
## Round( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el valor redondeado de un valor numérico según el número especificado de decimales.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Retrieve the rounded value of -16.238167
ResRound = Round(-16.238167, 2)
// Returns -16.24
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = Round(<Numeric value> [, <Number of decimals>])`
---

**`<Result>: (Entero o real)`**

Valor redondeado del valor numérico especificado.

**`<Numeric value>: (Real o moneda)`**

Valor numérico para redondear. Le recomendamos que use el tipo de moneda para que get obtenga resultados fiables.

**`<Number of decimals>: (Entero opcional)`**

Número de decimales a devolver. No se devuelve ningún decimal si no se especifica este parámetro.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Valor numérico
<a name="valor_numerico_ELTPARAGRAPHE000076"></a>

Si el valor &lt;Numérico&gt; está incluido entre:

- 0 y 0,49: el número se redondea a 0. Por ejemplo, 3.2 se redondea a 3 (sin decimales).

- 0,5 y 0,99: el número se redondea a 1. Por ejemplo, 3.7 se redondea a 4 (sin decimales).




Atención: El redondeo realizado con el tipo "Real" no es preciso.. En efecto, las operaciones realizadas con los tipos "reales" no son precisas debido a la representación informática de los reales. Para get un redondeo exacto, le recomendamos que utilice el tipo [Moneda](../Motscles/1514043.md) o [Numeric](../Motscles/1514072.md) que utiliza una representación de memoria exacta. Para más detalles, consulte el tipo [real](../Motscles/1514048.md).
<a name="NOTE0_2"></a>




### Separador decimal
<a name="separador_decimal_ELTPARAGRAPHE000098"></a>

El separador decimal para los números reales es ". (punto).

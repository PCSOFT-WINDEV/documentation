
## SysInstance( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la instancia de un módulo.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Instance of current application
ResInstance = SysInstance()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = SysInstance([<Module Name>])`
---

**`<Result>: (Integro)`**

Instancia del módulo especificado.

**`<Module Name>: (Cadena de caracteres opcional)`**

Corresponde a:

- el nombre y la ruta completa (o relativa) del archivo ejecutable (extensión".EXE") para la aplicación actual. Se permite el uso de un horario UNC ruta.

- el nombre y la ruta completa (o relativa) de una biblioteca WINDEV (archivo ".DLL") actualmente cargada en memoria ([CargaDLLL](../WDLang1/3014006.md)). Se permite el uso de un horario UNC ruta.




Si no se especifica este parámetro, se devuelve la instancia de la aplicación actual.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### "Parámetro Nombre del módulo
<a name="parametro_nombre_del_modulo_ELTPARAGRAPHE000042"></a>

El parámetro &lt;Module Name&gt; no debe contener el nombre y ruta de un archivo ejecutable diferente al de la aplicación actual.

Para saber si un ejecutable se está ejecutando actualmente, utilice [Ejecución ExeRunning](../WDLang1/3035003.md).
<a name="NOTE0_2"></a>




### Equivalencia
<a name="equivalencia_ELTPARAGRAPHE000055"></a>


```wl
ResInstance = SysInstance()
```


equivale a 


```wl
ResInstance = Instance()
```

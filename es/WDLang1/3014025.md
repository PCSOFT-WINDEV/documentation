


## Funciones de depuración
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


Las funciones de depuración son las siguientes:



|   |   |
| --- | --- |
| [dbgAssert](../WDLang1/3014022.md) | Permite detectar y señalar los posibles errores a través de una programación asertiva. |
| [dbgAuditOption](../WDLang1/1000018837.md) | Recupera o modifica las opciones de la auditoría dinámica. |
| [dbgAuditStatus](../WDLang1/1000018836.md) | Recupera y modifica el estado de la auditoría dinámica. |
| [dbgCheckDifferent](../WDLang1/1000019777.md) | Verifica si las expresiones especificadas son diferentes en la programación asertiva y en el modo de prueba automática. |
| [dbgCheckEqual](../WDLang1/1000019776.md) | Verifica si las expresiones son iguales en la programación asertiva y en el modo de prueba automática. |
| [dbgCheckFalse](../WDLang1/1000019781.md) | Verifica si una expresión es falsa en la programación asertiva y en el modo de prueba automática. |
| [dbgCheckNotNull](../WDLang1/1000019779.md) | Verifica si una expresión no es nula en la programación asertiva y en el modo de prueba automática. |
| [dbgCheckNull](../WDLang1/1000019778.md) | Verifica si una expresión es nula en la programación asertiva y en el modo de prueba automática. |
| [dbgCheckTrue](../WDLang1/1000019780.md) | Verifica si una expresión vale True en la programación asertiva y en el modo de prueba automática. |
| [dbgConnect](../WDLang1/1000024735.md) | Permite conectar un depurador remoto a una aplicación, servicio, Webservice, etc. |
| [dbgEnableAssert](../WDLang1/3014021.md) | Habilita o deshabilita la programación asertiva. |
| [dbgEnableAudit](../WDLang1/1000018835.md) | Permite la auditoría dinámica a través de la programación. |
| [dbgEnableLog](../WDLang1/1000017137.md) | Activa el registro de tiempo de ejecución mediante programación. |
| [dbgEndMemoryDiff](../WDLang1/1000018841.md) | Guarda un volcado de memoria que contiene los recursos de memoria que se han asignado y que no se han liberado desde la última llamada a [dbgStartMemoryDiff](../WDLang1/1000018840.md). |
| [dbgEndRecording](../WDLang1/1000017161.md) | Detiene la grabación del escenario. |
| [dbgError](../WDLang1/1000020526.md) | Provoca: <br><br>- una afirmación durante la prueba de elemento o proyecto (modo Go). <br><br>- un error en el modo de prueba automatizado. <br><br><br> |
| [dbgInfo](../WDLang1/3014024.md) | Devuelve una información de depuración. |
| [dbgLogOption](../WDLang1/1000017136.md) | Recupera o modifica las opciones del log de tiempo de ejecución. |
| [dbgLogStatus](../WDLang1/1000017138.md) | Devuelve y modifica el status del log de tiempo de ejecución. |
| [dbgOnError](../WDLang1/1000020722.md) | Define el WLanguage Procedure que se llamará cuando se detecte un error mediante una función dbgCheckXXX. |
| [dbgSaveDebugDump](../WDLang1/1000018834.md) | Guarda un archivo de volcado de la aplicación. |
| [dbgSaveMemoryDump](../WDLang1/1000018852.md) | Guarda un volcado de memoria utilizado por la aplicación. |
| [dbgStandardOutput](../WDLang1/3014020.md) | Escribe una información en el flujo de salida estándar "stdout" (también llamado "consola"). |
| [dbgStartMemoryDiff](../WDLang1/1000018840.md) | Inicia una operación diferencial de la memoria utilizada por la aplicación. |
| [dbgStartRecording](../WDLang1/1000017160.md) | Inicia la grabación de un escenario de prueba (también llamado escenario de reproducción). |
| [dbgWriteCommentAudit](../WDLang1/1000018839.md) | Escribe una comment en la actual auditoría dinámica. |
| [dbgWriteWarningAudit](../WDLang1/1000018838.md) | Escribe un aviso de tiempo de ejecución personalizado en la auditoría dinámica. |

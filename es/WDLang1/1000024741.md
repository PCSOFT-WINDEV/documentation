
## MaskPhoneNumber (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Recupera la máscara de entrada del número de teléfono de un país determinado.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Use Spain phone mask on EDT_Phone
EDT_Phone.InputMask = MaskPhoneNumber("ES")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = MaskPhoneNumber(<Country> [, <Option>])`
---

**`<Result>: (Variable de tipo InputMask)`**

Variable de tipo [InputMask](../WDLang1/1000024886.md) que contiene las características de la máscara de entrada del teléfono del país especificado. Este resultado puede asignarse directamente a la propiedad [InputMask](../Proprietes/2510033.md) .  

**`<Country>: (Cadena o variable de tipo Country)`**

País cuya máscara de entrada del número de teléfono debe recuperarse. Este parámetro puede corresponder a: 

- El código ISO alfabético o numérico del país deseado.

- Una variable de tipo [Country](../WDLang1/1000024921.md). 




**`<Option>: (Constante opcional de tipo Integer)`**

Características del número de teléfono:


|   |   |
| --- | --- |
| *mphoneForcePrefix* | Número de teléfono con prefijo internacional. <br>Ejemplo: +33 4 67 03 20 32. |
| *mphoneWithOrWithoutPrefix*<br>(Valor predeterminado) | Número de teléfono con o sin prefijo internacional. |
| *mphoneWithoutPrefix* | Número de teléfono sin prefijo internacional. <br>Ejemplo: 04 67 03 20 32. |

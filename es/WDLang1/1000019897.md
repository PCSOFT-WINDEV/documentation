
## WordCount (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el número de palabras en una cadena. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// The function returns 3
ResCount = WordCount("Example of use")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = WordCount(<String>)`
---

**`<Result>: (Entero)`**

Número de palabras en la cadena a analizar. 

**`<String>: (Cadena de caracteres)`**

Cadena de caracteres a manipular. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Todos los caracteres que no sean numéricos o alfabéticos se consideran separadores de palabras.

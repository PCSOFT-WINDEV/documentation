
## DateTimeLocalToUTC (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Convierte una fecha y hora local (zona horaria, hora de verano, hora de invierno) en tiempo universal (UTC). La información que se tiene en cuenta para realizar la conversión depende de la configuración del equipo actual.






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
Info(DateTimeLocalToUTC("20060101150000"))
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = DateTimeLocalToUTC(<Local date and time>)`
---

**`<Result>: (Cadena de caracteres o variable de tipo DateTime)`**

Fecha y hora UTC, expresadas en formato "AAAAMMDDHHmmSSCCC".

**`<Local date and time>: (Cadena de caracteres o variable de tipo DateTime)`**

Fecha y hora locales, expresadas en formato "AAAAMMDDHHmmSSCCC".  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
La fecha devuelta puede ser diferente de la fecha pasada como parámetro (por ejemplo, si la hora local es posterior a la medianoche y la hora universal es anterior a la medianoche).

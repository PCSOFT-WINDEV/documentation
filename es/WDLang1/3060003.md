
## FontList( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la lista de fuentes válidas instaladas en el ordenador actual (TrueType, Postscript, ...).

![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Sólo las fuentes soportadas en Java están listadas.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// List fonts in an information window
Info(FontList())
// List fonts in a List Box control
ListAdd(LIST_FontList, FontList())
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = FontList([<Font Type>])`
---

**`<Result>: (Cadena de caracteres)`**

Lista de fuentes en el siguiente formato: 

```txt
<Font 1> + CR + ... + <Font N>
```
Donde &lt;Font 1&gt; .. &lt;Font N&gt; corresponde al nombre de la fuente.

**`<Font Type>: (Constante opcional de tipo Integer)`**

Tipo de fuente a listar: 



|   |   |
| --- | --- |
| *fontDefault*<br>(Valor predeterminado) | Muestra la fuente del controlador de pantalla. |
| *fontPerFamily* | Enumera las familias de fuentes disponibles. Las familias de fuentes se enumeran en orden alfabético. Las variaciones de idioma no están listadas. <br><br>Observación: Esta constante puede combinarse con las constantes *fontDefault* y *fontPrinter*. |
| *fontPrinter* | Muestra la fuente del controlador de impresión. |


![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Este parámetro no está disponible. Sólo las fuentes soportadas en Java están listadas.

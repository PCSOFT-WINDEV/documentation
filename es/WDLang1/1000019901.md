


## StringReverse (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Invierte los caracteres de una cadena. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
Trace(StringReverse("ABC"))
// The trace window displays "CBA"
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = StringReverse(<String>)`
---

**`<Result>: (Cadena de caracteres Ansi, Unicode o Binaria)`**

Cadena de caracteres invertida. 

**`<String>: (Cadena de caracteres Ansi, Unicode o Binaria)`**

Cadena de caracteres a invertir.




## &lt;Hoja de cálculo&gt;.SelectMinus (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Deselecciona una o más células en una control Hoja de cálculo encontrada en una ventana.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
PSHEET_Spreadsheet.SelectMinus()
PSHEET_Spreadsheet.SelectPlus("A2", "B5")
PSHEET_Spreadsheet.StyleSelection(psheetBackgroundColor, LightRed)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Spreadsheet control>.SelectMinus([<Start cell> [, <End cell>]])`
---

**`<Spreadsheet control>: (Nombre del control)`**

Nombre del control Hoja de cálculo a utilizar.

**`<Start cell>: (Cadena de caracteres opcional)`**

Nombre de la celda a desmarcar. Este parámetro está en formato &lt;Letra de la columna&gt;&lt;Número de la fila&gt;. Por ejemplo: "A32". 

- Si no se especifica este parámetro, se desmarcan todas las celdas, así como las cabeceras de columna y fila.. 

- Si se especifica el parámetro &lt;Celda fin&gt;, &lt;Celda inicio&gt; corresponde a la primera celda que se deseleccionará de un rango de celdas.  

- Si no se especifica &lt;Celda final&gt;, sólo se deselecciona &lt;Celda inicial&gt;.




**`<End cell>: (Cadena de caracteres opcional)`**

Nombre de la celda correspondiente al final del rango de celdas a deseleccionar. Este parámetro está en formato &lt;Letra de la columna&gt;&lt;Número de la fila&gt;. Por ejemplo: "A32".  

- Si no se especifica este parámetro, sólo se deselecciona &lt;Celda inicial&gt;.. 

- Si &lt;Celda final&gt; corresponde a una celda posicionada antes de &lt;Celda inicial&gt;, las coordenadas se invierten y se realiza la deselección.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Si existe una selección y ésta contiene celdas en común con la selección realizada con **&lt;Hoja de cálculo&gt;.SelectMinus**, la selección existente se deselecciona por completo (no quedan espacios en blanco).

- Para seleccionar una sola celda y deseleccionar las demás, utilice la siguiente sintaxis:
	
	```wl
	PSHEET_Spreadsheet = "A3"
	```


- Esta función sólo se puede utilizar en un control Hoja de cálculo que se encuentre en una ventana..

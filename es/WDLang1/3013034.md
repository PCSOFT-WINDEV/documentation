
## EndAutomatedProcedure( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Detiene la ejecución actual o próxima de un automatic Procedure (definido por las propiedades del Procedure en el editor de código). Este automatic Procedure puede ser:

- un automatic Procedure no se ejecuta

- el procedimiento diferido

- un recurrente Procedure

- un Procedure en un Thread independiente.





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// The CheckInput procedure is an automatic delayed procedure called
// whenever an edit control is modified. 
// In the "Whenever the control is modified" code, the automatic procedure
// is stopped and restarted manually
EndAutomatedProcedure(CheckInput)
CheckInput(MySelf)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`EndAutomatedProcedure([<WLanguage procedure>])`
---

**`<WLanguage procedure>: (Nombre opcional de Procedure)`**

Nombre de la automatic Procedure a detener. Si no se especifica este parámetro, el actual Procedure se detiene.  

![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) El Procedure actualmente en funcionamiento continuará hasta que se llegue a su fin. Sólo se detendrá a la Repetition.

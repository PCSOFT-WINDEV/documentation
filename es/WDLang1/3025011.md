
## Altura del texto (función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Calcula la altura (en píxeles) del texto mostrado en una control Campo de entrada, en una control Estático o en una control Combo Box.. Esto le permite predecir la altura ocupada por el texto para ampliar la control si es necesario.








<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
Res is real = TextHeight(EDT_Edit1, "Hello")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = TextHeight(<Control name> , <Text> [, <Option>])`
---

**`<Result>: (Real)`**

Altura del texto (en píxeles) en la control especificada (según el estilo control).


**`<Control name>: (Cadena de caracteres)`**

Nombre de la edición control, control Estático o control Combo Box.


**`<Text>: (Cadena de caracteres)`**

Texto cuya altura se debe calcular.

**`<Option>: (Constante opcional de tipo Integer)`**

Configura el modo de cálculo de la altura:


|   |   |   |
| --- | --- | --- |
| *tWithMargin* | Permite tener en cuenta los márgenes set para el control (sangría izquierda, sangría derecha, etc.). <br>Esta constante puede ser combinada con la constante *tMultiline*. <br><br>![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Esta constante no está disponible. |
| *tDefault*<br>(valor predeterminado) | Altura del texto en la control. |   |
| *tMultiline* | Altura del texto teniendo en cuenta los caracteres CR (automatic o forzado por un "Carriage Return"). | ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=TexteHauteur_multi.gif) |
| *tSingleLine* | Altura de una Line teniendo en cuenta sólo los "Carriage Returns". | ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=TexteHauteur_mono.gif) |

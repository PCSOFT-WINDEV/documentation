


## DictionaryLoad( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Indica y carga el diccionario correspondiente a un idioma determinado.

Observaciones: 

- El diccionario debe ser un diccionario ortográfico en formato Hunspell.

- Los diccionarios francés e inglés/americano están cargados por Default. 

- En esta versión se utilizan los siguientes diccionarios: 

	- por la control Procesador de texto.

	- por la control Campo de entrada.

	- por las columnas Texto de los controles de tabla.






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Load the French dictionary
DictionaryLoad("Dictionary_fr.dic", languageFrench, 0)
DictionaryLoad("Dictionary_fr.dic", "fr-fr")

// Load the French Canadian dictionary
DictionaryLoad("Dictionary_fr.dic", "fr-ca")
DictionaryLoad("Dictionary_fr-ca.dic", languageFrench, 3) 
// 3 corresponds to the sub-language of Canada
```

## Observaciones


### Modo de funcionamiento del corrector ortográfico en un control Procesador de texto
<a name="modo_funcionamiento_del_corrector_ortografico_control_procesador_texto_ELTPARAGRAPHE000291"></a>



- Un docx Document contiene información sobre el lenguaje Document. Esta información se utiliza para identificar el lenguaje del diccionario que se utilizará. 

- Si el diccionario correspondiente al lenguaje Document se coloca al lado del ejecutable, se carga automáticamente y se utiliza para corregir el Document. Si no se encuentra el diccionario correspondiente al idioma Document, el control ortográfico no está disponible. 

- Cuando se crea un docx Document a través de la control Procesador de texto, el lenguaje Document corresponde al lenguaje de la aplicación actual (definido por [Nación](../WDLang1/3054014.md), por ejemplo). 




- [CheckSpell](../Proprietes/2510153.md) se utiliza para activar (o no) el corrector ortográfico en un control Procesador de texto. 




Para obtener más información, consulte [Revisar ortografía con diccionarios Hunspell](../WDChamp/1410087030.md).

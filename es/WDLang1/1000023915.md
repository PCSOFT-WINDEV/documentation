
## GetUUID( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Calcula un Identificador Único Universal de 128 bits ("UUID"). 

Este identificador es único en el mundo, independientemente de la máquina en la que se haya fabricado.. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
My UUID is UUID
MyUUID = GetUUID()
// Example of the UUID calculated: 95431e57-7d18-4fef-9d9d-96b231969149
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = GetUUID()`
---

**`<Result>: (UUID)`**

Identificador generado.
El formato UUID incluye separadores. Por ejemplo: "95431e57-7d18-4fef-9d9d-96b231969149".

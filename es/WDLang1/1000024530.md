


## JSONToString (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Convierte el contenido de una variable de tipo JSON a una cadena JSON.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
Person is JSON
Person.LastName = "MOORE"
Person.FirstName = "Vince"
// Displays: { "LastName":"MOORE", "FirstName":"Vince" }
Trace(JSONToString(Person))
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = JSONToString(<JSON variable> [, <Options>])`
---

**`<Result>: (Cadena de caracteres)`**

Cadena JSON resultante de la conversión de la variable JSON. 
Si se produce un error, la variable [ErrorOccurred](../WDLang1/3087001.md) se establece en <u><u><u><u>True</u></u></u></u> y la función [ErrorInfo](../WDLang1/3013008.md) permite conocer los detalles del error.

**`<JSON variable>: (Variable de tipo JSON)`**

Nombre de la variable de tipo [JSON](../Motscles/1000023611.md) que se convertirá en una cadena JSON.

**`<Options>: (Entero opcional)`**

Opción de formato de la cadena JSON:


|   |   |
| --- | --- |
| *psdFormatting* | Genera una cadena JSON con saltos de línea y sangría. |
| *psdMinified* | Genera una cadena JSON minificada, sin saltos de línea ni espacios. |

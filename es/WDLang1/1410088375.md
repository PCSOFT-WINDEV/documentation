


## &lt;Tabla&gt;.ToClipboard (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Copia el contenido de una Tabla o control Tabla TreeView al portapapeles. El contenido de control se copia en formato de texto: se convierte en una cadena y esta cadena se copia en el portapapeles.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
TABLE_TABLE1.ToClipboard(taNoTitle, ";")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Table control>.ToClipboard([<Options> [, <Separator> [, <Start row> [, <End row>]]]])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se copió el contenido del control en el portapapeles,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Table control>: (Nombre del control)`**

Nombre del control a manipular. Este control puede corresponder a: 

- un control Tabla. 

- un control Tabla TreeView.




**`<Options>: (Constante opcional)`**

Establece la información que se tendrá en cuenta en la cadena copiada en el portapapeles.


|   |   |
| --- | --- |
| *taColumnDisplayedOrder* | Exporta las columnas según el orden de las columnas que aparecen actualmente en la control (y no según el orden de las columnas definido en el editor). |
| *taColumnsTitles* | Se inserta el título de las columnas. |
| *taNoTitle*<br>(valor predeterminado) | Sólo se copian los datos. |
| *taNoTotal* | No exporta: <br><br>- las filas que contienen los totales, la media y la automática Count( estas filas son exportadas por Default).<br><br>- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) las filas que contienen cálculos personalizados (estas filas son exportadas por Default).<br><br><br> |
| *taSelectedLines* | Exporta sólo las filas seleccionadas (todas las filas son exportadas por Default). |
| *taWithInvisibleColumns* | También exporta las columnas de Invisible control.<br>![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Las columnas definidas como no imprimibles y no exportables se exportan. Estas columnas se definen: <br><br>- mediante programación: La propiedad [VisibleInExportAndPrint](../Proprietes/1000022320.md) se establece en <u><u><u><u>False</u></u></u></u>. <br><br>- en la pestaña "General" de la columna de la ventana de description: "Exportar e imprimir" set a "Nunca".<br><br><br> |



**`<Separator>: (Cadena de caracteres opcional)`**

Separador utilizado entre las columnas. El separador Default es la tabulación (TAB).

**`<Start row>: (Entero opcional)`**

Número de la fila donde se iniciará la exportación. Si no se especifica este parámetro, el fila de inicio es el primer fila del control.

**`<End row>: (Entero opcional)`**

Número de la fila donde terminará la exportación. Si no se especifica este parámetro, el fila final es el último fila del control.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Condiciones de uso
<a name="condiciones_uso_ELTPARAGRAPHE000256"></a>

La función **&lt;Tabla&gt;.ToClipboard** puede utilizarse en: 

- controles Tabla o Tabla TreeView basados en un archivo de datos.

- controles Tabla o Tabla TreeView rellenados mediante programación.

- controles de selección única o múltiple.



<a name="NOTE0_2"></a>


### Limitación
<a name="limitacion_ELTPARAGRAPHE000272"></a>

Las columnas combinadas se ignoran.

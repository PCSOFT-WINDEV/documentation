


## CalendarSelect (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la fecha seleccionada en un control Calendario.








<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Displays the selected date in a trace window
Trace(CalendarSelect(CAL_Calendar1))
```
<a name="Example2"></a>

```wl
// Retrieves the date in an edit control
EDT_Date = CalendarSelect(CAL_Calendar1)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = CalendarSelect(<Calendar control> [, <Index of the selected element>])`
---

**`<Result>: (Cadena de caracteres o variable de tipo Date)`**

Día seleccionado en la control Calendario. Este parámetro tiene el siguiente formato: "AAAAMMDD". Por ejemplo, si el día seleccionado es el 12 de noviembre de 2008, &lt;Resultado&gt; es igual a "20081112".

**`<Calendar control>: (Nombre del control)`**

Nombre de la control Calendario utilizada.

**`<Index of the selected element>: (Entero opcional)`**

Rango de selección para una multiselección Calendar. 
Si no se especifica este parámetro, se devuelve la fecha correspondiente al primer elemento seleccionado.

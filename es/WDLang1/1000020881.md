


## GanttTask (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **GanttTask** permite definir las características avanzadas de una tarea para un diagrama de Gantt. Las características de esta tarea pueden definirse y modificarse mediante diferentes propiedades de WLanguage. 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
NewTask1 is GanttTask
NewTask1.ID = "ID1"
NewTask1.DurationInHour = 0.10
NewTask1.Title = "Opening the trap"

NewTask2 is GanttTask
NewTask2.ID = 2
NewTask2.DurationInHour = 2.5
NewTask2.Title = "Launching the rocket"

// Adds the first task
GanttAddTask(COL_Gantt, NewTask1)

// Adds the task to the Table control by creating a new row
TableAddLine(TABLE_Example, NewTask2)

// Reorganizes the tasks in a Gantt Chart column
GanttOrganizeTask(COL_Gantt)
```

## Observaciones
<a name="NOTE0_1"></a>


### Propiedades específicas de las variables GanttTask
<a name="propiedades_especificas_las_variables_gantttask_ELTPARAGRAPHE000028"></a>

Las siguientes propiedades pueden ser utilizadas para manipular una tarea encontrada en un diagrama de Gantt:

| **Nombre de la propiedad** | **Tipo utilizado** | **Efecto** |
| --- | --- | --- |
| Acumulado | booleano | Gestión de tareas acumulativas: <br><br>- <u><u><u><u>True</u></u></u></u>: la tarea es acumulativa. La tarea incluye un marcador de "flecha" al inicio y al final. Este tipo de tarea se utiliza para simbolizar el progreso global de varias tareas subordinadas.<br><br>- <u><u><u><u>False</u></u></u></u>: la tarea no es acumulativa.<br><br><br> |
| BackgroundColor | Integro | Fondo color utilizado para mostrar la tarea en la columna del diagrama de Gantt.<br><br>Este color puede corresponder a:<br><br>- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),<br><br>- un color HSL (devuelto por la función [HSL](../WDLang1/3029057.md)),<br><br>- un color predefinido (ver [colores predefinidos de WLanguage](../WDLang5/3010002.md)).<br><br><br><br><br> |
| DuraciónEn el día | Real, entero o numérico | Duración de la tarea expresada en días laborables. En este caso, la fecha de finalización (**EndDate** Property) se calcula teniendo en cuenta los días laborables.<br><br>**Debe especificarse una duración.** <br><br>Si se especifica el **DuraciónEnDía** Property, no debe especificarse el **DuraciónEnHora** Property. |
| DuraciónEn hora | Real, entero o numérico | Duración de la tarea expresada en horas de trabajo. En este caso, la fecha de finalización (**EndDate** Property) se calcula teniendo en cuenta las horas de trabajo.<br><br>**Debe especificarse una duración.** <br><br>Si se especifica el **DuraciónEnHora** Property, no debe especificarse el **DuraciónEnDía** Property. |
| EndDate | Cadena de caracteres o variable de tipo DateTime | Fecha y hora de finalización de la tarea. <br><br>Esta propiedad es de solo lectura. Se calcula automáticamente teniendo en cuenta la duración especificada (en días o en horas), los días laborables y las horas de trabajo.. <br><br>Por ejemplo, si una tarea dura 24 horas y si el horario de trabajo se define de 8:00 a 20:00, la duración de la tarea será de set a 2 días en el diagrama de Gantt. |
| Fila | Integro | Número de la fila en el control Tabla donde se debe visualizar la tarea. |
| Hito | booleano | Gestión de tareas de hitos: <br><br>- <u><u><u><u>True</u></u></u></u>: la tarea es una tarea de hito. Una tarea de hito es una etiqueta usada para simbolizar un importante Event, una fecha límite intermedia. Una tarea de hito está representada por un diamante, y no tiene duración ni progreso.<br><br>- <u><u><u><u>False</u></u></u></u>: la tarea no es una tarea de Milestone.<br><br><br> |
| ID | Cadena de caracteres | Número utilizado para identificar la tarea.<br><br>**Debe especificarse este Property.** |
| Image | Image | Image mostrada en la tarea (si la tarea utiliza la ventana interna personalizada, IW_WinDevViewGanttTask). Esta propiedad puede corresponder a:<br><br>- el ruta a un Image accesible desde el ordenador actual,<br><br>- un archivo de Image que se encuentra en la biblioteca de aplicaciones,<br><br>- un control Imagen que contiene un Image,<br><br>- un dibujo realizado en una control Imagen con las funciones de dibujo y guardado en la memoria.<br><br><br> |
| Progress | Integro | Porcentaje de progreso de la tarea (entre 0 y 100). |
| ProgressColor | Integro | color utiliza para rellenar la barra de progresión de la tarea. Esta Property sólo se tiene en cuenta si el valor de la **Progress** Property no es 0. <br><br>Este color puede corresponder a:<br><br>- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),<br><br>- un color HSL (devuelto por la función [HSL](../WDLang1/3029057.md)),<br><br>- un color predefinido (ver [colores predefinidos de WLanguage](../WDLang5/3010002.md)).<br><br><br> |
| Responsable | Cadena de caracteres | Responsable de la tarea. Esta información no se visualiza en el diagrama de Gantt. |
| StartDate | Cadena de caracteres o variable de tipo DateTime | Fecha y hora de inicio de la tarea.<br><br>Si no se especifica esta Property, la fecha de inicio de la tarea se corresponde con los datos y la hora del sistema. |
| Título | Cadena de caracteres | Nombre de la tarea. Este nombre se muestra a la derecha de la tarea. |
| ToolTip | Cadena de caracteres | Texto del tooltip mostrado en la tarea. <br><br>De forma predeterminada, esta propiedad corresponde a una cadena vacía (""). |

### Utilización de variables GanttTask
<a name="utilizacion_variables_gantttask_ELTPARAGRAPHE000207"></a>

Las variables de tipo **GanttTask** son utilizadas por las siguientes funciones: 



|   |   |
| --- | --- |
| [GanttAddTask](../WDLang1/1000020904.md) | Añade una tarea en una columna del Diagrama de Gantt en una Tabla o control Tabla TreeView. |
| [GanttGetTask](../WDLang1/1000021026.md) | Recupera una tarea de su identificador en una columna del diagrama de Gantt (que se encuentra en una Tabla o control Tabla TreeView). |
| [GanttListTask](../WDLang1/1000021073.md) | Devuelve: <br><br>- la lista de tareas incluidas entre dos fechas determinadas en una columna del diagrama de Gantt, <br><br>- todas las tareas que se encuentran en una columna del diagrama de Gantt,<br><br>- una tarea específica (seleccionada o suspendida). <br><br><br> |




## mlzDeleteAll (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Borra todas las filas dinámicas de una Zona Mutilina control. 




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Reinitialize MZ_Network
mlzDeleteAll(MZ_Network)
mlzAddLine(MZ_Network)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`mlzDeleteAll(<Multiline Zone control>)`
---

**`<Multiline Zone control>: (Nombre del control)`**

Nombre de la control Zona multilínea a utilizar.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Todas las filas añadidas por [mlzAddLine](../WDLang1/1000020199.md) y [mlzInsertLine](../WDLang1/1000020207.md) son eliminadas.

- Las líneas estáticas definidas en el editor de ventanas no se ven afectadas.

- El tamaño de la control Zona multilínea se modifica para tener en cuenta las líneas borradas. : la control se reduce desde el fondo. Si los controles están situados debajo de la control Zona multilínea, se mueven hacia arriba.. La altura de la ventana de useful se reduce en consecuencia.

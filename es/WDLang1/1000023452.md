


## PDFReaderAbrir (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Abre un archivo PDF que se muestra en un lector de PDF control.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
PDFReaderOpen(PDF_MyPDFReader, "C:\MyDocument.pdf")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = PDFReaderOpen(<PDF Reader control> , <Path> [, <Password>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se abrió el archivo PDF correctamente,

- <u><u><u><u>False</u></u></u></u> en caso contrario. La Document que aparece actualmente en la control no se modifica. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.




**`<PDF Reader control>: (Nombre del control)`**

Nombre del control Lector PDF a manipular. 

**`<Path>: (Cadena de caracteres)`**

ruta completa o relativa del archivo PDF a abrir. 

**`<Password>: (Cadena de caracteres opcional)`**

Contraseña necesaria para abrir el archivo (archivo PDF protected en apertura). 

- Si no se especifica este parámetro y si el archivo PDF tiene contraseña protected, se mostrará una ventana que permitirá al usuario escribir la contraseña. 
	

- La función devolverá un error si la contraseña especificada no es válida para la Document. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Para cerrar el PDF Document que aparece en el lector de PDF control, utilice [PDFReaderClose](../WDLang1/1000023453.md).

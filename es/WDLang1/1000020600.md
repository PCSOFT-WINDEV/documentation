


## RadioButtonDelete (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Elimina una opción de un control Botón de opción. 




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Delete option #3 from a Radio Button control 
RadioButtonDelete(RADIO_Options, 3)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`RadioButtonDelete(<Radio Button control> , <Index>)`
---

**`<Radio Button control>: (Nombre del control)`**

Nombre de la control Botón de opción a utilizar.

**`<Index>: (Integro)`**

Índice de la opción a eliminar. Este índice debe estar comprendido entre 1 y el número de opciones de control Botón de opción. El [Conde](../Proprietes/2510097.md) Property obtiene el número de opciones en un control Botón de opción. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Cuando se eliminan las opciones de un control Botón de opción, el tamaño del control no se modifica. El tamaño de control puede modificarse mediante las propiedades [anchura](../Proprietes/2510059.md) y [Altura](../Proprietes/2510050.md).

- Tienes la capacidad de compactar las opciones para optimizar el aspecto de la control una vez que las opciones han sido eliminadas. 
	En el modo compactado, las opciones de una control Botón de opción se acercan.: 

	- cuando se hace una opción Invisible. 

	- cuando se elimina una opción.




- Para utilizar el modo compactado: 

	- marque "Compactar opciones invisibles" en la pestaña "Detalles" del control description.

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) utilice la propiedad [CompactOption](../Proprietes/1000020608.md). 




- Cuando se elimina una opción, si el botón de radio utiliza la numeración de opciones, las opciones que siguen a la opción eliminada se renumeran automáticamente. 

- Sólo se puede especificar el índice de la opción a eliminar. Si el botón de opción utiliza valores devueltos, no puede especificar el nombre de la opción que desea eliminar.. Sin embargo, puede especificar el índice que corresponde a la opción.

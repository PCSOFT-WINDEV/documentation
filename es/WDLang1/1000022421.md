
## EditCaret( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica el cursor de desplazamiento estándar para todos los controles de edición de la aplicación.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
EditCaret("MyCaret.cur")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`EditCaret(<Cursor>)`
---

**`<Cursor>: (Cadena de caracteres)`**



- Nombre del archivo del cursor (archivo .cur o .ani). 

- Vaciar la cadena ("") para volver a activar el cursor estándar.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- El [MouseCursor](../Proprietes/2510026.md) Property se utiliza para modificar el cursor de rastreo de un control. Si el cursor hover fue modificado por la propiedad [MouseCursor](../Proprietes/2510026.md), **EditCaret** no tiene efecto sobre el cursor control. 

- En el editor, se puede cambiar el cursor de desplazamiento utilizado para un control en la pestaña "UI" de la ventana de control description.

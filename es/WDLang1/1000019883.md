
## LastDayOfWeek (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la fecha del último día de la semana que corresponde a una fecha determinada. Esta opción permite identificar fácilmente la fecha a la que corresponde un domingo para una fecha determinada. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Click the Button control "Rest of the week"
EDT_END_DATE = LastDayOfWeek(EDT_START_DATE)
```
<a name="Example2"></a>

```wl
NbDaysBeforeWE = DateDifference(LastDayOfWeek(MyDate), MyDate)-2
```
<a name="Example3"></a>

```wl
LastDayOfWeek("20111103") // Returns 20110611
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = LastDayOfWeek(<Date>)`
---

**`<Result>: (Cadena de caracteres o variable de tipo Date)`**

Fecha en formato AAAAMMDD que corresponde al último día de la semana.  

**`<Date>: (Cadena de caracteres o variable de tipo Date)`**

Fecha en formato AAAAMMDD para la que se obtendrá el último día de la semana.




## DrawingStyle( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Permite averiguarlo y modificarlo:

- el modo de visualización de los botones de Grayed que se encuentran en las ventanas de su aplicación.

- el modo de visualización de Border translúcido para las ventanas de su aplicación.

- el modo de cargar imágenes grandes. 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Draws the grayed icons in shades of gray
DrawingStyle(styleGrayedIconGradient, True)
```
<a name="Example2"></a>

```wl
// Removes the translucent border effect
DrawingStyle(styleTranslucentWinBorder, False)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Modificar el modo de visualización de los elementos (botones, ventanas)

`DrawingStyle(<Element> , <Display mode>)`
---

**`<Element>: (Constante de tipo Integer)`**

Elemento afectado por el ajuste:


|   |   |
| --- | --- |
| *styleGrayedIconGradient* | Configura el efecto Grayed para los botones. |
| *styleImageLowQuality* | Configurar el modo de carga de imágenes grandes (mayores de 640x480). |
| *styleIndentBorderOverlap* | Configura el número de píxeles entre el texto y el Border en el modo de visualización "superposición". |
| *styleTranslucentWinBorder* | Configure el estilo translúcido de las ventanas. |



**`<Display mode>: (Booleano o entero)`**

Nuevo modo de visualización del elemento:

- **Para el efecto translúcido**:

	- <u><u><u><u>True</u></u></u></u>: Las ventanas con una Border translúcida se muestran con este efecto.

	- <u><u><u><u>False</u></u></u></u>: El efecto "Bordes translúcidos" de las ventanas se ignora.




- **Para el modo de carga Image**:

	- <u><u><u><u>True</u></u></u></u>: Carga rápida: el tiempo de carga de las imágenes tiene prioridad sobre su calidad.

	- <u><u><u><u>False</u></u></u></u>: Carga más lenta: la calidad de las imágenes tiene prioridad sobre el tiempo de carga.




- **Para el efecto Grayed fuera**:

	- <u><u><u><u>True</u></u></u></u>: Los iconos, imágenes de los botones de la Grayed se muestran en tonos de gris. Si se selecciona este modo, la pantalla puede ser más larga, pero get un mejor resultado.

	- <u><u><u><u>False</u></u></u></u>: Los iconos, las imágenes de los botones grises se muestran en tonos de gris automatic. En este modo, si la Image no contiene ningún píxel negro, la versión Grayed no dibuja nada.  




- **Para el desplazamiento del modo de dibujo "solapado**: Número de píxeles para el espacio entre la Line de la frontera y el texto de la leyenda.





<a name="SYNTAX2"></a>

### Averiguar el modo de visualización de los elementos (botones, ventanas)

`<Result> = DrawingStyle(<Element>)`
---

**`<Result>: (Booleano o entero)`**

Modo de visualización actual del elemento especificado o número de píxeles de desplazamiento para los subtítulos en modo "solapado".

**`<Element>: (Constante de tipo Integer)`**

Elemento afectado por el ajuste.


|   |   |
| --- | --- |
| *styleGrayedIconGradient* | Configurar el efecto Grayed para los botones.<br>Si &lt;Resultado&gt; = <u><u><u><u>True</u></u></u></u>, la imagen de los botones se muestra en tonos de gris.<br>Si &lt;Resultado&gt; = <u><u><u><u>False</u></u></u></u>, la imagen de los botones se pone en gris según el sistema de escala de grises de Windows 95. |
| *styleImageLowQuality* | Configurar el modo de carga de imágenes grandes (mayores de 640x480).<br>Si &lt;Resultado&gt; = <u><u><u><u>True</u></u></u></u>, el tiempo de carga de las imágenes tiene prioridad sobre la calidad.<br>Si &lt;Resultado&gt; = <u><u><u><u>False</u></u></u></u>, la calidad de las imágenes tiene prioridad sobre el tiempo de carga. |
| *styleIndentBorderOverlap* | Configura el espacio (en píxeles) entre el texto y la Border en el modo de visualización "overlap". |
| *styleTranslucentWinBorder* | Configurar el estilo translúcido de las ventanas. <br>Si &lt;Resultado&gt; = <u><u><u><u>True</u></u></u></u>, las ventanas con bordes translúcidos se muestran con este efecto. <br>Si &lt;Resultado&gt; = <u><u><u><u>False</u></u></u></u>, el efecto "Bordes traslúcidos" se ignora.<br> |

## Observaciones


### Gestión de las imágenes de Grayed
<a name="gestion_las_imagenes_grayed_ELTPARAGRAPHE000161"></a>

Según del modo de visualización seleccionado con la constante *styleGrayedIconGradient*:

- &lt;Modo de visualización&gt; = False: Los botones se dibujan de la siguiente manera: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=StyleDessin_stylePictoGrise11_2.GIF)
 **Atención**: Si el botón no contiene ningún píxel negro, la versión Grayed no dibujará nada.

- &lt;Modo de visualización&gt; = True: Los botones están dibujados en tonos de gris: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=StyleDessin_stylePictoGrise11_1.GIF)

### Gestión del efecto traslúcido
<a name="gestion_del_efecto_traslucido_ELTPARAGRAPHE000176"></a>

La gestión del efecto translúcido equivale a "Desactivar el efecto translúcido de las ventanas" de el menú contextual de las ventanas.

También tiene la posibilidad de desactivar la gestión del efecto translúcido con [WinDisableEffect](../WDLang1/1000017172.md).

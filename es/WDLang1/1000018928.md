


## TreeMapCount (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el número de elementos que se encuentran en una control TreeMap.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Number of elements in TMAP_File
ResNumberElements = TreeMapCount(TMAP_File)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = TreeMapCount(<TreeMap control>)`
---

**`<Result>: (Integro)`**

Número de elementos encontrados en el TreeMap (0 si el control está vacío). 
**Observación**: Se tienen en cuenta todos los elementos (incluso los elementos cuyo peso es 0). 

**`<TreeMap control>: (Nombre del control)`**

Nombre de control TreeMap a utilizar.
Si este parámetro corresponde a una cadena vacía (""), el control utilizado será el control TreeMap al que pertenece el Event actual.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
La función **TreeMapCount** y la [Conde](../Proprietes/2510097.md) Property son equivalentes:

```txt
<Nb elements> = TreeMapCount(<TreeMap control>)
```

```txt
<Nb elements> = <TreeMap control>.Count
```

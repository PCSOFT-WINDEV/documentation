
## WindowCount( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Calcula el número de ventanas abiertas actualmente en la aplicación actual.








<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF WindowCount() >= 1 THEN
	Info("There are " + WindowCount() + "opened windows")
ELSE
	Info("No window is opened")
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = WindowCount()`
---

**`<Result>: (Integro)`**



- Número de ventanas abiertas actualmente,

- 0 si la función se ejecuta en el código de inicialización del proyecto.

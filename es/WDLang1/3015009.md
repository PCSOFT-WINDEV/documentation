
## EndTimerSys( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Finaliza la ejecución de una timer activada por [TimerSys](../WDLang1/3015006.md).






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Procedure used to display the time in EDT_TIME1 on a regular basis
PROCEDURE Display_Time()
EDT_Time1 = TimeSys()

// Process when opening the window
// Display_Time will be automatically called every second
TimerNum is int
TimerNum = TimerSys("Display_Time", 100, 1)
IF TimerNum = 0 THEN
	Error("Unable to create the timer")
END
```



```wl
// Stop the timer during the click on a "BTN_StopTimer" button
EndTimerSys(TimerNum)
```





<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`EndTimerSys([<Timer number>])`
---

**`<Timer number>: (Entero opcional)`**

Número de la timer a destruir (devuelto por [TimerSys](../WDLang1/3015006.md)).
Si se especifica este número, la timer seleccionada se interrumpe. La función no tiene ningún efecto si la timer no existe o si ya ha sido interrumpida.
Si no se especifica este número y si se utiliza **EndTimerSys** en el código de un Procedure llamado por un timer, el timer se interrumpe. De lo contrario, la función **EndTimerSys** no tiene ningún efecto.




## GanttOrganizeTask (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Reorganiza las tareas de un diagrama de Gantt según las restricciones (dependencias y recursos).. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Reorganizes tasks
GanttOrganizeTask(COL_Gantt)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`GanttOrganizeTask(<Gantt column>)`
---

**`<Gantt column>: (Nombre del control)`**

Nombre de la columna (de un control Tabla o Tabla TreeView) de tipo Diagrama de Gantt a utilizar.

## Componente:


## Funciones de gestión áreas de memoria
			

<DIV class="specObsolete">
	<IMG style="float:left;margin-right:10px;" src="https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=warning_big.png"><B>Advertencia</B><BR>
	A partir de la versión <B>20</B>, esta funcionalidad se conserva por motivos de compatibilidad. Se recomienda utilizar la función [Associative arrays](../Motscles/1514058.md).

<a name="NOTE1_1"></a>


Para gestionar las zonas de memoria se utilizan las siguientes funciones: 



|   |   |
| --- | --- |
| [fMemExist](../WDLang1/1000018916.md) | Comprueba la existencia de una zona de memoria compartida. |
| [fMemOpen](../WDLang1/1000018917.md) | Abre una zona de memoria compartida entre varias aplicaciones. |
| [fMemWait](../WDLang1/1000018915.md) | Espera el final de Process sobre las notificaciones de modificación de una zona de memoria compartida. |

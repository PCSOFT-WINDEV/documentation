


## TableSearchEverywhere (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Busca un valor en todas las columnas de los siguientes controles: 

- Control Tabla,

- Control Tabla TreeView, 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) control Tabla se muestra en un control Combo Box.





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Click code of the Button control [Search] 
nPos is int = TableSearchEverywhere(TABLE_Order, EDT_Search, searchContains)
// If found
IF nPos > 0 THEN
	// Selects the row
	TABLE_Order = nPos
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = TableSearchEverywhere(<Table control> , <Search element> [, <Type of search> [, <Start>]])`
---

**`<Result>: (Integro)`**



- Índice del elemento encontrado, 

- -1 si la búsqueda falla.


El elemento encontrado no está seleccionado. Para seleccionar el elemento encontrado, utilice la función [TableSelectPlus](../WDLang1/3074031.md).

**`<Table control>: (Nombre del control)`**

Nombre de la Tabla o control Tabla TreeView en la que se realizará la búsqueda. La búsqueda se realiza en todas las columnas.

**`<Search element>: (Tipo del elemento buscado)`**

Valor a buscar en todas las columnas. El valor se busca en las columnas de texto de Visible.

Observación: La búsqueda no distingue entre mayúsculas y minúsculas, ni siquiera en las búsquedas de coincidencia exacta.

**`<Type of search>: (Constante opcional de tipo Integer)`**

Tipo de búsqueda que se realizará: 



|   |   |
| --- | --- |
| *searchContains*<br>(Valor predeterminado) | **Búsqueda genérica de tipo "Contiene"**<br><br>&lt;Resultado&gt; corresponde al índice del elemento con un valor que contiene &lt;Elemento de búsqueda&gt;. |
| *searchExactMatch* | **Búsqueda exacta** ("Igual a")<br><br>&lt;Resultado&gt; corresponde al índice del elemento con valor estrictamente igual a &lt;Elemento de búsqueda&gt;.<br><br>Atención: Los espacios al principio del texto se ignoran. |
| *searchStartsWith* | **Búsqueda genérica de tipo "Comienza por"**<br><br>&lt;Resultado&gt; corresponde al índice del elemento con un valor que comienza con &lt;Elemento de búsqueda&gt;. <br><br>Atención: Los espacios al principio del texto se ignoran. |



**`<Start>: (Entero opcional)`**

Índice de la fila de partida para la búsqueda (1 por Default).
Este parámetro se ignora en controles Tabla basados en un archivo de datos.

## Observaciones


### Condiciones de uso
<a name="condiciones_uso_ELTPARAGRAPHE000110"></a>

La función **TableSearchEverywhere** puede utilizarse en:

- una Tabla o Tabla TreeView con fuente de datos en memoria (esta función no está disponible para Table o TreeView controles Tabla con acceso directo a la fuente de datos),

- controles Tabla o Tabla TreeView rellenados mediante programación.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una tabla que se muestra en un control Combo Box.

- un control de selección única o de selección múltiple.




En el caso de los Table o TreeView controles Tabla basados en un archivo de datos, el modo de iteración debe ser automatic.

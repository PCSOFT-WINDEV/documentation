


## Funciones de gestión multitáctil
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


Las siguientes funciones sirven para gestionar los gestos en un Phone (iOS/Android) o en un sitio (multitáctil):



|   |   |
| --- | --- |
| [GestureCurrentPointer](../WDLang1/1000019751.md) | Devuelve el índice del puntero (dedo o lápiz óptico) que activó la llamada a un Process específico de la función multitáctil en un control Imagen. |
| [GestureNbPointer](../WDLang1/1000019750.md) | Devuelve el número de punteros en Contact con la pantalla. |
| [GesturePointerType](../WDLang1/1000023785.md) | Devuelve el tipo de puntero utilizado en el dispositivo: dedo, ratón o stylus. |
| [GesturePosX](../WDLang1/1000019741.md) | Devuelve la posición horizontal del puntero (dedo o lápiz). |
| [GesturePosY](../WDLang1/1000019745.md) | Devuelve la posición vertical del puntero (dedo o lápiz). |

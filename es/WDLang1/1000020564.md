
## IWListNext( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Muestra la próxima ventana interna en la lista de ventanas internas buscadas por un control Ventana Interna.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IWListNext(IW_InternalWindowControl)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = IWListNext(<Internal Window control>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se mostró la ventana interna siguiente,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Internal Window control>: (Nombre del control)`**

Nombre de la control Ventana interna a utilizar.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
**IWListNext** y [IWListPrevious](../WDLang1/1000020565.md) utilizan una animación al cambiar una ventana interna. Para cambiar una ventana interna sin utilizar animaciones, llame a [IWListPosition](../WDLang1/1000020562.md).

![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Las animaciones no están disponibles.




## <Diagram editor>Guardar (función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Guarda una Diagram como archivo "diag.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
DIAGEDT_MyDiagram.Save("diagram1.wddiag")
IF ErrorOccurred THEN
	Error("File not saved: " + ErrorInfo())
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Diagram Editor>.Save(<File>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se guardó correctamente,

- <u><u><u><u>False</u></u></u></u> en caso contrario. [ErrorInfo](../WDLang1/3013008.md) devuelve más detalles sobre el error.




**`<Diagram Editor>: (Nombre del control)`**

Nombre del editor de Diagram control a utilizar. 

**`<File>: (Cadena de caracteres)`**

ruta completa del fichero "diag".

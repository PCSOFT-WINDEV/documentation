
## dbgLogOption( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Recupera o modifica las opciones del log de tiempo de ejecución.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Find out the options of the runtime log
IF BinaryAND(dbgLogOption(), LogWLanguageFunctions) = LogWLanguageFunctions THEN
	Trace("Log with functions")
END
IF BinaryAND(dbgLogOption(), LogParameters) = LogParameters THEN
	Trace("Log with parameters")
END
IF BinaryAND(dbgLogOption(), LogProcessOnly) = LogProcessOnly THEN
	Trace("Log with processes")
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar las opciones del registro de tiempo de ejecución

`<Result> = dbgLogOption()`
---

**`<Result>: (Constante de tipo Integer)`**

Opciones del log de tiempo de ejecución:


|   |   |
| --- | --- |
| *LogAll* | Se recupera el log de tiempo de ejecución:<br><br>- las funciones de WLanguage<br><br>- los parámetros de las funciones<br><br>- los procesos<br><br><br> |
| *LogParameters* | Los parámetros de las funciones se registran en el log de tiempo de ejecución. |
| *LogProcessOnly* | El log de tiempo de ejecución sólo recupera información sobre los procesos. |
| *LogWLanguageFunctions* | Las funciones WLanguage se registran en el log de tiempo de ejecución. |

## Observaciones




### Modo de funcionamiento del log de tiempo de ejecución
<a name="modo_funcionamiento_del_log_tiempo_ejecucion_ELTPARAGRAPHE000128"></a>

Por Default, el registro de tiempo de ejecución se crea con todas las opciones: lista de funciones WLanguage así como de parámetros de función. Estas opciones se utilizan para presentar las acciones en detalle, pero el tamaño del registro creado aumentará rápidamente.

Para limitar el tamaño de Log( en tiempo de ejecución cuando un registro se realiza durante mucho tiempo, por ejemplo), tiene la posibilidad de ignorar estas opciones.

Para limitar el tamaño del registro en tiempo de ejecución, también tiene la posibilidad de habilitar el registro en tiempo de ejecución para la sección de la aplicación que está causando problemas con [dbgLogStatus](../WDLang1/1000017138.md).

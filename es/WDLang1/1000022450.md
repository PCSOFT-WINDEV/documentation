
## CameraCaptura (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Guarda el video que actualmente transmite una "cámara" como un Image o una secuencia de video.  

Los formatos de almacenamiento de vídeo son: 

- ![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) una Image en formato BMP o una secuencia de vídeo en formato AVI. 




Observaciones: 

- Esta función es equivalente a la función [VideoCapture](../WDLang1/3079007.md).





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 
```wl
// Save the video transmitted in the "CAM_Kitchen" control
// in the format of a 1-minute AVI ("C:\Temp\MyVideo.AVI")
ResSave = CameraCapture(CAM_Kitchen, "C:\Temp\MyVideo.AVI", viVideoCapture, 60)
IF ResSave = True THEN
	Info("Video successfully saved")
ELSE
	Error(ErrorInfo())
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = CameraCapture(<Camera control> , <Name of created file> [, <Type of capture> [, <Capture duration> [, <Number of images per second> [, <Display during capture>]]]])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se guardó correctamente el video,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Camera control>: (Nombre del control)`**

Nombre del control Cámara a utilizar. No es necesario mostrar el vídeo en un control Cámara para poder guardarlo.

**`<Name of created file>: (Cadena de caracteres)`**

Nombre y ruta completa (o relativa) del archivo a crear.

![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) Si se guarda un Image, se crea un archivo BMP. Si se guarda una secuencia de vídeo, se crea un archivo AVI.



**Atención**: No se puede utilizar ningún ruta UNC.

**`<Type of capture>: (Constante opcional de tipo Integer)`**

Tipo de captura:


|   |   |
| --- | --- |
| *viPictureCapture* | ![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) Captura un Image en el formato de un archivo BMP.<br> |
| *viVideoCapture*<br>(Valor predeterminado) | ![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) Capture una secuencia de vídeo en el formato de un archivo AVI.<br> |



**`<Capture duration>: (Entero o duración opcional)`**

Duración de la captura (en segundos). Si este parámetro es set a 0 o si no se especifica, la captura se detendrá:

- durante la llamada a [VideoStop](../WDLang1/3079001.md).

- cuando el disco especificado está lleno.


**Observación**: Este parámetro puede corresponder a: 

- un número entero correspondiente al número de segundos,

- una variable de tipo Duration,

- la duración en un formato legible ('1 s' por ejemplo). 




**`<Number of images per second>: (Entero opcional)`**

Número de imágenes guardadas por segundo (15 por Default). Este parámetro solo se tiene en cuenta si &lt;Tipo de captura&gt; es igual a *viVideoCapture*.

**`<Display during capture>: (Booleano opcional)`**



- <u><u><u><u>True</u></u></u></u> (por Default) si el vídeo transmitido actualmente debe ser visto en el control Cámara especificado,

- <u><u><u><u>False</u></u></u></u> en caso contrario.

<a name="NOTE0_1"></a>

## Observaciones




### Guardar como imagen
<a name="guardar_como_imagen_ELTPARAGRAPHE000140"></a>

Solo se guardará en formato BMP la imagen que se muestra al ejecutar la función **CameraCaptura**. Para guardar esta imagen, también se puede utilizar:

- la función [dSaveImageBMP](../WDLang1/3029026.md): la imagen se guardará en formato BMP.

- la función [dSaveImageGIF](../WDLang1/3029040.md): el Image se guardará en formato GIF.

- la función [dSaveImageJPEG](../WDLang1/3029038.md): el Image se guardará en formato JPEG.





### Condiciones de uso: 
<a name="condiciones_uso_ELTPARAGRAPHE000160"></a>

- ![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) Para utilizar esta función, se debe conectar una "cámara web" al ordenador actual.

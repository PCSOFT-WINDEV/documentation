


## &lt;Calendario&gt;.SelectCount (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el número de días seleccionados en una control Calendario.  








<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Browse the dates selected in the Calendar control
FOR i = 1 _TO_ CAL_Calendar1.SelectCount()
	Trace(CAL_Calendar1.Select(i))
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Calendar control>.SelectCount()`
---

**`<Result>: (Integro)`**

Número de elementos seleccionados en el control Calendario. Este parámetro:

- puede ser mayor que 1 sólo si la control Calendario es multiselección,

- es igual a 1 si la control Calendario no es multiselección.




**`<Calendar control>: (Nombre del control)`**

Nombre de control Calendario utilizado.

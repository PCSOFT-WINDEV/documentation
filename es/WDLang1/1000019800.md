


## &lt;Borrar todo (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Borra todos los elementos de un control TreeMap.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Deletes all elements from the TreeMap control
TMAP_Wine.DeleteAll()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<TreeMap control>.DeleteAll()`
---

**`<TreeMap control>: (Nombre del control)`**

Nombre de la control TreeMap utilizada.



<a name="XComponent"></a>

## Componente:
wd280obj.dll

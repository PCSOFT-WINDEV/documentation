


## &lt;Multimedia&gt;.duración (función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el tamaño del archivo multimedia que se está reproduciendo actualmente (en milisegundos) en el Multimedia control.  








<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 
```wl
MyFile is string = "C:\My music\mysong.mp3"
MM_Multimedia1.Open(Myfile)
Info("The duration of this file is " + MM_Multimedia1.Duration() + " milliseconds")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Multimedia control>.Duration()`
---

**`<Result>: (Integro)`**



- Duración (en milisegundos) del archivo multimedia,

- 0 si se produce un error. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Multimedia control>: (Nombre del control)`**

Nombre del Multimedia control que se va a utilizar.

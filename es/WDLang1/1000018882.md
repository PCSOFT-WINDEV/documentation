


## OrganizadorBorrarTodo (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Borra todas las citas de una control Agenda.

**Observaciones**: 

- En el caso de una control Agenda rellenada por programa, se borran todas las citas. 

- Para un control Agenda basado en un archivo de datos o en un Variable, las citas sólo se borran del control. Las citas se guardan en la fuente de datos relacionado al control. 








![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) En el modo de aplicación universal de Windows 10, esta función sólo se puede utilizar para los controles del organizador manipular en un reporte. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Delete appointments from the ORG_MyOrganizer control
OrganizerDeleteAll(ORG_MyOrganizer)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`OrganizerDeleteAll(<Organizer control>)`
---

**`<Organizer control>: (Nombre del control)`**

Nombre de la control Agenda que se va a utilizar. Este control puede corresponder a: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una control Agenda en una ventana.

- una control Agenda en una reporte.

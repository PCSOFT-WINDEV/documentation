
## Count( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el número de apariciones de una control en una ventana, en una página o en un grupo de control s.








<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Initialize the EDT_Label control if it exists
IF Count(EDT_Label) > 0 THEN
	Display_Label()
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = Count(<Search control>)`
---

**`<Result>: (Integro)`**



- Número de controles con el mismo nombre que se encuentran en la ventana o en la página.

- Número de controles que se encuentran en un grupo de controles.

- 0 si la control no existe.




**`<Search control>: (Nombre o cadena de caracteres de control)`**

Nombre del control o nombre del grupo de control s. 
Si este parámetro corresponde a una cadena vacía (""), se sustituye automáticamente por el nombre del control al que pertenece la Event actual. 
Para los controles indexados, no es necesario especificar la índice.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Invisible, Inactive o Grayed fuera control
<a name="invisible_inactive_grayed_fuera_control_ELTPARAGRAPHE000084"></a>

Se considera que un Invisible, Inactive o control en gris se encuentra en la ventana o en la página.
<a name="NOTE0_2"></a>




### Varios
<a name="varios_ELTPARAGRAPHE000092"></a>

**Atención**: Un control no debe ser usado directamente en un Process donde se comprueba su existencia. Se producirá un error de ejecución si el control no existe.

Ejemplo (de lo que no se debe hacer):


```wl
IF Count(EDT_Control1) > 0 THEN
	EDT_Control1 = "Foo"
END
```


Ejemplo (de qué hacer):


```wl
IF Count(EDT_Control1) > 0 THEN
	CallProcedure(EDT_Control1)
END
```

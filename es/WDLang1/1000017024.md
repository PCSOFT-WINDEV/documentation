
## Abandon( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Cierra la ventana actual ejecutando el "Click" Event del Cancel control Botón.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Click BTN_CANCEL
Abandon()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`Abandon()`
---



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Modo de funcionamiento
<a name="modo_funcionamiento_ELTPARAGRAPHE000056"></a>

Si la ventana contiene un control Botón de tipo CANCELAR, se ejecuta el evento "Clic" del control Botón . Atención:

- Si el "Click" Event del tipo Cancel control Botón cierra la ventana, la ventana se cierra.

- Si el "Click" Event del tipo Cancel control Botón no cierra la ventana, la ventana no se cierra.




Si la ventana no contiene Cancel control Botón, la ventana se cierra.. No se devuelve ningún valor de retorno.

Si se llama a la función **Abandon** desde un control Botón de tipo CANCELAR, se cierra la ventana (incluso si el evento "Clic" del control Botón no utiliza la función [Cerrar](../WDLang1/3038018.md)). Si un valor fue devuelto (por [Cerrar](../WDLang1/3038018.md) por ejemplo) en el evento "Click" del control Botón, este valor no es devuelto.

Llamar a la función **Abandon** es equivalente a Alt + F4.

**Observación**: si la ventana tiene varios controles del Botón Cancel, el comportamiento no está definido.
<a name="NOTE0_2"></a>




### Recordatorio: Los controles del Botón para usar en una ventana
<a name="recordatorio_los_controles_del_boton_para_usar_una_ventana_ELTPARAGRAPHE000085"></a>

- **Si la ventana no contiene controles Campo de entrada**, todos los controles Botón pueden ser de tipo NORMAL.

- **Si la ventana contiene al menos un control Campo de entrada**, se deben utilizar los siguientes tipos de controles Botón:

	- VALIDAR control Botón,

	- el Cancel control Botón,

	- un control Botón DE AYUDA (si los controles tienen ventanas de ayuda de associated),

	- cualquier otro botón de INTERRUPCIÓN Controles

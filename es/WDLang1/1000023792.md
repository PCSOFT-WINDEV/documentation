


## &lt;Chrono&gt;.Resume (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Reinicia un cronómetro que se detuvo con las funciones [&lt;Chrono&gt;.End](../WDLang1/1000023789.md) o [&lt;Chrono&gt;.Pause](../WDLang1/1000023790.md). La duración cronometrada a partir de la función **&lt;Chrono&gt;.Resume** se acumulará a la duración calculada previamente.








<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo



![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png)![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 
```wl
ProcessDuration is Duration 
MyChrono is Chrono 
MyChrono.Start()
MyFunction1()
ProcessDuration = MyChrono.Pause()
Info("The MyFunction1 process lasted " + ProcessDuration)
MyChrono.Resume()
MyFunction2()
ProcessDuration = MyChrono.End()
Info("The MyFunction1 and MyFunction2 processes lasted " + ProcessDuration)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Stopwatch>.Resumes()`
---

**`<Stopwatch>: (Variable de tipo Chrono)`**

Nombre de la variable de tipo [Chrono](../WDLang1/1000023560.md) que corresponde al cronómetro a utilizar.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Varios
<a name="varios_ELTPARAGRAPHE000303"></a>

 La función **&lt;Chrono&gt;.Resume** no tiene ningún efecto si se utiliza en un cronómetro que ya está en marcha.
<a name="NOTE0_2"></a>


### Error
<a name="error_ELTPARAGRAPHE000315"></a>

Si no se llama a la función [&lt;Chrono&gt;.Start](../WDLang1/1000023788.md) antes de la función **&lt;Chrono&gt;.Resume**, se produce un error WLanguage.
<a name="NOTE0_3"></a>


### Hilos
<a name="hilos_ELTPARAGRAPHE000332"></a>

Las funciones de cronometraje (Chronoxxx) relacionadas con el mismo cronómetro deben utilizarse en el mismo hilo.

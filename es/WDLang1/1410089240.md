


## KanbanCount (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el número de tarjetas o listas en un control Kanban. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Displays the total number of cards in the Kanban control
STC_NbTasks = "Total: " + KanbanCount(KANBAN_MyKanban, kcCardTotal)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = KanbanCount(<Kanban control> , <Count>)`
---

**`<Result>: (Entero)`**

Número de elementos. 

**`<Kanban control>: (Nombre del control)`**

Nombre del control Kanban utilizado. 

**`<Count>: (Constante de tipo Integer)`**

Tipo de información solicitada:


|   |   |
| --- | --- |
| *kcCardTotal* | Número total de tarjetas en el control. |
| *kcCardVisible* | Número de tarjetas visibles. |
| *kcListTotal* | Número total de listas Kanban en el control. |
| *kcListVisible* | Número de listas Kanban visibles. |


Observación: El número de tarjetas y listas Kanban visibles corresponde a las tarjetas y listas a las que se puede acceder al desplazarse por el control.

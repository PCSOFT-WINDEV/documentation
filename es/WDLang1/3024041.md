
## URLDecode (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Decodifica una URL codificada (es decir, con caracteres en formato %xx).

**Observación**: La sintaxis de la URL no se comprueba.






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Decodes a URL
DecodedURL is string = URLDecode("http://mysite/my%20report")
// DecodedURL contains: http://mysite/my report
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = URLDecode(<URL to decode> [, <Encoding/decoding format>])`
---

**`<Result>: (Cadena de caracteres)`**

Cadena de caracteres que contiene la URL decodificada. Los siguientes caracteres se convierten:

- '+' se convierte en espacios

- '%xx' (donde xx es un valor hexadecimal) se reemplaza por el carácter cuyo valor hexadecimal es xx.




**`<URL to decode>: (Cadena de caracteres)`**

URL a decodificar. 

**`<Encoding/decoding format>: (Constante de tipo Integer)`**

Especifica el tipo de codificación. Este parámetro debe ser el mismo que se utilizó en la codificación.


|   |   |
| --- | --- |
| *encodeURLToAnsi* | Formato de la URL. El texto codificado solo contiene caracteres permitidos en una URL. El texto decodificado es una cadena ANSI. Esta constante corresponde al la constante *encodeURLFromAnsi* utilizada en la función [URLEncode](../WDLang1/3024040.md). |
| *encodeURLToUnicode* | Formato de la URL. El texto codificado solo contiene caracteres permitidos en una URL. El texto decodificado es una cadena Unicode. Esta constante corresponde al la constante *encodeURLFromUnicode* utilizada en la función [URLEncode](../WDLang1/3024040.md). |
| *encodeURLToUTF8*<br>(Valor predeterminado) | Formato de la URL. El texto codificado solo contiene caracteres permitidos en una URL. El texto decodificado es una cadena UTF-8. Esta constante corresponde al la constante *encodeURLFromUTF8* utilizada en la función [URLEncode](../WDLang1/3024040.md).  |

## Observaciones


### Caracteres permitidos en una URL codificada
<a name="caracteres_permitidos_una_url_codificada_ELTPARAGRAPHE000112"></a>

La lista de caracteres permitidos en una URL codificada es la siguiente:


|   |   |   |   |   |
| --- | --- | --- | --- | --- |
| ; | / | ? | : | @ |
| & | = | + | $ | , |
| - | _ | . | ! | ~ |
| \* | ' | ( | \ | ) |
| % | # | dígitos | letras (mayúsculas y minúsculas) |   |


<a name="NOTE0_2"></a>




### Casos especiales
<a name="casos_especiales_ELTPARAGRAPHE000176"></a>

- La decodificación de una URL vacía (cadena vacía) devuelve una URL vacía.

- La cadena %26%238364%3B cuyo valor es #8364; se reemplaza por el símbolo €.

- Los caracteres % seguidos de caracteres no válidos se ignoran.

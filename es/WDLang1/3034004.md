
## Gestión de los errores de programación
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000118"></a>
El mecanismo de seguridad de WLanguage se activa cuando se produce un error de programación en una aplicación o en un sitio.. Aparece un mensaje de error en el ordenador del usuario final y finaliza la ejecución del programa.

**Para reducir el impacto de los errores de programación**, WINDEV, WINDEV Mobile y WEBDEV incluyen diferentes soluciones:

- La visualización de un mensaje de error personalizado. 
	![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Esta opción no está disponible.

- El mecanismo de las excepciones. Esto le permite personalizar la gestión del error.




**Atención**: Estas soluciones sólo se pueden utilizar para gestionar los errores de programación. Estas soluciones no se pueden utilizar para gestionar los errores de ejecución (como "no se puede escribir en un archivo de sólo lectura").. Para gestionar los errores de ejecución, véase [Gestión automática de errores.](../WDLang1/3034010.md)

## Mecanismo de excepción
<a name="mecanismo_excepcion_ELTTEXTE000166"></a>
El mecanismo de excepción se utiliza para Process los casos de error mediante la programación. Si se activa una excepción Process, no se muestra ningún mensaje de error y se ejecuta el código de la excepción. Este código le permite realizar todas las operaciones necesarias:

- para salir de la aplicación actual "correctamente" si el error es un error fatal,

- para devolver a control al usuario si el error puede ser corregido.




**WINDEV, WEBDEV y WINDEV Mobile proponen varios tipos de procesos de excepción:**

- [Las excepciones automatizadas](../WDLang1/3034013.md): la gestión de las excepciones se implementa en la interfaz de WINDEV, WEBDEV y WINDEV Mobile.

- [Las excepciones generales](../WDLang1/3034001.md): la excepción Process es válida para todo el objeto (proyecto, ventana, página, reporte, ...).

- [Las excepciones específicas](../WDLang1/3034003.md): la excepción Process es válida para una sección específica del código.

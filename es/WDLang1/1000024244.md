


## &lt;Tabla&gt;.MoveColumn (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Mueve una columna en una Tabla o control Tabla TreeView. 

**Observaciones**: 

- Para mover una fila en una control Tabla, utilice [&lt;Tabla&gt;.MoveLine](../WDLang1/1000024245.md).

- Esta función equivale a la siguiente operación realizada en tiempo de ejecución: haga clic en el encabezado de una columna y mueva la columna mientras mantiene pulsado el botón izquierdo del ratón.







<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Move the "COL_ProductPrice" column 
// on the far left side in the "TABLE_ProductTable" control
TABLE_ProductTable.MoveColumn(COL_ProductPrice, 1)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Table control>.MoveColumn(<Column> , <Subscript of destination column>)`
---

**`<Table control>: (Control name)`**

Nombre del control a manipular. Este control puede corresponder a: 

- un control Tabla.

- un control Tabla TreeView.




**`<Column>: (Control name)`**

Nombre de la columna a mover. El nombre de la columna es devuelto por [&lt;Tabla&gt;.EnumColumn](../WDLang1/1000024250.md).

**`<Subscript of destination column>: (Integer)`**

Nueva posición de la columna en la control. La primera columna (aunque sea Invisible) es la columna #1.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Condiciones de uso
<a name="condiciones_uso_ELTPARAGRAPHE000235"></a>

La función **&lt;Tabla&gt;.MoveColumn** puede utilizarse en:

- una Tabla o control Tabla TreeView basada en un archivo de datos.

- una Tabla o control Tabla TreeView rellenada programáticamente.

- un control de selección única o de selección múltiple.



<a name="NOTE0_2"></a>


### Los diferentes tipos de índices de columna
<a name="los_diferentes_tipos_indices_columna_ELTPARAGRAPHE000258"></a>

Existen dos tipos de índices para las columnas:

- Índice de la posición de Visible: índice de la columna en tiempo de ejecución.

- Índice de creación: índice de la columna al crear la control Tabla en el editor.




Estos índices cambian si el usuario mueve las columnas.

La función **&lt;Tabla&gt;.MoveColumn** utiliza el índice de posición visible.
<a name="NOTE0_3"></a>


### Mover y anclar columnas
<a name="mover_anclar_columnas_ELTPARAGRAPHE000277"></a>

- Si la columna se mueve entre algunas columnas ancladas, esta columna se anclará automáticamente.

- **Si la columna se mueve fuera de las columnas ancladas, esta columna ya no está anclada.**

### Desplazamiento de columnas en horizontal controles Tabla
<a name="desplazamiento_columnas_horizontal_controles_tabla_ELTPARAGRAPHE000290"></a>

- Si se llama a **&lt;Tabla&gt;.MoveColumn** en un control Tabla horizontal, las filas se desplazarán.

- Para desplazar las columnas en una control Tabla horizontal, utilice [&lt;Tabla&gt;.MoveLine](../WDLang1/1000024245.md).

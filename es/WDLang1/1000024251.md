


## &lt;Tabla&gt;.EnumBreak (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Enumera las rupturas encontradas en un control «Table».
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Lists the breaks of TABLE_MyTable
sBreakName is string
bPosition is boolean
nColNum is int

let i = 1
sDesc is string = TABLE_MyTable.EnumBreak(i)
WHILE sDesc <> "" 
	sBreakName = ExtractString(sDesc, 1, TAB)
	bPosition = ExtractString(sDesc, 2, TAB)
	nColNum = ExtractString(sDesc, 3, TAB)
   	IF {sBreakName, indControl}..Visible THEN
		Trace(sBreakName, bPosition, nColNum)
	END
	// Next break
	i++
	sDesc = TABLE_MyTable.EnumBreak(i)

END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Table control>.EnumBreak(<Break index>)`
---

**`<Result>: (Cadena de caracteres)`**



- description de la ruptura para el índice especificado. Este description tiene el siguiente formato: 
	
	```txt
	<Break> + TAB + <Break position> + TAB + <Column number>
	```
donde: 

	- &lt;Ruptura&gt;: Nombre de la pausa. 

	- &lt;Posición de descanso&gt;: 1 para un encabezado de ruptura y 0 para un pie de ruptura. 

	- &lt;Número de la columna&gt;: El número de la columna que desencadena la ruptura. 




- Cadena vacía ("") si la control Tabla no tiene rupturas o si &lt;índice de ruptura&gt; es igual al número de rupturas en la control Tabla + 1.




**`<Table control>: (Nombre del control)`**

Nombre de control Tabla a manipular.

**`<Break index>: (Integro)`**

Índice de la pausa de búsqueda, entre 1 y el número de pausas en la control Tabla + 1.
Si este parámetro es mayor que el número de rupturas encontradas en la control Tabla, la función devuelve una cadena vacía (""").




## EventChange( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica el estado de un evento.










<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Create an event
EventCreate("Let's go", eventManual, eventClose)
// Run a first thread
ThreadExecute("Thread1", threadNormal, "Thread_First")
// Locks the current thread until the event is obtained
ThreadWaitSignal()

// Run a second thread
ThreadExecute("Thread2", threadNormal, "Thread_Second")
// Locks the current thread until the event is obtained
ThreadWaitSignal()

// Modifies the event to run the threads
EventChange("Let's go", eventOpen)
```

```wl
//----------------------------------------------
PROCEDURE Thread_First()
Trace(dbgInfo(dbgProcess) + " - ThreadSendSignal(threadMain)")
ThreadSendSignal(threadMain)
EventWait("Let's go")
Trace(dbgInfo(dbgProcess) + " - Event received")
```

```wl
//--------------------------------------------
PROCEDURE Thread_Second()
Trace(dbgInfo(dbgProcess) + " - ThreadSendSignal(threadMain)")
ThreadSendSignal(threadMain)
EventWait("Let's go")
Trace(dbgInfo(dbgProcess) + " - Event received")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`EventChange(<Event> [, <New status>])`
---

**`<Event>: (Cadena de caracteres)`**

Nombre del Event a modificar. Si este evento no existe, se crea automatic ally con las opciones por defecto (modo automatic) y en el estado especificado.

**`<New status>: (Constante opcional)`**

Estado de la Event:


|   |   |
| --- | --- |
| *eventClose* | Cierra la Event.<br>Esta constante equivale a [EventClose](../WDLang1/1000021272.md). |
| *eventOpen*<br>(Valor predeterminado) | Abre la Event. Si los hilos están esperando a la Event, se desbloquean automáticamente. <br>Esta constante equivale a [EventOpen](../WDLang1/1000021271.md). |
| *eventOpenAndClose* | Abre la Event, desbloquea todos los hilos pendientes y cierra la Event.<br>Esta constante no es recomendable. Se mantiene para la compatibilidad con versiones anteriores. |

## Observaciones


### Resumen de tabla sobre el modo de funcionamiento de Event
<a name="resumen_tabla_sobre_modo_funcionamiento_event_ELTPARAGRAPHE000127"></a>

| Valor de EventChange | Event( automática de Default) | Manual Event |
| --- | --- | --- |
| *eventOpen* | Un solo hilo es procesado por la Event. <br>La Event se cierra automáticamente una vez que se procesa la el hilo. <br>**Hilos pendientes**: Si no hay ningún thread pendiente, el Event permanece abierto hasta el próximo thread, entonces se cierra. <br>**Nuevos hilos**: Los nuevos temas se añaden a la cola de espera. | El evento se abre. <br>**Hilos pendientes**: Se procesan todos los hilos pendientes. <br>**Nuevos hilos**: Si llegan otros hilos mientras la Event está todavía abierta, se procesan. |
| *eventOpenAndClose*<br>**Esta constante no es recomendable. Se mantiene para la compatibilidad con versiones anteriores.** | Un solo hilo es procesado por la Event. <br>La Event se cierra automáticamente una vez que se procesa la el hilo. <br>**Hilos pendientes**: Si no hay ningún hilo pendiente, la Event permanece cerrada. <br>**Nuevos hilos**: Los nuevos temas se añaden a la cola de espera. | Todos los hilos son procesados por la Event. <br>La Event se cierra automáticamente una vez que todos los hilos se procesan. <br>**Nuevos hilos**: Los nuevos temas se añaden a la cola de espera. |
| *eventClose* | El Event permanece cerrado. <br>**Nuevos hilos**: Los nuevos temas se añaden a la cola de espera. | El Event está cerrado. <br>**Nuevos hilos**: Los nuevos temas se añaden a la cola de espera. |


## DeleteVoiceCommand (Función)






	<B><B>Advertencia</B></B>

	A partir de la versión <B><B>17</B></B>, esta función se conserva por motivos de compatibilidad con versiones anteriores. Esta función se sustituye por [SpeechRecognitionDeleteCommand](../WDLang1/1000020017.md). 



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Borra un comando de voz que fue añadido por [AgregarComando de Voz](../WDLang1/1000017106.md).

**Observación**: se debe instalar un programa de reconocimiento de voz en el ordenador en tiempo de ejecución. Un programa de reconocimiento de voz se suministra con Windows Vista. Para los demás sistemas, se debe instalar "Dragon Naturally Speaking" (las pruebas se ejecutan con la versión 9 "Preferred" o "Standard").
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Start the "Proc_Anim_Window" procedure if the voice command is recognized
Voice1ID is int
Voice1ID = AddVoiceCommand("Proc_Anim_Window", "Animate the window")
...
// Delete the voice command
DeleteVoiceCommand(Voice1ID)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`DeleteVoiceCommand(<Identifier of voice command>)`
---

**`<Identifier of voice command>: (Integro)`**

Identificador del comando de voz a eliminar. Este identificador fue devuelto por [AgregarComando de Voz](../WDLang1/1000017106.md).

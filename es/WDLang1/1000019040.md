
## Constantes para la gestión del corrector ortográfico
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


| Constante | description | Utilizado por las funciones |
| --- | --- | --- |
| **spellScopePersistent** | La palabra se añade al diccionario de OpenOffice.<br>Esta constante tiene el mismo efecto que la opción "Añadir al diccionario" del menú contextual de AAF con el corrector ortográfico. | [HechizoAñadidoDiccionario](../WDLang1/1000018921.md) |
| **spellScopeProcess** | La palabra no se añade al diccionario de OpenOffice.<br>Por lo tanto, seguirá mostrándose como "False" si se reinicia la aplicación. | [HechizoAñadidoDiccionario](../WDLang1/1000018921.md) |

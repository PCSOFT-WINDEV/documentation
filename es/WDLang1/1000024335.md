


## &lt;Imagen&gt;.PixelColor (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Identifica el color de un pixel en un control Imagen.



![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los dibujos](../WDLang1/3029035.md). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Identify the color of the pixel (15, 25)
// in the Image control named "IMG_Drawing"
ResColor = IMG_Drawing.PixelColor(15, 25)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Image control>.PixelColor(<X-coordinate> , <Y-coordinate>)`
---

**`<Result>: (Entero)`**

Color del pixel especificado en formato [RGB](../WDLang1/3029012.md). 
Atención: el valor devuelto puede variar según la configuración de la pantalla (resolución, etc.).

**`<Image control>: (Nombre del control)`**

Nombre del control Imagen a manipular.

**`<X-coordinate>: (Entero)`**

Posición horizontal del pixel a manipular.

**`<Y-coordinate>: (Entero)`**

Posición vertical del pixel a manipular.

<a name="NOTE0_1"></a>

## Observaciones


### Coordenadas
<a name="coordenadas_ELTPARAGRAPHE000382"></a>
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=dpixel.gif)
Las coordenadas se indican en pixeles con respecto a la esquina superior izquierda de la imagen (coordenadas (0,0)).

Para obtener la posición de un clic en un control Imagen, utilice las funciones [MouseXPos](../WDLang1/3071007.md) y [MouseYPos](../WDLang1/3071008.md).
<a name="NOTE0_2"></a>


### ¿Cómo obtener los componentes del color?
<a name="¿como_obtener_los_componentes_del_color_ELTPARAGRAPHE000414"></a>

Obtener los componentes del color devuelto por la función **&lt;Imagen&gt;.PixelColor**:

- Para obtener el componente rojo, use [RGBRed](../WDLang1/3029050.md)  (o realice una división módulo 256 de &lt;Resultado&gt;), 

- Para obtener el componente verde, use [RGBGreen](../WDLang1/3029051.md) (o realice una división módulo 256 de &lt;Resultado&gt;/256),

- Para obtener el componente azul, use [RGBBlue](../WDLang1/3029049.md) (o realice una división módulo 256 de &lt;Resultado&gt;/256/256).

### Función &lt;Imagen&gt;.PixelColor, barras de desplazamiento y zoom automáticos
<a name="funcion_docparampagetitleshort_barras_desplazamiento_zoom_automaticos_ELTPARAGRAPHE000531"></a>

La opción "Barras de desplazamiento y zoom automáticos", disponible en los controles Imagen, no es compatible con la función **&lt;Imagen&gt;.PixelColor**. Si se utiliza esta opción en el control Imagen, la función **&lt;Imagen&gt;.PixelColor** inmoviliza la imagen. El zoom y las barras de desplazamiento también se inmovilizan.


## Log( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Calcula el logaritmo decimal de un valor numérico.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Calculate the logarithm of 10
ResLogarithm = Log(10)
// Returns 1
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = Log(<Numeric value>)`
---

**`<Result>: (Real)`**

Logaritmo decimal solicitado.

**`<Numeric value>: (Entero o real)`**

Valor numérico a utilizar. Se produce un error de WLanguage si este parámetro es un número negativo o un número nulo..




## ListSelect (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el índice del elemento seleccionado en un Cuadro de Lista, ListView o control Combo Box.

**Recordatorio**: El elemento seleccionado corresponde al elemento actual en el que se visualiza el banda de selección.

**Observación**: La función **ListSelect** puede utilizarse en:

- Controles de la caja de lista basados en un archivo de datos o rellenados mediante programación,

- Controles de cuadro de lista de selección única o de selección múltiple.













<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo
<a class="notetitle" target="_blank" href="$DOC$=1000003049011&name=listselect_funcion&product=WD">Ver ejemplos adicionales</a>

```wl
// Index of the selected element in "LIST_CustomerList"
ResElementIndex = ListSelect(LIST_CustomerList)
```


<a name="Example2"></a>

```wl
// Rows selected in a multi-selection List Box control
i is int = 1
SelectedRowIndex is int
SelectedRowIndex = ListSelect(LIST_List1, 1)
WHILE SelectedRowIndex <> -1
	Trace("Row #" + SelectedRowIndex + " is selected")
	i++
	SelectedRowIndex = ListSelect(LIST_List1, i)
END

// Equivalent to:
// FOR EACH SELECTED ROW OF LIST_List1
//	 Trace("Row #" + LIST_List1 + " is selected")
// END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = ListSelect(<List Box control> [, <Rank>])`
---

**`<Result>: (Integro)`**



- Índice del elemento seleccionado en el cuadro de lista especificado (o Combo Box) control,

- -1 si no hay ningún elemento seleccionado. 


En el caso de una control List Box multiselección, se debe especificar el rango del elemento seleccionado &lt;Rango&gt;).

**`<List Box control>: (Nombre del control)`**

Nombre de la casilla de la lista (o Combo Box) control a utilizar. Si este parámetro corresponde a una cadena vacía (""), se devuelve el índice del elemento seleccionado en el control List Box (o Combo Box) al que pertenece el Event actual.


**`<Rank>: (Entero opcional)`**

Rango de selección para una multiselección control List Box. Si no se especifica este parámetro, se devuelve el índice del primer elemento seleccionado.

Por ejemplo: Para recuperar el primer elemento seleccionado, el rango será 1, para recuperar el segundo elemento seleccionado, el rango será 2, etc..

## Observaciones


### Equivalencia
<a name="equivalencia_ELTPARAGRAPHE000124"></a>Las siguientes sintaxis son equivalentes:


```txt
Index is int
Index = <List Box control>
```
y

```txt
Index = ListSelect(<List Box control>)
```
<a name="NOTE0_2"></a>


### Valor del elemento seleccionado
<a name="valor_del_elemento_seleccionado_ELTPARAGRAPHE000139"></a>Para recuperar el valor del elemento seleccionado, la sintaxis es

```txt
Index is int
Index = ListSelect(<List Box control>)
<Element value> = <List Box control>[Index]
```


y


```txt
<Element value> = <List Box control>[<List Box control>]
```

<a name="NOTE0_3"></a>


### Varios
<a name="varios_ELTPARAGRAPHE000158"></a>

- Para get el índice del elemento seleccionado en un control Tabla, utilice [TableSelect](../WDLang1/3074020.md).

- **ListSelect** puede utilizarse en una columna "Combo Box" tabla.


## Funciones de gestión de entradas no válidas
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para gestionar la entrada de datos no válidos: 



|   |   |
| --- | --- |
| [InvalidInputDetect](../WDLang1/1000026180.md) | Detecta y señala un error en los controles requeridos que quedan vacíos y/o en los controles con datos no válidos. |
| [InvalidInputListControl](../WDLang1/1000026244.md) | Construye una lista de controles que requieren entrada de datos/contienen datos no válidos. |
| [InvalidInputShowIcon](../WDLang1/1000024917.md) | Muestra un ícono de error al lado de un control Campo de entrada para indicar que la entrada no es válida. |
| [InvalidInputShowMessage](../WDLang1/1000026247.md) | Muestra un error de entrada no válido para la control especificada. |
| [InvalidInputShowTooltip](../WDLang1/1000024896.md) | Muestra un aviso de error en un tooltip debajo de un control Campo de entrada para indicar por qué la entrada no es válida. |




## &lt;Caja de lista&gt;.display (función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Actualiza un cuadro de lista, ListView o control Combo Box basado en un archivo de datos. Los controles de List Box, ListView o Combo Box rellenados mediante programación se actualizan automáticamente

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Esta operación de actualización se puede realizar desde una posición determinada.**Observación**: La función **&lt;Caja de lista&gt;.display** puede utilizarse en los List Box de selección única y múltiple.










<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Refresh the "LOOP_ProductLooper" control
LIST_CustomerList.Display()
```


<a name="Example2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 
```wl
// Refresh "LOOP_ProductLooper" from the beginning of the List Box control
LIST_CustomerList.Display(taStart)
```

<a name="SYNTAX1"></a>

## Sintaxis

`<List Box control>.Display([<Position>])`
---

**`<List Box control>: (Nombre del control)`**

Nombre del buzón de la lista, ListView o control Combo Box en función de un archivo de datos que se va a actualizar.

**`<Position>: (Entero o constante (opcional))`**

Indica la posición de inicio para la actualización. Este parámetro puede corresponder a:

- un entero. La actualización se realiza a partir de esta posición. El elemento seleccionado no se modifica. Si este parámetro no es válido, la función **&lt;Caja de lista&gt;.display** no tiene ningún efecto.

- a constant: 
	


|   |   |
| --- | --- |
| *taCurrentFirst* | <br><br>	- Actualiza el control List Box del actual Record en el archivo de datos.<br><br>	- Se selecciona la primera fila de la control List Box. Este fila corresponde al actual Record del control.<br><br><br> |
| *taCurrentSelection* | <br><br>	- Actualiza la control List Box desde la posición de la banda de selección.<br><br>	- La posición de la banda de selección no se modifica.<br><br><br> |
| *taInit* | <br><br>	- Ejecuta los eventos de inicialización de control List Box (inicialización y fin de los eventos de inicialización).<br><br>	- Llena la control List Box.<br><br>	- Se selecciona la primera fila de la control List Box. Este fila corresponde al primer Record mostrado en el control.<br><br><br> |
| *taNoUpdEditCb* | Si el control es un control Combo Box editable, el control Campo de entrada associated con el control Combo Box no se modifica según el nuevo contenido de la lista.<br>Si no se usa esta constante, el control Campo de entrada asociado con el Combo Box es modificado por Default. <br>Esta constante se puede combinar con las otras constantes.<br><br>![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Esta constante no está disponible. |
| *taReExecuteQuery* | <br><br>	- Vuelve a ejecutar la consulta (sólo para los controles List Box en las consultas incrustadas).<br><br>	- Actualiza el control List Box y fija el banda de selección al principio de la lista.<br><br>	- Se selecciona la primera fila de la control List Box. Este fila corresponde al primer Record mostrado en el control.<br><br><br> |
| *taStart* | <br><br>	- Actualiza el control List Box y fija el banda de selección al principio de la lista.<br><br>	- Se selecciona la primera fila de la control List Box. Este fila corresponde al primer Record mostrado en el control.<br><br><br> |

## Observaciones


### Varios
<a name="varios_ELTPARAGRAPHE000330"></a>

- Para actualización el controles Tabla, utilice [&lt;Table&gt;.Display](../WDLang1/1000023505.md).

- **&lt;Caja de lista&gt;.display** puede utilizarse en una columna "Combo Box" tabla.



<a name="NOTE0_2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Conectores Nativos](https://doc.pcsoft.fr/ext/images/us/AN.png) 

### Conector Nativo SQLServer
<a name="conector_nativo_sqlserver_ELTPARAGRAPHE000350"></a>

Si la función **&lt;Caja de lista&gt;.display** manipula un control List Box relacionaldo con un archivo de datos al que se accede mendiante el Conector Nativo SQLServer, se recomineda utilizar las constantes *taCurrentFirst* y *taStart*.

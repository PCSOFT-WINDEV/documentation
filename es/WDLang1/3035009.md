
## ExeListDLL( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la lista de bibliotecas (archivos".DLL") utilizadas por una de las aplicaciones que se están ejecutando actualmente.. Esta aplicación puede corresponder a una aplicación WINDEV, una aplicación WINDEV Mobile u otra aplicación.

**Observación**: Sólo se listan las bibliotecas cargadas en memoria.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// -- Initialization code of "LIST_App" control
// Add the applications currently run into the "LIST_App" control
ListAdd(LIST_App, ExeListProcess(exePID, exeShortName))

// -- Code for selecting a row in "LIST_App"
// Extract the identifier of the application selected in the list
AppInfo is string = LIST_App[LIST_App]
AppID is string = ExtractString(AppInfo, 1)
// Display the DLLs found in memory in the "LIST_DLL" control
ListAdd(LIST_DLL, ExeListDLL(Val(AppID)))
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = ExeListDLL([<Application identifier>])`
---

**`<Result>: (Cadena de caracteres)`**

Lista de bibliotecas (archivos".DLL") utilizadas por la aplicación especificada en el siguiente formato:

```txt
<1st Library> + CR + <2nd Library> + CR + ... + <Nth Library>
```
Donde &lt;Biblioteca X&gt; corresponde al nombre y ruta completa de la biblioteca X.

**`<Application identifier>: (Constante opcional de tipo Integer)`**

Identificador de la aplicación a utilizar. Este identificador es devuelto por [Proceso ExeList](../WDLang1/3035008.md).
  
Si no se especifica este parámetro, &lt;Resultado&gt; corresponderá a la lista de bibliotecas para el actual Process. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
La función **ExeListDLL** no devuelve ningún código de error. Para determinar si esta función ha generado un error, utilice la función [ErrorOccurred](../WDLang1/3087001.md) Variable. Si se produce un error, puede get más detalles sobre el error con [ErrorInfo](../WDLang1/3013008.md).

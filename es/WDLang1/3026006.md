
## Funciones para el manejo de los controles Combo Box (sintaxis de prefijo)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones son específicas de los controles Combo Box (sintaxis de prefijo):



|   |   |
| --- | --- |
| [&lt;Cerrar](../WDLang1/3026004.md) | Fuerza el cierre de la lista desplegable en una control Combo Box (simple o multicolumna Combo Box). |
| [&lt;Combo Box&gt;.Abrir](../WDLang1/3026005.md) | Expands the list of a Combo Box control (single-column or multi-column Combo Box). |

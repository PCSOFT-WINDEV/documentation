
## AJAXCallInProgress (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Permite determinar si la solicitud que se está procesando es una solicitud AJAX.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF NOT AJAXCallInProgress() THEN
	// Fill the chart
	...
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = AJAXCallInProgress()`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si la solicitud que se está procesando es una solicitud AJAX,

- <u><u><u><u>False</u></u></u></u> en caso contrario. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Diferencia con la función InAJAXMode
<a name="diferencia_con_funcion_inajaxmode_ELTPARAGRAPHE000039"></a>

La función **AJAXCallInProgress** es diferente de la función [InAJAXMode](../WDLang1/3013074.md). La función [InAJAXMode](../WDLang1/3013074.md) devuelve <u><u><u><u>True</u></u></u></u> durante la acción AJAX (proceso o procedimiento) y <u><u><u><u>False</u></u></u></u> en los demás casos.

En páginas AWP, el código de inicialización del proyecto, clases, conjuntos de procedimientos, páginas y controles también se ejecuta en cada llamada. Durante estas inicializaciones: 

- La función [InAJAXMode](../WDLang1/3013074.md) devuelve <u><u><u><u>False</u></u></u></u>. 

- La función **AJAXCallInProgress** devuelve <u><u><u><u>True</u></u></u></u>. 


La función **AJAXCallInProgress** también devuelve <u><u><u><u>True</u></u></u></u> en el código de cierre del proyecto, clases, conjuntos de procedimientos, páginas, etc. 

La función **AJAXCallInProgress** permite hacer optimizaciones ignorando algunas inicializaciones innecesarias durante la llamada.
<a name="NOTE0_2"></a>


### Disponibilidad de la función
<a name="disponibilidad_funcion_ELTPARAGRAPHE000219"></a>

La función **AJAXCallInProgress** se puede utilizar en todos los IDE para escribir código multiplataforma. Las funciones o procedimientos que utilizan la función **AJAXCallInProgress** pueden compartirse entre varios productos sin generar errores de compilación.

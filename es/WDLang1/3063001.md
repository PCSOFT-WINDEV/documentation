


## Funciones de gestión del portapapeles
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>




Las siguientes funciones se utilizan para gestionar el portapapeles:



|   |   |
| --- | --- |
| [ClipboardFormat](../WDLang1/3063002.md) | Identifica el formato de la información que se encuentra en el portapapeles. |
| [ClipboardHeight](../WDLang1/3063006.md) | Calcula la altura de una Image (en formato de mapa de bits) que se encuentra en el portapapeles. |
| [ClipboardRTF](../WDLang1/1000017200.md) | Recupera una cadena RTF del portapapeles (el contenido del portapapeles está en formato RTF). |
| [ClipboardUnicode](../WDLang1/1000018858.md) | Devuelve el texto encontrado en el portapapeles en formato Unicode. |
| [ClipboardWidth](../WDLang1/3063007.md) | Calcula el ancho de una Image (en formato de mapa de bits) que se encuentra en el portapapeles. |
| [Portapapeles](../WDLang1/3063004.md) | Recupera el texto o Image que se encuentra en el portapapeles del sistema. |
| [ToClipboard](../WDLang1/3063005.md) | Escribe el texto o la información de Image en el portapapeles del sistema. |
| [ToClipboardRTF](../WDLang1/1000017199.md) | Escribe una cadena de caracteres en formato RTF en el portapapeles. |



También se pueden utilizar las siguientes funciones para manipular el portapapeles: 


|   |   |
| --- | --- |
| [TablaToClipboard](../WDLang1/1000017062.md) | Copia el contenido de una Tabla o control Tabla TreeView (basada en un archivo de datos o rellenada mediante programación) en el portapapeles. El contenido de control se copia en formato de texto: el contenido se convierte en una cadena, y la cadena se copia en el portapapeles. |
| [TwainToClipboard](../WDLang6/3078011.md) | Permite:<br><br>- copiar la Document del dispositivo Twain al portapapeles.<br><br>- ver la Document desde el dispositivo Twain en la interfaz de usuario del dispositivo.<br><br><br> |

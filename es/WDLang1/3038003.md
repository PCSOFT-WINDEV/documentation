


## MenuState( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Identifica el estado de una opción de menú (menú contextual o menú desplegable): activo, inactivo o Invisible.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// State of the "OPT_IconBar" option 
ResState = MenuState(OPT_IconBar)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = MenuState([<Window>, ] <Menu option>)`
---

**`<Result>: (Constante)`**

Estado actual de la opción de menú:


|   |   |
| --- | --- |
| *Active* | La opción del menú es Active. |
| *Grayed* | La opción del menú es Grayed. |
| *Invisible* | La opción del menú es Invisible. |



**`<Window>: (Nombre de la ventana opcional)`**

Nombre de la ventana que contiene la opción de menú. 
Si no se especifica este parámetro, **MenuState** manipular la ventana actual.

**`<Menu option>: (Nombre de la opción)`**

Nombre de la opción de menú a utilizar. Este nombre se define en la ventana de description de la opción de menú. Este nombre debe ser especificado sin comillas.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Equivalencia
<a name="equivalencia_ELTPARAGRAPHE000059"></a>La sintaxis: 


```txt
MenuState(<Menu option>)
```
equivale a la [Estado](../Proprietes/2510039.md) property: 


```txt
<Menu option>.State
```

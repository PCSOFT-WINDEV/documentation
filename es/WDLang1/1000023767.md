


## &lt;TreeView&gt;.Select (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la ruta completa del elemento seleccionado en un control TreeView.

Ejemplo de control TreeView:<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Arbre_Vocabulaire.gif)
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) **Observación**: La función **&lt;TreeView&gt;.Select** puede utilizarse en:

- un control TreeView rellenado mediante programación, basado en un archivo de datos, o basado en una variable. 

- un control TreeView de selección única o múltiple. En el caso de los controles TreeView de selección múltiple, se debe utilizar la función **&lt;TreeView&gt;.Select** en un bucle para obtener todas las filas seleccionadas. 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Full path of selected element in the "TREE_TVRecipe" TreeView control
ResPath = TREE_TVRecipe.Select()
```
<a name="Example2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 
```wl
// Selected rows in a multi-selection TreeView Table control populated programmatically
FOR i = 1 _TO_ TREE_TREE1.SelectOccurrence()
	Trace("Selected element: " + TREE_TREE1.Select(False, i))
END

// Equivalent to:
// i is int
// FOR EACH SELECTED ROW i OF TREE_TREE1
// 	Trace("Selected element: " + TREE_TREE1.Select(False, i-1))
// END
```

<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <TreeView control>.Select([<Management of duplicates> [, <Rank>]])`
---

**`<Result>: (Cadena de caracteres)`**

Ruta completa del elemento seleccionado, en el siguiente formato:

```txt
"<Root name>" + TAB + ["<Name of 1st node>" + TAB + ...
["<Name of 2nd node>" + TAB + [...]]]"<Leaf name>"
```


**`<TreeView control>: (Nombre del control)`**

Nombre del control TreeView a utilizar.

**`<Management of duplicates>: (Booleano opcional)`**

Modo de gestión de duplicados de los elementos:

- <u><u><u><u>False</u></u></u></u> (valor predeterminado): no se realiza ninguna gestión.

- <u><u><u><u>True</u></u></u></u>: los nombres de los elementos de la ruta se construyen con un identificador. Para obtener más información, consulte [Gestión de duplicados en controles TreeView](../WDChamp/1013325.md).




![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Este parámetro no se tiene en cuenta.

**`<Rank>: (Entero opcional)`**

Rango de la selección para un control TreeView de selección múltiple. Si no se especifica este parámetro, se devuelve el índice de la primera fila seleccionada.
Por ejemplo, para obtener el primer elemento seleccionado, &lt;Rank&gt; = 1. 
Para obtener el segundo elemento seleccionado, &lt;Rank&gt; = 2, etc..
  
![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Este parámetro no está soportado.

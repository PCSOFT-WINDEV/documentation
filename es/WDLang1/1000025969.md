


## HTMLExecuteControlProcess (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Ejecuta un Event associated con un control en una página HTML mostrada por un control. Esta función permite automatizar procesos en una Web Page, por ejemplo, un clic en un botón de un formulario HTML. 

Observación: Esta Event se ejecuta en JavaScript. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
HTMLControlValue(HTM_Display, "LOGIN", sLogin)
HTMLControlValue(HTM_Display, "PASSWD", sPassword)
HTMLExecuteControlProcess(HTM_Display, "BTN_OK", trtClick)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`HTMLExecuteControlProcess(<HTML Display control> , <Control in HTML page> , <Event>)`
---

**`<HTML Display control>: (Nombre del control)`**

Nombre de la pantalla HTML utilizada por control.  

**`<Control in HTML page>: (Cadena de caracteres)`**

Nombre de la control que se utilizará en el HTML Page. Si no se encuentra la control, se produce un error no fatal. 
Observación: Sólo se tienen en cuenta los atributos "name" e "id. 

**`<Event>: (Cadena de caracteres o constante entera)`**

Nombre de la Event( o Process) a ejecutar. Este nombre puede corresponder a: 

- una cadena de caracteres que contiene el nombre del código a ser ejecutado en JavaScript (por ejemplo: onclick', 'onchange', 'onfocus', etc.). 

- una de las siguientes constantes: 
	


|   |   |
| --- | --- |
|  *trtChange* | Cambio de código. |
| *trtClick* | Código de clic. |

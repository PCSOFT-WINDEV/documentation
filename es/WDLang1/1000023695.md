


## <PDF Reader>Cerrar (función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Cierra y libera el archivo PDF que se muestra en un lector de PDF control.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
PDF_MyPDFReader.Close()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<PDF Reader control>.Close()`
---

**`<PDF Reader control>: (Nombre del control)`**

Nombre del lector de PDF control que contiene el archivo PDF a cerrar.




## MenuSelectMinus( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Desactiva (hace gris) una opción de menú (menú emergente o menú desplegable). La opción de menú especificada ya no se puede seleccionar. La opción puede volver a activarse mediante la programación con [MenuSelectPlus](../WDLang1/3038005.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Grays out the "OPT_IconBar" option in 
// the drop-down menu ("Menu") of the current window
MenuSelectMinus(OPT_IconBar)

// Grays out the "OPT_IconBar" option in the "Menu" of the "WIN_Win1" window
// The WIN_Win1 window is not the current window
MenuSelectMinus(WIN_Win1, OPT_IconBar)

// Grays out the "OPT_IconBar" option in "PopupMenu1" 
MenuSelectMinus(WIN_Win1, PopupMenu1.OPT_IconBar)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`MenuSelectMinus([<Window>, ] <Menu option>)`
---

**`<Window>: (Nombre de la ventana opcional)`**

Nombre de la ventana que contiene la opción de menú para desactivar. 
Si no se especifica este parámetro, **MenuSelectMinus** manipular la ventana actual.

**`<Menu option>: (Nombre de la opción)`**

Nombre de la opción de menú a desactivar. Este nombre se define en la ventana de description de la opción de menú. 
Este nombre debe ser especificado sin comillas. 
Si este parámetro corresponde al nombre de un menú, también se deshabilitarán todas las subopciones de este menú (recursivo).. 
Para desdibujar una opción de un menú emergente, utilice la siguiente notación: &lt;Nombre del menú emergente&gt;.&lt;Nombre de la opción&gt;.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Deshabilitar un conjunto de opciones
<a name="deshabilitar_conjunto_opciones_ELTPARAGRAPHE000053"></a>

Para deshabilitar un conjunto de opciones, es necesario:

1. Asignar las opciones a un grupo de controles

2. Desengrasar las opciones (y los otros controles del grupo) con el [Estado](../Proprietes/2510039.md) Property. 



<a name="NOTE0_3"></a>


### Varios
<a name="varios_ELTPARAGRAPHE000066"></a>

Cuando una opción de menú no es visible (propiedad [Visible](../Proprietes/2510138.md)  = False), la opción se vuelve visible pero se muestra en gris después de llamar a la función **MenuSelectMinus**.

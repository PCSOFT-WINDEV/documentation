


## DiagramStartModification (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Indica el inicio de una serie de acciones realizadas en un documento de tipo Diagrama en un solo evento (acciones en el menú "Rehacer/Deshacer").
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MySelection is diagSelection <- DIAGEDT_MyDiagram.Selection
DiagramStartModification(DIAGEDT_MyDiagram)
MySelection.Background.Color = White
MySelection.Border.Color = Black
DiagramEndModification(DIAGEDT_MyDiagram)
```



<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`DiagramStartModification(<Diagram>)`
---

**`<Diagram>: (Nombre del control o variable de tipo Diagram)`**

Diagrama a utilizar. Este documento corresponde a:

- el nombre de un control Editor de diagramas.

- una variable de tipo [Diagram](../WDLang1/1410088055.md).






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Todos los cambios realizados entre la llamada a **DiagramStartModification** y [DiagramEndModification](../WDLang1/1410088736.md) se agrupan en un acción única que se puede deshacer fácilmente.




## ArrayDeleteDuplicate (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Borra los duplicados de una array.

Se pueden manipular los siguientes arrays:

- array unidimensional de elementos simples.

- array bidimensional de elementos simples. Los duplicados se eliminan en función del valor de una o varias columnas.

- array unidimensional de clases o estructuras. Los duplicados se eliminan según el valor de uno o más miembros.

- array unidimensional proporcionando un comparison Procedure.




Observación: [ArrayDistinct](../WDLang1/1000025337.md) se utiliza para obtener una nueva array sin duplicados.






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
arrMyArray is array of int = [ 1, 2, 3, 1, 4, 5, 2, 6 ]
// arrMyArray contains [ 1, 2, 3, 1, 4, 5, 2, 6 ]

ArrayDeleteDuplicate(arrMyArray)
// arrMyArray contains [ 1, 2, 3, 4, 5, 6 ]
```
<a name="Example2"></a>

```wl
// Use with an array of structures
StACategory is Structure
	CODE_CATEGORY is string(5)
	CATEGORY is string(300)
END
arrTheCategories is array of StACategory

// Code for filling the array
...

// Delete the duplicates
ArrayDeleteDuplicate(arrTheCategories, asMember, "CODE_CATEGORY;CATEGORY")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Borrar los duplicados de una array unidimensional de elementos simples

`ArrayDeleteDuplicate(<WLanguage array> [, <Deletion options>])`
---

**`<WLanguage array>: (Array)`**

Nombre de la variable de tipo Array a utilizar. Este array debe ser unidimensional. 

Este array también puede corresponder a un array de elementos simples de una variable avanzada (por ejemplo, array "Grupo" en las variables de tipo [gglContact](../WDLang5/1000017434.md)).

**`<Deletion options>: (Constante opcional de tipo Integer)`**

Opciones de borrado correspondientes a una o más constantes:


|   |   |
| --- | --- |
| *tccIgnorar la puntuación y el espacio* | Eliminar los duplicados sin tener en cuenta los espacios y la puntuación.<br>Esta opción es useful para arrays de cadenas.<br> |
| *tccIgnoreAccent* | Eliminar los duplicados ignorando los acentos.<br>Esta opción es useful para arrays de cadenas.<br> |
| *tccIgnoreCase* | Eliminar duplicados ignorando el caso.<br>Esta opción es useful para el arrays de las cadenas de caracteres. |
| *tccIgnoreInsideSpace* | Eliminar duplicados ignorando los espacios dentro de las cadenas.<br>Esta opción es useful para el arrays de las cadenas de caracteres. |
| *tccIgnoreSpace* | Eliminar duplicados ignorando los espacios al principio y al final de las cadenas.<br>Esta opción es useful para el arrays de las cadenas de caracteres. |




<a name="SYNTAX2"></a>

### Eliminación de duplicados de una array bidimensional de elementos simples según una o varias columnas

`ArrayDeleteDuplicate(<WLanguage array> , <asColumn> , <Column>)`
---

**`<WLanguage array>: (Array)`**

Nombre de la variable de tipo Array a utilizar. Este array debe ser bidimensional.

Este array también puede corresponder a un array bidimensional de tipo avanzado (por ejemplo, borrando los duplicados de un [xlsDocument](../WDLang5/1000018782.md) Variable según el valor de la columna).

**`<asColumn>: (Constante)`**

*asColumn*: Constante utilizado para manipular un array bidimensional de elementos simples según varias columnas.

La constante *asColumn* puede combinarse con una o más opciones de eliminación:


|   |   |
| --- | --- |
| *tccIgnorar la puntuación y el espacio* | Eliminar los duplicados sin tener en cuenta los espacios y la puntuación.<br>Esta opción se aplica a todas las columnas según las cuales se realiza el borrado de duplicados..<br>Esta opción es useful para el arrays de las cadenas de caracteres.<br><br> |
| *tccIgnoreAccent* | Eliminar los duplicados ignorando los acentos.<br>Esta opción se aplica a todas las columnas según las cuales se realiza el borrado de duplicados..<br>Esta opción es useful para arrays de cadenas.<br> |
| *tccIgnoreCase* | Eliminar duplicados ignorando el caso.<br>Esta opción se aplica a todas las columnas según las cuales se realiza el borrado de duplicados..<br>Esta opción es useful para el arrays de las cadenas de caracteres. |
| *tccIgnoreInsideSpace* | Eliminar duplicados ignorando los espacios dentro de las cadenas.<br>Esta opción se aplica a todas las columnas según las cuales se realiza el borrado de duplicados..<br>Esta opción es useful para el arrays de las cadenas de caracteres. |
| *tccIgnoreSpace* | Eliminar duplicados ignorando los espacios al principio y al final de las cadenas.<br>Esta opción se aplica a todas las columnas según las cuales se realiza el borrado de duplicados..<br>Esta opción es useful para el arrays de las cadenas de caracteres. |



**`<Column>: (Entero o cadena de caracteres)`**



- Si debe tenerse en cuenta una sola columna, índice de la columna que debe tenerse en cuenta para el borrado de duplicados.. 

- Si deben tenerse en cuenta varias columnas, lista de columnas que deben tenerse en cuenta para el borrado de duplicados.. Las diferentes columnas deben estar separadas por punto y coma. 





<a name="SYNTAX3"></a>

### Borrar los duplicados de una array unidimensional de clases, estructuras o tipos avanzados

`ArrayDeleteDuplicate(<WLanguage array> , <asMember> , <Members>)`
---

**`<WLanguage array>: (Array)`**

Nombre de la variable de tipo Array a utilizar. Este array debe ser unidimensional. Esta array puede ser una array de clases, estructuras o tipos avanzados.

**`<asMember>: (Constante)`**

*asMember*: Constante utilizado para manipular un array unidimensional de clases, estructuras o tipos avanzados.

La constante *asMember* puede combinarse con una o más opciones de eliminación:


|   |   |
| --- | --- |
| *tccIgnorar la puntuación y el espacio* | Eliminar los duplicados sin tener en cuenta los espacios y la puntuación.<br>Esta opción se aplica a todos los miembros según los cuales se realiza el borrado de duplicados..<br>Esta opción es useful para los miembros de la "cadena de caracteres"..<br> |
| *tccIgnoreAccent* | Eliminar los duplicados ignorando los acentos.<br>Esta opción se aplica a todos los miembros según los cuales se realiza el borrado de duplicados..<br>Esta opción es useful para los miembros de la "cadena de caracteres"..<br> |
| *tccIgnoreCase* | Eliminar duplicados ignorando el caso.<br>Esta opción se aplica a todos los miembros según los cuales se realiza el borrado de duplicados..<br>Esta opción es useful para los miembros "cadena de caracteres". |
| *tccIgnoreInsideSpace* | Eliminar duplicados ignorando los espacios dentro de las cadenas.<br>Esta opción se aplica a todos los miembros según los cuales se realiza el borrado de duplicados..<br>Esta opción es useful para los miembros "cadena de caracteres". |
| *tccIgnoreSpace* | Eliminar duplicados ignorando los espacios al principio y al final de las cadenas.<br>Esta opción se aplica a todos los miembros según los cuales se realiza el borrado de duplicados..<br>Esta opción es useful para los miembros "cadena de caracteres". |



**`<Members>: (Cadena de caracteres)`**

Nombre de los miembros utilizados. Si el borrado de duplicados se realiza según el valor de varios miembros, sus nombres deben estar separados por punto y coma.. 


<a name="SYNTAX4"></a>

### Borrar los duplicados de un array unidimensional con una especie de Procedure

`ArrayDeleteDuplicate(<WLanguage array> , <asFunction> , <WLanguage procedure>)`
---

**`<WLanguage array>: (Array)`**

Nombre de la variable de tipo Array a utilizar. Este array debe ser unidimensional. 

Esta array puede ser una array de tipos avanzados.

**`<asFunction>: (Constante)`**

*asFunction*: Constant utilizó para manipular un array unidimensional con una especie de Procedure.

**`<WLanguage procedure>: (Nombre de Procedure)`**

WLanguage Procedure en el siguiente formato: 


```txt
PROCEDURE <Procedure name>(<Element1>, <Element2>)
```
Esta Procedure es llamada automáticamente por el mecanismo de supresión para comparar los elementos entre sí, de dos en dos.

- Si el elemento 1 debe encontrarse antes que el elemento 2 en el orden de clasificación, la Procedure debe regresar -1.

- Si el elemento 1 debe encontrarse después del elemento 2 en el orden de clasificación, la Procedure debe devolver el 1.

- Si el elemento 1 y el elemento 2 son idénticos, la Procedure debe devolver 0. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Esta función no está disponible para la arrays fija.

- Esta función se puede utilizar en la arrays de variables avanzadas. 

- array dinámica de variantes: Sólo se puede utilizar una eliminación personalizada (con un WLanguage Procedure).

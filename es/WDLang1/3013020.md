
## LoadWDL( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Carga una biblioteca de objetos WINDEV en la memoria (.WDL). Esta función permite:

- usar bibliotecas personalizadas para cada usuario. Esto le permite crear varias bibliotecas e instalar una de ellas según el usuario.

- dividir una biblioteca "grande" en varias bibliotecas "más pequeñas", cargadas a petición.




Esta librería se cargará en la memoria hasta que:

- la aplicación termina,

- [DescargarWDL](../WDLang1/3013013.md) se ejecuta.



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Initialization code of project
// The library depends on the user
LibraryName, User are strings
hWDL_ is int
User = CommandLine("User")
SWITCH User
	CASE User1: LibraryName = "USER1.WDL"
	CASE User2: LibraryName = "USER2.WDL"
	CASE User3: LibraryName = "USER3.WDL"
END
hWDL_ = LoadWDL(LibraryName)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = LoadWDL(<Library name>)`
---

**`<Result>: (Integro)`**



- Número de identificación (manipular) de la biblioteca

- -1 si la biblioteca no se ha cargado correctamente.




**`<Library name>: (Cadena de caracteres)`**

Nombre y ruta completa (o relativa) de la biblioteca a cargar.

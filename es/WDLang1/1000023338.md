
## EpochToDateTime (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Convierte una fecha y hora en formato Epoch (número de unidades de tiempo desde el 01/01/1970 a las 00h00 UTC) en una fecha y hora local.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Displays "20220101000000000" (in France)
Info(EpochToDateTime("1640991600"))
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = EpochToDateTime(<Epoch datetime> [, <Conversion options>])`
---

**`<Result>: (Cadena de caracteres)`**

Fecha y hora locales resultantes de la conversión. Puede corresponder a:

- una variable de tipo DateTime.

- una cadena de caracteres en formato AAAAMMDDHHmmSSCCC (AAAA corresponde al año, MM al mes, DD al día, HH a las horas, mm a los minutos, SS a los segundos y CCC a los milisegundos).

- una cadena vacía ("") si se produce un error.




**`<Epoch datetime>: (Entero de 8 bytes)`**

Fecha y hora en formato Epoch (número de unidades de tiempo desde el 01/01/1970 a las 00h00 UTC) a convertir.

**`<Conversion options>: (Constante opcional de tipo Integer)`**

Opciones para convertir la fecha y la hora:


|   |   |
| --- | --- |
| *epochUnixMillisecond* | La fecha y hora en formato Epoch se expresa en milisegundos desde el 01/01/1970 a las 00h00 UTC (hora 0 en Unix). |
| *epochUnixSecond*<br>(valor predeterminado) | La fecha y hora en formato Epoch se expresa en segundos desde el 01/01/1970 a las 00h00 UTC (hora 0 en Unix). |

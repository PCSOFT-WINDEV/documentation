
## LoadError( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Carga en la memoria el archivo que contiene los mensajes de error en el idioma deseado. Este fichero debe haber sido creado por WDINT (herramienta de traducción).

Observación: Este archivo también contiene la traducción de las características de la aplicación automatic (AAF) y las interfaces de los controles avanzados (tratamiento de textos, hoja de cálculo, editor Image, etc.). 

Esta función debe ser llamada en el código de inicialización del proyecto para mostrar los mensajes de error generados por las DLLs de WINDEV, WINDEV Mobile y WEBDEV en el idioma deseado.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF LoadError("GERMAN.WDM") > 0 THEN
	Error("File containing the messages in German not found", ...
		"The messages will be displayed in French")
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = LoadError(<Name of the WDM file>)`
---

**`<Result>: (Integro)`**

estado reporte para cargar el fichero de errores:

- 0 si la operación de carga se ha realizado con éxito,

- 1 si &lt;Nombre del archivo WDM&gt; no fue encontrado, 

- 2 si &lt;Nombre del archivo WDM&gt; fue encontrado pero no tenía la estructura esperada, 

- 3 si &lt;Nombre del archivo WDM&gt; no se pudo inicializar, 

- 4 si &lt;Nombre del archivo WDM&gt; no es una biblioteca, 

- 5 si &lt;Nombre del archivo WDM&gt; no se pudo cargar en la memoria.




**`<Name of the WDM file>: (Cadena de caracteres)`**

Nombre del archivo de mensajes de error a utilizar (extensión.WDM). Este nombre puede ser un nombre completo o un nombre relativo. Este fichero debe haber sido creado por WDINT (herramienta para traducir mensajes no suministrados en estándar con WINDEV, WINDEV Mobile o WEBDEV)..  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Si se elige el idioma "Inglés" (por ejemplo) en el editor de ventanas (o en el editor de páginas), todos los programas ejecutables creados por WINDEV/WINDEV Mobile (o los sitios creados por WEBDEV) se ejecutarán en inglés.. Para ello, la traducción en inglés de los elementos encontrados en la aplicación (o en el sitio) debe haber sido introducida en los diferentes editores.. [Nación](../WDLang1/3054014.md) se utiliza para modificar el lenguaje de ejecución de la aplicación a través de la programación.

- **LoadError** no puede utilizarse para cargar archivos distintos de los creados con WDINT.EXE.

- Por Default, si no se carga ningún archivo con **LoadError**, los mensajes de error se muestran en francés.

- **LoadError** es independiente de [Nación](../WDLang1/3054014.md). Si se dispone de un archivo de errores para cada idioma soportado por la aplicación (o el sitio), este archivo WDM debe recargarse después de cada cambio de idioma realizado por [Nación](../WDLang1/3054014.md).

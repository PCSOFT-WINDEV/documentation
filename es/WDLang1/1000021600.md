
## ActionBarDeleteSearchHistory( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Borra la historia de la búsqueda que control encontró en el Action Bar de una ventana. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Deletes the search history from the current window
ActionBarDeleteSearchHistory()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`ActionBarDeleteSearchHistory([<Window>])`
---

**`<Window>: (Nombre de ventana)`**

Nombre de la ventana para la que se debe borrar el historial de búsqueda. Si este parámetro no se especifica o si corresponde a una cadena de caracteres vacía, la eliminación se realiza en la ventana Active. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Se produce un error fatal si la ventana no incluye ningún Action Bar control.




## Funciones de gestión de fechas y horas
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones permiten gestionar fechas y horas:



|   |   |
| --- | --- |
| [Age](../WDLang1/3027034.md) | Devuelve la edad según la fecha de nacimiento. |
| [BankHoliday](../WDLang1/1000017303.md) | Indica si un día es festivo o no. |
| [BankHolidayAdd](../WDLang1/1000017304.md) | Indica que un día (o una lista de días) es un día festivo. |
| [BankHolidayDeleteAll](../WDLang1/1000017301.md) | Borra la lista de todos los días festivos. |
| [BankHolidayList](../WDLang1/1000017302.md) | Devuelve la lista de los días festivos definidos con la función [BankHolidayAdd](../WDLang1/1000017304.md). |
| [ChronoEnd](../WDLang1/3027032.md) | Detiene un cronómetro y devuelve el tiempo transcurrido (en milisegundos) desde que se inició el cronometraje (con la función [ChronoStart](../WDLang1/3027031.md)). |
| [ChronoPause](../WDLang1/1000018829.md) | Pone en pausa un cronómetro para medir el tiempo de un proceso. |
| [ChronoReset](../WDLang1/1000020474.md) | Detiene y reinicia un cronómetro en marcha. |
| [ChronoResume](../WDLang1/1000018830.md) | Reinicia un cronómetro que se detuvo con las funciones [ChronoEnd](../WDLang1/3027032.md) o [ChronoPause](../WDLang1/1000018829.md). |
| [ChronoStart](../WDLang1/3027031.md) | Inicia un cronómetro para medir la duración de un proceso (en milisegundos) y reinicia un cronómetro en marcha. |
| [ChronoValue](../WDLang1/3027033.md) | Indica el tiempo transcurrido (en milisegundos) desde que se llamó a la función [ChronoStart](../WDLang1/3027031.md). |
| [CurrentMonth](../WDLang1/1000019415.md) | Devuelve el número del mes actual. |
| [CurrentYear](../WDLang1/1000019394.md) | Devuelve el año actual como número entero.. |
| [DateDifference](../WDLang1/3027001.md) | Calcula el número de días entre dos fechas. |
| [DatePicker](../WDLang1/1000020018.md) | Abre el selector de fecha del sistema. |
| [DateSys](../WDLang1/3027026.md) | Devuelve o modifica la fecha del sistema (fecha actual definida en el equipo). |
| [DateTimeByDefault](../WDLang1/3027015.md) | Permite definir el valor predeterminado de variables de tipo Date, Time y DateTime. |
| [DateTimeDifference](../WDLang1/3027020.md) | Calcula la diferencia entre dos fechas y horas. |

| [DateTimeTimeZoneToLocal](../WDLang1/1000020976.md) | Convierte la fecha y hora de una zona horaria determinada en una fecha y hora local. |
| [DateTimeToEpoch](../WDLang1/1000023337.md) | Convierte una fecha y hora local en formato Epoch (número de unidades de tiempo desde el 01/01/1970 a las 00h00 UTC). |
| [DateTimeToReal](../WDLang1/1000021361.md) | Convierte una fecha y hora en un real de 8 bytes. |
| [DateTimeToString](../WDLang1/1000025516.md) | Convierte una fecha/hora en una cadena en el formato especificado. |
| [DateTimeUTCToLocal](../WDLang1/3027037.md) | Convierte una fecha y hora UTC (Tiempo Universal) en hora local (zona horaria, horario de verano, horario de invierno). |
| [DateTimeValid](../WDLang1/1000020940.md) | Comprueba la validez de una fecha/hora entre el 01 de enero de 0001 a las 0 horas 0 minutos 0 segundos 0 centésimas de segundo y el 31 de diciembre de 9999 a las 23 horas 59 minutos 59 segundos 99 centésimas de segundo. |
| [DateToDay](../WDLang1/3027004.md) | Devuelve el día que corresponde a una fecha determinada. |
| [DateToDayInAlpha](../WDLang1/3027010.md) | Devuelve el nombre del día de una fecha determinada. |
| [DateToDayOfWeek](../WDLang1/1000024867.md) | Devuelve el día de la semana que corresponde a una fecha determinada. |
| [DateToInteger](../WDLang1/3027014.md) | Convierte una fecha en un número entero. |
| [DateToMonthInAlpha](../WDLang1/3027012.md) | Devuelve el nombre del mes de una fecha determinada. |
| [DateToString](../WDLang1/3027025.md) | Convierte: <br><br>- una fecha en cadena en el formato especificado.<br><br>- una fecha/hora en una cadena en el formato especificado.<br><br><br> |
| [DateToWeekNumber](../WDLang1/3027011.md) | Devuelve el número de la semana del año que corresponde a una fecha determinada. |
| [DateValid](../WDLang1/3027003.md) | Comprueba la validez de una fecha entre el 1 de enero de 0001 y el 31 de diciembre de 9999. |
| [DayNumberInAlpha](../WDLang1/1000019906.md) | Devuelve el nombre del día de la semana según su número. |
| [DurationToString](../WDLang1/3027007.md) | Convierte una duración en una cadena de caracteres en el formato especificado. |
| [Easter](../WDLang1/3027035.md) | Devuelve la fecha de Pascua del año especificado según el calendario cristiano occidental. |
| [EpochToDateTime](../WDLang1/1000023338.md) | Convierte una fecha y hora en formato Epoch (número de unidades de tiempo desde el 01/01/1970 a las 00h00 UTC) en una fecha y hora local. |
| [FirstDayOfMonth](../WDLang1/1000019417.md) | Devuelve el primer día del mes. |
| [FirstDayOfWeek](../WDLang1/1000019882.md) | Devuelve la fecha del primer día de la semana (es decir, el lunes). |
| [FirstDayOfYear](../WDLang1/1000021887.md) | Devuelve la fecha del primer día del año. |
| [IntegerToDate](../WDLang1/3027009.md) | Convierte un número entero en una fecha en formato AAAAMMDD. |
| [IntegerToDay](../WDLang1/3027008.md) | Devuelve el día de la semana que corresponde a una fecha determinada (esta fecha es un número entero). |
| [IntegerToDayInAlpha](../WDLang1/3027028.md) | Devuelve el nombre del día que corresponde a una fecha determinada (esta fecha es un número entero). |

| [IntegerToWeekNumber](../WDLang1/3027030.md) | Devuelve el número de la semana del año que corresponde a una fecha determinada (esta fecha es un entero). |
| [LastDayOfMonth](../WDLang1/1000019399.md) | Devuelve la fecha del último día del mes especificado. |
| [LastDayOfWeek](../WDLang1/1000019883.md) | Devuelve la fecha del último día de la semana que corresponde a una fecha determinada. |
| [LastDayOfYear](../WDLang1/1000025806.md) | Devuelve la fecha del último día del año. |

| [MonthNumberInAlpha](../WDLang1/1000019907.md) | Devuelve el nombre del mes según su número. |
| [Now](../WDLang1/3027024.md) | Devuelve o modifica la hora del sistema (hora actual definida en el equipo). |
| [RealToDateTime](../WDLang1/1000021364.md) | Convierte un real de 8 bytes en una fecha y hora. |
| [StringToDate](../WDLang1/3027013.md) | Convierte una fecha en formato de cadena de caracteres en una fecha en formato AAAAMMDD. |
| [StringToDuration](../WDLang1/3027002.md) | Convierte una duración en formato de cadena de caracteres en una duración que puede ser utilizada por una variable de tipo [Duration](../Motscles/1514069.md). |




## Funciones de control Cinta de opciones
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para gestionar los controles de la cinta de opciones:



|   |   |
| --- | --- |
| [RibbonAdaptSize](../WDLang1/1000020761.md) | Adapta el tamaño de los mandos que se encuentran en una control Cinta de opciones en función del tamaño de su título. |
| [RibbonCloseGrouping](../WDLang1/1000020136.md) | Cierra el grupo actualmente abierto en una control Cinta de opciones. |
| [RibbonEndExpandedMode](../WDLang1/1000020378.md) | Colpasa una cinta colapsada que se expande temporalmente. |
| [RibbonHighlightPane](../WDLang1/1000020530.md) | Destaca una pestaña en un control Cinta de opciones. |

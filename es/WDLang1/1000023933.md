


## AddReference (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Agrega la referencia de un objeto en la última posición en un array unidimensional.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
AnArray is array of AClass
AnObject is AClass
AddReference(AnArray, AnObject)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = AddReference(<WLanguage array> , <Object>)`
---

**`<Result>: (Entero)`**

Índice del array al que se agregó el elemento. 

**`<WLanguage array>: (Array)`**

Nombre de la variable de tipo Array a utilizar. Este array debe ser unidimensional.

**`<Object>: (Objeto)`**

Nombre del objeto cuya referencia se agrega al array.

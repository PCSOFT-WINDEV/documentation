


## &lt;Caja de Lista&gt;.InfoXY (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el índice de la fila (o Image) en una posición dada en un Cuadro de Lista o control ListView.

**Observaciones**: 

- Esta función no se puede utilizar con un control Combo Box ni con un control Carrusel. 

- La función **&lt;Caja de Lista&gt;.InfoXY** puede utilizarse en:

	- una control List Box de navegación o de memoria,

	- una selección única o una selección múltiple control List Box.










<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// -- Rollover code of control 
// Display a tooltip when the elements in a List Box control are hovered
Index is int 
Index = LIST_List1.InfoXY(tiLineNumber, MouseXPos(), MouseYPos())
IF Index > 0 THEN
	LIST_List1..Tooltip = LIST_List1[Index] 
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <List Box control>.InfoXY(<Type of information> , <X> , <Y>)`
---

**`<Result>: (Cadena de caracteres o entero)`**



- Información solicitada.

- -1 si no hay índice fila o si la posición especificada corresponde a un fila vacío.




**`<List Box control>: (Nombre del control)`**

Nombre del control a manipular: 

- Control List Box, 

- Control ListView.




**`<Type of information>: (Constante de tipo Integer)`**

Tipo de información solicitada:


|   |   |
| --- | --- |
| *tiLineNumber* | <br><br>- Número Fila para un control List Box estándar,<br><br>- El número de Image para un control ListView.. <br><br><br> |
| *tiOriginScreen* | Por Default, (0,0) corresponde a fila 1 (las barras de desplazamiento se encuentran en el origen).<br>Si la constante *tiOriginScreen* está asociada a la constante anterior, el punto (0,0) corresponde al origen de la pantalla.<br>Esta constante no puede utilizarse por sí sola. |



**`<X>: (Integro)`**

Coordenada X (en píxeles) a analizar. Esta coordenada se da en relación con el control List Box (si no se especifica la constante *oriScreenOrigin*).

**`<Y>: (Integro)`**

Coordenada Y (en píxeles) a analizar. Esta coordenada se da en relación con el control List Box (si no se especifica la constante *oriScreenOrigin*).



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Índice devuelto
<a name="indice_devuelto_ELTPARAGRAPHE000234"></a>

El índice devuelto por **&lt;Caja de Lista&gt;.InfoXY** corresponde al índice real del fila (no al índice mostrado). Este índice tiene en cuenta la fila ya mostrada pero no visible en la control List Box.

Además, si se utilizan las barras de desplazamiento, la función **&lt;Caja de Lista&gt;.InfoXY** tiene en cuenta este desplazamiento.


## ShapePolygon( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica la forma que se muestra actualmente en una control Forma para obtener una forma personalizada (especificando las coordenadas de los puntos de la Polygon). 

Le permite crear sus propias formas (para crear gráficos, por ejemplo).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
T1 is array of int = [5,5,5,505,505,505,505,5]
ShapePolygon(SHA_NoName1, T1)
```



```wl
arrPoints is array of int = ...
[10,10,
1000,10,
1000,800,
900,800,
700,1000,
750,800,
10,800]

ShapePolygon(SHA_NoName1, arrPoints)

SHA_NoName1.BackgroundColor = PastelOrange
SHA_NoName1.Color = DarkOrange
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`ShapePolygon(<Shape control> , <Points>)`
---

**`<Shape control>: (Nombre del control)`**

control Forma a utilizar. 

**`<Points>: (Array de enteros o puntos)`**

Array ot enteros o array de variables [Punto](../Motscles/1000018908.md) que contienen los puntos de paso de Polygon. 

Si esta array es una array de enteros, los índice pares representan las coordenadas X mientras que los subíndices impares representan las coordenadas Y de los puntos. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Las posiciones X e Y son relativas al tamaño de control y están expresadas en 1/1000.

La forma inicial se incluye en un área identificada por una Marker de (0,0) a (1000, 1000). Las coordenadas de los puntos de la nueva forma deben ser dadas de acuerdo con esta Marker. 
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=FormePolygone.gif)


Observación: El primer y el último punto son automáticamente relacionado.




## SchedulerDisplayNextPeriod (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Mueve el área de tiempo que se muestra en el área siguiente (por ejemplo, las próximas semanas) en un control Planificador. El control Planificador se visualiza a partir del día correspondiente al área de tiempo siguiente. 

**Observación**: Las funciones [SchedulerDisplayPreviousPeriod](../WDLang1/1000019495.md) y **SchedulerDisplayNextPeriod** permiten establecer fácilmente controles Botón "Adelante/Atrás". Estos controles le permiten desplazarse por las áreas de tiempo que muestra la pantalla de control Planificador.
 

<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Click the BTN_Next Button control
SchedulerDisplayNextPeriod(SCH_OfficeScheduling)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`SchedulerDisplayNextPeriod(<Scheduler control>)`
---

**`<Scheduler control>: (Nombre del control)`**

Nombre de control Planificador a utilizar.


## CameraStop (Función)



<a name="XUse"></a>


<a name="Use"></a>
<a name="description"></a>
Detiene la secuencia de vista previa o grabación de un video en un control Cámara.






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Stops the camera preview stream
// in the "CAM_Kitchen" control
ResStop = CameraStop(CAM_Kitchen, viStopDisplay)
```

<a name="XSYNTAX"></a>


<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = CameraStop(<Camera control> [, <Action to stop>])`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si la secuencia de vista previa de la cámara y/o la grabación de video en el control Cámara se detuvieron correctamente,

- <u><u><u><u>False</u></u></u></u> si se produce un error. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Camera control>: (Nombre del control)`**

Nombre del control Cámara a utilizar.

**`<Action to stop>: (Constante opcional de tipo Integer)`**

Tipo de acción a detener:


|   |   |
| --- | --- |
| *viStopAll*<br>(Valor predeterminado) | Detiene la secuencia de vista previa y la grabación de video. Se genera el archivo correspondiente. La visualización en el control Cámara especificado se detiene. |
| *viStopCapture* | Detiene la grabación de video. Se genera el archivo correspondiente. |
| *viStopDisplay* | Detiene la secuencia de vista previa de la cámara. La visualización en el control Cámara especificado se detiene. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Condiciones de uso
<a name="condiciones_uso_ELTPARAGRAPHE000065"></a>

- ![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) Para utilizar esta función, se debe conectar una cámara al equipo actual.

- **Observación**: Esta función es equivalente a la función [VideoStop](../WDLang1/3079001.md). 



<a name="NOTE0_2"></a>


### Casos especiales
<a name="casos_especiales_ELTPARAGRAPHE000084"></a>

- Para Record el vídeo capturado por una cámara web, utilice [CameraCapture](../WDLang1/1000022450.md).

- Para que se muestre el video capturado por una "cámara Web" en un control Cámara, utilice la función [CameraDisplay](../WDLang1/1410086595.md).


## ToastDeleteAll( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Borra todos los brindis mostrados por [ToastDisplay](../WDLang1/1000019358.md). Si no se muestra toast, la función no tiene ningún efecto.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Close the window: all the toasts are deleted
ToastDeleteAll()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`ToastDeleteAll()`

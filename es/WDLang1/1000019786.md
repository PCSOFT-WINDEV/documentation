
## Gesto Variable
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000176"></a>
El Gesto Variable se utiliza para recuperar los parámetros del gesto (swipe): dirección, velocidad, distancia, ....

Este Variable puede ser usado: 

- en el golpe Event de elementos en un control Looper, para saber la dirección del golpe. 

- en eventos multitáctiles (zoom, scroll y swipe) para evitar declarar parámetros en los eventos.

<a name="NOTE2_1"></a>


## Gesto Variable
<a name="gesto_variable_ELTTEXTE000200"></a>
El gesto Variable incluye los siguientes elementos:


|   |   |
| --- | --- |
|   | Real<br><br>Ángulo (expresado en grados) correspondiente a la inclinación de un Lápiz de Manzana con respecto al plano de la tableta.<br><br>Esta variable solo se especifica en el evento "Presionado/botón izquierdo presionado". |
|   | Constante de tipo Integer<br><br>Dirección del gesto. Puede corresponder a una de las siguientes constantes:<br><br>- *BottomToTop*: Pase de abajo hacia arriba.<br><br>- *RightToLeft*: Deslízate de derecha a izquierda.<br><br>- *LeftToRight*: Pase de izquierda a derecha.<br><br>- *TopToBottom*: Deslízate de arriba a abajo.<br><br><br>Este Variable está lleno de eventos de Pergamino y Golpe. |
|   | Para un pergamino: valor de la Move( en píxeles) en la dirección especificada por Gesture.Direction.<br><br>Para un gesto de zoom de pellizco: valor del espacio (en píxeles) entre dos puntos Contact.<br><br>Este Variable está lleno de eventos de zoom de desplazamiento y pellizco. |
|   | Real<br><br>Presión durante una prensa.<br><br>Esta presión puede tomar un valor comprendido entre 0,0 y 1,0.<br><br>Esta variable solo se especifica en el evento "Presionado/botón izquierdo presionado". |
|   | Real<br><br>Ángulo azimutal (expresado en grados) de un Lápiz de Manzana en relación con la marca de la actual control.<br><br>Esta variable solo se especifica en el evento "Presionado/botón izquierdo presionado". |
|   | Real<br><br>Presión máxima durante una prensa.<br><br>Esta presión puede tomar un valor comprendido entre 0,0 y 1,0.<br><br>Esta variable solo se especifica en el evento "Presionado/botón izquierdo presionado". |
|   | Integro<br><br>Velocidad de movimiento en la dirección especificada por Gesture.Direction. <br>Esta velocidad se expresa en píxeles por segundo.<br><br>Este Variable está lleno de eventos de Swipe. |

## Ejemplo
<a name="ejemplo_ELTTEXTE000224"></a>


### Recuperación de información de gestos
<a name="recuperacion_informacion_gestos_ELTPARAGRAPHE000127"></a>


```wl
// Scroll with Finger event of IMG_Image
SWITCH Gesture.Direction
	CASE RightToLeft: RADIO_Direction = 1
	CASE LeftToRight: RADIO_Direction = 2
	CASE TopToBottom: RADIO_Direction = 3
	CASE BottomToTop: RADIO_Direction = 4
END

EDT_Speed = Gesture.Speed
EDT_Distance = Gesture.Distance
```

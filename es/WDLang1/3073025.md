
## SysWindowsVersion( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve información sobre la versión de Windows utilizada en el equipo actual.



![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Devuelve la versión del sistema.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 
```wl
// Number of the Windows version used on the current computer
ResVersion = SysWindowsVersion(SysVersionNumber)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = SysWindowsVersion([<Type of information>])`
---

**`<Result>: (Cadena de caracteres)`**



- Información sobre la versión de Windows,

- Cadena vacía ("") si se produce un error.




**`<Type of information>: (Constante opcional)`**

Indica la información solicitada: 



|   |   |
| --- | --- |
| Parámetro no especificado | Versión para Windows:<br><br>- "11" para Windows 11<br><br>- "10" para Windows 10<br><br>- "8" para Windows 8<br><br>- "7" para Windows 7<br><br>- "VISTA"<br><br>- "LINUX"<br><br>- "2022S" para Windows 2022 Server<br><br>- "2019S" para Windows 2019 Server<br><br>- "2016S" para Windows 2016 Server<br><br>- "2012S" para Windows 2012 Server<br><br>- "2008S" para Windows 2008 Server<br><br>- "2003S" para Windows 2003 Server<br><br>- "XP" para Windows XP<br><br>- "ME" para Windows Millenium<br><br>- "NT 4", "NT 5" (para Windows 2000 por ejemplo)<br><br>- "NT 3.5"<br><br>- "98"<br><br>- "95"<br><br>- "32s" para Windows 3.1<br><br><br>Cadena vacía si la versión de Windows es desconocida.<br><br><br><br><br><br> |
| *sysProcessor* | Devuelve el modo de tiempo de ejecución del programa en el entorno actual:<br><br>- "32": el programa funciona en modo de 32 bits en un entorno de 32 bits.<br><br>- "64": el programa funciona en modo de 32 bits o 64 bits en un entorno de 64 bits.<br><br><br><br><br>![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Esta constante no está disponible. |
| *sysVersionBuild* | Número de compilación de Windows.<br>Por ejemplo: "1381".<br><br>![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Esta constante no está disponible. |
| *sysVersionInfo* | Información específica sobre la versión de Windows. Por ejemplo:<br><br>- "Service Pack x" para NT<br><br>- "para Windows 95A (cadena vacía)<br><br>- "B" para Windows 95B<br><br>- " "para Windows 98 (un espacio).<br><br>- "Pro" para Windows XP Pro<br><br>- "Home Edition" para Windows XP Home Edition.<br><br><br><br><br>![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Esta constante no está disponible. |
| *sysVersionNumber* | Número de versión del núcleo de Windows.<br>Por ejemplo:<br><br>- "3.51" para NT 3.51.<br><br>- "10" para Windows 11 21H2...<br><br><br><br><br>![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Esta constante no está disponible. |
| *sysVersionPlatform* | Plataforma del sistema:<br><br>- "Win32s" para WIN32 en 3.11<br><br>- "Windows" para Windows 95/98/ME<br><br>- "NT" para Windows NT 4.0/2000/XP/2003 Server<br><br>- "UNIX" para Linux.<br><br><br><br><br>![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Esta constante no está disponible. |


![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Este parámetro no está disponible.

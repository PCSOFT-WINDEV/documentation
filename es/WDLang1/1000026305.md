


## &lt;Editor de HTML&gt;.GetSelection (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Recupera de forma asíncrona la selección actual de un editor HTML control. 

**Observación**: Esta función es especialmente útil para las aplicaciones Android e iOS para las que el lenguaje WL de [Selección](../Proprietes/1000023033.md) Property no está disponible en modo de lectura.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Gray out the "Open" button (or not) according to the selection
HTMEDT_MyHTMLEditor.GetSelection(HTMLEditorGetSelection_Callback)
INTERNAL PROCEDURE HTMLEditorGetSelection_Callback(Selection is edtHtmlSelection)
   	BTN_Open.Grayed = NOT Left(selection.Text, "https:\\" )
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<HTML Editor control>.GetSelection(<WLanguage procedure>)`
---

**`<HTML Editor control>: (Nombre del control)`**

Nombre del editor HTML utilizado por control.

**`<WLanguage procedure>: (Nombre de Procedure)`**

WLanguage Procedure ("callback") ejecutado una vez recuperada la selección. Este procedimiento tiene el siguiente formato: 

```txt
PROCEDURE <Procedure name>(MySelection is edtHtmlSelection)
```
donde &lt;MiSelección&gt; es una [edtHTMLSelection](../WDLang1/1000026093.md) Variable que corresponde a la selección actual en el Editor HTML control.

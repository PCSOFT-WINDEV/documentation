
## Switzerland.CheckBER (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Comprueba la validez de un número BER (número de identificación en el registro de empresas en Suiza).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Check a variable
MyBERNumber is string = "73462600"
Switzerland.CheckBER(MyBERNumber)		// Returns True

// Check a control
EDT_BERNumber = "ABC"
Switzerland.CheckBER(EDT_BERNumber)	// Returns False
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = Switzerland.CheckBER(<Number to check>)`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si el elemento especificado contiene un número BER válido.

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Number to check>: (Nombre de control o variable de tipo String)`**

Número BER a comprobar. Este parámetro puede corresponder a:

- el nombre de una variable de tipo String.

- el nombre de un control.

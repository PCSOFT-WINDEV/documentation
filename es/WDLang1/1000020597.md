


## CheckBoxInsert (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Inserta una nueva opción en una control Casilla de verificación.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Inserts an option at position 2 
CheckBoxAdd(CBOX_Options, 2, "Read a book on a tablet")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = CheckBoxInsert(<Check Box control> , <Element index> , <Caption> [, <Sub-caption> [, <Image>]])`
---

**`<Result>: (Integro)`**



- Índice de la opción insertada. 

- -1 si se produce un error. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Check Box control>: (Nombre del control)`**

Nombre de control Casilla de verificación en la que se insertará la opción.

**`<Element index>: (Integro)`**

Índice del elemento a insertar. 

Si este parámetro:

- es mayor que el número de opciones encontradas en la casilla de verificación, la opción se agrega después de la última opción encontrada en la sección control Casilla de verificación. 

- es igual a 1 o 0, el elemento se añade antes de la primera opción que se encuentra en la control Casilla de verificación.




**`<Caption>: (Cadena de caracteres)`**

Texto de la opción a añadir.

**`<Sub-caption>: (Cadena de caracteres opcional)`**

Subtexto de la opción a añadir. 
Si no se especifica este parámetro, la nueva opción no tiene subtexto. 
Si se especifica este parámetro, el subtexto se mostrará incluso si la casilla de verificación utiliza un modo de visualización sin subtexto.


**`<Image>: (Cadena de caracteres opcional)`**

Imagen associated con la opción de insertar. 
Si no se especifica este parámetro, la opción añadida no tiene Image.
Si se especifica este parámetro, la Image se mostrará sólo si la casilla de verificación está usando un modo de visualización "estilo Vista" (con Image).




<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Las características de la opción insertada también pueden ser modificadas por las siguientes propiedades utilizadas en la opción: 

	- [Pie de foto](../Proprietes/2510053.md) property para cambiar el texto. 

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) [SubCaption](../Proprietes/1000017232.md) property para cambiar el subtexto. 

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) [Image](../Proprietes/2510034.md) property para modificar el image. 




- Al insertar las opciones en una control Casilla de verificación, el tamaño de la control no se modifica. Las opciones añadidas pueden no ser Visible si el control no es lo suficientemente grande. La control puede ampliarse mediante las propiedades [anchura](../Proprietes/2510059.md) y [Altura](../Proprietes/2510050.md).

- Para añadir una opción después de las opciones existentes, utilice [CheckBoxAdd](../WDLang1/1000020595.md). 

- Al insertar una opción: 

	- el índice de la opción insertada es &lt;índice del elemento&gt;. 

	- los índices de las opciones que siguen a la opción insertada se renumeran automáticamente.

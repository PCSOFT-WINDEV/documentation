## Presentación
<a name="presentacion_ELTTEXTE000118"></a>
Las aplicaciones pueden ser controladas por los movimientos del dispositivo móvil (Phone o tableta)!

Por ejemplo, un movimiento a la derecha va al siguiente formulario, un movimiento a la izquierda va al formulario anterior, un movimiento a las posiciones superiores al principio del archivo y un movimiento a las posiciones inferiores al final del archivo.

Este tipo de interfaz se puede programar fácilmente con WINDEV Mobile.

Las funciones del sensor permiten proponer este tipo de interfaz.

Entre las acciones apoyadas:

- bache (movimiento brusco) (principio y final),

- cambio de aceleración,

- cambio de orientación,

- orientación (brújula).




Este tipo de interfaz puede combinarse o coexistir con una interfaz estándar (botones, ...).

## Funciones disponibles
<a name="funciones_disponibles_ELTTEXTE000142"></a>
Las siguientes funciones se utilizan para manejar la brújula y el acelerómetro de un dispositivo móvil (Phone o tableta):



|   |   |
| --- | --- |
| [SensorDetectBeginShake](../WDLang1/1000019363.md) | Notifica cuando el dispositivo se agita en cualquier dirección. |
| [SensorDetectChangeAcceleration](../WDLang1/1000019362.md) | Notifica cuando se aplican fuerzas de aceleración al dispositivo a lo largo de cualquiera de los tres ejes. |
| [SensorDetectChangeOrientation](../WDLang1/1000019361.md) | Permite ser notificado cuando la orientación del dispositivo cambia en cualquiera de sus 3 ejes. |
| [SensorDetectEndShake](../WDLang1/1000019364.md) | Notifica cuando el dispositivo ha sido sacudido en cualquier dirección. |
| [SensorEnd](../WDLang1/1000019644.md) | Indica el fin del uso de las funciones del sensor. |
| [SensorGetOrientation](../WDLang1/1000019369.md) | Recupera la orientación actual del dispositivo en uno de sus 3 ejes. |




## SysDetectLockedSession( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Detecta el cambio de bloqueo de la sesión de Windows.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF NOT SysDetectLockedSession() THEN
	Error("Failure detecting the change of lock for the Windows session.")
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = SysDetectLockedSession(<WLanguage procedure>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se estableció la detección del cambio de bloqueo para la sesión de Windows,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<WLanguage procedure>: (Nombre de Procedure)`**

Nombre de WLanguage Procedure ("callback" Procedure) que será llamado cada vez que cambie el bloqueo de la sesión de Windows. 

Este procedimiento tiene el siguiente formato: 

```txt
PROCEDURE <Procedure name> (<Lock of Windows session>)
```


&lt;Lock of Windows session&gt; es una constante que es un set a:


|   |   |
| --- | --- |
| *sysSessionLocked* | La sesión de Windows está bloqueada. |
| *sysSessionUnlocked* | La sesión de Windows está desbloqueada. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Si se utiliza la función **SysDetectLockedSession** en un servicio, llame a la función [Multitask](../WDLang1/3015004.md) en el código de ejecución del servicio para recibir los mensajes de Windows. 


```wl
Multitask(-1)
```

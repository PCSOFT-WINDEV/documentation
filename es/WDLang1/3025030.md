
## NextIndex( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el número de la siguiente fila mostrada en la control Tabla actual.

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Esta función también puede utilizarse para devolver el índice de la siguiente control indexada .




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// -- Exit code of the row
RowNum is int = NextIndex()
Trace("Number of the next row: " + RowNum)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = NextIndex()`
---

**`<Result>: (Integro)`**

Número de la siguiente fila visualizada en el control Tabla actual. **NextIndex** debe utilizarse en el código de salida del fila. El resultado no tendrá sentido si esta función se utiliza en otro Process.

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Índice de la siguiente control indexada. **NextIndex** debe utilizarse en el código de salida del control. El resultado no tendrá sentido si esta función se utiliza en otro Process.


## Procedimiento WLanguage llamado por la función DialogAsynchronous



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Procedimiento ("Callback") llamado por la función [DialogAsynchronous](../WDLang1/1000025310.md) cuando el usuario hace clic en uno de los botones del cuadro de diálogo. 
Este procedimiento permite saber qué botón se pulsó, y ejecutar un proceso específico, si es necesario. 
Observación: Este procedimiento puede ser local, global, interno o lambda. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
nButton is int = Random(2)
DialogAsynchronous("Click %1 to start the application.", ...
		MyProcedure_Dialog, nButton)

INTERNAL PROCEDURE MyProcedure_Dialog(nParam1 int)
	IF nButton <> nParam1 THEN
		EndProgram()
	END
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`DialogAsynchronous_Callback(<Selected answer>)`
---

**`<Selected answer>: (Entero)`**

Identifica la respuesta seleccionada por el usuario. 
Los valores posibles se definieron en el asistente de creación de la función [DialogAsynchronous](../WDLang1/1000025310.md). 
Observación: Los valores que corresponden a las diferentes respuestas se integran automáticamente como comentarios en el editor de código al crear el diálogo mediante el asistente.

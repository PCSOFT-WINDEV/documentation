
## OrganizerZoom (Función)






	<B><B>Advertencia</B></B>

	A partir de la versión <B><B>17</B></B>, esta función se conserva por motivos de compatibilidad con versiones anteriores. Esta función se sustituye por [OrganizerMode](../WDLang1/1000019545.md).



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el intervalo de tiempo mostrado por un control Agenda: día, semana, mes, ....

**Observación**: [OrganizadorCambioZoom](../WDLang1/1000018883.md) se utiliza para modificar el rango de tiempo mostrado por el control Agenda.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Toggles between daily display and monthly display
IF OrganizerZoom(ORG_MyOrganizer) = orgzDay THEN
	OrganizerChangeZoom(ORG_MyOrganizer, orgzMonth)
ELSE
	OrganizerChangeZoom(ORG_MyOrganizer, orgzDay)
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = OrganizerZoom(<Organizer control>)`
---

**`<Result>: (Constante de tipo Integer)`**

Rango de tiempo mostrado por la control Agenda:


|   |   |
| --- | --- |
| *orgzDay* | El organizador muestra un día. |
| *orgzMonth* | El organizador muestra un mes completo. |
| *orgzWeek* | El organizador muestra una semana completa (o una semana Partial si se especificó un número de días en [OrganizadorCambioZoom](../WDLang1/1000018883.md)). |
| *orgzWeekOn5* | El organizador muestra una semana en 5 días (sin fin de semana). |



**`<Organizer control>: (Nombre del control)`**

Nombre de la control Agenda que se va a utilizar. Este control puede corresponder a: 

- una control Agenda en una ventana o en una página,

- una control Agenda en una reporte.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Las siguientes líneas de código son equivalentes: 


```wl
OrganizerZoom(ORG_Organizer, orgzWeek, 5)
```


y


```wl
OrganizerZoom(ORG_Organizer, orgzWeekOn5)
```


En ambos casos, la función **OrganizerZoom** devuelve la constante *orgzWeekOn5*.

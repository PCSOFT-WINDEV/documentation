


## &lt;Botón Radio&gt;.Count (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el número de opciones encontradas en una control Botón de opción.

Observación: Esta función devuelve el mismo resultado que [Conde](../Proprietes/2510097.md) utilizado en un control Botón de opción. 




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Number of options found in the "RADIO_Options" control
ResNumberOptions = RADIO_Options.Count()

// Identical to: 
// ResNumberOptions = RADIO_Options.Count
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <control Botón de opción>.Conde()`
---

**`<Result>: (Integer)`**

Número de opciones encontradas en el control Botón de opción especificado.

**`<control Botón de opción>: (Control name)`**

Nombre de la control Botón de opción a utilizar.

## Componente:

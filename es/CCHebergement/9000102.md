## Presentación
<a name="presentacion_ELTTEXTE000118"></a>
El Centro de Alojamiento control le permite crear una cuenta implementación. Esta cuenta podrá utilizarse para implementar sitios WEBDEV desde WEBDEV.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Crear un nuevo usuario
<a name="crear_nuevo_usuario_ELTTEXTE000142"></a>
Para crear una cuenta implementación desde el Centro de Alojamiento control:

1. Abra el Centro de Hosting control desde el menú de Inicio de Windows.

2. En el grupo "Usuarios", haga clic en "Nuevo usuario".. El asistente de creación de usuarios se abre. Siga las diferentes etapas.
	Observación: Si es necesario, el Centro de Control de Alojamiento le pedirá que defina un grupo de administración. En este ejemplo, responda "Sí" a la pregunta "¿Desea continuar sin este grupo?". 

3. Usted puede: 

	- Crear una cuenta Windows. Esta cuenta se utilizará para implementar y configurar los sitios. 

	- Utilizar una cuenta Windows existente. 




4. Si está creando una cuenta de Windows, introduzca el nombre de usuario y la contraseña (también puede generar la contraseña. En ese caso, no olvide guardarla).<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P6_Configuration%20CCH%E9bergement%20-%20HC%20N%B0005.jpg&type=thumb)
Pase a la etapa siguiente del asistente.

5. El asistente le pide que seleccione las opciones específicas de WEBDEV associated con la cuenta: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P6_Configuration%20CCH%E9bergement%20-%20HC%20N%B0006.jpg&type=thumb)
 Las opciones (opcionales) son: 

	- Esta cuenta es una cuenta de administrador WEBDEV: En este caso, esta cuenta podrá configurar las otras cuentas de usuario a través del administrador remoto.

	- Dejar cuenta inactiva al crearla: En este caso, esta cuenta no puede utilizarse para desplegar o actualizar sitios. Asimismo, esta cuenta no le permitirá utilizar el administrador remoto. 




6. Pase a la etapa siguiente.

7. El asistente sugiere utilizar una cuenta Windows para ejecutar las aplicaciones. Pase a la etapa siguiente del asistente. 

8. Introduzca la información del usuario. Pase a la etapa siguiente.

9. Los directorios de la cuenta de usuario se definen automáticamente según los datos especificados en la configuración del servidor.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P6_Configuration%20CCH%E9bergement%20-%20HC%20N%B0008.jpg&type=thumb)


10. Pase a la etapa siguiente.

11. Especifique el número máximo de conexiones permitidas para la cuenta y, si es necesario, el número máximo de sitios que se pueden instalar.

12. Seleccione o indique los parámetros del servidor HFSQL Client/Server que deben asociarse a la cuenta de usuario.

13. El siguiente paso de asistente incluye sitios web virtuales. 

	- Si decide crear un nuevo sitio virtual, simplemente introduzca el nombre DNS que apuntará al sitio (el DNS también debe estar configurado).

	- Si decide utilizar un sitio virtual existente, se reemplazará la configuración.
			Observación: el directorio raíz del sitio virtual cambiará. Si los sitios (desarrollados en WEBDEV o no) ya están operando en este mismo servidor Web virtual, pueden verse afectados. Es posible que tenga que restaurar el directorio raíz inicial (c:\\inetpub\\wwwroot\\ de Default).




14. Pase a la etapa siguiente.

15. Esta etapa permite especificar si se debe permitir el acceso y la implementación mediante FTP al usuario. Este paso no es necesario para las implantaciones HTTP. 

16. Pase a la etapa siguiente.

17. El asistente ha terminado. Marque todas las opciones. Puede desmarcar las operaciones que no desea que realice el asistente.

18. Valide el asistente. El servidor está listo para alojar sitios WEBDEV.

## Editar o eliminar una cuenta de usuario
<a name="editar_eliminar_una_cuenta_usuario_ELTTEXTE000166"></a>
Para editar una cuenta implementación desde el Centro de Alojamiento control:

1. Abra el Centro de Hosting control desde el menú de Inicio de Windows.

2. En el grupo "Usuarios", haga clic en "Cuentas de usuario".. 

3. Se muestra la lista de cuentas de usuario. 

	- Puede editar las cuentas de usuario a través del botón "Detalles" de cada cuenta. 

	- Puede eliminar las cuentas de usuario a través del botón "Eliminar" de cada cuenta.

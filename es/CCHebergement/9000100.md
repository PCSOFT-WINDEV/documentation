
## Centro de Control de Alojamiento
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000075"></a>
El Centro de Alojamiento control permite alojar fácilmente los sitios WEBDEV 28. Esta herramienta está destinada principalmente a los proveedores de alojamiento y a los Webmasters. El centro gestiona las cuentas WEBDEV así como la cuenta para el "IIS Web server" (versiones 5.xx, 6.xx, 7.xx y 8.xx) y los derechos para el sistema operativo Windows.

El Centro de Alojamiento control se instala en el ordenador de implementación (CCHebergement.exe) y sólo funciona con Windows y un servidor IIS.

Es muy fácil de usar. Para ello: 

1. [Configurar las características específicas del servidor](../CCHebergement/9000101.md). Esta configuración será utilizada por Default para todos los usuarios. 

2. Crear la [Las cuentas de implementación](../CCHebergement/9000102.md). Esta operación puede automatizarse utilizando una [comando Line](../CCHebergement/9000204.md).




## WDADMINEXE : Información sobre la última actualización disponible
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000080"></a>
La pestaña "Parámetros" contiene información sobre la última actualización disponible para la aplicación:<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=WDAdminExe%20-%20HC%20N%B0002.gif)


- El **ruta de WDADMINEXE** (archivo "WDADMINEXE.EXE") en el servidor.

- El **número de versión actual**. Este número corresponde al número de versión del último ejecutable instalado.

- El **número de la primera versión compatible**. Este número corresponde al número de versión del ejecutable que se encuentra en los equipos del usuario desde los que la actualización es opcional..
	Esta información fue especificada cuando se creó el programa instalación: 

	- en el asistente , "Actualización en vivo: reference ejecutable para verificar versiones". 

	- en el editor instalación WDInst, "Configuración ... Asistente" luego "Red instalación con actualización automatic". 




- La **historia** de la versión actual.. Este historial contiene las modificaciones realizadas por el desarrollador en la versión actual de la aplicación.. Esta historia fue introducida cuando se creó el programa instalación de la versión actual. Corresponde a la ayuda sobre las nuevas funcionalidades de la versión. 
	Se puede escribir el historial:

	- A través de la asistente para crear el programa instalación. Para más detalles, consulte la página web de [Parámetros para actualizar una aplicación instalada en la red](../Editeurs/2028035.md).

	- En el editor de instalación WDInst. Para obtener más información, consulte [Opciones avanzadas para la red instalación](../Editeurs/2028020.md).

## Presentación

La pestaña "Versiones" de WDADMINEXE sirve para gestionar desde el servidor las versiones instaladas en los ordenadores cliente.

## Gestión de versiones
<a name="gestion_versiones_ELTTEXTE000112"></a>
La pestaña "Versiones" permite: 

- Descubra la versión actual. Esta versión es utilizada por los ordenadores cliente que se actualizan.. 

- Ver las versiones disponibles: Estas versiones se instalan en el servidor. Para elegir el tipo de versión que desea mostrar, haga clic en el botón "Mostrar".: 

	- versión activa: versión utilizada actualmente. 

	- versión disponible: versión disponible en el servidor. 

	- versión prohibida: versión disponible en el servidor pero cuyo uso no está permitido. 







Para cada versión seleccionada, tiene la posibilidad de: 

- Ver la ayuda sobre las nuevas características. 

- **Prohibir la versión**: La versión seleccionada no se puede instalar en los ordenadores cliente. 

- **Permitir la versión**: Se usa para permitir una versión que estaba prohibida. 

- **Forzar la versión**: Permite forzar el uso de la versión seleccionada.. Sólo se puede forzar una versión activada. Si la versión es forzada, el ordenador cliente no puede rechazar la actualización durante la próxima comprobación.. 

- **Libera la versión**: Usado para liberar una versión que fue forzada. 

- **Habilitar la versión**: Habilita la versión seleccionada. Una versión instalada es automáticamente habilitada por Default. Sin embargo, se puede instalar sin estar habilitado y se puede habilitar posteriormente a través de WDADMINEXE.. 




**Observación**: La información disponible en esta pestaña se muestra sólo si el historial de versiones se habilitó al crear el programa instalación: 

- En la creación de asistente para instalación, esta activación se realiza en la pestaña "reference instalación - Historial de versiones"..  <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Info_sur_les_versions%20-%20HC%20N%B0001.gif&type=thumb)


## Centro control de ejecutables WINDEV (WDADMINEXE)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000093"></a>
Cuando se instala una aplicación WINDEV en la red, se crea automáticamente un programa utilizado para comprobar remotamente los ejecutables de la aplicación.: WDADMINEXE.

Desde el servidor, WDADMINEXE le permite hacerlo:

- [control la aplicación a distancia](../WDAdminExe/2028026.md): 

	- El administrador de una aplicación de la red puede detener (automatic o cierre forzado) la aplicación para todos los usuarios actuales.

	- El administrador de una aplicación de red puede configurar la frecuencia de la comprobación de las aplicaciones cliente.




- [get información sobre la última actualización disponible](../WDAdminExe/2028044.md): El administrador de una aplicación de red puede encontrar el número de la versión actual y el número de la última versión compatible.. 

- [Conocer las características de los usuarios de la aplicación](../WDAdminExe/2028014.md).

- [Gestionar las versiones instaladas en los ordenadores cliente](../WDAdminExe/2028094.md). 

- [Gestionar las configuraciones en modo Push](../WDAdminExe/2028110.md).

## estreno
<a name="estreno_ELTTEXTE000117"></a>
Para iniciar WDADMINEXE, haga doble clic en el archivo "WDADMINEXE.EXE" del subdirectorio "INSTALL" del directorio de instalación de la aplicación en el servidor.

**Observación**: Las opciones de la aplicación remota control y los detalles sobre la última actualización se encuentran en un archivo "INI".. Este archivo se llama "WDUPDATE.NET" de Default. Este archivo contiene las referencias de las diferentes aplicaciones a comprobar y, para cada una de ellas, las características de la comprobación a realizar (para más detalles, véase [Estructura del archivo control](../WDLang1/3013005.md)).

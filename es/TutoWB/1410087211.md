## Instalar un servidor HFSQL local
<a name="instalar_servidor_hfsql_local_ELTTEXTE000352"></a>
La primera operación a realizar consiste en instalar un servidor HFSQL.

Este servidor puede instalarse en el equipo local de desarrollo (es lo que vamos a hacer a continuación). En la fase de implementación, el servidor se puede instalar en un equipo específico o directamente en el servidor que aloja el sitio.

El programa de instalación del servidor HFSQL está incluido en los medios de instalación de WINDEV, WEBDEV y WINDEV Mobile. Si no dispone de este medio, la instalación del servidor HFSQL también está disponible en nuestro sitio web ([www.windev.es](https://www.windev.com)).

Para instalar el servidor HFSQL en el equipo local:

1. Inicie el programa de instalación de WINDEV, WEBDEV y WINDEV Mobile.

2. Seleccione "Instalar HFSQL Client/Server".

3. Enseguida, seleccione "Instalar o actualizar un servidor HFSQL".

4. Acepte los términos de la licencia.

5. Seleccione la plataforma (opción "Para Windows en este equipo").

6. Si ya hay servidores HFSQL instalados en el equipo actual, seleccione "Instalar un nuevo servidor".

7. Seleccione el directorio de instalación y especifique el nombre del servidor y el puerto. De forma predeterminada, se utiliza el puerto 4900.
	|   |   |   |
| --- | --- | --- |
| ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=note.png) | Nota | No olvide abrir este puerto en el firewall para conectarse al servidor HFSQL desde otro equipo. |





8. Especifique las credenciales de inicio de sesión del usuario con privilegios de administrador que se creará en el servidor HFSQL. De forma predeterminada, este usuario es "Admin" y no tiene contraseña. 
	|   |   |   |
| --- | --- | --- |
| ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=note.png) | Nota | Por razones de seguridad, recuerde cambiar la contraseña del administrador. |





9. Si el Centro de Control HFSQL no está instalado o no se puede acceder a él desde su equipo, seleccione la opción "Instalar el Centro de Control HFSQL". 
	|   |   |   |
| --- | --- | --- |
| ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=avertissement.png) | ¡Atención! | El Centro de Control HFSQL es necesario para gestionar la base de datos HFSQL Client/Server. |





10. El asistente le pide que configure el envío de notificaciones para identificar los problemas del servidor en tiempo real. En este caso, vaya a la etapa siguiente e indique que los ajustes se definirán "Más tarde". 

11. La asistente permite habilitar la autenticación de Active Directory. Active esta opción si es necesario. 

12. Puede ayudarnos a mejorar permitiéndonos recopilar información sobre el uso de nuestros productos. Esta recopilación de datos opcional y anónima permite que PC SOFT mejore sus productos. 

13. La instalación se ha completado. De forma predeterminada (si no ha cambiado la configuración del administrador), para conectarse al servidor en modo administrador, debe utilizar el usuario "Admin" sin contraseña.

## Crear un sitio con una base de datos HFSQL Client/Server
<a name="crear_sitio_con_una_base_datos_hfsql_clientserver_ELTTEXTE000430"></a>
Crear un sitio WEBDEV con una base de datos HFSQL Client/Server es muy sencillo. Siga los siguientes pasos:

1. Cree el proyecto con una nueva base de datos.

2. Cree el análisis especificando que las bases de datos utilizadas por el proyecto serán bases de datos "HFSQL Client/Server".

3. Especifique las características de conexión con el servidor HFSQL Client/Server que se utilizará.

4. Cuando cree un archivo de datos en el análisis, indique que el archivo de datos está en modo Client/Server y especifique la conexión utilizada.


|   |   |   |
| --- | --- | --- |
| ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=note.png) | Nota | También puede definir la conexión con el servidor HFSQL mediante programación. Para obtener más información, consulte la ayuda en línea: "[HDescribeConnection](../WDLang4/3044205.md)". |

## Adaptar un sitio para utilizar una base de datos HFSQL Client/Server
<a name="adaptar_sitio_para_utilizar_una_base_datos_hfsql_clientserver_ELTTEXTE000472"></a>


### Presentación
<a name="presentacion_ELTPARAGRAPHE000100"></a>

Pasar una base de datos HFSQL Classic a Client/Server es la operación más común.

WEBDEV incluye varias opciones para realizar esta operación:

- desde el editor de análisis.

- desde el Centro de Control HFSQL.


Para entender mejor las diferentes etapas, vamos a cambiar el sitio que creamos en la parte 4 del tutorial al modo Client/Server usando el editor de análisis.
<a name="NOTE5_2"></a>


### Adaptación del ejemplo
<a name="adaptacion_del_ejemplo_ELTPARAGRAPHE000116"></a>



- Vamos a adaptar el proyecto "Full WEBDEV Site" para utilizar una base de datos Cliente/Servidor. 






- Si es necesario, abra el proyecto en el que trabajó en las lecciones anteriores. 

	1. Vaya a la página de inicio de WEBDEV (Ctrl +&lt;).

	2. En la página de inicio, haga clic en "Tutorial", luego en "Parts 11 and 12", haga doble clic en "Full WEBDEV Site - Exercise".

	3. Un cuadro de diálogo le pide que abra el proyecto en el que trabajó en las lecciones anteriores. Puede abrir la copia local o el proyecto original. Seleccione "Abrir la copia local". 





|   |   |   |
| --- | --- | --- |
| ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=exemple-WB.png) | Corregido | Hay un proyecto completado disponible. Puede utilizar este proyecto para realizar los diferentes ejercicios de esta lección. Para abrir el proyecto completado, vaya a la página de inicio y haga clic en "Tutorial", luego en "Parts 11 and 12", haga doble clic en "Full WEBDEV Site - Answers". |







- Para adaptar el proyecto:

	1. Cargue el análisis del proyecto en el editor de análisis: haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ICO_Analyse_WM_GAF.jpg) en los botones de acceso rápido del menú de WEBDEV. El editor de análisis se abre.

	2. En la cinta de opciones, en la pestaña "Análisis", en el grupo "Conexión", haga clic en "Nueva conexión". Un asistente que permite crear una nueva conexión se abre.

	3. Seleccione el tipo de conexión que desea crear: "HFSQL Client/Server". <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P11_Passage%20analyse%20en%20CS%20WB%20-%20HC%20N%B0001.jpg&type=thumb)
Pase a la etapa siguiente.

	4. En las siguientes etapas, especifique: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P11_Passage%20analyse%20en%20CS%20WB%20-%20HC%20N%B0002.jpg&type=thumb)


		- el nombre del servidor en el que se instaló el servidor HFSQL Client/Server (localhost si se instaló en su equipo, por ejemplo), y el número de puerto. 

- el nombre de usuario y la contraseña (para utilizar el administrador, especifique las credenciales de inicio de sesión definidas cuando se instaló el servidor HFSQL). 

- el nombre de la base de datos ("Full_WEBDEV_Site" en nuestro ejemplo). 

5. Defina el nombre de la conexión (conserve el nombre predeterminado).

6. Vaya a la etapa siguiente y valide. La conexión a la base de datos se crea automáticamente. El asistente le pide que asocie los diferentes archivos de datos del análisis a la conexión que acaba de crear.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P11_Passage%20analyse%20en%20CS%20WB%20-%20HC%20N%B0003.jpg)

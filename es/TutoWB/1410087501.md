## Presentación
<a name="presentacion_ELTTEXTE000158"></a>
Existen dos programas para administrar sitios WEBDEV y Webservices:

- el administrador WEBDEV.

- el administrador WEBDEV remoto.




<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Administrador WEBDEV local
<a name="administrador_webdev_local_ELTTEXTE000182"></a>
El administrador WEBDEV es una aplicación instalada con WEBDEV Application Server.|   |   |   |
| --- | --- | --- |
| ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=note.png) | Nota | El administrador WEBDEV solo puede utilizarse si puede acceder directamente al servidor web en el que está implementado el sitio. |



Esta aplicación incluye siete partes distribuidas en siete pestañas:

- Conexiones: Muestra la lista de usuarios conectados a los sitios y permite desconectarlos. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P6_Administrateur%20WB%20-%20HC%20N%B0001.jpg&type=thumb)


- Sitios: Muestra los sitios implementados en el servidor, así como sus parámetros: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P6_Administrateur%20WB%20-%20HC%20N%B0002.jpg&type=thumb)


- Webservices: Muestra los servicios web implementados en el servidor, así como sus parámetros.

- WebSocket: Muestra los servidores WebSocket instalados en el servidor. 

- Configuración: Permite administrar la configuración del servidor. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P6_Administrateur%20WB%20-%20HC%20N%B0004.jpg&type=thumb)


- Avanzado: Permite definir ajustes adicionales del servidor como trabajos de impresión, cola de correo electrónico, etc, ... <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P6_Administrateur%20WB%20-%20HC%20N%B0005.jpg&type=thumb)


- Instalaciones: Permite bloquear el servidor para realizar el mantenimiento, permitir o no permitir instalaciones remotas y configurar el registro de las instalaciones. El registro muestra los elementos instalados en un sitio. Este registro es especialmente útil para ponerse en contacto con nuestro Soporte Técnico si encuentra errores durante la implementación. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P6_Administrateur%20WB%20-%20HC%20N%B0006.jpg&type=thumb)


- Registros: Permite ver los registros de sitios, Webservices o servidores WebSocket durante un período de tiempo determinado. Los elementos que se muestran son aquellos en los que se produjeron errores durante el período de tiempo especificado y para los que están habilitados los registros.

## Administrador WEBDEV remoto
<a name="administrador_webdev_remoto_ELTTEXTE000224"></a>
El administrador remoto es un sitio WEBDEV que ofrece funcionalidades similares a las del administrador WEBDEV pero que puede utilizarse para administrar sitios WEBDEV de forma remota.|   |   |   |
| --- | --- | --- |
| ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=avertissement.png) | ¡Atención! | Si utiliza el administrador remoto, es importante definir una conexión segura. Se recomienda acceder al administrador a través de una conexión HTTPS y establecer contraseñas seguras. |



Observación: Esta herramienta no está disponible con "WEBDEV Application Server - 10 conexiones".

Cuando publique un sitio dinámico en una plataforma de alojamiento, podrá utilizar el administrador remoto para actualizar los parámetros del sitio (si el proveedor de alojamiento permite la administración remota).



- Para iniciar WDAdminWeb:

	1. Abra un navegador.

	2. Introduzca la siguiente URL en la barra de direcciones (con las mismas mayúsculas y minúsculas):
			
		```txt
		http://computer/WDAdminWeb
		```

			donde "computer" es el nombre del servidor de implementación.

	3. Introduzca el nombre de usuario y la contraseña para iniciar sesión en el administrador WEBDEV remoto. 

	4. El administrador inicia.

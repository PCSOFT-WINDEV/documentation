## Controles específicos para los layouts
<a name="controles_especificos_para_los_layouts_ELTTEXTE000224"></a>
WEBDEV proporciona un control específico para los layouts: el control Barra de navegación. 

El control Barra de navegación permite crear un área de menú que puede adaptarse fácilmente al tamaño del navegador. Esta área puede contener todo tipo de controles.



- Veamos cómo funciona el control Barra de navegación.

	1. Abra WEBDEV 28. 

	2. Vaya a la página de inicio de WEBDEV si es necesario (Ctrl + &lt;). 

	3. En la página de inicio, haga clic en "Tutorial", luego en "Part 8 - Layouts", haga doble clic en "Layouts - Exercise".

	4. Abra la página PAGE_NavigationBar.







- El control Barra de navegación es el control de la parte superior que contiene el menú. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P8_agencement_Barre_navigation%20-%20HC%20N%B0001.jpg&type=thumb)
El control Barra de navegación está dividido en 3 secciones:

	- A la izquierda, el logotipo del sitio.

	- En el medio, el menú del sitio.

	- A la derecha, un cuadro de búsqueda con un ícono de búsqueda.









- Vamos a crear un layout: 

	1. En la pestaña "Página", en el grupo "Layouts", despliegue "Layouts" y seleccione "Agregar un layout a partir de la vista activa". 

	2. En la ventana que aparece, se muestra un layout adaptado a teléfonos y tabletas. Valide la ventana. 

	3. En nuestro ejemplo se crearon dos layouts. Estos layouts se muestran como miniaturas, a la derecha de la página "PAGE_NavigationBar". El layout Móvil se selecciona automáticamente y se muestra en el editor.  <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P8_agencement_Barre_navigation%20-%20HC%20N%B0002.jpg&type=thumb)

La disociación automática no está disponible para este control. Se deben configurar algunas opciones específicas para realizar la disociación. |







- Primero, vamos a configurar el control Barra de navegación para que se muestre en los dispositivos móviles. 

	1. Si es necesario, seleccione el control Barra de navegación en el layout Móvil. 

	2. Abra la ventana de descripción del control Barra de navegación (Alt + Entrar). 

	3. En la pestaña "UI", en "Modo de visualización", haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P8_agencement_Barre_navigation%20-%20HC%20N%B0003%201.jpg) para disociar el modo de visualización por layout.  <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P8_agencement_Barre_navigation%20-%20HC%20N%B0003.jpg&type=thumb)


	4. En la ventana que aparece, marque la opción "Disociar valor por layout" y, a continuación, en "Móvil", seleccione "Compacto".<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P8_agencement_Barre_navigation%20-%20HC%20N%B0004.jpg)
Confirme los cambios. 

5. Valide la ventana de descripción del control. 

6. El control Barra de navegación aparece en modo compacto en el layout Móvil. 




- Ahora veamos cómo se ve el control Barra de navegación en los diferentes layouts. 

	1. Abra el layout "Escritorio": la barra de navegación se muestra completa. 

	2. Abra el layout "Móvil": la barra de navegación se muestra en modo "Compacto".

	3. En el layout Móvil, seleccione el control Barra de navegación: el área central está sombreada.  <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P8_agencement_Barre_navigation%20-%20HC%20N%B0005.jpg&type=thumb)








- En el layout Móvil, el area central del control Barra de navegación está oculta.
	El menú de "hamburguesa" despliega una ventana emergente con los controles que suelen aparecer en el área central. Para personalizar la ventana emergente, simplemente haga clic en el panel central. A continuación, puede realizar los cambios deseados en la ventana.




- Analicemos el contenido del área central:

	1. Haga clic en el área sombreada del centro. La ventana emergente aparece en modo de edición.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P8_agencement_Barre_navigation%20-%20HC%20N%B0007.jpg&type=thumb)
El contorno amarillo indica que el control está en modo de edición. 

	2. El menú del sitio aparece. Este menú está dispuesto horizontalmente. Para que se muestre verticalmente, es necesario disociar la orientación del menú: 

		- Abra la ventana de descripción del menú (Alt + Entrar). 

		- En la pestaña "General", haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P8_agencement_Barre_navigation%20-%20HC%20N%B0003%201.jpg) para disociar la orientación del menú.  <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P8_agencement_Barre_navigation%20-%20HC%20N%B0008.jpg&type=thumb)


		- En la ventana que aparece: 

	- Seleccione la opción "Disociar valor por layout". 

	- Para el layout Móvil, seleccione "Vertical para móviles".  <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P8_agencement_Barre_navigation%20-%20HC%20N%B0009.jpg)

3. Valide la ventana de descripción.  <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P8_agencement_Barre_navigation%20-%20HC%20N%B0010.jpg&type=thumb)






- La última modificación consiste en cambiar el tamaño de las diferentes áreas de la barra de navegación para que se muestren todos los elementos. 

	1. Seleccione la barra de navegación. 
			Utilice los cuadrados rojos para cambiar el tamaño de las áreas del control Barra de navegación. 

	2. Reduzca el área del menú y amplíe el área de búsqueda. Obtendrá el siguiente resultado:  <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P8_agencement_Barre_navigation_2%20-%20HC%20N%B0002.jpg&type=thumb)








- Guarde la página (Ctrl + S) y pruébela (haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ICO_Go_Page_WB_GAF.jpg) en los botones de acceso rápido).

	1. La página se muestra en el navegador.  <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P8_agencement_Barre_navigation_2%20-%20HC%20N%B0003.jpg&type=thumb)


	2. Redimensione el navegador: el layout Móvil se muestra en el navegador.  <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P8_agencement_Barre_navigation_2%20-%20HC%20N%B0004.jpg&type=thumb)




- Cierre el navegador.




- Ahora, podemos probar el sitio en un teléfono o tableta: 

	1. Despliegue la opción ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ICO_Go_Page_WB_GAF.jpg) en el área de acceso rápido y seleccione "Probar desde un teléfono (acceso mediante un código QR)".  <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P7_Ancrage%20-%20Agencement%20bis%20-%20HC%20N%B0001.jpg)


	2. Una ventana aparece con un código QR que se puede escanear desde el teléfono. 

3. Una vez escaneado el código QR, el sitio se muestra en el teléfono. 

4. El layout "Móvil" aparece automáticamente.

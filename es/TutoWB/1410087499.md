### Recordatorio: Modo AWP
<a name="recordatorio_modo_awp_ELTPARAGRAPHE000044"></a>

Al inicio de este tutorial, presentamos las diferencias entre un sitio WEBDEV en modo Sesión y un sitio WEBDEV en modo AWP.

Solo el modo AWP permite tener direcciones URL de páginas independientes, fijas y directamente direccionables.

Para optimizar el sitio web, asegúrese de que esté en modo AWP.
<a name="NOTE2_3"></a>


### Métodos disponibles
<a name="metodos_disponibles_ELTPARAGRAPHE000055"></a>

WEBDEV incluye varias herramientas para mejorar el posicionamiento de páginas AWP en los motores de búsqueda:

- Un asistente de SEO.

- Las propiedades de la página.

- El mapa de navegación.


A continuación, presentaremos estos diferentes métodos en detalle.

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Asistente de SEO
<a name="asistente_seo_ELTTEXTE000345"></a>
WEBDEV incluye un asistente de SEO.



- Para abrir el asistente, en la pestaña "Proyecto", en el grupo "Web", despliegue "SEO" y seleccione "Asistente de SEO".  <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P5_R%E9f%E9rencement%20-%20HC%20N%B0001.jpg&type=thumb)





El asistente muestra algunos consejos para mejorar el SEO. Los consejos más importantes son los siguientes:

- **Títulos de las páginas**: Los títulos de las páginas deben reflejar el contenido y contener palabras clave importantes. El título de la página se define en la ventana de descripción de la página, para cada idioma.

- **Palabras clave de las páginas**: Le recomendamos asociar palabras clave a las páginas. Para definir las palabras clave:

	- Abra la ventana de descripción de la página "PAGE_List_of_new_products": abra el menú contextual y seleccione "Descripción".

	- Vaya a la pestaña "Detalles".

	- En la sección "SEO", haga clic en "Editar expresiones y palabras clave". <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P5_R%E9f%E9rencement%20-%20HC%20N%B0002.jpg&type=thumb)
La ventana que aparece permite introducir las palabras clave deseadas. Cierre las diferentes ventanas de descripción. 




- **Texto alternativo**: Los textos alternativos reemplazan a las imágenes cuando no se muestran (por ejemplo, cuando el usuario pide explícitamente que no se muestren las imágenes, si aún no se han cargado, o cuando el usuario utiliza un navegador en modo de texto o un lector de pantalla).
	Los textos alternativos describen la imagen y son indexados en los motores de búsqueda. Por lo tanto, deben ser elegidos con gran atención.
	El texto alternativo de una imagen se define en la ventana de descripción del control Imagen, en la pestaña "Ayuda".  <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P5_R%E9f%E9rencement%20-%20HC%20N%B0005%20bak.jpg&type=thumb)





- Si es necesario, cierre las ventanas de descripción y el asistente de SEO.

## Propiedades de la página
<a name="propiedades_pagina_ELTTEXTE000369"></a>
Como ya hemos visto, el asistente de SEO muestra los cambios que se pueden realizar en la ventana de descripción de la página:

- Título de la página,

- Palabras clave de la página, etc.




Otras propiedades que influyen en el SEO se pueden definir en la ventana de descripción de la página:

- Frecuencia de actualización,

- Prioridad en el sitio.



<a name="NOTE4_2"></a>


### Frecuencia de actualización
<a name="frecuencia_actualizacion_ELTPARAGRAPHE000130"></a>

La frecuencia de actualización le indica al motor de búsqueda con qué frecuencia se debe realizar la indexación de la página.  <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P5_R%E9f%E9rencement%20-%20HC%20N%B0002%201.jpg&type=thumb)


Indique la frecuencia de actualización más precisa:

- Si la página cambia con más frecuencia de la que se especifica, es posible que no se realice la indexación de algunas versiones.

- Si la página cambia con menos frecuencia, el motor de búsqueda podría sobrecargar el servidor y afectar la indexación.



<a name="NOTE4_3"></a>


### Prioridad en el sitio
<a name="prioridad_sitio_ELTPARAGRAPHE000143"></a>

Este criterio permite al motor de búsqueda saber qué página debe mostrar primero al usuario cuando varias páginas del sitio coinciden con una búsqueda.

<a name="NOTE5"></a>
<a name="NOTE5_1"></a>


## Mapa de navegación
<a name="mapa_navegacion_ELTTEXTE000405"></a>
El mapa de navegación define la estructura del sitio. En SEO, puede utilizarse para:

- mostrar el control Mapa del sitio para guiar a los usuarios dentro del sitio.

- generar el archivo sitemap.xml.




- Para definir el mapa de navegación: 

	1. En la cinta de opciones, vaya a la pestaña "Proyecto", grupo "Web" y seleccione "Mapa de navegación".

	2. Aparece una ventana que le permite construir la estructura jerárquica del sitio. Puede crear una estructura jerárquica utilizando el modo de construcción automática que detecta los enlaces entre las páginas. El mapa de navegación se puede configurar más adelante.







- WEBDEV construye automáticamente el archivo del mapa del sitio a partir del mapa de navegación. Este archivo se genera en el directorio _WEB del sitio con el nombre _sitemap.xml.




El archivo sitemap.xml debe enviarse a los motores de búsqueda para que puedan indexar el contenido del sitio.

<a name="NOTE6"></a>
<a name="NOTE6_1"></a>


## Nombres físicos de las páginas
<a name="nombres_fisicos_las_paginas_ELTTEXTE000429"></a>
Para mejorar aún más el SEO, puede definir el nombre físico de las páginas AWP para que sea diferente en cada idioma (y diferente también del nombre lógico).

Por ejemplo, una página que muestra un tablero de ventas cuyo nombre lógico (es decir, el nombre utilizado en el código) es PAGE_TableroDeVentas puede generarse en español en el archivo 'tablero-de-ventas.awp' y en inglés en el archivo 'sales-dashboard.awp'.

- Para definir el nombre de generación de la página:

	1. Abra la ventana de descripción de "PAGE_List_of_new_products".

	2. En la pestaña "General", haga clic en el botón "Idioma" (![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P5_MultiLangue%20-%20HC%20N%B0006%201.jpg)) a la derecha de la opción "Archivo generado".

	3. Una ventana que permite configurar el nombre de las páginas generadas se abre:  <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P5_R%E9f%E9rencement%20-%20HC%20N%B0004.jpg&type=thumb)


	4. Puede cambiar el nombre de la página y validar.

## Presentación
<a name="presentacion_ELTTEXTE000295"></a>
WEBDEV es un Entorno de Desarrollo Integrado de DevOps destinado a crear sitios de Internet e Intranet en diversos campos: comercio electrónico, multimedia, etc. 

Los sitios creados pueden dar acceso a información almacenada en las bases de datos.

WEBDEV permite crear:

- sitios de Internet e Intranet estáticos. Estos sitios gestionan datos que no cambian (sitio de presentación de la empresa, etc.).

- sitios de Internet e Intranet dinámicos que gestionan datos. Los sitios WEBDEV acceden a todas las bases de datos, relacionales o no, disponibles en el mercado.


A lo largo de este tutorial, aprenderá a crear sus sitios (con o sin base de datos) y a enriquecerlos utilizando todas las funcionalidades disponibles en WEBDEV.

## Entorno de desarrollo
<a name="entorno_desarrollo_ELTTEXTE000343"></a>


### Editor
<a name="editor_ELTPARAGRAPHE000075"></a>

El entorno de desarrollo de WEBDEV contiene una interfaz específica y varios editores que permiten crear los diferentes elementos de sus aplicaciones.

Por ejemplo, el editor de páginas permite crear páginas, el editor de reportes permite crear reportes, etc.

- Para descubrir WEBDEV, abriremos la página que contiene la lista de productos: 

	1. Presione Ctrl + E. 

	2. En la ventana que se muestra, escriba el nombre de la página que desea abrir: PAGE_List_of_products. 

	3. Valide. La página aparece en el editor. 







Todos los editores utilizan el mismo entorno: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P1_Environnement%20WB%20Complet.jpg&type=thumb)


1. **Menú de los editores** que se muestra como una cinta de opciones (veremos cómo se utiliza en el siguiente párrafo).

2. **Editor actual** (en este caso, editor de páginas). El editor muestra el elemento que se está creando o modificando en modo WYSIWYG (What You See Is What You Get).

3. **Paneles**. La interfaz de WEBDEV contiene varios paneles horizontales y verticales que le permiten acceder rápidamente a diferentes tipos de información.
	Por ejemplo:

	- El panel "Explorador de proyectos" (a la izquierda) muestra una lista de todos los elementos del proyecto por categoría.

	- El panel "Buscar - Reemplazar" (que se muestra en la parte inferior) permite realizar búsquedas rápidamente en el proyecto.


Los paneles se pueden ocultar pulsando Ctrl + W.

4. **Pestañas de documentos abiertos**. Estas pestañas permiten ver rápidamente todos los elementos abiertos. Al hacer clic en un elemento, este se muestra en el editor correspondiente.

### Cinta de opciones en detalle
<a name="cinta_opciones_detalle_ELTPARAGRAPHE000104"></a>

Las opciones de WEBDEV están disponibles en la cinta de opciones. La cinta contiene pestañas en las que se agrupan las diferentes opciones de los editores.

**Elementos de la cinta de opciones**
La cinta de opciones incluye tres áreas: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P1_Environnement%20WB%20-%20HC%20N%B0003.jpg&type=thumb)


- el área de botones, a la izquierda (1).

- el área de pestañas, en la parte superior (2).

- el área de opciones (3).


Veamos estas áreas en detalle.

**Área de botones (1)**



|   |   |
| --- | --- |
| <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P1_Environnement%20WB%20-%20HC%20N%B0003%202.jpg)<br> | El área de botones agrupa los **botones de acceso rápido**. Estos botones permiten realizar las operaciones más frecuentes, comunes a todos los editores: guardar, abrir, crear, etc.<br><br>Puede abrir la ventana "Acerca de" desde el logotipo del IDE. |

**Área de pestañas (2)**<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P1_Environnement%20WB%20-%20HC%20N%B0003%203.jpg&type=thumb)
Las diferentes pestañas de la cinta permiten acceder a las opciones de los editores del proyecto actual. Existen varios tipos de pestañas:

- la pestaña actual: El nombre de la pestaña aparece en negrita y se muestra una línea azul debajo del nombre de la pestaña

- las pestañas contextuales, específicas del elemento actual: El nombre de la pestaña se muestra en azul.

- las pestañas disponibles: El nombre de la pestaña se muestra en blanco (o en negro en el tema claro).




**Área de opciones (3)**<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P1_Environnement%20WB%20-%20HC%20N%B0003%201.jpg&type=thumb)
Las opciones de la cinta cambian dependiendo de la pestaña seleccionada. Existen varios tipos de opciones:

- Opciones a seleccionar,

- Botones para hacer clic, 

- Botones desplegables que muestran otras opciones. Existen dos tipos de botones desplegables:

	- botones desplegables para desplegar un menú, 

	- botones divididos que permiten desplegar un menú (haciendo clic en la flecha), o realizar una acción predeterminada (haciendo clic en el ícono del botón).





Las opciones están organizadas por grupos. Cada grupo de opciones tiene un nombre y un iniciador de cuadro de diálogo ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ICO_Regroupement_GAF.jpg). Este botón permite realizar una acción específica según el grupo actual: abrir la descripción del elemento actual, abrir la ayuda, etc. 

En este tutorial, para indicar una opción de menú, hablaremos de pestañas y grupos.. Por ejemplo: "Para abrir la ayuda, en la pestaña "Inicio", en el grupo "Ayuda en línea", haga clic en "Ayuda"."

### Opciones del entorno
<a name="opciones_del_entorno_ELTPARAGRAPHE000167"></a>

Puede cambiar varias opciones del entorno: 

- Tema: De forma predeterminada, el entorno utiliza un tema oscuro. Sin embargo, hay varios temas disponibles: 

	- Tema claro, cinta de opciones gris. En este modo, la barra de menú no tiene colores: está en gris. 

	- Tema gris. En este modo, el entorno y las ventanas de la interfaz se muestran sobre un fondo gris claro. 

	- Tema oscuro. En este modo, el entorno y las ventanas de la interfaz se muestran sobre un fondo negro o gris oscuro. 




- Idioma: De forma predeterminada, el entorno se muestra en el idioma elegido durante la instalación. El entorno puede mostrarse en español, inglés o francés.

- Panel: Algunos paneles se muestran de forma predeterminada en el entorno ("Explorador de proyectos", etc.). Puede elegir qué paneles se muestran y fijar su posición.




Todas estas opciones se encuentran en las opciones del entorno. Para cambiar estas opciones, vaya a la pestaña "Inicio", grupo "Entorno", despliegue "Opciones" y seleccione "Opciones generales de WEBDEV". Puede cambiar varias opciones del entorno en las diferentes pestañas de la ventana que aparece. 
Tenga en cuenta que para cambiar ciertas opciones puede ser necesario reiniciar WEBDEV.


## Robot de surveillance : Configuration des paramètres du contre-robot
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000153"></a>
Depuis le moniteur, il est possible de modifier les options de configuration :

- du robot,

- du contre-robot,

- du moniteur.




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Configuration des paramètres du contre-robot
<a name="configuration_des_parametres_contrerobot_ELTTEXTE000177"></a>
Les paramètres du contre-robot correspondent aux paramètres indiqués dans l'assistant de configuration du contre-robot.

**Pour afficher les paramètres du contre-robot** : 

1. Dans le menu du moniteur, dans le groupe Paramétrage", cliquez sur "Paramétrer" puis sur "Paramètres du contre-robot".

2. La fenêtre qui s'affiche reprend les caractéristiques du contre-robot :

	- Nom, Machine.

	- Options d'alerte.









<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Options d'alerte
<a name="options_alerte_ELTTEXTE000201"></a>


### Alerte par WDBal
<a name="alerte_par_wdbal_ELTPARAGRAPHE000038"></a>L'onglet "WDBal" permet de paramétrer l'envoi de rapport d'erreurs par la messagerie PC SOFT nommée WDBal. Pour utiliser ce mode de message d'alerte, la base de données des Centres de Contrôle doit être accessible depuis le poste où le contre-robot est installé.

La section "Configuration de WDBal" permet de définir le mode d'accès aux données des Centres de Contrôle :

- En mode partage réseau : répertoire des fichiers de données.

- En mode HFSQL Client/Serveur : 

	- serveur utilisé, port et base de données.

	- utilisateur et mot de passe. 







La section "Configuration des intervenants" permet de définir le ou les destinataires par défaut du message envoyé par la messagerie (bouton "Définir les destinataires"). Les destinataires proposés sont les utilisateurs ayant activé un envoi par WDBal. 


<a name="NOTE3_2"></a>


### Alerte par email
<a name="alerte_par_email_ELTPARAGRAPHE000056"></a>L'onglet "Email" permet de paramétrer l'envoi de rapport d'erreurs par email. Pour utiliser ce mode de message d'alerte, un serveur SMTP doit être accessible depuis le poste où le contre-robot est installé.

La section "Configuration de l'envoi d'emails" permet de définir les caractéristiques du serveur SMTP utilisé :

- Serveur et port SMTP

- Utilisateur et mot de passe associé pour utiliser le serveur SMTP.

- Type de sécurité utilisé. 




La section "Configuration des intervenants" permet de définir :

- L'expéditeur du message, c'est-à-dire l'adresse email utilisée comme expéditeur.

- Les destinataires par défaut du message envoyé par la messagerie. Il suffit d'indiquer les adresses email des destinataires.





<a name="NOTE3_3"></a>


### Utilisation d'un programme tiers
<a name="utilisation_programme_tiers_ELTPARAGRAPHE000075"></a>L'onglet "Programme tiers" permet de paramétrer l'envoi de rapport d'erreurs par un programme spécifique. Pour utiliser ce mode de message d'alerte, indiquez le chemin de l'exécutable voulu. Cet exécutable doit être accessible depuis le poste d'installation du contre-robot. 

Dans ce mode d'envoi, le rapport d'erreur est un fichier XML. Le chemin du rapport d'erreur sera passé en ligne de commande au programme tiers et devra être traité par ce programme.

Le format du fichier XML est le suivant :


```xml
<Test>
<NumeroTest>%1</NumeroTest>
<NomTest>%2</NomTest>
<CategorieTest>%3</CategorieTest>
<PrioriteTest>%4</PrioriteTest>
<SyntaxeTest>
%5
</SyntaxeTest>
<SolutionTest>
%6
</SolutionTest>
<DateErreur>%7</DateErreur>
<HeureErreur>%8</HeureErreur>
<MessageErreurPersonnel>
%9
</MessageErreurPersonnel>
<MessageErreur>
%10
</MessageErreur>
</Test>
<Test>
...
</Test>
```


Dans ce code :

- %1 = Numéro identifiant le test

- %2 = Nom du test

- %3 = Nom de la catégorie du test

- %4 = Priorité du test

- %5 = Syntaxe du test (paramètres associées)

- %6 = Solution proposée pour le test

- %7 = Date de l'erreur

- %8 = Heure de l'erreur

- %9 = Message d'erreur renseigné dans la fiche du test

- %10 = Message d'erreur renvoyé par le robot





<a name="NOTE3_4"></a>


### Paramétrage du rapport
<a name="parametrage_rapport_ELTPARAGRAPHE000102"></a>L'onglet "Paramétrage du rapport" permet de paramétrer le sujet du message envoyé (par email ou par WDBal).

La section "Objet de l'email envoyé" permet de construire le titre du message. Des macros permettent d'obtenir des informations spécifiques (nom du contre-robot, ...).

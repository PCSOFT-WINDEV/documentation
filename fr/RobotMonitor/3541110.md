
## Contrôle : Connectivité bases tierces (par ODBC)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000076"></a>
Le robot de surveillance va tenter de se connecter à une base de données tierce (MySQL, SQL Server, ...) par ODBC.

Le contrôle sera :

- réussi si la connexion est effective.

- en échec dans le cas contraire.






<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Paramètres spécifiques au contrôle de la connectivité des bases tierces
<a name="parametres_specifiques_controle_connectivite_des_bases_tierces_ELTTEXTE000100"></a>
Il est nécessaire d'indiquer le nom de la source ODBC à tester. Cette source ODBC doit être définie sur le poste d'installation du robot de surveillance.

**Remarque** : Pour effectuer un test en local, depuis le moniteur, il est nécessaire de définir la même source ODBC sur ce poste.

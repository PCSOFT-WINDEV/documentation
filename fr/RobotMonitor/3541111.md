
## Contrôle : Connectivité FTP
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000101"></a>
Le robot de surveillance va tenter de se connecter au serveur FTP avec les paramètres du serveur indiqués. Il est possible de tester l'existence d'un fichier sur le serveur FTP, ainsi que le nombre d'utilisateurs connectés. 

Le contrôle sera :

- réussi si la connexion est effective (et si le fichier indiqué existe).

- en échec dans le cas contraire.






<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Paramètres spécifiques au contrôle de la connectivité FTP
<a name="parametres_specifiques_controle_connectivite_ftp_ELTTEXTE000125"></a>


### Informations de connexion au serveur FTP
<a name="informations_connexion_serveur_ftp_ELTPARAGRAPHE000024"></a>Il est nécessaire de préciser les caractéristiques de connexion au serveur FTP :

- l'adresse du serveur FTP.

- le numéro de port du serveur et le mode de connexion (passif ou non).

- l'utilisateur et le mot de passe associé.





<a name="NOTE2_2"></a>


### Informations sur le fichier à contrôler
<a name="informations_sur_fichier_controler_ELTPARAGRAPHE000036"></a>Il est possible de tester la présence d'un fichier sur le serveur ainsi que son téléchargement. Les informations à donner sont :

- le nom complet du fichier sur le serveur FTP.

- si le fichier doit être téléchargé ou non.  





<a name="NOTE2_3"></a>


### Contrôle du nombre d'utilisateurs connectés
<a name="controle_nombre_utilisateurs_connectes_ELTPARAGRAPHE000047"></a>Il est possible de tester le nombre d'utilisateurs connectés. Les informations à donner sont :

- l'adresse de la page retournant la liste des utilisateurs.

- le nombre d'utilisateurs maximum.

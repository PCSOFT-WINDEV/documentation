
## Robot de surveillance : Créer un contrôle
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000162"></a>
Le moniteur du robot de surveillance permet de créer les différents contrôles à effectuer par le robot de surveillance.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000186"></a>
Les catégories permettent de regrouper les contrôles par thèmes. Chaque contrôle est associé à une catégorie.

Par défaut, si un contre-robot a été installé, une catégorie "Contre-robot" est créée. Cette catégorie contient un test de surveillance du contre-robot.


<a name="NOTE2_2"></a>


### Créer une catégorie
<a name="creer_une_categorie_ELTPARAGRAPHE000027"></a>Pour créer une catégorie :

1. Dans le menu, dans le groupe "Catégories et contrôles", cliquez sur "Ajouter" et sélectionnez l'option "Ajouter une catégorie" (ou utilisez l'option "Créer une catégorie" du menu contextuel de la table des contrôles).
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Robot_Cr%E9er%20un%20controle%20-%20HC%20N%B0002%201.gif)


2. Dans la fenêtre qui s'affiche, indiquez le nom de la catégorie.

3. Validez. La nouvelle catégorie apparaît dans le moniteur.





<a name="NOTE2_3"></a>


### Créer un contrôle
<a name="creer_controle_ELTPARAGRAPHE000040"></a>Pour créer un contrôle :

1. Dans le menu, dans le groupe "Catégories et contrôles", cliquez sur "Ajouter" et sélectionnez l'option "Ajouter un contrôle" (ou utilisez l'option "Créer un contrôle" du menu contextuel de la table des contrôles). 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Robot_Cr%E9er%20un%20controle%20-%20HC%20N%B0002.gif)


2. La fenêtre de configuration du contrôle s'affiche. 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Robot_Cr%E9er%20un%20controle%20-%20HC%20N%B0003.gif&type=thumb)


3. Indiquez : 

	- le nom du contrôle.

	- la catégorie du contrôle.

	- le type de contrôle à effectuer.

	- Si le contrôle dépend ou non d'un autre contrôle.
Remarque : Si le contrôle dépend d'un autre contrôle, il sera exécuté uniquement si le contrôle dont il dépend s'est exécuté sans erreur.  




4. Indiquez les [paramètres généraux](../RobotMonitor/3541108.md) du contrôle.

5. Indiquez les paramètres spécifiques du contrôle. Ces paramètres dépendent du type de contrôle sélectionné.

	- [Connectivité machine (ping)](../RobotMonitor/3541113.md).

	- [Connectivité FTP](../RobotMonitor/3541111.md).

	- [Connectivité HTTP](../RobotMonitor/3541112.md).

	- [Connectivité Serveur HFSQL Client/Serveur](../RobotMonitor/3541115.md).

	- [Connectivité bases tierces (par ODBC)](../RobotMonitor/3541110.md).

	- [Connectivité NNTP](../RobotMonitor/3541114.md).

	- [Activité d'une application WINDEV](../RobotMonitor/3541109.md).

	- [Envoi d'un email](../RobotMonitor/3541117.md).

	- [Log d'un site WEBDEV](../RobotMonitor/3541119.md).

	- [SNMP](../RobotMonitor/3541121.md).

	- [Espace disque (par SNMP)](../RobotMonitor/3541118.md).

	- [WLangage (code)](../RobotMonitor/3541122.md).

	- [Serveur WEBDEV](../RobotMonitor/3541120.md).

	- [Connectivité SMTP](../RobotMonitor/3541116.md).

	- Etat d'un cluster.

	- Activité d'un serveur de Télémétrie.

	- Sonde température.

	- Sonde température (par fichier).

	- [Etat SMART d'un disque](../RobotMonitor/3541125.md).




6. A tout moment, le bouton "Tester" permet d'exécuter le test en cours d'édition. Le test est exécuté depuis le moniteur.

7. Validez. Le test apparaît dans le moniteur du robot de surveillance.

## Dupliquer un contrôle
<a name="dupliquer_controle_ELTTEXTE000178"></a>
Pour dupliquer un contrôle existant :

1. Sélectionnez le contrôle à dupliquer dans le moniteur. 

2. Dans le menu du moniteur, dans le groupe "Catégories et contrôles", cliquez sur "Dupliquer". 

3. Validez la duplication. Un nouveau test est créé reprenant toutes les caractéristiques du test dupliqué.

4. La fenêtre de description du contrôle apparaît.

5. Modifiez les éléments voulus (notamment le nom du contrôle) et validez.




**Remarque** : Lors de la validation, il est possible de saisir les raisons de la modification du contrôle. Ces informations sont conservées dans l'historique de modification.

## Rechercher un contrôle
<a name="rechercher_controle_ELTTEXTE000202"></a>
Pour rechercher un contrôle :

1. Dans le menu du moniteur, cliquez sur "Rechercher".

2. Indiquez les critères de recherche. Ces critères peuvent porter sur le nom, la catégorie, le type ou le sujet du contrôle.

3. Validez. Les contrôles correspondant à la recherche sont automatiquement affichés avec une loupe en début de ligne.

## Exécuter un contrôle
<a name="executer_controle_ELTTEXTE000226"></a>
Pour exécuter un contrôle :

1. Sélectionnez le contrôle voulu dans le moniteur.

2. Affichez le menu contextuel du contrôle (clic droit) et sélectionnez "Tester le contrôle".

3. Indiquez si vous désirez que le résultat du test soit sauvegardé dans l'historique.

4. Le contrôle est exécuté et le moniteur affiche les différentes étapes de l'exécution.

5. Fermez la fenêtre d'exécution du contrôle.

## Afficher l'historique
<a name="afficher_historique_ELTTEXTE000250"></a>
Pour effectuer une recherche dans l'historique :

1. Dans le menu du moniteur, dans le groupe "Paramétrage", cliquez sur "Historique".

2. Saisissez les critères de recherche :

	- Période de recherche.

	- Type d'historique :

		- l'historique des erreurs concerne les contrôles en erreur.

		- l'historique des modifications concerne les modifications effectuées sur les contrôles.




- Contrôles concernés.

3. Cliquez sur le bouton "Rechercher" pour lancer la recherche. La liste des historiques correspondants est affichée. Pour chaque historique, le détail est affiché.

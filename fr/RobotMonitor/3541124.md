
## Modèle de contrôle
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000151"></a>
Pour simplifier la création des contrôles, il est possible de créer et d'utiliser des modèles de contrôles. Ces modèles permettent de définir les options de planification des contrôles. 

Ainsi, lors de la création d'un contrôle, au lieu de définir à chaque fois les options de planification du contrôle, il suffit de sélectionner le modèle de contrôle correspondant. 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Gestion des modèles de contrôle
<a name="gestion_des_modeles_controle_ELTTEXTE000175"></a>
Pour gérer les modèles de contrôle, dans le menu de RobotMonitor, dans le groupe "Catégories et contrôles", cliquez sur "Modèle de contrôle".  
<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Robot_Cr%E9er%20un%20controle%20-%20HC%20N%B0001.gif)


La fenêtre de gestion des contrôles est la suivante :  <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Robot_Cr%E9er%20un%20controle%20-%20HC%20N%B0004.gif)


Cette fenêtre permet de : 

- Créer un nouveau modèle, 

- Modifier un modèle. 

- Voir les tests (contrôles) associés à un modèle. 

- Supprimer un modèle. 





### Créer un modèle
<a name="creer_modele_ELTPARAGRAPHE000041"></a>Pour créer un nouveau modèle : 

1. Cliquez sur le bouton "Ajouter un nouveau modèle. Les champs de la fenêtre se vident. 

2. Indiquez les différentes caractéristiques du modèle : 

	- Nom du modèle : Ce nom sera utilisé dans la description du contrôle pour sélectionner les options de planifications voulues. 

	- Délai entre deux contrôles : Temps entre deux exécutions du contrôle. Par exemple, si le délai est de 2 heures, le contrôle sera lancé toutes les 2 heures.

	- Nombre de tentatives : Nombre de fois où le contrôle est relancé avant d'être considéré en erreur.

	- Attente entre deux tentatives : Les tests sont lancés par période de 2 minutes. Le temps entre deux tentatives ne doit pas dépasser 2 minutes (120 secondes).

	- Tranches horaires d'exécution : Période de temps pendant laquelle le contrôle doit être effectué. Pour effectuer le contrôle sur toute la journée, le début doit être "00:00" et la fin "23:59".

	- Jours d'exécution : Jours de la semaine où le contrôle doit être effectué.




3. Cliquez sur le bouton "Valider". Le modèle de contrôle est automatiquement créé. Il est ajouté à la liste des modèles et peut être associé à un contrôle. 





### Modifier un modèle
<a name="modifier_modele_ELTPARAGRAPHE000065"></a>Pour modifier un modèle : 

1. Sélectionnez le modèle voulu dans la liste des modèles. 

2. Modifiez les caractéristiques voulues. 

3. Cliquez sur le bouton "Valider". Le modèle de contrôle est automatiquement mis à jour. Si le modèle est associé à des contrôles, les contrôles sont également mis à jour.  





### Voir les contrôles associés à un modèle
<a name="voir_les_controles_associes_modele_ELTPARAGRAPHE000078"></a>Pour voir les contrôles associés à un modèle : 

1. Sélectionnez le modèle voulu dans la liste des modèles. 

2. Les caractéristiques du modèle sont affichées dans la partie gauche de la fenêtre (zone "Paramètre du modèle"). 

3. Les contrôles associés au modèle sont affichés dans la partie droite de la fenêtre (zone "Tests associés au modèle"). 







### Supprimer un modèle
<a name="supprimer_modele_ELTPARAGRAPHE000088"></a>Pour supprimer un modèle : 

1. Sélectionnez le modèle voulu dans la liste des modèles. 

2. Cliquez sur le bouton <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Robot_Cr%E9er%20un%20controle%20-%20HC%20N%B0004%201.gif)

Attention : Pour supprimer un modèle, ce modèle ne doit plus être associé à un contrôle. 

Remarque : Le bouton "Supprimer les modèles sans contrôles" permet de supprimer tous les modèles qui ne sont plus associés à un contrôle.

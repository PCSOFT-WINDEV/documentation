
## Contrôle : Envoi d'un email
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000093"></a>
Le robot de surveillance va :

- envoyer un email à une adresse email.

- lire la boîte de réception de l'adresse email pour vérifier que l'email a été reçu.




Le contrôle sera :

- réussi si l'email a été envoyé et s'il est présent dans la boîte de réception.

- en échec dans le cas contraire.






<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Paramètres spécifiques au contrôle d'envoi d'email
<a name="parametres_specifiques_controle_envoi_email_ELTTEXTE000117"></a>


### Paramètres SMTP (pour l'envoi du message)
<a name="parametres_smtp_pour_envoi_message_ELTPARAGRAPHE000027"></a>Les paramètres SMTP sont ceux du serveur utilisé pour l'envoi d'emails :

- l'adresse email à laquelle le message de test doit être envoyé.

- le serveur SMTP à utiliser et le port associé.

- l'utilisateur et mot de passe associé.





<a name="NOTE2_2"></a>


### Paramètres POP3 (pour la lecture du message reçu)
<a name="parametres_pop3_pour_lecture_message_recu_ELTPARAGRAPHE000039"></a>Les paramètres SMTP sont ceux du serveur utilisé pour l'envoi d'emails :

- l'adresse email à laquelle le message de test doit être relu.

- le serveur POP3 à utiliser et le port associé. La connexion peut être une connexion SSL.

- l'utilisateur et mot de passe associé.

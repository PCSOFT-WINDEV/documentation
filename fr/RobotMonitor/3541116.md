
## Contrôle : Connectivité SMTP
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000078"></a>
Le robot de surveillance va tenter d'effectuer une connexion à un serveur SMTP avec l'utilisateur indiqué.

Le contrôle sera :

- réussi si la connexion est effective.

- en échec dans le cas contraire.






<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Paramètres spécifiques au contrôle de la connectivité SMTP
<a name="parametres_specifiques_controle_connectivite_smtp_ELTTEXTE000102"></a>


### Informations de connexion au serveur SMTP
<a name="informations_connexion_serveur_smtp_ELTPARAGRAPHE000024"></a>Il est nécessaire de préciser les caractéristiques de connexion au serveur SMTP :

- l'adresse du serveur SMTP.

- le numéro de port de connexion.

- l'utilisateur et le mot de passe associé.

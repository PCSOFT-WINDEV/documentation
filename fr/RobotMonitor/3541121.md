
## Contrôle : SNMP
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000094"></a>
Le robot de surveillance va lire la valeur de l'OID indiquée et comparer sa valeur avec la valeur de référence.

Le contrôle sera :

- réussi si la valeur de l'OID a pu être lue et correspond à la valeur de référence.

- en échec dans le cas contraire.






<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Paramètres spécifiques au contrôle SNMP
<a name="parametres_specifiques_controle_snmp_ELTTEXTE000118"></a>


### Informations sur l'agent SNMP
<a name="informations_sur_agent_snmp_ELTPARAGRAPHE000024"></a>Pour effectuer un test SNMP, indiquez :

- l'adresse de l'agent SNMP à utiliser,

- la communauté de l'agent.





<a name="NOTE2_2"></a>


### Informations sur l'OID à contrôler
<a name="informations_sur_oid_controler_ELTPARAGRAPHE000035"></a>Pour effectuer un test SNMP, indiquez :

- l'OID prédéfini s'il existe,

- les conditions de test de la valeur. 




**Remarque** : Pour ajouter un OID prédéfini :

1. Saisissez son nom (champ "OID prédéfinis").

2. Saisissez sa valeur (champ "OID à lire").

3. Cliquez sur le bouton "+".

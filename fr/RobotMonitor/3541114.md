
## Contrôle : Connectivité NNTP
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000077"></a>
Le robot de surveillance va tenter de se connecter au serveur NNTP précisé (serveur de news).

Le contrôle sera :

- réussi si la connexion est effective.

- en échec dans le cas contraire.

### Informations sur le serveur NNTP
<a name="informations_sur_serveur_nntp_ELTPARAGRAPHE000024"></a>Pour tester un serveur NNTP, il est nécessaire d'indiquer :

- l'adresse du serveur NNTP.

- le port de connexion.

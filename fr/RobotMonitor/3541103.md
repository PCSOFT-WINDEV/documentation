
## Installation du moniteur
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000119"></a>
Le robot de surveillance est constitué de trois exécutables.

Le premier exécutable à installer est [l'exécutable correspondant au robot](../RobotMonitor/3541101.md). Le robot de surveillance tourne en permanence sur un poste et lance tous les tests demandés. Il doit être installé sur un poste qui ne sera jamais éteint.

Le second exécutable est le [contrôleur du robot (ou contre-robot)](../RobotMonitor/3541102.md). **Son installation n'est pas obligatoire**. Le contre-robot est une sécurité en plus. Il permet de surveiller le bon fonctionnement du robot de surveillance. S'il est installé, le contre-robot doit être installé sur un poste qui ne sera jamais éteint, différent du poste d'installation du robot de surveillance.  

Le troisième exécutable est le **moniteur des robots**. Le moniteur peut être installé sur n'importe quel poste. Il permet d'éditer les tests et les différents paramètres du robot.

## Assistant d'installation du moniteur
<a name="assistant_installation_moniteur_ELTTEXTE000143"></a>


### Installer le moniteur
<a name="installer_moniteur_ELTPARAGRAPHE000031"></a>Pour installer le moniteur :

1. Lancez le pack d'installation (WX280PACKROBOT.EXE) sur le poste où le moniteur doit être installé. L'assistant d'installation se lance. 
	Rappel : Le pack d'installation (WX280PACKROBOT.EXE) est disponible dans le sous-répertoire "Install\\Robot" du produit utilisé.

2. Validez la licence.

3. Sélectionnez l'option "Installer le moniteur des Robots" et passez à l'étape suivante.

4. Sélectionnez le répertoire d'installation du moniteur des robots. Passez à l'étape suivante.  

5. L'installation est terminée. Validez. L'installation du moniteur des robots est réalisée. Il vous suffit maintenant de paramétrer le moniteur des robots.





<a name="NOTE2_2"></a>


### Paramétrage du moniteur des robots
<a name="parametrage_moniteur_des_robots_ELTPARAGRAPHE000051"></a>A la fin de l'installation du moniteur, un assistant est automatiquement affiché pour configurer le moniteur.

1. Passez à l'étape suivante.

2. Indiquez l'emplacement de la base de données du robot de surveillance. Cette base de données peut être : 

	- Une base de données HFSQL Client/Serveur. Dans ce cas, indiquez : 

		- l'adresse du serveur, son port,

		- le nom d'utilisateur et le mot de passe associé, 

		- la base de données à utiliser. 




- Une base de données HFSQL Classic, présente dans un répertoire partagé. Ce répertoire correspond au répertoire partagé créé sur le poste où le robot a été installé. 

3. Passez à l'étape suivante.

4. Sélectionnez les options d'alerte du moniteur. Ces options correspondent à l'alerte effectuée sur le moniteur en cas d'erreur. Il est possible :

	- d'utiliser une alerte visuelle.

	- d'utiliser une alerte sonore.




5. Passez à l'étape suivante. Validez. Le paramétrage du moniteur est réalisé.




Vous pouvez créer des tests.

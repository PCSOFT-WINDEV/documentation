
## Contrôle : Espace disque par SNMP
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000087"></a>
Le robot de surveillance récupère l'espace disque disponible via un agent SNMP.

Le contrôle sera :

- réussi si l'espace disponible est supérieur au seuil d'alerte.

- en échec dans le cas contraire.






<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Paramètres spécifiques au contrôle d'espace disque
<a name="parametres_specifiques_controle_espace_disque_ELTTEXTE000111"></a>


### Informations sur l'agent SNMP
<a name="informations_sur_agent_snmp_ELTPARAGRAPHE000024"></a>Pour vérifier l'espace disque, indiquez :

- l'adresse de l'agent SNMP à utiliser,

- la communauté de l'agent.





<a name="NOTE2_2"></a>


### Informations sur le disque à contrôler
<a name="informations_sur_disque_controler_ELTPARAGRAPHE000035"></a>Lorsque l'adresse de l'agent SNMP est indiquée, il est possible de lister les disques accessibles par cet agent (bouton "Lister les disques"). 

Sélectionnez ensuite le ou les disques à contrôler et indiquez le seuil d'alerte.

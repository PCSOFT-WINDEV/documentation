
## Contrôle : Connectivité machine (requête PING)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000074"></a>
Le robot de surveillance va effectuer un ping sur l'adresse IP spécifiée.

Le contrôle sera :

- réussi si l'adresse répond au ping,

- en échec dans le cas contraire.


## Configuration des groupes d'utilisateurs
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000077"></a>
Le robot de surveillance permet de gérer des groupes d'utilisateurs qui seront concernés par les différents rapports d'erreur des contrôles. 

Pour gérer les groupes d'utilisateurs : 

1. Dans le menu du moniteur, dans le groupe "Paramétrage", cliquez sur "Gérer les groupes". 

2. La fenêtre qui s'affiche permet de : 

	- Ajouter ou supprimer un utilisateur dans un groupe. 

	- Ajouter un groupe.

	- Renommer un groupe.

	- Supprimer un groupe. 







<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Caractéristiques d'un groupe
<a name="caracteristiques_groupe_ELTTEXTE000101"></a>
Lors de l'ajout d'un groupe, il suffit d'indiquer son nom dans la fenêtre qui s'affiche. Le nouveau groupe apparaît ensuite dans la liste des groupes. 
Pour associer un utilisateur à un groupe, il suffit de sélectionner le groupe voulu dans la liste des groupes et de "cocher" les utilisateurs membres de ce groupe. 

Lorsque le groupe est créé, il peut être utilisé comme destinataire du rapport d'erreur d'un contrôle (dans les paramètres généraux du contrôle).

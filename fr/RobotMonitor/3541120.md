
## Contrôle : Serveur WEBDEV
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000080"></a>
Le robot de surveillance va tester le fonctionnement du serveur WEBDEV.

Le contrôle sera :

- réussi si le serveur WEBDEV répond.

- en échec dans le cas contraire.

<a name="NOTE2_1"></a>


## Paramètres spécifiques au contrôle d'un serveur WEBDEV
<a name="parametres_specifiques_controle_serveur_webdev_ELTTEXTE000104"></a>


### Informations sur le serveur WEBDEV
<a name="informations_sur_serveur_webdev_ELTPARAGRAPHE000024"></a>Les informations sur le serveur WEBDEV à tester sont :

- l'adresse du serveur WEBDEV.

- la version du serveur WEBDEV (par exemple 28).

- la plateforme (Windows ou Linux)


## 5. HFSQL Client/Serveur

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Une application WINDEV Mobile HFSQL peut également fonctionner en mode Client/Serveur.

Les caractéristiques du mode Client/Serveur sont les suivantes :

- Une application HFSQL Client/Serveur est exécutée sur différents périphériques mobiles (appelés machines clientes).

- Les fichiers de données sont présents sur un poste serveur. Seul le poste serveur accède physiquement aux fichiers de données.

- L'ensemble des traitements (requête, lecture/ajout dans un fichier de données, ...) est réalisé sur le serveur.




![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=WM%20HF%20Client%20serveur.svg)


HFSQL Client/Serveur est utilisable sur les plateformes iPhone, iPad, Android et Universal Windows 10 Platform.


## 10. Les requêtes intégrées

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les champs d'une fenêtre peuvent être liés à un fichier de données ou à une requête existante, ... Il est également possible de lier ces champs à une requête créée lors de la conception du champ.


|   |   |
| --- | --- |
| <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=requ%EAte_Int%E9gr%E9e_0.gif)<br> | <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=requ%EAte_Int%E9gr%E9e_1.gif)<br>Requête intégrée :<br>MaFenêtre_1$Requête |

Dans ce cas, la requête est intégrée à la fenêtre. Elle est présente dans le fichier WDW correspondant à la fenêtre. Si le fichier WDW est copié (dans un autre projet par exemple), les requêtes intégrées utilisées par cette fenêtre seront également copiées.

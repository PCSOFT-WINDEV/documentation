
## 5. Perso-dossiers : Organisez votre projet

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les éléments qui composent un projet sont tous listés dans le volet "Explorateur de Projet". Par défaut, les éléments sont rangés en fonction de leur type : fenêtres, états, classes...

Dans les projets de taille importante, il est souvent plus pertinent de regrouper les éléments correspondant à la même fonctionnalité : gestion du stock ou traitement des commandes par exemple.

Pour cela, il suffit de créer des "perso-dossiers" dans l'arborescence du volet "Explorateur de projet" et de glisser les différents éléments dans ces dossiers.

Des éléments peuvent être communs à plusieurs "perso-dossiers".

Il est ainsi plus simple de travailler sur une partie de l'application. 
<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=P2_WM_Perso_dossier%20-%20HC%20N%B0001.gif)


**Remarque** : Pour ranger, organiser vos projets, mais aussi pour partager un ensemble d'éléments entre différents projets, WINDEV Mobile met également à votre disposition les composants internes.

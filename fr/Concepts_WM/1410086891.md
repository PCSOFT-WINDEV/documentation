<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les applications créées avec WINDEV Mobile peuvent fonctionner sur les plateformes suivantes :

- Universal Windows Plateform apps. 

- Android version 6 et supérieures.

- iOS version 13 et supérieures.




Les processeurs suivants sont supportés :

- **ARM et compatibles** (Strong ARM, XScale, Samsung, Texas Instrument, etc.).

- **ARM v4T et compatibles** (XScale, etc.).




**Remarque** : 
Windows Mobile, Windows CE et Windows Embedded ne sont plus supportés par Microsoft, mais sont encore utilisés dans le domaine industriel. 
Pour continuer à développer pour Windows Mobile, vous devez utiliser une version 26. Votre version 28 donne accès à la version 26 qui peut être téléchargée [ici](http://www.pcsoft.fr/dl).

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
WINDEV Mobile vous permet de générer de nombreux autres types de projets.


|   |   |
| --- | --- |
| ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=P2_Mode%20generation%20Mobile%20-%20HC%20N%B0003%201.gif) | **Applications Android**<br><br>WINDEV Mobile permet de générer des applications pour la plateforme Android. Ces applications pourront être lancées sur des smartphones, des tablettes, des ultra-portables, utilisant ce système d'exploitation (à partir de la version 5.0). Ces applications peuvent également être distribuées sur Play Store par exemple. |
| ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=P2_Mode%20generation%20Mobile%20-%20HC%20N%B0003%202.gif) | **Applications iOS**<br><br>WINDEV Mobile permet de générer des applications pour la plateforme iPhone et iPad (à partir de iOS 9). Ces applications pourront être lancées sur des iPhone et des iPad. Ces applications peuvent également être distribuées sur Apple Store par exemple. |
| ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=P2_Mode%20generation%20Mobile%20-%20HC%20N%B0001%202.gif) | **Universal Windows 10 App**<br><br>WINDEV Mobile permet de générer des applications pour des tablettes et des téléphones Windows 10.<br>Ces applications pourront être lancées en mode tuile sur des tablettes Windows 10. Ces applications pourront être également distribuées sur Windows Store par exemple. |
| ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=P2_Mode%20generation%20Mobile%20-%20HC%20N%B0001%201.gif) | **Webservices SOAP ou REST**<br><br>A partir d'un projet WINDEV Mobile, il est possible de générer un Webservice (également appelé service Web XML). Un Webservice expose un ensemble de fonctions (généralement une ou plusieurs collections de procédures) et les rend accessibles à travers le Web (ou un réseau privé) en utilisant les protocoles standard de communication HTTP, SOAP et REST.<br><br>Remarque : Un Webservice doit être déployé sur un Serveur d'Application WEBDEV pour être utilisable. |
| ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=P2_Mode%20generation%20Mobile%20-%20HC%20N%B0001%203.gif) | **Composants externes**<br><br>Les composants externes sont des briques applicatives permettant de partager une ou plusieurs fonctionnalités spécifiques entre différentes applications. Un composant généré avec WINDEV peut également être utilisé dans un projet WEBDEV ou WINDEV Mobile. |


## 4. HFSQL Classic

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Le format HFSQL Classic est le format de base de données fourni avec WINDEV Mobile. Ce format de base de données est compatible entre WINDEV, WINDEV Mobile et WEBDEV.

Il s'agit d'un SGBD Relationnel redistribuable gratuitement.

Ce format est utilisable sur les plateformes iOS, Android et Universal Windows 10 App.

Ce format est identique au format HFSQL Classic de WINDEV et de WEBDEV (fichier ".WDD", fichiers de données, ...).

Cependant, la taille disponible sur les périphériques mobiles étant souvent restreinte, les fonctionnalités suivantes ne sont pas gérées par HFSQL Classic sur mobile :

- les transactions.

- la journalisation.

- la gestion des blocages des fichiers et des enregistrements.

- la gestion des fichiers au format Hyper File 5.5.

## Manipulation d'une base de données HFSQL Classic depuis le périphérique mobile
<a name="manipulation_une_base_donnees_hfsql_classic_depuis_peripherique_mobile_ELTTEXTE000135"></a>
Une base de données HFSQL Classic correspond à un ensemble de fichiers ".FIC", ".NDX", ".MMO".

Chaque fichier de données peut être manipulé par une application WINDEV Mobile. Ces manipulations sont réalisées grâce aux fonctions HFSQL (fonctions **Hxxx**). <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=pocketPc_HF.svg)

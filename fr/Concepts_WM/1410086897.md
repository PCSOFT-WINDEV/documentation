
## 10. Fenêtre interne
			

| [Page précédente](../Concepts_WM/1410086896.md) | [Sommaire](../Concepts_WM/1410086964.md) | [Page suivante](../Concepts_WM/1410086898.md) |

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Le champ Fenêtre interne permet d'inclure une fenêtre (et son code) dans une autre fenêtre. A l'exécution, la fenêtre à fusionner sera dynamiquement fusionnée à la fenêtre de réception.


### 1. Création d'une fenêtre interne
<a name="1_creation_une_fenetre_interne_ELTPARAGRAPHE000015"></a>


|   |   |
| --- | --- |
| La création d'une fenêtre interne se fait grâce à l'icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=P1_ICO_Cr%E9er_cpt.gif) disponible dans les boutons d'accès rapide. Dans la fenêtre de création d'un nouvel élément, cliquez sur "Fenêtre" puis sur "Fenêtre interne".<br><br>Une fenêtre interne est une fenêtre spécifique qui ne comporte ni barre de titre, ni menu.<br>Dans cette fenêtre, tous les types de champs peuvent être utilisés. | <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=P1_Fen%EAtre%20interne%201.gif&type=thumb)<br> |




### 2. Utilisation d'une fenêtre interne.
<a name="2_utilisation_une_fenetre_interne_ELTPARAGRAPHE000030"></a>


|   |   |
| --- | --- |
| Pour utiliser une fenêtre interne, il suffit de :<br><br>- créer un champ de type fenêtre interne.<br><br>- sélectionner dans la description du champ la fenêtre interne à utiliser et valider.<br><br><br> | <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=P1_Fen%EAtre%20interne%202.gif&type=thumb)<br> |

Remarques :

- Il est possible de modifier par programmation la fenêtre interne utilisée dans le champ "Fenêtre interne".

- Limitations : La zone d'accueil est rectangulaire et aucune surcharge n'est possible. Pour réaliser des surcharges, il est conseillé d'utiliser des modèles de champs.

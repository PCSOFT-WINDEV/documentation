
## 1. Fenêtres générées par le RAD

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Le RAD (Rapid Application Development) permet de créer des fenêtres à partir :

- de l'analyse liée au projet,

- des gabarits.




Les fenêtres générées contiennent tout le code nécessaire à leur fonctionnement. Ces fenêtres peuvent être testées immédiatement avec les données présentes sur le poste de développement.

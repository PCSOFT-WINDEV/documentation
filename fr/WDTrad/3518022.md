
## Propriétés du dictionnaire
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000155"></a>
Il est possible de configurer les options du dictionnaire de WDTRAD grâce à l'option "Edition .. Propriétés du dictionnaire". La fenêtre qui s'affiche est composée de plusieurs onglets dédockables. 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Onglet "Général" : Options disponibles
<a name="onglet_general_options_disponibles_ELTTEXTE000179"></a>
L'onglet "Général" permet : 

- d'obtenir des statistiques de traduction sur le dictionnaire en cours

- d'effectuer des opérations de maintenance. 





### Statistiques de traduction
<a name="statistiques_traduction_ELTPARAGRAPHE000023"></a>Pour obtenir les statistiques de traduction, cliquez sur le lien "Générer les statistiques du dictionnaire". Ces statistiques permettent d'obtenir le nombre de ressources présentes dans le dictionnaire. 


### Maintenance
<a name="maintenance_ELTPARAGRAPHE000032"></a>Les opérations de maintenance sont les suivantes : 

- **Paramétrage de la langue** : Pour chaque langue, il est possible de paramétrer les caractéristiques de la langue : clavier utilisé, symboles. Le paramétrage des symboles est utilisé par WDTRAD lors de la reconstruction d'une traduction à partir d'une traduction existante. 
	Pour paramétrer les caractéristiques d'une langue : 

	1. Sélectionnez la langue voulue. 

	2. Cliquez sur le bouton ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=icone_Proprietes.gif)
. L'onglet "Cla­vier" et l'onglet "Symboles" permettent de configurer les spécificités de la langue sélectionnée. 




- **Réindexation du dictionnaire** : Pour chaque langue, il est possible de réindexer le dictionnaire. Cette réindexation permet d'optimiser les performances d'accès aux données du dictionnaire. Cette réindexation peut prendre plusieurs minutes. 
	Pour réindexer une langue : 

	1. Sélectionnez la langue voulue. 

	2. Cliquez sur le bouton "Réindexer" et validez.

## Onglet "Terminologies personnalisées" : Options disponibles
<a name="onglet_terminologies_personnalisees_options_disponibles_ELTTEXTE000215"></a>
L'onglet "Terminologie" permet de figer des traductions à utiliser pour certains termes. Ces traductions seront utilisées dès que les termes spécifiés seront rencontrés lors d'une traduction automatique réalisée via DeepL par exemple. 

L'option "Synchroniser avec Amazon" permet de stocker les terminologies dans Amazon Translate. 

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Onglet "Recherche" : Options disponibles
<a name="onglet_recherche_options_disponibles_ELTTEXTE000239"></a>
L'onglet "Recherche" permet d'effectuer une recherche spécifique dans le dictionnaire. 

Il suffit de : 

1. Sélectionner la langue d'origine et la langue destination.

2. Saisir dans la zone "Recherche" le texte à rechercher dans la langue voulue. 

3. Cliquer sur le bouton "Rechercher". Selon les [options spécifiées](../WDTrad/3518021.md), les ressources trouvées sont listées et peuvent être modifiées. 




Remarque : Le menu contextuel de la table des ressources trouvées permet d'ajouter une entrée dans le dictionnaire (option "Ajouter une entrée dans le dictionnaire"). 

<a name="NOTE4"></a>
<a name="NOTE4_1"></a>


## Onglet "Rechercher / Remplacer" : Options disponibles
<a name="onglet_rechercher_remplacer_options_disponibles_ELTTEXTE000263"></a>
L'onglet "Rechercher / Remplacer" permet d'effectuer une recherche et un remplacement dans le dictionnaire. 

Il suffit de : 

1. Sélectionner la langue dans laquelle la recherche doit être effectuée.

2. Saisir dans la zone "Recherche" le texte à rechercher dans la langue spécifiée. 
	Remarque : Les caractères \* et ? peuvent être utilisés. Le caractère ~ permet d'exclure un terme de la recherche. 

3. Indiquer les options de recherche (casse, commence par, ...).

4. Cliquer sur le bouton "Rechercher". Selon les options spécifiées, les ressources trouvées sont listées. 

5. Indiquer le texte de remplacement. 

6. Effectuer le remplacement : 

	- Cliquer sur "Remplacer" pour remplacer la ressource sélectionnée. 

	- Cliquer sur "Remplacer tout" pour remplacer toutes les ressources listées.

## Onglet "Ajoutés récemment" : Options disponibles
<a name="onglet_ajoutes_recemment_options_disponibles_ELTTEXTE000287"></a>
L'onglet "Ajoutés récemment" permet de lister les dernières ressources ajoutées dans le dictionnaire. Il est possible de sélectionner les langues voulues ainsi que l'auteur de la traduction. Les 100 dernières ressources sont affichées.


## Les différents modes de copie
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000096"></a>
WDTRAD propose deux modes de copie entre les messages extraits et les messages traduits :

- copie simple.

- copie du message "Traduction" pour tous les messages "Référence" identiques.






<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Copie simple
<a name="copie_simple_ELTTEXTE000120"></a>
Cette méthode permet de copier le message "Référence" en cours dans le message "Traduction" correspondant.

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CopieSimple.gif)


Pour effectuer une copie simple, cliquez sur le bouton "Copier réf.".



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Copie du message "Traduction" pour tous les messages "Référence" identiques
<a name="copie_message_traduction_pour_tous_les_messages_reference_identiques_ELTTEXTE000144"></a>
Cette méthode affecte le message "Traduction" en cours à toutes les autres lignes ayant le même message "Référence".

Pour effectuer ce mode de copie, cliquez sur le bouton "Reporter trad.".

Ce mode de copie recherche toutes les lignes dont la colonne "Référence" est identique à celle de la ligne en cours. Le message "Traduction" de la ligne en cours est ensuite affecté à l'ensemble des lignes trouvées.

**Par exemple**, si la ligne en cours contient le message "Annuler" et sa traduction "Cancel", un clic sur le bouton "Reporter trad." permet de :

- rechercher tous les messages "Annuler",

- renseigner la traduction "Cancel" pour chacun des messages "Annuler" trouvés.




**Remarque** : Si des messages "Traduction" étaient présents dans les messages trouvés, ces messages "Traduction" sont remplacés avec la nouvelle traduction.

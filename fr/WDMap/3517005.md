
## WDMap : Visualisation d'enregistrements : cas particuliers
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000164"></a>
Cette page d'aide présente les cas particulier d'affichage sous WDMap :

- Les colonnes "Numéro d'enregistrement" et "Loupe".

- L'affichage des rubriques mémo et des clés composées.






<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Colonnes spécifiques
<a name="colonnes_specifiques_ELTTEXTE000194"></a>


### Colonne "Numéro d'enregistrement"
<a name="colonne_numero_enregistrement_ELTPARAGRAPHE000023"></a>Lors de la visualisation d'un fichier de données HFSQL, la colonne "Numéro d'enregistrement" apparaît. Le numéro d'enregistrement permet d'identifier et/ou de manipuler de façon unique chacun des enregistrements présents dans un fichier de données HFSQL.

**Remarque** : Il n'est pas possible de :

- Modifier les numéros d'enregistrement.

- Trier les numéros d'enregistrement (par ordre croissant ou décroissant).

- Déplacer la colonne "Numéro d'enregistrement".

- Rechercher un numéro d'enregistrement.





<a name="NOTE2_2"></a>


### Colonne "Loupe"
<a name="colonne_loupe_ELTPARAGRAPHE000038"></a>La colonne "Loupe" permet de visualiser les données liées. Par exemple, pour une commande, il est possible de voir les lignes de commande et les produits correspondants.

Pour visualiser les données liées : 

1. Cliquez sur la colonne "Loupe". Vous obtenez la liste des fichiers liés et le nom de la rubrique sur laquelle la liaison est effectuée.

2. Sélectionnez le fichier à visualiser et validez. La liste des enregistrements du fichier lié correspondant à la rubrique sélectionnée dans le fichier principal apparaît. ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Fichier_lie.gif)

## Cas particuliers
<a name="cas_particuliers_ELTTEXTE000224"></a>


### Cas particulier : Rubrique de type mémo Texte
<a name="cas_particulier_rubrique_type_memo_texte_ELTPARAGRAPHE000054"></a>L'option "Paramètres .. Colonne mémo multi-ligne" permet d'afficher :

- Soit uniquement la première ligne des mémos texte. Cette option permet d'optimiser la visualisation de l'ensemble des enregistrements d'un fichier.![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=MemoTexte1.gif)


- Soit le contenu complet des mémos texte. Les colonnes correspondantes deviennent multilignes. La hauteur des lignes de la table est proportionnelle à la taille des mémos.![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=MemoTexte2.gif)





Pour visualiser la totalité d'un mémo texte et/ou pour modifier le contenu d'un mémo texte :

1. Sélectionnez la colonne mémo de l'enregistrement voulu.

2. Sélectionnez l'option "Édition .. Colonne mémo".

3. Le contenu du mémo texte sélectionné apparaît dans une fenêtre spécifique.


**Remarque** : Il est également possible de modifier directement le contenu d'un mémo texte depuis la table de visualisation de WDMap.


<a name="NOTE3_2"></a>


### Cas particulier : Rubrique de type mémo Binaire
<a name="cas_particulier_rubrique_type_memo_binaire_ELTPARAGRAPHE000075"></a>WDMap permet de changer, d'exporter et de jouer les mémos binaires.

Pour changer le mémo binaire associé à la rubrique :

1. Double-cliquez sur la cellule contenant le mémo binaire. La fenêtre de modification du mémo binaire s'affiche.

2. Cliquez sur le bouton "Importer". Un sélecteur de fichiers s'affiche.

3. Sélectionnez le nouveau mémo binaire.




Pour exporter un mémo binaire vers un nouveau fichier :

1. Double-cliquez sur la cellule contenant le mémo binaire. La fenêtre de modification du mémo binaire s'affiche.

2. Cliquez sur le bouton "Exporter". Un sélecteur de fichiers s'affiche.

3. Saisissez le nom du fichier à générer. Par défaut, le nom et l'extension du fichier associé au mémo binaire sont proposés.




Pour jouer un mémo binaire :

1. Double-cliquez sur la cellule contenant le mémo binaire. La fenêtre de modification du mémo binaire s'affiche.

2. Cliquez sur le bouton "Jouer".
	**Remarque** : Ce bouton est actif uniquement si le mémo peut être exécuté.





<a name="NOTE3_3"></a>


### Cas particulier : Clés composées
<a name="cas_particulier_cles_composees_ELTPARAGRAPHE000102"></a>Par défaut, les rubriques de type "clé composée" ne sont pas affichées dans WDMap.

**Pour afficher les rubriques de type "clé composée"**, sélectionnez l'option "Affichage .. Afficher les clés composées".

Les différentes valeurs d'une rubrique de type "clé composée" sont séparées par des virgules.

Dans l'exemple suivant, la rubrique "clé composée" *DATECODENOM* est composée des rubriques *DATECMD*, *CODECMD* et *NOM*.

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=clecomposee.gif)


**Pour ne pas afficher les rubriques de type "clé composée"**, sélectionnez l'option "Affichage .. Enlever les clés composées".

**Remarque** : Pour modifier la valeur d'une rubrique de type "clé composée", modifiez les valeurs des différentes rubriques de la "clé composée".

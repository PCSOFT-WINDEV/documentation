
## WDMap : Modifier le mode d'affichage des enregistrements
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000201"></a>
Par défaut, WDMap affiche les enregistrements d'un fichier de données en appliquant les règles suivantes :

- Tous les enregistrements présents dans le fichier de données sont affichés.

- Les enregistrements sont triés selon leur ordre de création dans le fichier de données.

- Toutes les rubriques du fichier de données sont affichées (sauf les rubriques de type "Clé composée").

- Les rubriques sont classées par ordre logique. L'ordre logique correspond à l'ordre d'affichage défini sous l'éditeur d'analyses.




Cependant, il est possible de modifier à tout moment :

- La sélection des informations à afficher. En effet, WDMap permet de sélectionner :

	- les enregistrements à visualiser,

	- les rubriques à visualiser.




- Les options d'affichage de la table de visualisation :

	- recherche d'enregistrements,

	- tri des enregistrements visualisés,

	- modification de l'ordre d'affichage des rubriques,

	- ancrage des colonnes sur la gauche de la table.









<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Visualiser uniquement une sélection d'enregistrements
<a name="visualiser_uniquement_une_selection_enregistrements_ELTTEXTE000225"></a>
Pour simplifier vos recherches, WDMap permet d'afficher uniquement les enregistrements correspondant à un critère spécifique.
Par exemple, il est possible d'afficher uniquement les clients habitant "Montpellier" ou les fournisseurs dont le nom commence par la lettre "F".

Deux méthodes sont à votre disposition :

- Saisie directe d'un ou de plusieurs critère(s) de sélection sur les rubriques du fichier (option "Affichage .. Sélection d'enregistrements .. Assistant" ou Ctrl + Q).

- Utilisation d'une requête SQL sur les enregistrements du fichier (option "Affichage .. Sélection d'enregistrements .. Requête SQL").




**Remarque** : Ces critères de sélection ne sont pas sauvegardés. Lors d'une prochaine utilisation de WDMap, il sera nécessaire de redéfinir ces critères pour réaliser une nouvelle sélection.



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Visualiser uniquement certaines rubriques
<a name="visualiser_uniquement_certaines_rubriques_ELTTEXTE000249"></a>
Si vos fichiers de données contiennent de nombreuses rubriques, WDMap offre la possibilité d'afficher uniquement les rubriques nécessaires à votre recherche.

Par exemple, si votre fichier "Client" contient plus de 20 rubriques, il est possible de visualiser uniquement le nom, le prénom et la ville de chaque client.

Pour sélectionner les rubriques à visualiser, sélectionnez l'option "Affichage .. Sélection des colonnes à visualiser" (ou raccourci Ctrl + O) et cochez les rubriques à visualiser.

**Remarque** : Ce critère de sélection n'est pas sauvegardé. Lors d'une prochaine utilisation de WDMap, il sera nécessaire de redéfinir ce critère pour réaliser une nouvelle sélection.

## Configuration de la table de visualisation
<a name="configuration_table_visualisation_ELTTEXTE000273"></a>


### Présentation
<a name="presentation_ELTPARAGRAPHE000068"></a>Sous WDMap, les enregistrements sont visualisés dans une table. Toutes les options d'affichage spécifiques à ce type de champ sont donc disponibles.


<a name="NOTE4_2"></a>


### Rechercher des enregistrements
<a name="rechercher_des_enregistrements_ELTPARAGRAPHE000076"></a>Une recherche de type "Commence par" peut être réalisée sur une rubrique de type clé.

Les colonnes dans lesquelles une recherche peut être effectuée sont identifiées par l'icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=IconeRecherche.gif).

Pour rechercher des enregistrements selon la valeur d'une rubrique :

1. Cliquez sur l'icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=IconeRecherche.gif) dans l'entête de la colonne. Une zone de saisie apparaît.

2. Saisissez la valeur de la rubrique recherchée. Les enregistrements commençant par les lettres saisies s'affichent dynamiquement.




**Remarque** : Ce critère de recherche n'est pas sauvegardé. Lors d'une prochaine utilisation de WDMap, il sera nécessaire de redéfinir ce critère de recherche.


<a name="NOTE4_3"></a>


### Trier les enregistrements visualisés
<a name="trier_les_enregistrements_visualises_ELTPARAGRAPHE000097"></a>Un tri simple peut être réalisé sur une rubrique (par ordre croissant ou décroissant).

Par exemple, il est possible de trier les fournisseurs selon leur ville.

Les colonnes triables sont identifiées par l'icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=IconeTri.gif).

Pour trier les enregistrements visualisés selon une rubrique, cliquez sur l'entête de la colonne.

**Remarque** : Ce critère de tri n'est pas sauvegardé. Lors d'une prochaine utilisation de WDMap, il sera nécessaire de redéfinir ce critère pour réaliser un nouveau tri.


<a name="NOTE4_4"></a>


### Modifier l'ordre d'affichage des rubriques
<a name="modifier_ordre_affichage_des_rubriques_ELTPARAGRAPHE000115"></a>Par défaut, l'ordre d'affichage correspond à l'ordre logique des rubriques. L'ordre logique correspond à l'ordre d'affichage défini sous l'éditeur d'analyses.

WDMap permet de modifier facilement l'ordre d'affichage des rubriques.

Par exemple, il est possible d'organiser les colonnes par ordre d'importance ou de rassembler les colonnes contenant le même type d'informations.

Pour déplacer une colonne :

1. Cliquez sur l'entête de la colonne et maintenez le bouton gauche de la souris enfoncé.

2. Déplacez la colonne jusqu'à sa nouvelle position puis relâchez le bouton gauche de la souris.




**Remarques** :

- Cet ordre d'affichage n'est pas sauvegardé. Lors d'une prochaine utilisation de WDMap, l'ordre utilisé sera l'ordre par défaut (ordre logique des rubriques).

- La colonne "N° Enr." n'est pas déplaçable (cette colonne correspond au numéro des enregistrements du fichier de données).





<a name="NOTE4_5"></a>


### Ancrages des colonnes sur la gauche de la table
<a name="ancrages_des_colonnes_sur_gauche_table_ELTPARAGRAPHE000137"></a>Par défaut, pour visualiser toutes les colonnes, il est nécessaire d'utiliser l'ascenseur horizontal.

WDMap permet de fixer des colonnes sur la gauche de la table de visualisation. Lors de l'utilisation de l'ascenseur horizontal, seules les colonnes non fixées défileront.

Les colonnes contenant les rubriques les plus importantes peuvent ainsi être constamment visibles.

Par exemple, si la colonne "NomClient" est fixée sur la gauche, lors de l'utilisation de l'ascenseur horizontal, le nom des clients sera toujours visible.

Pour fixer une colonne, survolez le côté gauche de l'entête de la colonne à fixer et cliquez lorsque l'icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=IconeCadenas.gif) apparaît.

Pour défixer une colonne, survolez le côté gauche de l'entête de la colonne à défixer et cliquez lorsque l'icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=IconecadenasFerme.gif) apparaît.

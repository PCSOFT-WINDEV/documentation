
## WDMap : Imprimer des enregistrements
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000089"></a>
WDMap permet d'imprimer les enregistrements d'un fichier de données.

Lors de cette impression, il est possible de choisir les rubriques à imprimer.

Les enregistrements imprimés correspondent aux enregistrements visualisés sous WDMap. Pour imprimer uniquement une sélection d'enregistrements, il suffit de [sélectionner les enregistrements visualisés](../WDMap/3517011.md).



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000113"></a>
Pour imprimer tous les enregistrements visualisés :

1. Affichez le fichier de données contenant les enregistrements à imprimer (pour plus de détails, consultez [Visualiser les enregistrements présents dans un fichier de données](../WDMap/3517012.md)).

2. Sélectionnez les enregistrements à imprimer. Pour plus de détails, consultez [Sélection des enregistrements visualisés](../WDMap/3517011.md).
	**Rappel** : Seuls les enregistrements visualisés sous WDMap seront imprimés.

3. Sélectionnez l'option "Fichier .. Imprimer...". Un sélecteur des rubriques à imprimer s'affiche.

4. Sélectionnez les rubriques à imprimer et validez. Les enregistrements sélectionnés seront visualisés dans un aperçu avant impression et pourront être imprimés sur l'imprimante par défaut.




**Remarque** : Pour imprimer tous les enregistrements visualisés, il est également possible d'utiliser l'option "Imprimer" du menu contextuel de la table (clic droit).

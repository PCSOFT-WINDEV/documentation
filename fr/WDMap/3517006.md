
## WDMap : Réindexer un fichier de données HFSQL
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000124"></a>
WDMap permet à la fois de savoir si un fichier HFSQL doit être réindexé et à la fois de réindexer un fichier de données HFSQL.

Deux types de réindexation sont possibles :

- ***Réindexation normale*** : Réindexation classique (avec statistiques et sans compactage).

- ***Réindexation avec compactage*** : Réindexation avec compactage (ne conserve que les enregistrements actifs. Tous les enregistrements rayés ou supprimés sont définitivement supprimés).




**Rappel** :

- La réindexation ré-organise les index. Après réindexation, l'accès aux index est optimal. La réindexation peut être rendue nécessaire par la perte ou l'altération du fichier d'index.

- WDMap permet de réindexer simplement un fichier de données. Cependant, pour réindexer un fichier de façon plus complète, utilisez l'outil [WDOptimiseur](../WDOptimiseur/3519002.md).

- Il est également possible de réindexer un fichier depuis l'explorateur de Windows. Il suffit de sélectionner l'option "Optimiser" du menu contextuel du fichier à réindexer. La réindexation est alors effectuée grâce à l'outil [WDOptimiseur](../WDOptimiseur/3519002.md).

## Comment le faire ?
<a name="comment_faire_ELTTEXTE000154"></a>


### WDMap lancé depuis l'éditeur
<a name="wdmap_lance_depuis_editeur_ELTPARAGRAPHE000037"></a>Pour réindexer un fichier de données :

1. Affichez le fichier de données à réindexer (pour plus de détails, consultez [Visualiser les enregistrements présents dans un fichier de données](../WDMap/3517012.md)).

2. Pour réaliser :

	- une ***réindexation normale*** : sélectionnez l'option "Outils .. Réindexation .. Normale".

	- une ***réindexation avec compactage*** : sélectionnez l'option "Outils .. Réindexation .. Avec compactage".








<a name="NOTE2_2"></a>


### WDMap lancé par le programme "WDMAP.EXE"
<a name="wdmap_lance_par_programme_wdmapexe_ELTPARAGRAPHE000054"></a>Pour réindexer un fichier de données :

1. Dans l'assistant, sélectionnez l'option "Réindexer un fichier" et passez au plan suivant.

2. Sélectionnez le fichier à réindexer et validez.




Remarques :

- Si le fichier est protégé par un mot de passe (mot de passe défini soit à l'aide des fonctions [HPasse](../WDLang4/3044108.md), [HCréation](../WDLang4/3044255.md) ou [HCréationSiInexistant](../WDLang4/3044254.md), soit lors de la [modification automatique des données](../WDLang4/3044195.md)), précisez ce mot de passe.

- Si le fichier provient d'une analyse 5.5, précisez le chemin complet de l'analyse d'origine et le mot de passe de cette analyse.


## WDMap : Ajouter, modifier et supprimer les enregistrements d'un fichier de données
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000148"></a>
WDMap permet de réaliser des opérations simples (ajout, modification ou suppression) sur les enregistrements présents dans un fichier de données.

Ces manipulations peuvent être effectuées uniquement lors du développement de l'application (WDMap n'étant pas un outil redistribuable).

Il est ainsi possible par exemple d'ajouter, de modifier ou de supprimer des enregistrements avant même que les fenêtres correspondantes soient créées dans votre application. Vous pouvez également tester directement vos traitements.

**Attention** : Si les manipulations concernent des rubriques présentes dans des fichiers liés (sous l'éditeur d'analyses), il est conseillé d'activer la gestion de l'intégrité et la gestion des doublons. 
Par défaut, la gestion de l'intégrité et la gestion des doublons sont activées. 
La gestion de l'intégrité est activée uniquement si l'analyse d'origine du fichier est connue : la gestion de l'intégrité n'est pas activée si le fichier est ouvert directement sous l'explorateur avec WDMap.

**Remarques** :

- Pour activer la gestion de l'intégrité, sélectionnez l'option "Paramètres .. Gestion de l'intégrité". 
	**Attention** : Il est parfois nécessaire de localiser manuellement les fichiers liés au fichier manipulé. Pour localiser manuellement les fichiers manquants, cliquez sur l'icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=IconeGestionIntegrite.gif)
 et sélectionnez le répertoire de chaque fichier intervenant dans la gestion de l'intégrité du fichier en cours.

- Pour activer la gestion des doublons, sélectionnez l'option "Paramètres .. Gestions des doublons".

## Ajouter un enregistrement dans un fichier de données
<a name="ajouter_enregistrement_dans_fichier_donnees_ELTTEXTE000178"></a>
Pour ajouter un enregistrement dans un fichier de données :

1. Affichez le fichier de données dans lequel un enregistrement doit être ajouté (pour plus de détails, consultez [Visualiser les enregistrements présents dans un fichier de données](../WDMap/3517012.md)).

2. Sélectionnez l'option "Édition .. Ajouter un enregistrement" (ou cliquez sur la première ligne vide de la table de visualisation). Une nouvelle ligne est ajoutée dans la table de visualisation.

3. Saisissez dans cette nouvelle ligne les valeurs du nouvel enregistrement.
	**Remarque** : Pour passer d'une colonne à l'autre, utilisez les touches fléchées du clavier.

4. Cliquez sur une ligne différente pour valider l'ajout.
	**Remarque** : Si la gestion des doublons est activée et si une erreur de doublon est détectée, le mécanisme de sécurité de HFSQL est déclenché. Modifiez les valeurs nécessaires et validez.




**Remarque** : WDMap permet également de dupliquer un enregistrement :

1. Sélectionnez l'enregistrement à dupliquer.

2. Sélectionnez l'option "Édition .. Dupliquer un enregistrement" (ou raccourci Ctrl + D).

## Modifier un enregistrement d'un fichier de données
<a name="modifier_enregistrement_fichier_donnees_ELTTEXTE000202"></a>
Pour modifier un enregistrement d'un fichier de données :

1. Affichez le fichier de données dans lequel l'enregistrement doit être modifié (pour plus de détails, consultez [Visualiser les enregistrements présents dans un fichier de données](../WDMap/3517012.md)).

2. Sélectionnez l'enregistrement à modifier dans la table de visualisation.

3. Cliquez sur la colonne à modifier. Le curseur de saisie apparaît.

4. Saisissez la nouvelle valeur.

5. Cliquez sur une ligne différente pour valider la modification.
	**Remarque** : Si la gestion des doublons est activée et si une erreur de doublon est détectée, le mécanisme de sécurité de HFSQL est déclenché. Modifiez les valeurs nécessaires et validez.




**Traitements par lots** : WDMap permet de modifier en une seule opération tous les enregistrements visualisés :

1. Sélectionnez l'option "Édition .. Modifier ces enregistrements".

2. Sélectionnez l'option "&lt;Initialisation&gt;" dans les lignes correspondant aux rubriques à modifier.

3. Précisez la ou les nouvelle(s) valeur(s).

## Supprimer un enregistrement dans un fichier de données
<a name="supprimer_enregistrement_dans_fichier_donnees_ELTTEXTE000226"></a>
Pour supprimer un enregistrement dans un fichier de données :

1. Affichez le fichier de données dans lequel l'enregistrement doit être supprimé (pour plus de détails, consultez [Visualiser les enregistrements présents dans un fichier de données](../WDMap/3517012.md)).

2. Sélectionnez l'enregistrement à supprimer dans la table de visualisation.

3. Sélectionnez l'option "Édition .. Supprimer l'enregistrement" (ou appuyez sur la touche Suppr).

4. Confirmez la suppression.
	**Remarque** : Un message d'erreur apparaît si l'intégrité est gérée et si une règle d'intégrité est appliquée sur l'enregistrement supprimé. L'enregistrement ne sera alors pas supprimé.




**Astuce** : Tous les enregistrements visualisés peuvent être supprimés en une seule opération par l'option "Édition .. Supprimer tous les enregistrements".

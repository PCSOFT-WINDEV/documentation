
## WDMap : Exporter des enregistrements à partir d'un fichier de données
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000091"></a>
WDMap permet d'exporter des enregistrements vers :

- un fichier Excel. Les enregistrements pourront par exemple être manipulés avec Excel (statistiques, calcul, ...).

- un fichier XML. Les enregistrements pourront ainsi être affichés sur une page Internet.

- un fichier texte.




Les enregistrements exportés correspondent aux enregistrements visualisés sous WDMap. Pour exporter uniquement une sélection d'enregistrements, il suffit de [sélectionner les enregistrements visualisés](../WDMap/3517011.md).



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000115"></a>
Pour exporter tous les enregistrements visualisés :

1. Affichez le fichier de données à partir duquel les enregistrements doivent être exportés (pour plus de détails, consultez [Visualiser les enregistrements présents dans un fichier de données](../WDMap/3517012.md)).

2. Sélectionnez les enregistrements à exporter. Pour plus de détails, consultez [Sélection des enregistrements visualisés](../WDMap/3517011.md).
	**Rappel** : Seuls les enregistrements visualisés sous WDMap seront exportés.

3. Sélectionnez l'option "Fichier .. Exporter" et sélectionnez le type d'exportation voulue (vers Excel, dans un fichier XML ou dans un fichier texte). Un sélecteur de fichiers s'affiche.

4. Saisissez le nom (et le chemin) du fichier généré. Il n'est pas nécessaire de saisir l'extension du fichier généré.




**Remarque** : Pour exporter tous les enregistrements visualisés, il est également possible d'utiliser l'option "Exporter" du menu contextuel de la table (clic droit).

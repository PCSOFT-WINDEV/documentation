
## Connecteur Natif DB2 pour WINDEV et WEBDEV
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000122"></a>
En option, WINDEV et WEBDEV proposent un Connecteur Natif à DB2 (également appelé "Accès Natif DB2"). Ce Connecteur Natif permet d'accéder à une base DB2 sans driver externe depuis un programme en WLangage.

**Attention** : Cette version du Connecteur Natif DB2 ne fonctionne qu'avec WINDEV et WEBDEV.





<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Configuration nécessaire
<a name="configuration_necessaire_ELTTEXTE000146"></a>


### Conditions nécessaires à l'utilisation du Connecteur Natif DB2
<a name="conditions_necessaires_utilisation_connecteur_natif_db2_ELTPARAGRAPHE000020"></a>

- DB2 version 7.1 minimum (Connecteur Natif testé par nos services avec la version 8.2 et la version 10). 

- Le client DB2 sur chaque machine accédant à DB2.

- La DLL ***DB2CLI.DLL*** doit être accessible :

	- à partir du poste de développement,

	- à partir du poste de déploiement.








<a name="NOTE3_2"></a>


### Les fichiers nécessaires
<a name="les_fichiers_necessaires_ELTPARAGRAPHE000033"></a>Les fichiers nécessaires à l'utilisation du Connecteur Natif DB2 sont :

- en 32 bits : 

	- ***DB2CLI.DLL***. Cette DLL est installée par la couche client 32 bits de DB2. Pour les conditions de diffusion de cette DLL, reportez-vous à la licence de DB2.

	- ***wd280db2.dll***. Nécessaire au Connecteur Natif à une base DB2 en mode 32 bits. Ce fichier est livré avec le Connecteur Natif pour DB2.









- en 64 bits : 

	- ***DB2CLIO.DLL***. Cette DLL est installée par la couche client 64 bits de DB2. Pour les conditions de diffusion de cette DLL, reportez-vous à la licence de DB2.

	- ***wd280db2_64.dll***. Nécessaire au Connecteur Natif à une base DB2 en mode 64 bits. Ce fichier est livré avec le Connecteur Natif pour DB2.

## Utilisation
<a name="utilisation_ELTTEXTE000176"></a>


### Pour utiliser le Connecteur Natif DB2 dans vos applications WINDEV ou vos sites WEBDEV
<a name="pour_utiliser_connecteur_natif_db2_dans_vos_applications_windev_vos_sites_webdev_ELTPARAGRAPHE000058"></a>Pour utiliser DB2 en natif, il suffit de programmer :

- Soit avec les fonctions "SQL" du WLangage (voir [Programmation à l'aide des fonctions SQL](../WDDB2/5510002.md)).

- Soit avec les fonctions "HFSQL" du WLangage (voir [Programmation à l'aide des fonctions HFSQL](../WDDB2/5510003.md)).




Remarque : N'oubliez pas de consulter [les limitations concernant l'utilisation de le Connecteur Natif DB2](../WDDB2/5510007.md).

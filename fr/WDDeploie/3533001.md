## Présentation
<a name="presentation_ELTTEXTE000122"></a>


### L'utilitaire WDDéploie
<a name="utilitaire_wddeploie_ELTPARAGRAPHE000011"></a>WDDéploie est un utilitaire permettant de déployer facilement un site statique ou PHP. Deux méthodes de déploiement sont possibles :

- **déploiement distant (par FTP)** : Conseillé lorsque le serveur n'est pas directement disponible.

- **déploiement par réseau** : Conseillé lorsque le serveur est facilement accessible.




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Lancement de WDDéploie
<a name="lancement_wddeploie_ELTTEXTE000146"></a>
WDDéploie peut être lancé :

- sous le volet "Outils", dans le groupe "Utilitaires Web", cliquez sur "WDDeploie".

- en lançant directement le programme "WDDeploie.EXE".




<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Conditions d'utilisation
<a name="conditions_utilisation_ELTTEXTE000170"></a>
WDDéploie est un outil redistribuable. WDDéploie peut être utilisé pour installer n'importe quel site statique (créé avec WEBDEV ou non).

La licence d'utilisation de WEBDEV s'applique intégralement.

<a name="NOTE4"></a>
<a name="NOTE4_1"></a>


## Fonctionnalités de WDDéploie
<a name="fonctionnalites_wddeploie_ELTTEXTE000194"></a>
WDDéploie permet de :

- installer les fichiers d'un site statique ou PHP sur un serveur Web.

- compresser et regrouper les fichiers présents sur un site distant sous forme d'archives.

- restaurer les fichiers contenus dans une archive.




Pour plus de détails, consultez [Fonctionnalités de WDDéploie](../WDDeploie/3533002.md).


### Installation de WDDéploie sur un autre poste
<a name="installation_wddeploie_sur_autre_poste_ELTPARAGRAPHE000066"></a>Les fichiers nécessaires à l'installation de WDDéploie sont les suivants :

- WDDeploie.exe

- wd280xxx.dll

- WDOutil.WDK

- WDDeploie.pdf

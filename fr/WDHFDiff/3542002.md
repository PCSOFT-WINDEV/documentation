


## WDHFDiff - Utilisation
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000164"></a>
WDHFDiff permet de comparer :

- la structure de deux fichiers de données HFSQL.

- les données de deux fichiers de données HFSQL.

- les fichiers de données communs présents dans deux répertoires différents.






<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comparaison de deux fichiers de données HFSQL
<a name="comparaison_deux_fichiers_donnees_hfsql_ELTTEXTE000194"></a>


### Comparer la structure de deux fichiers de données HFSQL
<a name="comparer_structure_deux_fichiers_donnees_hfsql_ELTPARAGRAPHE000024"></a>Pour comparer la structure de deux fichiers de données HFSQL : 

1. Sous le volet "Outils", dans le groupe "Base de données", cliquez sur "WDHFDiff".

2. Cliquez sur le bouton "Comparer 2 fichiers".

3. Sélectionnez les deux fichiers à comparer.

4. Saisissez si nécessaire les mots de passe des fichiers à comparer.

5. Cliquez sur le bouton "Comparer la structure". Les différences trouvées sont automatiquement listées :

	- différences au niveau des propriétés WLangage des fichiers comparés.

	- liste des rubriques présentes dans le premier fichier, dans le second fichier et communes aux deux fichiers.

	- différences au niveau des propriétés WLangage des rubriques des fichiers comparés.






<a name="NOTE2_2"></a>


### Comparer les données de deux fichiers de données HFSQL
<a name="comparer_les_donnees_deux_fichiers_donnees_hfsql_ELTPARAGRAPHE000053"></a>Pour comparer les données de deux fichiers de données HFSQL :

1. Sous le volet "Outils", dans le groupe "Base de données", cliquez sur "WDHFDiff".

2. Comparez la structure des deux fichiers de données. Pour plus de détails, consultez le paragraphe ci-dessus.

3. Cliquez sur le bouton "Comparer les données".

4. Sélectionnez le type de comparaison à réaliser :

	- comparaison par numéro d'enregistrement. Les enregistrements présents dans les fichiers de données seront comparés selon leur identifiant.

	- comparaison par clé unique. Les enregistrements présents dans les fichiers de données seront comparés selon la ou les clés uniques communes des deux fichiers.
			Remarque : Dans ce type de comparaison, les enregistrements rayés et/ou supprimés ne seront pas comparés.




5. Sélectionnez les différentes rubriques à comparer.
	Rappel : Le bouton "Tous" permet de sélectionner toutes les rubriques disponibles. Le bouton "Aucun" permet de déselectionner les rubriques sélectionnées.

6. Cliquez sur le bouton "OK". Les différences trouvées sont automatiquement listées.

7. Double-cliquez sur une différence pour connaître ses caractéristiques.

## Comparaison des fichiers de données communs de deux répertoires
<a name="comparaison_des_fichiers_donnees_communs_deux_repertoires_ELTTEXTE000224"></a>
Pour comparer les fichiers de données communs à deux répertoires différents :

1. Sous le volet "Outils", dans le groupe "Base de données", cliquez sur "WDHFDiff".

2. Cliquez sur le bouton "Comparer 2 répertoires".

3. Sélectionnez les deux répertoires à comparer.

4. Cliquez sur le bouton "Scanner". Les fichiers de données communs aux deux répertoires sélectionnés sont automatiquement listés.

5. Sélectionnez les différents fichiers de données à comparer.
	Rappel : Le bouton "Tous" permet de sélectionner tous les fichiers disponibles. Le bouton "Aucun" permet de déselectionner les fichiers sélectionnés.

6. Cliquez sur le bouton "Comparer". Le nombre de différences trouvées dans chaque fichier de données s'affiche automatiquement.

7. Double-cliquez sur un fichier pour connaître ses différences :

	- différences au niveau des propriétés WLangage des fichiers comparés.

	- liste des rubriques présentes dans le premier fichier, dans le second fichier et communes aux deux fichiers.

	- différences au niveau des propriétés WLangage des rubriques des fichiers comparés.




8. Cliquez sur le bouton "Comparer les données" pour effectuer une comparaison par numéro d'enregistrement sur les fichiers de données. Les différences trouvées sont automatiquement listées.

9. Double-cliquez sur une différence pour connaître ses caractéristiques.

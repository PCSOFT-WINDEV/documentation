<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000108"></a>
WDHFDiff est un utilitaire permettant de comparer des fichiers de données HFSQL.

WDHFDiff permet de comparer :

- la [structure de deux fichiers de données HFSQL](../WDHFDiff/3542002.md).

- les [données de deux fichiers de données HFSQL](../WDHFDiff/3542002.md).

- les [fichiers de données communs présents dans deux répertoires différents](../WDHFDiff/3542002.md).




Les fichiers de données manipulés doivent être au format HFSQL (HFSQL Classic ou Hyper File 5.5).



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Lancement de WDHFDiff
<a name="lancement_wdhfdiff_ELTTEXTE000138"></a>
WDHFDiff peut être lancé :

- depuis WINDEV, WEBDEV ou WINDEV Mobile : sous le volet "Outils", dans le groupe "Base de données", cliquez sur "WDHFDiff".

- en lançant directement le programme "WDHFDiff.EXE" présent dans le sous-répertoire "Programmes" du répertoire d'installation de WINDEV, WEBDEV ou WINDEV Mobile.



<a name="NOTE2_2"></a>


### Conditions d'utilisation
<a name="conditions_utilisation_ELTPARAGRAPHE000057"></a>WDHFDiff est un outil non redistribuable. WDHFDiff ne peut pas être installé avec les applications développées avec WINDEV, WEBDEV ou WINDEV Mobile.

Pour utiliser WDHFDiff, WINDEV, WEBDEV ou WINDEV Mobile doit obligatoirement être installé sur le poste en cours.




## WDSQL : Création et exécution d'un script
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000146"></a>
Un script correspond à une suite d'instructions SQL permettant de réaliser une opération précise (création d'une nouvelle table dans la base de données, insertion multiple d'enregistrements, ...).

Un script est composé de plusieurs lignes. Chaque ligne du script comporte un code SQL pouvant être exécuté indépendamment.

Un script doit être exécuté ligne par ligne, par un mode Batch.

**Remarque** : Les opérations suivantes sont possibles uniquement sur les fichiers accédés via un provider OLE DB, un driver ODBC ou un accès natif :

- Création de la structure d'un fichier / table.

- Modification de la structure d'un fichier / table.

- Suppression de la structure d'un fichier / table.




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Créer un script
<a name="creer_script_ELTTEXTE000170"></a>
Un script peut être créé :

- directement depuis la fenêtre principale de WDSQL,

- à l'aide de l'assistant de création d'une nouvelle table (script de création de table).




**Remarques** :

- Pour enregistrer un script, sélectionnez l'option "Fichier .. Enregistrer une requête" ou cliquez sur l'icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=WDSQL%20-%20HC%20N%B0006%201.gif).

- Pour imprimer le code SQL d'un script, sélectionnez l'option "Fichier .. Imprimer le source de la requête" ou cliquez sur l'icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=WDSQL%20-%20HC%20N%B0006%202.gif).

<a name="NOTE2_3"></a>


### Créer un script de création d'une nouvelle table dans la base de données en cours
<a name="creer_script_creation_une_nouvelle_table_dans_base_donnees_cours_ELTPARAGRAPHE000065"></a>Pour créer un script de création d'une nouvelle table dans la base de données en cours :

1. Sélectionnez l'option "Fichier .. Créer une requête" (ou cliquez sur l'icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=WDSQL%20-%20HC%20N%B0006%205.gif)). L'assistant se lance. 

2. Sélectionnez l'option "Un script de création d'une nouvelle table". Passez à l'étape suivante.

3. Saisissez le nom de la table à créer (option "Nom de la table").

4. Pour chaque rubrique de la table :

	- Saisissez le nom de la rubrique (option "Nom du champ").

	- Cliquez sur le bouton "Ajouter".

	- Spécifiez le type de la rubrique. Les types proposés dépendent de la base de données en cours.

	- Spécifiez la taille de la rubrique.

	- Indiquez si la rubrique doit être indexée.

	- Indiquez si la rubrique peut être nulle ou non.




5. Cochez si nécessaire l'option "Générer la commande DROP TABLE" pour générer automatiquement la commande "DROP TABLE".
	**Rappel** : La commande DROP TABLE permet de supprimer une table.

6. Validez l'assistant. Le code SQL du script de création de table s'affiche automatiquement dans la fenêtre principale de WDSQL.

## Exécuter un script
<a name="executer_script_ELTTEXTE000206"></a>
Pour exécuter un script :

1. Chargez si nécessaire le script à exécuter sous WDSQL (option "Fichier .. Charger une requête" ou icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=WDSQL%20-%20HC%20N%B0006%207.gif)).

2. Sélectionnez l'option "Exécution .. Exécuter un script en batch" (ou cliquez sur l'icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=WDSQL%20-%20HC%20N%B0006%203.gif)).

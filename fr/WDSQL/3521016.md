


## WDSQL : Convertir la structure d'une base de données HFSQL
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000103"></a>
WDSQL permet de convertir la structure d'une base de données HFSQL en script SQL. Ce script pourra ensuite être exécuté sur une base de données SQL pour créer la base de données correspondante à l'analyse HFSQL.

Cette option permet par exemple de simplifier la création d'une analyse sur une base Oracle à partir d'une analyse existante au format WINDEV.

**Remarque** : Pour convertir la structure d'une base de données HFSQL, il est nécessaire d'être connecté à une base SQL. En effet, cette connexion permet d'identifier les types de colonnes reconnus par la base SQL sélectionnée. Ces types de colonnes seront alors utilisés lors de l'exportation de la base HFSQL vers la base SQL.

## Comment le faire ?
<a name="comment_faire_ELTTEXTE000133"></a>


### Convertir la structure d'une base de données à partir de WDSQL
<a name="convertir_structure_une_base_donnees_partir_wdsql_ELTPARAGRAPHE000024"></a>Pour convertir la structure d'une base de données à partir de WDSQL :

1. Connectez-vous si nécessaire à une base SQL. Pour plus de détails, consultez [Connexion à une base de données](../WDSQL/3521010.md).

2. Sélectionnez l'option "Outils .. Conversion d'analyse" ou cliquez sur l'icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=WDSQL%20-%20HC%20N%B0006%206.gif). L'assistant de conversion d'analyse se lance.

3. Sélectionnez l'analyse WINDEV à convertir (fichier ".WDD").

4. Spécifiez si nécessaire le mot de passe de cette analyse (mot de passe en exécution défini dans la description de l'analyse).

5. Sélectionnez le répertoire contenant les fichiers de données de l'analyse à convertir et passez à l'étape suivante.

6. Sélectionnez les fichiers de données à convertir au format SQL et passez à l'étape suivante.

7. Le script de conversion généré s'affiche directement dans l'assistant.
	**Remarque** : Dans l'assistant, il est possible de :

	- modifier directement ce script,

	- rechercher une expression spécifiée dans ce script,

	- remplacer la ou les expressions trouvée(s) par une autre expression.




8. Validez. Le code SQL du script généré s'affiche automatiquement dans la fenêtre principale de WDSQL.




**Remarque** : Ce script peut ensuite être :

- modifié directement dans la fenêtre principale de WDSQL,

- enregistré (option "Fichier .. Enregistrer une requête" ou icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=WDSQL%20-%20HC%20N%B0006%201.gif)),

- exécuté (option "Exécution .. Exécuter un script en batch" ou icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=WDSQL%20-%20HC%20N%B0006%203.gif)).

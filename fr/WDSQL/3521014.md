
## WDSQL : Exécution d'une requête SQL
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000177"></a>
WDSQL permet d'exécuter toutes les requêtes SQL créées avec WDSQL.


| Lors de l'exécution de : | Résultat : |
| --- | --- |
| <br><br>- Une requête de sélection<br><br><br> | Le résultat de la requête peut être visualisé :<br><br>- sous forme d'une table (icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=WDSQL%20-%20HC%20N%B0006%209.gif)<br> ou option "Exécution .. Exécuter en mode table").<br><br>- sous forme de fiches (icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=WDSQL%20-%20HC%20N%B0006%2010.gif)<br> ou option "Exécution .. Exécuter en mode fiche").<br><br><br>**Remarque** : Le résultat peut également [être exporté et/ou imprimé](../WDSQL/3521004.md). |
| <br><br>- Une requête d'insertion (icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=WDSQL%20-%20HC%20N%B0006%209.gif)<br> ou ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=WDSQL%20-%20HC%20N%B0006%2010.gif)<br> indifféremment).<br><br><br> | Les enregistrements sont directement ajoutés. |

## Exécuter une requête de sélection en mode table
<a name="executer_une_requete_selection_mode_table_ELTTEXTE000201"></a>


### Présentation
<a name="presentation_ELTPARAGRAPHE000064"></a>Lors de l'exécution d'une requête de sélection en mode table, le résultat de la requête est présenté dans une table. Cette table permet d'obtenir une vision globale du résultat de la requête.

Par exemple :


![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=WDSQL%20-%20HC%20N%B0008.gif)



<a name="NOTE2_2"></a>


### Exécuter une requête de sélection en mode table :
<a name="executer_une_requete_selection_mode_table_ELTPARAGRAPHE000076"></a>Pour exécuter une requête de sélection en mode table :

1. Chargez si nécessaire la requête à exécuter sous WDSQL (option "Fichier .. Charger une requête" ou icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=WDSQL%20-%20HC%20N%B0006%207.gif)
).

2. Sélectionnez l'option "Exécution .. Exécuter en mode table" (ou cliquez sur l'icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=WDSQL%20-%20HC%20N%B0006%209.gif)
). Le résultat de la requête s'affiche sous forme de table. Ce résultat peut être [exporté et/ou imprimé](../WDSQL/3521004.md).




**Remarque** : La visualisation du résultat sous forme de table :

- ***Rechercher un enregistrement*** : cliquez sur l'icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=WDSQL%20-%20HC%20N%B0008%201.gif)
 et saisissez les premières lettres de la valeur recherchée.

- ***Trier les enregistrements selon une rubrique*** : cliquez sur l'entête de la colonne à trier.

## Exécuter une requête de sélection en mode fiche
<a name="executer_une_requete_selection_mode_fiche_ELTTEXTE000231"></a>


### Présentation
<a name="presentation_ELTPARAGRAPHE000103"></a>Lors de l'exécution d'une requête de sélection en mode fiche, le résultat de la requête est présenté sous forme de fiches : chaque rubrique de l'enregistrement est visualisée dans son ensemble et il est possible de se déplacer d'une fiche à l'autre (grâce aux boutons ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=WDSQL%20-%20HC%20N%B0009%201.gif)
, ...).

Par exemple :


![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=WDSQL%20-%20HC%20N%B0009.gif)



<a name="NOTE3_2"></a>


### Exécuter une requête de sélection en mode fiche
<a name="executer_une_requete_selection_mode_fiche_ELTPARAGRAPHE000117"></a>Pour exécuter une requête de sélection en mode fiche :

1. Chargez si nécessaire la requête à exécuter sous WDSQL (option "Fichier .. Charger une requête" ou icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=WDSQL%20-%20HC%20N%B0006%207.gif)
).

2. Sélectionnez l'option "Exécution .. Exécuter en mode fiche" (ou cliquez sur l'icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=WDSQL%20-%20HC%20N%B0006%2010.gif)
). Le résultat de la requête s'affiche sous forme de fiche. Ce résultat peut être [exporté et/ou imprimé](../WDSQL/3521004.md).

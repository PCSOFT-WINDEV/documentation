## Présentation
<a name="presentation_ELTTEXTE000111"></a>
Le Centre de Contrôle Utilisateur permet de gérer les applications déployées sur un réseau d'entreprises. A partir de ce centre de contrôle, les utilisateurs peuvent installer les applications qui leur sont dédiées.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Lancement du Centre de Contrôle Utilisateur
<a name="lancement_centre_controle_utilisateur_ELTTEXTE000135"></a>


### Lancement du Centre de Contrôle Utilisateur
<a name="lancement_centre_controle_utilisateur_ELTPARAGRAPHE000018"></a>Le Centre de Contrôle Utilisateur peut être lancé :

- sous le volet "Outils", dans le groupe "Centres de Contrôle", cliquez sur "Contrôle Utilisateur".

- en lançant directement le programme "CCUser.EXE". 



<a name="NOTE2_2"></a>


### Conditions d'utilisation
<a name="conditions_utilisation_ELTPARAGRAPHE000040"></a>Le Centre de Contrôle Utilisateur est un outil redistribuable.

Il peut également être utilisé en réseau. Il suffit de copier les fichiers suivants dans un répertoire réseau :

- CCUSER.EXE,

- CCDEV.WDK, 

- WDOutil.WDK,

- CCxxxAdminExe.WDK,

- le framework de WINDEV (wdxxx\*.dll).

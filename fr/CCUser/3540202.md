


## Centre de Contrôle Utilisateur : Applications réseau disponibles
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000112"></a>
Le Centre de Contrôle Utilisateur permet de lister les applications pour lesquelles une installation réseau a été créée.

Lors de la création de l'installation réseau de l'application, une option permet de référencer l'installation de référence de l'application dans le Centre de Contrôle Utilisateur.

Dans l'assistant de création de l'installation, à l'étape "Installation Réseau : contrôle des versions", le bouton "Avancé ..." permet de définir si l'installation doit proposer de référencer ou non l'application dans le Centre de Contrôle Utilisateur.
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CC_User_Option_Install.gif)
Cette option est cochée par défaut. Si vous ne désirez pas que l'installateur propose de référencer l'application, il est nécessaire de décocher l'option.

Lors de l'installation de la version de référence de l'application, une option permet de référencer l'application dans le Centre de Contrôle Utilisateur.
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CC_User_Option_Install2.gif)

## Comment le faire ?
<a name="comment_faire_ELTTEXTE000136"></a>


### Lister les applications réseau disponibles
<a name="lister_les_applications_reseau_disponibles_ELTPARAGRAPHE000027"></a>Au lancement du Centre de Contrôle Utilisateur, la liste des applications réseau disponibles est automatiquement mise à jour.

Cette liste est visible dans l'encadré "Applications réseau disponibles" en haut à gauche dans la fenêtre principale du Centre de Contrôle.
<a name="NOTE2_2"></a>


### Installer une application réseau
<a name="installer_une_application_reseau_ELTPARAGRAPHE000036"></a>Pour chaque application réseau disponible dans le Centre de Contrôle, il est possible de lancer l'installation de l'application sélectionnée en double-cliquant dessus.
<a name="NOTE2_3"></a>


### Localisation des applications réseau disponible 
<a name="localisation_des_applications_reseau_disponible_ELTPARAGRAPHE000043"></a>Un fichier de paramètre global permet au Centre de Contrôle Utilisateur de lister les installations réseau disponibles. Ce fichier est placé dans le dossier \\AdminExe\\ du dossier partagé des Centres de Contrôle. Ce fichier se nomme "apps.ini".

L'emplacement du dossier partagé des Centres de Contrôle est paramétrable :

- sur le poste de développement : 

	- Affichez les options générales de WINDEV : sous le volet "Accueil", dans le groupe "Environnement", déroulez "Options" et sélectionnez "Options générales de WINDEV". 

	- Cliquez sur le bouton "Paramètres des Centres de Contrôle".




- sur un site de production : à l'étape "Référencement de l'application dans le Centre de Contrôle Utilisateur" (étape visible ci-dessus), au cours de l'installation de la version de référence de l'application sur le serveur.

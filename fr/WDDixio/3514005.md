
## Installer WDDIXIO
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000062"></a>
WDDIXIO est installé automatiquement lors de l'installation de WDMSG ou de WDINT.

Pour installer uniquement WDDIXIO, il est nécessaire d'installer et d'exécuter le fichier "InstallWDDixio.EXE" (présent dans le package d'installation de WDMSG ou WDINT). 

**Remarque** : Par défaut, le dictionnaire fourni avec WDDIXIO est automatiquement installé. Pour utiliser un dictionnaire personnalisé (créé et modifié avec WDTRAD) : 

1. Dans le répertoire d'installation de WDTrad, copiez le sous-répertoire "Dictionnaire" (contenant le dictionnaire personnalisé).

2. Collez ce sous-répertoire dans le répertoire d'installation de WDDIXIO.

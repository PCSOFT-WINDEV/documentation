## Présentation
<a name="presentation_ELTTEXTE000100"></a>
WDDIXIO est un outil de saisie de traduction des messages associés à une application. 

WDDIXIO facilite le travail de saisie de la traduction des messages et limite les risques d'erreurs de traduction.

WDDIXIO permet de :

- proposer des traductions correspondant au message à traduire.

- traduire les expressions d'une application de façon homogène.




WDDIXIO est accessible directement depuis les différentes fenêtres de description WINDEV, WEBDEV et WINDEV Mobile (bouton "Traduire" ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Combo_des_saisie_btn_multilangue.GIF)
).


![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=WDDixio%20-%20HC%20N%B0001.gif)


**Remarques** :

- Par défaut, WDDIXIO est livré avec un dictionnaire français - anglais contenant la traduction de plus de 6000 messages. Ce dictionnaire peut être enrichi grâce à WDTRAD.

- Pour plus de détails sur l'installation de WDDIXIO, consultez [Installer WDDIXIO](../WDDixio/3514005.md).

## Comment traduire les messages utilisés par votre projet avec WDDIXIO ?
<a name="comment_traduire_les_messages_utilises_par_votre_projet_avec_wddixio_ELTTEXTE000124"></a>
Pour traduire les différents messages utilisés par votre projet : 

1. Paramétrez l'utilisation de WDDIXIO dans votre produit. 

2. Paramétrez les options de WDDIXIO. 

3. Traduisez les différents éléments voulus. 

4. Récupérez la traduction. 


Pour plus de détails sur l'utilisation de WDDIXIO, consultez [Utiliser WDDIXIO](../WDDixio/3514006.md).

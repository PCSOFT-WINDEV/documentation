


## CaptchaAffiche (Fonction)

***En anglais : CaptchaDisplay***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Affiche un nouveau Captcha dans un champ Captcha. Si un captcha était déjà affiché dans le champ, ses paramètres sont réinitialisés : 

- nouveau texte.

- nouvelle déformation de texte, ...



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Propose un nouveau captcha à l'utilisateur
CaptchaAffiche(CPTCH_Captcha1)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`CaptchaAffiche(<Champ Captcha>)`
---

**`<Champ Captcha> : (Nom de champ)`**

Nom du champ Captcha dans lequel le captcha doit être affiché.

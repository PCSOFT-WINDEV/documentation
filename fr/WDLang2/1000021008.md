


## jQueryExécute (Fonction)

***En anglais : jQueryExecute***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Exécute une méthode Javascript de la librairie jQuery sur un élément de la page.

Remarques : 

- La fonction [jQuery](../WDLang2/1000021070.md) permet d'exécuter plusieurs méthodes chaînées. 

- Cette fonction est une fonctionnalité avancée : son utilisation nécessite une bonne connaissance du JavaScript et de la librairie jQuery



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Le champ CELL_Carré explose en morceaux avant de disparaître
jQueryExécute(CELL_Carré,"effect","explode")

// Modification de l'opacité du champ LIB_1
jQueryExécute(LIB_1, "fadeTo", "opacity", 0.5)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = jQueryExécute(<Champ manipulé> [, <Catégorie>] , <Fonction jQuery> [, <Paramètre 1> [... [, <Paramètre N>]]])`
---

**`<Résultat> : (Type attendu)`**

Valeur de retour de la méthode jQuery appelée.

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler dans la page. 

Remarque : L'option "Le champ peut être superposé" doit être cochée pour le champ (onglet "UI" de la fenêtre de description du champ).

**`<Catégorie> : (Chaîne de caractères optionnelle)`**

Nom de la catégorie de fonctions dans la librairie jQuery. 

**`<Fonction jQuery> : (Chaîne de caractères)`**

Nom de la fonction jQuery à exécuter. 

Attention : Le Javascript étant sensible à la casse, le nom de la fonction doit respecter la casse utilisée dans la librairie jQuery.

**`<Paramètre 1> : (Chaîne de caractères optionnelle)`**

Premier paramètre attendu par la fonction jQuery. 



**`<Paramètre N> : (Chaîne de caractères optionnelle)`**

Nième paramètre attendu par la fonction jQuery.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Paramètres de la fonction jQuery
<a name="parametres_fonction_jquery_ELTPARAGRAPHE000053"></a>Si une procédure (également appelée "Callback") doit être passée en paramètre à la fonction jQuery, la procédure navigateur doit être passée par référence (nom de la fonction précédée du signe "&"). Par exemple : 

```wl
jQueryExécute(BTN_Ajout, "click", &OnAjout)
```



### Chargement des librairies jQuery
<a name="chargement_des_librairies_jquery_ELTPARAGRAPHE000059"></a>Il existe plusieurs versions de la librairie jQuery. Il existe également plusieurs fichiers Javascript composant la librairie jQuery.

Il faut donc charger les librairies Javascript complémentaires. Pour cela, il existe 2 méthodes : 

- inclure le code de chargement en Javascript dans le code d'entête de la page. 

- télécharger sur le serveur les librairies Javascript jQuery et inclure les fichiers Javascript dans votre projet ou votre page.




**Méthode 1 : Pour inclure le code de chargement dans la page** : 

1. Affichez la fenêtre de description de la page (option "Description" du menu contextuel). 

2. Sélectionnez l'onglet "Avancé". 

3. Sélectionnez l'onglet "HTML". 

4. Copiez les lignes de code suivantes dans la zone "Code HTML ajouté à l'entête de page" :   



```html
<script src="http://code.jquery.com/jquery-1.9.1.js"></script>
<script src="http://code.jquery.com/ui/1.10.3/jquery-ui.js"></script>
```
Attention : Lors de l'écriture de cette page d'aide, les versions des librairies étaient 1.9.1 et 1.10.3. Il est conseillé de vérifier quelles sont les versions en cours. 

**Méthode 2 : Pour inclure les fichiers Javascript dans votre page (après téléchargement)** : 

1. Affichez la fenêtre de description de la page (option "Description" du menu contextuel). 

2. Sélectionnez l'onglet "Avancé". 

3. Sélectionnez l'onglet "HTML".

4. Cliquez sur le bouton "Ajouter" et sélectionnez les fichiers Javascript (.js) nécessaires. Attention : il est conseillé de vérifier quelles sont les versions en cours.

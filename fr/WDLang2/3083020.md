


## CheminNavigationSupprimeTout (Fonction)

***En anglais : BreadcrumbDeleteAll***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime tous les liens dans un champ Chemin de navigation.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
CheminNavigationSupprimeTout(SMP_MonChampChemin)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`CheminNavigationSupprimeTout(<Champ Chemin de navigation>)`
---

**`<Champ Chemin de navigation> : (Nom de champ)`**

Nom du champ Chemin de navigation dans lequel les liens doivent être supprimés.

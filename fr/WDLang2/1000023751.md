


## &lt;Image&gt;.PositionDéfilement (Fonction)

***En anglais : &lt;Image&gt;.ScrollingPosition***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Affiche le défilement à partir d'image spécifiée ou renvoie le nom de l'image affichée dans le défilement.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affiche l'image suivante (équivalent à NomImage.Suivante())
IMG_Bannière.PositionDéfilement(IMG_Bannière.PositionDéfilement() + 1)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Afficher un défilement à partir d'une image spécifiée

`<Champ Image>.PositionDéfilement(<Indice>)`
---

**`<Champ Image> : (Nom de champ)`**

Nom du champ à manipuler : 

- Champ Image.

- Champ Image cliquable.

- Champ Vignette.




**`<Indice> : (Entier)`**

Indice de l'image à afficher. Si ce paramètre est supérieur au nombre d'images à faire défiler (connu avec la fonction [&lt;Image&gt;.Occurrence](../WDLang2/1000023750.md)), la première image est affichée par défaut.


<a name="SYNTAXE2"></a>

### Connaître l'indice de l'image affichée dans un défilement

`<Résultat> = <Champ Image>.PositionDéfilement()`
---

**`<Résultat> : (Entier)`**

Indice de l'image actuellement affichée.

**`<Champ Image> : (Nom de champ)`**

Nom du champ à manipuler : 

- Champ Image.

- Champ Image cliquable.

- Champ Vignette.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Utilisée dans un code serveur, la fonction **&lt;Image&gt;.PositionDéfilement** retourne l'indice de l'image en cours uniquement si le code serveur a été exécuté par un champ de type "Envoyer les valeurs au serveur (submit)".

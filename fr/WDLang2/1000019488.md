
## NavigateurVersion (Fonction)

***En anglais : BrowserVersion***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Retourne la version du navigateur utilisé par l'internaute.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affiche une page particulière pour IE9 
SI NavigateurPlateforme() = NavigateurTypeInternetExplorer ALORS
	SELON NavigateurVersion(NavigateurVersionMajeure)
	CAS 9
		PageAffiche(PAGE_Fiche_IE9)
	AUTRE CAS
		PageAffiche(PAGE_Fiche_IE)
	FIN
SINON
	PageAffiche(PAGE_Fiche)
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = NavigateurVersion([<Information>])`
---

**`<Résultat> : (Variant)`**



- Version du navigateur (le type varie selon l'information demandée) :

	- Si la constante *NavigateurVersionComplète* a été utilisée, &lt;Résultat&gt; est une chaîne de caractères. 

	- si la constante *NavigateurVersionMajeure* a été utilisée, &lt;Résultat&gt; est un entier.




- 0 si le navigateur utilisé par l'internaute est inconnu. 




**`<Information> : (Constante optionnelle de type Entier)`**

Information demandée :



|   |   |
| --- | --- |
| *NavigateurVersionComplète* | Numéro de version complet du navigateur utilisé par l'internaute. |
| *NavigateurVersionMajeure* <br>(Valeur par défaut) | Numéro de version majeure du navigateur utilisé par l'internaute. |







<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Dans le cas d'un site dynamique (non AWP), la valeur retournée correspond au navigateur utilisé lors de la connexion au site. Si l'utilisateur change de navigateur (par exemple, avec un copier-coller d'URL dans un second navigateur), ce changement n'est pas détecté.

- Si votre projet utilise les sessions prélancées, cette fonction ne doit pas être utilisée dans l'événement "Initialisation" du projet. Cette fonction doit être utilisée dans l'événement "Initialisation du projet après connexion au site".

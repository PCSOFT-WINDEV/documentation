
## AjouteFavori (Fonction)

***En anglais : AddFavorite***

<DIV class="specObsolete">
	<IMG style="float:left;margin-right:10px;" src="https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=warning_big.png"><B>Avertissement</B><BR>
	 Cette fonction est conservée uniquement par compatibilité. En effet, cette fonction gère uniquement les favoris pour Internet Explorer. Depuis le 15 juin 2022, l'application de bureau Internet Explorer 11 n'est plus prise en charge sur certaines versions de Windows 10. 
</DIV><a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ajoute une adresse Internet dans la liste des favoris de l'internaute.

**Attention** : Cette opération fonctionne uniquement sur les navigateurs Internet Explorer.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ajoute le favori
SI Position(NavigateurNom(), "Netscape") = 0 ALORS
	AjouteFavori("PC SOFT, éditeur de WEBDEV", "wwww.pcsoft.fr")
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`AjouteFavori(<Titre> , <Adresse Internet>)`
---

**`<Titre> : (Chaîne de caractères)`**

Titre du favori visible dans le menu "Favoris" de Internet Explorer.

**`<Adresse Internet> : (Chaîne de caractères)`**

Adresse Internet du favori. Cette adresse peut être :

- une adresse IP au format http://xxx.xxx.xxx.xxx

- une URL (par exemple http://www.pcsoft.fr/webdev/)

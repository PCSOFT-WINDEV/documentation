
## NavigateurHauteur (Fonction)

***En anglais : BrowserHeight***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la hauteur en pixels de la zone cliente dans laquelle la page est affichée.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Si la taille disponible est trop faible.
SI NavigateurHauteur() < 500 ALORS
	...
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = NavigateurHauteur()`
---

**`<Résultat> : (Entier)`**

Hauteur de la zone cliente en pixels.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Cette hauteur inclut les éventuelles barres de défilement.

- Dans le cas d'une page affichée dans un champ de type "iFrame", la hauteur retournée est celle correspondant à la dimension du champ iFrame.

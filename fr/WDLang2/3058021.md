


## PagePosition (Fonction)

***En anglais : PagePosition***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Fait défiler une page pour positionner un champ dans la partie visible de la page (le plus haut) dans le navigateur.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// ZR_REPETE est un champ Zone répétée. 
// Positionnement sur la dernière ligne du champ Zone répétée
PagePosition("", ZR_REPETE, ZR_REPETE.Occurrence)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`PagePosition(<Nom de la page> , <Nom du champ> [, <Indice>])`
---

**`<Nom de la page> : (Chaîne de caractères)`**

Nom de la page à faire défiler. 
Si le nom de la page correspond à une chaîne vide (""), la page en cours sera manipulée. Si le nom de la page est incorrect, la fonction **PagePosition** n'a aucun effet.

**`<Nom du champ> : (Chaîne de caractères)`**

Nom du champ à afficher dans la partie visible de la page spécifiée. Si le champ est déjà visible dans la page ou si la page est entièrement visible dans le navigateur, la fonction **PagePosition** n'a aucun effet. 

**`<Indice> : (Entier optionnel (pour les zones répétées et les tables mémoire))`**

Indice de la ligne où se trouve le champ. La fonction **PagePosition** n'a aucun effet : 

- si le champ n'est pas un champ Zone répétée ou un champ Table par programmation,

- si l'indice est supérieur au nombre de lignes du champ Table par programmation ou du champ Zone répétée spécifié.




## &lt;Variable adminWBAuth&gt;.CréeCompte (Fonction)

***En anglais : &lt;WBAdminAuth variable&gt;.CreateAccount***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Crée un compte WEBDEV sur un Serveur d'application WEBDEV.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
wbAuthLogin est un adminWBAuth
wbAuthLogin.AdresseServeur = "https://mon-serveur.interne/"
wbAuthLogin.Utilisateur = "WWW\DATA"
wbAuthLogin.MotDePasse = "www"

compteWB est un adminWBCompte
compteWB.Utilisateur = "USER_X"
compteWB.ServeurVirtuel = compteWB.Utilisateur

RésultatCréation est un booléen = wbAuthLogin.CreeCompte(compteWB, "MDPUSERX")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Serveur d'application WEBDEV>.CréeCompte(<Compte> , <Mot de passe>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le compte a été créé, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Serveur d'application WEBDEV> : (Variable de type adminWBAuth)`**

Nom de la variable de type [adminWBAuth](../WDLang2/1410089328.md) décrivant le serveur et le compte administrateur utilisé pour interroger le Serveur d'application WEBDEV. 
Attention : Pour ajouter des comptes WEBDEV, le compte Windows utilisé doit être administrateur du poste Serveur.

**`<Compte> : (Variable de type adminWBCompte)`**

Nom de la variable de type [adminWBCompte](../WDLang2/1410089382.md) correspondant à la description du compte à créer.

**`<Mot de passe> : (Chaîne de caractères)`**

Mode de passe du compte à créer.




## Fonctions de pilotage du Serveur d'application WEBDEV
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions de gestion des utilisateurs et des connexions sur un Serveur d'application WEBDEV sont les suivantes :



|   |   |
| --- | --- |
| [AdminWBCréeCompte](../WDLang2/1410089450.md) | Crée un compte WEBDEV sur un Serveur d'application WEBDEV. |
| [AdminWBDéconnecte](../WDLang2/1410089343.md) | Supprime une connexion sur un Serveur d'application WEBDEV. |
| [AdminWBListeCompte](../WDLang2/1410089449.md) | Renvoie la liste des comptes WEBDEV sur un Serveur d'application WEBDEV. |
| [AdminWBListeConnexion](../WDLang2/1410089342.md) | Renvoie la liste des connexions en cours sur un Serveur d'application WEBDEV. |
| [AdminWBModifieCompte](../WDLang2/1410089471.md) | Modifie les caractéristiques d'un compte WEBDEV sur un Serveur d'application WEBDEV. |
| [AdminWBSupprimeCompte](../WDLang2/1410089472.md) | Supprime un compte WEBDEV sur un Serveur d'application WEBDEV. |

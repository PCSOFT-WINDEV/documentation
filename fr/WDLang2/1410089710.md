
## Piloter le Serveur d'application WEBDEV
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000126"></a>
A partir de la version 28, il est possible de piloter à distance le Serveur d'application WEBDEV. Cette nouveauté permet à la fois d'automatiser des manipulations, de proposer ces manipulations à travers une UI personnalisée ainsi que de créer des outils personnalisés de monitoring et de statistiques.

Le pilotage peut s'effectuer depuis une application WINDEV ou un site WEBDEV.

Parmi les manipulations automatisables, vous bénéficiez de :

- Gérer un compte WEBDEV : créer, supprimer, lister les comptes, ...

- Gérer les connexions : liste des connexions, déconnecter des utilisateurs, connaître le nombre d'utilisateurs connectés, etc.

- ...

## Comment le faire ?
<a name="comment_faire_ELTTEXTE000150"></a>


### Gérer les comptes WEBDEV 
<a name="gerer_les_comptes_webdev_ELTPARAGRAPHE000024"></a>Pour gérer les comptes WEBDEV par pilotage, il est nécessaire de : 

1. Définir une variable de type [adminWBAuth](../WDLang2/1410089328.md). Cette variable permet de définir les caractéristiques attendues par l'API authentifiée de gestion à distance d'un Serveur d'application WEBDEV.   

2. Manipuler une variable de type [adminWBCompte](../WDLang2/1410089382.md).

	- Pour créer un compte, il suffit de définir toutes les caractéristiques de la variable [adminWBCompte](../WDLang2/1410089382.md) puis d'utiliser la fonction [AdminWBCréeCompte](../WDLang2/1410089450.md). 

	- Pour modifier ou supprimer un compte, il suffit de récupérer la variable [adminWBCompte](../WDLang2/1410089382.md) correspondant au compte voulu  via la fonction [AdminWBListeCompte](../WDLang2/1410089449.md) puis d'utiliser les fonctions [AdminWBModifieCompte](../WDLang2/1410089471.md) et [AdminWBSupprimeCompte](../WDLang2/1410089472.md). 







Attention : Pour ajouter des comptes WEBDEV, le compte Windows utilisé doit être administrateur du poste Serveur. 


### Gérer les connexions
<a name="gerer_les_connexions_ELTPARAGRAPHE000061"></a>Pour gérer les connexions par pilotage, il est nécessaire de : 

1. Définir une variable de type [adminWBAuth](../WDLang2/1410089328.md). Cette variable permet de définir les caractéristiques attendues par l'API authentifiée de gestion à distance d'un Serveur d'application WEBDEV.   

2. Manipuler une variable de type [adminWBConnexion](../WDLang2/1410089344.md). La liste des connexions actives peut être connue avec la fonction [AdminWBListeConnexion](../WDLang2/1410089342.md). Il est possible de supprimer une connexion via la fonction [AdminWBDéconnecte](../WDLang2/1410089343.md).

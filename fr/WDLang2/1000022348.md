
## ServeurAppliModifieTâchePlanifiée (Fonction)

***En anglais : AppServerModifyScheduledTask***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie la description d'une tâche planifiée existante.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Change la programmation de la tâche pour l'exécuter toutes les heures
MaTâchePlanifiée.Mois = "*"
MaTâchePlanifiée.JourDuMois = "*"
MaTâchePlanifiée.JourDeLaSemaine = "*"
MaTâchePlanifiée.Heure = "*"
MaTâchePlanifiée.Minute = "0"

SI PAS ServeurAppliModifieTâchePlanifiée(MaTâchePlanifiée) ALORS
	// Cas d'erreur (par exemple si la tâche n'existe plus)
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = ServeurAppliModifieTâchePlanifiée(<Tâche planifiée>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la tâche planifiée a été modifiée. 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour connaître le libellé de l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.




**`<Tâche planifiée> : (Variable de type serveurAppliTâchePlanifiée)`**

Nom de la variable de type [serveurAppliTâchePlanifiée](../WDLang2/1000022367.md) correspondant à la description de la tâche planifiée à modifier. 
Remarque : La tâche planifiée est recherchée par son nom. Si la tâche planifiée n'est pas trouvée, la fonction retourne une erreur.

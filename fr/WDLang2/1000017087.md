


## PageActiveGFI (Fonction)

***En anglais : PageActivateDDW***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Active ou désactive le GFI (Grisage des pages inaccessibles) lors de l'affichage d'une page modale (par exemple les pages permettant de dialoguer avec l'utilisateur).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
PageActiveGFI(Vrai)
PageTauxGFI(40)
OuiNon("Voulez vous sortir ? ")
PageActiveGFI(Faux)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = PageActiveGFI(<Activation du GFI>)`
---

**`<Résultat> : (Booléen)`**

État d'activation du GFI précédent :

- <u><u><u><u>Vrai</u></u></u></u> : le GFI était activé sur les pages du site.

- <u><u><u><u>Faux</u></u></u></u> : le GFI n'était pas activé sur les pages du site.




**`<Activation du GFI> : (Booléen)`**

Nouvel état d'activation du GFI :

- <u><u><u><u>Vrai</u></u></u></u> : le GFI est activé sur les pages du site.

- <u><u><u><u>Faux</u></u></u></u> : le GFI n'est pas activé sur les pages du site.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- **Taux de grisage** : Par défaut, le taux de grisage des pages est égal à 60%. Pour modifier ce taux, utilisez la fonction [PageTauxGFI](../WDLang2/1000017086.md).

- La valeur utilisée en code navigateur est initialisée par la valeur donnée en code serveur. Les modifications effectuées en code navigateur ne sont pas reportées en code serveur.


## SiteAdresse (Fonction)

***En anglais : SiteAddress***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie l'adresse internet de connexion à un site dynamique WEBDEV (en mode Session) situé sur le même serveur.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Génération d'un lien qui lance le Site "MoneSite2"
HTM_ChampHTML.Valeur = "<A HREF=" + ...
	SiteAdresse(MonSite2) + ">Cliquez-moi pour aller sur le Site 2</A> "
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = SiteAdresse([<Nom du site>])`
---

**`<Résultat> : (Chaîne de caractères Ansi)`**

Adresse du site. .

**`<Nom du site> : (Chaîne de caractères optionnelle)`**

Nom du site dynamique WEBDEV (en mode Session) dont on veut connaître l'adresse. Ce site dynamique doit être installé sur le même serveur que le site en cours.  

Si ce paramètre est égal à une chaîne vide (""), le nom du site correspond au site en cours.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Exemples d'utilisation
<a name="exemples_utilisation_ELTPARAGRAPHE000046"></a>

- Cette adresse pourra par exemple être utilisée dans un email contenant un lien vers le site.

- Cette adresse est également utile lorsque du code HTML contenant des actions spécifiques est généré par programmation.

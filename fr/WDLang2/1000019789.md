


## PopupFerme (Fonction)

***En anglais : PopupClose***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Masque un champ Popup affiché dans la page via la fonction [PopupAffiche](../WDLang2/1000019788.md).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
PopupFerme(POPUP_Question)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`PopupFerme([<Popup>])`
---

**`<Popup> : (Nom de champ)`**

Nom du champ Popup qui sera masqué. Ce champ Popup a été affiché par la fonction [PopupAffiche](../WDLang2/1000019788.md). 
Si ce n'est pas le cas, la fonction **PopupFerme** n'a aucune action.
Ce paramètre est optionnel dans le code d'un champ présent dans la popup. Si ce paramètre n'est pas précisé, la popup contenant le champ est fermée.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Les navigateurs ne rafraîchissent pas l'affichage au milieu d'un code Javascript : les fonctions [PopupAffiche](../WDLang2/1000019788.md) et **PopupFerme** doivent être utilisées dans des traitements/événements différents. Si ces fonctions sont utilisées dans le même traitement/événement, le champ Popup n'est pas affiché.

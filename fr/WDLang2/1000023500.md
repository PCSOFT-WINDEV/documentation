


## &lt;Zone répétée&gt;.EnrouleTout (Fonction)

***En anglais : &lt;Looper&gt;.CollapseAll***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Enroule toutes les ruptures déroulées d'un champ Zone répétée.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Clic sur le champ Bouton [4 étoiles uniquement]
ZR_HOTEL.EnrouleTout()
nLigne4Etoile est un entier
nLigne4Etoile = ZR_HOTEL.Cherche(ATT_NBETOILE, 4)
ZR_HOTEL.Déroule(nLigne4Etoile)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Champ Zone répétée>.EnrouleTout()`
---

**`<Champ Zone répétée> : (Nom de champ)`**

Nom du champ Zone répétée à manipuler.




## Remarques


### Conditions d'utilisation
<a name="conditions_utilisation_ELTPARAGRAPHE000167"></a>La fonction **&lt;Zone répétée&gt;.EnrouleTout** peut être utilisée indifféremment sur : 

- un champ Zone répétée fichier,

- un champ Zone répétée par programmation.


## PleinEcranDésactive (Fonction)

***En anglais : FullScreenDisable***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Désactive le mode "Plein écran" du navigateur. 

Remarque : Pour activer le mode "Plein écran", utilisez la fonction [PleinEcranActive](../WDLang2/1000021012.md). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
PleinEcranDésactive()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`PleinEcranDésactive()`

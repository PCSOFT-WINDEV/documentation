


## ZoneRépétéeCherchePartout (Fonction)

***En anglais : LooperSearchEverywhere***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Recherche une valeur dans tous les attributs d'un champ Zone répétée. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Code de clic du champ Bouton [Rechercher]
nPos est entier
nPos = ZoneRépétéeCherchePartout(ZR_Commande, SAI_Recherche, rechercheContient)
// Si trouvé
SI nPos > 0 ALORS
	// Sélectionne la répétition
	ZR_Commande = nPos
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = ZoneRépétéeCherchePartout(<Champ Zone répétée> , <Valeur recherchée> [, <Type de recherche> [, <Début>]])`
---

**`<Résultat> : (Entier)`**



- Indice de l'élément trouvé, 

- -1 si la recherche a échoué.




**`<Champ Zone répétée> : (Nom de champ)`**

Nom du champ Zone répétée dans lequel la recherche doit être effectuée. La recherche est effectuée dans tous les attributs.

**`<Valeur recherchée> : (Chaîne de caractères)`**

Valeur recherchée dans tous les attributs.

**`<Type de recherche> : (Constante optionnelle de type Entier)`**

Type de recherche à effectuer : 



|   |   |
| --- | --- |
| *rechercheCommencePar* | **Recherche générique du type "Commence par"** : <br>&lt;Résultat&gt; correspond à l'indice de la répétition dans laquelle des éléments ont une valeur commençant par &lt;Valeur recherchée&gt;. |
| *rechercheContient*<br>(Valeur par défaut) | **Recherche générique du type "Contient"** : <br>&lt;Résultat&gt; correspond à l'indice de la répétition dans laquelle des éléments ont une valeur contenant la &lt;Valeur recherchée&gt;. |
| *rechercheIdentique* | **Recherche à l'identique du type "Egale à"** : <br>&lt;Résultat&gt; correspond à l'indice des répétitions dans lesquelles des éléments ont une valeur strictement égale à &lt;Valeur recherchée&gt;. |



**`<Début> : (Entier optionnel)`**

Indice de départ pour la recherche (1 par défaut).



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Conditions d'utilisation
<a name="conditions_utilisation_ELTPARAGRAPHE000083"></a>

- La fonction **ZoneRépétéeCherchePartout** peut être utilisée indifféremment sur : 

	- un champ Zone répétée fichier. Cependant, cette fonction n'est pas disponible pour les champs Zone répétée fichier avec accès direct.  

	- un champ Zone répétée par programmation.




- La valeur est recherchée parmi les attributs qui sont reliés à une valeur texte visible (par exemple, [Valeur](../Proprietes/2510130.md) sur un champ de saisie ou [Libellé](../Proprietes/2510053.md) sur un champ Libellé).

- La recherche n'est pas effectuée dans les attributs cachés.



<a name="NOTE0_2"></a>


### Sensibilité à la casse
<a name="sensibilite_casse_ELTPARAGRAPHE000106"></a>Aucune distinction n'est faite entre les minuscules et les majuscules et ce même si la recherche est effectuée à l'identique.

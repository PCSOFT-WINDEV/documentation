


## &lt;Bandeau défilant&gt;.Dernier (Fonction)

***En anglais : &lt;Sliding Banner&gt;.Last***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Affiche le dernier plan d'un champ Bandeau défilant.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affiche le dernier plan du bandeau (la dernière bannière)
BAN_Bannière.Dernier()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Champ Bandeau défilant>.Dernier()`
---

**`<Champ Bandeau défilant> : (Nom de champ)`**

Nom du champ Bandeau défilant à manipuler.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Les plans invisibles du champ Bandeau défilant sont ignorés.

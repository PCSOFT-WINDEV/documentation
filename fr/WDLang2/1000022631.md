


## &lt;Bandeau défilant&gt;.LanceDéfilement (Fonction)

***En anglais : &lt;Sliding Banner&gt;.StartSliding***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Active le défilement automatique sur un champ Bandeau défilant.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Démarre le défilement
BAN_Bannière.LanceDéfilement()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Champ Bandeau défilant>.LanceDéfilement()`
---

**`<Champ Bandeau défilant> : (Nom de champ)`**

Nom du champ Bandeau défilant à manipuler.




## PageFermeDialogue (Fonction)

***En anglais : PageCloseDialog***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ferme la page courante. Cette page a été ouverte avec la fonction [PageAfficheDialogue](../WDLang2/1000017124.md). Une valeur de retour peut être renvoyée à la page appelante.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ferme la page utilisée pour le dialogue en renvoyant la valeur Vrai
PageFermeDialogue(Vrai)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`PageFermeDialogue([<Valeur renvoyée>])`
---

**`<Valeur renvoyée> : (Type correspondant à la valeur renvoyée (optionnel))`**

Valeur renvoyée par la page lors de sa fermeture. Cette valeur est récupérée uniquement si la page a été ouverte avec la fonction [PageAfficheDialogue](../WDLang2/1000017124.md). Si la page n'a pas été ouverte avec la fonction [PageAfficheDialogue](../WDLang2/1000017124.md), une erreur est retournée.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- **PageFermeDialogue**() est équivalent à [ContexteFerme](../WDLang2/3058003.md)("").

- **PageFermeDialogue**(Valeur) est équivalent à [ContexteFerme](../WDLang2/3058003.md)("", Valeur).

- Si votre projet utilise les sessions prélancées, cette fonction ne doit pas être utilisée dans l'événement "Initialisation" du projet. Cette fonction doit être utilisée dans l'événement "Initialisation du projet après connexion au site".

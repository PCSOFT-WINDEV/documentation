


## &lt;Captcha&gt;.Vérifie (Fonction)

***En anglais : &lt;Captcha&gt;.Verify***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Vérifie si la valeur saisie par l'utilisateur correspond à la chaîne affichée dans un champ Captcha.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Valide la saisie de l'utilisateur
TestCaptcha est un booléen
TestCaptcha = CPTCH_Captcha.Vérifie(SAI_VerifCaptcha) 
SI TestCaptcha = Vrai ALORS
	Info("Félicitations, le Captcha a été saisi correctement.")
SINON
	Erreur("Vérifiez votre saisie, les valeurs sont différentes.")
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Champ Captcha>.Vérifie(<Valeur à tester>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la valeur du captcha est correcte, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Champ Captcha> : (Nom de champ)`**

Nom du champ Captcha à manipuler.

**`<Valeur à tester> : (Chaîne de caractères Unicode)`**

Valeur saisie par l'utilisateur. Ce paramètre peut correspondre au nom du champ dans lequel l'utilisateur a saisi la valeur du Captcha.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Permissivité
<a name="permissivite_ELTPARAGRAPHE000179"></a>Certaines polices de caractères affichent des caractères différents en utilisant le même dessin ou des dessins très similaires. Par exemple, le L minuscule et le chiffre 1 sont souvent très proches : " l " " 1 ". 

La fonction **&lt;Captcha&gt;.Vérifie** prend en compte certaines similitudes généralement constatées et effectue la comparaison en acceptant une substitution sur ces caractères.

La fonction **&lt;Captcha&gt;.Vérifie** est insensible à la casse si le Captcha contient uniquement des majuscules ou uniquement des minuscules.

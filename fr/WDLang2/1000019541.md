


## Fonctions de gestion des champs Jauge
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions permettant de gérer les champs Jauge dans un site WEBDEV sont les suivantes :



|   |   |
| --- | --- |
| [JaugeActive](../WDLang2/1000019477.md) | Active le rafraîchissement d'une jauge via Ajax. |
| [JaugeDésactive](../WDLang2/1000019478.md) | Stoppe le rafraîchissement périodique d'une jauge via Ajax. |
| [JaugeExécute](../WDLang2/1000019481.md) | Lance un traitement navigateur relativement long et fait avancer une jauge via Ajax selon l'avancement de ce traitement. |

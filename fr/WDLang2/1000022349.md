
## ServeurAppliSupprimeTâchePlanifiée (Fonction)

***En anglais : AppServerDeleteScheduledTask***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime une tâche planifiée associée à l'application (site ou Webservice).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
uneTâchePlanifiée est un serveurAppliTâchePlanifiée
uneTâchePlanifiée.Nom = "MaTâche" 
// Tente de supprimer la Tâche planifiée dont le nom est "MaTâche"
ServeurAppliSupprimeTâchePlanifiée(uneTâchePlanifiée)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = ServeurAppliSupprimeTâchePlanifiée(<Tâche planifiée>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la tâche planifiée a été supprimée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour connaître le libellé de l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.




**`<Tâche planifiée> : (Type associé)`**

Tâche planifiée à supprimer. Ce paramètre peut correspondre : 

- à une variable de type [serveurAppliTâchePlanifiée](../WDLang2/1000022367.md). 

- au nom d'une procédure. La tâche planifiée associée au nom de cette procédure sera supprimée. 

- au nom d'une tâche planifiée. La tâche planifiée correspondante sera supprimée.

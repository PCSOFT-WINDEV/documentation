


## ImageDernier (Fonction)

***En anglais : ImageLast***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Affiche la dernière image du défilement.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affiche la dernière bannière
ImageDernier(IMG_Bannière)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`ImageDernier(<Champ Image>)`
---

**`<Champ Image> : (Nom de champ)`**

Nom du champ à manipuler :

- Champ Image.

- Champ Image cliquable.

- Champ Vignette.


## AJAXDisponible (Fonction)

***En anglais : AJAXAvailable***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de savoir si la technologie AJAX est utilisable dans le navigateur en cours.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// AJAX supporté sur le navigateur en cours ?
SI AJAXDisponible() = Vrai ALORS
	// Exécution de la procédure serveur "MaProc"
	// Dès que le résultat de la procédure "MaProc" sera disponible, 
	// la procédure "ProcRésultat" sera exécutée
	Res est un entier
	Res = AJAXExécuteAsynchrone(MaProc, ProcRésultat)
	// Les traitements suivants s'exécutent sans attendre le résultat
	...
SINON
	// Traitements sans utilisation de AJAX
	... 
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = AJAXDisponible()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la technologie AJAX est utilisable dans le navigateur en cours, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Anciens navigateurs
<a name="anciens_navigateurs_ELTPARAGRAPHE000040"></a>La fonction [AJAXDisponible](../WDLang2/3055112.md) permet de savoir si le navigateur en cours supporte la technologie AJAX. Si un traitement utilisant la technologie AJAX est exécuté sur un navigateur ne supportant pas cette technologie, le traitement s'exécute "comme si" il n'utilisait pas la technologie AJAX (rafraîchissement de la page entière par exemple).




## ZoneRépétéeEnrouleTout (Fonction)

***En anglais : LooperCollapseAll***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Enroule toutes les ruptures déroulées d'un champ Zone répétée. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Clic sur le champ Bouton [4 étoiles uniquement]
ZoneRépétéeEnrouleTout(ZR_HOTEL)
nLigne4Etoile est un entier
nLigne4Etoile = ZoneRépétéeCherche(ZR_HOTEL, ATT_NBETOILE, 4)
ZoneRépétéeDéroule(ZR_HOTEL, nLigne4Etoile)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`ZoneRépétéeEnrouleTout(<Champ Zone répétée>)`
---

**`<Champ Zone répétée> : (Nom de champ)`**

Nom du champ Zone répétée à manipuler.
Si ce paramètre est égal à chaîne vide (""), le champ Zone répétée en cours est manipulé.




## Remarques


### Conditions d'utilisation
<a name="conditions_utilisation_ELTPARAGRAPHE000048"></a>La fonction **ZoneRépétéeEnrouleTout** peut être utilisée indifféremment sur : 

- un champ Zone répétée fichier,

- un champ Zone répétée par programmation.




## &lt;Popup&gt;.Ferme (Fonction)

***En anglais : &lt;Popup&gt;.Close***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Masque un champ Popup affiché dans la page via la fonction [&lt;Popup&gt;.Affiche](../WDLang2/1000021654.md).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
POPUP_Question.Ferme()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Popup>.Ferme()`
---

**`<Popup> : (Nom de champ)`**

Nom du champ Popup qui sera masqué.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Les navigateurs ne rafraîchissent pas l'affichage au milieu d'un code Javascript : les fonctions [&lt;Popup&gt;.Affiche](../WDLang2/1000021654.md) et **&lt;Popup&gt;.Ferme** doivent être utilisées dans des traitements/événements différents. Si ces fonctions sont utilisées dans le même traitement/événement, le champ Popup n'est pas affiché.

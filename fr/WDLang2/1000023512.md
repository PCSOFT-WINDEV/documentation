


## &lt;Zone répétée&gt;.Modifie (Fonction)

***En anglais : &lt;Looper&gt;.Modify***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie les valeurs des attributs dans une répétition d'un champ Zone répétée. 

**Remarque** : Il est conseillé d'utiliser la fonction [&lt;Zone répétée&gt;.ModifieLigne](../WDLang2/1000023520.md). En effet, la fonction [&lt;Zone répétée&gt;.ModifieLigne](../WDLang2/1000023520.md) permet d'insérer :

- des caractères TAB dans les attributs/champs.

- des valeurs de n'importe quel type dans les attributs/champs.


![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) Pour les champs Zone répétée sans attributs, il est conseillé de modifier directement la valeur des champs concernés. Pour plus de détails, consultez le paragraphe "Modifier un champ Zone répétée sans attributs". 










<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Modification d'une répétition dans un champ Zone répétée
ZR_Zone1.Modifie(NomProduit + TAB + CodeProduit, Indice)
```
<a name="Exemple2"></a>

```wl
// Modifie une valeur dans le champ Zone répétée en 12ème position
ZR_Zone1.Modifie(Nom + TAB + Prenom + TAB + ...
	NumTel + TAB + NumFax, 12)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Champ Zone répétée>.Modifie(<Valeur> [, <Indice>])`
---

**`<Champ Zone répétée> : (Nom de champ)`**

Nom du champ Zone répétée à manipuler.

**`<Valeur> : (Chaîne de caractères)`**

Liste des valeurs modifiées. Ce paramètre doit être de la forme : 

```txt
<Valeur 1> + TAB + ... + <Valeur N>
```

Les différentes valeurs doivent être indiquées dans l'ordre des attributs (fenêtre de description du champ Zone répétée).
Pour ne pas spécifier de valeur pour un attribut, utilisez une chaîne vide ("") ou 0 (selon le type de l'attribut).

**`<Indice> : (Entier optionnel)`**

Indice de la répétition à modifier.

- Si ce paramètre n'est pas précisé, la modification sera effectuée sur l'élément en cours.

- Si &lt;Indice&gt; est supérieur au nombre de répétitions, la fonction **&lt;Zone répétée&gt;.Modifie** est équivalente à la fonction [&lt;Zone répétée&gt;.Ajoute](../WDLang2/1000023469.md).






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Conditions d'utilisation
<a name="conditions_utilisation_ELTPARAGRAPHE000286"></a>La fonction **&lt;Zone répétée&gt;.Modifie** peut être utilisée indifféremment sur : 

- un champ Zone répétée fichier,

- un champ Zone répétée par programmation. 



<a name="NOTE0_2"></a>


### Tri
<a name="tri_ELTPARAGRAPHE000317"></a>Si la fonction **&lt;Zone répétée&gt;.Modifie** modifie un attribut de tri du champ Zone répétée en cours, le tri sur cet attribut sera annulé.
<a name="NOTE0_3"></a>


### Manipulation d'un mémo binaire
<a name="manipulation_memo_binaire_ELTPARAGRAPHE000330"></a>Pour modifier un mémo binaire dans un champ Zone répétée, utilisez la fonction [&lt;Zone répétée&gt;.ModifieLigne](../WDLang2/1000023520.md).
<a name="NOTE0_4"></a>

### Modifier un champ Zone répétée sans attributs
<a name="modifier_champ_zone_repetee_sans_attributs_ELTPARAGRAPHE000347"></a>Pour modifier un champ Zone répétée sans attributs, il est conseillé de modifier directement la valeur des champs concernés.

Par exemple :

```wl
// Modification de la valeur des champs
// de la zone répétée "ZR_Zone1" pour la répétition 23
ZR_Zone1[23].ChampNom = "Hobbes"
ZR_Zone1[23].ChampPrénom = "Samantha"
```

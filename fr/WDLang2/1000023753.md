


## &lt;Image&gt;.Premier (Fonction)

***En anglais : &lt;Image&gt;.First***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Affiche la première image du défilement.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affiche la première bannière
IMG_Bannière.Premier()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Champ Image>.Premier()`
---

**`<Champ Image> : (Nom de champ)`**

Nom du champ à manipuler : 

- Champ Image,

- Champ Image cliquable,

- Champ Vignette.


## Obtention d'un certificat SSL pour IIS2 (2.0 ou +)
			



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>


## Les différentes étapes
<a name="les_differentes_etapes_ELTTEXTE000138"></a>
Pour obtenir un certificat SSL pour IIS 2 à 6, les différentes étapes sont les suivantes : 

1. [Création d'un certificat](#NOTE1_1)

2. [Demande de certification](#NOTE2_1)

3. [Certification du certificat sur le serveur](#NOTE3_1)




Si vous utilisez IIS 7 ou supérieur, consultez [Obtenir un certificat SSL avec IIS 7](../WDLang2/1000019454.md). 



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Etape n°1 : Création d'un certificat
<a name="etape_creation_certificat_ELTTEXTE000162"></a>
Pour créer un certificat :

1. Ouvrez le Gestionnaire des services Internet (option "Démarrer .. Exécuter": Inetmgr.exe).

2. Sélectionnez le site Web sur lequel s'appliquera la sécurisation (par exemple : "Site Web par défaut").

3. Faites un clic droit sur ce site et sélectionnez l'option "Propriétés" du menu contextuel.

4. Sélectionnez l'onglet "Sécurité de répertoire" :

	- Cliquez sur le bouton "Certificat de serveur". Un assistant se lance.

	- Choisissez l'option "Créer un certificat".

	- Sur le volet de l'assistant "Nom usuel de votre site" : laissez le nom de votre ordinateur.

	- L'assistant crée un fichier texte "certreq.txt" qui permet de demander votre certificat.




5. Quittez les propriétés du serveur Web.






<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Etape n°2 : Demande de certification
<a name="etape_demande_certification_ELTTEXTE000186"></a>
La demande de certification se fait à un organisme de certification de clé (exemple de société : www.verisign.com). Cet exemple décrit la procédure à suivre avec l'organisme Vérisign. Les étapes sont sensiblement les mêmes pour chaque organisme.

1. Dans votre navigateur, tapez : http://www.verisign.com

2. Cliquez sur "Free Trial". Cette option permet d'obtenir un certificat valide 15 jours gratuitement.

3. Remplissez le petit questionnaire puis cliquez sur "Submit".

4. Suivez pas à pas l'assistant de certification :

	- *Step 1 of 5* : Generate CSR : Cliquez sur "Continue".

	- *Step 2 of 5* : Submit CSR : Copiez le contenu du fichier créé précédemment "certreq.txt" dans la zone de saisie prévue à cet effet, puis cliquez sur "Continue".

	- *Step 3 of 5* : Complete Application : Le contenu du certificat est listé à l'écran, tapez vos coordonnées (Attention les accents ne sont pas acceptés, la plupart des informations doivent être les mêmes que dans le contenu du certificat. Exemple : la région, la société). Renseignez bien votre adresse email : c'est à cette adresse que vous sera envoyé le certificat. Cliquez sur "Accept".





Dans l'heure qui suit vous recevrez votre certificat.

- Recommencez à partir de l'étape 1.

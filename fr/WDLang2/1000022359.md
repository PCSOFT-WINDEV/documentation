
## GglAnalyticsAjouteDurée (Fonction)

***En anglais : GglAnalyticsAddTiming***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ajoute une durée auprès de Google Analytics. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
GglAnalyticsAjouteDurée("dépendances js 3d", "chargement", dDurée.EnMillisecondes)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`GglAnalyticsAjouteDurée(<Catégorie> , <Action> , <Durée> [, <Libellé> [, <Options>]])`
---

**`<Catégorie> : (Chaîne de caractères)`**

Nom de la catégorie. Permet de regrouper plusieurs actions différentes (par exemple "Action Javascript").

**`<Action> : (Chaîne de caractères)`**

Nom de l'action à identifier (par exemple "chargement page").

**`<Durée> : (Entier ou Durée)`**

Durée de l'action exprimée en millisecondes à envoyer à Google Analytics. Ce paramètre peut correspondre à : 

- un entier correspondant au nombre de millisecondes, 

- une variable de type Durée, 

- l'indication directe de la durée (par exemple 1 s ou 10 ms). 




**`<Libellé> : (Chaîne de caractères optionnelle)`**

Libellé additionnel permettant de simplifier le classement dans les rapports de Google Analytics (par exemple "Google CDN").
Par défaut, ce paramètre correspond à une chaîne vide. 

**`<Options> : (Variant optionnel)`**

Options supplémentaires dépendant de Google Analytics. Pour connaître les valeurs possibles, consultez la documentation de Google Analytics [https://developers.google.com/analytics/devguides/collection/analyticsjs/user-timings](https://developers.google.com/analytics/devguides/collection/analyticsjs/user-timings). 
Par défaut, ce paramètre correspond à Null.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Pour utiliser cette fonction, vous devez posséder et enregistrer dans le projet un identifiant de suivi Google Analytics. Pour plus de détails, consultez [WEBDEV et Google Analytics](../WDLang2/1000022374.md). 

- **Remarque** : Au 1er Juillet 2023, Google Analytics passe de Universal Analytics à Google Analytics 4. WEBDEV 28 gère les deux versions. Les fonctions WLangage ont été adaptées. Par défaut, pour les projets existants, le mode géré est Universal Analytics et pour les nouveaux projets, le mode géré est Google Analytics 4. Pour plus de détails, consultez [WEBDEV et Google Analytics](../WDLang2/1000022374.md).

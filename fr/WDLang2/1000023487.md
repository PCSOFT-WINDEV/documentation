


## &lt;Zone répétée&gt;.DébutRemplissage (Fonction)

***En anglais : &lt;Looper&gt;.StartFilling***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Indique le début du remplissage d'un champ Zone répétée par programmation. L'avancement du remplissage est représenté :

- soit par une jauge infinie affichée au bas du champ.

- soit par une fenêtre interne.




**Attention** : Avant d'utiliser cette fonction, vous devez activer le mécanisme de gestion du remplissage dans la fenêtre de description du champ. Pour plus de détails, consultez [Caractéristiques d'un champ Zone répétée dans une fenêtre](../WDChamp/1013294.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
ZR_MaZone.DébutRemplissage()

POUR i = 1 A 10000
	ZR_MaZone.AjouteLigne()
	Multitâche(-1)
FIN

ZR_MaZone.FinRemplissage()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Champ Zone répétée>.DébutRemplissage()`
---

**`<Champ Zone répétée> : (Nom de champ)`**

Nom du champ Zone répétée à manipuler.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- A la fin du traitement de chargement du champ Zone répétée, il est nécessaire d'appeler la fonction [&lt;Zone répétée&gt;.FinRemplissage](../WDLang2/1000023501.md) afin d'indiquer que le remplissage est terminé.

- Cette fonction est sans effet sur les champs Zone répétée reliés à une source de données de type fichier de données ou requête.




## &lt;Chemin de navigation&gt;.SupprimeLien (Fonction)

***En anglais : &lt;Breadcrumb&gt;.DeleteLink***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime un lien dans un champ Chemin de navigation.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SMP_MonChampChemin.SupprimeLien(5)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Champ Chemin de navigation>.SupprimeLien(<Numéro>)`
---

**`<Champ Chemin de navigation> : (Nom de champ)`**

Nom du champ Chemin de navigation dans lequel le lien doit être supprimé.

**`<Numéro> : (Entier)`**

Numéro du lien à supprimer. Ce numéro est compris entre 1 et le nombre d'occurrences du &lt;Champ Chemin de navigation&gt; (connu avec la propriété [Occurrence](../Proprietes/2510097.md)).

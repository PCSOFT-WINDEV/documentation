


## ContexteOuvre (Fonction)

***En anglais : ContextOpen***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ouvre un nouveau contexte de page sans renvoyer les informations vers le navigateur. Cette fonction permet de passer des paramètres à une des pages d'un frameset.

Si un contexte de même nom est déjà ouvert, celui-ci est automatiquement détruit (le code de fermeture de la page est exécuté) puis ouvert.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ouverture du contexte de la page PAGE_Page4
ContexteOuvre(PAGE_Page4)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`ContexteOuvre(<Page> [, <Paramètre 1> [... [, <Paramètre N>]]])`
---

**`<Page> : (Nom de page)`**

Nom de la page dont le contexte doit être ouvert. Si ce paramètre est égal à une chaîne vide (""), le nom de la page correspond à la page à laquelle appartient le traitement en cours.

**`<Paramètre 1> : (Type de la valeur envoyée à la page (optionnel))`**

Paramètre(s) attendu par le traitement d'ouverture de la page à afficher. Ces paramètres sont passés par référence et sont considérés comme des variables globales à la page.



**`<Paramètre N> : (Type de la valeur envoyée à la page (optionnel))`**

Paramètre(s) attendu par le traitement d'ouverture de la page à afficher. Ces paramètres sont passés par référence et sont considérés comme des variables globales à la page.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Fonctions ContexteOuvre, PageAffiche et PageActualise
<a name="fonctions_docparampagetitleshort_pageaffiche_pageactualise_ELTPARAGRAPHE000044"></a>La fonction **ContexteOuvre** a la même action que la fonction [PageAffiche](../WDLang2/3058008.md), mais ne renvoie pas la page vers le navigateur. Pour renvoyer la page vers le navigateur utilisez la fonction [PageActualise](../WDLang2/3058011.md).
<a name="NOTE0_2"></a>


### Passage de paramètres à la fonction ContexteOuvre
<a name="passage_parametres_fonction_docparampagetitleshort_ELTPARAGRAPHE000063"></a>Lors de l'ouverture d'un frameset dont une ou plusieurs pages attendent des paramètres, il est nécessaire avant d'afficher le frameset, d'ouvrir le contexte des pages du frameset en spécifiant les paramètres attendus par ces pages.
<a name="NOTE0_3"></a>


### Sessions prélancées
<a name="sessions_prelancees_ELTPARAGRAPHE000074"></a>Si votre projet utilise les sessions prélancées, cette fonction ne doit pas être utilisée dans l'événement "Initialisation" du projet. Cette fonction doit être utilisée dans l'événement "Initialisation du projet après connexion au site".

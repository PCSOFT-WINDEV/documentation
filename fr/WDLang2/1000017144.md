
## IdentifiantContexteAWP (Fonction)

***En anglais : IdentifierAWPContext***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie l'identifiant du contexte AWP. Cet identifiant peut par exemple être utilisé pour construire une URL.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ajoute le contexte dans une URL construite
LIEN_Dyn = NomPage + ".awp?client=" + Client + "&" + IdentifiantContexteAWP()
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = IdentifiantContexteAWP()`
---

**`<Résultat> : (Chaîne de caractères Ansi)`**



- Identifiant du contexte AWP,

- Chaîne vide ("") quand la fonction est sans objet, par exemple dans les cas suivants :

	- La page courante n'est pas une page AWP.

	- La page courante est une page AWP sans contexte.

	- La page courante est une page AWP avec un contexte qui n'est pas sauvé sur le disque.  

	- La transmission est réalisée par cookie. 









<a name="NOTE0"></a>

## Remarques
Si le mode de transmission par défaut est utilisé : 

- Au premier affichage de la page, les deux modes de transmission sont utilisés (transmission par cookie et par url). La fonction **IdentifiantContexteAWP** renvoie l'identifiant du contexte AWP. 

- Lors des affichages suivants : 

	- Si les cookies fonctionnent, le mode de transmission de l'identifiant de contexte AWP par cookie est activé, la fonction **IdentifiantContexteAWP** renvoie une chaîne vide (""). 

	- Si les cookies ne fonctionnent pas, le mode de transmission de l'identifiant de contexte AWP par url est activé. La fonction **IdentifiantContexteAWP** renvoie l'identifiant du contexte AWP. 








### Sessions prélancées
<a name="sessions_prelancees_ELTPARAGRAPHE000057"></a>Si votre projet utilise les sessions prélancées, cette fonction ne doit pas être utilisée dans l'événement "Initialisation" du projet. Cette fonction doit être utilisée dans l'événement "Initialisation du projet après connexion au site".




## &lt;Upload&gt;.FichierEnCours (Fonction)

***En anglais : &lt;Upload&gt;.CurrentFile***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Indique le fichier en cours d'upload via le champ Upload. Cette fonction est utile notamment pour les champs Upload multifichiers.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affiche le nom du fichier actuellement uploadé
LIB_UploadMulti_Fichier = UPL_UploadMulti[UPL_UploadMulti.FichierEnCours()]
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Champ Upload>.FichierEnCours()`
---

**`<Résultat> : (Entier)`**

Indice du fichier en cours d'upload via le champ Upload : 1 pour le premier fichier de la liste, 2 pour le second, ...

**`<Champ Upload> : (Nom de champ)`**

Nom du champ Upload à manipuler.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Dans quel événement utiliser la fonction **&lt;Upload&gt;.FichierEnCours** ?
<a name="dans_quel_evenement_utiliser_fonction_docparampagetitleshort_ELTPARAGRAPHE000174"></a>La fonction **&lt;Upload&gt;.FichierEnCours** peut être utilisée uniquement dans l'événement navigateur de "Progression du transfert". Cet événement est associé : 

- soit au bouton "Envoyer" pour un champ Upload monofichier. 

- soit au bouton "Ajouter" pour un champ Upload multifichier. 




Utilisée dans un autre événement, la fonction **&lt;Upload&gt;.FichierEnCours** renverra 0.
<a name="NOTE0_2"></a>


### Taille maximale des fichiers à uploader
<a name="taille_maximale_des_fichiers_uploader_ELTPARAGRAPHE000200"></a>**Par défaut,** la taille des fichiers à uploader est illimitée dans le moteur WEBDEV. Cette taille peut être limitée par le serveur d'application WEBDEV. 
Si le fichier à uploader dépasse la taille limite, le fichier n'est pas uploadé.
Pour modifier la taille maximale des fichiers à uploader dans le serveur d'application WEBDEV, il est nécessaire de modifier la base de registre sur le serveur :  

1. Pour modifier la taille des fichiers uploadés pour toutes les applications WEBDEV 28, sélectionnez la clé : HKEY_LOCAL_MACHINE\\SOFTWARE\\PC SOFT\\WEBDEV\\28.0 sur une machine 32 bits et HKEY_LOCAL_MACHINE\\SOFTWARE\\WOW6432Node\\PC SOFT\\WEBDEV\\28.0 sur une machine 64 bits.

2. Ajoutez la chaîne MAX_UPLOAD. Cette chaîne aura pour valeur la taille maximale des fichiers à uploader (en Ko).

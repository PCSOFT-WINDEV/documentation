


## &lt;Image&gt;.ArrêteDéfilement (Fonction)

***En anglais : &lt;Image&gt;.StopScrolling***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Arrête le défilement automatique des images.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Stoppe le défilement
IMG_Bannière.ArrêteDéfilement()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Champ Image>.ArrêteDéfilement()`
---

**`<Champ Image> : (Nom de champ)`**

Nom du champ à manipuler :

- Champ Image.

- Champ Image cliquable.

- Champ Vignette.


## NavigateurLargeur (Fonction)

***En anglais : BrowserWidth***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la largeur en pixels de la zone cliente dans laquelle la page est affichée. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Si la taille disponible est trop étroite
SI NavigateurLargeur() < 500 ALORS
...
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = NavigateurLargeur()`
---

**`<Résultat> : (Entier)`**



- En mode Zoning, correspond à la largeur de la zone cliente en pixels. 

- En mode "Responsive Web Design", correspond à la largeur en pixels de la tranche en cours dans une page.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- La largeur inclut les éventuelles barres de défilement.

- Dans le cas d'une page affichée dans un champ de type "iFrame", la largeur retournée est celle correspondant à la dimension du champ iFrame.

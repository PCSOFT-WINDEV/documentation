
## AJAXAppelAsynchroneEnCours (Fonction)

***En anglais : AJAXAsynchronousCallPending***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de savoir si une procédure serveur appelée par la fonction [AJAXExécuteAsynchrone](../WDLang2/3055115.md) est en cours d'exécution.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// AJAX supporté sur le navigateur en cours ?
SI AJAXDisponible() = Vrai ALORS
	// Exécution de la procédure serveur "MaProc"
	// Dès que le résultat de la procédure serveur "MaProc"
	// sera disponible, 
	// la procédure navigateur "ProcRésultat" sera exécutée
	Res est un entier
	Res = AJAXExécuteAsynchrone(MaProc, ProcRésultat)
	// Les traitements suivants s'exécutent sans attendre le résultat
	...
	// État d'avancement de la procédure serveur "MaProc"
	SI AJAXAppelAsynchroneEnCours(Res) = Vrai ALORS
		// Annulation de l'exécution 
		// de la procédure navigateur "ProcRésultat"
		// Le résultat de la procédure serveur "MaProc" 
		// ne pourra pas être traité
		AJAXAnnuleAppelAsynchrone(Res)
	FIN
SINON
	// Traitements sans utilisation de AJAX
	... 
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = AJAXAppelAsynchroneEnCours(<Identifiant de la procédure serveur>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la procédure serveur spécifiée est en cours d'exécution, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Identifiant de la procédure serveur> : (Entier)`**

Identifiant de la procédure serveur à manipuler. Cet identifiant a été défini grâce à la fonction [AJAXExécuteAsynchrone](../WDLang2/3055115.md).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Rappel
<a name="rappel_ELTPARAGRAPHE000047"></a>La fonction [AJAXAnnuleAppelAsynchrone](../WDLang2/3055110.md) permet d'annuler l'exécution automatique de la procédure navigateur appelée par la fonction [AJAXExécuteAsynchrone](../WDLang2/3055115.md).


<a name="NOTE0_2"></a>


### Anciens navigateurs
<a name="anciens_navigateurs_ELTPARAGRAPHE000061"></a>La fonction [AJAXDisponible](../WDLang2/3055112.md) permet de savoir si le navigateur en cours supporte la technologie AJAX. Si un traitement utilisant la technologie AJAX est exécuté sur un navigateur ne supportant pas cette technologie, le traitement s'exécute "comme si" il n'utilisait pas la technologie AJAX (rafraîchissement de la page entière par exemple).

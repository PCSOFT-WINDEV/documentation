


## ZoneRépétéeSupprime (Fonction)

***En anglais : LooperDelete***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime une répétition dans un champ Zone répétée. Le champ Zone répété est automatiquement rafraîchit. 










<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Suppression de la répétition 1
ZoneRépétéeSupprime(ZR_ZoneProd, 1)
```
<a name="Exemple2"></a>

```wl
// Suppression de la répétition en cours
ZoneRépétéeSupprime(ZR_ZoneProd)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`ZoneRépétéeSupprime(<Champ Zone répétée> [, <Indice> [, <Animation>]])`
---

**`<Champ Zone répétée> : (Nom de champ)`**

Nom du champ Zone répétée à manipuler.
Si ce paramètre est égal à chaîne vide (""), la suppression sera effectuée sur le champ Zone répétée en cours.

**`<Indice> : (Entier optionnel)`**

Indice de la répétition à supprimer.

- Si ce paramètre n'est pas précisé, la suppression sera effectuée sur la répétition en cours.

- Si &lt;Indice&gt; est supérieur au nombre de répétitions, la fonction **ZoneRépétéeSupprime** est sans effet.




**`<Animation> : (Constante optionnelle de type Entier)`**

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Gestion de l'animation lors de la suppression : 



|   |   |
| --- | --- |
| *zrAnimationAucune*<br>(Valeur par défaut) | Aucune animation n'est jouée lors de la suppression. |
| *zrAnimationSuppression* | Joue une animation lors de la suppression. Cette animation correspond à un effet de disparition par diminution de la hauteur de la répétition. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Conditions d'utilisation
<a name="conditions_utilisation_ELTPARAGRAPHE000135"></a>La fonction **ZoneRépétéeSupprime** peut être utilisée indifféremment sur : 

- un champ Zone répétée fichier,

- un champ Zone répétée par programmation. 





### Utilisation avec un champ Zone répétée fichier
<a name="utilisation_avec_champ_zone_repetee_fichier_ELTPARAGRAPHE000148"></a>Si le champ Zone répétée est basé sur un fichier de données : 

- l'enregistrement est supprimé dans le fichier de données. 

- la répétition suivante est sélectionnée (si la dernière répétition est supprimée, la répétition précédente sera sélectionnée). L'enregistrement sélectionné dans le fichier de données lié correspond à la répétition sélectionnée.




## ZoneRépétéeSelect (Fonction)

***En anglais : LooperSelect***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie l'indice de l'élément sélectionné dans un champ Zone répétée. L'élément sélectionné correspond à la répétition en cours sur laquelle le bandeau de sélection est affiché.

**Remarque** : La fonction **ZoneRépétéeSelect** peut être utilisée indifféremment sur :

- un champ Zone répétée fichier ou Zone répétée par programmation.

- un champ Zone répétée monosélection ou multisélection. Dans le cas des champs Zone répétée multisélection, il est nécessaire d'utiliser la fonction **ZoneRépétéeSelect** dans une boucle pour connaître toutes les répétitions sélectionnées.















<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Indice de la répétition sélectionnée dans le champ Zone répétée "ZR_Produit"
ResSelect = ZoneRépétéeSelect(ZR_Produit)
```
<a name="Exemple2"></a>

```wl
// Répétitions sélectionnées dans le champ Zone répétée par programmation multisélection
i est un entier
NbSélectionné est un entier = ZoneRépétéeSelectOccurrence(ZR_Zone1)
POUR i = 1 A NbSélectionné
	Trace("Elément sélectionné : " + ZoneRépétéeSelect(ZR_Zone1, i))
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = ZoneRépétéeSelect(<Champ Zone répétée> [, <Rang>])`
---

**`<Résultat> : (Entier)`**



- Indice de la répétition sélectionnée dans le champ Zone répétée spécifié, 

- -1 si aucune répétition n'est sélectionnée.




**`<Champ Zone répétée> : (Nom de champ)`**

Nom du champ Zone répétée à manipuler.
Si ce paramètre correspond à une chaîne vide (""), le champ Zone répétée auquel appartient l'événement en cours est manipulé.

**`<Rang> : (Entier optionnel)`**

Rang de la sélection dans le cas d'un champ Zone répétée multisélection. 
Si ce paramètre n'est pas spécifié, l'indice de la première répétition sélectionnée est renvoyé.
Par exemple, pour récupérer le premier élément sélectionné, &lt;Rang&gt; = 1. 
Pour récupérer le deuxième élément sélectionné, &lt;Rang&gt; = 2, ...


## FramesetActualise (Fonction)

***En anglais : FramesetRefresh***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Actualise un frameset affiché dans le navigateur de l'internaute à partir du contexte présent sur le serveur. Les framesets réactualisés correspondent aux framesets définis sous l'éditeur, dans leur état initial. Les pages initiales du frameset sont affichées.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Actualisation de Frameset1
FramesetActualise(Frameset1)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`FramesetActualise(<Nom du frameset>)`
---

**`<Nom du frameset> : (Chaîne de caractères)`**

Nom du frameset à actualiser. 
Si ce paramètre est égal à une chaîne vide (""), le nom du frameset est celui du frameset auquel appartient l'événement en cours.

**Attention** : Le contexte du frameset doit exister sur le serveur. Le contexte du frameset est créé par les fonctions [FramesetAffiche](../WDLang2/3058006.md) et [ContexteOuvre](../WDLang2/3058004.md).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Si la fonction **FramesetActualise** est utilisée sans paramètre dans le code d'initialisation du frameset, une erreur WLangage apparaît.

- Si votre projet utilise les sessions prélancées, cette fonction ne doit pas être utilisée dans l'événement "Initialisation" du projet. Cette fonction doit être utilisée dans l'événement "Initialisation du projet après connexion au site".

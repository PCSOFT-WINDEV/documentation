


## AdminWBListeCompte (Fonction)

***En anglais : WBAdminListAccount***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la liste des comptes WEBDEV sur un Serveur d'application WEBDEV.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
wbAuthLogin est un adminWBAuth
wbAuthLogin.AdresseServeur = "https://mon-serveur.interne/"
wbAuthLogin.Utilisateur = "WWW\DATA"
wbAuthLogin.MotDePasse = "www"

ComptesWB est un tableau de adminWBCompte = AdminWBListeCompte(wbAuthLogin)
POUR i = 1 _À_ ComptesWB.Occurrence  
	Trace(ComptesWB[i].Utilisateur)
	Trace(ComptesWB[i].UtilisateurApplication)
	Trace(ComptesWB[i].CompteAdministrateur)
	Trace(ComptesWB[i].CompteActif)
	Trace(ComptesWB[i].RépertoireUtilisateur)
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = AdminWBListeCompte(<Serveur d'application WEBDEV>)`
---

**`<Résultat> : (Tableau)`**

Tableau de variables de type [adminWBCompte](../WDLang2/1410089382.md) correspondant au différents comptes WEBDEV disponibles sur le Serveur d'application WEBDEV. 
En cas d'erreur, la variable [ErreurDétectée](../WDLang1/3087001.md) passe à <u><u><u><u>Vrai</u></u></u></u>.

**`<Serveur d'application WEBDEV> : (Variable de type adminWBAuth)`**

Nom de la variable de type [adminWBAuth](../WDLang2/1410089328.md) décrivant le serveur et le compte administrateur utilisé pour interroger le Serveur d'application WEBDEV.

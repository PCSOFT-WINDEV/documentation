
## HTMLClasseAjoute (Fonction)

***En anglais : HTMLClassAdd***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ajoute une classe dans les classes HTML d'un champ.

Remarque : Il est également possible de connaître et de modifier la valeur de l'attribut HTML "class" d'un champ via la propriété [ClasseHTML](../Proprietes/1000021530.md). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
HTMLClasseAjoute(SAI_Saisie1, "SaisieEnErreur")
```




## Syntaxe
<a name="SYNTAXE1"></a>

`HTMLClasseAjoute(<Champ> , <Classe à ajouter>)`
---

**`<Champ> : (Chaîne de caractères)`**

Nom du champ à manipuler.

**`<Classe à ajouter> : (Chaîne de caractères)`**

Nom de la classe à ajouter. Si la classe existe déjà, elle n'est pas ajoutée une seconde fois.

Si plusieurs classes doivent être ajoutées, ce paramètre doit contenir le nom des différentes classes séparé par des espaces. Chaque classe est ajoutée indépendamment.

Attention : Le nom des classes est sensible à la casse.

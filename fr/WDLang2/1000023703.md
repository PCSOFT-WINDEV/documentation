


## &lt;Zone répétée&gt;.SupprimeTout (Fonction)

***En anglais : &lt;Looper&gt;.DeleteAll***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime toutes les répétitions du champ Zone répétée par programmation.










<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Remise à blanc
ZR_Zone1.SupprimeTout()
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Champ Zone répétée>.SupprimeTout()`
---

**`<Champ Zone répétée> : (Nom de champ)`**

Nom du champ Zone répétée à manipuler.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Conditions d'utilisation
<a name="conditions_utilisation_ELTPARAGRAPHE000215"></a>La fonction **&lt;Zone répétée&gt;.SupprimeTout** peut être utilisée uniquement sur un champ Zone répétée par programmation.

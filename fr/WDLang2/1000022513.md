


## BandeauPremier (Fonction)

***En anglais : BannerFirst***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Affiche le premier plan d'un champ Bandeau défilant.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affiche la première bannière
BandeauPremier(BAN_Bannière)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`BandeauPremier(<Champ Bandeau défilant>)`
---

**`<Champ Bandeau défilant> : (Nom de champ)`**

Nom du champ Bandeau défilant à manipuler.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Les plans invisibles du champ Bandeau défilant sont ignorés.

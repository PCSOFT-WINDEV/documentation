
## NotificationAffiche (Fonction)

***En anglais : NotificationDisplay***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Affiche un message hors navigateur. 

Un navigateur peut afficher des informations en dehors de sa fenêtre. Par exemple, un navigateur sous Windows peut afficher en bas à droite du bureau une information de réception d'email. Même si le navigateur est iconisé, l'information sera visualisée. 

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Notification_web.gif)

<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affiche un message hors navigateur
NotificationAffiche("Messagerie", "Vous avez reçu un nouveau mail", ...
		 "Message.gif")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`NotificationAffiche(<Titre> , <Message> [, <Image>])`
---

**`<Titre> : (Chaîne de caractères)`**

Titre du message affiché. Si ce paramètre correspond à une chaîne vide (""), le titre ne sera pas affiché. 

**`<Message> : (Chaîne de caractères)`**

Message à afficher dans la notification. 

**`<Image> : (Chaîne de caractères optionnelle)`**

Image du message affiché. 

Dans une page dynamique, si l'image est dans le répertoire "_WEB" du projet, il faut préciser le chemin virtuel de ce répertoire, en utilisant par exemple la fonction [RépertoireWeb](../WDLang2/3012015.md). Par exemple : RépertoireWeb() + "MonImage.gif".





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Les notifications doivent être activées sous Windows sur le poste de l'utilisateur. Dans le cas contraire, la notification ne sera pas affichée. 

- Les notifications ne sont pas supportées par tous les navigateurs. Si le navigateur ne supporte pas les notifications, la notification ne sera pas affichée.

- Si les notifications ne sont pas autorisées, une demande d'autorisation sera affichée dans le navigateur.
	Attention : Si l'appel de la fonction **NotificationAffiche** est réalisé dans un bouton qui exécute du code serveur, la demande d'autorisation de notification du navigateur sera effacée par le rafraîchissement de la page : il sera alors impossible à l'utilisateur d'autoriser les notifications.


## adminWBCompte (Type de variable)

***En anglais : WBAdminAccount***
			
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **adminWBCompte** permet de définir toutes les caractéristiques avancées d'un un compte WEBDEV sur un Serveur d'application WEBDEV. Les caractéristiques de ce compte WEBDEV peuvent être définies et modifiées à l'aide de différentes propriétés WLangage. 

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
wbAuthLogin est un adminWBAuth
wbAuthLogin.AdresseServeur = "https://mon-serveur.interne/"
wbAuthLogin.Utilisateur = "WWW\DATA"
wbAuthLogin.MotDePasse = "www"

ComptesWB est un tableau de adminWBComptes = AdminWBListeCompte(wbAuthLogin)
POUR i = 1 _À_ ComptesWB.Occurrence
	Trace(ComptesWB[i].Utilisateur)
FIN
```

## Propriétés


### Propriétés spécifiques à la description des variables de type adminWBCompte 
<a name="proprietes_specifiques_description_des_variables_type_adminwbcompte_ELTPARAGRAPHE000026"></a>Les propriétés suivantes peuvent être utilisées pour manipuler un compte WEBDEV :

| Nom de la propriété | Type manipulé | Effet |
| --- | --- | --- |
| Adresse | Chaîne de caractères | Adresse de l'utilisateur du compte. |
| CompteActif | Booléen | <br><br>- <u><u><u>Vrai</u></u></u> si le compte est activé, <br><br>- <u><u><u>Faux</u></u></u> dans le cas contraire. <br><br><br> |
| CompteAdministrateur | Booléen | <br><br>- <u><u><u>Vrai</u></u></u> si le compte est un compte administrateur, <br><br>- <u><u><u>Faux</u></u></u> dans le cas contraire.<br><br><br> |
| CompteDéploiement | Booléen | <br><br>- <u><u><u>Vrai</u></u></u> si l'utilisateur peut déployer des sites et des services, <br><br>- <u><u><u>Faux</u></u></u> dans le cas contraire.<br><br><br> |
| CompteStatistique | Booléen | <br><br>- <u><u><u>Vrai</u></u></u> si l'utilisateur peut consulter les statistiques des sites, <br><br>- <u><u><u>Faux</u></u></u> dans le cas contraire.<br><br><br> |
| Divers | Chaîne de caractères | Remarques supplémentaires. |
| Email | Chaîne de caractères | Adresse email de l'utilisateur du compte. |
| NombreMaxConnexion | Entier | Nombre maximum total de connexions autorisées pour l'ensemble des sites et webservices de l'utilisateur.  |
| NombreMaxREST | Entier | Nombre maximum de connexions autorisées pour les webservices REST de l'utilisateur.  |
| NombreMaxSite | Entier | Nombre maximum de connexions autorisées pour les sites de l'utilisateur. |
| NombreMaxSOAP | Entier | Nombre maximum de connexions autorisées pour les webservices SOAP de l'utilisateur. |
| NombreMaxWebSocket | Entier | Nombre maximum de connexions autorisées pour les websockets de l'utilisateur. |
| NomPrénom | Chaîne de caractères | Nom et prénom de l'utilisateur. |
| Photo | Buffer | Photo de l'utilisateur. |
| RépertoireDonnées | Chaîne de caractères | Répertoire des fichiers de données des applications et sites WEBDEV associées à ce compte. |
| RépertoireREST | Chaîne de caractères | Répertoire des Webservices REST associés à ce compte. |
| RépertoireServeurVirtuel | Chaîne de caractères | Répertoire racine du serveur virtuel. |
| RépertoireSite | Chaîne de caractères | Répertoire de déploiement des sites. |
| RépertoireSOAP | Chaîne de caractères | Répertoire des Webservices SOAP associés à ce compte. |
| RépertoireUtilisateur | Chaîne de caractères | Répertoire des applications et sites WEBDEV associées à ce compte. |
| RépertoireWebSocket | Chaîne de caractères | Répertoire des serveurs Websocket associés à ce compte. |
| ServeurVirtuel | Chaîne de caractères | Serveur virtuel sur lequel les sites/services sont déployées. |
| Société | Chaîne de caractères | Société de l'utilisateur du compte. |
| Téléphone | Chaîne de caractères | Numéro de téléphone de l'utilisateur du compte. |
| Utilisateur | Chaîne de caractères | Nom du compte Windows utilisé pour l'installation (déploiement) et l'administration distante. Le mot de passe associé est celui du compte Windows. |
| UtilisateurApplication | Chaîne de caractères | Nom du compte Windows utilisé pour l'exécution des sites et webservices. Le mot de passe associé est celui du compte Windows. |

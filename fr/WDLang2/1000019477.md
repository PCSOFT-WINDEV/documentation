
## JaugeActive (Fonction)

***En anglais : ProgressBarEnable***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Active le rafraîchissement d'une jauge via Ajax.

**Attention** : Cette fonction ne peut pas être utilisée dans un site PHP ou AWP. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple
<a class="notetitle" target="_blank" href="$DOC$=1000019562&name=jaugeactive_fonction&product=WD">Voir des exemples supplémentaires</a>

```wl
// Affiche la jauge au milieu de l'écran dans une cellule avec GFI
CelluleAfficheDialogue(CELL_Jauge)
// Et lance le suivi de l'avancement de la copie
JaugeActive(JAUGE_Copie)
```



```wl
// -- Code serveur du bouton BTN_Traitement
POUR I = JAUGE_SansNom1.BorneMin _A_ JAUGE_SansNom1.BorneMax
	JAUGE_SansNom1 = I
	Multitâche(-1)
FIN

// -- Code navigateur du bouton BTN_Traitement
JaugeActive(JAUGE_SansNom1, 10)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`JaugeActive(<Champ Jauge> [, <Période> [, <Arrêt automatique>]])`
---

**`<Champ Jauge> : (Nom de champ)`**

Champ Jauge à manipuler.

**`<Période> : (Entier optionnel ou Durée optionnelle)`**

Période de rafraîchissement de la jauge en centièmes de seconde. Ce paramètre peut correspondre à : 

- un entier correspondant au nombre de centièmes de seconde, 

- une variable de type Durée, 

- l'indication directe de la durée (par exemple '1s', '10cs' ou '10ms'). 


Par défaut, cette période est de 1000 centièmes de seconde.

**`<Arrêt automatique> : (Booléen optionnel)`**

Indique s'il faut arrêter automatiquement le rafraîchissement de la jauge (<u><u><u><u>Vrai</u></u></u></u>, valeur par défaut) à la fin de l'appel serveur.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Le rafraîchissement de la jauge est effectué périodiquement depuis le navigateur en interrogeant le serveur.

En code serveur, la position de la jauge est changée en modifiant simplement la valeur (propriété [Valeur](../Proprietes/2510130.md)) du champ Jauge.

Le rafraîchissement étant provoqué par le navigateur, les modifications en code serveur ne sont pas forcément répercutées immédiatement sur le champ navigateur.

**Attention** :  Il est important de ne pas appeler trop fréquemment le serveur pour ne pas le surcharger inutilement de requêtes.

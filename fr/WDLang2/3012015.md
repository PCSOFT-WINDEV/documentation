
## RépertoireWeb (Fonction)

***En anglais : FolderWeb***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le chemin relatif du répertoire contenant :

- les images,

- les fichiers Javascript,

- les fichiers applet Java,

- les autres fichiers accessibles depuis le navigateur.




Remarque : Pour atteindre le répertoire "&lt;Nom projet&gt;_WEB", il est possible d'utiliser : 

- la fonction **RépertoireWeb** : cette fonction renvoie un chemin relatif destiné à être interprété par le navigateur. 

- la fonction [fRepWeb](../WDLang1/3036052.md) : cette fonction renvoie un chemin absolu à utiliser depuis le serveur avec les [fonctions de gestion des fichiers externes (fxxx)](../WDLang1/3036034.md).

## Syntaxe

`<Résultat> = RépertoireWeb()`
---

**`<Résultat> : (Chaîne de caractères)`**

Chemin relatif du répertoire contenant les images, les fichiers Javascript, ... 





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png)![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) 

### Répertoire des images (en code navigateur)
<a name="repertoire_des_images_code_navigateur_ELTPARAGRAPHE000073"></a>Par défaut, l'option "Localiser l'image dans le répertoire _WEB en code navigateur" est cochée lors de la description d'une image. Dans ce cas, en code navigateur, l'image est automatiquement recherchée dans le répertoire _WEB du site. Aucun code spécifique n'est nécessaire.

Si l'option "Localiser l'image dans le répertoire _WEB en code navigateur" est décochée, le chemin complet de l'image doit être indiqué pour retrouver l'image. Il faut pour cela, utiliser la fonction **RépertoireWeb**.
<a name="NOTE0_2"></a>




### Spécifier le répertoire des images en WLangage pour la génération du code HTML
<a name="specifier_repertoire_des_images_wlangage_pour_generation_code_html_ELTPARAGRAPHE000086"></a>La fonction **RépertoireWeb** peut être utilisée pour spécifier le répertoire des images lors de la génération de code HTML en WLangage.

Si le code HTML doit contenir des références à des images ou tout autre objet statique (fichier Java ou Javascript), il est nécessaire d'utiliser la fonction **RépertoireWeb** pour localiser ces fichiers.


<a name="NOTE0_3"></a>




### Installation
<a name="installation_ELTPARAGRAPHE000103"></a>Dans le cas d'une installation par média physique, le répertoire des fichiers de données est spécifié lors de l'installation.

Dans le cas d'une installation par accès distant (FTP), l'hébergeur fournit le répertoire de base des fichiers de données et l'utilisateur fournit le nom du sous-répertoire dans lequel les fichiers de données sont installés.

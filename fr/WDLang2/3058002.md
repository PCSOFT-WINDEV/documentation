


## ContexteExiste (Fonction)

***En anglais : ContextExist***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de savoir si un contexte de page existe sur le serveur (c'est-à-dire si la page a été ouverte).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Actualise la page si le contexte existe sinon ouvre la page
SI ContexteExiste(PAGE_Page3) = Vrai ALORS
	PageActualise(PAGE_Page3)
SINON
	PageAffiche(PAGE_Page3)
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = ContexteExiste(<Page>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le contexte de la page spécifiée existe (c'est-à-dire si la page spécifiée est ouverte), 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Page> : (Nom de page)`**



- Nom de la page dont l'existence du contexte doit être vérifiée. 

- Chaîne vide ("") pour désigner la page à laquelle appartient le traitement en cours.




## Fonctions diverses WEBDEV
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions diverses WEBDEV sont les suivantes : 



|   |   |
| --- | --- |
| [ASPAffiche](../WDLang2/3012003.md) | Appelle un script ASP externe et renvoie la page résultat dans la fenêtre en cours du navigateur. |
| [ASPExécute](../WDLang2/3012007.md) | Appelle un script externe .asp et renvoie le résultat dans une chaîne. |
| [CapsLockVérifie](../WDLang2/1000020557.md) | Vérifie si la touche CapsLock est enfoncée. |
| [CertificatClientInfo](../WDLang2/1000019393.md) | Renvoie des informations sur le certificat utilisé par le poste client. |
| [ChaîneAffiche](../WDLang2/3012001.md) | Renvoie une chaîne spécifique (ou un buffer) sur le navigateur du client en réponse à la requête reçue. |
| [ConnexionOccurrence](../WDLang2/3012004.md) | Renvoie le nombre d'instances du site WEBDEV en cours d'exécution sur  le serveur. |
| [CouleurPalette](../WDLang2/1000019905.md) | Renvoie une couleur de la palette courante. |
| [DynamicServingConfigure](../WDLang2/1000021994.md) | Indique l'environnement à utiliser par le "Dynamic Serving" pour choisir le jeu de pages à afficher. |
| [ExécuteProcédureDifférée](../WDLang2/1000020682.md) | Exécute une procédure différée (dans le serveur d'application WEBDEV). |
| [FichierAffiche](../WDLang2/3012005.md) | Affiche un fichier sur le navigateur de l'internaute. |
| [HTMLClasseAjoute](../WDLang2/1000021550.md) | Ajoute une classe dans les classes HTML d'un champ. |
| [HTMLClasseBascule](../WDLang2/1000021551.md) | Bascule une classe dans les classes HTML d'un champ : si la classe n'existe pas, elle est ajoutée, si la classe existe déjà, elle est supprimée. |
| [HTMLClasseSupprime](../WDLang2/1000021552.md) | Supprime une classe des classes HTML d'un champ. |
| [jQuery](../WDLang2/1000021070.md) | Exécute une méthode Javascript (ou plusieurs méthodes chaînées) de la librairie jQuery sur un élément de la page. |
| [jQueryExécute](../WDLang2/1000021008.md) | Exécute une méthode Javascript de la librairie jQuery sur un élément de la page. |
| [JSEvénement](../WDLang2/1000018924.md) | Associe une procédure navigateur à un événement sur un objet en code navigateur. |
| [JSFinEvénement](../WDLang2/1000018925.md) | Supprime l'association d'une fonction WLangage navigateur à événement (mis en place grâce à la fonction [JSEvénement](../WDLang2/1000018924.md)). |
| [JSInfoEvénement](../WDLang2/3036067.md) | Permet de manipuler les propriétés JavaScript de l'événement navigateur qui a déclenché l'exécution du code. |
| [JSInterruptionEvénement](../WDLang2/1000019185.md) | Interrompt le traitement de l'événement en cours. |
| [JSMéthode](../WDLang2/3012002.md) | Permet d'exécuter une méthode Javascript sur un élément de la page en cours. |
| [JSONExécute](../WDLang2/1000017284.md) | Appelle une URL serveur qui retourne des données au format JSON (JavaScript Object Notation). |
| [JSONExécuteExterne](../WDLang2/1000017286.md) | Appelle une URL serveur externe qui retourne des données au format JSON (JavaScript Object Notation). |
| [JSPropriété](../WDLang2/3012019.md) | Permet de manipuler des fonctionnalités spécifiques sur des objets de la page en cours. |
| [PHPAffiche](../WDLang2/3012021.md) | Appelle un script PHP externe et renvoie la page résultat dans la fenêtre en cours du navigateur. |
| [PHPExécute](../WDLang2/3012013.md) | Appelle un script externe .php et renvoie le résultat dans une chaîne. |
| [RépertoireDonnées](../WDLang2/3036007.md) | La fonction RépertoireDonnées est conservée uniquement pour compatibilité. |
| [RépertoireWeb](../WDLang2/3012015.md) | Renvoie le chemin du répertoire contenant :<br><br>- les images,<br><br>- les fichiers Javascript,<br><br>- les fichiers applet Java,<br><br>- les autres fichiers accessibles depuis le navigateur.<br><br><br> |
| [RépertoireWeb_15](../WDLang2/3012018.md) | Attention : lors de la migration d'un projet WEBDEV 1.5 vers WEBDEV 17 : <br><br>- la fonction RépertoireWeb a été remplacée par la fonction RépertoireWeb_15. <br><br>- un warning est automatiquement généré dans votre projet.<br><br><br> |
| [ScriptAffiche](../WDLang2/3012017.md) | Appelle un script ou une page externe (par exemple .php, .asp, .mhtml ou .mht) et renvoie la page résultat dans la fenêtre en cours du navigateur. |
| [ScriptExécute](../WDLang2/3012016.md) | Appelle un script externe (par exemple .php ou .asp) et renvoie le résultat dans une chaîne. |
| [SessionHeartBeatActive](../WDLang2/1000026248.md) | Active ou désactive le "heartbeat" de la session courante. |
| [SessionPageDemandée](../WDLang2/1410089106.md) | Renvoie le nom de la page explicitement demandée au lancement de la session. |
| [SessionPrologue](../WDLang2/1000026262.md) | Active ou désactive une procédure WLangage appelée sur chaque requête dans une session. |
| [SSLActive](../WDLang2/3012014.md) | Permet d'activer ou de désactiver le mode sécurisé SSL. |

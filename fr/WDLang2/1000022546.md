
## NavigateurTranche (Fonction)

***En anglais : BrowserSlice***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le numéro de la tranche affichée dans une page en mode Responsive Web Design.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Si la tranche affichée est la tranche "Tablette"
SI NavigateurTranche() = 2 ALORS
	...
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Resultat> = NavigateurTranche()`
---

**`<Resultat> : (Entier)`**

Numéro de la tranche en cours.

Remarque : Si la page  n'utilise pas le mode **Responsive Web Design**, &lt;Résultat&gt; vaut 1.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Le numéro de la tranche par défaut est 1. 

- Le numéro de la tranche s'incrémente ensuite pour chaque tranche suivante :

	- Si la tranche par défaut est la tranche "Bureau", les numéros de tranches s'incrémentent de la tranche la plus large (tranche "Bureau") à la tranche la plus étroite (tranche "Mobile").

	- Si la tranche par défaut est la tranche "Mobile", les numéros de tranches s'incrémentent de la tranche la plus étroite (tranche "Mobile") à la tranche la plus large (tranche "Bureau").

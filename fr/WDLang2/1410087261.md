


## &lt;Page&gt;.FermePopup (Fonction)

***En anglais : &lt;Page&gt;.ClosePopup***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Masque une page affichée en popup via la fonction [&lt;Page&gt;.AffichePopup](../WDLang2/1410087260.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ferme la page affichée en popup
PAGE_DétailsProduit.FermePopup()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Page>.FermePopup()`
---

**`<Page> : (Nom de page)`**

Nom de la page affichée en popup qui sera masquée. Cette page doit avoir été affichée par la fonction [&lt;Page&gt;.AffichePopup](../WDLang2/1410087260.md). Si ce n'est pas le cas, la fonction **&lt;Page&gt;.FermePopup** n'a aucune action.




<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Les navigateurs ne rafraîchissent pas l'affichage au milieu d'un code Javascript : les fonctions [&lt;Page&gt;.AffichePopup](../WDLang2/1410087260.md) et **&lt;Page&gt;.FermePopup** doivent être utilisées dans des traitements/événements différents. Si ces fonctions sont utilisées dans le même traitement/événement, la page Popup n'est pas affichée.

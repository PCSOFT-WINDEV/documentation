


## ImagePrécédent (Fonction)

***En anglais : ImagePrevious***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Affiche l'image précédente du défilement.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Retourne à la bannière précédente
ImagePrécédent(IMG_Bannière)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`ImagePrécédent(<Champ Image>)`
---

**`<Champ Image> : (Nom de champ)`**

Nom du champ à manipuler : 

- Champ Image.

- Champ Image cliquable.

- Champ Vignette.

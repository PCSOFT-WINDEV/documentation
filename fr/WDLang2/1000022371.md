


## ZoneRépétéeFinRemplissage (Fonction)

***En anglais : LooperEndFilling***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Indique la fin du remplissage d'un champ Zone répétée par programmation. 

Rappel : Le début du remplissage a été indiqué avec la fonction [ZoneRépétéeDébutRemplissage](../WDLang2/1000022370.md). L'avancement du remplissage est représenté :

- soit par une jauge infinie affichée au bas du champ.

- soit par une fenêtre interne.




Attention : Avant d'utiliser cette fonction, vous devez activer le mécanisme de gestion du remplissage dans la fenêtre de description du champ. Pour plus de détails, consultez [Caractéristiques d'un champ Zone répétée dans une fenêtre](../WDChamp/1013294.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
ZoneRépétéeDébutRemplissage(ZR_MaZone)

POUR i = 1 A 10000
	ZoneRépétéeAjouteLigne(ZR_MaZone)
	Multitâche(-1)
FIN

ZoneRépétéeFinRemplissage(ZR_MaZone)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`ZoneRépétéeFinRemplissage(<Champ Zone répétée>)`
---

**`<Champ Zone répétée> : (Nom de champ)`**

Nom du champ Zone répétée à manipuler.

Si ce paramètre correspond à une chaîne vide (""), le champ Zone répétée manipulé sera celui auquel appartient l'événement en cours.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Cette fonction est sans effet sur les champs Zone répétée reliés à un fichier de données ou à une requête.


## NavigateurMobile (Fonction)

***En anglais : BrowserMobile***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de savoir si le navigateur utilisé par l'internaute est lancé sur un mobile.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Redirige vers la page mobile
SI NavigateurMobile() = Vrai ALORS
	PageAffiche(PAGE_Home_Mobile)
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = NavigateurMobile()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le navigateur est lancé sur un mobile. 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Les tablettes ne sont pas considérées comme des mobiles.

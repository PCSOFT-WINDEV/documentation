


## ZoneRépétéeVersWord (Fonction)

***En anglais : LooperToWord***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Crée un fichier Word (.RTF) avec les données d'un champ Zone répétée.

**Remarque** : Ce fichier pourra être ouvert sous Word ou sous OpenOffice Writer (selon le logiciel installé sur le poste en cours).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) 
```wl
sFic est une chaîne
// Nom du fichier
sFic = fSélecteur("", "", "Choisissez le fichier d'exportation", ...
	"Fichiers Word (*.rtf) *.rtf", "rtf", fSelCrée + fSelExiste)
// Si le nom n'a pas été fourni, on abandonne l'export
// Sinon, on exporte le contenu du champ Zone répétée ZR_Client
SI sFic <> "" ALORS ZoneRépétéeVersWord(ZR_Client, sFic)
```

## Syntaxe

`<Résultat> = ZoneRépétéeVersWord(<Champ Zone répétée> , <Nom du fichier Doc> [, <Options> [, <Répétition début> [, <Répétition fin>]]])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le fichier a été créé, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Champ Zone répétée> : (Nom de champ)`**

Nom du champ Zone répétée à manipuler. 

**`<Nom du fichier Doc> : (Chaîne de caractères)`**

Nom et chemin complet du fichier Word à créer. Si le fichier existe déjà, il est remplacé par le nouveau fichier.

**`<Options> : (Constante optionnelle de type Entier)`**

Paramètre le fichier Word à créer.


|   |   |
| --- | --- |
| *taLigneSélectionnées* | Exporte uniquement les répétitions sélectionnées (par défaut, toutes les répétitions sont exportées). |
| *taSansTitre* | Seules les données sont copiées dans le fichier. |
| *taTitreColonnes*<br>(Valeur par défaut) | Le titre des colonnes est inséré dans le fichier. |



**`<Répétition début> : (Entier optionnel)`**

Numéro de la répétition de début de l'exportation. Si ce paramètre n'est pas précisé, la répétition de début est la première répétition du champ Zone répétée.

**`<Répétition fin> : (Entier optionnel)`**

Numéro de la répétition de fin de l'exportation. Si ce paramètre n'est pas précisé, la répétition de fin est la dernière répétition du champ Zone répétée.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Le fichier créé pourra être ouvert avec Word version 95 ou supérieure ou avec OpenOffice Writer version 2 ou supérieure.

- Cette fonction peut être utilisée indifféremment sur les champs Zone répétée fichier et Zone répétée par programmation.

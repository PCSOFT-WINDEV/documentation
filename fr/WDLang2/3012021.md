


## PHPAffiche (Fonction)

***En anglais : PHPDisplay***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Appelle un script PHP externe et renvoie la page résultat dans la fenêtre en cours du navigateur.

Il est conseillé d'utiliser cette fonction pour exécuter un script PHP (plutôt que la fonction [ScriptAffiche](../WDLang2/3012017.md)).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Exécute le script "Script1.php" situé 
// à la racine du site Web et renvoie le résultat à l'internaute 
PHPAffiche("/Script1.php")
```
<a name="Exemple2"></a>

```wl
// Page de login en php. , envoie 3 paramètres avec la méthode GET
PHPAffiche("/PHP/Login.php", "Name=BOB&Pass=Mon+Secret&Lang=3")
```
<a name="Exemple3"></a>

```wl
// Retour à la "home" page du site
PHPAffiche("/default.php")
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`PHPAffiche(<Nom du script> [, <"Nom1=Param1&Nom2=Param2&...NomN=ParamN"> [, <Redirection>]])`
---

**`<Nom du script> : (Chaîne de caractères)`**

Chemin et nom du script. Le chemin peut être : 

- une URL complète "http://..." 

- un chemin relatif à la racine du site "/...". Dans ce cas, le script doit être sur le même serveur que le site Web, dans un des sous-répertoires du site.




**`<"Nom1=Param1&Nom2=Param2&...NomN=ParamN"> : (Chaîne de caractères optionnelle)`**

Paramètres destinés au script. Cette chaîne contient pour chaque paramètre :

- le nom du paramètre (par exemple "Nom1")

- le signe =

- la valeur du paramètre (par exemple "Param1")




Deux paramètres sont séparés par le signe &.

**`<Redirection> : (Booléen)`**

Permet de définir le code de retour HTTP de la fonction au navigateur. Ce code de retour est notamment utilisé lors du référencement par les moteurs de recherche. Ce paramètre peut correspondre à : 

- <u><u><u><u>Vrai</u></u></u></u> : redirection permanente (code HTTP 301). La page affichée par la fonction sera directement référencée.

- <u><u><u><u>Faux</u></u></u></u> (valeur par défaut) : redirection temporaire (code HTTP 302). Seule la page source sera référencée. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Afficher d'autres fichiers
<a name="afficher_autres_fichiers_ELTPARAGRAPHE000065"></a>La fonction **PHPAffiche** peut être également utilisée sur un document qui n'est pas un script PHP. Dans ce cas, le document est simplement renvoyé (équivalent à la fonction [FichierAffiche](../WDLang2/3012005.md), en précisant non pas une adresse physique mais une URL).
<a name="NOTE0_2"></a>


### Encodage des paramètres
<a name="encodage_des_parametres_ELTPARAGRAPHE000078"></a>Le contenu des paramètres est automatiquement encodé pour obtenir une URL valide :

- transformation en UTF-8 (selon les options de la page et du projet).

- encodage des caractères spéciaux.



<a name="NOTE0_3"></a>


### Sessions prélancées
<a name="sessions_prelancees_ELTPARAGRAPHE000090"></a>Si votre projet utilise les sessions prélancées, cette fonction ne doit pas être utilisée dans l'événement "Initialisation" du projet. Cette fonction doit être utilisée dans l'événement "Initialisation du projet après connexion au site".




## &lt;Zone répétée&gt;.AnnuleBalayage (Fonction)

***En anglais : &lt;Looper&gt;.CancelSwipe***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Annule l'affichage de la fenêtre interne affichée dans un champ Zone répétée suite au balayage d'une répétition (ou ligne).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Code du bouton d'annulation de la fenêtre interne
ZR_ZoneRepeteeClient.AnnuleBalayage()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Champ Zone répétée>.AnnuleBalayage()`
---

**`<Champ Zone répétée> : (Nom de champ)`**

Nom du champ Zone répétée à manipuler.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Il est possible de gérer le balayage d'une répétition (ou ligne) dans un champ Zone répétée. Ce balayage peut :

- Ne rien faire.

- Exécuter du code.

- Supprimer automatiquement l'élément balayé.

- Afficher une fenêtre interne afin de personnaliser l'action.




Pour plus de détails, consultez [Zone répétée : déplacements et gestures](../WDChamp/1013295.md).

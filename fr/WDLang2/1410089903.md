


## Fonctions de pilotage du Serveur d'application WEBDEV (syntaxe préfixée)



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions de gestion des utilisateurs et des connexions sur un Serveur d'application WEBDEV via une syntaxe préfixée sont les suivantes :




|   |   |
| --- | --- |
| [&lt;Variable adminWBAuth&gt;.CréeCompte](../WDLang2/1410089712.md) | Crée un compte WEBDEV sur un Serveur d'application WEBDEV. |
| [&lt;Variable adminWBAuth&gt;.Déconnecte](../WDLang2/1410089713.md) | Supprime une connexion sur un Serveur d'application WEBDEV. |
| [&lt;Variable adminWBAuth&gt;.ListeCompte](../WDLang2/1410089714.md) | Renvoie la liste des comptes WEBDEV sur un Serveur d'application WEBDEV. |
| [&lt;Variable adminWBAuth&gt;.ListeConnexion](../WDLang2/1410089715.md) | Renvoie la liste des connexions en cours sur un Serveur d'application WEBDEV. |
| [&lt;Variable adminWBAuth&gt;.ModifieCompte](../WDLang2/1410089716.md) | Modifie les paramètres d'un compte WEBDEV sur un Serveur d'application WEBDEV. |
| [&lt;Variable adminWBAuth&gt;.SupprimeCompte](../WDLang2/1410089717.md) | Supprime un compte WEBDEV sur un Serveur d'application WEBDEV. |

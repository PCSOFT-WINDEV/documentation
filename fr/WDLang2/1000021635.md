


## PageLitEntêteHTTP (Fonction)

***En anglais : PageReadHeaderHTTP***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Lit et renvoie un ou plusieurs entêtes HTTP reçus par la page en cours (ces entêtes sont envoyés par le client au serveur dans la requête courante).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
sReferer est une chaîne = PageLitEntêteHTTP("Referer")
```
<a name="Exemple2"></a>

```wl
// Exemple de récupération de tous les entêtes HTTP reçus
// Parcours des entêtes HTTP reçus
nIndiceEnTete est un entier
sUnEnTete est une chaîne
sTousLesEnTetes est une chaîne

BOUCLE
	// Entête suivant
	nIndiceEnTete ++
	sUnEnTete = PageLitEntêteHTTP(nIndiceEnTete)
	// Entête lu ?
	SI sUnEnTete = "" ALORS SORTIR  // plus d'entête
	sTousLesEnTetes += [RC] + sUnEnTete
FIN
Info(sTousLesEnTetes)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Lire un entête identifié par son nom

`<Résultat> = PageLitEntêteHTTP(<Entête à lire>)`
---

**`<Résultat> : (Chaîne de caractères)`**



- Valeur de l'entête. 

- Chaîne vide ("") si l'entête n'est pas présent.




**`<Entête à lire> : (Chaîne de caractères)`**

Nom de l'entête à lire.


<a name="SYNTAXE2"></a>

### Lire un entête identifié par son indice

`<Résultat> = PageLitEntêteHTTP(<Indice>)`
---

**`<Résultat> : (Chaîne de caractères)`**



- Valeur de l'entête. Le nom de l'entête est renvoyé en majuscules et les tirets sont remplacés par des "underscores" (le nom original de l'entête HTTP tel que envoyé par le client n'est pas disponible). 

- Chaîne vide ("") si il n'y a plus d'entêtes à lire.




**`<Indice> : (Entier)`**

Indice de l'entête à lire.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Dans un site dynamique classique, les entêtes sont les entêtes HTTP reçus lors de la connexion au site.

- Pour lister tous les entêtes, utilisez la syntaxe par indice jusqu'à ce que la fonction retourne une chaîne vide.

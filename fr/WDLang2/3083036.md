


## &lt;Chemin de navigation&gt;.ModifieLien (Fonction)

***En anglais : &lt;Breadcrumb&gt;.ModifyLink***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie un lien dans un champ Chemin de navigation.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Modifie un lien
SMP_MonChampChemin.ModifieLien(4, "Aide", "Page_Aide", "Aide")
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Champ Chemin de navigation>.ModifieLien(<Numéro> , <Libellé> , <URL ou nom de la page> [, <Bulle d'aide>])`
---

**`<Champ Chemin de navigation> : (Nom de champ)`**

Nom du champ Chemin de navigation dans lequel le lien doit être modifié.

**`<Numéro> : (Entier)`**

Numéro du lien à modifier. Ce numéro est compris entre 1 et le nombre d'occurrences du &lt;Champ Chemin de navigation&gt; (connu avec la propriété [Occurrence](../Proprietes/2510097.md)).

**`<Libellé> : (Chaîne de caractères)`**

Nouveau libellé du lien.

**`<URL ou nom de la page> : (Chaîne de caractères)`**

Nouvelle URL associée au lien ou nom logique de la page WEBDEV à ouvrir.

**`<Bulle d'aide> : (Chaîne de caractères optionnelle)`**

Description du lien, affichée dans une bulle lorsque la souris survole le lien.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Propriétés des liens
<a name="proprietes_des_liens_ELTPARAGRAPHE000189"></a>Pour accéder aux propriétés de chaque lien du champ Chemin de navigation, il est également possible d'utiliser l'opérateur [] avec la propriété associée.

Par exemple :

```wl
SMP_CheminNavigation[1].Libellé = "Accueil"
```

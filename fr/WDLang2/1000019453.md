
## Constantes de gestion des images dans des sites Web
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
[Afficher uniquement les constantes utilisées par les syntaxes préfixées](#" onclick="$('.objet').show();$('.pas_objet').hide();)

[Afficher uniquement les constantes utilisées par les syntaxes non préfixées](#" onclick="$('.objet').hide();$('.pas_objet').show();)Les constantes de gestion des champs Image sont les suivantes :

| Constante | Description | Utilisée par les fonctions |
| --- | --- | --- |
| **défilementCourant** | Remplit la liste des images et lance le défilement depuis la position courante si possible. | [ImageLanceDéfilement](../WDLang2/1000019427.md) |
| **défilementInitialise** | Remplit la liste des images et lance le défilement depuis le début. | [ImageLanceDéfilement](../WDLang2/1000019427.md) |
| **défilementLance** | Lance (ou relance) le défilement depuis la position courante. | [ImageLanceDéfilement](../WDLang2/1000019427.md) |




## &lt;Bandeau défilant&gt;.ArrêteDéfilement (Fonction)

***En anglais : &lt;Sliding Banner&gt;.StopSliding***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Arrête le défilement automatique sur un champ Bandeau défilant.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Stoppe le défilement (laisse le bandeau courant)
BAN_Bannière.ArrêteDéfilement()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Champ Bandeau défilant>.ArrêteDéfilement()`
---

**`<Champ Bandeau défilant> : (Nom de champ)`**

Nom du champ Bandeau défilant à manipuler.

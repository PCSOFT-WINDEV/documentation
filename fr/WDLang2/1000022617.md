


## Fonctions Bandeau défilant
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions de gestion des champs Bandeau défilant sont les suivantes : 



|   |   |
| --- | --- |
| [BandeauArrêteDéfilement](../WDLang2/1000022512.md) | Arrête le défilement automatique sur un champ Bandeau défilant. |
| [BandeauDernier](../WDLang2/1000022516.md) | Affiche le dernier plan d'un champ Bandeau défilant. |
| [BandeauLanceDéfilement](../WDLang2/1000022511.md) | Active le défilement automatique sur un champ Bandeau défilant. |
| [BandeauPrécédent](../WDLang2/1000022515.md) | Affiche le plan précédent d'un champ Bandeau défilant. |
| [BandeauPremier](../WDLang2/1000022513.md) | Affiche le premier plan d'un champ Bandeau défilant. |
| [BandeauSuivant](../WDLang2/1000022514.md) | Affiche le plan suivant d'un champ Bandeau défilant. |

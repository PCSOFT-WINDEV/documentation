


## AdminWBModifieCompte (Fonction)

***En anglais : WBAdminUpdateAccount***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie les caractéristiques d'un compte WEBDEV sur un Serveur d'application WEBDEV. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
wbAuthLogin est un adminWBAuth
wbAuthLogin.AdresseServeur = "https://mon-serveur.interne/"
wbAuthLogin.Utilisateur = "WWW\DATA"
wbAuthLogin.MotDePasse = "www"

liste_comptes est un tableau de adminWBCompte = AdminWBListeCompte(wbAuthLogin)
compteWB est un adminWBCompte = liste_comptes[1]
compteWB.CompteStatistique = Vrai

Resultat_Modif est un booléen = AdminWBModifieCompte(wbAuthLogin, compteWB)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = AdminWBModifieCompte(<Serveur d'application WEBDEV> , <Compte>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le compte a été modifié, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 




**`<Serveur d'application WEBDEV> : (Variable de type adminWBAuth)`**

Nom de la variable de type [adminWBAuth](../WDLang2/1410089328.md) décrivant le serveur et le compte administrateur utilisé pour interroger le Serveur d'application WEBDEV. 

**`<Compte> : (Variable de type adminWBCompte)`**

Nom de la variable de type [adminWBCompte](../WDLang2/1410089382.md) correspondant à la description du compte à modifier.

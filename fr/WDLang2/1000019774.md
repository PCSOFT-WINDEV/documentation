


## ZoneRépétéeDérouleTout (Fonction)

***En anglais : LooperExpandAll***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Déroule toutes les ruptures d'un champ Zone répétée avec rupture. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Clic sur le champ Bouton [1,2 étoiles uniquement]
ZoneRépétéeDérouleTout(ZR_HOTEL)
ZoneRépétéeEnroule(ZR_HOTEL, nLigne3Etoile)
ZoneRépétéeEnroule(ZR_HOTEL, nLigne4Etoile)
ZoneRépétéeEnroule(ZR_HOTEL, nLigne5Etoile)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`ZoneRépétéeDérouleTout(<Champ Zone répétée>)`
---

**`<Champ Zone répétée> : (Nom de champ)`**

Nom du champ Zone répétée à manipuler.
Si ce paramètre est égal à chaîne vide (""), le champ Zone répétée en cours sera manipulé.




## Remarques


### Conditions d'utilisation
<a name="conditions_utilisation_ELTPARAGRAPHE000048"></a>La fonction **ZoneRépétéeDérouleTout** peut être utilisée indifféremment sur : 

- un champ Zone répétée fichier,

- un champ Zone répétée par programmation.

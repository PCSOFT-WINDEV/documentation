


## BandeauPrécédent (Fonction)

***En anglais : BannerPrevious***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Affiche le plan précédent d'un champ Bandeau défilant.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Retourne au plan précédent
BandeauPrécédent(BAN_Bannière)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`BandeauPrécédent(<Champ Bandeau défilant>)`
---

**`<Champ Bandeau défilant> : (Nom de champ)`**

Nom du champ Bandeau défilant à manipuler.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Les plans invisibles du champ Bandeau défilant sont ignorés.

- Si le plan courant est le premier plan (visible), la fonction **BandeauPrécédent** affiche le dernier plan (visible).

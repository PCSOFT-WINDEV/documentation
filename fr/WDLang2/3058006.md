
## FramesetAffiche (Fonction)

***En anglais : FramesetDisplay***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Affiche un frameset WEBDEV dans le navigateur de l'internaute.

Si la fonction **FramesetAffiche** est utilisée dans le code d'un bouton, le frameset sera ouvert dans la destination définie pour ce bouton (onglet "Général" de la description du champ ou fonction [ChangeDestination](../WDLang2/3058005.md)).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Réaffichage du frameset FSET_LOGIN 
FrameSetAffiche(FSET_LOGIN)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`FramesetAffiche(<Nom du frameset> [, <Paramètre 1> [... [, <Paramètre N>]]])`
---

**`<Nom du frameset> : (Chaîne de caractères)`**

Nom du frameset à afficher. 
Si ce paramètre est égal à une chaîne vide (""), le nom du frameset est celui du frameset auquel appartient l'événement en cours.

**`<Paramètre 1> : (Type de la valeur envoyée à la page (optionnel))`**

Paramètre(s) à passer à l'événement "Déclarations globales" du frameset à ouvrir. Ces paramètres sont passés par référence et sont considérés comme des variables globales au frameset et à toutes les pages du frameset.



**`<Paramètre N> : (Type de la valeur envoyée à la page (optionnel))`**

Paramètre(s) à passer à l'événement "Déclarations globales" du frameset à ouvrir. Ces paramètres sont passés par référence et sont considérés comme des variables globales au frameset et à toutes les pages du frameset.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Récupération des paramètres
<a name="recuperation_des_parametres_ELTPARAGRAPHE000062"></a>Pour récupérer les paramètres, dans l'événement "Déclarations globales" du frameset, indiquez en début de code la ligne suivante :

```txt
PROCEDURE <Nom du frameset>(<Paramètre 1> [,...[, <Paramètre N>]])
```



<a name="NOTE0_2"></a>




### Principe détaillé de l'ouverture d'un frameset
<a name="principe_detaille_ouverture_frameset_ELTPARAGRAPHE000072"></a>La fonction **FramesetAffiche** effectue les opérations suivantes :

- création un contexte de frameset sur le serveur

- création un contexte de page sur le serveur pour toutes les pages du frameset sur le serveur

- renvoi du frameset dans le navigateur de l'internaute.





<a name="NOTE0_3"></a>




### Opérations effectuées à l'ouverture du frameset
<a name="operations_effectuees_ouverture_frameset_ELTPARAGRAPHE000088"></a>Les différentes opérations effectuées lors de l'affichage du frameset sont les suivantes :

- Exécution des événements "Déclarations globales" et "Initialisation" du frameset.

- Ouverture des pages du frameset (exécution des événements "Déclarations globales" et "Initialisation" de chaque page).

- Code WLangage suivant l'appel à la fonction **FramesetAffiche** (s'il existe).





<a name="NOTE0_4"></a>


### Sessions prélancées
<a name="sessions_prelancees_ELTPARAGRAPHE000105"></a>Si votre projet utilise les sessions prélancées, cette fonction ne doit pas être utilisée dans l'événement "Initialisation" du projet. Cette fonction doit être utilisée dans l'événement "Initialisation du projet après connexion au site".




## PageExiste (Fonction)

***En anglais : PageExist***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Vérifie si la page spécifiée est actuellement affichée dans le navigateur de l'internaute.

Permet de savoir par exemple si une page est affichée dans un cadre différent de celui en cours.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Vérifie si la page "PAGE_Inscription" est affichée
PageExiste(PAGE_Inscription)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = PageExiste(<Page>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la page est actuellement affichée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Page> : (Nom de page)`**

Nom de la page dont l'existence doit être vérifiée.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Pour vérifier si un contexte de page est ouvert sur le serveur, utilisez la fonction serveur [ContexteExiste](../WDLang2/3058002.md).




## UploadTailleFichier (Fonction)

***En anglais : UploadFileSize***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la taille totale (en octets) d'un fichier présent dans un champ Upload. 

**Remarque** : Cette fonction peut être utilisée même si aucun upload de fichier n'est en cours. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affiche la taille totale du fichier en Ko
LIB_TailleTotaleFichier = Arrondi(UploadTailleFichier(MoiMême, 1) / 1024, 0) + "Ko"
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = UploadTailleFichier(<Champ Upload> [, <Indice du fichier>])`
---

**`<Résultat> : (Entier)`**

Taille totale (en octets) du fichier présent dans le champ Upload. 

**`<Champ Upload> : (Nom de champ)`**

Nom du champ Upload manipulé.

**`<Indice du fichier> : (Entier optionnel)`**

Indice du fichier (présent dans le champ Upload) à manipuler. Par défaut, ce paramètre vaut 1.


## NavigateurOS (Fonction)

***En anglais : BrowserOS***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Indique le système d'exploitation annoncé par le navigateur de l'internaute.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Propose l'application dans le "store" correspondant à l'OS
SELON NavigateurOS()
	CAS navigateurOSiOS
		// ...
	CAS navigateurOSAndroid
		// ...
	CAS navigateurOSWindowsMobile
		// ...
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = NavigateurOS()`
---

**`<Résultat> : (Constante de type Entier)`**

Système d'exploitation de l'internaute : 



|   |   |
| --- | --- |
| *navigateurOSAndroid* | Navigateur lancé à partir d'un système Android. |
| *navigateurOSAutre* | Navigateur lancé à partir d'un système non répertorié. |
| *navigateurOSiOS* | Navigateur lancé à partir d'un système iOS. |
| *navigateurOSLinux* | Navigateur lancé à partir d'un système Linux. |
| *navigateurOSMacOS* | Navigateur lancé à partir d'un système MacOS. |
| *navigateurOSWindows* | Navigateur lancé à partir d'un système Windows. |
| *navigateurOSWindowsMobile* | Navigateur lancé à partir d'un système Windows pour mobile (par exemple : Windows CE, Windows 10 Mobile, ...). |




## JSMéthode (Fonction)

***En anglais : JSMethod***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet d'exécuter une méthode Javascript sur un élément de la page en cours.

Cette fonction est une fonction avancée : son utilisation nécessite une bonne connaissance de Javascript.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`JSMéthode(<Nom complet de l'élément> , <Méthode> [, <Paramètre 1> [... [, <Paramètre N>]]])`
---

**`<Nom complet de l'élément> : (Chaîne de caractères)`**

Nom "Javascript" de l'élément à manipuler. Généralement, ce nom commence par "document.". Il est nécessaire d'utiliser les alias des champs. Pour plus de détails, consultez le paragraphe Remarques.

**`<Méthode> : (Chaîne de caractères)`**

Méthode de l'élément à manipuler. Le nom de la méthode doit être directement indiqué : il n'est pas possible d'utiliser une chaîne construite.

**`<Paramètre 1> : (Chaîne de caractères optionnelle)`**

Paramètres de la méthode.



**`<Paramètre N> : (Chaîne de caractères optionnelle)`**

Paramètres de la méthode.

<a name="NOTE0_1"></a>

## Remarques


### La fonction JSMéthode manipule les alias des champs
<a name="fonction_jsmethode_manipule_les_alias_des_champs_ELTPARAGRAPHE000040"></a>Le Javascript est un langage sensible à la casse des identifiants : le nom complet de l'élément doit donc respecter la casse des alias de champs : tous les alias de page et de champs sont en majuscules.

Pour construire le nom de l'élément, il est nécessaire de connaître l'alias du champ. Cet alias correspond au nom du champ dans la page HTML. Pour connaître cet alias :

- sous l'éditeur de pages, survolez l'élément pour afficher l'alias utilisé dans la bulle d'aide du champ.

- sous l'éditeur de code, utilisez la combinaison de touches Maj + F1.

- par programmation, utilisez la propriété [Alias](../Proprietes/2510004.md).




Dans la construction du nom complet de l'élément, il est possible d'utiliser "_PAGE_" pour accéder à la page.

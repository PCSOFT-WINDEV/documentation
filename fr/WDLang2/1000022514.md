


## BandeauSuivant (Fonction)

***En anglais : BannerNext***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Affiche le plan suivant d'un champ Bandeau défilant.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affiche le plan suivant
BandeauSuivant(BAN_Bannière)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`BandeauSuivant(<Champ Bandeau défilant>)`
---

**`<Champ Bandeau défilant> : (Nom de champ)`**

Nom du champ Bandeau défilant à manipuler.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Les plans invisibles du champ Bandeau défilant sont ignorés.

- Si le plan courant est le dernier plan (visible), la fonction **BandeauSuivant** affiche le premier plan (visible).

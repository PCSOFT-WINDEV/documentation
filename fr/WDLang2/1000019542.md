
## Constantes Navigateur
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


| Constante | Description | Utilisée par les fonctions |
| --- | --- | --- |
| **cookieHTTPOnly** | Type du cookie : Cookie récupérable uniquement en code serveur. Ce cookie ne pourra pas être lu en code navigateur. | [CookieEcrit](../WDLang2/3012008.md) |
| **cookieNormal** | Type du cookie : Cookie sans options spécifiques. | [CookieEcrit](../WDLang2/3012008.md) |
| **cookieSameSiteStrict** | Type du cookie : Cookie de type "Same Site". Ce type de cookie permet de se prémunir des vulnérabilités de type CSRF (Cross Site Request Forgery). | [CookieEcrit](../WDLang2/3012008.md) |
| **cookieSecure** | Type du cookie : Cookie transmis uniquement dans une requête HTTPS. | [CookieEcrit](../WDLang2/3012008.md) |
| **navigateurOSAndroid** | Navigateur lancé à partir d'un système Android. | [NavigateurOS](../WDLang2/1000021857.md) |
| **navigateurOSAutre** | Navigateur lancé à partir d'un système non répertorié. | [NavigateurOS](../WDLang2/1000021857.md) |
| **navigateurOSiOS** | Navigateur lancé à partir d'un système iOS. | [NavigateurOS](../WDLang2/1000021857.md) |
| **navigateurOSLinux** | Navigateur lancé à partir d'un système Linux. | [NavigateurOS](../WDLang2/1000021857.md) |
| **navigateurOSMacOS** | Navigateur lancé à partir d'un système MacOS. | [NavigateurOS](../WDLang2/1000021857.md) |
| **navigateurOSWindows** | Navigateur lancé à partir d'un système Windows. | [NavigateurOS](../WDLang2/1000021857.md) |
| **navigateurOSWindowsMobile** | Navigateur lancé à partir d'un système Windows pour mobile. | [NavigateurOS](../WDLang2/1000021857.md) |
| **NavigateurPlateformeAndroid** | L'utilisateur affiche le site sur un smartphone sous Android. | [NavigateurPlateforme](../WDLang2/1000019486.md) |
| **NavigateurPlateformeBlackBerry** | L'utilisateur affiche le site sur un BlackBerry. | [NavigateurPlateforme](../WDLang2/1000019486.md) |
| **NavigateurPlateformeiPad** | L'utilisateur affiche le site sur un iPad. | [NavigateurPlateforme](../WDLang2/1000019486.md) |
| **NavigateurPlateformeiPhone** | L'utilisateur affiche le site sur un iPhone. | [NavigateurPlateforme](../WDLang2/1000019486.md) |
| **NavigateurPlateformeMobile** | L'utilisateur affiche le site sur un smartphone (autres que ceux cités). | [NavigateurPlateforme](../WDLang2/1000019486.md) |
| **NavigateurPlateformeRobot** | L'utilisateur est un robot de moteur de recherche (et qui indexe donc le site). | [NavigateurPlateforme](../WDLang2/1000019486.md) |
| **NavigateurPlateformeStandard** | L'utilisateur n'utilise pas une plateforme particulière (c'est le cas pour les PC de bureau, les PC portables, les Mac, ...). | [NavigateurPlateforme](../WDLang2/1000019486.md) |
| **NavigateurPlateformeWindowsMobile** | L'utilisateur affiche le site sur un smartphone sous Windows Mobile. | [NavigateurPlateforme](../WDLang2/1000019486.md) |
| **NavigateurPlateformeWindowsPhone** | L'utilisateur affiche le site sur un smartphone sous Windows Phone. | [NavigateurPlateforme](../WDLang2/1000019486.md) |
| **NavigateurTypeChrome** | Type du navigateur utilisé par l'internaute pour afficher le site : Chrome | [NavigateurType](../WDLang2/1000019487.md) |
| **NavigateurTypeEdge** | Type du navigateur utilisé par l'internaute pour afficher le site : Microsoft Edge | [NavigateurType](../WDLang2/1000019487.md) |
| **NavigateurTypeFirefox** | Type du navigateur utilisé par l'internaute pour afficher le site : Firefox | [NavigateurType](../WDLang2/1000019487.md) |
| **NavigateurTypeGecko** | Type du navigateur utilisé par l'internaute pour afficher le site : Navigateur utilisant le moteur de rendu Gecko (autre que Firefox). | [NavigateurType](../WDLang2/1000019487.md) |
| **NavigateurTypeInconnu** | Type du navigateur utilisé par l'internaute pour afficher le site : Le navigateur utilisé est inconnu ou les informations sont manquantes | [NavigateurType](../WDLang2/1000019487.md) |
| **NavigateurTypeInternetExplorer** | Type du navigateur utilisé par l'internaute pour afficher le site : Internet Explorer. | [NavigateurType](../WDLang2/1000019487.md) |
| **NavigateurTypeOpera** | Type du navigateur utilisé par l'internaute pour afficher le site : Opera | [NavigateurType](../WDLang2/1000019487.md) |
| **NavigateurTypeRobot** | Type du navigateur utilisé par l'internaute pour afficher le site : Robot de moteur de recherche. | [NavigateurType](../WDLang2/1000019487.md) |
| **NavigateurTypeSafari** | Type du navigateur utilisé par l'internaute pour afficher le site : Safari | [NavigateurType](../WDLang2/1000019487.md) |
| **NavigateurTypeWebKit** | Type du navigateur utilisé par l'internaute pour afficher le site : Navigateur utilisant le moteur de rendu WebKit (autre que Chrome et Safari). | [NavigateurType](../WDLang2/1000019487.md) |
| **NavigateurVersionComplète** | Numéro de version complet du navigateur utilisé par l'internaute. | [NavigateurVersion](../WDLang2/1000019488.md) |
| **NavigateurVersionMajeure** | Numéro de version majeure du navigateur utilisé par l'internaute. | [NavigateurVersion](../WDLang2/1000019488.md) |


## GglAnalyticsAjouteActionRéseauSocial (Fonction)

***En anglais : GglAnalyticsAddSocialNetworkAction***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ajoute une action de type réseau social auprès de Google Analytics. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
GglAnalyticsAjouteActionRéseauSocial("Facebook", "like", "http://monsite.com")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`GglAnalyticsAjouteActionRéseauSocial(<Réseau social> , <Action> , <Cible> [, <Autre>])`
---

**`<Réseau social> : (Chaîne de caractères)`**

Nom du réseau social pour lequel l'ajout de l'action doit être effectué (par exemple : Facebook, Twitter, ...). 

**`<Action> : (Chaîne de caractères)`**

Type de l'action à ajouter (par exemple Like, Send, Tweet, ..).

**`<Cible> : (Chaîne de caractères)`**

Cible de l'action sur le réseau social. Ce paramètre peut être l'adresse d'un site Web ou tout autre chaîne de caractères.

**`<Autre> : (Variant optionnel)`**

Options supplémentaires dépendant du réseau social. Pour connaître les valeurs possibles, consultez la documentation de Google Analytics ([https://developers.google.com/analytics/devguides/collection/analyticsjs/social-interactions](https://developers.google.com/analytics/devguides/collection/analyticsjs/social-interactions)) 
Par défaut, ce paramètre correspond à Null.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Pour utiliser cette fonction, vous devez posséder et enregistrer dans le projet un identifiant de suivi Google Analytics. Pour plus de détails, consultez [WEBDEV et Google Analytics](../WDLang2/1000022374.md). 

- **Remarque** : Au 1er Juillet 2023, Google Analytics passe de Universal Analytics à Google Analytics 4. WEBDEV 28 gère les deux versions. Les fonctions WLangage ont été adaptées. Par défaut, pour les projets existants, le mode géré est Universal Analytics et pour les nouveaux projets, le mode géré est Google Analytics 4. Pour plus de détails, consultez [WEBDEV et Google Analytics](../WDLang2/1000022374.md).

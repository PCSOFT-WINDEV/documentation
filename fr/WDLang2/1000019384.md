
## StockageLocalRécupère (Fonction)

***En anglais : LocalStorageGet***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Récupère une valeur du stockage local.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
StockageLocalRécupère(SAI_SaisieNom)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = StockageLocalRécupère(<Nom de l'élément> [, <Persistant>])`
---

**`<Résultat> : (Chaîne de caractères)`**

Valeur récupérée.

Remarque : Si la valeur d'un champ est récupérée, le champ est automatiquement affecté avec la valeur récupérée.

**`<Nom de l'élément> : (Chaîne de caractères)`**

Nom de l'élément dont la valeur doit être récupérée : nom de la valeur ou nom du champ.

**`<Persistant> : (Booléen optionnel)`**



- <u><u><u><u>Vrai</u></u></u></u> pour travailler sur le stockage local persistant (valeur par défaut), 

- <u><u><u><u>Faux</u></u></u></u> pour travailler sur le stockage local à une fenêtre de navigateur.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Stockage local persistant ou non
<a name="stockage_local_persistant_non_ELTPARAGRAPHE000045"></a>Lors d'un stockage local persistant, les valeurs stockées pourront être partagées par toutes les fenêtres et onglets d'un même navigateur d'une machine : il sera donc possible de les retrouver après avoir fermé puis ré-ouvert le navigateur ou dans une autre fenêtre du navigateur. 

Lors d'un stockage local non persistant, les valeurs stockées sont propres à une fenêtre (ou un onglet) du navigateur. Dans ce cas, les valeurs seront perdues lors de la fermeture de la fenêtre du navigateur (ou de l'onglet). Par contre, ces valeurs pourront être lues par les fenêtres du navigateur ouvertes depuis la fenêtre où ont été stockées les valeurs.

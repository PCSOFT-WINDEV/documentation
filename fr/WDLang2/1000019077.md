


## Fonctions Upload (syntaxe préfixée)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions (syntaxe préfixée) de gestion de l'upload de fichiers sont les suivantes : 



|   |   |
| --- | --- |
| [&lt;Upload&gt;.CopieFichier](../WDLang2/1000019046.md) | Enregistre sur le serveur un fichier "uploadé" par l'internaute. Cet upload a pu être effectué : <br><br>- via un champ de saisie de type "Upload". <br><br>- via un champ Upload mono ou multi-fichiers.<br><br><br> |
| [&lt;Upload&gt;.FichierEnCours](../WDLang2/1000019047.md) | Indique le fichier en cours d'upload via le champ Upload. |
| [&lt;Upload&gt;.Lance](../WDLang2/1000019063.md) | Lance l'envoi des fichiers sélectionnés dans un champ Upload. |
| [&lt;Upload&gt;.NomFichier](../WDLang2/1000019064.md) | Renvoie le nom d'un fichier "uploadé" par l'internaute. Cet upload a pu être effectué : <br><br>- via un champ de saisie de type "Upload". <br><br>- via un champ Upload mono ou multi-fichiers.<br><br><br> |
| [&lt;Upload&gt;.Supprime](../WDLang2/1000019065.md) | Supprime un fichier de la liste des fichiers à uploader : le fichier ne sera pas uploadé sur le serveur. |
| [&lt;Upload&gt;.SupprimeTout](../WDLang2/1000019069.md) | Vide la liste des fichiers à uploader : aucun fichier ne sera uploadé sur le serveur. |
| [&lt;Upload&gt;.Taille](../WDLang2/1000019070.md) | Renvoie la taille totale (en octets) des fichiers de l'upload en cours via un champ Upload. |
| [&lt;Upload&gt;.TailleEnvoyée](../WDLang2/1000019071.md) | Renvoie la taille totale (en octets) des fichiers déjà envoyés par l'upload en cours via un champ Upload. |
| [&lt;Upload&gt;.TailleEnvoyéeFichierEnCours](../WDLang2/1000019074.md) | Renvoie la taille (en octets) déjà envoyée du fichier en cours d'upload via un champ Upload. |
| [&lt;Upload&gt;.TailleFichier](../WDLang2/1000019075.md) | Renvoie la taille totale (en octets) d'un fichier présent dans un champ Upload. |
| [&lt;Upload&gt;.TailleFichierEnCours](../WDLang2/1000019076.md) | Renvoie la taille totale (en octets) du fichier en cours d'upload via un champ Upload. |

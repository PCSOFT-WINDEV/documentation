
## adminWBAuth (Type de variable)

***En anglais : WBAdminAuth***
			
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **adminWBAuth** permet de définir toutes les caractéristiques avancées de l'API authentifiée de gestion à distance d'un serveur d'application WEBDEV. Les caractéristiques de cette API peuvent être définies et modifiées à l'aide de différentes propriétés WLangage. 

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
wbAuthLogin est un adminWBAuth
wbAuthLogin.AdresseServeur = "https://mon-serveur.interne/"
wbAuthLogin.Utilisateur = "WWW\DATA"
wbAuthLogin.MotDePasse = "www"

ConnexionsWB est un tableau de adminWBConnexions = AdminWBListeConnexion(wbAuthLogin)

SI ErreurDétectée = Vrai ALORS
	Info(ErreurInfo)
	RETOUR
FIN

POUR i = 1 _À_ ConnexionsWB.Occurrence  
	Trace(ConnexionsWB[i].Site)
	Trace(DateHeureVersChaîne(ConnexionsWB[i].DateHeureDébutConnexion))
	Trace(DuréeVersChaîne(ConnexionsWB[i].TempsInactivité, duréeCentième))
FIN
```





<a name="PROPS"></a>

## Propriétés


### Propriétés spécifiques à la description des variables de type adminWBAuth
<a name="proprietes_specifiques_description_des_variables_type_adminwbauth_ELTPARAGRAPHE000026"></a>Les propriétés suivantes peuvent être utilisées pour manipuler une variable de type **adminWBAuth** :

| Nom de la propriété | Type manipulé | Effet |
| --- | --- | --- |
| AdresseServeur | Chaîne de caractères | Adresse du Serveur d'application WEBDEV de la forme "https://www.monserveur.com/". <br>Attention : il est nécessaire de préciser l'adresse avec le protocole utilisé (http/https).  |
| MotDePasse | Chaîne de caractères | Mot de passe associé à l'utilisateur. |
| Port | Entier | Numéro du port utilisé (par exemple, 443 par défaut pour https). |
| Utilisateur | Chaîne de caractères | Nom de l'utilisateur qui veut gérer le Serveur d'application WEBDEV. Cet utilisateur doit être un administrateur WEBDEV. |

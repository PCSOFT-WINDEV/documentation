


## CookieLit (Fonction)

***En anglais : CookieRead***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Récupère la valeur d'un cookie enregistré sur le poste de l'internaute.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple
<a class="notetitle" target="_blank" href="$DOC$=1000003012009&name=cookielit_fonction&product=WD">Voir des exemples supplémentaires</a>

```wl
// Récupère le nom et la date de la dernière visite
NomEtDate est une chaîne
NomEtDate = CookieLit("NOMETDATE") // NOMETDATE : nom du cookie
SI NomEtDate = "" ALORS
	// première visite
	MonMessage = "Bienvenue, nouveau visiteur !"
SINON
	LeNom = ExtraitChaîne(NomEtDate, 1, ",")
	MonMessage = "Bienvenue " + LeNom
	LaDate = ExtraitChaîne(NomEtDate, 0, ",")
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = CookieLit(<Nom du cookie>)`
---

**`<Résultat> : (Chaîne de caractères)`**



- Valeur du cookie, 

- Chaîne vide ("") si le cookie n'existe pas sur le poste de l'internaute.




**`<Nom du cookie> : (Chaîne de caractères)`**

Nom du cookie à lire sur le poste de l'internaute. Ce nom a été utilisé lors de la création du cookie grâce à la fonction [CookieEcrit](../WDLang2/3012008.md).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Si votre projet utilise les sessions prélancées, cette fonction ne doit pas être utilisée dans l'événement "Initialisation" du projet. Cette fonction doit être utilisée dans l'événement "Initialisation du projet après connexion au site".

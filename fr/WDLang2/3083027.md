


## ZoneRépétéeSelectOccurrence (Fonction)

***En anglais : LooperSelectCount***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le nombre de répétitions sélectionnées dans un champ Zone répétée. Les répétitions sélectionnées correspondent aux répétitions en cours sur lesquelles le bandeau de sélection est affiché.

**Remarque** : La fonction **ZoneRépétéeSelectOccurrence** peut être utilisée indifféremment sur :

- un champ Zone répétée fichier ou Zone répétée par programmation.

- un champ Zone répétée monosélection ou multisélection.













<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
i est un entier 
NbSélectionné est un entier 
NbSélectionné = ZoneRépétéeSelectOccurrence(ZR_PRODUIT) 
POUR i = 1 A NbSelectionné
	Trace("Répétition sélectionnée : " + i) 
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = ZoneRépétéeSelectOccurrence(<Champ Zone répétée>)`
---

**`<Résultat> : (Entier)`**



- Nombre de répétitions sélectionnées dans le champ Zone répétée spécifié, 

- 0 si aucune répétition n'est sélectionnée. 


Ce nombre peut être supérieur à 1 pour les champs Zone répétée multisélections.

**`<Champ Zone répétée> : (Nom de champ)`**

Nom du champ Zone répétée à manipuler.
Si ce paramètre correspond à une chaîne vide (""), le champ Zone répétée auquel appartient l'événement en cours est manipulé. 
Si ce paramètre ne correspond pas à un nom du champ Zone répétée, une erreur WLangage est générée.

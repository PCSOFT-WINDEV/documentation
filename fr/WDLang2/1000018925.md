
## JSFinEvénement (Fonction)

***En anglais : JSEndEvent***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime l'association d'une fonction WLangage navigateur à événement (mis en place grâce à la fonction [JSEvénement](../WDLang2/1000018924.md)). La procédure associée à l'événement ne sera plus appelée.

**Remarque** : L'interception des événements est automatiquement arrêtée lorsque la page associée est fermée.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Annule l'interception de tous les clics sur la page
// gnClic est un entier renvoyé par la fonction JSEvénement
JSFinEvénement(gnClicPage)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = JSFinEvénement(<Numéro de l'événement>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'interception de l'événement a été annulée, 

- <u><u><u><u>Faux</u></u></u></u> en cas d'erreur (numéro d'événement incorrect par exemple).




**`<Numéro de l'événement> : (Entier)`**

Numéro de l'événement à annuler. Cet entier a été renvoyé par la fonction [JSEvénement](../WDLang2/1000018924.md).




## Fonctions de gestion des images
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Différentes fonctions de gestion des images sont disponibles : 

- Les fonctions spécifiques au Web permettant de manipuler les images (notamment les fonctions de gestion de défilement d'images). 

- Les fonctions de gestion des images dans les champs Image. 





|   |   |
| --- | --- |
| [ImageArrêteDéfilement](../WDLang2/1000019428.md) | Arrête le défilement automatique des images. |
| [ImageCalqueVisible](../WDLang2/1000023504.md) | Permet de rendre visible ou invisible un calque d'une image de type WDPIC. |
| [ImageDernier](../WDLang2/1000019432.md) | Affiche la dernière image du défilement. |
| [ImageLanceDéfilement](../WDLang2/1000019427.md) | Lance le défilement automatique des images. |
| [ImageOccurrence](../WDLang2/1000019434.md) | Renvoie le nombre total d'images prises en compte par le défilement. |
| [ImagePositionDéfilement](../WDLang2/1000019433.md) | Retourne l'image affichée. |
| [ImagePosX](../WDLang2/3045003.md) | Renvoie la position horizontale (X) du curseur de souris par rapport au champ Image considéré. |
| [ImagePosY](../WDLang2/3045001.md) | Renvoie la position verticale (Y) du curseur de souris par rapport au champ Image cliquable considéré. |
| [ImagePrécédent](../WDLang2/1000019430.md) | Affiche l'image précédente du défilement. |
| [ImagePremier](../WDLang2/1000019429.md) | Affiche la première image du défilement. |
| [ImageSuivant](../WDLang2/1000019431.md) | Affiche l'image suivante du défilement. |
| [ImageZone](../WDLang2/3045002.md) | Renvoie le numéro de la zone de l'image cliquée par l'internaute. |

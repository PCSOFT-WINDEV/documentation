
## PleinEcranActive (Fonction)

***En anglais : FullScreenEnable***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Passe le navigateur en mode "Plein écran". Le navigateur occupe ainsi tout l'espace de l'écran. La barre d'outils n'est plus affichée.  

Remarque : Pour quitter le mode "Plein écran", utilisez la fonction [PleinEcranDésactive](../WDLang2/1000021013.md). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
PleinEcranActive()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`PleinEcranActive()`
---



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Le passage en mode "Plein écran" est disponible uniquement pour les navigateurs récents. 

- Attention : La fonction **PleinEcranActive** nécessite une action de l'utilisateur : Cette fonction ne peut pas être utilisée dans un événement de type "Onload". 
	Remarque : Les navigateurs bloquent cette commande pour des raisons de sécurité. Elle fait partie des commandes nécessitant obligatoirement une action utilisateur.

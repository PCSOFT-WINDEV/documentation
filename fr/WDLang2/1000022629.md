


## &lt;Bandeau défilant&gt;.Premier (Fonction)

***En anglais : &lt;Slidling Banner&gt;.First***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Affiche le premier plan d'un champ Bandeau défilant.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affiche la première bannière
BAN_Bannière.Premier()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Champ Bandeau défilant>.Premier()`
---

**`<Champ Bandeau défilant> : (Nom de champ)`**

Nom du champ Bandeau défilant à manipuler.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Les plans invisibles du champ Bandeau défilant sont ignorés.

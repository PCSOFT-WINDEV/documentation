


## ImagePosX (Fonction)

***En anglais : ImageXPos***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la position horizontale (X) du curseur de souris par rapport au champ Image considéré. 

Cette fonction est utilisable uniquement dans le code de clic d'un champ Image. Ce champ Image a les caractéristiques suivantes (dans l'onglet "Général" de la fenêtre de description du champ) : 

- Action lors du clic : "Appel serveur"

- Lors de l'action : "Envoi des coordonnées du clic".





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Code de clic de IMG_IMAGE1
X est un entier = ImagePosX()
Y est un entier = ImagePosY()
SI X > Y ALORS
	LIB_LIBELLE1 = "En Haut - A Droite"
SINON
	LIB_LIBELLE1 = "En Bas - A Gauche"
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = ImagePosX()`
---

**`<Résultat> : (Entier)`**

Position horizontale (ou abscisse) en pixels de la souris par rapport au coin haut gauche du champ Image sur lequel le bouton gauche de la souris a été enfoncé. 
Cette valeur est indéfinie si la fonction **ImagePosX** est utilisée dans un événement différent du code de clic d'un champ Image ayant pour action "Envoi des coordonnées du clic" (dans l'onglet "Général" de la fenêtre de description du champ).



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Sessions prélancées
<a name="sessions_prelancees_ELTPARAGRAPHE000054"></a>Si votre projet utilise les sessions prélancées, cette fonction ne doit pas être utilisée dans l'événement "Initialisation du projet". Cette fonction doit être utilisée dans l'événement "Initialisation du projet après connexion au site".




## CelluleFermeDialogue (Fonction)

***En anglais : CellCloseDialog***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Masque un champ Cellule précédemment affiché dans la page via la fonction [CelluleAfficheDialogue](../WDLang2/1000017139.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ferme le champ Cellule de question
CelluleFermeDialogue(CELL_Question)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`CelluleFermeDialogue(<Champ Cellule>)`
---

**`<Champ Cellule> : (Nom de champ)`**

Nom de la cellule qui sera masquée. Cette cellule est superposable et a été affichée par la fonction [CelluleAfficheDialogue](../WDLang2/1000017139.md). Si ce n'est pas le cas, la fonction **CelluleFermeDialogue** ne fait rien.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Les navigateurs ne rafraîchissent pas l'affichage au milieu d'un code Javascript : les fonctions [CelluleAfficheDialogue](../WDLang2/1000017139.md) et **CelluleFermeDialogue** doivent être utilisées dans des événements/traitements différents. Si ces fonctions sont utilisées dans le même événement/traitement, le champ Cellule n'est pas affiché.

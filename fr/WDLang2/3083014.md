


## ZoneRépétéeOccurrence (Fonction)

***En anglais : LooperCount***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le nombre de répétitions d'un champ Zone répétée.










<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Nombre de répétitions dans le champ Zone répétée "ZR_Zone1" ?
Résultat1 est un entier
Résultat1 = ZoneRépétéeOccurrence(ZR_Zone1)
```
<a name="Exemple2"></a>

```wl
// Affiche le nombre de réponses trouvées : (ex : "Il y a 31 réponses")
LIB_LIBELLE1 = "Il y a " + ZoneRépétéeOccurrence(ZR_ZoneResultat) + " réponses"
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = ZoneRépétéeOccurrence(<Champ Zone répétée> [, <Répétition/Attribut>])`
---

**`<Résultat> : (Entier)`**



- Nombre de répétitions du champ Zone répétée, 

- 0 si le champ Zone répétée est vide.




**`<Champ Zone répétée> : (Nom de champ)`**

Nom du champ Zone répétée à manipuler.
Si ce paramètre est égal à chaîne vide (""), le comptage sera effectué sur le champ Zone répétée en cours.

**`<Répétition/Attribut> : (Constante (optionnelle))`**




|   |   |
| --- | --- |
| *toAffichée*<br> | Nombre de répétitions non vides visibles.<br><br><br> |
| *toColonne* | Nombre d'attributs présents dans le champ Zone répétée. |
| *toTotal*<br>(Valeur par défaut)<br> | **Champ Zone répétée fichier** :<br><br>- Nombre total d'enregistrements :<br><br>	- du fichier de données lié (s'il n'y a pas de filtre). <br><br>	- filtrés du fichier de données lié (s'il y a un filtre).<br><br>	- de la requête (si le champ Zone répétée est liée à une requête).<br><br><br><br><br>- Nombre d'enregistrements visibles.<br><br><br>**Attention** : l'événement "Fin d'initialisation" est exécuté uniquement lorsque le total est calculé.<br><br>**Champ Zone répétée par programmation** :<br>Nombre total de répétitions (répétitions ajoutées par les fonctions [ZoneRépétéeAjoute](../WDLang2/3083006.md), [ZoneRépétéeAjouteLigne](../WDLang2/3083004.md), [ZoneRépétéeInsère](../WDLang2/3083003.md) et [ZoneRépétéeInsèreLigne](../WDLang2/3083002.md)). |
| *toVisible* | Nombre de répétitions visibles (y compris les répétitions vides ou affichées partiellement). Correspond au nombre maximum de répétitions que peut afficher le champ Zone répétée.<br><br><br> |

![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Ce paramètre n'est pas pris en compte dans cette version.

## Remarques


### Conditions d'utilisation
<a name="conditions_utilisation_ELTPARAGRAPHE000192"></a>La fonction **ZoneRépétéeOccurrence** peut être utilisée indifféremment sur : 

- un champ Zone répétée fichier,

- un champ Zone répétée par programmation.


## Obtention d'un certificat SSL pour IIS7 (ou supérieur)
			

<a name="NOTE0"></a>
<a name="NOTE0_1"></a>


## Les différentes étapes
<a name="les_differentes_etapes_ELTTEXTE000142"></a>
Pour obtenir un certificat SSL pour IIS 7 (et supérieur), les différentes étapes sont les suivantes : 

1. [Création de la demande de certification](#NOTE1_1). 

2. [Envoi de la demande de certification à un organisme](#NOTE2_1). 

3. [Mise en place du certificat](#NOTE3_1).




IIS 7 permet également de créer un [certificat auto-signé](#NOTE4_1).  



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Etape n°1 : Création de la demande de certification
<a name="etape_creation_demande_certification_ELTTEXTE000166"></a>
Pour créer une demande de certification :

1. Ouvrez le Gestionnaire des services Internet (option "Démarrer .. Exécuter": Inetmgr.exe).

2. Sélectionnez l'ordinateur en cours (la racine des connexions). 

3. Double-cliquez sur l'option "Certificats de serveur". 

4. Dans l'écran qui s'affiche, cliquez sur le lien "Créer une demande de certificat". Un assistant se lance. 

5. Renseignez les différentes informations demandées et indiquez le nom du fichier à générer. 






<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Etape n°2 : Envoi de la demande de certification à un organisme
<a name="etape_envoi_demande_certification_organisme_ELTTEXTE000190"></a>
La demande de certification se fait à un organisme de certification de clé (exemple de société : www.verisign.com). Cet exemple décrit la procédure à suivre avec l'organisme Vérisign. Les étapes sont sensiblement les mêmes pour chaque organisme.

Les différentes étapes sont les suivantes : 

1. Dans votre navigateur, tapez : http://www.verisign.com

2. Cliquez sur "Free Trial". Cette option permet d'obtenir un certificat valide 15 jours gratuitement.

3. Remplissez le petit questionnaire puis cliquez sur "Submit".

4. Suivez pas à pas l'assistant de certification :

	- *Step 1 of 5* : Generate CSR : Cliquez sur "Continue".

	- *Step 2 of 5* : Submit CSR : Copiez le contenu du fichier créé précédemment (Etape 1) dans la zone de saisie prévue à cet effet, puis cliquez sur "Continue".

	- *Step 3 of 5* : Complete Application : Le contenu du certificat est listé à l'écran, tapez vos coordonnées (Attention les accents ne sont pas acceptés, la plupart des informations doivent être les mêmes que dans le contenu du certificat. Exemple : la région, la société .). Renseignez bien votre adresse Email, c'est à cette adresse que vous sera envoyé le certificat. Cliquez sur "Accept".







Dans l'heure qui suit vous recevrez votre certificat.

## Certificat auto-signé
<a name="certificat_autosigne_ELTTEXTE000238"></a>
IIS 7 permet de créer un certificat auto-signé. Cette option permet de créer un certificat local pour tester le SSL. 

Pour créer un certificat auto-signé :

1. Ouvrez le Gestionnaire des services Internet (option "Démarrer .. Exécuter": Inetmgr.exe).

2. Sélectionnez l'ordinateur en cours (la racine des connexions). 

3. Double-cliquez sur l'option "Certificats de serveur". 

4. Dans l'écran qui s'affiche, cliquez sur le lien "Créer un certificat auto-signé". Un assistant se lance.

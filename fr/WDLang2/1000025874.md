
## CookieSupprime (Fonction)

***En anglais : CookieDelete***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime un cookie sur le poste de l'internaute.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Suppression d'un cookie en code serveur
// Le serveur n'a plus besoin de récupérer le nom et la date de dernière visite du client.
CookieSupprime("NOM_ET_DATE")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`CookieSupprime(<Cookie> [, <Chemin>])`
---

**`<Cookie> : (Chaîne de caractères)`**

Nom du cookie à supprimer. Ce nom a été spécifié avec la fonction [CookieEcrit](../WDLang2/3012008.md). 

**`<Chemin> : (Chaîne de caractères optionnelle)`**

Chemin d'enregistrement du cookie sur le serveur. Ce chemin a été spécifié lors de l'écriture du cookie avec la fonction [CookieEcrit](../WDLang2/3012008.md). 
Ce paramètre est sensible à la casse. 
Par défaut (ou si ce paramètre correspond à chaîne vide), le chemin correspond à "/". 
Exemple : "/COOKIES_WEB/FR/LOGIN"

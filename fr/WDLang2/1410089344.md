
## adminWBConnexion (Type de variable)

***En anglais : WBAdminConnection***
			
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **adminWBConnexion** permet de connaître toutes les caractéristiques avancées d'une connexion en cours sur le serveur. Cette connexion peut correspondre à une session, une requête AWP, une tâche planifiée, etc. Les caractéristiques de cette connexion peuvent être définies et modifiées à l'aide de différentes propriétés WLangage. 

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
wbAuthLogin est un adminWBAuth
wbAuthLogin.AdresseServeur = "https://mon-serveur.interne/"
wbAuthLogin.Utilisateur = "WWW\DATA"
wbAuthLogin.MotDePasse = "www"

ConnexionsWB est un tableau de adminWBConnexions = AdminWBListeConnexion(wbAuthLogin)
POUR i = 1 _À_ ConnexionsWB.Occurrence  
	Trace(ConnexionsWB[i].Site)
	Trace(DateHeureVersChaîne(ConnexionsWB[i].DateHeureDébutConnexion))
	Trace(DuréeVersChaîne(ConnexionsWB[i].TempsInactivité, duréeCentième))
FIN
```





<a name="PROPS"></a>

## Propriétés


### Propriétés spécifiques à la description des variables de type adminWBConnexion
<a name="proprietes_specifiques_description_des_variables_type_adminwbconnexion_ELTPARAGRAPHE000026"></a>Les propriétés suivantes peuvent être utilisées pour manipuler une connexion sur le serveur :

| Nom de la propriété | Type manipulé | Effet |
| --- | --- | --- |
| AdresseIP | Chaîne de caractères | Adresse IP associée à la connexion. |
| DateHeureDébutConnexion | Date-heure | Date et heure du début de la connexion. |
| Identifiant | Entier | Identifiant de la connexion. A utiliser par exemple pour une demande de déconnexion. |
| Site | Chaîne de caractères | Nom du site ou du Webservice utilisé par la connexion. |
| TempsInactivité | Durée | Durée depuis laquelle la connexion est inactive. |
| Type | Entier | Type de la connexion. |




## &lt;Zone répétée&gt;.DérouleTout (Fonction)

***En anglais : &lt;Looper&gt;.ExpandAll***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Déroule toutes les ruptures d'un champ Zone répétée avec rupture.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Clic sur le champ Bouton [1,2 étoiles uniquement]
ZR_HOTEL.DérouleTout()
ZR_HOTEL.Enroule(nLigne3Etoile)
ZR_HOTEL.Enroule(nLigne4Etoile)
ZR_HOTEL.Enroule(nLigne5Etoile)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Champ Zone répétée>.DérouleTout()`
---

**`<Champ Zone répétée> : (Nom de champ)`**

Nom du champ Zone répétée à manipuler.




## Remarques


### Conditions d'utilisation
<a name="conditions_utilisation_ELTPARAGRAPHE000167"></a>La fonction **&lt;Zone répétée&gt;.DérouleTout** peut être utilisée indifféremment sur : 

- un champ Zone répétée fichier,

- un champ Zone répétée par programmation.


## NavigateurFerme (Fonction)

***En anglais : BrowserClose***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ferme la fenêtre en cours du navigateur et stoppe l'exécution du code en cours. Il est également possible de fermer un navigateur spécifique.

Cas particuliers :

- **Internet Explorer** : Pour des raisons de sécurité Internet, le navigateur demande une autorisation avant de se fermer. Si l'utilisateur répond "Non" à l'autorisation de fermeture demandée par le navigateur, le navigateur n'est pas fermé, mais le code en cours d'exécution est tout de même interrompu.

- **FireFox** : Pour des raisons de sécurité Internet, FireFox refuse la fermeture du navigateur lorsque celui-ci n'a pas été ouvert par la fonction [NavigateurOuvre](../WDLang2/3055006.md).



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
NavigateurFerme()
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Fermer le navigateur en cours

`NavigateurFerme()`
---


<a name="SYNTAXE2"></a>

### Fermer le navigateur désigné par un identifiant

`NavigateurFerme(<Identifiant>)`
---

**`<Identifiant> : (Entier)`**

Identifiant de la fenêtre du navigateur à fermer. Cet identifiant est renvoyé par la fonction [NavigateurOuvre](../WDLang2/3055006.md).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Arrêter le site WEBDEV
<a name="arreter_site_webdev_ELTPARAGRAPHE000048"></a>La fonction **NavigateurFerme** permet uniquement de fermer la fenêtre en cours du navigateur : cette fonction ne permet pas d'arrêter un site WEBDEV. Pour arrêter un site WEBDEV, utilisez la fonction [FinProgramme](../WDLang1/3013033.md).




## &lt;Page&gt;.Existe (Fonction)

***En anglais : &lt;Page&gt;.Exist***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de savoir si un contexte de page existe sur le serveur (c'est-à-dire si la page a été ouverte).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Actualise la page si le contexte existe sinon ouvre la page
SI PAGE_Page3.Existe() = Vrai ALORS
	PAGE_Page3.Actualise()
SINON
	PAGE_Page3.Affiche()
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Page>.Existe()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le contexte de la page spécifiée existe (c'est-à-dire si la page spécifiée est ouverte), 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Page> : (Nom de page)`**

Nom de la page dont l'existence du contexte doit être vérifiée.

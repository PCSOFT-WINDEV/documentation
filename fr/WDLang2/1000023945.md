


## Fonctions de gestion des champs Zone répétée (syntaxe préfixée)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions permettant de gérer les champs Zone répétée (syntaxe préfixée) sont les suivantes : 



|   |   |
| --- | --- |
| [&lt;Zone répétée&gt;.Affiche](../WDLang2/1000023451.md) | Réactualise l'affichage d'une zone répétée fichier à partir d'une position donnée : les modifications effectuées sur le fichier lié sont répercutées dans la zone répétée. |
| [&lt;Zone répétée&gt;.AffichePopup](../WDLang2/1000023481.md) | Affiche la popup automatique de zoom du champ Galerie d'images. |
| [&lt;Zone répétée&gt;.Ajoute](../WDLang2/1000023469.md) | Ajoute une répétition à la fin d'un champ Zone répétée. |
| [&lt;Zone répétée&gt;.AjouteFI](../WDLang2/1000023474.md) | Ajoute une répétition sous forme d'une fenêtre interne à la fin d'un champ Zone répétée par programmation. |
| [&lt;Zone répétée&gt;.AjouteLigne](../WDLang2/1000023475.md) | Ajoute une répétition à la fin d'un champ Zone répétée. |
| [&lt;Zone répétée&gt;.AjoutEnCours](../WDLang2/1000023482.md) | Modifie ou renvoie l'état de visibilité de la fenêtre interne de patience lors de l'ajout d'éléments supplémentaires dans un champ Zone répétée infinie. |
| [&lt;Zone répétée&gt;.AjoutePI](../WDLang2/1410089490.md) | Ajoute une répétition sous forme d'une page interne à la fin d'un champ Zone répétée par programmation. |
| [&lt;Zone répétée&gt;.AnnuleBalayage](../WDLang2/1000023483.md) | Annule l'affichage de la fenêtre interne affichée dans un champ Zone répétée suite au balayage d'une répétition (ou ligne). |
| [&lt;Zone répétée&gt;.CherchePartout](../WDLang2/1000023485.md) | Recherche une valeur dans tous les attributs d'un champ Zone répétée. |
| [&lt;Zone répétée&gt;.DébutRemplissage](../WDLang2/1000023487.md) | Indique le début du remplissage d'un champ Zone répétée par programmation. |
| [&lt;Zone répétée&gt;.DéplaceLigne](../WDLang2/1000023488.md) | Déplace une répétition ou échange deux répétitions d'un champ Zone répétée. |
| [&lt;Zone répétée&gt;.Déroule](../WDLang2/1000023489.md) | Déroule une rupture du champ Zone répétée sur une ligne donnée. |
| [&lt;Zone répétée&gt;.DérouleTout](../WDLang2/1000023490.md) | Déroule toutes les ruptures d'un champ Zone répétée avec rupture. |
| [&lt;Zone répétée&gt;.Enregistre](../WDLang2/1000023496.md) | Valide la modification ou l'ajout de l'enregistrement associé à la répétition en cours dans le champ Zone répétée fichier. |
| [&lt;Zone répétée&gt;.Enroule](../WDLang2/1000023497.md) | Enroule une rupture précédemment déroulée d'un champ Zone répétée. |
| [&lt;Zone répétée&gt;.EnrouleDéroule](../WDLang2/1000023499.md) | Sur une répétition/ligne donnée d'un champ Zone répétée : <br><br>- replie une rupture (si elle est déroulée),<br><br>- déplie une rupture (si elle est enroulée).<br><br><br> |
| [&lt;Zone répétée&gt;.EnrouleTout](../WDLang2/1000023500.md) | Enroule toutes les ruptures déroulées d'un champ Zone répétée. |
| [&lt;Zone répétée&gt;.FinRemplissage](../WDLang2/1000023501.md) | Indique la fin du remplissage d'un champ Zone répétée par programmation. |
| [&lt;Zone répétée&gt;.IndiceRupture](../WDLang2/1000023502.md) | Renvoie l'indice du haut et du bas de rupture d'une rupture donnée dans un champ Zone répétée. |
| [&lt;Zone répétée&gt;.InfoXY](../WDLang2/1000023508.md) | Permet de connaître la répétition ou le champ se trouvant à une position donnée du champ Zone répétée (X, Y). |
| [&lt;Zone répétée&gt;.Insère](../WDLang2/1000023509.md) | Insère une répétition dans un champ Zone répétée. |
| [&lt;Zone répétée&gt;.InsèreFI](../WDLang2/1000023510.md) | Insère une répétition sous forme d'une fenêtre interne dans un champ Zone répétée par programmation. |
| [&lt;Zone répétée&gt;.InsèreLigne](../WDLang2/1000023511.md) | Insère une répétition dans un champ Zone répétée. |
| [&lt;Zone répétée&gt;.InsèrePI](../WDLang2/1410089491.md) | Insère une répétition sous forme d'une page interne dans un champ Zone répétée par programmation. |
| [&lt;Zone répétée&gt;.Modifie](../WDLang2/1000023512.md) | Modifie les valeurs des attributs dans une répétition d'un champ Zone répétée. |
| [&lt;Zone répétée&gt;.ModifieFI](../WDLang2/1000023519.md) | Modifie une répétition correspondant à une fenêtre interne dans un champ Zone répétée par programmation. |
| [&lt;Zone répétée&gt;.ModifieLigne](../WDLang2/1000023520.md) | Modifie les valeurs des attributs dans une répétition d'un champ Zone répétée. |
| [&lt;Zone répétée&gt;.ModifiePI](../WDLang2/1410089492.md) | Modifie une répétition correspondant à une page interne dans un champ Zone répétée par programmation. |
| [&lt;Zone répétée&gt;.NumEnr](../WDLang2/1000023521.md) | Renvoie le numéro de l'enregistrement en cours dans un champ Zone répétée fichier. |
| [&lt;Zone répétée&gt;.Occurrence](../WDLang2/1000023522.md) | Renvoie le nombre de répétitions d'un champ Zone répétée. |
| [&lt;Zone répétée&gt;.Position](../WDLang2/1000023523.md) | Déplace l'ascenseur d'un champ Zone répétée par programmation pour afficher une répétition spécifique ou renvoie l'indice de la première répétition affichée dans un champ Zone répétée par programmation. |
| [&lt;Zone répétée&gt;.RafraîchissementVisible](../WDLang2/1000023524.md) | Modifie ou renvoie l'état de visibilité du bandeau de rafraîchissement d'un champ Zone répétée. |
| [&lt;Zone répétée&gt;.RestaurePositionEtSélection](../WDLang2/1000023525.md) | Restaure la position de l'ascenseur d'un champ Zone répétée ainsi que le ou les éléments sélectionnés. |
| [&lt;Zone répétée&gt;.SauvePositionEtSélection](../WDLang2/1000023696.md) | Sauve :<br><br>- la position de l'ascenseur dans un champ Zone répétée,<br><br>- le ou les éléments sélectionnés.<br><br><br>Ces éléments peuvent être restaurés grâce à la fonction [&lt;Zone répétée&gt;.RestaurePositionEtSélection](../WDLang2/1000023525.md). |
| [&lt;Zone répétée&gt;.Select](../WDLang2/1000023697.md) | Renvoie l'indice de l'élément sélectionné dans un champ Zone répétée. |
| [&lt;Zone répétée&gt;.SelectMoins](../WDLang2/1000023698.md) | Dé-sélectionne une répétition dans un champ Zone répétée. |
| [&lt;Zone répétée&gt;.SelectOccurrence](../WDLang2/1000023699.md) | Renvoie le nombre de répétitions sélectionnées dans un champ Zone répétée. |
| [&lt;Zone répétée&gt;.SelectPlus](../WDLang2/1000023700.md) | Sélectionne une répétition dans un champ Zone répétée. |
| [&lt;Zone répétée&gt;.Supprime](../WDLang2/1000023701.md) | Supprime une répétition dans un champ Zone répétée. |
| [&lt;Zone répétée&gt;.SupprimeSelect](../WDLang2/1000023702.md) | Supprime la ou les lignes sélectionnées dans un champ Zone répétée. |
| [&lt;Zone répétée&gt;.SupprimeTout](../WDLang2/1000023703.md) | Supprime toutes les répétitions du champ Zone répétée par programmation. |
| [&lt;Zone répétée&gt;.Trie](../WDLang2/1000023704.md) | Permet de :<br><br>- Trier un champ Zone répétée selon un ou plusieurs de ses attributs.<br><br>- Trier un champ Zone répétée sur tous ses attributs.<br><br>- Annuler un tri précédemment effectué (c'est-à-dire ne plus tenir compte du tri).<br><br><br> |
| [&lt;Zone répétée&gt;.VersExcel](../WDLang2/1000023705.md) | Crée un fichier Excel avec les données d'un champ Zone répétée (champ Zone répétée fichier ou champ Zone répétée mémoire). |
| [&lt;Zone répétée&gt;.VersWord](../WDLang2/1000023706.md) | Crée un fichier Word (.RTF) avec les données d'un champ Zone répétée. |
| [&lt;Zone répétée&gt;.VersXML](../WDLang2/1000023707.md) | Crée un fichier XML avec les données d'un champ Zone répétée (champ Zone répétée fichier ou champ Zone répétée par programmation). |


## ConnexionOccurrence (Fonction)

***En anglais : ConnectionCount***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le nombre d'instances du site WEBDEV en cours d'exécution sur  le serveur.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
Info("Il y a " + ConnexionOccurrence() + " internautes connectés")
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = ConnexionOccurrence()`
---

**`<Résultat> : (Entier)`**

Nombre d'instances du site WEBDEV en cours d'exécution sur le serveur.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
La fonction **ConnexionOccurrence** permet de réaliser simplement :

- des tests de validité de licence, pour les sites limités à un certain nombre de connexions, ou facturés selon le nombre de connexions.

- des statistiques pour connaître par exemple le nombre maximum de personnes connectées à un site.


Chaque connexion peut être identifiée à partir de l'identifiant du processus grâce à la fonction [ExeDonnePID](../WDLang1/1000017239.md).

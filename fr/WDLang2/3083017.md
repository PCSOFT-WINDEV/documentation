


## CheminNavigationInsèreLien (Fonction)

***En anglais : BreadcrumbInsertLink***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Insère un lien dans un champ Chemin de navigation.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Insère un lien
CheminNavigationInsèreLien(SMP_MonChampChemin, 4, "Aide", "Page_Aide", "Aide")
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`CheminNavigationInsèreLien(<Champ Chemin de navigation> , <Numéro> , <Libellé> , <URL ou nom de la page> [, <Bulle d'aide>])`
---

**`<Champ Chemin de navigation> : (Nom de champ)`**

Nom du champ Chemin de navigation dans lequel le lien doit être inséré.

**`<Numéro> : (Entier)`**

Numéro du lien avant lequel l'insertion doit être faite. Ce numéro est compris entre 1 et le nombre d'occurrences du &lt;Champ Chemin de navigation&gt; (connu avec la propriété [Occurrence](../Proprietes/2510097.md)).

**`<Libellé> : (Chaîne de caractères)`**

Libellé du lien.

**`<URL ou nom de la page> : (Chaîne de caractères)`**

URL associée au lien ou nom logique de la page WEBDEV à ouvrir.

**`<Bulle d'aide> : (Chaîne de caractères optionnelle)`**

Description du lien, affichée dans une bulle lorsque la souris survole le lien.

<a name="description"></a>
Attention : lors de la migration d'un projet WEBDEV 1.5 vers WEBDEV 7 :

- la fonction **RépertoireWeb** a été remplacée par la fonction **RépertoireWeb_15**.

- un warning est automatiquement généré dans votre projet.




En effet, la structure du répertoire d'un projet WEBDEV a été modifiée entre WEBDEV 1.5 et WEBDEV 7.

Suivant l'utilisation de la fonction **RépertoireWeb_15** en WEBDEV 1.5, la fonction **RépertoireWeb_15** doit être remplacée :

- soit par la fonction [RépertoireWeb](../WDLang2/3012015.md).

- soit par la fonction [fRepWeb](../WDLang1/3036052.md).

<a name="NOTE0_1"></a>

## Remarques




### Comment remplacer la fonction RépertoireWEB_15
<a name="comment_remplacer_fonction_repertoireweb15_ELTPARAGRAPHE000040"></a>Par défaut, l'option "Localiser l'image dans le répertoire _WEB  en code navigateur" est cochée lors de la description d'une image. Dans ce cas, en code navigateur, l'image est automatiquement recherchée  dans le répertoire _WEB du site. Aucun code spécifique n'est nécessaire.

Si l'option "Localiser  l'image dans le répertoire _WEB en code navigateur" est décochée, le chemin complet de l'image doit être indiqué pour retrouver l'image. Il faut pour cela, utiliser la fonction [RépertoireWeb](../WDLang2/3012015.md).

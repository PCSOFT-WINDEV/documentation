


## &lt;Zone répétée&gt;.InfoXY (Fonction)

***En anglais : &lt;Looper&gt;.InfoXY***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de connaître l'élément se trouvant à une position donnée du champ Zone répétée (X, Y). Cet élément peut être : 

- une répétition, 

- un champ, 

- ![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png)![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) une rupture.




Cette fonction peut être utilisée indifféremment sur un champ Zone répétée fichier ou Zone répétée par programmation.  






<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Clic sur le bouton gauche de la souris
x est un entier = SourisPosX() 
y est un entier = SourisPosY() 
nRép est un entier = ZR_Zone1.InfoXY(ziNumLigne, x, y) 
Trace("Vous avez cliqué sur la répétition : " + nRép)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Champ Zone répétée>.InfoXY(<Type d'information> , <X> , <Y>)`
---

**`<Résultat> : (Chaîne de caractères ou entier)`**



- Information demandée.

- Une chaîne vide ("") s'il n'y a pas de champ au point spécifié.

- -1 s'il n'y a pas de numéro de répétition au point spécifié.




**`<Champ Zone répétée> : (Nom de champ)`**

Nom du champ Zone répétée à manipuler.

**`<Type d'information> : (Constante)`**

Type d'information demandée :


|   |   |
| --- | --- |
| *ziNomChamp* | Nom du champ au point (x,y). |
| *ziNomRupture* | Nom de la rupture au point (x,y).<br><br>![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Cette constante n'est pas disponible. |
| *ziNumLigne* | Numéro de la répétition au point (x,y). |
| *ziOrigineEcran* | Par défaut, le point (0,0) correspond à la ligne 1, colonne 1 (les ascenseurs étant situés à l'origine).<br>Si la constante *ziOrigineEcran* est combinée avec les constantes précédentes, le point (0,0) correspond à l'origine de l'écran.<br>Cette constante ne peut pas être utilisée seule. |



**`<X> : (Entier)`**

Coordonnée X (en pixels) à analyser. Cette coordonnée est relative au champ Zone répétée (si la constante *ziOrigineEcran* n'est pas précisée).

**`<Y> : (Entier)`**

Coordonnée Y (en pixels) à analyser. Cette coordonnée est relative au champ Zone répétée (si la constante *ziOrigineEcran* n'est pas précisée).

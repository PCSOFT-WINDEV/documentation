


## PageVisible (Fonction)

***En anglais : PageVisible***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Indique si la page est visible pour l'utilisateur. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Page visible ? Oui => on l'initialise
SI PageVisible() ALORS Initialise()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = PageVisible()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la page est visible pour l'utilisateur, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.

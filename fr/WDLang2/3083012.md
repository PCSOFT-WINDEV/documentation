


## ZoneRépétéeSupprimeTout (Fonction)

***En anglais : LooperDeleteAll***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime toutes les répétitions du champ Zone répétée par programmation.










<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Remise à blanc
ZoneRépétéeSupprimeTout(ZR_Zone1)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`ZoneRépétéeSupprimeTout(<Champ Zone répétée>)`
---

**`<Champ Zone répétée> : (Nom de champ)`**

Nom du champ Zone répétée à manipuler.
Si ce paramètre est égal à chaîne vide (""), la suppression sera effectuée sur le champ Zone répétée en cours.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Conditions d'utilisation
<a name="conditions_utilisation_ELTPARAGRAPHE000096"></a>La fonction **ZoneRépétéeSupprimeTout** peut être utilisée uniquement sur un champ Zone répétée par programmation.

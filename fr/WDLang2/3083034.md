


## &lt;Chemin de navigation&gt;.AjouteLien (Fonction)

***En anglais : &lt;Breadcrumb&gt;.AddLink***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ajoute un lien dans un champ Chemin de navigation.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SMP_MonChampChemin.AjouteLien("Aide", "Page_Aide", "Aide")
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Champ Chemin de navigation>.AjouteLien(<Libellé> , <URL ou nom de la page> [, <Bulle d'aide>])`
---

**`<Champ Chemin de navigation> : (Nom de champ)`**

Nom du champ Chemin de navigation dans lequel le lien doit être ajouté.

**`<Libellé> : (Chaîne de caractères)`**

Libellé du lien.

**`<URL ou nom de la page> : (Chaîne de caractères)`**

URL associée au lien ou nom logique de la page WEBDEV à ouvrir.
**Attention** : L'URL vers un site externe doit obligatoirement commencer par "http://".

**`<Bulle d'aide> : (Chaîne de caractères optionnelle)`**

Description du lien, affichée dans une bulle lorsque la souris survole le lien.

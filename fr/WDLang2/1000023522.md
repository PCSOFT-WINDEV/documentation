


## &lt;Zone répétée&gt;.Occurrence (Fonction)

***En anglais : &lt;Looper&gt;.Count***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le nombre de répétitions d'un champ Zone répétée.










<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Nombre de répétitions dans le champ Zone répétée "ZR_Zone1" ?
Résultat1 est un entier
Résultat1 = ZR_Zone1.Occurrence()
```
<a name="Exemple2"></a>

```wl
// Affiche le nombre de réponses trouvées : (ex : "Il y a 31 réponses")
LIB_LIBELLE1 = "Il y a " + ZR_ZoneResultat.Occurrence() + " réponses"
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Champ Zone répétée>.Occurrence([<Répétition/Attribut>])`
---

**`<Résultat> : (Entier)`**



- Nombre de répétitions du champ Zone répétée, 

- 0 si le champ Zone répétée est vide.




**`<Champ Zone répétée> : (Nom de champ)`**

Nom du champ Zone répétée à manipuler.

**`<Répétition/Attribut> : (Constante (optionnelle))`**




|   |   |
| --- | --- |
| *toAffichée*<br> | Nombre de répétitions non vides visibles.<br><br><br> |
| *toColonne* | Nombre d'attributs présents dans le champ Zone répétée. |
| *toTotal*<br>(Valeur par défaut)<br> | **Champ Zone répétée fichier** :<br><br>- Nombre total d'enregistrements :<br><br>	- du fichier de données lié (s'il n'y a pas de filtre). <br><br>	- filtrés du fichier de données lié (s'il y a un filtre).<br><br>	- de la requête (si le champ Zone répétée est liée à une requête).<br><br><br><br><br>- Nombre d'enregistrements visibles.<br><br><br>**Attention** : l'événement "Fin d'initialisation" est exécuté uniquement lorsque le total est calculé.<br><br>**Champ Zone répétée par programmation** :<br>Nombre total de répétitions (répétitions ajoutées par les fonctions [&lt;Zone répétée&gt;.Ajoute](../WDLang2/1000023469.md), [&lt;Zone répétée&gt;.AjouteLigne](../WDLang2/1000023475.md), [&lt;Zone répétée&gt;.Insère](../WDLang2/1000023509.md) et [&lt;Zone répétée&gt;.InsèreLigne](../WDLang2/1000023511.md)). |
| *toVisible* | Nombre de répétitions visibles (y compris les répétitions vides ou affichées partiellement). Correspond au nombre maximum de répétitions que peut afficher le champ Zone répétée.<br><br><br> |

![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Ce paramètre n'est pas pris en compte dans cette version.

## Remarques


### Conditions d'utilisation
<a name="conditions_utilisation_ELTPARAGRAPHE000319"></a>La fonction **&lt;Zone répétée&gt;.Occurrence** peut être utilisée indifféremment sur : 

- un champ Zone répétée fichier,

- un champ Zone répétée par programmation.


## HTMLClasseSupprime (Fonction)

***En anglais : HTMLClassDelete***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime une classe des classes HTML d'un champ.

Remarque : Il est également possible de connaître et de modifier la valeur de l'attribut HTML "class" d'un champ via la propriété [ClasseHTML](../Proprietes/1000021530.md). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
HTMLClasseSupprime(SAI_Saisie1, "SaisieEnErreur")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`HTMLClasseSupprime(<Champ> , <Classe à supprimer>)`
---

**`<Champ> : (Nom de champ)`**

Nom du champ à manipuler.

**`<Classe à supprimer> : (Chaîne de caractères)`**

Nom de la classe à supprimer. Si la classe n'existe pas, la fonction n'a pas d'effet.

Si plusieurs classes doivent être supprimées, ce paramètre doit contenir le nom des différentes classes séparé par des espaces. Chaque classe est supprimée indépendamment.

Attention : Le nom des classes est sensible à la casse.

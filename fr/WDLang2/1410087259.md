


## PopupFermePage (Fonction)

***En anglais : PopupClosePage***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Masque une page affichée en popup via la fonction [PopupAffichePage](../WDLang2/1410087258.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ferme la page affichée en popup
PopupFermePage(PAGE_DétailsProduit)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`PopupFermePage([<Page> [, <Valeur renvoyée>]])`
---

**`<Page> : (Nom de page optionnel)`**

Nom de la page affichée en popup qui sera masquée. Cette page doit avoir été affichée par la fonction [PopupAffichePage](../WDLang2/1410087258.md). Si ce n'est pas le cas, la fonction **PopupFermePage** n'a aucune action.

Si la fonction **PopupFermePage** est utilisée dans le code de la page principale, ce paramètre est obligatoire : le nom de la page popup à fermer doit être précisée. 
Si la fonction **PopupFermePage** est utilisée dans le code de la page courante (affichée comme popup), ce paramètre n'est pas obligatoire : la page courante est fermée. 

**`<Valeur renvoyée> : (Type correspondant à la valeur renvoyée (optionnel))`**





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Les navigateurs ne rafraîchissent pas l'affichage au milieu d'un code Javascript : les fonctions [PopupAffichePage](../WDLang2/1410087258.md) et **PopupFermePage** doivent être utilisées dans des traitements/événements différents. Si ces fonctions sont utilisées dans le même traitement/événement, la page Popup n'est pas affichée.

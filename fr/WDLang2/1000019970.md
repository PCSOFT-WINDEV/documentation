


## ZoneRépétéeEnrouleDéroule (Fonction)

***En anglais : LooperCollapseExpand***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Sur une répétition (ou ligne) donnée d'un champ Zone répétée : 

- replie une rupture (si elle est déroulée),

- déplie une rupture (si elle est enroulée).



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Déroule sur le clic utilisateur
ZoneRépétéeEnrouleDéroule(ZR_Produits)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = ZoneRépétéeEnrouleDéroule(<Champ Zone répétée> [, <Numéro de la répétition/ligne> [, <Nom de la rupture>]])`
---

**`<Résultat> : (Constante de type Entier)`**

Etat de la répétition/ligne avant l'appel à la fonction **ZoneRépétéeEnrouleDéroule** : 



|   |   |
| --- | --- |
| *taDéroule* | Répétition/ligne déroulée. |
| *taEnroule* | Répétition/ligne enroulée. |
| *taErreur* | Répétition/ligne inexistante. |



**`<Champ Zone répétée> : (Nom de champ)`**

Nom du champ Zone répétée à manipuler.
Si ce paramètre est égal à chaîne vide (""), le champ Zone répétée en cours est manipulé.

**`<Numéro de la répétition/ligne> : (Entier optionnel)`**

Numéro de la répétition/ligne à replier ou déplier.
Si ce paramètre n'est pas spécifié, la répétition/ligne en cours est manipulée.

**`<Nom de la rupture> : (Chaîne de caractères optionnelle)`**

Nom de la rupture à modifier. Ce paramètre doit être spécifié dans le cas de ruptures imbriquées. Il permet de définir la rupture qui sera manipulée. Par défaut, si des ruptures sont imbriquées, aucune rupture n'est enroulée ou déroulée.




## Remarques


### Conditions d'utilisation
<a name="conditions_utilisation_ELTPARAGRAPHE000084"></a>La fonction **ZoneRépétéeEnrouleDéroule** peut être utilisée indifféremment sur : 

- un champ Zone répétée fichier,

- un champ Zone répétée par programmation.

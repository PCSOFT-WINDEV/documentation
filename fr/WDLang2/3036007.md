 

## RépertoireDonnées (Fonction)

***En anglais : DataDirectory***
	

<a name="XUtilisation"></a>
<a name="Utilisation"></a>
|  |   |
| --- | --- |



|  
 | **But**<br><br>Connaître le répertoire des données HFSQL.<br><br>**Syntaxe**<br><br>&lt;Résultat&gt; = RépertoireDonnées()<br><br>&lt;Resultat&gt; est une chaîne de caractères, correspondant au chemin absolu permettant d'accéder aux données.<br><br>**Détail**<br><br>La fonction **RépertoireDonnées** renvoie le chemin des fichiers de données HFSQL.<br><br>Dans le cas d'une installation par média physique, ce répertoire est donné lors de l'installation.<br><br>Dans le cas d'une installation par accès distant (FTP), l'hébergeur fournit le répertoire de base des fichiers de données, et l'utilisateur fournit le sous-répertoire dans lequel les fichiers de données sont installés.<br><br>En mode test, cette fonction retourne une chaîne vide.<br><br>Le répertoire des données est le répertoire où se trouvent les fichiers HFSQL par défaut, c'est-à-dire :<br><br>- si la fonction [HSubstRep](../WDLang4/3044028.md) n'est pas utilisée.<br><br>- si les fichiers sont décrits dans l'analyse avec un répertoire spécifique (vivement déconseillé)<br><br><br><br><br>**Notes**<br><br>La chaîne renvoyée ne se termine pas par un '\' (anti slash)<br><br>Le chemin renvoyé est :<br><br>- soit un chemin complet. Exemple : "D:\DATA\Rewali.<br><br>- soit un nom UNC. Exemple : \\SERVEURA\DATA\Rewali<br><br><br><br><br> |
| --- | --- |

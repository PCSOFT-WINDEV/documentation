


## ImageCalqueVisible (Fonction)

***En anglais : ImageLayerVisible***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de rendre visible ou invisible un calque d'une image de type WDPIC.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affiche l'image d'un produit dans un champ Image
PROCEDURE AfficheProduit(sChampImage est un Champ, sNomWDPic est une chaîne, ...
		bPromo est un booléen)

MonImage est un WDPic
MonImage.NomFichier = sNomWDPic
ImageCalqueVisible(MonImage, "Calque Promo", bPromo)

sChampImage.Valeur = MonImage
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = ImageCalqueVisible(<Image WDPic> , <Nom du calque> , <Visible>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le calque existe, 

- <u><u><u><u>Faux</u></u></u></u> si le calque n'existe pas. En cas d'échec, la variable [ErreurDétectée](../WDLang1/3087001.md) vaut <u><u><u><u>Vrai</u></u></u></u> et la fonction [ErreurInfo](../WDLang1/3013008.md) permet de connaître le détail de l'erreur.




**`<Image WDPic> : (Variable de type WDPic)`**

Nom de la variable de type [WDPic](../WDLang1/1000023503.md) correspondant à l'image manipulée. 

**`<Nom du calque> : (Chaîne de caractères)`**

Nom du calque à manipuler. Ce nom doit être identique au nom défini sous l'éditeur d'images.

**`<Visible> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le calque doit être visible, 

- <u><u><u><u>Faux</u></u></u></u> si le calque doit être invisible. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Un calque peut être rendu visible ou invisible par programmation uniquement si le calque possède l'option "Visible/Invisible par programmation". Cette option est disponible dans le menu contextuel du calque. Cette option peut être activée : 

- sous l'éditeur d'images, 

- dans le champ Editeur d'images.


Pour plus de détails, consultez [Gestion des calques](../Editeurs/1410086606.md).

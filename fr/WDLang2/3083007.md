


## Fonctions de gestion des champs Zone répétée
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>






Les fonctions permettant de gérer les champs Zone répétée sont les suivantes :



|   |   |
| --- | --- |
| [GalerieAffichePopup](../WDLang2/1000023476.md) | Affiche la popup automatique de zoom du champ Galerie d'images. |
| [GalerieFermePopup](../WDLang2/1000023478.md) | Ferme la popup automatique de zoom de la galerie d'images. |
| [GaleriePopupAffichée](../WDLang2/1000024975.md) | Indique si la popup automatique de zoom du champ Galerie d'images est affichée. |
| [ZoneRépétéeAffiche](../WDLang2/3083001.md) | Réactualise l'affichage d'une zone répétée fichier à partir d'une position donnée : les modifications effectuées sur le fichier lié sont répercutées dans la zone répétée. |
| [ZoneRépétéeAjoute](../WDLang2/3083006.md) | Ajoute une répétition à la fin d'un champ Zone répétée. |
| [ZoneRépétéeAjouteFI](../WDLang2/1000023439.md) | Ajoute une répétition sous forme d'une fenêtre interne à la fin d'un champ Zone répétée par programmation. |
| [ZoneRépétéeAjouteLigne](../WDLang2/3083004.md) | Ajoute une répétition à la fin d'un champ Zone répétée. |
| [ZoneRépétéeAjoutEnCours](../WDLang2/1000021638.md) | Modifie ou renvoie l'état de visibilité de la fenêtre interne de patience lors de l'ajout d'éléments supplémentaires dans un champ Zone répétée infinie. |
| [ZoneRépétéeAjoutePI](../WDLang2/1410089366.md) | Ajoute une répétition sous forme d'une page interne à la fin d'un champ Zone répétée par programmation. |
| [ZoneRépétéeAnnuleBalayage](../WDLang2/1000021988.md) | Annule l'affichage de la fenêtre interne affichée dans un champ Zone répétée suite au balayage d'une répétition (ou ligne). |
| [ZoneRépétéeCherche](../WDLang2/3083005.md) | Recherche une valeur dans un attribut d'un champ Zone répétée. |
| [ZoneRépétéeCherchePartout](../WDLang2/1000021997.md) | Recherche une valeur dans tous les attributs d'un champ Zone répétée. |
| [ZoneRépétéeDébutRemplissage](../WDLang2/1000022370.md) | Indique le début du remplissage d'un champ Zone répétée par programmation. |
| [ZoneRépétéeDéplaceLigne](../WDLang2/1000017248.md) | Déplace une répétition ou échange deux répétitions d'un champ Zone répétée. |
| [ZoneRépétéeDéroule](../WDLang2/1000019771.md) | Déroule une rupture du champ Zone répétée sur une ligne donnée. |
| [ZoneRépétéeDérouleTout](../WDLang2/1000019774.md) | Déroule toutes les ruptures d'un champ Zone répétée avec rupture. |
| [ZoneRépétéeEnregistre](../WDLang2/3083028.md) | Valide la modification ou l'ajout de l'enregistrement associé à la répétition en cours dans le champ Zone répétée fichier. |
| [ZoneRépétéeEnroule](../WDLang2/1000019770.md) | Enroule une rupture précédemment déroulée d'un champ Zone répétée. |
| [ZoneRépétéeEnrouleDéroule](../WDLang2/1000019970.md) | Sur une répétition/ligne donnée d'un champ Zone répétée : <br><br>- replie une rupture (si elle est déroulée),<br><br>- déplie une rupture (si elle est enroulée).<br><br><br> |
| [ZoneRépétéeEnrouleTout](../WDLang2/1000019772.md) | Enroule toutes les ruptures déroulées d'un champ Zone répétée. |
| [ZoneRépétéeFinRemplissage](../WDLang2/1000022371.md) | Indique la fin du remplissage d'un champ Zone répétée par programmation. |
| [ZoneRépétéeIndiceRupture](../WDLang2/3083029.md) | Renvoie l'indice du haut et du bas de rupture d'une rupture donnée dans un champ Zone répétée. |
| [ZoneRépétéeInfoXY](../WDLang2/3083022.md) | Permet de connaître la répétition ou le champ se trouvant à une position donnée du champ Zone répétée (X, Y). |
| [ZoneRépétéeInsère](../WDLang2/3083003.md) | Insère une répétition dans un champ Zone répétée. |
| [ZoneRépétéeInsèreFI](../WDLang2/1000023440.md) | Insère une répétition sous forme d'une fenêtre interne dans un champ Zone répétée par programmation. |
| [ZoneRépétéeInsèreLigne](../WDLang2/3083002.md) | Insère une répétition dans un champ Zone répétée. |
| [ZoneRépétéeInsèrePI](../WDLang2/1410089367.md) | Insère une répétition sous forme d'une page interne dans un champ Zone répétée par programmation. |
| [ZoneRépétéeModifie](../WDLang2/3083013.md) | Modifie les valeurs des attributs dans une répétition d'un champ Zone répétée. |
| [ZoneRépétéeModifieFI](../WDLang2/1000023441.md) | Modifie une répétition correspondant à une fenêtre interne dans un champ Zone répétée par programmation. |
| [ZoneRépétéeModifieLigne](../WDLang2/3083015.md) | Modifie les valeurs des attributs dans une répétition d'un champ Zone répétée (fichier ou mémoire). |
| [ZoneRépétéeModifiePI](../WDLang2/1410089368.md) | Modifie une répétition correspondant à une page interne dans un champ Zone répétée par programmation. |
| [ZoneRépétéeNumEnr](../WDLang2/3083010.md) | Renvoie le numéro de l'enregistrement en cours dans un champ Zone répétée fichier. |
| [ZoneRépétéeOccurrence](../WDLang2/3083014.md) | Renvoie le nombre de répétitions d'un champ Zone répétée. |
| [ZoneRépétéePosition](../WDLang2/3083023.md) | Déplace l'ascenseur d'un champ Zone répétée par programmation pour afficher une répétition spécifique ou renvoie l'indice de la première répétition affichée dans un champ Zone répétée par programmation. |
| [ZoneRépétéeRafraîchissementVisible](../WDLang2/1000021195.md) | Modifie ou renvoie l'état de visibilité du bandeau de rafraîchissement d'un champ Zone répétée. |
| [ZoneRépétéeRestaurePositionEtSélection](../WDLang2/1000021789.md) | Restaure la position de l'ascenseur d'un champ Zone répétée ainsi que le ou les éléments sélectionnés. |
| [ZoneRépétéeSauvePositionEtSélection](../WDLang2/1000021788.md) | Sauve :<br><br>- la position de l'ascenseur dans un champ Zone répétée,<br><br>- le ou les éléments sélectionnés.<br><br><br>Ces éléments peuvent être restaurés grâce à la fonction [ZoneRépétéeRestaurePositionEtSélection](../WDLang2/1000021789.md). |
| [ZoneRépétéeSelect](../WDLang2/3083024.md) | Renvoie l'indice de l'élément sélectionné dans un champ Zone répétée. |
| [ZoneRépétéeSelectMoins](../WDLang2/3083026.md) | Dé-sélectionne une répétition dans un champ Zone répétée. |
| [ZoneRépétéeSelectOccurrence](../WDLang2/3083027.md) | Renvoie le nombre de répétitions sélectionnées dans un champ Zone répétée. |
| [ZoneRépétéeSelectPlus](../WDLang2/3083025.md) | Sélectionne une répétition dans un champ Zone répétée. |
| [ZoneRépétéeSupprime](../WDLang2/3083008.md) | Supprime une répétition dans un champ Zone répétée. |
| [ZoneRépétéeSupprimeSelect](../WDLang2/1000023336.md) | Supprime la ou les lignes sélectionnées dans un champ Zone répétée. |
| [ZoneRépétéeSupprimeTout](../WDLang2/3083012.md) | Supprime toutes les répétitions du champ Zone répétée par programmation. |
| [ZoneRépétéeTrie](../WDLang2/3083009.md) | Permet de :<br><br>- Trier un champ Zone répétée selon un ou plusieurs de ses attributs.<br><br>- Trier un champ Zone répétée sur tous ses attributs.<br><br>- Annuler un tri précédemment effectué (c'est-à-dire ne plus tenir compte du tri).<br><br><br> |
| [ZoneRépétéeVersExcel](../WDLang2/3083030.md) | Crée un fichier Excel avec les données d'un champ Zone répétée (champ Zone répétée fichier ou champ Zone répétée mémoire). |
| [ZoneRépétéeVersWord](../WDLang2/3083031.md) | Crée un fichier Word (.RTF) avec les données d'un champ Zone répétée. |
| [ZoneRépétéeVersXML](../WDLang2/3083032.md) | Crée un fichier XML avec les données d'un champ Zone répétée (champ Zone répétée fichier ou champ Zone répétée par programmation). |

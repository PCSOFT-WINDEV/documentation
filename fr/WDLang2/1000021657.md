


## Fonctions de gestion des pages (syntaxe préfixée)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions de gestion des pages (syntaxe préfixée) sont les suivantes :



|   |   |
| --- | --- |
| [&lt;Cellule&gt;.AfficheDialogue](../WDLang2/1000021652.md) | Affiche un champ Cellule dans la page avec un effet de GFI (Grisage des Fenêtres Inaccessibles). Cette fonction permet de simuler simplement une boîte de dialogue en code navigateur, en utilisant un champ Cellule dans une page. |
| [&lt;Cellule&gt;.FermeDialogue](../WDLang2/1000021653.md) | Masque le champ Cellule précédemment affiché dans la page via la fonction &lt;Champ Cellule&gt;.AfficheDialogue. |
| [&lt;Page&gt;.Actualise](../WDLang2/1000021659.md) | Actualise une page affichée dans le navigateur de l'internaute à partir du contexte présent sur le serveur. |
| [&lt;Page&gt;.Adresse](../WDLang2/1000021660.md) | Permet d'obtenir l'adresse Internet d'une page. |
| [&lt;Page&gt;.Affiche](../WDLang2/1000021661.md) | Affiche une page du site dans le navigateur de l'internaute. |
| [&lt;Page&gt;.AfficheDialogue](../WDLang2/1000021663.md) | Affiche une page de manière modale. Cette fonction permet de créer un dialogue avec l'utilisateur. La page est affichée devant et en arrière-plan se trouve la page ouvrante, grisée par le mécanisme de GFI. |
| [&lt;Page&gt;.AffichePopup](../WDLang2/1410087260.md) | Affiche une page WEBDEV comme une popup dans la page courante avec un effet de GFI (Grisage des Fenêtres Inaccessibles). |
| [&lt;Page&gt;.Existe](../WDLang2/1000021658.md) | Permet de savoir si un contexte de page existe sur le serveur (c'est-à-dire si la page a été ouverte). |
| [&lt;Page&gt;.FermePopup](../WDLang2/1410087261.md) | Masque une page affichée en popup affichée dans la page via la fonction [&lt;Page&gt;.AffichePopup](../WDLang2/1410087260.md). |
| [&lt;Page&gt;.Utilise](../WDLang2/1000021664.md) | Affiche une page WEBDEV dans le navigateur de l'internaute et ferme tous les contextes de pages en cours. |
| [&lt;Popup&gt;.Affiche](../WDLang2/1000021654.md) | Affiche une popup dans la page avec un effet de GFI (Grisage des Fenêtres Inaccessibles). |
| [&lt;Popup&gt;.Anime](../WDLang2/1000021655.md) | Affiche une popup dans une cellule de la page. |
| [&lt;Popup&gt;.Ferme](../WDLang2/1000021656.md) | Masque une popup affichée dans la page via la fonction [&lt;Popup&gt;.Affiche](../WDLang2/1000021654.md). |

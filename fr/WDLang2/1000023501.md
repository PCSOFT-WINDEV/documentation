


## &lt;Zone répétée&gt;.FinRemplissage (Fonction)

***En anglais : &lt;Looper&gt;.EndFilling***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Indique la fin du remplissage d'un champ Zone répétée par programmation. 

Rappel : Le début du remplissage a été indiqué avec la fonction [&lt;Zone répétée&gt;.DébutRemplissage](../WDLang2/1000023487.md). L'avancement du remplissage est représenté :

- soit par une jauge infinie affichée au bas du champ.

- soit par une fenêtre interne.




Attention : Avant d'utiliser cette fonction, vous devez activer le mécanisme de gestion du remplissage dans la fenêtre de description du champ. Pour plus de détails, consultez [Caractéristiques d'un champ Zone répétée dans une fenêtre](../WDChamp/1013294.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
ZR_MaZone.DébutRemplissage()

POUR i = 1 A 10000
	ZR_MaZone.AjouteLigne()
	Multitâche(-1)
FIN

ZR_MaZone.FinRemplissage()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Champ Zone répétée>.FinRemplissage()`
---

**`<Champ Zone répétée> : (Nom de champ)`**

Nom du champ Zone répétée à manipuler.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Cette fonction est sans effet sur les champs Zone répétée reliés à un fichier de données ou à une requête.

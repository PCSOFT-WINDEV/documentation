


## &lt;Zone répétée&gt;.ModifieLigne (Fonction)

***En anglais : &lt;Looper&gt;.ModifyLine***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie les valeurs des attributs dans une répétition d'un champ Zone répétée.

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Pour les champs Zone répétée sans attributs, il est conseillé de modifier directement la valeur des champs concernés. Pour plus de détails, consultez le paragraphe Modifier un champ Zone répétée sans attributs.










<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Modification d'une ligne dans un champ Zone répétée
ZR_Zone1.ModifieLigne(Indice, NomProduit, CodeProduit)
```
<a name="Exemple2"></a>

```wl
// Modifie une valeur dans un champ Zone répétée en 12ème position
ZR_Zone1.ModifieLigne(12, Nom, Prénom, NumTel, NumFax)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Champ Zone répétée>.ModifieLigne([<Indice> [, <Valeur 1> [, <Valeur N>]]])`
---

**`<Champ Zone répétée> : (Nom de champ)`**

Nom du champ Zone répétée à manipuler.

**`<Indice> : (Entier optionnel)`**

Indice de la répétition à modifier.

- Si ce paramètre n'est pas précisé, la modification sera effectuée sur l'élément en cours.

- Si &lt;Indice&gt; est supérieur au nombre de répétitions, une erreur WLangage est affichée.




**`<Valeur 1> : (Type de l'attribut associé (optionnel))`**

Valeur du 1er attribut de la ligne à modifier dans le champ Zone répétée spécifié.
Les différentes valeurs doivent être indiquées dans l'ordre des attributs (fenêtre de description du champ Zone répétée).
Si aucun de ces paramètres n'est spécifié, la fonction **&lt;Zone répétée&gt;.ModifieLigne** affecte chaîne vide ("") ou 0.

**`<Valeur N> : (Type de l'attribut associé (optionnel))`**

Valeur du Nème attribut de la ligne à modifier dans le champ Zone répétée spécifié.
Les différentes valeurs doivent être indiquées dans l'ordre des attributs (fenêtre de description du champ Zone répétée).
Si aucun des ces paramètres n'est spécifié, la fonction **&lt;Zone répétée&gt;.ModifieLigne** affecte chaîne vide ("") ou 0.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Conditions d'utilisation
<a name="conditions_utilisation_ELTPARAGRAPHE000275"></a>La fonction **&lt;Zone répétée&gt;.ModifieLigne** peut être utilisée indifféremment sur : 

- un champ Zone répétée fichier,

- un champ Zone répétée par programmation. 



<a name="NOTE0_2"></a>


### Caractéristiques de la répétition
<a name="caracteristiques_repetition_ELTPARAGRAPHE000306"></a>Si une valeur correspond au résultat d'un calcul, mettez l'expression numérique entre parenthèses. Par exemple :

```wl
ZR_ZoneCalcul.ModifieLigne(5, (53+29), (83-21))
```

<a name="NOTE0_3"></a>


### Tri
<a name="tri_ELTPARAGRAPHE000322"></a>Si la fonction **&lt;Zone répétée&gt;.ModifieLigne** modifie un attribut de tri du champ Zone répétée en cours, le tri sur cet attribut sera annulé.
<a name="NOTE0_4"></a>


### Modifier un champ Zone répétée sans attributs
<a name="modifier_champ_zone_repetee_sans_attributs_ELTPARAGRAPHE000335"></a>Pour modifier un champ Zone répétée sans attributs, il est conseillé de modifier directement la valeur des champs concernés.

Par exemple :

```wl
// Modification de la valeur des champs
// du champ Zone répétée "ZR_Zone1" pour la répétition 23
ZR_Zone1[23].ChampNom = "Hobbes"
ZR_Zone1[23].ChampPrénom = "Samantha"
```

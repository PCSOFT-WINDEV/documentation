
## HTMLClasseBascule (Fonction)

***En anglais : HTMLClassToggle***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Bascule une classe dans les classes HTML d'un champ : si la classe n'existe pas, elle est ajoutée, si la classe existe déjà, elle est supprimée.

Remarque : Il est également possible de connaître et de modifier la valeur de l'attribut HTML "class" d'un champ via la propriété [ClasseHTML](../Proprietes/1000021530.md). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
HTMLClasseBascule(SAI_Saisie1, "SaisieEnErreur")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`HTMLClasseBascule(<Champ> , <Classe à basculer>)`
---

**`<Champ> : (Nom de champ)`**

Nom du champ à manipuler.

**`<Classe à basculer> : (Chaîne de caractères)`**

Nom de la classe à basculer. Si une classe est présente n fois dans la chaîne, cette classe est basculée n fois.

Si plusieurs classes doivent être basculées, ce paramètre doit contenir le nom des différentes classes séparé par des espaces. Chaque classe sera basculée indépendamment.

Attention : Le nom des classes est sensible à la casse.

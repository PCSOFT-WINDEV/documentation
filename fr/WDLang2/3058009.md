


## PageCourante (Fonction)

***En anglais : CurrentPage***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le nom de la page contenant le code WLangage en cours d'exécution.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
Ch est une chaîne
Ch = PageCourante()
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = PageCourante()`
---

**`<Résultat> : (Chaîne de caractères)`**




|   |   |
| --- | --- |
| Nom en majuscules de la page en cours | Si le code WLangage en cours d'exécution est :<br><br>- un événement associé à la page : <br><br><br><br><br>- un événement associé à un champ (initialisation, clic).<br><br>- une procédure locale de la page.<br><br><br> |
| Chaîne vide ("") | Si le code en cours d'exécution est :<br><br><br> |

![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) La casse du nom de la page correspond à la casse réelle du nom de la page. Par exemple, si la page courante correspond à la page "Accueil.WWH", le nom retourné sera "Accueil".

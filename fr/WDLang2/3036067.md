
## JSInfoEvénement (Fonction)

***En anglais : JSInfoEvent***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de manipuler les propriétés JavaScript de l'événement navigateur qui a déclenché l'exécution du code. Permet par exemple : 

- de lire une propriété JavaScript de l'événement.

- d'affecter une valeur à une propriété JavaScript de l'événement. 




Cette fonction est une fonction avancée : son utilisation nécessite une bonne connaissance de JavaScript.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Lecture de la position de la souris lors du clic
nPosX est un entier = JSInfoEvénement("clientX")
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Récupérer la valeur d'une propriété de l'événement

`<Résultat> = JSInfoEvénement([<Propriété>])`
---

**`<Résultat> : (Variant)`**

Valeur de la propriété.

**`<Propriété> : (Chaîne de caractères optionnelle)`**

Propriété à manipuler. Si ce paramètre n'est pas précisé, la fonction **JSInfoEvénement** récupère l'objet directement. Il est ainsi possible d'utiliser la fonction **JSInfoEvénement** comme argument d'une procédure navigateur écrite en JavaScript.  


<a name="SYNTAXE2"></a>

### Modifier une propriété de l'événement 

`JSInfoEvénement(<Propriété>)`
---

**`<Propriété> : (Chaîne de caractères)`**

Propriété à manipuler.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Les objets "événement" sont différents selon les navigateurs. Certaines propriétés ne sont pas disponibles sur tous les navigateurs.

- L'événement qui a déclenché l'exécution du code n'est pas toujours disponible. Avant d'utiliser cet événement, il est conseillé de tester son existence, par exemple avec le code suivant :
	
	```wl
	SI JSInfoEvénement() ALORS ...
	```


- Dans les traitements WEBDEV saisis directement en Javascript, la fonction **JSInfoEvénement** ne doit pas être utilisée : il faut utiliser directement l'objet **Event**.

- La fonction **JSInfoEvénement** n'est pas disponible :

	- dans les procédures navigateur.

	- dans les événements appelés directement ou indirectement depuis une procédure navigateur (par l'intermédiaire de la fonction [ExécuteTraitement](../WDLang1/3013031.md) par exemple).

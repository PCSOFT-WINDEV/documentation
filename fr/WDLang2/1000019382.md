
## StockageLocalAjoute (Fonction)

***En anglais : LocalStorageAdd***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ajoute une valeur au stockage local.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
StockageLocalAjoute(SAI_SaisieNom)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Ajouter directement une valeur

`StockageLocalAjoute(<Nom de la valeur> , <Valeur> [, <Persistant>])`
---

**`<Nom de la valeur> : (Chaîne de caractères)`**

Nom de la valeur ajoutée.

**`<Valeur> : (Chaîne de caractères)`**

Valeur ajoutée.

**`<Persistant> : (Booléen optionnel)`**



- <u><u><u><u>Vrai</u></u></u></u> pour travailler sur le stockage local persistant (valeur par défaut). 
	Dans ce cas, les valeurs pourront être partagées par toutes les fenêtres et onglets d'un même navigateur d'une machine. Il sera donc possible de les retrouver après avoir fermé puis ré-ouvert le navigateur ou dans une autre fenêtre du navigateur.




- <u><u><u><u>Faux</u></u></u></u> pour travailler sur le stockage local à une fenêtre ou un onglet du navigateur. 
	Dans ce cas, les valeurs seront perdues lors de la fermeture de la fenêtre ou de l'onglet du navigateur. Par contre, ces valeurs pourront être lues par les fenêtres ou onglets du navigateur ouvertes depuis la fenêtre où ont été stockées les valeurs.





<a name="SYNTAXE2"></a>

### Ajouter la valeur d'un champ

`StockageLocalAjoute(<Nom de l'élément> [, <Persistant>])`
---

**`<Nom de l'élément> : (Chaîne de caractères)`**

Nom du champ dont la valeur est ajoutée.

**`<Persistant> : (Booléen optionnel)`**



- <u><u><u><u>Vrai</u></u></u></u> pour travailler sur le stockage local persistant (valeur par défaut). 
	Dans ce cas, les valeurs pourront être partagées par toutes les fenêtres et onglets d'un même navigateur d'une machine. Il sera donc possible de les retrouver après avoir fermé puis ré-ouvert le navigateur ou dans une autre fenêtre du navigateur.




- <u><u><u><u>Faux</u></u></u></u> pour travailler sur le stockage local à une fenêtre ou un onglet du navigateur. 
	Dans ce cas, les valeurs seront perdues lors de la fermeture de la fenêtre ou de l'onglet du navigateur. Par contre, ces valeurs pourront être lues par les fenêtres ou onglets du navigateur ouvertes depuis la fenêtre où ont été stockées les valeurs.

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les différentes constantes de gestion des champs Zone répétée sont les suivantes :




|   |   |   |
| --- | --- | --- |
| **Constante** | **Détail** | **Utilisée par les fonctions** |
| **ziNomChamp** | Nom du champ au point x, y. | [ZoneRépétéeInfoXY](../WDLang2/3083022.md) |
| **ziNomRupture** | Nom de la rupture au point x, y. | [ZoneRépétéeInfoXY](../WDLang2/3083022.md) |
| **ziNumLigne** | Numéro de la répétition au point x, y. | [ZoneRépétéeInfoXY](../WDLang2/3083022.md) |
| **ziOrigineEcran** | L'origine de l'écran correspond au point (0,0). | [ZoneRépétéeInfoXY](../WDLang2/3083022.md) |
| **zrAnimationAucune** | Aucune animation n'est jouée lors de la suppression. | [ZoneRépétéeSupprime](../WDLang2/3083008.md) |
| **zrAnimationSuppression** | Joue une animation lors de la suppression. Cette animation correspond à un effet de disparition par diminution de la hauteur de la répétition. | [ZoneRépétéeSupprime](../WDLang2/3083008.md) |

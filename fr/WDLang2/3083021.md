<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


Les fonctions permettant de gérer les champs Chemin de navigation sont les suivantes :



|   |   |
| --- | --- |
| [CheminNavigationAjouteLien](../WDLang2/3083016.md) | Ajoute un lien dans un champ Chemin de navigation. |
| [CheminNavigationInsèreLien](../WDLang2/3083017.md) | Insère un lien dans un champ Chemin de navigation. |
| [CheminNavigationModifieLien](../WDLang2/3083018.md) | Modifie un lien dans un champ Chemin de navigation. |
| [CheminNavigationSupprimeLien](../WDLang2/3083019.md) | Supprime un lien dans un champ Chemin de navigation. |
| [CheminNavigationSupprimeTout](../WDLang2/3083020.md) | Supprime tous les liens dans un champ Chemin de navigation. |

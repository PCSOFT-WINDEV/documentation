


## Fonctions de gestion des navigateurs
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions de gestion des navigateurs sont les suivantes :



|   |   |
| --- | --- |
| [AjouteFavori](../WDLang2/3055004.md) | Ajoute une adresse Internet dans la liste des favoris de l'internaute. |
| [CookieEcrit](../WDLang2/3012008.md) | Ecrit un cookie sur le poste de l'internaute. |
| [CookieLit](../WDLang2/3012009.md) | Récupère la valeur d'un cookie enregistré sur le poste de l'internaute. |
| [CookieSupprime](../WDLang2/1000025874.md) | Supprime un cookie sur le poste de l'internaute. |
| [EmailOuvreMessagerie](../WDLang2/3032021.md) | Ouvre le logiciel de messagerie par défaut : <br><br>- de l'internaute sur le poste navigateur.<br><br>- de l'utilisateur sur le poste Windows en cours.<br><br>- sur le téléphone.  <br><br><br> |
| [NavigateurAdresseIP](../WDLang2/3055003.md) | Renvoie l'adresse IP du poste de l'Internaute connecté au site WEBDEV. |
| [NavigateurEstConnecté](../WDLang2/1000019392.md) | Indique si le navigateur est connecté au réseau. |
| [NavigateurFerme](../WDLang2/3055007.md) | Ferme la fenêtre en cours du navigateur et stoppe l'exécution du code en cours. |
| [NavigateurHauteur](../WDLang2/1000022548.md) | Renvoie la hauteur en pixels de la zone cliente dans laquelle la page est affichée. |
| [NavigateurHistoriqueAjoute](../WDLang2/1000023458.md) | Ajoute une entrée dans l'historique de navigation en lui associant des données. Ces données seront transmises lors du retour sur cette entrée. |
| [NavigateurHistoriqueModifie](../WDLang2/1000023459.md) | Modifie les données de l'entrée courante dans l'historique de navigation. Ces données seront transmises lors d'un retour sur l'entrée courante. |
| [NavigateurImprime](../WDLang2/1000024545.md) | Imprime la page courante actuellement affichée dans le navigateur. |
| [NavigateurLanceAppli](../WDLang2/1000019424.md) | Ouvre le navigateur Web par défaut de l'appareil en cours. |
| [NavigateurLargeur](../WDLang2/1000022547.md) | Renvoie la largeur en pixels de la zone cliente dans laquelle la page est affichée. |
| [NavigateurMobile](../WDLang2/1000021856.md) | Permet de savoir si le navigateur utilisé par l'internaute est lancé sur un mobile. |
| [NavigateurNom](../WDLang2/3055005.md) | Renvoie le nom du navigateur de l'internaute ("user agent"). |
| [NavigateurOS](../WDLang2/1000021857.md) | Indique le système d'exploitation annoncé par le navigateur de l'internaute. |
| [NavigateurOuvre](../WDLang2/3055006.md) | Ouvre une nouvelle fenêtre du navigateur sur une page identifiée par son URL. |
| [NavigateurPlateforme](../WDLang2/1000019486.md) | Retourne la plateforme du navigateur utilisée par l'internaute. |
| [NavigateurTranche](../WDLang2/1000022546.md) | Renvoie le numéro de la tranche affichée dans une page en mode Responsive Web Design. |
| [NavigateurType](../WDLang2/1000019487.md) | Retourne le type du navigateur utilisé par l'internaute. |
| [NavigateurVersion](../WDLang2/1000019488.md) | Retourne la version du navigateur utilisé par l'internaute. |
| [NotificationAffiche](../WDLang2/1000021021.md) | Affiche un message hors navigateur. |
| [PleinEcranActive](../WDLang2/1000021012.md) | Passe le navigateur en mode "Plein écran". Le navigateur occupe ainsi tout l'espace de l'écran. |
| [PleinEcranDésactive](../WDLang2/1000021013.md) | Désactive le mode "Plein écran" du navigateur. |






- ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=WWDialogueNavigateur.gif) ***Exemples didactiques (WEBDEV)*** : **WWDialogueNavigateur** <br>Cet exemple montre un dialogue entre deux navigateurs distincts. Il permet d'ouvrir une nouvelle page dans un nouveau navigateur et de récupérer des valeurs modifiées.
- ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Accesauxvariablesd'environnement.gif) ***Exemples unitaires (WEBDEV)*** : **Accès aux variables d'environnement** <br>Récupération des variables d'environnement de l'Internaute (adresse IP, résolution, navigateur...)<br>Cela permet de faire des statistiques de consultations de votre site ou par exemple d'afficher une version de votre site différente selon les résolutions.<br>Vous pourrez également récupérer des informations sur le serveur comme le système d'exploitation, l'adresse IP ou encore le nom du serveur.

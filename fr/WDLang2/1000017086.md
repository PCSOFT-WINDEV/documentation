


## PageTauxGFI (Fonction)

***En anglais : PageRateDDW***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de connaître et de modifier le taux de grisage des pages utilisant le GFI (Grisage des pages inaccessibles).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
PageActiveGFI(Vrai)
PageTauxGFI(40)
OuiNon("Voulez vous sortir ? ")
PageActiveGFI(Faux)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le taux de grisage

`<Résultat> = PageTauxGFI()`
---

**`<Résultat> : (Entier)`**

Taux de grisage actuel (compris entre 0 et 100). Le taux de grisage par défaut est 60%.  


<a name="SYNTAXE2"></a>

### Modifier le taux de grisage

`<Résultat> = PageTauxGFI(<Taux de grisage>)`
---

**`<Résultat> : (Entier)`**

Taux de grisage avant modification (compris entre 0 et 100).

**`<Taux de grisage> : (Entier)`**

Nouveau taux de grisage (compris entre 0 et 100). 
Si une valeur supérieure à 100 est donnée, la valeur 100 sera utilisée. 
Si une valeur inférieure à 0 est donnée, la fonction **PageTauxGFI** sera sans effet. Le taux de grisage par défaut est 60%.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Pour activer le GFI sur les pages, utilisez la fonction [PageActiveGFI](../WDLang2/1000017087.md).

- La valeur utilisée en code navigateur est initialisée par la valeur donnée en code serveur. Les modifications effectuées en code navigateur ne sont pas reportées en code serveur.


## GaleriePopupAffichée (Fonction)

***En anglais : GalleryPopupDisplayed***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Indique si la popup automatique de zoom du champ Galerie d'images est affichée.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Code de sélection du champ ZR_Galerie
// Rendre visible le libellé lors du changement d'image
SI GaleriePopupAffichée(ZR_Galerie) ALORS
		ZR_Galerie[ZR_Galerie].LIB_IMAGE.Visible = Vrai
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = GaleriePopupAffichée(<Champ Zone répétée>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la popup est affichée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 




**`<Champ Zone répétée> : (Nom de champ)`**

Nom du champ Zone répétée en mode Galerie à manipuler.

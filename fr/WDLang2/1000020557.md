
## CapsLockVérifie (Fonction)

***En anglais : CapsLockVerify***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Vérifie si la touche CapsLock est enfoncée.

**Attention** : Cette fonction doit être utilisée uniquement dans le code navigateur de touche pressée (onkeypress). La vérification sera effectuée à la première saisie d'une lettre.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affiche un avertissement si la touche CapsLock est enfoncée
SI CapsLockVérifie() = Vrai ALORS Info("La touche CapsLock est enfoncée.")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = CapsLockVérifie()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la touche CapsLock est enfoncée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.

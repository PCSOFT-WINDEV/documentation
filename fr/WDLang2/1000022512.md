


## BandeauArrêteDéfilement (Fonction)

***En anglais : BannerStopSliding***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Arrête le défilement automatique sur un champ Bandeau défilant.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Stoppe le défilement (laisse le bandeau courant)
BandeauArrêteDéfilement(BAN_Bannière)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`BandeauArrêteDéfilement(<Champ Bandeau défilant>)`
---

**`<Champ Bandeau défilant> : (Nom de champ)`**

Nom du champ Bandeau défilant à manipuler.

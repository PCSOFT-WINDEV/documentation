
## JaugeDésactive (Fonction)

***En anglais : ProgressBarDisable***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Stoppe le rafraîchissement périodique d'une jauge via Ajax.

**Attention** : Cette fonction ne peut pas être utilisée dans un site PHP ou AWP. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Stoppe le rafraîchissement de la jauge
JaugeDésactive(JAUGE_Copie)
// Et masque la jauge affichée dans une cellule
CelluleFermeDialogue(CELL_Jauge)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`JaugeDésactive(<Champ Jauge>)`
---

**`<Champ Jauge> : (Nom de champ)`**

Nom du champ Jauge à manipuler.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Si aucun rafraîchissement n'est actif, la fonction ne fait rien.

- Si le rafraîchissement a été lancé par la fonction [JaugeActive](../WDLang2/1000019477.md), la fonction **JaugeDésactive** stoppe les appels périodiques au serveur. Si un traitement serveur est actif, son exécution continue.

- Si le rafraîchissement a été lancé par la fonction [JaugeExécute](../WDLang2/1000019481.md), la fonction **JaugeDésactive** stoppe les appels périodiques à la fonction navigateur. Les derniers appels à la fonction ne sont donc pas exécutés.

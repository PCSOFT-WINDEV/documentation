
## NavigateurType (Fonction)

***En anglais : BrowserType***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Retourne le type du navigateur utilisé par l'internaute.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affiche une page particulière pour IE9 
SI NavigateurType() = NavigateurTypeInternetExplorer ALORS
	SELON NavigateurVersion(NavigateurVersionMajeure)
	CAS 9
		PageAffiche(PAGE_Fiche_IE9)
	AUTRE CAS
		PageAffiche(PAGE_Fiche_IE)
	FIN
SINON
	PageAffiche(PAGE_Fiche)
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = NavigateurType()`
---

**`<Résultat> : (Constante de type Entier)`**

Type du navigateur utilisé par l'internaute pour afficher le site : 



|   |   |
| --- | --- |
| *NavigateurTypeChrome* | Chrome. |
| *NavigateurTypeEdge* | Microsoft Edge. |
| *NavigateurTypeFirefox* | Firefox. |
| *NavigateurTypeGecko* | Navigateur utilisant le moteur de rendu Gecko (autre que Firefox). |
| *NavigateurTypeInconnu* | Le navigateur utilisé est inconnu ou les informations sont manquantes. |
| *NavigateurTypeInternetExplorer* | Internet Explorer. |
| *NavigateurTypeOpera* | Opera. |
| *NavigateurTypeRobot* | Robot de moteur de recherche. |
| *NavigateurTypeSafari* | Safari. |
| *NavigateurTypeWebKit* | Navigateur utilisant le moteur de rendu WebKit (autre que Chrome et Safari). |







<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Si le navigateur utilisé par l'internaute est inconnu, la fonction retourne la constante *NavigateurTypeInconnu*.

- Dans le cas d'un site dynamique (non AWP), la valeur retournée correspond au navigateur utilisé lors de la connexion au site. Si l'utilisateur change de navigateur (par exemple, avec un copier-coller d'URL dans un second navigateur), ce changement n'est pas détecté.

- Si votre projet utilise les sessions prélancées, cette fonction ne doit pas être utilisée dans l'événement "Initialisation" du projet. Cette fonction doit être utilisée dans l'événement "Initialisation du projet après connexion au site".

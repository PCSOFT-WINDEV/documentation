
## ServeurAppliInfoTâchePlanifiée (Fonction)

***En anglais : AppServerInfoScheduledTask***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Lit la description d'une tâche planifiée.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Regarde si la tâche planifiée est activée
soit MonServeurAppliTâchePlanifiée = ServeurAppliInfoTâchePlanifiée(MaProc)
SI MonServeurAppliTâchePlanifiée.Activé ALORS
	// Code du si
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = ServeurAppliInfoTâchePlanifiée(<Tâche planifiée>)`
---

**`<Résultat> : (Variable de type serveurAppliTâchePlanifiée)`**

Variable de type [serveurAppliTâchePlanifiée](../WDLang2/1000022367.md) contenant la description de la tâche planifiée.

**`<Tâche planifiée> : (Chaîne de caractères)`**

Correspond : 

- au nom d'une procédure : le nom de la procédure est utilisé pour retrouver le nom de la tâche.

- au nom de la tâche. 




Remarque : Pour manipuler la tâche planifiée définie en édition sur procédure, le plus simple est d'utiliser la procédure comme paramètre.

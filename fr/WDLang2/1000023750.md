


## &lt;Image&gt;.Occurrence (Fonction)

***En anglais : &lt;Image&gt;.Occurrence***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le nombre total d'images prises en compte par le défilement.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Renvoie le nombre d'images prises en compte par le défilement
Trace(IMG_Banniere.Occurrence())
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Champ Image>.Occurrence()`
---

**`<Résultat> : (Entier)`**

Nombre d'images total du défilement.

**`<Champ Image> : (Nom de champ)`**

Nom du champ à manipuler :

- Champ Image.

- Champ Image cliquable.

- Champ Vignette.

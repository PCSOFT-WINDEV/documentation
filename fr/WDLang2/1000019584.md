


## Fonctions Captcha (syntaxe préfixée)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions permettant de manipuler les champs Captcha via une syntaxe préfixée sont les suivantes : 



|   |   |
| --- | --- |
| [&lt;Captcha&gt;.Affiche](../WDLang2/1000019493.md) | Affiche un nouveau Captcha dans un champ Captcha. |
| [&lt;Captcha&gt;.Vérifie](../WDLang2/1000019567.md) | Vérifie si la valeur saisie par l'utilisateur correspond à la chaîne affichée dans un champ Captcha. |

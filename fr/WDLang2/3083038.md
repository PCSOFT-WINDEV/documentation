


## &lt;Chemin de navigation&gt;.SupprimeTout (Fonction)

***En anglais : &lt;Breadcrumb&gt;.DeleteAll***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime tous les liens dans un champ Chemin de navigation.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SMP_MonChampChemin.SupprimeTout()
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Champ Chemin de navigation>.SupprimeTout()`
---

**`<Champ Chemin de navigation> : (Nom de champ)`**

Nom du champ Chemin de navigation dans lequel les liens doivent être supprimés.


## Gestion du Markdown
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000329"></a>
Markdown est un langage de balisage "léger" qui permet de formater des textes. Markdown est assez simple mais très pratique et facile à prendre en main, à retenir et à relire.

Markdown est supporté dans tous les champs WINDEV et WINDEV Mobile. Il suffit d'activer l'option et de saisir un libellé au format Markdown.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000353"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 

### Activation du rendu Markdown dans un champ 
<a name="activation_rendu_markdown_dans_champ_ELTPARAGRAPHE000020"></a>Dans la description du champ souhaité, onglet "UI", il suffit d'activer l'option "Support du Markdown".

Cette option s'applique uniquement:

- au libellé,

- à la bulle d'aide du champ,

- aux jetons associés à un champ de saisie.
	Remarque : Cette propriété ne s'applique pas au contenu du champ de saisie.




Remarque : dans l'éditeur d'états, il est également possible de gérer le Markdown dans les champs Libellé. 


![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 

### Champ de saisie Markdown
<a name="champ_saisie_markdown_ELTPARAGRAPHE000038"></a>À partir de la version 28 Update 2, un nouveau type est disponible pour le champ de saisie : "Visualisation Markdown".

Le contenu affecté au champ sera affiché (dans l'éditeur et en exécution) avec le rendu Markdown.

Cette fonctionnalité offre plusieurs avantages (par rapport à l'affichage d'un contenu dans un champ Libellé) :

- si le contenu formaté est trop grand pour être affiché, un ascenseur sera automatiquement affiché pour scroller dans le contenu.

- il est possible de sélectionner une portion du texte affiché (pour le copier par exemple).





### Conversions Markdown
<a name="conversions_markdown_ELTPARAGRAPHE000050"></a>Le WLangage propose en standard plusieurs fonctions pour convertir un contenu Markdown en d'autres formats :



|   |   |
| --- | --- |
| [MarkdownVersDOCX](../WDLang1/1410089153.md) | Convertit une chaîne de caractères au format Markdown en un document au format DOCX. |
| [MarkdownVersHTML](../WDLang1/1410088874.md) | Convertit une chaîne de caractères au format Markdown en un buffer au format HTML (encodé en UTF8). |
| [MarkdownVersPDF](../WDLang1/1410089119.md) | Convertit une chaîne (ou un buffer) au format Markdown en un buffer contenant un document PDF. |
| [MarkdownVersTexte](../WDLang1/1410089001.md) | Convertit une chaîne de caractères au format Markdown en une chaîne de caractères au format texte (sans balises). |





Lors de l'édition de documents au format Markdown (des comptes-rendus de réunion, des rapports d'interventions, etc.), ces fonctions permettent de :

- afficher le contenu dans un site Web (HTML),

- transmettre le document par email (PDF),

- ...

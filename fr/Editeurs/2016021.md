


## Palettes de couleurs
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000259"></a>
Les palettes de couleurs sont un moyen simple de personnaliser l'aspect visuel d'un site.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Ordre d'application des palettes de couleurs
<a name="ordre_application_des_palettes_couleurs_ELTTEXTE000283"></a>
La palette de couleurs d'une page peut être définie à trois niveaux différents (par ordre décroissant de priorité) :

- Dans le projet.

- Dans le modèle de pages dont la page hérite des propriétés.

- Dans la page elle-même.




Ainsi, il faut définir la palette générale au niveau du projet. Si un ensemble de pages doit utiliser une palette particulière (par exemple les pages de la partie "Administration" d'un site), il est possible de remplacer la palette du projet en la redéfinissant au niveau d'un modèle de pages. Enfin, si une page particulière doit avoir une palette différente, la palette peut être choisie au niveau de la page elle-même.

WEBDEV permet de : 

- [Définir la palette utilisée](#NOTE3_1). 

- [Créer une nouvelle palette](#NOTE4_2).

- [Modifier une palette existante](#NOTE4_3).

- [Dupliquer une palette](#NOTE4_4).




Remarque : les couleurs des palettes sont accessibles par programmation (fonction [CouleurPalette](../WDLang2/1000019905.md)). Il est ainsi possible par exemple de créer un code générique, indépendant de l'ambiance et de la palette utilisée.

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Définir la palette de couleurs utilisée
<a name="definir_palette_couleurs_utilisee_ELTTEXTE000307"></a>
Les palettes de couleurs définissent un jeu de couleurs qui sont proposées dans tous les sélecteurs de couleurs du projet.

Ces couleurs sont également appliquées sur les styles définis dans l'ambiance du projet afin de permettre de changer rapidement et facilement l'aspect visuel d'un projet.

Lors de la création d'un projet WEBDEV, l'assistant permet de choisir la palette de couleurs utilisée. Il est possible à tout moment de modifier ce choix. 

**Pour modifier la palette de couleurs utilisée par un projet** :

1. Affichez la fenêtre de description du projet : sous le volet "Projet", dans le groupe "Projet", cliquez sur "Description".

2. Sélectionnez l'onglet "Ambiances".

3. Choisissez une palette parmi les palettes prédéfinies.

4. Validez la fenêtre de description du projet. 




**Pour modifier la palette de couleurs utilisée par un modèle de pages ou par une page** :

1. Affichez la fenêtre de description de la page : sous le volet "Page", dans le groupe "Description", cliquez sur "Description".

2. Sélectionnez l'onglet "Style".

3. Cliquez sur le bouton de parcours pour sélectionner une palette de couleurs.

4. Si la palette sélectionnée est "Comme le projet", la palette utilisée sera celle du projet.




**Remarque** : La palette de couleurs utilisée par un projet (nouvelle palette ou palette modifiée) est copiée dans le répertoire du projet. Lorsque le projet est utilisé en GDS, la palette de couleurs est également intégrée aux éléments gérés par le GDS.

## Manipuler les palettes de couleurs
<a name="manipuler_les_palettes_couleurs_ELTTEXTE000331"></a>
Pour augmenter les possibilités des palettes, WEBDEV permet de :  

- Créer une nouvelle palette. 

- Modifier une palette existante. 

- Dupliquer une palette. 

- Modifier un élément présent dans la liste des derniers éléments manipulés.




Pour manipuler les palettes : 

- utilisez l'option "Créer une nouvelle palette" présente dans l'onglet "Ambiance" de la description de projet (pour afficher la description du projet, sous le volet "Projet", dans le groupe "Projet", cliquez sur "Description"). 

- sous le volet "Page", dans le groupe "Ambiance et styles", déroulez "Palette" et cliquez sur le lien "Créer une nouvelle palette".




L'éditeur de palettes apparaît. Cet éditeur permet de : 

- [Créer une nouvelle palette](#NOTE4_2). 

- [Modifier une palette existante.](#NOTE4_3)

- [Dupliquer une palette.](#NOTE4_4)



<a name="NOTE4_2"></a>


### Générer/Créer une palette
<a name="generercreer_une_palette_ELTPARAGRAPHE000155"></a>Dans l'éditeur de palettes : 

1. Cliquez sur l'option "Générer une palette". 

2. Indiquez le nom de la palette. 

3. Sélectionnez la couleur de base de la palette à l'aide du cercle chromatique et du nuancier. 

4. Validez. 

5. Les caractéristiques des différentes couleurs de la palette sont affichées. Il est possible de : 

	- modifier séparément chaque couleur de la palette. Un aperçu est affiché en haute de l'éditeur. 

	- sauvegarder les modifications effectuées. 

	- recharger les dernières modifications enregistrées (les modifications effectuées depuis la dernière sauvegarde sont perdues). 

	- sélectionner une couleur à l'aide de la pipette.




6. Validez la fenêtre. La nouvelle palette est automatiquement ajoutée à la liste des palettes disponibles dans la description du projet.



<a name="NOTE4_3"></a>


### Modifier une palette
<a name="modifier_une_palette_ELTPARAGRAPHE000176"></a>Dans l'éditeur de palettes : 

1. Cliquez sur l'option "Modifier une palette". 

2. Sélectionnez la palette à modifier. Si cette palette est une palette fournie avec WEBDEV, cette palette est automatiquement dupliquée : indiquez alors son nouveau nom. 

3. Validez. 

4. Les caractéristiques des différentes couleurs de la palette sont affichées. Il est possible de : 

	- modifier séparément chaque couleur de la palette. Un aperçu est affiché en haut de l'éditeur. 

	- sauvegarder les modifications effectuées. 

	- recharger les dernières modifications enregistrées (les modifications effectuées depuis la dernière sauvegarde sont perdues).

	- sélectionner une couleur à l'aide de la pipette.




5. Validez la fenêtre. La palette est automatiquement modifiée.



<a name="NOTE4_4"></a>


### Dupliquer une palette
<a name="dupliquer_une_palette_ELTPARAGRAPHE000196"></a>Dans l'éditeur de palettes : 

1. Cliquez sur l'option "Dupliquer une palette". 

2. Sélectionnez la palette à dupliquer. 

3. Validez. 

4. Indiquez le nom de la nouvelle palette (créée à partir de la palette sélectionnée). 

5. Les caractéristiques des différentes couleurs de la palette sont affichées. Il est possible de : 

	- modifier séparément chaque couleur de la palette. Un aperçu est affiché en haut de l'éditeur. 

	- sauvegarder les modifications effectuées. 

	- recharger les dernières modifications enregistrées (les modifications effectuées depuis la dernière sauvegarde sont perdues). 

	- sélectionner une couleur à l'aide de la pipette.




6. Validez la fenêtre. La nouvelle palette est automatiquement ajoutée à la liste des palettes disponibles dans la description du projet.

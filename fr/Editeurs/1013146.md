


## Vérifier l'orthographe dans les champs de saisie
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000186"></a>
WINDEV offre la possibilité d'utiliser la fonctionnalité de correction orthographique dans les champs de saisie et les colonnes de type texte des champs Table des applications WINDEV.

Cette correction orthographique peut être réalisée sur le poste de l'utilisateur : 

- soit via le correcteur orthographique intégré.

- soit avec Word ou OpenOffice 2.0 ou supérieur. 




**Pour utiliser le correcteur orthographique intégré** : 

1. Configurez si nécessaire le projet pour utiliser la vérification orthographique intégrée : 

	- Affichez la fenêtre de description du projet : sous le volet "Projet", dans le groupe "Projet", cliquez sur "Description". 

	- Dans l'onglet "Options" : 

		- Sélectionnez le correcteur orthographique "Correcteur intégré". 

		- Cochez l'option "Copier automatiquement les dictionnaires dans le répertoire de génération" pour que les dictionnaires soient disponibles à côté de l'exécutable. Il est également possible d'intégrer les fichiers correspondant dans la bibliothèque de l'application.




- Validez. 

2. Chargez les différents dictionnaires nécessaires via la fonction [DictionnaireCharge](../WDLang1/1000022958.md). 
	Remarque : Les dictionnaires Français et Anglais sont automatiquement chargés. 

3. Demandez la correction orthographique sur les différents champs de l'application (champ de saisie, colonne de champ Table) :

	- soit en cochant l'option "Vérifier l'orthographe" dans l'onglet "Détail" de la fenêtre de description du champ.

	- soit en utilisant la propriété [VérifieOrthographe](../Proprietes/2510153.md) dans le code WLangage.

	- soit en utilisant la fonction [ParamètreOrthographe](../WDLang1/3025052.md) dans le code WLangage.







Pour plus de détails, consultez [Vérification d'orthographe basée sur les dictionnaires Hunspell](../WDChamp/1410087030.md). 
**Pour utiliser le correcteur orthographique OpenOffice** : 

1. Configurez si nécessaire le projet pour utiliser la vérification orthographique avec OpenOffice : 

	- Affichez la fenêtre de description du projet : sous le volet "Projet", dans le groupe "Projet", cliquez sur "Description". 

	- Dans l'onglet "Options", sélectionnez le correcteur orthographique "Correcteur d'Open Office". 

	- Validez.  




2. Installez **OpenOffice** sur le poste de l'utilisateur final. Pour plus de détails, consultez [Comment installer OpenOffice 2.0 ?](../Editeurs/1013269.md)

3. Installez les **dictionnaires de OpenOffice** sur ce même poste. Pour plus de détails, consultez [Comment installer OpenOffice 2.0 ?](../Editeurs/1013269.md)

4. Demandez la correction orthographique sur les différents champs de l'application (champ de saisie, colonne de champ Table) :

	- soit en cochant l'option "Vérifier l'orthographe" dans l'onglet "Détail" de la fenêtre de description du champ.

	- soit en utilisant la propriété [VérifieOrthographe](../Proprietes/2510153.md) dans le code WLangage.

	- soit en utilisant la fonction [ParamètreOrthographe](../WDLang1/3025052.md) dans le code WLangage.

## Fonctionnement
<a name="fonctionnement_ELTTEXTE000210"></a>


### Correction orthographique intégrée
<a name="correction_orthographique_integree_ELTPARAGRAPHE000117"></a>Si le correcteur orthographique intégré est utilisé, la correction orthographique est automatiquement réalisée dans les champs de saisie et les colonnes de table : les mots non présents dans le dictionnaire intégré sont automatiquement soulignés en rouge. Le menu contextuel du mot souligné permet soit de sélectionner une des corrections proposées, soit d'ajouter le mot dans le dictionnaire.<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=FAA_Verifier_Orthographe_int%E9gr%E9%20-%20HC%20N%B0001.gif)
Les mots ajoutés au dictionnaire sont ajoutés dans le fichier "C:\\Users\\&lt;Utilisateur&gt;\\AppData\\Roaming\\PC SOFT\\perso.dic". Ils peuvent être édités et supprimés si nécessaire. 
<a name="NOTE2_2"></a>


### Correction orthographique de type OpenOffice
<a name="correction_orthographique_type_openoffice_ELTPARAGRAPHE000127"></a>Deux correcteurs d'orthographe peuvent être proposés selon le traitement de texte installé sur le poste. 

**Si le traitement de texte utilisé est Word**, il est possible de lancer automatiquement le correcteur d'orthographe de Word sur les champs de saisie (option "Orthographe" du menu contextuel).<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=FAA_Ortho_Word.gif)
**Remarque** : Cette option est proposée uniquement si Word est installé sur le poste en cours. 

**Si le traitement utilisé est OpenOffice**, la correction orthographique est automatiquement réalisée dans les champs de saisie et les colonnes de table : les mots non présents dans le dictionnaire de OpenOffice sont automatiquement soulignés en rouge. Le menu contextuel du mot souligné permet soit de sélectionner une des corrections proposées, soit d'ajouter le mot dans le dictionnaire. <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=FAA_Ortho_Office.gif)
**Remarque** : Cette option est proposée uniquement si OpenOffice et son dictionnaire sont installés sur la machine en cours. Si nécessaire, consultez la documentation de ce logiciel pour l'installation des dictionnaires.

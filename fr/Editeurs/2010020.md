


## Options de l'éditeur d'aide
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000131"></a>
Les options de l'éditeur d'aide permettent de définir :

- les différents compilateurs d'aide utilisés (onglet "Général").

- la vérification du système d'aide lors de son ouverture (onglet "Général").

- la qualité des images générées à la compilation (onglet "Images").

- la gestion de l'insertion d'une image dans une page (onglet "Images").




Pour afficher les options de l'éditeur d'aide, sous le volet "Accueil", dans le groupe "Environnement", déroulez "Options" et sélectionnez "Options de l'éditeur d'aide".

<a name="NOTE2_1"></a>


## Options de l'onglet "Général"
<a name="options_onglet_general_ELTTEXTE000155"></a>
Les options de l'onglet "Général" sont les suivantes : <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=option_editeur_aide%20-%20HC%20N%B0001.gif&type=thumb)


- Compilateur d'aide : WINDEV est livré avec deux compilateurs d'aide :

	- **Compilateur WinHelp** : ce compilateur correspond au programme **hcrtf.exe**. Ce programme est livré dans le sous-répertoire "Programmes" du répertoire d'installation de WINDEV.

	- **Compilateur HTML Help** : ce compilateur correspond au programme **hhc.exe**. Ce programme est livré dans le sous-répertoire "Programmes" du répertoire d'installation de WINDEV.


 **Remarque** : Pour compiler et tester votre système d'aide au format HLP ou CHM, les chemins des deux compilateurs doivent être indiqués.

- Intégrité du système d'aide : En cas de problème lors de l'ouverture du système d'aide, il est possible de vérifier (et de corriger) le système d'aide dès son ouverture. Il suffit de cocher l'option "Vérifier l'intégrité du système d'aide lors du chargement".
	**Attention** : Cette opération peut être longue. Il est conseillé de l'utiliser avant d'ouvrir un système d'aide posant problème et de la décocher ensuite.

## Options de l'onglet "Images"
<a name="options_onglet_images_ELTTEXTE000179"></a>
Les options de l'onglet "Images" sont les suivantes : <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=option_editeur_aide%20-%20HC%20N%B0002.gif&type=thumb)


- Qualité des images JPEG (format CHM uniquement) : Lors de la création de votre aide sous l'éditeur d'aide, n'importe quel type d'image peut être inséré dans vos pages d'aide. Cependant, lors de la compilation au format CHM, toutes les images utilisées seront automatiquement converties au format JPEG. En effet, ce format compressé permet d'optimiser la taille de vos fichiers d'aide.
	L'éditeur d'aide de WINDEV permet de choisir la qualité des images JPEG créées à la compilation.
	Il est conseillé d'utiliser un taux de qualité compris entre 50 et 70 %.
	**Attention** : Plus les images auront une qualité élevée, plus les images seront nettes, mais plus le fichier CHM compilé sera de taille importante.

- Gestion de l'insertion d'une image dans une page d'aide : Lors de l'insertion d'une image dans une page d'aide, plusieurs options sont possibles : 

	- **Demander pour chaque image si elle doit être copiée dans le répertoire du projet**.

	- **Toujours copier les images dans le répertoire du projet** : cette option permet de conserver toutes les images utilisées par le fichier d'aide dans le répertoire du projet.

	- **Ne jamais copier les images dans le répertoire du projet** :  toutes les images utilisées par le fichier d'aide restent dans leur répertoire d'origine. Si ces images ne sont plus disponibles lors de la compilation du fichier d'aide, un carré blanc remplacera l'image dans le fichier d'aide.

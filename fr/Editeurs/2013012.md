


## Encadrer le code
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000089"></a>
Vous souhaitez inclure vos lignes de code dans une condition, une structure, ...

Rien de plus simple, il suffit d'utiliser l'option "Encadrer" de l'éditeur de code.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000113"></a>
Pour insérer plusieurs lignes de code existantes dans une condition, une répétition, ou autre :

1. Sous l'éditeur de code, sélectionnez les lignes de code voulues.

2. Sous le volet "Code" du menu, dans le groupe "Edition", déroulez "Encadrer" et choisissez le type de code devant encadrer les lignes sélectionnées :

	- Conditions de type SI .. ALORS, SI ... ALORS ... SINON, SELON, ...

	- Instructions répétitives : POUR, POUR TOUT, BOUCLE, ...

	- Type de variable : Structure, Variable composée, Chaîne multilingue.
			**Remarque** : Vous pouvez également utiliser l'option "Encadrer" du menu contextuel de l'éditeur de code.




3. Le code sélectionné est immédiatement encadré.

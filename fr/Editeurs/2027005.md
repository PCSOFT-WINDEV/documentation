


## Le volet "Navigateur" 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000089"></a>
Lorsqu'un élément WINDEV, WEBDEV ou WINDEV Mobile (analyse, requête, ...) ne peut pas être affiché entièrement dans l'espace de travail, le volet "Navigateur" permet de se déplacer aisément dans cet élément.<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=FenNavigateur.gif&type=thumb)
Le menu contextuel (clic droit) du volet "Navigateur" permet de :

- Synchroniser l'élément affiché dans le volet "Navigateur" avec l'élément en cours sous l'éditeur.

- Copier l'image affichée dans le volet "Navigateur".

- Enregistrer l'image affichée dans le volet "Navigateur" au format EMF ("Windows Enhanced Meta File"). Cette image au format vectoriel peut ensuite être utilisée dans une présentation (document Word, PowerPoint, ...).

- Visualiser le graphe de l'analyse, la modélisation souple, le graphe UML ou le graphe des requêtes.

<a name="NOTE2_1"></a>


## Afficher le volet "Navigateur"
<a name="afficher_volet_navigateur_ELTTEXTE000113"></a>
**Pour afficher le volet "Navigateur"**, sous le volet "Accueil", dans le groupe "Environnement", déroulez "Volets" et sélectionnez "Volets" puis "Navigateur".

Pour plus de détails sur les manipulations des volets, consultez [Manipuler les volets ancrables](../Editeurs/2027001.md).

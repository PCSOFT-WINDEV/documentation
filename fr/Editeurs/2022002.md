


## Grille magnétique 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000202"></a>
Les différents éditeurs de WINDEV, WEBDEV et WINDEV Mobile offrent la possibilité d'aligner les éléments manipulés grâce à une grille magnétique.

La grille magnétique facilite le positionnement et l'alignement des champs de la fenêtre, de la page, de l'état, des requêtes, de l'UML , ...

La taille de la grille est exprimée en pixels (ou en centimètres dans l'éditeur d'états).

Lorsque la grille est en fonctionnement (magnétisme activé), elle peut être temporairement désactivée en maintenant la touche Maj enfoncée lors des déplacements.

De même, lorsque la grille est débranchée (magnétisme désactivé), elle peut être temporairement activée à l'aide de la touche Maj.<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Grille.gif)
Options d'utilisation conseillées pour la grille :

- Grille invisible.

- Grille active.

- Espacement vertical de la grille : 12 pixels.

- Espacement horizontal de la grille : 12 pixels.




Si l'espacement de la grille est égale à 1, la grille ne sera pas visualisée.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment afficher la grille ?
<a name="comment_afficher_grille_ELTTEXTE000226"></a>


### Éditeur de fenêtres ou éditeur de pages
<a name="editeur_fenetres_editeur_pages_ELTPARAGRAPHE000035"></a>Pour afficher la grille sous l'éditeur de fenêtres ou l'éditeur de pages, sous le volet "Affichage", dans le groupe "Aide à l'édition", cochez l'option "Grille".

Pour paramétrer les caractéristiques de la grille (espacement, magnétisme) : 

1. Sous le volet "Affichage", cliquez sur l'icône de regroupement du groupe "Options" (![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_regroup.gif)). 

2. Dans la fenêtre qui s'affiche, cliquez sur l'onglet " Magnétisme". 


**Remarque** : Il est possible d'afficher la grille par-dessus les champs de la fenêtre en cochant l'option "Dessus" dans les options de magnétisme.
<a name="NOTE2_3"></a>


### Éditeur d'analyses
<a name="editeur_analyses_ELTPARAGRAPHE000071"></a>Pour afficher la grille sous l'éditeur d'analyses, sous le volet "Affichage", dans le groupe "Aide à l'édition", cochez l'option "Grille".
<a name="NOTE2_4"></a>


### Éditeur d'UML
<a name="editeur_uml_ELTPARAGRAPHE000090"></a>Pour afficher la grille sous l'éditeur d'UML, sous le volet "Diagramme de xxx", dans le groupe "Affichage", cochez l'option "Grille visible".

Pour utiliser le magnétisme de la grille, sous le volet "Diagramme de xxx", dans le groupe "Affichage", cochez l'option "Grille active".
<a name="NOTE2_5"></a>


### Éditeur de requêtes
<a name="editeur_requetes_ELTPARAGRAPHE000107"></a>Pour afficher la grille sous l'éditeur de requêtes, sous le volet "Affichage", dans le groupe "Aide à l'édition", cochez l'option "Grille".
<a name="NOTE2_6"></a>


### Éditeur d'états
<a name="editeur_etats_ELTPARAGRAPHE000126"></a>Pour afficher la grille sous l'éditeur d'états, sous le volet "Affichage", dans le groupe "Aide à l'édition", cochez "Grille".

Pour paramétrer les caractéristiques de la grille (espacement, magnétisme) : 

1. Sous le volet "Affichage", cliquez sur l'icône de regroupement du groupe "Options" (![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_regroup.gif)). 

2. Dans la fenêtre qui s'affiche, cliquez sur l'onglet "Magnétisme".


## Le volet "Erreurs de compilation"
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000121"></a>
Le volet "Erreurs de compilation" permet de connaître :

- [les informations, les erreurs et les warnings de compilation](../Editeurs/3084022.md).

- [les erreurs de compilation d'UI](../Editeurs/9000104.md).




Pour accéder directement au code ou au champ ayant provoqué l'erreur, double-cliquez sur l'erreur.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Afficher le volet "Erreurs de compilation"
<a name="afficher_volet_erreurs_compilation_ELTTEXTE000145"></a>
Pour afficher le volet "Erreurs de compilation", sous le volet "Accueil", dans le groupe "Environnement", déroulez "Volets" et sélectionnez "Volets" puis "Erreurs de compilation".

Pour plus de détails sur les manipulations des volets, consultez [Manipuler les volets ancrables](../Editeurs/2027001.md).

<a name="NOTE3_1"></a>


## Contenu du volet "Erreurs de compilation"
<a name="contenu_volet_erreurs_compilation_ELTTEXTE000169"></a>


### Nombre d'erreurs
<a name="nombre_erreurs_ELTPARAGRAPHE000053"></a>Lorsque le volet est affiché mais pas sélectionné, le nombre d'erreurs de compilation détectées est affiché sur le volet. 

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=VoletErreur_nb.gif)

<a name="NOTE3_2"></a>


### Mode d'affichage des erreurs
<a name="mode_affichage_des_erreurs_ELTPARAGRAPHE000064"></a>

- Le volet "Erreurs de compilation" peut afficher les erreurs sous forme de liste ou de table hiérarchique. Le bouton ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_erreur_table.gif)
 permet de modifier le mode d'affichage. 

- L'activation de l'affichage des warnings est effectuée projet par projet et non plus globalement à l'environnement. Il est donc possible par exemple de débrancher cet affichage sur les anciens projets en maintenance, mais de l'activer sur les projets en cours de développement.




## Feuille de styles 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000151"></a>
Une feuille de styles (extension ".STY") regroupe :

- les aspects (styles) des champs des fenêtres (couleur de fond, police, encadrement, ...),

- les informations relatives à l'aspect des champs des fenêtres (image de fond, dimension, ...).




A la création d'un projet, plusieurs feuilles de styles sont automatiquement associées à ce projet :

- **une feuille de styles par défaut de WINDEV**. 
	Cette feuille de styles contient plusieurs styles par défaut pour tous les types de champs. Cette feuille de styles est présente dans le répertoire "Programmes\\Données" du répertoire d'installation de WINDEV.

- **une [feuille de styles personnalisée](../Editeurs/2016015.md)**.
	A la création d'un projet, cette feuille de styles vide est automatiquement associée au projet en cours et a pour nom "&lt;NomDuProjet&gt;.STY". Cette feuille de styles est créée dans le répertoire du projet. 
	Cette feuille de styles contient : 

	- des informations de style concernant les fenêtres (image de fond, couleur de fond). 

	- des informations de style concernant les options de menu. 

	- le style des champs (et certaines propriétés des champs, comme le cadrage). 
			Cette feuille de styles est entièrement manipulable et personnalisable.




- **une feuille de styles de gabarit** si le projet est associé à un gabarit.
	Cette feuille de styles contient les styles proposés par le gabarit du projet pour tous les types de champs. 
	**Remarque** : La feuille de styles de gabarit associée au projet dépend du gabarit de projet. Si vous modifiez le gabarit du projet, la feuille de styles de gabarit associée au projet sera changée. Pour modifier le gabarit du projet, vous pouvez utiliser une des méthodes suivantes : 

	- dans l'onglet "Style" de la description du projet, dans la zone "Gabarit du projet", sélectionnez le gabarit voulu. 

	- sous le volet "Projet", dans le groupe "Autres actions", option "Appliquer un gabarit au projet". Sélectionnez le gabarit voulu. 


 Pour plus de détails sur les gabarits, consultez [Gabarit](../Editeurs/2016011.md).

## Comment le faire ?
<a name="comment_faire_ELTTEXTE000175"></a>


### Opérations possibles sur les styles
<a name="operations_possibles_sur_les_styles_ELTPARAGRAPHE000065"></a>La majorité des manipulations réalisées sur les styles sont effectuées dans la feuille de styles personnalisée. Pour réaliser ces opérations, l'éditeur de fenêtres propose une interface spécifique.

Depuis l'interface de l'éditeur de fenêtres, il est possible de :

- [changer la feuille de styles personnalisée associée au projet](../Editeurs/2016015.md).

- [changer le style associé à un champ](../Editeurs/2016006.md).

- [modifier le style d'un champ](../Editeurs/2016006.md).

- [créer, modifier ou supprimer un style dans une feuille de styles personnalisée](../Editeurs/2016015.md).




Pour une gestion avancée des feuilles de styles (personnalisation des feuilles de styles de gabarit par exemple), WINDEV permet de manipuler les fichiers ".STY" directement sous l'éditeur de fenêtres.

Depuis le fichier ".STY", il est possible de :

- [modifier ou supprimer un style dans une feuille de styles](../Editeurs/2016010.md).

- [visualiser le contenu d'une feuille de styles](../Editeurs/2016010.md).

- [importer un style dans une feuille de styles](../Editeurs/2016010.md).

- [définir le style utilisé par défaut pour un type de champ](../Editeurs/2016010.md).




Remarque : Il est également possible de comparer deux feuilles de styles. Pour plus de détails, consultez [Comparateur de projets](../Editeurs/2030024.md).

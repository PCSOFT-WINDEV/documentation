
## MVP : Exemple simple mono fichier : Table + Fiche reliée
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000098"></a>
Voici un exemple simple et illustré de l'utilisation du MVP pour réaliser une fenêtre de type Table avec sa fiche reliée. Cet exemple est mono-fichier. 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Principe
<a name="principe_ELTTEXTE000122"></a>
Contrairement au mode de programmation classique, le champ Table est lié à une variable et non à un fichier de données. 

De même, les champs de la fiche ne sont pas liés à des rubriques, mais à des variables. 

Le concept du "Binding" est utilisé. 

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=IHM-et-Binding.gif)



### La fenêtre Table
<a name="fenetre_table_ELTPARAGRAPHE000025"></a>Dans cet exemple, la fenêtre "Table" permet de visualiser une liste de contacts.

Les actions possibles sont : 

- Créer un contact. 

- Supprimer un contact. 




![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Exemple-Gestion-Contacts-01.gif)





### La fenêtre Fiche
<a name="fenetre_fiche_ELTPARAGRAPHE000038"></a>Dans cet exemple, la fenêtre "Fiche" permet de créer et modifier un contact.

Les actions possibles sont : 

- Valider la saisie. 

- Annuler la saisie. 




![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Exemple-Gestion-Contacts-02.gif)




## WDInst : Options complémentaires (résumé, licence, etc.)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000277"></a>
Lors de l'installation de l'application, il est possible de :

- [proposer une sauvegarde des éléments remplacés par l'installation](#NOTE2_1). 

- [afficher le résumé des options choisies pour l'installation](#NOTE3_1).

- [afficher l'étape proposant l'installation directe de l'application](#NOTE4_1).

- [réaliser une installation silencieuse](#NOTE4_1_Silencieux). 

- [afficher une licence d'utilisation](#NOTE5_1).

- [afficher un fichier "Lisez-Moi"](#NOTE6_1).




Par défaut : 

- le résumé des options choisies pour l'installation sera affiché avant de commencer l'installation.

- l'assistant propose l'installation directe de l'application. 




Ces paramètres sont configurés dans l'éditeur d'installation WDInst. Pour lancer WDInst, sous le volet "Outils", dans le groupe "Utilitaires", cliquez sur "WDInst".

## Afficher le résumé des options choisies pour l'installation
<a name="afficher_resume_des_options_choisies_pour_installation_ELTTEXTE000331"></a>


### Présentation
<a name="presentation_ELTPARAGRAPHE000085"></a>Avant de commencer l'installation, vous pouvez afficher un résumé des options sélectionnées par l'utilisateur. Ce résumé contient les informations suivantes :

- le répertoire d'installation de l'application.

- le type d'installation choisi (complète, minimale ou personnalisée).

- la liste des composants installés (dans le cas d'une installation personnalisée).

- le détail des modifications effectuées dans les fichiers ".INI" et dans le fichier "AutoExec.BAT" (si des modifications sont effectuées).

- le détail des modifications apportées sur les clés de la base de registre (si des modifications sont effectuées).





![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Info_affiches_install%20-%20HC%20N%B0004.gif&type=thumb)

<a name="NOTE3_2"></a>


### Afficher un résumé des options choisies pour l'installation
<a name="afficher_resume_des_options_choisies_pour_installation_ELTPARAGRAPHE000100"></a>Pour afficher un résumé des options choisies pour l'installation :

1. Sélectionnez l'option "Paramètres de l'installation .. Assistant".

2. Cochez l'option "Afficher la page de résumé avant de commencer l'installation".

## Proposer ou non une installation rapide
<a name="proposer_non_une_installation_rapide_ELTTEXTE000361"></a>


### Présentation
<a name="presentation_ELTPARAGRAPHE000117"></a>Le programme d'installation peut proposer à l'utilisateur : 

- une installation rapide (également appelée "installation directe") : Si l'utilisateur choisi cette option, l'installation de l'application est automatiquement effectuée avec les options par défaut. 

- une installation personnalisée : L'utilisateur peut paramétrer les différentes options de l'installation. 



![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Info_affiches_install%20-%20HC%20N%B0002.gif&type=thumb)


Remarque : par défaut, l'installation rapide est proposée. 
<a name="NOTE4_2"></a>


### Proposer ou non une installation rapide
<a name="proposer_non_une_installation_rapide_ELTPARAGRAPHE000131"></a>Pour configurer l'installation rapide :

1. Sélectionnez l'option "Paramètres de l'installation .. Assistant".

2. Cochez l'option "Afficher la page proposant l'installation directe en un clic".




Remarque : Si cette option est décochée, l'assistant n'affichera pas l'étape proposant l'installation en un clic.

## Réaliser une installation silencieuse
<a name="realiser_une_installation_silencieuse_ELTTEXTE000391"></a>


### Présentation
<a name="presentation_ELTPARAGRAPHE000149"></a>Les installations de la version de référence et de la version cliente peuvent être réalisées de façon silencieuse. Dans ce cas, l'installation se fera sans demander aucune intervention de la part de l'utilisateur. Les options définies par défaut seront automatiquement utilisées.
<a name="NOTE4_2_Silencieux"></a>


### Proposer ou non une installation silencieuse
<a name="proposer_non_une_installation_silencieuse_ELTPARAGRAPHE000156"></a>Pour configurer l'installation silencieuse :

1. Sélectionnez l'option "Paramètres de l'installation .. Assistant".

2. Cochez l'option : 

	- **Installation cliente silencieuse** : l'installation de l'application pour l'utilisateur final ne nécessitera aucune action de la part de l'utilisateur. 

	- **Déploiement de la version de référence silencieux** (disponible uniquement lors d'une installation réseau avec mise à jour automatique) : l'installation de la version de référence de l'installation réseau sera effectuée sans intervention de l'utilisateur. 







Remarque : Si ces options sont décochées, l'assistant affichera toute les options nécessaires à l'installation de l'application.

## Afficher une licence d'utilisation
<a name="afficher_une_licence_utilisation_ELTTEXTE000421"></a>


### Présentation
<a name="presentation_ELTPARAGRAPHE000180"></a>L'installation de votre application peut commencer par la validation d'une licence d'utilisation. L'utilisateur devra accepter les termes de cette licence pour poursuivre l'installation.

Le fichier licence sera affiché par l'assistant d'installation. Il peut être :

- soit au format texte (".TXT").

- soit au format RTF (".RTF" ou ".WRI" par exemple).



<a name="NOTE5_2"></a>


### Afficher une licence d'utilisation au début de l'installation
<a name="afficher_une_licence_utilisation_debut_installation_ELTPARAGRAPHE000192"></a>Pour afficher une licence d'utilisation au début de l'installation : 

1. Sélectionnez l'option "Paramètres de l'installation .. Assistant".

2. Sélectionnez un fichier licence pour chaque langue gérée par le programme d'installation (bouton "Sélectionner"). 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Info_affiches_install%20-%20HC%20N%B0001%201.gif&type=thumb)

## Afficher un fichier "Lisez-Moi"
<a name="afficher_fichier_lisezmoi_ELTTEXTE000451"></a>


### Présentation
<a name="presentation_ELTPARAGRAPHE000205"></a>Vous pouvez proposer à l'utilisateur de lire un fichier "Lisez-Moi" une fois l'installation terminée. Ce fichier pourra par exemple donner des indications sur les limites du logiciel installé, aider l'utilisateur à utiliser ce logiciel, etc.

A la fin de l'installation, l'utilisateur devra cocher l'option "Afficher le fichier "Lisez-Moi"" pour lire ce fichier.

Ce fichier peut être :

- soit au format texte (".TXT"),

- soit au format Word (".DOC"),

- soit au format HTML (".HTM" ou ".HTML"),

- soit au format RTF (".RTF" ou ".WRI" par exemple),

- soit dans un format quelconque si celui-ci est reconnu sur le poste d'installation.





<a name="NOTE6_2"></a>


### Afficher un fichier "Lisez-Moi"
<a name="afficher_fichier_lisezmoi_ELTPARAGRAPHE000223"></a>Pour afficher un fichier "Lisez-Moi" : 

1. Sélectionnez l'option "Paramètres de l'installation .. Fin de l'installation".

2. Sélectionnez un fichier "Lisez-Moi" pour chaque langue gérée par le programme d'installation (bouton "Sélectionner"). 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Options%20install_WDInst%20-%20HC%20N%B0005%201.gif&type=thumb)

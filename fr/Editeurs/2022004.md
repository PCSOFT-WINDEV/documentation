


## Options d'alignement
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000241"></a>
Pour optimiser l'alignement des champs dans une fenêtre, une page ou un état, il est possible d'utiliser les options d'alignement. Ces options permettent d'aligner plusieurs champs présents dans la fenêtre ou l'état.

Il est possible d'aligner les champs grâce au volet "Alignement" du ruban de l'éditeur. 

Les options sont regroupées selon plusieurs thèmes : 

- Position : <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Alignement_Position.gif)


- Alignement interne et externe : <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Alignement_Int_Ext.gif)


- Taille : <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Alignement_Taille.gif)


- Centrage et répartition : <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Alignement_Centrage.gif)


- Autres : <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Alignement_Autres.gif)

## Comment le faire ?
<a name="comment_faire_ELTTEXTE000265"></a>


### Aligner plusieurs champs d'une fenêtre, d'une page ou d'un état
<a name="aligner_plusieurs_champs_une_fenetre_une_page_etat_ELTPARAGRAPHE000033"></a>Pour aligner plusieurs champs d'une fenêtre, d'une page ou d'un état : 

1. Sélectionnez les champs à aligner.

2. Effectuez :

	- Un alignement prédéfini : sous le volet "Alignement", sélectionnez les caractéristiques d'alignement voulues dans les groupes "Position", "Interne et externe", "Taille" et "Centrage et répartition".

	- Un alignement personnalisé : sous le volet "Alignement", dans le groupe "Autres alignements", cliquez sur "Alignement personnalisé". L'alignement personnalisé permet d'utiliser des propriétés spécifiques d'alignement. Ce mode est disponible uniquement pour les champs d'une fenêtre ou d'une page.







**Remarque** : **Visualisation de l'alignement avant application**
Si plusieurs champs sont sélectionnés, le survol d'une icône d'alignement permet de visualiser en temps réel le résultat du clic sur l'icône. Il est ainsi facile de visualiser l'effet des différentes options proposées. 
Un clic sur l'icône applique l'alignement sélectionné.
La sortie du survol laisse la fenêtre, la page ou l'état dans son état initial.

## Les différentes options détaillées
<a name="les_differentes_options_detaillees_ELTTEXTE000289"></a>


### Position
<a name="position_ELTPARAGRAPHE000074"></a>Les options du groupe "Position" permettent de gérer les positions des champs les uns par rapport aux autres. 
L'option **"Échanger les positions"** permet d'échanger la position de 2 champs ou d'échanger cycliquement la position de 3 champs ou plus. 

- **Exemple 1** : Echange de deux champs : Il suffit de sélectionner 2 champs : leurs positions sont échangées. <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Echange_Champ.gif)


- **Exemple 2** : Echange cyclique de position : Pour positionner le champ "Serveur SMTP" après le champ "Adresse email", il suffit de sélectionner en premier le champ "Serveur SMTP" puis "Mot de passe" et "Confirmation de mot de passe". <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Echange_Champ_2.gif&type=thumb)





Les options d'alignement des positions permettent d'aligner plusieurs champs entre eux (le champ utilisé pour base de l'alignement étant le premier sélectionné). Il est possible de : 

- aligner les champs selon le bord extérieur gauche ou droit. 

- aligner les champs selon le bord haut ou selon le bord bas. 

- aligner les centres des champs horizontalement ou verticalement. 


Des options de positionnement par rapport à la fenêtre permettent de positionner plus vite un champ ou un groupe de champs. Il est ainsi possible de : 

- Positionner les champs le plus à gauche puis le plus haut possible : Le ou les champs sont déplacés le plus à gauche possible (dans la limite des champs existants) puis ils sont déplacés le plus haut possible (dans la limite des champs existants). 

- Positionner les champs le plus à gauche possible : Le ou les champs sont déplacés le plus à gauche possible (dans la limite des champs existants). 

- Positionner les champs le plus haut possible : Le ou les champs sont déplacés le plus haut possible (dans la limite des champs existants). 


De la même manière, il est possible de : 

- Positionner les champs le plus bas possible. 

- Positionner les champs le plus à droite possible. 

- Positionner les champs le plus à droite puis le plus bas possible. 




Remarques : 

- Lors du positionnement, les ancrages sont pris en compte. 

- Si le champ est présent dans un champ Parent (superchamp par exemple), le positionnement est effectué par rapport au parent. 



<a name="NOTE3_2"></a>


### Interne/Externe
<a name="interneexterne_ELTPARAGRAPHE000119"></a>Les options du groupe "Interne et externe" permettent de gérer les bords internes et externes des champs. Ces options sont très utiles pour réaliser un alignement de champs avec libellés : champs de saisie, combo, ...
<a name="NOTE3_3"></a>


### Taille
<a name="taille_ELTPARAGRAPHE000126"></a>Les options du groupe "Taille" permettent : 

- d'uniformiser la taille des champs (la taille de référence est celle du premier champ sélectionné) : même largeur, même hauteur. 

- d'étendre les champs dans la fenêtre en fonctions des champs existants. Les options "Etendre en largeur", "Etendre en hauteur" et "Etendre en largeur et en hauteur" calculent la taille maximale que le champ peut occuper dans l'interface par rapport aux autres champs. Le redimensionnement est immédiat. 
	Par exemple, un champ Carte positionné dans une fenêtre peut facilement occuper tout l'espace disponible en tenant compte des champs déjà présents dans la fenêtre.
	Remarques : 

	- les ancrages sont pris en compte. 

	- la taille du champ parent est prise en compte. 






<a name="NOTE3_4"></a>


### Centrage et répartition
<a name="centrage_repartition_ELTPARAGRAPHE000145"></a>Les options du groupe "Centrage et répartition" permettent : 

- centrer l'élément sélectionné dans le parent (horizontalement et verticalement). 

- définir l'espacement vertical et horizontal entre les champs. 



<a name="NOTE2_3"></a>


### Autres alignements
<a name="autres_alignements_ELTPARAGRAPHE000155"></a>Les différentes options sont : 

- Alignement personnalisé. 

- Correcteur d'interface. 
	Pour plus de détails, consultez [Correcteur d'interface](../Editeurs/2022005.md). 

- Agrandir et aérer. 
	Pour plus de détails, consultez : 

	- [Aérer une sélection de champs](../Editeurs/2022007.md),

	- [Agrandir une fenêtre](../Editeurs/2022008.md). 







**Alignement personnalisé**
Lors de l'alignement personnalisé, il est possible de définir :

- L'alignement horizontal ou vertical : les champs peuvent être alignés à gauche ou à droite en fonction du début du champ ou en fonction du début de la zone de saisie. Ils peuvent aussi être alignés par le centre.

- L'espacement uniforme : l'espacement entre les champs est adapté pour être le même entre chaque champ (horizontalement ou verticalement).

- La taille : la taille des champs sélectionnés est automatiquement adaptée pour obtenir des champs de même hauteur et/ou de même largeur.

- Le champ de référence : pour l'alignement et la taille des champs, le champ de référence est au choix :

	- le premier champ sélectionné (celui qui a les poignées blanches),

	- le dernier champ sélectionné,

	- le plus grand champ sélectionné,

	- le champ situé le plus en haut à gauche des champs sélectionnés.

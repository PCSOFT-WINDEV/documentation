<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000097"></a>
WEBDEV offre la possibilité de gérer l'ensemble des styles WEBDEV utilisés dans un projet. Les styles WEBDEV disponibles sont visualisables par type de champ.  

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Editer la feuille de styles WEBDEV d'un projet
<a name="editer_feuille_styles_webdev_projet_ELTTEXTE000121"></a>
Pour éditer les styles WEBDEV d'un projet :

1. Sélectionnez un champ du type pour lequel vous souhaitez connaître les styles WEBDEV disponibles (par exemple un champ Bouton pour voir les différents styles WEBDEV associés au champ Bouton). 

2. Sous le volet "Modification", dans le groupe "Style", cliquez sur "Styles".

3. La liste des styles WEBDEV présents dans le projet et pouvant être associés au champ sélectionné (champ Bouton dans notre exemple) s'affiche.




Pour un type de champ donné, s'il existe un ou plusieurs styles WEBDEV dans le projet, il n'est pas possible de définir un style CSS par défaut pour ce type de champ. C'est le premier style WEBDEV de la liste qui correspondra au style par défaut pour ce type de champ. Le style par défaut n'est pas modifiable.

**Remarque** : La case à cocher "Local" permet d'obtenir la liste des styles associés à la feuille de styles associée à la page (dans l'onglet "Style" de la description de la page).

Cette fenêtre permet de :

- Modifier le style sélectionné. Si le style n'est pas un style local, cette modification sera reportée sur tous les champs utilisant ce style.

- Créer un nouveau style.

- Dupliquer le style sélectionné.

- Supprimer le style sélectionné : dans ce cas, le style associé au champ sera le même mais sera local au champ.




La fenêtre d'édition du style reprend les caractéristiques de style correspondant au champ pour lequel le style a été défini. Pour plus de détails, consultez [Fenêtre d'édition des styles WEBDEV](../Editeurs/2016002.md).




## Environnement de l'éditeur d'états 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=EtatEditeur.gif&type=thumb)
Les éléments importants de l'éditeur d'états sont les suivants :


|   |   |
| --- | --- |
| ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CERCLE1.gif)<br> | **Ruban**<br>Permet un accès rapide aux principales fonctionnalités de l'éditeur d'états. Ce ruban est composé de volets, de groupes et d'options. Pour plus de détails, consultez [Un menu sous forme de ruban](../Editeurs/9000149.md). |
| ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Cercle2.gif)<br> | [**Règles d'alignement**](../Editeurs/2022001.md)<br>Permettent d'aligner les champs dans l'état. Pour afficher les règles, utilisez le raccourci Ctrl + R. |
| ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CERCLE3.gif)<br> | **Etat en cours d'édition**<br>Des ascenseurs permettent de faire défiler les états volumineux. Il est également possible de zoomer sur l'état pour visualiser plus précisément les éléments de l'état (Ctrl + roulette de souris). |
| ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CERCLE4.gif)<br> | [**Volets verticaux**](../Editeurs/2027001.md)<br>Affiche :<br><br>- les éléments (fichiers, variables, états, champs, ...) disponibles dans le projet en cours ([volet "Explorateur de projet"](../Editeurs/2027029.md)).<br><br>- la description des éléments de l'état via le [modifieur](../Editeurs/2027006.md).<br><br><br>\   \   \   \<br>\ --- \ --- \ --- \<br>\ ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ER.png)<br><br> \ Important \ Dans le logiciel Etats & Requêtes, seuls les volets ["Explorateur de projet"](../Editeurs/2027029.md) et ["Modifieur"](../Editeurs/2027006.md) sont disponibles. \<br><br><br><br> |
| ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Cercle5.gif)<br> | **Fenêtre de description**<br>Permet de visualiser et/ou de modifier :<br><br>- les caractéristiques du ou des champs sélectionnés.<br><br>- les caractéristiques de l'état en cours.<br><br><br> |
| ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CERCLE6.gif)<br> | **Fenêtre de code**<br>Affiche les différents traitements associés à l'état, à ses blocs et à ses champs. |
| ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CERCLE7.gif)<br> | [**Documents ouverts**](../Editeurs/2029012.md)<br>Affiche un bouton pour chaque document ouvert sous l'éditeur. Pour afficher l'élément voulu au premier plan, il suffit de cliquer sur le "bouton" correspondant. |
| ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CERCLE8.gif)<br> | [**Volets horizontaux**](../Editeurs/2027001.md)<br>Affiche :<br><br>- les erreurs de compilation (erreurs présentes dans le code WLangage) ou erreurs d'IHM ([volet "Erreurs de compilation"](../Editeurs/2027031.md)),<br><br>- l'outil de recherche ([volet "Rechercher-Remplacer"](../Editeurs/2027013.md)),<br><br>- le débogueur lors du test d'un élément ([volet "Débogueur"](../Editeurs/2027030.md)),<br><br>- ...<br><br><br> |

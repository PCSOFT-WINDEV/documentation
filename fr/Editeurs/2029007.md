
## Environnement de l'éditeur de code 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=EnvEdCode.gif&type=thumb)


Les éléments importants de l'éditeur de code sont les suivants :


|   |   |
| --- | --- |
| ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CERCLE1.gif)<br> | **Ruban**<br>Permet un accès rapide aux principales fonctionnalités de l'éditeur de code. |
| ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CERCLE2.gif)<br> | **Volet du ruban spécifique à l'éditeur de code**<br>Permet une sélection rapide des codes à visualiser. |
| ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CERCLE3.gif)<br> | **Fenêtres de codes**<br>Plusieurs fenêtres de codes peuvent être ouvertes en même temps. |
| ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CERCLE4.gif)<br> | **Point d'arrêt pour le débogueur**<br>L'icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=IconeStop.gif)<br> signale un point d'arrêt. Lors du test du projet, le débogueur se lancera automatiquement lors de l'exécution de la ligne précédée de cette icône. Pour mettre ou enlever directement un point d'arrêt, il suffit de cliquer devant la ligne de code. |
| ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Cercle5.gif)<br> | **Erreur**<br>Un texte souligné en rouge signale une erreur rencontrée dans le code. La signification de l'erreur peut être connue :<br><br>- soit en survolant la ligne de code : une bulle d'aide apparaît contenant le texte de l'erreur.<br><br>- soit dans le volet "Erreurs de compilation".<br><br><br> |
| ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CERCLE6.gif)<br> | **Saisie assistée**<br>Liste déroulante proposant les différentes valeurs possibles pour chaque paramètre de la fonction en cours de saisie.<br><br>Pour plus de détails, consultez [Complétion automatique](../Editeurs/2013001.md).<br><br>L'option &lt;Assistant de la fonction xxx &gt; signale la présence d'un assistant permettant de saisir les différents paramètres de la fonction à l'aide de questions simples. Pour lancer l'assistant, sélectionnez l'option &lt;Assistant de la fonction xxx &gt;. |
| ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CERCLE7.gif)<br> | **Ensemble de volets**<br>Les volets "spécifiques" à l'éditeur de code sont les suivants :<br><br>- Volet "Erreurs de compilation" affichant le détail des erreurs,<br><br>- Volet "Visualisation contextuelle du code" permettant de visualiser le code de l'élément en cours sous l'éditeur.<br><br>- Volet "Débogueur" affichant les options du débogueur lorsque celui-ci est lancé.<br><br><br> |
| ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CERCLE8.gif)<br> | **Ascenseur visuel de code**<br>La partie droite (ou gauche) de l'éditeur de code affiche le code en réduction.<br>La fenêtre de visualisation active est matérialisée. Le déplacement de la petite fenêtre de visualisation déplace également le contenu de l'éditeur de code. |

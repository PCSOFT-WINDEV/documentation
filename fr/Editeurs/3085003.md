
## Personnaliser les raccourcis clavier
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000144"></a>
Les différents éditeurs de WINDEV, WEBDEV et WINDEV Mobile proposent de nombreuses options directement manipulables grâce aux raccourcis clavier. 

Le plus souvent, les raccourcis clavier correspondent à des raccourcis standard, disponibles dans de nombreux logiciels, par exemple :

- Ctrl + S pour enregistrer un document,

- Ctrl + A pour sélectionner le contenu d'un document,

- Ctrl + F pour faire une recherche, ...


WINDEV, WEBDEV et WINDEV Mobile proposent également de nombreux raccourcis correspondant à des actions spécifiques à l'éditeur manipulé. 

Pour simplifier la manipulation des éditeurs et améliorer la productivité, il est possible de redéfinir la grande majorité des raccourcis par défaut en modifiant la combinaison de touches associée à une action dans l'éditeur. Il est également possible d'associer une combinaison de touches à une action n'en n'ayant pas. 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000168"></a>


### Afficher la fenêtre de gestion des raccourcis clavier
<a name="afficher_fenetre_gestion_des_raccourcis_clavier_ELTPARAGRAPHE000026"></a>Pour afficher la fenêtre de gestion des raccourcis clavier : 

1. Sous le volet "Accueil", dans le groupe "Environnement", déroulez "Options" et sélectionnez "Personnaliser les raccourcis clavier". 

2. La fenêtre de gestion des raccourcis clavier s'affiche. Cette fenêtre affiche les différentes options du produit, regroupées par thème. 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Personnaliser_raccourcis%20-%20HC%20N%B0002.gif&type=thumb)





Remarques : 

- Il est possible de rechercher une commande ou un raccourci dans cette fenêtre.

- Si un raccourci clavier est associé à la commande, il est affiché. 





### Associer une combinaison de touches à une commande
<a name="associer_une_combinaison_touches_une_commande_ELTPARAGRAPHE000052"></a>Dans la fenêtre de gestion des raccourcis clavier : 

1. Cliquez sur l'icône "+". 

2. Une fenêtre popup apparaît : ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Personnaliser_raccourcis%20-%20HC%20N%B0003.gif)


3. Vous pouvez : 

	- soit presser une combinaison de touches : cette combinaison de touches sera détectée et associée à la commande en cours. 

	- soit annuler la détection. 




4. La combinaison de touches est automatiquement associée à la commande. 




Remarques : 

- Si une combinaison de touches existe déjà pour la commande, la nouvelle combinaison de touches est également associée à la commande. 

- Si la combinaison de touches choisie existe déjà pour une autre commande, un message et affiché et un picto spécifique apparaît à coté de chaque raccourci identique. ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Personnaliser_raccourcis%20-%20HC%20N%B0004.gif)
Ce raccourci n'est pas mémorisé. Pour forcer l'affectation du raccourci à la commande, il est nécessaire de supprimer l'association existante. 





### Modifier ou supprimer la combinaison de touches associée à une commande
<a name="modifier_supprimer_combinaison_touches_associee_une_commande_ELTPARAGRAPHE000073"></a>Dans la fenêtre de gestion des raccourcis clavier : 

1. Cliquez sur le raccourci associé à une commande. 

2. Une fenêtre popup apparaît : ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Personnaliser_raccourcis%20-%20HC%20N%B0005.gif)


3. Vous pouvez : 

	- détecter la prochaine touche : la prochaine combinaison de touches sera détectée et associée à la commande en cours. 

	- utiliser la combinaison par défaut (si par défaut la commande est associée à une combinaison).

	- supprimer le raccourci existant.

	- annuler l'opération. 








### Autres actions possibles
<a name="autres_actions_possibles_ELTPARAGRAPHE000088"></a>Le menu contextuel des raccourcis (affichable par l'icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Personnaliser_raccourcis%20-%20HC%20N%B0006.gif)
 ou par un clic droit sur une commande) permet de : ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Personnaliser_raccourcis%20-%20HC%20N%B0007.gif)


- choisir le mode d'affichage des raccourcis clavier. Il est possible d'afficher tout, uniquement les commandes avec raccourci, uniquement les raccourcis personnalisé. 

- importer / exporter le fichier de configuration des raccourcis clavier (sous forme de fichier ".ini"). 

- réinitialiser la liste des raccourcis, afin de revenir à la liste originale des raccourcis clavier.

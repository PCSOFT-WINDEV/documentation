


## Ordre de navigation dans une fenêtre, une page, ...
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000279"></a>
L'ordre de navigation des champs d'une fenêtre ou d'une page décrit l'ordre de prise de focus des champs lorsque l'utilisateur utilise la touche [Tabulation]. Par défaut, l'ordre de navigation correspond à l'ordre de création des champs.

L'ordre de navigation est pris en compte uniquement pour les champs gérés par la touche [Tabulation]. Le choix de la gestion du champ par la touche [Tabulation] est effectué :

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) dans l'onglet "UI" de la fenêtre de description de chaque champ d'une fenêtre.




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000303"></a>


### Connaître l'ordre de navigation
<a name="connaitre_ordre_navigation_ELTPARAGRAPHE000045"></a>Pour connaître l'ordre de navigation :

- soit utilisez la touche F5,

- soit sous le volet "Affichage", dans le groupe "Voir (tous les champs)", cochez "Ordre de navigation".


Un numéro apparaît sur tous les champs accessibles par la touche [Tabulation] dans la fenêtre / la page.

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Ordre%20navigation%202.gif)
Remarque : Si le champ appartient à un conteneur (cellule, zone répétée, ...), le numéro est de la forme : 

```txt
<Numéro tabulation conteneur>.<Numéro tabulation dans le conteneur>
```

<a name="NOTE2_2"></a>


### Modifier l'ordre de navigation
<a name="modifier_ordre_navigation_ELTPARAGRAPHE000079"></a>Pour modifier l'ordre de navigation dans la fenêtre ou la page en cours :

1. Sélectionnez :

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) sous WINDEV ou WINDEV Mobile : sous le volet "Fenêtre", dans le groupe "Ordre", déroulez "Navigation" et sélectionnez "Éditer".




2. La liste des champs de la fenêtre/page apparaît, classée selon l'ordre de navigation actuelle.

3. Utilisez les flèches situées sur la droite pour organiser la liste des champs selon l'ordre de navigation voulu.![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Ordre%20navigation.gif)





**Remarques** : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Si l'option "Définir automatiquement l'ordre de navigation à chaque modification de la fenêtre" est cochée, l'ordre de navigation des champs sera automatiquement recalculé : 

	- lors de l'ajout d'un champ dans la fenêtre, 

	- lors du déplacement d'un champ dans la fenêtre. 


L'ordre de tabulation est toujours correct quelle que soit la modification effectuée dans la fenêtre.

## Notes
<a name="notes_ELTTEXTE000333"></a>


### Définir l'ordre de navigation automatiquement
<a name="definir_ordre_navigation_automatiquement_ELTPARAGRAPHE000148"></a>Pour définir l'ordre de navigation automatiquement, en fonction de la position des champs dans la fenêtre ou la page, utilisez :

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) pour les fenêtres : sous le volet "Fenêtre", dans le groupe "Ordre", déroulez "Navigation" et sélectionnez "Définir en automatique".



<a name="NOTE3_2"></a>


### Définir l'ordre de navigation en fonction de la sélection
<a name="definir_ordre_navigation_fonction_selection_ELTPARAGRAPHE000186"></a>Il est possible de définir l'ordre de navigation en fonction de l'ordre de sélection des champs dans la fenêtre ou dans la page :

1. Sélectionnez les champs dans la fenêtre selon l'ordre voulu.

2. Sélectionnez :

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) pour les fenêtres : sous le volet "Fenêtre", dans le groupe "Ordre", déroulez "Navigation" et sélectionnez "Définir par la sélection".






<a name="NOTE3_3"></a>


### Connaître par programmation le numéro de navigation associé à un champ
<a name="connaitre_par_programmation_numero_navigation_associe_champ_ELTPARAGRAPHE000227"></a>Pour connaître et modifier la position d'un champ dans l'ordre de navigation par programmation, utilisez la propriété [NumTAB](../Proprietes/2510098.md).

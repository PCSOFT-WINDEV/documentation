


## Le volet "Catalogue d'images"
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000113"></a>
Le volet "Catalogue d'images" est un catalogue d'images contextuel. Ce volet permet d'accéder plus rapidement aux images pour les associer aux champs de vos interfaces (UI).

Lors de l'édition d'un champ nécessitant une image (champ Image, champ Bouton, ...), le catalogue d'images contextuel recherche automatiquement les images pouvant être utilisées (par exemple en se basant sur le libellé du champ).<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=voletCatalogueimage.gif)


<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Afficher le volet "Catalogue d'images"
<a name="afficher_volet_catalogue_images_ELTTEXTE000137"></a>
**Pour afficher le volet "Catalogue d'images"**, sous le volet "Accueil", dans le groupe "Environnement", déroulez "Volets" et sélectionnez "Volets" puis "Catalogue d'images".

Pour plus de détails sur les manipulations des volets, consultez [Manipuler les volets ancrables](../Editeurs/2027001.md).

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Utilisation du volet "Catalogue d'images"
<a name="utilisation_volet_catalogue_images_ELTTEXTE000161"></a>


### Recherche d'images
<a name="recherche_images_ELTPARAGRAPHE000045"></a>La recherche d'images peut s'effectuer de différentes façons :

- Directement dans le volet en saisissant un mot-clé : ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=VoletCatalogueimage%203.gif). 
	Un thème peut être sélectionné via le bouton ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=VoletCatalogueimage%201.gif). La recherche est automatiquement réalisée lors de l'appui sur la touche Entrée.

- Lors de la sélection d'un champ, le mot-clé utilisé pour la recherche correspond au libellé du champ.



<a name="NOTE3_2"></a>


### Affectation d'une image à un champ
<a name="affectation_une_image_champ_ELTPARAGRAPHE000061"></a>L'affectation d'une image présente dans le catalogue d'images à un champ peut s'effectuer de différentes façons :

- Lors d'un Drag and Drop de l'image du catalogue vers la fenêtre, la page ou l'état : un nouveau champ Image est automatiquement créé avec l'image sélectionnée.

- Lors d'un Drag and Drop de l'image du catalogue vers un champ de la fenêtre ou de la page : les caractéristiques du champ sont automatiquement modifiées.

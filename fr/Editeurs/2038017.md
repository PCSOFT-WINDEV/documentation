
## GDS : Optimisation de la gestion des heures
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000066"></a>
A partir de la version 110042, les dates et les heures sont stockées dans le Gestionnaire de Sources au format UTC (Temps Universel Coordonné) au lieu du temps local du poste en cours.

Conséquences : 

- Les heures lues à partir de la version 110042 dans une base de sources et écrites avec une version inférieure seront décalées de 1 ou 2 heures.

- Des pertes de performances peuvent être observées si des développeurs utilisent une base de sources du GDS simultanément avec la version 110042 et une version inférieure. Il est conseillé que tous les développeurs utilisent la même version du produit.

- L'ouverture des projets présents dans le GDS est plus rapide.

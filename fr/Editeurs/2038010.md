### Enlever un projet du gestionnaire de sources
<a name="enlever_projet_gestionnaire_sources_ELTPARAGRAPHE000011"></a>Pour enlever un projet du gestionnaire de sources :

1. Ouvrez le projet.

2. Affichez la description du projet : sous le volet "Projet", dans le groupe "Projet", cliquez sur "Description".

3. Dans l'onglet "GDS", sélectionnez l'option "Pas de groupware développeur ni de GDS" (ou "Travailler à plusieurs en utilisant le groupware développeur" selon vos besoins).

4. Validez.




**Remarque** : Si vous cochez à nouveau l'option "Utiliser le GDS (seul ou à plusieurs)", le bouton "Paramètres" propose différentes options : <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=GDS_Enlever%20-%20HC%20N%B0001.gif)


- Ajouter le projet dans une base de sources : Ouvre l'assistant d'import d'un projet dans une base de sources

- Reconnecter le projet à sa base de sources : Reconnecte le projet à la base de sources lui correspondant.




## Fenêtre iPad : Assistant de création de fenêtres
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000216"></a>
L'assistant de création de fenêtres iPad permet de créer en quelques clics une fenêtre contenant tous les éléments de base d'une fenêtre pour une application iPad.

Ces différents éléments peuvent être ajoutés à tout moment dans la fenêtre. 

Il est également possible de configurer différentes options dans la fenêtre de description de la fenêtre. 

A la fin de l'assistant, la fenêtre est créée sous l'éditeur avec les caractéristiques sélectionnées. Sous l'éditeur, en plus de la fenêtre, il est possible de visualiser : 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Fen%EAtre_iphone_Editeur%20-%20HC%20N%B0002.gif&type=thumb)


- la barre système,

- les zones correspondant au mode portrait ou paysage (matérialisées par des tirets).

## Les différentes étapes de l'assistant
<a name="les_differentes_etapes_assistant_ELTTEXTE000240"></a>


### Orientation de la fenêtre
<a name="orientation_fenetre_ELTPARAGRAPHE000028"></a>Plusieurs options sont possibles : 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=IOS_Ass_Orientation.gif)
Ces options peuvent être modifiées sous l'éditeur, dans l'onglet "Détail" de la fenêtre de description de la fenêtre (option "Orientation"). 

Il est également possible de spécifier si des ascenseurs automatiques doivent être affichés. Les ascenseurs seront automatiquement affichés si la fenêtre a une taille supérieure à la résolution de l'écran de l'appareil utilisé.![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=IOS_FenDetail.gif)



### Contenu de la fenêtre : simple ou double
<a name="contenu_fenetre_simple_double_ELTPARAGRAPHE000038"></a>
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=iPad_Simple_double.gif)
Vous pouvez créer : 

- une fenêtre **simple** : les données sont affichées dans la totalité de la fenêtre. 
	Dans ce cas, la suite de l'assistant permet de définir le contenu de la fenêtre. 

- une fenêtre **double** : la fenêtre est scindée en deux parties. Ce type d'interface permet par exemple de mettre en place un menu ou des choix d'options sur le coté gauche avec le détail à droite. 
	Dans ce cas, la suite de l'assistant permet de définir : 

	- le contenu de la zone gauche. 

	- le contenu de la zone droite. Ce contenu peut être : 

		- statique (défini sous l'éditeur) : l'assistant propose alors de définir le contenu de la zone. 

		- une fenêtre interne. Dans ce cas, un champ Fenêtre Interne est automatiquement créé dans la fenêtre. 









## Définir le contenu
<a name="definir_contenu_ELTTEXTE000270"></a>


### Quand définir le contenu ?
<a name="quand_definir_contenu_ELTPARAGRAPHE000058"></a>L'assistant permet de définir le contenu de la fenêtre dans les cas suivants : 

- fenêtre simple.

- partie gauche d'une fenêtre double. 

- partie droite d'une fenêtre double avec contenu statique.





### Contenu de la fenêtre
<a name="contenu_fenetre_ELTPARAGRAPHE000067"></a>
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=IOS_Ass_Contenu.gif)
Vous pouvez créer une fenêtre : 

- avec une **Action Bar**. Ce champ permet d'afficher plusieurs informations : l'icône de l'application, le titre de la fenêtre, certaines options de menu, le menu déroulant de votre fenêtre, ...
	Pour ajouter une "Action Bar" dans une fenêtre : sous le volet "Fenêtre", dans le groupe "Barres et menus", cliquez sur "Autres barres" et sélectionnez "Action Bar".

- avec une **zone répétée ou une zone multiligne**. Ce type de fenêtre permet par exemple de paramétrer une ou plusieurs options, de sélectionner une option, ...

- avec une **Tab bar (barre de thème)**. En règle générale, une "Tab bar" permet d'accéder aux différentes fenêtres de l'application.
	Pour ajouter une barre de thème dans une fenêtre : sous le volet "Fenêtre", dans le groupe "Barres et menus", déroulez "Autres barres" et sélectionnez "Barre de thème (Tab Bar iOS)".

- avec une **barre d'outils**. En règle générale, la barre d'outils permet d'effectuer des actions sur la fenêtre en cours. 
	Pour ajouter une barre d'outils dans une fenêtre : sous le volet "Fenêtre", dans le groupe "Barres et menus", déroulez "Autres barres" et sélectionnez "Barre d'outils (ToolBar iOS)".




**Différence entre une barre de thème et une barre d'outils** : 

- Une barre d'outils propose des actions sur la vue courante.

- Une barre de thèmes correspond à des onglets.





### Configuration de la zone répétée ou multiligne
<a name="configuration_zone_repetee_multiligne_ELTPARAGRAPHE000125"></a>Si vous avez choisi de créer une fenêtre avec une zone répétée ou une zone multiligne, les différents plans de l'assistant permettent de configurer cet élément. 

- Plusieurs types de zones répétées prédéfinies ou de zones multilignes sont disponibles : 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=IOS_Ass_TypeZR.gif)

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=IOS_ass_typeZM.gif)


- L'étape suivante permet de paramétrer les éléments affichés dans la zone répétée ou dans la zone multiligne. 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=IOS_ass_ParamZR.gif)


- Dans le cas d'une zone répétée, il est possible de choisir le type de remplissage de la zone répétée (programmation, liaison fichier, liaison variable).




A la validation de l'assistant, la zone répétée ou la zone multiligne choisie est automatiquement créée dans la fenêtre. 

Vous pouvez saisir le code permettant de la remplir et de la faire fonctionner. 

**Dans la cas d'une zone répétée**, les principaux événements à modifier sont les suivants :

- Code d'initialisation de la zone répétée : cet événement permet par exemple de remplir la zone répétée. 

- Code de sélection d'une ligne : cet événement permet de gérer les actions effectuées lorsque l'utilisateur clique sur la flèche standard. 

- Code de clic de l'image IMG_Disclosure : cet événement permet de gérer le clic sur la puce personnalisée. 




Pour plus de détails, consultez : 

- Champ [Zone répétée](../WDChamp/1013099.md). 

- Champ [Zone multiligne](../WDChamp/1000024001.md). 





### Barre de thème ou barre d'outils
<a name="barre_theme_barre_outils_ELTPARAGRAPHE000161"></a>Si vous avez choisi cette option, spécifiez les paramètres de la barre : 

- Couleur

- Style

- Boutons affichés




A la fin de l'assistant, la fenêtre est créée avec la barre spécifiée. Il ne reste plus qu'à saisir le code des boutons.

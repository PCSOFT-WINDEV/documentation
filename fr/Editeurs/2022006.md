
## Déplacement ancré sur les axes principaux
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000089"></a>
Pour simplifier le positionnement des champs, il est possible de forcer leur déplacement selon les quatre axes principaux (0°, 45°, 90°, 135°).

De cette façon, un champ peut être déplacé en restant sur une ligne donnée (pour être aligné avec une image de fond par exemple).

Cette fonctionnalité vient en complément du mécanisme de [magnétisme automatique](../Editeurs/2022002.md) des éditeurs de pages et de fenêtres.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000113"></a>
Pour déplacer un champ ou un groupe de champs d'une fenêtre, d'une page ou d'un état :

1. Sélectionnez le ou les champs à déplacer.

2. Commencez le déplacement à la souris.

3. Pendant le déplacement, pressez la touche Alt. Les champs seront alors automatiquement positionnés sur l'axe (horizontal, vertical ou oblique) le plus proche et resteront sur cet axe tant que la touche Alt restera enfoncée.




<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Remarque
<a name="remarque_ELTTEXTE000137"></a>
Le déplacement ancré est prioritaire sur la [grille magnétique](../Editeurs/2022002.md) lorsque les deux mécanismes sont susceptibles d'aider au positionnement d'un champ.




## Créer un gabarit à partir d'un gabarit existant
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000116"></a>
Il est possible de créer un gabarit de projet à partir d'un gabarit existant (par exemple un gabarit fourni par PC SOFT). 

Cette page d'aide décrit la procédure à suivre. 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000140"></a>


### Créer un gabarit à partir d'un gabarit existant
<a name="creer_gabarit_partir_gabarit_existant_ELTPARAGRAPHE000020"></a>**Pour créer un gabarit à partir d'un gabarit existant** : 

1. Fermez WINDEV. 

2. Sous l'explorateur de fichiers Windows, copiez le répertoire contenant le gabarit de base à utiliser et renommez ce répertoire. Les gabarits sont présents dans le répertoire "\\Personnel\\Gabarits\\WD" du répertoire d'installation de WINDEV. 
	Par exemple, copiez le répertoire "190 Gensteel" et renommez-le en "Perso Gensteel". 
	**Attention** : le nom du répertoire ne doit pas contenir un chiffre devant le nom. Cette notation est réservée aux gabarits PC SOFT.

3. Lancez WINDEV. 

4. Ouvrez sous l'éditeur le fichier ".sty" présent dans le nouveau répertoire du gabarit. 
	Dans notre exemple, il faut ouvrir le fichier "Gensteel.sty" présent dans le répertoire "Perso Gensteel". 

5. Enregistrez le fichier ".sty" sous un nouveau nom grâce à l'option "Enregistrer sous" (Ctrl + Maj + S). Le nom du fichier ".sty" doit correspondre au nom du gabarit. Validez la création d'un nouveau gabarit par copie. 
	Dans notre exemple, le fichier "Gensteel.sty" doit être renommé en "Perso Gensteel.sty". 

6. Le fichier ".sty" peut être modifié selon vos besoins. 

7. Fermez WINDEV. 

8. Sous l'explorateur de fichiers Windows, supprimez l'ancien fichier ".sty" dans le répertoire du gabarit. 
	Dans notre exemple, supprimez le fichier "Gensteel.sty" dans le répertoire "Perso Gensteel". 




Il est ensuite nécessaire de décrire les différentes fenêtres du gabarit. 

**Pour décrire les différentes fenêtres du gabarit** : 

1. Lancez WINDEV et n'ouvrez pas de projet (fermez si nécessaire la page d'accueil).

2. Ouvrez la fenêtre "&lt;Nom ancien gabarit&gt;_0_D" sous l'éditeur. 

3. Affichez la fenêtre de description de la fenêtre. 

4. Sous l'onglet "Style", dans la zone "Gabarit en cours", sélectionnez le nom du nouveau gabarit. 

5. Validez. Répondez "Oui" à la demande d'application de gabarit. 

6. Dans la fenêtre qui s'affiche, cochez toutes les options (sauf la première). Validez. 

7. Dans l'onglet "Général" de la fenêtre de description, modifiez le nom de la fenêtre en remplaçant le nom de l'ancien gabarit par le nouveau nom. 

8. Validez et enregistrez la fenêtre. 




Ces opérations peuvent être effectuées pour les fenêtres suivantes : 

- "&lt;Nom ancien gabarit&gt;_1" (manipulation obligatoire)

- "&lt;Nom ancien gabarit&gt;_13" (manipulation optionnelle mais conseillée)

- "&lt;Nom ancien gabarit&gt;_22" (manipulation optionnelle mais conseillée)

- "&lt;Nom ancien gabarit&gt;_23" (manipulation optionnelle mais conseillée)

- "&lt;Nom ancien gabarit&gt;_24" (manipulation optionnelle mais conseillée)





### Utiliser le nouveau gabarit
<a name="utiliser_nouveau_gabarit_ELTPARAGRAPHE000065"></a>Le nouveau gabarit est automatiquement proposé lors de la création de fenêtres ou de projets. 

Il est également possible de l'appliquer à un projet ou une fenêtre existante. Dans ce cas, il est nécessaire de cocher les options "Appliquer en se basant sur le nom des styles" et "Conserver, si possible, les surcharges de style".




## Manipuler les vues d'une fenêtre 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000086"></a>
Pour simplifier la manipulation de fenêtres importantes (par exemple ne pouvant pas être affichées entièrement sous l'éditeur), WINDEV et WINDEV Mobile permettent de manipuler plusieurs vues d'une même fenêtre. 

Chaque vue peut afficher une partie différente d'une même fenêtre. 

Toutes les modifications réalisées sur une des vues sera automatiquement reportée sur toutes les vues ouvertes.
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Vue_Fen.gif)


<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000110"></a>
Pour afficher une nouvelle vue de la fenêtre en cours sous l'éditeur, sous le volet "Affichage", dans le groupe "Options", cliquez sur "Dupliquer l'affichage".

Pour fermer une des vues, il suffit de fermer la fenêtre correspondante. 

**Remarque** : Les vues étant basées sur le même fichier, un seul élément ouvert apparaît dans la barre des éléments ouverts.

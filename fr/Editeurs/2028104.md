
## Mise à jour automatique (Applications réseau)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000088"></a>
Lors d'une installation réseau, une installation de référence est présente sur un ou plusieurs serveurs de référence. Les utilisateurs installent leur application depuis le serveur de référence.

**Cette étape de l'assistant permet de gérer les mises à jour automatiques des applications depuis le serveur de référence**.

Ces paramètres peuvent être si nécessaire modifiés lors de l'installation de l'application sur le serveur de référence (il suffit de cocher l'option "Ces paramètres pourront être modifiés au moment du déploiement sur le serveur"). 



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Fréquence de vérification de la disponibilité des mises à jour
<a name="frequence_verification_disponibilite_des_mises_jour_ELTTEXTE000112"></a>
Par défaut, la vérification de la disponibilité éventuelle d'une mise à jour sur le serveur est effectuée à chaque lancement de l'application. Cette fréquence de vérification est paramétrable. Elle peut être effectuée : 

- A chaque ouverture de l'application. 

- Tous les X jours. 



![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=install_verif.gif)


<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Interactions avec l'utilisateur lors de la vérification de la disponibilité des mises à jour
<a name="interactions_avec_utilisateur_lors_verification_disponibilite_des_mises_jour_ELTTEXTE000136"></a>
Une application installée via une installation réseau vérifie à chaque lancement si une mise à jour est disponible sur le serveur de référence. 

- Si une version plus récente est disponible sur le serveur, la mise à jour de l'application est proposée à l'utilisateur final. 

- Si la connexion réseau n'est pas trouvée à l'ouverture de l'application, le traitement par défaut est d'afficher un message d'avertissement. 
	Ce message peut être inutile dans le cadre d'une application utilisée sur un portable, connecté par intermittence. Dans ce cas, il suffit de sélectionner l'option "Vérification silencieuse : Ne pas informer l'utilisateur".

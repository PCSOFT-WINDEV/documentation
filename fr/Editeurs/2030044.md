
## Reporter les différences d'un élément à un autre
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000208"></a>
Le comparateur de projets permet de comparer deux éléments et de lister les différences trouvées.

Pour plus de détails, consultez [Comparateur de projets](../Editeurs/2030024.md).

Après avoir listé les différences trouvées, il est possible de reporter d'un élément à un autre :

- les différences de code.

- les différences de caractéristiques (taille et position d'une fenêtre ou d'une page par exemple).

- les champs existant uniquement dans un des deux éléments (fenêtre, page ou état).

- les procédures locales existant uniquement dans un des deux éléments (fenêtre, page ou état).

- les procédures globales existant uniquement dans une des deux collections de procédures.

- les méthodes existant uniquement dans une des deux classes.




Par exemple :
<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Merge%20code%20element%20-%20HC%20N%B0001.GIF&type=thumb)


Comparez et récupérez d'un clic les lignes qui vous intéressent !
<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Merge%20code%20element%20-%20HC%20N%B0002.GIF&type=thumb)

## Comment le faire ?
<a name="comment_faire_ELTTEXTE000232"></a>


### Reporter les différences de code d'un élément à un autre
<a name="reporter_les_differences_code_element_autre_ELTPARAGRAPHE000041"></a>Pour reporter les différences de code d'un élément à un autre :

1. Sélectionnez la ou les lignes de code à reporter.

2. Cliquez sur :

	- l'icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Merge%20code%20element%20-%20HC%20N%B0001%201.GIF) pour reporter le code sélectionné dans la fenêtre de droite dans celle de gauche.

	- l'icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Merge%20code%20element%20-%20HC%20N%B0001%202.GIF) pour reporter le code sélectionné dans la fenêtre de gauche dans celle de droite.





**Limitation :** Il n'est pas possible de reporter les traitements associés au projet.


<a name="NOTE2_2"></a>


### Reporter les différences de caractéristiques d'un élément à un autre
<a name="reporter_les_differences_caracteristiques_element_autre_ELTPARAGRAPHE000061"></a>Pour reporter les différences de caractéristiques d'un élément à un autre :

1. Sélectionnez la ligne de la caractéristique à reporter.

2. Cliquez sur :

	- l'icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Merge%20code%20element%20-%20HC%20N%B0001%201.GIF) pour reporter la caractéristique sélectionnée dans la fenêtre de droite dans celle de gauche.

	- l'icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Merge%20code%20element%20-%20HC%20N%B0001%202.GIF) pour reporter la caractéristique sélectionnée dans la fenêtre de gauche dans celle de droite.





**Limitation :** Il n'est pas possible de reporter le nom logique d'un élément à un autre.


<a name="NOTE2_3"></a>


### Reporter les champs existant uniquement dans un des deux éléments (fenêtre, page ou état)
<a name="reporter_les_champs_existant_uniquement_dans_des_deux_elements_fenetre_page_etat_ELTPARAGRAPHE000081"></a>Pour reporter les champs existant uniquement dans un des deux éléments (fenêtre, page ou état) :

1. Sélectionnez le champ à reporter.

2. Cliquez sur :

	- l'icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Merge%20code%20element%20-%20HC%20N%B0001%201.GIF) pour reporter le champ sélectionné dans la fenêtre de droite dans celle de gauche.

	- l'icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Merge%20code%20element%20-%20HC%20N%B0001%202.GIF) pour reporter le champ sélectionné dans la fenêtre de gauche dans celle de droite.





**Limitation :** Il n'est pas possible de reporter :

- les colonnes d'une table. Il est cependant possible de reporter un champ Table.

- les attributs d'une zone répétée. Il est cependant possible de reporter une zone répétée.

- un champ appartenant à un autre champ (superchamp, zone répétée, cellule, fenêtre interne, ...). Dans ce cas, il est nécessaire de reporter aussi le champ "conteneur". 
	Par contre, lors du report d'un champ pouvant contenir d'autres champs (superchamp, zone répétée, cellule, fenêtre interne, ...), il est possible de reporter uniquement certains des champs contenus dans ce champ.

- les menus contextuels et/ou leurs options de menus.





<a name="NOTE2_4"></a>


### Reporter les procédures locales existant uniquement dans un des deux éléments (fenêtre, page ou état)
<a name="reporter_les_procedures_locales_existant_uniquement_dans_des_deux_elements_fenetre_page_etat_ELTPARAGRAPHE000108"></a>Pour reporter les procédures locales existant uniquement dans un des deux éléments (fenêtre, page ou état) :

1. Sélectionnez la procédure locale à reporter.

2. Cliquez sur :

	- l'icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Merge%20code%20element%20-%20HC%20N%B0001%201.GIF) pour reporter la procédure locale sélectionnée dans la fenêtre de droite dans celle de gauche.

	- l'icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Merge%20code%20element%20-%20HC%20N%B0001%202.GIF) pour reporter la procédure locale sélectionnée dans la fenêtre de gauche dans celle de droite.








<a name="NOTE2_5"></a>


### Reporter les procédures globales existant uniquement dans une des deux collections de procédures
<a name="reporter_les_procedures_globales_existant_uniquement_dans_une_des_deux_collections_procedures_ELTPARAGRAPHE000126"></a>Pour reporter les procédures globales existant uniquement dans une des deux collections de procédures :

1. Sélectionnez la procédure globale à reporter.

2. Cliquez sur :

	- l'icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Merge%20code%20element%20-%20HC%20N%B0001%201.GIF) pour reporter la procédure globale sélectionnée dans la fenêtre de droite dans celle de gauche.

	- l'icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Merge%20code%20element%20-%20HC%20N%B0001%202.GIF) pour reporter la procédure globale sélectionnée dans la fenêtre de gauche dans celle de droite.





**Limitation** : Il n'est pas possible de reporter une collection de procédures. Pour reporter une procédure globale, il est nécessaire que la collection de procédures correspondante existe dans les deux projets comparés.


<a name="NOTE2_6"></a>


### Reporter les méthodes existant uniquement dans une des deux classes
<a name="reporter_les_methodes_existant_uniquement_dans_une_des_deux_classes_ELTPARAGRAPHE000146"></a>Pour reporter les méthodes existant uniquement dans une des deux classes :

1. Sélectionnez la méthode à reporter.

2. Cliquez sur :

	- l'icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Merge%20code%20element%20-%20HC%20N%B0001%201.GIF) pour reporter la méthode sélectionnée dans la fenêtre de droite dans celle de gauche.

	- l'icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Merge%20code%20element%20-%20HC%20N%B0001%202.GIF) pour reporter la méthode sélectionnée dans la fenêtre de gauche dans celle de droite.





**Limitation :** Il n'est pas possible de reporter une classe. Pour reporter une méthode, il est nécessaire que la classe correspondante existe dans les deux projets comparés.

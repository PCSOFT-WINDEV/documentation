


## Manipulations d'un Modèle Logique des Données (MLD)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000216"></a>
Les principales manipulations possibles sur un MLD sont les suivantes :

- [Dupliquer/Copier un MLD](#NOTE2_1).

- [Supprimer un MLD](#NOTE2_2).

- [Renommer un MLD](#NOTE2_3).

- [Associer un MLD à un projet](#NOTE2_4).

- [Agrandir ou réduire l'affichage d'un MLD](#NOTE2_5).

- [Déplacer l'affichage d'un MLD](#NOTE2_6).




**Remarque** : Les termes "MLD" et "Analyse" sont utilisés indifféremment pour définir la structure de la base de données associée à un projet.

## Manipulation d'un MLD
<a name="manipulation_mld_ELTTEXTE000240"></a>


### Dupliquer/Copier un MLD
<a name="dupliquercopier_mld_ELTPARAGRAPHE000039"></a>La duplication d'un MLD permet de disposer de deux MLD identiques de nom différent.

Pour dupliquer/copier un MLD :

1. Ouvrez l'analyse à dupliquer sous l'éditeur d'analyses.

2. Sous le volet "Accueil", dans le groupe "Général", déroulez "Enregistrer" et sélectionnez "Enregistrer sous". La description de l'analyse s'ouvre.

3. Modifiez le nom et le répertoire de l'analyse.

4. Validez les modifications :

	- L'ensemble des fichiers décrivant l'analyse dupliquée est automatiquement créé dans le répertoire spécifié.

	- L'analyse dupliquée est automatiquement associée au projet en cours.







**Remarque** : Les sous-répertoires de sauvegarde de l'analyse d'origine ne sont pas copiés. La nouvelle analyse ne contient aucune sauvegarde.
<a name="NOTE2_2"></a>


### Supprimer un MLD
<a name="supprimer_mld_ELTPARAGRAPHE000070"></a>Pour supprimer un MLD :

1. Ouvrez l'explorateur de Windows.

2. Supprimez le répertoire complet de l'analyse (répertoire &lt;Nom du projet&gt;.ANA).




**Remarque** : Si un projet utilisait cette analyse, il sera nécessaire d'associer le projet à une autre analyse lors de l'ouverture de ce projet.
<a name="NOTE2_3"></a>


### Renommer un MLD
<a name="renommer_mld_ELTPARAGRAPHE000082"></a>Pour renommer un MLD :

1. Ouvrez le MLD à renommer sous l'éditeur d'analyses.

2. Sous le volet "Accueil", dans le groupe "Général", déroulez "Enregistrer" et sélectionnez "Enregistrer sous". La description de l'analyse s'ouvre.

3. Saisissez le nouveau nom du MLD.

4. Validez la modification.

5. L'ensemble des fichiers décrivant l'analyse renommée est automatiquement créé dans le répertoire spécifié.

6. L'analyse renommée n'est pas automatiquement associée au projet en cours. Associez si nécessaire l'analyse renommée au projet en cours. Pour plus de détails, consultez Associer un MLD à un projet existant.

7. Supprimez si nécessaire sous l'explorateur de Windows le répertoire de l'analyse d'origine.



<a name="NOTE2_4"></a>


### Associer un MLD à un projet existant
<a name="associer_mld_projet_existant_ELTPARAGRAPHE000109"></a>Pour associer un MLD à un projet existant :

1. Ouvrez le projet à associer.

2. Sous le volet "Projet", dans le groupe "Projet", cliquez sur "Description". La description du projet s'ouvre.

3. Dans l'onglet "Analyse", indiquez le nom et le chemin du MLD à associer au projet. .




**Remarque** : Si des données de l'analyse d'origine sont manipulées dans le projet, l'association d'une nouvelle analyse peut entraîner des erreurs (erreurs de doublons, erreurs d'affichage, ...).
<a name="NOTE2_5"></a>


### Facteur de zoom
<a name="facteur_zoom_ELTPARAGRAPHE000134"></a>Il est possible de spécifier un facteur de zoom sous l'éditeur d'analyses pour réduire ou agrandir l'affichage du MLD en cours.

Pour spécifier un facteur de zoom :

- **1ère méthode** :

	- Soit sous le volet "Affichage", dans le groupe "Aide à l'édition", choisissez le facteur de zoom à utiliser.

	- Soit saisissez directement le facteur de zoom dans la barre de message de l'éditeur (en bas à droite).




<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ZoomBarreMessage.gif)


- **2ème méthode** :

	- Appuyez sur la touche Ctrl.

	- Modifiez le facteur de zoom avec la roulette de la souris tout en maintenant la touche Ctrl enfoncée.






<a name="NOTE2_6"></a>


### Déplacer l'affichage d'un MLD
<a name="deplacer_affichage_mld_ELTPARAGRAPHE000163"></a>Pour déplacer l'affichage d'un MLD :

1. Ouvrez le MLD dans l'éditeur d'analyses.

2. Appuyez sur la touche Alt.

3. Modifiez la position du MLD en maintenant à la fois le bouton gauche de la souris et la touche Alt enfoncés.

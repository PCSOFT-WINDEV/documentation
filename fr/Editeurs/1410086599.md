## Présentation
<a name="presentation_ELTTEXTE000177"></a>
WDBal est une messagerie sécurisée, intégrée à WINDEV, WEBDEV et WINDEV Mobile. Cette messagerie permet notamment aux différents développeurs d'une équipe de communiquer entre eux. 

Cette messagerie peut également être utilisée pour recevoir des messages des différents outils de l'environnement : 

- GDS (Gestionnaire de sources). 

- Centre de Contrôle (gestion des incidents). 




WDBal propose toutes les fonctionnalités d'une messagerie avancée : 

- Gestion des messages : Nouveau message, Accusé de réception, Suppression de messages, Transfert de messages, ...

- Gestion de la recherche : recherche selon l'expéditeur, la date, parmi les messages envoyés ou reçus, ...

- Gestion des dossiers pour conserver les messages reçus et les trier par thèmes.




Les intervenants proposés dans WDBal sont les intervenants spécifiés dans le Centre de Suivi de Projets. Pour plus de détails, consultez [Centre de Suivi de projets : Gestion des intervenants](../CCSuivi/3540703.md).

## Lancement de la messagerie
<a name="lancement_messagerie_ELTTEXTE000201"></a>
Pour lancer la messagerie WDBal : 

- directement depuis WINDEV, WEBDEV ou WINDEV Mobile : sous le volet "Outils", dans le groupe "Centres de Contrôle", cliquez sur "WDBal - Messagerie". 

- exécutez le programme WDBal.exe présent dans le sous-répertoire "Programmes" du répertoire d'installation de WINDEV, WEBDEV ou WINDEV Mobile.

- affichez le volet "Messagerie" dans votre environnement de travail

- ajoutez un onglet de type "Messagerie" dans le Centre de Suivi de projets. 




Si la messagerie est lancée de manière indépendante, elle s'affiche par défaut sous forme de "Mini-barre". ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Messagerie_Presentation%20-%20HC%20N%B0001.gif)
Il est possible de paramétrer le mode d'affichage de la messagerie dans ses options. 

Attention : La messagerie WDBal n'est pas un outil redistribuable.

## Paramétrage des options
<a name="parametrage_des_options_ELTTEXTE000225"></a>
Pour paramétrer les options de la messagerie : 

- en mode "Mini-barre" : sélectionnez l'option "Outils .. Options de la bal".

- en mode "Affichage complet" : Sous le volet "Outils", dans le groupe "Options", cliquez sur "Options". 




Les options disponibles sont les suivantes : ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Messagerie_Presentation%20-%20HC%20N%B0002.gif)


- **Mémoriser le mot de passe au démarrage** : Permet de mémoriser le mot de passe associé à l'utilisateur. La fenêtre de connexion à la messagerie propose automatiquement le mot de passe. 

- **Iconiser à côté de l'horloge (systray)** : Permet d'iconiser l'application à côté de l'horloge. Si un nouveau message est reçu, l'icône affichée devient rouge. 

- **Rafraîchir lors de la restauration (F5 automatique)** : Permet de rafraîchir automatiquement les messages reçus lors de l'affichage des messages. Il est également possible de rafraîchir la liste des messages par la touche F5. 

- **Fermeture automatique de l'application au bout de xx minutes** : Permet de fermer la messagerie au bout d'un certain temps d'inutilisation.

- **Mode mini-barre** : Permet d'afficher la messagerie en mode "Mini-barre" ou en mode complet. 

- **Biper quand plus de 3 caractères majuscules d'affilés** : Permet de signaler la saisie de texte en majuscules. 

- **Mode treeview** :  Permet d'afficher les messages sous forme d'arbre. Chaque message et les réponses correspondantes au message initial sont affichées sous forme hiérarchique. Par défaut (option non cochée), les messages reçus sont affichés dans une table, sous forme de liste. Le mode treeview est recommandé si la messagerie ne comporte pas trop de messages reçu en attente.

## Le mode mini-barre
<a name="mode_minibarre_ELTTEXTE000249"></a>
Le mode mini-barre présente les options suivantes : ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Messagerie_Presentation%20-%20HC%20N%B0001%201.gif)


1. Ecrire un nouveau message. 

2. Rechercher un message. 

3. Lire les messages reçus. 

4. Effectuer un appel général. 

5. Afficher la liste des numéros de téléphone.

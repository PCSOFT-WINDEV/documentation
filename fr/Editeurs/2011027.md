


## Génération de scripts SQL
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000089"></a>
Le contenu d'une analyse peut être exporté sous forme de scripts SQL pour une base de données spécifiée (SQL Server, Oracle, ...). Ce script génère les contraintes d'intégrité, les index, ...

Ce script peut être modifié, puis joué pour créer des tables sur un serveur. 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000113"></a>
Pour générer le script SQL correspondant à une analyse (ou un fichier) :

1. Sous le volet "Analyse", dans le groupe "Analyse", déroulez "Génération" et sélectionnez "Générer le script SQL". L'assistant se lance.

2. Sélectionnez les tables/fichiers pour lesquelles le script SQL doit être généré.

3. Sélectionnez le type de la base pour laquelle le script SQL doit être généré. Toutes les bases disponibles sont listées.

4. Indiquez les options de génération du code SQL :

	- Générer DROP TABLE (code de suppression des tables existantes).

	- Générer les index.

	- Générer les contraintes d'intégrité.




5. Le script généré est affiché. Ce script peut être enregistré dans un fichier spécifique pour être modifié, puis exécuté.

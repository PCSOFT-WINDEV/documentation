
## Application Universal Windows 10 App : Plateforme de déploiement
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000130"></a>
A la fin de la génération de l'application Universal Windows 10 App, l'assistant propose de sélectionner la plateforme de déploiement : 

- la machine locale. 

- un téléphone connecté en USB. 

- un téléphone du réseau. 




Remarque : L'application générée peut également être installée sur une tablette Windows 10. 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Déploiement sur la machine locale
<a name="deploiement_sur_machine_locale_ELTTEXTE000154"></a>
Pour déployer sur la machine locale, le mode développeur doit être activé sur le poste. 

Ce mode peut être activé directement depuis l'assistant de déploiement : 

1. Dans l'assistant de déploiement, sélectionnez l'option "Déployer et lancer l'application sur la machine locale". 

2. Passez à l'étape suivante de l'assistant. 

3. Ouvrez la fenêtre de paramétrage du mode développeur en cliquant sur le lien "Mode développeur". 

4. Dans cette fenêtre Windows, cliquez sur "Mode développeur". 

5. Un message d'avertissement de Windows apparaît : validez et fermez la fenêtre de paramétrage de Windows. 

6. Validez l'assistant. 

7. L'application est automatiquement déployée et lancée. 




<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Déploiement sur un téléphone connecté en USB
<a name="deploiement_sur_telephone_connecte_usb_ELTTEXTE000178"></a>
Pour déployer sur un téléphone connecté en USB, le mode développeur doit être activé sur le téléphone. 

Ce mode peut être activé directement depuis l'assistant de déploiement : 

1. Dans l'assistant de déploiement, sélectionnez l'option "Déployer l'application sur un téléphone connecté en USB". 

2. Passez à l'étape suivante de l'assistant. 

3. L'assistant indique les opérations à effectuer sur le téléphone afin d'activer : 

	- Le mode développeur. 

	- La détection d'appareils. 




4. Ces opérations sont les suivantes : 

	- Affichez les "Paramètres" du matériel. 

	- Dans "Mise à jour et sécurité", cliquez sur "Pour les développeurs". 

	- Dans la zone "Utiliser les fonctionnalités de développement", activez le "Mode développeur". 

	- Dans la zone "Détection d'appareils", activez l'option permettant de rendre votre appareil visible pour les connexions USB et le réseau local. 




5. Validez l'assistant. 

6. L'application est automatiquement déployée sur le téléphone. Il est nécessaire de la lancer manuellement.

## Déploiement sur un téléphone en réseau
<a name="deploiement_sur_telephone_reseau_ELTTEXTE000202"></a>
Pour déployer sur un téléphone en réseau, le mode développeur doit être activé sur le téléphone. 

Ce mode peut être activé directement depuis l'assistant de déploiement : 

1. Dans l'assistant de déploiement, sélectionnez l'option "Déployer l'application sur un téléphone du réseau". 

2. Passez à l'étape suivante de l'assistant. 

3. Indiquez l'identifiant ou l'adresse IP du téléphone. 

4. L'assistant indique les opérations à effectuer sur le téléphone afin d'activer : 

	- Le mode développeur. 

	- La détection d'appareils. 




5. Ces opérations sont les suivantes : 

	- Affichez les "Paramètres" du matériel. 

	- Dans "Mise à jour et sécurité", cliquez sur "Pour les développeurs". 

	- Dans la zone "Utiliser les fonctionnalités de développement", activez le "Mode développeur". 

	- Dans la zone "Détection d'appareils", activez l'option permettant de rendre votre appareil visible pour les connexions USB et le réseau local. 




6. Si l'appareil n'a pas encore été couplé avec cet ordinateur, sur le téléphone, dans la zone "Détection d'appareils", cliquez sur "Coupler". Le code s'affichant sur l'écran du téléphone doit être saisi dans le champ "Code" de l'assistant. 

7. Validez l'assistant. 

8. L'application est automatiquement déployée sur le téléphone. Il est nécessaire de la lancer manuellement.

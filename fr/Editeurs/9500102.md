


## Menu automatique des tables (FAA)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000162"></a>
Chaque champ Table présent dans une fenêtre propose par défaut un menu à l'utilisateur.

Ce menu, disponible sans programmation, permet des actions utiles :

- Export vers Word ou vers OpenOffice Writer (selon le logiciel installé sur le poste de l'utilisateur).

- Export vers Excel ou vers OpenOffice Calc (selon le logiciel installé sur le poste de l'utilisateur).

- Export vers XML.

- Copie de la ligne ou de la table.

- Création d'un graphique de représentation des données.

- Impression de la table (voir [FAA d'impression d'une table](../WDChamp/9500132.md)). 

- Sélection de plusieurs colonnes de table :




Pour simplifier le menu, seules les 32 premières colonnes sont visibles dans le menu. De plus, seules les colonnes ayant un titre sont proposées.<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Fonctionnalites_auto_Menu.gif)
 
Exemple de graphique généré automatiquement par le menu contextuel d'un champ Table :<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=FAA_Livre_Table_Graphe%20-%20HC%20N%B0002.gif)
Grâce à ce menu, l'utilisateur gagne en autonomie et peut réaliser des traitements qui n'étaient pas forcément prévus dans l'application :

- Exportation dans un fichier externe Excel, Word ou XML.

- Possibilités d'afficher ou masquer les colonnes (pour éviter d'avoir un menu trop important, seules les 32 première colonnes de la table sont affichées).

- Génération d'un graphique automatique à partir des données de la table. 
	Remarque : il n'est pas possible de sélectionner les colonnes à intégrer dans le graphe. Pour choisir uniquement certaines colonnes de la table, il est conseillé de créer un nouveau graphe à partir du champ Table s'affichant dans une fenêtre.




Remarques :

- Il est possible d'ajouter des choix à ce menu par programmation. Il suffit d'utiliser la propriété [MenuContextuel](../Proprietes/2510077.md).

- Lors de l'export d'une table par le menu contextuel, la fusion des colonnes est ignorée.

- Table horizontale : Le menu des FAA ne contient pas les options permettant de réaliser un export dans un fichier externe Excel, Word ou XML.

## Paramétrage
<a name="parametrage_ELTTEXTE000186"></a>


### Paramétrage du menu automatique dans un champ Table
<a name="parametrage_menu_automatique_dans_champ_table_ELTPARAGRAPHE000053"></a>Le menu contextuel d'un champ Table peut bien évidemment être désactivé. Pour paramétrer l'affichage du menu contextuel du champ Table, vous devez :

1. Ouvrir la fenêtre de description du champ Table.

2. Sélectionner l'onglet "UI".

3. Choisir le menu désiré dans la combo "Menu contextuel". Il est possible de : 

	- "Afficher le menu des FAA (Système)" : Dans ce cas, le menu contextuel affiché sera le menu automatique du champ Table (Menu des FAA). Pour plus de détails sur la configuration de ce menu, consultez [Configurer le menu des FAA](../Editeurs/2010040.md).

	- "Ajouter un menu contextuel" : Si cette option est cochée, vous pouvez sélectionner le menu contextuel personnalisé à afficher. <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Fonctionnalites_auto_Menu2.gif)






Remarques : 

- Si les deux options sont cochées, le menu contextuel personnalisé peut être ajouté avant ou après le menu des FAA. 

- Si aucune option n'est sélectionnée, aucun menu contextuel ne sera sélectionné. L'option &lt;Désactivé&gt; sera affichée dans la fenêtre de description. 

- La propriété [MenuContextuel](../Proprietes/2510077.md) permet également de supprimer le menu contextuel spécifique aux FAA et de retrouver le menu Windows classique pour un champ spécifique.

### Paramétrage de l'option de menu "Sélectionner les colonnes..."
<a name="parametrage_option_menu_selectionner_les_colonnes_ELTPARAGRAPHE000095"></a>Dans certains cas il peut être gênant de donner la possibilité à l'utilisateur de masquer des colonnes. Vous pouvez pour cela activer ou désactiver l'option de menu "Sélectionner les colonnes..." :

1. Ouvrez la fenêtre de description du champ Table.

2. Sélectionnez l'onglet "Détail".

3. Cochez ou décochez l'option "Menu 'Sélectionner les colonnes...'".


## Repérage immédiat de code modifié / code en cours
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000080"></a>
Sous l'éditeur de code : 

- Les lignes de code modifiées sont immédiatement visibles (1).

- L'événement dans lequel se trouve le curseur est facilement repérable (2). 



<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CodeModifie.GIF)


(1) : **Les lignes de code modifiées**
Seules les modifications de code réalisées depuis l'ouverture de l'élément en cours sont prises en compte :

- les lignes de code précédées d'une bande bleue ont été sauvegardées.

- les lignes de code précédées d'une bande orange n'ont pas encore été sauvegardées.




Lors de la fermeture de l'élément, les bandes colorées disparaissent automatiquement.

(2) : **Evénement en cours**
La barre verticale à gauche du code change de couleur pour identifier facilement l'événement WLangage où se trouve le curseur. Utile dans une page de code bien remplie.

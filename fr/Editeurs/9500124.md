


## Calendrier automatique sur les champs de saisie de type Date (FAA)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000112"></a>
Les champs de saisie de type Date proposent automatiquement une option "Calendrier" dans leur menu contextuel. Cette option permet d'afficher un calendrier et de sélectionner une date. Il suffit à l'utilisateur de cliquer sur une date pour que celle-ci soit sélectionnée et apparaisse dans le champ de saisie. <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=FAA_Calendrier.gif)

## Un calendrier automatique
<a name="calendrier_automatique_ELTTEXTE000136"></a>


### Mise en place
<a name="mise_place_ELTPARAGRAPHE000019"></a>L'option de menu "Calendrier" présente dans le menu contextuel des champs de saisie de type Date est automatique. 

Il est également possible d'afficher un bouton "Calendrier" à droite de la zone de saisie du champ. Pour cela, il suffit de cocher l'option "Afficher un bouton calendrier" dans l'onglet "Général" du champ de saisie. Le bouton "Éditer le calendrier..." permet de personnaliser le style du calendrier. 

En programmation : 

- la propriété [BoutonCalendrier](../Proprietes/1000017275.md) permet de savoir et d'indiquer si un champ de saisie de type Date propose un bouton pour afficher un calendrier en popup.

- la fonction [FAAExécute](../WDLang1/1000022099.md) utilisée avec la constante *faaCalendrier* permet d'afficher le calendrier sous le champ de saisie spécifié.

- la fonction [FAADésactive](../WDLang1/1000022018.md) utilisée avec la constante *faaCalendrier* supprime l'option "Calendrier" dans le menu contextuel du champ de saisie spécifié.

- la fonction [FAAChangeLibellé](../WDLang1/1000022100.md) permet de : 

	- modifier le libellé de l'option de menu "Calendrier" (constante *faaCalendrier*).

	- modifier le libellé du bouton "Aujourd'hui" présent dans la fenêtre du calendrier (constante *faaAujourdhui*). 








### Traduction du calendrier
<a name="traduction_calendrier_ELTPARAGRAPHE000058"></a>Par défaut, le calendrier est fourni en français et en anglais. 

Certains paramètres peuvent être traduits. Pour plus de détails, consultez [Champ Calendrier](../WDChamp/1016300.md).

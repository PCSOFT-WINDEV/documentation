


## Coloration du code WLangage
			



<a name="NOTE1_VMaj22"></a>
<a name="NOTE1_1_VMaj22"></a>


## Présentation
<a name="presentation_ELTTEXTE000148"></a>
A partir de la version 22, l'environnement de WINDEV, WEBDEV et WINDEV Mobile propose un thème clair et un thème foncé. Ces thèmes sont également disponibles dans l'éditeur de code. Vous avez également la possibilité de créer et configurer votre propre thème ou encore d'importer un thème existant. 

<a name="NOTE2_vmaj22"></a>
<a name="NOTE2_1_vmaj22"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000172"></a>
**Pour configurer les thèmes de l'éditeur de code** : 

1. Affichez les options d'édition de l'éditeur de code : sous le volet "Accueil", dans le groupe "Environnement", déroulez "Options" et sélectionnez "Options de l'éditeur de code".

2. Dans l'onglet "Coloration" les différents thèmes de l'éditeur de code sont listés : 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Coloration_Code_22%20-%20HC%20N%B0001.gif&type=thumb)


	- Clair (Version 21 et antérieures) : thème de l'éditeur de code correspondant à la version 21. 

- Clair : thème clair de l'éditeur de code (sur fond blanc).

- Foncé : thème foncé de l'éditeur de code (sur fond noir).

- Gris : thème gris de l'éditeur de code (sur fond gris).




Depuis cette fenêtre, vous pouvez : 

- utiliser un des thèmes proposés. 

- modifier un des thèmes proposés. 
	Remarque : Les thèmes proposés par défaut ne peuvent pas être modifiés directement. Il est nécessaire de les dupliquer pour les modifier. 

- supprimer un des thèmes proposés, 

- importer un thème. 




**Pour utiliser un thème**, double-cliquez sur le nom du thème : le libellé "(Actif)" est alors affiché sur la ligne du thème.

**Pour modifier un des thèmes proposés en standard** : 

1. Sélectionnez le thème voulu et cliquez sur "Dupliquer". 

2. Donnez un nom au nouveau thème et validez. 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Coloration_Code_22%20-%20HC%20N%B0002.gif)


3. Le nouveau thème apparaît dans la liste des thèmes disponible. Pour modifier ses caractéristiques, sélectionnez le thème et cliquez sur "Modifier". 

4. La fenêtre de paramétrage du thème est affichée. La liste de tous les éléments configurables est affichée. 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Coloration_Code_22%20-%20HC%20N%B0004.gif&type=thumb)
Pour chaque élément, il est possible de paramétrer (selon le type de l'élément): 

	- La couleur de fond. 

	- La couleur de la police. 

	- La police. 

	- La taille de la police. 

	- Les caractéristiques de la polices (gras, italique, souligné, barré). 




5. Pour quitter le mode modification, cliquez sur ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Coloration_Code_22%20-%20HC%20N%B0005%201.gif)
. L'éditeur  propose alors de sauvegarder ou d'annuler les modifications. 




**Pour supprimer un thème** : 

1. Sélectionnez un thème non actif. 

2. Cliquez sur le bouton "Supprimer" et validez. 


Attention : Les thèmes proposés par défaut ne peuvent pas être supprimés. 

**Pour importer un thème** : 

1. Cliquez sur le bouton "Importer". 

2. Sélectionnez le fichier ".sco" correspondant au thème à importer et validez. 


Remarque : Pour exporter un thème : 

1. Affichez l'écran de modification du thème. 

2. Cliquez sur le bouton ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Coloration_Code_22%20-%20HC%20N%B0005%202.gif)
 et sélectionnez l'option "Exporter un thème". 

3. Indiquez le nom du fichier ".sco" correspondant au thème et validez. 





### Cas particulier : Coloration du code hors configuration
<a name="cas_particulier_coloration_code_hors_configuration_ELTPARAGRAPHE000094"></a>Lorsqu'un élément hors configuration est ouvert sous l'éditeur, le code est coloré, mais affiché sur une couleur de fond spécifique (par défaut fond gris si vous utilisez le thème clair).

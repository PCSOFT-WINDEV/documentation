


## Créer les liens
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000177"></a>
Les liens permettent aux utilisateurs de se déplacer entre les différentes pages de l'aide et de parcourir le système d'aide. Un lien peut être fait :

- sur un mot ou un groupe de mots.

- sur une image.




Par exemple : 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=PageAide_Lien.gif)


WINDEV offre la possibilité de créer un lien vers :

- une page d'aide du système d'aide en cours.

- une ancre d'une page d'aide du système d'aide en cours.

- une page d'aide d'un autre système d'aide présent dans le projet.

- une page Internet.




Les caractéristiques des liens sont saisies dans les [Propriétés d'un lien](../Editeurs/2010022.md).

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000201"></a>


### Créer un lien sur du texte
<a name="creer_lien_sur_texte_ELTPARAGRAPHE000036"></a>Pour créer un lien :

1. Affichez la page d'aide devant contenir le lien dans l'éditeur d'aide.

2. Sélectionnez le groupe de mots sur lequel le lien doit être fait.

3. Sous le volet "Modification", dans le groupe "Edition", cliquez sur "Lien". Il est également possible d'utiliser l'option "Insérer un lien" du menu contextuel. La fenêtre des propriétés du lien s'affiche.

4. Définissez les caractéristiques du lien. Pour plus de détails, consultez [Propriétés d'un lien](../Editeurs/2010022.md).

5. Validez.





<a name="NOTE2_2"></a>


### Afficher les propriétés d'un lien sur du texte
<a name="afficher_les_proprietes_lien_sur_texte_ELTPARAGRAPHE000070"></a>Par défaut, lors de sa création, le lien permet d'afficher la page d'aide spécifiée dans la fenêtre d'aide en cours : la nouvelle page d'aide remplace la page d'aide actuellement affichée.

Il est possible de modifier ces caractéristiques grâce [aux propriétés du lien](../Editeurs/2010022.md). Il est par exemple possible d'afficher la page d'aide sous forme de "fenêtre popup".

Pour afficher les propriétés d'un lien :

1. Positionnez le curseur de la souris sur le texte du lien.

2. Double-cliquez sur le texte du lien ou sélectionnez l'option "Propriétés du lien" dans le menu contextuel (clic droit).



<a name="NOTE2_3"></a>


### Créer un lien sur une image
<a name="creer_lien_sur_une_image_ELTPARAGRAPHE000087"></a>Les images insérées dans une page d'aide peuvent contenir un ou plusieurs liens vers d'autres pages. Ces images sont alors appelées **images cliquables**. Ces liens peuvent être créés dès l'insertion de l'image dans la page d'aide, ou plus tard, grâce aux propriétés d'une image.

Pour plus de détails, consultez [Insérer une image dans une page d'aide](../Editeurs/2010026.md).
<a name="NOTE2_4"></a>


### Supprimer un lien sur du texte
<a name="supprimer_lien_sur_texte_ELTPARAGRAPHE000099"></a>Pour supprimer un lien :

1. Affichez la page d'aide contenant le lien dans l'éditeur d'aide.

2. Positionnez le curseur de la souris sur le mot ou l'image contenant le lien.

3. Sélectionnez l'option "Supprimer le lien" du menu contextuel. Le lien est automatiquement supprimé.



<a name="NOTE2_5"></a>


### Supprimer tous les liens sur du texte d'une page
<a name="supprimer_tous_les_liens_sur_texte_une_page_ELTPARAGRAPHE000110"></a>Pour supprimer tous les liens présents dans une page :

1. Affichez la page d'aide dans l'éditeur d'aide.

2. Positionnez le curseur de la souris dans la page.

3. Sélectionnez l'option "Supprimer tous les liens" du menu contextuel.



<a name="NOTE2_6"></a>


### Supprimer un ou plusieurs liens sur une image
<a name="supprimer_plusieurs_liens_sur_une_image_ELTPARAGRAPHE000121"></a>Pour supprimer les liens réalisés à partir d'une image :

1. Sélectionnez l'image cliquable.

2. Sélectionnez l'option "Propriétés de l'image" du menu contextuel.

3. Sélectionnez le lien à supprimer et cliquez sur le bouton "Supprimer". Pour plus de détails, consultez [Propriétés d'une image](../Editeurs/2010027.md).

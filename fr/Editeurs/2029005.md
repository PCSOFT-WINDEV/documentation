
## Environnement de l'éditeur de fenêtres 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a><br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=EnvEdFen.gif)


Les éléments importants de l'éditeur de fenêtres de WINDEV sont les suivants :


|   |   |
| --- | --- |
| <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CERCLE1.gif)<br> | **Ruban**<br>Permet un accès rapide aux principales fonctionnalités de l'éditeur de fenêtres. |
| <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CERCLE2.gif)<br> | [**Règles d'alignement**](../Editeurs/2022001.md)<br>Permettent d'aligner les champs dans la fenêtre. Pour afficher les règles, utilisez le raccourci Ctrl + R. |
| <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CERCLE3.gif)<br> | [Barre des documents ouverts](../Editeurs/2029012.md)<br>Chaque document ouvert correspond à un "bouton". Un clic sur le "bouton" permet d'afficher l'élément sous l'éditeur correspondant. |
| <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CERCLE4.gif)<br> | **Champ sélectionné**<br>Champ en cours de modification. Des poignées noires permettent d'identifier immédiatement le champ sélectionné.<br><br>Pour plus de détails, consultez [Sélectionnez un ou plusieurs champs](../Editeurs/2026007.md). |
| <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Cercle5.gif)<br> | [**Ancrage**](../WDChamp/2026002.md)<br>Les flèches rouges indiquent l'ancrage du champ.<br><br>Rappel : L'ancrage permet de gérer la position du champ lors du redimensionnement de la fenêtre. |
| <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CERCLE6.gif)<br> | **[Volets ancrables](../Editeurs/2027001.md)**<br>Affiche :<br><br>- les raccourcis vers les principales fonctionnalités disponibles sous l'éditeur en cours ([volet "Commandes"](../Editeurs/2027010.md)).<br><br>- les fonctionnalités les plus utilisées ([volet "Commandes"](../Editeurs/2027010.md)).<br><br>- les éléments (fenêtres, états, champs, ...) disponibles dans le projet en cours ([volet "Explorateur de Projet"](../Editeurs/2027029.md)).<br><br>- ...<br><br><br> |
| <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CERCLE7.gif)<br> | **Fenêtre de description**<br>Permet de visualiser et/ou de modifier les caractéristiques :<br><br>- du ou des champs sélectionnés.<br><br>- de la fenêtre sélectionnée.<br><br><br>Les informations affichées sont identiques à celles affichées dans le modifieur ([volet "Modifieur"](../Editeurs/2027006.md), affiché parmi les volets ancrables). |
| <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CERCLE8.gif)<br> | **[Volets ancrables](../Editeurs/2027001.md)**<br>Volets permettant de :<br><br>- afficher les différents éléments du code (liste des classes, des procédures locales ou globales, [volet "Code"](../Editeurs/2027009.md)),<br><br>- afficher les erreurs de compilation (erreurs présentes dans le code WLangage, [volet "Erreurs de compilation"](../Editeurs/2027031.md)),<br><br>- effectuer des recherches ([volet "Rechercher - Remplacer"](../Editeurs/2027013.md)),<br><br>- afficher le débogueur lors du test d'un élément ([volet "Débogueur"](../Editeurs/2027009.md)),<br><br>- ...<br><br><br> |

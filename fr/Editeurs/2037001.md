
## Connaître le compte qui exécute le protocole AWP
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Comment le faire sous IIS 5 ou 6 ?
<a name="comment_faire_sous_iis_5_6_ELTTEXTE000101"></a>
Pour connaître le compte Windows qui exécute le protocole AWP :

1. Lancez le Gestionnaire des services Internet (option "Démarrer .. Exécuter .. inetmgr.exe" sous Windows NT ou Windows 2000).

2. Affichez les propriétés du service "WWW".

3. Cliquez sur l'onglet "Sécurité de répertoires".

4. Affichez les méthodes d'authentification pour l'accès anonyme (bouton "Modifier").




Le compte Windows correspond au nom d'utilisateur utilisé pour ouvrir une session anonyme (bouton "Modifier").

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=iis5_compte_awp.gif)

## Comment le faire sous IIS 7 ?
<a name="comment_faire_sous_iis_7_ELTTEXTE000125"></a>
Pour connaître le compte Windows qui exécute le protocole AWP :

1. Lancez le Gestionnaire des services Internet (option "Démarrer .. Exécuter .. inetmgr.exe" sous Windows Vista).

2. Sélectionnez "Site Web par défaut".

3. Sélectionnez "Authentification" dans la liste qui s'affiche : ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=iis7_Authentification.gif)


4. Dans l'écran qui s'affiche, sélectionnez "Authentification anonyme".

5. Cliquez sur le bouton "Modifier" présent à droite.

6. Le nom de l'utilisateur anonyme s'affiche. Vous pouvez le modifier si nécessaire. Validez.


![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=IIS7_Authentification2.gif)

## Comment le faire sous Apache ?
<a name="comment_faire_sous_apache_ELTTEXTE000149"></a>
Pour connaître le compte Windows qui exécute le protocole AWP :

1. Lancez le panneau de configuration des services.

2. Affichez les propriétés du service "Apache".

3. Le compte Windows correspondant est affiché :

	- soit le compte système

	- soit un autre compte Windows.

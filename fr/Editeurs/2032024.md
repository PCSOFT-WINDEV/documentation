


## Documentation technique de la requête
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000139"></a>
L'éditeur de requêtes crée automatiquement une documentation technique pour chaque requête. Cette documentation technique contient :

- la représentation graphique de la requête.

- les informations générales de la requête (nom logique et physique de la requête, type de la requête, analyse associée à la requête, ...).

- les informations spécifiques aux résultats de la requête (rubriques résultat, paramètres de la requête, critères de tri, ...).

- les informations sur les caractéristiques de la requête (conditions de sélection, groupements, ...).

- le code SQL correspondant à la requête.




Le contenu de cette documentation technique est paramétrable, il peut contenir :

- soit toutes les informations sur la requête.

- soit une sélection d'informations.

## Comment le faire ?
<a name="comment_faire_ELTTEXTE000163"></a>


### Editer la documentation technique de la requête en cours sous l'éditeur
<a name="editer_documentation_technique_requete_cours_sous_editeur_ELTPARAGRAPHE000029"></a>Pour éditer la documentation technique de la requête en cours sous l'éditeur :

1. Sous le volet "Accueil", dans le groupe "Général", déroulez ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_imprimer.gif) et sélectionnez "Imprimer".

2. Spécifiez les informations contenues dans cette documentation technique.

3. Choisissez le type d'édition de la documentation technique.

4. Indiquez si cette documentation technique doit contenir : 

	- le vocabulaire de PC Soft, utilisé par défaut. 

	- un vocabulaire "générique" (par exemple, le terme "fichier de données" sera remplacé par "Table", "Rubrique" par "Colonne", "Champ" par "Contrôle", ...







Remarque : L'impression de la représentation de la requête peut être paramétrée selon :

- des zones d'impression.

- une mise en page spécifique. Pour définir cette mise en page, déroulez ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_imprimer.gif) et sélectionnez "Mise en page de la documentation".



<a name="NOTE2_2"></a>


### Spécifier les zones d'impression de la représentation graphique 
<a name="specifier_les_zones_impression_representation_graphique_ELTPARAGRAPHE000063"></a>Pour spécifier les zones d'impression de la représentation graphique :

1. Sous le volet "Affichage", dans le groupe "Aide à l'édition", cochez l'option "Zones d'impression". Des bordures représentant les zones imprimables dans la documentation technique apparaissent sous l'éditeur de requêtes.

2. Réduisez l'affichage de la représentation graphique : sous le volet "Affichage", dans le groupe "Aide à l'édition", modifiez la valeur du "Zoom" (ou Ctrl + Roulette de la souris) pour visualiser l'ensemble de la requête.

3. A l'aide de la souris :

	- déplacez ces bordures à l'endroit désiré en maintenant le bouton de la souris enfoncé (la couleur du curseur de la souris est noire).

	- choisissez le nombre de pages sur lesquelles la représentation graphique doit être imprimée (le curseur de la souris se transforme en double-flèche Nord-Ouest / Sud-Est).

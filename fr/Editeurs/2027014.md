## Présentation
<a name="presentation_ELTTEXTE000110"></a>
Le volet "Dictionnaire" permet de manipuler les dictionnaires WINDEV (fichiers ".DIC"). Il est possible de :

- créer un dictionnaire.

- ouvrir un ou plusieurs dictionnaires.

- afficher les éléments présents dans un dictionnaire.

- ajouter des éléments dans un dictionnaire.

- intégrer un élément d'un dictionnaire dans l'application en cours.



<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=VoletDictionnaire.gif)


Pour plus de détails sur les dictionnaires, consultez [Dictionnaire](../Editeurs/2023002.md).

## Afficher le volet "Dictionnaire"
<a name="afficher_volet_dictionnaire_ELTTEXTE000134"></a>
**Pour afficher le volet "Dictionnaire"**, sous le volet "Accueil", dans le groupe "Environnement", déroulez "Volets" et sélectionnez "Volets" puis "Dictionnaire". Les icônes représentant les volets actuellement affichés sont sur fond bleu.

Pour plus de détails sur les manipulations des volets, consultez [Manipuler les volets ancrables](../Editeurs/2027001.md).

## Présentation
<a name="presentation_ELTTEXTE000095"></a>
Le volet "Champs prédéfinis" propose plusieurs champs prédéfinis pouvant être intégrés dans la fenêtre ou la page en cours d'un simple "Drag and Drop".

## Afficher le volet "Champs pré-définis"
<a name="afficher_volet_champs_predefinis_ELTTEXTE000119"></a>
**Pour afficher le volet "Champs pré-définis"**, sous le volet "Accueil", dans le groupe "Environnement", déroulez "Volets" et sélectionnez "Volets" puis "Champs pré-définis". 

Pour plus de détails sur les manipulations des volets, consultez [Manipuler les volets ancrables](../Editeurs/2027001.md).

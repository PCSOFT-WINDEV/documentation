


## Eléments du projet 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000126"></a>
Le projet est composé de fenêtres, d'états, de requêtes, de fichiers d'aide, de collections de procédures, ...

Lors de la création d'un élément dans un projet, cet élément est automatiquement intégré à la liste des éléments du projet en cours. Les dépendances de cet élément (images, ...) sont également automatiquement intégrées à la liste des éléments du projet.

Les éléments présents dans la liste des éléments du projet seront automatiquement proposés pour être intégrés à la bibliothèque de l'application (si la bibliothèque est créée).

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000150"></a>
**Pour afficher la liste des éléments d'un projet**, sous le volet "Projet", dans le groupe "Projet", déroulez "Liste des éléments" et sélectionnez "Liste des éléments du projet".

Cette option permet de :

- **Ajouter des éléments appartenant à d'autres projets** : 
	Tout type de fichier peut être ajouté à la liste des éléments du projet : images, fichiers texte, ...
	Remarque : Les fichiers correspondants et leur dépendance feront partie du projet, mais les fichiers correspondants ne seront pas déplacés dans le répertoire de votre projet.

- **Supprimer des éléments de votre projet** :
	Remarque : Les fichiers correspondants et leur dépendance ne feront plus partie du projet, mais les fichiers correspondants ne sont pas supprimés physiquement.

- **Regénérer la liste des éléments de votre projet** :
	Cette option permet de reconstruire la liste des éléments du projet à partir des éléments présents dans le répertoire spécifié. Cette option est utile par exemple lors d'une exportation d'éléments WINDEV vers le répertoire d'un second projet. Pour prendre en compte les nouveaux éléments dans le second projet, il suffira de re-générer la liste des éléments.

- **Relocaliser les éléments** : 
	Cette option permet de modifier le répertoire du fichier physique correspondant à l'élément sélectionné.

## Notes
<a name="notes_ELTTEXTE000174"></a>


### Conseils : Ajouter des éléments à une bibliothèque
<a name="conseils_ajouter_des_elements_une_bibliotheque_ELTPARAGRAPHE000058"></a>Pour ajouter des éléments à une bibliothèque, il est conseillé d'ajouter ces éléments à la liste des éléments du projet. Ainsi, pour utiliser une image spécifique dans plusieurs fenêtres de votre projet par programmation, il suffit d'intégrer cette image dans la liste des éléments de votre projet (il n'est pas nécessaire de l'incorporer dans une fenêtre).
<a name="NOTE3_2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) **Inclure des éléments dans l'exécutable : catégorie "Autres" du volet "Explorateur de projet"**
Il est possible d'inclure des fichiers (fichiers de données HFSQL, images, sons, ...) directement dans l'exécutable généré. Il est ainsi possible d'utiliser des ressources qui ne doivent pas être visibles sur le disque de l'utilisateur final. 

Ces ressources doivent simplement être incluses dans le projet par l'intermédiaire du dossier "Autres" présent dans le volet "Explorateur de projet". 
Les fonctionnalités suivantes permettent de simplifier la manipulation du dossier "Autres" : 

- Le "Drag and Drop" des éléments à inclure dans le dossier "Autres" est possible depuis l'explorateur de fichiers Windows. 

- Un menu contextuel sur le menu "Autres" permet d'ajouter ou de supprimer des éléments. 

- Un dossier "Autres" est également disponible dans chaque composant interne, ce qui permet d'inclure des éléments dans les composants internes partagés. 

- La fonction de recherche dans le projet parcourt également le dossier "Autres" du projet.




## Créer des ancres
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000151"></a>
Les ancres permettent de définir des points spécifiques dans une page d'aide.

Il sera ensuite possible de se positionner directement sur ce point de la page d'aide à partir par exemple d'un lien.

Les ancres sont très utiles dans les pages d'aide relativement longues. Elles permettent au lecteur de se déplacer simplement et rapidement à l'intérieur de la page.

Une ancre peut être créée :

- sur un mot ou un groupe de mots.

- sur une image.




Pour visualiser simplement une ancre sous l'éditeur d'aide de WINDEV, le texte correspondant à l'ancre est surligné en vert pâle.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000175"></a>


### Créer une ancre
<a name="creer_une_ancre_ELTPARAGRAPHE000029"></a>Pour créer une ancre :

1. Affichez la page d'aide devant contenir l'ancre dans l'éditeur d'aide.

2. Sélectionnez le groupe de mots sur lequel l'ancre doit être créée.

3. Sous le volet "Modification", dans le groupe "Edition", cliquez sur "Ancre". 
	Remarque : Il est également possible de sélectionner l'option "Insérer une ancre" du menu contextuel.

4. Dans la fenêtre qui s'affiche, indiquez le nom de l'ancre. Ce nom permet d'identifier l'ancre lors de la création de liens vers cette ancre. Par défaut, le texte sélectionné est proposé.

5. Validez.



<a name="NOTE2_2"></a>


### Créer un lien vers une ancre
<a name="creer_lien_vers_une_ancre_ELTPARAGRAPHE000056"></a>Pour créer un lien vers une ancre :

1. Affichez la page d'aide devant contenir le lien dans l'éditeur d'aide.

2. Sélectionnez le groupe de mots sur lequel le lien doit être fait.

3. Sous le volet "Modification", dans le groupe "Edition", cliquez sur "Lien". 
	Remarque : Il est également possible de sélectionner l'option "Insérer un lien" du menu contextuel. La fenêtre des propriétés du lien s'affiche.

4. Sélectionnez les caractéristiques suivantes :

	- Hyperlien vers : "Lien vers une ancre". 

	- Page destination : sélectionnez la page et l'ancre voulue




5. Validez.



<a name="NOTE2_3"></a>


### Connaître le nom d'une ancre
<a name="connaitre_nom_une_ancre_ELTPARAGRAPHE000086"></a>Pour afficher le nom d'une ancre :

1. Positionnez le curseur de la souris sur le texte de l'ancre (partie surlignée en vert pâle).

2. Double-cliquez sur l'ancre ou sélectionnez l'option "Propriétés d'une ancre" dans le menu contextuel (clic droit). Il est possible de modifier le nom de l'ancre.



<a name="NOTE2_4"></a>


### Supprimer une ancre
<a name="supprimer_une_ancre_ELTPARAGRAPHE000096"></a>Pour supprimer une ancre :

1. Affichez la page d'aide contenant l'ancre dans l'éditeur d'aide. 

2. Positionnez le curseur de la souris sur le mot ou l'image correspondant à l'ancre.

3. Sélectionnez l'option "Supprimer une ancre" du menu contextuel. L'ancre est automatiquement supprimée.




**Attention** : Les liens portant sur cette ancre ne sont pas supprimés. Des erreurs peuvent survenir à la compilation si les liens ne sont pas modifiés.




## Gestion du temps
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000119"></a>
Pour chaque élément (fenêtres, états, ...) du projet, un cumul du temps passé est calculé :

- pour tous les participants au projet,

- pour chaque participant au projet.




Cette gestion du temps permet de réaliser des statistiques sur la durée de développement d'un projet, et de connaître rapidement les parties de développement les plus importantes.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000143"></a>


### Visualiser le tableau récapitulatif
<a name="visualiser_tableau_recapitulatif_ELTPARAGRAPHE000023"></a>Pour visualiser le tableau récapitulatif, sous le volet "Projet", dans le groupe "Autres actions", déroulez "Gestion du temps" et sélectionnez "Temps passé sur les éléments du projet".
<a name="NOTE2_2"></a>


### Activer ou non la gestion du temps
<a name="activer_non_gestion_temps_ELTPARAGRAPHE000042"></a>Pour activer / désactiver cette gestion du temps : 

- Affichez la fenêtre de description du projet : sous le volet "Projet", dans le groupe "Projet", cliquez sur "Description". 

- Affichez l'onglet "Avancé". 

- Cochez / décochez l'option "Activer le comptage du temps passé sur les éléments"

- Validez.



<a name="NOTE2_3"></a>


### Calcul des durées
<a name="calcul_des_durees_ELTPARAGRAPHE000066"></a>Si plusieurs éléments sont ouverts, seul l'élément en cours est pris en compte dans le calcul de la durée.

Si l'éditeur en cours est inactif (utilisation d'une autre application par exemple), le calcul de la durée se stoppe automatiquement.

Si un élément reste ouvert moins de 60 secondes ou reste actif moins de 10 secondes, ces durées ne sont pas prises en comptes.

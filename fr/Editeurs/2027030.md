<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000136"></a>
Le volet "Débogueur" permet d'afficher les options du débogueur lors du débogage du projet ou d'un élément du projet (option "Débogueur").


![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Partie2_PileDesAppels.gif&type=thumb)


Les différentes options de ce volet sont les suivantes : 

- ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Debug_IconeContinuerTest.gif)
 : Continuer (F5).

- ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Partie2_PileDesAppels%202.gif)
 : Terminer le test (Maj + F5).

- ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Partie2_PileDesAppels%205.gif)
 : Exécuter pas à pas en rentrant dans les fonctions (F7).

- ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Partie2_PileDesAppels%203.gif)
 : Exécuter pas à pas en sautant les fonctions (F8).

- ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Partie2_PileDesAppels%206.gif)
 : Exécuter jusqu'à la fin de la fonction (Alt + Fin).

- ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Partie2_PileDesAppels%208.gif)
 : Continuer jusqu'au curseur. 

- ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Partie2_PileDesAppels%204.gif)
 : Point d'arrêt (Ctrl + B).

- ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Partie2_PileDesAppels%207.gif)
 : Enlever tous les points d'arrêt.

- ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Partie2_PileDesAppels%209.gif)
 : Lister les points d'arrêt. 

- ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Partie2_PileDesAppels%2010.gif)
 : Visualiser l'état des sources de données. 

- ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Partie2_PileDesAppels%2013.gif)
 : Visualiser les éléments du système. 

- ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Partie2_PileDesAppels%2012.gif)
 : Afficher les informations du dump de débogage. 

- ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Partie2_PileDesAppels%2011.gif)
 : Voir les expressions automatiques. 

- ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Debug_IconeVariableLocale.gif)
 : Voir les variables locales. 

- ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Debug_IconeVariableGlobale.gif)
 : Voir les variables globales. 

- ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Partie2_PileDesAppels%2015.gif)
 : Voir les membres. 

- ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Debug_IconeCpteRenduHF.gif)
 : Voir le compte-rendu HFSQL. 

- ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Debug_IconeCpteRenduWL.gif)
 : Voir le compte-rendu WLangage.

- ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Partie2_PileDesAppels%201.gif)
 : Activer l'Analyseur Temps Réel. 

- ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Partie2_PileDesAppels%2014.gif)
 : Afficher la fenêtre de trace du débogueur.

- ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Partie2_PileDesAppels%2016.gif)
 : Affichage hexadécimal. 




Pour plus de détails, consultez le [Débogueur](../Editeurs/2019001.md).

## Afficher le volet "Débogueur"
<a name="afficher_volet_debogueur_ELTTEXTE000160"></a>
**Pour afficher le volet "Débogueur"**, sous le volet "Accueil", dans le groupe "Environnement", déroulez "Volets" et sélectionnez "Volets" puis "Débogueur".

Remarque : Ce volet est automatiquement affiché lors du débogage d'une fenêtre, d'une page ou du projet.

Pour plus de détails sur les manipulations des volets, consultez [Manipuler les volets ancrables](../Editeurs/2027001.md).

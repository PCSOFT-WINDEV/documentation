


## Créer les pages d'aide
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000218"></a>
Une aide est constituée de plusieurs pages. Les pages sont les éléments de base d'une aide. Quand l'utilisateur consulte l'aide, il lit le contenu d'une page d'aide.

Plusieurs types de pages d'aide peuvent être mises en place dans un fichier d'aide, voici les plus courants :

- **des pages d'aide contextuelle** : le contenu de la page d'aide donne des indications sur l'utilisation d'une fenêtre spécifique. Ce type de page d'aide est par exemple lancé à partir de l'icône d'aide (bouton "?") présent sur la fenêtre.

- **des pages d'aide "Comment le faire"** : ces pages d'aide décrivent pas à pas les différentes étapes pour réaliser une opération.

- **des pages d'aide "Conceptuelle"** : ces pages d'aide présentent un concept, une idée, définissent les termes utilisés.




Les pages d'aides sont reliées entre elles par des [liens](../Editeurs/2010023.md). Les liens permettent aux utilisateurs de se déplacer et de parcourir le système d'aide.

**Remarque** : Pour générer automatiquement les pages d'aide correspondant aux différents éléments du projet (fenêtres et états), sous le volet "Modification", dans le groupe "Système d'aide", cliquez sur "Générer les pages d'aide du projet".

## Comment le faire ?
<a name="comment_faire_ELTTEXTE000242"></a>


### Créer une page d'aide vierge
<a name="creer_une_page_aide_vierge_ELTPARAGRAPHE000043"></a>Pour créer une page d'aide vierge : 

- 1ère méthode : 

	- Sélectionnez une page dans le sommaire. 

	- Sous le volet "Modification", dans le groupe "Table des matières", déroulez "Nouveau" et sélectionnez "Nouvelle page vierge".

	- Une nouvelle page vierge est automatiquement créée et peut être modifiée.




- 2ème méthode : Dans le menu contextuel du sommaire, sélectionnez l'option "Nouveau .. Nouvelle page vierge". Une nouvelle page vierge est automatiquement créée et peut être modifiée.

- 3ème méthode : 

	- Sous le volet "Modification", dans le groupe "Table des matières", déroulez "Nouveau" et sélectionnez "Nouvelle page".

	- Dans l'assistant qui se lance, sélectionnez l'option "Créer une page vierge". 

	- Précisez les caractéristiques de la page et validez.




- 4ème méthode : Dans le menu contextuel du sommaire, sélectionnez l'option "Nouveau .. Nouvelle page". Dans l'assistant qui se lance, sélectionnez l'option "Créer une page vierge". Précisez les caractéristiques de la page et validez.



<a name="NOTE2_2"></a>


### Créer une page d'aide à partir d'un document existant
<a name="creer_une_page_aide_partir_document_existant_ELTPARAGRAPHE000087"></a>La création d'une page d'aide à partir d'un document existant est réalisée grâce à un assistant. Il suffit de sélectionner dans cet assistant :

- le nom et le chemin du fichier à importer (fichier au format txt, htm ou RTF). 

- le nom de la page d'aide à créer.




**Attention** : Si le document importé contient plusieurs pages, toutes les pages seront importées dans une seule page d'aide.

Pour lancer l'assistant d'importation, utilisez une des méthodes suivantes :

- sous le volet "Modification", dans le groupe "Table des matières", déroulez "Nouveau" et sélectionnez "Nouvelle page". Dans l'assistant qui se lance, sélectionnez l'option "Importer un document existant".

- sélectionnez l'option "Nouveau .. Nouvelle page" dans le menu contextuel du sommaire. Dans l'assistant qui se lance, sélectionnez l'option "Importer un document existant".



<a name="NOTE2_3"></a>


### Supprimer une page d'aide
<a name="supprimer_une_page_aide_ELTPARAGRAPHE000116"></a>Pour supprimer une page d'aide : 

1. Sélectionnez la page d'aide dans le sommaire.

2. Sélectionnez l'option "Supprime la page" dans le menu contextuel du sommaire (ou utilisez la touche Suppr).

## Modifier les pages d'aide
<a name="modifier_les_pages_aide_ELTTEXTE000278"></a>


### Parties d'une page d'aide
<a name="parties_une_page_aide_ELTPARAGRAPHE000128"></a>Une page d'aide est constituée de différentes parties :<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Editeur-Aide_page.gif)
Toutes ces parties peuvent être modifiées.
<a name="NOTE3_2"></a>


### Dans une page d'aide, il est possible de : 
<a name="dans_une_page_aide_est_possible_ELTPARAGRAPHE000137"></a>

- modifier le texte présent dans l'entête ou dans la page. Il suffit de saisir directement le texte sous l'éditeur d'aide. 
	Remarque : le correcteur d'orthographe est actif dans la page d'aide. 

- supprimer l'entête de la page (onglet "Apparence" des [propriétés de la page](../Editeurs/2010024.md)).

- modifier le titre de la page affiché dans le sommaire (modification du titre de la page dans les [propriétés de la page](../Editeurs/2010024.md)).

- configurer le mode d'affichage de la page, en modifiant par exemple le modèle de fenêtre associé à la page. Pour plus de détails, consultez [Définir un modèle de fenêtre d'aide](../Editeurs/2010006.md).

- ajouter des images (associées à des liens ou non). Pour plus de détails, consultez [Insérer une image dans une page d'aide](../Editeurs/2010026.md).

- ajouter des vidéos. Pour plus de détails, consultez [Insérer une vidéo dans une page d'aide](../Editeurs/2010026.md).

- ajouter des liens. Pour plus de détails, consultez [Créer les liens entre les pages d'aide](../Editeurs/2010023.md).

- associer des mots-clés d'index à la page d'aide. Pour plus de détails, consultez [Associer des mots-clés d'index à une page d'aide](../Editeurs/2010009.md).

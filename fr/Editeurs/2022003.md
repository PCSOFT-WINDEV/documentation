


## Positionnement automatique
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000109"></a>
Lors du déplacement d'un champ dans une fenêtre, une page ou un état, le correcteur d'interface en temps réel (appelé aussi positionnement automatique) affiche automatiquement des guides. Ces guides sont magnétisés et permettent d'aligner le champ déplacé avec les champs présents dans la fenêtre, la page ou l'état.
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CorrecteurIHM.gif)


<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000133"></a>


### Utiliser le positionnement automatique dans une fenêtre ou une page
<a name="utiliser_positionnement_automatique_dans_une_fenetre_une_page_ELTPARAGRAPHE000019"></a>Pour utiliser le positionnement automatique dans une fenêtre ou une page : 

1. Sous le volet "Affichage", cliquez sur le bouton de regroupement du groupe "Options" : ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_regroup.gif)


2. Dans l'onglet " Magnétisme", dans la zone "Positionnement automatique", sélectionnez les options "Actif" et "Afficher les règles". 




Pour ne pas utiliser le positionnement automatique dans une fenêtre, il suffit de décocher ces options.
<a name="NOTE2_2"></a>


### Utiliser le positionnement automatique dans un état
<a name="utiliser_positionnement_automatique_dans_etat_ELTPARAGRAPHE000044"></a>Pour utiliser le positionnement automatique dans un état : 
Sous le volet "Affichage", dans le groupe "Aide à l'édition", cochez l'option "Positionnement auto". 

- Si cette option est cochée, le positionnement automatique est activé. 

- Si cette option n'est pas cochée, le positionnement automatique n'est pas activé.

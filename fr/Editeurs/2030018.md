


## Dupliquer un projet 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000096"></a>
La duplication d'un projet permet de disposer de deux projets identiques de même nom mais présents dans des répertoires différents.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000120"></a>
Pour dupliquer le projet en cours :

1. Sous le volet "Accueil", dans le groupe "Général", déroulez "Enregistrer" et sélectionnez "Exporter .. Vers un répertoire".

2. Choisissez les fichiers à exporter. Pour dupliquer le projet, sélectionnez l'ensemble des fichiers. 
	Attention : Cette manipulation ne permet pas de dupliquer les fichiers de l'analyse et les fichiers de sauvegarde.

3. Spécifiez le répertoire dans lequel les fichiers doivent être exportés.

4. Validez la duplication du projet.


## La structure des modèles du RAD (conservé par compatibilité)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000114"></a>
**Attention** : en version 12, le RAD a été entièrement refait. La méthode exposée dans cette page est conservée par compatibilité. Pour plus de détails, consultez [Nouveaux Patterns RAD](../Editeurs/2031020.md). 

Les fichiers modèles du RAD (fichier MDL) permettent à la fois de décrire :

- Les différentes options affichées ou grisées dans l'assistant de création de fenêtres.

- Les fichiers nécessaires au fonctionnement de la fenêtre (par exemple les classes RAD à copier dans le répertoire de génération de la fenêtre).

- Les groupes de champs qui seront présents dans la fenêtre.

- Le code de la fenêtre et le code des procédures locales ou globales.

- Les caractéristiques des différentes zones où les champs seront affichés. Chaque zone de champs est associée à une zone de boutons, et une zone de boutons de parcours. Pour chaque zone de boutons, le modèle contient la description des boutons (taille, nom, libellé, ..) et leur code.




Toutes ces informations sont présentées dans le script selon une structure de modèle bien définie.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Structure des fichiers
<a name="structure_des_fichiers_ELTTEXTE000138"></a>
La structure des fichiers MDL peut être divisée en deux parties :

1. **Description du modèle**

	- Caractéristiques du modèle affichées dans l'assistant

	- Caractéristiques du modèle nécessaires en programmation (classes, groupes de champs, ..)

	- Code de la fenêtre et code des procédures locales ou globales




2. **Description de la fenêtre**

	- Caractéristiques des zones de champs (répété pour chaque zone de champs)

	- Caractéristiques des zones de boutons (répété pour chaque zone de boutons

	- Caractéristiques d'un bouton (répété pour chaque bouton)

	- Code du bouton

## Fichiers MDE
<a name="fichiers_mde_ELTTEXTE000162"></a>
Les fichiers MDE sont des fichiers MDL permettant de partager des scripts communs entre plusieurs fichiers MDL. Le contenu du fichier MDE est identique au fichier MDL. Cependant, le code n'est présent qu'une seule fois dans le fichier MDE. Les fichiers MDL utilisant ce code n'ont qu'un simple appel à réaliser.


<a name="NOTE3_2"></a>


### Appel du fichier MDE depuis un fichier MDL
<a name="appel_fichier_mde_depuis_fichier_mdl_ELTPARAGRAPHE000058"></a>La syntaxe à utiliser est la suivante :

```txt
%extern <Nom du fichier .MDE>
```

Exemple :

```txt
%extern Declare1fichier.mde
```


**Remarque** : Dans le fichier MDE, il est conseillé de terminer le fichier par un Retour chariot (une ligne vide) pour éviter tout problème lors de l'insertion du code dans le fichier MDL.

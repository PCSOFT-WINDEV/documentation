


## Options de compilation
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000107"></a>
Les options de compilation permettent de définir les paramètres pris en compte lors de la génération du système d'aide. Différents types de paramètres sont définis :

- Nom du fichier d'aide à générer.

- Compilateur à utiliser.

- Paramètres de compilation correspondant au compilateur sélectionné.




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Nom du fichier d'aide généré
<a name="nom_fichier_aide_genere_ELTTEXTE000131"></a>
Le nom du fichier d'aide généré par WINDEV peut être différent du nom du système d'aide. Par exemple, dans le cas d'une application multilingue, différents fichiers d'aide seront créés et générés à partir du même système d'aide WINDEV (fichier WDH).

Dans ce cas, le nom d'un fichier d'aide peut être constitué de plusieurs éléments :

- **le nom de base du fichier d'aide**. Par défaut, le nom proposé correspond au nom spécifié dans les propriétés du système d'aide : sous le volet "Modification", dans le groupe "Système d'aide", cliquez sur "Description". Il peut être modifié dans les paramètres de génération. 

- **le suffixe des fichiers générés**. Ce suffixe peut correspondre à : 

	- Aucun suffixe : le nom du fichier généré correspond au nom de base du fichier d'aide. 

	- Le numéro de la langue des fichiers. Ce numéro correspond à celui utilisé par la fonction [Nation](../WDLang1/3054014.md).

	- L'identifiant texte de la langue (FR pour Français).

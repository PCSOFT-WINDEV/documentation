## Choix du support
<a name="choix_support_ELTTEXTE000118"></a>
Lors de la création du programme d'installation à l'aide de l'assistant, il est possible de choisir les options concernant le support d'installation.

En effet, le programme d'installation est généré dans un répertoire en vue d'être copié sur différents supports.

Par défaut, le programme d'installation est généré dans le sous-répertoire "Install" du répertoire de votre application.

Les différents supports sont les suivants :

- ***Répertoires multiples***, découpés à la taille des disquettes (non disponible pour les installations compactes)
	Les fichiers du programme d'installation sont regroupés dans plusieurs sous-répertoires. La taille des fichiers contenus dans chaque sous-répertoire ne dépasse pas la taille d'une disquette (1,44 Mo par défaut). Il est ensuite possible de copier le contenu de chaque sous-répertoire sur une disquette.

- ***Répertoire unique***
	Le programme d'installation est créé dans un répertoire unique.

- ***Exécutable auto-extractible***
	Le programme d'installation est créé sous forme d'un exécutable auto-extractible. Lors du lancement de cet exécutable sur le poste utilisateurs, le programme d'installation est automatiquement décompressé et lancé.
	Cette solution est conseillée si le programme d'installation est transmis par Internet.

- ***Support avec autorun***
	Le programme d'installation est créé dans un répertoire unique en vue d'être copié sur un CD-Rom.




Remarque : Si l'option **Vider le répertoire avant de générer l'installation** est cochée, le répertoire de génération de l'installation sera vidé avant de générer le nouveau programme d'installation. Cette option permet d'éviter de conserver des fichiers inutiles dans le répertoire de génération du programme d'installation.

## Sécurité
<a name="securite_ELTTEXTE000142"></a>
Des options permettant de sécuriser l'installation sont également proposées : 

- **Protéger l'installation par un mot de passe** : Il est possible de protéger l'installation par un mot de passe (non disponible pour les installations compactes). Dans ce cas, il est nécessaire d'indiquer le mot de passe à utiliser. Ce mot de passe sera demandé lors de l'exécution du programme d'installation.

- **Signer l'installation avec le certificat** : Pour garantir l'intégrité du pack d'installation, il est possible de signer l'installation. Il suffit de sélectionner le certificat à utiliser. L'assistant propose automatiquement les certificats disponibles sur le poste en cours, dans le magasin personnel des certificats de Windows.

## Paramètres avancés
<a name="parametres_avances_ELTTEXTE000166"></a>
Les paramètres avancés permettent de modifier les différents paramètres (répertoire utilisé, taille d'une disquette, ...) du support choisi.

Pour vous aider à choisir le support adapté à votre mode d'installation, consultez le [Tableau récapitulatif sur l'installation d'une application WINDEV](../Editeurs/2028005.md).

Le bouton "Options avancées de l'installeur" permet de définir : 

- les options de gestion du framework de l'installeur (avec la personnalisation de l'installeur). 

- les options de l'UAC de l'installeur. 




Pour plus de détails, consultez [Options avancées de l'installeur](../Editeurs/2028108.md).

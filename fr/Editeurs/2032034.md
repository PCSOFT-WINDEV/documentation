


## Exemple de création d'une requête de sélection
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000143"></a>
Cette page d'aide présente la création de deux requêtes de sélection :

- une requête simple.

- une requête de type avancé.




Ces deux requêtes sont créées à partir du projet "WD Requêtes" (livré en standard avec WINDEV).

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Exemple 1 : Requête simple
<a name="exemple_1_requete_simple_ELTTEXTE000167"></a>


### Requête à créer
<a name="requete_creer_ELTPARAGRAPHE000023"></a>Sélection des caractéristiques des clients (numéro, nom et téléphone) habitant à Paris.
<a name="NOTE2_2"></a>


### Comment créer cette requête ? 
<a name="comment_creer_cette_requete_ELTPARAGRAPHE000030"></a>Les étapes sont les suivantes : 

1. Cliquez sur ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_nouveau.gif)
 parmi les boutons d'accès rapide. La fenêtre de création d'un nouvel élément s'affiche : cliquez sur "Requête". L'assistant de création d'une requête se lance.

2. Sélectionnez l'option "Sélection (SELECT)" et validez. La fenêtre de description de la requête s'affiche.

3. Saisissez le nom et le libellé de la requête.

4. Dans la partie gauche, double-cliquez sur les rubriques "NumClient", "NomClient", "Ville" et "Téléphone" du fichier de données CLIENT.

5. Dans la partie centrale, sélectionnez la rubrique "CLIENT.Ville".

6. Cliquez sur le bouton "Conditions de sélection" et sélectionnez l'option "Nouvelle condition". L'assistant de création d'une condition de sélection s'affiche.

7. Choisissez le critère de sélection "commence par". Sélectionnez "la valeur" et saisissez "Paris" dans le champ "Valeur". Validez.

8. Cliquez sur le bouton "Afficher/Renommer".

9. Répondez "Non" à la question "Afficher cette rubrique" et validez. 
	Remarque : Il n'est pas nécessaire d'afficher les valeurs de la rubrique "Ville" dans le résultat de la requête : tous les clients sélectionnés par la requête habitent Paris.

10. Validez la fenêtre de description de la requête. La représentation graphique de la requête s'affiche automatiquement sous l'éditeur de requêtes.

11. Sauvegardez la requête (![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_Enregistrer.gif)
).

## Exemple 2 : Requête de type avancé
<a name="exemple_2_requete_type_avance_ELTTEXTE000197"></a>


### Requête à créer
<a name="requete_creer_ELTPARAGRAPHE000057"></a>Affichage du nombre de commandes passées par chaque client habitant à Paris entre le 01/01/2014 et le 31/01/2014.

Les points importants de cette requête sont :

- la définition d'une condition de sélection sur la ville des clients.

- la définition d'une condition de sélection sur la date des commandes.

- le comptage des commandes passées par chaque client.



<a name="NOTE3_2"></a>


### Comment créer cette requête ?
<a name="comment_creer_cette_requete_ELTPARAGRAPHE000070"></a>Les étapes sont les suivantes : 

1. Cliquez sur ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_nouveau.gif)
 parmi les boutons d'accès rapide. La fenêtre de création d'un nouvel élément s'affiche : cliquez sur "Requête". L'assistant de création d'une requête se lance.

2. Sélectionnez l'option "Sélection (SELECT)" et validez. La fenêtre de description de la requête s'affiche.

3. Saisissez le nom et le libellé de la requête.

4. Dans la partie gauche, double-cliquez sur les rubriques "NumClient", "NomClient" et "Ville" du fichier de données CLIENT et sur la rubrique "DateCommande" du fichier de données COMMANDE.

5. Dans la partie centrale, sélectionnez la rubrique "CLIENT.Ville".

6. Cliquez sur le bouton "Conditions de sélection" et sélectionnez l'option "Nouvelle condition". L'assistant de création d'une condition de sélection s'affiche.

7. Choisissez le critère de sélection "commence par". Sélectionnez "La valeur" et saisissez "Paris" dans le champ "Valeur". Validez.

8. Cliquez sur le bouton "Afficher/Renommer".

9. Répondez "Non" à la question "Afficher cette rubrique" et validez. 
	Remarque : Il n'est pas nécessaire d'afficher les valeurs de la rubrique "Ville" dans le résultat de la requête : tous les clients sélectionnés par la requête habitent Paris.

10. Cliquez sur le bouton "Compter" et sélectionnez l'option "Sur une nouvelle rubrique". La fenêtre de sélection d'une rubrique s'affiche.

11. Affichez l'onglet "Autres rubriques", sélectionnez la rubrique "NumCommande" et validez la fenêtre (bouton "Sélectionner"). La fenêtre de description d'un comptage s'affiche.

12. Validez la fenêtre de description du comptage.

13. Dans la partie centrale, sélectionnez la rubrique "COMMANDE.DateCommande".

14. Cliquez sur le bouton "Entre deux dates". La fenêtre de description d'une condition s'affiche.

15. Saisissez la valeur "01/01/2022" dans le premier champ "Valeur" et la valeur "31/01/2022" dans le deuxième champ "Valeur". Validez.

16. Validez la fenêtre de description de la requête. La représentation graphique de la requête s'affiche automatiquement sous l'éditeur de requêtes.

17. Sauvegardez la requête (![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_Enregistrer.gif)
).

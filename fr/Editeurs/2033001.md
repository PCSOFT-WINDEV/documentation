## Aide en ligne
<a name="aide_ligne_ELTTEXTE000195"></a>


### Présentation
<a name="presentation_ELTPARAGRAPHE000025"></a>L'aide en ligne est commune à tous les produits : WINDEV, WEBDEV et WINDEV Mobile. 

Par défaut, l'aide proposée est **l'aide locale** (utilisant les fichiers installés sur le poste de développement). Elle est affichée dans un browser d'aide spécifique. 

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=P0_Aide_Browser.jpg&type=thumb)


A tout moment, si vous possédez un accès Internet valide, il est possible d'afficher la page d'aide actuellement affichée dans le "Browser d'aide" dans sa version Internet (lien "Voir la page correspondante sur Internet"). 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=P0_Aide_HTML.jpg&type=thumb)


L'aide en ligne Internet est disponible depuis n'importe quel poste disposant d'un accès Internet, sans que le produit soit forcément installé. Cette aide est mise à jour régulièrement. 

Chaque internaute peut ajouter des commentaires sur les pages de documentation : explications personnelles, exemples, liens, ...


### Comment afficher l'aide
<a name="comment_afficher_aide_ELTPARAGRAPHE000040"></a>L'aide est accessible depuis l'éditeur : sous le volet "Accueil", dans le groupe "Aide en ligne", déroulez "Aide". Les options suivantes sont alors disponibles : 

- "Index de l'aide". Cette option permet d'accéder aux pages d'aide grâce aux mots-clés qui ont été définis sur ces pages. 

- "Sommaire". Cette option permet d'accéder aux informations classées par thèmes. Chaque thème peut être "déroulé".

- "Rechercher dans l'aide...". Cette option permet de réaliser une recherche sur toutes les pages d'aides accessibles depuis ce fichier. Il suffit de saisir l'expression recherchée. Cette expression sera recherchée dans toutes les pages d'aide. Toutes les pages contenant le(s) mot(s) indiqué(s) sont listées et peuvent être affichées.




De plus, à tout moment, utilisez directement la touche F1 ou le bouton d'aide ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=BTN_Aide.Gif)
 pour obtenir l'aide correspondant à la fenêtre en cours ou à la fonction sélectionnée (par exemple, F1 permet d'afficher l'aide sur la fonction en cours de saisie sous l'éditeur de code).


### Lancer l'aide en ligne Internet depuis le produit
<a name="lancer_aide_ligne_internet_depuis_produit_ELTPARAGRAPHE000065"></a>Pour lancer directement l'aide en ligne Internet depuis le produit : 

1. Sous le volet "Accueil", dans le groupe "Environnement", déroulez "Options" et sélectionnez "Options générales de XXX".

2. Dans l'onglet "Aide", sélectionnez le mode d'accès à la base d'aide : "Utiliser l'aide en ligne" ainsi que la langue préférée.

3. Validez. 




**Remarque** : Pour repasser sur l'aide locale, il suffit d'afficher le même écran et de sélectionner l'option "Utiliser l'aide locale".

## Guide
<a name="guide_ELTTEXTE000231"></a>
Le guide permet à tout moment d'obtenir l'aide, les assistants ou les exemples disponibles dans WINDEV, WEBDEV ou WINDEV Mobile. 

**Pour utiliser le guide** :

1. Sous le volet "Accueil", dans le groupe "Aide en ligne", déroulez "Aide" et sélectionnez "Consulter le guide". La fenêtre de recherche dans le guide apparaît.

2. Saisissez le mot-clé de votre recherche. Automatiquement, les mots-clés similaires et les aides disponibles sont affichés.

3. Choisissez le document voulu et cliquez sur le bouton "Ouvrir" ou "Importer" selon le type de document.
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Guide.gif&type=thumb)



Pour plus de détails sur le guide, consultez [Le guide](../Editeurs/2033003.md).


## Qu'est-ce qu'une application ? 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000073"></a>
Une **application** est un ensemble d'objets (fenêtres, champs, ...) organisés dans un but précis (par exemple suivre les ventes des produits d'une entreprise). L'utilisateur utilise une application en lançant simplement un fichier "Exécutable" (par exemple "SuiviVente.exe").

WINDEV permet de créer une application grâce au **projet**.

Un projet est un ensemble logique d'objets et de traitements dont le but est de réaliser un objectif donné. Le projet gère :

- les relations des objets entre eux.

- les relations des objets et des fichiers de données (décrits dans l'analyse liée au projet).




Un projet ne peut être relié qu'à une seule analyse.

Une même analyse peut être associée à plusieurs projets.

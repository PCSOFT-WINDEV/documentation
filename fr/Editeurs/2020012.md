## Présentation
<a name="presentation_ELTTEXTE000146"></a>
WINDEV, WEBDEV et WINDEV Mobile proposent différents modes d'utilisation de l'environnement de développement.

Lors du premier lancement, les choix suivants sont proposés :

- **Récupérer la configuration de l'environnement d'une version précédente**. Cette option permet de retrouver rapidement tous vos paramétrages effectués dans une version précédente installée sur votre poste.

- **Environnement simplifié**. Cette option est conseillée pour découvrir WINDEV, WEBDEV ou WINDEV Mobile. Certaines fonctionnalités avancées n'apparaissent pas dans l'environnement (GDS, Centres de Contrôle, Règles Métier, ...). Ces fonctionnalités peuvent être réactivées à tout moment si nécessaire.

- **Environnement complet**. Cette option permet d'utiliser un environnement exhaustif. Certaines options peu utilisées des versions antérieures sont remplacées par des nouveautés.




Vous pouvez également choisir la configuration écran :

- ***mono-écran*** : les différents volets seront affichés sur le même écran que l'environnement de développement.

- ***multi-écrans*** : l'environnement de développement sera affiché sur un écran et les différents volets seront regroupés sur le second écran.

- ***écran Wide ou écran large*** : l'espace libre à droite de l'écran est optimisé pour libérer le bas de l'éditeur.  




A tout moment, il est possible de changer d'environnement ou de personnaliser l'environnement choisi.

## Modifier l'environnement de développement
<a name="modifier_environnement_developpement_ELTTEXTE000170"></a>
Lors de l'utilisation de WINDEV, WEBDEV, WINDEV Mobile, vous connaissez à tout moment le mode d'environnement sélectionné : il est affiché sous le logo de la version. Par exemple :

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Env_mode_dev.gif)


Pour changer d'environnement :

- soit cliquez sur le mode en cours.

- soit sélectionnez l'option "Outils .. Options .. Options de l'environnement".




La fenêtre de paramétrage de l'environnement apparaît.



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Options de l'environnement
<a name="options_environnement_ELTTEXTE000194"></a>
La fenêtre de paramétrage de l'environnement permet :

- sur l'onglet "Environnement", de définir les différentes fonctionnalités disponibles.

- sur l'onglet "Volets", de choisir les différentes fenêtres dockables à afficher.




A tout moment le bouton "Relancer l'assistant de configuration de l'environnement" permet de relancer l'assistant de configuration utilisé lors du premier lancement du produit.




### Couleurs des barres d'outils
<a name="couleurs_des_barres_outils_ELTPARAGRAPHE000063"></a>La couleur des barres d'outils peut être paramétrée. Pour cela, cliquez sur la petite icône de couleur située en haut à droite de la fenêtre de l'environnement de développement (en dessous du logo de la version). Une fenêtre popup s'ouvre alors et vous permet de configurer la luminosité du fond des barres d'outils ainsi que la couleur dominante.
<a name="NOTE3_2"></a>


### Onglet "Environnement"
<a name="onglet_environnement_ELTPARAGRAPHE000070"></a>Cochez les fonctionnalités à activer et décochez les fonctionnalités à désactiver. 


<a name="NOTE3_3"></a>


### Onglet "Volets"
<a name="onglet_volets_ELTPARAGRAPHE000078"></a>

- **Fenêtres dockables** : Sélectionnez les cases à cocher correspondant aux éléments à afficher dans l'environnement : volets, barre des documents ouverts, [modifieur](../Editeurs/2027006.md), [Aperçu Position](../Editeurs/2027002.md), Centres de Contrôle, ... Pour plus de détails sur ces éléments et leur manipulation, consultez [Manipuler les volets](../Editeurs/2027001.md).

- **Divers** : Permet d'afficher ou non la **barre d'onglets** et le mode d'activation du [FishEye](../Editeurs/9000067.md) :

	- **Jamais** : le mode "FishEye" ne sera jamais activable.

	- **Lorsque le zoom est entre 25% et 75%** : le mode "FishEye" sera activable en fonction de la valeur du zoom.

	- **Lorsque le zoom est inférieur à 100%** : le mode "FishEye" sera activable en fonction de la valeur du zoom.

	- **Lorsque Ctrl et Maj sont enfoncés** : le mode "FishEye" sera activable en fonction d'une combinaison de touches.




## Rétro-modélisation des héritages
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000139"></a>
Vos fenêtres, vos pages utilisent des modèles de fenêtres, des modèles de pages, des modèles de champs, et vous voulez connaître les interdépendances entre tous ces modèles ? Il suffit d'effectuer une rétro-modélisation des héritages.

Une modélisation des différents éléments utilisés est automatiquement créée sous l'éditeur de modélisation.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000163"></a>


### Effectuer une rétro-modélisation des modèles
<a name="effectuer_une_retromodelisation_des_modeles_ELTPARAGRAPHE000020"></a>Pour effectuer une rétro-modélisation des modèles :

1. Ouvrez sous l'éditeur l'élément voulu (page ou fenêtre).

2. Sous le volet "Fenêtre", dans le groupe "Modèles", déroulez "Modèles" et sélectionnez "Rétro-Modélisation des modèles". L'assistant de configuration de la rétro-modélisation se lance.

3. Répondez aux différentes questions de l'assistant :

	- Type de rétro-modélisation : Graphe des héritages.

	- Nom de la modélisation qui sera générée et affichée.

	- Affichage de la grille dans la modélisation générée.

	- groupement ou non des éléments dans la modélisation générée.




4. Validez. La rétro-modélisation des différents modèles utilisés est automatiquement générée et affichée.

### Changer le niveau de détails dans la modélisation
<a name="changer_niveau_details_dans_modelisation_ELTPARAGRAPHE000058"></a>Si vous avez choisi de générer un graphe avec regroupement par fonctionnalité, il est possible de changer le niveau de détails directement dans la modélisation.

Pour augmenter le niveau de détails : 

- d'un modèle : double-cliquez sur l'icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Retro_AugmenteDetails.gif)
 affiché dans la modélisation.

- de tous les modèles du graphe : cliquez sur l'icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Retro_AugmenteDetailsTout.gif)
 de la barre d'outils.




Pour diminuer le niveau de détails :

- d'un modèle : double-cliquez sur l'icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Retro_DiminueDetails.gif)
 du traitement.

- de tous les modèles du graphe : cliquez sur l'icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Retro_DiminueDetailsTout.gif)
 de la barre d'outils.




## Assistant d'installation : Les étapes pour créer l'installation d'une application WINDEV
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Lancement
<a name="lancement_ELTTEXTE000210"></a>
L'assistant de création du programme d'installation permet de définir l'ensemble des paramètres de votre programme d'installation.

Pour lancer cet assistant :

- soit sous le volet "Projet", dans le groupe "Génération", déroulez "Procédure d'installation" et sélectionnez "Créer la procédure d'installation".

- soit lancez l'éditeur d'installation WDInst : sous le volet "Outils", dans le groupe "Utilitaires", cliquez sur "WDInst". Dans l'éditeur de WDInst, sélectionnez l'option "Fichier .. Nouveau".






<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Les différentes étapes
<a name="les_differentes_etapes_ELTTEXTE000234"></a>
Les différentes étapes de la création du programme d'installation sont les suivantes :

1. [Sélection de l'application à installer](../Editeurs/2028041.md) (uniquement si l'assistant est lancé depuis WDInst et si aucun projet n'est ouvert sous WINDEV).

2. Création de l'exécutable. Pour plus de détails, consultez [Créer un exécutable](../Editeurs/2025002.md). Si vous venez de créer l'exécutable, cliquez directement sur "2. Installation" dans l'assistant de création du programme d'installation. Si nécessaire, cliquez sur "Accéder aux options d'installation" dans la fenêtre qui s'affiche. 

3. Sélection du [mode d'installation](../Editeurs/2028085.md) à créer.

	- [Installation dans un Store Privé](../Editeurs/9000203.md). 

	- Installation individuelle/autonome (standard, compacte, MSI).

	- Installation avec mise à jour automatique.
			Trois mode d'installation avec mise à jour automatique sont disponibles :

		- [Installation en réseau local avec prise en charge du "Push"](../Editeurs/2028098.md).

		- [Installation par Internet](../Editeurs/9000026.md).

		- [Installation multisite](../Editeurs/2028100.md).




4. [Choix du répertoire d'installation](../Editeurs/2028047.md). 

5. [Sélection des fichiers à installer](../Editeurs/2028031.md) (et des postes sur lesquels chacun des fichiers doit être installé - installation avec mise à jour uniquement).

6. [Installation du framework](../Editeurs/2028031.md).

7. Installation des [éléments optionnels pour la base de données](../Editeurs/2028086.md). En fonction des éléments sélectionnés, il sera nécessaire d'effectuer les opérations suivantes dans la suite de l'assistant :

	- Paramétrer le modification automatique des fichiers de données.

	- Réindexer les fichiers HFSQL pendant l'installation.

	- [Intégrer l'installation d'un serveur HFSQL](../Editeurs/2028081.md).

	- Déployer des fichiers HFSQL Client/Serveur.

	- Personnaliser le chemin des fichiers de données lors de l'installation.

	- Personnaliser le chemin des fichiers du groupware lors de l'installation.

	- Inclure le MDAC dans l'installation.

	- Inclure le driver ODBC pour HFSQL.
			




8. [Sélection des modules complémentaires](../Editeurs/2028045.md). En fonction des modules sélectionnés, il sera nécessaire d'effectuer les opérations suivantes dans la suite de l'assistant :

	- Installer l'aide pour le Macro-Code Utilisateur.

	- Intégrer l'installation du logiciel Etats & Requêtes.

	- Afficher une licence d'utilisation.

	- Afficher un fichier Lisez-moi à la fin de l'installation.

	- Paramétrer la modification automatique des fichiers de données.

	- Choisir les outils optionnels à inclure dans l'installation.

	- Inclure le désinstallateur.




9. [Sélection du gabarit de l'installateur](../Editeurs/2028103.md). 

10. [Sélection des langues proposées lors de l'installation](../Editeurs/2028036.md). Pour chaque langue, il est possible de définir le titre de la fenêtre principale du programme d'installation et de [Personnaliser le programme d'installation](../Editeurs/2028061.md).

11. [Choix du support sur lequel le programme d'installation sera généré](../Editeurs/2028040.md).

12. [Génération du programme d'installation](../Editeurs/2028055.md).




## Le volet "Briques de code"
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000100"></a>
Le volet "Briques de code" liste les briques de code actuellement disponibles.

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=BriqueCode_Gestionnaire16.gif)
Il permet de :

- Ajouter une brique de code,

- Modifier une brique de code,

- Utiliser une brique de code,

- Supprimer une brique de code.




Pour plus de détails, consultez [Gestion des briques de code](../Editeurs/9000066.md).

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Afficher le volet "Briques de code"
<a name="afficher_volet_briques_code_ELTTEXTE000124"></a>
**Pour afficher le volet "Briques de code"**, sous le volet "Accueil", dans le groupe "Environnement", déroulez "Volets" et sélectionnez "Volets" puis "Briques de code".

Pour plus de détails sur les manipulations des volets, consultez [Manipuler les volets ancrables](../Editeurs/2027001.md).

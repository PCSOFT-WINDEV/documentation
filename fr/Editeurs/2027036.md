<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000121"></a>
Le volet "Images" permet de visualiser et lister toutes les images présentes dans le répertoire et les sous-répertoires du projet. 

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Volet_images%20-%20HC%20N%B0001.gif)
Depuis le volet "Images", il est possible de : 

- ouvrir une image sous l'éditeur d'images, 

- créer une image sous l'éditeur d'images, 

- supprimer une image. 




Remarque : Sous l'éditeur d'images, le volet "Images" est automatiquement affiché. Dans ce cas, ce volet permet de : 

- visualiser l'aperçu et définir le zoom de l'image affichée sous l'éditeur d'images, 

- afficher les couleurs et paramétrer les différents outils utilisés (épaisseur du trait, opacité, ...), 

- gérer les calques. 


Pour plus de détails, consultez [Editeur d'images](../Editeurs/1410086603.md). 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Afficher le volet "Images"
<a name="afficher_volet_images_ELTTEXTE000145"></a>
**Pour afficher le volet "Images"**, sous le volet "Accueil", dans le groupe "Environnement", déroulez "Volets" et sélectionnez "Volets" puis "Images". 

Pour plus de détails sur les manipulations des volets, consultez [Manipuler les volets ancrables](../Editeurs/2027001.md).

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Utilisation du volet "Images"
<a name="utilisation_volet_images_ELTTEXTE000169"></a>
Le volet "Images" permet de : 

- ouvrir dans l'éditeur d'images, une image spécifique présente sur le disque (dans un répertoire différent de celui du projet en cours). Il suffit de cliquer sur l'icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Volet_images%20-%20HC%20N%B0001%202.gif)
 et de sélectionner le fichier à ouvrir sous l'éditeur d'images. 

- créer une nouvelle image sous l'éditeur d'images (icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Volet_images%20-%20HC%20N%B0001%201.gif)
). 

- manipuler une image via le menu contextuel : 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Volet_images%20-%20HC%20N%B0002.gif)


	- Modifier l'image : L'image est ouverte sous l'éditeur d'images pour être modifiée. 

- Ouvrir l'emplacement : Permet d'afficher l'explorateur de fichiers Windows sur le répertoire contenant le fichier image. 

- Supprimer : Permet de supprimer le fichier image. 




Remarque : La bulle de survol de l'image indique le chemin actuel de l'image.


## Différence de comportement sous Windows Vista (et supérieur)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentation
<a name="presentation_ELTTEXTE000111"></a>
Vous venez de tester votre projet en utilisant une des options de tests Windows Vista (ou supérieur) :

- Test depuis un répertoire spécifique.

- Test avec un manifeste.

- Test avec un utilisateur spécifique.




Un problème potentiel pouvant survenir sous Windows Vista (ou supérieur) a été détecté. Vous pouvez utiliser le bouton [Déboguer] pour corriger ce problème. <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Erreur_Vista.gif)

## Quelques problèmes pouvant être rencontrés et leur correction
<a name="quelques_problemes_pouvant_etre_rencontres_leur_correction_ELTTEXTE000135"></a>


### Écriture dans un répertoire non autorisée
<a name="ecriture_dans_repertoire_non_autorisee_ELTPARAGRAPHE000031"></a>Vous tentez d'écrire dans un répertoire où vous n'avez pas les droits (par exemple avec les fonctions fxxx ou les fonctions INIxxx).

En effet, si votre application n'a pas de privilèges spécifiques, vous n'avez pas le droit d'écrire :

- à la racine du disque C.

- dans le répertoire de Windows.

- dans le répertoire "Program Files".




Les données de vos applications doivent être écrites dans les répertoires appropriés. Ces répertoires peuvent être connus et créés par les fonctions [fRepDonnéesCommun](../WDLang1/3036070.md), [fRepDonnéesUtilisateur](../WDLang1/3036071.md), [fRepGlobalCommun](../WDLang1/3036072.md), [fRepGlobalUtilisateur](../WDLang1/3036073.md).


<a name="NOTE2_2"></a>


### Écriture dans une clé de la base de registre non autorisée
<a name="ecriture_dans_une_cle_base_registre_non_autorisee_ELTPARAGRAPHE000059"></a>Vous utilisez une application sans privilèges et donc vous n'avez pas le droit d'écrire dans la clé "HKEY_LOCAL_MACHINE\\Software".

Il est conseillé d'écrire dans une autre clé de la base de registre, accessible par tous les utilisateurs du poste.

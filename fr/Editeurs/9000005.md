


## Les perso-dossiers
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000204"></a>
Les "Perso-dossiers" permettent de simplifier l'organisation des projets en permettant de regrouper les éléments de votre application par thèmes. Les perso-dossiers sont disponibles : 

- sur chaque catégorie d'éléments présents dans le volet "Explorateur de projet" : fenêtres, états, ...

- directement au niveau du projet. 




**Remarques** :

- Il est également possible de créer des perso-dossiers dans l'éditeur d'analyses. Pour plus de détails, consultez [Perso-dossiers dans l'analyse](../Editeurs/2011003.md).

- Il est également possible d'organiser un projet de taille importante grâce aux [composants internes](../Editeurs/2014100.md).




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000228"></a>


### Créer un perso-dossier
<a name="creer_persodossier_ELTPARAGRAPHE000032"></a>Pour créer un perso-dossier :

1. Affichez si nécessaire le volet "Explorateur de projet".

2. Sélectionnez le dossier dans lequel le perso-dossier doit être créé : Fenêtres, Pages, ...

3. Dans le menu contextuel, sélectionnez l'option "Nouveau perso-dossier".

4. Saisissez le nom du perso-dossier et validez. Le perso-dossier apparaît dans le volet "Explorateur de projet" sous la forme d'un dossier jaune. Ce perso-dossier pourra être personnalisé. Pour plus de détails, consultez le paragraphe [Actions possibles sur un perso-dossier](#NOTE3_1).




**Remarques** :

- Si vous utilisez le GDS, il est nécessaire d'être Maître sur le projet pour créer un perso-dossier : le perso-dossier sera alors reporté lors de la prochaine synchronisation. Dans le cas contraire, le perso-dossier créé disparaîtra lors de la prochaine synchronisation.

- Les perso-dossiers sont disponibles dans les collections de procédures.



<a name="NOTE2_2"></a>


### Ouvrir un perso-dossier
<a name="ouvrir_persodossier_ELTPARAGRAPHE000054"></a>Pour ouvrir un perso-dossier, double-cliquez sur le nom du perso-dossier dans le volet "Explorateur de projet".
<a name="NOTE2_4"></a>


### Associer des éléments du projet à un perso-dossier
<a name="associer_des_elements_projet_persodossier_ELTPARAGRAPHE000061"></a>**Pour associer un élément du projet à un perso-dossier** : 

1. Sélectionnez l'élément voulu dans le volet "Explorateur du projet". 

2. Réalisez un "Drag and Drop" (Glisser / Déplacer) de l'élément vers le perso-dossier voulu. 




**Pour associer le même élément à plusieurs perso-dossiers**, il suffit de maintenir la touche Ctrl enfoncée lors du "Drag and Drop" vers le perso-dossier.

**Remarque** : Lors de la sauvegarde d'un nouvel élément, la fenêtre suivante s'affiche : 
<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=PersoDossier.GIF)

Cette fenêtre permet de spécifier le perso-dossier auquel l'élément en cours doit être associé. 
Pour que cette fenêtre s'affiche automatiquement à chaque sauvegarde d'un nouvel élément, cochez l'option "Demander l'emplacement des nouveaux éléments dans le projet" dans l'onglet "Options" de la description du projet (sous le volet "Projet", dans le groupe "Projet", cliquez sur "Description").

**Un élément du projet peut être associé à plusieurs perso-dossiers.**

**Remarque** : **Perso-dossier et GDS** : 
Si vous utilisez le GDS, il est nécessaire d'être maître sur le projet pour associer un nouvel élément à un perso-dossier. Dans ce cas, pour récupérer l'élément dans le perso-dossier, les autres développeurs doivent : 

- sous le volet "GDS", dans le groupe "Projet", dérouler "Maître / Invité". 

- sélectionner "Récupérer la version Maître (complet)".  Dans tous les autres cas, l'élément sera récupéré mais il ne sera pas mis dans le perso-dossier.

- Un invité peut mettre un élément dans un perso-dossier. Cependant, les autres développeurs ne peuvent pas récupérer l'élément dans le même perso-dossier : ils ne récupéreront que l'élément.



<a name="NOTE2_5"></a>


### Dissocier un élément à un perso-dossier
<a name="dissocier_element_persodossier_ELTPARAGRAPHE000121"></a>Pour dissocier un élément à un perso-dossier : 

1. Sélectionnez dans le volet "Explorateur du projet" l'élément à dissocier. 

2. Réalisez un "Drag and Drop" de l'élément présent dans le perso-dossier vers le nom du projet.



<a name="NOTE2_6"></a>


### Renommer un perso-dossier
<a name="renommer_persodossier_ELTPARAGRAPHE000131"></a>Pour renommer un perso-dossier : 

1. Sélectionnez dans le volet "Explorateur du projet" le perso-dossier à renommer. 

2. Sélectionnez "Description" du menu contextuel. 

3. Dans la fenêtre qui s'affiche, modifiez le nom du perso-dossier et validez.

## Actions possibles sur un perso-dossier
<a name="actions_possibles_sur_persodossier_ELTTEXTE000276"></a>


### Impression du contenu d'un perso-dossier
<a name="impression_contenu_persodossier_ELTPARAGRAPHE000144"></a>Il est possible d'imprimer le dossier des éléments présents dans un perso-dossier : 

1. Sélectionnez le perso-dossier dans le volet "Explorateur de projet". 

2. Dans le menu contextuel, sélectionnez l'option "Imprimer le dossier du perso-dossier".

3. L'assistant de génération du dossier technique des éléments du perso-dossier se lance.




<a name="NOTE4"></a>
<a name="NOTE4_1"></a>


## Composant
<a name="composant_ELTTEXTE000300"></a>
Lors de l'ajout d'un composant dans le projet en cours, les éléments utilisés par ce composant sont regroupés dans un perso-dossier.




## Faire une aide automatique
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000102"></a>
Les Fonctionnalités Automatiques de vos Applications (FAA) permettent d'ajouter des fonctionnalités à votre application sans aucune ligne de code. Il suffit de cocher les options appropriées afin de bénéficier de ces fonctionnalités.

Vous avez la possibilité de générer une aide pour votre application.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Principe
<a name="principe_ELTTEXTE000126"></a>
WINDEV dispose d'un éditeur d'aide vous permettant de générer une aide complète pour votre application.

L'éditeur d'aide de WINDEV génère un squelette d'aide afin de vous en faciliter la création. Les mécanismes généraux de votre application y sont expliqués.

Vous retrouverez ainsi des pages détaillant :

- **les fonctionnalités d'un champ Table** : l'utilisation des loupes, les tris dans les colonnes, l'affichage du menu contextuel, etc. 
	Cette page est automatiquement générée si votre application dispose au moins d'une fenêtre avec un champ Table et que vous choisissez de documenter ce type de fenêtre.

- **les fonctionnalités de la fenêtre du visualisateur de rapports**.
	Cette page est automatiquement générée si votre application dispose au moins d'un état et que vous choisissez de documenter au moins un état.




Cette génération automatique valorise les applications que vous diffusez, en permettant aux utilisateurs de mieux les utiliser.

**Remarque** : Dans la version actuelle, seules l'utilisation des champs Table et l'utilisation de la fenêtre du visualisateur de rapports sont documentées automatiquement. D'autre pages générées automatiquement seront intégrées dans les futures versions.

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Création d'une aide
<a name="creation_une_aide_ELTTEXTE000150"></a>
Les pages d'aide automatique sont générées lors de la création de l'aide. Ces pages sont incluses selon les choix sélectionnés dans l'assistant de création de l'aide. Par exemple, la page de description des fonctionnalités du visualisateur de rapports ne sera intégrée que si vous choisissez de documenter au moins un état.

Pour générer une aide :

1. Cliquez sur ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_nouveau.gif) parmi les boutons d'accès rapide. La fenêtre de création d'un nouvel élément s'affiche : cliquez sur "Architecture" puis sur "Aide".

2. Un assistant vous guide pour la création de l'aide. Remplissez les diverses informations jusqu'au plan de sélection des fenêtres à documenter.

3. Si vous sélectionnez au moins une fenêtre contenant un champ Table, une page d'aide est automatiquement générée pour expliquer les fonctionnalités de ce champ.

4. Dans l'étape suivante, vous pouvez sélectionner les états à documenter. En sélectionnant au moins un état, la page d'aide sur la fenêtre du visualisateur de rapports est automatiquement générée.

5. Terminez l'assistant : votre fichier d'aide est généré.




Vous pouvez constater qu'une partie "Divers" a été intégrée dans le sommaire du fichier d'aide. Cette partie contient les pages d'aide générées automatiquement.<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Fonctionnalites_auto_Aide2.gif)

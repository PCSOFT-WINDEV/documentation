<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000094"></a>
Le volet "Analyse UML" affiche les différents diagrammes UML (diagramme de classes, diagramme de déploiement, ...) du projet en cours.
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=VoletAnalyseUML.gif)
Le menu contextuel (clic droit) du volet "Analyse UML" permet de :

- Créer un nouveau modèle UML. Le modèle UML associé au projet peut contenir plusieurs diagrammes UML.

- Ouvrir un modèle UML.

- Ouvrir le modèle UML du projet en cours.

- Afficher la description des diagrammes.

- Ouvrir et fermer les diagrammes.

- Renommer les diagrammes.

- Supprimer les diagrammes.




Si aucun modèle UML n'est associé au projet en cours, le volet "Analyse UML" propose de créer un nouveau modèle.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Afficher le volet "Analyse UML"
<a name="afficher_volet_analyse_uml_ELTTEXTE000118"></a>
**Pour afficher le volet "Analyse UML"**, sous le volet "Accueil", dans le groupe "Environnement", déroulez "Volets" et sélectionnez "Volets" puis "Analyse UML".

Pour plus de détails sur les manipulations des volets, consultez [Manipuler les volets ancrables](../Editeurs/2027001.md).

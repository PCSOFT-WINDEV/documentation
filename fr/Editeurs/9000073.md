


## Correcteurs disponibles dans l'éditeur de code
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000151"></a>
L'éditeur de code met à votre disposition plusieurs outils pour corriger le code saisi :

- le correcteur automatique des éléments du projet,

- le correcteur d'orthographe pour les commentaires et les chaînes de caractères. 




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correcteur automatique
<a name="correcteur_automatique_ELTTEXTE000175"></a>
Lorsque le nom d'une variable ou d'un élément du projet n'est pas reconnu, le menu contextuel de l'éditeur de code propose une liste des corrections possibles. Cette liste présente le nom des variables et/ou des éléments du projet pouvant correspondre.

Par exemple : <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CorrecteurAuto.GIF)


Plus de risque d'erreur de frappe, de variable non déclarée, ...

Il suffit simplement de sélectionner le nom voulu.

<a name="NOTE3_1"></a>


## Correcteur orthographique
<a name="correcteur_orthographique_ELTTEXTE000199"></a>
Le correcteur d'orthographe de l'éditeur de code détecte les fautes dans :

- les commentaires,

- les chaînes utilisées dans le code (chaînes simples, messages multilingues, dialogues et chaînes multilingues).




Les fonctions et ordres WLangage sont reconnus comme orthographiquement corrects.

**Quel est le correcteur orthographique utilisé ?**
Le correcteur utilisé est le correcteur orthographique livré avec WINDEV/WEBDEV/WINDEV Mobile. Ce correcteur orthographique est disponible : 

- pour les commentaires (en français ou en anglais, selon la langue utilisée pour le code).

- pour les chaînes multilingues en Français et en Anglais.


Pour plus de détails, consultez [Vérification d'orthographe basée sur les dictionnaires Hunspell](../WDChamp/1410087030.md).

Le correcteur orthographique est activé par défaut. 

**Cas des chaînes multilingues**
Le dictionnaire utilisé est celui de la langue affichée. Pour modifier la langue affichée, sous le volet "Affichage", dans le groupe "Langue affichée", déroulez "Langue affichée". Le correcteur orthographique est une option globale à l'utilisateur.

**Remarque** : Par défaut, le correcteur d'orthographe est proposé uniquement en français et en anglais. Pour utiliser le correcteur d'orthographe pour d'autres langues dans les chaînes multilingues, il suffit d'ajouter les fichiers correspondant aux dictionnaires de type Hunspell voulus dans le répertoire "\\Programmes\\Donnees\\Dic". 
Ces dictionnaires peuvent facilement être trouvés sur Internet. Ces dictionnaires sont constitués d'un fichier ".dic" et d'un fichier ".aff" de même nom qui doivent se trouver au même emplacement.

**Attention** : Si la chaîne multilingue utilise une coloration différente de celle des chaînes, la correction orthographique est désactivée.

**Mots ajoutés au dictionnaire**
Les mots ajoutés au dictionnaire sont présents dans le fichier "C:\\Users\\&lt;Utilisateur&gt;\\AppData\\Roaming\\PC SOFT\\perso.dic".
<a name="NOTE3_2"></a>


### Désactiver le correcteur orthographique
<a name="desactiver_correcteur_orthographique_ELTPARAGRAPHE000087"></a>Pour désactiver le correcteur orthographique :

1. Sous le volet "Accueil", dans le groupe "Environnement", déroulez "Options" et sélectionnez "Options de l'éditeur de code".

2. Dans l'onglet "Code", décochez les endroits (commentaires ou chaînes) où le correcteur orthographique ne doit pas être utilisé (option "Activer le correcteur orthographique").




## Recharger un document
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000077"></a>
Pour recharger un document, sous le volet "Accueil", dans le groupe "Général", déroulez "Ouvrir" et sélectionnez "Recharger le document".

Cette fonctionnalité effectue les opérations suivantes :

- Fermer le document en cours sous l'éditeur (sans enregistrer les modifications).

- Ouvrir le même document.




Cette option permet par exemple de ré-ouvrir la fenêtre en cours en se basant sur le dernier enregistrement (le dernier Ctrl + S). Toutes les modifications effectuées depuis le dernier enregistrement ne sont pas prises en compte.

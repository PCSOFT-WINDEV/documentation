


## Type de fenêtres/pages créées par le RID
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000120"></a>
Le RID (Rapid Interface Design) permet de créer des fenêtres/pages à partir :

- de l'analyse liée au projet.

- du Pattern RID sélectionné.

- des gabarits.




Les fenêtres/pages générées contiennent uniquement les champs liés aux rubriques de l'analyse. Tout le code nécessaire au fonctionnement de ces fenêtres / pages reste à la charge du développeur. Vous pouvez directement saisir votre code personnalisé.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000144"></a>


### Créer une fenêtre/page RID 
<a name="creer_une_fenetrepage_rid_ELTPARAGRAPHE000024"></a>Pour créer une fenêtre/page RID :

1. Cliquez sur ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_nouveau.gif)
 parmi les boutons d'accès rapide. 

	- La fenêtre de création d'un nouvel élément s'affiche : cliquez sur "Fenêtre" (ou "Page") puis sur "Fenêtre" (ou "Page").

	- L'assistant se lance.




2. Affichez l'onglet "R.I.D.".

3. Sélectionnez le pattern RID à utiliser.

4. Sélectionnez le type de fenêtre/page à créer.



<a name="NOTE2_2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 

### Les différentes types de fenêtres RID
<a name="les_differentes_types_fenetres_rid_ELTPARAGRAPHE000043"></a>Les différents types de fenêtres proposés par le RID sont les suivantes :

- Fenêtre fiche simple.

- Fenêtre table simple.

- Fenêtre table avec fiche détail.




Pour plus de détails sur ces types de fenêtres, consultez [Types de fenêtres RAD](../Editeurs/2031003.md).

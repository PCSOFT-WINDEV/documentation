


## Déployer un plan d'action
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000109"></a>
Une fois créés, les plans d'action doivent être déployés pour être exécutés.  

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000133"></a>


### Déployer un plan d'action
<a name="deployer_plan_action_ELTPARAGRAPHE000018"></a>Pour déployer un plan d'action : 

1. Ouvrez le projet contenant le plan d'action à déployer. 

2. Cliquez sur l'icône de génération des plans d'action parmi les boutons d'accès rapide : ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_generation_Plan.gif)


3. Enregistrez le plan d'action si nécessaire. 

4. L'assistant de génération et de déploiement des plans d'action se lance. Indiquez : 

	- le nom de la bibliothèque des plans d'action.

	- la société.

	- la description.

	- la version. Cette version peut être incrémentée automatiquement à chaque génération. 

	- le copyright.




5. Sélectionnez les plans d'actions accessibles dans l'administrateur de la fabrique logicielle. 

6. Il est possible de déployer le plan d'action immédiatement ou plus tard. Dans ce cas, cette opération sera effectuée depuis l'administrateur de la fabrique logicielle. 
	**Remarque** : Si le déploiement est effectué immédiatement, indiquez le nom du poste correspondant au coordinateur de la fabrique logicielle. 

7. Validez. 

8. Il est possible de réaliser une sauvegarde du projet en cours. 

9. Si vous avez choisi de déployer immédiatement, le coordinateur s'affiche et indique le résultat du déploiement. 





### Déployer un plan d'action à partir de l'administrateur de la fabrique logicielle
<a name="deployer_plan_action_partir_administrateur_fabrique_logicielle_ELTPARAGRAPHE000042"></a>Pour déployer un plan d'action à partir de l'administrateur de la fabrique logicielle : 

1. Lancez l'administrateur de la fabrique logicielle : sous le volet "Outils", dans le groupe "Utilitaires", cliquez sur "WDFAdmin".

2. Dans la fenêtre qui s'affiche, indiquez le nom du coordinateur auquel l'administrateur doit se connecter. 

3. L'administrateur de la fabrique logicielle se lance. 

4. Dans l'onglet "Entrepôt", cliquez sur le bouton "Ajouter une bibliothèque". Sélectionnez le fichier "wdfab" correspondant à la bibliothèque du plan d'action à déployer. Ce fichier a été généré dans le répertoire Exe du projet lié au plan d'action. 

5. Le plan d'action sélectionné est déployé.

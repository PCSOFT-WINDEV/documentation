


## PersoMenu
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000179"></a>
Le PersoMenu vous permet de créer un menu personnalisé dans la barre de menu de WINDEV, WEBDEV et WINDEV Mobile. Vous pouvez ainsi ajouter des raccourcis vers vos applications préférées (Calculatrice, ...), ouvrir différents fichiers (Charte, ...).
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=PersoMenu18.gif)
Le PersoMenu est également accessible depuis le [Tableau de bord du projet](../Editeurs/9500221.md), grâce au Widget "PersoMenu".

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000203"></a>


### Ajouter une option au PersoMenu
<a name="ajouter_une_option_persomenu_ELTPARAGRAPHE000023"></a>Pour ajouter une option au PersoMenu : 

1. Cliquez sur le logo correspondant à la version du produit. Dans le menu qui s'affiche, sélectionnez l'option "PersoMenu .. Personnaliser le &lt;PersoMenu&gt;". ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=persomenu_creation.gif)


2. La fenêtre de configuration du PersoMenu s'ouvre.
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=PersoMenu_description.gif&type=thumb)


3. Cliquez sur le bouton "Ajouter".

4. Saisissez un libellé de menu.

5. Saisissez les lignes de commande à exécuter lorsque cette option sera choisie. Vous pouvez y saisir :

	- le chemin d'un exécutable : saisissez le chemin directement dans le champ ou utilisez le bouton "Parcourir...".

	- des macros : cliquez sur le bouton "Macro" et double-cliquez sur la macro que vous souhaitez ajouter à votre ligne de commande. Par exemple, la macro "Nom du projet en cours" {$NomProjet} sera remplacée par le nom de votre projet en cours d'édition.
			Les principales macros disponibles sont les suivantes : 

		- {$NomProjet} : Nom du projet en cours. 

		- {$RepProjet} : Répertoire du projet en cours. 

		- {$CheminCompletProjet} : Chemin complet du projet en cours. 

		- {$NomExe} : Nom de l'exécutable généré à partir du projet en cours. 

		- {$NomConfig} : Nom de la configuration en cours pour le projet en cours. 


 Pour connaître la liste complète des macros disponibles, il suffit de cliquer sur le bouton "Macro". 

6. Sélectionnez ensuite les options d'exécution de la ligne de commande :

	- Afficher la console : une console (invite de commande) sera affichée.

	- Laisser la console ouverte : si une console est affichée, elle restera ouverte à la fin de l'exécution du programme lancé.

	- Exécution non bloquante : l'exécution de la ligne de commande ne bloquera pas l'accès à WINDEV, WEBDEV ou WINDEV Mobile.




7. Sélectionnez le répertoire d'exécution du programme.

8. Saisissez ensuite :

	- le message affiché dans la barre de messages lors du survol de l'option.

	- l'icône affichée devant l'option dans le PersoMenu.







**Remarque** : Il est également possible d'ajouter un dossier ou un séparateur dans un PersoMenu. Un dossier permet de créer un sous-menu.
<a name="NOTE2_2"></a>


### Supprimer une option du PersoMenu
<a name="supprimer_une_option_persomenu_ELTPARAGRAPHE000075"></a>Pour supprimer une option du PersoMenu : 

1. Cliquez sur le logo correspondant à la version du produit. Dans le menu qui s'affiche, sélectionnez l'option "PersoMenu .. Personnaliser le &lt;PersoMenu&gt;".![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=persomenu_creation.gif)


2. La fenêtre de configuration du PersoMenu s'ouvre.

3. Sélectionnez l'option de menu que vous souhaitez supprimer.

4. Cliquez sur le bouton "Supprimer".



<a name="NOTE2_3"></a>


### Remarque
<a name="remarque_ELTPARAGRAPHE000088"></a>Une option d'un PersoMenu peut être exécutée avant ou après la génération d'une [configuration de projet](../Editeurs/9000030.md).

## Exemples
<a name="exemples_ELTTEXTE000239"></a>


### Créer un option pour lancer un exécutable
<a name="creer_option_pour_lancer_executable_ELTPARAGRAPHE000100"></a>Ce paragraphe décrit les manipulations à effectuer pour créer une option de menu permettant de lancer un exécutable (ici, la calculatrice de Windows) :

1. Cliquez sur le logo correspondant à la version du produit. Dans le menu qui s'affiche, sélectionnez l'option "PersoMenu .. Personnaliser le &lt;PersoMenu&gt;". ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=persomenu_creation.gif)


2. Dans la fenêtre de configuration du PersoMenu, cliquez sur le bouton "Ajouter une option".

3. Saisissez le libellé "Calculatrice".

4. Cliquez sur le bouton "Parcourir..." et sélectionnez le fichier "CALC.exe" (situé par exemple dans le répertoire System32 de Windows).

5. Saisissez "Lancer la calculatrice" dans le champ "Message" et sélectionnez à nouveau le fichier "CALC.exe" dans le champ "Icône".

6. Validez.




Vous pouvez maintenant ouvrir autant de calculatrices que vous le souhaitez directement depuis les différents éditeurs.
<a name="NOTE3_2"></a>


### Créer une option de menu avec macros
<a name="creer_une_option_menu_avec_macros_ELTPARAGRAPHE000117"></a>Ce paragraphe décrit comment créer une option de menu en utilisant des macros. Dans cet exemple, les macros seront utilisées pour ouvrir le fichier &lt;NomDeProjet&gt;.rtf. Ce fichier est automatiquement créé pour tout nouveau projet et contient les informations générales du projet.

1. Cliquez sur le logo correspondant à la version du produit. Dans le menu qui s'affiche, sélectionnez l'option "PersoMenu .. Personnaliser le &lt;PersoMenu&gt;".![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=persomenu_creation.gif)


2. Dans la fenêtre de configuration du PersoMenu, cliquez sur le bouton "Ajouter une option".

3. Saisissez le libellé "Informations du projet".

4. Cliquez sur le bouton "Macro" et sélectionnez la macro "{$RepProjet}".

5. Dans la ligne de commande, saisissez le caractère "\\" à la suite de "{$RepProjet}".

6. Cliquez à nouveau sur le bouton "Macro" et sélectionnez la macro "{$NomProjet}".

7. Dans la ligne de commande, saisissez les caractères ".rtf" à la suite de "{$NomProjet}".

8. Saisissez "Informations sur le projet" dans le champ "Message".

9. Validez.




Vous pouvez maintenant ouvrir le fichier ".rtf" spécifique à chaque projet.

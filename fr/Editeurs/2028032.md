


## WDInst : composants et modèles d'installation 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000191"></a>
Par défaut, le programme d'installation installe tous les fichiers spécifiés sur le poste d'installation.

L'éditeur d'installation WDInst permet de proposer plusieurs modèles d'installation :

- Installation complète.

- Installation minimale.

- Installation personnalisée.




Pour chaque modèle d'installation, WDInst permet de spécifier les groupes de fichiers à installer. Ces groupes sont appelés **"composants"**. L'installation d'un composant peut être obligatoire ou non (par exemple composant "Tutoriel" permettant d'installer les fichiers liés au tutoriel). Chaque composant peut être associé à un ou plusieurs modèles d'installation.

Par défaut, aucun composant d'installation n'est créé et l'utilisateur ne peut pas choisir de modèle d'installation spécifique (tous les fichiers de l'application seront automatiquement installés).

**Attention : Les composants proposés par l'éditeur d'installation n'ont aucun lien avec les composants créés par WINDEV, WEBDEV ou WINDEV Mobile**. Les composants de l'installation permettent uniquement de regrouper les fichiers par thème.

## Caractéristiques d'un composant et modèle d'installation
<a name="caracteristiques_composant_modele_installation_ELTTEXTE000215"></a>


### Caractéristiques d'un composant d'installation
<a name="caracteristiques_composant_installation_ELTPARAGRAPHE000030"></a>Un composant d'installation peut être :

- soit **obligatoire**. Les fichiers associés à ce composant seront obligatoirement installés sur le poste d'installation quel que soit le type d'installation choisi.

- soit **associé à un ou plusieurs modèles d'installation** :

	- *Installation complète* : les fichiers associés à ce composant seront installés sur le poste d'installation uniquement si une installation complète est réalisée.

	- *Installation minimale* : les fichiers associés à ce composant seront installés sur le poste d'installation uniquement si une installation minimale est réalisée.

	- *Installation personnalisée* : si une installation personnalisée est réalisée, l'utilisateur pourra choisir d'installer ou non les fichiers associés à ce composant.

## Comment le faire ?
<a name="comment_faire_ELTTEXTE000245"></a>


### Gérer les composants d'installation et les modèles d'installation
<a name="gerer_les_composants_installation_les_modeles_installation_ELTPARAGRAPHE000082"></a>La gestion des composants d'installation ainsi que celle des modèles d'installation sont réalisées sous l'éditeur d'installation WDInst : sous le volet "Outils", dans le groupe "Utilitaires", cliquez sur "WDInst".



L'assistant de création du programme d'installation ne permet pas de gérer les composants d'installation et de proposer plusieurs modèles d'installation à l'utilisateur.
<a name="NOTE3_2"></a>


### Créer un nouveau composant d'installation
<a name="creer_nouveau_composant_installation_ELTPARAGRAPHE000104"></a>Pour créer un nouveau composant d'installation : 

1. Cliquez sur le bouton "Nouveau" dans la fenêtre "Liste des composants".

2. Saisissez dans le tableau le libellé du composant dans toutes les langues gérées par le programme d'installation.

3. Sélectionnez le composant dans le tableau et spécifiez ses caractéristiques (composant obligatoire, installation complète, ...) grâce au bouton "Propriétés".



<a name="NOTE3_3"></a>


### Associer un ou plusieurs fichiers à un composant d'installation
<a name="associer_plusieurs_fichiers_composant_installation_ELTPARAGRAPHE000115"></a>Pour associer un ou plusieurs fichiers à un composant d'installation : 

1. Sélectionnez le ou les fichiers à associer dans la liste des fichiers de l'installation (partie haute de la fenêtre). 
	Pour sélectionner plusieurs fichiers simultanément, maintenez la touche Ctrl ou Maj enfoncée lors de la sélection des fichiers.

2. Sélectionnez le composant associé dans la fenêtre "Liste des composants". Si le fichier est associé au composant d'installation, la case située devant le nom du composant doit être cochée.




**Remarques** :

- Un même fichier peut être associé à plusieurs composants d'installation.

- La colonne "Composant" de la liste des fichiers de l'installation permet de visualiser rapidement les fichiers d'un composant.



<a name="NOTE3_4"></a>


### Proposer plusieurs modèles d'installation
<a name="proposer_plusieurs_modeles_installation_ELTPARAGRAPHE000132"></a>Pour proposer plusieurs modèles d'installation : 

1. Sélectionnez l'option "Paramètres de l'installation .. Assistant".

2. Sélectionnez le modèle d'installation proposé par le programme d'installation :

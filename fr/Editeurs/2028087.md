


## WDInst : Paramétrer le désinstalleur
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000152"></a>
Le désinstalleur d'application est l'application qui permet à l'utilisateur final de désinstaller l'application. 

Par défaut : 

- le désinstalleur est inclus dans le programme d'installation. 

- le désinstalleur fonctionne en français.

- le désinstalleur ne lance aucun autre programme. 




WDInst permet de paramétrer le désinstalleur en permettant de : 

- lancer un exécutable spécifique à la fin de la désinstallation.

- traduire les différents messages dans WDInst. 


Ce paramétrage est réalisé sous l'éditeur d'installation WDInst. Pour lancer WDinst, sous le volet "Outils", dans le groupe "Utilitaires", cliquez sur "WDInst".

Remarque : Pour indiquer sous WDInst si un désinstalleur doit être inclus ou non : 

1. Sélectionnez l'option "Paramètres de l'installation .. Fin de l'installation". 

2. Cochez ou décochez l'option "Inclure le désinstallateur".

## Lancer un exécutable
<a name="lancer_executable_ELTTEXTE000176"></a>
A la fin de la désinstallation de l'application, il est possible de lancer un exécutable supplémentaire. Cet exécutable sera lancé AVANT la désinstallation de l'application. 

Cet exécutable peut être : 

- L'application, lancée avec une ligne de commande spécifique. 

- Une application spécifique (présente dans le répertoire de l'application ou dans le poste en cours). 




**Pour lancer un exécutable lors de la désinstallation** : 

1. Sélectionnez l'option "Paramètres de l'installation .. Options du désinstalleur". 

2. Dans la fenêtre qui s'affiche, spécifiez la ligne de commande à exécuter. 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Options_desinstalleur%20-%20HC%20N%B0001.gif)
Par exemple : "&lt;RepInst&gt;\\Monappli.exe / UNINST".
	Il est possible d'utiliser les mots-clés suivants :

	- **&lt;RepInst&gt;** : répertoire d'installation choisi par l'utilisateur lors de l'installation.

	- **&lt;LectInst&gt;** : lecteur du répertoire d'installation choisi par l'utilisateur lors de l'installation.

	- **&lt;Langue&gt;** : numéro de la langue d'installation.

	- **&lt;FicInst&gt;** : répertoire d'installation des fichiers de données. 

	- **&lt;RepDonneesUser&gt;** : répertoire des données de l'utilisateur connecté (notamment pour une installation réalisée sur un poste Windows Vista ou supérieur). 

	- **&lt;RepDonneesCommun&gt;** : répertoire des données communes aux utilisateurs (notamment pour une installation réalisée sur un poste Windows Vista ou supérieur). 

	- **&lt;User&gt;** : Nom de l'utilisateur

	- **&lt;srXXX&gt;** : Constantes de la fonction [SysRep](../WDLang1/3073026.md). 




3. Validez.

## Traduire le désinstalleur
<a name="traduire_desinstalleur_ELTTEXTE000200"></a>
La traduction du désinstalleur est réalisée sous l'éditeur d'installation WDInst. Pour lancer WDinst, sous le volet "Outils", dans le groupe "Utilitaires", cliquez sur "WDInst".

**Pour traduire le désinstalleur** : 

1. Sélectionnez l'option "Paramètres de l'installation .. Options du désinstalleur".

2. Dans la fenêtre qui s'affiche, indiquez : 

	- le fichier de traduction à utiliser. Par défaut, le fichier de traduction WDIninst.INI est fourni dans le répertoire d'installation de WINDEV. Ce fichier contient par défaut les traductions en anglais. 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=WDInst_Tradoc.gif)


	- pour chaque message, les différentes traductions dans la langue voulue.




## Favicone : Définir l'icône du site
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000098"></a>
WEBDEV permet de définir simplement l'icône du site (également appelée Favicone). Cette icône s'affiche en raccourci sur le bureau, dans les listes de favoris du navigateur, en entête de la barre d'adresses, ...
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Favicone.gif)


<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000122"></a>
Pour associer un Favicone à votre adresse de site :

1. Affichez la fenêtre de description du projet : sous le volet "Projet", dans le groupe "Projet", cliquez sur "Description".

2. Dans l'onglet "Projet", le champ "Icône du site" permet de saisir l'image utilisée en favicone. 

3. Déroulez l'icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Menu_Image_Editeur%20-%20HC%20N%B0001.gif)
 : 

	- L'option "Parcourir" permet de sélectionner un fichier image présent sur votre disque. 

	- L'option "Catalogue" permet de sélectionner une image dans le catalogue d'images fourni en standard avec WEBDEV.




4. Validez la fenêtre de description. 




**Limitations** :

- La taille de l'image doit être 16x16 ou 32x32. 

- Il est conseillé d'utiliser une image au format .ICO pour que cette image soit correctement affichée par tous les navigateurs qui gèrent l'affichage d'un Favicone. 

- Evitez les formats ".PNG", ".GIF", ".JPG" car ces formats ne sont pas gérés par tous les navigateurs. Seuls certains navigateurs récents savent les afficher.

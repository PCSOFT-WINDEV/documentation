


## WDInst : grouper les fichiers à installer 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000169"></a>
Par défaut, lors de la génération du programme d'installation, les fichiers à installer sont compactés dans un seul fichier archive (".WDZ") pour être plus facilement diffusables.

A l'exécution du programme d'installation, ces fichiers sont automatiquement décompactés, extraits de l'archive et copiés sur le poste d'installation. Il est également possible de ne pas compacter les fichiers à installer (voir [Copier et compacter un ou plusieurs fichiers](../Editeurs/2028033.md)).

L'éditeur d'installation WDInst offre la possibilité de regrouper les fichiers à installer (groupement par type par exemple). Dans ce cas :

- La génération du programme d'installation crée autant de fichiers archive que de groupes. Le nom des fichiers archive créés correspond à "&lt;NomGroupe&gt;.WDZ".

- Les fichiers n'appartenant à aucun groupe seront archivés dans le fichier "__WDInst.WDZ".




Par défaut, aucun groupe de fichiers n'est créé.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Les groupes
<a name="les_groupes_ELTTEXTE000193"></a>


### Avantages des groupes
<a name="avantages_des_groupes_ELTPARAGRAPHE000030"></a>

- Optimisation de la génération du programme d'installation pour les mises à jour. 
	Si un groupe ne contient aucun fichier modifié depuis la dernière génération, l'archive correspondante (fichier ".WDZ") ne sera pas recréée.

- Hiérarchisation des fichiers à installer (création d'un groupe pour les fichiers image et d'un groupe pour les fichiers vidéo par exemple).

- Utilisation simplifiée des [composants](../Editeurs/2028032.md) (composant "Multimédia" associé aux groupes "image" et "vidéo" par exemple).

## Comment le faire ?
<a name="comment_faire_ELTTEXTE000217"></a>


### Gérer les groupes
<a name="gerer_les_groupes_ELTPARAGRAPHE000046"></a>La manipulation des groupes est réalisée sous l'éditeur d'installation WDInst. Pour lancer cet éditeur, sous le volet "Outils", dans le groupe "Utilitaires", cliquez sur "WDInst".

L'assistant de création du programme d'installation ne permet pas de gérer les groupes de fichiers.
<a name="NOTE3_2"></a>


### Créer un nouveau groupe de fichiers
<a name="creer_nouveau_groupe_fichiers_ELTPARAGRAPHE000067"></a>La création d'un groupe se fait directement en associant un des fichiers à installer à un nouveau groupe.

Pour créer un groupe et lui associer un fichier :

1. Sélectionnez le fichier à associer au groupe dans la fenêtre "Liste des fichiers de l'installation".

2. Dans la fenêtre "Description du fichier", affichez l'onglet "Option".

3. Cliquez sur le bouton "Nouveau".

4. Saisissez le nom du groupe de fichiers et validez.



<a name="NOTE3_3"></a>


### Associer un ou plusieurs fichiers à un groupe de fichiers
<a name="associer_plusieurs_fichiers_groupe_fichiers_ELTPARAGRAPHE000081"></a>Pour associer un ou plusieurs fichiers à un groupe de fichiers : 

1. Sélectionnez le ou les fichiers à associer dans la fenêtre "Liste des fichiers de l'installation".
	Pour sélectionner plusieurs fichiers simultanément, maintenez la touche Ctrl ou Maj enfoncée lors de la sélection des fichiers.

2. Affichez l'onglet "Option" de la fenêtre "Description du fichier".

3. Sélectionnez le groupe de fichiers associé (option "Groupe").



<a name="NOTE3_4"></a>


### Associer un ou plusieurs nouveaux fichiers à un groupe de fichiers
<a name="associer_plusieurs_nouveaux_fichiers_groupe_fichiers_ELTPARAGRAPHE000094"></a>

1. Sélectionnez l'option "Édition .. Ajouter des fichiers".

2. Sélectionnez le ou les fichiers à ajouter. Pour plus de détails, consultez [Ajouter un ou plusieurs fichiers](../Editeurs/2028066.md).

3. Sélectionnez le groupe de fichiers associé en cliquant sur le bouton "Autre".

4. Validez l'ajout du ou des fichiers.



<a name="NOTE3_5"></a>


### Dissocier un ou plusieurs fichiers d'un groupe de fichiers
<a name="dissocier_plusieurs_fichiers_groupe_fichiers_ELTPARAGRAPHE000107"></a>Pour dissocier un ou plusieurs fichiers d'un groupe de fichiers : 

1. Sélectionnez le ou les fichiers à dissocier dans la fenêtre "Liste des fichiers de l'installation". 
	Pour sélectionner plusieurs fichiers simultanément, maintenez la touche Ctrl ou Maj enfoncée lors de la sélection des fichiers.

2. Affichez l'onglet "Option" de la fenêtre "Description du fichier".

3. Sélectionnez "&lt;Aucun groupe&gt;" dans la combo "Groupe".



<a name="NOTE3_6"></a>


### Visualiser les fichiers associés à un groupe de fichiers
<a name="visualiser_les_fichiers_associes_groupe_fichiers_ELTPARAGRAPHE000120"></a>Pour visualiser les fichiers associés à un groupe de fichiers, dans la fenêtre "Liste des fichiers de l'installation", sélectionnez le groupe voulu dans la combo "Groupe affiché".

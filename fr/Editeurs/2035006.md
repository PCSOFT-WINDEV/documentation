## Comment le faire ?
<a name="comment_faire_ELTTEXTE000231"></a>


### Créer un diagramme de cas d'utilisation
<a name="creer_diagramme_cas_utilisation_ELTPARAGRAPHE000039"></a>Pour créer un diagramme de cas d'utilisation :

1. Cliquez sur ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_nouveau.gif) parmi les boutons d'accès rapide. 

	- La fenêtre de création d'un nouvel élément s'affiche : cliquez sur "Architecture" puis sur "UML".

	- L'assistant de création d'un modèle UML se lance.




2. Sélectionnez l'option "Diagramme de cas d'utilisation".

3. Spécifiez le modèle UML dans lequel le diagramme sera inséré.

4. Choisissez d'éditer directement le diagramme.

5. Validez la création du diagramme de cas d'utilisation. Le modèle créé devient le modèle en cours.



<a name="NOTE2_2"></a>


### Créer un cas d'utilisation
<a name="creer_cas_utilisation_ELTPARAGRAPHE000061"></a>Pour créer un cas d'utilisation : 

1. Sous le volet "Diagramme de cas d'utilisation", dans le groupe "Création", cliquez sur "Cas d'utilisation". Un nouveau cas d'utilisation apparaît dans le modèle UML.

2. Sélectionnez l'option "Description" du menu contextuel (clic droit) du nouveau cas d'utilisation. La fenêtre de spécifications du cas d'utilisation s'ouvre.

3. Spécifiez :

	- le nom du cas d'utilisation. Ce nom est affiché sous l'éditeur d'UML.

	- le libellé du cas d'utilisation décrivant l'intérêt du cas d'utilisation. Ce libellé sera utilisé dans le dossier d'impression de l'éditeur d'UML.




4. Validez la description du cas d'utilisation.



<a name="NOTE2_3"></a>


### Créer un acteur
<a name="creer_acteur_ELTPARAGRAPHE000088"></a>Pour créer un acteur : 

1. Sous le volet "Diagramme de cas d'utilisation", dans le groupe "Création", cliquez sur "Acteur".

2. Sélectionnez l'option "Description" du menu contextuel (clic droit) du nouvel acteur. La fenêtre de spécifications de l'acteur s'ouvre.

3. Spécifiez :

	- le nom de l'acteur. Ce nom est affiché dans l'acteur sous l'éditeur d'UML.

	- le libellé de l'acteur décrivant l'intérêt de l'acteur.




4. Validez la description de l'acteur.



<a name="NOTE2_4"></a>


### Créer une relation
<a name="creer_une_relation_ELTPARAGRAPHE000115"></a>Pour créer une relation :

1. Selon le type de relation à créer, sous le volet "Diagramme de cas d'utilisation", dans le groupe "Création" :

	- cliquez sur "Généralisation" pour créer une généralisation.

	- cliquez sur "Association" pour créer une association.

	- cliquez sur "Dépendance" pour créer une dépendance.




2. Renseignez les informations demandées.

3. Validez la description de la relation.



<a name="NOTE2_5"></a>


### Créer un paquetage
<a name="creer_paquetage_ELTPARAGRAPHE000138"></a>Pour créer un paquetage : 

1. Sous le volet "Diagramme de cas d'utilisation", dans le groupe "Création", cliquez sur "Paquetage".

2. Sélectionnez l'option "Description" du menu contextuel (clic droit) du nouveau paquetage. La fenêtre de spécifications du paquetage s'ouvre.

3. Spécifiez :

	- le nom du paquetage. Ce nom est affiché dans le paquetage sous l'éditeur d'UML.

	- le libellé du paquetage décrivant l'intérêt du paquetage.




4. Validez la description du paquetage.

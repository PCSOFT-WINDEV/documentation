


## Le guide 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000128"></a>
Le guide est un puissant moteur de recherche intégré à WINDEV, WEBDEV et WINDEV Mobile, permettant de rechercher tous les documents référencés concernant un domaine.

Ainsi, une recherche effectuée sur le mot "Carte bancaire" permet d'obtenir rapidement les différentes pages d'aide, les différents assistants, composants, superchamps ou exemples disponibles.

Cette page d'aide présente comment :

- Afficher le guide.

- Faire une recherche dans le guide.

- Utiliser un élément du guide.




De plus, le guide est entièrement personnalisable : vous pouvez ajouter vos propres mots-clés et documents dans cette base de données. Pour plus de détails, consultez [Personnaliser le guide](../Editeurs/2033002.md).

## Comment le faire ?
<a name="comment_faire_ELTTEXTE000152"></a>


### Afficher le guide
<a name="afficher_guide_ELTPARAGRAPHE000031"></a>Pour afficher le guide, sous le volet "Accueil", dans le groupe "Aide en ligne", déroulez "Aide" et sélectionnez "Consulter le guide".
<a name="NOTE2_2"></a>


### Faire une recherche dans le guide
<a name="faire_une_recherche_dans_guide_ELTPARAGRAPHE000050"></a>Une recherche dans le guide peut se faire soit par mot-clé, soit par thème. Pour cela, affichez le guide.

Pour faire une recherche dans le guide par mot-clé :

1. Saisissez un mot-clé.

2. Pour effectuer une recherche sur ce mot-clé, cliquez sur GO.

3. Pour effectuer une recherche sur plusieurs mots-clé, ajoutez (à l'aide des flèches) le mot-clé à la liste des critères de recherche et validez. Cette recherche combine les mots-clés sélectionnés et trouve tous les éléments référencés avec ces mots-clés.

4. Pour obtenir plus de résultats, cliquez sur le bouton "+ de résultats" : tous les documents correspondant à l'un ou l'autre des mots-clés sont alors affichés.<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Guide_sansnum.gif)





Pour faire une recherche dans le guide par catégorie :

1. Sélectionnez le type de document recherché.

2. Sélectionnez la catégorie voulue. La liste des documents correspondants est affichée.



<a name="NOTE2_3"></a>


### Utiliser un élément référencé dans le guide
<a name="utiliser_element_reference_dans_guide_ELTPARAGRAPHE000070"></a>Selon le type d'élément référencé dans le guide, les méthodes pour l'utiliser varient.

Dans tous les cas, il faut sélectionner l'élément.

Ainsi pour :

- Visualiser une page d'aide, une vidéo, un truc et astuce, cliquez sur le bouton "Ouvrir".

- Utiliser un assistant de code, effectuez votre recherche depuis l'éditeur de code et cliquez sur le bouton "Ouvrir".

- Visualiser un exemple, cliquez sur "Exécuter".

- Ouvrir un exemple, cliquez sur "Ouvrir".

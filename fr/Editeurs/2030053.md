


## Fichier Sitemap pour Google
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000119"></a>
Pour mieux analyser le contenu des sites, le moteur de recherche de Google utilise s'il est disponible un fichier XML nommé "Sitemap".

Ce fichier, présent sur le serveur qui héberge le site, décrit les liens sur les pages à référencer. Ce fichier est automatiquement généré à l'installation d'un site par WEBDEV. Ce fichier est disponible uniquement sur les sites AWP, les sites statiques et les sites PHP.

Ce fichier est complémentaire de l'assistant de référencement et permet de mieux référencer les pages d'un site. Par exemple, si une page n'a pas de lien utilisable par un robot (par exemple un lien contenant du code JavaScript), elle est décrite dans ce fichier (le lien est converti dans un format analysable par les moteurs) et donc référencée.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000143"></a>
Pour créer un fichier Sitemap : 

1. Créez votre site statique, PHP ou AWP.

2. Pour chaque page de votre site, indiquez :

	- si la page doit être référencée ou non (option "Interdire le référencement" dans l'onglet "Détail" de la description de la page). Si la page doit être référencée, elle apparaîtra dans le fichier Sitemap.

	- les expressions et mots-clés associées à chaque page référençable du site (onglet "Détail" de la fenêtre de description de la page). Ces mots-clés seront utilisés par les robots pour référencer les pages.

	- la fréquence de mise à jour de votre page (tous les jours, toutes les semaines, ...). Si vous ne savez pas, il est conseillé de mettre &lt;Non défini&gt; plutôt qu'une indication fausse.

	- la priorité interne de la page (élevée, faible, ...). Cette priorité permet de proposer en premier une page de votre site plutôt qu'une autre lors des recherches.




3. Installez votre site à l'aide des options d'installation prévues par WEBDEV :

	- pour un site WEBDEV AWP, utilisez l'assistant de création d'installation : sous le volet "Projet", dans le groupe "Génération", déroulez "Déployer le site" et sélectionnez "Générer et déployer le site".

	- pour un site WEBDEV statique ou un site PHP, utilisez WDDéploie : sous le volet "Outils", dans le groupe "Utilitaires Web", cliquez sur "WDDeploie".







Le fichier "Sitemap.xml" est automatiquement généré sur le poste serveur lors de l'installation à la racine du répertoire &lt;Nom_Projet&gt;_WEB, ainsi que à la racine de chaque répertoire des langues.

Remarques : 

- Sur le poste de développement, un fichier "_SiteMap.xml" est généré. Ce fichier contient les références aux pages intégrées dans le fichier SiteMap. Ce fichier n'est pas utilisable car il est automatiquement complété / modifié lors de l'installation du site.

- Si le site contient un [plan du site](../WDChamp/1013300.md) avec des URL avec paramètres, ces URL sont automatiquement référencées dans le fichier SiteMap.

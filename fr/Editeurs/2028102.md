


## Effacement des patchs
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000086"></a>
Pour modifier rapidement le fonctionnement d'une application, il est possible d'utiliser des [patchs d'application](../Editeurs/2030031.md). Lors de l'installation de l'application, il est possible d'effacer les bibliothèques de patch existantes. 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Effacer les bibliothèques de patch
<a name="effacer_les_bibliotheques_patch_ELTTEXTE000110"></a>
La configuration des bibliothèques de patch est réalisée sous l'éditeur d'installation WDInst. Pour lancer WDInst, sous le volet "Outils", dans le groupe "Utilitaires", cliquez sur "WDInst".

Pour effacer les bibliothèques de patch : 

1. Sélectionnez l'option "Paramètres de l'installation .. Effacement des patch". 

2. Grâce au bouton "Ajouter", sélectionnez les bibliothèques de patch à supprimer du poste client.  

3. Validez.

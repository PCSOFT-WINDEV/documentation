


## Le volet "Rechercher - Remplacer"
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000096"></a>
Le volet "Rechercher - Remplacer" permet de :

- Rechercher une expression spécifiée dans l'ensemble d'un projet.

- Remplacer la ou les expressions trouvée(s) par une autre expression. 

- Rechercher une expression spécifiée dans plusieurs projets.



![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Rechercher_Volet1.gif&type=thumb)
Pour plus de détails, consultez [Recherche dans les éditeurs](../Editeurs/9000150.md). 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Afficher le volet "Rechercher - Remplacer"
<a name="afficher_volet_rechercher_remplacer_ELTTEXTE000120"></a>
**Pour afficher le volet "Rechercher - Remplacer"**, sous le volet "Accueil", dans le groupe "Environnement", déroulez "Volets" et sélectionnez "Volets" puis "Rechercher - Remplacer".

Pour plus de détails sur les manipulations des volets ancrables, consultez [Manipuler les volets ancrables](../Editeurs/2027001.md). 

Remarque : Le volet "Rechercher - Remplacer" peut aussi bien être positionné horizontalement que verticalement.

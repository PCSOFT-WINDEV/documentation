## Présentation
<a name="presentation_ELTTEXTE000142"></a>


### Qu'est-ce qu'un métatype ?
<a name="questce_quun_metatype_ELTPARAGRAPHE000011"></a>Un métatype est une combinaison de caractéristiques d'une rubrique et du champ relié.

Par exemple un métatype "Fax" contiendra le type et la longueur de la rubrique, le masque de saisie, le cadrage, ...

Cette information sera ensuite reprise dans tous les modules, pour chaque action. La définition de l'analyse est ainsi simplifiée ainsi que le processus de génération RAD, la création automatique d'un champ, ...

Il est possible :

- [soit d'utiliser un métatype proposé sous l'éditeur d'analyses](#NOTE2_1).

- [soit de créer ses propres métatypes](#NOTE2_2).

## Comment le faire ?
<a name="comment_faire_ELTTEXTE000166"></a>


### Utiliser un métatype proposé par l'éditeur d'analyses
<a name="utiliser_metatype_propose_par_editeur_analyses_ELTPARAGRAPHE000034"></a>Pour créer une rubrique à partir d'un métatype dans l'éditeur d'analyses :

1. Sous l'éditeur d'analyses, affichez la description des rubriques du fichier de données voulu (double-cliquez sur le fichier de données par exemple).

2. Cliquez sur l'icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=MLD_Metatype_ico.gif)
.

3. La fenêtre suivante s'affiche :
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=MLD_Metatype.gif&type=thumb)


4. Indiquez un mot-clé dans la combo et cliquez sur le bouton "Rechercher". Lorsque le métatype voulu est trouvé, cliquez sur son image. Ses caractéristiques sont affichées.

5. Validez. La rubrique correspondante est automatiquement créée.




Remarque : Il est également possible lors de la création de la rubrique, de sélectionner l'option &lt;Métatype&gt; dans la liste des noms de rubrique disponibles.


<a name="NOTE2_2"></a>


### Créer un métatype qui sera proposé par l'éditeur d'analyses
<a name="creer_metatype_qui_sera_propose_par_editeur_analyses_ELTPARAGRAPHE000053"></a>Pour créer un métatype dans l'éditeur d'analyses :

1. Sous l'éditeur d'analyses, affichez la description des rubriques du fichier de données voulu (double-cliquez sur le fichier de données par exemple).

2. Cliquez sur l'icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=MLD_Metatype_ico.gif)
.

3. La fenêtre de gestion des métatypes s'affiche. 

4. Cliquez sur le bouton "Nouveau" (sous l'option "Métatypes utilisateurs") pour créer un métatype.

5. Dans la fenêtre à onglets qui s'affiche, indiquez :

	- dans l'onglet "Général" : 

		- Nom du métatype.

		- Description du métatype.

		- Image du métatype.




- dans l'onglet "Paramètres" : 

	- le type et le sous type de la rubrique (ces informations correspondent aux options sélectionnées pour la rubrique de base du métatype).

	- la valeur par défaut.

	- les infos partagées (ces informations correspondent aux options sélectionnées pour la rubrique de base du métatype).

6. Validez la création du métatype. Le métatype est enregistré dans un fichier au format XML. Par défaut, ce fichier est enregistré dans le sous-répertoire "\\Personnel\\Metatype" de WINDEV, WEBDEV ou WINDEV Mobile. Ce métatype pourra ensuite être importé dans l'éditeur d'analyses d'un autre poste.





<a name="NOTE2_3"></a>


### Importer un métatype dans l'éditeur d'analyses
<a name="importer_metatype_dans_editeur_analyses_ELTPARAGRAPHE000084"></a>Pour importer un métatype utilisateur dans l'éditeur d'analyses :

1. Sous l'éditeur d'analyses, sélectionnez l'option "Analyse .. Importer un métatype utilisateur".
	Remarque : Pour accéder à cette option, utilisez le menu principal accessible par le logo de la version du produit : 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Ruban_logo_menu17.gif)


2. Sélectionnez le fichier XML correspondant au métatype à importer. Ce fichier XML a été créé lors de la création du métatype.




## Importation d'éléments dans un projet
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000110"></a>
Tous les éléments manipulés par un projet (fenêtre, état, classes, collection de procédures, requêtes, ...) peuvent être transférés d'un projet à un autre, pour un même produit. L'importation permet d'ajouter au projet en cours des éléments provenant d'un autre projet. 

Par exemple, le projet WINDEV en cours peut ainsi importer des fenêtres WINDEV d'un autre projet WINDEV. 

Remarque : Il est également possible de réaliser des importations entre des projets appartenant à différents produits. Pour plus de détails consultez : 

- [Importer un projet WINDEV](../Editeurs/2030002.md)

- [Importer des fenêtres WINDEV](../Editeurs/2030001.md)




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000134"></a>
Pour importer des éléments d'un projet dans le projet en cours (les deux projets utilisent le même produit) : 

1. Sous le volet "Projet", dans le groupe "Projet", déroulez "Importer" et sélectionnez "Des éléments XXX et leurs dépendances" (où XXX représente le nom du produit en cours, WINDEV, WEBDEV ou WINDEV Mobile).

2. Sélectionnez le répertoire contenant les fichiers à importer. La liste des éléments est automatiquement affichée. Cette liste est composée :

	- du nom du fichier,

	- de la description du fichier (si elle existe).




3. Sélectionnez les éléments à importer. Une coche apparaît devant les fichiers sélectionnés.

4. Validez. Les fichiers sélectionnés sont automatiquement importés dans le répertoire du projet en cours et intégrés à la liste des éléments du projet.




<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Notes
<a name="notes_ELTTEXTE000158"></a>


### Dépendance des fichiers
<a name="dependance_des_fichiers_ELTPARAGRAPHE000054"></a>Les éléments sont importés avec leurs dépendances : tous les fichiers liés à l'élément importé sont également importés. Par exemple, la fenêtre et son image de fond sont automatiquement importées.
<a name="NOTE3_2"></a>


### Estimation de la taille des fichiers importés
<a name="estimation_taille_des_fichiers_importes_ELTPARAGRAPHE000061"></a>Il est possible d'estimer la taille des fichiers importés avec le bouton "Calculer". Cette taille prend en compte la taille de l'élément à importer et la taille des fichiers dépendant de cet élément.

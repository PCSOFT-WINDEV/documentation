


## WDInst : gérer les fichiers à installer 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000257"></a>
L'éditeur d'installation WDInst permet de gérer les fichiers à installer :

- [Visualiser la liste des fichiers](#NOTE2_1),

- [Ajouter un ou plusieurs fichiers](#NOTE3_1),

- [Supprimer un ou plusieurs fichiers](#NOTE4_1).




La gestion des fichiers à installer est réalisée sous l'éditeur d'installation WDInst. Pour lancer WDInst, sous le volet "Outils", dans le groupe "Utilitaires", cliquez sur "WDInst".

L'éditeur d'installation WDInst propose une gestion avancée des fichiers à installer par rapport à l'assistant de création du programme d'installation.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Visualiser la liste des fichiers à installer
<a name="visualiser_liste_des_fichiers_installer_ELTTEXTE000281"></a>
Les fichiers à installer peuvent être visualisés dans la fenêtre "Liste des fichiers de l'installation" de WDInst.<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Parametrer_caract_installation_fichier%20-%20HC%20N%B0001%201.gif&type=thumb)

<a name="NOTE2_2"></a>
Deux types de fichiers sont affichés dans cette liste :

- ***Fichiers individuels***. Le fichier est identifié par son nom complet (nom et extension) sur le poste de développement. Ce fichier sera intégré dans le programme d'installation (s'il est présent dans le répertoire d'origine au moment de la génération).

- ***Fichiers génériques***. Le nom du fichier contient des caractères jokers (\*,?). Tous les fichiers présents dans le répertoire d'origine et correspondant à ce filtre seront intégrés dans le programme d'installation lors de la génération.




Par exemple :<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=TypeFichier.gif)
Pour visualiser les fichiers associés à un [groupe de fichiers](../Editeurs/2028010.md), sélectionnez le groupe voulu dans la combo "Groupe affiché".
<a name="NOTE2_3"></a>


### Trier la liste des fichiers à installer
<a name="trier_liste_des_fichiers_installer_ELTPARAGRAPHE000065"></a>Pour trier la liste des fichiers à installer, cliquez sur l'entête de la colonne correspondant au critère de tri (nom du fichier, répertoire d'origine, répertoire de destination, ...).
<a name="NOTE2_4"></a>


### Modifier le répertoire de destination d'un fichier présent dans la liste des fichiers à installer
<a name="modifier_repertoire_destination_fichier_present_dans_liste_des_fichiers_installer_ELTPARAGRAPHE000072"></a>Le répertoire de destination correspond au répertoire dans lequel le fichier sera installé.

Pour définir le répertoire de destination, il est possible d'utiliser les mots-clés suivants :

- **&lt;RepInst&gt;** : répertoire d'installation choisi par l'utilisateur lors de l'installation.

- **&lt;LectInst&gt;** : lecteur du répertoire d'installation choisi par l'utilisateur lors de l'installation.

- **&lt;FicInst&gt;** : répertoire d'installation des fichiers de données. L'éditeur d'installation WDInst permet de paramétrer ce répertoire. Pour plus de détails, consultez [Paramétrer le répertoire des fichiers de données](../Editeurs/2028017.md).

- **&lt;RepWin&gt;** : répertoire de Windows des postes utilisateurs.

- **&lt;RepSys&gt;** : répertoire système.

- **&lt;RepRef&gt;** : répertoire de référence pour les installations avec mise à jour automatique uniquement.




Par exemple : "&lt;RepInst&gt;\\MonRépertoire".

Pour modifier le répertoire de destination d'un fichier présent dans la liste des fichiers à installer :

1. Sélectionner le fichier voulu.

2. Cliquez dans la colonne "Répertoire de destination".

3. Saisissez le nouveau répertoire de destination.



<a name="NOTE2_5"></a>


### Connaître et modifier les caractéristiques d'un fichier présent dans la liste des fichiers à installer
<a name="connaitre_modifier_les_caracteristiques_fichier_present_dans_liste_des_fichiers_installer_ELTPARAGRAPHE000099"></a>Les différentes caractéristiques du fichier sélectionné dans la liste peuvent être visualisées et modifiées dans la fenêtre "Description du fichier" de WDInst.

Pour connaître les différents éléments de l'interface de WDInst, consultez l'[interface de WDInst](../Editeurs/2028022.md)

Dans cette fenêtre, il est possible de :

- [Paramétrer les caractéristiques des fichiers à installer](../Editeurs/2028033.md).

- [Paramétrer les caractéristiques de l'installation des fichiers](../Editeurs/2028004.md).

## Installer un ou plusieurs fichiers supplémentaires
<a name="installer_plusieurs_fichiers_supplementaires_ELTTEXTE000329"></a>


### Par défaut, l'installation sélectionne l'ensemble des fichiers présents dans le répertoire "EXE" du projet.
<a name="par_defaut_installation_selectionne_ensemble_des_fichiers_presents_dans_repertoire_exe_projet_ELTPARAGRAPHE000121"></a>L'éditeur d'installation WDInst permet d'ajouter des :

- Fichiers individuels.

- Fichiers génériques.




Ces types de fichiers ont été présentés dans le paragraphe précédent.
<a name="NOTE3_2"></a>


### Ajouter un ou plusieurs fichiers individuels
<a name="ajouter_plusieurs_fichiers_individuels_ELTPARAGRAPHE000133"></a>Pour ajouter un ou plusieurs fichiers individuels :

1. Sélectionnez l'option "Edition .. Ajouter des fichiers". La fenêtre d'ajout de fichiers s'ouvre.

2. Sélectionnez le lecteur et le répertoire dans lesquels le ou les fichiers à ajouter sont présents. La liste des fichiers présents dans le répertoire s'affiche.

3. Pour filtrer les fichiers affichés en fonction d'un critère de recherche, saisissez le critère de recherche dans l'option "Filtre" ("\*.DLL" ou "\*client\*" par exemple) et cliquez sur le bouton "Filtrer". 
	**Remarque** : Les caractères jokers (\*,?) sont autorisés.

4. Cochez l'option "un ou plusieurs fichiers".

5. Sélectionnez le ou les fichiers à ajouter. 
	Pour sélectionner plusieurs fichiers simultanément, maintenez la touche Ctrl ou Maj enfoncée lors de la sélection des fichiers.

6. Spécifiez le répertoire de destination du ou des fichiers à installer.
	Pour définir le répertoire de destination, il est possible d'utiliser les mots-clés suivants :

	- **&lt;RepInst&gt;** : répertoire d'installation choisi par l'utilisateur lors de l'installation.

	- **&lt;LectInst&gt;** : lecteur du répertoire d'installation choisi par l'utilisateur lors de l'installation.

	- **&lt;FicInst&gt;** : répertoire d'installation des fichiers de données. L'éditeur d'installation WDInst permet de paramétrer ce répertoire. Pour plus de détails, consultez [Paramétrer le répertoire des fichiers de données](../Editeurs/2028017.md).

	- **&lt;RepWin&gt;** : répertoire de Windows des postes utilisateurs.

	- **&lt;RepSys&gt;** : répertoire système.

	- **&lt;RepRef&gt;** : répertoire de référence pour les installations avec mise à jour automatique uniquement.
			Par exemple : "&lt;RepInst&gt;\\MonRépertoire".




7. Pour associer le(s) fichier(s) à installer à un [groupe de fichiers](../Editeurs/2028010.md), cliquez sur le bouton "Autre" et spécifiez le groupe de fichiers à associer.

8. Validez l'ajout du ou des fichiers.



<a name="NOTE3_3"></a>


### Ajouter un ou plusieurs fichiers correspondant à un fichier générique
<a name="ajouter_plusieurs_fichiers_correspondant_fichier_generique_ELTPARAGRAPHE000170"></a>Pour ajouter un ou plusieurs fichiers correspondant à un fichier générique :

1. Sélectionnez l'option "Edition .. Ajouter des fichiers". La fenêtre d'ajout de fichiers s'ouvre.

2. Sélectionnez le lecteur et le répertoire dans lesquels le ou les fichiers à ajouter sont présents.

3. Cochez l'option "un fichier générique".

4. Saisissez le fichier générique ("\*.BMP" par exemple).
	**Remarque** : Les caractères jokers (\*,?) sont autorisés.

5. Pour sélectionner également les fichiers présents dans les sous-répertoires du répertoire spécifié, cochez l'option "Ajouter récursivement tous les fichiers correspondants".

6. Spécifiez le répertoire de destination du ou des fichiers à installer.
	Pour définir le répertoire de destination, il est possible d'utiliser les mots-clés suivants :

	- **&lt;RepInst&gt;** : répertoire d'installation choisi par l'utilisateur lors de l'installation.

	- **&lt;LectInst&gt;** : lecteur du répertoire d'installation choisi par l'utilisateur lors de l'installation.

	- **&lt;FicInst&gt;** : répertoire d'installation des fichiers de données. L'éditeur d'installation WDInst permet de paramétrer ce répertoire. Pour plus de détails, consultez [Paramétrer le répertoire des fichiers de données](../Editeurs/2028017.md).

	- **&lt;RepWin&gt;** : répertoire de Windows des postes utilisateurs.
			Par exemple : "&lt;RepInst&gt;\\MonRépertoire".




7. Pour associer le(s) fichier(s) à installer à un [groupe de fichiers](../Editeurs/2028010.md), cliquez sur le bouton "Autre" et spécifiez le groupe de fichiers à associer.

8. Validez l'ajout du ou des fichiers.

## Supprimer un ou plusieurs fichiers à installer
<a name="supprimer_plusieurs_fichiers_installer_ELTTEXTE000365"></a>
Pour supprimer un ou plusieurs fichiers à installer :

1. Sélectionnez le ou les fichiers à supprimer dans la fenêtre "Liste des fichiers de l'installation".
	Pour sélectionner plusieurs fichiers simultanément, maintenez la touche Ctrl ou Maj enfoncée lors de la sélection des fichiers.

2. Sélectionnez l'option "Édition .. Supprimer" (ou appuyez sur la touche Suppr).

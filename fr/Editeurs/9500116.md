


## Fonctionnalité automatique de copie de formulaire
			





## Présentation
<a name="presentation_ELTTEXTE000096"></a>
Lors de la saisie de plusieurs exemplaires d'un formulaire à la suite, il peut arriver qu'une partie des informations à saisir soit la même pour chaque instance du formulaire. Par exemple, dans le cas de la saisie d'une série d'articles, les nom et référence des articles changent à chaque fois mais le nom du fournisseur peut rester le même pendant un grand nombre de saisies.

Afin d'accélérer les manipulations, il est possible de demander à la fenêtre en cours de copier les informations saisies dans les différents champs vers le presse-papiers afin de la coller à chaque nouvelle saisie.



## Copier les données du formulaire
<a name="copier_les_donnees_formulaire_ELTTEXTE000120"></a>
Pour copier les données de la fenêtre en cours dans le presse-papiers, utilisez (au choix) :

- L'option "Copier le formulaire" du menu contextuel de la fenêtre.

- Le raccourci clavier Ctrl + Alt + C.

- La fonction [FenCopieFormulaire](../WDLang1/1000018992.md).

- La fonction [FAAExécute](../WDLang1/1000022099.md) de la façon suivante :
	
	```wl
	FAAExécute(MaFenêtre, faaCopieFormulaire)
	```







## Coller les données du formulaire
<a name="coller_les_donnees_formulaire_ELTTEXTE000144"></a>
Pour coller les données contenues dans le presse-papiers dans la fenêtre en cours, utilisez (au choix) :

- L'option "Coller le formulaire" du menu contextuel de la fenêtre.

- Le raccourci clavier Ctrl + Alt + V.

- La fonction [FenColleFormulaire](../WDLang1/1000018993.md).






## Remarques
<a name="remarques_ELTTEXTE000168"></a>
Seuls les champs en saisie dans la fenêtre sont modifiés lors d'un collage de formulaire. Les champs en affichage seulement ou grisé conservent leur valeur.

Tous les champs sont copiés dans le presse-papiers lors d'une copie de formulaire. Il est possible de coller le résultat dans un bloc-notes par exemple. Par contre, il n'est possible de copier du texte depuis un bloc-notes pour le coller dans une fenêtre.

Lors du collage, aucun traitement n'est exécuté (traitement de modification ou de sortie de champ par exemple). La fenêtre n'est pas informée que les valeurs de ses champs ont changé. Si cette fonctionnalité automatique est activée dans une application, il convient de vérifier la validité des champs au moment de la validation de la fenêtre (et non pas en sortie de champ).

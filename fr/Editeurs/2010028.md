


## Propriétés d'un style (système d'aide)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000104"></a>
Pour éditer les styles disponibles dans le système d'aide en cours : sous le volet "Modification", dans le groupe "Styles", cliquez sur "Editer les styles". La liste des styles disponibles apparaît.

Lors de la création ou de la modification d'un style, la fenêtre des propriétés d'un style s'affiche. Cette fenêtre est composée de 2 onglets :

- Description de la police,

- Description du paragraphe.




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Description de la police
<a name="description_police_ELTTEXTE000128"></a>
La fenêtre de description de la police est la suivante : <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Style_Paragraphe.gif)


Dans cette fenêtre, il est possible de sélectionner :

- le nom de la police utilisée,

- la taille de la police,

- la couleur de la police (une couleur personnalisée peut être choisie),

- le style de la police.




<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Description de paragraphe
<a name="description_paragraphe_ELTTEXTE000152"></a>
La fenêtre de description de paragraphe est la suivante : <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Style_Police.gif)
 
Dans cette fenêtre, il est possible de sélectionner :

- les marges de gauche et droite du paragraphe,

- une puce en début de paragraphe,

- l'espacement entre les lignes du paragraphe.

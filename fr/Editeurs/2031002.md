## Présentation
<a name="presentation_ELTTEXTE000252"></a>


### ATTENTION : A partir de la version 12, le RAD a été complètement refait. Cette page de documentation est conservée pour compatibilité. 
<a name="attention_partir_version_12_rad_ete_completement_refait_cette_page_documentation_est_conservee_pour_compatibilite_ELTPARAGRAPHE000011"></a>

**Pour plus de détails sur les nouveaux modèles du RAD, consultez [Nouveaux patterns RAD](../Editeurs/2031020.md).**

Le RAD (Rapid Application Development) permet de créer rapidement des fenêtres à partir de modèles prédéfinis. Par défaut, 10 modèles de fenêtre RAD sont proposés par WINDEV.

Chaque modèle correspond à un fichier Script, d'extension .MDL. Ce fichier comprend :

- La description du modèle RAD, affiché lors du lancement de l'assistant de création de fenêtres

- Les différentes étapes de l'assistant

- La description de la fenêtre générée et son code

- La description des champs générés et leur code

## Les fichiers modèles
<a name="les_fichiers_modeles_ELTTEXTE000276"></a>
Tous les fichiers utilisés par le RAD (RAD fenêtre ou projet) sont disponibles dans le sous-répertoire "\\Personnel\\Modeles\\RAD" de WINDEV.

Ce répertoire contient pour chaque modèle du RAD :

- **Un fichier BMP "RAD_FEN_xx"** :
	image correspondant à l'icône du type de fenêtre générée, XX correspond au numéro du modèle RAD.

- **Un fichier MDL** contenant le script du modèle RAD.

- Les fichiers des différentes classes WINDEV (.WDC) utilisées par les modèles du RAD Objet

## Modifier ou créer un modèle RAD
<a name="modifier_creer_modele_rad_ELTTEXTE000300"></a>


### Modifier un modèle RAD existant
<a name="modifier_modele_rad_existant_ELTPARAGRAPHE000148"></a>Pour modifier un modèle RAD existant : 

1. Dans le répertoire "\\Personnel\\Modeles\\RAD" de WINDEV, créez une copie de sauvegarde de tous vos fichiers script (extension mdl).

2. Ouvrez le fichier modèle à modifier sous le bloc-notes (notepad.exe) de Windows.

3. Modifiez le code du script selon vos besoins. Pour plus de détails sur le code du script, consultez [La structure des modèles RAD](../Editeurs/2031001.md).

4. Enregistrez les modifications

5. Relancez WINDEV. Les modifications réalisées dans le fichier de modèle seront automatiquement prises en compte.



<a name="NOTE3_2"></a>


### Créer un nouveau modèle RAD à partir d'un modèle existant
<a name="creer_nouveau_modele_rad_partir_modele_existant_ELTPARAGRAPHE000164"></a>Pour créer un nouveau modèle RAD à partir d'un modèle existant : 

1. Dans le répertoire "\\Personnel\\Modeles\\RAD" de WINDEV, créez une copie de sauvegarde de tous vos fichiers script (extension mdl).

2. Renommez le fichier "Copie de ..." du fichier modèle servant de base à votre nouveau modèle.

3. Ouvrez ce fichier modèle sous le bloc-Notes (notepad.exe) de Windows.

4. Modifiez le code suivant selon vos besoins :



```txt
#############################
# nom du modèle
#############################
Nom=Fiche avec fiche reliée
Titre=Fiche et fiche
Numero=11
#############################
# description du modèle
#############################
Description=Debut
Modèle Fiche avec Fiche reliée.
Ce modèle est adapté au fichiers reliés par une liaison simple.
Description=Fin
```



|   |   |
| --- | --- |
| Nom | Nom du modèle, affiché dans l'assistant de création de fenêtres. |
| Titre | Titre de la fenêtre générée. |
| Numéro | Numéro du modèle RAD. Ce numéro doit être modifié pour créer un nouveau modèle RAD. |
| Description = Début<br>Description = Fin | Texte de description du modèle affiché dans l'assistant de création de fenêtres. |

**Remarque** : le code des fichiers MDL n'est pas indenté et ne peut pas être indenté.

**5.** Modifiez le code du script selon vos besoins. Pour plus de détails sur le code du script, consultez [La structure des modèles RAD](../Editeurs/2031001.md).
**6.** Enregistrez les modifications.
**7.** Créez si nécessaire les images associées au nouveau modèle RAD (RAD_Fen_XX.BMP où xx correspond au numéro du modèle).
**8.** Relancez WINDEV. Le nouveau modèle apparaît lors du lancement de l'assistant de création de fenêtres.

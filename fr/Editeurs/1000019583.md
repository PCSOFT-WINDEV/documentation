


## Fabrique Logicielle : Options de l'administrateur
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000139"></a>
Plusieurs options sont paramétrables dans l'administrateur de la Fabrique Logicielle : 

- Authentification

- Coordinateur

- Automates




Pour paramétrer ces options : 

1. Lancez l'administrateur de la Fabrique Logicielle : sous le volet "Outils", dans le groupe "Utilitaires", cliquez sur "WDFAdmin".

2. Dans la fenêtre qui s'affiche, indiquez le nom du coordinateur auquel l'administrateur doit se connecter. 

3. L'administrateur de la Fabrique Logicielle se lance. 

4. Cliquez sur l'onglet "Options" pour réaliser le paramétrage de l'administrateur de la Fabrique Logicielle.

## Authentification - Paramètres des Centres de Contrôle
<a name="authentification_parametres_des_centres_controle_ELTTEXTE000163"></a>
Les paramètres d'authentification permettent de définir la base de données des comptes utilisée par la Fabrique Logicielle. La Fabrique Logicielle utilise les comptes des Centres de Contrôle. Il est nécessaire d'indiquer les options de connexion à cette base de données. Cette base de données peut être : 

- une base de données en mode partagé. Il suffit d'indiquer le répertoire de partage contenant les fichiers de données de la base de données des Centres de Contrôle. 

- une base de données HFSQL Client/Serveur. Il suffit d'indiquer le serveur et son port ainsi que la base utilisée. 
	Remarque : le bouton "Tester" permet de vérifier les paramètres saisis. 

- une plateforme Cloud. Il suffit d'indiquer le nom de la plateforme utilisée. 
	Remarque : Il est possible de créer directement une plateforme Cloud si nécessaire.




L'option "Demander le mot de passe au démarrage" permet de forcer l'identification de l'utilisateur au démarrage de l'administrateur de la Fabrique Logicielle.

## Coordinateur
<a name="coordinateur_ELTTEXTE000187"></a>
Le paramétrage du coordinateur permet de : 

- Choisir le coordinateur.

- Définir les options du coordinateur (lien "Options du coordinateur"). 




Les options du coordinateur concernent : 

- La base des Centres de Contrôle utilisée pour stocker les plans d'actions et mémoriser leurs comptes-rendus. 

- Le paramétrage de l'historique des résultats. Il est possible de fixer le nombre de résultats conservés pour chaque plan d'action. 

- Le paramétrage de la méthode d'envoi des comptes-rendus des plans d'action (messagerie interne des Centres de Contrôle, email, etc.). 

- Le paramétrage des destinataires en cas de défaillance d'un automate.

## Automates
<a name="automates_ELTTEXTE000211"></a>
Les options concernant les automates sont les suivantes : 

- **Modifier l'ordre de priorité des automates** : Lorsque plusieurs machines sont disponibles, il est possible de sélectionner une machine spécifique pour exécuter les plans d'action (par exemple la machine la plus rapide). 
	Rappel : Il est possible d'ajouter des automates dans l'onglet "Automates" de l'administrateur de la Fabrique Logicielle.

- **Gestion des boîtes de dialogue "Ne plus afficher"** : Cette option permet de réactiver si nécessaire les boîtes de dialogues précédemment désactivées (en cochant l'option "Ne plus demander" par exemple).

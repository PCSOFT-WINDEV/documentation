


## Editeur d'images
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000163"></a>
WINDEV, WEBDEV et WINDEV Mobile proposent un éditeur d'images intégré. 

Un ruban et un volet permettent d'accéder aux principales fonctionnalités de cet éditeur. 

Remarque : Cette page d'aide présente comment lancer l'éditeur d'images. Pour plus de détails sur les fonctionnalités disponibles, consultez [Editeur d'images : Fonctionnalités](../Editeurs/1410086606.md). 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Lancer l'éditeur d'images
<a name="lancer_editeur_images_ELTTEXTE000187"></a>
L'éditeur d'images peut être ouvert : 

- pour afficher une image ou une icône existante, 

- pour créer une nouvelle image ou une nouvelle icône. 



<a name="NOTE2_2"></a>


### Afficher une image existante
<a name="afficher_une_image_existante_ELTPARAGRAPHE000033"></a>Pour afficher une image existante, il est possible de : 

- utiliser le volet "Images" présent sous l'éditeur de WINDEV, WEBDEV et WINDEV Mobile. Par défaut, ce volet liste les images présentes dans le répertoire du projet en cours (ou dans le répertoire _WEB pour un projet WEBDEV). Il est également possible d'ouvrir une image présente sur le disque. Pour plus de détails, consultez [Le volet "Images"](../Editeurs/2027036.md). 

- utiliser le menu contextuel du champ affichant l'image : 

	- Affichez le menu contextuel du champ (champ Image ou Bouton par exemple). 

	- Sélectionnez l'option "Editer l'image".




- utiliser la fenêtre de description du champ : dans l'onglet "Général", déroulez ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Menu_Image_Editeur%20-%20HC%20N%B0001.gif)
 à droite du chemin de l'image et sélectionnez l'option "Editer l'image".




L'image sélectionnée s'affiche sous l'éditeur d'images et peut être modifiée. 
<a name="NOTE2_3"></a>


### Créer une image (ou une icône)
<a name="creer_une_image_une_icone_ELTPARAGRAPHE000054"></a>Pour créer une image dans l'éditeur d'images, il est possible : 

- d'utiliser le volet "Images" présent sous l'éditeur de WINDEV, WEBDEV et WINDEV Mobile. Pour plus de détails, consultez [Le volet "Images"](../Editeurs/2027036.md).

- d'utiliser la fenêtre de création d'un nouvel élément : 

	1. Cliquez sur ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ICO_Nouveau.gif)
 parmi les boutons d'accès rapide. 

	2. Dans la fenêtre qui s'affiche, cliquez sur "Atelier" puis sur "Image". 

	3. Une fenêtre permettant de définir l'image à créer s'affiche. Il est possible de créer : 

		- une image vierge ou une image basée sur une image du catalogue d'images. 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Editeur_img_nouveau%20-%20HC%20N%B0001.gif)


		- une icône vierge ou une icône basée sur une icône du catalogue d'images. 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Editeur_img_nouveau%20-%20HC%20N%B0002.gif)

4. Validez. Si nécessaire, choisissez l'image de base et ses caractéristiques dans le catalogue d'images. 

5. Une nouvelle image apparaît sous l'éditeur d'images.

## Enregistrer une image via l'éditeur d'images
<a name="enregistrer_une_image_via_editeur_images_ELTTEXTE000223"></a>


### Enregistrer une image modifiée
<a name="enregistrer_une_image_modifiee_ELTPARAGRAPHE000082"></a>L'enregistrement d'une image modifiée sous l'éditeur d'images est réalisée via l'icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ICO_Enregistrer.gif)
 du ruban. 

Plusieurs cas peuvent se présenter : 

- Si l'image ne contient aucun calque (image simple modifiée directement ou image aplatie), l'image est enregistrée automatiquement dans son format d'origine avec le même nom. L'image existante est donc directement modifiée (et mise à jour dans vos fenêtres, états, etc. utilisant cette image). 

- Si l'image contient au moins un calque, l'image peut être enregistrée sous différents formats : 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Editeur_img_sauver%20-%20HC%20N%B0001.gif)


	- **Image au format PNG uniquement**. Les calques seront perdus.

- **Image aux formats PNG et WDPIC**. Le nom du fichier image au format WDPIC est de la forme : "nomimageoriginal.extensionoriginale.wdpic". L'image au format WDPIC contient la description de tous les calques et de toutes les modifications effectuées sur l'image originale. L'image originale n'est pas modifiée. Lors du chargement d'une image sous l'éditeur d'images, si cette image possède un équivalent au format WDPIC, le fichier .wdpic est chargé automatiquement dans l'éditeur d'images.

- **Image au format WDPIC** uniquement. 
			Dans ce cas, l'image originale n'est pas modifiée. 
			L'image au format WDPIC peut être associée aux différents éléments de vos projets, comme tout autre format d'images.




Remarque : Une fois le format choisi, l'enregistrement des modifications suivantes est réalisé au même format. 


### Enregistrer une nouvelle image ou icône
<a name="enregistrer_une_nouvelle_image_icone_ELTPARAGRAPHE000105"></a>Si l'éditeur d'images vous a permis de créer une image ou une icône, l'icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ICO_Enregistrer.gif)
 du ruban permet d'enregistrer cette image ou cette icône. La fenêtre de sauvegarde d'un élément du projet s'affiche et propose d'enregistrer : 

- l'image au format "wdpic" par défaut. 

- l'icône au format "ico" par défaut. 




Pour modifier le format d'enregistrement de l'image, cliquez sur le bouton de sélection "..." et modifiez le format d'enregistrement. Les formats disponibles sont : WDPIC, TIF, ICO, PNG, JPEG, BMP et GIF. 

Remarque : Une fois le format choisi, l'enregistrement des modifications suivantes est réalisé au même format.

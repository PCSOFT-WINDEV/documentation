
## Volet "Analyseur Temps Réel"
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000098"></a>
Le volet "Analyseur Temps Réel" permet de visualiser pendant le test d'un élément l'activité de votre application ou de votre site. 


![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Analyseur_temps_r%E9el%20-%20HC%20N%B0001%204.gif&type=thumb)


Le volet "Analyseur temps réel" permet de : 

- suivre les différents threads de l'application. 

- connaître à tout moment le code exécuté. 

- accéder au code exécuté, ... 




Pour plus de détails, consultez [Analyseur Temps réel](../Editeurs/9500237.md). 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Afficher le volet "Analyseur Temps Réel"
<a name="afficher_volet_analyseur_temps_reel_ELTTEXTE000122"></a>
**Pour afficher le volet "Analyseur Temps Réel"**, sous le volet "Accueil", dans le groupe "Environnement", déroulez "Volets" et sélectionnez "Volets" puis "Analyseur Temps Réel".

Ce volet peut également être affiché directement via le bouton ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Partie2_PileDesAppels%201.gif)
 du volet "Débogueur", lors du débogage d'une application ou d'un site. 

Pour plus de détails sur les manipulations des volets, consultez [Manipuler les volets ancrables](../Editeurs/2027001.md).

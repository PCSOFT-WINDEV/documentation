## Présentation
<a name="presentation_ELTTEXTE000179"></a>
L'assistant de création du programme d'installation permet de personnaliser les fenêtres du programme d'installation en choisissant l'image de fond. 

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Installation_WD_defaut%20-%20HC%20N%B0001%201.gif)


Remarque : Il est également possible d'afficher une fenêtre de fond derrière la fenêtre d'installation. Les caractéristiques de cette fenêtre doivent être paramétrées dans l'étape "Titre et logo" de l'assistant. 







## Paramétrage
<a name="parametrage_ELTTEXTE000203"></a>


### Choix de l'image de décor
<a name="choix_image_decor_ELTPARAGRAPHE000070"></a>L'image de décor correspond à l'image de fond de la fenêtre de l'assistant. 

Il est possible d'utiliser : 

- aucune image (cas par défaut),

- une image spécifique ayant pour dimensions 708 x 460 pixels.

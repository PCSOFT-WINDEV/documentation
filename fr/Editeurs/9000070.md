


## Menu "?"
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000342"></a>
WINDEV propose en standard un menu d'aide adapté à vos applications. Ce menu permet aux utilisateurs de vos applications d'accéder directement à plusieurs fonctionnalités.

Ce menu s'intègre automatiquement à la fin du menu principal de votre application.

Par exemple : <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=MenuUtilitaire.GIF&type=thumb)


**Remarque** : Le menu "?" est également automatiquement proposé dans le menu contextuel système de la fenêtre (option "?").

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000366"></a>


### Intégrer automatiquement le menu "?"
<a name="integrer_automatiquement_menu_ELTPARAGRAPHE000025"></a>Pour intégrer automatiquement le menu "?" :

1. Ouvrez la fenêtre dans laquelle le menu "?" doit être ajouté.
	Cette fenêtre doit contenir un menu principal. Pour plus de détails, consultez [Menu principal d'une fenêtre](../WDChamp/1010010.md).

2. Sous le volet "Fenêtre", dans le groupe "Barres et menus", déroulez "Menu principal" et sélectionnez "Ajouter le menu '?'". L'assistant de création du menu utilitaire se lance.
	Passez au plan suivant de l'assistant.

3. Sélectionnez les options que le menu "?" doit proposer : <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=MenuUtilitaireAssistant.Gif&type=thumb)


4. Répondez aux diverses questions et terminez l'assistant.
	Le menu "?" ajoute automatiquement la fenêtre interne "FI_MenuHWinDevHelp" dans votre application. Le menu contextuel de cette fenêtre correspond au menu "?". Pour visualiser ce menu contextuel, cliquez sur "?" dans le menu de la fenêtre et sélectionnez l'option "Ouvrir le menu '?'".




**Remarque** : Si le composant WDRules a été intégré dans votre projet, il peut être supprimé.
<a name="NOTE2_2"></a>


### Options proposées par le menu "?"
<a name="options_proposees_par_menu_ELTPARAGRAPHE000061"></a>Le menu "?" propose les options suivantes :

- *** Nouveau dans la version*** : affiche la liste des modifications apportées à la version en cours. Cette liste a été spécifiée lors de la création de la version client de l'application. Pour plus de détails, consultez [Information des utilisateurs sur les évolutions](../Editeurs/2028084.md).

- *** Aide de l'application*** : lance l'aide (au format HLP ou CHM) de votre application. Ce fichier d'aide peut être créé automatiquement avec WINDEV.
	Sélectionnez le fichier d'aide à afficher ou choisissez de générer automatiquement l'aide de l'application.

- *** Aide des FAA (Fonctionnalités Automatiques de l'Application)*** : lance l'aide des Fonctionnalités Automatiques de vos Applications (au format PDF). Ce document permet aux utilisateurs de votre application de connaître les fonctionnalités automatiquement proposées dans les applications WINDEV. Ce fichier PDF est également présent dans le sous-répertoire "Aide" du répertoire d'installation de WINDEV et peut être modifié si nécessaire.
	Cette option ajoute automatiquement le composant "CCMenu" dans l'application. En effet, cette option utilise une procédure de ce composant pour fonctionner.

- *** Le saviez-vous...*** : affiche une fenêtre de type "Astuce du jour". 
	Cette fenêtre présente aux utilisateurs finaux différentes astuces d'utilisation d'une application WINDEV. Ces astuces correspondent à différentes FAA (Fonctionnalités Automatiques de vos Applications) disponibles en WINDEV. Ces astuces sont également lancées par défaut au démarrage de l'application.
	Pour plus de détails, consultez [Gestion du "Le saviez-vous"](../Editeurs/9500111.md). 
	Cette option ajoute automatiquement le composant "TipOfTheDay" dans l'application. En effet, cette option utilise une procédure de ce composant pour fonctionner.

- *** Outil de réplication***  : lance l'outil de réplication universelle assistée (ReplicSynchro). Cette option est conseillée uniquement si la [réplication universelle assistée](../WDLang4/9000045.md) a été mise en place dans l'application.
	Cette option ajoute automatiquement le composant "CCMenu" dans l'application. En effet, cette option utilise une procédure de ce composant pour fonctionner.

- *** Activer le débogage à distance*** : permet de déboguer l'application à distance. Cette option permet de lancer l'outil WDDebug sur le poste de l'utilisateur. L'utilisateur peut ainsi autoriser le [débogage à distance de l'application](../Editeurs/2019021.md). Cette option ajoute automatiquement le composant "CCMenu" dans l'application. En effet, cette option utilise une procédure de ce composant pour fonctionner.

- *** Activer l'audit d'exécution*** : permet de débuter l'enregistrement d'un rapport d'audit d'exécution contenant les asserts, les erreurs, l'occupation de la mémoire... Cette option enregistre les différents éléments collectés dans un fichier ".waudit" qui pourra ensuite être traité par le développeur de l'application.

- *** Créer une image de l'occupation mémoire*** : enregistre l'état de l'application en mémoire dans un fichier ".wmem" qui pourra être ouvert ultérieurement par le développeur de l'application.

- ***Créer, modifier un état...*** : lance le [logiciel Etats & Requêtes](../Presentation/3088003.md) afin de créer ou de personnaliser un état de l'application. Si le logiciel Etats & Requêtes n'est pas installé sur le poste en cours, une fenêtre s'affiche permettant à l'utilisateur de télécharger ce logiciel ou de l'installer depuis un emplacement spécifique.

- *** Créer, modifier une requête...*** : lance le [logiciel Etats & Requêtes](../Presentation/3088003.md) afin de créer ou de personnaliser une requête de l'application. Si le logiciel Etats & Requêtes n'est pas installé sur le poste en cours, une fenêtre s'affiche permettant à l'utilisateur de télécharger ce logiciel ou de l'installer depuis un emplacement spécifique.

- *** Envoyer une suggestion, un dysfonctionnement, une règle métier...*** : ouvre un assistant permettant à l'utilisateur d'envoyer un email au développeur de l'application pour lui signaler tous les dysfonctionnements rencontrés, lui faire des suggestions et lui proposer des modes opératoires. Les dysfonctionnements rencontrés, les suggestions et les règles métier proposées pourront être récupérés par le Centre de Suivi de Projets et/ou par le volet des règles métier.
	Cette option ajoute automatiquement :

	- le [composant "Feedback"](../Editeurs/2014013.md) dans l'application. 

	- le projet de travail correspondant à l'application WINDEV en cours dans le Centre de Suivi de Projets.
			Saisissez une adresse email pour recevoir ces informations.
			Pour plus de détails sur le fonctionnement de cette option, consultez [Gestion des retours Clients](../Editeurs/2014013.md).




- *** Consulter les demandes envoyées*** : affiche l'ensemble des suggestions, dysfonctionnements et règles métier envoyés par l'utilisateur en cours. Selon le filtre appliqué, les demandes correspondent à l'application en cours, à certaines applications ou à toutes les applications.
	Cette option peut être sélectionnée uniquement si l'option précédente est sélectionnée. Pour plus de détails, consultez [Gestion des retours Clients](../Editeurs/2014013.md).

- *** Mise à jour de l'application*** : permet de mettre à jour l'application (une installation réseau de l'application est nécessaire). Cette mise à jour peut être réalisée soit par le réseau local, soit par une page Web.
	Cette option ajoute automatiquement le composant "CCMenu" dans l'application. En effet, cette option utilise une procédure de ce composant pour fonctionner.

- *** Restaurer une version de l'application...*** : permet de revenir à une version antérieure de l'application. Ce choix permet, par exemple, dans le cas d'un dysfonctionnement de l'application de revenir à une version ne présentant pas le dysfonctionnement. Pour utiliser cette option, l'application doit être en [installation réseau et avoir activé l'historique des versions](../Editeurs/9000111.md).

- *** Licence d'utilisation*** : affiche la licence (au format texte, HLP ou CHM) de l'application.
	Cette option ajoute automatiquement le composant "CCMenu" dans l'application. En effet, cette option utilise une procédure de ce composant pour fonctionner. Le fichier de licence utilisé est celui défini lors de la génération de la procédure d'installation de l'application.

- *** Enregistrement du produit...*** : permet d'ouvrir une page Web indépendante afin de gérer l'enregistrement de la licence.
	Saisissez la page Web à lancer.

- *** A propos*** : ouvre la fenêtre "A propos" de l'application. Cette fenêtre peut être créée automatiquement avec WINDEV.
	Sélectionnez la fenêtre à afficher ou choisissez de générer automatiquement une fenêtre "A propos". Pour générer automatiquement cette fenêtre, sélectionnez ses différentes caractéristiques. Pour plus de détails, consultez [Fenêtre A propos](../Editeurs/9000088.md).

### Modifier les options proposées par le menu "?"
<a name="modifier_les_options_proposees_par_menu_ELTPARAGRAPHE000222"></a>Pour modifier les options proposées par le menu "?" :

1. Ouvrez la fenêtre contenant le menu "?".

2. Cliquez sur l'option "?".

3. Pour modifier :

	- les options proposées par défaut : 
			sélectionnez l'option "Assistant d'édition du menu '?'" et réalisez les modifications voulues. 
			Il est également possible d'utiliser le ruban : sous le volet "Fenêtre", dans le groupe "Barres et menus", déroulez "Menu principal" et sélectionnez "Edition assistée du menu '?'".

	- le look des options proposées, leur code, leur position, ... ou pour ajouter de nouvelles options personnels : 
			sélectionnez l'option "Ouvrir le menu '?'" et utilisez le menu contextuel de l'option à modifier (clic droit). 
			Il est également possible d'utiliser le ruban : sous le volet "Fenêtre", dans le groupe "Barres et menus", déroulez "Menu principal" et sélectionnez "Ouvrir le menu '?'".






<a name="NOTE2_4"></a>


### Supprimer le menu "?"
<a name="supprimer_menu_ELTPARAGRAPHE000268"></a>Pour supprimer le menu "?" :

1. Ouvrez la fenêtre contenant le menu "?".

2. Affichez le menu contextuel de l'option "?" (clic droit) et sélectionnez l'option "Supprimer".

3. Supprimez la fenêtre interne "FI_MenuHWinDevHelp" de l'application (par exemple, option "Supprimer du disque" du menu contextuel, dans le volet "Explorateur de projet").

4. Supprimez si nécessaire les composants utilisés par le menu :

	- le composant "CCMenu".

	- le composant "Feedback".

	- le composant "TipOfTheDay".





Ces composants peuvent être supprimés par exemple dans le volet "Explorateur de projet", grâce par l'option "Supprimer" du menu contextuel du composant.
<a name="NOTE2_5"></a>


### Menu "?" et groupware utilisateur
<a name="menu_groupware_utilisateur_ELTPARAGRAPHE000286"></a>Certaines options du menu automatique sont accessibles uniquement par le superviseur (et grisées si l'utilisateur n'est pas le superviseur). Ces options sont les suivantes :

- Optimiser la vitesse d'accès à la base de données.

- Sauvegarder les données.

- Sauvegarder l'application.

- Réparation de la base de données.




Pour qu'un utilisateur non superviseur accède à ces fonctions, il suffit de modifier les droits de la fenêtre interne FI_MenuHWinDevHelp qui contient le menu contextuel.

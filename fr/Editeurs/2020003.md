<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000099"></a>
Vous pouvez simplement personnaliser tous les assistants de WINDEV, WEBDEV ou WINDEV Mobile. Ainsi, la version de WINDEV, WEBDEV ou WINDEV Mobile installée sur votre poste utilisera votre image favorite dans tous les assistants (de code, de champs, de fenêtres, ...).

## Comment le faire ?
<a name="comment_faire_ELTTEXTE000123"></a>


### Personnaliser les assistants
<a name="personnaliser_les_assistants_ELTPARAGRAPHE000027"></a>Pour personnaliser les assistants :

1. Cliquez avec le bouton droit de la souris sur l'image de l'assistant. 

2. Dans la fenêtre qui s'ouvre, sélectionnez l'image à utiliser.




**Remarques** : 

- Le survol des différentes images change dynamiquement l'image affichée dans l'assistant.

- L'option "Choisir une image personnelle (PNG - 185x456)" permet de sélectionner un fichier image au format PNG présent sur votre poste. Ce fichier doit avoir les dimensions suivantes : 185 x 456 pixels. 

- La coche "Changer aléatoirement tous les jours" permet de varier les images utilisées. 





<a name="NOTE2_2"></a>


### Utiliser vos propres images dans les assistants
<a name="utiliser_vos_propres_images_dans_les_assistants_ELTPARAGRAPHE000046"></a>Pour utiliser vos propres images dans les assistants :

1. Créez une image au format BMP, GIF ou PNG. Cette image doit avoir les dimensions suivantes : 185 x 456 pixels.

2. Copiez cette image dans le sous-répertoire "\\Personnel\\Mon WINDEV\\Assistant" du répertoire d'installation de WINDEV, WEBDEV ou WINDEV Mobile.




## Sélection de l'application ou des fichiers à installer 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000117"></a>
WINDEV dispose d'un assistant pour générer un programme d'installation :

- pour installer une application WINDEV.

- pour installer d'autres fichiers. Par exemple, installation de goodies (images, vidéos, ...).




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Installer une application WINDEV
<a name="installer_une_application_windev_ELTTEXTE000141"></a>
L'assistant de création du programme d'installation permet de créer le programme d'installation d'une application WINDEV. Par défaut, si l'assistant ne demande pas le nom de l'application à installer, l'application à installer correspond au projet ouvert sous WINDEV.

Si l'assistant demande le nom de l'application à installer, il suffit de spécifier le projet (fichier ".WDP") à installer.

Le répertoire de base de l'application sélectionnée est automatiquement récupéré par l'assistant. Ce répertoire correspond au répertoire "EXE" du projet à installer. Par défaut, tous les fichiers présents dans ce répertoire à l'exception des fichiers de données (".FIC", ".NDX" et ".MMO") seront sélectionnés.

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Installer d'autres fichiers
<a name="installer_autres_fichiers_ELTTEXTE000165"></a>
L'assistant de création du programme d'installation (lancé depuis WDInst, l'éditeur d'installation) offre la possibilité de créer un programme d'installation associé à aucune application. Cette option permet par exemple de fournir à vos utilisateurs de nouveaux fichiers vidéo ou de nouveaux fichiers image (dans une installation indépendante de l'installation de l'application principale).

**Pour générer un programme d'installation associé à aucune application** :

1. Sous WINDEV, fermez le projet en cours : sous le volet "Accueil", dans le groupe "Général", déroulez "Fermer" et sélectionnez "Fermer le projet".

2. Ouvrez l'éditeur d'installation WDInst : sous le volet "Outils", dans le groupe "Utilitaires", cliquez sur "WDInst".

3. Sélectionnez l'option "Fichier .. Nouveau".

4. Ne spécifiez aucun projet (fichier ".WDP").

5. Cliquez sur le bouton "Avancé".

6. Sélectionnez le répertoire de base.
	Ce répertoire permet de définir la structure des répertoires à créer sur les postes utilisateurs.
	Par exemple, si le répertoire de base est "C:\\MesFichiers" et si les fichiers à installer sont présents dans le répertoire "C:\\MesFichiers\\MesImages\\ImagesFond", ces fichiers seront installés par défaut dans le sous-répertoire "MesImages\\ImagesFond" du répertoire d'installation.

7. Sélectionnez les fichiers à installer dans la suite de l'assistant.


## Création d'un exécutable : Langue de l'exécutable
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000078"></a>
Lors de la création de l'exécutable, il est possible de choisir la ou les langues à intégrer dans l'exécutable. Cette option permet par exemple de générer un exécutable différent par langue.

**Attention** : si vous choisissez de générer l'exécutable uniquement pour une seule langue, pensez à modifier si nécessaire les options de menu permettant de changer de langue.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Langues de l'exécutable
<a name="langues_executable_ELTTEXTE000102"></a>
Par défaut, seule la langue en cours sous WINDEV est proposée.

Il est possible de sélectionner :

- Une des langues utilisée par le projet,

- Toutes les langues du projet,

- Uniquement la langue en cours sous l'éditeur.




**Remarque** : La sélection d'un nombre limité de langues permet de réduire la taille de la bibliothèque (WDL) de l'application. Les configurations de projet permettent également de gérer les langues. Il est possible d'utiliser une configuration de projet pour générer une bibliothèque FR + US à partir d'un projet comportant par exemple 5 langues.

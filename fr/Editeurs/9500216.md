


## Renommer un projet
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000121"></a>
Le renommage d'un projet permet de modifier le nom du projet. Tous les éléments liés au projet avant son renommage sont conservés.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000145"></a>
Pour renommer le projet en cours :

1. Affichez le tableau de bord du projet. 

2. Sous le volet "Projet", dans le groupe "Autres actions", déroulez "Plus d'actions" et sélectionnez "Renommer le projet".

3. Indiquez le nouveau nom du projet et validez. 




Le nom du projet est automatiquement modifié. Les éléments liés au projet sont conservés.

**Remarques** :

- Seuls le nom du projet (fichier WDP, WWP ou WPP) et le nom du répertoire de compilation (répertoire .cpl) sont modifiés. 

- Les éléments suivants ne sont pas modifiés par le renommage du projet :  

	- le nom du répertoire du projet. 

	- le nom de l'analyse (si elle existe).




- Si l'exécutable a été généré avant le renommage, son nom n'est pas modifié à la prochaine génération de l'exécutable.




## Gestion des styles
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000092"></a>
L'éditeur d'aide permet de définir des styles de paragraphes. Ces styles contiennent toutes les caractéristiques d'un paragraphe :

- caractéristiques de la police (nom de la police, taille, graisse, soulignement, ...).

- caractéristiques de paragraphe (retraits, puce, alignement, espacement, ...).




Une fois définis, ces styles peuvent être utilisés sur toutes les pages d'aide du système d'aide.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment gérer les styles ?
<a name="comment_gerer_les_styles_ELTTEXTE000116"></a>
WINDEV offre la possibilité de :

- créer, modifier ou supprimer un style.

- appliquer un style à un paragraphe.
	Pour appliquer un style à un paragraphe, il suffit de sélectionner le paragraphe, puis de sélectionner le style correspondant dans la combo de styles :
	![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Combo_style.gif)


- dissocier un style : sous le volet "Modification", dans le groupe "Styles", cliquez sur "Dissocier les styles".
	Cette option permet de ne plus associer un style à un paragraphe. Pour ce paragraphe, la combo de style est alors vide. Ce paragraphe pourra ensuite être associé à un style.

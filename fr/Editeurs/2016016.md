


## Importer une feuille de styles CSS
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000095"></a>
WEBDEV offre la possibilité d'importer une feuille de styles CSS dans un projet WEBDEV.

Vous pouvez ainsi ré-utiliser des styles créés avec un autre outil et les intégrer à votre site WEBDEV.

**Remarque** : Lors de l'importation d'une page HTML existante, WEBDEV propose également d'importer les styles CSS associés à cette page. Pour plus de détails, consultez [Importation d'une page HTML existante](../Editeurs/2030029.md). 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Importer une feuille de styles existante
<a name="importer_une_feuille_styles_existante_ELTTEXTE000119"></a>
Pour importer une feuille de styles CSS existante :

1. Sous le volet "Projet", dans le groupe "Projet", déroulez "Importer" et sélectionnez "Une feuille de style CSS".

2. L'assistant d'importation se lance : ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Site%20Internet%20-%20Import%20CSS%20-%20HC%20N%B0001%201.gif)


3. Sélectionnez le mode de gestion des styles. Passez à l'étape suivante. 

4. Sélectionnez le fichier CSS contenant les styles à importer (bouton "..."). 
	La liste des styles CSS présents dans le fichier est affichée. ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Site%20Internet%20-%20Import%20CSS%20-%20HC%20N%B0002%201.gif)


5. Sélectionnez les styles à importer dans le projet (les styles cochés seront importés).

6. Validez. Les styles sont automatiquement intégrés à la feuille de styles associée au projet.

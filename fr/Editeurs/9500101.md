


## Recherche multi-champs : recherche dans une fenêtre
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000120"></a>
Dans une fenêtre, il est possible d'effectuer : 

- **Une recherche d'un mot dans un champ de saisie spécifique**. 
	Dans ce cas, utilisez l'option "Rechercher" du menu contextuel du champ. 

- **Une recherche d'un texte dans les différents champs de votre fenêtre**. 
	Dans ce cas, utilisez la combinaison de touches Ctrl + F et saisissez le mot à rechercher. 
	La recherche est effectuée : 

	- **avant la version 18** : dans les champs de saisie et les lignes de tables.

	- **à partir de la version 18** : dans tous les champs de la fenêtre. Tous les éléments correspondant à la recherche sont surlignés en jaune. Les boutons "Suivant" et "Précédent" permettent de se déplacer parmi les éléments sélectionnés.

## Modifier et personnaliser la fenêtre de recherche
<a name="modifier_personnaliser_fenetre_recherche_ELTTEXTE000144"></a>


### Présentation
<a name="presentation_ELTPARAGRAPHE000034"></a>WINDEV offre la possibilité d'intégrer la fenêtre de recherche utilisée par la FAA dans votre application. Il est ainsi possible de personnaliser par exemple :

- l'apparence de cette fenêtre (modification du gabarit par exemple).

- les langues gérées par cette fenêtre. Cette possibilité est très intéressante pour traduire cette fenêtre dans les différentes langues proposées par l'application.



<a name="NOTE2_2"></a>


### Intégration dans l'application
<a name="integration_dans_application_ELTPARAGRAPHE000044"></a>Pour intégrer la fenêtre de recherche dans votre application :

1. Sous le volet "Projet", dans le groupe "Projet", déroulez "Importer" et sélectionnez "Des éléments WINDEV et leurs dépendances".

2. Sélectionnez le sous-répertoire contenant les éléments à importer. La fenêtre de recherche est présente dans le sous-répertoire "\\Programmes\\Donnees\\Fenêtres prédéfinies\\FAA" du répertoire d'installation de WINDEV.

3. WINDEV liste les éléments présents dans le répertoire. Ce répertoire contient les différents éléments correspondant aux FAA personnalisables. Les éléments correspondant à la fenêtre de recherche sont :

	- la classe "CContexteRecherche.wdc". 

	- la fenêtre "WinDevFaa_FenetreRecherche.wdw", pour la recherche dans un champ de saisie. 

	- la fenêtre "WinDevFaa_FenetreRechercheTouteFen.wdw", pour la recherche dans toute la fenêtre. 




4. Validez. La classe et la fenêtre sont intégrées au projet. Ces éléments peuvent être modifiés. Les éléments modifiés seront automatiquement pris en compte lors de la recherche dans la fenêtre et/ou dans un champ de saisie.




**Remarques** :

- Les éléments importés ne doivent pas être renommés.

- Si vous intégrez ces éléments dans votre application, vous risquez de ne pas profiter des améliorations effectuées au cours des mises à jour. Pensez à vérifier leurs évolutions.

- Attention : Pour modifier ces éléments, les fichiers correspondants ne doivent pas être en lecture seule. Modifiez les caractéristiques des fichiers sous Windows si nécessaire.

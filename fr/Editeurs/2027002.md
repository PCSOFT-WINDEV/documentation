


## Le volet "Aperçu position fenêtres"
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000138"></a>
Le volet "Aperçu position fenêtres" permet de visualiser et/ou de modifier la position des fenêtres en cours de modification, lors de leur ouverture (selon la résolution d'écran utilisée).

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=FenApercuPosition.gif)


Le menu contextuel (clic droit) du volet "Aperçu position des fenêtres" propose les options suivantes :

- **Résolutions visibles** : Permet de visualiser une ou plusieurs résolutions utilisées lors du déploiement de l'application (résolutions "1024x768", "1152x864", ... dans l'exemple ci-dessus). Ces résolutions sont représentées par des pointillés.

- **Résolutions de référence** : Permet de visualiser une résolution de référence (résolution "1280x1024" dans l'exemple ci-dessus). Cette résolution est représentée par un trait bleu.

- **Texte résolution visible** : Affiche ou non les caractéristiques des résolutions affichées.

- Sélection du mode d'ouverture des fenêtres (centrée par rapport à l'écran, maximisée, ...). Pour plus de détails, consultez le paragraphe [Modifier le mode d'ouverture des fenêtres](#NOTE3_1).




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Afficher le volet "Aperçu position fenêtres"
<a name="afficher_volet_apercu_position_fenetres_ELTTEXTE000162"></a>
**Pour afficher le volet "Aperçu position fenêtres"**, sous le volet "Accueil", dans le groupe "Environnement", déroulez "Volets" et sélectionnez "Volets" puis "Aperçu position fenêtres".

Par exemple :

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ApercuPositionDansWinDev.gif&type=thumb)
Pour plus de détails sur les manipulations des volets, consultez [Manipuler les volets ancrables](../Editeurs/2027001.md).

<a name="NOTE3_1"></a>


## Mode et position d'ouverture des fenêtres
<a name="mode_position_ouverture_des_fenetres_ELTTEXTE000186"></a>
Les différents modes d'ouverture proposés sont les suivants :

- **Centré / écran** : la fenêtre est centrée dans l'écran.

- **Centré / mère** : la fenêtre est centrée par rapport à la fenêtre appelante.

- **Relatif à l'écran** : la fenêtre est ouverte à la position définie dans le volet "Aperçu position des fenêtres".

- **Relatif à la mère** : la fenêtre est ouverte à la position définie dans le volet "Aperçu position des fenêtres". Cette position est relative à la fenêtre appelante.

- **Maximisée** : la fenêtre est ouverte en plein écran.

- **Hors écran** : la fenêtre n'apparaît pas dans l'écran.



<a name="NOTE3_2"></a>


### Modifier le mode d'ouverture d'une fenêtre
<a name="modifier_mode_ouverture_une_fenetre_ELTPARAGRAPHE000067"></a>Pour modifier le mode d'ouverture d'une fenêtre :

1. Sélectionnez la fenêtre dans le volet "Aperçu position des fenêtres".
	**Remarque** : Pour apparaître dans le volet "Aperçu position fenêtres", les fenêtres doivent être ouvertes sous l'éditeur de fenêtres.

2. Affichez le menu contextuel (clic droit).

3. Sélectionnez le mode d'ouverture voulu.



<a name="NOTE3_3"></a>


### Définir la position d'ouverture d'une fenêtre dans le volet "Aperçu position des fenêtres"
<a name="definir_position_ouverture_une_fenetre_dans_volet_apercu_position_des_fenetres_ELTPARAGRAPHE000080"></a>Pour définir la position d'ouverture d'une fenêtre dans le volet "Aperçu position des fenêtres" :

1. Sélectionnez la fenêtre dans le volet "Aperçu position fenêtres".
	**Remarque** : Pour apparaître dans le volet "Aperçu position des fenêtres", les fenêtres doivent être ouvertes sous l'éditeur de fenêtres.

2. Maintenez le bouton gauche de la souris enfoncé.

3. Déplacez la fenêtre à la position voulue et relâchez le bouton gauche de la souris.


**Remarque** : Le mode et la position d'une fenêtre à son ouverture peuvent également être définis dans l'onglet "UI" de la fenêtre de description (option "Description" du menu contextuel de la fenêtre).

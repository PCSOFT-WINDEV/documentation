<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000179"></a>
Pour simplifier la saisie du code WLangage sous l'éditeur de code (fonctions, instructions, etc.), le mécanisme d'aide à la saisie suivant est proposé :

- **Visualisation de la syntaxe de la fonction en cours de saisie grâce à différentes bulles**. Cette aide apparaît lorsque le nom de la fonction est entièrement saisi. 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=saisie_assist%201.gif)
Si différentes syntaxes sont disponibles pour la fonction en cours, les différentes syntaxes sont affichées. 
	Pour modifier la syntaxe active, utilisez les touches Ctrl + Flèche bas ou Ctrl + Flèche haut.

- **Bulle d'aide contenant la description du paramètre survolé**. Cette bulle d'aide apparaît : 

	- lorsque le curseur de la souris est situé à la position d'un paramètre à saisir (le curseur de la souris survole la syntaxe sous forme de bulle).

	- si la combinaison de touches Alt + F1 est utilisée sur un paramètre déjà saisi. 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=saisie_assist.gif&type=thumb)





- **Bulle d'aide contenant la description du résultat**. Cette bulle d'aide apparaît : 

	- lorsque le curseur de la souris est situé sur le nom de la fonction (lors de la saisie d'une nouvelle fonction). 

	- si la combinaison de touches Alt + F1 est utilisée sur le nom d'une fonction déjà saisi. 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=saisie_assist_result.gif&type=thumb)





- **Bulle d'aide proposant les syntaxes disponibles sur les instructions WLangage**. Cette aide apparait lors de la saisie du premier terme de l'instruction WLangage. Dans certains cas, l'assistant propose également des syntaxes approchantes. Quelques exemples : 

	- Saisie du mot-clé TANTQUE : ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Aide_a_la_saisie_2%20-%20HC%20N%B0001.gif)


	- Saisie du mot-clé SI : ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Aide_a_la_saisie_2%20-%20HC%20N%B0002.gif)

## Fonctionnement
<a name="fonctionnement_ELTTEXTE000203"></a>


- L'aide à la saisie est disponible à tout moment sur tous les projets. 

- L'icône présente dans la bulle d'aide de description du paramètre en cours indique le type du paramètre attendu. Pour plus de détails sur ces icônes et leur signification, consultez le paragraphe "Notes" de cette page.

- Les différentes bulles d'aide apparaissent également lors de l'écriture de l'appel aux procédures ou méthodes documentées. Pour plus de détails, consultez l'aide sur [la documentation automatique des procédures et méthodes](../Motscles/1513006.md).

- La syntaxe active de la fonction WLangage s'affiche toujours dans la barre de message.
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=SyntaxeFonction.gif&type=thumb)


- L'éditeur de code propose également des options de complétion lors de la saisie des paramètres des fonctions par exemple. Pour plus de détails, consultez [Complétion automatique](../Editeurs/2013001.md).

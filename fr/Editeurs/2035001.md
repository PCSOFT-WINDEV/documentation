## Comment le faire ?
<a name="comment_faire_ELTTEXTE000318"></a>


### Créer un diagramme d'activité
<a name="creer_diagramme_activite_ELTPARAGRAPHE000048"></a>Pour créer un diagramme d'activité :

1. Cliquez sur ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_nouveau.gif)
 parmi les boutons d'accès rapide. 

	- La fenêtre de création d'un nouvel élément s'affiche : cliquez sur "Architecture" puis sur "UML".

	- L'assistant de création d'un modèle UML se lance.




2. Sélectionnez l'option "Diagramme d'activités".

3. Spécifiez le modèle UML dans lequel le diagramme sera inséré.

4. Choisissez d'éditer directement le diagramme.

5. Validez la création du diagramme d'activité. Le modèle créé devient le modèle en cours.



<a name="NOTE2_2"></a>


### Créer une activité
<a name="creer_une_activite_ELTPARAGRAPHE000070"></a>Pour créer une activité : 

1. Sous le volet "Diagramme d'activités", dans le groupe "Création", cliquez sur "Activité". Une nouvelle activité apparaît dans le modèle UML.

2. Sélectionnez l'option "Description" du menu contextuel (clic droit) de la nouvelle activité. La fenêtre de spécifications de l'activité s'ouvre.

3. Spécifiez :

	- le type de l'activité.

	- le nom de l'activité. Ce nom est affiché sous l'éditeur d'UML.




4. Sous le volet "Diagramme d'activités", dans le groupe "Création", cliquez sur "Barre de synchronisation". Une nouvelle barre de synchronisation apparaît dans le modèle UML.

5. Le menu contextuel (clic droit) de la barre de synchronisation permet de :

	- changer l'orientation de la barre.

	- changer la couleur de la barre.

### Créer un objet
<a name="creer_objet_ELTPARAGRAPHE000113"></a>Pour créer un objet : 

1. Sous le volet "Diagramme d'activités", dans le groupe "Création", cliquez sur "Objet". Un nouvel objet apparaît dans le modèle UML.

2. Sélectionnez l'option "Description" du menu contextuel (clic droit) du nouvel objet. La fenêtre de spécifications de l'objet s'ouvre.

3. Spécifiez :

	- le type de l'objet.

	- le nom de l'objet. Ce nom est affiché sous l'éditeur d'UML.

	- la classe de l'objet.

	- le stéréotype de l'objet.




4. Validez la description de l'objet.

2. Sélectionnez l'option "Description" du menu contextuel (clic droit) de la nouvelle émission de signal. La fenêtre de spécifications de la nouvelle émission de signal s'ouvre.

3. Spécifiez :

	- le type de l'émission.

	- le nom de l'émission. Ce nom est affiché sous l'éditeur d'UML.




4. Validez la description de l'émission de signal.

### Créer une transition
<a name="creer_une_transition_ELTPARAGRAPHE000196"></a>Pour créer une transition : 

1. Sous le volet "Diagramme d'activités", dans le groupe "Création", cliquez sur "Transition".

2. Sélectionnez les différents éléments à relier.

3. Sélectionnez l'option "Description" du menu contextuel (clic droit) de la nouvelle transition. La fenêtre de spécifications de la transition s'ouvre.

4. Spécifiez :

	- le type de la transition.

	- le nom de la transition. Ce nom est affiché sous l'éditeur d'UML.




5. Validez la description de la transition.

### Créer un paquetage
<a name="creer_paquetage_ELTPARAGRAPHE000224"></a>Pour créer un paquetage :

1. Sous le volet "Diagramme d'activités", dans le groupe "Création", cliquez sur "Paquetage". Un nouveau paquetage apparaît dans le modèle UML.

2. Sélectionnez l'option "Description" du menu contextuel (clic droit) du nouveau paquetage. La fenêtre de spécifications du paquetage s'ouvre.

3. Spécifiez :

	- le nom du paquetage. Ce nom est affiché dans le paquetage sous l'éditeur d'UML.

	- le libellé du paquetage décrivant l'intérêt du paquetage.

	- les relations du paquetage.




4. Validez la description du paquetage.




## Installation multiple d'un même site sur le même serveur
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000093"></a>
WEBDEV permet d'installer automatiquement sur un même serveur plusieurs exemplaires d'un même site, sans avoir à renommer le projet.

Il est par exemple possible pour un prestataire d'installer le même site pour différents clients sur un même serveur.

Les fichiers de données peuvent être communs ou séparés par site.

Le compte WEBDEV peut être commun ou différent, avec des paramétrages communs ou différents.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000117"></a>
Pour réaliser une installation multiple d'un même site dynamique WEBDEV sur un même serveur, il suffit de lancer l'assistant d'installation de site : sous le volet "Projet", dans le groupe "Génération", déroulez "Déployer le site" et sélectionnez "Générer et déployer le site".

Un des plans de l'assistant permet de définir le nom de déploiement du site. Il est ainsi possible d'installer sur le même serveur des sites ayant des noms de déploiement différents. La fonction [ProjetInfo](../WDLang1/3064004.md) utilisée avec la constante *piNomSiteDéployé* permet de connaître le nom réel du site

Pour plus de détails, consultez [Installation d'un site dynamique](../Editeurs/2028039.md).

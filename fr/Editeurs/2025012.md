<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000109"></a>
La mise en bibliothèque consiste à sélectionner les éléments manipulés par cette application (fenêtre, état, classes, collection de procédures, requêtes, fichiers de données...) et leurs dépendances (images utilisées dans les fenêtres par exemple). Pour plus de détails, consultez l'aide sur les [bibliothèques](../Editeurs/2030022.md).

Cet écran apparaît lors de la création de l'exécutable si vous avez choisi d'utiliser la bibliothèque principale de l'application.

**Remarque** : Pour créer l'exécutable, utilisez une des méthodes suivantes : 

- Sous le volet "Projet", dans le groupe "Génération", cliquez sur "Générer". 

- Déroulez ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_generation_exe.gif)
 parmi les boutons d'accès rapide et sélectionnez "Générer l'exécutable Windows 32 bits (exe)" ou "Générer l'exécutable Windows 64 bits (exe)".




L'assistant de création de l'exécutable permet de définir le mode de fonctionnement de l'exécutable et les modes d'utilisation de la bibliothèque.

L'écran "Fonctionnement de l'exécutable" permet d'indiquer si une bibliothèque est utilisée. Si vous avez choisi d'utiliser une bibliothèque (externe ou intégrée), l'écran "Mise en bibliothèque" dans la suite de l'assistant permet de sélectionner les éléments à intégrer dans la bibliothèque.

Rappel : 

- Bibliothèque intégrée à l'exécutable : Cette option est la plus courante. Le fichier WDL correspondant à la bibliothèque est intégré à l'exécutable. 

- Bibliothèque utilisée en externe. Dans ce cas, le fichier de la bibliothèque est un fichier WDL généré à côté de l'exécutable.

## Mise en bibliothèque : choix des éléments de la bibliothèque
<a name="mise_bibliotheque_choix_des_elements_bibliotheque_ELTTEXTE000133"></a>
La bibliothèque a par défaut le nom du projet en cours. Vous pouvez :

- **Modifier le nom de la bibliothèque** : il suffit de modifier directement le nom spécifié. 

- **Ajouter des éléments** (bouton "Ajouter"): 
	Tout type de fichier peut être ajouté à la liste des éléments insérés dans la bibliothèque : images, fichiers texte, ...

- **Supprimer des éléments** :  (bouton "Enlever")
	Les fichiers correspondants et leurs dépendances ne seront pas intégrés dans la bibliothèque.

- **Créer la bibliothèque à partir d'une description de bibliothèque existante** (bouton "WDU"). 
	Lors de la création d'une bibliothèque WINDEV ou WINDEV Mobile, un fichier ".WDU" du même nom est automatiquement créé. Ce fichier contient toutes les références des éléments intégrés à la bibliothèque. Il suffit de sélectionner le fichier WDU contenant les éléments à prendre en compte.

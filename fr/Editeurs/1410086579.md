
## Comment installer une application Universal Windows 10 App sur une tablette
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000122"></a>
La génération d'une application WINDEV Mobile de type Universal Windows 10 App rassemble tous les éléments du projet (fenêtres, code, etc.) dans une archive afin de permettre le déploiement de l'application sur un téléphone ou une tablette Windows 10 ou sur un PC Windows 10 en mode "tuile" et sa redistribution.

Lors de la génération de l'application, si vous avez choisi le mode "Générer et déployer l'application sur la machine locale", il est possible d'installer l'application directement sur une tablette Windows 10. 

Pour plus de détails : 

- sur la création d'une application Universal Windows 10 App, consultez [Développement d'applications Universal Windows 10 App](../Editeurs/1410086576.md).

- sur la génération de l'application, consultez [Générer une application Universal Windows 10 App](../Editeurs/1410086577.md).

<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000146"></a>


### Copier l'application générée sur une tablette
<a name="copier_application_generee_sur_une_tablette_ELTPARAGRAPHE000030"></a>Pour tester l'application générée sur une tablette Windows 10, il est nécessaire d'effectuer les manipulations suivantes : 

1. Générez l'application en choisissant l'option "Générer et déployer l'application sur la machine locale" dans l'assistant. 

2. Ouvrez le répertoire EXE du projet. un fichier &lt;NomProjet&gt;.appxbundle a été généré. 

3. Transférez ce fichier sur la tablette (via une clé USB par exemple). 

4. Installez si nécessaire le certificat qui a permis de générer l'application. 

	- Pour exporter le certificat depuis le gestionnaire de certificats : 

		- Utilisez la touche [Windows] [F] et saisissez "Certificat". 

		- Cliquez sur le résultat "Gérer les certificats d'ordinateur". Le gestionnaire de certificats s'ouvre. 

		- Sélectionnez le répertoire "Autorités de certification racines de confiance". 

		- Sélectionnez le certificat voulu. 

		- Affichez le menu contextuel du certificat, et sélectionnez l'option "Toutes les tâches .. Exporter". 

		- L'assistant d'exportation se lance. 




- Pour installer le certificat : 

	- Copiez le fichier ".cer" sur la tablette. 

	- Affichez le menu contextuel du fichier (clic droit) et sélectionnez l'option "Installer le certificat". L'assistant d'importation du certificat se lance. 

	- Dans l'assistant, sélectionnez l'emplacement de stockage "Ordinateur local". Cliquez sur "Suivant". 

	- Sélectionnez l'option "Placer tous les certificats dans le magasin suivant". 

	- Cliquez sur le bouton "Parcourir" et sélectionnez le magasin "Autorités de certification racines de confiance". 

	- Cliquez sur le bouton "Suivant" jusqu'à la fin de l'assistant. Un message apparaît indiquant que l'importation a réussi. 

5. Ouvrez l'interpréteur PowerShell en mode administrateur. Pour cela, sous Windows 10 : 

	- Affichez le menu de recherche grâce à la combinaison de touches [Windows] + [C]. 

	- Cliquez sur l'option "Rechercher". 

	- Dans la zone "Applications", saisissez "Powershell". 

	- Effectuez un clic droit sur le résultat "Windows PowerShell". 

	- Dans la partie basse de l'écran, cliquez sur "Exécuter comme administrateur". 




6. Dans l'interpréteur PowerShell, saisissez la ligne de commande suivante : 
	
	```txt
	Add-AppxPackage "<Chemin complet du package>"
	```

	Dans ce code, &lt;Chemin complet du package&gt; correspond au nom du package &lt;NomProjet&gt;.appxbundle. 

7. La tuile correspondant à l'application apparaît et l'application peut être utilisée.

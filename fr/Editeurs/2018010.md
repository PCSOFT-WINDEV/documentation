## Présentation

### Mode de gestion du groupware
<a name="mode_gestion_groupware_ELTPARAGRAPHE000017"></a>Deux modes d'intégration du groupware utilisateur sont disponibles :

- **l'intégration par défaut** : toute la programmation du groupware utilisateur est automatiquement intégrée à votre application.
	La fenêtre ou la page de login est automatiquement ajoutée à votre projet. La fenêtre ou la page de login apparaît lors du lancement de l'application, permettant à l'utilisateur de se connecter à l'application. Vous pouvez ainsi personnaliser cette fenêtre ou page en lui appliquant par exemple le gabarit de votre application.

- **l'intégration personnalisable** : toute la programmation du groupware utilisateur est automatiquement intégrée à votre application. 
	Toutes les fenêtres ou pages nécessaires à la gestion du groupware utilisateur (fenêtre ou page de login, fenêtres ou pages de gestion des utilisateurs) sont automatiquement ajoutées à votre projet. Vous pouvez ainsi personnaliser totalement toutes les fenêtres ou pages utilisées par le groupware utilisateur.

## Les différentes fenêtres et pages

### Les différentes fenêtres du groupware utilisateur
<a name="les_differentes_fenetres_groupware_utilisateur_ELTPARAGRAPHE000035"></a>Selon le mode d'intégration du groupware utilisateur dans votre application, une ou plusieurs fenêtres peuvent être intégrées dans votre projet. Ces fenêtres sont les suivantes.

**Remarque** : le fonctionnement de ces fenêtres est présenté en détail dans le paragraphe [Paramétrer le groupware utilisateur](../Editeurs/2018006.md).


|   |   |   |
| --- | --- | --- |
| **Fenêtre de login** | ***GPWLOGIN.WDW*** | Cette fenêtre permet à tous les utilisateurs de l'application de se connecter à l'application. Dans cette fenêtre, l'utilisateur doit saisir son nom et son mot de passe (optionnel) qui ont été préalablement définis par le superviseur de l'application.<br><br>Dès le lancement, il est possible d'utiliser le login "SUPERVISEUR" et le mot de passe "SUPERVISEUR".<br><br>**Remarque** : A partir de la version 120044, la fenêtre de login ne propose plus par défaut la liste des utilisateurs. Pour afficher la liste des utilisateurs, il suffit de passer la variable ***gbSecurite*** à <u><u><u><u>Faux</u></u></u></u> dans le code de déclaration des globales de la fenêtre GPWLOGIN. Pour obtenir cette fonctionnalité, il est nécessaire de ré-intégrer la fenêtre de login au projet. |
| **Menu du superviseur** | **GPWMenuSuperviseur. WDW** | Cette fenêtre est visualisable dans votre projet uniquement en mode personnalisé.<br><br>Cette fenêtre permet au superviseur soit de lancer l'application, soit de configurer les droits. |
| **Fiche d'un utilisateur** | **GPWFicheUtilisateur.WDW** | Cette fenêtre est visualisable dans votre projet uniquement en mode personnalisé.<br><br>Cette fenêtre permet au superviseur de saisir un nouvel utilisateur et les renseignements le concernant. |
| **Association des utilisateurs et des configurations** | **GPWAssociationConfiguration. WDW** | Cette fenêtre est visualisable dans votre projet uniquement en mode personnalisé.<br><br>Cette fenêtre permet au superviseur de gérer les utilisateurs, les groupes et les droits des utilisateurs. |
| **Détail d'une configuration** | **GPWDétailConfiguration.WDW** | Cette fenêtre est visualisable dans votre projet uniquement en mode personnalisé.<br><br>Cette fenêtre permet au superviseur de paramétrer pour chaque fenêtre les droits d'un utilisateur (ou d'un groupe) sur tous les champs, les groupes de champs et les options de menu de la fenêtre. |
| **Choix d'une configuration** | **GPWChoixConfiguration. WDW** | Cette fenêtre est visualisable dans votre projet uniquement en mode personnalisé.<br><br>Cette fenêtre permet au superviseur de copier une configuration précédemment définie (pour un groupe ou un utilisateur). |
| **Fiche configuration** | **GPWFicheConfiguration.WDW** | Cette fenêtre est visualisable dans votre projet uniquement en mode personnalisé.<br><br>Cette fenêtre permet au superviseur de créer un groupe ou un utilisateur. |
| **Historique des connexions** | **GPWHistoriqueConnexion.WDW** | Cette fenêtre est visualisable dans votre projet uniquement en mode personnalisé.<br><br>Cette fenêtre permet au superviseur de consulter l'historique des connexions (et de purger cet historique si nécessaire). |

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000126"></a>
Le volet "Analyse" liste :

- les fichiers de données et leurs rubriques présents dans l'analyse en cours. Les clés sont affichées en gras et les clés uniques sont affichées en gras et soulignées. 

- les procédures stockées, les requêtes stockées et les triggers présents dans l'analyse en cours.

- les vues SQL et les vues matérialisées présentes dans l'analyse en cours.

- les connexions décrites dans l'analyse en cours.

<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=VoletAnalyse.gif&type=thumb)
Le menu contextuel (clic droit) du volet "Analyse" permet de :

- Ouvrir l'analyse.

- Créer : 

	- un fichier de données.

	- une vue (SQL ou matérialisée).

	- un perso-dossier. Pour plus de détails, consultez [Perso-dossiers dans l'analyse](../Editeurs/2011003.md).

	- une collection de procédures stockées. 

	- une requête stockée.

	- un trigger.

	- une énumération.

	- une combinaison.




- Afficher la description de l'analyse. 

- Afficher la description des fichiers.

- Visualiser la description des fichiers.

- Gérer les connexions. 

- Générer l'analyse.

- Imprimer le dossier de l'analyse.




**Attention** : Les informations affichées correspondent à l'analyse en cours de description sous l'éditeur d'analyses. Ces informations peuvent être différentes de celles affichées dans le [volet "Explorateur de projet"](../Editeurs/2027029.md) (qui affiche les informations de la dernière analyse générée).

## Afficher le volet "Analyse"
<a name="afficher_volet_analyse_ELTTEXTE000150"></a>
**Pour afficher le volet "Analyse"**, sous le volet "Accueil", dans le groupe "Environnement", déroulez "Volets" et sélectionnez "Volets" puis "Analyse".

Pour plus de détails sur les manipulations des volets, consultez [Manipuler les volets ancrables](../Editeurs/2027001.md).

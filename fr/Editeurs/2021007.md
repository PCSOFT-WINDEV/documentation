


## Zoom dans les différents éditeurs 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000090"></a>
Tous les éditeurs de WINDEV, WEBDEV et WINDEV Mobile permettent de réaliser un zoom sur les différents éléments affichés sous l'éditeur. Il est ainsi possible de réaliser un zoom sur une fenêtre, afin par exemple de vérifier le bon alignement des champs, ...

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000114"></a>


### Spécifier un facteur de zoom précis
<a name="specifier_facteur_zoom_precis_ELTPARAGRAPHE000018"></a>Pour spécifier un facteur de zoom précis :

- Solution 1 : Sous le volet "Affichage", dans le groupe "Aide à l'édition", sélectionnez le zoom à utiliser.

- Solution 2 : Saisissez directement le facteur de zoom voulu dans la barre de message des éditeurs (case "Zoom").



![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Zoom.gif)

<a name="NOTE2_2"></a>


### Réaliser un zoom
<a name="realiser_zoom_ELTPARAGRAPHE000038"></a>Pour réaliser un zoom :

1. Appuyez sur la touche Ctrl.

2. Modifiez le facteur de zoom avec la roulette de la souris tout en maintenant la touche Ctrl enfoncée.




## Live Data : visualiser les données sous les éditeurs 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000302"></a>
Le "Live Data" consiste à visualiser le contenu des fichiers de données dans les champs en création sous les différents éditeurs du produit utilisé (éditeur de fenêtres, éditeur de pages, éditeur d'états, ...). Il est ainsi possible de configurer simplement la taille d'un champ en fonction des données pouvant être affichées dans le champ.

**Attention** : Pour utiliser le "Live Data" il est nécessaire de :

- associer une analyse au projet,

- manipuler des fichiers de données existants.

<a name="NOTE2_1"></a>


## Comment mettre en place le Live Data ?
<a name="comment_mettre_place_live_data_ELTTEXTE000332"></a>


### Mettre en place le "Live Data" sur le projet en cours (éditeur de fenêtres, de pages et d'états)
<a name="mettre_place_live_data_sur_projet_cours_editeur_fenetres_pages_etats_ELTPARAGRAPHE000025"></a>Pour mettre en place le "Live Data" sur le projet en cours (éditeur de fenêtres, de pages et d'états) : 

1. Sous le volet "Projet", dans le groupe "Projet", cliquez sur "Description". La fenêtre de description du projet s'affiche.

2. Affichez l'onglet "Live Data".

3. Cochez l'option "Activer le Live Data".

4. Pour chaque fichier de données, indiquez si nécessaire :

	- le type de fichier de données : Cette option vous permet de sélectionner le type du fichier de données à utiliser en aperçu. En effet, le fichier utilisé peut être un fichier de données HFSQL Classic, ou encore HFSQL Client/Serveur, Oracle, SQL Server, ... : il suffit de sélectionner la connexion correspondante.
			Des fichiers de données de différents types peuvent être utilisés simultanément.

	- le répertoire : Le répertoire peut correspondre à :

		- "Localisation automatique" : la localisation du fichier de données est effectuée automatiquement. Il est également possible d'utiliser le bouton "Localisation automatique".

		- "Répertoire du mode test" : les fichiers de données utilisés sont ceux définis pour le mode Test (le répertoire est défini dans l'onglet "Fichiers" de la description du projet.

		- "Répertoire défini dans l'analyse" : les fichiers de données utilisés sont ceux définis dans l'analyse.

		- "Parcourir" : permet de sélectionner le fichier de données voulu.




- le mot de passe du fichier de données.  

5. Validez.


**Remarque** : Dans ce cas, la mise en place du Live Data sera prise en compte uniquement à la prochaine ouverture des fenêtres, des pages et/ou des états.

		- "Localisation automatique" : la localisation du fichier de données est effectuée automatiquement. Il est également possible d'utiliser le bouton "Localisation automatique".

## Comment changer le répertoire des fichiers de données à utiliser pour le Live Data ?
<a name="comment_changer_repertoire_des_fichiers_donnees_utiliser_pour_live_data_ELTTEXTE000392"></a>


### Changer le répertoire des fichiers de données à utiliser pour le "Live Data"
<a name="changer_repertoire_des_fichiers_donnees_utiliser_pour_live_data_ELTPARAGRAPHE000153"></a>Pour changer le répertoire des fichiers de données à utiliser pour le "Live Data" :

1. Sous le volet "Projet", dans le groupe "Projet", cliquez sur "Description". La fenêtre de description du projet s'affiche.

2. Affichez l'onglet "Live Data".

3. Cochez l'option "Activer le Live Data".

4. Pour chaque fichier de données, indiquez la localisation du fichier. Le bouton "Localisation automatique" permet de rechercher automatiquement les fichiers. Cette recherche peut être relativement longue.

5. Validez.

## Comment spécifier le mot de passe des fichiers de données à utiliser pour le Live Data ?
<a name="comment_specifier_mot_passe_des_fichiers_donnees_utiliser_pour_live_data_ELTTEXTE000422"></a>


### Spécifier le mot de passe des fichiers à utiliser pour le "Live Data"
<a name="specifier_mot_passe_des_fichiers_utiliser_pour_live_data_ELTPARAGRAPHE000210"></a>Pour spécifier le mot de passe des fichiers à utiliser pour le "Live Data" : 
Si les fichiers de données utilisés pour le "Live Data" ont un mot de passe, il est possible de le spécifier dans les options du projet :

1. Sous le volet "Projet", dans le groupe "Projet", cliquez sur "Description". La fenêtre de description du projet s'affiche.

2. Affichez l'onglet "Live Data".

3. Cochez l'option "Activer le Live Data".

4. Pour chaque fichier de données, indiquez le mot de passe correspondant puis validez.


**Remarque** : Si le mot de passe n'est pas indiqué pour un fichier protégé, les données de ce fichier ne seront pas affichées.

## Afficher/Masquer les événements WLangage vides
<a name="affichermasquer_les_evenements_wlangage_vides_ELTTEXTE000197"></a>


### Présentation
<a name="presentation_ELTPARAGRAPHE000028"></a>L'éditeur de code permet d'afficher le code d'une multisélection (plusieurs champs, plusieurs procédures globales, plusieurs méthodes de classes, ...). Chaque élément propose un certain nombre d'événements WLangage.

L'affichage de ces événements peut rapidement devenir illisible en raison de leur nombre, alors que la plupart de ces événements sont vides.

Par défaut, les événements vides ne sont pas affichés.
<a name="NOTE2_2"></a>


### Afficher les événements WLangage vides
<a name="afficher_les_evenements_wlangage_vides_ELTPARAGRAPHE000039"></a>Pour afficher les événements WLangage vides :

1. Placez-vous sous l'éditeur de code.

2. Sous le volet "Affichage", dans le groupe "Aide à l'édition", cochez ou décochez "Afficher les traitements vides". Si les événements WLangage vides sont déjà affichés, ils sont automatiquement masqués et inversement.




Il est également possible d'utiliser la combinaison de touches Ctrl + F6.

## Afficher/Masquer les événements WLangage des modèles surchargés
<a name="affichermasquer_les_evenements_wlangage_des_modeles_surcharges_ELTTEXTE000227"></a>


### Présentation
<a name="presentation_ELTPARAGRAPHE000065"></a>Si l'élément affiché dans l'éditeur de code est associé à un modèle, il est possible d'afficher les événements WLangage des modèles surchargés.
<a name="NOTE3_2"></a>


### Afficher les événements WLangage des modèles surchargés
<a name="afficher_les_evenements_wlangage_des_modeles_surcharges_ELTPARAGRAPHE000072"></a>Pour afficher les événements WLangage des modèles surchargés :

1. Placez-vous sous l'éditeur de code.

2. Sous le volet "Affichage", dans le groupe "Aide à l'édition", cochez ou décochez "Afficher les traitements des modèles". Si ces événements sont déjà affichés, ils sont automatiquement masqués et inversement.




Il est également possible d'utiliser la combinaison de touches Maj + F6.

## Afficher/Masquer les numéros de ligne
<a name="affichermasquer_les_numeros_ligne_ELTTEXTE000257"></a>


### Présentation
<a name="presentation_ELTPARAGRAPHE000098"></a>L'éditeur de code permet d'afficher un numéro de ligne devant chaque ligne de code.

Par défaut, les numéros de ligne ne sont pas affichés.
<a name="NOTE4_2"></a>


### Afficher les numéros de ligne
<a name="afficher_les_numeros_ligne_ELTPARAGRAPHE000107"></a>Pour afficher les numéros de ligne :

1. Placez-vous sous l'éditeur de code.

2. Sous le volet "Affichage", dans le groupe "Aide à l'édition", cliquez sur "Afficher les numéros de ligne". Si les numéros de ligne sont déjà affichés, ils sont automatiquement masqués et inversement.




Il est également possible d'utiliser la combinaison de touches Ctrl + Shift + G.




## Administrateur du GDS : Fusion de bases de sources
			





## Présentation
<a name="presentation_ELTTEXTE000113"></a>
La fusion de bases de sources permet de réunir les projets de plusieurs bases de sources dans une seule base de sources.

Plusieurs cas peuvent se présenter :

- L'importation complète de la base de sources.

- L'importation partielle de la base de sources (sélection de un ou plusieurs projets).






## Comment le faire ?
<a name="comment_faire_ELTTEXTE000137"></a>


### Importer des projets d'une autre base de sources
<a name="importer_des_projets_une_autre_base_sources_ELTPARAGRAPHE000017"></a>Pour importer des projets d'une autre base de sources : 

1. Réintégrez tous les projets.

2. Lancez l'administrateur du GDS : sous le volet "GDS", dans le groupe "Base GDS", déroulez "Gérer" et sélectionnez "Administrateur du Gestionnaire de Sources". La base de sources du GDS sélectionnée sera la base de sources cible. L'administrateur du GDS doit être ouvert en mode Admin. 
	**Attention** : Cette opération doit être réalisée par un seul développeur.

3. Dans l'administrateur du GDS, sélectionnez la base de sources GDS source : sous le volet "Accueil", dans le groupe "Base de sources", cliquez sur "Importer une base". Un assistant démarre pour indiquer les paramètres d'importation.

4. Sélectionnez le type d'importation. Deux types d'importation sont possibles : 

	- importation complète 

	- importation partielle. Dans le cas d'une importation partielle, il est possible de sélectionner les projets à importer. 




5. Sauvegardez les bases de sources. Il est possible de sauvegarder les deux bases de sources "cible" et/ou "source".




**Remarques** : 

- L'importation d'une base de sources  peut être effectuée quel que soit le type de projet (WINDEV, WEBDEV ou WINDEV Mobile).

- L'importation d'une base de sources peut être relativement longue.

- L'importation d'une base de sources tient compte des [éléments partagés](../Editeurs/2038021.md). 

	- Dans le cas d'une importation complète de la base de sources, tous les partages sont recréés. 

	- Dans le cas d'une importation partielle, pour tous les projets importés en même temps leurs partages sont recréés.

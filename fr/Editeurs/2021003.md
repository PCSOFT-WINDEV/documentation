
## Rechercher / Remplacer dans le projet en cours
			






	<B><B>Avertissement</B></B>

	A partir de la version <B><B>18</B></B>,  la recherche dans un projet ou dans plusieurs projets a évolué. Pour plus de détails, consultez [Recherche dans les éditeurs](../Editeurs/9000150.md).



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000167"></a>
La fonctionnalité "Rechercher/Remplacer" permet de :

- Rechercher une expression spécifiée dans l'ensemble du projet en cours.

- Remplacer la ou les expressions trouvée(s) dans le projet en cours par une autre expression.




Remarque : il est également possible de réaliser une recherche sur plusieurs projets. Pour plus de détails, consultez [Recherche multi-projets](../Editeurs/2021009.md).



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000191"></a>


### Effectuer une recherche dans le projet en cours
<a name="effectuer_une_recherche_dans_projet_cours_ELTPARAGRAPHE000034"></a>Pour effectuer une recherche dans le projet en cours :

1. Affichez le [volet "Rechercher - Remplacer"](../Editeurs/2027013.md).

2. Saisissez l'expression à rechercher dans la liste "Rechercher".

3. Spécifiez les éléments dans lesquels la recherche doit être effectuée (liste "Dans"). Il est possible d'effectuer une recherche par défaut ou une recherche avancée.

4. Précisez les critères de recherche ("Contient", "Commence par", "Respecter la casse", ...).

5. Cliquez sur l'icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Ico_rech.gif) pour lancer la recherche. Si des éléments sont trouvés, la référence de ces éléments apparaît dans le volet "Rechercher - Remplacer". Pour accéder directement à un des éléments trouvés, double-cliquez sur la référence de l'élément.




**Remarque** : La recherche ne prend pas en compte les caractéristiques de l'analyse ou du modèle UML associé au projet.


<a name="NOTE3_2"></a>


### Effectuer un "Rechercher/Remplacer" dans le projet en cours
<a name="effectuer_rechercherremplacer_dans_projet_cours_ELTPARAGRAPHE000055"></a>Pour effectuer un "Rechercher/Remplacer" dans le projet en cours :

1. Affichez le [volet "Rechercher - Remplacer"](../Editeurs/2027013.md).

2. Saisissez l'expression à rechercher dans la liste "Rechercher".

3. Saisissez l'expression à remplacer dans la liste "Remplacer".

4. Spécifiez les éléments dans lesquels la recherche doit être effectuée (liste "Dans"). Il est possible d'effectuer soit une recherche par défaut, soit une recherche avancée.

5. Précisez les critères de recherche ("Contient", "Commence par", "Respecter la casse", ...).

6. Différentes icônes permettent de :

	- Remplacer l'expression sélectionnée.

	- Remplacer la dernière expression trouvée ou l'expression sélectionnée et rechercher l'expression suivante.

	- Remplacer la dernière expression trouvée ou l'expression sélectionnée et rechercher l'expression précédente.

	- Remplacer toutes les expressions trouvées.







**Remarque** : La recherche ne prend pas en compte les caractéristiques de l'analyse ou du modèle UML associé au projet.

## Notes
<a name="notes_ELTTEXTE000221"></a>


### Types de recherche
<a name="types_recherche_ELTPARAGRAPHE000082"></a>Plusieurs types de recherche sont disponibles :

- **Recherche par défaut** : La recherche peut être effectuée au choix soit dans le code source de l'élément actif, soit dans la totalité de l'élément actif (code source, messages du code, champs, rubriques, dépendances et options multilingues), soit dans l'ensemble du projet. Les options de la recherche par défaut sont sélectionnées directement dans le volet "Rechercher - Remplacer".

- **Recherche avancée** : La recherche avancée consiste à préciser dans quels éléments du projet la recherche doit être effectuée, ainsi que la portée de la recherche (c'est-à-dire dans quelles options de l'élément la recherche doit être effectuée). Pour effectuer une recherche avancée, il suffit de sélectionner &lt;Avancé...&gt; dans l'option "Dans" du volet "Rechercher - Remplacer".





<a name="NOTE4_2"></a>


### Critères de recherche
<a name="criteres_recherche_ELTPARAGRAPHE000093"></a>Quel que soit le type de recherche effectuée, il est possible de spécifier une combinaison des critères de recherche suivants :

- Contient : l'expression recherchée doit contenir l'expression saisie.

- Commence par : l'expression recherchée commence par l'expression saisie.

- Mot complet : l'expression recherchée correspond exactement à l'expression spécifiée.

- Respecter la casse : la recherche tient compte des majuscules et des minuscules.

- Respecter l'accentuation : la recherche tient compte des accents.

- Ignorer les commentaires : la recherche ne prend pas en compte les commentaires (commentaires de code commençant par //). 





<a name="NOTE4_3"></a>


### Marquer le résultat d'une recherche
<a name="marquer_resultat_une_recherche_ELTPARAGRAPHE000108"></a>Il est possible de marquer le résultat d'une recherche : tous les éléments trouvés dans le code correspondront alors à une marque de positionnement.

Pour marquer le résultat d'une recherche, il suffit de cliquer sur l'icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Ico_rech_marq.gif) avant de lancer la recherche.

Pour plus de détails sur les marques de positionnement, consultez [Marques de positionnement](../Editeurs/2013008.md).


## Groupware utilisateur : Configurer l'application
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000237"></a>
Lorsque le groupware utilisateur est paramétré, la configuration des utilisateurs et des accès est effectuée en exécution, en se connectant avec un compte "superviseur".

Lors du premier lancement, il est nécessaire d'utiliser l'utilisateur "Superviseur" et de choisir un mot de passe pour cet utilisateur. 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=gpu_identification_superv.gif)


Après identification, un utilisateur qui est superviseur du groupware aura le choix entre lancer l'application et configurer le groupware.
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=gpu_menu_superv.gif)


L'option "Configurer le groupware" permet d'accéder aux options d'administration du groupware utilisateur. Il est alors possible de : 

- [Configurer les utilisateurs et les groupes](#NOTE2_1). 

- [Configurer les droits](#NOTE3_1). 

- [Voir des statistiques](#NOTE4_1). 

- [Récupérer des données provenant d'un ancien groupware](#NOTE5_1). 

- [Crypter les mots de passe](#NOTE6_1).

## Configuration des utilisateurs et des groupes
<a name="configuration_des_utilisateurs_des_groupes_ELTTEXTE000261"></a>
Le groupware utilisateur permet de définir des groupes et des utilisateurs, selon des règles simples :

- un utilisateur peut appartenir à plusieurs groupes (ou à aucun).

- un groupe peut avoir plusieurs sous-groupes,

- un groupe ne peut avoir qu'un seul groupe "parent".




**Pour créer un utilisateur** : 

1. Dans la partie droite de l'écran, cliquez sur "Utilisateurs et groupes". Seul l'utilisateur "SUPERVISEUR" apparaît dans le tableau. 

2. Cliquez sur le bouton "Nouveau" situé en bas de la liste des utilisateurs. 

3. Dans l'écran qui s'affiche, saisissez les informations concernant l'utilisateur. 
	Remarques : 

	- Si vous saisissez le mot de passe, celui-ci apparaît masqué. Pour vérifier le mot de passe saisi, il suffit de cliquer sur l'icône en forme d'oeil : le mot de passe apparaît en clair tant que le bouton de la souris reste enfoncé. 

	- Le nouvel utilisateur créé peut être un utilisateur "superviseur". Dans ce cas, il pourra également créer des utilisateurs et configurer leurs droits. 




4. Cliquez sur "Enregistrer". Le nouvel utilisateur est automatiquement ajouté à la liste des utilisateurs. 


Les boutons "Modifier" et "Supprimer" permettent de manipuler les différents utilisateurs de la liste. 

**Pour créer un groupe** : 

1. Dans la partie droite de l'écran, cliquez sur "Utilisateurs et groupes". Seul l'utilisateur "SUPERVISEUR" apparaît dans le tableau. 

2. Cliquez sur le bouton "Nouveau" situé en bas de la liste des groupes.

3. Saisissez le nom du groupe. 

4. Cliquez sur "Enregistrer". Le nouveau groupe est automatiquement ajouté à la liste des groupes. 


Les boutons "Modifier" et "Supprimer" permettent de manipuler les différents groupes de la liste. 

**Pour associer un utilisateur à un groupe** : 

1. Sélectionnez l'utilisateur dans la liste des utilisateurs. 

2. Déplacez la ligne de l'utilisateur vers le groupe souhaité par un simple "Drag and Drop". 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=gpu_DragDrop.gif)





**Pour positionner un groupe comme sous-groupe d'un autre groupe** : 

1. Sélectionnez le groupe dans la liste des groupes. 

2. Déplacez le groupe sur la ligne du groupe "parent" par un simple "Drag and Drop". 




La création d'une hiérarchie "groupes/utilisateurs" permet de définir un héritage dans les droits d'accès. Par exemple, dans cette configuration : 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=gpu_UtilisateurGroupes.gif)


- le groupe "Secrétaire" héritera des droits d'accès du groupe "Commercial",

- l'utilisatrice "Loreleï" bénéficiera des droits d'accès du groupe "Secrétaire".

## Configuration des droits
<a name="configuration_des_droits_ELTTEXTE000285"></a>
La définition des droits peut être effectuée sur les groupes et les utilisateurs.

Astuce : Pour faciliter et accélérer la définition des droits d'accès, commencez toujours par définir les droits des groupes "parents", puis ceux des sous-groupes et enfin ceux des utilisateurs.

**Pour définir les droits** : 

1. Cliquez dans la partie gauche de l'écran sur "Gestion des droits". La liste des groupes, sous-groupes et utilisateurs apparaît. 

2. Sélectionnez l'utilisateur ou le groupe dont vous voulez configurer les droits et cliquez sur "Suivant". 

3. La définition des droits commence par autoriser ou interdire l'accès aux fenêtres et états de l'application : 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=gpu_droit_autoriserInterdir.gif&type=thumb)
Dans cette liste, apparaissent également les fenêtres et les états des composants utilisés par l'application. 

4. Il suffit de : 

	- Sélectionner l'élément. 

	- Cliquer sur les boutons "Autoriser l'accès" et "Interdire l'accès".
			Important : si vous interdisez l'accès à une fenêtre, pensez à bloquer tous les moyens d'accès à cette fenêtre (en grisant les boutons, options de menu, ...). Si un utilisateur a la possibilité d'ouvrir une fenêtre à laquelle il n'a pas accès, une erreur fatale sera affichée.

	- Un pictogramme (affiché en haut à droite de chaque fenêtre et état) indique le droit d'accès de la fenêtre ou de l'état en cours. 
			Remarques : 

		- Si l'accès à un élément est explicitement autorisé (option "Autoriser l'accès") pour un groupe, il peut être interdit pour un sous-groupe ou un utilisateur du groupe. 

		- Il est possible d'autoriser un élément qui est interdit dans le groupe "parent".




5. Si une fenêtre ou un état est autorisé pour le groupe ou pour l'utilisateur, il est alors possible de définir l'état de chaque champ, bouton, menu, option de menu, ... grâce au bouton "Suivant". 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=gpu_param_champ.gif&type=thumb)


	- L'état de l'élément peut être celui défini par défaut dans l'IHM ou un état particulier :

	- inactif,

	- grisé

	- invisible.

6. Lorsque le paramétrage est réalisé, validez.

## Récupération de données
<a name="recuperation_donnees_ELTTEXTE000333"></a>
Si vous avez déjà intégré le groupware utilisateur dans un projet avant les versions 18, deux possibilités s'offrent à vous :

- soit vous souhaitez conserver le groupware tel qu'il est. Il vous suffit alors de cocher l'option "Mode compatible 17 et inférieur" dans l'onglet "Intégration" de la description du groupware. 

- soit vous souhaitez bénéficier des fonctionnalités du nouveau groupware, sans perdre vos données existantes. Il vous suffit alors de passer par la récupération des données.




Pour importer des données existantes : 

1. Connectez-vous avec un utilisateur qui est superviseur (l'utilisateur "SUPERVISEUR" par exemple). 

2. Dans la fenêtre de gestion du groupware, cliquez sur "Récupération de données". 

3. Sélectionnez le type de la base de données qui contient les informations du groupware à importer. 

4. Indiquez :

	- le mot de passe des fichiers de données du groupware à importer. Si vous ne l'avez pas modifié, le mot de passe est “PCSGPW2001”.

	- les informations de localisation des fichiers (les répertoires pour des fichiers au format HFSQL Classic ou les paramètres de connexions au serveur dans les autres cas).




5. Validez.

## Crypter les mots de passe
<a name="crypter_les_mots_passe_ELTTEXTE000357"></a>
Si l'application a été déployée avec une version 200057 et supérieure, le bouton "Activer le cryptage des mots de passe" est affiché dans la fenêtre d'administration (en haut à droite). ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=GPU_Cryptage.gif)


L'utilisation du cryptage des mots de passe doit être réalisée avec précaution : 

- Les mots de passe existants seront migrés.

- Cette opération est irréversible.

- Si les fichiers des comptes utilisateurs sont partagés avec d'autres applications, toutes les applications doivent être mises à jour avec la version 200057 ou supérieure. Dans le cas contraire, il sera impossible de se connecter à l'application.




Cliquez sur le bouton "Activer le cryptage des mots de passe" pour utiliser le cryptage des mots de passe. 

Un message d'avertissement récapitulatif est affiché avant d'effectuer le cryptage des utilisateurs. Validez. 

Remarque : Lorsque les mots de passe ont été cryptés, le bouton "Activer le cryptage des mots de passe" n'est plus disponible. 

Pour plus de détails, consultez [Groupware utilisateur : Cryptage des mots de passe](../Editeurs/1000021283.md).

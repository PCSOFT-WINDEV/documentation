


## Ambiances
			





## Présentation
<a name="presentation_ELTTEXTE000138"></a>
Les ambiances permettent d'harmoniser l'aspect visuel d'un site et rendent les changements de look faciles et rapides.

Une ambiance est composée de :

- une feuille de styles contenant des styles WEBDEV.

- un répertoire d'images.

- une palette de couleurs par défaut.

- un ensemble d'états utilisés comme base pour construire les nouveaux états du projet.




Détaillons ces différents éléments : 

- La feuille de styles : 
	**Rappel** : Une feuille de styles est constituée de styles WEBDEV. Chaque style WEBDEV regroupe : 

	- un ou plusieurs styles CSS,

	- des valeurs de propriétés pour le champ.
			Un style WEBDEV s'applique à un type de champ particulier.
			Les styles d'une ambiance ont des noms normalisés. Ainsi, les ambiances sont interchangeables les unes avec les autres.




- Les images : 
	Les images livrées avec l'ambiance servent à définir l'aspect visuel des cellules, des tables, des champs onglets, etc.

- La palette de couleurs : 
	La palette de couleurs par défaut d'une ambiance peut être remplacée par n'importe quelle autre palette de couleurs. Pour plus de détails, consultez : [Palettes de couleurs](../Editeurs/2016021.md).






## Ordre d'application des ambiances
<a name="ordre_application_des_ambiances_ELTTEXTE000162"></a>
L'ambiance d'une page peut être définie à trois niveaux différents (par ordre décroissant de priorité) :

- Dans le projet.

- Dans le modèle de pages dont la page hérite des propriétés.

- Dans la page elle-même.




Il faut définir l'ambiance générale au niveau du projet. 

Si un ensemble de pages doit utiliser une ambiance particulière (par exemple les pages de "Promotion" d'un site de vente), il est possible de remplacer l'ambiance du projet : il suffit de la redéfinir au niveau d'un modèle de pages. 

Si une page particulière doit avoir une ambiance différente, cette ambiance peut être choisie au niveau de la page elle-même.



## Utilisation des ambiances
<a name="utilisation_des_ambiances_ELTTEXTE000186"></a>


### Choisir l'ambiance d'un projet
<a name="choisir_ambiance_projet_ELTPARAGRAPHE000055"></a>Pour choisir l'ambiance d'un projet :

1. Sous le volet "Projet", dans le groupe "Projet", cliquez sur "Description".

2. Sélectionnez l'onglet "Ambiance Styles".

3. Choisissez une ambiance parmi celles proposées (et sa palette de couleurs) et validez. 





### Changer l'ambiance d'un modèle de pages ou d'une page
<a name="changer_ambiance_modele_pages_une_page_ELTPARAGRAPHE000076"></a>Pour changer l'ambiance d'un modèle de pages ou d'une page : 

1. Dans le menu contextuel de la page, choisissez l'option de menu "Description".

2. Sélectionnez l'onglet "Style".

3. Cliquez sur le bouton de parcours pour sélectionner une ambiance.

4. Si l'ambiance sélectionnée est "Aucune", l'ambiance utilisée sera celle du projet.






## Mise à jour des ambiances
<a name="mise_jour_des_ambiances_ELTTEXTE000216"></a>
Si une ambiance utilisée par un projet a été mise à jour, c'est-à-dire modifiée, lors de l'ouverture de ce projet, WEBDEV propose la mise à jour du projet. 

Il n'est bien sur pas obligatoire de sélectionner cette nouvelle version de l'ambiance, l'ambiance précédente étant directement stockée dans chaque projet.


## Groupware utilisateur : Cryptage des mots de passe
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000098"></a>
Pour augmenter la sécurité du Groupware utilisateur, WINDEV et WEBDEV cryptent désormais les mots de passe des utilisateurs. 



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Mise en place
<a name="mise_place_ELTTEXTE000122"></a>
Le cryptage des mots de passe est automatiquement intégré à partir de la version 200057 lors de la mise en place du groupware utilisateur. 

Pour un projet en version 20 dont le groupware utilisateur est déjà en place, l'application doit être recompilée, déployée, et l'activation du cryptage des mots de passe doit être fait par l'utilisateur identifié en mode "Administrateur". 

**ATTENTION** : Lors de la mise en place du cryptage des mots de passe sur un projet existant : 

- Les mots de passe existants seront migrés.

- Cette opération est irréversible.

- Si les fichiers des comptes utilisateurs sont partagés avec d'autres applications ou sites, tous les sites et applications doivent être mis à jour avec la version 200057 ou supérieure. Dans le cas contraire, il sera impossible de se connecter à l'application ou au site.




Les différentes étapes sont les suivantes : 

1. Selon le mode d'intégration du groupware utilisateur : 

	- **Si le groupware utilisateur automatique est utilisé** : Recompilez l'application avec la nouvelle version de WINDEV ou WEBDEV (version supérieure ou égale à la version 200057). 

	- **Si le groupware personnalisé est utilisé** : Mettez à jour le composant interne "WDGPU" avec la nouvelle version de WINDEV ou WEBDEV (version supérieure ou égale à la version 200057) et recompilez l'application avec la nouvelle version de WINDEV ou WEBDEV (version supérieure ou égale à la version 200057). 




2. Déployez l'application. 

3. Lancez l'application, connectez-vous en mode administrateur et choisissez de configurer le groupware utilisateur. 

4. Un bouton "Activer le cryptage des mots de passe" apparaît dans la fenêtre d'administration (en haut à droite). <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=GPU_Cryptage.gif)
Cliquez sur ce bouton pour utiliser le cryptage des mots de passe. 

5. Un message d'avertissement est affiché avant d'effectuer le cryptage des mots de passe : 

	- Les mots de passe existants seront migrés.

	- Cette opération est irréversible.

	- Si les fichiers des comptes utilisateurs sont partagés avec d'autres applications, toutes les applications doivent être mises à jour avec la version 200057 ou supérieure. Dans le cas contraire, il sera impossible de se connecter à l'application.

	- Validez. 







Remarques : 

- Ces opérations sont identiques pour un projet WINDEV ou un projet WEBDEV. 

- Si vous utilisez le groupware utilisateur en mode compatible, il n'est pas possible de crypter les mots de passe. 

- Lorsque les mots de passe ont été cryptés, le bouton "Activer le cryptage des mots de passe" n'est plus disponible.

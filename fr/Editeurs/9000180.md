


## Ascenseur visuel de code
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000100"></a>
L'ascenseur visuel de code correspond à un mode de déplacement sous l'éditeur de code. 

La partie droite (ou gauche) de l'éditeur de code affiche le code en réduction.

La fenêtre de visualisation active est matérialisée. Et le déplacement de la petite fenêtre de visualisation déplace également le contenu de l'éditeur de code.<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Ascenseur_visuel_code%20-%20HC%20N%B0002.gif&type=thumb)
Ce mode de déplacement apporte précision, vitesse et confort considérable lors du scroll (du défilement). Ce mode offre également un aperçu de tous les événements et traitements. Par exemple toutes les procédures d'une collection de procédures ou toutes les méthodes d'une classe sont affichées.

L'ascenseur visuel signale : 

- les erreurs de compilation. Les lignes contenant une erreur de compilation sont signalées par un surlignage rouge. 

- les résultats de recherche. Les lignes contenant un élément recherché sont matérialisées par un surlignage orange.

## Comment le faire ?
<a name="comment_faire_ELTTEXTE000124"></a>
Pour gérer l'affichage de l'ascenseur visuel dans l'éditeur de code : 

1. Affichez les options de l'éditeur de code : Sous le volet "Accueil", dans le groupe "Environnement", déroulez "Options" et sélectionnez "Options de l'éditeur de code".

2. Dans la fenêtre qui s'affiche, sélectionnez l'onglet "Code". <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Ascenseur_visuel_code%20-%20HC%20N%B0001.gif&type=thumb)


3. Sélectionnez le type d'ascenseur voulu : 

	- Ascenseur standard.

	- Ascenseur visuel à gauche. 

	- Ascenseur visuel à droite. 




4. Validez.

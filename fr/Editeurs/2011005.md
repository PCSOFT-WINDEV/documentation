


## Entité d'un MCD
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000143"></a>
Pour regrouper des informations selon la méthode MERISE, il est nécessaire de créer des entités.

L'éditeur d'analyses permet de créer facilement des entités dans un MCD (Modèle Conceptuel des données).

Lors de la [génération du MCD en MLD](../Editeurs/2011020.md), les entités deviendront automatiquement des [fichiers](../Editeurs/2011010.md).

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Création d'une entité
<a name="creation_une_entite_ELTTEXTE000167"></a>
Pour créer une entité :

1. Sous le volet "MCD", dans le groupe "Création", déroulez "Nouveau" et sélectionnez "Nouvelle entité". La description de l'entité s'affiche.

2. Spécifiez :

	- le **nom de l'entité**. Ce nom sera le nom logique du fichier créé lors de la génération du MCD en MLD.

	- le **libellé de l'entité** résumant succinctement le sujet de l'entité.

	- si une **propriété de type "Identifiant automatique"** doit être définie dans l'entité. La valeur de cette propriété est unique pour chaque enregistrement et est automatiquement calculée.

	- les **notes** détaillant le fonctionnement de l'entité.




3. L'entité devient l'entité en cours. Il est possible de décrire automatiquement les propriétés de cette entité.

<a name="NOTE3_1"></a>


## Principales manipulations d'une entité
<a name="principales_manipulations_une_entite_ELTTEXTE000191"></a>


### Dupliquer/Copier une entité
<a name="dupliquercopier_une_entite_ELTPARAGRAPHE000056"></a>Pour dupliquer/copier une entité :

1. Sélectionnez l'entité à dupliquer.

2. Utilisez le raccourci Ctrl + C puis le raccourci Ctrl + V pour copier l'entité.

3. Dans la fenêtre de duplication, spécifiez les informations demandées et validez. L'entité dupliquée est automatiquement créée dans le MCD en cours.



<a name="NOTE3_2"></a>


### Supprimer une entité
<a name="supprimer_une_entite_ELTPARAGRAPHE000067"></a>Pour supprimer une entité :

1. Sélectionnez l'entité à supprimer.

2. Ouvrez le menu contextuel de l'entité sélectionnée (clic droit).

3. Sélectionnez l'option "Supprimer". L'entité est automatiquement supprimée du MCD. Si des relations pointaient sur cette entité, ces relations sont également supprimées.



<a name="NOTE3_3"></a>


### Renommer une entité
<a name="renommer_une_entite_ELTPARAGRAPHE000078"></a>Pour renommer une entité :

1. Sélectionnez l'entité à renommer.

2. Sous le volet "MCD", dans le groupe "Élément courant", cliquez sur "Description".

3. Modifiez le nom de l'entité et validez la description de l'entité. L'entité est automatiquement renommée.

## Configuration du serveur pour le protocole CGI (AWP)
<a name="configuration_serveur_pour_protocole_cgi_awp_ELTTEXTE000101"></a>
Pour configurer le serveur Web :

1. Affichez la fenêtre de configuration du serveur.

2. Cliquez sur le bouton "Default Virtual Settings ...".

3. Sélectionnez l'onglet "Standard CGI".

4. Définissez l'alias pour le protocole CGI :

	- Virtual : /WD280AWP

	- Actual : C:\\WEBDEV\\PROGRAMMES\\MOTEUR\\WINDOWS\\AWP (nom du répertoire d'installation du protocole CGI "WD280AWP.EXE")

	- Cliquez sur le bouton "ADD"

## Configuration du serveur pour les sites WEBDEV
<a name="configuration_serveur_pour_les_sites_webdev_ELTTEXTE000125"></a>
Pour configurer le serveur Web : 

1. Affichez la fenêtre de configuration du serveur.

2. Cliquez sur le bouton "Default Virtual Settings ...".

3. Sélectionnez l'onglet "Standard CGI".

4. Définissez pour chaque application WEBDEV l'alias des images de l'application :

	- Cliquez sur l'onglet "Alias"

		- Virtual : /&lt;Nom_Projet&gt;_WEB (par exemple MonAppli_WEB)

		- Actual : nom du répertoire des images de l'application  
						(par exemple c:\\WDPROJET\\MonAppli\\MonAppli_WEB)




- Cliquez sur le bouton "ADD"

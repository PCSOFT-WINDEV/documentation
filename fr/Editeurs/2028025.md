


## WDInst : langues gérées par l'installation 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000176"></a>
Si vos applications doivent être installées dans différents pays, l'éditeur d'installation WDInst offre la possibilité de générer une installation multilingue.

Les langues gérées par le programme d'installation seront proposées à l'utilisateur lors de l'installation de l'application.<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Installation_WD_defaut%20-%20HC%20N%B0002.gif)


<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Configurer une installation multilingue
<a name="configurer_une_installation_multilingue_ELTTEXTE000200"></a>
La configuration d'une installation multilingue est réalisée sous l'éditeur d'installation WDInst. Pour lancer WDInst, sous le volet "Outils", dans le groupe "Utilitaires", cliquez sur "WDInst".

L'assistant de création du programme d'installation permet également de configurer une installation multilingue.

Par défaut, les langues "Français" et "Américain" sont proposées à l'utilisateur lors de l'installation de l'application. Pour conserver cette gestion, configurez pour ces deux langues les éléments suivants :

- une licence, si une licence est affichée au début de l'installation de l'application.

- un fichier "Lisez-Moi", si un fichier "Lisez-Moi" est affiché à la fin de l'installation de l'application.

- un décor.

- des diapositives (WDInst uniquement).

- les libellés des raccourcis et les groupes de programmes (WDInst uniquement).




<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Gérer plusieurs langues dans le programme d'installation
<a name="gerer_plusieurs_langues_dans_programme_installation_ELTTEXTE000224"></a>
Pour gérer plusieurs langues dans le programme d'installation : 

1. Sélectionnez l'option "Paramètres de l'installation .. Décor". Les langues déjà gérées par le programme d'installation sont listées dans la zone "Langues disponibles à l'installation".

2. Cliquez sur le bouton "Ajouter" pour ajouter de nouvelles langues à gérer.
	**Remarque** : Après l'ajout d'une nouvelle langue, il est nécessaire de paramétrer les caractéristiques spécifiques à cette langue (options de décors, diapositives, ...). Le projet "WDSetup" doit également gérer ces langues. Pour plus de détails, consultez [WDSetup : Gérer des langues supplémentaires](../Editeurs/2028037.md). 

3. Cliquez sur le bouton "Supprimer" pour supprimer la langue sélectionnée dans la zone "Langues disponibles à l'installation".
	**Attention** : Toutes les caractéristiques définies pour la langue supprimée (options de décors, diapositives, ...) seront perdues.

## Configurer un programme d'installation multilingue
<a name="configurer_programme_installation_multilingue_ELTTEXTE000248"></a>
Après avoir sélectionné les différentes langues gérées dans le programme d'installation, il suffit de :

- [sélectionner une licence spécifique à chaque langue](../Editeurs/2028053.md) si nécessaire.

- [sélectionner un fichier "Lisez-Moi" spécifique à chaque langue](../Editeurs/2028053.md) si nécessaire..

- [configurer un décor spécifique à chaque langue](../Editeurs/2028030.md).

- [sélectionner des diapositives spécifiques à chaque langue](../Editeurs/2028030.md).

- [saisir le libellé des raccourcis](../Editeurs/2028004.md) (créés lors de l'installation) [dans toutes les langues proposées et de spécifier le groupe de programmes dans lequel les raccourcis doivent être créés](../Editeurs/2028004.md).
	Le libellé des raccourcis sera créé dans la langue d'exécution du programme d'installation.

- personnaliser le projet "WDSetup" si le programme d'installation doit gérer d'autres langues que le français et l'américain (langues proposées par défaut).

- Si l'installation comprend un désinstalleur, traduire les [options du désinstalleur](../Editeurs/2028087.md) dans les différentes langues. 




Pour que le programme d'installation gère des langues supplémentaires, le projet "WDSetup" doit être personnalisé avant de créer le programme d'installation. Ce projet (fourni avec les exemples complets de WINDEV) correspond au programme d'installation standard des applications WINDEV. Pour plus de détails, consultez [Utiliser un programme d'installation personnalisé](../Editeurs/2028037.md).

### Application installée en réseau
<a name="application_installee_reseau_ELTPARAGRAPHE000104"></a>Lors de la mise à jour d'une application installée en réseau, une fenêtre s'affiche au lancement de l'application depuis les postes utilisateurs. Cette fenêtre informe les utilisateurs qu'une mise à jour de l'application est disponible. Cette fenêtre permet selon le cas d'installer ou non la mise à jour.

<a name="NOTE5_1"></a>


## Lancement de l'application multilingue
<a name="lancement_application_multilingue_ELTTEXTE000278"></a>
Par défaut, une application multilingue est exécutée dans la langue principale du projet (option "Langue principale" dans l'onglet "Langues" de la description du projet).

**Pour lancer l'application installée dans la langue d'exécution du programme d'installation** : 

- récupérez la langue d'exécution du programme d'installation et enregistrez cette langue dans un fichier ".INI" ou dans la base de registre du poste d'installation. Cette opération peut être réalisée automatiquement lors de l'installation. Pour plus de détails, consultez [Gestion de la base de registre et des fichiers ".INI" lors de l'installation](../Editeurs/2028013.md).

- utilisez la fonction [Nation](../WDLang1/3054014.md) pour modifier la langue d'exécution de l'application.




Pour plus de détails sur les applications multilingues, consultez [Développer des applications multilingues](../WDLang1/3054008.md).

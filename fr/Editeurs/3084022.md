## Compilation du projet
<a name="compilation_projet_ELTTEXTE000126"></a>
Lors du développement d'un projet, le projet est régulièrement compilé. Cette compilation permet de vérifier le code saisi dans l'éditeur de code. En cas d'erreurs, les problèmes potentiels et leurs solutions sont affichés dans le volet "Erreurs de compilation".

Plusieurs types d'information peuvent être donnés :

- Erreurs de code (![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=picto_err_erreur.GIF)
).

- Warnings (![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=picto_err_warning.GIF)
).

- Informations (![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=picto_err_info.GIF)
).

- Erreurs d'UI (![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_Erreur_IHM.gif)
).




**Remarque** : Sur une machine disposant de plusieurs processeurs ou de plusieurs "core" (dual core, ...), la compilation de projets s'effectue en parallèle et utilise toute la puissance disponible. La phase de compilation est ainsi accélérée d'environ 30%.

## Les différents types d'erreurs possibles
<a name="les_differents_types_erreurs_possibles_ELTTEXTE000150"></a>


### Erreurs
<a name="erreurs_ELTPARAGRAPHE000036"></a>Les erreurs (![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=picto_err_erreur.GIF)
) signalent un problème grave (erreur de programmation, ...). Le test ou l'exécution du projet ne pourra pas fonctionner.

Les erreurs sont toujours visibles sous l'éditeur de code et doivent être corrigées pour que le code fonctionne.


<a name="NOTE2_2"></a>


### Warnings
<a name="warnings_ELTPARAGRAPHE000048"></a>Les warnings (![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=picto_err_warning.GIF)
) signalent des cas de programmation pouvant entraîner des comportements inattendus.

Les warnings sont affichés par défaut sous l'éditeur de code et n'empêchent pas l'exécution du programme.

Les warnings surviennent dans des cas de programmation qui peuvent réellement poser un problème


<a name="NOTE2_3"></a>


### Informations
<a name="informations_ELTPARAGRAPHE000062"></a>Les informations (![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=picto_err_info.GIF)
) expliquent les choix du compilateur ou présentent d'éventuels conseils pour améliorer le code.

Les informations peuvent être affichées uniquement pour comprendre par exemple pourquoi un code ne fournit pas le résultat attendu.


<a name="NOTE2_4"></a>


### Erreurs d'UI
<a name="erreurs_ELTPARAGRAPHE000074"></a>Les erreurs de compilation d'UI (![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_Erreur_IHM.gif)
) vous informent des problèmes potentiels détectés sur vos interfaces : images inexistantes, libellés tronqués, ...

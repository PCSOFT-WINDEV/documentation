## Présentation
<a name="presentation_ELTTEXTE000147"></a>
Nos études montrent que les utilisateurs finaux qui connaissent l'existence des FAA (Fonctionnalités Automatiques de vos Applications) en sont d'une part friands, mais surtout que ces FAA augmentent leur satisfaction d'utilisation de l'application et leur efficacité.

Afin de mieux faire découvrir les FAA aux utilisateurs finaux des applications, WINDEV propose les mécanismes suivants :

- Affichage d'un "Le saviez-vous" concernant les FAA au lancement de l'application. Cette fonctionnalité est désactivable par l'utilisateur.

- Présence d'un choix de menu d'affichage des astuces du jour ("Le saviez-vous") dans le menu "?" généré automatiquement.

<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000171"></a>


### Intégrer l'affichage d'un "Le saviez-vous" au lancement de l'application
<a name="integrer_affichage_saviezvous_lancement_application_ELTPARAGRAPHE000024"></a>Pour intégrer l'affichage de la fenêtre "Le saviez-vous" au lancement de votre application WINDEV :

1. Affichez la description du projet : sous le volet "Projet", dans le groupe "Projet", cliquez sur "Description".

2. Dans l'onglet "Avancé", cochez l'option "".
	**Remarque** : Le bouton "Voir les FAA proposées" permet d'ouvrir le fichier PDF contenant la présentation des différentes FAA disponibles dans une application WINDEV.

3. Validez la fenêtre de description du projet. Le composant "TipOfTheDay" est automatiquement intégré dans votre projet.




**Remarques** :

- Pour supprimer l'affichage de la fenêtre "Le saviez-vous", il suffit de décocher l'option "" (onglet "Avancé" de la fenêtre de description de projet).

- L'utilisateur a la possibilité de ne plus afficher la fenêtre "Le saviez-vous" : il suffit de cocher l'option "Ne plus montrer les astuces au démarrage". Pour réactiver l'affichage de la fenêtre, il est possible d'utiliser la touche Maj au lancement de l'exécutable. 



<a name="NOTE2_2"></a>


### Intégrer l'affichage d'un "Le saviez-vous" dans le menu d'aide automatique
<a name="integrer_affichage_saviezvous_dans_menu_aide_automatique_ELTPARAGRAPHE000066"></a>**Si votre application contient déjà un menu d'aide automatique "?"** : 

1. Dans la fenêtre contenant le menu d'aide automatique, sélectionnez l'option de menu '?'.

2. Dans le menu contextuel qui s'ouvre, sélectionnez l'option "Assistant d'édition du menu '?'".

3. Dans l'assistant, lors du choix des options de menu, sélectionnez l'option "Le saviez-vous".

4. Validez l'assistant. Le menu d'aide automatique est modifié. Le composant "TipOfTheDay" est automatiquement intégré dans votre projet.




**Si votre application ne contient pas de menu d'aide automatique** : 

1. Ajouter un menu automatique dans votre fenêtre principale : sous le volet "Fenêtre", dans le groupe "Barres et menus", déroulez "Menu principal" et sélectionnez "Ajouter le menu '?'". Pour plus de détails, consultez [Menu '?'](../Editeurs/9000070.md).

2. Dans l'assistant, l'option "Le saviez-vous" doit être cochée pour que l'option de menu correspondante soit créée.

3. Validez l'assistant. Le menu d'aide automatique est créé. Le composant "TipOfTheDay" est automatiquement intégré dans votre projet.


**Remarque** : Si le menu automatique propose l'option "Le saviez-vous", les astuces seront également proposées automatiquement au lancement de l'application : l'option "" est automatiquement cochée dans l'onglet "Avancé" de la description du projet.

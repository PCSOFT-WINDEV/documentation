## Présentation
<a name="presentation_ELTTEXTE000194"></a>
Une modélisation souple est composée de différents éléments reliés entre eux par des liaisons. Chaque élément a des caractéristiques.

Pour afficher les caractéristiques d'un élément, sélectionnez l'option "Description" du menu contextuel de l'élément. Selon l'élément sélectionné, les information affichées varient.

Deux onglets sont disponibles :

- l'onglet "Général" qui présente les caractéristiques générales de l'élément.

- l'onglet "Style" qui permet de définir les options de style de l'élément.

## Options générales
<a name="options_generales_ELTTEXTE000218"></a>


### Libellé/Nom
<a name="libellenom_ELTPARAGRAPHE000025"></a>Libellé affiché dans l'élément.

**Pour modifier ce libellé directement sous l'éditeur**, il suffit de :

- Sélectionner l'élément

- Cliquer sur le libellé : le libellé est en édition et peut être modifié.




**Pour la mise en forme du libellé ou sa position**, utilisez l'option "Couleur/Style/Police" du menu contextuel de l'élément.
<a name="NOTE2_2"></a>


### Image (Elément Image uniquement)
<a name="image_element_image_uniquement_ELTPARAGRAPHE000039"></a>Chemin de l'image à afficher dans l'élément Image. Il est également possible d'utiliser le bouton "Catalogue" pour sélectionner une des images proposées par le [catalogue d'images](../Editeurs/2020005.md) de WINDEV, WEBDEV, WINDEV Mobile. Dans ce cas, l'image sélectionnée est automatiquement copiée dans le répertoire du projet en cours.
<a name="NOTE2_3"></a>


### Liaison
<a name="liaison_ELTPARAGRAPHE000049"></a>Chemin du fichier lié à l'élément. Le fichier lié peut être un élément du projet en cours ou un autre fichier.

**Pour lier un élément de la modélisation souple à un élément du projet**, il suffit :

- soit de réaliser un "Drag and Drop" de l'élément du projet (par exemple depuis le volet "Explorateur de projet") vers l'élément voulu de la modélisation.

- soit de faire directement un "Drag and Drop" de l'élément du projet vers l'éditeur de modélisation. Dans ce cas, un nouvel élément est créé dans la modélisation.




**Pour modifier la liaison entre un élément de la modélisation et un élément du projet** :

1. Vérifiez que le volet "Explorateur de projet" est affiché dans votre environnement : sous le volet "Accueil", dans le groupe "Environnement", déroulez "Volets" et sélectionnez "Volets" puis "Explorateur de projet".

2. Affichez la description de l'élément de la modélisation.

3. Cliquez sur le bouton ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Modelisation_Cible.gif)
 et maintenez le bouton gauche de la souris enfoncé.

4. Déplacez la souris vers le nouvel élément du projet à lier dans le volet "Explorateur de projet".

5. Relâchez le bouton gauche de la souris au dessus de l'élément voulu. La nouvelle liaison apparaît dans la fenêtre de description de l'élément de la modélisation.




**Remarque** : Cette manipulation n'est pas possible sur les éléments "Fenêtres", "Etats" et "Composant" de la modélisation souple. Dans ce cas, pour changer la liaison, il est nécessaire de supprimer l'élément et de le recréer.

**Si l'élément de la modélisation est lié à un élément du projet**, l'élément de la modélisation contient alors l'icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=modelisation_liaison.gif)
. Cette flèche vers le bas indique qu'une liaison a été faite : un simple double-clic sur l'élément permet d'ouvrir directement l'objet lié. Il est également possible d'utiliser l'option "Ouvre l'élément lié" du menu contextuel de l'élément.

Dans les différents éditeurs, il est également possible de connaître les éléments présents dans une modélisation. Un bouton spécifique apparaît dans la description de l'élément du projet lorsque cet élément est relié à une modélisation.
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Modelisation_lien_fen.gif)
Si une seule modélisation est liée à l'élément, un clic sur le bouton permettra d'ouvrir la modélisation.

Si plusieurs modélisations sont liées à l'élément, un clic sur le bouton vous permet de sélectionner la modélisation à ouvrir.
<a name="NOTE2_4"></a>


### Niveau d'automatisme
<a name="niveau_automatisme_ELTPARAGRAPHE000095"></a>Le niveau d'automatisme permet de savoir et d'afficher dans la modélisation si nécessaire le niveau d'automatisme de l'élément en cours.

Si l'option "Niveau d'automatisme" est cochée, le niveau d'automatisme sera affiché sous forme d'icône dans la modélisation.

Les niveaux d'automatisme disponibles sont :

- Traitement informatisé automatique.

- Traitement informatisé avec intervention manuelle.

- Traitement manuel.



<a name="NOTE2_5"></a>


### Acteurs
<a name="acteurs_ELTPARAGRAPHE000110"></a>Les acteurs sont la ou les personnes intervenant dans la réalisation de la tâche décrite par l'élément.

Les acteurs présents dans les différentes listes peuvent être configurés grâce au bouton "Editer les acteurs". Pour plus de détails, consultez [Propriétés de la modélisation](../Editeurs/2010103.md).

- **Responsable** : Personne responsable de l'élément décrit.

- **Développeur** : Personne responsable du développement de l'élément.

- **Utilisateur** : Personne utilisant par exemple l'application.




Pour chaque type de responsabilité, il est possible de sélectionner plusieurs personnes.

Si une seule personne est sélectionnée, il est possible d'obtenir le détail des informations la concernant (adresse, numéro de téléphone, email, ...).

La case "Affiché" permet d'indiquer si les informations saisies doivent être affichées ou non dans la modélisation.

## Options de style
<a name="options_style_ELTTEXTE000266"></a>
Les options de style de l'élément permettent de définir simplement toutes les caractéristiques de style de l'élément sélectionné. Il est possible de sélectionner :

- la police utilisée, sa taille, ses attributs (gras, italique, ...) , sa couleur, son alignement.

- la couleur de fond de l'élément, la couleur de son cadre, son style et son épaisseur.




Lors de la modification d'un style d'un élément, il est possible de :

- Créer un nouveau style dans la modélisation. Dans ce cas, ce style pourra être ré-utilisé pour un autre élément.

- Modifier uniquement l'élément en cours.




Pour plus de détails sur la gestion des styles, consultez [Styles d'une modélisation](../Editeurs/2010105.md).


## Barre des documents ouverts
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000083"></a>
La barre des documents de l'environnement permet de naviguer entre les différents éléments ouverts sous les différents éditeurs. Le code et l'UI sont regroupés dans la barre.


![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Barre_document.gif)


Le survol d'un bouton affiche une vignette de l'aperçu : fenêtre, état, requête, code ....


![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Barre_Document_visu.gif)


Chaque document peut être fermé par une croix dans le bouton ou dans la vignette d'aperçu, ce qui est plus rapide. Il est également possible de cliquer sur la roulette de la souris : l'élément survolé est alors fermé. 

Le menu contextuel de chacun des boutons de cette barre propose de nombreuses options. L'option "Fermer tout sauf moi-même" est vraiment incontournable. 


![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Barre_document_Fermer.gif)


Si le nombre de choix est supérieur à l'espace disponible pour son affichage, la barre devient scrollable : il suffit d'utiliser la roulette de la souris pour faire défiler les différents éléments. Il est également possible de déplacer un bouton de la barre par un simple "Drag and Drop". 

Il est également possible d'afficher dans un menu la liste de tous les éléments ouverts. 


![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=barre_document_Menu.gif)


Remarque : Le menu contextuel de la barre des documents (et non pas d'un document) permet de rouvrir le ou les éléments qui ont été précédemment ouverts puis fermés (option "Récemment fermés"). 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Recent_Ouvert.gif)

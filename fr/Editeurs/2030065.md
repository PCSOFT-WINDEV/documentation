


## Générer et déployer des applications 64 bits
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000200"></a>
WINDEV permet de générer et d'installer une application 64 bits automatiquement. 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000224"></a>


### Créer un exécutable en mode 64 bits
<a name="creer_executable_mode_64_bits_ELTPARAGRAPHE000018"></a>Pour créer un exécutable en mode 64 bits : 

1. Créez une configuration de projet spécifique pour la gestion du 64 bits. Pour plus de détails, consultez [Configuration de projet](../Editeurs/9000030.md). Cette configuration de projet contient tous les éléments de votre projet. 

2. Sélectionnez si nécessaire la configuration de projet créée. 

3. Déroulez ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_generation_exe64.gif)
 parmi les boutons d'accès rapide et sélectionnez "Générer l'exécutable Windows 64 bits (.exe)". 

4. Suivez les différentes étapes de [l'assistant de création d'exécutable](../Editeurs/2025002.md).




Remarque : En utilisant une configuration de projet spécifique au 64 bits, l'exécutable et tous les fichiers nécessaires à son fonctionnement seront générés dans un répertoire spécifique. 
<a name="NOTE2_3"></a>


### Installer une application 64 bits
<a name="installer_une_application_64_bits_ELTPARAGRAPHE000040"></a>Pour installer une application 64 bits, WINDEV propose deux modes : 

- **Installation de l'application 64 bits uniquement** 
	Ce mode permet de générer le programme d'installation uniquement pour l'application 64 bits. Pour créer le programme d'installation, sous le volet "Projet", dans le groupe "Génération", cliquez sur "Procédure d'installation" (la configuration 64 bits de l'application doit être sélectionnée). L'assistant de création de la procédure d'installation se lance automatiquement (voir [Assistant de création d'installation](../Editeurs/2028062.md)). 
	Remarque : Le programme d'installation de l'application 64 bits est lui aussi un programme 64 bits.

- **Installation de l'application en mode 32 bits ou 64 bits** (avec adaptation en fonction du système de déploiement)
	WINDEV permet de créer une installation unique contenant à la fois l'application "32 bits" et l'application "64 bits". La version à installer sera automatiquement choisie en fonction du système d'exploitation de l'utilisateur. Nous allons détailler cette option.



<a name="NOTE2_4"></a>


### Installation combinée 32/64 bits
<a name="installation_combinee_3264_bits_ELTPARAGRAPHE000071"></a>Pour réaliser une installation combinée 32/64 bits : 

1. Générez l'exécutable et la procédure d'installation 32 bits. 

	- Affichez si nécessaire la configuration 32 bits : 

		- soit depuis le volet "Explorateur de projet". 

		- soit sous le volet "Projet", dans le groupe "Configuration de projet", cliquez sur "Gérer les configurations". 




- Générez l'exécutable et la procédure d'installation 32 bits : cliquez sur ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_generation_exe.gif)
 parmi les boutons d'accès rapide. 

2. Générez l'exécutable et la procédure d'installation 64 bits. 

	- Affichez si nécessaire la configuration 64 bits : 

		- soit depuis le volet "Explorateur de projet", 

		- soit sous le volet "Projet", dans le groupe "Configuration de projet", cliquez sur "Gérer les configurations".  




- Générez l'exécutable et la procédure d'installation 64 bits (cliquez sur ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_generation_exe64.gif)
 parmi les boutons d'accès rapide. 

3. Générez l'installation combinée : sous le volet "Projet", dans le groupe "Génération", cliquez sur "Procédure d'installation" et sélectionnez "Générer une installation combinée 32/64 bits". La fenêtre suivante s'affiche :
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Install_combinee.gif&type=thumb)


4. Indiquez les chemins des différents programmes d'installation (fichiers WDF des installations 32 et 64 bits). Si ces fichiers ne sont pas créés ou à jour, il est possible de re-générer les différentes installations. 
	Remarque : Il sera possible de modifier l'aide des nouveautés affichée lors de l'installation.

5. Dans l'étape suivante, indiquez les options de la génération de l'installation combinée : 

	- répertoire unique de génération.

	- vidage du répertoire avant génération.

	- répertoire de création de l'installation combinée. 

	- format de compression de l'installation combinée (Zip ou 7Z).




6. Validez. 

7. Si vous avez demandé la génération des programmes d'installation, validez les différents assistants de génération de l'exécutable et du programme d'installation. 

8. La génération du programme d'installation combinée est réalisée. 




Remarques : 

- L'installation combinée 32/64 bits n'est pas disponible pour les installations multisites.

- L'historique des versions défini pour chaque programme d'installation est géré : il est possible de revenir à une version précédente du logiciel. 

- La signature des installations combinées est disponible : chaque installation peut être signée.

- Support d'installation : chaque installation doit avoir pour support d'installation un répertoire unique. 

- Le programme d'installation de l'application 32 bits est une application 32 bits. Le programme d'installation de l'application 64 bits est une application 64 bits.

- L'installation combinée 32/64 bits peut utiliser des programmes d'installation différents et personnalisés. Il suffit de sélectionner le fichier "WDF" correspondant au programme d'installation voulu.

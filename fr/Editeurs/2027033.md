


## Le volet "Visualisation contextuelle du code"
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000087"></a>
Le volet "Visualisation contextuelle du code" permet d'afficher et/ou de modifier le code de l'élément (fenêtre, page, état, champ, ...) sélectionné sous l'éditeur.

Les caractéristiques du code affiché dans le volet correspondent aux caractéristiques de l'éditeur de code.<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=VoletCodectx.gif)
Le menu contextuel reprend les options du menu contextuel de l'éditeur de code.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Afficher le volet "Visualisation contextuelle du code"
<a name="afficher_volet_visualisation_contextuelle_code_ELTTEXTE000111"></a>
**Pour afficher le volet "Visualisation contextuelle du code"**, sous le volet "Accueil", dans le groupe "Environnement", déroulez "Volets" et sélectionnez "Volets" puis "Visualisation contextuelle du code".

Pour plus de détails sur les manipulations des volets , consultez [Manipuler les volets ancrables](../Editeurs/2027001.md).




## Ne plus afficher un message
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000116"></a>
L'environnement de WINDEV, WEBDEV, WINDEV Mobile affiche souvent des fenêtres d'avertissement.

Très importantes lors de la découverte d'une fonctionnalité, ces fenêtres peuvent vite s'avérer inutiles.

Le choix "Ne plus afficher" apparaît maintenant dans de nombreuses fenêtres, et permet de désactiver la fenêtre d'avertissement en cours.

WINDEV, WEBDEV et WINDEV Mobile proposent une gestion centralisée des fenêtres d'avertissement concernant tous les projets : il est ainsi possible de retrouver les fenêtres qui n'apparaissent plus et de les ré-activer si nécessaire.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000140"></a>


### Ne plus afficher un message
<a name="plus_afficher_message_ELTPARAGRAPHE000024"></a>Pour ne plus afficher un message, il suffit de cocher la case "Ne plus afficher" présente dans le message.

Si le message attend une réponse, la réponse sélectionnée sera automatiquement mémorisée et utilisée par défaut.
<a name="NOTE2_2"></a>


### Consulter la liste des "Ne plus afficher" et réactiver un message
<a name="consulter_liste_des_plus_afficher_reactiver_message_ELTPARAGRAPHE000033"></a>La liste des messages pour lesquels l'option "Ne plus afficher" a été cochée est disponible à tout moment :

1. Sous le volet "Accueil", dans le groupe "Environnement", déroulez "Options" et sélectionnez "Options générales de WINDEV/WEBDEV/WINDEV Mobile".

2. Sélectionnez l'onglet "Ne plus afficher". La liste affichée correspond aux différents messages qui ne sont plus affichés. Pour chaque message est affiché le nom de l'éditeur concerné, l'intitulé du message, la réponse sélectionnée si nécessaire.




Pour ré-activer un message, il suffit de décocher le message voulu dans la liste et de valider.

Remarques :

- Cette liste affiche uniquement les "Ne plus afficher" communs à tous les projets. Lorsqu'une option est spécifique à un projet, elle doit être paramétrée dans le projet. Exemple : la fenêtre demandant si le test doit être arrêté lors du retour sous l'éditeur : ce paramétrage est spécifique à chaque projet.

- La fenêtre de login n'est pas prise en compte dans les "Ne plus afficher". Cette fenêtre est paramétrée directement dans les options générales du produit. Pour afficher les options générales du produit, sous le volet "Accueil", dans le groupe "Environnement", déroulez "Options" et sélectionnez "Options générales de WINDEV/WEBDEV/WINDEV Mobile".

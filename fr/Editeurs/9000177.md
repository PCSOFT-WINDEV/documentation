


## Première fenêtre du projet
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000177"></a>
La première fenêtre du projet correspond à la première fenêtre ouverte lors du lancement de l'application WINDEV ou WINDEV Mobile. 

Cette fenêtre est également ouverte en premier lors du test du projet sur le poste de développement. 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment définir la première fenêtre du projet ?
<a name="comment_definir_premiere_fenetre_projet_ELTTEXTE000201"></a>
WINDEV propose plusieurs méthodes pour définir et choisir la première fenêtre d'un projet :

- [Par le volet "Explorateur de projet"](#NOTE2_2).

- [Dans la description de la configuration](#NOTE2_3).

- [Lors du premier test du projet](#NOTE2_4). 




**Remarque** : La première fenêtre du projet est également demandée lors du test du projet, si aucune fenêtre n'a été définie. Pour plus de détails, consultez [Tester un projet](../Editeurs/2019003.md).
<a name="NOTE2_2"></a>


### Définir la première fenêtre du projet dans le volet "Explorateur de projet"
<a name="definir_premiere_fenetre_projet_dans_volet_explorateur_projet_ELTPARAGRAPHE000040"></a>Pour définir la première fenêtre du projet :

1. Sélectionnez la fenêtre qui va être la première fenêtre dans le volet "Explorateur de projet". 

2. Affichez le menu contextuel de la fenêtre (clic droit) et sélectionnez l'option  "Première fenêtre du projet". <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Premi%E8re_fen_prjWD%20-%20HC%20N%B0002.gif)

	L'icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Premi%E8re_fen_prjWD%20-%20HC%20N%B0001.gif) permet d'identifier la première fenêtre du projet dans le volet "Explorateur de projet".






**Remarque** : Si le projet comporte plusieurs configurations, il est possible de définir une première fenêtre par configuration. Il suffit de : 

1. Sélectionner la configuration de projet voulue dans l'explorateur de projet. 

2. Définir la première fenêtre du projet via le menu contextuel de l'explorateur de projet.



<a name="NOTE2_3"></a>


### Définir la première fenêtre dans la description de la configuration
<a name="definir_premiere_fenetre_dans_description_configuration_ELTPARAGRAPHE000071"></a>Pour définir la première fenêtre dans la description du projet : 

1. Affichez la description du projet : sous le volet "Projet", dans le groupe "Projet", cliquez sur "Description".

2. Dans l'onglet "Général", cliquez sur "Description de la configuration courante". 

3. La fenêtre de description de la configuration courante s'affiche. 

4. Dans l'onglet "Général", la liste "Première fenêtre du projet" permet de sélectionner la première fenêtre du projet. 





<a name="NOTE2_4"></a>


### Définir la première fenêtre lors du test du projet
<a name="definir_premiere_fenetre_lors_test_projet_ELTPARAGRAPHE000103"></a>Pour tester un projet depuis l'éditeur : 

1. Sous le volet "Projet", dans le groupe "Mode test", déroulez "Mode test" et sélectionnez "Déboguer le projet" (Ctrl + F9). Il est également possible de cliquer sur ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ICO_GO_Projet_WD.gif) parmi les boutons d'accès rapide.

2. Si la première fenêtre du projet n'a pas été définie, une fenêtre s'affiche permettant de sélectionner la première fenêtre du projet. 
	

3. L'éditeur est automatiquement réduit en icône et le projet s'exécute.


## Bienvenue dans WINDEV, WEBDEV et WINDEV Mobile
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000221"></a>
La page d'accueil permet de réaliser simplement les opérations de création et d'ouverture de projets. 

La page d'accueil permet de : 

- [Voir les actualités concernant les produits PC SOFT](#NOTE2_2b). 

- [Créer un projet](#NOTE2_2). 

- [Ouvrir un projet](#NOTE2_3). 

- [Ouvrir un exemple](#NOTE2_4). 

- [Suivre et ouvrir un exemple du tutoriel](#NOTE2_5). 




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000245"></a>


### Afficher la page d'accueil
<a name="afficher_page_accueil_ELTPARAGRAPHE000036"></a>Il est possible d'accéder à tout moment à la page d'accueil. 

Pour afficher la page d'accueil : 

1. Cliquez sur l'icône 28 en haut de l'éditeur utilisé. 

2. Dans le menu contextuel qui s'affiche, sélectionnez "Bienvenue dans XXX". 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Bienvenue_WD_Afficher.gif)



Note : Il est également possible d'utiliser le raccourci Ctrl + &lt;. 
<a name="NOTE2_2b"></a>


### Afficher les actualités
<a name="afficher_les_actualites_ELTPARAGRAPHE000055"></a>Par défaut, la page d'accueil affiche les actualités concernant le produit utilisé : 

- nouvelle version disponible, 

- nouveaux téléchargements

- date des formations, 

- WINDEV Tour, ...




Sur cette page vous pouvez également accéder directement aux forums ou aux dernières FAQ. 










<a name="NOTE2_2"></a>


### Créer un projet
<a name="creer_projet_ELTPARAGRAPHE000073"></a>La page d'accueil permet de créer rapidement un projet pour une plateforme spécifique. Pour cela : 

1. Affichez la page d'accueil. 

2. Cliquez sur "Créer un projet". 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Bienvenue_WD_creer.gif&type=thumb)
La liste des différents types de projets pour le produit spécifié s'affiche. Vous pouvez : 

	- Cliquer sur le type de projet voulu. Dans ce cas, l'assistant de création d'un projet pour le type de projet voulu se lance. 

	- Cliquer à nouveau sur "Créer un projet". Dans ce cas, l'assistant de création complet (avec le choix du type de projet) se lance. 






<a name="NOTE2_3"></a>


### Ouvrir un projet
<a name="ouvrir_projet_ELTPARAGRAPHE000088"></a>La page d'accueil permet d'ouvrir rapidement un projet. Pour cela : 

1. Affichez la page d'accueil. 

2. Cliquez sur "Ouvrir un projet".

3. La liste des [projets épinglés](../Editeurs/9500217.md) et des projets précédemment ouverts s'affiche. 

4. Il est possible : 

	- d'ouvrir un des projets présents dans la liste. Il suffit de cliquer sur le lien correspondant. 

	- d'ouvrir un projet présent sur le disque. Il suffit soit de cliquer à nouveau sur le lien "Ouvrir un projet", soit de cliquer sur le bouton "Disque" puis de sélectionner le fichier projet voulu (WDP, WWP ou WPP).

	- d'ouvrir un projet présent dans le GDS (Gestionnaire de Sources). Il suffit de cliquer sur le lien "GDS" puis d'indiquer si nécessaire la localisation de la base et le projet à ouvrir. 

	- d'ouvrir un projet depuis GDSDrive. Il suffit de cliquer sur le lien "GDSDrive" puis d'indiquer les renseignements d'identification à GDSDrive : email, mot de passe et équipe. Pour plus de détails sur GDSDrive, connectez-vous à [GDSDrive](https://pcscloud-drive.net/FR/GDSDrive/index.awp).

	- d'ouvrir un projet depuis le CLOUD. Il suffit de cliquer sur le lien "PCSCloud" puis d'indiquer le nom de la plateforme cloud utilisée et le projet à ouvrir. Pour plus de détails, connectez-vous à [PCSCloud](https://pcscloud.net/FR/cloud_developpement.awp).







Remarques : 

- La zone de recherche permet de saisir le nom du projet à rechercher parmi les projets épinglés et les projets récents.

- Selon le produit lancé (WINDEV, WEBDEV ou WINDEV Mobile), seuls les projets pouvant être ouverts avec le produit en cours sont listés. Si WINDEV est installé en mode intégré (avec WEBDEV et WINDEV Mobile intégrés à WINDEV), tous les projets précédemment ouverts seront listés.



<a name="NOTE2_4"></a>


### Ouvrir un exemple
<a name="ouvrir_exemple_ELTPARAGRAPHE000122"></a>La page d'accueil permet d'ouvrir rapidement un exemple installé avec le produit utilisé. Pour cela : 

1. Affichez la page d'accueil. 

2. Cliquez sur "Ouvrir un exemple". 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Bienvenue_WD_Exemple%20-%20HC%20N%B0001.gif&type=thumb)


3. La liste des exemples s'affiche. Ces exemples sont regroupés par catégorie : 

	- Exemples complets,

	- Exemples didactiques,

	- Exemples unitaires,

	- Composants.




4. Il est possible : 

	- de rechercher un exemple, en saisissant un mot-clé dans le champ de recherche (identifié par une loupe). La liste des exemples est automatiquement mise à jour avec les exemples utilisant ce mot-clé dans leur nom ou leur description. 
			Un pictogramme permet d'identifier rapidement la plateforme d'exécution de l'exemple.

	- de filtrer les exemples par plateforme, en utilisant les différents pictogrammes situés à côté de la zone de recherche.  

	- d'ouvrir un des exemples présents dans la liste. Il suffit de cliquer sur le nom de l'exemple. 





**Remarque** : Lors de l'ouverture d'un exemple : 

- Si l'exemple n'est pas installé sur le poste, l'exemple est automatiquement téléchargé (depuis un serveur PC SOFT) et ouvert. 

- Si l'exemple est installé sur le poste, une copie est automatiquement réalisée dans le répertoire "Mes Projets\\Mes Exemples".  L'exemple ouvert correspond à la copie. Il est ainsi possible de travailler ou de faire des modifications sur cette copie. L'exemple original est ainsi conservé. A chaque nouvelle ouverture de l'exemple (via le bouton "Ouvrir un exemple"), il est possible de travailler sur l'exemple modifié ou de reprendre l'exemple original. 





<a name="NOTE2_5"></a>


### Ouvrir un exemple du tutoriel
<a name="ouvrir_exemple_tutoriel_ELTPARAGRAPHE000157"></a>WINDEV, WEBDEV et WINDEV Mobile sont livrés avec un tutoriel. Ce tutoriel est constitué d'un support (en ligne) et d'exercices. La page d'accueil permet d'ouvrir rapidement le support de cours sous forme électronique, ainsi que les exercices associés. Pour cela : 

1. Affichez la page d'accueil. 

2. Cliquez sur le bouton "Tutoriel". 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Bienvenue_WD_GAF.gif&type=thumb)


3. Vous pouvez : 

	- Consultez la version électronique du cours grâce au lien affiché en haut. 

	- Ouvrir un des exercices du tutoriel affiché dans la liste. Ces exemples sont regroupés selon leur ordre d'apparition dans le tutoriel. Pour ouvrir le projet voulu, il suffit de cliquer sur le lien correspondant. 







**Remarques** : 

- Si le tutoriel n'est pas installé, il est possible de télécharger les exemples du tutoriel depuis un serveur PC SOFT.

- Lors de l'ouverture d'un exercice du tutoriel, une copie est automatiquement réalisée dans le répertoire "Mes Projets\\Mes Exemples".  Le projet ouvert correspond à la copie. Il est ainsi possible de travailler ou faire des modifications sur cette copie. Le projet original est ainsi conservé. A chaque nouvelle ouverture de l'exercice via le bouton "Tutoriel", il est possible de travailler sur l'exercice modifié ou de reprendre l'exercice original.

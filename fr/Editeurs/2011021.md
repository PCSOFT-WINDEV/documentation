
## Qu'est-ce qu'une base de données ? 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000072"></a>
Une **base de données** est un ensemble d'informations associées à un sujet particulier (gestion des commandes par exemple). Ces informations sont organisées et stockées dans la base de données afin qu'un utilisateur puisse les retrouver facilement.

Les bases de données font partie de notre vie quotidienne : un simple carnet d'adresses est déjà une base de données.

Le carnet d'adresses permet de stocker les noms des personnes, les adresses, les numéros de téléphone, etc. L'index alphabétique permet de retrouver simplement une personne.
<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=base-de-d1.gif)


Dans une base de données informatisée, on retrouve les mêmes concepts que dans le carnet d'adresses :

- les informations stockées correspondent aux **rubriques**.

- les regroupements d'informations correspondent aux **fichiers de données**,

- les index sont créés à partir de rubriques de tri (appelées **Clés**).

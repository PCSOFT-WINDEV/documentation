
## Sous-modélisation
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000138"></a>
Dans une modélisation souple, les éléments de type "Application", "Traitement" et "Choix" peuvent permettre de rentrer dans une "sous-modélisation".

Par exemple, la description de l'ERP (Enterprise Resource Planning) d'une entreprise peut être composée des sous-modélisations suivantes : Gestion des clients, gestion des devis, marketing, production, comptabilité, ...

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Modelisation2.gif)


Le sigle ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=modelisation_sous_model.gif)
 permet d'identifier si une sous-modélisation peut être créée (l'icône est alors grisée) ou si une sous-modélisation est disponible.

## Manipuler les sous-modélisations
<a name="manipuler_les_sousmodelisations_ELTTEXTE000162"></a>


### Créer une sous-modélisation
<a name="creer_une_sousmodelisation_ELTPARAGRAPHE000029"></a>Pour créer une sous-modélisation à partir d'un élément :

1. Sélectionnez l'élément à lier à une sous-modélisation (Application, Traitement ou Choix).

2. Sélectionnez l'option "Ajouter une sous-modélisation" du menu contextuel de l'élément.

3. La sous-modélisation est alors créée et automatiquement ouverte sous l'éditeur.




**Remarque : il est également possible d'associer un élément à une sous-modélisation existante.** Pour cela, il suffit de :

1. Sélectionner l'élément à lier à la sous-modélisation.

2. Sélectionner l'option "Sélectionner une sous-modélisation existante" du menu contextuel de l'élément.

3. Sélectionner la sous-modélisation voulue. Cette sous-modélisation peut appartenir à la modélisation en cours ou à une autre modélisation.

4. Validez.

5. L'élément apparaît avec le sigle ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=modelisation_sous_vide.gif)
.





<a name="NOTE2_2"></a>


### Entrer dans une sous-modélisation
<a name="entrer_dans_une_sousmodelisation_ELTPARAGRAPHE000051"></a>Pour entrer dans la sous-modélisation d'un élément :

1. Sélectionnez l'élément voulu.

2. Dans le menu contextuel de l'élément, sélectionnez l'option "Entrer dans la sous-modélisation".

3. La sous-modélisation est alors automatiquement affichée.




**Remarque** : **si l'élément est lié à une sous-modélisation existante**, il est possible d'afficher cette sous-modélisation :

1. Sélectionnez l'élément voulu.

2. Dans le menu contextuel de l'élément, sélectionnez l'option "Suivre le lien vers la sous-modélisation".

3. La sous-modélisation est alors automatiquement affichée.




Une sous-modélisation peut avoir plusieurs modélisations "parentes" différentes.


<a name="NOTE2_3"></a>


### Remonter au niveau précédent de la modélisation
<a name="remonter_niveau_precedent_modelisation_ELTPARAGRAPHE000071"></a>Pour remonter au niveau précédent, vous pouvez utiliser :

- soit la barre de titre de la sous-modélisation. Les différents modélisations sont représentées sous la forme : "Modélisation1 &gt; Sous-Modélisation2 &gt; Sous-Modélisation3". Un simple clic sur la modélisation voulue permet de l'afficher.

- soit les différents volets à gauche de l'éditeur. Chaque volet correspond à une modélisation ou une sous-modélisation. Un simple clic sur la modélisation voulue permet de l'afficher.

- soit l'option de menu contextuel "Remonter au parent" de la sous-modélisation.





<a name="NOTE2_4"></a>


### Supprimer une sous-modélisation
<a name="supprimer_une_sousmodelisation_ELTPARAGRAPHE000083"></a>Pour supprimer une sous-modélisation :

1. Sélectionnez l'élément voulu.

2. Dans le menu contextuel, sélectionnez l'option "Supprimer la sous-modélisation". 




Remarque : Si l'élément est lié à une sous-modélisation existante, sélectionnez l'option de menu "Supprimer le lien vers la sous-modélisation".

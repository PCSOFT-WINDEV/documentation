
## Liste des tâches 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000119"></a>
Chaque projet peut être associé à une liste des tâches. Cette liste permet de détailler aux développeurs l'ensemble des modifications à apporter au projet.

Ces modifications peuvent concerner :

- un traitement dans le code de l'ensemble du projet,

- un objet (fenêtre, état, ...) à modifier ou à créer, etc.


**Remarques** : 

- **Cette liste des tâches n'est pas reliée au Centre de Suivi de Projets.** 

- Pour gérer une liste des tâches et un planning complet, il est conseillé d'utiliser le Centre de Suivi de Projets. Pour plus de détails, consultez [Comment gérer mes tâches et mon planning](../CCSuivi/3540721.md).

## Comment le faire ?
<a name="comment_faire_ELTTEXTE000143"></a>


### Visualiser la liste des tâches
<a name="visualiser_liste_des_taches_ELTPARAGRAPHE000032"></a>Pour visualiser la liste des tâches, affichez le [volet "Liste des tâches"](../Editeurs/2027015.md).


<a name="NOTE2_2"></a>


### Ajouter une tâche
<a name="ajouter_une_tache_ELTPARAGRAPHE000043"></a>Pour ajouter une tâche, deux solutions sont possibles, selon la nature de la tâche. 

**Cas 1 : Ajouter une tâche spécifique à la programmation** : Saisissez directement dans le code la description de la tâche précédée de "//A faire". Par exemple :

```wl
// A faire : Gestion des codes barres
```


Remarque : Il sera alors possible de se positionner dans le code à l'endroit précis de la tâche à réaliser en double-cliquant directement sur le libellé de la tâche dans le volet "Liste des tâches".

**Cas 2 : Ajouter une tâche par le volet "Liste des tâches"** : Cliquez sur ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Ico_Ajout_Tache.gif) dans le [volet "Liste des tâches"](../Editeurs/2027015.md). Chaque tâche peut :

- Être affectée à un des développeurs du projet.

- Correspondre à un niveau de priorité spécifique (priorité élevée, normale ou faible). Il est ainsi possible de trier les tâches à effectuer par ordre de priorité.





<a name="NOTE2_3"></a>


### Manipuler les tâches
<a name="manipuler_les_taches_ELTPARAGRAPHE000066"></a>Pour manipuler les tâches, utilisez le menu contextuel du volet "Liste des tâches".

**Remarque** : L'affichage des tâches dans le volet "Liste des tâches" n'est pas synchronisé avec le code. Par exemple, si la ligne de commentaire "// A faire : Gestion des codes barres" est supprimée, la tâche correspondante dans le volet "Liste des tâches" ne sera pas automatiquement supprimée.

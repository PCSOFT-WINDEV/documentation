## Comment le faire ?
<a name="comment_faire_ELTTEXTE000268"></a>


### Créer un diagramme d'objet
<a name="creer_diagramme_objet_ELTPARAGRAPHE000042"></a>Pour créer un diagramme d'objet : 

1. Cliquez sur ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_nouveau.gif)
 parmi les boutons d'accès rapide. 

	- La fenêtre de création d'un nouvel élément s'affiche : cliquez sur "Architecture" puis sur "UML".

	- L'assistant de création d'un modèle UML se lance.




2. Sélectionnez l'option "Diagramme d'objets".

3. Spécifiez le modèle UML dans lequel le diagramme sera inséré.

4. Choisissez d'éditer directement le diagramme.

5. Validez la création du diagramme d'objet. Le modèle créé devient le modèle en cours.



<a name="NOTE2_2"></a>


### Créer un objet
<a name="creer_objet_ELTPARAGRAPHE000064"></a>Pour créer un objet : 

1. Sous le volet "Diagramme d'objets", dans le groupe "Création", cliquez sur "Objet". L'assistant de création d'un objet se lance.

2. Spécifiez :

	- le nom de l'objet. Ce nom est affiché sous l'éditeur d'UML.

	- le nom de la classe. Ce nom est affiché sous l'éditeur d'UML.

	- le libellé de la classe décrivant l'intérêt de la classe. Ce libellé sera utilisé dans le dossier d'impression de l'éditeur d'UML.




3. Définissez les attributs de la classe. 

4. Validez la description de l'objet.



<a name="NOTE2_3"></a>


### Ajouter un attribut
<a name="ajouter_attribut_ELTPARAGRAPHE000092"></a>Pour ajouter un attribut : 

1. Sélectionnez l'objet dans lequel l'attribut doit être ajouté.

2. Sous le volet "Diagramme d'objets", dans le groupe "Création", cliquez sur "Attribut". L'assistant de création d'un attribut se lance.

3. Définissez les attributs de la classe.

4. Validez la description de l'attribut. L'attribut est automatiquement ajouté à l'objet en cours.




**Remarque** : Il est possible de modifier les caractéristiques d'un attribut dans les spécifications de l'objet (option "Description" du menu contextuel de l'objet).
<a name="NOTE2_4"></a>


### Créer un objet composite
<a name="creer_objet_composite_ELTPARAGRAPHE000118"></a>Pour créer un objet composite : 

1. Sous le volet "Diagramme d'objets", dans le groupe "Création", cliquez sur "Objet composite". Un nouvel objet composite apparaît dans le modèle UML.

2. Sélectionnez l'option "Description" du menu contextuel (clic droit) du nouvel objet composite. La fenêtre de spécifications de l'objet s'ouvre.

3. Spécifiez :

	- le nom de l'objet composite. Ce nom est affiché sous l'éditeur d'UML.

	- la classe de l'objet composite.

	- le libellé de l'objet composite décrivant l'intérêt de l'objet composite. Ce libellé sera utilisé dans le dossier d'impression de l'éditeur d'UML.




4. Validez la description de l'objet composite.



<a name="NOTE2_5"></a>


### Créer un lien
<a name="creer_lien_ELTPARAGRAPHE000146"></a>Pour créer un lien : 

1. Sous le volet "Diagramme d'objets", dans le groupe "Création", cliquez sur "Lien".

2. Sélectionnez les différents objets à relier.

3. Sélectionnez l'option "Description" du menu contextuel (clic droit) du nouveau lien. La fenêtre de spécifications du lien s'ouvre.

4. Spécifiez :

	- le type du lien.

	- le nom du lien. Ce nom est affiché sous l'éditeur d'UML.




5. Validez la description du lien.



<a name="NOTE2_6"></a>


### Créer un paquetage
<a name="creer_paquetage_ELTPARAGRAPHE000174"></a>Pour créer un paquetage : 

1. Sous le volet "Diagramme d'objets", dans le groupe "Création", cliquez sur "Paquetage". Un nouveau paquetage apparaît dans le modèle UML.

2. Sélectionnez l'option "Description" du menu contextuel (clic droit) du nouveau paquetage. La fenêtre de spécifications du paquetage s'ouvre.

3. Spécifiez :

	- le nom du paquetage. Ce nom est affiché dans le paquetage sous l'éditeur d'UML.

	- le libellé du paquetage décrivant l'intérêt du paquetage.

	- les relations du paquetage.




4. Validez la description du paquetage.

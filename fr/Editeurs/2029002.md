
## Environnement de l'éditeur d'analyses 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
L'environnement de l'éditeur d'analyses est le suivant : <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=EnvEditeurAnalyses.gif&type=thumb)


Les éléments importants de l'éditeur d'analyses de WINDEV sont les suivants :


|   |   |
| --- | --- |
| ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CERCLE1.gif) | **Menu de l'éditeur**<br>Permet un accès rapide aux principales fonctionnalités de l'éditeur d'analyses. |
| <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Cercle2.gif)<br> | **Graphe de l'analyse**<br>Des ascenseurs permettent de faire défiler les éléments du graphe (analyse contenant beaucoup de fichiers de données). Il est également possible de zoomer sur le graphe pour visualiser plus précisément les éléments de l'analyse (Ctrl + roulette de souris). |
| <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Cercle3.gif)<br> | **[Volet Analyse](../Editeurs/2027012.md)**<br>Liste les différents fichiers de données présents dans l'analyse en cours. Il suffit de dérouler le nom du fichier pour voir les noms et les types de rubriques présentes dans le fichier de données. |
| <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Cercle4.gif)<br> | **Fenêtre de description des fichiers de données**<br>Permet de :<br><br>- visualiser et/ou de modifier les caractéristiques d'un fichier de données de l'analyse.<br><br>- accéder à la fenêtre de description des rubriques du fichier de données.<br><br><br> |
| <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Cercle5.gif)<br> | **[Documents ouverts](../Editeurs/2029012.md)**<br>Affiche un bouton pour chaque document ouvert sous l'éditeur. Pour afficher l'élément voulu au premier plan, il suffit de cliquer sur le "bouton" correspondant. |
| <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Cercle6.gif)<br> | **Fenêtre du navigateur**<br>Permet de visualiser l'ensemble des éléments contenus dans le dossier ouvert. |
| <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Cercle7.gif)<br> | **Visualisation des rubriques**<br>Permet de visualiser l'ensemble des rubriques du fichier en cours. |




## Le volet "Messagerie"
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000092"></a>
Le volet "Messagerie" permet d'échanger des messages entre les différents collaborateurs de l'entreprise.
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=VoletMessagerie.gif)
Pour plus de détails, consultez [Messagerie](../Editeurs/2027023.md).

**Remarque** : Les différents collaborateurs pris en compte pour la messagerie sont les collaborateurs enregistrés dans le groupware développeur et / ou dans les différents Centres de Contrôle. Pour plus de détails, consultez [Base de données des Centres de Contrôle](../CCSuivi/3540720.md).

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Afficher le volet "Messagerie"
<a name="afficher_volet_messagerie_ELTTEXTE000116"></a>
**Pour afficher le volet "Messagerie"**, sous le volet  sous le volet "Accueil", dans le groupe "Environnement", déroulez "Volets" et sélectionnez "Volets" puis "Messagerie".

Pour plus de détails sur les manipulations des volets, consultez [Manipuler les volets ancrables](../Editeurs/2027001.md).


## Description du projet : Onglet Langues
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000206"></a>
La fenêtre de description du projet permet de configurer de nombreux éléments de votre projet. Cette fenêtre est constituée des onglets suivants :

- L'onglet ["Projet"](../Editeurs/2030046.md) qui permet de saisir les informations générales du projet

- L'onglet "Analyse" qui permet d'associer une analyse au projet et / ou un modèle UML

- L'onglet ["Fichiers"](../Editeurs/2030052.md) qui permet de saisir différentes informations sur les fichiers de données gérés par l'application (uniquement disponible si le projet est lié à une analyse).

- L'onglet "Live data" qui permet de configurer l'affichage des données (provenant des fichiers de données) sous les différents éditeurs.

- **L'onglet "Langues"** qui permet de paramétrer les différentes langues manipulées par le projet ainsi que les options linguistiques à utiliser pour chaque langue.

- L'onglet ["Style"](../Editeurs/9500234.md) qui permet de choisir le thème de gabarit, la feuille de styles personnalisée et de personnaliser l'aperçu avant impression. Il est également possible de paramétrer la taille des écrans.

- L'onglet "Groupe" qui permet de gérer le travail en groupe (GDS ou GIT).

- L'onglet ["Options"](../Editeurs/9500227.md) qui permet de configurer les options de mise à jour, de sauvegarde, de synchronisation des modèles. Cet onglet permet également d'activer le mode strict et de définir la charte de programmation.

- L'onglet ["Avancé"](../Editeurs/9500236.md) qui permet de gérer l'utilisation de "Etats & Requêtes", les FAA et le verrouillage de l'application.

- L'onglet ["Compilation"](../Editeurs/9500218.md) qui permet de configurer les options de compilation.

- L'onglet ["Télémétrie"](../Editeurs/9500223.md) qui permet de configurer les options de télémétrie pour le projet.

## Onglet "Langues"
<a name="onglet_langues_ELTTEXTE000230"></a>


### Langues du projet
<a name="langues_projet_ELTPARAGRAPHE000057"></a>
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Projet_multilangue%20-%20HC%20N%B0001.gif&type=thumb)


Cet onglet permet de gérer les caractéristiques des différentes langues du projet. 

Pour ajouter une nouvelle langue dans le projet : 

1. Cliquez sur le bouton "Ajouter". 

2. Dans la fenêtre qui s'affiche, sélectionnez les différentes langues gérées par le projet.
	Remarques : 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ProjetMultilangue2%20-%20HC%20N%B0001.gif&type=thumb)


	- Si la langue voulue n'apparaît pas dans la liste, il est possible de sélectionner une langue supplémentaire (les langues supplémentaires sont affichées dans la partie basse de la fenêtre). 

- Pour nommer une langue supplémentaire, cliquez sur la langue voulue et saisissez son nom dans la zone "Nom des langues personnalisées". 

3. Validez. Les différentes langues sont ajoutées dans la liste des langues du projet.




Remarques : 

- Les langues sélectionnées seront proposées pour toutes les informations des différents éléments pouvant être traduites (libellés des champs, options de menus, messages d'aide associés à un champ, ...). L'ordre d'affichage de ces langues est identique à celui de la liste des langues du projet. 
	Il est possible de modifier cet ordre à l'aide des boutons fléchés présents sous la liste des langues du projet.

- Lors de la modification des différentes langues du projet, les modifications seront automatiquement prises en compte :

	- pour tout nouvel objet créé sous les différents éditeurs (Fenêtre, Etat, Champ, ...).

	- pour tout élément ouvert sous les différents éditeurs (Fenêtre, Etat, ...).




- La langue par défaut (identifiée par le numéro "1") correspond à la langue utilisée par défaut en exécution. Pour la modifier, utilisez le menu contextuel de la langue à utiliser par défaut.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) L'option **"Changement de langue immédiat après l'appel à la fonction Nation du WLangage"** permet de modifier directement la langue d'affichage de l'exécutable après l'utilisation de la fonction [Nation](../WDLang1/3054014.md) (dans ce cas, la fonction [Utilise](../WDLang1/3038044.md) n'est plus nécessaire pour prendre en compte le changement de langue). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) L'option **"Sélectionner automatiquement la langue du système au démarrage de l'application"** permet d'utiliser systématiquement la langue correspondant au système de l'utilisateur.

### Options linguistiques
<a name="options_linguistiques_ELTPARAGRAPHE000120"></a>Il est possible de paramétrer l'affichage des différentes données numériques du projet (nombre, monétaire, date, ...) dans chaque langue gérée par le projet.

Par défaut, les paramètres utilisés sont ceux définis dans les options linguistiques de Windows (accessible depuis le panneau de configuration de Windows).

Pour paramétrer ces options :

1. Sélectionnez la langue voulue dans la partie gauche de l'écran. 

2. Sélectionnez l'onglet correspondant à l'élément à paramétrer (nombre, monétaire, ...). 

3. Modifiez les paramètres voulus et validez. 




**Remarque** : Un bouton permet à tout moment de réinitialiser les paramètres avec les valeurs par défaut.

**Lors de la création d'un champ de saisie ou d'une colonne de table** affichant des données de type numérique (nombre, monétaire, date, heure, durée, ...), le masque de saisie utilisé sera automatiquement le masque défini dans les options de langue du projet. Cette option est disponible dans les fenêtres et les pages.

**En exécution**,  lorsque un champ de saisie ou une colonne de table a pour masque "Défini par le projet", le masque de saisie / d'affichage s'adaptera automatiquement selon les options choisies dans le projet pour la langue affichée en exécution.

**Remarque** : L'option "Divers" permet de définir : 

- le sens d'écriture.

- l'alphabet utilisé.

- le fichier WDM de traduction.




![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Les options linguistiques supportées sont les suivantes :

- Séparateur de décimales (nombre et monétaire)

- Séparateur de milliers (nombre et monétaire)

- Symbole monétaire

- Format d'heure

- Format de date

- Jour et Mois

- Unité des tailles de fichier

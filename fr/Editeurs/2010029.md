


## Propriétés d'un mot-clé
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000108"></a>
Toutes les caractéristiques d'un mot-clé peuvent être modifiées à tout moment dans les propriétés du mot-clé. La fenêtre des propriétés d'un mot-clé permet de :

- modifier le libellé du mot-clé (onglet "Général").

- traduire le mot-clé dans les différentes langes gérées par le système d'aide (onglet "Général").

- associer le mot-clé à une ou plusieurs pages d'aide (onglet "Associations").




**Pour afficher les propriétés du mot-clé** : 

1. Affichez la liste des mots clés d'index sous l'éditeur d'aide (onglet "Index" ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Aide_Ico_Index.gif)).

2. Sélectionnez l'entrée d'index voulue.

3. Affichez les propriétés du mot-clé (option "Propriétés du mot-clé" du menu contextuel).

## Propriétés d'un mot-clé
<a name="proprietes_motcle_ELTTEXTE000132"></a>


### Onglet Général
<a name="onglet_general_ELTPARAGRAPHE000030"></a>Cet onglet permet de saisir et de modifier le libellé du mot-clé dans toutes les langues gérées par le système d'aide.

**Rappel** : Il est également possible de modifier le libellé du mot-clé directement sous l'éditeur d'aide. Il suffit de :

1. Afficher l'aide dans la langue voulue : sous le volet "Modification", dans le groupe "Traduire", déroulez "Langue affichée" et sélectionnez la langue voulue.

2. Cliquer sur le mot-clé : le mot-clé passe en édition.

3. Modifier le libellé.



<a name="NOTE2_2"></a>


### Onglet Associations
<a name="onglet_associations_ELTPARAGRAPHE000055"></a>Cet onglet permet d'associer ou non le mot-clé en cours à une ou plusieurs pages d'aide.

Le bouton "Ajouter" permet d'ouvrir la liste des pages d'aides présentes dans le système d'aide en cours. Il suffit de sélectionner une page d'aide pour l'associer au mot-clé.

Le bouton "Enlever" permet de ne plus associer la page d'aide sélectionnée au mot-clé en cours.

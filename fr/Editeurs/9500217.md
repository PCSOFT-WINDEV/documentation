


## Gestion des projets épinglés
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000144"></a>
La page d'accueil de WINDEV, WEBDEV et WINDEV Mobile permet de gérer les projets épinglés lors de l'ouverture d'un projet existant.

Vous pouvez épingler vos projets préférés et ainsi y accéder d'un clic. Les projets épinglés sont proposés directement en haut de la liste des derniers projets ouverts.<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Projet_epingl%E9%20-%20HC%20N%B0002.gif)


<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000168"></a>


### Gestion des projets épinglés
<a name="gestion_des_projets_epingles_ELTPARAGRAPHE000021"></a>La gestion des projets épinglés est réalisée depuis la page d'accueil, en cliquant sur l'option "Ouvrir un projet".
<a name="NOTE2_2"></a>


### Epingler un projet
<a name="epingler_projet_ELTPARAGRAPHE000028"></a>Il est possible d'épingler (ou de punaiser) un projet : 

- soit en utilisant le menu contextuel du projet récent à épingler (option "Epingler le projet"). 

- soit en cliquant sur la punaise présente à la fin de la ligne du projet récent à épingler. 



<a name="NOTE2_3"></a>


### Trier et organiser les projets épinglés
<a name="trier_organiser_les_projets_epingles_ELTPARAGRAPHE000038"></a>Pour organiser les projets épinglés, il est possible de créer des dossiers dans les projets épinglés et d'affecter un projet épinglé à un dossier. 

Pour créer un dossier dans les projets épinglés : 

1. Cliquez sur l'icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Projet_epingl%E9%20-%20HC%20N%B0001.gif) présente en haut de la liste des projets épinglés.

2. Indiquez le nom du dossier à créer et validez. 

3. Le dossier est créé à la fin de la liste des projets épinglés.




Pour déplacer un projet épinglé dans un dossier :

1. Sélectionnez l'icône présente devant le nom du projet épinglé. 

2. Réalisez un Drag and Drop de l'icône vers le dossier voulu. 

3. Le projet épinglé est associé au dossier sélectionné. 




Remarque : Le menu contextuel du dossier permet de renommer ou supprimer le dossier. 
<a name="NOTE2_4"></a>


### Dé-punaiser des projets
<a name="depunaiser_des_projets_ELTPARAGRAPHE000061"></a>Pour dé-punaiser des projets, il est possible : 

- soit de cliquer sur la punaise sur la ligne du projet épinglé. 

- soit de sélectionner l'option "Enlever l'épingle du projet" du menu contextuel du projet épinglé.

<a name="NOTE3_1"></a>


## Note
<a name="note_ELTTEXTE000210"></a>


### Sauvegarde de la liste des projets épinglés
<a name="sauvegarde_liste_des_projets_epingles_ELTPARAGRAPHE000073"></a>La liste des projets épinglés est propre à chaque utilisateur. 

Cette liste est sauvegardée dans le fichier bookmark.prj présent dans le répertoire C:\\Users\\&lt;utilisateur&gt;\\AppData\\Local\\PC SOFT\\&lt;WinDev/WebDev/WinDev Mobile&gt;. 

Ainsi, à chaque nouvelle version du produit utilisé, les projets épinglés sont toujours proposés. 
<a name="NOTE3_2"></a>


### Ouverture rapide
<a name="ouverture_rapide_ELTPARAGRAPHE000084"></a>Les projets épinglés peuvent être ouverts : 

- par la page d'accueil du produit utilisé. Il est possible d'accéder à tout moment à la page d'accueil :

	- Cliquez sur l'icône 28 en haut de l'éditeur utilisé. 

	- Dans le menu contextuel qui s'affiche, sélectionnez "Bienvenue dans XXX". <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Projet_epingl%E9%20-%20HC%20N%B0003.gif)





- depuis les boutons d'accès rapide, déroulez ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_ouvrir.gif) et sélectionnez "Projets épinglés".

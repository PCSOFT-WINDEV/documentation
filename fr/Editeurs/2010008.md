


## Identifiant des pages d'aide
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000116"></a>
Pour lancer directement une page d'aide spécifique à partir de votre application WINDEV, il est nécessaire de spécifier un identifiant pour chaque page de votre système d'aide. 

Cet identifiant est unique. Il sera utilisé :

- soit directement dans la description des champs (onglet "Aide").

- soit en programmation avec la fonction [WHelp](../WDLang1/3013053.md).




La définition des identifiants pour les pages d'aide peut se faire :

- soit directement dans la description de la page. 
	**Remarque** : Cette méthode ne permet pas d'avoir une vision globale de tous les identifiants utilisés dans le système d'aide.

- soit dans la fenêtre récapitulative des identifiants de page : sous le volet "Modification", dans le groupe "Système d'aide", cliquez sur "Identifiants des pages d'aide".




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Fenêtre de gestion des identifiants de page
<a name="fenetre_gestion_des_identifiants_page_ELTTEXTE000140"></a>
Pour chaque page, cette fenêtre présente l'identifiant associé. Il est possible :

- soit de modifier l'identifiant des pages directement dans la liste des pages.

- soit d'utiliser une des fonctions de numérotation automatique :

	- **"Num Auto"** : permet de numéroter les pages sans identifiant. L'identifiant est automatiquement calculé à partir de l'identifiant le plus grand utilisé.

	- **"Efface tout"** : permet de remettre à zéro tous les identifiants des pages d'aide.

	- **"Doublons"** : permet de rechercher les identifiants en double.

	- **"Imprimer"** : permet d'imprimer la liste des pages et des identifiants associés.

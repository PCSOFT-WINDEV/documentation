<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000106"></a>
Un Centre de Contrôle permet d'avoir une vision globale du domaine couvert, que ce soit en phase de développement, sur un site d'exploitation, ou encore pour les aspects de maintenance ou d'évolution des applications.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Les Centres de Contrôle dédiés à la gestion de projet
<a name="les_centres_controle_dedies_gestion_projet_ELTTEXTE000130"></a>


### Présentation
<a name="presentation_ELTPARAGRAPHE000018"></a>WINDEV, WEBDEV et WINDEV Mobile permettent de suivre le développement de projets grâce au [Centre de Suivi de Projets](../CCSuivi/3540701.md). 

Le Centre de Suivi de Projets permet de gérer les différents événements de la vie d'un projet : Exigences, Tâches, Planning personnel, Incidents, ... A partir de ce centre, les intervenants peuvent indiquer les tâches réalisées ou à réaliser, les incidents corrigés, ... Ce Centre permet également de récupérer les retours client sur les applications déployées.

## Les Centres de Contrôle redistribuables
<a name="les_centres_controle_redistribuables_ELTTEXTE000160"></a>


### Plusieurs Centres de Contrôle redistribuables sont livrés avec WINDEV, WEBDEV et WINDEV Mobile :
<a name="plusieurs_centres_controle_redistribuables_sont_livres_avec_windev_webdev_windev_mobile_ELTPARAGRAPHE000046"></a>

- **[Centre de Contrôle des moteurs et bases de données HFSQL (réseau et Client/Serveur)](../CCHF/3540001.md)**
	Ce Centre de Contrôle (également nommé Centre de Contrôle HFSQL) permet de gérer les différentes bases de données HFSQL (Réseau, Mobile et essentiellement Client/Serveur) qui sont présentes dans l'entreprise. Ce Centre de Contrôle peut être utilisé lors du développement d'une application. Il est redistribuable avec les applications HFSQL Client/Serveur.

- **[Centre de Contrôle Utilisateur](../CCUser/3540201.md)**
	Ce Centre de Contrôle permet de gérer les applications déployées sur un réseau d'entreprise. A partir de ce Centre de Contrôle, les utilisateurs peuvent installer les applications qui leur sont dédiées. Le Centre de Contrôle utilisateur est redistribuable.

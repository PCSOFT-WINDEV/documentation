


## Fenêtre de sélection rapide
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000075"></a>
Vous recherchez une élément de votre projet ? Vous connaissez une partie de son nom uniquement ? Utilisez la fenêtre de sélection rapide grâce à la combinaison de touches Ctrl + E.

L'essayer, c'est l'adopter.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000099"></a>
A n'importe quel endroit de l'éditeur (éditeur de fenêtres, de pages, d'états, ...), vous pouvez utiliser le raccourci Ctrl + E. Ce raccourci ouvre automatiquement la fenêtre de sélection rapide.

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Ctrl_E.gif)
Cette fenêtre liste tous les éléments du projet, et permet de sélectionner et d'afficher l'élément voulu sous l'éditeur correspondant. Pour chaque élément, un aperçu est disponible, permettant de savoir rapidement si l'élément proposé correspond à l'élément recherché.

**Vous recherchez un élément spécifique** contenant le mot "Archive" ? Il suffit d'indiquer dans la zone de recherche "Archive". Tous les éléments contenant ce mot seront listés.

**Vous recherchez uniquement les éléments contenant les mots "FEN" et "Test"**? Il suffit d'indiquer dans la zone de recherche "FEN\*Test" ou "FEN Test". Le caractère "\*" peut correspondre à n'importe quel caractère ou ensemble de caractères.

L'option "Rechercher aussi les sous-éléments (champs, procédures, ...)" permet d'activer ou non la recherche dans les sous-éléments du projet. Si cette option est activée, la recherche sera également effectuée dans les champs, les procédures, ... Selon le nombre d'éléments dans le projet, cette recherche pourra être plus longue.

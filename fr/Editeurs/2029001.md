
## Environnement de l'éditeur de projet 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a><br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=EnvEdProjet.gif)


Les éléments importants de l'éditeur de projet sont les suivants :


|   |   |
| --- | --- |
| <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CERCLE1.gif)<br> | **Ruban**<br>Permet un accès rapide aux principales fonctionnalités de l'éditeur de projet. |
| <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CERCLE2.gif)<br> | **Tableau de bord du projet en cours**<br>Tableau de bord permettant de voir l'état du projet et d'accéder à tous les éléments du projet. |
| <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CERCLE3.gif)<br> | **[Volets ancrables verticaux](../Editeurs/2027001.md)**<br>Affiche :<br><br>- les raccourcis vers les principales fonctionnalités disponibles sous l'éditeur en cours,<br><br>- les fonctionnalités les plus utilisées,<br><br>- les éléments (fenêtres, états, champs, ...) disponibles dans le projet en cours.<br><br><br> |
| <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CERCLE4.gif)<br> | **Fenêtre de description**<br>Permet de visualiser et/ou de modifier les caractéristiques du projet. |
| <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Cercle5.gif)<br> | **[Documents ouverts](../Editeurs/2029012.md)**<br>Affiche un bouton pour chaque document ouvert sous l'éditeur. Pour afficher l'élément voulu au premier plan, il suffit de cliquer sur le "bouton" correspondant. |
| <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CERCLE6.gif)<br> | **[Volets ancrables horizontaux](../Editeurs/2027001.md)**<br>Volets permettant de :<br><br>- afficher les différents éléments du code (liste des classes, des procédures locales ou globales, [volet "Code"](../Editeurs/2027009.md)),<br><br>- afficher les erreurs de compilation (erreurs présentes dans le code WLangage, [volet "Erreurs de compilation"](../Editeurs/2027031.md)),<br><br>- effectuer des recherches ([volet "Rechercher - Remplacer"](../Editeurs/2027013.md)),<br><br>- afficher le débogueur lors du test d'un élément ([volet "Débogueur"](../Editeurs/2027030.md)),<br><br>- ...<br><br><br> |

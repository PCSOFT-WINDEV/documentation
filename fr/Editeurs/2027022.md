


## Le volet "Répertoire du site"
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000089"></a>
Le volet "Répertoire du site" permet de gérer le répertoire "&lt;Mon projet&gt;_WEB" du projet en cours.
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=VoletRepSite.gif)
Ce répertoire contient :

- les images,

- les fichiers Javascript,

- les fichiers applet Java,

- les autres fichiers accessibles depuis le navigateur.




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Afficher le volet "Répertoire du site"
<a name="afficher_volet_repertoire_site_ELTTEXTE000113"></a>
**Pour afficher le volet "Répertoire du site"**, sous le volet "Accueil", dans le groupe "Environnement", déroulez "Volets" et sélectionnez "Volets" puis "Répertoire du site".

Pour plus de détails sur les manipulations des volets, consultez [Manipuler les volets ancrables](../Editeurs/2027001.md).

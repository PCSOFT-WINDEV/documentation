<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000156"></a>
WEBDEV permet d' "aspirer" des pages HTML existantes. La page est importée dans le projet WEBDEV et chaque élément de la page est automatiquement transformé en élément WEBDEV (champ Libellé, champ HTML, Lien, ...).

Le principe de cette fonctionnalité repose essentiellement sur la récupération du look d'une page HTML.

Par exemple : La page HTML suivante...<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ImportPageHTML.gif)
... devient la page suivante dans WEBDEV :<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ImportPageWW.gif)
Remarque : Si vous importez une page ou des éléments Dremweaver, consultez [Importer des éléments Dreamweaver dans un site WEBDEV](../Editeurs/2030059.md).

## Comment le faire ?
<a name="comment_faire_ELTTEXTE000180"></a>


### Importer une page HTML dans un site WEBDEV
<a name="importer_une_page_html_dans_site_webdev_ELTPARAGRAPHE000031"></a>Pour importer une page HTML dans un site WEBDEV :

1. Ouvrez le projet WEBDEV dans lequel la page doit être importée.

2. Sous le volet "Projet", dans le groupe "Projet", déroulez "Importer" et sélectionnez "Une page Html".

3. Spécifiez le chemin ou l'URL de la page HTML à importer. L'aperçu de la page HTML à importer s'affiche.
	**Remarque** : Il est possible également de lancer l'import de la page par un simple Drag and Drop du fichier HTM dans l'éditeur de pages de WEBDEV.

4. Sélectionnez le mode d'import. Les styles et les images associées à la page importée peuvent être gérés de deux façons différentes :

	- Si l'option "J'importe une page indépendante dans le projet" est sélectionnée :

		- une feuille de styles WEBDEV (fichier ".WWS) est créée et automatiquement associée à la page importée. Cette feuille de styles contient tous les styles CSS de la page originale.

		- les différentes images associées à la page importée seront copiées dans un répertoire spécifique (sous-répertoire "&lt;NomPage&gt;" du répertoire du projet par défaut).




- Si l'option "J'importe plusieurs pages du même site dans le projet" est sélectionnée :

	- les styles de la page importée seront automatiquement ajoutés dans la feuille de styles associée au projet.

	- les différentes images associées à la page importée seront copiées dans le répertoire "&lt;NomProjet&gt;_WEB" du projet.

5. Validez. La page HTML est automatiquement intégrée au projet WEBDEV en cours.

	- Si des styles identiques sont identifiés dans la page HTML à importer, un seul style est créé dans la page importée. 

	- Si un style utilisé dans une page HTML déjà importée est détecté, ce même style sera réutilisé. 





**Remarque** : Lors de l'importation d'une page HTML, il est conseillé de ne pas réaliser de "copier-coller".
<a name="NOTE2_2"></a>


### Éléments importés
<a name="elements_importes_ELTPARAGRAPHE000078"></a>Le tableau suivant présente les principales transformations effectuées entre la page HTML d'origine et la page WEBDEV créée :

| Page HTML d'origine | Page WEBDEV créée |
| --- | --- |
| Lien | Champ Lien associé à l'action du lien d'origine |
| Applet Java | Champ Applet Java associé à l'Applet Java d'origine |
| Balise &lt;OBJECT&gt; et &lt;BLINK&gt; | Champ HTML avec le contenu des balises d'origine |
| Image | Champ Image associé à l'image d'origine |
| Libellé | Champ Libellé ou Libellé HTML avec le contenu du libellé d'origine. |

 

**Remarque** : Les maparea ne sont pas importées.

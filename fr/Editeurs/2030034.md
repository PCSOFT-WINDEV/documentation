


## Importer une fenêtre WINDEV dans un site WEBDEV
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000113"></a>
Les fenêtres Windows réalisées avec WINDEV (version 7.5 ou supérieure) peuvent être importées dans un site WEBDEV. Les fenêtres sont alors automatiquement transformées en pages WEBDEV. Les champs et le code de la fenêtre sont récupérés.

L'import de fenêtres WINDEV permet de gagner énormément de temps lors du passage d'une application Windows à une application Internet ou Intranet.

**Attention** : Les pages résultant de l'importation peuvent être un peu différentes des fenêtres originales. Il est quelques fois nécessaire de remanier leur interface et leur code.

**Remarques** : 

- Il est également possible de copier les champs de la fenêtre directement dans une page. Pour plus de détails, consultez [Copie entre les éditeurs](../Editeurs/9000137.md).

- A partir de la version 24, l'assistant de webisation permet de transformer une application WINDEV en un site Internet. Pour plus de détails, consultez [Transformez vos applications WINDEV existantes en sites Internet](../Editeurs/9500225.md).

<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000137"></a>


### Importer une fenêtre WINDEV dans un site WEBDEV
<a name="importer_une_fenetre_windev_dans_site_webdev_ELTPARAGRAPHE000037"></a>Pour importer une fenêtre WINDEV dans un site WEBDEV :

1. Ouvrez le projet WEBDEV dans lequel la fenêtre WINDEV doit être importée.

2. Sous le volet "Projet", dans le groupe "Projet", déroulez "Importer" et sélectionnez "Des éléments WINDEV ou WINDEV Mobile".

3. Sélectionnez la fenêtre WINDEV (fichier ".WDW") à importer. La fenêtre WINDEV se transforme automatiquement en page WEBDEV.



<a name="NOTE2_2"></a>


### Importation des éléments WINDEV dans la page WEBDEV
<a name="importation_des_elements_windev_dans_page_webdev_ELTPARAGRAPHE000060"></a>Les éléments suivants ne sont pas importés dans la page WEBDEV créée :

- les champs WINDEV non présents dans WEBDEV (champ Potentiomètre, champ Jauge, ...).

- les fonctions, propriétés, mots-clés du WLangage non disponibles en WEBDEV (fonction **OuvreFenêtre**, mot-clé **MaFenêtre**, ...).

- les gabarits WINDEV. Si la fenêtre importée est associée à un gabarit, seules les images associées à la page et à ses champs seront importées.

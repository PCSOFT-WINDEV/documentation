


## Zoomer une fenêtre (FAA)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000106"></a>
Pour faciliter la lecture des fenêtres, l'utilisateur final peut zoomer le contenu d'une fenêtre à l'aide de la souris.<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=WD-Zoom%20Fenetre-6.jpg)
**Pour zoomer une fenêtre**, l'utilisateur doit utiliser la touche Ctrl et la roulette de la souris.

**Pour revenir à l'affichage initial**, il suffit d'utiliser l'option "Annuler le zoom" du menu contextuel de la fenêtre (ou la combinaison de touches Ctrl + 0).  

Remarque : Si nécessaire, des ascenseurs peuvent apparaître lors du zoom. 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Programmation
<a name="programmation_ELTTEXTE000130"></a>


### Programmer le zoom
<a name="programmer_zoom_ELTPARAGRAPHE000024"></a>Il est possible de programmer le zoom à appliquer à la fenêtre en utilisant la propriété [Zoom](../Proprietes/1000017201.md) sur la fenêtre. 

Par exemple : 

```wl
// Agrandissement à 150%
FEN_Client..Zoom = 150
```
Si le zoom est réalisé par programmation, le menu contextuel de la fenêtre permet d'annuler le zoom. 


### Désactiver le zoom
<a name="desactiver_zoom_ELTPARAGRAPHE000036"></a>Pour interdire le zoom sur une fenêtre, il suffit d'utiliser la fonction [FAADésactive](../WDLang1/1000022018.md) avec la constante *faaZoom*.


### Annuler le zoom
<a name="annuler_zoom_ELTPARAGRAPHE000044"></a>Pour annuler le zoom sur une fenêtre (et revenir à l'état initial), il suffit d'utiliser la fonction [FAAExécute](../WDLang1/1000022099.md) avec la constante *faaZoom*.


### Modifier le libellé de l'option permettant d'annuler le zoom (par programmation)
<a name="modifier_libelle_option_permettant_annuler_zoom_par_programmation_ELTPARAGRAPHE000052"></a>Pour annuler le zoom, l'utilisateur final peut afficher le menu contextuel de la fenêtre (clic droit) et sélectionner l'option "Annuler le zoom". 

Pour modifier le libellé de cette option, il suffit d'utiliser la fonction [FAAChangeLibellé](../WDLang1/1000022100.md) avec la constante *faaZoom*.

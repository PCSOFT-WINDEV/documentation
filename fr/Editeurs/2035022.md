## Comment le faire ?
<a name="comment_faire_ELTTEXTE000234"></a>


### Créer un diagramme de séquence
<a name="creer_diagramme_sequence_ELTPARAGRAPHE000039"></a>Pour créer un diagramme de séquence : 

1. Cliquez sur ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_nouveau.gif)
 parmi les boutons d'accès rapide. 

	- La fenêtre de création d'un nouvel élément s'affiche : cliquez sur "Architecture" puis sur "UML".

	- L'assistant de création d'un modèle UML se lance.




2. Sélectionnez l'option "Diagramme de séquence".

3. Spécifiez le modèle UML dans lequel le diagramme sera inséré.

4. Choisissez d'éditer directement le diagramme.

5. Validez la création du diagramme de séquence. Le modèle créé devient le modèle en cours.



<a name="NOTE2_2"></a>


### Créer un objet
<a name="creer_objet_ELTPARAGRAPHE000061"></a>Pour créer un objet : 

1. Sous le volet "Diagramme de séquence", dans le groupe "Création", cliquez sur "Objet". Un nouvel objet apparaît dans le modèle UML.

2. Sélectionnez l'option "Description" du menu contextuel (clic droit) du nouvel objet. La fenêtre de spécifications s'ouvre.

3. Spécifiez :

	- le type de l'objet.

	- le nom de l'objet. Ce nom est affiché sous l'éditeur d'UML.

	- la classe de l'objet.

	- le stéréotype de l'objet.

	- si la ligne de vie de l'objet doit être affiché ou non.




4. Validez la description de l'objet.



<a name="NOTE2_3"></a>


### Ajouter une période d'activation à un objet
<a name="ajouter_une_periode_activation_objet_ELTPARAGRAPHE000091"></a>Pour ajouter une période d'activation à un objet : 

1. Sélectionnez l'objet auquel une période d'activation doit être ajoutée.

2. Sous le volet "Diagramme de séquence", dans le groupe "Création", cliquez sur "Période d'activation". Une nouvelle période d'activation apparaît dans le modèle UML.



<a name="NOTE2_4"></a>


### Créer un message
<a name="creer_message_ELTPARAGRAPHE000113"></a>Pour créer un message :

1. Sous le volet "Diagramme de séquence", dans le groupe "Création", cliquez sur "Message".

2. Sélectionnez les différents éléments à relier. La fenêtre de spécifications s'ouvre.

3. Spécifiez :

	- le type du message.

	- le nom du message. Ce nom est affiché sous l'éditeur d'UML.




4. Validez la description du message.



<a name="NOTE2_5"></a>


### Créer un paquetage
<a name="creer_paquetage_ELTPARAGRAPHE000140"></a>Pour créer un paquetage : 

1. Sous le volet "Diagramme de séquence", dans le groupe "Création", cliquez sur "Paquetage". Un nouveau paquetage apparaît dans le modèle UML.

2. Sélectionnez l'option "Description" du menu contextuel (clic droit) du nouveau paquetage. La fenêtre de spécifications du paquetage s'ouvre.

3. Spécifiez :

	- le nom du paquetage. Ce nom est affiché dans le paquetage sous l'éditeur d'UML.

	- le libellé du paquetage décrivant l'intérêt du paquetage.

	- les relations du paquetage.




4. Validez la description du paquetage.

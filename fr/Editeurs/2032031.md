


## Fenêtre de description d'une requête composée 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000126"></a>
La fenêtre de description d'une requête composée est constituée de plusieurs parties. Cette fenêtre de description permet de :

- changer les requêtes utilisées dans la requête.

- modifier l'ordre d'exécution des requêtes.
	Rappel : Les rubriques apparaissant dans le résultat de la requête composée correspondent aux rubriques de la première requête exécutée.

- visualiser la requête en code SQL ou en langage naturel.

- gérer les enregistrements "doublons".

- trier les enregistrements.


Cette fenêtre de description s'affiche :

- lors de la création d'une requête de sélection. 
	Rappel : Pour créer une requête : 

	- Cliquez sur ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_nouveau.gif) parmi les boutons d'accès rapide. 

	- La fenêtre de création d'un nouvel élément s'affiche : cliquez sur "Requêtes".




- depuis une requête composée affichée sous l'éditeur de requêtes (option "Description de la requête" du menu contextuel de la requête).

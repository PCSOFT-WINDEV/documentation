


## Réparation de l'analyse
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000097"></a>
Dans certains cas, des messages d'erreur peuvent apparaître lors de l'ouverture de l'analyse liée à un projet. Les messages d'erreur peuvent être par exemple :

- Le fichier recherché est inconnu.

- Fichier non trouvé dans deffic.




Il est possible de réparer directement l'analyse.

**Remarque** : Il est conseillé de réaliser une sauvegarde de l'analyse avant de la réparer.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000121"></a>


### Réparer une analyse
<a name="reparer_une_analyse_ELTPARAGRAPHE000025"></a>Pour réparer une analyse :

1. Sous le volet "Analyse", dans le groupe "Analyse", déroulez "Maintenance" et sélectionnez "Réparer l'analyse".

2. Sélectionnez l'analyse à réparer. Il est possible de réparer :

	- L'analyse en cours.

	- Une analyse spécifique. Il suffit alors de sélectionner le fichier ".wda" correspondant à l'analyse à réparer.




3. Sélectionnez le mode de réparation à effectuer :

	- **Réparation rapide** : les fichiers de données correspondant à la description de l'analyse sont réindexés, l'analyse est synchronisée avec sa représentation graphique et elle est mise à niveau.

	- **Réparation complète** : Réparation rapide avec vérification de la cohérence de l'analyse. Il est possible de choisir les actions à effectuer.




4. Validez. La réparation de l'analyse est effectuée.

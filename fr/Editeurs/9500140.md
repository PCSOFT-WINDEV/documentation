## Présentation
<a name="presentation_ELTTEXTE000145"></a>
Le champ Interrupteur peut bénéficier automatiquement de plusieurs FAA (Fonctionnalités Automatiques de vos Applications) : 

- [Cocher tout](#NOTE2_1).

- [Décocher tout](#NOTE2_1).

- [Mémoriser la valeur sélectionnée](#NOTE3_1).

## Cocher tout / Décocher tout
<a name="cocher_tout_decocher_tout_ELTTEXTE000169"></a>


### Fonctionnement
<a name="fonctionnement_ELTPARAGRAPHE000034"></a>L'option "Cocher tout" permet de cocher toutes les options de l'interrupteur. 

L'option "Décocher tout" permet de décocher toutes les options de l'interrupteur. 


### Mise en place
<a name="mise_place_ELTPARAGRAPHE000041"></a>Les options "Cocher tout" et "Décocher tout" dans les champs Interrupteur sont automatiquement mises en place à partir de la version 20. Ces options sont disponibles sur tous les types de champs Interrupteur (standard, Vista, ...).  

Comme tous les menus contextuels proposés par les FAA, il est possible de configurer les options proposées par défaut par le champ. Pour plus de détails, consultez [Configurer les menus FAA](../Editeurs/2010040.md).

**En programmation** : 

- la fonction [FAADésactive](../WDLang1/1000022018.md) utilisée avec la constante *faaInterrupteurCocheTout* permet de supprimer les options permettant de cocher/décocher tout dans le menu contextuel du champ Interrupteur spécifié.

- la fonction [FAAExécute](../WDLang1/1000022099.md) utilisée avec la constante *faaInterrupteurCocheTout* permet de cocher toutes les options du champ Interrupteur spécifié.

- la fonction [FAAChangeLibellé](../WDLang1/1000022100.md) permet de modifier le libellé du menu contextuel permettant de cocher toutes les options d'un champ Interrupteur.

## Mémorisation de la valeur
<a name="memorisation_valeur_ELTTEXTE000199"></a>


### Fonctionnement
<a name="fonctionnement_ELTPARAGRAPHE000072"></a>Pour utiliser la mémorisation de la valeur sélectionnée d'un champ Liste/Combo, sélectionnez l'option "Mémoriser la valeur" du menu contextuel du champ.


### Mise en place
<a name="mise_place_ELTPARAGRAPHE000077"></a>L'option "Mémoriser la valeur" est disponible automatiquement dans les champs Combo et Liste. Cette option permet de gérer la [persistance des données](../WDChamp/9000019.md). 

Comme tous les menus contextuels proposés par les FAA, il est possible de configurer les options proposées par défaut par le champ. Pour plus de détails, consultez [Configurer les menus FAA](../Editeurs/2010040.md).

**En programmation** : La fonction [FAADésactive](../WDLang1/1000022018.md) utilisée avec la constante *faaSauveValeur* permet de supprimer l'option "Mémoriser la valeur" dans le menu contextuel du champ Interrupteur spécifié.

**Remarque** : Il est également possible de mémoriser la valeur grâce à l'option "Mémoriser la valeur" présente dans la fenêtre de description du champ (onglet "Détail"). Cette option permet de mémoriser la valeur même si l'utilisateur n'a pas activée l'option de menu. Si l'utilisateur a utilisé l'option de menu, l'option de menu est prioritaire.

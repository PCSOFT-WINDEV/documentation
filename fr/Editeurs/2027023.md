## Présentation
<a name="presentation_ELTTEXTE000093"></a>
Une messagerie sécurisée est incluse dans WINDEV, WINDEV Mobile ou WEBDEV.

Cette messagerie permet l'échange de messages entre les différents collaborateurs de l'entreprise.

La messagerie est disponible sous plusieurs formes :

- dans le volet "Messagerie" sous l'éditeur WINDEV, WEBDEV, et WINDEV Mobile. Pour plus de détails, consultez [Le volet "Messagerie"](../Editeurs/2027021.md). 

- sous forme d'un programme indépendant : WDBal. Pour plus de détails, consultez [Messagerie WDBal](../Editeurs/1410086599.md). 

- dans un onglet du Centre de Suivi de projets. Pour plus de détails, consultez [Centre de Suivi de projets : Gestion des messages](../CCSuivi/3540726.md)




**Remarque** : La messagerie utilise les informations enregistrées dans les Centres de Contrôle. Les différents collaborateurs pris en compte pour la messagerie sont les collaborateurs enregistrés dans les différents Centres de Contrôle. Les messages envoyés par la messagerie sont également enregistrés dans la base de données des Centres de Contrôle. Pour plus de détails, consultez [Base de données des Centres de Contrôle](../CCSuivi/3540720.md).

## Fonctionnalités de la messagerie
<a name="fonctionnalites_messagerie_ELTTEXTE000117"></a>
Quelque soit sont mode d'utilisation, la messagerie intégrée propose toutes les fonctionnalités d'une messagerie avancée : 

- Gestion des messages : Nouveau message, Accusé de réception, Suppression de messages, Transfert de messages, ...

- Gestion de la recherche : recherche selon l'expéditeur, la date, parmi les messages envoyés ou reçus, ...

- Gestion des dossiers pour conserver les messages reçus et les trier par thèmes.

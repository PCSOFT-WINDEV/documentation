## Configuration du serveur pour le protocole CGI
<a name="configuration_serveur_pour_protocole_cgi_ELTTEXTE000134"></a>
Pour configurer le serveur Web : 

1. Editez le fichier "httpd.conf" sous un éditeur de texte. 
	Ce fichier est présent dans le sous-répertoire "/Conf" du répertoire d'installation du serveur Apache.

2. Ajoutez dans le fichier "httpd.conf" la définition de l'alias du protocole CGI (WD280AWP.EXE) par la ligne suivante :
	
	```txt
	ScriptAlias /<Alias>/ "Répertoire du moteur"
	```

	**Exemple : Poste de développement** : le moteur est situé dans le répertoire "C:\\WEBDEV\\PROGRAMMES\\MOTEUR\\WINDOWS\\AWP" et le nom de l'alias est WD280AWP :
	
	```txt
	ScriptAlias /WD280AWP/ "C:/WEBDEV/PROGRAMMES/MOTEUR/WINDOWS/AWP/"
	```

	**Exemple : Poste de déploiement** : le moteur est situé dans le répertoire "C:\\WEBDEV\\MOTEUR\\WINDOWS\\AWP" et le nom de l'alias est WD280AWP :
	
	```txt
	ScriptAlias /WD280AWP/ "C:/WEBDEV/MOTEUR/WINDOWS/AWP/"
	```


3. Redémarrez le serveur Apache.




**Remarques**

- les caractères \\ deviennent des /

- ne pas oublier le / final dans le chemin.

- respecter les différences entre majuscules / minuscules.

## Configuration du serveur pour les sites WEBDEV
<a name="configuration_serveur_pour_les_sites_webdev_ELTTEXTE000158"></a>
Pour configurer le serveur Web :

1. Editez le fichier "httpd.conf" sous un éditeur de texte. Ce fichier est présent dans le sous-répertoire "/Conf" du répertoire d'installation du serveur Apache.

2. Ajoutez dans le fichier "httpd.conf" la définition de l'alias des images de chaque site WEBDEV :
	
	```txt
	Alias /<Nom alias Image>/ "Répertoire des images de l'application"
	```

	Par exemple pour le site WEBDEMO : 
	
	```txt
	Alias /WEBDEMO_WEB/ "C:/WEBDEMO/WEBDEMO_WEB/"
	```


3. Redémarrez le serveur Apache.





<a name="NOTE2_2"></a>


### Attention
<a name="attention_ELTPARAGRAPHE000065"></a>

- Avec le serveur Web Apache, le nom de l'alias est "case sensitive". Il est nécessaire de respecter la casse définie pour l'alias dans les pages HTML faisant référence au répertoire des images.

- Il est nécessaire de définir un alias d'images pour chaque site WEBDEV.

## Configuration du serveur pour les sites utilisant des pages AWP (Active Web Pages)
<a name="configuration_serveur_pour_les_sites_utilisant_des_pages_awp_active_web_pages_ELTTEXTE000188"></a>
Pour configurer le serveur Web :

1. Editez le fichier "httpd.conf" sous un éditeur de texte. Ce fichier est présent dans le sous-répertoire "/Conf" du répertoire d'installation du serveur Apache.

2. Dans la partie "AddType", ajoutez la ligne suivante :
	
	```txt
	AddType application/WEBDEV28-awp .awp
	```


3. Dans la partie "Action", ajoutez la ligne suivante :
	
	```txt
	Action application/WEBDEV28-awp /WD280AWP/WD280AWP.EXE
	```





**Remarque** : dans cette ligne, WD280AWP correspond à l'alias du moteur.


## Propriétés d'un lien
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000114"></a>
Lors de la création d'un lien, il suffit de préciser la page qui doit être ouverte lors du clic sur le lien. Par défaut, le lien créé permet d'ouvrir la page spécifiée dans la page d'aide en cours.

Cependant, il est possible de définir le type du lien et la destination de la page à afficher dans les propriétés du lien.

**Pour afficher les propriétés du lien** :

1. Positionnez le curseur de la souris sur le lien voulu.

2. Sélectionnez l'option "Propriétés du lien" du menu contextuel (clic droit de la souris). 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=proprieteLien.gif)





Remarque : Les liens sont disponibles dans tous les modes de génération du système d'aide.

<a name="NOTE2_1"></a>


## Propriétés d'un lien
<a name="proprietes_lien_ELTTEXTE000138"></a>
Les informations affichées dans cette fenêtre sont les suivantes :

- **Texte du lien** : Libellé du lien. Ce libellé sera souligné dans la page d'aide. Ce soulignement permet à l'utilisateur de savoir qu'un lien est disponible.

- **Hyperlien vers** : Nature du lien. Un lien permet d'ouvrir :

	- soit une autre page du système d'aide.

	- soit une page Internet.

	- soit une ancre d'une page.

	- soit un lien vers une page d'un autre fichier d'aide. Attention : le fichier WDH correspondant à ce second système d'aide doit être présent dans le répertoire du projet.




- **Page destination** : Page devant être ouverte par le lien. Cette option correspond :

	- soit à une des pages du système d'aide en cours.

	- soit à une adresse Internet (par exemple www.pcsoft.fr).
			

	- soit à une ancre sur une des pages du système d'aide en cours.

	- soit à une page d'un autre système d'aide présent dans le répertoire du projet.




- **Mode d'ouverture** : Mode d'ouverture de la page ouverte par le lien. Les options possibles sont :

	- *Fenêtre courante* : La page utilise la même description que la fenêtre d'aide en cours.

	- * Fenêtre popup* (ouverture d'une page d'aide uniquement) : La page d'aide est affichée sous forme de popup (bulle d'aide). 
			**Attention** : Dans ce cas, la page d'aide ne doit pas posséder de zone fixe.

	- *Fenêtre spécifique* : La page d'aide sera ouverte dans la fenêtre d'aide sélectionnée.




## Modèles de pages prédéfinis
			





## Présentation
<a name="presentation_ELTTEXTE000089"></a>
Les modèles de page prédéfinis fournissent des structures de pages correspondant à une grande variétés de sites et favorisent la ré-utilisabilité et l'harmonie graphique entre les projets.

De nombreux modèles sont livrés avec WEBDEV et de nouveaux modèles sont livrés régulièrement avec [la LST](../Presentation/3084010.md).



## Utilisation des modèles de pages prédéfinis
<a name="utilisation_des_modeles_pages_predefinis_ELTTEXTE000113"></a>
Pour utiliser un modèle de pages prédéfini :

1. Cliquez sur ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_nouveau.gif)
 parmi les boutons d'accès rapide.

2. La fenêtre de création d'un nouvel élément s'affiche : cliquez sur "Page" puis sur "Page".

3. Dans la fenêtre de création d'une nouvelle page, choisissez un modèle dans la partie "Utiliser un modèle prédéfini".




WEBDEV ajoute automatiquement deux éléments dans le projet :

- un modèle de pages correspondant à celui choisi et utilisant l'ambiance et la palette de couleur du projet.

- une page vierge utilisant le modèle.





### Remarques
<a name="remarques_ELTPARAGRAPHE000031"></a>

- Si le modèle prédéfini est déjà utilisé dans le projet, une nouvelle copie sera ajoutée. Si vous souhaitez que plusieurs pages soient basées sur le même modèle prédéfini, il faut :

	1. Créer une première page avec la méthode indiquée ci-dessus afin d'importer le modèle prédéfini dans le projet.

	2. Lors de la création des autres pages, choisir le modèle ajouté dans le projet. Ce modèle sera proposé dans la partie "Créer une page vierge ou associée à un modèle du projet".




- Il est possible d'ajouter plusieurs fois le même modèle prédéfini dans un projet pour l'utiliser avec plusieurs ambiances ou plusieurs palettes de couleurs par exemple.

- Après avoir été ajouté au projet, le modèle prédéfini peut être ouvert dans l'éditeur et modifié comme n'importe quel autre modèle de pages. Pour obtenir plus d'informations, consultez [Modèles de pages](../WDChamp/1012015.md).

- Un modèle de pages prédéfini peut également être utilisé comme base de création d'un autre modèle de pages.

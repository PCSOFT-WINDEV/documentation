


## Relations dans un MCD
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000123"></a>
Les relations permettent de lier les entités entre elles ou les associations aux entités.

L'éditeur d'analyses permet de créer facilement des relations.

Lors de la [génération du MCD en MLD](../Editeurs/2011020.md), les relations deviendront automatiquement des [liaisons](../Editeurs/2011008.md).

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Création d'une relation
<a name="creation_une_relation_ELTTEXTE000147"></a>
Pour créer une relation :

1. Sous le volet "MCD", dans le groupe "Création", déroulez "Nouveau" et sélectionnez "Tracer une relation". Le curseur de la souris se transforme en crayon.

2. Sélectionnez :

	- soit les deux entités à relier.

	- soit l'entité et l'association à relier.




3. Définissez les cardinalités de la relation.




<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Définition des cardinalités
<a name="definition_des_cardinalites_ELTTEXTE000171"></a>
Pour définir les cardinalités :

1. Sélectionnez la relation dont les cardinalités doivent être définies.

2. Sous le volet "MCD", dans le groupe "Élément courant", cliquez sur "Description".

3. Choisissez la cardinalité parmi celles proposées.

4. Validez la description des cardinalités.




Pour plus de détails sur les cardinalités, consultez les [Cardinalités](../Editeurs/2011014.md).




## WDInst : paramétrer les caractéristiques des fichiers à installer 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000199"></a>
L'éditeur d'installation WDInst permet de paramétrer les caractéristiques des fichiers à installer. Il est possible de :

- Spécifier le répertoire de destination d'un ou de plusieurs fichiers.

- Spécifier les postes sur lesquels chacun des fichiers doit être installé (cas d'une installation réseau).

- Copier et compacter un ou plusieurs fichiers.




Par défaut :

- Le répertoire de destination des fichiers correspond au répertoire d'installation choisi par l'utilisateur lors de l'installation.

- Les fichiers de données (".FIC", ".NDX", ".MMO") sont installés sur le poste serveur et tous les autres fichiers sont installés sur les postes utilisateurs.

- Les fichiers sont compactés lors de la génération du programme d'installation et installés sur le poste d'installation.




Le paramétrage des caractéristiques des fichiers à installer est réalisé sous l'éditeur d'installation WDInst. Pour lancer WDInst, sous le volet "Outils", dans le groupe "Utilitaires", cliquez sur "WDInst".

L'éditeur d'installation WDInst propose une gestion avancée du paramétrage des caractéristiques des fichiers à installer par rapport à l'assistant de création du programme d'installation.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Spécifier le répertoire de destination d'un ou de plusieurs fichiers à installer
<a name="specifier_repertoire_destination_plusieurs_fichiers_installer_ELTTEXTE000223"></a>
Le répertoire de destination a été spécifié lors de l'ajout du fichier dans la liste des fichiers à installer.

Pour modifier le répertoire de destination d'un ou de plusieurs fichiers :

1. Sélectionnez le ou les fichiers voulus dans la fenêtre "Liste des fichiers de l'installation".
	Pour sélectionner plusieurs fichiers simultanément, maintenez la touche Ctrl ou Maj enfoncée lors de la sélection des fichiers.

2. Affichez l'onglet "Chemin" de la fenêtre "Description du fichier".

3. Spécifiez le répertoire de destination du ou des fichiers sélectionnés (option "Répertoire"). Il est possible d'utiliser les mots-clés suivants :

	- **&lt;RepInst&gt;** : répertoire d'installation choisi par l'utilisateur lors de l'installation.

	- **&lt;LectInst&gt;** : lecteur du répertoire d'installation choisi par l'utilisateur lors de l'installation.

	- **&lt;Langue&gt;** : numéro de la langue d'installation.

	- **&lt;FicInst&gt;** : répertoire d'installation des fichiers de données. L'éditeur d'installation WDInst permet de paramétrer ce répertoire. Pour plus de détails, consultez [Paramétrer le répertoire des fichiers de données](../Editeurs/2028017.md).

	- **&lt;RepDonneesUser&gt;** : répertoire des données de l'utilisateur connecté (notamment pour une installation réalisée sur un poste Windows Vista ou supérieur). 

	- **&lt;RepDonneesCommun&gt;** : répertoire des données communes aux utilisateurs (notamment pour une installation réalisée sur un poste Windows Vista ou supérieur). 

	- **&lt;User&gt;** : Nom de l'utilisateur

	- **&lt;srXXX&gt;** : Constantes de la fonction [SysRep](../WDLang1/3073026.md). 
			Par exemple : "&lt;RepInst&gt;\\MonRépertoire".




4. Si un seul fichier est sélectionné, il est possible de renommer ce fichier lors de son installation. Il suffit de spécifier un nouveau nom du fichier (option "Fichier").




**Remarque** : Il est également possible de modifier le répertoire de destination d'un fichier directement dans la fenêtre "Liste des fichiers de l'installation". Il suffit de cliquer dans la colonne "Répertoire de destination" et de saisir le nouveau répertoire de destination.

## Spécifier les postes sur lesquels chacun des fichiers doit être installé (installation réseau)
<a name="specifier_les_postes_sur_lesquels_chacun_des_fichiers_doit_etre_installe_installation_reseau_ELTTEXTE000247"></a>


### Si vous réalisez une installation réseau
<a name="vous_realisez_une_installation_reseau_ELTPARAGRAPHE000079"></a>

- Uniquement sur le poste serveur (par exemple les fichiers de données partagés par tous les utilisateurs).

- Uniquement sur les postes utilisateurs (par exemple l'exécutable de l'application).

- A la fois sur le poste serveur et sur les postes utilisateurs (par exemple un fichier ".INI" permettant de configurer l'application installée sur le serveur avec les paramètres par défaut et les applications installées sur les postes utilisateurs avec les paramètres spécifiques aux utilisateurs).




**Pour spécifier les postes sur lesquels un ou plusieurs fichiers doivent être installés :**

1. Sélectionnez le ou les fichiers voulus dans la fenêtre "Liste des fichiers de l'installation".
	Pour sélectionner plusieurs fichiers simultanément, maintenez la touche Ctrl ou Maj enfoncée lors de la sélection des fichiers.

2. Affichez l'onglet "Option" de la fenêtre "Description du fichier".

3. Pour installer le ou les fichiers sélectionnés :

	- Sur le poste serveur, cochez l'option "Ce fichier s'installe sur le serveur".

	- Sur les postes utilisateurs, cochez l'option "Ce fichier s'installe sur le client".

	- Sur le poste serveur et sur les postes utilisateurs, cochez l'option "Ce fichier s'installe sur le serveur" et l'option "Ce fichier s'installe sur le client".

## Copier et compacter un ou plusieurs fichiers
<a name="copier_compacter_plusieurs_fichiers_ELTTEXTE000271"></a>


### Présentation
<a name="presentation_ELTPARAGRAPHE000102"></a>Par défaut, chaque fichier sélectionné pour l'installation sera :

- Compacté lors de la génération du programme d'installation.

- Installé sur le poste d'installation.




L'éditeur d'installation WDInst permet de modifier ce mode de fonctionnement par défaut.

Par exemple, votre installation utilise un "autorun". Cet "[autorun](../Editeurs/2028015.md)" permet de lancer un programme de présentation des différentes fonctionnalités de votre application présente sur le CD.

Pour cet exemple, l'éditeur d'installation WDInst offre la possibilité :

- Soit de ***ne pas compacter*** les fichiers utilisés par ce programme de présentation. Ce programme pourra ainsi être exécuté directement depuis le CD. Ce programme sera installé lors de l'installation de l'application.

- Soit de ***ne pas compacter et de ne pas installer*** les fichiers utilisés par ce programme de présentation. Ce programme pourra ainsi être exécuté directement depuis le CD. Ce programme **ne sera pas** installé lors de l'installation de l'application.



<a name="NOTE4_2"></a>


### Ne pas compacter un ou plusieurs fichiers lors de la génération du programme d'installation
<a name="pas_compacter_plusieurs_fichiers_lors_generation_programme_installation_ELTPARAGRAPHE000124"></a>Pour ne pas compacter un ou plusieurs fichiers lors de la génération du programme d'installation : 

1. Sélectionnez le ou les fichiers à ne pas compacter dans la fenêtre "Liste des fichiers de l'installation".
	Pour sélectionner plusieurs fichiers simultanément, maintenez la touche Ctrl ou Maj enfoncée lors de la sélection des fichiers.

2. Affichez l'onglet "Option" de la fenêtre "Description du fichier".

3. Cochez l'option "Ne pas compacter le fichier".



<a name="NOTE4_3"></a>


### Ne pas compacter et ne pas installer un ou plusieurs fichiers sur le poste d'installation
<a name="pas_compacter_pas_installer_plusieurs_fichiers_sur_poste_installation_ELTPARAGRAPHE000137"></a>Pour ne pas compacter et ne pas installer un ou plusieurs fichiers sur le poste d'installation : 

1. Sélectionnez le ou les fichiers voulus dans la fenêtre "Liste des fichiers de l'installation".
	Pour sélectionner plusieurs fichiers simultanément, maintenez la touche Ctrl ou Maj enfoncée lors de la sélection des fichiers.

2. Affichez l'onglet "Option" de la fenêtre "Description du fichier".

3. Cochez l'option "Copier dans la racine du CD ou de la dernière disquette".




Ces fichiers seront uniquement copiés :

- A la racine du CD, si le support d'installation correspond à un CD.

- A la racine de la dernière disquette, si le support d'installation correspond à des disquettes.

- Dans le répertoire unique spécifié, si le support d'installation correspond à un répertoire unique.

- Dans le dernier sous-répertoire créé, si le support d'installation correspond à un répertoire découpé à la taille d'une disquette.

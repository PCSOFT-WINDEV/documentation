


## Options de l'éditeur de fenêtres
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000295"></a>
Pour configurer les options générales de l'éditeur de fenêtres : 

1. Sous le volet "Accueil", dans le groupe "Environnement", déroulez "Options" et sélectionnez "Options de l'éditeur de fenêtres". 

2. La fenêtre de configuration est affichée. Les modifications possibles sont regroupées en 4 thèmes : 

	- Magnétisme.

	- Clic.

	- Options.

	- Champs.







<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Magnétisme
<a name="magnetisme_ELTTEXTE000319"></a>


### Paramétrage de la grille
<a name="parametrage_grille_ELTPARAGRAPHE000038"></a>La grille magnétique facilite le positionnement et l'alignement des champs de la fenêtre.

La taille de la grille est exprimée en pixels.

Options d'utilisation conseillées pour la grille :

- Grille invisible.

- Grille active.

- Espacement vertical de la grille : 12 pixels.

- Espacement horizontal de la grille : 12 pixels.




Si l'espacement de la grille est égale à 1, la grille ne sera pas visualisée.

Pour plus de détails, consultez [Grille magnétique](../Editeurs/2022002.md).


### Positionnement automatique
<a name="positionnement_automatique_ELTPARAGRAPHE000059"></a>Lors du déplacement d'un champ dans une fenêtre, le correcteur d'interface en temps réel (appelé aussi positionnement automatique) affiche automatiquement des guides. Ces guides sont magnétisés et permettent d'aligner le champ déplacé avec les champs présents dans la fenêtre.

Il est possible de rendre le positionnement actif ou non et les règles actives ou non. 

Pour plus de détails, consultez [Positionnement automatique](../Editeurs/2022003.md). 


### Paramétrage des règles
<a name="parametrage_des_regles_ELTPARAGRAPHE000071"></a>Sous l'éditeur de fenêtres, lors de la création ou la modification d'une fenêtre, il est possible de faire apparaître des règles d'alignement. 

Dans ces règles, des guides magnétiques sont affichés : tout champ approché d'un guide est automatiquement "attiré" par celui-ci. Ce magnétisme permet d'utiliser les guides pour faciliter le positionnement, l'alignement ou le redimensionnement des champs de la fenêtre.

Il est possible de rendre les règles et les guides visibles ou non et de paramétrer la couleur des guides. 

Pour plus de détails, consultez [Règles d'alignement (Editeur)](../Editeurs/2022001.md).

## Clic
<a name="clic_ELTTEXTE000355"></a>


### Action du double-clic
<a name="action_doubleclic_ELTPARAGRAPHE000089"></a>Par défaut, le double-clic sur un champ édite la fenêtre de description du champ. 

Il est possible de modifier ce comportement par défaut. 

Le double-clic peut : 

- **Éditer la description** : Afficher la fenêtre de description du champ (valeur par défaut).

- **Ouvrir le code** : Ouvrir la fenêtre de code associée au champ. 

- **Choisir un style** : Ouvrir la fenêtre permettant de choisir le style du champ. 





### Simple clic
<a name="simple_clic_ELTPARAGRAPHE000114"></a>Un simple clic sur un champ effectue une action particulière. Un simple clic sur le champ permet de :

- Editer le libellé du champ (valeur par défaut). 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=LibelleModifDirect.gif)


- Adapter la taille du champ après édition du libellé par simple clic. 

- Modifier si possible le nom du champ après modification du libellé par simple clic (valeur par défaut).

## Options
<a name="options_ELTTEXTE000385"></a>


### A chaque sélection
<a name="chaque_selection_ELTPARAGRAPHE000128"></a>A chaque sélection d'un champ dans la fenêtre, il est possible : 

- d'afficher les propriétés du champ sélectionné dans le volet "Modifieur". Pour plus de détails, consultez [Le volet "Modifieur"](../Editeurs/2027006.md). 

- d'afficher les traitements de l'élément sélectionné dans le volet "Visualisation contextuelle du code". Pour plus de détails, consultez [Le volet "Visualisation contextuelle du code"](../Editeurs/2027033.md). 





### Position de la fenêtre WINDEV dans la fenêtre d'accueil
<a name="position_fenetre_windev_dans_fenetre_accueil_ELTPARAGRAPHE000142"></a>Par défaut, la fenêtre WINDEV est placée au centre de la fenêtre d'accueil. Il est également possible de la placer en haut à gauche.

## Champs
<a name="champs_ELTTEXTE000421"></a>


### Liste des polices
<a name="liste_des_polices_ELTPARAGRAPHE000164"></a>Pour modifier la police des champs d'une fenêtre, il suffit d'utiliser la combo "Police" de l'onglet "Style" de la description des champs.

Par défaut, dans la combo "Police", seul le nom des polices installées sur le poste de développement est affiché.

Sous Windows, une même police peut être associée à plusieurs groupes de langues. Pour compléter chaque nom de police par la langue associée, il suffit de cocher l'option "Afficher toutes les langues disponibles".

Remarque : Cette option sera prise en compte uniquement à la prochaine ouverture d'une fenêtre.


### Autres options
<a name="autres_options_ELTPARAGRAPHE000175"></a>Les autres options disponibles sont les suivantes : 

- **Lettre d'appel automatique lors de la création d'un champ** : Cette option permet de créer automatiquement une lettre d'appel lors de la création du champ. Cette lettre d'appel peut être modifiée par la suite. Pour plus de détails, consultez [Raccourci clavier et lettre d'appel](../WDChamp/1014018.md). 

- **Afficher un symbole de représentation dans les champs Image en édition** : Cette option permet d'afficher ou non la mire dans les champs Image. 
	![Image avec mire](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Image_Mire%20-%20HC%20N%B0001.gif)
![Image sans mire](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Image_Mire%20-%20HC%20N%B0001%201.gif)

	Cette option peut également être désactivée champ par champ. Pour plus de détails, consultez [Le champ Image](../WDChamp/1013218.md). 

- **Calculer les ancrages lors du copier/coller** : Cette option permet de calculer automatiquement les meilleures options d'ancrage lors du copier/coller d'un champ dans une fenêtre. 

- **Forcer l'aperçu des fenêtres internes à 100%** : Permet de forcer l'aperçu des fenêtres internes à 100% dans les champs de type Fenêtre interne. Ainsi, la fenêtre interne est correctement visualisée et il est possible d'adapter sous l'éditeur la taille du champ Fenêtre interne. Pour plus de détails, consultez [Le champ Fenêtre interne](../WDChamp/1013255.md). 

- **Afficher un demi-cadre sur le champ survolé** : Permet de visualiser les limites du champ lors de son survol. 

- **Afficher une erreur de compilation d'UI lorsque la fenêtre n'est pas à jour du gabarit** : Permet d'afficher ou non une erreur d'UI spécifique si le gabarit n'est pas à jour. 

- **Adapter le zoom au DPI de l'écran à l'ouverture des fenêtres** : Permet d'optimiser la taille d'affichage de la fenêtre sous l'éditeur. 

- **Afficher l'ombre sous les fenêtres (sous l'éditeur)** : Permet d'afficher ou non une ombre sous le contour des fenêtres en édition sous l'éditeur de fenêtres. 

- **Désactiver l'audit d'UI pour tous les projets** : Permet de désactiver par défaut l'audit d'UI sur tous les projets. Il pourra être réactivé uniquement sur les projets voulus. 

- **Taille de saisie par défaut pour les champs de saisie** : Permet de définir la taille par défaut (en nombre de caractères) des champs de saisie lors de leur création.

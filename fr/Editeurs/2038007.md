


## Réintégrer un élément dans le gestionnaire de sources (GDS)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000118"></a>
La réintégration d'un élément dans la base de sources permet de mettre à disposition de tous les utilisateurs du gestionnaire de sources un élément qui a été modifié. L'élément modifié en local est recopié dans le projet de référence sur le serveur GDS.

**Remarque** : Les éléments extraits du Gestionnaire de sources apparaissent entourés d'un trait rouge sous l'éditeur de projet.

**Pour réintégrer un élément dans la base de sources** : 

1. Sélectionnez cet élément dans le volet "Explorateur de projet".

2. Sélectionnez l'option "Réintégrer" du menu contextuel.




Autre possibilité : sous le volet "GDS", dans le groupe "Elément en cours", cliquez sur "Réintégrer".

**Important** : La feuille de styles est un élément spécifique du projet. Pour réintégrer la feuille de styles, sous le volet "GDS", dans le groupe "Autres actions", déroulez "Feuille de styles" et sélectionnez "Réintégrer la feuille de styles du projet".

## Réintégration d'un élément
<a name="reintegration_element_ELTTEXTE000142"></a>
Lors de la réintégration d'un élément dans le GDS, l'écran suivant est affiché :
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Int%E9gration%20dans%20le%20GDS%20-%20HC%20N%B0010.gif)
Vous pouvez :

- Dans l'onglet "Tâches", indiquer que l'élément réintégré correspond à l'exécution d'une tâche ou à la réalisation d'une tâche.

- Dans l'onglet "Incidents", indiquer que l'élément réintégré corrige un incident.

- Dans l'onglet "Commentaire", saisir un commentaire sur la réintégration. Par exemple, vous pouvez indiquer que la fenêtre propose désormais une nouvelle option de menu.

- Connaître les modifications effectuées sur l'élément (bouton "Mes Modifs").

- Visualiser les propriétés de l'élément (bouton "Propriétés").




Il est possible de réintégrer les modifications effectuées dans l'élément, tout en conservant l'élément extrait (option "Garder l'élément extrait").

Lorsque un élément est réintégré dans le gestionnaire de sources et non extrait, il est entouré d'un trait noir sous l'éditeur de projet.

**Remarque** : si l'élément réintégré n'a pas été modifié, le GDS affiche un écran spécifique.

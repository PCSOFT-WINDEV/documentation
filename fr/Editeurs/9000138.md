
## Copie des champs depuis l'éditeur de fenêtres
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000609"></a>
Il est possible de copier les champs de l'éditeur de fenêtres : 

- soit vers l'éditeur de pages

- soit vers l'éditeur d'états. 




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>

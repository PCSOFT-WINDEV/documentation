## Rubriques pouvant être utilisées dans la requête
<a name="rubriques_pouvant_etre_utilisees_dans_requete_ELTTEXTE000101"></a>
Cette partie de la fenêtre de description d'une requête liste principalement les différentes rubriques pouvant être utilisées dans la requête.

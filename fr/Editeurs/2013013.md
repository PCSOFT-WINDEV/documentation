
## Historique de navigation dans le code
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000078"></a>
Les différents éditeurs permettent de visualiser directement le code d'un élément en utilisant la touche F2.

**Par exemple**, l'utilisation de la touche F2 sur le champ d'une fenêtre permet de lancer l'éditeur de code pour afficher le code de ce bouton.

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Code_Histo_Navigation.gif)


La touche F2 permet également de se déplacer dans les différentes procédures imbriquées dans l'éditeur de code.

Pour revenir à l'élément précédent, il suffit d'utiliser la combinaison de touches Ctrl + F2.

**Par exemple**, l'utilisation de la touche F2 sur le champ d'une fenêtre permet de lancer l'éditeur de code pour afficher le code de ce bouton. L'utilisation de Ctrl + F2 permet de fermer la fenêtre de code et de revenir sur le bouton original.

Une navigation plus complète peut également être effectuée : 

- en utilisant les boutons suivant et précédent de la barre d'outils de code. 

- à l'aide des combinaisons de touches Alt + Flèche Gauche et Alt + Flèche Droite. Les combinaisons Alt + Home et Alt + Fin positionnent au début et à la fin de l'historique. 




Ces options de navigation dans le code prennent également en compte tous les positionnements du curseur dans les  différents traitements.




## Tester un projet statique
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000151"></a>
Le test de l'ensemble du projet permet de simuler le lancement du site. Il est ainsi possible de tester le site dans son ensemble, même si son développement n'est pas terminé.

**Remarque** : Le test d'un projet peut être lancé quel que soit l'élément en cours sous l'éditeur.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000175"></a>


### Tester un site statique (ou la partie statique d'un site)
<a name="tester_site_statique_partie_statique_site_ELTPARAGRAPHE000020"></a>Pour tester un site statique depuis l'éditeur :

1. Sous le volet "Projet", dans le groupe "Mode test", déroulez "Mode test" et sélectionnez "Déboguer le projet depuis la page d'accueil".

2. L'éditeur est automatiquement réduit en icône et le projet s'exécute dans le navigateur par défaut.

3. La page d'accueil s'affiche. 




Au lancement du test du projet, avant l'ouverture de la première page du projet, les traitements suivants sont exécutés :

- chargement de la première page (code navigateur).

- entrée du premier champ de la première page (si ce champ a un traitement d'entrée) (code navigateur).




**Remarque** : Si votre site ne comporte pas de page d'accueil, WEBDEV vous propose de définir une page d'accueil pour votre site. Pour plus de détails, consultez [Page d'accueil](../WDChamp/2030010.md).

**Attention** : Pour tester un site statique, le serveur Web installé sur le poste de développement doit être lancé. 
<a name="NOTE2_2"></a>


### Arrêter le test d'un projet
<a name="arreter_test_projet_ELTPARAGRAPHE000056"></a>Pour arrêter le test, plusieurs méthodes sont possibles :

- 1ère méthode : Fermez le navigateur dans lequel le test est effectué. WEBDEV affiche l'éditeur en cours au moment du lancement du test.

- 2ème méthode : Revenez dans l'éditeur avec la barre des tâches ou avec Alt + Tab et confirmez l'arrêt du test. WEBDEV affiche l'éditeur en cours au moment du lancement du test.

## Notes
<a name="notes_ELTTEXTE000205"></a>


### Navigateur de test
<a name="navigateur_test_ELTPARAGRAPHE000068"></a>Pour sélectionner le navigateur de test, sous le volet "Projet", dans le groupe "Mode test", déroulez "Mode test" et sélectionnez "Navigateur de test...".

Le navigateur par défaut est défini dans les options de WEBDEV : 

- Sous le volet "Accueil", dans le groupe "Environnement", déroulez "Options" et sélectionnez "Options générales de WEBDEV". 

- Affichez l'onglet "Web".


Pour plus de détails, consultez [Options de WEBDEV](../Editeurs/2036001.md).

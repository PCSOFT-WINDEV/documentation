


## Configurer l'interface du programme d'installation 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000400"></a>
Pour rendre l'installation de votre application plus conviviale, l'assistant d'installation propose de personnaliser l'interface du programme d'installation. Ces options sont disponibles dans les étapes concernant le gabarit de l'installateur. 

Deux types de personnalisation de l'interface sont disponibles :

- [personnalisation de l'interface pour une installation standard ou réseau](#NOTE2_1).

- [personnalisation de l'interface pour une installation compacte](#NOTE3_1). 




**Remarque** : La personnalisation de l'interface peut être différente selon la langue d'exécution du programme d'installation.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Personnaliser l'interface du programme d'installation (installation standard ou réseau)
<a name="personnaliser_interface_programme_installation_installation_standard_reseau_ELTTEXTE000424"></a>


### Présentation
<a name="presentation_ELTPARAGRAPHE000151"></a>Par défaut, le programme d'installation correspond à une simple fenêtre dans laquelle les différentes options d'installation s'affichent. 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Installation_WD_defaut%20-%20HC%20N%B0001.gif&type=thumb)

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Installation_WD_defaut%20-%20HC%20N%B0004.gif&type=thumb)


Pour personnaliser votre programme d'installation, il est possible de :

- afficher une image en fond de la fenêtre. 

- afficher une fenêtre de fond derrière la fenêtre d'installation. Cette fenêtre peut afficher : 

	- un libellé ou un logo.

	- une couleur dégradée ou une image pour le décor de fond.








<a name="NOTE2_2"></a>


### Personnaliser la fenêtre du programme d'installation
<a name="personnaliser_fenetre_programme_installation_ELTPARAGRAPHE000182"></a>Il est possible d'utiliser une image de décor spécifique dans la fenêtre de l'assistant d'installation. 

Pour personnaliser la fenêtre du programme d'installation : 

1. Dans l'étape "Gabarit de l'installeur" de l'assistant, sélectionnez l'option "Afficher une image personnalisée". 

2. Sélectionnez l'image à utiliser. Il est conseillé d'utiliser une image de 708 x 460 pixels. 

3. Continuez l'assistant. 



<a name="NOTE2_3"></a>


### Utiliser une fenêtre de fond dans le programme d'installation
<a name="utiliser_une_fenetre_fond_dans_programme_installation_ELTPARAGRAPHE000195"></a>Une fenêtre de fond peut être affichée derrière la fenêtre d'installation de l'application WINDEV. 

Pour personnaliser la fenêtre de fond du programme d'installation : 

1. Dans l'étape "Titre et logo" de l'assistant, sélectionnez la langue pour laquelle l'interface doit être personnalisée.

2. Saisissez le titre de l'installation. Par défaut, ce titre correspond au nom de l'application. Ce titre sera affiché dans la barre de titre de la fenêtre du programme d'installation. 

3. Cliquez sur le bouton "Logo".

4. Configurez l'interface du programme d'installation.




**Par défaut, l'assistant d'installation n'est pas affiché sur une fenêtre de fond.**

Pour obtenir toutes les options, sélectionnez "Afficher la fenêtre de fond de l'installation".

## Personnaliser l'interface du programme d'installation (installation compacte)
<a name="personnaliser_interface_programme_installation_installation_compacte_ELTTEXTE000460"></a>
Par défaut, l'interface de l'installation compacte est la suivante :
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=installation%20compacte%20-%20HC%20N%B0002.gif)
L'assistant de création d'installation permet de personnaliser et de traduire :

- les textes de cette interface

- les messages d'erreur affichés.



<a name="NOTE3_2"></a>


### Pour personnaliser et traduire l'interface de l'installation compacte :
<a name="pour_personnaliser_traduire_interface_installation_compacte_ELTPARAGRAPHE000263"></a>

1. Dans l'étape "Titre et logo" de l'assistant, sélectionnez la langue pour laquelle l'interface doit être personnalisée.

2. Saisissez le titre de l'installation. Par défaut, ce titre correspond au nom de l'application. Ce titre sera affiché dans la barre de titre de la fenêtre du programme d'installation. 

3. Cliquez sur le bouton "Logo" de l'assistant.

4. Saisissez les nouveaux messages dans la langue sélectionnée.![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=installation%20compacte%20-%20HC%20N%B0002.gif)

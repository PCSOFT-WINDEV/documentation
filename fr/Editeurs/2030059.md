


## Importer des éléments Dreamweaver dans un site WEBDEV
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000166"></a>
WEBDEV permet d'importer des éléments Dreamweaver dans votre site WEBDEV. Il est possible d'importer : 

- [une bibliothèque Dreamweaver](#NOTE2_1) : la bibliothèque sera automatiquement convertie en modèle de champs. 

- [un modèle de page Dreamweaver](#NOTE2_2) : le modèle de page Dreamweaver sera automatiquement converti en modèle de pages WEBDEV. 

- [une page Dreamweaver](#NOTE2_3) : la page est importée avec ses bibliothèques et ses modèles. 




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000190"></a>


### Importer une bibliothèque Dreamweaver
<a name="importer_une_bibliotheque_dreamweaver_ELTPARAGRAPHE000028"></a>Pour importer une bibliothèque Dreamweaver dans un site WEBDEV :

1. Ouvrez le projet WEBDEV dans lequel la page doit être importée.

2. Sous le volet "Projet", dans le groupe "Projet", déroulez "Importer" et sélectionnez "Une bibliothèque Dreamweaver".

3. Spécifiez le chemin ou l'URL de la bibliothèque à importer (fichier d'extension ".lbi"). L'aperçu s'affiche.

4. Sélectionnez le mode d'import. 

5. Validez. La bibliothèque est automatiquement intégrée au projet WEBDEV en cours sous forme de modèle de champs.



<a name="NOTE2_2"></a>


### Importer un modèle de page Dreamweaver
<a name="importer_modele_page_dreamweaver_ELTPARAGRAPHE000053"></a>Pour importer une page HTML dans un site WEBDEV :

1. Ouvrez le projet WEBDEV dans lequel la page doit être importée.

2. Sous le volet "Projet", dans le groupe "Projet", déroulez "Importer" et sélectionnez "Un modèle de page Dreamweaver".

3. Spécifiez le chemin ou l'URL du modèle à importer. L'aperçu s'affiche.

4. Sélectionnez le mode d'import. 

5. Validez. Le modèle de pages est automatiquement intégré au projet WEBDEV en cours. Si le modèle utilise une bibliothèque ou un "template", ces éléments sont automatiquement détectés. 

6. Si les modèles de champs ou de pages sont déjà présents dans le projet, ces éléments sont automatiquement réutilisés par le modèle importé. 

7. Si les modèles de champs ou de pages ne sont pas présents dans le projet : 

	- Si les sources des bibliothèques et des templates sont disponibles, l'importation de ces éléments est automatiquement proposée. 

	- Si les sources des bibliothèques et des templates sont indisponibles ou si l'importation a été refusée, une rétro-analyse de la page est effectuée : les modèles de pages et de champs nécessaires sont automatiquement créés et associés à la page.






<a name="NOTE2_3"></a>


### Importer une page Dreamweaver
<a name="importer_une_page_dreamweaver_ELTPARAGRAPHE000083"></a>Pour importer une page HTML dans un site WEBDEV :

1. Ouvrez le projet WEBDEV dans lequel la page doit être importée.

2. Sous le volet "Projet", dans le groupe "Projet", déroulez "Importer" et sélectionnez "Une page Html".
	**Remarque** : Il est également possible de réaliser directement un "Drag And Drop" de la page vers l'éditeur de pages de WEBDEV.

3. Spécifiez le chemin ou l'URL de la page HTML à importer. L'aperçu de la page HTML à importer s'affiche.

4. Sélectionnez le mode d'import. Les styles et les images associées à la page importée peuvent être gérés de deux façons différentes :

	- Si l'option "J'importe une page indépendante dans le projet" est sélectionnée :

		- une feuille de styles WEBDEV (fichier ".WWS) est créée et automatiquement associée à la page importée. Cette feuille de styles contient tous les styles CSS de la page originale.

		- les différentes images associées à la page importée seront copiées dans un répertoire spécifique (sous-répertoire "&lt;NomPage&gt;" du répertoire du projet par défaut).




- Si l'option "J'importe plusieurs pages du même site dans le projet" est sélectionnée :

	- les styles de la page importée seront automatiquement ajoutés dans la feuille de styles associée au projet.

	- les différentes images associées à la page importée seront copiées dans le répertoire "&lt;NomProjet&gt;_WEB" du projet.

5. Validez. La page HTML est automatiquement intégrée au projet WEBDEV en cours. Si la page utilise une bibliothèque ou un "template", ces éléments sont automatiquement détectés. 

6. Si les modèles de champs ou de pages sont déjà présents dans le projet, ces éléments sont automatiquement réutilisés par la page importée. 

7. Si les modèles de champs ou de pages ne sont pas présents dans le projet : 

	- Si le source des bibliothèques et des templates sont disponibles, l'importation de ces éléments est automatiquement proposée. 

	- Si le source des bibliothèques et des templates sont indisponibles ou si l'importation a été refusée, une rétro-analyse de la page est effectuée : les modèles de pages et de champs nécessaires sont automatiquement créés et associés à la page.

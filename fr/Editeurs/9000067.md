## Présentation
<a name="presentation_ELTTEXTE000089"></a>
Le "FishEye" permet d'agrandir une partie des graphes des éditeurs (graphe de l'analyse, graphe UML, ....) en survolant une zone. Son action est équivalente à celle d'une loupe. Par exemple :<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=FishEye.gif)

## Comment le faire ?
<a name="comment_faire_ELTTEXTE000113"></a>


### Configurer le FishEye
<a name="configurer_fisheye_ELTPARAGRAPHE000019"></a>Pour configurer le "FishEye" :

1. Affichez les options du produit utilisé : sous le volet "Accueil", dans le groupe "Environnement", déroulez "Options" et sélectionnez "Options générales de WINDEV/WEBDEV/WINDEV Mobile".

2. Dans l'onglet "Editeur", sélectionnez le mode d'affichage du "FishEye" :

	- **Jamais** : le mode "FishEye" ne sera jamais activable.

	- **Lorsque le zoom est entre 25% et 75%** : le mode "FishEye" sera activable en fonction de la valeur du zoom.

	- **Lorsque le zoom est inférieur à 100%** : le mode "FishEye" sera activable en fonction de la valeur du zoom.

	- **Lorsque Ctrl et Maj sont enfoncés** : le mode "FishEye" sera activable en fonction d'une combinaison de touches.




3. Validez.

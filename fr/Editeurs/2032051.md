
## Requête de requête 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000077"></a>
Une requête de requête est une requête réalisée à partir de rubriques provenant d'une autre requête. La requête initiale doit avoir été exécutée avant la requête de requête.

Ce type de requête est disponible :

- en programmation, avec les fonctions [HExécuteRequête](../WDLang4/3044080.md) et [HExécuteRequêteSQL](../WDLang4/3044084.md).

- sous l'éditeur de requêtes.




**Attention** : Ne pas confondre "Requête de requête" et "Sous-requête". Pour plus de détails sur les sous-requêtes, consultez [Sous-requête](../Editeurs/2034009.md).

**Limitation** : Les requêtes de requête ne sont pas disponibles sur les bases externes (via OLE DB, ODBC ou Connecteur Natif). En effet, la base de données ne connaît pas la requête originale à partir de laquelle la seconde requête est créée.

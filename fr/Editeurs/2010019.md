## Recherche en mode intégral
<a name="recherche_mode_integral_ELTTEXTE000100"></a>
La recherche en mode intégral permet de disposer de l'onglet "Recherche" dans vos fichiers d'aide.

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Aide_Rechercher.gif)


Cet onglet permet à l'utilisateur de construire une base de mots à partir du contenu des pages d'aide. Cette base de mots permet de rechercher toutes les pages d'aide utilisant un mot spécifique. Cette base de mots (de taille plus ou moins importante) est stockée dans un fichier d'extension FTS.

WINDEV donne la possibilité de paramétrer la base de mots qui sera construite. Si l'option "Utiliser la recherche en mode intégral" est cochée, il est possible de :

- **Inclure les pages invisibles de la table des matières** : si cette option est cochée, la recherche se fera sur toutes les pages du système d'aide, sinon uniquement sur les pages présentes dans le sommaire.

- **Autoriser la recherche par phrase** : si cette option est cochée, l'utilisateur pourra saisir une phrase pour effectuer sa recherche. Seules les pages affichant cette phrase seront proposées.

- **Afficher les phrases trouvées** : si cette option est cochée, lors de la recherche d'une phrase, cette phrase sera mise en surbrillance dans les pages correspondant à la recherche.

- **Autoriser la recherche par similitude** : si cette option est cochée, la recherche sélectionnera tous les mots ayant la même racine.




**Remarque** : Les options sélectionnées seront les options utilisées par défaut pour créer la base de mots. L'utilisateur pourra à tout moment modifier ces options.

## Compression du fichier d'aide
<a name="compression_fichier_aide_ELTTEXTE000124"></a>
Cette option permet de compresser ou non le fichier d'aide. La compression du fichier d'aide permet d'optimiser la place occupée par le fichier HLP. Cette compression est conseillée.

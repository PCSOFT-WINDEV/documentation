## Création d'une rubrique
<a name="creation_une_rubrique_ELTTEXTE000238"></a>
Pour créer une rubrique : 

1. Double-cliquez sur le fichier de données dans lequel la rubrique doit être créée. La fenêtre de description des rubriques du fichier de données s'affiche.

2. Cliquez sur la première ligne vide du tableau des rubriques ou cliquez sur le bouton ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Manipulations_rubriques%20-%20HC%20N%B0001%201.GIF)
.

3. Spécifiez directement dans le tableau : 

	- le nom de la rubrique,

	- le libellé de la rubrique, 

	- le type de la rubrique et si nécessaire sa taille, 

	- si la rubrique concerne des données personnelles (coche RGPD). Pour plus de détails, consultez [Définition des rubriques de type "Données personnelles"](../Editeurs/1410086744.md).


 Remarque : Pour plus de détails sur les différents types de rubriques, consultez [Les différents types de rubrique](../Editeurs/2011026.md).

4. Indiquez dans la partie droite de l'écran les détails concernant la nouvelle rubrique (type de clé, sens de parcours, format, etc.).

5. Validez la description de la rubrique.

## Manipulation d'une rubrique
<a name="manipulation_une_rubrique_ELTTEXTE000262"></a>


### Dupliquer/Copier une rubrique
<a name="dupliquercopier_une_rubrique_ELTPARAGRAPHE000068"></a>Lors de la création d'une rubrique dont le nom est identique à celui d'une rubrique existante, WINDEV, WEBDEV et WINDEV Mobile proposent automatiquement d'utiliser la description de la rubrique d'origine.

Il est également possible de dupliquer une rubrique : 

- dans un même fichier.

- dans un fichier différent.  




Pour dupliquer une rubrique dans un même fichier, directement depuis la fenêtre de description des rubriques : 

1. Sélectionnez la rubrique à dupliquer dans la description des rubriques d'un fichier de données.

2. Cliquez sur le bouton ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Manipulations_rubriques%20-%20HC%20N%B0001%209.gif)
. La rubrique est automatiquement dupliquée. Son nom correspond au nom de la rubrique d'origine suffixée d'un chiffre.




Pour dupliquer une rubrique d'un fichier dans un autre fichier : 

1. Sélectionnez la rubrique à dupliquer dans la description des rubriques d'un fichier de données.

2. Cliquez sur le bouton ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Manipulations_rubriques%20-%20HC%20N%B0001%2010.gif)
. 

3. Dans la fenêtre qui s'affiche, indiquez la description de fichier dans laquelle la rubrique est automatiquement dupliquée. 

4. Validez. La rubrique apparaît dans la description du fichier sélectionné.



<a name="NOTE3_2"></a>


### Supprimer une rubrique
<a name="supprimer_une_rubrique_ELTPARAGRAPHE000098"></a>Pour supprimer une rubrique :

1. Sélectionnez la rubrique à supprimer dans la description des rubriques d'un fichier de données.

2. Cliquez sur le bouton ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Manipulations_rubriques%20-%20HC%20N%B0001%202.gif)
. La rubrique est automatiquement supprimée du fichier de données.




**Remarques** : 

- Lors de la création d'une rubrique dont le nom est identique à celui d'une rubrique précédemment supprimée, WINDEV, WEBDEV et WINDEV Mobile proposent automatiquement de restaurer la description de la rubrique supprimée.

- La suppression d'une rubrique entraîne la suppression des données lors de la modification automatique des données qui suivra, même si une nouvelle rubrique de même nom est créée.



<a name="NOTE3_3"></a>


### Renommer une rubrique
<a name="renommer_une_rubrique_ELTPARAGRAPHE000115"></a>Pour renommer une rubrique :

1. Sélectionnez la rubrique à renommer dans la description des rubriques d'un fichier de données.

2. Saisissez directement le nouveau nom de la rubrique.

3. Validez la modification. La rubrique est automatiquement renommée.



<a name="NOTE3_4"></a>


### Utiliser un dictionnaire
<a name="utiliser_dictionnaire_ELTPARAGRAPHE000126"></a>Une rubrique peut simplement être intégrée dans un dictionnaire pour être ré-utilisée par exemple dans une nouvelle application. L'intégration de la rubrique dans un dictionnaire se fait par simple "Drag and Drop" depuis l'éditeur d'analyses, vers le dictionnaire (affiché dans le [volet "Dictionnaire"](../Editeurs/2027014.md)).

Lors de son intégration dans le dictionnaire, la description de la rubrique peut être abonnée aux modifications. Toute modification réalisée dans la description de la rubrique pourra être reportée sur l'objet du dictionnaire et sur toutes les utilisations de cet objet.

Pour plus de détails, consultez le [Dictionnaire](../Editeurs/2023002.md).

## Trier les rubriques affichées dans la description des fichiers de données
<a name="trier_les_rubriques_affichees_dans_description_des_fichiers_donnees_ELTTEXTE000304"></a>
L'affichage des rubriques dans un fichier de données peut s'effectuer selon les critères suivants : 

- ordre logique. 

- ordre physique. 

- ordre alphabétique.




Pour modifier le mode d'affichage des rubriques : 

1. Double-cliquez sur le fichier de données. La fenêtre de description des rubriques du fichier de données s'affiche. 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Manipulations_rubriques%20-%20HC%20N%B0001.gif&type=thumb)


2. Les boutons de la zone "Affichage" permettent de trier les rubriques affichées dans cette fenêtre : 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Manipulations_rubriques%20-%20HC%20N%B0001%203.gif)


	- ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Manipulations_rubriques%20-%20HC%20N%B0001%204.gif)
 permet d'effectuer un tri alphabétique. 

- ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Manipulations_rubriques%20-%20HC%20N%B0001%205.gif)
 permet d'afficher les rubriques et index selon l'ordre physique. 

- ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Manipulations_rubriques%20-%20HC%20N%B0002.gif)
 permet d'afficher les rubriques et index selon l'ordre logique. 

3. L'entonnoir présent dans la colonne "Nom" (![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Manipulations_rubriques%20-%20HC%20N%B0001%206.gif)
) permet d'effectuer une recherche : il suffit de cliquer sur la l'entonnoir et de saisir les lettres de la rubrique recherchée. Toutes les rubriques dont le nom contiennent ces lettres seront affichées.




## Propriétés d'une image (système d'aide)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000120"></a>
Toutes les caractéristiques d'une image insérée dans une page d'aide peuvent être modifiées dans les propriétés de l'image. La fenêtre de gestion des propriétés de l'image permet de :

- visualiser et sélectionner le fichier image.

- gérer les zones cliquables de l'image.




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Propriétés d'une image
<a name="proprietes_une_image_ELTTEXTE000144"></a>


### Afficher les propriétés
<a name="afficher_les_proprietes_ELTPARAGRAPHE000021"></a>Pour afficher les propriétés de l'image :

1. Sélectionnez l'image.

2. Sélectionnez l'option "Propriétés de l'image" du menu contextuel (clic droit de la souris).<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Aide_image.gif&type=thumb)




<a name="NOTE2_2"></a>


### Sélection et visualisation de l'image
<a name="selection_visualisation_image_ELTPARAGRAPHE000032"></a>Le fichier image en cours est affiché en pré-visualisation. 
Il est possible à tout moment de modifier cette image grâce au bouton ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Menu_Image_Editeur%20-%20HC%20N%B0001.gif) : 

- L'option "Parcourir" ouvre un sélecteur de fichiers. 

- L'option "Catalogue" permet d'ouvrir le catalogue d'images livré avec WINDEV.



<a name="NOTE2_3"></a>


### Gestion des zones cliquables
<a name="gestion_des_zones_cliquables_ELTPARAGRAPHE000045"></a>Les zones cliquables permettent de définir sur les images des zones dans lesquelles l'utilisateur va pouvoir cliquer. Lors du survol de ces zones, le curseur de la souris se transformera en main, indiquant à l'utilisateur qu'il peut cliquer.

Une même image peut contenir plusieurs zones cliquables.

**Pour créer une zone cliquable** :

1. Cliquez sur le bouton "Nouvelle".

2. Définissez le lien associé à la zone de cliquage. Ce lien peut afficher une page ou une ancre du système d'aide en cours, une page d'un site Internet ou une page d'un autre système d'aide. Pour plus de détails sur les liens, consultez [Propriétés d'un lien](../Editeurs/2010022.md). 

3. Définissez la zone de cliquage soit à l'aide de la souris, soit grâce aux champs "Paramètres de la zone de clicage".
	Remarque : Pour que la zone cliquable recouvre entièrement l'image, cliquez sur le bouton "Toute l'image". 




**Pour supprimer une zone cliquable** :

1. Sélectionnez la zone cliquable à supprimer dans la liste.

2. Cliquez sur le bouton "Supprimer".

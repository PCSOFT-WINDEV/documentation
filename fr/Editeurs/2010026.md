


## Insérer une image ou une vidéo dans une page d'aide
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000200"></a>
Dans les pages d'aide, il est souvent utile d'utiliser des images ou des vidéos pour expliquer les manipulations à faire, désigner les boutons à utiliser, ...

En plus d'images "simples", il est possible d'intégrer des images cliquables dans les pages d'aide : lorsque l'utilisateur clique sur l'image (ou sur une partie de l'image), une page d'aide spécifique peut être affichée (sous forme de page ou de popup).

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Gestion des images
<a name="gestion_des_images_ELTTEXTE000224"></a>


### Insérer une image
<a name="inserer_une_image_ELTPARAGRAPHE000020"></a>Pour insérer une image dans une page d'aide :

1. Dans la page d'aide, positionnez le curseur à la position où l'image doit être insérée.

2. Sous le volet "Modification", dans le groupe "Edition", cliquez sur "Image". 

3. Dans l'explorateur de fichiers, sélectionnez l'image à insérer dans la page d'aide et validez. 

4. Si nécessaire, validez la copie de l'image dans le répertoire du projet. 

5. La fenêtre de description de l'image apparaît : <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Aide_image.gif&type=thumb)


6. Définissez si nécessaire des zones cliquables dans cette image :

	- Cliquez sur le bouton "Nouvelle".

	- Définissez le lien associé à la zone cliquable. Pour plus de détails, consultez [Propriétés d'un lien](../Editeurs/2010022.md).

	- Définissez la zone cliquable soit à l'aide de la souris, soit grâce aux champs "Paramètres de la zone de cliquage".


 Pour plus de détails sur cette fenêtre, consultez [Propriétés d'une image](../Editeurs/2010027.md).

7. Validez. L'image est intégrée dans la page d'aide. Elle peut être redimensionnée grâce aux poignées de sélection.




Remarque : Il est également possible d'utiliser l'option "Insérer une image" du menu contextuel de la page d'aide. 


### Insérer un clipart
<a name="inserer_clipart_ELTPARAGRAPHE000063"></a>Pour insérer un clipart dans une page d'aide :

1. Dans la page d'aide, positionnez le curseur à la position où le clipart doit être inséré.

2. Sous le volet "Modification", dans le groupe "Edition", cliquez sur "Catalogue".

3. Le catalogue d'images est affiché et vous permet de sélectionner l'image à afficher. 

4. Validez l'image sélectionnée dans le catalogue. 

5. La fenêtre de description de l'image apparaît : <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Aide_image.gif&type=thumb)


6. Définissez si nécessaire des zones cliquables dans cette image :

	- Cliquez sur le bouton "Nouvelle".

	- Définissez le lien associé à la zone cliquable. Pour plus de détails, consultez [Propriétés d'un lien](../Editeurs/2010022.md).

	- Définissez la zone cliquable soit à l'aide de la souris, soit grâce aux champs "Paramètres de la zone de cliquage".


 Pour plus de détails sur cette fenêtre, consultez [Propriétés d'une image](../Editeurs/2010027.md).

7. Validez. L'image est intégrée dans la page d'aide. Elle peut être redimensionnée grâce aux poignées de sélection.



<a name="NOTE2_2"></a>


### Afficher les propriétés d'une image
<a name="afficher_les_proprietes_une_image_ELTPARAGRAPHE000106"></a>Pour afficher les propriétés d'une image :

1. Sélectionnez l'image.

2. Sélectionnez l'option "Propriétés de l'image" du menu contextuel. 




Pour plus de détails, consultez [Propriétés d'une image](../Editeurs/2010027.md).

## Gestion des vidéos
<a name="gestion_des_videos_ELTTEXTE000260"></a>


### Insérer une vidéo
<a name="inserer_une_video_ELTPARAGRAPHE000125"></a>Pour insérer une vidéo dans une page d'aide :

1. Dans la page d'aide, positionnez le curseur à la position où la vidéo doit être insérée.

2. Sous le volet "Modification", dans le groupe "Edition", cliquez sur "Vidéo". 

3. Dans la fenêtre qui s'affiche, sélectionnez le fichier correspondant à la vidéo à insérer dans la page d'aide et validez. 
	Remarque : Le chemin de la vidéo peut correspondre à une vidéo accessible depuis le poste en cours (fichier mp4 ou WebM) ou à l'adresse d'une vidéo (YouTube ou autre). 

4. Si nécessaire, validez la copie de la vidéo dans le répertoire du projet. 

5. La vidéo est intégrée dans la page d'aide. Elle peut être redimensionnée grâce aux poignées de sélection.




Remarques : 

- Les vidéos sont disponibles uniquement pour les aides générées au format HTM et PDF. 

- Il est également possible d'utiliser l'option "Insérer une vidéo" du menu contextuel de la page d'aide. 

- Il est possible de modifier la vidéo utilisée en affichant les propriétés de la vidéo (option "Propriétés de la vidéo" du menu contextuel).

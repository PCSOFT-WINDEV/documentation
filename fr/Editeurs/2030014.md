


## Faire une sauvegarde d'un projet 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000116"></a>
Il est possible de faire régulièrement une sauvegarde des différents éléments d'un projet.

Lors de cette sauvegarde, il est possible de créer :

- un fichier archive (d'extension ZIP). Ce fichier contient tous les éléments du projet sauvegardés.

- une simple copie du projet.




A tout moment, il sera possible de restaurer le projet sauvegardé.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000140"></a>


### Pour faire une sauvegarde du projet en cours :
<a name="pour_faire_une_sauvegarde_projet_cours_ELTPARAGRAPHE000025"></a>

1. Sous le volet "Projet", dans le groupe "Autres actions", déroulez "Sauver/Restaurer" et sélectionnez "Sauvegarder le projet". L'assistant d'archivage s'affiche.

2. Sélectionnez les catégories de fichiers du projet à sauvegarder.

3. Sélectionnez le type de sauvegarde à réaliser :

	- ***Fichier de sauvegarde compressé (zip)***. Dans ce cas, précisez le nom du répertoire et du fichier de sauvegarde. Ce nom peut être personnalisé, ou contenir la date du jour de la sauvegarde. Attention : Si la sauvegarde est réalisée dans le répertoire des versions et si une sauvegarde de même nom a déjà été enregistrée, l'archive correspondante sera écrasée.

	- ***Copie de sauvegarde***. Dans ce cas, indiquez le nom du répertoire de sauvegarde. Ce nom peut être personnalisé ou contenir la date du jour de la sauvegarde.




4. Validez l'assistant. Le fichier archive (ou la copie du projet) est créé dans le répertoire spécifié.

## Notes
<a name="notes_ELTTEXTE000164"></a>


### Restauration
<a name="restauration_ELTPARAGRAPHE000052"></a>Pour restaurer une sauvegarde ou un fichier d'une sauvegarde, sous le volet "Projet", dans le groupe "Autres actions", déroulez "Sauver/Restaurer" et sélectionnez "Restaurer un projet". Pour plus de détails, consultez [Restaurer un projet](../Editeurs/2030038.md).


## Générateur de gabarits WINDEV : Personnaliser les éléments
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000479"></a>
Le générateur de gabarits WINDEV permet de générer les fichiers nécessaires à la création d'un gabarit personnalisé pour les projets WINDEV. 

Il est possible de personnaliser les différents éléments générés : 

- [Images](#NOTE2_1) : ces images seront utilisées dans le style des champs. 

- [Modèles de fenêtres](#NOTE3_1). 

- [Modèles d'états](#NOTE4_1). 




Cette page d'aide présente les principales images utilisées ainsi que les modifications pouvant être effectuées. 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Images du gabarit
<a name="images_gabarit_ELTTEXTE000503"></a>


### Conseils
<a name="conseils_ELTPARAGRAPHE000030"></a>

- L'extension des fichiers image peut être modifiée. Il est possible d'utiliser les formats suivants : JPG, PNG et GIF.

- Si vous souhaitez qu'une image ne soit pas utilisée dans un champ, il vous suffit de la rendre transparente et de dimension 1x1 pixel. Dans ce cas, l'image doit être au format PNG. Ainsi, lors de la génération du gabarit, l'image ne sera pas générée.

- Pour être certain d'un bon affichage des images dans vos gabarits, ne pas modifier les dimensions des fichiers.

## Modèles de fenêtres
<a name="modeles_fenetres_ELTTEXTE000533"></a>
Les modèles de fenêtres sont placés dans le répertoire "Modele" du projet source du gabarit. Les différents modèles de fenêtres disponibles sont : 

- Modele.wdt : Modèle général de vos fenêtres. Ce modèle permet de définir le fond de vos fenêtres (couleur de fond, image de fond, ...). 

- Modele_Menu.wdt : Modèle utilisé pour les fenêtres de type menu. 

- Modele_MessBox.wdt : Modèle utilisé pour les fenêtres de type boîte de message. 

- Modele_Popup.wdt : Modèle utilisé pour les fenêtres de type Popup. 

- Modele_Preview.wdt : Modèle utilisé pour l'aperçu avant impression. 




Pour personnaliser les modèles de fenêtres, il suffit de les ouvrir directement sous WINDEV. Il est conseillé d'ouvrir ces éléments dans WINDEV **hors projet** (par exemple, réalisez un Drag and Drop de l'élément vers WINDEV). 

<a name="NOTE4"></a>
<a name="NOTE4_1"></a>


## Modèles d'états
<a name="modeles_etats_ELTTEXTE000557"></a>
Les modèles d'états sont placés dans le répertoire "Modele" du projet source du gabarit. Les différents modèles d'états disponibles sont : 

- Modele_Etats.wte : Modèle général des états. 

- Modele_Etiquette.wte : Modèle utilisé pour les états de type Etiquette. 

- Modele_Mailing.wte : Modèle utilisé pour les états de type Mailing. 




Pour personnaliser les modèles d'états, il suffit de les ouvrir directement sous WINDEV. Il est conseillé d'ouvrir ces éléments dans WINDEV **hors projet** (par exemple, réalisez un Drag and Drop de l'élément vers WINDEV).

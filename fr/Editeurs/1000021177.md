


## Groupware utilisateur et fichiers de données Client/Serveur
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000138"></a>
Une application utilisant le groupware utilisateur peut tout à fait être utilisée avec une base de données HFSQL en mode Client/Serveur.

Pour qu'une application utilisant le groupware utilisateur fonctionne en mode Client/Serveur, il est nécessaire de :

- passer l'application existante en mode Client/Serveur. Pour plus de détails, consultez [Client/Serveur](../WDLang4/3044306.md).

- configurer correctement le groupware utilisateur afin d'utiliser les fichiers de données présents sur le serveur. Ces manipulations sont présentées dans cette page d'aide.




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000162"></a>


### Mode automatique
<a name="mode_automatique_ELTPARAGRAPHE000026"></a>Pour passer les fichiers de données du groupware utilisateur en mode Client/Serveur : 

1. Sous le volet "Projet", dans le groupe "Projet", cliquez sur "Groupware Utilisateur". 

2. Dans la fenêtre de description de la configuration de projet en cours, dans la zone "Groupware Utilisateur : Inactif", cliquez sur le bouton "Modifier". 

3. Dans la fenêtre qui s'affiche, activez le groupware utilisateur.

4. Sélectionnez le "Groupware utilisateur classique". 

5. Dans l'onglet "Fichiers", sélectionnez l'option "HFSQL Client/Serveur".

6. Cliquez sur le lien "Cliquez ici pour saisir les paramètres du serveur". Indiquez :

	- le nom du serveur utilisé.

	- le numéro de port utilisé.

	- le nom de l'utilisateur utilisé pour la base de données.

	- le mot de passe associé.

	- le nom de la base.

	- les répertoires des fichiers de données.




7. Si les fichiers de données spécifiques au groupware utilisateur existent, copiez ces fichiers de données sur le serveur.



<a name="NOTE2_2"></a>


### Mode manuel
<a name="mode_manuel_ELTPARAGRAPHE000060"></a>Pour passer les fichiers de données du groupware utilisateur en mode Client/Serveur : 

1. Sous le volet "Projet", dans le groupe "Projet", cliquez sur "Groupware Utilisateur".

2. Dans la fenêtre de description de la configuration de projet en cours, dans la zone "Groupware Utilisateur", cliquez sur le bouton "Modifier". 

3. Dans l'onglet "Général", sélectionnez "Lancement manuel par programmation". 
	Le groupware utilisateur ne doit pas être lancé en mode "Lancement automatique" dans votre application.

4. Sélectionnez les différentes options du groupware utilisateur et validez la fenêtre. 

5. Dans le code d'initialisation du projet, utilisez la fonction [gpwOuvreConnexion](../WDLang6/3041007.md) pour décrire la connexion à la base de données HFSQL Client/Serveur.

6. Si les fichiers de données spécifiques au groupware utilisateur existent, copiez ces fichiers de données sur le serveur.

7. Utilisez la fonction [gpwOuvre](../WDLang6/3041002.md) pour indiquer le positionnement des fichiers de données. Le répertoire spécifié doit être relatif au répertoire du serveur.

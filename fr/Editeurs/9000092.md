


## Associer des documents au projet
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000216"></a>
Pour permettre la description d'un projet avec précision, il est possible d'associer des documents au projet.

Ces documents peuvent correspondre à des fichiers texte, des images, des documents scannés, ...

La liste des documents associés au projet apparaît dans le [dossier du projet](../Editeurs/2015001.md).

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Associer un document au projet
<a name="associer_document_projet_ELTTEXTE000240"></a>
Il est possible d'associer un document à un projet :

- soit dans l'[assistant de création du projet](#NOTE2_2).

- soit depuis le [tableau de bord](#NOTE2_3).

- soit depuis le volet [Explorateur de projet](../Editeurs/2027029.md).



<a name="NOTE2_2"></a>


### Associer un document dans l'assistant de création du projet
<a name="associer_document_dans_assistant_creation_projet_ELTPARAGRAPHE000043"></a>Pour associer un document dans l'assistant de création du projet :

1. Allez dans le plan intitulé "Description - Documents joints".

2. Pour ajouter un document existant, cliquez sur le bouton "Ajouter un document existant" et sélectionnez le fichier à ajouter. Ce fichier peut être copié ou non dans le sous-répertoire "Documents" du projet en cours.

3. Pour ajouter un document scanné :

	- positionnez le document à scanner dans un scanneur accessible depuis le poste en cours.

	- cliquez sur le bouton "Scanner un document...".

	- si plusieurs scanneurs sont accessibles depuis le poste en cours, sélectionnez le scanneur à utiliser.

	- indiquez le nom du document scanné. Par défaut, ce document est enregistré dans le sous-répertoire "Documents" du répertoire en cours.




4. Les documents associés au projet sont listés.

5. Pour supprimer un document de cette liste, sélectionnez le document à supprimer et cliquez sur le bouton "Enlever de la liste".




**Remarque** : Dans ce plan de l'assistant, il est possible de créer le squelette du dossier des spécifications du projet (bouton "Générer un squelette pour le dossier de spécifications"). Un fichier "&lt;Nom du projet&gt;.RTF" est automatiquement créé dans le sous-répertoire "Documents" du projet en cours. Ce fichier contient le squelette des différentes spécifications du projet (objectif, vue d'ensemble, résumé des tâches, ...) et peut être modifié dans le plan suivant.
<a name="NOTE2_3"></a>


### Associer un document depuis le tableau de bord
<a name="associer_document_depuis_tableau_bord_ELTPARAGRAPHE000075"></a>Pour associer un document depuis le tableau de bord :

1. Affichez le tableau de bord : sous le volet "Projet", dans le groupe "Projet", cliquez sur ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_tableaubord.gif)
.

2. Affichez si nécessaire le Widget "Documents" : 

	- Affichez le menu contextuel du tableau de bord (clic droit sur une zone vide). 

	- Sélectionnez l'option "Mode édition". 

	- Affichez à nouveau le menu contextuel du tableau de bord. 

	- Sélectionnez l'option "Ajouter .. Documents". 

	- Affichez à nouveau le menu contextuel du tableau de bord et sélectionnez l'option "Mode édition".




3. Cliquez sur le bouton "+" du Widget "Documents". 

	- Pour ajouter un document existant, sélectionnez l'option "Sélectionner un document existant" puis sélectionnez le fichier à ajouter. Ce fichier peut être copié ou non dans le sous-répertoire "Documents" du projet en cours.

	- Pour ajouter un document scanné :

		- Positionnez le document à scanner dans un scanneur accessible depuis le poste en cours.

		- Sélectionnez l'option "Scanner un document".

		- Si plusieurs scanneurs sont accessibles depuis le poste en cours, sélectionnez le scanneur à utiliser.

		- Indiquez le nom du document scanné. Par défaut, ce document est enregistré dans le sous-répertoire "Documents" du répertoire en cours.







Pour supprimer un document depuis le tableau de bord : 

1. Sélectionnez le document à supprimer dans le widget "Documents". 

2. Affichez le menu contextuel du document et sélectionnez l'option "Enlever du projet" ou "Supprimer du disque".

## Visualiser les documents associés au projet
<a name="visualiser_les_documents_associes_projet_ELTTEXTE000276"></a>
Il est possible de visualiser les documents associés au projet :

- soit depuis le [tableau de bord](../Editeurs/9500221.md).

- soit depuis le volet [Explorateur de projet](../Editeurs/2027029.md).



<a name="NOTE3_2"></a>


### Visualiser un document depuis le tableau de bord
<a name="visualiser_document_depuis_tableau_bord_ELTPARAGRAPHE000131"></a>Pour visualiser un document depuis le tableau de bord :

1. Affichez le tableau de bord : sous le volet "Projet", dans le groupe "Projet", cliquez sur ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_tableaubord.gif)
.

2. Dans le widget "Documents", affichez le menu contextuel du document à visualiser et sélectionnez l'option "Ouvrir" (ou cliquez directement sur le fichier à visualiser).



<a name="NOTE3_3"></a>


### Visualiser un document depuis le volet "Explorateur de projet"
<a name="visualiser_document_depuis_volet_explorateur_projet_ELTPARAGRAPHE000151"></a>Pour visualiser un document depuis le volet "Explorateur de projet" :

1. Affichez le volet "Explorateur de projet" : sous le volet "Accueil", dans le groupe "Environnement", déroulez "Volets" et sélectionnez "Volets" puis "Explorateur de projet".

2. Cliquez sur l'élément "Documents". La liste des documents associés au projet s'affiche.

3. Sélectionnez le document à visualiser et sélectionnez l'option "Ouvrir" du menu contextuel (ou double-cliquez directement sur le fichier à visualiser).


## Statistiques de code 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000108"></a>
Les statistiques de code concernent :

- le nombre de lignes total du traitement concerné,

- le taux de commentaires,

- le nombre de lignes par traitement,

- le nombre de lignes de code avec ou sans commentaires,

- le nombre de lignes de commentaires.






<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000132"></a>


### Afficher les statistiques
<a name="afficher_les_statistiques_ELTPARAGRAPHE000025"></a>Ces statistiques peuvent être affichées :

- **pour l'objet en cours d'édition sous l'éditeur de code (fenêtre, état, ...).**
	Les statistiques concernent les différents traitements de l'élément en cours et de ses champs. 
	Utilisez l'option "Code .. Statistiques .. Objet". La fenêtre de configuration s'affiche. Cliquez sur le bouton "Afficher" pour calculer les statistiques.

- ** pour le projet en cours d'édition sous l'éditeur de code**. 
	Les statistiques affichées concernent l'ensemble du projet : chaque fenêtre, état, procédure globale ... est analysé. 
	Utilisez l'option "Code .. Statistiques .. Projet". La fenêtre de configuration s'affiche. Cliquez sur le bouton "Afficher" pour calculer les statistiques.




Dans la fenêtre de configuration des statistiques, indiquez :

- le pourcentage de commentaires maximum à prendre en compte.

- le nombre de lignes de code minimal à prendre en compte.




Remarque : Pour accéder à ces options, utilisez le menu principal accessible par le logo de la version du produit : <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Ruban_logo_menu17.gif)



<a name="NOTE2_2"></a>


### Impression des statistiques
<a name="impression_des_statistiques_ELTPARAGRAPHE000054"></a>L'impression des statistiques de code est réalisée à partir de la fenêtre de configuration des statistiques. Les éléments suivants sont imprimés :

- le nombre de lignes total du traitement concerné,

- le taux de commentaires,

- le nombre de lignes par traitement.

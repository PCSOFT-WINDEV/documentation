## Comment le faire ?
<a name="comment_faire_ELTTEXTE000243"></a>


### Créer un diagramme de collaboration
<a name="creer_diagramme_collaboration_ELTPARAGRAPHE000039"></a>Pour créer un diagramme de collaboration : 

1. Cliquez sur ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_nouveau.gif)
 parmi les boutons d'accès rapide. 

	- La fenêtre de création d'un nouvel élément s'affiche : cliquez sur "Architecture" puis sur "UML".

	- L'assistant de création d'un modèle UML se lance.




2. Sélectionnez l'option "Diagramme de collaboration".

3. Spécifiez le modèle UML dans lequel le diagramme sera inséré.

4. Choisissez d'éditer directement le diagramme.

5. Validez la création du diagramme de collaboration. Le modèle créé devient le modèle en cours.



<a name="NOTE2_2"></a>


### Créer un objet
<a name="creer_objet_ELTPARAGRAPHE000061"></a>Pour créer un objet :

1. Sous le volet "Diagramme de collaboration", dans le groupe "Création", cliquez sur "Objet". Un nouvel objet apparaît dans le modèle UML.

2. Sélectionnez l'option "Description" du menu contextuel (clic droit) du nouvel objet. La fenêtre de spécifications de l'objet s'ouvre.

3. Spécifiez :

	- le nom de l'objet. Ce nom est affiché sous l'éditeur d'UML.

	- le nom de la classe. Ce nom est affiché sous l'éditeur d'UML.

	- le libellé de la classe décrivant l'intérêt de la classe. Ce libellé sera utilisé dans le dossier d'impression de l'éditeur d'UML.




4. Définissez les attributs de l'objet. 

5. Validez la description de l'objet.



<a name="NOTE2_3"></a>


### Créer un acteur
<a name="creer_acteur_ELTPARAGRAPHE000090"></a>Pour créer un acteur :

1. Sous le volet "Diagramme de collaboration", dans le groupe "Création", cliquez sur "Acteur". Un nouvel objet apparaît dans le modèle UML.

2. Sélectionnez l'option "Description" du menu contextuel (clic droit) du nouvel acteur. La fenêtre de spécifications de l'acteur s'ouvre.

3. Spécifiez :

	- le nom de l'acteur. Ce nom est affiché dans l'acteur sous l'éditeur d'UML

	- le nom de la classe. Ce nom est affiché sous l'éditeur d'UML.

	- le libellé de l'acteur décrivant l'intérêt de l'acteur.




4. Validez la description de l'acteur.



<a name="NOTE2_4"></a>


### Créer un message
<a name="creer_message_ELTPARAGRAPHE000118"></a>Pour créer un message :

1. Sous le volet "Diagramme de collaboration", dans le groupe "Création", cliquez sur "Message".

2. Sélectionnez les différents éléments à relier. L'assistant de création d'un message se lance.

3. Spécifiez :

	- le nom du message.

	- le numéro de séquence du message.

	- les messages qui précèdent le message en cours de création.

	- les conditions d'émission du message.

	- les paramètres du message.

	- le nom de la valeur de retour.




4. Validez la description du message.



<a name="NOTE2_5"></a>


### Créer un paquetage
<a name="creer_paquetage_ELTPARAGRAPHE000149"></a>Pour créer un paquetage :

1. Sous le volet "Diagramme de collaboration", dans le groupe "Création", cliquez sur "Paquetage". Un nouveau paquetage apparaît dans le modèle UML.

2. Sélectionnez l'option "Description" du menu contextuel (clic droit) du nouveau paquetage. La fenêtre de spécifications du paquetage s'ouvre.

3. Spécifiez :

	- le nom du paquetage. Ce nom est affiché dans le paquetage sous l'éditeur d'UML.

	- le libellé du paquetage décrivant l'intérêt du paquetage.

	- les relations du paquetage.




4. Validez la description du paquetage.

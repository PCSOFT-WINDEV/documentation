


## Correcteur d'interface
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000093"></a>
Le correcteur d'interface avancé est un outil permettant d'harmoniser la disposition des champs dans les différentes fenêtres d'une application.

Le correcteur d'interface avancé propose pour la fenêtre en cours ou uniquement pour les champs sélectionnés, d'appliquer des règles de présentation d'interface issues du standard Windows : alignement des champs, normalisation de la taille des boutons, etc.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000117"></a>
Pour utiliser le correcteur d'interface avancé : 

1. Sélectionnez si nécessaire les champs dont la position, la taille ou l'alignement sont à revoir.

2. Lancez le correcteur d'interface avancé : sous le volet "Alignement", dans le groupe "Autres alignements", cliquez sur "Correcteur d'interface",

3. Lors de son lancement, le correcteur d'interface analyse la fenêtre (ou la sélection de champs) et établit des "Groupes de champs" en fonction de leur nature, de leur position, ...
	Pour chaque "groupe de champs", le correcteur propose d'appliquer les règles d'harmonisation non respectées. Les champs ou les groupes de champs concernés sont encadrés d'un trait coloré.




Plusieurs choix sont alors possibles:

- **Appliquer la règle** : Les modifications proposées sont effectuées dans la fenêtre. Le correcteur passe à la règle suivante.

- ** Ignorer la règle** : Aucune modification n'est réalisée. La règle suivante est proposée.

- **Rétablir** : Annule la modification effectuée par l'application de la règle précédente.

- **Fermer** : La règle proposée n'est pas appliquée. La fenêtre reste en l'état et le correcteur d'interface est fermé.

- **Annuler** : Toutes les règles appliquées depuis le début de la correction sont annulées. Le correcteur d'interface est fermé. La fenêtre est affichée dans son état initial (avant le début de la correction).

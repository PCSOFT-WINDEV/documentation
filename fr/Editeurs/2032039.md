


## Fenêtre de description d'une requête de sélection : Actions appliquées 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Actions appliquées sur les rubriques de la requête
<a name="actions_appliquees_sur_les_rubriques_requete_ELTTEXTE000135"></a>
Cette partie de la fenêtre de description d'une requête permet de définir les différentes actions à appliquer sur les rubriques de la requête.

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=DescrReqPartie3.gif)



|   |   |
| --- | --- |
| ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CERCLE1.GIF)<br> | Rubrique sur laquelle les actions seront appliquées. |
| ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CERCLE2.GIF)<br> | Permet :<br><br>- soit d'afficher ou non les valeurs de la rubrique sélectionnée dans le résultat de la requête.<br><br>- soit de renommer la rubrique sélectionnée dans le résultat de la requête (création d'un alias).<br><br><br> |
| ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CERCLE3.GIF)<br> | Trie les valeurs de la rubrique sélectionnée dans le résultat de la requête. |
| ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CERCLE4.GIF)<br> | Ajoute une nouvelle rubrique au résultat de la requête. Cette rubrique correspond :<br><br>- soit au nombre d'enregistrements présents dans un groupement d'enregistrements.<br><br>- soit au nombre d'enregistrements présents dans un fichier de données.<br><br><br> |
| ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CERCLE5.GIF)<br> | Ajoute une nouvelle rubrique au résultat de la requête. Cette rubrique correspond à la somme d'un ensemble d'enregistrements. |
| ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CERCLE6.GIF)<br> | Ajoute une nouvelle rubrique au résultat de la requête. Cette rubrique correspond :<br><br>- soit à la moyenne d'un ensemble d'enregistrements.<br><br>- soit à la valeur minimale d'un ensemble d'enregistrements.<br><br>- soit à la valeur maximale d'un ensemble d'enregistrements.<br><br><br> |
| ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CERCLE7.GIF)<br> | Modifie les valeurs affichées dans le résultat de la requête. Il est possible d'afficher uniquement :<br><br>- l'année à partir d'une rubrique de type date.<br><br>- le mois à partir d'une rubrique de type date.<br><br><br><br><br>- le département à partir d'une rubrique contenant un code postal.<br><br><br> |
| ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CERCLE8.GIF)<br> | Définit les conditions de sélection. Il est possible de :<br><br>- appliquer une condition de sélection sur la rubrique sélectionnée.<br><br>- modifier et/ou supprimer une condition existante.<br><br>- modifier l'enchaînement des conditions existantes.<br><br><br> |
| ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CERCLE9.gif)<br> | Sélectionne les enregistrements correspondant à une période donnée. |
| ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CERCLE10.GIF)<br> | Sélectionne les n premiers ou les n derniers enregistrements affichés dans le résultat de la requête. |
| ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CERCLE11.gif)<br> | Supprime ou non les enregistrements "doublons" du résultat de la requête. |
| ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CERCLE12.gif)<br> | Permet de :<br><br>- indiquer le type de la base de données utilisée (HFSQL, Oracle ou autre).<br><br>- gérer les jointures (ajouter, modifier, supprimer, ré-initialiser).<br><br>- vérifier ou non la syntaxe SQL utilisée (en mode test uniquement).<br><br>- préciser les options d'exécution (optimisation de l'espace mémoire, vérification du format des fichiers, ...).<br><br>- indiquer si la requête est visible sous le logiciel "Etats & Requêtes".<br><br><br> |

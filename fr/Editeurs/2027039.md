
## Le volet "Hashtags du code"
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000122"></a>
Le volet "Hashtags du code" affiche la liste des hashtags présents dans le code WLangage. 

Un hashtag permet de marquer une ligne de code avec une information permettant d'indiquer par exemple : 

- une action à faire, 

- une information sur la finalité du code,

- une correction de bug, 

- etc.




**Exemples** : Exemples de Hashtags pouvant exister dans le code : 

```wl
// #Exemple Exemple de Hashtag
// #TODO Finir de tester le code
```


**Règles à respecter** 
Un hashtag de code doit respecter les règles suivantes : 

- Un hashtag doit se trouver dans une ligne commentée. Par exemple : 
	```wl
	// #MonTag
	ou
	/* #MonTag
	```


- Le hashtag ne doit pas être précédé d'un caractère (excepté le marqueur de commentaire, un espace ou une tabulation). 

- Le hashtag doit commencer par une lettre et ne peut contenir que des caractères alphanumériques ou le caractère '_'. Il n'est pas possible de mettre un espace dans le nom du hashtag.

- Les hashtags ne sont pas sensibles à la casse. 




**Remarques** : 

- Il est possible de créer des hashtags de code dans des fenêtres internes, pages internes, modèles de champs, modèles de pages ou de fenêtres, composants internes, ...

- Il est possible de créer des hashtags de code dans un code WLangage navigateur. 





### Contenu du volet "Hashtags du code"
<a name="contenu_volet_hashtags_code_ELTPARAGRAPHE000037"></a>Le volet "Hashtags du code" liste tous les hashtags présents dans les différents événements WLangage du projet. Par défaut, le nom du hashtag est affiché puis le nom de l'élément contenant le hashtag.

Un double-clic sur le hashtag permet d'afficher le code WLangage associé à celui-ci. 

Les différents boutons du volet "Hashtags de code" permettent de : 

- parcourir les hashtags (suivant, précédent)

- parcourir les éléments du groupe (cas où plusieurs hashtags sont associés au même élément). 

- changer le mode d'affichage des hashtags : il est possible d'afficher au choix : 

	- Les hashtags regroupés par nom (par exemple, tous les hashtags "#A_Faire" présents dans tous les éléments du projet). 

	- Les hashtags regroupés par éléments (par exemple, tous les hashtags de la fenêtre FEN_Menu). 




- faire une recherche parmi les différents hashtags.

## Afficher le volet "Hashtags du code"
<a name="afficher_volet_hashtags_code_ELTTEXTE000152"></a>
**Pour afficher le volet "Hashtags du code"**, sous le volet "Accueil", dans le groupe "Environnement", déroulez "Volets" et sélectionnez "Volets" puis "Hashtags du code".

Pour plus de détails sur les manipulations des volets, consultez [Manipuler les volets ancrables](../Editeurs/2027001.md).

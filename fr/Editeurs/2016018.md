


## Adapter un projet pour utiliser les ambiances
			

<DIV class="specObsolete">
	<IMG style="float:left;margin-right:10px;" src="https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=warning_big.png"><B>Avertissement</B><BR>
	 Cette page d'aide présente les manipulations à effectuer pour adapter un projet à l'utilisation des ambiances. Cette page d'aide concerne les projets créés avec une version antérieure à WEBDEV 15.
</DIV>

## Présentation
<a name="presentation_ELTTEXTE000233"></a>
Les ambiances permettent d'harmoniser l'aspect visuel d'un site et rendent les changements de look faciles et rapides.

Il est possible d'adapter un projet existant (développé avec une version antérieure de WEBDEV par exemple) pour utiliser les éléments d'ambiance. Il sera ainsi possible par la suite de changer facilement l'ambiance complète du projet ou simplement la palette de couleurs.

Quelques adaptations sont nécessaires dans les cas suivants :  

- votre projet n'utilise peut être pas de modèles de pages.

- les styles de la feuille de styles n'utilisent peut être pas les noms normalisés nécessaires au changement d'ambiances.




Plusieurs cas peuvent se présenter :

- [Le projet ne possède pas de modèles de pages](#Adapt1).

- [Le projet possède un modèle de pages mais vous ne souhaitez pas le conserver](#Adapt2).

- [Le projet possède des modèles de pages que vous souhaitez conserver](#Adapt3).




<a name="Adapt1"></a>


## Le projet ne possède pas de modèles de pages
<a name="projet_possede_pas_modeles_pages_ELTTEXTE000257"></a>
La méthode à suivre pour l'adaptation d'un projet sans modèle de pages est la suivante :

1. Changez le style général du projet et choisissez une des ambiances.

	1. Sous le volet "Projet", dans le groupe "Projet", cliquez sur "Description".

	2. Sélectionnez l'onglet "Ambiance".

	3. Choisissez une ambiance et une palette de couleurs.




2. Créez un nouveau modèle de pages avec un modèle prédéfini ressemblant le plus possible au site existant.

	1. Cliquez sur ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_nouveau.gif) parmi les boutons d'accès rapide.

	2. La fenêtre de création d'un nouvel élément s'affiche : cliquez sur "Page" puis sur "Modèle de Pages".

	3. Dans la fenêtre de création d'un nouveau Modèle de Pages, sélectionnez le modèle de votre choix.




3. Si des champs sont présents dans chaque page de votre projet, vous pouvez les regroupez dans le modèle de pages.

4. Associez chaque page du projet au modèle de pages créé. Pour cela :

	1. Ouvrez la page dans l'éditeur. 

	2. Sous le volet "Page", dans le groupe "Description", déroulez "Modèles" et cliquez sur "Liste des modèles utilisés".

	3. Sélectionnez l'ancien modèle et cliquez sur le bouton "Supprimer".

	4. Confirmez la suppression.

	5. Cliquez sur le bouton "Ajouter".

	6. Choisissez le modèle de pages basé sur l'ambiance créée à l'étape 2.

	7. Répondez "Oui" à la question "Voulez-vous mettre à jour les propriétés de la page depuis ce modèle ?". De cette façon, toute modification de l'ambiance ou de la palette de couleur du projet se répercutera automatiquement sur le modèle puis sur la page.




5. Changez le style des champs des pages existantes pour un style de l'ambiance du projet.

6. Réorganisez les champs de chaque page si nécessaire.

## Le projet possède un modèle de pages mais vous ne souhaitez pas le conserver
<a name="projet_possede_modele_pages_mais_vous_souhaitez_pas_conserver_ELTTEXTE000281"></a>
La méthode à suivre pour l'adaptation d'un projet sans conserver les modèles de pages existants est la suivante :

1. Changez le style général du projet pour une des ambiances.

	1. Sous le volet "Projet", dans le groupe "Projet", cliquez sur "Description".

	2. Sélectionnez l'onglet "Ambiance".

	3. Choisissez une ambiance et une palette de couleurs.




2. Créez un nouveau modèle de pages avec un modèle prédéfini ressemblant le plus possible au site existant.

	1. Cliquez sur ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_nouveau.gif) parmi les boutons d'accès rapide.

	2. La fenêtre de création d'un nouvel élément s'affiche : cliquez sur "Page" puis sur "Modèle de Pages".

	3. Dans l'assistant de création d'un nouveau modèle de pages, sélectionnez une base "Vierge". Validez. 

	4. Copiez/collez les champs de l'ancien modèle.




3. Changez le style des champs copiés pour un style de l'ambiance du projet.

	1. Pour chaque champ du modèle, choisissez l'option "Choisir un style WEBDEV" dans le menu contextuel.

	2. Dans la liste des styles, n'affichez que les styles de l'ambiance en décochant la case "Styles du projet".

	3. Choisissez un style pour le champ.

	4. Validez la fenêtre.




4. Réorganisez les champs du modèle si nécessaire.

5. Changez le modèle de pages des pages existantes du projet. Pour chaque page :

	1. Ouvrez la page dans l'éditeur. 

	2. Sous le volet "Page", dans le groupe "Description", déroulez "Modèles" et cliquez sur "Liste des modèles utilisés".

	3. Sélectionnez l'ancien modèle et cliquez sur le bouton "Supprimer".

	4. Confirmez la suppression.

	5. Cliquez sur le bouton "Ajouter".

	6. Choisissez le modèle de pages basé sur l'ambiance créée à l'étape 2.

	7. Répondez "Oui" à la question "Voulez-vous mettre à jour les propriétés de la page depuis ce modèle ?". De cette façon, toute modification de l'ambiance ou de la palette de couleurs du projet se répercutera automatiquement sur le modèle puis sur la page.




6. Changez le style des champs des pages existantes pour un style de l'ambiance du projet.

7. Réorganisez les champs de chaque page si nécessaire.

## Le projet possède des modèles de pages que vous souhaitez conserver
<a name="projet_possede_des_modeles_pages_que_vous_souhaitez_conserver_ELTTEXTE000305"></a>
La méthode à suivre pour l'adaptation d'un projet en conservant les modèles de pages existants est la suivante :

1. Changez le style général du projet pour une des ambiances.

	1. Sous le volet "Projet", dans le groupe "Projet", cliquez sur "Description".

	2. Sélectionnez l'onglet "Ambiance".

	3. Choisissez une ambiance et une palette de couleurs.




2. Utilisez l'ambiance du projet dans chaque modèle de pages. Pour chaque modèle de pages :

	1. Ouvrez le modèle de pages dans l'éditeur.

	2. Dans le menu contextuel du modèle de pages, choisissez "Description".

	3. Dans la fenêtre de description, sélectionnez l'onglet "Style".

	4. Sélectionnez l'ambiance et la palette de couleurs du projet (si nécessaire) en utilisant le bouton de parcours et en choisissant "Aucune" dans les fenêtres d'ambiance personnalisées et de palette de couleurs personnalisées.




3. Changez le style des champs dans chaque modèle de pages du projet pour un style de l'ambiance du projet.

	1. Pour chaque champ de chaque modèle, dans le menu contextuel choisissez "Choisir un style WEBDEV".

	2. Dans la liste des styles, n'affichez que les styles de l'ambiance en décochant la case "Styles du projet".

	3. Choisissez un style pour le champ.

	4. Validez la fenêtre.




4. Réorganisez les champs du modèle si nécessaire.

5. Changez le style des champs des pages existantes pour un style de l'ambiance du projet.

6. Réorganisez les champs de chaque page si nécessaire.

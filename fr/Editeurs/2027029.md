


## Volet "Explorateur de projet"
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000340"></a>
Le volet "Explorateur de projet" permet de simplifier l'environnement de développement de WINDEV, WEBDEV et WINDEV Mobile.

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=VoletExplorateurProjet.gif)
Ce volet permet de :

- lister les éléments du projet.

- lister les éléments de l'analyse.

- lister les procédures, les classes, ...

- lister les descriptions externes.

- effectuer une recherche dans le projet en cours.

<a name="NOTE2_1"></a>


## Fonctionnalités du volet "Explorateur de projet"
<a name="fonctionnalites_volet_explorateur_projet_ELTTEXTE000364"></a>


### Manipulations générales dans le volet "Explorateur de projet"
<a name="manipulations_generales_dans_volet_explorateur_projet_ELTPARAGRAPHE000026"></a>Le volet "Explorateur de projet" liste des éléments du projet et si nécessaire les "sous-éléments" associés. 

Il est possible de réaliser des "Drag and Drop" des éléments sélectionnés vers l'environnement. Par exemple, le "Drag and Drop" d'une variable globale dans une fenêtre créera un champ de saisie.

Il est également possible de réaliser des "Drag and Drop" d'éléments dans le volet "Explorateur de projet" : par exemple, il est possible de déplacer un élément dans un perso-dossier ou dans un composant interne.

L'icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_synchro_explorateur.gif)
 permet de positionner le bandeau de sélection dans le volet "Explorateur de projet" sur l'élément en cours sous l'éditeur.

<a name="NOTE2_2"></a>


### Gestion du projet et de ses éléments
<a name="gestion_projet_ses_elements_ELTPARAGRAPHE000041"></a>Tous les éléments du projet sont regroupés dans le volet "Explorateur de projet". Un double-clic sur le nom de l'élément permet de l'afficher sous l'environnement.

**Le menu contextuel du projet** (de l'icône "P") permet d'effectuer les opérations les plus courantes sur le projet :

- Afficher la description du projet,

- Recompiler le projet,

- Créer un nouvel élément,

- Lister l'ensemble des éléments du projet,

- ...




Chaque élément du projet possède également son propre menu contextuel proposant toutes les fonctionnalités disponibles sur cet élément. Il est par exemple possible sur une fenêtre ou une page de réaliser les actions suivantes : 

- ouvrir l'élément sous l'éditeur correspondant, 

- tester l'élément, 

- dupliquer l'élément,

- afficher les codes de l'élément, ...




**L'élément "Configuration"** permet de gérer les [configurations de projet](../Editeurs/9000030.md). Le menu contextuel permet d'ajouter une nouvelle configuration ou de gérer les configurations existantes. Les éléments n'appartenant pas à la configuration sélectionnée sont grisés sous le volet "Explorateur de projet".

**Pour les fenêtres et les états**, le volet "Explorateur de projet" liste les champs présents dans l'élément sélectionné ainsi que les variables locales et les procédures locales.

**Pour les requêtes**, le volet "Explorateur de projet" liste les rubriques résultat de la requête ainsi que ses paramètres.

**La catégorie "Procédures"** regroupe les collections de procédures (collections de procédures du projet ou des composants présents dans le projet). Les constantes, les variables globales et les procédures sont listées pour chaque collection de procédures. Le volet "Explorateur de projet" permet également de créer des collections de procédures ou des procédures.

**Pour les classes**, le volet "Explorateur de projet" liste les membres et les méthodes. Le volet "Explorateur de projet" permet également de créer des nouvelles classes ou méthodes.

**Pour les composants internes**, le volet "Explorateur de projet" liste les composants internes et leurs éléments. Les éléments sont listés sous forme arborescente, regroupés par type (fenêtres, états, ...). 
Seules les branches contenant des éléments sont affichées. 

Le volet "Explorateur de projet" liste également les variables globales du projet et les fichiers et rubriques de l'analyse.
<a name="NOTE2_3"></a>


### Rétro-modélisation et références croisées
<a name="retromodelisation_references_croisees_ELTPARAGRAPHE000086"></a>Le menu contextuel des éléments du projet permet d'effectuer une rétro-modélisation et / ou d'afficher les références croisées.
<a name="NOTE2_4"></a>


### Gestion de l'analyse
<a name="gestion_analyse_ELTPARAGRAPHE000093"></a>Le volet "Explorateur de projet" affiche l'analyse du projet en cours, c'est-à-dire les fichiers de données et leurs rubriques. L'analyse affichée correspond à la dernière analyse générée et utilisable dans l'application.

**Rappel** : Le volet "Analyse" affiche l'analyse en cours d'édition et permet de manipuler la description des fichiers de données ou des rubriques.

L'éditeur d'analyses peut être ouvert depuis le volet "Explorateur de projet", via le menu contextuel des fichiers de données (option "Ouvrir l'analyse"). L'éditeur d'analyses se positionne alors sur le fichier de données sélectionné dans le volet "Explorateur de projet". Ce fichier de données est également sélectionné dans le volet ["Analyse"](../Editeurs/2027012.md). 

Un simple "Drag and Drop" des rubriques vers une fenêtre, une page ou un état permet de créer les champs correspondants.

**Remarque** : le volet "Explorateur de projet" affiche une bulle d'aide lors du survol d'une clé composée. Cette bulle contient les composantes de la clé composée. Le signe plus ou moins devant le nom de la composante indique le sens de tri. 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Volet_Clecomp.gif)

### Effectuer une recherche avec le volet "Explorateur de projet"
<a name="effectuer_une_recherche_avec_volet_explorateur_projet_ELTPARAGRAPHE000116"></a>Pour effectuer une recherche avec le volet "Explorateur de projet" :

1. Cliquez sur l'icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_Rech_Loupe.gif)
 pour lancer la recherche. 

2. La fenêtre de recherche parmi les éléments du projet s'affiche.
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CTRL_E.gif)


3. Saisissez le nom (ou une partie du nom) de l'élément recherché. La recherche est effectuée dans tous les noms des éléments du projet. Le résultat est affiché dans la fenêtre de recherche. Un double-clic sur l'élément permet de l'afficher sous l'éditeur correspondant. 

4. Pour effectuer une recherche plus approfondie, cochez l'option "Rechercher aussi les sous-éléments
(champs, procédures, ...)" : le résultat de la recherche prend en compte : 

	- toutes les références au mot recherché dans le code WLangage.

	- toutes les références au mot recherché dans le contenu des champs (libellé du champ, notes, ...).






<a name="NOTE2_6"></a>


### Catégorie "Autres" : inclure des éléments dans l'exécutable, la bibliothèque, ...
<a name="categorie_autres_inclure_des_elements_dans_executable_bibliotheque_ELTPARAGRAPHE000138"></a>Il est possible d'inclure des fichiers (fichiers de données HFSQL, images, sons, ...) directement dans l'exécutable ou l'archive généré. Il est ainsi possible d'utiliser des ressources qui ne doivent pas être visibles sur le disque de l'utilisateur final. 

Ces ressources doivent simplement être incluses dans le projet par l'intermédiaire du dossier "Autres" du volet "Explorateur de projet". 

Remarque : Les ressources présentes dans la catégorie "Autres" et inclues dans la bibliothèque (fichier WDL) pourront être extraites grâce à la fonction [fExtraitRessource](../WDLang1/1000019509.md). 

Les fonctionnalités suivantes permettent de simplifier la manipulation du dossier "Autres" : 

- Le "Drag and Drop" des éléments à inclure dans le dossier "Autres" est possible depuis l'explorateur de fichiers Windows. 

- Un menu contextuel sur le menu "Autres" permet d'ajouter ou de supprimer des éléments. 

- Un dossier "Autres" est également disponible dans chaque composant interne, ce qui permet d'inclure des éléments dans les composants internes partagés. 

- La fonction de recherche dans le projet parcourt également le dossier "Autres" du projet. 



<a name="NOTE2_6b"></a>


### Catégorie "Descriptions externes" : inclure des descriptions externes dans le projet
<a name="categorie_descriptions_externes_inclure_des_descriptions_externes_dans_projet_ELTPARAGRAPHE000163"></a>La catégorie "Descriptions externes" regroupe les différentes descriptions importées dans le projet : 

- Fichier XML, JSON, YAML. Pour plus de détails, consultez : 

	- [xlsDocument](../WDLang5/1000017464.md)

	- [YAML](../Motscles/1000024438.md)

	- [JSON](../Motscles/1000023611.md)




- Fichier XSD. Pour plus de détails, consultez [Gestion du XSD](../WDLang5/1000019170.md). 

- Module Python. Pour plus de détails, consultez [Python : Appels natifs](../WDLang6/1410088133.md). 




Le menu contextuel de cette catégorie permet d'importer ces diverses descriptions ou de les mettre à jour.

### Paramétrer l'affichage dans le volet "Explorateur de projet"
<a name="parametrer_affichage_dans_volet_explorateur_projet_ELTPARAGRAPHE000191"></a>Il est possible de configurer les options d'affichage et de tri utilisées dans le volet "Explorateur de projet" : 

1. Cliquez sur l'icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_propri%E9t%E9_Explorateur.gif)
.

2. Dans la fenêtre qui s'affiche, sélectionnez les options voulues : 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=WX_Explo_prj_Option.gif)





Les options d'affichage sont les suivantes : 

- Options d'affichage du volet "Explorateur de projet" :

	- **Trier par ordre alphabétique** : Tous les éléments du projet sont affichés dans leur dossier trié par ordre alphabétique (option par défaut).

	- **Trier par date de modification** : Tous les éléments du projet sont affichés dans leur dossier trié par ordre décroissant en fonction de leur dernière date de modification.




- Procédures et méthodes

	- **Trier par ordre alphabétique** : Les procédures et les méthodes sont affichées par ordre croissant en fonction de leur nom (option par défaut).

	- **Trier par date de modification** : Les procédures et les méthodes sont affichées par ordre décroissant en fonction de leur dernière date de modification.

	- **Trier par visibilité (public, protégé, privé)** : Les procédures et les méthodes sont classées et affichées en fonction de leur attribut de visibilité : Publique, Protégée et Privée.

	- **Trier par groupe de couleur** : Les procédures et les méthodes sont classées et affichées en fonction de la couleur qui leur a été attribuée.




- Classes

	- **Afficher les éléments des classes de base** : Cette option permet d'afficher ou non les éléments faisant partie des classes de base (par exemple, en cas de d'héritage ou de surcharge).

	- **Afficher uniquement les éléments publics** : Cette option permet d'afficher uniquement les éléments des classes (membres, méthodes) de type public.




- Rubriques de l'analyse

	- **Trier par ordre logique** : Les rubriques de chaque fichier de données sont affichées dans l'ordre dans lequel elles ont été créées lors de la saisie de la description du fichier de données.

	- **Trier par ordre alphabétique** : Les rubriques de chaque fichier de données sont affichées par ordre croissant en fonction de leur nom.




- Synchronisation : 

	- **Synchroniser automatiquement l'explorateur avec l'élément en cours d'édition** : Cette option permet de sélectionner automatiquement dans le volet "Explorateur de projet" l'élément en cours d'édition. Par exemple, si une fenêtre est affichée sous l'éditeur de fenêtres, son nom est automatiquement sélectionné dans le volet "Explorateur de projet".

## Afficher le volet "Explorateur de projet"
<a name="afficher_volet_explorateur_projet_ELTTEXTE000430"></a>
**Pour afficher le volet "Explorateur de projet"** :  sous le volet "Accueil", dans le groupe "Environnement", déroulez "Volets" et sélectionnez "Volets" puis "Explorateur de projet".

Pour plus de détails sur les manipulations des volets, consultez [Manipuler les volets ancrables](../Editeurs/2027001.md).

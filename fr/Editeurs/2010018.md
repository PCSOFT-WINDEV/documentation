## Autoriser la décompilation
<a name="autoriser_decompilation_ELTTEXTE000137"></a>
Si cette option est cochée, il sera possible (avec l'outil approprié) de décompiler le fichier d'aide au format CHM, c'est-à-dire d'obtenir toutes les pages HTML ainsi que les fichiers utilisés par ces pages (images par exemple).

La décompilation de fichiers d'aide permet d'obtenir rapidement les différentes pages d'un système d'aide directement à partir du fichier compilé, sans avoir les sources du système d'aide.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Compiler les informations pour la recherche en texte intégral
<a name="compiler_les_informations_pour_recherche_texte_integral_ELTTEXTE000161"></a>
Cette option permet de proposer un mode de recherche avancé (onglet "Rechercher" de l'aide).

Pour proposer ce mode, une liste des mots utilisés dans toutes les pages d'aide est automatiquement créée à la compilation.

Pour que cette option soit disponible dans les aides créées avec WINDEV, vous devez posséder le compilateur d'aide complet.

**Remarque : Recherche avancée :** Dans ce mode, le texte saisi par l'utilisateur sera recherché dans toutes les pages d'aide du fichier. Lors de la validation de la recherche, la liste des pages dans lesquelles le texte est trouvé est affiché. Dans cette liste, seuls les titres des pages sont affichés. Il suffit de sélectionner la page pour l'afficher.



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Créer un index binaire
<a name="creer_index_binaire_ELTTEXTE000185"></a>
Si cette option est cochée, un index binaire est créé. Cet index sera compilé directement dans le fichier d'aide CHM. La taille occupée par l'index sera donc réduite.

Cette option est nécessaire si vous voulez que votre index soit trié par ordre alphabétique.



<a name="NOTE4"></a>
<a name="NOTE4_1"></a>


## Créer une table des matières binaire
<a name="creer_une_table_des_matieres_binaire_ELTTEXTE000209"></a>
Cette option est conseillée pour les systèmes d'aide très importants car ainsi le sommaire est chargé plus rapidement lors de l'ouverture du fichier d'aide. De plus, le temps d'ouverture d'une page d'aide depuis le sommaire est également diminué.



<a name="NOTE5"></a>
<a name="NOTE5_1"></a>


## Remarque
<a name="remarque_ELTTEXTE000233"></a>
Pour des questions de sécurité, Windows autorise le lancement des fichiers CHM uniquement en local. Il n'est pas possible de lancer un CHM présent sur le réseau.

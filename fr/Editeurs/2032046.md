


## Fenêtre de description d'une requête de sélection : Utilisation 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000118"></a>
Voici quelques manipulations spécifiques à la fenêtre de description d'une requête de sélection :

- modifier l'ordre d'affichage de la liste des éléments de la requête.

- modifier les valeurs affichées dans le résultat de la requête.

- utiliser des rubriques provenant d'une requête existante.




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Trucs et astuces
<a name="trucs_astuces_ELTTEXTE000142"></a>


### Modifier l'ordre d'affichage de la liste des éléments de la requête
<a name="modifier_ordre_affichage_liste_des_elements_requete_ELTPARAGRAPHE000022"></a>Les éléments de la requête sont listés dans la partie centrale de la fenêtre de description (option "Description de la requête" du menu contextuel de la requête).

L'ordre d'affichage de la liste des éléments de la requête correspond :

- soit à l'ordre d'affichage des éléments dans le résultat de la requête : icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=OrdreAffichage.gif)
 (option sélectionnée par défaut).

- soit à l'enchaînement des tris dans le résultat de la requête : icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=OrdreTri.gif)
.




**Remarque** : Pour modifier l'ordre d'affichage / de tri des différentes rubriques dans le résultat de la requête, utilisez les boutons fléchés situés à droite de la liste des éléments.
<a name="NOTE2_3"></a>


### Modifier les valeurs affichées dans le résultat de la requête
<a name="modifier_les_valeurs_affichees_dans_resultat_requete_ELTPARAGRAPHE000040"></a>Il est possible d'afficher uniquement :

- l'année à partir d'une rubrique de type date.

- le mois à partir d'une rubrique de type date.

- le département à partir d'une rubrique contenant un code postal.




Pour modifier les valeurs affichées dans le résultat de la requête :

1. Sélectionnez une rubrique de type date ou contenant un code postal.

2. Cliquez sur le bouton "Année, département".

3. Sélectionnez l'option :

	- "Mois" pour afficher uniquement le mois dans le résultat de la requête.

	- "Année" pour afficher uniquement l'année dans le résultat de la requête.

	- "Département" pour afficher uniquement le département dans le résultat de la requête.




4. Validez.



<a name="NOTE2_4"></a>


### Utiliser des rubriques provenant d'une requête existante
<a name="utiliser_des_rubriques_provenant_une_requete_existante_ELTPARAGRAPHE000064"></a>Les rubriques pouvant être utilisées dans une requête appartiennent :

- soit à un fichier de l'analyse associée à l'application en cours.

- soit à une requête de l'application en cours.




La sélection d'une rubrique appartenant à un fichier de l'analyse est identique à la sélection d'une rubrique appartenant à une requête.

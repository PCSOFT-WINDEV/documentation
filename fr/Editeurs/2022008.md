
## Agrandir une fenêtre
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000111"></a>
Une tendance actuelle est l'agrandissement des fenêtres et de la taille des polices dans les champs. 

A partir de la version 28, il est possible d'agrandir automatiquement la taille des polices de l'ensemble des champs d'une sélection, d'une fenêtre ou d'un projet sans avoir à changer de gabarit et sans avoir à redimensionner les champs un à un. 

L'augmentation de la taille des polices agrandit et décale automatiquement tous les champs afin de conserver la mise en page. La taille de la fenêtre est également agrandie si nécessaire. 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000135"></a>
Pour agrandir une fenêtre : 

1. Ouvrez la fenêtre à agrandir sous l'éditeur. 

2. Sous le volet "Alignement", dans le groupe "Autres alignements", déroulez "Agrandir
Aérer" et sélectionnez "Agrandir". 

3. Dans la fenêtre qui s'affiche, indiquez les paramètres permettant de réaliser l'agrandissement : 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Agrandir_la_fen%EAtre%20-%20HC%20N%B0002.gif)
 Il est possible de : 

	- indiquer le facteur d'agrandissement. 

	- spécifier si la police et les styles doivent être agrandis. Dans ce cas, il est possible de : 

		- Créer des surcharges de style : le style de base sera surchargé pour prendre en compte les modifications. 

		- Créer / utiliser un gabarit agrandi : dans ce cas, un nouveau gabarit basé sur le gabarit en cours est généré. Par exemple, si la fenêtre utilise le gabarit "Phoenix", avec un agrandissement de 25%, un gabarit "Phoenix (x1.25)" sera créé.  




4. Validez. Les paramètres indiqués sont appliqués. 




Remarques : 

- L'agrandissement peut être effectué sur la sélection en cours, la fenêtre ou le projet. 

- Si la création d'un gabarit "agrandi" est demandée, ce gabarit pourra ensuite être appliqué aux autres éléments du projet. 

- Si le gabarit de base est mis à jour, le gabarit "agrandi" sera également mis à jour automatiquement. 

- Si la fenêtre contient un modèle de fenêtre, il est nécessaire d'agrandir également le modèle de fenêtre. Il est également possible d'agrandir le projet (le modèle sera automatiquement agrandi avec les autres éléments du projet). 

- Les modèles de champs ne sont pas agrandis.

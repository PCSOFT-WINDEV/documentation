## Développer un site Linux
<a name="developper_site_linux_ELTTEXTE000368"></a>
Le développement d'un site pour Linux se fait de manière identique au développement d'un site pour Windows.

Pour indiquer que votre site va être déployé sous Linux : 

1. Affichez la fenêtre de description de la configuration courante : sous le volet "Projet", dans le groupe "Configuration de projet", cliquez sur "Configuration courante". 

2. Cliquez sur l'onglet "Compatibilité" et cochez l'option "Activer les erreurs de compilation pour la compatibilité Linux".






A partir de la version 27, les distributions Linux supportées sont les suivantes :

- Debian en version 9 ou supérieur

- Fedora en version 25 ou supérieur

- Ubuntu en version 18.04 ou supérieur (version minimale de GCC : 5.1)

- OpenSUSE:Leap:15.0 avec GCC 7. La version OpenSUSE Leap 15.2 est également supportée.




A partir de la version 27, les distributions Linux suivantes ne sont plus supportées : 

- CentOS 7 (GCC 4.8)





Le site peut également être déployé dans une [image Docker](../Editeurs/3539080.md) du serveur d'application WEBDEV.

## Limitations
<a name="limitations_ELTTEXTE000440"></a>


- Les chemins de type UNC ne sont pas supportés en Linux.

- Seul l'alphabet occidental (constante *alphabetOccidental*) est supporté.

- Les fichiers XLS manipulés sur Linux doivent être des fichiers XLSX.

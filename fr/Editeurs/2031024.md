## Utilisation des patterns RAD
<a name="utilisation_des_patterns_rad_ELTTEXTE000147"></a>


### Les fichiers patterns
<a name="les_fichiers_patterns_ELTPARAGRAPHE000029"></a>Lors de la génération du pattern RAD, un fichier &lt;Nom Pattern&gt;.Pattern est généré dans le sous-répertoire EXE du projet. Pour être utilisé dans les différents produits, ce fichier doit être copié :

- dans le sous-répertoire "\\Programmes\\Donnees\\Modeles\\RAD\\WD\\Patterns" pour un pattern WINDEV.

- dans le sous-répertoire "\\Programmes\\Donnees\\Modeles\\RAD\\WB\\Patterns" pour un pattern WEBDEV.




Après copie, le nouveau pattern RAD est disponible dans le produit considéré (WINDEV ou WEBDEV) pour générer un RAD Application Complète (et si nécessaire un RAD fenêtre ou page).
<a name="NOTE2_2"></a>


### Génération d'un RAD Application complète
<a name="generation_rad_application_complete_ELTPARAGRAPHE000041"></a>Pour générer un RAD Application Complète :

1. Sous le volet "Projet", dans le groupe "Génération", cliquez sur "RAD Application complète".

2. Sélectionnez le pattern à utiliser et si nécessaire le gabarit à appliquer.

3. Suivez les différentes étapes de l'assistant.


**Remarque** : Si vous utilisez plusieurs fois sur le même projet l'assistant de création d'application complète du RAD, il est possible de générer uniquement les éléments correspondants aux fichiers ajoutés dans l'analyse. A chaque modification de l'analyse, une mise à jour du RAD est automatiquement proposée : les éléments nécessaires sont ajoutés à l'application générée par le RAD.
<a name="NOTE2_3"></a>


### Génération d'un RAD fenêtre ou d'un RAD page
<a name="generation_rad_fenetre_rad_page_ELTPARAGRAPHE000066"></a>Pour générer un RAD fenêtre ou page :

1. Cliquez sur ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_nouveau.gif)
 parmi les boutons d'accès rapide. La fenêtre de création d'un nouvel élément s'affiche : cliquez sur "Fenêtre" puis sur "Fenêtre" ou cliquez sur "Page" puis sur "Page".

2. Pour générer une fenêtre ou une page RAD, dans l'onglet "RAD", sélectionnez le pattern à utiliser (et si nécessaire le gabarit à utiliser).

3. Suivez les différentes étapes de l'assistant.


**Remarque** : En WEBDEV, seules les pages dynamiques utilisent les patterns RAD.




## Barre d'outils et menu du GDS
			





## Menu GDS
<a name="menu_gds_ELTTEXTE000165"></a>
Les opérations sur le GDS peuvent être effectuées directement depuis le volet GDS de WINDEV, WEBDEV ou WINDEV Mobile.

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=GDS_Barre_outils%20-%20HC%20N%B0002.gif&type=thumb)
Ce volet propose différentes zones : 

- "Base GDS".

- "Tous les éléments". 

- "Elément en cours". 

- "Projet". 

- "Autres actions". 




Voyons en détail les options de ces différentes zones. 


### Zone "Base GDS"
<a name="zone_base_gds_ELTPARAGRAPHE000021"></a>
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=GDS_Barre_outils%20-%20HC%20N%B0001.gif)
L'option "Gérer" permet de : 

- Lancer l'administrateur du Gestionnaire de Sources. Pour plus de détails, consultez [Administrateur du GDS](../Editeurs/2038008.md). 

- Créer une base de sources. Pour plus de détails, consultez [Créer une base de sources](../Editeurs/2038005.md). 

- Définir une politique de réintégration. Pour plus de détails, consultez [GDS : Politique de réintégration](../Editeurs/2038024.md).





### Zone "Tous les éléments"
<a name="zone_tous_les_elements_ELTPARAGRAPHE000040"></a>
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=GDS_Barre_outils%20-%20HC%20N%B0002%201.gif)
Cette zone permet de gérer tous les éléments du projet. 

Il est possible de : 

- **Tout récupérer** : Permet de récupérer toutes les nouvelles versions des éléments du projet qui sont présents dans la base de sources. Les éléments qui sont actuellement extraits par l'utilisateur ne sont pas proposés.

- **Tout réintégrer** : Permet de réintégrer dans la base de sources du GDS tous les éléments du projet en cours qui ont été extraits. Il est possible de garder extraits certains éléments.




L'option "Etat des éléments" permet de : 

- Connaître l'état des éléments : La liste des éléments du projet est affichée, avec pour chaque élément la date de la dernière extraction et son auteur.  

- Rafraîchir l'état des éléments : Permet de mettre à jour la liste des éléments du projet et l'état de chaque élément. 





### Zone "Elément en cours"
<a name="zone_element_cours_ELTPARAGRAPHE000056"></a>
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=GDS_Barre_outils%20-%20HC%20N%B0002%202.gif)
Cette zone permet de gérer l'élément en cours sous l'éditeur. 

Il est possible de : 

- **Extraire** : Permet d'extraire de la base de sources le fichier correspondant à l'élément en cours. La dernière version de l'élément est recopié sur le poste local. Plusieurs modes d'extraction sont disponibles. Pour plus de détails, consultez [Extraction d'un élément](../Editeurs/2038006.md).

- **Réintégrer** : Permet de réintégrer dans la base de sources le fichier correspondant à l'élément en cours. L'élément est recopié dans la base de sources du GDS. Il est mis à la disposition des autres utilisateurs du GDS. Pour plus de détails, consultez [Réintégration d'un élément](../Editeurs/2038007.md).

- **Annuler l'extraction** : Permet d'annuler la dernière extraction. Les modifications faites sont annulées, et le fichier repasse en mode "Lecture seule".

- **Récupérer la dernière version** : Permet d'obtenir la dernière version de l'élément en cours sous l'éditeur. Cette dernière version est celle présente dans la base de source du GDS. 

- **Comparer** : 

	- **Voir mes modifications depuis l'extraction** : Permet de voir les modifications réalisées sur l'élément en cours depuis son extraction.

	- **Comparer avec la version de la base** : Compare l'élément en cours (disponible en local) avec la version présente dans la base de sources du GDS. Cette comparaison peut être faite même si l'élément n'a pas été extrait. Permet par exemple de vérifier les modifications effectuées avant de récupérer l'élément.




- **Ajouter** : Permet d'ajouter l'élément actuellement présent dans l'éditeur dans la base de sources du GDS. Si cette option est utilisée depuis l'éditeur d'analyses, c'est l'analyse qui est ajoutée dans la base de sources. Pour plus de détails, consultez [Ajout d'un élément dans la base de sources](../Editeurs/2038009.md).

- **Propriétés** : Permet d'afficher les propriétés de l'élément en cours (les différentes extractions, les partages, ...).

- **Historique** : Permet d'obtenir la liste des modifications réalisées sur l'élément et par qui. Il est possible d'obtenir une ancienne version, de l'enregistrer, de comparer les différences (entre 2 versions par exemple), ou de fusionner la version du GDS avec votre version.





### Zone "Projet"
<a name="zone_projet_ELTPARAGRAPHE000085"></a>
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=GDS_Barre_outils%20-%20HC%20N%B0002%203.gif)
Cette zone permet de gérer le projet en cours sous l'éditeur. 

Il est possible de : 

- **Ajouter le projet** : Permet d'ajouter le projet actuellement ouvert dans l'éditeur dans la base de sources du GDS. 

- **Importer depuis le GDS** : Permet d'importer dans le projet en cours un élément présent dans le GDS (par exemple un composant, une fenêtre présente dans un autre projet). Il est possible de partager l'élément ou d'importer une copie.

- Gérer le mode **Maître / Invité** : cette option permet de gérer les options de manipulation des projets maître et invités. Pour plus de détails, consultez [Travailler avec le Gestionnaire de sources](../Editeurs/2038004.md).

- Afficher **l'historique du projet** : Permet d'obtenir la liste des modifications réalisées sur le projet dans son ensemble et sur le fichier du projet (fichier WDP, WWP ou WPP). Il est possible d'obtenir une ancienne version, de l'enregistrer, de comparer les différences (entre 2 versions par exemple), d'étiqueter la version du projet, ... 

- Gérer les **Branches** : Cette option permet de : 

	- Créer une branche. 

	- Récupérer les modifications depuis une branche. 
			Pour plus de détails, consultez [GDS : Gestion des branches](../Editeurs/2038016.md). 








### Zone "Autres actions"
<a name="zone_autres_actions_ELTPARAGRAPHE000110"></a>
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=GDS_Barre_outils%20-%20HC%20N%B0002%204.gif)
Cette zone permet de gérer la feuille de style du projet et le travail à distance. 

Il est possible de : 

- Extraire ou réintégrer la feuille de styles du projet. 

- Se déconnecter pour une utilisation nomade puis se reconnecter et se synchroniser.




## Ensembles de polices (site WEBDEV)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000175"></a>
Les navigateurs supportent par défaut les polices de caractères les plus courantes : Arial, Verdana, ...

WEBDEV offre la possibilité d'utiliser des polices de caractères personnalisées, différentes de celles proposées par défaut. 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000199"></a>


### Gestion des polices
<a name="gestion_des_polices_ELTPARAGRAPHE000020"></a>Lors de la création ou modification d'un style, il est possible de sélectionner une police parmi les polices standard.

Il est également possible de sélectionner de nouvelles polices. Ces polices peuvent provenir : 

- du poste de développement directement (ces polices ont été installées sur votre poste dans Windows). 

- d'un téléchargement (par exemple, télécharger des polices Google).



<a name="NOTE2_2"></a>


### Créer un ensemble de polices
<a name="creer_ensemble_polices_ELTPARAGRAPHE000032"></a>Pour ajouter un ensemble de polices : 

1. Ouvrez la fenêtre de gestion des styles (onglet "Style" de la description du champ). 

2. Dans la zone "Polices", cliquez sur le bouton "..." : 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Police_perso_WB_1.gif)


3. La fenêtre de gestion des polices s'affiche. 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Police_perso_WB_2.gif&type=thumb)
Cette fenêtre présente les ensembles de polices existants. 

4. Cliquez sur "Nouveau" pour créer un nouvel ensemble de polices. 

5. Dans l'assistant qui se lance, vous devez sélectionner les différentes polices qui constituent l'ensemble de polices. 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Police_perso_WB_3.gif)
Il est possible au choix : 

	- de sélectionner une police téléchargeable. Cette police sera automatiquement téléchargée sur le poste de l'internaute.  

	- de sélectionner une police système, qui sera affichée que si elle est installée sur le poste de l'internaute. Si cette option est sélectionnée, vous devez ensuite sélectionner la police voulue.  
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Police_perso_WB_4.gif)


	- de sélectionner une police générique, qui sera disponible pour tous les navigateurs. 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Police_perso_WB_5.gif)
Remarque : Ce type de police est obligatoire dans un ensemble de polices. Il est ainsi possible de s'assurer que le texte sera affiché quelque soit le poste de l'internaute. 

6. Sélectionnez le type de police voulu puis la police qui doit être ajoutée à l'ensemble de polices. 

7. La fenêtre présentant les différentes polices de l'ensemble est affichée. 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Police_perso_WB_6.gif)
Vous pouvez : 

	- Ajouter une nouvelle police (![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Police_perso_WB_6%203.gif)
). L'assistant se lance à nouveau. 

	- Supprimer la police sélectionnée (![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Police_perso_WB_6%202.gif)
). 

	- Modifier l'ordre des polices dans l'ensemble (grâce aux flèches ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Police_perso_WB_6%201.gif)
). 




8. Validez la fenêtre de création de l'ensemble de polices. 

9. Validez la fenêtre de gestion des ensembles de polices. L'ensemble de polices  sélectionné est appliqué au style en cours. 



<a name="NOTE2_3"></a>


### Importer une police
<a name="importer_une_police_ELTPARAGRAPHE000075"></a>Pour importer une nouvelle police : 

1. Dans la fenêtre de gestion des ensembles de polices, cliquez sur "Nouveau". 

2. Dans l'assistant qui se lance, sélectionnez "Police téléchargeable". Passez à l'étape suivante de l'assistant. 

3. Dans la fenêtre qui s'affiche, cliquez sur "Importer une police". 

4. L'assistant d'importation d'une police se lance. 

5. Sélectionnez le mode d'import de la police : 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Police_perso_WB%20-%20HC%20N%B0007.gif)


	- Importer une police Google Web Fonts : 

	- L'assistant propose alors la liste des polices Google à importer. 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Police_perso_WB%20-%20HC%20N%B0008.gif)


**Remarques** : 

- Assurez-vous que les fichiers de police de caractères utilisés sont libres de droits ou que vous possédez une licence vous autorisant à les distribuer. 

- Si vous importez une police et créez un ensemble de polices, les fichiers correspondants à cette police sont copiés dans le sous-répertoire "&lt;Nom_projet&gt;_WEB\\Styles\\fonts". Ainsi, il seront automatiquement pris en compte lors de l'installation du site.




**Attention : Ensemble de polices contenant des polices importées :** 
Avant d'utiliser une police téléchargée, il est nécessaire de l'intégrer dans un ensemble de polices similaires. 

Si l'ensemble de polices contient uniquement la police téléchargeable, le cas suivant peut être observé : si la police n'est pas présente sur le navigateur : 

- la police Times est affichée par défaut, le temps que la police demandée soit téléchargée. 

- si la police ne peut pas être téléchargée, la police Times est toujours affichée. 




Pour éviter l'affichage d'une police non désirée, il est conseillé de créer un ensemble de polices contenant les éléments suivants : 

- la ou les polices à télécharger, 

- une ou des polices de substitution qui correspondent aux polices des principaux systèmes, 

- une ou des polices qui correspondent à des polices très populaires, 

- le type de familles de polices associé. La liste des types de familles de polices est la suivante : 

	- "cursive" : style manuscrit,

	- "fantasy" : comme Impact, 

	- "monospace" comme Courier, 

	- "sans-serif" comme Arial, 

	- "serif" comme Times.





Si la première police n'est pas trouvée, la première présente dans la liste et sur le poste sera utilisée. 

**Exemple** : L'ensemble de polices suivant est défini : "Open Sans, Segoe UI, Arial, Helvetica, sans-serif". En l'absence de la police "Open Sans", la police "Segoe UI" sera utilisée, sinon la police Arial ou Helvetica est utilisée. Ces polices sont suffisamment connues pour avoir des alternatives correctes. Si ces deux polices ne sont pas trouvées, la première police disponible qui n'a pas de serif sera utilisée.

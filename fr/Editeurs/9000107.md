


## Correcteur d'orthographe dans l'interface
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000127"></a>
WINDEV, WEBDEV et WINDEV Mobile permettent d'utiliser un correcteur orthographique directement dans vos interfaces développées. Le correcteur orthographique est disponible dans l'éditeur de fenêtres, dans l'éditeur de pages et l'éditeur d'états.

Jusqu'à la version 23, le correcteur utilisé est le correcteur orthographique de OpenOffice. 

A partir de la version 24, le correcteur orthographique est intégré à WINDEV, WEBDEV et WINDEV Mobile. 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000151"></a>


### Principe
<a name="principe_ELTPARAGRAPHE000022"></a>Le correcteur orthographique détecte les fautes dans le texte saisi sous les différents éditeurs :

- Texte directement affiché sous l'éditeur (par exemple libellé d'un champ).

- Texte saisi dans les fenêtres de description de l'éditeur.




Le correcteur orthographique est intégré à WINDEV, WEBDEV, WINDEV Mobile. La correction orthographique est proposée par défaut en français et anglais. Pour utiliser des langues supplémentaires, il suffit d'ajouter les fichiers correspondant aux dictionnaires de type Hunspell voulus dans le répertoire "\\Programmes\\Donnees\\Dic". 

Ces dictionnaires peuvent facilement être trouvés sur Internet. Ces dictionnaires sont constitués d'un fichier ".dic" et d'un fichier ".aff" de même nom qui doivent se trouver au même emplacement. Pour plus de détails, consultez [Vérification d'orthographe basée sur les dictionnaires Hunspell](../WDChamp/1410087030.md). 
Remarque : Les fichiers ".dic" et ".aff" doivent avoir pour nom &lt;Abréviation de la langue&gt;-&lt;Abréviation sous-langue&lt;. Par exemple : 

- fr-FR.dic / fr-FR.aff

- de_DE.fic / de_DE.aff

- nl_NL.dic / nl_NL.aff


Le chargement des dictionnaires peut être vérifié avec la fonction [DictionnaireCharge](../WDLang1/1000022958.md). 
<a name="NOTE2_2"></a>


### Activer/Désactiver le correcteur orthographique
<a name="activerdesactiver_correcteur_orthographique_ELTPARAGRAPHE000050"></a>Pour activer/désactiver le correcteur orthographique :

1. Affichez les options générales de l'environnement : sous le volet "Accueil", dans le groupe "Environnement", déroulez "Options" et sélectionnez "Options générales de WINDEV/WEBDEV/WINDEV Mobile".

2. Dans l'onglet "Général", cochez ou décochez l'option "Désactiver le correcteur orthographique".




**Remarques** : 

- Il est également possible d'obtenir le correcteur orthographique dans l'éditeur de code. Pour plus de détails, consultez [Correcteurs sous l'éditeur de code](../Editeurs/9000073.md).

- Le correcteur orthographique s'arrête automatiquement si plus de 50% de mots sont faux. En effet, dans ce cas, le correcteur considère que la langue n'est pas connue ou bien qu'il s'agit d'un texte contenant des références, des noms propres, et donc que ce texte n'est pas à corriger. 

- Les mots ajoutés au dictionnaire sont présents dans le fichier "C:\\Users\\&lt;Utilisateur&gt;\\AppData\\Roaming\\PC SOFT\\perso.dic".

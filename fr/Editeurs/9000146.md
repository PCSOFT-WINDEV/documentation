
## Fenêtre à flèche (iPad)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000198"></a>
Le principe des fenêtres à flèche est simple : lorsque l'utilisateur clique sur un bouton, la fenêtre va s'afficher contre le bouton appelant avec une flèche de direction entre le bouton et la fenêtre. 

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=iPad_Fen_fleche.gif)


**Remarques** : 

- Ce type de fenêtre est disponible uniquement pour les applications iPad. Si l'option "Fenêtre avec flèche" est sélectionnée pour une application iPhone, la fenêtre sera non maximisée sans flèche.

- La fenêtre avec flèche utilise le look système : le cadre de la fenêtre et le dessin de la flèche sont fournis par le système iOS.

## Création d'une fenêtre avec flèche
<a name="creation_une_fenetre_avec_fleche_ELTTEXTE000222"></a>


### Création d'une fenêtre avec flèche
<a name="creation_une_fenetre_avec_fleche_ELTPARAGRAPHE000028"></a>Pour créer une fenêtre avec flèche : 

1. Créez une nouvelle fenêtre : 

	- Cliquez sur ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_nouveau.gif)
 parmi les boutons d'accès rapide. 

	- La fenêtre de création d'un nouvel élément s'affiche : cliquez sur "Fenêtre" puis sur "Fenêtre". 

	- Dans l'assistant de création d'une fenêtre, sélectionnez "Vierge non maximisée avec flèche" et validez. 




2. La fenêtre avec flèche apparaît sous l'éditeur. Le cadre et la flèche sont affichés sous l'éditeur mais ne peuvent être modifiés. Par contre, la taille de la fenêtre peut être modifiée. 



<a name="NOTE2_2"></a>


### Barre de titre
<a name="barre_titre_ELTPARAGRAPHE000049"></a>Par défaut, la fenêtre avec flèche créée ne contient pas de barre de titre. 

Pour afficher une barre de titre : 

1. Affichez la description de la fenêtre (option "Description" du menu contextuel). 

2. Dans l'onglet "Style", cochez l'option "Avec barre de titre". 

3. Validez. 




Le contenu affiché dans la barre de titre correspond au titre de la fenêtre saisi dans l'onglet "Général" de la fenêtre de description de la fenêtre avec flèche. 

**Remarque** : Le changement du titre par programmation (propriété [Titre](../Proprietes/2510090.md)) est possible uniquement dans l'événement "Déclarations globales" de la fenêtre. 
<a name="NOTE2_3"></a>


### Transformer une fenêtre maximisée en fenêtre à flèche
<a name="transformer_une_fenetre_maximisee_fenetre_fleche_ELTPARAGRAPHE000073"></a>Pour transformer une fenêtre maximisée en fenêtre à flèche : 

1. Affichez la description de la fenêtre (option "Description" du menu contextuel). 

2. Dans l'onglet "Style" : 

	- Sélectionnez l'option "Fenêtre non maximisée/détourée". 

	- Dans les paramètres, sélectionnez : 

		- l'option "Fenêtre avec flèche". 

		- l'option "Avec barre de titre" pour qu'une barre de titre soit automatiquement insérée dans la fenêtre. 




3. Validez.

## Programmation d'une fenêtre à flèche
<a name="programmation_une_fenetre_fleche_ELTTEXTE000258"></a>


### Ouverture d'une fenêtre à flèche
<a name="ouverture_une_fenetre_fleche_ELTPARAGRAPHE000104"></a>Pour ouvrir une fenêtre à flèche depuis un bouton de l'application : 

- Sous l'éditeur, utilisez l'action rapide de type "Ouvrir" associée au bouton. 

	- Affichez la fenêtre de description du champ Bouton. 

	- Sous l'onglet "Général", dans la zone "Action", sélectionnez la fenêtre à flèche à ouvrir. ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=iPad_Fen_fleche_ouvrir.gif)





- Par programmation, utilisez la fonction [OuvreFille](../WDLang1/3038021.md) dans le code de clic du bouton. 
	 Remarque : Il n'est pas autorisé d'utiliser plusieurs fois successivement la fonction [OuvreFille](../WDLang1/3038021.md) depuis la fenêtre mère. Par contre, une fenêtre mère peut ouvrir une fenêtre fille, qui elle-même peut ouvrir une nouvelle fenêtre fille.  



<a name="NOTE3_2"></a>


### Fermeture d'une fenêtre à flèche
<a name="fermeture_une_fenetre_fleche_ELTPARAGRAPHE000126"></a>La fermeture d'une fenêtre à flèche peut être réalisée : 

- via un bouton qui contient la fonction [Ferme](../WDLang1/3038018.md). 

- lors d'un clic de l'utilisateur en dehors de la fenêtre. Dans ce cas, l'événement "Fermeture" de la fenêtre est exécuté. 



<a name="NOTE3_3"></a>


### Remarques
<a name="remarques_ELTPARAGRAPHE000139"></a>

- Il est possible de passer des paramètres à une fenêtre à flèche. Pour plus de détails, consultez [Fenêtre paramétrée](../WDChamp/1010026.md).

- La fenêtre à flèche peut renvoyer une valeur qui sera récupérée dans la fenêtre principale (dans l'événement "Fermeture d'une fenêtre fille"). Pour plus de détails, consultez l'aide de la fonction [Ferme](../WDLang1/3038018.md).




## Installation au format MSI
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000121"></a>
Il est possible de créer un programme d'installation à la norme MSI. Ces procédures d'installation ont une taille plus réduite et permettent l'utilisation de l'installateur dans le cadre de stratégie de groupe.

**Remarque** : Le programme d'installation à la norme MSI ne peut être utilisé que dans le cadre d'une installation autonome. En effet, ce format ne gère pas le cas des installations réseau, ni la modification automatique des fichiers de données.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000145"></a>


### Création d'une installation MSI via l'assistant
<a name="creation_une_installation_msi_via_assistant_ELTPARAGRAPHE000020"></a>Pour créer une installation à la norme MSI :

1. Créez l'exécutable de l'application : sous le volet "Projet", dans le groupe "Génération", cliquez sur "Générer".

2. Lancez la création de la procédure d'installation : sous le volet "Projet", dans le groupe "Génération", cliquez sur "Procédure d'installation".

3. Dans l'assistant d'installation, sur la partie gauche de l'assistant, cliquez sur "2 - Installation". Dans la boîte de dialogue qui s'affiche, sélectionnez "Accéder aux options d'installation".

4. Choisissez une "Installation individuelle", puis l'option "Installation par un Package MSI" dans l'étape suivante.

5. Dans la suite de l'assistant :

	- spécifiez le répertoire d'installation. Le bouton "Avancé" permet de définir les modifications que l'installation doit apporter à la configuration système (base de registres, fichier INI, ...).

	- sélectionnez les fichiers à installer.

	- indiquez les options d'installation du framework WINDEV.

	- indiquez si une licence doit être affichée ou non.

	- sélectionnez le contexte d'installation : pour l'utilisateur courant ou pour tous les utilisateurs de la machine. Cette option permet de créer correctement les raccourcis de lancement de l'application. 

	- indiquez si le pack d'installation MSI doit être signé. Si le pack doit être signé, sélectionnez le certificat à utiliser.




6. Validez l'assistant. L'installation au format MSI est automatiquement créée. 




**Attention** : Il n'est actuellement pas possible de lancer un exécutable à la fin du programme d'installation de type "Package MSI".

Remarques : 

- L'installation MSI peut être certifiée (Windows App Certification Kit). 

- L'installation MSI peut être signée. 

- L'installation MSI est reconnue par le SCCM de Microsoft.

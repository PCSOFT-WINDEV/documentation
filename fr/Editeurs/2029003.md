


## Environnement de l'éditeur de requêtes 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Pour afficher l'éditeur de requêtes, il suffit de : 

- créer une nouvelle requête : 

	- cliquez sur ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_nouveau.gif)
 présent dans les boutons d'accès rapide. 

	- la fenêtre de création d'un nouvel élément s'affiche : cliquez sur "Requêtes".




- ouvrir une requête existante (via le volet "Explorateur de projet" par exemple).



![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Requete_Interface.gif&type=thumb)


## Propriété d'une entité / d'une association
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000113"></a>
Selon la méthode MERISE, les propriétés correspondent aux caractéristiques des entités.

L'éditeur d'analyses permet de créer facilement des propriétés.

Lors de la [génération du MCD en MLD](../Editeurs/2011020.md), les propriétés deviendront automatiquement des [rubriques](../Editeurs/2011009.md).



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Création d'une propriété
<a name="creation_une_propriete_ELTTEXTE000137"></a>
Pour décrire une propriété :

1. Double-cliquez sur l'entité ou sur l'association dans laquelle la propriété doit être créée. La fenêtre de description des propriétés s'affiche.

2. Cliquez sur la première ligne vide du tableau des propriétés ou cliquez sur ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=IconeAjouterPropriete.gif)


3. Spécifiez le nom, le libellé, le type, la taille et les notes de la propriété.

4. Dans le cas d'une propriété d'entité, cochez l'option "Identifiant" si la propriété est l'identifiant de l'entité.

5. Validez la description de la propriété.

## Principales manipulations d'une propriété
<a name="principales_manipulations_une_propriete_ELTTEXTE000161"></a>




### Supprimer une propriété
<a name="supprimer_une_propriete_ELTPARAGRAPHE000046"></a>Pour supprimer une propriété :

1. Sélectionnez la propriété à supprimer dans la description des propriétés d'une entité ou d'une association.

2. Cliquez sur ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=IconeSupprimerPropriete.gif)
. La propriété sera automatiquement supprimée de l'entité ou de l'association.





<a name="NOTE3_2"></a>




### Renommer une propriété
<a name="renommer_une_propriete_ELTPARAGRAPHE000060"></a>Pour renommer une propriété :

1. Sélectionnez la propriété à renommer dans la description des propriétés d'une entité ou d'une association.

2. Saisissez directement le nouveau nom de la propriété et validez la modification. La propriété est automatiquement renommée.


## Paramétrer la modification automatique des fichiers de données 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000124"></a>
La modification automatique des fichiers de données permet de mettre à jour la description des fichiers de données présents sur les postes utilisateurs.

Pour plus de détails, consultez la [Modification automatique des fichiers de données](../WDLang4/3044195.md).





<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Paramétrer la modification automatique des données grâce à l'assistant
<a name="parametrer_modification_automatique_des_donnees_grace_assistant_ELTTEXTE000154"></a>
Pour paramétrer la modification automatique des données grâce à l'assistant, sélectionnez (cochez) l'application (fichier ".EXE") pour laquelle la modification automatique des fichiers de données doit être effectuée.

Si les fichiers de données sont protégés par un mot de passe ou s'ils sont dans une base HFSQL Client/Serveur, il est nécessaire de fournir les mots de passe : 

1. Cliquez sur le bouton "Avancé".

2. Sélectionnez l'analyse au format WINDEV (fichier ".WDD") associée à l'application.

3. Pour chaque fichier présent dans l'analyse, spécifiez si nécessaire :

	- le mot de passe du fichier (s'il existe) dans l'analyse. 
			**Remarque** : Pour modifier ce mot de passe, cochez l'option "Modifier le mot de passe" et saisissez deux fois le nouveau mot de passe.

	- si l'application utilise une analyse WINDEV 5.5 migrée, indiquez :

		- l'analyse d'origine au format WINDEV 5.5 (fichier ".WDA").

		- le nom du fichier dans l'analyse au format WINDEV 5.5.

		- le mot de passe du fichier (s'il existe) dans l'analyse au format WINDEV 5.5.







Toutes ces informations à l'exception des mots de passe sont automatiquement affichées.

**Remarques** :

- Lors de la mise à jour, les fichiers de données automatiquement trouvés sont :

	- les fichiers présents dans le répertoire d'installation de l'application.

	- les fichiers listés dans le fichier ".REP". Pour plus de détails sur le fichier ".REP", consultez l'aide sur la [gestion du .REP](../WDLang4/3044186.md).




- L'assistant propose une modification automatique des données "silencieuse" : aucun écran de paramétrage ne sera affiché à l'utilisateur final. 

- Pour permettre à l'utilisateur d'accéder aux options de configuration de la modification automatique, l'éditeur d'installation WDInst permet de rendre visible le bouton "Options avancées" du programme d'installation.


## Copie des champs depuis l'éditeur d'états
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000211"></a>
Il est possible de copier les champs de l'éditeur d'états : 

- soit vers l'éditeur de fenêtres.

- soit vers l'éditeur de pages. 




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Conversion à partir de l'éditeur d'états
<a name="conversion_partir_editeur_etats_ELTTEXTE000235"></a>


### Copie de l'éditeur d'états vers l'éditeur de fenêtres
<a name="copie_editeur_etats_vers_editeur_fenetres_ELTPARAGRAPHE000020"></a>Les champs gérés lors de cette copie sont les suivants : 

| Champ présent dans l'éditeur d'états | Champ copié dans l'éditeur de fenêtres |
| --- | --- |
| Libellé | Bouton |
| Rubrique | Libellé |
| Image | Image |
| Code-barres | Code-barres |
| RTF | Libellé |
| HTML | Libellé |
| Prédéfini | Libellé |
| Graphe | Graphe |
| Interrupteur | Interrupteur |
| Lien | Bouton |
| Saisie | Saisie |
| Planning | Planning |
| Calcul | NA |
| Forme | NA |
| Etat Interne | NA |






<a name="NOTE2_2"></a>


### Copie de l'éditeur d'états vers l'éditeur de pages
<a name="copie_editeur_etats_vers_editeur_pages_ELTPARAGRAPHE000094"></a>Les champs gérés lors de cette copie sont les suivants : 

| Champ présent dans l'éditeur d'états | Champ copié dans l'éditeur de pages |
| --- | --- |
| Libellé | Bouton |
| Rubrique | Champ formaté |
| Image | Image |
| Code-barres | Code-barres |
| RTF | Libellé HTML |
| HTML | Champ d'affichage formaté |
| Prédéfini | Champ formaté |
| Graphe | Graphe |
| Interrupteur | Interrupteur |
| Lien | Lien |
| Saisie | Saisie |
| Planning | Planning |
| Calcul | NA |
| Forme | NA |
| Etat Interne | NA |

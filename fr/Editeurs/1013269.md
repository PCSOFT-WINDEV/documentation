
## Installation de OpenOffice
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000110"></a>
WINDEV, WEBDEV et WINDEV Mobile proposent une correction orthographique intégrée à plusieurs niveaux :

- Correction orthographique sous l'éditeur de fenêtres, de pages et d'états.

- [Correction orthographique sous l'éditeur de code](../Editeurs/9000073.md).  

- [Correction orthographique dans les champs de saisie en exécution](../Editeurs/1013146.md).




Dans tous les cas, la correction orthographique nécessite l'installation sur le poste de OpenOffice (2.0 ou supérieur).

Cette page d'aide présente comment installer OpenOffice 2.x et ses dictionnaires. En effet, les dictionnaires OpenOffice ne sont pas installés par défaut dans la version 2.x. 



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Installation de OpenOffice (2.x ou supérieur)
<a name="installation_openoffice_2x_superieur_ELTTEXTE000134"></a>
OpenOffice est une suite bureautique disponible gratuitement en téléchargement.

**Pour installer OpenOffice** :

1. Téléchargez OpenOffice à l'adresse suivante : [http://www.openoffice.org](http://www.openoffice.org).

2. Installez le logiciel.




Attention : Selon la version utilisée, les dictionnaires ne sont pas toujours installés par défaut.



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Installation des dictionnaires de OpenOffice
<a name="installation_des_dictionnaires_openoffice_ELTTEXTE000158"></a>
Par défaut, les dictionnaires servant à la correction orthographique ne sont pas installés avec OpenOffice.

**Pour installer ces dictionnaires (version 2.0)** : 

1. Lancez OpenOffice Writer 2.0.

2. Sélectionnez l'option de menu "Fichier .. Assistants .. Installer de nouveaux dictionnaires".

3. Suivez les différentes étapes de l'assistant, puis relancez OpenOffice.

4. Rebootez si nécessaire le poste pour prendre en compte les dictionnaires.




**Pour installer ces dictionnaires (version 3.0 et supérieure)** : 

1. Lancez OpenOffice Writer.

2. Sélectionnez l'option de menu "Outils .. Langue .. Dictionnaires supplémentaires en ligne".

3. Suivez les différentes étapes de l'assistant, puis relancez OpenOffice.

## Présentation
<a name="presentation_ELTTEXTE000184"></a>
Votre fenêtre affiche des champs qui se superposent ?

Vous voulez connaître l'altitude de vos champs : quels sont les champs affichés en premier plan ?

L'affichage en deux dimensions devient vite limité pour ce genre d'opération.

WINDEV permet d'avoir une vue spatiale d'une fenêtre sous l'éditeur en proposant de visualiser la fenêtre en trois dimensions. L'altitude des différents champs de la fenêtre apparaît alors immédiatement. Un champ sélectionné dans ce mode d'affichage reste sélectionné lors du passage à l'affichage en deux dimensions.

Vous pouvez choisir simplement le meilleur angle de vision grâce à la souris et aux touches du clavier.

**Attention** : pour utiliser cette fonctionnalité, une configuration spécifique est nécessaire.

## Comment le faire ?
<a name="comment_faire_ELTTEXTE000232"></a>
Pour afficher une vue en 3 dimensions de vos fenêtres :

1. Affichez la fenêtre sous l'éditeur.

2. Sous le volet "Affichage", dans le groupe "Options", cliquez sur "3D".

3. La fenêtre apparaît en 3 dimensions.




**Remarque** : Pour afficher une vue en 3 dimensions de votre fenêtre et animer cette vue, utilisez les touches Alt + Maj.

Pour optimiser la vue de la fenêtre, vous pouvez utiliser la souris ou le clavier.

| Déplacements à la souris |   |
| --- | --- |
| Rotation (bouton gauche de la souris) | Cliquez dans la vue de la fenêtre et déplacez la souris tout en maintenant le bouton gauche de la souris enfoncé. |
| Translation (bouton droit de la souris) | Cliquez dans la vue de la fenêtre et déplacez la souris tout en maintenant le bouton droit de la souris enfoncé. |
| **Déplacements au clavier** |   |
| Rotation | Utilisez les flèches ainsi que les touches + et - du clavier numérique.<br><br>Il est également possible d'utiliser les chiffres du clavier numérique. |
| Translation | Maintenez la touche Maj du clavier enfoncée et utilisez les flèches du clavier numérique. |
| Zoom | Utilisez la combinaison de touches :<br><br>- Maj + pour réaliser un zoom avant<br><br>- Maj - pour réaliser un zoom arrière.<br><br><br> |
| Modification de l'écart entre les champs (modification entre les différents plans d'affichage) | / (du pavé numérique) pour diminuer l'écart entre les champs.<br>"\*" (du pavé numérique) pour augmenter l'écart entre les champs |
| Réinitialisation de l'affichage | Touche 0 du clavier numérique : réinitialisation complète<br>Touche 5 du clavier numérique : réinitialisation en conservant l'espace entre les champs. |
| Animation de la vue en 3 Dimensions | Touche Arrêt défil.<br>Touche Echap pour annuler |

**Remarque** : Pour désactiver la visualisation en 3 dimensions, utilisez la touche ECHAP.

## Notes
<a name="notes_ELTTEXTE000256"></a>


### Actions possibles depuis l'affichage en 3D d'une fenêtre
<a name="actions_possibles_depuis_affichage_3d_une_fenetre_ELTPARAGRAPHE000116"></a>En mode 3 dimensions, seules certaines actions sont disponibles sur les champs de la fenêtre. Il est possible par exemple de :

- sélectionner un champ,

- afficher la description du champ,

- afficher le code source du champ.



<a name="NOTE4_2"></a>


### Conditions nécessaire à l'utilisation de la vue en 3 dimensions d'une fenêtre
<a name="conditions_necessaire_utilisation_vue_3_dimensions_une_fenetre_ELTPARAGRAPHE000127"></a>Pour visionner vos fenêtres en 3 dimensions, il est nécessaire de posséder sur le poste en cours :

- DirectX version 8.0.A minimum.

- Une carte accélératrice 3D avec ses drivers à jour.

- Un mode d'affichage en 16 bits ou 32 bits.




Si vous utilisez WINDEV ou WINDEV Mobile en mode 64 bits, pour utiliser la visualisation en 3 dimensions, il est nécessaire de télécharger et d'installer le pack à l'adresse suivante : [https://www.microsoft.com/fr-fr/download/details.aspx?id=35](https://www.microsoft.com/fr-fr/download/details.aspx?id=35)

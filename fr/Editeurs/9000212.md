


## Recherche rapide dans le code en cours
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000114"></a>
Lors de l'écriture de code, il arrive souvent de vouloir effectuer rapidement une recherche dans le code en cours.
Cette recherche peut bien entendu être effectuée par la recherche générale de l'environnement (Ctrl F).
Cela oblige cependant à saisir la souris et à régler cette recherche. 
A partir de la version 28, un nouveau mode de recherche simple dans le code en cours fait son apparition.
Cette recherche s'effectue au clavier (Alt F3).
La chaîne recherchée est mise en évidence (surlignage) dans le code en temps réel lors de la frappe. Le nombre d'occurrences est affiché.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment faire une recherche rapide ?
<a name="comment_faire_une_recherche_rapide_ELTTEXTE000138"></a>
Pour effectuer une recherche rapide dans le code : 

1. Ouvrez un projet sous l'éditeur et affichez le code d'un élément. 

2. Utilisez la combinaison de touches Alt F3 pour afficher la fenêtre de recherche rapide : 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Recherche_rapide_code%20-%20HC%20N%B0001.gif)


3. Pour afficher la fenêtre en mode "Rechercher / Remplacer", utilisez la combinaison de touches Ctrl R lorsque la fenêtre de recherche a le focus. 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Recherche_rapide_code%20-%20HC%20N%B0002.gif)

	Remarque : Pour quitter le mode de remplacement vous pouvez cliquer sur la croix ou utiliser la touche Echap. 




**Fonctionnement de la fenêtre de recherche**

- Il n'est pas possible d'afficher plusieurs fenêtres de recherche rapide. 

- La fenêtre se ferme via la croix ou la touche Echap. 

- Il est possible de parcourir les résultats via les touches fléchées ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Recherche_rapide_code%20-%20HC%20N%B0002%201.gif)
 ou les raccourcis F3 ou Maj F3.  

- Si une sélection monoligne est en cours lors de l'appel à la fenêtre de recherche, la sélection est automatiquement prise en compte dans la fenêtre de recherche et la recherche est lancée. 

- La recherche est automatiquement réalisée lors de la saisie de l'utilisateur dans la fenêtre de recherche. 

- Les options de filtrage pour la recherche sont les suivantes : 

	- ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Recherche_rapide_code%20-%20HC%20N%B0002%205.gif)
 Respect de la casse et des accents. 

	- ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Recherche_rapide_code%20-%20HC%20N%B0002%206.gif)
 Recherche des mots complets. 

	- ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Recherche_rapide_code%20-%20HC%20N%B0002%207.gif)
 Recherche via les expressions régulières. 




- Pour effectuer un remplacement : 

	- Ctrl R affiche la fenêtre de remplacement. Le curseur apparaît automatiquement dans la zone de saisie du texte de remplacement. 

	- Alt R (![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Recherche_rapide_code%20-%20HC%20N%B0002%203.gif)
) remplace l'élément courant et sélectionne l'élément suivant. 

	- Entrée (![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Recherche_rapide_code%20-%20HC%20N%B0002%202.gif)
) remplace tous les éléments trouvés.




- Ctrl F (![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Recherche_rapide_code%20-%20HC%20N%B0002%204.gif)
) permet d'ouvrir le volet de recherche.

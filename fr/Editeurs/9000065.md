


## Configuration d'environnement
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000216"></a>
Une configuration d'environnement correspond à une sauvegarde de l'environnement de travail sous les éditeurs de WINDEV, WEBDEV et WINDEV Mobile : les différents volets, palettes, barres d'outils sont sauvegardés (ainsi que leur position). Cette configuration peut être restaurée à tout moment. Il est possible de conserver jusqu'à 4 configurations par utilisateur / développeur.

**Astuce** : Les configurations d'environnement sont vite indispensables lors de l'utilisation de plusieurs écrans. Vous pouvez positionner les différents éléments de l'interface de WINDEV, WEBDEV et WINDEV Mobile sur n'importe quel écran. La position des différents éléments sur les différents écrans est automatiquement sauvegardée et restaurée.

**Remarque** : Pour éviter une modification intempestive de l'environnement de travail, il est possible de verrouiller l'environnement. 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000240"></a>


### Sauvegarder la configuration en cours
<a name="sauvegarder_configuration_cours_ELTPARAGRAPHE000024"></a>A tout moment, vous pouvez sauvegarder la configuration de travail en cours dans une des quatre configurations disponibles :

1. Utilisez une des méthodes suivantes :

	- utilisez le raccourci Ctrl + Alt + N° de configuration (1 à 4).

	- sous le volet "Accueil", dans le groupe "Environnement", déroulez "Volets" et sélectionnez "Volets" puis "Sauvegarde d'environnement .. Sauver dans ".




2. Dans la fenêtre qui s'affiche, indiquez le nom de votre configuration. Cette configuration pourra être affichée à tout moment grâce au raccourci Ctrl + Numéro de la configuration (1 à 4)

3. Validez. Le nom de la configuration en cours apparaît.




**Remarque** : Les configurations d'environnement sont sauvegardées dans un fichier INI présent dans le sous-répertoire "\\Personnel" du répertoire d'installation du produit utilisé. Ce fichier est de la forme "ConfigVoletEnv_&lt;Login de l'utilisateur&gt;.INI" où &lt;Login de l'utilisateur&gt; est remplacé par le login de l'utilisateur du produit utilisé.
<a name="NOTE2_2"></a>


### Afficher une configuration existante
<a name="afficher_une_configuration_existante_ELTPARAGRAPHE000052"></a>A tout moment, vous pouvez afficher une configuration existante :

1. Utilisez une des méthodes suivantes :

	- utilisez le raccourci Ctrl + N° de configuration (1 à 4).

	- sous le volet "Accueil", dans le groupe "Environnement", déroulez "Volets" et sélectionnez "Volets" puis "Sauvegarde d'environnement .. &lt;Nom de la configuration&gt;".




2. La configuration d'environnement voulue est affichée.



<a name="NOTE2_3"></a>


### Exporter une configuration
<a name="exporter_une_configuration_ELTPARAGRAPHE000077"></a>Il est possible d'exporter les configurations d'environnement associées à un utilisateur. Les configurations pourront ensuite être importées simplement (par exemple par un autre utilisateur ou dans un autre produit).

Pour exporter les configurations :

1. Sous le volet "Accueil", dans le groupe "Environnement", déroulez "Volets" et sélectionnez "Volets" puis "Sauvegarde d'environnement .. Configuration ".

2. Cliquez sur le bouton "Exporter".

3. Saisissez l'emplacement et le nom du fichier de configuration.

4. Validez. Le fichier INI contenant toutes les configurations d'environnement actuelles est créé.



<a name="NOTE2_4"></a>


### Importer une configuration
<a name="importer_une_configuration_ELTPARAGRAPHE000103"></a>Importer une ou plusieurs configurations permet par exemple d'harmoniser les interfaces de travail de WINDEV, WEBDEV et WINDEV Mobile. Vous créez vos configuration dans un de ces trois produits, et vous pouvez les exporter puis les importer dans les deux autres produits : les barres et volets communs aux différents produits seront ainsi affichés de la même façon.

Pour importer une configuration :

1. Sous le volet "Accueil", dans le groupe "Environnement", déroulez "Volets" et sélectionnez "Volets" puis "Sauvegarde d'environnement .. Configuration ".

2. Cliquez sur le bouton "Importer".

3. Sélectionnez le fichier INI contenant les configurations à importer.

4. Validez. Dans la fenêtre qui s'affiche, pour chaque configuration à importer, indiquez sa position dans les configurations en cours.

5. Validez : les configurations sont automatiquement importées dans le produit en cours. Elles sont directement utilisables.



<a name="NOTE2_5"></a>


### Supprimer une configuration
<a name="supprimer_une_configuration_ELTPARAGRAPHE000130"></a>Pour supprimer une configuration :

1. Sous le volet "Accueil", dans le groupe "Environnement", déroulez "Volets" et sélectionnez "Volets" puis "Sauvegarde d'environnement .. Configuration ".

2. Sélectionnez la configuration à supprimer.

3. Cliquez sur le bouton "Supprimer".



<a name="NOTE2_6"></a>


### Verrouiller l'environnement
<a name="verrouiller_environnement_ELTPARAGRAPHE000155"></a>Pour verrouiller l'environnement, sous le volet "Accueil", dans le groupe "Environnement", déroulez "Volets" et sélectionnez "Volets" puis "Verrouiller l'environnement". Les volets et les barres d'outils ne peuvent plus être déplacés ou redimensionnés.

## Présentation
<a name="presentation_ELTTEXTE000151"></a>
Il est courant de faire une recherche dans le projet en cours : recherche d'un libellé, du nom d'un champ, ... Cette recherche peut être réalisée :

- soit par le volet "Explorateur de projet",

- soit par le [volet "Rechercher / Remplacer"](../Editeurs/2027013.md).




Mais il peut être également intéressant d'effectuer une recherche dans différents projets (par exemple pour chercher une procédure utilisée dans un autre projet). Cette recherche peut bien entendu concerner des projets de n'importe quel type (WINDEV, WEBDEV ou WINDEV Mobile).

La recherche multi-projets permet d'effectuer une recherche dans :

- des projets spécifiques,

- tous les projets présents dans un répertoire,

- tous les projets présents sur un disque ou un serveur.

## Comment le faire ?
<a name="comment_faire_ELTTEXTE000175"></a>


### Effectuer une recherche multi-projets
<a name="effectuer_une_recherche_multiprojets_ELTPARAGRAPHE000042"></a>Pour effectuer une recherche multi-projets :

1. Sélectionnez l'option "Atelier .. Recherche multi-projets...". La fenêtre de configuration des répertoires de recherche s'affiche.

2. Indiquez les répertoires, disques ou projets concernés par la recherche :

	- le bouton "Répertoire" permet de sélectionner les projets présents dans un répertoire ou sur un disque. Tous les projets WINDEV, WEBDEV ou WINDEV Mobile présents à cet emplacement seront pris en compte. La recherche est également effectuée dans les sous-répertoires des emplacements spécifiés. Il est possible de préciser un chemin UNC.

	- le bouton "Projet" permet d'ajouter un projet spécifique (WINDEV, WEBDEV ou WINDEV Mobile). Il est possible d'ajouter plusieurs projets de types différents (un projet WINDEV et un projet WINDEV Mobile par exemple).




3. Validez. Le volet "Rechercher - Remplacer" est automatiquement affiché sous l'éditeur et activé (si ce n'est déjà fait).

4. Saisissez dans le volet "Rechercher - Remplacer" :

	- le mot ou l'expression à rechercher,

	- les critères de recherche (contient, mot complet, ...).




5. Lancez la recherche (icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Ico_rech.gif)). Si des éléments sont trouvés, la référence de ces éléments apparaît dans le volet "Rechercher - Remplacer".




Remarque : Il est également possible d'effectuer la recherche multi-projets directement depuis le volet "Rechercher - Remplacer". Il suffit de sélectionner l'option "Dans &lt;Multi-projets&gt;" pour configurer les répertoires de recherche.


<a name="NOTE2_2"></a>


### Manipuler le résultat de la recherche
<a name="manipuler_resultat_recherche_ELTPARAGRAPHE000066"></a>Le résultat de la recherche multi-projets est affiché dans le volet "Rechercher - Remplacer". Les résultats sont regroupés par projet.

Pour chaque élément trouvé sont indiqués :

- sa localisation,

- sa valeur.




Le menu contextuel du résultat de la recherche permet :

- de copier la liste des éléments trouvés dans le presse-papiers.

- d'imprimer la liste des éléments trouvés.




Pour accéder directement à un des éléments trouvés, double-cliquez sur la localisation de l'élément.

**Remarque** : Si l'élément trouvé est présent dans un projet différent du projet en cours :

- si le projet a été réalisé avec le même produit, le projet est ouvert dans une nouvelle instance du produit.

- si le projet a été réalisé avec un produit différent, le projet est ouvert dans le produit associé.

### Critères de recherche
<a name="criteres_recherche_ELTPARAGRAPHE000095"></a>Quel que soit le type de recherche effectuée, il est possible de spécifier une combinaison des critères de recherche suivants :

- Contient : l'expression recherchée doit contenir l'expression saisie.

- Commence par : l'expression recherchée commence par l'expression saisie.

- Mot complet : l'expression recherchée correspond exactement à l'expression spécifiée.

- Ignorer la ponctuation : la recherche ne prend pas en compte la ponctuation (point, virgule, ...).

- Ignorer la casse : la recherche ne tient pas compte des majuscules et des minuscules.

- Ignorer l'accentuation : la recherche ne tient pas compte des accents.

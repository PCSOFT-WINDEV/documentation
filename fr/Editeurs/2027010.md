<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000088"></a>
Le volet "Commandes" liste :

- les commandes les plus usuelles sur le projet en cours (création, ouverture, sauvegarde, ...).

- les principales commandes disponibles sous l'éditeur en cours.

- les dernières commandes utilisées.

<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=VoletCommande.gif)

<a name="NOTE2_1"></a>


## Afficher le volet "Commandes"
<a name="afficher_volet_commandes_ELTTEXTE000112"></a>
**Pour afficher le volet "Commandes"**, sous le volet "Accueil", dans le groupe "Environnement", déroulez "Volets" et sélectionnez "Volets" puis "Commandes".

Pour plus de détails sur les manipulations des volets, consultez [Manipuler les volets ancrables](../Editeurs/2027001.md).




## Fabrique logicielle : Installation
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000142"></a>
Pour utiliser la fabrique logicielle, il est nécessaire d'installer sur un poste un coordinateur ainsi que les différents automates qui vont réaliser les différentes tâches présentes dans les plans d'action. 

3 modules sont nécessaires : 

- Le module d'administration. Ce module est accessible directement depuis WINDEV, WEBDEV ou WINDEV Mobile sur la machine de développement : sous le volet "Outils", dans le groupe "Utilitaires", cliquez sur "WDFAdmin".

- Le coordinateur. Le coordinateur doit être installé sur une seule machine. Le coordinateur stocke les plans d'action, répartit leur exécution et mémorise les comptes-rendus.

- Les automates. Les automates exécutent les plans d'action. Ils doivent être installés sur des machines dédiées au déroulement des plans d'action.




**Attention** :

- Sur chaque machine automate, les produits WINDEV, WEBDEV ou WINDEV Mobile nécessaires aux plans d'action définis doivent être installés et une clé électronique (dongle) doit être présente.

- Il est conseillé de créer sur les machines automates un compte Windows dédié au déroulement des plans d'action. Utilisez ce compte pour : 

	- lancer l'installation de l'automate.

	- accepter le lancement de l'automate au démarrage de Windows et le paramétrage de Windows en "auto logon".




- Il est conseillé de ne pas activer le mécanisme de l'UAC sur ces machines automates. 

- Une fabrique logicielle ne doit pas être installée dans une session TSE non interactive, car les tests sortiront systématiquement en erreur. La fabrique logicielle doit être lancée dans une session TSE ouverte en mode interactif.

## Installation
<a name="installation_ELTTEXTE000166"></a>


### Installer le coordinateur et les différents automates de la fabrique logicielle
<a name="installer_coordinateur_les_differents_automates_fabrique_logicielle_ELTPARAGRAPHE000046"></a>Pour installer le coordinateur et les différents automates de la fabrique logicielle : 

1. Lancez le programme d'installation WXxx0PACKFABRIQUE.exe. Ce programme d'installation est présent dans le sous-répertoire "\\Install\\Fabrique" du produit utilisé. L'assistant d'installation se lance. 

2. Passez au plan suivant. Vous pouvez : 

	- Installer le coordinateur sur le poste en cours. 

	- Installer un automate sur le poste en cours. 

	- Désinstaller le coordinateur ou un automate. 








### Installer le coordinateur
<a name="installer_coordinateur_ELTPARAGRAPHE000058"></a>Pour installer le coordinateur :

1. Spécifiez dans l'assistant le répertoire d'installation du coordinateur. Passez à l'étape suivante. 

2. Indiquez les paramètres concernant : 

	- **les Centres de Contrôle** : La base de données des Centres de Contrôle permet : 

		- d'identifier l'utilisateur du coordinateur (login et mot de passe de l'utilisateur)
						Sélectionnez le type de base de données des Centres de Contrôle utilisée. Il est possible de ne pas gérer l'authentification au coordinateur. 

		- d'envoyer les comptes-rendus dans la messagerie intégrée de WINDEV, WEBDEV et WINDEV Mobile (application WDBal). 
						Selon le type de base de données sélectionné, indiquez les paramètres nécessaires. 




- les logs. Indiquez le nombre de résultats conservés dans l'historique. 

- l'envoi des comptes-rendus. Les messages peuvent être envoyés : 

	- via la messagerie interne à WINDEV, WEBDEV et WINDEV Mobile. Il suffit d'indiquer le nom et le mot de passe de l'expéditeur. 

	- via un email. Il est nécessaire d'indiquer les spécificités du serveur SMTP et de l'expéditeur. 

3. L'installation du coordinateur est automatiquement effectuée. Il est ensuite possible d'installer un automate. 





### Installer un automate
<a name="installer_automate_ELTPARAGRAPHE000085"></a>Pour installer un automate :

1. Spécifiez dans l'assistant : 

	- le répertoire d'installation de l'automate. 

	- le nom de la machine sur laquelle est installé le coordinateur de la fabrique logicielle. 

	- le nom d'un intervenant des Centres de Contrôle (et son mot de passe). Ces informations sont optionnelles si vous n'utilisez pas les Centres de Contrôle. 




2. L'installation de l'automate est effectuée. 

3. Paramétrez si nécessaire le lancement automatique de l'automate.




## WDInst : configurer l'interface du programme d'installation 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000441"></a>
Par défaut, l'éditeur d'installation WDInst propose un programme d'installation standard : 

- aucun fond d'écran,

- aucune diapositive n'est affichée, 

- gabarit standard de l'installeur.  




Pour rendre l'installation de votre application plus conviviale, l'éditeur d'installation WDInst permet de :

- [Personnaliser le décor du programme d'installation](#NOTE2_1).

- [Personnaliser le gabarit de l'assistant du programme d'installation](#NOTE3_1). 

- [Faire défiler à intervalles réguliers des images (appelées "diapositives") lors de l'installation des fichiers sur le poste d'installation](#NOTE4_1).

- [Utiliser un programme d'installation personnalisé (personnalisation de WDSetup)](../Editeurs/2028106.md).




**Remarque** : La personnalisation du décor, du gabarit et la sélection des diapositives peuvent être différentes selon la langue d'exécution du programme d'installation.

Le paramétrage de l'interface du programme d'installation est réalisé sous l'éditeur d'installation WDInst : sous le volet "Outils", dans le groupe "Utilitaires", cliquez sur "WDInst".

L'assistant de création du programme d'installation permet également de personnaliser l'interface et le gabarit du programme d'installation.

## Personnaliser l'interface du programme d'installation
<a name="personnaliser_interface_programme_installation_ELTTEXTE000465"></a>
Pour rendre l'installation de votre application plus conviviale, l'éditeur d'installation WDInst permet de personnaliser l'interface du programme d'installation.

Pour personnaliser l'interface du programme d'installation : 

1. Sélectionnez l'option "Paramètres de l'installation .. Décor".

2. Sélectionnez la langue pour laquelle l'interface doit être personnalisée.

3. Configurez l'interface du programme d'installation.
	Remarque : Pour afficher une image de fond, cochez l'option "Afficher la fenêtre de fond de l'installation".

## Personnaliser le gabarit de l'assistant
<a name="personnaliser_gabarit_assistant_ELTTEXTE000489"></a>
L'éditeur d'installation WDInst offre la possibilité de choisir le look des fenêtres de l'installation. 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Installation_WD_defaut%20-%20HC%20N%B0001%201.gif)


Pour personnaliser le gabarit de l'assistant : 

1. Sélectionnez l'option "Paramètres de l'installation .. Assistant".

2. cliquez sur "Gabarit de l'assistant". 

3. Sélectionnez l'image de décor. Il est possible d'utiliser : 




- aucune image (cas par défaut),

- une image spécifique ayant pour dimensions 708 x 460 pixels.

## Faire défiler à intervalles réguliers des diapositives
<a name="faire_defiler_intervalles_reguliers_des_diapositives_ELTTEXTE000513"></a>
L'éditeur d'installation WDInst offre la possibilité d'afficher des images (appelées diapositives) lors de l'installation des fichiers sur les postes utilisateurs.

**Remarques** :

- Si un logo est affiché dans le fond d'écran, les diapositives remplaceront ce logo.

- Plus la taille des fichiers "diapositive" sera petite, plus l'installation sera rapide. Pour diminuer la taille des diapositives, il est conseillé d'utiliser des formats d'images compressés (type ".GIF" ou ".JPEG" par exemple).



<a name="NOTE4_2"></a>


### Faire défiler des diapositives
<a name="faire_defiler_des_diapositives_ELTPARAGRAPHE000299"></a>Pour faire défiler à intervalles réguliers des diapositives : 

1. Sélectionnez l'option "Paramètres de l'installation .. Diapositives".

2. Sélectionnez la langue pour laquelle les diapositives doivent être paramétrées.

3. Sélectionnez les diapositives (fichiers image) à afficher (bouton "Ajouter").

4. Spécifiez la durée (en secondes) d'affichage minimale de chaque diapositive. 
	Pour optimiser la vitesse d'installation, le changement des diapositives est réalisé entre deux opérations d'installation (installation de deux fichiers différents par exemple). La durée d'affichage peut être plus importante que la durée spécifiée.

5. Modifiez si nécessaire l'ordre d'affichage des diapositives (boutons "Monter" et "Descendre").

6. Cochez l'option "Afficher les diapositives en boucle" pour afficher les diapositives tout au long de l'installation. 
	Si cette option est décochée et si à la dernière diapositive, l'installation n'est pas terminée, cette dernière diapositive restera affichée jusqu'à la fin de l'installation.

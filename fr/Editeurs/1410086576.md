
## Développement d'applications en mode Universal Windows 10 App
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000143"></a>
WINDEV et WINDEV Mobile permettent le développement d'applications pour tablettes Windows 10, téléphones Windows Phone 10 et Windows 10 (applications Universal Windows 10 App). 

Le développement d'applications de type Universal Windows 10 App à l'aide de WINDEV (ou WINDEV Mobile) se fait de façon similaire au développement pour Windows à quelques spécificités près. Il est également possible de tester les applications directement sur le poste de développement sous Windows 10.

**Remarque** :
 
Pour générer et tester des applications pour Windows Store apps, il est nécessaire d'installer le SDK Windows 10. Pour plus de détails, consultez [Applications Universal Windows 10 App : Configuration nécessaire](../Editeurs/1410086574.md). 
**ATTENTION** : Le SDK Windows 10 peut être installé uniquement sur Windows 8 et versions supérieures.

## Créer un projet pour Universal Windows 10 App avec WINDEV ou WINDEV Mobile
<a name="creer_projet_pour_universal_windows_10_app_avec_windev_windev_mobile_ELTTEXTE000167"></a>
Pour créer un projet Universal Windows 10 App avec WINDEV ou WINDEV Mobile : 

1. Cliquez sur ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_nouveau.gif) parmi les boutons d'accès rapide. 

2. La fenêtre de création d'un nouvel élément s'affiche : cliquez sur "Projet". 

3. L'assistant de création de projet se lance. Choisissez le nom et l'emplacement de votre projet. Vous pouvez saisir une brève description du projet. Passez à l'étape suivante. 

4. Ajoutez si nécessaire des documents de conception au projet. Passez à l'étape suivante. 

5. Dans l'étape "Description - Type de projet", sélectionnez "Universal Windows 10 App". Passez à l'étape suivante. 

6. Indiquez où trouver le SDK Windows 10, nécessaire à la génération d'applications. Si le SDK a été installé, le chemin d'installation est automatiquement proposé (par exemple "C:\\Program Files (x86)\\Windows Kits\\10\\bin"). 
	Pour plus de détails sur le téléchargement et l'installation du SDK, consultez la page [Installation du SDK Windows 10](../Editeurs/1410086574.md).

7. Passez à l'étape suivante. 

8. Sélectionnez le type d'appareils sur lequel l'application sera déployée (téléphone, tablette, PC). 

9. La suite de l'assistant se déroule de la même façon que pour n'importe quelle autre plateforme. Pour plus de détails, consultez [Créer un projet WINDEV Mobile](../Editeurs/2030043.md).

### Analyse d'un projet Universal Windows 10 App
<a name="analyse_projet_universal_windows_10_app_ELTPARAGRAPHE000060"></a>Un projet Universal Windows 10 App peut être associé à une analyse. Les fichiers de données pourront être au format HFSQL Classic et HFSQL Client/Serveur. 







## Tester un projet Universal Windows 10 App
<a name="tester_projet_universal_windows_10_app_ELTTEXTE000203"></a>
Plusieurs modes de test sont disponibles pour les applications Universal Windows 10 App :

- Le Go du projet dans le simulateur de WINDEV ou WINDEV Mobile. 

- L'exécution du projet sous Windows 10. 

<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=W8_test.gif)



### Go du projet directement dans le simulateur
<a name="projet_directement_dans_simulateur_ELTPARAGRAPHE000077"></a>Ce test lance une fenêtre de test du produit dans laquelle est exécutée l'application Universal Windows 10 App. Ce test permet d'utiliser le débogueur. Cependant, de nombreuses différences peuvent être observées avec une exécution réelle. En effet, la simulation utilise le PC et non le périphérique Windows 10 en mode Tuile. 






### Tester l'application en réel : exécution du projet en mode tuile
<a name="tester_application_reel_execution_projet_mode_tuile_ELTPARAGRAPHE000084"></a>L'option "Tester l'application en réel" permet d'exécuter votre projet en mode tuile, directement depuis Windows 10. Il est nécessaire de générer l'application Universal Windows 10 App. 

- Si l'application n'a pas été générée, WINDEV ou WINDEV Mobile propose de générer l'application. 

- Si l'application a déjà été générée, elle est automatiquement générée avec les derniers paramètres de génération spécifiés. 


Pour plus de détails, consultez [Générer une application Universal Windows 10 App](../Editeurs/1410086577.md).

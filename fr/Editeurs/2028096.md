


## WBSetup : Gérer des langues supplémentaires 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000117"></a>
Le projet "WBSetup" correspond au programme d'installation standard des sites WEBDEV. Ce projet est un projet WINDEV.

Par défaut, ce projet gère le français et l'américain.

Remarque : Vous pouvez ajouter dans ce projet toutes les fonctionnalités qui vous semblent importantes dans l'installation d'un site WEBDEV.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Gérer des langues supplémentaires dans WBSetup
<a name="gerer_des_langues_supplementaires_dans_wbsetup_ELTTEXTE000141"></a>
Pour gérer des langues supplémentaires dans WBSetup :

1. Ouvrez ce projet sous WINDEV : cliquez sur ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_ouvrir.gif) parmi les boutons d'accès rapide et sélectionnez "Ouvrir un projet". 
	Par défaut, ce projet est présent dans le répertoire "Exemples\\WBSetup" du répertoire d'installation de WINDEV.

2. Fermez toutes les fenêtres ouvertes sous l'éditeur de fenêtres.

3. Ajoutez les langues supplémentaires à gérer dans la description du projet : 

	- sous le volet "Projet", dans le groupe "Projet", cliquez sur "Description".

	- dans l'onglet "Langues", ajoutez les langues nécessaires.




4. Traduisez l'interface du projet (fenêtres, champs, ...) ainsi que les messages de code. 
	**Rappel** : WDMSG permet de faciliter la traduction d'un projet en extrayant l'ensemble des messages d'un projet (libellé des champs, titre des fenêtres, ...) et en réintégrant les messages après traduction. WDMSG est un module complémentaire de WINDEV. Pour plus d'informations sur WDMSG, contactez le service commercial de PC SOFT.

5. Créez un nouvel exécutable : cliquez sur ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_generer.gif) parmi les boutons d'accès rapide et sélectionnez "Générer l'exécutable Windows 32 bits". Dans l'assistant, pensez à intégrer les nouvelles langues dans l'exécutable. 
	**Remarque** : Vous pouvez modifier le nom de l'exécutable.




**Attention** : Nous vous conseillons de renommer le répertoire du projet WBSetup personnalisé. Ainsi, lors de l'installation d'une mise à jour de WEBDEV, le projet WBSetup personnalisé ne sera pas écrasé et vous conserverez toutes vos modifications.

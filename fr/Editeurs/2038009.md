


## Ajout d'un élément dans la base de sources du GDS
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000126"></a>
L'ajout d'un élément dans la base de sources du GDS (gestionnaire de sources) permet d'ajouter l'élément dans le projet de référence sur le Serveur GDS. L'élément sera ainsi mis à disposition de tous les utilisateurs du GDS lors de leur prochaine synchronisation.

**Remarque** : Par défaut, lors de l'ajout d'un élément dans le projet local, le gestionnaire de sources propose automatiquement d'ajouter l'élément dans la base de sources du GDS lors de l'enregistrement de l'élément.
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ajout_element_base%20-%20HC%20N%B0001.gif&type=thumb)

## Ajout d'un élément
<a name="ajout_element_ELTTEXTE000150"></a>
Pour ajouter un élément spécifique dans la base de sources du GDS :

1. Sélectionnez l'élément dans le volet "Explorateur de projet".

2. Utilisez l'option "Ajouter au gestionnaire de sources" du menu contextuel.
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ajout_element_base%20-%20HC%20N%B0002.gif)





Pour ajouter un élément dans le GDS, vous pouvez également sous le volet "GDS", dans le groupe "Elément en cours", cliquer sur "Ajouter".

Lors de l'ajout d'un élément dans le GDS, l'écran suivant est affiché :

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ajout_element_base%20-%20HC%20N%B0003.gif&type=thumb)
Vous pouvez :

- Dans l'onglet "Tâches", indiquer que le nouvel élément correspond à l'exécution d'une tâche ou à la réalisation d'une tâche.

- Dans l'onglet "Incidents", indiquer que le nouvel élément corrige un incident.

- Dans l'onglet "Commentaire", saisir un commentaire sur le nouvel élément. Par exemple, vous pouvez indiquer que cette nouvelle fenêtre ou page est destinée à saisir les logins des utilisateurs.




Les différentes options disponibles sont les suivantes : 

- **Extraire les éléments** : Cette option permet de mettre l'élément à disposition des autres utilisateurs, tout en le conservant extrait pour modification.

- **Interdire l'extraction multiple** : Cette option permet d'autoriser uniquement l'extraction de l'élément par un seul utilisateur en même temps. L'élément ne pourra pas être modifié en même temps par plusieurs utilisateurs.

- **Ne pas conserver d'historique** : Si cette option est cochée, aucun historique des différentes intégrations / extractions ne sera conservé pour cet élément.
	Remarque : Il est possible d'activer a posteriori la conservation de l'historique pour l'élément via les propriétés de l'élément. Pour afficher les propriétés de l'élément en cours, sous le volet "GDS", dans le groupe "Elément en cours", cliquez sur "Propriétés".

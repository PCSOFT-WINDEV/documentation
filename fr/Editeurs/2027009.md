


## Le volet "Code"
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000086"></a>
Le volet "Code" permet de manipuler les procédures (locales et globales) et les classes du projet en cours (options "Procédures globales", "Procédures locales" et "Classes"). Toutes les options permettant de manipuler les procédures et les classes sont disponibles dans le menu contextuel (clic droit).

**Ce volet est conservé par compatibilité avec les versions précédentes**. Les fonctionnalités de ce volet sont intégrées au [volet "Explorateur de projet".](../Editeurs/2027029.md)

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Afficher le volet "Code"
<a name="afficher_volet_code_ELTTEXTE000110"></a>
**Pour afficher le volet "Code"** : sous le volet "Accueil", dans le groupe "Environnement", déroulez "Volets" et sélectionnez "Volets" puis "Code".

Pour plus de détails sur les manipulations des volets, consultez [Manipuler les volets ancrables](../Editeurs/2027001.md).

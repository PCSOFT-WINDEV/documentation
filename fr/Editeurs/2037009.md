## Configuration du serveur pour le protocole CGI (AWP)
<a name="configuration_serveur_pour_protocole_cgi_awp_ELTTEXTE000095"></a>
Pour configurer le serveur Web : 

1. Editez le fichier "srm.cnf" sous un éditeur de texte. Ce fichier est présent dans le sous-répertoire "/Conf" du répertoire d'installation du serveur FrontPage.

2. Ajoutez dans le fichier "srm.cnf" la définition de l'alias du protocole CGI (WD280AWP.EXE) par la ligne suivante :





```txt
WinScriptAlias /<Alias>/ "Répertoire du moteur"
```


Par exemple, si le protocole CGI est situé dans le répertoire "C:\\WEBDEV\\PROGRAMMES\\MOTEUR\\WINDOWS\\AWP" et si le nom de l'alias est WD280AWP :


```txt
WinScriptAlias /WD220AWP/ C:/WEBDEV/PROGRAMMES/MOTEUR/WINDOWS/AWP/
```


où "C:/WEBDEV/PROGRAMMES/MOTEUR/WINDOWS/AWP" : est le répertoire d'installation du moteur WEBDEV (WD280AWP.EXE)

**Remarques** : 

- les caractères '\\' deviennent des '/'

- ne pas oublier le '/' final dans le chemin.

## Configuration du serveur pour les sites WEBDEV
<a name="configuration_serveur_pour_les_sites_webdev_ELTTEXTE000119"></a>
Pour configurer le serveur Web, ajoutez dans le fichier "srm.cnf" la définition de l'alias des images de chaque site WEBDEV :


```txt
Alias /<Nom alias Image>/ "Répertoire des images du site"
```


Par exemple pour le site WEBDEMO.


```txt
Alias /WEBDEMO_WEB/ "C:/WEBDEMO/WEBDEMO_WEB/"
```

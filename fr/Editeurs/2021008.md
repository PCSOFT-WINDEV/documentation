


## Perso-Notes
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000103"></a>
Les Perso-Notes vous permettent de saisir les idées à retenir, les remarques à ne pas oublier, ...Vous saisissez ainsi autant de pense-bêtes que vous voulez, dans n'importe quel élément de votre projet. Vous pouvez même coller vos Perso-Notes sur le bureau de Windows.

Une Perso-Note peut contenir du texte (libre ou formaté RTF), des images, du son, ...

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000127"></a>


### Ajouter une Perso-Note
<a name="ajouter_une_personote_ELTPARAGRAPHE000020"></a>Pour ajouter une Perso-Note :

1. Sélectionnez l'option "Nouvelle Perso-Note" du menu contextuel (clic droit).

2. L'assistant de création d'une nouvelle Perso-Note se lance. Sélectionnez le type des informations contenues dans la nouvelle Perso-Note :

	- texte simple.

	- texte avec mise en forme.

	- document scanné.

	- fichier texte, feuille Excel, image, ...

	- enregistrement vocale.




3. Selon le type de la Perso-Note, saisissez le contenu de la Perso-Note, sélectionnez le fichier attendu ou enregistrez le mémo vocal.

4. Donnez un titre à cette nouvelle Perso-Note et validez. <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Perso_notes%20-%20HC%20N%B0001.gif&type=thumb)




<a name="NOTE2_2"></a>


### Associer une Perso-Note à un élément du projet
<a name="associer_une_personote_element_projet_ELTPARAGRAPHE000039"></a>Pour associer une Perso-Note à un élément du projet :

1. Cliquez sur l'icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Perso_notes%20-%20HC%20N%B0003.GIF) (punaise attachée) dans la barre de titre de la Perso-Note. L'icône se transforme en punaise détachée : ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Perso_notes%20-%20HC%20N%B0002.GIF).

2. Ouvrez l'élément du projet auquel la Perso-Note doit être associée.

3. Cliquez une nouvelle fois sur la punaise pour associer la Perso-Note à l'élément en cours.



<a name="NOTE2_3"></a>


### Supprimer une Perso-Note
<a name="supprimer_une_personote_ELTPARAGRAPHE000054"></a>Pour supprimer une Perso-Note, cliquez simplement sur le "X" de la Perso-Note (en haut à droite) et validez la boîte de dialogue qui s'affiche.

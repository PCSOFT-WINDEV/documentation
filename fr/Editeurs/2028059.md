
## Gérer les utilisateurs d'une application installée en réseau 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000102"></a>
La gestion des utilisateurs permet de connaître à partir du poste serveur :

- les utilisateurs connectés actuellement à l'application,

- pour chaque utilisateur connecté :

	- le poste de lancement de l'application,

	- la version de l'application utilisée.









<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Caractéristiques des utilisateurs
<a name="caracteristiques_des_utilisateurs_ELTTEXTE000126"></a>
Pour visualiser les caractéristiques des utilisateurs d'une application sur le poste serveur :

1. Lancez l'application WDADMINEXE présent dans le sous-répertoire INSTALL du répertoire d'installation sur le poste serveur.

2. Affichez l'onglet "Utilisateurs".

**Remarque** : Les données affichées dans cet onglet sont contenues dans les fichiers LOKxxx.tmp. Ces fichiers sont présents dans le sous-répertoire LOCK du répertoire INSTALL sur le poste serveur. Chaque fichier correspond à un utilisateur connecté à l'application. Ces fichiers sont supprimés dès que la connexion est terminée.

Pour plus de détails sur l'application WDADMINEXE, consultez le [Contrôle à distance des exécutables WINDEV](../WDAdminExe/2028052.md).

## Notes
<a name="notes_ELTTEXTE000185"></a>


### Si vous choisissez de ne pas intégrer la gestion des utilisateurs
<a name="vous_choisissez_pas_integrer_gestion_des_utilisateurs_ELTPARAGRAPHE000045"></a>Si la gestion des utilisateurs n'est pas intégrée dans le programme d'installation, l'application WDADMINEXE sera tout de même installée sur le serveur. Cependant :

- les fichiers LOKxxx.tmp ne seront pas créés,

- l'onglet "Utilisateurs" sera grisé,

- le contrôle à distance de l'application sera disponible. Le contrôle à distance permet à tout moment de déconnecter les utilisateurs. Pour plus de détails sur le contrôle à distance de l'application, consultez le [Contrôle à distance des exécutables WINDEV](../WDAdminExe/2028052.md).

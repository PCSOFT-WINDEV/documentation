
## Programmation par action prédéfinie
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000158"></a>
La programmation par action prédéfinie permet de décrire le code correspondant à un événement d'un seul clic, sans avoir à saisir des lignes de code, simplement en sélectionnant l'action à exécuter dans une liste.

Par exemple, sous WINDEV :


![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ActionPredefListe.GIF&type=thumb)
*Exécution du code de clic du champ Bouton et ouverture de la fenêtre*

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ActionPredefAutre.GIF&type=thumb)
*Initialisation d'un composant, les paramètres sont indiqués dans l'interface*

Les actions prédéfinies sont proposées :

- sur les champs Bouton (dans l'événement "Clic").

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) sur les options de menu.

<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000182"></a>


### Définir une action prédéfinie
<a name="definir_une_action_predefinie_ELTPARAGRAPHE000041"></a>Pour définir une action prédéfinie :

1. Affichez la fenêtre de description du champ (option "Description" du menu contextuel).

2. Dans l'onglet "Général", définissez l'action à effectuer : 

	1. ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Sélectionnez l'action à exécuter dans la combo "Action". 
			Toutes les actions les plus courantes sont proposées dans cette liste. Pour sélectionner une action plus spécifique, cliquez sur le bouton "Toutes les actions" ou "Autres actions".





Les différentes actions proposées dépendent des éléments présents dans le projet en cours (fenêtres, pages, composant, procédure, ...).

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) **Remarque** : Pour choisir l'action prédéfinie, il est également possible de sélectionner l'option "Action prédéfinie..." du menu contextuel du champ.

## Note
<a name="note_ELTTEXTE000206"></a>


### Information affichée sous l'éditeur de code
<a name="information_affichee_sous_editeur_code_ELTPARAGRAPHE000070"></a>Lorsqu'une action prédéfinie est utilisée, une information indiquant cette action prédéfinie est spécifiée sous l'éditeur de code.

Par exemple :


![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ActionPredefInfo.GIF)


Le clic sur l'action prédéfinie affichée dans l'éditeur de code affiche : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) la fenêtre de sélection des actions prédéfinies.


Il est ainsi possible de modifier l'action prédéfinie directement depuis l'éditeur de code.

**Attention** : Tout code WLangage saisi sous l'éditeur de code en dessous de l'appel à l'action prédéfinie ne sera pas exécuté.

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Pour inclure une action prédéfinie dans du code WLangage, utilisez la fonction [ExécuteActionPrédéfinie](../WDLang1/3014028.md).

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png)

<a name="executer_une_action_predefinie_par_programmation_ELTPARAGRAPHE000106"></a>WINDEV propose également d'exécuter une action prédéfinie en utilisant la fonction [ExécuteActionPrédéfinie](../WDLang1/3014028.md).

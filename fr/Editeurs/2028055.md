


## Générer le programme d'installation 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000080"></a>
La génération du programme d'installation permet de créer le programme d'installation selon les caractéristiques spécifiées dans l'assistant.

A la fin de la génération, une fenêtre s'affiche permettant de : 

- Tester immédiatement le programme d'installation (bouton "Tester"). 

- Lancer un explorateur Windows sur le répertoire contenant le programme d'installation (bouton "Explorer"). 

- Valider la fenêtre de génération (bouton "Fermer"). 




Il est possible de continuer la personnalisation de l'installation sous WDInst : sous le volet "Outils", dans le groupe "Utilitaires", cliquez sur "WDInst". La configuration du programme d'installation spécifiée dans l'assistant sera automatiquement affichée. La configuration du programme d'installation est enregistrée dans le fichier "&lt;NomApplication.WDF&gt;" présent dans le répertoire de l'application.

**Remarque** : La description du programme d'installation spécifiée dans l'assistant est automatiquement sauvegardée. A la prochaine utilisation de l'assistant, les options de configuration précédemment spécifiées seront automatiquement reportées. La configuration du programme d'installation est enregistrée dans le fichier "&lt;NomApplication.WDF&gt;" présent dans le répertoire de l'application.

## Présentation
<a name="presentation_ELTTEXTE000228"></a>
L'utilisateur final peut créer des notes repositionnables dans toutes les fenêtres de ses applications WINDEV. C'est une Fonctionnalité Automatique de vos Applications disponible par défaut. <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=FAA_Note_Repostionnable%20-%20HC%20N%B0001.gif&type=thumb)

## Notes repositionnables : Fonctionnalités disponibles
<a name="notes_repositionnables_fonctionnalites_disponibles_ELTTEXTE000252"></a>


### Création d'une note repositionnable
<a name="creation_une_note_repositionnable_ELTPARAGRAPHE000019"></a>Pour créer une note repositionnable : 

1. Affichez le menu contextuel de la fenêtre (clic droit). 

2. Sélectionnez l'option "Notes repositionnables .. Créer une note repositionnable". <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=FAA_Note_Repostionnable%20-%20HC%20N%B0003.gif)


3. La note repositionnable apparaît dans la fenêtre en cours. Vous pouvez la positionner à votre convenance dans la fenêtre et saisir le texte de la note.
	Remarque : Il est possible de coller du texte dans la note (Ctrl + V). 




Par défaut, la note repositionnable créée a les caractéristiques suivantes : 

- Texte de la note de couleur rouge. 

- Taille du texte de la note moyen. 

- Couleur de fond de la note de couleur jaune. 

- Affichage du jour de création dans la note (Aujourd'hui). Cette date s'adaptera dans le temps (par exemple : il y a 3 jours, ...). 

- Réduction automatique : A chaque affichage de la fenêtre, la note est affichée 10 secondes puis elle est automatiquement réduite (en haut à droite de la fenêtre). 





### Configuration de la note repositionnable
<a name="configuration_note_repositionnable_ELTPARAGRAPHE000041"></a>Le menu contextuel de la note repositionnable permet de configurer cette note.<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=FAA_Note_Repostionnable%20-%20HC%20N%B0002.gif)
Les options disponibles sont les suivantes : 

- **Modifier le texte de la note** : Le curseur de saisie est alors affiché à la fin du texte de la note pour modification.
	Remarque : Pour modifier le texte de la note, il est également possible de cliquer directement dans la note affichée. 

- **Réduire** : Cette option permet de réduire la note. La note est alors positionnée sous forme de signet en haut à droite de la fenêtre.
	<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=FAA_Note_Repostionnable1%20-%20HC%20N%B0001.gif&type=thumb)
Le clic sur l'icône carré permet de restaurer la note à sa dimension initiale. 

- **Réduire automatiquement** : Si cette option est sélectionnée, la note sera affichée à l'ouverture de la fenêtre et sera réduite automatiquement après 5 secondes (si elle n'a pas été modifiée). 

- **Couleur de fond** : Cette option permet de personnaliser la couleur de fond de la note. 

- **Couleur du texte** : Cette option permet de personnaliser la couleur du texte de la note. 

- **Taille du texte** : Cette option permet de personnaliser la taille du texte de la note. 

- **Supprimer la note** : Cette option permet de supprimer la note. Dans ce cas, la note ne sera plus affichée. Il est possible d'annuler la suppression grâce à la combinaison de touches Ctrl + Z. 

- **Exporter les notes de cette fenêtre** : Cette option permet de créer un fichier "wdnote" dans le répertoire spécifié. Par défaut, le nom du fichier proposé correspond au nom de la fenêtre. Cette option permet par exemple de sauvegarder les notes associées à la fenêtre. Les notes exportées ne sont pas supprimées de la fenêtre. 

- **Exporter toutes les notes de l'application** : Cette option permet de créer un fichier "wdnote" dans le répertoire spécifié. Par défaut, le nom du fichier proposé correspond au nom de l'application. Cette option permet par exemple de sauvegarder les notes de l'application. Les notes exportées ne sont pas supprimées de l'application. 

- **Importer des notes** : Cette option permet d'importer un fichier "wdnote" dans l'application. Ce fichier peut aussi bien contenir les notes d'une fenêtre que celle de l'application. Les notes existantes à la fois dans l'application et dans le fichier ne seront pas modifiées, les notes existantes uniquement dans le fichier seront ajoutées dans l'application.

## Mise en place et configuration
<a name="mise_place_configuration_ELTTEXTE000282"></a>


### Activation des notes repositionnables
<a name="activation_des_notes_repositionnables_ELTPARAGRAPHE000070"></a>Par défaut, les notes repositionnables sont disponibles pour les utilisateurs d'applications compilées avec la version 21 et supérieure pour toutes les fenêtres utilisant le menu contextuel des FAA (Fonctionnalités Automatiques de vos Applications). Cette fonctionnalité est liée à la modification de l'UI par l'utilisateur final. 

**Pour activer les notes repositionnables au niveau du projet** : 

1. Affichez la fenêtre de description du projet : sous le volet "Projet", dans le groupe "Projet", cliquez sur "Description".

2. Vérifiez que la modification de l'UI par l'utilisateur final est activée et que les notes sont disponibles : 

	1. Dans l'onglet "Avancé", cliquez si nécessaire sur "Modification de l'UI par l'utilisateur final" ou cliquez sur "Options" pour voir les options. 

	2. Dans la fenêtre qui s'affiche : 

		- vérifiez que les options par défaut sont utilisées, 

		- si des options spécifiques sont définies, les options de gestion de la note repositionnable doivent être cochées. 




3. Vérifiez que le menu contextuel du projet "Notes repositionnables" est activé au niveau du projet : 

	1. Dans l'onglet "Avancé", cliquez sur le bouton "Menu des FAA". 

	2. Dans la fenêtre qui s'ouvre, sélectionnez l'élément "Fenêtre" : l'option "Notes repositionnables" doit être activée. 







**Pour activer les notes repositionnables au niveau d'une fenêtre** : 

1. Affichez la fenêtre de description de la fenêtre (option "Description" du menu contextuel). 

2. Vérifiez que la modification de l'UI par l'utilisateur final est activée et que les notes sont disponibles : 

	1. Dans l'onglet "Détail", cliquez si nécessaire sur "Modification de l'UI par l'utilisateur final" ou cliquez sur "Options" pour voir les options. 

	2. Dans la fenêtre qui s'affiche : 

		- vérifiez que les options par défaut sont utilisées, 

		- si des options spécifiques sont définies, les options de gestion de la note repositionnable doivent être cochées. 




3. Vérifiez que le menu contextuel du projet "Notes repositionnables" est activé au niveau de la fenêtre : 

	1. Dans l'onglet "UI", déroulez l'option "Menu contextuel". Cochez l'option "Afficher le menu des FAA" puis cliquez sur "Paramètres".

	2. Dans la fenêtre qui s'ouvre, l'option "Notes repositionnables" doit être activée. 








### Désactivation des notes repositionnables
<a name="desactivation_des_notes_repositionnables_ELTPARAGRAPHE000134"></a>Il est possible de désactiver la possibilité d'utiliser des notes repositionnables : 

- au niveau du projet, en désactivant les notes repositionnables dans le menu des FAA : 

	1. Affichez la fenêtre de description du projet : sous le volet "Projet", dans le groupe "Projet", cliquez sur "Description".

	2. Dans l'onglet "Avancé", cliquez sur le bouton "Menu des FAA". 

	3. Dans la fenêtre qui s'ouvre, sélectionnez l'élément "Fenêtre" et désactivez l'option "Notes repositionnables". 




- au niveau d'une fenêtre, en désactivant les notes repositionnables dans le menu des FAA : 

	1. Affichez la fenêtre de description de la fenêtre. 

	2. Dans l'onglet "UI", affichez les options du menu contextuel et cliquez sur le bouton "Paramètres". 

	3. Dans la fenêtre qui s'ouvre, désactivez l'option "Notes repositionnables". 







Remarque : Il est également possible de désactiver les options concernant les notes repositionnables dans la fenêtre de configuration de la modification de l'UI par l'utilisateur final, aussi bien pour le projet que pour les fenêtres. 


### Activation et désactivation par programmation
<a name="activation_desactivation_par_programmation_ELTPARAGRAPHE000168"></a>Pour désactiver les notes repositionnables, utilisez la fonction [FAADésactive](../WDLang1/1000022018.md) avec la constante *faaNoteRepositionnable*. 

Pour désactiver ou activer les notes repositionnables par programmation sur une fenêtre, il est également possible d'utiliser la propriété [FenEditionMode](../Proprietes/1000021824.md) avec la constante *feAjouteNoteRepositionnable* :

```wl
MaFenêtre.FenEditionMode = feAjouteNoteRepositionnable
```
Cette propriété permet de connaître et de modifier toutes les actions autorisées en mode "Modification de l'UI par l'utilisateur final".

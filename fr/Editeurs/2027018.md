<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000095"></a>
Le volet "Centre de Contrôle Qualité" affiche les différents incidents associés au projet sélectionné.<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=VoletCCQualite.gif)
Le menu contextuel (clic droit) du volet "Centre de Contrôle Qualité" permet de :

- Ajouter un incident.

- Modifier un incident.

- Corriger un incident.

- Réattribuer un incident.

- Lier une tâche à l'incident.




Pour plus de détails, consultez [Centre de Suivi de Projets : Gestion des incidents](../CCSuivi/3540723.md).

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Afficher le volet "Centre de Contrôle Qualité"
<a name="afficher_volet_centre_controle_qualite_ELTTEXTE000119"></a>
**Pour afficher le volet "Centre de Contrôle Qualité"**, sous le volet "Accueil", dans le groupe "Environnement", déroulez "Volets" et sélectionnez "Volets" puis "Centre de Contrôle Qualité".

Pour plus de détails sur les manipulations des volets, consultez [Manipuler les volets ancrables](../Editeurs/2027001.md).

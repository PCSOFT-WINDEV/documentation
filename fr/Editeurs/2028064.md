


## Etape 6 : Paramétrage du groupware utilisateur
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000082"></a>
Si le site est configuré pour utiliser le groupware utilisateur, les paramètres de la base de données du groupware peuvent être configurés lors de l'installation.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Paramétrage du site
<a name="parametrage_site_ELTTEXTE000106"></a><br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=wwinst_param_gpu_new.gif)
Si le site utilise le groupware utilisateur : 

- version 18 et supérieur : indiquez uniquement le répertoire de localisation du fichier des droits.

- version compatible : indiquez le répertoire de localisation du fichier des droits et des utilisateurs.




Ces fichiers peuvent, au choix, être placés dans le répertoire des données du site ou dans un répertoire spécifié.
<a name="NOTE3_2"></a>


### Remarque
<a name="remarque_ELTPARAGRAPHE000029"></a>Les chemins de données indiqués peuvent être relatifs ou absolus.

- Si des chemins relatifs sont utilisés, ils seront considérés comme étant relatifs au répertoire des données du site.

- Dans le cas de chemins absolus, il s'agit de chemins sur le serveur de déploiement.




## Champs Combo et Liste : Fonctionnalités automatiques (FAA)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000240"></a>
Les champs Liste et Liste déroulante (combo) peuvent bénéficier automatiquement de plusieurs FAA (Fonctionnalités Automatiques de vos Applications) : 

- **Pour le champ Combo** : 

	- [Tri des éléments](#NOTE2_1).

	- [Mémorisation de la valeur sélectionnée](#NOTE3_1). 

	- [Recherche dans la liste déroulée](#NOTE3B_1).




- **Pour le champ Liste** : 

	- [Tri des éléments](#NOTE2_1). 

	- [Mémorisation de la valeur sélectionnée](#NOTE2_1). 

	- [Copie de la ligne](#NOTE4_1).

	- [Copie de toutes les lignes](#NOTE3_1).

## Option de tri
<a name="option_tri_ELTTEXTE000264"></a>


### Conditions nécessaires
<a name="conditions_necessaires_ELTPARAGRAPHE000050"></a>Cette fonctionnalité est particulièrement utile si la liste est assez longue et si le développeur n'a pas pensé à en trier le contenu.

Pour disposer de cette fonctionnalité, les conditions suivantes sont requises :

- Le champ Liste ou Combo doit être alimenté par programmation ou par un fichier chargé en mémoire.

- Le champ Liste ou Combo ne doit pas déjà être trié.





### Fonctionnement
<a name="fonctionnement_ELTPARAGRAPHE000060"></a>Pour utiliser la fonctionnalité automatique de tri d'un champ Liste/Combo, sélectionnez l'option "Trier" du menu contextuel du champ.


### Mise en place
<a name="mise_place_ELTPARAGRAPHE000065"></a>L'option de tri dans les champs Combo et Liste est automatiquement mise en place à partir de la version 15.

Comme tous les menus contextuels proposés par les FAA, il est possible de configurer les options proposées par défaut par le champ. Pour plus de détails, consultez [Configurer les menus FAA](../Editeurs/2010040.md).

**En programmation** :

- la fonction [FAADésactive](../WDLang1/1000022018.md) utilisée avec la constante *faaTrieListeOuCombo* permet de supprimer l'option de tri dans le menu contextuel du champ Liste ou Combo spécifié.

- la fonction [FAAChangeLibellé](../WDLang1/1000022100.md) permet de modifier le libellé de l'option de tri.

- la propriété [Triée](../Proprietes/2510115.md) permet de savoir si le champ Liste ou Combo est trié.

## Mémorisation de la valeur
<a name="memorisation_valeur_ELTTEXTE000300"></a>


### Fonctionnement
<a name="fonctionnement_ELTPARAGRAPHE000096"></a>Pour utiliser la mémorisation de la valeur sélectionnée d'un champ Liste/Combo, sélectionnez l'option "Mémoriser la valeur" du menu contextuel du champ.


### Mise en place
<a name="mise_place_ELTPARAGRAPHE000101"></a>L'option "Mémoriser la valeur" est disponible automatiquement dans les champs Combo et Liste. Cette option permet de gérer la [persistance des données](../WDChamp/9000019.md). 

Comme tous les menus contextuels proposés par les FAA, il est possible de configurer les options proposées par défaut par le champ. Pour plus de détails, consultez [Configurer les menus FAA](../Editeurs/2010040.md).

**En programmation** : la fonction [FAADésactive](../WDLang1/1000022018.md) (ou [DésactiveFAA](../WDLang1/3013061.md)) utilisée avec la constante *faaSauveValeur* permet de supprimer l'option "Mémoriser la valeur" dans le menu contextuel du champ Liste ou Combo spécifié.

**Remarque** : Il est également possible de mémoriser la valeur grâce à l'option "Mémoriser la valeur" présente dans la fenêtre de description du champ (onglet "Détail"). Cette option permet de mémoriser la valeur même si l'utilisateur n'a pas activée l'option de menu. Si l'utilisateur a utilisé l'option de menu, l'option de menu est prioritaire.

## Recherche dans la liste déroulée
<a name="recherche_dans_liste_deroulee_ELTTEXTE000330"></a>


### Fonctionnement
<a name="fonctionnement_ELTPARAGRAPHE000129"></a>Pour utiliser la recherche dans la liste déroulée du champ Combo, il suffit de : 

- dérouler la liste du champ Combo, 

- commencer à saisir le texte recherché. Une zone de saisie permettant de saisir l'élément recherché apparaît. 





### Mise en place
<a name="mise_place_ELTPARAGRAPHE000137"></a>La recherche dans la liste déroulée du champ Combo est disponible automatiquement dans les champs Combo. 

**En programmation** : la fonction [FAADésactive](../WDLang1/1000022018.md) utilisée avec la constante *faaRechercheDansCombo* permet de désactiver la recherche pour le champ Combo spécifié.

**Remarque** : Il est également possible de proposer par défaut un champ de recherche dans la liste d'un champ Combo grâce à l'option "Avec recherche" présente dans la fenêtre de description du champ (onglet "Détail"). Cette option peut aussi être activée par le propriété [RechercheFAA](../Proprietes/1000023445.md).

## Copie de la ligne ou de toutes les lignes
<a name="copie_ligne_toutes_les_lignes_ELTTEXTE000360"></a>


### Fonctionnement
<a name="fonctionnement_ELTPARAGRAPHE000161"></a>Pour copier le contenu de la ligne sélectionnée dans un champ Liste, sélectionnez l'option "Copier la ligne" du menu contextuel. Le contenu de la ligne est copié dans le presse-papiers. 

Pour copier le contenu de toutes les lignes d'un champ Liste, sélectionnez l'option "Tout copier" du menu contextuel. Le contenu de toutes les lignes est copié dans le presse-papiers. 


### Mise en place
<a name="mise_place_ELTPARAGRAPHE000170"></a>Les options "Copier la ligne" et "Copier tout" sont disponibles automatiquement dans les champs Liste. 

Comme tous les menus contextuels proposés par les FAA, il est possible de configurer les options proposées par défaut par le champ. Pour plus de détails, consultez [Configurer les menus FAA](../Editeurs/2010040.md).

**En programmation** : 

- la fonction [FAADésactive](../WDLang1/1000022018.md) utilisée avec :

	- la constante *faaCopier* permet de désactiver la copie d'une ligne. 

	- la constante *faaToutCopier* permet de désactiver la copie de toutes les lignes.  




- la fonction [FAAChangeLibellé](../WDLang1/1000022100.md) permet de modifier le libellé des options de menu ("Copier la ligne" et "Copier tout").

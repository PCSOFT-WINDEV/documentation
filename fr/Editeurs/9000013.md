


## Tester et compiler un projet Java
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000131"></a>
Lorsque votre application Java est développée, il est possible de tester le projet Java et de le compiler avant de créer l'archive.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Test d'un projet Java sous l'éditeur
<a name="test_projet_java_sous_editeur_ELTTEXTE000155"></a>
Lors du test d'un projet depuis l'éditeur, le projet est compilé et lancé sous Windows. Le projet n'est pas compilé en Java.

Rappel : Pour lancer le test d'un projet, sous le volet "Projet", dans le groupe "Mode Test", déroulez "Mode test" et sélectionnez "Déboguer le projet".

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Compiler un projet Java sous WINDEV
<a name="compiler_projet_java_sous_windev_ELTTEXTE000179"></a>


### Compiler un projet Java
<a name="compiler_projet_java_ELTPARAGRAPHE000031"></a>La compilation d'un projet Java permet de :

- connaître les erreurs de programmation présentes dans le code WLangage.

- visualiser les champs, traitements, fonctions WLangage non disponibles en Java.




Ces erreurs apparaissent en violet dans le volet "Erreurs de compilation".

Pour plus de détails sur les éléments WINDEV non disponibles dans une application Java, consultez [Limitations de cette version](../Editeurs/9000056.md). 

**Remarque** : Le nombre de fonctions WLangage utilisables dans un projet JAVA est limité dans cette version. Ce nombre de fonctions est amené à croître régulièrement.

Pour compiler un projet Java, sous le volet "Projet", dans le groupe "Projet", déroulez "Recompiler et synchroniser" et sélectionnez "Recompiler le projet".
<a name="NOTE3_2"></a>


### Compiler un exécutable WINDEV en Java
<a name="compiler_executable_windev_java_ELTPARAGRAPHE000064"></a>A partir d'un projet WINDEV existant donnant lieu à la création d'un exécutable, vous pouvez créer en plus une application Java.

Pour cela, il faut ajouter une nouvelle configuration au projet : 

1. Sous le volet "Projet", dans le groupe "Configuration de projet", cliquez sur "Gérer les configurations".

2. Créez une nouvelle configuration (bouton "Nouvelle configuration"). Choisissez le type de génération "Java".

3. Sélectionnez les éléments du projet existant qui devront être intégrés dans cette configuration. Ces éléments seront également inclus dans l'archive ou l'applet qui sera générée.




## Ruban des éditeurs
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000154"></a>
<a name="Presentation_Ruban"></a>
WINDEV, WEBDEV et WINDEV Mobile présentent les différentes options des éditeurs à l'aide d'un ruban. Ce ruban est composé de volets dans lesquels sont regroupées les différentes options des éditeurs.

Le ruban est composé de différentes zones : ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Environnement%20WD%20-%20HC%20N%B0003.gif)


1. La zone des boutons d'accès rapide.

2. La zone des volets.

3. La zone des groupes d'options.

## Les boutons d'accès rapide
<a name="les_boutons_acces_rapide_ELTTEXTE000178"></a>


### Présentation
<a name="presentation_ELTPARAGRAPHE000026"></a>Les boutons d'accès rapide permettent d'effectuer les opérations les plus courantes, bien souvent communes à tous les éditeurs : enregistrer, ouvrir, créer, ...![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Environnement%20WD%20-%20HC%20N%B0003%201.gif)
Les 3 boutons en haut de cette zone sont également spécifiques : 

- le logo de la version permet d'afficher : 

	- la fenêtre "A propos...".

	- les perso-menus (menus personnalisés).

	- les menus déroulants des versions précédentes (version 17 et antérieures).




- les 2 pictogrammes suivants permettent de retrouver les menus et les barres d'outils des versions précédentes (version 17 et antérieures).





### Personnaliser les boutons d'accès rapide
<a name="personnaliser_les_boutons_acces_rapide_ELTPARAGRAPHE000040"></a>Pour personnaliser les boutons d'accès rapide : 

1. Faites un clic droit dans la zone des boutons d'accès rapide. 

2. Sélectionnez l'option "Personnaliser les boutons d'accès rapide". La fenêtre suivante s'affiche : 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Ruban_personnaliser.gif&type=thumb)
Dans cet écran : 

	- La zone de gauche correspond à la liste de toutes les options disponibles dans le produit. 

	- La zone de droite correspond aux 9 boutons de la zone d'accès rapide. 




3. Seuls 9 boutons peuvent être présents dans la zone d'accès rapide. Pour ajouter un ou plusieurs boutons, il est donc nécessaire de supprimer certains boutons de cette zone. Les boutons fléchés permettent de manipuler, enlever ou repositionner les options. 

4. Validez. La modification est automatiquement prise en compte.

## Les volets
<a name="les_volets_ELTTEXTE000208"></a>


### Présentation
<a name="presentation_ELTPARAGRAPHE000059"></a>Les différents volets du ruban permettent d'accéder aux options des différents éditeurs. ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Environnement%20WD%20-%20HC%20N%B0003%202.gif)
Plusieurs types de volets sont disponibles : 

- le volet en cours : L'onglet du volet apparaît en gris clair et un trait coloré est affiché en haut de l'onglet.

- les volets contextuels, spécifiques à l'élément en cours : Ces volets concernent l'éditeur en cours ou l'élément sélectionné. Le nom du volet est affiché : 

	- en jaune pour WINDEV, 

	- en orange pour WINDEV Mobile, 

	- en bleu pour WEBDEV.




- les autres volets disponibles : Le nom du volet apparaît en blanc.





### Rechercher une option dans le menu
<a name="rechercher_une_option_dans_menu_ELTPARAGRAPHE000074"></a>Vous recherchez une option spécifique ? Utilisez la loupe de la barre des volets : 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Ruban_Recherche.gif)


1. Cliquez sur la loupe : un champ de recherche est activé. 

2. Dès que vous saisissez un mot, l'éditeur propose automatiquement les options possibles associées à ce mot. Dans l'exemple ci-dessous, on recherche le mot "tableau" : 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Ruban_recherche_option.gif)


3. Si vous sélectionnez une des réponses proposées, vous êtes immédiatement positionné sur les options du menu à atteindre. Ces options sont encadrées d'un cadre jaune sur fond jaune pâle.
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Ruban_selection_option.gif)

## Les options
<a name="les_options_ELTTEXTE000238"></a>


### Présentation
<a name="presentation_ELTPARAGRAPHE000090"></a>Les options sont regroupées sous les volets dans des groupes d'options.![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Environnement%20WD%20-%20HC%20N%B0003%203.gif)
En fonction du volet sélectionné, les options affichées dans le ruban diffèrent. 

Plusieurs types d'options sont disponibles : 

- Options à cocher.

- Boutons à cliquer.

- Boutons à flèche permettant de dérouler des options. Deux types de boutons à flèche sont disponibles : 

	- les boutons à flèche permettant uniquement de dérouler un menu.

	- les boutons à flèche permettant soit de dérouler un menu (clic sur la flèche), soit d'effectuer une action par défaut (clic sur l'icône du bouton).







Les options sont regroupées par groupe. Chaque groupe d'options a un nom et peut avoir un bouton de regroupement : ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_regroup.gif)
. Ce bouton permet d'effectuer une action spécifique selon le groupe en cours : affichage de la description de l'élément en cours, affichage de l'aide, ...




## Définir un modèle de fenêtre d'aide
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000176"></a>
WINDEV offre la possibilité de créer des modèles de fenêtres d'aide. Ces modèles permettent de regrouper en une seule définition toutes les caractéristiques de la fenêtre d'aide : caractéristiques HTML, couleur des pages, etc.

Pour utiliser toutes les caractéristiques d'un modèle sur une page d'aide, il suffit d'associer le modèle à la page d'aide (onglet "Apparence" de la description de la page).

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Création d'un modèle
<a name="creation_modele_ELTTEXTE000200"></a>


### Comment créer un modèle de fenêtre d'aide ? 
<a name="comment_creer_modele_fenetre_aide_ELTPARAGRAPHE000020"></a>Pour créer un nouveau modèle de fenêtre d'aide :

1. Sous le volet "Modification", dans le groupe "Système d'aide", cliquez sur "Fenêtres d'aide".

2. Cliquez sur le bouton "Nouvelle" et indiquez le nom de la définition de fenêtre.

3. Sélectionnez les différentes options du modèle grâce aux différents onglets :

	- Panneaux affichés et boutons du système d'aide (onglet "Communes").

	- Les caractéristiques spécifiques aux pages d'aides compilées en HTML (onglet "HTML Help").

	- Les couleurs utilisées dans la page (onglet "Apparence").







<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Onglet "Communes"
<a name="onglet_communes_ELTTEXTE000224"></a>
Les options communes d'un modèle de page permettent de définir :

- le **nom logique** du modèle.

- le **titre de la fenêtre**du modèle. Si ce titre est précisé, ce titre sera affiché quel que soit le titre défini dans les caractéristiques de la page d'aide.

- les **panneaux** pouvant être affichés depuis la page.

- les **boutons** proposés dans la barre d'outils : Ces boutons permettront à l'utilisateur : 

	- d'afficher la page précédemment consultée. 

	- d'afficher toutes les options de l'aide (gestion des polices, aide toujours visible, etc.).

	- d'imprimer la page d'aide en cours.

## Onglet "HTML Help"
<a name="onglet_html_help_ELTTEXTE000248"></a>
Les options spécifiques à l'aide compilée au format HTML (fichier CHM) sont les suivantes :
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Modele_HTML.gif)


- **Afficher le panneau des Favoris** : Le panneau des favoris permet de référencer les pages d'aide fréquemment visualisées par l'utilisateur.

- **Options de présentation** :

	- ***Panneau latéral caché par défaut*** : Cache automatiquement l'ensemble des onglets (sommaire, index, rechercher, etc.) : seules la barre d'outils et la page sont affichées dans la fenêtre. Un bouton de la barre d'outils permet d'afficher les différents onglets.

	- ***Synchronisation automatique de la table des matières*** : Synchronise automatiquement le sommaire avec la page en cours dans l'aide. Si cette page est référencée dans le sommaire, le sommaire est déroulé et l'icône correspondante est sélectionnée.

	- ***Masquer le panneau latéral quand il perd le focus*** : Cache automatiquement l'ensemble des onglets lorsque le fichier d'aide n'a plus le focus. Dès que l'aide a de nouveau le focus, les onglets réapparaissent.




- **Boutons proposés sur la barre d'outils** : 

	- ***Masquer/Afficher le panneau latéral*** : Ce bouton permet à l'utilisateur d'afficher ou non les onglets (sommaire, recherche, index, etc.).

	- ***Page de démarrage*** : Ce bouton permet d'afficher la page par défaut du système d'aide.

	- ***Suivant*** : Ce bouton permet d'afficher la prochaine page (selon l'ordre d'apparition défini).

	- ***Synchroniser la table des matières*** : Ce bouton permet de synchroniser la table des matières avec la page d'aide en cours. 

	- ***Stop*** : Ce bouton arrête le chargement de la page d'aide. 

	- ***Actualiser*** : Ce bouton réaffiche la page d'aide en cours. 

	- ***Lien direct 1 / Lien direct 2*** : Ce bouton permet d'ouvrir par exemple un site Web, une page d'aide spécifique, etc. Les propriétés de ces liens doivent être saisies (libellé et URL).

## Onglet "Apparence"
<a name="onglet_apparence_ELTTEXTE000272"></a>
L'onglet "Apparence" permet de définir les options de couleur utilisées dans les pages d'aide.

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Modele_Apparence.gif)


- **Zone d'entête** : 
	Permet de définir : 

	- la couleur de fond de la zone d'entête.

	- l'image de fond utilisée pour cette zone.

	- le style du texte affiché dans cette zone.




- **Zone principale** :
	Permet de définir la couleur de fond de la zone de texte de la page et / ou l'image de fond utilisée pour cette zone.

- **Positionnement** :
	Permet de définir les coordonnées de la fenêtre d'aide (en pixels). L'assistant permet de simplifier la saisie des coordonnées :

	- cliquez sur le bouton "Assistant". Une fenêtre s'affiche.

	- modifiez la taille et la position de cette fenêtre et validez. La taille et les coordonnées de cette fenêtre sont automatiquement récupérées.

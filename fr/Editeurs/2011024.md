


## Synchroniser l'analyse avec les bases de données externes
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000116"></a>
La synchronisation de l'analyse avec une base de données externe est nécessaire si votre analyse manipule :

- des fichiers de données provenant d'une base de données externe (Oracle, SQL Server, ...).

- des fichiers de données dont les données n'ont pas été converties au format HFSQL Classic.




Cette synchronisation permet de prendre en compte les éventuels modifications apportées sur la structure des fichiers de données présents dans la base de données externe.

Il est possible :

- soit de forcer la synchronisation depuis l'éditeur d'analyses.

- soit de programmer une vérification automatique de l'évolution de la base de données externe à l'ouverture de l'analyse.




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000146"></a>


### Forcer la synchronisation
<a name="forcer_synchronisation_ELTPARAGRAPHE000030"></a>Pour forcer la synchronisation : 

1. Ouvrez l'analyse contenant des fichiers de données externes sous l'éditeur d'analyses.

2. Sous le volet "Analyse", dans le groupe "Analyse", déroulez "Synchronisation" et sélectionnez "Mettre à jour l'analyse depuis les bases de données externes". L'assistant de synchronisation avec les bases de données externes se lance.

3. Sélectionnez les bases de données utilisées dans l'analyse à vérifier.

4. Indiquez le type de vérification et validez. Il est possible de vérifier :

	- soit uniquement les fichiers de données (tables) communs à la base de données externe et à l'analyse.

	- soit tous les fichiers de données (tables) présents dans la base de données externe.




5. La liste des fichiers de données qui ne sont pas en phase avec l'analyse en cours apparaît. Il est possible de mettre à jour l'analyse en cours. Choisissez les modifications à appliquer.

6. Programmez si nécessaire une vérification régulière de la structure des bases de données externes. Cette vérification sera effectuée automatiquement à l'ouverture de l'analyse. Indiquez si nécessaire la fréquence de cette vérification.

7. Validez.



<a name="NOTE2_2"></a>


### Programmer une synchronisation régulière (à l'ouverture de l'analyse)
<a name="programmer_une_synchronisation_reguliere_ouverture_analyse_ELTPARAGRAPHE000060"></a>

1. Ouvrez l'analyse contenant des fichiers externes sous l'éditeur d'analyses.

2. Affichez la fenêtre de description de l'analyse : 

	- affichez le menu contextuel du graphe de l'analyse (clic droit). 

	- sélectionnez l'option "Description de l'analyse".




3. Dans l'onglet "Options", sélectionnez la fréquence de la synchronisation :<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Synchro.gif)


4. Validez.

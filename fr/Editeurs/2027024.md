<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000087"></a>
Le volet "Règles métier" affiche la liste des règles métier associées au projet.<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=RegleMetier.gif)
Pour plus de détails, consultez [Règles métier](../Editeurs/9000084.md).

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Afficher le volet "Règles métier"
<a name="afficher_volet_regles_metier_ELTTEXTE000111"></a>
**Pour afficher le volet "Règles métier"**, sous le volet "Accueil", dans le groupe "Environnement", déroulez "Volets" et sélectionnez "Volets" puis "Règles métier".

Pour plus de détails sur les manipulations des volets, consultez [Manipuler les volets ancrables](../Editeurs/2027001.md).

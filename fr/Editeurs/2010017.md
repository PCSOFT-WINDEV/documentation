
## Lancer un fichier d'aide (CHM, HTML ou PDF)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000131"></a>
Un fichier d'aide peut être lancé de différentes façons :

- [lancement **d'une aide complète**](#NOTE2_1) : permet d'ouvrir le fichier d'aide (affiche le sommaire). 
	Ce type de lancement est réalisé le plus souvent à partir d'une option de menu (du type "? .. Aide générale").

- [lancement **d'une aide sur une fenêtre ou un champ de l'application**](#NOTE3_1) : permet d'ouvrir le fichier d'aide sur une page spécifique. Cette page présente par exemple les différentes options disponibles dans la fenêtre ou sur le champ. 
	Le plus souvent, ce type de lancement est réalisé par l'appui sur la touche F1 ou par un bouton d'aide spécifique de la fenêtre.






<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Lancement d'une aide complète
<a name="lancement_une_aide_complete_ELTTEXTE000155"></a>
Le lancement de l'aide est réalisé grâce à la fonction [WHelp](../WDLang1/3013053.md). Il suffit de préciser le nom du fichier d'aide à lancer. Le fichier d'aide s'ouvre alors sur la première page de l'aide définie lors de la compilation.

Par exemple :


```wl
// Lancement de l'aide au format CHM
WHelp("MonAideChm.chm")
// Lancement de l'aide dans une application multilingue
WHelp("MonAide" + Nation() + ".chm")
```




<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Lancement d'une aide sur une fenêtre ou sur un champ
<a name="lancement_une_aide_sur_une_fenetre_sur_champ_ELTTEXTE000179"></a>


### Lancer l'aide associée à une fenêtre
<a name="lancer_aide_associee_une_fenetre_ELTPARAGRAPHE000047"></a>Pour lancer l'aide associée à une fenêtre, il suffit d'utiliser la fonction [WHelp](../WDLang1/3013053.md) en spécifiant l'identifiant de la page d'aide à afficher. Cet identifiant a été défini lors de la création du système d'aide (dans les [caractéristiques de la page](../Editeurs/2010024.md)).

**Remarque** : Pour créer automatiquement le bouton d'aide affichant la page correspondant à une fenêtre, effectuez un "Drag and Drop" de la page d'aide concernée vers la fenêtre voulue.

L'exemple suivant permet de lancer la page d'aide de la fenêtre "FEN_Configuration_Imp". Cette page d'aide a pour identifiant 5.


```wl
// Lancement de l'aide au format CHM
WHelp("MonAideChm.chm", 5)
```



<a name="NOTE3_2"></a>


### Lancer une aide sur un champ
<a name="lancer_une_aide_sur_champ_ELTPARAGRAPHE000067"></a>Pour lancer une page d'aide sur un champ :

1. Précisez le numéro de la page associée au champ dans la description du champ (onglet "Aide", champ "Numéro d'aide").

2. Utilisez la fonction [WHelp](../WDLang1/3013053.md) associée à la fonction [NumHelp](../WDLang1/3013027.md) pour lancer la page d'aide (par exemple à partir d'un bouton d'aide).




L'exemple suivant est le code d'un bouton d'aide. Ce bouton permet de lancer la page d'aide associée au champ en cours de saisie.


```wl
// -- Code du bouton Aide
WHelp("MonAide.chm", NumHelp(ChampEnCours()))
```

## Présentation
<a name="presentation_ELTTEXTE000119"></a>
Le volet "Trace du débogueur" permet de visualiser les messages affichés dans la trace lors du dernier test de l'application.<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=VoletTrace.gif)
Ces messages peuvent correspondre :

- aux appels de la fonction [Trace](../WDLang1/3013050.md).

- aux erreurs rencontrées lors du test. Le texte complet de l'erreur est affiché dans la fenêtre de trace.

- aux messages associés aux points d'arrêt. Pour plus de détails, consultez [Gestion des points d'arrêt](../Editeurs/2019005.md). 




Le menu contextuel (clic droit) de ce volet permet de :

- Vider la trace.

- Voir le code associé à la ligne en cours (il est également possible de double-cliquer sur la ligne dans la trace pour voir le code associé).

- Paramétrer les informations listées dans la trace du débogueur :

	- affichage des appels à la fonction Trace,

	- affichage des exceptions,

	- affichage des erreurs non fatales,

	- affichage des assertions,

	- affichage des caractères de contrôle sous forme de caractères (&lt;\\r&gt;&lt;\\n&gt;) ou les convertir dans leur valeur.




- Récupérer le contenu d'une ligne, exporter le contenu de la trace.




**Remarques** : 

- Le volet "Trace du débogueur" affiche uniquement les 1000 derniers affichages (1 affichage correspond à un appel de la fonction [Trace](../WDLang1/3013050.md) ou à une erreur).

## Afficher le volet "Trace du débogueur"
<a name="afficher_volet_trace_debogueur_ELTTEXTE000143"></a>
**Pour afficher le volet "Trace du débogueur"**, sous le volet "Accueil", dans le groupe "Environnement", déroulez "Volets" et sélectionnez "Volets" puis "Trace du débogueur".

Pour plus de détails sur les manipulations des volets, consultez [Manipuler les volets ancrables](../Editeurs/2027001.md).

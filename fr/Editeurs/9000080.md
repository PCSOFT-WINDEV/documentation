


## Analyse des dépendances et des images orphelines
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000103"></a>
Afin de vous aider à valider la circulation des internautes dans vos sites, une fonctionnalité avancée d'analyse des dépendances et des images orphelines est disponible dans WEBDEV. Cette fonctionnalité complète [l'analyse de code mort et éléments orphelins](../Editeurs/9000075.md).

Vous serez ainsi immédiatement averti en cas de liens brisés, réels ou potentiels. En effet, en mode dynamique, les parcours ne sont pas figés.

Les images inutilisées et les pages orphelines sont également signalées.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000127"></a>


### Lancer l'analyse des dépendances et des images orphelines
<a name="lancer_analyse_des_dependances_des_images_orphelines_ELTPARAGRAPHE000025"></a>Pour lancer l'analyse des dépendances et des images orphelines, sous le volet "Projet", dans le groupe "Audit et performances", déroulez "Audit statique" et sélectionnez "Analyser les dépendances".

La fenêtre résultat de cette analyse liste :

- **les images orphelines** (qui ne sont utilisées par aucune page). Il est possible de :

	- Supprimer les images sélectionnées du disque.

	- Archiver les images sélectionnées dans un fichier .ZIP et les supprimer.

	- Ignorer les images sélectionnées. Dans ce cas, les images ignorées pourront être affichées en cliquant sur "Images orphelines ignorées".




- **les dépendances**, c'est-à-dire les fichiers liés à d'autres fichiers. Pour chaque élément du projet WEBDEV (colonne "Elément"), toutes les dépendances sont données (colonne "Fichier"). Il est possible de visualiser :

	- toutes les dépendances,

	- uniquement les dépendances concernant les pages HTML,

	- uniquement les dépendances concernant les images,

	- uniquement les dépendances concernant les fichiers Javascript,

	- les autres dépendances.







Ces dépendances peuvent être affichées pour le projet, les pages du projet, les styles ou encore les états.

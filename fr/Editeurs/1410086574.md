
## Applications Universal Windows 10 App : Configuration nécessaire
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000102"></a>
Pour développer des applications de type Universal Windows 10 App avec WINDEV ou WINDEV Mobile, il est nécessaire d'installer : 

- **[le SDK Window 10 de Microsoft](#NOTE2_1)**. Le SDK pour Windows 10 (Software Development Kit ou Kit de développement logiciel) est un ensemble de fichiers et d'applications distribué par Microsoft afin de permettre la compilation d'applications pour le système d'exploitation Windows 10. 

- **Visual Studio 2010 Redistributable Package**. Ce package est téléchargeable sur le site Internet de Microsoft ([https://support.microsoft.com/fr-fr/help/2977003/the-latest-supported-visual-c-downloads](https://support.microsoft.com/fr-fr/help/2977003/the-latest-supported-visual-c-downloads)) (lien valide à la date de la rédaction de cette page). 




**Attention** : 

- Pour utiliser des API Universal Windows Platform (API UWP), il est également nécessaire d'installer Visual Studio (version 2015 minimum). 

- La création d'un projet de type Universal Windows Platform avec WINDEV ou WINDEV Mobile est disponible quel que soit le système du poste de développement. Pour générer et déployer l'application, le système du poste de développement doit être Windows 8 ou supérieur.

## SDK Windows 10
<a name="sdk_windows_10_ELTTEXTE000126"></a>


### Où trouver le SDK pour Windows 10 ?
<a name="trouver_sdk_pour_windows_10_ELTPARAGRAPHE000031"></a>Le SDK pour Windows 10 est disponible en téléchargement sur le site de Microsoft à l'adresse suivante : 

- [https://developer.microsoft.com/en-us/windows/downloads/sdk-archive/](https://developer.microsoft.com/en-us/windows/downloads/sdk-archive/) (version US).

- [https://developer.microsoft.com/fr-fr/windows/downloads/sdk-archive/](https://developer.microsoft.com/fr-fr/windows/downloads/sdk-archive/) (version FR).




**Important** : 

- Cette adresse est valide à l'heure de l'écriture de cette documentation.

- Il est conseillé de vérifier régulièrement si des mises à jour du SDK sont disponibles. 



<a name="NOTE2_2"></a>


### Comment installer le SDK pour Windows 10 ?
<a name="comment_installer_sdk_pour_windows_10_ELTPARAGRAPHE000050"></a>Après exécution du fichier téléchargé, tous les outils nécessaires au développement d'une application de type "Universal Windows 10 App" pour Windows 10 sont installés. 

Remarque : Le chemin du répertoire d'installation du SDK devra être indiqué dans l'assistant de création de projet.

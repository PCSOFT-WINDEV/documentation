## Configuration du serveur pour le protocole CGI (AWP)
<a name="configuration_serveur_pour_protocole_cgi_awp_ELTTEXTE000091"></a>
Pour configurer le serveur Web :

1. Ouvrez le gestionnaire de service Internet Microsoft.

2. Affichez les services du serveur.

3. Double cliquez sur le service WWW.

4. Cliquez sur l'onglet "Répertoire".

5. Cliquez sur "AJOUTER".

6. Sélectionnez le répertoire contenant le protocole CGI WD280AWP.EXE (sous-répertoire Programmes\\MOTEUR\\WINDOWS\\AWP du répertoire d'installation de WEBDEV).

7. Indiquez le nom de l'alias : WD280AWP.

8. Cochez les cases Lire - Exécuter.

9. Validez.

## Configuration du serveur pour les sites WEBDEV
<a name="configuration_serveur_pour_les_sites_webdev_ELTTEXTE000115"></a>
Pour configurer le serveur Web :

1. Ouvrez le gestionnaire de service Internet Microsoft.

2. Affichez les services du serveur.

3. Double-cliquez sur le service WWW.

4. Cliquez sur l'onglet "Répertoire".

5. Cliquez sur "AJOUTER".

6. Sélectionnez le répertoire contenant les images du projet (&lt;Nom Projet&gt;_WEB). Par exemple : C:\\WWprojet\\MARKET\\MARKET_WEB.

7. Indiquez le nom de l'alias &lt;NomProjet&gt;_WEB. Par exemple : MARKET_WEB.

8. Cochez les cases Lire - Scripts.

9. Validez.

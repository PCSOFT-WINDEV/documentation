


## Administrateur du GDS : Gestion des branches
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000136"></a>
L'administrateur de GDS (Gestionnaire de sources) permet de gérer directement les branches des différents projets présents dans le GDS.

Une branche est une copie séparée du projet réalisée à un instant donné. Par exemple, une branche peut être créée pour permettre la stabilisation d'une version envoyée en clientèle. Les développeurs peuvent ainsi travailler à la fois sur la branche et à la fois sur les éléments prévus pour une prochaine version. Pour plus de détails sur les branches, consultez [Gestion des branches](../Editeurs/2038016.md).

**Remarque** : L'administrateur du GDS permet également de réaliser des opérations de maintenance sur la base, ... Pour plus de détails, consultez [Présentation de l'administrateur du GDS](../Editeurs/2038008.md).

## Gestion des branches
<a name="gestion_des_branches_ELTTEXTE000160"></a>


### Créer une branche
<a name="creer_une_branche_ELTPARAGRAPHE000028"></a>Pour créer une branche :

1. Sélectionnez le projet voulu sous l'administrateur du GDS.

2. Sous le volet "Répertoires et projets", dans le groupe "Branches", cliquez sur "Créer une Branche".


L'assistant de création d'une branche se lance. Pour plus de détails, consultez [Création d'une branche](../Editeurs/2038016.md).
<a name="NOTE2_2"></a>


### Récupérer les modifications effectuées dans une branche
<a name="recuperer_les_modifications_effectuees_dans_une_branche_ELTPARAGRAPHE000055"></a>Pour récupérer les modifications effectuées dans une branche : 

1. Sélectionnez le projet voulu sous l'administrateur du GDS.

2. Sous le volet "Répertoires et projets", dans le groupe "Branches", cliquez sur "Récupérer des modifications".


L'assistant de récupération des modifications d'une branche se lance. Pour plus de détails, consultez [Récupération des modifications d'une branche](../Editeurs/2038016.md).
<a name="NOTE2_3"></a>


### Supprimer une branche
<a name="supprimer_une_branche_ELTPARAGRAPHE000082"></a>Pour supprimer une branche : 

1. Sélectionnez la branche voulue sous l'administrateur du GDS.

2. Dans le menu contextuel de la branche, sélectionnez l'option "Supprimer". 


ATTENTION : La suppression d'une branche du GDS n'est pas conseillée. Cette opération n'est pas annulable.

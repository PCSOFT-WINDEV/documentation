
## Environnement général de WINDEV 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a><br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=EnvWinDev.gif&type=thumb)


Les éléments importants de l'environnement général de WINDEV sont les suivants :


|   |   |
| --- | --- |
| <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Cercle1.gif)<br> | **Ruban**<br><br>Permet un accès rapide aux principales fonctionnalités de WINDEV. Ce ruban est composé de volets, de groupes et d'options. Pour plus de détails, consultez [Un menu sous forme de ruban](../Editeurs/9000149.md). |
| <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Cercle2.gif)<br> | **Tableau de bord du projet**<br><br>Regroupe les principaux accès aux différents éléments du projet. |
| <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Cercle3.gif)<br> | **Espace de travail**<br><br>Affiche les fenêtres des différents éditeurs (fenêtre de code, fenêtres ou états en création, analyse, etc.) |
| <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Cercle4.gif)<br> | [**Volets ancrables**](../Editeurs/2027001.md)<br><br>Affiche par exemple les éléments (fenêtres, états, champs, etc.) disponibles dans le projet en cours ([volet "Explorateur de projet"](../Editeurs/2027029.md)). |
| <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Cercle5.gif)<br> | **[Documents ouverts](../Editeurs/2029012.md)**<br><br>Affiche un bouton pour chaque document ouvert sous l'éditeur. Pour afficher l'élément voulu au premier plan, il suffit de cliquer sur le "bouton" correspondant. |
| <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Cercle6.gif)<br> | **[Volets ancrables](../Editeurs/2027001.md)**<br><br>Volets permettant de :<br><br>- afficher les erreurs de compilation (erreurs présentes dans le code WLangage, [volet "Erreurs de compilation"](../Editeurs/2027031.md)).<br><br>- effectuer des recherches ([volet "Rechercher - Remplacer"](../Editeurs/2027013.md)).<br><br>- afficher le débogueur lors du test d'un élément ([volet "Débogueur"](../Editeurs/2027030.md)).<br><br>- ...<br><br><br> |

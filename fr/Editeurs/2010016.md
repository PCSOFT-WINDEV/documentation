


## Propriétés du système d'aide
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000135"></a>
Les propriétés du système d'aide permettent de définir :

- le nom de l'aide ([onglet "Général"](#NOTE2_1)).

- la page affichée par défaut ([onglet "Général"](#NOTE2_1)).

- le titre de l'aide ([onglet "Général"](#NOTE2_1)).

- les langues gérées par le système d'aide ([onglet "Langues"](#NOTE3_1)).




Pour afficher les propriétés du système d'aide, sous le volet "Modification", dans le groupe "Système d'aide", cliquez sur "Description".

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Onglet "Général"
<a name="onglet_general_ELTTEXTE000159"></a>
Dans l'onglet "Général", il est possible de définir :

- **le nom logique du fichier d'aide**. Ce nom permettra de manipuler le fichier d'aide directement dans WINDEV. Par défaut, ce nom correspond au nom physique du fichier d'aide. Ce nom est utilisé dans les [options de génération](../Editeurs/2010015.md). 

- **la page par défaut du fichier d'aide**. Cette page sera la première page affichée par défaut par l'aide. Sélectionnez cette page parmi les pages de votre système d'aide.

- **le titre de l'aide**. Ce titre sera affiché dans la barre de titre du fichier d'aide.

## Onglet "Langues"
<a name="onglet_langues_ELTTEXTE000183"></a>
L'onglet "Langues" permet de définir les différentes langues gérées par le système d'aide. En effet, un système d'aide peut être multilingue. Pour plus de détails, consultez [Réaliser une aide multilingue](../Editeurs/2010025.md).

**Rappel** : Par défaut, votre système d'aide gère les mêmes langues que le projet. Pour modifier ces langues (par exemple pour gérer plus ou moins de langues) :

1. Affichez la description du système d'aide : 

	- sous le volet "Modification", dans le groupe "Système d'aide", cliquez sur "Description". 

	- affichez l'onglet "Langues".




2. Sélectionnez les langues à gérer par votre système d'aide. Le bouton "Synchroniser avec le projet" permet de prendre en compte automatiquement toutes les langues gérées par le projet.




## Manipuler un composant interne
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000159"></a>
Un composant interne est un regroupement d'éléments d'un projet. Ce regroupement permet de :

- **Organiser un projet** : vous pouvez créer des composants internes pour regrouper les éléments d'un projet par exemple par fonctionnalité.

- **Partager des éléments entre différents projets** : Les éléments présents dans un composant interne peuvent être partagés entre plusieurs projets par l'intermédiaire du GDS. Pour plus de détails, consultez [Partager un composant interne](../Editeurs/2014101.md).




L'éditeur permet à tout moment de manipuler un composant interne depuis le volet "Explorateur du projet". La visualisation des éléments d'un composant interne est hiérarchisée dans l'explorateur de projet : fenêtres, états, pages, ...

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Manipulations réalisables
<a name="manipulations_realisables_ELTTEXTE000183"></a>


### Afficher la description d'un composant interne
<a name="afficher_description_composant_interne_ELTPARAGRAPHE000026"></a>Pour afficher la description d'un composant interne :

1. Sélectionnez le composant interne dans le volet "Explorateur du projet".

2. Sélectionnez l'option "Description" du menu contextuel.




Pour plus de détails sur les différentes informations affichées, consultez [Fenêtre de description d'un composant interne](../Editeurs/2014103.md).
<a name="NOTE3_2"></a>


### Ajouter un élément dans le composant interne
<a name="ajouter_element_dans_composant_interne_ELTPARAGRAPHE000041"></a>Il est possible d'ajouter un élément dans un composant interne :

- par la description du composant interne (onglet "Eléments").

- par le volet "Explorateur du projet". Il suffit de faire un "Drag and Drop" d'un élément du projet vers le composant interne.




Le fichier correspondant est déplacé physiquement dans le répertoire du composant interne. Par défaut, l'élément intégré au composant interne est "Public". Pour rendre cet élément "Privé", utilisez par exemple l'option "Passer en privé" du menu contextuel de l'élément.
<a name="NOTE3_3"></a>


### Supprimer un élément du composant interne
<a name="supprimer_element_composant_interne_ELTPARAGRAPHE000053"></a>Il est possible de supprimer un élément du composant interne :

- par la description du composant interne (onglet "Eléments").

- par le volet "Explorateur du projet" : un Drag and Drop (glisser / déplacer) d'un élément d'un composant interne vers la catégorie correspondante du projet permet de "sortir" l'élément du composant interne. L'élément est alors associé au projet.




**Attention** : Si un élément d'un composant interne est supprimé avec la touche Suppr dans un projet qui n'utilise pas le GDS, cet élément ne fera plus partie du projet. Pour réintégrer cet élément du composant interne dans le projet, cet élément devra être ouvert et ajouté manuellement au projet.
<a name="NOTE3_5"></a>


### Modifier les éléments publics et privés d'un composant interne
<a name="modifier_les_elements_publics_prives_composant_interne_ELTPARAGRAPHE000065"></a>La modification des éléments publics et privés d'un composant interne peut être réalisée :

- Dans le volet "Explorateur du projet", sélectionnez l'élément et utilisez l'option du menu contextuel "Passer en privé" ou "Passer en public". Si l'élément est passé en privé, un carré rouge s'affiche sur l'élément dans le volet "Explorateur du projet".

- A partir de la description du composant interne (onglet "Eléments").



<a name="NOTE3_6"></a>


### Suppression d'un composant interne
<a name="suppression_composant_interne_ELTPARAGRAPHE000075"></a>Un composant interne peut être supprimé du projet en utilisant la touche Suppr.

**Attention** : Si un composant interne est supprimé d'un projet qui n'utilise pas le GDS, les éléments du composant interne ne feront plus partie du projet. Pour réintégrer les éléments du composant interne dans le projet, les éléments du composant interne devront être ouverts et ajoutés manuellement au projet.

Pour réintégrer le composant interne dans le projet, sous le volet "Projet", dans le groupe "Projet", déroulez "Importer" et sélectionnez "Un composant interne .. A partir d'un fichier".
<a name="NOTE3_7"></a>


### Lister les composants internes d'un projet
<a name="lister_les_composants_internes_projet_ELTPARAGRAPHE000098"></a>Pour lister les composants internes d'un projet :

1. Dans l'explorateur de projet : 

	- Sélectionnez le dossier "Composants internes". 

	- Dans le menu contextuel, sélectionnez l'option "Gérer les composants internes du projet".




2. La fenêtre qui s'affiche liste tous les composants internes utilisés par le projet. Depuis cette fenêtre il est possible de :

	- Afficher la description du composant interne sélectionné.

	- Mettre à jour le composant interne (lors d'un partage par le GDS).

	- Supprimer un composant interne.

	- Importer un composant interne présent dans le GDS.

	- Importer les fichiers d'un composant interne dans le projet. Cette option permet de recopier les fichiers d'un composant interne dans le projet en cours. Les fichiers sont recopiés et non partagés. Cette option est utile par exemple lors d'une suppression non voulue d'un composant dans le projet.




## Le volet "Favoris"
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000161"></a>
Le volet "Favoris" permet d'accéder facilement aux favoris de votre projet.<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=VoletFavoris.gif)
Un favori correspond à un élément important de votre projet, un élément sur lequel vous travaillez en ce moment, une procédure spécifique, ...

La liste des favoris d'un projet peut être modifiée par tous les utilisateurs de ce projet.

Vous pouvez visualiser :

- soit les favoris de tous les utilisateurs du projet.

- soit uniquement vos favoris.




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Afficher le volet "Favoris"
<a name="afficher_volet_favoris_ELTTEXTE000185"></a>
**Pour afficher le volet "Favoris"**, sous le volet  sous le volet "Accueil", dans le groupe "Environnement", déroulez "Volets" et sélectionnez "Volets" puis "Favoris".

Pour plus de détails sur les manipulations des volets, consultez [Manipuler les volets ancrables](../Editeurs/2027001.md).

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Manipuler les favoris d'un projet
<a name="manipuler_les_favoris_projet_ELTTEXTE000209"></a>


### Ajouter un favori
<a name="ajouter_favori_ELTPARAGRAPHE000051"></a>Pour ajouter un favori :

1. Affichez le volet "Favoris" (voir le paragraphe précédent).

2. Ouvrez ou affichez l'élément que vous souhaitez ajouter aux favoris.

3. Cliquez sur l'icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=IconeAjoutFavoris.GIF).

4. Saisissez le nom sous lequel cet élément doit apparaître dans la liste des favoris.

5. Validez.


**Remarque** : Tous les éléments d'un projet peuvent devenir des favoris (fenêtre, page, état, traitement, procédure, ...).
<a name="NOTE3_2"></a>


### Afficher la liste des favoris
<a name="afficher_liste_des_favoris_ELTPARAGRAPHE000068"></a>Pour afficher la liste des favoris :

1. Affichez le volet "Favoris" (voir le paragraphe précédent).

2. Pour visualiser :

	- les favoris de tous les utilisateurs du projet : sélectionnez l'option "Tous les Favoris".

	- uniquement vos favoris : sélectionnez l'option "Mes Favoris". <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ListeFavoris.GIF)







<a name="NOTE3_3"></a>


### Accéder à un favori
<a name="acceder_favori_ELTPARAGRAPHE000082"></a>Pour accéder à un favori :

1. Affichez le volet "Favoris" (voir le paragraphe précédent).

2. Sélectionnez le favori auquel vous souhaitez accéder.

3. Cliquez sur l'icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=IconeAccesFavoris.GIF).




**Remarque** : Vous pouvez aussi double-cliquer directement sur le favori auquel vous souhaitez accéder.
<a name="NOTE3_4"></a>


### Supprimer un favori
<a name="supprimer_favori_ELTPARAGRAPHE000097"></a>Pour supprimer un favori :

1. Affichez le volet "Favoris" (voir le paragraphe précédent).

2. Sélectionnez le favori à supprimer.

3. Cliquez sur l'icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=IconeSupprimeFavoris.GIF).




**Remarque** : Vous pouvez supprimer uniquement vos favoris.
<a name="NOTE3_5"></a>


### Favoris et GDS
<a name="favoris_gds_ELTPARAGRAPHE000112"></a>Le GDS (Gestionnaire de Sources) gère les favoris. Lors de l'ajout d'un élément dans les favoris du projet, aucun message n'est envoyé dans la messagerie WDBal.

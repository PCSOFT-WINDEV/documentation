


## Réparer le projet (Mise à niveau)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000082"></a>
Dans certains cas, il est possible de réparer le projet afin de manipuler un projet posant problèmes. Pour réparer le projet, sous le volet "Projet", dans le groupe "Projet", déroulez "Recompiler et synchroniser" et sélectionnez "Réparer le projet".

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Quelles sont les manipulations effectuées lors de la réparation d'un projet ?
<a name="quelles_sont_les_manipulations_effectuees_lors_reparation_projet_ELTTEXTE000106"></a>
La réparation d'un projet effectue les opérations suivantes :

- Ouverture et sauvegarde de tous les éléments du projet. Tous les éléments du projet sont re-sauvés au format de la version en cours d'utilisation.

- [Recompilation du projet](../Editeurs/9500205.md).

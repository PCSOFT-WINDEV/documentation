


## Portefeuille de projets
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000258"></a>
A partir de la version 25, il devient possible de créer des "portefeuilles de projets".

Un portefeuille de projets est un ensemble de projets logiquement regroupés.

Ces projets ont souvent des liens fonctionnels : ils partagent des composants, des classes, des collections de procédures, ...

Un portefeuille de projets permet d'effectuer des actions en bloc sur le portefeuille entier, comme par exemple :

- recompilation en nouvelle version,

- création de branches,

- ...


Finies les manipulations à répétition lors d'un changement de version ou d'un Sprint !

<a name="NOTE2_1"></a>


## Comment créer un portefeuille de projets ?
<a name="comment_creer_portefeuille_projets_ELTTEXTE000282"></a>


### Créer un portefeuille de projets 
<a name="creer_portefeuille_projets_ELTPARAGRAPHE000030"></a>Pour créer un portefeuille de projets : 

1. Affichez la page d'accueil du produit (Ctrl + &lt;). 

2. Cliquez sur l'option "Ouvrir un projet". 

3. Affichez le menu contextuel de l'option "Projets épinglés" et sélectionnez l'option "Ajouter un portefeuille". 

4. Dans la fenêtre qui s'affiche, indiquez : 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Portefeuille%20-%20HC%20N%B0001.gif)


	- Le nom du portefeuille de projets. 

- L'icône associée au portefeuille de projets. Cette icône a pour taille 32 x 32. Cette icône peut être sélectionnée via l'icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Portefeuille%20-%20HC%20N%B0001%201.gif)
 soit dans le catalogue d'images, soit dans une répertoire, soit via la galerie du projet. 

5. Validez. Le portefeuille de projets est créé. Il apparait dans la liste des projets présente dans la page d'accueil. Le menu contextuel du portefeuille de projets permet de réaliser les différentes manipulations possibles sur ce portefeuille. 




Remarque : La création et la manipulation d'un portefeuille de projets peut également être effectuée via le volet "Projets récents et épinglés". 

- Pour afficher le volet "Projets récents et épinglés", sous le volet "Accueil", dans le groupe "Environnement", déroulez "Volets" et sélectionnez "Volets" puis "Projets récents et épinglés". 

- Pour créer un portefeuille de projets, vous pouvez utiliser : 

	- le menu contextuel de l'option "Projets épinglés". 

	- l'icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Portefeuille%20-%20HC%20N%B0002%201.gif)
.

## Manipulations possibles sur un portefeuille de projets
<a name="manipulations_possibles_sur_portefeuille_projets_ELTTEXTE000306"></a>


### Présentation
<a name="presentation_ELTPARAGRAPHE000073"></a>Les manipulations disponibles sur un portefeuille de projets sont les suivantes : 

- [Ajouter un projet dans un portefeuille](#NOTE3_2).

- [Modifier la description du portefeuille de projets](#NOTE3_3). 

- [Enlever un projet d'un portefeuille](#NOTE3_4).

- [Supprimer un portefeuille de projets](#NOTE3_5). 




Ces manipulations peuvent être réalisées aussi bien depuis la page d'accueil du produit que depuis le volet "Projets récents et épinglés". 
<a name="NOTE3_2"></a>


### Ajouter un projet dans un portefeuille de projets
<a name="ajouter_projet_dans_portefeuille_projets_ELTPARAGRAPHE000095"></a>Pour ajouter un projet dans un portefeuille de projets : 

1. Affichez le menu contextuel du portefeuille de projets (présent dans la page d'accueil ou dans le volet "Projets récents et épinglés"). 

2. Sélectionnez l'option "Ajouter un projet". 

3. Dans le sélecteur de fichiers qui s'ouvre sélectionnez le projet WINDEV, WEBDEV ou WINDEV Mobile à ajouter au portefeuille (fichier \*.wdp, \*.wbp ou \*.wpp). 

4. Validez. Le projet sélectionné est ajouté au portefeuille sélectionné. 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Portefeuille%20-%20HC%20N%B0002.gif)





Remarques : 

- Seul le chemin du projet est référencé. Si le projet est déplacé, il ne pourra plus être manipulé par le portefeuille de projets : il sera nécessaire d'enlever le projet du portefeuille puis de l'ajouter à nouveau. 

- Il est également possible d'ajouter un projet au portefeuille de projets en réalisant un "Drag and drop" du projet voulu vers le portefeuille de projets. 

- Un projet présent dans le GDS peut être intégré à un portefeuille de projets : il suffit de sélectionner le répertoire local du projet correspondant. 

- Un portefeuille de projets peut contenir des projets correspondants à des produits différents. 

- Un projet peut être associé à plusieurs portefeuilles de projets. 



<a name="NOTE3_3"></a>


### Modifier la description d'un portefeuille de projets
<a name="modifier_description_portefeuille_projets_ELTPARAGRAPHE000116"></a>Pour modifier la description d'un portefeuille de projets : 

1. Affichez le menu contextuel du portefeuille de projets (présent dans la page d'accueil ou dans le volet "Projets récents et épinglés"). 

2. Sélectionnez l'option "Description du portefeuille". 

3. Dans la fenêtre qui s'affiche, vous pouvez renommer le portefeuille ou changer son icône. 

4. Validez. Les modifications sont prises en compte. 



<a name="NOTE3_4"></a>


### Enlever un projet du portefeuille de projets
<a name="enlever_projet_portefeuille_projets_ELTPARAGRAPHE000128"></a>Pour enlever un projet d'un portefeuille de projets : 

1. Affichez le menu contextuel d'un projet associé à un portefeuille de projets (présent dans la page d'accueil ou dans le volet "Projets récents et épinglés"). 

2. Sélectionnez l'option "Enlever le projet du portefeuille". 



<a name="NOTE3_5"></a>


### Supprimer un portefeuille de projets
<a name="supprimer_portefeuille_projets_ELTPARAGRAPHE000142"></a>Pour supprimer un portefeuille de projets : 

1. Affichez le menu contextuel du portefeuille de projets (présent dans la page d'accueil ou dans le volet "Projets récents et épinglés"). 

2. Sélectionnez l'option "Supprimer le portefeuille".

## Manipulations possibles sur les projets d'un portefeuille de projets
<a name="manipulations_possibles_sur_les_projets_portefeuille_projets_ELTTEXTE000354"></a>


### Présentation
<a name="presentation_ELTPARAGRAPHE000158"></a>Il est possible de réaliser des actions : 

- sur un projet présent dans un portefeuille de projets. Ces actions sont proposées par le menu contextuel du projet : 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Portefeuille%20-%20HC%20N%B0003.gif)


	- Ouvrir le projet : Permet d'ouvrir directement le projet. 

- Enlever le projet du portefeuille : Permet de ne plus associer le projet au portefeuille de projets. 

- Ouvrir le répertoire du projet : Permet d'ouvrir l'explorateur de fichiers dans le répertoire du projet. 

- Copier le chemin du projet : Permet de copier dans le presse-papiers le chemin du projet. 

- Ouvrir le projet dans un autre éditeur : Dans le cas de l'utilisation d'un produit WINDEV avec WEBDEV et / ou WINDEV Mobile intégré(s), permet de choisir le produit à utiliser pour le projet. 

- [Ajouter un projet dans le portefeuille](#NOTE3_2) : Permet d'ajouter un nouveau projet dans le portefeuille de projets. 

- sur l'ensemble des projets présents dans le portefeuille de projets. Ces actions sont proposées par le menu contextuel du portefeuille de projets : 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Portefeuille%20-%20HC%20N%B0004.gif)


	- [Créer une branche pour tous les projets](#NOTE4_2). 

- [Recompiler les projets en version 28](#NOTE4_3). 




Ces manipulations peuvent être réalisées aussi bien depuis la page d'accueil du produit que depuis le volet "Projets récents et épinglés". 
<a name="NOTE4_2"></a>


### Créer une branche pour tous les projets
<a name="creer_une_branche_pour_tous_les_projets_ELTPARAGRAPHE000189"></a>Attention : Pour utiliser cette option, au moins un des projets du portefeuille de projets doit être présent dans un GDS. 

Pour créer une branche pour tous les projets : 

1. Sélectionnez le portefeuille de projets. 

2. Affichez le menu contextuel et sélectionnez l'option "Créer une branche pour tous les projets". 

3. L'assistant de création de branche se lance. 

4. La première étape de l'assistant permet de : 

	- Choisir le nom de la branche. 

	- Sélectionner le ou les projets pour lesquels la branche doit être créée. 




5. Sélectionnez ensuite la localisation de la branche à créer qui contiendra l'ensemble des projets. 

6. Validez vos choix en conservant les partages des objets dans le GDS si nécessaire. 

7. La branche est créée pour tous les projets du portefeuille de projets. 



<a name="NOTE4_3"></a>


### Recompiler tous les projets
<a name="recompiler_tous_les_projets_ELTPARAGRAPHE000209"></a>Cette option permet de recompiler tous les projets d'un portefeuille de projets. Cette option est notamment intéressante lors d'un changement de version majeur du produit, par exemple lors du passage de la version 27 à la version 28.

## Présentation
<a name="presentation_ELTTEXTE000135"></a>
Une requête sert à interroger une base de données pour visualiser, insérer, modifier ou supprimer des données. La structure de la requête définit les données manipulées.

Une requête peut interroger un ou plusieurs fichiers de données.

L'éditeur de requêtes permet de créer très simplement des requêtes, sans avoir à programmer.
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=RequeteRepresentation.gif)
**Par exemple**, pour l'application Gestion Commerciale, vous voulez connaître :

- les commandes passées en mars 2015. Vous allez créer une requête qui va sélectionner dans le fichier COMMANDE uniquement les commandes dont la date est comprise entre le 01/03/2015 et le 31/03/2015.
	La requête REQ_Mars2015 interroge le fichier COMMANDE. 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Requete_ExempleUnFichier.gif)


- le numéro des commandes passées par les clients de Paris (France). Vous allez créer une requête qui va interroger :

	- le fichier CLIENT pour sélectionner uniquement les clients habitant Paris (France). 

	- le fichier COMMANDE pour afficher les numéros de commande.







La requête REQ_CommandesClientParisFrance interroge deux fichiers de données (CLIENT et COMMANDE). 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Requete_ExemplePlusieursFic.gif)
**Remarque** : Pour plus de détails sur la manipulation d'une requête dans un traitement WLangage, consultez [Faire référence à un élément dans un traitement WLangage](../WDChamp/1014007.md).

## Caractéristiques d'une requête
<a name="caracteristiques_une_requete_ELTTEXTE000159"></a>
Les caractéristiques d'une requête sont :

- [les rubriques](../Editeurs/2032004.md),

- [les jointures](../Editeurs/2032018.md),

- [les conditions de sélection](../Editeurs/2032019.md),

- [l'enchaînement des conditions de sélection](../Editeurs/2032055.md),

- [les opérations possibles dans une requête](../Editeurs/2032020.md),

- [l'ordre de tri des enregistrements](../Editeurs/2032005.md),

- [l'affichage des n premiers/derniers enregistrements d'une requête](../Editeurs/2032022.md),

- [la modification des valeurs affichées dans le résultat de la requête](../Editeurs/2032016.md),

- [l'affichage des enregistrements correspondant à une période donnée](../Editeurs/2032029.md).




Après la création de la requête, il est possible à tout moment de modifier les caractéristiques de cette requête. Pour plus de détails, consultez [Description d'une requête](../Editeurs/2032038.md).

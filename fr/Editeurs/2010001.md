
## Créer, modifier, supprimer un style
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000151"></a>
WINDEV, WEBDEV et WINDEV Mobile offrent la possibilité de manipuler les styles définis pour un système d'aide.

Pour éditer les styles disponibles dans le système d'aide en cours : sous le volet "Modification", dans le groupe "Styles", cliquez sur "Editer les styles". Cette option est disponible lorsque le système d'aide est affiché sous l'éditeur.


![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Edition_Styles.gif)


Cette fenêtre présente les différents styles présents dans le système d'aide, et pour chaque style, ses caractéristiques.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Ajouter un nouveau style
<a name="ajouter_nouveau_style_ELTTEXTE000175"></a>
Pour ajouter un nouveau style : 

1. Affichez la fenêtre de description des styles : sous le volet "Modification", dans le groupe "Styles", cliquez sur "Editer les styles".

2. Cliquez sur le bouton "Nouveau".

3. Saisissez le nom du nouveau style et ses caractéristiques dans la fenêtre de propriétés du style.

4. Validez la fenêtre de propriétés du style. Le style apparaît dans la fenêtre de description des styles.

5. Validez la fenêtre de description des styles.




<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Modifier un style
<a name="modifier_style_ELTTEXTE000199"></a>
Pour modifier un style : 

1. Affichez la fenêtre de description des styles : sous le volet "Modification", dans le groupe "Styles", cliquez sur "Editer les styles".

2. Sélectionnez dans la liste le style à modifier.

3. Cliquez sur le bouton "Modifier".

4. Saisissez les caractéristiques du style dans la fenêtre de propriétés du style.

5. Validez la fenêtre de propriétés du style. Le style apparaît dans la fenêtre de description des styles.

6. Validez la fenêtre de description des styles.




<a name="NOTE4"></a>
<a name="NOTE4_1"></a>


## Supprimer un style
<a name="supprimer_style_ELTTEXTE000223"></a>
Pour supprimer un style : 

1. Affichez la fenêtre de description des styles : sous le volet "Modification", dans le groupe "Styles", cliquez sur "Editer les styles".

2. Sélectionnez dans la liste le style à supprimer.

3. Cliquez sur le bouton "Supprimer". Le style est supprimé. Les paragraphes associés à ce style ne sont plus associés à un style (la combo "Style" est vide).

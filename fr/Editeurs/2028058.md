


## Etape 2 : Fichiers à installer
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000137"></a>
L'assistant de création du programme d'installation analyse et sélectionne automatiquement les fichiers à installer lors de l'installation du site.

Lors d'une installation par FTP, pour visualiser la liste des fichiers pris en compte, cliquez sur le bouton "Editer la liste des fichiers". 

Lors de la création d'un pack de déploiement ou de l'installation par média physique, la liste des fichiers est automatiquement proposée. 

Remarque : Il est possible lors d'une installation ou d'une mise à jour par FTP, de comparer les fichiers installés sur le serveur et ceux présents sur le poste de développement.

## Liste des fichiers à déployer
<a name="liste_des_fichiers_deployer_ELTTEXTE000161"></a>


### Liste des fichiers à installer (installation sur un serveur d'application distant)
<a name="liste_des_fichiers_installer_installation_sur_serveur_application_distant_ELTPARAGRAPHE000024"></a><br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=fichier_install_New.gif)
Par défaut, l'assistant sélectionne tous les éléments nécessaires au fonctionnement du site dynamique WEBDEV :

- bibliothèques (fichiers ".WDL")

- images et pages HTML statiques (répertoire _WEB du projet)

- feuilles de styles (fichiers CSS)

- pages HTML dynamiques (contenu des répertoires de langues du projet)

- ...




Pour chaque fichier (ou groupe de fichiers), il est possible d'indiquer via le menu contextuel si le fichier (ou le groupe de fichiers) doit être : 

- mis à jour. 

- ne pas être modifié sur le serveur (option "Ne rien faire"). Aucune action ne sera effectuée sur les fichiers sélectionnés. 

- supprimé sur le serveur.

- ne jamais être mis à jour sur le serveur.


Il est également possible d'utiliser les boutons correspondants ("Mettre à jour", "Ne rien faire", "Exclure des mises à jour" et "Supprimer").

Le bouton "Comparer" permet d'obtenir la comparaison entre le fichier local et le fichier distant : taille du fichier, date et heure de modification.

**Remarques** : 

- Dans le cas d'une mise à jour, seuls les nouveaux fichiers (ou les fichiers plus récents que ceux précédemment installés) sont marqués comme étant à déployer.

- Il est possible de filtrer les fichiers affichés dans cette fenêtre : fichiers à mettre à jour uniquement, fichiers à supprimer, à exclure ou sans action à réaliser.




**Cas des fichiers de données** :

- Lors d'une première installation, les fichiers de données sont automatiquement proposés par défaut.

- Lors d'une mise à jour, les fichiers de données sont proposés par défaut uniquement s'ils n'existent pas sur le serveur (cas de nouveaux fichiers de données par exemple).

- Si les fichiers de données sont ajoutés manuellement dans les fichiers à installer (bouton "Ajouter des fichiers supplémentaires"), à chaque mise à jour ces fichiers seront automatiquement inclus dans la mise à jour. Ils seront installés uniquement si ces fichiers sont plus récents que ceux existants.

- Pour ne pas installer les fichiers de données, il suffit de les sélectionner et de cliquer sur le bouton "Exclure définitivement les fichiers sélectionnés".




Lors d'un déploiement sur plusieurs serveurs simultanément, il est possible de sélectionner les fichiers à déployer sur chaque serveur. Le répertoire de destination de chaque fichier est commun à tous les serveurs.

### Sélection des fichiers à installer (pack de déploiement ou média physique)
<a name="selection_des_fichiers_installer_pack_deploiement_media_physique_ELTPARAGRAPHE000065"></a><br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=fichier_install_pack.gif)
Par défaut, l'assistant sélectionne tous les éléments nécessaires au fonctionnement du site dynamique WEBDEV :

- bibliothèques (fichiers ".WDL")

- images et pages HTML statique (répertoire _WEB du projet)

- feuilles de styles (fichiers CSS)

- pages HTML dynamiques (contenu des répertoires de langues du projet)

- ...




Cette liste correspond à la liste des fichiers qui seront installés. Il est possible de :

- supprimer des fichiers de cette liste (bouton "Exclure les fichiers"). Les fichiers exclus ne seront plus proposés lors d'une prochaine mise à jour du site. 

- ajouter des fichiers dans cette liste (bouton "Ajouter des fichiers"). Ces fichiers seront automatiquement proposés lors d'une mise à jour du site.

- modifier le répertoire de déploiement. Il suffit de modifier le répertoire destination directement dans la table. 




L'assistant permet également de déployer le site dans toutes les langues (par défaut) ou uniquement pour certaines langues. Si cette option est choisie, une popup s'ouvre permettant de sélectionner la ou les langues voulues.

**Cas des fichiers de données** :

- Par défaut, les fichiers de données sont proposés par l'assistant.

- Si les fichiers de données sont ajoutés manuellement dans les fichiers à installer (bouton "Ajouter des fichiers"), à chaque mise à jour ces fichiers seront automatiquement inclus dans la mise à jour. Ils seront installés uniquement si ces fichiers sont plus récents que ceux existants.

- Pour ne pas les installer, il suffit de les sélectionner et de cliquer sur le bouton "Exclure les fichiers".

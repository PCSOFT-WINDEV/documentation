


## Extraire un élément
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000166"></a>
Pour travailler sur un élément du projet présent dans le gestionnaire de sources, il est nécessaire d'extraire l'élément avant de le modifier.

Deux modes d'extraction sont disponibles :

- Extraction manuelle : L'extraction d'un élément doit être demandée avant de modifier l'élément.

- Extraction automatique : L'extraction de l'élément est automatiquement proposée dès la première tentative de modification de l'élément.




**Pour changer le mode d'extraction** : 

1. Sous le volet "Accueil", dans le groupe "Environnement", déroulez "Options" et sélectionnez "Options générales de WINDEV/WEBDEV/WINDEV Mobile".

2. Sélectionnez l'onglet "Général". L'option "Extraction des éléments à la première modification" permet de passer tous les projets ouverts depuis le logiciel en cours en mode automatique. 
	Attention : Dans cette version, le mode automatique est géré pour les fenêtres, les pages, les requêtes et les états.

3. Validez.

## Extraire un élément
<a name="extraire_element_ELTTEXTE000190"></a>


### Comment extraire un élément ?
<a name="comment_extraire_element_ELTPARAGRAPHE000043"></a>

- **1ère solution : extraction demandée** (disponible quel que soit le mode d'extraction configuré) :

	1. Sélectionnez l'élément dans le volet "Explorateur de projet".

	2. Sélectionnez l'option "Extraire" du menu contextuel de l'élément. 
			Autre solution : sous le volet "GDS", dans le groupe "Elément en cours", cliquez sur "Extraire" (icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=GDS_ico_Extraire.gif)).




- **2ème solution : extraction manuelle** (option "Extraction à la première modification" décochée) :

	1. Ouvrez l'élément sous l'éditeur. Un panneau orange indique que l'élément n'est pas extrait et qu'il ne peut pas être modifié.

	2. Dans le panneau orange, cliquez sur l'icône d'extraction pour modifier l'élément.




- **3ème solution : extraction automatique** (option "Extraction à la première modification" cochée) :

	1. Ouvrez l'élément sous l'éditeur.

	2. Effectuez la modification voulue. A la validation de la modification, le GDS propose d'extraire automatiquement l'élément pour le modifier. Si l'extraction est validée, la modification est réalisée sur l'élément extrait. Si l'extraction est refusée, l'élément n'est pas modifié.






<a name="NOTE2_2"></a>


### Extraire un élément
<a name="extraire_element_ELTPARAGRAPHE000073"></a>Quelle que soit la méthode utilisée pour extraire l'élément, le GDS propose les options suivantes lors de l'extraction : <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Int%E9gration%20dans%20le%20GDS%20-%20HC%20N%B0011.gif)
Le mode d'extraction peut être :

- ***Exclusif*** : personne ne pourra extraire cet élément jusqu'à sa réintégration. L'élément pourra uniquement être extrait pour test.

- ***Pour test*** : l'élément pourra être modifié mais les modifications ne pourront pas être réintégrées.

- ***Multiple*** : l'élément pourra également être extrait par d'autres utilisateurs. Dans ce cas, lors de la réintégration, il sera possible de visualiser les différences entre les différentes versions de l'élément.




Validez. L'élément est ouvert. La barre de titre indique que l'élément est extrait.

**Remarques** :

- Si vous tentez de modifier le code d'un élément qui n'a pas été extrait, le gestionnaire de sources propose automatiquement d'extraire l'élément.  

- Si vous cochez la case "Ne plus afficher ce message dans ce cas (clic sur 'extraire')", il sera possible de réafficher cette fenêtre par les [options de l'administrateur du GDS](../Editeurs/2038023.md).



<a name="NOTE2_3"></a>


### Extraction d'un élément déjà extrait en mode exclusif
<a name="extraction_element_deja_extrait_mode_exclusif_ELTPARAGRAPHE000096"></a>Si l'élément que vous tentez d'extraire a déjà été extrait en mode exclusif par un utilisateur, le gestionnaire de sources affiche la fenêtre suivante : <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=GDS_Deja_Extrait.gif)


Vous pouvez :

- Voir les caractéristiques de l'élément extrait (bouton "Propriétés")

- Envoyer un message à la personne ayant extrait l'élément (bouton "Envoyer un message"). 
	**Rappel** : une messagerie est livrée avec WINDEV, WEBDEV et WINDEV Mobile. Cette messagerie utilise les paramètres spécifiés dans les centres de contrôle (utilisateurs, emplacement des fichiers, ...). Pour afficher la messagerie, sous le volet "Outils", dans le groupe "Centres de Contrôle", cliquez sur "WDBal - Messagerie".

- Extraire l'élément pour tests uniquement. Dans ce cas, les modifications effectuées sur l'élément ne pourront pas être prises en compte dans le projet de référence (présent sur le serveur GDS).

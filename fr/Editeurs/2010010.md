## Récapitulatif
<a name="recapitulatif_ELTTEXTE000201"></a>
Dans une application WINDEV, différentes aides peuvent être proposées aux utilisateurs : 

- Bulle d'aide.

- Barre de message.

- Fenêtre d'aide.

- Bulle d'aide avancée.

- Page d'aide. 

- Fichier CHM.





### Bulle d'aide
<a name="bulle_aide_ELTPARAGRAPHE000021"></a><br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=BULLEAIDE.GIF)


- **Niveau d'aide apportée** : Aide courte (quelques mots)

- **Comment la créer** :

	- Fenêtre de description du champ, onglet "Aide"

	- Programmation : Propriété [Bulle](../Proprietes/2510005.md)




- **Comment lancer l'aide** : Lancement automatique lors du survol du champ.

- **Pour plus de détails** : [Aider l'utilisateur](../Editeurs/2010012.md)





### Bulle d'aide personnalisée
<a name="bulle_aide_personnalisee_ELTPARAGRAPHE000038"></a><br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Bulles_aide_personnalisees%20-%20HC%20N%B0005.gif)


- **Niveau d'aide apportée** : Aide contextuelle.

- **Comment la créer** : 

	- Editeur de projet, pour définir le look.  

	- Fenêtre de description du champ, onglet "Aide" pour le contenu




- **Comment lancer l'aide** : Lancement automatique lors du survol du champ.

- **Pour plus de détails** : [Bulles d'aide personnalisées](../Editeurs/9500233.md)





### Barre de message
<a name="barre_message_ELTPARAGRAPHE000053"></a><br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=BARREMSG.GIF)


- **Niveau d'aide apportée** : Aide simple (une phrase, l'heure, une jauge, ...)

- **Comment la créer** :

	- Fenêtre de description du champ, onglet "Aide"

	- Programmation : Fonction [Message](../WDLang1/3021006.md) ou propriété [Message](../WDLang1/3021006.md)




- **Comment lancer l'aide** : Lancement automatique (la fenêtre doit avoir une barre de message). 
	Les informations spécifiques à un champ s'affichent lorsque le champ a le focus.

- **Pour plus de détails** : [Aider l'utilisateur](../Editeurs/2010012.md)





### Fenêtre d'aide
<a name="fenetre_aide_ELTPARAGRAPHE000075"></a><br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Fenetre_Aide.gif)


- **Niveau d'aide apportée** : Aide pouvant contenir quelques phrases.

- **Comment la créer** : Fenêtre de description du champ, onglet "Aide"

- **Comment lancer l'aide** : Lancement automatique (touche F1).

- **Pour plus de détails** : [Aider l'utilisateur](../Editeurs/2010012.md)





### Page d'aide
<a name="page_aide_ELTPARAGRAPHE000087"></a><br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=P0_Aide_Browser.jpg&type=thumb)


- **Niveau d'aide apportée** : Aide évoluée (pages d'aide, liens, sommaire, images, ...).

- **Comment la créer** : Editeur d'aide : Cliquez sur ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_nouveau.gif) parmi les boutons d'accès rapide. La fenêtre de création d'un nouvel élément s'affiche : cliquez sur "Architecture" puis sur "Aide".

- **Comment lancer l'aide** : Lancement par programmation (fonction [WHelp](../WDLang1/3013053.md))

- **Comment lancer l'aide contextuelle sur une fenêtre, un champ** : 

	1. Page d'aide contextuelle sur un champ :

		- Indiquer le numéro de la page à afficher dans la fenêtre de description du champ.

		- Utiliser la fonction [WHelp](../WDLang1/3013053.md) et la fonction [NumHelp](../WDLang1/3013027.md).




2. Page d'aide contextuelle sur une fenêtre :

	- Utiliser la fonction [WHelp](../WDLang1/3013053.md) en précisant le numéro de la page à afficher.

- **Pour plus de détails :** [Créer l'aide (CHM)](../Editeurs/2010002.md)

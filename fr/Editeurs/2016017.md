


## Rechercher les styles WEBDEV inutilisés
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000096"></a>
Il arrive souvent de créer de nombreux styles de présentation lors de la phase d'élaboration d'un site.

Lorsque le site est terminé, il est courant que la moitié des styles créés ne soit pas utilisée. Cela alourdit les pages et rend l'édition plus fastidieuse.

WEBDEV propose donc de rechercher les styles inutilisés afin d'optimiser vos sites et vos projets.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000120"></a>


### Rechercher les styles inutilisés et les supprimer
<a name="rechercher_les_styles_inutilises_les_supprimer_ELTPARAGRAPHE000022"></a>Pour rechercher les styles inutilisés : 

1. Sous le volet "Projet", dans le groupe "Audit et performances", déroulez "Audit statique" et sélectionnez "Déclencher l'audit d'édition". Dans l'assistant, sélectionnez le type d'audit à réaliser.

2. La fenêtre qui s'affiche liste tous les problèmes potentiels de votre projet. Si des styles sont inutilisés, il sont présents dans la catégorie "Nettoyage". 

3. Si des styles inutilisés sont trouvés, cliquez sur le bouton [...]. La fenêtre qui s'affiche liste tous les styles présents dans votre projet WEBDEV et inutilisés. La liste présente les styles WEBDEV et CSS trouvés dans les différentes feuilles de styles liées au projet, pour chaque type de champ.




Pour supprimer un ou plusieurs styles :

1. Dans la fenêtre des styles inutilisés, sélectionnez le ou les styles à supprimer (une multisélection est possible).

2. Cliquez sur le bouton "Supprimer" (ou utilisez l'option "Supprimer les styles sélectionnés" du menu contextuel).

3. Le style est immédiatement supprimé de la liste. Il ne sera pas pris en compte lors de la prochaine génération des pages.




## Personnaliser le guide 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000097"></a>
Le guide est un puissant moteur de recherche intégré à WINDEV, permettant de rechercher tous les documents référencés concernant un domaine.

La personnalisation du guide vous permet d'ajouter et de référencer vos propres documents, applications, etc.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000121"></a>


### Personnaliser le guide
<a name="personnaliser_guide_ELTPARAGRAPHE000020"></a>Pour personnaliser le guide : 

1. Affichez le guide : sous le volet "Accueil", dans le groupe "Aide en ligne", déroulez "Aide" et sélectionnez "Consulter le guide".

2. Sélectionnez l'option "Afficher les options avancées".<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Guide_perso.gif&type=thumb)
Les boutons Imprimer, Propriétés et Ajouter s'affichent.

3. Cliquez sur le bouton "Ajouter". La fenêtre de description d'un document apparaît. Indiquez :

	- Dans l'onglet "Description" , le titre et la description rapide du document.

	- Dans l'onglet "Classement", le type du document référencé et sa catégorie.

	- Dans l'onglet "Source", indiquez les référence du document (exemple de code, répertoire de l'application, fichier d'aide, ...).

	- Dans l'onglet "Mots-clé", saisissez les mots clés affectés à ce document. Ces mots-clé permettront de retrouver facilement le document.




4. Validez. Ces caractéristiques peuvent être modifiées à tout moment pour le document sélectionné à l'aide du bouton "Propriétés".


## Style d'un champ : Caractéristiques de la police
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000086"></a>
De nombreuses options de style sont modifiables via l'onglet "Style" de la fenêtre de description des champs. Il est notamment possible de modifier les caractéristiques de la police, que ce soit la police du libellé du champ, de la zone de saisie, ou encore des éléments d'une liste. 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Caractéristiques de la police
<a name="caracteristiques_police_ELTTEXTE000110"></a>
Pour tous les éléments d'un champ affichant un texte, il est possible de configurer les éléments suivants depuis la fenêtre de description du champ (onglet "Style) : 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Stylle_surcharge%202.gif)


- **Police** : Famille de polices à utiliser. 

- **Position** : permet de déterminer la position du texte par rapport au champ. 

- **Couleur police** : Permet de choisir la couleur de la police utilisée. 

- **Taille** : Taille de la police à utiliser. Il est possible de choisir l'unité à utiliser : pixel CSS, point ou point WINDEV. 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Style_Unit%E9_interligne%20-%20HC%20N%B0002%201.gif)


- Options de mise en forme : Gras, Italique, Souligné, Barré. 

- Options avancées de mise en forme (bouton ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Style_Unit%E9_interligne%20-%20HC%20N%B0001%201.gif)
) : Dans la fenêtre qui s'affiche, il est possible de paramétrer : 

	- l'interligne (propriété [Interligne](../Proprietes/1410088366.md)),

	- l'espacement entre les caractères (propriété [Interlettre](../Proprietes/1410088367.md)),

	- l'inclinaison, uniquement pour les champs Libellé. 




- **Ombre** : Permet de définir si une ombre doit être appliquée au libellé, et dans quel sens. Il est possible de définir la couleur de l'ombre via l'option "Coul. ombre".


## Aérer une sélection de champs
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000104"></a>
Pour créer des UI actuelles, il est nécessaire d'espacer les champs de vos fenêtres. Avec WINDEV et WINDEV Mobile, il suffit d'indiquer l'espacement voulu et les champs sont automatiquement déplacés.  

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000128"></a>
Pour augmenter l'espacement entre les champs : 

1. Sélectionnez les champs concernés. 

2. Sous le volet "Alignement", dans le groupe "Autres alignements", déroulez "Agrandir
Aérer" et sélectionnez "Aérer la sélection". 

3. Dans la fenêtre qui s'affiche, indiquez les paramètres permettant d'aérer l'espace entre les champs sélectionnés : 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Aerer_la_selection%20-%20HC%20N%B0003.gif)
 Il est possible de : 

	- Indiquer le pas d'aération en largeur et/ou en hauteur. L'icône 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Aerer_la_selection%20-%20HC%20N%B0003%201.gif)
 permet de lier les deux valeurs.  

	- Indiquer si nécessaire la tolérance voulue. Cette tolérance permet d'affiner la détection des champs à aérer lorsque ceux-ci ne sont pas correctement alignés ou s'ils se chevauchent. 


 L'icône 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Aerer_la_selection%20-%20HC%20N%B0003%202.gif)
 permet d'obtenir un aperçu de la modification en temps réel dans votre fenêtre. 

4. Validez. Les paramètres indiqués sont appliqués aux champs sélectionnés. 




**Remarques** : 

- L'espacement se fait vers la droite et vers le bas. 

- Attention : La taille de la fenêtre n'est pas modifiée.

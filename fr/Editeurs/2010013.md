<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000135"></a>
Les utilisateurs sont familiers avec les sommaires, car le sommaire est le premier élément parcouru dans une documentation papier.

Le sommaire permet à l'utilisateur d'avoir une idée sur l'organisation de l'aide.

Les utilisateurs peuvent parcourir le sommaire pour trouver une page d'aide spécifique.

Le sommaire utilise deux catégories d'éléments : 

- les ***livrets***, qui permettent de regrouper les pages d'aides par chapitres ou sections. Dans la majorité des cas, les utilisateurs cliquent sur le livret pour afficher les pages du livret.

- les ***pages***, qui représentent les pages d'aide individuelles. Les utilisateurs cliquent sur la page pour afficher la page d'aide correspondante.




Dans l'aide au format CHM, il est possible d'organiser les pages et les livrets dans n'importe quel ordre.

**Remarques** : 

- Sous l'éditeur d'aide, il est possible de visualiser le sommaire en édition. Toutes les pages d'aide sont affichées sous l'éditeur. Ces pages d'aide peuvent être déplacées et ré-organisées. Si la page d'aide ne doit pas apparaître dans le sommaire en exécution, il est nécessaire de le spécifier dans les [caractéristiques d'une page d'aide](../Editeurs/2010024.md).

- Aide générée au format PDF : le sommaire est utilisé pour créer des signets dans le ficher PDF.

## Comment le faire ?
<a name="comment_faire_ELTTEXTE000159"></a>


### Gérer les livrets
<a name="gerer_les_livrets_ELTPARAGRAPHE000037"></a>Pour créer un livret :

1. Affichez le sommaire (onglet "Table des matières" ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Aide_Ico_sommaire.gif)).

2. Sélectionnez l'option "Nouveau .. Nouveau livret" du menu contextuel. 
	Autre possibilité :  Sous le volet "Modification", dans le groupe "Table des matières", déroulez "Nouveau" et sélectionnez "Nouveau livret".

3. Un livret est créé. Saisissez le libellé du livret.




Pour supprimer un livret :

1. Supprimez ou déplacez toutes les pages du livret à supprimer.

2. Sélectionnez le livret à supprimer.

3. Appuyez sur la touche Suppr.



<a name="NOTE2_3"></a>


### Gérer les pages dans les livrets
<a name="gerer_les_pages_dans_les_livrets_ELTPARAGRAPHE000070"></a>Pour faire apparaître une page à partir d'un livret :

1. Sélectionnez la page à associer.

2. Réalisez un "Drag and Drop" de cette page vers le livret.




Pour enlever une page d'un livret :

1. Sélectionnez la page à enlever.

2. Sous le volet "Modification", dans le groupe "Table des matières", utilisez les boutons fléchés pour déplacer les pages vers d'autres livrets.

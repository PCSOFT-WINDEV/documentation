


## Presse-papiers avec historique
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000063"></a>
Le presse-papiers mémorise les 10 derniers contenus coupés ou copiés (depuis n'importe quel éditeur). 

La combinaison de touches Ctrl + Maj + V (à la place de Ctrl + V) affiche un menu contextuel avec les 10 dernières valeurs copiées ou coupées : il suffit de sélectionner l'élément à copier. 

Le collage s'adapte à l'éditeur : par exemple, si un champ copié sous l'éditeur de fenêtres est collé dans l'éditeur de code, seul le nom du champ sera collé. <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=pressepaier_historise.gif)


## Exécutable : Sauvegarde du projet
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000087"></a>
A la fin de l'assistant de création de l'exécutable, WINDEV vous propose de réaliser automatiquement une sauvegarde des sources de votre projet.

Cette option permet de sauvegarder automatiquement votre projet à chaque version majeure, déployée en clientèle.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Mise en place
<a name="mise_place_ELTTEXTE000111"></a>


### Les différentes étapes
<a name="les_differentes_etapes_ELTPARAGRAPHE000021"></a>Si lors de la création de l'exécutable, vous sélectionnez l'option "Oui : Lancer l'assistant de sauvegarde du projet", l'assistant sera automatiquement lancé à la fin de la création de l'exécutable.

Cet assistant permet de :

1. Sélectionner les répertoires du projet à inclure dans la sauvegarde (analyse, exe, ...)

2. Choisir le type de sauvegarde à effectuer :

	- Fichier de sauvegarde compressé (format zip). Dans ce cas, il est nécessaire d'indiquer le nom du fichier de sauvegarde.

	- Copie de sauvegarde. Dans ce cas, il est nécessaire d'indiquer le répertoire de sauvegarde.








<a name="NOTE2_2"></a>


### Rappel
<a name="rappel_ELTPARAGRAPHE000037"></a>En cas de problème physique sur le disque, il est conseillé d'effectuer une sauvegarde sur un disque différent du disque de travail.

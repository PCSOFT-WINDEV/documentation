


## WDInst : inclure un "autorun" dans une installation sur CD 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000160"></a>
Lors de l'insertion du CD d'installation sur le poste d'installation, il est possible d'exécuter ou non un "autorun". Cet "autorun" permet de lancer automatiquement :

- soit le programme d'installation.

- soit une fenêtre spécifique de menu. L'éditeur d'installation WDInst permet de paramétrer cette fenêtre.
	Ce menu peut par exemple lancer le programme d'installation, afficher un fichier "Lisez-Moi", ouvrir un site Internet, ...




Par défaut, aucun "autorun" n'est exécuté lors de l'insertion du CD.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000184"></a>


### Gestion de l'autorun
<a name="gestion_autorun_ELTPARAGRAPHE000025"></a>La gestion d'un "autorun" est réalisée sous l'éditeur d'installation WDInst. 

Pour lancer WDInst, sous le volet "Outils", dans le groupe "Utilitaires", cliquez sur "WDInst".

L'assistant de création du programme d'installation ne permet pas de gérer l' "autorun".
<a name="NOTE2_2"></a>


### Inclure un "autorun" sur le CD d'installation
<a name="inclure_autorun_sur_installation_ELTPARAGRAPHE000048"></a>Pour inclure un "autorun" sur le CD d'installation :

1. Sous WDInst, sélectionnez l'option "Paramètres de l'installation .. Paramètres avancés".

2. Sélectionnez le type d'"autorun" à inclure (option "Action effectuée à l'insertion du CD"). Il est possible de : 

	- lancer automatiquement l'installeur.

	- lancer un menu d'exécution automatique. 







**Remarque** : Par défaut, aucun "autorun" n'est inclus.
<a name="NOTE2_3"></a>


### Paramétrer le menu de l'"autorun"
<a name="parametrer_menu_autorun_ELTPARAGRAPHE000067"></a>Pour paramétrer le menu de l'"autorun" :

1. Sous WDInst, sélectionnez l'option "Paramètres de l'installation .. Paramètres avancés".

2. Sélectionnez l'option "Lancement d'un menu d'exécution automatique". 

3. Cliquez sur le bouton "Paramètres".

4. Configurez les différentes options du menu dans les différents onglets de configuration :

	- **Onglet Général** : Permet de définir :

		- le titre de la fenêtre. Ce titre sera affiché dans la barre de titre de la fenêtre. Saisissez par exemple le nom de l'application : "WDScooter".

		- le grand titre et sa couleur. Ce titre sera la première ligne affichée dans la fenêtre. Saisissez par exemple le rôle de l'application : "Gestion des pièces en stock".
						**Attention** : Le nombre de caractères maximum est limité. En prenant les lettres "i" et "m" comme référence, le grand titre peut contenir au maximum entre 15 et 60 caractères.

		- le sous-titre et sa couleur. Ce titre sera affiché sous le grand titre. Saisissez par exemple un message de bienvenue : "Bienvenue dans le programme d'installation".
						**Attention** : Le nombre de caractères maximum est limité. En prenant les lettres "i" et "m" comme référence, le sous-titre peut contenir au maximum entre 30 et 100 caractères.

		- le fond de la fenêtre. Ce paramètre peut correspondre :

			- soit à une *couleur unie*. Sélectionnez la couleur de fond.

			- soit à une *image*. Cette image peut être répétée ou étirée. Sélectionnez l'image à utiliser.




- **Onglet Boutons** : La fenêtre menu peut contenir jusqu'à 8 **boutons**.

	1. Pour chaque bouton, spécifiez :

		- le ***libellé***. Ce libellé sera centré dans le bouton.

		- la ***description***. Cette description sera affichée dans la partie gauche de la fenêtre lors du survol du bouton correspondant.

		- la ***commande*** exécutée ou le ***document*** ouvert par ce bouton. Chaque bouton peut :

			- lancer un fichier exécutable. Par exemple : "Install.EXE". Pour être lancé, ce fichier doit être disponible sur le CD ou présent dans un des répertoires définis dans le PATH du poste d'installation.

			- afficher un document. Par exemple : "LisezMoi.DOC". Pour être affiché, ce fichier doit être disponible sur le CD ou présent dans un des répertoires définis dans le PATH du poste d'installation.

			- ouvrir un site Internet. Par exemple : "http://www.pcsoft.fr". Pour ouvrir ce site, le poste d'installation doit disposer d'une connexion permanente à Internet.

			- lancer un explorateur. Par exemple "E:\\MonAppli". Il suffit d'indiquer le répertoire sur lequel l'explorateur doit s'ouvrir dans la colonne "Commande / document".




A tout moment, le bouton "Tester" permet de tester les paramètres choisis pour le programme d'exécution automatique du CD. Le menu s'affiche avec les différents boutons spécifiés, les images et les couleurs paramétrées.

## Comment se déconnecter du groupware utilisateur ?
<a name="comment_deconnecter_groupware_utilisateur_ELTTEXTE000102"></a>
Le but : Faire un bouton avec déconnexion de l'utilisateur en cours du groupware utilisateur et retour sur la page de login.

Solution : Pour se déconnecter, il suffit d'utiliser la fonction [gpwDéconnecteUtilisateur](../WDLang6/1000021100.md). 

- En mode intégré, la déconnexion conserve des droits, ceux définis pour l'utilisateur non connecté.

- En mode non intégré, pour conserver les droits, il suffit de :

	- Créer un utilisateur "non connecté" par exemple. 

	- Définir des droits. 

	- Dans le code, utiliser : 
			
		```wl
		gpwDéconnecteUtilisateur()
		gpwLogin("non connecté", "mot de passe")
		// Rafraîchissement de la page en cours
		PageAffiche()
		```

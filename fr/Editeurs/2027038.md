
## Volet "Aide syntaxique"
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000090"></a>
Le volet "Aide syntaxique" permet d'afficher un résumé de l'aide de la fonction en cours de saisie. Vous pouvez ainsi continuer à saisir votre code tout en ayant sous les yeux un résumé de la fonction et le détail de la syntaxe.
<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Volet_Aide_syntaxique%20-%20HC%20N%B0001.gif)


Ce volet comporte : 

- la valeur de retour, 

- les paramètres, leur type et leur utilité, 

- un lien vers la page d'aide complète, 

- un lien vers le forum professionnel. 






<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Afficher le volet "Aide syntaxique"
<a name="afficher_volet_aide_syntaxique_ELTTEXTE000114"></a>
**Pour afficher le volet "Aide syntaxique"** :  sous le volet "Accueil", dans le groupe "Environnement", déroulez "Volets" et sélectionnez "Volets" puis "Aide syntaxique".

Pour plus de détails sur les manipulations des volets, consultez [Manipuler les volets ancrables](../Editeurs/2027001.md).

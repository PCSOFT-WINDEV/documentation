


## Eléments d'ambiance
			





## Présentation
<a name="presentation_ELTTEXTE000108"></a>
Les éléments d'ambiance permettent de changer très rapidement l'aspect visuel d'un projet sans avoir besoin de retoucher chaque page et chaque modèle de pages.

Les éléments d'ambiances sont composées de trois éléments :

- **[les modèles de pages prédéfinis](../Editeurs/2016019.md)** : ils sont constitués de champs (essentiellement des [cellules](../WDChamp/1013096.md)) et définissent une mise en page par défaut pour les pages du projet.

- **[l'ambiance](../Editeurs/2016020.md)** : elle est constituée d'une feuille de style normalisée pouvant être échangée avec celle de n'importe quelle autre ambiance. Une ambiance s'applique sur une page, un modèle de page ou sur l'ensemble d'un projet.

- **[la palette de couleurs](../Editeurs/2016021.md)** : elle est constitué d'une série de couleurs harmonieuses entre elles et dont les noms sont normalisés pour pouvoir être interchangés. Elle s'applique sur les styles de l'ambiance pour en redéfinir les tons.






## Utilisation des éléments d'ambiance
<a name="utilisation_des_elements_ambiance_ELTTEXTE000132"></a>


### Utilisation dans un nouveau projet
<a name="utilisation_dans_nouveau_projet_ELTPARAGRAPHE000030"></a>Lors de la création d'un nouveau projet, l'assistant de création de projet permet de choisir une ambiance et une palette de couleur. Celles-ci seront associées au projet et utilisées par défaut dans toutes les pages.

L'ambiance d'un projet peut être changée à tout moment dans l'éditeur. Pour cela :

1. Affichez la description du projet : sous le volet "Projet", dans le groupe "Projet", cliquez sur "Description".

2. Sélectionnez l'onglet "Ambiance Styles".

3. Choisissez la nouvelle ambiance et sa palette de couleurs.

4. Validez la fenêtre de description du projet. 




Lors de la création des pages, vous pourrez choisir le ou les modèles de pages prédéfinis à utiliser.


### Utilisation dans un projet existant
<a name="utilisation_dans_projet_existant_ELTPARAGRAPHE000056"></a>Il est possible d'adapter facilement un projet existant pour utiliser les éléments d'ambiance. Cette opération est détaillée dans [Adapter un projet pour utiliser les ambiances](../Editeurs/2016018.md).

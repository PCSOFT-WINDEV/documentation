<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000292"></a>
L'assistant d'installation permet d'inclure différents modules complémentaires dans l'installation. Ces modules complémentaires permettent d'ajouter des options supplémentaires au programme d'installation et d'accéder aux paramètres avancés de l'installation. Il est possible de :

- [Installer l'aide pour le Macro Code Utilisateur (MCU)](#NOTE2_1).

- Intégrer l'installation du [logiciel Etats & Requêtes](../Presentation/3088003.md) avec l'application en cours. Pour plus de détails, consultez [Diffuser le logiciel Etats & Requêtes](../Presentation/3088002.md).

- [Afficher une licence d'utilisation lors de l'installation](#NOTE2_3).

- [Afficher un fichier 'lisez-moi' à la fin de l'installation](#NOTE2_4).

- [Choisir des outils optionnels à inclure dans l'installation](#NOTE2_5).

- [Inclure le désinstallateur](#NOTE2_6).

- [Installer des polices de caractères](#NOTE2_7).

## Modules additionnels
<a name="modules_additionnels_ELTTEXTE000322"></a>


### Installer l'aide pour le Macro-Code Utilisateur (MCU)
<a name="installer_aide_pour_macrocode_utilisateur_mcu_ELTPARAGRAPHE000048"></a>Cette option permet d'inclure l'aide des fonctions, propriétés, mots-clés du WLangage dans votre application. Cette aide pourra être appelée directement depuis la fenêtre de saisie de code du MCU.

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Inst_Aide_MCU.gif)
Vous pouvez :

- soit inclure l'aide du WLangage dans l'installation. Dans ce cas, le programme d'installation sera plus important.

- soit télécharger l'aide sur le poste client pendant l'installation (une connexion Internet est nécessaire).




**Remarque** : Si vous installez les fichiers d'aide du MCU, l'archivage de ces fichiers d'aide (création d'un zip) est effectué à chaque nouvelle mise à jour de WINDEV. Cette opération n'est pas effectuée à chaque génération de l'installation de votre application.
<a name="NOTE2_2"></a>


### Intégrer l'installation de Etats & Requêtes
<a name="integrer_installation_etats_requetes_ELTPARAGRAPHE000062"></a>Cette option permet d'inclure l'installation du logiciel Etats & Requêtes dans l'installation de votre application. 


![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Inst_EtatsRequete.gif)


Dans l'écran de configuration de l'installation de "Etats & Requêtes" : 

1. Cochez l'option "Inclure Etats et Requêtes dans l'installation". Le package d'installation du logiciel Etats et Requêtes est automatiquement recherché. Ce pack d'installation est présent dans le sous-répertoire "Install\\Etats et Requêtes" du répertoire d'installation de WINDEV.
	**Attention** : Incorporer le logiciel Etats et Requêtes augmente la taille de votre installation de façon non négligeable (plus de 600 Mo).
	Il est également possible de distribuer le pack d'installation Etats et Requêtes séparément.
	Si le logiciel Etats et Requêtes n'est pas inclus dans l'installation, il pourra être téléchargé directement depuis l'application (soit par le menu automatique, soit depuis le visualisateur de rapports, soit par la fonction [LanceEtatsEtRequêtes](../WDLang1/3013067.md)). Par défaut, le logiciel est téléchargé sur un site FTP de PC SOFT. Il est possible de personnaliser l'adresse de téléchargement du logiciel (par exemple site d'une société, ...). Pour plus de détails, consultez [Etats & Requêtes : Personnaliser l'adresse de téléchargement](../Presentation/3088002.md).

2. Spécifiez si nécessaire les répertoires correspondant aux états et requêtes modifiés ou créés par l'utilisateur final :

	- le répertoire des états et requêtes personnels correspond au répertoire des états et requêtes visibles uniquement par l'utilisateur qui les a créés.

	- le répertoire des états et requêtes partagés correspond au répertoire des états et requêtes visibles par tous les utilisateurs de l'application.




3. Autorisez ou non la modification de ces répertoires lors de l'installation de l'application.

### Afficher une licence d'utilisation
<a name="afficher_une_licence_utilisation_ELTPARAGRAPHE000092"></a>Si l'option "Afficher une licence d'utilisation lors de l'installation" est cochée, vous devrez spécifier le fichier contenant la licence d'utilisation dans la suite de l'assistant. Ce fichier pourra être au format texte (".TXT") ou au format RTF (".RTF" ou ".WRI" par exemple).

Si le programme d'installation est multilingue, il sera nécessaire de sélectionner une licence pour chaque langue gérée par le programme d'installation.

La licence d'utilisation de votre application s'affichera au début de l'installation. L'utilisateur devra accepter les termes de cette licence pour poursuivre l'installation.
<a name="NOTE2_4"></a>


### Afficher un fichier 'Lisez Moi'
<a name="afficher_fichier_lisez_moi_ELTPARAGRAPHE000103"></a>Si l'option "Afficher un fichier 'lisez-moi' à la fin de l'installation" est cochée, vous devrez spécifier le fichier "lisez-moi" dans la suite de l'assistant. Ce fichier pourra être :

- soit au format texte (".TXT"),

- soit au format Word (".DOC"),

- soit au format HTML (".HTM" ou ".HTML"),

- soit au format RTF (".RTF" ou ".WRI" par exemple),

- soit dans un format quelconque si celui-ci est reconnu sur le poste utilisateurs.




Si le programme d'installation est multilingue, il sera nécessaire de sélectionner un fichier "lisez-moi" pour chaque langue gérée par le programme d'installation.

Le fichier "lisez-moi" pourra s'afficher à la fin de l'installation. A la fin de l'installation, l'utilisateur devra cocher l'option "Afficher le fichier "Lisez-Moi"" pour lire ce fichier.
<a name="NOTE2_5"></a>


### Choisir des outils optionnels à inclure dans l'installation
<a name="choisir_des_outils_optionnels_inclure_dans_installation_ELTPARAGRAPHE000120"></a>Certains outils WINDEV sont diffusables librement et gratuitement avec vos applications :

| Outils | Description |
| --- | --- |
| DplViewer | Permet de visualiser les duplicata. |
| [ReplicSynchro](../ReplicSynchro/3526550.md) | Permet de gérer la Réplication Universelle assistée. |
| [WDConver](../WDConver/3512006.md) | Convertit des fichiers externes vers des fichiers de données HFSQL. |
| [WDDebug](../Editeurs/2019021.md) | Permet de débugger une application à distance |
| [WDJournal](../WDJournal/3516012.md) | Sauvegarde et restaure les fichiers de données au format HFSQL et gère les journaux. |
| [WDOptimiseur](../WDOptimiseur/3519002.md) | Optimise l'accès aux fichiers HFSQL (ré-indexation, corrections des mémos, ...). |
| [WDReplic](../WDReplic/3520016.md) | Gère la réplication des données entre plusieurs postes utilisant la même application. |
| [WDSQL](../WDSQL/3521011.md) | Permet de définir une connexion et de créer des requêtes en code SQL sur cette connexion. |
| [WDScript](../WDScript/1410087244.md) | Permet de créer et d'éditer des scripts WLangage. |
| WDTelDiagno | Liste les lignes TAPI disponibles, et précise la ligne qui sera utilisée par défaut lors de l'utilisation des fonctions de téléphonie. |
| [WDTrans](../WDTrans/3524005.md) | Permet d'annuler des transactions ou de libérer des enregistrements en transaction. |


Pour ajouter un ou plusieurs de ces outils dans le programme d'installation, cochez l'option "Inclure des outils optionnels dans l'installation" et sélectionnez les outils à inclure dans la suite de l'assistant.

Pour limiter la multiplication des applications sur le poste de l'utilisateur final, les fichiers nécessaires à ces outils seront installés dans le répertoire "\\Program Files\\Fichiers communs\\PC SOFT\\28.0". Le chemin du répertoire des outils optionnels peut être connu grâce à la fonction [SysRep](../WDLang1/3073026.md). Par exemple : 


```wl
SysRep(srProgramFilesCommun) + "\PC SOFT\28.0"
```


Si un raccourci de l'application est créé dans le menu "Démarrer", le raccourci de chacun des outils installés sur les postes utilisateurs sera également ajouté dans le menu "Démarrer" (dans le même groupe de programmes que celui de l'application).

Par défaut, si l'application est associée à une base de données HFSQL, WDOptimiseur sera installé.

### Inclure le désinstallateur
<a name="inclure_desinstallateur_ELTPARAGRAPHE000205"></a>Si l'option "Inclure le désinstallateur" est cochée, le désinstallateur sera automatiquement installé sur les postes utilisateurs.

Après son installation, l'application apparaîtra dans la liste des programmes installés sur le système en cours (option "Ajout/Suppression de programmes" dans le panneau de configuration).

Remarque : Le fichier "uninst.INF" sera créé dans le répertoire d'installation de l'application. Ce fichier liste tous les fichiers de l'application installés sur les postes utilisateurs.

**Pour désinstaller l'application**, il suffira de :

1. Sélectionner l'application dans la liste des programmes.

2. Cliquer sur le bouton "Modifier/Supprimer". 
	Les éléments suivants seront supprimés :

	- Les fichiers listés dans le fichier "uninst.INF".

	- Les entrées dans la base de registre créées lors de l'installation de l'application.

	- Les raccourcis de l'application (sur le bureau et dans le menu "Démarrer") s'ils existent.

	- Les fichiers de données (".FIC", ".NDX" et ".MMO") après confirmation de l'utilisateur.

	- Les fichiers sauvegardés lors des différentes installations (s'ils existent) après confirmation de l'utilisateur.

	- Les répertoires dont les fichiers sont supprimés lors de la désinstallation.







**Cas particulier** : Si l'application crée des fichiers de données dans un répertoire spécifique (répertoire différent des sous-répertoires de l'application), ces fichiers de données ne sont pas supprimés automatiquement lors de la désinstallation.

**Remarque** : Pour pouvoir désinstaller l'application, le désinstallateur utilise une entrée dans la base de registre. L'assistant de création du programme d'installation permet de modifier cet identifiant. Pour plus de détails, consultez [Modifier l'identifiant utilisé pour désinstaller l'application](../Editeurs/2028062.md).
<a name="NOTE2_7"></a>


### Installer des polices de caractères
<a name="installer_des_polices_caracteres_ELTPARAGRAPHE000239"></a>Si l'option "Installer des polices de caractères" est cochée, les polices sélectionnées seront déployées sur la machine cliente. Dans l'écran qui s'affiche, ajoutez les fichiers TTF ou OTF correspondant aux polices à installer. 

Lors de l'installation de l'application, ces polices seront automatiquement installées sur le poste client. 

Remarque : Veuillez vérifier que vous possédez bien le droit de diffuser ces polices.

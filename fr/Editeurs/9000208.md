
## Outils bureautiques
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000072"></a>
WINDEV propose plusieurs champs "Bureautiques" permettant à l'utilisateur final d'utiliser ses outils préférés tout en restant dans leur application. 

Cette même fonctionnalité est désormais disponible pour les développeurs : vous pouvez utiliser les outils bureautiques directement depuis l'interface de votre logiciel préféré. Il suffit de cliquer sur l'icône correspondant dans le volet "Outils" du menu de WINDEV, WEBDEV et WINDEV Mobile. 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Les différents outils disponibles
<a name="les_differents_outils_disponibles_ELTTEXTE000096"></a>
5 outils bureautiques sont à votre disposition : 

- WD Docs : outil de type Traitement de texte, permettant de réaliser rapidement des documents au format Docx. Il est possible d'ouvrir des documents existants ou d'en créer. 

- WD Tableur : outil de type Tableur, permettant de réaliser rapidement des documents au format Xlsx. Il est possible d'ouvrir des documents existants ou d'en créer. 

- WD PDF : outil de type Lecteur PDF permettant de lire des documents au format PDF. 

- WD Pic : outil de type Editeur d'images permettant d'ouvrir, modifier, créer des images rapidement.  

- WD Diagramme : outil de type Editeur de diagrammes permettant de créer, d'ouvrir et de modifier des diagrammes.

## Présentation
<a name="presentation_ELTTEXTE000120"></a>
Le volet "Résultats des tests" permet de visualiser le résultat des différents tests autonomes d'un projet. 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=volet%20resultats%20de%20tests%20-%20HC%20N%B0001.gif)

## Afficher le volet "Résultats des tests"
<a name="afficher_volet_resultats_des_tests_ELTTEXTE000144"></a>
**Pour afficher le volet "Résultats des tests"** : 

- sous le volet "Accueil", dans le groupe "Environnement", déroulez "Volets" et sélectionnez "Volets" puis "Résultats des tests". 

- sous le volet "Tests automatiques", dans le groupe "Tests", cliquez sur "Voir les résultats". 




Pour plus de détails sur les manipulations des volets, consultez [Manipuler les volets ancrables](../Editeurs/2027001.md).

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Informations affichées
<a name="informations_affichees_ELTTEXTE000168"></a>
Le volet "Résultats des tests" affiche : 

- les différents tests et scénarios de votre projet (arborescence dans la partie gauche du volet). 

- pour chaque scénario, les résultats du dernier passage. Il est également possible de visualiser les résultats des passages précédents en choisissant la date et l'heure du passage du test à visualiser. 

- en cas d'erreur, une image correspondant à l'interface au moment de l'erreur. Cette copie d'écran permet de voir l'avancement du test lors du déclenchement de l'erreur ainsi que le contenu de l'interface. 




Le compte-rendu du test affiche : 

- les messages affichés par les fonctions spécifiques des tests (par exemple, fonction [TestEcritRésultat](../WDLang1/1000013001.md)),

- les erreurs du test, 

- les itérations testées, ...




Remarque : Les résultats des tests sont conservés dans la base de données des Centres de Contrôle. Seuls les 20 derniers passages sont affichés dans la combo de sélection du passage.




## Galerie d'images du projet
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000137"></a>
La galerie d'images du projet permet de visualiser d'un clic l'ensemble des images d'un projet. 

Cette galerie propose différentes fonctionnalités : 

- Une image présente dans un des répertoires du projet qui n'est pas utilisée est signalée. Si nécessaire, il est possible de la supprimer.

- Les images en doublon sont signalées : 

	- images strictement identiques, 

	- images binairement identiques mais avec un nom différent, ou stockées dans des sous-répertoires différents. 


 Supprimer des images permet d'alléger la taille des projets.

- Les images qui ne sont pas historisées (versionnées) dans le GDS sont également signalées.

## Fonctionnalités disponibles
<a name="fonctionnalites_disponibles_ELTTEXTE000185"></a>


### Affichage des images
<a name="affichage_des_images_ELTPARAGRAPHE000049"></a>La galerie d'images du projet permet de : 

- Lister et visualiser les images présentes dans les dossiers du projet en cours.  <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Galerie_projet%20-%20HC%20N%B0001.gif&type=thumb)
Il est possible de sélectionner uniquement le dossier voulu, de rechercher une image ou de trier les images affichées. Il est également possible d'afficher les images sous forme de liste avec vignettes ou de liste détaillée grâce aux icônes ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Galerie_projet%20-%20HC%20N%B0001%201.gif). 





### Manipulation des fichiers
<a name="manipulation_des_fichiers_ELTPARAGRAPHE000060"></a>La galerie d'images du projet permet de :

- Renommer, dupliquer ou supprimer le fichier sélectionné grâce aux options du groupe "Fichiers" présentes dans le ruban. 

- Imprimer l'image sélectionnée (Ctrl + P). La fenêtre de configuration et d'aperçu de l'impression s'affiche automatiquement. 

- Afficher l'image. Cette option permet d'afficher l'image en taille réelle dans la zone d'affichage des fichiers. 

- Editer l'image. Cette option permet d'éditer l'image sélectionnée dans l'éditeur d'images du produit utilisé. 
	Attention : Cette option ferme la galerie d'images du projet. 





### Sélection des fichiers
<a name="selection_des_fichiers_ELTPARAGRAPHE000072"></a>La galerie d'images du projet permet de sélectionner les images affichées via plusieurs filtres : 

- L'option "5 états" permet d'afficher uniquement les images au format "5 états" présentes dans les répertoires du projet. 

- L'option "Multi DPI" permet de lister uniquement les fichiers des images correspondant à la gestion du multi DPI. 

- L'option "Hors GDS" permet de lister uniquement les images qui ne sont pas historisées (versionnées) dans le GDS.

- L'option "Inutilisées" permet de lister uniquement les images qui ne sont pas utilisées par le projet. <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Galerie_projet%20-%20HC%20N%B0004.gif&type=thumb)


- L'option "Doublons" permet de lister les images potentiellement en double dans les différents répertoires du projet. <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Galerie_projet%20-%20HC%20N%B0003.gif&type=thumb)






### Nettoyage du projet
<a name="nettoyage_projet_ELTPARAGRAPHE000085"></a>L'option "Nettoyer" permet de lancer l'assistant de nettoyage du projet pour supprimer les fichiers inutilisés. Pour plus de détails, consultez [Nettoyage de projet](../Editeurs/9000068.md).

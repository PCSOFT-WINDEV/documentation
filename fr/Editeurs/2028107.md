
## Déployer dans le cloud pour applications PC SOFT
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000130"></a>
PC SOFT propose d'héberger vos sites dans le CLOUD pour les applications PC SOFT. L'installation s'effectue simplement, directement depuis l'environnement. 

Ainsi, une fois votre site terminé, il est possible de l'installer "en réel" directement sur votre plateforme présente dans le CLOUD. 

Pour plus d'informations, consultez [http://www.pcscloud.net](http://www.pcscloud.net).

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Les différentes étapes
<a name="les_differentes_etapes_ELTTEXTE000154"></a>
Les différentes étapes de l'installation d'un site WEBDEV sur une plateforme CLOUD PC SOFT sont les suivantes :

1. [Création si nécessaire d'un compte CLOUD](#NOTE3).

2. [Saisie des paramètres du compte](#NOTE3_2).

3. [Sélection des fichiers à installer](../Editeurs/2028058.md). 

4. [Modification automatique](../Editeurs/2028048.md). 

5. [Paramètres du site](../Editeurs/2028051.md). 

6. Transfert des fichiers et installation. 
	Des messages d'erreur peuvent apparaître à diverses étapes de l'installation à distance. Pour plus de détails, consultez [Erreurs en cas d'installation par FTP](../Editeurs/2028057.md).

7. Lorsque l'installation est terminée, l'assistant fournit la ligne de commande permettant de lancer le site installé. Cette ligne peut être copiée si nécessaire dans le presse-papiers grâce au menu contextuel.




Remarque : Les paramètres de déploiement spécifiés dans l'assistant sont mémorisés et re-proposés lors du prochain déploiement.





<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Création d'un compte CLOUD
<a name="creation_compte_cloud_ELTTEXTE000178"></a>
Un compte CLOUD peut être créé directement sur le site du CLOUD ([http://www.pcscloud.net](http://www.pcscloud.net)). 

Attention : L'utilisation du CLOUD est soumise à un contrat et une tarification spécifiques. 

<a name="NOTE3_2"></a>


## Paramètres du compte
<a name="parametres_compte_ELTTEXTE000202"></a>
Une fois le compte CLOUD créé : 

1. Vous devez renseigner les paramètres de votre compte CLOUD PC SOFT : 

	- l'utilisateur

	- le mot de passe 




2. Valider pour procéder à l'installation du site. 




Le bouton "Paramètres avancés" permet de définir les paramètres de déploiement : 

- Paramètres FTP.

- Paramètres HTTP/HTTPS.

- Paramètres de proxy HTTP/HTTPS.

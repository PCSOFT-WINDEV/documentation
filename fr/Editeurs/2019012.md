## Présentation
<a name="presentation_ELTTEXTE000109"></a>


### Avant de déployer votre site dynamique WEBDEV, il est conseillé d'effectuer des tests finaux à partir d'un serveur de test.
<a name="avant_deployer_votre_site_dynamique_webdev_est_conseille_effectuer_des_tests_finaux_partir_serveur_test_ELTPARAGRAPHE000011"></a>ATTENTION : Le moteur WEBDEV de déploiement pour test NE doit PAS être installé sur un poste où WEBDEV version Développement ou version Déploiement est installé.

Les limitations du moteur de déploiement pour test par rapport au moteur de déploiement sont les suivantes :

- Nombre maximum de connexions (pour toutes les applications WEBDEV installées sur le poste) limité à 10.

- Le moteur de déploiement pour test ne fonctionne PAS en mode TEST (c'est-à-dire que ce moteur ne fonctionne pas avec l'éditeur de WEBDEV)

- L'administrateur WEBDEV porte l'indication "Moteur de test".

- Dans la barre de titre du navigateur apparaît une indication précisant que l'application est utilisée avec un moteur de test : "Moteur de test; utilisation autre que le test ponctuel illégale"




**Éléments à installer sur le poste "Serveur de test"**

- Votre site dynamique WEBDEV (installation par FTP ou par média physique). Pour plus de détails, consultez [Installation d'un site](../Editeurs/2028039.md).

- Un navigateur Internet.

- Le moteur de "Test" de WEBDEV. Ce moteur est disponible dans le package d'installation de WEBDEV "Version Développement".

- Un serveur Web (le serveur Apache est fourni avec WEBDEV "Version Développement").

- Le protocole TCP/IP.

## Comment le faire ?
<a name="comment_faire_ELTTEXTE000133"></a>


### Tester un site dynamique sur un serveur de test
<a name="tester_site_dynamique_sur_serveur_test_ELTPARAGRAPHE000038"></a>Pour lancer le test du projet sur un serveur de test, plusieurs méthodes sont possibles :

- 1ère méthode :

	1. Lancez l'administrateur WEBDEV : sous le volet "Outils", dans le groupe "Utilitaires Web", cliquez sur "WDAdmin". 

	2. Cliquez sur le bouton "Page de test" (onglet "Avancé").




- 2ème méthode : (conseillée)
	Connectez-vous sur le site en saisissant l'adresse Internet suivante : 
	
	```txt
	http://<Adresse IP du serveur de test>/WD280AWP/WD280AWP.EXE/CONNECT/<Nom du projet>
	```

## Comment le faire ?
<a name="comment_faire_ELTTEXTE000240"></a>


### Créer un diagramme de déploiement
<a name="creer_diagramme_deploiement_ELTPARAGRAPHE000039"></a>Pour créer un diagramme de déploiement : 

1. Cliquez sur ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_nouveau.gif)
 parmi les boutons d'accès rapide. 

	- La fenêtre de création d'un nouvel élément s'affiche : cliquez sur "Architecture" puis sur "UML".

	- L'assistant de création d'un modèle UML se lance.




2. Sélectionnez l'option "Diagramme de déploiement".

3. Spécifiez le modèle UML dans lequel le diagramme sera inséré.

4. Choisissez d'éditer directement le diagramme.

5. Validez la création du diagramme de déploiement. Le modèle créé devient le modèle en cours.



<a name="NOTE2_2"></a>


### Créer une classe de noeuds
<a name="creer_une_classe_noeuds_ELTPARAGRAPHE000061"></a>Pour créer une classe de noeuds : 

1. Sous le volet "Diagramme de déploiement", dans le groupe "Création", cliquez sur "Classe de noeud". Une nouvelle classe de noeud apparaît dans le modèle UML.

2. Sélectionnez l'option "Description" du menu contextuel (clic droit) de la nouvelle classe de noeud. La fenêtre de spécifications s'ouvre.

3. Spécifiez :

	- le type de la classe de noeud.

	- le nom de la classe de noeud. Ce nom est affiché sous l'éditeur d'UML.

	- le stéréotype de la classe de noeud.

	- le facteur de multiplicité de la classe de noeud.




4. Validez la description de la classe de noeud.



<a name="NOTE2_3"></a>


### Créer une instance d'un noeud
<a name="creer_une_instance_noeud_ELTPARAGRAPHE000090"></a>Pour créer une instance d'un noeud : 

1. Sous le volet "Diagramme de déploiement", dans le groupe "Création", cliquez sur "Instance de noeud". Une nouvelle instance de noeud apparaît dans le modèle UML.

2. Sélectionnez l'option "Description" du menu contextuel (clic droit) de la nouvelle instance de noeud. La fenêtre de spécifications s'ouvre.

3. Spécifiez :

	- le type de l'instance de noeud.

	- le nom de l'instance de noeud. Ce nom est affiché sous l'éditeur d'UML.

	- le stéréotype de l'instance de noeud.




4. Validez la description de l'instance de noeud.



<a name="NOTE2_4"></a>


### Créer une connexion
<a name="creer_une_connexion_ELTPARAGRAPHE000118"></a>Pour créer une connexion :

1. Sous le volet "Diagramme de déploiement", dans le groupe "Création", cliquez sur "Connexion".

2. Sélectionnez les différents éléments à relier. L'assistant de création d'une connexion se lance.

3. Spécifiez :

	- les cardinalités de la connexion.

	- le rôle des éléments reliés par la connexion.

	- le libellé de la connexion. Ce libellé sera affiché sous l'éditeur d'UML.




4. Validez la description de la connexion.



<a name="NOTE2_5"></a>


### Créer un paquetage
<a name="creer_paquetage_ELTPARAGRAPHE000146"></a>Pour créer un paquetage : 

1. Sous le volet "Diagramme de déploiement", dans le groupe "Création", cliquez sur "Paquetage". Un nouveau paquetage apparaît dans le modèle UML.

2. Sélectionnez l'option "Description" du menu contextuel (clic droit) du nouveau paquetage. La fenêtre de spécifications du paquetage s'ouvre.

3. Spécifiez :

	- le nom du paquetage. Ce nom est affiché dans le paquetage sous l'éditeur d'UML.

	- le libellé du paquetage décrivant l'intérêt du paquetage.

	- les relations du paquetage.




4. Validez la description du paquetage.

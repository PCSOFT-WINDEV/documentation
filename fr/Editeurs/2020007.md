<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000064"></a>
Certaines fonctionnalités de WINDEV utilisent le framework GDI+.

Ce framework spécifique permet d'exploiter toutes les ressources graphiques du système.

Ce framework se compose d'une librairie spécifique : gdiplus.dll.

Cette librairie est livrée en standard sur les systèmes récents (supérieurs à Windows 2000).




## Installation de WINDEV/WEBDEV/WINDEV Mobile : Respect des normes de répertoires
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000179"></a>
Les différentes versions de Windows proposent des normes concernant l'emplacement de certains fichiers et l'usage des répertoires.

Depuis la version 14, l'installation de WINDEV, WEBDEV et WINDEV Mobile respecte ces normes. Lors de l'installation, vous pouvez :

- Installer le produit sélectionné dans un répertoire unique. Ce mode d'installation correspond à l'installation "Classique".

- Installer le produit en respectant la norme Windows.




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>

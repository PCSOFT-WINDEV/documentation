
## Propriétés d'une page d'aide
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000148"></a>
Les caractéristiques d'une page d'aide permettent de définir :

- le titre de la page et son nom ([Onglet "Général"](#NOTE2_1)).

- les mots-clés d'index associés à la page d'aide ([Onglet "Index"](#NOTE3_1)).

- l'apparence des différentes zones de la page d'aide ([Onglet "Apparence"](#NOTE4_1)).




Pour afficher les propriétés d'une page d'aide, sélectionnez l'option "Propriétés de la page" du menu contextuel d'une page d'aide.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Onglet "Général"
<a name="onglet_general_ELTTEXTE000172"></a>
 
Les options générales d'une page d'aide permettent de définir :
<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=PageAide_General.gif&type=thumb)


- le **nom logique** de la page d'aide.

- le** titre de la page**. Ce titre sera affiché dans le sommaire (si la page apparaît dans le sommaire de l'aide). Ce titre peut être saisi dans les différentes langues gérées par le système d'aide.

- le **numéro d'aide** de la page : Ce numéro permet d'identifier de manière unique la page d'aide. Il sera ainsi possible d'ouvrir l'aide sur une page spécifique (fonction [WHelp](../WDLang1/3013053.md)). Pour plus de détails, consultez [Lancer l'aide](../Editeurs/2010017.md).

- la **visibilité de la page dans la table des matières** : 

	- si cette option est cochée, la page apparaîtra à la position définie dans le sommaire de l'aide.

	- si cette option est décochée, la page apparaîtra uniquement dans le sommaire affiché sous l'éditeur d'aide. En exécution, le libellé de la page ne sera pas affiché dans le sommaire.


 Attention : cette option est prise en compte uniquement dans le cas de la génération au format chm. 




Pour simplifier la gestion des numéros d'aide dans votre système d'aide, consultez [Identifiant des pages d'aide](../Editeurs/2010008.md).

Pour plus de détails sur la gestion de l'aide multilingue, consultez : [Comment faire une aide multilingue](../Editeurs/2010025.md).

## Onglet "Index"
<a name="onglet_index_ELTTEXTE000196"></a>
L'onglet Index permet de :

- associer la page d'aide en cours d'édition à un des index existants dans l'aide ou à un nouveau mot-clé d'index (bouton "Ajouter").

- supprimer l'association de la page d'aide à un des mots-clé d'index (bouton "Enlever").

- générer un index automatique à partir du contenu de la page (bouton "Index Auto ...").



<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=PageAide_Index.gif&type=thumb)


Pour plus de détails sur la gestion des mots-clé, consultez [Associer des mot-clé d'index à une page d'aide](../Editeurs/2010009.md).

## Onglet "Apparence"
<a name="onglet_apparence_ELTTEXTE000220"></a>
L'onglet "Apparence" permet d'associer la page d'aide en cours à une description de fenêtre d'aide : il suffit de sélectionner la description voulue. Pour créer directement une nouvelle description, cliquez sur le bouton "Définitions". Pour plus de détails sur les descriptions de fenêtres, consultez [Définir un modèle de fenêtre](../Editeurs/2010006.md).

Pour créer un entête de page, il suffit de cocher la case "Cette page contient un en-tête" et de préciser sa hauteur (en pixels).<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=zoneEntete_description.gif)


Les caractéristiques de l'entête spécifiées dans la description de la fenêtre d'aide seront automatiquement appliquées.

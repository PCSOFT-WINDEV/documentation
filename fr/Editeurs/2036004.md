


## Installation de WINDEV, WEBDEV ou WINDEV Mobile en 64 bits
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000102"></a>
Cette page d'aide présente la configuration nécessaire pour installer WINDEV, WEBDEV ou WINDEV Mobile en mode 64 bits.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Pourquoi installer une version 64 bits ?
<a name="pourquoi_installer_une_version_64_bits_ELTTEXTE000126"></a>
L'installation de WINDEV, WEBDEV ou WINDEV Mobile 64 bits est conseillée (sauf si vos projets utilisent une fonctionnalité non disponible, voir liste ci-dessous). Cette installation est recommandée notamment si vous utilisez des projets comportant de nombreux éléments (plus de 1500). 

L'installation d'une version 64 bits entraîne les limitations suivantes : 

- Les Connecteurs Natifs (Accès Natifs) Progress et Sybase ne sont pas disponibles en 64 bits. 

- Les connexions depuis l'éditeur d'analyses sont effectuées en 64 bits. Si une connexion est faite vers une base tierce, le client ou pilote ODBC ou provider OLE DB de cette base doit donc également être installé en 64 bits.

- L'importation d'anciennes analyses Hyper File 5.5 n'est pas disponible. 

- L'ancien RAD compatible 11 n'est pas disponible. 




<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Configuration nécessaire
<a name="configuration_necessaire_ELTTEXTE000150"></a>
La configuration nécessaire pour le poste de développement est la suivante : 

- 4 Go de mémoire centrale (8 Go ou plus conseillés).

- 2 Go d'espace disque (4 Go pour une installation complète : il y a des centaines d'exemples livrés en standard).

- sous Windows 64 bits.




<a name="NOTE4"></a>
<a name="NOTE4_1"></a>


## Notes
<a name="notes_ELTTEXTE000174"></a>
Le format des projets et de tous les éléments des projets est indépendant du mode de compilation des éditeurs. Tous les éléments sources (projets, fenêtres, états, classes...) peuvent donc indifféremment être ouverts par un éditeur 32 bits puis un éditeur 64 bits ou l'inverse.

**Il n'est pas conseillé d'installer sur un même poste de développement une version 32 bits et une version 64 bits d'un même logiciel**.

Si toutefois une configuration de développement unique accueille une version 32 bits et une version 64 bits de WINDEV, WEBDEV ou WINDEV Mobile :

- les deux versions doivent être installées dans des dossiers différents comme proposé par défaut. 

- les deux versions ne doivent pas être exécutées simultanément.




## Fichier ".STY" : fichier de type "Feuille de styles"
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000224"></a>
Une feuille de styles est un fichier de type ".STY". Une feuille de styles regroupe :

- les aspects (styles) des champs des fenêtres (couleur de fond, police, encadrement, ...),

- les informations relatives à l'aspect des champs des fenêtres (image de fond, dimension, ...).




Les feuilles de styles personnalisées sont manipulables directement depuis l'interface de l'éditeur de fenêtres. Pour plus de détails, consultez [Feuille de styles personnalisée](../Editeurs/2016015.md).

La manipulation des styles par les fichiers ".STY" est une option avancée de WINDEV.

Depuis le fichier ".STY", il est possible de :

- [visualiser tous les styles contenus dans un fichier ".STY"](#NOTE2).

- [modifier un style dans un fichier ".STY"](#NOTE3).

- [supprimer un style dans un fichier ".STY"](#NOTE3).

- [importer un style dans un fichier ".STY"](#NOTE4).

- [définir le style par défaut d'un type de champ](#NOTE5).




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment visualiser le contenu d'une feuille de styles (fichier .STY) ?
<a name="comment_visualiser_contenu_une_feuille_styles_fichier_sty_ELTTEXTE000248"></a>
Pour visualiser le contenu d'une feuille de styles, il suffit d'ouvrir le fichier ".STY" correspondant sous WINDEV. Les styles présents dans la feuille de styles sont affichés les uns au-dessous des autres.

**Remarque** : Les feuilles de styles de gabarit sont présentes dans le répertoire "Personnel\\Gabarit" du répertoire d'installation de WINDEV.

## Comment modifier ou supprimer un style dans une feuille de styles (fichier .STY) ?
<a name="comment_modifier_supprimer_style_dans_une_feuille_styles_fichier_sty_ELTTEXTE000272"></a>


### Modifier un style dans une feuille de styles (fichier ".STY")
<a name="modifier_style_dans_une_feuille_styles_fichier_sty_ELTPARAGRAPHE000055"></a>Pour modifier un style dans une feuille de styles (fichier ".STY") : 

1. Ouvrez la feuille de styles (fichier ".STY") contenant le style à modifier.

2. Sélectionnez le style à modifier.

3. Double-cliquez sur le style (ou sélectionnez l'option "Description gabarit" du menu contextuel).

4. Effectuez les modifications voulues et appliquez ces modifications.

5. Enregistrez la feuille de styles (![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_Enregistrer.gif)
).




Remarques :

- Ces modifications seront reportées sur les champs utilisant le style modifié à l'ouverture du projet et/ou des fenêtres.

- Si un projet était ouvert lors de la modification de la feuille de styles (fichier ".STY"), pour prendre en compte les modifications effectuées dans le projet en cours, sous le volet "Projet", dans le groupe "Projet", déroulez "Recompiler et synchroniser" et sélectionnez "Réactualiser style et gabarit".




**Attention** : Lors d'une mise à jour de WINDEV, les modifications effectuées sur les feuilles de styles de gabarit risquent d'être écrasées. Si des modifications sont effectuées sur les feuilles de styles de gabarit, nous vous conseillons de faire des copies des fichiers modifiés.
<a name="NOTE3_2"></a>


### Supprimer un style dans une feuille de styles (fichier ".STY")
<a name="supprimer_style_dans_une_feuille_styles_fichier_sty_ELTPARAGRAPHE000089"></a>Pour supprimer un style dans une feuille de styles (fichier ".STY") : 

1. Ouvrez la feuille de styles (fichier ".STY") contenant le style à supprimer.

2. Sélectionnez le style à supprimer.

3. Sélectionnez l'option "Supprimer" du menu contextuel (ou appuyez sur la touche Suppr).

4. Enregistrez la feuille de styles (![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_Enregistrer.gif)
).




Remarques :

- A l'ouverture d'un projet et/ou d'une fenêtre utilisant le style supprimé, il sera possible :

	- soit de ne rien changer. Les champs utilisant le style supprimé ne seront pas modifiés. Il est conseillé d'associer ces champs à des styles différents.
			Le style supprimé pourra être ajouté à la feuille de styles personnalisée du projet à la prochaine ouverture.

	- soit d'ajouter le style supprimé à la feuille de styles personnalisée du projet.




- Si un projet était ouvert lors de la suppression d'un style dans une feuille de styles : 

	- fermez ce projet : sous le volet "Accueil", dans le groupe "Général", déroulez "Fermer" et sélectionnez "Fermer le projet".

	- ré-ouvrez ce projet pour prendre en compte la suppression du style dans le projet en cours : sous le volet "Accueil", dans le groupe "Général", déroulez "Ouvrir" et sélectionnez "Ouvrir un projet".

## Comment importer un style dans une feuille de styles (fichier ".STY") ?
<a name="comment_importer_style_dans_une_feuille_styles_fichier_sty_ELTTEXTE000302"></a>
Pour importer un style dans une feuille de styles (fichier ".STY") : 

1. Ouvrez la feuille de styles (fichier ".STY") contenant le style à exporter.

2. Sélectionnez le style à exporter.

3. Copiez le style sélectionné dans le presse-papiers (Ctrl + C).

4. Ouvrez la feuille de styles (fichier ".STY") dans laquelle le style va être importé.

5. Collez le contenu du presse-papiers (Ctrl + V).

6. Enregistrez la feuille de styles (fichier ".STY") dans laquelle le style a été importé (![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_Enregistrer.gif)
). Le style importé est à présent disponible dans les deux feuilles de styles.
	Remarque : Si le style importé utilise des fichiers spécifiques (par exemple le fichier image affiché en fond du champ), ces fichiers sont copiés dans le répertoire de la feuille de styles d'importation.




**Attention** : Lors d'une mise à jour de WINDEV, les importations de styles effectuées dans les feuilles de styles de gabarit risquent d'être écrasées. Si des modifications sont effectuées sur les feuilles de styles de gabarit, nous vous conseillons de faire des copies des fichiers modifiés.

## Comment définir le style par défaut d'un type de champ ?
<a name="comment_definir_style_par_defaut_type_champ_ELTTEXTE000326"></a>
Pour définir le style utilisé par défaut pour un type de champ dans le projet en cours : 

1. Ouvrez la feuille de styles (fichier ".STY") utilisée par le projet en cours.

2. Sélectionnez le style à utiliser par défaut pour un type de champ. Par exemple, deux styles sont disponibles pour un champ de type "Bouton" : "StyleBtnA" et "StyleBtnB". Pour utiliser par défaut le style "StyleBtnA", sélectionnez le style "StyleBtnA".

3. Double-cliquez sur le style (ou sélectionnez l'option "Description gabarit" du menu contextuel). La fenêtre de description du style s'affiche.

4. Affichez l'onglet "Détail".

5. Cochez l'option "Par défaut" et validez.

6. Enregistrez la feuille de styles (![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_Enregistrer.gif)
).




Remarque : Si votre projet est associé à plusieurs feuilles de gabarit, le style utilisé par défaut sera sélectionné dans l'ordre suivant :

- le style défini par défaut dans la feuille de styles personnalisée.

- le style défini par défaut dans la feuille de styles du gabarit associée à la fenêtre en cours.

- le style défini par défaut dans la feuille de styles par défaut de WINDEV.

## Principales manipulations d'un MCD
<a name="principales_manipulations_mcd_ELTTEXTE000202"></a>


### Supprimer un MCD
<a name="supprimer_mcd_ELTPARAGRAPHE000034"></a>Pour supprimer un MCD :

1. Ouvrez l'explorateur de Windows.

2. Supprimez le répertoire complet du MCD (répertoire &lt;Nom du MCD&gt;.ANA).



<a name="NOTE2_2"></a>


### Renommer un MCD
<a name="renommer_mcd_ELTPARAGRAPHE000044"></a>Pour renommer un MCD :

1. Ouvrez le MCD à renommer dans l'éditeur d'analyses.

2. Sous le volet "Accueil", dans le groupe "Général", déroulez "Enregistrer" et sélectionnez "Enregistrer sous".

3. Saisissez le nouveau nom du MCD et validez la modification. Le MCD renommé devient le MCD en cours dans le projet.



<a name="NOTE2_3"></a>


### Vérifier un MCD
<a name="verifier_mcd_ELTPARAGRAPHE000063"></a>Pour vérifier un MCD :

1. Sous le volet "MCD", dans le groupe "Analyse", cliquez sur "Vérifier le MCD".

2. Une fenêtre apparaît listant les erreurs et/ou les avertissements rencontrés. Ces erreurs / avertissements sont provoquées par :

	- les incohérences existantes dans le MCD.

	- les normes spécifiées par l'utilisateur dans l'onglet "Détail" de la description du MCD ("Informations sur le modèle" du menu contextuel de l'éditeur d'analyses).<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=OptionsMCD.gif&type=thumb)





3. Pour imprimer cette liste, cliquez sur le bouton "Imprimer".




**Remarque** : Lors de [la génération du MCD en MLD](../Editeurs/2011020.md), une vérification du MCD est automatiquement effectuée. En cas de problème, une fenêtre apparaît listant les erreurs et/ou les avertissements rencontrés.

- Si seuls des avertissements sont affichés, la génération du MCD en MLD peut continuer en cliquant sur le bouton "Fermer".

- Si au moins une erreur est affichée, la génération du MCD en MLD est arrêtée. Le bouton "Fermer" permet de revenir sous l'éditeur pour corriger le MCD.



<a name="NOTE2_4"></a>


### Modifier l'affichage d'un MCD
<a name="modifier_affichage_mcd_ELTPARAGRAPHE000098"></a>Il est possible de spécifier un facteur de zoom pour réduire ou agrandir l'affichage du MCD en cours.

Pour spécifier un facteur de zoom :

- **1ère méthode** :

	- Soit sous le volet "Affichage", dans le groupe "Aide à l'édition", choisissez le facteur de zoom à utiliser.

	- Soit saisissez directement le facteur de zoom dans la barre de message de l'éditeur (en bas à droite).<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ZoomBarreMessage.gif)





- **2ème méthode** :

	- Appuyez sur la touche Ctrl.

	- Modifiez le facteur de zoom avec la roulette de la souris tout en maintenant la touche Ctrl enfoncée.






<a name="NOTE2_5"></a>


### Déplacer l'affichage d'un MCD
<a name="deplacer_affichage_mcd_ELTPARAGRAPHE000125"></a>Pour déplacer l'affichage d'un MCD :

1. Ouvrez le MCD dans l'éditeur d'analyses.

2. Appuyez sur la touche Alt.

3. Modifiez la position du MCD en maintenant à la fois le clic gauche de la souris et la touche Alt enfoncés.

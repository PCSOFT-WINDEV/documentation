
## Importer les styles d'un thème WordPress avec WEBDEV
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000153"></a>
WordPress propose de nombreux templates pour créer des sites originaux. WEBDEV permet d'importer très simplement ces templates. 

Lors de l'import des styles d'un Template WordPress, WEBDEV analyse le contenu du template, extrait et génère les styles utilisés. WEBDEV importe également les images des pages exemples. L'import crée la feuille de styles correspondant au template, et permet l'import des images utilisées par les exemples fournis avec le template.

Il est possible d'importer plusieurs Templates différents dans le même projet : chaque feuille de styles reste cloisonnée.


<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Pré-requis
<a name="prerequis_ELTTEXTE000177"></a>
**Avant de réaliser l'import d'un template WordPress**, il est nécessaire de vérifier les points suivants : 

- Visual Studio 2013 Redistributable Package doit avoir été installé sur le poste. 
	Lors de l'écriture de cette documentation, ce package est téléchargeable sur le site Internet de Microsoft ([https://support.microsoft.com/fr-fr/help/2977003/the-latest-supported-visual-c-downloads](https://support.microsoft.com/fr-fr/help/2977003/the-latest-supported-visual-c-downloads)). 

- Il est nécessaire de copier le contenu de l'archive du serveur MySQL dans le répertoire WordPress de WEBDEV : 

	1. Téléchargez l'archive ZIP de MySQL Serveur 5.X 32 ou 64 bits (par exemple [https://dev.mysql.com/downloads/mysql/5.7.html](https://dev.mysql.com/downloads/mysql/5.7.html)). 

	2. Créez un dossier "mysql" dans le dossier des [éléments personnels](../Editeurs/2020015.md) de WEBDEV. Il s'agit du dossier : 

		- "\\Personnel\\Wordpress" si WEBDEV est installé dans un répertoire unique,

		- "&lt;User&gt;\\AppData\\Roaming\\PC SOFT\\28.0\\Wordpress" si WEBDEV est installé à la norme Windows.




3. Dans ce répertoire "\\WordPress\\mysql\\", décompressez le contenu de l'archive précédemment téléchargée. Les dossiers "bin", "lib" (...) doivent être à la racine du dossier "\\WordPress\\mysql\\".
			Exemples de répertoires : 

	- si WEBDEV est installé dans un répertoire unique :
						"C:\\&lt;votre dossier WEBDEV 28&gt;\\Personnel\\WordPress\\mysql\\bin\\mysqld.exe"

	- si WEBDEV est installé à la norme Windows :
						"C:\\Users\\&lt;User&gt;\\AppData\\Roaming\\PC SOFT\\28.0\\WordPress\\mysql\\bin\\mysqld.exe"

- Gestion des ports : Les ports TCP locaux 3399 et 8099 sont utilisés. Les dispositifs de sécurité type firewall doivent autoriser WEBDEV à communiquer en local sur ces ports (localhost).




**Si ces pré-requis n'ont pas été vérifiés, l'assistant d'importation du thème WordPress vous demandera de réaliser les opérations nécessaires**.

Attention : Avant d'importer un thème WordPress, veuillez consulter la licence de ce thème afin de vérifier si cet import est autorisé.

## Comment le faire ?
<a name="comment_faire_ELTTEXTE000201"></a>
**Pour importer un thème WordPress** : 

1. Ouvrez le projet WEBDEV dans lequel le thème WordPress doit être importé. 

2. Sous le volet "Projet", dans le groupe "Projet", déroulez "Importer" et sélectionnez l'option "Les styles d'un thème WordPress". 

3. L'assistant d'importation se lance. 
	Remarque : Si les pré-requis ne sont pas validés, l'assistant ne proposera pas d'importer le thème. 

4. Sélectionnez le fichier '.zip' contenant le thème WordPress à importer et cliquez sur le bouton "Terminer". 

5. L'importation se lance et peut prendre quelques minutes. 

6. A la fin de l'importation, WEBDEV propose d'ouvrir une page exemple contenant l'application du style importé sur les principaux champs d'une page WEBDEV. 




Les styles du thème WordPress sont intégrés au projet via une feuille de style spécifique. 

**Pour choisir un style du thème WordPress pour un de vos champs** : 

1. Sélectionnez le champ dans la page. 

2. Affichez le menu contextuel et sélectionnez l'option "Choisir un style WEBDEV". 

3. La fenêtre des styles disponibles s'affiche. 

4. Pour visualiser uniquement les styles du thème WordPress, sélectionnez son nom dans la liste "Styles du projet". 

5. Sélectionnez le style voulu et validez.

## Présentation
<a name="presentation_ELTTEXTE000127"></a>
Si votre application est multilingue, il est nécessaire de gérer le multilingue également dans les fenêtres/pages du groupware utilisateur. Par défaut, les fenêtres/pages du groupware utilisateur gèrent l'anglais/américain et le francais.

Cette page d'aide présente comment :

- Configurer le groupware utilisateur pour l'utiliser dans une application multilingue. 

- Traduire les différentes fenêtres/pages du groupware utilisateur.

- Exécuter l'application dans la langue souhaitée.




Pour plus de détails sur les applications multilingues, consultez [Créer une application multilingue](../WDLang1/3054008.md).

## Configurer le groupware utilisateur pour le multilingue
<a name="configurer_groupware_utilisateur_pour_multilingue_ELTTEXTE000151"></a>
Pour intégrer les fenêtres ou les pages du groupware utilisateur dans une application multilingue :

1. Sélectionnez l'option "Atelier .. Paramétrer le Groupware Utilisateur".

2. Vérifiez que les options suivantes sont sélectionnées :

	- Activer la gestion du groupware utilisateur

	- Intégration personnalisée




3. Vérifiez que l'option "Lancement manuel" est cochée.

## Traduction des fenêtres/pages du groupware utilisateur
<a name="traduction_des_fenetrespages_groupware_utilisateur_ELTTEXTE000175"></a>


### Traduction des fenêtres ou des pages pour les applications multilingues
<a name="traduction_des_fenetres_des_pages_pour_les_applications_multilingues_ELTPARAGRAPHE000051"></a>Par défaut, les fenêtres ou les pages du groupware utilisateur sont disponibles en français et anglais.

Pour traduire ces fenêtres ou ces pages dans d'autres langues, il suffit de traduire les différents libellés, ... de ces fenêtres ou de ces pages (en utilisant l'outil [WDMSG](../WDMsg/3518040.md) si nécessaire pour extraire les messages à traduire). WDMSG est un outil optionnel de WINDEV/WEBDEV/WINDEV Mobile permettant d'extraire et de ré-intégrer tous les messages à traduire.

## Exécuter l'application dans la langue souhaitée
<a name="executer_application_dans_langue_souhaitee_ELTTEXTE000199"></a>
Pour exécuter l'application multilingue en gérant le groupware utilisateur, il faut : 

1. Demander si nécessaire à l'utilisateur la langue dans laquelle l'application doit s'exécuter (par une fenêtre ou une page spécifique par exemple).

2. Configurer la langue d'exécution de l'application avec la fonction [Nation](../WDLang1/3054014.md).

3. Ouvrir la première fenêtre ou page de gestion du groupware utilisateur avec la fonction [gpwOuvre](../WDLang6/3041002.md).





```wl
// Code d'initialisation du projet
// ouverture de la fenêtre choix des langues
nLangue est un entier = Ouvre(FEN_Select_Langue)
// Configuration de la langue pour l'application
Nation(nLangue)
// Lancement de la première fenêtre du groupware utilisateur
gpwOuvre()
```




## Le volet "Modifieur"
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000185"></a>
Le volet "Modifieur" permet de visualiser et/ou de modifier directement depuis l'éditeur de fenêtres, de pages ou d'états, les principales caractéristiques :

- du ou des champs sélectionnés.

- de la fenêtre ou de la page sélectionnée.

- de l'état ou du bloc de l'état sélectionné.


![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ModifieurGeneral.gif)
Les différentes caractéristiques peuvent être :

- regroupées par catégorie (onglet "Catégorie").

- listées par ordre alphabétique (onglet "Nom").




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Afficher le volet "Modifieur"
<a name="afficher_volet_modifieur_ELTTEXTE000209"></a>
**Pour afficher le volet "Modifieur"**, sous le volet "Accueil", dans le groupe "Environnement", déroulez "Volets" et sélectionnez "Volets" puis "Modifieur".

Pour plus de détails sur les manipulations des volets, consultez [Manipuler les volets ancrables](../Editeurs/2027001.md).

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Visualiser et/ou modifier les caractéristiques
<a name="visualiser_etou_modifier_les_caracteristiques_ELTTEXTE000233"></a>
Le volet "Modifieur" permet de visualiser et/ou de modifier les principales caractéristiques :

- du ou des champs sélectionnés.

- de la fenêtre ou de la page sélectionnée.

- de l'état ou du bloc sélectionné.




**Remarques** :

- Pour annuler la dernière action effectuée dans le modifieur, sous le volet "Accueil", dans le groupe "Action", cliquez sur "Annuler" (ou Ctrl + Z).

- Certaines caractéristiques ne sont pas modifiables (date et heure de création par exemple).



<a name="NOTE3_2"></a>


### Visualiser et/ou modifier les caractéristiques d'un ou de plusieurs champs grâce au modifieur
<a name="visualiser_etou_modifier_les_caracteristiques_plusieurs_champs_grace_modifieur_ELTPARAGRAPHE000078"></a>Pour visualiser et/ou modifier les caractéristiques d'un ou de plusieurs champs grâce au modifieur :

1. Sélectionnez le ou les champs à modifier :

	- soit directement dans la fenêtre, la page ou l'état (voir [Sélectionner un ou plusieurs champs](../Editeurs/2026007.md)).

	- soit grâce à la liste déroulante du modifieur. Dans ce cas, seul un champ à la fois peut être sélectionné. Les poignées du champ sélectionné s'affichent automatiquement.
			Si plusieurs champs sont sélectionnés, seules les informations communes aux champs sélectionnés sont affichées dans le modifieur.




2. Effectuez les modifications nécessaires. Ces modifications sont réalisées sur tous les champs sélectionnés. Vous pouvez :

	- soit sélectionner la nouvelle valeur dans une liste déroulante.

	- soit sélectionner la nouvelle valeur dans un écran (modification d'une couleur par exemple).

	- soit modifier directement la valeur.







**Remarque** : Pour que les caractéristiques affichées dans le modifieur correspondent à l'élément sélectionné dans la liste déroulante du modifieur (et non à l'élément actuellement sélectionné dans WINDEV, WEBDEV ou WINDEV Mobile) :

1. Sous le volet "Affichage", dans le groupe "Options", cliquez sur l'icône de regroupement (![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_regroup.gif)
). 

2. Dans la fenêtre qui s'affiche, affichez l'onglet "Options".

3. Décochez l'option "Afficher les propriétés dans le modifieur" et validez.




Dans ce cas, le modifieur ne sera pas mis à jour avec les caractéristiques de l'élément sélectionné dans WINDEV, WEBDEV ou WINDEV Mobile.
<a name="NOTE3_3"></a>


### Visualiser et/ou modifier les caractéristiques d'une fenêtre, d'une page ou d'un état grâce au modifieur
<a name="visualiser_etou_modifier_les_caracteristiques_une_fenetre_une_page_etat_grace_modifieur_ELTPARAGRAPHE000122"></a>Pour visualiser et/ou modifier les caractéristiques d'une fenêtre, d'une page ou d'un état grâce au modifieur :

1. Ouvrez si nécessaire la fenêtre/page (![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_ouvrir.gif)
 parmi les boutons d'accès rapide).

2. Sélectionnez la fenêtre/page :

	- soit en cliquant sur le fond de la fenêtre, de la page ou de l'état.

	- soit en sélectionnant le nom de la fenêtre, de la page ou de l'état dans la liste déroulante du modifieur.




3. Les caractéristiques de la fenêtre, de la page ou de l'état s'affichent dans le modifieur.

4. Effectuez les modifications nécessaires. Vous pouvez :

	- soit sélectionner la nouvelle valeur dans une liste déroulante.

	- soit sélectionner la nouvelle valeur dans un écran (modification d'une couleur par exemple).

	- soit modifier directement la valeur.

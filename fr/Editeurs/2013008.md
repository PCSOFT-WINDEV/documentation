


## Marque de positionnement dans le code 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000140"></a>
Les marques de positionnement permettent de "marquer" des lignes de codes spécifiques. Ces marques permettent un parcours rapide du code d'une marque de positionnement à une autre. 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000164"></a>


### Manipuler les marques de positionnement
<a name="manipuler_les_marques_positionnement_ELTPARAGRAPHE000018"></a>Les options de manipulation des marques de positionnement sont :

- **Ajouter ou enlever une marque de positionnement sur la ligne de code** : sous le volet "Code", dans le groupe "Marques", cliquez sur "Marque" ou utilisez la combinaison de touches Ctrl + F7. 

- **Atteindre la marque suivante** : sous le volet "Code", dans le groupe "Marques", cliquez sur ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_suivante.gif) ou utilisez la touche F7. 

- **Atteindre la marque précédente** : sous le volet "Code", dans le groupe "Marques", cliquez sur ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_precedente.gif) ou utilisez la combinaison de touches Maj + F7. 

- **Supprimer toutes les marques du code** : sous le volet "Code", dans le groupe "Marques", cliquez sur "Marque" et sélectionnez "Enlever toutes les marques".



<a name="NOTE2_2"></a>


### Positionner des marques sur le résultat d'une recherche
<a name="positionner_des_marques_sur_resultat_une_recherche_ELTPARAGRAPHE000074"></a>Pour positionner des marques sur le résultat d'une recherche :

1. Affichez le [volet "Rechercher - Remplacer"](../Editeurs/2027013.md).

2. Saisissez l'expression à rechercher dans la liste "Rechercher".

3. Spécifiez les éléments dans lesquels la recherche doit être effectuée (liste "Dans"). Il est possible d'effectuer une recherche par défaut ou une recherche avancée.

4. Précisez les critères de recherche ("Contient", "Commence par", "Ignorer la casse", ...).

5. Cliquez sur l'icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Ico_rech_marq.gif) pour marquer tous les résultats de la recherche.

6. Validez par "Entrée" pour lancer la recherche. Si des éléments sont trouvés, la référence de ces éléments apparaît dans le volet "Rechercher - Remplacer". Pour accéder directement à un des éléments trouvés, double-cliquez sur la référence de l'élément.


Pour plus de détails sur le "Rechercher - Remplacer", consulter la page d'aide [Rechercher/Remplacer](../Editeurs/2021003.md).

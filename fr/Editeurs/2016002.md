
## Description d'un style WEBDEV
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000091"></a>
Cette fenêtre permet de décrire un style WEBDEV utilisé pour les champs de votre projet WEBDEV.

Cette fenêtre permet de définir les styles CSS et les options supplémentaires qui définissent le style WEBDEV.

Cette page d'aide détaille les différentes options de l'onglet "Général" et de l'onglet "Style".



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Onglet Général
<a name="onglet_general_ELTTEXTE000115"></a>
Cet onglet permet de définir le nom du style.

Pour un style associé à un champ Bouton, il est également possible de spécifier :

- L'aspect du bouton (image fixe ou avec survol, ...).

- Les images associées au bouton.

- Les dimensions du bouton.






<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Onglet Style
<a name="onglet_style_ELTTEXTE000139"></a>
Cet onglet permet de définir les différents styles CSS utilisés par le style WEBDEV. Pour plus de détails sur les styles CSS, consultez [Styles CSS](../Editeurs/2016004.md).

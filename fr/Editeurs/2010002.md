


## Créer le système d'aide sous WINDEV, WEBDEV ou WINDEV Mobile
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000205"></a>
Les principales étapes de la création d'un fichier d'aide (HTML, CHM ou PDF) sont les suivantes :

- créer le système d'aide.

- [créer les pages d'aide](../Editeurs/2010003.md) (avec des images si nécessaire).

- [créer les liens entre les pages d'aide](../Editeurs/2010023.md).

- [créer le sommaire](../Editeurs/2010013.md).

- [créer l'index](../Editeurs/2010009.md).

- [compiler l'aide et la tester](../Editeurs/2010014.md).

- [lancer l'aide](../Editeurs/2010017.md).






<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000229"></a>


### Créer une aide associée à votre application (ou à votre site)
<a name="creer_une_aide_associee_votre_application_votre_site_ELTPARAGRAPHE000042"></a>Pour créer une aide associée à votre application (ou à votre site) : 

1. Ouvrez votre projet sous WINDEV, WEBDEV ou WINDEV Mobile.

2. Cliquez sur ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_nouveau.gif)
 parmi les boutons d'accès rapide. 

	- Dans la fenêtre qui s'affiche, cliquez sur "Architecture" puis sur "Aide".

	- L'assistant de création d'un système d'aide se lance automatiquement.




3. Passez à l'étape suivante de l'assistant et choisissez le type de génération du système d'aide : 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Cr%E9er_l_aide%20-%20HC%20N%B0001.gif&type=thumb)


	- HTML : l'aide correspondra à un site statique avec un sommaire et une recherche full-text. 

- CHM : l'aide sera au format Microsoft HTML Help 1.3. 

- PDF : l'aide correspondra à un document au format PDF avec sommaire et signets. 

4. Passez à l'étape suivante. 

5. Indiquez le nom du système d'aide. Passez à l'étape suivante de l'assistant. 

6. Sélectionnez les fenêtres de votre application pour lesquelles une page d'aide doit être générée.
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Cr%E9er_l_aide%20-%20HC%20N%B0002.gif&type=thumb)

	Lors d'une génération au format CHM, l'option "Générer automatiquement le bouton d'aide dans chaque fenêtre" permet de créer automatiquement un bouton d'aide dans chaque fenêtre sélectionnée. Ce bouton ouvrira automatiquement la page d'aide correspondante. 

7. Passez à l'étape suivante. 

8. Sélectionnez les états de votre application pour lesquels une page d'aide doit être générée. Passez à l'étape suivante de l'assistant. 

9. Validez. L'assistant crée l'ensemble des pages d'aide spécifiées et propose de générer l'aide au format indiqué. 

10. Enregistrez l'aide : un fichier WDH est créé dans votre projet.




Le système d'aide est créé et affiché sous l'éditeur d'aide. Chaque page d'aide peut ensuite être modifiée, supprimée, ... sous l'éditeur d'aide.

**Remarque** : Lorsque votre système d'aide est créé, pour ouvrir ce système d'aide sous l'éditeur, il suffit d'utiliser une des méthodes suivantes :

- sous le volet "Accueil", dans le groupe "Général", déroulez "Ouvrir", sélectionnez "Ouvrir" puis le fichier d'aide correspondant (fichier WDH).

- sélectionnez le dossier "Aides" dans le volet "Explorateur de projet" et double-cliquez sur le nom du système d'aide.



<a name="NOTE2_2"></a>


### Afficher les propriétés du système d'aide
<a name="afficher_les_proprietes_systeme_aide_ELTPARAGRAPHE000132"></a>Les propriétés du système d'aide permettent de définir :

- le nom logique du système d'aide,

- la page par défaut affichée par l'aide,

- les langues gérées par le système d'aide,

- ...


Pour afficher les propriétés du système d'aide, sous le volet "Modification", dans le groupe "Système d'aide", cliquez sur "Description".

Pour plus de détails, consultez [Propriétés du système d'aide](../Editeurs/2010016.md).

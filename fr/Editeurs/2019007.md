


## Tester un projet dynamique depuis l'administrateur
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000133"></a>
Le lancement du test depuis l'administrateur WEBDEV (WDAdmin) permet par exemple de tester : 

- les fonctionnalités du site.

- les fonctionnalités spécifiques au Web (utilisation des cookies, ...).




**Remarque** : L'administrateur WEBDEV permet uniquement de tester les sites dynamiques ou la partie dynamique des sites "statiques + dynamiques".

Le lancement du test depuis l'administrateur WEBDEV est équivalent au lancement du site dynamique depuis un poste distant.

Avant de déployer un site dynamique WEBDEV, il est conseillé de tester au moins une fois ce site depuis l'administrateur WEBDEV.

Cette page d'aide présente comment tester un projet depuis l'administrateur WEBDEV. Il est également possible de tester un projet dynamique depuis l'éditeur WEBDEV. Pour plus de détails, consultez :

- [Tester un projet dynamique depuis l'éditeur WEBDEV](../Editeurs/2019009.md),

- [Différences entre le test d'un projet depuis l'éditeur et depuis l'administrateur](../Editeurs/2019008.md). 




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000157"></a>


### Tester un site dynamique (ou la partie dynamique d'un site) depuis l'administrateur
<a name="tester_site_dynamique_partie_dynamique_site_depuis_administrateur_ELTPARAGRAPHE000036"></a>Pour lancer le test depuis l'administrateur WEBDEV :

1. Lancez l'administrateur : sous le volet "Outils", dans le groupe "Utilitaires Web", cliquez sur "WDAdmin".

2. Dans l'onglet "Avancé", cliquez sur le bouton "Page de test".



<a name="NOTE2_2"></a>


### Arrêter le test d'un projet
<a name="arreter_test_projet_ELTPARAGRAPHE000058"></a>Pour arrêter le test d'un projet :

1. Affichez l'administrateur WEBDEV (en cliquant sur l'icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_WWAdmin.gif)
 présente dans la barre des tâches).

2. Dans l'onglet "Connexions", cliquez sur le bouton "Déconnecter".

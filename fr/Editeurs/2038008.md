
## Administrateur du GDS
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000078"></a>
L'administrateur de GDS (Gestionnaire de sources) permet de manipuler directement les différents projets inclus dans le gestionnaire de sources.

Il permet par exemple de :

- [gérer les bases de sources](../Editeurs/2038021.md) (création, connexion à une base de sources).

- [gérer les branches d'un projet](../Editeurs/2038022.md).

- [gérer les fichiers et les répertoires présents dans un projet de la base de sources](../Editeurs/2038021.md) (ajouter, supprimer, renommer des fichiers et des répertoires).

- [gérer les différents fichiers de la base de sources](../Editeurs/2038021.md) (extraction, réintégration, partage, ...).

- [lancer certains outils](../Editeurs/2038023.md) (options, maintenances, ...).

## Création d'une requête paramétrée
<a name="creation_une_requete_parametree_ELTTEXTE000100"></a>
Quel que soit le type de la requête (sélection, insertion, suppression, modification) et son mode de création (avec l'assistant ou directement par SQL), il est possible d'utiliser des paramètres. Le nom du paramètre est indiqué lors de la création de la requête.

Pour créer une requête paramétrée :

1. Créez une requête. Pour plus de détails sur la création d'une requête, consultez [Comment créer une requête ?](../Editeurs/2032059.md).

2. Lors de la [définition des conditions de sélection](../Editeurs/2032019.md) :

	- spécifiez que la condition de sélection correspond à un paramètre,

	- donnez le nom de ce paramètre.

## Test d'une requête paramétrée
<a name="test_une_requete_parametree_ELTTEXTE000124"></a>
Pour tester une requête paramétrée :

1. Ouvrez la requête paramétrée sous l'éditeur de requêtes.

2. Cliquez sur ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Ico_Go_Fenetre_WD_bl.gif) parmi les boutons d'accès rapide.

3. Spécifiez la valeur du ou des paramètres à utiliser pour le test de la requête. Dans cette fenêtre, vous pouvez sélectionner une valeur existante grâce à l'icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=OrdreAffichage.gif). Le menu suivant apparaît : <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Requete_MenuVisuValeur.gif)


4. Il est alors possible :

	- soit de sélectionner une des valeurs possibles parmi celles présentes dans le fichier

	- soit de visualiser la totalité du fichier.
			**Attention** : Cette option est disponible uniquement si le Live Data est activé. Pour plus de détails, consultez [Live Data](../Editeurs/2021005.md).




5. Validez. Le résultat de la requête s'affiche selon la ou les valeurs spécifiées.




**Remarque** : Si aucun paramètre n'est sélectionné (paramètre non coché), la ou les conditions utilisant les paramètres seront ignorées.

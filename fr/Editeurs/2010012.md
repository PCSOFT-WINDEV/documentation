
## Les différents types d'aide dans une application WINDEV
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000176"></a>
Pour aider l'utilisateur, plusieurs solutions sont à votre disposition :

- [les bulles d'aide](#NOTE2_1),

- [les informations affichées dans la barre de message](#NOTE3_1),

- [les fenêtres d'aide standard](#NOTE4_1),

- [l'aide au format HLP ou CHM](#NOTE5_1).




La création des trois premiers types d'aide se fait directement dans l'éditeur de fenêtres, dans la description de chaque champ (onglet "Aide"). Ces aides ont une présentation simple, et ne peuvent être liées entre elles.

La création du quatrième type d'aide se fait directement sous l'éditeur d'aide. Il s'agit en général d'une aide plus structurée, permettant par exemple d'indiquer à l'utilisateur une marche à suivre.

<a name="NOTE2_1"></a>


## Les bulles d'aide
<a name="les_bulles_aide_ELTTEXTE000200"></a>
Les bulles d'aide sont des rectangles de couleur jaune clair affichés lors du survol d'un champ. Une bulle d'aide contient un message d'aide concernant le champ survolé. Une bulle d'aide peut ainsi informer l'utilisateur de l'unité monétaire attendue lors de la saisie d'un prix.


![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=BULLEAIDE.GIF)


Par défaut, les bulles d'aides sont activées automatiquement dans une application WINDEV. 

Pour saisir le message affiché dans la bulle d'aide d'un champ : 

1. Affichez la fenêtre de description du champ (option "Description" du menu contextuel). 

2. Dans l'onglet "Aide", dans la zone "Bulle d'aide", saisissez le texte de la bulle d'aide. 

3. Validez. 




**Remarques** : 

- Dans une application multilingue, le texte de la bulle d'aide peut être saisi dans les différentes langues dans la fenêtre de description du champ. 

- Si l'option "RTF" est cochée, le texte peut être mis en forme grâce aux options de la barre d'outils RTF affichée. En exécution, la mise en forme sera respectée. 

- Le texte de la bulle d'aide peut être modifié en programmation grâce à la propriété [Bulle](../Proprietes/2510005.md). 

- La bulle d'aide dispose automatiquement d'une marge autour du texte : le texte de la bulle est ainsi plus lisible.

- L'utilisateur final a la possibilité de modifier les bulles d'aide de son application. Pour plus de détails, consultez [Bulle d'aide des champs (FAA)](../Editeurs/1000021821.md). 




Pour personnaliser les bulles d'aide, consultez [Bulles d'aide personnalisées](../Editeurs/9500233.md).

## La barre de message
<a name="barre_message_ELTTEXTE000224"></a>
Lorsque une fenêtre contient une barre de message, il est possible d'afficher des informations dans cette barre de message. Deux possibilités :

- Le message est lié à un champ. Le message peut être saisi dans l'onglet "Aide" de la fenêtre de description des champs. Le message sera affiché lorsque l'utilisateur sélectionnera le champ.

- Le message est lié à un traitement réalisé par programmation. Il suffit d'utiliser la fonction [Message](../WDLang1/3021006.md) pour préciser le message à afficher.





![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Barremsg.gif)


Pour plus de détails sur les barres de message, consultez [Barre de message des fenêtres](../WDChamp/1010038.md).

## Les fenêtres d'aide standard
<a name="les_fenetres_aide_standard_ELTTEXTE000248"></a>
Les fenêtres d'aide standard sont des fenêtres "Windows" classiques permettant d'afficher un texte explicatif, associé à un champ.

Le titre de cette fenêtre et son message sont saisis dans l'onglet "Aide" de la fenêtre de description des champs. Il n'est pas possible d'utiliser des images dans ces fenêtres.

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Fenetre_Aide.gif)


Si cette fenêtre a été définie pour un champ, cette fenêtre sera automatiquement lancée lors de l'utilisation de la touche F1.

## L'aide au format CHM
<a name="aide_format_chm_ELTTEXTE000272"></a>
Une aide est constituée d'un sommaire, de mots-clés (index) et de pages. Quand l'utilisateur consulte l'aide, il lit le contenu d'une page d'aide. Ces pages peuvent contenir des images, des liens permettant de changer de page, ...

WINDEV permet de créer une aide au format CHM : ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Aide_CHM.gif)


Il existe plusieurs manières de lancer cette aide :

- aide complète : le fichier d'aide est directement affiché.

- aide complète contextuelle : le fichier d'aide est lancé sur une page spécifique correspondant à la fenêtre ou au champ en cours.




Pour plus de détails sur la création d'un fichier d'aide, consultez [Créer un fichier d'aide](../Editeurs/2010004.md).

Pour plus de détails sur les différents modes de lancement d'un fichier CHM, consultez [Lancer l'aide](../Editeurs/2010017.md).

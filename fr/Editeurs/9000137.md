
## Copie entre les éditeurs
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000085"></a>
WINDEV, WEBDEV et WINDEV Mobile permettent de réaliser des copies (ou Drag and Drop) des différents champs entre les produits et leurs éditeurs. 

Il est ainsi possible de copier des champs : 

- [de l'éditeur de fenêtres vers l'éditeur de pages](../Editeurs/9000138.md). 

- [de l'éditeur de fenêtres vers l'éditeur d'états](../Editeurs/9000138.md). 

- [de l'éditeur de pages vers l'éditeur de fenêtres](../Editeurs/9000139.md). 

- [de l'éditeur de pages vers l'éditeur d'états](../Editeurs/9000139.md). 

- [de l'éditeur d'états vers l'éditeur de fenêtres](../Editeurs/9000140.md). 

- [de l'éditeur d'états vers l'éditeur de pages](../Editeurs/9000140.md). 




Lors de cette copie, les champs existant dans les différents éditeurs seront conservés ainsi que le code associé. 

Dans certains cas, les champs seront transformés lors de la copie, pour mieux correspondre à l'éditeur destination.

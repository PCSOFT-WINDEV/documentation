


## Fonctionnalités automatiques des champs de saisie
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000132"></a>
Les champs de saisie permettent de saisir les principales informations d'une application. 

Les champs de saisie proposent un certain nombre de fonctionnalités automatiques disponibles dans tous les champs de toutes vos applications sans que vous ayez besoin de coder quoi que ce soit. Ces fonctionnalités simplifient la manipulation des champs de saisie pour vos utilisateurs. 

Ces fonctionnalités sont directement accessibles depuis le menu contextuel (affiché par le clic droit de la souris).

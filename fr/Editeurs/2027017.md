<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000092"></a>
Le volet "Centre de Suivi de Projets" affiche les tâches du développeur en cours. Ces tâches correspondent au projet sélectionné ou à tous les projets.
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CC_Suivi_Presentation%20-%20HC%20N%B0001.gif)
Le menu contextuel (clic droit) du volet "Centre de Suivi de Projets" permet de :

- Ajouter une tâche.

- Modifier une tâche.

- Annuler une tâche.

- Réaliser une tâche.

- Actualiser la liste des tâches.

- Ouvrir l'élément relié à une tâche.

- Démarrer une tâche.




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Afficher le volet "Centre de Suivi de Projets"
<a name="afficher_volet_centre_suivi_projets_ELTTEXTE000116"></a>
**Pour afficher le volet "Centre de Suivi de Projets"**, sous le volet "Accueil", dans le groupe "Environnement", déroulez "Volets" et sélectionnez "Volets" puis "Centre de Suivi de Projets".

Pour plus de détails sur les manipulations des volets, consultez [Manipuler les volets ancrables](../Editeurs/2027001.md).

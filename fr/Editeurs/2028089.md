


## Package de déploiement
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000106"></a>
Le package de déploiement permet de créer un fichier ZIP contenant l'installation du site. Ce fichier pourra être ensuite fourni à l'hébergeur qui installera le site à l'aide de WDDéploie.

**Remarque** : WDDéploie est un outil WEBDEV permettant :

- à l'hébergeur de déployer un site dynamique à partir d'un package.

- de déployer des sites statiques WEBDEV.

- de déployer des sites PHP.




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000130"></a>
Pour créer un package de déploiement : 

1. Lancez l'assistant de création d'installation d'un site WEBDEV : sous le volet "Projet", dans le groupe "Génération", déroulez "Déployer le site" et sélectionnez "Générer et déployer le site".

2. Générez la bibliothèque du site. 

3. Sélectionnez le modèle "Créer un package de déploiement distant". 

4. [Saisissez les informations sur le serveur](../Editeurs/2028046.md).

5. [Sélectionnez les fichiers à installer](../Editeurs/2028058.md).

6. [Paramétrez la modification automatique](../Editeurs/2028048.md)

7. [Paramétrez le site](../Editeurs/2028051.md).

8. Indiquez les caractéristiques du package :

	- Le répertoire de génération du package. Il est possible de vider ou non le contenu du répertoire lors de la génération.

	- Le nom du package de déploiement.




9. Validez la création du package. A la fin de la création du package, WEBDEV propose :

	- de lancer le déploiement du site (à l'aide de WDDeploie).

	- d'ouvrir l'explorateur.

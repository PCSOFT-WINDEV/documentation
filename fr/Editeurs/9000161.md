
## Détacher les éditeurs
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000101"></a>
Qui n'a jamais voulu lors du développement d'une application pouvoir visualiser en même temps le code et l'UI, mais sans être contraint par l'environnement ? WINDEV, WEBDEV et WINDEV Mobile permettent de "détacher" les éditeurs de l'environnement de la fenêtre principale.

Un usage fréquent est l'édition simultanée d'une fenêtre d'UI et de son code. L'éditeur de code sera détaché de l'environnement et affiché à côté, sur un deuxième écran par exemple.

## Comment le faire ?
<a name="comment_faire_ELTTEXTE000125"></a>


### Détacher une fenêtre de l'environnement
<a name="detacher_une_fenetre_environnement_ELTPARAGRAPHE000021"></a>Pour détacher une fenêtre de l'environnement : 

1. Sélectionnez le bouton correspondant à l'élément à détacher. Ce bouton est présent dans la barre des documents ouverts sous l'éditeur. 

2. Déplacez le bouton en dehors de l'environnement. 

3. Relâchez. L'élément apparaît dans une nouvelle fenêtre, indépendante de l'environnement. 




Cette nouvelle fenêtre peut être : 

- placée sur un nouvel écran, 

- redimensionnée, 

- ...


Si une fenêtre de code est ouverte depuis l'élément détaché, cette fenêtre sera affichée par défaut dans le même environnement que l'élément détaché. 


### Rattacher une fenêtre à l'environnement
<a name="rattacher_une_fenetre_environnement_ELTPARAGRAPHE000038"></a>Pour rattacher un élément à l'environnement : 

- Méthode 1 : 

	1. Sélectionnez le bouton correspondant à l'élément à rattacher. Ce bouton est présent dans la barre des documents dans la fenêtre indépendante. 

	2. Dans le menu contextuel, sélectionnez l'option "Rattacher à la fenêtre principale". 
			Remarque : Si la fenêtre indépendante contient plusieurs éléments, il est possible de rattacher uniquement un élément à la fenêtre principale. 




- Méthode 2 : 

	1. Sélectionnez la fenêtre indépendante. 

	2. Déplacez la fenêtre indépendante vers la barre des documents ouverts de la fenêtre principale de l'éditeur. 

	3. Lorsque l'éditeur change de couleur, relâchez le bouton de la souris : la fenêtre indépendante est automatiquement rattachée à l'environnement.  
			Remarque : Si la fenêtre indépendante contient plusieurs éléments, tous les éléments sont rattachés à la fenêtre principale.

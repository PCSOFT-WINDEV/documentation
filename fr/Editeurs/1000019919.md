


## Développer une application pour Apple Watch
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000144"></a>
Une application Apple Watch est une application destinée à une montre couplée à un appareil iOS (téléphone, tablette, ...).

Une application Apple Watch est toujours associée à une application iOS avec laquelle elle peut échanger des données. 

Le développement d'une application Apple Watch est réalisé en plusieurs étapes : 

- [Création et développement du projet sous WINDEV Mobile](#NOTE2_1). 

- [Génération de l'application](../Editeurs/1000019908.md). 

- [Compilation sous Xcode](../Editeurs/1000019878.md). 




Cette page d'aide présente uniquement la création d'un projet Apple Watch et son développement. Sa génération et sa compilation sont identiques à celles pour une application iOS classique.

<a name="NOTE2_1"></a>


## Créer un projet pour Apple Watch avec WINDEV Mobile
<a name="creer_projet_pour_apple_watch_avec_windev_mobile_ELTTEXTE000168"></a>
Pour créer un projet Apple Watch avec WINDEV Mobile : 

1. Créez un projet pour iOS (pour plus de détails, consultez [Développer une application pour iPhone/iPad](../Editeurs/1000019818.md)).

2. Créez une configuration Apple Watch :

	- Lancez l'assistant de création d'une nouvelle configuration : sous le volet "Projet", dans le groupe "Configuration de projet", déroulez "Nouvelle configuration" et sélectionnez l'option "Nouvelle configuration (via assistant)". 
			Pour plus de détails, consultez [Configuration de projet](../Editeurs/9000030.md).

	- Dans l'étape "Informations générales" de l'assistant, sélectionnez "Application Apple Watch".<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Cr%E9erAppliAppleWatch%20-%20HC%20N%B0001.gif&type=thumb)


	- Dans l'étape "Application iOS associée" de l'assistant, sélectionnez la configuration iOS du projet associée à la configuration Apple Watch.<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Cr%E9erAppliAppleWatch%20-%20HC%20N%B0002.gif&type=thumb)

## Développer une application pour Apple Watch avec WINDEV Mobile
<a name="developper_une_application_pour_apple_watch_avec_windev_mobile_ELTTEXTE000192"></a>
Le développement d'une application Apple Watch et de son application iOS associée est similaire au développement pour une application classique.

**Pour l'application iOS**, vous créez les différents éléments de votre application. Le développement du projet se fait comme un projet WINDEV Mobile classique. Cependant, il existe quelques spécificités au développement pour iOS. Pour plus de détails, consultez [Particularités des applications iOS](../Editeurs/1000019865.md).

**Pour l'application Apple Watch**, vous pouvez :

- créer des fenêtres. Ces fenêtres sont au format Apple Watch.

- ajouter des champs dans les fenêtres. Dans une application Apple Watch, seuls les champs de type Libellé, Bouton et Image sont disponibles.

- utiliser des fonctions/propriétés/types WLangage. L'icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_AWatch.gif) permet de savoir si la fonctionnalité est disponible en Apple Watch.




**L'application iOS et l'application Apple Watch** peuvent échanger des données en utilisant les fonctions suivantes :


|   |   |
| --- | --- |
| [awEnvoieEtRécupèreInfo](../WDLang3/1000022116.md) | Envoie des données d'un téléphone iOS vers une montre Apple Watch (ou inversement) en demandant une réponse. |
| [awEnvoieInfo](../WDLang3/1000022117.md) | Envoie une information d'un téléphone iOS vers une montre Apple Watch ou inversement. |
| [awProcédureRéceptionEtEnvoiInfo](../WDLang3/1000022118.md) | Précise la procédure WLangage appelée lors de la réception des informations provenant de la fonction [awEnvoieEtRécupèreInfo](../WDLang3/1000022116.md). |
| [awProcédureRéceptionInfo](../WDLang3/1000022119.md) | Précise la procédure WLangage appelée lors de la réception des informations provenant de la fonction [awEnvoieInfo](../WDLang3/1000022117.md). |





Lorsque le développement est terminé, vous pouvez générer le projet Xcode permettant de compiler l'application Apple Watch et son application iOS associée sur le MAC : 

1. Cliquez sur ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_generation_IOS.gif) parmi les boutons d'accès rapide. 

2. Choisissez si nécessaire la première fenêtre affichée sur les différentes plateformes (iPhone, iPad et Apple Watch).

3. L'assistant de génération se lance. Pour plus de détails sur la génération du projet Xcode, consultez [Génération de l'application](../Editeurs/1000019908.md).

<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000205"></a>
La fenêtre de description du projet permet de configurer de nombreux éléments de votre projet. Cette fenêtre est constituée des onglets suivants :

- L'onglet ["Projet"](../Editeurs/2030046.md) qui permet de saisir les informations générales du projet. 

- L'onglet "Analyse" qui permet d'associer une analyse au projet et / ou un modèle UML. 

- L'onglet ["Fichiers"](../Editeurs/2030052.md) qui permet de saisir différentes informations sur les fichiers de données gérés par l'application (uniquement disponible si le projet est lié à une analyse).

- L'onglet "Live data" qui permet de configurer l'affichage des données (provenant des fichiers de données) sous les différents éditeurs.

- L'onglet ["Langues"](../Editeurs/9500219.md) qui permet de paramétrer les différentes langues manipulées par le projet ainsi que les options linguistiques à utiliser pour chaque langue.

- L'onglet **"Style"** qui permet de choisir le thème de gabarit, la feuille de styles personnalisée, et de personnaliser l'aperçu avant impression. Il est également possible de paramétrer la taille des écrans.

- L'onglet "Groupe" qui permet de gérer le travail en groupe (GDS ou GIT).

- L'onglet ["Options"](../Editeurs/9500227.md) qui permet de configurer les options de mise à jour, de sauvegarde, de synchronisation des modèles. Cet onglet permet également d'activer le mode strict et de définir la charte de programmation.

- L'onglet ["Avancé"](../Editeurs/9500236.md) qui permet de gérer l'utilisation de "Etats & Requêtes", les FAA et le verrouillage de l'application.

- L'onglet ["Compilation"](../Editeurs/9500218.md) qui permet de configurer les options de compilation.

- L'onglet ["Télémétrie"](../Editeurs/9500223.md) qui permet de configurer les options de télémétrie pour le projet.

## Onglet Style
<a name="onglet_style_ELTTEXTE000235"></a>


### Gabarit du projet
<a name="gabarit_projet_ELTPARAGRAPHE000054"></a>L'onglet "Style" de la fenêtre de description du projet permet de connaître et de définir le gabarit utilisé par le projet. Un aperçu permet de voir le gabarit en situation.

Remarque : Si le gabarit est modifié via la fenêtre de description du projet, cette modification est prise en compte uniquement pour les nouvelles fenêtres créées. Pour appliquer un gabarit sur les fenêtres existantes, sous le volet "Projet", dans le groupe "Autres actions", cliquez sur "Appliquer un gabarit au projet". 

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) L'option "Fenêtres système Info, OuiNon, Confirmer, Dialogue (pour les configurations WINDEV)" permet de personnaliser les fenêtres d'information système, en utilisant les FAA ou en intégrant directement les fenêtres correspondantes dans le projet. Pour plus de détails, consultez [Personnaliser les fenêtres d'information système (ou boîtes de dialogue)](../WDLang1/3021003.md).

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) L'option "Appliquer le thème de gabarit sur les composants intégrés dans le projet et sur les fenêtres des FAA" permet d'utiliser le thème du gabarit du projet sur tous les éléments intégrés dans le projet : composants, fenêtres spécifiques des FAA, etc.



![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) L'option "Bulle d'aide" permet de définir le mode d'affichage des bulles d'aide de votre application. Par défaut, le mode d'affichage est "Standard". Il est possible de personnaliser les bulles d'aide de l'application. Pour plus de détails, consultez [Bulles d'aide personnalisées](../Editeurs/9500233.md). 
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 

### Visualisateur de rapports
<a name="visualisateur_rapports_ELTPARAGRAPHE000140"></a>Le visualisateur de rapports permet à l'utilisateur final d'une application WINDEV de visualiser les états avant de les imprimer. Ce visualisateur de rapports peut être personnalisé pour suivre le look de votre application. Pour plus de détails, consultez [Personnalisation du visualisateur de rapports (aperçu)](../WDChamp/2024002.md). 


### Feuilles de styles
<a name="feuilles_styles_ELTPARAGRAPHE000148"></a>Cette option permet de sélectionner la ou les feuilles de styles associées au projet. 
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 

### Taille de fenêtres à leur création
<a name="taille_fenetres_leur_creation_ELTPARAGRAPHE000155"></a>Cette option permet de sélectionner la taille qui sera proposée par défaut lors de la création des fenêtres sous l'éditeur de fenêtres.

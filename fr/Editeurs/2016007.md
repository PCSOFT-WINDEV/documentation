## Présentation
<a name="presentation_ELTTEXTE000424"></a>
Cette fenêtre permet de décrire un style CSS utilisé pour les champs de votre projet WEBDEV. <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Sty.gif&type=thumb)


Cette fenêtre permet de :

- **Modifier les caractéristiques d'un style existant**. Il suffit de sélectionner le style, l'état et les caractéristiques du style sélectionné puis de valider.
	**Remarques** : 

	- Si le style n'est pas "Local" au champ, ces modifications seront effectuées sur tous les champs qui utilisent le style modifié.

	- Si le style est "local au champ", seul le style du champ sera modifié. Cependant, le style sera incorporé à la page HTML (conséquence : page plus lente à charger).




- **Créer un nouveau style** : Cliquez sur le bouton [Créer], saisissez le nom du style, puis ses caractéristiques et validez.

- **Renommer un style** : Cliquez sur le bouton [Renommer] et indiquez le nouveau nom du style. 
	**Attention** : cette option est déconseillée. En effet, cette option perd tous les liens entre les champs utilisant le style et le style renommé. Pour les champs utilisant le style avant renommage, ce style sera transformé en style "Local".




Cette page d'aide détaille les différentes options de l'onglet "Texte" et de l'onglet "Fond". L'aperçu permet de voir dynamiquement les caractéristiques du style.

## Onglet Texte
<a name="onglet_texte_ELTTEXTE000472"></a>


### Polices
<a name="polices_ELTPARAGRAPHE000224"></a>Sélectionnez une des familles de polices disponibles.

Une famille de polices est un ensemble ordonné de polices. Si la première police n'est pas disponible sur le poste de l'internaute, la deuxième police est recherchée. Si cette police est disponible, elle est utilisée, sinon la police suivante est recherchée.

Le bouton [...] permet de créer de nouvelles familles de polices, de dupliquer et de modifier des familles existantes. 

Pour plus de détails, consultez [Famille de polices](../Editeurs/2016022.md). 


<a name="NOTE3_2"></a>


### Taille
<a name="taille_ELTPARAGRAPHE000247"></a>Sélectionnez la taille de la police. Les tailles possibles sont :

- Une taille définie, précisée dans une unité spécifique (pixels, centimètres, millimètres, pouces, point, pica, pourcentage, ...).

- Une taille relative à la police en cours (au moment de l'affichage de la page) :

- xx-small : Taille extrêmement petite

- x-small : Taille très petite

- small : Taille petite

- medium : Taille normale

- x-large : Taille grande

- xx-large : Taille très grande

- -1 : la plus petite taille

- +1 : la plus grande taille
	



<a name="NOTE3_3"></a>


### Décoration
<a name="decoration_ELTPARAGRAPHE000266"></a>Options supplémentaire de mise en forme de la police. Ces options permettent par exemple de souligner des liens ou de les barrer.

- Aucune : pas de décoration supplémentaire particulière

- Barré : le texte est barré : un trait apparaît au milieu du texte. Exemple : a

- Souligné : le texte est souligné. Exemple : <u><u><u><u>B</u></u></u></u>

- Clignotant : le texte clignote (cette option n'a d'effet que si l'internaute utilise le navigateur Netscape)

- Surligné : un trait est présent au dessus du texte

### Autres paramètres de la police
<a name="autres_parametres_police_ELTPARAGRAPHE000283"></a>


|   |   |
| --- | --- |
| Alignement | Présentation du texte : aligné à gauche, centré, aligné à droite ou justifié (aligné à gauche et à droite). |
| Alignement vertical | Alignement du texte en hauteur par rapport à la zone disponible. |
| Casse | Mise en forme du caractère : 1ère lettre en majuscule, majuscule, minuscule ou aucune (par défaut) |
| Césure | Gestion de la césure des mots. Il est possible d'utiliser une césure classique ou bien de couper les mots si nécessaire. |
| Couleur | Sélectionnez la couleur de la police. |
| Graisse | Épaisseur du caractère. Il est possible de sélectionner :<br><br>- une graisse prédéfinie (normal, gras, épais, léger)<br><br>- une épaisseur (de 100 à 900)<br><br><br> |
| Indentation | Retrait de première ligne. Indiquez une valeur qui peut être exprimée en pixels, centimètres, millimètres, pouces, point, pica, pourcentage, ... |
| Interlettrage | Espace entre les lettres. Indiquez une valeur qui peut être exprimée en pixels, centimètres, millimètres, pouces, point, pica, pourcentage, etc. |
| Interligne | Espace entre les lignes. Indiquez une valeur qui peut être exprimée en pixels, centimètres, millimètres, pouces, point, pica, pourcentage, etc. |
| Style | Style des caractères : normal, italique, oblique |
| Variante | Mode d'affichage des caractères : normal ou petites majuscules. |

## Onglet Fond
<a name="onglet_fond_ELTTEXTE000520"></a>
Les caractéristiques modifiables sur l'onglet "Fond" sont les suivantes :


|   |   |
| --- | --- |
| Couleur | Sélectionnez la couleur de fond du champ. |
| Image | Image affichée en fond du champ. L'image choisie doit être présente dans le répertoire des images du projet (&lt;NomProjet&gt;_WEB). |
| Répétition | Répétition de l'image dans le champ. L'image peut être répétée :<br><br>- en mosaïque (option répétée)<br><br>- à l'horizontal (option répétée X)<br><br>- à la verticale (option répétée Y)<br><br><br> |
| Position fixe | Si cette option est choisie, l'image ne sera pas déplacée ("scrollée") lors de l'utilisation de l'ascenseur. |
| Position horizontale et verticale | Position de l'image par rapport au champ. La valeur indiquée peut être exprimée en pixels, centimètres, millimètres, pouces, point, pica, pourcentage, ... |




## FAA : Champs de saisie de type chemin de fichier
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000136"></a>
Les champs de saisie utilisant le masque de saisie de type "Nom et chemin complet d'un fichier" ou "Nom d'un fichier sans chemin" proposent diverses FAA (Fonctionnalités Automatiques de vos Applications) : 

- [un menu contextuel spécifique permettant de manipuler le fichier](#NOTE2_1). 

- [un aperçu du fichier en vignette (dans le cas d'un chemin de fichier de type Image ou PDF)](#NOTE3_1).  




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Un menu contextuel spécifique
<a name="menu_contextuel_specifique_ELTTEXTE000160"></a>


### Fonctionnement
<a name="fonctionnement_ELTPARAGRAPHE000025"></a>Sur les champs de saisie de nom ou de chemin de fichier, un menu contextuel est automatiquement proposé. ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=faa_Chemin_de_fichier%20-%20HC%20N%B0002.gif)


Ce menu contextuel contient notamment les options suivantes spécifiques aux fichiers : 

- "Parcourir" : Cette option permet d'ouvrir le sélecteur de fichiers Windows pour sélectionner le fichier voulu. 

- "Ouvrir" : Cette option permet d'ouvrir le fichier dans le logiciel associé par défaut sur le poste en cours. 

- "Ouvrir avec" : Cette option permet d'ouvrir le fichier dans un des logiciels proposés par le système. 

- "Ouvrir l'emplacement du fichier" : Cette option permet d'ouvrir l'explorateur de fichier sur le répertoire contenant le fichier. 

- "Imprimer" : Cette option permet d'imprimer directement le fichier. 

- "Envoyer par email" : Cette option permet d'envoyer le fichier par email. Le logiciel de messagerie utilisé est automatiquement ouvert et le fichier est ajouté en pièce jointe. 




Remarque : L'option "Parcourir" est également disponible pour les champs de saisie de type "Chemin de répertoire".


### Mise en place
<a name="mise_place_ELTPARAGRAPHE000046"></a>Le menu contextuel des fichiers et des chemins de fichier est automatiquement mis en place à partir de la version 17. 

Comme tous les menus contextuels proposés par les FAA, il est possible de configurer les options proposées par défaut par le champ. Pour plus de détails, consultez [Configurer les menus FAA](../Editeurs/2010040.md). 

En programmation : 

- la fonction [FAAExécute](../WDLang1/1000022099.md) utilisée avec des constantes spécifiques permet d'exécuter directement les options du menu contextuel.

- la fonction [FAADésactive](../WDLang1/1000022018.md) utilisée avec des constantes spécifiques permet de supprimer certaines options dans le menu contextuel du champ de saisie spécifié.

## Vignette d'aperçu des images ou des fichiers PDF
<a name="vignette_apercu_des_images_des_fichiers_pdf_ELTTEXTE000190"></a>


### Fonctionnement
<a name="fonctionnement_ELTPARAGRAPHE000071"></a>Lors de la saisie ou du survol d'un chemin de fichier, si ce fichier est une image ou un document PDF, une vignette (redimensionnable) apparaît automatiquement. 

L'utilisateur peut ainsi s'assurer qu'il a sélectionné la bonne image ou le bon document. 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=faa_Chemin_de_fichier%20-%20HC%20N%B0001.gif)
La dimension de la vignette est automatiquement calculée. 

Remarque : Si nécessaire, l'utilisateur peut désactiver l'affichage de la vignette par l'option "Désactiver la prévisualisation d'image" du menu contextuel du chemin de fichier. 


### Mise en place
<a name="mise_place_ELTPARAGRAPHE000082"></a>La visualisation des fichiers image et PDF dans une vignette est automatiquement mise en place à partir de la version 17. 

En programmation, la fonction [FAADésactive](../WDLang1/1000022018.md) utilisée avec la constante *faaAfficheVignette* supprime l'affichage automatique de la vignette du fichier pour le champ de saisie spécifié.

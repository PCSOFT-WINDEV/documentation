


## WDInst : paramétrer les caractéristiques de l'installation des fichiers 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000209"></a>
L'éditeur d'installation WDInst permet de paramétrer les caractéristiques de l'installation des fichiers :

- Créer un raccourci pour chaque fichier installé (avec une ligne de commande si nécessaire).

- Gérer les collisions lors de la mise à jour de l'application (remplacement ou non des fichiers existants).

- Sauvegarder les fichiers remplacés lors de la mise à jour de l'application (sauvegarde automatique des fichiers existants).




Le paramétrage des caractéristiques de l'installation des fichiers est réalisé sous l'éditeur d'installation WDInst. Pour lancer WDInst, sous le volet "Outils", dans le groupe "Utilitaires", cliquez sur "WDInst".

L'éditeur d'installation WDInst propose une gestion avancée du paramétrage des caractéristiques de l'installation des fichiers par rapport à l'assistant de création du programme d'installation.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Créer un raccourci
<a name="creer_raccourci_ELTTEXTE000233"></a>
Pour chaque fichier à installer, l'éditeur d'installation WDInst permet de créer :

- Un raccourci sur le bureau du poste d'installation.

- Un raccourci dans un groupe de programmes.

- Un raccourci dans la barre de démarrage.




Par défaut, pour chaque exécutable installé, un raccourci sera créé :

- sur le bureau du poste d'installation.

- dans un groupe de programmes.

- dans la barre de démarrage.




**Remarque** : Le libellé des raccourcis sera créé dans la langue d'exécution du programme d'installation.
<a name="NOTE2_2"></a>


### Créer un raccourci
<a name="creer_raccourci_ELTPARAGRAPHE000055"></a>Pour créer un raccourci : 

1. Sélectionnez le fichier voulu dans la fenêtre "Liste des fichiers de l'installation".

2. Affichez l'onglet "Raccourci" de la fenêtre "Description du fichier".

3. Cochez l'option voulue : 

	- "Créer un raccourci sur le bureau" pour créer le raccourci de l'application sur le bureau du poste utilisateur. 

	- "Créer un raccourci dans un groupe de programmes" pour créer le raccourci dans le groupe spécifié. 

	- "Créer un raccourci dans la barre de démarrage" pour créer le raccourci dans la barre de démarrage Windows. 
			Toutes ces options peuvent être cumulées. 




4. Saisissez le libellé du raccourci dans chaque langue gérée par le programme d'installation.

5. Si nécessaire, saisissez le groupe de programmes dans lequel le raccourci doit être créé.
	**Remarque** : Si ce groupe de programmes n'existe pas sur le poste utilisateur, ce groupe sera créé.

6. Spécifiez si nécessaire les paramètres du raccourci. L'exécution de ce raccourci lancera une ligne de commande. 
	Par exemple, la ligne de commande : "C:\\MonRépertoire\\MonApplication.EXE /Login = Will" est composée :

	- Du chemin complet du fichier associé au raccourci.
			Dans notre exemple : "C:\\MonRépertoire\\MonApplication.EXE".

	- Des paramètres du raccourci.
			Dans notre exemple : "/Login = Will".

## Gérer les collisions
<a name="gerer_les_collisions_ELTTEXTE000263"></a>
Par défaut, lors de la mise à jour d'une application, si des fichiers de même nom existent déjà dans le répertoire d'installation, ces fichiers sont remplacés (sauf si les fichiers déjà présents sur le poste sont plus récents que ceux contenus dans l'installation). Aucune fenêtre d'avertissement ne s'affiche pour confirmer ces remplacements.

L'éditeur d'installation WDInst permet de paramétrer le programme d'installation afin de :

- Installer les fichiers même si des fichiers de même nom plus récents sont présents dans le répertoire d'installation.

- Ne pas installer les fichiers si des fichiers de même nom sont présents dans le répertoire d'installation.

- Demander à l'utilisateur la confirmation du remplacement des fichiers.

- Renommer les fichiers existants.



<a name="NOTE3_2"></a>


### Gérer les collisions
<a name="gerer_les_collisions_ELTPARAGRAPHE000112"></a>Pour gérer les collisions :

1. Sélectionnez le ou les fichiers voulus dans la fenêtre "Liste des fichiers de l'installation".
	Pour sélectionner plusieurs fichiers simultanément, maintenez la touche Ctrl ou Maj enfoncée lors de la sélection des fichiers.

2. Affichez l'onglet "Version" de la fenêtre "Description du fichier" : 

	- Pour ne pas installer le(s) fichier(s) sélectionné(s) si des fichiers de même nom sont présents dans le répertoire d'installation, cochez l'option "Ne pas installer si le fichier existe déjà (sans faire d'autres contrôles)".

	- Pour remplacer le(s) fichier(s) existant(s) (même si ce(s) fichier(s) sont plus récent(s)), cochez l'option "Remplacer le fichier même s'il est plus récent". 

	- Pour demander à l'utilisateur la confirmation du remplacement de(s) fichier(s) sélectionné(s), cochez l'option "Demander confirmation du remplacement du fichier".

	- Pour renommer les fichiers existants dans le répertoire d'installation avant l'installation des nouveaux fichiers, cochez l'option "Renommer le fichier remplacé sous le nom ci-dessous". Saisissez le nom et l'extension du fichier remplacé.
			Par exemple : "MonImage_Bak.GIF".
			**Remarque** : Cette opération doit être effectuée pour chaque fichier remplacé à renommer.

## Sauvegarder les fichiers remplacés
<a name="sauvegarder_les_fichiers_remplaces_ELTTEXTE000293"></a>
Lors de la mise à jour d'une application, il est possible d'intégrer dans le programme d'installation un plan supplémentaire. Ce plan supplémentaire propose à l'utilisateur de sauvegarder les fichiers remplacés dans un répertoire de sauvegarde. Par défaut, ce répertoire de sauvegarde sera nommé "Sauvegarde_ &lt;DateSauvegarde&gt;_ &lt;HeureSauvegarde&gt;".

Cette sauvegarde pourra être utilisée par exemple en cas d'échec de l'installation. Pour utiliser cette sauvegarde, l'utilisateur devra copier les fichiers présents dans le répertoire de sauvegarde vers le répertoire d'installation de l'application.

Par défaut, la sauvegarde des fichiers remplacés est proposée aux utilisateurs.
<a name="NOTE4_2"></a>


### Proposer la sauvegarde des fichiers remplacés
<a name="proposer_sauvegarde_des_fichiers_remplaces_ELTPARAGRAPHE000152"></a>Pour proposer la sauvegarde des fichiers remplacés : 

1. Sous l'éditeur d'installation WDInst : sélectionnez l'option "Paramètres de l'installation .. Assistant" .

2. Cochez l'option "Proposer la sauvegarde des éléments remplacés par l'installation". 
	Lors de l'installation de l'application, l'utilisateur pourra :

	- Choisir de sauvegarder les fichiers remplacés.

	- Modifier le répertoire de sauvegarde proposé par défaut.

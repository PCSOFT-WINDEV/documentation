


## Positionnement sur une ligne d'un champ de saisie (FAA)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000107"></a>


### Comment le faire ? 
<a name="comment_faire_ELTPARAGRAPHE000011"></a>Pour positionner le curseur de saisie automatiquement sur une ligne précise d'un champ de saisie, plusieurs méthodes sont disponibles :

- Utiliser l'option "Atteindre la ligne..." du menu contextuel du champ. 

- Utiliser le raccourci Ctrl + G.

- Exécuter la fonction [FAAExécute](../WDLang1/1000022099.md) :
	
	```txt
	FAAExécute(<Champ de saisie>, faaAtteindreLaLigne)
	```



La fenêtre suivante s'affiche :
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=FAA_Atteindre_Ligne.gif)
L'utilisateur peut alors saisir le numéro de la ligne voulue et valider.

### Modifier la fenêtre de sélection d'une ligne
<a name="modifier_fenetre_selection_une_ligne_ELTPARAGRAPHE000033"></a>Par défaut, la fenêtre de sélection d'une ligne est disponible en français et anglais. 

Pour modifier cette fenêtre ou la traduire dans une autre langue, il suffit de l'intégrer au projet : 

1. Sous le volet "Projet", dans le groupe "Projet", déroulez "Importer" et sélectionnez "Des éléments WINDEV et leurs dépendances".

2. Sélectionnez le répertoire "Programmes\\Donnees\\Fenêtres prédéfinies\\FAA". 

3. Sélectionnez la fenêtre "WinDevFaa_FenetreAtteindreLigne.wdw". 

4. Validez l'importation. La fenêtre est intégrée à votre projet et peut être traduite et modifiée. 




Remarques :

- La fenêtre ne doit pas être renommée.

- Si vous intégrez cette fenêtre dans votre application, vous risquez de ne pas profiter des améliorations de cette fenêtre effectuées au cours des mises à jour. Pensez à vérifier si cette fenêtre n'évolue pas.

- Attention : Pour modifier cette fenêtre, le fichier correspondant ne doit pas être en lecture seule. Modifiez les caractéristiques du fichier sous Windows si nécessaire.

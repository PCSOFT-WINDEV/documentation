


## Association dans un MCD
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000147"></a>
Dans un **MCD** (Modèle Conceptuel des Données), les associations permettent de lier les entités entre elles.

L'éditeur d'analyses permet de créer facilement des associations.

Lors de la [génération du MCD en MLD](../Editeurs/2011020.md), les associations deviendront automatiquement des [fichiers](../Editeurs/2011010.md) ou des [liaisons](../Editeurs/2011008.md) (selon les cardinalités spécifiées).

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Création d'une association
<a name="creation_une_association_ELTTEXTE000171"></a>


### Créer une association
<a name="creer_une_association_ELTPARAGRAPHE000031"></a>Pour créer une association :

1. Sous le volet "MCD", dans le groupe "Création", déroulez "Nouveau" et sélectionnez "Nouvelle association".

2. Spécifiez :

	- le **nom de l'association**. Lors de la génération du MCD en MLD, ce nom sera le nom logique du fichier de données créé si cette association comporte au moins une propriété ou fait partie d'une liaison complexe.

	- le **libellé de l'association** résumant succinctement le sujet de l'association.

	- les **notes** détaillant le fonctionnement de l'association.




3. L'association devient l'association en cours. Il est possible de décrire automatiquement les propriétés de cette association.

## Principales manipulations d'une association
<a name="principales_manipulations_une_association_ELTTEXTE000195"></a>


### Dupliquer une association
<a name="dupliquer_une_association_ELTPARAGRAPHE000060"></a>Pour dupliquer/copier une association :

1. Sélectionnez l'association à dupliquer.

2. Utilisez le raccourci Ctrl + C puis le raccourci Ctrl + V pour copier l'association.

3. Dans la fenêtre de duplication, spécifiez les informations demandées et validez. L'association dupliquée est automatiquement créée dans le MCD en cours.



<a name="NOTE3_2"></a>


### Supprimer une association
<a name="supprimer_une_association_ELTPARAGRAPHE000071"></a>Pour supprimer une association :

1. Sélectionnez l'association à supprimer.

2. Ouvrez le menu contextuel de l'association sélectionnée (clic droit).

3. Sélectionnez l'option "Supprimer". L'association est automatiquement supprimée du MCD. Si des relations pointaient sur cette association, ces relations sont également supprimées.



<a name="NOTE3_3"></a>


### Renommer une association
<a name="renommer_une_association_ELTPARAGRAPHE000082"></a>Pour renommer une association :

1. Sélectionnez l'association à renommer.

2. Sous le volet "MCD", dans le groupe "Élément courant", cliquez sur "Description".

3. Modifiez le nom de l'association et validez la description de l'association. L'association est automatiquement renommée.

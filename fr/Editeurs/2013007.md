## Présentation
<a name="presentation_ELTTEXTE000092"></a>
Le glossaire liste l'ensemble des abréviations de l'éditeur de code. Chaque abréviation permet d'accéder à une fonction, une procédure, une propriété, ...

Le glossaire est entièrement personnalisable.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000116"></a>


### Utiliser le glossaire
<a name="utiliser_glossaire_ELTPARAGRAPHE000028"></a>Pour utiliser le glossaire :

1. Dans la fenêtre de code, saisissez une des abréviations du glossaire.

	- si la saisie assistée est activée : le nom de la fonction correspondante apparaît automatiquement dans la liste sous les caractères saisis.

	- si la saisie assistée n'est pas activée, appuyez sur la touche F5 : le nom de la fonction correspondante apparaît automatiquement dans la liste sous les caractères saisis.




2. Sélectionnez et validez le texte correspondant à l'abréviation saisie. La fonction, la procédure ou la propriété associée à l'abréviation est automatiquement saisie à l'emplacement du curseur.





<a name="NOTE2_2"></a>


### Contenu du glossaire
<a name="contenu_glossaire_ELTPARAGRAPHE000042"></a>Pour connaître et/ou modifier le contenu du glossaire, utilisez l'option "Code .. Glossaire" (Maj + F5).

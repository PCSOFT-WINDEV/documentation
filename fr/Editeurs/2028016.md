


## Configuration du programme d'installation 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000357"></a>
Pour créer un programme d'installation, WINDEV propose deux outils :

- [l'assistant de création du programme d'installation](../Editeurs/2028062.md) : Cet assistant permet une configuration rapide du programme d'installation.

	- Depuis WINDEV : sous le volet "Projet", dans le groupe "Génération", déroulez "Procédure d'installation" et sélectionnez "Créer la procédure d'installation"

	- Depuis l'éditeur d'installation WDInst : option "Fichier .. Nouveau". 




- [l'éditeur d'installation WDInst](../Editeurs/2028042.md) : sous le volet "Outils", dans le groupe "Utilitaires", cliquez sur "WDInst". Cet éditeur permet une configuration avancée du programme d'installation et propose la gestion de nombreuses options supplémentaires.




## Nettoyage de projet
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000097"></a>
Après quelques mois ou quelques années de développement et de maintenance, le répertoire de votre projet contient souvent plusieurs fichiers qui ne sont plus utilisés, mais que, dans le doute, vous n'osez pas supprimer.

Fichiers et fenêtres de test, images abandonnées, ... Un nettoyage s'impose !

Un assistant de WINDEV/WEBDEV/WINDEV Mobile permet d'une part de détecter automatiquement ces éléments inutilisés et d'autre part de les supprimer du projet, sans aucun risque. Les éléments supprimés du projet seront archivés (au format ZIP, ou dans un répertoire de sauvegarde) pour le cas où une de ces ressources aurait un jour une utilité...

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000121"></a>
Pour nettoyer le répertoire du projet :

1. Sous le volet "Projet", dans le groupe "Autres actions", déroulez "Sauver/Restaurer" et sélectionnez "Nettoyer le répertoire du projet". L'assistant de nettoyage de projet se lance. Passez à l'étape suivante.

2. Sélectionnez les répertoires du projet dans lesquels la recherche de fichiers non utilisés doit être effectuée. Les répertoires possibles sont les suivants :

	- répertoire racine du projet.

	- répertoire "Groupware" du projet.

	- répertoire des différentes sauvegardes du projet.

	- répertoire "Java" si le projet en cours permet de générer une archive Java.




3. Choisissez le type des fichiers à prendre en compte. L'assistant liste l'ensemble des fichiers trouvés.

4. Sélectionnez les fichiers à archiver.

5. Les fichiers peuvent être archivés :

	- soit dans un fichier compressé (au format ZIP) : sélectionnez l'option "Oui : Créer un fichier ZIP avec les fichiers".

	- soit dans un répertoire de sauvegarde : sélectionnez l'option "Non : Déplacer les fichiers dans un répertoire de sauvegarde".




6. Spécifiez le nom du fichier compressé à créer ou le répertoire dans lequel les fichiers à archiver seront déplacés. Cochez l'option "Ajouter la date du jour au nom du fichier / du répertoire" pour préfixer automatiquement le nom du fichier ou du répertoire avec la date du jour.

7. Validez pour archiver les fichiers non utilisés.

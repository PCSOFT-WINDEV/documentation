
## Volet "Exemples unitaires"
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000097"></a>
Le volet "Exemples unitaires" propose les exemples unitaires disponibles selon les différents produits. Ces exemples sont constitués d'une fenêtre ou d'une page et permettent de comprendre rapidement l'utilisation d'une fonction, d'une fonctionnalité ou d'une famille de fonctions.


![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Volet_Exemples_unitaires%20-%20HC%20N%B0001.gif)


Le volet "Exemples unitaires" permet de : 

- Rechercher un exemple unitaire : Il suffit de saisir un mot dans la zone de recherche. Tous les exemples unitaires correspondant à ce mot sont alors affichés.  

- Ouvrir un exemple unitaires dans le projet en cours : Il suffit de double-cliquer sur la ligne de l'exemple unitaire pour l'ouvrir dans le projet en cours. Vous pouvez alors enregistrer la fenêtre dans votre projet pour tester sont fonctionnement. 

- Voir la description de l'exemple unitaire : Il suffit de cliquer sur l'icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Volet_Exemples_unitaires%20-%20HC%20N%B0001%201.gif)
 à droite de la description rapide de l'exemple. Un bouton "Ouvrir l'exemple" permet alors d'afficher l'exemple sous l'éditeur. 

- Connaître  la plateforme d'utilisation de l'exemple unitaire : un picto indique la plateforme d'utilisation. 

- Filtrer les exemples unitaires par plateforme : Pour obtenir uniquement les exemples unitaires disponibles sur une plateforme spécifique, il suffit de cliquer sur la plateforme voulue dans les pictogrammes présents à droite de la zone de recherche.






<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Afficher le volet "Exemples unitaires"
<a name="afficher_volet_exemples_unitaires_ELTTEXTE000121"></a>
**Pour afficher le volet "Exemples unitaires"**, sous le volet "Accueil", dans le groupe "Environnement", déroulez "Volets" et sélectionnez "Volets" puis "Exemples unitaires".

Pour plus de détails sur les manipulations des volets, consultez [Manipuler les volets ancrables](../Editeurs/2027001.md).

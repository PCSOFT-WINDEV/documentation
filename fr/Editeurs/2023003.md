## Présentation
<a name="presentation_ELTTEXTE000180"></a>
WINDEV, WEBDEV et WINDEV Mobile permettent de réaliser simplement les manipulations suivantes sur un dictionnaire :

- [Création et ouverture d'un dictionnaire](#NOTE2_1). 

- [Ajout d'éléments dans un dictionnaire](#NOTE3_1).

- [Création de groupes d'éléments](#NOTE4_1).

- [Copie d'éléments depuis un dictionnaire](#NOTE5_1).




Toutes ces manipulations sont réalisées à partir du [volet "Dictionnaire"](../Editeurs/2027014.md).



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Créer ou ouvrir un dictionnaire
<a name="creer_ouvrir_dictionnaire_ELTTEXTE000204"></a>
Un dictionnaire correspond physiquement à un fichier ".DIC". Ce fichier est créé à partir du volet "Dictionnaire" (Option "Nouveau" du menu contextuel).

De même, l'ouverture d'un dictionnaire spécifique est réalisée à partir du volet "Dictionnaire" (option "Ouvrir" du menu contextuel). Le fichier sélectionné est ouvert dans le volet "Dictionnaire".

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Dico_ouvert.gif)


Pour afficher l'ensemble des éléments présents dans le dictionnaire :

1. Sélectionnez le nom du dictionnaire

2. Affichez le menu contextuel du dictionnaire (clic droit) et sélectionnez l'option "Dérouler tout".

## Ajouter des éléments dans un dictionnaire
<a name="ajouter_des_elements_dans_dictionnaire_ELTTEXTE000228"></a>


### Ajouter un élément dans un dictionnaire
<a name="ajouter_element_dans_dictionnaire_ELTPARAGRAPHE000057"></a>Pour ajouter un élément dans un dictionnaire :

1. Réalisez un "Drag and Drop" de l'élément choisi vers le dictionnaire. Selon l'objet à intégrer, le Drag and Drop peut être réalisé de différentes façons :

	- Directement depuis les éditeurs concernés (champ, groupe de champs, fenêtre, ligne de code, ...) :

		- Pour les champs, les groupes de champs ou les lignes de code : sélectionnez l'élément sous l'éditeur concerné et réalisez un Drag and Drop vers le dictionnaire.

		- Pour les pages, fenêtres et les requêtes : sélectionnez la barre de titre de la fenêtre d'accueil et réalisez un Drag and Drop vers le dictionnaire.




- Depuis le volet "Explorateur de projet" (fenêtre, fichier, rubrique ou procédure) : Sélectionnez l'élément dans le volet "Explorateur du projet" et réalisez un Drag and Drop vers le dictionnaire présent dans volet "Dictionnaire".

- Depuis le volet "Code" : Sélectionnez le nom de la procédure dans le volet "Code" et réalisez un Drag and Drop vers le dictionnaire.

2. Indiquez si l'élément de l'application en cours doit être abonné à l'élément présent dans le dictionnaire. En effet, lors de l'ajout d'un élément dans le dictionnaire, cet élément est dupliqué : l'application d'origine contient un exemplaire de l'élément et un autre exemplaire est copié dans le dictionnaire. 
	**Si l'abonnement est accepté** : En cas de modification de l'élément de l'application, l'abonnement permet de répercuter ces modifications à l'objet du dictionnaire, et par son intermédiaire à tous les objets abonnés. 
	Pour plus de détails sur l'abonnement, consultez [Abonnement d'un élément au dictionnaire](../Editeurs/2023001.md).




**Remarques** :

- Il est possible d'intégrer dans un dictionnaire de nombreux éléments : fenêtres, champs, procédures, lignes de code, fichiers ou rubriques. Ces différents éléments sont intégrés simplement par un "Drag and Drop" de l'objet vers le dictionnaire choisi.

- Les états ne peuvent pas être intégrés dans un dictionnaire.





<a name="NOTE3_2"></a>


### Exemple : Abonnement d'une procédure
<a name="exemple_abonnement_une_procedure_ELTPARAGRAPHE000087"></a>Pour conserver une procédure et l'incorporer facilement à vos applications, intégrez cette procédure à votre dictionnaire personnel !

1. Ouvrez le projet.

2. Ouvrez la fenêtre ou la page contenant la procédure.

3. Affichez les différentes procédures de la fenêtre (ou de la page) dans le volet "Explorateur de projet".

4. Ouvrez le dictionnaire voulu dans le volet "Dictionnaire".

5. Sélectionnez le nom de la procédure dans le volet "Explorateur de projet" et réalisez un "Drag and Drop" vers le dictionnaire.

6. Confirmez l'abonnement de la procédure. La procédure apparaît dans le dictionnaire.




**Remarque** : Pour associer une note descriptive à la procédure présente dans le dictionnaire, saisissez un texte explicatif dans les propriétés de l'élément (option "Propriétés" du menu contextuel)

## Créer des groupes d'éléments dans un dictionnaire
<a name="creer_des_groupes_elements_dans_dictionnaire_ELTTEXTE000258"></a>
Il est possible de créer des groupes d'éléments dans un dictionnaire. Ces groupes permettent de regrouper différents éléments WINDEV, WEBDEV ou WINDEV Mobile.

Pour créer un groupe d'éléments dans le dictionnaire en cours :

1. Dans le volet "Dictionnaire", sélectionnez le nom du dictionnaire.

2. Sélectionnez l'option "Créer un groupe" dans le menu contextuel (clic droit).

3. Saisissez le nom du groupe créé.




L'ajout d'éléments dans le groupe créé s'effectue par simple "Drag and Drop".

## Copier des éléments depuis un dictionnaire
<a name="copier_des_elements_depuis_dictionnaire_ELTTEXTE000282"></a>
Tous les éléments du dictionnaire peuvent être réutilisés dans n'importe quelle application WINDEV, WEBDEV ou WINDEV Mobile.

Pour réutiliser un élément du dictionnaire, il suffit de : 

1. Ouvrir le dictionnaire dans le volet.

2. Copier l'élément du dictionnaire vers l'éditeur concerné, par un simple "Drag and Drop". Lors de cette copie, l'abonnement de l'élément copié à l'objet du dictionnaire est proposé.




Pour plus de détails sur l'abonnement, consultez [Abonnement d'un élément au dictionnaire](../Editeurs/2023001.md).

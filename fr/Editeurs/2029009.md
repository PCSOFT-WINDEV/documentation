


## Editeur d'états : options et raccourcis clavier
			



<a name="NOTE4"></a>
<a name="NOTE4_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000245"></a>
L'éditeur d'états propose un menu sous forme de ruban regroupant toutes ses options. Les volets du ruban spécifiques à l'éditeur d'états sont les suivants : 

- Création : permet de créer tous les champs présents dans les états. 

- Modification : permet de modifier les caractéristiques de l'état. 

- Alignement : permet de gérer les options d'alignement des éléments présents dans l'état. 

- Affichage : permet de spécifier les options d'affichage de l'état et de ses champs sous l'éditeur.

## Les raccourcis clavier
<a name="les_raccourcis_clavier_ELTTEXTE000269"></a>
Voici les principaux raccourcis clavier utilisables sous l'éditeur d'états :

| Action | Raccourci |
| --- | --- |
| Activer / Désactiver le Live Data | F8 |
| Afficher la description du champ ou de la cellule | F4 |
| Afficher la liste des champs de l'état en cours | F12 |
| Afficher la liste des classes dans le volet "Explorateur de projet" (et dans le volet "Code" s'il est affiché) | Shift + F8 |
| Afficher la liste des procédures globales dans le volet volet "Explorateur de projet" (et dans le volet "Code" s'il est affiché) | Ctrl + F8 |
| Afficher le nom des blocs complet / Afficher le nom des blocs en abrégé | F10 |
| Afficher / Rendre invisible le bord des champs | F5 |
| Afficher / Rendre invisible le nom des champs | F6 |
| Afficher / Rendre invisible les marges de l'état | F11 |
| Afficher / Rendre invisible les règles | Ctrl + R |
| Aide | F1 |
| Annuler la dernière action | Ctrl + Z |
| Arrêter le mode test (lors de test du projet ou de l'état) | Shift + F5 |
| Coller le contenu du presse papier | Ctrl + V ou Shift + Inser |
| Copier le style de l'élément sélectionné | Ctrl + Shift + C |
| Copier l'élément sélectionné | Ctrl + C ou Ctrl + Inser |
| Couper l'élément sélectionné | Ctrl + X ou Alt + Suppr |
| Déplacer le champ ou la colonne sélectionnée à droite | Flèche droite |
| Déplacer le champ ou la colonne sélectionnée à gauche | Flèche gauche |
| Afficher le code associé à l'élément sélectionné dans le volet "Visualisation contextuelle du code" | Ctrl + Shift + F2 |
| Éditer le code de la sélection | F2 |
| Enregistrer l'état | Ctrl + S |
| Enregistrer sous | Ctrl + Shift + S |
| Enregistrer tout | Alt + S |
| Fermer tous les éléments ouverts | Ctrl + Shift + W |
| Imprimer le dossier concernant l'état en cours | Ctrl + P |
| Masquer les volets horizontaux | Ctrl + Q |
| Masquer tout (volets ancrables) | Ctrl + W |
| Mettre en gras le champ sélectionné | Ctrl + G |
| Mettre en italique le champ sélectionné | Ctrl + I |
| Nouveau (Création d'un élément) | Ctrl + N |
| Ouvrir (Ouverture d'un élément) | Ctrl + O |
| Ouvrir un projet | Ctrl + Shift + O |
| Rétablir (Restaurer la dernière action) | Ctrl + Y ou Ctrl + Shift + Z |
| Souligner le champ sélectionné | Ctrl + U |
| Supprimer la sélection | Suppr |
| Tracer le projet | Alt + F9 |
| Tester le projet | Ctrl + F9 |
| Tester l'état en cours | F9 |
| Visualiser l'état en 3D | Alt + D |
| Zoom à 100 % | Ctrl + Alt + 0 |
| Zoom + / - | Ctrl + roulette de souris |

Attention : Le signe "+" signifie que les touches doivent être utilisées simultanément

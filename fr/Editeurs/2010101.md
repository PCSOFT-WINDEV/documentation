


## Créer une modélisation souple
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000169"></a>
Pour créer une nouvelle modélisation souple, il est possible de : 

- Cliquer sur ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_nouveau.gif) parmi les boutons d'accès rapide. 

	- La fenêtre de création d'un nouvel élément s'affiche : cliquez sur "Architecture" puis sur "Modélisation Souple".

	- L'assistant de création d'une nouvelle modélisation souple se lance.




- Utiliser l'option "Nouvelle modélisation souple" du menu contextuel du volet "Explorateur de projet". [L'assistant de création d'une nouvelle modélisation](../Editeurs/2010101.md) se lance.

<a name="NOTE2_1"></a>


## Assistant de création de modélisation souple
<a name="assistant_creation_modelisation_souple_ELTTEXTE000193"></a>
L'assistant de création d'une modélisation souple vous permet de :

- indiquer le titre de la modélisation souple.

- choisir le contenu initial de la modélisation.

	- ***Vide*** : Modélisation vierge

	- ***Simple*** : Modélisation utilisant une base de données

	- ***Avec composants*** : Modélisation utilisant une base de données et des composants.


 Quelque soit le contenu choisi, il sera ensuite entièrement personnalisable.

- choisir les propriétés avancées de la modélisation. Ces propriétés concernent :

	- l'insertion automatique de cartouches : permet de configurer la présence automatique d'un cartouche (un titre) dans la modélisation principale et dans les sous-modélisations. Cette option pourra être modifiée à tout moment dans l'éditeur.

	- l'utilisation automatique des intervenants du Centre de Suivi de Projets dans la liste des acteurs du modèle. Cette option pourra être modifiée à tout moment dans l'éditeur.





La description du modèle est terminée. Le modèle apparaît sous l'éditeur.

## Les différents éléments pouvant être insérés dans un modèle
<a name="les_differents_elements_pouvant_etre_inseres_dans_modele_ELTTEXTE000217"></a>
L'éditeur de modélisation souple permet d'insérer de nombreux types d'éléments dans votre modélisation. Ces éléments sont disponibles directement dans le volet "Création" de l'éditeur de modélisation.

- Commencement (![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Modelisation_Commencement_i.gif)) : Permet de visualiser le commencement de la modélisation souple.

- Terminaison (![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Modelisation_Termine_ico.gif)) : Permet de visualiser la fin de la modélisation souple.

- Décision/Choix (![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Modelisation_Choix_ico.gif)) : Permet de visualiser simplement un choix dans la modélisation souple.

- Traitement (![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Modelisation_Trait_Ico.gif)) : Permet de visualiser simplement un traitement.

- Application (![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Modelisation_Appli_ico.gif)) : Permet de visualiser simplement une application.

- Fichier de données (![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Modelisation_HF_ico.gif)) : Permet de visualiser simplement un fichier de données.

- Image (![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Modelisation_Image_ico.gif)) : Permet d'inclure simplement une image (du catalogue d'image ou une image personnelle) dans la modélisation souple. Cette image a les mêmes caractéristiques qu'un traitement.

- Cartouche (![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Modelisation_Cart_ico.gif)) : Permet d'inclure un cartouche dans la modélisation souple. Permet par exemple de donner un titre à une modélisation souple ou une sous-modélisation.


Il est également possible d'insérer des formes simples (cercle, ellipse, rectangle, rectangle arrondi) et de lier tous ces éléments.
<a name="NOTE3_2"></a>


### Description des éléments
<a name="description_des_elements_ELTPARAGRAPHE000082"></a>Chaque élément d'une modélisation souple peut être configuré grâce à sa fenêtre de description (option "Description" du menu contextuel de l'élément).

Pour plus de détails, consultez [Description d'un élément](../Editeurs/2010102.md).
<a name="NOTE3_3"></a>


### Ajouter un élément du projet dans une modélisation souple
<a name="ajouter_element_projet_dans_une_modelisation_souple_ELTPARAGRAPHE000094"></a>Si vous effectuez une modélisation souple d'un projet WINDEV, WEBDEV ou WINDEV Mobile, il est possible d'insérer directement les éléments de votre projet dans votre modélisation. Il suffit de faire un "Drag and Drop" de l'élément du projet (présent par exemple dans le volet "Explorateur de projet" ou "Analyse") vers la modélisation.

Lorsque l'élément est inséré dans la modélisation souple, l'icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=modelisation_liaison.gif) indique que l'élément est lié à un objet du projet. Il est alors possible d'ouvrir directement cet objet :

- soit en double-cliquant sur la flèche.

- soit en utilisant l'option "Ouvre l'élément lié" du menu contextuel.



<a name="NOTE3_4"></a>


### Sous-modélisation
<a name="sousmodelisation_ELTPARAGRAPHE000108"></a>Les éléments de type "Application", "Traitement" et "Choix" peuvent permettre de rentrer dans une "sous-modélisation". Par exemple, la description de l'ERP (Enterprise Ressource Planning) d'une entreprise peut être composée des sous-modélisations suivantes : Gestion des clients, gestion des devis, marketing, Production, Comptabilité, ...

Pour plus de détails, consultez [Gestion des sous-modélisations](../Editeurs/2010104.md).
<a name="NOTE3_5"></a>


### Gestion de l'altitude des objets
<a name="gestion_altitude_des_objets_ELTPARAGRAPHE000120"></a>Tous les éléments présents dans l'éditeur de modélisation sont superposables. Il est possible de mettre un élément en premier plan, en arrière-plan, monter ou descendre l'élément d'un niveau. Il est ainsi possible par exemple de mettre une image en arrière-plan de la modélisation.


## Tester un site WEBDEV sur un mobile
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000114"></a>
Désormais, un site internet est très fréquemment utilisé sur des périphériques mobiles. WEBDEV permet de tester le comportement de ses projets et de ses pages directement sur un téléphone. 

Il est ainsi possible de vérifier l'affichage des pages "mobile-friendly" de votre site. 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000138"></a>


### Pré-requis
<a name="prerequis_ELTPARAGRAPHE000018"></a>

- Le périphérique mobile et le poste de développement doivent être sur le même réseau (Wifi par exemple).

- Le serveur WEB doit autoriser plusieurs connexions simultanées. Ce mode de test n'est pas disponible si vous utilisez le serveur Web intégré IIS Express. 



<a name="NOTE2_2"></a>


### Lancement du test
<a name="lancement_test_ELTPARAGRAPHE000026"></a>Le test sur mobile peut aussi bien être effectué pour le projet que pour une page. 

**Pour tester un projet sur mobile** : 

1. Déroulez l'icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_GO_Projet_WB_bl.gif) présente dans les boutons d'accès rapide. 

2. Sélectionnez l'option "Navigateur de test .. Téléphone Mobile (via un QR Code)".

3. Lancez le test du projet via l'icône "GO".

4. Une fenêtre avec un QR Code est affichée. <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=P8_Agencement_simple_1%20-%20HC%20N%B0016.jpg)


5. Flashez avec votre mobile le QR Code affiché. 

6. La première page du projet est affichée sur le mobile. 




**Pour tester la page WEBDEV affichée sous l'éditeur sur mobile** : 

1. Déroulez l'icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_Go_Page_bl.gif) présente dans les boutons d'accès rapide. <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=GO_PAGE_WB_deroule.gif)


2. Sélectionnez l'option "Tester depuis un téléphone (accès via un QR Code)". 

3. Une fenêtre avec un QR Code est affichée.   <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=P8_Agencement_simple_1%20-%20HC%20N%B0016.jpg)


4. Flashez avec votre mobile le QR Code affiché. 

5. La page en cours est affichée sur le mobile. 


Remarque : Tant que le mode de test n'est pas modifié, les Go de page suivants seront effectués sur le mobile (avec scan du QR Code). 




### Remarques
<a name="remarques_ELTPARAGRAPHE000058"></a>

- Les points d'arrêt (code serveur) sont actifs sur le téléphone. 

- Il est possible de positionner un point d'arrêt après le lancement du test. 

- Le mode débogage est disponible (mode pas à pas, visualisation des variables, etc.). 

- Il est possible de définir le navigateur de test par défaut dans les options de WEBDEV. Pour plus de détails, consultez [Options générales de WINDEV, WEBDEV, WINDEV Mobile](../Editeurs/2036001.md).

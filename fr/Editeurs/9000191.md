


## Composants utilitaires
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000131"></a>
WINDEV propose plusieurs composants utilitaires permettant de manipuler directement des données du produit. Vous pouvez les intégrer directement dans vos applications. 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000155"></a>
Pour intégrer un composant utilitaire dans votre projet : 

1. Sous le volet "Projet", dans le groupe "Projet", déroulez "Importer". Sélectionnez "Un composant Externe" puis "Composant utilitaire ou exemple". 

2. Une fenêtre apparaît avec la liste des composants utilitaires disponibles : 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Gestion%20des%20composants%2021%20-%20HC%20N%B0001.gif)


3. Sélectionnez le composant à importer dans votre projet. 

4. Validez. Le composant est importé. Il apparaît dans le volet "Explorateur de projet". 




<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Les différents composants disponibles
<a name="les_differents_composants_disponibles_ELTTEXTE000179"></a>
Les différents composants sont livrés avec leur documentation permettant d'utiliser les fonctionnalités du composant. 
Le composant **AccesExterne** est un composant permettant de consulter les données des Centre de Contrôle. Vous pouvez vous connecter à la base de données des Centres de Contrôle (HFSQL Classic, HFSQL Client/Serveur, [PCSCloud](https://pcscloud.net/FR/cloud_developpement.awp) ou GDSDrive) pour exporter au format JSON les données concernant les projets, les intervenants, les tâches, les incidents, les exigences. 

Pour plus de détails sur les Centres de Contrôle, consultez [Les Centres de Contrôle](../Editeurs/9000043.md). 
Le composant **CCMenu** permet de gérer certaines fonctionnalités proposées par le menu '?'. 

Pour plus de détails, consultez [Menu "?"](../Editeurs/9000070.md). 
Le composant **CCHCEdit** permet d'annoter les images. 
Le composant **CCFeedBack** permet de gérer les remontées des utilisateurs. Pour plus de détails, consultez [Composant Feedback](../Editeurs/2014013.md). 
Le composant **ComposantsRobot** correspond au composant de gestion du robot de surveillance. 

Pour plus de détails, consultez [Présentation du robot de surveillance](../RobotMonitor/3541100.md).
Le composant **TipOfTheDay** correspond au composant permettant de gérer les astuces du jour.

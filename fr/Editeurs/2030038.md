


## Restaurer une sauvegarde d'un projet 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000087"></a>
WINDEV, WEBDEV et WINDEV Mobile permettent de sauvegarder régulièrement les différents éléments d'un projet sous forme d'archive (fichier d'extension ZIP). Ce fichier contient tous les éléments du projet sauvegardés. Pour plus de détails, consultez [Sauvegarder un projet](../Editeurs/2030014.md). 

A tout moment, il est possible de restaurer le projet archivé. La restauration d'un projet est effectuée depuis WINDEV, WEBDEV ou WINDEV Mobile.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000111"></a>


### Pour restaurer une archive d'un projet :
<a name="pour_restaurer_une_archive_projet_ELTPARAGRAPHE000023"></a>

1. Sous le volet "Projet", dans le groupe "Autres actions", déroulez "Sauver/Restaurer" et sélectionnez "Restaurer un projet". L'assistant de restauration s'affiche.

2. Spécifiez le fichier archive contenant le projet à restaurer.

3. Sélectionnez le répertoire de la restauration.

4. Validez. La restauration est effectuée dans le répertoire spécifié.

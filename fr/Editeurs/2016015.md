


## Feuille de styles personnalisée 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000181"></a>
A la création d'un projet, une feuille de styles personnalisée vide est créée et associée au projet en cours. Cette feuille de styles est modifiable directement depuis l'éditeur de fenêtres et contiendra les différents styles créés par le développeur.

Cette feuille de styles est créée dans le répertoire du projet et a pour nom "&lt;NomDuProjet&gt;.STY".

Il est possible à tout moment d'associer une autre feuille de styles personnalisée au projet.

Cette page présente comment :

- [Associer un projet à une feuille de styles personnalisée quelconque](#NOTE2).

- [Créer, modifier ou supprimer un style dans une feuille de style personnalisée](#NOTE3).




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment associer un projet à une feuille de styles personnalisée quelconque ?
<a name="comment_associer_projet_une_feuille_styles_personnalisee_quelconque_ELTTEXTE000205"></a>
Pour associer le projet en cours à une feuille de styles personnalisée quelconque : 

1. Affichez la description du projet : sous le volet "Projet", dans le groupe "Projet", cliquez sur "Description".

2. Affichez l'onglet "Style".

3. Sélectionnez la feuille de styles personnalisée (fichier ".STY") à associer au projet dans l'option "Feuille de styles".




Remarques :

- La même feuille de styles personnalisée peut être associée à plusieurs projets.

- A l'ouverture d'une fenêtre utilisant des styles de l'ancienne feuille de styles du projet, il sera possible :

	- *soit de ne rien changer*. Les champs utilisant les styles de l'ancienne feuille de styles ne seront pas modifiés. Il est conseillé d'associer ces champs à des styles de la nouvelle feuille de styles.
			Les styles de l'ancienne feuille de styles pourront être ajoutés à la nouvelle feuille de styles personnalisée à la prochaine ouverture de la fenêtre.

	- *soit d'ajouter les styles de l'ancienne feuille de styles* à la nouvelle feuille de styles personnalisée.

## Comment créer, modifier ou supprimer un style d'une feuille de style personnalisée ?
<a name="comment_creer_modifier_supprimer_style_une_feuille_style_personnalisee_ELTTEXTE000229"></a>


### Créer un style pour un type de champ dans la feuille de styles personnalisée du projet en cours
<a name="creer_style_pour_type_champ_dans_feuille_styles_personnalisee_projet_cours_ELTPARAGRAPHE000064"></a>Pour créer un style pour un type de champ dans la feuille de styles personnalisée du projet en cours : 

1. Créez un champ de ce type dans une fenêtre de votre projet grâce au volet "Création". Par exemple, pour créer un nouveau style de bouton, créez un champ de type bouton.

2. Affichez la fenêtre d'édition des feuilles de style (option " Choisir un style" du menu contextuel du champ).

3. Cliquez sur le bouton "Créer". La fenêtre de description du style s'affiche.

4. Spécifiez les options de style voulues (onglets "Général" et "Style") et validez. Ce nouveau style est automatiquement ajouté à la feuille de style personnalisée du projet en cours. Ce style pourra être appliqué sur les champs de même type dans tous les projets utilisant cette feuille de styles.




Remarque : Il est également possible de créer un nouveau style à partir de la fenêtre de description d'un champ :

1. Affichez l'onglet "Style" de la fenêtre de description du champ (option "Description" du menu contextuel).

2. Modifiez les options de style voulues et validez (bouton "OK" ou "Appliquer"). La fenêtre de modification du style du champ s'affiche.

3. Sélectionnez le cadre "Ajouter ce style au projet", saisissez le nom de ce nouveau style et validez (bouton "OK").



<a name="NOTE3_2"></a>


### Modifier un style pour un type de champ dans la feuille de styles personnalisée du projet en cours
<a name="modifier_style_pour_type_champ_dans_feuille_styles_personnalisee_projet_cours_ELTPARAGRAPHE000082"></a>Pour modifier un style pour un type de champ dans la feuille de styles personnalisée du projet en cours :

1. Sélectionnez un champ de ce type dans une fenêtre de votre projet. Par exemple, pour modifier un style de bouton, sélectionnez un champ de type bouton.

2. Affichez la fenêtre d'édition des feuilles de styles (option "Choisir un style" du menu contextuel).

3. Sélectionnez le style à modifier.

4. Cliquez sur le bouton "Modifier". La fenêtre de description du style s'affiche.
	**Remarque** : Le bouton "Modifier" est grisé si le style sélectionné appartient à une feuille de styles non modifiable (feuille de styles de gabarit par exemple).

5. Effectuez les modifications voulues (onglets "Général" et "Style") et validez ces modifications (bouton "OK").

6. Appliquez les modifications effectuées (bouton "Appliquer"). Les modifications sont automatiquement reportées sur le champ en cours et sur les champs utilisant ce style dans les fenêtres ouvertes sous l'éditeur de fenêtres.




Remarques :

- Pour appliquer les modifications effectuées sur tous les champs utilisant ce style dans le projet en cours, sous le volet "Projet", dans le groupe "Projet", déroulez "Recompiler et synchroniser" et sélectionnez "Réactualiser style et gabarit".

- Il est également possible de modifier un style à partir de la description d'un champ (onglet "Style").

- Pour modifier le style de plusieurs champs de même type, sélectionnez les champs (en maintenant la touche Ctrl par exemple) avant d'ouvrir la fenêtre d'édition des feuilles de styles.



<a name="NOTE3_3"></a>


### Supprimer un style pour un type de champ dans la feuille de styles personnalisée du projet en cours
<a name="supprimer_style_pour_type_champ_dans_feuille_styles_personnalisee_projet_cours_ELTPARAGRAPHE000116"></a>Pour supprimer un style pour un type de champ dans la feuille de styles personnalisée du projet en cours :

1. Sélectionnez un champ de ce type dans une fenêtre de votre projet. Par exemple, pour supprimer un style de bouton, sélectionnez un champ de type bouton.

2. Affichez la fenêtre d'édition des feuilles de styles (option "Choisir un style" du menu contextuel).

3. Sélectionnez le style à supprimer.

4. Cliquez sur le bouton "Supprimer" et confirmez la suppression du style.
	**Remarque** : Le bouton "Supprimer" est grisé si le style sélectionné appartient à une feuille de styles non modifiable (feuille de styles de gabarit par exemple).

5. Sélectionnez le nouveau style du champ en cours et appliquez ce nouveau style (bouton "Appliquer").




Remarques :

- Si d'autres champs du projet utilisent le style supprimé, il est conseillé de sélectionner l'option "Ne rien faire" à l'ouverture des fenêtres et d'associer ces champs à un style différent. Pour plus de détails, consultez [Changer le style associé à un champ](../Editeurs/2016006.md).

- Il est également possible de supprimer un style à partir de la description d'un champ (onglet "Style").

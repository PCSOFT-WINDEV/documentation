


## Editeur de modélisation : Gestion des styles
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation et principe
<a name="presentation_principe_ELTTEXTE000127"></a>
L'éditeur de modélisation propose une gestion des styles avancée. Cette gestion des styles permet de définir et d'utiliser simplement des styles prédéfinis pour les différents objets d'une modélisation souple.

**Le principe est simple** : 

1. **La charte graphique** : Dès que le style d'un élément est modifié dans une modélisation souple, une charte graphique est automatiquement liée à la modélisation souple. Cette charte graphique est une sorte de "feuille de styles" de la modélisation souple. Le fichier associé à la charte graphique est un fichier qui a pour nom "Charte_&lt;Nom modélisation&gt;.wsm". Ce fichier est enregistré à côté de la modélisation souple et peut être utilisé par plusieurs modélisations.

2. **Lors de la première modification du style d'un élément, plusieurs solutions sont possibles** : 

	- **Créer un nouveau style dans la modélisation** : Cette option permet d'ajouter un style dans la charte graphique. Tous les éléments du même type créé utiliseront ce nouveau style.

	- **Modifier cet élément uniquement** : Cette option permet de modifier uniquement l'élément sélectionné. Cet élément n'est donc pas lié à la charte graphique.




3. **Lors de la modification du style d'un élément présent dans la charte graphique, plusieurs solutions sont possibles** : 

	- **Surcharger le style de l'élément** : Dans ce cas, seul l'élément en cours sera modifié. L'élément sera toujours lié au style défini dans la charte graphique et les évolutions du style associé à l'élément seront automatiquement prises en compte.

	- **Créer un nouveau style** : Il suffit d'indiquer le nom du nouveau style à créer dans la charte graphique.

	- **Modifier le style dans la charte graphique** : Dans ce cas, la modification sera reportée sur tous les éléments utilisant le même style.

	- **Dissocier l'élément** : Dans ce cas, les modifications de style seront appliquées à l'élément en cours uniquement. Cet élément ne sera plus lié au style et ne prendra plus en compte les évolutions du style.




4. Si plusieurs styles sont associés au même type d'élément, il est possible de sélectionner le style de l'élément grâce à l'option "Choisir un style..." du menu contextuel de l'élément.

## Manipulation des styles et de la charte graphique
<a name="manipulation_des_styles_charte_graphique_ELTTEXTE000151"></a>


### Associer une charte graphique à une modélisation
<a name="associer_une_charte_graphique_une_modelisation_ELTPARAGRAPHE000033"></a>Pour associer une charte graphique à une modélisation : 

1. Sous le volet "Modélisation", dans le groupe "Général", déroulez "Charte graphique" et sélectionnez "Sélectionner une charte graphique".

2. Dans la description de la modélisation, sélectionnez la charte graphique à utiliser.



<a name="NOTE2_2"></a>


### Associer un style à un élément
<a name="associer_style_element_ELTPARAGRAPHE000055"></a>Pour associer un style à un élément : 

1. Sélectionnez l'élément.

2. Sélectionnez l'option "Choisir un style..." du menu contextuel.

3. Sélectionnez le style voulu et validez.



<a name="NOTE2_3"></a>


### Actualiser les styles d'une modélisation suivant la charte graphique sélectionnée
<a name="actualiser_les_styles_une_modelisation_suivant_charte_graphique_selectionnee_ELTPARAGRAPHE000066"></a>Pour actualiser les styles d'une modélisation suivant la charte graphique sélectionnée, sous le volet "Modélisation", dans le groupe "Général", déroulez "Charte graphique" et sélectionnez "Réactualiser les styles depuis la charte graphique".

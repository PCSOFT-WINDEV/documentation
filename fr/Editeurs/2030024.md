


## Comparateur de projets 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000244"></a>
Comparer deux fenêtres (deux pages, deux états, deux classes, ...) est un besoin courant dans la vie d'une application.

Par exemple, une fenêtre qui fonctionnait il y a deux semaines a été modifiée par un stagiaire. Cette fenêtre ne fonctionne plus désormais. Comment connaître les modifications effectuées dans cette fenêtre ? Il suffit d'utiliser le comparateur de projets !

Le comparateur de projets permet de comparer deux projets ou deux éléments (fenêtres, pages, états, ...) entre eux.

**La comparaison est effectuée** :

- sur l'UI de l'élément et de ses sous-éléments (par exemple, une fenêtre et les champs de cette fenêtre). 
	**Remarque** : Si la fenêtre contient des [agencements](../Editeurs/9000153.md), la comparaison est effectuée sur chaque agencement.

- sur le code associé à l'élément et à ses sous-éléments (par exemple, un état et les champs de cet état).




Toutes les différences trouvées entre les deux éléments comparés sont listées.

Les éléments suivants peuvent être comparés :

- Projets.

- Tous les objets du projet. 

- Fenêtres ou modèles de fenêtres.

- Pages.

- Etats.

- Requêtes.

- Analyses.

- Classes.

- Collections de procédures.

- Analyses.

- Feuilles de styles.

- Tests automatiques.

- Descriptions de Webservice REST.

- Plans d'action.

- Composants internes.




Remarque : Lors de la comparaison de projets, les éléments suivants sont également comparés : 

- Procédures stockées.

- Requêtes stockées.

- Triggers.

- Propriétés du projet (numéros de version, informations de copyright, etc.).

- Images.

## Comparer des éléments
<a name="comparer_des_elements_ELTTEXTE000268"></a>
Le comparateur d'éléments permet de comparer :

- soit l'élément en cours avec un autre élément.

- soit deux éléments.

- soit trois éléments.

- soit deux projets.



<a name="NOTE2_2"></a>


### Comparer l'élément en cours avec un autre élément
<a name="comparer_element_cours_avec_autre_element_ELTPARAGRAPHE000090"></a>Pour comparer l'élément en cours avec un autre élément :

1. Sous le volet "Projet", dans le groupe "Autres actions", déroulez "Comparer" et sélectionnez "Comparer l'élément en cours". L'assistant de comparaison se lance.

2. Sélectionnez l'élément à comparer avec l'élément en cours.

3. Validez. L'ensemble des différences trouvées entre les deux éléments est listé.



<a name="NOTE2_3"></a>


### Comparer deux éléments
<a name="comparer_deux_elements_ELTPARAGRAPHE000113"></a>Pour comparer deux éléments :

1. Sous le volet "Projet", dans le groupe "Autres actions", déroulez "Comparer" et sélectionnez "Comparer deux éléments". L'assistant de comparaison se lance.

2. Sélectionnez le type d'élément à comparer (projet, fenêtre, page, état, ...).

3. Sélectionnez les deux éléments à comparer.

4. Validez. L'ensemble des différences trouvées entre les deux éléments est listé.<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Comparateur_prj%20-%20HC%20N%B0001.gif&type=thumb)




<a name="NOTE2_4"></a>


### Comparer deux projets
<a name="comparer_deux_projets_ELTPARAGRAPHE000138"></a>Pour comparer deux projets :

1. Sous le volet "Projet", dans le groupe "Autres actions", déroulez "Comparer" et sélectionnez "Comparer deux projets". L'assistant de comparaison se lance.

2. Sélectionnez les deux projets à comparer.

3. Validez. L'ensemble des différences trouvées entre les deux projets est listé.

## Différences trouvées entre deux éléments ou deux projets
<a name="differences_trouvees_entre_deux_elements_deux_projets_ELTTEXTE000310"></a>
Les différences sont classées selon l'ordre suivant :

- Différence de code : l'icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=OrdreAffichage.gif) permet de visualiser simultanément le code des deux éléments en cours de comparaison. <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Comparateur_prj%20-%20HC%20N%B0002.gif&type=thumb)


- Différence de propriétés : l'icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=OrdreAffichage.gif) permet de visualiser les caractéristiques différentes. <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Comparateur_prj%20-%20HC%20N%B0003.gif&type=thumb)

## Limitations
<a name="limitations_ELTTEXTE000340"></a>


### Comparaison de deux analyses
<a name="comparaison_deux_analyses_ELTPARAGRAPHE000191"></a>Lors de la comparaison de deux analyses, les caractéristiques suivantes sont ignorées :

- les dimensions et la position des différents objets graphiques représentant l'analyse.

- les informations partagées.

- les options présentes dans l'onglet "Options" de la fenêtre de description de l'analyse.

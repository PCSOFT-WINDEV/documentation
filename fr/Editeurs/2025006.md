<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000170"></a>
Les options générales de l'exécutable sont les suivantes :

- **le nom de l'exécutable** (fichier ".EXE"). Par défaut, le nom "&lt;NomProjet&gt;.EXE" est proposé.

- **l'icône du programme exécutable** (fichier ".ICO"). Cette icône peut être choisie dans le catalogue d'icônes de WINDEV (bouton "Catalogue").

- **l'image d'introduction du programme exécutable** (appelée "Splash Screen"). Cette image sera affichée à l'exécution pendant l'initialisation de l'application et le chargement des différentes librairies WINDEV/WINDEV Mobile.




Ces options générales sont définies dans l'assistant de création d'exécutable.

## Icône du programme exécutable
<a name="icone_programme_executable_ELTTEXTE000194"></a>
Pour associer une icône au programme exécutable, vous pouvez :

- utiliser une icône du catalogue d'icônes de WINDEV.

- choisir un fichier icône (.ICO) sur votre poste.


Ce choix est effectué via le bouton déroulant du champ "Nom de l'icône" :  ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Menu_Image_Editeur%20-%20HC%20N%B0002%202.gif)


Lors du choix d'un fichier icône (.ICO) sur votre poste, il est nécessaire de s'assurer que :

- soit le fichier icône est un fichier au format ICO 16 couleurs (Format Windows 3.x). Ces icônes n'ont pas de spécificités particulières.

- soit le fichier icône est un fichier au format ICO 256 (Format Win32, utilisé sous Windows 98, Me et 2000). Ces icônes n'ont pas de spécificités particulières.

- soit le fichier icône est un fichier utilisant un **groupes d'icônes** de différentes tailles et de différentes résolutions de couleurs. Les groupes d'icônes (également appelés icônes au format Windows XP) doivent répondre au standard suivant :

	- les icônes doivent avoir été créés par un outil classant les icônes par taille croissante et par résolution de couleurs décroissante.

## Image d'introduction de l'exécutable (Splash Screen)
<a name="image_introduction_executable_splash_screen_ELTTEXTE000218"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png) 

### Splash Screen d'une application WINDEV
<a name="splash_screen_une_application_windev_ELTPARAGRAPHE000072"></a>Le "Splash Screen" sera affiché à l'exécution pendant l'initialisation de l'application et le chargement des différentes librairies WINDEV. Lors de la création de l'exécutable, il est possible d'utiliser :

- un **Splash Screen classique** : une image sera affichée pendant l'initialisation. Cette image peut être choisie dans le catalogue d'images fourni par WINDEV.
	Les formats d'images supportés sont :

	- Bitmap (\*.BMP)

	- Graphics Interchange Format (\*.gif)

	- Joint Picture Experts Group (\*.jpg ; \*.jpeg)

	- PaintBrush (\*.pcx)

	- Adobe Photoshop Format (\*.psd)

	- TrueVision TARGA (\*.tga)

	- Tagged Image File Format (\*.tif ; \*.tiff)
			Remarque : l'option "tiff mosaïque" de la norme tiff 6.0 n'est pas supportée. Dans ce cas, il est conseillé d'enregistrer l'image à la norme tiff 5.0. 

	- Portable Network Graphics (\*.png)

	- Windows MetaFiles (\*.emf ; \*.wmf)

	- Icônes (\*.ico ; \*.icw)

	- Curseurs (\*.cur)


 Remarque : Si l'image n'est pas au format BMP, la librairie wdxxxpnt.dll doit être associée au programme exécutable.
	Ce type de splash screen est conseillé si le traitement d'initialisation de l'application est relativement long.
	Il est possible de :

- ***paramétrer les différentes informations affichées automatiquement*** sur cette image (bouton "Avancé"). Ces informations sont les suivantes :

	- *Numéro de version de l'exécutable* : affiche ou non le numéro de version de l'exécutable. Vous pouvez paramétrer la taille, la position, la police utilisée pour ce texte.

	- *Titre de l'application* : affiche ou non le titre de l'application. Par défaut, ce titre correspond au nom du projet. Vous pouvez paramétrer la taille, la position, la police utilisée pour ce texte.

	- *Message affiché* : permet de paramétrer les caractéristiques des messages affichés par programmation. Ces messages peuvent être affichés grâce à la fonction [Message](../WDLang1/3021006.md) utilisée dans le code d'initialisation du projet. Il est par exemple possible d'afficher un message pour indiquer les différentes étapes de l'initialisation.
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=SplashImage.gif&type=thumb)

- un **splash screen avec texte animé** : le texte indiqué (20 caractères maximum) sera affiché dans une animation. L'animation peut être visualisée en cliquant sur le bouton "Voir l'animation". Cette animation est paramétrable grâce au bouton "Options". Il est possible de :

	- choisir la couleur de fond du splash screen.

	- afficher le splash screen en plein écran ou non.

	- afficher le splash screen uniquement lors du premier lancement de l'application dans la journée.![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=SplashAnimation.gif)





- **pas de splash screen** : aucune image ne sera affichée pendant l'initialisation.




**Remarque** : L'utilisation d'un splash screen peut faire augmenter de manière conséquente la taille de l'exécutable. Par exemple, l'utilisation d'un splash screen avec texte animé peut faire augmenter de 400 Ko la taille de l'exécutable.

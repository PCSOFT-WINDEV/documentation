


## Messagerie WDBal : Lecture des messages
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000196"></a>
WDBal, la messagerie intégrée et sécurisée de WINDEV, WEBDEV et WINDEV Mobile permet très simplement de lire les messages reçus.

Pour lire les messages depuis WDBal : 

- Affichage en mode "Mini-barre" : cliquez sur ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Messagerie_Presentation%20-%20HC%20N%B0001%203.gif)
. 

- Affichage en mode complet : Par défaut, WDBal affiche les messages reçus en attente. 

- Affichage dans le Centre de Suivi de Projets : Par défaut, l'onglet messagerie affiche les messages reçus en attente.




Plusieurs types de messages peuvent être lus dans la messagerie : 

- Les messages reçus en attente. 

- Les messages reçus avec AR (Accusé de Réception). 

- Les messages envoyés en attente. 

- Les messages envoyés avec AR (Accusé de Réception).

<a name="NOTE2_1"></a>


## Lire les messages
<a name="lire_les_messages_ELTTEXTE000220"></a>
Les messages sont affichés dans une table. Cette table affiche : 

- L'expéditeur.

- La date de réception.

- L'heure de réception.

- Le nombre de pièces jointes. 

- La première ligne du message. 




En cliquant sur le message voulu, sont affichés : 

- La liste des destinataires du message. 

- Le contenu du message. 



![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Messagerie_Message%20-%20HC%20N%B0001%202.gif)
Un clic sur les différents dossiers standard (Reçus en attente, ...) permet de visualiser les différentes catégories de messages.

## Traiter les messages
<a name="traiter_les_messages_ELTTEXTE000244"></a>
Les différentes manipulations réalisables sur un message sont proposées : 

- dans le ruban "Accueil" de WDBal. 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Messagerie_Message%20-%20HC%20N%B0001%201.gif)


- dans le ruban de l'onglet "Messagerie" du Centre de Suivi de Projets. 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Messagerie_Message%20-%20HC%20N%B0003%201.gif)





Il est possible de : 

- faire un AR sur le message : le message est lu et a été traité. Il passe dans la liste des messages reçus avec AR. 

- répondre au message. Plusieurs possibilités : 

	- Répondre à l'expéditeur : Un nouveau message est créé reprenant le message existant. Seul l'expéditeur sera mis en destinataire du nouveau message. Il est possible d'ajouter de nouveaux destinataires. 

	- Répondre à tous : Un nouveau message est créé reprenant le message existant. L'expéditeur et tous les autres destinataires seront mis en destinataires du nouveau message. Il est possible d'ajouter de nouveaux destinataires. 

	- Transférer : Un nouveau message est créé reprenant le message existant. Ce nouveau message n'a pas de destinataire par défaut. Il est nécessaire d'ajouter de nouveaux destinataires. 




- Imprimer le message (directement ou via un aperçu). 

- Copier le texte du message. 

- Déplacer le message vers un dossier personnel. 

- Faire une recherche dans le message.

## Utiliser les dossiers personnels
<a name="utiliser_les_dossiers_personnels_ELTTEXTE000268"></a>
Un message peut être transféré dans un dossier personnel. Ces dossiers apparaissent sur la gauche de la messagerie. 

**Pour gérer les dossiers** : 

- Sous WDBal, affichez l'onglet "Dossiers". 

	- L'option "Nouveau" permet de créer un dossier personnel : indiquez le nom du dossier et validez. 

	- L'option "Supprimer" permet de supprimer le dossier personnel sélectionné. 




- Dans l'onglet "Messagerie" du Centre de Suivi de Projets", il est possible de : 

	- Créer un nouveau dossier via l'option "Nouveau dossier" du ruban ou via l'option "Nouveau dossier" du menu contextuel des dossiers personnels.

	- Supprimer un dossier via l'option "Supprimer le dossier" du menu contextuel des dossiers personnels.





**Pour transférer un message dans un dossier** : 

1. Sélectionnez le message dans la table. 

2. Transférer le message : 

	- Sous WDBal, sous le volet "Accueil", cliquez sur "Vers dossier". 

	- Dans l'onglet "Messagerie" du Centre de Suivi de Projets", cliquez sur "Vers dossier". 




3. Sélectionnez le dossier voulu dans la liste qui s'affiche et indiquez si le message transféré doit être considéré traité (option "Faire AR"). Validez.

## Rechercher un message
<a name="rechercher_message_ELTTEXTE000292"></a>
Il est possible de réaliser une recherche : 

- parmi les messages reçus ou envoyés, 

- d'un intervenant spécifique, 

- pendant une période spécifique, 

- contenant des mots spécifiques. 




Pour effectuer une recherche d'un message : 

- Sous WDBal : 

	1. Cliquez sur l'onglet "Recherche". 

	2. Spécifiez dans le ruban, les options de recherche. 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Messagerie_Message%20-%20HC%20N%B0002.gif)


	3. Cliquez sur le bouton "Démarrer". La liste des messages correspondant à la recherche est affichée. Pour arrêter la recherche en cours, cliquez sur le bouton "Arrêter". 

- Dans l'onglet "Messagerie" du Centre de Suivi de Projets" : 

	1. Cliquez sur "Rechercher". 

	2. Spécifiez les options de recherche. 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Messagerie_Message%20-%20HC%20N%B0004.gif)


	3. Cliquez sur le bouton "Rechercher". La liste des messages correspondant à la recherche est affichée. Pour arrêter la recherche en cours, cliquez sur le bouton "Stop". 




Le ou les messages trouvés peuvent être traités comme tout autre message. Il est possible : 

- de le renvoyer, de le transférer, 

- de l'imprimer, ...




## Gérer visuellement les branches du GDS
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000104"></a>
Pour optimiser la gestion des branches de vos projets dans le GDS, le gestionnaire de sources propose une édition graphique des branches. Cette visualisation offre une vision synthétique de la vie du projet. 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Fenêtre de gestion des branches
<a name="fenetre_gestion_des_branches_ELTTEXTE000128"></a>


### Afficher la fenêtre de gestion des branches
<a name="afficher_fenetre_gestion_des_branches_ELTPARAGRAPHE000018"></a>Pour afficher la fenêtre de gestion des branches, sous le volet "GDS", dans le groupe "Projet", déroulez "Branches" et sélectionnez "Gérer des branches". 

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=GDS_Visu_Branche%20-%20HC%20N%B0001.gif)
Remarque : Si l'option "Afficher toutes les révisions" est décochée, les réintégrations intermédiaires ne seront pas affichées permettant ainsi d'obtenir une vision encore plus synthétique. 


### Options disponibles à partir de la fenêtre de gestion des branches
<a name="options_disponibles_partir_fenetre_gestion_des_branches_ELTPARAGRAPHE000037"></a>Le menu contextuel du rond présent dans la colonne "Branche" permet d'effectuer plusieurs opérations sur la branche sélectionnée. 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=GDS_Visu_Branche%20-%20HC%20N%B0002.gif)
Il est possible de : 

- **Changer la couleur de la branche** : Cette option permet de modifier la couleur associée à la branche. Dans le sélecteur de couleur qui s'affiche, sélectionnez la couleur voulue et validez. 
	Remarque : Cette couleur est également utilisée sous l'éditeur lorsque vous travaillez sur une branche. 

- **Changer le libellé de la branche** : Cette option permet de modifier le libellé associé à la branche. Dans la fenêtre qui s'affiche, indiquez le nouveau libellé et validez. 
	Remarque : Ce libellé est également utilisé sous l'éditeur lorsque vous travaillez sur une branche. 

- **Créer une branche à partir de cette révision** : Cette option permet de créer des branches a posteriori. L'assistant de création d'une branche se lance immédiatement. Pour plus de détails, consultez [Créer une branche](../Editeurs/2038029.md). 

- **Fusionner avec cette branche** : Cette option permet de récupérer les modifications effectuées dans la branche. Pour plus de détails, consultez [Récupérer les modifications effectuées dans une branche](../Editeurs/2038030.md). 

- **Supprimer cette branche** : Cette option permet de supprimer la branche sélectionnée (par exemple si celle-ci devient inutile. Dans ce cas, le GDS demande confirmation de la suppression de la branche avant d'effectuer l'opération (cette opération ne peut pas être annulée).

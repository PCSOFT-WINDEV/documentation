


## Règles d'alignement (Editeur)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000203"></a>
Sous l'éditeur de fenêtres, lors de la création ou la modification d'une fenêtre, il est possible de faire apparaître des règles d'alignement. 
Ces règles d'alignement peuvent également être affichées sous l'éditeur de pages et l'éditeur d'états.

Dans ces règles, des guides magnétiques sont affichés : tout champ approché d'un guide est automatiquement "attiré" par celui-ci. Ce magnétisme permet d'utiliser les guides pour faciliter le positionnement, l'alignement ou le redimensionnement des champs de la fenêtre, de la page ou de l'état.

## Comment le faire ?
<a name="comment_faire_ELTTEXTE000227"></a>


### Afficher les règles
<a name="afficher_les_regles_ELTPARAGRAPHE000028"></a>Pour afficher les règles, sous le volet "Affichage", dans le groupe "Aide à l'édition", cochez l'option "Règles".

Deux types de guides peuvent alors être utilisés :

- Les guides, permettant d'aligner et de disposer les champs à l'intérieur des fenêtres, des pages ou des états.

- Les guides de bordure, permettant de déterminer une bordure de taille identique de chaque côté de la fenêtre, de la page ou de l'état.




**Remarque** : L'option "Afficher les guides" du menu contextuel des règles permet de visualiser la fenêtre, la page ou l'état avec ou sans ses guides.

**Guides actifs**
Visible ou non, les guides sont toujours actifs : un champ déplacé vers un repère avec la souris est automatiquement accolé au guide correspondant.

Lorsque le champ est accolé au guide, le déplacement du guide permet de déplacer automatiquement tous les champs accolés à ce guide.

Note : Un champ déplacé vers un repère avec les touches du clavier n'est pas toujours accolé au guide correspondant. En effet, cette opération est réalisée au pixel près.
<a name="NOTE2_2"></a>


### Paramétrer les règles
<a name="parametrer_les_regles_ELTPARAGRAPHE000062"></a>Il est également possible de paramétrer les règles depuis les options d'édition de l'éditeur de fenêtres, de pages, ou d'états. Il suffit d'utiliser la touche F12 (onglet "Magnétisme").

Cette option permet de :

- rendre les règles visibles ou non.

- rendre les guides visibles ou non.

- choisir la couleur des guides.

## Manipuler les repères
<a name="manipuler_les_reperes_ELTTEXTE000257"></a>
Les repères sont facilement manipulables à l'aide de la souris. Lorsque la souris survole un repère, le curseur se transforme en une double flèche.
<a name="NOTE3_2"></a>


### Créer un repère
<a name="creer_repere_ELTPARAGRAPHE000082"></a>Pour créer un repère :

- 1ère méthode : Cliquez sur une des règles. Le repère est créé sur la graduation la plus proche.

- 2ème méthode : Sélectionnez l'option "Ajouter un repère" dans le menu contextuel (clic droit) de la règle. Le repère est créé sur la graduation la plus proche.



<a name="NOTE3_3"></a>


### Déplacer un repère
<a name="deplacer_repere_ELTPARAGRAPHE000092"></a>Pour déplacer un repère :

1. Cliquez sur le repère présent dans la règle. Le curseur se transforme en une double flèche.

2. Faites glisser le curseur vers la nouvelle position du repère.



<a name="NOTE3_4"></a>


### Supprimer des repères
<a name="supprimer_des_reperes_ELTPARAGRAPHE000102"></a>

- Pour supprimer un repère, sélectionnez l'option "Supprimer le repère" du menu contextuel (clic droit) d'un repère.

- Pour supprimer tous les repères d'une règle, sélectionnez l'option "Tout supprimer" du menu contextuel d'une règle.



<a name="NOTE3_5"></a>


### Copier les repères et les guides
<a name="copier_les_reperes_les_guides_ELTPARAGRAPHE000110"></a>Pour copier les repères et les guides :

1. Sélectionnez l'option "Copier les guides" du menu contextuel des règles.

2. Ouvrez la fenêtre (la page ou l'état) dans laquelle la copie des repères et des guides doit être effectuée.

3. Affichez si nécessaire les règles dans cette fenêtre (page ou état) : sous le volet "Affichage", dans le groupe "Aide à l'édition", cochez l'option "Règles".

4. Sélectionnez l'option "Coller" dans le menu contextuel des règles.

## Cas particulier: Les repères de bordure
<a name="cas_particulier_les_reperes_bordure_ELTTEXTE000305"></a>


### Définition et déplacement
<a name="definition_deplacement_ELTPARAGRAPHE000136"></a>Les repères de bordure permettent d'établir une "marge" dans la fenêtre (la page ou l'état) en cours. Les repères de bordure sont situés à l'extrémité des règles.

Pour déplacer la marge gauche ou droite, pointez sur la règle horizontale sur un repère de bordure. Lorsque le pointeur prend la forme d'une double flèche, faites glisser la marge.

Pour déplacer la marge du haut ou du bas, pointez sur la règle verticale sur un repère de bordure. Lorsque le pointeur prend la forme d'une double flèche, faites glisser la marge.
<a name="NOTE4_2"></a>


### Remarques
<a name="remarques_ELTPARAGRAPHE000147"></a>

- **Les repères de bordure ne peuvent être supprimés**. Pour ignorer les bordures dans une fenêtre ou un état, les repères de bordure peuvent être réduits au minimum.

- **Par défaut, les repères de bordure opposés sont solidaires** : le déplacement d'un repère de bordure entraîne le déplacement du repère de bordure opposé.
	Pour modifier uniquement un des repères de bordure sous l'éditeur de fenêtres, il suffit de :

	1. Cliquer sur le repère de bordure : le curseur de souris se transforme en une flèche double.

	2. Appuyer sur la touche Ctrl et maintenir cette touche enfoncée.

	3. Déplacer le repère de bordure avec la souris.

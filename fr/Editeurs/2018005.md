## Analyse du groupware utilisateur
<a name="analyse_groupware_utilisateur_ELTTEXTE000543"></a>
L'analyse du groupware utilisateur est la suivante :

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=AnalyseGPW.gif)




**Fichiers de données permettant d'identifier les utilisateurs** : ces fichiers peuvent être communs à plusieurs applications :


|   |   |
| --- | --- |
| ***GPWHistoriqueConnexion *** | Fichier contenant l'historique des connexions. Ce fichier est présent uniquement si l'historique des connexions a été demandé. |
| ***GPWUtilisateur*** | Fichier des utilisateurs. Contient toutes les caractéristiques des utilisateurs (login, nom, prénom, mot de passe, ...) |
| ***GPWUtilisateurConfiguration*** | Fichier des groupes d'utilisateurs. |

****

****

**Fichiers de données permettant de gérer les droits** : ces fichiers sont spécifiques à l'application. Ils ne peuvent pas être partagés avec une autre application.


|   |   |
| --- | --- |
| ***GPWConfiguration*** | Fichier contenant les descriptions des différentes configurations disponibles. Chaque configuration est associée à un groupe d'utilisateurs. |
| ***GPWConfigurationElément*** | Etat de chaque élément de l'application pour chaque configuration. Par exemple permet de stocker si pour l'utilisateur "Secrétaire", l'option "Menu statistiques" doit être grisé ou non. |
| ***GPWElément*** | Fichier content les différents éléments de l'application à configurer (champs des fenêtres, options de menu, ...) |

## Fichier GPWConfiguration
<a name="fichier_gpwconfiguration_ELTTEXTE000567"></a>


### Caractéristiques des rubriques :
<a name="caracteristiques_des_rubriques_ELTPARAGRAPHE000063"></a>

| Nom | Explication | Type | Taille | Clé |
| --- | --- | --- | --- | --- |
| Application | Nom de l'application | Chaîne de caractères | 255 caractères | Avec doublons |
| Application_Configuration | Clé composée des rubriques Application et Configuration | Clé composée |   | Avec doublons |
| Configuration | Nom du groupe d'utilisateurs ou de l'utilisateur | Chaîne de caractères | 255 caractères | Unique |
| Groupe | Définit si la configuration correspond à un groupe d'utilisateurs ou à un utilisateur | Booléen |   |   |
| Groupe_Configuration | Clé composée des rubriques Groupe et Configuration | Clé composée |   | Unique |




<a name="NOTE2_2"></a>


### Remarque
<a name="remarque_ELTPARAGRAPHE000122"></a>Pour toutes les clés et pour toutes les composantes des clés composées :

- Le sens de parcours est ascendant.

- Les options de recherche sont : Sensible à la casse, Sensible à l'accentuation, Sensible aux espaces et à la ponctuation.

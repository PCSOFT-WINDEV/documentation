


## Importation d'une fenêtre WINDEV dans un projet Mobile
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000121"></a>
Les fenêtres WINDEV n'ayant pas le même format en WINDEV Mobile, WINDEV Mobile offre la possibilité d'importer une fenêtre WINDEV dans un projet WINDEV Mobile.

**Rappel** : Les analyses, requêtes, collections de procédures, états, classes ont le même format en WINDEV standard et WINDEV Mobile. Ces éléments peuvent être partagés entre un projet WINDEV standard et un projet WINDEV Mobile. Pour plus de détails, consultez [Partage d'éléments WINDEV](../WDLang3/3023045.md).

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000145"></a>
Pour importer une fenêtre WINDEV dans un projet WINDEV Mobile :

1. Sous le volet "Projet", dans le groupe "Projet", déroulez "Importer" et sélectionnez "Des éléments WINDEV".

2. Sélectionnez le fichier fenêtre (extension WDW) à importer.

3. La fenêtre sélectionnée est automatiquement ouverte sous WINDEV Mobile. Pour intégrer cette fenêtre au projet en cours, il suffit de l'enregistrer. La fenêtre est enregistrée au format WPW.




**Remarque** : La taille de la fenêtre importée correspond à sa taille d'origine. Il est nécessaire de revoir l'UI et les dimensions de la fenêtre pour les adapter à la plateforme utilisée (Android, iOS, etc.).

## Notes
<a name="notes_ELTTEXTE000169"></a>


### Opérations effectuées lors de l'importation
<a name="operations_effectuees_lors_importation_ELTPARAGRAPHE000048"></a>

- Les champs n'existant pas dans WINDEV Mobile sont automatiquement supprimés (barre d'outils,...).

- La liste des erreurs rencontrées (fonctions n'existant pas en WINDEV Mobile par exemple) est affichée dans le volet "Erreurs de compilation".



<a name="NOTE3_2"></a>


### Dépendance des fichiers
<a name="dependance_des_fichiers_ELTPARAGRAPHE000056"></a>Les fenêtre sont importées avec leurs dépendances : tous les fichiers liés à la fenêtre importée sont également importés. Les images sont automatiquement converties dans un format reconnu par WINDEV Mobile. Par exemple, l'image de fond d'une fenêtre est automatiquement importée et convertie au format GIF.
<a name="NOTE3_3"></a>


### Différences lors de l'exécution des événements d'une fenêtre
<a name="differences_lors_execution_des_evenements_une_fenetre_ELTPARAGRAPHE000063"></a>Les différences entre les systèmes d'exploitation Windows et les systèmes pour mobile entraînent une gestion différente des événements WLangage associés aux fenêtres. L'ordre d'exécution des événements WLangage associés à la fenêtre est différent sous WINDEV et sous WINDEV Mobile.

Lors de l'importation d'une fenêtre WINDEV, si votre code est basé sur l'ordre d'exécution des événements WLangage associés à la fenêtre, vérifiez que le comportement de l'application (en mode test, simulateur ou en exécution réelle) correspond bien au comportement attendu.
<a name="NOTE3_4"></a>


## Boutons spécifiques iOS
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000172"></a>
WINDEV Mobile propose plusieurs boutons standard Apple : 

- [Bouton Apple Sign in](#NOTE2_1).

- [Bouton Localisation](#NOTE3_1).




Ces boutons peuvent être créés directement à partir du champ Bouton ou via les champs Métier disponibles avec WINDEV Mobile. 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Bouton Apple Sign in
<a name="bouton_apple_sign_ELTTEXTE000196"></a>


### Création via un champ Métier
<a name="creation_via_champ_metier_ELTPARAGRAPHE000025"></a>Pour créer un champ Bouton "Apple Sign in" via un champ Métier : 

1. Sous le volet "Création", dans le groupe "Métier", déroulez "Champs Métier". Dans la fenêtre popup qui s'affiche, Saisissez "Connexion Apple". 

2. Droppez le champ "Connexion Apple" vers votre fenêtre.

3. Le champ est automatiquement créé. ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Boutons_specifiques_ios%20-%20HC%20N%B0001.gif)
Il est fonctionnel. Son code peut être modifié si nécessaire. Ce code utilise la fonction [AppleSignIn](../WDLang3/1000025404.md). 




Remarque : Il est possible de modifier le style du champ (voir ci-dessous). 


### Création via un champ Bouton
<a name="creation_via_champ_bouton_ELTPARAGRAPHE000053"></a>Pour créer un champ Bouton proposant une authentification Apple avec le look correspondant : 

1. Créez un champ Bouton dans votre fenêtre : sous le volet "Création", dans le groupe "Champs usuels", cliquez sur "Bouton". 

2. Affichez si nécessaire le volet "Modifieur" : sous le volet "Accueil", dans le groupe "Environnement", déroulez "Volets" et sélectionnez "Volets" puis "Modifieur". 

3. Sélectionnez le champ Bouton : dans le volet "Modifieur", les caractéristiques du champ sont affichées. 

4. Dans la zone "Détails" du volet "Modifieur" : 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Boutons_specifiques_ios%20-%20HC%20N%B0002.gif)


	- Pour l'option "Bouton Apple", sélectionnez "Enregistrement (Sign In)".

- Pour l'option "Apple Sign in - Style", sélectionnez le style voulu pour votre bouton ("Selon thème", "Noir", "Blanc", "Blanc avec bordure").  




Il ne reste plus qu'à écrire le code WLangage du champ pour réaliser l'authentification "Apple Sign In". 
Ce code doit utiliser la fonction [AppleSignIn](../WDLang3/1000025404.md).

## Bouton Localisation
<a name="bouton_localisation_ELTTEXTE000263"></a>


### Création via un champ Métier
<a name="creation_via_champ_metier_ELTPARAGRAPHE000067"></a>Pour créer un champ Bouton "Localisation" via un champ Métier : 

1. Sous le volet "Création", dans le groupe "Métier", déroulez "Champs Métier". Dans la fenêtre popup qui s'affiche, Saisissez "Localisation Apple". 

2. Droppez le champ "Localisation Apple" vers votre fenêtre.

3. Le champ est automatiquement créé. ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Boutons_specifiques_ios%20-%20HC%20N%B0002%201.gif)
Il est fonctionnel. Son code peut être modifié si nécessaire. Ce code utilise la fonction [GPSRécupèrePosition](../WDLang3/1000019205.md). 




Remarque : Il est possible de modifier l'icône ou le libellé du champ (voir ci-dessous). 


### Création via un champ Bouton
<a name="creation_via_champ_bouton_ELTPARAGRAPHE000095"></a>Pour créer un champ Bouton proposant une géolocalisation Apple avec le look correspondant :

1. Créez un champ Bouton dans votre fenêtre : sous le volet "Création", dans le groupe "Champs usuels", cliquez sur "Bouton".

2. Affichez si nécessaire le volet "Modifieur" : sous le volet "Accueil", dans le groupe "Environnement", déroulez "Volets" et sélectionnez "Volets" puis "Modifieur".

3. Sélectionnez le champ Bouton : dans le volet "Modifieur", les caractéristiques du champ sont affichées.

4. Dans la zone "Détails" du volet "Modifieur" : 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Boutons_specifiques_ios%20-%20HC%20N%B0003.gif)


	- Pour l'option "Bouton Apple", sélectionnez "Localisation (Location)".

- Pour l'option "Bouton Apple Localisation - Icône", sélectionnez le type d'icône voulue devant le texte ("Aucun", "Icône flèche pleine" ou "Icône flèche évidée").

- Pour l'option "Bouton Apple Localisation - Texte", sélectionnez le texte voulu pour votre bouton ("Position actuelle", "Envoyer la position actuelle", "Envoyer ma position actuelle", "Partager la position actuelle" ou "Partager ma position actuelle").


Il ne reste plus qu'à écrire le code WLangage du champ pour gérer la géolocalisation. Ce code doit utiliser la fonction [GPSRécupèrePosition](../WDLang3/1000019205.md).


## Audits
			



## Présentation
<a name="presentation_ELTTEXTE000118"></a>
Les audits apportent une série de fonctionnalités permettant d'améliorer automatiquement la qualité et les performances d'un projet et de suivre plus précisément ses conditions de mise en exploitation.

Plusieurs fonctionnalités d'audit sont disponibles :

- L'audit statique.

- l'audit d'UI.

- L'audit dynamique.

- La télémétrie. 




Ces différentes formes d'audit sont complémentaires et doivent être utilisées ensemble pour profiter de toutes leurs capacités.





## Audit statique
<a name="audit_statique_ELTTEXTE000142"></a>
L'audit statique consiste en une analyse détaillée d'un projet et de ses éléments.

Cet audit est mené directement depuis l'éditeur de projet.

Son fonctionnement est décrit dans la page d'aide : [Audit statique](../Editeurs/1014501.md).



## Audit d'UI
<a name="audit_ELTTEXTE000166"></a>
L'audit d'UI permet de détecter les technologies obsolètes utilisées dans vos interfaces. 

Cet audit est automatiquement activé lors de l'ouverture de vos projets (à partir de la version 27). 

Cet audit est mené directement dans vos fenêtres et pages. 

Son fonctionnement est décrit dans la page d'aide : [Audit d'UI](../Editeurs/9000207.md).

Remarque : Les problèmes détectés par l'audit d'UI sont également affichés dans l'audit statique. 



## Audit dynamique
<a name="audit_dynamique_ELTTEXTE000190"></a>
L'audit dynamique est une analyse du comportement d'un projet pendant son exécution.

Cet audit peut être réalisé en mode test ou directement dans l'environnement de production.

Son fonctionnement est décrit dans la page d'aide : [Audit dynamique](../Editeurs/1014502.md).





## Télémétrie
<a name="telemetrie_ELTTEXTE000214"></a>
La télémétrie collecte et agrège des données concernant l'utilisation de vos applications. 

Grâce à la télémétrie, vous connaissez tout de vos applications déployées et vous pouvez facilement les améliorer. 

Son fonctionnement est décrit dans la page d'aide : [Présentation de la télémétrie](../WDLang1/1410086580.md).

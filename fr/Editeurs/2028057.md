
## Erreurs en cas d'installation par FTP
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000452"></a>
La liste ci-dessous présente les différents messages d'erreur apparaissant dans l'assistant d'installation à distance (chez le responsable de sites). Certains de ces messages nécessitent des actions spécifiques de configuration au niveau du serveur Web de déploiement.

Pour chaque erreur, diverses solutions sont proposées.

Des messages d'erreur peuvent apparaître à diverses étapes de l'installation à distance.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Erreurs lors de la saisie des logins et mots de passe Serveur et FTP (installation du site)
<a name="erreurs_lors_saisie_des_logins_mots_passe_serveur_ftp_installation_site_ELTTEXTE000476"></a>
L'assistant d'installation effectue une vérification de la validité des logins et mots de passe. 

Voici la liste des erreurs pouvant apparaître.


### La vérification du mot de passe utilisateur a échoué : le serveur n'a pas retourné d'information. 
<a name="verification_mot_passe_utilisateur_echoue_serveur_pas_retourne_information_ELTPARAGRAPHE000028"></a>

Les problèmes possibles sont les suivants :

| Problème possible | Solution |
| --- | --- |
| Le serveur Web ne fonctionne pas | Relancer le serveur Web sur le poste serveur. |
| Le serveur FTP ne fonctionne pas. | Relancer le serveur FTP sur le poste serveur. |
| Le serveur FTP est mal configuré : nom d'utilisateur incorrect, ... | Vérifier la configuration du serveur FTP pour l'utilisateur. <br><br>Attention : Si vous utilisez le serveur FTP IIS, vérifiez que :<br><br>- l'utilisateur est défini au niveau de Windows<br><br>- l'alias du répertoire de transfert FTP correspond au nom d'utilisateur Windows.<br><br><br> |
| Le serveur d'application WEBDEV n'est pas installé correctement. | Réinstaller le serveur d'application WEBDEV. |




**Le mot de passe FTP n'est pas correct ou le serveur FTP n'est pas joignable**

Les problèmes possibles sont les suivants :

| Problème possible | Solution |
| --- | --- |
| Le mot de passe FTP indiqué par le responsable de sites n'est pas correct. | L'hébergeur doit vérifier le mot de passe FTP et vous communiquer ce mot de passe. |
| Le serveur FTP ne fonctionne pas. | Relancer le serveur FTP. |
| Le nombre de connexions autorisées sur le serveur FTP est dépassé. | Attendre qu'une connexion au serveur FTP soit libérée. |
| L'adresse du serveur Web est incorrecte. | L'hébergeur doit vous indiquer l'adresse exacte du serveur Web où l'installation du site doit être effectuée par FTP. |


**La vérification du mot de passe utilisateur a échoué : l'utilisateur est inconnu du serveur.**
**Vérifiez l'adresse du serveur et le nom d'utilisateur**
Les problèmes possibles sont les suivants :

| Problème possible | Solution |
| --- | --- |
| Le gestionnaire de compte WEBDEV ne reconnaît pas le responsable de sites. | L'hébergeur doit vérifier qu'un compte WEBDEV a été créé dans le gestionnaire de comptes et vous communiquer le "login" correspondant. |




**La vérification du mot de passe utilisateur a échoué : le serveur n'a pas retourné d'information.**

Les problèmes possibles sont les suivants :

| Problème possible | Solution |
| --- | --- |
| Le mot de passe saisit lors de l'installation à distance ne correspond pas au mot de passe défini dans le gestionnaire de comptes. | L'hébergeur doit vérifier le mot de passe associé à votre compte dans le gestionnaire de comptes WEBDEV et vous communiquer ce mot de passe. |

L'assistant d'installation effectue l'installation ou la mise à jour du site.

### Impossible de créer un répertoire temporaire sur le serveur. Contactez l'administrateur du serveur.
<a name="impossible_creer_repertoire_temporaire_sur_serveur_contactez_administrateur_serveur_ELTPARAGRAPHE000126"></a>

Les problèmes possibles sont les suivants :

| Problème possible | Solution |
| --- | --- |
| Le compte Windows qui exécute le gestionnaire de protocole WD28AWP n'a pas les droits nécessaires pour créer un répertoire temporaire. | L'hébergeur doit accorder les droits nécessaires au compte Windows. |






### Le nom de client annoncé est inconnu. Vérifiez votre nom de client et le mot de passe associé.
<a name="nom_client_annonce_est_inconnu_verifiez_votre_nom_client_mot_passe_associe_ELTPARAGRAPHE000143"></a>

Les problèmes possibles sont les suivants :

| Problème possible | Solution |
| --- | --- |
| Le gestionnaire de compte WEBDEV ne reconnaît pas le responsable de site | L'hébergeur doit vérifier que votre compte WEBDEV a été créé dans le gestionnaire de comptes et vous communiquer le "login" correspondant au responsable de site |






**Les fichiers nécessaires à l'installation n'ont pas été transmis correctement.**
**Vérifiez que le nom d'utilisateur et le mot de passe du FTP sont corrects et correspondent bien au compte propriétaire du site.**

Les problèmes possibles sont les suivants :

| Problème possible | Solution |
| --- | --- |
| Une erreur a eu lieu pendant le transfert FTP et le fichier installé sur le serveur n'est pas lisible. | Vous devez recommencer l'installation. <br>Attention : S'il s'agit d'une première installation, vous devez supprimer la description du serveur dans la liste "Mise à jour à distance". |






### Le fichier d'installation est invalide. La transmission du fichier ne s'est pas passée correctement ou le fichier transmis a été endommagé. 
<a name="fichier_installation_est_invalide_transmission_fichier_sest_pas_passee_correctement_fichier_transmis_ete_endommage_ELTPARAGRAPHE000178"></a>

Les problèmes possibles sont les suivants :

| Problème possible | Solution |
| --- | --- |
| Une erreur a eu lieu pendant le transfert FTP et le fichier installé sur le serveur n'est pas lisible. | Vous devez recommencer l'installation. <br>Attention : S'il s'agit d'une première installation, vous devez supprimer la description du serveur dans la liste "Mise à jour à distance". |




**Erreur pendant le décryptage des informations d'installation. Vérifiez le mot de passe utilisé.**

Lors de la programmation d'une installation différée, un des problèmes suivants est survenu :

| Problème possible | Solution |

**Impossible d'ajouter la programmation à cause de l'erreur suivante : XXX** : 

Les problèmes possibles sont les suivants :

| Problème possible | Solution |
| --- | --- |
| Pas assez de mémoire disponible | L'hébergeur doit vérifier votre mot de passe associé au responsable de sites dans le gestionnaire de comptes, et vous communiquer ce mot de passe |

### Un site de même nom est déjà installé sur ce serveur. 
<a name="site_meme_nom_est_deja_installe_sur_serveur_ELTPARAGRAPHE000235"></a>**Un même serveur ne peut pas abriter deux sites portant le même nom.**

**Vous devez renommer votre site ou désinstaller l'existant. S'il s'agit du même site, vous devez faire une mise à jour au lieu d'une installation.**

Les problèmes possibles sont les suivants :

| Problème possible | Solution |
| --- | --- |
| Une erreur a eu lieu pendant le transfert FTP et le fichier installé sur le serveur n'est pas lisible. | Vous devez recommencer l'installation. <br>Attention : S'il s'agit d'une première installation, vous devez supprimer la description du serveur dans la liste "Mise à jour à distance". |

Remarque : Pour effectuer une mise à jour par FTP, vous devez utiliser l'option "Mise à jour à distance" lors de l'installation du site par FTP (et non l'option "Installation à distance").
**Impossible de créer le répertoire du site. Contactez l'administrateur du serveur.**

Les problèmes possibles sont les suivants :

| Problème possible | Solution |
| --- | --- |
| Le répertoire de base des sites n'existe pas | L'hébergeur doit vérifier le répertoire de base des données indiqué dans le gestionnaire de comptes et vérifier l'existence de ce répertoire. |
| Les droits d'accès au répertoire de base des données sont insuffisants | L'hébergeur doit accorder les droits nécessaires au compte Windows qui exécute le gestionnaire de protocole WD28AWP.EXE. |






### Erreur d'écriture dans la base de registre. Contactez l'administrateur du serveur.
<a name="erreur_ecriture_dans_base_registre_contactez_administrateur_serveur_ELTPARAGRAPHE000278"></a>

Les problèmes possibles sont les suivants :

| Problème possible | Solution |
| --- | --- |
| Le compte Windows qui exécute le gestionnaire de protocole WD28AWP n'a pas les droits nécessaires pour écrire dans la base de registre. | L'hébergeur doit accorder les droits nécessaires au compte Windows. Utilisez le programme Regedt32.exe pour accorder les droits d'écriture dans la base de registre (option "Sécurité .. Permissions" dans Regedt32). |

## Mise à jour d'un site
<a name="mise_jour_site_ELTTEXTE000612"></a>
Lors de la mise à jour du site, l'assistant d'installation vérifie la bonne installation du site et sa configuration.
 
Voici la liste des erreurs pouvant apparaître. 



### Nom de site inconnu :
<a name="nom_site_inconnu_ELTPARAGRAPHE000338"></a>Les problèmes possibles sont les suivants :

| Problème possible | Solution |
| --- | --- |
| Le site n'est pas installé sur le serveur | Faire une installation distante complète. |
| Le site n'est plus référencé dans l'outil d'administration local WEBDEV | L'hébergeur doit référencer le site WEBDEV dans l'outil d'administration local WEBDEV (onglet "Sites", bouton "Ajouter"). |






### Le nom du client demandeur et le propriétaire du site ne correspondent pas. 
<a name="nom_client_demandeur_proprietaire_site_correspondent_pas_ELTPARAGRAPHE000357"></a>

Les problèmes possibles sont les suivants :



| Problème possible | Solution |
| --- | --- |
| Le site a été installé par un autre responsable de sites. | Dans le gestionnaire de comptes WEBDEV, l'hébergeur doit attribuer le site au compte WEBDEV correspondant, vérifier que les répertoires de base (de données, du site et de transfert FTP) sont corrects. |






### Echec lors du cryptage.
<a name="echec_lors_cryptage_ELTPARAGRAPHE000373"></a>

Les problèmes possibles sont les suivants :

| Problème possible | Solution |
| --- | --- |
| Le cryptage de la mise à jour a échoué. | L'hébergeur doit libérer de l'espace mémoire et / ou disque sur le poste effectuant la mise à jour (poste du responsable de sites par exemple). |






### Erreur lors de la copie du fichier d'information. Vérifiez les informations du compte WEBDEV et réessayez.
<a name="erreur_lors_copie_fichier_information_verifiez_les_informations_compte_webdev_reessayez_ELTPARAGRAPHE000388"></a>

Les problèmes possibles sont les suivants :

| Problème possible | Solution |
| --- | --- |
| Impossibilité de copier le fichier temporaire décrivant l'état du site. L'utilisateur n'a pas les droits nécessaires dans le répertoire FTP | L'hébergeur doit accorder les droits d'écriture dans le répertoire FTP. |
| Impossibilité de copier le fichier temporaire décrivant l'état du site. Il n'y a pas assez de place disponible sur le disque | L'hébergeur doit libérer de l'espace disque sur le serveur. |

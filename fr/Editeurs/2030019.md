


## Exportation d'éléments vers un répertoire 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000109"></a>
Tous les éléments manipulés par un projet (fenêtre, état, classes, collection de procédures, requêtes, ...) peuvent être transférés d'un projet à un autre. L'exportation permet de copier les éléments du projet en cours vers un autre répertoire (et donc un autre projet).

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000133"></a>


### Exporter des éléments du projet en cours
<a name="exporter_des_elements_projet_cours_ELTPARAGRAPHE000018"></a>Pour exporter des éléments du projet en cours : 

1. Sous le volet "Accueil", dans le groupe "Général", déroulez "Enregistrer" et sélectionnez "Exporter .. Vers un répertoire".

2. Sélectionnez le répertoire dans lequel doivent être exportés les fichiers. Une coche apparaît devant les fichiers sélectionnés.

3. Dans la liste des éléments WINDEV affichée, sélectionnez les éléments à exporter. 
	Remarque : Cette liste est composée du nom du fichier, de la description du fichier (si elle existe), de l'emplacement actuel du fichier.

4. Validez.
	Les fichiers sélectionnés sont automatiquement exportés dans le répertoire du projet spécifié.




**Remarque** : Pour intégrer les éléments exportés aux composants d'un projet, utilisez la [liste des composants du projet](../Editeurs/2030033.md).

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Notes
<a name="notes_ELTTEXTE000157"></a>


### Dépendance des fichiers
<a name="dependance_des_fichiers_ELTPARAGRAPHE000053"></a>Les éléments sont exportés avec leurs dépendances : tous les fichiers liés à l'élément exporté sont également exportés. Par exemple, la fenêtre et son image de fond sont automatiquement exportés.
<a name="NOTE3_2"></a>


### Estimation de la taille des fichiers exportés
<a name="estimation_taille_des_fichiers_exportes_ELTPARAGRAPHE000060"></a>Il est possible d'estimer la taille des fichiers exportés avec le bouton "Calculer". Cette taille prend en compte la taille de l'élément à exporter et la taille des fichiers dépendant de cet élément.

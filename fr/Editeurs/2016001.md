


## Edition de la feuille de styles CSS
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000141"></a>
WEBDEV offre la possibilité de gérer l'ensemble des styles CSS utilisés dans un projet. 

Il est possible :

- d'éditer la feuille de styles CSS d'un projet.

- de définir les styles CSS par défaut. 




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Editer la feuille de styles CSS d'un projet
<a name="editer_feuille_styles_css_projet_ELTTEXTE000165"></a>
Pour éditer la feuille de styles CSS d'un projet :

1. Sous le volet "Page", dans le groupe "Ambiance et styles", déroulez "Styles" et sélectionnez "Editer les styles CSS".

2. La liste des styles CSS présents dans le projet s'affiche. 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Edition-StyleCSS.gif&type=thumb)





Cette fenêtre permet de :

- Modifier le style sélectionné. Si le style n'est pas un style local, cette modification sera reportée sur tous les champs utilisant ce style.

- Créer un nouveau style.

- Dupliquer le style sélectionné.

- Supprimer le style sélectionné : dans ce cas, le style associé au champ sera le même mais sera local au champ.




**Remarque** : La case à cocher "Local" permet d'obtenir la liste des styles associés à la feuille de styles associée à la page (dans l'onglet "Style" de la description de la page).

## Définir les styles CSS par défaut
<a name="definir_les_styles_css_par_defaut_ELTTEXTE000189"></a>
Les styles CSS par défaut sont les styles CSS qui sont utilisés par défaut lors de la création d'un nouveau champ.

Pour un type de champ donné, s'il n'existe pas de style WEBDEV dans le projet, il est possible de définir un ou plusieurs styles CSS par défaut (cela dépend du type de champ).

**Pour définir les styles CSS par défaut d'un projet** :

1. Sous le volet "Page", dans le groupe "Ambiance et styles", déroulez "Styles" et sélectionnez "Définir les styles CSS par défaut". La liste des styles CSS présents dans le projet s'affiche ainsi que la liste des champs.
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Styles_CSS_Defaut.gif&type=thumb)


2. Sélectionnez un style CSS. Selon le style sélectionné :

	- Dans la liste des champs, les types de champs sur fond rouge sont ceux pour lesquels un autre style CSS a été défini par défaut.

	- Dans la liste des champs, les types de champs sur fond bleu sont ceux pour lesquels le style CSS sélectionné a été défini par défaut.

	- Dans la liste des champs, les lignes associées au style en cours apparaissent en vert.




3. Sélectionnez les champs auxquels ce style doit être associé par défaut (la liste des champs est multisélection).
	**Attention** :

	- Sélectionnez les autres types utilisant ce style. Sinon, les autres associations de ce style seront perdues.

	- Le bouton "Valider" valide la sélection.




4. Cliquez sur le bouton [Valider].




**Remarque** : Il est également possible de modifier, créer, supprimer, dupliquer un style présent dans la liste.

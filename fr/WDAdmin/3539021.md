## Présentation

### L'administrateur WEBDEV
<a name="administrateur_webdev_ELTPARAGRAPHE000011"></a>L'administrateur WEBDEV est un module permettant de gérer les sites WEBDEV dynamiques (en mode Session ou AWP).

Lors du développement d'un site WEBDEV dynamique, l'administrateur permet principalement de :

- tester le site grâce à la page de test (onglet "Avancé", option "Page de test").

- régler les paramètres de connexion au site pour vos tests : temps de déconnexion, ... (onglet "Configuration").

- supprimer directement les connexions de test en cours (onglet "Connexions").

- changer le serveur Web utilisé pour vos tests (onglet "Avancé", option "Serveur").

- effectuer un diagnostic en cas de problème lors du lancement d'un site WEBDEV (onglet "Configuration", option "Diagnostic").

- ...




En déploiement, deux types d'administrateur WEBDEV existent :

- **l'administrateur WEBDEV Déploiement**, livré avec le serveur d'application WEBDEV, qui permet à l'hébergeur de gérer les sites WEBDEV dynamiques installés sur un serveur Web.

- **l'administrateur WEBDEV Distant** qui permet au gestionnaire de site de gérer ses sites WEBDEV dynamiques à distance depuis n'importe quel poste.




Cette aide présente uniquement l'administrateur de WEBDEV version Développement.

Pour plus de détails sur les autres types d'administrateur WEBDEV, consultez leur aide respective.

## Lancement de l'administrateur de WEBDEV version Développement
<a name="lancement_administrateur_webdev_version_developpement_ELTTEXTE000187"></a>
Pour lancer l'administrateur de WEBDEV version Développement, utilisez une des méthodes suivantes :

- Sous le volet "Outils", dans le groupe "Utilitaires Web", cliquez sur "WDAdmin" depuis WEBDEV Développement.

- Sélectionnez "Programmes .. WEBDEV 28 .. Administrateur WEBDEV" depuis le menu "Démarrer".




L'administrateur WEBDEV est lancé automatiquement au premier test du site. L'administrateur peut ensuite être laissé en tâche de fond.

Par défaut, l'administrateur WEBDEV autorise dix connexions simultanées à un même site WEBDEV dynamique (icône GO).

**Remarque** : Lorsque l'administrateur WEBDEV est lancé, l'icône correspondante est présente dans la barre de tâches de Windows. Le menu contextuel de l'icône dans le "systray" permet : 

- d'afficher l'administrateur. 

- de quitter l'administrateur. 

- de déconnecter tous les utilisateurs.

## Interface de l'administrateur de WEBDEV version Développement
<a name="interface_administrateur_webdev_version_developpement_ELTTEXTE000211"></a>
L'administrateur de WEBDEV version Développement se présente sous la forme d'une fenêtre composée de plusieurs onglets :

- onglet ["Connexions"](../WDAdmin/3539041.md) pour paramétrer les caractéristiques des connexions en cours.

- onglet ["Sites"](../WDAdmin/3539008.md) listant les sites WEBDEV installés sur le poste en cours.

- onglet ["Webservices"](../WDAdmin/3539075.md) listant les webservices WEBDEV installés sur le poste en cours. Cet onglet est identique à l'onglet ["Sites"](../WDAdmin/3539008.md) mais pour les webservices. 

- onglet ["WebSockets"](../WDAdmin/3539082.md) listant les serveurs de Websockets installés sur le poste en cours. 

- onglet ["Configuration"](../WDAdmin/3539046.md) pour paramétrer la configuration générale des sites WEBDEV installés sur le poste en cours.

- onglet ["Installations/Comptes"](../WDAdmin/3539065.md) pour réaliser l'administration du serveur WEBDEV sur le poste en cours.

- onglet ["Avancé"](../WDAdmin/3539066.md) pour configurer des fonctionnalités spécifiques.

- onglet ["Journaux"](../WDAdmin/3539073.md) pour analyser le compte-rendu des journaux.

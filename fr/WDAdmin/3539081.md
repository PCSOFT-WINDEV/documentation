


## Serveur Web dédié
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000064"></a>
Lors de l'installation de WEBDEV version développement, si aucun serveur Web n'est disponible sur le poste en cours, WEBDEV installe automatiquement un serveur Web (IIS Express version 10). Ce serveur est automatiquement configuré pour fonctionner avec WEBDEV. 

Il est également possible d'installer ce serveur Web manuellement, puis de configurer son utilisation via l'administrateur WEBDEV.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Conseils
<a name="conseils_ELTTEXTE000092"></a>


### Installation automatique
<a name="installation_automatique_ELTPARAGRAPHE000110"></a>Lors de l'installation de WEBDEV 28, si aucun serveur Web n'est détecté sur le poste en cours, le serveur Web IIS Express version 10 est automatiquement installé et configuré pour WEBDEV sur le poste en cours. 

Ce serveur Web permet de tester rapidement les sites dynamiques (en mode Session ou AWP) créés avec WEBDEV. 


### Installation manuelle
<a name="installation_manuelle_ELTPARAGRAPHE000119"></a>Il est possible d'installer manuellement le serveur Web IIS Express version 10 sur le poste de développement (Windows 7 SP1 minimum). Cette installation est disponible à l'adresse suivante : [https://www.microsoft.com/fr-fr/download/details.aspx?id=48264](https://www.microsoft.com/fr-fr/download/details.aspx?id=48264)

Il suffit de sélectionner le téléchargement souhaité dans la langue voulue : 

- Version 64 bits (\*_amd64_\*) pour un OS 64 bits (cas général). 

- Version 32 bits (\*_x86_\*) pour un OS 32 bits.  




Le serveur Web pourra être ensuite sélectionné : 

- dans les options de WEBDEV : 

	1. Sous le volet "Accueil", dans le groupe "Environnement", déroulez "Options" et sélectionnez "Options générales de WEBDEV". 

	2. Dans l'onglet "Web", dans la zone "Serveur Web", cliquez sur le bouton "Modifier". 

	3. dans la fenêtre qui s'affiche sélectionnez le serveur Web voulu et validez. 




- dans l'administrateur WEBDEV : 

	1. Lancez l'administrateur WEBDEV : Sous le volet "Outils", dans le groupe "Utilitaires Web", cliquez sur "WDAdmin". 

	2. Dans l'onglet "Avancé", cliquez sur le bouton "Serveur". 

	3. Sélectionnez "Serveur Web dédié (IIS Express)" et validez. 








### Limitations
<a name="limitations_ELTPARAGRAPHE000168"></a>

- Le port utilisé pour le serveur Web dédié est imposé : 80xx, où xx est la version majeure de WEBDEV. Par exemple avec WEBDEV 28, le port utilisé est 8028.

- IIS Express ne dispose pas d'interface de configuration. Il faut éditer le fichier XML. En pratique, il n'est donc pas possible de modifier directement la configuration de IIS Express.

- Si le site a besoin d'une configuration non triviale (certificat SSL, etc.); il est nécessaire d'utiliser un serveur Web normal (IIS par exemple).





### Remarque
<a name="remarque_ELTPARAGRAPHE000179"></a>WEBDEV utilise une configuration séparée par version majeure, présente dans le sous-répertoire "&lt;Personnel&gt;\\IntegratedWebServer". En cas de problèmes, il est possible de supprimer ce répertoire : il sera récréé (ainsi que son contenu).

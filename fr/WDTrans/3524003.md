
## WDTrans : Annuler les opérations effectuées sur un fichier de transaction
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000118"></a>


### Si une transaction est en cours
<a name="une_transaction_est_cours_ELTPARAGRAPHE000011"></a>

**Si aucune transaction est en cours**, rétablit la cohérence de la base de données et annule la transaction qui a échoué (cas d'une coupure de courant par exemple).



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000142"></a>


### Annuler les opérations effectuées sur un fichier de transaction
<a name="annuler_les_operations_effectuees_sur_fichier_transaction_ELTPARAGRAPHE000021"></a>Pour annuler les opérations effectuées sur un fichier de transaction :

1. Lancez WDTrans.

2. Sélectionnez l'option "Annuler une transaction". 

3. Sélectionnez le fichier de transaction à l'aide du sélecteur de fichiers. 
	La liste des fichiers de données (avec leur chemin complet) en cours de transaction apparaît. Pour chaque fichier, l'identifiant du poste réalisant une opération en transaction est affiché.

4. Si certains fichiers de données sont protégés par un mot de passe, spécifiez ce mot de passe. En effet, ce mot de passe est nécessaire pour annuler la transaction.

5. Si nécessaire, débranchez la gestion des doublons et/ou de l'intégrité.  

6. Cliquez sur le bouton "Annuler la transaction". La transaction est annulée.





<a name="NOTE2_2"></a>


### Pour annuler les opérations effectuées sur un fichier de transaction en mode ligne de commande :
<a name="pour_annuler_les_operations_effectuees_sur_fichier_transaction_mode_ligne_commande_ELTPARAGRAPHE000038"></a>
```txt
WDTrans	/Fic = <Fichier>
		/Mdp = <Mot de passe>
		/Option = <Type d'action réalisée>
		/Log = <Fichier Log>
```


Le tableau ci-dessous liste les différents éléments pouvant être présents sur la ligne de commande :

| Paramètre | Signification |
| --- | --- |
| /Fic = &lt;Fichier&gt; | Chemin complet du fichier de transaction (fichier .TRS) |
| /Mdp = &lt;Mot de passe&gt; | Mot de passe associé au fichier à traiter (traitement d'un seul fichier) |
| /Option = &lt;Type d'action à réaliser&gt; | Option de WDTrans à lancer. Pour annuler les opérations effectuées sur un fichier de transaction, cette option correspond à 6. |
| /Log = &lt;Fichier Log&gt; | Chemin complet du fichier journal (.LOG) à créer. |



**Exemple** : Annuler les transactions enregistrées dans le fichier "WD Transactions.TRS" : 

```txt
WDTRANS.EXE /Fic="C:\Temp\WD Transaction.TRS" /option=6
```

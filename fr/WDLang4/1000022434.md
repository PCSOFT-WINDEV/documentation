


## MongoCréeCollection (Fonction)

***En anglais : MongoCreateCollection***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Crée une collection dans une base MongoDB en précisant des options de cette collection.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple
<a class="notetitle" target="_blank" href="$DOC$=1000022625&name=mongocreecollection_fonction&product=WD">Voir des exemples supplémentaires</a>

```wl
BaseMDB est un mongoBase
Option est un mongoCollectionOption
Option.TailleMax = 100*1024*1024
MongoCréeCollection(BaseMDB, "produit", Option)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = MongoCréeCollection(<Base> , <Collection> , <Options>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'opération a été réalisée,

- <u><u><u><u>Faux</u></u></u></u> en cas de problème. La fonction [ErreurInfo](../WDLang1/3013008.md) permet d'identifier l'erreur.




**`<Base> : (Variable de type mongoBase)`**

Nom de la variable de type [mongoBase](../WDLang4/1000022410.md) correspondant à la base MongoDB dans laquelle la collection doit être créée.
Si cette base n'existe pas, elle est automatiquement créée. 

**`<Collection> : (Chaîne de caractères)`**

Nom de la collection à créer.

**`<Options> : (Variable de type mongoCollectionOption)`**

Nom de la variable de type [mongoCollectionOption](../WDLang4/1000022413.md) correspondant aux options de création de la collection.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Lors de l'écriture dans une collection inexistante, cette collection est automatiquement créée avec les options par défaut.

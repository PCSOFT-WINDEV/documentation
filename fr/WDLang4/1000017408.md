


## Fonctions de liaison
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions de liaison sont les fonctions permettant de lier l'interface aux fichiers de données (et inversement). Ces fonctions sont les suivantes : 



|   |   |
| --- | --- |
| [ChampVersSource](../WDLang4/1000021384.md) | Affecte la source de la liaison d'un champ avec la valeur du champ. |
| [EcranVersFichier](../WDLang4/3044146.md) | Initialise automatiquement :<br><br>- la valeur mémoire des rubriques d'un fichier de données avec la valeur des champs de la fenêtre. <br><br>- la valeur des variables WLangage avec la valeur des champs de la fenêtre. <br><br><br>Cette fonction est également disponible pour les pages WEBDEV. |
| [EcranVersSource](../WDLang4/1000017342.md) | Initialise automatiquement : <br><br>- la valeur mémoire des rubriques d'un fichier de données avec la valeur des champs de la fenêtre.<br><br>- la valeur des variables WLangage avec la valeur des champs de la fenêtre.<br><br><br><br><br>Cette fonction est également disponible pour les pages WEBDEV. |
| [FenêtreDepuisSource](../WDLang4/1000021396.md) | Initialise automatiquement les champs d'une fenêtre avec : <br><br>- les valeurs des rubriques associées dans l'enregistrement en cours (chargé en mémoire) du fichier de données HFSQL décrit sous l'éditeur d'analyses.<br><br>- les valeurs des variables WLangage associées. <br><br><br> |
| [FichierVersEcran](../WDLang4/3044210.md) | Initialise automatiquement les champs d'une fenêtre ou d'une page avec :<br><br>- les valeurs des rubriques associées dans l'enregistrement en cours (chargé en mémoire) du fichier de données décrit sous l'éditeur d'analyses. <br><br>- les valeurs des variables WLangage associées. <br><br><br>Cette opération est effectuée quel que soit l'état des champs (grisés, inactifs, ou invisibles). |
| [FichierVersPage](../WDLang4/3044148.md) | Initialise automatiquement les champs d'une page avec :<br><br>- les valeurs des rubriques associées dans l'enregistrement en cours (chargé en mémoire) du fichier de données décrit sous l'éditeur d'analyses. Cette opération est effectuée quel que soit l'état des champs (grisés, inactifs, ou invisibles).<br><br>- les valeurs des variables WLangage associées. Cette opération est effectuée quel que soit l'état des champs (grisés, inactifs, ou invisibles).<br><br><br> |
| [PageDepuisSource](../WDLang4/1000021400.md) | Initialise automatiquement les champs d'une page avec : <br><br>- les valeurs des rubriques associées dans l'enregistrement en cours (chargé en mémoire) du fichier de données HFSQL décrit sous l'éditeur d'analyses. <br><br>- les valeurs des variables WLangage associées. <br><br><br>Cette opération est effectuée quel que soit l'état des champs (grisés, inactifs, ou invisibles). |
| [PageVersFichier](../WDLang4/3044022.md) | Initialise automatiquement :<br><br>- la valeur mémoire des rubriques d'un fichier de données avec la valeur des champs de la page. Les rubriques du fichier de données de base reliées aux champs de la fenêtre sont automatiquement affectées avec les valeurs des champs correspondants.<br><br>- la valeur des variables WLangage avec la valeur des champs de la fenêtre. Les variables WLangage reliées aux champs de la page sont automatiquement affectées avec les valeurs des champs correspondants.<br><br><br>Cette opération est effectuée quel que soit l'état des champs (grisés, inactifs, ou invisibles). |
| [PageVersSource](../WDLang4/1000017406.md) | Initialise automatiquement :<br><br>- la valeur mémoire des rubriques d'un fichier de données avec la valeur des champs de la page.<br><br>- la valeur des variables WLangage avec la valeur des champs de la page.<br><br><br> |
| [SourceDepuisFenêtre](../WDLang4/1000021402.md) | Initialise automatiquement : <br><br>- la valeur mémoire des rubriques d'un fichier de données avec la valeur des champs de la fenêtre.<br><br>- la valeur des variables WLangage avec la valeur des champs de la fenêtre.<br><br><br>Cette fonction est également disponible pour les pages WEBDEV. |
| [SourceDepuisPage](../WDLang4/1000021404.md) | Initialise automatiquement :<br><br>- la valeur mémoire des rubriques d'un fichier de données avec la valeur des champs de la page.<br><br>- la valeur des variables WLangage avec la valeur des champs de la page.<br><br><br> |
| [SourceVersEcran](../WDLang4/1000017341.md) | Initialise automatiquement les champs d'une fenêtre ou d'une page avec : <br><br>- les valeurs des rubriques associées dans l'enregistrement en cours (chargé en mémoire) du fichier de données HFSQL décrit sous l'éditeur d'analyses.<br><br>- les valeurs des variables WLangage associées. <br><br><br> |
| [SourceVersPage](../WDLang4/1000017405.md) | Initialise automatiquement les champs d'une page avec : <br><br>- les valeurs des rubriques associées dans l'enregistrement en cours (chargé en mémoire) du fichier de données HFSQL décrit sous l'éditeur d'analyses.<br><br>- les valeurs des variables WLangage associées.<br><br><br> |

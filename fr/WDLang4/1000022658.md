


## &lt;Variable Connexion&gt;.ExécutePlanification (Fonction)

***En anglais : &lt;Connection variable&gt;.ExecuteScheduling***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Exécute immédiatement une tâche planifiée sans tenir compte de sa planification : 

- procédure stockée, 

- sauvegarde, 

- optimisation, 

- rafraîchissement d'une vue matérialisée.





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Description de la connexion
MaConnexionBase est une Connexion

tabDescSvg est un tableau dynamique de hDescriptionSauvegarde 
tabDescSvg = MaConnexionBase.ListePlanification(hlpSauvegarde)
hDescSvg est une hDescriptionSauvegarde
POUR TOUT hDescSvg DE tabDescSvg
	MaConnexionBase.ExécutePlanification(hDescSvg)
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Exécuter immédiatement une tâche planifiée

`<Résultat> = <Connexion>.ExécutePlanification(<Tâche planifiée>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la tâche planifiée a été exécutée,

- <u><u><u><u>Faux</u></u></u></u> en cas d'erreur. La fonction [HErreurInfo](../WDLang4/3044071.md) permet d'identifier l'erreur.




**`<Connexion> : (Variable de type Connexion)`**

Nom de la variable de type [Connexion](../WDLang4/1514073.md) décrivant la connexion à manipuler.

**`<Tâche planifiée> : (Variable de type hTâchePlanifiée)`**

Nom de la variable de type [hTâchePlanifiée](../WDLang4/1000017448.md) décrivant la tâche déclenchant une procédure stockée.


<a name="SYNTAXE2"></a>

### Exécuter une tâche d'optimisation planifiée

`<Résultat> = <Connexion>.ExécutePlanification(<Tâche d'optimisation planifiée>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la tâche d'optimisation planifiée a été exécutée, 

- <u><u><u><u>Faux</u></u></u></u> en cas d'erreur. La fonction [HErreurInfo](../WDLang4/3044071.md) permet d'identifier l'erreur.




**`<Connexion> : (Variable de type Connexion)`**

Nom de la variable de type [Connexion](../WDLang4/1514073.md) décrivant la connexion à manipuler.

**`<Tâche d'optimisation planifiée> : (Variable de type hOptimisationPlanifiée)`**

Nom de la variable de type [hOptimisationPlanifiée](../WDLang4/1000018854.md) manipulée.


<a name="SYNTAXE3"></a>

### Exécuter une planification de sauvegarde

`<Résultat> = <Connexion>.ExécutePlanification(<Sauvegarde planifiée> [, <Type de sauvegarde>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la sauvegarde planifiée a été exécutée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Connexion> : (Variable de type Connexion)`**

Nom de la variable de type [Connexion](../WDLang4/1514073.md) décrivant la connexion à manipuler.

**`<Sauvegarde planifiée> : (Variable de type hDescriptionSauvegarde)`**

Nom de la variable de type [hDescriptionSauvegarde](../WDLang4/1000017455.md) manipulée.

**`<Type de sauvegarde> : (Booléen optionnel)`**

Type de sauvegarde à effectuer : 

- <u><u><u><u>Vrai</u></u></u></u> (valeur par défaut) : Sauvegarde différentielle. 

- <u><u><u><u>Faux</u></u></u></u> : Sauvegarde complète. 




**Remarque** : Une sauvegarde différentielle peut être effectuée uniquement si la tâche de sauvegarde comporte une planification différentielle.


<a name="SYNTAXE4"></a>

### Exécuter une planification de rafraîchissement d'une vue matérialisée

`<Résultat> = <Connexion>.ExécutePlanification(<Vue matérialisée>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la planification a été exécutée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Connexion> : (Variable de type Connexion)`**

Nom de la variable de type [Connexion](../WDLang4/1514073.md) décrivant la connexion à manipuler.

**`<Vue matérialisée> : (Variable de type hPlanificationVueMatérialisée)`**

Nom de la variable de type [hPlanificationVueMatérialisée](../WDLang4/1000021444.md) contenant la description des vues matérialisées à rafraîchir.


## HListePersoDossier (Fonction)

***En anglais : HListCustomFolder***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la liste des perso-dossiers (également appelés groupes) définis dans l'analyse. 




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
Trace(HListePersoDossier())
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = HListePersoDossier([<Chemin du WDD> [, <Mot de passe du WDD>]])`
---

**`<Résultat> : (Chaîne de caractères)`**

Liste des perso-dossiers définis dans l'analyse en cours ou dans l'analyse spécifiée. Les différents noms des perso-dossiers sont séparés par RC (Retour Chariot). Le résultat est de la forme : 

```txt
<Perso-dossier 1> + RC + <Perso-dossier 2> + ... + <Perso-dossier N>
```


**`<Chemin du WDD> : (Chaîne de caractères optionnelle)`**

Chemin complet de l'analyse (c'est-à-dire du ficher .WDD) à manipuler. Si ce paramètre n'est pas précisé ou est égal à une chaîne vide (""), l'analyse en cours est manipulée.

**`<Mot de passe du WDD> : (Chaîne de caractères optionnelle)`**

Mot de passe défini pour l'analyse. Si aucun mot de passe n'est défini, ce paramètre doit être égal à une chaîne vide ("").




## HFinInterditModif (Fonction)

***En anglais : HEndNoModif***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Débloque un fichier de données qui a été bloqué par le même programme avec la fonction [HInterditModif](../WDLang4/3044053.md).

**Attention** : Ne débloque pas les blocages effectués par d'autres programmes.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
HInterditModif(Facture)
// Traitement
HFinInterditModif(Facture)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = HFinInterditModif(<Fichier de données>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si aucun problème n'a été rencontré, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. La fonction [HErreur](../WDLang4/3044088.md) permet d'obtenir plus d'informations sur le problème rencontré.




**`<Fichier de données> : (Chaîne de caractères)`**

Nom du fichier de données manipulé.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Cette fonction n'est pas disponible lors de l'utilisation d'une base de données via .

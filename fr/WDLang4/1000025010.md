


## &lt;Source&gt;.SupprimeToutRayé (Fonction)

***En anglais : &lt;Source&gt;.Free***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Transforme les enregistrements rayés (fonction [&lt;Source&gt;.Raye](../WDLang4/1000025051.md)) d'un fichier de données en enregistrements supprimés. La place occupée par les enregistrements rayés est ainsi libérée et pourra être réutilisée lors des prochains ajouts (fonction [&lt;Source&gt;.Ajoute](../WDLang4/1000024187.md)).

**Rappel** : Un enregistrement rayé est inactif et disparaît de l'index. Les données de cet enregistrement rayé ne sont pas supprimées (contrairement à la fonction [&lt;Source&gt;.Supprime](../WDLang4/1000025071.md)).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Supprime tous les enregistrements rayés du fichier de données "Client"
Client.SupprimeToutRayé()
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Source>.SupprimeToutRayé()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'enregistrement a été supprimé, 

- <u><u><u><u>Faux</u></u></u></u> en cas de problème : l'enregistrement n'est pas supprimé. La fonction [HErreur](../WDLang4/3044088.md) permet d'identifier l'erreur.




**`<Source> : (Type correspondant à la source spécifiée)`**

Nom du fichier de données manipulé. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Cette fonction n'est pas adaptée à des fichiers xBase.

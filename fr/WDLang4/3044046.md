
## HListeConnexion (Fonction)

***En anglais : HListConnection***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la liste des connexions actuellement décrites dans l'application. Il est possible de lister les connexions décrites sous l'éditeur d'analyses ainsi que les connexions décrites par programmation (avec les fonctions [HDécritConnexion](../WDLang4/3044205.md) ou [HOuvreConnexion](../WDLang4/3044107.md)).






<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Récupérer la liste des connexions
ListeCnx est une chaîne = HListeConnexion(hLstDétail + hLstTout + hLstTriée)

// Insérer la liste dans la combo COMBO_Connexion
ListeAjoute(COMBO_Connexion, ListeCnx)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = HListeConnexion([<Options> [, <Séparateur de connexions> [, <Séparateur d'informations>]]])`
---

**`<Résultat> : (Chaîne de caractères)`**

Liste des connexions définies. Les différentes connexions sont séparées par le &lt;Séparateur de connexions&gt; (par défaut RC (Retour chariot)).


|   |   |
| --- | --- |
| *hLstDétail* non précisé | &lt;Nom de la connexion 1&gt; + &lt;Séparateur de connexions&gt; + &lt;Nom de la connexion 2&gt; + &lt;Séparateur de connexions&gt; + ... |
| *hLstDétail* précisé | &lt;Nom de la connexion 1&gt; + &lt;Séparateur d'informations&gt; + &lt;Utilisateur&gt; + &lt;Séparateur d'informations&gt; + &lt;Source de données&gt; + &lt;Séparateur d'informations&gt; + &lt;Nom du provider&gt; + &lt;Séparateur d'informations&gt; + &lt;Informations optionnelles&gt; + &lt;Séparateur de connexions&gt; + ... |



**`<Options> : (Constante ou combinaison de constantes de type entier (optionnel))`**

Paramètre le type de liste à effectuer :


|   |   |
| --- | --- |
| *hLstDétail* | Informations détaillées. |
| *hLstNormal*<br>(valeur par défaut) | Liste des connexions non triée, non détaillée, sans les connexions décrites par programmation. |
| *hLstOuvert* | Liste uniquement les connexions ouvertes. |
| *hLstTout* | Liste toutes les connexions (connexions de l'analyse et connexions définies par programmation avec les fonctions [HDécritConnexion](../WDLang4/3044205.md) ou [HOuvreConnexion](../WDLang4/3044107.md)). |
| *hLstTriée* | Trie la liste par ordre croissant. |



**`<Séparateur de connexions> : (Chaîne de caractères optionnelle)`**

Séparateur utilisé pour séparer les différentes connexions. Par défaut, ce séparateur correspond à un Retour Chariot (RC).

**`<Séparateur d'informations> : (Chaîne de caractères optionnelle)`**

Séparateur utilisé pour séparer les différentes informations renvoyées sur les connexions. Par défaut, ce séparateur correspond à une tabulation (TAB).



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Affichage du résultat dans un champ Liste ou un champ Combo
<a name="affichage_resultat_dans_champ_liste_champ_combo_ELTPARAGRAPHE000118"></a>&lt;Résultat&gt; peut être directement affecté à un champ Liste ou un champ Combo grâce à la fonction [ListeAjoute](../WDLang1/3049004.md). Si le &lt;Séparateur de connexion&gt; correspond à RC, chaque ligne contiendra le nom complet d'une des connexions trouvées.

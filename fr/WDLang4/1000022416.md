
## mongoGridFS (Type de variable)

***En anglais : mongoGridFS***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **mongoGridFS** permet de définir toutes les caractéristiques avancées d'un système de fichiers d'une base MongoDB. Les caractéristiques de ce système de fichiers peuvent être définies et modifiées à l'aide de différentes propriétés WLangage. 

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
cnxMDB est une mongoConnexion("mongodb://db1.exemple.net")
fs est un mongoGridFS
fs.BaseDeDonnées = cnxMDB.BaseDeDonnées["CRM"]
fs.Préfixe = "myfs"
Trace(MongoGridFSListeFichier(fs))
```





## Déclaration
<a name="declaration_ELTTEXTE000173"></a>
<a name="SYNTAXE1"></a>

`MaVariable est un mongoGridFS(<Base de données>, <Préfixe>)`
---

**`<Base de données> : (Chaîne de caractères)`**

Base de données MongoDB contenant le système de fichiers. 

**`<Préfixe> : (Chaîne de caractères)`**

Préfixe de la collection. 
Les fichiers seront stockés dans les collections sous la forme **&lt;prefixe&gt;.files** et **&lt;prefixe&gt;.chunck**.



<a name="NOTE0"></a>

## Remarques
<a name="NOTE0_1"></a>


### Propriétés spécifiques à la description des variables de type mongoGridFS
<a name="proprietes_specifiques_description_des_variables_type_mongogridfs_ELTPARAGRAPHE000051"></a>Les propriétés suivantes peuvent être utilisées pour manipuler un système de fichiers MongoDB :

| Nom de la propriété | Type manipulé | Effet |
| --- | --- | --- |
| BaseDeDonnées | Variable de type [mongoBase](../WDLang4/1000022410.md) | Base de données contenant le système de fichiers. |
| Préfixe | Chaîne de caractères | Préfixe de la collection. <br><br>Les fichiers seront stockés dans les collections sous la forme **&lt;prefixe&gt;.files** et **&lt;prefixe&gt;.chunck**. |

### Fonctions manipulant le type mongoGridFS
<a name="fonctions_manipulant_type_mongogridfs_ELTPARAGRAPHE000082"></a>Les fonctions manipulant le type mongoGridFS sont les suivantes : 



|   |   |
| --- | --- |
| [MongoGridFSEnvoieFichier](../WDLang4/1000022436.md) | Copie un fichier du poste local vers un système de fichiers MongoDB (GridFS). |
| [MongoGridFSListeFichier](../WDLang4/1000022438.md) | Récupère la liste des fichiers MongoDB présents dans le système de fichiers MongoDB (gridFS). |
| [MongoGridFSRécupèreFichier](../WDLang4/1000022439.md) | Récupère sur le poste local un fichier identifié par son nom et présent dans un système de fichiers MongoDB. |
| [MongoGridFSRécupèreFichierParID](../WDLang4/1000022595.md) | Récupère sur le poste local un fichier (identifié par son identifiant) présent dans un système de fichiers MongoDB. |
| [MongoGridFSRécupèreInfoFichier](../WDLang4/1000023495.md) | Permet de récupérer les informations concernant un fichier MongoDB (gridFS). |
| [MongoGridFSSupprimeFichier](../WDLang4/1000022440.md) | Supprime un fichier identifié par son nom dans un système de fichiers MongoDB. |
| [MongoGridFSSupprimeFichierParID](../WDLang4/1000022596.md) | Supprime un fichier identifié par son identifiant dans un système de fichiers MongoDB. |

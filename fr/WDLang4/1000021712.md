
## hbEcrit (Fonction)

***En anglais : hbWrite***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ecrit des cellules (valeurs de rubriques) dans une ou plusieurs lignes (enregistrements) d'une table HBase.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connexion
Connexion est une hbConnexion
Connexion.Serveur = "MonServeur"
// Description des données à écrire
Ecriture est une hbEcriture
Ecriture.Ligne = "post1"
Cellule est une hbCellule 
Cellule.Colonne.Famille = "post"
Cellule.Colonne.Qualificatif = "titre"
Cellule.Valeur = "Ceci est mon message sur le blog."
Ajoute(Ecriture.Cellule, Cellule)
// Ecriture des données
hbEcrit(Connexion, "blog", Ecriture)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = hbEcrit(<Connexion> , <Table HBase> , <Données à écrire>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'écriture a été réalisée,

- <u><u><u><u>Faux</u></u></u></u> en cas de problème. La fonction [ErreurInfo](../WDLang1/3013008.md) permet d'identifier l'erreur.




**`<Connexion> : (Variable de type hbConnexion)`**

Nom de la variable de type [hbConnexion](../WDLang4/1000021685.md) correspondant à la connexion au serveur HBase.

**`<Table HBase> : (Chaîne de caractères)`**

Nom de la table HBase à manipuler.

**`<Données à écrire> : (Variable de type hbEcriture)`**

Nom de la variable de type [hbEcriture](../WDLang4/1000021710.md) correspondant à la description de l'écriture à réaliser.

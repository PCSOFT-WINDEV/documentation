
## hOptimisationPlanifiée (Type de variable)

***En anglais : hScheduledOptimization***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **hOptimisationPlanifiée** permet de gérer une tâche d'optimisation planifiée sur un serveur HFSQL par programmation. Une tâche d'optimisation planifiée permet d'effectuer de façon périodique un calcul des statistiques ou une réindexation des fichiers de données du serveur HFSQL.

Les caractéristiques de cette tâche d'optimisation planifiée peuvent être définies et modifiées à l'aide de différentes propriétés WLangage.

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez ["Déclaration d'une variable"](../Motscles/1514032.md).






<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Recalculer les statistiques des fichiers de la base "MaBase", tous les samedis à 23H00
Optim est une hOptimisationPlanifiée
Optim.Description = "Recalcul des statistiques des fichiers de la base de données MaBase"

Optim.Planification.Mois = "*"
Optim.Planification.JourDeLaSemaine = "samedi"
Optim.Planification.Heure = "23"
Optim.Planification.Minute = "0"

Optim.Fichier[1].Nom = "MaBase"
Optim.Fichier[1].Option = hOptStat

HAjouteOptimisationPlanifiée(MaConnexion, Optim)
```

## Remarques


### Propriétés spécifiques à la description des variables de type hOptimisationPlanifiée
<a name="proprietes_specifiques_description_des_variables_type_hoptimisationplanifiee_ELTPARAGRAPHE000041"></a>Les propriétés suivantes peuvent être utilisées pour manipuler une variable de type **hOptimisationPlanifiée** :

| **Nom de la propriété** | **Type manipulé** | **Effet** |
| --- | --- | --- |
| Activé | Booléen | <br><br>- <u><u><u><u>Vrai</u></u></u></u> si la planification de l'optimisation est active, <br><br>- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.<br><br><br> |
| Description | Chaîne de caractères | Texte décrivant la planification de l'optimisation.<br>Par défaut, la description de la planification de l'optimisation correspond à une chaîne vide (""). |
| Fichier | Tableau de variables de type [hParamètreFichier](../WDLang4/1000018900.md) | Description des fichiers à prendre en compte lors de l'optimisation. |
| Identifiant | Entier | Identifiant de la tâche d'optimisation. <br>**Cette propriété est disponible en lecture seulement**. |
| NombreExécution | Entier | Nombre de fois où la tâche de planification d'optimisation doit être exécutée. Cette propriété peut également correspondre à la constante *Infini*. |
| Planification | Variable de type [hPlanification](../WDLang4/1000018853.md) | Caractéristiques de la planification. |
| Utilisateur | Chaîne de caractères | Utilisateur ayant créé la tâche de planification d'optimisation. <br>Cette propriété est en lecture uniquement. |

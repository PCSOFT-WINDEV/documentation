


## &lt;Variable Connexion&gt;.ListeTâche (Fonction)

***En anglais : &lt;Connection variable&gt;.ListTask***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Liste les tâches planifiées d'un serveur HFSQL Client/Serveur pour une connexion donnée.




<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Connexion>.ListeTâche()`
---

**`<Résultat> : (Chaîne de caractères)`**

Liste des tâches planifiées disponibles sur le serveur sous la forme :

```txt
<Identifiant de la tâche 1> + TAB + <Description de la tâche 1> + TAB + ...
<Tâche 1 active ou non> + TAB + <Utilisateur> + TAB + <Mois> + TAB + ...
<Jour du mois> + TAB + <Jour de la semaine> + TAB + <Heure> + TAB + ...
<Minute> + TAB + <Jour du mois ou jour de la semaine> + TAB + ...
<Nombre exécution> + TAB + <Nom de la procédure> + EOT + ...
```

**Remarque** : Pour obtenir les caractéristiques d'une tâche, utilisez la fonction [&lt;Variable Connexion&gt;.InfoTâche](../WDLang4/1000022686.md).

**`<Connexion> : (Variable de type Connexion)`**

Nom de la variable de type [Connexion](../WDLang4/1514073.md) décrivant la connexion pour laquelle les tâches planifiées doivent être listées.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Pour lister les tâches des autres utilisateurs, il est nécessaire de posséder le droit de visualisation des utilisateurs (constante *hDroitVisualisationUtilisateur* dans les fonctions [&lt;Variable Connexion&gt;.InfoDroitServeur](../WDLang4/1000022668.md) et [&lt;Variable Connexion&gt;.ModifieDroitServeur](../WDLang4/1000022710.md)).

- Pour lister les tâches planifiées, il est nécessaire de posséder le droit de connexion (constante *hDroitConnexion* ou *hDroitConnexionCryptée*) sur la base contenant la procédure stockée. 

- Il est possible d'utiliser la fonction **&lt;Variable Connexion&gt;.ListeTâche** pour vérifier l'existence d'une tâche. Il suffit de vérifier si l'identifiant de la tâche est présent parmi les tâches renvoyées par la fonction.

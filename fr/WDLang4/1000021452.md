


## HListePlanification (Fonction)

***En anglais : HListScheduling***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Liste les tâches planifiées définies sur un serveur HFSQL : 

- procédure stockée, 

- sauvegarde, 

- optimisation, 

- rafraîchissement d'une vue matérialisée.





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Modification de toutes les sauvegardes planifiées
tabDescSvg est un tableau dynamique de hDescriptionSauvegarde 
tabDescSvg = HListePlanification(MaConnexionBase, hlpSauvegarde)
hDescSvg est une hDescriptionSauvegarde
POUR TOUT hDescSvg DE tabDescSvg
	SI hDescSvg.LimiteNombreSauvegarde = 3 ALORS 
		hDescSvg.LimiteNombreSauvegarde = 5
		HModifiePlanification(MaConnexionBase, hDescSvg)
	FIN
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = HListePlanification(<Connexion> , <Type de planification>)`
---

**`<Résultat> : (Tableau)`**

Tableau contenant les descriptions listées. Le type des éléments du tableau dépend du paramètre &lt;Type de planification&gt;. 

**`<Connexion> : (Chaîne de caractères ou variable de type Connexion)`**

Connexion à manipuler. Cette connexion correspond : 

- soit au nom de la connexion définie sous l'éditeur d'analyses ou grâce aux fonctions [HDécritConnexion](../WDLang4/3044205.md) ou [HOuvreConnexion](../WDLang4/3044107.md).

- soit au nom d'une variable de type [Connexion](../WDLang4/1514073.md).




**`<Type de planification> : (Constante de type Entier)`**

Type de planification à lister : 


|   |   |
| --- | --- |
| *hlpOptimisation* | Tâches d'optimisation des fichiers de données (réindexation, calcul des statistiques). |
| *hlpSauvegarde* | Planifications des sauvegardes. |
| *hlpTâche* | Tâches planifiées exécutant une procédure stockée. |
| *hlpVueMatérialisée* | Tâches de rafraîchissement des vues matérialisées. |

## Remarques


### Droits nécessaires
<a name="droits_necessaires_ELTPARAGRAPHE000104"></a>Pour lister les tâches de rafraîchissement des vues matérialisées ou les tâches planifiées, il est nécessaire de posséder le droit de connexion (constante *hDroitConnexion* ou *hDroitConnexionCryptée*) pour chaque base de données concernée, c'est-à-dire pour chaque base de données qui contient une vue à traiter ou une procédure stockée à exécuter. 

Pour savoir si un utilisateur ou un groupe possède ces droits, utilisez la fonction [HInfoDroitServeur](../WDLang4/3044322.md) ou la fonction [HInfoDroitDaseDeDonnées](../WDLang4/3044320.md).

Pour modifier les droits d'un utilisateur ou d'un groupe d'utilisateurs, utilisez la fonction [HModifieDroitServeur](../WDLang4/3044328.md) ou la fonction  [HModifieDroitBaseDeDonnées](../WDLang4/3044326.md).

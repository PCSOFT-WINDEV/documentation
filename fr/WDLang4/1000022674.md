


## &lt;Variable Connexion&gt;.InfoPropriétéFichier (Fonction)

***En anglais : &lt;Connection variable&gt;.InfoFileProperty***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de connaître les propriétés d'un fichier de données situé sur un serveur HFSQL.



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Description de la connexion
MaConnexion est une Connexion

// Ouverture de la connexion
MaConnexion.OuvreConnexion()
...
NomPropriétaire est une chaîne
NomPropriétaire = MaConnexion.InfoPropriétéFichier(hPropriétaire, ...
	"MaBaseDeDonnées", "Client.fic")
SI NomPropriétaire <> "" ALORS 
	Trace("Propriétaire de la base de donnée : " + NomPropriétaire)
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Connexion>.InfoPropriétéFichier(<Propriété> , <Base de données> , <Fichier de données>)`
---

**`<Résultat> : (Chaîne de caractères)`**



- Valeur de la propriété demandée, 

- Chaîne vide ("") en cas d'erreur.




**`<Connexion> : (Variable de type Connexion)`**

Nom de la variable de type [Connexion](../WDLang4/1514073.md) décrivant la connexion pour laquelle la valeur d'une propriété est recherchée.

**`<Propriété> : (Constante de type Entier)`**

Propriété du fichier recherchée : 


|   |   |
| --- | --- |
| *hApplication* | Nom de l'application à partir de laquelle le fichier de données a été créé. <br>Si cette constante est utilisée, &lt;Résultat&gt; est une chaîne de caractères de la forme : <br>&lt;Nom de l'exécutable&gt;(&lt;Nom du projet&gt;). |
| *hCréateur* | Nom du propriétaire du fichier de données.<br>Si cette constant est utilisée, &lt;Résultat&gt; est une chaîne de caractères. |
| *hDate* | Date de la création du fichier de données. <br>Si cette constante est utilisée, &lt;Résultat&gt; est une chaîne de caractères de la forme AAAAMMJJHHmmSSCCC. |
| *hMachine* | Nom du poste à partir duquel a été créé le fichier de données. <br>Si cette constante est utilisée, &lt;Résultat&gt; est de la forme : <br>&lt;Nom du poste client&gt; + TAB + &lt;Adresse IP du poste Client&gt;. |
| *hNotes* | Description du fichier de données ou toute autre information associée au fichier de données grâce à la fonction [&lt;Variable Connexion&gt;.ModifiePropriétéFichier](../WDLang4/1000022718.md). <br>Si cette constante est utilisée, &lt;Résultat&gt; est une chaîne de caractères. |
| *hPropriétaire* | Propriétaire du fichier de données. <br>Si cette constante est utilisée, &lt;Résultat&gt; est une chaîne de caractères. |



**`<Base de données> : (Chaîne de caractères)`**

Nom de la base de données contenant le fichier pour lequel les propriétés sont recherchées. 
Si ce paramètre correspond à une chaîne vide (""), la base de données utilisée sera celle définie dans la connexion. 
Si aucune base de données n'est définie dans la connexion, une erreur WLangage sera générée.

**`<Fichier de données> : (Chaîne de caractères)`**

Nom du fichier de données HFSQL pour lequel les propriétés sont recherchées. L'extension du fichier doit être précisée.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Pour modifier les propriétés d'un fichier de données, utilisez la fonction [&lt;Variable Connexion&gt;.ModifiePropriétéFichier](../WDLang4/1000022718.md).

- Attention : le fichier de données manipulé doit avoir été créé sur le serveur (fonctions [&lt;Source&gt;.Création](../WDLang4/1000024207.md) ou [&lt;Source&gt;.CréationSiInexistant](../WDLang4/1000024208.md)).

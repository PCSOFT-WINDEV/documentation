
## HListeAnalyse (Fonction)

***En anglais : HListAnalysis***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la liste des analyses WINDEV ou WEBDEV (au format HFSQL Classic) disponibles dans un répertoire donné. La recherche est effectuée de façon récursive dans toute l'arborescence du répertoire spécifié.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple
<a class="notetitle" target="_blank" href="$DOC$=1000003044049&name=hlisteanalyse_fonction&product=WD">Voir des exemples supplémentaires</a>

```wl
// Remplit un champ Combo avec la liste des analyses
// disponibles sur \Mes répertoires
ListeAjoute(COMBO_Combo, HListeAnalyse("\Mes répertoires"))

// Remplit un champ de saisie multiligne 
// En utilisant comme séparateur la virgule et le RC
SAI_Saisie1 = HListeAnalyse("\Mes Projets", "," + RC)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = HListeAnalyse(<Répertoire de recherche> [, <Séparateur>])`
---

**`<Résultat> : (Chaîne de caractères)`**

Nom complet des analyses trouvées, exprimé sous la forme : 

```txt
<analyse 1> + <séparateur> + <analyse 2> + ...
```

où &lt;analyse x&gt; correspond au nom complet de l'analyse (.WDD) trouvée.

**`<Répertoire de recherche> : (Chaîne de caractères)`**

Chemin du répertoire par lequel la recherche des analyses doit débuter. Si ce paramètre est égal à "\*" ou "\*:\\", tous les répertoires de tous les disques accessibles seront parcourus (selon les cas, cette recherche peut être relativement longue et prendre quelques minutes).

**`<Séparateur> : (Chaîne de caractères optionnelle)`**

Chaîne de caractères utilisée pour séparer les différentes analyses dans &lt;Résultat&gt;. Si ce paramètre n'est pas spécifié, RC (Retour chariot) est utilisé par défaut.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Affichage du résultat dans un champ Liste ou un champ Combo
<a name="affichage_resultat_dans_champ_liste_champ_combo_ELTPARAGRAPHE000042"></a>Si le séparateur utilisé est le retour chariot (RC), &lt;Résultat&gt; peut être directement affecté à un champ Liste ou un champ Combo grâce à la fonction [ListeAjoute](../WDLang1/3049004.md). Chaque ligne contiendra le nom complet d'une des analyses trouvées.

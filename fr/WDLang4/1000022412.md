
## mongoBaseInfo (Type de variable)

***En anglais : mongoDatabaseInfo***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **mongoBaseInfo** permet de connaître les caractéristiques des informations d'une base MongoDB. Les caractéristiques de ces informations peuvent être connues à l'aide de différentes propriétés WLangage. 

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
Info est un mongoBaseInfo 
Info = MongoInfo(baseMDB)
```

## Remarques
<a name="NOTE0_1"></a>


### Propriétés spécifiques à la description des variables de type mongoBaseInfo
<a name="proprietes_specifiques_description_des_variables_type_mongobaseinfo_ELTPARAGRAPHE000040"></a>Les propriétés suivantes peuvent être utilisées pour manipuler les informations d'une base MongoDB :


| Nom de la propriété | Type manipulé | Effet |
| --- | --- | --- |
| NbCollections | Entier | Nombre de collections contenues dans la base.<br><br>**Cette propriété est disponible en lecture seulement**. |
| NbDocuments | Entier | Nombre de documents contenus dans la base (parmi toutes les collections).<br><br>**Cette propriété est disponible en lecture seulement**. |
| NbExtents | Entier | Nombre d'extents dans la base.<br><br>**Cette propriété est disponible en lecture seulement**. |
| NbIndexes | Entier | Nombre d'index.<br><br>**Cette propriété est disponible en lecture seulement**. |
| Nom | Chaîne de caractères | Nom de la base.<br><br>**Cette propriété est disponible en lecture seulement**. |
| TailleDonnées | Entier | Taille totale en octets des données contenues dans la base.<br><br>Remarques : <br><br>- Le padding est inclus dans ce total. <br><br>- Le facteur d'échelle affecte cette valeur. <br><br>- La taille ne diminue pas lorsque l'on réduit un document ou que l'on supprime des documents.<br><br><br>**Cette propriété est disponible en lecture seulement**. |
| TailleFichier | Entier | Taille totale en octets des fichiers de données de la base. <br><br>Remarques : <br><br>- L'espace pré-alloué est inclus dans ce total. <br><br>- Le facteur de padding est inclus dans ce total.<br><br><br>**Cette propriété est disponible en lecture seulement**. |
| TailleIndex | Entier | Taille totale en octets des index de la base.<br><br>**Cette propriété est disponible en lecture seulement**. |
| TailleMoyenneDocument | Entier | Taille moyenne en octets des documents.Cette taille correspond à la taille de la base divisée par le nombre de documents. <br><br>**Cette propriété est disponible en lecture seulement**. |
| TailleNamespace | Entier | Taille totale des fichiers de namespace (fichiers avec l'extension .ns) pour la base.<br><br>**Cette propriété est disponible en lecture seulement**. |
| TailleStockage | Entier | Taille totale en octets de l'espace alloué pour le stockage des données.<br><br>**Cette propriété est disponible en lecture seulement**. |
| VersionFichierDonnées | Chaîne de caractères | Version du format de stockage disque (uniquement pour mmapv1). La valeur est au format **majeur.mineur**.<br><br>**Cette propriété est disponible en lecture seulement**. |

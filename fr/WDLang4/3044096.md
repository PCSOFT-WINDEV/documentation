


## HFichierExiste (Fonction)

***En anglais : HFileExist***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de savoir :

- si un fichier de données existe, c'est-à-dire s'il a été créé physiquement (fonctions [HCréation](../WDLang4/3044255.md) ou [HCréationSiInexistant](../WDLang4/3044254.md)). 
	Remarque : Les substitutions en cours sur le fichier de données (effectuées avec les fonctions [HSubstRep](../WDLang4/3044028.md), [HChangeRep](../WDLang4/3044168.md) et [HChangeNom](../WDLang4/3044141.md)) sont prises en compte.

- si une vue ou une requête a été définie.





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple
<a class="notetitle" target="_blank" href="$DOC$=1000003044096&name=hfichierexiste_fonction&product=WD">Voir des exemples supplémentaires</a>

```wl
// Test de l'existence du fichier de données "Client"
SI HFichierExiste(Client) = Vrai ALORS
	// Traitements
	...
FIN
```

```wl
// Test de l'existence d'un fichier de données Client/Serveur
SI HFichierExiste(MaConnexion, "MonFichier.FIC") = Vrai ALORS
	// Traitements
	...
FIN
```

## Syntaxe

### Tester l'existence d'un fichier de données, d'une requête ou d'une vue

`<Résultat> = HFichierExiste([<Elément>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le fichier physique existe,

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Elément> : (Chaîne de caractères optionnelle)`**

Nom de l'élément à manipuler : 

- Nom du fichier de données.

- Nom de la vue ou de la requête.




Si ce nom n'est pas spécifié, la fonction **HFichierExiste** va manipuler le dernier fichier de données, la dernière requête ou vue manipulé par la dernière fonction de gestion HFSQL (fonction commençant par la lettre H).

### Tester l'existence d'un fichier de données en HFSQL Client/Serveur

`<Résultat> = HFichierExiste(<Connexion> , <Fichier physique>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le fichier de données physique existe sur le serveur, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Connexion> : (Chaîne de caractères ou variable de type Connexion)`**

Connexion associée au fichier de données. Cette connexion correspond : 

- soit au nom de la connexion définie sous l'éditeur d'analyses ou grâce aux fonctions [HDécritConnexion](../WDLang4/3044205.md) ou [HOuvreConnexion](../WDLang4/3044107.md).

- soit au nom d'une variable de type [Connexion](../WDLang4/1514073.md).




**`<Fichier physique> : (Chaîne de caractères)`**

Nom physique du fichier de données recherché. L'extension du fichier doit être précisée (par exemple "Client.fic").



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- La fonction **HFichierExiste** est utilisable que le fichier de données soit ouvert ou non.

- La fonction **HFichierExiste** n'ouvre pas le fichier de données si celui-ci est fermé.

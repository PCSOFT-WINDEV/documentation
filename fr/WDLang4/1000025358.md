


## &lt;Variable mongoGridFS&gt;.ListeFichier (Fonction)

***En anglais : &lt;mongoGridFS variable&gt;.ListFile***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Récupère la liste des fichiers MongoDB présents dans le système de fichiers MongoDB (gridFS).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple

<a name="Exemple2"></a>

```wl
gclGridFS est un mongoGridFS

TABLE_GridFS.SupprimeTout()

tabFichier est un tableau de mongoGridFSInfoFichier = gclGridFS.ListeFichier()
fic est un mongoGridFSInfoFichier
POUR TOUT fic DE tabFichier
	TABLE_GridFS.AjouteLigne(VariantVersJSON(fic.ID), ...
	fic.Nom, fic.Date, fic.Md5, fic.Taille, fic.TailleChunk, fic.TypeContenu)
FIN

TABLE_GridFS.SelectMoins()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Système de fichiers>.ListeFichier()`
---

**`<Résultat> : (Tableau de mongoGridFSInfoFichier)`**

Tableau de variables de type [mongoGridFSInfoFichier](../WDLang4/1000022417.md), correspondant à la description des fichiers listés.

**`<Système de fichiers> : (Variable de type mongoGridFS)`**

Nom de la variable de type [mongoGridFS](../WDLang4/1000022416.md) correspondant au système de fichiers Mongo à manipuler.

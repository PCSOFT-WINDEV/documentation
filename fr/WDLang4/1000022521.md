


## HDéclencheMiseAJourServeur (Fonction)

***En anglais : HStartServerUpdate***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de déclencher immédiatement la mise à jour automatique du serveur. Les utilisateurs seront déconnectés lors du lancement de cette opération.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Déclenche la mise à jour du serveur
HDéclencheMiseAJourServeur(MaConnexion)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = HDéclencheMiseAJourServeur(<Connexion> [, <Version>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'opération a été réalisée, 

- <u><u><u><u>Faux</u></u></u></u> en cas de problème.


La fonction [HErreurInfo](../WDLang4/3044071.md) permet d'identifier l'erreur.

**`<Connexion> : (Chaîne de caractères ou variable de type Connexion)`**

Connexion à une base de données. Cette connexion correspond :

- soit au nom de la connexion définie dans l'éditeur d'analyses.

- soit au nom de la connexion définie par la fonction [HDécritConnexion](../WDLang4/3044205.md). La description de la connexion doit être connue au moment de l'exécution de la fonction.

- soit au nom d'une variable de type [Connexion](../WDLang4/1514073.md).




**`<Version> : (Chaîne de caractères optionnelle)`**

Version pour laquelle la mise à jour doit être effectuée (par exemple : "240036"). Par défaut, si ce paramètre n'est pas spécifié, la dernière mise à jour disponible est utilisée. 




## Remarques
Le serveur de mise à jour (contenant le pack de mise à jour du serveur HFSQL) peut être défini par la fonction [HGèreMiseAJourServeur](../WDLang4/1000022635.md).




## &lt;Source&gt;.FichierExiste (Fonction)

***En anglais : &lt;Source&gt;.FileExist***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de savoir si un fichier de données existe, c'est-à-dire s'il a été créé physiquement (fonctions [&lt;Source&gt;.Création](../WDLang4/1000024207.md) ou [&lt;Source&gt;.CréationSiInexistant](../WDLang4/1000024208.md)). 

Remarque : Les substitutions en cours sur le fichier de données (effectuées avec les fonctions [HSubstRep](../WDLang4/3044028.md), [&lt;Source&gt;.ChangeRep](../WDLang4/1000024202.md) et [&lt;Source&gt;.ChangeNom](../WDLang4/1000024201.md)) sont prises en compte.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Test de l'existence du fichier de données "Client"
SI Client.FichierExiste() = Vrai ALORS
	// Traitements
	...
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Tester l'existence d'un fichier de données, d'une requête ou d'une vue

`<Résultat> = <Source>.FichierExiste()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le fichier physique existe,

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Source> : (Type correspondant à la source spécifiée)`**

Nom de l'élément à manipuler : 

- Nom du fichier de données.

- Nom de la vue ou de la requête.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- La fonction **&lt;Source&gt;.FichierExiste** est utilisable que le fichier de données soit ouvert ou non.

- La fonction **&lt;Source&gt;.FichierExiste** n'ouvre pas le fichier de données si celui-ci est fermé.

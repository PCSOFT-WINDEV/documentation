


## Fonctions de gestion des fichiers Hyper File conservées pour compatibilité uniquement
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Voici la liste des fonctions de gestion des fichiers Hyper File (utilisées en version 5.5) conservées par compatibilité uniquement avec leur équivalence en version 28.


|   |   |
| --- | --- |
| <a name="BinaireCharge"></a><br>BinaireCharge | Charge un fichier binaire dans une rubrique de type mémo binaire 4.1 (initialise un mémo binaire avec le contenu d'un fichier).<br><br>Fonction équivalente : [HAttacheMémo](../WDLang4/3044173.md). |
| <a name="BinaireInfo"></a><br>BinaireInfo | Permet de connaître les informations d'un mémo binaire 4.1 (type, nom, date, heure et taille du fichier d'origine).<br><br>Fonction équivalente : [HInfoMémo](../WDLang4/3044008.md) |
| <a name="BinaireSauve"></a><br>BinaireSauve | Sauve le contenu d'une rubrique mémo binaire 4.1 dans un fichier.<br><br>Fonction équivalente : [HExtraitMémo](../WDLang4/3044072.md) |
| <a name="HAlias_55"></a><br>HAlias_55 | Crée un alias d'un fichier logique ou annuler tous les alias d'un fichier logique.<br><br>Fonction équivalente : [HAlias](../WDLang4/3044176.md). |
| <a name="HAliasExterne"></a><br>HAliasExterne | Charge la description de fichiers Hyper File contenue dans une analyse externe à l'application sans fermer l'analyse en cours.<br><br>Fonctions équivalentes : <br><br>- [HDéclare](../WDLang4/3044203.md) : Importe une description de fichier à partir d'une analyse autre que l'analyse en cours.<br><br>- [HDéclareExterne](../WDLang4/3044204.md) : Importe une description de fichier à partir d'un fichier existant.<br><br><br> |
| <a name="HAssigne"></a><br>HAssigne | Modifie le chemin d'accès d'un fichier.<br><br>Fonction équivalente : [HChangeRep](../WDLang4/3044168.md). |
| <a name="HCopieEnreg_55"></a><br>HCopieEnreg_55 | Copie l'enregistrement en cours d'un fichier dans l'enregistrement en cours d'un autre fichier ayant la même structure.<br><br>Fonction équivalente : [HCopieEnreg](../WDLang4/3044261.md) |
| <a name="HCréeVue_55"></a><br>HCréeVue_55 | Crée une vue sur un fichier Hyper File.<br><br>Fonction équivalente : [HCréeVue](../WDLang4/3044159.md). |
| <a name="HDébutVerrou"></a><br>HDébutVerrou | Marque le début du verrouillage de lecture des fichiers. Toutes les fonctions de lecture 'simple' bloquent automatiquement les enregistrements lus.<br><br><br>Fonctions équivalentes : <br><br>- Utiliser les fonctions de lecture avec les constantes de blocage appropriées : [HLit](../WDLang4/3044047.md), [HLitPremier](../WDLang4/3044051.md), [HLitSuivant](../WDLang4/3044037.md), .... <br><br>- Ouvrir le fichier de données en lecture avec la fonction [HOuvre](../WDLang4/3044120.md).  <br><br>- Fonction [HBloqueFichier](../WDLang4/3044153.md)<br><br><br> |
| <a name="HDécritClé"></a><br>HDécritClé | Décrit une clé simple ou composée dans un fichier temporaire au format Hyper File. <br><br>Fonctionnement équivalent : <br><br>1. Déclarer une variable de type [Description de Rubrique](../WDLang4/1514071.md).<br><br>2. Définir toutes les propriétés de la rubrique grâce aux propriétés HFSQL.<br><br>3. Utiliser la fonction [HDécritRubrique](../WDLang4/3044233.md) pour valider la création de la rubrique.<br><br><br> |
| <a name="HDécritTrigger_55"></a><br>HDécritTrigger_55 | Ajoute ou modifie un trigger sur Hyper File. <br><br>Fonction équivalente : [HDécritTrigger](../WDLang4/3044074.md) |
| <a name="HDéjàBloque"></a><br>HDéjàBloque | Teste si la dernière opération WINDEV a pu être exécutée ou non. <br><br>Fonctionnement équivalent : cette fonction a été remplacée par :<br><br>- une gestion automatique des blocages (désactivable si nécessaire),<br><br>- la fonction [HErreurBlocage](../WDLang4/3044099.md).<br><br><br> |
| <a name="HDoublon"></a><br>HDoublon | Teste l'unicité d'une clé.<br><br>Fonctionnement équivalent : cette fonction a été remplacée par :<br><br>- une gestion automatique des doublons, permettant à l'utilisateur de saisir une valeur différente pour la rubrique générant une erreur de doublon.<br><br>- la fonction [HErreurDoublon](../WDLang4/3044077.md).<br><br><br> |
| <a name="HFinVerrou"></a><br>HFinVerrou | Marque la fin du verrouillage des fichiers : tous les enregistrements bloqués sont débloqués. <br><br>Fonctions équivalentes : <br><br>- Utiliser les fonctions de lecture avec les constantes appropriées : [HLit](../WDLang4/3044047.md), [HLitPremier](../WDLang4/3044051.md), [HLitSuivant](../WDLang4/3044037.md), .... <br><br>- Ouvrir le fichier de données en lecture/écriture avec la fonction [HOuvre](../WDLang4/3044120.md).  <br><br>- Fonction [HDébloqueFichier](../WDLang4/3044160.md). <br><br><br> |
| <a name="HImporteTexte_55"></a><br>HImporteTexte_55 | Importe un fichier texte dans un fichier Hyper File. <br><br>Fonction équivalente : [HImporteTexte](../WDLang4/3044011.md). |
| <a name="HInfoGéné"></a><br>HInfoGéné | Retourne les informations générales sur un fichier Hyper File.<br><br>Fonctionnement équivalent : cette fonction a été remplacée par :<br><br>- des fonctions (par exemple, [HNbEnr](../WDLang4/3044006.md) remplace la variable H.NbEnr).<br><br>- des [propriétés HFSQL](../Proprietes/2512070.md) (par exemple, la propriété **Répertoire** remplace la variable H.Rep).<br><br><br> |
| <a name="HInfoRubrique"></a><br>HInfoRubrique | Permet d'obtenir des informations sur une rubrique de l'analyse. <br><br>Fonctionnement équivalent : cette fonction a été remplacée par différentes [propriétés HFSQL](../Proprietes/2512070.md). |
| <a name="HLitBloque"></a><br>HLitBloque | Lit et bloque un enregistrement en fonction du numéro d'enregistrement spécifié. <br><br>Fonction équivalente : [HLit](../WDLang4/3044047.md) avec les constantes permettant la gestion des blocages. |
| <a name="HLitDernierBloque"></a><br>HLitDernierBloque | Lit et bloque le dernier enregistrement du fichier selon la clé passée en paramètre.<br><br>Fonction équivalente : [HLitDernier](../WDLang4/3044140.md) avec les constantes permettant la gestion des blocages. |
| <a name="HLitDernierSansBloquer"></a><br>HLitDernierSansBloquer | Lit le dernier enregistrement du fichier selon la clé passée en paramètre. Si le fichier est verrouillé, l'enregistrement lu n'est pas bloqué.<br><br>Fonction équivalente : [HLitDernier](../WDLang4/3044140.md) avec les constantes permettant la gestion des blocages. |
| <a name="HLitPrécédentBloque"></a><br>HLitPrécédentBloque | Lit et bloque l'enregistrement précédent selon la clé spécifiée.<br><br>Fonction équivalente : [HLitPrécédent](../WDLang4/3044048.md) avec les constantes permettant la gestion des blocages. |
| <a name="HLitPrécédentSansBloquer"></a><br>HLitPrécédentSansBloquer | Lit l'enregistrement précédent selon la clé spécifiée. Si le fichier est verrouillé, l'enregistrement lu n'est pas bloqué. <br><br>Fonction équivalente : [HLitPrécédent](../WDLang4/3044048.md) avec les constantes permettant la gestion des blocages. |
| <a name="HLitPremierBloque"></a><br>HLitPremierBloque | Lit et bloque le premier enregistrement du fichier relativement à la clé spécifiée.<br><br>Fonction équivalente : [HLitPremier](../WDLang4/3044051.md) avec les constantes permettant la gestion des blocages. |
| <a name="HLitPremierSansBloquer"></a><br>HLitPremierSansBloquer | Lit le premier enregistrement selon la clé spécifiée. Si le fichier est verrouillé, l'enregistrement lu n'est pas bloqué.<br><br>Fonction équivalente : [HLitPremier](../WDLang4/3044051.md) avec les constantes permettant la gestion des blocages. |
| <a name="HLitRechercheBloque"></a><br>HLitRechercheBloque | Lit et bloque le premier enregistrement dont la valeur de la clé donnée est égale ou supérieure à la valeur de l'argument de recherche.<br><br>Fonction équivalente : [HLitRecherche](../WDLang4/3044050.md) avec les constantes permettant la gestion des blocages.<br><br>Fonction équivalente : [HLitRecherche](../WDLang4/3044050.md) avec les constantes permettant la gestion des blocages. |
| <a name="HLitRechercheSansBloquer"></a><br>HLitRechercheSansBloquer | Lit le premier enregistrement dont la valeur de la clé donnée est égale ou supérieure à la valeur de l'argument de recherche. Si le fichier est verrouillé, l'enregistrement lu n'est pas bloqué.<br><br>Fonction équivalente : [HLitRecherche](../WDLang4/3044050.md) avec les constantes permettant la gestion des blocages. |
| <a name="HLitSansBloquer"></a><br>HLitSansBloquer | Lit l'enregistrement en fonction du numéro d'enregistrement spécifié. Si le fichier est verrouillé, l'enregistrement lu n'est pas bloqué.<br><br>Fonction équivalente : [HLit](../WDLang4/3044047.md) avec les constantes permettant la gestion des blocages. |
| <a name="HLitSuivantBloque"></a><br>HLitSuivantBloque | Lit et bloque l'enregistrement suivant selon la clé spécifiée.<br><br>Fonction équivalente : [HLitSuivant](../WDLang4/3044037.md) avec les constantes permettant la gestion des blocages. |
| <a name="HLitSuivantSansBloquer"></a><br>HLitSuivantSansBloquer | Lit l'enregistrement suivant selon la clé spécifiée. Si le fichier est verrouillé, l'enregistrement lu n'est pas bloqué.<br><br>Fonction équivalente : [HLitSuivant](../WDLang4/3044037.md) avec les constantes permettant la gestion des blocages. |
| <a name="HRéindexe_55"></a><br>HRéindexe_55 | Réindexe d'un fichier Hyper File de l'analyse en cours.<br><br>Fonction équivalente : [HRéindexe](../WDLang4/3044133.md). |
| <a name="HSubstNom"></a><br>HSubstNom | Modifie le nom (physique) d'un fichier de données.<br><br>Fonction équivalente : [HChangeNom](../WDLang4/3044141.md). |
| <a name="HVueVersTable"></a><br>HVueVersTable | Remplit un champ Table par programmation avec une vue.<br><br>Fonction équivalente : [FichierVersTableMémoire](../WDLang1/3074005.md). |

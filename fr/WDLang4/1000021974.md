
## HClusterInfoSynchronisation (Fonction)

***En anglais : HClusterSynchronizationInfo***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie l'état d'avancement de la synchronisation d'un noeud d'un cluster HFSQL. 

Rappel : Une synchronisation est automatiquement effectuée par exemple lorsqu'un noeud du cluster s'est arrêté anormalement : au redémarrage, ce noeud se synchronise avec un autre noeud qui contient les données à jour.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Etat de la synchronisation
EtatActuel est une chaîne
EtatActuel = HClusterInfoSynchronisation("MonCoordinateur","MonMotDePasse", "NomNoeud")
Trace("Etat en cours " + ExtraitChaîne(EtatActuel, 1, RC) + ...
	"Numéro étape " + ExtraitChaîne(EtatActuel, 2, RC) + ...
	"Nombre d'étapes " + ExtraitChaîne(EtatActuel, 3, RC) + ...
	"Avancement " + ExtraitChaîne(EtatActuel, 4, RC))
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = HClusterInfoSynchronisation(<Coordinateur> , <Mot de passe> , <Noeud>)`
---

**`<Résultat> : (Chaîne de caractères)`**

Informations sur la synchronisation du noeud. Ce résultat est de la forme : 

```txt
<Description de l'étape en cours> + RC + <Numéro de l'étape en cours> + RC +
<Nombre d'étapes> + RC + <Avancement (en pour mille)>
```


**`<Coordinateur> : (Chaîne de caractères)`**

Nom DNS de la machine où se trouve le coordinateur du cluster.

**`<Mot de passe> : (Chaîne de caractères)`**

Mot de passe permettant de se connecter au coordinateur. Ce mot de passe a été fourni lors de [l'installation du cluster](../WDLang4/1000019031.md).

**`<Noeud> : (Chaîne de caractères)`**

Nom DNS du noeud en cours de synchronisation.

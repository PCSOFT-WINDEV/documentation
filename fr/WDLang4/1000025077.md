


## &lt;Source&gt;.VérifieInaltéré (Fonction)

***En anglais : &lt;Source&gt;.CheckUnaltered***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Vérifie qu'un ou tous les enregistrements d'un fichier de données inaltérable n'ont pas été modifiés (par exemple avec un éditeur hexadécimal).

Attention : Un enregistrement altéré pourra être lu si l'en-tête de l'enregistrement n'est pas altéré.  
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Vérification de tous les enregistrements du fichier MonFichierClient
SI MonFichierClient.VérifieInaltéré() = Vrai ALORS
	Trace("Aucun enregistrement n'a été modifié")
SINON
	Trace("Il y a eu altération du fichier")
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Source>.VérifieInaltéré([<Enregistrement> [, <Jauge>]])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'enregistrement n'a pas été altéré, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Dans ce cas, la fonction [ErreurInfo](../WDLang1/3013008.md) permet d'obtenir la liste des enregistrements altérés.




**`<Source> : (Type correspondant à la source spécifiée)`**

Nom du fichier de données HFSQL à vérifier.

**`<Enregistrement> : (Entier ou constante de type Entier optionnels)`**



- Numéro de l'enregistrement à vérifier.

- Constante permettant de spécifier l'enregistrement à vérifier : 
	


|   |   |
| --- | --- |
| *hNumEnrEnCours* | Vérification de l'enregistrement courant. |
| *hNumEnrTous*<br>(Valeur par défaut) | Vérification de tous les enregistrements. |






**`<Jauge> : (Nom de champ ou nom de fenêtre optionnel)`**



- Nom de la fenêtre dans laquelle la jauge doit être affichée. 

- Nom du champ de type Jauge affichant la progression.




## HGèreTâche (Fonction)

***En anglais : HManageTask***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Active ou désactive une tâche planifiée d'un serveur HFSQL Client/Serveur. Cette fonction permet également de connaître l'état d'une tâche planifiée.

Il est ainsi possible par exemple de désactiver une tâche (pendant les vacances) puis de la réactiver.



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Activer/Désactiver une tâche planifiée

`<Résultat> = HGèreTâche(<Connexion> , <Tâche planifiée> [, <Activer/Désactiver>])`
---

**`<Résultat> : (Booléen)`**

Ancien mode de gestion de la tâche (avant utilisation de la fonction **HGèreTâche**) :

- <u><u><u><u>Vrai</u></u></u></u> si la tâche était active, 

- <u><u><u><u>Faux</u></u></u></u> si la tâche n'était pas active.




**`<Connexion> : (Chaîne de caractères ou variable de type Connexion)`**

Connexion pour laquelle la tâche doit être gérée. Cette connexion correspond : 

- soit au nom de la connexion définie sous l'éditeur d'analyses ou grâce aux fonctions [HDécritConnexion](../WDLang4/3044205.md) ou [HOuvreConnexion](../WDLang4/3044107.md).

- soit au nom d'une variable de type [Connexion](../WDLang4/1514073.md).




**`<Tâche planifiée> : (Entier)`**

Identifiant de la tâche planifiée à manipuler. Cet identifiant est fourni par les fonctions [HAjouteTâche](../WDLang4/1000017091.md) et [HListeTâche](../WDLang4/1000017092.md).

**`<Activer/Désactiver> : (Booléen optionnel)`**



- <u><u><u><u>Vrai</u></u></u></u> pour activer la tâche, 

- <u><u><u><u>Faux</u></u></u></u> pour désactiver la tâche.  





<a name="SYNTAXE2"></a>

### Connaître l'état d'une tâche planifiée

`<Résultat> = HGèreTâche(<Connexion> , <Tâche planifiée>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la tâche est active, 

- <u><u><u><u>Faux</u></u></u></u> si la tâche n'est pas active.




**`<Connexion> : (Chaîne de caractères ou variable de type Connexion)`**

Connexion pour laquelle la tâche doit être gérée. Cette connexion correspond : 

- soit au nom de la connexion définie sous l'éditeur d'analyses ou grâce aux fonctions [HDécritConnexion](../WDLang4/3044205.md) ou [HOuvreConnexion](../WDLang4/3044107.md).

- soit au nom d'une variable de type [Connexion](../WDLang4/1514073.md).




**`<Tâche planifiée> : (Entier)`**

Identifiant de la tâche planifiée à manipuler. Cet identifiant est fourni par les fonctions [HAjouteTâche](../WDLang4/1000017091.md) et [HListeTâche](../WDLang4/1000017092.md).

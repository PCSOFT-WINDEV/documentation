


## MongoGridFSListeFichier (Fonction)

***En anglais : MongoGridFSListFile***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Récupère la liste des fichiers MongoDB présents dans le système de fichiers MongoDB (gridFS).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
tabListeFichier est un tableau de mongoGridFSInfoFichier

tabListeFichier = MongoGridFSListeFichier(CnxMDB.BaseDeDonnées["CRM"])
POUR TOUT fic de tabListeFichier
	// Traite le fichier
	Trace(fic.Nom)
FIN
```
<a name="Exemple2"></a>

```wl
TableSupprimeTout(TABLE_GridFS)

tabFichier est un tableau de mongoGridFSInfoFichier = MongoGridFSListeFichier(gclGridFS)
fic est un mongoGridFSInfoFichier
POUR TOUT fic DE tabFichier
	TableAjouteLigne(TABLE_GridFS, VariantVersJSON(fic.ID), ...
	fic.Nom, fic.Date, fic.Md5, fic.Taille, fic.TailleChunk, fic.TypeContenu)
FIN

TableSelectMoins(TABLE_GridFS)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = MongoGridFSListeFichier(<Système de fichiers>)`
---

**`<Résultat> : (Tableau de mongoGridFSInfoFichier)`**

Tableau de variables de type [mongoGridFSInfoFichier](../WDLang4/1000022417.md), correspondant à la description des fichiers listés. 

**`<Système de fichiers> : (Variable de type mongoGridFS ou de type mongoBase)`**

Système de fichiers de la base MongoDB à manipuler. Correspond : 

- au nom de la variable de type [mongoGridFS](../WDLang4/1000022416.md) correspondant au système de fichiers Mongo.

- au nom de la variable de type [mongoBase](../WDLang4/1000022410.md) correspondant à la base MongoDB. Dans ce cas, le système de fichiers utilise le préfixe "fs".

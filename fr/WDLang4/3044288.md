


## HListeBaseDeDonnées (Fonction)

***En anglais : HListDatabase***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Liste les bases de données Client/Serveur associées à une connexion.



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Description de la connexion
HDécritConnexion("MaConnexion", "Julie", "MotDePasse", ...
		"MonServeurHF", "", hAccèsHFClientServeur, HOLecture)
// Ouverture de la connexion
HOuvreConnexion("MaConnexion")
...
// Ajoute dans un champ de saisie les bases de données de la connexion "MaConnexion"
SAI_Saisie1 = HListeBaseDeDonnées("MaConnexion")
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = HListeBaseDeDonnées(<Connexion> [, <Options>])`
---

**`<Résultat> : (Chaîne de caractères)`**

Liste des bases de données du serveur défini par la connexion. En fonction de la constante utilisée, le résultat est plus ou moins détaillé :


|   |   |
| --- | --- |
| *hLstNormal* | Liste les bases de données du serveur défini par la connexion :<br>&lt;Nom de la base de données 1&gt; + TAB + &lt;Nom de la base de données 2&gt; + TAB + ... |
| *hLstFichier* | Liste les bases de données et leurs fichiers pour le serveur défini par la connexion :<br>&lt;Nom de la base de données 1&gt; + TAB + &lt;Nom du fichier de données 1 dans la base de données 1&gt; + TAB + ... + RC + &lt;Nom de la base de données 2&gt; + TAB + &lt;Nom du fichier de données 1 dans la base de données 2&gt; + TAB + ... + RC + ... |
| *hLstTailleBaseDeDonnées* | Liste les bases de données et leur taille (en octets) pour le serveur défini par la connexion :<br>&lt;Nom de la base de données 1&gt; + TAB + &lt;Taille de la base de données 1&gt; + RC + &lt;Nom de la base de données 2&gt; + TAB + &lt;Taille de la base de données 2&gt; + RC + ...<br> |


Où :

- &lt;Nom de la base de données X&gt; : Nom de la base de données.

- &lt;Nom du fichier de données Y dans la base de données X&gt; : Nom du fichier de données physique Y présent dans la base de données X.

- &lt;Taille de la base de données X&gt; : Taille de l'ensemble des fichiers constituant la base de données en octets. 




En cas d'erreur, la liste des bases de données correspond à une chaîne vide ("") et la fonction [HErreur](../WDLang4/3044088.md) permet d'identifier l'erreur.

**`<Connexion> : (Chaîne de caractères ou variable de type Connexion)`**

Connexion à manipuler. Cette connexion correspond : 

- soit au nom de la connexion définie sous l'éditeur d'analyses ou grâce aux fonctions [HDécritConnexion](../WDLang4/3044205.md) ou [HOuvreConnexion](../WDLang4/3044107.md).

- soit au nom d'une variable de type [Connexion](../WDLang4/1514073.md). 




**`<Options> : (Constante ou combinaison de constantes de type entier (optionnel))`**

Type de liste à effectuer :


|   |   |
| --- | --- |
| *hLstFichier* | Liste détaillée, permettant de connaître les fichiers associés à chaque base de données. |
| *hLstNormal*<br>(valeur par défaut) | Liste non détaillée, contenant uniquement le nom des bases de données associées à une connexion. |
| *hLstTailleBaseDeDonnées* | Liste détaillée, permettant de connaître la taille de l'ensemble des fichiers associés à chaque base de données.<br><br>![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Non disponible. |

### Fonction HListeBaseDeDonnées et le Connecteur Natif MySQL
<a name="fonction_docparampagetitleshort_connecteur_natif_mysql_ELTPARAGRAPHE000168"></a>

- La fonction [HListeFichier](../WDLang4/3044045.md) renvoie uniquement la liste des fichiers de la base de données en cours. 

- La fonction **HListeBaseDeDonnées** liste les bases de données MySQL accessibles et leurs fichiers.

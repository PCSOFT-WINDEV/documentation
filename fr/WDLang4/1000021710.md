
## hbEcriture (Type de variable)

***En anglais : hbWriting***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **hbEcriture** permet de définir toutes les caractéristiques d'une écriture à réaliser dans une ligne (enregistrement) d'une table HBase. Les caractéristiques de cette écriture peuvent être définies et modifiées à l'aide de différentes propriétés WLangage. 

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connexion
Connexion est une hbConnexion
Connexion.Serveur = "MonServeur"
// Description des données à écrire
Ecriture est une hbEcriture
Ecriture.Ligne = "post1"
Cellule est une hbCellule 
Cellule.Colonne.Famille = "post"
Cellule.Colonne.Qualificatif = "titre"
Cellule.Valeur = "Ceci est mon message sur le blog."
Ajoute(Ecriture.Cellule, Cellule)
// Ecriture des données
hbEcrit(Connexion, "blog", Ecriture)
```

## Remarques
<a name="NOTE0_1"></a>


### Propriétés spécifiques à la description des variables de type hbEcriture
<a name="proprietes_specifiques_description_des_variables_type_hbecriture_ELTPARAGRAPHE000029"></a>Les propriétés suivantes peuvent être utilisées pour pour décrire l'écriture à réaliser :

| Nom de la propriété | Type manipulé | Effet |
| --- | --- | --- |
| Cellule | Tableau de [hbCellule](../WDLang4/1000021707.md) | Description des données à écrire. |
| Ligne | Buffer | Identifiant de la ligne. |

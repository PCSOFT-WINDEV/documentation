


## &lt;Variable Connexion&gt;.VérifieMiseAJourServeur (Fonction)

***En anglais : &lt;Connection variable&gt;.CheckServerUpdate***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Vérifie s'il existe une version plus récente du serveur HFSQL pouvant être mise à jour automatiquement.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MaConnexion est une Connexion

soit sNouvelleVersion = MaConnexion.VérifieMiseAJourServeur()
SI  sNouvelleVersion<>"" ALORS Info(sNouvelleVersion)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Connexion>.VérifieMiseAJourServeur()`
---

**`<Résultat> : (Chaîne de caractères)`**

Version de la mise à jour disponible sur le serveur HFSQL.

**`<Connexion> : (Variable de type Connexion)`**

Nom de la variable de type [Connexion](../WDLang4/1514073.md) décrivant la connexion au serveur HFSQL.




## Remarques
Le serveur de mise à jour (contenant le pack de mise à jour du serveur HFSQL) peut être défini par la fonction [&lt;Variable Connexion&gt;.GèreMiseAJourServeur](../WDLang4/1000022662.md).

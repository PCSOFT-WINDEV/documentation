


## &lt;Variable mongoGridFS&gt;.RécupèreInfoFichier (Fonction)

***En anglais : &lt;mongoGridFS variable&gt;.GetFileInfo***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de récupérer les informations concernant un fichier MongoDB (gridFS).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
cnxMDB est une mongoConnexion("srv-mongodb://db1.exemple.net")
gclGridFS est un mongoGridFS
gclGridFS.BaseDeDonnées = cnxMDB.BaseDeDonnées["CRM"]
Fi est mongoGridFSInfoFichier
Fi = gclGridFS.RécupèreInfoFichier("produit.png")
Trace(Fi.Taille)
Trace(Fi.TypeContenu)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Système de fichiers>.RécupèreInfoFichier(<Nom du fichier>)`
---

**`<Résultat> : (Variable de type mongoGridFSInfoFichier)`**

Variable de type [MongoGridFSInfoFichier](../WDLang4/1000022417.md) contenant les information sur le fichier.

**`<Système de fichiers> : (Variable de type mongoGridFS)`**

Nom de la variable de type [mongoGridFS](../WDLang4/1000022416.md) correspondant au système de fichiers Mongo à manipuler. 

**`<Nom du fichier> : (Chaîne de caractères)`**

Nom du fichier présent dans le système de fichiers MongoDB dont les informations doivent être récupérées.

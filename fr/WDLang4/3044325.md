


## HInfoPropriétéServeur (Fonction)

***En anglais : HInfoServerProperty***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de connaître les propriétés d'un serveur HFSQL.



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Description de la connexion
HDécritConnexion("MaConnexion", "Julie", "MotDePasse", "MonServeurHF", ...
	"MaBaseDeDonnées", hAccèsHFClientServeur, HOLecture)
// Ouverture de la connexion
HOuvreConnexion("MaConnexion")
...
NotesServeur est une chaîne
NotesServeur = HInfoPropriétéServeur("MaConnexion", hNotes)
SI NotesServeur <> "" ALORS 
	Trace("Description du serveur : " + NotesServeur)
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = HInfoPropriétéServeur(<Connexion> , <Propriété>)`
---

**`<Résultat> : (Chaîne de caractères)`**



- Valeur de la propriété demandée, 

- Chaîne vide ("") en cas d'erreur.




**`<Connexion> : (Chaîne de caractères ou variable de type Connexion)`**

Connexion pour laquelle la valeur d'une propriété est recherchée. Cette connexion correspond : 

- soit au nom de la connexion définie sous l'éditeur d'analyses ou grâce aux fonctions [HDécritConnexion](../WDLang4/3044205.md) ou [HOuvreConnexion](../WDLang4/3044107.md).

- soit au nom d'une variable de type [Connexion](../WDLang4/1514073.md).




**`<Propriété> : (Constante de type Entier)`**

Propriété du serveur recherchée : 


|   |   |
| --- | --- |
| *hNotes* | Description du serveur ou toute autre information associée au serveur grâce à la fonction [HModifiePropriétéServeur](../WDLang4/3044331.md). <br>Si cette constante est utilisée, &lt;Résultat&gt; est une chaîne de caractères. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Pour modifier les propriétés d'un serveur, utilisez la fonction [HModifiePropriétéServeur](../WDLang4/3044331.md).

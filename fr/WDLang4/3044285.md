


## HInfoUtilisateur (Fonction)

***En anglais : HInfoUser***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Met à jour [les variables de gestion des utilisateurs](../WDLang4/3044315.md) avec les informations concernant l'utilisateur spécifié. Pour des raisons de sécurité, le mot de passe de l'utilisateur n'est pas récupéré.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Description de la connexion
HDécritConnexion("MaConnexion", "Julie", "MotDePasse", "MonServeurHF", "MaBaseDeDonnées", ...
	hAccèsHFClientServeur, HOLecture)
// Ouverture de la connexion
HOuvreConnexion("MaConnexion")
// Association du fichier de données à la connexion
HChangeConnexion(Client, "MaConnexion")
SI HInfoUtilisateur("MaConnexion", "Florence") = Vrai ALORS
	Info("Voici les caractéristiques de l'utilisateur" + RC + ...
		"Login : " + HUtilisateur.Login + ...
		"Nom Complet : " + HUtilisateur.NomComplet + ...
		"Email : " + HUtilisateur.Email)
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = HInfoUtilisateur(<Connexion> , <Utilisateur>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la structure de gestion des utilisateurs a été mise à jour, 

- <u><u><u><u>Faux</u></u></u></u> en cas d'erreur. La fonction [HErreurInfo](../WDLang4/3044071.md) permet d'identifier l'erreur.




Remarque : Une erreur "Utilisateur inconnu" sera renvoyée par la fonction [HErreurInfo](../WDLang4/3044071.md) si l'utilisateur en cours n'a pas le droit de visualisation des utilisateurs (constante *hDroitVisualisationUtilisateur*). Ce droit peut être défini par la fonction [HModifieDroitServeur](../WDLang4/3044328.md).

**`<Connexion> : (Chaîne de caractères ou variable de type Connexion)`**

Connexion pour laquelle les renseignements concernant l'utilisateur doivent être récupérés. Cette connexion correspond : 

- soit au nom de la connexion définie sous l'éditeur d'analyses ou grâce aux fonctions [HDécritConnexion](../WDLang4/3044205.md) ou [HOuvreConnexion](../WDLang4/3044107.md).

- soit au nom d'une variable de type [Connexion](../WDLang4/1514073.md).




**`<Utilisateur> : (Chaîne de caractères)`**

Nom de l'utilisateur pour lequel les informations doivent être récupérées. Ce nom doit correspondre à la variable **HUtilisateur.Login** de l'un des utilisateurs de la connexion spécifiée.

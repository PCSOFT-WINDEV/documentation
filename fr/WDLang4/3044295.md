


## HSupprimeGroupe (Fonction)

***En anglais : HDeleteGroup***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime un groupe d'utilisateurs associé à une connexion, sur le serveur.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Description de la connexion
HDécritConnexion("MaConnexion", "Julie", "MotDePasse", "MonServeurHF", ...
		"MaBaseDeDonnées", hAccèsHFClientServeur, HOLecture)
// Ouverture de la connexion
HOuvreConnexion("MaConnexion")
// Association du fichier de données à la connexion
HChangeConnexion(Client, "MaConnexion")
...
SI HSupprimeGroupe("MaConnexion", "Util1") = Vrai ALORS
	Info("Groupe Util1 supprimé")
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = HSupprimeGroupe(<Connexion> , <Groupe>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le groupe a été supprimé, 

- <u><u><u><u>Faux</u></u></u></u> en cas d'erreur. La fonction [HErreurInfo](../WDLang4/3044071.md) permet d'identifier l'erreur.




Remarque : Une erreur "Groupe inconnu" sera renvoyée par la fonction [HErreurInfo](../WDLang4/3044071.md) si l'utilisateur en cours n'a pas le droit de visualisation des utilisateurs (constante *hDroitVisualisationUtilisateur*) et s'il demande la suppression d'un autre groupe.

**`<Connexion> : (Chaîne de caractères ou variable de type Connexion)`**

Connexion pour laquelle un groupe doit être supprimé. Cette connexion correspond :

- soit au nom de la connexion définie sous l'éditeur d'analyses ou grâce aux fonctions [HDécritConnexion](../WDLang4/3044205.md) ou [HOuvreConnexion](../WDLang4/3044107.md).

- soit au nom d'une variable de type [Connexion](../WDLang4/1514073.md).




**`<Groupe> : (Chaîne de caractères)`**

Nom du groupe à supprimer.

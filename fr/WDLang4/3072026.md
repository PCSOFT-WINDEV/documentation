
## SQLReqExiste (Fonction)

***En anglais : SQLReqExists***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Teste l'existence d'une requête.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Teste l'existence d'une requête
SI SQLReqExiste("REQ1") = 1 ALORS
	Info("La requête existe")
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = SQLReqExiste(<Nom de la requête>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la requête existe, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Nom de la requête> : (Chaîne de caractères)`**

Nom de la requête créée et exécutée avec la fonction [SQLExec](../WDLang4/3072007.md), ou exécutée avec la fonction [SQLExecWDR](../WDLang4/3072010.md).

![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Les requêtes créées avec l'éditeur de requêtes doivent être spécifiées sous forme d'une chaîne de caractères avec guillemets.

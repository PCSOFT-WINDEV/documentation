


## HMiseAJourVue (Fonction)

***En anglais : HUpdateView***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Crée ou met à jour la description d'une vue SQL ou d'une vue matérialisée depuis l'analyse vers un serveur HFSQL.

Remarque : La mise à jour des données de la vue est réalisée grâce à la fonction [HRafraîchitVue](../WDLang4/1000021376.md). 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Met à jour la description de la vue sur le serveur HFSQL
HMiseAJourVue(MaConnexion, MaVue)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = HMiseAJourVue(<Connexion> , <Vue> [, <Forcer>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la vue (SQL ou matérialisée) a été créée ou mise à jour. 

- <u><u><u><u>Faux</u></u></u></u> en cas de problème.  La fonction [HErreur](../WDLang4/3044088.md) permet d'identifier l'erreur.




**`<Connexion> : (Chaîne de caractères ou variable de type Connexion)`**

Connexion vers le serveur sur lequel la vue (SQL ou matérialisée) doit être créée ou mise à jour. Cette connexion correspond : 

- soit au nom de la connexion définie sous l'éditeur d'analyses ou grâce aux fonctions [HDécritConnexion](../WDLang4/3044205.md) ou [HOuvreConnexion](../WDLang4/3044107.md).

- soit au nom d'une variable de type [Connexion](../WDLang4/1514073.md).




**`<Vue> : (Chaîne de caractères)`**

Nom de la vue (SQL ou matérialisée) à manipuler. Cette vue doit être créée ou mise à jour sur le serveur HFSQL. 

**`<Forcer> : (Booléen optionnel)`**



- <u><u><u><u>Faux</u></u></u></u> (par défaut) : la vue sera créée ou mise à jour :

	- si le serveur ne possède pas la vue.

	- si le serveur possède une version plus ancienne de la vue.




- <u><u><u><u>Vrai</u></u></u></u> : la vue sera mise à jour dans tous les cas.




## HAnnuleRecherche (Fonction)

***En anglais : HCancelSeek***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Annule le critère de recherche en cours. Le parcours du fichier de données peut alors se poursuivre à partir de l'enregistrement en cours, sans utiliser de critère de recherche. Les enregistrements parcourus ne correspondront pas nécessairement à la condition de recherche.

**Remarque** : L'enregistrement en cours a été sélectionné grâce aux fonctions de recherche ([HLitRecherchePremier](../WDLang4/3044036.md), [HLitRechercheDernier](../WDLang4/3044034.md), ...).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Recherche des clients à partir du client "Martin"
HLitRecherchePremier(Client, Nom, "Martin")
SI HTrouve(Client) = Vrai ALORS
	// Elément trouvé : on annule la recherche
	HAnnuleRecherche(Client, Nom)
	SI ErreurDétectée = Vrai ALORS
		Erreur(HErreurInfo())
		RETOUR
	FIN
	TANTQUE HEnDehors(Client) = Faux
		 // Traitement 
		 ...
		 // Lecture du suivant sans utiliser le critère de recherche
		 HLitSuivant(Client, Nom)
	 FIN
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = HAnnuleRecherche([<Fichier de données> [, <Rubrique>]])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'opération a été réalisée, 

- <u><u><u><u>Faux</u></u></u></u> en cas de problème. La fonction [HErreur](../WDLang4/3044088.md) permet d'identifier l'erreur.




**`<Fichier de données> : (Chaîne de caractères optionnelle)`**

Nom du fichier de données (défini dans l'analyse) sur lequel le critère de recherche doit être annulé. Ce nom peut également correspondre à une vue ou à une requête. 
Si ce nom n'est pas spécifié, la fonction **HAnnuleRecherche** se réfère au dernier fichier de données utilisé.

**`<Rubrique> : (Chaîne de caractères optionnelle)`**

Nom de la rubrique du fichier de données sur laquelle le critère de recherche doit être annulé. 
Si ce nom n'est pas spécifié, la fonction **HAnnuleRecherche** se réfère à la dernière rubrique du fichier de données utilisée.

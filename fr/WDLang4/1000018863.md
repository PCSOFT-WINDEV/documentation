


## HListeOptimisationPlanifiée (Fonction)

***En anglais : HListScheduledOptimization***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Liste les tâches d'optimisation planifiées des fichiers de données HFSQL Client/Serveur pour une connexion.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MaConnexion est une Connexion
...
tabOptim est un tableau de hOptimisationPlanifiée
tabOptim = HListeOptimisationPlanifiée(MaConnexion)
POUR TOUT Optim DE tabOptim
	AfficheInfo(Optim)	
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = HListeOptimisationPlanifiée(<Connexion>)`
---

**`<Résultat> : (Tableau de variables de type hOptimisationPlanifiée)`**

Variable de type tableau de [hOptimisationPlanifiée](../WDLang4/1000018854.md). Chaque élément de ce tableau contient les caractéristiques d'une tâche d'optimisation planifiée. Le nombre d'éléments de ce tableau correspond au nombre de tâches d'optimisation planifiée disponibles.

**`<Connexion> : (Chaîne de caractères ou variable de type Connexion)`**

Connexion pour laquelle les tâches d'optimisation planifiées doivent être listées. Cette connexion correspond : 

- soit au nom de la connexion définie sous l'éditeur d'analyses ou grâce aux fonctions [HDécritConnexion](../WDLang4/3044205.md) ou [HOuvreConnexion](../WDLang4/3044107.md).

- soit au nom d'une variable de type [Connexion](../WDLang4/1514073.md).






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Droits nécessaires
<a name="droits_necessaires_ELTPARAGRAPHE000085"></a>Pour lister les tâches d'optimisation planifiées, il est nécessaire de posséder pour chaque base concernée, le droit de connexion (constante *hDroitConnexion* ou *hDroitConnexionCryptée*),

Pour une optimisation du serveur complet, l'utilisateur doit être super-utilisateur.

Pour savoir si un utilisateur ou un groupe possède ces droits, utilisez la fonction [HInfoDroitServeur](../WDLang4/3044322.md) ou [HInfoDroitDaseDeDonnées](../WDLang4/3044320.md).

Pour modifier les droits d'un utilisateur ou d'un groupe d'utilisateurs, utilisez la fonction [HModifieDroitServeur](../WDLang4/3044328.md) ou [HModifieDroitBaseDeDonnées](../WDLang4/3044326.md).

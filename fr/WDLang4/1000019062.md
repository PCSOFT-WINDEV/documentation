
## HClusterSupprimeNoeud (Fonction)

***En anglais : HClusterDeleteNode***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Désactive un noeud dans un cluster HFSQL.

Notes : 

- La fonction **HClusterSupprimeNoeud** est appelée automatiquement par le Centre de Contrôle HFSQL. Elle est réservée à un usage avancé.

- La fonction **HClusterSupprimeNoeud** ne désinstalle pas le serveur HFSQL et ne modifie pas les enregistrements de DNS.





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Désactive le noeud host3 dans le cluster dont le coordinateur est sur
// la machine clustermanager.mondomaine.com
HClusterSupprimeNoeud("clustermanager.mondomaine.com", "secret", "host3.mondomaine.com")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = HClusterSupprimeNoeud(<Coordinateur> , <Mot de passe> , <Noeud>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'ajout a réussi, 

- <u><u><u><u>Faux</u></u></u></u> sinon. En cas d'erreur, la fonction [ErreurInfo](../WDLang1/3013008.md) retourne le détail de l'erreur.




**`<Coordinateur> : (Chaîne de caractères)`**

Nom DNS de la machine où se trouve le coordinateur du cluster.

**`<Mot de passe> : (Chaîne de caractères)`**

Mot de passe permettant de se connecter au coordinateur. Ce mot de passe a été fourni lors de [l'installation du cluster](../WDLang4/1000019031.md).

**`<Noeud> : (Chaîne de caractères)`**

Nom DNS du noeud à désactiver.








## Remarques


### Fonctionnement des noeuds dans un cluster
<a name="fonctionnement_des_noeuds_dans_cluster_ELTPARAGRAPHE000184"></a>L'adresse du coordinateur du cluster est inscrite dans les fichiers de configuration de chacun des noeuds au moment de leur installation. De cette façon, au démarrage du service HFSQL Serveur, le noeud se connecte automatiquement au cluster.

Les fonctions [HClusterAjouteNoeud](../WDLang4/1000019061.md) et [HClusterSupprimeNoeud](../WDLang4/1000019062.md) peuvent ensuite être utilisées pour inclure ou exclure un noeud de la liste des noeuds actifs du cluster.

Lorsqu'un noeud est activé par la fonction [HClusterAjouteNoeud](../WDLang4/1000019061.md), ce noeud synchronise ses bases de données avec celles des autres noeuds et traite ensuite les demandes de connexion des clients.

Lorsqu'un noeud est désactivé par la fonction [HClusterSupprimeNoeud](../WDLang4/1000019062.md), ce noeud cesse de synchroniser ses bases de données et n'accepte plus de connexions de la part d'applications clientes.

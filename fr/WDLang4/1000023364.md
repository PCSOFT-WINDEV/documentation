
## MemcachedRécupèreGroupeClé (Fonction)

***En anglais : MemcachedGetKeyGroup***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Récupère les valeurs des clés stockées sur un groupe de serveurs Memcached.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
Résultats est un tableau de memcachedRésultat 
Résultats = MemcachedRécupèreGroupeClé(memConnexion, "grp", ["clé1", "clé2"])
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Récupérer la valeur d'une clé

`<Résultat> = MemcachedRécupèreGroupeClé(<Connexion> , <Groupe> , <Clé>)`
---

**`<Résultat> : (Variable de type memcachedRésultat)`**

Variable de type [memcachedRésultat](../WDLang4/1000023359.md) correspondant à la clé recherchée. Si la clé n'est pas trouvée, les propriétés **Clé** et **Valeur** de la variable [memcachedRésultat](../WDLang4/1000023359.md) seront vides. 

**`<Connexion> : (Variable de type memcachedConnexion)`**

Nom de la variable de type [memcachedConnexion](../WDLang4/1000023339.md) correspondant à la connexion au cache Memcached. 

**`<Groupe> : (Chaîne de caractères)`**

Partie de la clé permettant de choisir le serveur à utiliser.

**`<Clé> : (Chaîne de caractères)`**

Clé à rechercher.


<a name="SYNTAXE2"></a>

### Récupérer la valeur de plusieurs clés

`<Résultat> = MemcachedRécupèreGroupeClé(<Connexion> , <Groupe> , <Clés>)`
---

**`<Résultat> : (Tableau de variables de type memcachedRésultat)`**

Tableau de variables de type [memcachedRésultat](../WDLang4/1000023359.md) correspondant aux clés trouvées.

**`<Connexion> : (Variable de type memcachedConnexion)`**

Nom de la variable de type [memcachedConnexion](../WDLang4/1000023339.md) correspondant à la connexion au cache Memcached. 

**`<Groupe> : (Chaîne de caractères)`**

Partie de la clé permettant de choisir le serveur à utiliser.

**`<Clés> : (Tableau de chaînes de caractères)`**

Nom de la variable de type Tableau contenant les clés à rechercher.

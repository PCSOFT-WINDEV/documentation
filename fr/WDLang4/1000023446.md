


## HRSAttendTraitementDonnées (Fonction)

***En anglais : HRSWaitForDataProcess***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet d'attendre que : 

- les données à répliquer sur le serveur maître soient envoyées sur le serveur abonné.

- les données à répliquer reçues sur le serveur abonné soient entièrement appliquées.



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
cnx est une Connexion

HRSAttendTraitementDonnées(cnx, idRpl, 10000)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Attendre que les données à répliquer sur le serveur maître soient envoyées sur le serveur abonné

`<Résultat> = HRSAttendTraitementDonnées(<Connexion> , <Réplication> , <Timeout>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si les données sont envoyées, 

- <u><u><u><u>Faux</u></u></u></u> en cas d'erreur ou si le délai d'attente est écoulé. 




**`<Connexion> : (Chaîne de caractères ou variable de type Connexion)`**

Connexion vers le serveur maître HFSQL. Cette connexion correspond : 

- soit au nom de la connexion définie sous l'éditeur d'analyses ou grâce aux fonctions [HDécritConnexion](../WDLang4/3044205.md) ou [HOuvreConnexion](../WDLang4/3044107.md).

- soit au nom d'une variable de type [Connexion](../WDLang4/1514073.md).




**`<Réplication> : (Entier)`**

Identifiant de la réplication. Cet identifiant correspond à la propriété **Identifiant** de la variable de type [hRSConfig](../WDLang4/1000020484.md) décrivant la réplication. 

**`<Timeout> : (Entier)`**

Temps maximal en millisecondes d'attente. Ce paramètre peut correspondre à : 

- un entier correspondant au nombre de millisecondes, 

- une variable de type Durée, 

- l'indication directe de la durée (par exemple '10 ms'). 





<a name="SYNTAXE2"></a>

### Attendre que les données à répliquer reçues sur le serveur abonné soient entièrement appliquées

`<Résultat> = HRSAttendTraitementDonnées(<Connexion> , <Serveur Maître> , <Réplication> , <Timeout>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si les données sont appliquées, 

- <u><u><u><u>Faux</u></u></u></u> en cas d'erreur ou si le délai d'attente est écoulé. 




**`<Connexion> : (Chaîne de caractères ou variable de type Connexion)`**

Connexion vers le serveur abonné HFSQL. Cette connexion correspond : 

- soit au nom de la connexion définie sous l'éditeur d'analyses ou grâce aux fonctions [HDécritConnexion](../WDLang4/3044205.md) ou [HOuvreConnexion](../WDLang4/3044107.md).

- soit au nom d'une variable de type [Connexion](../WDLang4/1514073.md).




**`<Serveur Maître> : (Entier)`**

Identifiant du serveur maître. 

**`<Réplication> : (Entier)`**

Identifiant de la réplication. Cet identifiant correspond à la propriété **Identifiant** de la variable de type [hRSConfig](../WDLang4/1000020484.md) décrivant la réplication. 

**`<Timeout> : (Entier)`**

Temps maximal en millisecondes d'attente. Ce paramètre peut correspondre à : 

- un entier correspondant au nombre de millisecondes, 

- une variable de type Durée, 

- l'indication directe de la durée (par exemple '10 ms').




## HFTListeMot (Fonction)

***En anglais : HFTListWord***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Liste les mots de l'enregistrement courant qui seront pris en compte par l'indexation full-text.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
tabToken est un tableau de chaîne = HFTListeMot(Document, FTIndexTitreContenu)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = HFTListeMot(<Fichier de données> , <Index full-text>)`
---

**`<Résultat> : (Tableau de chaînes)`**

Variable de type Tableau contenant les mots pris en compte par l'indexation full-text.

**`<Fichier de données> : (Chaîne de caractères)`**

Nom du fichier de données HFSQL manipulé.

**`<Index full-text> : (Chaîne de caractères)`**

Nom de l'index full-text du fichier.

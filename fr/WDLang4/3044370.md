
## LINUX : Accès aux bases de données ODBC
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000091"></a>
L'accès natif à HFSQL Classic et Client/Serveur est disponible pour les applications WINDEV s'exécutant sous Linux (applications "console") et les sites WEBDEV hébergés sous Linux. 

L'accès Natif à MySQL est également disponible en Linux.

L'accès à toutes les bases de données disposant d'un driver ODBC Linux est également disponible : Oracle, DB2, Postgresql, Interbase, Ingres, SQL Server, Informix, ...



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000121"></a>


### Manipuler des bases de données via un driver ODBC Linux
<a name="manipuler_des_bases_donnees_via_driver_odbc_linux_ELTPARAGRAPHE000024"></a>Pour manipuler des bases de données via un driver ODBC Linux : 

1. Vérifiez si la couche standard de gestion ODBC n'est pas installée sur le poste Linux. Si ce n'est pas le cas, installez cette couche de gestion ODBC.

2. Installez si nécessaire le driver ODBC correspondant à la base de données à manipuler.

3. Dans votre application/site Linux, manipulez directement les bases de données grâce aux [fonctions SQL du WLangage](../WDLang4/3072002.md).





<a name="NOTE2_2"></a>


### Installer la couche ODBC sous Debian
<a name="installer_couche_odbc_sous_debian_ELTPARAGRAPHE000039"></a>Pour installer la couche ODBC sous Debian, le code à utiliser est le suivant :


```txt
apt-get install unixodbc
```

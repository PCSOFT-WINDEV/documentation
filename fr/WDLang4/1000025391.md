


## &lt;Variable mongoGridFS&gt;.RécupèreFichier (Fonction)

***En anglais : &lt;mongoGridFS variable&gt;.GetFile***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Récupère sur le poste local un fichier identifié par son nom et présent dans un système de fichiers MongoDB.

**Remarque** : Pour récupérer en local un fichier identifié à partir de son identifiant, utilisez la fonction [&lt;Variable mongoGridFS&gt;.RécupèreFichierParID](../WDLang4/1000025392.md). 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple

<a name="Exemple2"></a>

```wl
gclGridFS est un mongoGridFS

sRep est une chaîne = fRepSélecteur(SysRep(srTéléchargements), ...
	"Sélectionnez le répertoire de téléchargement")

SI sRep<>"" ALORS
	gclGridFS.RécupèreFichier(TABLE_GridFS.COL_Nom, ...
	ComplèteRep(sRep) + TABLE_GridFS.COL_Nom)
	ToastAffiche("Récupération terminée avec succès.")
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Système de fichiers>.RécupèreFichier(<Nom du fichier à récupérer> , <Nom du fichier local>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'opération a été réalisée,

- <u><u><u><u>Faux</u></u></u></u> en cas de problème. La fonction [ErreurInfo](../WDLang1/3013008.md) permet d'identifier l'erreur.




**`<Système de fichiers> : (Variable de type mongoGridFS)`**

Nom de la variable de type [mongoGridFS](../WDLang4/1000022416.md) correspondant au système de fichiers Mongo à manipuler. 

**`<Nom du fichier à récupérer> : (Chaîne de caractères)`**

Nom du fichier présent dans le système de fichiers MongoDB à récupérer sur le poste local.

**`<Nom du fichier local> : (Chaîne de caractères)`**

Chemin et nom du fichier récupéré sur le poste local.

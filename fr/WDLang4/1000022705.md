


## &lt;Variable Connexion&gt;.MiseAJourRequête (Fonction)

***En anglais : &lt;Connection variable&gt;.UpdateQuery***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Crée ou met à jour une requête sur un serveur HFSQL. La requête (fichier .WDR) sera alors utilisable dans une procédure stockée sur le serveur.



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Description de la connexion
MaConnexion est une Connexion

// Mettre à jour ou créer la requête
MaConnexion.MiseAJourRequête(Requête1)

// Mettre à jour la requête même si le serveur possède une version plus récente de la requête
MaConnexion.MiseAJourRequête(Requête1, Vrai)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Connexion>.MiseAJourRequête(<Requête> [, <Forcer>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la requête a été créée ou mise à jour, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. La fonction [HErreur](../WDLang4/3044088.md) permet d'identifier l'erreur.




**`<Connexion> : (Variable de type Connexion)`**

Nom de la variable de type [Connexion](../WDLang4/1514073.md) décrivant la connexion vers le serveur sur lequel la requête doit être créée ou mise à jour.

**`<Requête> : (Chaîne de caractères)`**

Nom de la requête à créer ou à mettre à jour sur le serveur. 
Si ce paramètre correspond à "\*", toutes les requêtes seront mises à jour ou créées sur le serveur.

**`<Forcer> : (Booléen optionnel)`**



- <u><u><u><u>Faux</u></u></u></u> (par défaut) : la requête sera créée ou mise à jour :

	- si le serveur ne possède pas la requête.

	- si le serveur possède une version plus ancienne de la requête.




- <u><u><u><u>Vrai</u></u></u></u> : la requête sera mise à jour dans tous les cas.

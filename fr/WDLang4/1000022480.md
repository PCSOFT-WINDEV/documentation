


## HOpérationEncours (Fonction)

***En anglais : HOperationInProgress***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la liste des opérations automatiques en cours sur un serveur HFSQL.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Liste de toutes les opérations en cours
tabOp est un tableau dynamique de hOpérationServeur 
tabOp = HOpérationEncours(cnx)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = HOpérationEncours(<Connexion>)`
---

**`<Résultat> : (Tableau de hOpérationServeur)`**

Tableau de variables de type [hOpérationServeur](../WDLang4/1000022471.md) contenant la description de toutes les opérations en cours sur le serveur HFSQL.

**`<Connexion> : (Chaîne de caractères ou variable de type Connexion)`**

Connexion à une base de données. Cette connexion correspond : 

- soit au nom de la connexion définie sous l'éditeur d'analyses ou grâce aux fonctions [HDécritConnexion](../WDLang4/3044205.md) ou [HOuvreConnexion](../WDLang4/3044107.md).

- soit au nom d'une variable de type [Connexion](../WDLang4/1514073.md).




## HExécuteProcédure (Fonction)

***En anglais : HExecuteProcedure***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Exécute une procédure stockée.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Exécution de la procédure "VenteParAnnée", avec comme paramètre "2019"
HExécuteProcédure("ConnexionHF", "VenteParAnnée", 2019)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = HExécuteProcédure(<Connexion> , <Procédure stockée> [, <Paramètre 1> [... [, <Paramètre N>]]])`
---

**`<Résultat> : (Type correspondant au résultat)`**



- Résultat de la procédure stockée : enregistrement, tableau d'enregistrements, tableau de types simples, ...

- Chaîne vide ("") si la procédure &lt;Procédure stockée&gt; ne retourne pas de résultat.




**`<Connexion> : (Chaîne de caractères ou variable de type Connexion)`**

Connexion vers le serveur qui exécute la procédure stockée. Cette connexion correspond : 

- soit au nom de la connexion définie sous l'éditeur d'analyses ou grâce aux fonctions [HDécritConnexion](../WDLang4/3044205.md) ou [HOuvreConnexion](../WDLang4/3044107.md).

- soit au nom d'une variable de type [Connexion](../WDLang4/1514073.md).




**`<Procédure stockée> : (Procédure WLangage)`**

Nom de la procédure stockée à exécuter.

**`<Paramètre 1> : (Variant optionnel)`**

Premier paramètre à passer à la procédure &lt;Procédure stockée&gt;.



**`<Paramètre N> : (Variant optionnel)`**

Nième paramètre à passer à la procédure &lt;Procédure stockée&gt;.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Gestion automatique des erreurs
<a name="gestion_automatique_des_erreurs_ELTPARAGRAPHE000116"></a>La gestion automatique des erreurs est désactivée dans les procédures stockées. Il est cependant possible d'utiliser la fonction [HSurErreur](../WDLang4/3044017.md) pour exécuter une procédure spécifique (procédure stockée).
<a name="NOTE0_2"></a>


### Passage de paramètres
<a name="passage_parametres_ELTPARAGRAPHE000128"></a>Les paramètres passés à la procédure stockée sont passés "par référence" : si ces paramètres sont modifiés dans la procédure stockée, ils seront également modifiés dans l'application qui a exécuté cette procédure stockée.
<a name="NOTE0_3"></a>


### Exécution d'une procédure stockée depuis un code SQL
<a name="execution_une_procedure_stockee_depuis_code_sql_ELTPARAGRAPHE000137"></a>Une procédure stockée peut être appelée : 

- depuis du code SQL, par exemple avec les fonctions [HExécuteRequêteSQL](../WDLang4/3044084.md) ou [SQLExec](../WDLang4/3072007.md), 

- depuis une application tierce utilisant le [driver ODBC pour HFSQL](../WDLang4/9000160.md) ou le [provider OLE DB pour HFSQL](../WDLang4/9000059.md). 




Pour plus de détails, consultez [Procédures et requêtes stockées (HFSQL Client/Serveur)](../WDLang4/3044360.md).

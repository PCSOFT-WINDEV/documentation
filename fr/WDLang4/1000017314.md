


## HChargeParamètre (Fonction)

***En anglais : HLoadParameter***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Lit un paramètre précédemment sauvegardé à partir d'une procédure stockée grâce à la fonction [HSauveParamètre](../WDLang4/1000017313.md).






<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
HChargeParamètre("MonParamètre")
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Chargement d'un paramètre sur le serveur (dans le code de la procédure stockée)

`<Résultat> = HChargeParamètre(<Paramètre> [, <Valeur par défaut>])`
---

**`<Résultat> : (Variant)`**

Valeur du paramètre si celui-ci existe, &lt;Valeur par défaut&gt; si le paramètre n'existe pas.

**`<Paramètre> : (Chaîne de caractères)`**

Nom du paramètre à restaurer. Ce paramètre ne doit pas correspondre à une chaîne vide et ne doit pas contenir le caractère "RC" (Retour Chariot).

**`<Valeur par défaut> : (Variant optionnel)`**

Valeur par défaut renvoyée si le paramètre n'est pas trouvée. Si ce paramètre n'est pas précisé, la valeur par défaut correspond à une chaîne vide ("").  


<a name="SYNTAXE2"></a>

### Chargement d'un paramètre sur le serveur (depuis le poste Client)

`<Résultat> = HChargeParamètre(<Connexion> , <Paramètre> , <Valeur par défaut>)`
---

**`<Résultat> : (Variant)`**

Valeur du paramètre si celui-ci existe, &lt;Valeur par défaut&gt; si le paramètre n'existe pas.

**`<Connexion> : (Chaîne de caractères ou variable de type Connexion)`**

Connexion à manipuler. Cette connexion correspond : 

- soit au nom de la connexion définie sous l'éditeur d'analyses ou grâce aux fonctions [HDécritConnexion](../WDLang4/3044205.md) ou [HOuvreConnexion](../WDLang4/3044107.md).

- soit au nom d'une variable de type [Connexion](../WDLang4/1514073.md).




**`<Paramètre> : (Chaîne de caractères)`**

Nom du paramètre à restaurer. Ce paramètre ne doit pas correspondre à une chaîne vide et ne doit pas contenir le caractère "RC" (Retour Chariot).

**`<Valeur par défaut> : (Variant)`**

Valeur par défaut renvoyée si le paramètre n'est pas trouvée. Si ce paramètre n'est pas précisé, la valeur par défaut correspond à une chaîne vide ("").

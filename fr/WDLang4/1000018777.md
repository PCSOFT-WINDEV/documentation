
## HListeServeur (Fonction)

***En anglais : HListServer***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Liste les serveurs HFSQL installés sur une machine.

**Attention** : Cette fonction nécessite l'accès au service MantaManager (port 4999).




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Remplit la liste LISTE_Serveur
ListeAjoute(LISTE_Serveur, HListeServeur("postefp"))
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = HListeServeur(<Nom du poste> [, <TimeOut>])`
---

**`<Résultat> : (Chaîne de caractères)`**

Liste des serveurs HFSQL présents sur le poste spécifié. Cette liste est de la forme :


```txt
<Nom du serveur 1> + TAB + <Port du serveur 1> + RC + ... 
+ RC + <Nom du serveur N> + TAB + <Port du serveur N>
```


**`<Nom du poste> : (Chaîne de caractères)`**

Nom du poste sur lequel les serveurs HFSQL sont recherchés.
Ce paramètre peut correspondre : 

- au nom du poste,

- à une adresse IP (Ipv4 ou Ipv6).




**`<TimeOut> : (Entier optionnel)`**

Temps maximal en millisecondes d'attente d'une réponse du service MantaManager. Ce paramètre peut correspondre à : 

- un entier correspondant au nombre de millisecondes, 

- une variable de type Durée, 

- l'indication directe de la durée (par exemple 10 ms). 


Par défaut, ce paramètre correspond à 5000 ms.

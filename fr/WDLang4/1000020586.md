
## Constantes utilisées par la réplication entre serveurs HFSQL
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


| Constante | Description | Utilisée par la fonction |
| --- | --- | --- |
| **hrcmPlusPrioritaire** | Méthode de résolution des conflits : Lors d'un conflit, seule la valeur de priorité définie pour le serveur est prise en compte. Plus la valeur de priorité définie pour le serveur est petite, plus le serveur sera prioritaire. | Variable de type [hRSConfig](../WDLang4/1000020484.md) |
| **hrcmPlusRécent** | Méthode de résolution des conflits : Lors d'un conflit, l'élément le plus récent est prioritaire.<br>Attention : Cette méthode nécessite une synchronisation de l'horloge des différents serveurs. | Variable de type [hRSConfig](../WDLang4/1000020484.md) |
| **hrcmProcédureAvecRubriquesModifiées** | Méthode de résolution des conflits : Lors d'un conflit, une procédure stockée est lancée (définie par la propriété **ProcédureRésolutionConflitModification**). Dans cette procédure, seules les rubriques modifiées seront accessibles. | Variable de type [hRSConfig](../WDLang4/1000020484.md) |
| **hrcmProcédureToutesRubriques** | Méthode de résolution des conflits : Lors d'un conflit, une procédure stockée est lancée (définie par la propriété **ProcédureRésolutionConflitModification**). Dans cette procédure, toutes les rubriques de l'enregistrement seront accessibles. | Variable de type [hRSConfig](../WDLang4/1000020484.md) |
| **hrsAbonné** | <br><br>- Rôle du serveur dans la réplication : Poste abonné dans la réplication entre serveurs HFSQL.<br><br>- Force la suppression d'une réplication orpheline côté abonné.<br><br><br> | <br><br>- [HRSInfo](../WDLang4/1000020574.md), [HRSInit](../WDLang4/1000020541.md), [HRSListeConfig](../WDLang4/1000020497.md)<br><br>- [HRSSupprimeConfig](../WDLang4/1000020493.md)<br><br><br> |
| **hrsDéfaut** | Supprime la réplication uniquement si le serveur abonné est accessible. | [HRSSupprimeConfig](../WDLang4/1000020493.md) |
| **hrsIgnoreAbonné** | Force la suppression même si l'autre serveur ne peut être contacté. | [HRSSupprimeConfig](../WDLang4/1000020493.md) |
| **hrsInactif** | Rôle du serveur dans la réplication : Poste inactif dans la réplication entre serveurs HFSQL. | [HRSInfo](../WDLang4/1000020574.md), [HRSInit](../WDLang4/1000020541.md) |
| **hrsMaître** | Rôle du serveur dans la réplication : Poste maître dans la réplication entre serveurs HFSQL. | [HRSInfo](../WDLang4/1000020574.md), [HRSInit](../WDLang4/1000020541.md), [HRSListeConfig](../WDLang4/1000020497.md) |




## HModifieSauvegardePlanifiée (Fonction)

***En anglais : HModifyScheduledBackup***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie une planification de sauvegarde. Cette planification a été décrite grâce à une variable de type [hDescriptionSauvegarde](../WDLang4/1000017455.md). 



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Description de la connexion
MaConnexionBase est une Connexion

tabSvg est un tableau dynamique de hDescriptionSauvegarde
tabSvg = HListeSauvegardePlanifiée(MaConnexionBase)
hSvg est une hDescriptionSauvegarde

POUR TOUT hSvg DE tabSvg
	SI hSvg.LimiteNombreSauvegarde = 3 ALORS 
		hSvg.LimiteNombreSauvegarde = 5
		hModifieSauvegardePlanifiée(MaConnexionBase, hSvg)
	FIN
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = HModifieSauvegardePlanifiée(<Connexion> , <Sauvegarde planifiée>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la sauvegarde planifiée a été modifiée, 

- <u><u><u><u>Faux</u></u></u></u> en cas d'erreur. 




**`<Connexion> : (Chaîne de caractères ou variable de type Connexion)`**

Connexion pour laquelle une sauvegarde planifiée doit être modifiée. Cette connexion correspond : 

- soit au nom de la connexion définie sous l'éditeur d'analyses ou grâce aux fonctions [HDécritConnexion](../WDLang4/3044205.md) ou [HOuvreConnexion](../WDLang4/3044107.md).

- soit au nom d'une variable de type [Connexion](../WDLang4/1514073.md). 




**`<Sauvegarde planifiée> : (Variable de type hDescriptionSauvegarde)`**

Nom de la variable de type [hDescriptionSauvegarde](../WDLang4/1000017455.md) à modifier.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Droits nécessaires
<a name="droits_necessaires_ELTPARAGRAPHE000096"></a>Pour avoir le droit de modifier une sauvegarde planifiée, il est nécessaire : 

- soit d'être le propriétaire de la sauvegarde planifiée. 

- soit d'être super-utilisateur.




## &lt;Variable Connexion&gt;.DésactiveTriggerServeur (Fonction)

***En anglais : &lt;Connection variable&gt;.DeactivateServerTrigger***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Désactive un trigger Serveur HFSQL Client/Serveur sur un serveur. Ce trigger est désactivé pour tous les clients de ce serveur. Ce trigger pourra ensuite être réactivé simplement grâce à la fonction [&lt;Variable Connexion&gt;.ActiveTriggerServeur](../WDLang4/1000022637.md).  

**Remarque** : Les fonctions de manipulation des triggers serveur sont des fonctions avancées. Les triggers serveur peuvent être créés directement dans l'éditeur d'analyses. Pour plus de détails, consultez [Triggers serveur](../WDLang4/3044369.md).




<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

### Désactiver un trigger serveur sur des fichiers HFSQL physiques d'un serveur

`<Résultat> = <Connexion>.DésactiveTriggerServeur(<Trigger> [, <Fichiers physiques HFSQL Client/Serveur>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u>Vrai</u></u></u> si l'opération a été réalisée, 

- <u><u><u>Faux</u></u></u> en cas de problème. La fonction [HErreur](../WDLang4/3044088.md) permet d'identifier l'erreur.




**`<Connexion> : (Variable de type Connexion)`**

Nom de la variable de type [Connexion](../WDLang4/1514073.md) décrivant la connexion au serveur HFSQL.

**`<Trigger> : (Chaîne de caractères)`**

Nom du trigger à désactiver. Ce nom a été donné grâce à la fonction [&lt;Variable Connexion&gt;.DécritTriggerServeur](../WDLang4/1000022653.md).

**`<Fichiers physiques HFSQL Client/Serveur> : (Chaîne de caractères optionnelle)`**

Nom physique de un ou plusieurs fichiers de données HFSQL Client/Serveur associés à la connexion (par exemple "CLIENT.FIC"). 
Pour indiquer plusieurs noms, séparez les noms de fichiers de données par une virgule (","). 
Si ce paramètre n'est pas précisé ou est égal à une chaîne vide (""), le trigger sera désactivé sur tous les fichiers de données de la base.

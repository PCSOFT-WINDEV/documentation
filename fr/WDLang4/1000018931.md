


## HModifieOptimisationPlanifiée (Fonction)

***En anglais : HModifyScheduledOptimization***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie une tâche d'optimisation planifiée sur le serveur HFSQL défini par la connexion. Cette tâche est présente dans une variable avancée de type [hOptimisationPlanifiée](../WDLang4/1000018854.md).



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = HModifieOptimisationPlanifiée(<Connexion> , <Tâche d'optimisation planifiée>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la tâche d'optimisation planifiée a été modifiée,

- <u><u><u><u>Faux</u></u></u></u> en cas d'erreur.




**`<Connexion> : (Chaîne de caractères ou variable de type Connexion)`**

Connexion pour laquelle une tâche d'optimisation planifiée doit être modifiée. Cette connexion correspond : 

- soit au nom de la connexion définie sous l'éditeur d'analyses ou grâce aux fonctions [HDécritConnexion](../WDLang4/3044205.md) ou [HOuvreConnexion](../WDLang4/3044107.md).

- soit au nom d'une variable de type [Connexion](../WDLang4/1514073.md). 




**`<Tâche d'optimisation planifiée> : (Variable de type hOptimisationPlanifiée)`**

Nom de la variable de type [hOptimisationPlanifiée](../WDLang4/1000018854.md) manipulée. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Droits nécessaires
<a name="droits_necessaires_ELTPARAGRAPHE000084"></a>Pour modifier une tâche, il est nécessaire de posséder : 

- le droit correspondant (constante *hDroitGèreTâche*) sur le serveur, 

- le droit d'exécuter la commande liée à la tâche planifiée.


La tâche sera exécutée avec le compte de l'utilisateur qui a créé la tâche. 

Remarque : Si le serveur est éteint, la tâche planifiée ne sera pas réalisée. La tâche planifiée sera réalisée à la prochaine date disponible lorsque le serveur sera allumé.

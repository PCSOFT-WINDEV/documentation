


## &lt;Variable Connexion&gt;.InfoFichier (Fonction)

***En anglais : &lt;Connection variable&gt;.InfoFile***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie les caractéristiques d'un fichier de données HFSQL Classic ou Client/Serveur.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple

<a name="3044311_Exemple2"></a>

```wl
// Récupération de la taille d'un fichier de données en utilisant une connexion 

// Paramètres de la connexion
CnxTest est une Connexion
CnxTest.Provider = hAccèsHFClientServeur
CnxTest.Utilisateur = "admin"
CnxTest.MotDePasse = "pass_admin"
CnxTest.Serveur = "serveur:4930"
CnxTest.BaseDeDonnées = "BaseTest"

Trace("Taille totale fichier Client : " + ...
	CnxTest.InfoFichier("NomFichier.FIC", hTailleTotale))
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

### Obtenir des informations concernant un fichier de données HFSQL Client/Serveur

`<Résultat> = <Connexion>.InfoFichier(<Fichier de données> , <Information>)`
---

**`<Résultat> : (Type de l'information recherchée)`**



- Information recherchée, 

- 0 si la connexion existe mais si le fichier de données n'existe pas sur le disque.




**`<Connexion> : (Variable de type Connexion)`**

Nom de la variable de type [Connexion](../WDLang4/1514073.md) décrivant la connexion à utiliser.

**`<Fichier de données> : (Chaîne de caractères)`**

Nom physique du fichier de données à analyser.

**`<Information> : (Constante de type Entier)`**

Information voulue sur le fichier de données :


|   |   |
| --- | --- |
| *hTailleFic* | Taille en octets du fichier de données (fichier ".Fic"). |
| *hTailleFtx* | Taille en octets du fichier d'index full-text (fichier ".Ftx"). |
| *hTailleMmo* | Taille en octets du fichier mémo (fichier ".Mmo"). |
| *hTailleNdx* | Taille en octets du fichier d'index (fichier ".Ndx"). |
| *hTailleTotale* | Taille en octets totale du fichier de données (taille cumulée du fichier de données, du fichier mémo et du fichier d'index). |




## HMiseAJourLiaison (Fonction)

***En anglais : HUpdateLink***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Met à jour sur le serveur l'ensemble des liaisons impliquant un fichier de données HFSQL Client/Serveur.

Remarque : L'ensemble des fichiers de données impliqués dans les liaisons doit avoir été configuré par la fonction [HChangeRep](../WDLang4/3044168.md) si besoin.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Met à jour les liaisons du fichier de données Client
HMiseAJourLiaison(Client)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = HMiseAJourLiaison(<Fichier de données> [, <Option>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la mise à jour a été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> en cas d'erreur. 




**`<Fichier de données> : (Chaîne de caractères)`**

Nom du fichier de données permettant de connaître la liste des liaisons considérées. Ce paramètre peut également correspondre à : 

- "\*", pour indiquer que tous les fichiers de données doivent être pris en compte, 

- un nom de groupe de fichiers de données, défini dans l'analyse. 




**`<Option> : (Constante ou combinaison de constantes optionnelle)`**

Options de la modification automatique des données : 


|   |   |
| --- | --- |
| *hmsNormal* <br>(Valeur par défaut) | Fonctionnement par défaut de la modification automatique des données. |
| *hmsSansContrôle* | La modification automatique des données est effectuée même si :<br><br>- la description du fichier de données est plus récente que celle de l'analyse,<br><br>- le GUID de l'analyse et le GUID du fichier de données sont différents.<br><br><br> |


**Remarque** : Les fichiers de données manipulés ne doivent pas être ouverts (par l'application ou par une autre application).



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Les rubriques utilisées par les liaisons doivent exister.

- Tous les fichiers de l'analyse doivent exister si vous utilisez le paraméttre "\*"

- Lors de la mise à jour, les liaisons supprimées dans l'analyse sont supprimées sur le serveur HFSQL. 

- Il est possible d'utiliser la fonction [HModifieStructure](../WDLang4/1000019439.md) avant d'utiliser la fonction **HMiseAJourLiaison**.


## HClusterEtat (Fonction)

***En anglais : HClusterState***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie l'état d'un cluster HFSQL en interrogeant son coordinateur.

**Note** : Cette fonction est réservée à un usage avancé. L'administration d'un cluster HFSQL doit être faite de préférence à l'aide du Centre de Contrôle HFSQL.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI HClusterEtat("clustermanager.mondomaine.com", "secret") = 5 ALORS
	LIB_Alerte = "ATTENTION : le cluster est en attente de synchronisation !"
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = HClusterEtat(<Coordinateur> , <Mot de passe>)`
---

**`<Résultat> : (Entier)`**

Etat du cluster. Les valeurs possibles sont les suivantes :

| Valeur | Etat |
| --- | --- |
| 0 | Une erreur est survenue lors de l'appel au coordinateur.<br><br>La fonction [ErreurInfo](../WDLang1/3013008.md) peut être utilisée pour connaître le détail de l'erreur. |
| 1 | Le cluster est arrêté.<br><br>Il peut être relancé par la fonction [HClusterDémarre](../WDLang4/1000019057.md). |
| 2 | Le cluster est démarré.<br><br>Il peut être suspendu par la fonction [HClusterArrête](../WDLang4/1000019058.md). |
| 3 | Le cluster est en cours d'arrêt. |
| 4 | Le cluster est en cours de démarrage. |
| 5 | Tous les noeuds du cluster actuellement connectés au coordinateur sont en attente de synchronisation. |



**`<Coordinateur> : (Chaîne de caractères)`**

Nom DNS de la machine où se trouve le coordinateur du cluster.

**`<Mot de passe> : (Chaîne de caractères)`**

Mot de passe utilisé pour se connecter au coordinateur. Ce mot de passe a été fourni lors de [l'installation du cluster](../WDLang4/1000019031.md).

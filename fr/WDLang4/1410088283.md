


## &lt;redisConnexion&gt;.PipelineAjoute (Fonction)

***En anglais : &lt;redisConnection&gt;.PipelineAdd***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ajoute une commande Redis dans un pipeline. Il est ainsi possible d'envoyer un groupe de commandes à un serveur Redis et d'optimiser les performances d'accès au réseau.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
gRedis.PipelineAjoute("set p1 pv1")
gRedis.PipelineAjoute("set p2 pv2")
gRedis.PipelineAjoute("get p2")
gRedis.PipelineAjoute("get p1")

r1 est un redisRéponse = gRedis.PipelineRécupère()
r2 est un redisRéponse = gRedis.PipelineRécupère()
r3 est un redisRéponse = gRedis.PipelineRécupère()
r4 est un redisRéponse = gRedis.PipelineRécupère()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Connexion>.PipelineAjoute(<Commande> [, <Paramètre 1> [, <Paramètre N>]])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'ajout a été effectué, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Connexion> : (Variable de type redisConnexion)`**

Nom de la variable de type [redisConnexion](../WDLang4/1000023527.md) correspondant à la connexion au serveur Redis.

**`<Commande> : (Chaîne de caractères)`**

Texte de la commande Redis à exécuter.
Cette commande peut contenir des paramètres séparés par des espaces.

**`<Paramètre 1> : (Type du paramètre optionnel)`**

Premier paramètre optionnel de la commande. Ce paramètre permet de passer des chaînes contenant des espaces ou des valeurs binaires à la commande exécutée.
Remarque : Les chaînes sont converties en UTF8 lors de l'envoi de la commande vers le serveur Redis.
$~ ... |

**`<Paramètre N> : (Type du paramètre optionnel)`**

Nième paramètre de la commande. Ce paramètre permet de passer des chaînes contenant des espaces ou des valeurs binaires à la commande exécutée.
Remarque : Les chaînes sont converties en UTF8 lors de l'envoi de la commande vers le serveur Redis.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Fonctionnement du pipeline
<a name="fonctionnement_pipeline_ELTPARAGRAPHE000196"></a>Lorsque l'ensemble des commandes souhaitées auront été ajoutées dans le pipeline, les résultats pourront être récupérés avec la fonction [&lt;redisConnexion&gt;.PipelineRécupère](../WDLang4/1410088285.md).

Pour ne pas récupérer les résultats, il est possible d'utiliser la fonction [&lt;redisConnexion&gt;.PipelineRAZ](../WDLang4/1410088284.md).

Si la fonction [&lt;redisConnexion&gt;.PipelineRAZ](../WDLang4/1410088284.md) n'a pas été utilisée, il doit y avoir autant d'appels à la fonction [&lt;redisConnexion&gt;.PipelineRécupère](../WDLang4/1410088285.md) que d'appels à la fonction **&lt;redisConnexion&gt;.PipelineAjoute**.

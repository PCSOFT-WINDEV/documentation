


## &lt;Variable Connexion&gt;.RafraîchitVue (Fonction)

***En anglais : &lt;Connection variable&gt;.RefreshView***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Demande le recalcul du contenu d'une vue matérialisée. 

**Remarque** : Il est également possible de mettre à jour la vue matérialisée grâce à l'instruction SQL ["REFRESH MATERIALIZED VIEW"](../Editeurs/2034001.md). 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Description de la connexion
MaConnexion est une Connexion

MaConnexion.RafraîchitVue(MaVue)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Connexion>.RafraîchitVue(<Vue> [, <Option>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la vue matérialisée a été rafraîchie,

- <u><u><u><u>Faux</u></u></u></u> en cas d'erreur. La fonction [HErreurInfo](../WDLang4/3044071.md) permet d'identifier l'erreur.




**`<Connexion> : (Variable de type Connexion)`**

Nom de la variable de type [Connexion](../WDLang4/1514073.md) décrivant la connexion à manipuler.

**`<Vue> : (Chaîne de caractères)`**

Nom de la vue matérialisée dont les données doivent être recalculées. Cette vue a été créée : 

- via l'éditeur d'analyses. 

- grâce à l'instruction SQL ["CREATE MATERIALIZED VIEW"](../Editeurs/2034001.md).




**`<Option> : (Constante optionnelle de type Entier)`**

Mode de rafraîchissement de la vue matérialisée. 


|   |   |
| --- | --- |
| *hSansAttente* | Le rafraîchissement de la vue matérialisée n'est pas bloquant : il est effectué en tâche de fond. La fonction [&lt;Variable Connexion&gt;.InfoVue](../WDLang4/1000022688.md) permet de connaître le résultat du rafraîchissement de la vue matérialisée. |
| Paramètre non spécifié (par défaut) | Le rafraîchissement de la vue matérialisée est bloquant pour celui qui l'a demandé. &lt;Résultat&gt; permet de connaître le compte-rendu du rafraîchissement. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Droits nécessaires
<a name="droits_necessaires_ELTPARAGRAPHE000244"></a>Pour recalculer du contenu d'une vue matérialisée, il est nécessaire de posséder : 

- le droit correspondant (constante *hDroitGèreTâche*) sur le serveur, 

- pour chaque base concernée (c'est-à-dire qui contient une vue à traiter), le droit de connexion (constante *hDroitConnexion* ou *hDroitConnexionCryptée*). 




Pour savoir si un utilisateur ou un groupe possède ces droits, utilisez la fonction [&lt;Variable Connexion&gt;.InfoDroitServeur](../WDLang4/1000022668.md) ou [HInfoDroitDaseDeDonnées](../WDLang4/1000022666.md).

Pour modifier les droits d'un utilisateur ou d'un groupe d'utilisateurs, utilisez la fonction [&lt;Variable Connexion&gt;.ModifieDroitServeur](../WDLang4/1000022710.md) ou [&lt;Variable Connexion&gt;.ModifieDroitBaseDeDonnées](../WDLang4/1000022707.md).

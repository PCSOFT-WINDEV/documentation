


## &lt;Variable Connexion&gt;.SupprimePlanification (Fonction)

***En anglais : &lt;Connection variable&gt;.DeleteScheduling***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime une tâche planifiée sur un serveur HFSQL : 

- procédure stockée, 

- sauvegarde, 

- optimisation, 

- rafraîchissement d'une vue matérialisée.





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
cnxMaBase est une Connexion

// Suppression de toutes les sauvegardes planifiées
tabDescSvg est un tableau dynamique de hDescriptionSauvegarde 
tabDescSvg = cnxMaBase.ListePlanification(hlpSauvegarde)
hDescSvg est une hDescriptionSauvegarde
POUR TOUT hDescSvg DE tabDescSvg
	SI Position(hDescSvg.Source, cnxMaBase.BaseDeDonnées, 0, SansCasse) > 0 ALORS
		cnxMaBase.SupprimePlanification(hDescSvg)
	FIN
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Supprimer une tâche planifiée

`<Résultat> = <Connexion>.SupprimePlanification(<Tâche planifiée>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la tâche planifiée a été supprimée,

- <u><u><u><u>Faux</u></u></u></u> en cas d'erreur. La fonction [HErreurInfo](../WDLang4/3044071.md) permet d'identifier l'erreur.




**`<Connexion> : (Variable de type Connexion)`**

Nom de la variable de type [Connexion](../WDLang4/1514073.md) décrivant la connexion pour laquelle une tâche planifiée doit être supprimée.

**`<Tâche planifiée> : (Variable de type hTâchePlanifiée)`**

Nom de la variable de type [hTâchePlanifiée](../WDLang4/1000017448.md), décrivant la tâche déclenchant une procédure stockée.

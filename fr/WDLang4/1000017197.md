
## RéplicInfo (Fonction)

***En anglais : ReplicInfo***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie des informations sur une réplication. Cette réplication a été définie avec l'éditeur de réplication (ReplicEdit).






<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI RéplicInfo("MaRéplication", replicExiste) = Vrai ALORS
	RéplicSynchronise("MaRéplication")
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = RéplicInfo(<Nom de la réplication> , <Information demandée>)`
---

**`<Résultat> : (Type correspondant à l'information demandée)`**

Information demandée par le paramètre &lt;Information demandée&gt;.

**`<Nom de la réplication> : (Chaîne de caractères)`**

Nom de la réplication à manipuler. Ce nom correspond :

- soit au nom du fichier ".wer", créé avec ReplicEdit (dans le cas d'une réplication connectée). Ce fichier doit être présent à côté de l'exécutable.

- soit au nom de publication (dans le cas d'un serveur de réplication).




**`<Information demandée> : (Constante de type Entier)`**

Information demandée sur la réplication spécifiée :


|   |   |
| --- | --- |
| *replicDateDemandeSynchronisation* | &lt;Résultat&gt; correspond à la date de la dernière demande de synchronisation.<br><br>Si la réplication n'existe pas, une erreur non fatale est déclenchée. Cette erreur peut être traitée grâce à la variable [ErreurDétectée](../WDLang1/3087001.md). |
| *replicDateSynchronisation* | &lt;Résultat&gt; correspond à la date de la dernière synchronisation.<br><br>Si la réplication n'existe pas, une erreur non fatale est déclenchée. Cette erreur peut être traitée grâce à la variable [ErreurDétectée](../WDLang1/3087001.md). |
| *replicExiste* | &lt;Résultat&gt; est à :<br><br>- <u><u><u><u>Vrai</u></u></u></u> si la réplication existe (c'est-à-dire si le fichier .WER est bien présent à côté de l'exécutable).<br><br>- <u><u><u><u>Faux</u></u></u></u> si la réplication n'existe pas.<br><br><br>**Remarque** : Dans le cas d'une réplication via serveur de réplication, &lt;Résultat&gt; est à <u><u><u><u>Vrai</u></u></u></u> si les paramètres passés à la fonction [RéplicOuvre](../WDLang4/1000017194.md) sont corrects. |
| *replicInitialisée* | &lt;Résultat&gt; est à :<br><br>- <u><u><u><u>Vrai</u></u></u></u> si la réplication est initialisée,<br><br>- <u><u><u><u>Faux</u></u></u></u> si la réplication n'est pas initialisée.<br><br><br>Si la réplication n'existe pas, une erreur non fatale est déclenchée. Cette erreur peut être traitée grâce à la variable [ErreurDétectée](../WDLang1/3087001.md). |

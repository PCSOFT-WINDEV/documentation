


## &lt;Source&gt;.VérifieMémo (Fonction)

***En anglais : &lt;Source&gt;.CheckMemo***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Vérifie l'intégrité structurelle des mémos d'un fichier de données HFSQL.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI Client.VérifieMémo(JAUGE_Jauge1) <> "" ALORS
	Info("Mémo corrompu")
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Source>.VérifieMémo([<Jauge>])`
---

**`<Résultat> : (Chaîne de caractères)`**



- Chaîne vide ("") si il n'y a aucun problème. 

- En cas de problème, chaîne de la forme : 
	
	```txt
	<Rubrique 1> + TAB + <Numéro d'enregistrement 1> + RC + ...
	<Rubrique N> + TAB + <Numéro d'enregistrement N>
	```
 où : 

	- &lt;Rubrique X&gt; est la rubrique mémo, 

	- &lt;Numéro d'enregistrement&gt; est le numéro de l'enregistrement contenant la rubrique Mémo. 





![HFSQL Client/Serveur](https://doc.pcsoft.fr/ext/images/fr/HFCS.png) En cas de corruption de la liste des blocs libres, &lt;Résultat&gt; contiendra :

```txt
<TAB> + "-1"
```
&lt;Résultat&gt; sera de la forme : 

```txt
<TAB> + "-1"+ RC + <Rubrique 1> + TAB + <Numéro d'enregistrement 1> + RC + ...
<Rubrique N> + TAB + <Numéro d'enregistrement N>
```


**`<Source> : (Type correspondant à la source spécifiée)`**

Nom du fichier de données HFSQL à vérifier.

**`<Jauge> : (Nom de fenêtre ou nom de champ optionnel)`**



- Nom de la fenêtre dans laquelle la jauge doit être affichée. 

- Nom du champ de type Jauge affichant la progression.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- La vérification de la liste des blocs libres est effectuée uniquement en HFSQL Client/Serveur.

- La fonction **&lt;Source&gt;.VérifieMémo** est plus rapide que la fonction [&lt;Source&gt;.Réindexe](../WDLang4/1000025060.md) utilisée avec la constante *hVérifieMémo*. 

- Si un problème est détecté, il est possible d'utiliser la fonction [&lt;Source&gt;.Réindexe](../WDLang4/1000025060.md) avec la constante *hVérifieMémo*.

- A partir de la version 27, la vérification du mémo est améliorée et approfondie, pouvant entraîner un allongement des durées de traitement.

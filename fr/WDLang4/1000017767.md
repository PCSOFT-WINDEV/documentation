
## HPurgeRepTravail (Fonction)

***En anglais : HClearWorkingDir***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Purge et détruit le répertoire temporaire précédemment créé lors de l'exécution de la fonction [HRepTravailServeur](../WDLang4/1000017768.md). Ce répertoire est automatiquement vidé et détruit lors de la déconnexion du client.

**Remarque** : Cette fonction peut être exécutée uniquement à partir d'une procédure stockée.  


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Code d'une procédure stockée
fSauveTexte(ComplèteRep(HRepTravailServeur()) + "MonFichier.txt", MaChaîneTexte)
...
HPurgeRepTravail()
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = HPurgeRepTravail()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le répertoire a été détruit, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. La fonction [HErreurInfo](../WDLang4/3044071.md) permet d'identifier l'erreur.

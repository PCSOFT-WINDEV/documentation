
## HClusterInfoErreurSynchronisation (Fonction)

***En anglais : HClusterSynchronizationErrorInfo***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie l'erreur survenue lors de la précédente phase de synchronisation du cluster HFSQL.

Rappel : Une synchronisation est automatiquement effectuée par exemple lorsqu'un noeud du cluster s'est arrêté anormalement : au redémarrage, ce noeud se synchronise avec un autre noeud qui contient les données à jour.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
sErreurSync est une chaîne
sErreurSync = HClusterInfoErreurSynchronisation("MonCoordinateur", "MonMotDePasse", "NomNoeud")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = HClusterInfoErreurSynchronisation(<Coordinateur> , <Mot de passe> , <Noeud>)`
---

**`<Résultat> : (Chaîne de caractères)`**

Erreur survenue lors de la dernière synchronisation.

**`<Coordinateur> : (Chaîne de caractères)`**

Nom DNS de la machine où se trouve le coordinateur du cluster.

**`<Mot de passe> : (Chaîne de caractères)`**

Mot de passe permettant de se connecter au coordinateur. Ce mot de passe a été fourni lors de [l'installation du cluster](../WDLang4/1000019031.md).

**`<Noeud> : (Chaîne de caractères)`**

Nom DNS du noeud à manipuler.

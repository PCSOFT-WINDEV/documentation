
## hbDescriptionTable (Type de variable)

***En anglais : hbTableDescription***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **hbDescriptionTable** permet de définir et de connaître toutes les caractéristiques d'une table HBase. Les caractéristiques de cette table HBase peuvent être définies et modifiées à l'aide de différentes propriétés WLangage. 

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connexion
Connexion est une hbConnexion
Connexion.Serveur = "MonServeur"
// Récupération de la description de la table
DescTable est une hbDescriptionTable = hbRécupèreDescriptionTable(Connexion, "blog")
// Affichage du nom de la table
Trace("table : " + DescTable.Nom)
// Affichage de la description des colonnes
POUR TOUT DescCol DE DescTable.Colonne
	Trace(TAB + "colonne : " + DescCol.Nom)
	Trace(TAB + TAB + "attribut : ")
	POUR TOUT ValAttribut, NomAttribut DE DescCol.Attribut
		Trace(TAB + TAB + NomAttribut + " = " + ValAttribut)
	FIN
FIN
```

## Remarques
<a name="NOTE0_1"></a>


### Propriétés spécifiques à la description des variables de type hbDescriptionTable
<a name="proprietes_specifiques_description_des_variables_type_hbdescriptiontable_ELTPARAGRAPHE000029"></a>Les propriétés suivantes peuvent être utilisées pour manipuler une table HBase :

| Nom de la propriété | Type manipulé | Effet |
| --- | --- | --- |
| Attribut | Tableau de Chaîne de caractères | Tableau associatif contenant les noms des attributs et les valeurs de chaque attribut. Ces noms et valeurs sont spécifiques à HBase et réservés à une utilisation avancée.<br><br>Pour plus de détails sur les attributs de table, consultez la documentation HBase. |
| Colonne | Tableau de [hbDescriptionColonne](../WDLang4/1000021694.md) | Description des colonnes. |
| Nom | Chaîne de caractères | Nom de la table HBase. |




<a name="NOTE0_2"></a>


### Fonctions WLangage manipulant le type hbDescriptionTable
<a name="fonctions_wlangage_manipulant_type_hbdescriptiontable_ELTPARAGRAPHE000066"></a>


|   |   |
| --- | --- |
| [hbCréeTable](../WDLang4/1000021696.md) | Crée une table HBase ou modifie la description d'une table HBase. |
| [hbRécupèreDescriptionTable](../WDLang4/1000021695.md) | Récupère la description d'une table HBase. |

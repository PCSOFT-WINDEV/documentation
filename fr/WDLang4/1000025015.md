


## &lt;Source&gt;.ListeMotVide (Fonction)

***En anglais : &lt;Source&gt;.ListStopWord***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la liste des mots vides utilisés par un index full-text. 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Récupère l'ensemble des mots ignorés par l'index full-text
soit sListeMotVide = FichierDonnées.ListeMotVide(MonIndexFullText)
// Pour chaque mot
POUR TOUTE CHAÎNE sMotVide DE sListeMotVide SEPAREE PAR TAB
	// Affiche le mot vide
	Trace(sMotVide)
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Source>.ListeMotVide(<Index full-text> [, <Chemin du WDD> [, <Mot de passe du WDD>]])`
---

**`<Résultat> : (Chaîne de caractères)`**

Liste des mots vides définis pour l'index full-text spécifié dans le fichier de données spécifié. Cette liste est de la forme : 

```txt
<Mot 1> + TAB + <Mot 2> + TAB + ... + TAB + <Mot N>
```
Les différents mots vides sont séparés par une tabulation (TAB). Ces mots seront ignorés par l'index full-text. 
Remarques : 

- Les mots peuvent avoir été mis en majuscules si l'index full-text est aussi insensible à la casse ou les accents retirés si l'index full-text est insensible à l'accentuation.

- Les mots possédant une longueur plus petite que la longueur minimale spécifiée pour l'index full-text peuvent ne pas apparaître dans cette liste.




**`<Source> : (Type correspondant à la source spécifiée)`**

Nom du fichier de données contenant l'index full-text.

**`<Index full-text> : (Chaîne de caractères)`**

Nom de l'index full-text manipulé.

**`<Chemin du WDD> : (Chaîne de caractères optionnelle)`**

Chemin complet de l'analyse (c'est-à-dire du ficher .WDD) dans laquelle le fichier de données est décrit. Si ce paramètre n'est pas précisé ou est égal à une chaîne vide (""), le fichier de données est recherché dans l'analyse en cours.

**`<Mot de passe du WDD> : (Chaîne de caractères optionnelle)`**

Mot de passe défini pour l'analyse. Si aucun mot de passe n'est défini, ce paramètre doit être égal à une chaîne vide ("").

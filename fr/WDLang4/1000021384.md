
## ChampVersSource (Fonction)

***En anglais : ControlToSource***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Affecte la source de la liaison d'un champ avec la valeur du champ.

**Remarque** : Cette fonction est utile pour le développement d'une application en mode MVP. Utilisée dans un traitement de modification, cette fonction permet de modifier uniquement un membre de la présentation plutôt que toute la présentation.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Modification de SAISIE_NOM
ChampVersSource()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`ChampVersSource([<Champ manipulé>])`
---

**`<Champ manipulé> : (Nom de champ optionnel)`**

Nom du champ à manipuler. Si ce paramètre n'est pas spécifié, l'élément désigné par le mot-clé "Moimême" est utilisé. Si cet élément n'est pas un champ, la fonction n'a aucun effet. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Cette fonction est équivalente à la fonction [EcranVersSource](../WDLang4/1000017342.md) utilisée sur un seul champ au lieu de toute la fenêtre. 

- Cette fonction est équivalente au code suivant : 

	- Champ lié à une rubrique :
			
		```txt
		FichierDeDonnées.Rubrique = Champ
		```


	- Champ lié à un membre de classe : 
		```txt
		Classe.Membre = Champ
		```





- Si le champ n'a pas de liaison, la fonction n'a aucun effet.

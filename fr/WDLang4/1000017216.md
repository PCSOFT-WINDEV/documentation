
## HRplGestionRubrique (Fonction)

***En anglais : HRplManageItem***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renseigne les options de réplication pour une rubrique : la rubrique peut être répliquée ou non.  




<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`HRplGestionRubrique(<Nom du fichier> , <Nom de la rubrique> , <Mode de réplication>)`
---

**`<Nom du fichier> : (Chaîne de caractères)`**

Nom du fichier de données à paramétrer.

**`<Nom de la rubrique> : (Chaîne de caractères)`**

Nom de la rubrique du fichier à paramétrer.

**`<Mode de réplication> : (Constante de type Entier)`**

Constante indiquant le mode de réplication de la rubrique :


|   |   |
| --- | --- |
| *rplRubriqueNonRépliquée* | La rubrique n'est pas répliquée. |
| *rplRubriqueRépliquée* | La rubrique est répliquée. |

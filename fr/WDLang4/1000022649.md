


## &lt;Variable Connexion&gt;.ChargeParamètre (Fonction)

***En anglais : &lt;Connection variable&gt;.LoadParameter***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Lit un paramètre précédemment sauvegardé à partir d'une procédure stockée grâce à la fonction [&lt;Variable Connexion&gt;.SauveParamètre](../WDLang4/1000022753.md).






<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Description de la connexion
MonParamètre est une Connexion

MonParamètre.ChargeParamètre()
```

<a name="XSYNTAXE"></a>

### Chargement d'un paramètre sur le serveur (depuis le poste Client)

`<Résultat> = <Connexion>.ChargeParamètre(<Paramètre> , <Valeur par défaut>)`
---

**`<Résultat> : (Variant)`**

Valeur du paramètre si celui-ci existe, &lt;Valeur par défaut&gt; si le paramètre n'existe pas.

**`<Connexion> : (Variable de type Connexion)`**

Nom de la variable de type [Connexion](../WDLang4/1514073.md) décrivant la connexion à manipuler.

**`<Paramètre> : (Chaîne de caractères)`**

Nom du paramètre à restaurer. Ce paramètre ne doit pas correspondre à une chaîne vide et ne doit pas contenir le caractère "RC" (Retour Chariot).

**`<Valeur par défaut> : (Variant)`**

Valeur par défaut renvoyée si le paramètre n'est pas trouvée. Si ce paramètre n'est pas précisé, la valeur par défaut correspond à une chaîne vide ("").




## &lt;Variable Connexion&gt;.PrioritéClient (Fonction)

***En anglais : &lt;Connection variable&gt;.PriorityClient***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie la priorité d'une application cliente. Les applications dont la priorité doit être modifiée sont les applications clientes décrites dans la structure [HClient](../WDLang4/3044340.md).

**Remarque** : Plus la priorité de l'application cliente est haute, plus le serveur traitera les demandes de cette application en premier.



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Description de la connexion
MaConnexion est une Connexion

// Définition de la structure HClient
HClient.Application = "MonApplicationCliente"

// Changement de priorité de l'application
MaConnexion.PrioritéClient(5)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Connexion>.PrioritéClient(<Priorité>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le changement de priorité a eu lieu,

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. La fonction [HErreur](../WDLang4/3044088.md) permet d'identifier l'erreur.




**`<Connexion> : (Variable de type Connexion)`**

Nom de la variable de type [Connexion](../WDLang4/1514073.md) décrivant la connexion vers le serveur HFSQL.

**`<Priorité> : (Entier)`**

Nouvelle valeur de la priorité à affecter à l'application cliente. Cette valeur est comprise entre -10 (basse priorité) et 10 (haute priorité). 
Par défaut, au lancement de l'application, la priorité est de 0 (priorité normale).



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Pour modifier la priorité de l'application cliente spécifiée, le droit de modifier la priorité est nécessaire. Ce droit correspond à la constante *hDroitPriorité* (utilisée dans les fonctions [&lt;Variable Connexion&gt;.InfoDroitServeur](../WDLang4/1000022668.md) et [&lt;Variable Connexion&gt;.ModifieDroitServeur](../WDLang4/1000022710.md)).




## HInfoGroupe (Fonction)

***En anglais : HInfoGroup***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie des informations sur le groupe d'utilisateurs spécifié. Ces informations sont automatiquement affectées [aux variables de gestion des groupes d'utilisateurs](../WDLang4/3044315.md).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Description de la connexion
HDécritConnexion("MaConnexion", "Julie", "MotDePasse", "MonServeurHF", "MaBaseDeDonnées", ...
	hAccèsHFClientServeur, HOLecture)
// Ouverture de la connexion
HOuvreConnexion("MaConnexion")
// Association du fichier de données à la connexion
HChangeConnexion(Client, "MaConnexion")
...
// Recherche des caractéristiques d'un groupe
Res est un booléen
Res = HInfoGroupe("MaConnexion", "Groupe_Test")
SI Res = Vrai ALORS
	Trace("Structure du groupe mise à jour")
	Trace("Nom du groupe : " + HGroupe.Nom)
	Trace("Description du groupe : " + HGroupe.Description)
	Trace("Utilisateurs du groupe : " + HGroupe.Utilisateurs)
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = HInfoGroupe(<Connexion> , <Groupe>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si les informations concernant le groupe ont été lues, 

- <u><u><u><u>Faux</u></u></u></u> en cas d'erreur. La fonction [HErreurInfo](../WDLang4/3044071.md) permet d'identifier l'erreur.




Remarque : Une erreur "Groupe inconnu" sera renvoyée par la fonction [HErreurInfo](../WDLang4/3044071.md) si l'utilisateur en cours n'a pas le droit de visualisation des utilisateurs (constante *hDroitVisualisationUtilisateur*). Ce droit peut être défini par la fonction [HModifieDroitServeur](../WDLang4/3044328.md).

**`<Connexion> : (Chaîne de caractères ou variable de type Connexion)`**

Connexion à utiliser. Cette connexion correspond : 

- soit au nom de la connexion définie sous l'éditeur d'analyses ou grâce aux fonctions [HDécritConnexion](../WDLang4/3044205.md) ou [HOuvreConnexion](../WDLang4/3044107.md).

- soit au nom d'une variable de type [Connexion](../WDLang4/1514073.md).




**`<Groupe> : (Chaîne de caractères)`**

Nom du groupe d'utilisateurs à manipuler.

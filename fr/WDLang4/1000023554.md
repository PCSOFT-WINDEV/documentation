


## &lt;Variable Connexion&gt;.RSListeConfig (Fonction)

***En anglais : &lt;Connection variable&gt;.RSListConfig***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Liste les réplications disponibles sur un serveur HFSQL maître ou abonné.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Déclaration des variables
Cnx est une Connexion
listeRS est un tableau de hRSConfig

// Configuration de la connexion
...

// Liste des réplications
listeRS = MaConnexion.RSListeConfig()
POUR TOUT ConfigRéplication de listeRS
	Trace(ConfigRéplication.Serveur)
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Connexion>.RSListeConfig([<Option>])`
---

**`<Résultat> : (Tableau de hRSConfig)`**

Tableau de variables de type [hRSConfig](../WDLang4/1000020484.md) décrivant les différentes réplications définies sur le serveur HFSQL.

**`<Connexion> : (Variable de type Connexion)`**

Nom de la variable de type [Connexion](../WDLang4/1514073.md) décrivant la connexion vers le serveur HFSQL maître (sur lequel sont définies les réplications).

**`<Option> : (Constante de type Entier)`**

Permet de lister les réplications pour lesquelles le serveur en cours est maître ou abonné : 


|   |   |
| --- | --- |
| *hrsAbonné* | Liste les réplications pour lesquelles le serveur en cours est abonné.<br><br>Remarque : Dans ce cas, la variable de type [hRSConfig](../WDLang4/1000020484.md) correspondant à la réplication a les caractéristiques suivantes :<br><br>- Réplication bidirectionnelle : l'identifiant correspond à l'identifiant de la réplication sur le serveur abonné.<br><br>- Réplication monodirectionnelle : l'identifiant vaut 0.<br><br>- La propriété **Serveur** correspond au nom du serveur maître.<br><br>- Les propriétés **Description** et **Planification** ne sont pas disponibles.<br><br><br> |
| *hrsMaître*<br>(Valeur par défaut) | Liste les réplications pour lesquelles le serveur en cours est maître. |

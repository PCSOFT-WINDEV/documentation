
## hParamètreFichier (Type de variable)

***En anglais : hFileParameter***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **hParamètreFichier** permet de définir les fichiers de données pris en compte par une tâche d'optimisation planifiée. Les caractéristiques de ces fichiers de données peuvent être définies et modifiées à l'aide de différentes propriétés WLangage. 

Ce type de variable est utilisé par le type [hOptimisationPlanifiée](../WDLang4/1000018854.md).

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Recalculer les statistiques des fichier de la base "MaBase", 
// tous les samedis à 23H00
Optim est une hOptimisationPlanifiée
Optim.Description = "Recalcul des statistiques des fichiers de la base de données MaBase"

Optim.Planification.Mois = "*"
Optim.Planification.JourDeLaSemaine = "samedi"
Optim.Planification.Heure = "23"
Optim.Planification.Minute = "0"

Optim.Fichier[1].Nom = "MaBase"
Optim.Fichier[1].Option = hOptStat

HAjouteOptimisationPlanifiée(MaConnexion, Optim)
```

<a name="NOTE0"></a>

## Remarques


### Propriétés spécifiques à la description des variables de type hParamètreFichier
<a name="proprietes_specifiques_description_des_variables_type_hparametrefichier_ELTPARAGRAPHE000040"></a>Les propriétés suivantes peuvent être utilisées pour manipuler une variable de type **hParamètreFichier** :

| **Nom de la propriété** | **Type manipulé** | **Effet** |
| --- | --- | --- |
| MotDePasse | Chaîne de caractères | Mot de passe des fichiers de données cryptés. <br><br>Cette propriété est disponible en écriture uniquement. |
| Nom | Chaîne de caractères | Nom du fichier à traiter. Ce nom peut correspondre à une chaîne de type : <br><br>- &lt;Nom de la base&gt;/&lt;Nom du fichier&gt; : Optimisation du fichier "Nom du fichier" dans la base "Nom de la base". <br><br>- &lt;Nom de la base&gt; : Optimisation de tous les fichiers de données contenus la base "Nom de la base".<br><br>- "\*" : Optimisation de tous les fichiers de données de toutes les bases de données du serveur.<br><br><br> |
| Option | Constante de type Entier | Option d'optimisation : <br><br>- *hOptIndex* : Provoque la réindexation des fichiers de données. Cette réindexation est effectuée à chaud sur les index et les index full-text. <br><br>- *hOptStat* : Provoque le recalcul des statistiques sur les fichiers de données. Le résultat est similaire à l'utilisation de la fonction [HStatCalcule](../WDLang4/3044134.md). Ce traitement est effectué à chaud.<br><br><br> |

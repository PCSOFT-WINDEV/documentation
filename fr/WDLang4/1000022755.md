


## &lt;Variable Connexion&gt;.SupprimeBaseDeDonnées (Fonction)

***En anglais : &lt;Connection variable&gt;.DeleteDatabase***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime une base de données présente sur un serveur HFSQL. Toutes les connexions utilisant la base de données supprimée sont fermées.

<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Description de la connexion
MaConnexion est une Connexion

// Ouverture de la connexion
MaConnexion.OuvreConnexion()
// Suppression de la base de données
MaConnexion.SupprimeBaseDeDonnées()
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Connexion>.SupprimeBaseDeDonnées([<Base de données>] [, <Options>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la base de données a été supprimée,

- <u><u><u><u>Faux</u></u></u></u> en cas d'erreur. La fonction [HErreur](../WDLang4/3044088.md) permet d'identifier l'erreur.




**`<Connexion> : (Variable de type Connexion)`**

Nom de la variable de type [Connexion](../WDLang4/1514073.md) décrivant la connexion pour laquelle la base de données doit être supprimée.

**`<Base de données> : (Chaîne de caractères optionnelle)`**

Nom de la base de données à supprimer. 
Si ce paramètre n'est pas précisé, la base de données utilisée sera celle définie dans la connexion. Si aucune base de données n'est définie dans la connexion, une erreur WLangage sera générée.

**`<Options> : (Constante ou combinaison de constantes de type entier optionnel)`**

Par défaut, lors de la suppression de la base de données, les fichiers correspondant aux journaux ne sont pas supprimés, et les fichiers correspondant aux transactions sont supprimés. Ce paramètre permet de modifier ce comportement :


|   |   |
| --- | --- |
| *hConserveTRS* | Conserve les fichiers de transaction. |
| *hSupprimeJournal* | Supprime le fichier Journal et toutes les opérations journalées. |

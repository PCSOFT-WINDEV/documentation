


## HDéconnecteClient (Fonction)

***En anglais : HDisconnectClient***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Affiche un message sur le(s) poste(s) Client et déconnecte l'application. Les postes Client concernés sont les postes renseignés dans la structure [HClient](../WDLang4/3044340.md).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Déconnexion de tous les utilisateurs de la base de données "BAL"
HClient.Base = "BAL"
HDéconnecteClient("MaConnexion", "Attention : Déconnexion imminente", 300)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = HDéconnecteClient(<Connexion> , <Message de déconnexion> , <Durée d'affichage>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'opération a été réalisée, 

- <u><u><u><u>Faux</u></u></u></u> en cas de problème. 


La fonction **HDéconnecteClient** est bloquante : la suite du programme ne s'exécutera que lorsque les différents clients seront déconnectés.

**`<Connexion> : (Chaîne de caractères ou variable de type Connexion)`**

Connexion à manipuler. Cette connexion correspond : 

- soit au nom de la connexion définie sous l'éditeur d'analyses ou grâce aux fonctions [HDécritConnexion](../WDLang4/3044205.md) ou [HOuvreConnexion](../WDLang4/3044107.md).

- soit au nom d'une variable de type [Connexion](../WDLang4/1514073.md).




**`<Message de déconnexion> : (Chaîne de caractères)`**

Message à afficher sur le poste Client avant la déconnexion.

**`<Durée d'affichage> : (Entier ou Durée)`**

Durée d'affichage du message exprimée en secondes. Ce paramètre peut correspondre à :

- un entier correspondant au nombre de secondes, 

- une variable de type Durée, 

- l'indication directe de la durée (par exemple 1 s). 








<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Fonctionnement
<a name="fonctionnement_ELTPARAGRAPHE000132"></a>Le poste Client spécifié dans la structure [HClient](../WDLang4/3044340.md) affiche le message pendant le durée indiquée et exécute la fonction [FinProgramme](../WDLang1/3013033.md) à la fermeture de la boîte de message.
<a name="NOTE0_2"></a>


### Personnalisation de l'affichage des messages
<a name="personnalisation_affichage_des_messages_ELTPARAGRAPHE000147"></a>La fonction [HSurAppelServeur](../WDLang4/3044343.md) permet de personnaliser la gestion de l'affichage d'un message sur le poste client.




## MongoSupprimeTout (Fonction)

***En anglais : MongoDeleteAll***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime tous les documents (correspondant au filtre spécifié) d'une collection d'une base MongoDB. 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
sFiltre est une chaîne
sFiltre = "{ ""name"" : ""Andy"" }"
MongoSupprimeTout(CollectMDB, sFiltre)
```
<a name="Exemple2"></a>

```wl
// Supprime tous les contacts dont le membre "Prénom" est "Robert"

// Filtre de recherche
sFiltre est une chaîne = "{ ""Prenom"": ""Robert"" } "

MongoSupprimeTout(gclCollection, sFiltre)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = MongoSupprimeTout(<Collection> , <Filtre>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'opération a été réalisée,

- <u><u><u><u>Faux</u></u></u></u> en cas de problème. La fonction [ErreurInfo](../WDLang1/3013008.md) permet d'identifier l'erreur.




**`<Collection> : (Variable de type mongoCollection)`**

Nom de la variable de type [mongoCollection](../WDLang4/1000022411.md) représentant la collection de la base MongoDB dans laquelle la suppression est effectuée.

**`<Filtre> : (Type correspondant au filtre utilisé)`**

Description du/des filtres au format MongoDB à appliquer pour sélectionner les documents à supprimer. Ce filtre correspond à : 

- un document au format JSON, sous forme de chaîne de caractères. 

- un document au format BSON, sous forme de buffer. 

- un variant.

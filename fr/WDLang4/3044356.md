
## HInfoLog (Fonction)

***En anglais : HInfoLog***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie des informations sur les logs du serveur.

**Rappel** : Les logs du serveur permettent de visualiser les appels au serveur effectués par les clients.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
Informations est une chaîne
// Récupération de la correspondance entre les numéros de fonctions 
// et le nom des fonctions WLangage
Informations = HInfoLog("MaConnexion", hCorrespondance)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = HInfoLog(<Nom de la connexion> , <Type d'information>)`
---

**`<Résultat> : (Chaîne de caractères)`**

Résultat détaillé du type d'information demandé. 

Si le type d'information demandé correspond à la constante *hCorrespondance,* &lt;Résultat&gt; est de la forme : 

```txt
<Numéro 1> + TAB + <Paramètre 1> + TAB + <Valeur 1> + TAB + <Nom Français 1> + TAB + 
<Nom Anglais 1> + RC + ... + <Numéro N> + TAB + <Paramètre N> + TAB + <Valeur N> + TAB + 
<Nom Français N> + TAB + <Nom Anglais N>
```

Où :

- &lt;Numéro&gt; correspond au numéro identifiant la fonction dans les logs.

- &lt;Paramètre&gt; correspond au paramètre dans le log qui permet de différencier des fonctions semblables (par exemple [HLitPremier](../WDLang4/3044051.md)/[HLitSuivant](../WDLang4/3044037.md)).

- &lt;Valeur&gt; correspond à la valeur du paramètre.

- &lt;Nom Français&gt; correspond au nom en français de la fonction WLangage.

- &lt;Nom Anglais&gt; correspond au nom en anglais de la fonction WLangage.




**`<Nom de la connexion> : (Chaîne de caractères)`**

Nom de la connexion vers le serveur pour lequel des informations sur les logs sont demandées.

**`<Type d'information> : (Constante de type entier)`**

Indique le type d'information demandée : 


|   |   |
| --- | --- |
| *hCorrespondance* | Fournit la correspondance entre les numéros de fonctions du log et le nom des fonctions WLangage |

<a name="NOTE0_1"></a>

## Remarques


### Récupération des logs
<a name="recuperation_des_logs_ELTPARAGRAPHE000063"></a>

- La fonction [HGèreServeur](../WDLang4/3044359.md) permet d'obtenir et de modifier des informations sur les logs (taille maximale, chemin, ...).

- La fonction [HRécupèreLog](../WDLang4/3044357.md) permet de récupérer les logs du serveur (entre deux dates) dans un fichier au format texte.




## Structure HClient
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000164"></a>
La structure **HClient** permet de gérer :

- la déconnexion d'un ou de plusieurs postes Client (fonction [HDéconnecteClient](../WDLang4/3044339.md))

- l'envoi d'un message à un ou plusieurs postes Client (fonction [HEnvoieMessageVersClient](../WDLang4/3044346.md))












<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Détail de la structure HClient
<a name="detail_structure_hclient_ELTTEXTE000194"></a>


### Variables de la structure
<a name="variables_structure_ELTPARAGRAPHE000066"></a>Les variables suivantes permettent de gérer les postes Client :


|   |   |
| --- | --- |
| HClient.Application | Chaîne de caractères.<br>Nom de l'application. <br>Ce nom est renvoyé par la fonction [HListeUtilisateurConnecté](../WDLang4/3044291.md). |
| HClient.Base | Chaîne de caractères.<br>Base de données HFSQL Client/Serveur. |
| HClient.Fichier | Chaîne de caractères.<br>Fichier de données. |
| HClient.Login | Chaîne de caractères<br>Login de l'utilisateur. |
| HClient.Machine | Chaîne de caractères.<br>Machine de l'utilisateur : Nom de la machine ou adresse IP de la machine. |


Les fonctions utilisant cette structure agiront sur les connexions dont l'utilisateur a le login **HClient.Login** et qui sont effectuées par une application **HClient.Application**. Cette application est lancée sur la machine **HClient.Machine** et utilise le fichier de données **HClient.Fichier** de la base de données **HClient.Base**.

Chaque variable de la structure peut prendre pour valeur "" (chaîne vide) qui signifie "n'importe quelle valeur".

## Fonctions utilisant la structure HClient
<a name="fonctions_utilisant_structure_hclient_ELTTEXTE000218"></a>
Les fonctions utilisant la structure **HClient** sont les suivantes : 



|   |   |
| --- | --- |
| [HDéconnecteClient](../WDLang4/3044339.md) | Affiche un message sur le(s) poste(s) Client et déconnecte l'application. |
| [HEnvoieMessageVersClient](../WDLang4/3044346.md) | Affiche un message sur le(s) poste(s) Client. |
| [HRAZClient](../WDLang4/3044341.md) | Initialise la structure de gestion des postes Client (structure [HClient](../WDLang4/3044340.md)). |

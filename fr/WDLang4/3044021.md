***En anglais : HViewToFile***

	<IMG style="float:left;margin-right:10px;" src="https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=warning_big.png"><B>Avertissement</B><BR>

<a name="Utilisation"></a>
<a name="description"></a>
Enregistre les modifications réalisées dans une vue HFSQL (grâce aux fonctions [HModifie](../WDLang4/3044042.md), [HSupprime](../WDLang4/3044018.md) ou [HRaye](../WDLang4/3044121.md)) dans le fichier de données correspondant.

**Remarque** : Cette opération ne peut être effectuée que si la vue a été créée avec la constante *hVueBlocage* (c'est-à-dire que les enregistrements importés dans la vue ont été bloqués dans le fichier de données). Après l'exécution de la fonction **HVueVersFichier**, les enregistrements sont débloqués dans le fichier source de la vue.

<a name="sample_code"></a>

## Exemple
<a class="notetitle" target="_blank" href="$DOC$=1000003044021&name=hvueversfichier_fonction&product=WD">Voir des exemples supplémentaires</a>

```wl
// Enregistre les modification de la vue Client34
// dans le fichier Client
HVueVersFichier(Client34)
```

## Syntaxe

`<Résultat> = HVueVersFichier(<Vue>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la vue a été transférée dans le fichier de données, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. La fonction [HErreur](../WDLang4/3044088.md) permet d'obtenir plus d'informations sur le problème rencontré.




**`<Vue> : (Chaîne de caractères)`**

Nom de la vue manipulée. Si ce nom n'est pas spécifié, la fonction **HVueVersFichier** va manipuler la dernière vue utilisée par une fonction HFSQL (fonction commençant par la lettre H).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
En cas d'erreur pendant la mise à jour du fichier de données, pour connaître les enregistrements qui ont posé problème et les problèmes associés, utilisez la fonction [HErreur](../WDLang4/3044088.md). Il suffit de parcourir la vue à nouveau et de tester la fonction [HErreur](../WDLang4/3044088.md) sur chaque enregistrement de la vue.

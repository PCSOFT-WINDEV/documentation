


## &lt;Variable Connexion&gt;.DupliqueBaseDeDonnées (Fonction)

***En anglais : &lt;Connection variable&gt;.DuplicateDatabase***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Duplique la base de données associée à une connexion sur un serveur HFSQL. Les tables, les droits, les procédures et les triggers sont dupliqués.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Description de la connexion
MaConnexion est une Connexion

MaConnexion.DupliqueBaseDeDonnées("MaBaseCopiée")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Connexion>.DupliqueBaseDeDonnées(<Base de données dupliquée> [, <Jauge>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la base de données a été dupliquée, 

- <u><u><u><u>Faux</u></u></u></u> en cas d'erreur. La fonction [HErreur](../WDLang4/3044088.md) permet d'identifier l'erreur.




**`<Connexion> : (Variable de type Connexion)`**

Nom de la variable de type [Connexion](../WDLang4/1514073.md) décrivant la connexion associée à la base de données à dupliquer.

**`<Base de données dupliquée> : (Chaîne de caractères)`**

Nom de la nouvelle base de données à créer (copie de la base en cours). 
Remarques :

- La nouvelle base de données à créer ne doit pas exister. 

- Si la nouvelle base de données existe déjà, une erreur WLangage sera générée.




**`<Jauge> : (Nom de fenêtre ou de champ optionnel)`**



- Nom de la fenêtre dans laquelle la jauge doit être affichée. 

- Nom du champ Jauge utilisé pour afficher la progression. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Conditions d'utilisation
<a name="conditions_utilisation_ELTPARAGRAPHE000215"></a>Pour dupliquer une base de données, l'utilisateur doit posséder : 

- le droit de faire des sauvegardes. 

- le droit de créer des bases de données.




## MongoGridFSSupprimeFichier (Fonction)

***En anglais : MongoGridFSDeleteFile***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime un fichier identifié par son nom dans un système de fichiers MongoDB.

**Remarque** : Pour supprimer un fichier identifié à partir de son identifiant, utilisez la fonction [MongoGridFSSupprimeFichierParID](../WDLang4/1000022596.md).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Supprime un fichier image
MongoGridFSSupprimeFichier(cnx.BaseDeDonnées, "produit.png")
```
<a name="Exemple2"></a>

```wl
// Supprime le fichier présent dans la colonne de la table
MongoGridFSSupprimeFichier(gclGridFS, TABLE_GridFS.COL_Nom)

ToastAffiche("Suppression terminée avec succès.")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = MongoGridFSSupprimeFichier(<Système de fichiers> , <Nom du fichier>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'opération a été réalisée,

- <u><u><u><u>Faux</u></u></u></u> en cas de problème. La fonction [ErreurInfo](../WDLang1/3013008.md) permet d'identifier l'erreur.




**`<Système de fichiers> : (Variable de type mongoGridFS ou de type mongoBase)`**

Système de fichiers de la base MongoDB à manipuler. Correspond : 

- au nom de la variable de type [mongoGridFS](../WDLang4/1000022416.md) correspondant au système de fichiers Mongo.

- au nom de la variable de type [mongoBase](../WDLang4/1000022410.md) correspondant à la base MongoDB. Dans ce cas, le système de fichiers utilise le préfixe "fs". 




**`<Nom du fichier> : (Chaîne de caractères)`**

Nom du fichier à supprimer.

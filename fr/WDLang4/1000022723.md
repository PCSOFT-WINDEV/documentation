


## &lt;Variable Connexion&gt;.ModifieUtilisateur (Fonction)

***En anglais : &lt;Connection variable&gt;.ModifyUser***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie les informations d'un utilisateur en fonction des éléments présents dans les [variables de gestion des utilisateurs](../WDLang4/3044315.md) correspondantes.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Description de la connexion
MaConnexion est une Connexion

// Ouverture de la connexion
MaConnexion.OuvreConnexion()
// Association du fichier de données à la connexion
Client.ChangeConnexion(MaConnexion)
SI MaConnexion.InfoUtilisateur("Florence") = Vrai ALORS
	// Modification des caractéristiques
	HUtilisateur.Groupes = "ADMIN" + RC + "TEST" + RC + "DOC"
	SI MaConnexion.ModifieUtilisateur("Florence") = Vrai ALORS
		Info("Utilisateur modifié")
	FIN
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Connexion>.ModifieUtilisateur(<Utilisateur>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'utilisateur a été modifié, 

- <u><u><u><u>Faux</u></u></u></u> en cas d'erreur. La fonction [HErreur](../WDLang4/3044088.md) permet d'identifier l'erreur.




**`<Connexion> : (Variable de type Connexion)`**

Nom de la variable de type [Connexion](../WDLang4/1514073.md) décrivant la connexion pour laquelle un utilisateur doit être modifié.

**`<Utilisateur> : (Chaîne de caractères)`**

Nom de l'utilisateur dont on veut modifier les caractéristiques. Ce nom doit correspondre à la variable **HUtilisateur.Login** de l'un des utilisateurs de la connexion spécifiée.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Droits nécessaires pour modifier un utilisateur
<a name="droits_necessaires_pour_modifier_utilisateur_ELTPARAGRAPHE000245"></a>Pour modifier un utilisateur (différent de l'utilisateur en cours), il est nécessaire de posséder le droit de visualisation des utilisateurs (constante *hDroitVisualisationUtilisateur*). Dans le cas contraire, la fonction **&lt;Variable Connexion&gt;.ModifieUtilisateur** renverra <u><u><u><u>Faux</u></u></u></u> et le message d'erreur sera "Utilisateur inconnu".

Pour modifier le nom d'un utilisateur, il est nécessaire de posséder le droit de visualisation des utilisateurs (constante *hDroitVisualisationUtilisateur*) et le droit de gestion des utilisateurs (constante *hDroitGestionUtilisateur*).

Remarque : Ces constantes sont utilisées dans les fonctions [&lt;Variable Connexion&gt;.InfoDroitServeur](../WDLang4/1000022668.md) et [&lt;Variable Connexion&gt;.ModifieDroitServeur](../WDLang4/1000022710.md).

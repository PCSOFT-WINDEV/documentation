
## Connecteur Natif AS/400
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000066"></a>
WINDEV et WEBDEV proposent en option un module d'accès à l'AS/400. Ce Connecteur Natif (également appelé "Accès Natif") permet d'accéder à une base AS/400 sans drivers externes depuis un programme en WLangage.

Vous pouvez ainsi proposer à vos utilisateurs des interfaces (Windows et WEB 2.0) qui manipuleront nativement les données de l'AS/400, aussi bien en lecture qu'en écriture.

La configuration des machines est très simple à réaliser. WINDEV et WEBDEV communiquent avec votre AS/400 soit via le protocole APPC de votre routeur, soit via le protocole TCP/IP.

**Remarque** : Le module "Connecteur natif AS/400" a été conçu par la société "Aura Equipements". Un fichier d'aide contenant toutes les spécificités de ce connecteur Natif est livré avec le Connecteur Natif AS/400.




## &lt;Variable Connexion&gt;.SupprimeRequête (Fonction)

***En anglais : &lt;Connection variable&gt;.DeleteQuery***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime une requête (utilisée par des procédures stockées) d'un serveur HFSQL.



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MaConnexion est une Connexion

// Supprimer la requête "GestionMail" du serveur
MaConnexion.SupprimeRequête(GestionMail)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Connexion>.SupprimeRequête(<Requête à supprimer>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la requête a été supprimée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. La fonction [HErreur](../WDLang4/3044088.md) permet d'identifier l'erreur.




**`<Connexion> : (Variable de type Connexion)`**

Nom de la variable de type [Connexion](../WDLang4/1514073.md) décrivant la connexion vers le serveur HFSQL contenant la requête à supprimer.

**`<Requête à supprimer> : (Chaîne de caractères)`**

Nom de la requête à supprimer du serveur.

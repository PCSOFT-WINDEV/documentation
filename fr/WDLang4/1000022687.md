


## &lt;Variable Connexion&gt;.InfoUtilisateur (Fonction)

***En anglais : &lt;Connection variable&gt;.InfoUser***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Met à jour [les variables de gestion des utilisateurs](../WDLang4/3044315.md) avec les informations concernant l'utilisateur spécifié. Pour des raisons de sécurité, le mot de passe de l'utilisateur n'est pas récupéré.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Description de la connexion
MaConnexion est une Connexion

// Ouverture de la connexion
MaConnexion.OuvreConnexion()
// Association du fichier de données à la connexion
Client.ChangeConnexion(MaConnexion)
SI MaConnexion.InfoUtilisateur("Florence") = Vrai ALORS
	Info("Voici les caractéristiques de l'utilisateur" + RC + ...
		"Login : " + HUtilisateur.Login + ...
		"Nom Complet : " + HUtilisateur.NomComplet + ...
		"Email : " + HUtilisateur.Email)
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Connexion>.InfoUtilisateur(<Utilisateur>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la structure de gestion des utilisateurs a été mise à jour, 

- <u><u><u><u>Faux</u></u></u></u> en cas d'erreur. La fonction [HErreurInfo](../WDLang4/3044071.md) permet d'identifier l'erreur.




Remarque : Une erreur "Utilisateur inconnu" sera renvoyée par la fonction [HErreurInfo](../WDLang4/3044071.md) si l'utilisateur en cours n'a pas le droit de visualisation des utilisateurs (constante *hDroitVisualisationUtilisateur*). Ce droit peut être défini par la fonction [&lt;Variable Connexion&gt;.ModifieDroitServeur](../WDLang4/1000022710.md).

**`<Connexion> : (Variable de type Connexion)`**

Nom de la variable de type [Connexion](../WDLang4/1514073.md) décrivant la connexion pour laquelle les renseignements concernant l'utilisateur doivent être récupérés.

**`<Utilisateur> : (Chaîne de caractères)`**

Nom de l'utilisateur pour lequel les informations doivent être récupérées. Ce nom doit correspondre à la variable **HUtilisateur.Login** de l'un des utilisateurs de la connexion spécifiée.

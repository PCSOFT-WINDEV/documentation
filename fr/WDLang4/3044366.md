


## HSupprimeRequête (Fonction)

***En anglais : HDeleteQuery***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime une requête (utilisée par des procédures stockées) d'un serveur HFSQL.



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Supprimer la requête "GestionMail" du serveur
HSupprimeRequête("MaConnexion", GestionMail)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = HSupprimeRequête(<Connexion> , <Requête à supprimer>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la requête a été supprimée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. La fonction [HErreur](../WDLang4/3044088.md) permet d'identifier l'erreur.




**`<Connexion> : (Chaîne de caractères ou variable de type Connexion)`**

Connexion vers le serveur HFSQL contenant la requête à supprimer. Cette connexion correspond :

- soit au nom de la connexion définie sous l'éditeur d'analyses ou grâce aux fonctions [HDécritConnexion](../WDLang4/3044205.md) ou [HOuvreConnexion](../WDLang4/3044107.md).

- soit au nom d'une variable de type [Connexion](../WDLang4/1514073.md).




**`<Requête à supprimer> : (Chaîne de caractères)`**

Nom de la requête à supprimer du serveur.

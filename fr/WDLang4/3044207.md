
## Accéder à des fichiers HFSQL via SAMBA sur Linux
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000080"></a>
Pour accéder à des fichiers HFSQL via Samba sur Linux, un paramétrage spécifique est nécessaire.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Paramétrage nécessaire
<a name="parametrage_necessaire_ELTTEXTE000110"></a>
Dans le fichier **smb.conf** qui se trouve communément sous /etc/samba, il faut rajouter ou modifier les paramètres suivants dans la section [global] du fichier.


```txt
[global]
locking = yes
strict locking = yes
share modes = yes
oplocks = no
kernel oplocks = no
blocking locks = no
fake oplocks = no
level2 oplocks = no
```


Ensuite, il est nécessaire de redémarrer le service Samba pour prendre en compte le nouveau paramétrage.




## HRSInfo (Fonction)

***En anglais : HRSInfo***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de lire la configuration de la réplication pour un serveur HFSQL participant à une ou plusieurs réplication(s).

Cette fonction peut être exécutée aussi bien sur un serveur maître que sur un serveur abonné. 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
Cnx est une Connexion

Trace(HRSInfo(Cnx))
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = HRSInfo(<Connexion>)`
---

**`<Résultat> : (Chaîne de caractères)`**

Informations concernant les différentes réplications effectuées sur ce serveur. Ces informations sont de la forme :  

```txt
<Rôle du serveur> + TAB + <Identifiant serveur> + TAB + <Port> + TAB + <Priorité>
```
où : 

- &lt;Rôle du serveur&gt; correspond à une constante de type entier permettant de connaître le rôle du serveur dans la réplication : 
	


|   |   |
| --- | --- |
| *hrsAbonné* | Poste abonné dans la réplication entre serveurs HFSQL. |
| *hrsInactif* | Poste inactif dans la réplication entre serveurs HFSQL. |
| *hrsMaître* | Poste maître dans la réplication entre serveurs HFSQL. |
| *hrsMaître* + *hrsAbonné* | Postes maître et abonné dans la réplication entre serveurs HFSQL (cas d'une réplication bi-directionnelle). |



- &lt;Identifiant serveur&gt; correspond au numéro identifiant le serveur HFSQL en cours parmi tous les serveurs participant à une réplication.

- &lt;Port&gt; correspond au port TCP/IP utilisé pour la communication entre les serveurs.

- &lt;Priorité&gt; correspond au numéro de priorité du serveur utilisé dans le cas d'une résolution par priorité lors d'un conflit de modification.




**`<Connexion> : (Chaîne de caractères ou variable de type Connexion)`**

Connexion vers le serveur HFSQL. Cette connexion correspond : 

- soit au nom de la connexion définie sous l'éditeur d'analyses ou grâce aux fonctions [HDécritConnexion](../WDLang4/3044205.md) ou [HOuvreConnexion](../WDLang4/3044107.md).

- soit au nom d'une variable de type [Connexion](../WDLang4/1514073.md).

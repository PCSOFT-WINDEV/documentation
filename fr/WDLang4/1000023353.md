
## MemcachedSupprimeGroupeClé (Fonction)

***En anglais : MemcachedDeleteKeyGroup***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime une clé dans le cache Memcached sur un groupe de serveurs. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MemcachedSupprimeGroupeClé(memConnexion, "MonApp", "clé")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = MemcachedSupprimeGroupeClé(<Connexion> , <Groupe> , <Clé>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'opération a été réalisée, 

- <u><u><u><u>Faux</u></u></u></u> en cas d'erreur. Pour connaître le libellé de l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.




**`<Connexion> : (Variable de type memcachedConnexion)`**

Nom de la variable de type [memcachedConnexion](../WDLang4/1000023339.md) correspondant à la connexion au cache Memcached. 

**`<Groupe> : (Chaîne de caractères)`**

Partie de la clé permettant de choisir le serveur à utiliser. 

**`<Clé> : (Chaîne de caractères)`**

Suite de la clé à supprimer.

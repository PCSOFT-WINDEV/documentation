
## hOpérationServeur (Type de variable)

***En anglais : hServerOperation***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **hOpérationServeur** permet de connaître toutes les caractéristiques d'une opération qui se déroule sur un serveur HFSQL. Les caractéristiques de cette opération peuvent être connues à l'aide de différentes propriétés WLangage. 

Ce type de variable est utilisé par la fonction [HOpérationEnCours](../WDLang4/1000022480.md). 

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
TabOp est un tableau de hOpérationServeur = HOpérationEncours(cnx)
```

## Remarques
<a name="NOTE0_1"></a>


### Propriétés spécifiques à la description des variables de type hOpérationServeur
<a name="proprietes_specifiques_description_des_variables_type_hoperationserveur_ELTPARAGRAPHE000034"></a>Les propriétés suivantes peuvent être utilisées pour manipuler une variable de type **hOpérationServeur** :

| Nom de la propriété | Type manipulé | Effet |
| --- | --- | --- |
| DateDébut | DateHeure | Date et Heure de début de l'opération.<br>**Cette propriété est disponible en lecture seulement**. |
| OptimisationPlanifiée | Variable de type [hOptimisationPlanifiée](../WDLang4/1000018854.md) | Cette propriété est renseignée uniquement si l'opération est de type "Optimisation de fichiers" (propriété **Type** vaut *hlpOptimisation*).<br>**Cette propriété est disponible en lecture seulement**. |
| Réplication | Variable de type [hRSConfig](../WDLang4/1000020484.md) | Cette propriété est renseignée uniquement si l'opération est de type "Réplication" (propriété **Type** vaut *hlpRéplication*). <br>**Cette propriété est disponible en lecture seulement**. |
| SauvegardePlanifiée | Variable de type [hDescriptionSauvegarde](../WDLang4/1000017455.md) | Cette propriété est renseignée uniquement si l'opération est de type "Sauvegarde" (propriété **Type** vaut *hlpSauvegarde*). <br>**Cette propriété est disponible en lecture seulement**. |
| TâchePlanifiée | Variable de type [hTâchePlanifiée](../WDLang4/1000017448.md) | Cette propriété est renseignée uniquement si l'opération est de type "Tâche" (propriété **Type** vaut *hlpTâche*). <br>**Cette propriété est disponible en lecture seulement**. |
| Type | Constante de type Entier | Type de l'opération : <br><br>- *hlpCalculStat* : Calcul de statistiques. <br><br>- *hlpDétectionClé* : Détection de clés. <br><br>- *hlpOptimisation* : Optimisation des fichiers de données. <br><br>- *hlpRéplication* : Réplication entre serveurs. <br><br>- *hlpSauvegarde* : Sauvegarde. <br><br>- *hlpTâche* : Tâche planifiée. <br><br>- *hlpVueMatérialisée* : Rafraîchissement d'une vue matérialisée. <br><br><br><br><br>**Cette propriété est disponible en lecture seulement**. |
| VueMatérialiséePlanifiée | Variable de type [hPlanificationVueMatérialisée](../WDLang4/1000021444.md) | Cette propriété est renseignée uniquement si l'opération est de type "Rafraîchissement de vue" (propriété **Type** vaut *hlpVueMatérialisée*).<br>**Cette propriété est disponible en lecture seulement**. |

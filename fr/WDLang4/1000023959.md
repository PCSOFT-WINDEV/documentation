


## &lt;Variable Connexion&gt;.FichierExiste (Fonction)

***En anglais : &lt;Connection variable&gt;.FileExist***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de savoir si un fichier de données existe, c'est-à-dire s'il a été créé physiquement (fonctions [&lt;Source&gt;.Création](../WDLang4/1000024207.md) ou [&lt;Source&gt;.CréationSiInexistant](../WDLang4/1000024208.md)). 

Remarque : Les substitutions en cours sur le fichier de données (effectuées avec les fonctions [HSubstRep](../WDLang4/3044028.md), [&lt;Source&gt;.ChangeRep](../WDLang4/1000024202.md) et [&lt;Source&gt;.ChangeNom](../WDLang4/1000024201.md)) sont prises en compte.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple

<a name="3044096_Exemple2"></a>

```wl
// Description de la connexion
MaConnexion est une Connexion

// Test de l'existence d'un fichier de données Client/Serveur
SI MaConnexion.FichierExiste("MonFichier.FIC") = Vrai ALORS
	// Traitements
	...
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

### Tester l'existence d'un fichier de données en HFSQL Client/Serveur

`<Résultat> = <Connexion>.FichierExiste(<Fichier physique>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u>Vrai</u></u></u> si le fichier de données physique existe sur le serveur, 

- <u><u><u>Faux</u></u></u> dans le cas contraire.




**`<Connexion> : (Variable de type Connexion)`**

Nom de la variable de type [Connexion](../WDLang4/1514073.md) décrivant la connexion associée au fichier de données.

**`<Fichier physique> : (Chaîne de caractères)`**

Nom physique du fichier de données recherché. L'extension du fichier doit être précisée (par exemple "Client.fic").


<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- La fonction **&lt;Variable Connexion&gt;.FichierExiste** est utilisable que le fichier de données soit ouvert ou non.

- La fonction **&lt;Variable Connexion&gt;.FichierExiste** n'ouvre pas le fichier de données si celui-ci est fermé.

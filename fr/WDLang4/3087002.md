
## H.NbEnregRequêteModification

***En anglais : H.NbRecModificationQuery***

<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de connaître le nombre d'enregistrements ayant été ajoutés, modifiés ou supprimés lors de l'exécution de la dernière requête du contexte en cours. 

Cette variable est disponible en lecture uniquement. Elle est mise à jour lors de l'exécution d'une requête de type ajout, modification, suppression (notamment par programmation avec les fonctions [HExécuteRequête](../WDLang4/3044080.md) et [HExécuteRequêteSQL](../WDLang4/3044084.md)).

![Connecteurs Natifs (Accès Natifs)](https://doc.pcsoft.fr/ext/images/fr/AN.png) Accès via un Connecteur Natif Oracle : Cette variable est également mise à jour.

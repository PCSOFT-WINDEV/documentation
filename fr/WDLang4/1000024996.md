


## &lt;Source&gt;.InfoFichier (Fonction)

***En anglais : &lt;Source&gt;.InfoFile***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie les caractéristiques d'un fichier de données HFSQL Classic ou Client/Serveur.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Récupération de la taille d'un fichier de données
// en utilisant la connexion en cours d'un fichier de données
Trace("Taille totale fichier Client : " + Client.InfoFichier(hTailleTotale))
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Obtenir des informations concernant un fichier de données HFSQL Classic

`<Résultat> = <Source>.InfoFichier(<Information>)`
---

**`<Résultat> : (Type de l'information recherchée)`**

Information recherchée.

**`<Source> : (Type correspondant à la source spécifiée)`**

Nom du fichier de données à analyser.

**`<Information> : (Constante de type Entier)`**

Information voulue sur le fichier de données :


|   |   |
| --- | --- |
| *hTailleFic* | Taille en octets du fichier de données (fichier ".Fic"). |
| *hTailleFtx* | Taille en octets du fichier d'index full-text (fichier ".Ftx"). |
| *hTailleMmo* | Taille en octets du fichier mémo (fichier ".Mmo"). |
| *hTailleNdx* | Taille en octets du fichier d'index (fichier ".Ndx"). |
| *hTailleTotale* | Taille totale en octets du fichier de données (taille cumulée du fichier de données, du fichier mémo et du fichier d'index). |

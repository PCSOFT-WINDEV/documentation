## Limitations connues
<a name="limitations_connues_ELTTEXTE000206"></a>



|   |   |
| --- | --- |
| <br><br>- Les noms longs ne sont pas supportés pour les fichiers xBase.<br><br><br> |   |
| <br><br>- Nombre maximum d'enregistrements par fichier de données :<br><br><br> | **illimité** |
| <br><br>- Nombre maximum de fichiers xBase ouverts :<br><br><br> | **1024** |
| <br><br>- Nombre maximum de champs par fichier :<br><br><br> |   |
| - fichier dBase 4 : | **255** |
| - fichier DBX : | **512** |
| - fichier Clipper5 : | **1.023** |
| - autres : | **128** |
| <br><br>- Taille maximale d'un enregistrement :<br><br><br> |   |
| - fichier Clipper5 : | **8.192** |
| - autre : | **4.000** |
| <br><br>- Taille maximale d'un champ texte :<br><br><br> |   |
| - fichier Clipper5 : | **2.048** |
| - autres : | **254** |
| <br><br>- Taille maximale d'un champ numérique :<br><br><br> |   |
| - fichier dBase 4 : | **20** |
| - fichier FoxPro : | **20** |
| - autres : | **19** |
| <br><br>- Taille maximale d'un champ numérique décimal :<br><br><br> |   |
| - fichier dBase 4 : | **18** |
| - fichier FoxPro : | **18** |
| - autres : | **15** |
| <br><br>- Nombre maximum d'enregistrements par fichier d'index :<br><br><br> | **illimité** |
| <br><br>- Taille maximale d'une clé :<br><br><br> | 100 |
| <br><br>- Longueur maximale d'une formule de clé :<br><br><br> | 100 |
| <br><br>- Fonctions autorisées dans les formules de clés :<br><br>	- **DTOC**<br><br>	- **DTOS**<br><br>	- **LEFT**<br><br>	- **RIGHT**<br><br>	- **STR**<br>			En Clipper, il est nécessaire de préciser une taille. <br><br>	- **SUBSTR**<br><br>	- **UPPER**<br><br>	- **LOWER**<br><br><br><br><br><br> |   |
| <br><br>- Nombre maximum d'index dans un fichier MDX (fichier d'index dBase 4) :<br><br><br> | 47 |
| <br><br>- Nombre maximum d'enregistrements par fichier mémo :<br><br><br> | **illimité** |
| <br><br>- Taille maximale d'un mémo texte (pouvant être saisi) :<br><br><br> | 24 ko |
| <br><br>- Taille maximale d'un nom de champ :<br><br><br> | 10 |
| <br><br>- Taille maximale d'un nom de clé :<br><br><br> | 10 |

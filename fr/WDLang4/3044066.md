


## HGèreTransaction (Fonction)

***En anglais : HSetTransaction***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Active ou désactive la gestion des transactions sur un ou plusieurs fichiers de données.

Lorsque la gestion des transaction est désactivée, toutes les fonctions relatives aux transactions sont inhibées ([HTransactionDébut](../WDLang4/3044002.md), [HTransactionFin](../WDLang4/3044032.md), [HTransactionAnnule](../WDLang4/3044001.md)).

**Remarque** : Par défaut, la gestion des transactions est active.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Désactive la gestion des transactions sur le fichier de données Client
HGèreTransaction(Client, Faux)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = HGèreTransaction([<Fichier de données>, ] <Options>)`
---

**`<Résultat> : (Booléen)`**

Correspond à l'ancienne gestion des transactions (avant utilisation de la fonction **HGèreTransaction**) : 

- <u><u><u><u>Vrai</u></u></u></u> si la gestion des transactions était active, 

- <u><u><u><u>Faux</u></u></u></u> si la gestion des transactions n'était pas active. 


Si &lt;Fichier de données&gt;= "\*", &lt;Résultat&gt; est à <u><u><u><u>Vrai</u></u></u></u> si dans l'ancien mode de gestion des transactions, la gestion des transactions était active au moins sur un des fichiers de données.

**`<Fichier de données> : (Chaîne de caractères optionnelle)`**

Nom du fichier de données manipulé. Si ce nom n'est pas spécifié ou correspond à "\*", la gestion des transactions est prise en compte pour tous les fichiers de données de l'analyse.

**`<Options> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la gestion des transactions sur le fichier de données doit être activée, 

- <u><u><u><u>Faux</u></u></u></u> si cette gestion doit être désactivée.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Gestion des transactions activée
<a name="gestion_des_transactions_activee_ELTPARAGRAPHE000122"></a>Si la gestion des transactions est activée pour un seul fichier de données, seules les manipulations réalisées sur ce fichier de données seront prises en compte dans la gestion des transactions (quels que soit les fichiers de données indiqués dans la fonction [HTransactionDébut](../WDLang4/3044002.md)).

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) 

### Limite 
<a name="limite_ELTPARAGRAPHE000136"></a>La gestion des transactions ne s'applique pas aux fichiers xBase.

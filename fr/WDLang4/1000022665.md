


## &lt;Variable Connexion&gt;.GèreTâche (Fonction)

***En anglais : &lt;Connection variable&gt;.ManageTask***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Active ou désactive une tâche planifiée d'un serveur HFSQL Client/Serveur. Cette fonction permet également de connaître l'état d'une tâche planifiée.

Il est ainsi possible par exemple de désactiver une tâche (pendant les vacances) puis de la réactiver.



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Activer/Désactiver une tâche planifiée

`<Résultat> = <Connexion>.GèreTâche(<Tâche planifiée> [, <Activer/Désactiver>])`
---

**`<Résultat> : (Booléen)`**

Ancien mode de gestion de la tâche (avant utilisation de la fonction **&lt;Variable Connexion&gt;.GèreTâche**) :

- <u><u><u><u>Vrai</u></u></u></u> si la tâche était active, 

- <u><u><u><u>Faux</u></u></u></u> si la tâche n'était pas active.




**`<Connexion> : (Variable de type Connexion)`**

Nom d'une variable de type [Connexion](../WDLang4/1514073.md) décrivant la connexion pour laquelle la tâche doit être gérée.

**`<Tâche planifiée> : (Entier)`**

Identifiant de la tâche planifiée à manipuler. Cet identifiant est fourni par les fonctions [&lt;Variable Connexion&gt;.AjouteTâche](../WDLang4/1000022645.md) et [&lt;Variable Connexion&gt;.ListeTâche](../WDLang4/1000022700.md).

**`<Activer/Désactiver> : (Booléen optionnel)`**



- <u><u><u><u>Vrai</u></u></u></u> pour activer la tâche, 

- <u><u><u><u>Faux</u></u></u></u> pour désactiver la tâche.





<a name="SYNTAXE2"></a>

### Connaître l'état d'une tâche planifiée

`<Résultat> = <Connexion>.GèreTâche(<Tâche planifiée>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la tâche est active, 

- <u><u><u><u>Faux</u></u></u></u> si la tâche n'est pas active.




**`<Connexion> : (Variable de type Connexion)`**

Nom de la variable de type [Connexion](../WDLang4/1514073.md) décrivant la connexion pour laquelle la tâche doit être gérée.

**`<Tâche planifiée> : (Entier)`**

Identifiant de la tâche planifiée à manipuler. Cet identifiant est fourni par les fonctions [&lt;Variable Connexion&gt;.AjouteTâche](../WDLang4/1000022645.md) et [&lt;Variable Connexion&gt;.ListeTâche](../WDLang4/1000022700.md).




## HDésactiveFiltre (Fonction)

***En anglais : HDeactivateFilter***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Désactive temporairement un filtre sur un fichier de données (une vue ou une requête). Ce filtre a été créé avec la fonction [HFiltre](../WDLang4/3044100.md). Si aucun filtre n'a été déclaré, la fonction **HDésactiveFiltre** est sans effet.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple
<a class="notetitle" target="_blank" href="$DOC$=1000003044085&name=hdesactivefiltre_fonction&product=WD">Voir des exemples supplémentaires</a>

```wl
// Déclaration du filtre : sélection des clients
// qui ont entre 20 et 60 ans
HFiltre(Client, Age_Client, 20, 60)
...
HDésactiveFiltre(Client) // Filtre désactivé
...
// Parcours du fichier de données avec filtre sur l'âge
// Traitement des clients qui ont entre 20 et 60 ans
// (procédure TraiteClient2060)
HActiveFiltre(Client)
HLitPremier(Client, Age_Client)
TANTQUE HTrouve(Client) = Vrai
	TraiteClient2060()
	HLitSuivant(Client, Age_Client)
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = HDésactiveFiltre([<Fichier de données>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le filtre est désactivé, 

- <u><u><u><u>Faux</u></u></u></u> en cas d'erreur (fichier de données ou rubrique inexistant).




**`<Fichier de données> : (Chaîne de caractères optionnelle)`**

Nom du fichier de données, de la vue ou de la requête manipulé. 
Si ce paramètre n'est pas spécifié, la fonction **HDésactiveFiltre** manipule le dernier fichier de données utilisé par la dernière fonction de gestion HFSQL (fonction commençant par la lettre H).



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Activation et désactivation d'un filtre
<a name="activation_desactivation_filtre_ELTPARAGRAPHE000096"></a>Après la déclaration d'un filtre (fonction [HFiltre](../WDLang4/3044100.md)), le filtre est actif. Après désactivation d'un filtre avec la fonction **HDésactiveFiltre**, le filtre peut être réactivé grâce à la fonction [HActiveFiltre](../WDLang4/3044158.md).

Remarque : Lors de l'appel de la fonction [HFerme](../WDLang4/3044073.md), le filtre activé par la fonction **HFiltre** est automatiquement supprimé.
<a name="NOTE0_2"></a>


### Nombre de filtres actifs possibles
<a name="nombre_filtres_actifs_possibles_ELTPARAGRAPHE000119"></a>Pour un fichier donné, un seul filtre peut être actif à la fois.
<a name="NOTE0_3"></a>


### Parcours
<a name="parcours_ELTPARAGRAPHE000128"></a>Les modifications apportées par les fonctions [HFiltre](../WDLang4/3044100.md) et **HDésactiveFiltre** sont prises en compte uniquement lors de l'utilisation des fonctions [HLitPremier](../WDLang4/3044051.md)/[HPremier](../WDLang4/3044113.md), [HLitDernier](../WDLang4/3044140.md)/[HDernier](../WDLang4/3044101.md) ou [HLitRecherchePremier](../WDLang4/3044036.md)/[HLitRechercheDernier](../WDLang4/3044034.md).

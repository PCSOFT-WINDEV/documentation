
## mongoGridFSInfoFichier (Type de variable)

***En anglais : mongoGridFSFileInfo***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **mongoGridFSInfoFichier** permet de connaître toutes les caractéristiques avancées d'un fichier contenu dans une base MongoDB. Les caractéristiques de ce fichier peuvent être connues à l'aide de différentes propriétés WLangage. 

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
tabListeFichier est un tableau de mongoGridFSInfoFichier

tabListeFichier = MongoGridFSListeFichier(CnxMDB.BaseDeDonnées["CRM"])
POUR TOUT Fic de tabListeFichier
	// Traite le fichier
	Trace(Fic.Nom)
FIN
```

## Remarques
<a name="NOTE0_1"></a>


### Propriétés spécifiques à la description des variables de type mongoGridFSInfoFichier
<a name="proprietes_specifiques_description_des_variables_type_mongogridfsinfofichier_ELTPARAGRAPHE000039"></a>Les propriétés suivantes peuvent être utilisées pour manipuler les informations d'un fichier :

| Nom de la propriété | Type manipulé | Effet |
| --- | --- | --- |
| Alias | Tableau de chaînes | Tableau des noms d'alias du fichier.<br><br>**Cette propriété est disponible en lecture seulement**. |
| Date | DateHeure | Date de création du fichier au format UTC.<br><br>**Cette propriété est disponible en lecture seulement**. |
| ID | Variant | Identifiant du fichier.<br><br>**Cette propriété est disponible en lecture seulement**. |
| Md5 | Chaîne de caractères | Clé de hachage (md5) pré-calculée sur le contenu du fichier.<br><br>**Cette propriété est disponible en lecture seulement**. |
| Métadonnées | Variant | Données additionnelles stockées avec le fichier.<br><br>**Cette propriété est disponible en lecture seulement**. |
| Nom | Chaîne de caractères | Nom du fichier.<br><br>**Cette propriété est disponible en lecture seulement**. |
| Taille | Entier | Taille du fichier en octets.<br><br>**Cette propriété est disponible en lecture seulement**. |
| TailleChunk | Entier | Taille des morceaux du fichier.<br><br>**Cette propriété est disponible en lecture seulement**. |
| TypeContenu | Chaîne de caractères | Type du contenu au format MIME.<br><br>**Cette propriété est disponible en lecture seulement**. |

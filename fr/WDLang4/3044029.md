


## HStatNbEnrIntervalle (Fonction)

***En anglais : HStatNbRecRange***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie une estimation majorée du nombre d'entrées pour une rubrique clé donnée dans un intervalle de valeurs donné. Cette estimation est déduite des informations obtenues lors du dernier calcul des statistiques.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Réalisation de statistiques sur la clé NomCli du fichier de données Client
RésultatStat = HStatNbEnrIntervalle(Client, NomCli, "Dupont", "Durant")
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = HStatNbEnrIntervalle([<Fichier de données> [, <Rubrique clé>, ]] <Borne minimale> , <Borne maximale>)`
---

**`<Résultat> : (Entier)`**

Nombre d'entrées trouvées pour la rubrique spécifiée dans l'intervalle voulu. 
Correspond à la constante *HNumEnrInvalide* si :

- les statistiques n'ont jamais été calculées pour la rubrique spécifiée.

- les statistiques ne sont pas supportées pour le type de la rubrique spécifiée. HFSQL permet de calculer des statistiques uniquement sur des clés.




**`<Fichier de données> : (Chaîne de caractères optionnelle)`**

Nom du fichier de données manipulé. Si ce nom n'est pas spécifié, la fonction **HStatNbEnrIntervalle** va manipuler le dernier fichier de données utilisé par la dernière fonction de gestion HFSQL (fonction commençant par la lettre H).

**`<Rubrique clé> : (Chaîne de caractères optionnelle)`**

Nom de la rubrique clé permettant de réaliser les statistiques. Si ce nom n'est pas spécifié, la fonction **HStatNbEnrIntervalle** va manipuler la dernière rubrique utilisée par la dernière fonction de gestion HFSQL (fonction commençant par la lettre H).

**`<Borne minimale> : (Type de la rubrique clé)`**

Valeur inférieure de l'intervalle.

**`<Borne maximale> : (Type de la rubrique clé)`**

Valeur supérieure de l'intervalle.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Statistiques sur des clés composées
<a name="statistiques_sur_des_cles_composees_ELTPARAGRAPHE000107"></a>Pour faire un calcul de statistiques sur une clé composée, il est possible d'utiliser la syntaxe suivante : 

```txt
HStatNbEnrIntervalle(<Fichier de données>, <Nom de la clé composée>, 
	[<Borne minimale du premier élément de la clé composée>, 
			<Borne minimale du second élément de la clé composée>], 
	[<Borne maximale du premier élément de la clé composée>, 
			<Borne maximale du second élément de la clé composée>, ...])
```

Exemple :

```wl
HStatNbEnrIntervalle(Client, Nom_Prenom, ["Blanc", "Alice"], ["Moulin", "Françoise"])
```

<a name="NOTE0_2"></a>


### Divers
<a name="divers_ELTPARAGRAPHE000130"></a>

- La mise à jour des statistiques peut être réalisée par les fonctions [HRéindexe](../WDLang4/3044133.md) et [HStatCalcule](../WDLang4/3044134.md).

- Dans les fichiers de données HFSQL Classic, seules les rubriques clés peuvent faire l'objet d'un calcul de statistiques.

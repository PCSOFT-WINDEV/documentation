
## Parcours de fichiers de données HFSQL
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000091"></a>
Les rubriques "Clé" permettent d'organiser les enregistrements dans un fichier de données pour optimiser le parcours ou la recherche d'enregistrements.





<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Recherche dans un fichier sans clé
<a name="recherche_dans_fichier_sans_cle_ELTTEXTE000121"></a>
Tous les enregistrements sont lus pour retrouver l'enregistrement recherché.
<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=rubrique_cle1.gif)




<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Recherche dans un fichier avec une clé
<a name="recherche_dans_fichier_avec_une_cle_ELTTEXTE000145"></a>
L'utilisation d'une clé permet de créer un INDEX. L'index est une boîte noire permettant un accès direct à l'enregistrement recherché.
<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=rubrique_cle2.gif)

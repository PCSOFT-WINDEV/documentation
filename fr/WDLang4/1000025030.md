


## &lt;Source&gt;.SurveilleStop (Fonction)

***En anglais : &lt;Source&gt;.TrackStop***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Arrête la surveillance d'un fichier de données sur un serveur HFSQL. Cette surveillance a été précédemment démarrée via la fonction [&lt;Source&gt;.Surveille](../WDLang4/1000025029.md). 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Lance la surveillance du fichier de données Commande
// La procédure TraiteCommande est appelée pour tout ajout d'enregistrement dans le fichier de données
Commande.Surveille(TraiteCommande, hNumEnrTous, hsAjout)

...
// Arrête la surveillance du fichier de données Commande
Commande.SurveilleStop(TraiteCommande, hNumEnrTous)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Source>.SurveilleStop(<Procédure WLangage> [, <Numéro d'enregistrement>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la surveillance a été arrêtée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire (par exemple, surveillance inexistante). Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Source> : (Type correspondant à la source spécifiée)`**

Nom du fichier de données HFSQL à manipuler. 

**`<Procédure WLangage> : (Nom de procédure)`**

Nom d'une procédure WLangage à ne plus appeler pour la surveillance du fichier de données.

**`<Numéro d'enregistrement> : (Entier optionnel)`**

Numéro de l'enregistrement à ne plus surveiller. 
Si ce paramètre n'est pas précisé ou correspond à la constante *hNumEnrTous*, la fonction **&lt;Source&gt;.SurveilleStop** va arrêter de surveiller tous les enregistrements du fichier de données (c'est-à-dire tout le fichier de données).



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Les fonctions [&lt;Source&gt;.Surveille](../WDLang4/1000025029.md) et **&lt;Source&gt;.SurveilleStop** peuvent être utilisées dans un service Windows.

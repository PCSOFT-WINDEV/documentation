


## &lt;Variable Connexion&gt;.SupprimeRépertoire (Fonction)

***En anglais : &lt;Connection variable&gt;.DeleteDirectory***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime un répertoire présent dans une base de données HFSQL Client/Serveur. Ce répertoire doit être vide. 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MaConnexion est une Connexion

MaConnexion.SupprimeFichier("ModuleCRM\Prospect.fic")
MaConnexion.SupprimeRépertoire("ModuleCRM")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Connexion>.SupprimeRépertoire(<Répertoire à supprimer>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le répertoire a été supprimé, 

- <u><u><u><u>Faux</u></u></u></u> en cas d'erreur (par exemple si le répertoire n'existe pas sur le serveur ou si le répertoire n'est pas vide). La fonction [HErreur](../WDLang4/3044088.md) permet d'identifier l'erreur.




**`<Connexion> : (Variable de type Connexion)`**

Nom de la variable de type [Connexion](../WDLang4/1514073.md) décrivant la connexion dans laquelle le répertoire doit être supprimé.

**`<Répertoire à supprimer> : (Chaîne de caractères)`**

Chemin du répertoire à supprimer. Ce chemin est relatif à la base de données : le nom de la base de données ne doit pas être précisé dans le chemin. Ce répertoire doit être vide.

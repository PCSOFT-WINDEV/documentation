


## &lt;redisConnexion&gt;.Désabonne (Fonction)

***En anglais : &lt;redisConnection&gt;.Unsubscribe***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime un abonnement à un canal de diffusion Redis.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
gRedis.Désabonne("MonCanal")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Connexion>.Désabonne(<Canal>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'opération a été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Connexion> : (Variable de type redisConnexion)`**

Nom de la variable de type [redisConnexion](../WDLang4/1000023527.md) correspondant à la connexion au serveur Redis.

**`<Canal> : (Chaîne de caractères)`**

Nom du canal de diffusion pour lequel le désabonnement doit être effectué.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Opérations effectuées lors du désabonnement
<a name="operations_effectuees_lors_desabonnement_ELTPARAGRAPHE000181"></a>Lors du désabonnement, la procédure spécifiée dans la fonction [&lt;redisConnexion&gt;.Abonne](../WDLang4/1410088279.md) sera exécutée.


## hbCellule (Type de variable)

***En anglais : hbCell***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **hbCellule** permet de définir toutes les caractéristiques avancées du contenu d'une cellule (valeur de rubrique) d'une table HBase. Les caractéristiques de ce contenu peuvent être définies et modifiées à l'aide de différentes propriétés WLangage. 
Ce type de variable est utilisé par : 

- la fonction [hbLit](../WDLang4/1000021706.md) : le résultat de la fonction [hbLit](../WDLang4/1000021706.md) est un tableau de **hbCellule**.

- les variables de type [hbEcriture](../WDLang4/1000021710.md), pour spécifier les valeurs à écrire. 




**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connexion
Connexion est une hbConnexion
Connexion.Serveur = "MonServeur"
// Description des données à écrire
Ecriture est une hbEcriture
Ecriture.Ligne = "post1"
Cellule est une hbCellule 
Cellule.Colonne.Famille = "post"
Cellule.Colonne.Qualificatif = "titre"
Cellule.Valeur = "Ceci est mon message sur le blog."
Ajoute(Ecriture.Cellule, Cellule)
// Ecriture des données
hbEcrit(Connexion, "blog", Ecriture)
```

## Remarques
<a name="NOTE0_1"></a>


### Propriétés spécifiques à la description des variables de type hbCellule
<a name="proprietes_specifiques_description_des_variables_type_hbcellule_ELTPARAGRAPHE000043"></a>Les propriétés suivantes peuvent être utilisées pour manipuler un contenu d'une cellule :

| Nom de la propriété | Type manipulé | Effet |
| --- | --- | --- |
| Colonne | Variable de type [hbColonne](../WDLang4/1000021698.md) | Colonne associée à la cellule. |
| Timestamp | DateHeure | Horodatage de la valeur lue. |
| Valeur | Buffer | Contenu de la cellule. |

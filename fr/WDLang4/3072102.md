
## WEBDEV : Connecteur Natif AS/400 pour PHP
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000090"></a>
WEBDEV permet de créer des sites Internet ou Intranet, en mode natif ou en PHP. Les sites PHP peuvent accéder aux ressources de votre AS/400 ou iSeries.

**Rappel** : La création d'un site en PHP ne nécessite aucune connaissance particulière en PHP. La programmation est réalisée en WLangage et WEBDEV génère automatiquement le code PHP nécessaire.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Conditions nécessaires
<a name="conditions_necessaires_ELTTEXTE000114"></a>


### Sur le poste de développement :
<a name="sur_poste_developpement_ELTPARAGRAPHE000021"></a>

- Installer le Connecteur Natif AS/400. Le Connecteur Natif AS/400 permet d'importer simplement les fichiers AS/400 dans l'analyse du projet.

- Installer le framework PHP (scripts PHP) pour AS/400. Lorsque ce framework a été installé, il est recommandé de recompiler le projet WEBDEV.

- Installer le Connecteur Natif AS/400 pour PHP.  




Lorsque tous ces éléments sont installés, il est possible de développer et de tester des sites WEBDEV PHP accédant à des données AS/400.


<a name="NOTE2_2"></a>


### Sur le poste de déploiement :
<a name="sur_poste_deploiement_ELTPARAGRAPHE000033"></a>

- Installer le site WEBDEV PHP (par exemple avec WDDéploie ou par FTP).

- Installer le Connecteur Natif AS/400 pour PHP.




**Remarque** : En déploiement, plusieurs architectures peuvent être utilisées :

- Un serveur frontal (équipé d'un moteur PHP version 4 ou 5) à votre serveur iSeries ou AS/400 (v4 R/5 et supérieure).

- Installation du site WEBDEV PHP directement sur votre IBM System i (avec Zend Core for i5/OS).

- Un serveur frontal Windows ou Linux, avec un serveur d'application WEBDEV.

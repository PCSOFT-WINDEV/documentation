


## HModifiePropriétéServeur (Fonction)

***En anglais : HModifyServerProperty***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie les propriétés d'un serveur HFSQL.

**Remarque** : Il est possible de connaître les propriétés d'un serveur HFSQL grâce à la fonction [HInfoPropriétéServeur](../WDLang4/3044325.md).




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
ResModif est un booléen
ResModif = HModifiePropriétéServeur("MaConnexion", hNotes, "Serveur HFSQL Client/Serveur")
SI ResModif = Vrai ALORS
	Info("Modification effectuée")
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = HModifiePropriétéServeur(<Connexion> , <Propriété à modifier> , <Nouvelle valeur>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si les propriétés ont été modifiées,

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. La fonction [HErreurInfo](../WDLang4/3044071.md) permet d'identifier l'erreur.




**`<Connexion> : (Chaîne de caractères ou variable de type Connexion)`**

Connexion pour laquelle la valeur d'une propriété doit être modifiée. Cette connexion correspond : 

- soit au nom de la connexion définie sous l'éditeur d'analyses ou grâce aux fonctions [HDécritConnexion](../WDLang4/3044205.md) ou [HOuvreConnexion](../WDLang4/3044107.md).

- soit au nom d'une variable de type [Connexion](../WDLang4/1514073.md). 




**`<Propriété à modifier> : (Constante de type Entier)`**

Propriété du serveur à modifier : 


|   |   |
| --- | --- |
| *hNotes* | Description du serveur ou toute autre information associée au serveur. <br>Si cette constante est utilisée, &lt;Nouvelle valeur&gt; est une chaîne de caractères contenant le texte de la note. |



**`<Nouvelle valeur> : (Chaîne de caractères)`**

Nouvelle valeur de la propriété.

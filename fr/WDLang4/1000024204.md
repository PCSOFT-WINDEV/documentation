


## &lt;Source&gt;.CompareRubrique (Fonction)

***En anglais : &lt;Source&gt;.CompareItem***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Compare deux valeurs en les triant selon la rubrique d'index HFSQL spécifiée : toutes les options de tri spécifiées pour la rubrique sont prises en compte (sensibilité à la casse, la ponctuation, sens de tri, langue Unicode, ...). 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SELON Client.CompareRubrique(Nom, sNom1, sNom2)
	CAS 0
		Info("Les deux valeurs sont identiques.")
	CAS 1
		Info("Nom1 est plus grand.")
	CAS -1
		Info("Nom2 est plus grand.")
FIN
```

```wl
MonFichierTest.CompareRubrique(RubSensibleCasseCroissant, "A", "B")
// Renvoie -1 (comme ChaîneCompare)
MonFichierTest.CompareRubrique(RubNonSensibleCasseCroissant, "A", "B")
// Renvoie +1 (contrairement à ChaîneCompare)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Source>.CompareRubrique(<Rubrique de tri> , <Valeur 1> , <Valeur 2>)`
---

**`<Résultat> : (Entier)`**



- 0 si les valeurs sont égales.

- -1 si la première valeur est inférieure à la seconde valeur.

- 1 si la première valeur est supérieure à la seconde valeur.




**`<Source> : (Type correspondant à la source spécifiée)`**

Nom du fichier de données HFSQL contenant l'index décrivant le tri.

**`<Rubrique de tri> : (Chaîne de caractères)`**

Nom de la rubrique correspondant à l'index de tri.

**`<Valeur 1> : (Type de la valeur)`**

Première valeur à comparer.

**`<Valeur 2> : (Type de la valeur)`**

Deuxième valeur à comparer.




## MongoGridFSSupprimeFichierParID (Fonction)

***En anglais : MongoGridFSDeleteFileByID***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime un fichier identifié par son identifiant dans un système de fichiers MongoDB.

**Remarque** : Pour supprimer un fichier identifié à partir de son nom, utilisez la fonction [MongoGridFSSupprimeFichier](../WDLang4/1000022440.md). 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
gCnxMONGO est une mongoConnexion

fsVV est un mongoGridFS
fsVV.Préfixe = "fs"
fsVV.BaseDeDonnées = gCnxMONGO.BaseDeDonnées[SAI_BASE]

tabFichier = MongoGridFSListeFichier(fsVV)
fic est un mongoGridFSInfoFichier
i est entier
POUR TOUT fic DE tabFichier
	i++
	Trace("-------------------------------------------------------------")
	Trace("n°: " + i)
	Trace("ID : " + VariantVersJSON(fic.ID))
	Trace("Nom : " + fic.Nom)
	MongoGridFSSupprimeFichierParID(fsVV, fic.ID)
	SI ErreurDétectée ALORS
		Info("Pb dans MongoGridFSSupprimeFichierParID : ", ErreurInfo(errComplet))
	FIN
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = MongoGridFSSupprimeFichierParID(<Système de fichiers> , <Identifiant du fichier>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'opération a été réalisée,

- <u><u><u><u>Faux</u></u></u></u> en cas de problème. La fonction [ErreurInfo](../WDLang1/3013008.md) permet d'identifier l'erreur.




**`<Système de fichiers> : (Variable de type mongoGridFS ou de type mongoBase)`**

Système de fichiers de la base MongoDB à manipuler. Correspond : 

- au nom de la variable de type [mongoGridFS](../WDLang4/1000022416.md) correspondant au système de fichiers Mongo.

- au nom de la variable de type [mongoBase](../WDLang4/1000022410.md) correspondant à la base MongoDB. Dans ce cas, le système de fichiers utilise le préfixe "fs". 




**`<Identifiant du fichier> : (Variant)`**

Identifiant du fichier à supprimer.

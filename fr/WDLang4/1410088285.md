


## &lt;redisConnexion&gt;.PipelineRécupère (Fonction)

***En anglais : &lt;redisConnection&gt;.PipelineGet***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Récupère le résultat de l'exécution d'une commande Redis ayant été ajouté par [&lt;redisConnexion&gt;.PipelineAjoute](../WDLang4/1410088283.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
gRedis.PipelineAjoute("set p1 pv1")
gRedis.PipelineAjoute("set p2 pv2")
gRedis.PipelineAjoute("get p2")
gRedis.PipelineAjoute("get p1")

r1 est un redisRéponse = gRedis.PipelineRécupère()
r2 est un redisRéponse = gRedis.PipelineRécupère()
r3 est un redisRéponse = gRedis.PipelineRécupère()
r4 est un redisRéponse = gRedis.PipelineRécupère()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Connexion>.PipelineRécupère()`
---

**`<Résultat> : (Variable de type redisRéponse)`**

Variable de type [redisRéponse](../WDLang4/1000023534.md) contenant le résultat de la commande Redis.

**`<Connexion> : (Variable de type redisConnexion)`**

Nom de la variable de type [redisConnexion](../WDLang4/1000023527.md) correspondant à la connexion au serveur Redis.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Fonctionnement du pipeline
<a name="fonctionnement_pipeline_ELTPARAGRAPHE000178"></a>Lorsque l'ensemble des commandes souhaitées auront été ajoutées dans le pipeline, les résultats pourront être récupérés avec la fonction [&lt;redisConnexion&gt;.PipelineRécupère](../WDLang4/1410088285.md).

Pour ne pas récupérer les résultats, il est possible d'utiliser la fonction [&lt;redisConnexion&gt;.PipelineRAZ](../WDLang4/1410088284.md).

En l'absence de l'appel à la fonction [&lt;redisConnexion&gt;.PipelineRAZ](../WDLang4/1410088284.md), il doit y avoir autant d'appels à la fonction **&lt;redisConnexion&gt;.PipelineRécupère** que d'appels à la fonction [&lt;redisConnexion&gt;.PipelineAjoute](../WDLang4/1410088283.md).

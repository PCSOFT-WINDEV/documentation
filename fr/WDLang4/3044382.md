
## H.CréationLangueUnicode (Variable)

***En anglais : H.LanguageCreationUnicode***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de définir les caractéristiques de tri des rubriques Unicode d'un fichier de données. Cette variable doit être utilisée avant la création des fichiers de données (fonctions [HCréation](../WDLang4/3044255.md) et [HCréationSiInexistant](../WDLang4/3044254.md)). 




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
H.CréationLangueUnicode = "AR" + TAB + "MA"
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`H.CréationLangueUnicode(<Caractéristiques de la langue>)`
---

**`<Caractéristiques de la langue> : (Chaîne de caractères)`**



- Caractéristiques de la langue à utiliser pour le tri, de la forme : 
	
	```txt
	<Identifiant de la langue> + TAB + <Identifiant du pays> + TAB + <Variante>
	```


- Constante permettant de définir un tri spécifique : 
	


|   |   |
| --- | --- |
| *hLangueTriBinaire* | Le tri sera effectué en comparant la valeur binaire des octets. Ce type de tri est très rapide. Il est conseillé par exemple pour effectuer une recherche à l'identique. |
| *hLangueTriRapide* | Le tri sera effectué en comparant les caractères un à un selon un tri générique. |

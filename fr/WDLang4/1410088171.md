


## RedisRécupère (Fonction)

***En anglais : RedisGet***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Récupère la valeur d'une clé sur un serveur Redis sous forme de chaîne.

Remarque : Pour récupérer la valeur d'une clé sous forme de chaîne binaire, utilisez la fonction [RedisExécuteCommande](../WDLang4/1000023541.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
gRedis est un redisConnexion
gRedis.Serveur = "MonServeurRedis:7001"

RedisAffecte(gRedis, "MaClé", "MaValeur", 1h)
Trace(RedisRécupère(gRedis, "MaClé"))
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = RedisRécupère(<Connexion> , <Clé>)`
---

**`<Résultat> : (Chaîne de caractères)`**



- Valeur de la clé.

- Null, si la clé n'a pas de valeur. 




**`<Connexion> : (Variable de type redisConnexion)`**

Nom de la variable de type [redisConnexion](../WDLang4/1000023527.md) correspondant à la connexion au serveur Redis.

**`<Clé> : (Chaîne de caractères)`**

Nom de la clé à récupérer.

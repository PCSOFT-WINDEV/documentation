


## Modification automatique des fichiers de données : nouveau format de date
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000218"></a>


### A partir de WINDEV 7 et de WEBDEV 7 version 149
<a name="partir_windev_7_webdev_7_version_149_ELTPARAGRAPHE000011"></a>

Désormais les formats disponibles pour les dates et les heures stockées dans des rubriques sont :


|   |   |
| --- | --- |
| **Formats disponibles pour les rubriques** | **Description** |
| Date au format AAAAMMJJ | Format de date disponible en WINDEV 5.5, WINDEV 7.5 et versions supérieures. |
| Date et Heure | Nouveau format de date, permettant de mémoriser à la fois la date et l'heure : AAAAMMJJHHmmSSCCC (précision : Millièmes de seconde) |
| Durée | Nouveau format, permettant de gérer les durées : +JHHMMSSCCC (précision : Millièmes de seconde) |
| Heure au format HHMM | Format d'heure disponible en WINDEV 5.5, WINDEV 7.5 et versions supérieures. |

## Que va faire la modification automatique des données ?
<a name="que_faire_modification_automatique_des_donnees_ELTTEXTE000242"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png) 

Tableau de conversion lors du passage d'un "ancien format" à un nouveau format lors de la modification automatique des fichiers de données.


|   |   |   |   |   |   |   |
| --- | --- | --- | --- | --- | --- | --- |
| Nouveau format | Heure | Date | Date/Heure | Durée | Chaîne / <br>Binaire /<br> Mémo | Numérique et monétaire |
| Ancien Format |
| Heure | X | 0 | Date/Heure<br>L'heure initialisée avec la valeur <br>La date est initialisée à 00/00/00 (date invalide) | Durée | Chaîne | 0 |
| Date | 0 | X | Date/Heure<br>L'heure initialisée avec 00 : 00<br>La date est initialisée avec la valeur | 0 | Chaîne | 0 |
| Date/Heure | Heure | Date | X | 0 | Chaîne | 0 |
| Durée | Heure | 0 | Date/Heure<br>L'heure initialisée avec la valeur <br>La date est initialisée à 00/00/00 (date invalide) | X | Chaîne | 0 |
| Chaîne/Binaire/Mémo | Heure | Date | 0 | 0 | X | Conversion du langage |
| Numérique et monétaire |   | 0 | 0 | Durée | Chaîne | X |


**Remarque** : Dans le cas d'une installation réseau, la modification automatique des données modifie à la fois les fichiers de données présents sur le poste serveur et ceux présents sur les postes utilisateurs.

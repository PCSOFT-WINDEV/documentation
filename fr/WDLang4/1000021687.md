
## hbListeTable (Fonction)

***En anglais : hbListTable***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Récupère la liste des tables présentes sur un serveur HBase.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connexion
Connexion est une hbConnexion
Connexion.Serveur = "MonServeur"
// Affichage de la liste des tables
Trace(hbListeTable(Connexion))
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = hbListeTable(<Connexion>)`
---

**`<Résultat> : (Chaîne de caractères)`**

Liste des tables présentes sur le serveur HBase. Cette liste est de la forme : 
```txt
<Nom de la table 1> + RC + <Nom de la table 2> + RC + ...
```


**`<Connexion> : (Variable de type hbConnexion)`**

Nom de la variable de type [hbConnexion](../WDLang4/1000021685.md) correspondant à la connexion au serveur HBase.

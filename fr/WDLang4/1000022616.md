<a name="NOTE1_1"></a>
Les constantes utilisées par les fonctions Mongoxxx sont les suivantes :

| Constante | Détail | Fonction |
| --- | --- | --- |
| **moeDéfaut** | Nombre de noeuds minimum devant avoir acquittés le traitement pour considérer l'écriture effectuée : un seul noeud doit avoir acquitté le traitement pour que l'écriture doit être considérée comme effectuée. | Type [mongoOptionEcriture](../WDLang4/1000022597.md) |
| **moeMajorité** | Nombre de noeuds minimum devant avoir acquittés le traitement pour considérer l'écriture effectuée : la majorité des noeuds doit avoir acquitté le traitement pour que l'écriture doit être considérée comme effectuée. | Type [mongoOptionEcriture](../WDLang4/1000022597.md) |
| **moeSansAcquittement** | Nombre de noeuds minimum devant avoir acquittés le traitement pour considérer l'écriture effectuée : Aucun noeud ne doit avoir acquitté le traitement pour que l'écriture doit être considérée comme effectuée. Attention, dans ce cas, aucune erreur n'est renvoyée. | Type [mongoOptionEcriture](../WDLang4/1000022597.md) |
| **moeTag** | Nombre de noeuds minimum devant avoir acquittés le traitement pour considérer l'écriture effectuée : le nombre de noeuds est celui défini dans une pré-configuration définie sure le serveur. | Type [mongoOptionEcriture](../WDLang4/1000022597.md) |
| **mongoAjouteSiInexistant** | Permet d'ajouter un document s'il n'existe pas dans la collection. | [MongoModifieTout](../WDLang4/1000022603.md)<br>[MongoModifieUn](../WDLang4/1000022443.md) |
| **mongoContinueSiErreur** | Permet d'ajouter le document suivant même si le document précédent a provoqué une erreur. | [MongoAjoute](../WDLang4/1000022432.md) |
| **mongoRenvoieValeurAprès** | Permet de renvoyer le document après modification. Par défaut, si cette constante n'est pas utilisée, le document renvoyé correspond au document avant modification. | [MongoModifieUn](../WDLang4/1000022443.md) |
| **mongoSansValidation** | N'effectue pas de vérification de la structure du document. | [MongoAjoute](../WDLang4/1000022432.md)<br>[MongoModifieTout](../WDLang4/1000022603.md) |




## &lt;redisConnexion&gt;.Récupère (Fonction)

***En anglais : &lt;redisConnection&gt;.Get***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Récupère la valeur d'une clé sur un serveur Redis sous forme de chaîne.

Remarque : Pour récupérer la valeur d'une clé sous forme de chaîne binaire, utilisez la fonction [&lt;redisConnexion&gt;.ExécuteCommande](../WDLang4/1410088282.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
gRedis est un redisConnexion
gRedis.Serveur = "MonServeurRedis:7001"

gRedis.Affecte("MaClé", "MaValeur", 1h)
Trace(gRedis.Récupère("MaClé"))
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Connexion>.Récupère(<Clé>)`
---

**`<Résultat> : (Chaîne de caractères)`**



- Valeur de la clé.

- Null, si la clé n'a pas de valeur.




**`<Connexion> : (Variable de type Connexion)`**

Nom de la variable de type [redisConnexion](../WDLang4/1000023527.md) correspondant à la connexion au serveur Redis.

**`<Clé> : (Chaîne de caractères)`**

Nom de la clé à récupérer.

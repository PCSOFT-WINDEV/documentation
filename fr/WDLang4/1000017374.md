


## HModifieTâche (Fonction)

***En anglais : HModifyTask***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie une tâche planifiée sur le serveur HFSQL défini par la connexion. Cette tâche est présente dans une variable de type [hTâchePlanifiée](../WDLang4/1000017448.md).

**Remarque** : La fonction [HInfoTâche](../WDLang4/1000017409.md) permet de récupérer les caractéristiques d'une tâche planifiée dans une variable de type [hTâchePlanifiée](../WDLang4/1000017448.md).




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MaConnexion est une Connexion
MaTâche est une hTâchePlanifiée

// Récupération des caractéristiques de la tâche
MaTâche = HInfoTâche(MaConnexion, IDTâche)
// Modification de la tâche
MaTâche.Heure = SAI_Heure.Heure+1
HModifieTâche(MaConnexion, IDTâche, MaTâche)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = HModifieTâche(<Connexion> , <Identifiant de la tâche> , <Tâche planifiée>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la tâche planifiée a été modifiée, 

- <u><u><u><u>Faux</u></u></u></u> en cas d'erreur.




**`<Connexion> : (Chaîne de caractères ou variable de type Connexion)`**

Connexion pour laquelle une tâche doit être ajoutée. Cette connexion correspond : 

- soit au nom de la connexion définie sous l'éditeur d'analyses ou grâce aux fonctions [HDécritConnexion](../WDLang4/3044205.md) ou [HOuvreConnexion](../WDLang4/3044107.md).

- soit au nom d'une variable de type [Connexion](../WDLang4/1514073.md). 




**`<Identifiant de la tâche> : (Entier)`**

Identifiant de la tâche à modifier. Cet identifiant est renvoyé par la fonction [HAjouteTâche](../WDLang4/1000017091.md).

**`<Tâche planifiée> : (Variable de type hTâchePlanifiée)`**

Nom de la variable de type [hTâchePlanifiée](../WDLang4/1000017448.md) contenant les modifications. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Droits nécessaires
<a name="droits_necessaires_ELTPARAGRAPHE000112"></a>Pour modifier une tâche, il est nécessaire de posséder :

- le droit correspondant (constante *hDroitGèreTâche*) sur le serveur,

- le droit d'exécuter la commande liée à la tâche planifiée.




La tâche sera exécutée avec le compte de l'utilisateur qui a créé la tâche.

**Remarque** : Si le serveur est éteint, la tâche planifiée ne sera pas réalisée. La tâche planifiée sera réalisée à la prochaine date disponible lorsque le serveur sera allumé.




## HMaintenanceServeur (Fonction)

***En anglais : HServerMaintenance***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Vérifie et optimise les tables systèmes d'un serveur HFSQL.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
HMaintenanceServeur(MaConnexion)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = HMaintenanceServeur(<Connexion> [, <Base de données>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si les tables système ont été vérifiées et optimisées, 

- <u><u><u><u>Faux</u></u></u></u> en cas d'erreur. La fonction [HErreur](../WDLang4/3044088.md) permet d'identifier l'erreur.




**`<Connexion> : (Chaîne de caractères ou variable de type Connexion)`**

Connexion pour laquelle les tables système doivent être vérifiées et optimisées. Cette connexion correspond : 

- soit au nom de la connexion définie sous l'éditeur d'analyses ou grâce aux fonctions [HDécritConnexion](../WDLang4/3044205.md) ou [HOuvreConnexion](../WDLang4/3044107.md).

- soit au nom d'une variable de type [Connexion](../WDLang4/1514073.md).




**`<Base de données> : (Chaîne de caractères optionnelle)`**

Nom de la base de données contenant les tables système à vérifier et optimiser. Si ce paramètre n'est pas précisé, cette fonction vérifiera et optimisera les tables système globales.

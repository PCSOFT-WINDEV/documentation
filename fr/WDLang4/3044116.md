


## HStatHeure (Fonction)

***En anglais : HStatTime***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie l'heure de la dernière mise à jour des statistiques d'index. Ces statistiques ont été réalisées grâce aux fonctions [HStatCalcule](../WDLang4/3044134.md) ou [HRéindexe](../WDLang4/3044133.md).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Réalisation de statistiques sur la clé NomCli du fichier de données Client
HStatCalcule(Client, NomCli)

// Connaître la date de réalisation de ces statistiques
ResHeure = HStatHeure()
Info("Les statistiques effectuées sur la rubrique NomCli " + ...
	"ont été réalisées à : " + ...
	HeureVersChaîne(ResHeure))
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = HStatHeure([<Fichier de données> [, <Rubrique clé>]])`
---

**`<Résultat> : (Chaîne de caractères)`**

Heure au format HHMMSS (HH correspond aux heures, MM aux minutes et SS aux secondes). L'heure renvoyée est au format UTC (Universal Time Coordinated).

**`<Fichier de données> : (Chaîne de caractères optionnelle)`**

Nom du fichier de données manipulé. Si ce nom n'est pas spécifié, la fonction **HStatHeure** manipule le dernier fichier de données utilisé par la dernière fonction de gestion HFSQL (fonction commençant par la lettre H).

**`<Rubrique clé> : (Chaîne de caractères optionnelle)`**

Nom de la rubrique clé sur laquelle les statistiques ont été effectuées. Si ce nom n'est pas spécifié, la fonction **HStatHeure** manipule la dernière rubrique clé utilisée par la dernière fonction de gestion HFSQL (fonction commençant par la lettre H).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Dans les fichiers de données HFSQL, seules les rubriques clés peuvent faire l'objet d'un calcul de statistiques.

- Cette fonction n'est pas disponible lors de l'utilisation d'une base de données via .

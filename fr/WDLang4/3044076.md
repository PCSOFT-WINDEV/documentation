
## H.NumEnr (Variable)

***En anglais : H.NumRec***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Indique le numéro de l'enregistrement physique chargé en mémoire (c'est-à-dire le numéro du dernier enregistrement lu).

Cette variable est mise à jour à chaque utilisation d'une fonction de lecture ([HLitPremier](../WDLang4/3044051.md), [HLitSuivant](../WDLang4/3044037.md), [HLit](../WDLang4/3044047.md), ...)


<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Utilisation de H.NumEnr
<a name="utilisation_hnumenr_ELTPARAGRAPHE000031"></a>La variable **H.NumEnr** peut être utilisée sur : 

- les fichiers de données HFSQL.

- les vues HFSQL.

- les requêtes sur les fichiers de données HFSQL. 




Dans ces deux derniers cas, la variable **H.NumEnr** correspond au numéro d'enregistrement du fichier de données associé à la vue ou à la requête.

Sur les fichiers xBase, la variable **H.NumEnr** est toujours égale à 0.

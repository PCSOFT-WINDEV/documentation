
## SQLLitMémo (Fonction)

***En anglais : SQLGetMemo***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Récupère le contenu d'une colonne de type Mémo dans le résultat de la requête, pour la ligne en cours.

**Attention** : Cette fonction doit être utilisée lors d'un parcours du résultat de la requête de type "SQLAvance/SQLLitCol". Pour plus de détails sur les différents types de parcours du résultat d'une requête SQL, consultez [Types de parcours SQL](../WDLang4/3072025.md).




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SQLExec("SELECT NOM, PRENOM, POSTE, PHOTO FROM CLIENT", "REQ1")
SQLAvance("REQ1") // Ou SQLFetch("REQ1")
// Récupérer les données
NOM = SQLLitCol("REQ1", 1)
PRENOM = SQLLitCol("REQ1", 2)
POSTE = SQLLitCol("REQ1", 3)
// Lire le mémo binaire et le stocker dans un fichier
// Photo du produit
HExtraitMemo(SQLLitMémo("REQ1", 4), "Produit.BMP")
SQLFerme("REQ1")
// Récupération d'un mémo binaire
SQLExec("SELECT NOM, MORCEAU FROM MUSIQUE", "REQ1")
// Joue le mémo binaire
Son(SQLLitMémo("REQ1", 2))
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = SQLLitMémo(<Nom de la requête> , <Numéro de la colonne>)`
---

**`<Résultat> : (Chaîne de caractères)`**

Mémo binaire.

**`<Nom de la requête> : (Chaîne de caractères)`**

Nom de la requête créée et exécutée avec la fonction [SQLExec](../WDLang4/3072007.md), ou exécutée avec la fonction [SQLExecWDR](../WDLang4/3072010.md).

**`<Numéro de la colonne> : (Entier)`**

Numéro de la colonne de type mémo à récupérer. Cette colonne correspond à une colonne de la requête (et non pas à une colonne de la table).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Ligne en cours de la requête
<a name="ligne_cours_requete_ELTPARAGRAPHE000085"></a>La ligne en cours est positionnée par la fonction [SQLAvance](../WDLang4/3072011.md) (également nommée SQLFetch).


<a name="NOTE0_2"></a>




### Colonnes de la requête
<a name="colonnes_requete_ELTPARAGRAPHE000097"></a>Pour lire les colonnes autres que les colonnes de type Mémo, utilisez la fonction [SQLLitCol](../WDLang4/3072019.md).


<a name="NOTE0_3"></a>




### Utilisation d'un mémo binaire
<a name="utilisation_memo_binaire_ELTPARAGRAPHE000109"></a>Le mémo binaire lu par la fonction **SQLLitMémo** peut être :

- Joué avec la fonction [Son](../WDLang1/3013047.md) si le mémo binaire est un fichier de type .WAV.

- Affiché dans un champ Image si le mémo binaire est au format BMP.

- Stocké dans un fichier grâce à la fonction [HExtraitMémo](../WDLang4/3044072.md).

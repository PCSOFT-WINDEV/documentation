


## HInfoPropriétéBaseDeDonnées (Fonction)

***En anglais : HInfoDatabaseProperty***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de connaître les propriétés d'une base de données située sur un serveur HFSQL.



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Description de la connexion
HDécritConnexion("MaConnexion", "Julie", "MotDePasse", ...
	"MonServeurHF", "MaBaseDeDonnées", hAccèsHFClientServeur, HOLecture)
// Ouverture de la connexion
HOuvreConnexion("MaConnexion")
...
NomPropriétaire est une chaîne
NomPropriétaire = HInfoPropriétéBaseDeDonnées("MaConnexion", ...
	hPropriétaire, "MaBaseDeDonnées")
SI NomPropriétaire <> "" ALORS 
	Trace("Propriétaire de la base de donnée : " + NomPropriétaire)
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = HInfoPropriétéBaseDeDonnées(<Connexion> , <Propriété> , <Base de données>)`
---

**`<Résultat> : (Chaîne de caractères)`**



- Valeur de la propriété demandée, 

- Chaîne vide ("") en cas d'erreur.




**`<Connexion> : (Chaîne de caractères ou variable de type Connexion)`**

Connexion pour laquelle la valeur d'une propriété est recherchée. Cette connexion correspond : 

- soit au nom de la connexion définie sous l'éditeur d'analyses ou grâce aux fonctions [HDécritConnexion](../WDLang4/3044205.md) ou [HOuvreConnexion](../WDLang4/3044107.md).

- soit au nom d'une variable de type [Connexion](../WDLang4/1514073.md).




**`<Propriété> : (Constante de type Entier)`**

Propriété de la base de données recherchée : 


|   |   |
| --- | --- |
| *hApplication* | Nom de l'application à partir de laquelle la base de données a été créée. <br>Si cette constante est utilisée, &lt;Résultat&gt; est une chaîne de caractères de la forme &lt;Nom de l'exécutable&gt;(&lt;Nom du projet&gt;). |
| *hCréateur* | Nom du créateur de la base de données. Cette information est purement informative et n'a pas d'influence. <br>Si cette constante est utilisée, &lt;Résultat&gt; est une chaîne de caractères. |
| *hDate* | Date de la création de la base de données. <br>Si cette constante est utilisée, &lt;Résultat&gt; est une chaîne de caractères de la forme AAAAMMJJHHmmSSCCC. |
| *hMachine* | Nom du poste à partir duquel a été créé l'élément. <br>Si cette constante est utilisée, &lt;Résultat&gt; est de la forme &lt;Nom du poste client&gt; + TAB + &lt;Adresse IP du poste Client&gt;. |
| *hNotes* | Description de la base de données ou toute autre information associée à la base de données grâce à la fonction [HModifiePropriétéBaseDeDonnées](../WDLang4/3044329.md). <br>Si cette constante est utilisée, &lt;Résultat&gt; est une chaîne de caractères. |
| *hPropriétaire* | Propriétaire de la base de données. <br>Le propriétaire est celui qui possède la base de données. Le propriétaire d'une base de données peut être modifié via la fonction [HModifiePropriétéBaseDeDonnées](../WDLang4/3044329.md). <br>Si cette constante est utilisée, &lt;Résultat&gt; est une chaîne de caractères. |



**`<Base de données> : (Chaîne de caractères)`**

Nom de la base de données pour laquelle les propriétés sont recherchées. 
Si ce paramètre correspond à une chaîne vide (""), la base de données utilisée sera celle définie dans la connexion. 
Si aucune base de données n'est définie dans la connexion, une erreur WLangage sera générée. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Pour modifier les propriétés d'une base de données, utilisez la fonction [HModifiePropriétéBaseDeDonnées](../WDLang4/3044329.md).

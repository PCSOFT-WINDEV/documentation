


## HVérifieMiseAJourServeur (Fonction)

***En anglais : HCheckServerUpdate***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Vérifie s'il existe une version plus récente du serveur HFSQL pouvant être mise à jour automatiquement. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MaConnexion est une Connexion

soit sNouvelleVersion = HVérifieMiseAJourServeur(MaConnexion)
SI sNouvelleVersion<>"" ALORS Info(sNouvelleVersion)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = HVérifieMiseAJourServeur(<Connexion>)`
---

**`<Résultat> : (Chaîne de caractères)`**

Version de la mise à jour disponible sur le serveur HFSQL. 

**`<Connexion> : (Chaîne de caractères ou variable de type Connexion)`**

Connexion au serveur HFSQL. Cette connexion correspond :

- soit au nom de la connexion définie sous l'éditeur d'analyses ou grâce aux fonctions [HDécritConnexion](../WDLang4/3044205.md) ou [HOuvreConnexion](../WDLang4/3044107.md).

- soit au nom d'une variable de type [Connexion](../WDLang4/1514073.md).







## Remarques
Le serveur de mise à jour (contenant le pack de mise à jour du serveur HFSQL) peut être défini par la fonction [HGèreMiseAJourServeur](../WDLang4/1000022635.md).




## MongoGridFSRécupèreInfoFichier (Fonction)

***En anglais : MongoGridFSGetFileInfo***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de récupérer les informations concernant un fichier MongoDB (gridFS).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
cnxMDB est une mongoConnexion("srv-mongodb://db1.exemple.net")
Fi est un mongoGridFSInfoFichier
Fi = MongoGridFSRécupèreInfoFichier(cnxMDB.BaseDeDonnées["CRM"], "produit.png")
Trace(Fi.Taille)
Trace(Fi.TypeContenu)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = MongoGridFSRécupèreInfoFichier(<Système de fichiers> , <Nom du fichier>)`
---

**`<Résultat> : (Variable de type mongoGridFSInfoFichier)`**

Variable de type [MongoGridFSInfoFichier](../WDLang4/1000022417.md) contenant les information sur le fichier. 

**`<Système de fichiers> : (Variable de type mongoGridFS ou de type mongoBase)`**

Système de fichiers de la base MongoDB à manipuler. Correspond : 

- au nom de la variable de type [mongoGridFS](../WDLang4/1000022416.md) correspondant au système de fichiers Mongo.

- au nom de la variable de type [mongoBase](../WDLang4/1000022410.md) correspondant à la base MongoDB. Dans ce cas, le système de fichiers utilise le préfixe "fs". 




**`<Nom du fichier> : (Chaîne de caractères)`**

Nom du fichier présent dans le système de fichiers MongoDB dont les informations doivent être récupérées.

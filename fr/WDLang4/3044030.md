


## HStatNbEnr (Fonction)

***En anglais : HStatNbRec***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le nombre d'entrées pour une rubrique clé donnée. Ce nombre a été calculé lors du dernier calcul de statistiques.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Réalisation de statistiques sur la clé NomCli du fichier de données Client
ResultatStat = HStatNbEnr(Client, NomCli)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = HStatNbEnr([<Fichier de données> [, <Rubrique clé>]])`
---

**`<Résultat> : (Entier)`**

Nombre d'entrées trouvées pour la rubrique spécifiée. 
Correspond à la constante *hNumEnrInvalide* si :

- les statistiques n'ont jamais été calculées pour la rubrique spécifiée.

- les statistiques ne sont pas supportées pour le type de la rubrique spécifiée. HFSQL permet de calculer des statistiques uniquement sur des clés.




**`<Fichier de données> : (Chaîne de caractères optionnelle)`**

Nom du fichier de données manipulé. Si ce nom n'est pas spécifié, la fonction **HStatNbEnr** va manipuler le dernier fichier de données utilisé par la dernière fonction de gestion HFSQL (fonction commençant par la lettre H).

**`<Rubrique clé> : (Chaîne de caractères optionnelle)`**

Nom de la rubrique clé permettant de réaliser les statistiques. Si ce nom n'est pas spécifié, la fonction **HStatNbEnr** va manipuler la dernière rubrique utilisée par la dernière fonction de gestion HFSQL (fonction commençant par la lettre H).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- La mise à jour des statistiques peut être réalisée par les fonctions [HRéindexe](../WDLang4/3044133.md) et [HStatCalcule](../WDLang4/3044134.md).

- Dans les fichiers de données HFSQL Classic, seules les rubriques clés peuvent faire l'objet d'un calcul de statistiques.

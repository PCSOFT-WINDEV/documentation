


## &lt;Variable Connexion&gt;.SupprimeCollection (Fonction)

***En anglais : &lt;Connection variable&gt;.DeleteSet***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime une collection de procédures stockées d'un serveur HFSQL.



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Description de la connexion
MaConnexion est une Connexion

// Supprimer la collection de procédures "GestionMail" du serveur
MaConnexion.SupprimeCollection(GestionMail)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Connexion>.SupprimeCollection(<Collection de procédures>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la collection de procédures a été supprimée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. La fonction [HErreur](../WDLang4/3044088.md) permet d'identifier l'erreur.




**`<Connexion> : (Variable de type Connexion)`**

Nom de la variable de type [Connexion](../WDLang4/1514073.md) décrivant la connexion vers le serveur HFSQL contenant la collection de procédures à supprimer.

**`<Collection de procédures> : (Chaîne de caractères)`**

Nom de la collection de procédures à supprimer du serveur.




## &lt;Variable Connexion&gt;.AnnuleSauvegarde (Fonction)

***En anglais : &lt;Connection variable&gt;.CancelBackup***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Annule une sauvegarde en cours.  



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Connexion>.AnnuleSauvegarde(<Sauvegarde à annuler>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la sauvegarde a été annulée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. La fonction [HErreurInfo](../WDLang4/3044071.md) permet d'identifier l'erreur.




**`<Connexion> : (Variable de type Connexion)`**

Nom de la variable de type [Connexion](../WDLang4/1514073.md) décrivant la connexion vers le serveur qui réalise la sauvegarde.

**`<Sauvegarde à annuler> : (Entier)`**

Identifiant de la sauvegarde à annuler. Cet identifiant est fourni par la fonction [&lt;Variable Connexion&gt;.Sauvegarde](../WDLang4/1000022752.md).

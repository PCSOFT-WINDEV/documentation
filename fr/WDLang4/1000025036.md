


## &lt;Source&gt;.MigreCléComposéeReliée (Fonction)

***En anglais : &lt;Source&gt;.MigrateLinkedCompositeKey***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Migre les valeurs des clés composées reliées provenant d'un fichier au format Hyper File 5.5 vers le format HFSQL Classic.


|   |
| --- |
| **Quand utiliser cette fonction ?**<br><br>Le format des clés composées a changé entre Hyper File 5.5 et HFSQL Classic (monoposte ou réseau).<br><br>Conséquence : la valeur des clés reliées correspondant à une clé composée n'est plus correcte après migration des fichiers du format Hyper File 5.5 au format HFSQL Classic. La fonction **&lt;Source&gt;.MigreCléComposéeReliée** permet de modifier la valeur des clés reliées et de retrouver la valeur de la clé composée dans la clé reliée.<br><br>Conditions d'utilisation :<br><br>Cette fonction doit être utilisée :<br><br>- uniquement avec des fichiers au format HFSQL Classic (fichier au format 5.5 migré au format Classic),<br><br>- une seule fois par clé à migrer<br><br>- juste après la migration de Hyper File 5.5 vers HFSQL Classic. En effet, si des enregistrements sont ajoutés entre la migration et l'utilisation de la fonction **&lt;Source&gt;.MigreCléComposéeReliée**, les valeurs des nouveaux enregistrements pourront être corrompues.<br><br><br> |


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Modifier les valeurs de la clé Commande.ClecompClient
Client.MigreCléComposéeReliée(Clecomp, Commande, ClecompClient)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Source>.MigreCléComposéeReliée(<Clé composée source> , <Fichier relié> , <Clé reliée> [, <Jauge>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'opération a été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> en cas de problème. La fonction [HErreur](../WDLang4/3044088.md) permet d'obtenir plus de détails sur le problème rencontré.




**`<Source> : (Type correspondant à la source spécifiée)`**

Nom du fichier source de la liaison, contenant la description de la clé composée.

**`<Clé composée source> : (Chaîne de caractères)`**

Nom de la rubrique de type "Clé composée" dans le fichier source de la liaison.

**`<Fichier relié> : (Chaîne de caractères)`**

Nom du fichier relié contenant la valeur de la clé composée à migrer

**`<Clé reliée> : (Chaîne de caractères)`**

Nom de la rubrique de clé dans le fichier relié. La valeur de cette rubrique doit être migrée.

**`<Jauge> : (Nom de fenêtre ou nom de champ optionnel)`**

Nom du champ de type Jauge ou nom de la fenêtre permettant de visualiser la progression de la migration.




## &lt;Variable Connexion&gt;.ModifieSauvegardePlanifiée (Fonction)

***En anglais : &lt;Connection variable&gt;.ModifyScheduledBackup***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie une planification de sauvegarde. Cette planification a été décrite grâce à une variable de type [hDescriptionSauvegarde](../WDLang4/1000017455.md). 



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Description de la connexion
MaConnexionBase est une Connexion

tabSvg est un tableau dynamique de hDescriptionSauvegarde 
tabSvg = MaConnexionBase.ListeSauvegardePlanifiée()
hSvg est une hDescriptionSauvegarde

POUR TOUT hSvg DE tabSvg
	SI hSvg.LimiteNombreSauvegarde = 3 ALORS 
		hSvg.LimiteNombreSauvegarde = 5
		MaConnexionBase.ModifieSauvegardePlanifiée(hSvg)
	FIN
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Connexion>.ModifieSauvegardePlanifiée(<Sauvegarde planifiée>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la sauvegarde planifiée a été modifiée, 

- <u><u><u><u>Faux</u></u></u></u> en cas d'erreur.




**`<Connexion> : (Variable de type Connexion)`**

Nom de la variable de type [Connexion](../WDLang4/1514073.md) décrivant la connexion pour laquelle une sauvegarde planifiée doit être modifiée.

**`<Sauvegarde planifiée> : (Variable de type hDescriptionSauvegarde)`**

Nom de la variable de type [hDescriptionSauvegarde](../WDLang4/1000017455.md) à modifier.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Droits nécessaires
<a name="droits_necessaires_ELTPARAGRAPHE000221"></a>Pour avoir le droit de modifier une sauvegarde planifiée, il est nécessaire : 

- soit d'être le propriétaire de la sauvegarde planifiée. 

- soit d'être super-utilisateur.

<a name="NOTE1_1"></a>

[Afficher uniquement les constantes utilisées par les syntaxes non préfixées](#" onclick="$('.objet').hide();$('.pas_objet').show();)Les constantes utilisées par la syntaxe classique des fonctions Redisxxx sont les suivantes :

| Constante | Détail | Fonction |
| --- | --- | --- |
| **redisAbonnement** | L'application vient de s'abonner au canal spécifié. | [RedisAbonne](../WDLang4/1000023552.md) |
| **redisDéconnexion** | La connexion au serveur Redis a été perdue. | [RedisAbonne](../WDLang4/1000023552.md) |
| **redisDésabonnement** | L'application vient de se désabonner du canal spécifié. | [RedisAbonne](../WDLang4/1000023552.md) |
| **redisMessage** | L'application vient de recevoir un message. | [RedisAbonne](../WDLang4/1000023552.md) |
| **redisTypeBuffer** | Type du résultat : Le résultat correspond à un buffer. | Type [redisRéponse](../WDLang4/1000023534.md) |
| **redisTypeEntier** | Type du résultat : Le résultat correspond à un entier. | Type [redisRéponse](../WDLang4/1000023534.md) |
| **redisTypeErreur** | Type du résultat : Le résultat correspond à une erreur. | Type [redisRéponse](../WDLang4/1000023534.md) |
| **redisTypeNil** | Type du résultat : Le résultat n'a pas de valeur. | Type [redisRéponse](../WDLang4/1000023534.md) |
| **redisTypeStatus** | Type du résultat : Le résultat correspond à un état. | Type [redisRéponse](../WDLang4/1000023534.md) |
| **redisTypeTableau** | Type du résultat : Le résultat correspond à un tableau. | Type [redisRéponse](../WDLang4/1000023534.md) |

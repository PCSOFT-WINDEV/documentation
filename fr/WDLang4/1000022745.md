


## &lt;Variable Connexion&gt;.OpérationEncours (Fonction)

***En anglais : &lt;Connection variable&gt;.OperationInProgress***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la liste des opérations automatiques en cours sur un serveur HFSQL.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Description de la connexion
MaConnexion est une Connexion

// Liste de toutes les opérations en cours
tabOp est un tableau dynamique de hOpérationServeur 
tabOp = MaConnexion.OpérationEncours()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Connexion>.OpérationEncours()`
---

**`<Résultat> : (Tableau de hOpérationServeur)`**

Tableau de variables de type [hOpérationServeur](../WDLang4/1000022471.md) contenant la description de toutes les opérations en cours sur le serveur HFSQL.

**`<Connexion> : (Variable de type Connexion)`**

Nom de la variable de type [Connexion](../WDLang4/1514073.md) décrivant la connexion à une base de données.

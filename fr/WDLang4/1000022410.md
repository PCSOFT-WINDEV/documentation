
## mongoBase (Type de variable)

***En anglais : mongoDatabase***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **mongoBase** permet de définir toutes les caractéristiques d'une base de données d'un serveur MongoDB. Ces caractéristiques peuvent être définies et modifiées à l'aide de différentes propriétés WLangage. 

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
cnxMDB est une mongoConnexion
cnxMDB = MongoCrée("mongodb://db1.exemple.net")
POUR TOUT MaBase DE cnxMDB.BaseDeDonnées
	Trace(MaBase.Nom)
FIN
```

## Remarques
<a name="NOTE0_1"></a>


### Propriétés spécifiques à la description des variables de type mongoBase
<a name="proprietes_specifiques_description_des_variables_type_mongobase_ELTPARAGRAPHE000039"></a>Les propriétés suivantes peuvent être utilisées pour manipuler une base MongoDB :

| Nom de la propriété | Type manipulé | Effet |
| --- | --- | --- |
| Collection | Tableau de [mongoCollection](../WDLang4/1000022411.md) | Tableau associatif contenant les collections de la base MongoDB. |
| Nom | Chaîne de caractères | Nom de la base de données MongoDB. |




<a name="NOTE0_2"></a>

<a name="fonctions_utilisant_type_mongobase_ELTPARAGRAPHE000068"></a>Les fonctions suivantes manipulent des variables de type mongoBase : 



|   |   |
| --- | --- |
| [MongoCréeCollection](../WDLang4/1000022434.md) | Crée une collection dans une base MongoDB en précisant des options de cette collection. |
| [MongoExécuteCommande](../WDLang4/1000022435.md) | Exécute une commande générique sur une base MongoDB ou sur une collection MongoDB. |
| [MongoGridFSEnvoieFichier](../WDLang4/1000022436.md) | Copie un fichier du poste local vers un système de fichiers MongoDB (GridFS). |
| [MongoGridFSListeFichier](../WDLang4/1000022438.md) | Récupère la liste des fichiers MongoDB présents dans le système de fichiers MongoDB (gridFS). |

| [MongoGridFSRécupèreFichierParID](../WDLang4/1000022595.md) | Récupère sur le poste local un fichier (identifié par son identifiant) présent dans un système de fichiers MongoDB. |
| [MongoGridFSRécupèreInfoFichier](../WDLang4/1000023495.md) | Permet de récupérer les informations concernant un fichier MongoDB (gridFS). |

| [MongoGridFSSupprimeFichierParID](../WDLang4/1000022596.md) | Supprime un fichier identifié par son identifiant dans un système de fichiers MongoDB. |

| [MongoSupprimeBase](../WDLang4/1000022444.md) | Supprime une base MongoDB et toutes ses collections. |

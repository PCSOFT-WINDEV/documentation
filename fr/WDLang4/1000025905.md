
## HSuggèreCléDébut (Fonction)

***En anglais : HSuggestKeyStart***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Lance la collecte d'informations sur les requêtes exécutées afin de proposer des clés d'optimisation. 

A partir de cette fonction, les requêtes exécutées sur la connexion passée en paramètre sont analysées pour ensuite proposer un ensemble de clés à ajouter à vos fichiers de données afin d'améliorer les performances de vos requêtes.

Pour obtenir le résultat de l'analyse, la fonction [HSuggèreCléFin](../WDLang4/1000025906.md) doit être exécutée. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
HSuggèreCléDébut(MaConnexion)
// Exécuter des requêtes sur la connexion MaConnexion 
// avec les fonctions HExécuteRequêteSQL et HExécuteRequête
...
// Obtenir les suggestions
sLesClésSuggérées est une chaîne = RTFVersTexte(HSuggèreCléFin(MaConnexion))
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`HSuggèreCléDébut(<Connexion>)`
---

**`<Connexion> : (Chaîne de caractères ou variable de type Connexion)`**

Connexion vers le serveur HFSQL contenant les requêtes à analyser. Cette connexion correspond :

- soit au nom de la connexion définie sous l'éditeur d'analyses ou grâce aux fonctions [HDécritConnexion](../WDLang4/3044205.md) ou [HOuvreConnexion](../WDLang4/3044107.md).

- soit au nom d'une variable de type [Connexion](../WDLang4/1514073.md).






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Le résultat obtenu avec la fonction [HSuggèreCléFin](../WDLang4/1000025906.md) va prendre en compte les 10 requêtes exécutées les plus lentes. Il est conseillé d'exécuter plusieurs fois une même requête avec des paramètres différents afin d'obtenir des résultats plus intéressants.

- Pour analyser une requête paramétrée, il est nécessaire de fournir la valeur du paramètre lors de l'exécution de la requête. 
	Si aucune valeur n'est donnée pour le ou les paramètres de la requête, la condition de la requête ne sera pas prise en compte. Dans ce cas-là, aucune clé n'est suggérée.

- Les clés sont suggérées en fonction des données contenues dans le fichier de données. Si un fichier de données ne contient aucune donnée, aucune clé ne sera suggérée.

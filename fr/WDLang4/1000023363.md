
## MemcachedRécupère (Fonction)

***En anglais : MemcachedGet***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Récupère les valeurs des clés stockées sur un serveur Memcached. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
Résultats est un tableau de memcachedRésultat = MemcachedRécupère(memConnexion, ["clé1", "clé2"])
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Récupérer la valeur d'une clé

`<Résultat> = MemcachedRécupère(<Connexion> , <Clé>)`
---

**`<Résultat> : (Variable de type memcachedRésultat)`**

Variable de type [memcachedRésultat](../WDLang4/1000023359.md) correspondant à la clé recherchée. Si la clé n'est pas trouvée, les propriétés **Clé** et **Valeur** de la variable [memcachedRésultat](../WDLang4/1000023359.md) seront vides. 

**`<Connexion> : (Variable de type memcachedConnexion)`**

Nom de la variable de type [memcachedConnexion](../WDLang4/1000023339.md) correspondant à la connexion au cache Memcached. 

**`<Clé> : (Chaîne de caractères)`**

Clé à rechercher.


<a name="SYNTAXE2"></a>

### Récupérer les valeurs d'un ensemble de clés

`<Résultat> = MemcachedRécupère(<Connexion> , <Clés>)`
---

**`<Résultat> : (Tableau de memcachedRésultat)`**

Tableau de variables de type [memcachedRésultat](../WDLang4/1000023359.md) contenant les clés trouvées.

**`<Connexion> : (Variable de type memcachedConnexion)`**

Nom de la variable de type [memcachedConnexion](../WDLang4/1000023339.md) correspondant à la connexion au cache Memcached. 

**`<Clés> : (Tableau de chaînes de caractères)`**

Nom de la variable de type Tableau contenant les clés à rechercher.

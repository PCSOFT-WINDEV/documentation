


## &lt;Source&gt;.RAZ (Fonction)

***En anglais : &lt;Source&gt;.Reset***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Initialise une ou toutes les variables des rubriques d'un fichier de données avec leurs valeurs par défaut. Les valeurs par défaut sont définies lors de la description du fichier de données sous l'éditeur d'analyses.

**Remarques** : 

- La fonction **&lt;Source&gt;.RAZ** ne modifie pas l'enregistrement dans le fichier de données mais uniquement les données en mémoire. 

- La fonction **&lt;Source&gt;.RAZ** modifie la propriété [NouvelEnregistrement](../Proprietes/1000017090.md) du fichier de données concerné : cette propriété passe à <u><u><u>Vrai</u></u></u>.



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affecte les rubriques avec leur valeur par défaut
Client.RAZ()
// Affichage
PAGE_Client.DepuisFichier(Client)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Initialiser les rubriques d'un fichier de données avec leurs valeurs par défaut

`<Résultat> = <Source>.RAZ([<Rubrique>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'initialisation a été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> en cas de problème. La fonction [HErreurInfo](../WDLang4/3044071.md) permet d'obtenir plus de détails.




**`<Source> : (Type correspondant à la source spécifiée)`**

Nom du fichier de données manipulé.

**`<Rubrique> : (Chaîne de caractères optionnelle)`**

Nom de la rubrique à initialiser. Si ce paramètre n'est pas spécifié ou est égal à "\*", l'initialisation sera effectuée pour toutes les rubriques du fichier de données.
![Hyper File 5.5](https://doc.pcsoft.fr/ext/images/fr/HF55.png) Ce paramètre n'est pas pris en compte. Toutes les rubriques sont automatiquement ré-initialisées. 


<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Ouverture de fichiers
<a name="ouverture_fichiers_ELTPARAGRAPHE000290"></a>La fonction **&lt;Source&gt;.RAZ** effectue une ouverture du fichier de données manipulé. Le fichier de données doit donc avoir été créé.
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Lors d'une gestion personnalisée des erreurs, il est nécessaire de tester la fonction [HErreurBlocage](../WDLang4/3044099.md) après l'utilisation de la fonction **&lt;Source&gt;.RAZ**.

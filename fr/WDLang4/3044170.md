
## HDBDécritRubrique (Fonction)

***En anglais : HDBDescribeField***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Décrit par programmation chaque rubrique de la structure d'un fichier xBase décrit avec [HDBDécritFichier](../WDLang4/3044172.md).

**Remarque** : les fichiers décrits par programmation sont des fichiers au format dBase3.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
HDBDécritFichier("CLIENT", "DA", "C:\FIC\CLIENT.DBF")
HDBDécritRubrique("NOM,C,20")
HDBDécritRubrique("PRENOM,C,20")
HDBDécritRubrique("MARIE,L")
HDBDécritRubrique("DATNAISS,D")
HDBDécritRubrique("INFOS,M")
HDBDécritIndex("C:\FIC\FICNOM.NDX", "NOM")
HDBDécritIndex("C:\FIC\FICNP.NDX", "NOMPRE,C,40", "NOM+PRENOM")
HDBCréation()
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = HDBDécritRubrique(<Nom et type de la rubrique>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'opération a été réalisée, 

- <u><u><u><u>Faux</u></u></u></u> en cas de problème. La fonction [HErreur](../WDLang4/3044088.md) permet d'identifier l'erreur.




**`<Nom et type de la rubrique> : (Chaîne de caractères)`**

Nom donné à la rubrique et type de la rubrique. Ce paramètre est de la forme : 

```txt
<Nom de la rubrique> , <Type de la rubrique>
```
où : 

- &lt;Nom de la rubrique&gt; : Nom qui sera ensuite utilisé pour manipuler directement par programmation cette rubrique.

- &lt;Type de la rubrique&gt; : Type de la rubrique à créer :
	


|   |   |
| --- | --- |
| C,&lt;Taille&gt; | Rubrique de type texte, &lt;Taille&gt; est la taille de la chaîne de caractères |
| N,&lt;Taille&gt;,&lt;NbDecimale&gt; | Rubrique de type numérique. &lt;Taille&gt; est la taille du numérique, &lt;NbDecimale&gt; est le nombre de décimales du numérique. |
| L | Rubrique de type booléen |
| D | Rubrique de type date (au format "AAMMJJ") |
| M | Rubrique de type Mémo texte |








<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Utilisation d'espaces dans les paramètres
<a name="utilisation_espaces_dans_les_parametres_ELTPARAGRAPHE000077"></a>Les paramètres ne doivent pas être séparés par des espaces, par exemple il ne faut pas écrire :


```wl
HDBDécritRubrique("PRENOM, C, 20")
```


Il faut écrire :


```wl
HDBDécritRubrique("PRENOM,C,20")
```



<a name="NOTE0_2"></a>




### Description d'une rubrique
<a name="description_une_rubrique_ELTPARAGRAPHE000092"></a>

- Si le fichier est déjà ouvert lors de l'appel de la fonction **HDBDécritRubrique**, une erreur 1009 sera générée.

- Les clés (ou index) sont définies avec la fonction [HDBDécritIndex](../WDLang4/3044166.md).

- Le nombre de rubriques créées dans un fichier avec la fonction **HDBDécritRubrique** est limité à 128.

- La taille maximale d'une rubrique :

	- de type texte non clé est de 254 caractères.

	- de type texte clé est de 100 caractères.

	- de type numérique est de 19 chiffres pour la partie entière et 15 chiffres pour la partie décimale.




- Pour utiliser la fonction **HDBDécritRubrique**, le fichier a été décrit avec [HDBDécritFichier](../WDLang4/3044172.md).





<a name="NOTE0_3"></a>




### Limitations
<a name="limitations_ELTPARAGRAPHE000124"></a>

- Cette fonction n'est pas disponible lors de l'utilisation d'une base de données via .

- Cette fonction n'est pas disponible lors de l'utilisation d'un exécutable autonome.




## &lt;redisConnexion&gt;.Affecte  (Fonction)

***En anglais : &lt;redisConnection&gt;.Set***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie la valeur d'une clé sur un serveur Redis.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
gRedis est un redisConnexion
gRedis.Serveur = "MonServeurRedis:7001"

gRedis.Affecte("MaClé", "MaValeur", 1h)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Connexion>.Affecte(<Clé> , <Valeur> [, <Durée de validité>])`
---

**`<Connexion> : (Variable de type Connexion)`**

Nom de la variable de type [redisConnexion](../WDLang4/1000023527.md) correspondant à la connexion au serveur Redis.

**`<Clé> : (Chaîne de caractères)`**

Nom de la clé à affecter.

**`<Valeur> : (Chaîne de caractères)`**

Nouvelle valeur de la clé. Cette nouvelle valeur peut être une chaîne au format UTF8, Unicode ou binaire.

**`<Durée de validité> : (Durée optionnelle)`**

Durée de validité de la clé en millisecondes. Ce paramètre peut correspondre à :

- un entier correspondant au nombre de millisecondes,

- une variable de type Durée,

- l'indication directe de la durée (par exemple 1 s).




## &lt;Variable Connexion&gt;.ListeGroupe (Fonction)

***En anglais : &lt;Connection variable&gt;.ListGroup***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la liste des groupes d'utilisateurs définis pour une connexion.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Description de la connexion
MaConnexion est une Connexion

// Ouverture de la connexion
MaConnexion.OuvreConnexion()
Res est une chaîne
Res = MaConnexion.ListeGroupe()
// Ajout des groupes dans un champ Liste
ListeAjoute(LISTE_Maliste, Res)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Connexion>.ListeGroupe()`
---

**`<Résultat> : (Chaîne de caractères)`**

Nom des différents groupes existants pour la connexion spécifiée. Les noms des différents groupes sont séparés par RC (Retour-Chariot).

**Remarque** : Si l'utilisateur en cours ne possède pas le droit de visualisation des utilisateurs (constante *hDroitVisualisationUtilisateur*) sur le serveur (fonction [&lt;Variable Connexion&gt;.ModifieDroitServeur](../WDLang4/1000022710.md) ou [&lt;Variable Connexion&gt;.InfoDroitServeur](../WDLang4/1000022668.md)), la fonction **&lt;Variable Connexion&gt;.ListeGroupe** renvoie uniquement le groupe en cours.

**`<Connexion> : (Variable de type Connexion)`**

Nom de la variable de type [Connexion](../WDLang4/1514073.md) décrivant la connexion manipulée.

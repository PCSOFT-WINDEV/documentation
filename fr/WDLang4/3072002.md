


## Fonctions SQL d'accès aux bases externes
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


### Rappel
<a name="rappel_ELTPARAGRAPHE000013"></a>




|   |   |
| --- | --- |
| [HExécuteRequête](../WDLang4/3044080.md) | Déclare une requête créée sous l'éditeur de requêtes au moteur HFSQL et exécute cette requête. |
| [HExécuteRequêteSQL](../WDLang4/3044084.md) | Initialise une requête écrite en langage SQL et déclare cette requête au moteur HFSQL. |




La liste des fonctions SQL disponibles en WLangage est la suivante : 




|   |   |
| --- | --- |
| [SQLAssocie](../WDLang4/3072004.md) | Associe automatiquement chaque colonne du résultat de la requête à un champ ou à une variable de l'application. |
| [SQLAvance](../WDLang4/3072011.md) | Passe à la ligne suivante (c'est-à-dire à l'enregistrement suivant) du résultat de la requête. |
| [SQLBloque](../WDLang4/3072003.md) | Permet de bloquer :<br><br>- La table de données dans sa totalité, en lecture et en écriture. Ses données sont inaccessibles aux autres postes.<br><br>- Les enregistrements sélectionnés par la requête. Ces données sont inaccessibles aux autres postes. Cette méthode permet par exemple de mettre à jour des enregistrements.<br><br><br> |
| [SQLChangeConnexion](../WDLang4/3072016.md) | Modifie la connexion en cours. |
| [SQLCol](../WDLang4/3072018.md) | Récupère le contenu de la colonne spécifiée dans le résultat de la requête, pour la ligne en cours. |
| [SQLColonne](../WDLang4/3072001.md) | Renvoie les caractéristiques de toutes les colonnes (ou rubriques) :<br><br>- soit d'une table donnée.<br><br>- soit d'une requête donnée.<br><br><br> |
| [SQLConnecte](../WDLang4/3072005.md) | Connecte l'application en cours à une base de données à interroger par SQL. |
| [SQLConnecteWS](../WDLang4/1000021031.md) | Connecte l'application en cours à une base de données à interroger par SQL en passant par l'intermédiaire d'un proxy Webservice (SOAP). |
| [SQLDéconnecte](../WDLang4/3072008.md) | Ferme la connexion en cours et libère l'espace mémoire utilisé par la connexion. |
| [SQLDernier](../WDLang4/3072009.md) | Positionne sur la dernière ligne du résultat de la requête. |
| [SQLExec](../WDLang4/3072007.md) | Nomme et exécute une requête SQL. |
| [SQLExecWDR](../WDLang4/3072010.md) | Exécute le code SQL d'une requête créée sous l'éditeur de requêtes. |
| [SQLFerme](../WDLang4/3072015.md) | Déclare la fin d'exécution d'une requête et libère les ressources mémoire allouées lors de l'exécution de la requête. |
| [SQLFormatColonne](../WDLang4/3072006.md) | Renvoie les différents types de colonnes supportés : <br><br>- soit par la source de données identifiée par la fonction [SQLConnecte](../WDLang4/3072005.md)<br><br>- soit par la base de données accédée par une connexion particulière (connexion décrite dans l'analyse ou décrite avec les fonctions [HDécritConnexion](../WDLang4/3044205.md) ou [HOuvreConnexion](../WDLang4/3044107.md)).<br><br><br> |
| [SQLInfoGene](../WDLang4/3072028.md) | Initialise les différentes variables SQL avec les informations relatives à la dernière requête exécutée. |
| [SQLListeSource](../WDLang4/3072027.md) | Liste les sources de données déclarées sur un site local. |
| [SQLListeTable](../WDLang4/3072021.md) | Liste toutes les tables accessibles avec le système de base de données connecté. |
| [SQLLitCol](../WDLang4/3072019.md) | Récupère le contenu de la colonne spécifiée dans le résultat de la requête, pour la ligne en cours. |
| [SQLLitMémo](../WDLang4/3072020.md) | Récupère le contenu d'une colonne de type Mémo dans le résultat de la requête, pour la ligne en cours. |
| [SQLLitMémoTexte](../WDLang4/3072012.md) | Récupère le contenu d'une colonne de type Mémo texte présente dans le résultat de la requête, pour la ligne en cours. |
| [SQLModifie](../WDLang4/3072023.md) | Modifie le contenu d'une ligne de résultat de la requête. |
| [SQLPositionne](../WDLang4/3072024.md) | Positionne directement sur la ligne de résultat spécifiée de la requête. |
| [SQLPrécédent](../WDLang4/3072022.md) | Positionne sur la ligne précédente du résultat de la requête. |
| [SQLPremier](../WDLang4/3072017.md) | Positionne sur la première ligne du résultat de la requête. |
| [SQLReqExiste](../WDLang4/3072026.md) | Teste l'existence d'une requête. |
| [SQLSuivant](../WDLang4/3072030.md) | Positionne sur la ligne suivante du résultat de la requête. |
| [SQLTable](../WDLang4/3072031.md) | Transfère directement le résultat d'une requête dans un champ Table par programmation (un champ Liste ou un champ Combo), avec possibilité de Fetch Partiel (récupération du résultat par blocs de lignes). |
| [SQLTransaction](../WDLang4/3072029.md) | Permet de débuter, valider ou annuler une transaction :<br><br>- soit sur les fichiers d'une base de données ouverte par la fonction [SQLConnecte](../WDLang4/3072005.md),<br><br>- soit sur les fichiers d'une base de données ouverte par une connexion (OLE DB ou Accès Natif/Connecteur Natif) décrite sous l'éditeur d'analyses, ou par la fonction [HOuvreConnexion](../WDLang4/3044107.md).<br><br><br> |

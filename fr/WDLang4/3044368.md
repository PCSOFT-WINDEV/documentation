
## Constantes de gestion de la réplication
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>




| Constante | Détail | Utilisée par les fonctions |
| --- | --- | --- |
| **rplAbonnéEstAJour** | Information à connaître : Le réplica abonné est à jour ou non | [HInfoRéplica](../WDLang4/3044015.md) |
| **rplAbonnéPrioritaire** | Les données présentes dans la base de données abonnée (ou le fichier abonné) sont prioritaires lors de la réplication. | [HSynchroniseRéplica](../WDLang4/3044014.md),<br>[HRplGestionFichier](../WDLang4/1000017215.md) |
| **rplBidirectionnel** | <br><br>- Mise à jour de la base de données maître ET de la base de données abonnée.<br><br>- Mise à jour du fichier maître ET du fichier abonné.<br><br><br> | [HSynchroniseRéplica](../WDLang4/3044014.md),<br>[HRplGestionFichier](../WDLang4/1000017215.md) |
| **rplConflitModification** | Enregistrement a été modifié à la fois sur le réplica source et sur le réplica destination | [HSynchroniseRéplica](../WDLang4/3044014.md) |
| **rplConflitSuppression** | Enregistrement supprimé sur le réplica destination et modifié ou supprimé sur le réplica source. | [HSynchroniseRéplica](../WDLang4/3044014.md) |
| **rplDateSynchroDepuisAbonné** | Information à connaître : Date de la dernière synchronisation | [HInfoRéplica](../WDLang4/3044015.md) |
| **rplDateSynchroDepuisMaître** | Information à connaître : Date de la dernière synchronisation depuis la base maître. | [HInfoRéplica](../WDLang4/3044015.md) |
| **rplDateSynchroVersAbonné** | Information à connaître : Date de la dernière synchronisation vers la base abonnée. | [HInfoRéplica](../WDLang4/3044015.md) |
| **rplDateSynchroVersMaître** | Information à connaître : Date de la dernière synchronisation vers la base maître. | [HInfoRéplica](../WDLang4/3044015.md) |
| **rplGestionConflitRéplication** | Le fichier utilisera le mode de gestion des conflits spécifié avec la fonction HSynchroniseRéplica. | [HRplGestionFichier](../WDLang4/1000017215.md) |
| **rplHAjoute** | Réplication d'un ajout | [HSynchroniseRéplica](../WDLang4/3044014.md) |
| **rplHModifie** | Réplication d'une modification | [HSynchroniseRéplica](../WDLang4/3044014.md) |
| **rplHSupprime** | Réplication d'une suppression | [HSynchroniseRéplica](../WDLang4/3044014.md) |
| **rplMaîtreEstAJour** | Information à connaître : Le réplica maître est à jour ou non. | [HInfoRéplica](../WDLang4/3044015.md) |
| **rplMaîtrePrioritaire** | Les données présentes dans la base de données maître sont prioritaires lors de la réplication. | [HSynchroniseRéplica](../WDLang4/3044014.md) |
| **rplNumRéplicaTransportable** | Information à connaître : Numéro de génération du dernier réplica transportable généré. | [HInfoRéplica](../WDLang4/3044015.md) |
| **rplPasDeConflit** | Aucun conflit détecté | [HSynchroniseRéplica](../WDLang4/3044014.md) |
| **rplPlusRécentPrioritaire** | Les données les plus récentes sont prioritaires. Attention, les données les plus récentes sont évaluées par rapport à l'heure sur le poste abonné. | [HSynchroniseRéplica](../WDLang4/3044014.md) |
| **rplRéplicationDésactivé** | Permet de désactiver temporairement la gestion de la réplication. | [HGèreRéplication](../WDLang4/3044067.md) |
| **rplRéplicationHétérogène** | Constante gardée par compatibilité. Cette constante est identique à *rplRéplicationUniverselle*. | [HGèreRéplication](../WDLang4/3044067.md) |
| **rplRéplicationJournalisé** | Permet de réaliser une réplication entre des fichiers HFSQL (utilise les fichiers HFSQL journalés). | [HGèreRéplication](../WDLang4/3044067.md) |
| **rplRéplicationUniverselle** | Permet de réaliser une réplication universelle (par exemple entre une base de données HFSQL et une base de données Oracle). | [HGèreRéplication](../WDLang4/3044067.md) |
| **rplRubriqueNonRépliquée** | Rubrique non répliquée. | [HRplGestionRubrique](../WDLang4/1000017216.md) |
| **rplRubriqueRépliquée** | Rubrique répliquée. | [HRplGestionRubrique](../WDLang4/1000017216.md) |
| **rplVersAbonné** | <br><br>- Mise à jour de la base de données abonnée en fonction de la base de données maître.<br><br>- Mise à jour du fichier abonné en fonction de la base de données maître.<br><br><br> | [HSynchroniseRéplica](../WDLang4/3044014.md),<br>[HRplGestionFichier](../WDLang4/1000017215.md) |
| **rplVersMaître** | <br><br>- Mise à jour de la base de données maître en fonction de la base de données abonnée.<br><br>- Mise à jour du fichier maître en fonction de la base de données abonnée.<br><br><br> | [HSynchroniseRéplica](../WDLang4/3044014.md),<br>[HRplGestionFichier](../WDLang4/1000017215.md) |

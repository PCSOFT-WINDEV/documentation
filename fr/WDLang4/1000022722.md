


## &lt;Variable Connexion&gt;.ModifieTâche (Fonction)

***En anglais : &lt;Connection variable&gt;.ModifyTask***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie une tâche planifiée sur le serveur HFSQL défini par la connexion. Cette tâche est présente dans une variable de type [hTâchePlanifiée](../WDLang4/1000017448.md).

**Remarque** : La fonction [&lt;Variable Connexion&gt;.InfoTâche](../WDLang4/1000022686.md) permet de récupérer les caractéristiques d'une tâche planifiée dans une variable de type [hTâchePlanifiée](../WDLang4/1000017448.md).




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MaConnexion est une Connexion
MaTâche est une hTâchePlanifiée

// Récupération des caractéristiques de la tâche
MaTâche = MaConnexion.InfoTâche(IDTâche)
// Modification de la tâche
MaTâche.Heure = SAI_Heure.Heure+1
MaConnexion.ModifieTâche(IDTâche, MaTâche)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Connexion>.ModifieTâche(<Identifiant de la tâche> , <Tâche planifiée>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la tâche planifiée a été modifiée, 

- <u><u><u><u>Faux</u></u></u></u> en cas d'erreur.




**`<Connexion> : (Variable de type Connexion)`**

Nom de la variable de type [Connexion](../WDLang4/1514073.md) décriant la connexion pour laquelle une tâche doit être ajoutée.

**`<Identifiant de la tâche> : (Entier)`**

Identifiant de la tâche à modifier. Cet identifiant est renvoyé par la fonction [&lt;Variable Connexion&gt;.AjouteTâche](../WDLang4/1000022645.md).

**`<Tâche planifiée> : (Variable de type hTâchePlanifiée)`**

Nom de la variable de type [hTâchePlanifiée](../WDLang4/1000017448.md) contenant les modifications.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Droits nécessaires
<a name="droits_necessaires_ELTPARAGRAPHE000241"></a>Pour modifier une tâche, il est nécessaire de posséder :

- le droit correspondant (constante *hDroitGèreTâche*) sur le serveur,

- le droit d'exécuter la commande liée à la tâche planifiée.




La tâche sera exécutée avec le compte de l'utilisateur qui a créé la tâche.

**Remarque** : Si le serveur est éteint, la tâche planifiée ne sera pas réalisée. La tâche planifiée sera réalisée à la prochaine date disponible lorsque le serveur sera allumé.




## HListeUtilisateur (Fonction)

***En anglais : HListUser***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la liste des utilisateurs définis pour une connexion.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Description de la connexion
HDécritConnexion("MaConnexion", "Julie", "MotDePasse", "MonServeurHF", "MaBaseDeDonnées", ...
	hAccèsHFClientServeur, HOLecture)
// Ouverture de la connexion
HOuvreConnexion("MaConnexion")
Résultat1 est une chaîne
Résultat1 = HListeUtilisateur("MaConnexion")
// Ajout des utilisateurs dans un champ Liste 
ListeAjoute(LISTE_Maliste, Résultat1)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = HListeUtilisateur(<Connexion>)`
---

**`<Résultat> : (Chaîne de caractères)`**

Nom des différents utilisateurs existants pour la connexion spécifiée. Les noms des utilisateurs sont séparés par RC (Retour-Chariot).

**Remarque** : Si l'utilisateur en cours ne possède pas le droit de visualisation des utilisateurs (constante *hDroitVisualisationUtilisateur*) sur le serveur (fonction [HModifieDroitServeur](../WDLang4/3044328.md) ou [HInfoDroitServeur](../WDLang4/3044322.md)), la fonction **HListeUtilisateur** renvoie uniquement l'utilisateur en cours. 

**`<Connexion> : (Chaîne de caractères ou variable de type Connexion)`**

Connexion à manipuler. Cette connexion correspond : 

- soit au nom de la connexion définie sous l'éditeur d'analyses ou grâce aux fonctions [HDécritConnexion](../WDLang4/3044205.md) ou [HOuvreConnexion](../WDLang4/3044107.md).

- soit au nom d'une variable de type [Connexion](../WDLang4/1514073.md).

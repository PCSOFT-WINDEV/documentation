
## HLibèrePosition (Fonction)

***En anglais : HFreePosition***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime une position sauvegardée par la fonction [HSauvePosition](../WDLang4/3044136.md). Permet de libérer la mémoire allouée à la gestion des positions.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Sauvegarde de la position
nPos est un entier = HSauvePosition(Client, Nom)
...
// Traitements
...
// Suppression de la position devenue inutile
HLibèrePosition(nPos)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = HLibèrePosition(<Position>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la position spécifiée a été libérée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. La fonction [HErreur](../WDLang4/3044088.md) permet d'obtenir plus d'informations sur le problème rencontré.




**`<Position> : (Entier)`**

Numéro d'une position précédemment sauvegardée, renvoyé par la fonction [HSauvePosition](../WDLang4/3044136.md).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Les positions non libérées par programmation seront automatiquement libérées lors de la fermeture de l'application.

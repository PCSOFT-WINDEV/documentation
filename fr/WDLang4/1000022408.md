
## MongoCrée (Fonction)

***En anglais : MongoCreate***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Crée une connexion à un serveur ou à un cluster MongoDB.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple
<a class="notetitle" target="_blank" href="$DOC$=1000022624&name=mongocree_fonction&product=WD">Voir des exemples supplémentaires</a>

```wl
cnxMDB est une mongoConnexion
cnxMDB = MongoCrée("mongodb://db1.exemple.net")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = MongoCrée(<Chaîne de connexion>)`
---

**`<Résultat> : (Variable de type mongoConnexion)`**

Variable de type [mongoConnexion](../WDLang4/1000022409.md) représentant la connexion à la base MongoDB. En cas de problème, la fonction [ErreurInfo](../WDLang1/3013008.md) permet d'identifier l'erreur.

**`<Chaîne de connexion> : (Chaîne de caractères)`**

Chaîne de connexion au serveur ou du cluster hébergeant la base MongoDB. Pour plus de détails, consultez la documentation de MongoDB. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Cette fonction est équivalente à la syntaxe suivante : 

```txt
<NomConnexion> est une mongoConnexion(<Chaîne de connexion>)
```

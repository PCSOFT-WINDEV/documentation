


## HTransactionListe (Fonction)

***En anglais : HTransactionList***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Liste les transactions en cours ou interrompues sur le serveur de la connexion spécifiée.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MaConnexion est une Connexion

// Récupérer la liste des transactions
sListeTrs est une chaîne = HTransactionListe(MaConnexion)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = HTransactionListe(<Connexion>)`
---

**`<Résultat> : (Chaîne de caractères)`**

Liste des transactions en cours ou interrompues sur le serveur associé à la connexion. Cette liste est de la forme : 

```txt
<Identifiant de la transaction> + TAB + <Nom de l'application> + TAB + 
<Poste Client> + TAB + <Valeur du HPoste> + TAB + 
<Etat de la transaction> + RC + ...
```
Le paramètre &lt;Etat de la transaction&gt; correspond à 0 pour une transaction en cours, et 1 pour une transaction interrompue.

**`<Connexion> : (Chaîne de caractères ou variable de type Connexion)`**

Connexion pour laquelle les transactions doivent être listées. Cette connexion correspond :

- soit au nom de la connexion définie sous l'éditeur d'analyses ou grâce aux fonctions [HDécritConnexion](../WDLang4/3044205.md) ou [HOuvreConnexion](../WDLang4/3044107.md).

- soit au nom d'une variable de type [Connexion](../WDLang4/1514073.md).

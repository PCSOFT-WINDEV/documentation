


## &lt;Variable Connexion&gt;.AjoutePlanification (Fonction)

***En anglais : &lt;Connection variable&gt;.AddScheduling***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Crée une tâche planifiée sur un serveur HFSQL : 

- procédure stockée, 

- sauvegarde, 

- optimisation, 

- rafraîchissement d'une vue matérialisée.





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Description de la connexion
MaConnexion est une Connexion
...
// Recalculer les statistiques des fichiers de données de la base de données "MaBase", 
// tous les samedis à 23H00
Optim est une hOptimisationPlanifiée
Optim.Description = "Recalcul des statistiques des fichiers de données" + ...
	" de la base de données MaBase"

Optim.Planification.Mois = "*"
Optim.Planification.JourDeLaSemaine = "samedi"
Optim.Planification.Heure = "23"
Optim.Planification.Minute = "0"

Optim.Fichier[1].Nom = "MaBase"
Optim.Fichier[1].Option = hOptStat

MaConnexion.AjoutePlanification(Optim)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Ajouter une tâche planifiée

`<Résultat> = <Connexion>.AjoutePlanification(<Tâche planifiée>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la tâche planifiée a été ajoutée,

- <u><u><u><u>Faux</u></u></u></u> en cas d'erreur. La fonction [HErreurInfo](../WDLang4/3044071.md) permet d'identifier l'erreur.




**`<Connexion> : (Variable de type Connexion)`**

Nom de la variable de type [Connexion](../WDLang4/1514073.md) décrivant la connexion pour laquelle une tâche planifiée doit être ajoutée.

**`<Tâche planifiée> : (Variable de type hTâchePlanifiée)`**

Nom de la variable de type [hTâchePlanifiée](../WDLang4/1000017448.md), décrivant la tâche déclenchant une procédure stockée.


<a name="SYNTAXE2"></a>

### Ajouter une tâche d'optimisation planifiée

`<Résultat> = <Connexion>.AjoutePlanification(<Tâche d'optimisation planifiée>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la tâche d'optimisation planifiée a été ajoutée,

- <u><u><u><u>Faux</u></u></u></u> en cas d'erreur. La fonction [HErreurInfo](../WDLang4/3044071.md) permet d'identifier l'erreur.




**`<Connexion> : (Variable de type Connexion)`**

Nom de la variable de type [Connexion](../WDLang4/1514073.md) décrivant la connexion pour laquelle une tâche d'optimisation planifiée doit être ajoutée.

**`<Tâche d'optimisation planifiée> : (Variable de type hOptimisationPlanifiée)`**

Nom de la variable de type [hOptimisationPlanifiée](../WDLang4/1000018854.md) manipulée.


<a name="SYNTAXE3"></a>

`<Résultat> = <Connexion>.AjoutePlanification(<Sauvegarde planifiée>)`
---

**`<Résultat> : (Booléen)`**

**`<Sauvegarde planifiée> : (Variable de type hDescriptionSauvegarde)`**

Nom de la variable de type [hDescriptionSauvegarde](../WDLang4/1000017455.md) manipulée.

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Connexion> : (Variable de type Connexion)`**

Nom de la variable de type [Connexion](../WDLang4/1514073.md) décrivant la connexion pour laquelle une planification de rafraîchissement de vue matérialisée doit être ajoutée.

**`<Vue matérialisée> : (Variable de type hPlanificationVueMatérialisée)`**

Nom de la variable de type [hPlanificationVueMatérialisée](../WDLang4/1000021444.md) contenant la description de la vue matérialisée à rafraîchir.

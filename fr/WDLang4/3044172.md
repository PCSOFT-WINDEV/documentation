
## HDBDécritFichier (Fonction)

***En anglais : HDBDescribeFile***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Décrit par programmation un fichier de données au format dBase 3 (format le plus répandu). Cette fonction permet d'indiquer le nom, l'abréviation et le chemin d'accès au fichier xBase.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
HDBDécritFichier("CLIENTDB", "CD", "C:\FIC\CLIENT.DBF")
HDBDécritRubrique("NOM,C,20")
HDBDécritRubrique("PRENOM,C,20")
HDBDécritRubrique("AGE,N,3,0")
HDBDécritRubrique("SOLDE,N,10,2")
HDBDécritRubrique("MARIE,L")
HDBDécritRubrique("DATNAISS,D")
HDBDécritRubrique("INFOS,M")
HDBCréation()
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = HDBDécritFichier(<Nom logique du fichier xBase> , <Préfixe du fichier> , <Nom complet du fichier physique>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'opération a été réalisée, 

- <u><u><u><u>Faux</u></u></u></u> en cas de problème. La fonction [HErreur](../WDLang4/3044088.md) permet d'identifier l'erreur.




**`<Nom logique du fichier xBase> : (Chaîne de caractères)`**

Nom logique du fichier xBase à créer. Ce nom sera utilisé dans toutes les fonctions manipulant les fichiers xBase.

**`<Préfixe du fichier> : (Chaîne de caractères)`**

Abréviation du fichier (2 caractères).

**`<Nom complet du fichier physique> : (Chaîne de caractères)`**

Chemin dans lequel sera créé le fichier sur disque et le nom complet du fichier (avec l'extension).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Utilisation du mot-clé EXTERNE
<a name="utilisation_motcle_externe_ELTPARAGRAPHE000052"></a>Il est conseillé d'utiliser le mot-clé EXTERNE pour déclarer le fichier qui va être décrit puis créé, afin d'éviter les warning sous l'éditeur de code. Par exemple :


```wl
EXTERNE Client
HDBDécritFichier("CLIENT", "DA", "C:\FIC\CLIENT.DBF")
```



<a name="NOTE0_2"></a>




### Description du fichier 
<a name="description_fichier_ELTPARAGRAPHE000063"></a>

- Si le fichier est déjà ouvert lors de l'appel de la fonction **HDBDécritFichier**, une erreur 1009 sera générée.

- Le fichier sera créé sur le disque avec la fonction [HDBCréation](../WDLang4/3044171.md).

- La structure du fichier est décrite avec la fonction [HDBDécritRubrique](../WDLang4/3044170.md). Le nombre de rubriques créées dans un fichier est limité à 128.

- Les index sont décrits avec la fonction [HDBDécritIndex](../WDLang4/3044166.md).





<a name="NOTE0_3"></a>




### Limitations
<a name="limitations_ELTPARAGRAPHE000087"></a>

- Cette fonction n'est pas disponible lors de l'utilisation d'une base de données via .

- Cette fonction n'est pas disponible lors de l'utilisation d'un exécutable autonome.




## &lt;Variable Connexion&gt;.Priorité (Fonction)

***En anglais : &lt;Connection variable&gt;.Priority***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de connaître ou de modifier la priorité de l'application appelante.

**Remarque** : Plus la priorité est haute, plus le serveur traitera les demandes de l'application en premier.



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Description de la connexion
MaConnexion est une Connexion

// Donner une priorité maximale à l'application
MaConnexion.Priorité(10)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître la priorité de l'application appelante

`<Résultat> = <Connexion>.Priorité()`
---

**`<Résultat> : (Entier)`**

Valeur de la priorité de l'application. Cette valeur est comprise entre -10 (basse priorité) et 10 (haute priorité). 
Par défaut, au lancement de l'application, la priorité est de 0 (priorité normale).

**`<Connexion> : (Variable de type Connexion)`**

Nom de la variable de type [Connexion](../WDLang4/1514073.md) décrivant la connexion vers le serveur HFSQL.


<a name="SYNTAXE2"></a>

### Modifier la priorité de l'application appelante

`<Résultat> = <Connexion>.Priorité(<Priorité>)`
---

**`<Résultat> : (Entier)`**

Précédente valeur de la priorité de l'application.

**`<Connexion> : (Variable de type Connexion)`**

Nom de la variable de type [Connexion](../WDLang4/1514073.md) décrivant la connexion vers le serveur HFSQL.

**`<Priorité> : (Entier)`**

Nouvelle valeur de la priorité à affecter à l'application cliente. Cette valeur est comprise entre -10 (basse priorité) et 10 (haute priorité). 
Par défaut, au lancement de l'application, la priorité est de 0 (priorité normale).



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Droit de modification
<a name="droit_modification_ELTPARAGRAPHE000265"></a>Pour augmenter la priorité de l'application en cours, le droit de modifier la priorité est nécessaire. Ce droit correspond à la constante *hDroitPriorité* (utilisée dans les fonctions [&lt;Variable Connexion&gt;.InfoDroitServeur](../WDLang4/1000022668.md) et [&lt;Variable Connexion&gt;.ModifieDroitServeur](../WDLang4/1000022710.md)).

Pour consulter ou diminuer la priorité de l'application en cours, le droit de modifier la priorité n'est pas nécessaire.




## Fonctions HDFS
			



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>
Les fonctions de gestion des fichiers sur un système HDFS sont les suivantes :



|   |   |
| --- | --- |
| [hdfsCopieDepuis](../WDLang4/1000021678.md) | Copie un fichier situé sur un serveur Hadoop vers la machine locale. |
| [hdfsCopieVers](../WDLang4/1000021679.md) | Copie un fichier situé sur la machine locale vers un serveur Hadoop. |
| [hdfsCréeRépertoire](../WDLang4/1000021675.md) | Crée un répertoire sur un serveur Hadoop. |
| [hdfsFermeConnexion](../WDLang4/1000021867.md) | Ferme une connexion à un serveur Hadoop. |
| [hdfsListe](../WDLang4/1000021677.md) | Liste le contenu d'un répertoire sur un serveur Hadoop. |
| [hdfsOuvreConnexion](../WDLang4/1000021866.md) | Ouvre une connexion vers un serveur Hadoop. |
| [hdfsRenomme](../WDLang4/1000021676.md) | Modifie le nom d'un fichier ou d'un répertoire sur un serveur Hadoop. |
| [hdfsSupprime](../WDLang4/1000021674.md) | Supprime un fichier ou un répertoire sur un serveur Hadoop. |



  


Les types manipulés par les fonctions HDFS sont les suivants :



|   |   |
| --- | --- |
| [hdfsConnexion](../WDLang4/1000021650.md) | Le type **hdfsConnexion** permet de définir toutes les caractéristiques avancées d'une connexion au système de fichiers Hadoop (HDFS). |
| [hdfsDroit](../WDLang4/1000021662.md) | Le type **hdfsDroit** permet de définir toutes les caractéristiques avancées des droits d'accès à un élément (fichier, lien, répertoire) du système de fichiers Hadoop. |
| [hdfsInformation](../WDLang4/1000021973.md) | Le type **hdfsInformation** permet de définir toutes les caractéristiques avancées de l'état d'un élément (fichier, répertoire, lien) du système de fichiers Hadoop. |




Attention : Les fonctions HBase sont disponibles en édition et en exécution uniquement à partir de Windows Vista ou Windows Server 2008.

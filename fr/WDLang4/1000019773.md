
## HInfoAnalyse (Fonction)

***En anglais : HInfoAnalysis***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie des informations sur une analyse (fichier WDD).




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
Trace(HInfoAnalyse())
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = HInfoAnalyse([<Chemin du WDD> [, <Mot de passe du WDD>]])`
---

**`<Résultat> : (Chaîne de caractères)`**

Informations sur l'analyse. Ces informations utilisent le format suivant : 

```txt
<GUID Analyse> + RC + <Numéro génération>
```


**`<Chemin du WDD> : (Chaîne de caractères optionnelle)`**

Chemin complet de l'analyse (c'est-à-dire du ficher .WDD) à manipuler. 
Si ce paramètre n'est pas précisé ou est égal à une chaîne vide (""), l'analyse en cours est manipulée.

**`<Mot de passe du WDD> : (Chaîne de caractères optionnelle)`**

Mot de passe défini pour l'analyse. Si aucun mot de passe n'est défini, ce paramètre doit être égal à une chaîne vide ("").

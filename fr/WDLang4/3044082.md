
## HErreurMotDePasse (Fonction)

***En anglais : HErrorPassword***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Après l'exécution d'une fonction de lecture ou d'écriture dans un fichier de données, permet de savoir si une erreur due à un mot de passe erroné a été rencontrée sur ce fichier de données.

En général, il est conseillé d'utiliser la fonction **HErreurMotDePasse** après la première fonction manipulant un fichier de données HFSQL.

Cette fonction peut être utilisée indifféremment avec les fichiers de données HFSQL, les vues HFSQL ou les requêtes.

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Par défaut, WINDEV propose une [gestion automatique des erreurs de mots de passe](../WDLang4/3044188.md). Cette fonctionnalité n'est pas disponible dans le code des procédures stockées.

![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Seules les erreurs de mot de passe survenues sur des fichiers de données HFSQL sont signalées. 






<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Vérification du compte-rendu de la fonction HLitPremier
SI HLitPremier(Client, Nom) = Faux ALORS
	// Problème lors de la lecture
	// Vérifie si l'erreur est due à un mot de passe erroné
	SI HErreurMotDePasse() = Vrai ALORS
		Info("Le mot de passe donné pour manipuler le fichier Client est faux")
		 // Ouverture de la fenêtre de saisie du mot de passe
		MotDePasse = Ouvre(FEN_Saisie_Passe)
		HPasse(Client, MotDePasse)
	FIN
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = HErreurMotDePasse()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> en cas de mot de passe erroné, 

- <u><u><u><u>Faux</u></u></u></u> si le mot de passe est correct.

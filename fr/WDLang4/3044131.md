


## HRéindexationEnCours (Fonction)

***En anglais : HIndexingInProgress***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Avertit si une ré-indexation est en cours sur un fichier de données HFSQL et renvoie le pourcentage de ré-indexation effectué.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI HRéindexationEnCours(Client) <> 0 ALORS
	Trace("Opération Impossible, réindexation en cours")
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = HRéindexationEnCours(<Fichier de données> [, <Jauge>])`
---

**`<Résultat> : (Entier)`**



- 0 si aucune ré-indexation n'est en cours sur le fichier de données, 

- un entier compris entre 1 et 100 indiquant le pourcentage de ré-indexation déjà effectué




**`<Fichier de données> : (Chaîne de caractères)`**

Nom du fichier de données manipulé.

**`<Jauge> : (Nom de fenêtre, nom de champ ou entier optionnel)`**



- Nom de la fenêtre dans laquelle la jauge doit être affichée, ou nom du champ de type Jauge.

- Identifiant (Handle) de la fenêtre qui affiche la jauge.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Ré-indexation
<a name="reindexation_ELTPARAGRAPHE000093"></a>La réindexation ré-organise les index. Pendant la ré-indexation, tout accès aux données est impossible.
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Par défaut, lors d'un accès à un fichier de données en cours de ré-indexation, le moteur HFSQL affiche une fenêtre indiquant qu'une ré-indexation est en cours. Cette fenêtre sera automatiquement fermée à la fin de la ré-indexation.
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=HF_Reindexation.gif)
Pour plus de détails sur cette fenêtre et sa personnalisation, consultez [Gestion assistée des erreurs HFSQL](../WDLang4/3044188.md).
<a name="NOTE0_2"></a>


### Limitation
<a name="limitation_ELTPARAGRAPHE000110"></a>Cette fonction n'est pas disponible lors de l'utilisation d'une base de données via  .

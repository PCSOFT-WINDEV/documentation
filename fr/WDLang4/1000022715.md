


## &lt;Variable Connexion&gt;.ModifieOptimisationPlanifiée (Fonction)

***En anglais : &lt;Connection variable&gt;.ModifyScheduledOptimization***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie une tâche d'optimisation planifiée sur le serveur HFSQL défini par la connexion. Cette tâche est présente dans une variable avancée de type [hOptimisationPlanifiée](../WDLang4/1000018854.md).



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Connexion>.ModifieOptimisationPlanifiée(<Tâche d'optimisation planifiée>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la tâche d'optimisation planifiée a été modifiée,

- <u><u><u><u>Faux</u></u></u></u> en cas d'erreur.




**`<Connexion> : (Variable de type Connexion)`**

Nom de la variable de type [Connexion](../WDLang4/1514073.md) décrivant la connexion pour laquelle une tâche d'optimisation planifiée doit être modifiée.

**`<Tâche d'optimisation planifiée> : (Variable de type hOptimisationPlanifiée)`**

Nom de la variable de type [hOptimisationPlanifiée](../WDLang4/1000018854.md) manipulée.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Droits nécessaires
<a name="droits_necessaires_ELTPARAGRAPHE000191"></a>Pour modifier une tâche, il est nécessaire de posséder : 

- le droit correspondant (constante *hDroitGèreTâche*) sur le serveur, 

- le droit d'exécuter la commande liée à la tâche planifiée.


La tâche sera exécutée avec le compte de l'utilisateur qui a créé la tâche. 

Remarque : Si le serveur est éteint, la tâche planifiée ne sera pas réalisée. La tâche planifiée sera réalisée à la prochaine date disponible lorsque le serveur sera allumé.




## &lt;Source&gt;.StatDate (Fonction)

***En anglais : &lt;Source&gt;.StatDate***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la date de la dernière mise à jour des statistiques d'index. Ces statistiques ont été réalisées grâce aux fonctions [&lt;Source&gt;.StatCalcule](../WDLang4/1000025064.md) ou [&lt;Source&gt;.Réindexe](../WDLang4/1000025060.md).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Réalisation de statistiques sur la clé NomCli du fichier de données Client
Client.StatCalcule(NomCli)

// Connaître la date de réalisation de ces statistiques
ResDate = Client.StatDate()
Info("Les statistiques effectuées sur la rubrique NomCli " + ...
	"ont été réalisées le : " + DateVersChaîne(ResDate, maskDateSystème))
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Source>.StatDate([<Rubrique clé>])`
---

**`<Résultat> : (Chaîne de caractères)`**



- Date de la dernière mise à jour des statistiques au format AAAAMMJJ (AAAA correspond à l'année, MM au mois et JJ au jour). La date renvoyée est au format UTC (Universal Time Coordinated). 

- Chaîne vide ("") si aucun calcul de statistiques n'a été effectué.




**`<Source> : (Type correspondant à la source spécifiée)`**

Nom du fichier de données manipulé. 

**`<Rubrique clé> : (Chaîne de caractères optionnelle)`**

Nom de la rubrique clé sur laquelle les statistiques ont été effectuées. Si ce nom n'est pas spécifié, la fonction **&lt;Source&gt;.StatDate** manipule la dernière rubrique clé utilisée par la dernière fonction de gestion HFSQL (fonction commençant par la lettre H).



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Dans les fichiers de données HFSQL, seules les rubriques clés peuvent faire l'objet d'un calcul de statistiques.

- Cette fonction n'est pas disponible lors de l'utilisation d'une base de données via  .

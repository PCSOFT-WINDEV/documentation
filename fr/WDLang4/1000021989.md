


## Fonctions HDFS (syntaxe préfixée)
			



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>
Les fonctions de gestion des fichiers sur un système HDFS (syntaxe préfixée) sont les suivantes : 



|   |   |
| --- | --- |
| [&lt;Variable hdfsConnexion&gt;.CopieDepuis](../WDLang4/1000021980.md) | Copie un fichier situé sur un serveur Hadoop vers la machine locale. |
| [&lt;Variable hdfsConnexion&gt;.CopieVers](../WDLang4/1000021981.md) | Copie un fichier situé sur la machine locale vers un serveur Hadoop. |
| [&lt;Variable hdfsConnexion&gt;.CréeRépertoire](../WDLang4/1000021982.md) | Crée un répertoire sur un serveur Hadoop. |
| [&lt;Variable hdfsConnexion&gt;.FermeConnexion](../WDLang4/1000021983.md) | Ferme une connexion à un serveur Hadoop. |
| [&lt;Variable hdfsConnexion&gt;.Liste](../WDLang4/1000021984.md) | Liste le contenu d'un répertoire sur un serveur Hadoop. |
| [&lt;Variable hdfsConnexion&gt;.OuvreConnexion](../WDLang4/1000021985.md) | Ouvre une connexion vers un serveur Hadoop. |
| [&lt;Variable hdfsConnexion&gt;.Renomme](../WDLang4/1000021986.md) | Modifie le nom d'un fichier ou d'un répertoire sur un serveur Hadoop. |
| [&lt;Variable hdfsConnexion&gt;.Supprime](../WDLang4/1000021987.md) | Supprime un fichier ou un répertoire sur un serveur Hadoop. |



Les types manipulés par les fonctions HDFS sont les suivants :



|   |   |
| --- | --- |
| [hdfsConnexion](../WDLang4/1000021650.md) | Le type **hdfsConnexion** permet de définir toutes les caractéristiques avancées d'une connexion au système de fichiers Hadoop (HDFS). |
| [hdfsDroit](../WDLang4/1000021662.md) | Le type **hdfsDroit** permet de définir toutes les caractéristiques avancées des droits d'accès à un élément (fichier, lien, répertoire) du système de fichiers Hadoop. |
| [hdfsInformation](../WDLang4/1000021973.md) | Le type **hdfsInformation** permet de définir toutes les caractéristiques avancées de l'état d'un élément (fichier, répertoire, lien) du système de fichiers Hadoop. |




Attention : Les fonctions HBase sont disponibles en édition et en exécution uniquement à partir de Windows Vista ou Windows Server 2008.




## &lt;Variable hdfsConnexion&gt;.OuvreConnexion (Fonction)

***En anglais : &lt;hdfsConnection variable&gt;.OpenConnection***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ouvre une connexion vers un serveur Hadoop.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connexion
MaConnexion est une hdfsConnexion
MaConnexion.namenode = "MonServeur"
MaConnexion.Utilisateur = "Admin"
SI MaConnexion.OuvreConnexion() = Faux ALORS
	Erreur(ErreurInfo())
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Connexion>.OuvreConnexion()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la connexion a été réalisée,

- <u><u><u><u>Faux</u></u></u></u> en cas de problème. La fonction [ErreurInfo](../WDLang1/3013008.md) permet d'identifier l'erreur.




**`<Connexion> : (Variable de type hdfsConnexion)`**

Nom de la variable de type [hdfsConnexion](../WDLang4/1000021650.md) correspondant à la connexion au serveur Hadoop à ouvrir.

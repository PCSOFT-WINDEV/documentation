


## &lt;Variable Connexion&gt;.ModifieGroupe (Fonction)

***En anglais : &lt;Connection variable&gt;.ModifyGroup***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie les informations du groupe en fonction des éléments présents dans [les variables de gestion de groupe](../WDLang4/3044315.md) correspondantes.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Description de la connexion
MaConnexion est une Connexion

// Ouverture de la connexion
MaConnexion.OuvreConnexion()
...
// Modification des membres du groupe
HGroupe.Utilisateurs = "Flo" + RC + "Julie" + RC + "Luc"
// Modification du groupe
SI MaConnexion.ModifieGroupe("TESTEURS") = Vrai ALORS
	Info("Groupe modifié")
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Connexion>.ModifieGroupe(<Groupe>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le groupe a été modifié,

- <u><u><u><u>Faux</u></u></u></u> en cas d'erreur. La fonction [HErreur](../WDLang4/3044088.md) permet d'identifier l'erreur.




**`<Connexion> : (Variable de type Connexion)`**

Nom de la variable de type [Connexion](../WDLang4/1514073.md) décrivant la connexion pour laquelle un groupe doit être modifié.

**`<Groupe> : (Chaîne de caractères)`**

Nom du groupe dont on veut modifier les caractéristiques.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Droits nécessaires pour modifier un groupe
<a name="droits_necessaires_pour_modifier_groupe_ELTPARAGRAPHE000250"></a>Pour modifier un groupe (différent de celui de l'utilisateur en cours) , il est nécessaire de posséder le droit de visualisation des utilisateurs (constante *hDroitVisualisationUtilisateur*). Dans le cas contraire, la fonction **&lt;Variable Connexion&gt;.ModifieGroupe** renverra <u><u><u><u>Faux</u></u></u></u> et le message d'erreur sera "Groupe inconnu".

Pour modifier le nom d'un groupe, il est nécessaire de posséder le droit de visualisation des utilisateurs (constante *hDroitVisualisationUtilisateur*) et le droit de gestion des utilisateurs (constante *hDroitGestionUtilisateur*).

Remarque : Ces constantes sont utilisées dans les fonctions [&lt;Variable Connexion&gt;.InfoDroitServeur](../WDLang4/1000022668.md) et [&lt;Variable Connexion&gt;.ModifieDroitServeur](../WDLang4/1000022710.md).
<a name="NOTE0_2"></a>


### Gestion des droits
<a name="gestion_des_droits_ELTPARAGRAPHE000281"></a>Il est possible de donner des droits à un groupe grâce aux fonctions :

- [&lt;Variable Connexion&gt;.ModifieDroitBaseDeDonnées](../WDLang4/1000022707.md) : permet de donner des droits sur une base de données.

- [&lt;Variable Connexion&gt;.ModifieDroitFichier](../WDLang4/1000022709.md) : permet de donner des droits sur un fichier de données.

- [&lt;Variable Connexion&gt;.ModifieDroitServeur](../WDLang4/1000022710.md) : permet de donner des droits sur un serveur.




Pour plus de détails sur la gestion des droits, consultez [Gestion des droits en HFSQL Client/Serveur](../WDLang4/3044333.md).

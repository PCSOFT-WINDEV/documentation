
## HAnnuleAlias (Fonction)

***En anglais : HCancelAlias***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Annule un alias précédemment déclaré avec la fonction [HAlias](../WDLang4/3044176.md). Après cette opération, le fichier de données associé à l'alias n'est plus connu par le moteur HFSQL.

**Remarque** : Si le fichier alias est ouvert, il est automatiquement fermé lors de l'annulation.






<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Crée 2 alias sur le fichier de données CLIENT
HAlias(CLIENT, CLIENT1)
HAlias(CLIENT, CLIENT2)
// Annule le premier alias
HAnnuleAlias(CLIENT1)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = HAnnuleAlias(<Fichier alias>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'opération a été réalisée, 

- <u><u><u><u>Faux</u></u></u></u> en cas de problème. La fonction [HErreur](../WDLang4/3044088.md) permet d'identifier l'erreur.




**`<Fichier alias> : (Chaîne de caractères)`**

Nom du fichier alias à annuler. Si le fichier alias n'existe pas, une erreur WLangage non fatale apparaît.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Lors de l'annulation d'un alias (fonction **HAnnuleAlias**) : 

- les modifications réalisées avec les fonctions [HChangeNom](../WDLang4/3044141.md) et [HChangeRep](../WDLang4/3044168.md) ne sont pas annulées. Si l'alias est à nouveau décrit, ces paramètres sont conservés et ré-appliqués.

- si un fichier de données a été créé avec un nom donné par la fonction [HChangeNom](../WDLang4/3044141.md) et un emplacement donné par la fonction [HChangeRep](../WDLang4/3044168.md), le fichier de données n'est pas supprimé physiquement. 

	- Pour supprimer un fichier de données créé à partir d'un alias, il faut utiliser la fonction [HSupprimeFichier](../WDLang4/3044294.md). 

	- Pour supprimer un dossier créé à partir d'un alias, il faut utiliser la fonction [HSupprimeRépertoire](../WDLang4/1000019001.md).

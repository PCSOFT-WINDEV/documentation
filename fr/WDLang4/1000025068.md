


## &lt;Source&gt;.StatNbEnr (Fonction)

***En anglais : &lt;Source&gt;.StatNbRec***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le nombre d'entrées pour une rubrique clé donnée. Ce nombre a été calculé lors du dernier calcul de statistiques.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Réalisation de statistiques sur la clé NomCli du fichier de données Client
ResultatStat = Client.StatNbEnr(NomCli)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Source>.StatNbEnr([<Rubrique clé>])`
---

**`<Résultat> : (Entier)`**

Nombre d'entrées trouvées pour la rubrique spécifiée. 
Correspond à la constante *hNumEnrInvalide* si :

- les statistiques n'ont jamais été calculées pour la rubrique spécifiée.

- les statistiques ne sont pas supportées pour le type de la rubrique spécifiée. HFSQL permet de calculer des statistiques uniquement sur des clés.




**`<Source> : (Type correspondant à la source spécifiée)`**

Nom du fichier de données manipulé. 

**`<Rubrique clé> : (Chaîne de caractères optionnelle)`**

Nom de la rubrique clé permettant de réaliser les statistiques. Si ce nom n'est pas spécifié, la fonction **&lt;Source&gt;.StatNbEnr** va manipuler la dernière rubrique utilisée par la dernière fonction de gestion HFSQL (fonction commençant par la lettre H).



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- La mise à jour des statistiques peut être réalisée par les fonctions [&lt;Source&gt;.Réindexe](../WDLang4/1000025060.md) et [&lt;Source&gt;.StatCalcule](../WDLang4/1000025064.md).

- Dans les fichiers de données HFSQL Classic, seules les rubriques clés peuvent faire l'objet d'un calcul de statistiques.

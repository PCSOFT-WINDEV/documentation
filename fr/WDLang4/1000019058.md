
## HClusterArrête (Fonction)

***En anglais : HClusterStop***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Suspend le fonctionnement d'un cluster HFSQL. Les noeuds du cluster n'accepteront plus de connexions de la part des applications clientes.

Le cluster pourra être redémarré grâce à la fonction [HClusterDémarre](../WDLang4/1000019057.md).

**Note** : Cette fonction est réservée à une utilisation avancée. Il est recommandé d'utiliser le Centre de Contrôle HFSQL pour gérer un cluster HFSQL.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
HClusterArrête("Host1", "MonMotDePasse")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = HClusterArrête(<Coordinateur> , <Mot de passe>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la fonction a réussi, 

- <u><u><u><u>Faux</u></u></u></u> sinon. En cas d'erreur, la fonction [ErreurInfo](../WDLang1/3013008.md) retourne le détail de l'erreur.




**`<Coordinateur> : (Chaîne de caractères)`**

Nom DNS de la machine où se trouve le coordinateur du cluster. 

**`<Mot de passe> : (Chaîne de caractères)`**

Mot de passe permettant de se connecter au coordinateur. Ce mot de passe a été fourni lors de [l'installation du cluster](../WDLang4/1000019031.md).




## &lt;Variable hdfsConnexion&gt;.Liste (Fonction)

***En anglais : &lt;hdfsConnection variable&gt;.List***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Liste le contenu d'un répertoire sur un serveur Hadoop.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connexion
MaConnexion est une hdfsConnexion
MaConnexion.namenode = "MonServeur"
MaConnexion.Utilisateur = "Admin"
MaConnexion.OuvreConnexion()
// Récupération du contenu du répertoire
Contenu est un tableau de hdfsInformation = MaConnexion.Liste("/")
// Affichage du contenu du répertoire
POUR TOUT Elem DE Contenu
	Trace(Elem.Chemin)
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Connexion>.Liste(<Chemin>)`
---

**`<Résultat> : (Tableau)`**

Tableau de variables de type [hdfsInformation](../WDLang4/1000021973.md) correspondant au contenu du répertoire.

**`<Connexion> : (Variable de type hdfsConnexion)`**

Nom de la variable de type [hdfsConnexion](../WDLang4/1000021650.md) correspondant à la connexion au serveur Hadoop.

**`<Chemin> : (Chaîne de caractères)`**

Chemin absolu du répertoire sur le serveur Hadoop dont le contenu doit être listé. Ce chemin est sensible à la casse.

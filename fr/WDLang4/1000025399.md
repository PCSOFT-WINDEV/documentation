


## &lt;Variable mongoBase&gt;.SupprimeBase (Fonction)

***En anglais : &lt;mongoDatabase variable&gt;.DeleteDatabase***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime une base MongoDB et toutes ses collections.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
CnxBaseMDB est un mongoConnexion
MaBaseMongo est un mongoBase
MaBaseMongo = CnxBaseMDB.BaseDeDonnées["CRM"]
MaBaseMongo.SupprimeBase()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Base>.SupprimeBase()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'opération a été réalisée,

- <u><u><u><u>Faux</u></u></u></u> en cas de problème. La fonction [ErreurInfo](../WDLang1/3013008.md) permet d'identifier l'erreur.




**`<Base> : (Variable de type mongoBase)`**

Nom de la variable de type [mongoBase](../WDLang4/1000022410.md) correspondant à la base MongoDB à supprimer.




## &lt;Variable mongoCollection&gt;.SupprimeUn (Fonction)

***En anglais : &lt;mongoCollection variable&gt;.DeleteOne***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime un document dans une collection d'une base MongoDB.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MaCollection est un mongoCollection
soit sFiltre = "{ ""name"": ""Andy"" }"
soit sTri = "{ ""rating"": 1 }"
soit d = MaCollection.SupprimeUn(sFiltre, sTri)
```
<a name="Exemple2"></a>

```wl
gclCollection est un mongoCollection
// Supprime le premier contact dont le membre "Prénom" vaut "Jules"
// Filtre de recherche
sFiltre est une chaîne = "{ ""Prenom"": ""Jules"" }"
gclCollection.SupprimeUn(sFiltre)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Collection>.SupprimeUn(<Filtre> [, <Tri>])`
---

**`<Résultat> : (Variant)`**

Document supprimé. 

En cas de problème, la fonction [ErreurInfo](../WDLang1/3013008.md) permet d'identifier l'erreur.

**`<Collection> : (Variable de type mongoCollection)`**

Nom de la variable de type [mongoCollection](../WDLang4/1000022411.md) représentant la collection de la base MongoDB dans laquelle la suppression est effectuée.

**`<Filtre> : (Type correspondant au filtre utilisé)`**

Description du/des filtres au format MongoDB à appliquer pour sélectionner le document à supprimer. Ce filtre correspond à : 

- un document au format JSON, sous forme de chaîne de caractères. 

- un document au format BSON, sous forme de buffer. 

- un variant.




**`<Tri> : (Type optionnel correspondant au format utilisé)`**

Options de tri permettant de déterminer le premier document qui sera supprimé lorsque plusieurs documents correspondent au filtre. Correspond à : 

- un document au format JSON, sous forme de chaîne de caractères. 

- un document au format BSON, sous forme de buffer. 

- un variant.




## RedisPublie (Fonction)

***En anglais : RedisPublish***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Publie un message sur un canal de diffusion Redis.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
RedisPublie(gRedis, "MonCanal", "Hello Redis from WINDEV")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = RedisPublie(<Connexion> , <Canal> , <Message>)`
---

**`<Résultat> : (Entier)`**

Nombre d'applications ayant reçu le message. 

Remarque : Cette information n'est pas disponible lors de la connexion à un cluster Redis.

**`<Connexion> : (Variable de type redisConnexion)`**

Nom de la variable de type [redisConnexion](../WDLang4/1000023527.md) correspondant à la connexion au serveur Redis.

**`<Canal> : (Chaîne de caractères)`**

Nom du canal sur lequel le message doit être publié. Cette chaîne est encodée au format UTF8.

**`<Message> : (Chaîne de caractères ou Buffer)`**

Message à publier sur le canal : 

- Si ce paramètre est de type Chaîne, il sera converti au format UTF8.

- Si ce paramètre est de type Buffer, il sera envoyé sous forme binaire.

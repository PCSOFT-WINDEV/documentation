


## &lt;Variable Connexion&gt;.ListePlanification (Fonction)

***En anglais : &lt;Connection variable&gt;.ListScheduling***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Liste les tâches planifiées définies sur un serveur HFSQL : 

- procédure stockée, 

- sauvegarde, 

- optimisation, 

- rafraîchissement d'une vue matérialisée.





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Description de la connexion
MaConnexionBase est une Connexion

// Modification de toutes les sauvegardes planifiées
tabDescSvg est un tableau dynamique de hDescriptionSauvegarde 
tabDescSvg = MaConnexionBase.ListePlanification(hlpSauvegarde)
hDescSvg est une hDescriptionSauvegarde
POUR TOUT hDescSvg DE tabDescSvg
	SI hDescSvg.LimiteNombreSauvegarde = 3 ALORS 
		hDescSvg.LimiteNombreSauvegarde = 5
		MaConnexionBase.ModifiePlanification(hDescSvg)
	FIN
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Connexion>.ListePlanification(<Type de planification>)`
---

**`<Résultat> : (Tableau)`**

Tableau contenant les descriptions listées. Le type des éléments du tableau dépend du paramètre &lt;Type de planification&gt;.

**`<Connexion> : (Variable de type Connexion)`**

Nom de la variable de type [Connexion](../WDLang4/1514073.md) décrivant la connexion à manipuler.

**`<Type de planification> : (Constante de type Entier)`**

Type de planification à lister : 


|   |   |
| --- | --- |
| *hlpOptimisation* | Tâches d'optimisation des fichiers de données (réindexation, calcul des statistiques). |
| *hlpSauvegarde* | Planifications des sauvegardes. |
| *hlpTâche* | Tâches planifiées exécutant une procédure stockée. |
| *hlpVueMatérialisée* | Tâches de rafraîchissement des vues matérialisées. |

<a name="NOTE0_1"></a>

## Remarques


### Droits nécessaires
<a name="droits_necessaires_ELTPARAGRAPHE000223"></a>Pour lister les tâches de rafraîchissement des vues matérialisées ou les tâches planifiées, il est nécessaire de posséder le droit de connexion (constante *hDroitConnexion* ou *hDroitConnexionCryptée*) pour chaque base de données concernée, c'est-à-dire pour chaque base de données qui contient une vue à traiter ou une procédure stockée à exécuter. 

Pour savoir si un utilisateur ou un groupe possède ces droits, utilisez la fonction [&lt;Variable Connexion&gt;.InfoDroitServeur](../WDLang4/1000022668.md) ou la fonction [HInfoDroitDaseDeDonnées](../WDLang4/1000022666.md).

Pour modifier les droits d'un utilisateur ou d'un groupe d'utilisateurs, utilisez la fonction [&lt;Variable Connexion&gt;.ModifieDroitServeur](../WDLang4/1000022710.md) ou la fonction  [&lt;Variable Connexion&gt;.ModifieDroitBaseDeDonnées](../WDLang4/1000022707.md).

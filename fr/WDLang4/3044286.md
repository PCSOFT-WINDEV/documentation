


## HInterditAccèsBaseDeDonnées (Fonction)

***En anglais : HNoDatabaseAccess***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Interdit toutes les nouvelles connexions à une base de données ou bien à toutes les bases de données. 

Remarques : 

- Les lectures et les écritures fonctionnent normalement. Seuls les nouvelles connexions à la base de données sont interdites. 

- Pour ré-autoriser les connexions, utilisez la fonction [HFinInterditAccèsBaseDeDonnées](../WDLang4/3044287.md).





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Description de la connexion
HDécritConnexion("MaConnexion", "Julie", "MotDePasse", "MonServeurHF", "BaseClient", ...
	hAccèsHFClientServeur, hOLecture)
// Ouverture de la connexion
HOuvreConnexion("MaConnexion")
SI HInterditAccèsBaseDeDonnées("MaConnexion", "BaseClient") ALORS
	Info("Il est impossible de se connecter à la base de données")
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = HInterditAccèsBaseDeDonnées(<Connexion> [, <Base de données>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'accès (c'est-à-dire la connexion) à la base de données spécifiée (ou à toutes les bases de données accédées par la connexion) est interdit, 

- <u><u><u><u>Faux</u></u></u></u> en cas d'erreur. La fonction [HErreur](../WDLang4/3044088.md) permet d'identifier l'erreur.




**`<Connexion> : (Chaîne de caractères ou variable de type Connexion)`**

Connexion contenant la base de données dont l'accès doit être interdit. Cette connexion correspond : 

- soit au nom de la connexion définie sous l'éditeur d'analyses ou grâce aux fonctions [HDécritConnexion](../WDLang4/3044205.md) ou [HOuvreConnexion](../WDLang4/3044107.md).

- soit au nom d'une variable de type [Connexion](../WDLang4/1514073.md).




**`<Base de données> : (Chaîne de caractères optionnelle)`**

Nom de la base de données pour laquelle l'accès doit être interdit.
Si ce paramètre vaut "\*", l'accès sera interdit pour toutes les bases de données.
Si ce paramètre n'est pas spécifié, l'accès sera interdit pour les bases de données utilisées par la connexion &lt;Connexion&gt;.

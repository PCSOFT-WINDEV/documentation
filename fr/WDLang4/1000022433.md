


## MongoCherche (Fonction)

***En anglais : MongoFind***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Cherche des documents dans une collection MongoDB.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple
<a class="notetitle" target="_blank" href="$DOC$=1000022623&name=mongocherche_fonction&product=WD">Voir des exemples supplémentaires</a>

```wl
sFiltre est une chaîne
sFiltre = "{ ""name"": ""France"" }"

soit MaRecherche = MongoCherche(CollecMDB, sFiltre)

// Lecture du résultat
POUR TOUT item de MaRecherche
	Trace(VariantVersJson(item))
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = MongoCherche(<Collection> , <Filtre> [, <Option>])`
---

**`<Résultat> : (Variable de type mongoRésultat)`**

Variable de type [mongoRésultat](../WDLang4/1000022415.md) correspondant au résultat de la recherche. En cas de problème, la fonction [ErreurInfo](../WDLang1/3013008.md) permet d'identifier l'erreur.

**`<Collection> : (Variable de type mongoCollection)`**

Nom de la variable de type [mongoCollection](../WDLang4/1000022411.md) correspondant à la collection MongoDB dans laquelle la recherche doit être effectuée.

**`<Filtre> : (Type correspondant au filtre utilisé)`**

Description du/des filtres au format MongoDB à appliquer pour effectuer la recherche. Ce filtre correspond à : 

- un document au format JSON, sous forme de chaîne de caractères. 

- un document au format BSON, sous forme de buffer. 

- un variant. 




**`<Option> : (Variable optionnelle de type mongoChercheOption)`**

Nom de la variable de type [mongoChercheOption](../WDLang4/1000022414.md) correspondant aux options de recherche.

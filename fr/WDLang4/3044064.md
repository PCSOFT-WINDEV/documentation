


## HLibère (Fonction)

***En anglais : HFree***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Transforme les enregistrements rayés (fonction [HRaye](../WDLang4/3044121.md)) d'un fichier de données en enregistrements supprimés. La place occupée par les enregistrements rayés est ainsi libérée et pourra être réutilisée lors des prochains ajouts (fonction [HAjoute](../WDLang4/3044147.md)).

**Rappel** : Un enregistrement rayé est inactif et disparaît de l'index. Les données de cet enregistrement rayé ne sont pas supprimées (contrairement à la fonction [HSupprime](../WDLang4/3044018.md)).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Supprime tous les enregistrements rayés du fichier de données "Client"
HLibère(Client)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = HLibère([<Fichier de données>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'enregistrement a été supprimé, 

- <u><u><u><u>Faux</u></u></u></u> en cas de problème : l'enregistrement n'est pas supprimé. La fonction [HErreur](../WDLang4/3044088.md) permet d'identifier l'erreur.




**`<Fichier de données> : (Chaîne de caractères optionnelle)`**

Nom du fichier de données manipulé. Si ce nom n'est pas spécifié, la fonction **HLibère** va manipuler le dernier fichier de données utilisé par la dernière fonction de gestion HFSQL (fonction commençant par la lettre H).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Cette fonction n'est pas adaptée à des fichiers xBase.

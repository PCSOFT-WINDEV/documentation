


## &lt;Variable Connexion&gt;.MaintenanceServeur (Fonction)

***En anglais : &lt;Connection variable&gt;.ServerMaintenance***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Vérifie et optimise les tables systèmes d'un serveur HFSQL.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Description de la connexion
MaConnexion est une Connexion

MaConnexion.MaintenanceServeur()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Connexion>.MaintenanceServeur([<Base de données>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si les tables système ont été vérifiées et optimisées, 

- <u><u><u><u>Faux</u></u></u></u> en cas d'erreur. La fonction [HErreur](../WDLang4/3044088.md) permet d'identifier l'erreur.




**`<Connexion> : (Variable de type Connexion)`**

Nom de la variable de type [Connexion](../WDLang4/1514073.md) décrivant la connexion pour laquelle les tables système doivent être vérifiées et optimisées.

**`<Base de données> : (Chaîne de caractères optionnelle)`**

Nom de la base de données contenant les tables système à vérifier et optimiser. Si ce paramètre n'est pas précisé, cette fonction vérifiera et optimisera les tables système globales.




## &lt;Variable Connexion&gt;.DéclencheMiseAJourServeur (Fonction)

***En anglais : &lt;Connection variable&gt;.StartServerUpdate***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de déclencher immédiatement la mise à jour automatique du serveur. Les utilisateurs seront déconnectés lors du lancement de cette opération.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Description de la connexion
MaConnexion est une Connexion

// Déclenche la mise à jour du serveur
MaConnexion.DéclencheMiseAJourServeur()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Connexion>.DéclencheMiseAJourServeur([<Version>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'opération a été réalisée, 

- <u><u><u><u>Faux</u></u></u></u> en cas de problème.


La fonction [HErreurInfo](../WDLang4/3044071.md) permet d'identifier l'erreur.

**`<Connexion> : (Variable de type Connexion)`**

Nom de la variable de type [Connexion](../WDLang4/1514073.md) décrivant la connexion à utiliser pour la mise à jour du serveur.

**`<Version> : (Chaîne de caractères optionnelle)`**

Version pour laquelle la mise à jour doit être effectuée (par exemple : "240036"). Par défaut, si ce paramètre n'est pas spécifié, la dernière mise à jour disponible est utilisée. 




## Remarques
Le serveur de mise à jour (contenant le pack de mise à jour du serveur HFSQL) peut être défini par la fonction [&lt;Variable Connexion&gt;.GèreMiseAJourServeur](../WDLang4/1000022662.md).

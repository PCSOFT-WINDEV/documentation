


## HJournalStop (Fonction)

***En anglais : HLogStop***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Arrête la journalisation du fichier de données. Les manipulations effectuées dans le fichier journalé ne sont plus enregistrées.

**Remarque** : La journalisation n'est pas disponible lors de l'accès à une base de données via .


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
HJournalInfo(Client, "début impression de " + Critère)
HJournalStop(Client) // Stopper le journal
HLitPremier(Client, Critère)
TANTQUE HTrouve() = Vrai
	// Traitement
	 HLitSuivant(Client)
FIN

// Redémarrer le journal
HJournalRedémarre(Client) 
// Ecrire dans le journal
HJournalInfo(Client, "fin impression de " + Critère)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = HJournalStop(<Fichier de données>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'opération a été réalisée, 

- <u><u><u><u>Faux</u></u></u></u> en cas de problème. La fonction [HErreur](../WDLang4/3044088.md) permet d'identifier l'erreur.




**`<Fichier de données> : (Chaîne de caractères)`**

Nom du fichier de données journalé dont la journalisation doit être arrêtée.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Les opérations effectuées après l'utilisation de cette fonction ne seront pas journalées et seront perdues en cas d'incident.

- Ne pas oublier de relancer la journalisation par la fonction [HJournalRedémarre](../WDLang4/3044019.md) en s'assurant que le fichier de données est rigoureusement dans son état initial.

- Le journal est arrêté jusqu'à l'exécution de la fonction [HJournalRedémarre](../WDLang4/3044019.md).

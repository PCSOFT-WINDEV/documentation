<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000170"></a>
WINDEV, WEBDEV et WINDEV Mobile proposent différentes façons de décrire une connexion :

- Soit directement sous l'éditeur d'analyses,

- Soit par les fonctions [HDécritConnexion](../WDLang4/3044205.md), [HChangeConnexion](../WDLang4/3044150.md), ...

- Soit en utilisant le type [Connexion](../WDLang4/1514073.md) et les propriétés du WLangage.

## Description d'une connexion sous l'éditeur d'analyses
<a name="description_une_connexion_sous_editeur_analyses_ELTTEXTE000200"></a>


### Pour décrire une connexion sous l'éditeur d'analyses : 
<a name="pour_decrire_une_connexion_sous_editeur_analyses_ELTPARAGRAPHE000040"></a>

1. Ouvrez l'éditeur d'analyses.

2. Sous le volet "Analyse", dans le groupe "Connexion", cliquez sur "Connexions".

3. Dans l'assistant de création d'une connexion, indiquez :

	- Le type de connexion à créer (HFSQL Client/Serveur ou Base externe),

	- Les caractéristiques de la connexion (port, base de données, utilisateur, mot de passe, ...).




4. Si une connexion a déjà été décrite dans l'analyse, la liste des connexions ainsi que leurs caractéristiques apparaît et peut être modifiée et testée.

## Description d'une connexion par les fonctions HDécritConnexion, ...
<a name="description_une_connexion_par_les_fonctions_hdecritconnexion_ELTTEXTE000224"></a>
Les fonctions [HDécritConnexion](../WDLang4/3044205.md), [HChangeConnexion](../WDLang4/3044150.md), ... permettent de décrire une connexion et de l'utiliser. Le principe est le suivant :

- La fonction [HDécritConnexion](../WDLang4/3044205.md) décrit une nouvelle connexion à une base de données externe. Cette connexion est nommée. Des paramètres spécifiques peuvent être définis tels que le mode de gestions des curseurs (si la base de données accédée le permet).

- Pour associer cette connexion à une table ou un fichier, utilisez la fonction [HChangeConnexion](../WDLang4/3044150.md).

- Pour ouvrir la connexion, utilisez la fonction [HOuvreConnexion](../WDLang4/3044107.md). Si les paramètres de la connexion décrite avec **HDécritConnexion** ne sont pas corrects, la fonction [HOuvreConnexion](../WDLang4/3044107.md) renverra une erreur.

- Grâce à la fonction [HDécritConnexion](../WDLang4/3044205.md), une description de table Oracle importée dans l'analyse peut être manipulée directement avec des fichiers de données HFSQL Classic. Il suffit de décrire une connexion temporaire à une base de données HFSQL Classic puis de l'associer à la description de fichier présente dans l'analyse.

## Description d'une connexion grâce au type Connexion et à ses propriétés
<a name="description_une_connexion_grace_type_connexion_ses_proprietes_ELTTEXTE000248"></a>
Une variable de type [Connexion](../WDLang4/1514073.md) permet de décrire une ou plusieurs connexions à des bases de données externes (HFSQL Client/Serveur, Connecteur Natif, ...).

La connexion pourra être ouverte grâce à la fonction [HOuvreConnexion](../WDLang4/3044107.md).

La variable de type [Connexion](../WDLang4/1514073.md) permet aussi bien de :

- décrire une nouvelle connexion (équivalent à la fonction [HDécritConnexion](../WDLang4/3044205.md)).

- créer une connexion (qui sera ouverte grâce à la fonction [HOuvreConnexion](../WDLang4/3044107.md)).

- modifier une connexion existante.




## &lt;Variable hdfsConnexion&gt;.FermeConnexion (Fonction)

***En anglais : &lt;hdfsConnection variable&gt;.CloseConnection***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ferme une connexion à un serveur Hadoop.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connexion
MaConnexion est une hdfsConnexion
MaConnexion.namenode = "MonServeur"
MaConnexion.Utilisateur = "Admin"
SI MaConnexion.OuvreConnexion() ALORS
	// Fermeture de la connexion
	MaConnexion.FermeConnexion()
SINON
	Erreur()
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Connexion>.FermeConnexion()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la déconnexion a été réalisée,

- <u><u><u><u>Faux</u></u></u></u> en cas de problème. La fonction [ErreurInfo](../WDLang1/3013008.md) permet d'identifier l'erreur.




**`<Connexion> : (Variable de type hdfsConnexion)`**

Nom de la variable de type [hdfsConnexion](../WDLang4/1000021650.md) correspondant à la connexion au serveur Hadoop à fermer.

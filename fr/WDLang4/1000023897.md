


## Fonctions MongoDB (syntaxe préfixée)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions de gestion des bases MongoDB (syntaxe préfixée) sont les suivantes : 



|   |   |
| --- | --- |
| [&lt;Variable mongoBase&gt;.CréeCollection](../WDLang4/1000023612.md) | Crée une collection dans une base MongoDB en précisant des options de cette collection. |
| [&lt;Variable mongoBase&gt;.ExécuteCommande](../WDLang4/1000023636.md) | Exécute une commande générique sur une base MongoDB. |
| [&lt;Variable mongoBase&gt;.Info](../WDLang4/1000025396.md) | Renvoie des informations sur une base MongoDB. |
| [&lt;Variable mongoBase&gt;.SupprimeBase](../WDLang4/1000025399.md) | Supprime une base MongoDB et toutes ses collections. |
| [&lt;Variable mongoCollection&gt;.Ajoute](../WDLang4/1000025345.md) | Ajoute un ou plusieurs documents dans une collection d'une base MongoDB. |
| [&lt;Variable mongoCollection&gt;.Cherche](../WDLang4/1000025356.md) | Cherche des documents dans une collection MongoDB. |
| [&lt;Variable mongoCollection&gt;.ExécuteCommande](../WDLang4/1000023819.md) | Exécute une commande générique sur une collection MongoDB. |
| [&lt;Variable mongoCollection&gt;.ExécuteLotOpérations](../WDLang4/1410089633.md) | Effectue un ensemble d'opérations d'écriture MongoDB en lot. Il est ainsi possible d'éviter des latences réseau qui pourraient subvenir en effectuant les opérations séquentiellement. |
| [&lt;Variable mongoCollection&gt;.ModifieTout](../WDLang4/1000025397.md) | Modifie des documents dans une collection d'une base MongoDB. |
| [&lt;Variable mongoCollection&gt;.ModifieUn](../WDLang4/1000025398.md) | Modifie un document dans une collection d'une base MongoDB. |
| [&lt;Variable mongoCollection&gt;.SupprimeCollection](../WDLang4/1000025400.md) | Supprime une collection d'une base MongoDB. |
| [&lt;Variable mongoCollection&gt;.SupprimeTout](../WDLang4/1000025401.md) | Supprime tous les documents (correspondant au filtre spécifié) d'une collection d'une base MongoDB. |
| [&lt;Variable mongoCollection&gt;.SupprimeUn](../WDLang4/1000025402.md) | Supprime un document dans une collection d'une base MongoDB. |
| [&lt;Variable mongoGridFS&gt;.EnvoieFichier](../WDLang4/1000025357.md) | Copie un fichier du poste local vers un système de fichiers MongoDB (GridFS). |
| [&lt;Variable mongoGridFS&gt;.ListeFichier](../WDLang4/1000025358.md) | Récupère la liste des fichiers MongoDB présents dans le système de fichiers MongoDB (gridFS). |
| [&lt;Variable mongoGridFS&gt;.RécupèreFichier](../WDLang4/1000025391.md) | Récupère sur le poste local un fichier identifié par son nom et présent dans un système de fichiers MongoDB. |
| [&lt;Variable mongoGridFS&gt;.RécupèreFichierParID](../WDLang4/1000025392.md) | Récupère sur le poste local un fichier (identifié par son identifiant) présent dans un système de fichiers MongoDB. |
| [&lt;Variable mongoGridFS&gt;.RécupèreInfoFichier](../WDLang4/1000025393.md) | Permet de récupérer les informations concernant un fichier MongoDB (gridFS). |
| [&lt;Variable mongoGridFS&gt;.SupprimeFichier](../WDLang4/1000025394.md) | Supprime un fichier identifié par son nom dans un système de fichiers MongoDB. |
| [&lt;Variable mongoGridFS&gt;.SupprimeFichierParID](../WDLang4/1000025395.md) | Supprime un fichier identifié par son identifiant dans un système de fichiers MongoDB. |

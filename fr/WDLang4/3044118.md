


## HRAZ (Fonction)

***En anglais : HReset***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Initialise : 

- une ou toutes les variables des rubriques d'un fichier de données avec leurs valeurs par défaut. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) une ou toutes les rubriques d'une variable de type [Enregistrement](../WDLang4/1000020962.md) avec leurs valeurs par défaut.


Les valeurs par défaut sont définies lors de la description du fichier de données sous l'éditeur d'analyses.

**Remarques** : 

- La fonction **HRAZ** ne modifie pas l'enregistrement dans le fichier de données mais uniquement les données en mémoire. 

- La fonction **HRAZ** modifie la propriété [NouvelEnregistrement](../Proprietes/1000017090.md) du fichier de données concerné : cette propriété passe à <u><u><u>Vrai</u></u></u>.





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affecte les rubriques avec leur valeur par défaut
HRAZ(Client)
// Affichage
FichierVersPage()
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Initialiser les rubriques d'un fichier de données avec leurs valeurs par défaut

`<Résultat> = HRAZ([<Fichier de données> [, <Rubrique>]])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'initialisation a été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> en cas de problème. La fonction [HErreurInfo](../WDLang4/3044071.md) permet d'obtenir plus de détails.




**`<Fichier de données> : (Chaîne de caractères optionnelle)`**

Nom du fichier de données manipulé. Si ce nom n'est pas spécifié, la fonction **HRAZ** va manipuler le dernier fichier de données utilisé par la dernière fonction de gestion HFSQL (fonction commençant par la lettre H).

**`<Rubrique> : (Chaîne de caractères optionnelle)`**

Nom de la rubrique à initialiser. Si ce paramètre n'est pas spécifié ou est égal à "\*", l'initialisation sera effectuée pour toutes les rubriques du fichier de données.
![Hyper File 5.5](https://doc.pcsoft.fr/ext/images/fr/HF55.png) Ce paramètre n'est pas pris en compte. Toutes les rubriques sont automatiquement ré-initialisées. 


<a name="SYNTAXE2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) 
### Initialiser les rubriques d'un enregistrement avec leurs valeurs par défaut (variable de type Enregistrement)

`<Résultat> = HRAZ(<Enregistrement> [, <Rubrique>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'initialisation a été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> en cas de problème. La fonction [HErreurInfo](../WDLang4/3044071.md) permet d'obtenir plus de détails.




**`<Enregistrement> : (Variable de type Enregistrement)`**

Nom de la variable de type [Enregistrement](../WDLang4/1000020962.md) manipulée. 

**`<Rubrique> : (Chaîne de caractères optionnelle)`**

Nom de la rubrique à initialiser. Si ce paramètre n'est pas spécifié ou est égal à "\*", l'initialisation sera effectuée pour toutes les rubriques de l'enregistrement.
![Hyper File 5.5](https://doc.pcsoft.fr/ext/images/fr/HF55.png) Ce paramètre n'est pas pris en compte. Toutes les rubriques sont automatiquement ré-initialisées.

<a name="NOTE0_1"></a>

## Remarques


### Ouverture de fichiers
<a name="ouverture_fichiers_ELTPARAGRAPHE000203"></a>La fonction **HRAZ** effectue une ouverture du fichier de données manipulé. Le fichier de données doit donc avoir été créé.
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Lors d'une gestion personnalisée des erreurs, il est nécessaire de tester la fonction [HErreurBlocage](../WDLang4/3044099.md) après l'utilisation de la fonction **HRAZ**.
<a name="NOTE0_2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) 

### Variable de type Enregistrement
<a name="variable_type_enregistrement_ELTPARAGRAPHE000230"></a>Pour réinitialiser une variable de type [Enregistrement](../WDLang4/1000020962.md), il est également possible d'utiliser la fonction [VariableRAZ](../WDLang1/3013069.md).

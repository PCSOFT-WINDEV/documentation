


## &lt;Source&gt;.GèreTransaction (Fonction)

***En anglais : &lt;Source&gt;.SetTransaction***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Active ou désactive la gestion des transactions sur un ou plusieurs fichiers de données.

Lorsque la gestion des transaction est désactivée, toutes les fonctions relatives aux transactions sont inhibées ([&lt;Variable Connexion&gt;.TransactionDébut](../WDLang4/1000023967.md), [&lt;Variable Connexion&gt;.TransactionFin](../WDLang4/1000023968.md), [&lt;Variable Connexion&gt;.TransactionAnnule](../WDLang4/1000023966.md)).

**Remarque** : Par défaut, la gestion des transactions est active.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Désactive la gestion des transactions sur le fichier de données Client
Client.GèreTransaction(Faux)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Source>.GèreTransaction(<Options>)`
---

**`<Résultat> : (Booléen)`**

Correspond à l'ancienne gestion des transactions (avant utilisation de la fonction **&lt;Source&gt;.GèreTransaction**) : 

- <u><u><u><u>Vrai</u></u></u></u> si la gestion des transactions était active, 

- <u><u><u><u>Faux</u></u></u></u> si la gestion des transactions n'était pas active. 




**`<Source> : (Type correspondant à la source spécifiée)`**

Nom du fichier de données manipulé. 

**`<Options> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la gestion des transactions sur le fichier de données doit être activée, 

- <u><u><u><u>Faux</u></u></u></u> si cette gestion doit être désactivée.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Gestion des transactions activée
<a name="gestion_des_transactions_activee_ELTPARAGRAPHE000258"></a>Si la gestion des transactions est activée pour un seul fichier de données, seules les manipulations réalisées sur ce fichier de données seront prises en compte dans la gestion des transactions (quels que soit les fichiers de données indiqués dans la fonction [&lt;Variable Connexion&gt;.TransactionDébut](../WDLang4/1000023967.md)).

### Limite 
<a name="limite_ELTPARAGRAPHE000275"></a>La gestion des transactions ne s'applique pas aux fichiers xBase.




## &lt;Variable Connexion&gt;.SupprimeVue (Fonction)

***En anglais : &lt;Connection variable&gt;.DropView***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime une vue matérialisée sur le serveur HFSQL.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Création d'une vue matérialisée sur un serveur
SI PAS MaConnexion1.MiseAJourVue(MaVueMatérialisée) ALORS
	Trace(HErreurInfo())
FIN

// Rafraichissement de la vue matérialisée sur le serveur
SI PAS MaConnexion1.RafraîchitVue(MaVueMatérialisée) ALORS
	Trace(HErreurInfo())
FIN

// Suppression de la vue matérialisée sur le serveur
SI PAS MaConnexion1.SupprimeVue(MaVueMatérialisée) ALORS 
	Trace(HErreurInfo())
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Connexion>.SupprimeVue(<Vue> [, <Suppression en cascade>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'opération a été réalisée, 

- <u><u><u><u>Faux</u></u></u></u> en cas de problème. La fonction [HErreurInfo](../WDLang4/3044071.md) permet d'identifier l'erreur.




**`<Connexion> : (Nom de champ)`**

Connexion vers le serveur HFSQL qui contient la vue matérialisée. Cette connexion correspond :

- soit au nom de la connexion définie sous l'éditeur d'analyses ou grâce aux fonctions [HDécritConnexion](../WDLang4/3044205.md) ou [&lt;Variable Connexion&gt;.OuvreConnexion](../WDLang4/1000023962.md).

- soit au nom d'une variable de type [Connexion](../WDLang4/1514073.md).




**`<Vue> : (Chaîne de caractères)`**

Nom de la vue matérialisée à manipuler, présente sur le serveur.

**`<Suppression en cascade> : (Booléen optionnel)`**



- <u><u><u><u>Vrai</u></u></u></u> pour gérer la suppression en cascade, 

- <u><u><u><u>Faux</u></u></u></u> (valeur par défaut) dans le cas contraire.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Si la vue matérialisée n'existe pas, la fonction retourne une erreur non fatale.

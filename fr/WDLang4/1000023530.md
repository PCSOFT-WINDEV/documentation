


## RedisConnecte (Fonction)

***En anglais : RedisConnect***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ouvre une connexion vers un serveur Redis (ou un cluster Redis).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
gRedis est un redisConnexion
gRedis.Serveur = "MonServeurRedis:7001"
SI RedisConnecte(gRedis) = Faux ALORS
	ErreurInfo()
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = RedisConnecte(<Connexion>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la connexion a réussi, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. La fonction [ErreurInfo](../WDLang1/3013008.md) permet d'identifier l'erreur. Pour plus de détails, consultez la documentation de Redis.




**`<Connexion> : (Variable de type redisConnexion)`**

Nom de la variable de type [redisConnexion](../WDLang4/1000023527.md) correspondant à la connexion à un serveur Redis (ou à un cluster Redis).

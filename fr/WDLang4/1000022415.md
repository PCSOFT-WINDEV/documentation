
## mongoRésultat (Type de variable)

***En anglais : mongoResult***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **mongoRésultat** permet de connaître le résultat lors d'une recherche dans une base de type MongoDB. 

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
sFiltre est une chaîne
sFiltre = "{ ""name"": ""France"" }"

soit MaRecherche = MongoCherche(CollecMDB, sFiltre)

// Lecture du résultat
POUR TOUT item de MaRecherche
	Trace(VariantVersJson(item))
FIN
```





<a name="NOTE0"></a>

## Remarques
<a name="NOTE0_1"></a>


### Fonctionnement
<a name="fonctionnement_ELTPARAGRAPHE000040"></a>

- Le type de variable **mongoRésultat** peut être parcouru grâce à l'instruction POUR TOUT. 

- Chaque élément parcouru est de type **Variant**. 

- Chaque élément représente un document MongoDB.







### Fonctions manipulant le type mongoRésultat
<a name="fonctions_manipulant_type_mongoresultat_ELTPARAGRAPHE000048"></a>Les fonctions manipulant une variable de type mongoRésultat sont les suivantes : 



|   |   |
| --- | --- |
| [MongoCherche](../WDLang4/1000022433.md) | Cherche des documents dans une collection MongoDB. |
| [MongoExécuteCommande](../WDLang4/1000022435.md) | Exécute une commande générique sur une base MongoDB ou sur une collection MongoDB. |
| [MongoPipelineAgrégation](../WDLang4/1000025267.md) | Permet de récupérer des documents issus de la transformation et de l'agrégation en plusieurs étapes. |

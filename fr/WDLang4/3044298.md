


## HDésactiveFiltreAuto (Fonction)

***En anglais : HDeactivateAutoFilter***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Désactive un filtre automatique sur les fichiers de données reliés lors du parcours d'un fichier XML. Ce filtre a été activé grâce à la fonction [HActiveFiltreAuto](../WDLang4/3044297.md).

**Attention** : Cette fonction est disponible uniquement pour l'Accès Natif XML.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Activation de tous les filtres auto possibles du fichier de données Client
HActiveFiltreAuto(Client, "*")
// sauf entre Client et Commande
HDésactiveFiltreAuto(Client, Commande)

/////////////////////////////

// Parcours des commandes
HLitPremier(Commande, IDCommande)
TANTQUE HEnDehors() = Faux
	// Parcours les lignes de commande de la commande en cours
	HLitPremier(LigneCde, IDCommande)
	TANTQUE HEnDehors() = Faux
		HLitSuivant(LigneCde, IDCommande)
	FIN
	// Commande suivante
	HLitSuivant(Commande, IDCommande)
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = HDésactiveFiltreAuto(<Fichier de données de parcours> , <Fichier de données à ne pas filtrer>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le filtre est désactivé, 

- <u><u><u><u>Faux</u></u></u></u> en cas d'erreur (fichier de données ou rubrique inexistant).




**`<Fichier de données de parcours> : (Chaîne de caractères)`**



- Nom du fichier de données utilisé pour le parcours.

- "\*" pour désactiver tous les filtres automatiques possibles.




Si les paramètres &lt;Fichier de données de parcours&gt; et &lt;Fichier de données à ne pas filtrer&gt; correspondent à "\*", &lt;Résultat&gt; vaut <u><u><u><u>Vrai</u></u></u></u> et aucun filtre n'est désactivé.

**`<Fichier de données à ne pas filtrer> : (Chaîne de caractères)`**



- Nom du fichier de données à ne pas filtrer lors d'un positionnement dans le fichier de données de parcours.

- "\*" pour désactiver tous les filtres automatiques possibles.




Si les paramètres &lt;Fichier de données de parcours&gt; et &lt;Fichier de données à ne pas filtrer&gt; correspondent à "\*", &lt;Résultat&gt; vaut <u><u><u><u>Vrai</u></u></u></u> et aucun filtre n'est désactivé.

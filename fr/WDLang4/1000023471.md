


## &lt;Variable Connexion&gt;.RSAjouteConfig (Fonction)

***En anglais : &lt;Connection variable&gt;.RSAddConfig***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ajoute une réplication entre deux serveurs HFSQL sur le poste serveur maître. 

<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MaConnexion est une Connexion

ConfigRéplication1 est un hRSConfig
ConfigRéplication1.Serveur = "atlas.masociete.com:4996"
Ajoute(ConfigRéplication1.Fichier, "Mabase")
Ajoute(ConfigRéplication1.Fichier, "-Mabase\Monfichier.fic")
MaConnexion.RSAjouteConfig(ConfigRéplication1, hrsSansCopie)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Connexion>.RSAjouteConfig(<Réplication à effectuer> , <Option> [, <Jauge>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la réplication a été ajoutée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. La fonction [HErreurInfo](../WDLang4/3044071.md) permet d'identifier l'erreur.




**`<Connexion> : (Variable de type Connexion)`**

Nom de la variable de type [Connexion](../WDLang4/1514073.md) décrivant la connexion vers le serveur HFSQL maître.

**`<Réplication à effectuer> : (Variable de type hRSConfig)`**

Nom de la variable de type [hRSConfig](../WDLang4/1000020484.md) décrivant la réplication à effectuer.

**`<Option> : (Constante de type Entier)`**

Action à réaliser : 


|   |   |
| --- | --- |
| *hrsCopie* | Active la copie initiale des éléments répliqués du serveur de la connexion (serveur maître) vers l'autre serveur (serveur abonné, défini dans la variable de type [hRSConfig](../WDLang4/1000020484.md)). Cette copie est bloquante. <br><br>Compatibilité avec les versions précédentes : Correspond à <u><u><u><u>Vrai</u></u></u></u> dans les versions précédentes. |
| *hrsCopieTâcheDeFond* | Active la copie initiale des éléments répliqués du serveur de la connexion (serveur maître) vers l'autre serveur (serveur abonné, défini dans la variable de type [hRSConfig](../WDLang4/1000020484.md)). Cette copie est réalisée en tâche de fond et n'est pas bloquante. <br><br>Pour savoir si la copie a été réalisée : <br><br>1. Listez les réplications disponibles avec la fonction [&lt;Variable Connexion&gt;.RSListeConfig](../WDLang4/1000023554.md). <br><br>2. Si la réplication voulue n'est pas listée, elle n'a pas été ajoutée. <br><br>3. Si la réplication voulue est présente, consultez la valeur de la propriété **Initialisation** de la variable [hRSConfig](../WDLang4/1000020484.md). Si cette propriété : <br><br>	- est comprise entre 0 et 1000, la réplication est en cours d'ajout. <br><br>	- vaut 1000, la réplication a été ajoutée. <br><br>	- n'est pas renseignée, l'ajout de la réplication est en échec.  <br><br><br><br><br><br> |
| *hrsSansCopie* | Aucune copie n'est réalisée. <br><br>Compatibilité avec les versions précédentes : Correspond à <u><u><u><u>Faux</u></u></u></u> dans les versions précédentes. |



**`<Jauge> : (Nom de champ optionnel)`**

Nom du champ Jauge permettant de visualiser l'avancement de la copie initiale.

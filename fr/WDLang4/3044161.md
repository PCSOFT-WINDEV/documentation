
## HDBTypeTri (Fonction)

***En anglais : HDBSortType***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie ou modifie la relation d'ordre des rubriques textes des fichiers xBase.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
nTypeTri est un entier
// Récupère le type de tri
nTypeTri = HDBTypeTri()

// Change le type de tri
SELON nTypeTri
	CAS hDBASCII : Info("Tri ASCII")
	CAS hDBLexicographique
			  HDBTypeTri(hDBASCII)
			  Info("Tri Lexicographique changé en tri ASCII")
	AUTRES CAS
			  Info("Erreur : Tri inconnu")
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Modifier la relation d'ordre utilisée

`<Résultat> = HDBTypeTri(<Type de tri>)`
---

**`<Résultat> : (Constante)`**

Type de tri utilisé avant le changement de tri :


|   |   |
| --- | --- |
| *hDBASCII* | Tri effectué selon l'ordre ASCII. |
| *hDBLexicographique* | Tri lexicographique. |



**`<Type de tri> : (Constante)`**

Type de tri à utiliser :


|   |   |
| --- | --- |
| *hDBASCII*<br>(Valeur par défaut) | Tri effectué selon l'ordre ASCII. |
| *hDBLexicographique* | Tri lexicographique. |






<a name="SYNTAXE2"></a>

### Connaître la relation d'ordre utilisée

`<Résultat> = HDBTypeTri()`
---

**`<Résultat> : (Constante)`**

Type de tri actuellement utilisé :


|   |   |
| --- | --- |
| *hDBASCII*<br>(Valeur par défaut) | Tri effectué selon l'ordre ASCII. |
| *hDBLexicographique* | Tri lexicographique. |







<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Cette fonction n'est pas disponible lors de l'utilisation d'une base de données via .

- Cette fonction n'est pas disponible lors de l'utilisation d'un exécutable autonome.

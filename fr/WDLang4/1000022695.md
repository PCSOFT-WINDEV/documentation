


## &lt;Variable Connexion&gt;.ListeOptimisationPlanifiée (Fonction)

***En anglais : &lt;Connection variable&gt;.ListScheduledOptimization***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Liste les tâches d'optimisation planifiées des fichiers de données HFSQL Client/Serveur pour une connexion.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MaConnexion est une Connexion
...
tabOptim est un tableau de hOptimisationPlanifiée
tabOptim = MaConnexion.ListeOptimisationPlanifiée()
POUR TOUT Optim DE tabOptim
	AfficheInfo(Optim)	
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Connexion>.ListeOptimisationPlanifiée()`
---

**`<Résultat> : (Tableau de variables de type hOptimisationPlanifiée)`**

Variable de type tableau de [hOptimisationPlanifiée](../WDLang4/1000018854.md). Chaque élément de ce tableau contient les caractéristiques d'une tâche d'optimisation planifiée. Le nombre d'éléments de ce tableau correspond au nombre de tâches d'optimisation planifiée disponibles.

**`<Connexion> : (Variable de type Connexion)`**

Nom de la variable de type [Connexion](../WDLang4/1514073.md) décrivant la connexion pour laquelle les tâches d'optimisation planifiées doivent être listées.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Droits nécessaires
<a name="droits_necessaires_ELTPARAGRAPHE000210"></a>Pour lister les tâches d'optimisation planifiées, il est nécessaire de posséder pour chaque base concernée, le droit de connexion (constante *hDroitConnexion* ou *hDroitConnexionCryptée*),

Pour une optimisation du serveur complet, l'utilisateur doit être super-utilisateur.

Pour savoir si un utilisateur ou un groupe possède ces droits, utilisez la fonction [&lt;Variable Connexion&gt;.InfoDroitServeur](../WDLang4/1000022668.md) ou [HInfoDroitDaseDeDonnées](../WDLang4/1000022666.md).

Pour modifier les droits d'un utilisateur ou d'un groupe d'utilisateurs, utilisez la fonction [&lt;Variable Connexion&gt;.ModifieDroitServeur](../WDLang4/1000022710.md) ou [&lt;Variable Connexion&gt;.ModifieDroitBaseDeDonnées](../WDLang4/1000022707.md).

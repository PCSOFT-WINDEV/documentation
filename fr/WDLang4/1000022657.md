


## &lt;Variable Connexion&gt;.EnvoieMessageVersClient (Fonction)

***En anglais : &lt;Connection variable&gt;.SendMessageToClient***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Affiche un message sur le(s) poste(s) Client. Les postes Client concernés sont les postes renseignés dans la structure [HClient](../WDLang4/3044340.md).




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Description de la connexion
MaConnexion est une Connexion
...
// Message à tous les utilisateurs de la base de données "BAL"
HClient.Base = "BAL"
MaConnexion.EnvoieMessageVersClient("Attention : Déconnexion imminente", 300)
```
<a name="3044346_Exemple2"></a>

```wl
// Description de la connexion
MaConnexion est une Connexion
...
// Message à l'utilisateur "Admin" de la base de données "MaBase"
HClient.Base = "MaBase"
HClient.Login = "Admin"
MaConnexion.EnvoieMessageVersClient("Test d'envoi de message", 15)
```
<a name="3044346_Exemple3"></a>

```wl
// Description de la connexion
MaConnexion est une Connexion
...
// Message à toute application accédant au fichier Client.fic 
// de la base de données "MaBase"
HClient.Base = "MaBase"
HClient.Fichier = "Client.fic"
MaConnexion.EnvoieMessageVersClient("Vous utilisez le fichier Client", 300)
```
<a name="3044346_Exemple4"></a>

```wl
// Description de la connexion
MaConnexion est une Connexion
...
// Message à toute application exécutée sur "Machine1"
//  utilisant le login "Admin"
HClient.Machine = "Machine1"
HClient.Login = "Admin"
MaConnexion.EnvoieMessageVersClient("Attention : Déconnexion imminente", 300)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Connexion>.EnvoieMessageVersClient(<Message> , <Durée d'affichage>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'opération a été réalisée, 

- <u><u><u><u>Faux</u></u></u></u> en cas de problème. 


La fonction **&lt;Variable Connexion&gt;.EnvoieMessageVersClient** est bloquante pendant la durée de l'envoi du message vers le serveur. Lorsque le serveur a reçu le message, l'application continue à fonctionner normalement.

**`<Connexion> : (Variable de type Connexion)`**

Nom de la variable de type [Connexion](../WDLang4/1514073.md) décrivant la connexion à manipuler.

**`<Message> : (Chaîne de caractères)`**

Message à afficher sur le poste Client.

**`<Durée d'affichage> : (Entier ou Durée)`**

Durée d'affichage du message exprimée en secondes. Ce paramètre peut correspondre à : 

- un entier correspondant au nombre de secondes, 

- une variable de type Durée, 

- l'indication directe de la durée (par exemple 1 s). 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Personnalisation
<a name="personnalisation_ELTPARAGRAPHE000287"></a>La fonction [HSurAppelServeur](../WDLang4/3044343.md) permet de personnaliser la gestion de l'affichage d'un message sur le poste client.
<a name="NOTE0_2"></a>


### Conditions d'utilisation
<a name="conditions_utilisation_ELTPARAGRAPHE000300"></a>

- Pour utiliser cette fonction, il est nécessaire d'avoir :

	- le droit "Droit d'envoyer des messages aux postes clients" (constante *hDroitEnvoiMessageVersClient* dans la fonction [&lt;Variable Connexion&gt;.ModifieDroitServeur](../WDLang4/1000022710.md)). 

	- le droit "Droit de voir les utilisateurs et les groupes" (constante *hDroitVisualisationUtilisateur* dans la fonction [&lt;Variable Connexion&gt;.ModifieDroitServeur](../WDLang4/1000022710.md)). 




- Le message sera envoyé aux utilisateurs que le client appelant la fonction est autorisé à voir.

- Cette fonction peut être utilisée pour envoyer un message à tous les utilisateurs d'un cluster HFSQL.

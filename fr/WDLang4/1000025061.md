


## &lt;Source&gt;.RetourPosition (Fonction)

***En anglais : &lt;Source&gt;.RestorePosition***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Restaure le contexte précédemment sauvegardé d'un fichier de données (fonction [&lt;Source&gt;.SauvePosition](../WDLang4/1000025062.md)) : enregistrement en cours, filtre, pointeurs de lecture.

La fonction **&lt;Source&gt;.RetourPosition** permet de restaurer :

- les différents pointeurs dans l'index,

- le filtre en cours.




Cette fonction peut être utilisée indifféremment avec les fichiers de données, les vues HFSQL ou les requêtes.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Recherche d'un enregistrement
Client.LitRecherchePremier(Nom, Nom)
TANTQUE Client.Trouve() = Vrai
	// Sauve le contexte en cours
	nPos = Client.SauvePosition(Nom)
	// Vérification d'un client d'un autre nom
	Client.LitRecherche(Nom, Nom2)
	SI Client.Trouve() = Vrai ALORS Compteur = Compteur + 1
	// Restaure le contexte
	HRetourPosition(nPos)
	Client.LitSuivant(Nom)
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Source>.RetourPosition()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la restauration de la position indiquée a été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. La fonction [HErreur](../WDLang4/3044088.md) permet d'obtenir plus d'informations sur le problème rencontré.




**`<Source> : (Type correspondant à la source spécifiée)`**

Nom du fichier de données dont le contexte doit être restauré. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
<a name="NOTE0_2"></a>


### Divers
<a name="divers_ELTPARAGRAPHE000254"></a>

- Le nombre d'appels à la fonction [&lt;Source&gt;.SauvePosition](../WDLang4/1000025062.md) n'est pas limité.

- Il est possible d'imbriquer les fonctions [&lt;Source&gt;.SauvePosition](../WDLang4/1000025062.md) et **&lt;Source&gt;.RetourPosition** pour un même fichier de données.


## HDBCréation (Fonction)

***En anglais : HDBCreation***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Termine la description de la structure du fichier de données xBase par programmation. Le fichier de données qui vient d'être décrit est créé sur le disque dans le chemin indiqué dans la fonction [HDBDécritFichier](../WDLang4/3044172.md). Les fichiers de données, mémos et d'index sont créés sur le disque.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
HDBDécritFichier("CLIENTDB", "CD", "C:\FIC\CLIENT.DBF")
HDBDécritRubrique("NOM,C,20")
HDBDécritRubrique("PRENOM,C,20")
HDBDécritRubrique("AGE,N,3,0")
HDBDécritRubrique("SOLDE,N,10,2")
HDBDécritRubrique("MARIE,L")
HDBDécritRubrique("DATNAISS,D")
HDBDécritRubrique("INFOS,M")
HDBCréation()
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = HDBCréation()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'opération a été réalisée, 

- <u><u><u><u>Faux</u></u></u></u> en cas de problème. La fonction [HErreur](../WDLang4/3044088.md) permet d'identifier l'erreur.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Création d'un fichier xBase via le Connecteur Natif xBase
<a name="creation_fichier_xbase_via_connecteur_natif_xbase_ELTPARAGRAPHE000048"></a>

- Les fichiers de données et d'index sont créés au format dBase 3 (fichier NDX).

- Après la fonction **HDBCréation**, le fichier peut être utilisé sans appeler les fonctions [HDBOuvre](../WDLang4/3044163.md) et [HDBIndex](../WDLang4/3044165.md).





<a name="NOTE0_2"></a>




### Création d'un fichier xBase
<a name="creation_fichier_xbase_ELTPARAGRAPHE000067"></a>

- Si le fichier existe déjà, il est écrasé par le nouveau fichier créé.

- Si le fichier est ouvert lors de l'appel de **HDBCréation**, une erreur 1009 sera générée.





<a name="NOTE0_3"></a>




### Création d'un fichier xBase sous l'éditeur d'analyses
<a name="creation_fichier_xbase_sous_editeur_analyses_ELTPARAGRAPHE000080"></a>Un fichier xBase peut également être décrit et créé depuis l'éditeur. Il est également possible d'importer sous l'éditeur d'analyses, un fichier xBase existant.


<a name="NOTE0_4"></a>


### Limitations
<a name="limitations_ELTPARAGRAPHE000088"></a>

- Cette fonction n'est pas disponible lors de l'utilisation d'une base de données via .

- Cette fonction n'est pas disponible lors de l'utilisation d'un exécutable autonome.

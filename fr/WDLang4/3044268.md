
## OLE DB : Configuration nécessaire 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000085"></a>
Pour manipuler une base de données via OLE DB, les éléments suivants doivent être installés sur le poste de développement et sur le poste de déploiement :

- un driver OLE DB (par exemple OLEDBORA, ORACLE, ...),

- le MDAC (Microsoft DATA Access Component) version 2.7. minimum.






<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Notes
<a name="notes_ELTTEXTE000109"></a>


- Si le MDAC n'est pas installé, seule la connexion à la base OLE DB sera possible. 

- Par défaut, l'installation de WINDEV et WEBDEV propose d'installer le MDAC 2.7.
	Il est également possible de le télécharger sur le site de téléchargement suivant : [http://www.microsoft.com/downloads](http://www.microsoft.com/downloads).




**Remarque** : A la date de rédaction de cette documentation, ce module était librement diffusable. Le module à télécharger et à installer a pour nom : MDAC_TYP.EXE.


<a name="NOTE2_3"></a>


### Cas particulier
<a name="cas_particulier_ELTPARAGRAPHE000035"></a>Le décimal de la couche ADO utilisé par l'accès OLE DB est limité à 28 digits. Si la valeur dépasse cette limite, elle est renvoyée à la base sous forme de réel.

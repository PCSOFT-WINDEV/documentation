
## HListeProvider (Fonction)

***En anglais : HListProvider***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la liste des providers OLE DB et/ou des Connecteurs Natifs installés sur le poste en cours.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Récupérer la liste des Connecteurs Natifs
sListeCN est une chaîne
sCNExtrait est une chaîne
sDescription est une chaîne
nIndice est un entier = 1
sListeCN = HListeProvider()
BOUCLE
	// Extraire chaque Connecteur Natif
	sCNExtrait = ExtraitChaîne(sListeCN, nIndice, RC)
	sDescription = ExtraitChaîne(sCNExtrait, 1, TAB)
	sCNExtrait = ExtraitChaîne(sCNExtrait, 2, TAB)
	SI sCNExtrait = EOT ALORS SORTIR
	// Ajouter le Connecteur Natif dans le champ Combo
	ListeAjoute(COMBO_ChFournisseur, sDescription + gValeurMémorisée(sCNExtrait))
	nIndice++
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = HListeProvider()`
---

**`<Résultat> : (Chaîne de caractères)`**

Liste des providers et/ou des Connecteurs Natifs (également appelées Accès Natifs) trouvés sous la forme: 

```txt
<Description du provider 1> + TAB + <Nom du provider 1> + <RC> +...
<Description du provider n> + TAB + <Nom du provider n> + <RC>
```
où

- &lt;Description du provider&gt; correspond :

	- soit au nom générique du provider (Excel 2000 par exemple).

	- soit au nom du Connecteur Natif.

	- soit à une description extraite de la base de registre.




- &lt;Nom du provider&gt; correspond à la constante identifiant le provider OLE DB ou Connecteur Natif. Cet identifiant peut être utilisé dans les fonctions HFSQL ([HOuvreConnexion](../WDLang4/3044107.md), [HDécritConnexion](../WDLang4/3044205.md), ...).

### Connecteur natif SQL Server
<a name="connecteur_natif_sql_server_ELTPARAGRAPHE000052"></a>La fonction **HListeProvider** prend en compte la valeur de la variable [H.ModeSQLServer](../WDLang4/3044276.md). 

En mode par défaut, si un des deux Connecteurs Natifs SQL Server est installé sur le poste, le Connecteur Natif SQL Server sera listé.

Si l'ancien Connecteur Natif est forcé (H.ModeSQLServer=2) et n'est pas  présent sur le poste, le Connecteur Natif SQL Server ne sera pas listé.

Si le nouveau Connecteur Natif est forcé (H.ModeSQLServer=0) et n'est pas présent sur le poste, le Connecteur Natif SQL Server ne sera pas listé.

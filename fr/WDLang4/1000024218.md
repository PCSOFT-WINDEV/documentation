


## &lt;Source&gt;.DésactiveFiltre (Fonction)

***En anglais : &lt;Source&gt;.DeactivateFilter***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Désactive temporairement un filtre sur un fichier de données (une vue ou une requête). Ce filtre a été créé avec la fonction [&lt;Source&gt;.Filtre](../WDLang4/1000024960.md). Si aucun filtre n'a été déclaré, la fonction **&lt;Source&gt;.DésactiveFiltre** est sans effet.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Déclaration du filtre : sélection des clients
// qui ont entre 20 et 60 ans
Client.Filtre(Age_Client, 20, 60)
...
Client.DésactiveFiltre() // Filtre désactivé
...
// Parcours du fichier de données avec filtre sur l'âge
// Traitement des clients qui ont entre 20 et 60 ans
// (procédure TraiteClient2060)
Client.ActiveFiltre()
Client.LitPremier(Age_Client)
TANTQUE Client.Trouve() = Vrai
	TraiteClient2060()
	Client.LitSuivant(Age_Client)
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Source>.DésactiveFiltre()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le filtre est désactivé, 

- <u><u><u><u>Faux</u></u></u></u> en cas d'erreur (fichier de données ou rubrique inexistant).




**`<Source> : (Type correspondant à la source spécifiée)`**

Nom du fichier de données, de la vue ou de la requête manipulé. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Activation et désactivation d'un filtre
<a name="activation_desactivation_filtre_ELTPARAGRAPHE000240"></a>Après la déclaration d'un filtre (fonction [&lt;Source&gt;.Filtre](../WDLang4/1000024960.md)), le filtre est actif. Après désactivation d'un filtre avec la fonction **&lt;Source&gt;.DésactiveFiltre**, le filtre peut être réactivé grâce à la fonction [&lt;Source&gt;.ActiveFiltre](../WDLang4/1000023969.md).

Remarque : Lors de l'appel de la fonction [&lt;Source&gt;.Ferme](../WDLang4/1000024958.md), le filtre activé par la fonction **HFiltre** est automatiquement supprimé.
<a name="NOTE0_2"></a>


### Nombre de filtres actifs possibles
<a name="nombre_filtres_actifs_possibles_ELTPARAGRAPHE000270"></a>Pour un fichier donné, un seul filtre peut être actif à la fois.
<a name="NOTE0_3"></a>


### Parcours
<a name="parcours_ELTPARAGRAPHE000280"></a>Les modifications apportées par les fonctions [&lt;Source&gt;.Filtre](../WDLang4/1000024960.md) et **&lt;Source&gt;.DésactiveFiltre** sont prises en compte uniquement lors de l'utilisation des fonctions [&lt;Source&gt;.LitPremier](../WDLang4/1000025022.md)/[&lt;Source&gt;.Premier](../WDLang4/1000025048.md), [&lt;Source&gt;.LitDernier](../WDLang4/1000025020.md)/[&lt;Source&gt;.Dernier](../WDLang4/1000024217.md) ou [&lt;Source&gt;.LitRecherchePremier](../WDLang4/1000025034.md)/[&lt;Source&gt;.LitRechercheDernier](../WDLang4/1000025024.md).

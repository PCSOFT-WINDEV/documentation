<a name="NOTE1_1"></a>
Les constantes utilisées par les fonctions HDFS sont les suivantes : 

| Constante | Description | Utilisée par les fonctions ou types de variables |
| --- | --- | --- |
| **hdfsDéfaut** | Options de copie ou de renommage : si le fichier destination existe déjà sur le serveur, il n'est pas écrasé. | [hdfsCopieVers](../WDLang4/1000021679.md), <br>[hdfsRenomme](../WDLang4/1000021676.md), <br>[hdfsSupprime](../WDLang4/1000021674.md) |
| **hdfsEcraser** | <br><br>- Options de copie : si le fichier destination existe déjà sur le serveur, il est remplacé par le fichier copié depuis la machine locale.<br><br>- Options de renommage : si le fichier destination existe déjà, il est remplacé par le fichier renommé.<br><br><br> | <br><br>- [hdfsCopieVers](../WDLang4/1000021679.md)<br><br>- [hdfsRenomme](../WDLang4/1000021676.md)<br><br><br> |
| **hdfsFichier** | L'élément est un fichier. | Type [hdfsInformation](../WDLang4/1000021973.md) |
| **hdfsLien** | L'élément est un lien vers un autre fichier. | Type [hdfsInformation](../WDLang4/1000021973.md) |
| **hdfsRécursif** | L'élément est un lien vers un répertoire. | Type [hdfsInformation](../WDLang4/1000021973.md) |
| **hdfsRépertoire** | Suppression : Les contenus du répertoire et des sous-répertoires sont également supprimés. | [hdfsSupprime](../WDLang4/1000021674.md) |

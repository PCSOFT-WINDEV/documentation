
## HClusterParamètre (Fonction)

***En anglais : HClusterParameter***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Lit et modifie les paramètres d'un cluster HFSQL.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Lecture du nom DNS du cluster
sDNS est une chaîne 
sDNS = HClusterParamètre("clustermanager.mondomaine.com", "secret", hCPDNS)

InfoConstruit("Utilisez <%1> comme adresse de serveur dans vos applications" + ...
    " clientes pour vous connecter au cluster.", sDNS)
```




## Syntaxe

### Connaître la valeur d'un paramètre du cluster

`<Résultat> = HClusterParamètre(<Coordinateur> , <Mot de passe> , <Paramètre>)`
---

**`<Résultat> : (Chaîne de caractères)`**

Valeur du paramètre. 

**`<Coordinateur> : (Chaîne de caractères)`**

Nom DNS de la machine où se trouve le coordinateur du cluster.

**`<Mot de passe> : (Chaîne de caractères)`**

Mot de passe permettant de se connecter au coordinateur. Ce mot de passe a été fourni lors de l'[installation du cluster](../WDLang4/1000019031.md).

**`<Paramètre> : (Constante de type Entier)`**

Paramètre à lire. Les valeurs possibles sont :


|   |   |
| --- | --- |
| *hCPDNS* | Nom DNS du cluster. Ce paramètre est purement informatif.<br><br>**Remarque** : Il est important de ne pas confondre :<br><br>- le nom désignant le cluster (correspondant à la valeur associé à ce paramètre) : nom à utiliser pour accéder aux bases de données du cluster.<br><br>- le nom désignant le coordinateur (utilisé dans le paramètre &lt;Coordinateur&gt;) : nom (DNS) de la machine sur laquelle a été installé le coordinateur du cluster.<br><br>- le nom désignant le noeud (non utilisé par cette fonction) : nom (DNS) de la machine sur laquelle a été installé le nouveau serveur HFSQL.<br><br><br> |
| *hCPPortClient* | Port d'écoute des connexions clientes. |







### Modifier la valeur d'un paramètre du cluster

`<Résultat> = HClusterParamètre(<Coordinateur> , <Mot de passe> , <Paramètre> [, <Nouvelle valeur>])`
---

**`<Résultat> : (Chaîne de caractères)`**

Ancienne valeur du paramètre.

**`<Coordinateur> : (Chaîne de caractères)`**

Nom DNS de la machine où se trouve le coordinateur du cluster.

**`<Mot de passe> : (Chaîne de caractères)`**

Mot de passe permettant de se connecter au coordinateur. Ce mot de passe a été fourni lors de l'[installation du cluster](../WDLang4/1000019031.md).

**`<Paramètre> : (Constante de type Entier)`**

Paramètre à modifier. Les valeurs possibles sont :


|   |   |
| --- | --- |
| *hCPDNS* | Nom DNS du cluster. Ce paramètre est purement informatif.<br><br>**Remarque** : Il est important de ne pas confondre :<br><br>- le nom désignant le cluster (correspondant à la valeur associée à ce paramètre) : nom à utiliser pour accéder aux bases de données du cluster.<br><br>- le nom désignant le coordinateur (utilisé dans le paramètre &lt;Coordinateur&gt;) : nom (DNS) de la machine sur laquelle a été installé le coordinateur du cluster.<br><br>- le nom désignant le noeud (non utilisé par cette fonction) : nom (DNS) de la machine sur laquelle a été installé le nouveau serveur HFSQL.<br><br><br> |
| *hCPPortClient* | Port d'écoute des connexions clientes.<br><br>**Remarque** : La modification de ce paramètre sera prise en compte au prochain redémarrage du cluster. |





**`<Nouvelle valeur> : (Chaîne de caractères optionnelle)`**

Nouvelle valeur du paramètre.

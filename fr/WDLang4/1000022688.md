


## &lt;Variable Connexion&gt;.InfoVue (Fonction)

***En anglais : &lt;Connection variable&gt;.InfoView***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie des informations sur une vue matérialisée.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Description de la connexion
MaConnexion est une Connexion

Info("Les données de la vue ont été produites le " + ...
	MaConnexion.InfoVue(MaVue, hDateDernierRafraîchissement)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Connexion>.InfoVue(<Vue> , <Information demandée>)`
---

**`<Résultat> : (Chaîne de caractères)`**

Information demandée.

**`<Connexion> : (Variable de type Connexion)`**

Nom de la variable de type [Connexion](../WDLang4/1514073.md) décrivant la connexion vers le serveur HFSQL qui contient la vue matérialisée.

**`<Vue> : (Chaîne de caractères)`**

Nom de la vue matérialisée présente sur le serveur et pour laquelle les informations sont demandées.

**`<Information demandée> : (Constante de type Entier)`**

Information demandée sur la vue matérialisée spécifiée. 


|   |   |
| --- | --- |
| *hCodeSQL* | &lt;Résultat&gt; contiendra le code SQL du contenu de la vue. |
| *hDateDernierRafraîchissement* | &lt;Résultat&gt; contiendra la date (au format AAAAMMJJ) du dernier rafraîchissement réussi de la vue. |
| *hErreurDernierRafraîchissement* | &lt;Résultat&gt; correspondra à : <br><br>- l'erreur produite pendant le dernier rafraîchissement. <br><br>- Chaîne vide ("") si le dernier rafraîchissement a réussi.<br><br><br> |
| *hRafraîchissementEnCours* | &lt;Résultat&gt; correspondra à : <br><br>- 1 si la vue est en cours de rafraîchissement, <br><br>- 0 sinon.<br><br><br> |

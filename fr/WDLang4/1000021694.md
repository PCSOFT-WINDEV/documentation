
## hbDescriptionColonne (Type de variable)

***En anglais : hbColumnDescription***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **hbDescriptionColonne** permet de définir toutes les caractéristiques avancées d'une famille de colonnes (rubriques). Les caractéristiques de cette famille de colonnes peuvent être définies et modifiées à l'aide de différentes propriétés WLangage. 

Ce type de variable est utilisé par les variables de type [hbDescriptionTable](../WDLang4/1000021688.md). 

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connexion
Connexion est une hbConnexion
Connexion.Serveur = "MonServeur"
// Description de la table
DescTable est une hbDescriptionTable
DescTable.Nom = "blog"
// Description de la colonne "Post"
DescCol est une hbDescriptionColonne
DescCol.Nom = "Post"
Ajoute(DescTable.Colonne, DescCol)
// Création de la table
hbCréeTable(Connexion, DescTable)
```

## Remarques
<a name="NOTE0_1"></a>


### Propriétés spécifiques à la description des variables de type hbDescriptionColonne
<a name="proprietes_specifiques_description_des_variables_type_hbdescriptioncolonne_ELTPARAGRAPHE000034"></a>Les propriétés suivantes peuvent être utilisées pour manipuler une famille de colonnes :

| Nom de la propriété | Type manipulé | Effet |
| --- | --- | --- |
| Attribut | Tableau de Chaîne de caractères | Tableau associatif contenant les noms des attributs et les valeurs de chaque attribut. Ces noms et valeurs sont spécifiques à HBase et réservés à une utilisation avancée. Ils permettent de définir par exemple le type des données de la colonne.<br><br>Pour plus de détails sur les attributs de colonnes, consultez la documentation HBase. |
| Nom | Chaîne de caractères | Nom de la famille de colonnes. |

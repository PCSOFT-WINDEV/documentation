


## &lt;Variable Connexion&gt;.RSDéclenche (Fonction)

***En anglais : &lt;Connection variable&gt;.RSExecute***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Exécute immédiatement une réplication périodique entre serveurs HFSQL : la réplication se déclenche avant la planification prévue.

<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MaConnexion est une Connexion

ConfigRéplication1 est un hRSConfig
...
MaConnexion.RSDéclenche(ConfigRéplication1..Identifiant)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Connexion>.RSDéclenche(<Réplication à déclencher>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la réplication a été déclenchée,

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. La fonction [HErreurInfo](../WDLang4/3044071.md) permet d'identifier l'erreur.




**`<Connexion> : (Variable de type Connexion)`**

Nom de la variable de type [Connexion](../WDLang4/1514073.md) décrivant la connexion vers le serveur HFSQL maître de la réplication.

**`<Réplication à déclencher> : (Entier)`**

Identifiant la réplication à déclencher. Cet identifiant correspond à la propriété **Identifiant** de la variable de type [hRSConfig](../WDLang4/1000020484.md) décrivant la réplication.

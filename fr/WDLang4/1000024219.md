


## &lt;Source&gt;.DésactiveFiltreAuto (Fonction)

***En anglais : &lt;Source&gt;.DeactivateAutoFilter***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Désactive un filtre automatique sur les fichiers de données reliés lors du parcours d'un fichier XML. Ce filtre a été activé grâce à la fonction [&lt;Source&gt;.ActiveFiltreAuto](../WDLang4/1000024171.md).

**Attention** : Cette fonction est disponible uniquement pour l'Accès Natif XML.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Activation de tous les filtres auto possibles du fichier de données Client
Client.ActiveFiltreAuto("*")
// sauf entre Client et Commande
Client.DésactiveFiltreAuto(Commande)

/////////////////////////////

// Parcours des commandes
Commande.LitPremier(IDCommande)
TANTQUE Commande.EnDehors() = Faux
	// Parcours les lignes de commande de la commande en cours
	LigneCde.LitPremier(IDCommande)
	TANTQUE LigneCde.EnDehors() = Faux
		LigneCde.LitSuivant(IDCommande)
	FIN
	// Commande suivante
	Commande.LitSuivant(IDCommande)
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Source>.DésactiveFiltreAuto(<Fichier de données à ne pas filtrer>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le filtre est désactivé, 

- <u><u><u><u>Faux</u></u></u></u> en cas d'erreur (fichier de données ou rubrique inexistant).




**`<Source> : (Type correspondant à la source spécifiée)`**

Nom du fichier de données utilisé pour le parcours.

**`<Fichier de données à ne pas filtrer> : (Chaîne de caractères)`**



- Nom du fichier de données à ne pas filtrer lors d'un positionnement dans le fichier de données de parcours.

- "\*" pour désactiver tous les filtres automatiques possibles.




## HSauveParamètre (Fonction)

***En anglais : HSaveParameter***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Sauve une valeur persistante. Cette valeur pourra être lue grâce à la fonction [HChargeParamètre](../WDLang4/1000017314.md).  

Remarque : Les paramètres sont sauvegardés par base de données dans le fichier "user_param.fic".


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Sauvegarde d'une valeur à partir d'une procédure stockée
// (Code serveur)
HSauveParamètre("MonParamètre", 5)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Sauvegarde d'un paramètre sur le serveur (dans le code de la procédure stockée)

`<Résultat> = HSauveParamètre(<Paramètre> , <Valeur du paramètre>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la sauvegarde a été réalisée,

- <u><u><u><u>Faux</u></u></u></u> en cas d'erreur. La fonction [HErreurInfo](../WDLang4/3044071.md) permet d'identifier l'erreur.




**`<Paramètre> : (Chaîne de caractères)`**

Nom du paramètre à sauvegarder. Ce paramètre ne doit pas correspondre à une chaîne vide et ne doit pas contenir le caractère "RC" (Retour Chariot).

**`<Valeur du paramètre> : (Type correspondant à la valeur à sauver)`**

Valeur à sauver.  


<a name="SYNTAXE2"></a>

### Sauvegarde d'un paramètre sur le serveur (depuis le poste Client)

`<Résultat> = HSauveParamètre(<Connexion> , <Paramètre> , <Valeur du paramètre>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la sauvegarde a été réalisée, 

- <u><u><u><u>Faux</u></u></u></u> en cas d'erreur. La fonction [HErreurInfo](../WDLang4/3044071.md) permet d'identifier l'erreur.




**`<Connexion> : (Chaîne de caractères ou variable de type Connexion)`**

Connexion à utiliser. Cette connexion correspond : 

- soit au nom de la connexion définie sous l'éditeur d'analyses ou grâce aux fonctions [HDécritConnexion](../WDLang4/3044205.md) ou [HOuvreConnexion](../WDLang4/3044107.md).

- soit au nom d'une variable de type [Connexion](../WDLang4/1514073.md).




**`<Paramètre> : (Chaîne de caractères)`**

Nom du paramètre à sauvegarder. Ce paramètre ne doit pas correspondre à une chaîne vide et ne doit pas contenir le caractère "RC" (Retour Chariot).

**`<Valeur du paramètre> : (Type correspondant à la valeur à sauver)`**

Valeur à sauver.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Type de variables sauvegardées
<a name="type_variables_sauvegardees_ELTPARAGRAPHE000142"></a>La fonction **HSauveParamètre** permet de sauvegarder des variables de type simple (chaîne, entier, ...).

Pour sauvegarder des variables complexes (objets, structures, ...), il est nécessaire de sérialiser ces variables sous forme de chaîne grâce à la fonction [Sérialise](../WDLang1/3013065.md).

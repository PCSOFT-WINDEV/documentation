


## HSupprimeRépertoire (Fonction)

***En anglais : HDeleteDirectory***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime un répertoire présent dans une base de données HFSQL Client/Serveur. Ce répertoire doit être vide. 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MaConnexion est une Connexion

HSupprimeFichier(MaConnexion, "ModuleCRM\Prospect.fic")
HSupprimeRépertoire(MaConnexion, "ModuleCRM")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = HSupprimeRépertoire(<Connexion> , <Répertoire à supprimer>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le répertoire a été supprimé, 

- <u><u><u><u>Faux</u></u></u></u> en cas d'erreur (par exemple si le répertoire n'existe pas sur le serveur ou si le répertoire n'est pas vide). La fonction [HErreur](../WDLang4/3044088.md) permet d'identifier l'erreur.




**`<Connexion> : (Chaîne de caractères ou variable de type Connexion)`**

Connexion dans laquelle le répertoire doit être supprimé. Cette connexion correspond :

- soit au nom de la connexion définie sous l'éditeur d'analyses ou grâce aux fonctions [HDécritConnexion](../WDLang4/3044205.md) ou [HOuvreConnexion](../WDLang4/3044107.md).

- soit au nom d'une variable de type [Connexion](../WDLang4/1514073.md).




**`<Répertoire à supprimer> : (Chaîne de caractères)`**

Chemin du répertoire à supprimer. Ce chemin est relatif à la base de données : le nom de la base de données ne doit pas être précisé dans le chemin. Ce répertoire doit être vide.

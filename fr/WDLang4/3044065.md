
## HGèreTrigger (Fonction)

***En anglais : HSetTrigger***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Active ou désactive la gestion des triggers. Pour plus de détails, consultez [Gestion des triggers](../WDLang4/3044271.md). 

**Remarque** : Par défaut, la gestion des triggers est active. **Pour accélérer votre application**, la désactivation des triggers est conseillée si ces derniers ne sont pas utilisés.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Désactive la gestion des triggers
HGèreTrigger(Faux)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = HGèreTrigger(<Gestion>)`
---

**`<Résultat> : (Booléen)`**

Ancien mode de gestion des triggers (avant l'utilisation de la fonction **HGèreTrigger**) : 

- <u><u><u><u>Vrai</u></u></u></u> si la gestion des triggers était active, 

- <u><u><u><u>Faux</u></u></u></u> si la gestion des triggers n'était pas active.




**`<Gestion> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la gestion des triggers est activée, 

- <u><u><u><u>Faux</u></u></u></u> si cette gestion doit être désactivée.

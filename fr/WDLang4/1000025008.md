


## &lt;Source&gt;.JournalStop (Fonction)

***En anglais : &lt;Source&gt;.LogStop***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Arrête la journalisation du fichier de données. Les manipulations effectuées dans le fichier journalé ne sont plus enregistrées.

**Remarque** : La journalisation n'est pas disponible lors de l'accès à une base de données via .


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
Client.JournalInfo("début impression de " + Critère)
Client.JournalStop() // Stopper le journal
Client.LitPremier(Critère)
TANTQUE Client.Trouve() = Vrai
	// Traitement
	 Client.LitSuivant()
FIN

// Redémarrer le journal
Client.JournalRedémarre() 
// Ecrire dans le journal
Client.JournalInfo("fin impression de " + Critère)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Source>.JournalStop()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'opération a été réalisée, 

- <u><u><u><u>Faux</u></u></u></u> en cas de problème. La fonction [HErreur](../WDLang4/3044088.md) permet d'identifier l'erreur.




**`<Source> : (Type correspondant à la source spécifiée)`**

Nom du fichier de données journalé dont la journalisation doit être arrêtée.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Les opérations effectuées après l'utilisation de cette fonction ne seront pas journalées et seront perdues en cas d'incident.

- Ne pas oublier de relancer la journalisation par la fonction [&lt;Source&gt;.JournalRedémarre](../WDLang4/1000025007.md) en s'assurant que le fichier de données est rigoureusement dans son état initial.

- Le journal est arrêté jusqu'à l'exécution de la fonction [&lt;Source&gt;.JournalRedémarre](../WDLang4/1000025007.md).




## Fonctions Memcached
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions permettant de manipuler des bases Memcached sont les suivantes : 



|   |   |
| --- | --- |
| [MemcachedAffecte](../WDLang4/1000023342.md) | Modifie un couple clé/valeur dans le cache Memcached :<br><br>- Si la clé existe, la valeur est modifiée.<br><br>- Si la clé n'existe pas, le couple clé/valeur est ajouté.<br><br><br> |
| [MemcachedAffecteGroupeClé](../WDLang4/1000023354.md) | Modifie un couple clé/valeur dans le cache Memcached sur un groupe de serveurs :<br><br>- Si la clé existe, la valeur est modifiée.<br><br>- Si la clé n'existe pas, le couple clé/valeur est ajouté.<br><br><br> |
| [MemcachedAjoute](../WDLang4/1000023340.md) | Ajoute un nouveau couple clé/valeur dans le cache Memcached. |
| [MemcachedAjouteGroupeClé](../WDLang4/1000023349.md) | Ajoute un nouveau couple clé/valeur dans le cache Memcached sur un groupe de serveurs. |
| [MemcachedAjoutePréfixe](../WDLang4/1000023343.md) | Ajoute une valeur au début de la valeur existante dans le cache pour la clé donnée. |
| [MemcachedAjoutePréfixeGroupeClé](../WDLang4/1000023357.md) | Ajoute une valeur au début de la valeur existante dans le cache pour la clé donnée sur un groupe de serveurs. |
| [MemcachedAjouteSuffixe](../WDLang4/1000023344.md) | Ajouter une valeur à la fin de la valeur existante dans le cache pour la clé donnée. |
| [MemcachedAjouteSuffixeGroupeClé](../WDLang4/1000023350.md) | Ajoute une valeur à la fin de la valeur existante dans le cache pour la clé donnée sur un groupe de serveurs. |
| [MemcachedCAS](../WDLang4/1000023348.md) | Modifie la valeur d'une clé si celle-ci n'a pas été modifiée. |
| [MemcachedCASGroupeClé](../WDLang4/1000023356.md) | Modifie la valeur d'une clé sur un groupe de serveurs si celle-ci n'a pas été modifiée. |
| [MemcachedDécrémente](../WDLang4/1000023347.md) | Décrémente la valeur associée à une clé. Si la clé n'existe pas, elle est créée. |
| [MemcachedDécrémenteGroupeClé](../WDLang4/1000023355.md) | Décrémente la valeur associée à une clé sur un groupe de serveurs. |
| [MemcachedIncrémente](../WDLang4/1000023346.md) | Incrémente la valeur associée à une clé. |
| [MemcachedIncrémenteGroupeClé](../WDLang4/1000023352.md) | Incrémente la valeur associée à une clé sur un groupe de serveurs. |
| [MemcachedModifie](../WDLang4/1000023341.md) | Modifie un couple clé/valeur dans le cache Memcached. |
| [MemcachedModifieGroupeClé](../WDLang4/1000023351.md) | Modifie un couple clé/valeur dans le cache Memcached sur un groupe de serveurs. |
| [MemcachedRécupère](../WDLang4/1000023363.md) | Récupère les valeurs des clés stockées sur un serveur Memcached. |
| [MemcachedRécupèreGroupeClé](../WDLang4/1000023364.md) | Récupère les valeurs des clés stockées sur un groupe de serveurs Memcached. |
| [MemcachedSupprime](../WDLang4/1000023345.md) | Supprime une clé dans le cache Memcached. |
| [MemcachedSupprimeGroupeClé](../WDLang4/1000023353.md) | Supprime une clé dans le cache Memcached sur un groupe de serveurs. |

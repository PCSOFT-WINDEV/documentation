


## &lt;Variable mongoBase&gt;.ExécuteCommande (Fonction)

***En anglais : &lt;mongoDatabase variable&gt;.ExecuteCommand***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Exécute une commande générique sur une base MongoDB.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
BaseMDB est un MongoBase
// Exécuter une commande sur une base MongoDB
BaseMDB.ExécuteCommande("{ ""ping"" : 1 }" )
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Exécuter une commande sur une base MongoDB

`<Résultat> = <Base>.ExécuteCommande(<Commande>)`
---

**`<Résultat> : (Variable de type mongoRésultat)`**

Variable de type [mongoRésultat](../WDLang4/1000022415.md) correspondant au résultat de la commande. En cas de problème. La fonction [ErreurInfo](../WDLang1/3013008.md) permet d'identifier l'erreur.

**`<Base> : (Variable de type mongoBase)`**

Nom de la variable de type [mongoBase](../WDLang4/1000022410.md) correspondant à la base MongoDB dans laquelle la commande s'exécutera.

**`<Commande> : (Chaîne de caractères, Buffer ou variant)`**

Document décrivant la commande à exécuter. Ce document peut correspondre : 

- à un document au format JSON. Dans ce cas, ce paramètre correspond à une chaîne de caractères. 

- à un document au format BSON. Dans ce cas, ce paramètre correspond à un buffer. 

- à un variant.




## HGèreMémo (Fonction)

***En anglais : HSetMemo***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de modifier la gestion des rubriques de type mémo. Il est possible de gérer tous les mémos d'un fichier de données ou uniquement une rubrique mémo spécifique d'un fichier de données. Par défaut, la gestion des mémos est activée.
![Hyper File 5.5](https://doc.pcsoft.fr/ext/images/fr/HF55.png) Seule la gestion des mémos au niveau du fichier de données est possible : il est impossible de gérer les mémos au niveau des rubriques.![OLE DB](https://doc.pcsoft.fr/ext/images/fr/OLEDB.png) L'appel de la fonction **HGèreMémo** provoque la fermeture de tous les parcours commencés. Les critères de filtrage et de recherche sont conservés.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Désactive la gestion des mémos sur les observations du fichier de données Client
HGèreMémo(Client, Rub_Observations, hMémoNon)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = HGèreMémo([<Fichier de données> [, <Rubrique mémo>, ]] <Mode de gestion>)`
---

**`<Résultat> : (Entier)`**

Ancien mode de gestion des mémos utilisé (avant utilisation de la fonction **HGèreMémo**).


|   |   |
| --- | --- |
| *hMémoAuto*<br>(valeur par défaut) | Lecture des mémos lors de l'accès à la rubrique. |
| *hMémoNon* | Pas de gestion des mémos (aucune lecture et aucune écriture ne sera effectuée dans le fichier de données ou dans la rubrique spécifié). |
| *hMémoOui* | Lecture des mémos texte à chaque lecture de l'enregistrement.<br>Lecture des mémos binaires uniquement lors de l'accès à la rubrique. |



**`<Fichier de données> : (Chaîne de caractères optionnelle)`**

Nom du fichier de données manipulé. 
Si ce nom n'est pas spécifié ou est égal à "\*", tous les fichiers mémo sont pris en compte.

**`<Rubrique mémo> : (Chaîne de caractères optionnelle)`**

Nom de la rubrique mémo à gérer ou non. 
Si ce nom n'est pas spécifié ou est égal à "\*", toutes les rubriques mémo sont prises en compte.
![Hyper File 5.5](https://doc.pcsoft.fr/ext/images/fr/HF55.png) Seule la gestion des mémos au niveau du fichier de données est possible : il est impossible de gérer les mémos au niveau des rubriques.

**`<Mode de gestion> : (Constante de type Entier)`**

Mode de gestion des rubriques de type mémo voulu :


|   |   |
| --- | --- |
| *hMémoAuto*<br>(valeur par défaut) | Lecture des mémos lors de l'accès à la rubrique. |
| *hMémoNon* | Pas de gestion des mémos (aucune lecture et aucune écriture ne sera effectuée dans le fichier de données ou dans la rubrique spécifié). |
| *hMémoOui* | Lecture des mémos texte à chaque lecture de l'enregistrement.<br>Lecture des mémos binaires uniquement lors de l'accès à la rubrique. |


Si seul ce paramètre est précisé, le mode de gestion spécifié s'applique à tous les fichiers de données et à toutes les rubriques.
![OLE DB](https://doc.pcsoft.fr/ext/images/fr/OLEDB.png) La constante *hMémoAuto* est équivalente à la constante *hMémoOui*.

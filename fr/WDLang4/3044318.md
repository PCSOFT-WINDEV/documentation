
## HDémarreServeur (Fonction)

***En anglais : HStartServer***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de démarrer un serveur HFSQL (cette fonction utilise le service MantaManager).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Redémarrage du serveur 
HDémarreServeur("PosteServeurCS")

// Redémarrage du serveur en utilisant un numéro de port
HDémarreServeur("PosteServeurCS:4901")

// Redémarrage du serveur en utilisant un nom de serveur
HDémarreServeur("PosteServeurCS", "Serveur HFSQL")
```


<a name="Exemple2"></a>



```wl
// Redémarrage du serveur en utilisant une variable connexion

Cnx_HFSQL est une Connexion
Cnx_HFSQL.Provider = hAccèsHFClientServeur
Cnx_HFSQL.Utilisateur = "admin"
Cnx_HFSQL.MotDePasse = ""
Cnx_HFSQL.Serveur = "GB:4916"
Cnx_HFSQL.BaseDeDonnées = "CRM"

HDémarreServeur(Cnx_HFSQL.Serveur)

SELON HEtatServeur(Cnx_HFSQL.Serveur)
	CAS hArrêté : Info("Serveur arrêté")
	CAS hDémarré : Info("Serveur démarré")
	CAS hInaccessible : Info("Serveur inaccessible")
FIN
```







<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Redémarrage d'un serveur (avec possibilité de préciser le port du serveur HFSQL)

`<Résultat> = HDémarreServeur(<Poste serveur>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le serveur a été redémarré, 

- <u><u><u><u>Faux</u></u></u></u> en cas d'erreur. La fonction [HErreur](../WDLang4/3044088.md) permet d'identifier l'erreur.




**`<Poste serveur> : (Chaîne de caractères)`**

Nom de la machine serveur. Il est possible de préciser le numéro de port utilisé par le serveur HFSQL. Par défaut, le numéro de port utilisé est 4900. Si le numéro de port du serveur HFSQL est différent, utilisez la notation suivante :

&lt;Nom du poste&gt; : &lt;Numéro de port&gt;

Exemple : "NomPoste:4901"

Le numéro de port correspond au numéro de port utilisé dans la fonction [HOuvreConnexion](../WDLang4/3044107.md).  


<a name="SYNTAXE2"></a>

### Redémarrage d'un serveur en précisant le nom du serveur HFSQL

`<Résultat> = HDémarreServeur(<Poste serveur> , <Serveur HFSQL>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le serveur a été redémarré, 

- <u><u><u><u>Faux</u></u></u></u> en cas d'erreur. La fonction [HErreur](../WDLang4/3044088.md) permet d'identifier l'erreur.




**`<Poste serveur> : (Chaîne de caractères)`**

Nom de la machine serveur.

**`<Serveur HFSQL> : (Chaîne de caractères)`**

Nom du serveur HFSQL à redémarrer (ce nom apparaît dans le Centre de Contrôle HFSQL).



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Redémarrage d'un serveur
<a name="redemarrage_serveur_ELTPARAGRAPHE000080"></a>

- Si le serveur est déjà démarré lors de l'utilisation de la fonction **HDémarreServeur**, le serveur n'est pas redémarré. La fonction [HEtatServeur](../WDLang4/3044319.md) permet de connaître l'état d'un serveur.

- Aucun droit spécifique n'est nécessaire pour démarrer un serveur.




**Remarque** : Suite à un arrêt du serveur, il est possible que le serveur ne puisse pas être redémarré tout de suite. Le système d'exploitation ne ferme pas toujours immédiatement les connexions qui existaient entre les clients connectés et le serveur. Il suffit d'attendre quelques minutes (selon les timeouts du système d'exploitation) et le serveur peut être relancé.

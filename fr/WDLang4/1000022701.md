


## &lt;Variable Connexion&gt;.ListeUtilisateur (Fonction)

***En anglais : &lt;Connection variable&gt;.ListUser***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la liste des utilisateurs définis pour une connexion.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Description de la connexion
MaConnexion est une Connexion

// Ouverture de la connexion
MaConnexion.OuvreConnexion()
Résultat1 est une chaîne
Résultat1 = MaConnexion.ListeUtilisateur()
// Ajout des utilisateurs dans un champ Liste
LISTE_Maliste.Ajoute(Résultat1)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Connexion>.ListeUtilisateur()`
---

**`<Résultat> : (Chaîne de caractères)`**

Nom des différents utilisateurs existants pour la connexion spécifiée. Les noms des utilisateurs sont séparés par RC (Retour-Chariot).

**Remarque** : Si l'utilisateur en cours ne possède pas le droit de visualisation des utilisateurs (constante *hDroitVisualisationUtilisateur*) sur le serveur (fonction [&lt;Variable Connexion&gt;.ModifieDroitServeur](../WDLang4/1000022710.md) ou [&lt;Variable Connexion&gt;.InfoDroitServeur](../WDLang4/1000022668.md)), la fonction **&lt;Variable Connexion&gt;.ListeUtilisateur** renvoie uniquement l'utilisateur en cours.

**`<Connexion> : (Variable de type Connexion)`**

Nom de la variable de type [Connexion](../WDLang4/1514073.md) décrivant la connexion à manipuler.

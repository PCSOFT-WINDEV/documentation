


## &lt;Source&gt;.FTListeMot (Fonction)

***En anglais : &lt;Source&gt;.FTListWord***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Liste les mots de l'enregistrement courant qui seront pris en compte par l'indexation full-text.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
tabToken est un tableau de chaîne = Document.FTListeMot(FTIndexTitreContenu)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Source>.FTListeMot(<Index full-text>)`
---

**`<Résultat> : (Tableau de chaînes)`**

Variable de type Tableau contenant les mots pris en compte par l'indexation full-text.

**`<Source> : (Type correspondant à la source spécifiée)`**

Nom du fichier de données HFSQL manipulé.

**`<Index full-text> : (Chaîne de caractères)`**

Nom de l'index full-text du fichier.

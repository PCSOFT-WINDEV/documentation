


## Fonctions de gestion d'une base Big Data Redis (syntaxe préfixée)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions de gestion d'une base de données Redis via une syntaxe préfixée sont les suivantes : 



|   |   |
| --- | --- |
| [&lt;redisConnexion&gt;.Abonne](../WDLang4/1410088279.md) | Souscrit un abonnement à un canal de diffusion Redis. |
| [&lt;redisConnexion&gt;.Affecte](../WDLang4/1410088245.md) | Modifie la valeur d'une clé sur un serveur Redis. |
| [&lt;redisConnexion&gt;.Connecte](../WDLang4/1410088280.md) | Ouvre une connexion vers un serveur Redis (ou un cluster Redis). |
| [&lt;redisConnexion&gt;.Désabonne](../WDLang4/1410088281.md) | Supprime un abonnement à un canal de diffusion Redis. |
| [&lt;redisConnexion&gt;.ExécuteCommande](../WDLang4/1410088282.md) | Exécute une commande sur un serveur Redis et récupère le résultat. |
| [&lt;redisConnexion&gt;.PipelineAjoute](../WDLang4/1410088283.md) | Ajoute une commande Redis dans un pipeline. |
| [&lt;redisConnexion&gt;.PipelineRAZ](../WDLang4/1410088284.md) | Ignore les résultats de l'exécution d'une commande Redis ayant été ajoutée par la fonction [&lt;redisConnexion&gt;.PipelineAjoute](../WDLang4/1410088283.md). |
| [&lt;redisConnexion&gt;.PipelineRécupère](../WDLang4/1410088285.md) | Récupère le résultat de l'exécution d'une commande Redis ayant été ajouté par [&lt;redisConnexion&gt;.PipelineAjoute](../WDLang4/1410088283.md). |
| [&lt;redisConnexion&gt;.Publie](../WDLang4/1410088286.md) | Publie un message sur un canal de diffusion Redis. |
| [&lt;redisConnexion&gt;.Récupère](../WDLang4/1410088278.md) | Récupère la valeur d'une clé sur un serveur Redis sous forme de chaîne. |

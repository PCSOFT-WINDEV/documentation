


## &lt;Variable hdfsConnexion&gt;.CréeRépertoire (Fonction)

***En anglais : &lt;hdfsConnection variable&gt;.CreateDirectory***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Crée un répertoire sur un serveur Hadoop.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connexion
MaConnexion est une hdfsConnexion
MaConnexion.namenode = "MonServeur"
MaConnexion.Utilisateur = "Admin"
MaConnexion.OuvreConnexion()
// Définition des droits
Droits est un hdfsDroit
Droits.UtilisateurLecture = Vrai
Droits.UtilisateurEcriture = vrai
// Création du répertoire
MaConnexion.CréeRépertoire("/tmp/sous_repertoire", Droits)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Connexion>.CréeRépertoire(<Chemin> [, <Droits>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la création du répertoire a été réalisée,

- <u><u><u><u>Faux</u></u></u></u> en cas de problème. La fonction [ErreurInfo](../WDLang1/3013008.md) permet d'identifier l'erreur.




**`<Connexion> : (Variable de type hdfsConnexion)`**

Nom de la variable de type [hdfsConnexion](../WDLang4/1000021650.md) correspondant à la connexion au serveur Hadoop.

**`<Chemin> : (Chaîne de caractères)`**

Chemin absolu du répertoire à créer sur le serveur Hadoop. Ce chemin est sensible à la casse.

**`<Droits> : (Variable de type hdfsDroit optionnelle)`**

Nom de la variable de type [hdfsDroit](../WDLang4/1000021662.md) correspondant aux droits à appliquer au répertoire créé.
Par défaut, tous les droits sont donnés à tous les utilisateurs.

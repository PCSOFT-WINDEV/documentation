


## Fonctions de gestion de HFSQL Cluster
			





## Fonctions d'administration d'un cluster HFSQL
<a name="fonctions_administration_cluster_hfsql_ELTTEXTE000066"></a>
Les fonctions ci-dessous permettent d'administrer un cluster HFSQL :



|   |   |
| --- | --- |
| [HClusterAjouteNoeud](../WDLang4/1000019061.md) | Active un noeud dans un cluster HFSQL. |
| [HClusterArrête](../WDLang4/1000019058.md) | Suspend le fonctionnement d'un cluster HFSQL. Les noeuds du cluster n'accepteront plus de connexions de la part des applications clientes. |
| [HClusterDémarre](../WDLang4/1000019057.md) | Démarre un cluster HFSQL. Ce cluster a été arrêté précédemment (par exemple avec la fonction [HClusterArrête](../WDLang4/1000019058.md)). |
| [HClusterEtat](../WDLang4/1000019059.md) | Renvoie l'état d'un cluster HFSQL en interrogeant son coordinateur. |
| [HClusterIgnoreSynchro](../WDLang4/1000019243.md) | Désigne un noeud du cluster HFSQL comme source de données pour réaliser la synchronisation du cluster. |
| [HClusterInfoErreurSynchronisation](../WDLang4/1000021975.md) | Renvoie l'erreur survenue lors de la précédente phase de synchronisation du cluster HFSQL. |
| [HClusterInfoNoeud](../WDLang4/1000019060.md) | Renvoie l'état de chaque noeud du cluster en interrogeant le coordinateur. |
| [HClusterInfoSynchronisation](../WDLang4/1000021974.md) | Renvoie l'état d'avancement de la synchronisation d'un noeud d'un cluster HFSQL. |
| [HClusterNoeudMaître](../WDLang4/1410087885.md) | Renvoie le nom du noeud maître dans un cluster HFSQL. |
| [HClusterParamètre](../WDLang4/1000019305.md) | Lit et modifie les paramètres d'un cluster HFSQL. |
| [HClusterSupprimeNoeud](../WDLang4/1000019062.md) | Désactive un noeud dans un cluster HFSQL. |





Il est également possible d'utiliser les fonctions suivantes : 



|   |   |
| --- | --- |
| [HChargeParamètre](../WDLang4/1000017314.md) | Lit un paramètre précédemment sauvegardé à partir d'une procédure stockée grâce à la fonction [HSauveParamètre](../WDLang4/1000017313.md). |
| [HSauveParamètre](../WDLang4/1000017313.md) | Sauve une valeur persistante sur le serveur HFSQL. |
| [HSupprimeParamètre](../WDLang4/1000017315.md) | Supprime un paramètre précédemment sauvé avec la fonction [HSauveParamètre](../WDLang4/1000017313.md). |

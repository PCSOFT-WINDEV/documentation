


## &lt;Variable Connexion&gt;.ListeElémentStocké (Fonction)

***En anglais : &lt;Connection variable&gt;.ListStoredElement***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la liste des éléments stockés sur un serveur HFSQL (collections de procédures, procédures stockées ou requêtes).




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Description de la connexion
MaConnexion est une Connexion

// Récupérer les éléments présents sur le serveur
ListeElément est une chaîne
ListeElément = MaConnexion.ListeElémentStocké(hLstDetail + hLstTout)

// Ajoute la liste des éléments dans COMBO_COMBO1
COMBO_COMBO1.Ajoute(ListeElément)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Connexion>.ListeElémentStocké([<Options>])`
---

**`<Résultat> : (Chaîne de caractères)`**

Liste des éléments demandés. En fonction de la constante utilisée, le résultat est plus ou moins détaillé.


|   |   |
| --- | --- |
| *hLstDétail* non précisé | &lt;Nom Elément 1&gt; + TAB + &lt;Type&gt; + RC + &lt;Nom Elément 2&gt; + RC + ... |
| *hLstDétail* précisé | &lt;Nom Elément 1&gt; + TAB + &lt;Type&gt; + TAB + &lt;Nom Auteur&gt; + TAB + &lt;Date Heure MAJ&gt; + TAB + &lt;Nom auteur MAJ&gt; + TAB + &lt;Nom Machine MAJ&gt; + TAB + &lt;Taille&gt; + RC + ... |

Où :

- &lt;Nom Elément X&gt; : Nom logique de la collection de procédures, de la procédure ou de la requête. Le nom de la procédure est de la forme &lt;Nom de la collection&gt;.&lt;Nom de la procédure&gt;. 

- &lt;Type&gt; : Type de l'élément : Valeur permettant d'identifier l'élément listé :
	


|   |   |
| --- | --- |
| *eltCollectionProcédure* | L'élément listé  est une collection de procédures. |
| *eltProcédure* | L'élément listé est une procédure. |
| *eltRequête* | L'élément listé est une requête. |



- &lt;Nom Auteur&gt; : Nom de la personne ayant créé l'élément.

- &lt;Date Heure MAJ&gt; : Date et heure de la dernière mise à jour de l'élément.

- &lt;Nom Auteur MAJ&gt; : Nom de la personne ayant effectué la dernière mise à jour de l'élément.

- &lt;Nom Machine MAJ&gt; : Nom de la machine ayant effectué la dernière mise à jour de l'élément.

- &lt;Taille&gt; : Taille de la collection ou de la requête (0 pour une procédure).




**`<Connexion> : (Variable de type Connexion)`**

Nom de la variable de type [Connexion](../WDLang4/1514073.md) décrivant la connexion vers le serveur HFSQL.

**`<Options> : (Constante ou combinaison de constantes de type entier (optionnel))`**

Type de liste à effectuer :


|   |   |
| --- | --- |
| *hLstCollection* | Liste les collections de procédures. |
| *hLstDétail* | Liste les informations détaillées. |
| *hLstProcédureStockée* | Liste les procédures stockées. |
| *hLstRequête* | Liste les requêtes. |
| *hLstTout*<br>(Valeur par défaut) | Liste tous les éléments dans l'ordre suivant : collections de procédures, procédures stockées et requêtes. |


## HDécritLiaison (Fonction)

***En anglais : HDescribeLink***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Décrit par programmation une liaison entre deux fichiers de données.

Après l'appel à la fonction **HDécritLiaison** :

- La description de la liaison est validée.

- La description de la liaison ne peut plus être modifiée.

- La variable de description de la liaison est réinitialisée. Une nouvelle description de liaison peut être effectuée.




**Remarque** : Pour ajouter une liaison dans une base de données HFSQL Client/Serveur, utilisez la fonction [HAjouteLiaison](../WDLang4/3044312.md). 




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
HDécritLiaison(LiaisonClientProduit)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = HDécritLiaison(<Liaison>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'opération a été réalisée, 

- <u><u><u><u>Faux</u></u></u></u> en cas de problème. La fonction [HErreurInfo](../WDLang4/3044071.md) permet d'identifier l'erreur.




**`<Liaison> : (Variable de type Description de Liaison)`**

Nom de la variable de type [Description de Liaison](../WDLang4/1514039.md) correspondant à la liaison à créer. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Comment décrire une liaison par programmation ? 
<a name="comment_decrire_une_liaison_par_programmation_ELTPARAGRAPHE000079"></a>Pour décrire une liaison par programmation, il faut :

1. Utiliser des variables de type [Description de Liaison](../WDLang4/1514039.md).

2. Décrire les caractéristiques de la liaison grâce aux propriétés HFSQL.

3. Utiliser la fonction **HDécritLiaison** pour enregistrer la description de la liaison.




**Remarque** : Pour décrire un fichier de données par programmation, utilisez la fonction [HDécritFichier](../WDLang4/3044206.md). Pour décrire des rubriques par programmation, utilisez la fonction [HDécritRubrique](../WDLang4/3044233.md).

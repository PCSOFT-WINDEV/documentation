


## HSupprimeParamètre (Fonction)

***En anglais : HDeleteParameter***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime un paramètre précédemment sauvé avec la fonction [HSauveParamètre](../WDLang4/1000017313.md). 
 

<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
HSupprimeParamètre("MonParamètre")
```



```wl
MaConnexion est une Connexion

HSupprimeParamètre(MaConnexion, "MonParamètre")
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Supprime un paramètre sur le serveur (dans le code de la procédure stockée)

`<Résultat> = HSupprimeParamètre(<Paramètre>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la suppression a été réalisée, 

- <u><u><u><u>Faux</u></u></u></u> en cas d'erreur. La fonction [HErreurInfo](../WDLang4/3044071.md) permet d'identifier l'erreur.




**`<Paramètre> : (Chaîne de caractères)`**

Nom du paramètre à supprimer. Ce paramètre ne doit pas correspondre à une chaîne vide et ne doit pas contenir le caractère "RC" (Retour Chariot).  


<a name="SYNTAXE2"></a>

### Supprime un paramètre sur le serveur (depuis le poste Client)

`<Résultat> = HSupprimeParamètre(<Connexion> , <Paramètre>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la suppression a été réalisée, 

- <u><u><u><u>Faux</u></u></u></u> en cas d'erreur. La fonction [HErreurInfo](../WDLang4/3044071.md) permet d'identifier l'erreur.




**`<Connexion> : (Chaîne de caractères ou variable de type Connexion)`**

Connexion à utiliser. Cette connexion correspond :

- soit au nom de la connexion définie sous l'éditeur d'analyses ou grâce aux fonctions [HDécritConnexion](../WDLang4/3044205.md) ou [HOuvreConnexion](../WDLang4/3044107.md).

- soit au nom d'une variable de type [Connexion](../WDLang4/1514073.md).




**`<Paramètre> : (Chaîne de caractères)`**

Nom du paramètre à supprimer. Ce paramètre ne doit pas correspondre à une chaîne vide et ne doit pas contenir le caractère "RC" (Retour Chariot).

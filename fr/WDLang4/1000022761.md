


## &lt;Variable Connexion&gt;.SupprimeParamètre (Fonction)

***En anglais : &lt;Connection variable&gt;.DeleteParameter***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime un paramètre précédemment sauvé avec la fonction [&lt;Variable Connexion&gt;.SauveParamètre](../WDLang4/1000022753.md). 
 

<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MaConnexion est une Connexion

MaConnexion.SupprimeParamètre("MonParamètre")
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

### Supprime un paramètre sur le serveur (depuis le poste Client)

`<Résultat> = <Connexion>.SupprimeParamètre(<Paramètre>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u>Vrai</u></u></u> si la suppression a été réalisée, 

- <u><u><u>Faux</u></u></u> en cas d'erreur. La fonction [HErreurInfo](../WDLang4/3044071.md) permet d'identifier l'erreur.




**`<Connexion> : (Variable de type Connexion)`**

Nom de la variable de type [Connexion](../WDLang4/1514073.md) décrivant la connexion à utiliser.

**`<Paramètre> : (Chaîne de caractères)`**

Nom du paramètre à supprimer. Ce paramètre ne doit pas correspondre à une chaîne vide et ne doit pas contenir le caractère "RC" (Retour Chariot).

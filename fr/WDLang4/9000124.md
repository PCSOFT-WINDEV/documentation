
## Informations optionnelles de connexion
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000757"></a>
Lors de la définition des connexions sous l'éditeur d'analyses ou lors de la définition des connexions par programmation, il est possible de spécifier des informations optionnelles de connexion. 

Sous l'éditeur d'analyses, ces informations sont saisies dans la description de la connexion, onglet "Avancé". 

Par programmation, ces informations étendues sont configurées : 

- via la propriété [InfosEtendues](../Proprietes/2512041.md) utilisée avec les variables de  type [Connexion](../WDLang4/1514073.md). 

- via la fonction [HOuvreConnexion](../WDLang4/3044107.md) ou la fonction [HDécritConnexion](../WDLang4/3044205.md). 




Ces informations sont définies grâce aux mots-clés suivants. 

**Attention** : Cette page d'aide présente uniquement les informations optionnelles de connexion disponibles pour les Connecteurs Natifs (également appelés Accès Natifs) et OLE DB.

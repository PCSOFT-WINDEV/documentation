


## &lt;Variable Connexion&gt;.ModifiePropriétéServeur (Fonction)

***En anglais : &lt;Connection variable&gt;.ModifyServerProperty***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie les propriétés d'un serveur HFSQL.

**Remarque** : Il est possible de connaître les propriétés d'un serveur HFSQL grâce à la fonction [&lt;Variable Connexion&gt;.InfoPropriétéServeur](../WDLang4/1000022675.md).




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Description de la connexion
MaConnexion est une Connexion

ResModif est un booléen
ResModif = MaConnexion.ModifiePropriétéServeur(hNotes, "Serveur HFSQL Client/Serveur")
SI ResModif = Vrai ALORS
	Info("Modification effectuée")
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Connexion>.ModifiePropriétéServeur(<Propriété à modifier> , <Nouvelle valeur>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si les propriétés ont été modifiées,

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. La fonction [HErreurInfo](../WDLang4/3044071.md) permet d'identifier l'erreur.




**`<Connexion> : (Variable de type Connexion)`**

Nom de la variable de type [Connexion](../WDLang4/1514073.md) décrivant la connexion pour laquelle la valeur d'une propriété doit être modifiée.

**`<Propriété à modifier> : (Constante de type Entier)`**

Propriété du serveur à modifier : 


|   |   |
| --- | --- |
| *hNotes* | Description du serveur ou toute autre information associée au serveur. <br>Si cette constante est utilisée, &lt;Nouvelle valeur&gt; est une chaîne de caractères contenant le texte de la note. |



**`<Nouvelle valeur> : (Chaîne de caractères)`**

Nouvelle valeur de la propriété.

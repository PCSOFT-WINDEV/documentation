


## Fonctions OLE
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>



|   |   |
| --- | --- |
| [ObjetActif](../WDLang3/3057006.md) | Permet d'utiliser un objet Automation déjà lancé sur le poste en cours. |
| [OLECharge](../WDLang3/3057002.md) | Initialise un champ OLE avec un objet OLE. |
| [OLEColle](../WDLang3/3057003.md) | Colle le contenu du presse-papiers de Windows dans un champ OLE. |
| [OLECopie](../WDLang3/3057004.md) | Copie l'objet présent dans un champ OLE vers le presse-papiers de Windows. |
| [OLECréeCopie](../WDLang3/3057005.md) | Associe un champ OLE à un objet OLE en précisant un fichier de base. |
| [OLECréeVide](../WDLang3/3057001.md) | Associe un champ OLE à un objet OLE en invoquant un serveur OLE spécifique. |
| [OLEEdite](../WDLang3/3057008.md) | Édite un objet OLE contenu dans un champ OLE. |
| [OLEJoue](../WDLang3/3057009.md) | Permet :<br><br>- soit de récupérer la liste des opérations possibles (autres que l'édition) depuis un serveur, <br><br>- soit d'effectuer une opération.<br><br><br> |
| [OLESauve](../WDLang3/3057011.md) | Sauve l'objet contenu dans un champ OLE sous forme binaire. |
| [OLEServeur](../WDLang3/3057010.md) | Liste les serveurs OLE enregistrés auprès du système en cours. |



**Remarque** : Les fonctions OLE fonctionnent avec Office 2007.

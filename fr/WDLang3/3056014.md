
## NetNomMachine (Fonction)

***En anglais : NetMachineName***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le nom de la machine locale ou le nom d'un poste dont on connaît l'adresse IP (résolution DNS inverse).  




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affiche le nom de la machine locale
Info(NetNomMachine())
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = NetNomMachine([<Adresse IP>])`
---

**`<Résultat> : (Chaîne de caractères)`**

Nom du poste en cours. 


**`<Adresse IP> : (Chaîne de caractères optionnelle)`**

Adresse IP (au format XXX.XXX.XXX.XXX) du poste dont on veut connaître le nom. Si ce paramètre n'est pas spécifié, le nom du poste en cours sera retourné. Cette adresse doit être une adresse valide.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Adresse IP valide
<a name="adresse_valide_ELTPARAGRAPHE000050"></a>Pour vérifier qu'une adresse IP est valide, utilisez la fonction [Ping](../WDLang1/3014013.md).


## SaaSAdminDésactiveLicence (Fonction)

***En anglais : SaaSAdminSubscriptionDeallocateLicense***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Retire une licence d'abonnement à un utilisateur SaaS.

**Remarques** : 

- L'utilisation de cette fonction nécessite d'avoir ouvert une session d'administration SaaS au préalable en utilisant la fonction [SaaSAdminConnecte](../WDLang3/1000019098.md).

- L'Administrateur SaaS aura accès à toutes les licences. Un gestionnaire de compte client aura accès uniquement aux licences de ses utilisateurs. 



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
S est un saasSite dynamique 
T est une saasTarification dynamique 
C est un saasClient dynamique 
Ab est un saasAbonnement dynamique 
U est un saasUtilisateur dynamique 

S = SaaSAdminChercheSite("Forum")
T = SaaSAdminChercheTarification("Promo 2010", S)
C = SaaSAdminChercheClient("Hybrid")
Ab = SaaSAdminChercheAbonnement(T, C)
U = SaaSAdminChercheUtilisateur("marc@hybrid.com")

// L'utilisateur marc n'est plus autorisé à profiter de la promo 2010
SaaSAdminDésactiveLicence(Ab, U)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Retrait d'une licence en utilisant l'abonnement

`<Résultat> = SaaSAdminDésactiveLicence(<Abonnement> , <Utilisateur>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'opération a été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Abonnement> : (Variable de type saasAbonnement)`**

Nom de la variable de type [saasAbonnement](../WDLang3/1000019088.md) correspondant à l'abonnement à modifier dans la base SaaS.

**`<Utilisateur> : (Variable de type saasUtilisateur)`**

Nom de la variable de type [saasUtilisateur](../WDLang3/1000019083.md) identifiant l'utilisateur à qui la licence doit être retirée. Cet utilisateur ne pourra plus profiter de l'abonnement.


<a name="SYNTAXE2"></a>

### Retrait d'une licence en utilisant la tarification

`<Résultat> = SaaSAdminDésactiveLicence(<Tarification> , <Site SaaS> , <Utilisateur>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'opération a été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Tarification> : (Chaîne de caractères)`**

Nom de la tarification concernée.

Un abonnement lié à cette tarification doit exister pour le compte client de l'utilisateur.

**`<Site SaaS> : (Chaîne de caractères)`**

Nom du site possédant la tarification.

**`<Utilisateur> : (Chaîne de caractères)`**

Login de l'utilisateur à qui la licence doit être retirée. Cet utilisateur ne pourra plus profiter de l'abonnement.

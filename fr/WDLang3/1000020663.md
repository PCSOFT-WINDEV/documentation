
## Calendrier (Type de variable)

***En anglais : Calendar***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **Calendrier** permet de manipuler un calendrier (ou agenda) associé à un compte utilisateur ou bien se trouvant sur l'appareil. Il est possible de lister les rendez-vous du calendrier grâce à la fonction [RendezVousListeCalendrier](../WDLang3/1000020661.md).

Les caractéristiques de ce calendrier peuvent être lues à l'aide de différentes propriétés WLangage.

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Remplit une liste avec les calendriers disponibles sur l'appareil
tabCalendrier est un tableau de Calendrier = RendezVousListeCalendrier()
POUR TOUT Calendrier de tabCalendrier
	ListeAjoute(LISTE_Liste1, Calendrier.Nom)
FIN
```

## Remarques
<a name="NOTE0_1"></a>


### Propriétés spécifiques à la description des variables de type Calendrier
<a name="proprietes_specifiques_description_des_variables_type_calendrier_ELTPARAGRAPHE000041"></a>Les propriétés suivantes peuvent être utilisées pour manipuler un Calendrier :

| **Nom de la propriété** | **Type manipulé** | **Effet** |
| --- | --- | --- |
| Couleur | Entier | Couleur associé au calendrier. Cette couleur est le résultat de la formule suivante :<br>Couleur = 65536 \* Composante Bleu + 256 \* Composante Verte + Composante Rouge. |
| ID | Entier sur 8 | Identifiant associé au calendrier. Cette propriété sera renseignée automatiquement lors de lecture du calendrier. Il est déconseillé de mémoriser cet identifiant au-delà de la durée de vie de l'application car cet identifiant peut être modifié lors de la synchronisation du calendrier. |
| Nom | Chaîne de caractères | Nom du calendrier. |
| NomCompte | Chaîne de caractères | Nom du compte utilisateur auquel est associé le calendrier. <br><br><br><br>Si cette propriété correspond à une chaîne vide (""), le calendrier n'est associé à aucun compte : il se trouve uniquement stocké localement sur l'appareil. |
| TypeCompte | Chaîne de caractères | Type de compte. <br><br><br><br><br><br>Si cette propriété correspond à une chaîne vide (""), le calendrier n'est associé à aucun compte et se trouve uniquement stocké localement sur l'appareil. |


Attention : Les propriétés du type **Calendrier** sont toutes en lecture seule et renseignées automatiquement lors de la lecture des calendriers avec la fonction [RendezVousListeCalendrier](../WDLang3/1000020661.md).

**Remarque** : Ces propriétés peuvent être utilisées avec une des deux syntaxes suivantes :

- &lt;Nom de la variable&gt;.&lt;Nom de la propriété&gt;

- &lt;Nom de la variable&gt;.&lt;Nom de la propriété&gt;

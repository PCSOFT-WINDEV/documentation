
## KiosqueDésactive (Fonction)

***En anglais : KioskDisable***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Désactive le mode kiosque pour l'application en cours. Les boutons système sont de nouveau affichés dans l'interface. 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Arrête le mode kiosque pour l'application en cours
KiosqueDésactive()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`KiosqueDésactive()`
---

<a name="NOTE0_1"></a>

## Remarques


### Cas particuliers

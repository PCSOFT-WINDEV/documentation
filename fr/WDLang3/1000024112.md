


## &lt;Graphe&gt;.OptionMenu (Fonction)

***En anglais : &lt;Chart&gt;.MenuOption***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie le texte d'une option du menu contextuel d'un graphe.

**Remarque** : 

- Par défaut, le menu contextuel est actif. Pour modifier l'état du menu contextuel, utilisez la fonction [&lt;Graphe&gt;.Menu](../WDLang3/1000024094.md).

- Cette fonction est utilisable uniquement si les menus des graphes sont en mode compatible (mode activé via la constante *grMenuPourCompatibilité* de la fonction [&lt;Graphe&gt;.Menu](../WDLang3/1000024094.md)).



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Modification de la ligne 50 du menu contextuel du graphe GRF_MonGraphe
GRF_MonGraphe.OptionMenu(50, "Imprimer le graphe")
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Champ Graphe>.OptionMenu(<Identifiant de l'option> , <Texte>)`
---

**`<Champ Graphe> : (Nom de champ)`**

Nom du champ Graphe à manipuler (présent dans l'éditeur de fenêtres).

**`<Identifiant de l'option> : (Entier)`**

Identifiant de l'option du menu contextuel.

**`<Texte> : (Chaîne de caractères)`**

Nouveau texte de l'option de menu spécifiée. Si ce paramètre correspond à une chaîne vide (""), la ligne de menu spécifiée est désactivée : cette ligne ne sera plus affichée dans le menu.

### Ligne active/désactive
<a name="ligne_activedesactive_ELTPARAGRAPHE000479"></a>

- Pour désactiver une option de menu, utilisez la fonction **&lt;Graphe&gt;.OptionMenu** en spécifiant une chaîne vide ("") dans le paramètre &lt;Texte&gt;.

- Pour réactiver une option de menu désactivée par la fonction **&lt;Graphe&gt;.OptionMenu**, il suffit de rappeler cette fonction avec le même identifiant de menu mais avec un libellé non vide.

- Les données définies avec la fonction **&lt;Graphe&gt;.OptionMenu** seront prises en compte lors du prochain appel à la fonction [&lt;Graphe&gt;.Dessine](../WDLang3/1000024062.md).



<a name="NOTE0_3"></a>


### Langues
<a name="langues_ELTPARAGRAPHE000505"></a>

- Le menu contextuel est prédéfini dans deux langues (Français et Anglais). Il n'est pas possible de rajouter une langue.

- Le texte est modifié dans la langue en cours. Pour gérer des langues étrangères, modifiez les textes prédéfinis.

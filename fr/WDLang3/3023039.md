
## Communication avec WINDEV Mobile
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000248"></a>
WINDEV Mobile permet de faire communiquer : 

- deux terminaux mobiles entre eux, 

- des téléphones Android, iOS, ...




Ces "dialogues" se font par infrarouge, par Wi-Fi, par GPRS, ...

### Coupures, perturbations de communication en FTP et HTTP
<a name="coupures_perturbations_communication_ftp_http_ELTPARAGRAPHE000195"></a>En cas de perturbations de la communication (coupure, parasite, ...), l'application WINDEV peut rester bloquée : il est nécessaire d'éteindre puis de redémarrer le Pocket PC pour ré-initialiser la couche de communication de Windows Mobile.

Pour afficher un message à l'utilisateur lui indiquant le problème et la marche à suivre, il est conseillé d'inclure le traitement à surveiller dans un thread (par exemple le transfert FTP). Ce thread sera surveillé régulièrement. En cas de problème, il sera alors possible d'afficher un message clair (par exemple "Une coupure réseau a été détectée. Le fonctionnement de certaines applications peut être perturbé. Nous vous conseillons de redémarrer votre Mobile").

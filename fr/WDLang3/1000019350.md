
## WiFiActive (Fonction)

***En anglais : WiFiActivate***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Active ou désactive le Wi-Fi sur l'appareil.

Remarques : 

- Pour connaître l'état d'activation du Wi-Fi sur l'appareil ou être notifié en cas de changement d'état, utilisez la fonction [WiFiEtat](../WDLang3/1000019354.md).

- L'accès Wi-Fi n'est pas disponible dans l'émulateur Android.







<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI Val(SysVersionAndroid(sysVersionNuméro))<10 ALORS  
	SI WiFiEtat() = wifiDésactivé ALORS
		WiFiActive(Vrai)
	FIN
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = WiFiActive([<Activation du Wi-Fi>])`
---

**`<Résultat> : (Constante de type Entier)`**

Etat d'activation du Wi-Fi après la modification :


|   |   |
| --- | --- |
| *wifiActif* | Le Wi-Fi est activé sur l'appareil. |
| *wifiActivationEnCours* | Le Wi-Fi est en cours d'activation sur l'appareil. |
| *wifiDésactivationEnCours* | Le Wi-Fi est en cours de désactivation sur l'appareil. |
| *wifiDésactivé* | Le Wi-Fi est désactivé sur l'appareil. |
| *wifiErreur* | L'état d'activation du Wi-Fi n'a pas pu être récupéré. |





**`<Activation du Wi-Fi> : (Booléen optionnel)`**

Nouvel état d'activation du Wi-Fi : 

- <u><u><u><u>Vrai</u></u></u></u> (valeur par défaut) : Active le Wi-Fi sur l'appareil. 

- <u><u><u><u>Faux</u></u></u></u> : Désactive le Wi-Fi sur l'appareil.

## Remarques


### Permissions requises
<a name="permissions_requises_ELTPARAGRAPHE000090"></a>L'utilisation de cette fonction entraîne la modification des permissions requises par l'application.

Permissions requises :  

- ACCESS_WIFI_STATE : cette permission permet aux applications d'accéder à des informations concernant les réseaux Wi-Fi.

- CHANGE_WIFI_STATE : cette permission permet aux applications de modifier l'état de connectivité Wi-Fi.

- WAKE_LOCK : cette permission permet d'utiliser PowerManager WakeLocks afin d'empêcher le processeur de se mettre en veille ou d'empêcher l'écran de se griser.


## TâcheSupprime (Fonction)

***En anglais : TaskDelete***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime la tâche en cours de la liste des tâches d'une messagerie Lotus Notes ou Outlook.

**Attention** : Cette fonction **ne permet pas** de supprimer la tâche en cours de la liste des tâches d'une messagerie Outlook Express.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ouverture d'une session Lotus Notes
IDSession est un entier
IDSession = EmailOuvreSessionNotes("MotDePasse", "ServeurMars", "mail\julie.nsf",...
					"C:\Users\Julie\AppData\Local\Lotus\Notes\Data\user.id")
// Lecture de la première tâche
TâchePremier(IDSession)
// Suppression de cette tâche
TâcheSupprime(IDSession)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = TâcheSupprime(<Identifiant de la session>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la suppression a été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour connaître le libellé de l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*. 




**`<Identifiant de la session> : (Entier)`**

Identifiant de la session à utiliser. Cet identifiant est renvoyé par une des fonctions suivantes : [EmailOuvreSessionNotes](../WDLang3/3032121.md), [EmailOuvreSessionOutlook](../WDLang3/3032128.md), [NotesOuvreBase](../WDLang3/1000005101.md) ou [OutlookOuvreSession](../WDLang3/3032123.md).

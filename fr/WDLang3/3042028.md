


## grEpaisseurTrait (Fonction)

***En anglais : grLineThickness***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Récupère et / ou change la valeur de l'épaisseur du trait dans un graphe de type Courbe.

![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Les graphes](../WDChamp/3042060.md). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Graphe créé par programmation
// Modifie l'épaisseur du trait pour toutes les séries du graphe
grEpaisseurTrait("MonGraphe", grToutesSéries, 4)
```
<a name="Exemple2"></a>

```wl
// Champ Graphe
// Modifie l'épaisseur du trait pour toutes les séries du graphe
grEpaisseurTrait(GRF_MonGraphe, grToutesSéries, 4)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Récupérer l'épaisseur du trait (graphe de type Courbe)

`<Résultat> = grEpaisseurTrait(<Nom du graphe> , <Numéro de série>)`
---

**`<Résultat> : (Entier)`**

Epaisseur du trait.

**`<Nom du graphe> : (Nom de champ ou chaîne de caractères)`**

Nom du graphe à manipuler. Ce nom correspond :

- soit au nom du graphe défini par programmation avec la fonction [grCrée](../WDLang3/3042005.md).

- soit au nom du champ Graphe (présent dans l'éditeur de fenêtres ou d'états).


Ce graphe doit être de type Courbe.



**`<Numéro de série> : (Entier)`**

Numéro de la série pour laquelle on veut connaître l'épaisseur du trait.  


<a name="SYNTAXE2"></a>

### Modifier l'épaisseur du trait (graphe de type Courbe)

`grEpaisseurTrait(<Nom du graphe> , <Numéro de série> , <Nouvelle épaisseur>)`
---

**`<Nom du graphe> : (Nom de champ ou chaîne de caractères)`**

Nom du graphe à manipuler. Ce nom correspond :

- soit au nom du graphe défini par programmation avec la fonction [grCrée](../WDLang3/3042005.md).

- soit au nom du champ Graphe (présent dans l'éditeur de fenêtres ou d'états).


Ce graphe doit être de type Courbe.



**`<Numéro de série> : (Entier)`**



- Numéro de la série pour laquelle on veut modifier l'épaisseur du trait. 

- Constante *grToutesSéries* pour changer l'épaisseur de toutes les séries (y compris les nouvelles séries).




**`<Nouvelle épaisseur> : (Entier)`**

Nouvelle épaisseur du trait. Ce paramètre peut correspondre à une valeur supérieure à 0. Par défaut, l'épaisseur du trait est 1.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Unités prises en compte pour l'épaisseur
<a name="unites_prises_compte_pour_epaisseur_ELTPARAGRAPHE000176"></a>L'épaisseur d'un trait correspond à des valeurs supérieures ou égales à 0.

- Si l'épaisseur correspond à 0, l'épaisseur sera de 1 pixel à l'écran (soit 1 pica à l'impression).

- Si l'épaisseur est supérieure ou égale à 1, l'épaisseur sera de x pixels à l'écran, soit x pica à l'impression.



<a name="NOTE0_2"></a>


### Divers
<a name="divers_ELTPARAGRAPHE000188"></a>

- Si la fonction **grEpaisseurTrait** est utilisée sur un type de graphe autre que courbe, aucune erreur n'est renvoyée. L'information est stockée et sera utilisée lorsque le type du graphe sera modifié (avec la fonction [grType](../WDLang3/3042062.md) ou avec le menu contextuel du graphe).

- Équivalence : la fonction **grEpaisseurTrait** est équivalente à la propriété [Epaisseur](../Proprietes/2510144.md).

- Les données définies avec la fonction **grEpaisseurTrait** seront prises en compte lors du prochain appel à la fonction [grDessine](../WDLang3/3042026.md).

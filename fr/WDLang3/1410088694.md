
## iOSRépertoireAppGroup (Fonction)

***En anglais : iOSAppGroupDirectory***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le chemin du répertoire du conteneur permettant de partager des fichiers entre plusieurs applications appartenant à un même App Group iOS.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
soit sConteneur = iOSRépertoireAppGroup("group.6233ABCD3.fr.pcsoft.devteam")
soit nFichier = fOuvre(sConteneur + [fSep] + "partage.txt")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = iOSRépertoireAppGroup(<AppGroup>)`
---

**`<Résultat> : (Chaîne de caractères)`**

Chemin du répertoire qui contient les fichier partagés par le groupe d'applications.

**`<AppGroup> : (Chaîne de caractères)`**

Nom du groupe "App Group".



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
**Pour utiliser un espace de stockage partagé entre applications (ou Widgets) du même éditeur** : 

1. Dans le console développeur d'Apple : 

	- Créez un identifiant "App Groups".  

	- Dans le profile de l'application, ajoutez la capability "App Groups" et indiquer l'identifiant spécifié à l'étape précédente.




2. Dans le code de votre application, utilisez la fonction **iOSRépertoireAppGroup**. 

3. Générez l'application iOS. Dans l'assistant de génération de l'application WINDEV Mobile, une étape spécifique "Dynamic Island" apparaît.
	 Cette étape permet d'indiquer les différents identifiants App Group utilisés pour partager des fichiers avec d'autres applications.

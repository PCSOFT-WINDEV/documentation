
## SaaSAdminAjouteAbonnement (Fonction)

***En anglais : SaaSAdminAddSubscription***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ajoute un abonnement à une tarification d'un site SaaS pour un compte client.

**Remarques** : 

- L'utilisation de cette fonction nécessite d'avoir ouvert une session d'administration SaaS au préalable en utilisant la fonction [SaaSAdminConnecte](../WDLang3/1000019098.md).

- L'Administrateur SaaS gère tous les abonnements. Un gestionnaire de compte client pourra uniquement ajouter à son propre compte un nouvel abonnement à un site auquel il est déjà abonné. 



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
S est un saasSite dynamique
T est une saasTarification dynamique
C est un saasClient dynamique

S = SaaSAdminChercheSite("Forum")
T = SaaSAdminChercheTarification("Promo 2010", S)
C = SaaSAdminChercheClient("Hybrid")

Ab est un saasAbonnement
Ab.NbLicences = 5

// Abonnement du client "Hybrid" à la tarification "Promo 2010" du site "Forum"
SaaSAdminAjouteAbonnement(Ab, T, C)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = SaaSAdminAjouteAbonnement(<Nouvel abonnement> , <Tarification> , <Compte client>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'ajout a été effectué, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Nouvel abonnement> : (Variable de type saasAbonnement)`**

Nom de la variable de type [saasAbonnement](../WDLang3/1000019088.md) contenant la description de l'abonnement à ajouter.

**`<Tarification> : (Variable de type saasTarification)`**

Nom de la variable de type [saasTarification](../WDLang3/1000019091.md) correspondant à la tarification à laquelle l'abonnement est rattaché.

**`<Compte client> : (Variable de type saasClient)`**

Nom de la variable de type [saasClient](../WDLang3/1000019072.md) correspondant au compte client à abonner.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Ajouter un nouvel abonnement à une tarification pour un client ne peut se faire que si aucun abonnement pour cette tarification n'est en cours pour ce client.


## SaaSAdminLitParamètreSite (Fonction)

***En anglais : SaaSAdminReadSiteParameter***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Lit un paramètre du site précédemment enregistré avec la fonction [SaaSAdminEcritParamètreSite](../WDLang3/1000019769.md). Cette fonction peut être exécutée en dehors du site SaaS concerné.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
s est une chaîne 
s = SaaSAdminLitParamètreSite("Jet-Thames", "Dupont" ,"logo")
SI ErreurDétectée = Vrai ALORS IMG_ImageLogo = s
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Lire une information

`<Résultat> = SaaSAdminLitParamètreSite(<Site SaaS> , <Client SaaS> , <Nom du paramètre>)`
---

**`<Résultat> : (Variant)`**

Valeur de la donnée recherchée. 

**`<Site SaaS> : (Chaîne de caractères)`**

Nom du site SaaS à manipuler.

**`<Client SaaS> : (Chaîne de caractères)`**

Nom du compte client SaaS.

**`<Nom du paramètre> : (Chaîne de caractères)`**

Nom de la donnée à lire. Cette donnée est unique par client. Si cette donnée n'existe pas, la variable [ErreurDétectée](../WDLang1/3087001.md) est à <u><u><u><u>Vrai</u></u></u></u>. 


<a name="SYNTAXE2"></a>

### Lire une information (utilisation des variables de type saasxxx)

`<Résultat> = SaaSAdminLitParamètreSite(<Site SaaS> , <Compte Client> , <Nom du paramètre>)`
---

**`<Résultat> : (Variant)`**

Valeur de la donnée recherchée. 

**`<Site SaaS> : (Variable de type saasSite)`**

Nom de la variable de type [saasSite](../WDLang3/1000019068.md) correspondant au site SaaS à manipuler. 

**`<Compte Client> : (Variable de type saasClient)`**

Nom de la variable de type [saasClient](../WDLang3/1000019072.md) correspondant au compte client SaaS. 

**`<Nom du paramètre> : (Chaîne de caractères)`**

Nom de la donnée à lire. Cette donnée est unique par client. Si cette donnée n'existe pas, la variable [ErreurDétectée](../WDLang1/3087001.md) est à <u><u><u><u>Vrai</u></u></u></u>.

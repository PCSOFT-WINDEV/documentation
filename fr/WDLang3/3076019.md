


## telEnregistre (Fonction)

***En anglais : tapiRecord***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Enregistre la communication en cours sous forme d'un fichier ".WAV". Cette fonction peut être utilisée par exemple pour créer le message d'un répondeur. L'enregistrement s'arrête automatiquement après la durée indiquée.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Le répondeur a été mis en route
// Le message de la personne qui téléphone est actuellement enregistré
// La durée maximale est fixée à 60 secondes
telEnregistre("M_" + DateDuJour() + Maintenant() + ".wav", 60, nIDAppel)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = telEnregistre(<Nom du fichier WAV> , <Durée maximale> [, <Identifiant de l'appel>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'enregistrement est effectué, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Nom du fichier WAV> : (Chaîne de caractères)`**

Nom et chemin du fichier ".WAV" à créer. L'extension du fichier doit être précisée.

**`<Durée maximale> : (Entier ou Durée)`**

Durée maximale de l'enregistrement (en secondes). Ce paramètre peut correspondre à : 

- un entier correspondant au nombre de secondes, 

- une variable de type Durée, 

- l'indication directe de la durée (par exemple 1 s ou 10 ms). 




**`<Identifiant de l'appel> : (Entier optionnel)`**

Identifiant de l'appel à manipuler. Cet identifiant est connu dans la procédure WLangage appelée par la fonction [telDémarreDétectionAppel](../WDLang3/3076029.md). Si ce paramètre n'est pas spécifié, l'appel en cours est manipulé.

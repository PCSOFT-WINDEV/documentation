
## SocketConnecteInfraRouge (Fonction)

***En anglais : SocketConnectInfrared***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Connecte un poste client à un socket donné via le port infrarouge.

**Rappel** : Un socket est une ressource de communication utilisée par les applications pour communiquer d'une machine à une autre sans se soucier du type de réseau.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI SocketConnecteInfraRouge("client", "service") = Faux ALORS
	Erreur("Erreur de connexion " + ErreurInfo(errMessage))
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = SocketConnecteInfraRouge(<Nom du socket> , <Nom du service>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la fonction a réussi, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. En cas d'erreur, il est possible de connaître le détail de l'erreur grâce à la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Nom du socket> : (Chaîne de caractères)`**

Nom à donner à la connexion ouverte sur le socket du serveur. Ce nom sera utilisé dans toutes les fonctions socket.

**`<Nom du service> : (Chaîne de caractères)`**

Nom du service spécifié dans la fonction [SocketCréeInfraRouge](../WDLang3/3070012.md).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Protocole IRDA
<a name="protocole_irda_ELTPARAGRAPHE000052"></a>La fonction **SocketConnecteInfraRouge** utilise le protocole IRDA (Infrared Data Association). Il est ainsi possible de connecter une application WINDEV avec n'importe quel appareil utilisant ce protocole (à condition de connaître le nom du service utilisé).


<a name="NOTE0_2"></a>




### Limite Windows
<a name="limite_windows_ELTPARAGRAPHE000064"></a>Le nombre maximum de sockets infrarouge utilisés simultanément est 80.


<a name="NOTE0_3"></a>




### Connexion à un socket donné
<a name="connexion_socket_donne_ELTPARAGRAPHE000073"></a>Pour connecter un poste client à un socket donné, utilisez la fonction [SocketConnecte](../WDLang3/3070016.md).




## &lt;Carte&gt;.SupprimeItinéraire (Fonction)

***En anglais : &lt;Map&gt;.DeleteItinerary***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime un itinéraire d'un champ Carte.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 
```wl
// Affiche un itinéraire automobile entre les deux adresses spécifiées
	
sID est une chaîne
sID = CARTE_ChampCarte.AjouteItinéraire(["Place de la comédie, Montpellier", ...
	"Avenue des Champs Elysées, Paris"], itinéraireAutomobile)

// Suppression de l'itinéraire 
CARTE_ChampCarte.SupprimeItinéraire(sID)
```


<a name="Exemple2"></a>

```wl
// Affiche un itinéraire correspondant à la liste de positions géographiques spécifiées
posDépart est une géoPosition
posDépart.Latitude = XXX
posDépart.Longitude = XXX

posArrivée est une géoPosition
posArrivée.Latitude = XXX
posArrivée.Longitude = XXX

sID est une chaîne
sID = CARTE_ChampCarte.AjouteItinéraire([posDépart, posArrivée])

// Suppression de l'itinéraire
CARTE_ChampCarte.SupprimeItinéraire(sID)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Champ Carte>.SupprimeItinéraire(<Identifiant>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'itinéraire a été supprimé, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Champ Carte> : (Nom de champ)`**

Nom du champ Carte à manipuler.

**`<Identifiant> : (Chaîne de caractères)`**

Identifiant de l'itinéraire à supprimer. Cet identifiant correspond à la valeur retournée par la fonction [&lt;Carte&gt;.AjouteItinéraire](../WDLang3/1000022420.md).



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Pour supprimer tous les itinéraires ajoutés sur un champ Carte, utilisez la fonction [&lt;Carte&gt;.SupprimeTout](../WDLang3/1000022472.md).

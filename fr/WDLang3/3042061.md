


## grTailleImage (Fonction)

***En anglais : grImageSize***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Définit la taille de l'image contenant le graphe.
![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Les graphes](../WDChamp/3042060.md). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Définition de la taille de l'image contenant le graphe "MonGraphe"
grTailleImage("MonGraphe", 10, 10, 100, 150)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`grTailleImage(<Nom du graphe> , <x1> , <y1> , <x2> , <y2>)`
---

**`<Nom du graphe> : (Chaîne de caractères)`**

Nom du graphe à manipuler, défini avec la fonction [grCrée](../WDLang3/3042005.md).

**`<x1> : (Réel)`**

Abscisse (en pixels) du coin haut gauche de l'image.

**`<y1> : (Réel)`**

Ordonnée (en pixels) du coin haut gauche de l'image.

**`<x2> : (Réel)`**

Abscisse (en pixels) du coin bas droit de l'image.

**`<y2> : (Réel)`**

Ordonnée (en pixels) du coin bas droit de l'image.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Fichier image
<a name="fichier_image_ELTPARAGRAPHE000054"></a>Si la destination du graphe correspond à un fichier BMP, EMF ou WMF, la fonction **grTailleImage** initialise les dimensions de l'image du graphe pour son enregistrement.
<a name="NOTE0_2"></a>


### Prise en compte
<a name="prise_compte_ELTPARAGRAPHE000064"></a>Les paramètres définis avec la fonction **grTailleImage** seront pris en compte lors du prochain appel à la fonction [grDessine](../WDLang3/3042026.md).




## &lt;Graphe&gt;.Graduation (Fonction)

***En anglais : &lt;Chart&gt;.Interval***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Indique le pas des graduations sur l'axe vertical/horizontal d'un graphe (c'est-à-dire l'intervalle entre deux graduations). 

**Rappel** : Par défaut, le pas des graduations est égal à 0. Ce pas est ajusté au mieux selon la place disponible dans la destination choisie.

![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Les graphes](../WDChamp/3042060.md). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Champ Graphe
// Axe horizontal du graphe gradué avec un pas de 10
GRF_MonGraphe.Graduation(10, grAbscisse)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Champ Graphe>.Graduation(<Pas> [, <Type de l'axe>])`
---

**`<Champ Graphe> : (Nom de champ)`**

Intervalle entre deux graduations. Si ce paramètre est égal à 0, le pas sera ajusté au mieux selon la place disponible dans la destination choisie.

**`<Pas> : (Réel)`**

Intervalle entre deux graduations. Si ce paramètre est égal à 0, le pas sera ajusté au mieux selon la place disponible dans la destination choisie.

**`<Type de l'axe> : (Constante optionnelle)`**

Type de l'axe pour lequel le pas des graduations est défini :


|   |   |
| --- | --- |
| *grAbscisse* <br>(Valeur par défaut) | Graduations sur l'axe des abscisses. |
| *grAltitude* | Graduations sur l'axe des altitudes (graphe de type Surface uniquement).<br><br>![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Cette constante n'est pas disponible. |
| *grOrdonnée* | Graduations sur l'axe des ordonnées. |
| *grOrdonnéeSecondaire* | Graduations sur l'axe secondaire des ordonnées. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Graduations des axes
<a name="graduations_des_axes_ELTPARAGRAPHE000257"></a>

- Limites : 100 graduations au-dessus et 100 graduations en dessous de l'axe des "X".

- La dernière graduation affichée est le premier multiple de &lt;Pas&gt; supérieur à la plus grande valeur des données. La totalité des données est donc visible quelle que soit la dimension de la destination.

- Pour modifier les graduations de début et de fin sur l'axe horizontal, vertical ou secondaire d'un graphe, utilisez la fonction [&lt;Graphe&gt;.Origine](../WDLang3/1000024114.md).

- Les données définies avec la fonction **&lt;Graphe&gt;.Graduation** seront prises en compte lors du prochain appel à la fonction [&lt;Graphe&gt;.Dessine](../WDLang3/1000024062.md).

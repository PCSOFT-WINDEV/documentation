


## EmailAjouteDossier (Fonction)

***En anglais : EmailAddFolder***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ajoute un dossier dans le logiciel de messagerie (Outlook, logiciel de messagerie utilisant le protocole IMAP (Internet Message Access Protocol), ...).  

Ce dossier est créé dans la branche "Dossiers personnels".

**Rappel** : Cette fonction ne permet pas d'ajouter un dossier dans le logiciel de messagerie Outlook Express.






<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Protocole IMAP
MaSession est un emailSessionIMAP
...
// Création d'un dossier dans le dossier en cours 
EmailAjouteDossier(MaSession, EmailDossierCourant(MaSession) + "\Travail")

// Création d'un dossier dans le dossier personnel
EmailAjouteDossier(MaSession, "Dossiers personnels\Boîte de réception\TravailPerso")
```


<a name="Exemple2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 
```wl
// Ouverture de la session Outlook
IDSession = EmailOuvreSessionOutlook("MonProfil")
...
// Création d'un dossier dans le dossier en cours 
EmailAjouteDossier(IDSession, EmailDossierCourant(IDSession) + "\Travail")

// Création d'un dossier dans le dossier personnel
EmailAjouteDossier(IDSession, "Dossiers personnels\Boîte de réception\TravailPerso")
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = EmailAjouteDossier(<Session> , <Chemin du dossier>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'ajout a bien été effectué, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. En cas d'erreur, la variable [ErreurDétectée](../WDLang1/3087001.md) est à <u><u><u><u>Vrai</u></u></u></u>. Pour connaître le libellé de l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.




**`<Session> : (Chaîne de caractères, Entier ou variable de type emailSessionOutlook ou emailSessionIMAP)`**

Identifiant de la session email en cours. Cet identifiant peut être :

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) un entier renvoyé par la fonction [EmailOuvreSessionOutlook](../WDLang3/3032128.md) ou [OutlookOuvreSession](../WDLang3/3032123.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [emailSessionOutlook](../WDLang3/1000018767.md).

- une variable de type [emailSessionIMAP](../WDLang3/1000018957.md).

- le nom d'utilisateur passé à la fonction [EmailOuvreSessionIMAP](../WDLang3/1000019189.md).




**`<Chemin du dossier> : (Chaîne de caractères Unicode)`**

Chemin du dossier à créer. Ce chemin doit obligatoirement commencer par "Dossiers personnels".

Si ce chemin contient plusieurs branches n'existant pas, toutes les branches sont créées.

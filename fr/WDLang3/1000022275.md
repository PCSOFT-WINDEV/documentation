


## &lt;Variable emailSessionIMAP&gt;.ChangeDossier (Fonction)

***En anglais : &lt;emailIMAPSession variable&gt;.ChangeFolder***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie le dossier en cours dans le logiciel de messagerie utilisant le protocole IMAP.   

Ce dossier est présent dans la branche "Dossiers personnels". 

Attention : Les dossiers de type "dossier générique" ne sont pas gérés. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Protocole IMAP
SAI_USER est un emailSessionIMAP
...
SI EmailOuvreSessionIMAP(SAI_USER, SAI_PASSWORD, ...
		"imap.mondomaine.fr") = Vrai ALORS
	SAI_USER.ChangeDossier("Dossiers personnels\Boîte de réception\Travail") 
	...
SINON
	Erreur("Impossible d'établir la connexion")
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Session>.ChangeDossier(<Chemin du dossier>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la modification a bien été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire (par exemple, le chemin spécifié n'existe pas). En cas d'erreur, la variable [ErreurDétectée](../WDLang1/3087001.md) est à <u><u><u><u>Vrai</u></u></u></u>. Pour connaître le libellé de l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.




**`<Session> : (Variable de type emailSessionIMAP)`**

Nom de la variable de type [emailSessionIMAP](../WDLang3/1000018957.md) correspondant à la session email en cours.

**`<Chemin du dossier> : (Chaîne de caractères Unicode)`**

Chemin du nouveau dossier en cours.




## ContactDernier (Fonction)

***En anglais : ContactLast***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Lit le dernier contact présent dans le carnet d'adresses d'une messagerie Lotus Notes ou Outlook.
**Attention** : Cette fonction **ne permet pas** de lire le dernier contact présent dans le carnet d'adresses d'une messagerie Outlook Express.

Lors de la lecture d'un contact, les données de ce contact sont mis à jour dans la [structure Contact](../WDLang3/1000006111.md). Pour récupérer les données du contact en cours, il suffit de récupérer les valeurs de la [structure Contact](../WDLang3/1000006111.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ouverture d'une session Lotus Notes
IDSession est un entier
IDSession = EmailOuvreSessionNotes("MotDePasse", "ServeurMars", ...
		"mail/julie.nsf", "C:\Lotus Notes\julie.id")
// Lecture du dernier contact
ContactDernier(IDSession)
// Ajout des contacts trouvés dans un champ Liste
TANTQUE mContact.EnDehors = Faux
	ListeAjoute(LISTE_ListeContact, mContact.Nom + TAB + mContact.Prénom)
	ContactPrécédent(IDSession)
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Lire le dernier contact dans la messagerie Lotus Notes ou Outlook

`<Résultat> = ContactDernier(<Identifiant de la session>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le dernier contact a été lu, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour connaître le libellé de l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.




**`<Identifiant de la session> : (Entier)`**

Identifiant de la session à utiliser. Cet identifiant est renvoyé par la fonction [EmailOuvreSessionNotes](../WDLang3/3032121.md), [EmailOuvreSessionOutlook](../WDLang3/3032128.md), [NotesOuvreBase](../WDLang3/1000005101.md) ou [OutlookOuvreSession](../WDLang3/3032123.md).




## EmailSupprimeDossier (Fonction)

***En anglais : EmailRemoveFolder***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime un dossier dans le logiciel de messagerie Outlook ou sur un serveur IMAP. 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 
```wl
// Ouverture de la session
IDSession = EmailOuvreSessionOutlook("MonProfil")
...
// Suppression d'un dossier
EmailSupprimeDossier(IDSession, "Dossiers personnels\Boîte de réception\Travail")

// Suppression d'un dossier dans le dossier en cours
EmailSupprimeDossier(IDSession, EmailDossierCourant(IDSession) + "\Travail")
```


<a name="Exemple2"></a>

```wl
// Protocole IMAP
MaSession est un emailSessionIMAP
...
// Suppression d'un dossier dans le dossier en cours 
EmailSupprimeDossier(MaSession, EmailDossierCourant(MaSession) + "\Travail")

// Suppression d'un dossier dans le dossier personnel
EmailSupprimeDossier(MaSession, "Dossiers personnels\Boîte de réception\TravailPerso")
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = EmailSupprimeDossier(<Session> , <Chemin du dossier>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la suppression a été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. En cas d'erreur, la variable [ErreurDétectée](../WDLang1/3087001.md) est à <u><u><u><u>Vrai</u></u></u></u>. Pour connaître le libellé de l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.




**`<Session> : (Chaîne de caractères, Entier ou Variable de type emailSessionOutlook ou emailSessionIMAP)`**

Identifiant de la session email en cours. Cet identifiant peut être :

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) un entier renvoyé par la fonction [EmailOuvreSessionOutlook](../WDLang3/3032128.md) ou [OutlookOuvreSession](../WDLang3/3032123.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [emailSessionOutlook](../WDLang3/1000018767.md).

- une variable de type [emailSessionIMAP](../WDLang3/1000018957.md).

- le nom d'utilisateur passé à la fonction [EmailOuvreSessionIMAP](../WDLang3/1000019189.md).




**`<Chemin du dossier> : (Chaîne de caractères Unicode)`**

Chemin du dossier à supprimer. 



<a name="NOTE0"></a>

## Remarques
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 

### Cas d'un serveur Outlook
<a name="cas_serveur_outlook_ELTPARAGRAPHE000168"></a>Le dossier doit être présent dans la branche "Dossiers personnels".

Lors de l'appel de la fonction **EmailSupprimeDossier**, les manipulations suivantes sont automatiquement effectuées :

- les messages contenus dans le dossier à supprimer sont déplacés dans la Corbeille (dossier "Eléments supprimés").

- les sous-dossiers du dossier à supprimer et leur contenu sont déplacés dans la Corbeille (dossier "Eléments supprimés").

- le dossier à supprimer est définitivement supprimé.




Si le dossier à supprimer est déjà présent dans la Corbeille, le dossier et son contenu sont définitivement supprimés.

**Rappel** : Cette fonction ne permet pas de supprimer un dossier du logiciel de messagerie Outlook Express.


## App Tracking Transparency - Suivi d'activité
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000080"></a>
App Tracking Transparency permet à l'utilisateur d'autoriser une application iOS à suivre son activité sur les applications mobiles et les sites Web d'autres sociétés à des fins publicitaires ou de partage avec des gestionnaires de données. 

Sous iOS 14.5, les applications doivent obtenir l'autorisation de l'utilisateur avant de suivre son activité. 

A quoi correspond le suivi d'activité ? On parle de "suivi d'activité" lorsque les données permettant d'identifier l'utilisateur ou son appareil sont associées à des données similaires provenant d'autres applications (ou sites Web) à des fins publicitaires (publicité ciblée ou campagne publicitaire) ou lorsque les informations recueillies sont partagées avec des gestionnaires de données (ventes d'informations). 

Si l'utilisateur autorise le suivi d'activité, il sera possible de suivre ou accéder à l'identifiant publicitaire de l'appareil, appelé IDFA (Identifier for Advertisers). L'application pourra vous proposer des publicités personnalisées sur d'autres applications ou sites web. Si l'utilisateur refuse, le ciblage publicitaire sera alors réduit.

## Comment gérer l'App Tracking Transparency ?
<a name="comment_gerer_app_tracking_transparency_ELTTEXTE000104"></a>
WINDEV Mobile permet de : 

- ouvrir la fenêtre permettant de demander l'autorisation de suivi d'activité. ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=AppleATT.gif)
Cette fenêtre peut être personnalisée comme toutes les demandes d'autorisation, en spécifiant un message personnalisé à afficher dans l'étape "Permissions" de l'assistant de génération de l'application iOS. Il suffit de saisir un texte spécifique pour la permission "Tracking Usage Description". 

- vérifier si l'autorisation de suivi d'activité a été donnée.

- obtenir l'identifiant publicitaire. Cet identifiant permet d'identifier de manière unique le périphérique utilisé. 




WINDEV Mobile met à votre disposition les fonctions suivantes afin de gérer l'App Tracking Transparency : 



|   |   |
| --- | --- |
| [AppleIdentifiantPublicitaire](../WDLang3/1410087936.md) | Renvoie l'identifiant publicitaire de l'appareil (iOS uniquement). |
| [AppleTrackingDemandeAutorisationAsynchrone](../WDLang3/1410087935.md) | Affiche la demande de permission concernant le suivi de l'activité de l'utilisateur. |
| [AppleTrackingEtatAutorisation](../WDLang3/1410087934.md) | Permet de savoir si l'application est autorisée à réaliser un suivi d'activité sur les données de l'utilisateur (iOS uniquement). |

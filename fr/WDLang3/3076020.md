
## telErreur (Fonction)

***En anglais : tapiError***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Indique si la dernière fonction telxxx ([fonctions de gestion des appels téléphoniques](../WDLang3/3076005.md)) a renvoyé une erreur dans le module TAPI. Cette fonction est particulièrement utile dans le cas des fonctions ne renvoyant pas Vrai ou Faux comme résultat.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI telNuméroAppelé() = "" ALORS
	SI telErreur() = Vrai ALORS
		Erreur(ErreurInfo(errMessage))
	FIN
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = telErreur()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Faux</u></u></u></u> si aucune erreur n'est survenue lors de l'utilisation d'une fonction de téléphonie, 

- <u><u><u><u>Vrai</u></u></u></u> en cas d'erreur. Le libellé ou le code de l'erreur TAPI peut être connu avec la fonction [ErreurInfo](../WDLang1/3013008.md).

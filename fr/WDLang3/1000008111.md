
## GroupeAjoute (Fonction)

***En anglais : GroupAdd***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ajoute un groupe de contacts dans le carnet d'adresses d'une messagerie Lotus Notes ou Outlook. La [structure Groupe](../WDLang3/1000008120.md) permet de définir les données de ce groupe de contacts.

**Attention** : Cette fonction **ne permet pas** d'ajouter un groupe de contacts dans le carnet d'adresses d'une messagerie Outlook Express. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ouverture d'une session Lotus Notes
IDSession est un entier
IDSession = EmailOuvreSessionNotes("MotDePasse", "ServeurMars", ...
		"mail\julie.nsf", "C:\Users\Julie\AppData\Local\Lotus\Notes\Data\user.id")
// Initialisation des variables de la structure Groupe
GroupeRAZ()
// Renseignements des variables pour le nouveau groupe
mGroupe.Nom = "Mes amis"
mGroupe.Description = "Groupe de mes amis"
mGroupe.NbContact = 3
mGroupe.Contact[1] = "Pierre"
mGroupe.Contact[2] = "Paul"
mGroupe.Contact[3] = "Jacques"
// Ajout du groupe
SI GroupeAjoute(IDSession) = Vrai ALORS
	Info("Groupe de contacts ajouté")
SINON
	Erreur("Erreur lors de l'ajout", ErreurInfo())
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = GroupeAjoute(<Identifiant de la session>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'ajout a été effectué, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour connaître le libellé de l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.




**`<Identifiant de la session> : (Entier)`**

Identifiant de la session à utiliser. Cet identifiant est renvoyé par une des fonctions suivantes : [EmailOuvreSessionNotes](../WDLang3/3032121.md), [EmailOuvreSessionOutlook](../WDLang3/3032128.md), [NotesOuvreBase](../WDLang3/1000005101.md) ou [OutlookOuvreSession](../WDLang3/3032123.md).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Les groupes de contacts ne sont pas gérés par Outlook version 97. Il n'est donc pas possible d'ajouter un groupe de contacts dans cette version de Outlook.

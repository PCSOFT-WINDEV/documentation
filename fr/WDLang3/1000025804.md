


## &lt;Carte&gt;.LimiteZone (Fonction)

***En anglais : &lt;Map&gt;.LimitArea***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Limite la zone géographique affichée dans le champ Carte.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
p1 est une géoPosition
p1.Latitude = 0
p1.Longitude = 0
p2 est une géoPosition
p2.Latitude = 3
p2.Longitude = 3
CARTE_MaCarte.LimiteZone(p1, p2)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Définir la zone géographique

`<Résultat> = <Champ Carte>.LimiteZone(<Position Nord Ouest> , <Position Sud Est>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la limitation de la zone affichable a été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Champ Carte> : (Nom de champ)`**

Nom du champ Carte à manipuler.

**`<Position Nord Ouest> : (Variable de type géoPosition)`**

Nom de la variable de type [géoPosition](../WDLang3/1000019191.md) correspondant au point Nord Ouest de la zone affichée dans le champ Carte.

**`<Position Sud Est> : (Variable de type géoPosition)`**

Nom de la variable de type [géoPosition](../WDLang3/1000019191.md) correspondant au point Sud Est de la zone affichée dans le champ Carte.


<a name="SYNTAXE2"></a>

### Supprimer la zone géographique

`<Résultat> = <Champ Carte>.LimiteZone()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la limitation de la zone affichable a été supprimée,

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Champ Carte> : (Nom de champ)`**

Nom du champ Carte à manipuler.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Selon le niveau de zoom, le champ Carte peut afficher une zone plus étendue que la zone spécifiée, mais le centre de la carte sera toujours situé à l'intérieur de la zone spécifiée en paramètre.

- Lors de l'appel de la fonction **&lt;Carte&gt;.LimiteZone**, la carte affichée dans le champ Carte sera automatiquement positionnée sur la zone spécifiée.

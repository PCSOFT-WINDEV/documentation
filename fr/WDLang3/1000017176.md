


## Fonctions SNMP
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions SNMP sont les suivantes : 



|   |   |
| --- | --- |
| [SNMPChaîneVersOID](../WDLang3/1000017076.md) | Convertit un OID de sa représentation textuelle vers sa représentation numérique. |
| [SNMPChargeMIB](../WDLang3/1000017074.md) | Charge en mémoire et analyse un fichier MIB. |
| [SNMPEcrit](../WDLang3/1000017071.md) | Écrit une ou plusieurs valeurs sur un agent SNMP pour lequel une session SNMP a été ouverte. |
| [SNMPFermeSession](../WDLang3/1000017068.md) | Ferme une session SNMP. |
| [SNMPLit](../WDLang3/1000017069.md) | Lit une ou plusieurs valeurs d'un agent avec lequel une session SNMP a été ouverte. |
| [SNMPLitSuivant](../WDLang3/1000017070.md) | Lit la valeur suivant la dernière valeur récupérée d'un agent SNMP. |
| [SNMPLitTableau](../WDLang3/1000017217.md) | Lit un tableau de valeurs SNMP. |
| [SNMPOIDAccès](../WDLang3/1000017077.md) | Renvoie les modes d'accès autorisés d'un OID spécifique. |
| [SNMPOIDDescription](../WDLang3/1000017079.md) | Renvoie la description d'un OID. |
| [SNMPOIDEtat](../WDLang3/1000017078.md) | Renvoie l'état d'un OID. |
| [SNMPOIDType](../WDLang3/1000017080.md) | Renvoie le type de données contenu dans un OID. |
| [SNMPOIDVersChaîne](../WDLang3/1000017075.md) | Convertit un OID de sa représentation numérique vers sa représentation textuelle. |
| [SNMPOuvreSession](../WDLang3/1000017067.md) | Ouvre une session SNMP. |
| [SNMPTrapActive](../WDLang3/1000017072.md) | Active l'interception d'une "trap" SNMP envoyée par un agent. |
| [SNMPTrapDésactive](../WDLang3/1000017073.md) | Désactive l'interception d'une trap SNMP. |

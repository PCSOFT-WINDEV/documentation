
## BufferVersURI (Fonction)

***En anglais : BufferToURI***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Sauvegarde le contenu d'un buffer dans une ressource identifiée par son URI.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ouvre le sélecteur de ressource en mode création
uriDocImage est une URI = URISélecteur(uriImage, uriCreation)
SI PAS ErreurDétectée ALORS
	// Sauvegarde le contenu du buffer contenant l'image dans le document créé 
	BufferVersURI(bufImage, uriDocImage)
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = BufferVersURI(<Buffer à manipuler> , <URI destination> [, <Mode>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la sauvegarde a été effectuée, 

- <u><u><u><u>faux</u></u></u></u> dans le cas contraire. 


En cas d'erreur, la variable [ErreurDétectée](../WDLang1/3087001.md) est à <u><u><u><u>Vrai</u></u></u></u>. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).

**`<Buffer à manipuler> : (Buffer)`**

Buffer correspondant au contenu à sauvegarder dans la ressource (identifiée par son URI).

**`<URI destination> : (Chaîne de caractères ou variable de type URI)`**

URI de la ressource dans laquelle le fichier doit être sauvegardé. Cette ressource peut être identifiée : 

- par une chaîne de caractères correspondant à une URI, 

- par une variable de type [URI](../WDLang3/1000024027.md). 


Remarque : L'URI spécifiée doit être de type "file" (fichier externe) ou "content" (ressource provenant d'un 'content provider') et être accessible en écriture.

**`<Mode> : (Constante optionnelle de type Entier)`**

Mode de sauvegarde : 



|   |   |
| --- | --- |
| *uriAjout* | Le contenu du buffer est ajouté à la fin du contenu initial de la ressource. |
| *uriCréation*<br>(Valeur par défaut) | Le contenu initial de la ressource est remplacé par celui du buffer. <br> |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Pour sélectionner une ressource et obtenir son URI, utilisez la fonction [URISélecteur](../WDLang3/1000024026.md).

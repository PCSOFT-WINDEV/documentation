


## UPNPModifieVolume (Fonction)

***En anglais : UPNPModifyVolume***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie le volume d'un périphérique UPnP.

**Attention** : Pour utiliser cette fonction, le périphérique doit posséder un service "RenderingControl".
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
tabPériphérique est un tableau de upnpPériphériques 
tabPériphérique = UPNPListePériphérique(...
		"urn:schemas-upnp-org:device:MediaRenderer:1", 5)
POUR TOUT Périphérique de tabPériphérique
	UPNPModifieVolume(Périphérique, 50)
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = UPNPModifieVolume(<Périphérique UPnP> , <Nouvelle valeur>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le volume a été modifié, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 




**`<Périphérique UPnP> : (Variable de type upnpPériphérique)`**

Nom de la variable [upnpPériphérique](../WDLang3/1000020903.md) correspondant au périphérique UPnP à manipuler.

**`<Nouvelle valeur> : (Entier)`**

Nouveau volume à appliquer. Ce paramètre doit être compris entre 0 et 100.

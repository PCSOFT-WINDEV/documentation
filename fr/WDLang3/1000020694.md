
## WiFiSelectCarteRéseau (Fonction)

***En anglais : WiFiSelectNetworkCard***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Sélectionne la carte réseau Wi-Fi qui sera utilisée pour les prochains appels des fonctions WiFi.

**Remarque** : L'appel à cette fonction n'est pas obligatoire. Par défaut, la première carte réseau Wi-Fi disponible est utilisée.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
sMaListe est une chaîne
nID est un entier  
sDescription est une chaîne

sMaListe = WiFiListeCarteRéseau()
// Récupère l'identifiant de la première carte trouvée
nID = Val(ExtraitChaîne(sMaListe, 1, TAB))
// Sélectionne la carte 
WiFiSelectCarteRéseau(nID)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = WiFiSelectCarteRéseau(<Identifiant carte réseau>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la sélection de la carte réseau Wi-Fi a été effectuée. 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. La fonction [ErreurInfo](../WDLang1/3013008.md) permet d'obtenir plus de renseignements sur le problème rencontré (par exemple si la carte réseau Wi-Fi n'est plus disponible).




**`<Identifiant carte réseau> : (Chaîne de caractères)`**

Identifiant de la carte réseau Wi-Fi à sélectionner. Cet identifiant peut être obtenu grâce à la fonction [WiFiListeCarteRéseau](../WDLang3/1000020693.md).

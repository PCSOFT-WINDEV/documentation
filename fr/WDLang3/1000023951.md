


## Fonctions SOAP (syntaxe préfixée)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


Les fonctions permettant de manipuler le protocole SOAP sont les suivantes : 



|   |   |
| --- | --- |
| [&lt;Variable wsRequête&gt;.AjouteAssertionSAML](../WDLang3/1000023948.md) | Ajoute une assertion SAML à une requête de Webservice. |
| [&lt;Variable wsRequête&gt;.AjouteSignatureXML](../WDLang3/1000023950.md) | Ajoute une signature XML à une requête de Webservice. |
| [&lt;Variable xsdEntité&gt;.AjouteAttribut](../WDLang3/1000023949.md) | Ajoute un attribut à une variable automatique générée par l'import d'un Webservice. |

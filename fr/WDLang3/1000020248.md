
## Evénement (Type de variable)

***En anglais : Event***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **Evénement** permet de définir toutes les caractéristiques avancées d'un événement : anniversaire, fête, ... 

Les caractéristiques de cet événement peuvent être définies et modifiées à l'aide de différentes propriétés WLangage. 

Ce type de variable est utilisé par les variables de type [Contact](../WDLang3/1000020254.md).

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Récupère la date d'anniversaire d'un contact
sDate est une chaîne
POUR TOUT evt de MonContact.Evénement
	SI evt.Type = evtAnniversaire ALORS
		sDate = evt.DateBrute
		RETOUR
	FIN
FIN
```

## Remarques


### Propriétés spécifiques à la description des variables de type Evénement
<a name="proprietes_specifiques_description_des_variables_type_evenement_ELTPARAGRAPHE000050"></a>Les propriétés suivantes peuvent être utilisées pour manipuler un événement :

| **Nom de la propriété** | **Type manipulé** | **Effet** |
| --- | --- | --- |
| DateBrute | Chaîne de caractères | Date de début de l'événement. La chaîne de caractères est non formatée. |
| Etiquette | Chaîne de caractères | Etiquette pour les types d'événement personnalisé (Optionnel).<br>Si la propriété **Type** ne correspond pas à la constante *evtPersonnalisé*, la valeur de cette propriété sera ignorée. |
| Type | Constante de type Entier | ![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Type d'événement : <br><br>- *evtAnniversaire* : Anniversaire / date de naissance. <br><br>- *evtAutre* : autres types.<br><br>- *evtFête* : Fête.<br><br>- *evtPersonnalisé* : type personnalisé correspondant à la valeur de la propriété **Etiquette**.<br><br><br> |


<a name="NOTE0_2"></a>


### Réinitialisation
<a name="reinitialisation_ELTPARAGRAPHE000097"></a>Pour réinitialiser le contenu d'une variable de type **Evénement**, il est possible d'utiliser la fonction [VariableRAZ](../WDLang1/3013069.md).


## Structure FaxEtatComplet
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000364"></a>
La structure FaxEtatComplet est automatiquement remplie lors de l'appel à la fonction [FaxEtat](../WDLang3/3037001.md).



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Détail de la structure FaxEtatComplet
<a name="detail_structure_faxetatcomplet_ELTTEXTE000388"></a>



|   |   |
| --- | --- |
| **FaxEtatComplet.DateHeureExpédition** | DateHeure<br><br>Date et heure à laquelle le fax sera envoyé. |
| **FaxEtatComplet.DépartementExpéditeur** | Chaîne de caractères<br><br>Département de l'expéditeur. |
| **FaxEtatComplet.EnErreur** | Booléen<br><br>Vrai : Indique si le fax a été abandonné pour quelque raison que ce soit (même par l'utilisateur) |
| **FaxEtatComplet.EnvoyéParConnexionEnCours** | Booléen<br><br>Vrai : Indique si un fax est envoyé via la connexion précisée dans la fonction [FaxEtat](../WDLang3/3037001.md) par le paramètre &lt;Identifiant Connexion&gt;. |
| **FaxEtatComplet.Etat** | Constante de type Entier<br><br>État du fax au niveau du périphérique :<br><br><br>\   \   \<br>\ --- \ --- \<br>\ *FaxEtatAnnulé* \ La transmission du fax est annulé. \<br>\ *FaxEtatAppel* \ Le fax est en phase de numérotation. \<br>\ *FaxEtatBloqué* \ Numéro du destinataire bloqué. \<br>\ *FaxEtatDéconnecté* \ Déconnexion en cours de transmission. \<br>\ *FaxEtatEnvoi* \ Le fax est en train d'être envoyé. \<br>\ *FaxEtatEnvoyé* \ Le fax a été envoyé. \<br>\ *FaxEtatErreur* \ Impossible d'obtenir l'état du fax. \<br>\ *FaxEtatErreurFatale* \ Erreur fatale. \<br>\ *FaxEtatFauxNuméro* \ Numéro de fax erroné. \<br>\ *FaxEtatGestionnaireEnAttente* \ Le périphérique est disponible. \<br>\ *FaxEtatHorsConnexion* \ Le périphérique est hors ligne et indisponible. \<br>\ *FaxEtatInactif* \ Envoi de fax inactif (pause entre deux appels par exemple) \<br>\ *FaxEtatIndisponible* \ Le fax est indisponible (utilisé par une autre application) \<br>\ *FaxEtatInit* \ La transmission est en phase d'initialisation \<br>\ *FaxEtatOccupé* \ Le destinataire est occupé \<br>\ *FaxEtatPasDeFax* \ Le périphérique a reçu un appel qui n'est pas un fax (appel vocal par exemple) \<br>\ *FaxEtatPasDeTonalité* \ Pas de tonalité \<br>\ *FaxEtatPrisEnCharge* \ Le gestionnaire de fax a pris en charge le document de fax et va le transmettre. \<br>\ *FaxEtatRéception* \ Le périphérique est en train de recevoir un fax \<br>\ *FaxEtatRépondu* \ Le destinataire a répondu \<br>\ *FaxEtatReporté* \ L'appel est reporté \<br>\ *FaxEtatRoutage* \ Le périphérique est en train de router un fax reçu. \<br>\ *FaxEtatSansRéponse* \ Le destinataire ne répond pas \<br>\ *FaxEtatSonnerie* \ Le fax du destinataire sonne \<br><br> |
| **FaxEtatComplet.EtatDansGestionnaire** | Entier.<br><br>Etat du fax dans le gestionnaire de fax. Peut correspondre à une combinaison des constantes suivantes :<br><br><br>\   \   \<br>\ --- \ --- \<br>\ *FaxEtatFileEchec* \ L'envoi du fax a échoué \<br>\ *FaxEtatFileEnAttente* \ Le fax est en attente \<br>\ *FaxEtatFileEnCours* \ Le fax est en cours d'envoi \<br>\ *FaxEtatFileEssaisExcédés* \ Le nombre d'essais maximal a été atteint et le fax ne sera pas envoyé. \<br>\ *FaxEtatFilePasDeLigne* \ Il n'y a pas de ligne disponible pour envoyer le fax. Il sera envoyé quand une ligne se libèrera. \<br>\ *FaxEtatFilePause* \ Le fax est en pause. \<br>\ *FaxEtatFileRéessaie* \ L'envoi du fax a échoué et un nouvel essai sera fait plus tard. \<br>\ *FaxEtatFileSuppression* \ Le fax est en cours de suppression. \<br><br> |
| **FaxEtatComplet.Identifiant** | Entier<br><br>Identifiant numérique du fax. |
| **FaxEtatComplet.LibelléEtat** | Chaîne de caractères<br><br>Description de l'état du fax. |
| **FaxEtatComplet.NbTotalPages** | Entier.<br><br>Nombre total de pages à transmettre. |
| **FaxEtatComplet.NomDestinataire** | Chaîne de caractères<br><br>Nom du destinataire. |
| **FaxEtatComplet.NomDocument** | Chaîne de caractères<br><br>Nom du fax. |
| **FaxEtatComplet.NomExpéditeur** | Chaîne de caractères<br><br>Nom de l'expéditeur. |
| **FaxEtatComplet.NomPériphérique** | Chaîne de caractères<br><br>Nom du périphérique (modem) utilisé pour la transmission. |
| **FaxEtatComplet.NuméroDestinataire** | Chaîne de caractères<br><br>Numéro du destinataire. |
| **FaxEtatComplet.NumPageEnCours** | Entier.<br><br>Numéro de la page en cours de transmission. |
| **FaxEtatComplet.Planification** | Constante de type entier.<br><br>Planification de l'envoi :<br><br><br>\   \   \<br>\ --- \ --- \<br>\ *FaxPlanificationHeureCreuse* \ Fax envoyé pendant les heures creuses. \<br>\ *FaxPlanificationHeureSpécifique* \ Fax envoyé à une heure précisée dans FaxEtatComplet.DateHeureExpédition. \<br>\ *FaxPlanificationMaintenant* \ Fax envoyé le plus vite possible. \<br><br> |
| **FaxEtatComplet.Propriétaire** | Chaîne de caractères.<br><br>Nom de l'utilisateur ayant émis le fax. |
| **FaxEtatComplet.SociétéExpéditrice** | Chaîne de caractères.<br><br>Société de l'expéditeur. |
| **FaxEtatComplet.TailleEnOctets** | Entier<br><br>Taille du document transmis, en octets. |
| **FaxEtatComplet.Terminé** | Booléen.<br><br>Indique si la transmission du fax est terminée (c'est-à-dire finie ou abandonnée). |
| **FaxEtatComplet.Type** | Constante de type entier.<br><br>Type de fax :<br><br><br>\   \   \<br>\ --- \ --- \<br>\ *FaxTypeEchecRéception* \ Le fax n'a pas encore été routé car il n'a pas été reçu. \<br>\ *FaxTypeEmission* \ Fax émis. \<br>\ *FaxTypeEnRoutage* \ Le routage du fax a échoué. Il sera re-tenté plus tard. \<br>\ *FaxTypeInconnu* \ Type inconnu car le fax n'a pas encore été planifié. \<br>\ *FaxTypeRéception* \ Fax reçu. \<br><br> |

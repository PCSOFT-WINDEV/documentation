
## SaaSAdminAjouteClient (Fonction)

***En anglais : SaaSAdminAddClient***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ajoute un compte client SaaS.

**Remarques** : 

- L'utilisation de cette fonction nécessite d'avoir ouvert une session d'administration SaaS au préalable en utilisant la fonction [SaaSAdminConnecte](../WDLang3/1000019098.md).

- L'utilisation de cette fonction est réservée à l'Administrateur SaaS. Un gestionnaire de compte client ne peut pas utiliser cette fonction. 



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Déclare un compte pour la société "Hybrid" dans la base du Webservice SaaS
S est un saasClient
S.Nom = "Hybrid"
SaaSAdminAjouteClient(S)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Ajouter un compte client (utilisation des variables de type saasxxx)

`<Résultat> = SaaSAdminAjouteClient(<Nouveau client>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'ajout a été effectué, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Nouveau client> : (Variable de type saasClient)`**

Nom de la variable de type [saasClient](../WDLang3/1000019072.md) contenant la description du compte client à ajouter.


<a name="SYNTAXE2"></a>

### Ajouter un nouveau client identifié par son nom

`<Résultat> = SaaSAdminAjouteClient(<Compte client> [, <Référence externe>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'ajout a été effectué, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Compte client> : (Chaîne de caractères)`**

Nom du client à rajouter dans la base SaaS.

**`<Référence externe> : (Chaîne de caractères optionnelle)`**

Référence externe du compte client. Si ce paramètre n'est pas précisé, la valeur de la référence externe correspond à chaîne vide ("").


## SaaSAdminChercheSite (Fonction)

***En anglais : SaaSAdminFindSite***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Recherche un site dans la base du Webservice SaaS.

**Remarques** : 

- L'utilisation de cette fonction nécessite d'avoir ouvert une session d'administration SaaS au préalable en utilisant la fonction [SaaSAdminConnecte](../WDLang3/1000019098.md).

- L'Administrateur SaaS aura accès à tous les sites SaaS. Un gestionnaire de compte client aura accès uniquement aux sites auxquels il est abonné. 



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Suppression du site SaaS Forum
S est un saasSite dynamique
S = SaaSAdminChercheSite("Forum")
SI ErreurDétectée() ALORS
	Erreur("Site non trouvé")
SINON
	SI SaaSAdminSupprimeSite(S) = Vrai ALORS
		Info("Site supprimé")
	FIN
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = SaaSAdminChercheSite(<Site>)`
---

**`<Résultat> : (Variable de type saasSite)`**

Variable de type [saasSite](../WDLang3/1000019068.md) contenant la description du site recherché.

Si le site n'est pas trouvé, la variable renvoyée est vide.

**`<Site> : (Chaîne de caractères)`**

Nom du site SaaS recherché.

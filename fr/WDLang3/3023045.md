## Partage des différents éléments WINDEV
<a name="partage_des_differents_elements_windev_ELTTEXTE000068"></a>
Il est possible d'utiliser les mêmes éléments WINDEV (analyse, états, requêtes, ...) dans une application WINDEV et dans une application WINDEV Mobile :

- **Fenêtres** : Les fenêtres n'ont pas le même format en WINDEV (fichier ".WDW") et en WINDEV Mobile (fichier ".WPW"). En effet, les fenêtres n'ont pas la même taille, les mêmes caractéristiques, les mêmes spécificités, la même approche utilisateur, ... WINDEV Mobile offre donc la possibilité d'importer des fenêtres WINDEV standard dans un projet WINDEV Mobile.

- **Aide** : Ce type d'élément n'existe pas sous Android ou iOS.

- **Analyse, états, requêtes, classes, collection de procédures, ...** : Ces éléments peuvent être directement partagés entre une application WINDEV et une application WINDEV Mobile.

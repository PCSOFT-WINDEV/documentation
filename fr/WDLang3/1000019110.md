
## SaaSAdminChercheService (Fonction)

***En anglais : SaaSAdminFindService***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Recherche de la description d'un service SaaS.

**Remarques** : 

- L'utilisation de cette fonction nécessite d'avoir ouvert une session d'administration SaaS au préalable en utilisant la fonction [SaaSAdminConnecte](../WDLang3/1000019098.md).

- L'Administrateur SaaS aura accès à tous les services SaaS. Un gestionnaire de compte client aura accès uniquement aux services associés aux sites auxquels il est abonné. 



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
S est un saasSite dynamique 
S = SaaSAdminChercheSite("Forum")

MonService est un saasService dynamique 
MonService = SaaSAdminChercheService("Poster un message", S)
SI ErreurDétectée = Faux ALORS
	Info(" Service trouvé ")
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = SaaSAdminChercheService(<Service> , <Site SaaS>)`
---

**`<Résultat> : (Variable de type saasService)`**

Variable de type [saasService](../WDLang3/1000019096.md) contenant la description du service. La variable renvoyée est vide si la recherche n'a pas aboutie.

**`<Service> : (Chaîne de caractères)`**

Nom du service à rechercher dans le site.

**`<Site SaaS> : (Variable de type saasSite ou chaîne de caractères)`**

Nom de la variable de type [saasSite](../WDLang3/1000019068.md) (ou nom du site) correspondant au site SaaS possédant le service recherché.




## &lt;Graphe&gt;.ComparatifPositionSérie (Fonction)

***En anglais : &lt;Chart&gt;.ComparativeSeriesPosition***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Fixe la position (gauche ou droite) d'une série dans un graphe de type Histogramme comparatif.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
GRF_SansNom1.ComparatifPositionSérie(1, grADroite)
GRF_SansNom1.ComparatifPositionSérie(2, grAGauche)

GRF_SansNom1.CouleurSérie(1, DonneCouleur())
GRF_SansNom1.CouleurSérie(2, DonneCouleur())

rDroite est un réel
rGauche est un réel

nRayon est un entier = 250

POUR n = 1 À 360
	rDroite = nRayon + nRayon * Cosinus(n)
	rGauche = nRayon + nRayon * Sinus(n + 180)
	GRF_SansNom1.AjouteDonnée(1, n, rDroite)
	GRF_SansNom1.AjouteDonnée(2, n, rGauche)
	GRF_SansNom1.EtiquetteCatégorie(n, n)
FIN

GRF_SansNom1.Dessine()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Champ Graphe>.ComparatifPositionSérie(<Numéro de la série> , <Position>)`
---

**`<Champ Graphe> : (Nom de champ)`**

Nom du champ Graphe à manipuler, présent dans l'éditeur de fenêtres.

**`<Numéro de la série> : (Entier)`**

Numéro de la série à modifier.

**`<Position> : (Entier)`**

Position de la série manipulée dans le graphe :


|   |   |
| --- | --- |
| *grADroite* | Série affichée à droite du graphe. |
| *grAGauche* | Série affichée à gauche du graphe. |

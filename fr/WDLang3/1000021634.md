


## grHMSupprimeCouleur (Fonction)

***En anglais : grHMDeleteColor***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime les couleurs spécifiées par la fonction [grHMCouleur](../WDLang3/1000021629.md) dans un graphe de type Heatmap. 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// On fixe les couleurs
grHMSupprimeCouleur(GRF_Graphe)
grHMCouleur(GRF_Graphe, -65, RVB(0, 105, 255))
grHMCouleur(GRF_Graphe, 0, RVB(240, 255, 0))
grHMCouleur(GRF_Graphe, 35, RVB(255, 31, 0))
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`grHMSupprimeCouleur(<Nom du graphe> [, <Palier>])`
---

**`<Nom du graphe> : (Nom de champ ou chaîne de caractères)`**

Nom du graphe à manipuler. Ce nom correspond :

- soit au nom du graphe défini par programmation avec la fonction [grCrée](../WDLang3/3042005.md).

- soit au nom du champ Graphe présent dans l'éditeur de fenêtres ou d'états.




**`<Palier> : (Réel optionnel)`**

Palier de la couleur qui doit être supprimée. Si ce paramètre n'est pas spécifié, toutes les couleurs de palier sont supprimées et la couleur de la première série est utilisée.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques

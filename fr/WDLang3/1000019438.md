
## Constantes de gestion des fonctions Wi-Fi
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


| Constante | Détail | Utilisée par les fonctions |
| --- | --- | --- |
| **wifiActif** | Le Wi-Fi est activé sur l'appareil. | [WiFiActive](../WDLang3/1000019350.md), [WiFiEtat](../WDLang3/1000019354.md) |
| **wifiActivationEnCours** | Le Wi-Fi est en cours d'activation sur l'appareil. | [WiFiActive](../WDLang3/1000019350.md), [WiFiEtat](../WDLang3/1000019354.md) |
| **wifiAdresseIP** | Adresse IP associée à la connexion. | [WiFiInfoConnexion](../WDLang3/1000019355.md) |
| **wifiAdresseMac** | Adresse Mac Wi-Fi de l'appareil. | [WiFiInfoConnexion](../WDLang3/1000019355.md) |
| **wifiBSSID** | BSSID (Basic Service Set Identifier) du point d'accès de la connexion. | [WiFiInfoConnexion](../WDLang3/1000019355.md) |
| **wifiDésactivationEnCours** | Le Wi-Fi est en cours de désactivation sur l'appareil. | [WiFiActive](../WDLang3/1000019350.md), [WiFiEtat](../WDLang3/1000019354.md) |
| **wifiDésactivé** | Le Wi-Fi est désactivé sur l'appareil. | [WiFiActive](../WDLang3/1000019350.md), [WiFiEtat](../WDLang3/1000019354.md) |
| **wifiErreur** | L'état d'activation du Wi-Fi n'a pas pu être récupéré. | [WiFiActive](../WDLang3/1000019350.md), [WiFiEtat](../WDLang3/1000019354.md) |
| **wifiIDRéseau** | Identifiant du réseau auquel l'appareil est connecté, chaîne vide ("") si l'appareil n'est connecté à aucun réseau connu. | [WiFiInfoConnexion](../WDLang3/1000019355.md) |
| **wifiSécuritéAucune** | Aucune sécurité n'est utilisée. | [WiFiAjouteRéseau](../WDLang3/1000019351.md), [WiFiDétectePointAccès](../WDLang3/1000019353.md) |
| **wifiSécuritéIEEE8021X** | Sécurité utilisant la norme IEEE 802.1x. | [WiFiAjouteRéseau](../WDLang3/1000019351.md), [WiFiDétectePointAccès](../WDLang3/1000019353.md), [WiFiListeRéseau](../WDLang3/1000019356.md) |
| **wifiSécuritéWEP** | Sécurité basée sur une clé WEP (Wired Equivalent Privacy) | [WiFiAjouteRéseau](../WDLang3/1000019351.md), [WiFiDétectePointAccès](../WDLang3/1000019353.md), [WiFiListeRéseau](../WDLang3/1000019356.md) |
| **wifiSécuritéWPA** | Sécurité utilisant le mécanisme WPA (Wi-Fi Protected Access). | [WiFiAjouteRéseau](../WDLang3/1000019351.md), [WiFiDétectePointAccès](../WDLang3/1000019353.md), [WiFiListeRéseau](../WDLang3/1000019356.md) |
| **wifiSécuritéWPA2** | Sécurité utilisant le mécanisme WPA2 (Wi-Fi Protected Access). | [WiFiAjouteRéseau](../WDLang3/1000019351.md), [WiFiDétectePointAccès](../WDLang3/1000019353.md), [WiFiListeRéseau](../WDLang3/1000019356.md) |
| **wifiSécuritéWPA_EAP** | Sécurité utilisant les mécanismes WPA (Wi-Fi Protected Access) et EAP (Extensible Authentification Protocol). | [WiFiAjouteRéseau](../WDLang3/1000019351.md), [WiFiDétectePointAccès](../WDLang3/1000019353.md), [WiFiListeRéseau](../WDLang3/1000019356.md) |
| **wifiSSID** | SSID (Service Set Identifier) du réseau Wi-Fi auquel l'appareil est connecté. | [WiFiInfoConnexion](../WDLang3/1000019355.md) |
| **wifiTout** | Informations complètes sur la connexion (correspond à la combinaison de constantes *wifiSSID* + *wifiBSSID* + *wifiAdresseIP* + *wifiAdresseMac* + *wifiVitesse* + *wifiIDRéseau*). | [WiFiInfoConnexion](../WDLang3/1000019355.md) |
| **wifiVitesse** | Vitesse de la liaison Wi-Fi (exprimée en Mbps). | [WiFiInfoConnexion](../WDLang3/1000019355.md) |

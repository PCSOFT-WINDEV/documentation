
## MarqueurImage (Type de variable)

***En anglais : MarkerImage***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **MarqueurImage** permet personnaliser l'image associée à un marqueur ou à un cluster de marqueurs affiché dans le champ Carte. Les caractéristiques de cette image peuvent être définies et modifiées à l'aide de différentes propriétés WLangage. 

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonImg est un MarqueurImage
MonImg.Taille = 60
MonImg.CouleurFond = VertClair
MonImg.Texte = 100

MonMarqueur est un Marqueur
MonMarqueur.Position = MaPosition
MonMarqueur.Image = MonImg

CarteAjouteMarqueur(CARTE_Carte1, MonMarqueur)
```

<a name="NOTE0"></a>

## Remarques
<a name="NOTE0_1"></a>


### Propriétés spécifiques à la description des variables de type MarqueurImage
<a name="proprietes_specifiques_description_des_variables_type_marqueurimage_ELTPARAGRAPHE000028"></a>Les propriétés suivantes peuvent être utilisées pour manipuler l'image d'un marqueur :

| Nom de la propriété | Type manipulé | Effet |
| --- | --- | --- |
| Couleur | [Couleur](../WDLang1/1000019661.md), entier, constante | Couleur du contour de la forme et du libellé du marqueur. Cette couleur peut correspondre à :<br><br>- une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)),<br><br>- [une couleur prédéfinie du WLangage](../WDLang5/3010002.md),<br><br>- une variable [Couleur](../WDLang1/1000019661.md). <br><br><br>Couleur par défaut : noir. |
| CouleurFond | [Couleur](../WDLang1/1000019661.md), entier, constante | Contours du cercle. Cette couleur peut correspondre à :<br><br>- une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)),<br><br>- [une couleur prédéfinie du WLangage](../WDLang5/3010002.md),<br><br>- une variable [Couleur](../WDLang1/1000019661.md). <br><br><br>Remarque : Pour modifier l'opacité du remplissage, il est possible d'écrire :<br><br><pre><code>MonMarqueurImage.CouleurFond.Opacité = 50</code></pre><br>Couleur par défaut : noir. |
| Forme | Constante de type Entier | Forme à représenter parmi les constantes suivantes : <br><br>- *formeCarré* : Forme carrée. <br><br>- *formeCarréArrondi* : Forme carrée à coin arrondi.<br><br>- *formeRond* (valeur par défaut) : Forme ronde.<br><br><br> |
| Taille | Entier | Taille de la forme (rayon pour un cercle, côté pour un carré) exprimée en pixels. |
| Texte | Chaîne de caractères | Libellé à afficher dans la forme. |

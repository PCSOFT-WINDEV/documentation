
## AdresseSIP (Type de variable)

***En anglais : SIPAddress***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **AdresseSIP** permet de définir toutes les caractéristiques avancées d'une adresse SIP. SIP (Session Initiation Protocol) est un protocole standardisé utilisé pour établir des communications multimédia (son et images). Il est notamment utilisé pour la VoIP (voix sur IP).

Les caractéristiques de cette adresse SIP peuvent être définies et modifiées à l'aide de différentes propriétés WLangage. 

Ce type de variable est utilisé par les variables de type [Contact](../WDLang3/1000020254.md).

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).

<a name="NOTE0_1"></a>

## Remarques


### Propriétés spécifiques à la description des variables de type AdresseSIP
<a name="proprietes_specifiques_description_des_variables_type_adressesip_ELTPARAGRAPHE000038"></a>Les propriétés suivantes peuvent être utilisées pour manipuler une adresse SIP :

| **Nom de la propriété** | **Type manipulé** | **Effet** |
| --- | --- | --- |
| Adresse | Chaîne de caractères | Adresse SIP. |
| Etiquette | Chaîne de caractères | Etiquette pour les types d'adresses SIP personnalisées (optionnel).<br>Si la propriété **Type** ne correspond pas à la constante *sipPersonnalisé*, la valeur de cette propriété sera ignorée. |
| Type | Constante de type Entier | Type de l'adresse SIP : <br><br>- *sipAutre* : autres types.<br><br>- *sipBureau* : adresse SIP du bureau.<br><br>- *sipDomicile* : adresse SIP du domicile.<br><br>- *sipPersonnalisé* : type personnalisé correspondant à la valeur de la propriété **Etiquette**.<br><br><br> |


<a name="NOTE0_2"></a>


### Réinitialisation
<a name="reinitialisation_ELTPARAGRAPHE000077"></a>Pour réinitialiser le contenu d'une variable de type **AdresseSIP**, il est possible d'utiliser la fonction [VariableRAZ](../WDLang1/3013069.md).


## Constantes iCloud
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les constantes iCloud sont les suivantes :

| Constante | Détail | Fonction |
| --- | --- | --- |
| **iccvChangementCompte** | Cause de l'événement : L'utilisateur a changé de compte. | [iCloudCléValeurSurveille](../WDLang3/1410088760.md) |
| **iccvEchecSynchronisation** | Cause de l'événement : Erreur lors de la tentative de synchronisation entre le périphérique et le serveur Apple iCloud. | [iCloudCléValeurSurveille](../WDLang3/1410088760.md) |
| **iccvModificationServeur** | Cause de l'événement : Les clés ont été modifiées depuis le serveur Apple iCloud. | [iCloudCléValeurSurveille](../WDLang3/1410088760.md) |
| **iccvModificationSynchronisationInitiale** | Cause de l'événement : Synchronisation initiale avec le serveur Apple iCloud. | [iCloudCléValeurSurveille](../WDLang3/1410088760.md) |
| **iccvViolationQuota** | Cause de l'événement : Le quota concernant les clés a été dépassé (Nombre de clés, espace de stockage, etc.). | [iCloudCléValeurSurveille](../WDLang3/1410088760.md) |


## géoSuiviEtat (Fonction)

***En anglais : geoTrackingStatus***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Récupère l'état du système de suivi de position.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
Si géoSuiviEtat() = géoSuiviDésactivé ALORS
	RETOUR
FIN

// traitement dépendant du suivi de position actif
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = géoSuiviEtat()`
---

**`<Résultat> : (Constante de type Entier)`**

Etat du système de suivi : 



|   |   |
| --- | --- |
| *géoSuiviActivé* | Le système de suivi de position est activé. |
| *géoSuiviDésactivé*  | Le système de suivi de position est désactivé. |
| *géoSuiviErreur* | Erreur pendant la récupération de l'état d'activation du système. Utilisez la fonction [ErreurInfo](../WDLang1/3013008.md) pour obtenir le détail de l'erreur. |

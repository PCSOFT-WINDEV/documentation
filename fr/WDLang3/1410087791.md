


## grCPSensDeRotation (Fonction)

***En anglais : grRBRotationDirection***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Fixe le sens de rotation d'un graphe circulaire de progrès. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
grCPSensDeRotation(GRF_Graphe1, grSensAntihoraire)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`grCPSensDeRotation(<Nom du graphe> , <Rotation>)`
---

**`<Nom du graphe> : (Nom de champ ou chaîne de caractères)`**

Nom du graphe à manipuler. Ce nom correspond :

- soit au nom du graphe défini par programmation avec la fonction [grCrée](../WDLang3/3042005.md).

- soit au nom du champ Graphe présent dans l'éditeur de fenêtres ou d'états.




**`<Rotation> : (Constante de type Entier)`**

Sens de rotation du graphe : 


|   |   |
| --- | --- |
| *grSensAntihoraire* | Les données du graphe circulaire seront affichées en suivant le sens inverse de rotation des aiguilles d'une montre. |
| *grSensHoraire* | Les données du graphe circulaire seront affichées en suivant le sens de rotation des aiguilles d'une montre. |




## ContactModifie (Fonction)

***En anglais : ContactModify***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie le contact en cours dans le carnet d'adresses d'une messagerie Lotus Notes ou Outlook.
**Attention** : Cette fonction **ne permet pas** de lire le dernier contact présent dans le carnet d'adresses d'une messagerie Outlook Express.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ouverture d'une session Lotus Notes
IDSession est un entier
IDSession = EmailOuvreSessionNotes("MotDePasse", "ServeurMars", ...
	"mail/julie.nsf", "C:\Lotus Notes\julie.id")
// Recherche du contact à modifier
SI ContactCherche(IDSession, "Dupont", mContactNom, "Axelle", mContactPrénom) = Vrai ALORS
	// Modification du numéro de téléphone de ce contact
	mContact.TéléphoneMobile = "06.15.14.84.52"
	ContactModifie(IDSession)
	Info("Le numéro de téléphone a été modifié")
SINON
	Info("Aucun contact n'a été trouvé")
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = ContactModifie(<Identifiant de la session>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la modification a été effectuée. 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour connaître le libellé de l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.




**`<Identifiant de la session> : (Entier)`**

Identifiant de la session à utiliser. Cet identifiant est renvoyé par une des fonctions suivantes : [EmailOuvreSessionNotes](../WDLang3/3032121.md), [EmailOuvreSessionOutlook](../WDLang3/3032128.md), [NotesOuvreBase](../WDLang3/1000005101.md) ou [OutlookOuvreSession](../WDLang3/3032123.md).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Principe de la modification en utilisant la structure Contact
<a name="principe_modification_utilisant_structure_contact_ELTPARAGRAPHE000056"></a>Lors de la lecture d'un contact (à l'aide des fonctions [ContactLit](../WDLang3/1000006112.md), [ContactPremier](../WDLang3/1000006106.md), [ContactDernier](../WDLang3/1000006103.md), ...), les données de ce contact sont mises à jour dans la [structure Contact](../WDLang3/1000006111.md). 

Pour modifier le contact en cours, il suffit de : 

1. Modifier les valeurs de la [structure Contact](../WDLang3/1000006111.md). 

2. Valider les modifications avec la fonction **ContactModifie**.

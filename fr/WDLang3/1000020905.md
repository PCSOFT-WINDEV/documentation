
## upnpImage (Type de variable)

***En anglais : upnpImage***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **upnpImage** permet de définir toutes les caractéristiques avancées d'une image d'un périphérique UPnP. Les caractéristiques de cette image UPnP peuvent être définies et modifiées à l'aide de différentes propriétés WLangage. 

Ce type de variable est utilisé par les variables de type [upnpPériphérique](../WDLang3/1000020903.md). 

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).
<a name="NOTE0"></a>

## Remarques
<a name="NOTE0_1"></a>


### Propriétés spécifiques à la description des variables de type upnpImage
<a name="proprietes_specifiques_description_des_variables_type_upnpimage_ELTPARAGRAPHE000028"></a>Les propriétés suivantes peuvent être utilisées pour manipuler une image d'un périphérique UPnP :

| **Nom de la propriété** | **Type manipulé** | **Effet** |
| --- | --- | --- |
| BitParPixel | Entier | Nombre de bits par pixels. |
| Hauteur | Entier | Hauteur de l'image en pixels. |
| Image | Variable de type [Image](../WDLang1/1000019650.md) | Image |
| Largeur | Entier | Largeur de l'image en pixels. |
| TypeMIME | Chaîne de caractères | Type MIME de l'image. |
| URL | Chaîne de caractères | URL de l'image. |

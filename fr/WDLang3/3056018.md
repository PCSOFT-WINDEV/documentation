
## NetSupprimeFichier (Fonction)

***En anglais : NetEraseFile***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime un fichier présent sur le serveur FTP (File Transfer Protocol) ou RPC (Remote Procedure Call) réalisé avec WINDEV.

**Remarque** : Il est possible d'autoriser ou non l'utilisation de cette fonction sur un serveur FTP/RPC grâce à la fonction [NetOptionServeur](../WDLang3/3056022.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
IDServeurFTP est un entier
Res est un entier
IDServeurFTP = NetConnecte("148.61.125.245", ServeurFTP, "GUEST", "")
// Supprimer un fichier sur le serveur
Res = NetSupprimeFichier(IDServeurFTP, "C:\HISTO\ANNEE98.TXT")
SI Res <> 0 ALORS
	TraiteErreur() // Procédure de traitement de l'erreur
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = NetSupprimeFichier(<Identifiant connexion> , <Nom du fichier>)`
---

**`<Résultat> : (Entier)`**

Résultat de la suppression :

- 0 : Le fichier présent sur le serveur a été supprimé

- 2 : Le fichier à supprimer n'existe pas

- 5 : Accès refusé

- 19 : Disquette protégée en écriture




**`<Identifiant connexion> : (Entier)`**

Identifiant de la connexion du poste client (obtenu par la fonction [NetConnecte](../WDLang3/3056004.md)).

**`<Nom du fichier> : (Chaîne de caractères Ansi)`**

Nom et chemin d'accès complet du fichier à supprimer sur le serveur.

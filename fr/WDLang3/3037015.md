
## FaxSuspend (Fonction)

***En anglais : FaxPause***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Met en pause un fax dans la file d'attente. Si le fax est actif, il sera mis en pause lorsqu'il retournera dans la file d'attente.

**Remarque** : Cette fonction ne peut être utilisée que sous Windows 2000, XP ou version supérieure. Sous Windows Vista (et supérieur), la gestion des fax est disponible uniquement avec Windows Professionnel et Windows Edition Intégrale.
<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = FaxSuspend(<Identifiant connexion> , <Nom du fax ou Identifiant du fax>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le fax est mis en pause, 

- <u><u><u><u>Faux</u></u></u></u> en cas d'erreur (par exemple si le nom du fax n'existe pas).




**`<Identifiant connexion> : (Entier)`**

Identifiant de la connexion obtenu avec la fonction [FaxConnecte](../WDLang3/3037005.md).

**`<Nom du fax ou Identifiant du fax> : (Chaîne de caractères)`**

Correspond :

- soit au nom permettant d'identifier le fax envoyé. Ce nom a été donné avec la fonction [FaxEnvoie](../WDLang3/3037006.md).

- soit à l'identifiant du fax (obtenu par les fonctions [FaxBoîteEnvoi](../WDLang3/3037008.md) ou [FaxBoîteRéception](../WDLang3/3037009.md))

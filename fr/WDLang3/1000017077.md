
## SNMPOIDAccès (Fonction)

***En anglais : SNMPOIDAccess***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie les modes d'accès autorisés d'un OID spécifique.

**Remarque** : Cette fonction ne peut être utilisée qu'après avoir chargé le fichier MIB (fonction [SNMPChargeMIB](../WDLang3/1000017074.md)) qui décrit la correspondance entre les OID au format texte et numérique.  
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
bMIBChargé est un booléen
sCheminMIBs est une chaîne = ComplèteRep(fRepExe()) + "mibs\"
sMib est une chaîne
sMib = "RFC1382-MIB.mib" // MIB du protocole X25
sAccès est un entier
bMIBChargé = SNMPChargeMIB(sCheminMIBs + sMib)
SI bMIBChargé = Faux ALORS 
	Erreur("Erreur MIB :" + ErreurInfo())
SINON
	sAccès = SNMPOIDAccès("dod.internet.mgmt.mib-2.transmission.x25.x25AdmnInterruptTimer")
	SELON sAccès
		CAS snmpAccèsAucun : Info("OID inaccessible")
		CAS snmpAccèsEcritureSeule : Info("OID en écriture")
		CAS snmpAccèsLectureEcriture : Info("OID en lecture / écriture")
		CAS snmpAccèsLectureSeule : Info("OID en lecture")
		AUTRES CAS : Erreur(ErreurInfo())
	FIN
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = SNMPOIDAccès(<OID à traiter>)`
---

**`<Résultat> : (Constante de type Entier)`**

Constante d'accès correspondant au mode d'accès décrit dans le fichier MIB :


|   |   |
| --- | --- |
| *snmpAccèsAucun* | L'OID n'est pas accessible. |
| *snmpAccèsEcritureSeule* | L'OID est accessible en écriture seulement. |
| *snmpAccèsInconnu* | Erreur : la fonction [ErreurInfo](../WDLang1/3013008.md) permet d'obtenir plus de renseignements sur le problème rencontré. |
| *snmpAccèsLectureEcriture* | L'OID est accessible en lecture et en écriture. |
| *snmpAccèsLectureSeule* | L'OID est accessible en lecture seulement. |



**`<OID à traiter> : (Chaîne de caractères)`**

Chaîne contenant l'OID sous forme de texte ou de numérique.

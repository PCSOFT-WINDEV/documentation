
## J2EEErreur (Fonction)

***En anglais : J2EEError***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie l'erreur de la dernière procédure J2EE ayant échoué.

**Remarque** : Cette procédure a été exécutée par la fonction [J2EEExécute](../WDLang3/3047007.md) ou [J2EEExécuteXML](../WDLang3/3047008.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Renvoie le message d'erreur de la dernière procédure J2EE exécutée
ResMessageErreur = J2EEErreur(J2EEErrMessage)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = J2EEErreur(<Type d'erreur>)`
---

**`<Résultat> : (Chaîne de caractères)`**



- Erreur de la dernière procédure J2EE exécutée ayant échoué.

- Une chaîne vide ("") si :

	- aucune erreur n'a été retournée par le serveur,

	- aucune connexion n'a été établie avec le serveur.







**`<Type d'erreur> : (Constante)`**

Type d'erreur à retourner :


|   |   |
| --- | --- |
| *J2EEErrActeur* | URL du serveur ayant provoqué l'erreur (utile si la procédure est exécutée par plusieurs serveurs). |
| *J2EEErrCode* | Erreur de la dernière procédure J2EE exécutée :<br><br>- "VersionMismatch" : La version J2EE utilisée sur le poste en cours est différente de celle utilisée sur le serveur.<br><br>- "MustUnderstand" : Le serveur ne comprend pas le paramètre spécifié dans l'entête (uniquement si le paramètre &lt;Exécution de la procédure&gt; de l'entête est initialisé à <u><u><u><u>Vrai</u></u></u></u>).<br><br>- "Client" : Mauvais paramétrage au niveau de l'exécution de la procédure.<br><br>- "Server" : Le serveur n'a pas exécuté la procédure.<br><br><br>**Remarque** : Cette chaîne de caractères peut être complétée d'informations supplémentaires. Le format utilisé est "&lt;Erreur&gt;.&lt;Information&gt;". Par exemple : "Client.Authentication" si l'erreur est due à l'authentification du client. |
| *J2EEErrDétail* | Détail de l'erreur. Ce message est au format XML. |
| *J2EEErrMessage* | Message d'erreur renvoyé par le serveur sur la dernière procédure J2EE exécutée. |

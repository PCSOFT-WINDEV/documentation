
## FichierVersURI (Fonction)

***En anglais : FileToURI***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Sauvegarde le contenu d'un fichier dans une ressource identifiée par son URI.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ouvre le sélecteur de ressources en mode création
uriDocImage est une URI = URISélecteur(uriImage, uriCréation)
SI PAS ErreurDétectée ALORS
	// Sauvegarde de l'image dans le document créé 
	FichierVersURI(image, uriDocImage)
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = FichierVersURI(<Fichier à sauvegarder> , <URI destination> [, <Mode>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la sauvegarde a été effectuée, 

- <u><u><u><u>faux</u></u></u></u> dans le cas contraire. 


En cas d'erreur, la variable [ErreurDétectée](../WDLang1/3087001.md) est à <u><u><u><u>Vrai</u></u></u></u>. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).

**`<Fichier à sauvegarder> : (Chaîne de caractères)`**

Chemin du fichier à sauvegarder. Ce fichier doit exister.

**`<URI destination> : (Chaîne de caractères ou variable de type URI)`**

Ressource dans laquelle le fichier doit être sauvegardé. Cette ressource peut être identifiée : 

- par une chaîne de caractères correspondant à une URI, 

- par une variable de type [URI](../WDLang3/1000024027.md). 


Remarque : L'URI spécifiée doit être de type "file" (fichier externe) ou "content" (ressource provenant d'un content provider) et être accessible en écriture.

**`<Mode> : (Constante optionnelle de type Entier)`**

Mode de sauvegarde : 



|   |   |
| --- | --- |
| *uriAjout* | Le contenu du fichier est ajouté à la fin du contenu initial de la ressource. |
| *uriCréation*<br>(Valeur par défaut) | Le contenu initial de la ressource est remplacé par celui du fichier. <br> |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Pour sélectionner une ressource et obtenir son URI, utilisez la fonction [URISélecteur](../WDLang3/1000024026.md).


## SaaSAdminSupprimeSite (Fonction)

***En anglais : SaaSAdminDeleteSite***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime un site SaaS.

**Remarques** : 

- L'utilisation de cette fonction nécessite d'avoir ouvert une session d'administration SaaS au préalable en utilisant la fonction [SaaSAdminConnecte](../WDLang3/1000019098.md).

- L'utilisation de cette fonction est réservée à l'Administrateur SaaS. Un gestionnaire de compte client ne peut pas utiliser cette fonction. 



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Suppression du site SaaS "Forum"
S est un saasSite 
S = SaaSAdminChercheSite("Forum")

SI SaaSAdminSupprimeSite(S) = Vrai ALORS
	Info("Site supprimé avec succès.")
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Supprimer un site SaaS (utilisation d'une variable de type saasSite)

`<Résultat> = SaaSAdminSupprimeSite(<Site SaaS>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la suppression a été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Site SaaS> : (Variable de type saasSite)`**

Nom de la variable de type [saasSite](../WDLang3/1000019068.md) correspondant au site SaaS à supprimer de la base SaaS.


<a name="SYNTAXE2"></a>

### Supprimer un site SaaS identifié par son nom

`<Résultat> = SaaSAdminSupprimeSite(<Site SaaS>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la suppression a été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Site SaaS> : (Chaîne de caractères)`**

Nom du site SaaS à supprimer de la base SaaS.

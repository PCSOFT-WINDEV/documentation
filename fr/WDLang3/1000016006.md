
## NetAdresseMAC (Fonction)

***En anglais : NetMACAddress***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie l'adresse MAC (adresse physique) de l'une des cartes réseau de la machine.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Renvoie l'adresse MAC du poste
Info(NetAdresseMAC())
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = NetAdresseMAC([<Rang carte réseau>])`
---

**`<Résultat> : (Chaîne de caractères)`**



- Adresse MAC obtenue (au format "aa:bb:cc:dd:ee:ff"), 

- Chaîne vide ("") en cas de problème.




**`<Rang carte réseau> : (Entier optionnel)`**

Permet de préciser le numéro de la carte réseau pour laquelle l'adresse MAC doit être récupérée (pour les machines disposant de plusieurs cartes réseau). La première carte est au rang 0. 
Si ce paramètre est négatif, l'adresse obtenue sera celle de la première carte réseau.

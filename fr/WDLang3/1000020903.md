
## upnpPériphérique (Type de variable)

***En anglais : upnpDevice***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **upnpPériphérique** permet de définir toutes les caractéristiques avancées d'un périphérique UPnP. Les caractéristiques de ce périphérique uPnP peuvent être définies et modifiées à l'aide de différentes propriétés WLangage. 

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonPériph est une upnpPériphérique 
MonPériph = UPNPCherchePériphérique(CST_UPNP_UUID_SONY_BRAVIA, 51300)
SI ErreurDétectée ALORS
	ErreurInfo(errComplet)
SINON
	Ajoute(m_tabPeripherique, MonPériph)
FIN
```

### Propriétés spécifiques à la description des variables de type upnpPériphérique
<a name="proprietes_specifiques_description_des_variables_type_docparampagetitleshort_ELTPARAGRAPHE000029"></a>

Les propriétés suivantes peuvent être utilisées pour manipuler un périphérique UPnP :

| Nom de la propriété | Type manipulé | Effet |
| --- | --- | --- |
| CodeProduit | Chaîne de caractères | Code produit (UPC). |
| DescriptionModèle | Chaîne de caractères | Description des fonctionnalités du périphérique. |
| Icone | Tableau de [upnpImage](../WDLang3/1000020905.md) | Liste des différentes icônes disponibles. |
| Nom | Chaîne de caractères | Nom du périphérique. |
| NomFabricant | Chaîne de caractères | Nom du fabricant. |
| NomModèle | Chaîne de caractères | Nom du modèle. |
| NomUnique | Chaîne de caractères | Nom unique du périphérique (UDN). |
| NuméroModèle | Chaîne de caractères | Numéro du modèle. |
| NuméroSérie | Chaîne de caractères | Numéro de série du périphérique. |
| Service | Tableau de [upnpService](../WDLang3/1000020908.md) | Liste des différents services UPnP disponibles. |
| Type | Chaîne de caractères | URI contenant le type du périphérique. |
| URL | Chaîne de caractères | URL contenant le XML de description du périphérique. |
| URLFabricant | Chaîne de caractères | Lien vers le site du fabricant. |
| URLModèle | Chaîne de caractères | URL d'une page Web contenant les informations spécifiques au modèle. |
| URLPrésentation | Chaîne de caractères | URL de la page Web qui peut être utilisée pour contrôler l'appareil. |

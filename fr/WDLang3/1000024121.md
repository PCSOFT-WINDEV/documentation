


## &lt;Graphe&gt;.Quadrillage (Fonction)

***En anglais : &lt;Chart&gt;.GridLines***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Affiche ou rend invisible le quadrillage d'un graphe.

Exemple de quadrillage : ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=quadrillage.gif)


**Rappel** : Par défaut, le quadrillage d'un graphe est invisible.



![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Les graphes](../WDChamp/3042060.md). 












<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affichage du quadrillage horizontal du graphe GRF_MonGraphe
GRF_MonGraphe.Quadrillage(Vrai, grAbscisse)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Champ Graphe>.Quadrillage(<Affiche> , <Sens de l'axe>)`
---

**`<Champ Graphe> : (Nom de champ)`**

Nom du champ Graphe à manipuler présent dans l'éditeur de fenêtres, de pages ou d'états.


**`<Affiche> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le quadrillage doit être affiché, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Sens de l'axe> : (Constante ou combinaison de constantes de type Entier)`**

Axe utilisé pour le quadrillage : 



|   |   |
| --- | --- |
| *grAbscisse*<br>(valeur par défaut) | Graduations sur l'axe des abscisses (axe horizontal). |
| *grOrdonnée* | Graduations sur l'axe des ordonnées (axe vertical). |


La combinaison de ces deux constantes permet d'obtenir un quadrillage complet (vertical et horizontal).



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Le quadrillage est pris en compte :

	- dans les graphes de type histogramme (constantes *grHisto* et *grHistoEmpilé*), nuage de points (constante *grNuage*), courbe (constante *grCourbe*) et boursier (constantes *grChandelier*, *grBarCharts* et *grMinMax*) lorsque le paramètre &lt;Sens de l'axe&gt; correspond à la constante *grAbscisse*.

	- dans les graphes de type nuage de points (constante *grNuage*), courbe (constante *grCourbe*) et boursier (constantes *grChandelier* et *grBarCharts*) lorsque le paramètre &lt;Sens de l'axe&gt; correspond à la constante *grOrdonnée*.




- Le quadrillage est sans effet sur les graphes de type secteur (constante *grSecteur*) ou entonnoir (constante *grEntonnoir*).

- Les données définies avec la fonction **&lt;Graphe&gt;.Quadrillage** seront prises en compte lors du prochain appel à la fonction [&lt;Graphe&gt;.Dessine](../WDLang3/1000024062.md).




## &lt;Carte&gt;.ChangeStyle (Fonction)

***En anglais : &lt;Map&gt;.ChangeStyle***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie le style de la carte affichée dans le champ Carte.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Applique un style "Aubergine" à la carte
CARTE_MaCarte.ChangeStyle(ccsAubergine)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Champ Carte>.ChangeStyle(<Style> [, <Chemin du fichier de configuration>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le nouveau style a pu être appliqué, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Champ Carte> : (Nom de champ)`**

Nom du champ Carte à manipuler.

**`<Style> : (Entier)`**

Modifie le style appliqué à la carte affichée par le champ Carte. Ce paramètre peut prendre l'une des valeurs suivantes :


|   |   |
| --- | --- |
| *ccsArgenté* | Style "argenté". |
| *ccsAubergine* | Style "aubergine". |
| *ccsAutomatique* | Le style sera appliqué en fonction du mode clair ou sombre de l'application :<br><br>- Thème clair : le style utilisé sera le style "standard",<br><br>- Thème sombre : le style utilisé sera le style "nuit".<br><br><br>Si le thème de l'application change alors que le champ Carte est affiché, le style sera automatiquement modifié. |
| *ccsNuit* | Style "nuit". |
| *ccsPersonnalisé* | Style personnalisé. dans ce cas, le paramètre &lt;Chemin du fichier de configuration&gt; doit être spécifié. |
| *ccsRétro* | Style "retro". |
| *ccsSombre* | Style "sombre". |
| *ccsStandard* | style "standard". |



**`<Chemin du fichier de configuration> : (Chaîne de caractères optionnelle)`**

Chemin du fichier JSON décrivant le style personnalisé qui doit être appliqué à la carte. Ce fichier doit être un fichier JSON valide encodé en UTF-8. Pour plus d'information sur la configuration personnalisé du style, consultez : [https://mapstyle.withgoogle.com/](https://mapstyle.withgoogle.com/)

Ce paramètre sera pris en compte uniquement si le paramètre &lt;Style&gt; correspond à la constante *ccsPersonnalisé*.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Par défaut le champ Carte utilise le style automatique.

- Pour récupérer le style utilisé par la carte affichée depuis le champ Carte, utilisez la fonction [&lt;Carte&gt;.RécupèreStyle](../WDLang3/1000025798.md).

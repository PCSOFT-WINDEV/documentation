


## telAppelDébut (Fonction)

***En anglais : tapiCallStart***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la date et l'heure du début de l'appel (entrant ou sortant).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
DateHeureAppel est une DateHeure
DateHeureAppel = telAppelDébut()
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = telAppelDébut([<Identifiant de l'appel>])`
---

**`<Résultat> : (DateHeure)`**

Date et heure du début de l'appel.

**`<Identifiant de l'appel> : (Entier optionnel)`**

Identifiant de l'appel à manipuler. Cet identifiant est connu dans la procédure WLangage appelée par la fonction [telDémarreDétectionAppel](../WDLang3/3076029.md). Si ce paramètre n'est pas spécifié, l'appel en cours (sortant ou entrant) est manipulé.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Disponibilité de l'information
<a name="disponibilite_information_ELTPARAGRAPHE000035"></a>L'information est disponible :

- à n'importe quel moment pour le dernier appel sortant

- jusqu'à la fin de l'appel pour les appels entrants. Si cette information doit être stockée, il est nécessaire de la mémoriser avant la fin de l'appel.



<a name="NOTE0_2"></a>


### Configuration nécessaire
<a name="configuration_necessaire_ELTPARAGRAPHE000047"></a>Les fonctions de téléphonie utilisent la technologie TAPI 2.0 et/ou la technologie TAPI 3.1. Ces technologies sont utilisables sur tous les systèmes.

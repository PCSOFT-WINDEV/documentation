


## Fonctions de gestion de la domotique
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions de gestion du protocole X10 sont les suivantes :



|   |   |
| --- | --- |
| [DomoAllume](../WDLang3/1000003101.md) | Met en marche un ou plusieurs appareils électriques compatibles avec le protocole X10. |
| [DomoEnvoie](../WDLang3/1000003104.md) | Envoie une commande à un ou plusieurs appareils électriques compatibles avec le protocole X10. |
| [DomoEtat](../WDLang3/1000003105.md) | Permet de savoir si un appareil électrique compatible avec le protocole X10 est en marche ou à l'arrêt. |
| [DomoEteint](../WDLang3/1000003103.md) | Arrête un ou plusieurs appareils électriques compatibles avec le protocole X10. |
| [DomoFerme](../WDLang3/1000003106.md) | Ferme la communication ouverte grâce à la fonction [DomoOuvre](../WDLang3/1000003102.md). |
| [DomoOuvre](../WDLang3/1000003102.md) | Permet d'ouvrir une connexion avec une interface X10. |

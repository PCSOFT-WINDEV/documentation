
## RendezVousAjoute (Fonction)

***En anglais : AppointmentAdd***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ajoute un rendez-vous : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) dans l'agenda d'une messagerie Lotus Notes ou Outlook. La structure [RendezVous](../WDLang3/1000011010.md) permet de définir les données de ce contact.
	**Rappel** : Cette fonction ne permet pas d'ajouter un rendez-vous dans l'agenda d'une messagerie Outlook Express.







<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple
<a class="notetitle" target="_blank" href="$DOC$=1000020711&name=rendezvousajoute_fonction&product=WD">Voir des exemples supplémentaires</a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 
```wl
// Ouverture d'une session Lotus Notes
IDSession est un entier
IDSession = EmailOuvreSessionNotes("MotDePasse", "ServeurMars", ...
	"mail/julie.nsf", "C:\Lotus Notes\julie.id")
// Initialisation des variables de la structure RendezVous
RendezVousRAZ()
// Renseignements des variables pour le nouveau rendez-vous
mRendezVous.Objet = "Réunion commerciaux"
mRendezVous.DateDébut = "20070125100000"
mRendezVous.Lieu = "Salle 3B"
mRendezVous.NbParticipant = 3
mRendezVous.Participant[1] = "Romain"
mRendezVous.Participant[2] = "Lucas"
mRendezVous.Participant[3] = "Stéphane"
// Ajout du rendez-vous
SI RendezVousAjoute(IDSession) = Vrai ALORS
	Info("Rendez-vous ajouté")
SINON
	Erreur("Erreur lors de l'ajout", ErreurInfo())
FIN
```

## Syntaxe

### Ajouter un rendez-vous dans l'agenda d'une messagerie Lotus Notes ou Outlook

`<Résultat> = RendezVousAjoute(<Identifiant de la session>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'ajout a été effectué, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Identifiant de la session> : (Entier)`**

Identifiant de la session à utiliser. Cet identifiant est renvoyé par une des fonctions suivantes : [EmailOuvreSessionNotes](../WDLang3/3032121.md), [EmailOuvreSessionOutlook](../WDLang3/3032128.md), [NotesOuvreBase](../WDLang3/1000005101.md) ou [OutlookOuvreSession](../WDLang3/3032123.md).

## Remarques
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) 

### Agenda d'une messagerie Lotus Notes ou Outlook
<a name="agenda_une_messagerie_lotus_notes_outlook_ELTPARAGRAPHE000129"></a>

- Lors de la création d'un rendez-vous avec rappel dans Outlook, le rappel s'affiche immédiatement quelle que soit la date du rendez-vous.

- Les rendez-vous ajoutés sont des rendez-vous de type "meeting" (et non pas des rendez-vous de type "appointment"). Il est possible de modifier le type du rendez-vous en modifiant le champ "AppointmentType" du rendez-vous créé (par exemple avec la fonction [NotesModifieChamp](../WDLang3/1000005113.md)).
	Les différentes valeurs possibles pour les types sont :

	- "0" : appointment

	- "1" : anniversary

	- "2" : event

	- "3" : meeting

	- "4" : reminder


 Exemple :
	
	```wl
	NotesModifieChamp(ID, "AppointmentType", "0")
	```


- Sous Outlook, il est possible de spécifier le mode de gestion des contacts/participants lors de l'ouverture de la session (fonction [EmailOuvreSessionOutlook](../WDLang3/3032128.md) ou fonction [OutlookOuvreSession](../WDLang3/3032123.md)).


## telAppelEstTerminé (Fonction)

***En anglais : tapiCallIsOver***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de savoir si l'appel (sortant ou entrant) est terminé.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// L'appel en cours est-il terminé ?
SI telAppelEstTerminé() = Vrai ALORS
	// Raccrochage de la ligne
	telRaccroche()
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = telAppelEstTerminé([<Identifiant de l'appel>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'appel est terminé, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Identifiant de l'appel> : (Entier optionnel)`**

Identifiant de l'appel à manipuler. Cet identifiant est connu dans la procédure WLangage appelée par la fonction [telDémarreDétectionAppel](../WDLang3/3076029.md). Si ce paramètre n'est pas spécifié, l'appel en cours (sortant ou entrant) est manipulé.

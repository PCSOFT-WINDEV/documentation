
## WiFiConnecte (Fonction)

***En anglais : WiFiConnect***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Connecte l'appareil au réseau Wi-Fi spécifié.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI WiFiConnecte("14758") = Vrai ALORS
	Info("Connexion Wi-Fi établie.")
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = WiFiConnecte(<Identifiant> [, <Mot de passe>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la connexion a réussi, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. La fonction [ErreurInfo](../WDLang1/3013008.md) permet d'obtenir plus de renseignements sur le problème rencontré.




**`<Identifiant> : (Chaîne de caractères)`**

Identifiant du réseau auquel on souhaite se connecter. Cet identifiant peut être : 

- l'identifiant renvoyé par la fonction [WiFiAjouteRéseau](../WDLang3/1000019351.md) (cas d'un réseau qui vient d'être ajouté). 




**`<Mot de passe> : (Chaîne de caractères optionnelle)`**

Mot de passe utilisé pour se connecter à un réseau sécurisé. Si ce mot de passe n'est pas spécifié, le mot de passe utilisé sera celui enregistré dans la configuration du réseau (si ce mot de passe existe).

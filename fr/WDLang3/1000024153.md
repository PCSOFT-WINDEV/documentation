


## &lt;Graphe&gt;.TitreAxe (Fonction)

***En anglais : &lt;Chart&gt;.AxisTitle***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Définit le titre de l'axe des abscisses et le titre de l'axe des ordonnées d'un graphe.

Exemple de titres des axes d'un graphe : ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=titreaxe.gif)


**Remarque** : Par défaut, les axes d'un graphe n'ont pas de titre.



![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Les graphes](../WDChamp/3042060.md). 












<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Modification du titre de l'axe horizontal du graphe "GRF_MonGraphe"
GRF_MonGraphe.TitreAxe("Temps", grAbscisse)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Champ Graphe>.TitreAxe(<Titre> , <Sens de l'axe>)`
---

**`<Champ Graphe> : (Nom de champ)`**

Nom du champ Graphe présent dans l'éditeur de fenêtres, de pages ou d'états.


**`<Titre> : (Chaîne de caractères)`**

Titre de l'axe spécifié.

**`<Sens de l'axe> : (Constante)`**

Orientation de l'axe à manipuler :


|   |   |
| --- | --- |
| *grAbscisse*<br>(valeur par défaut) | Graduations sur l'axe des abscisses (axe horizontal). |
| *grOrdonnée* | Graduations sur l'axe des ordonnées (axe vertical). |
| *grOrdonnéeSecondaire* | Graduations sur l'axe des ordonnées secondaires (axe vertical). |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Types de graphes
<a name="types_graphes_ELTPARAGRAPHE000289"></a>Selon le type de graphe et le &lt;Sens de l'axe&gt;, la fonction **&lt;Graphe&gt;.TitreAxe** est prise en compte ou non :

| Type de graphe | Axe horizontal | Axe vertical |
| --- | --- | --- |
| Courbe | Fonction **&lt;Graphe&gt;.TitreAxe** prise en compte. | Fonction **&lt;Graphe&gt;.TitreAxe** prise en compte. |
| Graphe Boursier de type Chandelier et BarChart | Fonction **&lt;Graphe&gt;.TitreAxe** prise en compte. | Fonction **&lt;Graphe&gt;.TitreAxe** prise en compte. |
| Histogramme | Fonction **&lt;Graphe&gt;.TitreAxe** prise en compte. | Fonction **&lt;Graphe&gt;.TitreAxe** prise en compte. |
| Graphe Boursier de type Minimum/Maximum | Fonction **&lt;Graphe&gt;.TitreAxe** prise en compte. | Fonction **&lt;Graphe&gt;.TitreAxe** prise en compte. |
| Nuage de points | Fonction **&lt;Graphe&gt;.TitreAxe** prise en compte. | Fonction **&lt;Graphe&gt;.TitreAxe** prise en compte. |
| Secteur | Fonction **&lt;Graphe&gt;.TitreAxe** non prise en compte. | Fonction **&lt;Graphe&gt;.TitreAxe** non prise en compte. |


Les modifications réalisées avec la fonction **&lt;Graphe&gt;.TitreAxe** seront prises en compte lors du prochain appel à la fonction [&lt;Graphe&gt;.Dessine](../WDLang3/1000024062.md).
<a name="NOTE0_2"></a>


### Titre des axes
<a name="titre_des_axes_ELTPARAGRAPHE000392"></a>Le titre des axes du graphe peut être affiché le long des différents axes.
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Titre.gif)
Le contenu du titre des axes du graphe est défini par la fonction **&lt;Graphe&gt;.TitreAxe**. Cette fonction permet également de positionner ce titre selon l'axe voulu. Il n'est pas possible de définir le positionnement et/ou l'orientation du titre par rapport à l'axe.  

La police utilisée pour le titre du graphe peut être modifiée grâce à la fonction [&lt;Graphe&gt;.PoliceTitreAxe](../WDLang3/1000024120.md).
<a name="NOTE0_3"></a>


### Langues utilisant un alphabet non latin
<a name="langues_utilisant_alphabet_non_latin_ELTPARAGRAPHE000414"></a>La fonction **&lt;Graphe&gt;.TitreAxe** utilisera l'alphabet non latin sélectionné uniquement si la configuration du projet associée utilise des chaînes UNICODE en exécution.

Pour vérifier l'option choisie dans la configuration courante : 

1. Sous le volet "Projet", dans le groupe "Configuration de projet", cliquez sur "Configuration courante". 

2. Affichez l'onglet "Unicode" : l'option "Utiliser des chaînes UNICODE en exécution" doit être cochée. 

3. Validez si nécessaire.

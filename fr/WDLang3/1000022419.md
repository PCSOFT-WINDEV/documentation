
## Fonctions de gestion du champ Carte (syntaxe préfixée)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions de gestion des champs Carte via la syntaxe préfixée sont les suivantes : 



|   |   |
| --- | --- |
| [&lt;Carte&gt;.AffichePopup](../WDLang3/1000025799.md) | Ouvre la popup d'un marqueur affiché dans un champ Carte. |
| [&lt;Carte&gt;.AffichePosition](../WDLang3/1000022394.md) | Centre la carte affichée dans un champ Carte sur une position géographique ou un lieu donné. |
| [&lt;Carte&gt;.AfficheZone](../WDLang3/1000025800.md) | Déplace la carte affichée dans le champ Carte pour que la zone géographique spécifiée soit entièrement visible dans le champ. |
| [&lt;Carte&gt;.AjouteForme](../WDLang3/1000025801.md) | Ajoute une nouvelle forme (polygone, ligne, cercle) sur la carte affichée dans un champ Carte. |
| [&lt;Carte&gt;.AjouteImage](../WDLang3/1000025802.md) | Ajoute une nouvelle image sur la carte affichée dans un champ Carte. |
| [&lt;Carte&gt;.AjouteItinéraire](../WDLang3/1000022420.md) | Ajoute un itinéraire sur un champ Carte. |
| [&lt;Carte&gt;.AjouteMarqueur](../WDLang3/1000022422.md) | Ajoute un nouveau marqueur sur une carte affichée dans un champ Carte. |
| [&lt;Carte&gt;.ChangeStyle](../WDLang3/1000025797.md) | Modifie le style de la carte affichée dans le champ Carte. |
| [&lt;Carte&gt;.DistanceItinéraire](../WDLang3/1000022431.md) | Calcule la distance d'un itinéraire dans un champ Carte. |
| [&lt;Carte&gt;.ExécuteJS](../WDLang3/1000022384.md) | Exécute du code natif de l'API Google Maps dans un champ Carte en Javascript. |
| [&lt;Carte&gt;.FermePopup](../WDLang3/1000025803.md) | Ferme la popup d'un marqueur affichée dans le champ Carte. |
| [&lt;Carte&gt;.FinDéplacement](../WDLang3/1000022437.md) | Arrête le suivi du déplacement de l'appareil dans un champ Carte. |
| [&lt;Carte&gt;.InfoPosition](../WDLang3/1000022447.md) | Renvoie les coordonnées en pixel du point d'un champ Carte correspondant à une position géographique (latitude et longitude). |
| [&lt;Carte&gt;.InfoXY](../WDLang3/1000022448.md) | Renvoie la position géographique (latitude et longitude) correspondant à un point d'un champ Carte. |
| [&lt;Carte&gt;.LimiteZone](../WDLang3/1000025804.md) | Limite la zone géographique affichée dans le champ Carte. |
| [&lt;Carte&gt;.ModifieForme](../WDLang3/1000025805.md) | Modifie une forme dessinée sur la carte affichée dans un champ Carte. |
| [&lt;Carte&gt;.ModifieImage](../WDLang3/1000025818.md) | Modifie une image dessinée sur la carte affichée dans un champ Carte. |
| [&lt;Carte&gt;.ModifieMarqueur](../WDLang3/1000022383.md) | Modifie un marqueur affiché dans un champ Carte. |
| [&lt;Carte&gt;.PopupAffichée](../WDLang3/1000025819.md) | Permet de savoir si la popup d'un marqueur du champ Carte est affichée. |
| [&lt;Carte&gt;.RécupèreImage](../WDLang3/1000022462.md) | Récupère l'image d'une carte affichée dans un champ Carte. Les marqueurs et les itinéraires affichés sur cette carte seront également présents dans l'image. |
| [&lt;Carte&gt;.RécupèreObjetJS](../WDLang3/1000022385.md) | Retourne l'instance de l'API d'un champ Carte. |
| [&lt;Carte&gt;.RécupèrePosition](../WDLang3/1000022463.md) | Renvoie la position géographique du point situé au centre de la carte actuellement affiché dans un champ Carte. |
| [&lt;Carte&gt;.RécupèreStyle](../WDLang3/1000025798.md) | Renvoie le style utilisé par la carte affichée dans le champ Carte. |
| [&lt;Carte&gt;.SuitDéplacement](../WDLang3/1000022464.md) | Affiche la position courante de l'appareil dans un champ Carte et met à jour cette position durant son déplacement. |
| [&lt;Carte&gt;.SupprimeForme](../WDLang3/1000025820.md) | Supprime une forme dessinée sur la carte affichée dans un champ Carte. |
| [&lt;Carte&gt;.SupprimeImage](../WDLang3/1000025829.md) | Supprime une image dessinée sur la carte affichée dans un champ Carte. |
| [&lt;Carte&gt;.SupprimeItinéraire](../WDLang3/1000022465.md) | Supprime un itinéraire d'un champ Carte. |
| [&lt;Carte&gt;.SupprimeMarqueur](../WDLang3/1000022466.md) | Supprime un marqueur d'une carte affichée par un champ Carte. |
| [&lt;Carte&gt;.SupprimeTout](../WDLang3/1000022472.md) | Supprime dans un champ Carte : <br><br>- tous les marqueurs, <br><br>- toutes les formes,<br><br>- toutes les images.<br><br><br> |





Les types de variables manipulés par le champ Carte ou ses fonctions sont les suivants : 



|   |   |
| --- | --- |
| [CarteCercle](../WDLang3/1000025504.md) | Le type **CarteCercle** permet de définir les caractéristiques d'une forme de type cercle à dessiner sur la carte affichée dans un champ Carte. |
| [CarteImage](../WDLang3/1000025523.md) | Le type **CarteImage** permet de définir les caractéristiques d'une image à dessiner sur la carte affichée dans un champ Carte. |
| [CartePolygone](../WDLang3/1000025476.md) | Le type **CartePolygone** permet de définir les caractéristiques d'une forme de type polygone à dessiner sur la carte affichée dans un champ Carte. |
| [CartePolyligne](../WDLang3/1000025489.md) | Le type **CartePolyligne** permet de définir les caractéristiques d'une ligne composée de plusieurs segments à dessiner sur la carte affichée dans un champ Carte. |
| [Marqueur](../WDLang3/1000019940.md) | Le type **Marqueur** permet de définir les caractéristiques d'un marqueur à placer sur une carte affichée dans un champ Carte. |
| [MarqueurImage](../WDLang3/1000025796.md) | Le type **MarqueurImage** permet personnaliser l'image associée à un marqueur ou à un cluster de marqueurs affiché dans le champ Carte. |

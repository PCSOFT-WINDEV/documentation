


## &lt;Carte&gt;.ExécuteJS (Fonction)

***En anglais : &lt;Map&gt;.ExecuteJS***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Exécute du code natif de l'API Google Maps dans un champ Carte en Javascript. 

Cette fonction permet notamment de personnaliser le champ Carte WINDEV en proposant des fonctionnalités spécifiques à l'API Google Maps.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Centre la carte sur 45.20,26.51
soit sCode = [
	var LatLng = new google.maps.LatLng(45.20,26.51);
	CarteRécupèreObjetJS().panTo(LatLng);
]
CARTE_MaCarte.ExécuteJS(sCode)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Champ Carte>.ExécuteJS(<Code Javascript>)`
---

**`<Résultat> : (Chaîne de caractères)`**



- Résultat de l'exécution du code Javascript à exécuter, si ce code renvoie un résultat. Ce résultat est renvoyé sous forme d'une chaîne de caractères. 

- "" (chaîne vide) si le code Javascript ne renvoie pas de résultat.




**`<Champ Carte> : (Nom de champ)`**

Nom du champ Carte à manipuler.

**`<Code Javascript> : (Chaîne de caractères)`**

Code Javascript à exécuter. Dans ce code, le nom de l'objet Javascript représentant la carte Google est connu par la fonction [&lt;Carte&gt;.RécupèreObjetJS](../WDLang3/1000022385.md).



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Pour plus d'informations sur les API Google disponibles sur le champ Carte, consultez la documentation Google ([https://developers.google.com/maps/documentation/javascript/](https://developers.google.com/maps/documentation/javascript/)). 

- Si le code génère des erreurs, ces erreurs peuvent être affichées à l'utilisateur final et/ou entraîner des dysfonctionnements du champ Carte. 

- L'utilisation des API Google Maps n'est pas garantie dans le futur.

- L'utilisation des API Google Maps n'est pas compatible avec les applications iOS/Android.


## telAppelEnAttente (Fonction)

***En anglais : tapiCallIsWaiting***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de savoir si l'appel est en attente.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// L'appel en cours est-il en attente ?
SI telAppelEnAttente() = Vrai ALORS
	// Affichage d'un message
	Message("L'appel a été mis en attente")
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = telAppelEnAttente([<Identifiant de l'appel>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le numéro appelé ou l'appel entrant est en attente, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Identifiant de l'appel> : (Entier optionnel)`**

Identifiant de l'appel à manipuler. Cet identifiant est connu dans la procédure WLangage appelée par la fonction [telDémarreDétectionAppel](../WDLang3/3076029.md). Si ce paramètre n'est pas spécifié, l'appel en cours (sortant ou entrant) est manipulé.

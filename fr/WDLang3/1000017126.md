
## BTInfoRadio (Fonction)

***En anglais : BTInfoRadio***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie des informations spécifiques sur une radio Bluetooth connectée à la machine.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
sListeRadios est une chaîne
sUneRadio est une chaîne
sId est une chaîne
sNom est une chaîne
sAdresseMAC est une chaîne
// Vide la liste
TableSupprimeTout(TABLE_Radios)
// Liste les radios
sListeRadios = BTListeRadio(btListeSimple)
// Parcours des radios
POUR TOUTE CHAINE sUneRadio DE sListeRadios SEPAREE PAR RC
	// Mémorise les informations du périphérique
	sId = ExtraitChaîne(sUneRadio, 1)
	sNom = ExtraitChaîne(sUneRadio, 3)
	sAdresseMAC = ExtraitChaîne(sUneRadio, 2)
	// Si le nom n'est pas renseigné, demande le nom explicitement
	SI sNom ~= "" ALORS sNom = BTInfoRadio(sId, btNom) 
	// Remplit le champ Table avec les radios
	TableAjouteLigne(TABLE_Radios, Faux, sId, sNom, sAdresseMAC)
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = BTInfoRadio(<Identifiant de la radio> , <Information demandée>)`
---

**`<Résultat> : (Chaîne de caractères)`**

Information demandée sur la radio spécifiée.

**`<Identifiant de la radio> : (Entier)`**

Identifiant de la radio Bluetooth à manipuler. Cet identifiant est retourné par la fonction [BTListeRadio](../WDLang3/1000017125.md).

**`<Information demandée> : (Constante de type Entier)`**

Information sur la radio. Le tableau suivant liste les informations disponibles et les valeurs pouvant être renvoyées :


|   |   |
| --- | --- |
| *btAdresseMAC* | Adresse MAC du périphérique |
| *btClasseDePériphérique* | Classe du périphérique, pouvant correspondre à une des constantes suivantes :<br><br>- btCoD_AccèsRéseau<br><br>- btCoD_AudioVidéo<br><br>- btCoD_Divers<br><br>- btCoD_Image<br><br>- btCoD_Inconnu<br><br>- btCoD_Jouet<br><br>- btCoD_Ordinateur<br><br>- btCoD_Périphérique<br><br>- btCoD_Téléphone<br><br>- btCoD_Vêtement<br><br><br> |
| *btClasseDeService* | Classe de service, pouvant correspondre à une ou à une combinaison des constantes suivantes :<br><br>- btCoS_Acquisition : Périphérique d'acquisition de données (scanner, micro, ...).<br><br>- btCoS_Audio : Périphérique audio (micro, haut-parleur, ...).<br><br>- btCoS_Information : Service d'information (Serveur Web ou WAP, ...).<br><br>- btCoS_Positionnement : Périphérique de positionnement (GPS, ...).<br><br>- btCoS_Réseau : Périphérique réseau (point d'accès, ...).<br><br>- btCoS_Restitution : Imprimante, haut-parleur.<br><br>- btCoS_Téléphonie : Appareil de téléphonie (téléphone, modem, ...).<br><br>- btCoS_Transfert : Support de transfert (carte mémoire, ...).<br><br><br> |
| *btFabricant* | Nom du fabricant. |
| *btIDClasseDeService* | Valeur numérique (entier sur 11 bits) correspondant à la classe de service. |
| *btIDFabricant* | Identifiant du fabricant. |
| *btIDMajeurClasseDePériphérique* | Valeur numérique (entier sur 5 bits) correspondant au numéro majeur de la classe de périphérique. |
| *btIDMineurClasseDePériphérique* | Valeur numérique (entier sur 6 bits) correspondant au numéro mineur de la classe de périphérique. |
| *btIDVersion* | Numéro de version fixé par le fabricant. |
| *btNom* | Nom de la radio. |







<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Seules les piles bluetooth Microsoft, Bluesoleil et Broadcom (Widcomm) sont supportées par cette fonction. Pour connaître la pile utilisée, utilisez la fonction [BTPile](../WDLang3/1000017229.md).  

Les piles Broadcom (Widcomm) ne sont pas disponibles en 64 bits.

Pour plus de détails sur les clés supportées, consultez [Quelles piles utiliser ?](../WDLang3/1000017231.md)


## Constantes de gestion des Sockets
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


Les constantes suivantes sont utilisées par les fonctions Socket.

| Constante | Détail | Fonction |
| --- | --- | --- |
| **HashCertificat** | Certificat SSL identifié par son hash | [SocketCréeSSL](../WDLang3/1000017041.md), [SocketConnecteSSL](../WDLang3/1000017042.md) |
| **NomCertificat** | Certificat SSL identifié par son nom | [SocketCréeSSL](../WDLang3/1000017041.md), [SocketConnecteSSL](../WDLang3/1000017042.md) |
| **ProtocoleDéfaut** | Utilisation du protocole par défaut | Type [httpRequête](../WDLang3/1000021158.md) |
| **ProtocoleSSL2** | Utilisation du protocole SSL2 | [SocketCréeSSL](../WDLang3/1000017041.md), [SocketConnecteSSL](../WDLang3/1000017042.md), [WebSocketClientConnecteSSL](../WDLang3/1410087752.md) |
| **ProtocoleSSL3** | Utilisation du protocole SSL3 | [SocketCréeSSL](../WDLang3/1000017041.md), [SocketConnecteSSL](../WDLang3/1000017042.md), [WebSocketClientConnecteSSL](../WDLang3/1410087752.md) |
| **ProtocoleTLS1** | Utilisation du protocole TLS1 | [SocketCréeSSL](../WDLang3/1000017041.md), [SocketConnecteSSL](../WDLang3/1000017042.md), Type [httpRequête](../WDLang3/1000021158.md), [WebSocketClientConnecteSSL](../WDLang3/1410087752.md) |
| **ProtocoleTLS1_1** | Utilisation du protocole TLS 1.1 | [SocketCréeSSL](../WDLang3/1000017041.md), [SocketConnecteSSL](../WDLang3/1000017042.md), Type [httpRequête](../WDLang3/1000021158.md), [WebSocketClientConnecteSSL](../WDLang3/1410087752.md) |
| **ProtocoleTLS1_2** | Utilisation du protocole TLS 1.2 | [SocketCréeSSL](../WDLang3/1000017041.md), [SocketConnecteSSL](../WDLang3/1000017042.md), Type [httpRequête](../WDLang3/1000021158.md), [WebSocketClientConnecteSSL](../WDLang3/1410087752.md) |
| **SocketAdresse** | Information à récupérer : Adresse IP du socket client | [SocketClientInfo](../WDLang3/3070009.md) |
| **SocketErreur** | Une erreur s'est produite durant la communication avec le serveur. | [SocketConnecte](../WDLang3/3070016.md) |
| **SocketFermeture** | Le socket est fermé. | [SocketConnecte](../WDLang3/3070016.md) |
| **SocketMarqueurFin** | Type de transmission : Permet de préciser un marqueur de fin | [SocketChangeModeTransmission](../WDLang3/3070005.md) |
| **SocketMarqueurFinBuffer** | Type de transmission : Permet de préciser un marqueur de fin de buffer | [SocketChangeModeTransmission](../WDLang3/3070005.md) |
| **SocketMessage** | Le socket a reçu un message | [SocketConnecte](../WDLang3/3070016.md) |
| **SocketOuverture** | Le socket est connecté au serveur. | [SocketConnecte](../WDLang3/3070016.md) |
| **socketParamètreMode** | Choix du module utilisé pour la gestion des sockets TCP sécurisées. | [SocketParamètre](../WDLang3/1410088949.md) |
| **SocketPort** | Information à récupérer : Numéro de port auquel est connecté le socket | [SocketClientInfo](../WDLang3/3070009.md) |
| **SocketSansMarqueurFin** | Type de transmission : Aucun marqueur n'est ajouté et/ou enlevé de la chaîne de caractère transmise | [SocketChangeModeTransmission](../WDLang3/3070005.md) |
| **SocketTailleDébut** | Type de transmission : Permet d'indiquer le message à transmettre | [SocketChangeModeTransmission](../WDLang3/3070005.md) |

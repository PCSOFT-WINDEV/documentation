


## &lt;Graphe&gt;.NuageRécupèreDonnéeXY (Fonction)

***En anglais : &lt;Chart&gt;.ScatterGetDataXY***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Récupère une donnée dans un graphe "Nuage de points". 

![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Les graphes](../WDChamp/3042060.md). 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affichage d'une donnée dans le champ "GRF_MonGraphe" de type "Nuage de points". 
// Cette donnée est la première abscisse de la première série.
Info(GRF_MonGraphe.NuageRécupèreDonnéeXY(1, 1, grValeurX))
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Champ Graphe>.NuageRécupèreDonnéeXY(<Numéro de la série> , <Indice> , <Valeur à récupérer>)`
---

**`<Résultat> : (Réel)`**

Valeur récupérée.

**`<Champ Graphe> : (Nom de champ)`**

Nom du champ Graphe manipulé (présent dans l'éditeur de fenêtres, de pages ou d'états).


**`<Numéro de la série> : (Entier)`**

Numéro de la série pour laquelle une valeur doit être récupérée. 
Si la série spécifiée n'existe pas, la valeur récupérée est 0.

**`<Indice> : (Entier)`**

Indice de la donnée dans la série pour laquelle la valeur doit être récupérée.
Si la donnée correspondant à l'indice spécifié n'existe pas dans la série, la valeur récupérée est 0.

**`<Valeur à récupérer> : (Constante de type Entier)`**

Indique la valeur à récupérer :


|   |   |
| --- | --- |
| *grValeurX* | Valeur X de la série. |
| *grValeurY* | Valeur Y de la série. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Pour récupérer une donnée dans un graphe Boursier, il est conseillé d'utiliser la fonction [&lt;Graphe&gt;.BourseRécupèreDonnée](../WDLang3/1000023566.md).

- Pour récupérer une donnée dans un graphe Surface, il est conseillé d'utiliser la fonction [grSurfaceRécupèreDonnée](../ErrCompil/1000001036.md).

- Pour récupérer une donnée dans un autre type de graphe (Secteur, Histogramme, Courbe, etc.), utilisez [&lt;Graphe&gt;.RécupèreDonnée](../WDLang3/1000024122.md).

- Pour récupérer une donnée dans un graphe de type Secteur multiniveau, il est conseillé d'utiliser la fonction [&lt;Graphe&gt;.SMNRécupèreDonnée](../WDLang3/1000024137.md). 

- Pour récupérer une donnée dans un graphe de type Heatmap, il est conseillé d'utiliser la fonction [&lt;Graphe&gt;.HMRécupèreDonnée](../WDLang3/1000024083.md).

- Il est préférable d'utiliser la fonction **&lt;Graphe&gt;.NuageRécupèreDonnéeXY** pour récupérer des données dans des graphes de type "Nuage de points". Il est également possible d'utiliser la fonction [&lt;Graphe&gt;.RécupèreDonnée](../WDLang3/1000024122.md).
	La syntaxe suivante :
	
	```wl
	GRF_MonGraphe.NuageRécupèreDonnéeXY(i, 1, grValeurX)
	GRF_MonGraphe.NuageRécupèreDonnéeXY(i, 1, grValeurY)
	```
est équivalente à la syntaxe : 
	
	```wl
	GRF_MonGraphe.RécupèreDonnée(i*2-1, 1)
	GRF_MonGraphe.RécupèreDonnée(i*2, 1)
	```

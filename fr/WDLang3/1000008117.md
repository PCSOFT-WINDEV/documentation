
## GroupeSuivant (Fonction)

***En anglais : GroupNext***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Lit le groupe de contacts suivant le groupe en cours dans le carnet d'adresses d'une messagerie Lotus Notes ou Outlook.

Lors de la lecture d'un groupe de contacts, les données de ce groupe sont mises à jour dans la [structure Groupe](../WDLang3/1000008120.md). Pour récupérer les données du groupe de contacts en cours, il suffit de récupérer les valeurs de la [structure Groupe](../WDLang3/1000008120.md).

**Attention** : Cette fonction **ne permet pas** de lire le groupe de contacts suivant le groupe en cours dans le carnet d'adresses d'une messagerie Outlook Express.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ouverture d'une session Lotus Notes
IDSession est un entier
IDSession = EmailOuvreSessionNotes("MotDePasse", "ServeurMars", ...
	"mail\julie.nsf", "C:\Users\Julie\AppData\Local\Lotus\Notes\Data\user.id")
// Lecture du premier groupe de contacts
GroupePremier(IDSession)
// Ajout des groupes de contacts trouvés dans un champ Liste
TANTQUE mGroupe.EnDehors = Faux
	ListeAjoute(LISTE_GroupeContact, mGroupe.Nom + TAB + mGroupe.Description)
	GroupeSuivant(IDSession)
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = GroupeSuivant(<Identifiant de la session>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le groupe de contacts suivant a été lu, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour connaître le libellé de l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.




**`<Identifiant de la session> : (Entier)`**

Identifiant de la session à utiliser. Cet identifiant est renvoyé par une des fonctions suivantes : [EmailOuvreSessionNotes](../WDLang3/3032121.md), [EmailOuvreSessionOutlook](../WDLang3/3032128.md), [NotesOuvreBase](../WDLang3/1000005101.md) ou [OutlookOuvreSession](../WDLang3/3032123.md).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Groupe de contact en cours
<a name="groupe_contact_cours_ELTPARAGRAPHE000067"></a>La variable mGroupe.EnDehors permet de savoir s'il y a un groupe de contacts courant.


<a name="NOTE0_2"></a>




### Version de Outlook utilisée
<a name="version_outlook_utilisee_ELTPARAGRAPHE000076"></a>Les groupes de contacts ne sont pas gérés par Outlook version 97. Il n'est donc pas possible de rechercher un groupe de contacts dans cette version de Outlook.


## BTChangeVisibilité (Fonction)

***En anglais : BTChangeVisibility***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Change la visibilité d'une radio Bluetooth.

**Remarque** : Une radio Bluetooth correspond à un émetteur Bluetooth. Cet émetteur peut par exemple correspondre à une clé USB Bluetooth, ou être un élément intégré au poste (par exemple dans le cas d'un portable).




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple

![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) 
```wl
bVisible est un booléen
// Identifiant des radios Bluetooth listés dans un champ Table 
nIdRadio est un entier
nIdRadio = TABLE_Radios.COL_ID[TABLE_Radios]
// Teste si la radio sélectionnée est visible
bVisible = BTEstVisible(nIdRadio)
// Affichage 
SI bVisible = Vrai ALORS
	Trace("Visible")
SINON 
	Trace("Non visible")
FIN
// Changement de la visibilité
BTChangeVisibilité(nIdRadio, PAS bVisible)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe
![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) 
### Syntaxe utilisable sous Windows

`<Résultat> = BTChangeVisibilité(<Identifiant de la radio> , <Nouvel état>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le changement de visibilité a réussi, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. La fonction [ErreurInfo](../WDLang1/3013008.md) permet d'obtenir plus de renseignements sur le problème rencontré.




**`<Identifiant de la radio> : (Entier)`**

Identifiant de la radio Bluetooth à manipuler. Cet identifiant est retourné par la fonction [BTListeRadio](../WDLang3/1000017125.md).

**`<Nouvel état> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> pour que la radio devienne visible, 

- <u><u><u><u>Faux</u></u></u></u> pour que la radio soit cachée.

## Remarques




### Radio visible et demande de connexion
<a name="radio_visible_demande_connexion_ELTPARAGRAPHE000118"></a>Une radio Bluetooth visible accepte automatiquement les demandes de connexion. Lorsque la fonction **BTChangeVisibilité** permet de rendre la radio visible, l'appel à la fonction [BTChangeConnectivité](../WDLang3/1000017130.md) est automatiquement réalisé.

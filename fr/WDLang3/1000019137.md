
## SaaSAdminChercheAbonnement (Fonction)

***En anglais : SaaSAdminFindSubscription***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Recherche un abonnement SaaS.

**Remarques** : 

- L'utilisation de cette fonction nécessite d'avoir ouvert une session d'administration SaaS au préalable en utilisant la fonction [SaaSAdminConnecte](../WDLang3/1000019098.md).

- L'Administrateur SaaS aura accès à tous les abonnements. Un gestionnaire de compte client aura accès uniquement à ses propres abonnements. 



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
S est un saasSite 
T est une saasTarification 
C est un saasClient 
Ab est un saasAbonnement 

S = SaaSAdminChercheSite("Forum")
T = SaaSAdminChercheTarification("Promo 2010", S)
C = SaaSAdminChercheClient("Hybrid")
Ab = SaaSAdminChercheAbonnement(T, C)
SI ErreurDétectée = Faux ALORS
	Info("Le client est bien abonné à cette tarification ")
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Rechercher un abonnement (utilisation des variables de type saasxxx)

`<Résultat> = SaaSAdminChercheAbonnement(<Tarification> , <Compte client>)`
---

**`<Résultat> : (Variable de type saasAbonnement)`**

Variable de type [saasAbonnement](../WDLang3/1000019088.md) contenant la description de l'abonnement recherché.

Si la recherche n'a pas aboutie, la variable renvoyée est vide.

**`<Tarification> : (Variable de type saasTarification)`**

Nom de la variable de type [saasTarification](../WDLang3/1000019091.md) correspondant à la tarification pour laquelle l'abonnement est recherché.

**`<Compte client> : (Variable de type saasClient)`**

Nom de la variable de type [saasClient](../WDLang3/1000019072.md) correspondant au compte client pour lequel l'abonnement est recherché.


<a name="SYNTAXE2"></a>

### Rechercher un abonnement grâce au nom de la tarification

`<Résultat> = SaaSAdminChercheAbonnement(<Tarification> , <Site SaaS> , <Compte client>)`
---

**`<Résultat> : (Variable de type saasAbonnement)`**

Variable de type [saasAbonnement](../WDLang3/1000019088.md) contenant la description de l'abonnement recherché.

Si la recherche n'a pas aboutie, la variable renvoyée est vide.

**`<Tarification> : (Chaîne de caractères)`**

Nom de la tarification pour laquelle l'abonnement est recherché.

**`<Site SaaS> : (Chaîne de caractères)`**

Nom du site SaaS possédant la tarification.

**`<Compte client> : (Chaîne de caractères)`**

Nom du compte client pour lequel l'abonnement est recherché.

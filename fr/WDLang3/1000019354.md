
## WiFiEtat (Fonction)

***En anglais : WiFiStatus***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie l'état actuel d'activation du Wi-Fi sur l'appareil ou demande à être notifié des changements d'état d'activation du Wi-Fi.

**Remarques** : 





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI WiFiEtat() = wifiDésactivé ALORS
	WiFiActive(Vrai)
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître l'état actuel d'activation du Wi-Fi sur le poste en cours

`<Résultat> = WiFiEtat()`
---

**`<Résultat> : (Entier)`**

Etat actuel d'activation du Wi-Fi :


|   |   |
| --- | --- |
| *wifiActif* | Le Wi-Fi est activé sur l'appareil. |
| *wifiActivationEnCours* | Le Wi-Fi est en cours d'activation sur l'appareil.<br><br> |
| *wifiDésactivationEnCours* | Le Wi-Fi est en cours de désactivation sur l'appareil.<br><br> |
| *wifiDésactivé* | Le Wi-Fi est désactivé sur l'appareil. |
| *wifiErreur* | L'état d'activation du Wi-Fi n'a pas pu être récupéré.<br><br> |

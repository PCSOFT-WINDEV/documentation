
## SaaSAdminChercheClient (Fonction)

***En anglais : SaaSAdminFindClient***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Recherche un client SaaS.

**Remarques** : 

- L'utilisation de cette fonction nécessite d'avoir ouvert une session d'administration SaaS au préalable en utilisant la fonction [SaaSAdminConnecte](../WDLang3/1000019098.md).

- L'Administrateur SaaS aura accès à tous les comptes client. Un gestionnaire de compte client aura accès uniquement à son propre compte client. 



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Recherche des informations du client "Hybrid"
S est un saasClient dynamique 
S = SaaSAdminChercheClient("Hybrid")
SI S = Null ALORS
	Erreur("Client non trouvé")
SINON
	SI SaaSAdminSupprimeClient(S) = Vrai ALORS
		Info("Client supprimé")
	FIN
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = SaaSAdminChercheClient(<Client recherché>)`
---

**`<Résultat> : (Variable de type saasClient)`**

Variable de type [saasClient](../WDLang3/1000019072.md) contenant la description du compte client si la recherche a abouti.

Si le client n'a pas été trouvé, la variable renvoyée est vide.

**`<Client recherché> : (Chaîne de caractères)`**

Information permettant d'identifier le client : 

- Nom du client.

- Raison sociale.

- Référence externe.




## NetListeAdresseIP (Fonction)

***En anglais : NetListIPAddress***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la liste des adresses IP (Internet Protocol) d'une machine. 

<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affiche les adresses IP de la machine locale
Info(NetListeAdresseIP())
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = NetListeAdresseIP([<Nom de la machine>] [, <Options>])`
---

**`<Résultat> : (Chaîne de caractères)`**

Liste des adresses IP séparées par RC (Retour Chariot). 

**`<Nom de la machine> : (Chaîne de caractères optionnelle)`**

Nom de la machine pour laquelle la liste des adresses IP est recherchée. Si ce paramètre n'est pas précisé, la liste des adresses IP pour la machine locale est renvoyée.


**`<Options> : (Constante ou combinaison de constantes optionnelle de type Entier)`**

Type d'adresse à récupérer : 


|   |   |
| --- | --- |
| *IPv4* | Liste les adresses IPv4. |
| *IPv6* | Liste les adresses IPv6. |


Si ce paramètre n'est pas spécifié, la liste des adresses IPv4 et IPv6 est renvoyée (combinaison de constantes *IPv6* + *IPv4*).

## Remarques


### Equivalence
<a name="equivalence_ELTPARAGRAPHE000063"></a>Les codes suivants sont équivalents : 

```wl
Trace(NetListeAdresseIP(ipv4))
```


```wl
n est un entier = 0
TANTQUE NetAdresseIP(n) <> ""
	Trace(NetAdresseIP(n))
	n = n + 1
FIN
```

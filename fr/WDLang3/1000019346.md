
## OutlookListeProfil (Fonction)

***En anglais : OutlookListProfile***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Liste les profils Outlook disponibles.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
ListeProfil est une chaîne 
ListeProfil = OutlookListeProfil()
POUR TOUTE CHAINE sUnProfil DE ListeProfil SEPAREE PAR RC
	Trace(sUnProfil)
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = OutlookListeProfil()`
---

**`<Résultat> : (Chaîne de caractères)`**

Liste des profils Outlook disponibles séparés par RC (Retour Chariot).

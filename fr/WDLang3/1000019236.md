
## telDialerAffiche (Fonction)

***En anglais : tapiDialerDisplay***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ouvre l'application de téléphonie (dialer) par défaut et affiche le numéro spécifié. Aucun appel n'est déclenché.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
telDialerAffiche("0612345678")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = telDialerAffiche(<Numéro de téléphone>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'application de téléphonie (dialer) a pu être ouverte, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Numéro de téléphone> : (Chaîne de caractères)`**

Numéro de téléphone à afficher dans l'écran de numérotation. 

Si ce paramètre est une chaîne vide (""), l'application de téléphonie sera ouverte sur l'écran de numérotation.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Appel automatique
<a name="appel_automatique_ELTPARAGRAPHE000054"></a>Pour déclencher automatiquement l'appel du numéro spécifié, utilisez la fonction [telDialerCompose](../WDLang3/1000019235.md).

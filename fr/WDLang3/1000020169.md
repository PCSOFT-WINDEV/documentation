
## VolumeModifie (Fonction)

***En anglais : VolumeModify***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie le réglage du volume d'une source sonore.

**Remarque** : Cette fonction peut être utilisée dans l'émulateur.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Coupe la sonnerie du téléphone et active le mode vibreur
VolumeModifie(0, volumeSonnerie, volumeVibreurOn)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = VolumeModifie(<Volume> [, <Source> [, <Options>]])`
---

**`<Résultat> : (Entier)`**



- Valeur du volume de la source spécifiée après modification, 

- -1 en cas d'erreur. Dans ce cas, la variable [ErreurDétectée](../WDLang1/3087001.md) est à <u><u><u><u>Vrai</u></u></u></u>. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Volume> : (Entier ou constante de type Entier)`**

Nouvelle valeur du volume : 

- Valeur comprise entre 0 (muet) et 100 (volume maximum). 

- Constante permettant de faire varier le volume en cours : 
	


|   |   |
| --- | --- |
| *volumeMoins* | Décrémente la valeur du volume d'un niveau. |
| *volumePlus* | Incrémente la valeur du volume d'un niveau. |






**`<Source> : (Constante optionnelle de type Entier)`**

Source sonore à manipuler : 


|   |   |
| --- | --- |
| *volumeAlarme* | Volume de l'alarme. |
| *volumeAppel* | Volume des appels. |
| *volumeDéfaut* | Volume de la source par défaut. Cette source dépend de la configuration du téléphone. |
| *volumeMusique* | Volume utilisé lors de la lecture d'un fichier audio ou vidéo. |
| *volumeNotification* | Volume du son émit lors d'une nouvelle notification. |
| *volumeSonnerie* | Volume de la sonnerie du téléphone. |
| *volumeSystème* | Volume des sons système. |





**`<Options> : (Constante ou combinaison de constantes optionnelle de type Entier)`**

Options disponibles :


|   |   |
| --- | --- |
| *volumeToast* | Provoque l'affichage d'un toast indiquant la nouvelle valeur du volume pour la source sonore spécifiée. |
| *volumeVibreurOff* | Désactive la vibration de l'appareil lors de l'arrivée d'une notification ou d'un appel. Cette option est prise en compte uniquement lorsque la source sonore spécifiée correspond à la constante *volumeSonnerie* ou *volumeNotification*. |
| *volumeVibreurOn* | Déclenche la vibration de l'appareil lors de l'arrivée d'une notification ou d'un appel. Cette option est prise en compte uniquement lorsque la source sonore spécifiée correspond à la constante *volumeSonnerie* ou *volumeNotification*. |
| *volumeVibreurSiMuet* | Déclenche la vibration de l'appareil lors de l'arrivée d'une notification ou d'un appel si la source sonore correspondante est muette. Cette option est prise en compte uniquement lorsque la source sonore spécifiée correspond à la constante *volumeSonnerie* ou *volumeNotification*. |







<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

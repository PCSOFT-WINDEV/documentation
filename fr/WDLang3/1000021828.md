
## RendezVousVersiCalendar (Fonction)

***En anglais : AppointmentToiCalendar***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Exporte un rendez-vous ou une liste de rendez-vous au format iCalendar.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Crée un fichier au format iCalendar 
// contenant la liste des rendez-vous affichés dans un champ Planning
tabRDV est un tableau de RendezVous
tabRDV = PlanningListeRendezVous(MonPlanning)
sChaîne est une chaîne = RendezVousVersiCalendar(tabRDV)
SI PAS ErreurDétectée ALORS
	fSauveTexte("agenda.ics", sChaîne)
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Exporter une liste de rendez-vous au format iCalendar

`<Résultat> = RendezVousVersiCalendar(<Tableau de rendez-vous>)`
---

**`<Résultat> : (Chaîne de caractères ANSI au format UTF-8)`**



- Liste des rendez-vous au format iCalendar,

- Chaîne vide ("") en cas d'erreur. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md). 




**`<Tableau de rendez-vous> : (Tableau de variables de type RendezVous)`**

Nom de la variable de type Tableau contenant les variables de type [RendezVous](../WDLang1/1000019244.md) à exporter au format iCalendar.


<a name="SYNTAXE2"></a>

### Exporter un rendez-vous au format iCalendar

`<Résultat> = RendezVousVersiCalendar(<Rendez-vous>)`
---

**`<Résultat> : (Chaîne de caractères ANSI au format UTF-8)`**



- Rendez-vous au format iCalendar,

- Chaîne vide ("") en cas d'erreur. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md). 




**`<Rendez-vous> : (Variable de type RendezVous)`**

Nom de la variable de type [RendezVous](../WDLang1/1000019244.md) à exporter au format iCalendar.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Pour récupérer une liste de rendez-vous à partir d'une chaîne au format iCalendar, utilisez la fonction [iCalendarVersRendezVous](../WDLang3/1000021827.md).

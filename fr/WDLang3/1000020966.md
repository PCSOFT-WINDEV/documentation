


## Fonctions DLNA / UPnP
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions WLangage permettant de manipuler des périphériques UPnP sont les suivantes : 




|   |   |
| --- | --- |
| [UPNPCherchePériphérique](../WDLang3/1000020912.md) | Recherche un périphérique à partir de son nom unique (UDN). |
| [UPNPExécuteAction](../WDLang3/1000020918.md) | Exécute une action sur le service UPnP. |
| [UPNPListePériphérique](../WDLang3/1000020901.md) | Liste les périphériques UPnP disponibles sur le réseau. |
| [UPNPMédiaArrêt](../WDLang3/1000020944.md) | Arrête la lecture du média en cours sur le périphérique UPnP. |
| [UPNPMédiaJoue](../WDLang3/1000020941.md) | Lance la lecture du média en cours sur le périphérique UPnP. |
| [UPNPMédiaOuvre](../WDLang3/1000020942.md) | Ouvre un média sur le périphérique UPnP. |
| [UPNPMédiaPause](../WDLang3/1000020943.md) | Met en pause la lecture du média en cours sur le périphérique UPnP. |
| [UPNPModifieContraste](../WDLang3/1000020938.md) | Modifie le contraste d'un périphérique UPnP. |
| [UPNPModifieLuminosité](../WDLang3/1000020939.md) | Modifie la luminosité d'un périphérique UPnP. |
| [UPNPModifieMuet](../WDLang3/1000020931.md) | Modifier l'état 'Muet' d'un périphérique UPnP. |
| [UPNPModifieNetteté](../WDLang3/1000020936.md) | Modifie la netteté d’un périphérique UPnP. |
| [UPNPModifieVolume](../WDLang3/1000020937.md) | Modifie le volume d’un périphérique UPnP. |
| [UPNPRécupèreContraste](../WDLang3/1000020933.md) | Récupère le contraste d’un périphérique UPnP. |
| [UPNPRécupèreLuminosité](../WDLang3/1000020932.md) | Récupère la luminosité d'un périphérique UPnP. |
| [UPNPRécupèreMuet](../WDLang3/1000020929.md) | Récupère l'état 'Muet' d'un périphérique UPnP. |
| [UPNPRécupèreNetteté](../WDLang3/1000020935.md) | Récupère la netteté d’un périphérique UPnP. |
| [UPNPRécupèreVolume](../WDLang3/1000020934.md) | Récupère le volume d’un périphérique UPnP. |

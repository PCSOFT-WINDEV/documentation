
## Constantes OBEX
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


| Constante | Description | Utilisée par la fonction |
| --- | --- | --- |
| **obexBluetooth** | Transport des données effectué par Bluetooth | [OBEXConnecte](../WDLang3/1000017153.md) |
| **obexInfraRouge** | Transport des données effectué par Infrarouge | [OBEXConnecte](../WDLang3/1000017153.md) |
| **obexListeDétaillée** | Liste les fichiers OBEX sous forme détaillée : <br>&lt;Nom du fichier 1&gt; + TAB + &lt;Type 1&gt; + TAB + &lt;Taille en octets 1&gt; + TAB + &lt;Permissions 1&gt; + TAB + &lt;Date de modification 1&gt; + TAB + &lt;Date de création 1&gt; + TAB + &lt;Date d'accès 1&gt; + RC + ... | [OBEXListeFichier](../WDLang3/1000017148.md) |
| **obexListeRépertoire** | Liste les fichiers et les répertoires OBEX. | [OBEXListeFichier](../WDLang3/1000017148.md) |
| **obexListeSimple** | Liste les fichiers OBEX sous forme simple : <br>&lt;Nom du fichier 1&gt; + TAB + &lt;Type 1&gt; + RC + ... &lt;Nom du fichier N&gt; + TAB + &lt;Type N&gt; <br>où &lt;Type&gt; correspond à "F" si l'élément est un fichier, et "D" si l'élément est un répertoire. | [OBEXListeFichier](../WDLang3/1000017148.md) |


## &lt;Graphe&gt;.IncrémenteDonnée (Fonction)

***En anglais : &lt;Chart&gt;.IncreaseData***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ajoute une valeur à une donnée dans un graphe. 

![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Les graphes](../WDChamp/3042060.md). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ajoute la valeur 7 à la deuxième donnée de la première série
GRF_MonGraphe.IncrémenteDonnée(1, 2, 7)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Champ Graphe>.IncrémenteDonnée(<Numéro de la série> , <Indice> , <Valeur>)`
---

**`<Champ Graphe> : (Nom de champ)`**

Nom du champ Graphe à manipuler (présent dans l'éditeur de fenêtres, de pages ou d'états).


**`<Numéro de la série> : (Entier)`**

Numéro de la série dans laquelle une valeur doit être ajoutée. Si la série spécifiée n'existe pas, le grapheur crée autant de séries que nécessaire pour obtenir une matrice de données cohérente. Les séries ainsi créées sont toutes initialisées à 0.

**`<Indice> : (Entier)`**

Indice de la donnée dans la série. Si la donnée spécifiée n'existe pas, le grapheur crée autant de données que nécessaire pour obtenir une matrice de données cohérente. Les données ainsi créées sont toutes initialisées à 0.



**`<Valeur> : (Entier ou réel)`**

Valeur à ajouter à la donnée spécifiée.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Types de graphes
<a name="types_graphes_ELTPARAGRAPHE000252"></a>Pour les graphes de type Secteur, seules les données de la première série sont utilisées.
<a name="NOTE0_2"></a>


### Cas particuliers
<a name="cas_particuliers_ELTPARAGRAPHE000262"></a>

- Les données définies avec la fonction **&lt;Graphe&gt;.IncrémenteDonnée** seront prises en compte lors du prochain appel à la fonction [&lt;Graphe&gt;.Dessine](../WDLang3/1000024062.md).

- Une série peut être composée de 8000 éléments maximum.

- Il est impossible de dessiner un graphe de type Secteur si toutes les données de la première série sont nulles.

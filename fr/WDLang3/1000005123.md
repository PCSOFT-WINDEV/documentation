
## NotesListeVue (Fonction)

***En anglais : NotesListView***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la liste des vues présentes dans la base Lotus Notes en cours.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ouverture de la connexion
IDConnexion est un entier
IDConnexion = NotesOuvreConnexion("MotDePasse", "ServeurMars", ...
		"C:\Users\Julie\AppData\Local\Lotus\Notes\Data\user.id")
// Spécification de la base Lotus Notes à utiliser
IDBase est un entier
IDBase = NotesOuvreBase(IDConnexion, "julie.nsf")
...
// Liste des vues de la base
ListeVue est une chaîne
NomVue est une chaîne
ListeVue = NotesListeVue(IDBase)
// Pour chaque vue
POUR TOUTE CHAINE NomVue DE ListeVue SEPAREE PAR RC
	// Ajoute le nom de la vue dans un champ Liste
	ListeAjoute(LISTE_ListeNomVue, NomVue)
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = NotesListeVue(<Identifiant de la session>)`
---

**`<Résultat> : (Chaîne de caractères)`**

Nom des vues présentes dans la base Lotus Notes, séparé par un RC.

**`<Identifiant de la session> : (Entier)`**

Identifiant de la session à utiliser. Cet identifiant est renvoyé par la fonction [EmailOuvreSessionNotes](../WDLang3/3032121.md) ou [NotesOuvreBase](../WDLang3/1000005101.md).

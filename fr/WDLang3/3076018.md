
## telNumérote (Fonction)

***En anglais : tapiSendKey***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de simuler l'utilisation des touches du clavier téléphonique.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
telNumérote("123456789ABCD*#", LISTE_ListeAppel[LISTE_ListeAppel])
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = telNumérote(<Suite de touches> [, <Identifiant de l'appel>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le la numérotation a été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Suite de touches> : (Chaîne de caractères)`**

Suite de touches à simuler.

**`<Identifiant de l'appel> : (Entier optionnel)`**

Identifiant de l'appel à manipuler. Cet identifiant est connu dans la procédure WLangage appelée par la fonction [telDémarreDétectionAppel](../WDLang3/3076029.md). Si ce paramètre n'est pas spécifié, l'appel en cours est manipulé.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Limitation
<a name="limitation_ELTPARAGRAPHE000047"></a>Il n'est pas possible d'appeler la fonction **telNumérote** dans la procédure de détection d'appels (définie grâce à la fonction [telDémarreDétectionAppel](../WDLang3/3076029.md)).

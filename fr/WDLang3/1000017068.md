
## SNMPFermeSession (Fonction)

***En anglais : SNMPCloseSession***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ferme une session SNMP.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
IDSession est un entier
// Ouverture de session
IDSession = SNMPOuvreSession("TestDoc", "public")
SI IDSession = -1 ALORS Erreur("Erreur d'ouverture de session " + ErreurInfo())
...
SNMPFermeSession(IDSession)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = SNMPFermeSession(<Identifiant de la session SNMP>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la session a été correctement fermée, 

- <u><u><u><u>Faux</u></u></u></u> en cas d'erreur. La fonction [ErreurInfo](../WDLang1/3013008.md) permet d'obtenir plus de renseignements sur le problème rencontré.




**`<Identifiant de la session SNMP> : (Entier)`**

Identifiant de la session SNMP à fermer. Cet identifiant a été renvoyé par la fonction [SNMPOuvreSession](../WDLang3/1000017067.md).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Lors de la fermeture de l'application (ou lors de la fermeture de la session d'un site dynamique), les sessions SNMP encore ouvertes sont automatiquement fermées.




## &lt;Graphe&gt;.HMAjouteDonnée (Fonction)

***En anglais : &lt;Chart&gt;.HMAddData***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ajoute une donnée dans un graphe de type Heatmap.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
GRF_Graphe1.HMAjouteDonnée(1, 1, 10)
GRF_Graphe1.HMAjouteDonnée(1, 2, 30)
GRF_Graphe1.HMAjouteDonnée(2, 1, 20.5)
GRF_Graphe1.HMAjouteDonnée(2, 2, 5)
```

```wl
// Définition des couleurs
GRF_HMap.HMCouleur(-50, BleuFoncé)
GRF_HMap.HMCouleur(+50, RougeClair)
POUR i = 1 A 500
	GRF_HMap.HMAjouteDonnée(Hasard(-100, 100), Hasard(-100, 100), -100)
	GRF_HMap.HMAjouteDonnée(Hasard(-100, 100), Hasard(-100, 100), 100)
FIN
GRF_SansNom1.Dessine()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Champ Graphe>.HMAjouteDonnée(<X> , <Y> , <Valeur>)`
---

**`<Champ Graphe> : (Nom de champ)`**

Nom du champ Graphe à manipuler (présent dans l'éditeur de fenêtres ou d'états).


**`<X> : (Entier)`**

Position de la donnée sur l'axe des abscisses.

**`<Y> : (Entier)`**

Position de la donnée sur l'axe des ordonnées.

**`<Valeur> : (Entier ou réel)`**

Valeur à ajouter. Si une valeur se trouve déjà à la position (X, Y) indiquée, la valeur existante est remplacée par la nouvelle valeur.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Les données définies avec la fonction **&lt;Graphe&gt;.HMAjouteDonnée** seront prises en compte lors du prochain appel à la fonction [&lt;Graphe&gt;.Dessine](../WDLang3/1000024062.md).

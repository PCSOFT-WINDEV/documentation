
## Créer un Widget Android
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000125"></a>
Les Widgets sont de petites applications qui peuvent être affichées sur l'écran d'accueil du téléphone ou de la tablette. Ces Widgets permettent d'afficher des informations mises à jour à intervalle régulier et offrent des interactions simples avec l'utilisateur. 

WINDEV Mobile permet de créer des Widgets à partir de vos applications Android grâce aux fenêtres de type Widget. 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Création et déploiement d'un Widget Android
<a name="creation_deploiement_widget_android_ELTTEXTE000149"></a>
Le principe est simple : 

1. Vous créez un projet Android. Pour plus de détails, consultez [Développement d'applications pour Android](../Editeurs/9000099.md). 

2. Dans votre projet Android, vous créez des fenêtres de type Widget. Ces fenêtres contiennent toutes les fonctionnalités nécessaires à votre Widget. 
	Remarques : 

	- Seuls 3 types de champs sont disponibles dans les fenêtres Widget : [les champs Bouton](../WDChamp/1013076.md), [les champs Libellé](../WDChamp/1013179.md) et [les champs Image](../WDChamp/1013218.md).

	- Les styles des champs d'un Widget définis en édition seront ignorés en exécution. Il est uniquement possible de définir la couleur de fond des champs et l'image de fond pour les champs Bouton.

	- Les fenêtres Widget présentent un certains nombre de spécificités. Pour plus de détails, consultez [Fenêtres Widget](../WDChamp/9000143.md). 




3. Générez votre application Android. Pour plus de détails, consultez [Génération d'une application Android](../Editeurs/9000109.md). 

4. Déployez le fichier apk généré. 

5. Les Widgets contenus dans l'application seront présents dans la liste des Widgets disponibles et pourront être installés sur le bureau. 







### Spécificité des Widgets
<a name="specificite_des_widgets_ELTPARAGRAPHE000052"></a>Le ou les Widget(s) d'une application générée avec WINDEV Mobile s'exécute(nt) dans un processus différent de l'application elle-même. 

En revanche, tous les Widgets d'une même application s'exécutent dans le même processus et partageront donc le même contexte d'exécution (variables globales, parcours HFSQL, ...). 

L'utilisateur a la possibilité d'ajouter sur l'écran d'accueil autant d'instances d'un même Widget qu'il le souhaite. Chacune de ces instances correspondra à la même instance de fenêtre : les modifications effectuées sur la fenêtre seront automatiquement répercutées sur toutes les instances affichées du Widget correspondant.

## Fonctions WLangage spécifiques aux Widgets Android
<a name="fonctions_wlangage_specifiques_aux_widgets_android_ELTTEXTE000179"></a>


### Fonctions de gestion des Widgets
<a name="fonctions_gestion_des_widgets_ELTPARAGRAPHE000066"></a>Les fonctions suivantes permettent de gérer les spécificités des Widgets Android : 



|   |   |
| --- | --- |
| [EnModeWidget](../WDLang1/1000020649.md) | Identifie si le code est exécuté en mode Widget Android ou non. |
| [WidgetAffiche](../WDLang3/1000020647.md) | Rafraîchit l'affichage d'une fenêtre de type Widget. |
| [WidgetLanceAppli](../WDLang3/1000020648.md) | Lance l'application à laquelle appartient un Widget. |






### Fonctions utilisables dans les Widgets
<a name="fonctions_utilisables_dans_les_widgets_ELTPARAGRAPHE000072"></a>La plupart des fonctions disponibles pour les applications Android sont également disponibles en mode Widget. Pour identifier facilement ces fonctions, l'icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=androidwidget.gif)
 apparaît dans la barre des plateformes de la fonction.




## &lt;Graphe&gt;.NuageEtiquetteSérie (Fonction)

***En anglais : &lt;Chart&gt;.ScatterSeriesLabel***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Initialise l'étiquette d'une série de données dans un graphe de type "Nuage de points".

Exemple d'étiquette de série : ![Etiquette de série](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=EtiquetteSerie.gif)


**Rappel** : Par défaut, un graphe n'a aucune étiquette.



![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Les graphes](../WDChamp/3042060.md). 








<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Initialise l'étiquette "Achat" de la série de données 3 dans le champ GRF_MonGraphe
GRF_MonGraphe.NuageEtiquetteSérie(3, "Achat")
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Champ Graphe>.NuageEtiquetteSérie(<Numéro de la série> , <Étiquette>)`
---

**`<Champ Graphe> : (Nom de champ)`**

Nom du champ Graphe à manipuler (présent dans l'éditeur de fenêtres, de pages ou d'états).


**`<Numéro de la série> : (Entier)`**

Numéro de la série pour laquelle une étiquette doit être initialisée.

**`<Étiquette> : (Chaîne de caractères)`**

Libellé de l'étiquette à initialiser.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Il est possible de modifier la police de l'étiquette grâce à la fonction [&lt;Graphe&gt;.PoliceEtiquette](../WDLang3/1000024117.md).

- Les données définies avec la fonction **&lt;Graphe&gt;.NuageEtiquetteSérie** seront prises en compte lors du prochain appel à la fonction [&lt;Graphe&gt;.Dessine](../WDLang3/1000024062.md).

- La fonction **&lt;Graphe&gt;.NuageEtiquetteSérie** est équivalente à la fonction [&lt;Graphe&gt;.EtiquetteSérie](../WDLang3/1000024073.md). 
	La syntaxe de [&lt;Graphe&gt;.EtiquetteSérie](../WDLang3/1000024073.md) utilisée avec des graphes de type "Nuage de points" est la suivante :
	
	```txt
	<Champ Graphe>.EtiquetteSérie(2*<Indice>-1, <Étiquette>)
	```
**Rappel** : Un graphe de type "Nuage de points" utilise les données par groupe de deux séries (pour plus de détails, consultez la fonction [&lt;Graphe&gt;.NuageAjouteDonnéeXY](../WDLang3/1000024105.md)).

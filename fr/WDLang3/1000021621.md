


## &lt;Variable httpRequête&gt;.LitCookie (Fonction)

***En anglais : &lt;httpRequest variable&gt;.ReadCookie***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Retourne la valeur d'un cookie reçu suite à une requête HTTP.






<a name="SYNTAXE2"></a>

### Syntaxe utilisable avec la fonction HTTPEnvoie

`<Résultat> = <Requête HTTP>.LitCookie(<Domaine> , <Nom>)`
---

**`<Résultat> : (Chaîne de caractères)`**



- Valeur du cookie.

- Chaîne vide ("") si le cookie n'existe pas. La variable [ErreurDétectée](../WDLang1/3087001.md) est alors à <u><u><u>Vrai</u></u></u>.




**`<Requête HTTP> : (Variable de type httpRequête)`**

Nom de la variable [httpRequête](../WDLang3/1000021158.md) pour laquelle le cookie doit être lu.

**`<Domaine> : (Chaîne de caractères)`**

Nom du domaine pour lequel le cookie a été reçu.

**`<Nom> : (Chaîne de caractères)`**

Nom du cookie.


<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Pour utiliser la fonction **&lt;Variable httpRequête&gt;.LitCookie**, la gestion des cookies doit avoir été activée par la fonction [&lt;Variable httpRequête&gt;.GèreCookie](../WDLang3/1000021608.md).

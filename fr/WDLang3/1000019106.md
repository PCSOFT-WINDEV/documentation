
## SaaSAdminListeSite (Fonction)

***En anglais : SaaSAdminListSite***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Liste les sites SaaS déclarés dans la base d'un Webservice SaaS WEBDEV.

**Remarques** : 

- L'utilisation de cette fonction nécessite d'avoir ouvert une session d'administration SaaS au préalable en utilisant la fonction [SaaSAdminConnecte](../WDLang3/1000019098.md).

- L'Administrateur SaaS aura accès à tous les sites SaaS. Un gestionnaire de compte client aura accès uniquement aux sites auxquels il est abonné. 



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Initialisation d'une liste avec les sites SaaS
lstSite est un tableau de saasSite
lstSite = SaaSAdminListeSite()

POUR TOUT SiteSaaS DE lstSite
	ListeAjoute(LISTE_Sites, SiteSaaS.Nom)
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = SaaSAdminListeSite()`
---

**`<Résultat> : (Tableau de variables de type saasSite)`**

Liste des sites SaaS déclarés dans la base du Webservice.


## AlbumSauve (Fonction)

***En anglais : AlbumSave***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Sauve une image, une photo ou une vidéo dans l'album de photos de l'appareil mobile. Cette image pourra être réutilisée avec la fonction [AlbumSélecteur](../WDLang3/1000020186.md).




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Sauvegarde d'une image présente dans une variable de type image
MonImage est une Image = "Test.png"
AlbumSauve(MonImage)
```


<a name="Exemple2"></a>



```wl
// Sauvegarde d'une photo
SI AlbumSauve(VidéoLanceAppli(viCaptureImage)) = Faux ALORS
	Info("Image non sauvegardée")
FIN
```


<a name="Exemple3"></a>



<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Sauve une image présente dans une variable de type Image

`<Résultat> = AlbumSauve(<Image>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'enregistrement de l'image a réussi, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Image> : (Variable de type Image)`**

Nom de la variable de type [Image](../WDLang1/1000019650.md) correspondant à l'image à ajouter dans l'album.


<a name="SYNTAXE2"></a>

### Sauve une image présente dans un champ (Image ou Graphe)

`<Résultat> = AlbumSauve(<Nom du champ>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'enregistrement de l'image a réussi, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Nom du champ> : (Chaîne de caractères)`**

Nom du champ Image ou du champ Graphe contenant l'image à ajouter dans l'album.


<a name="SYNTAXE3"></a>

### Sauve une image présente dans un fichier

`<Résultat> = AlbumSauve(<Nom du fichier Image>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'enregistrement de l'image a réussi, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Nom du fichier Image> : (Chaîne de caractères)`**

Nom complet ou relatif du fichier image ou vidéo à ajouter dans l'album (par exemple, nom du fichier renvoyé par la fonction [VidéoLanceAppli](../WDLang1/1000019368.md)).

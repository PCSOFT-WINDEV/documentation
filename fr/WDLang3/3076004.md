
## telDuréeAppel (Fonction)

***En anglais : tapiCallDuring***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la durée de l'appel (différence entre la date et l'heure du début de l'appel et la date et l'heure de fin de l'appel).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
DuréeAppel est une Durée
DuréeAppel = telDuréeAppel()
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = telDuréeAppel([<Identifiant de l'appel>])`
---

**`<Résultat> : (Durée)`**

Durée de l'appel. Correspond à la durée en cours si l'appel est en cours, ou à la durée totale, si l'appel est terminé.

**`<Identifiant de l'appel> : (Entier optionnel)`**

Identifiant de l'appel à manipuler. Cet identifiant est connu dans la procédure WLangage appelée par la fonction [telDémarreDétectionAppel](../WDLang3/3076029.md). Si ce paramètre n'est pas spécifié, l'appel en cours est manipulé.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Disponibilité de l'information
<a name="disponibilite_information_ELTPARAGRAPHE000038"></a>L'information est disponible :

- à n'importe quel moment pour le dernier appel sortant.

- jusqu'à la fin de l'appel pour les appels entrants. Si cette information doit être stockée, il est nécessaire de la mémoriser avant la fin de l'appel.





<a name="NOTE0_2"></a>


### Configuration nécessaire
<a name="configuration_necessaire_ELTPARAGRAPHE000176"></a>Les fonctions de téléphonie utilisent la technologie TAPI 2.0 et/ou la technologie TAPI 3.1. Ces technologies sont utilisables sur tous les systèmes.

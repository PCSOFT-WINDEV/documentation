


## telTransfertDirect (Fonction)

***En anglais : tapiBlindTransfer***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Effectue un transfert d'appel "en aveugle". Aucune reprise de ligne n'est possible.
<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = telTransfertDirect(<Identifiant de l'appel> , <Numéro de transfert>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le transfert d'appel est effectué, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Identifiant de l'appel> : (Entier)`**

Identifiant de l'appel à transférer. Cet identifiant est connu dans la procédure WLangage appelée par la fonction [telDémarreDétectionAppel](../WDLang3/3076029.md).

**`<Numéro de transfert> : (Chaîne de caractères)`**

Numéro de téléphone à composer.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Limitation
<a name="limitation_ELTPARAGRAPHE000039"></a>Il n'est pas possible d'appeler la fonction **telTransfertDirect** dans la procédure de détection d'appels (définie grâce à la fonction [telDémarreDétectionAppel](../WDLang3/3076029.md)).
<a name="NOTE0_2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) 

### Transfert d'appel avec possibilité de reprise de la communication
<a name="transfert_appel_avec_possibilite_reprise_communication_ELTPARAGRAPHE000054"></a>Pour effectuer un transfert d'appel avec possibilité de reprise du numéro initial (par exemple si le numéro de transfert refuse de prendre la communication), utilisez la fonction [telExécuteTransfert](../WDLang3/3076021.md).
<a name="NOTE0_3"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) Les fonctions de téléphonie utilisent la technologie TAPI 2.0 et/ou la technologie TAPI 3.1. Ces technologies sont utilisables sur tous les systèmes.

Attention : le transfert d'appel n'est pas disponible pour tous les matériels. Le transfert d'appel est pris en compte uniquement dans le cas d'un PABX.

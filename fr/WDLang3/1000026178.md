


## &lt;Variable Email&gt;.ExporteHTML (Fonction)

***En anglais : &lt;Email variable&gt;.ExportHTML***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Récupère le code HTML d'un email pour l'afficher dans un champ HTML (champ HTML, champ d'affichage HTML, ...). Les images sont extraites sur le disque.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
oMailReçu est un Email
//...
// Lecture de la boîte mail pour renseigner oMailReçu
//...
EDHTML_Mail.RépertoireDeTravail = "C:\Temp\mail"
EDHTML_Mail = oMailReçu.ExporteHTML(EDHTML_Mail.RépertoireDeTravail)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Email>.ExporteHTML(<Répertoire>)`
---

**`<Résultat> : (Chaîne de caractères)`**

Code HTML pouvant être affecté à un champ HTML (champ HTML, champ d'affichage HTML, ...).

**`<Email> : (Variable de type Email)`**

Nom de la variable de type [Email](../WDLang3/1000018713.md) correspondant à l'email dont le code HTML doit être récupéré.

**`<Répertoire> : (Chaîne de caractères)`**

Chemin du répertoire dans lequel les images seront enregistrées. 
Remarque : Si au moment de la récupération, l'image existe déjà, un suffixe est ajouté au nom de l'image enregistrée sur le disque (_1, _2, ..., _n).



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Les images extraites sont référencées dans le code HTML résultat par une balise de type "./monimage.png". 

- Les références aux images de type &lt;img src="cid:dddd"&gt; sont remplacées par des références de type &lt;img src="./monimage.png"&gt;.

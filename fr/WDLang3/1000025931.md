
## ModbusDéconnecte (Fonction)

***En anglais : ModbusDisconnect***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ferme la session Modbus. 

Cette fonction déconnecte le poste client du serveur ou de l'esclave Modbus et libère le port série dans le cas d'une session RTU.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Crée une session et se connecte à l'esclave Modbus sur le port série COM4
Session est un modbusSession
Session = ModbusConnecteRTU("COM4")

// Si la connexion échoue, affiche l'erreur
SI ErreurDétectée = Vrai ALORS
	Trace(ErreurInfo())
SINON
	// Ecrit une valeur dans le registre 0x42 de l'esclave
	ModbusEcritRegistre(Session, 0x42, 0xCAFE)
FIN

// Déconnexion
ModbusDéconnecte(Session)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`ModbusDéconnecte(<Session>)`
---

**`<Session> : (Variable de type modbusSession)`**

Nom de la variable de type [modbusSession](../WDLang3/1000025924.md) correspondant à la session Modbus qui doit être fermée.




## Fonctions de gestion de l'album de photos
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions de gestion de l'album de photos sont les suivantes :



|   |   |
| --- | --- |
| [AlbumSauve](../WDLang3/1000020191.md) | Sauve une image, une photo ou une vidéo dans l'album de photos de l'appareil mobile. |
| [AlbumSélecteur](../WDLang3/1000020186.md) | Récupère une photo, une image ou une vidéo stockée dans l'album de photos de l'appareil mobile. |
| [PhotoLanceAppli](../WDLang3/1000021601.md) | Lance l'application caméra native de l'appareil afin de prendre une photo. |

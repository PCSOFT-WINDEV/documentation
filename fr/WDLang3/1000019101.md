
## SaaSAdminEstConnecté (Fonction)

***En anglais : SaaSAdminIsConnected***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Indique si une session d'administration SaaS a été ouverte.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI SaaSAdminEstConnecté() = Faux ALORS
	Erreur("Vous devez vous connecter en administrateur.")
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = SaaSAdminEstConnecté([<Niveau requis>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> dans les cas suivants : 

	- L'administrateur SaaS est connecté et aucun paramètre n'est spécifié. 

	- L'administrateur SaaS est connecté et la constante *saasAdministrateur* est spécifiée.

	- Un Gestionnaire de compte est connecté et la constante *saasGestionnaireDeCompte* est spécifiée.

	- L'un ou l'autre est connecté et les deux constantes sont spécifiées. 




- <u><u><u><u>Faux</u></u></u></u> sinon. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Niveau requis> : (Constante optionnelle ou combinaison de constantes optionnelle de type Entier)`**

Niveau requis pour se connecter : 


|   |   |
| --- | --- |
| *saasAdministrateur* <br>(valeur par défaut) | Niveau requis : Administrateur. |
| *saasGestionnaireDeCompte* | Niveau requis : Gestionnaire de compte. |

Ces deux constantes peuvent être combinées.

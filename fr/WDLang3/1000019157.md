


## Fonctions d'archivage (gestion des ZIP, WDZ, CAB, RAR, 7z, TAR et TGZ (TAR.GZ)) (syntaxe préfixée)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>










Les fonctions WLangage (syntaxe préfixée) permettant de créer et de gérer des archives sont les suivantes :




|   |   |
| --- | --- |
| [&lt;Variable zipArchive&gt;.AjouteFichier](../WDLang3/1000019050.md) | Ajoute et compresse automatiquement un fichier (de n'importe quel type) dans une archive de type CAB, ZIP, WDZ, 7z, TAR ou TGZ (TAR.GZ). |
| [&lt;Variable zipArchive&gt;.AjouteListeFichier](../WDLang3/1000019159.md) | Ajoute et compresse automatiquement une liste de fichiers (de n'importe quel type) dans une archive au format ZIP, CAB, WDZ ou 7z. |
| [&lt;Variable zipArchive&gt;.AjouteRépertoire](../WDLang3/1000019078.md) | Ajoute et compresse automatiquement la totalité des fichiers d'un répertoire et de ses sous-répertoires dans une archive. |
| [&lt;Variable zipArchive&gt;.ChangeChemin](../WDLang3/1000026329.md) | Modifie le chemin mémorisé d'un fichier d'une archive (au format ZIP, WDZ ou 7z). |
| [&lt;Variable zipArchive&gt;.ChercheFichier](../WDLang3/1000026330.md) | Recherche un fichier dans une archive (au format CAB, ZIP, RAR, WDZ ou 7z). |
| [&lt;Variable zipArchive&gt;.Crée](../WDLang3/1000019160.md) | Crée une archive. |
| [&lt;Variable zipArchive&gt;.CréeExe](../WDLang3/1000019079.md) | Crée une archive auto-extractible sous forme d'un exécutable en 32 bits (.EXE). |
| [&lt;Variable zipArchive&gt;.Découpe](../WDLang3/1000019161.md) | Découpe une archive en plusieurs sous-archives d'une taille donnée. |
| [&lt;Variable zipArchive&gt;.EstMulti](../WDLang3/1000019080.md) | Renvoie le type de l'archive : archive mono-partie ou archive multi-parties. |
| [&lt;Variable zipArchive&gt;.Existe](../WDLang3/1000026332.md) | Permet de savoir si une archive existe. |
| [&lt;Variable zipArchive&gt;.ExtraitChemin](../WDLang3/1000019081.md) | Renvoie le chemin d'un fichier de l'archive. |
| [&lt;Variable zipArchive&gt;.ExtraitFichier](../WDLang3/1000019082.md) | Extrait et décompresse automatiquement un fichier d'une archive à un emplacement physique ou en mémoire. |
| [&lt;Variable zipArchive&gt;.ExtraitListeFichier](../WDLang3/1000019084.md) | Extrait et décompresse automatiquement une liste de fichiers présents dans une archive vers un emplacement physique. |
| [&lt;Variable zipArchive&gt;.ExtraitRépertoire](../WDLang3/1000025755.md) | Extrait et décompresse automatiquement les fichiers présents dans un répertoire de l'arborescence des fichiers d'une archive vers un emplacement physique. |
| [&lt;Variable zipArchive&gt;.ExtraitTout](../WDLang3/1000019085.md) | Extrait et décompresse automatiquement la totalité des fichiers d'une archive à un emplacement physique. |
| [&lt;Variable zipArchive&gt;.Ferme](../WDLang3/1000019086.md) | Ferme une archive. |
| [&lt;Variable zipArchive&gt;.FichierEnCours](../WDLang3/1000019090.md) | Renvoie le nom du fichier archivé en cours de manipulation par les fonctions d'ajout et d'extraction de fichiers. |
| [&lt;Variable zipArchive&gt;.InfoFichier](../WDLang3/1000019092.md) | Renvoie les caractéristiques d'un fichier de l'archive :<br><br>- Le nom du fichier et son chemin mémorisé.<br><br>- La taille originale du fichier.<br><br>- La taille compressée du fichier.<br><br>- La date et l'heure de création du fichier.<br><br>- La date et l'heure de modification du fichier.<br><br>- La date et l'heure d'accès du fichier.<br><br>- Les attributs du fichier.<br><br>- Le mode de cryptage du fichier.<br><br><br> |
| [&lt;Variable zipArchive&gt;.ListeFichier](../WDLang3/1000019093.md) | Renvoie la liste des fichiers présents dans une archive. |
| [&lt;Variable zipArchive&gt;.NbPartieNécessaire](../WDLang3/1000019094.md) | Renvoie le nombre de parties d'une taille donnée nécessaires pour découper une archive mono-partie. |
| [&lt;Variable zipArchive&gt;.Ouvre](../WDLang3/1000019097.md) | Ouvre une archive existante. |
| [&lt;Variable zipArchive&gt;.OuvreCAB](../WDLang3/1000019148.md) | Ouvre une archive existante au format CAB. |
| [&lt;Variable zipArchive&gt;.OuvreRAR](../WDLang3/1000019150.md) | Ouvre une archive existante au format RAR. |
| [&lt;Variable zipArchive&gt;.Regroupe](../WDLang3/1000019151.md) | Regroupe les différentes sous-archives d'une archive multi-parties pour en faire une archive mono-partie. |
| [&lt;Variable zipArchive&gt;.SupprimeFichier](../WDLang3/1000019152.md) | Supprime un fichier contenu dans une archive. |
| [&lt;Variable zipArchive&gt;.SupprimeListeFichier](../WDLang3/1000019153.md) | Supprime un ensemble de fichiers contenu dans une archive. |
| [&lt;Variable zipArchive&gt;.SupprimeTout](../WDLang3/1000019154.md) | Supprime tous les fichiers présents dans une archive. |
| [&lt;Variable zipArchive&gt;.TailleFichier](../WDLang3/1000019155.md) | Renvoie la taille d'un des fichiers de l'archive (au format CAB, RAR, WDZ, ZIP, 7z, TAR ou TGZ (TAR.GZ)) :<br><br>- soit la taille du fichier compressé.<br><br>- soit la taille du fichier avant compression.<br><br><br> |
| [&lt;zipArchive&gt;.SélectionFichier](../WDLang3/1000026294.md) | Définit une sélection des fichiers contenus dans l'archive. |



**Remarque** : Les fonctions d'archivage peuvent être utilisées dans un thread sans bloquer les autres threads pendant leur exécution.

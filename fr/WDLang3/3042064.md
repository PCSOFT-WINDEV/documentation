


## grChargeParamètre (Fonction)

***En anglais : grLoadParameter***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Restaure les paramètres de dessin d'un graphe. Ces paramètres ont été sauvegardés par la fonction [grSauveParametre](../WDLang3/3042065.md).

**Rappel** : Par défaut, les paramètres d'un graphe modifiés par l'utilisateur sont sauvegardés lors de l'arrêt de l'application. Ces paramètres sont automatiquement restaurés au prochain lancement de l'application. Pour plus de détails, consultez la fonction [grCrée](../WDLang3/3042005.md).

![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Les graphes](../WDChamp/3042060.md). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Sauvegarde des paramètres d'un graphe créé par programmation
INIEcrit("Parametres", "MonGraphe", "Param", grSauveParamètre("MonGraphe"))

// Restauration des paramètres d'un graphe créé par programmation
grChargeParamètre("MonGraphe", INILit("Parametres", "MonGraphe", "Param"))
```
<a name="Exemple2"></a>

```wl
// Sauvegarde des paramètres d'un champ Graphe
INIEcrit("Parametres", "GRF_Graphe", "Param", grSauveParamètre(GRF_Graphe))

// Restauration des paramètres d'un champ Graphe
grChargeParamètre(GRF_Graphe, INILit("Parametres", "GRF_Graphe", "Param"))
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = grChargeParamètre(<Nom du graphe> , <Paramètres>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le chargement des paramètres a été effectué, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Nom du graphe> : (Nom de champ ou chaîne de caractères)`**

Nom du graphe à manipuler. Ce nom correspond :

- soit au nom du graphe défini par programmation avec la fonction [grCrée](../WDLang3/3042005.md).

- soit au nom du champ Graphe présent dans l'éditeur de fenêtres, de pages ou d'états.




**`<Paramètres> : (Chaîne de caractères Ansi)`**

Paramètre à restaurer. Ce paramètre est dans un format compressé. Il peut uniquement être obtenu par la fonction [grSauveParamètre](../WDLang3/3042065.md). La valeur de ce paramètre peut être sauvegardée dans un fichier, un mémo ou dans la base de registre.

Si ce paramètre correspond à une chaîne vide (""), la fonction n'a aucun effet et renvoie <u><u><u><u>Faux</u></u></u></u>.

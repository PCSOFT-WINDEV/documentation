
## OutlookOuvreSession (Fonction)

***En anglais : OutlookStartSession***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet d'accéder aux différentes données manipulées par le logiciel de messagerie Outlook (emails, contacts, groupes de contacts, tâches, rendez-vous et dossiers). Cette fonction est identique à la fonction [EmailOuvreSessionOutlook](../WDLang3/3032128.md). Pour fermer la session, il suffit d'utiliser la fonction [FermeSession](../WDLang3/3032124.md) ou la fonction [EmailFermeSession](../WDLang3/3032006.md).

**Attention** : Cette fonction ne permet pas d'accéder aux différentes données manipulées par le logiciel de messagerie Outlook Express.

**Remarque** : Lors de l'utilisation de la fonction **OutlookOuvreSession**, le répertoire en cours est automatiquement modifié par Outlook.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple
<a class="notetitle" target="_blank" href="$DOC$=1000003032123&name=outlookouvresession_fonction&product=WD">Voir des exemples supplémentaires</a>

```wl
// Ouverture de la session
SI OutlookOuvreSession("MonProfil") <> 0 ALORS
	Info("Session ouverte")
SINON
	Erreur(ErreurInfo())
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = OutlookOuvreSession(<Profil> [, <Options d'ouverture>])`
---

**`<Résultat> : (Entier)`**



- Identifiant de la session. 

- 0 ou <u><u><u><u>Faux</u></u></u></u> en cas d'erreur. Pour obtenir le libellé de l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.




**`<Profil> : (Chaîne de caractères)`**

Identifie le profil de la connexion.



- Le profil par défaut peut être connu grâce à la fonction [OutlookProfilDéfaut](../WDLang3/1000019347.md). 

- La liste des profils disponibles peut être connue grâce à la fonction [OutlookListeProfil](../WDLang3/1000019346.md). 




Si ce paramètre correspond à une chaîne vide ("") :

- si plusieurs profils sont définis sur le poste en cours, la fonction **OutlookOuvreSession** affiche une fenêtre contenant tous les profils dans laquelle l'utilisateur peut choisir son profil.

- si un seul profil est défini sur le poste en cours, la fonction **OutlookOuvreSession** sélectionne automatiquement ce profil.




**`<Options d'ouverture> : (Constante de type entier optionnelle)`**

Mode d'ouverture de la session Outlook : 


|   |   |
| --- | --- |
| *outlookRDVAvecContact*<br>(Valeur par défaut) | Les fonctions RendezVousXXX listeront les contacts associés aux rendez-vous dans la variable mRendezVous.Participant (voir [Structure RendezVous](../WDLang3/1000011010.md)). |
| *outlookRDVAvecParticipant* | Les fonctions RendezVousXXX listeront les participants associés aux rendez-vous dans les variables :<br><br>- mRendezVous.Participant<br><br>- mRendezVous.ParticipantCc<br><br>- mRendezVous.ParticipantCci<br><br><br>Pour plus de détails, consultez [Structure RendezVous](../WDLang3/1000011010.md). |

## Remarques




### Création d'un profil
<a name="creation_profil_ELTPARAGRAPHE000108"></a>Pour ouvrir une nouvelle session avec Outlook, il est nécessaire de définir un "profil". Ce "profil" est défini dans la configuration de la connexion Internet avec Outlook.

Pour créer un profil :

1. Exécutez Outlook.

2. Dans la fenêtre "Choix d'un profil", cliquez sur [Nouveau].

3. Sélectionnez le service "Microsoft Exchange Server".

4. Donnez un nom au profil.

5. Donnez le nom du serveur Microsoft Exchange.





<a name="NOTE0_2"></a>




### Répertoire en cours
<a name="repertoire_cours_ELTPARAGRAPHE000131"></a>**Attention** : La fonction **OutlookOuvreSession** modifie le chemin du répertoire en cours (renvoyé par la fonction [fRepEnCours](../WDLang1/3036022.md)).

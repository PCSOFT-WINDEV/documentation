
## OLECharge (Fonction)

***En anglais : OLELoad***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Initialise un champ OLE avec un objet OLE. Le serveur OLE est invoqué pour charger l'objet OLE. L'objet OLE apparaît dans le champ : le champ est mis à jour. Le serveur n'est pas lancé.

**Remarque** : Selon le serveur qui a créé l'objet OLE, l'objet peut s'afficher dans le champ sous forme de :

- sa représentation bitmap. Exemple : image créée avec Paint Brush.

- sa représentation vectorielle. Exemple : feuille de calcul créée avec Excel.

- l'icône représentant le serveur. Exemple : icône WinWord.

- texte.



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Chargement de l'objet
OLECharge(OLE_OLE2, "OBJET1.OLE")

// Traitement de l'objet

 // Sauvegarde de l'objet
OLESauve(OLE_OLE2, "OBJET1.OLE")
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = OLECharge(<Champ OLE> , <Nom du fichier>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'objet OLE a été chargé, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Champ OLE> : (Nom de champ)`**

Nom du champ OLE à initialiser. Si ce paramètre correspond à une chaîne vide (""), le champ considéré est le champ auquel appartient l'événement en cours.

**`<Nom du fichier> : (Chaîne de caractères)`**

Nom de l'objet OLE. Cet objet doit avoir été créé avec la fonction [OLESauve](../WDLang3/3057011.md).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Connaître le nom du fichier OLE associé à un champ
<a name="connaitre_nom_fichier_ole_associe_champ_ELTPARAGRAPHE000054"></a>Pour connaître le nom du fichier OLE associé à un champ, la syntaxe est la suivante :

```wl
NomFichier = OLE_OLE1 // OLE_OLE1 est le nom du champ
```



<a name="NOTE0_2"></a>




### Modifier le contenu d'un champ OLE
<a name="modifier_contenu_champ_ole_ELTPARAGRAPHE000064"></a>Pour modifier le nom du fichier associé à un champ, la syntaxe est la suivante :


```wl
OLE_OLE1 = NomFichier // OLE_OLE1 est le nom du champ
```


Cette ligne de code équivaut à **OLECharge**(OLE_OLE1, NomFichier).


<a name="NOTE0_3"></a>




### Afficher le contenu d'un mémo binaire OLE 
<a name="afficher_contenu_memo_binaire_ole_ELTPARAGRAPHE000080"></a>Pour afficher le contenu d'un mémo binaire de type OLE, la syntaxe est la suivante :

```txt
OLECharge(NomChamp, NomFichierDeDonnées.NomRubrique)
```


Par exemple : 


```wl
OLECharge(OLE_OLE1, Produit.OLEMémo)
```



<a name="NOTE0_4"></a>




### Divers
<a name="divers_ELTPARAGRAPHE000094"></a>Seul OLE version 1 est géré.

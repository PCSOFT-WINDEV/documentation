
## SaaSAdminDésactiveService (Fonction)

***En anglais : SaaSAdminPricingDisableService***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Désactive un service dans une tarification SaaS.

**Remarques** : 

- L'utilisation de cette fonction nécessite d'avoir ouvert une session d'administration SaaS au préalable en utilisant la fonction [SaaSAdminConnecte](../WDLang3/1000019098.md).

- L'utilisation de cette fonction est réservée à l'Administrateur SaaS. Un gestionnaire de compte client ne peut pas utiliser cette fonction. 



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
S est un saasSite dynamique 
T est une saasTarification dynamique 
ServiceCRM est un saasService dynamique 

S = SaaSAdminChercheSite("Forum")
T = SaaSAdminChercheTarification("Promo 2010", S)
ServiceCRM = SaaSAdminChercheService("CRM", S)

// Désactive le service CRM pour les abonnés de "Promo 2010"
SaaSAdminDésactiveService(T, ServiceCRM)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Désactiver un service (utilisation des variables de type saasxxx)

`<Résultat> = SaaSAdminDésactiveService(<Tarification> , <Service>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la désactivation a été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Tarification> : (Variable de type saasTarification)`**

Nom de la variable de type [saasTarification](../WDLang3/1000019091.md) correspondant à la tarification à manipuler. Le service sera retiré de cette tarification.

**`<Service> : (Variable de type saasService)`**

Nom de la variable de type [saasService](../WDLang3/1000019096.md) identifiant le service qui devra être retiré de la liste des services fournis par la tarification.


<a name="SYNTAXE2"></a>

### Désactiver un service identifié par son nom

`<Résultat> = SaaSAdminDésactiveService(<Tarification> , <Service> , <Site SaaS>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la désactivation a été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Tarification> : (Chaîne de caractères)`**

Nom de la tarification. Le service devra être retiré de cette tarification.

**`<Service> : (Chaîne de caractères)`**

Nom du service qui devra être retiré de la liste des services fournis par la tarification.

**`<Site SaaS> : (Chaîne de caractères)`**

Nom du site SaaS concerné par la tarification et le service.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Le service sera automatiquement rendu inaccessible à tous les abonnés à cette tarification.

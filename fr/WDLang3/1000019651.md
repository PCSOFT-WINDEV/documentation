
## Comment intégrer les services SESAM VITALE ? 
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000071"></a>
A partir de la version 18 de WINDEV, l'importation des webservices intègre les outils nécessaires pour permettre la consommation des Webservices sécurisés utilisés pour gérer les cartes Sesam Vitale.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000095"></a>
L'utilisation d'une carte Sesam Vitale dans une application n'utilise pas un jeu d'instructions WLangage spécifiques : il n'y a pas de fonction Sesam\*. 

La gestion des cartes Sesam Vitale est effectuée par l'intermédiaire de Webservices : il suffit d'importer les webservices requis par la fonctionnalité à implémenter.

Ces webservices sont fournis par le GIE SESAM-VITALE, au travers d'un processus d'agrément des logiciels qui vont les exploiter. Cette mission d'agrément est assurée par le CNDA (Centre National de Dépôt et d'Agrément). Pour plus de détails, consultez [http://www.cnda.ameli.fr/](http://www.cnda.ameli.fr/)

La réalisation de solutions applicatives utilisant les Webservices Sesam Vitale nécessite donc de s'enregistrer auprès du CNDA, afin d'entreprendre le processus d'agrément nécessaire. Le site du CNDA fournit l'ensemble des éléments administratifs et techniques nécessaires pour cette démarche.

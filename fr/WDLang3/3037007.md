
## FaxDéconnecte (Fonction)

***En anglais : FaxDisconnect***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ferme une connexion à un serveur de fax (précédemment ouverte avec la fonction [FaxConnecte](../WDLang3/3037005.md)).

**Remarque** : Cette fonction ne peut être utilisée que sous Windows 2000, XP ou version supérieure. Sous Windows Vista (et supérieur), la gestion des fax est disponible uniquement avec Windows Professionnel et Windows Edition Intégrale.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
IDConnect est un entier
IDConnect = FaxConnecte()
...
SI FaxDéconnecte(IDConnect) = Faux ALORS 
	Erreur("La déconnexion a échoué.", ErreurInfo(errMessageSystème))
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = FaxDéconnecte(<Identifiant connexion>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la déconnexion a été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> en cas d'erreur. Le message d'erreur correspondant peut être obtenu avec la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Identifiant connexion> : (Entier)`**

Identifiant de la connexion obtenu avec la fonction [FaxConnecte](../WDLang3/3037005.md).

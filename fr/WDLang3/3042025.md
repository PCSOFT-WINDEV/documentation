


## grExiste (Fonction)

***En anglais : grExist***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Teste l'existence d'un graphe en mémoire. 

Le graphe manipulé peut être :

- soit un graphe créé par programmation (fonction [grCrée](../WDLang3/3042005.md)).

- soit un champ Graphe dans l'éditeur de fenêtres, de pages ou d'états.




![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Les graphes](../WDChamp/3042060.md). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Teste l'existence du graphe "MonGraphe"
ResExiste = grExiste("MonGraphe")
```
<a name="Exemple2"></a>

```wl
// Teste l'existence du graphe GRF_MonGraphe
ResExiste = grExiste(GRF_MonGraphe)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = grExiste(<Nom du graphe>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le graphe existe, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Nom du graphe> : (Nom de champ ou chaîne de caractères)`**

Nom du graphe à manipuler. Ce nom correspond :

- soit au nom du graphe défini par programmation avec la fonction [grCrée](../WDLang3/3042005.md).

- soit au nom du champ Graphe présent dans l'éditeur de fenêtres, de pages ou d'états.

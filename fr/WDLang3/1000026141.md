
## Fonctions WebSocket (syntaxe préfixée)

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions utilisables dans un projet de type Serveur de WebSocket (syntaxe préfixée) sont les suivantes : 



|   |   |
| --- | --- |
| [&lt;Variable websocketClient&gt;.Déconnecte](../WDLang3/1000026067.md) | Déconnecte un client actuellement connecté sur un serveur de WebSocket. |
| [&lt;Variable websocketClient&gt;.Envoie](../WDLang3/1000026068.md) | Envoie un message pour un client du serveur de WebSocket. |
| [&lt;Variable websocketClient&gt;.Exécute](../WDLang3/1000026113.md) | Exécute une procédure globale dans le contexte d'un client connecté au serveur de WebSocket. |

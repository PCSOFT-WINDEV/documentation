


## &lt;Graphe&gt;.Lissage (Fonction)

***En anglais : &lt;Chart&gt;.Smoothing***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Lisse un graphe de type Courbe ou Nuage de points par la méthode des "splines cubiques". 

**Rappel** : Par défaut, un graphe est non lissé.



![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Les graphes](../WDChamp/3042060.md).








<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Lisse le graphe affiché dans le champ GRF_MonGraphe
GRF_MonGraphe.Lissage(Vrai)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Lisser un graphe

`<Champ Graphe>.Lissage(<Lissage>)`
---

**`<Champ Graphe> : (Nom de champ)`**

Nom du champ Graphe à manipuler (présent dans l'éditeur de fenêtres, de pages ou d'états).


**`<Lissage> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le lissage doit être activé, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.





<a name="SYNTAXE2"></a>

<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Le lissage est pris en compte :

- dans les graphes de type courbe (constante *grCourbe*),

- dans les graphes de type nuage de points (constante *grNuage*) dont les points sont reliés (fonction [&lt;Graphe&gt;.NuageReliePoint](../WDLang3/1000024109.md)).




**Attention** : Cette fonction n'a aucun effet sur un graphe 3D.

Les paramètres définis avec la fonction **&lt;Graphe&gt;.Lissage** seront pris en compte lors du prochain appel à la fonction [&lt;Graphe&gt;.Dessine](../WDLang3/1000024062.md).

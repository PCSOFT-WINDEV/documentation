
## WebserviceLitEntêteHTTP (Fonction)

***En anglais : WebserviceReadHTTPHeader***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Lit l'entête HTTP reçu lors d'un appel à un webservice (SOAP ou REST). Ces entêtes HTTP sont envoyées par le client au serveur dans la requête courante.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Valide l'origine de la requête (protocole CORS)
sOrigine est une chaîne = WebserviceLitEntêteHTTP("Origin")
// Validation de l'origine
...
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Lire l'entête HTTP identifiée par son nom

`<Entête HTTP> = WebserviceLitEntêteHTTP(<Nom de l'entête à lire>)`
---

**`<Entête HTTP> : (Chaîne de caractères)`**

Valeur de l'entête HTTP. L'entête est formatée comme suit:

- Le nom de l'entête est renvoyée en majuscules.

- Les tirets sont remplacés par des "underscores" (le nom original de l'en-tête HTTP tel que envoyé par le client n'est pas disponible). 

- Une chaîne vide ("") est retournée dans le cas où il n'y a plus d'entêtes à lire.




**`<Nom de l'entête à lire> : (Chaîne de caractères)`**

Nom de l'entête à lire.


<a name="SYNTAXE2"></a>

### Lire l'entête HTTP identifiée par son indice

`<Entête HTTP> = WebserviceLitEntêteHTTP(<Indice>)`
---

**`<Entête HTTP> : (Chaîne de caractères)`**

Valeur de l'entête HTTP. L'entête est formatée comme suit:

- Le nom de l'entête est renvoyée en majuscules.

- Les tirets sont remplacés par des "underscores" (le nom original de l'en-tête HTTP tel que envoyé par le client n'est pas disponible). 

- Une chaîne vide ("") est retournée dans le cas où il n'y a plus d'entêtes à lire.




**`<Indice> : (Entier)`**

Indice de l'entête à lire. Pour lire toutes les entêtes HTTP, il faut incrémenter cet indice jusqu'à ce que la fonction retourne une chaîne vide.

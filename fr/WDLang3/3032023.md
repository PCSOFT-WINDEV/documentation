


## EmailRAZ (Fonction)

***En anglais : EmailReset***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ré-initialise toutes les variables de la structure [email](../WDLang3/3032029.md) ou d'une variable de type [Email](../WDLang3/1000018713.md).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ré-initialiser toutes les variables de la structure email 
EmailRAZ()
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`EmailRAZ([<Email>])`
---

**`<Email> : (Variable optionnelle de type Email)`**

Nom de la variable [Email](../WDLang3/1000018713.md) à réinitialiser. Si ce paramètre n'est pas spécifiée, la structure [Email](../WDLang3/3032029.md) sera utilisée.

***En anglais : grDonutHolePercentage***

	<IMG style="float:left;margin-right:10px;" src="https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=warning_big.png"><B>Avertissement</B><BR>

<a name="Utilisation"></a>
<a name="description"></a>
Modifie le pourcentage correspondant au rayon du trou d'un graphe de type Beignet (ou "Donut"). 
 
![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Les graphes](../WDChamp/3042060.md). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Le trou représentera 50% du rayon total du donut/beignet
grDonutPourcentageTrou(GRF_MonGraphe, 50)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`grDonutPourcentageTrou(<Nom du graphe> , <Pourcentage>)`
---

**`<Nom du graphe> : (Nom de champ ou chaîne de caractères)`**

Nom du graphe à manipuler. Ce nom correspond :

- soit au nom du graphe défini par programmation avec la fonction [grCrée](../WDLang3/3042005.md).

- soit au nom du champ Graphe (présent dans l'éditeur de fenêtres, de pages ou d'états).




**`<Pourcentage> : (Entier)`**

Pourcentage représenté par le trou du donut/beignet (compris entre 0 et 70).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Le paramétrage réalisé avec la fonction **grDonutPourcentageTrou** sera pris en compte lors du prochain appel à la fonction [grDessine](../WDLang3/3042026.md).

- Cette fonction peut être utilisée uniquement avec les graphes de type Beignet.

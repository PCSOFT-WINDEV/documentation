
## Gérer les accès distants
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000124"></a>
Les accès distants permettent de créer une connexion réseau non permanente avec certains outils.

WINDEV permet de gérer les accès distants en proposant plusieurs fonctions permettant d'ouvrir, de fermer ces accès distants, mais aussi de les lister et d'obtenir des informations sur ces accès.

Quelques exemples "classiques" d'utilisation d'accès distants :

- Utilisation d'un modem

- Utilisation d'un VPN (réseau privé virtuel)

- ...






<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Fonctions de gestion des accès distants
<a name="fonctions_gestion_des_acces_distants_ELTTEXTE000148"></a>
Les fonctions WLangage suivantes permettent de gérer les accès distants :



|   |   |
| --- | --- |
| [NetFermeAccèsDistant](../WDLang3/3056021.md) | Ferme la ligne précédemment ouverte par la fonction [NetOuvreAccèsDistant](../WDLang3/3056026.md). |
| [NetInfoAccèsDistant](../WDLang3/3056028.md) | Renvoie des informations sur l'état d'une connexion ainsi que des statistiques sur cette connexion. |
| [NetListeAccèsDistant](../WDLang3/3056027.md) | Renvoie la liste des accès distants disponibles. |
| [NetOuvreAccèsDistant](../WDLang3/3056026.md) | Établit une connexion distante. |

## Spécificités
<a name="specificites_ELTTEXTE000205"></a>


### Connexion pour l'accès réseau à distance
<a name="connexion_pour_acces_reseau_distance_ELTPARAGRAPHE000036"></a>La création d'une connexion pour l'accès réseau à distance est réalisée dans le panneau de configuration sous Windows. Les informations suivantes seront demandées lors de la création de la connexion :

- le libellé de la connexion (utilisé par la fonction [NetOuvreAccèsDistant](../WDLang3/3056026.md)),

- le numéro de téléphone à composer,

- le nom de login ou d'utilisateur,

- le modem, le protocole, etc.


Ces données sont fournies par le fournisseur d'accès.

Lors de l'exécution de la connexion par un simple double clic sur l'icône correspondante, il est possible de spécifier un mot de passe pour cette connexion. Ce mot de passe peut être enregistré afin d'être utilisé à chaque lancement de cette connexion.


<a name="NOTE3_2"></a>


### Gestion de la fermeture automatique d'un accès distant
<a name="gestion_fermeture_automatique_acces_distant_ELTPARAGRAPHE000056"></a>La fonction [NetOuvreAccèsDistant](../WDLang3/3056026.md) permet de spécifier si l'accès distant ouvert doit être fermé ou non automatiquement à la fermeture de l'application.

Si la fermeture automatique est demandée, la fermeture de la connexion sera automatique à la fin de l'application ou lors de l'appel à la fonction [NetFermeAccèsDistant](../WDLang3/3056021.md) sans paramètre. Dans ce cas, la fermeture sera valable uniquement pour la dernière connexion ouverte par [NetOuvreAccèsDistant](../WDLang3/3056026.md).

Si la fermeture automatique n'est pas demandée, la fermeture de l'application ne fermera pas la connexion. Cette option permet par exemple de faire une application qui ouvre une connexion.
Dans ce cas, la fonction [NetFermeAccèsDistant](../WDLang3/3056021.md) sans paramètres ne fermera pas la connexion. Pour fermer cette connexion, il sera nécessaire de préciser le nom de l'accès en paramètre.

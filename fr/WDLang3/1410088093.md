


## &lt;Graphe&gt;.CPSensDeRotation (Fonction)

***En anglais : &lt;Chart&gt;.RBRotationDirection***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Fixe le sens de rotation d'un graphe circulaire de progrès.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
GRF_Graphe1.CPSensDeRotation(grSensAntihoraire)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Champ Graphe>.CPSensDeRotation(<Rotation>)`
---

**`<Champ Graphe> : (Nom de champ)`**

Nom du champ Graphe manipulé (présent dans l'éditeur de fenêtres ou d'états).

**`<Rotation> : (Constante de type Entier)`**

Sens de rotation du graphe : 


|   |   |
| --- | --- |
| *grSensAntihoraire* | Les données du graphe circulaire seront affichées en suivant le sens inverse de rotation des aiguilles d'une montre. |
| *grSensHoraire* | Les données du graphe circulaire seront affichées en suivant le sens de rotation des aiguilles d'une montre. |

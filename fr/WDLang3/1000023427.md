


## Fonctions Beacon
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions permettant de gérer les balises Beacon sont les suivantes : 



|   |   |
| --- | --- |
| [BeaconArrêteDétectionEnArrièrePlan](../WDLang3/1000023114.md) | Arrête une ou plusieurs détections de balises Beacon en arrière-plan. |
| [BeaconArrêteDétectionPrécise](../WDLang3/1000023116.md) | Arrête la détection de balises Beacon lancée avec la fonction [BeaconDétectePrécis](../WDLang3/1000023115.md). |
| [BeaconDétecteEnArrièrePlan](../WDLang3/1000023112.md) | Permet à l'application d'être notifiée lorsque l'appareil entre ou sort d'une zone correspondant au rayon d'émission d'un ensemble de balises Beacon. |
| [BeaconDétectePrécis](../WDLang3/1000023115.md) | Permet de trouver les balises Beacon à proximité de l'appareil. |
| [BeaconListeDétectionEnArrièrePlan](../WDLang3/1000023113.md) | Retourne la liste des détections actives de balises Beacon lancées en arrière-plan avec la fonction [BeaconDétecteEnArrièrePlan](../WDLang3/1000023112.md). |

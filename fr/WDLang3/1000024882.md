


## grComparatifPositionSérie (Fonction)

***En anglais : grComparativeSeriesPosition***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Fixe la position (gauche ou droite) d'une série dans un graphe de type Histogramme comparatif. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
grComparatifPositionSérie(GRF_SansNom1, 1, grADroite)
grComparatifPositionSérie(GRF_SansNom1, 2, grAGauche)

grCouleurSérie(GRF_SansNom1, 1, DonneCouleur())
grCouleurSérie(GRF_SansNom1, 2, DonneCouleur())

rDroite est un réel
rGauche est un réel

nRayon est un entier = 250

POUR n = 1 À 360
	rDroite = nRayon + nRayon * Cosinus(n)
	rGauche = nRayon + nRayon * Sinus(n + 180)
	grAjouteDonnée(GRF_SansNom1, 1, n, rDroite)
	grAjouteDonnée(GRF_SansNom1, 2, n, rGauche)
	grEtiquetteCatégorie(GRF_SansNom1, n, n)
FIN

grDessine(GRF_SansNom1)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`grComparatifPositionSérie(<Nom du graphe> , <Numéro de la série> , <Position>)`
---

**`<Nom du graphe> : (Nom de champ ou chaîne de caractères)`**

Nom du graphe à manipuler. Ce nom correspond :

- soit au nom du graphe défini par programmation avec la fonction [grCrée](../WDLang3/3042005.md).

- soit au nom du champ Graphe présent dans l'éditeur de fenêtres.




**`<Numéro de la série> : (Entier)`**

Numéro de la série à modifier. 

**`<Position> : (Entier)`**

Position de la série manipulée dans le graphe :


|   |   |
| --- | --- |
| *grADroite* | Série affichée à droite du graphe. |
| *grAGauche* | Série affichée à gauche du graphe. |

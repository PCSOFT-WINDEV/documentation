
## NFCEtat (Fonction)

***En anglais : NFCStatus***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Retourne l'état d'activation du capteur NFC de l'appareil.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = NFCEtat()`
---

**`<Résultat> : (Constante de type Entier)`**

Etat d'activation du capteur : 


|   |   |
| --- | --- |
| *nfcActivé* | Capteur NFC activé.<br><br> |
| *nfcDésactivé* | Capteur NFC désactivé. |
| *nfcIndisponible* | L'appareil ne possède pas de capteur NFC ou celui-ci est indisponible. |







<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Cas particuliers
<a name="cas_particuliers_ELTPARAGRAPHE000060"></a>

- La fonction **NFCEtat** ne peut être utilisée ni dans le simulateur ni dans l'émulateur : la fonction retournera la constante *nfcIndisponible*.

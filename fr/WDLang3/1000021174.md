


## grBourseRécupèreDonnée (Fonction)

***En anglais : grStockGetData***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Récupère une donnée dans un graphe boursier (chandelier, barcharts ou minmax). 
![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Les graphes](../WDChamp/3042060.md). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affichage d'une donnée dans le champ "GRF_MonGraphe". 
// Cette donnée est la première valeur minimale de la première série.
Info(grBourseRécupèreDonnée(GRF_MonGraphe, 1, 1, grValeurMin))
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = grBourseRécupèreDonnée(<Nom du graphe> , <Numéro de la série> , <Indice> , <Valeur à récupérer>)`
---

**`<Résultat> : (Réel)`**

Valeur récupérée.

**`<Nom du graphe> : (Nom de champ ou chaîne de caractères)`**

Nom du graphe à manipuler. Ce nom correspond :

- soit au nom du graphe défini par programmation avec la fonction [grCrée](../WDLang3/3042005.md). Par défaut, un graphe créé avec la fonction [grCrée](../WDLang3/3042005.md) ne contient aucune donnée.

- soit au nom du champ Graphe (présent dans l'éditeur de fenêtres, de pages ou d'états).




**`<Numéro de la série> : (Entier)`**

Numéro de la série pour laquelle une valeur doit être récupérée. 
Si la série spécifiée n'existe pas, la valeur récupérée est 0.

**`<Indice> : (Entier)`**

Indice de la donnée dans la série pour laquelle la valeur doit être récupérée. 
Si la donnée correspondant à l'indice spécifié n'existe pas dans la série, la valeur récupérée est 0.

**`<Valeur à récupérer> : (Constante de type Entier)`**

Indique la valeur à récupérer :



|   |   |
| --- | --- |
| *grValeurDébut* | Valeur de début. |
| *grValeurFin* | Valeur de fin. |
| *grValeurMax* | Valeur maximale. |
| *grValeurMin* | Valeur minimale. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Pour récupérer une donnée dans un graphe de type Nuage de points, il est conseillé d'utiliser la fonction [grNuageRécupèreDonnéeXY](../WDLang3/1000021175.md).

- Pour récupérer une donnée dans un graphe de type Secteur multiniveau, il est conseillé d'utiliser la fonction [grSMNRécupèreDonnée](../WDLang3/1000021287.md). 

- Pour récupérer une donnée dans un graphe Surface, il est conseillé d'utiliser la fonction [grSurfaceRécupèreDonnée](../WDLang3/1000021179.md).

- Pour récupérer une donnée dans un graphe de type Heatmap, il est conseillé d'utiliser la fonction [grHMRécupèreDonnée](../WDLang3/1000021632.md).

- Pour récupérer une donnée dans un autre type de graphe (Secteur, Histogramme, Courbe, etc.), utilisez la fonction [grRécupèreDonnée](../WDLang3/1000021171.md).

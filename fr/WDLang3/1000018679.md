
## zipArchive (Type de variable)

***En anglais : zipArchive***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **zipArchive** permet de manipuler un fichier d'archive correspondant à l'un des formats suivants : ZIP, CAB, RAR, WDZ, 7z, TAR ou TGZ (TAR.GZ). Les caractéristiques de ce fichier peuvent être définies et modifiées à l'aide de différentes propriétés WLangage.



**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Réalisation d'une archive de sauvegarde du répertoire "Mes Documents"
MonArchive est un zipArchive

// Protection de l'archive par un mot de passe
MonArchive.MotDePasse = "secret"

// Création de l'archive
zipCrée(MonArchive, "s:\Sauvegardes\" + DateVersChaîne(DateSys(), "AAAA-MM-JJ") + ".zip")

// Ajout des fichiers
zipAjouteRépertoire(MonArchive, SysRep(srMesDocuments))

// Fermeture de l'archive
zipFerme(MonArchive)
```

## Remarques


### Propriétés spécifiques à la description des variables de type zipArchive
<a name="proprietes_specifiques_description_des_variables_type_ziparchive_ELTPARAGRAPHE000062"></a>Les propriétés suivantes peuvent être utilisées pour manipuler une variable de type **zipArchive** :

| **Nom** | **Type manipulé** | **Effet** |
| --- | --- | --- |
| Fichier | Tableau de [zipFichierArchivé](../WDLang3/1000019044.md) | Tableau des fichiers présents dans l'archive. |
| FormatArchive | Constante de type Entier | Format du fichier d'archive :<br><br>- *zipFormatCAB* : Fichier au format CAB<br><br>- *zipFormatRAR* : Fichier au format RAR<br><br>- *zipFormatWDZ* : Fichier au format WDZ<br><br>- *zipFormatZIP* : Fichier au format ZIP<br><br>- *zipFormat7Z* : Fichier au format 7z<br><br>- *zipFormatTAR* : Fichier au format TAR <br><br>- *zipFormatTARGZ* : Fichier au format TGZ (TAR.GZ).  <br><br><br> |
| MotDePasse | Chaîne de caractères | Mot de passe à utiliser pour les ajouts ou les extractions de fichiers de l'archive. <br>Pour plus de détails sur le fonctionnement des mots de passe sur les archives, consultez la fonction [zipMotDePasse](../WDLang3/3082053.md).<br>Cette propriété est disponible en écriture uniquement.<br><br>Format 7z : Il n'est pas possible d'utiliser un mot de passe.<br>Format TAR et TGZ (TAR.GZ) : Il n'est pas possible d'utiliser un mot de passe. |
| NbFichier | Entier | Nombre de fichiers présents dans l'archive.<br>**Cette propriété est disponible en lecture seulement**. |
| NbPartie | Entier | Nombre de parties d'une archive multi-volumes.<br>Dans le cas d'une archive mono-volume, cette propriété est égale à 1.<br>**Cette propriété est disponible en lecture seulement**.<br> |
| NiveauCompression | Entier | Niveau de compression à utiliser pour les ajouts de fichiers dans l'archive.<br>Pour plus de détails, consultez la fonction [zipNiveauCompression](../WDLang3/3082054.md). |
| Taille | Entier sur 8 octets | Taille totale des fichiers contenus dans l'archive (taille décompressée).<br>**Cette propriété est disponible en lecture seulement**. |
| TailleCompressée | Entier sur 8 octets | Taille totale compressée des fichiers contenus dans l'archive.<br>**Cette propriété est disponible en lecture seulement**. |

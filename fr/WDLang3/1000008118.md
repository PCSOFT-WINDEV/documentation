
## GroupeSupprime (Fonction)

***En anglais : GroupDelete***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime le groupe de contacts en cours du carnet d'adresses d'une messagerie Lotus Notes ou Outlook.

**Attention** : Cette fonction **ne permet pas** de supprimer le groupe de contacts en cours du carnet d'adresses d'une messagerie Outlook Express.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ouverture d'une session Lotus Notes
IDSession est un entier
IDSession = EmailOuvreSessionNotes("MotDePasse", "ServeurMars", ...
	"mail\julie.nsf", "C:\Users\Julie\AppData\Local\Lotus\Notes\Data\user.id")
// Lecture du premier groupe de contacts
GroupePremier(IDSession)
// Suppression de ce groupe
GroupeSupprime(IDSession)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = GroupeSupprime(<Identifiant de la session>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la suppression a été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour connaître le libellé de l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*. 




**`<Identifiant de la session> : (Entier)`**

Identifiant de la session à utiliser. Cet identifiant est renvoyé par la fonction [EmailOuvreSessionNotes](../WDLang3/3032121.md), [EmailOuvreSessionOutlook](../WDLang3/3032128.md), [NotesOuvreBase](../WDLang3/1000005101.md) ou [OutlookOuvreSession](../WDLang3/3032123.md).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Version de Outlook utilisée
<a name="version_outlook_utilisee_ELTPARAGRAPHE000059"></a>Les groupes de contacts ne sont pas gérés par Outlook version 97. Il n'est donc pas possible de supprimer un groupe de contacts dans cette version de Outlook.

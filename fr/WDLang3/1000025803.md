


## &lt;Carte&gt;.FermePopup (Fonction)

***En anglais : &lt;Map&gt;.ClosePopup***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ferme la popup d'un marqueur affichée dans le champ Carte.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
CARTE_MaCarte.FermePopup(MonMarqueur)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Champ Carte>.FermePopup(<Marqueur>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la popup a pu être fermée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Champ Carte> : (Nom de champ)`**

Nom du champ Carte à manipuler.

**`<Marqueur> : (Variable de type Marqueur)`**

Nom de la variable de type [Marqueur](../WDLang3/1000019940.md) pour laquelle la popup associée doit être fermée. Ce marqueur doit exister sur le champ Carte.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Pour afficher la popup, utilisez la fonction [&lt;Carte&gt;.AffichePopup](../WDLang3/1000025799.md).

- Pour savoir si la popup d'un marqueur est affichée, utilisez la fonction [&lt;Carte&gt;.PopupAffichée](../WDLang3/1000025819.md).

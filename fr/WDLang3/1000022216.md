


## &lt;Variable btlePériphérique&gt;.RécupèreTousServices (Fonction)

***En anglais : &lt;btleDevice variable&gt;.GetAllServices***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Récupère tous les services d'un périphérique Bluetooth Low Energy correspondant à un identifiant (UUID).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonPériph est un btlePériphérique
...
tabService est un tableau de btleService
tabService = MonPériph.RécupéreTousServices("ACC54-DAD4D5-DADD")
POUR TOUT ELEMENT service de tabService 
	...
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Périphérique>.RécupèreTousServices(<Identifiant>)`
---

**`<Résultat> : (Tableau de variables de type btleService)`**

Tableau de variables de type [btleService](../WDLang3/1000022057.md) correspondant aux services trouvés. Si aucun service ne correspond à l'identifiant spécifié, le tableau retourné sera vide.

**`<Périphérique> : (Variable de type btlePériphérique)`**

Nom de la variable de type [btlePériphérique](../WDLang3/1000022054.md) correspondant au périphérique Bluetooth manipulé.

**`<Identifiant> : (Chaîne de caractères)`**

Identifiant (UUID) des services à récupérer.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Plusieurs services d'un même périphérique Bluetooth peuvent correspondre au même identifiant (UUID). Pour récupérer uniquement le premier service correspondant à un identifiant, utilisez la fonction [&lt;Variable btlePériphérique&gt;.RécupèreUnService](../WDLang3/1000022222.md).




## &lt;Variable emailSessionOutlook&gt;.AjouteDossier (Fonction)

***En anglais : &lt;emailOutlookSession variable&gt;.AddFolder***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>


Ajoute un dossier dans le logiciel de messagerie Outlook. Ce dossier est créé dans la branche "Dossiers personnels".

**Rappel** : Cette fonction ne permet pas d'ajouter un dossier dans le logiciel de messagerie Outlook Express.






<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ouverture de la session Outlook
IDSession est un EmailSessionOutlook
...
// Création d'un dossier dans le dossier en cours 
IDSession.AjouteDossier(IDSession.DossierCourant() + "\Travail")
// Création d'un dossier dans le dossier personnel
IDSession.AjouteDossier("Dossiers personnels\Boîte de réception\TravailPerso")
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Session>.AjouteDossier(<Chemin du dossier>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'ajout a bien été effectué, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. En cas d'erreur, la variable [ErreurDétectée](../WDLang1/3087001.md) est à <u><u><u><u>Vrai</u></u></u></u>. Pour connaître le libellé de l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.




**`<Session> : (Variable de type emailSessionOutlook)`**

Nom de la variable de type [emailSessionOutlook](../WDLang3/1000018767.md) correspondant à la session email en cours.

**`<Chemin du dossier> : (Chaîne de caractères Unicode)`**

Chemin du dossier à créer. Ce chemin doit obligatoirement commencer par "Dossiers personnels".

Si ce chemin contient plusieurs branches n'existant pas, toutes les branches sont créées.

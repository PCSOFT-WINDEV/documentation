
## GroupeRAZ (Fonction)

***En anglais : GroupReset***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Réinitialise la structure [Groupe](../WDLang3/1000008120.md) utilisée pour accéder à un groupe de contacts d'une messagerie Outlook ou Lotus Notes. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ouverture d'une session Lotus Notes
IDSession est un entier
IDSession = EmailOuvreSessionNotes("MotDePasse", "ServeurMars", ...
		"mail\julie.nsf", "C:\Users\Julie\AppData\Local\Lotus\Notes\Data\user.id")
// Initialisation des variables de la structure Groupe
GroupeRAZ()
// Renseignements des variables pour le nouveau groupe
mGroupe.Nom = "Mes amis"
mGroupe.Description = "Groupe de mes amis"
mGroupe.NbContact = 3
mGroupe.Contact[1] = "Pierre"
mGroupe.Contact[2] = "Paul"
mGroupe.Contact[3] = "Jacques"
// Ajout du groupe
SI GroupeAjoute(IDSession) = Vrai ALORS
	Info("Groupe de contacts ajouté")
SINON
	Erreur("Erreur lors de l'ajout", ErreurInfo())
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`GroupeRAZ()`
---



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Les groupes de contacts ne sont pas gérés par Outlook version 97. Il n'est donc pas possible de réinitialiser les groupes de contacts dans cette version de Outlook.

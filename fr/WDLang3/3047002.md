


## Fonctions J2EE
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions J2EE sont les suivantes :



|   |   |
| --- | --- |
| [J2EEAjouteEntête](../WDLang3/3047001.md) | Ajoute un entête à la prochaine procédure à exécuter sur le serveur. |
| [J2EEDonneRésultat](../WDLang3/3047003.md) | Renvoie le résultat de la dernière procédure J2EE exécutée avec succès. |
| [J2EEErreur](../WDLang3/3047010.md) | Renvoie l'erreur de la dernière procédure J2EE ayant échoué. |
| [J2EEExécute](../WDLang3/3047007.md) | Exécute une procédure sur un serveur de services Web XML J2EE. |
| [J2EEExécuteXML](../WDLang3/3047008.md) | Exécute une procédure sur un serveur de services Web XML J2EE. |


## OLEColle (Fonction)

***En anglais : OLEPaste***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Colle le contenu du presse-papiers de Windows dans un champ OLE.

**Remarque** : Si le contenu du presse-papiers a été créé depuis un serveur OLE, celui-ci est automatiquement associé à l'objet, sinon l'objet ne sera pas collé. Selon le serveur qui a créé l'objet OLE, l'objet peut s'afficher dans le champ sous forme de :

- sa représentation bitmap (exemple d'image créée avec Paint Brush),

- sa représentation vectorielle (exemple de feuille de calcul créée avec Excel),

- l'icône représentant le serveur (exemple d'icône WinWord),

- texte.



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// La portion d'image est collée depuis Paint Brush dans le presse-papiers.
// Le contenu du presse-papiers est collé dans le champ.
// Le serveur associé est Paint Brush.
OLEColle(OLE_OLE1)
OLESauve(OLE_OLE1, "OBJET1.OLE")
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = OLEColle(<Champ OLE>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'objet a été collé, 

- <u><u><u><u>Faux</u></u></u></u> en cas de problème.




**`<Champ OLE> : (Nom de champ)`**

Nom du champ OLE qui va accueillir l'objet OLE. Si le champ contenait un objet, cet objet est remplacé par le nouvel objet. Si ce paramètre correspond à une chaîne vide (""), le champ considéré sera le champ auquel appartient l'événement en cours.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Contenu du presse-papiers
<a name="contenu_pressepapiers_ELTPARAGRAPHE000049"></a>

- Le contenu du presse-papiers peut être de la forme texte, graphique ou binaire.

- Le contenu du presse-papiers a été initialisé par la fonction [OLECopie](../WDLang3/3057004.md), par une option (d'un éditeur) ou par une commande de copie dans le presse-papiers.





<a name="NOTE0_2"></a>




### Divers
<a name="divers_ELTPARAGRAPHE000062"></a>

- Seul OLE version 1 est géré.

- L'objet peut être sauvé avec la fonction [OLESauve](../WDLang3/3057011.md).


## SaaSAdminListeTarification (Fonction)

***En anglais : SaaSAdminListPricing***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Liste les tarifications d'un site SaaS.

**Remarques** : 

- L'utilisation de cette fonction nécessite d'avoir ouvert une session d'administration SaaS au préalable en utilisant la fonction [SaaSAdminConnecte](../WDLang3/1000019098.md).

- L'Administrateur SaaS aura accès à toutes les tarifications. Un gestionnaire de compte client aura accès uniquement aux tarifications des sites auxquels il est abonné. 



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affichage des tarifications du site CRM dans un champ Liste
tabTarif est un tableau de saasTarification
tabTarif = SaaSAdminListeTarification("CRM")

POUR TOUT Tarif DE tabTarif
	ListeAjoute(LISTE_Tarif, Tarif..Nom)
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = SaaSAdminListeTarification(<Site SaaS>)`
---

**`<Résultat> : (Tableau de variables de type saasTarification)`**

Liste des tarifications SaaS définies pour le site.

**`<Site SaaS> : (Variable de type saasSite ou chaîne de caractères)`**

Nom de la variable [saasSite](../WDLang3/1000019068.md) correspondant au site SaaS pour lequel les tarifications sont recherchées.

Ce paramètre peut également être une chaîne de caractères correspondant au nom du site SaaS manipulé.


## awProcédureRéceptionEtEnvoiInfo (Fonction)

***En anglais : awProcedureGetAndSendInfo***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Précise la procédure WLangage appelée lors de la réception des informations provenant de la fonction [awEnvoieEtRécupèreInfo](../WDLang3/1000022116.md). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
PROCEDURE INTERNE RecoitDonnées(b est Buffer) : Buffer
	Info(b)
	RENVOYER "retour"
FIN 

awProcédureRéceptionEtEnvoiInfo(RecoitDonnées)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`awProcédureRéceptionEtEnvoiInfo(<Procédure WLangage>)`
---

**`<Procédure WLangage> : (Nom de procédure)`**

Nom de la procédure WLangage (également nommée "callback") appelée lors de la réception des informations provenant de la fonction [awEnvoieEtRécupèreInfo](../WDLang3/1000022116.md).
Cette procédure est de la forme :

```txt
PROCEDURE <Nom de la procédure>(<Information>) : Buffer
...
RENVOYER <Réponse>
```

où &lt;Information&gt; et &lt;Réponse&gt; sont des variables de type Buffer.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
**Conseil** :  Appelez cette fonction dans le code d'initialisation du projet pour que la procédure soit accessible dès le démarrage de l'application.

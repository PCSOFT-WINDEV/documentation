


## EmailCopie (Fonction)

***En anglais : EmailCopy***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Copie un email présent dans un répertoire vers un autre répertoire d'un serveur IMAP. 

**Remarque** : Cette fonction n'est pas disponible pour les sessions POP3, Outlook et Notes. 




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonEmail est un Email
MaSession est une emailSessionIMAP
...
// Ouverture de la session
SI EmailOuvreSession(MaSession) = Vrai ALORS
	// Lecture du premier email
	SI EmailLitPremier(MaSession, MonEmail) = Vrai ALORS
		// Copie l'email dans un autre dossier
		EmailCopie(MaSession, MonEmail, "Inbox/Lu")
	FIN
	EmailFermeSession(MaSession)
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Copier un email contenu dans une variable de type Email

`<Résultat> = EmailCopie(<Email> , <Dossier>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la copie a été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Email> : (Variable de type Email)`**

Nom de la variable de type [Email](../WDLang3/1000018713.md) correspondant à l'email à copier. Cet email est associé à une session IMAP. 

**`<Dossier> : (Chaîne de caractères)`**

Dossier destination de l'email. Ce dossier doit exister. La liste des dossiers peut être connue par la fonction [EmailListeDossier](../WDLang3/3032133.md). 


<a name="SYNTAXE2"></a>

### Copier un email contenu dans une variable de type Email en précisant la session IMAP

`<Résultat> = EmailCopie(<Session> , <Email> , <Dossier>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la copie a été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Session> : (Variable de type emailSessionIMAP)`**

Nom de la session de type [emailSessionIMAP](../WDLang3/1000018957.md) correspondant à la session à laquelle appartient l'email. 

**`<Email> : (Variable de type Email)`**

Nom de la variable de type [Email](../WDLang3/1000018713.md) correspondant à l'email à copier. 

**`<Dossier> : (Chaîne de caractères)`**

Dossier destination de l'email. Ce dossier doit exister. La liste des dossiers peut être connue par la fonction [EmailListeDossier](../WDLang3/3032133.md).


## FTPNom (Fonction)

***En anglais : FTPName***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le nom du dernier fichier accédé par une fonction FTP (File Transfer Protocol).

**Remarque** : La fonction **FTPNom** est utile notamment lors d'un parcours de fichiers sur un serveur FTP au moyen de la fonction [FTPListeFichier](../WDLang3/3040018.md).




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Nom du dernier fichier accédé par une fonction FTP
ResNom = FTPNom()
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = FTPNom()`
---

**`<Résultat> : (Chaîne de caractères)`**



- Nom du dernier fichier accédé par une fonction FTP. Ce fichier a été modifié par une des fonctions FTP suivantes : [FTPAttribut](../WDLang3/3040006.md), [FTPDate](../WDLang3/3040015.md), [FTPHeure](../WDLang3/3040013.md), [FTPTaille](../WDLang3/3040007.md), [FTPListeFichier](../WDLang3/3040018.md).  

- Chaîne vide ("") en cas d'erreur.  Pour connaître le libellé de l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Requête au serveur FTP
<a name="requete_serveur_ftp_ELTPARAGRAPHE000084"></a>La fonction **FTPNom** ne provoque pas l'envoi d'une requête au serveur FTP.


<a name="NOTE0_2"></a>


### Délai
<a name="delai_ELTPARAGRAPHE000095"></a>Par défaut, toutes les fonctions FTP échouent si aucune réponse du serveur FTP n'est reçue après 20 secondes. Ce délai peut être modifié lors de la connexion au serveur FTP avec la fonction [FTPConnecte](../WDLang3/3040016.md).


<a name="NOTE0_3"></a>
<a name="NOTE0_4"></a>


### Droits des utilisateurs
<a name="droits_des_utilisateurs_ELTPARAGRAPHE000125"></a>La récupération des noms des fichiers présents sur un serveur FTP n'est possible que pour un utilisateur ayant les droits en lecture sur le serveur FTP. En règle générale, les utilisateurs "anonymes" ont les droits en lecture.

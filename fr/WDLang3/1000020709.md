
## Constantes de gestion des rendez-vous
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


| Constante | Description | Utilisée par la fonction |
| --- | --- | --- |
| **rappelDéfaut** | Méthode à utiliser pour effectuer le rappel : méthode par défaut. Cette méthode varie en fonction du paramétrage de l'appareil. | Type [Rappel](../WDLang3/1000020688.md) |
| **rappelEmail** | Méthode à utiliser pour effectuer le rappel : envoi d'un email. Ce type de rappel est disponible uniquement si le compte associé au calendrier du rendez-vous supporte ce type de rappel et si ce compte est configuré. | Type [Rappel](../WDLang3/1000020688.md) |
| **rappelNotification** | Méthode à utiliser pour effectuer le rappel : envoi d'une notification. | Type [Rappel](../WDLang3/1000020688.md) |
| **rappelSMS** | Méthode à utiliser pour effectuer le rappel : envoi d'un SMS. Ce type de rappel est disponible uniquement si le compte associé au calendrier du rendez-vous supporte ce type de rappel et si ce compte est configuré. | Type [Rappel](../WDLang3/1000020688.md) |
| **rdvContenu** | Filtre sur les rendez-vous : Filtre sur la description du rendez-vous (recherche de type contient). | [RendezvousListe](../WDLang3/1000020690.md) |
| **rdvIdentifiant** | Filtre sur les rendez-vous : Filtre sur les identifiants de rendez-vous (recherche à l'identique). | [RendezvousListe](../WDLang3/1000020690.md) |
| **rdvLieu** | Filtre sur les rendez-vous : Filtre sur le lieu du rendez-vous (recherche de type contient). | [RendezvousListe](../WDLang3/1000020690.md) |
| **rdvTitre** | Filtre sur les rendez-vous : Filtre sur le titre du rendez-vous (recherche de type contient). | [RendezvousListe](../WDLang3/1000020690.md) |

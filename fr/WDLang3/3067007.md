
## EtatAlimentation (Fonction)

***En anglais : PowerStatus***

	<IMG style="float:left;margin-right:10px;" src="https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=warning_big.png"><B>Avertissement</B><BR>

<a name="Utilisation"></a>
<a name="description"></a>
Permet de connaître diverses informations sur la batterie (principale ou de secours) de l'appareil sur lequel l'application est exécutée.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Etat de la batterie principale ?
Info("La batterie est chargée à : " + EtatAlimentation(PourcentageCharge) + " %")
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = EtatAlimentation(<Type d'information> [, <Batterie>])`
---

**`<Résultat> : (Entier ou booléen selon le type d'information demandé)`**



- Information demandée, 

- Chaîne vide ("") si l'information demandée n'est pas disponible.




**`<Type d'information> : (Constante de type entier)`**

Indique le type d'information à rechercher : 



|   |   |
| --- | --- |
| *EnRecharge* | <br><br>- <u><u><u><u>Vrai</u></u></u></u> si la batterie est en charge,<br><br>- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.<br><br><br> |
| *NiveauCharge* | Niveau de charge de la batterie :<br><br>- 0 : batterie inexistante.<br><br>- 1 : niveau haut.<br><br>- 2 : niveau bas.<br><br>- 3 : niveau critique.<br><br><br> |
| *PourcentageCharge* | Pourcentage de charge de la batterie. |
| *SurSecteur* | <br><br>- <u><u><u><u>Vrai</u></u></u></u> si l'appareil est branché sur le secteur,<br><br>- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.<br><br><br> |
| *VieRestante* | Durée de vie restante (en secondes) de la batterie.<br><br> |
| *VieTotale* | Durée de vie totale (en secondes) de la batterie.<br><br> |



**`<Batterie> : (Booléen optionnel)`**



- <u><u><u><u>Vrai</u></u></u></u> (par défaut) pour obtenir des informations sur la batterie principale de l'appareil, 

- <u><u><u><u>Faux</u></u></u></u> pour obtenir des informations sur la batterie de secours de l'appareil.

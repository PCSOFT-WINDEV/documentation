


## MQTTDésabonneTout (Fonction)

***En anglais : MQTTUnsubscribeAll***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de se désabonner de tous les abonnements préalablement souscrits.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MQTTDésabonneTout(gMqttSession)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = MQTTDésabonneTout(<Session MQTT>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si tous les désabonnements ont été effectués,

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. La fonction [ErreurInfo](../WDLang1/3013008.md) permet d'obtenir plus de détails. 




**`<Session MQTT> : (Variable de type mqttSession)`**

Nom de la variable [mqttSession](../WDLang3/1000023023.md) correspondant à la session MQTT à manipuler.

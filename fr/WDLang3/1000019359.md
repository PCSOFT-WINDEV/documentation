
## BTActive (Fonction)

***En anglais : BTActivate***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Active ou désactive la gestion du Bluetooth sur l'appareil. Pour connaître l'état d'activation du Bluetooth sur l'appareil ou être notifié en cas de changement d'état, utilisez la fonction [BTEtat](../WDLang3/1000019360.md).

**Remarque** :  Les fonctions Bluetooth ne sont pas disponibles dans l'émulateur Android. 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI BTEtat() = btDesactivé ALORS
	BTActive(Vrai)
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = BTActive([<Activation du Bluetooth>])`
---

**`<Résultat> : (Constante de type Entier)`**

Etat d'activation du Bluetooth après la modification. Les valeurs possibles sont les suivantes : 


|   |   |
| --- | --- |
| *btActif* | Le Bluetooth est activé sur l'appareil. |
| *btActivationEnCours* | Le Bluetooth est en cours d'activation sur l'appareil. |
| *btDésactivationEnCours* | Le Bluetooth est en cours de désactivation sur l'appareil. |
| *btDésactivé* | Le Bluetooth est désactivé sur l'appareil. |
| *btErreur* | L'état d'activation du Bluetooth n'a pas pu être récupéré. |





**`<Activation du Bluetooth> : (Booléen optionnel)`**

Nouvel état d'activation du Bluetooth : 

- <u><u><u><u>Vrai</u></u></u></u> (valeur par défaut) : Active le Bluetooth sur l'appareil. 

- <u><u><u><u>Faux</u></u></u></u> : Désactive le Bluetooth sur l'appareil.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Permissions requises
<a name="permissions_requises_ELTPARAGRAPHE000078"></a>L'utilisation de cette fonction entraîne la modification des permissions requises par l'application.

Permissions requises :

- BLUETOOTH : cette permission permet aux applications de se connecter à des services bluetooth. 

- BLUETOOTH_ADMIN : cette permission permet aux applications de découvrir et de connecter des périphériques bluetooth.

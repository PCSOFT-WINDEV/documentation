
## SaaSConnexionClient (Fonction)

***En anglais : SaaSClientConnexion***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie les caractéristiques de la connexion à la base de données du client. Cette fonction permet à un utilisateur connecté grâce à la fonction [SaaSConnecte](../WDLang3/1000018959.md) d'accéder à la base de données cliente lui appartenant.  
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MaConnexion est une Connexion

SI SaaSEstConnecté() = Vrai ALORS
	MaConnexion = SaaSConnexionClient()
	HChangeConnexion("*", MaConnexion)
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = SaaSConnexionClient()`
---

**`<Résultat> : (Variable de type Connexion)`**

Variable de type [Connexion](../WDLang4/1514073.md) contenant les caractéristiques de connexion à la base de données du client.




## CarteInfoPosition (Fonction)

***En anglais : MapInfoPosition***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie les coordonnées en pixels du point d'un champ Carte correspondant à une position géographique (latitude et longitude).




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Retourne le point de la carte correspondant à la position courante de l'utilisateur
MaPosition est une géoPosition
MaPosition.Latitude = 1.442951
MaPosition.Longitude = 43.604363
Trace(CarteInfoPosition(CARTE_Position, MaPosition))
```


<a name="Exemple2"></a>
![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png) 
```wl
// Retourne le point de la carte correspondant à la position courante de l'utilisateur
MaPosition est une géoPosition
MaPosition = GPSRécupèrePosition()
SI ErreurDétectée = Faux ALORS
	RENVOYER CarteInfoPosition(CARTE_Position, MaPosition)
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = CarteInfoPosition(<Champ Carte> , <Position>)`
---

**`<Résultat> : (Chaîne de caractères)`**

Coordonnées du point correspondant à la position géographique. Les coordonnées renvoyées sont au format : 

```txt
<X> + TAB + <Y>
```


**`<Champ Carte> : (Nom de champ)`**

Nom du champ Carte à manipuler. Si ce paramètre correspond à une chaîne vide (""), le champ Carte auquel appartient l'événement en cours sera utilisé.

**`<Position> : (Variable de type géoPosition)`**

Nom d'une variable de type [géoPosition](../WDLang3/1000019191.md) correspondant à la position géographique dont les coordonnées sont recherchées sur la carte.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Les coordonnées renvoyées par la fonction **CarteInfoPosition** peuvent correspondre à un point actuellement non visible dans le champ Carte (coordonnées négatives par exemple).

- Le calcul de la position tient compte du niveau de zoom dans le champ Carte au moment de l'appel de la fonction. Plus le niveau de zoom est important, plus les coordonnées renvoyées sont précises.

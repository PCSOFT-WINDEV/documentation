


## CarteAffichePopup (Fonction)

***En anglais : MapDisplayPopup***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ouvre la popup d'un marqueur affiché dans un champ Carte.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affiche un marqueur 
MaPosition est une géoPosition
MaPosition.Latitude = 43.613708 
MaPosition.Longitude = 3.876972
MonMarqueur est un Marqueur
MonMarqueur.Position = MaPosition
MonMarqueur.AvecPopup = Vrai
MonMarqueur.Nom = "Position"
MonMarqueur.Description = "Position du lieu"
CarteAjouteMarqueur(CARTE_MaCarte, MonMarqueur)

// Affiche la popup du marqueur
CarteAffichePopup(CARTE_MaCarte, MonMarqueur)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = CarteAffichePopup(<Champ Carte> , <Marqueur>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la popup du marqueur a pu être affichée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Champ Carte> : (Nom de champ)`**

Nom du champ Carte à manipuler. Si ce paramètre correspond à une chaîne vide (""), le champ Carte auquel appartient le traitement en cours sera utilisé.

**`<Marqueur> : (Variable de type Marqueur)`**

Nom de la variable de type [Marqueur](../WDLang3/1000019940.md) décrivant la popup du marqueur à afficher. Ce marqueur doit avoir été ajouté sur la Carte.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Pour masquer la popup d'un marqueur déjà affichée, utilisez la fonction [CarteFermePopup](../WDLang3/1000025554.md).

- Pour savoir si la popup d'un marqueur est affichée, utilisez la fonction [CartePopupAffichée](../WDLang3/1000025555.md).

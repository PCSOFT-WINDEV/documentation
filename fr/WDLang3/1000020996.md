


## &lt;Variable upnpPériphérique&gt;.RécupèreMuet (Fonction)

***En anglais : &lt;upnpDevice variable&gt;.GetMute***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Récupère l'état 'Muet' d'un périphérique UPnP.

**Attention** : Pour utiliser cette fonction, le périphérique doit posséder un service "RenderingControl".
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
tabPériphérique est un tableau de upnpPériphériques 
tabPériphérique = UPNPListePériphérique(...
		"urn:schemas-upnp-org:device:MediaRenderer:1", 5)
POUR TOUT Périphérique de tabPériphérique
	SI Périphérique.RécupèreMuet() ALORS
		Trace(Périphérique.Nom + " = Muet")
	SINON
		Trace(Périphérique.Nom + " = Non muet")
	FIN
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Périphérique UPnP>.RécupèreMuet()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le périphérique est muet, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Périphérique UPnP> : (Variable de type upnpPériphérique)`**

Nom de la variable [upnpPériphérique](../WDLang3/1000020903.md) correspondant au périphérique UPnP à manipuler.




## CarteLicenceBing (Fonction)

***En anglais : MapLicenseBing***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Indique la clé de licence "Bing map" à utiliser dans les champs Carte utilisés en mode Universal Windows 10 App.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Initialisation du projet
CarteLicenceBing("AIzaSyBjsINSH5x39Ks6c0_CoS1yr1Mb3cB3cVo")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`CarteLicenceBing(<clé>)`
---

**`<clé> : (Chaîne de caractères)`**

Clé fournie par Bing. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Obtenir une clé Bing
<a name="obtenir_une_cle_bing_ELTPARAGRAPHE000031"></a>Dans les applications Universal Windows 10 App, le champ Carte utilise le service "Bing Map". 

Pour employer ce service, il est nécessaire d'obtenir et d'utiliser une clé de licence Bing Map. 

Pour obtenir une clé de licence Bing Map : 

1. Inscrivez-vous sur le site [https://www.bingmapsportal.com/](https://www.bingmapsportal.com/). 

2. Dans le menu "My Account", sélectionnez l'option "Create or view keys". 

3. Spécifiez les informations demandées : 

	- Nom de l'application. 

	- Type de clé. 

	- Type d'application : "Universal Windows App". 




4. Cliquez sur "Create". 

5. La clé générée est affichée sous le navigateur. Le lien "Copy key" permet de copier la clé. 




Rappel : L'utilisation d'un champ Carte est soumise à des conditions d'utilisation spécifiques. Pour plus de détails, consultez  [http://www.microsoft.com/maps/Licensing/licensing.aspx](http://www.microsoft.com/maps/Licensing/licensing.aspx). 
<a name="NOTE0_2"></a>


### Conditions d'utilisation
<a name="conditions_utilisation_ELTPARAGRAPHE000060"></a>

- Cette fonction permet de spécifier la licence à utiliser pour que le fournisseur de cartes Bing donne accès aux fonctionnalités étendues à l'application (marqueurs, itinéraires, ...). 

- La licence est appliquée à tous les champs Carte de l'application. Elle ne sera pas utilisée pour tous les nouveaux champs Carte créés après l'appel de la fonction. 

- Il est recommandé d'utiliser la fonction **CarteLicenceBing** dans le code d'initialisation du projet.

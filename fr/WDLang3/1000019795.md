


## CarteSupprimeMarqueur (Fonction)

***En anglais : MapDeleteMarker***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime un marqueur d'une carte affichée par un champ Carte. 




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Suppression d'un marqueur
MonMarqueur est un Marqueur
MonMarqueur.Nom = "Destination"
MonMarqueur.Position.Latitude = 43.613708 
MonMarqueur.Position.Longitude = 3.876972

SI INT_AfficheMaDestination = Vrai ALORS
	CarteAjouteMarqueur(CARTE_Position, MonMarqueur)
SINON
	CarteSupprimeMarqueur(CARTE_Position, MonMarqueur)
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = CarteSupprimeMarqueur([<Champ Carte> [, <Marqueur>]])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le marqueur a pu être supprimé, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Champ Carte> : (Nom de champ optionnel)`**

Nom du champ Carte à manipuler. Si ce paramètre correspond à une chaîne vide ("") ou n'est pas spécifié, le champ Carte auquel appartient l'événement en cours sera utilisé.

**`<Marqueur> : (Variable de type Marqueur ou chaîne de caractères optionnelle)`**

Marqueur à supprimer. Ce paramètre peut correspondre : 

- à une variable de type [Marqueur](../WDLang3/1000019940.md) correspondant au marqueur à supprimer. 

- à une chaîne de caractères correspondant au nom du marqueur à supprimer. Ce nom correspond à la propriété **Nom** de la variable de type [Marqueur](../WDLang3/1000019940.md).




Si ce paramètre n'est pas spécifié, tous les marqueurs de la carte seront supprimés (équivalent à la fonction [CarteSupprimeTout](../WDLang3/1000019952.md)). 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Seul un marqueur précédemment ajouté au champ Carte peut être supprimé. 

- Pour ajouter un nouveau marqueur dans un champ Carte, utilisez la fonction [CarteAjouteMarqueur](../WDLang3/1000019794.md).

- Pour modifier un marqueur affiché dans un champ Carte, utilisez la fonction [CarteModifieMarqueur](../WDLang3/1000019796.md).




## &lt;Graphe&gt;.Titre (Fonction)

***En anglais : &lt;Chart&gt;.Title***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie le libellé et/ou la position du titre d'un graphe.

Exemple de titre dans un graphe : ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Titre.gif)


Le graphe manipulé peut être :

- soit un graphe créé par programmation (fonction [grCrée](../WDLang3/3042005.md)).

- soit un champ Graphe dans l'éditeur de fenêtres, de pages ou d'états.




**Remarque** : Par défaut, un graphe n'a pas de titre.



![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Les graphes](../WDChamp/3042060.md). 












<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Modification du libellé et de la position du titre du graphe GRF_MonGraphe
GRF_MonGraphe.Titre("Évolution du Chiffre d'affaires", grEnHaut)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Champ Graphe>.Titre([<Titre> [, <Position>]])`
---

**`<Champ Graphe> : (Nom de champ)`**

Nom du champ Graphe à manipuler (présent dans l'éditeur de fenêtres, de pages ou d'états).


**`<Titre> : (Chaîne de caractères optionnelle)`**

Titre du graphe.

**`<Position> : (Constante optionnelle)`**

Indique la position du titre :


|   |   |
| --- | --- |
| *grEnBas* | Titre affiché en bas du graphe. |
| *grEnHaut* | Titre affiché en haut du graphe. |
| *grSans* | Sans titre. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Affichage du titre
<a name="affichage_titre_ELTPARAGRAPHE000296"></a>Les modifications réalisées avec la fonction **&lt;Graphe&gt;.Titre** seront prises en compte lors du prochain appel à la fonction [&lt;Graphe&gt;.Dessine](../WDLang3/1000024062.md).
<a name="NOTE0_2"></a>


### Positionnement manuel du titre : &lt;Graphe&gt;.Titre sans effet
<a name="positionnement_manuel_titre_docparampagetitleshort_sans_effet_ELTPARAGRAPHE000314"></a>Si le titre du graphe est positionné manuellement (option "Positionnement manuel" dans la fenêtre du titre du graphe), la fonction **&lt;Graphe&gt;.Titre** n'a plus d'effet.
<a name="NOTE0_3"></a>


### Langues utilisant un alphabet non latin
<a name="langues_utilisant_alphabet_non_latin_ELTPARAGRAPHE000334"></a>La fonction **&lt;Graphe&gt;.Titre** utilisera l'alphabet non latin sélectionné uniquement si la configuration du projet associée utilise des chaînes UNICODE en exécution.

Pour vérifier l'option choisie dans la configuration courante : 

1. Sous le volet "Projet", dans le groupe "Configuration de projet", cliquez sur "Configuration courante". 

2. Affichez l'onglet "Unicode" : l'option "Utiliser des chaînes UNICODE en exécution" doit être cochée. 

3. Validez si nécessaire.




## &lt;Graphe&gt;.NuageCouleurSérie (Fonction)

***En anglais : &lt;Chart&gt;.ScatterSeriesColor***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Initialise la couleur d'une série pour un graphe de type "Nuage de points". 

**Rappel** : Par défaut, les couleurs des séries (ou des parts de secteur) sont sélectionnées selon un ordre prédéfini parmi un éventail de 13 couleurs.



![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Les graphes](../WDChamp/3042060.md). 




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ajoute une donnée dans le champ GRF_MonGraphe de type "Nuage de points".
// Cette valeur (20, 20) est ajoutée à la fin de la première série.
GRF_MonGraphe.NuageAjouteDonnéeXY(1, 20, 20)
// La série 1 du graphe GRF_MonGraphe est de couleur verte
GRF_MonGraphe.NuageCouleurSérie(1, VertClair)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Champ Graphe>.NuageCouleurSérie(<Numéro de la série> , <Couleur>)`
---

**`<Champ Graphe> : (Nom de champ)`**

Nom du champ Graphe à manipuler (présent dans l'éditeur de fenêtres, de pages ou d'états).


**`<Numéro de la série> : (Entier)`**

Numéro de la série à manipuler.

**`<Couleur> : (Entier ou constante)`**

Couleur de la série spécifiée. Cette couleur peut correspondre à :

- une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)),

- une couleur TSL (obtenue avec la fonction [TSL](../WDLang1/3029057.md)),

- [une couleur prédéfinie du WLangage](../WDLang5/3010002.md).






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Les données définies avec la fonction **&lt;Graphe&gt;.NuageCouleurSérie** seront prises en compte lors du prochain appel à la fonction [&lt;Graphe&gt;.Dessine](../WDLang3/1000024062.md).

- Pour obtenir des couleurs dégradées, utilisez la fonction [&lt;Graphe&gt;.Dégradé](../WDLang3/1000024061.md).

- La fonction **&lt;Graphe&gt;.NuageCouleurSérie** est équivalente à la fonction [&lt;Graphe&gt;.CouleurSérie](../WDLang3/1000023599.md). 
	La syntaxe de la fonction [&lt;Graphe&gt;.CouleurSérie](../WDLang3/1000023599.md) utilisée avec les graphes de type "Nuage de points" est la suivante :
	
	```txt
	<Champ Graphe>.CouleurSérie(2*<Indice>-1, <Couleur>)
	```

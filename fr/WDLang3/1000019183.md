
## SaaSAdminListeAbonnementClient (Fonction)

***En anglais : SaaSAdminListClientSubscription***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Liste les abonnements SaaS d'un compte client.

**Remarques** : 

- L'utilisation de cette fonction nécessite d'avoir ouvert une session d'administration SaaS au préalable en utilisant la fonction [SaaSAdminConnecte](../WDLang3/1000019098.md).

- L'Administrateur SaaS aura accès à tous les abonnements. Un gestionnaire de compte client aura accès uniquement à ses propres abonnements. 



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affiche les abonnements d'un client dans un champ Table (lié à une variable de type Tableau)
tabAbo est un tableau de saasAbonnement 
tabAbo = SaaSAdminListeAbonnementClient(SAI_NomClient)
SourceVersEcran()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Lister les abonnements SaaS d'un compte client (utilisation des variables saasxxx)

`<Résultat> = SaaSAdminListeAbonnementClient(<Compte client> [, <Site SaaS>])`
---

**`<Résultat> : (Tableau de variable de type saasAbonnement)`**

Liste des abonnements.

**`<Compte client> : (Variable de type saasClient)`**

Nom de la variable de type [saasClient](../WDLang3/1000019072.md) correspondant au compte client SaaS pour lequel les abonnements sont recherchés.

**`<Site SaaS> : (Variable de type saasSite optionnelle)`**

Nom de la variable de type [saasSite](../WDLang3/1000019068.md) contenant la description du site SaaS à manipuler. Si ce paramètre est spécifié, seuls les abonnements au site SaaS indiqué seront listés, sinon tous les abonnements du compte client seront listés.


<a name="SYNTAXE2"></a>

### Lister les abonnements d'un compte client identifié par son nom

`<Résultat> = SaaSAdminListeAbonnementClient(<Compte client> [, <Site SaaS>])`
---

**`<Résultat> : (Tableau de variable de type saasAbonnement)`**

Liste des abonnements.

**`<Compte client> : (Chaîne de caractères)`**

Nom du compte client SaaS pour lequel les abonnements sont recherchés.

**`<Site SaaS> : (Chaîne de  caractères optionnelle)`**

Nom du site SaaS à manipuler. Si ce paramètre est spécifié, seuls les abonnements au site SaaS indiqué seront listés, sinon tous les abonnements du compte client seront listés.


## Constantes de gestion des champs Carte

Les constantes de gestion des champs Carte sont les suivantes : 

| Constante | Description | Utilisée par les fonctions |
| --- | --- | --- |
| **adresseAutre** | Type d'adresse : autre type d'adresse. | Variable de type [Adresse](../WDLang3/1000019944.md) |
| **adresseBureau** | Type d'adresse : adresse professionnelle (du bureau). | Variable de type [Adresse](../WDLang3/1000019944.md) |
| **adresseDomicile** | Type d'adresse : adresse du domicile. | Variable de type [Adresse](../WDLang3/1000019944.md) |
| **adressePersonnalisée** | Type d'adresse : adresse personnalisée. | Variable de type [Adresse](../WDLang3/1000019944.md) |
| **ccsArgenté** | Style appliqué à la carte : Style "argenté". | [CarteRécupèreStyle](../WDLang3/1000025557.md), [CarteChangeStyle](../WDLang3/1000025556.md) |
| **ccsAubergine** | Style appliqué à la carte : Style "aubergine". | [CarteRécupèreStyle](../WDLang3/1000025557.md), [CarteChangeStyle](../WDLang3/1000025556.md) |
| **ccsAutomatique** | Style appliqué à la carte : Le style sera appliqué en fonction du mode clair ou sombre de l'application :<br><br>- Thème clair : le style utilisé sera le style "standard",<br><br>- Thème sombre : le style utilisé sera le style "nuit".<br><br><br>Si le thème de l'application change alors que le champ Carte est affiché, le style sera automatiquement modifié. | [CarteRécupèreStyle](../WDLang3/1000025557.md), [CarteChangeStyle](../WDLang3/1000025556.md) |
| **ccsNuit** | Style appliqué à la carte : Style "nuit". | [CarteRécupèreStyle](../WDLang3/1000025557.md), [CarteChangeStyle](../WDLang3/1000025556.md) |
| **ccsPersonnalisé** | Style appliqué à la carte : Style personnalisé. | [CarteRécupèreStyle](../WDLang3/1000025557.md), [CarteChangeStyle](../WDLang3/1000025556.md) |
| **ccsRétro** | Style appliqué à la carte : Style "rétro". | [CarteRécupèreStyle](../WDLang3/1000025557.md), [CarteChangeStyle](../WDLang3/1000025556.md) |
| **ccsSombre** | Style appliqué à la carte : Style "sombre". | [CarteRécupèreStyle](../WDLang3/1000025557.md), [CarteChangeStyle](../WDLang3/1000025556.md) |
| **ccsStandard** | Style appliqué à la carte : style "standard". | [CarteRécupèreStyle](../WDLang3/1000025557.md), [CarteChangeStyle](../WDLang3/1000025556.md) |
| **itinéraireAutomobile** | L'itinéraire sera tracé en respectant les voies accessibles aux automobiles.<br><br>Caractéristiques de ce mode :<br><br><br><br>- Ce mode utilise le mode de géolocalisation de Google. Un accès Internet est nécessaire.<br><br>- L'ordre des étapes peut être modifié afin d'obtenir le trajet le plus optimal.<br><br><br> | [CarteAjouteItinéraire](../WDLang3/1000020383.md) |
| **itinéraireAutomobileNonOptimisé** | L'itinéraire sera tracé en respectant les voies accessibles aux automobiles.<br><br>Caractéristiques de ce mode :<br><br><br><br>- Ce mode utilise le mode de géolocalisation de Google. Un accès Internet est nécessaire.<br><br>- L'ordre des étapes ne sera pas modifié.<br><br><br> | [CarteAjouteItinéraire](../WDLang3/1000020383.md) |
| **itinéraireCycliste** | L'itinéraire sera tracé en respectant les voies accessibles aux vélos. Les itinéraires cyclistes ne sont pas disponibles dans tous les pays. Pour plus d'informations, consultez la documentation du service de géolocalisation de Google.<br><br>Caractéristiques de ce mode :<br><br><br><br>- Ce mode utilise le mode de géolocalisation de Google. Un accès Internet est nécessaire.<br><br>- L'ordre des étapes peut être modifié afin d'obtenir le trajet le plus optimal.<br><br><br> | [CarteAjouteItinéraire](../WDLang3/1000020383.md) |
| **itinéraireManuel** | L'itinéraire sera tracé de manière rectiligne entre chaque position spécifiée sans tenir compte des routes.<br><br>Ce mode permet de ne pas utiliser le mode de géolocalisation de Google. Dans ce cas, le paramètre &lt;Positions&gt; doit correspondre à un tableau de variables de type géoPosition ou de variables de type Marqueur.<br><br><br><br>Dans ce mode, les étapes intermédiaires sont prises en compte selon l'ordre d'ajout dans le tableau. | [CarteAjouteItinéraire](../WDLang3/1000020383.md) |
| **itinérairePiéton** | L'itinéraire sera tracé en respectant les voies accessibles aux piétons.<br><br>Caractéristiques de ce mode :<br><br><br><br>- Ce mode utilise le mode de géolocalisation de Google. Un accès Internet est nécessaire.<br><br>- L'ordre des étapes peut être modifié afin d'obtenir le trajet le plus optimal.<br><br><br> | [CarteAjouteItinéraire](../WDLang3/1000020383.md) |
| **mqBas** | Mode d'alignement du marqueur lorsqu'il est affiché sur une carte :  Marqueur affiché en bas de la position. | Type [Marqueur](../WDLang3/1000019940.md) |
| **mqCentre** | Mode d'alignement du marqueur lorsqu'il est affiché sur une carte :  Marqueur affiché sur la position. | Type [Marqueur](../WDLang3/1000019940.md) |
| **mqDroite** | Mode d'alignement du marqueur lorsqu'il est affiché sur une carte :  Marqueur affiché à droite de la position. | Type [Marqueur](../WDLang3/1000019940.md) |
| **mqGauche** | Mode d'alignement du marqueur lorsqu'il est affiché sur une carte :  Marqueur affiché à gauche de la position. | Type [Marqueur](../WDLang3/1000019940.md) |
| **mqHaut** | Mode d'alignement du marqueur lorsqu'il est affiché sur une carte :  Marqueur affiché en haut de la position. | Type [Marqueur](../WDLang3/1000019940.md) |

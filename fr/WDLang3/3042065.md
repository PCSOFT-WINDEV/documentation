


## grSauveParamètre (Fonction)

***En anglais : grSaveParameter***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Sauve les paramètres d'un graphe sous forme d'une chaîne. Ces paramètres pourront être restaurés avec la fonction [grChargeParamètre](../WDLang3/3042064.md).

**Rappel** : Par défaut, la configuration d'un graphe est automatiquement sauvegardée lors de l'arrêt de l'application. 



![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Les graphes](../WDChamp/3042060.md). 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Sauvegarde des paramètres d'un graphe créé par programmation
INIEcrit("Parametres", "MonGraphe", "Param", grSauveParamètre("MonGraphe"))

// Restauration des paramètres d'un graphe créé par programmation
grChargeParamètre("MonGraphe", INILit("Parametres", "MonGraphe", "Param"))
```
<a name="Exemple2"></a>

```wl
// Sauvegarde des paramètres d'un champ Graphe
INIEcrit("Parametres", "GRF_Graphe", "Param", grSauveParamètre(GRF_Graphe))

// Restauration des paramètres d'un champ Graphe
grChargeParamètre(GRF_Graphe, INILit("Parametres", "GRF_Graphe", "Param"))
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = grSauveParamètre(<Nom du graphe>)`
---

**`<Résultat> : (Chaîne de caractères Ansi)`**



- Paramètres du graphe. Cette chaîne peut être sauvegardée pour être ensuite restaurée par la fonction [grChargeParamètre](../WDLang3/3042064.md).

- Chaîne vide ("") en cas d'erreur.




**`<Nom du graphe> : (Nom de champ ou chaîne de caractères)`**

Nom du graphe à manipuler. Ce nom correspond :

- soit au nom du graphe défini par programmation avec la fonction [grCrée](../WDLang3/3042005.md).

- soit au nom du champ Graphe présent dans l'éditeur de fenêtres, de pages ou d'états.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Les paramètres du graphe sauvegardés sont notamment : 

- le type du graphe.

- le titre du graphe et sa position.

- la position de la légende.

- l'espacement entre les barres pour les histogrammes.

- la profondeur pour les histogrammes.

- les caractéristiques du dégradé.

- l'orientation des axes.

- le quadrillage.

- les paramètres 3D pour les graphes en 3 dimensions.

- les caractéristiques des polices utilisées.

- les caractéristiques de lissage.

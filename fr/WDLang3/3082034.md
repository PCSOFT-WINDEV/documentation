


## zipMsgErreur (Fonction)

***En anglais : zipMsgError***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le message associé à un code d'erreur sur une archive.










<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple
<a class="notetitle" target="_blank" href="$DOC$=1000003082034&name=zipmsgerreur_fonction&product=WD">Voir des exemples supplémentaires</a>

```wl
MonArchive est un zipArchive 
// Ouverture d'une archive
ResOuvreArchive = zipOuvre(MonArchive)
// Erreur d'ouverture d'une archive
SI ResOuvreArchive <> 0 ALORS
	// Affichage d'un message d'erreur
	Info(zipMsgErreur(ResOuvreArchive))
	...
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = zipMsgErreur(<Code d'erreur>)`
---

**`<Résultat> : (Chaîne de caractères)`**

Message d'erreur correspondant au code d'erreur spécifié.

**`<Code d'erreur> : (Entier)`**

Code d'erreur renvoyé lors de la manipulation d'une archive.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Codes d'erreur
<a name="codes_erreur_ELTPARAGRAPHE000077"></a>Les codes d'erreur retournés par les fonctions d'archivage sont les suivants :

- 1 : Le chemin passé en paramètre n'existe pas.

- 2 : Accès interdit : l'utilisateur n'a pas les droits nécessaires.

- 3 : L'archive est corrompue.

- 4 : Le chemin n'existe pas dans l'archive.

- 5 : Impossible d'écrire dans l'archive.

- 6 : Les fichiers des sous-archives ne sont pas dans l'ordre (cas d'une extraction dans une archive multi-parties).

- 7 : Le fichier est déjà présent dans l'archive.

- 8 : Une archive ne peut pas être ajoutée à elle-même. Par exemple, l'archive "MonArchive" ne peut pas être ajoutée à l'archive "MonArchive".

- 9 : Pas assez de mémoire pour effectuer l'opération. 

- 10 : Pas assez de place pour créer l'exécutable, ou l'utilisateur n'a pas les droits en écriture sur l'archive originale (.WDZ).

- 11 : L'icône n'a pas été trouvée ou l'icône n'est pas au bon format.

- 12 : Pas assez de place pour créer l'exécutable, ou l'utilisateur n'a pas les droits en lecture sur l'archive originale (.WDZ).

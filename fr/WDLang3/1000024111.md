


## &lt;Graphe&gt;.OpacitéSérie (Fonction)

***En anglais : &lt;Chart&gt;.SeriesOpacity***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie le facteur d'opacité d'une série dans un graphe.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Passe le pourcentage d'opacité de la série 1 à 50%
GRF_MonGraphe.OpacitéSérie(1, 50)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Champ Graphe>.OpacitéSérie(<Numéro de la série> , <Pourcentage d'opacité>)`
---

**`<Champ Graphe> : (Nom de champ)`**

Nom du champ Graphe manipulé (présent dans l'éditeur de fenêtres ou d'états).


**`<Numéro de la série> : (Entier)`**

Numéro de la série pour laquelle l'opacité doit être modifiée.

**`<Pourcentage d'opacité> : (Entier)`**

Pourcentage d'opacité de la série. Ce pourcentage est compris entre 0 (série complètement invisible) et 100 (série complètement opaque).



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
<a name="NOTE0_2"></a>


- Le changement d'opacité est pris en compte par tous les types de graphe. 

- L'opacité fixée pour la série est prioritaire sur l'opacité globale du champ Graphe (propriété [Opacité](../Proprietes/2514006.md)).

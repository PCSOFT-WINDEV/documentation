
## OLEServeur (Fonction)

***En anglais : OLEServer***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Liste les serveurs OLE enregistrés auprès du système en cours.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Initialiser le champ Liste avec les serveurs OLE disponibles
ListeAjoute(LISTE_SERVOLE, OLEServeur())
// Récupérer le serveur à lancer
Choix est une chaîne
Ind est un entier = ListeSelect(LISTE_SERVOLE)
Choix = LISTE_SERVOLE[Ind] + TAB
Choix = ExtraitChaîne(Choix, 2)
OLECréeVide(OLE_OLE1, Choix)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = OLEServeur()`
---

**`<Résultat> : (Chaîne de caractères)`**



- Liste des serveurs OLE au format : 
	
	```txt
	<Description 1> + TAB + <Nom du Serveur 1> + RC + ... 
	+ RC + <Description N> + TAB + <Nom du Serveur N>
	```
Le nom du serveur pourra ensuite être utilisé dans la fonction [OLECréeVide](../WDLang3/3057001.md). 
	Remarque : la liste des serveurs peut contenir des chaînes vides (à supprimer si nécessaire).

- Chaîne vide ("") en cas d'erreur.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Initialiser un champ Liste avec les serveurs OLE
<a name="initialiser_champ_liste_avec_les_serveurs_ole_ELTPARAGRAPHE000042"></a>Il est possible de remplir un champ Liste (ou un champ Combo) avec la liste des serveurs OLE en utilisant directement la fonction [ListeAjoute](../WDLang1/3049004.md). La syntaxe à utiliser est la suivante :

```txt
ListeAjoute(<Nom du champ Liste>, OLEServeur())
```



<a name="NOTE0_2"></a>




### Divers
<a name="divers_ELTPARAGRAPHE000055"></a>Seul OLE version 1 est géré.

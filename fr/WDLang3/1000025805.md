


## &lt;Carte&gt;.ModifieForme (Fonction)

***En anglais : &lt;Map&gt;.ModifyShape***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie une forme dessinée sur la carte affichée dans un champ Carte.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Modifie le rayon d'un cercle affiché dans le champ Carte
Cercle est un CarteCercle
Cercle.Rayon = 50
CARTE_MaCarte.ModifieForme(Cercle)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Champ Carte>.ModifieForme(<Forme>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la forme a pu être modifiée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Champ Carte> : (Nom de champ)`**

Nom du champ Carte à manipuler.

**`<Forme> : (Variable de type CarteCercle, CartePolygone ou CartePolyligne)`**

Forme à manipuler correspondant au nom d'une variable de type [CartePolygone](../WDLang3/1000025476.md), [CartePolyligne](../WDLang3/1000025489.md) ou [CarteCercle](../WDLang3/1000025504.md).



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- La forme doit avoir été ajoutée dans le champ Carte pour pouvoir être modifiée.

- Pour ajouter une nouvelle forme à dessiner sur une carte, utilisez la fonction [&lt;Carte&gt;.AjouteForme](../WDLang3/1000025801.md).

- Pour supprimer une forme dessinée dans un champ Carte, utilisez la fonction [&lt;Carte&gt;.SupprimeForme](../WDLang3/1000025820.md).

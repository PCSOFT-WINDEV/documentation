
## NotifAjoute (Fonction)

***En anglais : NotifAdd***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ajoute une notification dans la barre système de l'appareil (Android) ou dans le centre de notification (iOS).








<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
notif est une Notification
notif.Titre = "Nouveau message reçu"
notif.Message = "Cliquez pour afficher le message"
NotifAjoute(notif)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = NotifAjoute(<Notification>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la notification a été ajoutée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**Remarque** : Dans le simulateur, cette fonction est sans effet et renvoie toujours <u><u><u><u>Faux</u></u></u></u>. 

**`<Notification> : (Variable de type Notification)`**

Nom de la variable de type [Notification](../WDLang3/1000019441.md) contenant la description de la notification à afficher.

Si ce paramètre correspond à une notification déjà ajoutée, cette notification sera mise à jour. La fonction **NotifAjoute** aura alors le même effet que la fonction [NotifModifie](../WDLang3/1000019443.md).

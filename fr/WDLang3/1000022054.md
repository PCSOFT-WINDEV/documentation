


## btlePériphérique (Type de variable)

***En anglais : btleDevice***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **btlePériphérique** permet de manipuler un périphérique utilisant la technologie Bluetooth Low Energy.

Les caractéristiques de ce périphérique peuvent être définies et modifiées à l'aide de différentes propriétés WLangage. 

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
monPériph est un btlePériphérique
BTLEConnecte(monPériph)
```





<a name="NOTE0"></a>

## Remarques
<a name="NOTE0_1"></a>


### Propriétés spécifiques à la description des variables de type btlePériphérique
<a name="proprietes_specifiques_description_des_variables_type_btleperipherique_ELTPARAGRAPHE000039"></a>Les propriétés suivantes peuvent être utilisées pour manipuler une variable de type **btlePériphérique** :

| Nom de la propriété | Type manipulé | Effet |
| --- | --- | --- |
| Identifiant | Chaîne de caractères | Identifiant du périphérique.<br><br><br><br>Cette propriété est disponible en lecture seulement. |
| Nom | Chaîne de caractères | Nom du périphérique. Ce nom peut correspondre à une chaîne vide (""). <br><br>Cette propriété est disponible en lecture seulement. |
| Service | Tableau de [btleService](../WDLang3/1000022057.md) | Liste des services Bluetooth LE supportés par le périphérique. Il est nécessaire d'être connecté au périphérique (avec la fonction [BTLEConnecte](../WDLang3/1000021884.md)) pour accéder à cette propriété. Une erreur fatale sera déclenchée dans le cas contraire. |

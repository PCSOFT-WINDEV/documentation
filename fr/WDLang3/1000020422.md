
## Constantes NFC
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


Les constantes suivantes sont utilisées par les fonctions NFC.

| Constante | Détail | Fonction |
| --- | --- | --- |
| **nfcActivé** | Capteur NFC activé. | [NFCEtat](../WDLang3/1000020415.md) |
| **nfcDésactivé** | Capteur NFC désactivé. | [NFCEtat](../WDLang3/1000020415.md) |
| **nfcErrDésactivé** | Le NFC n'est pas activé sur l'appareil. | [NFCEcritTag](../WDLang3/1000020417.md) |
| **nfcErrAnnulation** | L'action a été annulée par l'utilisateur. | [NFCEcritTag](../WDLang3/1000020417.md) |
| **nfcErrDétection** | Le tag NFC est sorti du champ de détection pendant le processus d'écriture. | [NFCEcritTag](../WDLang3/1000020417.md) |
| **nfcErrEmulateur** | Appel de la fonction depuis l'émulateur. | [NFCEcritTag](../WDLang3/1000020417.md) |
| **nfcErrFormat** | Le format des données à écrire est invalide. | [NFCEcritTag](../WDLang3/1000020417.md) |
| **nfcErrInconnue** | Erreur inconnue. | [NFCEcritTag](../WDLang3/1000020417.md) |
| **nfcErrIndisponible** | Le capteur NFC n'existe pas ou est indisponible. | [NFCEcritTag](../WDLang3/1000020417.md) |
| **nfcErrLectureSeule** | Le tag NFC est en protégé en écriture. | [NFCEcritTag](../WDLang3/1000020417.md) |
| **nfcErrNDEF** | Le tag NFC ne peut pas être formaté au format NDEF (NFC Data Exchange Format). | [NFCEcritTag](../WDLang3/1000020417.md) |
| **nfcErrProtection** | Le tag NFC ne peut pas être protégé en écriture (mode non supporté par le tag). | [NFCEcritTag](../WDLang3/1000020417.md) |
| **nfcErrTailleMax** | La taille des données à écrire est supérieure à la taille maximale des données pouvant être stockées sur le tag NFC. | [NFCEcritTag](../WDLang3/1000020417.md) |
| **nfcInconnu** | Type du tag :  type inconnu. Valeur par défaut si aucune opération de lecture n'a été effectuée avec le tag. | Variable de type [nfcTag](../WDLang3/1000020411.md) |
| **nfcIndisponible** | L'appareil ne possède pas de capteur NFC ou celui-ci est indisponible. | [NFCEtat](../WDLang3/1000020415.md) |
| **nfcMedia** | Type de données : Données de type média. Peut correspondre à toute ressource identifiée par un type MIME (image, vidéo, document, etc.). | Variable de type [nfcDonnée](../WDLang3/1000020406.md) |
| **nfcMifareClassic** | Type du tag : NFC Mifare Classic | Variable de type [nfcTag](../WDLang3/1000020411.md) |
| **nfcOK** | L'opération a été effectuée. | [NFCEcritTag](../WDLang3/1000020417.md)<br><br> |
| **nfcTexte** | Type de données : Données de type texte. | Variable de type [nfcDonnée](../WDLang3/1000020406.md) |
| **nfcType1** | Type du tag : NFC Forum Type 1 | Variable de type [nfcTag](../WDLang3/1000020411.md) |
| **nfcType2** | Type du tag : NFC Forum Type 2 | Variable de type [nfcTag](../WDLang3/1000020411.md) |
| **nfcType3** | Type du tag : NFC Forum Type 3 | Variable de type [nfcTag](../WDLang3/1000020411.md) |
| **nfcType4** | Type du tag : NFC Forum Type 4 | Variable de type [nfcTag](../WDLang3/1000020411.md) |
| **nfcURI** | Type de données : Données de type URI (Uniform Ressource Identifier). Peut correspondre à :<br><br>- l'url d'une page web (http://www.pcsoft.fr),<br><br>- une adresse FTP (ftp://ftp.pcsoft.fr/),<br><br>- un numéro de téléphone (tel:0123456789),<br><br>- une adresse mail (mailto:adresse@mail.com), etc.<br><br><br> | Variable de type [nfcDonnée](../WDLang3/1000020406.md) |
| **nfcVide** | Type de données : Type vide | Variable de type [nfcDonnée](../WDLang3/1000020406.md) |

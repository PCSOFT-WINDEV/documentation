
## telCapacité (Fonction)

***En anglais : tapiCapability***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie les caractéristiques d'un périphérique de téléphonie. Permet de savoir si ce périphérique supporte les appels entrants de n'importe quel type, les appels vocaux et les appels fax.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI ETBinaire(telCapacité("LTP2"), telOptionTypeVoix) = telOptionTypeVoix ALORS 
	Info("Voix gérée")
FIN

SI ETBinaire(telCapacité("LTP2"), telOptionTypeFax) = telOptionTypeFax ALORS
	Info("Fax géré")
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = telCapacité(<Nom du périphérique>)`
---

**`<Résultat> : (Entier ou constante de type entier)`**



- 0 en cas d'erreur, 

- une ou plusieurs des constantes suivantes : 
	


|   |   |
| --- | --- |
| *telOptionTypeFax* | Périphérique acceptant les appels Fax |
| *telOptionTypeInconnu* | Périphérique acceptant les appels entrant de type inconnu. |
| *telOptionTypeVoix* | Périphérique acceptant les appels vocaux.<br>Remarque : Les modems Fax autorisant la numérotation automatique peuvent renvoyer cette constante. |






Pour identifier simplement le résultat, utilisez la fonction [ETBinaire](../WDLang1/3050010.md).

**`<Nom du périphérique> : (Chaîne de caractères)`**

Nom du périphérique à utiliser. La liste des périphériques de téléphonie est connue avec la fonction [telListePériphérique](../WDLang3/3076024.md).

## Remarques


### Configuration nécessaire
<a name="configuration_necessaire_ELTPARAGRAPHE000180"></a>Les fonctions de téléphonie utilisent la technologie TAPI 2.0 et/ou la technologie TAPI 3.1. Ces technologies sont utilisables sur tous les systèmes.


## OLECopie (Fonction)

***En anglais : OLECopy***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Copie l'objet présent dans un champ OLE vers le presse-papiers de Windows.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Création d'un objet Paint Brush
OLECréeVide(OLE_OLE1, "PBrush")
// L'objet est copié dans le presse-papiers
OLECopie(OLE_OLE1)
// L'objet est collé et sauvé dans un autre champ
OLEColle(OLE_OLE2)
OLESauve(OLE_OLE2, "OBJET.OLE")
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = OLECopie(<Champ OLE>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'objet a été copié, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Champ OLE> : (Nom de champ)`**

Nom du champ OLE qui contient l'objet OLE à copier. Si ce paramètre correspond à une chaîne vide (""), le champ considéré sera le champ auquel appartient l'événement en cours.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Seul OLE version 1 est géré.

- Le contenu du presse-papiers est écrasé. Le contenu du presse-papiers peut être de la forme texte, graphique ou binaire.

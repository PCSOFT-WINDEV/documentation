


## btleService (Type de variable)

***En anglais : btleService***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **btleService** permet de manipuler un service Bluetooth Low Energy ("Bluetooth GATT Service"). 

Chaque service Bluetooth LE est identifié par une chaîne de caractères unique (UUID) et contient un ensemble de caractéristiques. Une liste des différents services standard peut être trouvée sur le site 
[https://www.bluetooth.com/specifications/gatt/services](https://www.bluetooth.com/specifications/gatt/services).
Pour plus de détails sur les spécifications standard, vous pouvez également consulter : 
[https://www.bluetooth.com/specifications/protocol-specifications](https://www.bluetooth.com/specifications/protocol-specifications).

Les caractéristiques du service Bluetooth LE peuvent être définies et modifiées à l'aide de différentes propriétés WLangage. 

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
monPériph est un btlePériphérique
monService est un btleService <- monPériph.Service[1]
```

### Propriétés spécifiques à la description des variables de type btleService
<a name="proprietes_specifiques_description_des_variables_type_btleservice_ELTPARAGRAPHE000048"></a>Les propriétés suivantes peuvent être utilisées pour manipuler un service :

| Nom de la propriété | Type manipulé | Effet |

| UUID | Chaîne de caractères | Identifiant du service.<br><br>Cette propriété est disponible en lecture seulement. |

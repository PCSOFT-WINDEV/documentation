
## SaaSAdminSupprimeAbonnement (Fonction)

***En anglais : SaaSAdminDeleteSubscription***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime un abonnement SaaS.

**Remarques** : 

- L'utilisation de cette fonction nécessite d'avoir ouvert une session d'administration SaaS au préalable en utilisant la fonction [SaaSAdminConnecte](../WDLang3/1000019098.md).

- L'utilisation de cette fonction est réservée à l'Administrateur SaaS. Un gestionnaire de compte client ne peut pas utiliser cette fonction. 



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
S est un saasSite 
T est une saasTarification 
C est un saasClient 
Ab est un saasAbonnement

S = SaaSAdminChercheSite("Forum")
T = SaaSAdminChercheTarification("Promo 2010", S)
C = SaaSAdminChercheClient("Hybrid")

// Recherche de l'abonnement à supprimer
Ab = SaaSAdminChercheAbonnement(T, C)

// Le client Hybrid n'est plus abonné à la Promo 2010 du Forum
SaaSAdminSupprimeAbonnement(Ab)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Supprimer un abonnement (utilisation des variables de type saasxxx)

`<Résultat> = SaaSAdminSupprimeAbonnement(<Abonnement>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la suppression a été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Abonnement> : (Variable de type saasAbonnement)`**

Nom de la variable de type [saasAbonnement](../WDLang3/1000019088.md) correspondant à l'abonnement à supprimer de la base SaaS.


<a name="SYNTAXE2"></a>

### Supprime un abonnement identifié par le nom de la tarification associée

`<Résultat> = SaaSAdminSupprimeAbonnement(<Tarification> , <Site SaaS> , <Compte client>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la suppression a été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Tarification> : (Chaîne de caractères)`**

Nom de la tarification à laquelle le client est abonné.

**`<Site SaaS> : (Chaîne de caractères)`**

Nom du site SaaS possédant la tarification.

**`<Compte client> : (Chaîne de caractères)`**

Nom du compte client (ou référence externe).

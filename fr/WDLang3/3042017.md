


## grDestinationEMF (Fonction)

***En anglais : grDestinationEMF***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Définit un méta fichier comme destination d'un graphe (EMF signifie Enhanced Meta File). Ce fichier sera créé lors du prochain dessin du graphe (fonction [grDessine](../WDLang3/3042026.md)).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Définition du fichier "C:\Graphes\Graphe.EMF" comme destination du graphe "MonGraphe"
grDestinationEMF("MonGraphe", "C:\Graphes\Graphe.EMF")
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`grDestinationEMF(<Nom du graphe> , <Nom du fichier EMF>)`
---

**`<Nom du graphe> : (Chaîne de caractères)`**

Nom du graphe à manipuler, défini avec la fonction [grCrée](../WDLang3/3042005.md).

**`<Nom du fichier EMF> : (Chaîne de caractères)`**

Nom du fichier EMF dans lequel le graphe doit être enregistré. Ce fichier est créé lors de l'appel à la fonction [grDessine](../WDLang3/3042026.md).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Fichier EMF pour destination
<a name="fichier_emf_pour_destination_ELTPARAGRAPHE000041"></a>

- Lors du prochain appel à la fonction [grDessine](../WDLang3/3042026.md), le graphe sera dessiné dans le fichier &lt;Nom du fichier EMF&gt;.

- Ce fichier pourra être lu grâce à un outil reconnaissant le format EMF.

- Il est possible de définir la taille de ce fichier grâce à la fonction [grFenTaille](../WDLang3/3042016.md).

- Il est conseillé de visualiser le graphe à l'écran avant de l'enregistrer dans un fichier.

- La fonction **grDestinationEMF** doit être préférée à la fonction [grDestinationWMF](../WDLang3/3042019.md).



<a name="NOTE0_2"></a>


### Différences entre les fonctions grDestinationEMF et grSauveEMF
<a name="differences_entre_les_fonctions_grdestinationemf_grsauveemf_ELTPARAGRAPHE000064"></a>

- La fonction **grDestinationEMF** permet de définir la destination du prochain dessin du graphe. Lors de l'utilisation de la fonction [grDessine](../WDLang3/3042026.md), le graphe sera dessiné dans le fichier spécifié et toutes les modifications effectuées depuis le dernier dessin du graphe seront prises en compte.

- La fonction [grSauveEMF](../WDLang3/3042049.md) permet de sauvegarder le graphe directement dans un fichier au format EMF. L'image enregistrée correspond au graphe lors du dernier appel de la fonction [grDessine](../WDLang3/3042026.md). Si des modifications ont été effectuées sur le graphe depuis l'appel de la fonction [grDessine](../WDLang3/3042026.md), ces modifications ne seront pas prises en compte.



<a name="NOTE0_3"></a>


### Limites
<a name="limites_ELTPARAGRAPHE000087"></a>

- Cette fonction ne peut pas être utilisée avec les champs Graphe. 

- Cette fonction ne peut pas être utilisée sous l'éditeur d'états.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Graphe en 3 dimensions : Cette fonction ne peut pas être utilisée : le logiciel DirectX utilisé pour manipuler les graphes en 3 dimensions ne permet pas de créer des fichiers au format EMF.



<a name="NOTE0_4"></a>


### Créer et dessiner un graphe par programmation
<a name="creer_dessiner_graphe_par_programmation_ELTPARAGRAPHE000098"></a>Pour dessiner un graphe :

- créez le graphe (fonction [grCrée](../WDLang3/3042005.md)),

- ajoutez les données (fonctions [grAjouteDonnée](../WDLang3/3042023.md), [grBourseAjouteDonnée](../WDLang3/3042009.md) et [grNuageAjouteDonnéeXY](../WDLang3/3042032.md)),

- définissez la destination du graphe (fonctions [grDestinationBMP](../WDLang3/3042018.md), [grDestinationChamp](../WDLang3/3042020.md), **grDestinationEMF**, [grDestinationFen](../WDLang3/3042031.md) et [grDestinationWMF](../WDLang3/3042019.md)),

- définissez la destination du graphe à l'aide des fonctions suivantes : 

	- [grDestinationBMP](../WDLang3/3042018.md), 

	- [grDestinationChamp](../WDLang3/3042020.md), 

	- **grDestinationEMF**, 

	- [grDestinationFen](../WDLang3/3042031.md),

	- [grDestinationImage](../WDLang3/1000020952.md),

	- [grDestinationWMF](../WDLang3/3042019.md). 




- dessinez le graphe (fonction [grDessine](../WDLang3/3042026.md)).

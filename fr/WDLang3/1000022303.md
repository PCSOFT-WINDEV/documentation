


## &lt;Variable EmailSessionNotes&gt;.OuvreSession (Fonction)

***En anglais : &lt;EmailNotesSession variable&gt;.StartSession***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ouvre une session d'emails vers un serveur Lotus Domino afin de manipuler les messages. 

**Remarque** :
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Pour ouvrir une session avec le logiciel de messagerie Lotus Notes, utilisez la fonction [EmailOuvreSessionNotes](../WDLang3/3032121.md).
<a name="sample_code"></a>

## Exemple

<a name="3032028_Exemple1"></a>

```wl
// Ouvre une session d'emails 
MaSessionNotes est un emailSessionNotes
SI MaSessionNotes.OuvreSession() = Vrai ALORS
	// Connecté
SINON
	NomUser = ""
	Erreur("Impossible d'établir la connexion")
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Session>.OuvreSession()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la session a été ouverte, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. En cas d'erreur, la variable [ErreurDétectée](../WDLang1/3087001.md) est à <u><u><u><u>Vrai</u></u></u></u>. 
	Pour connaître le libellé de l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.




**`<Session> : (Variable de type emailSessionNotes)`**

Nom de la variable de type [emailSessionNotes](../WDLang3/1000018768.md) correspondant à la session d'emails à manipuler.

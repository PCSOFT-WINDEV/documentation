


## &lt;Carte&gt;.RécupèreImage (Fonction)

***En anglais : &lt;Map&gt;.GetImage***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Récupère l'image d'une carte affichée dans un champ Carte. Les marqueurs et les itinéraires affichés sur cette carte seront également présents dans l'image.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Récupération de l'image affichée dans le champ Carte et affectation à un champ Image
CARTE_MonChampCarte.RécupèreImage(CallbackCarteRécupèreImage)

PROCEDURE INTERNE CallbackCarteRécupèreImage(ImageCarte est une Image)
	IMG_Image = ImageCarte
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Champ Carte>.RécupèreImage(<Procédure WLangage>)`
---

**`<Champ Carte> : (Nom de champ)`**

Nom du champ Carte à manipuler.

**`<Procédure WLangage> : (Nom de procédure)`**

Nom de la procédure WLangage (également nommée "callback") appelée lorsque l'image du champ Carte sera disponible. 
Cette procédure est de la forme :

```txt
PROCEDURE <Nom de la procédure>(<Image>)
```
où &lt;Image&gt; est une variable de type [Image](../WDLang1/1000019650.md) contenant la carte affichée dans le champ.

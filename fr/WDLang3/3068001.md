
## SMSRAZ (Fonction)

***En anglais : SMSReset***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ré-initialise toutes les variables de la [structure SMS](../WDLang3/3068003.md).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ré-initialisation de la structure SMS
SMSRAZ()
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`SMSRAZ()`

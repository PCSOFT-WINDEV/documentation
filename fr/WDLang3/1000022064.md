


## btleDescripteur (Type de variable)

***En anglais : btleDescriptor***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **btleDescripteur** permet de manipuler le descripteur d'une caractéristique Bluetooth Low Energy. 

Ce descripteur décrit les données échangées entre deux appareils compatibles Bluetooth LE : unité de la valeur, plage de valeurs, ... Il est identifié par une chaîne de caractère unique (UUID).

Les caractéristiques de ce descripteur peuvent être définies et modifiées à l'aide de différentes propriétés WLangage. 

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
cMonService est un btleService
...
cMaCaractéristique est un btleCaractéristique <- cMonService[1]
cMonDesc est un btleDescripteur <- cMaCaractéristique.Descripteur[1]
```

<a name="proprietes_specifiques_description_des_variables_type_btledescripteur_ELTPARAGRAPHE000041"></a>Les propriétés suivantes peuvent être utilisées pour manipuler un descripteur :

| Nom de la propriété | Type manipulé | Effet |
| --- | --- | --- |

## Présentation
<a name="presentation_ELTTEXTE000067"></a>
Le DNS (Domain Name System) est un service permettant d'établir une correspondance entre une adresse IP et un nom de domaine. 

Le WLangage propose plusieurs fonctions permettant de manipuler simplement les adresses IP et les domaines sur un serveur DNS. 



## Fonctions WLangage
<a name="fonctions_wlangage_ELTTEXTE000091"></a>
Les fonctions de gestion des serveurs DNS sont les suivantes : 



|   |   |
| --- | --- |
| [DNSAjouteAdresse](../WDLang3/1000019400.md) | Ajoute une relation IP / Nom dans le serveur DNS. |
| [DNSListeAdresse](../WDLang3/1000019401.md) | Liste les adresses IP associées à un nom dans le serveur DNS. |
| [DNSSupprimeAdresse](../WDLang3/1000019402.md) | Supprime une relation Nom/Adresse IP dans un serveur DNS. |

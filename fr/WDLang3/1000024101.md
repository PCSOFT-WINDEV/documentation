


## &lt;Graphe&gt;.Nuage3DQuadrillage (Fonction)

***En anglais : &lt;Chart&gt;.Scatter3DGridlines***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Affiche le quadrillage pour les différents plans d'un graphe de type Nuage de points 3D. 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=grSurfaceQuadrillage.gif)
Remarque : Le quadrillage d'un graphe de type Nuage de points 3D peut également être activé/désactivé dans l'onglet "Général" de la fenêtre de description du champ Graphe.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
GRF_Graphe1.Nuage3DQuadrillage(grPlanXZ, grQuadrillageX + grQuadrillageZ)
GRF_Graphe1.Nuage3DQuadrillage(grPlanXY, grSansQuadrillage)
GRF_Graphe1.Nuage3DQuadrillage(grPlanXY, grSansQuadrillage)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Champ Graphe>.Nuage3DQuadrillage(<Plan> , <Axe>)`
---

**`<Champ Graphe> : (Nom de champ)`**

Nom du champ Graphe à manipuler (présent dans l'éditeur de fenêtres).

**`<Plan> : (Constantes de type Entier)`**

Plan sur lequel s'applique le quadrillage :


|   |   |
| --- | --- |
| *grPlanXY* | Quadrillage du plan XY. |
| *grPlanXZ* | Quadrillage du plan XZ. |
| *grPlanYZ* | Quadrillage du plan YZ. |



**`<Axe> : (Constante ou combinaison de constantes de type Entier)`**

Axe sur lequel s'applique le quadrillage : 


|   |   |
| --- | --- |
| *grPlanXY* | Le quadrillage est affiché sur le plan XY (disponible si le paramètre &lt;Plan&gt; correspond à la constante *grPlanXY*). |
| *grPlanXZ* | Le quadrillage est affiché sur le plan XZ (disponible si le paramètre &lt;Plan&gt; correspond à la constante *grPlanXZ*). |
| *grPlanYZ* | Le quadrillage est affiché sur le plan YZ (disponible si le paramètre &lt;Plan&gt; correspond à la constante *grPlanYZ*). |
| *grQuadrillageX* | Le quadrillage est affiché sur l'axe des X (disponible si le paramètre &lt;Plan&gt; correspond à la constante *grPlanXZ* ou à la constante *grPlanXY*). |
| *grQuadrillageY* | Le quadrillage est affiché sur l'axe des Y (disponible si le paramètre &lt;Plan&gt; correspond à la constante *grPlanXY* ou à la constante *grPlanYZ*). |
| *grQuadrillageZ* | Le quadrillage est affiché sur l'axe des Z (disponible si le paramètre &lt;Plan&gt; correspond à la constante *grPlanXZ* ou à la constante *grPlanYZ*). |
| *grSansQuadrillage* | Le quadrillage n'est pas affiché. |

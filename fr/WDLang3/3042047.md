


## grLégende (Fonction)

***En anglais : grLegend***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Indique la présence et la position de la légende dans un graphe.

Exemple de légende dans un graphe : <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Legende.gif)


**Remarque** : La légende regroupe les couleurs et les étiquettes des séries (ou de secteurs) d'un graphe. 



![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Les graphes](../WDChamp/3042060.md). 






<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Graphe créé par programmation
grCrée("MonGraphe", GrHisto)
grDestinationFen("MonGraphe", "", "Titre de la fenêtre")
// La légende est affichée à droite du graphe "MonGraphe"
grLégende("MonGraphe", grADroite)
...
grDessine("MonGraphe")
```
<a name="Exemple2"></a>

```wl
// Manipulation d'un champ Graphe
// La légende est affichée à gauche du graphe GRF_MonGraphe
grLégende(GRF_MonGraphe, grAGauche)
...
grDessine(GRF_MonGraphe)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`grLégende(<Nom du graphe> , <Position>)`
---

**`<Nom du graphe> : (Nom de champ ou chaîne de caractères)`**

Nom du graphe à manipuler. Ce nom correspond :

- soit au nom du graphe défini par programmation avec la fonction [grCrée](../WDLang3/3042005.md).

- soit au nom du champ Graphe présent dans l'éditeur de fenêtres, de pages ou d'états.




**`<Position> : (Entier)`**

Position de la légende du graphe :


|   |   |
| --- | --- |
| *grADroite* | Légende affichée à droite du graphe. |
| *grAGauche* | Légende affichée à gauche du graphe. |
| *grEnBas* | Légende affichée en-dessous du graphe. |
| *grEnHaut* | Légende affichée au-dessus du graphe. |
| *grSans* | Pas de légende. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Légende d'un graphe
<a name="legende_graphe_ELTPARAGRAPHE000153"></a>L'ensemble des étiquettes de séries correspond à la légende du graphe.<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=EtiquetteSerie.gif)
Les étiquettes de série ne sont pas utilisables avec les graphes de type Secteur.

La définition des étiquettes de séries est réalisée avec la fonction [grEtiquetteSérie](../WDLang3/3042024.md). 

Il est possible de modifier la police de la légende grâce à la fonction [grPoliceLégende](../WDLang3/3042037.md).  

La fonction **grLégende** permet d'indiquer la présence et la position de la légende (c'est-à-dire de l'ensemble des étiquettes de série). Les modifications effectuées avec la fonction **grLégende** seront prises en compte lors du prochain appel à la fonction [grDessine](../WDLang3/3042026.md). Il est possible de modifier la position de la légende d'un graphe à partir d'un menu contextuel.

**Remarque** : Pour définir des étiquettes de série dans des graphes de type "Nuage de points", il est conseillé d'utiliser la fonction [grNuageEtiquetteSérie](../WDLang3/3042038.md). Pour plus de détails, consultez [Légendes d'un graphe](../WDChamp/3042069.md). 
<a name="NOTE0_2"></a>


### Positionnement manuel de la légende : grLégende sans effet
<a name="positionnement_manuel_legende_grlegende_sans_effet_ELTPARAGRAPHE000193"></a>Si la légende est positionnée manuellement (option "Positionnement manuel" dans la fenêtre de la légende du graphe), la fonction **grLégende** n'a plus d'effet.

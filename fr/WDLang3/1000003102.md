
## DomoOuvre (Fonction)

***En anglais : DomoOpen***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet d'ouvrir une connexion avec une interface X10.

Dans cette version, seules les interfaces de type CM11 (de Marmitek) et CM15 sont supportées.

**Remarque** : Le bon fonctionnement du protocole X10 par courant porteur requiert une continuité électrique entre les appareils. Par exemple la présence de transformateur d'isolement empêche la transmission du courant porteur.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ouverture de la connexion X10
IDConnexion est un entier
IDConnexion = DomoOuvre("COM4", "A")
// Erreur rencontrée ?
SI IDConnexion = 0 ALORS
	Erreur(ErreurInfo(errMessage))
SINON	
	// Allumage des lampes reliées aux modules A3 et A4
	DomoAllume(IDConnexion, "A3 A4")
	// Equivalent à : DomoEnvoie(IDConnexion, "A3 A4", domoMarche)
FIN
...
// Fermeture de la connexion
DomoFerme(IDConnexion)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = DomoOuvre(<Nom du port> , <Code maison> [, <Type d'interface>])`
---

**`<Résultat> : (Entier)`**

Identifiant de la connexion, 0 en cas d'erreur. Cet identifiant sera utilisé dans toutes fonctions DomoXXX. En cas d'erreur, pour connaître le libellé de l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.

**`<Nom du port> : (Chaîne de caractères)`**

Nom du port série à ouvrir : "COM1", "COM2", "COM3, ... "COM32".

**Remarque** : ce paramètre est ignoré pour les interfaces CM15.

**`<Code maison> : (Chaîne de caractères)`**

Code maison que l'interface X10 doit gérer. Le code maison correspond à une lettre comprise entre A et P.

En cas de réinitialisation de l'interface X10, ce code maison est utilisé pour reconfigurer cette interface.

**`<Type d'interface> : (Chaîne de caractères optionnelle)`**

Type d'interface X10 reliée au poste en cours.


|   |   |
| --- | --- |
| "CM11" <br>(Valeur par défaut) | Interface de type CM11. |
| "CM15" | Interface de type CM15. |


## GroupeLit (Fonction)

***En anglais : GroupRead***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Lit un groupe de contacts présent dans le carnet d'adresses d'une messagerie Lotus Notes ou Outlook. Pour être lu, ce groupe doit avoir été précédemment : 

- lu (fonctions [GroupeDernier](../WDLang3/1000008112.md), [GroupePrécédent](../WDLang3/1000008114.md), [GroupePremier](../WDLang3/1000008115.md) ou [GroupeSuivant](../WDLang3/1000008117.md)), 

- ajouté (fonction [GroupeAjoute](../WDLang3/1000008111.md)), 

- modifié (fonction [GroupeModifie](../WDLang3/1000008113.md)).




**Attention** : Cette fonction **ne permet pas** de lire un groupe de contacts présent dans le carnet d'adresses d'une messagerie Outlook Express.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ouverture d'une session Lotus Notes
IDSession est un entier
IDSession = EmailOuvreSessionNotes("MotDePasse", "ServeurMars", ...
	 "mail\julie.nsf", "C:\Users\Julie\AppData\Local\Lotus\Notes\Data\user.id")
// Lecture du premier groupe de contacts
GroupePremier(IDSession)
...
// Re-lecture de ce même groupe de contacts
GroupeLit(IDSession, mGroupe.ID)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = GroupeLit(<Identifiant de la session> , <Identifiant du groupe>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le groupe de contacts a été lu, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour connaître le libellé de l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.




**`<Identifiant de la session> : (Entier)`**

Identifiant de la session à utiliser. Cet identifiant est renvoyé par une des fonctions suivantes : [EmailOuvreSessionNotes](../WDLang3/3032121.md), [EmailOuvreSessionOutlook](../WDLang3/3032128.md), [NotesOuvreBase](../WDLang3/1000005101.md) ou [OutlookOuvreSession](../WDLang3/3032123.md).

**`<Identifiant du groupe> : (Chaîne de caractères)`**

Identifiant du groupe de contacts à lire. Cet identifiant est contenu dans la variable mGroupe.ID.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Groupe de contact en cours
<a name="groupe_contact_cours_ELTPARAGRAPHE000083"></a>La variable mGroupe.EnDehors permet de savoir s'il y a un groupe de contacts courant.


<a name="NOTE0_2"></a>




### Version de Outlook utilisée
<a name="version_outlook_utilisee_ELTPARAGRAPHE000092"></a>Les groupes de contacts ne sont pas gérés par Outlook version 97. Il n'est donc pas possible de rechercher un groupe de contacts dans cette version de Outlook.

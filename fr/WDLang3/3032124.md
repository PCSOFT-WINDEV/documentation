
## FermeSession (Fonction)

***En anglais : CloseSession***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ferme une session avec le logiciel de messagerie Lotus Notes ou Outlook.

**Rappel** : Une session est ouverte grâce à la fonction [EmailOuvreSessionNotes](../WDLang3/3032121.md), [EmailOuvreSessionOutlook](../WDLang3/3032128.md), [OutlookOuvreSession](../WDLang3/3032123.md) ou [NotesOuvreConnexion](../WDLang3/1000005102.md).

Une session non terminée sera automatiquement fermée à la fin du programme WINDEV ou de la session WEBDEV.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ouverture d'une session 
IDSession est un entier
IDSession = EmailOuvreSessionNotes("MotDePasse", "ServeurMars", ...
		"mail/julie.nsf", "C:\Lotus Notes\julie.id")
// Si une session est ouverte, fermeture de cette session
SI IDSession <> 0 ALORS
	FermeSession(IDSession)
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`FermeSession(<Identifiant session>)`
---

**`<Identifiant session> : (Entier)`**

Identifiant de la session en cours. Cette valeur est renvoyée par la fonction [EmailOuvreSessionNotes](../WDLang3/3032121.md), [EmailOuvreSessionOutlook](../WDLang3/3032128.md), [OutlookOuvreSession](../WDLang3/3032123.md) ou [NotesOuvreConnexion](../WDLang3/1000005102.md).

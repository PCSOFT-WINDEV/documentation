
## SNMPOIDEtat (Fonction)

***En anglais : SNMPOIDStatus***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie l'état d'un OID.

**Remarque** : Cette fonction ne peut être utilisée qu'après avoir chargé le fichier MIB (fonction [SNMPChargeMIB](../WDLang3/1000017074.md)) qui décrit la correspondance entre les OID au format texte et numérique.  
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
bMIBChargé est un booléen
sCheminMIBs est une chaîne = ComplèteRep(fRepExe()) + "mibs\"
sMib est une chaîne
sMib = "RFC1382-MIB.mib" // MIB du protocole X25
nEtat est un entier
bMIBChargé = SNMPChargeMIB(sCheminMIBs + sMib)
SI bMIBChargé = Faux ALORS 
	Erreur("Erreur MIB :" + ErreurInfo())
SINON
	nEtat = SNMPOIDEtat("dod.internet.mgmt.mib-2.transmission.x25.x25AdmnInterruptTimer")
	SELON nEtat
		CAS snmpEtatObligatoire : Info("OID obligatoire")
		CAS snmpEtatOptionnel : Info("OID optionnel")
		CAS snmpEtatAncien : Info("OID ancien")
		CAS snmpEtatObsolète : Info("OID à ne plus utiliser")
		AUTRES CAS : Erreur(ErreurInfo())
	FIN
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = SNMPOIDEtat(<OID à traiter>)`
---

**`<Résultat> : (Constante de type Entier)`**

Constante d'état correspondant à l'état de l'OID décrit dans le fichier MIB :


|   |   |
| --- | --- |
| *snmpEtatAncien* | L'OID est ancien : il a été remplacé par un ou plusieurs autres OID mais sa valeur est toujours renseignée. |
| *snmpEtatInconnu* | Erreur : la fonction [ErreurInfo](../WDLang1/3013008.md) permet d'obtenir plus de renseignements sur le problème rencontré. |
| *snmpEtatObligatoire* | L'OID est obligatoire. |
| *snmpEtatObsolète* | L'OID est obsolète et ne doit plus être renseigné. |
| *snmpEtatOptionnel* | L'OID est optionnel. |



**`<OID à traiter> : (Chaîne de caractères)`**

Chaîne contenant l'OID sous forme de texte ou de numérique.

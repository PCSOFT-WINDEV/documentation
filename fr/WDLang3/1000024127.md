


## &lt;Graphe&gt;.SauveParamètre (Fonction)

***En anglais : &lt;Chart&gt;.SaveParameter***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Sauve les paramètres d'un graphe sous forme d'une chaîne. Ces paramètres pourront être restaurés avec la fonction [&lt;Graphe&gt;.ChargeParamètre](../WDLang3/1000023568.md).

**Rappel** : Par défaut, la configuration d'un graphe est automatiquement sauvegardée lors de l'arrêt de l'application. 



![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Les graphes](../WDChamp/3042060.md). 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Sauvegarde des paramètres d'un champ Graphe
INIEcrit("Parametres", "GRF_Graphe", "Param", GRF_Graphe.SauveParamètre())

// Restauration des paramètres d'un champ Graphe
GRF_Graphe.ChargeParamètre(INILit("Parametres", "GRF_Graphe", "Param"))
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Champ Graphe>.SauveParamètre()`
---

**`<Résultat> : (Chaîne de caractères Ansi)`**



- Paramètres du graphe. Cette chaîne peut être sauvegardée pour être ensuite restaurée par la fonction [&lt;Graphe&gt;.ChargeParamètre](../WDLang3/1000023568.md).

- Chaîne vide ("") en cas d'erreur.




**`<Champ Graphe> : (Nom de champ)`**

Nom du champ Graphe à manipuler (présent dans l'éditeur de fenêtres, de pages ou d'états).



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Les paramètres du graphe sauvegardés sont notamment : 

- le type du graphe.

- le titre du graphe et sa position.

- la position de la légende.

- l'espacement entre les barres pour les histogrammes.

- la profondeur pour les histogrammes.

- les caractéristiques du dégradé.

- l'orientation des axes.

- le quadrillage.

- les paramètres 3D pour les graphes en 3 dimensions.

- les caractéristiques des polices utilisées.

- les caractéristiques de lissage.


## zipFiltre (Fonction)

***En anglais : zipFilter***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la liste des formats d'archives gérés par la plateforme en cours, au format attendu par le filtre de la fonction [fSélecteur](../WDLang1/3036053.md).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Récupération de la liste des formats d'archives
ResFormatArchive = zipFiltre()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = zipFiltre()`
---

**`<Résultat> : (Chaîne de caractères)`**

Liste des formats d'archives gérés par la plateforme en cours sous la forme :

```txt
<Description> + TAB + <*.Extension> + RC + <Description2> + TAB + <*.Extension2>
```

où :

- &lt;Description&gt; correspond au nom du format. Par exemple : "Archive Winzip (\*.zip)".

- &lt;Extension&gt; correspond aux différentes extensions de &lt;Description&gt;.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Formats d'archives supportés
<a name="formats_archives_supportes_ELTPARAGRAPHE000053"></a>Les formats d'archives supportés sous Windows sont :

- Archive Winzip (\*.zip)

- Archive WINDEV (\*.wdz)

- Cabinet file (\*.cab)

- Roshal ARchive (\*.rar)

- Archive 7z (\*.7z)

- Tape ARchive (\*.tar)

- Tape ARchive compressée (\*.tgz;\*.tar.gz;\*.gz)

- Archive WINDEV auto-extractible (\*.exe)

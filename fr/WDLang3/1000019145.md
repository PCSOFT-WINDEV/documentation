
## SaaSAdminConnexionSite (Fonction)

***En anglais : SaaSAdminSiteConnection***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la connexion à la base commune ou à la base de référence d'un site SaaS.

**Remarques** : 

- L'utilisation de cette fonction nécessite d'avoir ouvert une session d'administration SaaS au préalable en utilisant la fonction [SaaSAdminConnecte](../WDLang3/1000019098.md).

- L'utilisation de cette fonction est réservée à l'Administrateur SaaS. Un gestionnaire de compte client ne peut pas utiliser cette fonction. 



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
S est un saasSite dynamique 
C est une Connexion

S = SaaSAdminChercheSite("Forum")
C = SaaSAdminConnexionSite(S)

// Affichage de la base de référence du site SaaS "Forum"
Info(C.BaseDeDonnées)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = SaaSAdminConnexionSite(<Site SaaS>)`
---

**`<Résultat> : (Variable de type Connexion)`**

Variable de type [Connexion](../WDLang4/1514073.md) contenant les paramètres de connexion à la base de données.

- Si le site n'utilise pas de base de données, la connexion renvoyée sera vide.

- Si le site utilise une base de données commune à tous les comptes clients, les informations de connexion à cette base seront renvoyées.

- Si le site utilise une base de données propre à chaque compte client, les informations de connexion à la base par défaut seront renvoyées.




**`<Site SaaS> : (Variable de type saasSite ou chaîne de caractères)`**

Nom de la variable de type [saasSite](../WDLang3/1000019068.md) (ou nom du site) correspondant au site SaaS pour lequel la connexion doit être récupérée.

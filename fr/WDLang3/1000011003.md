
## RendezVousModifie (Fonction)

***En anglais : AppointmentModify***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie le rendez-vous en cours : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) dans l'agenda d'une messagerie Lotus Notes ou Outlook.
	Lors de la lecture d'un rendez-vous, les données de ce rendez-vous sont mis à jour dans la [structure RendezVous](../WDLang3/1000011010.md). Pour modifier le rendez-vous en cours, il suffit de modifier les valeurs de la [structure RendezVous](../WDLang3/1000011010.md) et de valider les modifications avec la fonction **RendezVousModifie**.
	**Rappel** : Cette fonction ne permet pas de modifier le rendez-vous en cours dans l'agenda d'une messagerie Outlook Express.







<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ouverture d'une session Lotus Notes
IDSession est un entier
IDSession = EmailOuvreSessionNotes("MotDePasse", "ServeurMars", ...
	"mail/julie.nsf", "C:\Lotus Notes\julie.id")
// Lecture du dernier rendez-vous
RendezVousDernier(IDSession)
// Modification du lieu
mRendezVous.Lieu = "Salle 5H"
RendezVousModifie(IDSession)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Modifier un rendez-vous dans l'agenda d'une messagerie Lotus Notes ou Outlook

`<Résultat> = RendezVousModifie(<Identifiant de la session> [, <Identifiant du rendez-vous>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la modification a été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md). 




**`<Identifiant de la session> : (Entier)`**

Identifiant de la session à utiliser. Cet identifiant est renvoyé par une des fonctions suivantes : [EmailOuvreSessionNotes](../WDLang3/3032121.md), [EmailOuvreSessionOutlook](../WDLang3/3032128.md), [NotesOuvreBase](../WDLang3/1000005101.md) ou [OutlookOuvreSession](../WDLang3/3032123.md).  

**`<Identifiant du rendez-vous> : (Chaîne de caractères optionnelle)`**

Identifiant du rendez-vous à modifier. Cet identifiant est contenu dans la variable mRendezVous.ID.

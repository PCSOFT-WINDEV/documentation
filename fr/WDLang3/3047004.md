
## Constantes J2EE
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les constantes utilisées par les fonctions J2EE sont les suivantes :

| Constante | Description | Détail | Utilisée par les fonctions |
| --- | --- | --- | --- |
| **J2EEerrActeur** | Type d'erreur à retourner | URL du serveur ayant provoqué l'erreur (utile si la procédure est exécutée par plusieurs serveurs). | [J2EEErreur](../WDLang3/3047010.md) |
| **J2EEErrCode** | Type d'erreur à retourner | Erreur de la dernière procédure J2EE exécutée :<br><br>- "VersionMismatch" : La version J2EE utilisée sur le poste en cours est différente de celle utilisée sur le serveur.<br><br>- "MustUnderstand" : Le serveur ne comprend pas le paramètre spécifié dans l'entête (uniquement si le paramètre &lt;Exécution de la procédure&gt; de l'entête est initialisé à <u><u><u><u>Vrai</u></u></u></u>).<br><br>- "Client" : Mauvais paramétrage au niveau de l'exécution de la procédure.<br><br>- "Server" : Le serveur n'a pas exécuté la procédure.<br><br><br>**Remarque** : Cette chaîne de caractères peut être complétée d'informations supplémentaires. Le format utilisé est "&lt;Erreur&gt;.&lt;Information&gt;". Par exemple : "Client.Authentication" si l'erreur est due à l'authentification du client. | [J2EEErreur](../WDLang3/3047010.md) |
| **J2EEErrDétail** | Type d'erreur à retourner | Détail de l'erreur. Ce message est au format XML. | [J2EEErreur](../WDLang3/3047010.md) |
| **J2EEErrMessage** | Type d'erreur à retourner | Message d'erreur renvoyé par le serveur sur la dernière procédure J2EE exécutée. | [J2EEErreur](../WDLang3/3047010.md) |
| **J2EERésultat** | Type de résultat | Résultat de la dernière procédure J2EE exécutée | [J2EEDonneRésultat](../WDLang3/3047003.md) |
| **J2EERésultatHTTP** | Type de résultat | Résultat sous forme HTTP de la dernière procédure J2EE exécutée | [J2EEDonneRésultat](../WDLang3/3047003.md) |
| **J2EERésultatXML** | Type de résultat | Résultat sous forme XML de la dernière procédure J2EE exécutée | [J2EEDonneRésultat](../WDLang3/3047003.md) |
| **J2EETypeBinaireBase64** | Type du paramètre | Binaire | [J2EEExécute](../WDLang3/3047007.md) |
| **J2EETypeBinaireHexa** | Type du paramètre | Binaire hexadécimal | [J2EEExécute](../WDLang3/3047007.md) |
| **J2EETypeBooléen** | Type du paramètre | Booléen | [J2EEExécute](../WDLang3/3047007.md) |
| **J2EETypeChaîne** | Type du paramètre | Chaîne de caractères | [J2EEExécute](../WDLang3/3047007.md) |
| **J2EETypeChaîneNormalisée** | Type du paramètre | Chaîne de caractères sans RC ni TAB | [J2EEExécute](../WDLang3/3047007.md) |
| **J2EETypeDécimal** | Type du paramètre | Tout nombre sans limite | [J2EEExécute](../WDLang3/3047007.md) |
| **J2EETypeEntier** | Type du paramètre | Entier limité | [J2EEExécute](../WDLang3/3047007.md) |
| **J2EETypeEntierCourt** | Type du paramètre | Entier court | [J2EEExécute](../WDLang3/3047007.md) |
| **J2EETypeEntierCourtNonSigné** | Type du paramètre | Entier court non signé | [J2EEExécute](../WDLang3/3047007.md) |
| **J2EETypeEntierIllimité** | Type du paramètre | Entier illimité | [J2EEExécute](../WDLang3/3047007.md) |
| **J2EETypeEntierLong** | Type du paramètre | Entier long | [J2EEExécute](../WDLang3/3047007.md) |
| **J2EETypeEntierLongNonSigné** | Type du paramètre | Entier long non signé | [J2EEExécute](../WDLang3/3047007.md) |
| **J2EETypeEntierNégatif** | Type du paramètre | Entier négatif (sauf 0) | [J2EEExécute](../WDLang3/3047007.md) |
| **J2EETypeEntierNonNégatif** | Type du paramètre | Entier non négatif | [J2EEExécute](../WDLang3/3047007.md) |
| **J2EETypeEntierNonPositif** | Type du paramètre | Entier non positif | [J2EEExécute](../WDLang3/3047007.md) |
| **J2EETypeEntierNonSigné** | Type du paramètre | Entier non signé | [J2EEExécute](../WDLang3/3047007.md) |
| **J2EETypeEntierPositif** | Type du paramètre | Entier positif (sauf 0) | [J2EEExécute](../WDLang3/3047007.md) |
| **J2EETypeOctet** | Type du paramètre | Octet | [J2EEExécute](../WDLang3/3047007.md) |
| **J2EETypeOctetNonSigné** | Type du paramètre | Octet non signé | [J2EEExécute](../WDLang3/3047007.md) |
| **J2EETypeRéel** | Type du paramètre | Réel | [J2EEExécute](../WDLang3/3047007.md) |
| **J2EETypeRéelDouble** | Type du paramètre | Réel double | [J2EEExécute](../WDLang3/3047007.md) |
| **J2EETypeURL** | Type du paramètre | Adresse URL | [J2EEExécute](../WDLang3/3047007.md) |

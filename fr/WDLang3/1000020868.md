


## inAppProduit (Type de variable)

***En anglais : inAppProduct***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **inAppProduit** permet de définir toutes les caractéristiques avancées d'un produit ou d'un abonnement dans le cadre d'un achat In-App. Les caractéristiques de ce produit peuvent être définies et modifiées à l'aide de différentes propriétés WLangage.

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MesProduits est un tableau de inAppProduit
// Récupération à l'aide de inAppInfoProduit
// ...
Pour i = 1 _A_ MesProduits.Occurrence
	ZoneRépétéeAjouteLigne(ZR_Produits, MesProduits[i].Libellé, MesProduits[i].Description)
FIN
```

## Remarques
<a name="NOTE0_1"></a>


### Propriétés spécifiques à la description des variables de type inAppProduit
<a name="proprietes_specifiques_description_des_variables_type_inappproduit_ELTPARAGRAPHE000028"></a>Les propriétés suivantes peuvent être utilisées pour manipuler un produit In-App ou un abonnement In-App :

| Nom de la propriété | Type manipulé | Effet |
| --- | --- | --- |
| Description | Chaîne de caractères | Description du produit. <br><br>**Cette propriété est disponible en lecture seulement**. |
| Identifiant | Chaîne de caractères | Identifiant du produit ou de l'abonnement dans le store (iTunes Connect ou Google Play Store).<br><br>**Cette propriété est disponible en lecture seulement**. |
| Libellé | Chaîne de caractères | Libellé ou titre du produit ou de l'abonnement.<br><br>**Cette propriété est disponible en lecture seulement**. |
| Prix | Chaîne de caractères | <br><br>- Prix du produit dans la monnaie courante, formaté en utilisant le masque du téléphone. <br><br>- Tarif de l'abonnement dans la monnaie courante, formaté en utilisant le masque du téléphone. <br><br><br><br><br>**Cette propriété est disponible en lecture seulement**. |
|   | Constante de type Entier | Type de produit : <br><br>- *inAppAbonnement* : Abonnement.<br><br>- *inAppProduit* : Produit consommable et non consommable. <br><br><br><br><br>**Cette propriété est disponible en lecture seulement**. |




<a name="NOTE0_2"></a>


### Fonctions utilisant le type inAppProduit
<a name="fonctions_utilisant_type_inappproduit_ELTPARAGRAPHE000093"></a>Les fonctions manipulant des variables de type **inAppProduit** sont les suivantes : 



|   |   |
| --- | --- |
| [inAppAchèteProduit](../WDLang3/1000020873.md) | Envoie une demande d'achat d'un produit "In-App" associé à l'application. |
| [inAppConsommeProduit](../WDLang3/1000020880.md) | Permet de consommer dans une application un produit précédemment acheté par l'utilisateur. |
| [inAppListeInfoProduit](../WDLang3/1000020872.md) | Récupère à partir du store les informations sur les produits "In-App" associés à l'application. |


## DoubleAuthGénèreLien (Fonction)

***En anglais : TwoFactorAuthenticationGenerateLink***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Génère un lien pour enregistrer un compte dans une application d'authentification. Affecté à un champ Code-barres, ce lien permet d'afficher un QRCode pouvant être scanné dans l'application d'authentification choisie sur le téléphone. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
soit sURL = DoubleAuthGénèreLien(sCléTOTP, sEmail, sNomAppli)
CBA_MonQRCode = sURL
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = DoubleAuthGénèreLien(<Clé TOTP> , <Compte> , <Application>)`
---

**`<Résultat> : (Chaîne de caractères)`**

URL permettant de créer un compte dans une application d'authentification.

**`<Clé TOTP> : (Chaîne de caractères)`**

Chaîne ANSI qui contient la clé TOTP de l'utilisateur à authentifier. Cette clé TOTP a été générée avec la fonction [DoubleAuthGénèreClé](../WDLang3/1000025974.md). 

**`<Compte> : (Chaîne de caractères)`**

Nom du compte à enregistrer (généralement c'est l'adresse email qui est utilisée).

**`<Application> : (Chaîne de caractères)`**

Nom de l'application à enregistrer sur l'application d'authentification.

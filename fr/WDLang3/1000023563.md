


## &lt;Graphe&gt;.AxePosition (Fonction)

***En anglais : &lt;Chart&gt;.AxisPosition***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Indique la position à laquelle sera dessiné un des axes (abscisses ou ordonnées) du graphe. 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
GRF_MonGraphe.AxePosition(grAbscisse, grMin)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Champ Graphe>.AxePosition(<Axe> , <Position>)`
---

**`<Champ Graphe> : (Nom de champ)`**

Nom du champ Graphe à manipuler (présent dans l'éditeur de fenêtres, de pages ou d'états).



**`<Axe> : (Constante de type Entier)`**

Axe à positionner : 


|   |   |
| --- | --- |
| *grAbscisse* | Axe des abscisses (axe des X). |
| *grOrdonnée* | Axe des ordonnées (axe des Y). |



**`<Position> : (Constante de type Entier)`**

Position à laquelle l'axe doit être placé. 


|   |   |
| --- | --- |
| *grMax* | L'axe est placé sur la graduation maximale du graphe. |
| *grMin* | L'axe est placé sur la graduation minimale du graphe. |
| *grZéro* | L'axe est placé sur la graduation 0. <br><br>Cette constante correspond à l'affichage par défaut des graphes. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
<a name="NOTE0_2"></a>


- Cette fonction peut être utilisée sur tous les types de graphes affichant un axe.

- Cette fonction permet de modifier l'axe des ordonnées primaire. L'axe des ordonnées secondaire est toujours placé à l'opposé de l'axe des ordonnées primaire.

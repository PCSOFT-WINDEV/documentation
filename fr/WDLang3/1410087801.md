


## grCPValeurMaximale (Fonction)

***En anglais : grRBMaximumValue***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Fixe la valeur maximale du graphe circulaire de progrès. Par défaut, cette valeur est 100, ainsi les données sont exprimées en pourcentage. Cette valeur peut être changée et les pourcentages sont recalculés automatiquement.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
grCPValeurMaximale(GRF_Graphe1, 1212)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`grCPValeurMaximale(<Nom du graphe> , <Valeur maximale>)`
---

**`<Nom du graphe> : (Nom de champ ou chaîne de caractères)`**

Nom du graphe à manipuler. Ce nom correspond :

- soit au nom du graphe défini par programmation avec la fonction [grCrée](../WDLang3/3042005.md).

- soit au nom du champ Graphe présent dans l'éditeur de fenêtres ou d'états.




**`<Valeur maximale> : (Entier)`**

Nouvelle valeur maximale à prendre en compte. Cette valeur est strictement positive.
Par défaut, cette valeur vaut 100, ainsi les données sont exprimées en pourcentage. Lors du changement de cette valeur, les données du graphe sont recalculées automatiquement.

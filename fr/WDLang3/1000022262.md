


## grTypeSérie (Fonction)

***En anglais : grSeriesType***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Fixe le type d'une série utilisée dans un graphe composite (graphe composé de plusieurs graphes de types différents).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
grTypeSérie(GRF_MonGraphe, 1, grCourbe)
grTypeSérie(GRF_MonGraphe, 2, grNuage)
grTypeSérie(GRF_MonGraphe, 3, grHisto)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`grTypeSérie(<Nom du graphe> , <Numéro de la série> , <Type> [, <Paramètre>])`
---

**`<Nom du graphe> : (Nom de champ ou chaîne de caractères)`**

Nom du graphe à manipuler. Ce nom correspond :

- soit au nom du graphe défini par programmation avec la fonction [grCrée](../WDLang3/3042005.md).

- soit au nom du champ Graphe (présent dans l'éditeur de fenêtres ou d'états).




**`<Numéro de la série> : (Entier)`**

Numéro de la série pour laquelle le type de graphe doit être modifié.

**`<Type> : (Constante de type Entier)`**

Type de graphe à utiliser pour représenter les données de la série :



|   |   |
| --- | --- |
| *grAire* | Graphe de type Aire. |
| *grAirePolaire* | Graphe de type Aire Polaire. |
| *grBarCharts* | Graphe de type Boursier. |
| *grChandelier* | Graphe boursier de type Chandelier Japonais. |
| *grCourbe* | Graphe de type Courbe. |
| *grCourbeRégressionQuadratique* | Graphe de type Courbe de la forme "ax2 + bx + c". La courbure est déterminée en réalisant une régression quadratique sur la série spécifiée dans &lt;Paramètre&gt;. |
| *grDroiteConstante* | Affiche une droite horizontale à la position indiquée dans &lt;Paramètre&gt;. |
| *grDroiteMoyenne* | Affiche une droite horizontale à la moyenne de la série indiquée dans &lt;Paramètre&gt;. |
| *grDroiteRégressionLinéaire* | Affiche une droite dont la pente est calculée en faisant une régression linéaire sur la série spécifiée dans &lt;Paramètre&gt;. |
| *grGrapheBulle* | Graphe de type Graphe à bulle. |
| *grHisto* | Graphe de type Histogramme. |
| *grHistoEmpilé* | Graphe de type Histogramme empilé. |
| *grMinMax* | Graphe boursier affichant la différence entre la valeur minimale et la valeur maximale du cours. |
| *grNuage* | Graphe de type Nuage de points. |
| *grWaterfall* | Graphe de type Waterfall. |
| *grZoneIntervalle* | Graphe de type Zone d'intervalle. |



**`<Paramètre> : (Constante de type Entier)`**

Paramètre nécessaire pour certains types de graphe. Si &lt;Type&gt; correspond à : 

- *grCourbeRégressionQuadratique* : &lt;Paramètre&gt; correspond au numéro de la série utilisée pour faire la régression quadratique. 

- *grDroiteConstante* : &lt;Paramètre&gt; correspond à la position d'affichage de la droite horizontale. 

- *grDroiteMoyenne* : &lt;Paramètre&gt; correspond au numéro de la série utilisée pour calculer la moyenne.   

- *grDroiteRégréssionLinéaire* : &lt;Paramètre&gt; correspond au numéro de la série utilisée pour faire la régression linéaire. 
	Attention : La régression linéaire fonctionne uniquement avec une seule série. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
<a name="NOTE0_2"></a>
La fonction [grType](../WDLang3/3042062.md) annule tous les types de série spécifiques fixés par la fonction **grTypeSérie** : toutes les séries auront le même type.

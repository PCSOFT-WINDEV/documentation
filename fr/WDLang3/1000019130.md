
## SaaSAdminChercheTarification (Fonction)

***En anglais : SaaSAdminFindPricing***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Cherche une tarification SaaS.

**Remarques** : 

- L'utilisation de cette fonction nécessite d'avoir ouvert une session d'administration SaaS au préalable en utilisant la fonction [SaaSAdminConnecte](../WDLang3/1000019098.md).

- L'Administrateur SaaS aura accès à toutes les tarifications. Un gestionnaire de compte client aura accès uniquement aux tarifications des sites auxquels il est abonné. 



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
S est un saasSite dynamique
S = SaaSAdminChercheSite("Forum")

T est une saasTarification dynamique
T = SaaSAdminChercheTarification("Promo 2009", S)
SI ErreurDétectée = Vrai ALORS
	Erreur("La tarification n'a pas été trouvée")
SINON
	Info("La tarification a été trouvée")
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = SaaSAdminChercheTarification(<Tarification> , <Site SaaS>)`
---

**`<Résultat> : (Variable de type saasTarification)`**

Variable de type [saasTarification](../WDLang3/1000019091.md) contenant la description de la tarification recherchée.

Si la tarification n'est pas trouvée, la variable renvoyée est vide.

**`<Tarification> : (Chaîne de caractères)`**

Nom de la tarification recherchée dans le site SaaS.

**`<Site SaaS> : (Variable de type saasSite ou chaîne de caractères)`**

Nom de la variable de type [saasSite](../WDLang3/1000019068.md) ou nom du site SaaS dans lequel la tarification est recherchée.

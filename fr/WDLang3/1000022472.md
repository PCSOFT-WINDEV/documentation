


## &lt;Carte&gt;.SupprimeTout (Fonction)

***En anglais : &lt;Map&gt;.DeleteAll***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime dans un champ Carte : 

- tous les marqueurs, 

- toutes les formes,

- toutes les images.









<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Suppression de tous les marqueurs du champ Carte
CARTE_MaCarte.SupprimeTout()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Champ Carte>.SupprimeTout()`
---

**`<Champ Carte> : (Nom de champ)`**

Nom du champ Carte à manipuler.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Gestion des marqueurs : 

	- Pour supprimer un marqueur en particulier affiché dans un champ Carte, utilisez la fonction [&lt;Carte&gt;.SupprimeMarqueur](../WDLang3/1000022466.md).

	- Pour ajouter un nouveau marqueur sur un champ Carte, utilisez la fonction [&lt;Carte&gt;.AjouteMarqueur](../WDLang3/1000022422.md).

	- Pour modifier un marqueur affiché dans un champ Carte, utilisez la fonction [&lt;Carte&gt;.ModifieMarqueur](../WDLang3/1000022383.md).




- Gestion des itinéraires : Pour supprimer un itinéraire particulier, utilisez la fonction [&lt;Carte&gt;.SupprimeItinéraire](../WDLang3/1000022465.md).

- Gestion des images : 

	- Pour supprimer une image spécifique affichée dans un champ Carte, utilisez la fonction [&lt;Carte&gt;.SupprimeImage](../WDLang3/1000025829.md). 

	- Pour ajouter une nouvelle image dans un champ Carte, utilisez la fonction [&lt;Carte&gt;.AjouteImage](../WDLang3/1000025802.md). 

	- Pour modifier une image dans un champ Carte, utilisez la fonction [&lt;Carte&gt;.ModifieImage](../WDLang3/1000025818.md). 




- Gestion des formes : 

	- Pour supprimer une forme spécifique affichée dans un champ Carte, utilisez la fonction [&lt;Carte&gt;.SupprimeForme](../WDLang3/1000025820.md). 

	- Pour ajouter une nouvelle forme dans un champ Carte, utilisez la fonction [&lt;Carte&gt;.AjouteForme](../WDLang3/1000025801.md). 

	- Pour modifier une forme dans un champ Carte, utilisez la fonction [&lt;Carte&gt;.ModifieForme](../WDLang3/1000025805.md).

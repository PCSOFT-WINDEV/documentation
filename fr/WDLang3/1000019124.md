
## SaaSAdminChercheUtilisateur (Fonction)

***En anglais : SaaSAdminFindUser***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Recherche un utilisateur d'un compte client SaaS.

**Remarques** : 

- L'utilisation de cette fonction nécessite d'avoir ouvert une session d'administration SaaS au préalable en utilisant la fonction [SaaSAdminConnecte](../WDLang3/1000019098.md).

- L'Administrateur SaaS gère tous les utilisateurs de tous les clients. Un gestionnaire de compte client ne peut chercher des utilisateurs que dans son propre compte client. 



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonUtilisateur est un saasUtilisateur
MonUtilisateur = SaaSAdminChercheUtilisateur("sam@hybrid.com")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = SaaSAdminChercheUtilisateur(<Login>)`
---

**`<Résultat> : (Variable de type saasUtilisateur)`**

Variable de type [saasUtilisateur](../WDLang3/1000019083.md) contenant la description de l'utilisateur recherché.

Si la recherche échoue, la variable renvoyée est vide.

**`<Login> : (Chaîne de caractères)`**

Login de l'utilisateur recherché.

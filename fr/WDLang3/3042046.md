


## grLibellé (Fonction)

***En anglais : grLabel***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Indique divers paramètres de légende complémentaire dans un graphe.

Exemple de légendes complémentaires : ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=grLibell%E9%20-%20HC%20N%B0001.gif)




![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Les graphes](../WDChamp/3042060.md).






<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Graphe créé par programmation
// Affichage du libellé des étiquettes des catégories de données dans le graphe "MonGraphe"
grLibellé("MonGraphe", grAfficheEtiquette, Vrai)
```
<a name="Exemple2"></a>

```wl
// Champ Graphe
// Affichage du libellé des pourcentages des catégories de données dans le graphe "GRF_MonGraphe"
grLibellé(GRF_Graphe, grAffichePourcentage, Vrai)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`grLibellé(<Nom du graphe> , <Légende complémentaire> , <Affichage>)`
---

**`<Nom du graphe> : (Nom de champ ou chaîne de caractères)`**

Nom du graphe à manipuler. Ce nom correspond :

- soit au nom du graphe défini par programmation avec la fonction [grCrée](../WDLang3/3042005.md).

- soit au nom du champ Graphe présent dans l'éditeur de fenêtres, de pages ou d'états.




**`<Légende complémentaire> : (Constante)`**

Légende complémentaire à afficher avec le graphe :

- ***avec un graphe en secteurs*** : 
	


|   |   |
| --- | --- |
| *grAfficheEtiquette* | Étiquette des catégories de données. |
| *grAffichePourcentage* | Pourcentage des catégories de données. |
| *grAfficheValeur* | Valeur des catégories de données. |






- ***avec un histogramme*** :
	


|   |   |
| --- | --- |
| *grAfficheValeur* | Valeur des catégories de données. |






- ***avec un histogramme empilé*** : 
	


|   |   |
| --- | --- |
| *grAfficheValeurSiNulle* | Étiquette des catégories de données, même pour les valeurs nulles dans l'histogramme empilé. |






**`<Affichage> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la légende complémentaire doit être affichée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.

<a name="NOTE0_1"></a>

## Remarques


### Libellé d'un graphe
<a name="libelle_graphe_ELTPARAGRAPHE000173"></a>Le libellé du graphe peut être constitué de trois éléments :

- les étiquettes de catégories.

- la valeur des catégories.

- le pourcentage des catégories.


![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=grLibell%E9%20-%20HC%20N%B0001.gif)
Le libellé du graphe peut être affiché uniquement pour les graphes en secteur, histogrammes et histogrammes groupés.

La fonction **grLibellé** permet d'afficher les différents éléments du libellé du graphe. Les éléments affichés différent selon le type de graphe.

Les étiquettes de catégories affichées correspondent à celles définies avec la fonction [grEtiquetteCatégorie](../WDLang3/3042015.md).

La valeur des catégories dépendent des données du graphe.

Le pourcentage des catégories est automatiquement calculé.

L'affichage de ces informations peut être mis en forme grâce à la fonction [grMasque](../WDLang3/3042034.md).

Les modifications effectuées avec la fonction **grLibellé** seront prises en compte lors du prochain appel à la fonction [grDessine](../WDLang3/3042026.md).

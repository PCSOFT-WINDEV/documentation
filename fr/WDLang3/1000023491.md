


## grNuage3DLibelléGraduation (Fonction)

***En anglais : grScatter3DMarkCaption***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de spécifier le libellé d'une graduation pour un graphe de type Nuage de points 3D. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
grNuage3DLibelléGraduation(GRF_MonGraphe, grAbscisse, 1, "Janvier")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`grNuage3DLibelléGraduation(<Nom du graphe> , <Axe de la graduation> , <Valeur> , <Libellé>)`
---

**`<Nom du graphe> : (Nom de champ ou chaîne de caractères)`**

Nom du graphe de type Nuage de points 3D à manipuler. Ce nom correspond :

- soit au nom du graphe défini par programmation avec la fonction [grCrée](../WDLang3/3042005.md).

- soit au nom du champ Graphe (présent dans l'éditeur de fenêtres).




**`<Axe de la graduation> : (Constante de type Entier)`**

Type de l'axe sur lequel la graduation doit être affichée :


|   |   |
| --- | --- |
| *grAbscisse* | Axe des abscisses. |
| *grAltitude* | Axe des altitudes. |
| *grOrdonnée* | Axe des ordonnées. |



**`<Valeur> : (Réel)`**

Valeur de la graduation pour laquelle le libellé doit être modifié.  

**`<Libellé> : (Chaîne de caractères)`**

Nouveau libellé de la graduation. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Gestion des graduations
<a name="gestion_des_graduations_ELTPARAGRAPHE000080"></a>Pour que le libellé de la graduation soit affiché, il est nécessaire : 

- qu'une graduation existe à la valeur indiquée. Dans le cas contraire, le libellé de la graduation ne sera pas affiché.

- que la graduation ait une valeur. S'il existe une graduation sans valeur, le libellé ne sera pas affiché.




**Rappel** : Les graduations d'un graphe de type Nuage de points 3D peuvent être définies : 

- sous l'éditeur, dans la fenêtre de description du champ Graphe (onglet "Général"). 

- par programmation, grâce à la fonction [grGraduation](../WDLang3/3042027.md).

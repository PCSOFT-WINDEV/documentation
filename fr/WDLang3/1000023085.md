


## &lt;Variable mqttSession&gt;.Déconnecte (Fonction)

***En anglais : &lt;mqttSession variable&gt;.Disconnect***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de se déconnecter d'un serveur MQTT (également appelé "Broker").


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MaSessionMQTT est un mqttSession
...

MaSessionMQTT.Déconnecte()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Session MQTT>.Déconnecte()`
---

**`<Session MQTT> : (Variable de type mqttSession)`**

Nom de la variable [mqttSession](../WDLang3/1000023023.md) correspondant aux paramètres de la connexion à terminer.


## saasTarification (Type de variable)

***En anglais : saasPricing***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **saasTarification** permet de définir toutes les caractéristiques avancées d'une tarification SaaS. 

Les caractéristiques de cette tarification peuvent être définies et modifiées à l'aide de différentes propriétés WLangage. 

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonSite est un saasSite dynamique 
MonSite = SaaSAdminChercheSite("Gestion de voitures")

T est un saasTarification
T.Nom = "Offre Premium"
T.Prix = 150
SaaSAdminAjouteTarification(T, MonSite)

S est un saasService
S.Nom = "Accès Complet"

// Ajoute le service dans le site
SaaSAdminAjouteService(T, MonSite)

// La tarification proposera ce service
SaaSAdminActiveService(T, S)
```

## Remarques


### Propriétés spécifiques à la description des variables de type saasTarification
<a name="proprietes_specifiques_description_des_variables_type_saastarification_ELTPARAGRAPHE000030"></a>Les propriétés suivantes peuvent être utilisées pour manipuler une variable de type **saasTarification** :

| **Nom** | **Type** | **Effet** |
| --- | --- | --- |
| Facturation | Constante de type Entier | Indique si la tarification est par utilisateur ou au forfait.<br><br>Les constantes suivantes sont disponibles :<br><br>- *saasFacturationForfaitaire* : La tarification est facturée au forfait<br><br>- *saasFacturationParUtilisateur* : La tarification est facturée par utilisateur<br><br><br> |
| Nom | Chaîne de caractères | Nom de la tarification.<br>Ce nom doit être unique pour chaque Site. |
| Périodicité | Entier ou Constante de type Entier | Durée d'une période de facturation.<br>Cette durée est exprimée en nombre de jours.<br>La constante *saasPériodicitéIllimitée* permet d'indiquer une tarification sur une période illimitée. |
| Prix | Monétaire | Coût de la tarification.<br>Si la tarification est facturée par période ce prix concerne une période. |
| Site | Variable de type [saasSite](../WDLang3/1000019068.md) | Référence vers le site qui offre cette tarification.<br>Cette propriété est accessible en lecture seulement. |


## SMSLanceAppli (Fonction)

***En anglais : SMSRunApp***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Lance l'application native d'envoi de SMS de l'appareil (Android ou iPhone/iPad). Le contenu et les destinataires du SMS sont initialisés à partir des informations renseignées dans la [structure SMS](../WDLang3/3068003.md) du WLangage.

Attention : Le SMS n'est pas envoyé. Seuls les champs de l'application d'envoi de SMS sont initialisés. 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SMS.Numéro = "0610203040" 
SMS.Message = "J'envoie des SMS avec WINDEV Mobile !"
SMSLanceAppli()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`SMSLanceAppli()`
---

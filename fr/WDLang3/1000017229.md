
## BTPile (Fonction)

***En anglais : BTStack***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la pile Bluetooth en cours d'utilisation. Il est ainsi possible de proposer certaines fonctionnalités en fonction du type de la clé utilisée. En effet, certaines fonctions de manipulation des clés ne sont pas disponibles selon le type de clé utilisé.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
PileUtilisée est un entier
PileUtilisée = BTPile() 
SELON PileUtilisée 
	CAS btPileAucune : Info("Aucune pile n'est installée")
	CAS btPileBlueSoleil : Info("Pile BlueSoleil")
	CAS btPileBroadcom : Info("Pile Broadcom")
	CAS btPileMicrosoft : Info("Pile Microsoft")
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = BTPile()`
---

**`<Résultat> : (Constante de type Entier)`**

Type de pile utilisé par la clé Bluetooth :


|   |   |
| --- | --- |
| *btPileAucune* | La clé locale utilise une pile non supportée ou aucune clé Bluetooth n'est installée. |
| *btPileBlueSoleil* | La clé locale utilise une pile BlueSoleil. |
| *btPileBroadcom* | La clé locale utilise une pile Broadcom (anciennement nommée Widcomm). |
| *btPileMicrosoft* | La clé locale utilise une pile Microsoft. |

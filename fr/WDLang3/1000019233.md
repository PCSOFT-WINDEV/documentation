


## HTTPCookieLit (Fonction)

***En anglais : HTTPCookieRead***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Retourne la valeur d'un cookie reçu suite à une requête HTTP.

**Remarque** : Les cookies lus et envoyés par la fonction [HTTPRequête](../WDLang3/3043007.md) sont uniquement conservés en mémoire dans l'application. Ils n'ont pas de rapport avec les cookies des navigateurs Web installés sur le poste.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Effectue une requête HTTP
HTTPRequête("www.monsite.fr")
// Affiche le cookie d'identification renvoyé par le serveur
sID est une chaîne = HTTPCookieLit(".monsite.fr", "ID")
Info("Cookie d'identification = " + sID)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Syntaxe utilisable avec la fonction HTTPRequête

`<Résultat> = HTTPCookieLit(<Domaine> , <Nom>)`
---

**`<Résultat> : (Chaîne de caractères)`**



- Valeur du cookie.

- Chaîne vide ("") si le cookie n'existe pas. La variable [ErreurDétectée](../WDLang1/3087001.md) est alors à <u><u><u><u>Vrai</u></u></u></u>.




**`<Domaine> : (Chaîne de caractères)`**

Nom du domaine pour lequel le cookie a été reçu. 

**`<Nom> : (Chaîne de caractères)`**

Nom du cookie. 


<a name="SYNTAXE2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 
### Syntaxe utilisable avec la fonction HTTPEnvoie

`<Résultat> = HTTPCookieLit(<Requête HTTP> , <Domaine> , <Nom>)`
---

**`<Résultat> : (Chaîne de caractères)`**



- Valeur du cookie.

- Chaîne vide ("") si le cookie n'existe pas. La variable [ErreurDétectée](../WDLang1/3087001.md) est alors à <u><u><u><u>Vrai</u></u></u></u>.




**`<Requête HTTP> : (Variable de type httpRequête)`**

Nom de la variable [httpRequête](../WDLang3/1000021158.md) pour laquelle le cookie doit être lu.  

**`<Domaine> : (Chaîne de caractères)`**

Nom du domaine pour lequel le cookie a été reçu. 

**`<Nom> : (Chaîne de caractères)`**

Nom du cookie. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Pour utiliser la fonction **HTTPCookieLit**, la gestion des cookies doit avoir été activée par la fonction [HTTPCookieGère](../WDLang3/1000019218.md).

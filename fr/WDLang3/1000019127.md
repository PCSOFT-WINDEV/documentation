
## SaaSAdminAjouteTarification (Fonction)

***En anglais : SaaSAdminAddPricing***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ajoute une tarification SaaS.

**Remarques** : 

- L'utilisation de cette fonction nécessite d'avoir ouvert une session d'administration SaaS au préalable en utilisant la fonction [SaaSAdminConnecte](../WDLang3/1000019098.md).

- L'utilisation de cette fonction est réservée à l'Administrateur SaaS. Un gestionnaire de compte client ne peut pas utiliser cette fonction. 



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
S est un saasSite dynamique 
Tarification2010 est un saasTarification

S = SaaSAdminChercheSite("Forum")
Tarification2010..Nom = "Promo 2010"

// Création d'une tarification "Promo 2010" sur le site "Forum"
SaaSAdminAjouteTarification(Tarification2010, S)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Ajouter une tarification (utilisation des variables de type saasxxx)

`<Résultat> = SaaSAdminAjouteTarification(<Tarification> , <Site SaaS>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'ajout a été effectué, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Tarification> : (Variable de type saasTarification)`**

Nom de la variable de type [saasTarification](../WDLang3/1000019091.md) correspondant à la tarification à ajouter dans le site SaaS.

**`<Site SaaS> : (Variable de type saasSite ou chaîne de caractères)`**

Site SaaS recevant la nouvelle tarification.


<a name="SYNTAXE2"></a>

### Ajouter une tarification identifiée par son nom

`<Résultat> = SaaSAdminAjouteTarification(<Tarification> , <Site>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'ajout a été effectué, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Tarification> : (Chaîne de caractères)`**

Nom de la nouvelle tarification à ajouter dans le site SaaS.

**`<Site> : (Chaîne de caractères)`**

Nom du site SaaS recevant la nouvelle tarification.

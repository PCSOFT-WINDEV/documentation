
## GPSTermine (Fonction)

***En anglais : GPSEnd***





<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Indique la fin de l'utilisation des fonctions de géolocalisation. En effet, les fonctions de géolocalisation peuvent, selon le paramétrage choisi et la fréquence d'appel, consommer beaucoup de ressources sur le périphérique (batterie, bande passante, etc.). La fonction **GPSTermine** doit être appelée lorsque les fonctions de localisation ne sont plus utilisées par l'application.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Termine l'utilisation du GPS
GPSTermine()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`GPSTermine()`

## Remarques


### Fonctionnement
<a name="fonctionnement_ELTPARAGRAPHE000067"></a>

- Après l'appel à la fonction **GPSTermine**, les envois de notifications effectués par les fonctions [GPSSuitDéplacement](../WDLang3/1000019206.md), [GPSEtat](../WDLang3/1000019207.md) et [GPSDétectePosition](../WDLang3/1000019209.md) seront stoppés.

### Fonctionnalité d'application requise
<a name="fonctionnalite_application_requise_ELTPARAGRAPHE000228"></a>L'utilisation de cette fonction entraîne la déclaration d'une fonctionnalité d'application dans l'assistant de génération de l'application.

Fonctionnalité requise : Géolocalisation

Cette fonctionnalité permet aux applications d'accéder aux données de géolocalisation de l'appareil ou de l'ordinateur. 




- ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=WMGeolocalisation.gif) ***Exemples multiplateforme (WINDEV Mobile)*** : **WM Geolocalisation** <br>Cet exemple présente comment gérer des recherches de proximité avec géolocalisation :<br>- recherche autour de moi<br>- recherche dans une ville, à une adresse donnée, proche d'une adresse.<br>Les résultats apparaissent dans une zone répétée et dans une carte avec des marqueurs.
- ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=AndroidGPS.gif) ***Android (WINDEV Mobile)*** : **Android GPS** <br>Cet exemple montre l'utilisation des fonctions GPS du WLangage dans une application Android.<br>Il permet de récupérer à intervalle régulier : <br>- La latitude<br>- La longitude<br>- L'altitude<br>- La vitesse<br>- la direction




## Fonctions SMS
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Ces fonctions permettent d'envoyer et de lire facilement des SMS : 



|   |   |
| --- | --- |
| [SMSEnvoie](../WDLang3/3068002.md) | Envoie un SMS. |
| [SMSLanceAppli](../WDLang3/1000020864.md) | Lance l'application native d'envoi de SMS de l'appareil (Android ou iPhone/iPad). |
| [SMSNbMessage](../WDLang3/3068009.md) | Renvoie le nombre de SMS stockés dans la mémoire du téléphone Android. |
| [SMSPremier](../WDLang3/3068006.md) | Lit le premier SMS stocké dans la mémoire du téléphone Android. |
| [SMSRAZ](../WDLang3/3068001.md) | Ré-initialise toutes les variables de la [structure SMS](../WDLang3/3068003.md). |
| [SMSSuivant](../WDLang3/3068007.md) | Lit le SMS suivant stocké dans la mémoire du téléphone Android. |
| [SMSSupprime](../WDLang3/3068008.md) | Supprime un SMS stocké dans la mémoire du téléphone Android. Cette fonction ne doit plus être utilisée. |


## TâcheModifie (Fonction)

***En anglais : TaskModify***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie la tâche en cours dans la liste des tâches d'une messagerie Lotus Notes ou Outlook.

Lors de la lecture d'une tâche, les données de cette tâche sont mis à jour dans la [structure Tâche](../WDLang3/1000012010.md). Pour modifier la tâche en cours, il suffit de modifier les valeurs de la [structure Tâche](../WDLang3/1000012010.md) et de valider les modifications avec la fonction **TâcheModifie**.

**Attention** : Cette fonction **ne permet pas** de modifier la tâche en cours dans la liste des tâches d'une messagerie Outlook Express.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ouverture d'une session Lotus Notes
IDSession est un entier
IDSession = EmailOuvreSessionNotes("MotDePasse", "ServeurMars", "mail\julie.nsf",...
					"C:\Users\Julie\AppData\Local\Lotus\Notes\Data\user.id")

// Lecture de la première tâche
TâchePremier(IDSession)
// Modification de la date de début
mTâche.DateDébut = "20070125100000"
TâcheModifie(IDSession)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = TâcheModifie(<Identifiant de la session>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la modification a été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour connaître le libellé de l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*. 




**`<Identifiant de la session> : (Entier)`**

Identifiant de la session à utiliser. Cet identifiant est renvoyé par une des fonctions suivantes : [EmailOuvreSessionNotes](../WDLang3/3032121.md), [EmailOuvreSessionOutlook](../WDLang3/3032128.md), [NotesOuvreBase](../WDLang3/1000005101.md) ou [OutlookOuvreSession](../WDLang3/3032123.md).

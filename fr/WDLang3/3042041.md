


## grNuageTaillePoint (Fonction)

***En anglais : grScatterPointSize***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Définit la taille des points dans un graphe de type "Nuage de points". 

**Rappel** : Par défaut, la taille d'un point dans un graphe de type "Nuage de points" est égale à 1 pixel.



![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Les graphes](../WDChamp/3042060.md). 








<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Augmentation de la taille des points dans le graphe "MonGraphe".
// Ce graphe est de type "Nuage de points".
grNuageTaillePoint(GRF_MonGraphe, 3)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`grNuageTaillePoint(<Nom du graphe> , <Taille>)`
---

**`<Nom du graphe> : (Nom de champ ou chaîne de caractères)`**

Nom du graphe à manipuler. Ce nom correspond :

- soit au nom du graphe défini par programmation avec la fonction [grCrée](../WDLang3/3042005.md).

- soit au nom du champ Graphe présent dans l'éditeur de fenêtres, de pages ou d'états.




**`<Taille> : (Entier)`**

Taille d'un point (en pixels).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Taille d'un point
<a name="taille_point_ELTPARAGRAPHE000114"></a>Le point affiché est un carré dont la taille du côté est calculée grâce à la formule suivante : (&lt;Taille&gt;\*2)-1 pixels.<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Graphe2.gif)

<a name="NOTE0_2"></a>


### Points dans les graphes de type "Nuage de points"
<a name="points_dans_les_graphes_type_nuage_points_ELTPARAGRAPHE000124"></a>

- Dans les graphes de type "Nuage de points", un point est défini par ses coordonnées (abscisse et ordonnée).

- Les données définies avec la fonction **grNuageTaillePoint** seront prises en compte lors du prochain appel à la fonction [grDessine](../WDLang3/3042026.md).

- La fonction **grNuageTaillePoint** est utilisée uniquement pour les graphes de type "Nuage de points". Cette fonction est sans effet pour tous les autres types de graphes.

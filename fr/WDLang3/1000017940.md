


## grInfoPoint (Fonction)

***En anglais : grPointInfo***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie les coordonnées à l'écran d'un point ou d'une valeur présent dans un graphe.

Cette fonction permet par exemple de récupérer les coordonnées d'une zone d'un graphe pour pouvoir ensuite dessiner dans cette zone avec les fonctions de dessin.

**Remarque** : Cette fonction doit être utilisée après la fonction [grDessine](../WDLang3/3042026.md) (lorsque le graphe a été dessiné).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Coordonnées à l'écran d'un point du graphe créé par programmation
x est un entier
x = grInfoPoint("MonGraphe", grAbscisse, 1, 1)
```
<a name="Exemple2"></a>

```wl
// Coordonnées du centre d'un graphe de type Radar affiché dans un champ Graphe
Coordonnées est une chaîne
Coordonnées = grInfoPoint(GRF_Radar, grAbscisse, grAxeX, grMin) + "," + ...
		grInfoPoint(GRF_Radar, grOrdonnée, grAxeY, grMin)
```
<a name="Exemple3"></a>

```wl
// Coordonnées de la zone utile d'un graphe affiché dans un champ Graphe
Zone_Utile est une chaîne
Zone_Utile = grInfoPoint(GRF_Courbes, grAbscisse, grAxeX, grMin) + "," + ...
		grInfoPoint(GRF_Courbes, grAbscisse, grAxeX, grMax) + ":" + ...
		grInfoPoint(GRF_Courbes, grOrdonnée, grAxeY, grMin) + ","+ ...
		grInfoPoint(GRF_Courbes, grOrdonnée, grAxeY, grMax)
```
<a name="Exemple4"></a>

```wl
// Coordonnées de la zone comprise entre la coordonnée Y minimale et maximale
Zone est une chaîne
Zone = grInfoPoint(GRF_Courbes, grOrdonnée, 1, grMin) + "," + ...
		grInfoPoint(GRF_Courbes, grOrdonnée, 1, grMax)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = grInfoPoint(<Nom du graphe> , <Type de l'axe> , <Numéro de la série> , <Numéro de la catégorie>)`
---

**`<Résultat> : (Entier)`**

Coordonnée du point spécifié (en pixels) selon l'axe spécifié. Les coordonnées sont calculées par rapport au graphe (en non par rapport à l'écran).

**`<Nom du graphe> : (Nom de champ ou chaîne de caractères)`**

Nom du graphe à manipuler. Ce nom correspond :

- soit au nom du graphe défini par programmation avec la fonction [grCrée](../WDLang3/3042005.md).

- soit au nom du champ Graphe (présent dans l'éditeur de fenêtres ou de pages).




**`<Type de l'axe> : (Constante de type Entier)`**

Type de l'axe pris en compte pour le calcul des coordonnées :


|   |   |
| --- | --- |
| *grAbscisse*<br>(Valeur par défaut) | Calcul de la coordonnée sur l'axe des abscisses. |
| *grOrdonnée* | Calcul de la coordonnée sur l'axe des ordonnées. |



**`<Numéro de la série> : (Entier)`**



- Numéro de la série concernée.

- Constante permettant de connaître les coordonnées du point correspondant aux bornes des axes :
	


|   |   |
| --- | --- |
| *grAxeX* | Coordonnées des bornes de l'axe des X. |
| *grAxeY* | Coordonnées des bornes de l'axe des Y. |






**`<Numéro de la catégorie> : (Entier)`**



- Numéro de la catégorie concernée.

- Constante permettant de connaître les coordonnées du point correspondant aux bornes des axes :
	


|   |   |
| --- | --- |
| *grMax* | Coordonnées de la borne maximale. |
| *grMin* | Coordonnées de la borne minimale. |


## ContactLit (Fonction)

***En anglais : ContactRead***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Lit un contact présent dans le carnet d'adresses d'une messagerie Lotus Notes ou Outlook.
**Attention** : Cette fonction **ne permet pas** de lire un contact présent dans le carnet d'adresses d'une messagerie Outlook Express.

Pour être lu, ce contact doit avoir été précédemment : 

- lu (fonction [ContactDernier](../WDLang3/1000006103.md), [ContactPrécédent](../WDLang3/1000006105.md), [ContactPremier](../WDLang3/1000006106.md) ou [ContactSuivant](../WDLang3/1000006108.md)), 

- ajouté (fonction [ContactAjoute](../WDLang3/1000006101.md)), 

- modifié (fonction [ContactModifie](../WDLang3/1000006104.md)), 

- recherché (fonction [ContactCherche](../WDLang3/1000006102.md)). 




La fonction **ContactLit** ne positionne pas sur le contact lu.   

Lors de la lecture d'un contact, les données de ce contact sont mises à jour dans la [structure Contact](../WDLang3/1000006111.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ouverture d'une session Lotus Notes
IDSession est un entier
IDSession = EmailOuvreSessionNotes("MotDePasse", "ServeurMars", "mail/julie.nsf", ...
		"C:\Lotus Notes\julie.id")
// Lecture du premier contact
ContactPremier(IDSession)
...
// Re-lecture de ce même contact
ContactLit(IDSession, mContact.ID)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = ContactLit(<Identifiant de la session> , <Identifiant du contact>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le contact a été lu, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour connaître le libellé de l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.




**`<Identifiant de la session> : (Entier)`**

Identifiant de la session à utiliser. Cet identifiant est renvoyé par une des fonctions suivantes : [EmailOuvreSessionNotes](../WDLang3/3032121.md), [EmailOuvreSessionOutlook](../WDLang3/3032128.md), [NotesOuvreBase](../WDLang3/1000005101.md) ou [OutlookOuvreSession](../WDLang3/3032123.md).

**`<Identifiant du contact> : (Chaîne de caractères)`**

Identifiant du contact à lire. Cet identifiant est contenu dans la variable mContact.ID.

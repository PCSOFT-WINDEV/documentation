


## &lt;Carte&gt;.ModifieImage (Fonction)

***En anglais : &lt;Map&gt;.ModifyImage***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie une image dessinée sur la carte affichée dans un champ Carte.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Modifie la largeur d'une image affiché dans le champ Carte
MonImage est un CarteImage

MonImage.Largeur = 100
CARTE_MaCarte.ModifieImage(MonImage)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Champ Carte>.ModifieImage(<Image>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'image a pu être modifiée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Champ Carte> : (Nom de champ)`**

Nom du champ Carte à manipuler.

**`<Image> : (Variable de type CarteImage)`**

Nom de la variable de type [CarteImage](../WDLang3/1000025523.md) correspondant à l'image à modifier.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- L'image doit avoir été ajoutée dans le champ Carte pour pouvoir être modifiée.

- Pour ajouter une nouvelle image à dessiner dans le champ Carte, utilisez la fonction [&lt;Carte&gt;.AjouteImage](../WDLang3/1000025802.md).

- Pour supprimer une image dessinée dans le champ Carte, utilisez la fonction [&lt;Carte&gt;.SupprimeImage](../WDLang3/1000025829.md).

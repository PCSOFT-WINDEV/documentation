
## MagnetoAction (Fonction)

***En anglais : RecorderAction***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Exécute une action sur le magnétophone / dictaphone de l'appareil en cours.

**Remarque** : Cette fonction est disponible uniquement si l'appareil en cours possède une fonctionnalité magnétophone.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = MagnetoAction(<Action> [, <Nom du fichier>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'action définie a été lancée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.


En cas d'erreur, la variable [ErreurDétectée](../WDLang1/3087001.md) est à <u><u><u><u>Vrai</u></u></u></u>. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).

**`<Action> : (Constante de type entier)`**

Type d'action à exécuter :


|   |   |
| --- | --- |
| *magnetoEnregistre* | Ouvre le magnétophone et commence un enregistrement.<br>L'enregistrement est actif dès que l'utilisateur appuie sur le bouton appelant la fonction **MagnetoAction**. |
| *magnetoFerme* | Ferme le magnétophone. |
| *magnetoJoue* | Ouvre le magnétophone et joue le fichier son spécifié. |
| *magnetoStop* | Arrête l'enregistrement en cours.<br>L'enregistrement s'arrête dès que l'utilisateur relâche le bouton appelant la fonction **MagnetoAction**. |



**`<Nom du fichier> : (Chaîne de caractères optionnelle)`**

Nom et chemin complet du fichier son :

- à enregistrer (constante *magnetoEnregistre*).

- à jouer (constante *magnetoJoue*).

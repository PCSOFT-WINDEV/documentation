
## RécupèreThème (Fonction)

***En anglais : GetTheme***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de connaître le thème actuel d'une application mobile. Il est ainsi possible de gérer des cas particuliers (sélection d'une image spécifique en fonction du thème actuel par exemple) via l'événement "Application du thème clair/sombre". 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Evénement "Application du thème clair/sombre"
SI RécupèreThème() = stThèmeSombre ALORS
	IMG_ImageFond = "FondSombre.png"
SINON 
	IMG_ImageFond = "FondClair.png"
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = RécupèreThème()`
---

**`<Résultat> : (Constante de type Entier)`**

Thème actuellement utilisé par l'application : 



|   |   |
| --- | --- |
| *stThèmeClair* | <br><br>- L'application utilise le thème clair.<br><br>- La gestion du thème clair/sombre n'est pas activé dans l'application (option "Autoriser le passage en thème sombre (automatique ou manuellement via la fonction ChangeThème)" dans l'onglet "Style" de la description du projet). <br><br><br><br><br> |
| *stThèmeSombre* | L'application utilise le thème sombre. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Pour modifier le thème de l'application, utilisez la fonction [ChangeThème](../WDLang3/1000025266.md).

- Pour connaitre le thème actif sur l'appareil mobile, utilisez la fonction [SysThème](../WDLang1/1000019646.md).

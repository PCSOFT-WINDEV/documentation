


## Fonctions spécifiques à la communication iOS - Apple Watch
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions WLangage spécifiques à la communication entre les applications iOS et les applications Apple Watch associées sont les suivantes : 



|   |   |
| --- | --- |
| [awEnvoieEtRécupèreInfo](../WDLang3/1000022116.md) | Envoie des données d'un téléphone iOS vers une montre Apple Watch (ou inversement) en demandant une réponse. |
| [awEnvoieInfo](../WDLang3/1000022117.md) | Envoie une information d'un téléphone iOS vers une montre Apple Watch ou inversement. |
| [awProcédureRéceptionEtEnvoiInfo](../WDLang3/1000022118.md) | Précise la procédure WLangage appelée lors de la réception des informations provenant de la fonction [awEnvoieEtRécupèreInfo](../WDLang3/1000022116.md). |
| [awProcédureRéceptionInfo](../WDLang3/1000022119.md) | Précise la procédure WLangage appelée lors de la réception des informations provenant de la fonction [awEnvoieInfo](../WDLang3/1000022117.md). |


## iCloudCléValeurListeClé (Fonction)

***En anglais : iCloudKeyValueListKey***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Liste les clés contenues dans un conteneur de "clé-valeur" de Apple iCloud.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
soit tabClé = iCloudCléValeurListeClé()
POUR TOUT Clé DE tabClé
	Trace(Clé, iCloudCléValeurLit(Clé))
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = iCloudCléValeurListeClé()`
---

**`<Résultat> : (Tableau)`**

Tableau de chaînes de caractères correspondant à la liste des clés contenues dans le conteneur de "clé-valeur".



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Le nom de la clé doit faire moins de 64 octets en encodage UTF8.

- Une application peut utiliser au maximum 1024 clés.

- L'espace total pour le stockage des clés ne peut pas dépasser un 1 MégaOctets.

- Le partage des clés/valeurs s'effectue pour une même application sur des appareils utilisant le même compte iCloud.

- La synchronisation entre appareils peut prendre plusieurs minutes.

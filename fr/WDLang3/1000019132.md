
## SaaSAdminActiveService (Fonction)

***En anglais : SaaSAdminPricingEnableService***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Active un service dans une tarification SaaS.

**Remarques** : 

- L'utilisation de cette fonction nécessite d'avoir ouvert une session d'administration SaaS au préalable en utilisant la fonction [SaaSAdminConnecte](../WDLang3/1000019098.md).

- L'utilisation de cette fonction est réservée à l'Administrateur SaaS. Un gestionnaire de compte client ne peut pas utiliser cette fonction. 



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
S est un saasSite dynamique
T est une saasTarification dynamique
ServiceCRM est un saasService dynamique

S = SaaSAdminChercheSite("Forum")
T = SaaSAdminChercheTarification("Promo 2010", S)
ServiceCRM = SaaSAdminChercheService("CRM", S)

// Les abonnés de la tarification "Promo 2010" peuvent maintenant accéder à la section CRM du Forum
SaaSAdminActiveService(T, ServiceCRM)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Activer un service (utilisation des variables de type saasxxx)

`<Résultat> = SaaSAdminActiveService(<Tarification> , <Service SaaS>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'activation a réussi, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Tarification> : (Variable de type saasTarification)`**

Nom de la variable de type [saasTarification](../WDLang3/1000019091.md) correspondant à la tarification à laquelle un service est ajouté.

**Remarque** : Le service  doit avoir été créé au préalable par la fonction [SaaSAdminAjouteService](../WDLang3/1000019107.md).

**`<Service SaaS> : (Variable de type saasService)`**

Nom de la variable de type [saasService](../WDLang3/1000019096.md) correspondant au service SaaS qui devra être ajouté à la liste des services fournis par la tarification.


<a name="SYNTAXE2"></a>

### Activer un service identifié par son nom

`<Résultat> = SaaSAdminActiveService(<Tarification> , <Service SaaS> , <Site SaaS>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'activation a réussi, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Tarification> : (Chaîne de caractères)`**

Nom de la tarification à laquelle un service SaaS est ajouté.

**`<Service SaaS> : (Chaîne de caractères)`**

Nom du service SaaS qui devra être ajouté à la liste des services fournis par la tarification.

**`<Site SaaS> : (Chaîne de caractères)`**

Nom du site SaaS concerné par la tarification et le service.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Le service sera automatiquement rendu accessible à tous les abonnés à cette tarification (et pas seulement aux nouveaux abonnés).

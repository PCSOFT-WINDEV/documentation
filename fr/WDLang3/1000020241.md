
## Téléphone (Type de variable)

***En anglais : Phone***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **Téléphone** permet de manipuler un numéro de téléphone ainsi que son type (domicile, bureau, ...).

Les caractéristiques de cette variable peuvent être définies et modifiées à l'aide de différentes propriétés WLangage. 

Ce type de variable est utilisé par les variables de type [Contact](../WDLang3/1000020254.md).

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Compose le premier numéro de téléphone professionnel d'un contact
tel est un Téléphone
POUR TOUT tel DE contact.Téléphone
	SI tel.Type = telMobileBureau OU tel.Type = telBureau ALORS
		telDialerCompose(tel.Numéro)
		RETOUR
	FIN
FIN
```

## Remarques


### Propriétés spécifiques à la description des variables de type Téléphone
<a name="proprietes_specifiques_description_des_variables_type_telephone_ELTPARAGRAPHE000050"></a>Les propriétés suivantes peuvent être utilisées pour manipuler un numéro de téléphone :

| **Nom de la propriété** | **Type manipulé** | **Effet** |
| --- | --- | --- |
| Etiquette | Chaîne de caractères | Etiquette pour les types de numéro de téléphone personnalisé (Optionnel).<br>Si la propriété **Type** ne correspond pas à la constante *telPersonnalisé*, la valeur de cette propriété sera ignorée. |
| Numéro | Chaîne de caractères | Numéro de téléphone. Aucun formatage n'est effectué. |
| Type | Constante de type Entier | <br><br>![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Type du numéro : <br><br>- *telAutre* : autres types<br><br>- *telBureau* : téléphone fixe du bureau<br><br>- *telDomicile* : téléphone fixe du domicile<br><br>- *telFaxBureau* : fax du bureau<br><br>- *telFaxDomicile* : fax du domicile<br><br>- *telMobileBureau* : téléphone mobile du bureau<br><br>- *telMobileDomicile* : téléphone mobile du domicile<br><br><br><br><br> |


<a name="NOTE0_2"></a>


### Réinitialisation
<a name="reinitialisation_ELTPARAGRAPHE000118"></a>Pour réinitialiser le contenu d'une variable de type **Téléphone**, il est possible d'utiliser la fonction [VariableRAZ](../WDLang1/3013069.md).




## &lt;Graphe&gt;.Menu (Fonction)

***En anglais : &lt;Chart&gt;.Menu***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de choisir le type du menu contextuel affiché par un graphe.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Active le menu contextuel sous forme de fenêtre pour le champ GRF_MonGraphe
GRF_MonGraphe.Menu(grMenuFenêtre)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Champ Graphe>.Menu(<Type de menu>)`
---

**`<Champ Graphe> : (Nom de champ)`**

Nom du champ Graphe à manipuler (présent dans l'éditeur de fenêtres).

**`<Type de menu> : (Constante de type Entier)`**

Type de menu à afficher : 


|   |   |
| --- | --- |
| *grMenuAucun* | Aucun menu contextuel n'est affiché. <br><br>Remarque : Cette constante correspond à la valeur Faux utilisée dans les versions précédentes. |
| *grMenuFenêtre* | Un menu contextuel sous forme de fenêtre est affiché. |
| *grMenuPourCompatibilité* | Un menu contextuel déroulant est affiché. Cette option permet d'afficher le menu des graphes présent dans les versions antérieures à la version 23.  <br><br>Remarque : Cette constante correspond à la valeur Vrai utilisée dans les versions précédentes. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Pour modifier le texte des options du menu contextuel conservé par compatibilité, utilisez la fonction [&lt;Graphe&gt;.OptionMenu](../WDLang3/1000024112.md).

- Les données définies avec la fonction **&lt;Graphe&gt;.Menu** seront prises en compte lors du prochain appel à la fonction [&lt;Graphe&gt;.Dessine](../WDLang3/1000024062.md).

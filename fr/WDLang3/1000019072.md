
## saasClient (Type de variable)

***En anglais : saasClient***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **saasClient** permet de définir toutes les caractéristiques avancées d'un Compte Client SaaS. 

Les caractéristiques de ce compte client peuvent être définies et modifiées à l'aide de différentes propriétés WLangage. 

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
C est un saasClient
C.Nom = "Hybrid"

SaaSAdminAjouteClient(C)
```

<a name="NOTE0_1"></a>

## Remarques


### Propriétés spécifiques à la description des variables de type saasClient
<a name="proprietes_specifiques_description_des_variables_type_saasclient_ELTPARAGRAPHE000030"></a>Les propriétés suivantes peuvent être utilisées pour manipuler un compte client SaaS :

| **Propriété** | **Type manipulé** | **Effet** |
| --- | --- | --- |
| Informations | Chaîne de caractères | Informations diverses.<br>Cette propriété n'est accessible que par l'administrateur du Webservice SaaS (le compte "admin"). |
| Nom | Chaîne de caractères | Nom du compte client.<br>Généralement, le nom de la société cliente, sa raison sociale, ou les nom et prénom de la personne physique.<br>Cette propriété ne peut pas être vide et doit être unique pour chaque Webservice SaaS. |
| RéférenceExterne | Chaîne de caractères | La référence externe permet de stocker un lien vers un Système d'Information externe au Serveur d'Application WEBDEV.<br>Le contenu de cette propriété est libre et son emploi n'est pas obligatoire. |

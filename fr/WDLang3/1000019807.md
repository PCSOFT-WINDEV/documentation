
## VolumeRécupère (Fonction)

***En anglais : VolumeGet***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Récupère le réglage actuel du volume d'une source sonore.

**Remarque** : Cette fonction peut être utilisée dans l'émulateur.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
nVolume est un entier = VolumeRécupère(volumeSonnerie)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = VolumeRécupère([<Source> [, <Options>]])`
---

**`<Résultat> : (Entier)`**



- Valeur du volume de la source spécifiée, 

- -1 en cas d'erreur. Dans ce cas, la variable [ErreurDétectée](../WDLang1/3087001.md) est à <u><u><u><u>Vrai</u></u></u></u>. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Source> : (Constante optionnelle de type Entier)`**

Source sonore à manipuler. 


|   |   |
| --- | --- |
| *volumeAlarme* | Volume de l'alarme. |
| *volumeAppel* | Volume des appels. |
| *volumeDéfaut* | Volume de la source par défaut. Cette source dépend de la configuration du téléphone. |
| *volumeMusique* | Volume utilisé lors de la lecture d'un fichier audio ou vidéo. |
| *volumeNotification* | Volume du son émit lors d'une nouvelle notification. |
| *volumeSonnerie*<br>(Valeur par défaut) | Volume de la sonnerie du téléphone. |
| *volumeSystème* | Volume des sons système. |



**`<Options> : (Constante optionnelle de type Entier)`**

Type de volume à récupérer : 


|   |   |
| --- | --- |
| *volumeCourant*<br>(Valeur par défaut) | Volume courant de la source spécifiée. Ce volume est compris entre 0 et la valeur maximale du volume de la source. |
| *volumeCourantPourcentage* | Volume courant de la source spécifiée, exprimée en pourcentage de sa valeur maximale. Ce volume est compris entre 0 et 100. |
| *volumeMax* | Volume maximal de la source spécifiée. |







<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Pour modifier le réglage du volume pour une source sonore, utilisez la fonction [VolumeModifie](../WDLang3/1000020169.md).

- Pour modifier la source sonore associée aux boutons physiques de réglage du volume de l'appareil, utilisez la fonction [VolumeBoutonPhysique](../WDLang3/1000020168.md).

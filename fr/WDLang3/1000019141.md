
## SaaSAdminCréeBaseClient (Fonction)

***En anglais : SaaSAdminCreateClientDatabase***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Crée la base de données d'un compte client pour un site SaaS.

**Remarques** : 

- L'utilisation de cette fonction nécessite d'avoir ouvert une session d'administration SaaS au préalable en utilisant la fonction [SaaSAdminConnecte](../WDLang3/1000019098.md).

- L'utilisation de cette fonction est réservée à l'Administrateur SaaS. Un gestionnaire de compte client ne peut pas utiliser cette fonction. 



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
S est un saasSite dynamique 
C est un saasClient dynamique 

S = SaaSAdminChercheSite("Forum")
C = SaaSAdminChercheClient("Hybrid")

SI SaaSAdminCrééBaseClient(S, C) = Faux ALORS
	SI SaaSAdminCrééBaseClient(S, C, Faux) = Faux ALORS
		Erreur("Impossible de créer la base client.")
	SINON
		Info("La base a été créée vide.")
	FIN
SINON
	Info("La base a été créée depuis la base de référence.")
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = SaaSAdminCrééBaseClient(<Site SaaS> , <Compte client> [, <Base vide>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la base cliente a été créée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Site SaaS> : (Variable de type saasSite ou chaîne de caractères)`**

Nom de la variable de type [saasSite](../WDLang3/1000019068.md) (ou nom du site) correspondant au site SaaS pour lequel la base doit être créée.

**`<Compte client> : (Variable de type saasClient ou chaîne de caractères)`**

Nom de la variable de type [saasClient](../WDLang3/1000019072.md) ou nom du compte client SaaS pour lequel la base doit être créée.

Si ce paramètre est une chaîne de caractères, elle peut représenter le nom ou la propriété [RéférenceExterne](../WDLang3/1000019072.md) du compte client.

**`<Base vide> : (Booléen optionnel)`**



- <u><u><u><u>Faux</u></u></u></u> (valeur par défaut), la base de référence sera copiée pour créer la nouvelle base de données.

- <u><u><u><u>Vrai</u></u></u></u>, la base de référence définie pour le site sera ignorée et une base vide sera créée.

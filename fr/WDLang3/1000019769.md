
## SaaSAdminEcritParamètreSite (Fonction)

***En anglais : SaaSAdminWriteSiteParameter***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Enregistre une information spécifique au client. Cette fonction peut être exécutée en dehors du site SaaS concerné. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SaaSAdminEcritParamètreSite("Jet-Thames", "Dupont", "logo", "./logoPCSOFT.png")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Enregistrer une information 

`<Résultat> = SaaSAdminEcritParamètreSite(<Site SaaS> , <Client SaaS> , <Nom du paramètre> , <Valeur>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'écriture a réussi, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md). 




**`<Site SaaS> : (Chaîne de caractères)`**

Nom du site SaaS à manipuler. 

**`<Client SaaS> : (Chaîne de caractères)`**

Nom du compte client SaaS.

**`<Nom du paramètre> : (Chaîne de caractères)`**

Nom de la donnée à enregistrer. 

**`<Valeur> : (Chaîne de caractères)`**

Valeur de la donnée à enregistrer pour ce client. 


<a name="SYNTAXE2"></a>

### Enregistrer une information (utilisation des variables de type saasxxx)

`<Résultat> = SaaSAdminEcritParamètreSite(<Site SaaS> , <Compte Client> , <Nom du paramètre> , <Valeur>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'écriture a réussi, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md). 




**`<Site SaaS> : (Variable de type saasSite)`**

Nom de la variable de type [saasSite](../WDLang3/1000019068.md) correspondant au site SaaS à manipuler. 

**`<Compte Client> : (Variable de type saasClient)`**

Nom de la variable de type [saasClient](../WDLang3/1000019072.md) correspondant au compte client SaaS. 

**`<Nom du paramètre> : (Chaîne de caractères)`**

Nom de la donnée à enregistrer. 

**`<Valeur> : (Chaîne de caractères)`**

Valeur de la donnée à enregistrer pour ce client.

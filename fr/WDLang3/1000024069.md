***En anglais : &lt;Chart&gt;.DonutHolePercentage***

<DIV class="specObsolete">
	<IMG style="float:left;margin-right:10px;" src="https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=warning_big.png"><B>Avertissement</B><BR>
	A partir de la version <B>27</B>,  cette fonction est conservée par compatibilité (cette fonction sera encore supportée dans la prochaine version). Il est conseillé d'utiliser la fonction [&lt;Graphe&gt;.PourcentageTrou](../WDLang3/1410088090.md).

<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie le pourcentage correspondant au rayon du trou d'un graphe de type Beignet (ou "Donut"). 
 
![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Les graphes](../WDChamp/3042060.md). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Le trou représentera 50% du rayon total du donut/beignet
GRF_MonGraphe.DonutPourcentageTrou(50)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Champ Graphe>.DonutPourcentageTrou(<Pourcentage>)`
---

**`<Champ Graphe> : (Nom de champ)`**

Nom du champ Graphe à manipuler (présent dans l'éditeur de fenêtres, de pages ou d'états).


**`<Pourcentage> : (Entier)`**

Pourcentage représenté par le trou du donut/beignet (compris entre 0 et 70).



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Le paramétrage réalisé avec la fonction **&lt;Graphe&gt;.DonutPourcentageTrou** sera pris en compte lors du prochain appel à la fonction [&lt;Graphe&gt;.Dessine](../WDLang3/1000024062.md).

- Cette fonction peut être utilisée uniquement avec les graphes de type Beignet.


## LEDEteint (Fonction)

***En anglais : LEDSwitchOff***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Éteint la LED (Light-Emitting Diode) de l'appareil. Cette LED doit avoir été allumée par la fonction [LEDAllume](../WDLang3/1000019446.md). 

Attention : 

- L'appareil doit disposer d'une LED.

- Cette fonction est sans effet dans le simulateur ou l'émulateur.



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Allumage de la diode en rouge pendant 10 secondes
LEDAllume(RougeClair, 1000)

// Extinction de la LED
LEDEteint()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`LEDEteint()`

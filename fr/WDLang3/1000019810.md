
## LuminositéRécupère (Fonction)

***En anglais : BrightnessGet***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le réglage actuel de la luminosité de l'écran de l'appareil.

**Remarque** : Cette fonction peut être utilisée dans l'émulateur.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
nLuminosité est un entier = LuminositéRécupère()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = LuminositéRécupère()`
---

**`<Résultat> : (Entier)`**

Réglage actuel de la luminosité de l'écran en mode manuel, compris entre 0 et 100.





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Pour modifier la valeur du réglage actuel de la luminosité de l'écran, utilisez la fonction [LuminositéModifie](../WDLang3/1000020170.md).

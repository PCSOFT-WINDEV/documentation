
## RéseauMobileEtat (Fonction)

***En anglais : MobileNetworkStatus***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie l'état actuel de connexion aux données sur le réseau mobile ou demande à être notifié des changements d'état de la connexion aux données.

**Remarques** : 

- Cette fonction n'est pas utilisable dans le simulateur. Dans ce cas, elle renverra toujours 0. 

- Cette fonction ne concerne pas la connexion Wi-Fi mais la connexion par un opérateur de téléphonie. 





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI RéseauMobileEtat() = réseauConnecté ALORS
	NavigateurLanceAppli("www.pcsoft.fr")
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Récupérer l'état de la connexion aux données sur le réseau mobile

`<Résultat> = RéseauMobileEtat()`
---

**`<Résultat> : (Constante de type Entier)`**

Etat de la connexion aux données :


|   |   |
| --- | --- |
| *réseauConnecté* | Connecté. |
| *réseauConnexionEnCours* | Connexion en cours.<br><br>![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Cette constante n'est pas disponible. |
| *réseauDéconnecté* | Déconnecté. |




<a name="SYNTAXE2"></a>

### Être notifié du changement de l'état de la connexion aux données sur le réseau mobile

`<Résultat> = RéseauMobileEtat(<Procédure WLangage>)`
---

**`<Résultat> : (Constante de type Entier)`**

Etat de la connexion aux données :


|   |   |
| --- | --- |
| *réseauConnecté* | Connecté. |
| *réseauConnexionEnCours* | Connexion en cours.<br><br>![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Cette constante n'est pas disponible. |
| *réseauDéconnecté* | Déconnecté. |



**`<Procédure WLangage> : (Nom de procédure)`**

Procédure WLangage (également nommée "callback") appelée lorsque l'état de la connexion aux données change. 

Cette procédure est de la forme : 

```txt
PROCEDURE <Nom de la procédure>(<Nouvel état>)
```


Le paramètre &lt;Nouvel état&gt; est une constante de type entier correspondant au nouvel état de la connexion aux données : 


|   |   |
| --- | --- |
| *réseauConnecté* | Connecté. |
| *réseauConnexionEnCours* | Connexion en cours.<br><br>![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Cette constante n'est pas disponible. |
| *réseauDéconnecté* | Déconnecté. |

Si ce paramètre correspond à une chaîne vide (""), les notifications de changement d'état de la connexion ne sont plus envoyées. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Un changement d'état de la connexion peut intervenir lorsque le type de réseau utilisé change : par exemple lors du passage du réseau 2G au réseau 3G ou inversement. 

Pour connaître le type de réseau correspondant à la connexion courante, utilisez la fonction [RéseauMobileInfoConnexion](../WDLang3/1000020176.md).

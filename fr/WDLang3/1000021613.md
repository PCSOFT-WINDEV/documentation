
## FluxChangeDriver (Fonction)

***En anglais : StreamChangeDriver***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Indique la caméra à utiliser comme source pour un flux et/ou un champ Conférence. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Clic sur Caméra arrière :
FluxChangeDriver(CONF_Flux1, 2)

// Passe la caméra sur un autre champ Conférence
FluxChangeDriver(CONF_Flux2, 0) // Libère la caméra sinon l'appel suivant échoue

FluxChangeDriver(CONF_Flux1, 1)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Définir la caméra source à partir du numéro de caméra

`FluxChangeDriver(<Champ Conférence> , <Numéro de la caméra>)`
---

**`<Champ Conférence> : (Nom de champ ou chaîne de caractères)`**

Nom du champ Conférence ou nom du flux à manipuler. 

**`<Numéro de la caméra> : (Entier)`**

Numéro de la caméra à utiliser. 

- Si ce paramètre vaut 0, le flux est déconnecté de la caméra associée. 

- Si ce paramètre est invalide, la fonction échoue et renvoie une erreur non fatale. 

- Si le numéro de la caméra n'existe pas, le flux n'enverra plus d'images. 





<a name="SYNTAXE2"></a>

### Définir la caméra source à partir du driver de la caméra

`FluxChangeDriver(<Champ Conférence> , <Nom du driver>)`
---

**`<Champ Conférence> : (Nom de champ ou chaîne de caractères)`**

Nom du champ Conférence ou nom du flux à manipuler. 

**`<Nom du driver> : (Chaîne de caractères)`**

Nom de la caméra à utiliser. Ce nom correspond à un des noms renvoyé par la fonction [VidéoListeDriver](../WDLang1/3079008.md) (utilisée avec la constante *viDirectShow*). 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Si une communication est en cours, la source vidéo est modifiée.

- Pour connaître le nom des flux présents dans une fenêtre, affichez la fenêtre de description des flux de la fenêtre : 

	- Depuis la fenêtre de description du champ Conférence : Dans l'onglet "Général", cliquez sur "Editer les flux de la fenêtre". 

	- Depuis le menu contextuel du champ Conférence : Sélectionnez l'option "Editer les flux de la fenêtre".


## SaaSAdminActiveLicence (Fonction)

***En anglais : SaaSAdminSubscriptionAllocateLicense***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Attribue une licence d'abonnement à un utilisateur SaaS.

**Remarques** : 

- L'utilisation de cette fonction nécessite d'avoir ouvert une session d'administration SaaS au préalable en utilisant la fonction [SaaSAdminConnecte](../WDLang3/1000019098.md).

- L'Administrateur SaaS aura accès à toutes les licences. Un gestionnaire de compte client aura accès uniquement aux licences de ses utilisateurs. 



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
S est un saasSite
T est une saasTarification
C est un saasClient
Ab est un saasAbonnement
U est un saasUtilisateur

S = SaaSAdminChercheSite("Forum")
T = SaaSAdminChercheTarification("Promo 2010", S)
C = SaaSAdminChercheClient("Hybrid")
Ab = SaaSAdminChercheAbonnement(T, C)
U = SaaSAdminChercheUtilisateur("marc@hybrid.com")

// Autorise l'utilisateur marc@hybrid.com à profiter de la promo 2010
SaaSAdminActiveLicence(Ab, U)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Allouer une licence sur un abonnement

`<Résultat> = SaaSAdminActiveLicence(<Abonnement> , <Utilisateur>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'allocation a été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Abonnement> : (Variable de type saasAbonnement)`**

Nom de la variable de type [saasAbonnement](../WDLang3/1000019088.md) correspondant à l'abonnement à modifier dans la base SaaS.

**`<Utilisateur> : (Variable de type saasUtilisateur)`**

Nom de la variable de type [saasUtilisateur](../WDLang3/1000019083.md) identifiant l'utilisateur concerné. Cet utilisateur reçoit une licence l'autorisant à profiter de l'abonnement.





### Allouer d'une licence sur une tarification

`<Résultat> = SaaSAdminActiveLicence(<Tarification> , <Site SaaS> , <Utilisateur>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'allocation a été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Tarification> : (Chaîne de caractères)`**

Nom de la tarification concernée. Il doit exister un abonnement associé à cette tarification pour le compte client auquel appartient l'utilisateur.

**`<Site SaaS> : (Chaîne de caractères)`**

Nom du site SaaS possédant la tarification.

**`<Utilisateur> : (Chaîne de caractères)`**

Login de l'utilisateur qui reçoit une licence l'autorisant à profiter de l'abonnement.

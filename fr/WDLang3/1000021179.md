


## grSurfaceRécupèreDonnée (Fonction)

***En anglais : grSurfaceGetData***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Récupère une donnée dans un graphe de type Surface. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affichage d'une donnée dans le champ "GRF_MonGraphe". 
// Cette donnée est l'abscisse du premier point.
Info(grSurfaceRécupèreDonnée(GRF_MonGraphe, 1, grAbscisse))
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = grSurfaceRécupèreDonnée(<Nom du graphe> , <Indice> , <Valeur à récupérer>)`
---

**`<Résultat> : (Réel)`**

Valeur récupérée.

**`<Nom du graphe> : (Nom de champ ou chaîne de caractères)`**

Nom du graphe à manipuler. 
Ce nom correspond :

- soit au nom du graphe défini par programmation avec la fonction [grCrée](../WDLang3/3042005.md),

- soit au nom du champ Graphe (présent dans l'éditeur de fenêtres, de pages ou d'états).




**`<Indice> : (Entier)`**

Indice de la donnée déterminé par l'ordre d'ajout dans le graphe. 
Si la donnée correspondant à l'indice spécifié n'existe pas, la valeur récupérée est 0.

**`<Valeur à récupérer> : (Entier)`**

Indique la valeur à récupérer :


|   |   |
| --- | --- |
| *grAbscisse* | Abscisse du point à récupérer. |
| *grAltittude* | Altitude du point à récupérer. |
| *grOrdonnée* | Ordonnée du point à récupérer. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Rappel : Pour les données d'un graphe de type Surface, le plan XY constitue le plancher de la surface, tandis que Z indique la hauteur du point.

- Pour récupérer une donnée dans un graphe de type Boursier, il est conseillé d'utiliser la fonction [grBourseRécupèreDonnée](../WDLang3/1000021174.md).

- Pour récupérer une donnée dans un graphe de type Nuage de points, il est conseillé d'utiliser la fonction [grNuageRécupèreDonnéeXY](../WDLang3/1000021175.md).

- Pour récupérer une donnée dans un graphe, utilisez la fonction [grRécupèreDonnée](../WDLang3/1000021171.md).

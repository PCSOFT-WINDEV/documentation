


## &lt;Carte&gt;.PopupAffichée (Fonction)

***En anglais : &lt;Map&gt;.PopupDisplayed***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de savoir si la popup d'un marqueur du champ Carte est affichée.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonMarqueur est un Marqueur

SI CARTE_MaCarte.PopupAffichée(MonMarqueur) ALORS
	...
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Champ Carte>.PopupAffichée(<Marqueur>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la popup du marqueur est affichée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Champ Carte> : (Nom de champ)`**

Nom du champ Carte à manipuler.

**`<Marqueur> : (Variable de type Marqueur)`**

Nom de la variable de type [Marqueur](../WDLang3/1000019940.md) dont on veut savoir si une popup est affichée.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Pour afficher la popup, utilisez la fonction [&lt;Carte&gt;.AffichePopup](../WDLang3/1000025799.md).

- Pour fermer la popup, utilisez la fonction [&lt;Carte&gt;.FermePopup](../WDLang3/1000025803.md).

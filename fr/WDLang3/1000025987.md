
## Constantes de gestion de la synthèse vocale
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les constantes utilisées par les fonctions de la synthèse vocale sont les suivantes :

| Constante | Détail | Fonction / Type de variable |
| --- | --- | --- |
| **svQualitéBasse** | Qualité de la voix : Basse qualité. | Type [VoixDeSynthèse](../WDLang3/1000025986.md) |
| **svQualitéHaute** | Qualité de la voix : Haute qualité. | Type [VoixDeSynthèse](../WDLang3/1000025986.md) |
| **svQualitéNormale** | Qualité de la voix : Qualité normale. | Type [VoixDeSynthèse](../WDLang3/1000025986.md) |
| **svQualitéTrèsBasse** | Qualité de la voix : Très basse qualité. | Type [VoixDeSynthèse](../WDLang3/1000025986.md) |
| **svQualitéTrèsHaute** | Qualité de la voix : Très haute qualité. | Type [VoixDeSynthèse](../WDLang3/1000025986.md) |
| **svVoixFéminine** | Genre de la voix de synthèse : Voix féminine. | Type [VoixDeSynthèse](../WDLang3/1000025986.md) |
| **svVoixIndéfinie** | Genre de la voix de synthèse : Voix neutre. | Type [VoixDeSynthèse](../WDLang3/1000025986.md) |
| **svVoixMasculine** | Genre de la voix de synthèse : Voix masculine. | Type [VoixDeSynthèse](../WDLang3/1000025986.md) |

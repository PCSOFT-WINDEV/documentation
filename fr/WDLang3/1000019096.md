
## saasService (Type de variable)

***En anglais : saasService***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **saasService** permet de définir toutes les caractéristiques avancées d'un service SaaS. 

Les caractéristiques de ce Service peuvent être définies et modifiées à l'aide de différentes propriétés WLangage. 

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonService est un saasService
MonService.Nom = "Accès"

SiteActuel est un saasSite

SaaSAdminAjouteService(MonService, SiteActuel)
```

<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Propriétés spécifiques à la description des variables de type saasService
<a name="proprietes_specifiques_description_des_variables_type_saasservice_ELTPARAGRAPHE000030"></a>Les propriétés suivantes peuvent être utilisées pour manipuler un service SaaS :

| **Nom de la propriété** | **Type manipulé** | **Effet** |
| --- | --- | --- |
| Nom | Chaîne de caractères | Nom du service. Ce nom sera utilisé pour vérifier l'autorisation d'accès grâce à la fonction [SaaSVérifieService](../WDLang3/1000018961.md). |
| Site | Variable de type [saasSite](../WDLang3/1000019068.md) | Site SaaS auquel le service appartient. <br>Cette propriété est accessible en lecture seulement. |




<a name="NOTE0_2"></a>


### Fonctions utilisant le type SaaService
<a name="fonctions_utilisant_type_saaservice_ELTPARAGRAPHE000064"></a>

Les fonctions suivantes peuvent être utilisées pour manipuler une variable de type **saasService** :



|   |   |
| --- | --- |
| [SaaSAdminActiveService](../WDLang3/1000019132.md) | Active un service dans une tarification SaaS. |
| [SaaSAdminAjouteService](../WDLang3/1000019107.md) | Ajoute un service SaaS dans la description d'un site. |
| [SaaSAdminChercheService](../WDLang3/1000019110.md) | Recherche de la description d'un service SaaS. |
| [SaaSAdminDésactiveService](../WDLang3/1000019133.md) | Désactive un service dans une tarification SaaS. |
| [SaaSAdminModifieService](../WDLang3/1000019108.md) | Modifie un service SaaS. |
| [SaaSAdminSupprimeService](../WDLang3/1000019109.md) | Supprime un service SaaS. |


## NetListeRep (Fonction)

***En anglais : NetDirList***

<a name="Utilisation"></a>
<a name="description"></a>
Liste les répertoires et les fichiers présents :

- sur le poste serveur FTP (File Transfer Protocol) ou RPC (Remote Procedure Call) réalisé avec WINDEV,

- sur le poste client.




**Remarque** : Il est possible d'autoriser ou non l'utilisation de cette fonction sur un serveur FTP/RPC grâce à la fonction [NetOptionServeur](../WDLang3/3056022.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
ConnectRPC est un entier
ConnectRPC = NetConnecte("148.61.125.245", ServeurRPC, "GUEST", "")
...
// Affiche la liste des répertoires et des fichiers présents sur le disque C: du poste serveur
Info(NetListeRep(ConnectRPC, "C:\*.*"))
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = NetListeRep([<Identifiant connexion>, ] <Nom répertoire de base>)`
---

**`<Résultat> : (Chaîne de caractères Ansi)`**

Liste des répertoires et des fichiers (avec leur taille) présents sur le poste serveur ou client. Ce résultat est de la forme : 

```txt
<Répertoire1> + TAB + <Répertoire2> + TAB + ... + TAB + <RépertoireN> + RC + ...
<Fichier1/Taille du fichier 1 en Ko> + TAB + <Fichier2/Taille du fichier 2 en Ko> + ...
TAB + ... + TAB + <FichierN/Taille du fichier N en Ko>
```


**`<Identifiant connexion> : (Entier optionnel)`**

Identifiant de la connexion du poste client (obtenu par la fonction [NetConnecte](../WDLang3/3056004.md)). Si ce paramètre n'est pas précisé, la fonction **NetListeRep** renvoie la liste des répertoires et des fichiers présents sur le poste client en cours.

**`<Nom répertoire de base> : (Chaîne de caractères Ansi)`**

Répertoire du poste serveur ou client dont on veut lister les fichiers et les sous-répertoires. Il est nécessaire de préciser le filtre utilisé pour la recherche des fichiers (par exemple, \*.\*).


## NetMACVersIP (Fonction)

***En anglais : NetMACToIP***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie l'adresse IP correspondant à une adresse MAC donnée.  
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Renvoie l'adresse IP
Info(NetMACVersIP("11:22:33:44:55:66"))
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = NetMACVersIP(<Adresse MAC> [, <Délai>])`
---

**`<Résultat> : (Chaîne de caractères)`**



- Adresse IP obtenue (au format xxx.xxx.xxx.xxx), 

- Chaîne vide ("") en cas de problème.




**`<Adresse MAC> : (Chaîne de caractères)`**

Adresse MAC au format "aa:bb:cc:dd:ee:ff" ou au format "aa-bb-cc-dd-ee-ff".

**`<Délai> : (Entier optionnel)`**

Durée maximale (en millisecondes) pendant laquelle la conversion de l'adresse MAC vers IP peut être réalisée. Ce paramètre peut correspondre à : 

- un entier correspondant au nombre de millisecondes, 

- une variable de type Durée, 

- l'indication directe de la durée (par exemple 1 s ou 10 ms). 


Si ce paramètre n'est pas précisé, cette valeur vaut 10000 (10 secondes).

<a name="NOTE0_1"></a>

## Remarques




### Adresse MAC et adresse IP
<a name="adresse_mac_adresse_ELTPARAGRAPHE000046"></a>L'adresse MAC est l'adresse physique spécifiée sur les cartes réseau des postes MAC. L'adresse IP est l'adresse permettant de communiquer entre plusieurs postes via le protocole IP (Internet Protocol).

**Remarque** : Cette fonction ne peut être utilisée que pour des machines reliées en réseau local.


## NetIPVersMAC (Fonction)

***En anglais : NetIPToMAC***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie l'adresse MAC correspondant à une adresse IP donnée.  
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Renvoie l'adresse MAC 
Info(NetIPVersMAC("11.22.33.44"))
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = NetIPVersMAC(<Adresse IP>)`
---

**`<Résultat> : (Chaîne de caractères)`**



- Adresse MAC obtenue (au format "aa:bb:cc:dd:ee:ff"), 

- Chaîne vide ("") en cas de problème.




**`<Adresse IP> : (Chaîne de caractères)`**

Adresse IP à convertir au format "aa.bb.cc.dd". Cette adresse IP doit être associée à une carte réseau physique pour être convertie en une adresse MAC.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Adresse MAC et adresse IP
<a name="adresse_mac_adresse_ELTPARAGRAPHE000033"></a>L'adresse MAC est l'adresse physique spécifiée sur les cartes réseau des postes MAC. L'adresse IP est l'adresse permettant de communiquer entre plusieurs postes via le protocole IP (Internet Protocol).

**Remarque** : 

- Cette fonction ne peut être utilisée que pour des machines reliées en réseau local.

- Les adresses IP au format IPv6 ne sont pas supportées.


## RendezVousSupprime (Fonction)

***En anglais : AppointmentDelete***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) le rendez-vous en cours de l'agenda d'une messagerie Lotus Notes ou Outlook.
	**Rappel** : Cette fonction ne permet pas de supprimer le rendez-vous en cours de l'agenda d'une messagerie Outlook Express.







<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 
```wl
// Ouverture d'une session Lotus Notes
IDSession est un entier
IDSession = EmailOuvreSessionNotes("MotDePasse", "ServeurMars", ...
	"mail/julie.nsf", "C:\Lotus Notes\julie.id")
// Lecture du dernier rendez-vous
RendezVousDernier(IDSession)
// Suppression du rendez-vous
RendezVousSupprime(IDSession)
```


<a name="Exemple2"></a>







<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Supprimer le rendez-vous en cours de l'agenda d'une messagerie Lotus Notes ou Outlook 

`<Résultat> = RendezVousSupprime(<Identifiant de la session> [, <Identifiant du rendez-vous>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la suppression a été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour connaître le libellé de l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*. 




**`<Identifiant de la session> : (Entier)`**

Identifiant de la session à utiliser. Cet identifiant est renvoyé par une des fonctions suivantes : [EmailOuvreSessionNotes](../WDLang3/3032121.md), [EmailOuvreSessionOutlook](../WDLang3/3032128.md), [NotesOuvreBase](../WDLang3/1000005101.md) ou [OutlookOuvreSession](../WDLang3/3032123.md).  

**`<Identifiant du rendez-vous> : (Chaîne de caractères optionnelle)`**

Identifiant du rendez-vous à supprimer. Cet identifiant est contenu dans la variable mRendezVous.ID.




## Fonctions de gestion des rendez-vous (Outlook, Lotus Notes, appareil mobile)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions de gestion des rendez-vous permettent : 

- de gérer les contacts présents dans l'agenda d'une messagerie Lotus Notes ou Outlook. 









|   |   |
| --- | --- |
| [iCalendarVersRendezVous](../WDLang3/1000021827.md) | Liste les rendez-vous décrits dans une chaîne de caractères au format iCalendar. |
| [RendezVousAffiche](../WDLang3/1000020692.md) | Affiche un rendez-vous dans l'application native de gestion des rendez-vous de l'appareil mobile (Android ou iOS). |
| [RendezVousAjoute](../WDLang3/1000011001.md) | Ajoute un rendez-vous : <br><br>- dans l'agenda d'une messagerie Lotus Notes ou Outlook. <br><br>- dans un calendrier (ou agenda) d'un appareil mobile (Android ou iOS). <br><br><br> |
| [RendezVousCrée](../WDLang3/1000020691.md) | Affiche la fenêtre de création de rendez-vous de l'application native de gestion des rendez-vous de l'appareil Android. |
| [RendezVousDernier](../WDLang3/1000011002.md) | Lit le dernier rendez-vous présent dans l'agenda d'une messagerie Lotus Notes ou Outlook. |
| [RendezVousListe](../WDLang3/1000020690.md) | Liste les rendez-vous présents sur le mobile (Android/iOS) et correspondant aux critères spécifiés. |
| [RendezVousListeCalendrier](../WDLang3/1000020661.md) | Liste les calendriers disponibles sur l'appareil mobile (Android/iOS). |
| [RendezVousLit](../WDLang3/1000011011.md) | Lit un rendez-vous présent dans l'agenda d'une messagerie Lotus Notes ou Outlook. |
| [RendezVousModifie](../WDLang3/1000011003.md) | Modifie le rendez-vous en cours : <br><br>- dans l'agenda d'une messagerie Lotus Notes ou Outlook.<br><br>- dans un calendrier (ou agenda) d'un appareil mobile (Android/iOS). <br><br><br> |
| [RendezVousPrécédent](../WDLang3/1000011004.md) | Lit le rendez-vous précédent le rendez-vous en cours, dans l'agenda d'une messagerie Lotus Notes ou Outlook. |
| [RendezVousPremier](../WDLang3/1000011005.md) | Lit le premier rendez-vous présent dans l'agenda d'une messagerie Lotus Notes ou Outlook. |
| [RendezVousRAZ](../WDLang3/1000011006.md) | Réinitialise : <br><br>- la structure [RendezVous](../WDLang3/1000011010.md).<br><br>- la variable de type [RendezVous](../WDLang1/1000019244.md).<br><br><br> |
| [RendezVousSuivant](../WDLang3/1000011007.md) | Lit le rendez-vous suivant le rendez-vous en cours, dans l'agenda d'une messagerie Lotus Notes ou Outlook. |
| [RendezVousSupprime](../WDLang3/1000011008.md) | Supprime : <br><br>- le rendez-vous en cours de l'agenda d'une messagerie Lotus Notes ou Outlook. <br><br>- un rendez-vous dans un calendrier (ou agenda) d'un appareil mobile (Android/iOS).<br><br><br> |
| [RendezVousVersiCalendar](../WDLang3/1000021828.md) | Exporte un rendez-vous ou une liste de rendez-vous au format iCalendar. |

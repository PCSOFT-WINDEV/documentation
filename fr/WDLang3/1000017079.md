
## SNMPOIDDescription (Fonction)

***En anglais : SNMPOIDDescription***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la description d'un OID.

**Remarque** : Cette fonction ne peut être utilisée qu'après avoir chargé le fichier MIB (fonction [SNMPChargeMIB](../WDLang3/1000017074.md)) qui décrit la correspondance entre les OID au format texte et numérique.  
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MIBChargé est un booléen
CheminMIBs est une chaîne = ComplèteRep(fRepExe()) + "mibs\"
Mib est une chaîne
Mib = "RFC1382-MIB.mib" // MIB du protocole X25
MaDescription est une chaîne
MIBChargé = SNMPChargeMIB(CheminMIBs + Mib)
SI MIBChargé = Faux ALORS 
	Erreur("Erreur MIB : " + ErreurInfo())
SINON
	MaDescription = SNMPOIDDescription("
			dod.internet.mgmt.mib-2.
			transmission.x25.x25AdmnInterruptTimer
			")
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = SNMPOIDDescription(<OID à traiter>)`
---

**`<Résultat> : (Chaîne de caractères)`**

Description de l'OID. Cette description est contenue dans le fichier MIB.

**`<OID à traiter> : (Chaîne de caractères)`**

Chaîne contenant l'OID sous forme de texte ou de numérique.

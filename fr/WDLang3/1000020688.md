
## Rappel (Type de variable)

***En anglais : Reminder***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **Rappel** permet de décrire et manipuler un rappel pour notifier l'utilisateur qu'un rendez-vous est en approche.

Ce type de variable est utilisé par les variables de type [RendezVous](../WDLang1/1000019244.md). 

Les caractéristiques de ce rappel peuvent être définies et modifiées à l'aide de différentes propriétés WLangage.

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ajoute un nouveau rappel par notification une heure avant le début du rendez-vous
rappelRDV est un Rappel
rappelRDV.Type = rappelNotification
rappelRDV.Délai = 60
Ajoute(rdv.Rappel, rappelRDV)
RendezVousModifie(rdv)

// Ajoute un nouveau rappel par email une semaine avant le début du rendez-vous
rappelRDV est un Rappel
rappelRDV.Type = rappelEmail
rappelRDV.Délai = 7 * 24 * 60 
Ajoute(rdv.Rappel, rappelRDV)
RendezVousModifie(rdv)
```

<a name="NOTE0"></a>

## Remarques
<a name="NOTE0_1"></a>


### Propriétés spécifiques à la description des variables de type Rappel
<a name="proprietes_specifiques_description_des_variables_type_rappel_ELTPARAGRAPHE000043"></a>Les propriétés suivantes peuvent être utilisées pour manipuler un rappel :

| **Nom de la propriété** | **Type manipulé** | **Effet** |
| --- | --- | --- |
| Délai | Entier ou Durée | Délai en minutes avant le début du rendez-vous pour lequel le rappel doit être effectué. Cette propriété peut correspondre à : <br><br>- un entier correspondant au nombre de minutes, <br><br>- une variable de type Durée, <br><br>- l'indication directe de la durée (par exemple 1 min). <br><br><br> |
| Type | Constante de type Entier | Méthode à utiliser pour effectuer le rappel : <br><br>- *rappelDéfaut* : méthode par défaut. Cette méthode varie en fonction du paramétrage de l'appareil. <br><br>- *rappelEmail* : envoi d'un email. Ce type de rappel est disponible uniquement si le compte associé au calendrier du rendez-vous supporte ce type de rappel et si ce compte est configuré.<br><br>- *rappelNotification* : envoi d'une notification. <br><br>- *rappelSMS* : envoi d'un SMS. Ce type de rappel est disponible uniquement si le compte associé au calendrier du rendez-vous supporte ce type de rappel et si ce compte est configuré.<br><br><br><br><br> |




<a name="NOTE0_2"></a>


### Note
<a name="note_ELTPARAGRAPHE000085"></a>Pour réinitialiser le contenu d'une variable de type Rappel, il est possible d'utiliser la fonction [VariableRAZ](../WDLang1/3013069.md).

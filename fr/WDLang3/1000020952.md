


## grDestinationImage (Fonction)

***En anglais : grDestinationImage***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Définit une variable de type Image comme destination d'un graphe. 

<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonImage est une Image
MonImage.Largeur = 100
MonImage.Hauteur = 100
grCrée("MonGraphe", grSecteur)

// Définit la destination du dessin du graphe
grDestinationImage("MonGraphe", MonImage)
// Ajoute les données
grAjouteDonnée("MonGraphe", 1, 10)
...
// Dessine le graphe
grDessine("MonGraphe")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`grDestinationImage(<Nom du graphe> , <Nom de la variable Image>)`
---

**`<Nom du graphe> : (Chaîne de caractères)`**

Nom du graphe à manipuler, défini avec la fonction [grCrée](../WDLang3/3042005.md).

**`<Nom de la variable Image> : (Variable de type Image)`**

Nom de la variable de type [Image](../WDLang1/1000019650.md) dans laquelle le graphe sera dessiné. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Conditions d'utilisation et limites
<a name="conditions_utilisation_limites_ELTPARAGRAPHE000046"></a>

- Le graphe sera dessiné dans la variable de type [Image](../WDLang1/1000019650.md) spécifiée. Le graphe sera dessiné sur un fond blanc. Le précédent dessin contenu dans la variable [Image](../WDLang1/1000019650.md) sera effacé. 

- Cette fonction ne peut pas être utilisée avec les champs Graphe.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Cette fonction ne permet pas d'obtenir l'image d'un graphe en 3D. Le graphe sera automatiquement converti en 2D. 



<a name="NOTE0_2"></a>


### Créer et dessiner un graphe par programmation
<a name="creer_dessiner_graphe_par_programmation_ELTPARAGRAPHE000063"></a>Pour dessiner un graphe :

- créez le graphe (fonction [grCrée](../WDLang3/3042005.md)),

- ajoutez les données (fonctions [grAjouteDonnée](../WDLang3/3042023.md), [grBourseAjouteDonnée](../WDLang3/3042009.md) et [grNuageAjouteDonnéeXY](../WDLang3/3042032.md)),

- définissez la destination du graphe à l'aide des fonctions suivantes : 

	- [grDestinationBMP](../WDLang3/3042018.md), 

	- [grDestinationChamp](../WDLang3/3042020.md), 

	- [grDestinationEMF](../WDLang3/3042017.md), 

	- [grDestinationFen](../WDLang3/3042031.md),

	- **grDestinationImage**,

	- [grDestinationWMF](../WDLang3/3042019.md). 




- dessinez le graphe (fonction [grDessine](../WDLang3/3042026.md)).

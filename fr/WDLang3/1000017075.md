
## SNMPOIDVersChaîne (Fonction)

***En anglais : SNMPOIDToString***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Convertit un OID de sa représentation numérique vers sa représentation textuelle.

**Remarque** : Cette fonction ne peut être utilisée qu'après avoir chargé le fichier MIB (fonction [SNMPChargeMIB](../WDLang3/1000017074.md)) qui décrit la correspondance entre les OID au format texte et numérique.  
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
bMIBChargé est un booléen
sCheminMIBs est une chaîne = ComplèteRep(fRepExe()) + "mibs\"
sMib est une chaîne
sMib = "RFC1382-MIB.mib" // MIB du protocole X25
sOID est une chaîne
bMIBChargé = SNMPChargeMIB(sCheminMIBs + sMib)
SI bMIBChargé = Faux ALORS 
	Erreur("Erreur MIB : " + ErreurInfo())
SINON
	sOID = SNMPOIDVersChaîne("1.3.6.1.2.1.25.1.1.0")
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = SNMPOIDVersChaîne(<OID à convertir>)`
---

**`<Résultat> : (Chaîne de caractères)`**



- Chaîne contenant la valeur de l'OID sous forme texte, 

- Chaîne vide ("") en cas d'erreur.




**`<OID à convertir> : (Chaîne de caractères)`**

Chaîne contenant l'OID sous forme numérique à convertir.




## grSurfaceSupprimeLibelléGraduation (Fonction)

***En anglais : grSurfaceDeleteMarkCaption***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime les libellés de graduation d'un graphe de type Surface. 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
grSurfaceSupprimeLibelléGraduation(GRF_MonGraphe, grAbscisse, 1)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`grSurfaceSupprimeLibelléGraduation(<Nom du graphe> [, <Axe de la graduation> [, <Valeur>]])`
---

**`<Nom du graphe> : (Nom de champ ou chaîne de caractères)`**

Nom du graphe à manipuler. Ce nom correspond :

- soit au nom du graphe défini par programmation avec la fonction [grCrée](../WDLang3/3042005.md).

- soit au nom du champ Graphe (présent dans l'éditeur de fenêtres ou d'états).




**`<Axe de la graduation> : (Constante de type Entier)`**

Type de l'axe sur lequel la graduation doit être supprimée :



|   |   |
| --- | --- |
| *grAbscisse* | Axe des abscisses. |
| *grAltitude* | Axe des altitudes. |
| *grOrdonnée* | Axe des ordonnées. |


Si ce paramètre n'est pas précisé, les graduations sont supprimées sur tous les axes. 

**`<Valeur> : (Réel)`**

Valeur de la graduation pour laquelle le libellé doit être supprimé. 

Si ce paramètre n'est pas précisé, tous les libellés des graduations seront supprimés.

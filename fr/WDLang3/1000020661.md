
## RendezVousListeCalendrier (Fonction)

***En anglais : AppointmentListCalendar***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Liste les calendriers disponibles sur l'appareil mobile (Android/iOS).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple
<a class="notetitle" target="_blank" href="$DOC$=1000020712&name=rendezvouslistecalendrier_fonction&product=WD">Voir des exemples supplémentaires</a>

```wl
// Remplit un champ Liste avec les calendriers disponibles sur l'appareil
tabCalendrier est un tableau de Calendrier 
tabCalendrier = RendezVousListeCalendrier()
POUR TOUT Calendrier de tabCalendrier
	ListeAjoute(LISTE_Liste1, Calendrier.Nom)
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Lister les calendriers disponibles

`<Résultat> = RendezVousListeCalendrier()`
---

**`<Résultat> : (Tableau de variables de type Calendrier)`**

Tableau de variables de type [Calendrier](../WDLang3/1000020663.md) initialisé avec la liste des calendriers disponibles. 
En cas d'erreur, le tableau est vide et la variable [ErreurDétectée](../WDLang1/3087001.md) est à <u><u><u><u>Vrai</u></u></u></u>. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).


<a name="SYNTAXE2"></a>

### Lister les calendriers correspondant aux filtres spécifiés

`<Résultat> = RendezVousListeCalendrier(<Filtre> , <Valeur recherchée>)`
---

**`<Résultat> : (Tableau de variables de type Calendrier)`**

Tableau de variables de type [Calendrier](../WDLang3/1000020663.md) initialisé avec la liste des calendriers disponibles. 
En cas d'erreur, le tableau est vide et la variable [ErreurDétectée](../WDLang1/3087001.md) est à <u><u><u><u>Vrai</u></u></u></u>. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).

**`<Filtre> : (Constante de type Entier)`**

Propriété sur laquelle la liste des calendriers doit être filtrée : 


|   |   |
| --- | --- |
| *rdvIdentifiant* | Filtre sur les identifiants des calendriers. Une recherche à l'identique est effectuée. |
| *rdvNom* | Filtre sur le nom du calendrier. Une recherche de type "Contient" est effectuée. |
| *rdvNomCompte* | Filtre sur le nom du compte auquel appartient le calendrier. Une recherche de type "Contient" est effectuée. |
| *rdvTypeCompte* | Filtre sur le type du compte auquel appartient le calendrier. Une recherche de type "Contient" est effectuée. |





**`<Valeur recherchée> : (Chaîne de caractères)`**

Valeur recherchée dans les propriétés filtrées du calendrier. Cette recherche est insensible à la casse.

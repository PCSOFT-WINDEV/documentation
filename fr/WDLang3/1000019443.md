
## NotifModifie (Fonction)

***En anglais : NotifModify***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie une notification actuellement affichée dans la barre système de l'appareil.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
notif est une Notification
notif.Titre = "Nouveau message reçu"
notif.Message = "Cliquez pour afficher le message"
NotifAjoute(notif)
...
notif.Titre = "2 messages reçus"
notif.Message = "Cliquez pour afficher les messages"
NotifModifie(notif)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = NotifModifie(<Notification>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la notification a été modifiée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**Remarque** : Dans le simulateur, cette fonction est sans effet et renvoie toujours <u><u><u><u>Faux</u></u></u></u>. 

**`<Notification> : (Variable de type Notification)`**

Nom de la variable de type [Notification](../WDLang3/1000019441.md) contenant la description de la notification à mettre à jour.


## ContactCrée (Fonction)

***En anglais : ContactCreate***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Affiche la fenêtre de création de contact de l'application native de gestion des contacts de l'appareil (Android, iPhone ou iPad, Universal Windows).

## Syntaxe
<a name="SYNTAXE1"></a>

### Créer un contact et initialiser la variable Contact

`<Résultat> = ContactCrée([<Contact>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si un contact a été créé, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) 

- <u><u><u><u>Vrai</u></u></u></u> si l'interface utilisateur est affichée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 




**`<Contact> : (Variable de type Contact)`**

Nom de la variable de type [Contact](../WDLang3/1000020254.md) contenant les données correspondant au contact créé. 





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Cas particuliers
<a name="cas_particuliers_ELTPARAGRAPHE000086"></a>

- Si l'utilisateur crée un nouveau contact, la [structure Contact](../WDLang3/1000006111.md) ou la variable [Contact](../WDLang3/1000020254.md) est automatiquement renseignée avec les informations relatives à ce nouveau contact.

- ![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) La fonction **ContactCrée** est non bloquante. 

- La fonction **ContactCrée** peut être utilisée dans l'émulateur. Elle ne peut pas être utilisée dans le simulateur.



<a name="NOTE0_2"></a>


### Création d'un contact (sans utiliser la fenêtre de création)
<a name="creation_contact_sans_utiliser_fenetre_creation_ELTPARAGRAPHE000117"></a>Pour créer un nouveau contact entièrement par programmation (sans ouvrir de fenêtre) : 

1. Décrivez le contact à l'aide de la variable [Contact](../WDLang3/1000020254.md). 

2. Utilisez la fonction [ContactAjoute](../WDLang3/1000006101.md).

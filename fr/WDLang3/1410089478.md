
## Procédure WLangage appelée par la fonction DeepLinkProcédure



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Procédure WLangage (également nommée "callback") appelée par la fonction [DeepLinkProcédure](../WDLang3/1410089051.md) en cas de clic sur un "deep link" dont l'URL cible a été associée à l'application. Cette procédure doit être une procédure globale. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
DeepLinkProcédure(ProcDeepLink)
```
<a name="1410089051_Exemple2"></a>



```wl
// ---------------------------------
// Procédure globale
// ---------------------------------
PROCEDURE ProcDeepLink(url est une URI)
// Extrait l'identifiant du produit contenu dans l'url
sId est une chaine = ExtraitChaine(url.Requête, 2, " = ")
SI sId <> EOT ALORS
	SI FenEtat(FEN_FicheProduit) <> Inexistant ALORS
		Ferme(FEN_FicheProduit)			
	FIN
	OuvreFenêtreMobile(FEN_FicheProduit, sId)
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`DeepLinkProcédure_Callback(<URL>)`
---

**`<URL> : (Variable de type URI)`**

Variable de type [URI](../WDLang3/1000024027.md) correspondant à la cible du lien "deep link" cliqué.

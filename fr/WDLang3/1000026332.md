


## &lt;Variable zipArchive&gt;.Existe (Fonction)

***En anglais : &lt;zipArchive variable&gt;.Exist***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de savoir si une archive existe. Une archive existe si elle a été ouverte par la fonction [&lt;Variable zipArchive&gt;.Ouvre](../WDLang3/1000019097.md) ou créée par la fonction [&lt;Variable zipArchive&gt;.Crée](../WDLang3/1000019160.md).








<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) 
```wl
MonArchive est un zipArchive
// Création de l'archive
ResCréation = MonArchive.Crée("C:\Temp\Archive.zip")
...
// Test de l'existence de l'archive 
SI MonArchive.Existe() = Vrai ALORS
	// Ajout d'un fichier dans une archive
	ResAjoutFichier = MonArchive.AjouteFichier(...
			"C:\MonRépertoire\MesFichiers\Fichier.pdf", zipDisque)
	// Affichage du message d'erreur si le fichier n'a pas été ajouté
	SI ResAjoutFichier <> 0 ALORS 
		Erreur(zipMsgErreur(ResAjoutFichier))
	FIN
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Archive>.Existe()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'archive existe, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Archive> : (Variable de type zipArchive)`**

Nom d'une variable de type [zipArchive](../WDLang3/1000018679.md) correspondant à l'archive à manipuler.




## &lt;Graphe&gt;.EpaisseurTrait (Fonction)

***En anglais : &lt;Chart&gt;.LineThickness***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Récupère et / ou change la valeur de l'épaisseur du trait dans un graphe de type Courbe.

![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Les graphes](../WDChamp/3042060.md). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Champ Graphe
// Modifie l'épaisseur du trait pour toutes les séries du graphe
GRF_MonGraphe.EpaisseurTrait(grToutesSéries, 4)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Récupérer l'épaisseur du trait (graphe de type Courbe)

`<Résultat> = <Champ Graphe>.EpaisseurTrait(<Numéro de série>)`
---

**`<Résultat> : (Entier)`**

Epaisseur du trait.

**`<Champ Graphe> : (Nom de champ)`**

Nom du champ Graphe à manipuler (présent dans l'éditeur de fenêtres ou d'états). Ce graphe doit être de type Courbe.


**`<Numéro de série> : (Entier)`**

Numéro de la série pour laquelle on veut connaître l'épaisseur du trait.


<a name="SYNTAXE2"></a>

### Modifier l'épaisseur du trait (graphe de type Courbe)

`<Champ Graphe>.EpaisseurTrait(<Numéro de série> , <Nouvelle épaisseur>)`
---

**`<Champ Graphe> : (Nom de champ)`**

Nom du champ Graphe à manipuler (présent dans l'éditeur de fenêtres ou d'états). Ce graphe doit être de type Courbe.


**`<Numéro de série> : (Entier)`**



- Numéro de la série pour laquelle on veut modifier l'épaisseur du trait. 

- Constante *grToutesSéries* pour changer l'épaisseur de toutes les séries (y compris les nouvelles séries).




**`<Nouvelle épaisseur> : (Entier)`**

Nouvelle épaisseur du trait. Ce paramètre peut correspondre à une valeur supérieure à 0. Par défaut, l'épaisseur du trait est 1.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Unités prises en compte pour l'épaisseur
<a name="unites_prises_compte_pour_epaisseur_ELTPARAGRAPHE000306"></a>L'épaisseur d'un trait correspond à des valeurs supérieures ou égales à 0.

- Si l'épaisseur correspond à 0, l'épaisseur sera de 1 pixel à l'écran (soit 1 pica à l'impression).

- Si l'épaisseur est supérieure ou égale à 1, l'épaisseur sera de x pixels à l'écran, soit x pica à l'impression.



<a name="NOTE0_2"></a>


### Divers
<a name="divers_ELTPARAGRAPHE000319"></a>

- Si la fonction **&lt;Graphe&gt;.EpaisseurTrait** est utilisée sur un type de graphe autre que courbe, aucune erreur n'est renvoyée. L'information est stockée et sera utilisée lorsque le type du graphe sera modifié (avec la fonction [&lt;Graphe&gt;.Type](../WDLang3/1000024168.md) ou avec le menu contextuel du graphe).

- Équivalence : la fonction **&lt;Graphe&gt;.EpaisseurTrait** est équivalente à la propriété [Epaisseur](../Proprietes/2510144.md).

- Les données définies avec la fonction **&lt;Graphe&gt;.EpaisseurTrait** seront prises en compte lors du prochain appel à la fonction [&lt;Graphe&gt;.Dessine](../WDLang3/1000024062.md).

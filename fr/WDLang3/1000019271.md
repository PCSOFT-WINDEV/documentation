
## Constantes des fonctions de géolocalisation
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>




Les constantes utilisées par les fonctions de géolocalisation sont les suivantes : 

| Constante | Description | Utilisée par les fonctions |
| --- | --- | --- |
| **géoCarte2D** | La position est affichée sur une carte en deux dimensions dans l'application de cartographie native de l'appareil. | [géoLanceAppli](../WDLang3/1000019954.md) |
| **géoKilomètre** | Renvoie la distance en kilomètres. | [géoDistance](../WDLang3/1000019211.md) |
| **géoMètre** | Renvoie la distance en mètres. | [géoDistance](../WDLang3/1000019211.md) |
| **géoMile** | Renvoie la distance en miles. | [géoDistance](../WDLang3/1000019211.md) |
| **géoSuiviActivé** | Etat du système de suivi : Le système de suivi de position est activé. | [géoSuiviEtat](../WDLang3/1000021592.md) |
| **géoSuiviDésactivé** | Etat du système de suivi : Le système de suivi de position est désactivé. | [géoSuiviEtat](../WDLang3/1000021592.md) |
| **géoSuiviErreur** | Etat du système de suivi : Erreur pendant la récupération de l'état d'activation du système. Utilisez la fonction [ErreurInfo](../WDLang1/3013008.md) pour obtenir le détail de l'erreur. | [géoSuiviEtat](../WDLang3/1000021592.md) |
| **géoVue360** | La position est affichée sous forme d'image avec une vue à 360° dans l'application de cartographie native de l'appareil. | [géoLanceAppli](../WDLang3/1000019954.md) |
| **gpsActivé** | Le fournisseur de localisation est activé. | [GPSEtat](../WDLang3/1000019207.md) |
| **gpsAltitude** | Le fournisseur de localisation fournit des informations d'altitude. | [GPSInitParamètre](../WDLang3/1000019204.md), [GPSInfo](../WDLang3/1000019208.md) |
| **gpsAuto** | Détermination automatique du fournisseur de localisation. | [GPSInitParamètre](../WDLang3/1000019204.md) |
| **gpsDésactivé** | Le fournisseur de localisation est désactivé. | [GPSEtat](../WDLang3/1000019207.md) |
| **gpsDirection** | Le fournisseur de localisation fournit des informations de direction. | [GPSInitParamètre](../WDLang3/1000019204.md), [GPSInfo](../WDLang3/1000019208.md) |
| **gpsDisponible** | Le fournisseur de localisation est devenu disponible. | [GPSEtat](../WDLang3/1000019207.md) |
| **gpsEnergie** | Demande des informations sur la consommation énergétique du fournisseur de localisation. | [GPSInfo](../WDLang3/1000019208.md) |
| **gpsEnergieElevée** | Le fournisseur de localisation peut requérir une quantité d'énergie élevée de la part de l'appareil. | [GPSInitParamètre](../WDLang3/1000019204.md), [GPSInfo](../WDLang3/1000019208.md) |
| **gpsEnergieFaible** | Le fournisseur de localisation doit requérir une quantité d'énergie faible de la part de l'appareil. | [GPSInitParamètre](../WDLang3/1000019204.md), [GPSInfo](../WDLang3/1000019208.md) |
| **gpsEnergieMoyenne** | Le fournisseur de localisation peut requérir une quantité d'énergie moyenne de la part de l'appareil. | [GPSInitParamètre](../WDLang3/1000019204.md), [GPSInfo](../WDLang3/1000019208.md) |
| **gpsErreur** | Une erreur s'est produite lors de la récupérationde l'état du fournisseur de localisation. | [GPSEtat](../WDLang3/1000019207.md) |
| **gpsErreurDroit** | Le navigateur n'a pas les droits pour utiliser la géolocalisation. | [GPSRécupèrePosition](../WDLang3/1000019205.md), [GPSSuitDéplacement](../WDLang3/1000019206.md) |
| **gpsErreurOK** | Aucune erreur n'a été détectée. | [GPSRécupèrePosition](../WDLang3/1000019205.md), [GPSSuitDéplacement](../WDLang3/1000019206.md) |
| **gpsErreurPosition** | Impossible de déterminer la position (erreur interne du fournisseur d'accès par exemple). | [GPSRécupèrePosition](../WDLang3/1000019205.md), [GPSSuitDéplacement](../WDLang3/1000019206.md) |
| **gpsErreurTimeout** | La position n'a pas pu être récupérée dans le temps imparti. | [GPSRécupèrePosition](../WDLang3/1000019205.md), [GPSSuitDéplacement](../WDLang3/1000019206.md) |
| **gpsHorsService** | Le fournisseur de localisation est hors service. | [GPSEtat](../WDLang3/1000019207.md) |
| **gpsIndisponible** | Le fournisseur de localisation est devenu temporairement indisponible. | [GPSEtat](../WDLang3/1000019207.md) |
| **gpsPrécision** | Demande des informations sur la précision de mesure du fournisseur de localisation. | [GPSInfo](../WDLang3/1000019208.md) |
| **gpsPrécisionElevée** | Calcul de position avec une précision élevée. | [GPSInitParamètre](../WDLang3/1000019204.md), [GPSInfo](../WDLang3/1000019208.md) |
| **gpsPrécisionFaible** | Calcul de position avec une précision faible. | [GPSInitParamètre](../WDLang3/1000019204.md), [GPSInfo](../WDLang3/1000019208.md) |
| **gpsRéseau** | Calcul de la position en utilisant le réseau d'antenne mobile ou Wi-Fi. | [GPSInitParamètre](../WDLang3/1000019204.md), [GPSInfo](../WDLang3/1000019208.md) |
| **gpsSatellite** | Calcul de la position en utilisant le système GPS. | [GPSInitParamètre](../WDLang3/1000019204.md), [GPSInfo](../WDLang3/1000019208.md) |
| **gpsType** | Demande des informations sur le type de fournisseur de localisation. | [GPSInfo](../WDLang3/1000019208.md) |
| **gpsVitesse** | Le fournisseur de localisation fournit des informations de vitesse. | [GPSInitParamètre](../WDLang3/1000019204.md), [GPSInfo](../WDLang3/1000019208.md) |

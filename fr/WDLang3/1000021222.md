
## httpAvancement (Type de variable)

***En anglais : httpProgress***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **httpAvancement** permet de gérer l'avancement de l'envoi ou de la réception d'une requête HTTP. Les caractéristiques de la gestion de l'avancement peuvent être définies et modifiées à l'aide de différentes propriétés WLangage. 

Ce type de variable est utilisé par : 

- le type [httpRequête](../WDLang3/1000021158.md). 

- le type [restRequête](../WDLang3/1000021481.md). 




**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
cMaRequête est un httpRequête
cMaRequête.URL = "http://www.pcsoft.fr"
cMaRequête.AvancementTéléchargement.Procédure = MaCallBack
cMaRéponse est un httpRéponse = HTTPEnvoie(cMaRequête)
SI ErreurDétectée ALORS
	Erreur(ErreurInfo(errComplet))
SINON
	Info(cMaRéponse.Contenu)
FIN
```





<a name="NOTE0"></a>

## Remarques
<a name="NOTE0_1"></a>


### Propriétés spécifiques à la description des variables de type httpAvancement
<a name="proprietes_specifiques_description_des_variables_type_httpavancement_ELTPARAGRAPHE000054"></a>Les propriétés suivantes peuvent être utilisées pour manipuler l'avancement de l'envoi ou de la réception d'une requête HTTP :

| Nom de la propriété | Type manipulé | Effet |
| --- | --- | --- |
| Fréquence | Entier | Fréquence d'appel de la jauge ou de la procédure, définie en nombre d'octets. <br><br>Par défaut, la fréquence est de 1000 octets.<br><br>**Remarque** : La fréquence indiquée peut ne pas être respectée. <br><br> |
| Jauge | Chaîne de caractères | Nom du champ Jauge utilisé pour suivre l'avancement. Ce champ sera notifié de l'avancement à la fréquence indiquée. <br><br>**Attention** : Si les propriétés **Procédure** et **Jauge** sont renseignées, la propriété **Procédure** est prioritaire pour l'appel.<br><br> |
| Procédure | Chaîne de caractères | Nom de la procédure utilisée pour suivre l'avancement. Cette procédure sera notifiée de l'avancement à la fréquence indiquée. <br><br>Cette procédure est de la forme : <br><br><pre><code>PROCEDURE <Nom de la procédure>(<Total>, <Progression>)</code></pre><br><br><br>où : <br><br>- &lt;Total&gt; est le nombre total d'octets à transférer. <br><br>- &lt;Progression&gt; est le nombre d'octets déjà transférés. <br><br><br><br><br>**Attention** : Si les propriétés **Procédure** et **Jauge** sont renseignées, la propriété **Procédure** est prioritaire pour l'appel. |

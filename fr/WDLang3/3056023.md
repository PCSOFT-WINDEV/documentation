


## NetAdresseIP (Fonction)

***En anglais : NetIPAddress***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie l'adresse IP (Internet Protocol) d'un poste. 
**Attention** : Seules les adresses IPV4 sont renvoyées. Ce fonctionnement est conservé pour compatibilité avec les versions précédentes. Pour connaître les adresses IPV4 et/ou IPV6, utilisez la fonction [NetListeAdresseIP](../WDLang3/1000019683.md). 




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affiche l'adresse IP de la machine locale
Info(NetAdresseIP())
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = NetAdresseIP([<Nom du poste>] [, <Indice>])`
---

**`<Résultat> : (Chaîne de caractères)`**



- Adresse IP du poste spécifié (au format XXX.XXX.XXX.XXX), 

- Chaîne vide ("") si le poste possède moins d'adresses IP que l'&lt;Indice&gt; demandé.




**`<Nom du poste> : (Chaîne de caractères optionnelle)`**

Nom du poste à manipuler. Si ce paramètre n'est pas spécifié, le poste considéré sera le poste en cours.


**`<Indice> : (Entier optionnel)`**

Indice de l'adresse IP dans la liste des adresses du poste. Par défaut, l'adresse renvoyée est la première adresse de la liste des adresses du poste. La première adresse a pour indice 0.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Nombre d'adresses IP
<a name="nombre_adresses_ELTPARAGRAPHE000062"></a>

- Il existe une adresse IP pour chaque protocole TCP/IP installé sur un même poste.

- Pour lister les différentes adresses IP d'un poste (IPV4 et/ou IPV6), utilisez la fonction [NetListeAdresseIP](../WDLang3/1000019683.md).

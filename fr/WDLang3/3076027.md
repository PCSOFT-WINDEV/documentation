
## telToucheEnfoncée (Fonction)

***En anglais : tapiKeyPressed***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de connaître l'historique des touches enfoncées sur le clavier téléphonique depuis la dernière utilisation de la fonction **telToucheEnfoncée**. L'historique des touches enfoncées est automatiquement ré-initialisé.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
ListeTouches est une chaîne 
ListeTouches = telToucheEnfoncée()
SI Position(ListeTouches, "*") <> 0 ALORS
	Trace("La touche * a été utilisée")
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = telToucheEnfoncée([<Identifiant de l'appel>])`
---

**`<Résultat> : (Chaîne de caractères)`**

Liste des touches enfoncées depuis le dernier appel à la fonction **telToucheEnfoncée**.

**Attention** : Cette liste correspond à l'historique des touches utilisées des deux cotés de la ligne téléphonique (coté entrant et sortant).

**`<Identifiant de l'appel> : (Entier optionnel)`**

Identifiant de l'appel à manipuler. Cet identifiant est connu dans la procédure WLangage appelée par la fonction [telDémarreDétectionAppel](../WDLang3/3076029.md). Si ce paramètre n'est pas spécifié, l'appel en cours est manipulé.

## Remarques




### Comment savoir si une touche a été enfoncée ? 
<a name="comment_savoir_une_touche_ete_enfoncee_ELTPARAGRAPHE000046"></a>**Pour savoir si une touche été enfoncée**, utilisez la constante *telToucheDétectée* dans la procédure WLangage appelée par les fonctions [telCompose](../WDLang3/3076013.md) ou [telDémarreDétectionAppel](../WDLang3/3076029.md).

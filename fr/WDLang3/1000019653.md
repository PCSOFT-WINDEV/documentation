


## Fonctions de gestion des Webservices
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions de gestion des Webservices sont les suivantes : 



|   |   |
| --- | --- |
| [AnnuleContexteWebservice](../WDLang3/1000019591.md) | Supprime du contexte du webservice une variable ajoutée par la fonction [DéclareContexteWebservice](../WDLang3/1000019460.md). |
| [DéclareContexteWebservice](../WDLang3/1000019460.md) | Permet de déclarer une liste de variables dont la valeur sera persistante entre les appels successifs à un Webservice. |
| [WebserviceAdresseIPClient](../WDLang3/1000019641.md) | Renvoie l'adresse IP du poste utilisant et appelant le Webservice. |
| [WebserviceEcritCodeHTTP](../WDLang3/1000022785.md) | Renseigne le code HTTP qui sera retourné à la fin de l'exécution de la fonction du Webservice REST. |
| [WebserviceEcritCodeHTTPPersonnalisé](../WDLang3/1410087803.md) | Renseigne le code HTTP personnalisé qui sera retourné à la fin de l'exécution de la fonction du Webservice REST. |
| [WebserviceEcritEntêteHTTP](../WDLang3/1000022788.md) | Ajoute un entête HTTP supplémentaire à une requête à un Webservice REST. |
| [WebserviceEcritTypeMIME](../WDLang3/1000022681.md) | Indique le type MIME de la réponse renvoyée par le Webservice (cas d'un appel à un Webservice REST). |
| [WebserviceLitEntêteHTTP](../WDLang3/1000022787.md) | Lit l'entête HTTP reçu lors d'un appel à un webservice (SOAP ou REST). |
| [WebserviceParamètre](../WDLang3/1000022786.md) | Permet de récupérer la valeur d'un paramètre lors de l'appel d'une requête d'un Webservice (SOAP ou REST). |




## &lt;Graphe&gt;.TypePointSérie (Fonction)

***En anglais : &lt;Chart&gt;.SeriesPointType***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie la forme du point d'une série dans un graphe de type Courbe, Aire, Nuage de points et Radar.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Transforme tous les points de la série 1 en cercle
GRF_MonGraphe.TypePointSérie(1, grPointCercle)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Champ Graphe>.TypePointSérie(<Numéro de la série> , <Type>)`
---

**`<Champ Graphe> : (Nom de champ)`**

Nom du champ Graphe à manipuler (présent dans l'éditeur de fenêtres ou d'états).


**`<Numéro de la série> : (Entier)`**

Numéro de la série dans laquelle le type de point doit être modifié.

**`<Type> : (Constante de type Entier)`**

Nouveau type de point de la catégorie : 


|   |   |
| --- | --- |
| *grNonDessiné* | Les points ne sont pas dessinés. |
| *grPointCarré* | Point en forme de carré. |
| *grPointCercle* | Point en forme de cercle. |
| *grPointCroix* | Point en forme de croix. |
| *grPointLosange* | Point en forme de losange. |
| *grPointTriangle* | Point en forme de triangle. |

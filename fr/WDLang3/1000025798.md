


## &lt;Carte&gt;.RécupèreStyle (Fonction)

***En anglais : &lt;Map&gt;.GetStyle***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le style utilisé par la carte affichée dans le champ Carte.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Applique une style "rétro" à la carte
CARTE_MaCarte.ChangeStyle(ccsRétro)
...
soit StyleUtilisé = CARTE_MaCarte.RécupèreStyle()
SI StyleUtilisé = ccsRétro ALORS Trace("Style rétro")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Champ Carte>.RécupèreStyle()`
---

**`<Résultat> : (Constante de type Entier)`**

Style utilisé par la carte affichée dans le champ Carte : 


|   |   |
| --- | --- |
| *ccsArgenté* | Style "Argenté". |
| *ccsAubergine* | Style "Aubergine". |
| *ccsNuit* | Style "Nuit". |
| *ccsPersonnalisé* | Style "Personnalisé". |
| *ccsRétro* | Style "Retro". |
| *ccsSombre* | Style "Sombre". |
| *ccsStandard* | Style "Standard". |



**`<Champ Carte> : (Nom de champ)`**

Nom du champ Carte à manipuler.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Pour modifier le style de la carte affichée par le champ Carte, utilisez la fonction [&lt;Carte&gt;.ChangeStyle](../WDLang3/1000025797.md).

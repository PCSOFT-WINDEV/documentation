


## &lt;Graphe&gt;.Zoom (Fonction)

***En anglais : &lt;Chart&gt;.Zoom***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Règle le zoom d'un graphe. 

**Remarque** : Cette fonction est similaire à la fonction [&lt;Graphe&gt;.Origine](../WDLang3/1000024114.md). Cependant, la fonction **&lt;Graphe&gt;.Zoom** permet à l'utilisateur de revenir au zoom initial du graphe en utilisant une des méthodes suivantes : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) grâce au menu contextuel du graphe (option "Zoom initial"). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) grâce à la combinaison de touches Ctrl + 0. 







<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
GRF_MonGraphe.Zoom(5, 12, grAbscisse)
GRF_MonGraphe.Dessine()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Champ Graphe>.Zoom(<Graduation minimale> [, <Graduation maximale> [, <Axe>]])`
---

**`<Champ Graphe> : (Nom de champ)`**

Nom du champ Graphe à manipuler (présent dans l'éditeur de fenêtres).

**`<Graduation minimale> : (Réel)`**

Graduation de départ sur l'axe spécifié.
Ce paramètre peut également correspondre à une durée, une date, une heure ou une date-heure. 

**`<Graduation maximale> : (Réel optionnel)`**

Graduation de fin sur l'axe spécifié. 

Si ce paramètre n'est pas spécifié, &lt;Graduation maximale&gt; correspond à la valeur maximale des données du graphe.
Ce paramètre peut également correspondre à une durée, une date, une heure ou une date-heure. 

**`<Axe> : (Constante optionnelle de type Entier)`**

Axe pour lequel les graduations doivent être modifiées : 



|   |   |
| --- | --- |
| *grAbscisse*<br>(valeur par défaut) | Axe des abscisses (axe horizontal). |
| *grOrdonnée* | Axe des ordonnées (axe vertical). |
| *grOrdonnéeSecondaire* | Axe secondaire des ordonnées (axe vertical). |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Graduations des axes
<a name="graduations_des_axes_ELTPARAGRAPHE000267"></a>

- Axe des abscisses (axe horizontal) : les paramètres &lt;Graduation minimale&gt; et &lt;Graduation maximale&gt; correspondent à l'indice minimal et maximal des données dans les séries affichées.

- Axe des ordonnées (axe vertical) : les paramètres &lt;Graduation minimale&gt; et &lt;Graduation maximale&gt; correspondent à la valeur minimale et maximale des données.

- Limites : 100 graduations au-dessus et 100 graduations en dessous de l'axe des abscisses.

- Pour modifier la fréquence des graduations sur l'axe vertical ou horizontal d'un graphe, utilisez la fonction [&lt;Graphe&gt;.Graduation](../WDLang3/1000024076.md).

- Les données définies avec la fonction **&lt;Graphe&gt;.Zoom** seront prises en compte lors du prochain appel à la fonction [&lt;Graphe&gt;.Dessine](../WDLang3/1000024062.md).



<a name="NOTE0_2"></a>


### Types de graphes
<a name="types_graphes_ELTPARAGRAPHE000296"></a>La graduation des axes est sans effet sur les graphes de type Secteur.
<a name="NOTE0_3"></a>


### Restauration des valeurs par défaut
<a name="restauration_des_valeurs_par_defaut_ELTPARAGRAPHE000306"></a>Pour restaurer l'affichage du graphe avant le zoom, il est nécessaire d'utiliser les valeurs minimales et maximales des axes. Ces valeurs peuvent être connues grâce aux propriétés suivantes : 



|   |   |
| --- | --- |
| [AxeXMax](../Proprietes/1000018169.md) | La propriété **AxeXMax** permet de connaître la borne maximale des abscisses (axe X) d'un champ de type Graphe. Cette propriété peut être utilisée par exemple dans l'événement "Zoom" d'un champ Graphe pour identifier la partie affichée. |
| [AxeXMin](../Proprietes/1000018168.md) | La propriété **AxeXMin** permet de connaître la borne minimale des abscisses (axe X) d'un champ Graphe. Cette propriété peut être utilisée par exemple dans l'événement "Zoom" d'un champ Graphe pour identifier la partie affichée. |
| [AxeYMax](../Proprietes/1000018171.md) | La propriété **AxeYMax** permet de connaître la borne maximale des ordonnées (axe Y) d'un champ Graphe. Cette propriété peut être utilisée par exemple dans l'événement "Zoom" d'un champ Graphe pour identifier la partie affichée. |
| [AxeYMin](../Proprietes/1000018170.md) | La propriété **AxeYMin** permet de connaître la borne minimale des ordonnées (axe Y) d'un champ Graphe. |
| [AxeZMax](../Proprietes/1000020626.md) | La propriété **AxeZMax** permet de connaître la borne maximale des altitudes (axe Z) d'un champ Graphe (graphe Surface). |
| [AxeZMin](../Proprietes/1000020662.md) | La propriété **AxeZMin** permet de connaître la borne minimale des altitudes (axe Z) d'un champ Graphe (graphe Surface). |



Par exemple : 

```wl
// Code permettant de zoomer
GRF_MonGraphe.Zoom(5, 12, grAbscisse)
GRF_MonGraphe.Dessine()
```

```wl
// Code permettant de dé-zoomer
GRF_MonGraphe.Zoom(GRF_MonGraphe.AxeXMin, GRF_MonGraphe.AxeXMax, grAbscisse)
GRF_MonGraphe.Dessine()
```

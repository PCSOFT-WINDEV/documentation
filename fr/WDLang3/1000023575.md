


## Fonctions de gestion des protocoles OAuth et OpenID
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions de gestion des protocoles OAuth et OpenID sont les suivantes :  



|   |   |
| --- | --- |
| [AuthIdentifie](../WDLang3/1000022219.md) | Effectue une authentification utilisant le protocole OAuth 2.0 sur un webservice quelconque. |
| [AuthRenouvelleToken](../WDLang3/1000026168.md) | Renouvelle un token OAuth2 d'un utilisateur si ce token est expiré. |
| [OpenIDLitIdentité](../WDLang3/1000023573.md) | Récupère l'identité d'un utilisateur contenue dans une variable de type [AuthToken](../WDLang3/1000022220.md). |



  
Les types de variables spécifiques aux protocoles OAuth et OpenID sont les suivants : 



|   |   |
| --- | --- |
| [AuthToken](../WDLang3/1000022220.md) | Le type **AuthToken** contient les caractéristiques d'un token d'accès à un service Web. |
| [OAuth2Paramètres](../WDLang3/1000022212.md) | Le type **OAuth2Paramètres** permet de définir toutes les informations nécessaires pour s'authentifier sur un service Web implémentant le standard OAuth 2.0. |
| [OpenIDIdentité](../WDLang3/1000023567.md) | Le type **OpenIDIdentité** permet de définir toutes les caractéristiques avancées d'un utilisateur connecté à un service via le protocole OAuth 2.0. |
| [OpenIDParamètres](../WDLang3/1000023561.md) | Le type **OpenIDParamètres** permet de définir toutes les caractéristiques avancées d'une authentification pour OpenID. |

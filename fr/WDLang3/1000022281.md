


## &lt;Variable emailSessionOutlook&gt;.DossierCourant (Fonction)

***En anglais : &lt;emailOutlookSession variable&gt;.CurrentFolder***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le nom du dossier en cours dans le logiciel de messagerie Outlook. Ce dossier est présent dans la branche "Dossiers personnels".

**Rappel** : Cette fonction ne permet pas de connaître le nom du dossier en cours dans le logiciel de messagerie Outlook Express.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple



<a name="3032129_Exemple2"></a>

```wl
// Ouverture de la session Outlook
IDSession est un EmailSessionOutlook
...
// Création d'un dossier dans le dossier en cours 
IDSession.AjouteDossier(IDSession.DossierCourant() + "\Travail")
// Création d'un dossier dans le dossier personnel
IDSession.AjouteDossier("Dossiers personnels\Boîte de réception\TravailPerso")
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Session>.DossierCourant()`
---

**`<Résultat> : (Chaîne de caractères)`**



- Nom du dossier en cours, 

- Chaîne vide ("") en cas d'erreur. En cas d'erreur, la variable [ErreurDétectée](../WDLang1/3087001.md) est à <u><u><u><u>Vrai</u></u></u></u>. Pour connaître le libellé de l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.




**`<Session> : (Variable de type emailSessionOutlook)`**

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Nom de la variable de type [emailSessionOutlook](../WDLang3/1000018767.md) correspondant à la session email en cours.

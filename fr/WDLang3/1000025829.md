


## &lt;Carte&gt;.SupprimeImage (Fonction)

***En anglais : &lt;Map&gt;.DeleteImage***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime une image dessinée sur la carte affichée dans un champ Carte.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Suppression d'une image présente dans le champ Carte
CARTE_MaCarte.SupprimeImage(Image)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Supprimer une image identifiée par la variable associée

`<Résultat> = <Champ Carte>.SupprimeImage([<Image>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'image a pu être supprimée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Champ Carte> : (Nom de champ)`**

Nom du champ Carte à manipuler.

**`<Image> : (Variable de type CarteImage optionnel)`**

Nom de la variable de type [CarteImage](../WDLang3/1000025523.md) correspondant à l'image à supprimer. Si ce paramètre n'est pas spécifié, toutes les images de la carte seront supprimées.


<a name="SYNTAXE2"></a>

### Supprimer une image identifiée par son nom

`<Résultat> = <Champ Carte>.SupprimeImage(<Nom de l'image>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'image a pu être supprimée, 

- <u><u><u><u>faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Champ Carte> : (Nom de champ)`**

Nom du champ Carte à manipuler.

**`<Nom de l'image> : (Chaîne de caractères)`**

Nom de l'image à supprimer. Ce nom correspond à la propriété **Nom** de la variable de type [CarteImage](../WDLang3/1000025523.md) à supprimer. Si ce paramètre n'est pas spécifié, toutes les images de la carte seront supprimées.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Pour ajouter une nouvelle image à dessiner sur une carte, utilisez la fonction [&lt;Carte&gt;.AjouteImage](../WDLang3/1000025802.md).

- Pour modifier une image dessinée dans un champ Carte, utilisez la fonction [&lt;Carte&gt;.ModifieImage](../WDLang3/1000025818.md).

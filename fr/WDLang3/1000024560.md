
## saasVisite (Type de variable)

***En anglais : saasVisit***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **saasVisite** permet de connaître toutes les caractéristiques d'une visite d'un utilisateur SaaS. Les caractéristiques de ce type de variable peuvent être connues via différentes propriétés WLangage. 

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
T est un tableau de saasVisite 
T = SaaSAdminListeVisite("MonSiteSaaS", "Client1")
```





<a name="NOTE0"></a>

## Remarques
<a name="NOTE0_1"></a>


### Propriétés spécifiques à la description des variables de type saasVisite
<a name="proprietes_specifiques_description_des_variables_type_saasvisite_ELTPARAGRAPHE000029"></a>Les propriétés suivantes peuvent être utilisées pour manipuler une variable de type **saasVisite** :

| Nom de la propriété | Type manipulé | Effet |
| --- | --- | --- |
| Date | DateHeure | Date et heure à laquelle l'utilisateur s'est connecté. <br><br>**Cette propriété est disponible en lecture seulement**. |
| Durée | Durée | Durée de la connexion. <br><br>**Cette propriété est disponible en lecture seulement**. |
| Site | [saasSite](../WDLang3/1000019068.md) | Site pour lequel l'utilisateur s'est connecté. <br><br>**Cette propriété est disponible en lecture seulement**. |
| Utilisateur | [saasUtilisateur](../WDLang3/1000019083.md) | Utilisateur qui a effectué la visite. <br><br>**Cette propriété est disponible en lecture seulement**. |

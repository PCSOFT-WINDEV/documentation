
## TâcheAjoute (Fonction)

***En anglais : TaskAdd***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ajoute une tâche à la liste des tâches d'une messagerie Lotus Notes ou Outlook. La [structure Tâche](../WDLang3/1000012010.md) permet de définir les données de cette tâche.

**Attention** : Cette fonction **ne permet pas** d'ajouter une tâche à la liste des tâches d'une messagerie Outlook Express.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ouverture d'une session Lotus Notes
IDSession est un entier
IDSession = EmailOuvreSessionNotes("MotDePasse", "ServeurMars", "mail\julie.nsf", ...
				"C:\Users\Julie\AppData\Local\Lotus\Notes\Data\user.id")

// Initialisation des variables de la structure Tâche
TâcheRAZ()
// Renseignements des variables pour le nouveau rendez-vous
mTâche.Objet = "Nouvelle tâche"
mTâche.DateDébut = "20070125100000"
mTâche.NbParticipant = 3
mTâche.Participant[1] = "Romain"
mTâche.Participant[2] = "Lucas"
mTâche.Participant[3] = "Stéphane"
// Ajout de la tâche
SI TâcheAjoute(IDSession) = Vrai ALORS
	Info("Tâche ajoutée")
SINON
	Erreur("Erreur lors de l'ajout", ErreurInfo())
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = TâcheAjoute(<Identifiant de la session>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'ajout a été effectué, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md). 




**`<Identifiant de la session> : (Entier)`**

Identifiant de la session à utiliser. Cet identifiant est renvoyé par une des fonctions suivantes : [EmailOuvreSessionNotes](../WDLang3/3032121.md), [EmailOuvreSessionOutlook](../WDLang3/3032128.md), [NotesOuvreBase](../WDLang3/1000005101.md) ou [OutlookOuvreSession](../WDLang3/3032123.md).

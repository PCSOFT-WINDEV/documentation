
## SaaSAdminDéconnecte (Fonction)

***En anglais : SaaSAdminDisconnect***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Termine la session d'administration d'une base SaaS.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connexion d'administration à la base SaaS
SI SaaSAdminConnecte("admin", "secret") = Faux ALORS
	Erreur("Vous n'êtes pas autorisé à vous connecter en tant qu'administrateur")
	RETOUR
FIN

...

// Fin de la session d'administration
SaaSAdminDéconnecte()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = SaaSAdminDéconnecte()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la déconnexion a été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).

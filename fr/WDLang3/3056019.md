
## NetMsgErreur (Fonction)

***En anglais : NetMsgError***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le message d'erreur correspondant au numéro d'erreur retourné par les fonctions [NetOuvreAccèsDistant](../WDLang3/3056026.md) et [NetFermeAccèsDistant](../WDLang3/3056021.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
RapportOuverture est un entier = NetOuvreAccèsDistant("Ma connexion")
SI RapportOuverture = 0 ALORS
	...
	NetFermeAccèsDistant()
SINON
	Erreur("Erreur lors de la connexion", NetMsgErreur(RapportOuverture))
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = NetMsgErreur(<Numéro de l'erreur>)`
---

**`<Résultat> : (Chaîne de caractères)`**

Message de l'erreur (voir Notes).

**`<Numéro de l'erreur> : (Entier)`**

Numéro de l'erreur retourné par les fonctions [NetOuvreAccèsDistant](../WDLang3/3056026.md) et [NetFermeAccèsDistant](../WDLang3/3056021.md).

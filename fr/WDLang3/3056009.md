


## Fonctions FTP WINDEV
			





<a name="NOTE1"></a>
<a name="NOTE1_1"></a>

	<B><B><B><B>Avertissement</B></B></B></B>

	 Les fonctions permettant de gérer le FTP/RPC WINDEV sont conservées par compatibilité.  Il est conseillé d'utiliser un serveur FTP, SSH ou SCP standard. Pour plus de détails, consultez [Manipuler des fichiers sur un serveur FTP](../WDLang3/3040009.md).






|   |   |
| --- | --- |
| [NetAdresseIP](../WDLang3/3056023.md) | Renvoie l'adresse IP (Internet Protocol) d'un poste. |
| [NetAdresseMAC](../WDLang3/1000016006.md) | Renvoie l'adresse MAC (adresse physique) de l'une des cartes réseau de la machine. |
| [NetConnecte](../WDLang3/3056004.md) | Connecte à un serveur FTP (File Transfer Protocol) ou RPC (Remote Procedure Call) réalisé avec WINDEV.<br><br>**Cette fonction est conservée par compatibilité**. |
| [NetDéconnecte](../WDLang3/3056011.md) | Déconnecte d'un serveur FTP (File Transfer Protocol) ou RPC (Remote Procedure Call) réalisé avec WINDEV.<br><br>**Cette fonction est conservée par compatibilité**. |
| [NetDémarreServeur](../WDLang3/3056013.md) | Lance un serveur FTP (File Transfer Protocol) ou RPC (Remote Procedure Call) réalisé avec WINDEV.<br><br>**Cette fonction est conservée par compatibilité**. |
| [NetEnvoieBuffer](../WDLang3/3056001.md) | Transmet une chaîne de caractères à un serveur FTP (File Transfer Protocol) ou RPC (Remote Procedure Call) réalisé avec WINDEV.<br><br>**Cette fonction est conservée par compatibilité**. |
| [NetEnvoieFichier](../WDLang3/3056005.md) | Transmet un fichier à un serveur FTP (File Transfer Protocol) réalisé avec WINDEV.<br><br>**Cette fonction est conservée par compatibilité**. |
| [NetEnvoieMessage](../WDLang3/3056003.md) | Déclenche un événement Windows sur le système d'un poste serveur FTP (File Transfer Protocol) ou RPC (Remote Procedure Call) réalisé avec WINDEV.<br><br>**Cette fonction est conservée par compatibilité**. |
| [NetEnvoieMessageClient](../WDLang3/3056007.md) | Envoie un message du poste serveur vers tous les postes clients connectés.<br><br>**Cette fonction est conservée par compatibilité**. |
| [NetExécute](../WDLang3/3056002.md) | Exécute un programme spécifique sur un serveur FTP (File Transfer Protocol) ou RPC (Remote Procedure Call) réalisé avec WINDEV.<br><br>**Cette fonction est conservée par compatibilité**. |

| [NetFinServeur](../WDLang3/3056024.md) | Arrête un serveur FTP (File Transfer Protocol) ou RPC (Remote Procedure Call) réalisé avec WINDEV et précédemment lancé avec la fonction [NetDémarreServeur](../WDLang3/3056013.md).<br><br>**Cette fonction est conservée par compatibilité**. |
| [NetInfoAccèsDistant](../WDLang3/3056028.md) | Renvoie des informations sur l'état d'une connexion ainsi que des statistiques sur cette connexion. |

| [NetListeAccèsDistant](../WDLang3/3056027.md) | Renvoie la liste des accès distants disponibles. |
| [NetListeAdresseIP](../WDLang3/1000019683.md) | Renvoie la liste des adresses IP (Internet Protocol) d'une machine. |
| [NetListeDisque](../WDLang3/3056015.md) | Liste les disques disponibles :<br><br>- sur le poste serveur FTP (File Transfer Protocol) ou RPC (Remote Procedure Call) réalisé avec WINDEV,<br><br>- sur le poste client.<br><br><br><br><br>**Cette fonction est conservée par compatibilité**. |
| [NetListeRep](../WDLang3/3056016.md) | Liste les répertoires et les fichiers présents : <br><br>- sur le poste serveur FTP (File Transfer Protocol) ou RPC (Remote Procedure Call) réalisé avec WINDEV,<br><br>- sur le poste client.<br><br><br><br><br>**Cette fonction est conservée par compatibilité**. |
| [NetMACVersIP](../WDLang3/1000016004.md) | Renvoie l'adresse IP correspondant à une adresse MAC donnée. |

| [NetMessageBoxDistant](../WDLang3/3056020.md) | Affiche une boîte de dialogue sur l'écran du poste serveur FTP (File Transfer Protocol) ou RPC (Remote Procedure Call) réalisé avec WINDEV.<br><br>**Cette fonction est conservée par compatibilité**. |
| [NetMsgErreur](../WDLang3/3056019.md) | Renvoie le message d'erreur correspondant au numéro d'erreur retourné par les fonctions [NetOuvreAccèsDistant](../WDLang3/3056026.md) et [NetFermeAccèsDistant](../WDLang3/3056021.md). |
| [NetNomMachine](../WDLang3/3056014.md) | Renvoie le nom de la machine locale ou le nom d'un poste dont on connaît l'adresse IP (résolution DNS inverse). |

| [NetRécupèreFichier](../WDLang3/3056025.md) | Récupère un fichier existant sur un serveur FTP (File Transfer Protocol) réalisé avec WINDEV.<br><br>**Cette fonction est conservée par compatibilité**. |
| [NetRenommeFichier](../WDLang3/3056006.md) | Renomme (ou déplace) un fichier présent sur le serveur FTP (File Transfer Protocol) ou RPC (Remote Procedure Call) réalisé avec WINDEV.<br><br>**Cette fonction est conservée par compatibilité**. |
| [NetSupprimeFichier](../WDLang3/3056018.md) | Supprime un fichier présent sur le serveur FTP (File Transfer Protocol) ou RPC (Remote Procedure Call) réalisé avec WINDEV.<br><br>**Cette fonction est conservée par compatibilité**. |

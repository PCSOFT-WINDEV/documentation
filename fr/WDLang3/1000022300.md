


## &lt;Variable emailSessionOutlook&gt;.SupprimeDossier (Fonction)

***En anglais : &lt;emailOutlookSession variable&gt;.RemoveFolder***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime un dossier dans le logiciel de messagerie Outlook.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 
```wl
// Ouverture de la session Outlook 
IDSession est un EmailSessionOutlook
...
// Suppression d'un dossier
IDSession.SupprimeDossier("Dossiers personnels\Boîte de réception\Travail")
// Suppression d'un dossier dans le dossier en cours
IDSession.SupprimeDossier(IDSession.DossierCourant() + "\Travail")
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Session>.SupprimeDossier(<Chemin du dossier>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la suppression a été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. En cas d'erreur, la variable [ErreurDétectée](../WDLang1/3087001.md) est à <u><u><u><u>Vrai</u></u></u></u>. Pour connaître le libellé de l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.




**`<Session> : (Variable de type emailSessionOutlook)`**

Nom de la variable de type [emailSessionOutlook](../WDLang3/1000018767.md) correspondant à la session d'email à manipuler.

**`<Chemin du dossier> : (Chaîne de caractères Unicode)`**

Chemin du dossier à supprimer.



<a name="NOTE0"></a>

## Remarques
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 

### Cas d'un serveur Outlook
<a name="cas_serveur_outlook_ELTPARAGRAPHE000283"></a>Le dossier doit être présent dans la branche "Dossiers personnels".

Lors de l'appel de la fonction **&lt;Variable emailSessionOutlook&gt;.SupprimeDossier**, les manipulations suivantes sont automatiquement effectuées :

- les messages contenus dans le dossier à supprimer sont déplacés dans la Corbeille (dossier "Eléments supprimés").

- les sous-dossiers du dossier à supprimer et leur contenu sont déplacés dans la Corbeille (dossier "Eléments supprimés").

- le dossier à supprimer est définitivement supprimé.




Si le dossier à supprimer est déjà présent dans la Corbeille, le dossier et son contenu sont définitivement supprimés.

**Rappel** : Cette fonction ne permet pas de supprimer un dossier du logiciel de messagerie Outlook Express.

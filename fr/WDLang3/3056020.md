
## NetMessageBoxDistant (Fonction)

***En anglais : NetRemoteMessageBox***

<a name="Utilisation"></a>
<a name="description"></a>
Affiche une boîte de dialogue sur l'écran du poste serveur FTP (File Transfer Protocol) ou RPC (Remote Procedure Call) réalisé avec WINDEV.

**Remarque** : Il est possible d'autoriser ou non l'utilisation de cette fonction sur un serveur FTP/RPC grâce à la fonction [NetOptionServeur](../WDLang3/3056022.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
ConnectRPC est un entier
ConnectRPC = NetConnecte("148.61.125.245", ServeurRPC, "GUEST", "")
...
// Affiche une boîte de dialogue OuiNon sur le serveur
SI NetMessageBoxDistant(ConnectRPC, ...
	"Voulez-vous continuer ?", "", netOuiNon) = 1 ALORS
	...
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = NetMessageBoxDistant(<Identifiant connexion> , <Message> , <Titre de la fenêtre> , <Type de la fenêtre>)`
---

**`<Résultat> : (Entier)`**

Numéro du bouton cliqué dans la boîte de message sur le serveur.

**`<Identifiant connexion> : (Entier)`**

Identifiant de la connexion du poste client (obtenu par la fonction [NetConnecte](../WDLang3/3056004.md)).

**`<Message> : (Chaîne de caractères Ansi)`**

Texte à afficher dans la boîte de dialogue.

**`<Titre de la fenêtre> : (Chaîne de caractères Ansi)`**

Texte affiché dans la barre de titre de la fenêtre.

**`<Type de la fenêtre> : (Constante de type entier)`**

Type de la boîte de dialogue à afficher.


|   |   |
| --- | --- |
| *netConfirmer* | Boîte de message avec les boutons Oui, Non et Annuler. |
| *netInfo* | Boîte de message de type information (bouton OK uniquement). |
| *netOKAnnuler* | Boîte de message avec les boutons OK et Annuler. |
| *netOuiNon* | Boîte de message avec les boutons Oui et Non. |
| *netRéessayerAnnuler* | Boîte de message avec les boutons Réessayer et Annuler. |
| *netRéessayerIgnorer* | Boîte de message avec les boutons Abandon, Réessayer et Ignorer. |

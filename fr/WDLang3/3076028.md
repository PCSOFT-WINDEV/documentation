
## telOrigine (Fonction)

***En anglais : tapiOrigin***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de connaître l'origine d'un appel (appel entrant, appel sortant, ...).
<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = telOrigine([<Identifiant de l'appel>])`
---

**`<Résultat> : (Constante de type Entier)`**

Origine de l'appel.


|   |   |
| --- | --- |
| *telOrigineAppelConférence* | Appel en mode conférence (à plusieurs). |
| *telOrigineAppelEntrant* | Appel entrant : communication provenant de l'extérieur. <br>L'utilisation de cette variable dépend du type de matériel utilisé, du driver, ... |
| *telOrigineAppelExterne* | Appel provenant de l'extérieur. <br>L'utilisation de cette variable dépend du type de matériel utilisé, du driver, ... |
| *telOrigineAppelInconnue* | Appel d'origine indéterminée. <br>L'utilisation de cette variable dépend du type de matériel utilisé, du driver, ... |
| *telOrigineAppelIndisponible* | Appel d'origine indéterminée. <br>L'utilisation de cette variable dépend du type de matériel utilisé, du driver, ... |
| *telOrigineAppelInterne* | Appel interne au PABX |
| *telOrigineAppelSortant* | Appel sortant : communication effectuée vers l'extérieur. |



**`<Identifiant de l'appel> : (Entier optionnel)`**

Identifiant de l'appel à manipuler. Cet identifiant est connu dans la procédure WLangage appelée par la fonction [telDémarreDétectionAppel](../WDLang3/3076029.md).

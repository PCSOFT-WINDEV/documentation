


## grDestinationChamp (Fonction)

***En anglais : grDestinationControl***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Définit un champ Image comme destination d'un graphe.

**Remarques** :

- Cette fonction ne peut pas être utilisée avec les champs Graphe. 

- Cette fonction ne peut pas être utilisée sous l'éditeur d'états.




![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Les graphes](../WDChamp/3042060.md). 








<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Définition du champ "IMG_ImageGraphe" comme destination du graphe "MonGraphe"
grDestinationChamp("MonGraphe", IMG_ImageGraphe)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`grDestinationChamp(<Nom du graphe> , <Champ Image>)`
---

**`<Nom du graphe> : (Chaîne de caractères)`**

Nom du graphe à manipuler, défini avec la fonction [grCrée](../WDLang3/3042005.md).  

**`<Champ Image> : (Nom de champ)`**

Nom du champ Image dans lequel le graphe doit être dessiné.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Créer et dessiner un graphe par programmation
<a name="creer_dessiner_graphe_par_programmation_ELTPARAGRAPHE000086"></a>Pour dessiner un graphe :

- créez le graphe (fonction [grCrée](../WDLang3/3042005.md)),

- ajoutez les données (fonctions [grAjouteDonnée](../WDLang3/3042023.md), [grBourseAjouteDonnée](../WDLang3/3042009.md) et [grNuageAjouteDonnéeXY](../WDLang3/3042032.md)),

- définissez la destination du graphe à l'aide des fonctions suivantes : 

	- [grDestinationBMP](../WDLang3/3042018.md), 

	- **grDestinationChamp**, 

	- [grDestinationEMF](../WDLang3/3042017.md), 

	- [grDestinationFen](../WDLang3/3042031.md),

	- [grDestinationImage](../WDLang3/1000020952.md),

	- [grDestinationWMF](../WDLang3/3042019.md). 




- dessinez le graphe (fonction [grDessine](../WDLang3/3042026.md)).


Lors du prochain appel à la fonction [grDessine](../WDLang3/3042026.md), le graphe sera dessiné à l'intérieur du champ Image &lt;Champ Image&gt;. Il est possible d'associer une autre image à ce champ : le graphe sera alors dessiné en surimpression devant l'image.

<a name="changement_destination_ELTPARAGRAPHE000142"></a>Si le graphe était précédemment affiché dans une fenêtre indépendante, cette fenêtre indépendante sera fermée lors du prochain appel à la fonction [grDessine](../WDLang3/3042026.md).

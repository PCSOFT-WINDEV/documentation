


## Fonctions OBEX
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions OBEX sont les suivantes : 



|   |   |
| --- | --- |
| [OBEXConnecte](../WDLang3/1000017153.md) | Permet de se connecter à un périphérique supportant le protocole OBEX. <br>Rappel : Le protocole OBEX est un protocole de transfert de données utilisable par les faisceaux infrarouges et par Bluetooth. |
| [OBEXDéconnecte](../WDLang3/1000017154.md) | Permet de se déconnecter d'un périphérique supportant le protocole OBEX. |
| [OBEXEnvoieFichier](../WDLang3/1000017151.md) | Envoie un fichier sur un périphérique gérant le protocole OBEX. |
| [OBEXEnvoieVCard](../WDLang3/1000017155.md) | Envoie un fichier VCard (carte de visite virtuelle) à un périphérique gérant le protocole OBEX. |
| [OBEXListeFichier](../WDLang3/1000017148.md) | Liste les fichiers partagés par un périphérique utilisant le protocole OBEX. |
| [OBEXRécupèreFichier](../WDLang3/1000017152.md) | Récupère un fichier sur un périphérique gérant le protocole OBEX. |
| [OBEXRécupèreVCard](../WDLang3/1000017156.md) | Récupère une VCard sur un périphérique gérant le protocole OBEX. |






- ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=WDBluetooth.gif) ***Exemples didactiques (WINDEV)*** : **WD Bluetooth** <br>WINDEV met à votre disposition des fonctions de gestion du Bluetooth.<br>Ainsi depuis vos applications WINDEV, vous pouvez gérer tous types de périphériques Bluetooth : oreillettes, enceintes, téléphones portables, imprimantes, ...<br>Cet exemple didactique montre l'utilisation de ces fonctions Bluetooth :<br>- pour lister les périphériques Bluetooth accessibles<br>- pour lister les services offerts par un périphérique  <br>- pour se connecter à un périphérique Bluetooth<br>- pour envoyer un fichier à un périphérique via les fonctions "OBEX" du WLangage.

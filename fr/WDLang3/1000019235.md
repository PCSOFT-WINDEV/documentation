
## telDialerCompose (Fonction)

***En anglais : tapiDialerCall***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ouvre l'application de téléphonie (dialer) par défaut du téléphone et compose le numéro spécifié.








<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
telDialerCompose("0612345678")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = telDialerCompose(<Numéro de téléphone>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'application de téléphonie a été correctement lancée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. En cas d'erreur, il est possible d'obtenir le détail de l'erreur grâce à la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Numéro de téléphone> : (Chaîne de caractères)`**

Numéro de téléphone à composer. 
Si ce paramètre correspond à une chaîne vide (""), la fonction retournera <u><u><u><u>Faux</u></u></u></u>.
Attention : le numéro ne doit pas contenir d'espaces.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Cas particuliers
<a name="cas_particuliers_ELTPARAGRAPHE000084"></a>

- Il n'est pas possible de déclencher automatiquement l'appel des numéros d'urgence.

- Il n'est pas possible de composer des codes USSD.

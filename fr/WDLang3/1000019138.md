
## SaaSAdminListeAbonnement (Fonction)

***En anglais : SaaSAdminListSubscription***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Liste les abonnements SaaS associés à un site ou une tarification.

**Remarques** : 

- L'utilisation de cette fonction nécessite d'avoir ouvert une session d'administration SaaS au préalable en utilisant la fonction [SaaSAdminConnecte](../WDLang3/1000019098.md).

- L'Administrateur SaaS aura accès à tous les abonnements. Un gestionnaire de compte client aura accès uniquement à ses propres abonnements. 



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Comptage du nombre global d'abonnements
tabAbo est un tableau de saasAbonnement 
tabClient est un tableau de saasClient 

tabAbo = SaaSAdminListeAbonnement("CRM")
tabClient = SaaSAdminListeClient()
Info("Le site CRM possède " + tabAbo.Occurrence + " abonnés (sur l'ensemble des comptes clients).")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Lister les abonnements à un site SaaS

`<Résultat> = SaaSAdminListeAbonnement(<Site SaaS>)`
---

**`<Résultat> : (Tableau de variables de type saasAbonnement)`**

Variable de type [saasAbonnement](../WDLang3/1000019088.md) contenant la liste des abonnements au site recherché.

**`<Site SaaS> : (Variable de type saasSite ou chaîne de caractères)`**

Nom de la variable de type [saasSite](../WDLang3/1000019068.md) (ou nom du site) correspondant au site SaaS pour lequel les abonnements sont recherchés.

Si ce paramètre est de type chaîne de caractères, il doit correspondre au nom du site.


<a name="SYNTAXE2"></a>

### Lister les abonnements à une tarification (utilisation des variables de type saasxxx)

`<Résultat> = SaaSAdminListeAbonnement(<Tarification>)`
---

**`<Résultat> : (Tableau de variables de type saasAbonnement)`**

Variable de type [saasAbonnement](../WDLang3/1000019088.md) contenant la liste des abonnements à la tarification spécifiée.

**`<Tarification> : (Variable de type saasTarification)`**

Nom de la variable de type [saasTarification](../WDLang3/1000019091.md) correspondant à la tarification pour laquelle les abonnements doivent être listés.


<a name="SYNTAXE3"></a>

### Lister les abonnements à une tarification (identifiée par son nom)

`<Résultat> = SaaSAdminListeAbonnement(<Tarification> , <Site SaaS>)`
---

**`<Résultat> : (Tableau de variables de type saasAbonnement)`**

Variable de type [saasAbonnement](../WDLang3/1000019088.md) correspondant à la liste des abonnements à la tarification spécifiée.

**`<Tarification> : (Chaîne de caractères)`**

Nom de la tarification pour laquelle les abonnements doivent être listés.

**`<Site SaaS> : (Chaîne de caractères)`**

Nom du site SaaS dont dépend la tarification.

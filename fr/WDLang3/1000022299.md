


## &lt;Variable emailSessionIMAP&gt;.SupprimeDossier (Fonction)

***En anglais : &lt;emailIMAPSession variable&gt;.RemoveFolder***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime un dossier sur un serveur IMAP.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Protocole IMAP
MaSession est un emailSessionIMAP
...
// Suppression d'un dossier dans le dossier en cours 
MaSession.SupprimeDossier(MaSession.DossierCourant() + "\Travail")

// Suppression d'un dossier dans le dossier personnel
MaSession.SupprimeDossier("Dossiers personnels\Boîte de réception\TravailPerso")
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Session>.SupprimeDossier(<Chemin du dossier>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la suppression a été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. En cas d'erreur, la variable [ErreurDétectée](../WDLang1/3087001.md) est à <u><u><u><u>Vrai</u></u></u></u>. Pour connaître le libellé de l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.




**`<Session> : (Variable de type emailSessionIMAP)`**

Nom de la variable de type [emailSessionIMAP](../WDLang3/1000018957.md) correspondant à la session d'email à manipuler.

**`<Chemin du dossier> : (Chaîne de caractères Unicode)`**

Chemin du dossier à supprimer.

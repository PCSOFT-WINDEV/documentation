


## &lt;Variable emailSessionIMAP&gt;.DossierCourant (Fonction)

***En anglais : &lt;emailIMAPSession variable&gt;.CurrentFolder***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le nom du dossier en cours dans le logiciel de messagerie utilisant le protocole IMAP (Internet Message Access Protocol). Ce dossier est présent dans la branche "Dossiers personnels".
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Protocole IMAP
MaSession est un emailSessionIMAP
...
// Création d'un dossier dans le dossier en cours 
MaSession.AjouteDossier(MaSession.DossierCourant() + "\Travail")

// Création d'un dossier dans le dossier personnel
MaSession.AjouteDossier("Dossiers personnels\Boîte de réception\TravailPerso")
```


<a name="3032129_Exemple2"></a>

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Session>.DossierCourant()`
---

**`<Résultat> : (Chaîne de caractères)`**



- Nom du dossier en cours, 

- Chaîne vide ("") en cas d'erreur. En cas d'erreur, la variable [ErreurDétectée](../WDLang1/3087001.md) est à <u><u><u><u>Vrai</u></u></u></u>. Pour connaître le libellé de l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.




**`<Session> : (Variable de type emailSessionIMAP)`**

Nom de la variable de type [emailSessionIMAP](../WDLang3/1000018957.md) correspondant à la session email en cours.




## grTypePoint (Fonction)

***En anglais : grPointType***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie la forme du point d'une catégorie ou d'une valeur particulière dans un graphe de type Courbe, Aire, Nuage de points et Radar. 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Transforme en cercle le point (catégorie) 2 de la série 1
grTypePoint(GRF_MonGraphe, 1, 2, grPointCercle)

// Transforme en carré le point (catégorie) 3 de toutes les séries
grTypePoint(GRF_MonGraphe, 3, grPointCarre)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Modifier la forme du point d'une catégorie (pour toutes les séries)

`grTypePoint(<Nom du graphe> , <Numéro de la catégorie> , <Type>)`
---

**`<Nom du graphe> : (Nom de champ ou chaîne de caractères)`**

Nom du graphe à manipuler. Ce nom correspond :

- soit au nom du graphe défini par programmation avec la fonction [grCrée](../WDLang3/3042005.md).

- soit au nom du champ Graphe (présent dans l'éditeur de fenêtres ou d'états).




**`<Numéro de la catégorie> : (Entier)`**

Indice de la catégorie à modifier. 

**`<Type> : (Constante de type Entier)`**

Nouveau type de point de la catégorie : 



|   |   |
| --- | --- |
| *grNonDessiné* | Point non dessiné. |
| *grPointCarré* | Point en forme de carré. |
| *grPointCercle* | Point en forme de cercle. |
| *grPointCroix* | Point en forme de croix. |
| *grPointLosange* | Point en forme de losange. |
| *grPointTriangle* | Point en forme de triangle. |




<a name="SYNTAXE2"></a>

### Modifier la forme d'un point particulier

`grTypePoint(<Nom du graphe> , <Numéro de la série> , <Numéro de la catégorie> , <Type>)`
---

**`<Nom du graphe> : (Nom de champ ou chaîne de caractères)`**

Nom du graphe à manipuler. Ce nom correspond :

- soit au nom du graphe défini par programmation avec la fonction [grCrée](../WDLang3/3042005.md).

- soit au nom du champ Graphe (présent dans l'éditeur de fenêtres ou d'états).




**`<Numéro de la série> : (Entier)`**

Numéro de la série dans laquelle le type de point doit être modifié.

**`<Numéro de la catégorie> : (Entier)`**

Indice de la catégorie à modifier. 

**`<Type> : (Constante de type Entier)`**

Nouveau type de point de la catégorie : 



|   |   |
| --- | --- |
| *grNonDessiné* | Point non dessiné. |
| *grPointCarré* | Point en forme de carré. |
| *grPointCercle* | Point en forme de cercle. |
| *grPointCroix* | Point en forme de croix. |
| *grPointLosange* | Point en forme de losange. |
| *grPointTriangle* | Point en forme de triangle. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
<a name="NOTE0_2"></a>
Pour changer le type de tous les points d'une série, il est recommandé d'utiliser la fonction [grTypePointSérie](../WDLang3/1000022271.md).


## SaaSAdminModifieAbonnement (Fonction)

***En anglais : SaaSAdminModifySubscription***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie un abonnement SaaS.

**Remarques** : 

- L'utilisation de cette fonction nécessite d'avoir ouvert une session d'administration SaaS au préalable en utilisant la fonction [SaaSAdminConnecte](../WDLang3/1000019098.md).

- L'Administrateur SaaS gère tous les abonnements. Un gestionnaire de compte client pourra uniquement modifier un abonnement de son propre compte. 



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
S est un saasSite dynamique 
T est une saasTarification dynamique 
C est un saasClient dynamique 
Ab est un saasAbonnement dynamique

S = SaaSAdminChercheSite("Forum")
T = SaaSAdminChercheTarification("Promo 2010", S)
C = SaaSAdminChercheClient("Hybrid")

Ab = SaaSAdminChercheAbonnement(T, C)
Ab.NbLicences = 3

// Le client Hybrid possède maintenant 3 licences d'utilisation pour la Promo 2010 du Forum
SaaSAdminModifieAbonnement(Ab, T, C)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = SaaSAdminModifieAbonnement(<Abonnement SaaS>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la modification a été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Abonnement SaaS> : (Variable de type saasAbonnement)`**

Nom de la variable de type [saasAbonnement](../WDLang3/1000019088.md) correspondant à l'abonnement à modifier dans la base SaaS.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Toute modification d'un abonnement se traduit sur les relevés d'utilisation de la façon suivante :

- Terminaison de l'abonnement,

- Début d'un nouvel abonnement (portant le même nom) avec les nouvelles caractéristiques.


Ceci est nécessaire pour permettre de calculer automatiquement les tarifs en fonction du nombre de licences et des durées d'utilisation.

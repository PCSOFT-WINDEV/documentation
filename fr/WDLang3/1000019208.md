
## GPSInfo (Fonction)

***En anglais : GPSInfo***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie les informations sur le fournisseur de localisation utilisé par l'application pour les fonctions de géolocalisation.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple

![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) 
```wl
// Test du type de fournisseur utilisé
SI GPSInfo(gpsType) = gpsSatellite ALORS
	Info("La géolocalisation du téléphone utilise le système GPS.")
FIN
```



```wl
SI GPSInfo(gpsVitesse) = Faux ALORS
	Erreur("La vitesse de déplacement du téléphone ne peut pas être déterminée.")
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = GPSInfo(<Information>)`
---

**`<Résultat> : (Variant)`**

Information demandée en fonction du paramètre &lt;Information&gt;.

**`<Information> : (Constante de type Entier)`**

Type d'information demandée.


|   |   |   |
| --- | --- | --- |
| *gpsAltitude* | Altitude du périphérique disponible. | &lt;Résultat&gt; correspondra à :<br><br>- <u><u><u><u>Vrai</u></u></u></u> si le fournisseur de localisation permet de connaître l'altitude du périphérique,<br><br>- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire ou si aucun fournisseur de localisation n'a été trouvé.<br><br><br> |
| *gpsDirection* | Direction du déplacement du périphérique disponible. | &lt;Résultat&gt; correspondra à :<br><br>- <u><u><u><u>Vrai</u></u></u></u> si le fournisseur de localisation permet de connaître la direction du déplacement du périphérique,<br><br>- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire ou si aucun fournisseur de localisation n'a été trouvé.<br><br><br> |
| *gpsEnergie* | Consommation d'énergie requise par le fournisseur. | &lt;Résultat&gt; correspondra à :<br><br>- *gpsEnergieFaible* : Le fournisseur doit requérir une faible quantité d'énergie.<br><br>- *gpsEnergieMoyenne* : Le fournisseur peut requérir une quantité moyenne d'énergie.<br><br>- *gpsEnergieElevée* : Le fournisseur peut requérir une quantité d'énergie élevée.<br><br><br> |
| *gpsPrécision* | Précision de la mesure de position effectuée par le fournisseur. | &lt;Résultat&gt; correspondra à :<br><br>- *gpsPrécisionElevée* : calcul de la position avec une précision élevée.<br><br>- *gpsPrécisionFaible* : calcul de la position avec une précision faible.<br><br><br> |
| *gpsType* | Type du fournisseur de localisation. | &lt;Résultat&gt; correspondra à :<br><br>- *gpsSatellite* : fournisseur de localisation par le réseau de satellites GPS.<br><br>- *gpsRéseau* : fournisseur de localisation par le réseau des antennes mobiles ou Wi-Fi.<br><br>- 0 si le type du fournisseur de localisation n'a pas été déterminé. <br><br><br><br><br>![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Cette constante n'est pas disponible. |
| *gpsVitesse* | Vitesse de déplacement du périphérique disponible. | &lt;Résultat&gt; correspondra à :<br><br>- <u><u><u><u>Vrai</u></u></u></u> si le fournisseur de localisation permet de connaître la vitesse de déplacement du périphérique,<br><br>- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire ou si aucun fournisseur de localisation n'a été trouvé.<br><br><br> |











<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Cas particuliers
<a name="cas_particuliers_ELTPARAGRAPHE000142"></a>

- Il est conseillé d'initialiser les paramètres du fournisseur de localisation à l'aide de la fonction [GPSInitParamètre](../WDLang3/1000019204.md) avant de récupérer les informations de ce fournisseur.

- L'état d'activation du fournisseur de localisation peut être connu grâce à la fonction [GPSEtat](../WDLang3/1000019207.md).



![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) 

### Fonctionnalité d'application requise
<a name="fonctionnalite_application_requise_ELTPARAGRAPHE000357"></a>L'utilisation de cette fonction entraîne la déclaration d'une fonctionnalité d'application dans l'assistant de génération de l'application.

Fonctionnalité requise : Géolocalisation

Cette fonctionnalité permet aux applications d'accéder aux données de géolocalisation de l'appareil ou de l'ordinateur.

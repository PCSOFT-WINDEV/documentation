
## SaaSAdminSupprimeService (Fonction)

***En anglais : SaaSAdminDeleteService***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime un service SaaS.

**Remarques** : 

- L'utilisation de cette fonction nécessite d'avoir ouvert une session d'administration SaaS au préalable en utilisant la fonction [SaaSAdminConnecte](../WDLang3/1000019098.md).

- L'utilisation de cette fonction est réservée à l'Administrateur SaaS. Un gestionnaire de compte client ne peut pas utiliser cette fonction. 



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Retire le service "Forum Support" du site SaaS "CRM"
S est un saasSite dynamique 
MonService est un saasService dynamique 

S = SaaSAdminChercheSite("CRM")
MonService = SaaSAdminChercheService("Forum Support", S)

SaaSAdminSupprimeService(MonService)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Supprimer un service SaaS (utilisation des variables de type saasxxx)

`<Résultat> = SaaSAdminSupprimeService(<Service SaaS>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la suppression a été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Service SaaS> : (Variable de type saasService)`**

Nom de la variable de type [saasService](../WDLang3/1000019096.md) correspondant au service à supprimer de la base SaaS.


<a name="SYNTAXE2"></a>

### Supprimer un service SaaS identifié par son nom

`<Résultat> = SaaSAdminSupprimeService(<Service SaaS> , <Site SaaS>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la suppression a été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Service SaaS> : (Chaîne de caractères)`**

Nom du service SaaS à supprimer de la base SaaS.

**`<Site SaaS> : (Chaîne de caractères)`**

Nom du site SaaS possédant le service à supprimer.

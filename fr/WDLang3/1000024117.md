


## &lt;Graphe&gt;.PoliceEtiquette (Fonction)

***En anglais : &lt;Chart&gt;.LabelFont***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie la police utilisée pour les étiquettes de catégorie d'un graphe. ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=EtiquetteCat.gif)




![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Les graphes](../WDChamp/3042060.md). 








<a name="sample_code"></a>

## Exemple

<a name="Exemple1"></a>
<a name="3042012_Exemple1"></a>

```wl
// Modification de la police des étiquettes des axes du champ Graphe : GRF_Graphe

// Création d'une police spécifique
polEtiquettes est une Police
polEtiquettes.Nom = "Verdana"
polEtiquettes.Taille = 10
polEtiquettes.Gras = Vrai

// Changement de police
GRF_Graphe.PoliceEtiquette(polEtiquettes)
GRF_Graphe.Dessine()
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Définition de la police utilisée pour les étiquettes d'un graphe

`<Champ Graphe>.PoliceEtiquette(<Police utilisée>)`
---

**`<Champ Graphe> : (Nom de champ)`**

Nom du champ Graphe présent dans l'éditeur d'états.


**`<Police utilisée> : (Police)`**

Police à utiliser. Ce paramètre correspond à une variable de type [Police](../Motscles/1514045.md). Les caractéristiques de la police ont été définies grâce à la fonction [PoliceCrée](../WDLang1/3060002.md) ou grâce aux [propriétés de gestion des polices](../Proprietes/2515045.md).


<a name="SYNTAXE2"></a>

### Définition de la police utilisée pour les étiquettes d'un graphe (syntaxe conservée par compatibilité)

`<Champ Graphe>.PoliceEtiquette(<Identifiant de la police>)`
---

**`<Champ Graphe> : (Nom de champ)`**

Nom du champ Graphe présent dans l'éditeur d'états.


**`<Identifiant de la police> : (Entier)`**

Identifiant de la police, défini avec la fonction [grCréePolice](../WDLang3/3042001.md). Si cette police n'existe pas, une erreur du WLangage est générée.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Manipulation des étiquettes de catégories
<a name="manipulation_des_etiquettes_categories_ELTPARAGRAPHE000296"></a>Les étiquettes de catégories sont affichées :

- sous les graduations horizontales pour les graphes de type courbe ou les graphes boursiers de type chandelier japonais et BarCharts,

- sous l'axe des X pour les histogrammes ou les graphes de type boursier Minimum/Maximum.


![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=EtiquetteCat.gif)
Les étiquettes de catégories ne sont pas utilisables avec les graphes de type nuage de points.

La définition des étiquettes de catégories est réalisée avec la fonction [&lt;Graphe&gt;.EtiquetteCatégorie](../WDLang3/1000024072.md).

Il est possible de modifier la police de l'étiquette grâce à la fonction **&lt;Graphe&gt;.PoliceEtiquette**.

Les étiquettes de catégories définies avec la fonction [&lt;Graphe&gt;.EtiquetteCatégorie](../WDLang3/1000024072.md) peuvent être affichées dans le libellé du graphe de type Secteur (constante *grAfficheEtiquette* utilisée avec la fonction [&lt;Graphe&gt;.Libellé](../WDLang3/1000024165.md)).
<a name="NOTE0_2"></a>


### Initialisation des polices
<a name="initialisation_des_polices_ELTPARAGRAPHE000336"></a>La police utilisée par défaut pour les étiquettes a les caractéristiques suivantes :

- Nom de la police : Arial

- Taille : 14

- Style : Normal

- Couleur : Noir



<a name="NOTE0_3"></a>


### Orientation de la police
<a name="orientation_police_ELTPARAGRAPHE000351"></a>Dans le cas des étiquettes d'un graphe, la propriété "orientation" de la police est ignorée. 

Pour changer l'orientation de la police des étiquettes, il faut modifier directement la description du champ Graphe dans la fenêtre à 7 onglets. 

Par défaut, l'orientation est choisie de manière automatique. Cette orientation tient compte de la taille et du nombre d'étiquettes à afficher.




## &lt;Graphe&gt;.ChargeParamètre (Fonction)

***En anglais : &lt;Chart&gt;.LoadParameter***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Restaure les paramètres de dessin d'un graphe. Ces paramètres ont été sauvegardés par la fonction [&lt;Graphe&gt;.SauveParamètre](../WDLang3/1000024127.md).

**Rappel** : Par défaut, les paramètres d'un graphe modifiés par l'utilisateur sont sauvegardés lors de l'arrêt de l'application. Ces paramètres sont automatiquement restaurés au prochain lancement de l'application.

![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Les graphes](../WDChamp/3042060.md). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Sauvegarde des paramètres d'un champ Graphe
INIEcrit("Parametres", "GRF_Graphe", "Param", GRF_Graphe.SauveParamètre())

// Restauration des paramètres d'un champ Graphe
GRF_Graphe.ChargeParamètre(INILit("Parametres", "GRF_Graphe", "Param"))
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Champ Graphe>.ChargeParamètre(<Paramètres>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le chargement des paramètres a été effectué, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Champ Graphe> : (Nom de champ)`**

Nom du champ Graphe à manipuler (présent dans l'éditeur de fenêtres, de pages ou d'états).

**`<Paramètres> : (Chaîne de caractères Ansi)`**

Paramètre à restaurer. Ce paramètre est dans un format compressé. Il peut uniquement être obtenu par la fonction [&lt;Graphe&gt;.SauveParamètre](../WDLang3/1000024127.md). La valeur de ce paramètre peut être sauvegardée dans un fichier, un mémo ou dans la base de registre.

Si ce paramètre correspond à une chaîne vide (""), la fonction n'a aucun effet et renvoie <u><u><u><u>Faux</u></u></u></u>.

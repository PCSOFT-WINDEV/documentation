


## CarteRécupèreObjetJS (Fonction)

***En anglais : MapGetJSObject***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Retourne l'instance de l'API d'un champ Carte. Il est ainsi possible de manipuler le champ Carte directement dans du code Javascript. 

Combinée à la fonction [CarteConfigureAPIGgl](../WDLang3/1000021611.md), cette fonction permet de gérer des paramètres supplémentaires, non proposés par défaut par le champ Carte. 

**Remarque** : En WINDEV, cette fonction peut être utilisée dans le code Javascript exécuté par la fonction [CarteExécuteJS](../WDLang3/1000021615.md) pour manipuler le champ Carte de l'application WINDEV.  
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Code serveur
CarteConfigureAPIGgl("libraries=geometry,places","language=fr")
```

```wl
// Code navigateur
MaCarte est un objet dynamique
MaCarte = CarteRécupèreObjetJS(CARTE_Navigateur)
RechercheAdresse(MaCarte)
```

```txt
// Fonction Javascript utilisant l'objet dynamique
function RechercheAdresse(MaCarte)
{
	var infowindow;
	// Rome : 41.873144, 12.480391
	var pyrmont = new google.maps.LatLng(41.873144, 12.480391);
	var request = {
		location: pyrmont,
		radius: 500,
		types: ['store']
		};
	infowindow = new google.maps.InfoWindow();
	var service = new google.maps.places.PlacesService(MaCarte);
	service.nearbySearch(request, RetourAdresse_WB);
	
}
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = CarteRécupèreObjetJS([<Champ Carte>])`
---

**`<Résultat> : (Objet dynamique)`**

Nom de la variable de type Objet dynamique contenant l'instance de l'API du champ Carte. 

**`<Champ Carte> : (Nom de champ optionnel)`**

Nom du champ Carte à manipuler. Si ce paramètre correspond à une chaîne vide ("") ou n'est pas spécifié, le champ Carte auquel appartient l'événement en cours sera utilisé.

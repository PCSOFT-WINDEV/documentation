


## CarteLicenceGgl (Fonction)

***En anglais : MapLicenseGgl***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Indique la clé de licence "Google Maps API" à utiliser dans les champs Carte (de WINDEV ou WEBDEV). Cette clé peut être nécessaire pour utiliser les fonctionnalités "Google Maps API for work". Pour plus de détails, consultez la licence de "Google Maps API".

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) **Important** : L'appel de la fonction **CarteLicenceGgl** doit être réalisé **avant** l'ouverture de la fenêtre contenant le champ Carte. Si le champ Carte se trouve dans la première fenêtre du projet, l'appel de la fonction **CarteLicenceGgl** doit être réalisé dans le code d'initialisation du projet.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Initialisation de la clé Google
CarteLicenceGgl("ABCDEF12345789abcdef")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`CarteLicenceGgl(<Clé Google>)`
---

**`<Clé Google> : (Chaîne de caractères)`**

Clé fournie par Google.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- L'utilisation d'un champ Carte est soumise à des conditions d'utilisations spécifiques. Consultez la licence Google sur l'utilisation des services de cartographie. 
	
	


|   |
| --- |
| **Attention : A partir du 22 Juin 2016, les conditions d'utilisation des services de cartographie de Google ont évolué**. Il est désormais **obligatoire** de spécifier une clé de licence pour utiliser ce service. Vous devez : <br><br>	1. Générer la clé dans la console développeur de Google ([https://console.developers.google.com](https://console.developers.google.com)). Les manipulations à réaliser sont les suivantes (à la date d'écriture de cette page) : <br><br>		- Utilisez un compte Google pour se connecter à la console développeur. <br><br>		- Créez un projet Google si nécessaire. <br><br>		- Accédez au "Gestionnaire d'API" du projet.<br><br>		- Dans la rubrique "API Google Maps", cliquez sur le lien "Google Maps JavaScript API".<br><br>		- Cliquez sur "Activer" afin d'activer l'API Google Maps Javascript API. <br><br>		- Dans la rubrique "Identifiants", déroulez le bouton "Créer des identifiants". <br><br>		- Sélectionnez "Clé d'API". <br><br>		- Sélectionnez "Clé d'API". <br><br>			- Dans le cas d'un site WEBDEV, sélectionnez "Clé navigateur". Il est recommandé de préciser le domaine qui utilisera la clé pour le suivi des requêtes et pour éviter toute usurpation.<br><br>			- Dans le cas d'une application WINDEV, sélectionnez "Clé serveur". Il est recommandé de préciser l'adresse qui utilisera la clé pour le suivi des requêtes et pour éviter toute usurpation.<br><br><br><br><br>	- Validez la création de la clé, puis copiez la clé créée par Google. <br><br>	2. Indiquez la clé utilisée dans l'application ou le site grâce à la fonction **CarteLicenceGgl**.<br><br><br> |



- Cette fonction permet de spécifier la licence à utiliser pour que le fournisseur de carte Google Maps donne à l'application : 

	- **à partir du 22 juin 2016**, accès aux fonctionnalités "Google Maps JavaScript API". 

	- accès aux fonctionnalités étendues de "Maps API for work" (nombre de requêtes plus important, suivi, quotas, ...).




- La licence est appliquée à toutes les cartes de l'application et sera utilisée pour tous les nouveaux champs Carte créés après cet appel. Il est recommandé d'appeler cette fonction dans le code d'initialisation du projet.

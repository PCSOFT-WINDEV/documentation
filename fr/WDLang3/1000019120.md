
## SaaSAdminAjouteUtilisateur (Fonction)

***En anglais : SaaSAdminAddUser***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ajoute un utilisateur SaaS.

**Remarques** : 

- L'utilisation de cette fonction nécessite d'avoir ouvert une session d'administration SaaS au préalable en utilisant la fonction [SaaSAdminConnecte](../WDLang3/1000019098.md).

- L'Administrateur SaaS gère tous les utilisateurs de tous les clients. Un gestionnaire de compte client ne peut ajouter des utilisateurs que dans son propre compte client. 



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Recherche du compte client
C est un saasClient dynamique 
C = SaaSAdminChercheClient("Hybrid")

// Ajout d'un nouvel utilisateur
SAM est un saasUtilisateur 
SAM.Login = "sam@hybrid.com"
SAM.MotDePasse = "secret"
SaaSAdminAjouteUtilisateur(SAM, C)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Ajouter un utilisateur (utilisation des variables de type saasxxx)

`<Résultat> = SaaSAdminAjouteUtilisateur(<Utilisateur> , <Compte client>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'ajout a été effectué, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Utilisateur> : (Variable de type saasUtilisateur)`**

Nom de la variable de type [saasUtilisateur](../WDLang3/1000019083.md) correspondant à la description du nouvel utilisateur.

**`<Compte client> : (Variable de type saasClient ou chaîne de caractères)`**

Compte client auquel l'utilisateur doit être rattaché.

Si ce paramètre est une chaîne de caractères, celle-ci peut représenter le nom ou la propriété [RéférenceExterne](../WDLang3/1000019072.md) du compte client.


<a name="SYNTAXE2"></a>

### Ajouter un utilisateur identifié par son nom

`<Résultat> = SaaSAdminAjouteUtilisateur(<Login> , <Mot de passe> , <Compte client>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'ajout a été effectué, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Login> : (Chaîne de caractères)`**

Login du nouvel utilisateur.

**`<Mot de passe> : (Chaîne de caractères)`**

Mot de passe du nouvel utilisateur.

**`<Compte client> : (Chaîne de caractères)`**

Nom ou référence externe du compte client auquel l'utilisateur doit être rattaché.

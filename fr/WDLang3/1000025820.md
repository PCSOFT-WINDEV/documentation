


## &lt;Carte&gt;.SupprimeForme (Fonction)

***En anglais : &lt;Map&gt;.DeleteShape***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime une forme dessinée sur la carte affichée dans un champ Carte.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Suppression d'une forme
CARTE_MaCarte.SupprimeForme(MaForme)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Supprimer une forme via la variable associée

`<Résultat> = <Champ Carte>.SupprimeForme([<Forme>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la forme a pu être supprimée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Champ Carte> : (Nom de champ)`**

Nom du champ Carte à manipuler.

**`<Forme> : (Variable de type CarteCercle, CartePolygone ou CartePolyligne)`**

Forme à supprimer sur la carte affichée dans le champ Carte. Cette forme est décrite à l'aide d'un des types de variables suivants : 

- une variable de type [CartePolygone](../WDLang3/1000025476.md), 

- une variable de type [CartePolyligne](../WDLang3/1000025489.md), 

- une variable de type [CarteCercle](../WDLang3/1000025504.md).


Si ce paramètre n'est pas spécifié, toutes les formes présentes sur la carte seront supprimées.


<a name="SYNTAXE2"></a>

### Supprimer une forme à partir de son nom

`<Résultat> = <Champ Carte>.SupprimeForme(<Identifiant de la forme>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la forme a pu être supprimée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Champ Carte> : (Nom de champ)`**

Nom du champ Carte à manipuler.

**`<Identifiant de la forme> : (Chaîne de caractères)`**

Identifiant de la forme à supprimer. Cet identifiant correspond à la propriété **Nom** de la variable associée à la forme. Si ce paramètre n'est pas spécifié, toutes les formes présentes sur la carte seront supprimées.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Pour ajouter une nouvelle forme à dessiner dans un champ Carte, utilisez la fonction [&lt;Carte&gt;.AjouteForme](../WDLang3/1000025801.md).

- Pour modifier une forme dessinée dans un champ Carte, utilisez la fonction [&lt;Carte&gt;.ModifieForme](../WDLang3/1000025805.md).

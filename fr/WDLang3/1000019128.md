
## SaaSAdminModifieTarification (Fonction)

***En anglais : SaaSAdminModifyPricing***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie une tarification SaaS.

**Remarques** : 

- L'utilisation de cette fonction nécessite d'avoir ouvert une session d'administration SaaS au préalable en utilisant la fonction [SaaSAdminConnecte](../WDLang3/1000019098.md).

- L'utilisation de cette fonction est réservée à l'Administrateur SaaS. Un gestionnaire de compte client ne peut pas utiliser cette fonction. 



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Recherche la tarification à modifier
S est un saasSite dynamique 
T est une saasTarification dynamique 

S = SaaSAdminChercheSite("Forum")
T = SaaSAdminChercheTarification("Promo 2010", S)

// Baisse de 20€ du prix
T.Prix -= 20 

SaaSAdminModifieTarification(T)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = SaaSAdminModifieTarification(<Tarification>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la modification a été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Tarification> : (Variable de type saasTarification)`**

Nom de la variable de type [saasTarification](../WDLang3/1000019091.md) correspondant à la tarification à modifier dans la base SaaS.



<a name="NOTE0"></a>

## Remarques
La modification d'une tarification impacte directement tous les abonnements liés à cette tarification. Par exemple, si le prix de la tarification baisse, tous les abonnements en cours seront impactés rétroactivement.

Pour faire une modification non rétroactive, il faut : 

- supprimer les abonnements liés (fonction [SaaSAdminSupprimeAbonnement](../WDLang3/1000019136.md)).

- modifier la tarification.


## SynthèseVocaleInitialise (Fonction)

***En anglais : SpeechSynthesisInitialize***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Initialise les paramètres de la synthèse vocale pour l'application en cours. Il est possible d'initialiser : 

- Uniquement la langue utilisée (syntaxe 1). 

- Toutes les caractéristiques de la voix via une variable de type [VoixDeSynthèse](../WDLang3/1000025986.md) (syntaxe 2). 




**Remarque** : L'utilisation de la fonction **SynthèseVocaleInitialise** n'est pas obligatoire. Si la fonction [SynthèseVocaleLitTexte](../WDLang3/1000019805.md) ou la fonction [SynthèseVocaleLitFichier](../WDLang3/1000020172.md) est appelée sans que les paramètres de la synthèse vocale n'aient été précisés avec la fonction **SynthèseVocaleInitialise**, les paramètres par défaut seront utilisés.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI PAS SynthèseVocaleInitialise(NationVersLangue(Nation())) ALORS
	Erreur("Echec de l'initialisation de la synthèse vocale.", ErreurInfo())
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Initialiser uniquement la langue

`<Résultat> = SynthèseVocaleInitialise(<Langue> [, <Vitesse>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'initialisation a fonctionné, 

- <u><u><u><u>Faux</u></u></u></u> en cas d'échec. Dans ce cas, la variable [ErreurDétectée](../WDLang1/3087001.md) est à <u><u><u><u>Vrai</u></u></u></u>. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Langue> : (Entier ou constante de type Entier)`**



- Langue à utiliser pour la lecture du texte. Si la langue spécifiée n'est pas supportée par le moteur de synthèse vocale, la fonction retournera <u><u><u><u>Faux</u></u></u></u> et  la première langue disponible parmi la séquence suivante sera utilisée : 

	- Langue de l'application (langue par défaut ou définie par la fonction [Nation](../WDLang1/3054014.md)). 

	- Langue de l'appareil

	- Anglais

	- Langue par défaut du moteur de synthèse vocale.




- 0 pour utiliser la langue courante de l'application.

**`<Vitesse> : (Entier optionnel)`**

Vitesse d'énonciation du texte, comprise entre 0 (très lent) et 10 (très rapide). Si ce paramètre n'est pas spécifié, sa valeur sera de 5 (vitesse moyenne).

![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Ce paramètre n'est pas pris en compte. La vitesse d'énonciation est réglable uniquement dans les paramètres du mobile utilisé (option "Paramètres .. Heure et langue .. Voix").


<a name="SYNTAXE2"></a>

### Initialiser l'ensemble des caractéristiques de la voix

`<Résultat> = SynthèseVocaleInitialise(<Voix> [, <Vitesse>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'initialisation a fonctionné, 

- <u><u><u><u>Faux</u></u></u></u> en cas d'échec. Dans ce cas, la variable [ErreurDétectée](../WDLang1/3087001.md) est à <u><u><u><u>Vrai</u></u></u></u>. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Voix> : (Variable de type VoixDeSynthèse)`**

Nom de la variable de type [VoixDeSynthèse](../WDLang3/1000025986.md) à utiliser pour la lecture du texte. 

**`<Vitesse> : (Entier optionnel)`**

Vitesse d'énonciation du texte, comprise entre 0 (très lent) et 10 (très rapide). Si ce paramètre n'est pas spécifié, sa valeur sera de 5 (vitesse moyenne).

![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Ce paramètre n'est pas pris en compte. La vitesse d'énonciation est réglable uniquement dans les paramètres du mobile utilisé (option "Paramètres .. Heure et langue .. Voix").

## Remarques


### Fonctionnement
<a name="fonctionnement_ELTPARAGRAPHE000362"></a>

- Pour synthétiser une chaîne de caractères ou un fichier texte, utilisez respectivement les fonctions [SynthèseVocaleLitTexte](../WDLang3/1000019805.md) et [SynthèseVocaleLitFichier](../WDLang3/1000020172.md).

- ![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Par défaut, seul le moteur de synthèse vocale de la langue par défaut du système est installé. Pour installer un autre moteur de synthèse vocale, utilisez les paramètres de l'appareil (option "Paramètres .. Heure et langue .. Langue").



<a name="NOTE0_2"></a>
![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) **Fonctionnalité d'application requise**
L'utilisation de cette fonction entraîne la déclaration d'une fonctionnalité d'application dans l'assistant de génération de l'application.
Fonctionnalité requise : Microphone : Cette fonctionnalité permet aux applications d'effectuer des enregistrements audio.


## TâcheRAZ (Fonction)

***En anglais : TaskReset***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Réinitialise la [structure Tâche](../WDLang3/1000012010.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ouverture d'une session Lotus Notes
IDSession est un entier
IDSession = EmailOuvreSessionNotes("MotDePasse", "ServeurMars", "mail\julie.nsf", ...
				"C:\Users\Julie\AppData\Local\Lotus\Notes\Data\user.id")

// Initialisation des variables de la structure Tâche
TâcheRAZ()
// Renseignements des variables pour le nouveau rendez-vous
mTâche.Objet = "Nouvelle tâche"
mTâche.DateDébut = "20070125100000"
mTâche.NbParticipant = 3
mTâche.Participant[1] = "Romain"
mTâche.Participant[2] = "Lucas"
mTâche.Participant[3] = "Stéphane"
// Ajout de la tâche
SI TâcheAjoute(IDSession) = Vrai ALORS
	Info("Tâche ajoutée")
SINON
	Erreur("Erreur lors de l'ajout", ErreurInfo())
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`TâcheRAZ()`

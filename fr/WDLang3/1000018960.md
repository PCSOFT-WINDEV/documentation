
## SaaSDéconnecte (Fonction)

***En anglais : SaaSDisconnect***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Déconnecte l'utilisateur d'un site SaaS auprès du webservice SaaS gestionnaire du site.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI SaaSConnecte("login", "motdepasse") = Vrai ALORS
	SI SaaSDéconnecte() = Vrai ALORS
		Info("Déconnexion réussie")
	FIN
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = SaaSDéconnecte()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la déconnexion a été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Si la fonction **SaaSDéconnecte** est utilisée plusieurs fois (sans reconnexion par la fonction [SaasConnecte](../WDLang3/1000018959.md)), &lt;Résultat&gt; est également à <u><u><u><u>Faux</u></u></u></u>.

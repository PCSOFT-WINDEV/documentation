
## RendezVousAffiche (Fonction)

***En anglais : AppointmentDisplay***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Affiche un rendez-vous dans l'application native de gestion des rendez-vous de l'appareil mobile (Android ou iOS).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI PAS RendezVousAffiche(rdv) ALORS
	Erreur("Impossible d'afficher le rendez-vous.", ErreurInfo())
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = RendezVousAffiche(<Rendez-vous>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la description du rendez-vous a pu être affichée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Rendez-vous> : (Variable de type RendezVous)`**

Nom de la variable de type [RendezVous](../WDLang1/1000019244.md) correspondant au rendez-vous dont la description doit être affichée. Ce rendez-vous doit exister et appartenir à un calendrier de l'appareil (obtenu par la fonction [RendezVousListe](../WDLang3/1000020690.md)).



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Cas particuliers
<a name="cas_particuliers_ELTPARAGRAPHE000058"></a>

- La fonction **RendezVousAffiche** rend la main à l'application immédiatement après l'affichage de la description du rendez-vous dans l'application native.

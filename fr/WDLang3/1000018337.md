


## grParamètre (Fonction)

***En anglais : grParameter***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Récupère ou modifie un paramètre d'un graphe.

**Remarques** : 

- Cette fonction propose toutes les options de la fonction [grParamètre3DS](../WDLang3/3042063.md).

- Cette fonction ne doit pas être utilisée sur des champs Graphe présents dans l'éditeur d'états.






![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Les graphes](../WDChamp/3042060.md). 










<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Désactive l'optimisation des graduations sur l'axe X de GRF_Graphe1
grParamètre(GRF_Graphe1, grGraduationsOptimiséesX, Faux)
```

## Remarques


### Paramètres d'un graphe
<a name="parametres_graphe_ELTPARAGRAPHE000622"></a>Les différents paramètres d'un graphe sont les suivants :

| Constante | Effet | Valeur du paramètre |
| --- | --- | --- |
| *gr3DSPropAnimation* | Type d'animation jouée lors de l'affichage du graphe | Entier<br>Les valeurs possibles sont :<br><br>- 0 : aucune animation<br><br>- 1 : zoom avant<br><br>- 2 : effet de rotation<br><br>- 4 : construction progressive des barres / secteurs<br><br>- 8 : affichage progressif des couleurs<br><br><br>Ces valeurs peuvent être additionnées.<br>La valeur par défaut est 15. |
| *gr3DSPropAutoZoom* | Zoom automatique du graphe | Booléen<br><br>- <u><u><u><u>Vrai</u></u></u></u> (par défaut) si le zoom doit être automatiquement calculé,<br><br>- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.<br><br><br>Si le zoom est automatiquement calculé, la valeur du paramètre *gr3DSPropDistance* n'est pas prise en compte. |
| *gr3DSPropCouleurPlanX* | Couleur du plan vertical (axe à droite ou à gauche) | Entier<br><br><br>- Composante RVB de la couleur (obtenue avec la fonction [RVB](../WDLang1/3029012.md))<br><br>- [Couleur prédéfinie](../WDLang5/3010002.md).<br><br><br>Pour ne pas dessiner cet axe, utilisez la constante *Transparent*.<br>Cette constante est prise en compte uniquement sur les graphes de type histogramme ou courbe.<br>La valeur par défaut est *GrisClair*. |
| *gr3DSPropCouleurPlanY* | Couleur du plan horizontal (axe au-dessus ou au-dessous) | Entier<br><br><br>- Composante RVB de la couleur (obtenue avec la fonction [RVB](../WDLang1/3029012.md))<br><br>- [Couleur prédéfinie](../WDLang5/3010002.md).<br><br><br>Pour ne pas dessiner cet axe, utilisez la constante *Transparent*.<br>Cette constante est prise en compte uniquement sur les graphes de type histogramme ou courbe.<br>La valeur par défaut est *GrisClair*. |
| *gr3DSPropCouleurPlanZ* | Couleur du plan du fond | Entier<br><br><br>- Composante RVB de la couleur (obtenue avec la fonction [RVB](../WDLang1/3029012.md))<br><br>- [Couleur prédéfinie](../WDLang5/3010002.md).<br><br><br>Pour ne pas dessiner cet axe, utilisez la constante *Transparent*.<br>Cette constante est prise en compte uniquement sur les graphes de type histogramme ou courbe.<br>La valeur par défaut est *GrisClair*. |
| *gr3DSPropDistance* | Zoom du graphe | Réel<br>Les valeurs possibles sont comprises entre 0 et l'infini.<br>Cette constante est prise en compte uniquement si la constante *gr3DSPropAutoZoom* est à <u><u><u><u>Faux</u></u></u></u>.<br>La valeur par défaut est 189. |
| *gr3DSPropElevation* | Angle de rotation (en degrés) du graphe autour de l'axe des ordonnées (axe Y) | Réel<br>Les valeurs possibles sont comprises entre 0 et 360.<br>La valeur 0 permet de voir le graphe de face.<br>La valeur 90 permet de voir le graphe du dessus.<br>Par défaut, l'angle de rotation optimal est automatiquement calculé. |
| *gr3DSPropLégendeSuperposée* | Superposition de la légende et du titre du graphe | Booléen.<br><br>- <u><u><u><u>Vrai</u></u></u></u> si la légende et le titre sont superposés au graphe,<br><br>- <u><u><u><u>Faux</u></u></u></u> (par défaut) dans le cas contraire.<br><br><br> |
| *gr3DSPropLumièreX* | Abscisse (X) de la lumière éclairant le graphe. | Réel<br>Les valeurs possibles sont comprises entre 0 et l'infini.<br>La valeur par défaut est 78. |
| *gr3DSPropLumièreY* | Ordonnée (Y) de la lumière éclairant le graphe. | Réel<br>Les valeurs possibles sont comprises entre 0 et l'infini.<br>La valeur par défaut est 80. |
| *gr3DSPropRotation* | Angle de rotation (en degrés) du graphe autour de l'axe des abscisses (axe X). | Réel<br>Les valeurs possibles sont comprises entre 0 et 360.<br>La valeur 0 permet de voir le graphe de face.<br>La valeur 90 permet de voir le graphe de droite.<br>Par défaut, l'angle de rotation optimal est automatiquement calculé. |
| *gr3DSPropTransparence* | Taux de transparence (en pourcentage) des barres ou des secteurs du graphe | Entier<br>Les valeurs possibles sont comprises entre 0 (opaque) et 100 (transparent/invisible).<br>La valeur par défaut est 27%. |
| *grAlgoEntonnoir* | Mode de représentation des données d'un graphe de type Entonnoir. | Constante de type Entier :<br><br>- *grEntonnoirAire* : La valeur est symbolisée par l'aire de la portion de l'entonnoir.<br><br>- *grEntonnoirHauteur* : La valeur est symbolisée par la hauteur de la portion de l'entonnoir.<br><br><br> |
| *grAlgoPyramide* | Mode de représentation des données d'un graphe de type Pyramide. | Constante de type Entier :<br><br>- *grPyramideHauteur* : La valeur est symbolisée par la hauteur de la portion de la pyramide.<br><br>- *grPyramideVolume* : La valeur est symbolisée par le volume de la portion de la pyramide.<br><br><br><br><br> |
| *grAscenseurZoomToujoursVisible* | Gestion des ascenseurs lors du zoom. | Booléen.<br><br>- <u><u><u><u>Vrai</u></u></u></u> (par défaut) si les ascenseurs du zoom doivent être toujours visibles, même lorsque le graphe est affiché à 100%. <br><br>- <u><u><u><u>Faux</u></u></u></u> si les ascenseurs de zoom doivent devenir invisibles lorsque le graphe est affiché à 100%. <br><br><br> |

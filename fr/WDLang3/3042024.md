


## grEtiquetteSérie (Fonction)

***En anglais : grSeriesLabel***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Initialise l'étiquette d'une série de données d'un graphe.

Exemple d'étiquette de série :
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=EtiquetteSerie.gif)
Pour afficher l'étiquette, utilisez la fonction [grLégende](../WDLang3/3042047.md).

**Rappel** : Par défaut, un graphe n'a aucune étiquette. Pour plus de détails sur la gestion des légendes, étiquettes, ... consultez [Gestion des légendes dans les graphes](../WDChamp/3042069.md). 

![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Les graphes](../WDChamp/3042060.md). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Initialisation de l'étiquette "Ventes" de la série de données 2 dans le graphe "MonGraphe"
grEtiquetteSérie("MonGraphe", 2, "Ventes")
```
<a name="Exemple2"></a>

```wl
// Initialisation de l'étiquette "Ventes" de la série de données 2 dans le champ GRF_MonGraphe
grEtiquetteSérie(GRF_MonGraphe, 2, "Ventes")
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`grEtiquetteSérie(<Nom du graphe> , <Numéro de la série> , <Étiquette>)`
---

**`<Nom du graphe> : (Nom de champ ou chaîne de caractères)`**

Nom du graphe à manipuler. Ce nom correspond :

- soit au nom du graphe défini par programmation avec la fonction [grCrée](../WDLang3/3042005.md).

- soit au nom du champ Graphe présent dans l'éditeur de fenêtres, de pages ou d'états.




**`<Numéro de la série> : (Entier)`**

Numéro de la série pour laquelle une étiquette doit être initialisée.

**`<Étiquette> : (Chaîne de caractères)`**

Libellé de l'étiquette à initialiser.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Etiquettes de série
<a name="etiquettes_serie_ELTPARAGRAPHE000139"></a>L'ensemble des étiquettes de séries correspond à la légende du graphe.
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=EtiquetteSerie.gif)
Les étiquettes de série ne sont pas utilisables avec les graphes de type secteur.

La définition des étiquettes de séries est réalisée avec la fonction **grEtiquetteSérie**. 

Les données définies avec la fonction **grEtiquetteSérie** seront prises en compte lors du prochain appel à la fonction [grDessine](../WDLang3/3042026.md).

Il est possible de modifier la police de la légende grâce à la fonction [grPoliceLégende](../WDLang3/3042037.md).  

La fonction [grLégende](../WDLang3/3042047.md) permet d'indiquer la présence et la position de la légende (c'est-à-dire de l'ensemble des étiquettes de série).

**Remarque** : Pour définir des étiquettes de série dans des graphes de type "Nuage de points", il est conseillé d'utiliser la fonction [grNuageEtiquetteSérie](../WDLang3/3042038.md).

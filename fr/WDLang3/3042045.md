


## grNuageCouleurSérie (Fonction)

***En anglais : grScatterSeriesColor***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Initialise la couleur d'une série pour un graphe de type "Nuage de points". 

**Rappel** : Par défaut, les couleurs des séries (ou des parts de secteur) sont sélectionnées selon un ordre prédéfini parmi un éventail de 13 couleurs.



![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Les graphes](../WDChamp/3042060.md). 




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ajoute une donnée dans le graphe "MonGraphe" de type "Nuage de points".
// Cette valeur (10, 20) est ajoutée à la fin de la première série.
grNuageAjouteDonnéeXY("MonGraphe", 1, 10, 20)
// La série 1 du graphe "MonGraphe" est de couleur jaune
grNuageCouleurSérie("MonGraphe", 1, JauneClair)
```
<a name="Exemple2"></a>

```wl
// Ajoute une donnée dans le champ GRF_MonGraphe de type "Nuage de points".
// Cette valeur (20, 20) est ajoutée à la fin de la première série.
grNuageAjouteDonnéeXY(GRF_MonGraphe, 1, 20, 20)
// La série 1 du graphe GRF_MonGraphe est de couleur verte
grNuageCouleurSérie(GRF_MonGraphe, 1, VertClair)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`grNuageCouleurSérie(<Nom du graphe> , <Numéro de la série> , <Couleur>)`
---

**`<Nom du graphe> : (Nom de champ ou chaîne de caractères)`**

Nom du graphe à manipuler. Ce nom peut correspondre :

- au nom du graphe défini par programmation avec la fonction [grCrée](../WDLang3/3042005.md).

- au nom du champ Graphe présent dans l'éditeur de fenêtres, de pages ou d'états.




**`<Numéro de la série> : (Entier)`**

Numéro de la série à manipuler.

**`<Couleur> : (Entier ou constante)`**

Couleur de la série spécifiée. Cette couleur peut correspondre à :

- une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)),

- une couleur TSL (obtenue avec la fonction [TSL](../WDLang1/3029057.md)),

- [une couleur prédéfinie du WLangage](../WDLang5/3010002.md).  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Les données définies avec la fonction **grNuageCouleurSérie** seront prises en compte lors du prochain appel à la fonction [grDessine](../WDLang3/3042026.md).

- Pour obtenir des couleurs dégradées, utilisez la fonction [grDégradé](../WDLang3/3042029.md).

- La fonction **grNuageCouleurSérie** est équivalente à la fonction [grCouleurSérie](../WDLang3/3042013.md). 
	La syntaxe de la fonction [grCouleurSérie](../WDLang3/3042013.md) utilisée avec les graphes de type "Nuage de points" est la suivante :
	
	```txt
	grCouleurSérie(<Nom du graphe>, 2*<Indice>-1, <Couleur>)
	```

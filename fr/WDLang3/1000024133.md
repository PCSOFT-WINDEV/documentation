


## &lt;Graphe&gt;.SMNAngleDépart (Fonction)

***En anglais : &lt;Chart&gt;.SunburstStartAngle***

<DIV class="specObsolete">
	<IMG style="float:left;margin-right:10px;" src="https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=warning_big.png"><B>Avertissement</B><BR>
	A partir de la version <B>27</B>,  cette fonction est conservée par compatibilité (cette fonction sera encore supportée dans la prochaine version). Il est conseillé d'utiliser la fonction [&lt;Graphe&gt;.AngleDépart](../WDLang3/1410088091.md).

<a name="Utilisation"></a>
<a name="description"></a>
Indique l'angle de départ de la première part dans un graphe de type Secteur multiniveau.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Augmentation de l'angle de départ du graphe "GRF_MonGraphe"
GRF_MonGraphe.SMNAngleDépart(45)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Champ Graphe>.SMNAngleDépart(<Angle>)`
---

**`<Champ Graphe> : (Nom de champ)`**

Nom du champ Graphe à manipuler (présent dans l'éditeur de fenêtres, de pages ou d'états).



**`<Angle> : (Entier)`**

Angle de départ de la première part (exprimé en degrés dans le sens des aiguilles d'une montre à partir du point haut du disque). 
Par défaut, l'angle de départ dans un graphe de type Secteur multiniveau est égal à 0.

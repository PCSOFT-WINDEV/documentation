
## WiFiListeCarteRéseau (Fonction)

***En anglais : WiFiListNetworkCard***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la liste des cartes réseaux Wi-Fi actuellement disponibles sur le poste en cours.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
sMaListe est une chaîne
nId est un entier  
sDescription est une chaîne

sMaListe = WiFiListeCarteRéseau()
POUR TOUTE CHAINE RéseauWifi de sMaListe SEPAREE PAR RC
	nId = Val(ExtraitChaîne(RéseauWifi, 1 , TAB))
	sDescription = ExtraitChaîne(RéseauWifi, 2, TAB)
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = WiFiListeCarteRéseau()`
---

**`<Résultat> : (Variable)`**

Liste des cartes réseau Wi-Fi séparées par RC (Retour Chariot). Pour chaque carte réseau, une série d'informations est retournée au format suivant : 

```txt
<Identifiant> + TAB + <Description>
```
où : 

- &lt;Identifiant&gt; : Identifiant de la carte réseau Wi-Fi. 

- &lt;Description&gt; : Nom en clair de la carte réseau Wi-Fi.

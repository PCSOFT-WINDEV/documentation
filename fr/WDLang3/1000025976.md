
## DoubleAuthVérifieCode (Fonction)

***En anglais : TwoFactorAuthenticationCheckCode***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Vérifie le code donné par une application d'authentification.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI DoubleAuthVérifieCode(nCode, sCléTOTP) = Faux ALORS
	Erreur("Le code ne correspond pas.")
	RETOUR
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = DoubleAuthVérifieCode(<Code> , <Clé TOTP>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le code est correct,

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Code> : (Entier ou chaîne de caractères)`**

Code à 6 chiffres donné par l'application d'authentification.



**`<Clé TOTP> : (Chaîne de caractères)`**

Chaîne ANSI contenant la clé TOTP de l'utilisateur à authentifier.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Durée de validité
<a name="duree_validite_ELTPARAGRAPHE000051"></a>

- La durée de validité du code est légèrement supérieure à celle montrée par l'application d'authentification. Par exemple, lorsque Google Authenticator affiche un nouveau code, le précédent reste encore valide 1 minute.

- La fonction **DoubleAuthVérifieCode** vérifie les codes générés 1 minute avant et 1 minute après l'instant où la demande de vérification a été effectuée.




## &lt;Graphe&gt;.SurfaceSupprimeCouleurAltitude (Fonction)

***En anglais : &lt;Chart&gt;.SurfaceDeleteAltitudeColor***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime les couleurs d'altitude spécifiées par la fonction [&lt;Graphe&gt;.SurfaceCouleurAltitude](../WDLang3/1000024144.md). 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
GRF_Graphe1.SurfaceSupprimeCouleurAltitude()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Champ Graphe>.SurfaceSupprimeCouleurAltitude([<Altitude>])`
---

**`<Champ Graphe> : (Nom de champ)`**

Nom du champ Graphe à manipuler (présent dans l'éditeur de fenêtres ou d'états).

**`<Altitude> : (Réel optionnel)`**

Altitude de la couleur qui doit être supprimée. Si ce paramètre n'est pas spécifié, toutes les couleurs d'altitude sont supprimées : le graphe utilise alors la couleur de la première série.

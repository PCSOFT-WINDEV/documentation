


## &lt;Variable upnpPériphérique&gt;.MédiaPause (Fonction)

***En anglais : &lt;upnpDevice variable&gt;.MediaPause***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Met en pause la lecture du média en cours sur le périphérique UPnP.

**Attention** : Pour utiliser cette fonction, le périphérique doit posséder un service "AVTransport".
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
tabPériphérique est un tableau de upnpPériphériques 
tabPériphérique = UPNPListePériphérique(...
		"urn:schemas-upnp-org:device:MediaRenderer:1", 5)
POUR TOUT Périphérique de tabPériphérique
	Périphérique.MédiaPause()
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Périphérique UPnP>.MédiaPause()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la lecture a été mise en pause, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Périphérique UPnP> : (Variable de type upnpPériphérique)`**

Nom de la variable [upnpPériphérique](../WDLang3/1000020903.md) correspondant au périphérique UPnP à manipuler.

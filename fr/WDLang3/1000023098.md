


## Fonctions MQTT (syntaxe préfixée)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions permettant de gérer les clients MQTT (syntaxe préfixée) sont les suivantes :



|   |   |
| --- | --- |
| [&lt;Variable mqttSession&gt;.Abonne](../WDLang3/1000023083.md) | Permet de s'abonner à une notification MQTT. |
| [&lt;Variable mqttSession&gt;.Connecte](../WDLang3/1000023084.md) | Permet de se connecter à un serveur MQTT (également appelé "Broker"). |
| [&lt;Variable mqttSession&gt;.Déconnecte](../WDLang3/1000023085.md) | Permet de se déconnecter d'un serveur MQTT (également appelé "Broker"). |
| [&lt;Variable mqttSession&gt;.Désabonne](../WDLang3/1000023086.md) | Permet de se désabonner d'une notification MQTT. |
| [&lt;Variable mqttSession&gt;.DésabonneTout](../WDLang3/1000023090.md) | Permet de se désabonner de tous les abonnements préalablement souscrits. |
| [&lt;Variable mqttSession&gt;.Publie](../WDLang3/1000023091.md) | Publie un message sur le broker MQTT. |

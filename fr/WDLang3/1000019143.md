
## SaaSAdminModifieConnexionClient (Fonction)

***En anglais : SaaSAdminModifyClientConnection***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie la connexion à la base de données du client pour un site SaaS.

**Remarques** : 

- L'utilisation de cette fonction nécessite d'avoir ouvert une session d'administration SaaS au préalable en utilisant la fonction [SaaSAdminConnecte](../WDLang3/1000019098.md).

- L'utilisation de cette fonction est réservée à l'Administrateur SaaS. Un gestionnaire de compte client ne peut pas utiliser cette fonction. 



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// La société Hybrid est déjà cliente du site SaaS Forum
S est un saasSite dynamique 
C est un saasClient dynamique 
S = SaaSAdminChercheSite("Forum")
C = SaaSAdminChercheClient("Hybrid")

// Association avec une nouvelle base de données
NouvelleBase est une Connexion 
NouvelleBase = SaaSAdminConnexionClient(S, C)
NouvelleBase.BaseDeDonnées = "Forum_Hybrid_2010"

// Modifie les paramètres de connexion à la base cliente
SaaSAdminModifieConnexionClient(S, C, NouvelleBase)

// Création de la nouvelle base cliente à partir de la base de référence
SaaSAdminCrééBaseClient(S, C)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = SaaSAdminModifieConnexionClient(<Site SaaS> , <Compte Client> , <Connexion>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la modification a été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Site SaaS> : (Variable de type saasSite ou Chaîne de caractères)`**

Site SaaS concerné par la modification.

**`<Compte Client> : (Variable de type saasClient ou Chaîne de caractères)`**

Compte client SaaS (ou nom du compte client) à modifier.

**`<Connexion> : (Variable de type Connexion)`**

Nom de la variable de type [Connexion](../WDLang4/1514073.md) contenant les nouveaux paramètres de connexion à la base de données.




## grNiveauTemporelMasque (Fonction)

***En anglais : grTimeLevelMask***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Définit le masque d'affichage d'un niveau de l'échelle temporelle d'un graphe. 

Remarque : Cette fonction peut être utilisée avec la fonction [grNiveauTemporelMax](../WDLang3/1000023016.md) qui permet de définir le nombre de niveaux à afficher dans l'échelle temporelle d'un graphe. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affiche les heures sous la forme 10h
grNiveauTemporelMasque(GRF_MonGraphe, grntHeure, "HHh")
```

```wl
// Définition de l'échelle temporelle pour afficher plusieurs valeur par mois
// mais en affichant uniquement les mois dans l'échelle 
// (les jours ne sont pas affichés)
grNiveauTemporelMasque(GRF_MonGraphe, grntMois, "Mmm")
grNiveauTemporelMax(GRF_MonGraphe, 1, grntMois)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`grNiveauTemporelMasque(<Nom du graphe> , <Niveau temporel> , <Masque>)`
---

**`<Nom du graphe> : (Nom de champ ou chaîne de caractères)`**

Nom du graphe à manipuler. Ce nom correspond :

- soit au nom du graphe défini par programmation avec la fonction [grCrée](../WDLang3/3042005.md).

- soit au nom du champ Graphe présent dans l'éditeur de fenêtres ou de pages.




**`<Niveau temporel> : (Constante de type Entier)`**

Niveau temporel sur lequel le masque doit être fixé : 



|   |   |
| --- | --- |
| *grntAnnée* | Le masque doit être utilisé sur les années. |
| *grntHeure* | Le masque doit être utilisé sur les heures. |
| *grntJour* | Le masque doit être utilisé sur les jours. |
| *grntMilliseconde* | Le masque doit être utilisé sur les millisecondes. |
| *grntMinute* | Le masque doit être utilisé sur les minutes. |
| *grntMois* | Le masque doit être utilisé sur les mois. |
| *grntSeconde* | Le masque doit être utilisé sur les secondes. |



**`<Masque> : (Chaîne de caractères)`**

Masque à utiliser. 

Attention :  Le masque s'applique uniquement à un seul niveau temporel. Il est donc recommandé de le spécifier en utilisant uniquement les données de ce niveau (par exemple, "HH" pour le niveau heure ou "AAAA" pour le niveau année).



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Combinaison de masques
<a name="combinaison_masques_ELTPARAGRAPHE000112"></a>Il est possible de combiner les masques pour utiliser un format spécifique. Par exemple : 

```wl
// Affiche les mois sous la forme mois + année
grNiveauTemporelMasque(GRF_MonGraphe, grntMois, "Mmm AAAA")
```

<a name="NOTE0_2"></a>


### Masques disponibles
<a name="masques_disponibles_ELTPARAGRAPHE000127"></a>Les masques disponibles sont les suivants : 

- AAAA représente l'année sur 4 positions.

- AA représente la date sur 2 positions.

- M représente le mois (le zéro n'est pas affiché devant le numéro du mois).

- MM représente le mois sur 2 chiffres.

- MMM représente le mois en lettres sur 3 caractères (par exemple jan). La casse utilisée dépend des options linguistiques du projet. 

- Mmm représente le mois en lettres sur 3 caractères avec la première lettre forcée en majuscule (par exemple Jan).

- mmm représente le mois en lettres sur 3 caractères avec la première lettre forcée en minuscule (par exemple jan).

- MMMM représente le mois en lettres complet (par exemple janvier). La casse utilisée dépend des options linguistiques du projet.

- Mmmm représente le mois en lettres complet avec la première lettre forcée en majuscule (par exemple Janvier).

- mmmm représente le mois en lettres complet avec la première lettre forcée en minuscule (par exemple janvier).

- J représente le jour (le zéro n'est pas affiché devant le numéro du jour).

- JJ représente le jour sur 2 chiffres.

- JJJ représente le jour en lettres sur 3 caractères (par exemple lun). La casse utilisée dépend des options linguistiques du projet.

- Jjj représente le jour en lettres sur 3 caractères avec la première lettre forcée en majuscule (par exemple Lun).

- jjj représente le jour en lettres sur 3 caractères avec la première lettre forcée en minuscule (par exemple lun).

- JJJJ représente le jour en lettres complet (par exemple lundi). La casse utilisée dépend des options linguistiques du projet.

- Jjjj représente le jour en lettres complet avec la première lettre forcée en majuscule (par exemple Lundi).

- jjjj représente le jour en lettres complet avec la première lettre forcée en minuscule (par exemple lundi).

- L représente la première lettre du jour (par exemple L,M,M,J,V,S,D).

- HH représente le nombre d'heures.

- mm représente le nombre de minutes.

- SS représente le nombre de secondes.

- CC représente le nombre de centièmes de seconde.

- AP représente "AM" ou "PM" (si "AP" est présent dans le format, l'heure est affichée sur 12 heures).

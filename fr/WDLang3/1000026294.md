


## &lt;zipArchive&gt;.SélectionFichier (Fonction)

***En anglais : &lt;zipArchive&gt;.SelectFile***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Définit une sélection des fichiers contenus dans l'archive. Cette sélection de fichiers sera traitéé par les fonctions WLangage de gestion des archives. 

Il est ainsi possible par exemple de sélectionner uniquement les fichiers ".doc" d'une archive. Dans ce cas, toutes les fonctions WLangage permettant de manipuler la totalité de l'archive (fonction [&lt;Variable zipArchive&gt;.ExtraitTout](../WDLang3/1000019085.md) par exemple) n'agiront que sur les fichiers sélectionnés (fichiers ".doc" dans notre exemple).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
Archive est un zipArchive
// Ouverture d'une archive
SI Archive.Ouvre("C:\Archives\Archive.zip") = 0 ALORS
	// On veut extraire uniquement les documents
	sAncienneSélection est une chaîne = Archive.SélectionFichier("*.doc")
	// Extraction de tous les documents de l'archive 
	Archive.ExtraitTout(zipDisque)
	// Restitution de la sélection précédente
	Archive.SélectionFichier(sAncienneSélection)
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Archive>.SélectionFicher([<Sélection>])`
---

**`<Résultat> : (Chaîne de caractères)`**

Ancienne sélection appliquée sur l'archive.

**`<Archive> : (Variable de type zipArchive)`**

Nom d'une variable de type [zipArchive](../WDLang3/1000018679.md) correspondant à l'archive à manipuler.

**`<Sélection> : (Chaîne de caractères optionnelle)`**

Nom générique des fichiers à sélectionner. 
Les caractères génériques (\* et ?) sont autorisés. 
Si ce paramètre n'est pas précisé ou vaut chaîne vide (""), la sélection est supprimée et tous les fichiers seront traités.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Principe
<a name="principe_ELTPARAGRAPHE000239"></a>Après l'appel de la fonction **&lt;zipArchive&gt;.SélectionFichier**, les prochaines fonctions de gestion des archives manipulant l'ensemble des fichiers de l'archive respecteront la sélection : ces fonctions manipuleront uniquement la sélection de fichiers spécifiée. 

Pour manipuler à nouveau l'ensemble des fichiers de l'archive, il suffit d'appeler la fonction **&lt;zipArchive&gt;.SélectionFichier** sans spécifier le paramètre &lt;Sélection&gt;.




## RéseauMobileInfoConnexion (Fonction)

***En anglais : MobileNetworkConnectionInfo***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie les informations demandées concernant la connexion actuelle aux données sur le réseau mobile.

**Remarque** : Cette fonction n'est pas utilisable dans le simulateur et renverra 0 dans ce cas. 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Activation du Wi-Fi si l'intensité du signal est suffisante pour l'accès aux données 
// via le réseau mobile
SI Val(RéseauMobileInfoConnexion(réseauSignalGSM)) <= 20 _ET_ WiFiEtat() <> wifiActif  ALORS
	SI OuiNon("L'intensité du signal GSM est insuffisante. Activer le Wi-Fi ?") ALORS
		WiFiActive()
	FIN
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = RéseauMobileInfoConnexion(<Type d'information>)`
---

**`<Résultat> : (Chaîne de caractères ou constante de type Entier)`**



- Information demandée. 

- Si plusieurs types d'informations sont demandés, chaque type d'information est séparé par un Retour Chariot (RC). 

- L'ordre dans lequel sont renvoyées les informations (si elles sont demandées) est le suivant : 

	1. &lt;Type de réseau&gt;

	2. &lt;Opérateur&gt;

	3. &lt;Signal GSM&gt;

	4. &lt;Itinérance&gt;

	5. &lt;Mode avion&gt;







**`<Type d'information> : (Constante de type Entier)`**

Information demandée : 


|   |   |
| --- | --- |
| *réseauItinérance* | Mode d'itinérance des données.<br><br>**Si cette information est demandée, &lt;Résultat&gt; vaut** :<br><br>- "1" si la connexion actuelle utilise l'itinérance des données (roaming)<br><br>- "0" dans le cas contraire.<br><br><br> |
| *réseauModeAvion* | Gestion du mode avion.<br><br>**Si cette information est demandée, &lt;Résultat&gt; vaut** :<br><br>- "1" si l'appareil est en mode avion.<br><br>- "0" dans le cas contraire.<br><br><br><br><br>![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Cette constante n'est pas disponible. |
| *réseauOpérateur* | Nom de l'opérateur réseau utilisé.<br><br>Selon le type de réseau ou si le mode avion est activé, cette information peut ne pas être renseignée.<br><br>En l'absence de carte SIM, cette information peut correspondre à un nom d'opérateur si le téléphone est "Sim-locké". |
| *réseauSignalGSM* | Intensité du signal GSM comprise entre "0" (pas de signal) et "100" (signal optimal).<br><br> |
| *réseauType* | Type de réseau utilisé pour l'accès aux données sur le réseau mobile.<br><br>Mode d'itinérance des données.<br><br>**Si cette information est demandée, &lt;Résultat&gt; vaut une constante de type entier** :<br><br>- *réseau2G* : Réseau 2G<br><br>- *réseauEdge* : Réseau Edge<br><br>- *réseau3G* : Réseau 3G<br><br>- *réseau4G* : Réseau 4G<br><br>- *réseau5G* : Réseau 5G<br><br>- *réseauInconnu* : Réseau inconnu (ou absence de carte SIM). <br><br><br> |

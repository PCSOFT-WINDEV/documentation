


## InternetConnecté (Fonction)

***En anglais : InternetConnected***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Détermine si une connexion Internet est active sur le poste en cours. 






<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connexion Internet active ?
ResConnexion = InternetConnecté()
SI ResConnexion = Vrai ALORS
	Info("Une connexion Internet est active")
SINON
	Info("Aucune connexion Internet n'est active")
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Savoir si une connexion Internet est active

`<Résultat> = InternetConnecté()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si une connexion Internet est active sur le poste en cours, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.

## Remarques
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) 

### Firewall
<a name="firewall_ELTPARAGRAPHE000135"></a>La fonction **InternetConnecté** utilise une requête **ping** afin de tester l'accès à Internet. Il faut donc que les paramètres de sécurité du réseau autorise le ping pour que le résultat de la fonction soit pertinent.

Remarque : Le ping est réalisé sur le DNS de Google (6 secondes, 1 essai). Le code équivalent est : 

```wl
Ping("8.8.8.8", 6000, 1)
```

## Environnement de développement Mobile
<a name="environnement_developpement_mobile_ELTTEXTE000105"></a>
WINDEV Mobile est un environnement de développement sous poste PC qui permet de développer : 

- ![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) des applications en mode Universal Windows 10 App.




**Remarques** : 

- Les applications générées fonctionnent uniquement sous des terminaux mobiles (équipés de processeur ARM, XScale, Samsung et compatible ARM). Ces applications ne fonctionnent pas sous des postes PC.

- Le développement d'applications Windows Mobile n'est plus disponible à partir de la version 27. Windows Mobile, Windows CE et Windows Embedded ne sont plus supportés par Microsoft, mais sont encore utilisés dans le domaine industriel. 
	Pour continuer à développer pour Windows Mobile, vous devez utiliser une version 26. Votre version 28 donne accès à la version 26 qui peut être téléchargée [ici](http://www.pcsoft.fr/dl).

## Fonctions WLangage
<a name="fonctions_wlangage_ELTTEXTE000138"></a>
La plupart des fonctions WLangage présentes dans WINDEV standard sont également disponibles dans WINDEV Mobile. Il s'agit des fonctions communes aux deux produits.

Seules les fonctions spécifiques à Windows ne sont pas disponibles sous WINDEV Mobile.

Le schéma suivant présente les fonctions du WLangage utilisables selon le type d'application développée :

## Base de données
<a name="base_donnees_ELTTEXTE000162"></a>
Les données manipulées dans une application WINDEV Mobile peuvent provenir :

- ![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) d'une base de données HFSQL Classic ou HFSQL Client/Serveur.

- de fichiers ".INI".

- de fichiers texte (au format ANSI ou UNICODE).

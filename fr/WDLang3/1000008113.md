
## GroupeModifie (Fonction)

***En anglais : GroupModify***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie le groupe de contacts en cours dans le carnet d'adresses d'une messagerie Lotus Notes ou Outlook.

Lors de la lecture d'un groupe de contacts, les données de ce groupe sont mis à jour dans la [structure Groupe](../WDLang3/1000008120.md). Pour modifier le groupe en cours, il suffit de modifier les valeurs de la [structure Groupe](../WDLang3/1000008120.md) et de valider les modifications avec la fonction **GroupeModifie**.

**Attention** : Cette fonction **ne permet pas** de modifier le groupe de contacts en cours dans le carnet d'adresses d'une messagerie Outlook Express.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ouverture d'une session Lotus Notes
IDSession est un entier
IDSession = EmailOuvreSessionNotes("MotDePasse", "ServeurMars", ...
	"mail\julie.nsf", "C:\Users\Julie\AppData\Local\Lotus\Notes\Data\user.id")

// Lecture du premier groupe de contacts
GroupePremier(IDSession)
// Ajout d'un nouveau contact dans ce groupe
mGroupe.NbContact = 4
mGroupe.Contact = "Arthur"
GroupeModifie(IDSession)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = GroupeModifie(<Identifiant de la session>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la modification a été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour connaître le libellé de l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.




**`<Identifiant de la session> : (Entier)`**

Identifiant de la session à utiliser. Cet identifiant est renvoyé par une des fonctions suivantes : [EmailOuvreSessionNotes](../WDLang3/3032121.md), [EmailOuvreSessionOutlook](../WDLang3/3032128.md), [NotesOuvreBase](../WDLang3/1000005101.md) ou [OutlookOuvreSession](../WDLang3/3032123.md).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Version de Outlook utilisée
<a name="version_outlook_utilisee_ELTPARAGRAPHE000070"></a>Les groupes de contacts ne sont pas gérés par Outlook version 97. Il n'est donc pas possible de modifier un groupe de contacts dans cette version de Outlook.

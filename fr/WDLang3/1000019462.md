
## SaaSChangeMotDePasse (Fonction)

***En anglais : SaaSChangePassword***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Change le mot de passe de l'utilisateur connecté. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI SaaSConnecte("login", "motdepasse") = Vrai ALORS
	SI SaaSChangeMotDePasse("M[]t2P4zz") = Vrai ALORS
		Info("Votre mot de passe a été correctement changé")
	FIN
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = SaaSChangeMotDePasse(<Mot de passe>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le mot de passe a été changé, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire (ou si aucun utilisateur n'est connecté). Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Mot de passe> : (Chaîne de caractères)`**

Nouveau mot de passe.

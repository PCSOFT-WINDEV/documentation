
## SaaSAdminListeService (Fonction)

***En anglais : SaaSAdminListService***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Liste les services SaaS associés à un site SaaS, à une tarification ou à un utilisateur.

**Remarques** : 

- L'utilisation de cette fonction nécessite d'avoir ouvert une session d'administration SaaS au préalable en utilisant la fonction [SaaSAdminConnecte](../WDLang3/1000019098.md).

- L'Administrateur SaaS aura accès à tous les services. Un gestionnaire de compte client aura accès uniquement aux services des sites auxquels il est abonné. 



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Nombre de services du site CRM
SiteCRM est un saasSite
tabSvc est un tableau de saasService

SiteCRM.Nom = "CRM"
tabSvc = SaaSAdminListeService(SiteCRM)

Trace("Le site CRM possède " + tabSvc.Occurrence + " services.")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître les services d'un site SaaS

`<Résultat> = SaaSAdminListeService(<Site SaaS>)`
---

**`<Résultat> : (Tableau de variables de type saasService)`**

Liste des services proposés par le site.

**`<Site SaaS> : (Variable de type saasSite ou chaîne de caractères)`**

Site SaaS pour lequel les services sont recherchés : 

- Nom de la variable de type [saasSite](../WDLang3/1000019068.md) correspondant au site.

- Chaîne de caractères correspondant au nom du site SaaS.  





<a name="SYNTAXE2"></a>

### Connaître les services proposés par une tarification SaaS

`<Résultat> = SaaSAdminListeService(<Tarification>)`
---

**`<Résultat> : (Tableau de variables de type saasService)`**

Liste des services accessibles par la tarification.

**`<Tarification> : (Variable de type saasTarification)`**

Nom de la variable de type [saasTarification](../WDLang3/1000019091.md) correspondant à la tarification SaaS pour laquelle les services sont recherchés.


<a name="SYNTAXE3"></a>

### Connaître les services d'un site SaaS proposés par une tarification SaaS

`<Résultat> = SaaSAdminListeService(<Tarification> , <Site SaaS>)`
---

**`<Résultat> : (Tableau de variables de type saasService)`**

Liste des services accessibles par la tarification pour le site demandé.

**`<Tarification> : (Chaîne de caractères)`**

Nom de la tarification SaaS pour laquelle les services doivent être listés.

**`<Site SaaS> : (Chaîne de caractères)`**

Nom du site SaaS pour lequel les services doivent être listés.

**Remarque** : La tarification doit appartenir au site.


<a name="SYNTAXE4"></a>

### Connaître les services d'un site SaaS auxquels un utilisateur est abonné

`<Résultat> = SaaSAdminListeService(<Utilisateur SaaS> [, <Site SaaS> [, <Avec Doublons>]])`
---

**`<Résultat> : (Tableau de variables de type saasService)`**

Liste des services proposés pour l'utilisateur.

**`<Utilisateur SaaS> : (Variable de type saasUtilisateur)`**

Nom de la variable de type [saasUtilisateur](../WDLang3/1000019083.md) correspondant à l'utilisateur SaaS pour lequel les services sont recherchés.

**`<Site SaaS> : (Variable de type saasSite optionnelle ou chaîne de caractères optionnelle)`**

Site SaaS pour lequel les services sont recherchés : 

- Nom de la variable de type [saasSite](../WDLang3/1000019068.md) correspondant au site.

- Chaîne de caractères correspondant au nom du site SaaS.  


Si ce paramètre n'est pas précisé, les services sont recherchés pour tous les sites. 

**`<Avec Doublons> : (Booléen optionnel)`**



- <u><u><u><u>Vrai</u></u></u></u> (valeur par défaut) si le résultat peut contenir des doublons, 

- <u><u><u><u>Faux</u></u></u></u> pour supprimer les doublons du résultat.




## &lt;Graphe&gt;.Paramètre3DS (Fonction)

***En anglais : &lt;Chart&gt;.Parameter3DS***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Récupère ou modifie un paramètre de dessin d'un graphe en mode 3D Spatiale. 

Remarques : 

- Cette fonction ne doit pas être utilisée sur des champs Graphe présents dans l'éditeur d'états.

- Cette fonction est identique à la fonction [&lt;Graphe&gt;.Paramètre](../WDLang3/1000024115.md).



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Modification du type d'animation jouée lors 
// de l'affichage du champ Graphe "GRF_MonGraphe"
// Les barres se construiront progressivement et
// le graphe apparaîtra avec un effet de rotation
GRF_MonGraphe.Paramètre3DS(gr3DSPropAnimation, 10)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Récupérer la valeur d'un paramètre de dessin du graphe

`<Résultat> = <Champ Graphe>.Paramètre3DS(<Paramètre de dessin>)`
---

**`<Résultat> : (Entier, réel ou booléen)`**

Valeur du paramètre de dessin. Le type dépend du paramètre de dessin spécifié (voir les Remarques).

**`<Champ Graphe> : (Nom de champ)`**

Nom du champ Graphe à manipuler, présent sous l'éditeur de fenêtres.

**`<Paramètre de dessin> : (Constante de type entier)`**

Paramètre de dessin dont la valeur est recherchée :



|   |   |
| --- | --- |
| *gr3DSPropAnimation* | Type d'animation jouée lors de l'affichage du graphe. |
| *gr3DSPropAutoZoom* | Zoom automatique du graphe. |
| *gr3DSPropCouleurPlanX* | Couleur du plan vertical (axe à droite ou à gauche). |
| *gr3DSPropCouleurPlanY* | Couleur du plan horizontal (axe au-dessus ou au-dessous). |
| *gr3DSPropCouleurPlanZ* | Couleur du plan du fond. |
| *gr3DSPropDistance* | Zoom du graphe. |
| *gr3DSPropElevation* | Angle de rotation du graphe autour de l'axe des ordonnées (axe Y). |
| *gr3DSPropLégendeSuperposée* | Superposition de la légende et du titre du graphe. |
| *gr3DSPropLumièreX* | Abscisse (X) de la lumière éclairant le graphe. |
| *gr3DSPropLumièreY* | Ordonnée (Y) de la lumière éclairant le graphe. |
| *gr3DSPropRotation* | Angle de rotation du graphe autour de l'axe des abscisses (axe X). |
| *gr3DSPropTransparence* | Taux de transparence des barres ou des secteurs du graphe. |

### Modifier la valeur d'un paramètre de dessin du graphe

`<Champ Graphe>.Paramètre3DS(<Paramètre de dessin> , <Valeur>)`
---

**`<Champ Graphe> : (Nom de champ)`**

Nom du champ Graphe à manipuler, présent sous l'éditeur de fenêtres.

**`<Paramètre de dessin> : (Constante)`**

Paramètre de dessin à modifier :



|   |   |
| --- | --- |
| *gr3DSPropAnimation* | Type d'animation jouée lors de l'affichage du graphe. |
| *gr3DSPropAutoZoom* | Zoom automatique du graphe. |
| *gr3DSPropCouleurPlanX* | Couleur du plan vertical (axe à droite ou à gauche). |
| *gr3DSPropCouleurPlanY* | Couleur du plan horizontal (axe au-dessus ou au-dessous). |
| *gr3DSPropCouleurPlanZ* | Couleur du plan du fond. |
| *gr3DSPropDistance* | Zoom du graphe. |
| *gr3DSPropElevation* | Angle de rotation du graphe autour de l'axe des ordonnées (axe Y). |
| *gr3DSPropLégendeSuperposée* | Superposition de la légende et du titre du graphe. |
| *gr3DSPropLumièreX* | Abscisse (X) de la lumière éclairant le graphe. |
| *gr3DSPropLumièreY* | Ordonnée (Y) de la lumière éclairant le graphe. |
| *gr3DSPropRotation* | Angle de rotation du graphe autour de l'axe des abscisses (axe X). |
| *gr3DSPropTransparence* | Taux de transparence des barres ou des secteurs du graphe. |



**`<Valeur> : (Entier, réel ou booléen)`**

Valeur du paramètre de dessin. Le type dépend du paramètre de dessin spécifié (voir les Remarques).



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Paramètre de dessin
<a name="parametre_dessin_ELTPARAGRAPHE000329"></a>Les différents paramètres de dessin d'un graphe 3D Spatiale sont les suivants :


| Constante | Effet | Valeur du paramètre de dessin |
| --- | --- | --- |
| *gr3DSPropAnimation* | Type d'animation jouée lors de l'affichage du graphe | Entier<br>Les valeurs possibles sont :<br><br>- 0 : aucune animation<br><br>- 1 : zoom avant<br><br>- 2 : effet de rotation<br><br>- 4 : construction progressive des barres / secteurs<br><br>- 8 : affichage progressif des couleurs<br><br><br>Ces valeurs peuvent être additionnées.<br>La valeur par défaut est 15. |
| *gr3DSPropAutoZoom* | Zoom automatique du graphe | Booléen<br><u><u><u><u>Vrai</u></u></u></u> (par défaut) si le zoom doit être automatiquement calculé, <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.<br>Si le zoom est automatiquement calculé, la valeur du paramètre *gr3DSPropDistance* n'est pas prise en compte. |
| *gr3DSPropCouleurPlanX* | Couleur du plan vertical (axe à droite ou à gauche) | Entier<br>Composante RVB de la couleur (obtenue avec la fonction [RVB](../WDLang1/3029012.md)) ou [une couleur prédéfinie](../WDLang5/3010002.md).<br>Pour ne pas dessiner cet axe, utilisez la constante *Transparent*.<br>Cette constante est prise en compte uniquement sur les graphes de type histogramme ou courbe.<br>La valeur par défaut est *GrisClair*. |
| *gr3DSPropCouleurPlanY* | Couleur du plan horizontal (axe au-dessus ou au-dessous) | Entier<br>Composante RVB de la couleur (obtenue avec la fonction [RVB](../WDLang1/3029012.md)) ou [une couleur prédéfinie](../WDLang5/3010002.md).<br>Pour ne pas dessiner cet axe, utilisez la constante *Transparent*.<br>Cette constante est prise en compte uniquement sur les graphes de type histogramme ou courbe.<br>La valeur par défaut est *GrisClair*. |
| *gr3DSPropCouleurPlanZ* | Couleur du plan du fond | Entier<br>Composante RVB de la couleur (obtenue avec la fonction [RVB](../WDLang1/3029012.md)) ou [une couleur prédéfinie](../WDLang5/3010002.md).<br>Pour ne pas dessiner cet axe, utilisez la constante *Transparent*.<br>Cette constante est prise en compte uniquement sur les graphes de type histogramme ou courbe.<br>La valeur par défaut est *GrisClair*. |
| *gr3DSPropDistance* | Zoom du graphe | Réel<br>Les valeurs possibles sont comprises entre 0 et l'infini.<br>Cette constante est prise en compte uniquement si la constante *gr3DSPropAutoZoom*est à <u><u><u><u>Faux</u></u></u></u>.<br>La valeur par défaut est 189. |
| *gr3DSPropElevation* | Angle de rotation (en degrés) du graphe autour de l'axe des ordonnées (axe Y). | Réel<br>Les valeurs possibles sont comprises entre 0 et 360.<br>La valeur 0 permet de voir le graphe de face.<br>La valeur 90 permet de voir le graphe du dessus.<br>Par défaut, l'angle de rotation optimal est automatiquement calculé. |
| *gr3DSPropLégendeSuperposée* | Superposition de la légende et du titre du graphe. | Booléen<br><u><u><u><u>Vrai</u></u></u></u> si la légende et le titre sont superposés au graphe, <u><u><u><u>Faux</u></u></u></u> (par défaut) dans le cas contraire. |
| *gr3DSPropLumièreX* | Abscisse (X) de la lumière éclairant le graphe. | Réel<br>Les valeurs possibles sont comprises entre 0 et l'infini.<br>La valeur par défaut est 78. |
| *gr3DSPropLumièreY* | Ordonnée (Y) de la lumière éclairant le graphe | Réel<br>Les valeurs possibles sont comprises entre 0 et l'infini.<br>La valeur par défaut est 80. |
| *gr3DSPropRotation* | Angle de rotation (en degrés) du graphe autour de l'axe des abscisses (axe X). | Réel<br>Les valeurs possibles sont comprises entre 0 et 360.<br>La valeur 0 permet de voir le graphe de face.<br>La valeur 90 permet de voir le graphe de droite.<br>Par défaut, l'angle de rotation optimal est automatiquement calculé. |
| *gr3DSPropTransparence* | Taux de transparence (en pourcentage) des barres ou des secteurs du graphe. | Entier<br>Les valeurs possibles sont comprises entre 0 (opaque) et 100 (transparent/invisible).<br>La valeur par défaut est 27%. |


<a name="NOTE0_2"></a>


### Divers
<a name="divers_ELTPARAGRAPHE000531"></a>Si la fonction **&lt;Graphe&gt;.Paramètre3DS** est utilisée sur un type de graphe autre que 3D Spatiale, aucune erreur n'est renvoyée. L'information est stockée et sera utilisée lorsque le type du graphe sera modifié (avec la fonction [&lt;Graphe&gt;.Type](../WDLang3/1000024168.md) ou avec le menu contextuel du graphe).


## SocketCréeInfraRouge (Fonction)

***En anglais : SocketCreateInfrared***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Créé un socket utilisant le port infrarouge. Un socket est une ressource de communication utilisée par les applications pour communiquer d'une machine à une autre sans se soucier du type de réseau.

**Remarque** : Cette fonction doit être appelée sur le poste serveur utilisé pour la communication. Si le poste client utilise une application WINDEV pour se connecter à ce socket, cette application devra utiliser la fonction [SocketConnecteInfraRouge](../WDLang3/3070018.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI SocketCréeInfraRouge("Serveur", "Service") = Faux ALORS
	Erreur("Erreur création " + ErreurInfo(errMessage))
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = SocketCréeInfraRouge(<Nom du socket> , <Nom du service>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la fonction a réussi, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. En cas d'erreur, il est possible de connaître le détail de l'erreur grâce à la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Nom du socket> : (Chaîne de caractères)`**

Nom à donner au socket. Ce nom sera utilisé dans toutes les fonctions socket.

**`<Nom du service> : (Chaîne de caractères)`**

Nom à donner au service. Ce nom sera utilisé dans la fonction [SocketConnecteInfraRouge](../WDLang3/3070018.md).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Re-création de socket
<a name="recreation_socket_ELTPARAGRAPHE000055"></a>Si un socket n'est pas fermé correctement par la fonction [SocketFerme](../WDLang3/3070015.md) (redémarrage du poste par exemple), la recréation du socket peut être relativement longue. Il est en effet nécessaire d'attendre que le système libère le socket.


<a name="NOTE0_2"></a>




### Création de socket
<a name="creation_socket_ELTPARAGRAPHE000067"></a>Pour créer un socket sur un poste serveur, utilisez la fonction [SocketCrée](../WDLang3/3070011.md).


<a name="NOTE0_3"></a>




### Protocole IRDA
<a name="protocole_irda_ELTPARAGRAPHE000079"></a>La fonction **SocketCréeInfraRouge** utilise le protocole IRDA (Infrared Data Association).


<a name="NOTE0_4"></a>




### Limite Windows
<a name="limite_windows_ELTPARAGRAPHE000091"></a>Le nombre maximum de sockets infrarouge utilisés simultanément est 80.

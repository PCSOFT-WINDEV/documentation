


## &lt;Variable sshSession&gt;.Ecrit (Fonction)

***En anglais : &lt;sshSession variable&gt;.Write***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ecrit des données dans la session SSH spécifiée.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
cMaSession est une sshSession
// La session aura été ouverte avec la fonction <sshSession>.ConnecteShell. 
// Caract(10) permet de terminer la ligne de commande à envoyer. 

// Positionne dans le dossier /var/log
cMaSession.Ecrit("cd /var/log" + Caract(10))

// Demande à lire les lignes contenant "error"
cMaSession.Ecrit("grep -i error /var/log/messages" + Caract(10))
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Session SSH>.Ecrit(<Données>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si les données ont été envoyées correctement, 

- <u><u><u><u>Faux</u></u></u></u> en cas d'erreur. La fonction [ErreurInfo](../WDLang1/3013008.md) permet d'obtenir plus de renseignements sur le problème rencontré.




**`<Session SSH> : (Variable de type sshSession)`**

Nom de la variable [sshSession](../WDLang3/1000021157.md) correspondant à la session SSH à manipuler.

**`<Données> : (Buffer)`**

Données à écrire dans la session SSH.

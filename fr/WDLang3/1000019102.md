
## SaaSAdminAjouteSite (Fonction)

***En anglais : SaaSAdminAddSite***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ajoute un site SaaS.

**Remarques** : 

- L'utilisation de cette fonction nécessite d'avoir ouvert une session d'administration SaaS au préalable en utilisant la fonction [SaaSAdminConnecte](../WDLang3/1000019098.md).

- L'utilisation de cette fonction est réservée à l'Administrateur SaaS. Un gestionnaire de compte client ne peut pas utiliser cette fonction. 



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ajout d'un site "Forum" dans la base du Webservice SaaS
S est un saasSite
S.Nom = "Forum"
SaaSAdminAjouteSite(S)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = SaaSAdminAjouteSite(<Site SaaS>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'ajout a été effectué, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Site SaaS> : (Variable de type saasSite ou chaîne de caractères)`**

Description du nouveau site SaaS.

Si ce paramètre est une chaîne de caractères, ce paramètre doit correspondre au nom du site tel qu'il a été déployé (en tenant compte de la casse).

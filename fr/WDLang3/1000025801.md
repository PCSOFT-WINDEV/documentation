


## &lt;Carte&gt;.AjouteForme (Fonction)

***En anglais : &lt;Map&gt;.AddShape***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ajoute une nouvelle forme (polygone, ligne, cercle) sur la carte affichée dans un champ Carte.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ajoute un polygone dans le champ Carte
Polygone est une CartePolygone
POUR TOUTE PositionPoint DE tabPositions
	Polygone.Point.Ajoute(PositionPoint)
FIN
Polygone.CouleurTrait = RougeClair
Polygone.ActionClic = ProcClic
CARTE_MaCarte.AjouteForme(Polygone)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Champ Carte>.AjouteForme(<Forme>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la forme a pu être ajoutée dans le champ Carte,

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Champ Carte> : (Nom de champ)`**

Nom du champ Carte à manipuler.

**`<Forme> : (Variable de type CarteCercle, CartePolygone ou CartePolyligne)`**

Forme à dessiner sur la carte affichée dans le champ Carte. Cette forme est décrite à l'aide d'un des types de variables suivants : 

- une variable de type [CartePolygone](../WDLang3/1000025476.md), 

- une variable de type [CartePolyligne](../WDLang3/1000025489.md), 

- une variable de type [CarteCercle](../WDLang3/1000025504.md).






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Lorsque la forme a été ajoutée, il sera possible de la manipuler par son nom (propriété **Nom** de la variable associée). 

- Si la forme a été déjà été ajoutée dans le champ Carte ou si la forme est invalide (exemple : polygone avec un nombre insuffisant de points), une erreur fatale sera affichée.

- Pour supprimer une forme dessinée dans un champ Carte, utilisez la fonction [&lt;Carte&gt;.SupprimeForme](../WDLang3/1000025820.md). 

- Pour modifier une forme dessinée dans un champ Carte, utilisez la fonction [&lt;Carte&gt;.ModifieForme](../WDLang3/1000025805.md).

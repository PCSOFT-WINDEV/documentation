


## ContactAjoute (Fonction)

***En anglais : ContactAdd***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ajoute un contact : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) dans le carnet d'adresses d'une messagerie Lotus Notes ou Outlook. 
	**Attention** : Cette fonction **ne permet pas** d'ajouter un contact dans le carnet d'adresses d'une messagerie Outlook Express.

- ![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) dans la liste des contacts de l'appareil (Android, iPhone, iPad, Universal Windows).

<a name="sample_code"></a>

## Exemple
<a class="notetitle" target="_blank" href="$DOC$=1000020272&name=contactajoute_fonction&product=WD">Voir des exemples supplémentaires</a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) 
```wl
// Initialisation des variables de la structure Contact
ContactRAZ()
// Renseignements des variables pour le nouveau contact
mContact.Nom = "Dupont"
mContact.Prénom = "Axelle"
mContact.Email = "axelle.dupont@mesmails.com"
// Ajout du contact
SI ContactAjoute(IDSession) = Vrai ALORS
	Info("Contact ajouté")
SINON
	Erreur("Erreur lors de l'ajout", ErreurInfo())
FIN
```
<a name="Exemple2"></a>
![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) 
```wl
// Variables locales
tContact est un Contact
tTéléphone est un Téléphone

// Identité
tContact.Nom = SAI_Nom
tContact.Prénom = SAI_Prénom

// Numéros de téléphone Fixe
SI SAI_TéléphoneFixe <> "" ALORS
	tTéléphone.Numéro = SAI_TéléphoneFixe
	tTéléphone.Type = telDomicile
	Ajoute(tContact.Téléphone, tTéléphone)
FIN

// Ajout
SI PAS ContactAjoute(tContact, gSource) ALORS
	Erreur("Impossible de créer le contact.")
SINON
	FEN_TableContacts.RafraichitListeContact()
	Ferme()
FIN
```

## Syntaxe
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) 
### Ajouter un contact dans la messagerie Lotus Notes ou Outlook

`<Résultat> = ContactAjoute(<Identifiant de la session>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'ajout a été effectué, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour connaître le libellé de l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.




**`<Identifiant de la session> : (Entier)`**

Identifiant de la session à utiliser. Cet identifiant est renvoyé par la fonction [EmailOuvreSessionNotes](../WDLang3/3032121.md), [EmailOuvreSessionOutlook](../WDLang3/3032128.md), [NotesOuvreBase](../WDLang3/1000005101.md) ou [OutlookOuvreSession](../WDLang3/3032123.md).


<a name="SYNTAXE2"></a>
![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) 
### Ajouter un contact dans la liste des contacts de l'appareil

`<Résultat> = ContactAjoute(<Contact> [, <Source>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'ajout a été effectué, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour connaître le libellé de l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.




**`<Contact> : (Variable de type Contact)`**

Nom de la variable de type [Contact](../WDLang3/1000020254.md) correspondant au contact à ajouter. 

**`<Source> : (Variable optionnelle de type ContactSource)`**

Nom de la variable de type [ContactSource](../WDLang3/1000020211.md) correspondant à la source ou au compte dans lequel le contact doit être ajouté. Si ce paramètre n'est pas précisé, le contact ne sera associé à aucun compte.

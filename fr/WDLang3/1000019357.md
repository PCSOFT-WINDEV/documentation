
## WiFiSupprimeRéseau (Fonction)

***En anglais : WiFiDeleteNetwork***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime un réseau de la liste des réseaux Wi-Fi configurés de l'appareil.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
WiFiSupprimeRéseau("1478")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = WiFiSupprimeRéseau(<Identifiant>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la suppression a été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. La fonction [ErreurInfo](../WDLang1/3013008.md) permet d'obtenir plus de renseignements sur le problème rencontré.




**`<Identifiant> : (Chaîne de caractères)`**

Identifiant du réseau à supprimer.

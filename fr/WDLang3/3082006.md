


## Fonctions d'archivage (gestion des ZIP, WDZ, CAB, RAR, 7z, TAR et TGZ (TAR.GZ))
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>










Les fonctions WLangage permettant de créer et de gérer des archives sont les suivantes :




|   |   |
| --- | --- |
| [zipAjouteFichier](../WDLang3/3082008.md) | Ajoute et compresse automatiquement un fichier (de n'importe quel type) dans une archive de type CAB, ZIP, WDZ, 7z, TAR ou TGZ (TAR.GZ). |
| [zipAjouteListeFichier](../WDLang3/3082038.md) | Ajoute et compresse automatiquement une liste de fichiers (de n'importe quel type) dans une archive au format ZIP, CAB, WDZ ou 7z. |
| [zipAjouteRépertoire](../WDLang3/3082014.md) | Ajoute et compresse automatiquement la totalité des fichiers d'un répertoire et de ses sous-répertoires dans une archive. |
| [zipChangeChemin](../WDLang3/3082027.md) | Modifie le chemin mémorisé d'un fichier d'une archive (au format ZIP, WDZ ou 7z). |
| [zipChercheFichier](../WDLang3/3082002.md) | Recherche un fichier dans une archive (au format CAB, ZIP, RAR, WDZ ou 7z). |
| [zipCrée](../WDLang3/3082003.md) | Crée une archive. |
| [zipCréeExe](../WDLang3/3082013.md) | Crée une archive auto-extractible sous forme d'un exécutable en 32 bits (.EXE). |
| [zipDécoupe](../WDLang3/3082020.md) | Découpe une archive en plusieurs sous-archives d'une taille donnée. |
| [zipEstMulti](../WDLang3/3082021.md) | Renvoie le type de l'archive : archive mono-partie ou archive multi-parties. |
| [zipExiste](../WDLang3/1000017233.md) | Permet de savoir si une archive existe. |
| [zipExtraitChemin](../WDLang3/3082023.md) | Renvoie le chemin d'un fichier de l'archive. |
| [zipExtraitFichier](../WDLang3/3082019.md) | Extrait et décompresse automatiquement un fichier d'une archive à un emplacement physique ou en mémoire. |
| [zipExtraitListeFichier](../WDLang3/1000017234.md) | Extrait et décompresse automatiquement une liste de fichiers présents dans une archive vers un emplacement physique. |
| [zipExtraitRépertoire](../WDLang3/1000025518.md) | Extrait et décompresse automatiquement les fichiers présents dans un répertoire de l'arborescence des fichiers d'une archive vers un emplacement physique. |
| [zipExtraitTout](../WDLang3/3082011.md) | Extrait et décompresse automatiquement la totalité des fichiers d'une archive à un emplacement physique. |
| [zipFerme](../WDLang3/3082005.md) | Ferme une archive. |
| [zipFichierEnCours](../WDLang3/3082026.md) | Renvoie le nom du fichier archivé en cours de manipulation par les fonctions d'ajout et d'extraction de fichiers. |
| [zipFiltre](../WDLang3/1000026293.md) | Renvoie la liste des formats d'archives gérés par la plateforme en cours, au format attendu par le filtre de la fonction [fSélecteur](../WDLang1/3036053.md). |
| [zipInfoFichier](../WDLang3/3082040.md) | Renvoie les caractéristiques d'un fichier de l'archive :<br><br>- Le nom du fichier et son chemin mémorisé.<br><br>- La taille originale du fichier.<br><br>- La taille compressée du fichier.<br><br>- La date et l'heure de création du fichier.<br><br>- La date et l'heure de modification du fichier.<br><br>- La date et l'heure d'accès du fichier.<br><br>- Les attributs du fichier.<br><br>- Le mode de cryptage du fichier.<br><br><br> |
| [zipListeFichier](../WDLang3/3082031.md) | Renvoie la liste des fichiers présents dans une archive. |
| [zipMotDePasse](../WDLang3/3082053.md) | Définit le mot de passe utilisé pour les opérations d'ajout et d'extraction de fichiers dans l'archive spécifiée (format RAR, ZIP et WDZ). |
| [zipMsgErreur](../WDLang3/3082034.md) | Renvoie le message associé à un code d'erreur sur une archive. |
| [zipNbFichier](../WDLang3/3082035.md) | Renvoie le nombre de fichiers présents dans une archive (au format CAB, RAR, ZIP, WDZ ou 7z). |
| [zipNbPartie](../WDLang3/3082029.md) | Renvoie le nombre de sous-archives (parties) d'une archive au format CAB, RAR, ZIP, WDZ, 7z, TAR ou TGZ (TAR.GZ). |
| [zipNbPartieNécessaire](../WDLang3/3082052.md) | Renvoie le nombre de parties d'une taille donnée nécessaires pour découper une archive mono-partie. |
| [zipNiveauCompression](../WDLang3/3082054.md) | Change le niveau de compression utilisé dans la création d'une archive au format ZIP ou au format 7z. |
| [zipOuvre](../WDLang3/3082044.md) | Ouvre une archive existante. |
| [zipOuvreCAB](../WDLang3/1000017250.md) | Ouvre une archive existante au format CAB. |
| [zipOuvreRAR](../WDLang3/1000017294.md) | Ouvre une archive existante au format RAR. |
| [zipRegroupe](../WDLang3/3082046.md) | Regroupe les différentes sous-archives d'une archive multi-parties pour en faire une archive mono-partie. |
| [zipSélectionFichier](../WDLang3/1000026152.md) | Définit une sélection des fichiers contenus dans l'archive. |
| [zipSupprimeFichier](../WDLang3/3082041.md) | Supprime un fichier contenu dans une archive. |
| [zipSupprimeListeFichier](../WDLang3/3082049.md) | Supprime un ensemble de fichiers contenu dans une archive. |
| [zipSupprimeTout](../WDLang3/3082051.md) | Supprime tous les fichiers présents dans une archive. |
| [zipTaille](../WDLang3/3082012.md) | Renvoie la taille de l'archive (au format ZIP, RAR, CAB, WDZ ou 7z)  :<br><br>- soit la taille de l'ensemble des fichiers compressés présents dans l'archive.<br><br>- soit la taille de l'ensemble des fichiers avant compression présents dans l'archive.<br><br><br> |
| [zipTailleFichier](../WDLang3/3082017.md) | Renvoie la taille d'un des fichiers de l'archive (au format CAB, RAR, WDZ, ZIP, 7z, TAR ou TGZ (TAR.GZ)) :<br><br>- soit la taille du fichier compressé.<br><br>- soit la taille du fichier avant compression.<br><br><br> |



**Remarque** : Les fonctions d'archivage peuvent être utilisées dans un thread sans bloquer les autres threads pendant leur exécution.

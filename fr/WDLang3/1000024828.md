


## Procédure WLangage appelée par la fonction URISélecteur



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Procédure WLangage (également nommée "callback") appelée par la fonction [URISélecteur](../WDLang3/1000024026.md) pour le fichier sélectionné. Cette procédure est appelée lorsque l'utilisateur ferme la fenêtre de sélection d'URI.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ouvre la fenêtre système pour la sélection d'une image ou d'un pdf
URISélecteur(MaProcURISélecteur, uriImage + TAB + "application/pdf")

PROCEDURE INTERNE MaProcURISélecteur(Succès est un booléen, Résultat1 est une URI)
	SI Succès = Vrai ALORS
		// Affiche le nom de la ressource sélectionnée
		ToastAffiche(URIRécupèreInfo(Résultat1, uriInfoNom))
	FIN
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`URISélecteur_Callback(<Succès> , <Résultat>)`
---

**`<Succès> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la ressource a été sélectionnée. 
	Dans ce cas, &lt;Résultat&gt; est une variable de type [URI](../WDLang3/1000024027.md) correspondant à la ressource sélectionnée. 

- <u><u><u><u>Faux</u></u></u></u> en cas d'erreur. 




**`<Résultat> : (Variable de type URI)`**

Variable de type [URI](../WDLang3/1000024027.md) correspondant à la ressource sélectionnée.




## &lt;Graphe&gt;.Nuage3DCouleurAltitude (Fonction)

***En anglais : &lt;Chart&gt;.Scatter3DAltitudeColor***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Définit les différentes couleurs utilisées selon les niveaux d'altitude d'une série d'un graphe de type Nuage de points 3D.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
GRF_MonNuage3D.Nuage3DCouleurAltitude(1, RougeClair, 100)
GRF_MonNuage3D.Nuage3DCouleurAltitude(1, BleuPastel, 0)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Champ Graphe>.Nuage3DCouleurAltitude(<Numéro de la série> , <Couleur> , <Altitude>)`
---

**`<Champ Graphe> : (Nom de champ)`**

Nom du champ Graphe à manipuler (présent dans l'éditeur de fenêtres).

**`<Numéro de la série> : (Entier)`**

Numéro de la série à manipuler.

**`<Couleur> : (Entier ou constante de type Entier)`**

Couleur de l'altitude spécifiée. Cette couleur peut correspondre à :

- une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)),

- une couleur TSL (obtenue avec la fonction [TSL](../WDLang1/3029057.md)),

- une couleur prédéfinie (voir [les couleurs prédéfinies du WLangage](../WDLang5/3010002.md)).




**`<Altitude> : (Réel)`**

Altitude pour laquelle la couleur sera appliquée.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Si des couleurs d'altitude sont spécifiées, les points se situant à ces altitudes recevront ces couleurs. 

Pour les points se situant entre deux couleurs, un dégradé linéaire sera appliqué.

Pour les points en dehors des bornes de couleurs, la couleur de la série sera appliquée.

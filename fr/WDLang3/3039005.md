
## Constantes de gestion des flux (son et vidéo)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


| Constante | Description | Utilisée par les fonctions ou propriétés |
| --- | --- | --- |
| **FluxAffichageAucun** | Vidéo affichée dans le champ Conférence : Aucune vidéo n'est affichée. | Propriété [Affichage](../Proprietes/2510010.md) |
| **FluxAffichageEmission** | Vidéo affichée dans le champ Conférence : La vidéo affichée correspond à la vidéo émise par le poste en cours. | Propriété [Affichage](../Proprietes/2510010.md) |
| **FluxAffichageRéception** | Vidéo affichée dans le champ Conférence : La vidéo affichée correspond à la vidéo reçue sur le poste en cours. | Propriété [Affichage](../Proprietes/2510010.md) |
| **FluxNomAppelant** | Nom de l'émetteur du flux. | [FluxInfo](../WDLang3/3039009.md), [FluxStop](../WDLang3/3039002.md) |
| **FluxNuméroAppelant** | Numéro de l'émetteur de flux (par exemple son adresse IP). | [FluxInfo](../WDLang3/3039009.md), [FluxStop](../WDLang3/3039002.md) |
| **FluxSon** | Type de données émise par le flux : Les données de type son sont émises. | Propriété [Emission](../Proprietes/2510037.md) |
| **FluxSonDuplex** | Flux de type son reçu et émis (correspond à FluxSonEmission + FluxSonRéception). | [FluxJoue](../WDLang3/3039010.md), [FluxStop](../WDLang3/3039002.md) |
| **FluxSonEmission** | Flux de type son émis. | [FluxJoue](../WDLang3/3039010.md), [FluxStop](../WDLang3/3039002.md) |
| **FluxSonRéception** | Flux de type son reçu. | [FluxJoue](../WDLang3/3039010.md), [FluxStop](../WDLang3/3039002.md) |
| **FluxTout** | Flux de type vidéo + son (FluxSonDuplex + FluxVidéoDuplex). | [FluxJoue](../WDLang3/3039010.md), [FluxStop](../WDLang3/3039002.md) |
| **FluxVidéo** | Type de données émise par le flux : Les données de type Vidéo sont émises. | Propriété [Emission](../Proprietes/2510037.md) |
| **FluxVidéoDuplex** | Flux de type vidéo reçu et émis (correspond à FluxVidéoEmission + FluxVidéoRéception). | [FluxJoue](../WDLang3/3039010.md), [FluxStop](../WDLang3/3039002.md) |
| **FluxVidéoEmission** | Flux de type Vidéo émis. | [FluxJoue](../WDLang3/3039010.md), [FluxStop](../WDLang3/3039002.md) |
| **FluxVidéoRéception** | Flux de type Vidéo reçu. | [FluxJoue](../WDLang3/3039010.md), [FluxStop](../WDLang3/3039002.md) |


## ContactSource (Type de variable)

***En anglais : ContactSource***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **ContactSource** permet de définir toutes les caractéristiques avancées d'une source (ou compte) de contacts (exemple : Google, Exchange, Facebook).

Les caractéristiques de cette source peuvent être définies et modifiées à l'aide de différentes propriétés WLangage. 

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Liste les comptes Google du téléphone
tabSource est un tableau de ContactSource = ContactListeSource()
SI PAS ErreurDétectée ALORS
	POUR TOUT ELEMENT source de tabSource
		SI tabSource.Type = " com.google " ALORS
			// Traitement 	
		FIN
	FIN
FIN
```

<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Propriétés spécifiques à la description des variables de type ContactSource
<a name="proprietes_specifiques_description_des_variables_type_contactsource_ELTPARAGRAPHE000045"></a>Les propriétés suivantes peuvent être utilisées pour manipuler la source des contacts :

| **Nom de la propriété** | **Type manipulé** | **Effet** |
| --- | --- | --- |
| Nom | Chaîne de caractères | Nom de la source ou du compte. Cette propriété peut par exemple correspondre à une adresse email dans le cas d'un compte Google (exemple "nom.prenom@gmail.com"). |
| Type | Chaîne de caractères | Type de la source ou du compte (exemple "com.google" pour un compte Google). |


<a name="NOTE0_2"></a>


### Cas particuliers
<a name="cas_particuliers_ELTPARAGRAPHE000078"></a>

- Pour lister les sources des contacts de l'appareil, utilisez la fonction [ContactListeSource](../WDLang3/1000020214.md).

- Pour réinitialiser le contenu d'une variable de type **ContactSource**, il est possible d'utiliser la fonction [VariableRAZ](../WDLang1/3013069.md).

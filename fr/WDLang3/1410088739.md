
## iCloudCléValeurEcrit (Fonction)

***En anglais : iCloudKeyValueWrite***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet d'enregistrer la valeur d'une clé dans un conteneur de paires "clé-valeur" basé sur Apple iCloud.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Tableau de réel
tabRéels est un tableau de 5 réels
tabRéelsLu est un tableau de 5 réels
tabRéels[1] = 45.698
tabRéels[2] = 149.01
tabRéels[3] = 5.589
tabRéels[4] = 154892.025
tabRéels[5] = 7820

// Ecriture
iCloudCléValeurEcrit("Tableau de réels", tabRéels)

// Lecture
tabRéelsLu = iCloudCléValeurLit("Tableau de réels")

// Suppression
iCloudCléValeurSupprime("Tableau de réels")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = iCloudCléValeurEcrit(<Clé> , <Valeur>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'écriture a été réalisée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 




**`<Clé> : (Chaîne de caractères)`**

Nom de la clé à manipuler. 

**`<Valeur> : (Variant)`**

Valeur de la clé. Cette valeur peut correspondre à une variable de type entier, réel, dateheure, booléen, chaine, binaire et tableau d'un de ces types.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Le nom de la clé doit faire moins de 64 octets en encodage UTF8.

- Une application peut utiliser au maximum 1024 clés.

- L'espace total pour le stockage des clés ne peut pas dépasser un 1 MégaOctets.

- Le partage des clés/valeurs s'effectue pour une même application sur des appareils utilisant le même compte iCloud.

- La synchronisation entre appareils peut prendre plusieurs minutes.

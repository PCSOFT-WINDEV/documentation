
## &lt;Graphe&gt;.AxeVisible (Fonction)

***En anglais : &lt;Chart&gt;.AxisVisible***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Paramètre l'affichage des axes dans un graphe de type Courbe ou Histogramme. L'axe peut :

- être visible, 

- être invisible, 

- afficher uniquement le libellé.




![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Les graphes](../WDChamp/3042060.md). 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
GRF_Graphe1.AxeVisible(grAbscisse, grInvisible)
// ou par compatibilité
// GRF_Graphe1.AxeVisible(grAbscisse, Faux)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Champ Graphe>.AxeVisible(<Type de l'axe> , <Visible>)`
---

**`<Champ Graphe> : (Nom de champ)`**

Nom du champ Graphe à manipuler (présent dans l'éditeur de fenêtres, de pages ou d'états).



**`<Type de l'axe> : (Constante de type Entier)`**

Axe à manipuler :


|   |   |
| --- | --- |
| *grAbscisse* | Axe des abscisses. |
| *grOrdonnée* | Axe des ordonnées. |



**`<Visible> : (Booléen)`**

Mode d'affichage de l'axe : 


|   |   |
| --- | --- |
| *grInvisible* | L'axe devient invisible.<br>Par compatibilité, il est également possible d'utiliser &lt;Visible&gt; = Faux. |
| *grLibelléUniquement* | Seuls les libellés de l'axe sont visibles.<br><br> |
| *grVisible* | L'axe devient visible. <br>Par compatibilité, il est également possible d'utiliser &lt;Visible&gt; = Vrai. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Si l'axe des ordonnées est invisible, l'axe des ordonnées secondaires est également invisible. 

- Les paramètres définis avec la fonction **&lt;Graphe&gt;.AxeVisible** seront pris en compte lors du prochain appel à la fonction [&lt;Graphe&gt;.Dessine](../WDLang3/1000024062.md).




## grRotation (Fonction)

***En anglais : grRotation***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de : 

- Effectuer une rotation d'un graphe sur un de ses axes afin de permettre une meilleure visualisation des données. 

- Connaître l'angle de rotation d'un graphe sur un de ses axes. 




Le graphe manipulé peut être : 

- un graphe de type Surface, 

- un graphe de type Pyramide.





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Effectue une légère rotation de 35 degrés
// pour mettre en évidence l'effet de relief 
// sur un graphe de type Surface
grRotation(GRF_Surface, grAxeX, -35)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Effectuer une rotation d'un graphe

`grRotation(<Nom du graphe> , <Axe> , <Angle>)`
---

**`<Nom du graphe> : (Nom de champ ou chaîne de caractères)`**

Nom du graphe à manipuler. Ce graphe peut être : 

- un graphe de type Surface, 

- un graphe de type Pyramide.




Ce nom correspond :

- soit au nom du graphe défini par programmation avec la fonction [grCrée](../WDLang3/3042005.md).

- soit au nom du champ Graphe (présent dans l'éditeur de fenêtres ou d'états).




**`<Axe> : (Constante de type Entier)`**

Axe de rotation : 


|   |   |
| --- | --- |
| *grAxeX* | Rotation selon l'axe des X. |
| *grAxeY* | Rotation selon l'axe des Y. |
| *grAxeZ* | Rotation selon l'axe des Z. |



**`<Angle> : (Réel)`**

Angle de rotation en degrés. 


<a name="SYNTAXE2"></a>

### Connaître l'angle de rotation d'un graphe

`<Résultat> = grRotation(<Nom du graphe> , <Axe de rotation>)`
---

**`<Résultat> : (Réel)`**

Angle de rotation en degrés. 

**`<Nom du graphe> : (Nom de champ ou chaîne de caractères)`**

Nom du graphe à manipuler. Ce graphe peut être : 

- un graphe de type Surface, 

- un graphe de type Pyramide.




Ce nom correspond :

- soit au nom du graphe défini par programmation avec la fonction [grCrée](../WDLang3/3042005.md).

- soit au nom du champ Graphe (présent dans l'éditeur de fenêtres ou d'états).




**`<Axe de rotation> : (Constante de type Entier)`**

Axe de rotation : 


|   |   |
| --- | --- |
| *grAxeX* | Rotation selon l'axe des X. |
| *grAxeY* | Rotation selon l'axe des Y. |
| *grAxeZ* | Rotation selon l'axe des Z. |


## BTListeService (Fonction)

***En anglais : BTListService***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Liste les services offerts par un périphérique Bluetooth.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
nNuméroPériph est un entier
sListeServices est une chaîne
sChaîneTmp est une chaîne
// Vide le champ LISTE_Services
ListeSupprimeTout(LISTE_Services)
// Liste les services du périphérique sélectionné
SI TableSelect(TABLE_Appareils) <> -1 ALORS
	// Identifiant du périphérique
	nNuméroPériph = TABLE_Appareils.COL_ID[TABLE_Appareils]
	// Récupération de la liste des services
	sListeServices = BTListeService(nNuméroPériph)
	// Ajout des services dans la liste
	POUR TOUTE CHAINE sChaîneTmp DE sListeServices SEPAREE PAR RC
		ListeAjoute(LISTE_Services, sChaîneTmp)
	FIN
SINON
	Info("Sélectionnez un périphérique Bluetooth")
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = BTListeService(<Identifiant du périphérique> [, <Niveau de détail> [, <Identifiant de la radio>]])`
---

**`<Résultat> : (Chaîne de caractères)`**

Liste des services supportés par le périphérique. Cette liste est de la forme :

- en mode simple : &lt;Nom Service 1&gt; + RC + ... + RC + &lt;Nom Service N&gt;

- en mode détaillé : &lt;Nom Service 1&gt; + TAB + &lt;Description Service 1&gt; + TAB + &lt;Identifiant Service 1&gt; + RC + ... + &lt;Nom Service N&gt; + TAB + &lt;Description Service N&gt; + TAB + &lt;Identifiant Service N&gt;




**`<Identifiant du périphérique> : (Entier)`**

Identifiant du périphérique Bluetooth à manipuler. Cet identifiant est retourné par la fonction [BTListePériphérique](../WDLang3/1000017132.md) ou par la fonction [BTSélecteurPériphérique](../WDLang3/1000017135.md).

**`<Niveau de détail> : (Constante optionnelle de type Entier)`**

Niveau de détail du résultat :


|   |   |
| --- | --- |
| *btListeDétaillée* | &lt;Résultat&gt; contient la liste des services en mode détaillé. |
| *btListeSimple*<br>(Valeur par défaut) | &lt;Résultat&gt; contient la liste des services en mode simple. |



**`<Identifiant de la radio> : (Entier optionnel)`**

Identifiant de la radio Bluetooth à manipuler. Cet identifiant est retourné par la fonction [BTListeRadio](../WDLang3/1000017125.md).

- Pour connaître la pile utilisée, utilisez la fonction [BTPile](../WDLang3/1000017229.md). Pour plus de détails sur les clés supportées, consultez [Quelles piles utiliser ?](../WDLang3/1000017231.md)

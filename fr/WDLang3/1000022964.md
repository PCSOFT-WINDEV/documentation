


## grNuage3DAjouteDonnée (Fonction)

***En anglais : grScatter3DAddData***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ajoute un nouveau point à afficher dans un graphe de type Nuage de points 3D. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ajoute un point avec la couleur et la forme de la série
grNuage3DAjouteDonnée(GRF_NomNuage3D, 1, 10, 10, 10)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`grNuage3DAjouteDonnée(<Nom du graphe> , <Numéro de la série> , <Valeur X> , <Valeur Y> , <Valeur Z> [, <Couleur> [, <Type>]])`
---

**`<Nom du graphe> : (Nom de champ ou chaîne de caractères)`**

Nom du graphe à manipuler. Ce nom peut correspondre :

- au nom du graphe défini par programmation avec la fonction [grCrée](../WDLang3/3042005.md).

- au nom du champ Graphe (présent dans l'éditeur de fenêtres).




**`<Numéro de la série> : (Entier)`**

Numéro de la série dans laquelle une valeur doit être ajoutée. Si la série spécifiée n'existe pas, le grapheur crée autant de séries que nécessaire pour obtenir une matrice de données cohérente. Les séries ainsi créées sont toutes initialisées à 0.

**`<Valeur X> : (Réel)`**

Coordonnée horizontale (abscisse) du point à ajouter.

**`<Valeur Y> : (Réel)`**

Coordonnée verticale (ordonnée) du point à ajouter. 

**`<Valeur Z> : (Réel)`**

Coordonnée en altitude du point à ajouter. 

**`<Couleur> : (Entier ou constante de type Entier)`**

Couleur du point. Cette couleur peut correspondre à :

- une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)),

- une couleur TSL (obtenue avec la fonction [TSL](../WDLang1/3029057.md)),

- [une couleur prédéfinie](../WDLang5/3010002.md).

- la constante *CouleurIndéfinie* (valeur pas défaut) pour appliquer la couleur de la série. 




**`<Type> : (Constante optionnelle de type Entier)`**

Type du point  :


|   |   |
| --- | --- |
| *grPointCarré* | Point en forme de carré. |
| *grPointCercle* | Point en forme de cercle. |
| *grPointCroix* | Point en forme de croix. |
| *grPointCube* | Point en forme de cube (3D). |
| *grPointLosange* | Point en forme de losange. |
| *grPointPyramide* | Point en forme de pyramide (3D). |
| *grPointSphère* | Point en forme de sphère (3D). |
| *grPointTriangle* | Point en forme de triangle. |


Si ce paramètre n'est pas précisé, le type du point correspond au type de point défini pour la série. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
La taille du point est basée sur la taille minimale d'une graduation du graphe.

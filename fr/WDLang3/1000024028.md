
## Constantes URI
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


Les constantes utilisées par les fonctions URIxxx sont les suivantes :

| Constante | Détail | Fonction |
| --- | --- | --- |
| **uriAjout** | Le contenu du fichier ou du buffer est ajouté à la fin du contenu initial de la ressource. | [FichierVersURI](../WDLang3/1410087413.md)<br>[BufferVersURI](../WDLang3/1410087415.md) |
| **uriAudio** | Types MIME gérés : Autorise tous les types de fichier audio. | [URISélecteur](../WDLang3/1000024026.md) |
| **uriCréation** | <br><br>- Ouvre le sélecteur en mode "Création" pour créer un nouveau document. <br><br>- Le contenu initial de la ressource est remplacé par celui du fichier ou du buffer.<br><br><br> | <br><br>- [URISélecteur](../WDLang3/1000024026.md)<br><br>- [FichierVersURI](../WDLang3/1410087413.md)<br>	[BufferVersURI](../WDLang3/1410087415.md)<br><br><br> |
| **uriImage** | Types MIME gérés : Autorise tous les types de fichier image. | [URISélecteur](../WDLang3/1000024026.md) |
| **uriInfoNom** | Information demandée : Nom de la ressource (sans extension). | [URIRécupèreInfo](../WDLang3/1000024037.md) |
| **uriInfoTaille** | Information demandée : Taille de la ressource (en octets). | [URIRécupèreInfo](../WDLang3/1000024037.md) |
| **uriInfoTypeConversion** | Information demandée : Types MIME (et extensions de fichier correspondantes) dans lesquels la ressource peut être convertie lors de l'export de la ressource. | [URIRécupèreInfo](../WDLang3/1000024037.md) |
| **uriInfoTypeOrigine** | Information demandée : Type MIME d'origine de la ressource et extension de fichier correspondante. | [URIRécupèreInfo](../WDLang3/1000024037.md) |
| **uriOuverture** | Ouvre le sélecteur en mode "Ouverture" pour ouvrir un document existant. | [URISélecteur](../WDLang3/1000024026.md) |
| **uriVidéo** | Types MIME gérés : Autorise tous les types de fichier image. | [URISélecteur](../WDLang3/1000024026.md) |

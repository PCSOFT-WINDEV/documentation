


## &lt;Variable httpRéponse&gt;.RécupèreCookie (Fonction)

***En anglais : &lt;httpResponse variable&gt;.GetCookie***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Récupère les cookies lus dans la variable [httpRéponse](../WDLang3/1000021165.md) (contenant le résultat de la fonction [&lt;Variable httpRequête&gt;.Envoie](../WDLang3/1000021624.md)).






<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE3"></a>

### Récupérer les cookies lus via une variable de type httpRéponse

`<Résultat> = <Réponse HTTP>.RécupèreCookie()`
---

**`<Résultat> : (Tableau de variables de type Cookie)`**

Liste des cookies stockés en mémoire pour le domaine spécifié.

**`<Réponse HTTP> : (Variable de type httpRéponse)`**

Nom de la variable [httpRéponse](../WDLang3/1000021165.md) pour laquelle les cookies doivent être listés.




## &lt;Graphe&gt;.PoliceTitre (Fonction)

***En anglais : &lt;Chart&gt;.TitleFont***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie la police utilisée pour le titre d'un graphe.

Exemple de titre dans un graphe : ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Titre.gif)




![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Les graphes](../WDChamp/3042060.md). 








<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Modification de la police utilisée pour le titre du graphe GRF_MonGraphe
// L'identifiant de la police utilisée est 7.
GRF_MonGraphe.PoliceTitre(7)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Définition de la police utilisée pour le titre d'un graphe

`<Champ Graphe>.PoliceTitre(<Police utilisée>)`
---

**`<Champ Graphe> : (Nom de champ)`**

Nom du champ Graphe à manipuler présent dans l'éditeur de fenêtres, de pages ou d'états.


**`<Police utilisée> : (Police)`**

Police à utiliser. Ce paramètre correspond à une variable de type [Police](../Motscles/1514045.md). Les caractéristiques de la police ont été définies grâce à la fonction [PoliceCrée](../WDLang1/3060002.md) ou grâce aux [propriétés de gestion des polices](../Proprietes/2515045.md).


<a name="SYNTAXE2"></a>

### Définition de la police utilisée pour le titre d'un graphe (syntaxe conservée pour compatibilité)

`<Champ Graphe>.PoliceTitre(<Identifiant de la police>)`
---

**`<Champ Graphe> : (Nom de champ)`**

Nom du champ Graphe présent dans l'éditeur de fenêtres, de pages ou d'états.


**`<Identifiant de la police> : (Entier)`**

Identifiant de la police, défini avec la fonction [grCréePolice](../WDLang3/3042001.md). Si cette police n'existe pas, une erreur du WLangage est générée.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Titre d'un graphe
<a name="titre_graphe_ELTPARAGRAPHE000288"></a>Le titre du graphe peut être affiché au-dessous ou au-dessus du graphe.
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Titre.gif)
Le contenu du titre du graphe est défini par la fonction [&lt;Graphe&gt;.Titre](../WDLang3/1000024166.md). Cette fonction permet également de positionner ce titre.

La police utilisée pour le titre du graphe peut être modifiée grâce à la fonction **&lt;Graphe&gt;.PoliceTitre**.

Les données définies avec la fonction **&lt;Graphe&gt;.PoliceTitre** seront prises en compte lors du prochain appel à la fonction [&lt;Graphe&gt;.Dessine](../WDLang3/1000024062.md).
<a name="NOTE0_2"></a>


### Initialisation des polices
<a name="initialisation_des_polices_ELTPARAGRAPHE000320"></a>La police utilisée par défaut pour le titre d'un graphe a les caractéristiques suivantes :

- Nom de la police : Arial

- Taille : 14

- Style : Normal

- Couleur : Noir

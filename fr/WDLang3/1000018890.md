
## HTTPListeCertificat (Fonction)

***En anglais : HTTPListCertificate***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Liste les certificats client présents sur la machine pour s'authentifier sur un serveur. Il est possible d'utiliser un certificat spécifique grâce à la fonction [HTTPCertificat](../WDLang3/1000018898.md). 

**Exemple d'utilisation** : Cette fonction peut être utilisée par exemple pour éviter d'avoir la fenêtre standard de sélection du certificat si le site demande au client de s'authentifier. 

**Remarque** : La fonction [CertificatListe](../WDLang1/1000019293.md) permet de lister les certificats disponibles dans le magasin des certificats de Windows. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
ListeCertif est une chaîne
ListeCertif = HTTPListeCertificat()
POUR TOUTE CHAINE Certif DE ListeCertif SEPAREE PAR RC
	ListeAjoute(LISTE_Liste1, ExtraitChaîne(Certif, 1, TAB) + ...
			gValeurMémorisée(ExtraitChaîne(Certif, 3, TAB)))
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = HTTPListeCertificat()`
---

**`<Résultat> : (Chaîne de caractères)`**

Liste des certificats trouvés sur le poste en cours. Les différents certificats sont séparés par un Retour Chariot (RC). Cette liste est de la forme : 

```txt
<Nom du certificat> + TAB + <Fournisseur> + TAB + <Empreinte numérique> + RC + ...
```

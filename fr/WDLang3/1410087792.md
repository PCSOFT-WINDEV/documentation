


## grCPAjouteDonnée (Fonction)

***En anglais : grRBAddData***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ajoute une donnée à un graphe circulaire de progrès. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
grCPAjouteDonnée(GRF_Graphe1, "Montpellier", 10)
grCPAjouteDonnée(GRF_Graphe1, "France", 50)
grCPAjouteDonnée(GRF_Graphe1, "Europe", 80)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`grCPAjouteDonnée(<Nom du graphe> , <Catégorie> , <Valeur>)`
---

**`<Nom du graphe> : (Nom de champ ou chaîne de caractères)`**

Nom du graphe à manipuler. Ce nom correspond :

- soit au nom du graphe défini par programmation avec la fonction [grCrée](../WDLang3/3042005.md).

- soit au nom du champ Graphe présent dans l'éditeur de fenêtres ou d'états.




**`<Catégorie> : (Chaîne de caractères)`**

Nom de la catégorie associée à la donnée. 

**`<Valeur> : (Entier ou réel)`**

Valeur à ajouter. 
Par défaut, la valeur est considérée en pourcentage. Si ce n'est pas le cas, il est possible de donner la valeur "maximale" avec la fonction [grCPValeurMaximale](../WDLang3/1410087801.md) (le pourcentage sera automatiquement calculé à partir de cette valeur).

## Remarques


### Ajout de données dans un graphe
<a name="ajout_donnees_dans_graphe_ELTPARAGRAPHE000068"></a>La fonction [grAjouteDonnée](../WDLang3/3042023.md) permet d'ajouter des données dans un graphe (secteur, histogramme, ...).

Selon le type de graphe, des fonctions spécifiques permettent également d'ajouter des données : 

- Pour les graphes boursiers, il est conseillé d'utiliser la fonction [grBourseAjouteDonnée](../WDLang3/3042009.md).

- Pour les graphes Heatmap, il est conseillé d'utiliser la fonction [grHMAjouteDonnée](../WDLang3/1000021628.md).

- Pour les graphes de type Nuage de points, il est conseillé d'utiliser la fonction [grNuageAjouteDonnéeXY](../WDLang3/3042032.md).

- Pour les graphes Secteur Multiniveau, il est conseillé d'utiliser la fonction [grSMNAjouteDonnée](../WDLang3/1000021285.md). 

- Pour les graphes de type Surface, il est conseillé d'utiliser la fonction [grSurfaceAjouteDonnée](../WDLang3/1000020425.md).




Rappel : Pour ajouter une valeur à une donnée d'un graphe, utilisez la fonction [grIncrémenteDonnée](../WDLang3/3042030.md).

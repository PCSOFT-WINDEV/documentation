
## HTTPTimeOut (Fonction)

***En anglais : HTTPTimeOut***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Règle le timeout pour les fonctions WLangage utilisant le protocole HTTP (fonctions [HTTPRequête](../WDLang3/3043007.md), [SOAPExécute](../WDLang3/3069013.md), [DotNetExécute](../WDLang3/3031016.md), [J2EEExécute](../WDLang3/3047007.md), [webservice importé](../WDLang3/3086001.md), ...).

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Attention, si vous utilisez une variable de type [httpRequête](../WDLang3/1000021158.md), le timeout est défini par la propriété **DuréeNonRéponse** de la variable. 




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI HTTPTimeOut() > 200000 ALORS HTTPTimeOut(20000)
// Récupération du code HTML de la page Web "www.pcsoft.com"
ResLancement = HTTPRequête("http://www.pcsoft.fr")
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Lire le réglage actuel du timeout

`<Résultat> = HTTPTimeOut()`
---

**`<Résultat> : (Entier)`**

Valeur du timeout exprimée en millisecondes. Par défaut, la valeur du timeout est fixée à 200 secondes (200 000 millisecondes).  


<a name="SYNTAXE2"></a>

### Modifier le réglage du timeout

`HTTPTimeOut(<Nouveau timeout>)`
---

**`<Nouveau timeout> : (Entier ou Durée)`**

Nouvelle valeur du timeout exprimée en millisecondes. Ce paramètre peut correspondre à : 

- un entier correspondant au nombre de millisecondes, 

- une variable de type Durée, 

- l'indication directe de la durée (par exemple 20 s ou 10 ms). 


Par défaut, la valeur du timeout est fixée à 200 secondes (200 000 millisecondes).

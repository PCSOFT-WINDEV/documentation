
## SaaSAdminConnexionClient (Fonction)

***En anglais : SaaSAdminClientConnection***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la connexion à la base de données d'un compte client pour un site SaaS.

**Remarques** : 

- L'utilisation de cette fonction nécessite d'avoir ouvert une session d'administration SaaS au préalable en utilisant la fonction [SaaSAdminConnecte](../WDLang3/1000019098.md).

- L'Administrateur SaaS aura accès à toutes les connexions. Un gestionnaire de compte client aura accès uniquement à la connexion concernant son propre compte client. 



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ouverture d'une connexion à la base du client "Jet Tames" pour le site "CRM"
UnSiteSaaS est un saasSite
UnClientSaaS est un saasClient 
UneBaseClient est une Connexion

UnSiteSaaS = SaaSAdminChercheSite("CRM")
UnClientSaaS = SaaSAdminChercheClient("JetTames")
UneBaseClient = SaaSAdminConnexionClient(UnSiteSaaS, UnClientSaaS)

HOuvreConnexion(UneBaseClient)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Récupérer les paramètres de la connexion (utilisation des variables de type saasxxx)

`<Résultat> = SaaSAdminConnexionClient(<Site SaaS> , <Compte Client>)`
---

**`<Résultat> : (Chaîne de caractères ou variable de type Connexion)`**

Paramètres de connexion à la base de données du compte client.

**`<Site SaaS> : (Variable de type saasSite)`**

Nom de la variable de type [saasSite](../WDLang3/1000019068.md) correspondant au site SaaS dont les paramètres de connexion doivent être récupérés.

**`<Compte Client> : (Variable de type saasClient)`**

Nom de la variable de type [saasClient](../WDLang3/1000019072.md) correspondant au compte client SaaS dont les paramètres de connexion doivent être récupérés.


<a name="SYNTAXE2"></a>

### Récupérer les paramètres de la connexion

`<Résultat> = SaaSAdminConnexionClient(<Site> , <Client SaaS>)`
---

**`<Résultat> : (Chaîne de caractères ou variable de type Connexion)`**

Paramètres de connexion à la base de données du compte client.

**`<Site> : (Chaîne de caractères)`**

Nom du site SaaS dont les paramètres de connexion doivent être récupérés.

**`<Client SaaS> : (Chaîne de caractères)`**

Nom du compte client SaaS dont les paramètres de connexion doivent être récupérés.

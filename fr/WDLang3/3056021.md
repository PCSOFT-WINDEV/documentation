
## NetFermeAccèsDistant (Fonction)

***En anglais : NetCloseRemoteAccess***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ferme la ligne précédemment ouverte par la fonction [NetOuvreAccèsDistant](../WDLang3/3056026.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
RapportAccès est un entier = NetOuvreAccèsDistant("Ma connexion")
SI RapportAccès = 0 ALORS
	// Traitement
	...
	NetFermeAccèsDistant()
SINON
	Erreur("Erreur lors de la connexion", NetMsgErreur(RapportAccès))
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = NetFermeAccèsDistant([<Nom de l'accès>])`
---

**`<Résultat> : (Entier)`**



- 0 si la fermeture a été réalisée, 

- un code d'erreur dans le cas contraire. Ce code d'erreur peut être utilisé avec la fonction [NetMsgErreur](../WDLang3/3056019.md) afin de connaître la cause exacte de l'erreur et le message correspondant.




**`<Nom de l'accès> : (Chaîne de caractères optionnelle)`**

Nom de la connexion "Accès réseau à distance" défini dans Windows. La liste des accès distants installés sur un poste peuvent être obtenus avec la fonction [NetListeAccèsDistant](../WDLang3/3056027.md).

Si ce paramètre est précisé, l'accès distant demandé est fermé, quel que soit son mode d'ouverture (par la fonction [NetOuvreAccèsDistant](../WDLang3/3056026.md) ou non).

Si ce paramètre n'est pas précisé, seule la dernière connexion ouverte par la fonction [NetOuvreAccèsDistant](../WDLang3/3056026.md) avec le paramètre &lt;Fermeture automatique&gt; à <u><u><u><u>Vrai</u></u></u></u> est fermé.

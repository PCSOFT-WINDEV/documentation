


## grTypeTraitSérie (Fonction)

***En anglais : grSeriesLineType***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie le type de trait d'une série dans un graphe de type Courbe, Aire, Nuage de points et Radar.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Définit des traits en pointillé pour la série 2
grTypeTraitSérie(GRF_MonGraphe, 2, TraitPointillé)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`grTypeTraitSérie(<Nom du graphe> , <Numéro de la série> , <Type>)`
---

**`<Nom du graphe> : (Nom de champ ou chaîne de caractères)`**

Nom du graphe à manipuler. Ce nom correspond :

- soit au nom du graphe défini par programmation avec la fonction [grCrée](../WDLang3/3042005.md).

- soit au nom du champ Graphe (présent dans l'éditeur de fenêtres ou d'états).




**`<Numéro de la série> : (Entier)`**

Numéro de la série pour laquelle le type de trait doit être défini.

**`<Type> : (Constante de type Entier)`**

Type de trait de la série : 



|   |   |
| --- | --- |
| *TraitContinu* | Trait plein continu. |
| *TraitMixte* | Trait alternant pointillé et tiret (.-.-.-). |
| *TraitPointillé* | Trait en pointillé (.......). |
| *TraitTiret* | Trait avec des tirets (---). |

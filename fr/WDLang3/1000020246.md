
## AdresseWeb (Type de variable)

***En anglais : WebAddress***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **AdresseWeb** permet de définir toutes les caractéristiques avancées d'une adresse de site Web ainsi que son type (page personnelle, blog, ftp, etc.). Les caractéristiques de cette adresse de site peuvent être définies et modifiées à l'aide de différentes propriétés WLangage. 

Ce type de variable est utilisé par les variables de type [Contact](../WDLang3/1000020254.md).

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ouvre un navigateur sur le site web d'un contact
SI MesContacts.SiteWeb.Occurrence > 0 ALORS
	UnSite est une AdresseWeb = MesContacts.SiteWeb[1]
	NavigateurLanceAppli(UnSite.Adresse)
FIN
```

## Remarques


### Propriétés spécifiques à la description des variables de type AdresseWeb
<a name="proprietes_specifiques_description_des_variables_type_adresseweb_ELTPARAGRAPHE000048"></a>

Les propriétés suivantes peuvent être utilisées pour manipuler une adresse de site Web :

| **Nom de la propriété** | **Type manipulé** | **Effet** |
| --- | --- | --- |
| Adresse | Chaîne de caractères | Adresse web. |
| Etiquette | Chaîne de caractères | Etiquette pour les adresses de type "personnalisé" (optionnel).<br>Si la propriété **Type** ne correspond pas à la constante *webPersonnalisé*, la valeur de cette propriété sera ignorée. |
| Type | Constante de type Entier | ![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Type d'adresse web : <br><br>- *webAutre* : autre type.<br><br><br> |


<a name="NOTE0_2"></a>


### Réinitialisation
<a name="reinitialisation_ELTPARAGRAPHE000101"></a>Pour réinitialiser le contenu d'une variable de type **AdresseWeb**, il est possible d'utiliser la fonction [VariableRAZ](../WDLang1/3013069.md).




## grSurfaceSupprimeCouleurAltitude (Fonction)

***En anglais : grSurfaceDeleteAltitudeColor***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime les couleurs d'altitude spécifiées par la fonction [grSurfaceCouleurAltitude](../WDLang3/1000020427.md). 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
grSurfaceSupprimeCouleurAltitude(GRF_Graphe1)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`grSurfaceSupprimeCouleurAltitude(<Nom du graphe> [, <Altitude>])`
---

**`<Nom du graphe> : (Nom de champ ou chaîne de caractères)`**

Nom du graphe à manipuler. Ce nom correspond :

- soit au nom du graphe défini par programmation avec la fonction [grCrée](../WDLang3/3042005.md).

- soit au nom du champ Graphe (présent dans l'éditeur de fenêtres ou d'états).




**`<Altitude> : (Réel optionnel)`**

Altitude de la couleur qui doit être supprimée. Si ce paramètre n'est pas spécifié, toutes les couleurs d'altitude sont supprimées : le graphe utilise alors la couleur de la première série.

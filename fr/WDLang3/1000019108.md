
## SaaSAdminModifieService (Fonction)

***En anglais : SaaSAdminModifyService***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie un service SaaS.

**Remarques** : 

- L'utilisation de cette fonction nécessite d'avoir ouvert une session d'administration SaaS au préalable en utilisant la fonction [SaaSAdminConnecte](../WDLang3/1000019098.md).

- L'utilisation de cette fonction est réservée à l'Administrateur SaaS. Un gestionnaire de compte client ne peut pas utiliser cette fonction. 



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Renommage du service "Poster un message" du site "Forum" en "Poster des messages"
S est un saasSite dynamique 
MonService est un saasService dynamique 

S = SaaSAdminChercheSite("Forum")

MonService = SaaSAdminChercheService("Poster un message", S)
MonService.Nom = "Poster des messages"

SaaSAdminModifieService(MonService)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Renommer un service SaaS (utilisation des variables de type saasxxx)

`<Résultat> = SaaSAdminModifieService(<Service SaaS>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la modification a été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Service SaaS> : (Variable de type saasService)`**

Nom de la variable de type [saasService](../WDLang3/1000019096.md) correspondant au service à modifier.


<a name="SYNTAXE2"></a>

### Renommer un service Saas identifié par son nom

`<Résultat> = SaaSAdminModifieService(<Ancien nom du service> , <Site SaaS> , <Nouveau nom du service>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la modification a été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Ancien nom du service> : (Chaîne de caractères)`**

Ancien nom du service à modifier dans la base SaaS.

**`<Site SaaS> : (Chaîne de caractères)`**

Nom du site SaaS auquel appartient le service.

**`<Nouveau nom du service> : (Chaîne de caractères)`**

Nouveau nom à donner au service.

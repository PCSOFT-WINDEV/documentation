
## DomoEnvoie (Fonction)

***En anglais : DomoSend***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Envoie une commande à un ou plusieurs appareils électriques compatibles avec le protocole X10.

**Remarque** : Le bon fonctionnement du protocole X10 par courant porteur requiert une continuité électrique entre les appareils. Par exemple la présence de transformateur d'isolement empêche la transmission du courant porteur.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ouverture de la connexion X10
IDConnexion est un entier
IDConnexion = DomoOuvre("COM4", "A")  
// Allumage des lampes reliées aux modules A3 et A4
DomoEnvoie(IDConnexion, "A3 A4", domoMarche)
// Equivalent à : DomoAllume(IDConnexion, "A3 A4")
// Modification de l'intensité de ces lampes
DomoEnvoie(IDConnexion, "A3 A4", domoDiminue, -60)
// Equivalent à : DomoEnvoie(IdConexion, "A3 A4", domoModifie, -60)
...
// Extinction des lampes
DomoEnvoie(IDConnexion, "A3 A4", domoArrête)
// Equivalent à : DomoEteint(IDConnexion, "A3 A4")
```


<a name="Exemple2"></a>



```wl
// Extinction de tous les appareils sur A
DomoEnvoie(IDConnexion, "A", domoArrêteTout)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = DomoEnvoie(<Identifiant de la connexion> , <Adresse> , <Commande à envoyer> [, <Valeur de la commande>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la commande a bien été envoyée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire (erreur de transmission, périphérique n'ayant pas répondu, ...). En cas d'erreur, pour connaître le libellé de l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.




**`<Identifiant de la connexion> : (Entier)`**

Identifiant de la connexion à utiliser. Cet identifiant est défini grâce à la fonction [DomoOuvre](../WDLang3/1000003102.md).

**`<Adresse> : (Chaîne de caractères)`**

Adresse du ou des appareils électriques compatibles avec le protocole X10 concerné.

Le format d'une adresse est le suivant : &lt;Code maison&gt;&lt;Code module&gt;

Où :

&lt;Code maison&gt; correspond à une lettre comprise entre A et P.

&lt;Code module&gt; correspond à un chiffre compris entre 1 et 16.

Par exemple : "A1", "B14".

Pour spécifier plusieurs adresses, séparez les différentes adresses par un espace. Par exemple : "A1 A3 B15".

**`<Commande à envoyer> : (Constante de type Entier)`**

Commande à envoyer aux appareils électriques spécifiés :


|   |   |
| --- | --- |
| *domoArrête* | Arrête les appareils électriques spécifiés. |
| *domoArrêteTout* | Arrête tous les appareils électriques spécifiés. Dans ce cas, &lt;Adresse&gt; doit contenir uniquement un ou des codes maison. |
| *domoAugmente* | Augmente l'intensité des appareils électriques spécifiés de &lt;Valeur de la commande&gt; pour-cent (intensité d'éclairage des lampes généralement). |
| *domoDiminue* | Diminue l'intensité des appareils électriques spécifiés de &lt;Valeur de la commande&gt; pour-cent (intensité d'éclairage des lampes généralement). |
| *domoMarche* | Allume les appareils électriques spécifiés. |
| *domoModifie* | Modifie l'état en cours des appareils électriques spécifiés. Dans ce cas, le paramètre &lt;Valeur de la commande&gt; correspond à :<br><br>- Vrai : Allume les appareils électriques spécifiés.<br><br>- Faux : Arrête les appareils électriques spécifiés.<br><br>- &lt;Valeur positive&gt; : Augmente l'intensité des appareils électriques spécifiés de &lt;Valeur positive&gt; pour-cent<br><br>- &lt;Valeur négative&gt; : Diminue l'intensité des appareils électriques spécifiés de &lt;Valeur négative&gt; pour-cent<br><br><br> |



**`<Valeur de la commande> : (Booléen ou Entier optionnel)`**

Valeur de la commande envoyée :


|   |   |
| --- | --- |
| <u><u><u><u>Vrai</u></u></u></u> | Allume les appareils électriques spécifiés. |
| <u><u><u><u>Faux</u></u></u></u> | Arrête les appareils électriques spécifiés. |
| &lt;Valeur positive&gt; | Augmente l'intensité des appareils électriques spécifiés de &lt;Valeur positive&gt; pour-cent. |
| &lt;Valeur négative&gt; | Diminue l'intensité des appareils électriques spécifiés de &lt;Valeur négative&gt; pour-cent. |

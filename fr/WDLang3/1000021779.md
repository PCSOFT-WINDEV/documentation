
## iOSExclutFichieriCloud (Fonction)

***En anglais : iOSExcludeFileFromiCloud***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Exclut un fichier des sauvegardes réalisées par le cloud Apple. Permet notamment d'éviter la sauvegarde de très gros fichiers.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
iOSExclutFichieriCloud(fRepDonnées() + "monfichier.txt")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = iOSExclutFichieriCloud(<Chemin du fichier>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le fichier a été exclu, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Chemin du fichier> : (Chaîne de caractères)`**

Nom et chemin complet du fichier du fichier à exclure de la sauvegarde dans le cloud Apple.

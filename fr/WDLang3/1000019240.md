
## GPSArrêteDétection (Fonction)

***En anglais : GPSStopDetection***





<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Arrête la détection de position déclenchée par la fonction [GPSDétectePosition](../WDLang3/1000019209.md).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Active la notification de détection de position
nID est un entier 
nID = GPSDétectePosition(ProcDétection, 1.145258, 30.968745)

...

// Arrête la notification de détection de position
GPSArrêteDétection(nID)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`GPSArrêteDétection(<Identifiant de détection>)`
---

**`<Identifiant de détection> : (Entier)`**

Identifiant correspondant à la détection de position à interrompre. Cet identifiant est retourné par la fonction [GPSDétectePosition](../WDLang3/1000019209.md). 

Si cet identifiant ne correspond à aucune détection de position en cours, la fonction est sans effet.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Cas particuliers
<a name="cas_particuliers_ELTPARAGRAPHE000070"></a>

- Pour ne plus recevoir de notifications, utilisez la fonction [GPSTermine](../WDLang3/1000019210.md).

- Pour arrêter la détection de la proximité d'une position en particulier, il est également possible de renvoyer <u><u><u><u>Faux</u></u></u></u> dans la procédure associée (procédure définie dans la fonction [GPSDétectePosition](../WDLang3/1000019209.md)).



![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) 

### Fonctionnalité d'application requise
<a name="fonctionnalite_application_requise_ELTPARAGRAPHE000158"></a>L'utilisation de cette fonction entraîne la déclaration d'une fonctionnalité d'application dans l'assistant de génération de l'application.

Fonctionnalité requise : Géolocalisation

Cette fonctionnalité permet aux applications d'accéder aux données de géolocalisation de l'appareil ou de l'ordinateur.

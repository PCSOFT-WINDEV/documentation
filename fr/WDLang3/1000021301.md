
## notificationFormat (Type de variable)

***En anglais : notificationFormat***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
A partir d'Android 4.1 (Jelly Bean), les notifications affichées dans la liste des notifications de l'appareil peuvent être "déroulées" pour afficher plus de contenu (image, texte multiligne, etc.). 

Les notifications affichées sont automatiquement enroulées ou déroulées par le système en fonction de la place disponible dans la liste. L'utilisateur a toutefois la possibilité d'enrouler ou dérouler une notification en effectuant un geste de balayage vers le haut (enrouler) ou vers le bas (dérouler) à l'aide de deux doigts.

Le type **notificationFormat** permet de décrire le format d'affichage d'une notification lorsque celle-ci est déroulée. Les caractéristiques de ce format d'affichage peuvent être définies et modifiées à l'aide de différentes propriétés WLangage. 

Pour associer un format à une notification, il suffit d'utiliser la propriété **Format** du type [Notification](../WDLang3/1000019441.md).

**Remarques** :

- Si la notification est affichée sur un appareil en version inférieure à Android 4.1, le format sera ignoré.

- Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).







<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
n est une notification
n.Titre = "Photo enregistrée"
n.Message = "Date de la prise de vue : 18/07/2014 19:45"
n.GrandeIcone = "photo.png"
n.MessageSecondaire = "desert.png"
n.Format.Type = notifFormatImage
n.Format.Contenu = fRepEnCours + "photos/desert.png"
NotifAjoute(n)
```





<a name="NOTE0"></a>

## Remarques
<a name="NOTE0_1"></a>


### Propriétés spécifiques à la description des variables de type notificationFormat
<a name="proprietes_specifiques_description_des_variables_type_notificationformat_ELTPARAGRAPHE000042"></a>Les propriétés suivantes peuvent être utilisées pour manipuler un format de notification :

| Nom de la propriété | Type manipulé | Effet |
| --- | --- | --- |
| Contenu | Chaîne de caractères | Contenu de la notification à afficher. La valeur de cette propriété dépend du type de format utilisé (propriété **Type**). <br><br>- Si la propriété **Type** vaut *notifFormatImage*, la propriété **Contenu** correspond au chemin de l'image à afficher.<br><br>- Si la propriété **Type** vaut *notifFormatListe*, la propriété **Contenu** correspond aux éléments à afficher dans la liste, séparés par des Retours Chariot (RC). Il est possible d'afficher jusqu'à 5 éléments dans la liste. Les éléments supplémentaires seront ignorés.<br><br>- Si la propriété **Type** vaut *notifFormatTexte*, la propriété **Contenu** correspond au texte multiligne à afficher. Si le texte spécifié est trop long, il pourra être tronqué à l'affichage.<br><br><br> |
| GrandeIcone | Chaîne de caractères | Chemin de l'image correspondant à la grande icône associée à la notification. Cette icône sera affichée à gauche du titre et du message de la notification. <br><br>Si cette propriété n'est pas renseignée, la grande icône de la notification sera identique à celle utilisée lorsque la notification est enroulée.<br><br>Cette propriété est prise en compte uniquement si la propriété **Type** correspond à la constante *notifFormatImage*. |
| MessageSecondaire | Chaîne de caractères | Message secondaire de la notification. Si cette propriété n'est pas renseignée, le message secondaire de la notification sera identique à celui utilisé lorsque la notification est enroulée. |
| Titre | Chaîne de caractères | Titre de la notification lorsque celle-ci est déroulée. Si cette propriété n'est pas renseignée, le titre de la notification sera identique à celui utilisé lorsque la notification est enroulée. |
| Type | Constante de type Entier | Type de format à appliquer à la notification lorsque celle-ci est déroulée : <br><br>- *notifFormatImage* : Affichage d'une image. <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=NotifFormatimage.gif)<br><br><br>- *notifFormatListe* : Affichage d'une liste d'éléments.<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=NotifFormatListe.gif)<br><br><br>- *notifFormatTexte* (valeur par défaut) : Affichage d'un texte multiligne.<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=NotifFormatTexte.gif)<br><br><br><br> |




<a name="NOTE0_2"></a>


### Divers
<a name="divers_ELTPARAGRAPHE000102"></a>Pour réinitialiser le contenu d'une variable de type **notificationFormat**, il est possible d'utiliser la fonction [VariableRAZ](../WDLang1/3013069.md).

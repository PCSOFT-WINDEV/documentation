


## &lt;Variable upnpService&gt;.ExécuteAction (Fonction)

***En anglais : &lt;upnpService variable&gt;.ExecuteAction***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Exécute une action sur le service UPnP.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonService est un upnpService
...


SELON TABLE_ParamIN.Occurrence()
	CAS 0 
		tabRésultat = MonService.ExécuteAction(SAI_Action)
		SI ErreurDétectée ALORS
			Info(ErreurInfo(errComplet))
			RETOUR
		FIN
	CAS 1
		tabRésultat = MonService.ExécuteAction(SAI_Action, ...
				TABLE_ParamIN[1].COL_Param)
		SI ErreurDétectée ALORS
			Info(ErreurInfo(errComplet))
			RETOUR
		FIN
	CAS 2
		tabRésultat = MonService.ExécuteAction(SAI_Action, ...
				TABLE_ParamIN[1].COL_Param, ...
				TABLE_ParamIN[2].COL_Param)
		SI ErreurDétectée ALORS
			Info(ErreurInfo(errComplet))
			RETOUR
		FIN
	AUTRE CAS
		Info("Non géré")	
		RETOUR
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Service>.ExécuteAction(<Action> [, <Paramètre 1> [... [, <Paramètre N>]]])`
---

**`<Résultat> : (Tableau de chaîne de caractères)`**

Variable de type Tableau de chaînes contenant le résultat de l'action UPnP exécutée.

**`<Service> : (Variable de type upnpService)`**

Nom de la variable de type [upnpService](../WDLang3/1000020908.md) correspondant au service sur lequel l'action doit être effectuée.

**`<Action> : (Chaîne de caractères)`**

Action à appeler sur le service.

**`<Paramètre 1> : (Type de la valeur attendu (optionnel))`**

Premier paramètre à passer au service UPnP.



**`<Paramètre N> : (Type de la valeur attendu (optionnel))`**

Nième paramètre à passer au service UPnP.


<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Si l'action n'est pas exécutée, la variable [ErreurDétecté](../WDLang1/3087001.md) est à <u><u><u><u>Vrai</u></u></u></u> et la fonction [ErreurInfo](../WDLang1/3013008.md) permet de connaître le détail de l'erreur.

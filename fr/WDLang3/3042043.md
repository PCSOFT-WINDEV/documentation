


## grOrientation (Fonction)

***En anglais : grOrientation***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie l'orientation des axes d'un graphe. 



![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Les graphes](../WDChamp/3042060.md). 








<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Inverse l'axe des abscisses et des ordonnées
// Graphe créé par programmation
grOrientation("MonGraphe", grOrdonnée, grAGauche, grEnBas)
```
<a name="Exemple2"></a>

```wl
// Inverse l'axe des abscisses et des ordonnées
// Champ Graphe
grOrientation(GRF_MonGraphe, grOrdonnée, grAGauche, grEnBas)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`grOrientation(<Nom du graphe> [, <Premier axe> [, <Orientation axe horizontal> [, <Orientation axe vertical>]]])`
---

**`<Nom du graphe> : (Nom de champ ou chaîne de caractères)`**

Nom du graphe à manipuler. Ce nom correspond :

- soit au nom du graphe défini par programmation avec la fonction [grCrée](../WDLang3/3042005.md).

- soit au nom du champ Graphe présent dans l'éditeur de fenêtres, de pages ou d'états.




**`<Premier axe> : (Constante optionnelle)`**

Type de coordonnées à afficher sur l'axe horizontal :


|   |   |
| --- | --- |
| *grAbscisse*<br>(valeur par défaut) | Les abscisses sont affichées sur l'axe horizontal. |
| *grOrdonnée* | Les ordonnées sont affichées sur l'axe vertical. |



**`<Orientation axe horizontal> : (Constante optionnelle)`**

Orientation de l'axe horizontal :


|   |   |
| --- | --- |
| *grADroite*<br>(valeur par défaut) | L'axe horizontal est orienté vers la droite (les valeurs négatives sont à gauche et les valeurs positives à droite). |
| *grAGauche* | L'axe horizontal est orienté vers la gauche (les valeurs négatives sont à droite et les valeurs positives à gauche). |



**`<Orientation axe vertical> : (Constante optionnelle)`**

Orientation de l'axe vertical :


|   |   |
| --- | --- |
| *grEnBas* | L'axe vertical est orienté vers le bas (les valeurs négatives sont en haut et les valeurs positives en bas). |
| *grEnHaut*<br>(valeur par défaut) | L'axe vertical est orienté vers le haut (les valeurs négatives sont en bas et les valeurs positives en haut). |

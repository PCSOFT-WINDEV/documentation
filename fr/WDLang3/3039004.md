


## Fonctions de gestion des flux (vidéo et son)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions de gestion des flux sont les suivantes :



|   |   |
| --- | --- |
| [FluxAccepte](../WDLang3/3039003.md) | Associe une demande de connexion de flux à un champ Conférence. |
| [FluxChangeDriver](../WDLang3/1000021613.md) | Indique la caméra à utiliser comme source pour un flux et/ou un champ Conférence. |
| [FluxConfigure](../WDLang3/1000021645.md) | Spécifie les paramètres réseau permettant d'utiliser des champs Conférence sur Internet. |
| [FluxConnecte](../WDLang3/3039008.md) | Connecte un flux (et le champ associé) à une adresse spécifique. |
| [FluxDéconnecte](../WDLang3/3039007.md) | Arrête la réception ou l'émission d'un flux. |
| [FluxInfo](../WDLang3/3039009.md) | Renvoie le nom et / ou le numéro de l'émetteur du flux. |
| [FluxJoue](../WDLang3/3039010.md) | Reprend la réception ou l'émission d'un type de données dans un flux. |
| [FluxRefuse](../WDLang3/3039006.md) | Refuse une demande de connexion d'un flux. |
| [FluxStop](../WDLang3/3039002.md) | Arrête la réception ou l'émission d'un type de données dans un flux. |

- ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=WDVisioconference.gif) ***Exemples complets (WINDEV)*** : **WD Visioconférence** <br>Cet exemple utilise le champ Conférence et illustre aussi l'utilisation des sockets grâce à une messagerie instantanée.<br>WD VisioConférence permet d'entrer en conférence (son + vidéo) avec un utilisateur distant. De plus grâce à la messagerie instantanée, vous avez la possibilité d'envoyer un fichier à votre correspondant et de dialoguer simplement avec lui.




## Manipuler des fax reçus ou envoyés
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000130"></a>
Pour manipuler des fax directement depuis une application ou un site, WINDEV et WEBDEV mettent à votre disposition des fonctions permettant de manipuler des fax, par l'intermédiaire du serveur de fax système disponible sous Windows XP et Windows 2000. Sous Windows Vista, la gestion des fax est disponible uniquement avec Windows Vista Professionnel et Windows Vista Edition Intégrale.

Rappel : il est également possible d'envoyer des fax depuis une application WINDEV ou un site WEBDEV. Pour plus de détails, consultez [Envoyer des fax](../WDLang3/3037002.md).

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment manipuler des fax ?
<a name="comment_manipuler_des_fax_ELTTEXTE000154"></a>
Pour manipuler des fax (reçus ou envoyés) depuis une application WINDEV ou un site WEBDEV, il faut :

- [configurer le poste "Serveur de fax"](../WDLang3/3037016.md).

- créer l'application WINDEV/WEBDEV. 

- [configurer si nécessaire par programmation les options du serveur de fax](../WDLang3/3037016.md).




<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Programmer la manipulation de fax
<a name="programmer_manipulation_fax_ELTTEXTE000178"></a>
Pour manipuler un fax (reçu ou envoyé) depuis une application WINDEV, il faut : 

1. Lire le contenu de la boîte de réception des fax (fonction [FaxBoîteRéception](../WDLang3/3037009.md)) ou le contenu de la boîte d'envoi (fonction [FaxBoîteEnvoi](../WDLang3/3037008.md)). Ces fonctions permettent de récupérer les identifiants numériques des fax reçus ou en attente.

2. A partir de l'identifiant du fax, il est possible de :

	- Connaître l'état du fax, et ses caractéristiques (fonction [FaxEtat](../WDLang3/3037001.md) et structure [FaxEtatComplet](../WDLang3/3037011.md)).

	- Manipuler le fax grâce aux fonctions suivantes :
			


|   |   |
| --- | --- |
| [FaxRéinitialise](../WDLang3/3037012.md) | Redémarre un fax dans la file d'attente. Cette fonction ne peut être utilisée que sur les fax en attente de transmission. |
| [FaxReprend](../WDLang3/3037013.md) | Ré-active un fax mis en pause dans la file d'attente (fax mis en attente de transmission). |
| [FaxSupprime](../WDLang3/3037014.md) | Supprime un fax de la file d'attente et annule son envoi. Il est impossible d'utiliser ensuite la fonction [FaxEtat](../WDLang3/3037001.md) sur ce fax. |
| [FaxSuspend](../WDLang3/3037015.md) | Met en pause un fax dans la file d'attente. Si le fax est actif, il sera mis en pause lorsqu'il retournera dans la file d'attente. |

Le fax peut être envoyé grâce à un serveur de fax mais aussi par un modem correctement installé et configuré sur le poste de l'utilisateur.




## grNiveauTemporelMax (Fonction)

***En anglais : grTimeLevelMax***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Définit le nombre de niveaux à afficher dans l'échelle temporelle d'un graphe ainsi que le niveau maximum.

Remarque : Cette fonction peut être utilisée avec la fonction [grNiveauTemporelMasque](../WDLang3/1000023017.md) qui permet de définir le masque d'un niveau de l'échelle temporelle du graphe.  
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Fixe le nombre de niveaux maximum à afficher à 2
grNiveauTemporelMax(GRF_MonGraphe, 2)

// Interdit l'affichage des années pour GRF_MonGraphe
grNiveauTemporelMax(GRF_MonGraphe, 3, grntMois)
```

```wl
// Définition de l'échelle temporelle pour afficher plusieurs valeur par mois
// mais en affichant uniquement les mois dans l'échelle 
// (les jours ne sont pas affichés)
grNiveauTemporelMasque(GRF_MonGraphe, grntMois, "Mmm")
grNiveauTemporelMax(GRF_MonGraphe, 1, grntMois)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`grNiveauTemporelMax(<Nom du graphe> , <Nombre de niveaux> [, <Niveau maximum>])`
---

**`<Nom du graphe> : (Nom de champ ou chaîne de caractères)`**

Nom du graphe à manipuler. Ce nom correspond :

- soit au nom du graphe défini par programmation avec la fonction [grCrée](../WDLang3/3042005.md).

- soit au nom du champ Graphe présent dans l'éditeur de fenêtres ou de pages.




**`<Nombre de niveaux> : (Entier)`**

Nombre de niveaux maximum affichés dans le graphe. 

**`<Niveau maximum> : (Constante optionnelle de type Entier)`**

Niveau maximum à afficher : 


|   |   |
| --- | --- |
| *grntAnnée* | Niveau maximum affiché : les années. |
| *grntHeure* | Niveau maximum affiché : les heures. |
| *grntJour* | Niveau maximum affiché : les jours. |
| *grntMilliseconde* | Niveau maximum affiché : les millisecondes. |
| *grntMinute* | Niveau maximum affiché : les minutes. |
| *grntMois* | Niveau maximum affiché : les mois. |
| *grntSeconde* | Niveau maximum affiché : les secondes. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Lors de l'affichage d'une échelle temporelle sur des données de type DateHeure, le grapheur dessine plusieurs niveaux correspondant aux différentes unités temporelles affichées. 

- Le niveau minimum à afficher est déterminé en fonction de la plage de données.

- Le niveau maximum est déterminé en utilisant deux paramètres :

	- Le nombre de niveaux maximum à afficher dans le graphe (par défaut : 3). 

	- Le niveau maximum autorisé pour l'affichage (par défaut : année).





La fonction **grNiveauTemporelMax** permet de modifier ces paramètres. Il est ainsi possible de ne pas surcharger le graphe ou de ne pas afficher certains niveaux qui n'ont pas de sens pour les données affichées.

Par exemple : 

- pour ne jamais afficher les années, il suffit de régler le niveau maximum avec la constante *grntMois*.

- pour ne jamais afficher plus de deux niveaux temporels dans le graphe, il suffit de régler le paramètre &lt;Nombre de niveaux&gt; à 2.

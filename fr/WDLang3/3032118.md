


## Fonctions LDAP
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions LDAP disponibles sont les suivantes :



|   |   |
| --- | --- |
| [LDAPAjouteAttribut](../WDLang3/3032110.md) | Ajoute un attribut ou ajoute une nouvelle valeur à un attribut existant. |
| [LDAPConnecte](../WDLang3/3032100.md) | Permet de se connecter à un serveur LDAP grâce aux caractéristiques définies dans la [structure LDAPSession](../WDLang3/3032101.md). |
| [LDAPDébutAjout](../WDLang3/3032109.md) | Initialise l'ajout d'un nouvel objet dans un serveur LDAP. |
| [LDAPDébutModification](../WDLang3/3032112.md) | Initialise la modification d'un objet existant dans un serveur LDAP. |
| [LDAPDéconnecte](../WDLang3/3032102.md) | Permet de se déconnecter d'un serveur LDAP spécifique. |
| [LDAPListeAttribut](../WDLang3/3032104.md) | Liste les attributs d'un objet d'un serveur LDAP. |
| [LDAPListeFils](../WDLang3/3032105.md) | Liste les fils d'un objet d'un serveur LDAP. |
| [LDAPMode](../WDLang3/3032126.md) | Modifie le mode de fonctionnement d'une session LDAP. |
| [LDAPNbValeur](../WDLang3/3032106.md) | Renvoie le nombre de valeurs d'un attribut. |
| [LDAPRAZ](../WDLang3/3032103.md) | Ré-initialise la [structure LDAPSession](../WDLang3/3032101.md) avec les valeurs par défaut. |
| [LDAPRecherche](../WDLang3/3032108.md) | Lance une recherche dans un serveur LDAP. |
| [LDAPRemplaceAttribut](../WDLang3/3032113.md) | Remplace toutes les valeurs d'un attribut par une valeur donnée. |
| [LDAPRemplaceValeurAttribut](../WDLang3/3032119.md) | Remplace une valeur donnée d'un attribut par une nouvelle valeur donnée. |
| [LDAPRenommeAttribut](../WDLang3/3032120.md) | Renomme un attribut. |
| [LDAPSupprime](../WDLang3/3032114.md) | Supprime un objet du serveur LDAP. |
| [LDAPSupprimeAttribut](../WDLang3/3032115.md) | Supprime un attribut et toutes ses valeurs. |
| [LDAPSupprimeValeurAttribut](../WDLang3/3032116.md) | Supprime une valeur d'un attribut spécifié. |
| [LDAPValeur](../WDLang3/3032107.md) | Renvoie la valeur correspondant à l'indice passé en paramètre pour l'attribut spécifié. |
| [LDAPValideAjout](../WDLang3/3032111.md) | Valide l'ajout d'un nouvel objet dans un serveur LDAP. |
| [LDAPValideModification](../WDLang3/3032117.md) | Valide la modification d'un objet existant dans un  serveur LDAP. |





Autres fonctions liées à la gestion du LDAP : 



|   |   |
| --- | --- |
| [gpwEnModeLDAP](../WDLang6/1000017056.md) | Permet de savoir si l'identification des utilisateurs dans le groupware utilisateur sera réalisée via un annuaire LDAP. |
| [gpwImportUtilisateursLDAP](../WDLang6/1000017058.md) | Importe les utilisateurs enregistrés dans un annuaire LDAP dans le fichier des utilisateurs du groupware utilisateur. |
| [gpwOuvreSessionLDAP](../WDLang6/1000017055.md) | Indique au groupware utilisateur que l'authentification des utilisateurs sera réalisée via un annuaire LDAP. |
| [gpwRecupInfoLDAP](../WDLang6/1000017063.md) | Récupère des les caractéristiques du serveur LDAP utilisé pour le groupware utilisateur.  |

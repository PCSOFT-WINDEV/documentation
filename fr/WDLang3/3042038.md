


## grNuageEtiquetteSérie (Fonction)

***En anglais : grScatterSeriesLabel***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Initialise l'étiquette d'une série de données dans un graphe de type "Nuage de points".

Exemple d'étiquette de série : ![Etiquette de série](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=EtiquetteSerie.gif)


**Rappel** : Par défaut, un graphe n'a aucune étiquette.



![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Les graphes](../WDChamp/3042060.md). 








<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Initialise l'étiquette "Vente" de la série de données 2 dans le graphe "MonGraphe"
grNuageEtiquetteSérie("MonGraphe", 2, "Vente")
```
<a name="Exemple2"></a>

```wl
// Initialise l'étiquette "Achat" de la série de données 3 dans le champ GRF_MonGraphe
grNuageEtiquetteSérie(GRF_MonGraphe, 3, "Achat")
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`grNuageEtiquetteSérie(<Nom du graphe> , <Numéro de la série> , <Étiquette>)`
---

**`<Nom du graphe> : (Nom de champ ou chaîne de caractères)`**

Nom du graphe à manipuler. Ce nom correspond :

- soit au nom du graphe défini par programmation avec la fonction [grCrée](../WDLang3/3042005.md).

- soit au nom du champ Graphe présent dans l'éditeur de fenêtres, de pages ou d'états.




**`<Numéro de la série> : (Entier)`**

Numéro de la série pour laquelle une étiquette doit être initialisée.

**`<Étiquette> : (Chaîne de caractères)`**

Libellé de l'étiquette à initialiser.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Il est possible de modifier la police de l'étiquette grâce à la fonction [grPoliceEtiquette](../WDLang3/3042012.md).

- Les données définies avec la fonction **grNuageEtiquetteSérie** seront prises en compte lors du prochain appel à la fonction [grDessine](../WDLang3/3042026.md).

- La fonction **grNuageEtiquetteSérie** est équivalente à la fonction [grEtiquetteSérie](../WDLang3/3042024.md). 
	La syntaxe de [grEtiquetteSérie](../WDLang3/3042024.md) utilisée avec des graphes de type "Nuage de points" est la suivante :
	
	```txt
	grEtiquetteSérie(<Nom graphe>, 2*<Indice>-1, <Étiquette>)
	```
**Rappel** : Un graphe de type "Nuage de points" utilise les données par groupe de deux séries (pour plus de détails, consultez la fonction [grNuageAjouteDonnéeXY](../WDLang3/3042032.md)).

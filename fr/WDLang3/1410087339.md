
## Constantes_telechargement
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les constantes utilisées par les fonctions de gestion du téléchargement sont les suivantes :

| Constante | Détail | Type de variable |
| --- | --- | --- |
| **téléchargementErreurStockage** | Erreur rencontrée : Erreur liée au stockage. | Type [TéléchargementInfo](../WDLang3/1410087306.md) |
| **téléchargementErreurInconnu** | Erreur rencontrée : L'erreur n'est pas connue. | Type [TéléchargementInfo](../WDLang3/1410087306.md) |
| **téléchargementErreurEspaceInsuffisant** | Erreur rencontrée : L'espace de stockage est insuffisant. | Type [TéléchargementInfo](../WDLang3/1410087306.md) |
| **téléchargementErreurFichierExistant** | Erreur rencontrée : L'URI pointe vers un fichier existant. | Type [TéléchargementInfo](../WDLang3/1410087306.md) |
| **téléchargementPauseAttentePourWifi** | La raison de la pause correspond à l'attente d'être en réseau WIFI. | Type [TéléchargementInfo](../WDLang3/1410087306.md) |
| **téléchargementPauseAttenteRéseau** | La raison de la pause correspond à l'attente d'une connexion réseau. | Type [TéléchargementInfo](../WDLang3/1410087306.md) |
| **téléchargementPauseInconnu** | La raison de la pause est inconnue. | Type [TéléchargementInfo](../WDLang3/1410087306.md) |
| **téléchargementPauseAttenteNouvelEssai** | La raison de la pause est une erreur réseau, le système attend avant de réessayer. | Type [TéléchargementInfo](../WDLang3/1410087306.md) |
| **téléchargementNotifAucune** | Mode de notification du téléchargement : Aucune notification n'est visible. | Type [TéléchargementParamètre](../WDLang3/1410087293.md) |
| **téléchargementNotifProgression** | Mode de notification du téléchargement : Tant que le téléchargement est en cours, une notification montre la progression et disparaît lorsque le téléchargement est terminé. | Type [TéléchargementParamètre](../WDLang3/1410087293.md) |
| **téléchargementNotifProgressionEtFin** | Mode de notification du téléchargement : Tant que le téléchargement est en cours, une notification montre la progression et reste visible lorsque le téléchargement est terminé. | Type [TéléchargementParamètre](../WDLang3/1410087293.md) |
| **téléchargementEtatEchec** | Statut du téléchargement :  Le téléchargement a échoué. | Type [TéléchargementInfo](../WDLang3/1410087306.md) |
| **téléchargementEtatSuccès** | Statut du téléchargement : Le téléchargement a réussi. | Type [TéléchargementInfo](../WDLang3/1410087306.md) |
| **téléchargementEtatEnAttente** | Statut du téléchargement : Le téléchargement est en attente. | Type [TéléchargementInfo](../WDLang3/1410087306.md) |
| **téléchargementEtatEnCours** | Statut du téléchargement : Le téléchargement est en cours. | Type [TéléchargementInfo](../WDLang3/1410087306.md) |
| **téléchargementEtatPause** | Statut du téléchargement : Le téléchargement est en pause. | Type [TéléchargementInfo](../WDLang3/1410087306.md) |

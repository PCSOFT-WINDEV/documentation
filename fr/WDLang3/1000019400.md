
## DNSAjouteAdresse (Fonction)

***En anglais : DNSAddAddress***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ajoute une relation IP / Nom dans le serveur DNS.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
c est une Durée
c..EnJours = 1
ResultDNS est un booléen

// Ajout dans un serveur DNS spécifique
ResultDNS = DNSAjouteAdresse("testdoc.doc.local.", "192.168.1.1", c, "Andromede_2")

SI ResultDNS = Faux ALORS
	Erreur(ErreurInfo(errComplet))
FIN

// Ajout dans le serveur DNS par défaut
ResultDNS = DNSAjouteAdresse("testAp.towerqual.local.", "192.168.1.2")
SI ResultDNS = Faux ALORS
	Erreur(ErreurInfo(errComplet))
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = DNSAjouteAdresse(<Nom> , <Adresse IP> [, <Durée de validité> [, <Serveur DNS> [, <Utilisateur> [, <Mot de passe>]]]])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'ajout de l'adresse DNS a réussi, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. La fonction [ErreurInfo](../WDLang1/3013008.md) permet d'obtenir plus de renseignements sur le problème rencontré. 




**`<Nom> : (Chaîne de caractères Unicode)`**

Nom complet à associer à l'adresse IP. Ce paramètre est de la forme : &lt;Nom&gt; + "." + &lt;Domaine&gt;. Exemple : "testdoc.pcsoft.com"

**`<Adresse IP> : (Chaîne de caractères)`**

Adresse IP à associer au &lt;Nom&gt;. Cette adresse IP peut être une adresse IP version 4 (au format xxx.xxx.xxx.xxx) ou version 6 (au format aaaa :bbbb :cccc :dddd :eeee :ffff :gggg :hhhh :iiii :jjjj).

**`<Durée de validité> : (Durée optionnelle)`**

Durée (en secondes) de validité de la requête. Ce paramètre peut correspondre à : 

- un entier correspondant au nombre de secondes, 

- une variable de type Durée, 

- l'indication directe de la durée (par exemple 1 s). 


Par défaut, ce paramètre vaut 0. 

**`<Serveur DNS> : (Chaîne de caractères optionnelle)`**

Adresse du serveur DNS à mettre à jour. L'association "Adresse IP / Nom" sera ajoutée à ce serveur. Par défaut, le serveur DNS actuel est utilisé. 

**`<Utilisateur> : (Chaîne de caractères Unicode optionnelle)`**

Nom de l'utilisateur utilisé pour signer la requête. 

**`<Mot de passe> : (Chaîne de caractères Unicode optionnelle)`**

Mot de passe associé à l'utilisateur pour signer la requête. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Cette fonction gère les serveurs DNS Linux et Windows ne nécessitant pas de signature des requêtes.

- Cette fonction gère les serveurs DNS Windows nécessitant une signature des requêtes.

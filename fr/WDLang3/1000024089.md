


## &lt;Graphe&gt;.Légende (Fonction)

***En anglais : &lt;Chart&gt;.Legend***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Indique la présence et la position de la légende dans un graphe.

Exemple de légende dans un graphe : ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Legende.gif)


**Remarque** : La légende regroupe les couleurs et les étiquettes des séries (ou de secteurs) d'un graphe. 



![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Les graphes](../WDChamp/3042060.md). 






<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Manipulation d'un champ Graphe
// La légende est affichée à gauche du graphe GRF_MonGraphe
GRF_MonGraphe.Légende(grAGauche)
...
GRF_MonGraphe.Dessine()
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Champ Graphe>.Légende(<Position>)`
---

**`<Champ Graphe> : (Nom de champ)`**

Nom du champ Graphe à manipuler (présent dans l'éditeur de fenêtres, de pages ou d'états).


**`<Position> : (Entier)`**

Position de la légende du graphe :


|   |   |
| --- | --- |
| *grADroite* | Légende affichée à droite du graphe. |
| *grAGauche* | Légende affichée à gauche du graphe. |
| *grEnBas* | Légende affichée en-dessous du graphe. |
| *grEnHaut* | Légende affichée au-dessus du graphe. |
| *grSans* | Pas de légende. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Légende d'un graphe
<a name="legende_graphe_ELTPARAGRAPHE000277"></a>L'ensemble des étiquettes de séries correspond à la légende du graphe.
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=EtiquetteSerie.gif)
Les étiquettes de série ne sont pas utilisables avec les graphes de type Secteur.

La définition des étiquettes de séries est réalisée avec la fonction [&lt;Graphe&gt;.EtiquetteSérie](../WDLang3/1000024073.md). 

Il est possible de modifier la police de la légende grâce à la fonction [&lt;Graphe&gt;.PoliceLégende](../WDLang3/1000024118.md).  

La fonction **&lt;Graphe&gt;.Légende** permet d'indiquer la présence et la position de la légende (c'est-à-dire de l'ensemble des étiquettes de série). Les modifications effectuées avec la fonction **&lt;Graphe&gt;.Légende** seront prises en compte lors du prochain appel à la fonction [&lt;Graphe&gt;.Dessine](../WDLang3/1000024062.md). Il est possible de modifier la position de la légende d'un graphe à partir d'un menu contextuel.

**Remarque** : Pour définir des étiquettes de série dans des graphes de type "Nuage de points", il est conseillé d'utiliser la fonction [&lt;Graphe&gt;.NuageEtiquetteSérie](../WDLang3/1000024107.md). Pour plus de détails, consultez [Légendes d'un graphe](../WDChamp/3042069.md).
<a name="NOTE0_2"></a>


### Positionnement manuel de la légende : grLégende sans effet
<a name="positionnement_manuel_legende_grlegende_sans_effet_ELTPARAGRAPHE000326"></a>Si la légende est positionnée manuellement (option "Positionnement manuel" dans la fenêtre de la légende du graphe), la fonction **&lt;Graphe&gt;.Légende** n'a plus d'effet.

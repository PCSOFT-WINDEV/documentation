
## DNSListeAdresse (Fonction)

***En anglais : DNSListAddress***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Liste les adresses IP associées à un nom dans le serveur DNS.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
ListeIP est une chaîne 
ListeIP = DNSListeAdresse("testdoc.doc.local", "Andromede_2")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = DNSListeAdresse(<Nom> [, <Serveur DNS>])`
---

**`<Résultat> : (Chaîne de caractères)`**

Liste des adresses IP séparées par RC (Retours Chariot).

**`<Nom> : (Chaîne de caractères Unicode)`**

Nom complet sur lequel les adresses IP doivent être récupérées. Ce paramètre est de la forme : &lt;Nom&gt; + "." + &lt;Domaine&gt;. 
Exemple : "testdoc.pcsoft.com"

**`<Serveur DNS> : (Chaîne de caractères optionnelle)`**

Adresse du serveur DNS à manipuler. Par défaut, le serveur DNS actuel est utilisé.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Cette fonction gère les serveurs DNS Linux et Windows ne nécessitant pas de signature des requêtes.

- Cette fonction gère les serveurs DNS Windows nécessitant une signature des requêtes.

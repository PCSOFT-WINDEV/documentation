


## Procédure WLangage appelée par la fonction GPSDétectePosition



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Procédure WLangage (également nommée "callback") appelée par la fonction [GPSDétectePosition](../WDLang3/1000019209.md) lorsque le périphérique entre ou sort de la région délimitée par les paramètres spécifiés. Cette procédure peut être une procédure locale, globale ou interne.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Active la demande de notification
GPSDétectePosition(ProcDétection, 1.145258, 30.968745)
```

```wl
// Procédure appelée lors de la notification
PROCEDURE ProcDétection(Entrée est un booléen)
SI Entrée = Vrai ALORS
	Info("Vous êtes arrivé à destination.")
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = GPSDétectePosition_Callback(<Entrée> , <Identifiant de détection>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> pour continuer la détection de la position spécifiée par la fonction [GPSDétectePosition](../WDLang3/1000019209.md), 

- <u><u><u><u>Faux</u></u></u></u> pour arrêter la détection de la position spécifiée par la fonction [GPSDétectePosition](../WDLang3/1000019209.md).




**`<Entrée> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le périphérique entre dans la région délimitée par les paramètres spécifiés par la fonction [GPSDétectePosition](../WDLang3/1000019209.md), 

- <u><u><u><u>Faux</u></u></u></u> s'il en sort.




**`<Identifiant de détection> : (Entier)`**

Valeur renvoyée par la fonction [GPSDétectePosition](../WDLang3/1000019209.md). Cet identifiant permet de différencier les positions détectées lorsqu'une même procédure est utilisée pour la détection de plusieurs positions.


## SaaSAdminSupprimeUtilisateur (Fonction)

***En anglais : SaaSAdminDeleteUser***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime un utilisateur SaaS.

**Remarques** : 

- L'utilisation de cette fonction nécessite d'avoir ouvert une session d'administration SaaS au préalable en utilisant la fonction [SaaSAdminConnecte](../WDLang3/1000019098.md).

- L'Administrateur SaaS gère tous les utilisateurs de tous les clients. Un gestionnaire de compte client ne peut supprimer des utilisateurs que dans son propre compte client. 



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Suppression de l'utilisateur marc (du compte client Hybrid)
MonUtilisateur est un saasUtilisateur

MonUtilisateur = SaaSAdminChercheUtilisateur("marc@hybrid.com")

SI SaaSAdminSupprimeUtilisateur(MonUtilisateur) = Vrai ALORS
	Info("Utilisateur supprimé")
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = SaaSAdminSupprimeUtilisateur(<Utilisateur>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la suppression a été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Utilisateur> : (Variable de type saasUtilisateur ou chaîne de caractères)`**

Utilisateur SaaS (ou login de l'utilisateur SaaS) à supprimer de la base SaaS.

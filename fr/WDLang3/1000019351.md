
## WiFiAjouteRéseau (Fonction)

***En anglais : WiFiAddNetwork***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ajoute un nouveau réseau à la liste des réseaux Wi-Fi configurés de l'appareil. La fonction [WiFiConnecte](../WDLang3/1000019352.md) permet ensuite de connecter l'appareil au réseau Wi-Fi ajouté. 

**Remarque** : L'accès Wi-Fi n'est pas disponible dans l'émulateur Android.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
Identifiant est une chaîne 
Identifiant = WiFiAjouteRéseau("SSID", "BSSID", wifiSécuritéWPA, "mdp", Vrai)
SI Identifiant <> " "  ALORS
	WiFiConnecte(Identifiant)
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = WiFiAjouteReseau(<SSID> , <BSSID> , <Sécurité> [, <Clé> [, <Permanent>]])`
---

**`<Résultat> : (Chaîne de caractères)`**



- Identifiant réseau du nouveau réseau. 

- Chaîne vide si le réseau n'a pas pu être créé. La fonction [ErreurInfo](../WDLang1/3013008.md) permet d'obtenir plus de renseignements sur le problème rencontré.




**Remarque** : Si le réseau ajouté correspond à un réseau déjà configuré sur l'appareil, &lt;Résultat&gt;correspond à l'identifiant du réseau existant.

**`<SSID> : (Chaîne de caractères)`**

SSID (Service Set Identifier) du réseau Wi-Fi.

**`<BSSID> : (Chaîne de caractères)`**

BSSID (Basic Service Set Identifier) du point d'accès associé au réseau.

**`<Sécurité> : (Constante de type chaîne de caractères)`**

Type de sécurité utilisé par le réseau Wi-Fi : 


|   |   |
| --- | --- |
| *wifiSécuritéAucune* | Aucune sécurité n'est utilisée. |
| *wifiSécuritéIEEE8021X* | Sécurité utilisant la norme IEEE 802.1x. |
| *wifiSécuritéWEP* | Sécurité basée sur une clé WEP (Wired Equivalent Privacy) |
| *wifiSécuritéWPA2* | Sécurité utilisant le mécanisme WPA2 (Wi-Fi Protected Access). |
| *wifiSécuritéWPA* | Sécurité utilisant le mécanisme WPA (Wi-Fi Protected Access). |
| *wifiSécuritéWPA_EAP* | Sécurité utilisant les mécanismes WPA (Wi-Fi Protected Access) et EAP (Extensible Authentification Protocol). |





**`<Clé> : (Chaîne de caractères optionnelle)`**

Clé utilisée pour se connecter au réseau sécurisé. Si le protocole de sécurité spécifié est de type WPA ou WPA2, la clé est obligatoire.

**`<Permanent> : (Booléen optionnel)`**

Indique si l'ajout du réseau est permanent :	

- <u><u><u><u>Vrai</u></u></u></u> : le réseau ajouté sera mémorisé de façon permanente sur l'appareil. 

- <u><u><u><u>Faux</u></u></u></u> (valeur par défaut) : le réseau ajouté ne sera pas conservé lors du prochaine redémarrage de l'appareil.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Cas particuliers
<a name="cas_particuliers_ELTPARAGRAPHE000098"></a>

- Si le réseau ajouté est mémorisé de façon permanente sur l'appareil (paramètre &lt;Permanent&gt; à <u><u><u><u>Vrai</u></u></u></u>), les identifiants attribués à chacun des réseaux peuvent être modifiés après l'exécution de la fonction **WiFiAjouteRéseau**. Utilisez la fonction [WiFiListeRéseau](../WDLang3/1000019356.md) pour connaître les nouveaux identifiants.

- Pour supprimer un réseau de la liste des réseaux configurés de l'appareil, utilisez la fonction [WiFiSupprimeRéseau](../WDLang3/1000019357.md).



<a name="NOTE0_2"></a>


### Permissions requises
<a name="permissions_requises_ELTPARAGRAPHE000119"></a>L'utilisation de cette fonction entraîne la modification des permissions requises par l'application.

Permissions requises :  

- ACCESS_WIFI_STATE : permet aux applications d'accéder à des informations concernant les réseaux Wi-Fi. 

- CHANGE_WIFI_STATE : permet aux applications de modifier l'état de connectivité Wi-Fi.

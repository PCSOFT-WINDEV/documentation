


## &lt;Graphe&gt;.CouleurPoint (Fonction)

***En anglais : &lt;Chart&gt;.PointColor***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Initialise la couleur d'un point particulier sur un graphe de type :

- Courbe,

- Nuage de points,

- Graphe à bulles.





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Change le point 2 de la série 1
GRF_MonGraphe.CouleurPoint(1, 2, RVB(200,150,45))
GRF_MonGraphe.Dessine()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Champ Graphe>.CouleurPoint(<Numéro série> , <Numéro catégorie> , <Couleur de fond> [, <Couleur de bordure>])`
---

**`<Champ Graphe> : (Nom de champ)`**

Nom du champ Graphe à manipuler (présent dans l'éditeur de fenêtres, de pages ou d'états).



**`<Numéro série> : (Entier)`**

Numéro de la série à manipuler.

**`<Numéro catégorie> : (Entier)`**

Numéro de la catégorie à modifier.

**`<Couleur de fond> : (Entier ou constante de type Entier)`**

Couleur de fond du point. Cette couleur peut correspondre à :

- une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)),

- une couleur TSL (obtenue avec la fonction [TSL](../WDLang1/3029057.md)),

- [une couleur prédéfinie](../WDLang5/3010002.md).




**`<Couleur de bordure> : (Entier ou constante de type Entier)`**

Couleur de la bordure du point (lorsque le point possède une bordure). Cette couleur peut correspondre à :

- une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)),

- une couleur TSL (obtenue avec la fonction [TSL](../WDLang1/3029057.md)),

- [une couleur prédéfinie](../WDLang5/3010002.md).






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Pour modifier la couleur de toutes les points d'une catégorie, utilisez la fonction [&lt;Graphe&gt;.CouleurCatégorie](../WDLang3/1000023570.md).

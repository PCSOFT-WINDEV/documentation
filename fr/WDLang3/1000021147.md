


## SSHConnecteShell (Fonction)

***En anglais : SSHConnectShell***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ouvre une nouvelle session SSH de type "Shell". Cette session pourra être fermée à l'aide de la fonction [SSHDéconnecteShell](../WDLang3/1000021148.md).

**Remarque** :  Pour envoyer une commande directement au serveur SSH, utilisez la fonction [SSHCommande](../WDLang3/1000021143.md).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
cMaSession est une SSHSession
cMaSession.Adresse = "127.0.0.1"
cMaSession.Port = 22
cMaSession.Utilisateur = "login"
cMaSession.MotDePasseUtilisateur = "pass"
SI SSHConnecteShell(cMaSession) ALORS
	Info("Session ouverte")
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = SSHConnecteShell(<Session SSH>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la session a été ouverte correctement,

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Session SSH> : (Variable de type sshSession)`**

Nom d'une variable [sshSession](../WDLang3/1000021157.md) contenant les paramètres de connexion de la session SSH à ouvrir.

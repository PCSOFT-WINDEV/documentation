


## grMAJAutomatiqueEtiquetteCatégorie (Fonction)

***En anglais : grAutoRefreshCategoryLabel***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Identifie ou définit si les étiquettes de catégorie sont mises à jour à chaque dessin du graphe. Dans ce cas, à chaque appel de la fonction [grDessine](../WDLang3/3042026.md), le contenu de la série est automatiquement recalculé pour refléter les derniers changements de la source de données. Les étiquettes de catégories sont alors mises à jour si nécessaire.

**Exemple d'utilisation** : une série dans un graphe sert de série de référence : cette série n'est pas mise à jour automatiquement.



![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Les graphes](../WDChamp/3042060.md). 






<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Graphe créé par programmation
SI grMAJAutomatiqueEtiquetteCatégorie("MonGraphe") = Vrai ALORS
	grDessine("MonGraphe")
FIN
```
<a name="Exemple2"></a>

```wl
// Champ Graphe
SI grMAJAutomatiqueEtiquetteCatégorie(GRF_MonGraphe) = Vrai ALORS
	grDessine(GRF_MonGraphe)
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Identifier le mode de mise à jour des étiquettes de catégorie

`<Résultat> = grMAJAutomatiqueEtiquetteCatégorie(<Nom du graphe>)`
---

**`<Résultat> : (Booléen)`**

Mode de mise à jour actuel des étiquettes de catégories : 

- <u><u><u><u>Vrai</u></u></u></u> si les étiquettes sont mises à jour automatiquement, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Nom du graphe> : (Nom de champ ou chaîne de caractères)`**

Nom du graphe à manipuler. Ce nom correspond :

- soit au nom du graphe défini par programmation avec la fonction [grCrée](../WDLang3/3042005.md).

- soit au nom du champ Graphe présent dans l'éditeur de fenêtres, de pages ou d'états.  





<a name="SYNTAXE2"></a>

### Définir le mode de mise à jour des étiquettes de catégorie

`grMAJAutomatiqueEtiquetteCatégorie(<Nom du graphe> , <Mise à jour automatique>)`
---

**`<Nom du graphe> : (Nom de champ ou chaîne de caractères)`**

Nom du graphe à manipuler. Ce nom correspond :

- soit au nom du graphe défini par programmation avec la fonction [grCrée](../WDLang3/3042005.md).

- soit au nom du champ Graphe présent dans l'éditeur de fenêtres, de pages ou d'états.




**`<Mise à jour automatique> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> pour une mise à jour automatique, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Si les étiquettes de catégories sont en mode automatique, les étiquettes de catégories sont automatiquement recalculées lors du prochain appel à la fonction [grDessine](../WDLang3/3042026.md).

- Si les étiquettes de catégories ne sont pas en mode automatique, les étiquettes de catégories seront recalculées uniquement lors du prochain appel de la fonction [grSourceEtiquetteCatégorie](../WDLang3/3042072.md).




## &lt;Variable upnpPériphérique&gt;.RécupèreNetteté (Fonction)

***En anglais : &lt;upnpDevice variable&gt;.GetSharpness***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Récupère la netteté d'un périphérique UPnP.

**Attention** : Pour utiliser cette fonction, le périphérique doit posséder un service "RenderingControl".
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
tabPériphérique est un tableau de upnpPériphériques
tabPériphérique = UPNPListePériphérique(...
		"urn:schemas-upnp-org:device:MediaRenderer:1", 5)
POUR TOUT Périphérique de tabPériphérique
	Trace(Périphérique..Nom + " : " + Périphérique.RécupèreNetteté())
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Périphérique UPnP>.RécupèreNetteté()`
---

**`<Résultat> : (Entier)`**

Netteté du périphérique (comprise entre 0 et 100).

**`<Périphérique UPnP> : (Variable de type upnpPériphérique)`**

Nom de la variable [upnpPériphérique](../WDLang3/1000020903.md) correspondant au périphérique UPnP à manipuler.


## HTTPRAZFormulaire (Fonction)

***En anglais : HTTPResetForm***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Vide un formulaire HTTP en cours d'édition.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
HTTPCréeFormulaire("FORM")
...
HTTPRAZFormulaire("FORM")
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`HTTPRAZFormulaire(<Nom du formulaire>)`
---

**`<Nom du formulaire> : (Chaîne de caractères)`**

Nom du formulaire HTTP à vider. Ce nom a été donné lors de la création du formulaire HTTP (fonction [HTTPCréeFormulaire](../WDLang3/1000017119.md)). Si le formulaire n'existe pas, une erreur est affichée.

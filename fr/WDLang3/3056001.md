
## NetEnvoieBuffer (Fonction)

***En anglais : NetSendBuffer***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Transmet une chaîne de caractères à un serveur FTP (File Transfer Protocol) ou RPC (Remote Procedure Call) réalisé avec WINDEV.

**Remarque** : Il est possible d'autoriser ou non l'utilisation de cette fonction sur un serveur FTP/RPC grâce à la fonction [NetOptionServeur](../WDLang3/3056022.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple
<a class="notetitle" target="_blank" href="$DOC$=1000003056001&name=netenvoiebuffer_fonction&product=WD">Voir des exemples supplémentaires</a>

```wl
ConnectFTP est un entier
ConnectFTP = NetConnecte("148.61.125.245", ServeurFTP, "GUEST", "")
...
NetEnvoieBuffer(ConnectFTP, "La suite dans quelques instants")
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`NetEnvoieBuffer(<Identifiant connexion> , <Chaîne à envoyer> [, <Informations complémentaires>])`
---

**`<Identifiant connexion> : (Entier)`**

Identifiant de la connexion (obtenu par la fonction [NetConnecte](../WDLang3/3056004.md)).

**`<Chaîne à envoyer> : (Chaîne de caractères)`**

Texte à envoyer au serveur.

**`<Informations complémentaires> : (Entier optionnel)`**

Informations complémentaires ajoutées à la chaîne.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Caractéristiques de l'envoi
<a name="caracteristiques_envoi_ELTPARAGRAPHE000045"></a>La chaîne est envoyée grâce au message Windows WM_COPYDATA(74). 


<a name="NOTE0_2"></a>




### Récupération du message sur le serveur
<a name="recuperation_message_sur_serveur_ELTPARAGRAPHE000054"></a>Pour récupérer la chaîne envoyée et les informations complémentaires sur le serveur, il faut intercepter l'événement 74 grâce à la fonction [Evénement](../WDLang1/3015011.md) (voir l'exemple complet).

La variable _EVE.lParam pointe sur une structure de la forme :


```wl
RecupChaîne est une structure
	Wdata est un entier // Information complémentaire envoyée
	lTaille est un entier // Taille de la chaîne envoyée (Nombre d'octets)
	pInfo est un entier // Adresse de la chaîne envoyée
FIN
```

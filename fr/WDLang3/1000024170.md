


## &lt;Graphe&gt;.SupprimeTout (Fonction)

***En anglais : &lt;Chart&gt;.DeleteAll***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Efface un graphe dessiné et créé avec la fonction [grCrée](../WDLang3/3042005.md) et détruit ce graphe. La destruction d'un graphe permet de libérer la place mémoire occupée par la matrice utilisée par ce graphe.

**Remarque** : Tous les graphes utilisés dans une application sont systématiquement détruits à la fermeture de la fenêtre ou de l'application.



![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Les graphes](../WDChamp/3042060.md). 










<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Destruction du graphe "MonGraphe"
GRF_MonGraphe.SupprimeTout()
```

<a name="SYNTAXE1"></a>

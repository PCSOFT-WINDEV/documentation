
## BeaconArrêteDétectionPrécise (Fonction)

***En anglais : BeaconStopPreciseDetection***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Arrête la détection de balises Beacon lancée avec la fonction [BeaconDétectePrécis](../WDLang3/1000023115.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Lance une détection d'un groupe de balises Beacon en arrière-plan
BeaconDétecteEnArrièrePlan(tabGroupeBalises, ProcDétection)
PROCEDURE INTERNE ProcDétection(groupe est un beaconGroupe, nType est un entier)
// Entrée dans la zone d'émission des balises de groupes
	SI nType = bdapEntrée
		// On recherche les balises à proximité
		BeaconDétectePrécis(groupe, procDétectionPrécise)
		// Sortie de la zone d'émission des balises de groupes
	SINON
		// On stoppe la détection précise
		BeaconArrêteDétectionPrécise()
	FIN
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = BeaconArrêteDétectionPrécise()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la détection a bien été arrêtée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




## &lt;Graphe&gt;.PoliceTitreAxe (Fonction)

***En anglais : &lt;Chart&gt;.AxisTitleFont***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie la police utilisée pour les titres des axes d'un graphe.

Exemple de titres des axes : <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Titreaxe.gif)




![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Les graphes](../WDChamp/3042060.md). 








<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Modification de la police utilisée pour
// les titre des axes du graphe "GRF_MonGraphe".
MaPolice est une police
MaPolice.Nom = "Arial"
MaPolice.Taille = 12
MaPolice.Gras = Vrai
GRF_MonGraphe.PoliceTitreAxe(MaPolice)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Définition de la police utilisée pour le titre des axes d'un graphe

`<Champ Graphe>.PoliceTitreAxe(<Police utilisée>)`
---

**`<Champ Graphe> : (Nom de champ)`**

Nom du champ Graphe manipulé présent dans l'éditeur de fenêtres, de pages ou d'états.


**`<Police utilisée> : (Police)`**

Police à utiliser. Ce paramètre correspond à une variable de type [Police](../Motscles/1514045.md). Les caractéristiques de la police ont été définies grâce à la fonction [PoliceCrée](../WDLang1/3060002.md) ou grâce aux [propriétés de gestion des polices](../Proprietes/2515045.md).


<a name="SYNTAXE2"></a>

### Définition de la police utilisée pour le titre des axes d'un graphe (syntaxe pour compatibilité)

`<Champ Graphe>.PoliceTitreAxe(<Identifiant de la police>)`
---

**`<Champ Graphe> : (Nom de champ)`**

Nom du champ Graphe manipulé présent dans l'éditeur de fenêtres, de pages ou d'états.


**`<Identifiant de la police> : (Entier)`**

Identifiant de la police, défini avec la fonction [grCréePolice](../WDLang3/3042001.md). Si cette police n'existe pas, une erreur du WLangage est générée.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Titre d'un graphe
<a name="titre_graphe_ELTPARAGRAPHE000288"></a>Le titre des axes du graphe peut être affiché le long des différents axes.<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Titre.gif)
Le contenu du titre des axes du graphe est défini par la fonction [&lt;Graphe&gt;.TitreAxe](../WDLang3/1000024153.md). Cette fonction permet également de positionner ce titre.

La police utilisée pour le titre du graphe peut être modifiée grâce à la fonction **&lt;Graphe&gt;.PoliceTitreAxe**.
<a name="NOTE0_2"></a>


### Initialisation des polices
<a name="initialisation_des_polices_ELTPARAGRAPHE000310"></a>La police utilisée par défaut pour le titre des axes a les caractéristiques suivantes :

- Nom de la police : Arial,

- Taille : 14,

- Style : Normal,

- Couleur : Noir.

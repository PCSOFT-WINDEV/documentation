


## grAxeVisible (Fonction)

***En anglais : grAxisVisible***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Paramètre l'affichage des axes dans un graphe de type Courbe ou Histogramme. L'axe peut :

- être visible, 

- être invisible, 

- afficher uniquement le libellé.




![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Les graphes](../WDChamp/3042060.md). 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
grAxeVisible(GRF_Graphe1, grAbscisse, grInvisible)
// ou par compatibilité
// grAxeVisible(GRF_Graphe1, grAbscisse, Faux)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`grAxeVisible(<Nom du graphe> , <Type de l'axe> , <Visible>)`
---

**`<Nom du graphe> : (Nom de champ ou chaîne de caractères)`**

Nom du graphe à manipuler. Ce nom correspond :

- soit au nom du graphe défini par programmation avec la fonction [grCrée](../WDLang3/3042005.md).

- soit au nom du champ Graphe présent dans l'éditeur de fenêtres, de pages ou d'états.




**`<Type de l'axe> : (Constante de type Entier)`**

Axe à manipuler :


|   |   |
| --- | --- |
| *grAbscisse* | Axe des abscisses. |
| *grOrdonnée* | Axe des ordonnées. |



**`<Visible> : (Constante de type Entier)`**

Mode d'affichage de l'axe : 


|   |   |
| --- | --- |
| *grInvisible* | L'axe devient invisible.<br>Par compatibilité, il est également possible d'utiliser &lt;Visible&gt; = Faux. |
| *grLibelléUniquement* | Seuls les libellés de l'axe sont visibles.<br><br> |
| *grVisible* | L'axe devient visible. <br>Par compatibilité, il est également possible d'utiliser &lt;Visible&gt; = Vrai. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Si l'axe des ordonnées est invisible, l'axe des ordonnées secondaires est également invisible. 

- Les paramètres définis avec la fonction **grAxeVisible** seront pris en compte lors du prochain appel à la fonction [grDessine](../WDLang3/3042026.md).

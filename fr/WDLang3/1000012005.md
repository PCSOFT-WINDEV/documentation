
## TâchePremier (Fonction)

***En anglais : TaskFirst***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Lit la première tâche de la liste des tâches d'une messagerie Lotus Notes ou Outlook. Cette tâche devient la tâche en cours.

Lors de la lecture d'une tâche, les données de cette tâche sont mis à jour dans la structure [Tâche](../WDLang3/1000012010.md). Pour récupérer les données de la tâche en cours, il suffit de récupérer les valeurs de la structure [Tâche](../WDLang3/1000012010.md).

**Attention** : Cette fonction **ne permet pas** de lire la première tâche de la liste des tâches d'une messagerie Outlook Express.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ouverture d'une session Lotus Notes
IDSession est un entier
IDSession = EmailOuvreSessionNotes("MotDePasse", "ServeurMars", "mail\julie.nsf",...
					"C:\Users\Julie\AppData\Local\Lotus\Notes\Data\user.id")

// Lecture de la première tâche
TâchePremier(IDSession)
// Ajout des tâches trouvées dans une liste
TANTQUE mTâche.EnDehors = Faux
	ListeAjoute(LISTE_ListeTâche, mTâche.Objet + TAB + mTâche.Catégorie)
	TâcheSuivant(IDSession)
FIN
```


<a name="Exemple2"></a>



```wl
// Ouverture d'une session Outlook
IDSession est un entier
IDSession = OutlookOuvreSession("MonProfil")
// Lecture de la première tâche
TâchePremier(IDSession)
// Ajout des tâches trouvées dans une liste
TANTQUE mTâche.EnDehors = Faux
	ListeAjoute(LISTE_ListeTâche, mTâche.Objet + TAB + mTâche.Catégorie)
	TâcheSuivant(IDSession)
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = TâchePremier(<Identifiant de la session>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la première tâche a été lue, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour connaître le libellé de l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*. 




**`<Identifiant de la session> : (Entier)`**

Identifiant de la session à utiliser. Cet identifiant est renvoyé par la fonction [EmailOuvreSessionNotes](../WDLang3/3032121.md), [EmailOuvreSessionOutlook](../WDLang3/3032128.md), [NotesOuvreBase](../WDLang3/1000005101.md) ou [OutlookOuvreSession](../WDLang3/3032123.md).

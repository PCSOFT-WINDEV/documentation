


## &lt;Graphe&gt;.BourseAjouteDonnée (Fonction)

***En anglais : &lt;Chart&gt;.StockAddData***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ajoute une donnée dans un graphe boursier (chandelier, barcharts ou minmax).

**Attention** : Selon le type de graphe à manipuler, il existe une fonction WLangage spécifique pour ajouter les données (voir paragraphe "Remarques"). 
![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Les graphes](../WDChamp/3042060.md). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple

<a name="3042009_Exemple2"></a>

```wl
// Champ Graphe
// Ajout de données dans le graphe "GRF_MonGraphe"
GRF_MonGraphe.BourseAjouteDonnée(1, 5.4, 6.8, 6.1, 6.3)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Champ Graphe>.BourseAjouteDonnée(<Numéro de la série> [, <Indice>] , <Valeur minimale> , <Valeur maximale> , <Première valeur> , <Dernière valeur>)`
---

**`<Champ Graphe> : (Nom de champ)`**

Nom du champ Graphe à manipuler (présent dans l'éditeur de fenêtres, de pages ou d'états).


**`<Numéro de la série> : (Entier)`**

Numéro de la série dans laquelle une valeur doit être ajoutée. Si la série spécifiée n'existe pas, le grapheur crée autant de séries que nécessaire pour obtenir une matrice de données cohérente. Les séries ainsi créées sont toutes initialisées à 0.

**Attention : Seule la première série sera dessinée**.

**`<Indice> : (Entier optionnel)`**

Indice de la donnée dans la série où les valeurs doivent être ajoutées. Si ce paramètre :

- n'est pas spécifié : les valeurs sont ajoutées à la fin de la série.

- est spécifié : l'ancienne valeur est remplacée.




**`<Valeur minimale> : (Entier ou réel)`**

Valeur minimale de la journée.

**`<Valeur maximale> : (Entier ou réel)`**

Valeur maximale de la journée.

**`<Première valeur> : (Entier ou réel)`**

Première valeur de la journée.

**`<Dernière valeur> : (Entier ou réel)`**

Dernière valeur de la journée.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Cohérence des valeurs
<a name="coherence_des_valeurs_ELTPARAGRAPHE000276"></a>Si les valeurs ne sont pas cohérentes, les données ne seront pas affichées dans le graphe :

- &lt;Première valeur&gt;, &lt;Dernière valeur&gt; et &lt;Valeur minimale&gt; doivent être strictement inférieures à &lt;Valeur maximale&gt;.

- &lt;Première valeur&gt;, &lt;Dernière valeur&gt; et &lt;Valeur maximale&gt; doivent être strictement supérieures à &lt;Valeur minimale&gt;.



<a name="NOTE0_2"></a>


### Ajout de données
<a name="ajout_donnees_ELTPARAGRAPHE000289"></a>

- Les données ajoutées avec la fonction **&lt;Graphe&gt;.BourseAjouteDonnée** seront prises en compte lors du prochain appel à la fonction [&lt;Graphe&gt;.Dessine](../WDLang3/1000024062.md).

- Seule la première série sera dessinée.



<a name="NOTE0_3"></a>


### Ajout de données dans les différents types de graphes
<a name="ajout_donnees_dans_les_differents_types_graphes_ELTPARAGRAPHE000308"></a>

- Pour ajouter des données dans les graphes standard (courbe, secteur, ...), utilisez la fonction [&lt;Graphe&gt;.AjouteDonnée](../WDLang3/1000023533.md). 

- Pour les graphes de type Nuage de points, il est conseillé d'utiliser la fonction [&lt;Graphe&gt;.NuageAjouteDonnéeXY](../WDLang3/1000024105.md).

- Pour les graphes Heatmap, il est conseillé d'utiliser la fonction [&lt;Graphe&gt;.HMAjouteDonnée](../WDLang3/1000024079.md).

- Pour les graphes Secteur Multiniveau, il est conseillé d'utiliser la fonction [&lt;Graphe&gt;.SMNAjouteDonnée](../WDLang3/1000024132.md). 

- Pour les graphes de type Surface, il est conseillé d'utiliser la fonction [&lt;Graphe&gt;.SurfaceAjouteDonnée](../WDLang3/1000024143.md).

- Pour ajouter une valeur à une donnée d'un graphe, utilisez la fonction [&lt;Graphe&gt;.IncrémenteDonnée](../WDLang3/1000024086.md).

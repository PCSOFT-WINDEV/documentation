
## SaaSAdminListeClient (Fonction)

***En anglais : SaaSAdminListClient***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Liste les comptes client de la base d'un Webservice SaaS.

**Remarques** : 

- L'utilisation de cette fonction nécessite d'avoir ouvert une session d'administration SaaS au préalable en utilisant la fonction [SaaSAdminConnecte](../WDLang3/1000019098.md).

- L'utilisation de cette fonction est réservée à l'Administrateur SaaS. Un gestionnaire de compte client ne peut pas utiliser cette fonction. 



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
tabClient est un tableau de saasClient 
tabClient = SaaSAdminListeClient()
POUR TOUT Client DE tabClient
	Trace(Client.Nom)
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = SaaSAdminListeClient()`
---

**`<Résultat> : (Tableau de variables de type saasClient)`**

Liste des comptes client du Webservice SaaS.

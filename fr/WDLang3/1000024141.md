


## &lt;Graphe&gt;.SupprimeSérie (Fonction)

***En anglais : &lt;Chart&gt;.DeleteSeries***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime une série de données d'un graphe. Toutes les données de la série sont effacées. 

Attention : Les séries restantes sont re-numérotées. Par exemple, lors de la suppression de la série 1, la série 2 devient la série 1, ...



![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Les graphes](../WDChamp/3042060.md). 












<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Suppression de la troisième série de données du graphe "GRF_MonGraphe"
GRF_MonGraphe.SupprimeSérie(3)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Champ Graphe>.SupprimeSérie([<Numéro de la série> [, <Options>]])`
---

**`<Champ Graphe> : (Nom de champ)`**

Nom du champ Graphe à manipuler.


**`<Numéro de la série> : (Entier optionnel)`**

Numéro de la série à supprimer. Si ce paramètre n'est pas spécifié, toutes les données du graphe sont effacées.

**`<Options> : (Constante de type Entier)`**

Type de suppression à réaliser : 


|   |   |
| --- | --- |
| *grDonnées* | Suppression uniquement des données de la série. |
| *grDonnéesEtStyle*<br>(Valeur par défaut) | Suppression des données et du style de la série. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Si toutes les séries d'un graphe sont supprimées, il faut ajouter de nouvelles données afin de pouvoir re-dessiner le graphe.

- Les modifications effectuées avec la fonction **&lt;Graphe&gt;.SupprimeSérie** seront prises en compte lors du prochain appel à la fonction [&lt;Graphe&gt;.Dessine](../WDLang3/1000024062.md).

- Dans le cas d'une série basée sur une rubrique, les données sont effacées. La source de données est conservée, et les données peuvent donc être reconstruites. Pour couper le lien à la source, utilisez la fonction [&lt;Graphe&gt;.SourceSérie](../WDLang3/1000024139.md) en modifiant la source de la série.


## NotifSupprimeCatégorie (Fonction)

***En anglais : NotifDeleteCategory***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime une catégorie de notification de l'application.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Suppression de toutes les catégories de notification de l'application avec une priorité faible
tabCatégorie est un tableau de notificationCatégorie = NotifListeCatégorie()
POUR TOUTE Catégorie DE tabCatégorie
	SI Catégorie.Priorité = notifPrioritéFaible ALORS
		NotifSupprimeCatégorie(Catégorie.Identifiant)
	FIN
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = NotifSupprimeCatégorie(<Identifiant>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la catégorie a été supprimée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. En cas d'échec de la suppression, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md) pour connaître le détail de l'erreur.




**`<Identifiant> : (Chaîne de caractères)`**

Identifiant de la catégorie de notification à supprimer. L'identifiant de catégorie est sensible à la casse et à l'accentuation.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Lors de  la suppression d'une catégorie, les modifications effectuées par l'utilisateur final dans les paramètres de notification de l'application sont perdues.

- Pour lister les catégories existantes de l'application, utilisez la fonction [NotifListeCatégorie](../WDLang3/1000023979.md).

- La fonction **NotifSupprimeCatégorie** est disponible uniquement sur les appareils disposant d'une version d'Android supérieure ou égale à la version 8.0 (api level 26). 
	Si la fonction est utilisée avec une version antérieure du système, une erreur fatale sera déclenchée. 
	Pour connaître la version du système Android sur laquelle s'exécute l'application, utilisez la fonction [SysVersionAndroid](../WDLang1/1000019785.md).

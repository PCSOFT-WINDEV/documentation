


## MQTTDésabonne (Fonction)

***En anglais : MQTTUnsubscribe***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de se désabonner d'une notification MQTT. 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MQTTDésabonne(gMqttSession, "sensors/sensor1/temperature1")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = MQTTDésabonne(<Session MQTT> , <Sujet>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le désabonnement est mis en place,

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. La fonction [ErreurInfo](../WDLang1/3013008.md) permet d'obtenir plus de détails. 




**`<Session MQTT> : (Variable de type mqttSession)`**

Nom de la variable [mqttSession](../WDLang3/1000023023.md) correspondant à la session MQTT à manipuler.

**`<Sujet> : (Chaîne de caractères)`**

Sujet pour lequel le désabonnement est demandé.

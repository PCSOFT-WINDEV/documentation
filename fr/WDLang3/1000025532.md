


## CarteAjouteImage (Fonction)

***En anglais : MapAddImage***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ajoute une nouvelle image sur la carte affichée dans un champ Carte.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ajout d'une image dans le champ Carte
ImageSurCarte est un CarteImage
ImageSurCarte.Position = MaPosition
ImageSurCarte.Image = "image.png"
ImageSurCarte.ActionClic = ProcClic
CarteAjouteImage(CARTE_MaCarte, ImageSurCarte)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = CarteAjouteImage(<Champ Carte> , <Image>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'image a pu être ajoutée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Champ Carte> : (Nom de champ)`**

Nom du champ Carte à manipuler. Si ce paramètre correspond à une chaîne vide (""), le champ Carte auquel appartient l'événement en cours sera utilisé.

**`<Image> : (Variable de type CarteImage)`**

Nom de la variable de type [CarteImage](../WDLang3/1000025523.md) correspondant à l'image à ajouter.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Si l'image a été déjà été ajoutée dans le champ Carte ou si l'image est invalide (ex : largeur non spécifiée), une erreur fatale sera affichée. 

- Pour supprimer une image dessinée dans un champ Carte, utilisez la fonction [CarteSupprimeImage](../WDLang3/1000025534.md). 

- Pour modifier une image dessinée dans un champ Carte, utilisez la fonction [CarteModifieImage](../WDLang3/1000025533.md).

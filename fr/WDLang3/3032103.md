
## LDAPRAZ (Fonction)

***En anglais : LDAPReset***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ré-initialise la [structure LDAPSession](../WDLang3/3032101.md) avec les valeurs par défaut. Cette structure est utilisée par la fonction [LDAPConnecte](../WDLang3/3032100.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
LDAPRAZ()
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`LDAPRAZ()`

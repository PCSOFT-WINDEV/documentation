
## ContactEdite (Fonction)

***En anglais : ContactEdit***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ouvre en édition la fiche d'un contact en cours dans l'application native de gestion des contacts de l'appareil  (Android, iPhone ou iPad, Universal Windows).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ouvre en édition la fiche du contact en cours
// Variables globales
gContactEnCours est un Contact

... 

// - Clic sur BTN_Modifier
// Lance l'édition du contact
ContactEdite(gContactEnCours)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Ouvrir la fiche d'un contact en édition

`<Résultat> = ContactEdite(<Contact>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le contact a été modifié, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. En cas d'erreur, la variable [ErreurDétectée](../WDLang1/3087001.md) est à <u><u><u><u>Vrai</u></u></u></u>. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) 

- <u><u><u><u>Vrai</u></u></u></u> si l'interface utilisateur est affichée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 




**`<Contact> : (Variable de type Contact)`**

Nom de la variable de type [Contact](../WDLang3/1000020254.md) correspondant au contact dont la fiche doit être ouverte. Ce contact doit exister dans la base des contacts de l'appareil. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Cas particuliers
<a name="cas_particuliers_ELTPARAGRAPHE000086"></a>

- ![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) La fonction **ContactEdite** n'est pas bloquante. 

- Pour mettre à jour la variable [Contact](../WDLang3/1000020254.md) avec les informations relatives au contact précédemment édité, utilisez la fonction [ContactLit](../WDLang3/1000006112.md).

- La fonction **ContactEdite** peut être utilisée dans l'émulateur. Elle ne peut pas être utilisée dans le simulateur.

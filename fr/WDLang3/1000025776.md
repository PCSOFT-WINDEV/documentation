
## géoRécupèreZone (Fonction)

***En anglais : geoGetArea***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Récupère les coordonnées de la zone géographique correspondant à la description.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affiche la France dans un champ Carte 
soit (p1,p2) = géoRécupèreZone("France")
SI PAS ErreurDétectée() ALORS
	CarteAfficheZone(CARTE_Carte1, p1, p2)
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`(<Position NO>, <Position SE>) = géoRécupèreZone(<Description>)`
---

**`<Position NO> : (Variable de type géoPosition)`**

Variable de type [géoPosition](../WDLang3/1000019191.md) correspondant à la zone géographique Nord Ouest.

**`<Position SE> : (Variable de type géoPosition)`**

Variable de type [géoPosition](../WDLang3/1000019191.md) correspondant à la zone géographique Sud Est.

**`<Description> : (Chaîne de caractères)`**

Zone géographique dont les coordonnées doivent être récupérées. Par exemple : "Paris, France".

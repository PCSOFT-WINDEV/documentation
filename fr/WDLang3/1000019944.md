
## Adresse (Type de variable)

***En anglais : Address***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **Adresse** permet de manipuler l'adresse d'un lieu ou d'une personne. Une adresse est définie par une adresse postale (rue, ville, département, code postal, pays) et/ou une position géographique (latitude et longitude). Les caractéristiques de cette adresse peuvent être définies et modifiées à l'aide de différentes propriétés WLangage. 

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonAdresse est une Adresse
MonAdresse.Ville = "Paris"
MonAdresse.Pays = "France"

// Centre la carte affichée par le champ "CARTE_Position" à partir d'une adresse
CarteAffichePosition(CARTE_Position, MonAdresse)
```

## Remarques


### Propriétés spécifiques à la description des variables de type Adresse
<a name="proprietes_specifiques_description_des_variables_type_adresse_ELTPARAGRAPHE000059"></a>Les propriétés suivantes peuvent être utilisées pour manipuler une variable de type **Adresse** :

| **Nom de la propriété** | **Type manipulé** | **Effet** |
| --- | --- | --- |
| CodePostal | Chaîne de caractères | Code postal de l'adresse. <br><br>Cette propriété est optionnelle. |
|   | Chaîne de caractères | Etiquette pour les types d'adresses personnalisées (optionnel).<br>Si la propriété **Type** ne correspond pas à la constante *adressePersonnalisée*, la valeur de cette propriété sera ignorée.<br><br>Cette propriété est notamment utilisée pour la gestion des contacts. |
| Pays | Chaîne de caractères | Nom du pays. Cette propriété est optionnelle. |
| Position | Variable de type [géoPosition](../WDLang3/1000019191.md) | Position géographique de l'adresse : latitude et longitude. <br><br>Cette propriété est optionnelle. <br><br>Si la position géographique n'est pas spécifiée, la propriété **PositionValide** de la variable [géoPosition](../WDLang3/1000019191.md) vaut <u><u><u><u>Faux</u></u></u></u>. |
| Région | Chaîne de caractères | Région associée à l'adresse (état pour une adresse aux Etats-Unis, land pour une adresse en Allemagne, Comté, province, ...). <br><br>Cette propriété est optionnelle. |
| Rue | Chaîne de caractères | Nom de la rue, route ou avenue. <br><br>Cette propriété est optionnelle. |
|   | Constante de type Entier | Type d'adresse : <br><br>- *adresseAutre* : autre type d'adresse.<br><br>- *adresseBureau* : adresse professionnelle (du bureau).<br><br>- *adresseDomicile* : adresse du domicile. <br><br>- *adressePersonnalisée* : type personnalisé correspondant à la valeur de la propriété **Etiquette**.<br><br><br><br><br>Cette propriété est notamment utilisée pour la gestion des contacts. |
| Ville | Chaîne de caractères | Nom de la ville. <br><br>Cette propriété est optionnelle. |


<a name="NOTE0_2"></a>


### Utilisation des variables de type Adresse
<a name="utilisation_des_variables_type_adresse_ELTPARAGRAPHE000161"></a>Les variables de type **Adresse** sont utilisées par les fonctions suivantes : 



|   |   |
| --- | --- |
| [CarteAffichePosition](../WDLang3/1000019787.md) | Centre la carte affichée dans un champ Carte sur une position géographique ou un lieu donné. |
| [CarteAjouteItinéraire](../WDLang3/1000020383.md) | Ajoute un itinéraire sur un champ Carte. |
| [géoRécupèreAdresse](../WDLang3/1000019797.md) | Récupère une adresse ou une liste d'adresses correspondant à la description d'un lieu géographique. |





Les variables de type **Adresse** sont utilisées par les types de variables suivants : 



|   |   |
| --- | --- |
| [Contact](../WDLang3/1000020254.md) | Le type **Contact** permet de définir toutes les caractéristiques avancées d'un contact du répertoire téléphonique. |

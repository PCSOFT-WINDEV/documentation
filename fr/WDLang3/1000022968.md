


## grNuage3DReliePoint (Fonction)

***En anglais : grScatter3DLinkPoint***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Relie ou non les points (à l'aide de lignes) dans un graphe de type "Nuage de points 3D".

**Rappel** : Par défaut, les points d'un graphe de type "Nuage de points 3D" sont reliés.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
grNuage3DReliePoint(GRF_MonNuage3D, 1, TraitContinu)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`grNuage3DReliePoint(<Nom du graphe> , <Numéro de la série> [, <Type de trait>])`
---

**`<Nom du graphe> : (Nom de champ ou chaîne de caractères)`**

Nom du graphe à manipuler. Ce nom correspond :

- soit au nom du graphe défini par programmation avec la fonction [grCrée](../WDLang3/3042005.md).

- soit au nom du champ Graphe présent dans l'éditeur de fenêtres.




**`<Numéro de la série> : (Entier)`**

Numéro de la série à manipuler. 

**`<Type de trait> : (Constante optionnelle de type Entier)`**

Type du trait à tracer entre les points du graphe : 



|   |   |
| --- | --- |
| *TraitAucun* | Aucun trait n'est tracé. |
| *TraitContinu* <br>(Valeur par défaut) | Trait plein continu. |
| *TraitMixte* | Trait alternant pointillé et tiret (.-.-.-). |
| *TraitPointillé* | Trait en pointillé (........). |
| *TraitTiret* | Trait avec des tirets (------). |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Les données définies avec la fonction **grNuage3DReliePoint** seront prises en compte lors du prochain appel à la fonction [grDessine](../WDLang3/3042026.md).

- La fonction **grNuage3DReliePoint** est utilisée uniquement pour les graphes de type "Nuage de points 3D". Cette fonction est sans effet pour tous les autres types de graphes.

- Les points sont reliés dans l'ordre dans lequel ils ont été ajoutés.

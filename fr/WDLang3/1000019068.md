
## saasSite (Type de variable)

***En anglais : saasSite***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **saasSite** permet de définir toutes les caractéristiques avancées d'un site WEBDEV SaaS.

Les caractéristiques de ce site peuvent être définies et modifiées à l'aide de différentes propriétés WLangage. 

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonSite est un saasSite
MonSite.Nom = "SiteSAV"

SaaSAdminAjouteSite(S)
```

<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Propriétés spécifiques à la description des variables de type saasSite
<a name="proprietes_specifiques_description_des_variables_type_saassite_ELTPARAGRAPHE000030"></a>Les propriétés suivantes peuvent être utilisées pour manipuler un site SaaS :


|   |   |   |
| --- | --- | --- |
| **Nom de la propriété** | **Type manipulé** | **Effet** |
| Nom | Chaîne de caractères | Nom du site. Ce nom sera utilisé dans l'[Administrateur SaaS](../SaaSAdminWeb/1600000.md) et en programmation pour identifier le site. Il correspond généralement au nom du projet ou au nom de déploiement du site WEBDEV. |

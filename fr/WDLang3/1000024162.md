


## &lt;Graphe&gt;.AjouteDonnéeTemporelle (Fonction)

***En anglais : &lt;Chart&gt;.AddTimeData***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ajoute une donnée dans un graphe à échelle temporelle. Un graphe à échelle temporelle est un graphe dont l'unité de l'échelle des abscisses est le temps (date, heure, durée). 

Cette échelle temporelle peut être utilisée par exemple pour représenter une évolution d'une mesure (capteur de température par exemple) en fonction du temps passé sur une journée.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
nSérie est un entier = 1
rTempérature est un tableau de 24*60 réels

POUR i = 1 À 24*60 // 24h * 60 mn
	d est une Durée
	d.Minute = i
	GRF_CapteurTempérature.AjouteDonnéeTemporelle(nSérie, d, rTempérature[i])
FIN

GRF_CapteurTempérature.Dessine()
```

```wl
// Dessin sinusoïde
POUR i = 1 À 360
	d est une Durée
	d.Minute = i
	GRF_Sinus.AjouteDonnéeTemporelle(1, d, 2 + Sinus(i))
FIN

GRF_Sinus.Dessine()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Champ Graphe>.AjouteDonnéeTemporelle(<Numéro de la série> , <Temps> , <Valeur>)`
---

**`<Champ Graphe> : (Nom de champ)`**

Nom du champ Graphe à manipuler.

**`<Numéro de la série> : (Entier)`**

Numéro de la série dans laquelle une valeur doit être ajoutée. Si la série spécifiée n'existe pas, le grapheur crée autant de séries que nécessaire pour obtenir une matrice de données cohérente. Les séries ainsi créées sont toutes initialisées à 0.

**`<Temps> : (Durée ou DateHeure ou Date ou Heure)`**

Position dans le temps pour laquelle la valeur doit être ajoutée.

**`<Valeur> : (Entier ou réel)`**

Valeur à ajouter. 
Si une valeur est déjà présente pour une position dans le temps, cette valeur sera remplacée par la nouvelle valeur.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Cette fonction permet d'ajouter des données dans les graphes utilisant une échelle temporelle et également les graphes en temps réel. Pour plus de détails, consultez [Graphe avec affichage en temps réel](../WDChamp/1000021037.md).

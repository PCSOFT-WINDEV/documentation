<a name="Utilisation"></a>
<a name="description"></a>
Déclenche un événement Windows sur le système d'un poste serveur FTP (File Transfer Protocol) ou RPC (Remote Procedure Call) réalisé avec WINDEV. Cet événement sera envoyé à toutes les fenêtres ouvertes sur le serveur.

**Attention** : Cette fonction doit être utilisée sur un poste client.

**Remarque** : Il est possible d'autoriser ou non l'utilisation de cette fonction sur un serveur FTP/RPC grâce à la fonction [NetOptionServeur](../WDLang3/3056022.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple
<a class="notetitle" target="_blank" href="$DOC$=1000003056003&name=netenvoiemessage_fonction&product=WD">Voir des exemples supplémentaires</a>

```wl
ConnectRPC est un entier
ConnectRPC = NetConnecte("148.61.125.245", ServeurRPC, "GUEST", "")
...
// Coté client : envoi d'un message Windows au serveur
NetEnvoieMessage(ConnectRPC, "MonMessage")
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`NetEnvoieMessage(<Identifiant connexion> , <Événement> [, <Paramètre 1> [, <Paramètre 2>]])`
---

**`<Identifiant connexion> : (Entier)`**

Identifiant de la connexion du poste client (obtenu par la fonction [NetConnecte](../WDLang3/3056004.md)).

**`<Événement> : (Entier ou chaîne de caractères Ansi)`**

Identifiant de l'événement Windows transmis aux fenêtres du serveur (numéro d'événement Windows ou chaîne de caractères spécifique).

**`<Paramètre 1> : (Entier optionnel)`**

Paramètre associé à l'événement.

**`<Paramètre 2> : (Entier optionnel)`**

Paramètre associé à l'événement.

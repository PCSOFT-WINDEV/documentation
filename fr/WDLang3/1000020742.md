


## xmlSignatureInformationCléX509 (Type de variable)

***En anglais : xmlSignatureInformationKeyX509***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **xmlSignatureInformationCléX509** permet de définir les informations à mettre dans la partie **KeyInfo** de la signature XML. Il est ainsi possible d'ajouter les informations sur un certificat X509. Les caractéristiques de ce type de variable peuvent être définies et modifiées à l'aide de différentes propriétés WLangage. 

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MaSignature est un xmlSignature
Info est un xmlSignatureInformationCléX509
Info.AvecCertificat = Vrai
Ajoute(MaSignature.InformationClé, Info)
```

## Remarques
<a name="NOTE0_1"></a>


### Propriétés spécifiques à la description des variables de type xmlSignatureInformationCléX509
<a name="proprietes_specifiques_description_des_variables_type_xmlsignatureinformationclex509_ELTPARAGRAPHE000052"></a>Les propriétés suivantes peuvent être utilisées pour manipuler une variable de type **xmlSignatureInformationCléX509** :

| **Nom de la propriété** | **Type manipulé** | **Effet** |
| --- | --- | --- |
| AvecCertificat | Booléen | <br><br>- <u><u><u><u>Vrai</u></u></u></u> : le certificat est ajouté dans la description de la signature, <br><br>- <u><u><u><u>Faux</u></u></u></u> : le certificat n'est pas ajouté dans la description de la signature.<br><br><br> |
| AvecEmetteur | Booléen | <br><br>- <u><u><u><u>Vrai</u></u></u></u> : le nom de l'émetteur du certificat est ajouté dans la description de la signature, <br><br>- <u><u><u><u>Faux</u></u></u></u> : le nom de l'émetteur du certificat n'est pas ajouté dans la description de la signature.<br><br><br> |
| AvecNomSujet | Booléen | <br><br>- <u><u><u><u>Vrai</u></u></u></u> : le nom du sujet du certificat est ajouté dans la description de la signature, <br><br>- <u><u><u><u>Faux</u></u></u></u> : le nom du sujet du certificat n'est pas ajouté dans la description de la signature.<br><br><br> |

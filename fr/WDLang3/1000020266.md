
## ContactListe (Fonction)

***En anglais : ContactList***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Liste les contacts de l'appareil correspondant aux critères spécifiés.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Rafraîchissement de la liste des contacts dans le champ Zone répétée
ZoneRépétéeSupprimeTout(ZR_CONTACTS)

// Récupération de la liste des contacts
SI SAI_Recherche <> "" ALORS
	gtabContacts = ContactListe(Null, contactNom, SAI_Recherche)
SINON
	gtabContacts = ContactListe(Null)
FIN

// Ajout dans le champ Zone répétée
POUR TOUT tContact DE gtabContacts
	SI tContact.NomAffiché <> "" ALORS
		ZoneRépétéeAjouteLigne(ZR_CONTACTS, tContact.NomAffiché, tContact)	
	FIN
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = ContactListe([<Source> [, <Filtre> [, <Borne minimale> [, <Borne maximale>]]]])`
---

**`<Résultat> : (Tableau de variables de type Contact)`**

Tableau de variables de type [Contact](../WDLang3/1000020254.md) initialisé avec la liste des contacts correspondant aux critères spécifiées. 
En cas d'erreur, la variable [ErreurDétectée](../WDLang1/3087001.md) est à <u><u><u><u>Vrai</u></u></u></u>. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).

**`<Source> : (Variable optionnelle de type ContactSource)`**

Nom de la variable de type [ContactSource](../WDLang3/1000020211.md) décrivant la source ou le compte pour lequel les contacts doivent être listés. 
Si ce paramètre vaut NULL, les contacts de toutes les sources de l'appareil seront pris en compte.

**`<Filtre> : (Constante optionnelle de type Entier)`**

Propriété sur laquelle la liste des contacts doit être filtrée : 


|   |   |
| --- | --- |
| *contactEmail* | Filtrage sur les adresses Email. |
| *contactNom* | Filtrage sur les noms de famille. |
| *contactPrénom* | Filtrage sur les prénoms. |
| *contactSociété* | Filtrage sur les sociétés. |
| *contactTéléphone*<br>(Valeur par défaut) | Filtrage sur les numéros de téléphone. |





**`<Borne minimale> : (Chaîne de caractères optionnelle)`**

Borne minimale du filtre.

**`<Borne maximale> : (Chaîne de caractères optionnelle)`**

Borne maximale du filtre.
Si ce paramètre n'est pas précisé, la borne maximale du filtre correspond à la valeur du paramètre &lt;Borne minimale&gt;.


## BTListeRadio (Fonction)

***En anglais : BTListRadio***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Liste les radios Bluetooth connectées à la machine.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
sChaîneTmp est une chaîne
sListePériphs est une chaîne
// Liste les radios
sListePériphs = BTListeRadio()
// Vide le champ Table affichant la liste des radios
TableSupprimeTout(TABLE_Radios)
// Remplit le champ Table
SI sListePériphs <> "" ALORS 
	POUR TOUTE CHAINE sChaîneTmp DE sListePériphs SEPAREE PAR RC
		TableAjoute(TABLE_Radios, sChaîneTmp)
	FIN
SINON
	Info(ErreurInfo(errComplet)) 
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = BTListeRadio([<Niveau de détail>])`
---

**`<Résultat> : (Chaîne de caractères)`**

Liste des radios disponibles. Cette liste est de la forme :

- en mode simple : 
	
	```txt
	<Identifiant Radio 1> + TAB + <Adresse MAC 1> + TAB + <Nom 1> + RC + ... + 
	RC + <Identifiant Radio N> + TAB + <Adresse MAC N> + TAB + <Nom N>
	```


- en mode détaillé : 
	
	```txt
	<Identifiant Radio 1> + TAB + <Adresse MAC 1> + TAB + <Nom 1> + TAB + 
	<Identifiant Classe 1> + TAB + <Identifiant fabricant 1> + TAB + 
	<Identifiant Version 1> + RC + ... + RC + <Identifiant Radio N> + TAB + 
	<Adresse MAC N> + TAB + <Nom N> + TAB + <Identifiant Classe N> + TAB + 
	<Identifiant fabricant N> + TAB + <Identifiant Version N>
	```





Les différentes informations obtenues sont les suivantes :

- &lt;Identifiant Radio&gt; : Identifiant de la radio, à utiliser dans les fonctions Bluetooth manipulant des radios. 

- &lt;Adresse MAC&gt; : Adresse physique de la radio

- &lt;Nom&gt; : Nom retourné par la radio (peut être une chaîne vide

- &lt;Identifiant Classe&gt; : Classe de radio (entier sur 32 permettant d'obtenir des informations sur le type de la radio)

- &lt;Identifiant fabricant&gt; : Identifiant du fabricant de la radio

- &lt;Identifiant Version&gt; : Identifiant propre à la radio (consulter sa documentation)




**`<Niveau de détail> : (Constante optionnelle de type Entier)`**

Niveau de détail du résultat :



|   |   |
| --- | --- |
| *btListeDétaillée* | &lt;Résultat&gt; contient la liste des radios en mode détaillé. |
| *btListeSimple*<br>(Valeur par défaut) | &lt;Résultat&gt; contient la liste des radios en mode simple. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Seules les piles bluetooth Microsoft, Bluesoleil et Broadcom (Widcomm) sont supportées par cette fonction. Pour connaître la pile utilisée, utilisez la fonction [BTPile](../WDLang3/1000017229.md).  

Les piles Broadcom (Widcomm) ne sont pas disponibles en 64 bits.

Pour plus de détails sur les clés supportées, consultez [Quelles piles utiliser ?](../WDLang3/1000017231.md)

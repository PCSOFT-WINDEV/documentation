


## &lt;Variable AuthToken&gt;.OpenIDLitIdentité (Fonction)

***En anglais : &lt;AuthToken variable&gt;.OpenIDReadIdentity***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Récupère l'identité d'un utilisateur contenue dans une variable de type [AuthToken](../WDLang3/1000022220.md). 

Remarque : Cette fonction peut être utilisée dans le cas d'une authentification utilisant le protocole OAuth 2.0 sur un Webservice quelconque.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
OpenIDParams est un OpenIDParamètres
OpenIDParams.URLConfiguration = "https://accounts.google.com/.well-known/openid-configuration"
OpenIDParams.ClientID = "106034950318?????????k59q.apps.googleusercontent.com" 
OpenIDParams.ClientSecret = "oBTQL52J??????-76MpiM"
OpenIDParams.URLRedirection = "http://localhost:9846"
	
MonToken est un AuthToken = AuthIdentifie(OpenIDParams)
SI ErreurDétectée ALORS
	Erreur(ErreurInfo())
	RETOUR
FIN

MonIdentité est un OpenIDIdentité = MonToken.OpenIDLitIdentité()
SI MonIdentité.Valide ALORS
	Trace(MonIdentité.Email)
	Trace(MonIdentité.Nom)
	Trace(MonIdentité.Prénom)
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Token d'authentification>.OpenIDLitIdentité()`
---

**`<Résultat> : (Variable de type OpenIDIdentité)`**

Variable de type [OpenIDIdentité](../WDLang3/1000023567.md) contenant les informations de l'utilisateur authentifié.

**`<Token d'authentification> : (Variable de type AuthToken)`**

Nom de la variable de type [AuthToken](../WDLang3/1000022220.md) correspondant au token (ou jeton) contenant les informations d'accès pour les prochaines requêtes authentifiées.

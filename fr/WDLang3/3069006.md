


## Fonctions SOAP
			

<DIV class="specObsolete">
	<IMG style="float:left;margin-right:10px;" src="https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=warning_big.png"><B>Avertissement</B><BR>
	 Cette fonctionnalité est réservée à un usage avancé pour gérer des cas particuliers. Dans la majorité des cas, il est recommandé d'importer le wsdl du webservice SOAP. Pour plus de détails, consultez [Importation/consommation de Webservices](../WDLang3/3086001.md). 
</DIV><a name="NOTE1"></a>
<a name="NOTE1_1"></a>


Les fonctions permettant de manipuler le protocole SOAP sont les suivantes :



|   |   |
| --- | --- |
| [SOAPAjouteAssertionSAML](../WDLang3/1000020762.md) | Ajoute une assertion SAML à une requête de Webservice. |
| [SOAPAjouteAttribut](../WDLang3/1000019241.md) | Ajoute un attribut à une variable automatique générée par l'import d'un Webservice. |
| [SOAPAjouteEntête](../WDLang3/3069001.md) | Ajoute un entête à la prochaine fonction SOAP d'un Webservice à exécuter. |
| [SOAPAjouteSignatureXML](../WDLang3/1000020754.md) | Ajoute une signature XML à une requête de Webservice. |
| [SOAPDonneRésultat](../WDLang3/3069002.md) | Renvoie le résultat de la dernière procédure SOAP exécutée avec succès. |
| [SOAPEncodeRésultat](../WDLang3/3069003.md) | Permet de paramétrer le format de la valeur retournée par le serveur SOAP WINDEV au programme client SOAP. |
| [SOAPErreur](../WDLang3/3069005.md) | Renvoie l'erreur de la dernière procédure SOAP ayant échoué. |
| [SOAPExécute](../WDLang3/3069013.md) | Exécute une procédure sur un serveur SOAP. |
| [SOAPExécuteXML](../WDLang3/3069014.md) | Exécute une procédure sur un serveur SOAP. |
| [SOAPPrépare](../WDLang3/1000019238.md) | Construit la requête SOAP d'appel à une fonction d'un WebService. |
| [SOAPRécupèreEntête](../WDLang3/1000019237.md) | Récupère un entête dans le message SOAP reçu suite à l'appel d'un Webservice. |
| [SOAPVersHTTP](../WDLang3/1000023920.md) | Construit la requête HTTP d'appel à une fonction d'un Webservice. Cette requête peut être modifiée pour l'adapter si nécessaire aux attentes du Webservice. Elle pourra ensuite être envoyée grâce à la fonction [HTTPEnvoie](../WDLang3/1000021183.md). |

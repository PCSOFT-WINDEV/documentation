
## telRaccroche (Fonction)

***En anglais : tapiHangUp***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Raccroche une ligne téléphonique ouverte avec la fonction [telCompose](../WDLang3/3076013.md) ou par la fonction [telRépondAppel](../WDLang3/3076026.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI telCompose(SAI_NuméroTéléphone, "EtatLigne") = Vrai ALORS
	Info("Cliquez sur OK pour raccrocher")
	telRaccroche()
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Raccrocher une ligne ouverte par la fonction telCompose

`telRaccroche()`
---


<a name="SYNTAXE2"></a>

### Raccrocher une ligne ouverte par la fonction telRépondAppel

`telRaccroche(<Identifiant de l'appel>)`
---

**`<Identifiant de l'appel> : (Entier)`**

Identifiant de l'appel à terminer. Cet identifiant est connu dans la procédure WLangage appelée par la fonction [telRépondAppel](../WDLang3/3076026.md).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Limitation
<a name="limitation_ELTPARAGRAPHE000046"></a>Il n'est pas possible d'appeler la fonction **telRaccroche** dans la procédure de détection d'appels (définie grâce à la fonction [telDémarreDétectionAppel](../WDLang3/3076029.md)).


<a name="NOTE0_2"></a>


### Configuration nécessaire
<a name="configuration_necessaire_ELTPARAGRAPHE000193"></a>Les fonctions de téléphonie utilisent la technologie TAPI 2.0 et/ou la technologie TAPI 3.1. Ces technologies sont utilisables sur tous les systèmes.

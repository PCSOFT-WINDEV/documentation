
## SaaSAdminSupprimeClient (Fonction)

***En anglais : SaaSAdminDeleteClient***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime un compte client SaaS.

**Remarques** : 

- L'utilisation de cette fonction nécessite d'avoir ouvert une session d'administration SaaS au préalable en utilisant la fonction [SaaSAdminConnecte](../WDLang3/1000019098.md).

- L'utilisation de cette fonction est réservée à l'Administrateur SaaS. Un gestionnaire de compte client ne peut pas utiliser cette fonction. 



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
S est un saasClient 
S = SaaSAdminChercheClient("Hybrid")

SI SaaSAdminSupprimeClient(S) = Vrai ALORS
	Info("Client supprimé.")
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = SaaSAdminSupprimeClient(<Compte client>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la suppression a été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Compte client> : (Variable de type saasClient ou chaîne de caractères)`**

Nom de la variable de type [saasClient](../WDLang3/1000019072.md) ou chaîne de caractères correspondant au compte client à supprimer le la base SaaS.

Si ce paramètre est une chaîne de caractères, celle-ci représente le nom du compte client ou sa propriété [RéférenceExterne](../WDLang3/1000019072.md).

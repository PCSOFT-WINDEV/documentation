


## &lt;Graphe&gt;.HMLibelléY (Fonction)

***En anglais : &lt;Chart&gt;.HMCaptionY***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie le libellé d'une catégorie sur l'axe des ordonnées d'un graphe de type Heatmap.

Note : Par défaut, le libellé d'une catégorie correspond à la valeur numérique de la catégorie.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
GRF_HMap.HMLibelléY(1, "Y1")
GRF_HMap.HMLibelléY(2, "Y2")
```

```wl
// Libellé des axes
POUR TOUT ELEMENT nIndice, nTempérature DE gtaTempérature
	GRF_Graphe.HMLibelléX(nIndice, "" + nTempérature)
FIN
POUR TOUT ELEMENT nIndice, nVent DE gtaVent
	GRF_Graphe.HMLibelléY(nIndice, "" + nVent)
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Champ Graphe>.HMLibelléY(<Y> , <Libellé>)`
---

**`<Champ Graphe> : (Nom de champ)`**

Nom du champ Graphe à manipuler (présent dans l'éditeur de fenêtres ou d'états).


**`<Y> : (Entier)`**

Position de la catégorie sur l'axe des ordonnées.

**`<Libellé> : (Chaîne de caractères)`**

Nouveau libellé de la catégorie. Si ce paramètre correspond à une chaîne vide (""), la valeur numérique de la catégorie est affichée.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Le libellé défini avec la fonction **&lt;Graphe&gt;.HMLibelléY** sera pris en compte lors du prochain appel à la fonction [&lt;Graphe&gt;.Dessine](../WDLang3/1000024062.md).




## EmailChangeDossier (Fonction)

***En anglais : EmailChangeFolder***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie le dossier en cours dans le logiciel de messagerie (Outlook, logiciel de messagerie utilisant le protocole IMAP, ...).   

Ce dossier est présent dans la branche "Dossiers personnels". 

Attention : Les dossiers de type "dossier générique" ne sont pas gérés. 

**Rappel** : Cette fonction ne permet pas de modifier le dossier en cours dans le logiciel de messagerie Outlook Express.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple
<a class="notetitle" target="_blank" href="$DOC$=1000003032135&name=emailchangedossier_fonction&product=WD">Voir des exemples supplémentaires</a>

```wl
// Protocole IMAP
SI EmailOuvreSessionIMAP(SAI_USER, SAI_PASSWORD, ...
		"imap.mondomaine.fr") = Vrai ALORS
	EmailChangeDossier(SAI_USER, "Dossiers personnels\Boîte de réception\Travail") 
	...
SINON
	Erreur("Impossible d'établir la connexion")
FIN
```
<a name="Exemple2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 
```wl
// Ouverture de la session
IDSession = EmailOuvreSessionOutlook("MonProfil")
...
// Changer le dossier en cours
EmailChangeDossier(IDSession, "Dossiers personnels\Boîte de réception\Travail")
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = EmailChangeDossier(<Session> , <Chemin du dossier>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la modification a bien été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire (par exemple, le chemin spécifié n'existe pas). En cas d'erreur, la variable [ErreurDétectée](../WDLang1/3087001.md) est à <u><u><u><u>Vrai</u></u></u></u>. Pour connaître le libellé de l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.




**`<Session> : (Chaîne de caractères, Entier ou variable de type emailSessionOutlook ou emailSessionIMAP)`**

Identifiant de la session email en cours. Cet identifiant peut être :

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) un entier renvoyé par la fonction [EmailOuvreSessionOutlook](../WDLang3/3032128.md) ou [OutlookOuvreSession](../WDLang3/3032123.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [emailSessionOutlook](../WDLang3/1000018767.md).

- une variable de type [emailSessionIMAP](../WDLang3/1000018957.md).

- le nom d'utilisateur passé à la fonction [EmailOuvreSessionIMAP](../WDLang3/1000019189.md).




**`<Chemin du dossier> : (Chaîne de caractères Unicode)`**

Chemin du nouveau dossier en cours.

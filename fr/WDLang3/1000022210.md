
## TypeEcran (Fonction)

***En anglais : ScreenType***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le type d'écran sur lequel s'exécute l'application : téléphone, tablette, ordinateur ou montre.

Remarques :

- En mode simulateur, le type d'écran correspond à un écran d'ordinateur. 



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Si l'application est sur un grand écran (tablette ou PC)
SI TypeEcran() DANS (teTablette, teOrdinateur) ALORS
	...
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = TypeEcran()`
---

**`<Résultat> : (Constante de type Entier)`**

Type de l'écran :


|   |   |
| --- | --- |
| *teMontre* | Ecran de montre (diagonale inférieure à 2 pouces). |
| *teOrdinateur* | Ecran d'ordinateur (diagonale supérieure à 13 pouces). |
| *teTablette* | Ecran de tablette (diagonale comprise entre 7 et 13 pouces). |
| *teTéléphone* | Ecran de téléphone (diagonale comprise entre 2 et 7 pouces). |

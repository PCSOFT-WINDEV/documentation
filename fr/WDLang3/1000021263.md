


## EmailRécupèreIndicateur (Fonction)

***En anglais : EmailGetIndicator***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Récupère les indicateurs définis sur un email d'une session IMAP.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
cMaSession est une emailSessionIMAP
...
tabIndicateur est un tableau de chaînes
cMonEmail est un Email
... 
SI EmailOuvreSession(cMaSession) ALORS 
	SI EmailLitPremier(cMaSession, cMonEmail) ALORS
		tabIndicateur = EmailRécupèreIndicateur(cMaSession, cMonEmail)
		POUR TOUT sIndicateur de tabIndicateur
			Trace(sIndicateur)
		FIN
	FIN
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = EmailRécupèreIndicateur(<Session IMAP> , <Email>)`
---

**`<Résultat> : (Tableau de chaîne de caractères)`**

Tableau contenant l'ensemble des indicateurs définis sur le message. Ces indicateurs correspondent : 

- à la liste des indicateurs définis par la RFC (\\Seen, \\Answered, \\Flagged, \\Deleted, \\Draft et \\Recent), 

- aux indicateurs définis par le client. 




**`<Session IMAP> : (Variable de type emailSessionIMAP)`**

Nom de la variable de type [emailSessionIMAP](../WDLang3/1000018957.md) correspondant à la session IMAP associée à l'email. 

**`<Email> : (Variable de type Email)`**

Nom de la variable de type [Email](../WDLang3/1000018713.md) correspondant à l'email à manipuler.

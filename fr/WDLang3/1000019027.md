Les types avancés SaaS sont les suivants :



|   |   |
| --- | --- |
| [saasAbonnement](../WDLang3/1000019088.md) | Le type **saasAbonnement** permet de définir toutes les caractéristiques avancées d'un abonnement d'un compte client SaaS à une tarification. |
| [saasClient](../WDLang3/1000019072.md) | Le type **saasClient** permet de définir toutes les caractéristiques avancées d'un Compte Client SaaS. |
| [saasService](../WDLang3/1000019096.md) | Le type **saasService** permet de définir toutes les caractéristiques avancées d'un service SaaS. |
| [saasSite](../WDLang3/1000019068.md) | Le type **saasSite** permet de définir toutes les caractéristiques avancées d'un site WEBDEV SaaS. |
| [saasTarification](../WDLang3/1000019091.md) | Le type **saasTarification** permet de définir toutes les caractéristiques avancées d'une tarification SaaS. |
| [saasUtilisateur](../WDLang3/1000019083.md) | Le type **saasUtilisateur** permet de définir toutes les caractéristiques avancées d'un utilisateur de site SaaS. |


## NotesDimensionChamp (Fonction)

***En anglais : NotesItemDimension***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la dimension d'un champ du document en cours dans Lotus Notes.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ouverture d'une session Lotus Notes
IDSession est un entier
IDSession = EmailOuvreSessionNotes("MotDePasse", "ServeurMars", ...
		"mail\julie.nsf", "C:\Users\Julie\AppData\Local\Lotus\Notes\Data\user.id")
...
// Lecture du premier email
EmailLitPremier(IDSession)
// Dimension du champ Lotus Notes "INetSendTo"
ResDimension est un entier
ResDimension = NotesDimensionChamp(IDSession, "INetSendTo")
Info("Cet email a été envoyé à " + ResDimension + " personnes")
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = NotesDimensionChamp(<Identifiant de la session> , <Champ Lotus Notes>)`
---

**`<Résultat> : (Entier)`**



- Dimension du champ Lotus Notes spécifié, 

- 0 si le champ Lotus Notes spécifié n'existe pas.




**`<Identifiant de la session> : (Entier)`**

Identifiant de la session à utiliser. Cet identifiant est renvoyé par la fonction [EmailOuvreSessionNotes](../WDLang3/3032121.md) ou [NotesOuvreBase](../WDLang3/1000005101.md).

**`<Champ Lotus Notes> : (Chaîne de caractères)`**

Nom du champ Lotus Notes dont on veut connaître la dimension.

Pour connaître le nom et le type des champs du document Notes en cours, utilisez la fonction [NotesListeChamp](../WDLang3/1000005112.md).

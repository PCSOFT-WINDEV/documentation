


## &lt;Graphe&gt;.EtiquetteCatégorie (Fonction)

***En anglais : &lt;Chart&gt;.CategoryLabel***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de : 

- Initialiser l'étiquette d'une catégorie de données (ou d'un secteur) d'un graphe. 
	

- Connaître l'étiquette d'une catégorie de données (ou d'un secteur) d'un graphe.




Exemple d'étiquette de catégorie :
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=EtiquetteCat.gif)


**Rappel** : Par défaut, un élément de graphe n'a aucune étiquette.

![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Les graphes](../WDChamp/3042060.md). 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Initialisation de l'étiquette "Février" de la seconde catégorie de données
// dans le graphe "GRF_MonGraphe"
GRF_MonGraphe.EtiquetteCatégorie(2, "Février")
```
<a name="3042015_Exemple3"></a>

```wl
// Lecture de l'étiquette de la seconde catégorie de données
// dans le graphe "GRF_MonGraphe"
MonEtiquette est une chaîne = GRF_MonGraphe.EtiquetteCatégorie(2)
```

<a name="SYNTAXE1"></a>

## Syntaxe

### Initialiser l'étiquette d'une catégorie de données (ou d'un secteur) d'un graphe

`<Champ Graphe>.EtiquetteCatégorie(<Indice> , <Étiquette>)`
---

**`<Champ Graphe> : (Nom de champ)`**

Nom du champ Graphe à manipuler (présent dans l'éditeur de fenêtres, de pages ou d'états).


**`<Indice> : (Entier)`**

Indice de la donnée dans la série.

**`<Étiquette> : (Chaîne de caractères)`**

Libellé de l'étiquette à initialiser. 
Ce paramètre peut également correspondre à une durée, une date, une heure ou une date-heure. 
**Note** : Si vous utilisez une échelle des abscisses temporelle, il est nécessaire d'appeler la fonction **&lt;Graphe&gt;.EtiquetteCatégorie** systématiquement à chaque ajout de données dans le graphe avec la fonction [&lt;Graphe&gt;.AjouteDonnée](../WDLang3/1000023533.md).


<a name="SYNTAXE2"></a>

### Connaître l'étiquette d'une catégorie de données (ou d'un secteur) d'un graphe

`<Résultat> = grEtiquetteCatégorie(<Champ Graphe> , <Indice>)`
---

**`<Résultat> : (Chaîne de caractères)`**

Libellé de l'étiquette.

**`<Champ Graphe> : (Nom de champ)`**

Nom du champ Graphe à manipuler (présent dans l'éditeur de fenêtres, de pages ou d'états).


**`<Indice> : (Entier)`**

Indice de la donnée (dont l'étiquette est recherchée) dans la série.

## Remarques


### Gestion des étiquettes de catégories
<a name="gestion_des_etiquettes_categories_ELTPARAGRAPHE000341"></a>Les étiquettes de catégories sont affichées :

- sous les graduations horizontales pour les graphes de type courbe ou les graphes boursiers de type Chandelier japonais et BarCharts,

- sous l'axe des X pour les histogrammes ou les graphes de type boursier Minimum/Maximum.


![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=EtiquetteCat.gif)
Les étiquettes de catégories ne sont pas utilisables avec les graphes de type Nuage de points.

La position des étiquettes de catégorie ne peut pas être modifiée. 

La définition des étiquettes de catégories est réalisée avec la fonction **&lt;Graphe&gt;.EtiquetteCatégorie**.

Les données définies avec la fonction **&lt;Graphe&gt;.EtiquetteCatégorie** seront prises en compte lors du prochain appel à la fonction [&lt;Graphe&gt;.Dessine](../WDLang3/1000024062.md).

Il est possible de modifier la police de l'étiquette grâce à la fonction [&lt;Graphe&gt;.PoliceEtiquette](../WDLang3/1000024117.md).

Les étiquettes de catégories définies avec la fonction **&lt;Graphe&gt;.EtiquetteCatégorie** peuvent être affichées dans le libellé du graphe de type Secteur (constante *grAfficheEtiquette* utilisée avec la fonction [&lt;Graphe&gt;.Libellé](../WDLang3/1000024165.md)).


## HTTPAjouteFichier (Fonction)

***En anglais : HTTPAddFile***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ajoute un fichier dans le formulaire HTTP. Ce formulaire a été créé grâce à la fonction [HTTPCréeFormulaire](../WDLang3/1000017119.md) et sera envoyé grâce à la fonction [HTTPEnvoieFormulaire](../WDLang3/1000017190.md).




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
HTTPCréeFormulaire("FORM")
HTTPAjouteParamètre("FORM", "Prix", "3400")
HTTPAjouteFichier("FORM", "Photo", "C:\Temp\MaPhoto.bmp")
...
HTTPEnvoieFormulaire("FORM", "www.monsite.fr")
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = HTTPAjouteFichier(<Nom du formulaire> , <Nom du paramètre> , <Nom du fichier> [, <Type de fichier>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le fichier a été ajouté au formulaire HTTP, 

- <u><u><u><u>Faux</u></u></u></u> en cas d'erreur. Pour connaître le libellé de l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.




**`<Nom du formulaire> : (Chaîne de caractères)`**

Nom du formulaire HTTP à manipuler. Ce nom a été donné grâce à la fonction [HTTPCréeFormulaire](../WDLang3/1000017119.md). Si le formulaire n'existe pas, une erreur est affichée.

**`<Nom du paramètre> : (Chaîne de caractères)`**

Nom du paramètre à ajouter dans le formulaire. Ce paramètre sera lié au fichier. Si le nom du paramètre existe déjà dans le formulaire, sa valeur sera écrasée.

**`<Nom du fichier> : (Chaîne de caractères)`**

Chemin complet du fichier à ajouter dans le formulaire. Le fichier doit être accessible depuis le poste en cours. En effet, le fichier est immédiatement ouvert en lecture et bloqué en écriture. Le fichier sera lu uniquement pendant l'exécution de la fonction [HTTPEnvoieFormulaire](../WDLang3/1000017190.md).

**`<Type de fichier> : (Chaîne de caractères)`**

Description du type MIME du fichier. Ce paramètre correspond à l'entête "Content-Type".

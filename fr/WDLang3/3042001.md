 



## grCréePolice (Fonction)

***En anglais : grCreateFont***
	

<a name="XUtilisation"></a>
<a name="Utilisation"></a>
|  |   |
| --- | --- |

|  
 | **But**<br><br>Créer une police pour les graphes.<br><br>**Syntaxe**<br><br>grCréePolice(&lt;NumPolice&gt;, &lt;NbPixelsHauteur&gt;, &lt;Style&gt;, &lt;NomPoliceWindows&gt; [,&lt;Couleur&gt;])<br><br>&lt;NumPolice&gt; est le numéro (entier) identifiant la police créée<br><br>&lt;NbPixelsHauteur&gt; est un entier correspondant à la hauteur de la police en pixels.<br><br>&lt;Style&gt; est l'attribut (ou style) de la police<br><br>- iNormal police normale (sans attribut)<br><br>- iGras police en gras (attribut gras)<br><br>- iSouligne police soulignée (attribut souligné)<br><br>- iItalique police italique (attribut italique)<br><br>- iBarre police barrée<br><br><br><br><br>&lt;NomPoliceWindows&gt; est le nom de la police choisie (parmi celles installées sur le poste)<br><br>&lt;Couleur&gt; est la couleur dans laquelle est affiché le texte. Cette couleur correspond à une [constante prédéfinie](../WDLang5/3010002.md) (Noir par défaut)<br><br>**Détail**<br><br>Les polices créées avec **grCréePolice** sont automatiquement détruites à la fermeture de l'application.<br><br>La création d'une police avec un numéro de police existante écrase automatiquement l'ancienne police.<br><br>La couleur utilisée par défaut pour les polices est le noir (Noir).<br><br>**Notes**<br><br>- La fonction [grSupprimeTout](../WDLang3/3042055.md) ne détruit pas les polices créées avec **grCréePolice	**.<br><br>- Une série ne doit pas être composée de plus de 8000 éléments.<br><br><br> |
| --- | --- |

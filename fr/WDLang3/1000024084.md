


## &lt;Graphe&gt;.HMSupprimeCouleur (Fonction)

***En anglais : &lt;Chart&gt;.HMDeleteColor***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime les couleurs spécifiées par la fonction [&lt;Graphe&gt;.HMCouleur](../WDLang3/1000024080.md) dans un graphe de type Heatmap. 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// On fixe les couleurs
GRF_Graphe.HMSupprimeCouleur()
GRF_Graphe.HMCouleur(-65, RVB(0, 105, 255))
GRF_Graphe.HMCouleur(0, RVB(240, 255, 0))
GRF_Graphe.HMCouleur(35, RVB(255, 31, 0))
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Champ Graphe>.HMSupprimeCouleur([<Palier>])`
---

**`<Champ Graphe> : (Nom de champ)`**

Nom du champ Graphe à manipuler (présent dans l'éditeur de fenêtres ou d'états).


**`<Palier> : (Réel optionnel)`**

Palier de la couleur qui doit être supprimée. Si ce paramètre n'est pas spécifié, toutes les couleurs de palier sont supprimées et la couleur de la première série est utilisée.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Les données modifiées avec la fonction **&lt;Graphe&gt;.HMSupprimeCouleur** seront prises en compte lors du prochain appel à la fonction [&lt;Graphe&gt;.Dessine](../WDLang3/1000024062.md).

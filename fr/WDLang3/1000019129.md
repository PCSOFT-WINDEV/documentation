
## SaaSAdminSupprimeTarification (Fonction)

***En anglais : SaaSAdminDeletePricing***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime une tarification SaaS. Ses abonnements sont automatiquement supprimés. 

**Remarques** : 

- L'utilisation de cette fonction nécessite d'avoir ouvert une session d'administration SaaS au préalable en utilisant la fonction [SaaSAdminConnecte](../WDLang3/1000019098.md).

- L'utilisation de cette fonction est réservée à l'Administrateur SaaS. Un gestionnaire de compte client ne peut pas utiliser cette fonction. 



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Suppression de la tarification "Promo 2009" dans le site "Forum"
S est un saasSite 
T est une saasTarification

S = SaaSAdminChercheSite("Forum")
T = SaaSAdminChercheTarification("Promo 2009", S)

SaaSAdminSupprimeTarification(T)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Supprimer une tarification (utilisation des variables de type saasxxx)

`<Résultat> = SaaSAdminSupprimeTarification(<Tarification>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la suppression a été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Tarification> : (Variable de type saasTarification)`**

Nom de la variable de type [saasTarification](../WDLang3/1000019091.md) correspondant à la tarification à supprimer de la base SaaS.


<a name="SYNTAXE2"></a>

### Supprimer une tarification identifiée par son nom

`<Résultat> = SaaSAdminSupprimeTarification(<Tarification> , <Site SaaS>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la suppression a été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Tarification> : (Chaîne de caractères)`**

Nom de la tarification à supprimer de la base SaaS.

**`<Site SaaS> : (Chaîne de caractères)`**

Nom du site SaaS possédant la tarification.

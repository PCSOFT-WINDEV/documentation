<a name="Utilisation"></a>
<a name="description"></a>
Envoie un message du poste serveur vers tous les postes clients connectés. Le poste serveur est un serveur FTP (File Transfer Protocol) ou RPC (Remote Procedure Call) réalisé avec WINDEV.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple
<a class="notetitle" target="_blank" href="$DOC$=1000003056007&name=netenvoiemessageclient_fonction&product=WD">Voir des exemples supplémentaires</a>

```wl
NetEnvoieMessageClient(IDServeur, "Reçu", 0, 0)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`NetEnvoieMessageClient(<Identifiant du serveur> , <Événement> [, <Paramètre 1> [, <Paramètre 2>]])`
---

**`<Identifiant du serveur> : (Entier)`**

Identifiant du serveur utilisé (obtenu par la fonction [NetDémarreServeur](../WDLang3/3056013.md)).

**`<Événement> : (Entier ou chaîne de caractères Ansi)`**

Identifiant de l'événement Windows transmis aux fenêtres des clients (numéro d'événement Windows ou chaîne de caractères spécifique).

**`<Paramètre 1> : (Entier optionnel)`**

Paramètre associé à l'événement.

**`<Paramètre 2> : (Entier optionnel)`**

Paramètre associé à l'événement.

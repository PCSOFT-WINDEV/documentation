


## EmailConstruitSource (Fonction)

***En anglais : EmailBuildSource***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Génère le code source de l'email à envoyer à partir des variables actuellement présentes dans la structure [Email](../WDLang3/3032029.md) ou dans une variable de type [Email](../WDLang3/1000018713.md). 

Le code source est généré dans la variable **Email.Source** ou dans la propriété **Source** de la variable [Email](../WDLang3/1000018713.md). Ce code source peut ensuite être sauvé dans un fichier .eml par exemple.

**Remarque** : Cette fonction peut être utile pour modifier un élément dans un email à envoyer. Le mail pourra être envoyé par la fonction [EmailEnvoie](../WDLang3/3032003.md). Pour plus de détails, consultez [Ecrire un email personnalisé](../WDLang3/3032002.md).






<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple



### Exemple avec une variable de type Email
<a name="exemple_avec_une_variable_type_email_ELTPARAGRAPHE000109"></a>
```wl
// Initialisation de la variable
MonMessage est un Email

Ajoute(MonMessage.Destinataire, "support_test@pcsoft_test.fr")
MonMessage.Expéditeur = "client@societe.fr"
MonMessage.Message = "Test Auto eMail"

// Création du fichier .eml
SI EmailConstruitSource(MonMessage) = Vrai ALORS
	fSauveTexte("c:\MesEmails\Test.eml", MonMessage.Source)
FIN
```
<a name="Exemple2"></a>


### Exemple avec la structure Email
<a name="exemple_avec_structure_email_ELTPARAGRAPHE000121"></a>
```wl
// Initialisation de la structure email
Email.NbDestinataire = 1
Email.Destinataire = "support_test@pcsoft_test.fr"
Email.Expéditeur = "client@societe.fr"
Email.Message = "Test Auto eMail"

// Création du fichier .eml
SI EmailConstruitSource() = Vrai ALORS
	fSauveTexte("c:\MesEmails\Test.eml", Email.Source)
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = EmailConstruitSource([<Email> [, <Options avancées>]])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le code source a été généré, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. En cas d'erreur, la variable [ErreurDétectée](../WDLang1/3087001.md) est à <u><u><u><u>Vrai</u></u></u></u>. 
	Pour connaître le libellé de l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*. Le code généré peut être lu dans la variable Email.Source du thread en cours.




**`<Email> : (Variable optionnelle de type Email)`**

Nom de la variable de type [Email](../WDLang3/1000018713.md) à manipuler.
Si ce paramètre n'est pas spécifié, la structure [Email](../WDLang3/3032029.md) est utilisée.

**`<Options avancées> : (Constante optionnelle de type Entier)`**

Combinaison des constantes suivantes :


|   |   |
| --- | --- |
| *emailOptionEncodeEntête* | Encode les entêtes du message en Quoted-Printable si nécessaire. <br>Par défaut, l'entête du message n'est pas encodé. Cette constante évite les messages avec des accents mal encodés. |
| *emailOptionEntêteCCi* | Permet d'ajouter les destinataires non officiels dans le code source de l'entête de l'email. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Pour remplir la structure [Email](../WDLang3/3032029.md) ou une variable de type [Email](../WDLang3/1000018713.md) à partir du contenu de sa propriété **Source**, utilisez la fonction [EmailImporteSource](../WDLang3/1000017226.md).

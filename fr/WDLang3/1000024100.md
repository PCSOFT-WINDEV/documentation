


## &lt;Graphe&gt;.Nuage3DProjection (Fonction)

***En anglais : &lt;Chart&gt;.Scatter3DProjection***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Indique si les points d'une série d'un graphe de type Nuage de points 3D doivent être projetés sur un plan donné. Dans ce cas, une ligne droite reliera le point à sa projection sur le plan correspondant.
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=grSurfaceQuadrillage.gif)

<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Projette en pointillés les points de la série 1 sur le plan XY
GRF_MonNuage3D.Nuage3DProjection(1, grPlanXY, TraitPointillé)
```

```wl
// Initialisation du champ Combo
// Remplit la liste des types de trait
ListeAjoute(MoiMême, "Aucun trait" + gValeurMémorisée(TraitAucun))
ListeAjoute(MoiMême, "Trait continu" + gValeurMémorisée(TraitContinu))
ListeAjoute(MoiMême, "Trait tiret" + gValeurMémorisée(TraitTiret))
ListeAjoute(MoiMême, "Trait pointillé" + gValeurMémorisée(TraitPointillé))
ListeAjoute(MoiMême, "Trait mixte" + gValeurMémorisée(TraitMixte))

// Par défaut, le trait appliqué est mixte pour ce plan
MoiMême = TraitMixte

// -----------------------------------------------------
// Code de sélection d'une ligne du champ Combo

// Applique le type de projection pour le plan XY
GRF_Graphe.Nuage3DProjection(1, grPlanXY, MoiMême)
GRF_Graphe.Dessine()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Champ Graphe>.Nuage3DProjection(<Numéro de la série> , <Plan de projection> [, <Type de trait>])`
---

**`<Champ Graphe> : (Nom de champ)`**

Nom du champ Graphe à manipuler (présent dans l'éditeur de fenêtres).

**`<Numéro de la série> : (Entier)`**

Numéro de la série à manipuler.

**`<Plan de projection> : (Constante de type Entier)`**

Plan sur lequel les points doivent être projetés : 


|   |   |
| --- | --- |
| *grPlanXY* | Plan XY. |
| *grPlanXZ* | Plan XZ. |
| *grPlanYZ* | Plan YZ. |



**`<Type de trait> : (Constante optionnelle de type Entier)`**

Type du trait permettant de visualiser la projection : 



|   |   |
| --- | --- |
| *TraitAucun* | Aucun trait n'est tracé. |
| *TraitContinu* <br>(Valeur par défaut) | Trait plein continu. |
| *TraitMixte* | Trait alternant pointillé et tiret (.-.-.-). |
| *TraitPointillé* | Trait en pointillé (........). |
| *TraitTiret* | Trait avec des tirets (------). |

## Remarques


### Paramétrer l'affichage des projections
<a name="parametrer_affichage_des_projections_ELTPARAGRAPHE000232"></a>Par défaut, les projections sont toujours visibles. Il est possible de paramétrer l'affichage des projections à l'aide de la fonction [&lt;Graphe&gt;.Paramètre](../WDLang3/1000024115.md). 

Il est possible : 

- de ne pas afficher les projections.
	
	```wl
	GRF_MonNuage3D.Paramètre(grProjectionNuage3D, grProjectionAucune)
	```


- d'afficher toujours les projections.
	
	```wl
	GRF_MonNuage3D.Paramètre(grProjectionNuage3D, grProjectionToujoursVisible)
	```


- d'afficher les projections uniquement lors du clic sur le point.
	
	```wl
	GRF_MonNuage3D.Paramètre(grProjectionNuage3D, grProjectionClic)
	```

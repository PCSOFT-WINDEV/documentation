
## RendezVousCrée (Fonction)

***En anglais : AppointmentCreate***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Affiche la fenêtre de création de rendez-vous de l'application native de gestion des rendez-vous de l'appareil Android.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
rdv est un RendezVous
SI RendezVousCrée(rdv) ALORS
	Info(ChaîneConstruit("Le rendez-vous %1 a été ajouté au calendrier %2", ...
			rdv.Nom, rdv.Calendrier.Nom))
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = RendezVousCrée(<RendezVous>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si un rendez-vous a été créé, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<RendezVous> : (Variable de type RendezVous)`**

Nom de la variable de type [RendezVous](../WDLang1/1000019244.md) à renseigner avec les données correspondant au rendez-vous créé.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Cas particuliers
<a name="cas_particuliers_ELTPARAGRAPHE000047"></a>

- Pour créer un nouveau rendez-vous entièrement par programmation, sans ouvrir de fenêtre, utilisez la fonction [RendezVousAjoute](../WDLang3/1000011001.md).

- La fonction **RendezVousCrée** est bloquante jusqu'à ce que l'utilisateur valide ou annule la fenêtre de création de rendez-vous.

- La fonction **RendezVousCrée** peut être utilisée dans l'émulateur mais pas dans le simulateur.

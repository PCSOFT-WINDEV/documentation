


## &lt;Variable zipArchive&gt;.NbPartieNécessaire (Fonction)

***En anglais : &lt;zipArchive variable&gt;.NbPartNeeded***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le nombre de parties d'une taille donnée nécessaires pour découper une archive mono-partie. 

**Remarque** : Cette fonction est disponible uniquement pour les archives au format ZIP et WDZ.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) 
```wl
MonArchive est un zipArchive
// Ouverture d'une archive
ResOuvreArchive = MonArchive.Ouvre("C:\MesArchives\Archive.zip")

SI ResOuvreArchive = 0 ALORS
	...
	// Nombre de parties de 1,4 Mo nécessaires pour contenir l'archive
	ResNbPartieNécessaire = MonArchive.NbPartieNécessaire(1400*1024)
	Info("Pour découper cette archive, " + ResNbPartieNécessaire + ...
				" partie(s) de 1,4 Mo sont nécessaires")
	...
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Archive>.NbPartieNécessaire(<Taille des parties>)`
---

**`<Résultat> : (Entier)`**

Nombre de parties d'une taille donnée nécessaires pour contenir une archive.

**`<Archive> : (Variable de type zipArchive)`**

Nom d'une variable de type [zipArchive](../WDLang3/1000018679.md) correspondant à l'archive à manipuler.

**`<Taille des parties> : (Entier)`**

Taille des parties en octets. La taille par défaut est égale à 1,4 Mo (1400\*1024 octets). La taille minimale est égale à 300 Ko par partie (300\*1024 octets).



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Découpage d'une archive
<a name="decoupage_une_archive_ELTPARAGRAPHE000225"></a>La création d'une archive multi-parties est réalisable grâce à la fonction [&lt;Variable zipArchive&gt;.Découpe](../WDLang3/1000019161.md). La fonction **&lt;Variable zipArchive&gt;.NbPartieNécessaire** permet de connaître le nombre de parties nécessaires au découpage de l'archive.
<a name="NOTE0_2"></a>


### Archive multi-parties
<a name="archive_multiparties_ELTPARAGRAPHE000243"></a>L'utilisation de la fonction **&lt;Variable zipArchive&gt;.NbPartieNécessaire** sur une archive multi-parties permet de calculer le nombre de parties nécessaires en vue d'un nouveau découpage.

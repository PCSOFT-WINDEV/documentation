
## sDansFileSortie (Fonction)

***En anglais : sInExitQueue***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Récupère le nombre d'octets en attente de transfert dans le buffer de sortie du port série, du port parallèle ou du port infrarouge spécifié.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Le port COM2 est ouvert
// Envoi d'un message au buffer de sortie de COM2
sEcrit(2, MonMessage)
// Attendre la fin de l'écriture
BOUCLE
	SI sDansFileSortie(2) = 0 ALORS SORTIR
FIN
Info("Fin de l'écriture")
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = sDansFileSortie(<Numéro de port>)`
---

**`<Résultat> : (Entier)`**

Nombre d'octets en attente de transfert. 

**`<Numéro de port> : (Entier)`**



- Numéro du port série concerné : 1, 2, 3, ... 32 pour COM1, COM2, COM3, ... COM32.

- Numéro de port parallèle concerné : -1, -2, -3 ou -4 pour LPT1, LPT2, LPT3 ou LPT4.

- Numéro du port infrarouge concerné : 1, 2, 3, ... 32 pour IR1, IR2, IR3, ... IR32.

- Numéro de port renvoyé par la fonction [sOuvre](../WDLang3/3062013.md) (si cette fonction a été appelée avec un nom de port).

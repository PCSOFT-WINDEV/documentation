


## &lt;Graphe&gt;.CPAjouteDonnée (Fonction)

***En anglais : &lt;Chart&gt;.RBAddData***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ajoute une donnée à un graphe circulaire de progrès.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
GRF_Graphe1.CPAjouteDonnée("Montpellier", 10)
GRF_Graphe1.CPAjouteDonnée("France", 50)
GRF_Graphe1.CPAjouteDonnée("Europe", 80)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Champ Graphe>.CPAjouteDonnée(<Catégorie> , <Valeur>)`
---

**`<Champ Graphe> : (Nom de champ)`**

Nom du champ Graphe manipulé (présent dans l'éditeur de fenêtres ou d'états).

**`<Catégorie> : (Chaîne de caractères)`**

Nom de la catégorie associée à la donnée.

**`<Valeur> : (Entier ou réel)`**

Valeur à ajouter. 
Par défaut, la valeur est considérée en pourcentage. Si ce n'est pas le cas, il est possible de donner la valeur "maximale" avec la fonction [&lt;Graphe&gt;.CPValeurMaximale](../WDLang3/1410088094.md) (le pourcentage sera automatiquement calculé à partir de cette valeur).

## Remarques


### Ajout de données dans un graphe
<a name="ajout_donnees_dans_graphe_ELTPARAGRAPHE000189"></a>La fonction [&lt;Graphe&gt;.AjouteDonnée](../WDLang3/1000023533.md) permet d'ajouter des données dans un graphe (secteur, histogramme, ...).

Selon le type de graphe, des fonctions spécifiques permettent également d'ajouter des données : 

- Pour les graphes boursiers, il est conseillé d'utiliser la fonction [&lt;Graphe&gt;.BourseAjouteDonnée](../WDLang3/1000023565.md).

- Pour les graphes Heatmap, il est conseillé d'utiliser la fonction [&lt;Graphe&gt;.HMAjouteDonnée](../WDLang3/1000024079.md).

- Pour les graphes de type Nuage de points, il est conseillé d'utiliser la fonction [&lt;Graphe&gt;.NuageAjouteDonnéeXY](../WDLang3/1000024105.md).

- Pour les graphes Secteur Multiniveau, il est conseillé d'utiliser la fonction [&lt;Graphe&gt;.SMNAjouteDonnée](../WDLang3/1000024132.md). 

- Pour les graphes de type Surface, il est conseillé d'utiliser la fonction [&lt;Graphe&gt;.SurfaceAjouteDonnée](../WDLang3/1000024143.md).




Rappel : Pour ajouter une valeur à une donnée d'un graphe, utilisez la fonction [&lt;Graphe&gt;.IncrémenteDonnée](../WDLang3/1000024086.md).

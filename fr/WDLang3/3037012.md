
## FaxRéinitialise (Fonction)

***En anglais : FaxRestart***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Redémarre un fax dans la file d'attente. Cette fonction ne peut être utilisée que sur les fax en attente de transmission.

**Remarque** : Cette fonction ne peut être utilisée que sous Windows 2000, XP ou version supérieure. Sous Windows Vista, la gestion des fax est disponible uniquement avec Windows Vista Professionnel et Windows Vista Edition Intégrale.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
IDConnect est un entier
IDConnect = FaxConnecte()
SI IDConnect = 0 ALORS 
	Erreur("La connexion au serveur de fax a échoué." + ...
		 "Vérifiez que celui-ci est bien démarré", ...
		ErreurInfo(errMessageSystème))
SINON
	 ListeAjoute(LISTE_FaxEnvoi, FaxBoîteEnvoi(IDConnect))
FIN
...
// Code de clic de la liste
SI FaxEtat(LISTE_FaxEnvoi..ValeurAffichée) = FaxEtatPrisEnCharge ALORS
	 FaxRéinitialise(LISTE_FaxEnvoi..ValeurAffichée) 
FIN

...
FaxDéconnecte(IDConnect)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = FaxRéinitialise(<Identifiant connexion> , <Nom du fax ou Identifiant du fax>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le fax est redémarré dans la file d'attente, 

- <u><u><u><u>Faux</u></u></u></u> en cas d'erreur (par exemple si le nom du fax n'existe pas).




**`<Identifiant connexion> : (Entier)`**

Identifiant de la connexion obtenu avec la fonction [FaxConnecte](../WDLang3/3037005.md).

**`<Nom du fax ou Identifiant du fax> : (Chaîne de caractères)`**

Correspond :

- soit au nom permettant d'identifier le fax envoyé. Ce nom a été donné avec la fonction [FaxEnvoie](../WDLang3/3037006.md).

- soit à l'identifiant du fax (obtenu par les fonctions [FaxBoîteEnvoi](../WDLang3/3037008.md) ou [FaxBoîteRéception](../WDLang3/3037009.md))




## &lt;Variable Email&gt;.ChargeFichierAttaché (Fonction)

***En anglais : &lt;Email variable&gt;.LoadAttachment***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ajoute un fichier en attachement d'un email.

<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ajout d'un fichier dans les pièces jointes d'un email
MonMessage est un Email
MonMessage.ChargeFichierAttaché(...
	SysRep(srMesDocuments) + ["\"] + "Rapport Annuel.pdf")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Email>.ChargeFichierAttaché(<Nom et chemin d'un fichier>)`
---

**`<Résultat> : (Entier)`**



- Rang du fichier dans le tableau des fichiers attachés (variable [Email.Attache](../WDLang3/1000018713.md)), 

- -1 en cas d'erreur. Pour connaître le libellé de l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Email> : (Variable de type Email)`**

Nom de la variable de type [Email](../WDLang3/1000018713.md) auquel le fichier doit être attaché.

**`<Nom et chemin d'un fichier> : (Chaîne de caractères)`**

Nom et chemin complet (ou relatif) du fichier à attacher. Un chemin de type UNC peut être utilisé. Il n'est pas possible d'utiliser des caractères joker ("\*" ou "?").

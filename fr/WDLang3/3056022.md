
## NetOptionServeur (Fonction)

***En anglais : NetServerOption***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Gère les droits des clients sur le poste serveur (serveur FTP (File Transfer Protocol) ou RPC (Remote Procedure Call) réalisé avec WINDEV). Les autorisations données ou refusées sont valables pour tous les clients qui se connecteront au serveur.

**Attention** : **Cette fonction peut être exécutée uniquement sur le serveur.** Par défaut, le serveur autorise l'exécution de toutes les fonctions Net.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
IDServeurFTP est un entier = NetDémarreServeur(ServeurFTP)
...
// Interdit les fonctions NetExécute et NetSupprimeFichier sur le serveur
NetOptionServeur(IDServeurFTP, netOptionExécute, Faux)
NetOptionServeur(IDServeurFTP, netOptionSupprimeFichier, Faux)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`NetOptionServeur(<Identifiant connexion serveur> , <Nom de l'action> , <Autorisation>)`
---

**`<Identifiant connexion serveur> : (Entier)`**

Identifiant de la connexion (obtenu par la fonction [NetDémarreServeur](../WDLang3/3056013.md)).

**`<Nom de l'action> : (Constante de type entier)`**

A chaque fonction Net ayant une action sur le serveur RPC/FTP correspond une constante du WLangage permettant ou non d'autoriser l'action correspondante. La limitation des actions réalisables sur le serveur est réalisée grâce aux constantes suivantes :


|   |   |
| --- | --- |
| NetEnvoieBuffer | *netOptionEnvoieBuffer* |
| NetEnvoieFichier | *netOptionEnvoieFichier* |
| NetEnvoieMessage | *netOptionEnvoieMessage* |
| NetExécute | *netOptionExécute* |
| NetListeDisque | *netOptionListeDisque* |
| NetListeRep | *netOptionListeRep* |
| NetMessageBoxDistant | *netOptionMessageBoxDistant* |
| NetRécupèreFichier | *netOptionRécupèreFichier* |
| NetRenommeFichier | *netOptionRenommeFichier* |
| NetSupprimeFichier | *netOptionSupprimeFichier* |



**`<Autorisation> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> : Autorise l'action spécifiée. 

- <u><u><u><u>Faux</u></u></u></u> : N'autorise pas l'action spécifiée.

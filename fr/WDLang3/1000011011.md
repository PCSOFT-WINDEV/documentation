
## RendezVousLit (Fonction)

***En anglais : AppointmentRead***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Lit un rendez-vous présent dans l'agenda d'une messagerie Lotus Notes ou Outlook. 

**Rappel** : Cette fonction ne permet pas de lire un rendez-vous présent dans l'agenda d'une messagerie Outlook Express.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ouverture d'une session Lotus Notes
IDSession est un entier
IDSession = EmailOuvreSessionNotes("MotDePasse", "ServeurMars", "mail/julie.nsf", "C:\Lotus Notes\julie.id")
// Lecture du premier rendez-vous
RendezVousPremier(IDSession)
...
// Re-lecture de ce même rendez-vous
RendezVousLit(IDSession, mRendezVous.ID)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = RendezVousLit(<Identifiant de la session> , <Identifiant du rendez-vous>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le rendez-vous a été lu, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour connaître le libellé de l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*. 




**`<Identifiant de la session> : (Entier)`**

Identifiant de la session à utiliser. Cet identifiant est renvoyé par une des fonctions suivantes : [EmailOuvreSessionNotes](../WDLang3/3032121.md), [EmailOuvreSessionOutlook](../WDLang3/3032128.md), [NotesOuvreBase](../WDLang3/1000005101.md) ou [OutlookOuvreSession](../WDLang3/3032123.md).

**`<Identifiant du rendez-vous> : (Chaîne de caractères)`**

Identifiant du rendez-vous à lire. Cet identifiant est contenu dans la variable mRendezVous.ID.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- La variable mRendezVous.EnDehors permet de savoir s'il y a un rendez-vous courant.

- Sous Outlook, il est possible de spécifier le mode de gestion des contacts/participants lors de l'ouverture de la session (fonction [EmailOuvreSessionOutlook](../WDLang3/3032128.md) ou fonction [OutlookOuvreSession](../WDLang3/3032123.md)).

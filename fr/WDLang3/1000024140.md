


## &lt;Graphe&gt;.SuperposeGraphe (Fonction)

***En anglais : &lt;Chart&gt;.OverlayChart***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet d'afficher deux graphes de type différent dans le même graphe. Les deux graphes sont superposés.

**Remarque** : Le premier graphe doit être un graphe de type Histogramme (Histogramme empilé, groupé, ...), de type Boursier (MinMax, chandelier, ...) ou de type Nuage de points et le second graphe doit être de type Courbe.



![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Les graphes](../WDChamp/3042060.md). 








<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI GRF_MonHistogramme.SuperposeGraphe(GRF_MaCourbe) = Vrai ALORS
	Info("Les deux graphes sont superposés")
FIN
GRF_MonHistogramme.Dessine()
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Champ Graphe destination>.SuperposeGraphe(<Champ Graphe source>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si les graphes ont été superposés, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Champ Graphe destination> : (Nom de champ)`**

Nom du champ Graphe qui affichera les deux graphes en même temps. Ce champ Graphe contient le premier graphe (de type histogramme, boursier ou nuage de points).

**`<Champ Graphe source> : (Chaîne de caractères)`**

Nom du champ Graphe qui doit s'afficher dans le graphe destination. Ce graphe doit être de type Courbe. Il se superposera au &lt;Champ Graphe destination&gt;. Ce graphe est pris en compte en tant que source de données : son titre et ses paramètres ne sont pas pris en compte. Seules les informations concernant la légende sont conservées.
Si ce paramètre correspond à une chaîne vide (""), la liaison entre les deux graphes est supprimée.
Ce paramètre doit correspondre à un graphe différent du &lt;Champ Graphe destination&gt; (dans le cas contraire, les graphes s'écrasent).



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- La superposition des graphes définie avec la fonction **&lt;Graphe&gt;.SuperposeGraphe** sera prise en compte lors du prochain appel à la fonction [&lt;Graphe&gt;.Dessine](../WDLang3/1000024062.md).

- Attention : La fonction **&lt;Graphe&gt;.SuperposeGraphe** ne gère pas la mire des différents graphes superposés.




## HTTPCookieRécupère (Fonction)

***En anglais : HTTPCookieGet***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Récupère les cookies lus : 

- par un appel à la fonction [HTTPRequête](../WDLang3/3043007.md) pour un domaine HTTP.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) par un appel à la fonction [HTTPEnvoie](../WDLang3/1000021183.md) pour un domaine HTTP. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) dans la variable [httpRéponse](../WDLang3/1000021165.md) (contenant le résultat de la fonction [HTTPEnvoie](../WDLang3/1000021183.md)). 


**Remarque** : Les cookies lus et envoyés par la fonction [HTTPRequête](../WDLang3/3043007.md) sont uniquement conservés en mémoire dans l'application. Ils n'ont pas de rapport avec les cookies des navigateurs Web installés sur le poste.






<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Requête vers le site google.fr
HTTPRequete("www.google.fr")

// Lecture du tableau de cookies renvoyé par le site
tabCookie est un tableau de Cookies
tabCookie = HTTPCookieRécupère(".www.google.fr")
Info("Le site www.google.fr stocke actuellement " + tabCookie.Occurrence + " cookies localement.")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Récupérer les cookies lus par un appel à la fonction HTTPRequête

`<Résultat> = HTTPCookieRécupère(<Domaine>)`
---

**`<Résultat> : (Tableau de variables de type Cookie)`**

Liste des cookies stockés en mémoire pour le domaine spécifié.

**`<Domaine> : (Chaîne de caractères)`**

Domaine pour lequel la liste des cookies actuellement en mémoire doit être récupérée.


<a name="SYNTAXE2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 
### Récupérer les cookies lus par un appel à la fonction HTTPEnvoie

`<Résultat> = HTTPCookieRécupère(<Requête HTTP> , <Domaine>)`
---

**`<Résultat> : (Tableau de variables de type Cookie)`**

Liste des cookies stockés en mémoire pour le domaine spécifié.

**`<Requête HTTP> : (Variable de type httpRequête)`**

Nom de la variable [httpRequête](../WDLang3/1000021158.md) pour laquelle les cookies doivent être listés. 

**`<Domaine> : (Chaîne de caractères)`**

Domaine pour lequel la liste des cookies actuellement en mémoire doit être récupérée.


<a name="SYNTAXE3"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 
### Récupérer les cookies lus via une variable de type httpRéponse

`<Résultat> = HTTPCookieRécupère(<Réponse HTTP>)`
---

**`<Résultat> : (Tableau de variables de type Cookie)`**

Liste des cookies stockés en mémoire pour le domaine spécifié.

**`<Réponse HTTP> : (Variable de type httpRéponse)`**

Nom de la variable [httpRéponse](../WDLang3/1000021165.md) pour laquelle les cookies doivent être listés.




## UPNPMédiaJoue (Fonction)

***En anglais : UPNPMediaPlay***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Lance la lecture du média en cours sur le périphérique UPnP.

**Attention** : Pour utiliser cette fonction, le périphérique doit posséder un service "AVTransport". 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
tabPériphérique est un tableau de upnpPériphériques
tabPériphérique = UPNPListePériphérique(...
		"urn:schemas-upnp-org:device:MediaRenderer:1", 5)
POUR TOUT Périphérique de tabPériphérique
	UPNPMédiaJoue(Périphérique)
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = UPNPMédiaJoue(<Périphérique UPnP>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la lecture a été lancée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 




**`<Périphérique UPnP> : (Variable de type upnpPériphérique)`**

Nom de la variable [upnpPériphérique](../WDLang3/1000020903.md) correspondant au périphérique UPnP à manipuler.

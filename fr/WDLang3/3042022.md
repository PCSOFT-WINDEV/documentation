***En anglais : grPieStartAngle***

<a name="Utilisation"></a>
<a name="description"></a>
Indique l'angle de départ de la première part dans un graphe de type Secteur, Hémicycle ou Donut.

Exemple d'angle de départ d'un graphe : ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=grSecteurPartExcentr%E9e%20-%20HC%20N%B0001.gif)




![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Les graphes](../WDChamp/3042060.md). 














<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Augmentation de l'angle de départ du champ Graphe "GRF_MonGraphe"
grSecteurAngleDépart(GRF_MonGraphe, 45)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`grSecteurAngleDépart(<Nom du graphe> , <Angle>)`
---

**`<Nom du graphe> : (Nom de champ ou chaîne de caractères)`**

Nom du graphe à manipuler. Ce nom correspond :

- soit au nom du graphe défini par programmation avec la fonction [grCrée](../WDLang3/3042005.md).

- soit au nom du champ Graphe présent dans l'éditeur de fenêtres, de pages ou d'états.




**`<Angle> : (Entier)`**

Angle de départ de la première part (exprimé en degrés dans le sens des aiguilles d'une montre à partir du point haut du disque). 
Par défaut, l'angle de départ dans un graphe de type Secteur, Hémicycle ou Donut est égal à 0.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- La définition de l'angle de départ de la première part d'un graphe est disponible uniquement sur les graphes suivants : 

	- graphes de type Secteur (constante *grSecteur*). 

	- graphes de type Hémicycle (constante *grHémicycle*).
			![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Ce type de graphe n'est pas disponible.

	- graphes de type Donut (constante *grDonut*).
			![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Ce type de graphe n'est pas disponible.




- Les données définies avec la fonction **grSecteurAngleDépart** seront prises en compte lors du prochain appel à la fonction [grDessine](../WDLang3/3042026.md).

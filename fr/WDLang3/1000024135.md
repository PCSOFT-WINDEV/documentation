


## &lt;Graphe&gt;.SMNPartExcentrée (Fonction)

***En anglais : &lt;Chart&gt;.SunburstPullOut***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Excentre une part ou toutes les parts d'un graphe de type Secteur multiniveau. 

**Remarque** : Chaque part du graphe peut être excentrée indépendamment des autres.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ajout de données
GRF_Graphe.SMNAjouteDonnée("Europe" + TAB + "France" + TAB + "Montpellier", 320000)
GRF_Graphe.SMNAjouteDonnée("Europe" + TAB + "France" + TAB + "Paris", 789000)
GRF_Graphe.SMNAjouteDonnée("Europe" + TAB + "France", 1109000) 
GRF_Graphe.SMNAjouteDonnée("Europe" + TAB + "Belgique", 900000)
GRF_Graphe.SMNAjouteDonnée("Asie", 250000)
GRF_Graphe.Dessine()
// Décalage d'une part du graphe "GRF_Graphe" de 20 pixels
GRF_Graphe.SMNPartExcentrée("Europe" + TAB + "France" + TAB + "Montpellier", 20)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Excentrage d'une part

`<Champ Graphe>.SMNPartExcentrée(<Chemin de la part> , <Décalage>)`
---

**`<Champ Graphe> : (Nom de champ)`**

Nom du champ Graphe à manipuler (présent dans l'éditeur de fenêtres, de pages ou d'états).


**`<Chemin de la part> : (Chaîne de caractères)`**

Chemin complet de la portion à excentrer dans le graphe. Ce paramètre est de la forme :

```txt
"<Nom du niveau de base>" + TAB + ["<Nom du 1er niveau>" + TAB + ...
["<Nom du dernier niveau>" + TAB + [...]]]"<Nom de la portion>"
```


**`<Décalage> : (Entier)`**

Décalage de la part spécifiée (en pixels).


<a name="SYNTAXE2"></a>

### Excentrage de toutes les parts

`<Champ Graphe>.SMNPartExcentrée(<Décalage>)`
---

**`<Champ Graphe> : (Nom de champ)`**

Nom du champ Graphe à manipuler (présent dans l'éditeur de fenêtres, de pages ou d'états).


**`<Décalage> : (Entier)`**

Décalage des différentes parts (en pixels).



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Les modifications effectuées avec la fonction **&lt;Graphe&gt;.SMNPartExcentrée** seront prises en compte lors du prochain appel à la fonction [&lt;Graphe&gt;.Dessine](../WDLang3/1000024062.md).

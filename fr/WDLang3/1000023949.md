


## &lt;Variable xsdEntité&gt;.AjouteAttribut (Fonction)

***En anglais : &lt;xsdEntity variable&gt;.AddAttribute***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ajoute un attribut à une variable automatique générée par l'import d'un Webservice.






<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
xsdJob est un DefinedReportJob
xsdJob.AjouteAttribut("xsi:type", xsdJob..Type)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Variable>.AjouteAttribut(<Nom> , <Valeur>)`
---

**`<Variable> : (Variable générée par l'import d'un Webservice)`**

Variable automatique du Webservice à modifier.

**`<Nom> : (Chaîne de caractères Unicode)`**

Nom de l'attribut à ajouter.

**`<Valeur> : (Chaîne de caractères Unicode)`**

Valeur de l'attribut à ajouter.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Fonctionnement
<a name="fonctionnement_ELTPARAGRAPHE000195"></a>Lorsqu'un Webservice est importé dans un projet, WINDEV, WEBDEV et WINDEV Mobile génèrent automatiquement les types de variables permettant de manipuler le flux XML utilisé par le protocole du Webservice. 

La fonction **&lt;Variable xsdEntité&gt;.AjouteAttribut** permet d'ajouter un attribut qui ne serait pas déclaré dans le WSDL (et donc non généré automatiquement) à une variable de l'un des types importés.

**Remarque** : La fonction **&lt;Variable xsdEntité&gt;.AjouteAttribut** est une fonction avancée. Dans la plupart des cas, le WSDL déclare tous les éléments nécessaires à l'utilisation des Webservices.

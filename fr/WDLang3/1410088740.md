
## iCloudCléValeurExiste (Fonction)

***En anglais : iCloudKeyValueExist***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Vérifie l'existence d'une clé dans un conteneur de paires clé-valeur basé sur Apple iCloud.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI iCloudCléValeurEcrit("ArticleParPage") = Faux ALORS
	iCloudCléValeurEcrit("ArticleParPage", 10)
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = iCloudCléValeurExiste(<Clé>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la clé existe,

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 




**`<Clé> : (Chaîne de caractères)`**

Nom de la clé manipulée.




## &lt;Carte&gt;.AjouteMarqueur (Fonction)

***En anglais : &lt;Map&gt;.AddMarker***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ajoute un nouveau marqueur sur une carte affichée dans un champ Carte.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affiche un marqueur : lors du clic sur ce marqueur, une procédure est exécutée
MaPosition est une géoPosition
MaPosition.Latitude = 43.613708 
MaPosition.Longitude = 3.876972

MonMarqueur est un Marqueur
MonMarqueur.Position = MaPosition
MonMarqueur.ActionClic = ProcClicMarqueur

CARTE_Position.AjouteMarqueur(MonMarqueur)
```

```wl
PROCEDURE ProcClicMarqueur(MaMarque est un marqueur)

ToastAffiche(ChaîneConstruit("Position : %1 N, %2 E ", MaMarque.Position.Latitude, ...
		MaMarque.Position.Longitude))
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Ajoute un nouveau marqueur à partir d'une variable de type Marqueur

`<Résultat> = <Champ Carte>.AjouteMarqueur(<Marqueur>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le marqueur a été ajouté, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Champ Carte> : (Nom de champ)`**

Nom du champ Carte à manipuler.

**`<Marqueur> : (Variable de type Marqueur)`**

Nom de la variable de type [Marqueur](../WDLang3/1000019940.md) décrivant le marqueur à afficher.

Attention : Si la propriété **Position** du marqueur n'a pas été renseignée, une erreur fatale sera déclenchée.


<a name="SYNTAXE2"></a>

### Ajoute un nouveau marqueur à partir d'une variable de type géoPosition

`<Résultat> = <Champ Carte>.AjouteMarqueur(<Position>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le marqueur a été ajouté, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.  Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Champ Carte> : (Nom de champ)`**

Nom du champ Carte à manipuler.

**`<Position> : (Variable de type géoPosition)`**

Nom de la variable de type [géoPosition](../WDLang3/1000019191.md) décrivant la position du marqueur à afficher.


<a name="SYNTAXE3"></a>

### Ajoute un nouveau marqueur à partir d'une latitude et d'une longitude

`<Résultat> = <Champ Carte>.AjouteMarqueur(<Latitude> , <Longitude>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le marqueur a été ajouté, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.  Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Champ Carte> : (Nom de champ)`**

Nom du champ Carte à manipuler.

**`<Latitude> : (Réel)`**

Latitude du marqueur à ajouter sur la carte.

**`<Longitude> : (Réel)`**

Longitude du marqueur à ajouter sur la carte.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Pour supprimer un marqueur affiché dans un champ Carte, utilisez la fonction [&lt;Carte&gt;.SupprimeMarqueur](../WDLang3/1000022466.md).

- Pour modifier un marqueur affiché dans un champ Carte, utilisez la fonction [&lt;Carte&gt;.ModifieMarqueur](../WDLang3/1000022383.md).

- Pour supprimer tous les marqueurs d'un champ Carte, utilisez la fonction [&lt;Carte&gt;.SupprimeTout](../WDLang3/1000022472.md).

- Pour ajouter plusieurs marqueurs dans un champ Carte, il est nécessaire que la propriété **Nom** de la variable [Marqueur](../WDLang3/1000019940.md) soit unique pour chaque marqueur (même si les positions GPS sont différentes).
	Exemple :
	
	```wl
	Pos est un géoPosition
	POI est un Marqueur
	
	POUR TOUT PRODUIT
		Pos.Latitude = PRODUIT.Latitude
		Pos.Longitude = PRODUIT.Longitude
	
		POI..Position = Pos
		// POI.Nom doit être différent à chaque itération dans la boucle.
		POI.Nom = "Produit " + PRODUIT.Reference
		CARTE_MaCarte.AjouteMarqueur(POI)
	FIN
	```


- Il est possible de gérer la position du marqueur grâce à la propriété **Alignement** de la variable [Marqueur](../WDLang3/1000019940.md).

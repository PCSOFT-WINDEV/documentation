


## Fonctions Bluetooth
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions permettant de gérer les appareils Bluetooth sont les suivantes :



|   |   |
| --- | --- |
| [BTAccepteConnexion](../WDLang3/1000017128.md) | Permet de savoir si une radio Bluetooth accepte ou non les demandes de connexion en provenance des périphériques de son entourage. |
| [BTActive](../WDLang3/1000019359.md) | Active ou désactive la gestion du Bluetooth sur l'appareil. |
| [BTActiveService](../WDLang3/1000017142.md) | Active un service offert par un périphérique Bluetooth. |
| [BTChangeConnectivité](../WDLang3/1000017130.md) | Permet de paramétrer une radio Bluetooth pour accepter ou non les demandes de connexion provenant des périphériques de son entourage. |
| [BTChangeVisibilité](../WDLang3/1000017129.md) | Change la visibilité d'une radio Bluetooth. |
| [BTConnectePériphérique](../WDLang3/1000017146.md) | Permet de se faire authentifier auprès d'un périphérique Bluetooth. |
| [BTDéconnectePériphérique](../WDLang3/1000017147.md) | Annule l'authentification auprès d'un périphérique Bluetooth. |
| [BTDésactiveService](../WDLang3/1000017143.md) | Désactive un service sur un périphérique Bluetooth. |
| [BTEstVisible](../WDLang3/1000017127.md) | Permet de savoir si une radio Bluetooth est visible. |
| [BTEtat](../WDLang3/1000019360.md) | Renvoie l'état actuel d'activation du Bluetooth sur l'appareil ou demande à être notifié des changements d'état d'activation. |
| [BTInfoPériphérique](../WDLang3/1000017133.md) | Renvoie des informations spécifiques sur un périphérique Bluetooth. |
| [BTInfoRadio](../WDLang3/1000017126.md) | Renvoie des informations spécifiques sur une radio Bluetooth connectée à la machine. |
| [BTListePériphérique](../WDLang3/1000017132.md) | Liste les périphériques Bluetooth accessibles. |
| [BTListeRadio](../WDLang3/1000017125.md) | Liste les radios Bluetooth connectées à la machine. |
| [BTListeService](../WDLang3/1000017141.md) | Liste les services offerts par un périphérique Bluetooth. |
| [BTOuvrePropriétésPériphérique](../WDLang3/1000017134.md) | Ouvre la fenêtre des propriétés d'un périphérique Bluetooth. |
| [BTPile](../WDLang3/1000017229.md) | Renvoie la pile Bluetooth en cours d'utilisation. |
| [BTSélecteurPériphérique](../WDLang3/1000017135.md) | Ouvre une fenêtre système pour sélectionner un périphérique Bluetooth. |





**Remarques** : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Seules la pile bluetooth Microsoft et la pile Bluesoleil sont supportées.

- Il n'est pas nécessaire de signer l'application pour utiliser les fonctions Bluetooth.




Il est également possible de gérer les Sockets Bluetooth grâce aux fonctions suivantes : 



|   |   |
| --- | --- |
| [SocketConnecteBluetooth](../WDLang3/1000017145.md) | Connecte un poste client à un socket Bluetooth spécifique. |
| [SocketCréeBluetooth](../WDLang3/1000017167.md) | Créé un socket en utilisant un périphérique Bluetooth. |

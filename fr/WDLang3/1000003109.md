
## Constantes Domo
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


| Constante | Description | Utilisée par les fonctions |
| --- | --- | --- |
| **domoArrête** | Arrête les appareils électriques spécifiés | [DomoEnvoie](../WDLang3/1000003104.md) |
| **domoArrêteTout** | Arrête tous les appareils électriques spécifiés | [DomoEnvoie](../WDLang3/1000003104.md) |
| **domoAugmente** | Augmente l'intensité des appareils électriques spécifiés (intensité d'éclairage des lampes généralement). | [DomoEnvoie](../WDLang3/1000003104.md) |
| **domoDiminue** | Diminue l'intensité des appareils électriques spécifiés (intensité d'éclairage des lampes généralement). | [DomoEnvoie](../WDLang3/1000003104.md) |
| **domoMarche** | Allume les appareils électriques spécifiés. | [DomoEnvoie](../WDLang3/1000003104.md) |
| **domoModifie** | Modifie l'état en cours des appareils électriques spécifiés. | [DomoEnvoie](../WDLang3/1000003104.md) |

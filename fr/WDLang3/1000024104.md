


## &lt;Graphe&gt;.Nuage3DSupprimeLibelléGraduation (Fonction)

***En anglais : &lt;Chart&gt;.Scatter3DDeleteMarkCaption***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime les libellés de graduation d'un graphe de type Nuage de points 3D.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
GRF_MonGraphe.Nuage3DSupprimeLibelléGraduation(grAbscisse, 1)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Champ Graphe>.Nuage3DSupprimeLibelléGraduation([<Axe de la graduation> [, <Valeur>]])`
---

**`<Champ Graphe> : (Nom de champ)`**

Nom du champ Graphe à manipuler (présent dans l'éditeur de fenêtres).

**`<Axe de la graduation> : (Constante de type Entier)`**

Type de l'axe sur lequel la graduation doit être supprimée : 



|   |   |
| --- | --- |
| *grAbscisse* | Axe des abscisses. |
| *grAltitude* | Axe des altitudes. |
| *grOrdonnée* | Axe des ordonnées. |


Si ce paramètre n'est pas précisé, les graduations sont supprimées sur tous les axes.

**`<Valeur> : (Réel)`**

Valeur de la graduation pour laquelle le libellé doit être supprimé. 

Si ce paramètre n'est pas précisé, tous les libellés des graduations seront supprimés.

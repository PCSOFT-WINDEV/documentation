


## grHMAjouteDonnée (Fonction)

***En anglais : grHMAddData***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ajoute une donnée dans un graphe de type Heatmap.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
grHMAjouteDonnée(GRF_Graphe1, 1, 1, 10)
grHMAjouteDonnée(GRF_Graphe1, 1, 2, 30)
grHMAjouteDonnée(GRF_Graphe1, 2, 1, 20.5)
grHMAjouteDonnée(GRF_Graphe1, 2, 2, 5)
```

```wl
// Définition des couleurs
grHMCouleur(GRF_HMap, -50, BleuFoncé)
grHMCouleur(GRF_HMap, +50, RougeClair)
POUR i = 1 A 500
	grHMAjouteDonnée(GRF_HMap, Hasard(-100, 100), Hasard(-100, 100), -100)
	grHMAjouteDonnée(GRF_HMap, Hasard(-100, 100), Hasard(-100, 100), 100)
FIN
grDessine(GRF_HMap)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`grHMAjouteDonnée(<Nom du graphe> , <X> , <Y> , <Valeur>)`
---

**`<Nom du graphe> : (Nom de champ ou chaîne de caractères)`**

Nom du graphe à manipuler. Ce nom correspond :

- soit au nom du graphe défini par programmation avec la fonction [grCrée](../WDLang3/3042005.md).

- soit au nom du champ Graphe présent dans l'éditeur de fenêtres ou d'états.




**`<X> : (Entier)`**

Position de la donnée sur l'axe des abscisses.

**`<Y> : (Entier)`**

Position de la donnée sur l'axe des ordonnées.

**`<Valeur> : (Entier ou réel)`**

Valeur à ajouter. Si une valeur se trouve déjà à la position (X, Y) indiquée, la valeur existante est remplacée par la nouvelle valeur. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Les données définies avec la fonction **grHMAjouteDonnée** seront prises en compte lors du prochain appel à la fonction [grDessine](../WDLang3/3042026.md).

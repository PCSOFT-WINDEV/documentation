


## &lt;Graphe&gt;.HistoAgrégation (Fonction)

***En anglais : &lt;Chart&gt;.ColumnAggregation***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Définit l'algorithme d'agrégation des données d'un graphe de type Histogramme (groupé ou empilé).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Code du champ Sélecteur permettant de sélectionner le type d'agrégation
// Mémorise le type d'agrégation sélectionné
soit nAgrégation = SELON (SEL_TypeAgrégation, grhaAucun, grhaMax, grhaMin, grhaMoyenne, grhaSomme)

// Applique l'agrégation
GRF_Graphe.HistoAgrégation(nAgrégation)

// Redessine l'exemple
GRF_Graphe.Dessine()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Champ Graphe>.HistoAgrégation(<Algorithme>)`
---

**`<Champ Graphe> : (Nom de champ)`**

Nom du champ Graphe à manipuler (présent dans l'éditeur de fenêtres, de pages ou d'états).

**`<Algorithme> : (Constante de type Entier)`**

Algorithme d'agrégation à utiliser : 


|   |   |
| --- | --- |
| *grhaAucun* | Aucune agrégation. |
| *grhaMax* | Agrégation selon le maximum des données. |
| *grhaMin* | Agrégation selon le minimum des données. |
| *grhaMoyenne* | Agrégation selon la moyenne des données. |
| *grhaSomme* | Agrégation selon la somme des données. |

<a name="NOTE0_1"></a>

## Remarques


### Quand utiliser l'agrégation ?
<a name="quand_utiliser_agregation_ELTPARAGRAPHE000203"></a>Lors de l'affichage d'un graphe de type Histogramme, il peut arriver que les barres deviennent trop petites pour être affichées. C'est notamment le cas pour les graphes présentant une échelle temporelle. Dans ce cas, il devient nécessaire d'afficher une barre faisant (au minimum) la taille d'une graduation et agrégeant les données de toutes les barres potentielles présentes dans cette graduation.

### Agrégation et type de graphes
<a name="agregation_type_graphes_ELTPARAGRAPHE000228"></a>

- Pour les graphes sans échelle temporelle, l'agrégation n'est pas activée par défaut.

- Pour les graphes avec échelle temporelle, l'agrégation par défaut est la somme des barres.

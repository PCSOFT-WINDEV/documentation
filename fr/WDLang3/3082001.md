| Constante | Détail | Utilisée par les fonctions |
| --- | --- | --- |
| **zipAdministrateur** | Force l'exécutable créé à se lancer en tant qu'administrateur. Ce mode permet d'éviter la virtualisation des fichiers à partir de Windows Vista. | [zipCréeExe](../WDLang3/3082013.md) |
| **zipAnnule** | Annule l'extraction des fichiers de l'archive. | [zipExtraitListeFichier](../WDLang3/1000017234.md), [zipExtraitTout](../WDLang3/3082011.md) |
| **zipAnsi** | Permet de forcer la sauvegarde des noms de fichier au format Ansi dans le fichier Archive.<br><br>Cette constante est utilisée par défaut dans une application utilisant le mode Ansi. | [zipCrée](../WDLang3/3082003.md) |
| **zipArborescence** | Les fichiers de l'archive seront extraits en respectant l'arborescence des répertoires contenus dans l'archive. | [zipCréeExe](../WDLang3/3082013.md) |
| **zipAucun** | Mémorise uniquement le nom et l'extension du fichier. Par exemple : NomFichier.pdf<br>Extraction dans le répertoire en cours sans restauration de l'arborescence du fichier (si elle a été mémorisée). | [zipAjouteFichier](../WDLang3/3082008.md)<br>[zipExtraitFichier](../WDLang3/3082019.md) |
| **zipAvecUI** | L'auto-extraction affichera une boîte de dialogue permettant à l'utilisateur de choisir le répertoire de destination.<br><br>Remarque : Dans les versions précédentes, cette constante avait pour nom **zipAvecIHM**. | [zipCréeExe](../WDLang3/3082013.md) |
| **zipChoixArborescence** | L'auto-extraction affichera dans la boîte de dialogue l'option "Respecter l'arborescence". | [zipCréeExe](../WDLang3/3082013.md) |
| **zipCréationSiInexistant** | Création de l'archive si l'archive est inexistante. | [zipOuvre](../WDLang3/3082044.md) |
| **zipDisque** | Mémorise le chemin complet et absolu du fichier (nom du disque, des répertoires, du fichier et l'extension du fichier). Par exemple : C:\\Répertoire\\Fichiers\\NomFichier.pdf <br>Cette constante n'a pas d'effet pour le format ZIP.<br>Renvoie uniquement le nom de l'unité. | [zipAjouteFichier](../WDLang3/3082008.md)<br>[zipAjouteRépertoire](../WDLang3/3082014.md)<br>[zipExtraitChemin](../WDLang3/3082023.md)<br>[zipExtraitFichier](../WDLang3/3082019.md)<br>[zipExtraitTout](../WDLang3/3082011.md) |
| **zipEcrase** | Lors de l'extraction, les fichiers existants en lecture seulement sont automatiquement écrasés par les fichiers extraits. | [zipExtraitTout](../WDLang3/3082011.md) |
| **zipEcriture** | Ouverture en lecture/écriture. | [zipOuvre](../WDLang3/3082044.md) |
| **zipEnMémoire** | Permet de gérer les archives en mémoire (ajout d'un fichier présent dans un buffer ou extraction d'un fichier dans un buffer). | [zipAjouteFichier](../WDLang3/3082008.md), [zipExtraitFichier](../WDLang3/3082019.md) |
| **zipEntête64** | Permet de créer une archive de plus de 4 gigas sur laquelle la fonction [zipChangeChemin](../WDLang3/3082027.md) pourra être utilisée. | [zipCrée](../WDLang3/3082003.md), <br>[zipOuvre](../WDLang3/3082044.md), <br>[zipOuvreCAB](../WDLang3/1000017250.md), <br>[zipOuvreRAR](../WDLang3/1000017294.md) |
| **zipExtension** | Renvoie uniquement l'extension du fichier (précédée du "."). | [zipExtraitChemin](../WDLang3/3082023.md) |
| **zipFichier** | Renvoie uniquement le nom du fichier (sans son extension). | [zipExtraitChemin](../WDLang3/3082023.md) |
| **zipFormat7Z** | Fichier d'archive au format 7z. | Variable de type [zipArchive](../WDLang3/1000018679.md) |
| **zipFormatAucun** | Aucun fichier d'archive n'est créé. | Variable de type [hDescriptionSauvegarde](../WDLang4/1000017455.md) |
| **zipFormatCAB** | Fichier d'archive au format CAB. | Variable de type [zipArchive](../WDLang3/1000018679.md) |
| **zipFormatRAR** | Fichier d'archive au format RAR. | Variable de type [zipArchive](../WDLang3/1000018679.md) |
| **zipFormatTAR** | Fichier d'archive au format TAR (format non compressé). | Variable de type [zipArchive](../WDLang3/1000018679.md) |
| **zipFormatTARGZ** | Fichier d'archive au format TARGZ. | Variable de type [zipArchive](../WDLang3/1000018679.md) |
| **zipFormatWDZ** | Fichier d'archive au format WDZ. | Variable de type [zipArchive](../WDLang3/1000018679.md) |
| **zipFormatZIP** | Fichier d'archive au format ZIP. | Variable de type [zipArchive](../WDLang3/1000018679.md)<br>Variable de type [hDescriptionSauvegarde](../WDLang4/1000017455.md) |
| **zipIgnore** | Ignore l'extraction du fichier en erreur. | [zipExtraitListeFichier](../WDLang3/1000017234.md), [zipExtraitTout](../WDLang3/3082011.md) |
| **zipLecture** | Ouverture en lecture seule | [zipOuvre](../WDLang3/3082044.md) |
| **zipNonCompressé** | Le contenu de l'archive ne sera pas compressé.<br>Ce paramètre est utilisable avec les archives de type CAB, pour utiliser uniquement la fonction d'archivage de ce format. | [zipCrée](../WDLang3/3082003.md) |
| **zipRecommence** | Retente l'extraction du fichier. | [zipExtraitListeFichier](../WDLang3/1000017234.md), [zipExtraitTout](../WDLang3/3082011.md) |
| **zipRelatif** | Le chemin passé en paramètre est considéré comme la racine de l'archivage. Le chemin mémorisé pour tous les fichiers et les sous-répertoires est relatif à cette racine. | [zipAjouteRépertoire](../WDLang3/3082014.md) |
| **zipRepCourant** | Tous les fichiers de l'archive seront extraits dans le répertoire en cours. | [zipCréeExe](../WDLang3/3082013.md) |
| **zipRepDéfaut** | Utilise le répertoire spécifié par défaut (paramètre &lt;Répertoire d'extraction par défaut&gt;). | [zipCréeExe](../WDLang3/3082013.md) |
| **zipRépertoire** | Mémorise les différents répertoires composant le chemin, le nom et l'extension du fichier. Par exemple : \\Répertoire\\Fichiers\\NomFichier.pdf<br>Renvoie les différents répertoires du fichier (le nom et l'extension du fichier ne sont pas renvoyés) | [zipAjouteFichier](../WDLang3/3082008.md)<br>[zipAjouteRépertoire](../WDLang3/3082014.md)<br>[zipExtraitChemin](../WDLang3/3082023.md)<br>[zipExtraitFichier](../WDLang3/3082019.md)<br>[zipExtraitTout](../WDLang3/3082011.md) |
| **zipRepTemporaire** | Les fichiers de l'archive seront extraits dans le répertoire temporaire de la machine sur laquelle l'exécutable est lancé. Tous les fichiers sont extraits au même endroit. | [zipCréeExe](../WDLang3/3082013.md) |
| **zipUnicode** | Permet de forcer la sauvegarde des noms de fichier au format Unicode dans le fichier Archive.<br><br>Cette constante est utilisée par défaut dans une application utilisant le mode Unicode. | [zipCrée](../WDLang3/3082003.md) |

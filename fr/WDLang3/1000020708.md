
## Gestion des agendas (Android/iOS)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000121"></a>
WINDEV Mobile permet d'accéder simplement aux agendas et aux rendez-vous des téléphones et tablettes sous Android et iOS. Il est par exemple possible de : 

- Lister les agendas existants.

- Lister les rendez-vous.

- Ajouter un nouveau rendez-vous à un agenda.

- Modifier les données d'un rendez-vous.

- Supprimer un rendez-vous.

- Supprimer tous les rendez-vous d'un agenda.

- Définir la répétition d'un rendez-vous (tous les jours, tous les mois, etc.).

- Ajouter/Modifier/Supprimer les rappels pour un rendez-vous donné (exemple : envoi d'un SMS une heure avant le rendez-vous).

- Lancer l'application native de gestion des agendas pour créer, éditer ou visualiser un rendez-vous.

- Plusieurs types de données : 
	


|   |   |
| --- | --- |
| [Calendrier](../WDLang3/1000020663.md) | Le type **Calendrier** permet de manipuler un calendrier (ou agenda) associé à un compte utilisateur ou bien se trouvant sur l'appareil. |
| [RendezVous](../WDLang1/1000019244.md) | Le type **RendezVous** permet de définir toutes les caractéristiques avancées d'un rendez-vous : horaires, lieu, ... |

- Diverses fonctions : 

	- Fonctions pour gérer l'ajout, la suppression et la modification des rendez-vous : 
			


|   |   |
| --- | --- |

| [RendezVousModifie](../WDLang3/1000011003.md) | Modifie le rendez-vous en cours : <br><br>		- dans l'agenda d'une messagerie Lotus Notes ou Outlook.<br><br>		- dans un calendrier (ou agenda) d'un appareil mobile (Android/iOS). <br><br><br> |
| [RendezVousSupprime](../WDLang3/1000011008.md) | Supprime : <br><br>		- le rendez-vous en cours de l'agenda d'une messagerie Lotus Notes ou Outlook. <br><br>		- un rendez-vous dans un calendrier (ou agenda) d'un appareil mobile (Android/iOS).<br><br><br> |

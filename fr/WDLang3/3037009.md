
## FaxBoîteRéception (Fonction)

***En anglais : FaxInbox***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Énumère les fax en attente ou en cours de réception.

**Remarque** : Cette fonction peut être utilisée uniquement sous Windows 2000 ou XP. A partir de Windows Vista, la gestion des fax est disponible uniquement avec Windows Vista Professionnel et Windows Vista Edition Intégrale.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
IDConnect est un entier
IDConnect = FaxConnecte()
...
ResEnvoiFax est un booléen
ResEnvoiFax = FaxEnvoie(IDConnect, "MonFax", "Message.txt", "0102030405")
...
ListeRéception est une chaîne 
ListeRéception = FaxBoîteRéception(IDConnect)
...
FaxDéconnecte(IDConnect)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = FaxBoîteRéception(<Identifiant connexion>)`
---

**`<Résultat> : (Chaîne de caractères)`**

Liste des identifiants numériques de fax séparés par des RC (Retour Chariot). Cet identifiant permet de manipuler le fax grâce aux fonctions de gestion des fax. Les identifiants de fax correspondent à tous les fax reçus.

**`<Identifiant connexion> : (Entier)`**

Identifiant de la connexion obtenu avec la fonction [FaxConnecte](../WDLang3/3037005.md).

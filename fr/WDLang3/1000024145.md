


## &lt;Graphe&gt;.SurfaceLibelléGraduation (Fonction)

***En anglais : &lt;Chart&gt;.SurfaceMarkCaption***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de spécifier le libellé d'une graduation pour un graphe de type Surface. 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
GRF_MonGraphe.SurfaceLibelléGraduation(grAbscisse, 1, "Janvier")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Champ Graphe>.SurfaceLibelléGraduation(<Axe de la graduation> , <Valeur> , <Libellé>)`
---

**`<Champ Graphe> : (Nom de champ)`**

Nom du champ Graphe à manipuler (présent dans l'éditeur de fenêtres ou d'états).

**`<Axe de la graduation> : (Constante de type Entier)`**

Type de l'axe sur lequel la graduation doit être affichée :


|   |   |
| --- | --- |
| *grAbscisse* | Axe des abscisses. |
| *grAltitude* | Axe des altitudes. |
| *grOrdonnée* | Axe des ordonnées. |



**`<Valeur> : (Réel)`**

Valeur de la graduation pour laquelle le libellé doit être modifié.

**`<Libellé> : (Chaîne de caractères)`**

Nouveau libellé de la graduation.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Gestion des graduations
<a name="gestion_des_graduations_ELTPARAGRAPHE000209"></a>Pour que le libellé de la graduation soit affiché, il est nécessaire : 

- qu'une graduation existe à la valeur indiquée. Dans le cas contraire, le libellé de la graduation ne sera pas affiché.

- que la graduation ait une valeur. S'il existe une graduation sans valeur, le libellé ne sera pas affiché.




**Rappel** : Les graduations d'un graphe de type Surface peuvent être définies : 

- sous l'éditeur, dans la fenêtre de description du champ Graphe (onglet "Général"). 

- par programmation, grâce à la fonction [&lt;Graphe&gt;.Graduation](../WDLang3/1000024076.md).

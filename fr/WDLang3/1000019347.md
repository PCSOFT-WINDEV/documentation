
## OutlookProfilDéfaut (Fonction)

***En anglais : OutlookDefaultProfile***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Récupère le profil défini par défaut dans Outlook.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Récupération du profil par défaut
ProfilDéfaut est une chaîne 
ProfilDéfaut = OutlookProfilDéfaut()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = OutlookProfilDéfaut()`
---

**`<Résultat> : (Chaîne de caractères)`**



- Profil par défaut, 

- Chaîne vide ("") si aucun profil par défaut n'est défini.




**Remarque** : Si le profil par défaut a été défini mais n'est pas actif, il est tout de même renvoyé.

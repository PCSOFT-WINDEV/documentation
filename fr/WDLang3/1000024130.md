


## &lt;Graphe&gt;.SecteurPartExcentrée (Fonction)

***En anglais : &lt;Chart&gt;.PiePullOut***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Excentre une part ou toutes les parts d'un graphe. Cette fonction est utilisable sur les graphes de type Secteur, les graphes de type Beignet (donut) et les graphes de type Hémicycle.

Exemple de part excentrée dans un graphe : <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=grSecteurPartExcentr%E9e%20-%20HC%20N%B0002.gif)


**Remarque** : Chaque part du graphe peut être excentrée indépendamment des autres.



![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Les graphes](../WDChamp/3042060.md). 












<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Décalage de la troisième part du graphe "GRF_MonGraphe" de 20 pixels
GRF_MonGraphe.SecteurPartExcentrée(3, 20)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Excentrage d'une part

`<Champ Graphe>.SecteurPartExcentrée(<Indice de la part> , <Décalage>)`
---

**`<Champ Graphe> : (Nom de champ)`**

Nom du champ Graphe à manipuler (présent dans l'éditeur de fenêtres, de pages ou d'états).


**`<Indice de la part> : (Entier)`**

Indice de la part à excentrer.

**`<Décalage> : (Entier)`**

Décalage de la part spécifiée (en pixels).


<a name="SYNTAXE2"></a>

### Excentrage de toutes les parts

`<Champ Graphe>.SecteurPartExcentrée(<Décalage>)`
---

**`<Champ Graphe> : (Nom de champ)`**

Nom du champ Graphe à manipuler (présent dans l'éditeur de fenêtres, de pages ou d'états).


**`<Décalage> : (Entier)`**

Décalage des différentes parts (en pixels).



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Le décalage d'une part d'un graphe est sans effet sur les graphes autres que les graphes de type Secteur, les graphes de type Beignet (donut) et les graphes de type Hémicycle.
	![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Les graphes de type Hémicycle ne sont pas disponibles.

- Les modifications effectuées avec la fonction **&lt;Graphe&gt;.SecteurPartExcentrée** seront prises en compte lors du prochain appel à la fonction [&lt;Graphe&gt;.Dessine](../WDLang3/1000024062.md).

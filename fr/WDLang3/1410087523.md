
## Constantes Permission
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les constantes de gestion des permissions sont les suivantes :

| Constante | Détail | Fonction |
| --- | --- | --- |
| **permCaméra** | Permission d'accéder aux caméras de l'appareil. | [PermissionDemande](../WDLang3/1410087423.md), <br>[PermissionListe](../WDLang3/1410087429.md) |
| **permEcritureContact** | Permission de modification des contacts. | [PermissionDemande](../WDLang3/1410087423.md), <br>[PermissionListe](../WDLang3/1410087429.md) |
| **permEcritureStockageExterne** | Permission d'écriture sur le stockage externe. | [PermissionDemande](../WDLang3/1410087423.md), <br>[PermissionListe](../WDLang3/1410087429.md) |
| **permEnregistrementAudio** | Permission d'enregistrement du flux audio. | [PermissionDemande](../WDLang3/1410087423.md), <br>[PermissionListe](../WDLang3/1410087429.md) |
| **permEnvoiSMS** | Permission d'envoi de SMS. | [PermissionDemande](../WDLang3/1410087423.md), <br>[PermissionListe](../WDLang3/1410087429.md) |
| **permGestionStockageExterne** | Permission de gestion du stockage externe. | [PermissionDemande](../WDLang3/1410087423.md), <br>[PermissionListe](../WDLang3/1410087429.md)<br><br> |
| **permLectureContact** | Permission d'accès en lecture aux contacts. | [PermissionDemande](../WDLang3/1410087423.md), <br>[PermissionListe](../WDLang3/1410087429.md) |
| **permLectureEtatTéléphone** | Permission d'accès aux informations du téléphone. | [PermissionDemande](../WDLang3/1410087423.md), <br>[PermissionListe](../WDLang3/1410087429.md) |
| **permLocalisation** | Permission d'accès à la position de l'appareil. | [PermissionDemande](../WDLang3/1410087423.md), <br>[PermissionListe](../WDLang3/1410087429.md) |
| **permLocalisationEnArrièrePlan** | Permission d'accès à la position de l'appareil lorsque l'application est en arrière-plan. | [PermissionDemande](../WDLang3/1410087423.md), <br>[PermissionListe](../WDLang3/1410087429.md) |
| **permLocalisationPrécise** | Permission d'accès à la position de l'appareil avec une précision élevée. | [PermissionDemande](../WDLang3/1410087423.md), <br>[PermissionListe](../WDLang3/1410087429.md) |

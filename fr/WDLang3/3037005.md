
## FaxConnecte (Fonction)

***En anglais : FaxConnect***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ouvre une connexion à un serveur de fax.

**Remarque** : Cette fonction peut être utilisée uniquement sous Windows 2000, XP ou version supérieure ou sur un serveur possédant Windows 2000, XP ou version supérieure. Sous Windows Vista, la gestion des fax est disponible uniquement avec Windows Vista Professionnel et Windows Vista Edition Intégrale.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
IDConnect est un entier
IDConnect = FaxConnecte()
SI IDConnect = 0 ALORS 
	Erreur("La connexion au serveur de fax a échoué." + ...
		"Vérifiez que celui-ci est bien démarré", ...
		ErreurInfo(errMessageSystème))
FIN
...
FaxDéconnecte(IDConnect)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = FaxConnecte([<Nom du serveur de fax>])`
---

**`<Résultat> : (Entier)`**



- Identifiant de la connexion, 

- 0 en cas d'erreur. Le message d'erreur correspondant peut être obtenu avec la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Nom du serveur de fax> : (Chaîne de caractères optionnelle)`**

Nom du poste serveur de fax. Si ce paramètre n'est pas précisé, le serveur de fax correspond au poste en cours. 
Attention : il est nécessaire de démarrer le serveur de fax et de le configurer en mode d'envoi. Pour plus de détails, consultez [Envoyer des fax](../WDLang3/3037002.md).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Utiliser un serveur de fax installé sur un poste différent du poste en cours
<a name="utiliser_serveur_fax_installe_sur_poste_different_poste_cours_ELTPARAGRAPHE000046"></a>Il est possible d'utiliser un serveur de fax installé sur un poste différent du poste en cours. Dans ce cas, le poste sur lequel est installé le serveur de fax doit être un poste avec Windows Server 2003. Le serveur de fax doit être partagé. Il suffit alors d'utiliser le nom du poste serveur de fax dans la fonction **FaxConnecte**.

Le poste à partir duquel les fax sont envoyés doit posséder un serveur de fax local.

Par exemple, si le serveur de fax est installé sur le poste "ServeurFax1", il suffit d'utiliser le code :


```wl
IDConnect est un entier
IDConnect = FaxConnecte("ServeurFax1")
```

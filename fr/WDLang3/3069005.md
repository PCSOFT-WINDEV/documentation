


## SOAPErreur (Fonction)

***En anglais : SOAPError***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie l'erreur de la dernière procédure SOAP ayant échoué.

**Remarque** : Cette procédure a été exécutée par la fonction [SOAPExécute](../WDLang3/3069013.md) ou [SOAPExécuteXML](../WDLang3/3069014.md).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple
<a class="notetitle" target="_blank" href="$DOC$=1000003069005&name=soaperreur_fonction&product=WD">Voir des exemples supplémentaires</a>

```wl
// Renvoie le message d'erreur de la dernière procédure SOAP exécutée
ResMessageErreur = SOAPErreur(SOAPerrMessage)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = SOAPErreur(<Type d'erreur>)`
---

**`<Résultat> : (Chaîne de caractères)`**



- Erreur de la dernière procédure SOAP exécutée ayant échoué.

- Une chaîne vide ("") si :

	- aucune erreur n'a été retournée par le serveur SOAP,

	- aucune connexion n'a été établie avec le serveur SOAP.







**`<Type d'erreur> : (Constante)`**

Type d'erreur à retourner :


|   |   |
| --- | --- |
| *SOAPErrActeur* | URL du serveur SOAP ayant provoqué l'erreur (utile si la procédure est exécutée par plusieurs serveurs SOAP). |
| *SOAPErrCode* | Erreur de la dernière procédure SOAP exécutée :<br><br>- "VersionMismatch" : La version SOAP utilisée sur le poste en cours est différente de celle utilisée sur le serveur SOAP.<br><br>- "MustUnderstand" : Le serveur SOAP ne comprend pas le paramètre spécifié dans l'entête (uniquement si le paramètre &lt;Exécution de la procédure&gt; de l'entête est initialisé à <u><u><u><u>Vrai</u></u></u></u>).<br><br>- "Client" : Mauvais paramétrage au niveau de l'exécution de la procédure.<br><br>- "Server" : Le serveur SOAP n'a pas exécuté la procédure.<br><br><br>Remarque : Cette chaîne de caractères peut être complétée d'informations supplémentaires. Le format utilisé est "&lt;Erreur&gt;.&lt;Information&gt;". Par exemple : "Client.Authentication" si l'erreur est due à l'authentification du client. |
| *SOAPErrDétail* | Détail de l'erreur. Ce message est au format XML. |
| *SOAPErrMessage* | Message d'erreur renvoyé par le serveur SOAP sur la dernière procédure SOAP exécutée. |

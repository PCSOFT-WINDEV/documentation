


## BeaconArrêteDétectionEnArrièrePlan (Fonction)

***En anglais : BeaconStopBackgroundDetection***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Arrête une ou plusieurs détections de balises Beacon en arrière-plan. Ces détections ont été lancées avec la fonction [BeaconDétecteEnArrièrePlan](../WDLang3/1000023112.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Suppression de toutes les détections en arrière-plan
BeaconArrêteDétectionEnArrièrePlan()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Supprimer toutes les détections en arrière-plan

`<Résultat> = BeaconArrêteDétectionEnArrièrePlan()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si toutes les détections ont été arrêtées,

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).





<a name="SYNTAXE2"></a>

### Supprimer une détection en arrière-plan

`<Résultat> = BeaconArrêteDétectionEnArrièrePlan(<Groupe de balises Beacon>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la détection a bien été arrêtée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Groupe de balises Beacon> : (Variable de type beaconGroupe)`**

Nom de la variable de type [beaconGroupe](../WDLang3/1000023105.md) correspondant au groupe de balises Beacon dont la détection doit être arrêtée. 


<a name="SYNTAXE3"></a>

### Supprimer plusieurs détections en arrière-plan

`<Résultat> = BeaconArrêteDétectionEnArrièrePlan(<Tableau de groupes de balises>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la détection a bien été arrêtée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Tableau de groupes de balises> : (Tableau de variables de type beaconGroupe)`**

Nom du tableau de variables de type [beaconGroupe](../WDLang3/1000023105.md) correspondant aux groupes de balises Beacon dont la détection doit être arrêtée.


## La structure Groupe
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000120"></a>
La structure Groupe est une structure prédéfinie du WLangage (aucune déclaration n'est nécessaire). Cette structure permet de :

- créer ou modifier un groupe de contacts.

- récupérer le contenu d'un groupe de contacts.






<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Les différentes variables de la structure Groupe
<a name="les_differentes_variables_structure_groupe_ELTTEXTE000144"></a>
La structure est composée des membres suivants :


|   |   |
| --- | --- |
| **mGroupe.Contact** | Tableau de chaîne de caractères. <br>Nom des contacts présents dans le groupe. <br><br>Pour accéder au nom des différents contacts du groupe, utilisez le format suivant : <br>mGroupe.Contact[&lt;Num contact&gt;]<br>Où &lt;Num contact&gt; correspond au numéro du contact voulu. Le nombre de contacts présents dans le groupe peut être connu grâce à la variable mGroupe.NbContact. |
| **mGroupe.Description** | Chaîne de caractères. <br>Description du groupe de contacts. |
| **mGroupe.EnDehors** | Booléen. <br>Indique s'il y a un groupe de contacts courant. |
| **mGroupe.ID** | Chaîne de caractères.<br>Identifiant unique du groupe de contacts |
| **mGroupe.NbContact** | Entier. <br>Nombre de contacts présents dans le groupe. |
| **mGroupe.Nom** | Chaîne de caractères.<br>Nom du groupe de contacts. |

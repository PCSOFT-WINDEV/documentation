


## grEffaceDessin (Fonction)

***En anglais : grClearPict***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Efface le dessin d'un graphe (dans un champ Image ou dans une fenêtre).

**Remarque** : Le graphe n'est pas détruit. Les données et les paramètres du graphe sont conservés en mémoire. Il suffit d'appeler la fonction [grDessine](../WDLang3/3042026.md) pour ré-afficher le graphe.

![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Les graphes](../WDChamp/3042060.md). 








<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Effacement du graphe "MonGraphe"
grEffaceDessin("MonGraphe")
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`grEffaceDessin(<Nom du graphe>)`
---

**`<Nom du graphe> : (Nom de champ ou chaîne de caractères)`**

Nom du graphe à effacer. Ce nom correspond :

- soit au nom du graphe défini par programmation avec la fonction [grCrée](../WDLang3/3042005.md).

- soit au nom du champ Graphe (présent dans l'éditeur de fenêtres ou de pages).






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Si le graphe est dessiné dans une fenêtre indépendante, la fenêtre est fermée.

- La fonction **grEffaceDessin** n'a aucun effet si le graphe n'est pas dessiné.

- La fonction **grEffaceDessin** n'a aucun effet dans les états.

- Pour supprimer les données d'un graphe, utilisez la fonction [grSupprimeSérie](../WDLang3/3042054.md) ou [grSupprimeTout](../WDLang3/3042055.md).

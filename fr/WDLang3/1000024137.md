


## &lt;Graphe&gt;.SMNRécupèreDonnée (Fonction)

***En anglais : &lt;Chart&gt;.SunburstGetData***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Récupère une valeur numérique dans un graphe de type Secteur multiniveau.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MaVal est réel = GRF_Graphe1.SMNRécupèreDonnée("Europe" + TAB + "France" + TAB + "Montpellier")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Champ Graphe>.SMNRécupèreDonnée(<Chemin de la portion>)`
---

**`<Résultat> : (Réel)`**



- Valeur à récupérer.

- 0 si le chemin est invalide ou si la portion ne contient pas de donnée.




**`<Champ Graphe> : (Nom de champ)`**

Nom du champ Graphe à manipuler (présent dans l'éditeur de fenêtres, de pages ou d'états).


**`<Chemin de la portion> : (Chaîne de caractères)`**

Chemin complet de la portion dont la valeur doit être récupérée. Ce paramètre est de la forme :

```txt
"<Nom du niveau de base>" + TAB + ["<Nom du 1er niveau>" + TAB + ...
["<Nom du dernier niveau>" + TAB + [...]]]"<Nom de la portion>"
```




<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Pour récupérer une donnée dans un graphe Boursier, il est conseillé d'utiliser la fonction [&lt;Graphe&gt;.BourseRécupèreDonnée](../WDLang3/1000023566.md).

- Pour récupérer une donnée dans un graphe Surface, il est conseillé d'utiliser la fonction [grSurfaceRécupèreDonnée](../ErrCompil/1000001036.md).

- Pour récupérer une donnée dans un autre type de graphe (Secteur, Histogramme, Courbe, etc.), utilisez [&lt;Graphe&gt;.RécupèreDonnée](../WDLang3/1000024122.md).

- Pour récupérer une donnée dans un graphe de type Heatmap, il est conseillé d'utiliser la fonction [&lt;Graphe&gt;.HMRécupèreDonnée](../WDLang3/1000024083.md).

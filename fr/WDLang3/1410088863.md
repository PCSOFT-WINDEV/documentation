
## Fonctions iCloud
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions de manipulation d'une clé dans un conteneur de paires "clé-valeur" basé sur Apple iCloud sont les suivantes : 



|   |   |
| --- | --- |
| [iCloudCléValeurEcrit](../WDLang3/1410088739.md) | Permet d'enregistrer la valeur d'une clé dans un conteneur de paires "clé-valeur" basé sur Apple iCloud. |
| [iCloudCléValeurExiste](../WDLang3/1410088740.md) | Vérifie l'existence d'une clé dans un conteneur de paires "clé-valeur" basé sur Apple iCloud. |
| [iCloudCléValeurListeClé](../WDLang3/1410088856.md) | Liste les clés contenues dans un conteneur de "clé-valeur" de Apple iCloud. |
| [iCloudCléValeurLit](../WDLang3/1410088741.md) | Lit la valeur d'une clé dans un conteneur de paires clé-valeur basé sur Apple iCloud. |
| [iCloudCléValeurSupprime](../WDLang3/1410088742.md) | Supprime une clé dans un conteneur de paires clé-valeur basé sur Apple iCloud. |
| [iCloudCléValeurSurveille](../WDLang3/1410088760.md) | Définit la procédure appelée lors du changement de la valeur d'une clé dans un conteneur de paires clé-valeur basé sur Apple iCloud. |





Remarque : Le but de des fonctions iCloud est d'enregistrer des données par application / par utilisateur (compte iCloud) pour que ces données soient synchronisées et partagées entre chaque périphérique de l'utilisateur. Pour utiliser ces fonctions : 

- en exécution, il est nécessaire d'activer la capabilities "iCloud" et de mettre à jour le fichier de provisioning sous Xcode.

- en Go simulateur, les fonctions fonctionnent mais sans mémorisation.

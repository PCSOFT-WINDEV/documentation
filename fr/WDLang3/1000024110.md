


## &lt;Graphe&gt;.NuageTaillePoint (Fonction)

***En anglais : &lt;Chart&gt;.ScatterPointSize***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Définit la taille des points dans un graphe de type "Nuage de points". 

**Rappel** : Par défaut, la taille d'un point dans un graphe de type "Nuage de points" est égale à 1 pixel.



![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Les graphes](../WDChamp/3042060.md). 








<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Augmentation de la taille des points dans le champ "GRF_MonGraphe".
// Ce graphe est de type "Nuage de points".
GRF_MonGraphe.NuageTaillePoint(3)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Champ Graphe>.NuageTaillePoint(<Taille>)`
---

**`<Champ Graphe> : (Nom de champ)`**

Nom du champ Graphe manipulé (présent dans l'éditeur de fenêtres, de pages ou d'états).


**`<Taille> : (Entier)`**

Taille d'un point (en pixels).



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Taille d'un point
<a name="taille_point_ELTPARAGRAPHE000239"></a>Le point affiché est un carré dont la taille du côté est calculée grâce à la formule suivante : (&lt;Taille&gt;\*2)-1 pixels.
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Graphe2.gif)

<a name="NOTE0_2"></a>


### Points dans les graphes de type "Nuage de points"
<a name="points_dans_les_graphes_type_nuage_points_ELTPARAGRAPHE000250"></a>

- Dans les graphes de type "Nuage de points", un point est défini par ses coordonnées (abscisse et ordonnée).

- Les données définies avec la fonction **&lt;Graphe&gt;.NuageTaillePoint** seront prises en compte lors du prochain appel à la fonction [&lt;Graphe&gt;.Dessine](../WDLang3/1000024062.md).

- La fonction **&lt;Graphe&gt;.NuageTaillePoint** est utilisée uniquement pour les graphes de type "Nuage de points". Cette fonction est sans effet pour tous les autres types de graphes.

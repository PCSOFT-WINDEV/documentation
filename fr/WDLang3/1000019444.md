
## NotifSupprime (Fonction)

***En anglais : NotifDelete***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime une notification affichée dans la barre système de l'appareil.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
notif est une Notification
notif.Titre = "Nouveau message reçu"
notif.Message = "Cliquez pour afficher le message"
NotifAjoute(notif)
...
NotifSupprime(notif)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = NotifSupprime(<Notification>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la notification a été supprimée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**Remarque** : Dans le simulateur, cette fonction est sans effet et renvoie toujours <u><u><u><u>Faux</u></u></u></u>. 

**`<Notification> : (Variable de type Notification)`**

Nom de la variable de type [Notification](../WDLang3/1000019441.md) correspondant à la notification à supprimer.


## SaaSEcritParamètreSite (Fonction)

***En anglais : SaaSWriteSiteParameter***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Enregistre une information spécifique pour un site SaaS dans la configuration d'un compte client.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Personnalisation du logo
SaaSEcritParamètreSite("logo", "./logo_PCSOFT.png")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = SaaSEcritParamètreSite(<Nom du paramètre> , <Valeur>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le paramètre a pu être modifié, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).


Remarque : L'utilisateur connecté doit posséder les droits pour écrire cette information (droits d'administration).

**`<Nom du paramètre> : (Chaîne de caractères)`**

Nom du paramètre à modifier dans le site en cours.

Les paramètres d'un site doivent avoir été définis dans l'[Administrateur SaaS](../SaaSAdminWeb/1600000.md) lors de la configuration du site.

**`<Valeur> : (Chaîne de caractères)`**

Nouvelle valeur du paramètre.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Les paramètres d'un site sont définis par l'administrateur du système SaaS lors de la configuration du site dans l'[Administrateur SaaS](../SaaSAdminWeb/1600000.md). Ils peuvent avoir des valeurs par défaut.

Les paramètres des sites sont destinés à permettre la personnalisation de l'IHM ou du comportement des sites SaaS. Pour plus de détails sur leur utilisation, consultez "[Développement d'un site SaaS](../WDLang3/9500214.md)".

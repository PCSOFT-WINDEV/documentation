


## &lt;Graphe&gt;.NuageReliePoint (Fonction)

***En anglais : &lt;Chart&gt;.ScatterLinkPoint***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Relie ou non les points dans un graphe de type "Nuage de points".

Exemple de graphe avec les points reliés : ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=TraitRelie.gif)


**Rappel** : Par défaut, les points d'un graphe de type "Nuage de points" sont reliés.



![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Les graphes](../WDChamp/3042060.md). 








<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Délier les points du graphe GRF_MonGraphe de type "Nuage de points"
GRF_MonGraphe.NuageReliePoint(Faux)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Relier ou délier les points d'un graphe de type Nuage de points

`<Champ Graphe>.NuageReliePoint(<Relier>)`
---

**`<Champ Graphe> : (Nom de champ)`**

Nom du champ Graphe à manipuler (présent dans l'éditeur de fenêtres, de pages ou d'états).


**`<Relier> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si les points doivent être reliés entre eux, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.





<a name="SYNTAXE2"></a>

### Relier ou délier les points d'une série d'un graphe de type Nuage de points

`<Champ Graphe>.grNuageReliePoint(<Numéro de la série> , <Relier>)`
---

**`<Champ Graphe> : (Nom de champ)`**

Nom du champ Graphe à manipuler (présent dans l'éditeur de fenêtres, de pages ou d'états).


**`<Numéro de la série> : (Entier)`**

Numéro de la série à manipuler.

**`<Relier> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si les points doivent être reliés entre eux, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Dans les graphes de type "Nuage de points", un point est défini par ses coordonnées (abscisse et ordonnée).

- Les données définies avec la fonction **&lt;Graphe&gt;.NuageReliePoint** seront prises en compte lors du prochain appel à la fonction [&lt;Graphe&gt;.Dessine](../WDLang3/1000024062.md).

- La fonction **&lt;Graphe&gt;.NuageReliePoint** est utilisée uniquement pour les graphes de type "Nuage de points". Cette fonction est sans effet pour tous les autres types de graphes.

- Les points sont reliés dans l'ordre dans lequel ils ont été ajoutés.

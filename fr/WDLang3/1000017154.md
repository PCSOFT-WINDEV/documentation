
## OBEXDéconnecte (Fonction)

***En anglais : OBEXDisconnect***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de se déconnecter d'un périphérique supportant le protocole OBEX.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Déconnexion OBEX
SI OBEXDéconnecte(nIDObex) = Vrai ALORS
	Info("Déconnexion effectuée")
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = OBEXDéconnecte(<Identifiant de la connexion OBEX>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la déconnexion a réussi, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. La fonction [ErreurInfo](../WDLang1/3013008.md) permet d'obtenir plus de renseignements sur le problème rencontré.




**`<Identifiant de la connexion OBEX> : (Entier)`**

Identifiant de la connexion OBEX à fermer. Cet identifiant est retourné par la fonction [OBEXConnecte](../WDLang3/1000017153.md).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Seules les piles bluetooth Microsoft sont supportées par cette fonction. Pour connaître la pile utilisée, utilisez la fonction [BTPile](../WDLang3/1000017229.md).

Pour plus de détails sur les clés supportées, consultez [Quelles piles utiliser ?](../WDLang3/1000017231.md)

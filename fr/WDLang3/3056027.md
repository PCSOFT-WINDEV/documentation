
## NetListeAccèsDistant (Fonction)

***En anglais : NetListRemoteAccess***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la liste des accès distants disponibles.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Présentation dans une liste des accès disponibles
ListeAjoute(LISTE_Liste1, NetListeAccèsDistant())
...
// Ouverture de l'accès sélectionné
NetOuvreAccèsDistant(LISTE_Liste1[LISTE_Liste1])
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = NetListeAccèsDistant()`
---

**`<Résultat> : (Chaîne de caractères)`**



- Liste des accès distants installés sur le poste en cours, sous la forme :
	
	```txt
	<Nom de l'accès 1> + RC + <Nom de l'accès 2> ...
	```


- Chaîne vide ("") si aucun accès distant n'est installé.

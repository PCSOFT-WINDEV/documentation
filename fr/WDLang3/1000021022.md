## Présentation
<a name="presentation_ELTTEXTE000108"></a>
Pour utiliser les notifications Push en Android, il est nécessaire de posséder un compte Google et d'effectuer une configuration spécifique. 

Cette page d'aide présente les différentes manipulations à effectuer. 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Les différentes étapes
<a name="les_differentes_etapes_ELTTEXTE000132"></a>
Les différentes étapes sont les suivantes : 

**1. Création d'un Google API Project**

- Depuis le navigateur, ouvrez la console [Google API](https://code.google.com/apis/console). 

- Authentifiez-vous en utilisant un compte Google. 

- Créez un nouveau "API Project" ou utilisez un projet existant. 




**2. Récupération du numéro de projet Google API**

- Récupérez le numéro de projet Google API : 

	1. Affichez les informations sur votre projet en cours. <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Push_Google_2.gif)


	2. Dans la fenêtre qui s'affiche, copiez le numéro : <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Push_Google_2B.gif)

- Ce numéro doit être indiqué dans l'assistant de génération de l'application Android. <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Push_Google_3.gif)





**3. Génération d'une "clé de l'API"**

- Dans un navigateur, allez sur [https://developers.google.com/mobile/add?platform=android&cntapi=gcm](https://developers.google.com/mobile/add?platform=android&cntapi=gcm)

- Sélectionnez le projet créé à l'étape précédente. 

- Renseignez le nom de package de l'application (défini dans l'assistant de génération Android).

- Cliquez sur "Choose and configure services".

- Cliquez sur "Enable Google Cloud Messaging".<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=push_Google_8.gif)


- Récupérez le numéro de clé de l'API (server API Key). Ce numéro de clé doit être fourni en paramètre à la fonction [NotifPushEnvoie](../WDLang3/1000020819.md). Cette clé apparaît ensuite dans la liste des clés du projet dans la console développeur.


## SaaSAdminConfigureDevise (Fonction)

***En anglais : SaaSAdminConfigureCurrency***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Configure la devise utilisée dans les relevés de connexions du système SaaS.

**Remarques** : 

- L'utilisation de cette fonction nécessite d'avoir ouvert une session d'administration SaaS au préalable en utilisant la fonction [SaaSAdminConnecte](../WDLang3/1000019098.md).

- L'utilisation de cette fonction est réservée à l'Administrateur SaaS. Un gestionnaire de compte client ne peut pas utiliser cette fonction. 



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Passage en Livres Sterling
SaaSAdminConfigureDevise("£")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = SaaSAdminConfigureDevise(<Devise>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la configuration a été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md). 




**`<Devise> : (Chaîne de caractères)`**

Caractère représentant la devise à employer dans les relevés de connexions du système SaaS.

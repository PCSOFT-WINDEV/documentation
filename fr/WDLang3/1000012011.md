
## TâcheLit (Fonction)

***En anglais : TaskRead***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Lit une tâche dans la liste des tâches d'une messagerie Lotus Notes ou Outlook. Pour être lue, cette tâche doit avoir été précédemment lue (fonction [TâcheDernier](../WDLang3/1000012002.md), [TâchePrécédent](../WDLang3/1000012004.md), [TâchePremier](../WDLang3/1000012005.md) ou [TâcheSuivant](../WDLang3/1000012007.md)), ajoutée (fonction [TâcheAjoute](../WDLang3/1000012001.md)) ou modifiée (fonction [TâcheModifie](../WDLang3/1000012003.md)).

**Attention** : Cette fonction **ne permet pas** de lire une tâche dans la liste des tâches d'une messagerie Outlook Express.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ouverture d'une session Lotus Notes
IDSession est un entier
IDSession = EmailOuvreSessionNotes("MotDePasse", "ServeurMars", "mail\julie.nsf", ...
					"C:\Users\Julie\AppData\Local\Lotus\Notes\Data\user.id")

// Lecture de la première tâche
TâchePremier(IDSession)
...
// Re-lecture de cette même tâche
TâcheLit(IDSession, mTâche.ID)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = TâcheLit(<Identifiant de la session> , <Identifiant de la tâche>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la tâche a été lue, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md). 




**`<Identifiant de la session> : (Entier)`**

Identifiant de la session à utiliser. Cet identifiant est renvoyé par une des fonctions suivantes : [EmailOuvreSessionNotes](../WDLang3/3032121.md), [EmailOuvreSessionOutlook](../WDLang3/3032128.md), [NotesOuvreBase](../WDLang3/1000005101.md) ou [OutlookOuvreSession](../WDLang3/3032123.md).

**`<Identifiant de la tâche> : (Chaîne de caractères)`**

Identifiant de la tâche à lire. Cet identifiant est contenu dans la variable mTâche.ID.

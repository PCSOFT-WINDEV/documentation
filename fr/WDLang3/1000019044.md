
## zipFichierArchivé (Type de variable)

***En anglais : zipArchivedFile***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **zipFichierArchivé** permet de manipuler : 

- un fichier contenu dans une archive correspondant à l'un des formats suivants : ZIP, CAB, RAR, WDZ, 7z, TAR ou TGZ (TAR.GZ).  

- un fichier à ajouter dans une archive.




Les caractéristiques de ce fichier peuvent être définies et modifiées à l'aide de différentes propriétés WLangage.

Ce type de variable est utilisé : 

- par les variables de type [zipArchive](../WDLang3/1000018679.md). 

- par la fonction [ zipAjouteFichier](../WDLang3/3082008.md). 






**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ouverture d'un fichier d'archive
MonArchive est un zipArchive
zipOuvre(MonArchive, "c:\Sauvegardes\archive.zip")

// Affichage dans la trace de la liste des fichiers contenus dans l'archive
POUR TOUT FichierArc DE MonArchive.Fichier
	Trace(FichierArc.Nom)
FIN

// Ajout d'un nouveau fichier dans l'archive
NouveauFichier est un zipFichierArchivé
NouveauFichier.Nom = "FichierArchivé.txt"
NouveauFichier.Contenu = "Ce fichier a été archivé"
zipAjouteFichier(MonArchive, NouveauFichier)
```

## Remarques


### Propriétés spécifiques à la description des variables de type zipFichierArchivé
<a name="proprietes_specifiques_description_des_variables_type_zipfichierarchive_ELTPARAGRAPHE000073"></a>Les propriétés suivantes peuvent être utilisées pour manipuler une variable de type **zipFichierArchivé** :

| **Nom** | **Type manipulé** | **Effet** |
| --- | --- | --- |
| AttributFichier | Chaîne de caractères | Une ou plusieurs lettres correspondant aux attributs du fichier ("AH" par exemple) mémorisés lors de son ajout dans l'archive.<br>Les lettres utilisées sont :<br><br>- R : Fichier en lecture seule. Le fichier est uniquement accessible en lecture.<br><br>- H : Fichier caché. Le nom du fichier est généralement grisé ou invisible dans l'explorateur.<br><br>- S : Fichier système. Le fichier est un fichier système Windows.<br><br>- A : Fichier "archive". Cet attribut indique que le fichier peut être archivé. Certains programmes se servent de cet attribut pour savoir quels fichiers ont été sauvegardés.<br><br><br>Cette information est restaurée lors de l'extraction du fichier.<br><br>![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) Cette propriété n'est pas disponible. |
| Contenu | Buffer | Contenu du fichier.<br>Cette propriété est disponible en lecture seulement si le fichier a déjà été ajouté dans une archive. |
| DateCréation | Chaîne de caractères ou variable de type DateHeure | Date de création du fichier. Cette date a été mémorisée lors de l'ajout du fichier dans l'archive.<br>Cette information est restaurée lors de l'extraction du fichier.<br>Cette information est conservée uniquement pour le format d'archive WDZ.<br><br> |
| DateDernierAccès | Chaîne de caractères ou variable de type DateHeure | Date de dernier accès au fichier. Cette date a été mémorisée lors de l'ajout du fichier dans l'archive.<br>Cette information est restaurée lors de l'extraction du fichier.<br>Cette information est conservée uniquement  pour le format d'archive WDZ.<br><br> |
| DateModification | Chaîne de caractères ou variable de type DateHeure | Date de modification du fichier. Cette date a été mémorisée lors de l'ajout du fichier dans l'archive.<br>Cette information est restaurée lors de l'extraction du fichier.<br><br> |
| ModeCryptage | Chaîne de caractères | Caractère indiquant si le fichier archivé est protégé par cryptage.<br>Les valeurs possibles sont :<br><br>- N : Fichier normal.<br><br>- + : Fichier crypté (il sera nécessaire de fournir un mot de passe pour l'extraire de l'archive).<br><br><br>Cette propriété est disponible en lecture seulement si le fichier a déjà été ajouté dans une archive. |
| Nom | Chaîne de caractères | Chemin du fichier mémorisé lors de l'ajout du fichier dans l'archive. |
| Taille | Entier sur 8 octets | Taille originale du fichier mémorisée lors de l'ajout du fichier dans l'archive.<br>**Cette propriété est disponible en lecture seulement**. |
| TailleCompressée | Entier sur 8 octets | Taille occupée par le fichier compressé dans l'archive.<br>**Cette propriété est disponible en lecture seulement**. |


## SocketCréeBluetooth (Fonction)

***En anglais : SocketCreateBluetooth***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Créé un socket en utilisant un périphérique Bluetooth.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Création du socket Bluetooth en spécifiant un nom de service
SI SocketCréeBluetooth(gsNomSocketBT, "SerialPortServiceClass_UUID") = Faux ALORS
	Erreur("Impossible de créer le socket Bluetooth", ErreurInfo(errComplet))
	RETOUR
FIN
```



```wl
// Création du socket Bluetooth en donnant un identifiant "GUID"
// indiqué dans les spécifications d'un périphérique (automate, ...)
SI SocketCréeBluetooth(gsNomSocketBT, "5") = Faux ALORS
	Erreur("Impossible de créer le socket Bluetooth", ErreurInfo(errComplet))
	RETOUR
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = SocketCréeBluetooth(<Nom du socket> , <Service> [, <Description>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le socket a été créé, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. En cas d'erreur, il est possible de connaître le détail de l'erreur grâce à la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Nom du socket> : (Chaîne de caractères)`**

Nom à donner au socket. Ce nom sera utilisé dans toutes les fonctions socket. 
Attention : le nom du socket est "case-sensitive".

**`<Service> : (Chaîne de caractères)`**

Correspond :

- soit au nom d'un service connu (voir Notes).

- soit au GUID d'un service.




**`<Description> : (Chaîne de caractères optionnelle)`**

Description du socket utilisé par le service de découverte des services Bluetooth.

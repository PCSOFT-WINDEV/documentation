


## &lt;Graphe&gt;.Dessine (Fonction)

***En anglais : &lt;Chart&gt;.Draw***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Dessine un graphe selon les paramètres spécifiés.

**Remarques** : 

- La fonction **&lt;Graphe&gt;.Dessine** prend en compte les nouveaux paramètres spécifiés depuis le dernier appel à cette même fonction.

- Cette fonction ne doit pas être utilisée pour les champs Graphe présents dans l'éditeur d'états.


![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Les graphes](../WDChamp/3042060.md). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Dessin d'un graphe dans un champ Graphe
GRF_MonGraphe.Dessine()
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Champ Graphe>.Dessine()`
---

**`<Champ Graphe> : (Nom de champ)`**

Nom du champ Graphe à manipuler (présent dans une fenêtre ou une page).

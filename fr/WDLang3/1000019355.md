
## WiFiInfoConnexion (Fonction)

***En anglais : WiFiConnectionInfo***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie l'information spécifiée à propos de la connexion Wi-Fi actuelle de l'appareil.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
Info(ChaîneConstruit("Votre vitesse de connexion actuelle est de %1 Mbps", ...
	WiFiInfoConnexion(wifiVitesse)))
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = WiFiInfoConnexion([<Type d'information>])`
---

**`<Résultat> : (Chaîne de caractères)`**



- Information demandée. Si plusieurs types d'informations sont demandées, chaque type d'information est séparé par un Retour Chariot (RC). 
	L'ordre dans lequel sont renvoyées les informations (si elles sont demandées) est le suivant : *wifiSSID* + *wifiBSSID* + *wifiAdresseIP* + *wifiAdresseMac* + *wifiVitesse* + *wifiIDRéseau*

- Chaîne vide ("") si l'appareil n'est pas connecté à un point d'accès Wi-Fi. La variable [ErreurDétectée](../WDLang1/3087001.md) est à <u><u><u><u>Vrai</u></u></u></u>. La fonction [ErreurInfo](../WDLang1/3013008.md) permet d'obtenir plus de renseignements sur le problème rencontré.




**`<Type d'information> : (Constante optionnelle de type Entier)`**

Information demandée : 


|   |   |
| --- | --- |
| *wifiAdresseIP* | Adresse IP associée à la connexion.<br><br>![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Constante non disponible |
| *wifiAdresseMac* | Adresse Mac Wi-Fi de l'appareil.<br><br>Attention : A partir de la version 6 du SDK Android, l'adresse Mac retournée est de la forme "02:00:00:00:00:00".<br><br>![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Constante non disponible |
| *wifiBSSID* | BSSID (Basic Service Set Identifier) du point d'accès de la connexion. |
| *wifiIDRéseau* | <br><br>- Identifiant du réseau auquel l'appareil est connecté,<br><br>- Chaîne vide ("") si l'appareil n'est connecté à aucun réseau connu.<br><br><br> |
| *wifiSSID* | SSID (Service Set Identifier) du réseau Wi-Fi auquel l'appareil est connecté. |
| *wifiTout*<br>(Valeur par défaut) | Informations complètes sur la connexion (correspond à la combinaison de constantes *wifiSSID*+ *wifiBSSID*+ *wifiAdresseIP*+ *wifiAdresseMac*+ *wifiVitesse*+ *wifiIDRéseau*).<br><br>![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) La valeur renvoyée pour les constantes *wifiAdresseIP* et *wifiAdresseMac* correspond à une chaîne vide. |
| *wifiVitesse* | Vitesse de la liaison Wi-Fi (exprimée en Mbps). |

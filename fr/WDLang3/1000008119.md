


## Fonctions de gestion des groupes de contacts
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions de gestion des groupes de contacts présents dans le carnet d'adresses d'une messagerie Lotus Notes, Domino ou Outlook sont les suivantes :



|   |   |
| --- | --- |
| [GroupeAjoute](../WDLang3/1000008111.md) | Ajoute un groupe de contacts dans le carnet d'adresses d'une messagerie Lotus Notes ou Outlook. |
| [GroupeDernier](../WDLang3/1000008112.md) | Lit le dernier groupe de contacts présent dans le carnet d'adresses d'une messagerie Lotus Notes ou Outlook. |
| [GroupeLit](../WDLang3/1000008121.md) | Lit un groupe de contacts présent dans le carnet d'adresses d'une messagerie Lotus Notes ou Outlook. |
| [GroupeModifie](../WDLang3/1000008113.md) | Modifie le groupe de contacts en cours dans le carnet d'adresses d'une messagerie Lotus Notes ou Outlook. |
| [GroupePrécédent](../WDLang3/1000008114.md) | Lit le groupe de contacts précédant le groupe en cours, dans le carnet d'adresses d'une messagerie Lotus Notes ou Outlook. |
| [GroupePremier](../WDLang3/1000008115.md) | Lit le premier groupe de contacts présent dans le carnet d'adresses d'une messagerie Lotus Notes ou Outlook. |
| [GroupeRAZ](../WDLang3/1000008116.md) | Réinitialise la [structure Groupe](../WDLang3/1000008120.md). |
| [GroupeSuivant](../WDLang3/1000008117.md) | Lit le groupe de contacts suivant le groupe en cours, dans le carnet d'adresses d'une messagerie Lotus Notes ou Outlook. |
| [GroupeSupprime](../WDLang3/1000008118.md) | Supprime le groupe de contacts en cours du carnet d'adresses d'une messagerie Lotus Notes ou Outlook. |

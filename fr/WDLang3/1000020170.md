
## LuminositéModifie (Fonction)

***En anglais : BrightnessModify***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie le réglage de la luminosité de l'écran de l'appareil.

**Remarque** : Cette fonction peut être utilisée dans l'émulateur.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Applique une luminosité maximale à l'écran
LuminositéModifie(100)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`LuminositéModifie(<Luminosité>)`
---

**`<Luminosité> : (Entier ou constante de type Entier)`**

Nouvelle valeur de la luminosité : 

- valeur comprise entre 0 et 100. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Cas particuliers 
<a name="cas_particuliers_ELTPARAGRAPHE000067"></a>

- La modification de la luminosité agit sur le paramétrage de l'appareil. La nouvelle luminosité est donc globale à toutes les applications et sera conservée à la fermeture de l'application.

- Pour connaître la valeur du réglage actuel de la luminosité de l'écran, utilisez la fonction [LuminositéRécupère](../WDLang3/1000019810.md).

- Il est déconseillé de spécifier une luminosité trop faible (inférieure à 10). En effet, la luminosité minimale peut varier d'un appareil à l'autre et l'écran du téléphone peut s'éteindre si la limite est atteinte.


## NetMessageBoxClient (Fonction)

***En anglais : NetClientMessageBox***

<a name="Utilisation"></a>
<a name="description"></a>
Affiche une boîte de dialogue sur chaque poste client connecté au serveur. Le serveur est un serveur FTP (File Transfer Protocol) ou RPC (Remote Procedure Call) réalisé avec WINDEV.

**Remarque** : Cette fonction n'est validée que lorsque toutes les boîtes de dialogue sont fermées sur tous les postes clients.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple
<a class="notetitle" target="_blank" href="$DOC$=1000003056017&name=netmessageboxclient_fonction&product=WD">Voir des exemples supplémentaires</a>

```wl
IDServeur est un entier = NetDémarreServeur()
// Affiche une boîte de dialogue OuiNon sur le poste client
SI NetMessageBoxClient(IDServeur, "Voulez vous continuer ?", ...
	"", netOuiNon) = 1 ALORS
	...
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = NetMessageBoxClient(<Identifiant du serveur> , <Message> , <Titre de la fenêtre> , <Type de la fenêtre>)`
---

**`<Résultat> : (Entier)`**

Numéro du bouton cliqué dans la boîte de message par le dernier poste client.

**`<Identifiant du serveur> : (Entier)`**

Identifiant du serveur (obtenu par la fonction [NetDémarreServeur](../WDLang3/3056013.md)).

**`<Message> : (Chaîne de caractères Ansi)`**

Texte à afficher dans la boîte de dialogue.

**`<Titre de la fenêtre> : (Chaîne de caractères Ansi)`**

Texte affiché dans la barre de titre de la fenêtre.

**`<Type de la fenêtre> : (Constante de type entier)`**

Type de la boîte de dialogue à afficher.


|   |   |
| --- | --- |
| *netConfirmer* | Boîte de message avec les boutons Oui, Non et Annuler. |
| *netInfo* | Boîte de message de type information (bouton OK uniquement). |
| *netOkAnnuler* | Boîte de message avec les boutons OK et Annuler. |
| *netOuiNon* | Boîte de message avec les boutons Oui et Non. |
| *netRéessayerAnnuler* | Boîte de message avec les boutons Réessayer et Annuler. |
| *netRéessayerIgnorer* | Boîte de message avec les boutons Abandon, Réessayer et Ignorer. |

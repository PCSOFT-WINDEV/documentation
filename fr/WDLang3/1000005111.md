
## NotesFermeBase (Fonction)

***En anglais : NotesCloseDatabase***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ferme la base locale ou distante Lotus Notes ou Domino utilisée. Cette base a été préalablement ouverte à l'aide de la fonction [NotesOuvreBase](../WDLang3/1000005101.md).

**Remarque** : Cette base sera automatiquement fermée à la fin du programme WINDEV ou de la session WEBDEV.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ouverture de la session
IDSession est un entier
IDSession = NotesOuvreConnexion("MotDePasse", "ServeurMars", ...
		"C:\Users\Julie\AppData\Local\Lotus\Notes\Data\user.id")
// Ouverture de la base
SI NotesOuvreBase(IDSession, "mail/julie.nsf") <> 0 ALORS
	Info("Session ouverte et base spécifiée")
SINON
	Erreur(ErreurInfo())
FIN
...
// Fermeture de la base
NotesFermeBase(IDSession)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = NotesFermeBase(<Identifiant de la session>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la base Lotus Notes a été fermée, 

- <u><u><u><u>Faux</u></u></u></u> en cas d'erreur. Pour obtenir le libellé de l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.




**`<Identifiant de la session> : (Chaîne de caractères)`**

Identifiant de la session à manipuler. Cet identifiant est retourné par la fonction [NotesOuvreConnexion](../WDLang3/1000005102.md).

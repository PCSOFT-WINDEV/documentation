
## SaaSAdminModifieUtilisateur (Fonction)

***En anglais : SaaSAdminModifyUser***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie un utilisateur SaaS.

**Remarques** : 

- L'utilisation de cette fonction nécessite d'avoir ouvert une session d'administration SaaS au préalable en utilisant la fonction [SaaSAdminConnecte](../WDLang3/1000019098.md).

- L'Administrateur SaaS gère tous les utilisateurs de tous les clients. Un gestionnaire de compte client ne peut modifier des utilisateurs que dans son propre compte client. 



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonUtilisateur est un saasUtilisateur
...
MonUtilisateur = SaaSAdminChercheUtilisateur("marc@hybrid.com")
MonUtilisateur.Nom = "marc.durand@hybrid.com"

SaaSAdminModifieUtilisateur(MonUtilisateur)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = SaaSAdminModifieUtilisateur(<Utilisateur>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la modification a été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Utilisateur> : (Variable de type saasUtilisateur)`**

Nom de la variable de type [saasUtilisateur](../WDLang3/1000019083.md) correspondant à l'utilisateur à modifier dans la base SaaS.

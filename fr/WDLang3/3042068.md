


## grSuperposeGraphe (Fonction)

***En anglais : grOverlayChart***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet d'afficher deux graphes de type différent dans le même graphe. Les deux graphes sont superposés.

**Remarque** : Le premier graphe doit être un graphe de type Histogramme (Histogramme empilé, groupé, ...), de type Boursier (MinMax, chandelier, ...) ou de type Nuage de points et le second graphe doit être de type Courbe.



![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Les graphes](../WDChamp/3042060.md). 








<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI grSuperposeGraphe(GRF_MonHistogramme, GRF_Courbe) = Vrai ALORS
	Info("Les deux graphes sont superposés")
FIN
grDessine(GRF_MonHistogramme)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = grSuperposeGraphe(<Graphe destination> , <Graphe source>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si les graphes ont été superposés, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Graphe destination> : (Chaîne de caractères)`**

Nom du graphe qui affichera les deux graphes en même temps. Ce graphe contient le premier graphe (de type histogramme, boursier ou nuage de points).

**`<Graphe source> : (Chaîne de caractères)`**

Nom du graphe qui doit s'afficher dans le graphe destination. Ce graphe doit être de type Courbe. Il se superposera au &lt;Graphe destination&gt;. Ce graphe est pris en compte en tant que source de données : son titre et ses paramètres ne sont pas pris en compte. Seules les informations concernant la légende sont conservées.
Si ce paramètre correspond à une chaîne vide (""), la liaison entre les deux graphes est supprimée.
Ce paramètre doit correspondre à un graphe différent du &lt;Graphe destination&gt; (dans le cas contraire, les graphes s'écrasent). 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- La superposition des graphes définie avec la fonction **grSuperposeGraphe** sera prise en compte lors du prochain appel à la fonction [grDessine](../WDLang3/3042026.md).

- Attention : La fonction **grSuperposeGraphe** ne gère pas la mire des différents graphes superposés.

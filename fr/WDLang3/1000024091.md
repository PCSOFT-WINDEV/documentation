


## &lt;Graphe&gt;.MAJAutomatiqueEtiquetteCatégorie (Fonction)

***En anglais : &lt;Chart&gt;.AutoRefreshCategoryLabel***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Identifie ou définit si les étiquettes de catégorie sont mises à jour à chaque dessin du graphe. Dans ce cas, à chaque appel de la fonction [&lt;Graphe&gt;.Dessine](../WDLang3/1000024062.md), le contenu de la série est automatiquement recalculé pour refléter les derniers changements de la source de données. Les étiquettes de catégories sont alors mises à jour si nécessaire.

**Exemple d'utilisation** : une série dans un graphe sert de série de référence : cette série n'est pas mise à jour automatiquement.



![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Les graphes](../WDChamp/3042060.md). 






<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Champ Graphe
SI GRF_MonGraphe.MAJAutomatiqueEtiquetteCatégorie() = Vrai ALORS
	GRF_MonGraphe.Dessine()
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Identifier le mode de mise à jour des étiquettes de catégorie

`<Résultat> = <Champ Graphe>.MAJAutomatiqueEtiquetteCatégorie()`
---

**`<Résultat> : (Booléen)`**

Mode de mise à jour actuel des étiquettes de catégories : 

- <u><u><u><u>Vrai</u></u></u></u> si les étiquettes sont mises à jour automatiquement, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Champ Graphe> : (Nom de champ)`**

Nom du champ Graphe à manipuler (présent dans l'éditeur de fenêtres, de pages ou d'états).  



<a name="SYNTAXE2"></a>

### Définir le mode de mise à jour des étiquettes de catégorie

`<Champ Graphe>.MAJAutomatiqueEtiquetteCatégorie(<Mise à jour automatique>)`
---

**`<Champ Graphe> : (Nom de champ)`**

Nom du champ Graphe à manipuler (présent dans l'éditeur de fenêtres, de pages ou d'états).  


**`<Mise à jour automatique> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> pour une mise à jour automatique, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Si les étiquettes de catégories sont en mode automatique, les étiquettes de catégories sont automatiquement recalculées lors du prochain appel à la fonction [&lt;Graphe&gt;.Dessine](../WDLang3/1000024062.md).

- Si les étiquettes de catégories ne sont pas en mode automatique, les étiquettes de catégories seront recalculées uniquement lors du prochain appel de la fonction [&lt;Graphe&gt;.SourceEtiquetteCatégorie](../WDLang3/1000024138.md).

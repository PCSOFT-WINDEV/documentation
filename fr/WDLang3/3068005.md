
## WINDEV Mobile et les SMS
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000149"></a>
WINDEV Mobile permet d'envoyer et de lire simplement des SMS grâce aux fonctions WLangage SMS.

Un SMS (Short Message Service, appelé également Textos, Mini messages, Télémessages, ...) correspond à un message texte (160 caractères maximum) envoyé sur un téléphone portable.

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=EnvoiSMS.gif)


**Remarque** : Les caractéristiques des SMS sont contenues dans la [structure SMS](../WDLang3/3068003.md).

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Configuration nécessaire
<a name="configuration_necessaire_ELTTEXTE000173"></a>
Pour pouvoir utiliser les fonctions SMS, l'application doit être installée :

- soit sur un téléphone Android.

- soit sur un téléphone iOS. 




![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Le SMS sera préparé par l'application WINDEV Mobile, mais le message sera envoyé par l'application native du téléphone.

## Notes
<a name="notes_ELTTEXTE000197"></a>
<a name="NOTE3_2"></a>


### Gestion des différents types de numéros
<a name="gestion_des_differents_types_numeros_ELTPARAGRAPHE000043"></a>Deux types de numéros peuvent être utilisés pour envoyer des SMS :

- Les numéros courts (également appelés "Nationaux"). Ces numéros sont accessibles uniquement depuis le territoire national.

- Les numéros classiques (également appelés "Internationaux"). Ces numéros sont accessibles depuis n'importe quel endroit.





<a name="NOTE3_4"></a>


### Fonctionnement différent en mode GO et en utilisation réelle
<a name="fonctionnement_different_mode_utilisation_reelle_ELTPARAGRAPHE000054"></a>En **mode GO** (simulation sur le poste de développement), une erreur WLangage est générée lors de l'appel d'une fonction SMS ou lors de l'utilisation d'une des variables de la structure SMS.

### Appels téléphoniques
<a name="appels_telephoniques_ELTPARAGRAPHE000094"></a>WINDEV Mobile permet également de gérer simplement les appels téléphoniques grâce aux fonctions suivantes : 



|   |   |
| --- | --- |
| [telDialerAffiche](../WDLang3/1000019236.md) | Ouvre l'application de téléphonie (dialer) par défaut et affiche le numéro spécifié. Aucun appel n'est déclenché. |
| [telDialerCompose](../WDLang3/1000019235.md) | Ouvre l'application de téléphonie (dialer) par défaut du téléphone et compose le numéro spécifié. |




## Fonctions de gestion des ports série et parallèle
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>









|   |   |
| --- | --- |
| [sDansFileEntrée](../WDLang3/3062004.md) | Récupère le nombre d'octets en attente dans le buffer d'entrée du port série ou du port infrarouge spécifié. |
| [sDansFileSortie](../WDLang3/3062005.md) | Récupère le nombre d'octets en attente de transfert dans le buffer de sortie du port série, du port parallèle ou du port infrarouge spécifié. |
| [sEchap](../WDLang3/3062006.md) | Lance diverses fonctions agissant directement sur les bits du port série, du port parallèle ou du port infrarouge, indépendamment du protocole de communication. |
| [sEcrit](../WDLang3/3062009.md) | Écrit une chaîne de caractères dans le buffer de sortie du port série, du port parallèle ou du port infrarouge spécifié. |
| [sEvénement](../WDLang3/3062014.md) | Branche un événement sur un port série. |
| [sFerme](../WDLang3/3062010.md) | Ferme le port série, le port parallèle ou le port infrarouge spécifié. |
| [sFinEvénement](../WDLang3/3062008.md) | Débranche la détection d'un événement sur un port série. |
| [sFixeParamètre](../WDLang3/3062011.md) | Fixe ou modifie les paramètres de configuration du port série, du port parallèle ou du port infrarouge spécifié. |
| [sListePort](../WDLang3/1000025980.md) | Renvoie la liste des noms des ports séries et des ports parallèles présents sur la machine. |
| [sLit](../WDLang3/3062012.md) | Lit une chaîne de caractères dans le buffer d'entrée du port série spécifié. |
| [sOuvre](../WDLang3/3062013.md) | Ouvre et initialise :<br><br>- soit un port série.<br><br>- soit un port parallèle.<br><br>- soit un port infrarouge.<br><br><br> |



Autres fonctions liées : 



|   |   |
| --- | --- |
| [sCalculeCrc16](../WDLang1/3062007.md) | Contrôle une chaîne de caractères avant et après transmission entre applications WINDEV / entre sites WEBDEV. |
| [sCalculeCrc32](../WDLang1/3062015.md) | Calcule le code de redondance cyclique (CRC) d'un tampon. |






- ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Impressiondirecte.gif) ***Exemples unitaires (WINDEV)*** : **Impression directe** <br>Impression en direct sur une imprimante matricielle avec les fonctions WLangage.<br><br>Dans cet exemple, nous abordons les principaux thèmes suivants :<br>1/ les fonctions de gestion des ports de communications<br>2/ l'envoi de séquences à une imprimante.<br><br>Résumé de l'exemple livré avec WINDEV :	<br>Ce programme réalisé avec WINDEV permet d'imprimer directement sur l'imprimante matricielle connectée au port parallèle de votre choix. <br>Cet exemple utilise l'ordre du WLangage "sEcrit".<br>Dans le domaine des applications exploitant des caisses enregistreuses, cette fonctionnalité est très utile.<br>Cet exemple n'est pas adapté à l'utilisation d'une imprimante laser ; on préférera dans ce cas utiliser l'éditeur d'états de WINDEV.




## UPNPCherchePériphérique (Fonction)

***En anglais : UPNPFindDevice***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Recherche un périphérique à partir de son nom unique (UDN).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
Device est un upnpPériphérique
Device = UPNPCherchePériphérique("uuid:72047cc4-1dd2-11b2-87b7-0000aaae85f5", 51300)
SI ErreurDétectée ALORS
	Info(ErreurInfo(errComplet))
SINON
	Info("Périphérique trouvé")
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = UPNPCherchePériphérique(<Nom du périphérique> , <Port> [, <Adresse> [, <Délai>]])`
---

**`<Résultat> : (Variable de type upnpPériphérique)`**

Nom de la variable de type [upnpPériphérique](../WDLang3/1000020903.md) contenant les caractéristiques du périphérique recherché.

**`<Nom du périphérique> : (Chaîne de caractères)`**

Nom unique (UDN) du périphérique à rechercher.

**`<Port> : (Entier)`**

Numéro du port utilisé pour la recherche.

**`<Adresse> : (Chaîne de caractères optionnelle)`**

Adresse sur laquelle la socket de recherche est créée.

**`<Délai> : (Entier optionnel ou Durée optionnelle)`**

Temps maximal de recherche (timeout) exprimé en secondes. Ce paramètre peut correspondre à : 

- un entier correspondant au nombre de secondes, 

- une variable de type Durée, 

- l'indication directe de la durée (par exemple '1s', '10cs' ou '10ms'). 


Par défaut, ce paramètre correspond à 5 secondes. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

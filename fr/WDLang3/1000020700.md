
## Utiliser le Wi-Fi
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000161"></a>
WINDEV et WINDEV Mobile proposent les fonctions WiFi pour accéder facilement aux réseaux Wi-Fi. Cette famille de fonctions est utilisable aussi bien dans les applications WINDEV que dans les applications Android. 





<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Wi-Fi et WINDEV : Principe
<a name="wifi_windev_principe_ELTTEXTE000185"></a>
Le principe d'utilisation des fonctions WiFi est le suivant : 

1. Lister les cartes réseaux Wi-Fi accessibles à partir du poste en cours (fonction [WiFiListeCarteRéseau](../WDLang3/1000020693.md)). 

2. Sélectionner la carte réseau Wi-Fi à utiliser (fonction [WiFiSelectCarteRéseau](../WDLang3/1000020694.md)). 

3. Détecter les points d'accès Wi-Fi disponibles grâce à la fonction [WiFiDétectePointAccès](../WDLang3/1000019353.md). 

4. Lister les réseaux Wi-Fi configurés sur le poste (fonction [WiFiListeRéseau](../WDLang3/1000019356.md)). 

5. Si un point d'accès Wi-Fi disponible est présent dans la liste des réseaux Wi-Fi enregistrés, il est possible de s'y connecter (fonction [WiFiConnecte](../WDLang3/1000019352.md)). 




Les fonctions disponibles sont les suivantes : 



|   |   |
| --- | --- |
| [WiFiConnecte](../WDLang3/1000019352.md) | Connecte l'appareil au réseau Wi-Fi spécifié. |
| [WiFiDétectePointAccès](../WDLang3/1000019353.md) | Lance la détection des points d'accès Wi-Fi actuellement accessibles depuis l'appareil. |
| [WiFiInfoConnexion](../WDLang3/1000019355.md) | Renvoie l'information spécifiée à propos de la connexion Wi-Fi actuelle de l'appareil. |
| [WiFiListeCarteRéseau](../WDLang3/1000020693.md) | Renvoie la liste des cartes réseaux Wi-Fi actuellement disponibles sur le poste en cours. |
| [WiFiListeRéseau](../WDLang3/1000019356.md) | Renvoie la liste des réseaux Wi-Fi actuellement configurés sur l'appareil. |
| [WiFiSelectCarteRéseau](../WDLang3/1000020694.md) | Sélectionne la carte réseau Wi-Fi qui sera utilisée pour les prochains appels des fonctions WiFi. |
| [WiFiSupprimeRéseau](../WDLang3/1000019357.md) | Supprime un réseau de la liste des réseaux Wi-Fi configurés de l'appareil. |

## Wi-Fi et Android : Principe
<a name="wifi_android_principe_ELTTEXTE000260"></a>
Le principe d'utilisation des fonctions WiFi est le suivant : 

1. Vérifier l'état Wi-Fi du mobile (fonction [WiFiEtat](../WDLang3/1000019354.md)) et l'activer si nécessaire (fonction [WiFiActive](../WDLang3/1000019350.md)). Lors de l'activation, si un réseau Wi-Fi accessible est présent dans la liste des réseaux Wifi disponibles, le mobile se connecte au meilleur réseau Wi-Fi. 

2. Détecter les points d'accès Wi-Fi (fonction [WiFiDétectePointAccès](../WDLang3/1000019353.md)). Il est ainsi possible de connaître tous les réseaux Wi-Fi accessibles. 

3. Lister les réseaux Wi-Fi enregistrés (fonction [WiFiListeRéseau](../WDLang3/1000019356.md)). 

4. Si un point d'accès Wi-Fi disponible est présent dans la liste des réseaux enregistrés, il est possible de s'y connecter (fonction [WiFiConnecte](../WDLang3/1000019352.md)). 

5. Si un point d'accès Wi-Fi disponible n'est pas présent dans la liste des réseaux enregistrés, il est possible de le configurer (fonction [WiFiAjouteRéseau](../WDLang3/1000019351.md)) et de s'y connecter (fonction [WiFiConnecte](../WDLang3/1000019352.md)).




Les fonctions disponibles sont les suivantes : 



|   |   |
| --- | --- |
| [WiFiActive](../WDLang3/1000019350.md) | Active ou désactive le Wi-Fi sur l'appareil. |
| [WiFiAjouteRéseau](../WDLang3/1000019351.md) | Ajoute un nouveau réseau à la liste des réseaux Wi-Fi configurés de l'appareil. |
| [WiFiConnecte](../WDLang3/1000019352.md) | Connecte l'appareil au réseau Wi-Fi spécifié. |
| [WiFiDétectePointAccès](../WDLang3/1000019353.md) | Lance la détection des points d'accès Wi-Fi actuellement accessibles depuis l'appareil. |
| [WiFiEtat](../WDLang3/1000019354.md) | Renvoie l'état actuel d'activation du Wi-Fi sur l'appareil ou demande à être notifié des changements d'état d'activation du Wi-Fi. |
| [WiFiInfoConnexion](../WDLang3/1000019355.md) | Renvoie l'information spécifiée à propos de la connexion Wi-Fi actuelle de l'appareil. |
| [WiFiListeRéseau](../WDLang3/1000019356.md) | Renvoie la liste des réseaux Wi-Fi actuellement configurés sur l'appareil. |
| [WiFiSupprimeRéseau](../WDLang3/1000019357.md) | Supprime un réseau de la liste des réseaux Wi-Fi configurés de l'appareil. |

## Wi-Fi et iOS : Principe
<a name="wifi_ios_principe_ELTTEXTE000342"></a>
Dans les applications iOS, les fonctions Wi-Fi permettent de : 

- Connaître l'état d'activation du Wi-Fi. 

- Obtenir la liste des réseaux Wi-Fi. 


Les fonctions disponibles sont les suivantes : 



|   |   |
| --- | --- |
| [WiFiEtat](../WDLang3/1000019354.md) | Renvoie l'état actuel d'activation du Wi-Fi sur l'appareil ou demande à être notifié des changements d'état d'activation du Wi-Fi. |
| [WiFiListeRéseau](../WDLang3/1000019356.md) | Renvoie la liste des réseaux Wi-Fi actuellement configurés sur l'appareil. |

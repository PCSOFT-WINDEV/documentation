
## DotNetVersion (Fonction)

***En anglais : DotNetVersion***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le numéro de version du Framework .Net disponible sur le poste en cours.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affichage de la version du Framework .NET
Info("Version du Framework .NET : " + DotNetVersion())
```


<a name="Exemple2"></a>



```wl
// Est-ce-que le Framework .NET 3.0 est installé sur le poste en cours ?
SI DotNetVersion(dnvInstallée, "3.0") = Vrai ALORS
	Info("Le Framework .NET 3.0 est installé")
SINON
	Info("Le Framework .NET 3.0 n'est pas installé")
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = DotNetVersion([<Information demandée> [, <Paramètre>]])`
---

**`<Résultat> : (Chaîne de caractères)`**



- Version du Framework .NET, 

- Chaîne vide ("") si le Framework .NET n'est pas disponible.




**`<Information demandée> : (Constante optionnelle de type entier)`**

Version du Framework .NET à renvoyer :


|   |   |
| --- | --- |
| *dnvCourant*<br>(Valeur par défaut) | Renvoie la version courante installée (c'est-à-dire la version la plus récente) |
| *dnvInstallée* | Renvoie Vrai si la version demandée dans &lt;Paramètre&gt; est installée sur le poste en cours. |
| *dnvListe* | Renvoie la liste des versions installées sur le poste en cours. Les différentes versions sont séparées par des RC. |



**`<Paramètre> : (Variant optionnel)`**

Si le paramètre &lt;Information demandée&gt; vaut dnvInstallée, &lt;Paramètre&gt; est une chaîne contenant la version à tester (par exemple "3.0"). La version à tester est comparée aux différentes versions installées sur le poste. La comparaison des numéros de version est du type "Commence par".

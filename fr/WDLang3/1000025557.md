


## CarteRécupèreStyle (Fonction)

***En anglais : MapGetStyle***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le style utilisé par la carte affichée dans le champ Carte.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Applique une style "rétro" à la carte
CarteChangeStyle(CARTE_MaCarte, ccsRétro)
...
soit StyleUtilisé = CarteRécupèreStyle(CARTE_MaCarte)
SI StyleUtilisé = ccsRétro ALORS Trace("Style rétro")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = CarteRécupèreStyle(<Champ Carte>)`
---

**`<Résultat> : (Constante de type Entier)`**

Style utilisé par la carte affichée dans le champ Carte : 


|   |   |
| --- | --- |
| *ccsArgenté* | Style "Argenté". |
| *ccsAubergine* | Style "Aubergine". |
| *ccsNuit* | Style "Nuit". |
| *ccsPersonnalisé* | Style "Personnalisé". |
| *ccsRétro* | Style "Retro". |
| *ccsSombre* | Style "Sombre". |
| *ccsStandard* | Style "Standard". |





**`<Champ Carte> : (Nom de champ)`**

Nom du champ Carte à manipuler. Si ce paramètre correspond à une chaîne vide (""), le champ Carte auquel appartient l'événement en cours sera utilisé.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Pour modifier le style de la carte affichée par le champ Carte, utilisez la fonction [CarteChangeStyle](../WDLang3/1000025556.md).

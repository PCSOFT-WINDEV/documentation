


## grTempsRéelIntervalle (Fonction)

***En anglais : grRealTimeInterval***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Fixe l'intervalle de temps visible dans un graphe temporel avec affichage en temps réel. 

Exemple : Dans cet exemple, l'intervalle de temps est de 10 secondes. <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Graphe%20temps%20r%E9el%20-%20HC%20N%B0001%201.gif)
Remarque : Cette fonction s'applique uniquement aux graphes ayant une échelle temporelle, pour lesquelles les données sont ajoutées avec la fonction [grAjouteDonnéeTemporelle](../WDLang3/1000023532.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
POUR i = 1 À 10
	d est Durée
	d.Minute = i
	grAjouteDonnéeTemporelle(GRF_MonGraphe, 1, d, 2 + Sinus(i*36))
FIN

d est Durée = 2min
grTempsRéelIntervalle(GRF_MonGraphe, d)
grDessine(GRF_SansNom1)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`grTempsRéelIntervalle(<Champ Graphe> , <Intervalle>)`
---

**`<Champ Graphe> : (Nom de champ)`**

Nom du champ Graphe à manipuler. 

**`<Intervalle> : (Durée)`**

Intervalle de temps visible dans le graphe (exprimé en millisecondes). Par défaut, cette valeur est de 60 secondes. Si ce paramètre correspond à 0, la valeur par défaut est utilisée.

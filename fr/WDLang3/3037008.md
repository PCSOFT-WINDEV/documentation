
## FaxBoîteEnvoi (Fonction)

***En anglais : FaxOutbox***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Énumère les fax en attente ou en cours d'envoi.

**Remarque** : Cette fonction peut être utilisée uniquement sous Windows 2000 ou XP. Sous Windows Vista (et supérieur), la gestion des fax est disponible uniquement avec Windows Professionnel et Windows Edition Intégrale.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
IDConnect est un entier
IDConnect = FaxConnecte()
...
ResEnvoiFax est un booléen
ResEnvoiFax = FaxEnvoie(IDConnect, "MonFax", "Message.txt", "0102030405")
...
ListeEnvoi est une chaîne 
ListeEnvoi = FaxBoîteEnvoi(IDConnect)
...
FaxDéconnecte(IDConnect)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = FaxBoîteEnvoi(<Identifiant connexion>)`
---

**`<Résultat> : (Chaîne de caractères)`**

Liste des identifiants numériques de fax séparés par des RC (Retour Chariot). Cet identifiant permet de manipuler le fax grâce aux fonctions de gestion des fax. Les identifiants de fax correspondent à tous les fax en attente ou en cours d'envoi (pas uniquement les fax envoyés avec la fonction [FaxEnvoie](../WDLang3/3037006.md)).

**`<Identifiant connexion> : (Entier)`**

Identifiant de la connexion obtenu avec la fonction [FaxConnecte](../WDLang3/3037005.md).

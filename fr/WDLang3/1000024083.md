


## &lt;Graphe&gt;.HMRécupèreDonnée (Fonction)

***En anglais : &lt;Chart&gt;.HMGetData***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Récupère une donnée d'un graphe de type Heatmap. 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Récupération d'une donnée
soit MaValeur = GRF_HMap.HMRécupèreDonnée(1, 1)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Champ Graphe>.HMRécupèreDonnée(<X> , <Y>)`
---

**`<Résultat> : (Réel)`**



- Valeur récupérée. 

- 0 si aucune valeur n'existe à la position indiquée.




**`<Champ Graphe> : (Nom de champ)`**

Nom du champ Graphe manipulé (présent dans l'éditeur de fenêtres, de pages ou d'états).


**`<X> : (Entier)`**

Position de la donnée à récupérer sur l'axe des abscisses.

**`<Y> : (Entier)`**

Position de la donnée à récupérer sur l'axe des ordonnées.


## Constantes DotNet

<a name="NOTE1_1"></a>


| Constante | Description | Détail | Utilisée par les fonctions |
| --- | --- | --- | --- |
| **dnvCourant** | Version du framework .NET | Version courante installée (c'est-à-dire la version la plus récente) | [DotNetVersion](../WDLang3/3031017.md) |

| **DotNetErrCode** | Type d'erreur | Erreur de la dernière procédure DotNet exécutée | [DotNetErreur](../WDLang3/3031004.md) |
| **DotNetErrDétail** | Type d'erreur | Détail de l'erreur. Ce message est au format XML. | [DotNetErreur](../WDLang3/3031004.md) |
| **DotNetErrMessage** | Type d'erreur | Message d'erreur renvoyé par le serveur sur la dernière procédure DotNet exécutée. | [DotNetErreur](../WDLang3/3031004.md) |

| **DotNetRésultatXML** | Type de résultat | Résultat sous forme XML de la dernière procédure DotNet exécutée | [DotNetDonneRésultat](../WDLang3/3031005.md) |
| **DotNetTypeBinaireBase64** | Type du paramètre | Binaire | [DotNetExécute](../WDLang3/3031016.md) |

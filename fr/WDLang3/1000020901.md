


## UPNPListePériphérique (Fonction)

***En anglais : UPNPListDevice***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Liste les périphériques UPnP disponibles sur le réseau. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png) 
```wl
tabPériphériques est un tableau de upnpPériphérique
tabPériphériques = UPNPListePériphérique("upnp:rootdevice", 51300)
SI ErreurDétectée ALORS
	Info(ErreurInfo(errComplet))
SINON
	InitListePériphérique()
FIN
```
<a name="Exemple2"></a>

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = UPNPListePériphérique(<Type> , <Port> [, <Adresse> [, <Délai>]])`
---

**`<Résultat> : (Tableau de variables de type upnpPéripérique)`**

Nom du tableau de variables de type [upnpPériphérique](../WDLang3/1000020903.md) contenant la liste des périphériques trouvés sur le réseau. 

**`<Type> : (Chaîne de caractères)`**

Type des périphériques à lister. 

**`<Port> : (Entier)`**

Numéro du port utilisé pour la recherche.

**`<Adresse> : (Chaîne de caractères optionnelle)`**

Adresse sur laquelle la socket de recherche est créée.

**`<Délai> : (Entier optionnel)`**

Temps maximal de recherche (timeout) exprimé en secondes. Ce paramètre peut correspondre à : 

- un entier correspondant au nombre de secondes, 

- une variable de type Durée, 

- l'indication directe de la durée (par exemple 1 s ou 10 ms). 


Par défaut, ce paramètre correspond à 5 secondes.




## grSMNCouleurPortion (Fonction)

***En anglais : grSunburstSectionColor***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Change la couleur d'une portion d'un graphe de type Secteur multiniveau.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ajoute les données
grSMNAjouteDonnée(GRF_Graphe, "Europe" + TAB + "France" + TAB + "Montpellier", 320000)
grSMNAjouteDonnée(GRF_Graphe, "Europe" + TAB + "France" + TAB + "Paris", 789000)
grSMNAjouteDonnée(GRF_Graphe, "Europe" + TAB + "France", 1109000) 
grSMNAjouteDonnée(GRF_Graphe, "Europe" + TAB + "Belgique", 900000)
grSMNAjouteDonnée(GRF_Graphe, "Asie", 250000)

// Personnalise les couleurs du graphe
grSMNCouleurPortion(GRF_Graphe, "Europe", RVB(106, 188, 227))
grSMNCouleurPortion(GRF_Graphe, "Asie", RVB(182, 115, 251)) 

grDessine(GRF_Graphe)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`grSMNCouleurPortion(<Nom du graphe> , <Chemin de la portion> , <Couleur>)`
---

**`<Nom du graphe> : (Nom de champ ou chaîne de caractères)`**

Nom du graphe à manipuler. Ce nom correspond :

- soit au nom du graphe défini par programmation avec la fonction [grCrée](../WDLang3/3042005.md).

- soit au nom du champ Graphe présent dans l'éditeur de fenêtres ou d'états.




**`<Chemin de la portion> : (Chaîne de caractères)`**

Chemin complet de la portion à colorer dans le graphe. Ce paramètre est de la forme : 

```txt
"<Nom du niveau de base>" + TAB + ["<Nom du 1er niveau>" + TAB + ...
["<Nom du dernier niveau>" + TAB + [...]]]"<Nom de la portion>"
```


**`<Couleur> : (Entier ou constante de type Entier)`**

Couleur de la portion spécifiée. Cette couleur peut correspondre à :

- une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)),

- une couleur TSL (obtenue avec la fonction [TSL](../WDLang1/3029057.md)),

- une couleur prédéfinie (voir [les couleurs prédéfinies du WLangage](../WDLang5/3010002.md)).  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Si une portion n'a pas une couleur spécifiée, une couleur lui est automatiquement attribuée. Il est ainsi possible d'obtenir un dégradé avec les portions de niveau supérieur si elles existent. 

- La couleur peut être modifiée à tout moment avant le dessin du graphe.




## &lt;Graphe&gt;.SérieAxeSecondaire (Fonction)

***En anglais : &lt;Chart&gt;.SeriesSecondaryAxis***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Détermine si une série sera dessinée sur l'axe secondaire.



![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Les graphes](../WDChamp/3042060.md). 












<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Place la série 2 de GRF_MonGraphe sur l'axe secondaire
GRF_MonGraphe.SérieAxeSecondaire(2, Vrai)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Champ Graphe>.SérieAxeSecondaire(<Numéro de la série> , <Position>)`
---

**`<Champ Graphe> : (Nom de champ)`**

Nom du champ Graphe à manipuler. 


**`<Numéro de la série> : (Entier)`**

Numéro de la série à modifier.

**`<Position> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> pour placer la série sur l'axe secondaire, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Cette fonction est utilisable uniquement avec les graphes possédant des axes. Elle ne fonctionne pas avec des graphes de type Secteur, Radar ou Entonnoir.

- Cette fonction est équivalente à la propriété [AxeSecondaire](../Proprietes/1000019187.md).

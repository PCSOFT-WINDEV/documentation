


## &lt;Variable emailSessionOutlook&gt;.FermeSession (Fonction)

***En anglais : &lt;emailOutlookSession variable&gt;.CloseSession***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Ferme la session Outlook.






<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ouverture de la session
MaSession est un emailSessionOutlook

MaSession..Nom = "ProfilOutlook"
MaSession.OuvreSession()
...
MaSession.FermeSession()
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Session Outlook>.FermeSession()`
---

**`<Session Outlook> : (Variable de type emailSessionOutlook)`**

Nom de la variable de type [emailSessionOutlook](../WDLang3/1000018767.md) correspondant à la session d'email à manipuler.

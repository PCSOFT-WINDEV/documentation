
## AppleIdentifiantPublicitaire (Fonction)

***En anglais : AppleAdvertisingIdentifier***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie l'identifiant publicitaire de l'appareil (iOS uniquement). 

L'identifiant publicitaire est un UUID unique pour chaque appareil. Cet identifiant est utilisé uniquement pour la publicité. Cette chaîne est utilisée par les publicitaires notamment dans les cas suivants :

- limitation du nombre d'expositions à la publicité, 

- attribution de la publicité selon le profil, 

- événements de conversion (publicité entraînant une visite du site), 

- estimation du nombre d'utilisateurs uniques, 

- détection de la fraude publicitaire,

- débogage.



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SELON AppleTrackingEtatAutorisation()
     CAS ateaIndéterminé
             AppleTrackingDemandeAutorisationAsynchrone(OnTracking)
     CAS ateaAutorisé
             PisteUtilisateur(AppleIdentifiantPublicitaire())
FIN
```

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = AppleIdentifiantPublicitaire()`
---

**`<Résultat> : (UUID)`**

Identifiant publicitaire de l'appareil.

<a name="NOTE0_1"></a>

## Remarques


### Autorisation nécessaire
<a name="autorisation_necessaire_ELTPARAGRAPHE000043"></a>

- Pour obtenir l'identifiant publicitaire de l'appareil, il est nécessaire d'avoir une autorisation spécifique. La présence de cette autorisation peut être testée avec la fonction [AppleTrackingEtatAutorisation](../WDLang3/1410087934.md). Si l'autorisation n'a pas été validée, il est possible de l'obtenir avec la fonction [AppleTrackingDemandeAutorisationAsynchrone](../WDLang3/1410087935.md).

- En cas d'absence d'autorisation, l'identifiant sera composé uniquement de zéro. 

- Cet identifiant n'est pas disponible dans le simulateur Xcode.





### Réinitialiser l'identifiant publicitaire
<a name="reinitialiser_identifiant_publicitaire_ELTPARAGRAPHE000056"></a>L'utilisateur a la possibilité de ré-initialiser son identifiant publicitaire dans les paramètres de son périphérique : 

- A partir de iOS 14 : 

	- Affichez les paramètres et sélectionnez "Confidentialité".

	- Défilez la liste et sélectionnez "Publicité Apple".

	- Désactivez "Annonces personnalisées".




- Avant iOS 14 : 

	- Affichez les paramètres et sélectionnez "Confidentialité".

	- Sélectionnez "Publicité" puis cliquez sur "Réinitialiser l'identifiant de publicité".

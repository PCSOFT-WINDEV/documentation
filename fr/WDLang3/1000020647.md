
## WidgetAffiche (Fonction)

***En anglais : WidgetDisplay***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Rafraîchit l'affichage d'une fenêtre de type Widget et exécute l'événement "Rafraîchissement du Widget". 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
WidgetAffiche(FEN_Widget1)
```


<a name="Exemple2"></a>

```wl
// Evénement "Rafraîchissement du Widget" de FEN_Widget
// Rafraîchit le RDV 
RafraîchitRDV()

WidgetAffiche(FEN_Détails)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`WidgetAffiche([<Fenêtre manipulée>])`
---

**`<Fenêtre manipulée> : (Nom de fenêtre optionnel)`**

Nom de la fenêtre Widget à rafraîchir. 





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Affichage de la fenêtre Widget
<a name="affichage_fenetre_widget_ELTPARAGRAPHE000048"></a>L'affichage d'une fenêtre Widget est effectué par le processus qui gère l'écran d'accueil de l'appareil. Le rafraîchissement de l'affichage n'est pas effectué en permanence mais uniquement dans les cas suivants : 

- Après l'ajout d'une nouvelle instance du Widget sur l'écran d'accueil.

- Après un clic sur le Widget (ou un de ses champs). 

- Après la mise à jour du contenu du Widget. 




La fonction **WidgetAffiche** permet de forcer manuellement le rafraîchissement de l'affichage. Cette fonction exécute également l'événement "Rafraîchissement du Widget" présent dans la fenêtre de type Widget.

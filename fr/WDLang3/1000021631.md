


## grHMLibelléY (Fonction)

***En anglais : grHMCaptionY***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie le libellé d'une catégorie sur l'axe des ordonnées d'un graphe de type Heatmap.

Note : Par défaut, le libellé d'une catégorie correspond à la valeur numérique de la catégorie.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
grHMLibelléY(GRF_HMap, 1, "Y1")
grHMLibelléY(GRF_HMap, 2, "Y2")
```

```wl
// Libellé des axes
POUR TOUT ELEMENT nIndice, nTempérature DE gtaTempérature
	grHMLibelléX(GRF_Graphe, nIndice, "" + nTempérature)
FIN
POUR TOUT ELEMENT nIndice, nVent DE gtaVent
	grHMLibelléY(GRF_Graphe, nIndice, "" + nVent)
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`grHMLibelléY(<Nom du graphe> , <Y> , <Libellé>)`
---

**`<Nom du graphe> : (Nom de champ ou chaîne de caractères)`**

Nom du graphe à manipuler. Ce nom correspond :

- soit au nom du graphe défini par programmation avec la fonction [grCrée](../WDLang3/3042005.md).

- soit au nom du champ Graphe présent dans l'éditeur de fenêtres ou d'états.




**`<Y> : (Entier)`**

Position de la catégorie sur l'axe des ordonnées. 

**`<Libellé> : (Chaîne de caractères)`**

Nouveau libellé de la catégorie. Si ce paramètre correspond à une chaîne vide (""), la valeur numérique de la catégorie est affichée.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Le libellé défini avec la fonction **grHMLibelléY** sera pris en compte lors du prochain appel à la fonction [grDessine](../WDLang3/3042026.md).




## &lt;Graphe&gt;.SurfaceAjouteDonnée (Fonction)

***En anglais : &lt;Chart&gt;.SurfaceAddData***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ajoute une donnée dans un graphe de type Surface. 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
x, y, z sont des réels
wImage est une Image

// Chargement de la photo en mémoire dans une variable de type Image
wImage = "couchersoleil.jpg"


// Balayage de l'image et de tous les points
// pour déterminer le taux de luminosité du point
// à l'aide de la fonction CouleurLuminosité
POUR i = 1 _A_ wImage.Largeur PAS 4
	POUR j = 1 _A_ wImage.Hauteur PAS 4
		x = i
		y = j
		z = CouleurLuminosité(dPixelCouleur(wImage, x, y))
		GRF_Surface.SurfaceAjouteDonnée(x, y, z)
	FIN
FIN

// On dessine le graphe dans le champ Graphe
GRF_Surface.Dessine()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Champ Graphe>.SurfaceAjouteDonnée(<X> , <Y> , <Z>)`
---

**`<Champ Graphe> : (Nom de champ)`**

Nom du champ Graphe (présent dans l'éditeur de fenêtres ou d'états).

**`<X> : (Réel)`**

Abscisse du point à ajouter.

**`<Y> : (Réel)`**

Ordonnée du point à ajouter.

**`<Z> : (Réel)`**

Altitude du point à ajouter.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Graphe Surface
<a name="graphe_surface_ELTPARAGRAPHE000200"></a>Pour les données d'un graphe de type Surface, le plan XY constitue le plancher de la surface, tandis que Z indique la hauteur du point.
<a name="NOTE0_2"></a>


### Ajout de données dans un graphe
<a name="ajout_donnees_dans_graphe_ELTPARAGRAPHE000210"></a>La fonction [&lt;Graphe&gt;.AjouteDonnée](../WDLang3/1000023533.md) permet d'ajouter des données dans un graphe (secteur, histogramme, ...).

Selon le type de graphe, des fonctions spécifiques permettent également d'ajouter des données : 

- Pour les graphes boursiers, il est conseillé d'utiliser la fonction [&lt;Graphe&gt;.BourseAjouteDonnée](../WDLang3/1000023565.md).

- Pour les graphes de type Nuage de points, il est conseillé d'utiliser la fonction [&lt;Graphe&gt;.NuageAjouteDonnéeXY](../WDLang3/1000024105.md).

- Pour les graphes Heatmap, il est conseillé d'utiliser la fonction [&lt;Graphe&gt;.HMAjouteDonnée](../WDLang3/1000024079.md).

- Pour les graphes Secteur Multiniveau, il est conseillé d'utiliser la fonction [&lt;Graphe&gt;.SMNAjouteDonnée](../WDLang3/1000024132.md). 




Rappel : Pour ajouter une valeur à une donnée d'un graphe, utilisez la fonction [&lt;Graphe&gt;.IncrémenteDonnée](../WDLang3/1000024086.md).

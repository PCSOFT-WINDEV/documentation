
## iCalendarVersRendezVous (Fonction)

***En anglais : iCalendarToAppointment***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Liste les rendez-vous décrits dans une chaîne de caractères au format iCalendar.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 
```wl
// Charge les rendez-vous décrits dans un fichier iCalendar dans un champ Planning 
// sChaîne est ici une chaîne ANSI
sChaîne est une chaîne = fChargeTexte("agenda.ics")
tabRDV est un tableau de RendezVous = iCalendarVersRendezVous(sChaîne)
SI PAS ErreurDétectée ALORS
	POUR TOUT ELEMENT rdv de tabRDV
		PlanningAjouteRendezVous(PLN_MonPlanning, rdv)
	FIN
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = iCalendarVersRendezVous(<iCalendar>)`
---

**`<Résultat> : (Tableau de RendezVous)`**

Tableau de variables de type [RendezVous](../WDLang1/1000019244.md) contenant la liste des rendez-vous décrits dans la chaîne de caractères.

**`<iCalendar> : (Chaîne de caractères ANSI au format UTF-8 ou Buffer)`**

Données au format iCalendar.





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Pour construire une chaîne de caractères au format iCalendar à partir d'une liste de rendez-vous, utilisez la fonction [RendezVousVersiCalendar](../WDLang3/1000021828.md).

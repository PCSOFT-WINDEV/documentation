


## Fonctions HTTP (syntaxe préfixée)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions suivantes (syntaxe préfixée) permettent de manipuler des pages Internet, d'envoyer des formulaires HTTP, ... :



|   |   |
| --- | --- |
| [&lt;Variable httpRéponse&gt;.RécupèreCookie](../WDLang3/1000021633.md) | Récupère les cookies lus dans la variable [httpRéponse](../WDLang3/1000021165.md) (contenant le résultat de la fonction [&lt;Variable httpRequête&gt;.Envoie](../WDLang3/1000021624.md)). |
| [&lt;Variable httpRequête&gt;.EcritCookie](../WDLang3/1000021607.md) | Ajoute ou modifie un cookie dans une requête HTTP. |
| [&lt;Variable httpRequête&gt;.Envoie](../WDLang3/1000021624.md) | Envoie une requête HTTP et attend la réponse du serveur HTTP. |
| [&lt;Variable httpRequête&gt;.GèreCookie](../WDLang3/1000021608.md) | Active ou désactive la gestion des cookies dans une requête [HTTPRequête](../WDLang3/3043007.md). |
| [&lt;Variable httpRequête&gt;.LitCookie](../WDLang3/1000021621.md) | Retourne la valeur d'un cookie reçu suite à une requête HTTP. |
| [&lt;Variable httpRequête&gt;.RAZCookie](../WDLang3/1000021622.md) | Supprime l'ensemble des cookies (globalement ou pour un domaine) stockés par les appels à la fonction [HTTPRequête](../WDLang3/3043007.md). |
| [&lt;Variable httpRequête&gt;.RécupèreCookie](../WDLang3/1000021627.md) | Récupère les cookies lus par un appel à la fonction [&lt;Variable httpRequête&gt;.Envoie](../WDLang3/1000021624.md) pour un domaine HTTP. |
| [&lt;Variable httpRequête&gt;.RemplaceCookie](../WDLang3/1000021623.md) | Remplace l'ensemble des cookies stockés pour un domaine par la fonction [&lt;Variable httpRequête&gt;.Envoie](../WDLang3/1000021624.md). |
| [&lt;Variable restRequête&gt;.Envoie](../WDLang3/1410089811.md) | Envoie une requête REST et attend la réponse du serveur. |


## NotesDocumentCherche (Fonction)

***En anglais : NotesDocumentSeek***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Recherche un document dans Lotus Notes.

**Remarque** : Pour rechercher un document dans une vue, utilisez la fonction [NotesVueChercheDernier](../WDLang3/1000005127.md) ou [NotesVueCherchePremier](../WDLang3/1000005128.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ouverture d'une session Lotus Notes
IDSession est un entier
IDSession = EmailOuvreSessionNotes("MotDePasse", "ServeurMars", ...
		"mail\julie.nsf", "C:\Users\Julie\AppData\Local\Lotus\Notes\Data\user.id")
...
// Recherche des documents contenant la chaîne "pc soft" 
SI NotesDocumentCherche(IDSession, "@contains(@lowercase(categories);'pc soft')") = Vrai ALORS
	// Enumération des documents trouvés
	TANTQUE NotesEnDehors(IDSession) = Faux
		// Affiche le nom des documents trouvés dans un champ Liste
		ListeAjoute(LISTE_DocumentTrouve, NotesChamp(IDSession, "Nom"))
		// Document trouvé suivant
		NotesDocumentSuivant(IDSession)
	FIN
SINON
	Info("Aucun document n'a été trouvé")
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = NotesDocumentCherche(<Identifiant de la session> , <Formule Lotus Notes>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si un ou plusieurs documents ont été trouvés, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. En cas d'erreur, la variable [ErreurDétectée](../WDLang1/3087001.md) est à <u><u><u><u>Vrai</u></u></u></u>. Pour connaître le libellé de l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.




Si aucun document n'a été trouvé, la variable [ErreurDétectée](../WDLang1/3087001.md) reste à <u><u><u><u>Faux</u></u></u></u> et la fonction [NotesEnDehors](../WDLang3/1000005109.md) renvoie <u><u><u><u>Vrai</u></u></u></u>.

**`<Identifiant de la session> : (Entier)`**

Identifiant de la session à utiliser. Cet identifiant est renvoyé par la fonction [EmailOuvreSessionNotes](../WDLang3/3032121.md) ou [NotesOuvreBase](../WDLang3/1000005101.md).

**`<Formule Lotus Notes> : (Chaîne de caractères)`**

Formule de sélection des documents dans Lotus Notes. Pour plus de détails sur ces formules de sélection, consultez la documentation de Lotus Notes.

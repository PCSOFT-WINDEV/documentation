<a name="Utilisation"></a>
<a name="description"></a>
Liste les disques disponibles :

- sur le poste serveur FTP (File Transfer Protocol) ou RPC (Remote Procedure Call) réalisé avec WINDEV,

- sur le poste client.




**Remarque** : Il est possible d'autoriser ou non l'utilisation de cette fonction sur un serveur FTP/RPC grâce à la fonction [NetOptionServeur](../WDLang3/3056022.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
ConnectRPC est un entier
ConnectRPC = NetConnecte("148.61.125.245", ServeurRPC, "GUEST", "")
...
// Affiche la liste des disques du serveur
Info(NetListeDisque(ConnectRPC))
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = NetListeDisque([<Identifiant connexion>])`
---

**`<Résultat> : (Chaîne de caractères Ansi)`**

Liste des disques disponibles sur le poste serveur ou client. Ce résultat est de la forme : 

```txt
Disque1 + TAB + Type1 + RC + Disque2 + TAB + Type2 + RC + ...
```


Les différents types de disques sont les suivants :


|   |   |
| --- | --- |
| CD | Disque CD ROM |
| FD | Lecteur de disquettes |
| HD | Disque dur |
| NW | Disque réseau |
| RD | Ramdisk |
| UN | Type inconnu |



**`<Identifiant connexion> : (Entier optionnel)`**

Identifiant de la connexion du poste client (obtenu par la fonction [NetConnecte](../WDLang3/3056004.md)). Si ce paramètre n'est pas précisé, la fonction **NetListeDisque** renvoie la liste des disques disponibles sur le poste en cours.


## Fonctions Modbus (syntaxe préfixée)



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions Modbus en syntaxe préfixée sont les suivantes : 



|   |   |
| --- | --- |
| [&lt;modbusSession&gt;.Déconnecte](../WDLang3/1000025954.md) | Ferme la session Modbus. |
| [&lt;modbusSession&gt;.EcritBit](../WDLang3/1000025967.md) | Écrit un ou plusieurs bits à une adresse précise dans un esclave Modbus. |
| [&lt;modbusSession&gt;.EcritFlottant](../WDLang3/1000026035.md) | Ecrit une ou plusieurs flottants simple précision à partir d'une adresse précise dans un esclave Modbus. |
| [&lt;modbusSession&gt;.EcritRegistre](../WDLang3/1000026036.md) | Ecrit un ou plusieurs registres (valeur 16 bits) à partir d'une adresse précise dans un esclave Modbus. |
| [&lt;modbusSession&gt;.LitBit](../WDLang3/1000026065.md) | Lit un ou plusieurs bits à partir d’une adresse précise sur un esclave Modbus. |
| [&lt;modbusSession&gt;.LitFlottant](../WDLang3/1000026079.md) | Lit un ou plusieurs flottants "simple précision" à partir d'une adresse précise dans un esclave Modbus. |
| [&lt;modbusSession&gt;.LitRegistre](../WDLang3/1000026080.md) | Lit un ou plusieurs registres à partir d'une adresse précise dans un esclave Modbus. |

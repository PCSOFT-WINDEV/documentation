
## FaxEtat (Fonction)

***En anglais : FaxStatus***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie l'état d'un fax précédemment envoyé avec la fonction [FaxEnvoie](../WDLang3/3037006.md). La fonction **FaxEtat** remplit la [structure FaxEtatComplet](../WDLang3/3037011.md) qui contient toutes les caractéristiques du fax.


**Remarque** : Cette fonction ne peut être utilisée que sous Windows 2000, XP ou version supérieure.

**Rappel** : Sous Windows Vista (et supérieur), la gestion des fax est disponible uniquement avec Windows Professionnel et Windows Edition Intégrale.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
IDConnect est un entier
IDConnect = FaxConnecte()
...
NomFax est une chaîne = "MonFax"
IDFax = FaxEnvoie(IDConnect, NomFax, "Message.txt", "0102030405")
...
SI FaxEtat(IDConnect, NomFax) = FaxEtatEnvoyé ALORS
	Info("Le fax a été envoyé")
SINON
	Info("Etat du fax : " + FaxEtatComplet.LibelleEtat)
FIN
...
FaxDéconnecte(IDConnect)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = FaxEtat(<Identifiant connexion> , <Nom du fax ou Identifiant du fax>)`
---

**`<Résultat> : (Constante de type Entier)`**

Etat du fax spécifié :


|   |   |
| --- | --- |
| *FaxEtatAnnulé* | La transmission du fax est annulé. |
| *FaxEtatAppel* | Le fax est en phase de numérotation. |
| *FaxEtatBloqué* | Numéro du destinataire bloqué. |
| *FaxEtatDéconnecté* | Déconnexion en cours de transmission. |
| *FaxEtatEnvoi* | Le fax est en train d'être envoyé. |
| *FaxEtatEnvoyé* | Le fax a été envoyé. |
| *FaxEtatErreur* | Impossible d'obtenir l'état du fax. |
| *FaxEtatErreurFatale* | Erreur fatale. |
| *FaxEtatFauxNuméro* | Numéro de fax erroné. |
| *FaxEtatGestionnaireEnAttente* | Le périphérique est disponible. |
| *FaxEtatHorsConnexion* | Le périphérique est hors ligne et indisponible. |
| *FaxEtatInactif* | Envoi de fax inactif (pause entre deux appels par exemple). |
| *FaxEtatIndisponible* | Le fax est indisponible (utilisé par une autre application). |
| *FaxEtatInit* | La transmission est en phase d'initialisation. |
| *FaxEtatOccupé* | Le destinataire est occupé. |
| *FaxEtatPasDeTonalité* | Pas de tonalité. |
| *FaxEtatPasFax* | Le périphérique a reçu un appel qui n'est pas un fax (appel vocal par exemple). |
| *FaxEtatPrisEnCharge* | Le gestionnaire de fax a pris en charge le document de fax et va le transmettre. |
| *FaxEtatRéception* | Le périphérique est en train de recevoir un fax. |
| *FaxEtatRépondu* | Le destinataire a répondu. |
| *FaxEtatReporté* | L'appel est reporté. |
| *FaxEtatRoutage* | Le périphérique est en train de router un fax reçu. |
| *FaxEtatSansRéponse* | Le destinataire ne répond pas. |
| *FaxEtatSonnerie* | Le fax du destinataire sonne. |



**`<Identifiant connexion> : (Entier)`**

Identifiant de la connexion obtenu avec la fonction [FaxConnecte](../WDLang3/3037005.md). Cet identifiant doit correspondre à celui de la connexion grâce à laquelle le fax a été envoyé.

**`<Nom du fax ou Identifiant du fax> : (Chaîne de caractères)`**

Correspond :

- soit au nom permettant d'identifier le fax envoyé. Ce nom a été donné avec la fonction [FaxEnvoie](../WDLang3/3037006.md).

- soit à l'identifiant du fax (obtenu par les fonctions [FaxBoîteEnvoi](../WDLang3/3037008.md) ou [FaxBoîteRéception](../WDLang3/3037009.md))




## TableurFerme (Fonction)

***En anglais : SpreadsheetClose***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ferme le document XLSX ouvert précédemment par la fonction [TableurCharge](../WDLang1/1000021340.md). 
<a name="sample_code"></a>

## Exemple


```wl
TableurCharge(TBLR_Tableur, "doc1.xlsx")
TableurFerme(TBLR_Tableur)
LanceAppliAssociée("doc1.xlsx")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`TableurFerme(<Champ Tableur>)`
---

**`<Champ Tableur> : (Nom de champ)`**

Nom du champ Tableur à manipuler. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Par défaut, la fonction [TableurCharge](../WDLang1/1000021340.md) laisse le fichier XLSX ouvert en écriture. Lors de l'enregistrement, seules les cellules modifiées dans le champ Tableur sont écrasées. 

- Pour ouvrir le fichier ".xlxs" avec une autre application, il est nécessaire de le fermer avec la fonction **TableurFerme**.

- Si aucun fichier ".xlsx" n'a été ouvert, la fonction **TableurFerme** ne fait rien.

- Cette fonction est utilisable uniquement sur un champ Tableur présent dans une fenêtre.




## Tests automatiques : Fonctions spécifiques aux tests
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Dans le code d'un test automatique, il est possible d'utiliser les fonctions suivantes :



|   |   |
| --- | --- |
| [TestAjouteItération](../WDLang1/1000017015.md) | Ajoute une itération dans le test en cours et donne les valeurs des paramètres du test pour cette itération. |
| [TestEcritRésultat](../WDLang1/1000013001.md) | Écrit une information ou une erreur dans le compte-rendu d'un test unitaire, sous l'éditeur de tests. |
| [TestErreur](../WDLang1/1000023506.md) | Provoque : <br><br>- une assertion lors du test de l'élément ou du projet (mode Go). <br><br>- une erreur en mode test automatique.<br><br><br> |
| [TestSurErreur](../WDLang1/1000023507.md) | Provoque : <br><br>- une assertion lors du test de l'élément ou du projet (mode Go). <br><br>- une erreur en mode test automatique.<br><br><br> |
| [TestVérifie](../WDLang1/1000017023.md) | Écrit un résultat dans le compte-rendu de test en fonction du résultat d'une expression booléenne. |
| [TestVérifieDifférence](../WDLang1/1000023397.md) | Vérifie si les expressions spécifiées sont différentes en programmation défensive et en mode test automatique. |
| [TestVérifieEgalité](../WDLang1/1000023398.md) | Vérifie si les expressions sont égales en programmation défensive et en mode test automatique. |
| [TestVérifieFaux](../WDLang1/1000023399.md) | Vérifie si une expression est fausse en programmation défensive et en mode test automatique. |
| [TestVérifieNonNull](../WDLang1/1000023400.md) | Vérifie si l'expression n'est pas Null en programmation défensive et en mode test automatique. |
| [TestVérifieNull](../WDLang1/1000023401.md) | Vérifie si l'expression est Null en programmation défensive et en mode test automatique. |
| [TestVérifieVrai](../WDLang1/1000023402.md) | Vérifie si une expression vaut Vrai en programmation défensive et en mode test automatique. |



Remarque : Dans le code d'un test automatique, il est également possible d'utiliser l'instruction [SUR](../Motscles/1511020.md).




## JSONVersChaîne (Fonction)

***En anglais : JSONToString***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Convertit le contenu d'une variable de type JSON en une chaîne au format JSON.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
Personne est un JSON
Personne.Nom = "DUPONT"
Personne.Prenom = "Michel"
// Affiche : { "Nom":"DUPONT", "Prenom":"Michel" }
Trace(JSONVersChaîne(Personne))
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = JSONVersChaîne(<Variable JSON> [, <Options>])`
---

**`<Résultat> : (Chaîne de caractères)`**

Chaîne au format JSON résultant de la conversion de la variable de type JSON. 
En cas d'échec, la variable [ErreurDétectée](../WDLang1/3087001.md) vaut <u><u><u><u>Vrai</u></u></u></u> et la fonction [ErreurInfo](../WDLang1/3013008.md) permet de connaître le détail de l'erreur.

**`<Variable JSON> : (Variable de type JSON)`**

Nom de la variable de type [JSON](../Motscles/1000023611.md) à convertir en chaîne au format JSON.

**`<Options> : (Entier optionnel)`**

Option de mise en forme de la chaîne au format JSON :	


|   |   |
| --- | --- |
| *psdMinifié* | Génère une chaîne au format JSON minifiée, sans aucun retour à la ligne ou caractère d'espacement. |
| *psdMiseEnForme* | Génère une chaîne au format JSON avec des retours à la ligne et une indentation. |


## SysRectangleEcran (Fonction)

***En anglais : SysScreenRectangle***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie les coordonnées du rectangle d'affichage correspondant à tous les écrans. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
Trace(SysRectangleEcran())
```


<a name="Exemple2"></a>

```wl
sRect est une chaîne = SysRectangleEcran()
GaucheEcran est un entier = ExtraitChaîne(sRect, 1)
HautEcran est un entier = ExtraitChaîne(sRect, 2)
DroitEcran est un entier = ExtraitChaîne(sRect, 3)
BasEcran est un entier = ExtraitChaîne(sRect, 4)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = SysRectangleEcran()`
---

**`<Résultat> : (Chaîne de caractères)`**

Dimensions de l'écran. Ces dimensions sont de la forme : 


```txt
<X> + TAB + <Y> + TAB + <X Maximum> + TAB + <Y Maximum>
```


- Les coordonnées (X, Y) correspondent au point haut gauche de l'écran.

- X Maximum : Largeur de l'écran ou somme des largeurs des écrans

- Y Maximum : Hauteur de l'écran ou somme des hauteurs des écrans

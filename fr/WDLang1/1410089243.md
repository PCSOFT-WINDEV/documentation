
## Italie.VérifieCF (Fonction)

***En anglais : Italy.CheckCF***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Teste la validité d'un numéro CF (numéro d'identification personnel en Italie).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Test d'une variable
MonNuméroCF est une chaîne = "RSSMRA74D22A001Q"
Italie.VérifieCF(MonNuméroCF)	// renvoie Vrai

// Test d'un champ
SAI_NuméroCF = "ABC"
Italie.VérifieCF(SAI_NuméroCF)	// renvoie Faux
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = Italie.VérifieCF(<Numéro à vérifier>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'élément spécifié contient un numéro CF valide.

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Numéro à vérifier> : (Nom de champ ou variable de type chaîne)`**

Numéro CF à vérifier. Ce paramètre peut correspondre : 

- au nom d'une variable de type chaîne.

- au nom d'un champ.


## fListeDisque (Fonction)

***En anglais : fListDisk***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la liste des disques installés sur la machine.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
sListeDisque est une chaîne 
sListeDisque = fListeDisque() // Renvoie "C:" + RC + "D:" + RC + "I:"
ListeAjoute(COMBO_Disque, sListeDisque)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = fListeDisque()`
---

**`<Résultat> : (Chaîne de caractères)`**

Liste des disques installés sur le poste au format suivant : 

```txt
<Nom du lecteur 1> + RC + <Nom du lecteur 2> + RC + ... + RC + <Nom du lecteur N>
```




<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Pour afficher la liste des disques dans un champ Liste ou Combo, utilisez la syntaxe : 


```txt
ListeAjoute(<Nom champ Liste ou Combo>, <Résultat>)
```




## &lt;Fenêtre&gt;.Restaure (Fonction)

***En anglais : &lt;Window&gt;.Restore***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Affiche une fenêtre WINDEV dans sa taille initiale.

**Remarque** : La fonction **&lt;Fenêtre&gt;.Restaure** réalise la même action que le bouton de restauration présent dans le coin haut droit des fenêtres.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affichage de la fenêtre "FEN_SaisieClient" dans sa taille initiale
FEN_SaisieClient.Restaure()
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Fenêtre>.Restaure()`
---

**`<Fenêtre> : (Nom de fenêtre)`**

Nom de la fenêtre à afficher dans sa taille initiale.

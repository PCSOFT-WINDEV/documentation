
## AppliInfoVersion (Fonction)

***En anglais : AppVersionInfo***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie des informations sur une version disponible sur le serveur d'installation.

**Remarque** : Cette fonction n'est utilisable qu'avec une installation réseau sur un réseau local. Elle ne peut être employée avec une installation par IP.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
Res est une chaîne = AppliListeVersion()
IV est un InfoVersion

AfficheInfo est une chaîne = [
Numéro = %1
Date = %2
Taille = %3 octets
Commentaire = %4
Actif = %5
Forcé = %6
Interdite = %7
]

POUR TOUTE CHAINE MaChaîne DE Res SEPAREE PAR RC
	IV = AppliInfoVersion(MaChaîne)
	Trace(ChaîneConstruit(AfficheInfo, ...
		IV.NuméroVersion, IV.Date, ...
		IV.Taille, IV.Commentaire, ...
		IV.Activé, IV.Forcé, IV.Interdit))
	Trace("===============================================")
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = AppliInfoVersion(<Version de l'application> [, <Chemin de l'application>])`
---

**`<Résultat> : (Variable de type InfoVersion)`**

Variable de type [InfoVersion](../WDLang1/1000018905.md) contenant les informations concernant la version spécifiée de l'application. 

**`<Version de l'application> : (Chaîne de caractères)`**

Numéro de la version de l'application pour laquelle les informations sont demandées. 

**Remarque** : Les numéros de version disponibles peuvent être connus par la fonction [AppliListeVersion](../WDLang1/1000018843.md). 

**`<Chemin de l'application> : (Chaîne de caractères optionnelle)`**

Chemin complet de l'exécutable à manipuler. Si ce paramètre n'est pas précisé, l'application manipulée est l'application en cours.

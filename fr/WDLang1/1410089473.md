
## dCopieImageChamp (Fonction)

***En anglais : dCopieImageChamp***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Copie l'image d'un champ d'une fenêtre WINDEV dans une variable de type [Image](../WDLang1/1000019650.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Image du champ de saisie du nom
MonImage est une Image = dCopieImageChamp(SAI_Nom)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = dCopieImageChamp(<Champ à copier>)`
---

**`<Résultat> : (Variable de type Image)`**

Variable de type [Image](../WDLang1/1000019650.md) contenant l'image du champ. 
En cas d'erreur, la variable [ErreurDétectée](../WDLang1/3087001.md) vaut <u><u><u><u>Vrai</u></u></u></u> et le message d'erreur correspondant peut être connu avec la fonction [ErreurInfo](../WDLang1/3013008.md).

**`<Champ à copier> : (Nom de champ)`**

Nom du champ dont l'image doit être copiée.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- L'image du champ prend en compte l'état du champ (actif, grisé, ...) au moment de l'appel de la fonction.

- L'image du champ sera récupérée même si le champ est invisible, hors fenêtre, ou s'il n'est pas dans le plan de la fenêtre affiché.

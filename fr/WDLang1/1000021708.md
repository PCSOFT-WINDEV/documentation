


## &lt;Onglet&gt;.SauveConfiguration (Fonction)

***En anglais : &lt;Tab&gt;.SaveConfiguration***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Sauvegarde la configuration des volets dynamiques d'un champ Onglet sous forme d'une chaîne de caractères. Cette configuration pourra être restaurée grâce à la fonction [&lt;Onglet&gt;.ChargeConfiguration](../WDLang1/1000021699.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Demande à l'utilisateur le fichier de sauvegarde
sFichier est une chaîne

// Ouvre le sélecteur de fichiers
sFichier = fSélecteur("", "", ...
	"Sélectionnez un fichier de configuration ...", ...
	"Configuration du tableau de bord" + TAB + "*.conf", "*.conf", ...
	fselCrée + fselExiste)
// Si un fichier est sélectionné
SI sFichier <> "" ALORS
	// Récupère la configuration courante dans une chaîne
	sConfiguration est chaîne = ONG_TableauDeBord.SauveConfiguration()
	// Sauve la configuration
	SI fSauveTexte(sFichier, sConfiguration) ALORS
		ToastAffiche("La configuration a été sauvée", toastCourt, cvMilieu, chCentre)
	SINON
		Erreur("Erreur en sauvegarde du fichier de configuration", ErreurInfo())
	FIN
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Champ Onglet>.SauveConfiguration()`
---

**`<Résultat> : (Chaîne de caractères)`**

Configuration sauvée sous forme de chaîne de caractères.

**`<Champ Onglet> : (Nom de champ)`**

Nom du champ Onglet à manipuler.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- La configuration des volets dynamiques d'un champ Onglet est constituée des éléments suivants :

	- les volets dynamiques existants avec leur rang. 

	- la fenêtre interne associée à chaque volet dynamique. 




- Pour sauver ou restaurer automatiquement la configuration utilisateur, vous pouvez : 

	- sous l'éditeur, cocher l'option "Mémoriser la configuration des volets" dans l'onglet "Détail" de la fenêtre de description du champ.

	- par programmation, utiliser la propriété [MémoriserLaConfiguration](../Proprietes/1000021383.md).

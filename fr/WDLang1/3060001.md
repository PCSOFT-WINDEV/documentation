


## Fonctions de gestion des polices
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions permettant de manipuler les polices (police des dessins, polices d'impression, ...) sont les suivantes :  



|   |   |
| --- | --- |
| [dPolice](../WDLang1/3029020.md) | Déclare la police de caractères qui sera utilisée par la fonction [dTexte](../WDLang1/3029032.md) : <br><br>- dans un champ Image, <br><br>- dans une variable de type [Image](../WDLang1/1000019650.md), <br><br>- dans une variable de type [WDPic](../WDLang1/1000023503.md) (sur le calque d'arrière-plan),<br><br>- dans une variable de type [picCalque](../WDLang1/1000024604.md). <br><br><br> |
| [gPolice](../WDLang1/3029047.md) | Construit une séquence de caractères décrivant la police d'affichage à utiliser pour le texte. |
| [gPoliceGras](../WDLang1/3029061.md) | Construit une séquence de caractères permettant d'indiquer si le texte suivant l'appel de cette fonction est en gras ou non. |
| [gPoliceItalique](../WDLang1/3029062.md) | Construit une séquence de caractères permettant d'indiquer si le texte suivant l'appel de cette fonction est en italique ou non. |
| [gPoliceSoulignée](../WDLang1/3029063.md) | Construit une séquence de caractères permettant d'indiquer si le texte suivant l'appel de cette fonction est souligné ou non. |
| [gPoliceTaille](../WDLang1/3029064.md) | Construit une séquence de caractères permettant d'indiquer la taille de la police d'affichage du texte d'une chaîne graphique. |
| [grPoliceEtiquette](../WDLang3/3042012.md) | Modifie la police utilisée pour les étiquettes de catégorie d'un graphe. |
| [grPoliceLégende](../WDLang3/3042037.md) | Modifie la police utilisée pour la légende d'un graphe. |
| [grPoliceTitre](../WDLang3/3042048.md) | Modifie la police utilisée pour le titre d'un graphe. |
| [grPoliceTitreAxe](../WDLang3/3042059.md) | Modifie la police utilisée pour les titres des axes d'un graphe. |
| [iCréePolice](../WDLang5/3046045.md) | Crée une nouvelle police d'impression. |
| [iPolice](../WDLang5/3046014.md) | Sélectionne la police par défaut. |
| [iPoliceDidot](../WDLang5/3046016.md) | Sélectionne l'unité à utiliser pour la hauteur des polices (créées avec la fonction [iCréePolice](../WDLang5/3046045.md)) : point DIDOT ou point PICA. |
| [iSelectPolice](../WDLang5/3046055.md) | Crée une nouvelle police d'impression depuis la fenêtre standard de sélection des polices de Windows... |
| [PoliceCrée](../WDLang1/3060002.md) | Crée une définition de police. |
| [PoliceListe](../WDLang1/3060003.md) | Renvoie la liste des polices valides installées sur le poste en cours (TrueType, Postscript, ...). |
| [PoliceSelect](../WDLang1/3060004.md) | Ouvre la fenêtre standard de sélection de polices de Windows en sélectionnant la police spécifiée. |

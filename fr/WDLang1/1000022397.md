
## NoteChargeTout (Fonction)

***En anglais : NoteLoadAll***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Importe et affiche les notes repositionnables de la fenêtre en cours précédemment sauvées par la fonction [NoteSauveTout](../WDLang1/1000022396.md). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Clic sur [Exporter mes notes]
NoteSauveTout()
SI ErreurDétectée = Vrai ALORS
	Erreur(ErreurInfo())
FIN

// Clic sur [Importer mes notes]
NoteChargeTout()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`NoteChargeTout([<Fichier>])`
---

**`<Fichier> : (Chaîne de caractères optionnelle)`**

Nom du fichier contenant les notes repositionnables à charger. 

Si ce paramètre n'est pas précisé, les notes chargées sont celles correspondant au fichier : &lt;Nom fenêtre&gt;.wdnote présent dans le répertoire des données. Le fichier recherché correspond à : 
```txt
fRepDonnées()\<MaFenetre..Nom>.wdnote
```




<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Cette fonction est équivalente à la FAA (Fonctionnalités Automatiques de vos Applications) d'importation des notes sans l'affichage du sélecteur de fichiers. Pour plus de détails, consultez [Note repositionnable (FAA)](../Editeurs/1000021820.md). 

- Les éléments restaurés sont : la position de la note, son contenu, sa couleur de fond et les caractéristiques de la police utilisée.




## dDécoupe (Fonction)

***En anglais : dCrop***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la partie découpée d'une image. Cette image est présente dans une variable de type [Image](../WDLang1/1000019650.md). 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonImageDécoupée est une Image
MonImage est une image
MonImage = "Image.png"

MonImageDécoupée = dDécoupe(MonImage, 10, 10, 100, 100)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = dDécoupe(<Image> , <X> , <Y> , <Largeur> , <Hauteur>)`
---

**`<Résultat> : (Variable de type Image)`**

Variable de type [Image](../WDLang1/1000019650.md) contenant l'image découpée. 

**`<Image> : (Variable de type Image)`**

Nom de la variable de type [Image](../WDLang1/1000019650.md) correspondant à l'image à découper. 

**`<X> : (Entier)`**

Abscisse (en pixels) du coin haut gauche de la zone à découper.

**`<Y> : (Entier)`**

Ordonnée (en pixels) du coin haut gauche de la zone à découper.

**`<Largeur> : (Entier)`**

Largeur de la zone à découper. 

**`<Hauteur> : (Entier)`**

Hauteur de la zone à découper. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Si la zone à découper est invalide (zone plus grande que l'image source ou zone vide), cette fonction provoque une erreur non fatale.

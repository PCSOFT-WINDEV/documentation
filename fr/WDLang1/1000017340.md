


## CalendrierSelectMoins (Fonction)

***En anglais : CalendarSelectMinus***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Désélectionne une date dans un champ Calendrier multisélection. 

 



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Désélectionne le jour correspondant à aujourd'hui
CalendrierSelectMoins(CAL_Calendrier1, DateDuJour())
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`CalendrierSelectMoins(<Champ Calendrier> , <Date à supprimer>)`
---

**`<Champ Calendrier> : (Nom de champ)`**

Nom du champ Calendrier manipulé. Ce champ doit avoir l'option "Multisélection" cochée (onglet "Détail" de la description du champ). Si cette option n'est pas cochée, la fonction est sans effet.

**`<Date à supprimer> : (Chaîne de caractères)`**

Date au format AAAAMMJJ à supprimer de la sélection. Si cette date n'est pas sélectionnée, la fonction est sans effet.

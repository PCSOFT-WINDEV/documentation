
## CouleurLuminosité (Fonction)

***En anglais : ColorLightness***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Calcule la luminosité d'une couleur.






<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SelectCouleur(MaCouleur)
Info("Teinte : " + CouleurTeinte(MaCouleur) + "°", ...
	"Luminosité : " + CouleurLuminosité(MaCouleur) + "%", ...
	"Saturation : " + CouleurSaturation(MaCouleur) + "%")
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = CouleurLuminosité(<Couleur>)`
---

**`<Résultat> : (Entier)`**

Valeur de la luminosité de la couleur spécifiée (pourcentage compris entre 0 et 100).

**`<Couleur> : (Entier)`**

Couleur à manipuler. Cette couleur peut correspondre à :

- une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)),

- [une couleur prédéfinie du WLangage](../WDLang5/3010002.md).




## LecteurPDFRecherchePrécédent (Fonction)

***En anglais : PDFReaderSeekPrevious***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Positionne sur le résultat précédent d'une recherche initiée par la fonction [LecteurPDFRecherchePremier](../WDLang1/1000023409.md) dans le contenu d'un champ Lecteur PDF. 

Remarque : La fonction [LecteurPDFRechercheSuivant](../WDLang1/1000023410.md) permet de trouver l'élément suivant correspondant à la recherche. 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Recherche un texte dans un champ Lecteur PDF et affiche le texte
LecteurPDFRecherchePremier(PDF_MonLecteurPDF, "société")
// Recherche l'élément suivant
SI LecteurPDFRechercheSuivant(PDF_MonLecteurPDF) = Faux ALORS
	Info("Recherche terminée")
FIN

// Recherche l'élément précédent
LecteurPDFRecherchePrécédent(PDF_MonLecteurPDF)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = LecteurPDFRecherchePrécédent(<Champ Lecteur PDF>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le texte spécifié dans la fonction [LecteurPDFRecherchePremier](../WDLang1/1000023409.md) a été trouvé dans le document en cours dans le champ Lecteur PDF, 

- <u><u><u><u>Faux</u></u></u></u> si le texte n'a pas été trouvé. 




**`<Champ Lecteur PDF> : (Nom de champ)`**

Nom du champ Lecteur PDF à manipuler.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Le texte et les options de recherche ont été spécifiés dans la fonction [LecteurPDFRecherchePremier](../WDLang1/1000023409.md).

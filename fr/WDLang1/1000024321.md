
## CertificatDeConfianceRAZ (Fonction)

***En anglais : TrustedCertificateReset***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Retire tous les certificats ajoutés par la fonction [CertificatDeConfianceAjoute](../WDLang1/1000024320.md). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
CertificatDeConfianceAjoute("MonCA.pem")
// code
CertificatDeConfianceRAZ()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = CertificatDeConfianceRAZ()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si les certificats de confiance (ajoutés par la fonction [CertificatDeConfianceAjoute](../WDLang1/1000024320.md)) ont été retirés, 

- <u><u><u><u>Faux</u></u></u></u> en cas d'erreur.




## &lt;Type Chaîne Ansi&gt;.AnsiVersOem (Fonction)

***En anglais : &lt;Ansi string&gt;.AnsiToOem***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Convertit une chaîne de caractères au format ANSI (Windows) en une chaîne de caractères au format OEM (DOS). Les caractères accentués sont différents dans les deux normes. La conversion consiste à modifier les caractères accentués.

**Rappel** :

- ANSI est la norme des caractères au standard Windows.

- OEM est la norme des caractères au standard DOS (ou des fenêtres "console" de Windows).





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
Res est une chaîne
ChaîneAConvertir est une chaîne = "Crème brûlée"
Res = ChaîneAConvertir.AnsiVersOem()
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Chaîne à convertir>.AnsiVersOem()`
---

**`<Résultat> : (Chaîne de caractères Ansi)`**

Chaîne de caractères au format OEM.

**`<Chaîne à convertir> : (Chaîne de caractères Ansi)`**

Chaîne de caractères au format ANSI à convertir.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- La fonction **&lt;Type Chaîne Ansi&gt;.AnsiVersOem** ne convertit pas les caractères ayant un code ASCII compris entre 0 et 31 (caractères de contrôle).

- **Attention** : Ne pas utiliser la fonction **&lt;Type Chaîne Ansi&gt;.AnsiVersOem** avec une chaîne de caractères contenant des caractères spécifiques OEM (caractères accentués) : ces caractères seront transformés et le résultat sera incorrect.


## AscenseurPosition (Fonction)

***En anglais : ScrollbarPosition***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de gérer la position des ascenseurs : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Modifie ou récupère la position d'un des ascenseurs (horizontal ou vertical) d'un champ ou d'une fenêtre dans une application WINDEV ou WINDEV Mobile. 







<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 
```wl
// Synchronisation des ascenseurs de deux champs Table
nPos = AscenseurPosition(TABLE_MaTable, ascVert)
AscenseurPosition(TABLE_MaTable2, ascVert, nPos)
```





<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître la position de l'ascenseur

`<Résultat> = AscenseurPosition(<Elément manipulé> [, <Orientation>])`
---

**`<Résultat> : (Entier)`**



- Position de la cage de l'ascenseur. 

- 0 si l'ascenseur n'existe pas ou si l'ascenseur est grisé.  




**`<Elément manipulé> : (Nom de champ, de fenêtre ou de page)`**



- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Nom du champ manipulé. 
	

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Nom de la fenêtre manipulée.




**`<Orientation> : (Constante optionnelle de type Entier)`**

Orientation de l'ascenseur pour lequel la position doit être récupérée : 


|   |   |
| --- | --- |
| *ascHorz* | La position est calculée pour l'ascenseur horizontal du champ. |
| *ascVert*<br>(Valeur par défaut) | La position est calculée pour l'ascenseur vertical du champ. |




<a name="SYNTAXE2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) 
### Modifier la position de l'ascenseur

`<Résultat> = AscenseurPosition(<Champ manipulé> , <Orientation> , <Nouvelle position>)`
---

**`<Résultat> : (Entier)`**

Nouvelle position de la cage de l'ascenseur.  

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ manipulé.


**`<Orientation> : (Constante optionnelle de type Entier)`**

Orientation de l'ascenseur pour lequel la position doit être modifiée : 


|   |   |
| --- | --- |
| *ascHorz* | La position est modifiée pour l'ascenseur horizontal du champ. |
| *ascVert* | La position est modifiée pour l'ascenseur vertical du champ. |



**`<Nouvelle position> : (Entier optionnel)`**

Nouvelle position de l'ascenseur. La position de l'ascenseur commence à 1. 

Si la valeur spécifiée n'est pas dans les bornes spécifiées pour l'ascenseur du champ, une position correcte est forcée. 

Si l'ascenseur n'existe pas ou est grisé, la nouvelle position n'est pas prise en compte : la fonction ne fait rien.


## RegistreExiste (Fonction)

***En anglais : RegistryExist***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Teste l'existence d'une clé de la base de registre de Windows.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Teste l'existence de la clé "HKEY_LOCAL_MACHINE\SOFTWARE\MonAppli"
ResExiste = RegistreExiste("HKEY_LOCAL_MACHINE\SOFTWARE\MonAppli")
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Tester l'existence d'une clé ou d'une valeur de clé

`<Résultat> = RegistreExiste([<Mode d'accès>, ] <Chemin de la clé> [, <Nom de la valeur>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la clé ou la valeur spécifiée existe, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Mode d'accès> : (Constante de type Entier)`**

Mode d'accès à la base de registre :



|   |   |
| --- | --- |
| *registreMode32* | Mode d'accès forcé pour accéder à la base de registre comme un programme 32 bits. |
| *registreMode64* | Mode d'accès forcé pour accéder à la base de registre comme un programme 64 bits. |
| *registreModeAuto*<br>(Valeur par défaut) | Mode d'accès à la base de registre automatique : <br><br>- une application 32 bits exécutée sur un système 32 bits manipule la base de registre comme un programme 32 bits.<br><br>- une application 32 bits exécutée sur un système 64 bits manipule la base de registre à partir de la branche suivante :<br>	<br><pre><code>HKEY_LOCAL_MACHINE\SOFTWARE\Wow6432Node</code></pre><br><br><br>- une application 64 bits exécutée sur un système 64 bits manipule la base de registre comme un programme 64 bits.<br><br><br> |



**`<Chemin de la clé> : (Chaîne de caractères)`**

Chemin de la clé à tester.

**`<Nom de la valeur> : (Chaîne de caractères optionnelle)`**

Nom de la valeur à tester. Si ce paramètre n'est pas spécifié, la valeur testée sera la valeur "&lt;Défaut&gt;". Pour tester la valeur "&lt;Défaut&gt;", &lt;Nom de la valeur&gt; doit correspondre à "0".  


<a name="SYNTAXE2"></a>

### Tester l'existence d'une valeur de clé identifiée par son indice

`<Résultat> = RegistreExiste([<Mode d'accès>, ] <Chemin de la clé> , <Indice de la valeur>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la valeur spécifiée existe, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Mode d'accès> : (Constante de type Entier)`**

Mode d'accès à la base de registre :



|   |   |
| --- | --- |
| *registreMode32* | Mode d'accès forcé pour accéder à la base de registre comme un programme 32 bits. |
| *registreMode64* | Mode d'accès forcé pour accéder à la base de registre comme un programme 64 bits. |
| *registreModeAuto*<br>(Valeur par défaut) | Mode d'accès à la base de registre automatique : <br><br>- une application 32 bits exécutée sur un système 32 bits manipule la base de registre comme un programme 32 bits.<br><br>- une application 32 bits exécutée sur un système 64 bits manipule la base de registre à partir de la branche suivante :<br>	<br><pre><code>HKEY_LOCAL_MACHINE\SOFTWARE\Wow6432Node</code></pre><br><br><br>- une application 64 bits exécutée sur un système 64 bits manipule la base de registre comme un programme 64 bits.<br><br><br> |



**`<Chemin de la clé> : (Chaîne de caractères)`**

Chemin de la clé à tester.

**`<Indice de la valeur> : (Entier)`**

Indice de la valeur à tester. Si ce paramètre n'est pas spécifié, la valeur testée sera la valeur "&lt;Défaut&gt;". Pour tester la valeur "&lt;Défaut&gt;", &lt;Indice de la valeur&gt; doit correspondre à "0".

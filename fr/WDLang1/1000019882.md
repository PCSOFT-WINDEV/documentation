
## PremierJourDeLaSemaine (Fonction)

***En anglais : FirstDayOfWeek***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la date du premier jour de la semaine (c'est-à-dire le Lundi). 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Clic sur le bouton "BTN_debut_semaine"
SAI_DATE_DEBUT = PremierJourDeLaSemaine(SAI_DATE_DEBUT)
```
<a name="Exemple2"></a>

```wl
NbJourTravaille = DateDifférence(SAI_Date, PremierJourDeLaSemaine(SAI_Date), SAI_Date)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = PremierJourDeLaSemaine(<Date>)`
---

**`<Résultat> : (Chaîne de caractères ou variable de type Date)`**

Date correspondant au premier jour de la semaine demandée (au format AAAAMMJJ). 

**`<Date> : (Chaîne de caractères ou variable de type Date)`**

Date pour laquelle le premier jour de la semaine doit être calculé. Cette date est au format AAAAMMJJ.

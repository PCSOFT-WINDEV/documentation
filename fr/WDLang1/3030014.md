
## Drag and Drop depuis l'explorateur 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000127"></a>
Le "Drag and Drop" depuis l'explorateur consiste à sélectionner un ou plusieurs fichiers dans l'explorateur de Windows et à les transférer dans un champ d'une fenêtre WINDEV.

Par exemple, l'exemple "WD Drag and Drop" permet ainsi d'afficher les fichiers image "droppés" depuis l'explorateur.

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=drag_drop_explorateur.gif)




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Principe
<a name="principe_ELTTEXTE000151"></a>
Pour permettre un "Drag and Drop" depuis l'explorateur vers une application WINDEV, les différentes étapes sont les suivantes :

1. Activez la possibilité de réaliser un "Drag and Drop" depuis l'explorateur (fonction [ExplorerAccepte](../WDLang1/3030015.md)).

2. Pour traiter l'action effectuée par l'utilisateur, associez une procédure du WLangage au lâcher du fichier dans la fenêtre (fonction [Evénement](../WDLang1/3015011.md)).

3. Dans la procédure appelée à chaque lâcher de fichier dans la fenêtre, récupérez les caractéristiques des fichiers "droppés" grâce à la fonction [ExplorerRécupère](../WDLang1/3030009.md).






<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Fonctions spécifiques au "Drag and Drop" depuis l'explorateur
<a name="fonctions_specifiques_drag_and_drop_depuis_explorateur_ELTTEXTE000175"></a>
Les fonctions suivantes permettent de gérer le "Drag and Drop" depuis l'explorateur Windows vers une fenêtre d'une application WINDEV.



|   |   |
| --- | --- |
| [ExplorerAccepte](../WDLang1/3030015.md) | Active ou désactive la possibilité de faire du "Drag and Drop" (Glisser/Déplacer) depuis l'explorateur Windows vers une fenêtre ou un champ WINDEV. |
| [ExplorerRécupère](../WDLang1/3030009.md) | Récupère le nombre et le nom des fichiers "droppés" depuis l'explorateur. |





<a name="NOTE4"></a>
<a name="NOTE4_1"></a>


## Programmation
<a name="programmation_ELTTEXTE000215"></a>


### Activation de la gestion du "Drag and Drop" depuis l'explorateur vers la fenêtre
<a name="activation_gestion_drag_and_drop_depuis_explorateur_vers_fenetre_ELTPARAGRAPHE000052"></a>Pour activer la possibilité de "Drag and Drop" depuis l'explorateur vers une fenêtre, il suffit de déclarer dans le code d'initialisation de la fenêtre :

- la prise en compte du "Drag and Drop",

- l'événement permettant de lier une procédure WLangage spécifique (RecupFichiers dans notre exemple) à l'action du lâcher (événement Windows WM_DROPFILES égal à 563). Cette procédure est une procédure locale à la fenêtre.
	
	```wl
	ExplorerAccepte(Vrai, "")
	Evénement("RecupFichiers", "*.", 563)
	```






<a name="NOTE4_2"></a>


### Traitement des fichiers "droppés"
<a name="traitement_des_fichiers_droppes_ELTPARAGRAPHE000065"></a>**Dans la procédure appelée à chaque "lâcher" de fichiers dans la fenêtre, il est possible d'obtenir des renseignements et de traiter les fichiers concernés.**

Dans notre exemple, il est ainsi possible grâce à la fonction [ExplorerRécupère](../WDLang1/3030009.md) de :

- connaître le nombre de fichiers "lâchés" depuis l'explorateur,

- récupérer le nom et le chemin de chaque fichier et de l'afficher dans une nouvelle fenêtre.





```wl
// Nombre de fichiers posés dans l'application par Drag and Drop
nbFic est un entier = ExplorerRécupère(_EVE.wparam)
sNomFic est une chaîne
// Récupération de tous les fichiers
POUR i = 1 A nbFic
	sNomFic = ExplorerRécupère(_EVE.wparam, i)
	OuvreDocument(sNomFic)
FIN
```


## docTableDesMatières (Type de variable)

***En anglais : docTableOfContents***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **docTableDesMatières** permet de définir toutes les options de génération d'une table des matières pour un document de type Traitement de texte. Les caractéristiques de cette table des matières peuvent être définies et modifiées à l'aide de différentes propriétés WLangage. 

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
doc est un Document
...
// Définition de la table des matières
tdm est un docTableDesMatières
tdm.AvecNuméroDePage = Vrai

// Les paragraphes utilisant le style par défaut de niveau 1 (Titre 1) 
// sont utilisés pour les entrées de niveau 1 de la table des matières
tdm.StylesParNiveau[1] = styleTDMDéfaut1

// Les paragraphes utilisant le style par défaut de niveau 1 (Titre 2) 
// et les paragraphes utilisant le style "MonStylePerso"
// sont utilisés pour les entrées de niveau 2 de la table des matières
tdm.StylesParNiveau[2] = styleTDMDéfaut2 + RC + "MonStylePerso"

// Les paragraphes utilisant le style perso "MonStylePerso3"
// sont utilisés pour les entrées de niveau 3 de la table des matières
tdm.StylesParNiveau[3] = "MonStylePerso3"

// Insère la table des matières au début du document
fDébutDoc est docFragment = doc[1 SUR 0]
DocInsère(fDébutDoc, tdm)
```

## Remarques
<a name="NOTE0_1"></a>


### Propriétés spécifiques à la description des variables de type docTableDesMatières
<a name="proprietes_specifiques_description_des_variables_type_doctabledesmatieres_ELTPARAGRAPHE000029"></a>Les propriétés suivantes peuvent être utilisées pour manipuler une table des matières :


| Nom de la propriété | Type manipulé | Effet |
| --- | --- | --- |
| AvecNuméroDePage | Booléen | <br><br>- <u><u><u><u>Vrai</u></u></u></u> si le numéro de page doit être affiché sur chaque ligne de la table des matières, <br><br>- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. <br><br><br> |
| StylesParNiveau | Tableau de chaînes | Permet de spécifier le ou les styles à utiliser lors de la génération du niveau correspondant de la table des matières.<br><br>Si plusieurs styles sont utilisés, ils doivent être séparés par RC (Retour chariot). <br><br>Il est possible d'utiliser : <br><br>- un ou des styles personnalisés, <br><br>- les constantes correspondant aux styles par défaut : <br><br>	- *styleTDMDéfaut1* : Style de paragraphe par défaut pour le niveau 1 de la table des matières (Titre 1). <br><br>	- *styleTDMDéfaut2* : Style de paragraphe par défaut pour le niveau 2 de la table des matières (Titre 2). <br><br>	- *styleTDMDéfaut3* : Style de paragraphe par défaut pour le niveau 3 de la table des matières (Titre 3). <br><br>	- *styleTDMDéfaut4* : Style de paragraphe par défaut pour le niveau 4 de la table des matières (Titre 4). <br><br>	- *styleTDMDéfaut5* : Style de paragraphe par défaut pour le niveau 5 de la table des matières (Titre 5). <br><br>	- *styleTDMDéfaut6* : Style de paragraphe par défaut pour le niveau 6 de la table des matières (Titre 6). <br><br>	- *styleTDMDéfaut7* : Style de paragraphe par défaut pour le niveau 7 de la table des matières (Titre 7). <br><br>	- *styleTDMDéfaut8* : Style de paragraphe par défaut pour le niveau 8 de la table des matières (Titre 8). <br><br>	- *styleTDMDéfaut9* : Style de paragraphe par défaut pour le niveau 9 de la table des matières (Titre 9). <br><br><br><br><br><br> |


<a name="NOTE0_2"></a>


### Divers
<a name="divers_ELTPARAGRAPHE000080"></a>Une table des matières peut être ajoutée dans le document grâce à la fonction [DocInsère](../WDLang1/1000022732.md).

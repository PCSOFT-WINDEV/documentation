
## PiloteSélecteur (Fonction)

***En anglais : EmulateRadioButton***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Pilote la sélection d'une option dans un champ Sélecteur.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Sélection de la 5ème option du sélecteur
PiloteSélecteur(SEL_Selecteur, 5)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`PiloteSélecteur(<Champ> , <Numéro de l'option>)`
---

**`<Champ> : (Chaîne de caractères)`**

Nom du champ de type Sélecteur.

**`<Numéro de l'option> : (Entier)`**

Indice de l'option à sélectionner dans le champ.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Cette fonction doit être utilisée uniquement dans le code des tests automatiques.

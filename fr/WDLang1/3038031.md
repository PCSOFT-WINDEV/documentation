
## FenIntLarg (Fonction)

***En anglais : WinInWidth***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la largeur interne d'une fenêtre (sans le cadre).
<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=FenExtInt.gif)

<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Largeur interne de la fenêtre "FEN_SaisieClient"
ResLargeurInterne = FenIntLarg(FEN_SaisieClient)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = FenIntLarg([<Fenêtre manipulée>])`
---

**`<Résultat> : (Entier)`**

Largeur interne de la fenêtre (en pixels).

**`<Fenêtre manipulée> : (Nom de fenêtre optionnel ou chaîne de caractères optionnelle)`**

Nom ou alias de la fenêtre à manipuler. Si ce paramètre n'est pas précisé ou correspond à une chaîne vide (""), la largeur interne de la fenêtre en cours est retournée.


## Canada.VérifieNAS (Fonction)

***En anglais : Canada.CheckSIN***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Teste la validité d'un numéro NAS (numéro d'assurance sociale du Canada). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Test d'une variable
MonNuméroNAS est une chaîne = "046454286"
Canada.VérifieNAS(MonNuméroNAS)		// renvoie Vrai

// Test d'un champ
SAI_NuméroNAS = "ABC"
Canada.VérifieNAS(SAI_NuméroNAS)	// renvoie Faux
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = Canada.VérifieNAS(<Numéro à vérifier>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'élément spécifié contient un numéro NAS valide.

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Numéro à vérifier> : (Nom de champ ou variable de type chaîne)`**

Numéro NAS à vérifier. Ce paramètre peut correspondre :

- au nom d'une variable de type chaîne.

- au nom d'un champ.


## SysRésolutionEcran (Fonction)

***En anglais : SysScreenResolution***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la résolution d'un écran.  
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affichage de la résolution de l'écran principal
ChaîneRes est une chaîne
ChaîneRes = SysRésolutionEcran()
Trace("Résolution Largeur =" + ExtraitChaîne(ChaîneRes, 1) + RC + ...
	"Résolution Hauteur =" + ExtraitChaîne(ChaîneRes, 2) + RC + ...
	"Nb bits par pixel = " + ExtraitChaîne(ChaîneRes, 3) + RC + ...
	"Taux de rafraîchissement =" + ExtraitChaîne(ChaîneRes, 4))
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = SysRésolutionEcran([<Nom de l'écran>])`
---

**`<Résultat> : (Chaîne de caractères)`**



- Résolution de l'écran donnée sous la forme :
	
	```txt
	<Résolution Largeur> + TAB + <Résolution Hauteur> + TAB + 
	<Nombre de bits par pixel> + TAB + <Taux de Rafraîchissement>
	```


- Chaîne vide ("") en cas d'erreur. La fonction [ErreurInfo](../WDLang1/3013008.md) permet d'obtenir plus de renseignements sur le problème rencontré.




**`<Nom de l'écran> : (Chaîne de caractères optionnelle)`**

Nom de l'écran pour lequel la résolution doit être connue.

Si ce paramètre n'est pas spécifié, la résolution de l'écran principal est renvoyée. Si le nom spécifié n'existe pas, &lt;Résultat&gt; correspond à une chaîne vide ("").

**Remarque** : La liste des noms des écrans peut être obtenue par la fonction [SysListeEcran](../WDLang1/1000017014.md).

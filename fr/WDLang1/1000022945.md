
## docPropriété (Type de variable)

***En anglais : docProperty***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **docPropriété** permet de définir toutes les propriétés d'un document de type Traitement de texte. Les caractéristiques de ces propriétés peuvent être définies et modifiées à l'aide de différentes propriétés WLangage. 

**ATTENTION : Ce type ne doit pas être utilisé directement.** Il doit être utilisé uniquement via les variables de type [Document](../WDLang1/1000022461.md). 

Remarque : Les propriétés du document sont utilisées pour certains champs calculés pouvant être inclus dans le document.






<a name="NOTE0"></a>

## Remarques
<a name="NOTE0_1"></a>


### Propriétés spécifiques à la description des variables de type docPropriété
<a name="proprietes_specifiques_description_des_variables_type_docpropriete_ELTPARAGRAPHE000031"></a>Les propriétés suivantes peuvent être utilisées pour manipuler une variable de type **docPropriété** :

| Nom de la propriété | Type manipulé | Effet |
| --- | --- | --- |
| Auteur | Chaîne de caractères | Auteur du document. |
| Catégorie | Chaîne de caractères | Catégorie du document. |
| Commentaire | Chaîne de caractères | Commentaire associé au document. |
| DateCréation | Date et heure | Date et heure de création du document. <br><br>**Cette propriété est disponible en lecture seulement.**<br><br> |
| DateDernièreModification | Date et heure | Date et heure de dernière modification du document. <br><br>**Cette propriété est disponible en lecture seulement.** |
| EtatContenu | Chaîne de caractères | Etat du document : brouillon, revu, ... |
| MotClé | Chaîne de caractères | Mot-clé associé au document. |
| Révision | Chaîne de caractères | Numéro de révisions du document. <br><br>**Cette propriété est disponible en lecture seulement.** |
| Société | Chaîne de caractères | Société ayant créé le document. <br><br> |
| Sujet | Chaîne de caractères | Sujet du document. |
| Titre | Chaîne de caractères | Titre du document. |

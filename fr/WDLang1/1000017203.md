


## MultimédiaArrêt (Fonction)

***En anglais : MultimediaStop***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Arrête la lecture en cours d'un fichier multimédia dans un champ Multimédia. 

Le fichier multimédia reste chargé en mémoire. Pour libérer la mémoire, utilisez la fonction [MultimédiaFerme](../WDLang1/1000017205.md).








<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) 
```wl
MonFichier est une chaîne = "C:\Ma musique\mysong.mp3"
MultimédiaJoue(MM_Multimédia1, MonFichier)
MultimédiaArrêt(MM_Multimédia1)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = MultimédiaArrêt(<Champ Multimédia>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'affichage dans le champ Multimédia est correctement arrêté, 

- <u><u><u><u>Faux</u></u></u></u> en cas d'erreur. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Champ Multimédia> : (Nom de champ)`**

Nom du champ Multimédia à manipuler.

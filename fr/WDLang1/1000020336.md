
## ToastSupprimeTout (Fonction)

***En anglais : ToastDeleteAll***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime tous les toasts affichés par la fonction [ToastAffiche](../WDLang1/1000019358.md). Si aucun toast n'est affiché, la fonction est sans effet.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Fermeture de la fenêtre : tous les toasts sont effacés
ToastSupprimeTout()
```

## Syntaxe




## &lt;Tableau de bord&gt;.Occurrence (Fonction)

***En anglais : &lt;Dashboard&gt;.Count***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le nombre de Widgets présents dans un champ Tableau de bord. Les Widgets pris en compte sont : 

- les Widgets déclarés sous l'éditeur de fenêtres ou l'éditeur de pages, 

- les Widgets ajoutés par la fonction [&lt;Tableau de bord&gt;.AjouteWidget](../WDLang1/1000022502.md).







<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affiche le nombre de widgets affichés et le nombre total de widgets
InfoConstruit("Le tableau de bord contient %1 widget(s) au total dont %2 affiché(s)", ...
	TDB_TableauDeBord.Occurrence(toTotal), 
	TDB_TableauDeBord.Occurrence(toAffichée))
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Champ Tableau de bord>.Occurrence(<Type de Widget>)`
---

**`<Résultat> : (Entier)`**

Nombre de Widgets présents dans le champ Tableau de Bord.

**`<Champ Tableau de bord> : (Nom de champ)`**

Nom du champ Tableau de bord à manipuler.

**`<Type de Widget> : (Constante de type Entier)`**

Type de Widget à prendre en compte :


|   |   |
| --- | --- |
| *toAffichée* | Seuls les Widgets affichés sont pris en compte. |
| *toTotal* | Tous les Widgets sont pris en compte (Widgets affichés ou non). |

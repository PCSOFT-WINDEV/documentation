


## DocAjoute (Fonction)

***En anglais : DocAdd***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ajoute : 

- à la fin d'un document de type Traitement de texte : un élément, un document, un paragraphe, un texte, une image ou un fragment.

- à la fin d'un fragment : un élément, un texte ou une image.



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ajoute une image
// Sélection du fichier image 
sFichier est chaîne  
sFichier = fSélecteurImage(fRepEnCours(), "", "Ajouter une image",
	"JPEG (*.JPG)" + TAB + "*.jpg" + RC + "PNG (*.PNG)" + TAB + 
	"*.png" + RC + "Tous les fichiers (*.*)" + TAB + "*.*")
	
SI sFichier = "" ALORS
	RETOUR
FIN

// Le fichier image est transféré dans une variable de type Image 
img est une Image = sFichier

// Ajout de l'image en fin de document 
DocAjoute(TT_SansNom1, img)
```
<a name="Exemple2"></a>

```wl
// Ajoute un saut de page
DocAjoute(TT_SansNom1, Caract(12))
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Ajouter à la fin d'un document de type Traitement de texte

`<Résultat> = DocAjoute(<Document> , <Objet à ajouter>)`
---

**`<Résultat> : (Variable de type docFragment)`**

Variable de type [docFragment](../WDLang1/1000022482.md) contenant le fragment ajouté. 

**`<Document> : (Nom de champ ou variable de type Document)`**

Document à manipuler. Ce document correspond : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) au nom d'un champ Traitement de texte. 

- à une variable de type [Document](../WDLang1/1000022461.md).




**`<Objet à ajouter> : (Type de l'objet ajouté)`**

Objet à ajouter : 

- Elément : Nom de la variable de type [docElément](../WDLang1/1000022484.md) correspondant à l'élément à ajouter.

- Document : Nom de la variable de type [Document](../WDLang1/1000022461.md) correspondant au document à ajouter.

- Paragraphe : Nom de la variable de type [docParagraphe](../WDLang1/1000022483.md) correspondant au paragraphe à ajouter. Ce paragraphe doit avoir été récupéré depuis un document, il ne peut être créé en partant de rien car le tableau des éléments d'un paragraphe est en lecture seule.

- Texte sous forme de chaîne de caractères : Texte à ajouter.
	Il est possible d'utiliser les constantes suivantes dans le texte à ajouter pour gérer les différents sauts possibles : 
	


|   |   |
| --- | --- |
| *docSautDeColonne* | Ajoute un saut de colonne dans une section multicolonne. Si la section n'est pas multicolonne, un saut de page est ajouté. |
| *docSautDeLigne* | Ajoute un saut de ligne. |
| *docSautDePage* | Ajoute un saut de page. |
| *docSautDeParagraphe* | Ajoute un saut de paragraphe. |



- Image : Nom de la variable de type [Image](../WDLang1/1000019650.md) à ajouter.

- Fragment : Nom de la variable de type [docFragment](../WDLang1/1000022482.md) à ajouter.





<a name="SYNTAXE2"></a>

### Ajouter à la fin d'un fragment

`<Résultat> = DocAjoute(<Fragment> , <Objet à ajouter>)`
---

**`<Résultat> : (Variable de type docFragment)`**

Variable de type [docFragment](../WDLang1/1000022482.md) contenant le fragment ajouté. 

**`<Fragment> : (Variable de type docFragment)`**

Nom de la variable de type [docFragment](../WDLang1/1000022482.md) à manipuler.

**`<Objet à ajouter> : (Type de l'objet ajouté)`**

Objet à ajouter : 

- Elément : Nom de la variable de type [docElément](../WDLang1/1000022484.md) correspondant à l'élément à ajouter.

- Texte sous forme de chaîne de caractères : Texte à ajouter.
	Il est possible d'utiliser les constantes suivantes dans le texte à ajouter pour gérer les différents sauts possibles : 
	


|   |   |
| --- | --- |
| *docSautDeColonne* | Ajoute un saut de colonne dans une section multicolonne. Si la section n'est pas multicolonne, un saut de page est ajouté. |
| *docSautDeLigne* | Ajoute un saut de ligne. |
| *docSautDePage* | Ajoute un saut de page. |
| *docSautDeParagraphe* | Ajoute un saut de paragraphe. |



- Image : Nom de la variable de type [Image](../WDLang1/1000019650.md) à ajouter.

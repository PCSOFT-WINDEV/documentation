


## Fonctions de gestion des clés d'activation
			



Les fonctions de gestion des clés d'activation sont les suivantes :



|   |   |
| --- | --- |
| [CléCalculeCléActivation](../WDLang1/1000018849.md) | Calcule la clé d'activation de l'application à partir de la clé initiale. |
| [CléCompareClé](../WDLang1/1000018850.md) | Compare une clé initiale avec une clé d'activation. |
| [CléGénèreCléInitiale](../WDLang1/1000018848.md) | Génère la clé initiale (numéro de licence unique) pour le système d'activation de l'application. |
| [CléRécupèreIdentifiant](../WDLang1/1000018851.md) | Récupère l'identifiant ayant servi à générer une clé initiale. |

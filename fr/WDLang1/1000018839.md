
## dbgEcritCommentaireAudit (Fonction)

***En anglais : dbgWriteCommentAudit***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ecrit un commentaire dans l'audit dynamique en cours. Si l'audit n'est pas actif, la fonction **dbgEcritCommentaireAudit** est sans effet. 

**Remarque** : L'audit dynamique peut être activé grâce à la fonction [dbgActiveAudit](../WDLang1/1000018835.md).




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Active l'audit dynamique
dbgActiveAudit()

// Place un commentaire pour rappeler que si la fonction fSupprime génère une erreur, 
// cela n'a pas d'importance
dbgEcritCommentaireAudit("L'appel à fSupprime peut générer une erreur " + ...
	 "quand le fichier n'existe pas. Cette erreur est sans intérêt.")
fSupprime("existepas.txt")
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`dbgEcritCommentaireAudit(<Commentaire>)`
---

**`<Commentaire> : (Chaîne de caractères)`**

Commentaire à écrire dans le fichier d'audit.

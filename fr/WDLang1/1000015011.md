


## Fonctions Xaml (syntaxe préfixée)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions (syntaxe préfixée) permettant de gérer les champs Xaml sont les suivantes : 



|   |   |
| --- | --- |
| [&lt;Xaml&gt;.Evénement](../WDLang1/1000015009.md) | Associe une procédure écrite en WLangage à un événement d'un champ Xaml. |
| [&lt;Xaml&gt;.Propriété](../WDLang1/1000015010.md) | Permet de connaître ou de modifier la valeur d'une propriété .Net d'un champ Xaml ou d'un champ Carrousel. |

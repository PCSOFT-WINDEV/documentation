
## ChampExécuteModification (Fonction)

***En anglais : ControlExecuteModification***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Considère que le champ a été modifié par l'utilisateur (la propriété [Modifié](../Proprietes/2510080.md) passe à <u><u><u>Vrai</u></u></u> pour le champ et si nécessaire pour la fenêtre) et exécute l'événement WLangage "A chaque modification" associé au champ. 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
ChampExécuteModification(SAI_Saisie)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`ChampExécuteModification(<Champ manipulé>)`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler.

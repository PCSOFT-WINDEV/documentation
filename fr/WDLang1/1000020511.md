
## DécimalVersSexagésimal (Fonction)

***En anglais : DecimalToSexagesimal***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie l'angle sexagésimal (en base 60) correspondant à un angle décimal.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Cherche l'angle sexagésimal correspondant à un angle de 128.91°
rAngle est un réel = 128.91
sAngleSexagésimal est une chaîne = DécimalVersSexagésimal(rAngle)
Info("Un angle de " + rAngle + "° correspond à : " + sAngleSexagésimal)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = DecimalVersSexagésimal(<Angle>)`
---

**`<Résultat> : (Chaîne de caractères)`**

Résultat de la conversion au format sexagésimal. Cette chaîne est de la forme : 
```txt
XXX°XX'XX''
```


**`<Angle> : (Réel)`**

Angle au format décimal à convertir.

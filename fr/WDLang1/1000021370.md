


## OngletChargeConfiguration (Fonction)

***En anglais : TabLoadConfiguration***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Restaure la configuration des volets dynamiques d'un champ Onglet. Cette configuration a été précédemment sauvée sous forme d'une chaîne de caractères avec la fonction [OngletSauveConfiguration](../WDLang1/1000021371.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Demande à l'utilisateur la configuration qu'il souhaite charger
sFichier est une chaîne

// Ouvre le sélecteur de fichiers
sFichier = fSélecteur("", "", "Sélectionnez un fichier de configuration...", ...
	"Configuration de tableau de bord" + TAB + "*.conf", "*.conf")
// Si un fichier est sélectionné
SI sFichier <> "" ALORS
	// Récupère la configuration contenue dans le fichier
	sConfiguration est chaîne = fChargeTexte(sFichier)
	// Charge la configuration
	SI OngletChargeConfiguration(ONG_OngletDyn, sConfiguration) ALORS
		ToastAffiche("La configuration a été chargée", toastCourt, cvMilieu, chCentre)
	SINON
		Erreur("Erreur en chargement de configuration", ErreurInfo())
	FIN
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = OngletChargeConfiguration(<Champ Onglet> , <Configuration>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la configuration a été chargée, 

- <u><u><u><u>Faux</u></u></u></u> en cas d'erreur. En cas d'erreur, la variable [ErreurDétectée](../WDLang1/3087001.md) est à <u><u><u><u>Vrai</u></u></u></u>. Pour connaître le libellé de l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*. 




**`<Champ Onglet> : (Nom de champ)`**

Nom du champ Onglet à manipuler. 

**`<Configuration> : (Chaîne de caractères)`**

Configuration des volets dynamiques sauvegardée avec la fonction [OngletSauveConfiguration](../WDLang1/1000021371.md).

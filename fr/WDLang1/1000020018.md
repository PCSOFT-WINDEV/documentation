
## DateSélecteur (Fonction)

***En anglais : DatePicker***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Affiche le sélecteur de date du système.

**Remarque** : Pour sélectionner une date à l'aide du sélecteur de date, il est conseillé d'utiliser directement le masque de saisie de type "Rouleau Date" pour le champ de saisie de type "Date". Ce masque de saisie est également disponible pour les applications iPhone/iPad. 
<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=DateSelecteur.gif)



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
d est une Date = DateSélecteur()
```



```wl
DateInitiale est une Date = "20170101"
MaDate est une Date
MaDate = DateSélecteur(DateInitiale, "Date de départ")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = DateSélecteur([<Date initiale> [, <Titre>]])`
---

**`<Résultat> : (Date ou Chaîne de caractères)`**



- Date sélectionnée dans la sélecteur, 

- Chaîne vide ("") si la sélection a été annulée par l'utilisateur.




**`<Date initiale> : (Date ou Chaîne de caractères optionnelle)`**

Date affichée à l'ouverture du sélecteur. Si cette date n'est pas spécifiée, le sélecteur affichera la date du jour.

**`<Titre> : (Chaîne de caractères optionnelle)`**

Titre à afficher dans le sélecteur. Si ce paramètre n'est pas spécifié, le titre affiché sera une représentation textuelle de la date affichée par le sélecteur.

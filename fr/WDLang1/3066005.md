
## RéseauNomRep (Fonction)

***En anglais : NetworkDirName***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Identifie le répertoire partagé associé à un lecteur réseau. Le chemin UNC correspondant à la lettre du lecteur spécifiée est renvoyée.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Identifier le répertoire associé à la lettre réseau "E:"
// Par exemple \\Sdoc\Doc
ResNomRep = RéseauNomRep("E:")
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = RéseauNomRep(<Lettre du lecteur>)`
---

**`<Résultat> : (Chaîne de caractères)`**



- Nom du répertoire associé au lecteur réseau spécifié. Le nom du répertoire peut correspondre à un chemin UNC.

- Chaîne vide ("") si le lecteur spécifié n'est pas un lecteur réseau. 




**`<Lettre du lecteur> : (Chaîne de caractères)`**

Lecteur réseau à manipuler au format "&lt;Lettre du lecteur&gt;**:**".

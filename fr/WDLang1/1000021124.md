


## &lt;Diagramme de Gantt&gt;.HeureOuvrable (Fonction)

***En anglais : &lt;Gantt chart&gt;.WorkingHour***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Surcharge les heures ouvrables d'une ligne dans une colonne d'un champ Table (ou Table hiérarchique) de type Diagramme de Gantt.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Pour la ligne 2, les heures ouvrables vont de 9h30 à 12h puis de 13h à 18h30
COL_Gantt.HeureOuvrable(2, "0930", "1200", "1300", "1830")
// Pour la ligne 3, les heures ouvrables vont de 10h00 à 17h00
COL_Gantt.HeureOuvrable(3, "1000", "1700")
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Colonne Gantt>.HeureOuvrable(<Ligne> , <Heure de début> , <Heure de fin> [, <Heure de début 2> [, <Heure de fin 2>]])`
---

**`<Colonne Gantt> : (Nom de champ)`**

Nom de la colonne (d'un champ Table ou Table hiérarchique) de type Diagramme de Gantt à manipuler.

**`<Ligne> : (Entier)`**

Numéro de la ligne à modifier. 
Remarque : La ligne associée à une tâche peut être connue par la propriété **Ligne** de la variable de type [TâcheGantt](../WDLang1/1000020881.md).

**`<Heure de début> : (Chaîne de caractères ou variable de type Heure)`**

Heure de début de la première plage horaire ouvrable.

**`<Heure de fin> : (Chaîne de caractères ou variable de type Heure)`**

Heure de fin de la première plage horaire ouvrable.

**`<Heure de début 2> : (Chaîne de caractères ou variable de type Heure)`**

Heure de début de la seconde plage horaire ouvrable.

**`<Heure de fin 2> : (Chaîne de caractères ou variable de type Heure)`**

Heure de fin de la seconde plage horaire ouvrable.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Pour les lignes non spécifiées, les heures utilisées sont : 

	- les heures ouvrables définies pour le champ sous l'éditeur de fenêtres. 

	- les heures ouvrables définies par les propriétés [HeureOuvrableDébut](../Proprietes/1000018958.md) et [HeureOuvrableFin](../Proprietes/1000019056.md). 




- Cette fonction affecte le dessin du diagramme de Gantt si les heures sont affichées dans le diagramme de Gantt. 

- Cette fonction affecte le calcul de la durée des tâches si celles-ci sont spécifiées en heures.

- Deux plages horaires peuvent être utilisées pour définir par exemple les heures ouvrables du matin et celles de l'après-midi.

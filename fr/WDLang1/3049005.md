


## ListeAffiche (Fonction)

***En anglais : ListDisplay***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Rafraîchit l'affichage d'un champ Liste, Liste image ou Combo fichier. Pour un champ Liste, Liste image ou Combo par programmation, le rafraîchissement est automatique.

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Ce rafraîchissement peut être réalisé à partir d'un position donnée.**Remarque** : La fonction **ListeAffiche** peut être utilisée indifféremment sur les listes monosélection et multisélections.










<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Rafraîchissement du champ "LISTE_ListeClient"
ListeAffiche(LISTE_ListeClient)
```


<a name="Exemple2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) 
```wl
// Rafraîchissement du champ "LISTE_ListeClient" à partir du début du champ Liste
ListeAffiche(LISTE_ListeClient, taDébut)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`ListeAffiche(<Champ Liste> [, <Position>])`
---

**`<Champ Liste> : (Nom de champ)`**

Nom du champ Liste, Liste image ou Combo fichier à rafraîchir.
Si ce paramètre correspond à une chaîne vide (""), le champ Liste (Liste image ou Combo) auquel appartient l'événement en cours est rafraîchi.

**`<Position> : (Entier ou constante (optionnel))`**

Indique la position de départ du rafraîchissement. Ce paramètre peut correspondre à :

- un entier. Le rafraîchissement est effectué à partir de cette position. L'élément sélectionné n'est pas modifié. Si ce paramètre est invalide, la fonction **ListeAffiche** est sans effet.

- une constante : 
	


|   |   |
| --- | --- |
| *taCourantBandeau* | <br><br>	- Rafraîchit le champ Liste à partir de la position du bandeau de sélection.<br><br>	- La position du bandeau de sélection n'est pas modifiée.<br><br><br> |
| *taCourantPremier* | <br><br>	- Rafraîchit le champ Liste à partir de l'enregistrement courant dans le fichier de données.<br><br>	- La première ligne du champ Liste est sélectionnée. Cette ligne correspond à l'enregistrement courant du champ.<br><br><br> |
| *taDébut* | <br><br>	- Rafraîchit le champ Liste en se positionnant sur la première ligne du champ Liste.<br><br>	- La première ligne du champ Liste est sélectionnée. Cette ligne correspond au premier enregistrement affiché dans le champ.<br><br><br> |
| *taInit* | <br><br>	- Exécute les événements d'initialisation du champ Liste (événements "Initialisation" et "Fin d'initialisation").<br><br>	- Remplit le champ Liste.<br><br>	- La première ligne du champ Liste est sélectionnée. Cette ligne correspond au premier enregistrement affiché dans le champ.<br><br><br> |
| *taRéExécuteRequête* | <br><br>	- Ré-exécute la requête (cas des champs Liste sur requête intégrée uniquement).<br><br>	- Rafraîchit le champ Liste en se positionnant sur la première ligne du champ Liste.<br><br>	- La première ligne du champ Liste est sélectionnée. Cette ligne correspond au premier enregistrement affiché dans le champ.<br><br><br> |
| *taSansMajSaisieCb* | Si le champ est un champ Combo avec saisie, le champ de saisie associé au champ Combo n'est pas modifié en fonction du nouveau contenu de la liste.<br>Si cette constante n'est pas utilisée, par défaut, le champ de saisie associé au champ Combo est modifié. <br>Cette constante peut être combinée avec les autres constantes.<br><br>![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Cette constante n'est pas disponible. |

<a name="NOTE0_1"></a>

## Remarques


### Divers
<a name="divers_ELTPARAGRAPHE000208"></a>

- Pour réactualiser les champs Table, utilisez la fonction [TableAffiche](../WDLang1/3074003.md).

- La fonction **ListeAffiche** peut être utilisée sur une colonne d'un champ Table de type "Combo".

### Connecteur Natif SQLServer
<a name="connecteur_natif_sqlserver_ELTPARAGRAPHE000226"></a>Si la fonction **ListeAffiche** manipule un champ Liste lié à un fichier de données accédé par le Connecteur Natif SQLServer, il est conseillé d'utiliser les constantes *taCourantPremier* et *taDébut*.

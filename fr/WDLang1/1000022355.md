


## VidéoDemandePermission (Fonction)

***En anglais : VideoRequestPermission***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Demande la permission d'utiliser la caméra de l'appareil.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI VidéoDemandePermission() = Vrai ALORS
	OuvreFenêtreMobile(FEN_Caméra)
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = VidéoDemandePermission()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la permission d'utiliser la caméra a été accordée,

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.

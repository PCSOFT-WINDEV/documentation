
## PlanningVersiCalendar (Fonction)

***En anglais : SchedulerToiCalendar***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Exporte les rendez-vous d'un champ Planning au format iCalendar.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
siCal est une chaîne ANSI
siCal = PlanningVersiCalendar(PLN_PLANNING, "user", "20150101", "20151231")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = PlanningVersiCalendar(<Champ Planning> , <Ressource> [, <Date de début> [, <Date de fin>]])`
---

**`<Résultat> : (Chaîne de caractères ANSI au format UTF-8)`**



- Liste des rendez-vous du champ Planning au format iCalendar. 

- Chaîne vide ("") en cas d'erreur. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md). 




**`<Champ Planning> : (Nom de champ)`**

Nom du champ Planning à manipuler. Ce champ peut correspondre :

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) au champ Planning d'une fenêtre, 

- au champ Planning d'un état. 




**`<Ressource> : (Chaîne de caractères)`**

Nom de la ressource à exporter. 

**`<Date de début> : (Chaîne de caractères ou variable de type DateHeure optionnelle)`**

Date de début des éléments pris en compte pour l'export.

**`<Date de fin> : (Chaîne de caractères ou variable de type DateHeure optionnelle)`**

Date de fin des éléments pris en compte pour l'export.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Pour importer des données au format iCalendar dans un champ Planning, utilisez la fonction [iCalendarVersPlanning](../WDLang1/1000021917.md).




## ArbreTrie (Fonction)

***En anglais : TreeSort***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Trie les éléments présents dans un champ Arbre.

Exemple de champ Arbre :<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Arbre_Vocabulaire.gif)

<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Tri des éléments présents dans le champ Arbre "ARBRE_RecetteTV"
// Tri alphabétiquement à partir du noeud "Desserts"
Res = ArbreTrie(ARBRE_RecetteTV, "Recettes" + TAB + "Desserts", aAscendant)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = ArbreTrie(<Champ Arbre> [, <Chemin de l'élément de départ> [, <Type de tri>]])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le tri a été effectué, 

- <u><u><u><u>Faux</u></u></u></u> si l'élément de départ spécifié n'existe pas.


![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Après l'utilisation de la fonction **ArbreTrie**, l'ensemble des branches du champ Arbre manipulé s'enroulent automatiquement.

**`<Champ Arbre> : (Nom de champ)`**

Nom du champ Arbre à trier.

**`<Chemin de l'élément de départ> : (Chaîne de caractères optionnelle)`**

Chemin complet de l'élément à partir duquel le tri va être effectué. Si ce paramètre correspond à la constante *NULL* ou n'est pas spécifié, le tri est effectué sur l'ensemble des éléments du champ Arbre. Ce paramètre est de la forme :

```txt
"<Nom de la racine>" + TAB + ["<Nom du 1er noeud>" + TAB + ...
["<Nom du 2ème noeud>" + TAB + [...]]]"<Nom de la feuille>"
```
En cas de doublons sur un élément du chemin, ce chemin peut contenir l'identifiant de l'élément (indiqué à la suite du nom grâce à la fonction [ArbreID](../WDLang1/3018027.md)).

**`<Type de tri> : (Constante optionnelle)`**

Type de tri à effectuer.


|   |   |
| --- | --- |
| *aAscendant*<br>(Valeur par défaut) | Tri par ordre alphabétique. |
| *aDescendant* | Tri à l'inverse de l'ordre alphabétique. |

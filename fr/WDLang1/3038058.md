


## MenuSupprime (Fonction)

***En anglais : MenuDelete***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime un menu ou une option de menu.

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Il est possible de supprimer des options de menu principal ou des options de menu contextuel.

![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) Seuls les menus, options et séparateurs ajoutés par programmation (avec les fonctions [MenuAjouteMenu](../WDLang1/1000017266.md), [MenuAjouteOption](../WDLang1/1000017262.md) et [MenuAjouteSéparateur](../WDLang1/1000017270.md)) peuvent être supprimés.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Suppression de l'option "OPT_Quitter" du menu "MenuFichier"
MenuSupprime(MenuFichier.OPT_Quitter)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Supprimer un menu

`MenuSupprime(<Nom du menu>)`
---

**`<Nom du menu> : (Chaîne de caractères)`**

Nom du menu à supprimer.  


<a name="SYNTAXE2"></a>

### Supprimer une option de menu

`MenuSupprime(<Nom de l'option>)`
---

**`<Nom de l'option> : (Chaîne de caractères)`**

Nom de l'option de menu à supprimer, au format suivant "&lt;Nom du menu&gt;.&lt;Nom de l'option&gt;".

Cette option sera définitivement supprimée du menu.  


<a name="SYNTAXE3"></a>

### Supprimer une option de menu par son indice

`MenuSupprime(<Nom du menu> , <Rang de l'option>)`
---

**`<Nom du menu> : (Chaîne de caractères)`**

Nom du menu contenant l'option à supprimer. Il peut s'agir d'un menu principal ou d'un menu contextuel.

**`<Rang de l'option> : (Entier)`**

Rang de l'option à supprimer dans le menu (les rangs commencent à 1).

Cette option sera définitivement supprimée du menu.

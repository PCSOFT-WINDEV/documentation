


## Gestion des vidéos (retransmises par une Web Caméra) (syntaxe préfixée)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000072"></a>
Pour gérer la séquence vidéo actuellement retransmise par la Web caméra installée sur le poste en cours, WINDEV met à votre disposition :

- le [champ Web Caméra](../WDChamp/1013188.md).

- des [fonctions WLangage](../WDLang1/1000022386.md).




Il est par exemple possible de :

- visualiser la vidéo retransmise par une Web caméra.

- enregistrer une image de la vidéo retransmise dans un fichier image (BMP, JPEG, ...).

- enregistrer une partie de la vidéo retransmise dans un fichier vidéo.

- ...




**Remarque** : Pour visualiser une vidéo sous forme d'AVI, utilisez un [champ Image](../WDChamp/1013218.md) et les [fonctions MCI](../WDLang1/3051010.md).

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Principe d'utilisation
<a name="principe_utilisation_ELTTEXTE000122"></a>
Pour manipuler des vidéos :

1. Créez une fenêtre contenant un champ Web Caméra.

2. Affichez la fenêtre de description de ce champ (option "Description" du menu contextuel).

3. Dans cette fenêtre, sélectionnez la Web Caméra à utiliser. 
	**Remarque** : Il est également possible de spécifier la Web Caméra à utiliser par programmation (fonction [&lt;Caméra&gt;.ChangeDriver](../WDLang1/1000022378.md)).

4. Par programmation :

	- Affichez la séquence vidéo actuellement retransmise par la Web Caméra spécifiée dans le champ Web Caméra (fonction [&lt;Caméra&gt;.Affiche](../WDLang1/1000022380.md)).

	- Sauvegardez si nécessaire la vidéo actuellement retransmise par la Web Caméra spécifiée sous forme d'une image (fichier BMP) ou d'une séquence vidéo (fichier AVI) à l'aide de la fonction [&lt;Caméra&gt;.Capture](../WDLang1/1000022382.md).

	- Modifiez si nécessaire la Web Caméra utilisée (fonction [&lt;Caméra&gt;.ChangeDriver](../WDLang1/1000022378.md)).

	- Configurez si nécessaire le driver de capture vidéo (fonction [&lt;Caméra&gt;.Configure](../WDLang1/1000022379.md)).

	- Listez si nécessaire les drivers de capture vidéo installés sur le poste en cours (fonction [VidéoListeDriver](../WDLang1/3079008.md)).

	- Renvoyez si nécessaire l'état du champ Web Caméra (fonction [&lt;Caméra&gt;.Etat](../WDLang1/1000022476.md)).

	- Arrêtez l'affichage ou la sauvegarde d'une séquence vidéo dans le champ Web Caméra (fonction [&lt;Caméra&gt;.Arrête](../WDLang1/1000022381.md)).






<a name="NOTE2_2"></a>


### Remarques
<a name="remarques_ELTPARAGRAPHE000206"></a>

- En exécution, dès l'ouverture de la fenêtre contenant le champ Web Caméra, la séquence vidéo retransmise par la Web Caméra associée s'affiche automatiquement. Dans ce cas, l'utilisation de la fonction [&lt;Caméra&gt;.Affiche](../WDLang1/1000022380.md) n'est pas nécessaire.

- Si aucune Web Caméra n'a été associée au champ Web Caméra, les fonctions [&lt;Caméra&gt;.Affiche](../WDLang1/1000022380.md) et/ou [&lt;Caméra&gt;.Capture](../WDLang1/1000022382.md) manipulent la séquence vidéo retransmise par la Web Caméra par défaut (du poste en cours).

- Pour sauvegarder la vidéo actuellement retransmise par la Web Caméra spécifiée sous forme d'un fichier image, il est possible d'utiliser :

	- soit la fonction [&lt;Caméra&gt;.Capture](../WDLang1/1000022382.md).

	- soit les fonctions [&lt;Image&gt;.SauveBMP](../WDLang1/1000024341.md), [&lt;Image&gt;.SauveGIF](../WDLang1/1000024342.md) et [&lt;Image&gt;.SauveJPEG](../WDLang1/1000024344.md).


## LienCouleurRTF (Fonction)

***En anglais : LinkColorRTF***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie la couleur des liens dans tous les champs de saisie de type RTF présents dans l'application. Par défaut, les liens dans les champs de saisie de type RTF utilisent le bleu, ou la couleur précisée avec la fonction [LienCouleur](../WDLang1/1000022354.md). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Personnalise la couleur des liens (champs de saisie et champs Libellé)
LienCouleur(RVB(250,80,84), RVB(255,90,89), RVB(250,70,84))
// Personnalise la couleur des liens dans les champs de saisie RTF
LienCouleurRTF(RVB(25,80,84), RVB(25,90,89), RVB(25,70,84))
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`LienCouleurRTF(<Couleur par défaut> [, <Couleur de survol> [, <Couleur du lien enfoncé> [, <Couleur du lien visité>]]])`
---

**`<Couleur par défaut> : (Entier ou constante de type Entier)`**

Couleur du lien par défaut. Cette couleur peut correspondre à :

- une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)),

- [une couleur prédéfinie du WLangage](../WDLang5/3010002.md). 


Si ce paramètre correspond à la constante *CouleurDéfaut*, les couleurs du style original sont rétablies. 

**`<Couleur de survol> : (Entier ou constante de type Entier optionnel)`**

Couleur du lien en survol. Cette couleur peut correspondre à :

- une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)),

- [une couleur prédéfinie du WLangage](../WDLang5/3010002.md). 


Si ce paramètre n'est pas précisé, la couleur de survol est déduite à partir du paramètre &lt;Couleur par défaut&gt;.

**`<Couleur du lien enfoncé> : (Entier ou constante de type Entier optionnel)`**

Couleur du lien durant le clic. Cette couleur peut correspondre à :

- une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)),

- [une couleur prédéfinie du WLangage](../WDLang5/3010002.md). 


Si ce paramètre n'est pas précisé, la couleur du lien enfoncé est déduite à partir du paramètre &lt;Couleur par défaut&gt;.

**`<Couleur du lien visité> : (Entier ou constante de type Entier optionnel)`**

Couleur du lien déjà visité. Cette couleur peut correspondre à :

- une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)),

- [une couleur prédéfinie du WLangage](../WDLang5/3010002.md). 


Si ce paramètre n'est pas précisé, la couleur du lien visité est déduite à partir du paramètre &lt;Couleur par défaut&gt;.


## FenOrientation (Fonction)

***En anglais : WinOrientation***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie ou récupère l'orientation d'affichage de la fenêtre courante.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI FenOrientation() = fenOrientationPaysage ALORS
	FenOrientation(fenOrientationPortrait)
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Modifier l'orientation de la fenêtre

`<Résultat> = FenOrientation(<Orientation>)`
---

**`<Résultat> : (Constante de type Entier)`**

Orientation de la fenêtre après modification : 


|   |   |
| --- | --- |
| *fenOrientationPaysage* | Fenêtre en mode paysage (la largeur est supérieure à la hauteur). |
| *fenOrientationPortrait* | Fenêtre en mode portrait (la hauteur est supérieure à la largeur). |





**`<Orientation> : (Constante de type Entier)`**

Nouvelle orientation à donner à la fenêtre : 


|   |   |
| --- | --- |
| *fenOrientationPaysage* | Fenêtre en mode paysage (la largeur est supérieure à la hauteur). |
| *fenOrientationPortrait* | Fenêtre en mode portrait (la hauteur est supérieure à la largeur). |






<a name="SYNTAXE2"></a>

### Connaître l'orientation actuelle de la fenêtre

`<Résultat> = FenOrientation()`
---

**`<Résultat> : (Constante de type Entier)`**

Orientation actuelle de la fenêtre : 


|   |   |
| --- | --- |
| *fenOrientationPaysage* | Fenêtre en mode paysage (la largeur est supérieure à la hauteur). |
| *fenOrientationPortrait* | Fenêtre en mode portrait (la hauteur est supérieure à la largeur). |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
**Attention** : Si la fonction **FenOrientation** est appelée alors qu'il n'y a pas de fenêtre courante, la fonction est sans effet et retourne -1.

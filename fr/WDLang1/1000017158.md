
## CorbeilleSupprime (Fonction)

***En anglais : RecycleBinDelete***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Affiche un message de confirmation pour la suppression d'un fichier dans la corbeille. Si l'utilisateur valide, le fichier est supprimé, sinon le fichier reste dans la corbeille.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// -- Bouton de remplissage d'un champ Liste
// Affiche les éléments de la corbeille dans un champ Liste
sMaListeCorbeille est une chaîne
ListeSupprimeTout(LISTE_Corbeille)
sMaListeCorbeille = CorbeilleListeFichier()
ListeAjoute(LISTE_Corbeille, sMaListeCorbeille)

// -- Bouton de suppression de l'élément sélectionné dans le champ Liste
CorbeilleSupprime(LISTE_Corbeille[LISTE_Corbeille])
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = CorbeilleSupprime(<Nom du fichier>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le message de confirmation de la suppression a été affiché, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Nom du fichier> : (Chaîne de caractères)`**

Nom complet d'origine du fichier (chemin complet et nom du fichier). Si le fichier est présent en plusieurs exemplaires dans la corbeille, tous les fichiers seront supprimés. 
Pour supprimer uniquement un fichier, il est nécessaire de faire suivre le nom du fichier par l'heure d'effacement en utilisant le format suivant : 

```txt
<Nom du fichier> + TAB + <Heure d'effacement>
```

Ce format est renvoyé par la fonction [CorbeilleListeFichier](../WDLang1/1000017162.md).

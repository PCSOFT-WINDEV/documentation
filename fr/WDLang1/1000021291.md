


## &lt;Interrupteur&gt;.Ajoute (Fonction)

***En anglais : &lt;Check box&gt;.Add***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ajoute une nouvelle option dans un champ Interrupteur. L'option est ajoutée à la suite des options existantes.  




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ajout de 2 options 
INT_Options.Ajoute("Lire un livre sur une tablette")
INT_Options.Ajoute("Lire un livre de poche")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Champ Interrupteur>.Ajoute(<Libellé> [, <Sous-libellé> [, <Image>]])`
---

**`<Résultat> : (Entier)`**



- Indice de l'option ajoutée. 

- -1 en cas d'erreur. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Champ Interrupteur> : (Nom de champ)`**

Nom du champ Interrupteur dans lequel l'option doit être ajoutée. L'option est ajoutée à la suite des options existantes.

**`<Libellé> : (Chaîne de caractères)`**

Libellé de l'option à ajouter.

**`<Sous-libellé> : (Chaîne de caractères optionnelle)`**

Sous-libellé de l'option à ajouter. 
Si ce paramètre n'est pas spécifié, l'option ajoutée n'a pas de sous-libellé. 
Si ce paramètre est spécifié, le sous-libellé sera affiché même si l'interrupteur utilise un mode d'affichage sans sous-libellé.


**`<Image> : (Chaîne de caractères optionnelle)`**

Image associée à l'option à ajouter. 
Si ce paramètre n'est pas spécifié, l'option ajoutée n'a pas d'image.
Si ce paramètre est spécifié, l'image sera affichée uniquement si l'interrupteur utilise un mode d'affichage "Style Vista" (avec image).




<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Les caractéristiques de l'option ajoutée peuvent également être modifiées grâce aux propriétés suivantes utilisées sur l'option : 

	- Propriété [Libellé](../Proprietes/2510053.md) pour modifier le libellé. 

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Propriété [SousLibellé](../Proprietes/1000017232.md) pour modifier le sous-libellé. 

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Propriété [Image](../Proprietes/2510034.md) pour modifier l'image. 




- Lors de l'ajout d'options dans un champ Interrupteur, la taille du champ n'est pas modifiée. Si le champ n'est pas suffisamment grand, les options ajoutées pourront ne pas être visibles. Il est possible d'augmenter la taille du champ avec les propriétés [Largeur](../Proprietes/2510059.md) et [Hauteur](../Proprietes/2510050.md).

- Pour ajouter une option à une position spécifique, utilisez la fonction [&lt;Interrupteur&gt;.Insère](../WDLang1/1000021307.md).

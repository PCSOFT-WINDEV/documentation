


## &lt;Variable WDPic&gt;.Flou (Fonction)

***En anglais : &lt;WDPic variable&gt;.Blur***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Floute la totalité d'une zone donnée dans une variable de type [WDPic](../WDLang1/1000023503.md) (sur le calque d'arrière-plan).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonWDPic est un WDPic = "Test.bmp"
// Floute l'image
MonWDPic.Flou()

IMG_MonDessin = MonWDPic
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Flouter la totalité d'une image

`<Variable Image>.Flou([<Rayon>])`
---

**`<Variable Image> : (Variable de type WDPic)`**

Nom de la variable de type [WDPic](../WDLang1/1000023503.md) à manipuler. Seul le calque d'arrière-plan sera manipulé. 

**`<Rayon> : (Entier optionnel)`**

Rayon autour d'un point dans lequel on tient compte des points voisins pour effectuer le floutage. Plus le rayon est grand, plus l'image sera floutée, moins le floutage sera rapide. La valeur par défaut est 10.


<a name="SYNTAXE2"></a>

### Flouter une zone d'une image

`<Variable Image>.Flou(<Zone à flouter> [, <Rayon>])`
---

**`<Variable Image> : (Variable de type WDPic)`**

Nom de la variable de type [WDPic](../WDLang1/1000023503.md) à manipuler. Seul le calque d'arrière-plan sera manipulé. 

**`<Zone à flouter> : (Variable de type Rectangle ou Polygone)`**

Zone de l'image à flouter. Cette zone peut être définie par 

- une variable de type [Rectangle](../Motscles/1514079.md) (pour les zones simples).

- une variable de type [Polygone](../WDLang1/1000025624.md) (pour les zones complexes). 
	Remarques : 

	- Il n'est pas nécessaire de fermer le polygone (les coordonnées du dernier point n'ont pas à être égales aux coordonnées du premier point). La "fermeture" du polygone est réalisée automatiquement.

	- Un polygone peut avoir au maximum 100 000 points.





Les coordonnées sont indiquées par rapport au coin haut gauche de l'image (coordonnées (0,0)).

**`<Rayon> : (Entier optionnel)`**

Rayon autour d'un point dans lequel on tient compte des points voisins pour effectuer le floutage. Plus le rayon est grand, plus l'image sera floutée, moins le floutage sera rapide. La valeur par défaut est 10.

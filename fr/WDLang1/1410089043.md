


## &lt;Variable FichierDisque&gt;.Ferme (Fonction)

***En anglais : &lt;DiskFile variable&gt;.Close***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ferme un fichier externe.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) 
```wl
ResFerme est un entier

// Ouverture d'un fichier externe
MonFichier est un FichierDisque
MonFichier = fOuvre("C:\MesRépertoires\Fichier.txt", foLectureEcriture)
SI PAS ErreurDétectée ALORS
	...
	// Fermeture du fichier externe
	ResFerme = MonFichier.Ferme()
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Fichier manipulé>.Ferme()`
---

**`<Résultat> : (Entier)`**



- 0 si le fichier externe est fermé, 

- -1 en cas d'erreur. Pour connaître le libellé de l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*. La fonction **&lt;Variable FichierDisque&gt;.Ferme** génère une erreur si le fichier n'est pas ouvert.




**`<Fichier manipulé> : (Variable de type FichierDisque)`**

Nom de la variable de type [FichierDisque](../WDLang1/1410088883.md) initialisée avec la fonction [fOuvre](../WDLang1/3036036.md) ou [fCrée](../WDLang1/3036017.md).




<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Fermeture d'un fichier ouvert plusieurs fois
<a name="fermeture_fichier_ouvert_plusieurs_fois_ELTPARAGRAPHE000273"></a>Il est possible d'ouvrir un même fichier plusieurs fois en même temps. Chaque version du fichier doit être fermée individuellement par son propre identifiant.
<a name="NOTE0_2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) 

### Fermeture d'un fichier bloqué
<a name="fermeture_fichier_bloque_ELTPARAGRAPHE000285"></a>Un fichier bloqué sera automatiquement débloqué lors de sa fermeture.

**Rappel** : La fonction [&lt;Variable FichierDisque&gt;.Bloque](../WDLang1/1410089037.md) permet de bloquer un fichier externe. La fonction [&lt;Variable FichierDisque&gt;.Débloque](../WDLang1/1410089040.md) permet de débloquer un fichier externe.

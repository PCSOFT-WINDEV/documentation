
## PlanificateurSupprimeTâche (Fonction)

***En anglais : SchedulerDeleteTask***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime une tâche planifiée.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Récupération du nom des tâches planifiées
sListeTâche est une chaîne = PlanificateurListeTâche()
SI sListeTâche = "" ALORS
	Info("Aucune tâche n'est planifiée sur ce poste")
SINON
	Info("On supprime les tâches déjà présentes pour partir de zéro")
	POUR n = 1 A ChaîneOccurrence(sListeTâche, RC) + 1
		// Suppression de chaque tâche planifiée
		PlanificateurSupprimeTâche(ExtraitChaîne(sListeTâche, n, RC))
	FIN
	// On vérifie qu'il n'y a plus de tâches
	sListeTâche = PlanificateurListeTâche()
	SI sListeTâche <> "" ALORS 
		Info("Toutes les tâches planifiées n'ont pas été supprimées. Il reste : " + sListeTâche)
	FIN
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = PlanificateurSupprimeTâche(<Nom de la tâche>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> en cas de succès, 

- <u><u><u><u>Faux</u></u></u></u> en cas d'erreur (si la tâche n'existe pas ou si la tâche ne peut pas être supprimée).




**`<Nom de la tâche> : (Chaîne de caractères)`**

Nom de la tâche à supprimer. Ce nom a été défini lors de la création de la tâche avec la fonction [PlanificateurAjouteTâche](../WDLang1/3089003.md).

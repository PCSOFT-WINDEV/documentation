
## TexteHauteur (Fonction)

***En anglais : TextHeight***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Calcule la hauteur (en pixels) d'un texte affiché dans un champ de saisie, un champ Libellé ou un champ Combo. Il est ainsi possible de prévoir la hauteur occupée par le texte afin d'agrandir le champ si nécessaire.








<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
Res est un réel = TexteHauteur(SAI_Saisie1, "Bonjour")
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = TexteHauteur(<Nom du champ> , <Texte> [, <Option>])`
---

**`<Résultat> : (Réel)`**

Hauteur du texte (en pixels) dans le champ spécifié (en fonction du style du champ).


**`<Nom du champ> : (Chaîne de caractères)`**

Nom du champ de saisie, du champ Libellé ou du champ Combo.


**`<Texte> : (Chaîne de caractères)`**

Texte dont la hauteur doit être calculée.

**`<Option> : (Constante optionnelle de type entier)`**

Configure le mode de calcul de la hauteur :


|   |   |   |
| --- | --- | --- |
| *tAvecMarge* | Permet de prendre en compte les marges définies pour le champ (retrait gauche, retrait droit, ...). <br>Cette constante peut être combinée avec la constante *tMultiligne*. <br><br>![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Cette constante n'est pas disponible. |
| *tDéfaut*<br>(valeur par défaut) | Hauteur du texte dans le champ. |   |
| *tMultiligne* | Hauteur du texte en tenant compte des retours à la ligne (automatiques ou forcés par un "Retour Chariot"). | ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=TexteHauteur_multi.gif)<br> |
| *tMonoligne* | Hauteur d'une ligne en tenant compte uniquement des "Retour chariot". | ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=TexteHauteur_mono.gif)<br> |

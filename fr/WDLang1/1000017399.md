
## AnimationActive (Fonction)

***En anglais : AnimationEnabled***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Active ou désactive les animations dans les champs.

**Remarque** : Seules les animations de champs sont concernées : animations automatiques (configurées dans le projet ou dans la fenêtre) ou animations programmées (avec les fonctions [AnimationPrépare](../WDLang1/1000017386.md), [AnimationJoue](../WDLang1/1000017387.md), ...). Les animations des champs Image et des champs Libellé ainsi que les animations de fenêtres ne sont pas concernées.  
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Les animations ne seront jamais activées
AnimationActive(Faux)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = AnimationActive([<Mode d'activation>])`
---

**`<Résultat> : (Booléen)`**

Correspond à l'ancien mode de gestion des animations de champs (avant l'utilisation de la fonction **AnimationActive**) : 

- <u><u><u><u>Vrai</u></u></u></u> si la gestion des animations était active, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Mode d'activation> : (Booléen optionnel)`**

Mode d'activation des animations :



|   |   |
| --- | --- |
| *Faux* | Les animations ne sont jamais affichées. |
| *Vrai* | Les animations sont en mode automatique. L'animation est effectuée en fonction de la puissance de la machine utilisée.<br>Si le poste ne dispose pas des API nécessaires aux animations, les animations ne seront pas affichées. |

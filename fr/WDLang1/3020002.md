


## Fonctions de gestion des champs Barre d'outils
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions de gestion des champs Barre d'outils sont les suivantes :



|   |   |
| --- | --- |
| [BarreAttache](../WDLang1/3020003.md) | Ancre un champ Barre d'outils à la fenêtre ou change la position d'ancrage d'un champ Barre d'outils ancré. |
| [BarreDétache](../WDLang1/3020001.md) | Transforme un champ Barre d'outils ancré en champ Barre d'outils flottant. |
| [BarrePosition](../WDLang1/3020004.md) | Indique le mode d'affichage d'un champ Barre d'outils. |




## &lt;Variable SignalManuel&gt;.Ferme (Fonction)

***En anglais : &lt;ManualEvent variable&gt;.Close***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ferme un signal de synchronisation entre plusieurs threads.


<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Signal>.Ferme()`
---

**`<Signal> : (Variable de type SignalManuel)`**

Nom d'une variable de type [SignalManuel](../WDLang1/1000021275.md) à fermer.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Cette fonction est équivalente à la syntaxe suivante de la fonction [SignalModifie](../WDLang1/3077017.md) : 
	
	```txt
	SignalModifie(<Signal>, signalFermé)
	```


- **Cas d'un signal manuel** : Le signal se ferme. 
	**Nouveaux threads** : Si d'autres threads arrivent, ils sont mis en attente.

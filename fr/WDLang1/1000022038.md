
## Fond (Type de variable)

***En anglais : Background***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **Fond** permet de définir toutes les caractéristiques avancées du fond d'un élément. Les caractéristiques de ce fond peuvent être définies et modifiées à l'aide de différentes propriétés WLangage. 

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Dessine un rectangle avec bords bleus, fond vert et coins arrondis
// dans un champ Image

// Initialise le dessin dans le champ Image
dDébutDessin(IMG_Dessin)

Rect est un Rectangle
Rect.X = 10
Rect.Y = 15
Rect.Largeur = 100
Rect.Hauteur = 150
StyleCadre est un Cadre
StyleCadre.Couleur  = BleuClair
StyleCadre.Arrondi = 30
StyleCadre.Epaisseur = 1
StyleFond est un Fond
StyleFond.Couleur = VertClair
dCadre(Rect, StyleCadre, StyleFond)
```

## Remarques
<a name="NOTE0_1"></a>


### Propriétés spécifiques à la description des variables de type Fond
<a name="proprietes_specifiques_description_des_variables_type_fond_ELTPARAGRAPHE000044"></a>Les propriétés suivantes peuvent être utilisées pour manipuler le fond d'un élément :

| Nom de la propriété | Type manipulé | Effet |
| --- | --- | --- |
| Couleur | Variable de type [Couleur](../WDLang1/1000019661.md) | Couleur de fond. |
| Dégradé | <br><br> | Dégradé de la couleur de fond : <br><br>- Propriété **CouleurDébut** (type [Couleur](../WDLang1/1000019661.md)) : Couleur de début du dégradé. <br><br>- Propriété **CouleurFin** (type [Couleur](../WDLang1/1000019661.md)) : Couleur de fin du dégradé.<br><br>- Propriété **Angle** (entier) : Angle à utiliser pour la sélection des couleurs dans le dégradé. La valeur de l'angle est comprise entre 0 (angle horizontal, valeur par défaut) et 360°.<br><br>- Propriété **Couleur3** (type [Couleur](../WDLang1/1000019661.md)) : Couleur à utiliser dans le dégradé.<br><br>- Propriété **DistanceCouleur3** (entier) : Pourcentage séparant la couleur de début et la couleur 3. <br><br>- Propriété **Couleur4** (type [Couleur](../WDLang1/1000019661.md)) : Couleur à utiliser dans le dégradé.<br><br>- Propriété **DistanceCouleur4** (entier) : Pourcentage séparant la couleur de début et la couleur 4.  <br><br><br><br><br>Les paramètres de couleur sont utilisés de la façon suivante :<br><br><br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=drectangledegrade.gif)<br><br><br>L'angle permet de définir l'orientation du dégradé.<br><br><br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=angle_degrade.gif)<br><br><br>Les dégradés réalisés avec 3 ou 4 couleurs sont disponibles uniquement pour un angle de 0 ou de 90 degrés.<br><br>**Remarques** : <br><br>- Sur certains systèmes (TSE 256 couleurs par exemple), les dégradés ne sont pas disponibles. Dans ce cas, la couleur définie via la propriété **CouleurDébut** est affichée.<br><br>- Sur les documents docx, les fonds dégradés ne sont pas gérés. <br><br><br> |
| Hachure | Constante de type Entier | Type de hachures utilisées : <br><br>- *HachureAucune* : Pas de hachures. <br><br>- *HachureCroisé* : Hachures croisées.<br><br>- *HachureCroiséDiagonale* : Hachures diagonales croisées.<br><br>- *HachureDiagonale* : Hachures diagonales à 60%.<br><br>- *HachureDiagonaleBas* : Hachures diagonales à 120%.<br><br>- *HachureHorizontale* : Hachures horizontales.<br><br>- *HachureVerticale* : Hachures verticales. <br><br><br> |
| Type | Constante de type Entier | Type de fond : <br><br>- *FondPlein* : Fond plein.<br><br>- *FondTransparent* : Fond transparent (la propriété **Couleur** est alors ignorée).<br><br>- *FondHachuré* : Fond avec hachures (la propriété **Hachure** est prise en compte). <br><br>- *FondDégradé* : Fond avec un dégradé de couleurs (la propriété **Dégradé** est prise en compte).<br><br><br> |


## FenCopieFormulaire (Fonction)

***En anglais : WinCopyForm***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Copie le contenu d'un formulaire dans le presse-papiers.

Remarques : 

- Le contenu copié peut être collé dans un autre formulaire grâce à la fonction [FenColleFormulaire](../WDLang1/1000018993.md).

- Il est également possible de copier le contenu d'un formulaire grâce à la FAA (Fonctionnalité Automatique de vos Applications) de copie de formulaire. 



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Copie les valeurs de formulaire
FenCopieFormulaire(FEN_SaisieNom)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`FenCopieFormulaire([<Nom de la fenêtre>])`
---

**`<Nom de la fenêtre> : (Chaîne de caractères optionnelle)`**

Nom ou alias de la fenêtre à manipuler. Si ce paramètre n'est pas précisé ou correspond à une chaîne vide (""), la fenêtre en cours est manipulée.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Les types de champs supportés sont :

- Champs de saisie,

- Sélecteurs/Interrupteurs,

- Potentiomètres, potentiomètres rotatifs,

- Calendriers,

- Combos en saisie.

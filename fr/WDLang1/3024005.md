
## ChaîneRécupère (Fonction)

***En anglais : StringRetrieve***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Récupère une chaîne de caractères à partir d'un format externe à PC Soft (chaîne de caractères résultat d'une fonction de l'API Windows par exemple).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// La DLL "MaDLL32.dll" renvoie une adresse sur laquelle est stockée une chaîne ASCIIZ
ChargeDLL("MaDLL32.dll")
nAdrChaîneASCIIZ est un entier = AppelDLL32("MaDLL32.dll", "RenvoieAdresseChaîne")
sContenuChaîne est une chaîne = ChaîneRécupère(nAdrChaîneASCIIZ, crAdresseASCIIZ)
DéchargeDLL("MaDLL32.dll")
Info(sContenuChaîne)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = ChaîneRécupère(<Paramètre> , <Option>)`
---

**`<Résultat> : (Chaîne de caractères)`**

Chaîne résultat au format PC Soft.

**`<Paramètre> : (Entier système)`**

Adresse de la chaîne de caractères à récupérer.

**`<Option> : (Constante de type Entier)`**

Format de la chaîne à récupérer.


|   |   |
| --- | --- |
| *crAdresseASCIIZ* | &lt;Paramètre&gt; doit être un entier contenant l'adresse d'une chaîne au format ASCIIZ. |
| *crAdresseUNICODE* | &lt;Paramètre&gt; doit être un entier contenant l'adresse d'une chaîne au format UNICODE. |

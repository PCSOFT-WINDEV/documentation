


## OrgaModifie (Fonction)

***En anglais : OrgModify***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie un élément dans un champ Organigramme.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Syntaxe 1 : Modification uniquement du titre
Boss est un entier 
Boss = OrgaAjoute(ORGA_MonOrganigramme, "Matthieu Boss", "big boss", "Goldbar.gif")

OrgaModifie(ORGA_MonOrganigramme, 1, "Tim C.", "CEO")
```
<a name="Exemple2"></a>

```wl
// Syntaxe 2 : Modification de l'élément
ElémentPDG est un OrgaElément
ElémentPDG.Titre = "Big Boss"
OrgaAjoute(ORGA_DIRECTION, ElémentPDG)

ElémentPDG.Titre="Tim C."
ElémentPDG.Image = "Goldbar.gif"
OrgaModifie(ORGA_DIRECTION, 1, ElémentPDG)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Modifier un élément d'un champ Organigramme en précisant ses caractéristiques

`OrgaModifie(<Champ Organigramme> , <Indice> , <Titre> [, <Contenu> [, <Image>]])`
---

**`<Champ Organigramme> : (Nom de champ)`**

Nom du champ Organigramme dans lequel l'élément doit être modifié. 

**`<Indice> : (Entier)`**

Indice de l'élément à modifier. Cet indice a été renvoyé par exemple par la fonction [OrgaAjoute](../WDLang1/1000019729.md). 

**`<Titre> : (Chaîne de caractères)`**

Nouveau titre de l'élément. 

**`<Contenu> : (Chaîne de caractères optionnelle)`**

Nouveau contenu de l'élément. 
Si ce paramètre n'est pas précisé, le contenu existant ne sera pas modifié. 

**`<Image> : (Chaîne de caractères optionnelle)`**

Nouvelle image associée à l'élément. Cette image s'affiche à gauche du titre de l'élément. Ce paramètre correspond au nom et au chemin complet (ou relatif) de l'image. Un chemin de type UNC peut être utilisé. 
Si ce paramètre n'est pas précisé, l'image existante ne sera pas modifiée. 


<a name="SYNTAXE2"></a>

### Modifier un élément d'un champ Organigramme (en utilisant le type OrgaElément)

`OrgaModifie(<Champ Organigramme> , <Indice> , <Elément>)`
---

**`<Champ Organigramme> : (Nom de champ)`**

Nom du champ Organigramme dans lequel l'élément doit être modifié. 

**`<Indice> : (Entier)`**

Indice de l'élément à modifier. Cet indice a été renvoyé par exemple par la fonction [OrgaAjoute](../WDLang1/1000019729.md).

**`<Elément> : (Variable de type OrgaElément)`**

Nom de la variable de type [OrgaElément](../WDLang1/1000019713.md) décrivant les caractéristiques complètes de l'élément à modifier. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Utilisation du type OrgaElément (2ème syntaxe)
<a name="utilisation_type_orgaelement_2eme_syntaxe_ELTPARAGRAPHE000098"></a>L'élément à l'indice &lt;Indice&gt; est remplacé par &lt;Elément&gt;. Dans ce cas, la fonction **OrgaModifie** est équivalente à la syntaxe suivante :

```txt
Champ_Organigramme[Indice] = Elément
```

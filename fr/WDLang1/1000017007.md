


## TableauVersChaîne (Fonction)

***En anglais : ArrayToString***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Convertit un tableau à 1 ou 2 dimensions en chaîne de caractères.

**Remarque** : Cette fonction est utilisable uniquement sur les [tableaux WLangage](../Motscles/1514030.md).








<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Tableau à une dimension
sMesValeurs est une chaîne 
// Tableau initial
MonTableau est un tableau de 3 chaînes
MonTableau[1] = "A"
MonTableau[2] = "B"
MonTableau[3] = "C"
sMesValeurs = TableauVersChaîne(MonTableau)
// sMesValeurs contient : "A" + RC + "B" + RC + "C"
```
<a name="Exemple2"></a>

```wl
// Tableau à deux dimensions
sMesValeurs est une chaîne 
// Tableau initial
MonTableau est un tableau de 2 par 3 chaînes
MonTableau[1,1] = "A"; MonTableau[1,2] = "B"
MonTableau[1,3] = "C"; MonTableau[2,1] = "D"
MonTableau[2,2] = "E"; MonTableau[2,3] = "F"

sMesValeurs = TableauVersChaîne(MonTableau)
// sMesValeurs contient : "A" + TAB + "B" + TAB + "C" +RC+ "D" + TAB + "E" + TAB + "F"
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Convertir un tableau à une dimension

`<Résultat> = TableauVersChaîne(<Tableau WLangage> [, <Séparateur d'éléments>])`
---

**`<Résultat> : (Chaîne de caractères)`**

Chaîne de caractères contenant les différentes valeurs du tableau. Les éléments du tableau sont convertis en chaîne de caractères, puis ajoutés dans la chaîne en étant séparés par le &lt;Séparateur d'éléments&gt;.

**`<Tableau WLangage> : (Variable de type Tableau)`**

Nom du tableau à une dimension à convertir en chaîne de caractères. Ce tableau doit être alloué.

**`<Séparateur d'éléments> : (Chaîne de caractères optionnelle)`**

Séparateur utilisé pour séparer les valeurs des différents éléments du tableau. Par défaut, le séparateur d'éléments utilisé est RC (Retour Chariot).  


<a name="SYNTAXE2"></a>

### Convertir un tableau à 2 dimensions

`<Résultat> = TableauVersChaîne(<Tableau WLangage> [, <Séparateur de ligne> [, <Séparateur de colonne>]])`
---

**`<Résultat> : (Chaîne de caractères)`**

Chaîne de caractères contenant les différentes valeurs du tableau. Les éléments du tableau sont convertis en chaîne de caractères, puis ajoutés dans la chaîne. Les lignes sont séparées par &lt;Séparateur de ligne&gt; et les colonnes sont séparées par &lt;Séparateur de colonne&gt;.

**`<Tableau WLangage> : (Variable de type Tableau)`**

Nom du tableau à 2 dimensions à convertir en chaîne de caractères. Ce tableau doit être alloué. 

**`<Séparateur de ligne> : (Chaîne de caractères optionnelle)`**

Séparateur utilisé pour séparer les valeurs des différentes lignes. Par défaut, le séparateur de lignes utilisé est RC (Retour Chariot).

**`<Séparateur de colonne> : (Chaîne de caractères optionnelle)`**

Séparateur utilisé pour séparer les valeurs des différentes colonnes. Par défaut, le séparateur de colonnes utilisé est TAB.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Les tableaux de type structure gérés sont uniquement des tableaux à une dimension. Dans ce cas, l'ordre des paramètres détermine l'ordre dans la chaîne résultat.

- Les tableaux contenant des chaînes UNICODE ou des variants ne sont pas gérés.

- Pour le type Durée, la valeur est convertie en millièmes de seconde.

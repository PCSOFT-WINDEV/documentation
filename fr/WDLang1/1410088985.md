
## docRemplacementAutomatique (Type de variable)

***En anglais : docAutoReplace***
			
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **docRemplacementAutomatique** permet de définir toutes les caractéristiques avancées d'une correction automatique en cours de saisie dans un champ Traitement de texte. Les caractéristiques de cette correction automatique peuvent être définies et modifiées à l'aide de différentes propriétés WLangage. 

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
c est un docRemplacementAutomatique
c.NouvelleChaîne = "10 fois plus vite"
TT_MondocCommercial.CorrectionAutomatique.Remplacement["10f"] = c

// Autre syntaxe acceptée (sans passer par la variable docRemplacementAutomatique)
TT_MondocCommercial.CorrectionAutomatique.Remplacement["20f"] = "20 fois plus vite"
```
<a name="Exemple2"></a>

```wl
d est un Document
DocInsère(d, 1, "Bonjour")

TT_MondocCommercial.CorrectionAutomatique.Remplacement["bb"] = d

d2 est un Document 
d2 <- TT_MondocCommercial..CorrectionAutomatique.Remplacement["bb"]..NouvelleChaîneFormatée
```



<a name="PROPS"></a>

## Propriétés


### Propriétés spécifiques à la description des variables de type docRemplacementAutomatique
<a name="proprietes_specifiques_description_des_variables_type_docremplacementautomatique_ELTPARAGRAPHE000028"></a>Les propriétés suivantes peuvent être utilisées pour manipuler une variable de type **docRemplacementAutomatique** :

| Nom de la propriété | Type manipulé | Effet |
| --- | --- | --- |
| NouvelleChaîne | Chaîne de caractères | Chaîne contenant la correction. <br><br>Remarque : La propriété **NouvelleChaîne** est prioritaire sur la propriété **NouvelleChaîneFormatée** (si elle existe). |
| NouvelleChaîneFormatée | [Document](../WDLang1/1000022461.md) | Chaîne corrigée sous forme d'un document pouvant contenir des mises en forme.<br><br>Remarque : Cette propriété peut correspondre également à une variable de type [docParagraphe](../WDLang1/1000022483.md), [docElément](../WDLang1/1000022484.md), ... |

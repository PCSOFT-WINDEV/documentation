
## MCIRécupère (Fonction)

***En anglais : MCIRetrieve***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le dernier résultat de la commande MCI exécutée avec la fonction [MCIExécute](../WDLang1/3051005.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Dernier résultat de la fonction MCIExécute
ResRécupère = MCIRécupère()
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = MCIRécupère()`
---

**`<Résultat> : (Chaîne de caractères)`**



- Résultat retourné par la fonction [MCIExécute](../WDLang1/3051005.md), 

- Chaîne vide ("") si la fonction [MCIExécute](../WDLang1/3051005.md) n'a retourné aucun résultat.

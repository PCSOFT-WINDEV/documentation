
## USBDéconnecte (Fonction)

***En anglais : USBDisconnect***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Déconnecte ou éjecte un périphérique USB.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Récupère le premier périphérique de la liste
MonPériph est une chaîne
MonPériph = ExtraitChaîne(USBListePériphérique(), rangPremier)
USBDéconnecte(MonPériph)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`USBDéconnecte(<Identifiant USB>)`
---

**`<Identifiant USB> : (Chaîne de caractères)`**

Identifiant du périphérique USB à manipuler. Cet identifiant est obtenu grâce à la fonction [USBListePériphérique](../WDLang1/3090002.md).

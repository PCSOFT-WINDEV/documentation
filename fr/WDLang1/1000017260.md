
## AppliChangeParamètre (Fonction)

***En anglais : AppChangeParameter***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Change une information du mécanisme de mise à jour automatique de l'application spécifiée.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple

![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) 
```wl
SI AppliParamètre(appMAJMode) = 3 ALORS
	AppliChangeParamètre(appMAJMsgArrêt, ...
			"L'application " + ProjetInfo(piNomEXE) + ...
			"va être arrêtée automatiquement.")
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = AppliChangeParamètre(<Information> , <Nouvelle valeur> [, <Chemin de l'application>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'information demandée a été modifiée, 

- <u><u><u><u>Faux</u></u></u></u> en cas de problème. Pour plus de détails sur le problème rencontré, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Information> : (Constante de type Entier)`**

Caractéristique de l'application à modifier :  


|   |   |
| --- | --- |
| *appMAJAfficheErreur* | Permet d'afficher ou non un message en cas de problème lors de la vérification des mises à jour de l'application : <br><br>- <u><u><u><u>Vrai</u></u></u></u> pour afficher un message, <br><br>- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. <br><br><br><br><br>![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Constante non disponible. |
| *appMAJChemin* | Chemin des informations de mise à jour sur le serveur.<br><br> |
| *appMAJDélai* | Délai entre deux messages (si appMAJMode = 2) ou avant l'arrêt automatique de l'application (si appMAJMode = 3). Ce délai est exprimé en secondes.<br><br> |
| *appMAJMode* | Mode de fonctionnement actuel de l'application :<br><br>- 0 : Fonctionnement normal.<br><br>- 1 : Lancement interdit : les exécutables en cours continuent de fonctionner.<br><br>- 2 : Lancement interdit : les exécutables en cours reçoivent un message d'avertissement périodiquement.<br><br>- 3 : Lancement interdit : les exécutables en cours sont arrêtés automatiquement au bout d'un certain délai.<br><br><br> |
| *appMAJModeAffichage* | Mode d'affichage choisi pour prévenir l'utilisateur qu'une mise à jour de l'application est disponible :<br><br>- 0 : Une fenêtre dans l'application est utilisée pour avertir l'utilisateur.<br><br>- 1 : Une notification est utilisée pour avertir l'utilisateur.<br><br><br><br><br>![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Constante non disponible. |
| *appMAJMsgArrêt* | Message affiché avant l'arrêt automatique de l'application (quand appMAJMode = 3).<br><br> |
| *appMAJMsgAvertissement* | Message affiché périodiquement (quand appMAJMode = 2). |
| *appMAJMsgInterdit* | Message affiché lorsque le lancement de l'application est interdit. |
| *appMAJProtocole* | Protocole réseau utilisé pour la recherche des mises à jour. Ce protocole peut correspondre à "LOCAL" ou "HTTP". |
| *appMAJRéseau* | Niveau de mise à jour :<br><br>- 0 : Pas de mise à jour. <br>	La mise à jour en réseau n'est pas utilisée.<br><br>- 1 : Mise à jour réseau activée. <br>	Les mises à jour disponibles sont vérifiées et la boîte de dialogue de la mise à jour automatique est affichée.<br><br>- 2 : Mise à jour réseau activée et inscription des applications auprès du serveur de mise à jour. <br>	Les mises à jour disponibles sont vérifiées et la boîte de dialogue de la mise à jour automatique est affichée. <br>	Lors de son lancement, l'exécutable crée un fichier ".LOK" sur le serveur. Ce fichier permet à l'application [WDAdminExe](../WDAdminExe/2028052.md) de connaître les clients utilisant l'application.<br><br><br> |
| *appMAJServeur* | Serveur sur lequel sont recherchées les mises à jour.<br>Si le protocole HTTP est utilisé, adresse IP ou nom du poste.<br>Si le protocole LOCAL est utilisé, nom du partage réseau. |
| *appMAJVerifAuto* | Activation de la vérification automatique des mises à jour : <br><br>- <u><u><u><u>Vrai</u></u></u></u> pour activer, <br><br>- <u><u><u><u>Faux</u></u></u></u> pour désactiver. <br><br><br><br><br>![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Constante non disponible. |
| *appMAJVersionLocale* | Numéro de version de l'exécutable local. |



**`<Nouvelle valeur> : (Type attendu)`**

Nouvelle valeur à donner à l'information spécifiée.

**`<Chemin de l'application> : (Chaîne de caractères optionnelle)`**

Chemin complet de l'exécutable à manipuler. Si ce paramètre n'est pas précisé, l'application manipulée est l'application en cours.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Les nouveaux paramètres seront pris en compte uniquement lors du prochain lancement de l'application.

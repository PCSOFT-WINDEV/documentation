
## TailleVersChaîne (Fonction)

***En anglais : LengthToString***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Convertit une taille donnée en octets en une chaîne de caractères formatée en kilo-octets, méga-octets ou tera-octets. Cette fonction permet par exemple de formater le résultat renvoyé par les fonctions [fTaille](../WDLang1/3036055.md), [FTPTaille](../WDLang3/3040007.md), ...


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
nTaille est un entier = 123596900
Info(TailleVersChaîne(nTaille)) // Affiche "117,87 Mo"
Info(TailleVersChaîne(nTaille, tailleKo)) // Affiche "120700,10 Ko"
Info(TailleVersChaîne(nTaille, tailleGo)) // Affiche "0,12 Go"
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = TailleVersChaîne(<Taille à convertir> [, <Format d'affichage>])`
---

**`<Résultat> : (Chaîne de caractères)`**



- Résultat de la conversion. Ce résultat est composé de 2 chiffres après la virgule.

- Chaîne vide ("") en cas d'erreur. 




**`<Taille à convertir> : (Entier)`**

Taille en octets à convertir. 

**`<Format d'affichage> : (Constante de type Entier optionnelle)`**

Par défaut, la chaîne est retournée dans l'unité la plus simple à lire. Les constantes suivantes permettent de forcer l'utilisation d'une unité :


|   |   |
| --- | --- |
| *tailleGo* | Taille convertie en giga-octets. |
| *tailleKo* | Taille convertie en kilo-octets. |
| *tailleMo* | Taille convertie en méga-octets. |
| *tailleO* | Taille convertie en octets. |
| *tailleTO* | Taille convertie en tera-octets. |

## Remarques




### Formatage selon les langues utilisées
<a name="formatage_selon_les_langues_utilisees_ELTPARAGRAPHE000107"></a>Le formatage du résultat dépend des caractéristiques des langues définies dans la description du projet (onglet "Langues"). Il est ainsi possible de modifier les unités de taille de fichier et / ou le séparateur de décimales.




## DonneIdentifiant (Fonction)

***En anglais : GetIdentifier***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Calcule un identifiant unique de type entier pour un exécutable ou une session donné. Ces données ne sont pas mémorisées : elles sont perdues dès la fin de l'exécution de l'application ou de la session.

<a name="sample_code"></a>

## Syntaxe

`<Résultat> = DonneIdentifiant()`
---

**`<Résultat> : (Entier)`**

Identifiant généré. Le premier appel à la fonction **DonneIdentifiant** renvoie 1, le second 2, ...  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Limites
<a name="limites_ELTPARAGRAPHE000078"></a>

- L'identifiant généré est un entier, il est donc limité à 231. En cas de dépassement, l'identifiant devient négatif.

- La fonction **DonneIdentifiant** peut être utilisée dans une tâche parallèle.





### Calcul d'un identifiant
<a name="calcul_identifiant_ELTPARAGRAPHE000095"></a>Il est également possible d'obtenir un identifiant global (GUID) grâce à la fonction [DonneGUID](../WDLang1/1000018857.md).




## TCDVersExcel (Fonction)

***En anglais : PVTToExcel***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Crée un fichier Excel avec les données d'un champ Tableau croisé dynamique.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
sFichier est une chaîne

// Construit le nom du fichier
sFichier = SysRep(srAppData) + ["\"] + "StatsParPays.xls"

TCDVersExcel(TCD_Statistiques, sFichier)
LanceAppliAssociée(SysRep(srAppData) + ["\"] + "StatsParPays.xls")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = TCDVersExcel(<Champ Tableau croisé dynamique> , <Fichier XLS> [, <Options>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le fichier a été créé, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour obtenir plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Champ Tableau croisé dynamique> : (Nom de champ)`**

Nom du champ Tableau croisé dynamique à manipuler.

**`<Fichier XLS> : (Chaîne de caractères)`**

Nom et chemin complet du fichier Excel à créer. Si le fichier existe déjà, il est remplacé par le nouveau fichier.

**`<Options> : (Constante optionnelle)`**

Paramétrage du fichier XLS à créer.


|   |   |
| --- | --- |
| *taAvecMiseEnForme* | Exporte la couleur et la police du champ Tableau croisé dynamique. |
| *taSansTitre* | Seules les données sont copiées dans le fichier. |
| *taTitreColonnes*<br>(valeur par défaut) | Le titre des colonnes est inséré dans le fichier. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Les fichiers Excel sont limités à 65536 lignes et 256 colonnes si l'extension utilisée est ".xls". Cette limite peut être dépassée en utilisant l'extension ".xlsx" afin de créer des fichiers au format Excel 2007. Attention : dans ce cas, les anciennes versions de Excel devront être équipées du "Pack de compatibilité Office 2007", téléchargeable gratuitement sur le site de Microsoft afin de pouvoir ouvrir les documents générés.

- Le fichier créé pourra être ouvert avec Excel version 97 ou supérieure ou avec OpenOffice Calc version 2 ou supérieure.

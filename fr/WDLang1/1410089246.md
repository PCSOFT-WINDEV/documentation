
## Espagne.VérifieNIF (Fonction)

***En anglais : Spain.CheckNIF***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Teste la validité d'un numéro NIF (numéro d'identification d'entreprise en Espagne).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Test d'une variable
MonNuméroNIF est une chaîne = "B82661935"
Espagne.VérifieNIF(MonNuméroNIF)	// renvoie Vrai

// Test d'un champ
SAI_NuméroNIF = "ABC"
Espagne.VérifieNIF(SAI_NuméroNIF)	// renvoie Faux
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = Espagne.VérifieNIF(<Numéro à vérifier>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'élément spécifié contient un numéro NIF valide.

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Numéro à vérifier> : (Nom de champ ou variable de type chaîne)`**

Numéro NIF à vérifier. Ce paramètre peut correspondre : 

- au nom d'une variable de type chaîne.

- au nom d'un champ.


## EnumérationDepuisValeur (Fonction)

***En anglais : EnumerationFromValue***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie une valeur d'énumération connue par sa valeur associée.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Description de l'énumération
UneEnumération est une Enumération
	Valeur1 = "A"
	Valeur2 = "B"
FIN

MonEnumération est UneEnumération

// MonEnumération est affecté avec la valeur "Valeur1"
MonEnumération = EnumérationDepuisValeur(UneEnumération, "A")
// La trace donne Valeur1 -> "A"
Trace(MonEnumération..Nom + " -> " + MonEnumération..Valeur)

// MonEnumération est affecté avec la valeur "Valeur2"
MonEnumération = EnumérationDepuisValeur(UneEnumération, "B")
// la trace donne Valeur2 -> "B"
Trace(MonEnumération..Nom + " -> " + MonEnumération..Valeur)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = EnumérationDepuisValeur(<Enumération> , <Valeur> [, <Défaut>])`
---

**`<Résultat> : (Variable de type Enumération)`**

Variable de type [Enumération](../Motscles/1514067.md). 

**`<Enumération> : (Variable de type Enumération)`**

Nom de la variable de type [Enumération](../Motscles/1514067.md) à manipuler. 

**`<Valeur> : (Variant)`**

Valeur simple (chaîne, entier, réel, booléen, ...) de l'énumération. 

**`<Défaut> : (Variable optionnelle)`**

Valeur par défaut à donner à l'énumération si la &lt;Valeur&gt; n'est pas trouvée. 

Si ce paramètre n'est pas spécifié et si la &lt;Valeur&gt; n'est pas trouvée, une erreur fatale (exception) est générée.

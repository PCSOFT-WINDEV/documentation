
## SysRep (Fonction)

***En anglais : SysDir***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le chemin d'un répertoire du système.





![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Dans cette version, cette fonction permet uniquement de renvoyer le répertoire à utiliser par les applications pour stocker leurs données (spécifique à l'utilisateur).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) 
```wl
// Récupérer le chemin du répertoire système
ResRep = SysRep(srSystème)
```

<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Répertoire temporaire
<a name="repertoire_temporaire_ELTPARAGRAPHE000874"></a>Pour connaître le répertoire temporaire de Windows, utilisez la fonction [fRepertoireTemp](../WDLang1/3036044.md).


<a name="NOTE0_2"></a>


### Corbeille
<a name="corbeille_ELTPARAGRAPHE000885"></a>La "Corbeille" correspond à un répertoire virtuel géré par Windows. La fonction **SysRep** ne permet donc pas de connaître le chemin de la "Corbeille".


<a name="NOTE0_3"></a>
![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) 

### Fonctionnalité d'application requise
<a name="fonctionnalite_application_requise_ELTPARAGRAPHE000900"></a>L'utilisation de cette fonction entraîne la déclaration de plusieurs fonctionnalités d'application dans l'assistant de génération de l'application.
Fonctionnalités requises : 

- Bibliothèque d'images

- Vidéothèque

- Médiathèque

- Bibliothèque de documents

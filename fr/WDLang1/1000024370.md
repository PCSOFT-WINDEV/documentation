


## &lt;Variable Image&gt;.SymétrieHorizontale (Fonction)

***En anglais : &lt;Image variable&gt;.HorizontalSymmetry***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Retourne l'image contenue dans une variable de type [Image](../WDLang1/1000019650.md) par rapport à un axe horizontal (symétrie par rapport à un axe horizontal).<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=dSymetrieHorizontale.gif)


![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Les dessins](../WDLang1/3029035.md). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonImage est une Image = "Test.bmp"
// Symétrie de l'image contenue dans la variable Image par rapport à un axe horizontal
MonRésultat est un booléen
MonRésultat = MonImage.SymétrieHorizontale()
SI MonRésultat = Faux ALORS Erreur("L'opération a échoué")
IMG_MonDessin = MonImage
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Variable Image>.SymétrieHorizontale()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la symétrie a été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Variable Image> : (Variable de type Image)`**

Nom de la variable de type [Image](../WDLang1/1000019650.md) à manipuler.

La symétrie est réalisée par rapport à l'axe horizontal médian de l'image. L'image est directement modifiée.

<a name="NOTE0_1"></a>

## Remarques


### Fin du dessin
<a name="fin_dessin_ELTPARAGRAPHE000346"></a>Si la fonction [&lt;Variable Image&gt;.FinDessin](../WDLang1/1000024351.md) est appelée dans le même traitement que la fonction **&lt;Variable Image&gt;.SymétrieHorizontale**, la transformation de l'image ne sera pas affichée.

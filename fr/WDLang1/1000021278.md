
## TâcheParallèleAnnule (Fonction)

***En anglais : ParallelTaskCancel***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Termine immédiatement l'exécution de la tâche courante : la tâche passe dans l'état *tpeAnnulée*.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Termine l'exécution de la tâche courante
TâcheParallèleAnnule()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`TâcheParallèleAnnule()`
---



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- La fonction **TâcheParallèleAnnule** termine immédiatement l'exécution de la tâche en cours.

- La fonction **TâcheParallèleAnnule** peut être utilisée uniquement dans le code exécuté par une tâche. L'appel de la fonction termine immédiatement l'exécution de la tâche.

- La fonction [TâcheParallèleDemandeAnnulation](../WDLang1/1000021213.md) ne peut pas être utilisée dans le code exécuté par une tâche.


## Procédure WLangage appelée par la fonction AppliDétecteMAJDisponible



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Procédure (également appelée "Callback") appelée par la fonction [AppliDétecteMAJDisponible](../WDLang1/1410087705.md) lorsqu'une mise à jour de l'application en cours d'exécution est détectée. Cette procédure peut être une procédure globale ou interne.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
AppliDétecteMAJDisponible(MiseAJourDisponible)
	PROCEDURE INTERNE MiseAJourDisponible(MiseAJour, Version)
		SI MiseAJour = appMAJDispoObligatoire ALORS
			ToastAffiche("Une mise à jour obligatoire est disponible (" + Version + ").")
		FIN
	FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`AppliDétecteMAJDisponible_Callback(<Niveau de mise à jour> , <Version>)`
---

**`<Niveau de mise à jour> : (Constante de type Entier)`**

Type de la mise à jour disponible : 



|   |   |
| --- | --- |
| *appMAJDispoFacultative* | Une mise à jour facultative est disponible. |
| *appMAJDispoObligatoire* | Une mise à jour obligatoire est disponible. |





**`<Version> : (Chaîne de caractères)`**

Numéro de la version disponible. 







<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Personnalisation de la détection automatique
<a name="personnalisation_detection_automatique_ELTPARAGRAPHE000048"></a>Cette procédure permet de personnaliser l'action effectuée lors de la détection de la mise à jour (pour plus de détails, consultez [Détection automatique des mises à jour disponibles](../WDLang1/1410087705.md)). Son action remplace l'affichage de la notification Windows. Cette procédure peut par exemple : 

- ne rien faire : dans ce cas, l'utilisateur ne sera pas informé qu'une mise à jour est disponible. 

- faire une action spécifique (afficher un toast explicatif par exemple). Cette action remplacera la notification Windows affichée par défaut.


## gRectangle (Fonction)

***En anglais : gRectangle***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Construit une séquence de caractères permettant de dessiner un rectangle.

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Dans une fenêtre, cette séquence de caractères permet de dessiner un rectangle :  

- dans un élément d'un champ Liste ou Combo.

- dans la barre de titre et/ou de message d'une fenêtre. 

- dans un champ Libellé. 







<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 
```wl
// Dessin d'un rectangle dans un élément du champ Liste "LISTE_Figure"
// Le fond du rectangle est vert clair et le bord jaune clair
ListeAjoute(LISTE_Figure, gRectangle(0, 0, 25, 18, VertClair, JauneClair))
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = gRectangle(<X1> , <Y1> , <X2> , <Y2> [, <Couleur du fond> [, <Couleur du trait>]])`
---

**`<Résultat> : (Chaîne de caractères)`**

Séquence de caractères permettant de dessiner un rectangle. 

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Cette séquence de caractères peut être : 

- utilisée dans les fonctions [ListeAjoute](../WDLang1/3049004.md), [ListeInsère](../WDLang1/3049002.md) ou [ListeModifie](../WDLang1/3049009.md).

- affectée à un champ Libellé. 

- utilisée dans la barre de titre ou de message d'une fenêtre. 




**`<X1> : (Réel)`**

Abscisse (en pixels) du coin haut gauche du rectangle à dessiner.

**`<Y1> : (Réel)`**

Ordonnée (en pixels) du coin haut gauche du rectangle à dessiner.

**`<X2> : (Réel)`**

Abscisse (en pixels) du coin bas droit du rectangle à dessiner.

**`<Y2> : (Réel)`**

Ordonnée (en pixels) du coin bas droit du rectangle à dessiner.

**`<Couleur du fond> : (Entier ou constante (optionnel))`**

Couleur du fond du rectangle. Cette couleur peut correspondre à :

- une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)),

- une couleur TSL (obtenue avec la fonction [TSL](../WDLang1/3029057.md)),

- [une couleur prédéfinie du WLangage](../WDLang5/3010002.md).




Si ce paramètre n'est pas précisé, la couleur du fond correspond à la couleur du fond du texte en cours (définie par la fonction [gFond](../WDLang1/3029037.md)).

**`<Couleur du trait> : (Entier ou constante (optionnel))`**

Couleur du trait du rectangle. Cette couleur peut correspondre à :

- une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)),

- une couleur TSL (obtenue avec la fonction [TSL](../WDLang1/3029057.md)),

- [une couleur prédéfinie du WLangage](../WDLang5/3010002.md).




Si ce paramètre n'est pas précisé, la couleur du trait correspond à la couleur du texte en cours (définie par la fonction [gStylo](../WDLang1/3029039.md)).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Coordonnées
<a name="coordonnees_ELTPARAGRAPHE000133"></a>Toutes les coordonnées sont exprimées en pixels par rapport au coin haut gauche de l'élément dans lequel le rectangle doit être dessiné (par exemple, coin haut gauche de la ligne du champ Liste ou Combo).


<a name="NOTE0_2"></a>




### Largeur/Hauteur du rectangle
<a name="largeurhauteur_rectangle_ELTPARAGRAPHE000142"></a>La largeur du rectangle de référence vaut &lt;X2&gt; - &lt;X1&gt;.

La hauteur du rectangle de référence vaut &lt;Y2&gt; - &lt;Y1&gt;.

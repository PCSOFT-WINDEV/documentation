


## &lt;Table&gt;.SaisieEnCours (Fonction)

***En anglais : &lt;Table&gt;.InputInProgress***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Indique si un champ Table ou Table hiérarchique est en cours de saisie.

**Remarque** : Si l'utilisateur utilise le mode Recherche dans l'entête de colonne (loupe), le champ Table ou Table hiérarchique est considéré en cours de saisie.








<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Si l'utilisateur n'est pas en train de saisir dans le champ
SI TABLE_TABLE1.SaisieEnCours() = Faux ALORS
	// Met à jour le champ Table
	TABLE_TABLE1.Affiche()
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Champ Table>.SaisieEnCours()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le champ est en cours de saisie (c'est-à-dire si l'utilisateur est en train de saisir des informations dans le champ), 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Champ Table> : (Nom de champ)`**

Nom du champ à manipuler. Ce champ peut correspondre à : 

- un champ Table. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) un champ Table hiérarchique. 
	![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) Les champs Table hiérarchique ne sont pas disponibles.

<a name="NOTE0_1"></a>

## Remarques


### Conditions d'utilisation
<a name="conditions_utilisation_ELTPARAGRAPHE000216"></a>Cette fonction est disponible sur : 

- un champ Table ou Table hiérarchique fichier.

- un champ Table ou Table hiérarchique par programmation.

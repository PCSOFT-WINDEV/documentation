
## fChargeRessource (Fonction)

***En anglais : fLoadResource***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Charge le contenu d'une ressource depuis une bibliothèque.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Charge les conditions générales de vente depuis la WDL
CGV est un Buffer = fChargeRessource("CGV.PDF")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = fChargeRessource(<Ressource>)`
---

**`<Résultat> : (Buffer)`**

Ressource demandée. 

**`<Ressource> : (Chaîne de caractères)`**

Nom de la ressource à charger. 
Rappel : Les ressources sont des fichiers ajoutés à la bibliothèque au moment de la génération de l'application, de la bibliothèque, du composant externe, ...

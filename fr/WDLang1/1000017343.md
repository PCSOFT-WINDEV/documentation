


## &lt;Calendrier&gt;.SelectPlus (Fonction)

***En anglais : &lt;Calendar&gt;.SelectPlus***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Sélectionne une date dans un champ Calendrier. Si le champ Calendrier est multisélections, ajoute la date spécifiée à la sélection courante.






<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ajoute le 3 Octobre 2008 à la sélection
CAL_Calendrier1.SelectPlus("20081003")
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Champ Calendrier>.SelectPlus(<Date à sélectionner>)`
---

**`<Champ Calendrier> : (Nom de champ)`**

Nom du champ Calendrier manipulé.

**`<Date à sélectionner> : (Chaîne de caractères)`**

Date au format AAAAMMJJ à sélectionner ou à ajouter à la sélection courante (cas d'un calendrier multisélection). 
Si cette date est déjà sélectionnée, la fonction est sans effet. 

Si la &lt;Date à sélectionner&gt; n'est pas affichée sur le calendrier, le calendrier se positionne automatiquement sur cette date.

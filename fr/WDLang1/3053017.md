***En anglais : MemNext***

<a name="Utilisation"></a>
<a name="description"></a>
Positionne sur l'élément suivant d'une zone mémoire et renvoie la valeur de retour de cet élément.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Positionne sur l'élément suivant de la zone mémoire "MaZoneMémoire" 
// et renvoie la valeur de cet élément
ResSuivant = MemSuivant("MaZoneMémoire")
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = MemSuivant(<Zone mémoire>)`
---

**`<Résultat> : (Tout type)`**



- Si l'élément suivant existe : valeur de retour de l'élément suivant, définie par les fonctions [MemAjoute](../WDLang1/3053002.md) ou [MemModifie](../WDLang1/3053007.md).

- Si l'élément suivant n'existe pas : une chaîne vide ("").




**`<Zone mémoire> : (Chaîne de caractères)`**

Nom de la zone mémoire, défini par la fonction [MemCrée](../WDLang1/3053005.md).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Tri de la zone mémoire
<a name="tri_zone_memoire_ELTPARAGRAPHE000088"></a>Les tris effectués par la fonction [MemTrie](../WDLang1/3053020.md) sont pris en compte lors de l'appel à la fonction **MemSuivant**. 

Si aucun tri n'a été demandé, la fonction **MemSuivant** retourne la valeur de l'élément suivant selon l'ordre des éléments ajoutés avec la fonction [MemAjoute](../WDLang1/3053002.md).


<a name="NOTE0_2"></a>


### Existence de la zone mémoire
<a name="existence_zone_memoire_ELTPARAGRAPHE000264"></a>L'existence de la zone mémoire n'est pas vérifiée.

Si la zone mémoire spécifiée n'existe pas, une erreur du WLangage est générée lors de l'exécution de la fonction **MemSuivant**. 

Pour vérifier l'existence d'une zone mémoire, utilisez la fonction [MemExiste](../WDLang1/3053006.md).
<a name="NOTE0_3"></a>




### Recherche dans une zone mémoire
<a name="recherche_dans_une_zone_memoire_ELTPARAGRAPHE000116"></a>Si la fonction [MemRecherche](../WDLang1/3053015.md) est utilisée avant l'appel de la fonction **MemSuivant**, la fonction **MemSuivant** se positionnera sur le prochain élément recherché.


<a name="NOTE0_4"></a>


### Équivalence
<a name="equivalence_ELTPARAGRAPHE000133"></a>Les deux syntaxes suivantes sont équivalentes :


```txt
Res = MemSuivant(<Zone mémoire>)

Res = MemRécupère(<Zone mémoire>, MemEnCours(<Zone mémoire>)+1)
```




## &lt;Editeur HTML&gt;.FormateSelection (Fonction)

***En anglais : &lt;HTML editor&gt;.FormatSelection***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie le style de la sélection dans un champ Editeur HTML.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
EDHTML_MonEditeurHTML.FormateSelection(htmlItalique)
EDHTML_MonEditeurHTML.FormateSelection(htmlTaillePolice, "14px")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Champ Editeur HTML>.FormateSelection(<Attribut> [, <Valeur>])`
---

**`<Champ Editeur HTML> : (Nom de champ)`**

Nom du champ Editeur HTML manipulé.

**`<Attribut> : (Constante de type Entier)`**

Attribut de style à modifier :


|   |   |
| --- | --- |
| *htmlAlignement* | Aligne horizontalement la sélection. Si cette constante est utilisée, le paramètre &lt;Valeur&gt; permet de spécifier l'alignement voulu. Les valeurs possibles sont : <br><br>- *chCentre* : La sélection est centrée. <br><br>- *chDroite* : La sélection est alignée à droite. <br><br>- *chGauche* : La sélection est alignée à gauche. <br><br><br> |
| *htmlBarré* | Barre la sélection. Si la sélection est déjà barrée, la barre est supprimée. Si une partie de la sélection est barrée, la sélection toute entière est barrée.  |
| *htmlCouleur* | Change la couleur de la sélection. Si cette constante est utilisée, le paramètre &lt;Valeur&gt; permet de spécifier la couleur voulue. Le paramètre &lt;Valeur&gt; peut correspondre à : <br><br>- un entier. La couleur sera alors une couleur RVB (par exemple obtenue avec la fonction [RVB](../WDLang1/3029012.md)). <br><br>- une chaîne. La couleur sera alors au format HTML (par exemple "00ff00" ou "green"). <br><br><br> |
| *htmlCouleurFond* | Change la couleur de fond de la sélection. Si cette constante est utilisée, le paramètre &lt;Valeur&gt; permet de spécifier la couleur de fond voulue. Le paramètre &lt;Valeur&gt; peut correspondre à : <br><br>- un entier. La couleur sera alors une couleur RVB (par exemple obtenue avec la fonction [RVB](../WDLang1/3029012.md)). <br><br>- une chaîne. La couleur sera alors au format HTML (par exemple "00ff00" ou "green"). <br><br><br> |
| *htmlCréeLien* | Crée un lien sur l'URL spécifiée dans le paramètre &lt;Valeur&gt;. |
| *htmlExposant* | Passe la sélection en mode "exposant". Si la sélection est déjà en mode "exposant", le mode "exposant" est supprimé. Si une partie de la sélection est en mode "exposant", la sélection toute entière passe en mode "exposant".   |
| *htmlGras* | Passe la sélection en gras. Si la sélection est déjà en gras, le gras est supprimé. Si une partie de la sélection est en gras, la sélection toute entière passe en gras.   |
| *htmlIndice* | Passe la sélection en mode "indice". Si la sélection est déjà en mode "indice", le mode "indice" est supprimé. Si une partie de la sélection est en mode "indice", la sélection toute entière passe en mode "indice". |
| *htmlItalique* | Passe la sélection en italique. Si la sélection est déjà en italique, l'italique est supprimé. Si une partie de la sélection est en italique, la sélection toute entière passe en italique.   |
| *htmlNomPolice* | Modifie le nom de la police utilisée pour la sélection. Si cette constante est utilisée, le paramètre &lt;Valeur&gt; permet de spécifier le nom de la famille de la police à utiliser. Par exemple "Arial, Helvetica, sans-serif". |
| *htmlNomStyleCss* | Modifie le style CSS utilisé par la sélection. Si cette constante est utilisée, le paramètre &lt;Valeur&gt; permet de spécifier le nom du style CSS à utiliser. |
| *htmlSouligné* | Souligne la sélection. Si la sélection est déjà soulignée, le souligné est supprimé. Si une partie de la sélection est soulignée, la sélection toute entière passe en souligné.   |
| *htmlSupprimeLien* | Supprime le lien de la sélection.   |
| *htmlTaillePolice* | Modifie la taille de la police utilisée dans la sélection. Si cette constante est utilisée, le paramètre &lt;Valeur&gt; permet de spécifier la taille de la police à utiliser. Cette taille est exprimée en unités HTML. Par exemple : "12px" ou "2.5em".  |
| *htmlTaillePoliceMoins* | Réduit la taille de la police de la sélection. |
| *htmlTaillePolicePlus* | Augmente la taille de la police de la sélection. |



**`<Valeur> : (Variant optionnel)`**

Valeur à utiliser en fonction de la constante utilisée dans le paramètre &lt;Attribut&gt;.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Après l'utilisation de cette fonction, l'événement "Modification de la sélection" du champ est automatiquement exécuté (sauf si la fonction est utilisée dans cet événement). 

- La modification n'est pas immédiate : il faut attendre l'appel de l'événement "Modification de la sélection" pour relire la valeur du champ.

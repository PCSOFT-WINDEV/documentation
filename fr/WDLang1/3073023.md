
## SysRécupèreDC (Fonction)

***En anglais : SysGetDC***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Récupère le DC (Device Context) d'une fenêtre, d'un champ ou de l'écran. Ce DC peut ensuite être utilisé pour manipuler la fenêtre, le champ, ... avec les API Windows (notamment les fonctions de dessin de l'API Windows).

**Remarque** : Si la fonction **SysRécupèreDC** a déjà été appelée et si le DC précédent n'a pas été libéré par la fonction [SysLibèreDC](../WDLang1/3073021.md), la fonction **SysRécupèreDC** libère automatiquement l'ancien DC.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Récupération du DC de la fenêtre "Microsoft Word - Ventes 2000.DOC"
ResHandle = SysFenHandle("Microsoft Word - Ventes 2000.DOC")
ResRécupèreDC = SysRécupèreDC(ResHandle)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = SysRécupèreDC(<Handle>)`
---

**`<Résultat> : (Entier)`**



- DC de la fenêtre, du champ ou de l'écran, 

- 0 en cas d'erreur.




**`<Handle> : (Entier)`**



- Handle de la fenêtre ou du champ à manipuler.

- 0 pour récupérer le DC de l'écran.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Le handle d'une fenêtre est obtenu :

- avec la fonction [Handle](../WDLang1/3015005.md) pour une fenêtre WINDEV.

- avec la fonction [SysFenHandle](../WDLang1/3073007.md) pour une fenêtre non créée avec WINDEV.




Le handle d'un champ est obtenu avec la fonction [Handle](../WDLang1/3015005.md).

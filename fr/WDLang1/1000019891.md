


## Moyenne (Fonction)

***En anglais : Mean***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Calcule la moyenne de plusieurs éléments : 

- éléments d'un tableau, 

- valeurs numériques, ...





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Moyenne des éléments d'un tableau d'entiers 
MonTableau est un tableau d'entiers = [1,2,3]
MaMoyenne est un numérique
MaMoyenne = Moyenne(MonTableau) // MaMoyenne = 2
```



<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Calculer la moyenne des éléments d'un tableau à une dimension (types numériques)

`<Résultat> = Moyenne(<Tableau>)`
---

**`<Résultat> : (Numérique)`**

Moyenne des éléments numériques du tableau. 

**`<Tableau> : (Variable de type Tableau)`**

Nom du tableau à une dimension de types numériques contenant les éléments dont la moyenne doit être calculée. 


<a name="SYNTAXE2"></a>

### Calculer la moyenne des éléments d'un tableau à une dimension (types structurés)

`<Résultat> = Moyenne(<Tableau> , <Membre>)`
---

**`<Résultat> : (Numérique)`**

Moyenne des valeurs des membres. 

**`<Tableau> : (Variable de type Tableau)`**

Nom du tableau à une dimension de types structurés. Les types structurés autorisés sont les suivants : classe, classe dynamique, structure, structure dynamique.

Remarque : Dans un tableau de classes dynamiques ou de structures dynamiques, si l'élément n'est pas alloué, il est ignoré.

**`<Membre> : (Chaîne de caractères)`**

Nom du membre contenant les valeurs à utiliser pour le calcul de la moyenne. 


<a name="SYNTAXE3"></a>

### Calculer la moyenne des éléments d'une colonne d'un tableau à 2 dimensions

`<Résultat> = Moyenne(<Tableau> , <Colonne>)`
---

**`<Résultat> : (Numérique)`**

Moyenne des valeurs de la colonne. 

**`<Tableau> : (Variable de type Tableau)`**

Nom du tableau à 2 dimensions de type numérique. 

**`<Colonne> : (Entier)`**

Indice de la colonne du tableau contenant les valeurs à utiliser pour le calcul de la moyenne. 


<a name="SYNTAXE4"></a>

### Calculer la moyenne de plusieurs valeurs

`<Résultat> = Moyenne(<Valeur 1> , <Valeur 2> [, <Valeur 3> [, <Valeur N>]])`
---

**`<Résultat> : (Numérique)`**

Moyenne des valeurs.

**`<Valeur 1> : (Numérique)`**

Première valeur à utiliser pour le calcul de la moyenne. 

**`<Valeur 2> : (Numérique)`**

Seconde valeur à utiliser pour le calcul de la moyenne. 

**`<Valeur 3> : (Numérique)`**

Troisième valeur à utiliser pour le calcul de la moyenne. 
$~ ... |

**`<Valeur N> : (Numérique)`**

Dernière valeur à utiliser pour le calcul de la moyenne.

## Remarques


### Types gérés
<a name="types_geres_ELTPARAGRAPHE000154"></a>Les types gérés pour les tableaux de numériques et pour les membres des tableaux de type structurés sont les suivants : 

- tous les types entier (sur 1, 2, 4 ou 8 octets, signés ou non signés + entier système).

- les types réel (sur 4 ou 8 octets), monétaire et numérique.

- le type Date.





### Moyenne de durées
<a name="moyenne_durees_ELTPARAGRAPHE000165"></a>Si la fonction **Moyenne** est utilisée pour calculer une moyenne de durées, tous les paramètres doivent être de type Durée.



**Enchaînement de fonctions**
Il est possible d'enchaîner les différentes fonctions de manipulation d'un tableau. 
Les fonctions suivantes peuvent être utilisées dans un enchaînement : 

- [Filtre](../WDLang1/1000024845.md)

- [Transforme](../WDLang1/1000024734.md)

- [Union](../WDLang1/1000025408.md)

- [Intersection](../WDLang1/1000025407.md)

- [Distinct](../WDLang1/1000025406.md). 


Cet enchaînement pourra être utilisé comme source d'une instruction POUR TOUT ou se terminer par une des fonctions suivantes :

- [Agrège](../WDLang1/1000024744.md)

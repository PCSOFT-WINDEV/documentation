
## UUEncode (Fonction)

***En anglais : UUEncode***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Encode une chaîne de caractères selon l'algorithme UUEncode. Cette chaîne encodée pourra ensuite être décodée avec la fonction [UUDécode](../WDLang1/1000018864.md). 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Encodage d'une chaîne
Res = UUEncode("Le numéro de mon compte est 0123456789")
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = UUEncode(<Chaîne à encoder>)`
---

**`<Résultat> : (Chaîne de caractères)`**

Chaîne encodée selon l'algorithme UUEncode. 

**`<Chaîne à encoder> : (Chaîne de caractères)`**

Texte à encoder.

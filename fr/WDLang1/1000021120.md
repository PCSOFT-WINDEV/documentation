


## &lt;Diagramme de Gantt&gt;.AjouteLien (Fonction)

***En anglais : &lt;Gantt chart&gt;.AddLink***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Crée un lien de type "Fin à Début" entre deux tâches affichées dans une colonne d'un champ Table (ou Table hiérarchique) de type Diagramme de Gantt.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Création des tâches puis du lien
t1 est une TâcheGantt
t1.ID = "T1"
t2 est une TâcheGantt
t2.ID = "T2"
COL_Gantt.AjouteLien(t1.ID, t2.ID)

// Autre syntaxe en spécifiant le type du lien
COL_Gantt.AjouteLien(t1.ID, t2.ID, tlgFinADébut)
```


<a name="1000021075_Exemple2"></a>

```wl
// Création d'un lien via une variable de type LienGantt
Lien est un LienGantt
Lien.IDSource = "T1"
Lien.IDDestination = "T2"
Lien.Type = tlgFinADébut
COL_Gantt.AjouteLien(Lien)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Créer un lien entre deux tâches existantes

`<Colonne Gantt>.GanttAjouteLien(<Identifiant Tâche Source> , <Identifiant Tâche Destination> [, <Type du lien>])`
---

**`<Colonne Gantt> : (Nom de champ)`**

Nom de la colonne (d'un champ Table ou Table hiérarchique) de type Diagramme de Gantt à manipuler.

**`<Identifiant Tâche Source> : (Chaîne de caractères)`**

Identifiant de la tâche source du lien.

**`<Identifiant Tâche Destination> : (Chaîne de caractères)`**

Identifiant de la tâche destination du lien.

**`<Type du lien> : (Constante de type Entier)`**

Type du lien à ajouter : 


|   |   |
| --- | --- |
| *tlgDébutADébut* | Lien de type "Début à Début" : La tâche destination ne peut pas commencer avant que la tâche source ne soit commencée. |
| *tlgDébutAFin* | Lien de type "Début à Fin" : La tâche destination ne peut pas se terminer avant que la tâche source ne soit commencée (cas rare). |
| *tlgFinADébut*<br>(valeur par défaut) | Lien de type "Fin à Début" : La tâche destination ne peut pas commencer avant que la tâche source ne soit terminée. |
| *tlgFinAFin* | Lien de type "Fin à Fin" : La tâche destination ne peut pas se terminer avant que la tâche source ne soit terminée. |




<a name="SYNTAXE2"></a>

### Créer un lien via une variable de type LienGantt

`GanttAjouteLien(<Colonne Gantt> , <Lien>)`
---

**`<Colonne Gantt> : (Nom de champ)`**

Nom de la colonne (d'un champ Table ou Table hiérarchique) de type Diagramme de Gantt à manipuler.

**`<Lien> : (Variable de type LienGantt)`**

Nom de la variable de type [LienGantt](../WDLang1/1000021076.md) décrivant le lien à créer.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Si le lien existe déjà, la fonction ne fait rien.

- Si un des identifiants n'existe pas, le lien est inactif : le lien n'est pas affiché et n'a aucun effet. 

- Selon leurs types, les liens sont dessinés sur les bords de début ou de fin des tâches.


## iCalendarVersAgenda (Fonction)

***En anglais : iCalendarToOrganizer***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Importe des données au format iCalendar dans un champ Agenda.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
siCal est une chaîne ANSI
siCal = fChargeBuffer("calendar.ics")

iCalendarVersAgenda(AGD_MonAgenda, siCal)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`iCalendarVersAgenda(<Champ Agenda> , <Données iCalendar>)`
---

**`<Champ Agenda> : (Nom de champ)`**

Nom du champ Agenda à manipuler. Ce champ peut correspondre : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) au champ Agenda d'une fenêtre. 

- au champ Agenda d'un état.




**`<Données iCalendar> : (Chaîne de caractères ANSI au format UTF-8)`**

Données au format iCalendar. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Pour exporter les données d'un champ Agenda au format iCalendar, utilisez la fonction [AgendaVersiCalendar](../WDLang1/1000021919.md). 

- ![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) Cette fonction est disponible uniquement dans les sites WEBDEV pour Linux. Cette fonction n'est pas disponible dans les applications WINDEV pour Linux.

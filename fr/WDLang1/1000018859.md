
## RéseauNomDomaine (Fonction)

***En anglais : NetworkDomainName***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le nom du domaine auquel est associée la machine.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
NomDomaine est une chaîne
NomDomaine = RéseauNomDomaine()
Info("Votre poste est associé au domaine :  " + NomDomaine)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = RéseauNomDomaine([<Options>])`
---

**`<Résultat> : (Chaîne de caractères)`**

Nom du domaine de l'ordinateur.

**`<Options> : (Constante de type Entier optionnelle)`**

Type de domaine considéré : 


|   |   |
| --- | --- |
| *resDomaineDNS*<br>(Valeur par défaut) | Nom du dommaine DNS associé à la machine. |
| *resDomaineNetBIOS* | Nom du domaine NetBIOS associé à la machine. |

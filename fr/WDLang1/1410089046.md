


## &lt;Variable FichierDisque&gt;.LitLigneExpressionRégulière (Fonction)

***En anglais : &lt;DiskFile variable&gt;.ReadLineRegularExpression***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Lit une ligne d'un fichier externe et récupère dans des variables les morceaux de cette ligne en fonction d'une expression régulière.

Cette fonction permet notamment de simplifier l'analyse des fichiers de log.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ouverture d'un fichier externe
MonFichier est un FichierDisque
MonFichier = fOuvre("C:\MesRépertoires\AccesData.log", foLecture)
SI PAS ErreurDétectée ALORS
       // Lecture d'une ligne dans ce fichier
	// Format : 20001221  - Achat
	sDate est une chaîne
	sInfo est une chaîne
	MonFichier.LitLigneExpressionRégulière("([0-8]*) - ([A-Za-z]*)", sDate, sInfo)
	...
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Fichier manipulé>.LitLigneExpressionRégulière(<Format> , <Variable 1> [, <Variable 2> [... [, <Variable N>]]])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si les données ont été lues et récupérées. 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Fichier manipulé> : (Variable de type FichierDisque)`**

Nom de la variable de type [FichierDisque](../WDLang1/1410088883.md) initialisée avec la fonction [fOuvre](../WDLang1/3036036.md) ou [fCrée](../WDLang1/3036017.md).

**`<Format> : (Chaîne de caractères)`**

Format de référence, exprimé sous forme d'une expression régulière. Ce format utilise les caractères suivants : 



|   |   |
| --- | --- |
| A B C - / _ | Lettres et symboles à vérifier |
| [A-Z] ou [0-9] | Intervalle de lettres, de chiffres ou de symboles à vérifier |
| \* | 0 ou plusieurs symboles à vérifier |
| + | 1 ou plusieurs symboles à vérifier |
| ( ) | Délimitation d'une partie du format à extraire |
| { } | Nombre de répétitions autorisées pour l'expression précédente (de la forme {min, max}) |



**`<Variable 1> : (Chaîne de caractères, Entier, ...)`**

Variables qui seront automatiquement initialisées avec les différentes parties du format à extraire. Chaque partie est définie par les caractères '(' et ')'.

**`<Variable 2> : (Chaîne de caractères, Entier, ...)`**

Variables qui seront automatiquement initialisées avec les différentes parties du format à extraire. Chaque partie est définie par les caractères '(' et ')'.



**`<Variable N> : (Chaîne de caractères, Entier, ...)`**

Variables qui seront automatiquement initialisées avec les différentes parties du format à extraire. Chaque partie est définie par les caractères '(' et ')'.

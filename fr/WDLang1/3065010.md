
## RegistreSupprimeClé (Fonction)

***En anglais : RegistryDeleteKey***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime une sous-clé dans la base de registre de Windows.

**Remarque** : La fonction **RegistreSupprimeClé** supprime la sous-clé, les valeurs et les descendants de la clé spécifiée.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
Chemin est une Chaîne
// Connaître le nom de la clé "root"
Chemin = RegistrePremièreSousClé()
// Supprimer la clé pour mon application
Chemin = Chemin + "\MonAppli\"
SI RegistreSupprimeClé(Chemin) = Faux ALORS
	Erreur("Erreur lors de la suppression de la clé")
	Retour
FIN
```


<a name="Exemple2"></a>



```wl
// Supprime la sous-clé "HKEY_LOCAL_MACHINE\SYSTEM\Appli"
ResSupClé = RegistreSupprimeClé("HKEY_LOCAL_MACHINE\SYSTEM\Appli")
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = RegistreSupprimeClé([<Mode d'accès>, ] <Chemin de la sous-clé>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la sous-clé a été supprimée ou si la clé n'existe pas, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Mode d'accès> : (Constante de type Entier)`**

Mode d'accès à la base de registre :



|   |   |
| --- | --- |
| *registreMode32* | Mode d'accès forcé pour accéder à la base de registre comme un programme 32 bits. |
| *registreMode64* | Mode d'accès forcé pour accéder à la base de registre comme un programme 64 bits. |
| *registreModeAuto*<br>(Valeur par défaut) | Mode d'accès à la base de registre automatique : <br><br>- une application 32 bits exécutée sur un système 32 bits manipule la base de registre comme un programme 32 bits.<br><br>- une application 32 bits exécutée sur un système 64 bits manipule la base de registre à partir de la branche suivante :<br>	<br><pre><code>HKEY_LOCAL_MACHINE\SOFTWARE\Wow6432Node</code></pre><br><br><br>- une application 64 bits exécutée sur un système 64 bits manipule la base de registre comme un programme 64 bits.<br><br><br> |



**`<Chemin de la sous-clé> : (Chaîne de caractères)`**

Chemin de la sous-clé à supprimer.

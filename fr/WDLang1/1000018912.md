


## DonneCouleur (Fonction)

***En anglais : GetColor***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet d'obtenir une suite de couleurs harmonieuses sans avoir à utiliser une table de couleurs.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Remplit les données du champ Graphe
POUR I = 1 A 12
	// Ajoute le chiffre d'affaires du mois numéro I
	grAjouteDonnée(GRF_MonGraphe, I, tabValeurs[I])
	
	// Affecte une couleur à la série du mois numéro I
	grCouleurSérie(GRF_MonGraphe, I, DonneCouleur())	
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = DonneCouleur([<Indice> [, <Série>]])`
---

**`<Résultat> : (Entier)`**

Couleur au format RVB.



**`<Indice> : (Entier optionnel)`**

Numéro de la couleur à récupérer. 
Si ce paramètre n'est pas spécifié, la couleur renvoyée est une couleur qui n'a jamais été utilisée. 
Si ce paramètre est spécifié, pour un indice donné, la même couleur sera toujours renvoyée.


**`<Série> : (Constante optionnelle de type Entier)`**

Constante indiquant le style de série de couleur à récupérer : 


|   |   |
| --- | --- |
| *coulSérieDéfaut*<br>(Valeur par défaut) | La couleur renvoyée est choisie parmi une série de couleurs standard. |
| *coulSérieFoncée* | La couleur renvoyée est choisie parmi une série de couleurs plus foncées (adaptée au champ TreeMap par exemple). |
| *coulSérieGabarit* | La couleur renvoyée est choisie parmi la série de couleurs définie pour le gabarit en cours. <br><br> |
| *coulSériePalette* | La couleur renvoyée est choisie parmi la série de couleurs définie pour la palette en cours. <br><br>![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Cette constante n'est pas disponible. |
| *coulSériePastel* | La couleur renvoyée est choisie parmi une série de couleurs plus claires. |


## TableurModifieNom (Fonction)

***En anglais : SpreadsheetModifyName***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie le nom utilisé pour identifier une cellule d'un champ Tableur.

Remarques : 

- Les noms peuvent ensuite être utilisés dans les formules pour qu'elles soient plus faciles à comprendre. Par exemple : "=PrixHT\*TVA" est plus clair que = "=A1\*A2". 

- Les noms sont proposés dans la complétion en saisie de formules. 



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Crée une formule qui référence A1 et A2 avec des noms plus clairs
TableurAjouteNom(TBLR_TABLEUR1, "PrixHT", "$A$1")
TableurAjouteNom(TBLR_TABLEUR1, "TVA", "$A$2")
TableurAjouteFormule(TBLR_TABLEUR1, "A3", "PrixHT*TVA")
TableurModifieNom(TBLR_TABLEUR1, "TVA", "TVAIntra" ,"$A$2") 
// La formule de A3 devient "PrixHT*TVAIntra"
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`TableurModifieNom(<Champ Tableur> , <Nom de la cellule> [, <Nouveau nom> [, <Cellule> [, <Portée> [, <Commentaire>]]]])`
---

**`<Champ Tableur> : (Nom de champ)`**

Nom du champ Tableur à manipuler. 

**`<Nom de la cellule> : (Chaîne de caractères)`**

Nom actuel de la cellule à modifier. 

**`<Nouveau nom> : (Chaîne de caractères optionnelle)`**

Nouveau nom de la cellule. 

Attention : 

- Ce paramètre ne doit pas correspondre à un nom de cellule (par exemple "A1") ou à un nom de fonction standard (par exemple "SOMMME").

- Ce paramètre ne doit pas contenir d'espaces. 

- Si le nom spécifié est invalide ou existe déjà, la fonction **TableurModifieNom** échoue : la variable [ErreurDétectée](../WDLang1/3087001.md) est à <u><u><u><u>Vrai</u></u></u></u> et la fonction [ErreurInfo](../WDLang1/3013008.md) permet de connaître la cause de l'erreur. 




**`<Cellule> : (Chaîne de caractères optionnelle)`**

Cellule à nommer. 
Attention : Il est nécessaire d'utiliser des coordonnées absolues pour désigner la cellule (par exemple : "$A$1"). 
En effet, dans un nom de cellule, les coordonnées sont relatives. L'utilisation de la notation "A1", "A2", ... peut entraîner des erreurs de références circulaires. 

**`<Portée> : (Entier optionnel)`**

Numéro de la feuille dans laquelle le nom est utilisable. Si ce paramètre est omis ou vaut 0, le nom est utilisable dans toutes les feuilles.

**`<Commentaire> : (Chaîne de caractères optionnelle)`**

Commentaire associé au nom de la cellule (255 caractères maximum).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Les noms ne sont pas sensibles à la casse (majuscules/minuscules).

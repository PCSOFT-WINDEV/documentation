
## CouleurTeinte (Fonction)

***En anglais : ColorHue***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Calcule la teinte d'une couleur.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SelectCouleur(MaCouleur)
Info("Teinte : " + CouleurTeinte(MaCouleur) + "°", ...
	"Luminosité : " + CouleurLuminosité(MaCouleur) + "%", ...
	"Saturation : " + CouleurSaturation(MaCouleur) + "%")
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = CouleurTeinte(<Couleur>)`
---

**`<Résultat> : (Entier)`**

Valeur de la teinte de la couleur spécifiée. Cette valeur est exprimée en tant qu'angle de 0 à 360° dans le cercle chromatique (0 = Rouge).

**`<Couleur> : (Entier)`**

Couleur manipulée. Cette couleur peut correspondre à :

- une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)),

- [une couleur prédéfinie du WLangage](../WDLang5/3010002.md).

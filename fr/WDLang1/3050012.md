


## Fonctions de gestion des valeurs binaires
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


Les fonctions de gestion des valeurs binaires sont les suivantes :



|   |   |
| --- | --- |
| [ETBinaire](../WDLang1/3050010.md) | Renvoie le résultat du ET logique (bit à bit) entre deux valeurs. |
| [NONBinaire](../WDLang1/3050011.md) | Renvoie le résultat du NON logique (bit à bit) d'une valeur. |
| [OUBinaire](../WDLang1/3050038.md) | Renvoie le résultat du OU logique (bit à bit) entre deux valeurs. |
| [OUExclusifBinaire](../WDLang1/3050023.md) | Renvoie le résultat du OU Exclusif logique (bit à bit) entre deux valeurs. |




## TableFin (Fonction)

***En anglais : TableEnd***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Arrête le parcours manuel du fichier de données lié à un champ Table fichier. Après l'exécution de la fonction **TableFin**, les enregistrements situés après l'enregistrement en cours dans le champ Table ne seront pas lus.

<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Arrêt du parcours du fichier lié au champ Table "TABLE_TableProduit"
TableFin(TABLE_TableProduit)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`TableFin([<Champ Table>])`
---

**`<Champ Table> : (Nom de champ optionnel)`**

Nom du champ Table à manipuler.

Si ce paramètre n'est pas spécifié ou correspond à une chaîne vide (""), le champ Table manipulé est le champ Table auquel appartient l'événement en cours.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Conditions d'utilisation
<a name="conditions_utilisation_ELTPARAGRAPHE000060"></a>

- La fonction **TableFin** peut être utilisée indifféremment sur un champ Table monosélection ou multisélection.

- La fonction **TableFin** ne doit pas être utilisée sur les champs Table fichier utilisant un parcours automatique.





<a name="NOTE0_2"></a>


### Evénements du champ Table
<a name="evenements_champ_table_ELTPARAGRAPHE000080"></a>La fonction **TableFin** peut être utilisée uniquement dans les événements du champ Table suivants :

- Initialisation.

- Lecture de l'enregistrement précédent.

- Lecture de l'enregistrement suivant.



<a name="NOTE0_3"></a>


### Réinitialisation du champ Table
<a name="reinitialisation_champ_table_ELTPARAGRAPHE000103"></a>Après l'appel de la fonction **TableFin** dans un événement d'un champ Table, ré-initialisez l'affichage des données du champ Table. Par exemple :

```wl
TableAffiche(TABLE_TABLE1, taInit)
```

<a name="NOTE0_4"></a>


### Visualisation des enregistrements entre deux valeurs
<a name="visualisation_des_enregistrements_entre_deux_valeurs_ELTPARAGRAPHE000121"></a>Pour visualiser dans un champ Table les enregistrements correspondant à un intervalle de valeurs, il est conseillé d'utiliser la fonction [HFiltre](../WDLang4/3044100.md) (et non la fonction **TableFin**).


## Procédure WLangage appelée par la fonction KanbanFiltre



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Procédure WLangage (également nommée "callback") appelée par la fonction [KanbanFiltre](../WDLang1/1410089534.md) pour effectuer un filtre avancé. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = KanbanFiltre_Callback(<Carte>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la carte manipulée respecte la procédure de filtre et doit apparaître dans le résultat du filtre, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 




**`<Carte> : (Variable de type kbCarte)`**

Variable de type [kbCarte](../WDLang1/1410089167.md) manipulée par la procédure de filtre.




## FenEdCréeNote (Fonction)

***En anglais : WinEdCreateNote***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Crée un champ Note repositionnable par programmation. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
NOTE_Manote est un Champ
NOTE_Manote <- FenEdCréeNote(MaFenêtre, "Nouvelle note")
NOTE_Manote.CouleurFond = BleuClair
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`[<Résultat> <- ] FenEdCréeNote(<Fenêtre> , <Texte initial> [, <X> [, <Y>]])`
---

**`<Résultat> : (Variable de type Champ)`**

Le champ Note créé peut être manipulé via une variable de type [Champ](../WDLang1/1000020967.md). Dans ce cas, l'affectation doit être réalisée avec l'opérateur **'<-'**. 

Si la fonction **FenEdCréeNote** est utilisée directement, le champ Note est créé dans la fenêtre et il est visible par défaut. 

**`<Fenêtre> : (Nom de fenêtre)`**

Nom ou alias de la fenêtre dans laquelle le champ Note sera créé. 
Si ce paramètre n'est pas précisé ou correspond à une chaîne vide (""), le champ Note sera créé dans la fenêtre en cours.

**`<Texte initial> : (Chaîne de caractères)`**

Texte affiché dans le champ Note.

**`<X> : (Entier optionnel)`**

Abscisse du champ Note à créer (position sur l'axe des X) en pixels. Position horizontale du coin haut gauche du champ par rapport au coin haut gauche de la zone client de la fenêtre (c'est-à-dire la fenêtre sans les barres de titre et de menu ni le cadre). Ce paramètre correspond à la propriété [XInitial](../Proprietes/2510140.md). 

Si ce paramètre n'est pas précisé, l'abscisse du champ Note correspond à la position X de la souris. 

Il est ensuite possible de modifier cette valeur avec la propriété [X](../Proprietes/2510135.md).

**`<Y> : (Entier optionnel)`**

Ordonnée du champ Note à créer (position sur l'axe des Y) en pixels. Position verticale du coin haut gauche du champ par rapport au coin haut gauche de la zone client de la fenêtre (c'est-à-dire la fenêtre sans les barres de titre et de menu ni le cadre). Ce paramètre correspond à la propriété [YInitial](../Proprietes/2510087.md). 

Si ce paramètre n'est pas précisé, l'ordonnée du champ Note correspond à la position Y de la souris. 

Il est ensuite possible de modifier cette valeur avec la propriété [Y](../Proprietes/2510137.md).

## Remarques


### Caractéristiques du champ Note repositionnable
<a name="caracteristiques_champ_note_repositionnable_ELTPARAGRAPHE000072"></a>

- La création peut être annulée par Ctrl + Z. 

- Le champ Note repositionnable créé est une note persistante : si la fenêtre est fermée puis ré-ouverte, la note sera restaurée. 
	Pour créer un champ Note non persistant, utilisez la fonction [ChampCrée](../WDLang1/1000020876.md) avec la constante *typNoteRepositionnable*. 

- Le champ Note repositionnable créé sera sauvé lors de l'appel aux fonctions : 

	- [FenEdSauve](../WDLang1/1000021848.md),

	- [FenEdSauveNote](../WDLang1/1000022120.md),

	- [NoteSauveTout](../WDLang1/1000022396.md). 






<a name="NOTE0_2"></a>


### Equivalences
<a name="equivalences_ELTPARAGRAPHE000096"></a>

- Cette fonction est équivalente à la création d'une note dans la fenêtre par l'utilisateur final via la FAA (Fonctionnalité Automatique de votre Application). Pour plus de détails, consultez [Note repositionnable (FAA)](../Editeurs/1000021820.md). 

- Utilisée pour créer un champ Note repositionnable dans la fenêtre en cours, cette fonction est équivalente à la fonction [NoteCrée](../WDLang1/1000022395.md).

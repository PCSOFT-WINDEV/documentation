


## Fonctions de gestion des champs
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


Les fonctions de gestion des champs sont les suivantes :



|   |   |
| --- | --- |
| [ActiveXEvénement](../WDLang1/3025010.md) | Associe une procédure écrite en WLangage à un événement d'un contrôle ActiveX. |
| [AjouteLien](../WDLang1/1000019038.md) | Ajoute un lien sur une portion de texte dans un champ de saisie. |
| [AscenseurAffiche](../WDLang1/3025018.md) | Affiche ou non un ascenseur dans un champ WINDEV (champ de saisie ou liste), même si cet ascenseur n'a pas été sélectionné lors de la description du champ. |
| [AscenseurEstVisible](../WDLang1/1000021644.md) | Permet de savoir si un ascenseur d'un champ dans une application WINDEV est visible. |
| [AscenseurPosition](../WDLang1/1000021057.md) | Permet de gérer la position des ascenseurs : <br><br>- Modifie ou récupère la position d'un des ascenseurs (horizontal ou vertical) d'un champ dans une application WINDEV ou WINDEV Mobile. <br><br>- Récupère la position d'un des ascenseurs (horizontal ou vertical) d'un champ dans un site WEBDEV.<br><br><br> |
| [AscenseurPositionMax](../WDLang1/1000021826.md) | Récupère la position maximale d'un des ascenseurs (horizontal ou vertical) d'un champ dans une application WINDEV. |
| [BulleActive](../WDLang1/3025014.md) | Active ou désactive le mécanisme d'affichage des bulles d'aide sur les champs. |
| [BulleCouleur](../WDLang1/3025015.md) | Modifie les couleurs utilisées pour l'affichage des bulles d'aide. |
| [BulleDélai](../WDLang1/3025051.md) | Permet de modifier :<br><br>- le délai à attendre avant l'affichage des bulles d'aide.<br><br>- la durée d'affichage des bulles d'aide des champs.<br><br><br> |
| [BulleLargeurMax](../WDLang1/1000022257.md) | Modifie la largeur maximale des bulles d'aide. |
| [BulleMarge](../WDLang1/1000021844.md) | Modifie la marge du texte dans les bulles d'aide. |
| [ChampActif](../WDLang1/1000017377.md) | Dégrise un champ ou un groupe de champs d'une fenêtre. Pendant cette opération, une animation sur les champs peut être effectuée. |
| [ChampAlias](../WDLang1/3025016.md) | Identifie, modifie ou restaure la liaison fichier d'un champ ou d'un ensemble de champs. |
| [ChampCauseEntréeSortie](../WDLang1/3025013.md) | Indique l'origine de l'entrée dans le champ. |
| [ChampClone](../WDLang1/3025017.md) | Crée un nouveau champ à partir d'un champ existant. |
| [ChampContextuel](../WDLang1/3025012.md) | Identifie le champ sur lequel le menu contextuel a été ouvert. |
| [ChampCrée](../WDLang1/1000020876.md) | Crée un nouveau champ de n'importe quel type dans une fenêtre par programmation. |
| [ChampEnCours](../WDLang1/3025019.md) | Renvoie le nom du champ en cours de saisie. |
| [ChampErreur](../WDLang1/3025036.md) | Personnalise la valeur affichée dans les champs en cas d'impossibilité d'afficher la valeur. |
| [ChampExiste](../WDLang1/3025035.md) | Teste l'existence d'un champ (ou d'un groupe de champs) dans une fenêtre, dans une page ou dans un état. |
| [ChampGrisé](../WDLang1/1000017375.md) | Grise un champ ou un groupe de champs. Pendant cette opération, une animation sur les champs peut être effectuée. |
| [ChampInfoXY](../WDLang1/1000020503.md) | Renvoie le nom du champ se trouvant à une position donnée. |
| [ChampInvisible](../WDLang1/1000017369.md) | Rend invisible un champ ou un groupe de champs dans une fenêtre. Pendant cette opération, une animation sur les champs peut être effectuée. |
| [ChampPrécédent](../WDLang1/3025033.md) | Indique le nom du champ en saisie avant le champ en cours. |
| [ChampPremier](../WDLang1/3025022.md) | Indique le nom du premier champ spécifié dans l'ordre de navigation pour la fenêtre spécifiée. |
| [ChampSansEspace](../WDLang1/3025008.md) | Indique si les espaces à droite de la valeur présente dans le champ sont supprimés. |
| [ChampSuivant](../WDLang1/3025020.md) | Renvoie le nom du champ suivant en saisie. |
| [ChampSupprime](../WDLang1/3025037.md) | Supprime définitivement un champ d'une fenêtre, d'une page ou d'un état. |
| [ChampSurvol](../WDLang1/3025021.md) | Identifie le champ survolé par le curseur de la souris. |
| [ChampTypeInfo](../WDLang1/1000020520.md) | Renvoie l'icône correspondant au type de champ spécifié. |
| [ChampVisible](../WDLang1/1000017338.md) | Rend visible un champ ou un groupe de champs dans une fenêtre. Pendant cette opération, une animation sur les champs peut être effectuée. |
| [ChangeStyle](../WDLang1/1000017171.md) | Modifie dynamiquement le style d'un champ. |
| [ChargeParamètre](../WDLang1/3025047.md) | Lit une valeur persistante. |
| [ColonneEnCours](../WDLang1/3025006.md) | Renvoie :<br><br>- soit l'indice de la colonne indicée en cours dans un champ Table (Table par programmation ou Table fichier).<br><br>- soit le numéro de la colonne en cours de saisie dans le champ Table indiqué (Table par programmation ou Table fichier).<br><br><br> |
| [CoordonnéeChampImageVersImage](../WDLang1/1000026037.md) | Convertit les coordonnées d'un rectangle exprimées par rapport au champ Image vers les coordonnées correspondantes exprimées par rapport à l'image affichée dans le champ. |
| [CoordonnéeEcranVersEditeur](../WDLang1/1000021276.md) | Convertit une taille ou une position donnée dans le dpi natif de l'écran de l'appareil sur lequel s'exécute l'application vers le dpi de référence sous l'éditeur (160 dpi). |
| [CoordonnéeEditeurVersEcran](../WDLang1/1000021277.md) | Convertit une taille ou une position donnée dans le dpi de référence sous l'éditeur (160 dpi) vers le dpi natif de l'écran de l'appareil sur lequel s'exécute l'application. |
| [CoordonnéeImageVersChampImage](../WDLang1/1000026038.md) | Convertit les coordonnées d'un rectangle exprimées par rapport à l'image vers les coordonnées correspondantes exprimées par rapport au champ Image qui l'affiche. |
| [CouleurSélection](../WDLang1/3025003.md) | Change les caractéristiques du bandeau de sélection dans un champ Table, un champ Combo ou un champ Liste. |
| [DictionnaireCharge](../WDLang1/1000022958.md) | Indique et charge le dictionnaire correspondant à une langue donnée. |
| [DonneFocus](../WDLang1/3025001.md) | Donne le focus :<br><br>- à un champ (d'une fenêtre, d'une page ou d'un frameset).<br><br>- à une fenêtre.<br><br><br> |
| [EcranPremier](../WDLang1/3025039.md) | Définit le prochain champ qui sera en saisie dans la fenêtre en cours. |
| [EnumèreChamp](../WDLang1/3025005.md) | Renvoie le nom du Nème champ de l'élément considéré. |
| [FormePolygone](../WDLang1/1000022000.md) | Modifie la forme actuellement affichée dans un champ Forme pour obtenir une forme personnalisée (en spécifiant les coordonnées des points du polygone voulu). |
| [HTMLNavigue](../WDLang1/3025042.md) | Lance une opération de navigation dans un champ d'affichage HTML. |
| [IndiceEnCours](../WDLang1/3025040.md) | Renvoie selon le champ spécifié :<br><br>- le numéro de la ligne en cours affichée dans le champ Table en cours,<br><br>- le numéro de l'option d'un champ Interrupteur ayant le focus,<br><br>- le numéro de la répétition en cours dans le champ Zone répétée en cours,<br><br>- l'indice du champ indicé en cours.<br><br><br> |
| [IndiceSuivant](../WDLang1/3025030.md) | Renvoie le numéro de la ligne suivante affichée dans le champ Table en cours. |
| [InitParamètre](../WDLang1/3025044.md) | Initialise la gestion des valeurs persistantes. |
| [LienCouleur](../WDLang1/1000022354.md) | Modifie la couleur des liens dans les champs de saisie et les champs Libellé. |
| [Occurrence](../WDLang1/3025009.md) | Renvoie le nombre d'occurrences d'un champ dans une fenêtre, une page ou un groupe de champs. |
| [OuvreMenuContextuel](../WDLang1/3025029.md) | Ouvre automatiquement un menu contextuel pour le champ, la fenêtre ou la page en cours. |

| [RAZ](../WDLang1/3025026.md) | Ré-initialise :<br><br>- soit tous les champs de la fenêtre ou de la page en cours (y compris les groupes de champs, les superchamps et les fenêtres internes).<br><br>- soit les champs présents dans un groupe de champs, un superchamp, une fenêtre interne ou une page interne.<br><br><br> |
| [RepriseSaisie](../WDLang1/3025027.md) | Arrête le traitement en cours et force la reprise de saisie dans le champ, la fenêtre ou la page spécifié(e). |

- ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=WDPersistance.gif) ***Exemples didactiques (WINDEV)*** : **WD Persistance** <br>Cet exemple montre l'utilisation des fonctions InitParamètre, ChargeParamètre, SauveParamètre.<br>Ces fonctions permettent de configurer la sauvegarde de champs, de variables et de tout autre paramètre.<br>Cet exemple est décomposé en trois parties :<br>	- La configuration<br>	- La gestion manuelle de variables (Fonctions SauveParamètre et ChargeParamètre)<br>	- L'optimisation<br>1°) Configuration<br>	<br>	La fenêtre de configuration vous permet de modifier l'emplacement de la sauvegarde des paramètres (Base de registre, fichier INI, fichier XML) ainsi que son chemin (clé de registre ou le chemin du fichier).<br>2°) Gestion Manuelle de variables<br>	La gestion manuelle vous permet de sauvegarder le contenu de variables grâce à la fonction SauveParamètre et de les restaurer grâce à la fonction ChargeParamètre. Chaque paramètre est identifié par un nom. Ces valeurs sont sauvegardées à l'emplacement défini dans la fenêtre de configuration.<br>3°) L'optimisation<br>	Cette fenêtre vous explique comment optimiser votre code pour éviter les lenteurs dues à la restauration des valeurs sauvegardées. En effet, lors de la restauration de la valeur d'un champ, son code de modification est exécuté. Si vous avez des traitements longs (Requêtes paramétrées ou autres), cette fenêtre vous montre comment accélérer le chargement de la fenêtre.

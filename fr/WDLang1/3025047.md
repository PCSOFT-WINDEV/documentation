
## ChargeParamètre (Fonction)

***En anglais : LoadParameter***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Lit une valeur persistante. La valeur persistante a été sauvegardée grâce à la fonction [SauveParamètre](../WDLang1/3025046.md) : 

- soit dans la base de registre (ou le fichier équivalent sur la plateforme d'exécution). Par exemple : 

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) Dans une application Linux, les paramètres sont sauvegardés dans le dossier $HOME/.local/share.




- ![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) soit dans le fichier spécifié avec la fonction [InitParamètre](../WDLang1/3025044.md).













<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Sauvegarde
SauveParamètre("LOGIN", SAI_Saisie1)

// Restauration
SAI_Saisie1 = ChargeParamètre("LOGIN", "")
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = ChargeParamètre(<Nom du paramètre> [, <Valeur par défaut>])`
---

**`<Résultat> : (Chaîne de caractères ou entier)`**



- Valeur du paramètre recherché. 

- Si le paramètre recherché n'est pas trouvé, correspond à la &lt;Valeur par Défaut&gt; ou à chaîne vide ("") si aucune valeur par défaut n'a été spécifiée.




**`<Nom du paramètre> : (Chaîne de caractères)`**

Nom du paramètre à rechercher. Ce paramètre ne doit pas correspondre à une chaîne vide.

![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Si le mode d'initialisation choisi est la base de registre, la fonction **ChargeParamètre** n'est sensible ni à la casse, ni à l'accentuation du nom du paramètre (Paramètre et parametre sont équivalents).

**`<Valeur par défaut> : (Chaîne de caractères optionnelle ou entier optionnel)`**

Valeur par défaut du paramètre. Cette valeur sera renvoyée si la valeur du paramètre n'est pas trouvée.

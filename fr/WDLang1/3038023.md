
## TitreEnCours (Fonction)

***En anglais : CurrentTitle***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie le titre de la fenêtre en cours (connu par la fonction [FenEnCours](../WDLang1/3038033.md)) et la barre de titre.

**Rappel** : Par défaut, le titre de la fenêtre est celui défini sous l'éditeur.






<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Remplacer le titre de la fenêtre en cours
TitreEnCours(gFond(RougeClair) + gStylo(Noir) + "Modification d'une fiche client")
```
<a name="Exemple2"></a>

```wl
// Modifier le titre de la fenêtre en cours
// Ajoute "en cours"
TitreEnCours(EOT + gFond(RougeClair) + gStylo(Noir) + " en cours")
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Modifier le titre de la fenêtre

`TitreEnCours(<Nouveau titre de la fenêtre>)`
---

**`<Nouveau titre de la fenêtre> : (Chaîne de caractères)`**

Nouveau titre de la fenêtre en cours.  


<a name="SYNTAXE2"></a>

### Modifier le titre de la fenêtre et la barre de titre

`TitreEnCours([<Couleur de Fond> [, <Couleur Texte>, ]] <Nouveau titre de la fenêtre>)`
---

**`<Couleur de Fond> : (Entier optionnel)`**

Couleur utilisée pour le fond de la barre de titre. Cette couleur peut correspondre à :

- une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)),

- une couleur TSL (obtenue avec la fonction [TSL](../WDLang1/3029057.md)),

- [une couleur prédéfinie du WLangage](../WDLang5/3010002.md).




**`<Couleur Texte> : (Entier optionnel)`**

Couleur utilisée pour le texte de la barre de titre. Cette couleur peut correspondre à :

- une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)),

- une couleur TSL (obtenue avec la fonction [TSL](../WDLang1/3029057.md)),

- [une couleur prédéfinie du WLangage](../WDLang5/3010002.md).




**`<Nouveau titre de la fenêtre> : (Chaîne de caractères)`**

Nouveau titre de la fenêtre en cours. Si la constante EOT est précisée, le nouveau titre sera ajouté au titre existant.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Titre des boîtes de dialogues
<a name="titre_des_boites_dialogues_ELTPARAGRAPHE000111"></a>Les fenêtres ouvertes par les fonctions [Avertissement](../WDLang1/3021009.md), [Confirmer](../WDLang1/3021007.md), [Dialogue](../WDLang1/3021015.md), [Erreur](../WDLang1/3021013.md), [ErreurInfo](../WDLang1/3013008.md), [Info](../WDLang1/3021011.md), [OKAnnuler](../WDLang1/3021004.md) ou [OuiNon](../WDLang1/3021005.md) ont le même titre que la fenêtre précédente.

Pour modifier le titre de la prochaine fenêtre à ouvrir, utilisez la fonction [TitreSuivant](../WDLang1/3038043.md).


<a name="NOTE0_2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 

### Titre des fenêtres détourées
<a name="titre_des_fenetres_detourees_ELTPARAGRAPHE000150"></a>Sur les fenêtres détourées, la fonction **TitreEnCours** permet également de modifier le titre de la fenêtre. Cependant, ce titre n'est pas affiché dans la fenêtre.

Si vous avez utilisé un gabarit détouré, le titre de la fenêtre est affiché dans un champ de type libellé. Le contenu de ce champ n'est pas pris en compte par la fonction **TitreEnCours**. Il est nécessaire de le modifier spécifiquement. 
<a name="NOTE0_3"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) 

### Présentation du titre
<a name="presentation_titre_ELTPARAGRAPHE000167"></a>Les [fonctions de manipulation des chaînes graphiques](../WDLang1/3029033.md) (fonctions commençant par "g") permettent de :

- ajouter des dessins au titre,

- modifier la mise en forme d'un titre (couleur de fond, couleur de texte, etc.) avec les fonctions [gFond](../WDLang1/3029037.md) et [gStylo](../WDLang1/3029039.md).




Par exemple : pour modifier la couleur de texte et la couleur de fond d'un titre :


```wl
TitreEncours(EOT + gFond(RVB(0, 255, 255)) + gStylo(RVB(255, 0, 0)) + "Test de couleur")
```


**Cas particulier sous Windows Vista et 7** : Les fenêtres utilisant le look "Aero system" ne peuvent pas avoir de titre graphique : la partie graphique est ignorée. Si "Aero" n'est pas actif, le titre graphique sera affiché normalement.
<a name="NOTE0_4"></a>




### Équivalence
<a name="equivalence_ELTPARAGRAPHE000193"></a>Le code suivant : 

```txt
TitreEnCours(<Nouveau titre de la fenêtre>)
```

est équivalent à la propriété [Titre](../Proprietes/2510090.md) :


```txt
MaFenêtre.Titre = <Nouveau titre de la fenêtre>
```

### Affichage du titre
<a name="affichage_titre_ELTPARAGRAPHE000220"></a>Pour afficher un titre, il est nécessaire que la fenêtre possède un champ Action Bar.


## PoidsFaible (Fonction)

***En anglais : LoWord***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie les deux octets de poids faible d'un entier.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
Res est un entier
Res = 0x000A001F
Trace(PoidsFort(Res))	// Affiche 10 (correspond à 0x000A)
Trace(PoidsFaible(Res))	// Affiche 31 (correspond à 0x001F)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = PoidsFaible(<Valeur> [, <Taille>])`
---

**`<Résultat> : (Entier sans signe sur 2 octets)`**

Les deux octets de poids faible.

**`<Valeur> : (Entier)`**

Valeur dont le poids faible est recherché. 
Si cette valeur est d'un type numérique différent de entier, cette valeur sera automatiquement convertie en entier. 
Si cette valeur est une chaîne de caractères, une erreur sera affichée.

**`<Taille> : (Entier optionnel)`**

Taille de l'entier retourné : 

- 1 pour un entier sur 1 octet, 

- 2 pour un entier sur 2 octets, 

- 4 pour un entier sur 4 octets.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- L'opérateur [&](../Motscles/1512002.md) permet de connaître l'adresse d'une variable.

- La fonction **PoidsFaible** peut être utilisée pour récupérer le poids faible d'un entier retourné par la fonction [API](../WDLang1/3014005.md) ou [AppelDLL32](../WDLang1/3014003.md).

- La fonction [PoidsFort](../WDLang1/3014017.md) permet de récupérer le poids fort d'un entier.

- Pour construire un entier de 4 octets à partir de deux entiers de 2 octets, utilisez la fonction [ConstruitEntier](../WDLang1/3062016.md).

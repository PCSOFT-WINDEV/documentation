
## Suisse.VérifieNAVS (Fonction)

***En anglais : Switzerland.CheckOASI***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Teste la validité d'un numéro NAVS (numéro d'assurance vieillesse survivants de Suisse).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Test d'une variable
MonNuméroNAVS est une chaîne = "7566523572040"
Suisse.VérifieNAVS(MonNuméroNAVS)		// renvoie Vrai

// Test d'un champ
SAI_NuméroNAVS = "ABC"
Suisse.VérifieNAVS(SAI_NuméroNAVS)	// renvoie Faux
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = Suisse.VérifieNAVS(<Numéro à vérifier>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'élément spécifié contient un numéro NAVS valide.

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Numéro à vérifier> : (Nom de champ ou variable de type chaîne)`**

Numéro NAVS à vérifier. Ce paramètre peut correspondre :

- au nom d'une variable de type chaîne.

- au nom d'un champ.

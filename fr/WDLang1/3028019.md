
## DDESujet (Fonction)

***En anglais : DDETopic***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Identifie le sujet de la conversation associée à une connexion DDE.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ouverture d'une connexion DDE
NumConnexion = DDEConnecte(NomServeur, "MonSujet")

...

// Sujet de la conversation ?
Sujet = DDESujet(NumConnexion)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = DDESujet([<Identifiant connexion>])`
---

**`<Résultat> : (Chaîne de caractères)`**

Sujet de la conversation.

**`<Identifiant connexion> : (Entier optionnel)`**

Identifiant de la connexion renvoyé par la fonction [DDEConnecte](../WDLang1/3028015.md). Si ce paramètre n'est pas précisé, **DDESujet** utilise la dernière connexion manipulée.

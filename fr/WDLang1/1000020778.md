


## &lt;TimeLine&gt;.ChangeMode (Fonction)

***En anglais : &lt;TimeLine&gt;.ChangeMode***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie la période (c'est-à-dire le nombre de secondes, millisecondes, microsecondes, ...) affichée par un champ TimeLine.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Clic sur BTN_Secondes
// Affiche 2 minutes 
TL_TimeLine.ChangeMode(plnModeSeconde, 120)

TL_TimeLine.GranularitéDéplacement = 5
TL_TimeLine.GranularitéDurée = 1
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Champ TimeLine>.ChangeMode(<Mode d'affichage> [, <Etendue>])`
---

**`<Champ TimeLine> : (Nom de champ)`**

Nom du champ TimeLine manipulé.

**`<Mode d'affichage> : (Constante de type Entier)`**

Plage de temps à afficher dans le champ TimeLine : 



|   |   |
| --- | --- |
| *plnModeJournée* | Affiche le champ TimeLine en prenant comme résolution d'affichage la journée. |
| *plnModeMicroseconde* | Affiche le champ TimeLine en prenant comme résolution d'affichage la microseconde. |
| *plnModeMilliseconde* | Affiche le champ TimeLine en prenant comme résolution d'affichage la milliseconde. |
| *plnModeMois* | Affiche le champ TimeLine en prenant comme résolution d'affichage le mois. |
| *plnModeNbJourPersonnalisé* | Affiche le champ TimeLine sur le nombre de jours indiqué dans le paramètre &lt;Etendue&gt;. |
| *plnModeSeconde* | Affiche le champ TimeLine en prenant comme résolution d'affichage la seconde. |
| *plnModeSemaine* | Affiche le champ TimeLine sur une semaine de 5 jours ou sur une semaine de 7 jours. Le nombre de jours est indiqué dans le paramètre &lt;Etendue&gt;. |



**`<Etendue> : (Entier optionnel)`**

Etendue totale à afficher dans le champ TimeLine en prenant en compte la nouvelle unité.

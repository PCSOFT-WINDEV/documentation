
## Répète (Fonction)

***En anglais : RepeatString***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Concatène N répétitions d'une même chaîne de caractères ou d'un même buffer.








<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
Res = Répète("x", 4)     // Renvoie "xxxx"
Res = Répète("Boby", 2)  // Renvoie "BobyBoby"
Res = Répète("", 2)      // Renvoie ""
Res = Répète("", 0)      // Renvoie ""
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Concaténer N répétitions d'une même chaîne de caractères

`<Résultat> = Répète(<Chaîne initiale> , <Nombre de répétitions>)`
---

**`<Résultat> : (Chaîne de caractères)`**



- Nouvelle chaîne de caractères contenant la répétition voulue, 

- Chaîne vide ("") si &lt;Nombre de répétitions&gt; est négatif ou nul.




**`<Chaîne initiale> : (Chaîne de caractères)`**

Chaîne de caractères à répéter.

**`<Nombre de répétitions> : (Entier)`**

Nombre de fois où la &lt;Chaîne initiale&gt; doit être répétée.  


<a name="SYNTAXE2"></a>

### Concaténer N répétitions d'un même buffer

`<Résultat> = Répète(<Buffer initial> , <Nombre de répétitions>)`
---

**`<Résultat> : (Buffer)`**



- Nouveau buffer contenant la répétition voulue, 

- Buffer vide si &lt;Nombre de répétitions&gt; est négatif ou nul.




**`<Buffer initial> : (Buffer)`**

Buffer à répéter.

**`<Nombre de répétitions> : (Entier)`**

Nombre de fois où le &lt;Buffer initial&gt; doit être répété.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Fonction Répète et UNICODE (syntaxe 1)
<a name="fonction_docparampagetitleshort_unicode_syntaxe_1_ELTPARAGRAPHE000094"></a>Si la chaîne passée en paramètre est au format ANSI, le résultat sera au format ANSI.

Si la chaîne passée en paramètre est au format UNICODE, le résultat sera au format UNICODE.

**Remarque** : Si le résultat de la fonction **Répète** utilisée sur une chaîne ANSI est affecté à une chaîne UNICODE (et inversement), la conversion ANSI/UNICODE sera implicitement réalisée.

Pour plus de détails, consultez [Gestion de l'UNICODE](../WDLang1/3024038.md).

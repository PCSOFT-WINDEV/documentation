


## ChronoReprend (Fonction)

***En anglais : ChronoResume***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Redémarre un chronomètre précédemment arrêté par les fonctions [ChronoFin](../WDLang1/3027032.md) ou [ChronoPause](../WDLang1/1000018829.md). La durée chronométrée à partir de la fonction **ChronoReprend** sera cumulée à la durée précédemment calculée.








<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
DuréeTraitement est une Durée
ChronoDébut()
MaFonction1()
DuréeTraitement = ChronoPause()
Info("Le traitement MaFonction1 a duré " + DuréeTraitement) 
ChronoReprend() 
MaFonction2() 
DuréeTraitement = ChronoFin()
Info("Les traitements MaFonction1 et MaFonction2 ont duré " + DuréeTraitement)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Reprendre le chronométrage réalisé avec un numéro de chronomètre

`ChronoReprend([<Numéro du chronomètre>])`
---

**`<Numéro du chronomètre> : (Entier optionnel)`**

Identifiant du chronomètre utilisé. Si ce paramètre n'est pas précisé, sa valeur par défaut est 1.

### Reprendre le chronométrage réalisé avec une variable de type Chrono

`ChronoReprend(<Chronomètre>)`
---

**`<Chronomètre> : (Variable de type Chrono)`**

Nom de la variable de type [Chrono](../WDLang1/1000023560.md) correspondant au chronomètre à utiliser. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Divers
<a name="divers_ELTPARAGRAPHE000184"></a>Si la fonction **ChronoReprend** est utilisée sur un chronomètre déjà activé, cette fonction n'a aucun effet. 
<a name="NOTE0_2"></a>


### Erreur
<a name="erreur_ELTPARAGRAPHE000196"></a>Si la fonction [ChronoDébut](../WDLang1/3027031.md) n'a pas été appelée avant la fonction **ChronoReprend**, une erreur WLangage est affichée.
<a name="NOTE0_3"></a>


### Threads
<a name="threads_ELTPARAGRAPHE000211"></a>Les fonctions de chronométrage (Chronoxxx) concernant un même chronomètre doivent être utilisées dans un même thread.

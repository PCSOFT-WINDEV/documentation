


## EnModeTestAutomatique (Fonction)

***En anglais : InAutomaticTestMode***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de savoir si un test automatique est en cours ou non.

**Remarque** : Les tests automatiques peuvent être lancés uniquement sur des fenêtres, des classes ou des procédures.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI EnModeTestAutomatique() = Vrai ALORS 
	Trace("Variable NumClient : " + NumClient)
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = EnModeTestAutomatique()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si un test automatique est en cours, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Disponibilité de la fonction
<a name="disponibilite_fonction_ELTPARAGRAPHE000215"></a>La fonction **EnModeTestAutomatique** est disponible dans tous les produits pour permettre l'écriture d'un code multi-produits. Des fonctions ou procédures utilisant la fonction **EnModeTestAutomatique** peuvent être partagées entre plusieurs produits, sans que des erreurs de compilation soient générées.


## COMRécupèreInterface (Fonction)

***En anglais : COMQueryInterface***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Appelle la méthode QueryInterface de l'objet COM spécifié pour obtenir une interface spécifique de cet objet afin de l'utiliser avec la fonction [COMAppelleMéthode](../WDLang1/1000018832.md). 




<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = COMRécupèreInterface(<Objet COM> , <GUID>)`
---

**`<Résultat> : (Variable de type COMObjet)`**

Variable de type [COMObjet](../WDLang1/1000018983.md) qui pourra être utilisée avec la fonction [COMAppelleMéthode](../WDLang1/1000018832.md). 

**`<Objet COM> : (Objet COM)`**

Objet COM depuis lequel l'interface doit être récupérée. 

**`<GUID> : (Chaîne de caractères)`**

GUID de l'interface de l'objet à obtenir. Les chaînes de GUID doivent être composées de 32 caractères hexadécimaux séparés ou non par des tirets (par exemple : 265EBF50-D4A0-448D-ADC6-EAD971CFB537).

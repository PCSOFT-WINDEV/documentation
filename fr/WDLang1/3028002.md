
## DDEChaîne (Fonction)

***En anglais : DDEString***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Retourne les informations récupérées par la fonction [DDERécupère](../WDLang1/3028017.md).

**Rappel** : La fonction [DDERécupère](../WDLang1/3028017.md) permet de récupérer une donnée. Cette donnée est stockée dans **DDEChaîne**.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple
<a class="notetitle" target="_blank" href="$DOC$=1000003028002&name=ddechaine_fonction&product=WD">Voir des exemples supplémentaires</a>

```wl
// Exécute une action par le serveur
DDEExécute(NumConnexion, "EXE1")
SI DDERécupère(NumConnexion, "Item1") ALORS
	Info("La donnée récupérée est" + DDEChaîne())
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = DDEChaîne([<Identifiant connexion>])`
---

**`<Résultat> : (Chaîne de caractères)`**

Informations récupérées.

**`<Identifiant connexion> : (Entier optionnel)`**

Identifiant de la connexion renvoyé par la fonction [DDEConnecte](../WDLang1/3028015.md). Si ce paramètre n'est pas précisé, **DDEChaîne** utilise la dernière connexion manipulée.


## CaméraParamètre (Fonction)

***En anglais : CameraParameter***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie ou modifie la valeur d'un paramètre de la caméra de l'appareil.


	<B><B><B>Avertissement</B></B></B>

	 cette fonctionnalité n'est plus disponible



**Remarques** : 

- Pour connaître la liste des valeurs supportées par la caméra pour un paramètre donné, utilisez la fonction [CaméraListeParamètre](../WDLang1/1000022458.md).

- Cette fonction est équivalente à la fonction [VidéoParamètre](../WDLang1/1000019694.md). 





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Utilisation de la résolution maximale pour prendre une photo
sListeRésolution est une chaîne 
sListeRésolution = CaméraListeParamètre(vipRésolutionPhoto)
SI ErreurDétectée = Faux ALORS
	CaméraParamètre(vipRésolutionPhoto, ExtraîtChaîne(sListeRésolution, rangDernier, RC))
SINON
	Erreur("Erreur durant la récupération des résolutions supportées par la caméra.")
FIN

// Utilisation de la valeur maximale du zoom
sListeZoom est une chaîne
sListeZoom = CaméraListeParamètre(vipZoom)
SI ErreurDétectée = Faux ALORS
	nZoom est un entier = Val(ExtraîtChaîne(sListeZoom, rangDernier, RC))
	SI CaméraParamètre(vipZoom, nZoom) = Faux ALORS
		Erreur("La caméra ne peut pas zoomer.")
	FIN
FIN

// Géolocalisation des photos prises par la caméra
MaPosition est une géoPosition = GPSRécupèrePosition()
CaméraParamètre(vipGéoPosition, MaPosition)

// Réglage de la sensibilité ISO
CaméraParamètre(vipISO, 800)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Récupérer la valeur d'un paramètre de la caméra de l'appareil

`<Résultat> = CaméraParamètre(<Paramètre>)`
---

**`<Résultat> : (Type du paramètre récupéré)`**



- Valeur du paramètre, 

- Chaîne vide ("") si le paramètre spécifié n'est pas supporté par la caméra ou en cas d'erreur lors de la récupération des paramètres de la caméra.




En cas d'erreur, la variable [ErreurDétectée](../WDLang1/3087001.md) est à <u><u><u><u>Vrai</u></u></u></u>. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).

**`<Paramètre> : (Constante de type chaîne de caractères ou chaîne de caractères)`**

Paramètre pour lequel la valeur doit être récupérée. Ce paramètre peut correspondre : 

- soit à une des constantes prédéfinies. Ces constantes correspondent aux paramètres les plus courants. 

- soit à une Chaîne de caractères correspondant au nom du paramètre voulu. 
	**Attention** : le nom du paramètre est sensible à la casse et à l'accentuation. Consultez la documentation du SDK Android ou de l'appareil pour connaître le nom des paramètres supplémentaires supportés. Si le nom spécifié ne correspond pas à un paramètre supporté par la caméra, la fonction retournera chaîne vide.


Les constantes disponibles sont les suivantes : 


|   |   |
| --- | --- |
| *vipBalance* | Mode de balance des blancs actuellement supporté par la caméra : automatique, lumière du jour, nuageux, ...<br> |
| *vipCaméra* | Indice de la caméra en cours d'utilisation :<br><br>- Indice compris entre 1 et le nombre de caméras présentes sur l'appareil.<br><br>- -1 si aucune caméra n'est en cours d'utilisation.<br><br><br> |
| *vipCompressionJPEG* | Compression des images JPEG, correspondant à un entier compris entre 1 et 100.<br><br>- 1 : Compression maximale (qualité faible)<br><br>- 100 : Pas de compression (très haute qualité)<br><br><br> |
| *vipEffet* | Effet de couleurs actuellement pris en compte : noir et blanc, poster, sépia, ...<br> |
| *vipFlash* | Etat actuel du flash de la caméra : automatique, sans, anti yeux rouges, ... |
| *vipFocus* | Etat actuel du focus de la caméra : automatique, macro, fixe, ...<br> |
| *vipGéoPosition* | Informations de géolocalisation actuelles : la valeur retournée est une variable de type [géoPosition](../WDLang3/1000019191.md). Les propriétés suivantes peuvent être renseignées en fonction de ce que gère la caméra : latitude, longitude, altitude, date de mesure.<br> |
| *vipISO* | Sensibilité ISO actuellement utilisée par la caméra : automatique, 100, 200, 400, ...<br> |
| *vipRésolutionPhoto* | Résolution des photos prises par la caméra. <br>La valeur renvoyée est au format : <br><br><pre><code><Largeur><TAB><Hauteur></code></pre><br><br><br>Exemple : <br><br><pre><code>640<TAB>480</code></pre><br> |
| *vipRotationAuto* | Rotation automatique des photos en fonction de l'orientation de l'appareil au moment de la capture.<br><br>- <u><u><u><u>Vrai</u></u></u></u> : la rotation automatique est activée.<br><br>- <u><u><u><u>Faux</u></u></u></u> : la rotation automatique est désactivée.<br><br><br>Si la rotation automatique est activée, la photo sera enregistrée en tenant compte de l'orientation de l'appareil au moment de la capture.<br><br>La rotation automatique est activée par défaut, cependant certains appareils ne supportent pas la prise de photos en mode portrait. Dans ce cas, il est conseillé de désactiver la rotation automatique et de gérer a posteriori la rotation de l'image par programmation.<br> |
| *vipScène* | Type de scènes actuellement utilisé par la caméra : automatique, nuit, sport, ...<br> |
| *vipScintillement* | Ajustement du scintillement (antibanding) actuellement utilisé par la caméra : automatique, aucun, 50Hz, 60Hz, ...<br> |
| *vipZoom* | Valeur de zoom actuellement utilisée par la caméra.<br>Les valeurs retournées sont de la forme :<br><br>- 100 (Zoom 1x = pas de zoom)<br><br>- 200 (zoom 2x)<br><br>- 300 (zoom 3x)<br><br>- 350 (zoom 3,5x)<br><br>- ...<br><br><br>La première valeur retournée est toujours 100 et la dernière valeur correspond au zoom maximal. <br>Exemple : <br><br><pre><code>100<RC>200<RC>300<RC>350</code></pre><br><br><br>**Remarque** : La fonction retournera une chaîne vide ("") si la caméra de l'appareil ne supporte pas le zoom.<br> |




<a name="SYNTAXE2"></a>

### Modifier la valeur d'un paramètre de la caméra de l'appareil

`<Résultat> = CaméraParamètre(<Paramètre> , <Valeur>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la modification du paramètre a été effectuée avec succès, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire ou en cas d'erreur lors de la récupération des paramètres de la caméra. Par exemple, la fonction **CaméraParamètre** retournera <u><u><u><u>Faux</u></u></u></u> si le paramètre spécifié ou sa valeur n'est pas supporté par la caméra.




En cas d'erreur, la variable [ErreurDétectée](../WDLang1/3087001.md) est à <u><u><u><u>Vrai</u></u></u></u>. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).

**`<Paramètre> : (Constante de type chaîne de caractères ou chaîne de caractères)`**

Paramètre pour lequel la valeur doit être modifiée. Ce paramètre peut correspondre : 

- soit à une des constantes prédéfinies. Ces constantes correspondent aux paramètres les plus courants. 

- soit à une Chaîne de caractères correspondant au nom du paramètre voulu. 
	**Attention** : le nom du paramètre est sensible à la casse et à l'accentuation. Consultez la documentation du SDK Android ou de l'appareil pour connaître le nom des paramètres supplémentaires supportés. Si le nom spécifié ne correspond pas à un paramètre supporté par la caméra, la fonction retournera chaîne vide.




Les constantes disponibles sont les suivantes : 
Les constantes disponibles sont les suivantes : 


|   |   |
| --- | --- |
| *vipBalance* | Mode de balance des blancs de la caméra.<br> |
| *vipCaméra* | Indice de la caméra à utiliser.<br><br>Attention : le changement de la caméra entraîne la réinitialisation des paramètres précédemment définis avec la fonction **CaméraParamètre**. |
| *vipCompressionJPEG* | Compression des images JPEG.<br> |
| *vipEffet* | Effet de couleurs.<br> |
| *vipFlash* | Mode de flash à utiliser par la caméra. |
| *vipFocus* | Mode de focus à utiliser par la caméra.<br> |
| *vipGéoPosition* | Informations de géolocalisation à utiliser. <br> |
| *vipISO* | Sensibilités ISO supportées par la caméra.<br> |
| *vipRésolutionPhoto* | Définition de la résolution pour les photos prises par la caméra.<br> |
| *vipRotationAuto* | Rotation automatique des photos en fonction de l'orientation de l'appareil au moment de la capture.<br> |
| *vipScène* | Type de scènes à utiliser par la caméra.<br> |
| *vipScintillement* | Ajustement du scintillement (antibanding) à utiliser par la caméra.<br> |
| *vipZoom* | Valeur de zoom à utiliser par la caméra.<br> |



**`<Valeur> : (Type correspondant au paramètre à modifier)`**

Nouvelle valeur du paramètre.

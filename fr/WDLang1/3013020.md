
## ChargeWDL (Fonction)

***En anglais : LoadWDL***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Charge en mémoire une bibliothèque d'objets WINDEV (.WDL). Cette fonction permet :

- d'utiliser des bibliothèques personnalisées pour chaque utilisateur. Il est ainsi possible de créer plusieurs bibliothèques et d'installer l'une ou l'autre de ces bibliothèques en fonction de l'utilisateur.

- de découper une "grosse" bibliothèque en plusieurs "petites" bibliothèques, chargées selon les besoins de l'application.




Cette bibliothèque sera chargée en mémoire jusqu'à ce que :

- l'application se termine,

- la fonction [DéchargeWDL](../WDLang1/3013013.md) soit exécutée.



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple
<a class="notetitle" target="_blank" href="$DOC$=1000003013020&name=chargewdl_fonction&product=WD">Voir des exemples supplémentaires</a>

```wl
// Code d'initialisation du projet
// La bibliothèque dépend de l'utilisateur
NomBibli, User sont des chaînes
hWDL_ est un entier
User = LigneCommande("Utilisateur")
SELON User
	CAS User1 : NomBibli = "USER1.WDL"
	CAS User2 : NomBibli = "USER2.WDL"
	CAS User3 : NomBibli = "USER3.WDL"
FIN
hWDL_ = ChargeWDL(NomBibli)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = ChargeWDL(<Nom de la bibliothèque>)`
---

**`<Résultat> : (Entier)`**



- Numéro d'identification (handle) de la bibliothèque

- -1 en cas d'échec de chargement de la bibliothèque.




**`<Nom de la bibliothèque> : (Chaîne de caractères)`**

Nom et chemin complet ou relatif de la bibliothèque à charger.

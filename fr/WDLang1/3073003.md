
## SysFenAffiche (Fonction)

***En anglais : SysWinShow***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie l'état d'affichage d'une fenêtre.

**Remarque** : La fonction **SysFenAffiche** permet principalement d'afficher une fenêtre non créée par WINDEV. Le numéro de handle de cette fenêtre est obtenu avec la fonction [SysFenHandle](../WDLang1/3073007.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affichage de la fenêtre "Microsoft Word - Ventes 2000.DOC"
ResHandle = SysFenHandle("Microsoft Word - Ventes 2000.DOC")
ResAfficheFen = SysFenAffiche(ResHandle, sfaVisible)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = SysFenAffiche(<Handle de la fenêtre> [, <Type d'affichage>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la fenêtre spécifiée était précédemment affichée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Handle de la fenêtre> : (Entier)`**

Handle de la fenêtre à manipuler. Ce handle peut être obtenu avec la fonction [SysFenHandle](../WDLang1/3073007.md).

**`<Type d'affichage> : (Constante optionnelle)`**

Indique l'état de la fenêtre spécifiée :



|   |   |
| --- | --- |
| *sfaCache* | Rend la fenêtre invisible et active une autre fenêtre. |
| *sfaIconise* | Iconise la fenêtre. |
| *sfaMaximise* | Maximise la fenêtre. |
| *sfaMinimise* | Minimise la fenêtre. |
| *sfaNormal*<br>(Valeur par défaut) | Affiche la fenêtre à sa taille et à sa position initiales |
| *sfaRafraîchie* | Rafraîchit la fenêtre |
| *sfaRéactive* | Rend active une fenêtre inactive |
| *sfaRestaure* | Restaure une fenêtre à sa taille et à sa position initiales. |
| *sfaVisible* | Rend visible une fenêtre et l'affiche à sa taille et à sa position initiales (si la fenêtre a été rendue invisible avec la constante *sfaCache*) |







<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Fenêtre Windows
<a name="fenetre_windows_ELTPARAGRAPHE000095"></a>L'utilisation de la fonction **SysFenAffiche** est déconseillée avec des fenêtres WINDEV. Il est préférable d'utiliser les fonctions spécifiques suivantes : [Restaure](../WDLang1/3038042.md), [Maximise](../WDLang1/3038001.md), [FenEtat](../WDLang1/3038030.md), [Iconise](../WDLang1/3038015.md), ...

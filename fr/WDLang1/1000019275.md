


## PlanningSupprimeTout (Fonction)

***En anglais : SchedulerDeleteAll***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime tous les rendez-vous et toutes les ressources d'un champ Planning.

Remarque : 

- Dans le cas d'un champ Planning par programmation, tous les rendez-vous et toutes les ressources sont supprimés. 

- Dans le cas d'un champ Planning fichier ou sur variable, tous les rendez-vous et toutes les ressources sont uniquement supprimées dans le champ Planning. Les rendez-vous et les ressources sont conservés dans la source de données liée au champ. 






![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) En mode Universal Windows 10 App, cette fonction permet uniquement de manipuler les champs Planning présents dans un état. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Suppression de tout le contenu du champ Planning PLN_Planning
PlanningSupprimeTout(PLN_Planning)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`PlanningSupprimeTout(<Champ Planning>)`
---

**`<Champ Planning> : (Nom de champ)`**

Nom du champ Planning dont tous les rendez-vous et ressources doivent être supprimés. Ce champ peut correspondre : 

- au champ Planning d'une fenêtre, 

- au champ Planning d'une page,

- au champ Planning d'un état.


## Fonctions de gestion des jetons (syntaxe préfixée)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions de gestion des jetons dans les champs de saisie via une syntaxe préfixée sont les suivantes :



|   |   |
| --- | --- |
| [&lt;Saisie&gt;.AjouteJeton](../WDLang1/1000024043.md) | Ajoute un jeton dans un champ de saisie de type "Jetons de texte". |
| [&lt;Saisie&gt;.InsèreJeton](../WDLang1/1000024044.md) | Insère un jeton dans un champ de saisie de type "Jetons de texte". |
| [&lt;Saisie&gt;.OccurrenceJeton](../WDLang1/1000024045.md) | Renvoie le nombre de jetons présent un champ de saisie de type "Jetons de texte". |
| [&lt;Saisie&gt;.SupprimeJeton](../WDLang1/1000024046.md) | Supprime un jeton dans un champ de saisie de type "Jetons de texte". |
| [&lt;Saisie&gt;.SupprimeToutJeton](../WDLang1/1000024047.md) | Supprime tous les jetons d'un champ de saisie de type "Jetons de texte". |


## ExtraitLigne (Fonction)

***En anglais : ExtractLine***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Extrait une ligne spécifique présente dans une chaîne de caractères.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Extraction de la ligne 5 du fichier "test.txt"
sContenu est une chaîne = fChargeTexte("C:\test.txt")
sLigne5 est une chaîne = ExtraitLigne(sContenu, 5)
```


<a name="Exemple2"></a>

```wl
// Extraction de la ligne 5 du champ de saisie multiligne SAI_Info
sLigne5 est une chaîne = ExtraitLigne(SAI_Info, 5)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = ExtraitLigne(<Chaîne initiale> , <Numéro de ligne>)`
---

**`<Résultat> : (Chaîne de caractères)`**



- Contenu de la ligne extraite. 

- EOT si le numéro de la ligne est inconnu. 




**`<Chaîne initiale> : (Chaîne de caractères)`**

Chaîne de caractères dans laquelle la ligne est recherchée. 

**`<Numéro de ligne> : (Entier)`**

Numéro de la ligne recherchée. Ce numéro de ligne est compris entre 1 et le nombre total de lignes de &lt;Chaîne initiale&gt;. Le séparateur de ligne correspond au Retour-Chariot (RC).

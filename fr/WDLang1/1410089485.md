


## &lt;Liste de Kanban&gt;.IndiceListe (Fonction)

***En anglais : &lt;Kanban list&gt;.ListIndex***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie l'indice d'une liste de Kanban présente dans un champ Kanban. 

Remarque : La visibilité des listes n'est pas prise en compte.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
Indice est un entier 
Indice = LISTE_A_revoir.IndiceListe()
SI Indice = 1 ALORS
	KANBAN_MonKanban.AjouteListe("A commencer")
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Liste de Kanban>.IndiceListe()`
---

**`<Résultat> : (Entier)`**



- Indice de la liste de Kanban dans le champ Kanban, 

- 0 si la liste de Kanban n'existe pas. 


Remarque : Cet indice correspond à l'indice d'édition.

**`<Liste de Kanban> : (Nom de champ)`**

Nom du champ Liste de Kanban à manipuler. Ce nom peut correspondre : 

- au nom du champ Liste de Kanban défini sous l'éditeur, 

- au nom du champ Liste de Kanban défini par programmation.

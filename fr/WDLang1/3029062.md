
## gPoliceItalique (Fonction)

***En anglais : gFontItalic***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Construit une séquence de caractères permettant d'indiquer si le texte suivant l'appel de cette fonction est en italique ou non. 

**Remarque** : Tout le texte écrit **à la suite** de la fonction est affecté.

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Dans une fenêtre, cette séquence de caractères permet d'utiliser un texte en italique (ou non) :  

- dans un élément d'un champ Liste ou Combo.

- dans la barre de titre et/ou de message d'une fenêtre. 

- dans un champ Libellé. 







<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 
```wl
// Affichage du titre "WINDEV, AGL n°1 en France." dans la prochaine fenêtre
TitreSuivant(gPoliceItalique() + "WINDEV" + gPoliceItalique(Faux) + ", AGL n°1 en France.")
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = gPoliceItalique([<Italique>])`
---

**`<Résultat> : (Chaîne de caractères)`**

Séquence de caractères définissant si la suite du texte doit être en italique ou non.

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Cette séquence de caractères peut être : 

- utilisée dans les fonctions [ListeAjoute](../WDLang1/3049004.md), [ListeInsère](../WDLang1/3049002.md) ou [ListeModifie](../WDLang1/3049009.md).

- affectée à un champ Libellé. 

- utilisée dans la barre de titre ou de message d'une fenêtre. 




**`<Italique> : (Booléen optionnel)`**



- <u><u><u><u>Vrai</u></u></u></u> (par défaut) si le texte suivant la fonction **gPoliceItalique** doit être en italique, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- La police par défaut est rétablie a la fin de l'appel des fonctions [gPoliceXXX](../WDLang1/3029028.md).

- Toutes les autres caractéristiques de la police sont conservées (Gras, ...).

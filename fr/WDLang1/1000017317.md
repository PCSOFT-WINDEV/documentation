
## Constantes de gestion des événements Windows
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
 
Les constantes de gestion des événements Windows sont les suivantes :

| Constante | Détail | Fonction |
| --- | --- | --- |
| **HandleAscenseurH** | Handle de l'ascenseur horizontal du champ. | [Handle](../WDLang1/3015005.md) |
| **HandleAscenseurV** | Handle de l'ascenseur vertical du champ. | [Handle](../WDLang1/3015005.md) |
| **HandleChampNatif** | Champ natif iOS associé au champ spécifié : champ Carte (MKMapView) et champ HTML (UIWebView). | [Handle](../WDLang1/3015005.md) |
| **HandleListeCombo** | Handle de la liste déroulante associée à une combo (le champ doit être un champ Combo). | [Handle](../WDLang1/3015005.md) |
| **HandleListeSaisieAssistée** | Renvoie le handle de la liste déroulante associée à un champ proposant la saisie assistée. | [Handle](../WDLang1/3015005.md) |
| **HandleSaiseCombo** | Handle du champ de saisie associé à une combo (le champ doit être un champ Combo). | [Handle](../WDLang1/3015005.md) |
| **HandleUIViewController** | Renvoie le UIViewController (iOS) associé au champ spécifié. | [Handle](../WDLang1/3015005.md) |
| **tempoDessin** | Type d'événement pouvant s'exécuter pendant la temporisation : Les champs et les fenêtres peuvent être redessinés. | [Temporisation](../WDLang1/3015012.md) |
| **tempoRien** | Type d'événement pouvant s'exécuter pendant la temporisation : Aucun événement ne peut s'exécuter. | [Temporisation](../WDLang1/3015012.md) |
| **tempoSourisEtClavier** | Type d'événement pouvant s'exécuter pendant la temporisation :<br><br>- Les champs et les fenêtres peuvent être redessinés.<br><br>- Les timers peuvent être exécutés.<br><br>- Les actions souris (clic sur les champs par exemple) ou les événements clavier peuvent être exécutés.<br><br><br> | [Temporisation](../WDLang1/3015012.md) |
| **tempoTimer** | Type d'événement pouvant s'exécuter pendant la temporisation :<br><br>- Les champs et les fenêtres peuvent être redessinés.<br><br>- Les timers peuvent être exécutés.<br><br><br> | [Temporisation](../WDLang1/3015012.md) |


## EnModeSysNotification (Fonction)

***En anglais : InSysNotificationMode***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de savoir si l'application est lancée depuis une notification interactive "Windows 10".
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI EnModeSysNotification() = Vrai ALORS 
	Trace("Application lancée depuis une notification Windows 10")
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = EnModeSysNotification()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'application est lancée depuis une notification "Windows 10", 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.

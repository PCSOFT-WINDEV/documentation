


## TélémétrieAjouteMesure (Fonction)

***En anglais : TelemetryAddMeasure***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ajoute une mesure pour un capteur de valeur ou de temps.

Il est ainsi possible d'établir des statistiques sur les mesures d'une valeur ou d'un temps : nombre total, minimal, moyen, maximal, ...


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Un capteur de valeur "NombreProduit" a été défini sous l'éditeur
// A chaque validation d'une commande, le nombre de produits de la commande est ajouté
TélémétrieAjouteMesure(NombreProduit, TableauProduit.Occurrence)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`TélémétrieAjouteMesure(<Capteur> [, <Valeur de la catégorie> [, <Valeur de la mesure>]])`
---

**`<Capteur> : (Chaîne de caractères)`**

Nom du capteur de valeur ou de temps défini sous l'éditeur. 

**`<Valeur de la catégorie> : (Chaîne de caractères optionnelle)`**

Valeur pour la catégorie définie sous l'éditeur.

**`<Valeur de la mesure> : (Type correspondant au capteur, optionnel)`**

Valeur de la mesure. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Les capteurs doivent être définis dans une télémétrie personnalisée.

- Si le capteur n'est pas un capteur de valeur ou de temps, une erreur WLangage sera affichée.

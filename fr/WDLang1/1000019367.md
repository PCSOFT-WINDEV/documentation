
## ChercheProcédure (Fonction)

***En anglais : SeekProcedure***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Cherche une procédure connue par son nom. La procédure trouvée pourra être exécutée directement. 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Nom de la procédure qu'il faut chercher puis exécuter
NomProcédureCallBack est une chaîne
NomProcédureCallBack = "Procedures_globales_de_MonProjet.CallBack"
// Variable permettant de récupérer le résultat de la recherche de la procédure
procCallback est une Procédure

// Cherche la procédure dont le nom est dans la variable "NomProcédureCallBack"
procCallback = ChercheProcédure(NomProcédureCallBack)

// Vérifie si la procédure cherchée a été trouvée
SI procCallback = NULL ALORS
	// Procédure non trouvée
	Info(ChaîneConstruit("La procédure <%1> n'a pas été trouvée.", ...
			NomProcédureCallBack))
SINON
	// Procédure trouvée
	// Appel de la procédure trouvée
	procCallback("Paramètres")
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = ChercheProcédure(<Nom de la procédure> [, <Option>])`
---

**`<Résultat> : (Variable de type Procédure)`**



- Nom de la variable de type [Procédure](../Motscles/1000018907.md) qui correspondra à la procédure trouvée. 

- "NULL" si aucune procédure n'est trouvée. 




**`<Nom de la procédure> : (Chaîne de caractères)`**

Nom de la procédure à rechercher. Ce nom peut être préfixé de la collection de procédures ou du nom de la fenêtre dans laquelle la procédure se trouve. 

**`<Option> : (Constante optionnelle de type Entier)`**

Mode de recherche de la procédure : 


|   |   |
| --- | --- |
| *trtProcédure*<br>(Valeur par défaut) | Recherche parmi les procédures du projet. |
| *trtProcédureProjet* | Recherche parmi des procédures du projet hôte du composant. |


## dbgEtatAudit (Fonction)

***En anglais : dbgAuditStatus***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Récupère et modifie l'état de l'audit dynamique.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Lance l'audit dynamique
dbgActiveAudit()
...
// Met l'audit dynamique en pause
dbgEtatAudit(AuditPause)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître l'état de l'audit dynamique

`<Résultat> = dbgEtatAudit()`
---

**`<Résultat> : (Constante de type Entier)`**

Etat actuel de l'audit dynamique : 


|   |   |
| --- | --- |
| *AuditActif* | L'audit dynamique est actif. |
| *AuditInactif* | L'audit dynamique n'a pas été activé. |
| *AuditPause* | L'audit dynamique est en pause. |






<a name="SYNTAXE2"></a>

### Modifier l'état de l'audit dynamique

`<Résultat> = dbgEtatAudit(<Nouvel état>)`
---

**`<Résultat> : (Constante de type Entier)`**

Etat d'activité de l'audit dynamique avant la modification effectuée par la fonction **dbgEtatAudit**. 


|   |   |
| --- | --- |
| *AuditActif* | L'audit dynamique est actif. |
| *AuditInactif* | L'audit dynamique n'a pas été activé. |
| *AuditPause* | L'audit dynamique est en pause. |





**`<Nouvel état> : (Constante de type entier)`**

Nouvel état de l'audit dynamique : 


|   |   |
| --- | --- |
| *AuditActif* | Active l'audit dynamique (après une pause). |
| *AuditInactif* | Arrête définitivement l'audit dynamique. Pour le réactiver, il faudra utiliser la fonction [dbgActiveAudit](../WDLang1/1000018835.md). |
| *AuditPause* | Arrête momentanément l'audit dynamique. |


## VersionWindows (Fonction)

***En anglais : WindowsVersion***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le mode d'exécution de Windows.

**Rappel** : WINDEV permet de créer des [exécutables 32 bits et 64 bits](../Editeurs/2030048.md).






<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connaître la version de Windows : 32 bits ou 64 bits

Version est un entier
Version = VersionWindows()
SELON Version 
	CAS 32 : Info("Windows s'exécute en 32 bits")
	CAS 64 : Info("Windows s'exécute en 64 bits")
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = VersionWindows()`
---

**`<Résultat> : (Chaîne de caractères)`**

Mode d'exécution du programme dans l'environnement en cours :

- "32" : le programme fonctionne en 32 bits dans un environnement 32 bits.

- "64" : le programme fonctionne en 32 bits ou en 64 bits dans un environnement 64 bits.

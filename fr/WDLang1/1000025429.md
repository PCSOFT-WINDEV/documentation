
## Procédure WLangage appelée par la fonction OKAnnulerAsynchrone

<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Procédure (également appelée "Callback") appelée par la fonction [OKAnnulerAsynchrone](../WDLang1/1000025308.md) lorsque l'utilisateur clique sur l'un des boutons de la boîte de dialogue. Cette procédure permet de connaître le bouton cliqué et permet d'exécuter si nécessaire un traitement spécifique. 
Remarque : Cette procédure peut être une procédure locale, globale, interne ou lambda. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
OKAnnulerAsynchrone("Quel bouton choisissez vous ?", MaProcédure_OKAnnuler)

PROCEDURE INTERNE MaProcédure_OKAnnuler(nRéponse entier){
	SELON nRéponse
		CAS 1 : ToastAffiche("Bouton OK")
		CAS 0 : ToastAffiche("Bouton Annuler")
	FIN
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`OKAnnulerAsynchrone_Callback(<Bouton sélectionné>)`
---

**`<Bouton sélectionné> : (Entier)`**

Identifie le bouton sélectionné par l'utilisateur :


|   |   |
| --- | --- |
| *Faux* (ou 0) | L'utilisateur a choisi le bouton "Annuler". |
| *Vrai* (ou 1) | L'utilisateur a choisi le bouton "OK". |




## Fonctions DDE
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions permettant de gérer le DDE sont les suivantes :



|   |   |
| --- | --- |
| [DDEChaîne](../WDLang1/3028002.md) | Retourne les informations récupérées par la fonction [DDERécupère](../WDLang1/3028017.md). |
| [DDEConnecte](../WDLang1/3028015.md) | Ouvre une connexion DDE entre le programme en cours et le destinataire selon un sujet donné. |
| [DDEDéconnecte](../WDLang1/3028003.md) | Supprime une connexion entre le programme en cours et un destinataire. |
| [DDEDestinataire](../WDLang1/3028001.md) | Identifie le destinataire d'une connexion DDE. |
| [DDEEnvoie](../WDLang1/3028004.md) | Envoie une donnée au programme connecté via DDE. |
| [DDEErreur](../WDLang1/3028006.md) | Retourne le compte-rendu d'exécution d'une fonction de gestion du dialogue DDE. |
| [DDEEvénement](../WDLang1/3028009.md) | Associe une procédure WLangage à un événement DDE : lorsque l'événement DDE se produit, la procédure WLangage associée est automatiquement lancée. |
| [DDEExécute](../WDLang1/3028008.md) | Envoie un ordre à exécuter sur le serveur. |
| [DDEFinLien](../WDLang1/3028005.md) | Interrompt un lien entre un item et une donnée. |
| [DDEItem](../WDLang1/3028007.md) | Identifie l'item concerné par un événement DDE. |
| [DDELance](../WDLang1/3028013.md) | Lance l'exécution d'un programme (exécutable par exemple) depuis l'application en cours. |
| [DDELien](../WDLang1/3028010.md) | Crée un lien chaud avec une donnée. |
| [DDELienTiède](../WDLang1/3028014.md) | Crée un lien tiède avec une donnée. |
| [DDEModifie](../WDLang1/3028011.md) | Modifie une donnée liée. |
| [DDERécupère](../WDLang1/3028017.md) | Récupère une donnée envoyée par un programme (le destinataire de la connexion pour l'objet spécifié). |
| [DDESujet](../WDLang1/3028019.md) | Identifie le sujet de la conversation associée à une connexion DDE. |

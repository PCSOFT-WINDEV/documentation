
## SysThème (Fonction)

***En anglais : SysTheme***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le thème visuel affiché actuellement sur le mobile. 

Remarque : Ce thème est celui affiché automatiquement par le mobile (configuré dans les paramètres de l'appareil). Il ne correspond pas au thème choisi par programmation avec la fonction [ChangeThème](../WDLang3/1000025266.md). 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affichage d'une image claire si le thème sombre est affiché
SI SysThème() = stThèmeSombre ALORS
	IMG_Image = "image_claire.jpg"
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = SysThème()`
---

**`<Résultat> : (Constante de type Entier)`**

Thème visuel affiché sur le mobile : 


|   |   |
| --- | --- |
| *stThèmeClair* | Le thème clair du mobile est utilisé. |
| *stThèmeInconnu* | Le thème utilisé est inconnu. |
| *stThèmeSombre* | Le thème sombre du mobile est utilisé. |

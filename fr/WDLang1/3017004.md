


## Fonctions de gestion des animations
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000240"></a>
Plusieurs fonctions WLangage permettent de gérer des animations. Il est possible de :

- gérer l'animation des champs Image et des champs Libellé.

- animer la modification de champs. 

- animer le passage d'une fenêtre à une autre.






<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Animation des champs Image et Libellé
<a name="animation_des_champs_image_libelle_ELTTEXTE000264"></a>
Ces fonctions permettent de gérer l'animation d'une image dans un champ Image ou d'un texte dans un champ Libellé.


|   |   |
| --- | --- |
| [AnimationChangeParamètre](../WDLang1/3017002.md) | Change un paramètre d'animation d'une image affichée dans un champ Image ou d'un texte affiché dans un champ Libellé |
| [AnimationChangeType](../WDLang1/3017001.md) | Change le type d'animation d'une image dans un champ Image ou d'un texte dans un champ Libellé (les paramètres spécifiques au nouveau type peuvent être modifiés avec la fonction [AnimationChangeParamètre](../WDLang1/3017002.md)) |
| [AnimationLitParamètre](../WDLang1/3017003.md) | Lit la valeur d'un des paramètres d'animation défini pour un champ Image ou pour un champ Libellé |
| [FenDésactiveEffet](../WDLang1/1000017172.md) | Désactive des effets visuels du moteur graphique de WINDEV (pour une utilisation sous TSE par exemple). Cette fonction est utile pour permettre à une application de fonctionner sur un poste ayant des capacités graphiques limitées ou pour le mode TSE si la bande passante est limitée. |


Pour plus de détails, consultez [Animer un champ Libellé](../WDChamp/1013160.md) et [Animer une image](../WDChamp/1013244.md).

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Animation des champs
<a name="animation_des_champs_ELTTEXTE000288"></a>
Ces fonctions permettent d'animer la modification d'un ou de plusieurs champs réalisée par programmation.




|   |   |
| --- | --- |
| [AnimationActive](../WDLang1/1000017399.md) | Active ou désactive les animations dans les champs (animations définies dans le projet ou par programmation). |
| [AnimationJoue](../WDLang1/1000017387.md) | Joue une animation sur une modification d'un champ, d'un groupe de champs ou d'une fenêtre. |
| [AnimationJoueSurPropriétéChamp](../WDLang1/1000017398.md) | Anime la modification d'une propriété entière entre 2 valeurs. |
| [AnimationMinFPS](../WDLang1/1000017400.md) | Paramètre le nombre minimal d'images par seconde pour que les animations dans les champs (effets de transitions) soient activées. |
| [AnimationPrépare](../WDLang1/1000017386.md) | Prépare l'animation sur un champ, un groupe de champs ou une fenêtre. L'animation sera ensuite jouée avec la fonction [AnimationJoue](../WDLang1/1000017387.md). |


Il existe également des fonctions permettant de jouer une animation lors du changement d'état d'un champ ou d'un groupe de champs :


|   |   |
| --- | --- |
| [ChampActif](../WDLang1/1000017377.md) | Dégrise un champ ou un groupe de champs d'une fenêtre. Pendant cette opération, une animation sur les champs peut être effectuée. |
| [ChampGrisé](../WDLang1/1000017375.md) | Grise un champ ou un groupe de champs. Pendant cette opération, une animation sur les champs peut être effectuée. |
| [ChampInvisible](../WDLang1/1000017369.md) | Rend invisible un champ ou un groupe de champs dans une fenêtre. Pendant cette opération, une animation sur les champs peut être effectuée. |
| [ChampVisible](../WDLang1/1000017338.md) | Rend visible un champ ou un groupe de champs dans une fenêtre. Pendant cette opération, une animation sur les champs peut être effectuée. |

## Animation des fenêtres
<a name="animation_des_fenetres_ELTTEXTE000312"></a>
Ces fonctions permettent d'animer le passage d'une fenêtre à une autre. Pour plus de détails, consultez [Animation de fenêtres](../WDLang1/3038054.md).


|   |   |
| --- | --- |
| [FenAnimationFermeture](../WDLang1/3038050.md) | Modifie le type et la durée des animations utilisées lors de la fermeture des fenêtres de l'application. |
| [FenAnimationOuverture](../WDLang1/3038051.md) | Modifie le type et la durée des animations utilisées lors de l'ouverture des fenêtres de l'application. |
| [FenAnimationSuivante](../WDLang1/3038052.md) | Modifie le type et la durée des animations utilisées lors de l'ouverture et de la fermeture de la prochaine fenêtre. |

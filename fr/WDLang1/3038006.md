


## FenRepeint (Fonction)

***En anglais : WinRedraw***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Redessine immédiatement les champs de la fenêtre nécessitant une mise à jour graphique.

**Remarque** : Cette fonction est identique à la fonction [MultitâcheRepeint](../WDLang1/3038004.md).








<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
i est un entier 
POUR i = 1 A 1000
	TraitementLong()
	FenRepeint()
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`FenRepeint([<Fenêtre>])`
---

**`<Fenêtre> : (Nom de fenêtre)`**

Nom de la fenêtre à redessiner.
Si ce paramètre n'est pas précisé, la fenêtre en cours est redessinée.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Comparaison avec la fonction Multitâche
<a name="comparaison_avec_fonction_multitache_ELTPARAGRAPHE000088"></a>Contrairement à la fonction [Multitâche](../WDLang1/3015004.md), l'appel de la fonction **FenRepeint** n'interrompt pas le code en cours (les fonctions Timer ne sont pas appelées par exemple).

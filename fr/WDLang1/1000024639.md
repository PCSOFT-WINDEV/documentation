


## &lt;Variable picCalque&gt;.SymétrieHorizontale (Fonction)

***En anglais : &lt;picLayer variable&gt;.HorizontalSymmetry***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Retourne l'image contenue dans une variable de type [picCalque](../WDLang1/1000024604.md) par rapport à un axe horizontal (symétrie par rapport à un axe horizontal).![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=dSymetrieHorizontale.gif)


![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Les dessins](../WDLang1/3029035.md). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonImageWDPic est un WDPic = "Test.bmp"
MonCalque est un picCalque

// Symétrie du calque par rapport à un axe horizontal
MonRésultat est un booléen
MonRésultat = MonCalque.SymétrieHorizontale()
SI MonRésultat = Faux ALORS Erreur("L'opération a échoué")
IMG_MonDessin = MonImageWDPic
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = dSymétrieHorizontale(<Image>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la symétrie a été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Image> : (Variable de type picCalque)`**

Nom de la variable de type [picCalque](../WDLang1/1000024604.md) à manipuler.

La symétrie est réalisée par rapport à l'axe horizontal médian de l'image. L'image est directement modifiée.

<a name="NOTE0_1"></a>

## Remarques


### Fin du dessin
<a name="fin_dessin_ELTPARAGRAPHE000346"></a>Si la fonction [&lt;Variable picCalque&gt;.FinDessin](../WDLang1/1000024473.md) est appelée dans le même traitement que la fonction **&lt;Variable picCalque&gt;.SymétrieHorizontale**, la transformation de l'image ne sera pas affichée.

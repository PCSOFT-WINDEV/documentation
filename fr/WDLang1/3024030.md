
## Minuscule (Fonction)

***En anglais : Lower***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Convertit une chaîne de caractères en minuscules en fonction des paramètres de localisation définis dans Windows.

**Remarque** : La fonction **Minuscule** est conservée par compatibilité et fonctionne uniquement avec les alphabets latins. Pour paramétrer de manière plus précise le formatage d'une chaîne ou manipuler d'autres alphabets, utilisez la fonction [ChaîneFormate](../WDLang1/3024019.md).






<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
Minuscule("ABCD") 		// Renvoie "abcd"
Minuscule("BONJOUR")            // Renvoie "bonjour"
Minuscule("Pierre et Paul")		// Renvoie "pierre et paul"
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = Minuscule(<Chaîne de caractères à convertir>)`
---

**`<Résultat> : (Chaîne de caractères)`**

Chaîne de caractères en minuscules.

**`<Chaîne de caractères à convertir> : (Chaîne de caractères)`**

Chaîne de caractères à passer en minuscules.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Règles de conversion
<a name="regles_conversion_ELTPARAGRAPHE000079"></a>

- Les lettres minuscules non accentuées restent en minuscules non accentuées.

- Les lettres minuscules accentuées restent en minuscules accentuées.

- Les lettres majuscules non accentuées passent en minuscules non accentuées.

- Les lettres majuscules accentuées restent en majuscules accentuées.
	




**Remarque** : La fonction **Minuscule** est conservée par compatibilité et fonctionne uniquement avec les alphabets latins. Pour paramétrer de manière plus précise le formatage d'une chaîne ou manipuler d'autres alphabets, utilisez la fonction [ChaîneFormate](../WDLang1/3024019.md).
<a name="NOTE0_2"></a>




### Divers
<a name="divers_ELTPARAGRAPHE000096"></a>

- Les "blancs" sont conservés.

- Le moteur HFSQL effectue automatiquement des recherches sans tenir compte de la casse. La fonction **Minuscule** est donc inutile pour effectuer ce type d'opération.

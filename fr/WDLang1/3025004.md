
## TexteLargeur (Fonction)

***En anglais : TextWidth***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Calcule la largeur (en pixels) d'un texte affiché dans un champ de saisie, un champ Combo ou un champ Libellé. Il est ainsi possible de prévoir la largeur occupée par le texte afin d'agrandir le champ si nécessaire.






<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
ResNbPixels est un réel = TexteLargeur(SAI_Saisie1, "Bonjour")
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = TexteLargeur(<Nom du champ> , <Texte> [, <Options>])`
---

**`<Résultat> : (Réel)`**

Largeur du texte (en pixels) dans le champ spécifié (en fonction du style du champ).

Il est possible de connaître la largeur en millimètres grâce à la formule suivante : 
&lt;Nb pixels&gt; = &lt;Résolution écran&gt;\*&lt;Nb millimètres&gt;/25.4


**`<Nom du champ> : (Chaîne de caractères)`**

Nom du champ de saisie, du champ Libellé ou du champ Combo.


**`<Texte> : (Chaîne de caractères)`**

Texte dont la largeur doit être calculée.

**`<Options> : (Constante optionnelle de type entier)`**

Configure le mode de calcul de la largeur du texte :


|   |   |
| --- | --- |
| *tAvecMarge* | Permet de prendre en compte les marges définies pour le champ (retrait gauche, retrait droit, ...). <br>Cette constante peut être combinée avec la constante *tMultiligne*. <br><br>![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Cette constante n'est pas disponible. |
| *tDéfaut*<br>(valeur par défaut) | Largeur du texte dans le champ. |
| *tMultiligne* | Largeur du texte dans le champ. Au maximum, cette largeur correspondra à la largeur du champ.<br><br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=TexteLargeur.gif)<br> |
| *tMonoligne* | Largeur du texte sans tenir compte de la largeur du champ. Cette constante permet d'obtenir une précision supérieure à celle fournie par la constante *tDéfaut*.<br><br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=TexteLargeur.gif)<br> |

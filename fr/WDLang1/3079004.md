


## VidéoAffiche (Fonction)

***En anglais : VideoDisplay***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Affiche dans un champ Caméra la séquence vidéo actuellement retransmise par une "web caméra" ou par la caméra de l'appareil mobile.






<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affiche la séquence vidéo actuellement retransmise dans le champ "CAM_Caméra"
ResFilm = VidéoAffiche(CAM_Caméra)
SI ResFilm = Vrai ALORS
	// Affiche un libellé
	CAM_Caméra.Libellé = "En ce moment dans la cuisine"
SINON
	Erreur(ErreurInfo())
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = VidéoAffiche(<Champ Caméra>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la séquence vidéo actuellement retransmise est correctement affichée dans le champ Caméra, 

- <u><u><u><u>Faux</u></u></u></u> en cas d'erreur. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Champ Caméra> : (Nom de champ)`**

Nom du champ Caméra à manipuler.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Conditions d'utilisation
<a name="conditions_utilisation_ELTPARAGRAPHE000075"></a>

- ![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png) Pour utiliser cette fonction, une "caméra" doit être connectée au poste en cours.




**Remarque** : Cette fonction est équivalente à la fonction [CaméraAffiche](../WDLang1/1410086595.md).
<a name="NOTE0_2"></a>


### Cas particuliers
<a name="cas_particuliers_ELTPARAGRAPHE000103"></a>

- En exécution, dès l'ouverture de la fenêtre contenant le champ Caméra, la séquence vidéo retransmise par la Caméra associée s'affiche automatiquement. Dans ce cas, l'utilisation de la fonction **VidéoAffiche** n'est pas nécessaire.

- Pour arrêter l'affichage d'une séquence vidéo, utilisez la fonction [VidéoArrête](../WDLang1/3079001.md).

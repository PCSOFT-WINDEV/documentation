


## MenuSelectPlus (Fonction)

***En anglais : MenuSelectPlus***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Active une option de menu (menu contextuel ou déroulant). L'option de menu spécifiée peut être sélectionnée. Cette fonction permet de dégriser une option de menu grisée avec la fonction [MenuSelectMoins](../WDLang1/3038010.md).

<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Dégrise l'option "OPT_BarreIcone" dans 
// le menu déroulant "Menu" de la fenêtre en cours
MenuSelectPlus(OPT_BarreIcone) 
// Dégrise l'option "OPT_BarreIcone" dans le menu "Menu" de la fenêtre "FEN_Fen1"
// La fenêtre FEN_Fen1 n'est pas la fenêtre en cours
MenuSelectPlus(FEN_Fen1, OPT_BarreIcone)
// Dégrise l'option "OPT_BarreIcone" dans le menu contextuel "MenuContextuel1" 
MenuSelectPlus(FEN_Fen1, MenuContextuel1.OPT_BarreIcone)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`MenuSelectPlus([<Nom de la fenêtre>, ] <Nom de l'option de menu>)`
---

**`<Nom de la fenêtre> : (Chaîne de caractères optionnelle)`**

Nom de la fenêtre contenant l'option de menu à activer. 
Si ce paramètre n'est pas spécifié, la fonction **MenuSelectPlus** manipulera la fenêtre en cours.

**`<Nom de l'option de menu> : (Chaîne de caractères)`**

Nom de l'option de menu à activer. Ce nom est défini dans la fenêtre de description de l'option de menu. Ce nom doit être spécifié sans guillemets. 
Si ce paramètre correspond au nom d'un menu, toutes les sous-options de ce menu seront également accessibles (récursif). Pour dégriser l'option d'un menu contextuel, il est nécessaire d'utiliser la notation &lt;Nom du menu contextuel&gt;.&lt;Nom de l'option&gt;.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Astuce : Dégriser un ensemble d'options
<a name="astuce_degriser_ensemble_options_ELTPARAGRAPHE000049"></a>Pour dégriser un ensemble d'options, il suffit de :

1. Affecter les différentes options à un groupe de champs.

2. Dégriser les options (et autres champs du groupe) grâce à la propriété [Etat](../Proprietes/2510039.md) = actif appliquée au groupe de champs.




## Fonctions de gestion des champs Interrupteur (syntaxe préfixée)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions permettant de gérer les champs Interrupteur via une syntaxe préfixée sont les suivantes :



|   |   |
| --- | --- |
| [&lt;Interrupteur&gt;.Ajoute](../WDLang1/1000021291.md) | Ajoute une nouvelle option dans un champ Interrupteur. |
| [&lt;Interrupteur&gt;.Insère](../WDLang1/1000021307.md) | Insère une nouvelle option dans un champ Interrupteur. |
| [&lt;Interrupteur&gt;.Occurrence](../WDLang1/1000021308.md) | Renvoie le nombre d'options présentes dans un champ Interrupteur. |
| [&lt;Interrupteur&gt;.Supprime](../WDLang1/1000021309.md) | Supprime une option d'un champ Interrupteur. |

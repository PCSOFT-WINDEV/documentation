
## ThreadArrêtDemandé (Fonction)

***En anglais : ThreadStopRequested***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Vérifie si une demande d'arrêt a été envoyée au thread en cours d'exécution. 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Si une demande d'arrêt a été envoyée au thread
SI ThreadArrêtDemandé() ALORS
	// Libérer les ressources du thread
	...
	// Terminer le thread
	ThreadFin()
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = ThreadArrêtDemandé()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si une demande d'arrêt a été envoyée au thread en cours d'exécution, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- L'arrêt d'un thread est demandé par un autre thread grâce à la fonction [ThreadDemandeArrêt](../WDLang1/1000021218.md).

- Le code du thread courant doit vérifier régulièrement (boucles, ... ) si un arrêt a été demandé et il doit se terminer proprement le cas échéant.

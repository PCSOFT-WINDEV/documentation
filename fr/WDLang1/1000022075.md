
## PiloteVitesse (Fonction)

***En anglais : EmulateSpeed***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Règle la vitesse d'exécution des fonctions de pilotage.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ralentit la vitesse d'exécution du test
PiloteVitesse(pvRalenti)
PiloteSouris(SAI_Nom, psClicGauche, 10, 5)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`PiloteVitesse(<Vitesse>)`
---

**`<Vitesse> : (Constante de type Entier)`**

Vitesse d'exécution des fonctions PiloteXXXX : 


|   |   |
| --- | --- |
| *pvDéfaut*<br>(Valeur par défaut) | Vitesse d'exécution normale. |
| *pvRalenti* | Vitesse d'exécution ralentie (pour visualiser les actions effectuées). |




## &lt;Table&gt;.ColonnesFiltrées (Fonction)

***En anglais : &lt;Table&gt;.FilteredColumn***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la liste des colonnes filtrées par l'utilisateur dans un champ Table ou Table hiérarchique. 

Ce filtre a été mis en place : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) grâce au menu contextuel de l'icône "loupe" de la colonne. Les filtres utilisateur font partie des FAA (Fonctionnalités Automatiques de vos Applications) disponibles sur les champs Table et Table hiérarchique. Pour plus de détails, consultez [FAA sur les colonnes d'un champ Table](../WDChamp/9500107.md). 

- par programmation grâce à la fonction [TableActiveFiltre](../WDLang1/1000017149.md). 







<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Récupération des filtres
sListeFiltre est chaîne = TABLE_Client.ColonnesFiltrées()
POUR TOUTE CHAÎNE sFiltre DE sListeFiltre SEPAREE PAR RC
	sCol est chaîne = ExtraitChaîne(sFiltre, 1, ";")
	nType est entier = ExtraitChaîne(sFiltre, 2, ";")
	sVal est chaîne = ExtraitChaîne(sFiltre, 3, ";")
	Trace(sCol, nType, sVal)
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Champ Table>.ColonnesFiltrées()`
---

**`<Résultat> : (Chaîne de caractères)`**



- Liste et caractéristiques des colonnes filtrées. 

- Chaîne vide ("") si aucun filtre n'est actif. 




La liste des colonnes filtrées est de la forme : 

```txt
<Colonne filtrée 1> ; <Type du filtre 1> ; <Valeur du filtre 1> + RC + ... + 
<Colonne filtrée N> ; <Type du filtre N> ; <Valeur du filtre N>
```
où : 

- &lt;Colonne filtrée X&gt; correspond au nom de la colonne filtrée. 

- &lt;Type du filtre&gt; correspond au type du filtre sélectionné par l'utilisateur. Les différents types disponibles sont les suivants : 
	


|   |   |
| --- | --- |
| *filtreCommencePar* | Affiche les lignes dont la valeur de la colonne spécifiée "Commence par" les caractères indiqués dans la &lt;Valeur du filtre&gt;. |
| *filtreContient* | Affiche les lignes dont la valeur de la colonne spécifiée "Contient" les caractères indiqués dans la &lt;Valeur du filtre&gt;. |
| *filtreDifférent* | Affiche les lignes dont la valeur de la colonne spécifiée "Est différente" des caractères indiqués dans la &lt;Valeur du filtre&gt;. |
| *filtreEgal* | Affiche les lignes dont la valeur de la colonne spécifiée "Est égale" aux caractères indiqués dans la &lt;Valeur du filtre&gt;. |
| *filtreInférieur* | Affiche les lignes dont la valeur de la colonne spécifiée "Est inférieure" aux caractères indiqués dans la &lt;Valeur du filtre&gt;. |
| *filtreInférieurOuEgal* | Affiche les lignes dont la valeur de la colonne spécifiée "Est inférieure ou égale" aux caractères indiqués dans la &lt;Valeur du filtre&gt;. |
| *filtreNeCommencePasPar* | Affiche les lignes dont la valeur de la colonne spécifiée "Ne commence pas par" les caractères indiqués dans la &lt;Valeur du filtre&gt;. |
| *filtreNeContientPas* | Affiche les lignes dont la valeur de la colonne spécifiée "Ne contient pas" les caractères indiqués dans la &lt;Valeur du filtre&gt;. |
| *filtreNeTerminePasPar* | Affiche les lignes dont la valeur de la colonne spécifiée "Ne termine pas par" les caractères indiqués dans la &lt;Valeur du filtre&gt;. |
| *filtreSupérieur* | Affiche les lignes dont la valeur de la colonne spécifiée "Est supérieure" aux caractères indiqués dans la &lt;Valeur du filtre&gt;. |
| *filtreSupérieurOuEgal* | Affiche les lignes dont la valeur de la colonne spécifiée "Est supérieure ou égale" aux caractères indiqués dans la &lt;Valeur du filtre&gt;. |
| *filtreTerminePar* | Affiche les lignes dont la valeur de la colonne spécifiée "Se termine par" les caractères indiqués dans la &lt;Valeur du filtre&gt;. |



- &lt;Valeur du filtre&gt; correspond à la valeur du filtre saisi par l'utilisateur.




**`<Champ Table> : (Nom de champ)`**

Nom du champ Table ou Table hiérarchique manipulé.

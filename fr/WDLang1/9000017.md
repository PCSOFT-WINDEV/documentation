


## Fonctions de gestion du MCU
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions de gestion du MCU sont les suivantes : 



|   |   |
| --- | --- |
| [EnModeMCU](../WDLang1/1000017027.md) | Identifie si le code est exécuté depuis un code écrit dans une macro utilisateur. |
| [MCUActif](../WDLang1/3030020.md) | Permet de connaître et de modifier l'activation de la gestion du MCU (Macro Code Utilisateur). |
| [MCUOuvre](../WDLang1/3030021.md) | Ouvre la fenêtre d'édition du Macro Code Utilisateur pour la fenêtre en cours. |

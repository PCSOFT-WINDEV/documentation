


## TableEnrouleTout (Fonction)

***En anglais : TableCollapseAll***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Enroule :

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) toute la hiérarchie (tous les noeuds) d'un champ Table hiérarchique.

- toutes les ruptures d'un champ Table. 




![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Les ruptures ne sont pas disponibles sur les tables dans cette version.






<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Enroule tous les noeuds du champ Table hiérarchique "TABLEH_Planning"
TableEnrouleTout(TABLEH_Planning)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`TableEnrouleTout(<Champ Table>)`
---

**`<Champ Table> : (Nom de champ)`**

Nom du champ à manipuler. Ce nom peut correspondre :

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) au nom du champ Table hiérarchique pour lequel tous les noeuds doivent être enroulés.

- au nom du champ Table pour lequel toutes les ruptures doivent être enroulées.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Conditions d'utilisation
<a name="conditions_utilisation_ELTPARAGRAPHE000099"></a>La fonction **TableEnrouleTout** peut être utilisée sur : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) un champ Table hiérarchique.

- un champ Table par programmation. 

- un champ Table fichier chargé en mémoire.  




Cette fonction ne peut pas être utilisée sur les champs Table fichier (accès direct).




## &lt;Variable WDPic&gt;.DessinePoint (Fonction)

***En anglais : &lt;WDPic variable&gt;.DrawPoint***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Dessine un point dans une variable de type [WDPic](../WDLang1/1000023503.md).

![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Les dessins](../WDLang1/3029035.md). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonImageWDPic est un WDPic = "Test.bmp"
// Dessin d'un point jaune clair dans une variable WDPic
MonImageWDPic.DessinePoint(10, 50, JauneClair)
IMG_MonDessin = MonImageWDPic
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Dessiner un point en donnant ses coordonnées 

`<Image WDPic>.DessinePoint(<Abscisse> , <Ordonnée> [, <Couleur du point> [, <Opacité>]])`
---

**`<Image WDPic> : (Variable de type WDPic)`**

Nom de la variable de type [WDPic](../WDLang1/1000023503.md) à manipuler. Seul le calque d'arrière-plan sera manipulé.

**`<Abscisse> : (Entier)`**

Position horizontale (en pixels) du point à dessiner.

**`<Ordonnée> : (Entier)`**

Position verticale (en pixels) du point à dessiner.

**`<Couleur du point> : (Entier ou constante (optionnel))`**

Couleur à affecter au point. Cette couleur peut correspondre à :

- une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)),

- une couleur TSL (obtenue avec la fonction [TSL](../WDLang1/3029057.md)),

- [une couleur prédéfinie du WLangage](../WDLang5/3010002.md).




Si ce paramètre n'est pas précisé, la couleur du point :

- est *Transparent* si la fonction [&lt;Variable WDPic&gt;.Stylo](../WDLang1/1000024636.md) n'a pas été utilisée auparavant,

- correspond à la couleur précisée dans le dernier appel à la fonction [&lt;Variable WDPic&gt;.Stylo](../WDLang1/1000024636.md).




**`<Opacité> : (Entier optionnel)`**

Opacité de la couleur, comprise entre 0 (transparent) et 255 (opacité complète). Une opacité de 128 indique que le pixel concerné aura une transparence de 50% lorsqu'il sera dessiné à l'écran ou sauvé dans une image au format PNG. 

**Remarque** : Cette opacité sera prise en compte si le dessin dans l'image a été initialisé avec la constante *dAvecOpacité* (fonction [&lt;Variable WDPic&gt;.DébutDessin](../WDLang1/1000024471.md)).

### Dessiner un point via une variable de type Point 

`<Image WDPic>.DessinePoint(<Point> [, <Couleur du point> [, <Opacité>]])`
---

**`<Image WDPic> : (Variable de type WDPic)`**

Nom de la variable de type [WDPic](../WDLang1/1000023503.md) à manipuler. Seul le calque d'arrière-plan sera manipulé.

**`<Point> : (Variable de type Point)`**

Nom de la variable de type [Point](../Motscles/1000018908.md) contenant les coordonnées du point à dessiner.

**`<Couleur du point> : (Entier ou constante (optionnel))`**

Couleur à affecter au point. Cette couleur peut correspondre à :

- une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)),

- une couleur TSL (obtenue avec la fonction [TSL](../WDLang1/3029057.md)),

- [une couleur prédéfinie du WLangage](../WDLang5/3010002.md).




Si ce paramètre n'est pas précisé, la couleur du point :

- est *Transparent* si la fonction [&lt;Variable WDPic&gt;.Stylo](../WDLang1/1000024636.md) n'a pas été utilisée auparavant,

- correspond à la couleur précisée dans le dernier appel à la fonction [&lt;Variable WDPic&gt;.Stylo](../WDLang1/1000024636.md).




**`<Opacité> : (Entier optionnel)`**

Opacité de la couleur, comprise entre 0 (transparent) et 255 (opacité complète). Une opacité de 128 indique que le pixel concerné aura une transparence de 50% lorsqu'il sera dessiné à l'écran ou sauvé dans une image au format PNG. 

**Remarque** : Cette opacité sera prise en compte si le dessin dans l'image a été initialisé avec la constante *dAvecOpacité* (fonction [&lt;Variable WDPic&gt;.DébutDessin](../WDLang1/1000024471.md)).

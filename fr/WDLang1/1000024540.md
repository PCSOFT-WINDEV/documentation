


## &lt;Variable Document&gt;.UndoRedoSupprimeTout (Fonction)

***En anglais : &lt;Document variable&gt;.UndoRedoDeleteAll***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime tout l'historique du mécanisme d'Undo/Redo d'une variable de type [Document](../WDLang1/1000022461.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonDocument est un Document

// Supprime l'historique des actions effectuées sur le document
MonDocument.UndoRedoSupprimeTout()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Document>.UndoRedoSupprimeTout()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la suppression a bien été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire




**`<Document> : (Variable de type Document)`**

Nom de la variable de type [Document](../WDLang1/1000022461.md) à manipuler.

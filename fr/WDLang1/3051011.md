
## MCIErr (Fonction)

***En anglais : MCIErr***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le numéro de la dernière erreur survenue sur une fonction MCI (Media Control Interface).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Numéro de la dernière erreur
ResErreur = MCIErr()
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = MCIErr()`
---

**`<Résultat> : (Entier)`**

Numéro de la dernière erreur survenue sur une fonction MCI.

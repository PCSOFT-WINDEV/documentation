


## &lt;Kanban&gt;.Occurrence (Fonction)

***En anglais : &lt;Kanban&gt;.Count***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le nombre de cartes ou de listes dans un champ Kanban.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affiche le nombre total de cartes présentes dans le champ Kanban
LIB_NbTâches = "Total : " + KANBAN_MonKanban.Occurrence(koCarteTotal)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Champ Kanban>.Occurrence(<Comptage>)`
---

**`<Résultat> : (Entier)`**

Nombre d'éléments.

**`<Champ Kanban> : (Nom de champ)`**

Nom du champ Kanban manipulé.

**`<Comptage> : (Constante de type Entier)`**

Type d'information demandée :


|   |   |
| --- | --- |
| *koCarteTotal* | Nombre total de cartes dans le champ. |
| *koCarteVisible* | Nombre de cartes visibles. |
| *koListeTotal* | Nombre total de listes de Kanban dans le champ. |
| *koListeVisible* | Nombre de listes de Kanban visibles. |


Remarque : Le nombre de cartes et de listes de Kanban visibles tient compte des cartes et des listes accessibles par le déplacement de l'ascenseur.




## &lt;Variable Image&gt;.ModifieTSL (Fonction)

***En anglais : &lt;Image variable&gt;.ModifyHSL***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie la teinte, la saturation et la luminosité d'une image contenue dans une variable de type [Image](../WDLang1/1000019650.md).

![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Les dessins](../WDLang1/3029035.md). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonImage est une Image = "Test.bmp"
// Change la teinte, la luminosité, la saturation
// de l'image contenue dans le champ IMG_Image1
MonImage.ModifieTSL(50, 50, 50)
IMG_MonDessin = MonImage
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Variable Image>.ModifieTSL(<Variation de Teinte> , <Variation de Saturation> , <Variation de Luminosité> [, <Image destination>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le changement a été effectué, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Variable Image> : (Variable de type Image)`**

Nom de la variable de type [Image](../WDLang1/1000019650.md) à manipuler.

**`<Variation de Teinte> : (Entier)`**

Valeur comprise entre -360 et +360. Cette valeur sera ajoutée à la teinte actuelle de chaque pixel pour obtenir une nouvelle teinte.

**`<Variation de Saturation> : (Entier)`**

Valeur comprise entre -100 et +100. Cette valeur sera ajoutée à la saturation actuelle de chaque pixel pour obtenir une nouvelle saturation.

**`<Variation de Luminosité> : (Entier)`**

Valeur comprise entre -100 et +100. Cette valeur sera ajoutée à la luminosité actuelle de chaque pixel pour obtenir une nouvelle luminosité.

**`<Image destination> : (Nom de champ ou variable de type Image)`**

Image dans laquelle l'image transformée sera dessinée. Cette image peut correspondre : 

- au nom d'un champ Image. 

- au nom d'une variable de type [Image](../WDLang1/1000019650.md).
	




Si ce paramètre n'est pas spécifié, la transformation s'effectuera sur &lt;Variable Image&gt;.

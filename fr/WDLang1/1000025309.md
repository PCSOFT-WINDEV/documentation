
## PiloteCombo (Fonction)

***En anglais : EmulateCombo***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Pilote une des actions suivantes sur un champ Combo : 

- clic sur le bouton permettant de dérouler la liste du champ Combo. 

- sélection d'un élément dans un champ Combo.



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Sélection du 5ème élément du champ Combo
PiloteCombo(COMBO_Choix, 5)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Piloter le champ Combo via l'identifiant de l'élément

`PiloteCombo(<Champ Combo> [, <Indice de l'élément>])`
---

**`<Champ Combo> : (Nom de champ)`**

Nom du champ de type Combo.

**`<Indice de l'élément> : (Entier optionnel)`**

Indice de l'élément à sélectionner dans le champ Combo. Si ce paramètre n'est pas précisé, un clic sur le bouton déroulant la liste du champ Combo est simulé.


<a name="SYNTAXE2"></a>

### Piloter le champ Combo via le libellé de l'élément

`PiloteCombo(<Champ Combo> [, <Libellé de l'élément>])`
---

**`<Champ Combo> : (Nom de champ)`**

Nom du champ de type Combo.

**`<Libellé de l'élément> : (Chaîne de caractères optionnelle)`**

Libellé de l'élément à sélectionner dans le champ (le premier élément ayant le libellé recherché sera sélectionné).



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Cette fonction doit être utilisée uniquement dans le code des tests automatiques.

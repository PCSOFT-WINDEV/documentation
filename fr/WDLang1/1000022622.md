


## DocOuvre (Fonction)

***En anglais : DocOpen***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ouvre un fichier Traitement de texte (format docx) en mode Lecture/Ecriture. Le fichier docx est ouvert et bloqué jusqu'à sa fermeture.

**Remarque** : Pour créer un fichier docx sans utiliser un fichier initial, il suffit de créer une variable de type [Document](../WDLang1/1000022461.md) (la fonction **DocOuvre** est alors inutile).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Utilisation d'une variable Document
d est un Document
d = DocOuvre("C:\Users\test\Documents\fichier.docx")

SI ErreurDétectée() ALORS
	Erreur(ErreurInfo())
	RETOUR
FIN
```



```wl
// Utilisation d'un champ Traitement de texte
DocOuvre(TT_MonTT, "C:\Users\test\Documents\fichier.docx")

SI ErreurDétectée() ALORS
	Erreur(ErreurInfo())
	RETOUR
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Associer un fichier docx à une variable Document

`<Résultat> = DocOuvre(<Source du document> [, <Mot de passe>])`
---

**`<Résultat> : (Variable de type Document)`**

Nom de la variable de type [Document](../WDLang1/1000022461.md) correspondant au fichier docx. Cette variable peut être utilisée dans toutes les fonctions [DocXXX](../WDLang1/1000022618.md).

En cas d'erreur, la variable [ErreurDétectée](../WDLang1/3087001.md) est à <u><u><u><u>Vrai</u></u></u></u> et la fonction [ErreurInfo](../WDLang1/3013008.md) permet de connaître la cause de l'erreur.

**`<Source du document> : (Type correspondant à la source)`**

Correspond à un des éléments suivants : 

- Chemin complet ou relatif du fichier docx à ouvrir (260 caractères maximum).

- Nom de la variable de type Buffer contenant le document docx à afficher dans le champ Traitement de texte. 

- Nom de la rubrique mémo contenant le document docx à afficher dans le champ Traitement de texte. Ce paramètre est de la forme : &lt;Fichier de données&gt;.&lt;Rubrique&gt;.




**`<Mot de passe> : (Chaîne de caractères optionnelle)`**

Mot de passe d'ouverture du fichier docx à manipuler (si le fichier est protégé par un mot de passe d'ouverture).

### Associer un fichier docx à un champ Traitement de texte

`<Résultat> = DocOuvre(<Champ Traitement de texte> , <Source du document> [, <Mot de passe>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le fichier docx a été ouvert dans le champ Traitement de texte, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 




**`<Champ Traitement de texte> : (Nom de champ)`**

Nom du champ Traitement de texte dans lequel le fichier docx doit être ouvert.

**`<Source du document> : (Type correspondant à la source)`**

Correspond à un des éléments suivants : 

- Chemin complet ou relatif du fichier docx à ouvrir (260 caractères maximum).

- Nom de la variable de type Buffer contenant le document docx à afficher dans le champ Traitement de texte. 

- Nom de la rubrique mémo contenant le document docx à afficher dans le champ Traitement de texte. Ce paramètre est de la forme : &lt;Fichier de données&gt;.&lt;Rubrique&gt;.




**`<Mot de passe> : (Chaîne de caractères optionnelle)`**

Mot de passe d'ouverture du fichier docx à manipuler (si le fichier est protégé par un mot de passe d'ouverture).

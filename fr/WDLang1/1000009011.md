


## PiloteSouris (Fonction)

***En anglais : EmulateMouse***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Simule un événement de la souris (clic gauche, clic droit, double-clic, ...).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Clic sur le champ de saisie SAI_SaisieLogin
PiloteSouris(SAI_SaisieLogin, psClicGauche, 3, 10)

// Saisie de "MARC" dans ce champ
PiloteSaisie(SAI_SaisieLogin, "Marc", psShift)

// Clic sur le champ Bouton Valider
PiloteSouris(BTN_BoutonValider, psClicGauche, 5, 5)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`PiloteSouris(<Fenêtre ou champ> , <Action à effectuer> [, <X> , <Y>])`
---

**`<Fenêtre ou champ> : (Nom de fenêtre ou nom de champ)`**

Nom de la fenêtre ou du champ sur lequel l'événement de la souris doit avoir lieu.

**`<Action à effectuer> : (Constante (ou combinaison de constantes) de type entier)`**

Indique l'événement de la souris à simuler :


|   |   |
| --- | --- |
| *psAlt* | Maintient la touche Alt enfoncée lors de l'action.<br>Cette constante peut être combiné avec d'autres.<br><br> |
| *psClicDroit* | Effectue un clic droit.<br><br> |
| *psClicGauche* | Effectue un clic gauche.<br><br> |
| *psControl* | Maintient la touche Ctrl enfoncée lors de l'action.<br>Cette constante peut être combiné avec d'autres.<br><br> |
| *psDoubleClicDroit* | Effectue un double-clic droit.<br><br> |
| *psDoubleClicGauche* | Effectue un double-clic gauche.<br><br> |
| *psEnfonceBoutonDroit* | Enfonce le bouton droit.<br><br> |
| *psEnfonceBoutonGauche* | Enfonce le bouton gauche.<br><br> |
| *psRelacheBoutonDroit* | Relâche le bouton droit.<br><br> |
| *psRelacheBoutonGauche* | Relâche le bouton gauche.<br><br> |
| *psShift* | Maintient la touche Maj enfoncée lors de l'action.<br>Cette constante peut être combiné avec d'autres.<br><br> |



**`<X> : (Entier optionnel)`**

Position horizontale (ou abscisse) en pixels où l'action doit avoir lieu. Cette position est relative au coin haut gauche de la fenêtre ou du champ manipulé dans le référentiel de l'écran.

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Les paramètres &lt;X&gt; et &lt;Y&gt; sont optionnels. Si ces paramètres ne sont pas indiqués, l'événement de la souris à simuler est effectué au milieu de la fenêtre ou du champ.![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Ce paramètre est obligatoire.

**`<Y> : (Entier optionnel)`**

Position verticale (ou ordonnée) en pixels où l'action doit avoir lieu. Cette position est relative au coin haut gauche de la fenêtre ou du champ manipulé dans le référentiel de l'écran. 
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Les paramètres &lt;X&gt; et &lt;Y&gt; sont optionnels. Si ces paramètres ne sont pas indiqués, l'événement de la souris à simuler est effectué au milieu de la fenêtre ou du champ.![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Ce paramètre est obligatoire.

## Remarques


### Limitation des champs manipulés
<a name="limitation_des_champs_manipules_ELTPARAGRAPHE000156"></a>La fonction **PiloteSouris** n'est pas compatible avec les champs suivants :

- Champ Image.

- Champ Jauge.

- Champ Libellé.

- Champ OLE.

- Champ Forme.

- Champ HTML.

- Champ Web Caméra.

- Champ Conférence.

- Champ Code-barres.

- Champ Fenêtre interne

- Champ Graphe.




**Attention** : La fonction **PiloteSouris** ne fonctionne pas sur une session TSE non interactive.

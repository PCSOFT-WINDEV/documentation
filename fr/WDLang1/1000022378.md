


## &lt;Caméra&gt;.ChangeDriver (Fonction)

***En anglais : &lt;Camera&gt;.SetDriver***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Connecte le champ "Caméra" à un driver de capture vidéo spécifique. 

**Attention** : 

- Pour utiliser cette fonction, une "caméra" doit être connectée au poste en cours.

- Pour obtenir la liste des drivers de capture vidéo installé sur le poste en cours, utilisez la fonction [VidéoListeDriver](../WDLang1/3079008.md) (avec la constante *viDirectShow*). Pour savoir quel driver est actuellement associé à un champ Caméra, utilisez la propriété [Driver](../Proprietes/2510061.md).



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affiche la liste des drivers vidéo disponibles
// dans le champ LISTE_DriverVidéo
LISTE_DriverVidéo.Ajoute(VidéoListeDriver(viDirectShow))
// Utilise le deuxième driver Vidéo
SI CAM_Camera1.ChangeDriver(LISTE_DriverVidéo[2]) = Faux ALORS
	Erreur("Changement de driver impossible : " + RC + ...
	ErreurInfo())
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Champ Caméra>.ChangeDriver(<Identifiant du driver vidéo>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le driver vidéo associé au champ a été modifié, 

- <u><u><u><u>Faux</u></u></u></u> en cas de problème. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Champ Caméra> : (Nom de champ)`**

Nom du champ Caméra à manipuler.

**`<Identifiant du driver vidéo> : (Chaîne de caractères)`**

Identifiant du driver vidéo à utiliser. Cet identifiant est de la forme :

```txt
<Nom du driver> + TAB + <Version du driver>
```


Cet identifiant correspond au format de liste renvoyé par la fonction [VidéoListeDriver](../WDLang1/3079008.md) (avec la constante *viDirectShow*).

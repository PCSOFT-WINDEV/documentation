


## LecteurPDFInfoXY (Fonction)

***En anglais : PDFReaderInfoXY***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie pour une position donnée dans un champ Lecteur PDF (coordonnées d'un point du champ) : 

- soit le numéro de la page pour la position indiquée, 

- soit la position X (en mm) pour la position indiquée, 

- soit la position Y (en mm) pour la position indiquée.



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Numéro de page
nPage est un entier
nPage = LecteurPDFInfoXY(PDF_SansNom1, pdfNuméroPage, SourisPosX(), SourisPosY())

// Position de X en millimètres
rPosX est un réel
rPosX = LecteurPDFInfoXY(PDF_SansNom1, pdfCoordonnéeX, SourisPosX(), SourisPosY())

// Position de Y en millimètres
rPosY est un réel
rPosY = LecteurPDFInfoXY(PDF_SansNom1, pdfCoordonnéeY, SourisPosX(), SourisPosY())
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = LecteurPDFInfoXY(<Champ Lecteur PDF> , <Type d'information> , <X> , <Y>)`
---

**`<Résultat> : (Réel)`**

Information demandée. 

**`<Champ Lecteur PDF> : (Nom de champ)`**

Nom du champ Lecteur PDF à manipuler. 

Si ce paramètre correspond à une chaîne vide (""), le champ manipulé est le champ auquel appartient l'événement en cours.

**`<Type d'information> : (Constante de type Entier)`**

Type d'information demandée :


|   |   |
| --- | --- |
| *pdfCoordonnéeX* | Abscisse de la position spécifiée sur la page (exprimée en millimètres). |
| *pdfCoordonnéeY* | Ordonnée de la position spécifiée sur la page (exprimée en millimètres). |
| *pdfNuméroPage* | Numéro de la page contenant la position spécifiée. |
| *pdfOrigineEcran* | Par défaut, le point (0,0) correspond à la ligne 1, colonne 1 du champ Traitement de texte manipulé (les ascenseurs étant situés à l'origine).<br>Si la constante *pdfOrigineEcran* est combinée avec les constantes précédentes, le point (0,0) correspond à l'origine de l'écran.<br>Cette constante ne peut pas être utilisée seule. |



**`<X> : (Entier)`**

Coordonnée X (en pixels) à analyser. Cette coordonnée est relative au champ (si la constante *pdfOrigineEcran* n'est pas précisée).

**`<Y> : (Entier)`**

Coordonnée Y (en pixels) à analyser. Cette coordonnée est relative au champ (si la constante *pdfOrigineEcran* n'est pas précisée).


## PiloteVoletAttache (Fonction)

***En anglais : EmulatePaneDock***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Pilote la réintégration d'un volet détaché dans le champ Onglet dynamique associé.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Réintègre le volet "Alias_1" dans le champ Onglet à la 3ème position
PiloteVoletAttache(ONG_MonOngletDyn, "Alias_1", 3)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`PiloteVoletAttache(<Nom du champ> , <Alias> , <Numéro du volet>)`
---

**`<Nom du champ> : (Chaîne de caractères)`**

Nom du champ Onglet dynamique auquel le volet doit être rattaché. 

**`<Alias> : (Chaîne de caractères)`**

Alias du volet à réintégrer dans le champ Onglet dynamique associé. 

**`<Numéro du volet> : (Entier)`**

Numéro correspondant à la position souhaitée du volet dans le champ Onglet dynamique après réintégration. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Cette fonction doit être utilisée uniquement dans le code des tests automatiques.


## SansCaractèreDroite (Fonction)

***En anglais : NoRightCharacter***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie une chaîne de caractères sans les caractères spécifiés situés à droite de la chaîne initiale.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
sChaîne est une chaîne = "<<Je suis une chaîne de caractères>>"
sRésultat est une chaîne = SansCaractèreDroite(sChaîne, ">")
// sRésultat contient "<<Je suis une chaîne de caractères"
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = SansCaractèreDroite(<Chaîne initiale> , <Caractères à supprimer>)`
---

**`<Résultat> : (Chaîne de caractères)`**

Chaîne de caractères sans les caractères spécifiés situés à droite de la chaîne initiale. 

**`<Chaîne initiale> : (Chaîne de caractères)`**

Chaîne de caractères initiale. Cette chaîne n'est pas modifiée. 

**`<Caractères à supprimer> : (Chaîne de caractères)`**

Liste des caractères à supprimer à droite de la chaîne initiale. 

Si ce paramètre correspond à une chaîne vide (""), &lt;Résultat&gt; correspondra à la chaîne initiale. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Fonction SansCaractèreDroite et UNICODE
<a name="fonction_docparampagetitleshort_unicode_ELTPARAGRAPHE000056"></a>Si la chaîne passée en paramètre est au format ANSI, le résultat sera au format ANSI.

Si la chaîne passée en paramètre est au format UNICODE, le résultat sera au format UNICODE.

**Remarque** : Si le résultat de la fonction **SansCaractèreDroite** utilisée sur une chaîne ANSI est affecté à une chaîne UNICODE (et inversement), la conversion ANSI/UNICODE sera implicitement réalisée.

Pour plus de détails, consultez [Gestion de l'UNICODE](../WDLang1/3024038.md).

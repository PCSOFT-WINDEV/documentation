
## RéseauDéconnecte (Fonction)

***En anglais : NetworkDisconnect***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Déconnecte un lecteur réseau.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Déconnexion de la lettre réseau "E:"
ResDéconnexion est un entier = RéseauDéconnecte("E:")

SELON ResDéconnexion
	CAS 16
		Erreur("Mauvaise valeur pour la déconnexion")
	CAS 17
		Erreur("Connexion en cours d'utilisation")
	CAS 18
		Erreur("Un fichier est ouvert, impossible de se déconnecter")
	CAS 19
		Erreur("Tentative de déconnexion sans connexion")
	AUTRE CAS
		Info("Lecteur déconnecté avec succès")
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = RéseauDéconnecte(<Lettre du lecteur>)`
---

**`<Résultat> : (Entier)`**



- 0 si la déconnexion a été effectuée, 

- un code d'erreur (valeur supérieure à 0) dans le cas contraire.




**`<Lettre du lecteur> : (Chaîne de caractères)`**

Lecteur réseau à déconnecter au format "&lt;Lettre du lecteur&gt;**:**".  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Codes d'erreur
<a name="codes_erreur_ELTPARAGRAPHE000036"></a>Les codes d'erreur retournés sont les suivants :

- 16 : Mauvaise valeur pour la déconnexion.

- 17 : Connexion en cours d'utilisation.

- 18 : Un fichier est ouvert, impossible de se déconnecter.

- 19 : Tentative de déconnexion sans connexion.




Le message correspondant au code d'erreur peut être connu grâce à la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.




## &lt;Variable picCalque&gt;.DessineLigne (Fonction)

***En anglais : &lt;picLayer variable&gt;.DrawLine***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Dessine une ligne dans une variable de type [picCalque](../WDLang1/1000024604.md).![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Les dessins](../WDLang1/3029035.md). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonImage est un WDPic = "Test.bmp"
MonCalque est un picCalque

// Dessin d'une ligne jaune clair dans une variable de type picCalque
MonCalque.DessineLigne(10, 10, 50, 50, JauneClair)
IMG_MonDessin = MonImage
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Image picCalque>.DessineLigne(<X1> , <Y1> , <X2> , <Y2> [, <Couleur de la ligne> [, <Largeur du trait>]])`
---

**`<Image picCalque> : (Variable de type picCalque)`**

Nom de la variable de type [picCalque](../WDLang1/1000024604.md) à manipuler.

**`<X1> : (Entier)`**

Abscisse en pixels du point de départ de la ligne.

**`<Y1> : (Entier)`**

Ordonnée en pixels du point de départ de la ligne.

**`<X2> : (Entier)`**

Abscisse en pixels du point d'arrivée de la ligne.

**`<Y2> : (Entier)`**

Ordonnée en pixels du point d'arrivée de la ligne.

**`<Couleur de la ligne> : (Entier ou constante (optionnel))`**

Couleur de la ligne. Cette couleur peut correspondre à :

- une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)),

- une couleur TSL (obtenue avec la fonction [TSL](../WDLang1/3029057.md)),

- [une couleur prédéfinie du WLangage](../WDLang5/3010002.md),

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) une variable de type [Couleur](../WDLang1/1000019661.md).


Si ce paramètre n'est pas précisé, la couleur de la ligne :

- est *Transparent* si la fonction [&lt;Image&gt;.Stylo](../WDLang1/1000024016.md) n'a pas été utilisée auparavant,

- correspond à la couleur précisée dans le dernier appel à la fonction [&lt;Image&gt;.Stylo](../WDLang1/1000024016.md).




**`<Largeur du trait> : (Entier optionnel)`**

Épaisseur (en pixels) de la ligne à dessiner. Si ce paramètre n'est pas précisé, la valeur prise en compte peut être :

- la dernière valeur précisée dans le dernier appel à la fonction [&lt;Image&gt;.Stylo](../WDLang1/1000024016.md).

- 1 si la fonction [&lt;Image&gt;.Stylo](../WDLang1/1000024016.md) n'a pas été utilisée auparavant.

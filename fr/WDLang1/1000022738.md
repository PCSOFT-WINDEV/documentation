
## RuptureSupprime (Fonction)

***En anglais : BreakDelete***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime une rupture dans un champ Table ou dans un champ Zone répétée. Cette rupture a été précédemment ajoutée via la fonction [RuptureAjoute](../WDLang1/1000022737.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ajout d'une rupture sur la date
xRupture est Champ
xRupture <- RuptureAjoute(TABLE_Commande.COL_Date, "RuptDate", ruptHaut, 50) 

//...

// Supprime la rupture "xRupture"
RuptureSupprime(xRupture)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`RuptureSupprime(<Rupture>)`
---

**`<Rupture> : (Variable de type Champ)`**

Nom de la variable de type [Champ](../WDLang1/1000020967.md) correspondant à la rupture à supprimer. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- La fonction **RuptureSupprime** supprime en même temps le haut et le bas de rupture s'ils existent.

- Pour rendre invisible une rupture définie sous l'éditeur, utilisez la propriété [Visible](../Proprietes/2510138.md).

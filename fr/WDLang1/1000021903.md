
## PointDAttacheSupprimeTout (Fonction)

***En anglais : AttachPointDeleteAll***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime tous les points d'attache d'un champ. 

Rappel : Les points d'attache permettent d'utiliser un champ Fil sur un champ. Tant que le champ n'a aucun point d'attache, aucun fil ne peut partir ou arriver du champ.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Supprime tous les points d'attache existant
PointDAttacheSupprimeTout(IMG_Logo)
// Définit un nouveau point d'attache
IdPointDattache est un entier
IdPointDattache = PointDAttacheAjoute(IMG_Logo, 100, 30))

NomFil est un Champ
NomFil <- ChampCrée("", typFil)
NomFil.ChampDépart = IMG_Logo
NomFil.ChampArrivée = BTN_Modifier

// Utilisation du nouveau point d'attache
NomFil.PointDAttacheDépart = IdPointDattache
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`PointDAttacheSupprimeTout(<Champ manipulé>)`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ pour lequel tous les points d'attache doivent être supprimés.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- La fonction [PointDAttacheAjoute](../WDLang1/1000021902.md) permet de définir un nouveau point d'attache.

- Les propriétés [PointDAttacheArrivée](../Proprietes/1000021908.md) et [PointDAttacheDépart](../Proprietes/1000021907.md) permettent d'utiliser ces nouveaux points d'attache.

- Tant que le champ n'a aucun point d'attache, aucun fil ne peut partir ou arriver du champ.

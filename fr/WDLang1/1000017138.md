
## dbgEtatLog (Fonction)

***En anglais : dbgLogStatus***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie et modifie l'état du log d'exécution.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Lance le log d'exécution
dbgActiveLog()
...
// Met le log d'exécution en pause
dbgEtatLog(LogPause)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître l'état du log d'exécution

`<Résultat> = dbgEtatLog()`
---

**`<Résultat> : (Constante de type Entier)`**

Etat actuel du log d'exécution :


|   |   |
| --- | --- |
| *LogActif* | Le log d'exécution est actif. |
| *LogInactif* | Le log d'exécution n'a pas été activé.<br><br> |
| *LogPause* | Le log d'exécution est en pause. |






<a name="SYNTAXE2"></a>

### Modifier l'état du log d'exécution

`dbgEtatLog(<Ancien état> , <Nouvel état>)`
---

**`<Ancien état> : (Constante de type Entier)`**

Etat d'activité du log d'exécution avant la modification effectuée par la fonction **dbgEtatLog**.


|   |   |
| --- | --- |
| *LogActif* | Le log d'exécution est actif. |
| *LogInactif* | Le log d'exécution n'a pas été activé.<br><br> |
| *LogPause* | Le log d'exécution est en pause. |



**`<Nouvel état> : (Constante de type Entier)`**

Nouvel état du log d'exécution :


|   |   |
| --- | --- |
| *LogActif* | Active le log d'exécution (après une pause). |
| *LogInactif* | Arrête définitivement le log d'exécution. Pour le réactiver, il faudra utiliser la fonction [dbgActiveLog](../WDLang1/1000017137.md).<br><br> |
| *LogPause* | Arrête momentanément le log d'exécution. |

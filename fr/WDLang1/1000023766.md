


## &lt;Arbre&gt;.RécupèreIdentifiant (Fonction)

***En anglais : &lt;TreeView&gt;.Identifier***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie l'identifiant associé à un élément (noeud ou feuille) dans un champ Arbre.

Exemple de champ Arbre :
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Arbre_Vocabulaire.gif)

<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Renvoie l'identifiant du noeud "Desserts" dans le champ Arbre "ARBRE_RecetteTV"
Res = ARBRE_RecetteTV.RécupèreIdentifiant("Recettes" + TAB + "Desserts")
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Champ Arbre>.RécupèreIdentifiant(<Chemin de l'élément>)`
---

**`<Résultat> : (Chaîne de caractères)`**



- Identifiant de l'élément spécifié, 

- Chaîne vide ("") si aucun identifiant n'est associé à l'élément ou si l'élément spécifié n'existe pas.




Si l'identifiant (défini avec la fonction [&lt;Arbre&gt;.Ajoute](../WDLang1/1000023422.md) ou [&lt;Arbre&gt;.Modifie](../WDLang1/1000023763.md)) n'est pas une chaîne de caractères mais un entier, il peut être nécessaire de convertir le résultat en entier pour manipuler cet identifiant (fonction [Val](../WDLang1/3024037.md)).

**`<Champ Arbre> : (Nom de champ)`**

Nom du champ Arbre à manipuler.

**`<Chemin de l'élément> : (Chaîne de caractères)`**

Chemin complet de l'élément à manipuler. Ce paramètre est de la forme :

```txt
"<Nom de la racine>" + TAB + ["<Nom du 1er noeud>" + TAB + ...
["<Nom du 2ème noeud>" + TAB + [...]]]"<Nom de la feuille>"
```




<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Récupérer l'identifiant d'un élément avec gestion des doublons
<a name="recuperer_identifiant_element_avec_gestion_des_doublons_ELTPARAGRAPHE000191"></a>Pour récupérer l'identifiant de l'élément sélectionné en prenant en compte la gestion des doublons, le paramètre &lt;Chemin de l'élément&gt; doit correspondre au code suivant :

```txt
<Champ Arbre>.Select(Vrai)
```


Par exemple :

```wl
Trace(ARBRE_Arbre1.Select() + "/" + ...
	ARBRE_Arbre1.RécupèreIdentifiant(ARBRE_Arbre1.Select(Vrai)))
```

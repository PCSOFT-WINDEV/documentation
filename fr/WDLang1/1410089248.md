
## Allemagne.VérifieUStIdNr (Fonction)

***En anglais : Germany.CheckUStIdNr***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Teste la validité d'un numéro UStIdNr (numéro d'identification d'entreprise en Allemagne).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Test d'une variable
MonNuméroUStIdNr est une chaîne = "DE262843185"
Allemagne.VérifieUStIdNr(MonNuméroUStIdNr)	// renvoie Vrai

// Test d'un champ
SAI_NuméroUStIdNr = "ABC"
Allemagne.VérifieUStIdNr(SAI_NuméroUStIdNr)	// renvoie Faux
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = Allemagne.VérifieUStIdNr(<Numéro à vérifier>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'élément spécifié contient un numéro UStIdNr valide.

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Numéro à vérifier> : (Nom de champ ou variable de type chaîne)`**

Numéro UStIdNr à vérifier. Ce paramètre peut correspondre : 

- au nom d'une variable de type chaîne.

- au nom d'un champ.

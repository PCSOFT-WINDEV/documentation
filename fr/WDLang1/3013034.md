
## FinAutomatismeProcédure (Fonction)

***En anglais : EndAutomatedProcedure***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Arrête l'exécution (en cours ou à venir) d'une procédure automatique (définie par les propriétés de la procédure sous l'éditeur de code). Cette procédure automatique peut être :

- une procédure automatique non exécutée

- une procédure à appel différé

- une procédure à répétition temporisée

- une procédure dans un thread indépendant.





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// La procédure VérifSaisie est une procédure différée automatique appelée
// à chaque modification d'un champ de saisie. 
// Dans le code A chaque modification du champ, la procédure automatique
// est arrêtée et relancée manuellement
FinAutomatismeProcédure(VérifSaisie)
VérifSaisie(MoiMême)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`FinAutomatismeProcédure([<Procédure WLangage>])`
---

**`<Procédure WLangage> : (Nom de procédure optionnel)`**

Nom de la procédure automatique à arrêter. Si ce paramètre n'est pas spécifié, la procédure en cours est arrêtée.  

![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) La procédure en cours d'exécution ira jusqu'à son terme. Seule la répétition sera stoppée.

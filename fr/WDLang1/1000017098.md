
## ServiceDésinstalle (Fonction)

***En anglais : ServiceUninstall***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Désinstalle un service. Ce service peut être un service présent sur le poste local ou sur un poste distant.

![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) Désactive tous les "runlevels" du service mais n'efface aucun fichier (y compris les scripts de démarrage).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Désinstalle le service présent dans la table TABLE_Service
SI ServiceDésinstalle(COL_Nom[TABLE_Service]) = Faux ALORS
	Erreur(ErreurInfo(errMessage))
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = ServiceDésinstalle(<Nom du service> [, <Poste distant>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le service s'est désinstallé, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Nom du service> : (Chaîne de caractères)`**

Nom du service à désinstaller.

**`<Poste distant> : (Chaîne de caractères optionnelle)`**

Adresse IP du poste distant au format xxx.xxx.xxx.xxx (obtenu par la fonction [NetAdresseIP](../WDLang3/3056023.md)) ou nom du poste distant (obtenu par la fonction [NetNomMachine](../WDLang3/3056014.md)). Ce paramètre permet de désinstaller un service sur un autre poste. Pour utiliser cette option, l'utilisateur doit être administrateur de la machine distante. 
Si ce paramètre n'est pas précisé ou correspond à une chaîne vide, le service est manipulé en local.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Cas particulier
<a name="cas_particulier_ELTPARAGRAPHE000070"></a>La désinstallation d'un service n'est pas immédiate. Une demande est envoyée au système. Cette demande sera réalisée lorsque le service se terminera. Le service pourra être terminé lorsque aucune instance du service ne sera utilisée.

Si le service est en cours d'exécution, il sera automatiquement arrêté avant d'être désinstallé.




## DocVersPDF (Fonction)

***En anglais : DocToPDF***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Convertit un document de type Traitement de texte en un fichier PDF. 

<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Export en PDF d'une variable de type Document
d est un Document = "texte.docx"
SI DocVersPDF(d, fRepExe() + [fSep] + "MonDoc.pdf") ALORS
	Info("Document converti en PDF")
FIN
```
<a name="Exemple2"></a>

```wl
// Export en PDF du contenu d'un champ Traitement de texte
// Code de clic du bouton "Export PDF" 
DocVersPDF(TT_MonDoc, fRepExe() + [fSep] + "tt.pdf")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = DocVersPDF(<Document à convertir> , <Nom du fichier PDF>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le fichier PDF a été créé, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Document à convertir> : (Variable de type Document ou chaîne de caractères)`**

Document à manipuler. Ce document peut correspondre à : 

- une variable de type [Document](../WDLang1/1000022461.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) un champ Traitement de texte.




**`<Nom du fichier PDF> : (Chaîne de caractères)`**

Nom et chemin complet du fichier PDF à créer. Si le fichier existe déjà, il est remplacé par le nouveau fichier.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Les polices "otf" (OpenType) ne sont pas gérées par la génération PDF. 

- Les filigranes ne sont pas gérés par la génération PDF.

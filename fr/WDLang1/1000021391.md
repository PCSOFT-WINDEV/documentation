


## Fonctions de gestion des champs Onglet
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions permettant de manipuler les champs Onglet sont les suivantes : 



|   |   |
| --- | --- |
| [OngletChargeConfiguration](../WDLang1/1000021370.md) | Restaure la configuration des volets dynamiques d'un champ Onglet. |
| [OngletDuChamp](../WDLang1/3025028.md) | Renvoie le nom du champ Onglet qui contient le champ spécifié. |
| [OngletEtat](../WDLang1/1000021637.md) | Renvoie l'état d'un volet d'onglet dynamique (identifié par son alias). |
| [OngletFerme](../WDLang1/1000021362.md) | Ferme un volet dynamique dans un champ Onglet. |
| [OngletFermeTout](../WDLang1/1000021372.md) | Ferme tous les volets dynamiques d'un champ Onglet. |
| [OngletOuvre](../WDLang1/1000021369.md) | Ouvre un nouveau volet dans un champ Onglet dynamique. |
| [OngletSauveConfiguration](../WDLang1/1000021371.md) | Sauvegarde la configuration des volets dynamiques d'un champ Onglet sous forme d'une chaîne de caractères. |
| [VoletDuChamp](../WDLang1/1000021625.md) | Renvoie le nom du volet (statique ou dynamique) d'un champ Onglet associé à un champ. |

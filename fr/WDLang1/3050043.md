
## MatCompresse (Fonction)

***En anglais : MatStack***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Compresse la place mémoire occupée par une matrice.

**Remarque** : La compression est nécessaire si vous manipulez une matrice importante sur laquelle de nombreux calculs ont été effectués. Si cette matrice comporte plusieurs colonnes et/ou lignes nulles, la compression permet de réduire sensiblement la taille mémoire de la matrice.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple
<a class="notetitle" target="_blank" href="$DOC$=1000003050043&name=matcompresse_fonction&product=WD">Voir des exemples supplémentaires</a>

```wl
// Compression d'une matrice
MatCompresse("MaMatrice")
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`MatCompresse(<Matrice>)`
---

**`<Matrice> : (Chaîne de caractères)`**

Nom de la matrice à compresser, défini avec la fonction [MatCrée](../WDLang1/3050058.md).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Existence de la matrice à compresser
<a name="existence_matrice_compresser_ELTPARAGRAPHE000081"></a>L'existence de la matrice n'est pas vérifiée. Pour vérifier l'existence d'une matrice, utilisez la fonction [MatExiste](../WDLang1/3050076.md).


<a name="NOTE0_2"></a>




### Gestion des erreurs
<a name="gestion_des_erreurs_ELTPARAGRAPHE000093"></a>**Attention** : La fonction **MatCompresse** ne retourne pas de résultat. Pour savoir si des erreurs ont été générées par la compression, utilisez la fonction [MatErreur](../WDLang1/3050044.md) sur cette matrice. Le libellé de l'erreur peut être connu grâce à la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.

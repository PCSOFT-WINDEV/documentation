


## &lt;Traitement de texte&gt;.RechercheElémentParNom (Fonction)

***En anglais : &lt;Word Processing&gt;.FindElementByName***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Recherche un élément par son nom dans l'ensemble d'un champ Traitement de texte.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Supprime l'image appelée "Image logo" du document
ElémentTrouvé est un docElément <- TT_MonDocument.RechercheElémentParNom("Image Logo")
SI ElémentTrouvé <> Null ALORS
	DocSupprime(ElémentTrouvé)
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Champ Traitement de texte>.DocRechercheElementParNom(<Nom de l'élément>)`
---

**`<Résultat> : (Variable de type docElément)`**



- Variable de type [docElément](../WDLang1/1000022484.md) correspondant à l'élément cherché, 

- Null si aucun élément n'a été trouvé.




**`<Champ Traitement de texte> : (Nom de champ)`**

Nom du champ Traitement de texte à manipuler.

**`<Nom de l'élément> : (Chaîne de caractères)`**

Texte à rechercher dans le champ.

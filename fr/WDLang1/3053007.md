
## MemModifie (Fonction)

***En anglais : MemModify***

<a name="Utilisation"></a>
<a name="description"></a>
Modifie un élément dans une zone mémoire.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Remet à 0 toutes les valeurs supérieures à 100
// dans la zone mémoire "MaZoneMémoire"
POUR Indice = 1 _A_ MemOccurrence("MaZoneMémoire")
	ResValeur = MemRécupère("MaZoneMémoire", Indice)
	SI ResValeur > 100 ALORS MemModifie("MaZoneMémoire", 0, 0, Indice)
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`MemModifie(<Zone mémoire> , <Valeur modifiée> , <Valeur de retour> , <Indice>)`
---

**`<Zone mémoire> : (Chaîne de caractères)`**

Nom de la zone mémoire, défini par la fonction [MemCrée](../WDLang1/3053005.md).

**`<Valeur modifiée> : (Chaîne de caractères, entier ou réel)`**

Valeur modifiée dans la zone mémoire. Cette valeur correspond à l'argument de tri utilisé par la fonction [MemTrie](../WDLang1/3053020.md). Cette valeur peut être récupérée grâce à la fonction [MemValClé](../WDLang1/3053016.md).

**`<Valeur de retour> : (Chaîne de caractères)`**

Valeur retournée par les fonctions [MemRécupère](../WDLang1/3053012.md) ou [MemPositionne](../WDLang1/3053018.md).

**`<Indice> : (Entier)`**

Indice de l'élément à modifier. L'indice d'un élément peut être obtenu grâce aux fonctions [MemEnCours](../WDLang1/3053008.md) ou [MemRecherche](../WDLang1/3053015.md).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Existence de la zone mémoire
<a name="existence_zone_memoire_ELTPARAGRAPHE000236"></a>L'existence de la zone mémoire n'est pas vérifiée.

Si la zone mémoire spécifiée n'existe pas, une erreur du WLangage est générée lors de l'exécution de la fonction **MemModifie**. 

Pour vérifier l'existence d'une zone mémoire, utilisez la fonction [MemExiste](../WDLang1/3053006.md).
<a name="NOTE0_2"></a>




### Tri des éléments
<a name="tri_des_elements_ELTPARAGRAPHE000109"></a>L'élément modifié dans la zone mémoire n'est pas trié. Il est nécessaire d'utiliser la fonction [MemTrie](../WDLang1/3053020.md) pour trier l'ensemble de la zone mémoire.

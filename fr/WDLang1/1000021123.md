


## &lt;Diagramme de Gantt&gt;.HeureAffichée (Fonction)

***En anglais : &lt;Gantt chart&gt;.TimeDisplayed***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Définit la plage horaire affichée dans une colonne d'un champ Table (ou Table hiérarchique) de type Diagramme de Gantt.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Change le mode d'affichage
COL_Gantt.ChangeMode(ganttModeEnJour)
// Définit la plage horaire affichée
COL_Gantt.HeureAffichée("0900", "1930")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Colonne Gantt>.HeureAffichée(<Heure de début> , <Heure de fin>)`
---

**`<Colonne Gantt> : (Nom de champ)`**

Nom de la colonne (d'un champ Table ou Table hiérarchique) de type Diagramme de Gantt à manipuler.

**`<Heure de début> : (Chaîne de caractères)`**

Heure de début de l'affichage (au format HHMM ou HH).

**`<Heure de fin> : (Chaîne de caractères)`**

Heure de fin de l'affichage (au format HHMM ou HH).

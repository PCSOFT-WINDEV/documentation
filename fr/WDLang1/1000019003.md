


## AgendaSelectPlus (Fonction)

***En anglais : OrganizerSelectPlus***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Sélectionne un rendez-vous dans un champ Agenda.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Sélectionne le premier rendez-vous du champ Agenda
AgendaSelectPlus(AGD_MonAgenda, 1)

// Récupère la description de ce rendez-vous
tabRendezVous est un tableau de RendezVous
tabRendezVous = AgendaListeRendezVous(AGD_MonAgenda, agRdvSélectionné)
```
<a name="Exemple2"></a>

```wl
// Ajoute un rendez-vous dans le champ Agenda puis sélectionne ce rendez-vous
IndiceAjout est un entier
gMonRendezVous est un RendezVous

IndiceAjout = AgendaAjouteRendezVous(AGD_MonAgenda, gMonRendezVous)
AgendaSelectPlus(AGD_MonAgenda, IndiceAjout)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Sélectionner un rendez-vous identifié par son indice

`AgendaSelectPlus(<Champ Agenda> , <Indice du rendez-vous>)`
---

**`<Champ Agenda> : (Nom de champ)`**

Nom du champ Agenda à manipuler. Ce champ peut correspondre : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) au champ Agenda d'une fenêtre.




**`<Indice du rendez-vous> : (Entier)`**

Indice du rendez-vous à sélectionner. Les indices des rendez-vous commencent à 1.


<a name="SYNTAXE2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 
### Sélectionner un rendez-vous identifié par une variable de type RendezVous

`AgendaSelectPlus(<Champ Agenda> , <Rendez-vous à sélectionner>)`
---

**`<Champ Agenda> : (Nom de champ)`**

Nom du champ Agenda à manipuler dans une fenêtre. 

**`<Rendez-vous à sélectionner> : (Variable de type RendezVous)`**

Nom de la variable de type [RendezVous](../WDLang1/1000019244.md) correspondant au rendez-vous à sélectionner.

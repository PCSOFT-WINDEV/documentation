


## Continent (Type de variable)

***En anglais : Continent***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **Continent** permet de connaître les caractéristiques d'un continent. Les caractéristiques de ce continent peuvent être : 

- récupérées avec les fonctions [ContinentRécupère](../WDLang1/1000024933.md) et [ContinentListe](../WDLang1/1000024932.md). 

- connues à l'aide de différentes propriétés WLangage. 




**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Récupération des informations sur l'Europe
oContinent est un Continent = ContinentRécupère(continentEurope)
// Affichage du libellé 
LIB_Continent = oContinent.Nom
```

<a name="NOTE0"></a>

## Remarques
<a name="NOTE0_1"></a>


### Propriétés spécifiques à la description des variables de type Continent
<a name="proprietes_specifiques_description_des_variables_type_continent_ELTPARAGRAPHE000048"></a>Les propriétés suivantes peuvent être utilisées pour manipuler une variable de type **Continent** :

| Nom de la propriété | Type manipulé | Effet |
| --- | --- | --- |
| Code | Constante de type Chaîne de caractères | Constante identifiant le continent :<br><br>- *continentAfrique* : Afrique (la constante vaut "AF")<br><br>- *continentAmériqueNord* : Amérique du Nord (la constante vaut "NA")<br><br>- *continentAmériqueSud* : Amérique du Sud (la constante vaut "SA")<br><br>- *continentAntarctique* : Antarctique (la constante vaut "AN")<br><br>- *continentAsie* : Asie (la constante vaut "AS")<br><br>- *continentEurope* : Europe (la constante vaut "EU")<br><br>- *continentIndéfini* : Le continent n'a pas pu être identifié. <br><br>- *continentOcéanie* :  Océanie (la constante vaut "OC")<br><br><br><br><br>**Cette propriété est disponible en lecture seulement**. |
| Nom | Chaîne de caractères | Nom du continent (dans la langue du framework). <br>**Cette propriété est disponible en lecture seulement**. |




<a name="NOTE0_2"></a>


### Fonctions WLangage utilisant les variables de type Continent

|   |   |
| --- | --- |
| [ContinentListe](../WDLang1/1000024932.md) | Renvoie la liste des continents. |
| [ContinentRécupère](../WDLang1/1000024933.md) | Renvoie la variable de type [Continent](../WDLang1/1000024919.md) correspondant à une des valeurs suivantes : <br><br>- une constante continent (code continent sur 2 caractères), <br><br>- un libellé de continent.<br><br><br> |
| [PaysListe](../WDLang1/1000024930.md) | Renvoie la liste de tous les pays ou des pays d'un continent donné selon la norme ISO 3166-1. |

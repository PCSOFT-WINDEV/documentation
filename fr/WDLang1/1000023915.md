
## DonneUUID (Fonction)

***En anglais : GetUUID***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Calcule un identifiant unique Universel (nommé "UUID") ayant pour taille 128 bits. 

Cet identifiant est unique dans le monde quelle que soit la machine sur laquelle il a été produit. 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonUUID est un UUID
MonUUID = DonneUUID()
// Exemple du UUID calculé : 95431e57-7d18-4fef-9d9d-96b231969149
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = DonneUUID()`
---

**`<Résultat> : (UUID)`**

Identifiant généré.
Le UUID renvoyé est formaté pour un affichage avec des séparateurs. Par exemple : "95431e57-7d18-4fef-9d9d-96b231969149".

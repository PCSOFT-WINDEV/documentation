
## CertificatExtrait (Fonction)

***En anglais : CertificateExtract***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Extrait un certificat d'un buffer de signature ou d'un duplicata d'impression.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Extraction du certificat depuis le buffer
MonCertificat est un Certificat 
MonCertificat = CertificatExtrait(bufSignature, certificatDepuisBuffer)

// Teste si la signature ne correspond pas 
// ou si le buffer est incorrect (détail dans ErreurInfo)
SI MonCertificat = Null ALORS
	Erreur("Erreur lors de la récupération du certificat." + RC + ErreurInfo())
	RETOUR
FIN

// Affiche le niveau de fiabilité du certificat
SELON MonCertificat.Fiabilité
	CAS certificatOk : Info("Certificat valide")
	CAS certificatInvalide : Info("Certificat invalide")
	CAS certificatNonFiable : Info("Racine de confiance du certificat non fiable")
	CAS certificatExpiré : Info("Dates de validité du certificat expiré")
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = CertificatExtrait(<Elément signé> [, <Option>])`
---

**`<Résultat> : (Variable de type Certificat)`**

Variable de type [Certificat](../WDLang1/1000019287.md) extrait de l'élément contenant la signature. 

**`<Elément signé> : (Variable de type Buffer ou chaîne de caractères)`**

Correspond : 

- soit à un buffer de signature. Ce buffer a été obtenu grâce aux fonctions : 

	- [CertificatSigneChaîne](../WDLang1/1000019296.md) ou [CertificatSigneFichier](../WDLang1/1000019297.md), 

	- [PDFRécupèreSignature](../WDLang6/1410088120.md).




- soit au chemin d'un duplicata d'impression signé. Le duplicata d'impression a été signé avec la fonction [iParamètreDuplicata](../WDLang5/1000018845.md). 




**`<Option> : (Constante optionnelle de type Entier)`**

Indique le type d'élément utilisé : 


|   |   |
| --- | --- |
| *certificatDepuisBuffer*<br>(Valeur par défaut) | L'élément est un buffer de signature. |
| *certificatDepuisDuplicata* | L'élément est le chemin d'un fichier duplicata signé. |

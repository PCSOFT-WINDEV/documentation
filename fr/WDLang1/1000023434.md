
## Fonctions de gestion du champ Editeur d'images et des variables WDPic (syntaxe préfixée)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions de gestion du champ Editeur d'images et des variables WDPic (via une syntaxe préfixée) sont les suivantes : 



|   |   |
| --- | --- |
| [&lt;Éditeur d'images&gt;.Crée](../WDLang1/1000023430.md) | Crée une image dans le champ Editeur d'images. |
| [&lt;Éditeur d'images&gt;.Ferme](../WDLang1/1000023431.md) | Ferme l'image se trouvant dans un champ Editeur d'images. |
| [&lt;Éditeur d'images&gt;.Ouvre](../WDLang1/1000023432.md) | Ouvre une image dans un champ Editeur d'images. |
| [&lt;Éditeur d'images&gt;.Sauve](../WDLang1/1000023433.md) | Sauvegarde l'image affichée dans le champ Editeur d'images. |
| [&lt;Variable WDPic&gt;.FusionneCalque](../WDLang1/1000024866.md) | Fusionne les calques d'une image au format WDPic. |
| [&lt;Variable WDPic&gt;.Redimensionne](../WDLang1/1000024920.md) | Redimensionne une image au format WDPic. |
| [&lt;Variable WDPic&gt;.Rotation](../WDLang1/1000024922.md) | Effectue une rotation d'une image au format WDPic. |
| [&lt;Variable WDPic&gt;.Sauve](../WDLang1/1000024923.md) | Sauve l'image présente dans une variable de type WDPic. |

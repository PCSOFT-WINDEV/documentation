
## Erreurs DDE
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000416"></a>
Lorsqu'une fonction DDE échoue (renvoie Faux par exemple), il est possible de connaître par programmation le numéro de l'erreur correspondante grâce à la fonction [ErreurInfo](../WDLang1/3013008.md). Il est ainsi possible de traiter l'erreur et de retenter l'opération.

Cette page présente les principales erreurs DDE pouvant survenir. Vous trouverez pour chaque erreur :

- son numéro,

- les fonctions DDE concernées par cette erreur,

- l'explication de l'erreur.

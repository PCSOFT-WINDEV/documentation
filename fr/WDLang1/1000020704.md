


## Fonctions de gestion des champs Tableau croisé dynamique
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Ces fonctions permettent de manipuler les champs de type Tableau croisé dynamique :



|   |   |
| --- | --- |
| [TCDAxeXY](../WDLang1/1000021113.md) | Permet de : <br><br>- Spécifier les entêtes de lignes et de colonnes affichés dans un champ Tableau croisé dynamique. <br><br>- Connaître les entêtes de lignes et de colonnes affichés dans un champ Tableau croisé dynamique.<br><br><br> |
| [TCDCalculeMiseAJour](../WDLang1/1000020534.md) | Met à jour une partie des données d'un champ Tableau croisé dynamique à partir d'une date. |
| [TCDCalculeTout](../WDLang1/1000020539.md) | Calcule ou recalcule entièrement un champ Tableau croisé dynamique. |
| [TCDCharge](../WDLang1/1000020507.md) | Charge le résultat du calcul d'un champ Tableau croisé dynamique depuis une sauvegarde (fichier présent sur le disque ou sauvegarde dans un buffer). |
| [TCDComparePériode](../WDLang1/1000024869.md) | Calcule et affiche une comparaison de périodes dans un champ Tableau croisé dynamique. |
| [TCDDéroule](../WDLang1/1000020699.md) | Déroule toutes les données d'un champ Tableau croisé dynamique selon un entête de ligne ou de colonne. |
| [TCDDérouleTout](../WDLang1/1000020623.md) | Déroule tous les entêtes de lignes et/ou de colonnes d'un champ Tableau croisé dynamique. |
| [TCDEnroule](../WDLang1/1000020698.md) | Enroule toutes données d'un champ Tableau croisé dynamique selon un entête de ligne ou de colonne. |
| [TCDEnrouleTout](../WDLang1/1000020622.md) | Enroule tous les entêtes de lignes et/ou de colonnes d'un champ Tableau croisé dynamique. |
| [TCDFiltre](../WDLang1/1000020531.md) | Active ou désactive un filtre sur un champ Tableau croisé dynamique. |
| [TCDInfoXY](../WDLang1/1000020735.md) | Renvoie les valeurs d'entêtes correspondant à une position en pixels dans un champ Tableau croisé dynamique. |
| [TCDListePositionEntête](../WDLang1/1000020975.md) | Renvoie toutes les valeurs associées à un entête de ligne ou de colonne d'un champ Tableau croisé dynamique. |
| [TCDSauve](../WDLang1/1000020471.md) | Sauve le résultat du calcul d'un champ Tableau croisé dynamique (dans un fichier ou en mémoire). |
| [TCDSelect](../WDLang1/1000021024.md) | Permet de connaître la position des cellules sélectionnées et de sélectionner des cellules dans un champ Tableau croisé dynamique. |
| [TCDSelectOccurrence](../WDLang1/1000022175.md) | Renvoie le nombre de cellules sélectionnées dans un champ Tableau croisé dynamique. |
| [TCDValeurEntête](../WDLang1/1000021135.md) | Renvoie la valeur affichée dans un entête du champ Tableau croisé dynamique. |
| [TCDVersExcel](../WDLang1/1000020703.md) | Crée un fichier Excel avec les données d'un champ Tableau croisé dynamique. |

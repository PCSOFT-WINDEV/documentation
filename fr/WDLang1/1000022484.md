
## docElément (Type de variable)

***En anglais : docElement***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **docElément** permet de manipuler chaque élément composant une variable de type [docParagraphe](../WDLang1/1000022483.md). 

Les caractéristiques de cet élément peuvent être définies et modifiées à l'aide de différentes fonctions et propriétés du WLangage. 

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ouvre un fichier docx dans le champ Traitement de texte
TT_MonDoc = DocOuvre(fRepExe() + fSep() + "MonDoc.docx")

// Fragment positionné à la fin du document
fFin est un docFragment(TT_MonDoc, -1)
// Sélectionne le document
FragmentTexte est un docFragment(TT_MonDoc, 1, fFin.PositionFin)

POUR TOUT MonElementCourant DE FragmentTexte.Elément
	MonParagraphe est un docParagraphe = MonElementCourant.Paragraphe
	Trace(MonParagraphe.PositionDébut)
	Trace(MonParagraphe.PositionFin)
FIN
```

## Remarques
<a name="NOTE0_1"></a>


### Propriétés spécifiques à la description des variables de type docElément
<a name="proprietes_specifiques_description_des_variables_type_docelement_ELTPARAGRAPHE000036"></a>Les propriétés suivantes peuvent être utilisées pour manipuler une variable de type **docElément** :

| **Nom de la propriété** | Type manipulé | Effet |
| --- | --- | --- |
| Cellule | Variable de type [docCellule](../WDLang1/1000022927.md) | Caractéristiques de la cellule (uniquement dans le cas où l'élément correspond à une cellule d'un tableau). |
| Formule | Variable de type [docElémentFormule](../WDLang1/1000022937.md) | Caractéristiques de la formule : formule, mise en forme de la formule, résultat de la formule (uniquement dans le cas d'un élément de type formule). |
| Image | Variable de type [docElémentImage](../WDLang1/1000022696.md) | Caractéristiques de l'image (utilisable uniquement dans le cas d'un élément de type Image). |
| Lien | Variable de type [docElémentLien](../WDLang1/1000022970.md) | Caractéristiques du lien (uniquement dans le cas où l'élément correspond à un lien). <br><br>Ces caractéristiques sont disponibles en lecture uniquement. |
| Paragraphe | Variable de type [docParagraphe](../WDLang1/1000022483.md) | Paragraphe qui contient l'élément. |
| Parent | Variable de type [docElément](../WDLang1/1000022484.md) | Elément qui contient l'élément.<br>**Cette propriété est disponible en lecture seulement**. |
| PositionDébut | Entier | Position de début de l'élément dans le document.<br>**Cette propriété est disponible en lecture seulement**. |
| PositionFin | Entier | Position de fin de l'élément dans le document.<br>**Cette propriété est disponible en lecture seulement**. |
| TableDesMatières | Variable de type [docTableDesMatières](../WDLang1/1000024893.md) | Caractéristiques de la table des matières du document. |
| Texte | Variable de type [docElémentTexte](../WDLang1/1000022493.md) | Caractéristiques du texte (utilisable uniquement dans le cas d'un élément de type Texte). |
| Type | Entier | Type de l'élément correspondant à une des constantes suivantes : <br><br>- *typeDocElémentCellule* : Elément de type Cellule d'un tableau.<br><br>- *typeDocElémentDocument* : Elément de type Document.<br><br>- *typeDocElémentEnTête* : Elément de type entête de page.<br><br>- *typeDocElémentFormule* : Elément de type Formule. <br><br>- *typeDocElémentImage* : Elément de type Image.<br><br>- *typeDocElémentInconnu* : Elément de type non reconnu. <br><br>- *typeDocElémentLien* : Elément de type Lien.<br><br>- *typeDocElémentParagraphe* : Elément de type Paragraphe avec aucun texte (texte vide).<br><br>- *typeDocElémentPiedDePage* : Elément de type pied de page.<br><br>- *typeDocElémentTableDesMatières* : Elément de type Table des matières.<br><br>- *typeDocElémentTexte* : Elément de type Texte.<br><br>- *typeDocElémentZoneDeTexte* : Elément de type Zone de texte. <br><br><br> |
| ZoneDeTexte | Variable de type [docElémentZoneDeTexte](../WDLang1/1000023420.md) | Caractéristiques de la zone de texte (utilisable uniquement dans le cas d'un élément de type Zone de texte). |

### Fonctions manipulant les variables de type docElément 
<a name="fonctions_manipulant_les_variables_type_docelement_ELTPARAGRAPHE000175"></a>


|   |   |
| --- | --- |
| [DocAjoute](../WDLang1/1000022919.md) | Ajoute : <br><br>- à la fin d'un document de type Traitement de texte : un élément, un document, un paragraphe, un texte, une image ou un fragment.<br><br>- à la fin d'un fragment : un élément, un texte ou une image.<br><br><br> |
| [DocChangeAltitude](../WDLang1/1000023970.md) | Modifie l'altitude d'un élément de type Image ou de type Zone de texte dans un document DocX. |
| [DocInsère](../WDLang1/1000022732.md) | Insère un objet dans un document de type Traitement de texte ou remplace le contenu du fragment en cours. Les objets manipulables sont : <br><br>- un élément,<br><br>- un document, <br><br>- un paragraphe, <br><br>- un texte, <br><br>- une image, <br><br>- un fragment.<br><br><br> |
| [DocSupprime](../WDLang1/1000024172.md) | Supprime un élément d'un document. Cet élément est supprimé du document auquel il est associé. |


## NationFramework (Fonction)

***En anglais : NationFramework***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Définit ou identifie la langue du framework.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Force le framework en anglais
NationFramework(nationAnglais)
```

```wl
// Trinité et Tobago en FR et en US
MonPays est un Pays = PaysRécupère("TT")
// Affichage du pays en FR
Trace(MonPays..Nom) // Trinité-et-Tobago
// Affichage du pays en US
// Changement de la langue du framework
NationFramework(nationAnglais)
Trace(MonPays..Nom) // Trinidad and Tobago
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître la langue actuelle du framework

`<Résultat> = NationFramework()`
---

**`<Résultat> : (Constante de type Entier)`**

Langue actuelle du framework :


|   |   |
| --- | --- |
| *nationAnglais* | Framework en langue anglaise. |
| *nationEspagnol* | Framework en langue espagnole. |
| *nationFrançais* | Framework en langue française. |




<a name="SYNTAXE2"></a>

### Modifier la langue du framework

`<Résultat> = NationFramework([<Nouvelle langue>])`
---

**`<Résultat> : (Constante de type Entier)`**

Langue actuelle du framework :


|   |   |
| --- | --- |
| *nationAnglais* | Framework en langue anglaise. |
| *nationEspagnol* | Framework en langue espagnole. |
| *nationFrançais* | Framework en langue française. |



**`<Nouvelle langue> : (Constante de type Entier)`**

Langue à utiliser pour le framework :


|   |   |
| --- | --- |
| *nationAnglais* | Framework en langue anglaise. |
| *nationEspagnol* | Framework en langue espagnole. |
| *nationFrançais* | Framework en langue française. |


Remarque : Si une autre langue est précisée, la langue réellement utilisée sera l'anglais ou le français selon la langue.

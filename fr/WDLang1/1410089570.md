


## &lt;Tableur&gt;.DéfusionneCellule (Fonction)

***En anglais : &lt;Spreadsheet&gt;.UnmergeCell***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime la fusion de cellule pour la cellule spécifiée ou pour la cellule sélectionnée dans un champ Tableur.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Dé-fusionne la cellule B2
TBLR_MonTableur.DéfusionneCellule("B2")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Champ Tableur>.DéfusionneCellule([<Cellule>])`
---

**`<Champ Tableur> : (Nom de champ)`**

Nom du champ Tableur à manipuler.

**`<Cellule> : (Chaîne de caractères optionnelle)`**

Nom d'une des cellules de la plage de cellules à dé-fusionner. Ce paramètre est au format &lt;Lettre de la colonne&gt;&lt;Numéro de la ligne&gt;. Par exemple : "A32".
Si ce paramètre n'est pas précisé, la cellule sélectionnée dans le champ Tableur sera dé-fusionnée.

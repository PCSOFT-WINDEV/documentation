
## PanneauEnumère (Fonction)

***En anglais : PanelEnumerate***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Enumère les champs Panneau dockable d'une fenêtre. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Détache tous les champs Panneau dockable
POUR TOUT p DE PanneauEnumère()
	PanneauDétache(p)
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = PanneauEnumère()`
---

**`<Résultat> : (Tableau)`**

Tableau de champs Panneau dockable.

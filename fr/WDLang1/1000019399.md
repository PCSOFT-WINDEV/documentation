
## DernierJourDuMois (Fonction)

***En anglais : LastDayOfMonth***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la date du dernier jour du mois spécifié. 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Renvoie le dernier jour du mois de Février 2017
MaDate est une Date
MaDate = DernierJourDuMois(2017, 2)
```
<a name="Exemple2"></a>

```wl
// Renvoie le dernier jour du mois en cours
MaDate est une Date
MaDate = DernierJourDuMois()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître le dernier jour du mois à partir d'une date

`<Résultat> = DernierJourDuMois([<Date>])`
---

**`<Résultat> : (Chaîne de caractères ou variable de type Date)`**

Date correspondant au dernier jour du mois demandé (au format AAAAMMJJ).

**`<Date> : (Chaîne de caractères, variable de type Date ou variable de type DateHeure optionnelle)`**

Date pour laquelle le dernier jour du mois doit être calculé. Cette date peut correspondre à : 

- une chaîne de caractères au format AAAAMMJJ (AAAA correspond à l'année, MM au mois et JJ au jour).  

- une chaîne de caractères au format AAAAMMJJHHmmSSCC (AAAA correspond à l'année, MM au mois, JJ au jour, HH aux heures, mm aux minutes, SS aux secondes et CC aux centièmes de seconde (utiliser CCC pour les millièmes de seconde)). Dans ce cas, seule la date est prise en compte. 

- une variable de type Date. 

- une variable de type DateHeure. Dans ce cas, seule la date est prise en compte.


Si ce paramètre n'est pas spécifié, la date du jour est utilisée par défaut.


<a name="SYNTAXE2"></a>

### Connaître le dernier jour du mois à partir d'une année et d'un mois

`<Résultat> = DernierJourDuMois(<Année> , <Mois>)`
---

**`<Résultat> : (Chaîne de caractères ou variable de type Date)`**

Date correspondant au dernier jour du mois demandé.

**`<Année> : (Entier)`**

Année pour laquelle le dernier jour du mois doit être calculé. 

**`<Mois> : (Entier)`**

Mois pour lequel le dernier jour doit être calculé.

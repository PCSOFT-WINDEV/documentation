
## Toast (Type de variable)

***En anglais : Toast***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **Toast** permet de définir toutes les caractéristiques avancées d'un toast à afficher. Les caractéristiques de ce toast peuvent être définies et modifiées à l'aide de différentes propriétés WLangage. 

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonToast est un Toast
MonToast.Titre = "Le saviez-vous ?"
MonToast.Texte = "Les FAA sont un truc génial."
MonToast.Bouton[1].Libellé = "En savoir plus..."
MonToast.Bouton[1].ActionClic = FoncAide
MonToast.ActionFermeture = clicFerme
ToastAffiche(MonToast)
PROCÉDURE INTERNE FoncAide
	Ouvre("FEN_Aide")
FIN
PROCÉDURE INTERNE clicFerme(nBouton)
	Trace("Clic fermeture, bouton " + nBouton)
FIN
```



```wl
// Affiche un toast avec le nom produit supprimé
MonToast est un Toast
MonToast.DuréeAffichage = toastLong
MonToast.CadrageVertical = cvBas 
MonToast.CadrageHorizontal = chCentre
ToastAfficheFI(FI_TOAST_SUPPR, MonToast, PRODUIT.NOM)

//------------------------------------
// Déclarations globales de la fenêtre interne
PROCÉDURE FI_TOAST_SUPPR(nomProduit)
LIB_TEXTE = ChaîneConstruit("Le produit %1 a été supprimé", nomProduit)
```

## Remarques
<a name="NOTE0_1"></a>


### Propriétés spécifiques à la description des variables de type Toast
<a name="proprietes_specifiques_description_des_variables_type_toast_ELTPARAGRAPHE000058"></a>Les propriétés suivantes peuvent être utilisées pour manipuler un toast :

   | --- |
| --- | --- | --- |
| **Nom de la propriété** | Type manipulé | Effet |
| ActionFermeture |   |   | Action à réaliser lors de la fermeture du toast. Cette propriété correspond au nom de la procédure à lancer lors du clic sur le bouton affiché dans le toast. <br>Cette procédure prend en paramètre optionnel le numéro du bouton cliqué. Si aucun bouton spécifique n'a été cliqué pour fermer le toast (fermeture par timer, clic sur le toast, fermeture de la fenêtre parent, etc.), ce paramètre vaut 0. |
| Bouton[n] |   |   | Boutons affichés dans le toast. <br><br> |
|   | Bouton[n].ActionClic |   | Action à réaliser lors du clic sur le bouton. <br>Cette propriété correspond au nom de la procédure à lancer lors du clic sur le bouton affiché dans le toast. |
|   | Bouton[n].Libellé |   | Libellé du bouton. <br><br>Remarque : il est possible d'utiliser la fonction [gStylo](../WDLang1/3029039.md) pour modifier la couleur du texte affiché dans le toast. Par exemple : <br><br><pre><code>MonToast.Bouton[3].Libellé = gStylo(JauneClair) + <br>	"Ne plus demander"</code></pre><br> |
| CadrageHorizontal |   | Constante de type Entier | Cadrage horizontal du message, relatif à la fenêtre : <br><br>- *chCentre* (Valeur par défaut) : Affichage du message Toast au centre.<br><br>- *chDroite* : Affichage du message Toast sur la droite.<br><br>- *chGauche* : Affichage du message Toast sur la gauche.<br><br><br><br><br> |
| CadrageVertical |   | Constante de type Entier | Cadrage vertical du message, relatif à la fenêtre : <br><br>- *cvBas* (Valeur par défaut) : Affichage du message Toast en bas.<br><br>- *cvHaut* : Affichage du message Toast en haut.<br><br>- *cvMilieu* : Affichage du message Toast au milieu.<br><br><br> |
| CouleurFond |   | Entier | Couleur de fond utilisée pour l'affichage du message. Ce paramètre peut correspondre à : <br><br>- une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)),<br><br>- une couleur TSL (obtenue avec la fonction [TSL](../WDLang1/3029057.md)),<br><br>- [une couleur prédéfinie du WLangage](../WDLang5/3010002.md).<br><br><br> |
| DuréeAffichage |   | Constante de type Entier | Durée d'affichage du message toast : <br><br>- *toastCourt* (Valeur par défaut) : Le message Toast sera affiché pendant une faible période de temps.<br><br>- *toastLong* : Le message Toast sera affiché pendant une longue période de temps.<br><br><br><br><br>**Remarque** : La durée d'affichage exacte dépend de l'appareil utilisé. |
| FermeAuClic |   | Booléen | Gestion de la fermeture du toast : <br><br>- <u><u><u><u>Vrai</u></u></u></u> : le toast se ferme automatiquement si l'utilisateur clique dessus, <br><br>- <u><u><u><u>Faux</u></u></u></u> : le toast se ferme après la durée d'affichage spécifiée. <br><br><br><br><br> |
| Opacité |   | Entier | Niveau d'opacité du toast dont la valeur est comprise entre 1 et 100 : <br><br>- 1, toast sans aucune opacité, <br><br>- 100, pour obtenir un toast complètement opaque. <br><br><br><br><br>Par défaut, le niveau d'opacité est 0 (ce qui correspond à une opacité à 100). |
| Superposable |   | Booléen | <br><br>- <u><u><u><u>Vrai</u></u></u></u> (valeur par défaut) : si plusieurs toasts sont affichés, les toasts s'empilent. <br><br>- <u><u><u><u>Faux</u></u></u></u> : si plusieurs toasts sont affichés, les toasts sont affichés les uns sous les autres (compatibilité version 24). <br><br><br><br><br> |
| Texte |   | Chaîne de caractères | Message à afficher. |
| Titre |   | Chaîne de caractères | Titre du toast. <br><br> |

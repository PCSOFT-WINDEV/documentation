


## &lt;Éditeur d'images&gt;.Sauve (Fonction)

***En anglais : &lt;Image Editor&gt;.Save***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Sauve l'image présente dans un champ Editeur d'images.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Demande de sauvegarde de l'image présente dans un champ Editeur d'images
SI EI_MonEditeur.Sauve() = Faux ALORS
	Erreur(ErreurInfo())
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Champ Editeur d'images>.Sauve([<Fichier>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'image a été créée,

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire (erreur, annulation du sélecteur de fichier, ...). En cas d'erreur, la variable [ErreurDétectée](../WDLang1/3087001.md) est à <u><u><u><u>Vrai</u></u></u></u> et la fonction [ErreurInfo](../WDLang1/3013008.md) permet de connaître le détail de l'erreur.




**`<Champ Editeur d'images> : (Nom de champ)`**

Nom du champ Editeur d'images à manipuler. 

**`<Fichier> : (Chaîne de caractères optionnelle)`**

Nom et chemin du fichier image à sauver. L'extension du fichier détermine le type de l'image à sauver.
Si ce paramètre n'est pas précisé, l'action effectuée correspond à la fonctionnalité d'enregistrement de l'éditeur d'images (clic sur la disquette dans le ruban).
Si l'image n'a jamais été enregistrée, le sélecteur de fichier s'affiche.

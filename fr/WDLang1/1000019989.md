


## &lt;Multimédia&gt;.Durée (Fonction)

***En anglais : &lt;Multimedia&gt;.Duration***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la taille du fichier multimédia en cours de lecture (en millisecondes) dans le champ Multimédia.  








<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) 
```wl
MonFichier est une chaîne = "C:\Ma musique\mysong.mp3"
MM_Multimédia1.Ouvre(MonFichier)
Info("La durée de ce fichier est " + MM_Multimédia1.Durée() + " millisecondes")
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Champ Multimédia>.Durée()`
---

**`<Résultat> : (Entier)`**



- Durée (en millisecondes) du fichier multimédia, 

- 0 en cas d'erreur. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Champ Multimédia> : (Nom de champ)`**

Nom du champ Multimédia à manipuler.

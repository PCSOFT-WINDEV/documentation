
## FenChangeAlias (Fonction)

***En anglais : WinChangeAlias***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie l'alias d'une fenêtre.

**Rappel** : Un alias permet de différencier la même fenêtre ouverte plusieurs fois (cas fréquent dans une application MDI).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ouvre une nouvelle fenêtre MDI.
// WINDEV choisit un alias.
sNomAlias = MDIOuvre(FEN_SaisieClient, "Nouveau client", Valeur1, Valeur2)
// Utilisation d'un alias personnalisé
FenChangeAlias(sNomAlias, "TEMP0001")
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = FenChangeAlias(<Ancien alias> , <Nouvel alias>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le changement de nom a été effectué, 

- <u><u><u><u>Faux</u></u></u></u> si l'opération n'a pas pu être réalisée (une fenêtre avec le même alias existe déjà).




**`<Ancien alias> : (Chaîne de caractères)`**

Alias qui doit être modifié.

**`<Nouvel alias> : (Chaîne de caractères)`**

Nouvel alias de la fenêtre.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Cette fonction est équivalente à la propriété [Alias](../Proprietes/2510004.md).


```txt
{<Ancien Alias>}..Alias = <Nouveau Alias>
```

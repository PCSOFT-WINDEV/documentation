
## diagSélection (Type de variable)

***En anglais : diagSelection***
			
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **diagSélection** permet de définir toutes les caractéristiques avancées d'une sélection réalisée dans un diagramme. Les caractéristiques de cette sélection peuvent être définies et modifiées à l'aide de différentes propriétés WLangage.

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Obtenir la sélection dans un champ Editeur de diagrammes
MaSélection est un diagSélection <- EDIAG_MonDiagramme.Sélection
SI MaSélection.Forme.Occurrence > 0 ALORS
	// Obtenir les formes
	POUR TOUT stForme DE MaSélection.Forme	
		LIB_INFO_SELECTION = "Sélection de " + stForme.Nom + 
		" [ " + stForme.X + ", " + stForme.Y + " - " + stForme.Largeur + 
		"x" + stForme.Hauteur + " ]"	
	FIN
SINON
	LIB_INFO_SELECTION = "Cliquez sur une forme dans le diagramme pour connaître la sélection"
FIN
```





<a name="PROPS"></a>

## Propriétés


### Propriétés spécifiques à la description des variables de type diagSélection
<a name="proprietes_specifiques_description_des_variables_type_diagselection_ELTPARAGRAPHE000028"></a>Les propriétés suivantes peuvent être utilisées pour manipuler une sélection :

| Nom de la propriété | Type manipulé | Effet |
| --- | --- | --- |
| Altitude | Entier | Numéro d'ordre correspondant à l'altitude de la sélection. |
| Angle | Réel | Angle de rotation de la sélection. |
| Arrondi | Réel | <br><br>- Courbure de l'arrondi de la forme sélectionnée, <br><br>- -1 si la forme n'a pas d'arrondi ou si plusieurs formes sont sélectionnées. <br><br><br> |
| Bordure | [diagBordure](../WDLang1/1410088179.md) | Caractéristiques de la bordure de la forme sélectionnée. |
| Commentaire | Chaîne de caractères | Commentaire associé à la sélection. |
| Fond | <br><br> | <br><br>- Couleur de fond de la forme sélectionnée, <br><br>- -1 si aucune forme n'est sélectionnée ou si plusieurs formes sont sélectionnées.  <br><br><br> |
| Forme | Tableau de [diagForme](../WDLang1/1410088082.md) | Formes présentes dans la sélection. |
| Hauteur | Réel | <br><br>- Hauteur de la sélection, exprimée en pixels, <br><br>- -1 si aucune sélection. <br><br><br> |
| Largeur | Réel | <br><br>- Largeur de la sélection, exprimée en pixels, <br><br>- -1 si aucune sélection. <br><br><br> |
| Lien | [diagLien](../WDLang1/1410088454.md) | Caractéristiques du lien sélectionné. |
| Ombre | Booléen | <br><br>- <u><u><u>Vrai</u></u></u> si la forme possède une ombre, <br><br>- <u><u><u>Faux</u></u></u> dans le cas contraire.<br><br><br> |
| Opacité | Entier | <br><br>- Pourcentage d'opacité de la forme sélectionnée. Ce pourcentage varie de 0 (invisible) à 100 (complètement opaque).<br><br>- -1 si aucune forme n'est sélectionnée.<br><br><br> |
| Texte | [diagTexte](../WDLang1/1410088393.md) | Caractéristiques du texte de la forme sélectionnée. |
| Type | Constante de type Entier | Type de l'élément sélectionné |
| X | Réel | <br><br>- Position en X du rectangle englobant la sélection (exprimée en pixels),<br><br>- -1 si aucune sélection.<br><br><br> |
| Y | Réel | <br><br>- Position en Y du rectangle englobant la sélection (exprimée en pixels),<br><br>- -1 si aucune sélection.<br><br><br> |

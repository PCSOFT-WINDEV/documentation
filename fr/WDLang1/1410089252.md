
## dEfface (Fonction)

***En anglais : dClear***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Efface le contenu d'une variable de type Image ou d'un champ Image avec la couleur spécifiée.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
dDébutDessin(IMG_Dessin)

// Dessin d'un rectangle dont le fond est jaune clair
// et le bord bleu clair
dRectangle(10, 10, 50, 50, JauneClair, BleuClair)

// Efface le dessin avec la couleur Bleue
dEfface(BleuClair)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Effacer avec la couleur spécifiée dans l'image en cours

`dEfface([<Couleur>])`
---

**`<Couleur> : (Constante ou Variable de type Couleur optionnelle)`**

Couleur utilisée pour effacer. Cette couleur peut correspondre à : 

- une des constantes WLangage de couleur, 

- une variable de type [Couleur](../WDLang1/1000019661.md). 

- une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)),

- une couleur TSL (obtenue avec la fonction [TSL](../WDLang1/3029057.md)),

- une couleur prédéfinie (voir [les couleurs prédéfinies du WLangage](../WDLang5/3010002.md)).


Si ce paramètre n'est pas précisé, l'état initial de l'image sera appliqué : 

- transparent pour une variable de type [Image](../WDLang1/1000019650.md), 

- couleur du fond de la fenêtre pour un champ Image. 





<a name="SYNTAXE2"></a>

### Effacer avec la couleur spécifiée dans le champ Image ou dans la variable de type Image

`dEfface(<Image> [, <Couleur>])`
---

**`<Image> : (Nom de champ ou variable de type Image)`**

Nom du champ Image ou de la variable de type [Image](../WDLang1/1000019650.md) à manipuler. 

**`<Couleur> : (Constante ou Variable de type Couleur optionnelle)`**

Couleur utilisée pour effacer. Cette couleur peut correspondre à : 

- une des constantes WLangage de couleur, 

- une variable de type [Couleur](../WDLang1/1000019661.md). 

- une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)),

- une couleur TSL (obtenue avec la fonction [TSL](../WDLang1/3029057.md)),

- une couleur prédéfinie (voir [les couleurs prédéfinies du WLangage](../WDLang5/3010002.md)).


Si ce paramètre n'est pas précisé, l'état initial de l'image sera appliqué : 

- transparent pour une variable de type [Image](../WDLang1/1000019650.md), 

- couleur du fond de la fenêtre pour un champ Image.

## Remarques


### Gestion de l'opacité
<a name="gestion_opacite_ELTPARAGRAPHE000095"></a>

- Si le dessin a été activé dans le champ sans l'option d'opacité (fonction dDébutDessin sans la constante *dAvecOpacité*) : 

	- ![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png) Sous Windows, la fonction **dEfface** utilisée avec une couleur ne prendra pas en compte l'opacité. 

	- ![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) Sous Linux, seules les couleurs dont l'opacité est maximum seront utilisables.




- Si le dessin a été activé dans le champ avec l'option d'opacité (fonction dDébutDessin avec la constante *dAvecOpacité*), la fonction **dEfface** utilisée avec une couleur prendra en compte l'opacité.

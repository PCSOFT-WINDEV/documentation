
## &lt;Kanban&gt;.AjouteCarte (Fonction)

***En anglais : &lt;Kanban&gt;.AddCard***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ajoute une carte dans une liste d'un champ Kanban.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ajoute une carte dans la liste de Kanban LISTE_Matière_à_Revoir
// Cette liste de kanban a été définie sous l'éditeur 
maCarte est un kbCarte
maCarte.Titre = "Anglais"
KANBAN_Révision.AjouteCarte(LISTE_Matière_à_Revoir, maCarte)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Champ Kanban>.AjouteCarte(<Liste de Kanban> , <Carte>)`
---

**`<Résultat> : (Entier)`**

Indice de l'élément ajouté.

**`<Champ Kanban> : (Nom de champ)`**

Nom du champ Kanban manipulé.

**`<Liste de Kanban> : (Nom de champ)`**

Nom du champ Liste de Kanban dans lequel la carte doit être ajoutée. Ce nom peut correspondre : 

- au nom du champ Liste de Kanban défini sous l'éditeur, 

- au nom du champ Liste de Kanban défini par programmation.




**`<Carte> : (Variable de type kbCarte)`**

Nom de la variable de type [kbCarte](../WDLang1/1410089167.md) correspondant à la carte à ajouter. La carte est ajoutée à la fin de la liste de Kanban spécifiée.

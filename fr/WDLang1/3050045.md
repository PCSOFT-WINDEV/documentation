
## MatDéterminant (Fonction)

***En anglais : MatDeterminant***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Calcule le déterminant d'une matrice carrée (nombre de lignes égal au nombre de colonnes).




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple
<a class="notetitle" target="_blank" href="$DOC$=1000003050045&name=matdeterminant_fonction&product=WD">Voir des exemples supplémentaires</a>

```wl
// Calcul du déterminant d'une matrice
ResDéterminant = MatDéterminant("MaMatrice")
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = MatDéterminant(<Matrice>)`
---

**`<Résultat> : (Réel)`**

Déterminant de la matrice. Si cette valeur est égale à 0, le déterminant de la matrice est nul.

**`<Matrice> : (Chaîne de caractères)`**

Nom de la matrice à manipuler, défini avec la fonction [MatCrée](../WDLang1/3050058.md).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Existence de la matrice
<a name="existence_matrice_ELTPARAGRAPHE000078"></a>L'existence de la matrice n'est pas vérifiée. Pour vérifier l'existence d'une matrice, utilisez la fonction [MatExiste](../WDLang1/3050076.md).


<a name="NOTE0_2"></a>




### Gestion des erreurs
<a name="gestion_des_erreurs_ELTPARAGRAPHE000090"></a>**Attention** : La fonction **MatDéterminant** ne retourne pas de code d'erreur. Pour savoir si des erreurs ont été générées par le calcul du déterminant, utilisez la fonction [MatErreur](../WDLang1/3050044.md). Le libellé de l'erreur peut être connu grâce à la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.

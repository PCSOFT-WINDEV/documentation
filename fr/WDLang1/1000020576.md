
## fSurveilleStopTout (Fonction)

***En anglais : fTrackStopAll***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Arrête toutes les surveillances en cours sur les fichiers et les répertoires. Ces surveillances ont pu être lancées par la fonction [fSurveilleFichier](../WDLang1/1000020522.md) ou par la fonction [fSurveilleRépertoire](../WDLang1/1000020523.md). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Arrêt de toutes les surveillances en cours
SI PAS fSurveilleStopTout() ALORS
	Erreur("Aucune surveillance n'a été lancée.")
SINON
	Info("La suppression des surveillances a réussi.")
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = fSurveilleStopTout()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si toutes les surveillances précédemment activées ont été arrêtées, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire (surveillance inexistante).




## &lt;TreeMap&gt;.SelectGroupe (Fonction)

***En anglais : &lt;TreeMap&gt;.SelectGroup***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Sélectionne visuellement un groupe d'éléments dans un champ TreeMap. Un groupe d'éléments est identifié par son chemin. Le chemin a été spécifié lors de l'ajout de l'élément dans le champ TreeMap grâce à la fonction [&lt;TreeMap&gt;.Ajoute](../WDLang1/1000019740.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Le champ TMAP_Vin est rempli à partir d'un champ Table. 
// Sélection du groupe correspondant au type de vin sélectionné dans le champ Table
RépertoireSélection est une chaîne
RépertoireSélection = COL_Typevin[TABLE_VIN]
TMAP_Vin.SelectGroupe(RépertoireSélection)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Champ TreeMap>.SelectGroupe(<Chemin du groupe>)`
---

**`<Champ TreeMap> : (Nom de champ)`**

Nom du champ TreeMap à manipuler.

**`<Chemin du groupe> : (Chaîne de caractères)`**

Chemin du groupe à sélectionner. Le séparateur des différents éléments du chemin peut être :

- soit le caractère "\\". Par exemple : "MonRépertoire\\MonFichier". 

- soit le caractère "TAB". Par exemple : "Desserts"+TAB+"Crème brulée". 




Ce chemin a été spécifié lors de l'ajout de l'élément dans le champ TreeMap avec la fonction [&lt;TreeMap&gt;.Ajoute](../WDLang1/1000019740.md).

Si ce paramètre correspond à une chaîne vide (""), aucun groupe n'est sélectionné.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Si un groupe était déjà sélectionné avant l'utilisation de la fonction **&lt;TreeMap&gt;.SelectGroupe**, ce groupe est automatiquement désélectionné. 

- Le style du groupe sélectionné est défini dans l'onglet "Style" de la fenêtre de description du champ TreeMap.




## &lt;Variable picCalque&gt;.FinDessin (Fonction)

***En anglais : &lt;picLayer variable&gt;.EndDrawing***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Libère les ressources associées à la variable de type [picCalque](../WDLang1/1000024604.md).
**Remarque** : L'appel de la fonction **&lt;Variable picCalque&gt;.FinDessin** est optionnel. Les ressources sont automatiquement libérées au moment de la fermeture de la fenêtre ou de la page.



![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Les dessins](../WDLang1/3029035.md). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonWDPic est un WDPic = "Test.bmp"
MonCalque est un picCalque
...
// Destruction de l'image présente dans le calque
MonCalque.FinDessin()
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Image picCalque>.FinDessin()`
---

**`<Image picCalque> : (Variable de type picCalque)`**

Nom de la variable de type [picCalque](../WDLang1/1000024604.md) à manipuler.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Utilisation dans le cadre de la syntaxe préfixée
<a name="utilisation_dans_cadre_syntaxe_prefixee_ELTPARAGRAPHE000479"></a>La fonction [&lt;Variable picCalque&gt;.DébutDessin](../WDLang1/1000024470.md) doit être appelée avant l'utilisation des fonctions de dessin. La fonction [&lt;Variable picCalque&gt;.DébutDessin](../WDLang1/1000024470.md) permet de définir les options de dessin utilisées sur le type [picCalque](../WDLang1/1000024604.md).

La fonction **&lt;Variable picCalque&gt;.FinDessin** ne doit pas être appelée dans le même traitement que la fonction [&lt;Variable picCalque&gt;.DébutDessin](../WDLang1/1000024470.md).
Si la fonction **&lt;Variable picCalque&gt;.FinDessin** est appelée dans le même traitement que la fonction [&lt;Variable picCalque&gt;.DébutDessin](../WDLang1/1000024470.md), le dessin n'apparaîtra pas : il sera automatiquement effacé.

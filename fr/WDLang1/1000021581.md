
## PiloteVoletDétache (Fonction)

***En anglais : EmulatePaneUndock***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Pilote le détachement d'un volet d'un champ Onglet dynamique.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Détache le volet "Alias_1" du champ Onglet ONG_Onglet
PiloteVoletDétache(ONG_Onglet, "Alias_1", 105, 3)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`PiloteVoletDétache(<Nom du champ> , <Alias> , <X> , <Y>)`
---

**`<Nom du champ> : (Chaîne de caractères)`**

Nom du champ Onglet dynamique duquel le volet doit être détaché. 

**`<Alias> : (Chaîne de caractères)`**

Alias du volet à détacher dans le champ Onglet dynamique associé. 

**`<X> : (Entier)`**

Abscisse à laquelle le volet doit être positionné. 

**`<Y> : (Chaîne de caractères)`**

Ordonnée à laquelle le volet doit être positionné.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Cette fonction doit être utilisée uniquement dans le code des tests automatiques.

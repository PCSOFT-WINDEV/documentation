


## docLigne (Type de variable)

***En anglais : docRow***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **docLigne** permet de définir toutes les caractéristiques avancées d'une ligne d'un tableau présent dans un document de type Traitement de texte. Les caractéristiques de cette ligne de tableau peuvent être définies et modifiées à l'aide de différentes propriétés WLangage. 

**ATTENTION : Ce type ne doit pas être utilisé directement.** Il doit être utilisé uniquement via les variables de type [docParagraphe](../WDLang1/1000022483.md) correspondant à un tableau. 


<a name="NOTE0"></a>

## Remarques
<a name="NOTE0_1"></a>


### Propriétés spécifiques à la description des variables de type docLigne
<a name="proprietes_specifiques_description_des_variables_type_docligne_ELTPARAGRAPHE000028"></a>Les propriétés suivantes peuvent être utilisées pour manipuler une variable de type **docLigne** :

| Nom de la propriété | Type manipulé | Effet |
| --- | --- | --- |
| Hauteur | Entier | Hauteur de la ligne (en millimètres). |
| ModeHauteur | Constante de type Entier | Mode de calcul de la hauteur de la ligne : <br><br>- *docModeHauteurAuMoins* : La hauteur de la ligne sera au moins celle définie par la propriété **Hauteur**. <br><br>- *docModeHauteurAutomatique* : La hauteur de la ligne est calculée automatiquement en fonction du texte présent dans les cellules de la ligne. <br><br>- *docModeHauteurExact* : La hauteur de la ligne sera celle définie par la propriété **Hauteur**. <br><br>- *docModeHauteurNonDéfini* : Le mode de calcul de la hauteur de la ligne n'est pas défini (par défaut, le mode de calcul automatique est effectué). <br><br><br> |
| NombreCellules | Entier | Nombre de cellules de la ligne. <br><br>Cette propriété est disponible en lecture seulement. |

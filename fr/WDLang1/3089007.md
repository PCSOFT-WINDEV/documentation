
## PlanificateurModifieHoraire (Fonction)

***En anglais : SchedulerModifyTrigger***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie les paramètres d'un horaire de tâche planifiée, c'est-à-dire les paramètres qui vont définir l'exécution d'une tâche. Les paramètres utilisés pour la modification de l'horaire de la tâche sont ceux saisis dans la structure [HoraireTâchePlanifiée](../WDLang1/3089002.md).

**Remarque** : Pour remplir la structure HoraireTâchePlanifiée avec les paramètres d'un horaire d'une tâche, utilisez la fonction [PlanificateurPropriétésHoraire](../WDLang1/3089010.md).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// La tâche est quotidienne, exécutée tous les jours à 10H à partir d'aujourd'hui
HoraireTâchePlanifiée.Type = htQuotidien
HoraireTâchePlanifiée.Heure = 1000
HoraireTâchePlanifiée.DateDébut = DateDuJour()
NumHoraire = PlanificateurAjouteHoraire("MaTache")

...

// Modification de l'heure d'exécution
HoraireTâchePlanifiée.Heure = 1030
PlanificateurModifieHoraire("MaTache", NumHoraire)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = PlanificateurModifieHoraire(<Nom de la tâche> , <Identifiant de l'horaire>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> en cas de succès, 

- <u><u><u><u>Faux</u></u></u></u> en cas d'erreur.




**`<Nom de la tâche> : (Chaîne de caractères)`**

Nom de la tâche à manipuler.

**`<Identifiant de l'horaire> : (Entier)`**

Identifiant de l'horaire à manipuler. Cet identifiant est renvoyé lors de la création de l'horaire avec la fonction [PlanificateurAjouteHoraire](../WDLang1/3089001.md).


## FinTimerSys (Fonction)

***En anglais : EndTimerSys***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Termine l'exécution d'un timer déclenché par la fonction [TimerSys](../WDLang1/3015006.md).






<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Procédure permettant l'affichage régulier de l'heure dans SAI_HEURE1
PROCEDURE Affiche_Heure()
SAI_HEURE1 = HeureSys()

// Traitement à l'ouverture de la fenêtre
// Affiche_Heure sera automatiquement appelée chaque seconde
NumTimer est un entier
NumTimer = TimerSys("Affiche_Heure", 100, 1)
SI NumTimer = 0 ALORS
	Erreur("Création du timer impossible")
FIN
```



```wl
// Arrêt du timer lors du clic sur un bouton "BTN_ArrêtTimer"
FinTimerSys(NumTimer)
```







<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`FinTimerSys([<Numéro du timer>])`
---

**`<Numéro du timer> : (Entier optionnel)`**

Numéro du timer à détruire (renvoyé par la fonction [TimerSys](../WDLang1/3015006.md)).
Si ce numéro est précisé, le timer sélectionné est interrompu. Si le timer n'existe pas ou s'il a déjà été interrompu, la fonction n'a aucun effet.
Si ce numéro n'est pas précisé et si la fonction **FinTimerSys** est utilisée dans le code d'une procédure appelée par timer, le timer est interrompu. Sinon, la fonction **FinTimerSys** n'a aucun effet.


## MCIExécute (Fonction)

***En anglais : MCIExecute***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Exécute une commande MCI (Media Control Interface).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ouverture du fichier "C:\MesFichiers\Fichier.AVI"
// dans le champ Image "IMG_ImageAVI"
ResExécuteMCI = MCIExécute("Open C:\MesFichiers\Fichier.AVI" + ...
	"Alias AVI Style Child Parent" + Handle(IMG_ImageAVI))
ResExécuteMCI2 = MCIExécute("Play AVI Notify")
```


<a name="Exemple2"></a>



```wl
// Lancer une vidéo "C:\MesFichiers\Fichier.AVI" en mode répétition
// Ouverture de la vidéo
MCIOuvre("C:\MesFichiers\Fichier.AVI", "AVI", MciStyleFille, IMG_Vidéo, MciModeNormal)
// Exécution de la vidéo
MCIExécute("Play AVI repeat")
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = MCIExécute(<Commande à exécuter> [, <Nom de la fenêtre>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la commande MCI a été exécutée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour connaître :

	- le numéro de l'erreur, utilisez la fonction [MCIErr](../WDLang1/3051011.md).

	- le libellé de l'erreur, utilisez la fonction [MCIMsgErr](../WDLang1/3051021.md) ou la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.







**`<Commande à exécuter> : (Chaîne de caractères)`**

Commande MCI à exécuter.

Les différentes commandes MCI sont accessibles en standard avec Windows grâce à la DLL MMSYSTEM. Cette librairie, fournie par Microsoft, contient toutes les fonctions nécessaires à l'exploitation des périphériques multimédias. Pour plus de détails sur les commandes MCI, consultez la documentation Microsoft.
Il est possible d'ajouter la constante *MciAttente* à cette chaîne. Cette constante permet d'indiquer que le programme doit attendre la fin de la commande MCI pour exécuter la suite de l'application. 

**`<Nom de la fenêtre> : (Chaîne de caractères optionnelle)`**

Nom de la fenêtre permettant d'intercepter et de gérer les différents messages envoyés par MMSYSTEM.DLL. Ces messages sont du type MM_xxxxxx et sont récupérables grâce à la fonction [Evénement](../WDLang1/3015011.md). Il est par exemple possible d'obtenir des messages concernant la gestion du lecteur de CD ROM ou du Joystick.

<a name="NOTE0_1"></a>

## Remarques
Pour connaître le résultat de la commande MCI, utilisez la fonction [MCIRécupère](../WDLang1/3051022.md).

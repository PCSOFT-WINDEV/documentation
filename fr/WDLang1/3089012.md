
## PlanificateurRAZ (Fonction)

***En anglais : SchedulerReset***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Remplit les structures [TâchePlanifiée](../WDLang1/3089004.md) et/ou [HoraireTâchePlanifiée](../WDLang1/3089002.md) avec les valeurs par défaut.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Initialisation des structures TâchePlanifiée et HoraireTâchePlanifiée
PlanificateurRAZ()
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`PlanificateurRAZ([<Structure à ré-initialiser>])`
---

**`<Structure à ré-initialiser> : (Constante ou combinaison de constantes optionnelle)`**

Type de la structure à ré-initialiser.


|   |   |
| --- | --- |
| *planificateurRAZHoraire* | Ré-initialise la structure HoraireTâchePlanifiée. |
| *planificateurRAZTâche* | Ré-initialise la structure TâchePlanifiée. |
| *planificateurRAZTâche* + *planificateurRAZHoraire*<br>(par défaut) | Ré-initialise les deux structures. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Ré-initialisation des structures
<a name="reinitialisation_des_structures_ELTPARAGRAPHE000061"></a>Les structures sont ré-initialisées de manière à créer une tâche par défaut (identique à celle crée par la commande "Nouvelle tâche" du planificateur Windows) : une tâche exécutée tous les jours à 9h à partir du jour de sa création, ne pouvant pas s'exécuter lorsque l'ordinateur fonctionne sur batteries, et limitée à 72 heures d'exécution.


## ArcTang2 (Fonction)

***En anglais : ArcTan2***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie l'arc tangente 2 des valeurs passées en paramètre.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
ArcTang2(1, 1) 	// Renvoie 45
ArcTang2(1, -1) 	// Renvoie 135
ArcTang2(-1, 1) 	// Renvoie -45
ArcTang2(-1, -1) 	// Renvoie -135
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = ArcTang2(<Y> , <X>)`
---

**`<Résultat> : (Réel)`**



- Angle (en degrés) entre : 

	- la partie positive de l'axe des X d'un plan,

	- et le point de ce plan aux coordonnées (X,Y). 
			Cet angle est positif pour les angles dans le sens anti-horaire dit sens trigonométrique (moitié haute du plan, Y &gt; 0) et négatif dans l'autre (moitié basse du plan, Y &lt; 0).
			Cet angle est compris dans l'intervalle [-180,180].




- 0 si les paramètres &lt;X&gt; et &lt;Y&gt; sont égaux à 0. 




**`<Y> : (Réel)`**

Valeur Y. 

**`<X> : (Réel)`**

Valeur X.

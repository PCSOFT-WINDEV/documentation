
## SysNbStockageExterne (Fonction)

***En anglais : SysNbExternalStorage***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le nombre d'espaces de stockage externe présents sur un appareil Android.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Récupère le chemin du répertoire à utiliser pour le stockage des fichiers de l'application 
// sur l'espace de stockage externe secondaire de l'application
nNB est un entier = SysNbStockageExterne()
SI nNb > 1 ALORS
	RENVOYER SysRepStockageExterne(2, sseAppFichier)
SINON
	RENVOYER ""
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = SysNbStockageExterne()`
---

**`<Résultat> : (Entier)`**

Nombre d'espaces de stockage externe présents sur l'appareil.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Un même appareil peut contenir plusieurs zones de stockage externe (par exemple, une zone correspond à la partition "externe" de la mémoire permanente et une autre zone correspond à la SDCard).

- La fonction [SysStockageExterneAmovible](../WDLang1/1000021818.md) permet de savoir si un espace de stockage correspond à de la mémoire amovible (SDCard) ou non.
	Avant d'accéder à un répertoire situé dans la zone de stockage externe de l'appareil, il est conseillé de tester sa disponibilité avec la fonction [SysEtatCarteStockage](../WDLang1/1000019554.md).

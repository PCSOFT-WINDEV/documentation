
## Fonctions de gestion des champs Planning (syntaxe préfixée)
			

Les fonctions de gestion des champs Planning (syntaxe préfixée) sont les suivantes :



|   |   |
| --- | --- |
| [&lt;Planning&gt;.Affiche](../WDLang1/1000022900.md) | Réinitialise le contenu d'un champ Planning à partir de sa source de données (fichier HFSQL par exemple). |
| [&lt;Planning&gt;.AffichePériodePrécédente](../WDLang1/1000022901.md) | Déplace la zone temporelle affichée sur la zone précédente (par exemple, semaines précédentes) dans un champ Planning. |
| [&lt;Planning&gt;.AffichePériodeSuivante](../WDLang1/1000022902.md) | Déplace la zone temporelle affichée sur la zone suivante (par exemple, semaines suivantes) dans un champ Planning.  |
| [&lt;Planning&gt;.AjouteRendezVous](../WDLang1/1000022904.md) | Ajoute un nouveau rendez-vous dans un champ Planning. |
| [&lt;Planning&gt;.AjouteRessource](../WDLang1/1000022905.md) | Ajoute une nouvelle ressource dans un champ Planning. |
| [&lt;Planning&gt;.ChangeMode](../WDLang1/1000022906.md) | Modifie la période (c'est-à-dire le nombre de jours) affichée par un champ Planning. |
| [&lt;Planning&gt;.DateVisible](../WDLang1/1000022918.md) | Rend visible ou invisible un jour dans un champ Planning. |
| [&lt;Planning&gt;.HeureAffichée](../WDLang1/1000022907.md) | Définit la plage horaire affichée par le champ Planning en mode Jour ou Semaine. |
| [&lt;Planning&gt;.InfoXY](../WDLang1/1000022908.md) | Renvoie le rendez-vous, la ressource ou la date se trouvant à une position donnée dans un champ Planning. |
| [&lt;Planning&gt;.ListeRendezVous](../WDLang1/1000022909.md) | Renvoie : <br><br>- la liste des rendez-vous compris entre deux dates données dans un champ Planning, <br><br>- tous les rendez-vous d'un champ Planning,<br><br>- un rendez-vous particulier (sélectionné ou survolé). <br><br><br> |
| [&lt;Planning&gt;.PositionDateHeure](../WDLang1/1000022920.md) | Affiche un champ Planning à partir d'une date spécifiée ou renvoie la date du premier jour (ou dernier jour) affiché dans un champ Planning. |
| [&lt;Planning&gt;.PositionRessource](../WDLang1/1000022921.md) | Affiche une ressource donnée dans un champ Planning ou renvoie la position la première ressource visible dans un champ Planning. |
| [&lt;Planning&gt;.SelectMoins](../WDLang1/1000022910.md) | Désélectionne un rendez-vous dans un champ Planning. |
| [&lt;Planning&gt;.SelectPlus](../WDLang1/1000022911.md) | Sélectionne un rendez-vous dans un champ Planning. |
| [&lt;Planning&gt;.SupprimeRendezVous](../WDLang1/1000022912.md) | Supprime un rendez-vous dans un champ Planning. |
| [&lt;Planning&gt;.SupprimeRessource](../WDLang1/1000022913.md) | Supprime une ressource dans un champ Planning. |
| [&lt;Planning&gt;.SupprimeTout](../WDLang1/1000022914.md) | Supprime tous les rendez-vous et toutes les ressources d'un champ Planning. |

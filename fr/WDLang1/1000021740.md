


## TableurRécupèreFusion (Fonction)

***En anglais : SpreadsheetGetMerge***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de savoir si une cellule d'un champ Tableur est fusionnée avec d'autres cellules et d'obtenir le nom des cellules fusionnées. 

Remarque : Cette fonction est utilisable uniquement sur un champ Tableur présent dans une fenêtre. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
sFusion est chaîne = TableurRécupèreFusion(TBLR_MonTableur, "B1")
SI sFusion = "" ALORS
	Info("Aucune fusion")
SINON
	sMin est chaîne = ExtraitChaîne(sFusion, 1, ":")
	sMax est chaîne = ExtraitChaîne(sFusion, 2, ":")
	Info("Fusion  entre " + sMin + " et " +  sMax)
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Récupère les caractéristiques d'une cellule fusionnée identifiée par son nom

`<Résultat> = TableurRécupèreFusion(<Champ Tableur> , <Cellule>)`
---

**`<Résultat> : (Chaîne de caractères)`**



- Chaîne vide ("") si la cellule n'est pas fusionnée, 

- Plage de cellules fusionnées au format : 
	
	```txt
	"<Nom cellule haut gauche>:<Nom cellule bas droite>"
	```
Par exemple : "A2:C5" 




**`<Champ Tableur> : (Nom de champ)`**

Nom du champ Tableur à manipuler.

**`<Cellule> : (Chaîne de caractères)`**

Nom de cellule à manipuler. Par exemple : "A4".


<a name="SYNTAXE2"></a>

### Récupère les caractéristiques d'une cellule fusionnée identifiée par sa colonne et sa ligne

`<Résultat> = TableurRécupèreFusion(<Champ Tableur> , <Colonne> , <Ligne>)`
---

**`<Résultat> : (Chaîne de caractères)`**



- Chaîne vide ("") si la cellule n'est pas fusionnée, 

- Plage de cellules fusionnées au format : 
	
	```txt
	"<Nom cellule haut gauche>:<Nom cellule bas droite>"
	```
Par exemple : "A2:C5" 




**`<Champ Tableur> : (Nom de champ)`**

Nom du champ Tableur à manipuler.

**`<Colonne> : (Entier)`**

Numéro de la colonne de la cellule à manipuler. 

**`<Ligne> : (Entier)`**

Numéro de la ligne de la cellule à manipuler.

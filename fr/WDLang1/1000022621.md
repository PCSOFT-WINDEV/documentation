


## DocFerme (Fonction)

***En anglais : DocClose***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ferme le fichier docx et libère le document de type Traitement de texte. 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
DocFerme(TT_MonDocument)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`DocFerme(<Document>)`
---

**`<Document> : (Variable de type Document ou nom de champ)`**

Document à manipuler. Ce document correspond : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) au nom d'un champ Traitement de texte.

- à une variable de type [Document](../WDLang1/1000022461.md).


Le fichier physique associé au champ ou à la variable (s'il existe) est fermé. Les modifications réalisées dans le fichier ne sont pas sauvées avant sa fermeture (utilisez la fonction [DocSauve](../WDLang1/1000022387.md) si nécessaire).




## Fonctions MCI
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions MCI sont les suivantes :



|   |   |
| --- | --- |
| [MCIArrêt](../WDLang1/3051003.md) | Arrête l'exécution d'un fichier multimédia, précédemment ouvert (fonction [MCIOuvre](../WDLang1/3051017.md)). |
| [MCIDébut](../WDLang1/3051002.md) | Place au début d'un fichier multimédia, précédemment ouvert (fonction [MCIOuvre](../WDLang1/3051017.md)). |
| [MCIErr](../WDLang1/3051011.md) | Renvoie le numéro de la dernière erreur survenue sur une fonction MCI (Media Control Interface). |
| [MCIEtat](../WDLang1/3051009.md) | Renvoie l'état d'un fichier multimédia, précédemment ouvert (fonction [MCIOuvre](../WDLang1/3051017.md)). |
| [MCIExécute](../WDLang1/3051005.md) | Exécute une commande MCI (Media Control Interface). |
| [MCIFerme](../WDLang1/3051006.md) | Ferme un fichier multimédia. |
| [MCIFin](../WDLang1/3051007.md) | Place à la fin d'un fichier multimédia, précédemment ouvert (fonction [MCIOuvre](../WDLang1/3051017.md)). |
| [MCIFormatPosition](../WDLang1/3051004.md) | Fixe le format utilisé pour définir la position dans un fichier multimédia (ouvert avec la fonction [MCIOuvre](../WDLang1/3051017.md)). |
| [MCIHauteur](../WDLang1/3051008.md) | Renvoie la hauteur en pixels d'un fichier multimédia affichant des images. |
| [MCIJoue](../WDLang1/3051001.md) | Joue un fichier un fichier multimédia. |
| [MCILargeur](../WDLang1/3051020.md) | Renvoie la largeur en pixels d'un fichier multimédia affichant des images. |
| [MCIMsgErr](../WDLang1/3051021.md) | Renvoie le libellé de la dernière erreur survenue sur une fonction MCI (Media Control Interface). |
| [MCINombrePiste](../WDLang1/3051014.md) | Renvoie le nombre de pistes d'un fichier multimédia, précédemment ouvert (fonction [MCIOuvre](../WDLang1/3051017.md)). |
| [MCIOuvre](../WDLang1/3051017.md) | Ouvre un fichier multimédia. |
| [MCIPause](../WDLang1/3051015.md) | Fait une pause dans la lecture d'un fichier multimédia, précédemment ouvert (fonction [MCIOuvre](../WDLang1/3051017.md)). |
| [MCIPiste](../WDLang1/3051016.md) | Renvoie la piste courante d'un fichier multimédia, précédemment ouvert (fonction [MCIOuvre](../WDLang1/3051017.md)). |
| [MCIPosition](../WDLang1/3051013.md) | Renvoie ou modifie la position en cours dans un fichier multimédia, précédemment ouvert (fonction [MCIOuvre](../WDLang1/3051017.md)). |
| [MCIRécupère](../WDLang1/3051022.md) | Renvoie le dernier résultat de la commande MCI exécutée avec la fonction [MCIExécute](../WDLang1/3051005.md). |
| [MCITaille](../WDLang1/3051018.md) | Renvoie ou modifie la taille d'un fichier multimédia, précédemment ouvert (fonction [MCIOuvre](../WDLang1/3051017.md)). |
| [MCITailleFenêtre](../WDLang1/3051019.md) | Modifie la taille et la position de la fenêtre d'affichage d'un fichier multimédia, précédemment ouvert (fonction [MCIOuvre](../WDLang1/3051017.md)). |
| [MCIVolume](../WDLang1/3051012.md) | Modifie ou renvoie le volume sonore. |


## Description de Paramètre (Type de variable)

***En anglais : Parameter Description***
			
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Une variable de type **Description de Paramètre** permet de décrire un paramètre d'une syntaxe par programmation. 

Les caractéristiques de ce type peuvent être connues à l'aide de différentes propriétés WLangage. Ce type de variable est utilisé par les variables de type [Description de Syntaxe](../WDLang1/1410089154.md). 

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
Description_état est une Définition = RécupèreDéfinitionType(ETAT_CommandesClients)
Description_syntaxe est une Description de Syntaxe = Description_état.SyntaxeDéclaration
// Parcourt les paramètres de l'état
POUR TOUT x DE Description_syntaxe.Paramètre
	Trace(x.Nom)
FIN
```

## Propriétés


### Propriétés spécifiques à la description des variables de type Description de Paramètre
<a name="proprietes_specifiques_description_des_variables_type_description_parametre_ELTPARAGRAPHE000031"></a>Les propriétés suivantes peuvent être utilisées pour manipuler une Description de Paramètre :

| Nom de la propriété | Type manipulé | Effet |
| --- | --- | --- |
| Nom | Chaîne de caractères | Nom du paramètre. <br><br>**Cette propriété est disponible en lecture seulement**. |
| Type | [Définition](../WDLang1/1000019540.md) | Type du paramètre. <br><br>**Cette propriété est disponible en lecture seulement**. |

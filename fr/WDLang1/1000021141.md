


## Fonctions Gantt (syntaxe préfixée)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions permettant de manipuler une colonne d'un champ Table (ou Table hiérarchique) de type Diagramme de Gantt via une syntaxe préfixée sont les suivantes : 




|   |   |
| --- | --- |
| [&lt;Diagramme de Gantt&gt;.AjouteCongé](../WDLang1/1000021118.md) | Déclare une nouvelle période de congés pour une ligne dans une colonne d'un champ Table (ou Table hiérarchique) de type Diagramme de Gantt. |
| [&lt;Diagramme de Gantt&gt;.AjouteLien](../WDLang1/1000021120.md) | Crée un lien de type "Fin à Début" entre deux tâches affichées dans une colonne d'un champ Table (ou Table hiérarchique) de type Diagramme de Gantt. |
| [&lt;Diagramme de Gantt&gt;.AjouteTâche](../WDLang1/1000021121.md) | Ajoute une tâche dans une colonne d'un champ Table (ou Table hiérarchique) de type Diagramme de Gantt. |
| [&lt;Diagramme de Gantt&gt;.ChangeMode](../WDLang1/1000021122.md) | Modifie la période (c'est-à-dire le nombre de jours) affichée dans une colonne d'un champ Table (ou Table hiérarchique) de type Diagramme de Gantt. |
| [&lt;Diagramme de Gantt&gt;.HeureAffichée](../WDLang1/1000021123.md) | Définit la plage horaire affichée dans une colonne d'un champ Table (ou Table hiérarchique) de type Diagramme de Gantt. |
| [&lt;Diagramme de Gantt&gt;.HeureOuvrable](../WDLang1/1000021124.md) | Surcharge les heures ouvrables d'une ligne dans une colonne d'un champ Table (ou Table hiérarchique) de type Diagramme de Gantt. |
| [&lt;Diagramme de Gantt&gt;.JourSemaineOuvré](../WDLang1/1000021125.md) | Indique si un jour de la semaine est ouvré dans une colonne d'un champ Table (ou Table hiérarchique) de type Diagramme de Gantt. |
| [&lt;Diagramme de Gantt&gt;.ListeLien](../WDLang1/1000021128.md) | Liste les liens présents dans une colonne d'un champ Table (ou Table hiérarchique) de type Diagramme de Gantt. |
| [&lt;Diagramme de Gantt&gt;.ListeTâche](../WDLang1/1000021129.md) | Renvoie : <br><br>- la liste des tâches comprises entre deux dates données dans une colonne de type Diagramme de Gantt, <br><br>- toutes les tâches d'une colonne de type Diagramme de Gantt,<br><br>- une tâche spécifique (sélectionnée ou survolée). <br><br><br> |
| [&lt;Diagramme de Gantt&gt;.OrganiseTâche](../WDLang1/1000021130.md) | Réorganise les tâches d'un diagramme de Gantt selon les contraintes (dépendances et ressources). |
| [&lt;Diagramme de Gantt&gt;.PositionDateHeure](../WDLang1/1000021131.md) | Paramètre le premier jour et la première heure affichés dans une colonne (d'un champ Table ou Table hiérarchique) de type Diagramme de Gantt. |
| [&lt;Diagramme de Gantt&gt;.RécupèreTâche](../WDLang1/1000021137.md) | Récupère une tâche à partir de son identifiant dans une colonne (d'un champ Table ou Table hiérarchique) de type Diagramme de Gantt. |
| [&lt;Diagramme de Gantt&gt;.SupprimeLien](../WDLang1/1000021138.md) | Supprime des liens dans une colonne d'un champ Table (ou Table hiérarchique) de type Diagramme de Gantt. |
| [&lt;Diagramme de Gantt&gt;.SupprimeTâche](../WDLang1/1000021139.md) | Supprime une tâche dans une colonne d'un champ Table (ou Table hiérarchique) de type Diagramme de Gantt. |
| [&lt;Diagramme de Gantt&gt;.SupprimeTout](../WDLang1/1000021140.md) | Supprime toutes les tâches dans une colonne d'un champ Table (ou Table hiérarchique) de type Diagramme de Gantt. |




## Fonctions de gestion des champs Kanban
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
 
Les fonctions permettant de gérer les champs Kanban sont les suivantes :



|   |   |
| --- | --- |
| [KanbanAffiche](../WDLang1/1410089522.md) | Rafraîchit l'affichage d'un champ Kanban ou d'une carte d'un champ Kanban. |
| [KanbanAjouteCarte](../WDLang1/1410089140.md) | Ajoute une carte dans une liste d'un champ Kanban. |
| [KanbanAjouteListe](../WDLang1/1410089137.md) | Ajoute une nouvelle liste dans un champ Kanban. |
| [KanbanDéplaceCarte](../WDLang1/1410089144.md) | Déplace une carte dans un champ Kanban. |
| [KanbanDéplaceListe](../WDLang1/1410089540.md) | Déplace une liste de Kanban dans un champ Kanban. |
| [KanbanEnregistre](../WDLang1/1410089562.md) | Modifie l'enregistrement associé à la carte en cours dans le champ Kanban fichier. |
| [KanbanFiltre](../WDLang1/1410089534.md) | Affiche uniquement les cartes correspondant au filtre spécifié. |
| [KanbanIndiceListe](../WDLang1/1410089317.md) | Renvoie l'indice d'une liste présente dans un champ Kanban. |
| [KanbanInfoXY](../WDLang1/1410089224.md) | Renvoie pour une position donnée dans un champ Kanban (coordonnées d'un point du champ) :<br><br>- soit le numéro de la carte pour la position indiquée, <br><br>- soit le numéro de la liste pour la position indiquée.<br><br><br> |
| [KanbanInsèreCarte](../WDLang1/1410089316.md) | Insère une carte dans une liste de Kanban. |
| [KanbanInsèreListe](../WDLang1/1410089539.md) | Insère une liste de Kanban dans un champ Kanban. |
| [KanbanOccurrence](../WDLang1/1410089240.md) | Renvoie le nombre de cartes ou de listes dans un champ Kanban. |
| [KanbanPositionListe](../WDLang1/1410089241.md) | Permet de faire défiler une liste d'un champ Kanban afin d'afficher une carte donnée. |
| [KanbanSupprimeCarte](../WDLang1/1410089141.md) | Supprime une carte d'un champ Kanban ou d'un champ Liste de Kanban. |
| [KanbanSupprimeListe](../WDLang1/1410089239.md) | Supprime une liste de Kanban d'un champ Kanban. |
| [KanbanSupprimeToutCarte](../WDLang1/1410089143.md) | Supprime toutes les cartes d'un champ Kanban ou d'un champ Liste de Kanban. |

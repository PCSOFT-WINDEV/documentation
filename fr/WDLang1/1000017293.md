
## AppliMAJDisponible (Fonction)

***En anglais : AppUPDAvailable***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de savoir si une mise à jour est disponible pour l'application spécifiée.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI AppliMAJDisponible() = appMAJDispoObligatoire ALORS
	SI OuiNon("Une mise à jour de l'application est disponible.", ...
		  "Voulez-vous effectuer cette mise à jour ?") = Oui ALORS
			  AppliDéclencheMAJ()
	FIN
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = AppliMAJDisponible([<Chemin de l'application>])`
---

**`<Résultat> : (Constante de type Entier)`**

Renvoie :

- 0 en cas d'erreur lors de la recherche de la mise à jour. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).

- une des constantes suivantes :
	


|   |   |
| --- | --- |
| *appMAJDispoFacultative* | Une mise à jour facultative est disponible. |
| *appMAJDispoObligatoire* | Une mise à jour obligatoire est disponible. |
| *appMAJNonDispo* | Aucune mise à jour n'est disponible. |






**`<Chemin de l'application> : (Chaîne de caractères optionnelle)`**

Chemin complet de l'exécutable à manipuler. Si ce paramètre n'est pas précisé, l'application manipulée est l'application en cours.

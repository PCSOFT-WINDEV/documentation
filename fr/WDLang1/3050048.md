
## MatCopie (Fonction)

***En anglais : MatCopy***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Crée la copie d'une matrice. La nouvelle matrice a les mêmes dimensions que la matrice de base. Tous les éléments de la matrice d'origine sont copiés dans la nouvelle matrice.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple
<a class="notetitle" target="_blank" href="$DOC$=1000003050048&name=matcopie_fonction&product=WD">Voir des exemples supplémentaires</a>

```wl
// Copie d'une matrice
ResCopie = MatCopie("MaMatrice", "MaMatriceCopie")
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = MatCopie(<Matrice à copier> , <Matrice copiée>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la matrice a été copiée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour connaître le libellé de l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.




**`<Matrice à copier> : (Chaîne de caractères)`**

Nom de la matrice à copier, défini avec la fonction [MatCrée](../WDLang1/3050058.md).

**`<Matrice copiée> : (Chaîne de caractères)`**

Nom de la matrice copiée, automatiquement créée lors de la copie.

<a name="NOTE0_1"></a>

## Remarques




### Existence de la matrice copiée
<a name="existence_matrice_copiee_ELTPARAGRAPHE000090"></a>L'existence de la matrice n'est pas vérifiée. Si la matrice existait auparavant, cette matrice est écrasée. Pour vérifier l'existence d'une matrice, utilisez la fonction [MatExiste](../WDLang1/3050076.md).

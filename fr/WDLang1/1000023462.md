
## TableurRécupèreInfoTri (Fonction)

***En anglais : SpreadsheetGetSortInfo***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie différentes informations sur le tri effectué dans un champ Tableur. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Montre la zone qui va être triée
sSel est chaîne 
sSel = TableurRécupèreInfoTri(TBLR_Tableur, tblrInfoTriSélection) 
SI sSel = "" ALORS
	RETOUR // non triable
FIN
sMin est une chaîne = ExtraitChaine(sSel, 1, ":")
sMax est une chaîne = ExtraitChaine(sSel, 2, ":")

// Sélectionne la zone 
TableurSelectPlus(TBLR_Tableur, sMin, sMax)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = TableurRécupèreInfoTri(<Champ Tableur> , <Type d'information>)`
---

**`<Résultat> : (Chaîne de caractères)`**

Information demandée. 

**`<Champ Tableur> : (Nom de champ)`**

Nom du champ Tableur à manipuler. 

**`<Type d'information> : (Constante de type Entier)`**

Information recherchée : 


|   |   |
| --- | --- |
| *tblrInfoTriAvecTitre* | Permet de savoir si la sélection triée par la FAA (Fonctionnalité Automatique de vos Applications) de tri du champ Tableur contient un titre. &lt;Résultat&gt; peut correspondre à : <br><br>- 1 si la sélection à trier comporte un titre, <br><br>- 0 dans le cas contraire. <br><br><br> |
| *tblrInfoTriCritères* | Permet de connaître les derniers critère de tri utilisés pour trier le champ Tableur. &lt;Résultat&gt; est de la forme : <br><br><pre><code>"<Nom de la colonne1>[;< Option>]" + TAB + <br>"<Nom de la colonne2>" + TAB + ...</code></pre><br>où : <br><br>- &lt;Nom de la colonne&gt; est le nom de la colonne triée, <br><br>- &lt;Option&gt; correspond à une des constantes de tri de la fonction [TableurTrieSélection](../WDLang1/1000023463.md). <br><br><br><br><br>Exemple : "A&lt;Tab&gt;B;3" |
| *tblrInfoTriSélection* | Permet de connaître la sélection qui sera triée par la FAA (Fonctionnalité Automatique de vos Applications) de tri du champ Tableur. Exemple : "A2:B4". <br><br>Remarque : Si la sélection ne permet pas de tri, &lt;Résultat&gt; correspond à une chaîne vide (""). |

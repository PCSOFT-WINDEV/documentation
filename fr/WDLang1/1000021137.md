


## &lt;Diagramme de Gantt&gt;.RécupèreTâche (Fonction)

***En anglais : &lt;Gantt chart&gt;.GetTask***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Récupère une tâche à partir de son identifiant dans une colonne (d'un champ Table ou Table hiérarchique) de type Diagramme de Gantt.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Récupération d'une tâche pour la modifier
MaTâche est une TâcheGantt
MaTâche <- COL_Gantt.RécupèreTâche("ID2")
MaTâche.Titre = "Nouveau titre"
```


<a name="1000021026_Exemple2"></a>

```wl
// Récupération d'une tâche
MaTâche est un TâcheGantt
MaTâche = COL_Gantt.RécupèreTâche("ID1")
InfoConstruit("Tâche récupérée : %1", MaTâche.Titre)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Colonne Gantt>.RécupèreTâche(<Identifiant>)`
---

**`<Résultat> : (Variable de type TâcheGantt)`**

Nom de la variable de type [TâcheGantt](../WDLang1/1000020881.md) dans laquelle les caractéristiques de la tâche sont récupérées.

**`<Colonne Gantt> : (Nom de champ)`**

Nom de la colonne (d'un champ Table ou Table hiérarchique) de type Diagramme de Gantt à manipuler.

**`<Identifiant> : (Chaîne de caractères)`**

Identifiant de la tâche à récupérer. Cet identifiant correspond à la propriété **ID** définie dans la variable de type [TâcheGantt](../WDLang1/1000020881.md). 

Si l'identifiant spécifié n'existe pas, une erreur WLangage est affichée.

## Remarques


### Récupération d'une tâche
<a name="recuperation_une_tache_ELTPARAGRAPHE000187"></a>Pour récupérer une tâche, il est possible : 

- soit d'utiliser l'opérateur "=". Dans ce cas, il est possible de connaître les caractéristiques de la tâche récupérée. 

- soit d'utiliser l'opérateur "&lt;-". **Dans ce cas, il est possible de modifier dynamiquement la tâche récupérée**. 




**1er cas : Utilisation de l'opérateur "="**. 

Dans ce cas, il est possible de connaître les caractéristiques de la tâche récupérée. 

```txt
<Nom de la variable> = GanttRécupèreTâche(<Colonne Gantt>, <Identifiant>)
```

Par exemple :

```wl
// Récupération d'une tâche
MaTâche est un TâcheGantt
MaTâche = COL_Gantt.RécupèreTâche("ID1")
InfoConstruit("Tâche récupérée : %1", MaTâche.Titre)
```

Il est également possible de récupérer une tâche en utilisant la syntaxe suivante : 

```txt
<Nom de la variable> = <Colonne Gantt>["<Identifiant>"]
```

Par exemple : 

```wl
MaTâche est un TâcheGantt
MaTâche = COL_Gantt["ID1"]
InfoConstruit("Tâche récupérée : %1", MaTâche.Titre)
```


**2ème cas : Utilisation de l'opérateur "&lt;-".**

Dans ce cas, il est possible de modifier dynamiquement la tâche récupérée. 

```txt
<Nom de la variable> <- GanttRécupèreTâche(<Colonne Gantt>, <Identifiant>)
```
où Identifiant est l'identifiant de la tâche à récupérer. 

Par exemple : 

```wl
// Modification d'une tâche
MaTâche est un TâcheGantt
MaTâche <- COL_Gantt.RécupèreTâche("ID1")
MaTâche.Avancement = 70
MaTâche.CouleurAvancement = VertClair
```

Il est également possible de récupérer une tâche en utilisant la syntaxe suivante : 

```txt
<Nom de la variable> <- <Colonne Gantt>["<Identifiant>"]
```
Par exemple : 

```wl
t <-COL_Gantt["ID1"]
t.CouleurFond = BleuClair
```


## Constantes de gestion des notifications interactives
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les constantes de gestion des notifications interactives sont les suivantes :

| **sysNotifBoutonContextuel** | Comportement ou positionnement spécifique du bouton : Bouton à ajouter au menu contextuel de la notification. | [sysNotificationBouton](../WDLang1/1410087947.md) |
| --- | --- | --- |
| **sysNotifBoutonNormal** | Comportement ou positionnement spécifique du bouton : Bouton simple. | [sysNotificationBouton](../WDLang1/1410087947.md) |
| **sysNotifBoutonRappel** | Comportement ou positionnement spécifique du bouton : Bouton système de type "Rappeler plus tard" | [sysNotificationBouton](../WDLang1/1410087947.md) |
| **sysNotifBoutonFerme** | Comportement ou positionnement spécifique du bouton : Bouton système qui ferme automatiquement la notification. | [sysNotificationBouton](../WDLang1/1410087947.md) |
| **sysNotifNormal** | Comportement spécifique de la notification : La notification est affichée quelques secondes à l'écran avant de se mettre dans le centre d'actions de Windows. | [sysNotification](../WDLang1/1410087893.md) |
| **sysNotifAppel** | Comportement spécifique de la notification : La notification reste à l'écran jusqu'à intervention de l'utilisateur. Un son système d'appel entrant est joué en boucle et un style différent est utilisé pour les boutons. | [sysNotification](../WDLang1/1410087893.md) |
| **sysNotifRappel** | Comportement spécifique de la notification : La notification reste à l'écran jusqu'à intervention de l'utilisateur. Un son système de "rappel" est joué à l'apparition de la notification. | [sysNotification](../WDLang1/1410087893.md) |
| **sysNotifAlarme** | Comportement spécifique de la notification : La notification reste à l'écran jusqu'à intervention de l'utilisateur. Un son système de "rappel" est joué en boucle. | [sysNotification](../WDLang1/1410087893.md) |
| **sysNotifAucunContour** | Contour à appliquer sur le logo : Aucun contour, image carrée. | [sysNotification](../WDLang1/1410087893.md) |
| **sysNotifContourCercle** | Contour à appliquer sur le logo : Image avec un contour circulaire. | [sysNotification](../WDLang1/1410087893.md) |
| Constante | Détail | Type de variable |
| **sysNotifEntréeCombo** | Type du champ permettant une interaction avec l'utilisateur : champ Combo. | [sysNotificationEntrée](../WDLang1/1410088024.md) |
| **sysNotifEntréeTexte** | Type du champ permettant une interaction avec l'utilisateur : champ de saisie texte. | [sysNotificationEntrée](../WDLang1/1410088024.md) |

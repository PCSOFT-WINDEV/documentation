
## SysCouleurRes (Fonction)

***En anglais : SysColorRes***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la résolution de l'écran en nombre de couleurs.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connaître la résolution de l'écran
ResCouleurRes = SysCouleurRes()
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = SysCouleurRes()`
---

**`<Résultat> : (Entier)`**

Nombre de couleurs de la résolution de l'écran. Ce nombre est défini dans le panneau de configuration de Windows.

## Composante :
wd280std.dll

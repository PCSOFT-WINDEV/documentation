


## GanttListeTâche (Fonction)

***En anglais : GanttListTask***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie : 

- la liste des tâches comprises entre deux dates données dans une colonne de type Diagramme de Gantt, 

- toutes les tâches d'une colonne de type Diagramme de Gantt,

- une tâche spécifique (sélectionnée ou survolée). 



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple
<a class="notetitle" target="_blank" href="$DOC$=1000021119&name=ganttlistetache_fonction&product=WD">Voir des exemples supplémentaires</a>

```wl
// Liste toutes les tâches
tabListe est un tableau de TâcheGantt

// Récupère la liste des tâches
tabListe = GanttListeTâche(COL_Gantt)

InfoConstruit("Les tâches ont bien été récupérées" + RC + 
	"Le tableau contient %1 tâche(s).", tabListe.Occurrence)
```


<a name="Exemple2"></a>

```wl
// Liste les tâches à partir de la date du jour
tabListe est un tableau de TâcheGantt

// Récupère la liste des tâches
tabListe = GanttListeTâche(COL_Gantt, DateSys())
```


<a name="Exemple3"></a>

```wl
// Récupère la tâche sélectionnée
tabListe est un tableau de TâcheGantt

// Récupère la liste des tâches
tabListe = GanttListeTâche(COL_Gantt, ganttTâcheSélectionnée)

InfoConstruit("La tâche récupérée : %1.", tabListe[1].Titre)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Récupérer la liste des tâches entre deux dates

`<Résultat> = GanttListeTâche(<Colonne Gantt> [, <Date de début> [, <Date de fin>]])`
---

**`<Résultat> : (Tableau de variables de type TâcheGantt)`**

Nom du tableau contenant les variables de type [TâcheGantt](../WDLang1/1000020881.md). 
Si une tâche correspondant aux critères spécifiés existe, ce tableau contiendra un seul élément. 
Si aucune tâche ne correspond aux critères spécifiés, ce tableau contiendra 0 tâche.

**`<Colonne Gantt> : (Nom de champ)`**

Nom de la colonne (d'un champ Table ou Table hiérarchique) de type Diagramme de Gantt à manipuler.

**`<Date de début> : (Variable optionnelle de type DateHeure)`**

Date de début de sélection des tâches.

Si ce paramètre n'est pas spécifié, toutes les tâches du Diagramme de Gantt seront renvoyées.

**`<Date de fin> : (Variable optionnelle de type DateHeure)`**

Date de fin de sélection des tâches.

Si ce paramètre n'est pas spécifié, toutes les tâches du Diagramme de Gantt seront renvoyées.


<a name="SYNTAXE2"></a>

### Récupérer les informations d'une tâche spécifique

`<Résultat> = GanttListeTâche(<Colonne Gantt> , <Type de tâche>)`
---

**`<Résultat> : (Tableau de variables de type TâcheGantt)`**

Nom du tableau contenant la variable de type [TâcheGantt](../WDLang1/1000020881.md). 
Si une tâche correspondant aux critères spécifiés existe, ce tableau contiendra un seul élément. 
Si aucune tâche ne correspond aux critères spécifiés, ce tableau contiendra 0 tâche.

**`<Colonne Gantt> : (Nom de champ)`**

Nom de la colonne (d'un champ Table ou Table hiérarchique) de type Diagramme de Gantt à manipuler.

**`<Type de tâche> : (Constante de type Entier)`**

Type de la tâche recherchée :


|   |   |
| --- | --- |
| *ganttTâcheSélectionnée* | Tâche actuellement sélectionnée dans le champ Diagramme de Gantt. Si aucune tâche n'est sélectionnée, &lt;Résultat&gt; correspondra à un tableau de 0 élément. |
| *ganttTâcheSurvolée* | Tâche survolée par la souris dans le champ Diagramme de Gantt. Si aucune tâche n'est survolée, &lt;Résultat&gt; correspondra à un tableau de 0 élément. |




## NoteCrée (Fonction)

***En anglais : NoteCreate***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Crée un champ Note repositionnable dans la fenêtre en cours par programmation. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Crée un champ Note repositionnable coloré
NouvelleNote est un Champ
NouvelleNote <- NoteCrée("Notez vos remarques ici")
NouvelleNote.CouleurFond = DonneCouleur()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = NoteCrée([<Texte de la note> [, <X> [, <Y>]]])`
---

**`<Résultat> : (Variable de type Champ)`**

Le champ Note repositionnable créé peut être manipulé via une variable de type [Champ](../WDLang1/1000020967.md). Dans ce cas, l'affectation doit être réalisée avec l'opérateur **'<-'**. 

Si la fonction **NoteCrée** est utilisée directement, le champ est créé dans la fenêtre et est visible par défaut. 

**`<Texte de la note> : (Chaîne de caractères optionnelle)`**

Texte à afficher dans le champ Note repositionnable. 

**`<X> : (Entier optionnel)`**

Abscisse du champ Note repositionnable à créer (position sur l'axe des X) en pixels. Position horizontale du coin haut gauche du champ Note Repositionnable par rapport au coin haut gauche de la zone client de la fenêtre (c'est-à-dire la fenêtre sans les barres de titre et de menu ni le cadre). Ce paramètre correspond à la propriété [XInitial](../Proprietes/2510140.md). 

Si ce paramètre n'est pas précisé, l'abscisse du champ est 0. 

Il est ensuite possible de modifier cette valeur avec la propriété [X](../Proprietes/2510135.md).

**`<Y> : (Entier optionnel)`**

Ordonnée du champ Note repositionnable à créer (position sur l'axe des Y) en pixels. Position verticale du coin haut gauche du champ Note repositionnable par rapport au coin haut gauche de la zone client de la fenêtre (c'est-à-dire la fenêtre sans les barres de titre et de menu ni le cadre). Ce paramètre correspond à la propriété [YInitial](../Proprietes/2510087.md). 

Si ce paramètre n'est pas précisé, l'ordonnée du champ est 0. 

Il est ensuite possible de modifier cette valeur avec la propriété [Y](../Proprietes/2510137.md).

## Remarques


### Caractéristiques de la note repositionnable
<a name="caracteristiques_note_repositionnable_ELTPARAGRAPHE000080"></a>

- La création peut être annulée par Ctrl + Z. 

- La note repositionnable créée est une note persistante : si la fenêtre est fermée puis ré-ouverte, la note sera restaurée. 
	Pour créer une note non persistante, utilisez la fonction [ChampCrée](../WDLang1/1000020876.md) avec la constante *typNoteRepositionnable*. 

- Toutes les notes repositionnables présentes dans la fenêtre peuvent être sauvées et restaurées via les fonctions [NoteChargeTout](../WDLang1/1000022397.md) et [NoteSauveTout](../WDLang1/1000022396.md). 





### Equivalences
<a name="equivalences_ELTPARAGRAPHE000100"></a>

- Cette fonction est équivalente à la création d'une note par FAA (Fonctionnalité Automatique de vos Applications) dans la fenêtre par l'utilisateur final. Pour plus de détails, consultez [Note repositionnable (FAA)](../Editeurs/1000021820.md). 

- Cette fonction est équivalente à la fonction [FenEdCréeNote](../WDLang1/1000022270.md) utilisée pour créer une note dans la fenêtre en cours.


## Fonctions de gestion des champs Tableau de bord (syntaxe préfixée)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions de gestion des champs Tableau de bord (syntaxe préfixée) sont les suivantes :



|   |   |
| --- | --- |
| [&lt;Tableau de bord&gt;.Affiche](../WDLang1/1000022500.md) | Rafraîchit l'affichage des éléments d'un champ Tableau de bord. |
| [&lt;Tableau de bord&gt;.AjouteWidget](../WDLang1/1000022502.md) | Ajoute et affiche un Widget (fenêtre interne) dans un champ Tableau de bord. |
| [&lt;Tableau de bord&gt;.ChangeTailleWidget](../WDLang1/1000022504.md) | Change les dimensions d'un Widget présent dans un champ Tableau de bord. |
| [&lt;Tableau de bord&gt;.ChargeConfiguration](../WDLang1/1000022506.md) | Restaure la configuration d'un champ Tableau de bord. |
| [&lt;Tableau de bord&gt;.ConfigurationInitiale](../WDLang1/1000022525.md) | Modifie la configuration d'un champ Tableau de bord (définie sous l'éditeur) avant son premier affichage. |
| [&lt;Tableau de bord&gt;.DéplaceWidget](../WDLang1/1000022527.md) | Change la position d'un widget présent dans un champ Tableau de bord. |
| [&lt;Tableau de bord&gt;.InfoXY](../WDLang1/1000022529.md) | Permet de savoir :<br><br>- si un Widget est présent dans un champ Tableau de bord à une position donnée.<br><br>- quel est le Widget présent dans un champ Tableau de bord à une position donnée.<br><br><br> |
| [&lt;Tableau de bord&gt;.Mode](../WDLang1/1000022532.md) | Ouvre ou ferme le mode "Edition" d'un champ Tableau de bord. |
| [&lt;Tableau de bord&gt;.Occurrence](../WDLang1/1000022538.md) | Renvoie le nombre de Widgets présents dans un champ Tableau de bord. |
| [&lt;Tableau de bord&gt;.RécupèreWidgetPosition](../WDLang1/1000022533.md) | Renvoie la position d'un Widget présent dans un champ Tableau de bord. |
| [&lt;Tableau de bord&gt;.SauveConfiguration](../WDLang1/1000022534.md) | Sauvegarde la configuration d'un champ Tableau de bord sous forme d'une chaîne. |
| [&lt;Tableau de bord&gt;.Supprime](../WDLang1/1000022535.md) | Supprime un Widget affiché dans un champ Tableau de bord. Le champ Tableau de bord est mis à jour et un emplacement vide remplace le Widget supprimé. |
| [&lt;Tableau de bord&gt;.SupprimeTout](../WDLang1/1000022536.md) | Supprime tous les Widgets affichés dans un champ Tableau de bord. |


## Description de TâcheParallèle (Type de variable)

***En anglais : Description of ParallelTask***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **Description de TâcheParallèle** permet de définir toutes les caractéristiques d'une tâche parallèle. Les caractéristiques de cette tâche peuvent être définies et modifiées à l'aide de différentes propriétés WLangage. 

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
d est une Description de TâcheParallèle
d.Procédure = ProcédureAExécuter
d.Option = tpoCopieLégèreContexteHFSQL
TâcheParallèleExécute(d)
```





<a name="NOTE0"></a>

## Remarques
<a name="NOTE0_1"></a>


### Propriétés spécifiques à la description des variables de type Description de TâcheParallèle
<a name="proprietes_specifiques_description_des_variables_type_description_tacheparallele_ELTPARAGRAPHE000051"></a>Les propriétés suivantes peuvent être utilisées pour manipuler une Description de TâcheParallèle  :

| Nom de la propriété | Type manipulé | Effet |
| --- | --- | --- |
| Option | Constante de type Entier | Options de la tâche : <br><br>- *tpoCopieLégèreContexteHFSQL* : Effectue une copie légère du contexte HFSQL courant au moment de la demande d'exécution de la tâche. <br>	Seuls les répertoires où se trouvent les fichiers de données en HFSQL Classic et/ou les connexions en HFSQL Client/Serveur sont mémorisés.<br><br>- *tpoCopieComplèteContexteHFSQL* : Effectue une copie complète du contexte HFSQL courant au moment de la demande d'exécution de la tâche. <br>	Conseillé par exemple si la tâche doit tenir compte des positions en cours dans les fichiers et requêtes du contexte de l'appelant.<br><br>- *tpoCopieDifféréeContexteHFSQL* (Valeur par défaut) : Effectue une copie légère du contexte HFSQL **lors du premier accès aux données HFSQL**.<br><br>- *tpoThreadPrincipal* : Exécute la tâche dans le thread principal.<br><br><br> |
| Paramètre | Tableau de variants | Tableau contenant les paramètres de la tâche. |
| Procédure | Procédure ou chaîne de caractères | Procédure à exécuter lors de l'exécution de la tâche parallèle. Cette propriété peut correspondre : <br><br>- à une chaîne de caractères correspondant au nom d'une procédure WLangage. <br><br>- à une variable de type [Procédure](../Motscles/1000018907.md). <br><br><br> |




## &lt;Menu&gt;.AjouteOptionURL (Fonction)

***En anglais : &lt;Menu&gt;.AddURLOption***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ajoute une nouvelle option de menu à la suite de la dernière option d'un menu (déroulant ou contextuel). Cette option de menu permet d'afficher la page correspondant à l'URL indiquée.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ajoute un nouveau menu
_Menu.AjouteMenu("MNU_Site", "Mon Site")
// Ajoute une option dans le nouveau menu
MNU_Site.AjouteOptionURL("OPT_PCSOFT", "PC Soft", "http://www.pcsoft.fr")
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Menu d'origine>.AjouteOptionURL(<Option à créer> , <Libellé> , <URL de l'option>)`
---

**`<Résultat> : (Entier)`**

Position de l'option dans le menu. Si l'ajout de l'option n'a pas été effectué, une erreur fatale est affichée.

**`<Menu d'origine> : (Nom de menu)`**

Nom du menu dans lequel l'option de menu doit être ajoutée. L'option de menu sera ajoutée à la suite de la dernière option de ce menu.
Remarque : Ce nom de menu peut correspondre à un menu contextuel. 

**`<Option à créer> : (Chaîne de caractères)`**

Nom de l'option de menu à ajouter. Ce nom sera utilisé pour manipuler l'option de menu par programmation. Si ce nom correspond à une option existante, une erreur fatale est affichée.

**`<Libellé> : (Chaîne de caractères)`**

Libellé de la nouvelle option de menu. Ce libellé sera affiché dans la page et permettra à l'internaute de sélectionner l'option.

**`<URL de l'option> : (Chaîne de caractères)`**

Adresse de la page à afficher lors de la sélection de l'option.

## Remarques
![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png) 

### Lancer une page AWP du projet
<a name="lancer_une_page_awp_projet_ELTPARAGRAPHE000193"></a>Il est possible d'utiliser la fonction **&lt;Menu&gt;.AjouteOptionURL** pour lancer une page AWP du projet. Dans ce cas, il est possible d'utiliser la fonction [&lt;Page&gt;.Adresse](../WDLang2/1000021660.md) pour construire l'URL de cette page. Par exemple :


```wl
MENU_Menu1.AjouteOptionURL("OPT_PageAWP", "Page AWP", PAGE_AWP.Adresse())
```

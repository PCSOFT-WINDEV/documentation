


## ExeTermine (Fonction)

***En anglais : ExeTerminate***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Termine l'exécution d'une application (32 bits ou 64 bits) sur le poste en cours : toutes les instances de l'exécutable sont "tuées". L'application peut être identifiée par son nom ou son PID.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) 
```wl
bRes est un booléen
bRes = ExeTermine("C:\Draw\psp\psp.exe")
```

<a name="SYNTAXE1"></a>

## Syntaxe

### Terminer une application identifiée par son nom

`<Résultat> = ExeTermine(<Nom de l'exécutable>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si toutes les instances du programme ont été supprimées, 

- <u><u><u><u>Faux</u></u></u></u> en cas de problème. Cette fonction peut échouer :

	- si l'exécutable n'est pas instancié (l'exécutable n'est pas lancé). 

	- si l'utilisateur ne dispose pas de droits suffisants pour détruire une des instances. 







**`<Nom de l'exécutable> : (Chaîne de caractères)`**

Nom et chemin complet (ou relatif) de l'exécutable (260 caractères maximum).



<a name="SYNTAXE2"></a>

### Terminer une application identifiée par son identificateur de processus (PID)

`<Résultat> = ExeTermine(<Identificateur de processus>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si toutes les instances du programme ont été supprimées, 

- <u><u><u><u>Faux</u></u></u></u> en cas de problème. Cette fonction peut échouer :

	- si l'exécutable n'est pas instancié (l'exécutable n'est pas lancé). 

	- si l'utilisateur ne dispose pas de droits suffisants pour détruire une des instances. 







**`<Identificateur de processus> : (Entier)`**

Identificateur de processus (PID) de l'application. Cet identifiant peut être connu par la fonction [LanceAppli](../WDLang1/3035006.md) (avec la constante *exePID*) ou la fonction [ExeDonnePID](../WDLang1/1000017239.md).

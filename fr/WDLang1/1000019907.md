


## NuméroDeMoisVersLettre (Fonction)

***En anglais : MonthNumberInAlpha***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le nom du mois en fonction de son numéro.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
sWarning = "Le bureau est fermé en " + NuméroDeMoisVersLettre(2)
// Affiche "Le bureau est fermé en Février"
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = NuméroDeMoisVersLettre(<Numéro du mois>)`
---

**`<Résultat> : (Chaîne de caractères)`**

Nom du mois correspondant au numéro spécifié : Janvier, Février, Mars, Avril, Mai, Juin, Juillet, Août, Septembre, Octobre, Novembre, Décembre. 

**`<Numéro du mois> : (Entier)`**

Numéro du mois compris entre 1 et 12 (1 correspond à Janvier et 12 correspond à Décembre).



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Les noms des mois dépendent : 

- de la langue en cours (définie par la fonction [Nation](../WDLang1/3054014.md)),

- des noms des mois personnalisés spécifiés dans les caractéristiques du projet (onglet "Langues" de la fenêtre de description du projet). 
	Rappel : Pour afficher la description du projet, sous le volet "Projet", dans le groupe "Projet", cliquez sur "Description".

- des paramètres régionaux de Windows définis pour la langue en cours.

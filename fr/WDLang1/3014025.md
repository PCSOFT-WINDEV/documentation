


## Fonctions de débogage
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


Les différentes fonctions de débogage sont les suivantes :



|   |   |
| --- | --- |
| [dbgActiveAssertion](../WDLang1/3014021.md) | Active ou désactive la programmation défensive. |
| [dbgActiveAudit](../WDLang1/1000018835.md) | Active l'audit dynamique par programmation. |
| [dbgActiveLog](../WDLang1/1000017137.md) | Active par programmation le log d'exécution. |
| [dbgAssertion](../WDLang1/3014022.md) | Permet de trouver et de signaler les erreurs potentielles grâce à un mode de programmation défensif. |
| [dbgConnecte](../WDLang1/1000024735.md) | Permet de connecter un débogueur à distance sur une application, un service, un Webservice, ... |
| [dbgDébutDiffMémoire](../WDLang1/1000018840.md) | Débute une opération de différence de la mémoire utilisée par l'application. |
| [dbgDébutEnregistrement](../WDLang1/1000017160.md) | Démarre l'enregistrement d'un scénario de test (également appelé scénario de reproduction). |
| [dbgEcritCommentaireAudit](../WDLang1/1000018839.md) | Ecrit un commentaire dans l'audit dynamique en cours. |
| [dbgEcritWarningAudit](../WDLang1/1000018838.md) | Ecrit un warning d'exécution personnalisé dans l'audit dynamique. |
| [dbgErreur](../WDLang1/1000020526.md) | Provoque : <br><br>- une assertion lors du test de l'élément ou du projet (mode Go). <br><br>- une erreur en mode test automatique. <br><br><br> |
| [dbgEtatAudit](../WDLang1/1000018836.md) | Récupère et modifie l'état de l'audit dynamique. |
| [dbgEtatLog](../WDLang1/1000017138.md) | Renvoie et modifie l'état du log d'exécution. |
| [dbgFinDiffMémoire](../WDLang1/1000018841.md) | Sauve un dump mémoire contenant uniquement les ressources mémoire qui ont été allouées et qui n'ont pas encore été libérées depuis l'appel précédent de la fonction [dbgDébutDiffMémoire](../WDLang1/1000018840.md). |
| [dbgFinEnregistrement](../WDLang1/1000017161.md) | Arrête l'enregistrement du scénario en cours. |
| [dbgInfo](../WDLang1/3014024.md) | Récupère une information de débogage. |
| [dbgOptionAudit](../WDLang1/1000018837.md) | Récupère ou modifie les options de l'audit dynamique. |
| [dbgOptionLog](../WDLang1/1000017136.md) | Récupère ou modifie les options du log d'exécution. |
| [dbgSauveDumpDébogage](../WDLang1/1000018834.md) | Sauve un dump de débogage de l'application. |
| [dbgSauveDumpMémoire](../WDLang1/1000018852.md) | Sauve un dump de la mémoire utilisée par l'application. |
| [dbgSortieStandard](../WDLang1/3014020.md) | Écrit une information dans le flux de sortie standard "stdout" (également appelé "console"). |
| [dbgSurErreur](../WDLang1/1000020722.md) | Définit la procédure WLangage qui sera appelée en cas d'erreur détectée par une fonction de la famille dbgVérifieXXXX. |
| [dbgVérifieDifférence](../WDLang1/1000019777.md) | Vérifie si les expressions spécifiées sont différentes en programmation défensive et en mode test automatique. |
| [dbgVérifieEgalité](../WDLang1/1000019776.md) | Vérifie si les expressions sont égales en programmation défensive et en mode test automatique. |
| [dbgVérifieFaux](../WDLang1/1000019781.md) | Vérifie si une expression est fausse en programmation défensive et en mode test automatique. |
| [dbgVérifieNonNull](../WDLang1/1000019779.md) | Vérifie si l'expression n'est pas Null en programmation défensive et en mode test automatique. |
| [dbgVérifieNull](../WDLang1/1000019778.md) | Vérifie si l'expression est Null en programmation défensive et en mode test automatique. |
| [dbgVérifieVrai](../WDLang1/1000019780.md) | Vérifie si une expression vaut Vrai en programmation défensive et en mode test automatique. |


## DateSys (Fonction)

***En anglais : DateSys***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie ou modifie la date système (date du jour définie pour le poste en cours).

**Remarques** : 

- Cette fonction est identique à la fonction [DateDuJour](../WDLang1/3027016.md).







<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
Res = DateSys()   // Res = "20011225"
Res = DateVersChaîne(DateSys()) // Res = "25/12/2001"
Année = Gauche(DateSys(), 4)
// Année retourne l'année en cours (par exemple 2001)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître la date système

`<Résultat> = DateSys()`
---

**`<Résultat> : (Chaîne de caractères ou variable de type Date)`**

Date système au format AAAAMMJJ (AAAA correspond à l'année, MM au mois et JJ au jour).  


<a name="SYNTAXE2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) 
### Modifier la date système

`DateSys(<Nouvelle date>)`
---

**`<Nouvelle date> : (Chaîne de caractères, variable de type Date ou variable de type DateHeure)`**

Nouvelle date système. Cette date peut correspondre à : 

- une chaîne de caractères au format AAAAMMJJ (AAAA correspond à l'année, MM au mois et JJ au jour).  

- une chaîne de caractères au format AAAAMMJJHHmmSSCC (AAAA correspond à l'année, MM au mois, JJ au jour, HH aux heures, mm aux minutes, SS aux secondes et CC aux centièmes de seconde (utiliser CCC pour les millièmes de seconde)). Dans ce cas, seule la date est prise en compte. 

- une variable de type Date. 

- une variable de type DateHeure. Dans ce cas, seule la date est prise en compte.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Pour obtenir la date système sous un autre format que AAAAMMJJ, il faut formater la chaîne (par exemple avec les fonctions [DateVersChaîne](../WDLang1/3027025.md), [Droite](../WDLang1/3024016.md), [Gauche](../WDLang1/3024001.md)) ou utiliser les [propriétés sur les dates](../Proprietes/2514010.md). Le format AAAAMMJJ a été choisi car il permet de tester la chronologie de deux dates selon ce même format, alors que cela ne serait pas possible avec une date au format AAAAJJMM ou AAMMJJ par exemple. 

- Sur tous les systèmes actuels, l'année de la date système peut être comprise entre 1920 et 2120.

- **Attention** : Pour modifier la date système, il est nécessaire de posséder les droits administrateur sur le poste en cours.

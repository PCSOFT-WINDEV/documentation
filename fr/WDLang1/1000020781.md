


## &lt;TimeLine&gt;.SupprimePiste (Fonction)

***En anglais : &lt;TimeLine&gt;.DeleteTrack***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime une piste dans un champ TimeLine. 

**Attention** : Si le champ TimeLine est lié à une source de données, la suppression est réalisée uniquement dans le champ : les événements liés à la piste ne sont pas supprimés de la source de données.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Supprime la piste SON
TL_MONTIMELINE.SupprimePiste("Son")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Champ TimeLine>.SupprimePiste(<Piste>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si une piste a été supprimée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Champ TimeLine> : (Nom de champ)`**

Nom du champ TimeLine dans lequel la piste doit être supprimée.

**`<Piste> : (Chaîne de caractères)`**

Nom de la piste à supprimer.

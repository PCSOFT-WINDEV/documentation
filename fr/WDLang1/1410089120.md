
## Composant (Type de variable)

***En anglais : Component***
			
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **Composant** permet de manipuler facilement un composant externe. Il permet de récupérer une [interface](../POO/6010017.md) globale sur le composant. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Charment d'un composant
compo est un Composant
SI ComposantCharge(compo, fRepExe() + fSep() + "CompoExemple.wdk") ALORS
	Trace("OK")
	// Exécution des procédures du composant
	Trace(ComposantExécute("ProcéduresGlobales.Test"))
	Trace(ComposantExécute("ProcéduresGlobales.TestAvecParam", (1)))
FIN
```




## Syntaxe
<a name="SYNTAXE1"></a>

`MaVariable est un Composant`
---







<a name="NOTE0"></a>

## Remarques


- Pour initialiser une variable de type Composant, utilisez la fonction [ComposantCharge](../WDLang1/3014023.md).

- Une variable de type **Composant** peut être manipulée grâce aux fonctions : 
	


|   |   |
| --- | --- |
| [ComposantCharge](../WDLang1/3014023.md) | Charge un composant externe. Ce composant peut être :<br><br>	- un composant externe présent dans le projet en cours. <br><br>	- un composant externe indépendant du projet en cours.<br><br><br> |
| [ComposantExécute](../WDLang1/1410089107.md) | Exécute une procédure globale présente dans un composant externe. Ce composant externe a été précédemment chargé par la fonction [ComposantCharge](../WDLang1/3014023.md). |

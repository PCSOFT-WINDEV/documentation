
## StatCorrélation (Fonction)

***En anglais : StatCorrelation***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Calcule le coefficient de corrélation entre deux séries de valeurs.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple
<a class="notetitle" target="_blank" href="$DOC$=1000003050022&name=statcorrelation_fonction&product=WD">Voir des exemples supplémentaires</a>

```wl
// Calcul du coefficient de corrélation entre la ligne 1 et la ligne 2 d'une matrice
ResCoefCorrélation = StatCorrélation("MaMatrice", 1, 2, Faux)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = StatCorrélation(<Matrice> [, <1ère ligne/1ère colonne> [, <2ème ligne/2ème colonne> [, <Ligne/Colonne>]]])`
---

**`<Résultat> : (Réel)`**



- Coefficient de corrélation existant entre les deux séries de valeurs, 

- 0 si le coefficient de corrélation est nul.




**`<Matrice> : (Chaîne de caractères)`**

Nom de la matrice contenant les données, défini avec la fonction [MatCrée](../WDLang1/3050058.md).

**`<1ère ligne/1ère colonne> : (Entier optionnel)`**

1ère ligne ou 1ère colonne contenant les données (1 par défaut).

**`<2ème ligne/2ème colonne> : (Entier optionnel)`**

2ème ligne ou 2ème colonne contenant les données (2 par défaut).

**`<Ligne/Colonne> : (Booléen optionnel)`**



- <u><u><u><u>Vrai</u></u></u></u> (par défaut) si les séries de valeurs correspondent à des colonnes de la matrice, 

- <u><u><u><u>Faux</u></u></u></u> si les séries de valeurs correspondent à des lignes de la matrice.

## Remarques




### Calcul du coefficient de corrélation
<a name="calcul_coefficient_correlation_ELTPARAGRAPHE000095"></a>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=STATcORReLATION1.gif) StatCorrélation("MaMatrice", 1, 3, Faux) retourne 0,997054.
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=STATcORReLATION2.gif) StatCorrélation("MaMatrice", 1, 3, Faux) retourne 1.
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Statcorrelation3.gif) StatCorrélation("MaMatrice", 1, 3, Faux) retourne 0.


<a name="NOTE0_2"></a>




### Nombre d'éléments présents dans la matrice
<a name="nombre_elements_presents_dans_matrice_ELTPARAGRAPHE000111"></a>Le nombre d'éléments présents dans la matrice est pris comme référence. Ce nombre d'éléments correspond :

- au nombre de lignes de la matrice, connu avec la fonction [MatNbLigne](../WDLang1/3050060.md).

- au nombre de colonnes de la matrice, connu avec la fonction [MatNbColonne](../WDLang1/3050073.md).




Si les séries comportent moins d'éléments que la matrice, les valeurs manquantes sont automatiquement complétées par des 0. Ces valeurs sont prises en compte lors des calculs.


<a name="NOTE0_3"></a>




### Gestion des erreurs
<a name="gestion_des_erreurs_ELTPARAGRAPHE000131"></a>**Attention** : La fonction **StatCorrélation** ne retourne pas de code d'erreur. Pour savoir si le calcul du coefficient de corrélation a généré des erreurs, utilisez la fonction [StatErreur](../WDLang1/3050019.md). Le libellé de l'erreur peut être connu grâce à la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.

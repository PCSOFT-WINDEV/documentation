


## Fonctions de gestion des projets
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Ces fonctions permettent de manipuler facilement des éléments d'un projet WINDEV :




|   |   |
| --- | --- |
| [ComposantCharge](../WDLang1/3014023.md) | Charge un composant externe. Ce composant peut être :<br><br>- un composant externe présent dans le projet en cours. <br><br>- un composant externe indépendant du projet en cours.<br><br><br> |
| [ComposantChargeTout](../WDLang1/1000018926.md) | Charge en mémoire de manière récursive tous les composants externes utilisés dans le projet. |
| [ComposantExécute](../WDLang1/1410089107.md) | Exécute une procédure globale présente dans un composant externe. Ce composant externe a été précédemment chargé par la fonction [ComposantCharge](../WDLang1/3014023.md). |
| [ComposantListe](../WDLang1/3064007.md) | Retourne la liste des composants externes (fichiers ".WDK") utilisés par l'application WINDEV, WINDEV Mobile ou par le site WEBDEV en cours. |
| [ComposantLocalise](../WDLang1/3064009.md) | Spécifie le chemin d'accès à un composant externe du projet. |
| [EnumèreElément](../WDLang1/3064001.md) | Permet d'énumérer les éléments du projet : fenêtres, pages, états, requêtes, ... |
| [EnumèreSousElément](../WDLang1/3064002.md) | Permet d'énumérer les sous-éléments d'un élément du projet. |
| [ListeDLL](../WDLang1/3064006.md) | Retourne :<br><br>- **en exécution**: la liste des librairies (fichiers ".DLL") utilisées par l'application WINDEV, WINDEV Mobile ou par le site WEBDEV en cours. Seules les librairies en mémoire sont listées.<br><br>- **en mode test** : toutes les librairies installées sur le poste en cours.<br><br><br> |
| [NombreFenêtre](../WDLang1/3064005.md) | Calcule le nombre de fenêtres de l'application en cours actuellement ouvertes. |
| [ProjetInfo](../WDLang1/3064004.md) | Renvoie une information spécifique sur le projet en cours d'exécution. |

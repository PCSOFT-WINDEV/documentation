


## &lt;Liste&gt;.Select (Fonction)

***En anglais : &lt;List Box&gt;.Select***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie l'indice de l'élément sélectionné dans un champ Liste, Liste image ou Combo.

**Rappel** : L'élément sélectionné correspond à l'élément en cours sur lequel le bandeau de sélection est affiché.

**Remarque** : La fonction **&lt;Liste&gt;.Select** peut être utilisée indifféremment sur :

- les champs Liste fichier ou Liste par programmation,

- les champs Liste monosélection ou multisélection.













<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Indice de l'élément sélectionné dans le champ "LISTE_ListeClient"
ResIndiceElément = LISTE_ListeClient.Select()
```
<a name="Exemple2"></a>

```wl
// Lignes sélectionnées dans un champ Liste multisélection
i est un entier = 1
IndiceLigneSelectionnee est un entier
IndiceLigneSelectionnee = LISTE_Liste1.Select(1)
TANTQUE IndiceLigneSelectionnee <> -1
	Trace("La ligne n°" + IndiceLigneSelectionnee + " est sélectionnée")
	i++
	IndiceLigneSelectionnee = LISTE_Liste1.Select(i)
FIN

// Équivalent à :
// POUR TOUTE LIGNE SELECTIONNEE DE LISTE_Liste1
//	 Trace("La ligne n°" + LISTE_Liste1 + " est sélectionnée")
// FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Champ Liste>.Select([<Rang>])`
---

**`<Résultat> : (Entier)`**



- Indice de l'élément sélectionné dans le champ Liste (ou Combo) spécifiée, 

- -1 si aucun élément n'est sélectionné. 


Dans le cas d'un champ Liste multisélection, il est nécessaire de préciser le rang de l'élément sélectionné (paramètre &lt;Rang&gt;).

**`<Champ Liste> : (Nom de champ)`**

Nom du champ Liste (ou Combo) à manipuler.

**`<Rang> : (Entier optionnel)`**

Rang de la sélection dans le cas d'un champ Liste multisélection. Si ce paramètre n'est pas spécifié, l'indice du premier élément sélectionné est renvoyé.

Par exemple : Pour récupérer le premier élément sélectionné, le rang sera 1, pour récupérer le deuxième élément sélectionné, le rang sera 2, etc.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Équivalence
<a name="equivalence_ELTPARAGRAPHE000249"></a>Les syntaxes suivantes sont équivalentes :

```txt
Indice est un entier
Indice = <Champ Liste>
```
et

```txt
Indice = <Champ Liste>.Select()
```

<a name="NOTE0_3"></a>


### Divers
<a name="divers_ELTPARAGRAPHE000267"></a>

- Pour connaître l'indice de l'élément sélectionné dans un champ Table, utilisez la fonction [&lt;Table&gt;.Select](../WDLang1/1000024284.md).

- La fonction **&lt;Liste&gt;.Select** peut être utilisée sur une colonne d'un champ Table de type "Combo".

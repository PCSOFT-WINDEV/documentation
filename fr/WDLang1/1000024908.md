


## &lt;Type Chaîne&gt;.Inverse (Fonction)

***En anglais : &lt;String type&gt;.Reverse ***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Inverse les caractères d'une chaîne. 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MaChaîne est une chaîne = "ABC"
Trace(MaChaîne.Inverse())
// La fenêtre de trace affiche "CBA"
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Chaîne>.Inverse()`
---

**`<Résultat> : (Chaîne de caractères Ansi, Unicode ou Binaire)`**

Chaîne de caractère inversée.

**`<Chaîne> : (Chaîne de caractères)`**

Chaîne de caractères à inverser.

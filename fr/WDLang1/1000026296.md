


## &lt;Editeur HTML&gt;.DepuisEmail (Fonction)

***En anglais : &lt;HTML editor&gt;.FromEmail***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Importe le contenu HTML d'un email et ses images dans un champ Editeur HTML. Les images contenues dans l'email sont également affichées dans le champ.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Lit un message et l'affiche dans le champ Editeur HTML pour répondre
MonMessage est un Email
EmailLitMessage(IDSession, MonMessage)
EDHTML_MonEditeurHTML.DepuisEmail(MonMessage, emailOptionRépondreAvant)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Champ Editeur HTML>.DepuisEmail(<Email> [, <Option>])`
---

**`<Champ Editeur HTML> : (Nom de champ)`**

Nom du champ Editeur HTML manipulé.

**`<Email> : (Variable de type Email)`**

Nom de la variable de type [Email](../WDLang3/1000018713.md) à afficher dans le champ Editeur HTML.

**`<Option> : (Constante optionnelle de type Entier)`**

Options d'affichage de l'email : 


|   |   |
| --- | --- |
| *emailOptionRépondreAprès* | Le contenu de l'email est affiché en décalé, la partie réponse se trouve après le contenu. |
| *emailOptionRépondreAvant* | Le contenu de l'email est affiché en décalé, la partie réponse se trouve avant le contenu. |
| *emailOptionTransféré* | Ajoute un entête de réponse au début de l'email. |


Si ce paramètre n'est pas précisé ou vaut 0, l'email affiché dans le champ est directement modifiable.

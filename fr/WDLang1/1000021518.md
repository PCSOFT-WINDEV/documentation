


## TableurInsèreColonne (Fonction)

***En anglais : SpreadsheetInsertColumn***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Insère une colonne vide dans un champ Tableur présent dans une fenêtre. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Insère plusieurs colonnes
POUR I = 2 A 5
	TableurInsèreColonne(TBLR_Tableur, I)
 FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`TableurInsèreColonne(<Champ Tableur> [, <Colonne>])`
---

**`<Champ Tableur> : (Nom de champ)`**

Nom du champ Tableur à manipuler. 

**`<Colonne> : (Entier optionnel)`**

Numéro de la colonne avant laquelle la nouvelle colonne sera insérée. Par exemple, si ce paramètre vaut 2, la nouvelle colonne sera insérée à la position 2. Les colonnes existantes seront décalées vers la droite et les formules seront mises à jour. 
Si ce paramètre n'est pas précisé et si une colonne est sélectionnée dans le champ, l'insertion sera réalisée avant la colonne sélectionnée. Dans le cas contraire (aucune colonne sélectionnée), la fonction ne fait rien. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- L'opération d'insertion est ajoutée dans la liste des opérations à annuler : l'utilisateur peut annuler cette opération via le menu contextuel ou via la combinaison de touches Ctrl + Z.

- Cette fonction est utilisable uniquement sur un champ Tableur présent dans une fenêtre.

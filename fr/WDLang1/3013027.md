
## NumHelp (Fonction)

***En anglais : NumHelp***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le numéro de contexte d'aide associé au champ spécifié. Ce numéro d'aide a été précisé lors de la création du champ sous l'éditeur de fenêtre.

**Remarque** : Ce numéro d'aide pourra être utilisé avec la fonction [WHelp](../WDLang1/3013053.md) pour afficher la page d'aide (format HLP ou CHM) correspondant au champ.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ouverture de l'aide MonAide.HLP sur la page
// ayant pour numéro de contexte NumHelp(NomChampEnCours)
// NomChampEnCours est une variable contenant le nom du champ en cours
WHelp("MonAide.hlp", NumHelp(NomChampEnCours))
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = NumHelp(<Nom du champ>)`
---

**`<Résultat> : (Entier)`**

Numéro de l'aide contextuelle.

**`<Nom du champ> : (Chaîne de caractères)`**

Nom du champ dont on recherche le numéro d'aide contextuelle. Pour les options d'interrupteur ou de sélecteur, indiquez le numéro de l'option entre crochet (par exemple, Interrupteur[1] pour la première option d'un interrupteur).




## GanttPositionDateHeure (Fonction)

***En anglais : GanttDateTimePosition***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Paramètre le premier jour et la première heure affichés dans une colonne (d'un champ Table ou Table hiérarchique) de type Diagramme de Gantt.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
GanttPositionDateHeure(COL_Gantt, DateSys())
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`GanttPositionDateHeure(<Colonne Gantt> , <Date de début>)`
---

**`<Colonne Gantt> : (Nom de champ)`**

Nom de la colonne (d'un champ Table ou Table hiérarchique) de type Diagramme de Gantt à manipuler.

**`<Date de début> : (Chaîne de caractères ou variable de type Date/DateHeure)`**

Date de début de l'affichage du Diagramme de Gantt. Ce paramètre peut être : 

- une date au format "AAAAMMJJ",

- une date et une heure au format "AAAAMMJJHHmmSSCC".






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Dans un état, la fonction **GanttPositionDateHeure** devra être utilisée dans le code d'initialisation de la colonne de type Gantt.

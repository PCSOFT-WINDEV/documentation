


## &lt;Interrupteur&gt;.Supprime (Fonction)

***En anglais : &lt;Check box&gt;.Delete***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime une option d'un champ Interrupteur. 




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Supprime l'option numéro 3 d'un champ Interrupteur
INT_Options.Supprime(3)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Champ Interrupteur>.Supprime(<Indice>)`
---

**`<Champ Interrupteur> : (Nom de champ)`**

Nom du champ Interrupteur dans lequel l'option doit être supprimée.

**`<Indice> : (Entier)`**

Indice de l‘option à supprimer. Cet indice doit être compris entre 1 et le nombre d'options contenues dans le champ Interrupteur. La propriété [Occurrence](../Proprietes/2510097.md) permet de connaître le nombre d'options d'un champ Interrupteur.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Lors de la suppression d'options dans un champ Interrupteur, la taille du champ n'est pas modifiée. Il est possible de modifier la taille du champ avec les propriétés [Largeur](../Proprietes/2510059.md) et [Hauteur](../Proprietes/2510050.md).

- Il est possible de compacter les options pour optimiser l'aspect du champ après suppression des options. 
	En mode compacté, les options d'un champ Interrupteur se resserrent : 

	- lorsqu'une option est rendue invisible. 

	- lorsqu'une option est supprimée.




- Pour utiliser le mode compacté : 

	- cochez l'option "Compacter les options invisibles" dans l'onglet "Détail" de la description du champ.

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) utilisez la propriété [CompacteOption](../Proprietes/1000020608.md).




- Lors de la suppression d'une option, les indices des options suivant l'option supprimée sont automatiquement renumérotés.

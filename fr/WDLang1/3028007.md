
## DDEItem (Fonction)

***En anglais : DDEItem***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Identifie l'item concerné par un événement DDE.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Équivalent à : Item = _DDE.Item
Item = DDEItem()
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = DDEItem([<Identifiant connexion>])`
---

**`<Résultat> : (Chaîne de caractères)`**

Item concerné par l'événement DDE.

**`<Identifiant connexion> : (Entier optionnel)`**

Identifiant de la connexion renvoyé par la fonction [DDEConnecte](../WDLang1/3028015.md). Si ce paramètre n'est pas précisé, **DDEItem** utilise la dernière connexion manipulée.

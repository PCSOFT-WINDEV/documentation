


## Fonctions de gestion des fenêtres
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Voici la liste des principales fonctions permettant de gérer les fenêtres WINDEV et WINDEV Mobile par programmation.

Pour plus de détails :

- Sur la programmation des menus, consultez [Fonctions de gestion des menus](../WDLang1/3038016.md)

- Sur la programmation des fenêtres MDI, consultez [Fonctions de programmation des fenêtres MDI](../WDLang1/3052003.md)









|   |   |
| --- | --- |
| [Abandonne](../WDLang1/1000017024.md) | Ferme la fenêtre en cours en exécutant l'événement "Clic" du champ Bouton de type ABANDON. |
| [ChangeGabarit](../WDLang1/3013026.md) | Change dynamiquement le gabarit : <br><br>- de toutes les fenêtres de l'application en cours, <br><br>- des fenêtres présentes dans les composants WINDEV utilisés par l'application,<br><br>- d'une fenêtre spécifique du projet.<br><br><br> |
| [DélaiAvantFermeture](../WDLang1/3038040.md) | Limite le temps d'affichage d'une fenêtre ou d'une boîte de dialogue. <br>Si aucune action de l'utilisateur n'est effectuée, lorsque le temps spécifié est écoulé : <br><br>- dans le cas d'une fenêtre, le code d'un bouton de la fenêtre sera exécuté. <br><br>- dans le cas d'une boîte de dialogue, la boîte de dialogue est automatiquement fermée.<br><br><br> |
| [DemandeMiseAJourIHM](../WDLang1/1000021378.md) | Demande l'exécution de l'événement "Demande de mise à jour de l'affichage" (et rend la main **sans attendre l'exécution de l'événement**). Cette fonction est conservée par compatibilité. |
| [DemandeMiseAJourIHMParent](../WDLang1/1000022961.md) | Demande l'exécution du traitement "Demande de mise à jour de l'affichage" du parent. **Cette fonction est conservée par compatibilité.** |
| [DemandeMiseAJourUI](../WDLang1/1000023899.md) | Demande l'exécution de l'événement "Demande de mise à jour de l'affichage" (et rend la main **sans attendre l'exécution de l'événement**). |
| [DemandeMiseAJourUIÉlément](../WDLang1/1410088703.md) | Demande l'exécution de l'événement "Demande de mise à jour de l'affichage" de l'élément spécifié (et rend la main **sans attendre l'exécution de l'événement**). |
| [DemandeMiseAJourUIParent](../WDLang1/1000023900.md) | Demande l'exécution du traitement "Demande de mise à jour de l'affichage" du parent. |

| [ExécuteMiseAJourIHM](../WDLang1/1000021379.md) | Exécute immédiatement le traitement "Demande de mise à jour de l'affichage" de la fenêtre, de la page, de la fenêtre interne, de la page interne, du modèle de champs ou du superchamp. **Cette fonction est conservée par compatibilité.** |
| [ExécuteMiseAJourIHMParent](../WDLang1/1000022962.md) | Exécute immédiatement l'événement "Demande de mise à jour de l'affichage" du parent. **Cette fonction est conservée par compatibilité.** |
| [ExécuteMiseAJourUI](../WDLang1/1000023901.md) | Exécute immédiatement l'événement "Demande de mise à jour de l'affichage" de la fenêtre, de la page, de la fenêtre interne, de la page interne, du modèle de champs ou du superchamp. |
| [ExécuteMiseAJourUIÉlément](../WDLang1/1410088704.md) | Exécute immédiatement l'événement "Demande de mise à jour de l'affichage" de l'élément spécifié. |
| [ExécuteMiseAJourUIParent](../WDLang1/1000023902.md) | Exécute immédiatement l'événement "Demande de mise à jour de l'affichage" du parent. |
| [FenActiveGFI](../WDLang1/3038056.md) | Active ou désactive le GFI (Grisage des Fenêtres Inaccessibles) :<br><br>- soit d'une fenêtre spécifique.<br><br>- soit de toutes les fenêtres de l'application.<br><br><br> |

| [FenAdapteLargeur](../WDLang1/1000019920.md) | Adapte la largeur de la fenêtre au contenu des champs. |
| [FenAdapteTaille](../WDLang1/3038049.md) | Adapte la taille de la fenêtre au contenu des champs. |
| [FenAgencementEnCours](../WDLang1/1000021133.md) | Renvoie l'indice de l'agencement en cours : <br><br>- dans la fenêtre en cours.<br><br>- dans la fenêtre ou dans la fenêtre interne spécifiée.<br><br><br> |
| [FenAttenteDébut](../WDLang1/1000023416.md) | Ouvre la fenêtre d'attente de l'application. |

| [FenChangeAlias](../WDLang1/3038028.md) | Modifie l'alias d'une fenêtre. |
| [FenColleFormulaire](../WDLang1/1000018993.md) | Colle le contenu d'un formulaire stocké dans le presse-papiers. |
| [FenCopieFormulaire](../WDLang1/1000018992.md) | Copie le contenu d'un formulaire dans le presse-papiers. |
| [FenCoulissanteVisible](../WDLang1/1000020978.md) | Affiche ou masque une fenêtre coulissante (sliding menu). |
| [FenDésactiveEffet](../WDLang1/1000017172.md) | Désactive des effets visuels du moteur graphique de WINDEV (pour une utilisation sous TSE par exemple). |
| [FenDonneHauteurAdaptée](../WDLang1/1000019888.md) | Renvoie la hauteur d'une fenêtre, adaptée au contenu des champs. |
| [FenDonneLargeurAdaptée](../WDLang1/1000019889.md) | Renvoie la largeur d'une fenêtre, adaptée au contenu des champs. |
| [FenEffetInactif](../WDLang1/3038055.md) | Active ou désactive le GFI (Grisage des Fenêtres Inaccessibles) :<br><br>- soit d'une fenêtre spécifique.<br><br>- soit de toutes les fenêtres de l'application.<br><br><br> |
| [FenEnCours](../WDLang1/3038033.md) | Identifie la fenêtre en cours de saisie. |
| [FenEnExécution](../WDLang1/3038034.md) | Identifie la fenêtre contenant le code du WLangage en cours d'exécution. |
| [FenEtat](../WDLang1/3038030.md) | Identifie ou modifie l'état d'une fenêtre. |
| [FenExtHaut](../WDLang1/3038032.md) | Renvoie la hauteur totale d'une fenêtre (cadre, barre de menu et titre compris). |
| [FenExtLarg](../WDLang1/3038029.md) | Renvoie la largeur totale d'une fenêtre (cadre compris). |

| [FenRafraîchissementVisible](../WDLang1/1000023419.md) | Modifie ou retourne l'état de visibilité du bandeau de rafraîchissement d'une fenêtre. |




## EditeurHTMLSélectionne (Fonction)

***En anglais : HTMLEditorSelect***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Sélectionne une portion de texte dans un champ Editeur HTML. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Passe les 5 premiers caractères en gras
EditeurHTMLSelectionne(EDHTML_MonEditeurHTML, 1, 5)
EditeurHTMLFormateSélection(EDHTML_MonEditeurHTML, htmlGras)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`EditeurHTMLSélectionne(<Champ Editeur HTML> , <Début de la sélection> , <Fin de la sélection>)`
---

**`<Champ Editeur HTML> : (Nom de champ)`**

Nom du champ Editeur HTML manipulé.

**`<Début de la sélection> : (Entier)`**

Premier caractère à sélectionner dans le champ Editeur HTML. 

**`<Fin de la sélection> : (Entier)`**

Dernier caractère de la sélection. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- La sélection peut être récupérée par la fonction [EditeurHTMLRécupèreSélection](../WDLang1/1000025992.md). 

- Le format de la sélection peut ensuite être modifié par la fonction [EditeurHTMLFormateSélection](../WDLang1/1000025996.md).

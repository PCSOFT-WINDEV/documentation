
## GesteTypePointeur (Fonction)

***En anglais : GesturePointerType***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le type de pointeur utilisé sur l'appareil : doigt, souris ou stylet.

Remarque : Cette fonction est disponible sous WINDEV uniquement pour être disponible dans le simulateur. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
nType est un entier = GesteTypePointeur()
SI nType = typePointeurDoigt ALORS Info("Le pointeur utilisé est le doigt.")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = GesteTypePointeur([<Pointeur>])`
---

**`<Résultat> : (Entier)`**



- Type de pointeur utilisé, 

- -1 en cas d'erreur. 




Le type de pointeur peut correspondre à l'une des constantes suivantes :



|   |   |
| --- | --- |
| *typePointeurDoigt* | Le doigt est utilisé comme pointeur. |
| *typePointeurSouris* | La souris est utilisée comme pointeur. |
| *typePointeurStylet* | Le stylet est utilisé comme pointeur. |



**`<Pointeur> : (Entier optionnel)`**

Indice du pointeur à connaître. Le premier pointeur a pour indice 1. 

- Si ce paramètre n'est pas spécifié, la fonction retournera la position du premier pointeur. 

- Si l'indice ne correspond pas un pointeur valide, la fonction retournera -1.








<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- La fonction **GesteTypePointeur** doit être utilisée dans les événements liés aux mouvements du pointeur : clic, déplacement, pointeur enfoncé, pointeur relevé, scroll, etc.  Si la fonction **GesteTypePointeur** est appelée dans un autre événement ou traitement, &lt;Résultat&gt; correspondra à la valeur '-1'.

- Pour connaître l'indice du pointeur ayant déclenché l'exécution d'un événement lié à un mouvement sur l'écran, utilisez la fonction [GestePointeurEncours](../WDLang1/1000019751.md).

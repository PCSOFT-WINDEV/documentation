
## PressePapierRTF (Fonction)

***En anglais : ClipboardRTF***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Récupère une chaîne RTF dans le presse-papiers (le contenu du presse-papiers est au format RTF).

Remarques :

- Pour identifier le contenu du presse-papiers, utilisez la fonction [PressePapierFormat](../WDLang1/3063002.md).

- Pour récupérer du texte ou une image, utilisez la fonction [PressePapier](../WDLang1/3063004.md).  



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Récupère le contenu du presse-papiers au format RTF
sRTF est une chaîne = PressePapierRTF()
SI sRTF = "" ALORS
	Erreur()
SINON
	// Colle le texte RTF dans le champ SAI_RTF
	SAI_RTF = sRTF
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = PressePapierRTF()`
---

**`<Résultat> : (Chaîne de caractères)`**



- Chaîne de caractères au format RTF. La chaîne de caractères récupérée peut être affectée à un champ RTF.  

- Chaîne vide ("") en cas d'erreur.

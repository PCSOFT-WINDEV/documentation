
## ChaîneVersJSON (Fonction)

***En anglais : StringToJSON***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Convertit une chaîne au format JSON en une variable de type JSON.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
sJZon est une chaîne ANSI = [
{
"glossary": {
"title": "example glossary",
"GlossDiv": {
"title": "S",
"GlossList": {
"GlossEntry": {
"ID": "SGML",
"SortAs": "SGML",
"GlossTerm": "Standard Generalized Markup Language",
"Acronym": "SGML",
"Abbrev": "ISO 8879:1986",
"GlossDef": {
"para": "A meta-markup language, used to create markup languages such as DocBook.",
"GlossSeeAlso": ["GML", "XML"]
},
"GlossSee": "markup"
}
}
}
}
}
]

jz est un JSON = ChaîneVersJSON(sJZon)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = ChaîneVersJSON(<Chaîne au format JSON>)`
---

**`<Résultat> : (Variable de type JSON)`**

Variable de type [JSON](../Motscles/1000023611.md) contenant le résultat de la conversion.
En cas d'échec, la variable [ErreurDétectée](../WDLang1/3087001.md) vaut <u><u><u><u>Vrai</u></u></u></u> et la fonction [ErreurInfo](../WDLang1/3013008.md) permet de connaître le détail de l'erreur.

**`<Chaîne au format JSON> : (Chaîne de caractères)`**

Chaîne de caractères contenant les données au format JSON.

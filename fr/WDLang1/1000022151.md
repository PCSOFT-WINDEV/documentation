
## PleinEcran (Fonction)

***En anglais : FullScreen***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Active ou désactive le mode plein écran dans les applications en mode Universal Windows 10 App. 

Le mode plein écran consiste à masquer : 

- la barre système quand l'application est lancée sur un PC,

- la barre système et la barre de navigation sur un mobile. 


Ces barres peuvent être ré-affichées par un simple survol de la zone. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
PleinEcran(Vrai)
PleinEcran(Faux)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`PleinEcran([<Activation>])`
---

**`<Activation> : (Booléen optionnel)`**



- <u><u><u><u>Vrai</u></u></u></u> pour activer le mode plein écran, 

- <u><u><u><u>Faux</u></u></u></u> pour désactiver le mode plein écran.

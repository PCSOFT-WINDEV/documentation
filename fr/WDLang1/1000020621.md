


## Fonctions de gestion des champs TimeLine
			



Les fonctions de gestion des champs TimeLine sont les suivantes :



|   |   |
| --- | --- |
| [TimelineAffiche](../WDLang1/1000020683.md) | Réinitialise le contenu d'un champ TimeLine à partir de sa source de données (fichier de données HFSQL par exemple). |
| [TimelineAjouteEvénement](../WDLang1/1000020400.md) | Ajoute un événement dans un champ TimeLine. |
| [TimelineAjoutePiste](../WDLang1/1000020405.md) | Ajoute une nouvelle piste dans un champ TimeLine. |
| [TimelineChangeMode](../WDLang1/1000020397.md) | Modifie la période (c'est-à-dire le nombre de secondes, millisecondes, microsecondes, ...) affichée par un champ TimeLine. |
| [TimelineListeEvénement](../WDLang1/1000020401.md) | Renvoie : <br><br>- la liste des événements compris entre deux moments donnés dans un champ TimeLine, <br><br>- tous les événements d'un champ TimeLine,<br><br>- un événement particulier (sélectionné ou survolé). <br><br><br> |
| [TimelineSupprimeEvénement](../WDLang1/1000020403.md) | Supprime un événement dans un champ TimeLine. |
| [TimelineSupprimePiste](../WDLang1/1000020760.md) | Supprime une piste dans un champ TimeLine. |
| [TimelineSupprimeTout](../WDLang1/1000020404.md) | Supprime tous les événements et toutes les pistes d'un champ TimeLine. |

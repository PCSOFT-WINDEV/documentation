


## Fonctions de gestion de la télémétrie
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


Les fonctions de gestion de la télémétrie sont les suivantes : 



|   |   |
| --- | --- |
| [TélémétrieActive](../WDLang1/1000021900.md) | Active manuellement la télémétrie. |
| [TélémétrieAjouteMesure](../WDLang1/1000022168.md) | Ajoute une mesure pour un capteur de valeur ou de temps. |
| [TélémétrieAjoutePassage](../WDLang1/1000022174.md) | Signale l'exécution du code au niveau d'un capteur de passage. |
| [TélémétrieDésactive](../WDLang1/1000021912.md) | Désactive la télémétrie. Les données de télémétrie ne sont plus envoyées au serveur. |
| [TélémétrieEnvoieDonnées](../WDLang1/1000024736.md) | Force l'envoi immédiat des données de la télémétrie. |
| [TélémétrieEstActive](../WDLang1/1000021911.md) | Permet de savoir si la télémétrie est activée. |
| [TélémétrieIdentifieUtilisateur](../WDLang1/1000022160.md) | Identifie l'utilisateur actuel pour lequel les données de télémétrie sont enregistrées. |
| [TélémétrieParamètre](../WDLang1/1000021913.md) | Permet de connaître et de modifier les paramètres de la télémétrie. |

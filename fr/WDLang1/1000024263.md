


## &lt;Traitement de texte&gt;.EntreEdition (Fonction)

***En anglais : &lt;Word Processing&gt;.GoIntoEdit***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Entre en édition dans le champ Traitement de texte dans une partie spécifique du document : corps, en-tête ou pied de page.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
TT_ChampMonDoc.EntreEdition(docEnTête)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Champ Traitement de texte>.EntreEdition(<Partie>)`
---

**`<Champ Traitement de texte> : (Nom de champ)`**

Nom du champ Traitement de texte à manipuler.

**`<Partie> : (Constante de type Entier)`**

Partie du document dans laquelle il faut entrer en édition : 



|   |   |
| --- | --- |
| *docCorps* | Entre en édition dans le corps du document pour la page en cours. |
| *docEnTête* | Entre en édition dans l'en-tête de page de la page en cours. |
| *docPiedDePage* | Entre en édition dans le pied de page de la page en cours. |

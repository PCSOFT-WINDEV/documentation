


## HTMLValideFormulaire (Fonction)

***En anglais : HTMLValidateForm***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Valide le formulaire d'une page HTML affichée dans un champ d'affichage HTML. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
HTMLValeurChamp(HTM_Affichage, "LOGIN", sLogin)
HTMLValeurChamp(HTM_Affichage, "PASSWD", sMotDePasse)
HTMLValideFormulaire(HTM_Affichage)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`HTMLValideFormulaire(<Champ d'affichage HTML> [, <Formulaire>])`
---

**`<Champ d'affichage HTML> : (Nom de champ)`**

Nom du champ d'affichage HTML manipulé. 

**`<Formulaire> : (Chaîne de caractères optionnelle)`**

Nom du formulaire à valider. Si ce paramètre n'est pas précisé, le premier formulaire de la page affichée dans le champ est validé.




## &lt;Kanban&gt;.Enregistre (Fonction)

***En anglais : &lt;Kanban&gt;.Save***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie l'enregistrement associé à la carte en cours dans le champ Kanban fichier. L'enregistrement est mis à jour : 

- dans le fichier de données lié au champ Kanban. 

- dans la variable liée au champ.



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Modification du titre et du texte de la première carte de la liste
// puis enregistrement dans la base de données
Carte_1 est kbCarte <- KANBAN_Liste.LISTE_TODO[1]
Carte_1.Titre = SAI_TITRE
Carte_1.Texte = SAI_TEXTE
KANBAN_Liste.Enregistre(Carte_1)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Champ Kanban>.Enregistre(<Carte>)`
---

**`<Champ Kanban> : (Nom de champ)`**

Champ Kanban à manipuler. 
 Si le champ Kanban n'est pas lié a une source de données, la fonction échoue avec une erreur fatale.

**`<Carte> : (Variable de type kbCarte)`**

Nom de la variable de type [kbCarte](../WDLang1/1410089167.md) correspondant à la carte à modifier dans l'enregistrement lié au champ Kanban.
Si la carte spécifiée n'est pas présente dans le champ kanban, la fonction échoue avec un erreur fatale.

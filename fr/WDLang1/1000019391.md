
## SansCaractère (Fonction)

***En anglais : NoCharacter***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie une chaîne de caractères sans les caractères spécifiés : 

- situés à gauche et à droite de la chaîne initiale.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) situés à l'intérieur de la chaîne initiale.







<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MaChaîne est une chaîne
MaChaîne = SansCaractère("<Texte>", "<>") 
// MaChaîne contient "Texte"
```


<a name="Exemple2"></a>

```wl
SansCaractère("Abracadabra", "a", sscIntérieur)   // Renvoie "Abrcdbra"
SansCaractère("Abracadabra", "a", sscTout)   // Renvoie "brcdbr"
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = SansCaractère(<Chaîne initiale> , <Caractères à supprimer> [, <Position>])`
---

**`<Résultat> : (Chaîne de caractères)`**

Chaîne de caractères sans les caractères spécifiés situés à gauche et à droite de la chaîne initiale. 

**`<Chaîne initiale> : (Chaîne de caractères)`**

Chaîne de caractères initiale. 

**`<Caractères à supprimer> : (Chaîne de caractères)`**

Liste des caractères à supprimer à gauche et à droite de la chaîne initiale. 

Si ce paramètre correspond à une chaîne vide (""), &lt;Résultat&gt; correspondra à la chaîne initiale. 

**`<Position> : (Constante optionnelle de type Entier)`**

Position des caractères à supprimer. Cette position peut correspondre à une des constantes suivantes : 


|   |   |
| --- | --- |
| *sscDroite* | Supprime les caractères spécifiés situés à droite de la chaîne initiale. |
| *sscExtérieur*<br>(Valeur par défaut) | Supprime les caractères spécifiés situés à l'extérieur de la chaîne initiale. <br>Equivalent à *sscDroite* + *sscGauche*. |
| *sscGauche* | Supprime les caractères spécifiés situés à gauche de la chaîne initiale. |
| *sscIntérieur* | Supprime les caractères spécifiés présents à l'intérieur de la chaîne initiale. |
| *sscTout* | Supprime tous les caractères spécifiés présents dans la chaîne initiale. <br>Equivalent à *sscDroite* + *sscGauche* + *sscIntérieur*. |





![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Ce paramètre n'est pas disponible.

## Remarques




### Fonction SansCaractère et UNICODE
<a name="fonction_docparampagetitleshort_unicode_ELTPARAGRAPHE000128"></a>Si la chaîne passée en paramètre est au format ANSI, le résultat sera au format ANSI.

Si la chaîne passée en paramètre est au format UNICODE, le résultat sera au format UNICODE.

**Remarque** : Si le résultat de la fonction **SansCaractère** utilisée sur une chaîne ANSI est affecté à une chaîne UNICODE (et inversement), la conversion ANSI/UNICODE sera implicitement réalisée.

Pour plus de détails, consultez [Gestion de l'UNICODE](../WDLang1/3024038.md).


## FinTauxIntérêt (Fonction)

***En anglais : FinInterestRate***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Évalue le taux d'intérêt pour un emprunt donné sur une période avec des versements constants.

**Remarque** : Le taux d'intérêt obtenu est supposé constant sur la période.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple
<a class="notetitle" target="_blank" href="$DOC$=1000003050047&name=fintauxinteret_fonction&product=WD">Voir des exemples supplémentaires</a>

```wl
// Taux d'intérêt pour un emprunt de 25 000 Euros. Six versements
// de 5 000 Euros sont nécessaires au remboursement de l'emprunt.
ResTauxIntérêt = FinTauxIntérêt(6, -5000, 25000)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = FinTauxIntérêt(<Nombre de remboursements nécessaires> [, <Valeur d'un remboursement> [, <Capital>]])`
---

**`<Résultat> : (Réel ou monétaire)`**

Taux d'intérêt recherché.

**`<Nombre de remboursements nécessaires> : (Réel ou monétaire)`**

Nombre total de remboursements à effectuer.

**`<Valeur d'un remboursement> : (Réel optionnel)`**

Valeur d'un remboursement (0 par défaut). Cette valeur doit être négative.

**`<Capital> : (Réel ou monétaire optionnel)`**

Montant de l'emprunt (1 par défaut).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
**Attention** : La fonction **FinTauxIntérêt** ne retourne pas de code d'erreur. Pour savoir si cette fonction a généré des erreurs, utilisez la fonction [FinErreur](../WDLang1/3050071.md). Le libellé de l'erreur peut être connu grâce à la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.

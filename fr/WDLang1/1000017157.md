
## CorbeilleVide (Fonction)

***En anglais : RecycleBinClear***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Vide la corbeille du système.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
CorbeilleVide()
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = CorbeilleVide()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la corbeille a été vidée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.

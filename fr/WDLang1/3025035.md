
## ChampExiste (Fonction)

***En anglais : ControlExist***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Teste l'existence d'un champ (ou d'un groupe de champs) dans une fenêtre, dans une page ou dans un état.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Test de l'existence du champ "SaisieNom"
ResExiste = ChampExiste("SaisieNom")
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = ChampExiste(<Nom du champ>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le champ (ou le groupe de champs) existe, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Nom du champ> : (Chaîne de caractères)`**

Nom du champ à tester **entre guillemets**. Ce paramètre peut aussi correspondre au nom d'un groupe de champs. 
Pour tester l'existence des champs des autres fenêtres, pages ou états, &lt;Nom du champ&gt; est de la forme :
"&lt;Nom de la fenêtre, de la page ou de l'état&gt;.&lt;Nom du champ&gt;".


## UTF8VersUnicode (Fonction)

***En anglais : UTF8ToUnicode***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Convertit une chaîne au format UTF8 en une chaîne Unicode. 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
sChaîne est une chaîne = "Antananarivo"
// Conversion en chaîne Unicode
sChaîneUnicode est une chaîne UNICODE = UTF8VersUnicode(sChaîne)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = UTF8VersUnicode(<Chaîne UTF8 à convertir>)`
---

**`<Résultat> : (Chaîne de caractères)`**



- Chaîne de caractères au format Unicode.

- EOT en cas de problème de conversion. 




**`<Chaîne UTF8 à convertir> : (Chaîne de caractères)`**

Chaîne de caractères à convertir. Cette chaîne de caractères n'est pas modifiée.


## PlanificateurAjouteHoraire (Fonction)

***En anglais : SchedulerAddTrigger***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Crée un horaire pour une tâche planifiée. Un horaire est un des événements qui vont déclencher l'exécution d'une tâche. Avant de créer l'horaire, il est nécessaire de définir tous les paramètres de cet horaire dans la [structure HoraireTâchePlanifiée](../WDLang1/3089002.md).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// La tâche est quotidienne, exécutée tous les jours à 10H
// à partir d'aujourd'hui
HoraireTâchePlanifiée.Type = htQuotidien
HoraireTâchePlanifiée.Heure = 1000
HoraireTâchePlanifiée.DateDébut = DateDuJour()
PlanificateurAjouteHoraire("MaTache")
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = PlanificateurAjouteHoraire(<Nom de la tâche>)`
---

**`<Résultat> : (Entier)`**



- Numéro de l'horaire qui vient d'être créé. Cet identifiant pourra être utilisé dans les fonctions permettant de manipuler des horaires.

- 0 en cas d'erreur. 




**`<Nom de la tâche> : (Chaîne de caractères)`**

Nom de la tâche à manipuler. Cette tâche a été créée grâce à la fonction [PlanificateurAjouteTâche](../WDLang1/3089003.md).

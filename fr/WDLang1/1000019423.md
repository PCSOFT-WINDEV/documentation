
## ThreadCourant (Fonction)

***En anglais : ThreadCurrent***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le nom du thread en cours d'exécution.




<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = ThreadCourant()`
---

**`<Résultat> : (Chaîne de caractères)`**



- Nom du thread en cours d'exécution, 

- "" (chaîne vide) si le thread courant est le thread principal.

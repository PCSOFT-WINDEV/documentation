
## PoliceSelect (Fonction)

***En anglais : FontSelect***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ouvre la fenêtre standard de sélection de polices de Windows en sélectionnant la police spécifiée.

La police sélectionnée par l'utilisateur pourra ensuite être utilisée :

- pour écrire dans des dessins (fonction [dPolice](../WDLang1/3029020.md)).

- pour écrire dans les graphes (fonctions [grPoliceEtiquette](../WDLang3/3042012.md), [grPoliceLégende](../WDLang3/3042037.md), [grPoliceTitre](../WDLang3/3042048.md), [grPoliceTitreAxe](../WDLang3/3042059.md)).

- pour imprimer (fonction [iPolice](../WDLang5/3046014.md)).

- dans les champs d'une fenêtre ou d'un état (propriété [Police](../Proprietes/2515045.md)).



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Création d'une police par défaut
PDéfaut est une police = PoliceCrée("Arial", 12)
// Utilisation d'une variable pour stocker la police de l'utilisateur
PUtilisateur est une police
// Demande à l'utilisateur de modifier sa police
PoliceSelect(PUtilisateur, Vrai, PDéfaut)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Ouvrir le sélecteur de polices en spécifiant une police par défaut

`<Résultat> = PoliceSelect(<Police sélectionnée> [, <Option> [, <Police par défaut>]])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'utilisateur a sélectionné une police, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire (l'utilisateur a utilisé le bouton "Annuler").




**`<Police sélectionnée> : (Police)`**

Nom de la variable de type [Police](../Motscles/1514045.md) qui contiendra les caractéristiques de la police sélectionnée par l'utilisateur.

**`<Option> : (Booléen optionnel)`**



- <u><u><u><u>Vrai</u></u></u></u> pour afficher les options avancées de la police (soulignement, couleur), 

- <u><u><u><u>Faux</u></u></u></u> pour ne par afficher ces options dans le sélecteur de polices.




**`<Police par défaut> : (Variable de type Police optionnelle)`**

Nom de la variable de [type Police](../Motscles/1514045.md) qui correspond aux caractéristiques de police à sélectionner par défaut dans le sélecteur de polices de Windows.


<a name="SYNTAXE2"></a>

### Ouvrir le sélecteur de polices en donnant toutes les caractéristiques de la police par défaut

`<Résultat> = PoliceSelect(<Police sélectionnée> [, <Option> [, <Nom de la police> [, <Taille> [, <Style> [, <Couleur>]]]]])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'utilisateur a sélectionné une police, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire (l'utilisateur a utilisé le bouton ANNULER).




**`<Police sélectionnée> : (Police)`**

Nom de la variable de type [Police](../Motscles/1514045.md) qui contiendra les caractéristiques de la police sélectionnée par l'utilisateur. Si seul ce paramètre est spécifié, la police par défaut sélectionnée dans le sélecteur de polices aura les caractéristiques de cette police.

**`<Option> : (Booléen optionnel)`**



- <u><u><u><u>Vrai</u></u></u></u> pour afficher les options avancées de la police (soulignement, couleur), 

- <u><u><u><u>Faux</u></u></u></u> pour ne par afficher ces options dans le sélecteur de polices.




**`<Nom de la police> : (Chaîne de caractères optionnelle)`**

Nom de la police sélectionnée par défaut dans le sélecteur de polices. Ce nom doit correspondre à une des polices installées sur le poste en cours.

**`<Taille> : (Entier optionnel optionnelle)`**

Taille de la police sélectionnée par défaut dans le sélecteur de police. Cette taille est exprimée en points pica (1 point Pica = 0,353 mm).

**`<Style> : (Constante ou combinaison de constantes optionnelle de type entier)`**

Attributs de la police sélectionnés par défaut :



|   |   |
| --- | --- |
| *iBarre* | Police barrée |
| *iGras* | Police en gras |
| *iItalique* | Police italique |
| *iNormal* | Police normale, sans attributs |
| *iSouligné* | Police soulignée |



**`<Couleur> : (Entier optionnel)`**

Couleur de la police sélectionnée par défaut. Cette couleur peut correspondre à :

- une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)).

- une couleur TSL (obtenue avec la fonction [TSL](../WDLang1/3029057.md)),

- une [couleur prédéfinie dans le WLangage](../WDLang5/3010002.md).

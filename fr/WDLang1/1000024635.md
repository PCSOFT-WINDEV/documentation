


## PicRedimensionne (Fonction)

***En anglais : PicResize***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Redimensionne une image au format WDPic. Tous les calques de l'image sont redimensionnés. 

Remarque : Pour redimensionner un seul calque, utilisez la fonction [dRedimensionne](../WDLang1/3029011.md). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ouverture d'une image source
MonImageWDPic est un WDPic
MonImageWDPic.NomFichier = fRepExe() + ["\"] + "Source.png"

// Redimensionnement de l'image
PicRedimensionne(MonImageWDPic, 100, 100, drHomothétique)

// Sauvegarde du résultat
PicSauve(MonImageWDPic, fRepExe() + ["\"] + "Resultat.png")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = PicRedimensionne(<Image WDPic> , <Nouvelle largeur> , <Nouvelle hauteur> [, <Option>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le redimensionnement a été effectué, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Image WDPic> : (Variable de type WDPic)`**

Nom de la variable de type [WDPic](../WDLang1/1000023503.md) contenant l'image à manipuler. 

**`<Nouvelle largeur> : (Entier)`**

Nouvelle largeur de l'image exprimée en pixels.

**`<Nouvelle hauteur> : (Entier)`**

Nouvelle hauteur de l'image exprimée en pixels.

**`<Option> : (Constante de type Entier optionnelle)`**

Options de redimensionnement : 


|   |   |
| --- | --- |
| *drEtiré*<br>(Valeur par défaut) | L'image est étirée si nécessaire pour correspondre à la nouvelle largeur et à la nouvelle hauteur spécifiées. |
| *drHauteQualité* | Cette constante peut être combinée avec les options de redimensionnement de type homothétique.<br><br>En mode homothétique, cette constante permet d'améliorer la qualité de l'image lorsque celle-ci est réduite par rapport à sa taille initiale. |
| *drHomothétique* | L'image est étirée ou réduite pour occuper de manière optimale l'espace défini sans être déformée. Elle occupe :<br><br>- soit toute la hauteur spécifiée.<br><br>- soit toute la largeur spécifiée.<br><br><br> |
| *drHomothétiqueCentré* | L'image est étirée ou réduite pour occuper de manière optimale l'espace défini sans être déformée et elle est centrée. Elle occupe :<br><br>- soit toute la hauteur spécifiée : l'image est centrée horizontalement.<br><br>- soit toute la largeur spécifiée : l'image est centrée verticalement.<br><br><br> |
| *drHomothétiqueEtendu* | L'image est étirée ou réduite pour que le plus petit côté de l'image soit affiché entièrement. |
| *drRépété* | L'image est répétée pour occuper toute la nouvelle largeur et toute la nouvelle hauteur spécifiées. |

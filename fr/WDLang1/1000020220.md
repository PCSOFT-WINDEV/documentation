<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions de gestion des champs Zone multiligne via une syntaxe préfixée sont les suivantes :



|   |   |
| --- | --- |
| [&lt;Zone multiligne&gt;.AjouteLigne](../WDLang1/1000020216.md) | Ajoute une nouvelle ligne dynamique dans un champ Zone multiligne. |
| [&lt;Zone multiligne&gt;.InsèreLigne](../WDLang1/1000020217.md) | Insère une nouvelle ligne dynamique dans un champ Zone multiligne. |
| [&lt;Zone multiligne&gt;.Supprime](../WDLang1/1000020218.md) | Supprime une ligne dynamique dans un champ Zone multiligne. |
| [&lt;Zone multiligne&gt;.SupprimeTout](../WDLang1/1000020219.md) | Supprime toutes les lignes dynamiques dans un champ Zone multiligne. |




## Fonctions de gestion des champs Liste, Liste image et Combo (syntaxe préfixée)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions permettant de gérer les champs Liste, Liste image et Combo (via une syntaxe préfixée) sont les suivantes : 



|   |   |
| --- | --- |
| [&lt;Liste&gt;.Affiche](../WDLang1/1000017318.md) | Rafraîchit l'affichage d'un champ Liste, Liste image ou Combo fichier. |
| [&lt;Liste&gt;.Ajoute](../WDLang1/1000017319.md) | Ajoute un élément dans un champ Liste par programmation, un champ Liste image par programmation, un champ Carrousel ou un champ Combo par programmation. |
| [&lt;Liste&gt;.Cherche](../WDLang1/1000017320.md) | Recherche un élément dans un champ Liste (liste classique ou liste image) ou un champ Combo (y compris les valeurs décrites par la fonction [gLien](../WDLang1/3029043.md)). |
| [&lt;Liste&gt;.DéplaceLigne](../WDLang1/1000017322.md) | Déplace une ligne ou échange deux lignes d'un champ Liste ou Combo. |
| [&lt;Liste&gt;.InfoXY](../WDLang1/1000017323.md) | Renvoie, pour une position dans un champ Liste (liste classique ou liste image), l'indice de la ligne (ou de l'image) correspondant à cette position. |
| [&lt;Liste&gt;.Insère](../WDLang1/1000017326.md) | Insère un élément dans un champ Liste par programmation (y compris Liste image) ou dans un champ Combo par programmation. |
| [&lt;Liste&gt;.Modifie](../WDLang1/1000017327.md) | Modifie un élément dans un champ Liste par programmation (liste classique ou liste image) ou dans un champ Combo par programmation. |
| [&lt;Liste&gt;.Occurrence](../WDLang1/1000017334.md) | Renvoie le nombre d'éléments présents dans un champ Liste, Liste image ou Combo. |
| [&lt;Liste&gt;.Position](../WDLang1/1000017335.md) | Affiche un champ Liste (y compris Liste image) à partir d'un élément spécifié ou renvoie l'indice du premier élément affiché dans un champ Liste (lorsque l'ascenseur de la liste a été utilisé). |
| [&lt;Liste&gt;.Select](../WDLang1/1000017336.md) | Renvoie l'indice de l'élément sélectionné dans un champ Liste, Liste image ou Combo. |
| [&lt;Liste&gt;.SelectMoins](../WDLang1/1000017393.md) | Désélectionne un élément dans un champ Liste, Liste image ou Combo. |
| [&lt;Liste&gt;.SelectOccurrence](../WDLang1/1000017394.md) | Renvoie le nombre d'éléments sélectionnés dans un champ Liste, Liste image ou Combo. |
| [&lt;Liste&gt;.SelectPlus](../WDLang1/1000017414.md) | Sélectionne un ou plusieurs éléments dans un champ Liste, Liste image ou Combo. |
| [&lt;Liste&gt;.Supprime](../WDLang1/1000017415.md) | Supprime un élément d'un champ Liste par programmation (y compris Liste image) ou d'un champ Combo par programmation. |
| [&lt;Liste&gt;.SupprimeTout](../WDLang1/1000017416.md) | Vide de tous ses éléments un champ Liste par programmation (y compris Liste image) ou un champ Combo par programmation. |
| [&lt;Liste&gt;.Trie](../WDLang1/1000017417.md) | Trie les éléments d'un champ Liste, Liste image ou Combo rempli par programmation. |

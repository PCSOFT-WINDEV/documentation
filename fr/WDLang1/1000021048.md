


## Fonctions de gestion des champs Tableau de bord
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions de gestion des champs Tableau de bord sont les suivantes :



|   |   |
| --- | --- |
| [TDBAffiche](../WDLang1/1000020970.md) | Rafraîchit l'affichage des éléments d'un champ Tableau de bord. |
| [TDBAjouteWidget](../WDLang1/1000020953.md) | Ajoute et affiche un Widget (fenêtre interne) dans un champ Tableau de bord. |
| [TDBChangeTailleWidget](../WDLang1/1000021286.md) | Change les dimensions d'un Widget présent dans un champ Tableau de bord. |
| [TDBChargeConfiguration](../WDLang1/1000020961.md) | Restaure la configuration d'un champ Tableau de bord. |
| [TDBConfigurationInitiale](../WDLang1/1000021043.md) | Modifie la configuration d'un champ Tableau de bord (définie sous l'éditeur) avant son premier affichage. |
| [TDBDéplaceWidget](../WDLang1/1000021289.md) | Change la position d'un widget présent dans un champ Tableau de bord. |
| [TDBInfoXY](../WDLang1/1000020974.md) | Permet de savoir :<br><br>- si un Widget est présent dans un champ Tableau de bord à une position donnée.<br><br>- quel est le Widget présent dans un champ Tableau de bord à une position donnée.<br><br><br> |
| [TDBMode](../WDLang1/1000020956.md) | Ouvre ou ferme le mode "Edition" d'un champ Tableau de bord. |
| [TDBOccurrence](../WDLang1/1000021042.md) | Renvoie le nombre de Widgets présents dans un champ Tableau de bord. |
| [TDBRécupèreWidgetPosition](../WDLang1/1000022498.md) | Renvoie la position d'un Widget présent dans un champ Tableau de bord. |
| [TDBSauveConfiguration](../WDLang1/1000020960.md) | Sauvegarde la configuration d'un champ Tableau de bord sous forme d'une chaîne. |
| [TDBSupprime](../WDLang1/1000020955.md) | Supprime un Widget affiché dans un champ Tableau de bord. Le champ Tableau de bord est mis à jour et un emplacement vide remplace le Widget supprimé. |
| [TDBSupprimeTout](../WDLang1/1000020954.md) | Supprime tous les Widgets affichés dans un champ Tableau de bord. |





Autres fonctions disponibles : 



|   |   |
| --- | --- |
| [DnDDonneElémentTDB](../WDLang1/1000020971.md) | Ajoute un Widget dans un champ Tableau de bord lors d'un "Drag and Drop". |

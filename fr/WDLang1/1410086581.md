


## Télémétrie : Installation du serveur
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000173"></a>
Pour utiliser la télémétrie, il est nécessaire d'installer sur un poste : 

- [Un serveur HFSQL](#NOTE2_1). 

- [Un serveur de télémétrie](#NOTE3_1). 




Remarques : 

- Le poste où est installé le serveur de télémétrie doit être un serveur Web. Un serveur Web (IIS, Apache ou autre) doit être installé sur le poste. 

- Il est possible d'installer le serveur HFSQL et le serveur de télémétrie sur deux postes différents. 

- Le serveur de télémétrie peut être installé sur un serveur Web hébergeant également un serveur d'application WEBDEV. 

- Attention : La télémétrie ne fonctionne pas avec les proxy non transparents.

## Installation d'un serveur HFSQL
<a name="installation_serveur_hfsql_ELTTEXTE000197"></a>


### Installation du serveur HFSQL
<a name="installation_serveur_hfsql_ELTPARAGRAPHE000046"></a>Le programme d'installation du serveur HFSQL est disponible dans le package d'installation de WINDEV ou WINDEV Mobile. Si vous ne possédez pas ce package d'installation, l'installation du serveur HFSQL est également disponible sur le site de PC SOFT ([http://www.pcsoft.fr](https://www.pcsoft.fr)).

Pour installer le serveur HFSQL depuis le package d'installation de WINDEV/WINDEV Mobile :

1. Lancez le programme d'installation de WINDEV/WINDEV Mobile.

2. Choisissez l'option "Installation du serveur HFSQL".

3. Sélectionnez ensuite l'option "Installer ou mettre à jour un serveur HFSQL".

4. Acceptez la licence.

5. Choisissez la plateforme (option "Pour Windows sur cette machine").

6. Si des serveurs HFSQL sont déjà installés sur le poste en cours, choisissez l'option "Installer un nouveau serveur".

7. Sélectionnez le répertoire d'installation et indiquez le nom du serveur et le port. Par défaut, le port utilisé est 4900. 

8. Indiquez le nom et le mot de passe de l'utilisateur ayant les droits d'administration qui sera créé par défaut sur le serveur HFSQL. Par défaut, cet utilisateur est "Admin" sans mot de passe.

9. Installez si nécessaire le Centre de Contrôle HFSQL si celui-ci n'est pas présent ou accessible depuis votre poste. 

10. L'assistant propose de paramétrer l'envoi de notifications afin de connaître les dysfonctionnements du serveur en tant réel. Sélectionnez l'option voulue. 

11. L'assistant propose d'activer l'authentification par Active Directory. Activez cette option si nécessaire.

12. Validez (ou non) la participation à l'amélioration du produit installé en autorisant la collecte d'informations relatives à l'utilisation du produit. Cette collecte, facultative et anonyme, permet à PC SOFT d'améliorer les fonctionnalités du produit.

13. L'installation est effectuée. Par défaut (si vous n'avez pas modifié les caractéristiques de l'administrateur), pour se connecter au serveur en mode administrateur, il faut utiliser l'utilisateur "Admin" sans mot de passe.
	Note : Pour des raisons évidentes de sécurité, pensez à changer le mot de passe de l'administrateur.

## Installation d'un serveur de télémétrie
<a name="installation_serveur_telemetrie_ELTTEXTE000221"></a>
Le serveur de télémétrie est un serveur d'application WEBDEV réservé à la télémétrie.

**Pour installer le serveur de télémétrie** : 

1. Lancez le pack d'installation du serveur réservé (fichier WX280PACKRSRV.exe disponible dans le sous-répertoire "Install\\WDServeurR" du répertoire d'installation de WINDEV).

2. Sélectionnez si nécessaire la langue d'installation. 

3. Validez la licence. 

4. Sélectionnez le module à installer : "Installer le serveur de télémétrie (WDAnalytics)". <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Install_serveur_telemetrie%20-%20HC%20N%B0001.gif&type=thumb)
Passez à l'étape suivante. 

5. Sélectionnez si nécessaire la version du Serveur d'application réservé à installer (32 ou 64 bits). Passez à l'étape suivante.

6. Sélectionnez le répertoire d'installation. <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Install_serveur_telemetrie%20-%20HC%20N%B0002.gif&type=thumb)
Passez à l'étape suivante.

7. Spécifiez les caractéristiques du serveur HFSQL utilisé pour la télémétrie (installé précédemment) : <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Install_serveur_telemetrie%20-%20HC%20N%B0003.gif&type=thumb)


	- Nom du serveur HFSQL, 

- Port. 

- Nom et mot de passe de l'administrateur du Serveur HFSQL. Ces informations sont nécessaires pour configurer automatiquement la base de données de la télémétrie et l'utilisateur associé. 

8. Testez la connexion pour vérifier vos paramètres. 

9. Indiquez les paramètres spécifiques à la base de données utilisée par la télémétrie : 

	- le nom de la base de données dédiée à la télémétrie (TLM par défaut). 

	- le nom de l'utilisateur administrateur de la télémétrie et son mot de passe. 




10. Cette base de données et son utilisateur vont être automatiquement créés. 

11. Passez à l'étape suivante.  

12. Indiquez le premier jour de la semaine. Cette information permettra de réaliser les statistiques par semaine. <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Install_serveur_telemetrie%20-%20HC%20N%B0004.gif&type=thumb)


13. Validez. 

14. L'assistant vérifie si un serveur Web est présent sur le poste d'installation. 

	- Si aucun serveur Web n'est installé sur le poste d'installation, l'assistant propose d'en installer un et de le configurer pour utiliser le serveur de télémétrie. 

	- Si un serveur Web est présent sur le poste d'installation, l'assistant propose de le configurer pour utiliser le serveur de télémétrie. 




15. A la fin de l'installation, l'assistant indique le login et le mot de passe de la télémétrie. **Copiez et conservez ces informations**. Ces login et mot de passe doivent être utilisés pour vous connecter au serveur de télémétrie depuis WINDEV ou WINDEV Mobile.
	**Important** : La mémorisation des paramètres de configuration du serveur réservé est enregistrée dans le fichier suivant : "C:\\ProgramData\\WDAnalytics\\WDAnalytics.default.xml"
	Ce fichier doit être ajouté au plan de sauvegarde du serveur. En effet, ce fichier devra obligatoirement être restauré en cas de réinstallation du serveur ou de duplication sur une nouvelle machine physique ou virtuelle en reprenant les mêmes réglages.

16. Terminez l'assistant. 




Remarques : 

- Si le nom de la base de données est erroné, il est possible de relancer l'installation du serveur de télémétrie pour indiquer le nom correct de la base tout en conservant les identifiants. 

- Un serveur de télémétrie en version 28 est compatible avec des applications Windows compilées avec des versions inférieures (21, ...).

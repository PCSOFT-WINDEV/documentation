
## Encode (Fonction)

***En anglais : Encode***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Encode une chaîne de caractères ou un buffer. Pour décoder le résultat, il suffit d'utiliser la fonction [Décode](../WDLang1/1000022259.md).




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Encodage d'une image en Base 64
Res est une chaîne = Encode(fChargeBuffer("MonImage.jpg"), encodeBASE64)
```
<a name="Exemple2"></a>

```wl
// Encodage d'une URL
Info(Encode("https://www.google.fr/?", encodeURLDepuisAnsi) + ...
	Encode(Encode("https://www.google.fr/?é", encodeURLDepuisAnsi), encodeURLParamètre))
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = Encode(<Données> [, <Format d'encodage>])`
---

**`<Résultat> : (Chaîne de caractères)`**

Données encodées selon le format choisi.

**`<Données> : (Chaîne de caractères ou Buffer)`**

Données à encoder.

**`<Format d'encodage> : (Constante optionnelle de type Entier)`**

Format d'encodage des données :


|   |   |
| --- | --- |
| *encodeBASE64*<br>(Valeur par défaut) | Format BASE 64 (respecte la norme RFC 4648). Le texte encodé comporte uniquement des caractères imprimables et les RC (Retour Chariot).<br><br> |
| *encodeBASE64SansRC* | Format BASE 64 (respecte la norme RFC 4648) sans les RC (Retour Chariot). Le texte encodé comporte uniquement des caractères imprimables.<br><br><br><br> |
| *encodeBASE64URL* | Format BASE 64 URL (respecte la norme RFC 4648). Le texte encodé comporte uniquement des caractères imprimables.<br><br> |
| *encodeBASE85* | Format BASE 85. Le texte encodé comporte uniquement des caractères imprimables.<br><br><br><br> |
| *encodePCS* | Format ASCII. Le texte encodé comporte uniquement des caractères imprimables.<br><br> |
| *encodeURLDepuisAnsi* | Format URL. Le texte encodé comporte uniquement des caractères autorisés dans une URL. Le texte à encoder est une chaîne ANSI.<br><br> |
| *encodeURLDepuisUnicode* | Format URL. Le texte encodé comporte uniquement des caractères autorisés dans une URL. Le texte à encoder est une chaîne Unicode.<br><br> |
| *encodeURLDepuisUTF8* | Format URL. Le texte encodé comporte uniquement des caractères autorisés dans une URL. Le texte à encoder est une chaîne UTF-8.<br><br><br><br> |
| *encodeURLParamètre* | Cette constante est utilisée quand il est nécessaire de passer une URL A en paramètre d'une URL B. Les caractères réservés de l'URL A (//, ?...) sont encodés.<br><br>L'URL A (à encoder) doit être une URL valide. Si elle a elle-même des paramètres, ils doivent avoir été encodés avec une des constantes suivantes : *encodeURLDepuisAnsi*, *encodeURLDepuisUnicode* ou *encodeURLDepuisUTF8*.<br><br><br><br> |
| *encodeUUEncode* | Format UUEncode. Le texte encodé comporte uniquement des caractères imprimables.<br><br> |




## Fonctions diverses Windows
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions diverses Windows sont les suivantes :  



|   |   |
| --- | --- |
| [API](../WDLang1/3014005.md) | Exécute une fonction présente dans une DLL externe. |
| [APIParamètre](../WDLang1/3014026.md) | Modifie les options d'appel de la fonction [API](../WDLang1/3014005.md) et de la fonction [AppelDLL32](../WDLang1/3014003.md). |
| [AppelDLL32](../WDLang1/3014003.md) | Exécute une fonction présente dans une DLL externe. |
| [AppelInterface](../WDLang1/1000017088.md) | Exécute une méthode d'une interface d'un objet implémenté dans une DLL externe au framework WINDEV. |
| [BitcoinAdresseValide](../WDLang1/1410088858.md) | Vérifie si une adresse d'un portefeuille Bitcoin est valide. |
| [ChargeDLL](../WDLang1/3014006.md) | Charge en mémoire la librairie (DLL) spécifiée. |
| [COMAppelleMéthode](../WDLang1/1000018832.md) | Exécute une méthode d'une interface d'un objet COM instancié grâce à la fonction [COMCréeInstance](../WDLang1/1000018831.md). |
| [COMCréeInstance](../WDLang1/1000018831.md) | Instancie un objet COM (Component Object Model). |
| [COMRécupèreInterface](../WDLang1/1000019784.md) | Appelle la méthode QueryInterface de l'objet COM spécifié pour obtenir une interface spécifique de cet objet afin de l'utiliser avec la fonction [COMAppelleMéthode](../WDLang1/1000018832.md). |
| [ConstruitEntier](../WDLang1/3062016.md) | Construit un entier de 4 octets à partir de deux entiers de 2 octets. |
| [CréeRaccourci](../WDLang1/3014008.md) | Crée un raccourci sur le bureau de Windows, dans le menu "Démarrer" de Windows ou dans un répertoire spécifique. |
| [DéchargeDLL](../WDLang1/3014009.md) | Décharge la librairie 32 bits (DLL) qui a été précédemment chargée en mémoire avec la fonction [ChargeDLL](../WDLang1/3014006.md). |
| [EnvoieTouche](../WDLang1/3014014.md) | Simule des manipulations de touches au clavier et à la souris. |
| [IBANValide](../WDLang1/1410089019.md) | Teste la validité d'un IBAN (International Bank Acount Number). |
| [InstalleIcone](../WDLang1/3014016.md) | Crée une icône (associée à un programme) dans un groupe de programme de Windows, avec la ligne de commande associée. |
| [Instance](../WDLang1/3014019.md) | Renvoie le "Handle système" (HINSTANCE) du programme en cours. |
| [OpenSSLVérifie](../WDLang1/1000020432.md) | Vérifie grâce à une clé publique que la signature de données réalisée avec la bibliothèque OpenSSL est correcte. |
| [Ping](../WDLang1/3014013.md) | Teste si une adresse réseau est joignable en utilisant le protocole ICMP (équivalent de l'utilitaire réseau PING). |
| [PoidsFaible](../WDLang1/3014018.md) | Renvoie les deux octets de poids faible d'un entier. |
| [PoidsFort](../WDLang1/3014017.md) | Renvoie les deux octets de poids fort d'un entier. |
| [sCalculeCrc16](../WDLang1/3062007.md) | Contrôle une chaîne de caractères avant et après transmission entre applications WINDEV / entre sites WEBDEV. |
| [sCalculeCrc32](../WDLang1/3062015.md) | Calcule le code de redondance cyclique (CRC) d'un tampon. |
| [SupprimeRaccourci](../WDLang1/3014012.md) | Supprime un raccourci précédemment créé avec la fonction [CréeRaccourci](../WDLang1/3014008.md) sur le bureau de Windows, dans le menu "Démarrer" de Windows ou dans un répertoire spécifique. |
| [ToucheEnfoncée](../WDLang1/3014007.md) | Vérifie quelle est la touche enfoncée. |
| [Transfert](../WDLang1/3014015.md) | Copie un bloc d'octets ou une chaîne de taille fixe d'une adresse mémoire dans une autre. |
| [VersionWindows](../WDLang1/3014002.md) | Renvoie le mode d'exécution de Windows. |





Il est également possible d'utiliser les types de variables suivants : 



|   |   |
| --- | --- |
| [Description d'API](../WDLang1/1000019149.md) | Une variable de type **Description d'API** permet de décrire par programmation une API, ses paramètres, sa valeur de retour et ses options d'appel. |

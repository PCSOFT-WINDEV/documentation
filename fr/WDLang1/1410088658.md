
## Constantes Diagramme
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les constantes de gestion des diagrammes sont les suivantes :

| Constante | Détail | Type de variable |
| --- | --- | --- |
| **diagActionAucune** | Actions autorisées sur une forme : Aucune action n'est autorisée pour l'utilisateur final sur la forme.  | Type [diagForme](../WDLang1/1410088082.md) |
| **diagActionToute** | Actions autorisées sur une forme : Toutes les actions sont autorisées pour l'utilisateur final sur la forme. | Type [diagForme](../WDLang1/1410088082.md) |
| **lienDestinationAuto** | Position de l'extrémité du lien sur la forme destination : La position de l'extrémité du lien est déterminée automatiquement selon la position du lien et de la forme. | Type [diagExtrémité](../WDLang1/1410088470.md) |
| **lienDestinationForme** | Position de l'extrémité du lien sur la forme destination : La position de l'extrémité du lien est situé au milieu de la forme. | Type [diagExtrémité](../WDLang1/1410088470.md) |
| **lienDestinationPointDAttache** | Position de l'extrémité du lien sur la forme destination : La position de l'extrémité du lien correspond à un point d'attache. | Type [diagExtrémité](../WDLang1/1410088470.md) |
| **lienDestinationRectangleEnglobant** | Position de l'extrémité du lien sur la forme destination : La position de l'extrémité du lien correspond à un point du rectangle englobant la forme. | Type [diagExtrémité](../WDLang1/1410088470.md) |
| **routeDirect** | Type de route empruntée par le lien : Le tracé du lien est une ligne droite du point de départ au point d'arrivée en passant par les points de passage. | Type [diagLien](../WDLang1/1410088454.md) |
| **routeOrthogonal** | Type de route empruntée par le lien : Le tracé du lien peut faire des changements de direction à angle droit entre point de départ et le point d'arrivée en passant par les points de passage. | Type [diagLien](../WDLang1/1410088454.md) |

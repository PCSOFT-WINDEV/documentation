
## SysIMEI (Fonction)

***En anglais : SysIMEI***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le numéro IMEI d'un téléphone Android.

**Remarque** : La fonction [SysNumSérie](../WDLang1/3073033.md) permet d'obtenir le numéro IMEI d'un téléphone Android. 
Attention : Cette information est disponible jusqu'à Android 9. A partir d'Android 10, cette information n'est plus accessible : cette fonction provoque une erreur fatale.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI Val(SysVersionAndroid(sysVersionNuméro))<10 ALORS
	// Récupération du numéro IMEI de l'appareil
	Info("Le numéro IMEI de cet appareil est " + SysIMEI("Cellular Line"))
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = SysIMEI()`
---

**`<Résultat> : (Chaîne de caractères)`**



- Numéro IMEI, 

- Chaîne vide ("") en cas d'erreur.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### IMEI
<a name="imei_ELTPARAGRAPHE000055"></a>L'International Mobile Equipment Identity (IMEI) est un numéro qui permet d'identifier de manière unique chacun des terminaux mobiles (ME) GSM ou UMTS.

Ce numéro permet à l'opérateur du réseau de téléphonie mobile d'identifier le mobile appelant et ainsi de l'autoriser ou non à se connecter. Il permet ainsi l'établissement des appels d'urgence sans SIM. Ce numéro est également utile pour pouvoir bloquer auprès de l'ensemble des opérateurs un mobile volé.

Tous les appareils qui n'ont pas d'emplacement de carte SIM n'ont pas de numéro IMEI (cas de la majorité des tablettes). Pour les appareils avec carte SIM, le numéro IMEI peut-être récupéré jusqu'à Android 9. A partir d'Android 10, cette information n'est plus accessible.

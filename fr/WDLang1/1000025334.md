


## TableauUnion (Fonction)

***En anglais : ArrayUnion***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie un tableau correspondant à l'union de deux tableaux.

Les tableaux suivants peuvent être manipulés :

- Tableau d'éléments simples (entier, réel, chaîne de caractères) à une dimension.

- Tableau de structures à une dimension. 





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Premier tableau d'entiers
tabMonTableau1 est un tableau d'entiers
TableauAjoute(tabMonTableau1, 1)
TableauAjoute(tabMonTableau1, 6)
TableauAjoute(tabMonTableau1, 1)
TableauAjoute(tabMonTableau1, 3)

// Second tableau d'entiers
tabMonTableau2 est un tableau d'entiers
TableauAjoute(tabMonTableau2, 3)
TableauAjoute(tabMonTableau2, 5)
TableauAjoute(tabMonTableau2, 8)
TableauAjoute(tabMonTableau2, 2)

// Tableau de l'union
tabMonUnion est un tableau d'entiers
tabMonUnion = TableauUnion(tabMonTableau1, tabMonTableau2)
// le résultat est 1;6;3;5;8;2
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = TableauUnion(<Premier tableau WLangage> , <Second tableau WLangage>)`
---

**`<Résultat> : (Tableau)`**

Variable de type Tableau contenant le résultat de l'union. 

**`<Premier tableau WLangage> : (Tableau)`**

Nom de la variable de type Tableau correspondant au premier tableau à manipuler. Ce tableau doit être un tableau à une dimension. Ce tableau ne sera pas modifié. 

**`<Second tableau WLangage> : (Tableau)`**

Nom de la variable de type Tableau correspondant au second tableau à manipuler. Ce tableau doit être un tableau à une dimension. Ce tableau doit être du même type que le tableau spécifié avec le paramètre &lt;Premier tableau WLangage&gt;. Ce tableau ne sera pas modifié. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Si un tableau contient plusieurs doublons, ces doublons ne sont pas conservés. 

- Si les tableaux sont triés, le tableau résultat n'est pas trié.

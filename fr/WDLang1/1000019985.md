


## &lt;Organigramme&gt;.SupprimeTout (Fonction)

***En anglais : &lt;Organization Chart&gt;.DeleteAll***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime tous les éléments d'un champ Organigramme. Le champ Organigramme est entièrement vidé. 

**Remarque** : Si le champ Organigramme est relié à une source fichier, la fonction **&lt;Organigramme&gt;.SupprimeTout** n'a aucun effet sur la source : aucun enregistrement n'est supprimé si la source n'est pas préalablement vidée.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Suppression de tous les éléments du champ Organigramme
ORGA_DIRECTION.SupprimeTout()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Champ Organigramme>.SupprimeTout()`
---

**`<Champ Organigramme> : (Nom de champ)`**

Nom du champ Organigramme dont tous les éléments doivent être supprimés.

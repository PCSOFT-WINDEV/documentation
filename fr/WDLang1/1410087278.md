


## &lt;Champ d'affichage HTML&gt;.DepuisEmail (Fonction)

***En anglais : &lt;HTML Display control&gt;.FromEmail***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Importe le contenu HTML d'un email et ses images dans un champ d'affichage HTML. Les images contenues dans l'email sont également affichées dans le champ.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Lire un message et l'afficher dans le champ HTML_Visu
MonMessage est un Email
EmailLitMessage(IDSession, MonMessage)
HTM_AffichageHTML.DepuisEmail(MonMessage)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Champ d'affichage HTML>.DepuisEmail(<Email>)`
---

**`<Champ d'affichage HTML> : (Nom de champ)`**

Nom du champ d'affichage HTML manipulé.

**`<Email> : (Variable de type Email)`**

Nom de la variable de type [Email](../WDLang3/1000018713.md) à afficher dans le champ d'affichage HTML.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Le contenu HTML du mail est affecté au champ d'affichage HTML avec ses images. 

- La propriété [RépertoireDeTravail](../Proprietes/1000026169.md) est affectée avec le chemin du répertoire temporaire contenant les images associées au mail.

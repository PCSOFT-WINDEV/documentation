 

## dSauveImage (Fonction)

***En anglais : dSaveImage***
	

<a name="XUtilisation"></a>
<a name="Utilisation"></a>
|  |   |
| --- | --- |



|  
 | **But**<br><br>Sauver une image sur disque.<br><br>**Syntaxe**<br><br>&lt;Résultat&gt; = dSauveImage(&lt;NomChampImage&gt; , &lt;NomFichier&gt; [,&lt;NombreCouleur&gt; [, &lt;Format&gt; [, &lt;Qualité&gt;]]])<br><br>- &lt;Résultat&gt; est un booléen qui vaut vrai si la sauvegarde a réussi.<br><br>- &lt;NomChampImage&gt; est le nom du champ Image que l'on veut sauver.<br><br>- &lt;NomFichier&gt; est le nom du fichier.<br><br>- &lt;NombreCouleur&gt; est un entier qui permet de définir le nombre de couleurs de l'image sauvegardée (0 par défaut).<br><br>- &lt;Format&gt; "BMP" (par défaut) ou "JPG" ou "GIF".<br><br>- &lt;Qualité&gt; qualité de sauvegarde : entre 1 et 100 pour JPEG, entre 16 et 256 pour GIF (=nombre de couleurs dans la palette), inutilisé pour BMP<br><br><br><br><br>**Détail**<br><br>**dSauveImage** sauvegarde la partie visible du champ Image considéré.<br><br>**Récapitulatif : Interaction entre les différents paramètres de la fonction**<br><br><br>\   \   \   \   \<br>\ --- \ --- \ --- \ --- \<br>\ &lt;Format&gt; \ &lt;NombreCouleur&gt; \ &lt;Qualité&gt; \ Détail \<br>\ "BMP" \ 0 ou 16 \ Inutilisé \ Si NombreCouleur=0, alors utilise le nombre de couleurs de la résolution \<br>\ "JPG" \ Inutilisé \ 1-100 \ Profondeur de couleur sur 24 bits \<br>\ "GIF" \ Inutilisé \ 16-256 \ Le paramètre &lt;qualité&gt; indique le nombre de couleur dans la palette \<br><br>**Notes**<br><br>L'existence du fichier n'est pas vérifiée. Si le fichier existait, il est écrasé.<br><br>La fonction **dSauveImage** permet de sauvegarder des images pour des résolutions d'écran allant jusqu'à 32 bits.<br><br>Les fonctions de dessin (commençant par "d") ne doivent être utilisées que pour des champs de type "image".<br><br> |

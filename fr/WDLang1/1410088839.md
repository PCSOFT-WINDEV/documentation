
## PicCaptureLassoOCR (Fonction)

***En anglais : PicRectangularSnipOCR***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Lance une capture d'écran par le lasso pour en extraire le texte via une analyse avec le système OCR intégré.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
sTexteImage est une chaîne
sTexteImage = PicCaptureLassoOCR()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = PicCaptureLassoOCR([<Iconise>])`
---

**`<Résultat> : (Chaîne de caractères)`**

Texte contenu dans l'image capturée. 

**`<Iconise> : (Booléen optionnel)`**



- <u><u><u><u>Vrai</u></u></u></u> pour iconiser l'application lors de la capture, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.

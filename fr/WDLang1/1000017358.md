
## Déplace (Fonction)

***En anglais : Move***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet :

- de déplacer un élément dans un tableau WLangage à une dimension ou dans une propriété tableau d'un type avancé (tableau des événements de gglAgenda, ...). Les autres éléments du tableau sont décalés.

- d'échanger deux éléments dans un tableau WLangage à une dimension ou dans une propriété tableau d'un type avancé (tableau des événements de gglAgenda, ...).




**Remarque** : Cette fonction est équivalente à la fonction [TableauDéplace](../WDLang1/3075011.md).




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonTableau est un tableau de 6 chaînes
MonTableau[1] = "A"
MonTableau[2] = "B"
MonTableau[3] = "C"
MonTableau[4] = "D"
MonTableau[5] = "E"
MonTableau[6] = "F"
// Déplacer l'élément 2 à la 5° position
Déplace(MonTableau, 2, 5)
// Le tableau contient les éléments "A", "C", "D", "E", "B", "F"
```


<a name="Exemple2"></a>



```wl
MonTableau est un tableau de 6 chaînes
MonTableau[1] = "A"
MonTableau[2] = "B"
MonTableau[3] = "C"
MonTableau[4] = "D"
MonTableau[5] = "E"
MonTableau[6] = "F"
// Échanger l'élément 2 avec l'élément 5
Déplace(MonTableau, 2, 5, tdEchange)
// Le tableau contient les éléments "A", "E", "C", "D", "B", "F"
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Déplacer un élément dans un tableau WLangage ou dans la propriété tableau d'un type avancé

`Déplace(<Nom du tableau> , <Indice départ> , <Indice arrivée> [, <tdDéplace>])`
---

**`<Nom du tableau> : (Tableau WLangage)`**

Nom de la variable de type Tableau à manipuler. Ce tableau doit être un tableau à une dimension.

**`<Indice départ> : (Entier)`**

Indice de l'élément à déplacer.

**`<Indice arrivée> : (Entier)`**

Nouvel indice de l'élément déplacé.

**`<tdDéplace> : (Constante optionnelle)`**

*tdDéplace* : Constante permettant de déplacer un élément dans un tableau WLangage.  


<a name="SYNTAXE2"></a>

### Échanger deux éléments dans un tableau WLangage ou dans la propriété tableau d'un type avancé

`Déplace(<Nom du tableau> , <Indice du 1° élément> , <Indice du 2° élément> , <tdEchange>)`
---

**`<Nom du tableau> : (Tableau WLangage)`**

Nom de la variable de type Tableau à manipuler. Ce tableau doit être un tableau à une dimension.

**`<Indice du 1° élément> : (Entier)`**

Indice du premier élément à échanger.

**`<Indice du 2° élément> : (Entier)`**

Indice du second élément à échanger.

**`<tdEchange> : (Constante)`**

*tdEchange* : Constante permettant d'échanger deux éléments dans un tableau WLangage.




## Fonctions de gestion des champs Liste, Liste image et Combo
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


Les fonctions permettant de gérer les champs Liste, Liste image et Combo sont les suivantes :



|   |   |
| --- | --- |
| [FichierVersListeMémoire](../WDLang1/3049007.md) | Remplit un champ Liste par programmation (ou Combo) avec tous les enregistrements d'un fichier de données ou d'une requête. |
| [ListeAffiche](../WDLang1/3049005.md) | Rafraîchit l'affichage d'un champ Liste, Liste image ou Combo fichier |
| [ListeAjoute](../WDLang1/3049004.md) | Ajoute un élément dans un champ Liste par programmation (y compris Liste image) ou un champ Combo par programmation. |
| [ListeCherche](../WDLang1/3049003.md) | Recherche un élément dans un champ Liste (liste classique ou liste image) ou un champ Combo (y compris les valeurs décrites par la fonction [gValeurMémorisée](../WDLang1/1410088363.md)). |
| [ListeDéplaceLigne](../WDLang1/1000017279.md) | Déplace une ligne ou échange deux lignes d'un champ Liste ou Combo. |
| [ListeInfoXY](../WDLang1/3049006.md) | Renvoie, pour une position dans un champ Liste (liste classique ou liste image), l'indice de la ligne (ou de l'image) correspondant à cette position. |
| [ListeInsère](../WDLang1/3049002.md) | Insère un élément dans un champ Liste par programmation (y compris Liste image) ou dans un champ Combo par programmation. |
| [ListeModifie](../WDLang1/3049009.md) | Modifie un élément dans un champ Liste par programmation (liste classique ou liste image) ou dans un champ Combo par programmation. |
| [ListeOccurrence](../WDLang1/3049014.md) | Renvoie le nombre d'éléments présents dans un champ Liste, Liste image ou Combo. |
| [ListePosition](../WDLang1/3049015.md) | Affiche un champ Liste (y compris Liste image) à partir d'un élément spécifié ou renvoie l'indice du premier élément affiché dans un champ Liste (lorsque l'ascenseur du champ a été utilisé). |
| [ListeSelect](../WDLang1/3049011.md) | Renvoie l'indice de l'élément sélectionné dans un champ Liste, Liste image ou Combo. |
| [ListeSelectMoins](../WDLang1/3049008.md) | Désélectionne un élément dans un champ Liste, Liste image ou Combo. |
| [ListeSelectOccurrence](../WDLang1/3049010.md) | Renvoie le nombre d'éléments sélectionnés dans un champ Liste, Liste image ou Combo. |
| [ListeSelectPlus](../WDLang1/3049012.md) | Sélectionne un ou plusieurs éléments dans un champ Liste, Liste image ou Combo. |
| [ListeSupprime](../WDLang1/3049016.md) | Supprime un élément d'un champ Liste par programmation (y compris Liste image) ou d'un champ Combo par programmation. |
| [ListeSupprimeTout](../WDLang1/3049013.md) | Vide de tous ses éléments un champ Liste par programmation (y compris Liste image) ou un champ Combo par programmation. |
| [ListeTrie](../WDLang1/1000017022.md) | Trie les éléments d'un champ Liste, Liste image ou Combo rempli par programmation. |






- ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=LechampListeImage.gif) ***Exemples unitaires (WINDEV)*** : **Le champ Liste Image** <br>Utilisation du champ Liste Image de WINDEV.<br>Le champ Liste Image permet d'afficher le contenu d'une liste sous forme de vignettes.<br>Tout comme les champs Liste ou Table, le champ Liste Image peut être "mémoire" (rempli par programmation) ou "fichier" (lié à une base de données).

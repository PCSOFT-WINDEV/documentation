
## Constantes de gestion des applications avec "Live update"
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


| Constante | Utilisation | Valeur |
| --- | --- | --- |
| **acArrêt** | Mode de contrôle à distance : Arrêt de l'application. | [AppliContrôle](../WDLang1/3013003.md) |
| **acAvertissement** | Mode de contrôle à distance : Avertissement d'un arrêt imminent. | [AppliContrôle](../WDLang1/3013003.md) |
| **acInterdit** | Mode de contrôle à distance : Accès interdit à l'application. | [AppliContrôle](../WDLang1/3013003.md) |
| **appMAJAfficheErreur** | Permet d'afficher ou non un message en cas de problème lors de la vérification des mises à jour de l'application. | [AppliChangeParamètre](../WDLang1/1000017260.md), <br>[AppliParamètre](../WDLang1/1000017268.md) |
| **appMAJChemin** | Chemin des informations de mise à jour sur le serveur. | [AppliChangeParamètre](../WDLang1/1000017260.md), <br>[AppliParamètre](../WDLang1/1000017268.md) |
| **appMAJDélai** | Délai entre deux messages (si appMAJMode = 2) ou avant l'arrêt automatique de l'application (si appMAJMode = 3). Ce délai est exprimé en secondes.  | [AppliChangeParamètre](../WDLang1/1000017260.md), <br>[AppliParamètre](../WDLang1/1000017268.md) |
| **appMAJDispoFacultative** | Une mise à jour facultative est disponible. | [AppliMAJDisponible](../WDLang1/1000017293.md), [Procédure de la fonction AppliDétecteMAJDisponible](../WDLang1/1410087706.md) |
| **appMAJDispoObligatoire** | Une mise à jour obligatoire est disponible. | [AppliMAJDisponible](../WDLang1/1000017293.md), [Procédure de la fonction AppliDétecteMAJDisponible](../WDLang1/1410087706.md) |
| **appMAJMode** | Mode de fonctionnement actuel de l'application :<br><br>- 0 : Fonctionnement normal.<br><br>- 1 : Lancement interdit : les exécutables en cours continuent de fonctionner.<br><br>- 2 : Lancement interdit : les exécutables en cours reçoivent un message d'avertissement périodiquement.<br><br>- 3 : Lancement interdit : les exécutables en cours sont arrêtés automatiquement au bout d'un certain délai.<br><br><br> | [AppliChangeParamètre](../WDLang1/1000017260.md), <br>[AppliParamètre](../WDLang1/1000017268.md) |
| **appMAJModeAffichage** | Mode d'affichage choisi pour prévenir l'utilisateur qu'une mise à jour de l'application est disponible :<br><br>- 0 : Une fenêtre dans l'application est utilisée pour avertir l'utilisateur.<br><br>- 1 : Une notification est utilisée pour avertir l'utilisateur.<br><br><br> | [AppliChangeParamètre](../WDLang1/1000017260.md), <br>[AppliParamètre](../WDLang1/1000017268.md) |
| **appMAJMsgArrêt** | Message affiché avant l'arrêt automatique de l'application (quand appMAJMode = 3). | [AppliChangeParamètre](../WDLang1/1000017260.md), <br>[AppliParamètre](../WDLang1/1000017268.md) |
| **appMAJMsgAvertissement** | Message affiché périodiquement (quand appMAJMode = 2). | [AppliChangeParamètre](../WDLang1/1000017260.md), <br>[AppliParamètre](../WDLang1/1000017268.md) |
| **appMAJMsgInterdit** | Message affiché lorsque le lancement de l'application est interdit. | [AppliChangeParamètre](../WDLang1/1000017260.md), <br> <br>[AppliParamètre](../WDLang1/1000017268.md) |
| **appMAJNonDispo** | Aucune mise à jour n'est disponible. | [AppliMAJDisponible](../WDLang1/1000017293.md) |
| **appMAJProtocole** | Protocole réseau utilisé pour la recherche des mises à jour. Ce protocole peut correspondre à "LOCAL" ou "HTTP". | [AppliChangeParamètre](../WDLang1/1000017260.md), <br> <br>[AppliParamètre](../WDLang1/1000017268.md) |
| **appMAJRéseau** | Niveau de mise à jour :<br><br>- 0 : Pas de mise à jour.<br><br>- 1 : Mise à jour réseau activée.<br><br>- 2 : Mise à jour réseau activée et inscription des applications auprès du serveur de mise à jour.<br><br><br> | [AppliChangeParamètre](../WDLang1/1000017260.md), <br>[AppliParamètre](../WDLang1/1000017268.md) |
| **appMAJServeur** | Serveur sur lequel sont recherchées les mises à jour.  Si le protocole HTTP est utilisé, adresse IP ou nom du poste.  Si le protocole LOCAL est utilisé, nom du partage réseau. | [AppliChangeParamètre](../WDLang1/1000017260.md), <br> <br>[AppliParamètre](../WDLang1/1000017268.md) |
| **appMAJTempsAffichage** | Temps d'affichage des messages. | [AppliParamètre](../WDLang1/1000017268.md) |
| **appMAJVerifAuto** | Vérification automatique des mises à jour. | [AppliChangeParamètre](../WDLang1/1000017260.md), <br>[AppliParamètre](../WDLang1/1000017268.md) |
| **appMAJVersionCompatible** | Numéro de version minimale compatible avec la version de référence. | [AppliParamètre](../WDLang1/1000017268.md) |
| **appMAJVersionLocale** | Numéro de version de l'exécutable local. | [AppliChangeParamètre](../WDLang1/1000017260.md),<br>[AppliParamètre](../WDLang1/1000017268.md) |
| **appMAJVersionRéférence** | Numéro de version de la version de référence. | [AppliParamètre](../WDLang1/1000017268.md) |




## &lt;Diagramme de Gantt&gt;.JourSemaineOuvré (Fonction)

***En anglais : &lt;Gantt chart&gt;.WorkingWeekDay***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Indique si un jour de la semaine est ouvré dans une colonne d'un champ Table (ou Table hiérarchique) de type Diagramme de Gantt.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
COL_Gantt.JourSemaineOuvré(6, Vrai) // Samedi ouvré
COL_Gantt.JourSemaineOuvré(7, Faux) // Dimanche non ouvré
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Colonne Gantt>.JourSemaineOuvré(<Numéro du jour> , <Ouvré>)`
---

**`<Colonne Gantt> : (Nom de champ)`**

Nom de la colonne d'un champ Table (ou Table hiérarchique) de type Diagramme de Gantt à manipuler.

**`<Numéro du jour> : (Entier)`**

Numéro du jour de la semaine à manipuler :

- 1 : lundi,

- 2 : mardi,

- 3 : mercredi,

- 4 : jeudi,

- 5 : vendredi,

- 6 : samedi, 

- 7 : dimanche.




**`<Ouvré> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> : si le jour est ouvré.

- <u><u><u><u>Faux</u></u></u></u> : si le jour n'est pas ouvré.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Les jours non ouvrés sont utilisés pour calculer la date de fin d'une tâche en fonction de sa durée.
	Par exemple, une tâche de 2 jours qui commence un vendredi finira lundi si le samedi et le dimanche ne sont pas ouvrés.

- Si la durée des tâches est spécifiée en heures (propriété **DuréeEnHeure** de la variable de type [TâcheGantt](../WDLang1/1000020881.md)), les heures ouvrables sont aussi utilisées pour calculer la date de fin.

- Par défaut, les jours non ouvrés sont samedi et dimanche.

- Les jours fériés définis avec la fonction [JourFériéAjoute](../WDLang1/1000017304.md) sont considérés comme des jours non ouvrés.

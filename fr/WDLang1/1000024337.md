


## &lt;Image&gt;.Redimensionne (Fonction)

***En anglais : &lt;Image&gt;.Resize***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Redimensionne une image contenue dans un champ Image.



![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Les dessins](../WDLang1/3029035.md). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Zoom du champ IMG_Image1 : on grossit 2 fois l'image affichée dans le champ
Résultat1 est un booléen
Résultat1 = IMG_Image1.Redimensionne(IMG_Image1.Largeur * 2, IMG_Image1.Hauteur * 2)
SI Résultat1 = Faux ALORS Erreur("L'opération a échoué")
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Champ Image>.Redimensionne(<Nouvelle largeur> , <Nouvelle hauteur> [, <Option>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le redimensionnement a été effectué, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Champ Image> : (Nom de champ)`**

Nom du champ Image à manipuler.

**`<Nouvelle largeur> : (Entier)`**

Nouvelle largeur de l'image exprimée en pixels.

**`<Nouvelle hauteur> : (Entier)`**

Nouvelle hauteur de l'image exprimée en pixels.

**`<Option> : (Constante de type Entier optionnelle)`**

Options de redimensionnement : 


|   |   |
| --- | --- |
| *drEtiré*<br>(Valeur par défaut) | L'image est étirée si nécessaire pour correspondre à la nouvelle largeur et à la nouvelle hauteur spécifiées.<br><br>![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Cette constante n'est pas disponible. |
| *drHauteQualité* | Cette constante peut être combinée avec les options de redimensionnement de type homothétique.<br><br>En mode homothétique, cette constante permet d'améliorer la qualité de l'image lorsque celle-ci est réduite par rapport à sa taille initiale.<br>![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Cette constante n'est pas disponible. |
| *drHomothétique* | L'image est étirée ou réduite pour occuper de manière optimale l'espace défini sans être déformée. Elle occupe :<br><br>- soit toute la hauteur spécifiée.<br><br>- soit toute la largeur spécifiée.<br><br><br><br><br>![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Cette constante n'est pas disponible. |
| *drHomothétiqueCentré* | L'image est étirée ou réduite pour occuper de manière optimale l'espace défini sans être déformée et elle est centrée. Elle occupe :<br><br>- soit toute la hauteur spécifiée : l'image est centrée horizontalement.<br><br>- soit toute la largeur spécifiée : l'image est centrée verticalement.<br><br><br>![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Cette constante n'est pas disponible. |
| *drHomothétiqueEtendu* | L'image est étirée ou réduite pour que le plus petit côté de l'image soit affiché entièrement.<br>![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Cette constante n'est pas disponible. |
| *drRépété* | L'image est répétée pour occuper toute la nouvelle largeur et toute la nouvelle hauteur spécifiées.<br>![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Cette constante n'est pas disponible. |







![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Ce paramètre n'est pas disponible.

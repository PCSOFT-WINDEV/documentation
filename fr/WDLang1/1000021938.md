


## &lt;Table hiérarchique&gt;.SupprimeFils (Fonction)

***En anglais : &lt;TreeView Table&gt;.DeleteChild***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime toutes les lignes filles d'une branche de la hiérarchie. La branche elle-même n'est pas supprimée.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Supprime toutes les lignes filles de l'élément "Desserts"
TABLEH_Recette.SupprimeFils("Desserts")
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Supprimer les lignes filles d'une branche en spécifiant son indice

`<Champ Table hiérarchique>.SupprimeFils(<Indice de l'élément parent>)`
---

**`<Champ Table hiérarchique> : (Nom de champ)`**

Nom du champ Table hiérarchique à manipuler.

**`<Indice de l'élément parent> : (Entier)`**

Indice de la ligne dont toutes les lignes filles doivent être supprimées.


<a name="SYNTAXE2"></a>

### Supprimer les lignes filles d'une branche en spécifiant son nom

`<Champ Table hiérarchique>.SupprimeFils(<Nom de l'élément parent>)`
---

**`<Champ Table hiérarchique> : (Nom de champ)`**

Nom du champ Table hiérarchique à manipuler.

**`<Nom de l'élément parent> : (Chaîne de caractères)`**

Contenu de la première colonne de l'élément dont toutes les lignes filles doivent être supprimées.

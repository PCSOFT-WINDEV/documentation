
## Couleur (Type de variable)

***En anglais : Color***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **Couleur** permet de définir toutes les caractéristiques avancées d'une couleur WLangage. Les caractéristiques de cette couleur peuvent être définies et modifiées à l'aide de différentes propriétés WLangage. 

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MaCouleur est une Couleur = JauneClair
MaCouleur.Luminosité += 10
```

## Remarques


### Propriétés spécifiques à la description des variables de type Couleur
<a name="proprietes_specifiques_description_des_variables_type_couleur_ELTPARAGRAPHE000059"></a>Les propriétés suivantes peuvent être utilisées pour manipuler une variable de type **Couleur** :

| **Nom de la propriété** | **Type manipulé** | **Effet** |
| --- | --- | --- |
| Bleu | Entier | Valeur de la composante bleue de la couleur. |
| Couleur | Entier | Valeur de la couleur. Cette valeur est compatible avec : <br><br>- les fonctions [RVB](../WDLang1/3029012.md), [RVBBleu](../WDLang1/3029049.md), [RVBRouge](../WDLang1/3029050.md), [RVBVert](../WDLang1/3029051.md), ...<br><br>- les constantes WLangage de couleurs. <br><br><br>Cette propriété correspond à la propriété par défaut utilisée par le type **Couleur**. |
| Luminosité | Entier | Luminosité de la couleur (comprise entre 0 et 255). |
| Opacité | Entier | Opacité par rapport à la couche alpha associée à la couleur. <br>Cette propriété doit être comprise entre 0 et 255. |
| Rouge | Entier | Valeur de la composante rouge de la couleur. |
| Saturation | Entier | Saturation de la couleur (comprise entre 0 et 255). |
| Teinte | Entier | Teinte de la couleur (comprise entre 0 et 255). |
| Vert | Entier | Valeur de la composante verte de la couleur. |

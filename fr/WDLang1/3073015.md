
## SysNomExe (Fonction)

***En anglais : SysNameExe***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le nom et le chemin complet d'une application.

**Remarque** : Pour récupérer le nom de l'exécutable courant, utilisez la fonction [ExeInfo](../WDLang1/3035001.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Récupérer le nom et le chemin complet de l'application en cours
ResNomExe = SysNomExe(SysInstance())
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = SysNomExe(<Instance de l'exécutable>)`
---

**`<Résultat> : (Chaîne de caractères)`**

Nom et chemin complet de l'application spécifiée.

**`<Instance de l'exécutable> : (Entier)`**

Instance de l'exécutable de l'application à manipuler. L'instance d'une application peut être obtenue avec la fonction [SysInstance](../WDLang1/3073018.md).

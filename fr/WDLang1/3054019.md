
## LangueVersNation (Fonction)

***En anglais : LanguageToNation***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la nation correspondant à une langue.






<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connaître la nation correspondant
// à la langue arménienne
ResNation est un entier
ResNation = LangueVersNation(langueArménien) 
SELON ResNation
	CAS nationAllemand :
	 ...
	CAS nationAustralien :
	 ...
	CAS 0 :
	 ...
FIN
```


<a name="Exemple2"></a>



```wl
// Connaître la nation correspondant
// à la langue anglaise en Australie
ResNation est un entier
ResNation = LangueVersNation(langueAnglais, 3)
SELON ResNation
	CAS nationAnglais :
	 ...
	CAS nationAustralien :
	 ...
	CAS 0 :
	 ...
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = LangueVersNation(<Langue> [, <Sous-langue>])`
---

**`<Résultat> : (Constante de type Entier)`**



- Nation correspondant de la langue spécifiée, 

- 0 si la langue spécifiée ne correspond à aucune nation. 


Pour plus de détails sur ces constantes, consultez [Correspondance entre les langues, les sous-langues, les alphabets et les nations](../WDLang1/3054022.md).

**`<Langue> : (Constante de type Entier)`**

Langue dont on veut connaître la nation. Pour plus de détails sur ces constantes, consultez [Correspondance entre les langues, les sous-langues, les alphabets et les nations](../WDLang1/3054022.md).

**`<Sous-langue> : (Entier optionnel)`**

Sous-langue dont on veut connaître la nation. Pour plus de détails sur ces sous-langues, consultez [Correspondance entre les langues, les sous-langues, les alphabets et les nations](../WDLang1/3054022.md).
Si ce paramètre n'est pas spécifié, la nation standard de la &lt;Langue&gt; spécifiée est donnée.

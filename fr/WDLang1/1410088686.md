


## &lt;Variable diagForme&gt;.VersImage (Fonction)

***En anglais : &lt;diagShape variable&gt;.ToImage***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Crée une image à partir d'une forme d'un diagramme (la forme peut être affichée ou non dans le diagramme). Cette fonction est utile par exemple pour créer une vignette d'une forme.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affiche la prévisualisation d'un forme
MaForme est un diagRectangle
MaForme.Largeur = 320
MaForme.Hauteur = 200
MonImage est une Image 
MonImage = MaForme.VersImage(64, 64, 8)
IMG_Previsualisation = MonImage
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Forme>.VersImage(<Largeur> , <Hauteur> [, <Marge>])`
---

**`<Résultat> : (Variable de type Image)`**

Variable de type [Image](../WDLang1/1000019650.md) correspondant à la forme voulue.

**`<Forme> : (Variable de type diagForme (ou une de ses dérivées))`**

Nom de la variable de type diagForme (ou une de ses dérivées) à manipuler. Il est possible d'utiliser les variables suivantes : 

- Variable de type [diagChemin](../WDLang1/1410088384.md),

- Variable de type [diagGroupe](../WDLang1/1410088127.md),

- Variable de type [diagImage](../WDLang1/1410088414.md),

- Variable de type [diagLien](../WDLang1/1410088454.md),

- Variable de type [diagOvale](../WDLang1/1410088135.md),

- Variable de type [diagPolygone](../WDLang1/1410088137.md),

- Variable de type [diagRectangle](../WDLang1/1410088083.md).




**`<Largeur> : (Réel)`**

Largeur totale de l'image à créer (en pixels).

**`<Hauteur> : (Réel)`**

Hauteur totale de l'image à créer (en pixels).

**`<Marge> : (Réel optionnel)`**

Taille du cadre autour du dessin de la forme (en pixels)

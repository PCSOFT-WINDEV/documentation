


## &lt;TreeMap&gt;.Occurrence (Fonction)

***En anglais : &lt;TreeMap&gt;.Count***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le nombre d'éléments présents dans un champ TreeMap.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Nombre d'éléments présents dans le champ TreeMap TMAP_Fichier
ResNombreElement = TMAP_Fichier.Occurrence()
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Champ TreeMap>.Occurrence()`
---

**`<Résultat> : (Entier)`**

Nombre d'éléments présents dans le champ TreeMap (0 si le champ est vide). 
**Remarque** : Tous les éléments sont pris en compte (même les éléments de poids 0).

**`<Champ TreeMap> : (Nom de champ)`**

Nom du champ TreeMap à manipuler.

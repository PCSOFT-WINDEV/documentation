
## SaisieAssistée (Type de variable)

***En anglais : AssistedInput***
			
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **SaisieAssistée** permet de définir toutes les caractéristiques avancées d'une ligne dans la saisie assistée. Les caractéristiques de cette ligne peuvent être définies et modifiées à l'aide de différentes propriétés WLangage. 

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ajout d'une ligne
_sa est SaisieAssistée
_sa.Libellé = "34300(Agde)"
_sa.ValeurMémorisée = 343300
SaisieAssistéeAjoute(SAI_CP, _sa)

// -- Evénement "Sélection d'une valeur dans la liste de saisie assistée"
PROCEDURE Sélection(MaSélection est SaisieAssistée)
SAI_CP = MaSélection.ValeurMémorisée
```





<a name="PROPS"></a>

## Propriétés


### Propriétés spécifiques à la description des variables de type SaisieAssistée
<a name="proprietes_specifiques_description_des_variables_type_saisieassistee_ELTPARAGRAPHE000033"></a>Les propriétés suivantes peuvent être utilisées pour manipuler une ligne proposée dans la saisie assistée :

| Nom de la propriété | Type manipulé | Effet |
| --- | --- | --- |
| Libellé | Chaîne de caractères | Libellé affiché dans la ligne de saisie assistée. |
| ValeurMémorisée | Tous types | Valeur mémorisée pour la ligne de la saisie assistée. Cette valeur peut être utilisée par le développeur pour mémoriser des informations. |


<a name="NOTE0"></a>

## Remarques
<a name="NOTE0_1"></a>


### Utilisation des variables de type SaisieAssistée
<a name="utilisation_des_variables_type_saisieassistee_ELTPARAGRAPHE000062"></a>Une variable de type SaisieAssistée peut être : 

- utilisée avec la fonction [SaisieAssistéeAjoute](../WDLang1/1000020043.md). 

- passée en paramètre à l'événement "Sélection d'une valeur dans la liste de saisie assistée".


## Fonctions de gestion de la saisie invalide
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions permettant de gérer la saisie invalide sont les suivantes : 



|   |   |
| --- | --- |
| [SaisieInvalideAfficheBulle](../WDLang1/1000024896.md) | Affiche une bulle d'erreur sous un champ de saisie pour indiquer la raison de la saisie invalide. |
| [SaisieInvalideAfficheErreur](../WDLang1/1000026247.md) | Affiche une erreur de saisie invalide pour le champ spécifié. |
| [SaisieInvalideAfficheIcone](../WDLang1/1000024917.md) | Affiche une icône d'erreur à côté d'un champ de saisie pour indiquer que la saisie est invalide. |
| [SaisieInvalideDétecte](../WDLang1/1000026180.md) | Détecte et met en erreur les champs dont la saisie est invalide et /ou obligatoire. |
| [SaisieInvalideListeChamp](../WDLang1/1000026244.md) | Construit la liste des champs dont la saisie est invalide/obligatoire. |

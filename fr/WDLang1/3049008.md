


## ListeSelectMoins (Fonction)

***En anglais : ListSelectMinus***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Désélectionne un élément dans un champ Liste, Liste image ou Combo.

**Rappel** : L'élément sélectionné correspond à l'élément en cours sur lequel le bandeau de sélection est affiché.

**Remarque** : La fonction **ListeSelectMoins** peut être utilisée indifféremment sur :

- les champs Liste fichier ou Liste par programmation,

- les champs Liste monosélection ou multisélection.













<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Désélection de l'élément dont l'indice est 7 dans le champ "LISTE_ListeClient"
ListeSelectMoins(LISTE_ListeClient, 7)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`ListeSelectMoins(<Champ Liste> [, <Indice 1> [... [, <Indice N>]]])`
---

**`<Champ Liste> : (Nom de champ)`**

Nom du champ Liste (ou Combo) à manipuler. Si ce paramètre correspond à une chaîne vide (""), l'élément spécifié selon son indice est désélectionné dans le champ Liste (ou Combo) auquel appartient l'événement en cours.



**`<Indice 1> : (Entier optionnel)`**

Indice du premier élément à désélectionner. 
Si aucun indice n'est précisé, tous les éléments du champ Liste (ou Combo) spécifié sont désélectionnés. 
Si ces indices correspondent à des éléments non sélectionnés, la fonction **ListeSelectMoins** est sans effet.



**`<Indice N> : (Entier optionnel)`**

Indice du Nième élément à désélectionner. 
Si aucun indice n'est précisé, tous les éléments du champ Liste (ou Combo) spécifié sont désélectionnés. 
Si ces indices correspondent à des éléments non sélectionnés, la fonction **ListeSelectMoins** est sans effet.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Champ Combo
<a name="champ_combo_ELTPARAGRAPHE000123"></a>Si la fonction **ListeSelectMoins** est utilisée avec un champ Combo :

- aucun élément du champ Combo n'est affiché,

- si le champ Combo est en saisie, le champ de saisie est mis à vide.



<a name="NOTE0_2"></a>


### Divers
<a name="divers_ELTPARAGRAPHE000138"></a>

- Pour désélectionner un élément dans un champ Table, utilisez la fonction [TableSelectMoins](../WDLang1/3074029.md).

- La fonction **ListeSelectMoins** peut être utilisée sur une colonne d'un champ Table de type "Combo".

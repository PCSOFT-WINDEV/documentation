
## RTFInsère (Fonction)

***En anglais : RTFInsert***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Insère une chaîne de caractères au format RTF dans le contenu d'un champ RTF (dans une fenêtre ou un état WINDEV) ou dans le contenu d'une chaîne de caractères au format RTF.

Rappel : Pour ajouter une chaîne au format RTF à la fin d'une autre chaîne RTF, utilisez la fonction [RTFAjoute](../WDLang1/3025024.md). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Insère au début de SAI_SAISIE1 le contenu du champ SAI_SAISIE2
RTFInsère(SAI_SAISIE1, SAI_SAISIE2, 1)

// Insère la date du jour à la position du curseur
RTFInsère(SAI_SAISIE1, DateSys(), SAI_SAISIE1..Curseur)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Insère une chaîne de caractères dans un champ RTF

`RTFInsère(<Champ RTF> , <Chaîne de caractères à insérer> , <Position d'insertion>)`
---

**`<Champ RTF> : (Chaîne de caractères)`**

Nom du champ RTF (champ de saisie, champ d'un état WINDEV).

**`<Chaîne de caractères à insérer> : (Chaîne de caractères)`**

Chaîne de caractères au format RTF à insérer dans le contenu du champ de type RTF.

**`<Position d'insertion> : (Entier)`**

Position d'insertion de la chaîne de caractères au format RTF. Le premier caractère contenu dans le champ a pour indice 1.  


<a name="SYNTAXE2"></a>

### Insère une chaîne de caractères dans une chaîne RTF

`<Résultat> = RTFInsère(<Chaîne RTF> , <Chaîne de caractères à insérer> , <Position d'insertion>)`
---

**`<Résultat> : (Chaîne de caractères)`**

Nouvelle chaîne de caractères dans laquelle l'ajout a été effectué. La chaîne initiale n'est pas modifiée.

**`<Chaîne RTF> : (Chaîne de caractères)`**

Nom de la chaîne de caractères au format RTF.

**`<Chaîne de caractères à insérer> : (Chaîne de caractères)`**

Chaîne de caractères au format RTF à insérer dans le contenu de la chaîne de type RTF.

**`<Position d'insertion> : (Entier)`**

Position d'insertion de la chaîne de caractères au format RTF. Le premier caractère contenu dans la chaîne a pour indice 1.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Cette fonction pilote la gestion RTF de WIndows. Selon la version de Windows utilisée, il se peut qu'une fin de paragraphe soit ajoutée à la fin de la ligne.

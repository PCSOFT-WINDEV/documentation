


## TableRécupèreParent (Fonction)

***En anglais : TableGiveParent***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le "parent" d'un élément (niveau immédiatement supérieur) dans un champ Table hiérarchique.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Parent du noeud en cours dans le champ Table hiérarchique "TABLEH_RecetteTV"
Res est un entier
Res = TableRécupèreParent(TABLEH_RecetteTV)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = TableRécupèreParent(<Champ Table hiérarchique> [, <Numéro de la ligne>])`
---

**`<Résultat> : (Entier)`**



- Numéro de la ligne du "parent" trouvé, 

- 0 si la ligne n'a pas de parent, 

- -1 si la ligne n'existe pas.




**`<Champ Table hiérarchique> : (Nom de champ)`**

Nom du champ Table hiérarchique à manipuler.

**`<Numéro de la ligne> : (Entier optionnel)`**

Numéro de la ligne contenant la branche à manipuler. Si ce paramètre n'est pas spécifié, la ligne en cours est manipulée.

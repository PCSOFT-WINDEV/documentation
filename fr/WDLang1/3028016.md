
## Constantes DDE
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


| Constante | Description | Utilisée par les fonctions |
| --- | --- | --- |
| **DDEActif** | Mode de lancement du programme : le programme lancé est actif et il "prend la main" sur le programme en cours. | [DDELance](../WDLang1/3028013.md) |
| **DDEAExécuter** | Événement DDE à gérer : le programme reçoit une commande à exécuter provenant d'une autre application. | [DDEEvénement](../WDLang1/3028009.md) |
| **DDEALier** | Événement DDE : le programme reçoit une demande de lien "tiède" ou "chaud" à une donnée (cette demande provient d'une autre application). | [DDEEvénement](../WDLang1/3028009.md) |
| **DDEAModifier** | Événement DDE : le programme reçoit un signal de modification de la valeur d'une donnée liée (ce signal provient d'une autre application). | [DDEEvénement](../WDLang1/3028009.md) |
| **DDEARecevoir** | Événement DDE : le programme reçoit une donnée d'une autre application. | [DDEEvénement](../WDLang1/3028009.md) |
| **DDEARépondre** | Événement DDE : le programme reçoit une demande de donnée d'une autre application. | [DDEEvénement](../WDLang1/3028009.md) |
| **DDEIconise** | Mode de lancement du programme : le programme lancé est inactif et iconisé. | [DDELance](../WDLang1/3028013.md) |
| **DDEInactif** | Mode de lancement du programme : le programme lancé est inactif. | [DDELance](../WDLang1/3028013.md) |
| **DDEMaximise** | Mode de lancement du programme : le programme lancé est actif et il "prend la main" sur le programme en cours (la fenêtre d'exécution est maximale). | [DDELance](../WDLang1/3028013.md) |

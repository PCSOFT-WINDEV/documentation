
## LuhnCalcule (Fonction)

***En anglais : LuhnCalculate***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Calcule le prochain chiffre de contrôle de Luhn pour une suite donnée. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
sNuméroSiretIncomplet est une chaîne = "120 027 016 0056"
cChiffreContrôle est un caractère = LuhnCalcule(sNuméroSiretIncomplet)
Info("Le chiffre de contrôle est [%cChiffreContrôle%]")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = LuhnCalcule(<Identifiant>)`
---

**`<Résultat> : (Chaîne de caractères)`**

Chiffre de contrôle de Luhn pour la suite donnée

**`<Identifiant> : (Chaîne de caractères)`**

Suite à partir de laquelle le chiffre de Luhn va être déterminé. Cette suite doit impérativement être composée uniquement de chiffres et d'espaces (les espaces ne sont pas déterminants pour la formule).

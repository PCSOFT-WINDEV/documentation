


## fFichierExiste (Fonction)

***En anglais : fFileExist***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Teste l'existence d'un fichier externe.

**Remarque** : Sous Windows 64 bits, l'accès à un répertoire système à partir d'un exécutable 32 bits peut être effectué dans un répertoire différent de celui attendu. Pour plus de détails, consultez la page d'aide [64 bits natif et 32 bits natif](../Editeurs/2030048.md).

<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) 
```wl
UnFichier est une chaîne
UnFichier = "C:\Temp\Exemple.txt"
SI fFichierExiste(UnFichier) = Vrai ALORS 
	Info("Le fichier " + UnFichier + " existe bien sur le disque")
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = fFichierExiste(<Nom du fichier>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le fichier existe, 

- <u><u><u><u>Faux</u></u></u></u> sinon (par exemple si le nom du fichier correspond à un nom de répertoire).




**`<Nom du fichier> : (Chaîne de caractères)`**

Nom et chemin complet du fichier. Les caractères jokers (\*, ?) ne sont pas utilisables. Pour utiliser les caractères jokers, la fonction [fRep](../WDLang1/3036056.md) peut être utilisée. Si seul le nom du fichier est précisé (sans le chemin), le fichier est recherché dans le répertoire en cours (par exemple : fFichierExiste("Test.jpg"))
![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png)![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) Ce paramètre peut être au format Ansi ou Unicode.


## AppliSupprimeVersion (Fonction)

***En anglais : AppDeleteVersion***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime une version de l'historique des versions présent sur le serveur d'installation de référence.

**Attention** : Cette fonction est irréversible : les fichiers de l'installation de la version correspondante seront supprimés du serveur d'installation de référence. 

**Remarque** : Cette fonction n'est utilisable qu'avec une installation réseau sur un réseau local. Elle ne peut être employée avec une installation par IP.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Crée la liste des versions disponibles
ListeVersions est une chaîne 
ListeVersions = AppliListeVersion()
// Extraction de la première version
PremièreVersion est une chaîne
PremièreVersion = ExtraitChaine(ListeVersions, 1, RC)
// Suppression de la version
SI AppliSupprimeVersion(PremièreVersion) = Faux ALORS
	Erreur("Erreur à la suppression de la version " + ...
			PremièreVersion + RC + ErreurInfo())
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = AppliSupprimeVersion(<Version> [, <Exécutable>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la suppression a réussi, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**Remarques** : Les causes d'erreurs les plus courantes sont : 

- l'historique des versions n'est pas activé. 

- la version demandée n'existe pas.

- la version supprimée correspond à la version en cous d'utilisation. Dans ce cas, il est possible par exemple de changer la version active grâce à la fonction [AppliActiveVersion](../WDLang1/1000018846.md). 




**`<Version> : (Chaîne de caractères)`**

Numéro de la version à supprimer. 

**`<Exécutable> : (Chaîne de caractères optionnelle)`**

Chemin complet de l'exécutable de l'application pour laquelle la version doit être supprimée. Si ce paramètre n'est pas précisé, c'est l'application en cours qui est manipulée.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
La fonction **AppliSupprimeVersion** doit être utilisée par un utilisateur possédant les droits d'accès en écriture sur le serveur d'installation de référence.

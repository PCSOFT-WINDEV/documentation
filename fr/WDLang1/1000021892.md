
## SysAssocieExtension (Fonction)

***En anglais : SysLinkExtension***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Associe une extension (.txt, .avi, etc.) à un programme d'exécution par défaut.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Associe l'extension ".txt" à l'exécutable "app.exe"
SysAssocieExtension(".txt", "C:\Program Files\MyApp\app.exe ""%1""", sysUtilisateurCourant)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = SysAssocieExtension(<Extension> , <Commande> [, <Utilisateur>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'association est effectuée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. La fonction [ErreurInfo](../WDLang1/3013008.md) permet d'obtenir plus de détails.




**`<Extension> : (Chaîne de caractères)`**

Extension à associer. 

**`<Commande> : (Chaîne de caractères)`**

Commande à exécuter. Cette commande est de la forme : "CheminCompletDeLExécutable %1"
où %1 représente le fichier sélectionné à passer en argument. Il est conseillé de l'entourer de guillemets pour s'assurer de sa bonne exécution.
Remarque : Ce paramètre peut contenir des raccourcis système tels que %systemroot%.

**`<Utilisateur> : (Constante optionnelle de type Entier)`**

L'association entre l'application et l'extension est enregistrée dans la base de registres. Deux options sont possibles : 


|   |   |
| --- | --- |
| *sysTousUtilisateurs* | L'association sera prise en compte pour tous les utilisateurs du poste en cours. <br><br>Remarque : Il est nécessaire de posséder les droits "Administrateur" pour exécuter cette opération. |
| *sysUtilisateurCourant*<br>(Valeur par défaut) | L'association sera prise en compte uniquement pour l'utilisateur courant. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Une seule commande est acceptée par extension. Si plusieurs commandes sont associées à une même extension, seule la dernière commande sera prise en compte.




## KanbanAjouteCarte (Fonction)

***En anglais : KanbanAddCard***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ajoute une carte dans une liste d'un champ Kanban.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ajoute une carte dans la liste de Kanban LISTE_Matière_à_Revoir
// Cette liste de kanban a été définie sous l'éditeur 
maCarte est un kbCarte
maCarte.Titre = "Anglais"
KanbanAjouteCarte(KANBAN_Révision, LISTE_Matière_à_Revoir, maCarte)
```



```wl
// Crée dynamiquement une liste de Kanban
LISTE_Matières_Revues est un Champ
LISTE_Matières_Revues <- KanbanAjouteListe(KANBAN_Révision, "Revues")

// Ajoute une carte dans la liste de Kanban créée dynamiquement
// (Liste de Kanban manipulée par son libellé) 
maCarte2 est un kbCarte
maCarte2.Titre = "Espagnol"
KanbanAjouteCarte(KANBAN_Révision, "Revues", maCarte2)

// Ajoute la carte dans la liste de Kanban créée dynamiquement 
// (Liste de Kanban manipulée par son nom)
maCarte1 est un kbCarte
maCarte1.Titre = "Français"
KanbanAjouteCarte("Revues", maCarte1)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Ajouter une carte dans une liste de Kanban en spécifiant le champ Kanban

`<Résultat> = KanbanAjouteCarte(<Champ Kanban> , <Liste de Kanban> , <Carte>)`
---

**`<Résultat> : (Entier)`**

Indice de l'élément ajouté. 

**`<Champ Kanban> : (Nom de champ)`**

Nom du champ Kanban manipulé. 

**`<Liste de Kanban> : (Nom de champ)`**

Nom du champ Liste de Kanban dans lequel la carte doit être ajoutée. Ce nom peut correspondre : 

- au nom du champ Liste de Kanban défini sous l'éditeur, 

- au nom du champ Liste de Kanban défini par programmation. 




**`<Carte> : (Variable de type kbCarte)`**

Nom de la variable de type [kbCarte](../WDLang1/1410089167.md) correspondant à la carte à ajouter. La carte est ajoutée à la fin de la liste de Kanban spécifiée. 


<a name="SYNTAXE2"></a>

### Ajouter une carte dans une liste de Kanban

`<Résultat> = KanbanAjouteCarte(<Liste de Kanban> , <Carte>)`
---

**`<Résultat> : (Entier)`**

Indice de l'élément ajouté. 

**`<Liste de Kanban> : (Chaîne de caractères)`**

Nom du champ Liste de Kanban dans lequel la carte doit être ajoutée. Ce nom peut correspondre : 

- au nom du champ Liste de Kanban défini sous l'éditeur, 

- au nom du champ Liste de Kanban défini par programmation. 




**`<Carte> : (Variable de type kbCarte)`**

Nom de la variable de type [kbCarte](../WDLang1/1410089167.md) correspondant à la carte à ajouter. La carte est ajoutée à la fin de la liste de Kanban spécifiée.




## &lt;Multimédia&gt;.Position (Fonction)

***En anglais : &lt;Multimedia&gt;.Position***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie ou modifie la position en cours du fichier multimédia en cours dans le champ Multimédia.







<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) 
```wl
MonFichier est une chaîne = "C:\Ma musique\mysong.mp3"
MM_Multimédia1.Joue(MonFichier)
// Positionne à 10 secondes du début du fichier
MM_Multimédia1.Position(10000) 
MM_Multimédia1.Arrêt()
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître la position dans le fichier

`<Résultat> = <Champ Multimédia>.Position()`
---

**`<Résultat> : (Entier)`**

Position actuelle dans le fichier exprimée en millisecondes.

**`<Champ Multimédia> : (Nom de champ)`**

Nom du champ Multimédia à manipuler.


<a name="SYNTAXE2"></a>

### Modifier la position dans le fichier

`<Résultat> = <Champ Multimédia>.Position(<Nouvelle position>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le positionnement demandé a été effectué, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Champ Multimédia> : (Nom de champ)`**

Nom du champ Multimédia à manipuler.

**`<Nouvelle position> : (Entier ou Durée)`**

Nouvelle position dans le fichier multimédia, exprimée en millisecondes. Ce paramètre peut correspondre à : 

- un entier correspondant au nombre de millisecondes, 

- une variable de type Durée, 

- l'indication directe de la durée (par exemple 1 s ou 10 ms).

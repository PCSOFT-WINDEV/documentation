
## SysVersionAndroid (Fonction)

***En anglais : SysAndroidVersion***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Retourne des informations sur la version d'Android utilisée par l'application.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Les fonctions Bluetooth ne sont disponibles 
// qu'à partir de l'Api Level 5 d'Android (Android "Eclair" 2.0)
SI Val(SysVersionAndroid(sysVersionApiLevel)) >= 5 ALORS
	// Active le bluetooth sur l'appareil
	BTActive(vrai)
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = SysVersionAndroid([<Type d'information>])`
---

**`<Résultat> : (Chaîne de caractères)`**



- Information demandée sur la version d'Android. 

- Chaîne vide ("") en cas d'erreur. 




**`<Type d'information> : (Constante de type Entier optionnelle)`**

Indique l'information à demander : 


|   |   |
| --- | --- |
| *sysVersionApiLevel* | Api Level d'Android. |
| *sysVersionNuméro* | Numéro de version d'Android.<br>Par exemple : "1.5", "2.2", "2.3.3", ...). |
| *sysVersionPlateForme* | Nom de la version majeure d'Android :<br><br>- "Cupcake"<br><br>- "Donut"<br><br>- "Eclair"<br><br>- "Froyo"<br><br>- "Gingerbread"<br><br>- "Honeycomb"<br><br>- "Ice Cream Sandwich"<br><br><br>Si le nom de la version n'est pas connu, &lt;Résultat&gt; correspond à une chaîne vide (""). |

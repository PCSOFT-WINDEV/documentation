


## &lt;Variable Pile&gt;.Info (Fonction)

***En anglais : &lt;Stack variable&gt;.Info***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Récupère les caractéristiques d'une pile : types des éléments et nombre d'éléments.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
stMaClasse est une Structure
	MaClasseProp est un entier
FIN

// Pile dont les éléments sont des instances de stMaClasse
MaPile est une Pile de stMaClasse
// Récupère la définition du type des éléments
MaDéfinition est une Définition 
MaDéfinition = MaPile.Info(tiDéfinitionElément)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Nom de la pile>.Info(<Information>)`
---

**`<Résultat> : (Variant)`**

Information demandée.

**`<Nom de la pile> : (Pile)`**

Nom de la variable de type [Pile](../WDLang1/1514022.md) à manipuler.

**`<Information> : (Constante de type Entier)`**

Information à récupérer :


|   |   |
| --- | --- |
| *tiDéfinitionElément* | &lt;Résultat&gt; est une variable de type [Définition](../WDLang1/1000019540.md) et contient la définition du type des éléments.<br><br>![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Cette constante n'est pas disponible. |
| *tiNombreTotal* | &lt;Résultat&gt; est une variable de type Entier et contient le nombre total d'éléments dans la pile. |
| *tiTypeElément* | &lt;Résultat&gt; est une variable de type Entier et contient le type des éléments de la pile. Ce résultat est compatible avec la fonction [TypeVar](../WDLang1/3013056.md). |

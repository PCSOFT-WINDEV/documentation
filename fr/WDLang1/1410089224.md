
## KanbanInfoXY (Fonction)

***En anglais : KanbanInfoXY***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie pour une position donnée dans un champ Kanban (coordonnées d'un point du champ) :

- soit le numéro de la carte pour la position indiquée, 

- soit le numéro de la liste pour la position indiquée. 



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
IndiceListe est un entier = KanbanInfoXY(KANBAN_Planning, kbInfoListe, SourisPosX, SourisPosY)
Trace(KANBAN_Planning.Liste[IndiceListe].Titre)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = KanbanInfoXY(<Champ Kanban> , <Type d'information> , <X> , <Y>)`
---

**`<Résultat> : (Entier)`**

Information demandée : 

- Numéro de la carte pour la position indiquée, 

- Numéro de la liste de Kanban pour la position indiquée, 

- -1 si la position ne correspond pas à un élément du champ Kanban. 




**`<Champ Kanban> : (Nom de champ)`**

Nom du champ Kanban à manipuler. 
Si ce paramètre correspond à une chaîne vide (""), le champ manipulé est le champ auquel appartient l'événement en cours. 

**`<Type d'information> : (Constante de type Entier)`**

Type d'information demandée :


|   |   |
| --- | --- |
| *kbInfoCarte* | Indice de la carte présente à la position spécifiée. |
| *kbInfoListe* | Indice de la liste présente à la position spécifiée. |



**`<X> : (Entier)`**

Coordonnée X (en pixels) à analyser. Cette coordonnée est relative au champ.

**`<Y> : (Entier)`**

Coordonnée Y (en pixels) à analyser. Cette coordonnée est relative au champ.

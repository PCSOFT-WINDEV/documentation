
## PiloteInterrupteur (Fonction)

***En anglais : EmulateCheckBox***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Pilote la sélection d'une option dans un champ Interrupteur.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Sélection de la 2ème option de l'interrupteur
PiloteInterrupteur(INT_Interrupteur, 2)
```





<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`PiloteInterrupteur(<Champ> , <Numéro de l'option>)`
---

**`<Champ> : (Chaîne de caractères)`**

Nom du champ de type Interrupteur.

**`<Numéro de l'option> : (Entier)`**

Indice de l'option à sélectionner dans le champ.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Cette fonction doit être utilisée uniquement dans le code des tests automatiques.

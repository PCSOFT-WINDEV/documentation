
## Description de Syntaxe (Type de variable)

***En anglais : Syntax Description***
			
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Une variable de type **Description de Syntaxe** permet de décrire par programmation la syntaxe d'un état, d'une fenêtre ou d'une page. Les caractéristiques de ce type de variable peuvent être définies et modifiées à l'aide de différentes propriétés WLangage. 

Remarque : La description de la syntaxe est disponible dans la définition de l'état, de la fenêtre ou de la page. Cette définition peut être récupérée grâce à la fonction [RécupèreDéfinitionType](../WDLang1/1000019559.md).

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
Description_état est une Définition = RécupèreDéfinitionType(ETAT_CommandesClients)
Description_syntaxe est une Description de Syntaxe = Description_état.SyntaxeDéclaration
```





<a name="PROPS"></a>

## Propriétés


### Propriétés spécifiques à la description des variables de type Description de Syntaxe
<a name="proprietes_specifiques_description_des_variables_type_description_syntaxe_ELTPARAGRAPHE000031"></a>

Les propriétés suivantes peuvent être utilisées pour manipuler une variable de type **Description de Syntaxe** :

| Nom de la propriété | Type manipulé | Effet |
| --- | --- | --- |
| NombreParamètresVariable | Booléen | <br><br>- <u><u><u>Vrai</u></u></u> si le nombre de paramètres de la syntaxe est variable, <br><br>- <u><u><u>Faux</u></u></u> dans le cas contraire. <br><br><br><br><br>**Cette propriété est disponible en lecture seulement**. |
| Paramètre | Tableau de [Description de Paramètre](../WDLang1/1410089122.md) | Détail de chaque paramètre de la syntaxe. |

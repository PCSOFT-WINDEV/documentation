
## PlanningMode (Fonction)

***En anglais : SchedulerMode***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le mode d'affichage d'un champ Planning : jour, semaine, mois, ...

**Remarque** : La fonction [PlanningChangeMode](../WDLang1/1000019440.md) permet de modifier le mode d'affichage du champ Planning. 

![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) En mode Universal Windows 10 App, cette fonction permet uniquement de manipuler les champs Planning présents dans un état. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
nMode est un entier = PlanningMode(PLN_MonPlanning)
SI nMode = plnModeJournée ALORS
	PLN_MonPlanning..GranularitéRendezVous = 15 // 15 minutes en mode Jour
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = PlanningMode(<Champ Planning>)`
---

**`<Résultat> : (Entier)`**

Plage de temps affichée par le champ Planning : 


|   |   |
| --- | --- |
| *plnModeJournée* | Planning sur une journée. |
| *plnModeMois* | Planning sur un mois. |
| *plnModeNbJourPersonnalisé* | Planning sur un nombre de jours spécifique. |
| *plnModeSelonRdv* | Planning utilisant un nombre de jours permettant de voir tous les rendez-vous disponibles.<br><br> |
| *plnModeSemaine* | Planning sur une semaine de 5 jours ou sur une semaine de 7 jours. |



**`<Champ Planning> : (Nom de champ)`**

Nom du champ Planning à manipuler. Ce champ peut correspondre :

- au champ Planning d'une fenêtre,

- au champ Planning d'un état.




## Fonctions de recherche dans les champs d'une fenêtre
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000076"></a>
WINDEV offre la possibilité à l'utilisateur d'effectuer une recherche parmi tous les textes présents dans une fenêtre ou dans un champ. 

Il est possible d'effectuer cette recherche : 

- via la Fonctionnalité Automatique de vos Applications de recherche dans une fenêtre. Pour plus de détails, consultez [Recherche dans une fenêtre](../Editeurs/9500101.md). 

- via les fonctions de recherche WLangage. 




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Fonctions de recherche dans les champs
<a name="fonctions_recherche_dans_les_champs_ELTTEXTE000100"></a>
Les fonctions permettant de rechercher un texte dans les différents champs d'une fenêtre sont les suivantes : 



|   |   |
| --- | --- |
| [ChampRechercheSurligne](../WDLang1/1000020535.md) | Surligne dans une fenêtre ou dans un champ tous les libellés et textes correspondant à un texte donné. |
| [ChampRechercheTextePrécédent](../WDLang1/1000020765.md) | Recherche et sélectionne l'élément précédent correspondant au texte recherché dans une fenêtre ou un champ. |
| [ChampRechercheTextePremier](../WDLang1/1000020763.md) | Recherche et sélectionne le premier élément correspondant au texte recherché dans une fenêtre ou un champ. |
| [ChampRechercheTexteSuivant](../WDLang1/1000020764.md) | Recherche et sélectionne l'élément suivant correspondant au texte recherché dans une fenêtre ou un champ. |

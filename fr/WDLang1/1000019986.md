
## Fonctions de gestion des champs Organigramme (syntaxe préfixée)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions de gestion des champs Organigramme (via la syntaxe préfixée) sont les suivantes : 



|   |   |
| --- | --- |
| [&lt;Organigramme&gt;.Affiche](../WDLang1/1000019974.md) | Réinitialise le contenu d'un champ Organigramme à partir de sa source de données (fichier de données HFSQL par exemple). |
| [&lt;Organigramme&gt;.Ajoute](../WDLang1/1000019975.md) | Ajoute un nouvel élément "racine" dans un champ Organigramme. |
| [&lt;Organigramme&gt;.AjouteFils](../WDLang1/1000019976.md) | Ajoute un élément fils dans un champ Organigramme. |
| [&lt;Organigramme&gt;.InfoXY](../WDLang1/1000019977.md) | Renvoie l'indice de l'élément se trouvant à une position donnée dans le champ Organigramme. |
| [&lt;Organigramme&gt;.Modifie](../WDLang1/1000019978.md) | Modifie un élément dans un champ Organigramme. |
| [&lt;Organigramme&gt;.Occurrence](../WDLang1/1000019979.md) | Renvoie le nombre d'éléments présents dans un champ Organigramme. |
| [&lt;Organigramme&gt;.SauveImage](../WDLang1/1000019980.md) | Crée une image représentant la totalité d'un champ Organigramme. |
| [&lt;Organigramme&gt;.Select](../WDLang1/1000019981.md) | Renvoie l'indice de l'élément sélectionné dans un champ Organigramme. |
| [&lt;Organigramme&gt;.SelectMoins](../WDLang1/1000019982.md) | Dé-sélectionne un élément dans un champ Organigramme. |
| [&lt;Organigramme&gt;.SelectPlus](../WDLang1/1000019983.md) | Sélectionne un élément dans un champ Organigramme. |
| [&lt;Organigramme&gt;.Supprime](../WDLang1/1000019984.md) | Supprime un élément et l'ensemble de ses "fils" d'un champ Organigramme. |
| [&lt;Organigramme&gt;.SupprimeTout](../WDLang1/1000019985.md) | Supprime tous les éléments d'un champ Organigramme. |








- ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=WDOrganigramme.gif) ***Exemples didactiques (WINDEV)*** : **WD Organigramme** <br>Cet exemple permet de créer des organigrammes.<br>Un organigramme sert à indiquer la répartition des responsables à l'intérieur d'une entreprise, et les relations de commandement qui existent entre eux.

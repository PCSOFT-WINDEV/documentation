
## diagExtrémité (Type de variable)

***En anglais : diagHead***
			
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **diagExtrémité** permet de définir toutes les caractéristiques avancées d'une extrémité d'un lien entre 2 formes d'un diagramme. Les caractéristiques de cette extrémité peuvent être définies et modifiées à l'aide de différentes propriétés WLangage. 

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Crée une liaison entre rect1 et rect2
rect1 est un diagRectangle
rect2 est un diagRectangle
Lien est un diagLien

Lien.Départ.PointDAttache = rect1.PointDAttache(100, 50) 
Lien.Départ.Dessin = formeAucune
Lien.Arrivée.PointDAttache = rect2.PointDAttache(0, 50) 
Lien.Départ.Dessin = formeTriangle
```





<a name="PROPS"></a>

## Propriétés


### Propriétés spécifiques à la description des variables de type diagExtrémité
<a name="proprietes_specifiques_description_des_variables_type_diagextremite_ELTPARAGRAPHE000026"></a>Les propriétés suivantes peuvent être utilisées pour manipuler l'extrémité d'un lien :

| Nom de la propriété | Type manipulé | Effet |
| --- | --- | --- |
| Dessin | Constante de type Entier | Dessin de la flèche : <br><br>- *formeAucune* : L'extrémité n'a pas de forme particulière. <br><br>- *formeCarré* : L'extrémité est un carré. <br><br>- *formeCarréArrondi* : L'extrémité est un carré arrondi. <br><br>- *formeFlèche* : L'extrémité est une flèche. <br><br>- *formeLosange* : L'extrémité est un losange. <br><br>- *formeRond* : L'extrémité est un rond. <br><br>- *formeTriangle* : L'extrémité est un triangle. <br><br><br> |
| Destination | Constante de type Entier | Position de l'extrémité du lien sur la forme destination :  <br><br>- *lienDestinationAuto* : La position de l'extrémité du lien est déterminée automatiquement selon la position du lien et de la forme. <br><br>- *lienDestinationForme* : La position de l'extrémité du lien est situé au milieu de la forme. <br><br>- *lienDestinationPointDAttache* : La position de l'extrémité du lien correspond à un point d'attache. <br><br>- *lienDestinationRectangleEnglobant* : La position de l'extrémité du lien correspond à un point du rectangle englobant la forme. <br><br><br> |
| PointDAttache | [diagPointDAttache](../WDLang1/1410088457.md) | Point d'attache du lien. |
| X | Réel | Position de l'extrémité (X) dans le diagramme. |
| Y | Réel | Position de l'extrémité (Y) dans le diagramme. |

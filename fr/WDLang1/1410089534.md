


## KanbanFiltre (Fonction)

***En anglais : KanbanFilter***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Affiche uniquement les cartes correspondant au filtre spécifié. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Modification du champ SAI_Filtre
KanbanFiltre(KANBAN_Exemple, SAI_Filtre, ccSansCasse + ccSansAccent)
```


<a name="Exemple2"></a>

```wl
// Afficher les tâches sans intervenant
KanbanFiltre(KANBAN_Exemple, Proc_Filtre)
	PROCEDURE INTERNE Proc_Filtre(Carte est un kbCarte)
		SI Carte.Intervenant = "" ALORS
			RENVOYER Vrai
		FIN
	FIN
// Identique à 
// KanbanFiltre(KANBAN_Exemple, (Carte)=> { RENVOYER Carte.Intervervenant = "" } )
```


<a name="Exemple3"></a>

```wl
// Annuler le filtre
KanbanFiltre(KANBAN_Exemple, "")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Filtrer les cartes selon une chaîne de caractères contenue dans le titre

`KanbanFiltre(<Kanban> , <Filtre> [, <Option>])`
---

**`<Kanban> : (Nom de champ)`**

Nom du champ Kanban manipulé. 

**`<Filtre> : (Chaîne de caractères)`**

Filtre à utiliser. Les cartes sélectionnées seront celles dont le titre et/ou le contenu contient la chaîne &lt;Filtre&gt;. Si ce paramètre correspond à une chaîne vide (""), le filtre est annulé. 

**`<Option> : (Constante optionnelle de type Entier)`**

Options de comparaison : 


|   |   |
| --- | --- |
| *ccSansAccent* | Comparaison sans tenir compte des accents. |
| *ccSansCasse* | Comparaison sans tenir compte de la casse des caractères (majuscules/minuscules). |
| *MotComplet* | Comparaison du mot complet, c'est-à-dire entouré de caractères de ponctuation (caractère différent d'une lettre ou d'un chiffre) ou d'espaces. |




<a name="SYNTAXE2"></a>

### Filtrer les cartes selon une procédure de filtre

`KanbanFiltre(<Kanban> , <Procédure WLangage>)`
---

**`<Kanban> : (Nom de champ)`**

Nom du champ Kanban manipulé.

**`<Procédure WLangage> : (Nom de procédure)`**

Nom de la procédure WLangage (également nommée "callback") appelée pour filtrer les cartes affichées dans le champ Kanban. Pour plus de détails sur cette procédure, consultez [Paramètres de la procédure utilisée par la fonction KanbanFiltre](../WDLang1/1410089535.md).
Si ce paramètre correspond à une chaîne vide (""), le filtre est annulé. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Les cartes ajoutées après un appel à la fonction **KanbanFiltre** ne sont pas prises en compte par le filtre.

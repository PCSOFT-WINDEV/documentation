


## &lt;Variable WDPic&gt;.Rotation (Fonction)

***En anglais : &lt;WDPic variable&gt;.Rotation***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Effectue une rotation d'une image au format WDPic. Tous les calques de l'image effectuent la rotation. 

Remarque : Pour effectuer la rotation d'un seul calque, utilisez la fonction [&lt;Image&gt;.Rotation](../WDLang1/1000024338.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ouverture d'une image source
MonImageWDPic est un WDPic
MonImageWDPic.NomFichier = fRepExe() + ["\"] + "Source.png"

// Rotation de l'image
MonImageWDPic.Rotation(45)

// Sauvegarde du résultat
MonImageWDPic.Sauve(fRepExe() + ["\"] + "Resultat.png")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Effectuer une rotation simple

`<Résultat> = <Image WDPic>.Rotation(<Angle> [, <Options>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la rotation a été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Image WDPic> : (Variable de type WDPic)`**

Nom de la variable de type [WDPic](../WDLang1/1000023503.md) contenant l'image à manipuler.

**`<Angle> : (Réel)`**

Angle de rotation (en degrés). 

- Si l'angle est positif, l'image est tournée dans le sens horaire. 

- Si l'angle est négatif, l'image est tournée dans le sens anti-horaire.




**`<Options> : (Constante de type Entier)`**

Options de rotation : 


|   |   |
| --- | --- |
| *drAdapte* | L'image est tournée avec un agrandissement. Elle est ensuite réduite pour correspondre à la taille initiale de l'image. |
| *drDéfaut*<br>(valeur par défaut) | Agrandissement si nécessaire de l'image pour correspondre aux dimensions de l'image tournée. |
| *drSansAgrandissement* | L'image est tournée mais n'est pas agrandie : l'image peut être tronquée. |




<a name="SYNTAXE2"></a>

### Effectuer une rotation en modifiant le centre de rotation

`<Résultat> = <Image WDPic>.Rotation(<Angle> , <X> , <Y> [, <Options>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la rotation a été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Image WDPic> : (Variable de type WDPic)`**

Nom de la variable de type [WDPic](../WDLang1/1000023503.md) contenant l'image à manipuler.

**`<Angle> : (Réel)`**

Angle de rotation (en degrés). 

- Si l'angle est positif, l'image est tournée dans le sens horaire. 

- Si l'angle est négatif, l'image est tournée dans le sens anti-horaire.




**`<X> : (Réel)`**

Nouvelle abscisse du centre de rotation.

**`<Y> : (Réel)`**

Nouvelle ordonnée du centre de rotation.

**`<Options> : (Constante optionnelle de type Entier)`**

Options de rotation : 


|   |   |
| --- | --- |
| *drDéfaut*<br>(Valeur par défaut) | Agrandissement si nécessaire de l'image pour correspondre aux dimensions de l'image tournée. |
| *drSansAgrandissement* | L'image est tournée mais n'est pas agrandie. |

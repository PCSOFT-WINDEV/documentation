
## AppliParamètre (Fonction)

***En anglais : AppParameter***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie des informations sur le mécanisme de mise à jour automatique de l'application spécifiée.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple

![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) 
```wl
SI AppliParamètre(appMAJMode) = 3 ALORS
	AppliChangeParamètre(appMAJMsgArrêt, ...
			"L'application " + ProjetInfo(piNomEXE) + ...
			"va être arrêtée automatiquement.")
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = AppliParamètre(<Information> [, <Chemin de l'application>])`
---

**`<Résultat> : (Type de l'information)`**

Information demandée.

**`<Information> : (Constante de type Entier)`**

Caractéristique de l'application voulue :  


|   |   |
| --- | --- |
| *appMAJAfficheErreur* | Affichage ou non d'un message en cas de problème lors de la vérification des mises à jour de l'application : <br><br>- <u><u><u><u>Vrai</u></u></u></u> pour afficher un message, <br><br>- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. <br><br><br><br><br>![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Constante non disponible. |
| *appMAJChemin* | Chemin des informations de mise à jour sur le serveur.<br><br> |
| *appMAJDélai* | Délai entre deux messages (si appMAJMode = 2) ou avant l'arrêt automatique de l'application (si appMAJMode = 3)<br><br> |
| *appMAJMode* | Mode de fonctionnement actuel de l'application :<br><br>- 0 : Fonctionnement normal.<br><br>- 1 : Lancement interdit : les exécutables en cours continuent de fonctionner.<br><br>- 2 : Lancement interdit : les exécutables en cours reçoivent un message d'avertissement périodiquement.<br><br>- 3 : Lancement interdit : les exécutables en cours sont arrêtés automatiquement au bout d'un certain délai.<br><br><br><br><br> |
| *appMAJModeAffichage* | Mode d'affichage choisi pour prévenir l'utilisateur qu'une mise à jour de l'application est disponible :<br><br>- 0 : Une fenêtre dans l'application est utilisée pour avertir l'utilisateur.<br><br>- 1 : Une notification est utilisée pour avertir l'utilisateur.<br><br><br><br><br>![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Constante non disponible. |
| *appMAJMsgArrêt* | Message affiché avant l'arrêt automatique de l'application (quand appMAJMode = 3).<br><br> |
| *appMAJMsgAvertissement* | Message affiché périodiquement (quand appMAJMode = 2).<br><br> |
| *appMAJMsgInterdit* | Message affiché lorsque le lancement de l'application est interdit.<br><br> |
| *appMAJProtocole* | Protocole réseau utilisé pour la recherche des mises à jour. Ce protocole peut correspondre à "LOCAL" ou "HTTP".<br><br> |
| *appMAJRéseau* | Niveau de mise à jour :<br><br>- 0 : Pas de mise à jour.<br><br>- 1 : Mise à jour réseau activée.<br><br>- 2 : Mise à jour réseau activée et inscription des applications auprès du serveur de mise à jour.<br><br><br> |
| *appMAJServeur* | Serveur sur lequel sont recherchées les mises à jour. |
| *appMAJTempsAffichage* | Durée d'affichage des messages.<br><br> |
| *appMAJVerifAuto* | Activation de la vérification automatique des mises à jour : <br><br>- <u><u><u><u>Vrai</u></u></u></u> pour activer, <br><br>- <u><u><u><u>Faux</u></u></u></u> pour désactiver. <br><br><br><br><br>![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Constante non disponible. |
| *appMAJVersionCompatible* | Numéro de version minimale compatible avec la version de référence.<br><br> |
| *appMAJVersionLocale* | Numéro de version de l'exécutable local. |
| *appMAJVersionRéférence* | Numéro de version de la version de référence. |



**`<Chemin de l'application> : (Chaîne de caractères optionnelle)`**

Chemin complet de l'exécutable à manipuler. Si ce paramètre n'est pas précisé, l'application manipulée est l'application en cours.

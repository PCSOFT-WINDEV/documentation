<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le code ISO d'un pays correspondant au libellé du pays à la norme ISO 3166-1.

Remarque : Il est également possible d'obtenir le code ISO d'un pays correspondant à un code ISO de type différent (par exemple, le code ISO alphabétique sur 2 caractères correspondant à un code ISO numérique). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Code ISO France
sCodeISOFrance est une chaîne = DonneISOPays("France")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = DonneISOPays(<Libellé ou code ISO du pays> [, <Type du code ISO>])`
---

**`<Résultat> : (Chaîne de caractères)`**

Code ISO du pays.

**`<Libellé ou code ISO du pays> : (Chaîne de caractères)`**



- Libellé ISO du pays. 

- Code ISO du pays. Ce code ISO utilise un type de codage différent de celui recherché. 




**`<Type du code ISO> : (Constante optionnelle de type Entier)`**

Type de code ISO renvoyé :



|   |   |
| --- | --- |
| *lpCodeAlpha2* | Code ISO alphabétique sur 2 caractères. |
| *lpCodeAlpha3* | Code ISO alphabétique sur 3 caractères. |
| *lpCodeNumérique* | Code ISO numérique. |

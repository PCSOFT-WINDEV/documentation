


## &lt;Menu&gt;.AjouteSéparateur (Fonction)

***En anglais : &lt;Menu&gt;.AddSeparator***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ajoute un nouveau séparateur dans un menu (menu déroulant ou contextuel). Pour insérer un séparateur à un emplacement spécifique, utilisez la fonction [&lt;Menu&gt;.InsèreSéparateur](../WDLang1/1000020610.md).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 
```wl
// Ajoute un nouveau menu dans le menu MENU_MonMenu
MENU_MonMenu.AjouteMenu("OPT_Planning", "Planning")
// Ajoute une option dans le nouveau menu OPT_Planning
OPT_Planning.AjouteOption("OPT_Créer", "Créer", Créer_Planning)
// Associe une image à l'option de menu OPT_Créer
{"OPT_Créer", indChamp}..Image = "planning.png"
// Ajoute un séparateur
OPT_Planning.AjouteSéparateur()
// Ajoute une nouvelle option (OPT_Voir)
OPT_Planning.AjouteOption("OPT_Voir", "Voir", Voir_planning)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Menu d'origine>.AjouteSéparateur()`
---

**`<Résultat> : (Entier)`**

Position du séparateur dans le menu (déroulant ou contextuel). Si l'ajout du séparateur n'a pas été effectué, une erreur fatale est affichée.

**`<Menu d'origine> : (Nom de menu)`**

Nom du menu dans lequel le séparateur doit être ajouté. Le séparateur sera ajouté après la dernière option de ce menu. 
Remarque : Ce nom de menu peut correspondre à un menu contextuel.

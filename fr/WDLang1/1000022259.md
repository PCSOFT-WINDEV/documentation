
## Décode (Fonction)

***En anglais : Decode***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Décode une chaîne de caractères précédemment encodée avec la fonction [Encode](../WDLang1/1000022258.md).






<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Encodage d'une image en base64
Res est une chaîne = Encode(fChargeBuffer("MonImage.jpg"), encodeBASE64)

// ...

// Décodage de l'image
ImageDécodée est une Image = Décode(Res, encodeBASE64)
```



```wl
// Décodage d'un buffer reçu dans une requête HTTP
bufLu est un Buffer = retourHTTP.Contenu
<COMPILE SI ConfigurationUNICODE>
 	bufResultat = Décode(AnsiVersUnicode(bufLu), encodeBASE64) 
<SINON>
 	bufResultat = Décode(bufLu, encodeBASE64) 
<FIN>
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = Décode(<Elément à décoder> [, <Format d'encodage/décodage>])`
---

**`<Résultat> : (Chaîne de caractères ou Buffer)`**

Résultat du décodage.

**`<Elément à décoder> : (Chaîne de caractères)`**

Données à décoder. Ces données ont été encodées avec la fonction [Encode](../WDLang1/1000022258.md).
Attention : La chaîne à décoder doit être du même format que les chaînes par défaut du projet : ANSI ou UNICODE.

**`<Format d'encodage/décodage> : (Constante de type Entier)`**

Indique le type d'encodage effectué. Ce paramètre doit être identique à celui choisi lors du codage.


|   |   |
| --- | --- |
| *encodeBASE64*<br>(Valeur par défaut) | Format BASE 64 (respecte la norme RFC 4648). Le texte encodé comporte uniquement des caractères imprimables et les RC (Retour Chariot).<br><br> |
| *encodeBASE64SansRC* | Format BASE 64 (respecte la norme RFC 4648) sans les RC (Retour Chariot). Le texte encodé comporte uniquement des caractères imprimables.<br><br> |
| *encodeBASE64URL* | Format BASE 64 URL (respecte la norme RFC 4648). Le texte encodé comporte uniquement des caractères imprimables.<br><br> |
| *encodeBASE85* | Format BASE 85. Le texte encodé comporte uniquement des caractères imprimables.<br><br><br><br> |
| *encodePCS* | Format ASCII. Le texte encodé comporte uniquement des caractères imprimables.<br><br><br><br> |
| *encodeURLVersAnsi* | Format URL. Le texte encodé comporte uniquement des caractères autorisés dans une URL. Le texte décodé est une chaîne ANSI. Cette constante correspond à la constante *encodeURLDepuisAnsi* utilisée dans la fonction [Encode](../WDLang1/1000022258.md). <br><br> |
| *encodeURLVersUnicode* | Format URL. Le texte encodé comporte uniquement des caractères autorisés dans une URL. Le texte décodé est une chaîne Unicode. Cette constante correspond à la constante *encodeURLDepuisUnicode* utilisée dans la fonction [Encode](../WDLang1/1000022258.md). <br><br> |
| *encodeURLVersUTF8* | Format URL. Le texte encodé comporte uniquement des caractères autorisés dans une URL. Le texte décodé est une chaîne UTF-8. Cette constante correspond à la constante *encodeURLDepuisUTF8* utilisée dans la fonction [Encode](../WDLang1/1000022258.md). <br><br> |
| *encodeUUEncode* | Format UUEncode. Le texte encodé comporte uniquement des caractères imprimables.<br><br> |

<a name="NOTE0_1"></a>

## Remarques


### Format des chaînes de caractères à décoder
<a name="format_des_chaines_caracteres_decoder_ELTPARAGRAPHE000165"></a>La chaîne à décoder doit être du même format que les chaînes par défaut de la configuration de projet en cours : ANSI ou UNICODE.

Pour connaître ou modifier le format des chaînes par défaut : 

1. Affichez la fenêtre de description de la configuration en cours : sous le volet "Projet", dans le groupe "Configuration de projet", cliquez sur "Configuration courante". 

2. L'onglet "Unicode" permet de connaître le type de chaînes de caractères géré par défaut.




## SaisieSupprimeJeton (Fonction)

***En anglais : EditDeleteToken***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime un jeton dans un champ de saisie de type "Jetons de texte". 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Clic sur [Supprimer le robot de test] 
SaisieSupprimeJeton(SAI_Destinataire, "robotmail@adresse.test.fr")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Supprimer un jeton identifié par sa valeur

`<Résultat> = SaisieSupprimeJeton(<Champ de saisie> , <Valeur>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le jeton a pu être supprimé, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Champ de saisie> : (Nom de champ)`**

Nom du champ de saisie de type "Jetons de texte". 

**`<Valeur> : (Chaîne de caractères)`**

Valeur mémorisée du jeton à supprimer. Si plusieurs jetons ont la même valeur, seul le premier jeton sera supprimé.


<a name="SYNTAXE2"></a>

### Supprimer un jeton identifié par son indice

`<Résultat> = SaisieSupprimeJeton(<Champ de saisie> , <Indice>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le jeton a pu être supprimé, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Champ de saisie> : (Nom de champ)`**

Nom du champ de saisie de type "Jetons de texte". 

**`<Indice> : (Entier)`**

Indice du jeton à supprimer.


## RVBVersHTML (Fonction)

***En anglais : RGBToHTML***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la couleur HTML correspondant à la couleur RVB. 




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
RVBVersHTML(255) // Renvoie "#0000FF"
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = RVBVersHTML(<Couleur RVB>)`
---

**`<Résultat> : (Chaîne de caractères)`**

Couleur HTML exprimée en hexadécimal.



**`<Couleur RVB> : (Entier)`**

Couleur RVB à convertir en couleur HTML. Cette couleur peut correspondre à : 

- une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)),

- [une couleur prédéfinie du WLangage](../WDLang5/3010002.md).

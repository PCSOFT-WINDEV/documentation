
## EstPair (Fonction)

***En anglais : IsEven***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Identifie un nombre pair.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Nombre pair ?
ResPair = EstPair(66)     // Renvoie Vrai
ResPair = EstPair(6 + 13) // Renvoie Faux
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = EstPair(<Nombre>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le nombre spécifié est pair, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Nombre> : (Expression numérique entière ou entier)`**

Expression numérique entière ou entier à identifier.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Le chiffre zéro est considéré comme étant un chiffre pair.




## ArbreEnroule (Fonction)

***En anglais : TreeCollapse***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Enroule une branche précédemment déroulée. Les noeuds "fils" ne sont alors plus visibles dans le champ Arbre. Les noeuds fils restent dans leur état (enroulé ou déroulé).

Exemple de champ Arbre :
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Arbre_Vocabulaire.gif)
**Rappel** : En exécution, un noeud est enroulé lorsque le signe "+" précède le nom du noeud.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Enroule le noeud "Desserts" dans le champ Arbre "ARBRE_RecetteTV"
Res = ArbreEnroule(ARBRE_RecetteTV, "Recettes" + TAB + "Desserts")
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = ArbreEnroule(<Champ Arbre> , <Noeud à enrouler>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le noeud a été enroulé, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Champ Arbre> : (Nom de champ)`**

Nom du champ Arbre à manipuler.

**`<Noeud à enrouler> : (Chaîne de caractères)`**

Chemin du noeud à enrouler. Ce paramètre est de la forme :

```txt
"<Nom de la racine>" + TAB + ["<Nom du 1er noeud>" + TAB + ...
["<Nom du 2ème noeud>" + TAB + [...]]]"<Nom de la feuille>"
```
En cas de doublons sur un élément du chemin, ce chemin peut contenir l'identifiant de l'élément (indiqué à la suite du nom grâce à la fonction [ArbreID](../WDLang1/3018027.md)).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Noeud à enrouler
<a name="noeud_enrouler_ELTPARAGRAPHE000068"></a>Si le noeud à enrouler :

- est introuvable, la fonction **ArbreEnroule** renvoie <u><u><u><u>Faux</u></u></u></u>.

- est une feuille, la fonction **ArbreEnroule** renvoie <u><u><u><u>Vrai</u></u></u></u>.

- est déjà enroulé, la fonction **ArbreEnroule** renvoie <u><u><u><u>Vrai</u></u></u></u>.



<a name="NOTE0_2"></a>


### Bandeau de sélection
<a name="bandeau_selection_ELTPARAGRAPHE000102"></a>

- Si le bandeau de sélection était positionné sur un des éléments du noeud à enrouler, le bandeau se positionne sur le noeud enroulé après l'appel de la fonction **ArbreEnroule**.

- Le code de modification est appelé à chaque déplacement du bandeau de sélection. Si deux niveaux sont enroulés et si le bandeau de sélection se positionne sur le noeud supérieur, les codes de modification de chacun des niveaux sont exécutés.

- Le chemin complet de l'élément sélectionné peut être connu grâce à la fonction [ArbreSelect](../WDLang1/3018003.md).

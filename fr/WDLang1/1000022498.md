


## TDBRécupèreWidgetPosition (Fonction)

***En anglais : DashGetWidgetPosition***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la position d'un Widget présent dans un champ Tableau de bord.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Récupère la position du Widget courant en nombre de cases
(x,y) = TDBRécupèreWidgetPosition()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Récupérer la position du Widget en cours dans le champ Tableau de bord en cours

`(<X>, <Y>) = TDBRécupèreWidgetPosition([<Options>])`
---

**`<X> : (Entier)`**

Position horizontale du Widget en cours exprimée en nombre de cases ou en pixels. 
Le Widget manipulé correspond au champ auquel appartient l'événement en cours (Widget ou champ appartenant au Widget).

**`<Y> : (Entier)`**

Position verticale du Widget en cours exprimée en nombre de cases ou en pixels. 
Le Widget manipulé correspond au champ auquel appartient l'événement en cours (Widget ou champ appartenant au Widget).

**`<Options> : (Constante optionnelle de type Entier)`**

Unité utilisée :


|   |   |
| --- | --- |
| *tdbUnitéCase*<br>(Valeur par défaut) | L'unité utilisée pour la position du Widget est un nombre de cases. La première case est la case [1, 1]. |
| *tdbUnitéPixel* | L'unité utilisée pour la position du Widget est le pixel. |




<a name="SYNTAXE2"></a>

### Récupérer la position d'un Widget dans un champ Tableau de bord 

`(<X>, <Y>) = TDBRécupèreWidgetPosition(<Champ Tableau de bord> , <Indice du Widget> [, <Options>])`
---

**`<X> : (Entier)`**

Position horizontale du Widget en nombre de cases ou en pixels.

**`<Y> : (Entier)`**

Position verticale du Widget en nombre de cases ou en pixels.

**`<Champ Tableau de bord> : (Nom de champ)`**

Nom du champ Tableau de bord à manipuler.

**`<Indice du Widget> : (Entier)`**

Indice du Widget à manipuler. Cet indice correspond :

- au numéro d'ajout du Widget défini sous l'éditeur de fenêtres ou de pages.

- au numéro d'ajout du Widget renvoyé par la fonction [TDBAjouteWidget](../WDLang1/1000020953.md).




**`<Options> : (Constante optionnelle de type Entier)`**

Unité utilisée :


|   |   |
| --- | --- |
| *tdbUnitéCase* <br>(Valeur par défaut) | L'unité utilisée pour la position du Widget est un nombre de cases. La première case est la case [1, 1]. |
| *tdbUnitéPixel* | L'unité utilisée pour la position du Widget est le pixel. |

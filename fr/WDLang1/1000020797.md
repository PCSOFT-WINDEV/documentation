


## SablierRéseau (Fonction)

***En anglais : HourglassNetwork***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Affiche ou masque une jauge sans fin : 

- dans la barre de statut d'une application iOS. 

- dans la barre de titre des fenêtres d'une application Android.



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affichage de la jauge
SablierRéseau(Vrai)
// Traitement
...
Multitâche(-1)
...
// Masquage de la jauge
SablierRéseau(Faux)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`SablierRéseau(<Activation>)`
---

**`<Activation> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> pour activer la jauge sans fin, 

- <u><u><u><u>Faux</u></u></u></u> pour masquer la jauge sans fin.




## RubanFinModeDéplié (Fonction)

***En anglais : RibbonEndExpandedMode***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Replie un ruban enroulé et temporairement déplié. 

**Rappel** : La propriété [Enroulé](../Proprietes/2513014.md) permet d'enrouler un ruban. Dans ce cas, seuls les libellés des volets sont affichés (comme pour un menu classique). Le contenu du volet est affiché lorsque l'utilisateur clique sur le libellé du volet. Si l'utilisateur clique à nouveau sur le libellé du volet ou sur un bouton, le contenu du volet n'est plus affiché automatiquement.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Replie le ruban après sélection de la taille dans un champ Combo. 
// Sélection de COMBO_Taille
nTaille est un entier = COMBO_Taille[COMBO_Taille]
RTFSélection(SAI_TEXTERTF, rtfTaillePolice, nTaille)
RubanFinModeDéplié()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`RubanFinModeDéplié([<Champ Ruban>])`
---

**`<Champ Ruban> : (Nom de champ optionnel)`**

Nom du champ Ruban à manipuler. Si ce paramètre n'est pas spécifié, manipule le champ Ruban de la fenêtre en cours.




<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Si le ruban n'est pas enroulé cette fonction n'a aucun effet. 

- Si aucun ruban n'est présent dans la fenêtre en cours, la fonction n'a aucun effet.

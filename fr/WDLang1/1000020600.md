


## SélecteurSupprime (Fonction)

***En anglais : RadioButtonDelete***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime une option d'un champ Sélecteur. 




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Supprime l'option numéro 3 d'un champ Sélecteur 
SélecteurSupprime(SEL_Options, 3)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`SélecteurSupprime(<Champ Sélecteur> , <Indice>)`
---

**`<Champ Sélecteur> : (Nom de champ)`**

Nom du champ Sélecteur à manipuler.

**`<Indice> : (Entier)`**

Indice de l'option à supprimer. Cet indice doit être compris entre 1 et le nombre d'options contenues dans le champ Sélecteur. La propriété [Occurrence](../Proprietes/2510097.md) permet de connaître le nombre d'options d'un champ Sélecteur. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Lors de la suppression d'options dans un champ Sélecteur, la taille du champ n'est pas modifiée. Il est possible de modifier la taille du champ avec les propriétés [Largeur](../Proprietes/2510059.md) et [Hauteur](../Proprietes/2510050.md).

- Il est possible de compacter les options pour optimiser l'aspect du champ après suppression des options. 
	En mode compacté, les options d'un champ Sélecteur se resserrent : 

	- lorsqu'une option est rendue invisible. 

	- lorsqu'une option est supprimée.




- Pour utiliser le mode compacté : 

	- cochez l'option "Compacter les options invisibles" dans l'onglet "Détail" de la description du champ.

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) utilisez la propriété [CompacteOption](../Proprietes/1000020608.md). 




- Lors de la suppression d'une option, si le sélecteur utilise la numérotation des options, les indices des options suivant l'option supprimée sont automatiquement renumérotées. 

- Seul l'indice de l'option à supprimer peut être indiqué. Si le sélecteur utilise les valeurs renvoyées, il n'est pas possible de spécifier le nom de l'option à supprimer. Il est cependant possible d'indiquer l'indice correspondant à l'option.




## &lt;Arbre&gt;.RécupèreFils (Fonction)

***En anglais : &lt;TreeView&gt;.GiveChild***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie les "fils" d'un élément (niveau immédiatement inférieur) dans un champ Arbre.

Exemple de champ Arbre : <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Arbre_Vocabulaire.gif)

<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Premier "fils" du noeud "Desserts" dans le champ Arbre "ARBRE_RecetteTV"
Res = ARBRE_RecetteTV.RécupèreFils("Recettes" + TAB + "Desserts", aPremier)
// Res vaut : "Desserts Chauds"
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Champ Arbre>.RécupèreFils(<Chemin du noeud> , <Fils à récupérer>)`
---

**`<Résultat> : (Chaîne de caractères)`**



- Nom du "fils" trouvé, 

- Chaîne vide ("") si aucun "fils" n'est trouvé.




**`<Champ Arbre> : (Nom de champ)`**

Nom du champ Arbre à manipuler.

**`<Chemin du noeud> : (Chaîne de caractères)`**

Chemin complet du noeud dont les éléments "fils" sont recherchés. Ce paramètre :

- est de la forme : 
	
	```txt
	"<Nom de la racine>" + TAB + ["<Nom du 1er noeud>" + TAB + ...
	["<Nom du 2ème noeud>" + TAB + [...]]]"<Nom de la feuille>"
	```


- correspond à la constante *NULL* si les "fils" de la racine du champ Arbre doivent être renvoyés.




**`<Fils à récupérer> : (Constante de type Entier)`**

Indique le "fils" voulu : 



|   |   |
| --- | --- |
| *aPremier* | Premier élément fils. |
| *aSuivant* | Fils suivant le dernier fils récupéré. |




## &lt;Tableau croisé dynamique&gt;.SelectOccurrence (Fonction)

***En anglais : &lt;Pivot table&gt;.SelectCount***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le nombre de cellules sélectionnées dans un champ Tableau croisé dynamique.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Parcours des cellules sélectionnées
tcdPos est un tcdPosition de TCD_MonTCD

nbSel est un entier = TCD_MonTCD.SelectOccurrence()
POUR i = 1 A nbSel
	tcdPos = TCD_MonTCD.Select(i)
	Trace(tcdPos.COL_Date_Jour + ":" + tcdPos.COL_Produit))
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Champ Tableau croisé dynamique>.SelectOccurrence()`
---

**`<Résultat> : (Entier)`**

Nombre de cellules sélectionnées dans le champ Tableau croisé dynamique.

**`<Champ Tableau croisé dynamique> : (Nom de champ)`**

Nom du champ Tableau croisé dynamique à manipuler.

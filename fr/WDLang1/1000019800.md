


## &lt;TreeMap&gt;.SupprimeTout (Fonction)

***En anglais : &lt;TreeMap&gt;.DeleteAll***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime tous les éléments d'un champ TreeMap.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Supprime tous les éléments du champ TreeMap
TMAP_Vin.SupprimeTout()
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Champ TreeMap>.SupprimeTout()`
---

**`<Champ TreeMap> : (Nom de champ)`**

Nom du champ TreeMap manipulé.

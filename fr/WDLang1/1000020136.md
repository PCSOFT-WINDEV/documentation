
## RubanFermeRegroupement (Fonction)

***En anglais : RibbonCloseGrouping***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ferme le regroupement actuellement ouvert dans un champ Ruban. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI Taille(SAI_DansRuban) >= 3 ALORS
	RubanFermeRegroupement()
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`RubanFermeRegroupement()`
---



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Fonctionnement des regroupements
<a name="fonctionnement_des_regroupements_ELTPARAGRAPHE000030"></a>Dans un champ Ruban, les regroupements trop petits n'affichent qu'une icône et une flèche pour ouvrir une popup qui contient tous les champs.

Par défaut, cette popup est refermée : 

- si l'utilisateur clique sur un bouton, 

- si l'utilisateur effectue une sélection dans un champ Combo.




Pour fermer le regroupement sur une autre action (par exemple après la saisie de 3 caractères, ...), utilisez la fonction **RubanFermeRegroupement** dans le traitement voulu. 

**Remarque** : Si aucune popup n'est ouverte, la fonction **RubanFermeRegroupement** ne fait rien et ne renvoie pas d'erreur.

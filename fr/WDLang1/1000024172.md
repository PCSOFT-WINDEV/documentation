


## DocSupprime (Fonction)

***En anglais : DocDelete***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime un élément d'un document. Cet élément est supprimé du document auquel il est associé. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Supprime les éléments de type Image dans un document
MonDoc est un Document
POUR TOUT MonPara DE MonDoc.Paragraphe
	POUR TOUT MonElement DE MonPara.Elément
			SI  MonElement.Type = typeDocElémentImage ALORS
				
				DocSupprime(MonElement)
			FIN	
	FIN
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`DocSupprime(<Elément>)`
---

**`<Elément> : (Variable de type docElément)`**

Nom de la variable de type [docElément](../WDLang1/1000022484.md) correspondant à l'élément à supprimer.

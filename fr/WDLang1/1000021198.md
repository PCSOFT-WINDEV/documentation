
## PiloteVolet (Fonction)

***En anglais : EmulatePane***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Pilote la sélection d'un volet d'un champ Onglet, Ruban ou Boîte à outils.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Sélection du 5ème volet du champ Onglet ONG_Onglet
PiloteVolet(ONG_Onglet, 5)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`PiloteVolet(<Champ> , <Numéro du volet>)`
---

**`<Champ> : (Chaîne de caractères)`**

Nom du champ à manipuler. Ce champ doit être de type Onglet, Ruban ou Boîte à Outils.

**`<Numéro du volet> : (Entier)`**

Indice du volet à sélectionner.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Cette fonction doit être utilisée uniquement dans le code des tests automatiques.

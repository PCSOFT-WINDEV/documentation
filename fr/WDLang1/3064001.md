
## EnumèreElément (Fonction)

***En anglais : EnumElement***

<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet d'énumérer les éléments du projet : fenêtres, pages, états, requêtes, ...
Utilisée dans une boucle, cette fonction peut permettre d'énumérer les éléments d'un projet.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple
<a class="notetitle" target="_blank" href="$DOC$=1000003064001&name=enumereelement_fonction&product=WD">Voir des exemples supplémentaires</a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) 
```wl
// Énumération des fenêtres d'un projet
NomFenêtre est une chaîne = EnumèreElément(enumFenêtre)
// Parcours de toutes les fenêtres
TANTQUE NomFenêtre <> ""
	// Passage à la fenêtre suivante
	NomFenêtre = EnumèreElément()
FIN
```

## Syntaxe

`<Résultat> = EnumèreElément([<Objet à énumérer> [, <Nom du composant>]])`
---

**`<Résultat> : (Chaîne de caractères)`**



- Nom de l'élément trouvé, 

- Chaîne vide ("") si aucun élément n'a été trouvé.




**`<Objet à énumérer> : (Constante optionnelle)`**

Paramètre à spécifier au premier appel à la fonction (pour lancer l'énumération). Pour continuer l'énumération, ce paramètre n'est pas nécessaire.


|   |   |
| --- | --- |
| *enumComposantExterne* | Énumère les éléments publics d'un composant externe intégré au projet. Le nom du composant est spécifié dans le paramètre &lt;Nom du composant&gt;. |
| *enumEtat* | Énumère tous les états du projet. |
| *enumFenêtre* | Énumère toutes les fenêtres du projet. |
| *enumFenêtreInterne* | Énumère toutes les fenêtres internes du projet. |
| *enumGPU* | Constante réservée pour le Groupware utilisateur. |
| *enumModèleEtat* | Énumère tous les modèles d'états du projet. |
| *enumModèleFenêtre* | Énumère tous les modèles de fenêtres du projet. |
| *enumModèlePage* | Enumère tous les modèles de pages du projet. |
| *enumPage* | Énumère toutes les pages du projet. |
| *enumPageDynamique* | Énumère toutes les pages dynamiques du projet. |
| *enumPageInterne* | Énumère toutes les pages internes du projet. |
| *enumRequête* | Énumère toutes les requêtes du projet. |
| *enumTout* | Énumère tous les éléments du projet. |





![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Seules les constantes *enumFenêtre*, *enumRequête* et *enumTout* sont disponibles.

**`<Nom du composant> : (Chaîne de caractères)`**

Nom du composant pour lequel les éléments publics doivent être énumérés. 

![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Ce paramètre n'est pas disponible.

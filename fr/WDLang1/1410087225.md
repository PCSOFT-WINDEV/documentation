


## CaméraPhoto (Fonction)

***En anglais : CameraPhoto***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Prend une photo via un champ Caméra dans une application Android ou iOS. La photo est enregistrée dans le répertoire par défaut de l'application, ou dans le répertoire spécifié avec la propriété [RépertoireDeTravail](../Proprietes/1000026169.md). Les différentes options précisées pour le champ Caméra (flash, caméra dorsale ou frontale, ...) sont prises en compte. 

Remarque : Cette fonction est utilisable uniquement avec un champ Caméra en version 27 ou supérieure (le champ ne doit pas avoir l'option "Mode compatible 26" cochée dans l'onglet "Général" de la fenêtre de description du champ). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
CaméraPhoto(CAM_MaCaméra, CaméraPhoto_Callback)

PROCÉDURE INTERNE CaméraPhoto_Callback(bRéussite est un booléen, CheminPhoto est une chaîne)
	SI bRéussite = Vrai ALORS
		IMG_PHOTO = CheminPhoto
		LIB_CHEMIN_PHOTO = CheminPhoto
		MaFenêtre.Plan = 3
	SINON
		ToastAffiche("Erreur lors de la prise de photo." + RC + ErreurInfo())
	FIN
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`CaméraPhoto(<Champ Caméra> , <Procédure WLangage>)`
---

**`<Champ Caméra> : (Nom de champ)`**

Nom du champ Caméra qui va prendre la photo. 

**`<Procédure WLangage> : (Nom de procédure)`**

Nom de la procédure WLangage (également nommée "callback") appelée lorsque la photo est enregistrée.
Pour plus de détails sur cette procédure, consultez [Paramètres de la procédure utilisée par la fonction CaméraPhoto](../WDLang1/1410087851.md).

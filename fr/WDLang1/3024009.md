


## Décompresse (Fonction)

***En anglais : Uncompress***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Décompresse un bloc mémoire (buffer) préalablement compressé par la fonction [Compresse](../WDLang1/3024012.md).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Compresse puis décompresse une chaîne
Comp est un Buffer = Compresse("Chaîne à compresser")
Info(Décompresse(Comp))
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = Décompresse(<Buffer compressé>)`
---

**`<Résultat> : (Buffer)`**



- Bloc mémoire décompressé, 

- Chaîne vide ("") en cas d'erreur. Le compte-rendu détaillé de l'erreur est connu grâce à la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Buffer compressé> : (Buffer)`**

Buffer compressé obtenu avec la fonction [Compresse](../WDLang1/3024012.md).

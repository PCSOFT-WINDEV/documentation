


## VidéoListeDriver (Fonction)

***En anglais : VideoListDriver***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Liste les drivers de capture vidéo installés sur le poste en cours.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connaître les drivers Vidéo installés
// Affichage dans le champ LISTE_ListeDriver
ListeAjoute(LISTE_ListeDriver, VidéoListeDriver(viDirectShow))
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = VidéoListeDriver([<Technologie utilisée>])`
---

**`<Résultat> : (Chaîne de caractères)`**

Liste des drivers vidéo installés sur le poste, sous la forme :

```txt
<Nom du driver 1> + TAB + <Version du driver 1> + RC + ... 
<Nom du driver N> + TAB + <Version du driver N>
```


Où : 

- &lt;Nom du driver 1&gt; ... &lt;Nom du driver N&gt; correspondent au nom des drivers vidéos, 

- &lt;Version du driver 1&gt; ... &lt;Version du driver N&gt; correspondent à la version du driver.




**`<Technologie utilisée> : (Constante de type Entier)`**

Type de technologie utilisée pour lister les drivers : 


|   |   |
| --- | --- |
| *viDirectShow* | Technologie utilisée : Direct Show. <br><br>Cette constante doit être utilisée : <br><br>- pour lister des drivers correspondant aux différentes caméras installées (par exemple sur une tablette). <br><br>- pour sélectionner un driver utilisé dans un champ Conférence (fonction [FluxChangeDriver](../WDLang3/1000021613.md)).<br><br><br> |
| *viVideoForWindows*<br>(Valeur par défaut) | Technologie utilisée : Video for Windows |






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Pour afficher la liste des drivers vidéo dans un champ Liste ou un champ Combo, utilisez la syntaxe : 
	
	```txt
	ListeAjoute(<Nom du champ Liste>, <Résultat>)
	```

	ou
	
	```txt
	ListeAjoute(<Nom du champ Liste>, VidéoListeDriver())
	```


- Cette fonction est équivalente à la fonction [CaméraListeDriver](../WDLang1/1000022457.md).

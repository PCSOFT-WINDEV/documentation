
## Fonctions Calendrier (syntaxe préfixée)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Ces fonctions permettent de manipuler les champs Calendrier ainsi que les champs de saisie de type Date via une syntaxe préfixée :



|   |   |
| --- | --- |
| [&lt;Calendrier&gt;.Position](../WDLang1/1000017361.md) | Renvoie le mois affiché dans un champ Calendrier. |
| [&lt;Calendrier&gt;.Select](../WDLang1/1000017385.md) | Renvoie la date sélectionnée dans un champ Calendrier. |
| [&lt;Calendrier&gt;.SelectMoins](../WDLang1/1000017363.md) | Désélectionne une date dans un champ Calendrier multisélection. |
| [&lt;Calendrier&gt;.SelectOccurrence](../WDLang1/1000017362.md) | Renvoie le nombre de jours sélectionnés dans un champ Calendrier. |
| [&lt;Calendrier&gt;.SelectPlus](../WDLang1/1000017343.md) | Sélectionne une date dans un champ Calendrier. |
| [&lt;Saisie&gt;.CalendrierOuvre](../WDLang1/1000017423.md) | Dans un champ de saisie de type Date, ouvre une fenêtre popup affichant un calendrier. |

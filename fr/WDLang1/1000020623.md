


## TCDDérouleTout (Fonction)

***En anglais : PVTExpandAll***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Déroule tous les entêtes de lignes et/ou de colonnes d'un champ Tableau croisé dynamique. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Charge le fichier des données
TCDCharge(TCD_Statistiques, sFichier)
// Déroule tous les entêtes du champ Tableau croisé dynamique
TCDDérouleTout(TCD_Statistiques)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`TCDDérouleTout(<Champ Tableau croisé dynamique> [, <Type de l'entête>])`
---

**`<Champ Tableau croisé dynamique> : (Nom de champ)`**

Nom du champ Tableau croisé dynamique à manipuler. Ce champ Tableau croisé dynamique doit avoir été rempli soit avec la fonction [TCDCalculeTout](../WDLang1/1000020539.md), soit avec la fonction [TCDCharge](../WDLang1/1000020507.md). 

**`<Type de l'entête> : (Constante ou combinaison de constantes optionnelle de type Entier)`**

Type de l'entête à dérouler : 


|   |   |
| --- | --- |
| *tcdEntêteColonne* | Déroule les entêtes de colonnes. |
| *tcdEntêteLigne* | Déroule les entêtes de lignes. |


Par défaut, si ce paramètre n'est pas spécifié, les entêtes de lignes **ET** de colonnes sont déroulés.

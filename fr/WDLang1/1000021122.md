


## &lt;Diagramme de Gantt&gt;.ChangeMode (Fonction)

***En anglais : &lt;Gantt chart&gt;.ChangeMode***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie la période (c'est-à-dire le nombre de jours) affichée dans une colonne d'un champ Table (ou Table hiérarchique) de type Diagramme de Gantt.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Clic sur BTN_Semaine
COL_Gantt.ChangeMode(ganttModeEnJour)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Colonne Gantt>.ChangeMode(<Mode d'affichage> [, <Nombre de jours>])`
---

**`<Colonne Gantt> : (Nom de champ)`**

Nom de la colonne (d'un champ Table ou Table hiérarchique) de type Diagramme de Gantt à manipuler.

**`<Mode d'affichage> : (Constante de type Entier)`**

Plage de temps à afficher dans la colonne de type Diagramme de Gantt : 


|   |   |
| --- | --- |
| *ganttModeEnJour* | Affiche le diagramme de Gantt en fonction de la largeur de la colonne. Dans ce cas, il est nécessaire de spécifier le nombre de jours à afficher. |
| *ganttModeSelonTâche* | Affiche le diagramme de Gantt de la première à la dernière tâche. |



**`<Nombre de jours> : (Entier optionnel)`**

Nombre de jours affichés. Ce paramètre est pris en compte uniquement si le paramètre &lt;Mode d'affichage&gt; correspond à la constante *ganttModeEnJour*. Dans le cas contraire, une erreur WLangage est affichée.

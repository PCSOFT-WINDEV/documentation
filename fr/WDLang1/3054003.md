
## ChargeErreur (Fonction)

***En anglais : LoadError***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Charge en mémoire le fichier contenant les messages d'erreur dans la langue souhaitée. Ce fichier doit avoir été créé avec l'utilitaire de traduction WDINT.

Remarque : Ce fichier contient également la traduction des Fonctionnalités Automatiques de votre Application (FAA) et les interfaces des champs avancés (champ Traitement de texte, champ Tableur, champ Editeur d'images, ...). 

Cette fonction doit être appelée dans le code d'initialisation du projet afin d'afficher les messages d'erreur générés par les DLL de WINDEV, WINDEV Mobile et WEBDEV dans la langue voulue.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI ChargeErreur("ALLEMAND.WDM") > 0 ALORS
	Erreur("Fichier des messages en allemand non trouvé", ...
		"Les messages seront affichés en français")
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = ChargeErreur(<Nom du fichier WDM>)`
---

**`<Résultat> : (Entier)`**

compte-rendu du chargement du fichier des erreurs :

- 0 si le chargement s'est correctement effectué,

- 1 si &lt;Nom du fichier WDM&gt; n'a pas été trouvé,

- 2 si &lt;Nom du fichier WDM&gt; a été trouvé mais n'a pas la structure attendue,

- 3 si &lt;Nom du fichier WDM&gt; n'a pas pu être initialisé,

- 4 si &lt;Nom du fichier WDM&gt; n'est pas une bibliothèque,

- 5 si &lt;Nom du fichier WDM&gt; n'a pas pu être chargé en mémoire.




**`<Nom du fichier WDM> : (Chaîne de caractères)`**

Nom du fichier des messages d'erreur à utiliser (d'extension .WDM). Ce nom peut être complet ou relatif. Ce fichier doit avoir été créé avec l'utilitaire de traduction des messages WDINT (outil non fourni en standard avec WINDEV, WINDEV Mobile ou WEBDEV).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Si la langue "Anglais" (par exemple) est choisie dans l'éditeur de fenêtres (ou de pages), tous les programmes exécutables créés avec WINDEV/WINDEV Mobile (ou les sites créés avec WEBDEV) s'exécuteront en anglais. Pour cela, la traduction en anglais des différents éléments de l'application (ou du site) doit avoir été saisie dans les différents éditeurs. La fonction [Nation](../WDLang1/3054014.md) permet de modifier la langue d'exécution de l'application par programmation.

- La fonction **ChargeErreur** ne peut pas être utilisée pour charger des fichiers autres que ceux créés avec WDINT.EXE.

- Par défaut, si aucun fichier n'est chargé avec la fonction **ChargeErreur**, les messages d'erreur seront affichés en français.

- La fonction **ChargeErreur** est indépendante de la fonction [Nation](../WDLang1/3054014.md). Si un fichier d'erreur est disponible pour chaque langue gérée par l'application (ou le site), il est nécessaire de recharger ce fichier WDM après chaque changement de langue effectué avec la fonction [Nation](../WDLang1/3054014.md).




## Fonctions de gestion des files, des piles et des listes
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions suivantes permettent de gérer les types File, Pile et Liste :



|   |   |
| --- | --- |
| [Défile](../WDLang1/1000018322.md) | Défile un élément d'une file. |
| [Dépile](../WDLang1/1000017372.md) | Dépile un élément d'une pile. |
| [Empile](../WDLang1/1000017366.md) | Empile un élément dans une pile. |
| [Enfile](../WDLang1/1000018323.md) | Enfile un élément dans une file. |
| [FileInfo](../WDLang1/1000019547.md) | Récupère les caractéristiques d'une file : types des éléments et nombre d'éléments. |
| [ListeInfo](../WDLang1/1000019548.md) | Récupère les caractéristiques d'une liste WLangage : types des éléments et nombre d'éléments. |
| [PileInfo](../WDLang1/1000019546.md) | Récupère les caractéristiques d'une pile : types des éléments et nombre d'éléments. |

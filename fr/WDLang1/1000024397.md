


## &lt;Traitement de texte&gt;.Undo (Fonction)

***En anglais : &lt;Word Processing&gt;.Undo***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Annule la dernière modification effectuée sur un champ Traitement de texte.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Annule la dernière action effectuée sur le champ
TT_MonDocument.Undo()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Champ Traitement de texte>.Undo()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la dernière modification a été annulée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Champ Traitement de texte> : (Nom de champ)`**

Nom du champ Traitement de texte à manipuler.

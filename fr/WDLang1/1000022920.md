


## &lt;Planning&gt;.PositionDateHeure (Fonction)

***En anglais : &lt;Scheduler&gt;.PositionDateTime***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Affiche un champ Planning à partir d'une date spécifiée ou renvoie la date du premier jour (ou dernier jour) affiché dans un champ Planning. 

![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) En mode Universal Windows 10 App, cette fonction permet uniquement de manipuler les champs Planning présents dans un état. 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Positionne le champ Planning sur la date du jour
PLN_Planning.PositionDateHeure(DateDuJour())

// Positionne le champ Planning au 20 décembre 2018
PLN_Planning.PositionDateHeure("20181220")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Positionner le planning sur une date spécifiée

`<Champ Planning>.PositionDateHeure(<Date de début>)`
---

**`<Champ Planning> : (Nom de champ)`**

Nom du champ Planning à manipuler. Ce champ peut correspondre : 

- au champ Planning d'une fenêtre, 

- au champ Planning d'un état.




**`<Date de début> : (Chaîne de caractères ou variable de type Date)`**

Date de début de l'affichage.


<a name="SYNTAXE2"></a>

### Connaître la première date ou la dernière date affichée par le champ Planning

`<Résultat> = <Champ Planning>.PositionDateHeure([<Date recherchée>])`
---

**`<Résultat> : (Chaîne de caractères, variable de type Date ou DateHeure)`**

Date (ou Date et Heure) affichée par le champ Planning.

**`<Champ Planning> : (Nom de champ)`**

Nom du champ Planning à manipuler. Ce champ peut correspondre : 

- au champ Planning d'une fenêtre, 

- au champ Planning d'un état.




**`<Date recherchée> : (Constante optionnelle de type Entier)`**

Type de date recherchée : 


|   |   |
| --- | --- |
| *agDateHeure1erVisible* | Première date et heure affichée dans le champ Planning (en fonction de la position de l'ascenseur).<br><br> |
| *plnDateDébut* <br>(Valeur par défaut) | Première date affichée dans le champ Planning. |
| *plnDateFin* | Dernière date affichée dans le champ Planning. |
| *plnMoisEnCours* | Premier jour du mois affiché dans le champ Planning. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Positionnement dans le champ Planning
<a name="positionnement_dans_champ_planning_ELTPARAGRAPHE000253"></a>Lors de l'utilisation de la fonction **&lt;Planning&gt;.PositionDateHeure** pour afficher une date, le champ Planning conserve vos préférences d'affichage. 

Par exemple, si le champ Planning est affiché à la semaine commençant un Lundi, la fonction **&lt;Planning&gt;.PositionDateHeure** utilisée sur la date du vendredi affiche le planning du Lundi au Dimanche pour la semaine contenant le Vendredi spécifié.

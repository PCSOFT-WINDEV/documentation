
## ArcCosinus (Fonction)

***En anglais : ArcCos***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie l'arc cosinus d'une valeur numérique (entière ou réelle).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Récupération de l'arc cosinus de 0.5
ResArcCosinus = ArcCosinus(0.5)
// Renvoie 60
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = ArcCosinus(<Valeur numérique>)`
---

**`<Résultat> : (Réel)`**

Arc cosinus (angle en degrés du cosinus) de la valeur numérique spécifiée, compris dans l'intervalle [0,180].

**`<Valeur numérique> : (Entier ou réel)`**

Valeur numérique (cosinus) à manipuler. Cette valeur doit être comprise dans l'intervalle [-1,1]. Dans le cas contraire, une erreur du WLangage est générée.

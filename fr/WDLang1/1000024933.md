
## ContinentRécupère (Fonction)

***En anglais : ContinentGet***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la variable de type [Continent](../WDLang1/1000024919.md) correspondant à une des valeurs suivantes : 

- une constante continent (code continent sur 2 caractères), 

- un libellé de continent.





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Récupération des informations sur l'Europe
oContinent est un Continent = ContinentRécupère(continentEurope)
// Affichage du libellé 
LIB_Continent = oContinent.Nom
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = ContinentRécupère([<Libellé, constante ou code du continent>])`
---

**`<Résultat> : (Variable de type Continent)`**

Variable de type [Continent](../WDLang1/1000024919.md) correspondant au libellé ou au code de continent. 

**`<Libellé, constante ou code du continent> : (Chaîne de caractères optionnelle)`**



- Libellé du continent recherché (en français, anglais ou espagnol). 

- Constante identifiant le continent (ou code continent sur 2 caractères) :
	


|   |   |
| --- | --- |
| *continentAfrique* | Afrique (la constante vaut "AF"). |
| *continentAmériqueNord* | Amérique du Nord (la constante vaut "NA"). |
| *continentAmériqueSud* | Amérique du Sud (la constante vaut "SA"). |
| *continentAntarctique* | Antarctique (la constante vaut "AN"). |
| *continentAsie* | Asie (la constante vaut "AS"). |
| *continentEurope* | Europe (la constante vaut "EU"). |
| *continentOcéanie* | Océanie (la constante vaut "OC"). |





Si ce paramètre n'est pas précisé, le continent sera le celui de la machine.

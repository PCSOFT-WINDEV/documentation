| **aErreur** | État d'un noeud / Type d'un élément : noeud ou élément inexistant. | [ArbreEtat](../WDLang1/3018015.md), [ArbreTypeElément](../WDLang1/3018014.md) |
| **aFeuille** | Type d'un élément : élément feuille. | [ArbreTypeElément](../WDLang1/3018014.md), [ArbreCherche](../WDLang1/3018030.md) |

| **aizBouton** | Type de la zone survolée : Bouton "+ / -" situé devant le libellé de l'élément de l'arbre. | [ArbreInfoXY](../WDLang1/3018026.md) |
| **aizCoche** | Type de la zone survolée : Coche de l'élément. | [ArbreInfoXY](../WDLang1/3018026.md) |
| **aizDroite** | Type de la zone survolée : Espace à droite du libellé de l'élément. | [ArbreInfoXY](../WDLang1/3018026.md) |
| **aizEnDehors** | Type de la zone survolée : Espace situé sous les éléments de l'arbre. | [ArbreInfoXY](../WDLang1/3018026.md) |
| **aizGauche** | Type de la zone survolée : Espace situé à gauche du libellé de l'élément. | [ArbreInfoXY](../WDLang1/3018026.md) |

| **aizLibellé** | Type de la zone survolée : Libellé de l'élément. | [ArbreInfoXY](../WDLang1/3018026.md) |
| **aiZone** | Type d'information demandée : Type de la zone survolée. | [ArbreInfoXY](../WDLang1/3018026.md) |

| **aSuivant** | Indique le "fils" voulu : fils suivant. | [ArbreRécupèreFils](../WDLang1/3018010.md) |
| **aTout** | Niveau des éléments à prendre en compte : tous les "fils" sont pris en compte. | [ArbreOccurrence](../WDLang1/3018009.md) |
| **aTriAlpha** | Position de la feuille par rapport aux éléments du même niveau : feuille insérée ou déplacée suivant l'ordre alphabétique. | [ArbreAjoute](../WDLang1/3018012.md), [ArbreDéplaceElément](../WDLang1/3018024.md) |
| **aUnNiveau** | Niveau des éléments à prendre en compte : seuls les "fils" immédiats sont pris en compte. | [ArbreOccurrence](../WDLang1/3018009.md), [ArbreCherche](../WDLang1/3018030.md) |

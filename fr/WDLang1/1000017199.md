
## VersPressePapierRTF (Fonction)

***En anglais : ToClipboardRTF***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Écrit une chaîne de caractères au format RTF dans le presse-papiers.

**Remarque** : Le contenu du presse-papiers au format RTF pourra être récupéré avec la fonction [PressePapierRTF](../WDLang1/1000017200.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Copie le contenu du champ SAI_RTF dans le presse-papiers au format RTF
VersPressePapierRTF(SAI_RTF)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = VersPressePapierRTF(<Texte ou champ au format RTF>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le texte a été copié dans le presse-papiers, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Texte ou champ au format RTF> : (Chaîne de caractères)`**

Texte RTF à écrire dans le presse-papiers ou nom du champ contenant le texte au format RTF.




## &lt;Liste de Kanban&gt;.SupprimeToutCarte (Fonction)

***En anglais : &lt;Kanban list&gt;.DeleteAllCard***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime toutes les cartes d'un champ Kanban ou d'un champ Liste de Kanban.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Supprime toutes les cartes de la liste LISTE_A_Faire
LISTE_A_Faire.SupprimeToutCarte()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Champ manipulé>.SupprimeToutCarte()`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom d'un champ Liste de Kanban.

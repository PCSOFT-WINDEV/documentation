


## Fonctions de gestion du planificateur Windows
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


Les fonctions permettant de gérer le planificateur Windows sont les suivantes :



|   |   |
| --- | --- |
| [PlanificateurAjouteHoraire](../WDLang1/3089001.md) | Crée un horaire pour une tâche planifiée. |
| [PlanificateurAjouteTâche](../WDLang1/3089003.md) | Crée une tâche dans le planificateur de tâches Windows (si cette tâche n'existe pas déjà). |
| [PlanificateurEtatTâche](../WDLang1/3089005.md) | Récupère ou modifie l'état d'une tâche créée dans le planificateur de tâches Windows. |
| [PlanificateurListeTâche](../WDLang1/3089006.md) | Récupère la liste des tâches planifiées. |
| [PlanificateurModifieHoraire](../WDLang1/3089007.md) | Modifie les paramètres d'un horaire de tâche planifiée, c'est-à-dire les paramètres qui vont définir l'exécution d'une tâche. |
| [PlanificateurModifieTâche](../WDLang1/3089008.md) | Modifie les paramètres d'une tâche planifiée. |
| [PlanificateurOuvrePropriétésTâche](../WDLang1/3089009.md) | Ouvre la page de propriétés d'une tâche planifiée. |
| [PlanificateurPropriétésHoraire](../WDLang1/3089010.md) | Lit les propriétés d'un horaire de tâche planifiée et met à jour la [structure HoraireTâchePlanifiée](../WDLang1/3089002.md). |
| [PlanificateurPropriétésTâche](../WDLang1/3089011.md) | Lit les propriétés d'une tâche planifiée et met à jour la [structure TâchePlanifiée](../WDLang1/3089004.md). |
| [PlanificateurRAZ](../WDLang1/3089012.md) | Remplit les structures [TâchePlanifiée](../WDLang1/3089004.md) et/ou [HoraireTâchePlanifiée](../WDLang1/3089002.md) avec les valeurs par défaut. |
| [PlanificateurSupprimeHoraire](../WDLang1/3089013.md) | Supprime un horaire d'une tâche planifiée. |
| [PlanificateurSupprimeTâche](../WDLang1/3089014.md) | Supprime une tâche planifiée. |






- ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=LesfonctionsTachesPlanifiees.gif) ***Exemples unitaires (WINDEV)*** : **Les fonctions Tâches Planifiées** <br>Utilisation des fonctions PlanificateurXXX. <br>Vous pouvez voir la liste des tâches planifiées, en ajouter, les modifier.<br><br>Vous pouvez également planifier les horaires, ainsi que différents paramètres de la tâche planifiée comme des options d'alimentation, ou encore des options de fin d'exécution de la tâche planifiée.




## &lt;TimeLine&gt;.ListeEvénement (Fonction)

***En anglais : &lt;TimeLine&gt;.ListEvent***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie : 

- la liste des événements compris entre deux moments donnés dans un champ TimeLine, 

- tous les événements d'un champ TimeLine,

- un événement particulier (sélectionné ou survolé).



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Liste tous les événements du champ TimeLine
tabEvt est un tableau de EvénementTimeline
tabEvt = TL_TimeLine.ListeEvénement()

// Ajoute les événements dans un champ Table
POUR TOUT gMonEvénement DE tabEvt
	TABLE_Table1.AjouteLigne(gMonEvénement.Piste, ...
			gMonEvénement.Début, gMonEvénement.Fin, gMonEvénement.Titre)
FIN
```
<a name="Exemple2"></a>

```wl
// Liste les événements à partir d'un moment
tabEvt est un tableau de EvénementTimeline
tabEvt = TL_Automates.ListeEvénement("Automate 1", 70)

POUR TOUT gMonEvénement DE tabEvt
	Trace(gMonEvénement.Début, gMonEvénement.Fin, gMonEvénement.Titre)
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Récupérer la liste des événements entre deux moments

`<Résultat> = <Champ TimeLine>.ListeEvénement([<Piste> [, <Début> [, <Fin>]]])`
---

**`<Résultat> : (Tableau de variables de type EvénementTimeline)`**

Tableau contenant les variables de type [EvénementTimeline](../WDLang1/1000020612.md). 
Si un événement correspondant aux critères spécifiés existe, ce tableau contiendra un seul élément. 
Si aucun événement ne correspond aux critères spécifiés, ce tableau contiendra 0 événement.

**`<Champ TimeLine> : (Nom de champ)`**

Nom du champ TimeLine à manipuler.

**`<Piste> : (Chaîne de caractères optionnelle)`**

Nom de la piste à laquelle les événements sont associés. Si ce paramètre n'est pas spécifié (ou correspond à une chaîne vide), tous les événements de toutes les pistes seront renvoyés.

**`<Début> : (Entier optionnel)`**

Début de sélection des événements. 
Si ce paramètre n'est pas spécifié, tous les événements du champ TimeLine seront renvoyés.

**`<Fin> : (Entier optionnel)`**

Fin de sélection des événements. 
Si ce paramètre n'est pas spécifié, tous les événements à partir de &lt;Début&gt; seront renvoyés.


<a name="SYNTAXE2"></a>

### Récupérer les informations d'un événement particulier

`<Résultat> = <Champ TimeLine>.ListeEvénement(<Type d'événement>)`
---

**`<Résultat> : (Tableau de variables de type EvénementTimeline)`**

Nom du tableau contenant les variables de type [EvénementTimeline](../WDLang1/1000020612.md). 
Si un événement correspondant aux critères spécifiés existe, ce tableau contiendra un seul élément. 
Si aucun événement ne correspond aux critères spécifiés, ce tableau contiendra 0 événement.

**`<Champ TimeLine> : (Nom de champ)`**

Nom du champ TimeLine à manipuler.

**`<Type d'événement> : (Constante de type Entier)`**

Type d'événement recherché : 



|   |   |
| --- | --- |
| *tlEveSélectionné* | Evénement actuellement sélectionné dans le champ TimeLine. Si aucun événement n'est sélectionné, &lt;Résultat&gt; correspondra à un tableau de 0 élément. |
| *tlEveSurvolé* | Evénement survolé par la souris dans le champ TimeLine. Si aucun événement n'est survolé, &lt;Résultat&gt; correspondra à un tableau de 0 élément. |

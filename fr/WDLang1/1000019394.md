
## AnnéeEnCours (Fonction)

***En anglais : CurrentYear***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie l'année en cours sous forme d'entier. 






<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonAnnée est un entier
MonAnnée = AnnéeEnCours()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = AnnéeEnCours()`
---

**`<Résultat> : (Entier)`**

Année en cours.

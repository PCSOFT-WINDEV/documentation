


## &lt;Tableau de bord&gt;.DéplaceWidget (Fonction)

***En anglais : &lt;Dashboard&gt;.MoveWidget***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Change la position d'un Widget présent dans un champ Tableau de bord. Ce Widget est déjà configuré ou affiché. 

Rappel : Un Widget correspond à : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une fenêtre interne.





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Déplace le Widget 2 en haut du champ Tableau de bord
TDB_MonTableauDeBord.DéplaceWidget(2, 1, 1)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Déplacer un Widget identifié par son indice

`<Champ Tableau de bord>.DéplaceWidget(<Indice du Widget> , <X> , <Y> [, <Options>])`
---

**`<Champ Tableau de bord> : (Nom de champ)`**

Nom du champ Tableau de bord à manipuler.

**`<Indice du Widget> : (Entier)`**

Indice du Widget à manipuler. Cet indice correspond :

- au numéro d'ajout du Widget défini sous l'éditeur de fenêtres ou de pages.

- au numéro d'ajout du Widget renvoyé par la fonction [&lt;Tableau de bord&gt;.AjouteWidget](../WDLang1/1000022502.md).




**`<X> : (Entier)`**

Position horizontale du Widget en nombre de cases ou en pixels. La première case est la case [1, 1]. La valeur en pixels est arrondie pour correspondre à un nombre entier de cases.

**`<Y> : (Entier)`**

Position verticale du Widget en nombre de cases ou en pixels. La première case est la case [1, 1]. La valeur en pixels est arrondie pour correspondre à un nombre entier de cases.

**`<Options> : (Constante optionnelle de type Entier)`**

Permet de définir l'unité utilisée et le mode de réorganisation : 


|   |   |
| --- | --- |
| *tdbSansRéorganisation* | Déplace le Widget sans réorganiser les Widgets adjacents.<br><br>Attention : L'utilisation de la constante *tdbSansRéorganisation* peut conduire à un affichage incorrect (avec des Widgets superposés). En utilisant cette constante, il est important de réorganiser entièrement le champ Tableau de bord à l'aide des fonctions [&lt;Tableau de bord&gt;.ChangeTailleWidget](../WDLang1/1000022504.md) et **&lt;Tableau de bord&gt;.DéplaceWidget**. |
| *tdbUnitéCase*<br>(Valeur par défaut) | L'unité utilisée pour le déplacement du Widget est un nombre de cases. |
| *tdbUnitéPixel* | L'unité utilisée pour le déplacement du Widget est le pixel. |

<a name="NOTE0_1"></a>

## Remarques


### Règle utilisée pour le calcul des cases
<a name="regle_utilisee_pour_calcul_des_cases_ELTPARAGRAPHE000244"></a>Pour un Widget de taille 100 px par 100 px, avec une marge de 10 px, voici quelques exemples de redimensionnement : 


```wl
TDB_FuseauxHoraires.DéplaceWidget(1, 80, 80, tdbUnitéPixel)
// Le Widget se positionnera dans la case 1 x 1

TDB_FuseauxHoraires.DéplaceWidget(1, 160, 200, tdbUnitéPixel)
// Le Widget se positionnera dans la case 2 x 2

TDB_FuseauxHoraires.DéplaceWidget(1, 211, 211, tdbUnitéPixel)
// Le Widget se positionnera dans la case 3 x 3

TDB_FuseauxHoraires.DéplaceWidget(1, 400, 500, tdbUnitéPixel)
// Le Widget se positionnera dans la case 4 x 5
```




## Fonctions Notes repositionnables
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions permettant de manipuler le champ Note Repositionnable par programmation sont les suivantes : 



|   |   |
| --- | --- |
| [NoteChargeTout](../WDLang1/1000022397.md) | Importe et affiche les notes repositionnables de la fenêtre en cours précédemment sauvées par la fonction [NoteSauveTout](../WDLang1/1000022396.md). |
| [NoteCrée](../WDLang1/1000022395.md) | Crée un champ Note repositionnable dans la fenêtre en cours par programmation. |
| [NoteSauveTout](../WDLang1/1000022396.md) | Sauvegarde les notes redimensionnables de la fenêtre en cours dans un fichier. |
| [NoteSupprime](../WDLang1/1000022893.md) | Supprime une note repositionnable présente dans la fenêtre en cours. |

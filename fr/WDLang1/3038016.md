


## Fonctions de gestion des menus
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions de gestion des menus sont les suivantes : 



|   |   |
| --- | --- |
| [MenuAjoute](../WDLang1/3038048.md) | Ajoute un menu contextuel existant dans le menu principal de la fenêtre en cours. |
| [MenuAjouteMenu](../WDLang1/1000017266.md) | Ajoute un nouveau menu dans un menu existant (déroulant ou contextuel) d'une fenêtre ou d'une page. |
| [MenuAjouteOption](../WDLang1/1000017262.md) | Ajoute une nouvelle option de menu à la suite de la dernière option d'un menu (déroulant ou contextuel). |
| [MenuAjouteOptionURL](../WDLang1/1000018529.md) | Ajoute une nouvelle option de menu à la suite de la dernière option d'un menu (déroulant ou contextuel). Cette option de menu permet d'afficher la page correspondant à l'URL indiquée. |
| [MenuAjoutePopup](../WDLang1/1000020583.md) | Transforme une option de menu d'une page pour que cette option ouvre une popup. |
| [MenuAjouteSéparateur](../WDLang1/1000017270.md) | Ajoute un nouveau séparateur dans un menu (menu déroulant ou contextuel). |
| [MenuClone](../WDLang1/1000017257.md) | Clone un menu ou une option de menu ainsi que le code associé. |
| [MenuDémarque](../WDLang1/3038002.md) | Enlève la marque placée devant l'option de menu (menu contextuel ou déroulant). |
| [MenuEtat](../WDLang1/3038003.md) | Identifie l'état d'une option de menu (menu contextuel ou déroulant) : active, inactive ou invisible. |
| [MenuExiste](../WDLang1/1000019365.md) | Indique si une option de menu existe dans un menu. |
| [MenuInsèreMenu](../WDLang1/1000017267.md) | Insère un menu avant un autre menu dans une fenêtre. |
| [MenuInsèreOption](../WDLang1/1000017265.md) | Insère une nouvelle option à une position spécifique. |
| [MenuInsèreSéparateur](../WDLang1/1000017271.md) | Insère un séparateur dans un menu. |
| [MenuInvisible](../WDLang1/3038022.md) | Rend une option de menu invisible (menu contextuel ou déroulant). |
| [MenuLibellé](../WDLang1/3038009.md) | Identifie ou modifie le libellé d'une option de menu (menu contextuel ou déroulant). |
| [MenuMarquage](../WDLang1/3038008.md) | Permet de savoir si la marque est présente devant l'option de menu (menu contextuel ou déroulant). |
| [MenuMarque](../WDLang1/3038007.md) | Positionne la marque devant l'option de menu (menu contextuel ou déroulant). |
| [MenuSelectMoins](../WDLang1/3038010.md) | Désactive (grise) une option de menu (menu contextuel ou déroulant). |
| [MenuSelectPlus](../WDLang1/3038005.md) | Active une option de menu (menu contextuel ou déroulant). |
| [MenuSupprime](../WDLang1/3038058.md) | Supprime un menu ou une option de menu. |



Les fonctions suivantes permettent également de gérer les menus : 


|   |   |
| --- | --- |
| [ChampContextuel](../WDLang1/3025012.md) | Identifie le champ sur lequel le menu contextuel a été ouvert. |
| [EnumèreMenu](../WDLang1/3038039.md) | Utilisée dans une boucle, cette fonction permet d'énumérer les options de menu d'une fenêtre, comme les sous options d'une option de menu. |
| [EnumèreSousElément](../WDLang1/3064002.md) | Utilisée dans une boucle, cette fonction permet d'énumérer les menus déroulants ou les menus contextuels d'une fenêtre ou d'un champ. |
| [grMenu](../WDLang3/3042035.md) | Active ou désactive le menu contextuel d'un graphe. |
| [OuvreMenuContextuel](../WDLang1/3025029.md) | Ouvre automatiquement un menu contextuel pour le champ ou la fenêtre en cours. |
| [PiloteMenu](../WDLang1/1000009010.md) | Pilote le prochain menu qui sera ouvert, puis exécute automatiquement une option de ce menu indiquée en paramètre. |

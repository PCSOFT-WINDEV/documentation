


## KanbanAjouteListe (Fonction)

***En anglais : KanbanAddList***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ajoute une nouvelle liste dans un champ Kanban.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ajout d'une liste de Kanban avec comme titre "Tâches à revoir"
// Dans le champ KANBAN_Kanban1
LISTE_Liste_a_revoir est un Champ
LISTE_Liste_a_revoir <- KanbanAjouteListe(KANBAN_Kanban1, "A_revoir")
LISTE_Liste_a_revoir.Titre = "Tâches à revoir"
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = KanbanAjouteListe(<Champ Kanban> , <Liste de Kanban>)`
---

**`<Résultat> : (Champ)`**

Champ de type "Liste de Kanban". 

**`<Champ Kanban> : (Nom de champ)`**

Nom du champ Kanban à manipuler. 

**`<Liste de Kanban> : (Chaîne de caractères)`**

Nom de la liste de Kanban à créer.

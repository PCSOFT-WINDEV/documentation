


## Fonctions de gestion des champs d'affichage HTML
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions suivantes permettent de gérer le champ d'affichage HTML : 



|   |   |
| --- | --- |
| [HTMLDepuisEmail](../WDLang1/1410087263.md) | Importe le contenu HTML d'un email et ses images dans un champ d'affichage HTML. Les images contenues dans l'email sont également affichées dans le champ. |
| [HTMLExécuteProcédureNavigateur](../WDLang1/1000025920.md) | Exécute une procédure WLangage navigateur WEBDEV dans un champ d'affichage HTML avec une page créée par WEBDEV. |
| [HTMLExécuteTraitementChamp](../WDLang1/1000025969.md) | Lance l'exécution d'un événement WLangage associé à un champ présent dans une page HTML affichée par une champ d'affichage HTML. |
| [HTMLNavigue](../WDLang1/3025042.md) | Lance une opération de navigation dans un champ d'affichage HTML. |
| [HTMLSurErreurJS](../WDLang1/1000026259.md) | Indique la procédure à appeler en cas d'erreur dans l'exécution du Javascript dans un champ d'affichage HTML. |
| [HTMLValeurChamp](../WDLang1/1000025968.md) | Permet de connaître ou modifier un champ d'un formulaire HTML présent dans un champ d'affichage HTML. |
| [HTMLValideFormulaire](../WDLang1/1000025970.md) | Valide le formulaire d'une page HTML affichée dans un champ d'affichage HTML. |

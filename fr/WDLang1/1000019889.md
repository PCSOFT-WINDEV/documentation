
## FenDonneLargeurAdaptée (Fonction)

***En anglais : WinGiveSuitableWidth***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la largeur d'une fenêtre, adaptée au contenu des champs. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affiche le champ Table
TableAffiche(FEN_Prix.TABLE_Prix)
// Calcule la largeur adaptée
nLargeurOK est un entier
nLargeurOK = FenDonneLargeurAdaptée(FEN_Prix)
SI nLargeurOK < 640 ALORS
	FEN_Prix.Largeur = nLargeurOK
SINON
	FEN_Prix.Largeur = 640
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = FenDonneLargeurAdaptée([<Fenêtre>])`
---

**`<Résultat> : (Entier)`**

Largeur adaptée au contenu des champs pour optimiser l'affichage des champs dans la fenêtre. Cette largeur est exprimée en pixels. 

**`<Fenêtre> : (Nom de fenêtre optionnel)`**

Nom de la fenêtre à manipuler. Si ce paramètre n'est pas précisé, la fenêtre en cours sera manipulée.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Le calcul de la largeur adaptée d'une fenêtre est possible uniquement si la fenêtre contient un des champs suivants ancré en largeur :

- Champ Liste,

- Champ Table,

- Champ Zone répétée,

- Champ Arbre,

- Champ Liste image,

- Champ de saisie multiligne.


## Fonctions de gestion des champs Arbre (syntaxe préfixée)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions de gestion des champs Arbre (via la syntaxe préfixée) sont les suivantes : 



|   |   |
| --- | --- |
| [&lt;Arbre&gt;.Affiche](../WDLang1/1000023421.md) | Rafraîchit l'affichage d'un champ Arbre à partir d'une position donnée. |
| [&lt;Arbre&gt;.Ajoute](../WDLang1/1000023422.md) | Ajoute un élément "feuille" (et tous les noeuds nécessaires) dans un champ Arbre. |
| [&lt;Arbre&gt;.Cherche](../WDLang1/1000023423.md) | Recherche un élément (feuille ou noeud) dans un champ Arbre et renvoie son chemin. |
| [&lt;Arbre&gt;.CopieElément](../WDLang1/1000023424.md) | Copie un élément (feuille ou noeud) dans un champ Arbre. |
| [&lt;Arbre&gt;.DéplaceElément](../WDLang1/1000023755.md) | Déplace un élément (feuille ou noeud) dans un champ Arbre. |
| [&lt;Arbre&gt;.Déroule](../WDLang1/1000023756.md) | Déroule un noeud dans un champ Arbre. |
| [&lt;Arbre&gt;.DérouleTout](../WDLang1/1000023757.md) | Déroule un noeud et tous ses fils dans un champ Arbre. |
| [&lt;Arbre&gt;.Enroule](../WDLang1/1000023758.md) | Enroule une branche précédemment déroulée. |
| [&lt;Arbre&gt;.EnrouleTout](../WDLang1/1000023759.md) | Enroule une branche (ainsi que tous ses fils) précédemment déroulée. |
| [&lt;Arbre&gt;.EtatElément](../WDLang1/1000023878.md) | Renvoie l'état d'un noeud : enroulé, déroulé ou inexistant. |
| [ArbreID](../WDLang1/3018027.md) | Permet de construire le chemin des éléments d'un champ Arbre en différenciant les doublons. |
| [&lt;Arbre&gt;.InfoXY](../WDLang1/1000023760.md) | Renvoie l'élément d'un arbre se trouvant à une position donnée. |
| [&lt;Arbre&gt;.Insère](../WDLang1/1000023761.md) | Insère une feuille à une position spécifique dans un champ Arbre. |
| [&lt;Arbre&gt;.ListeFils](../WDLang1/1000023762.md) | Liste les "fils" d'un noeud et les éléments "fils" de ces "fils" dans un champ Arbre. |
| [&lt;Arbre&gt;.Modifie](../WDLang1/1000023763.md) | Modifie un élément (noeud ou feuille) dans un champ Arbre. |
| [ArbreOccurrence](../WDLang1/3018009.md) | Renvoie le nombre d'éléments présents dans une branche d'un champ Arbre. |
| [&lt;Arbre&gt;.Position](../WDLang1/1000023764.md) | Affiche un champ Arbre à partir d'un élément (noeud ou feuille) spécifié ou renvoie le nom du premier élément affiché dans un champ Arbre. |
| [&lt;Arbre&gt;.RécupèreFils](../WDLang1/1000023765.md) | Renvoie les "fils" d'un élément (niveau immédiatement inférieur) dans un champ Arbre. |
| [&lt;Arbre&gt;.RécupèreIdentifiant](../WDLang1/1000023766.md) | Renvoie l'identifiant associé à un élément (noeud ou feuille) dans un champ Arbre. |
| [ArbreRécupèreIndice](../WDLang1/1000023418.md) | Renvoie l'indice d'un élément (noeud ou feuille) dans un champ Arbre. |
| [&lt;Arbre&gt;.Select](../WDLang1/1000023767.md) | Renvoie le chemin complet de l'élément sélectionné dans un champ Arbre. |
| [&lt;Arbre&gt;.SelectMoins](../WDLang1/1000023768.md) | Déselectionne un élément dans un champ Arbre. |
| [&lt;Arbre&gt;.SelectOccurrence](../WDLang1/1000023769.md) | Renvoie le nombre d'éléments sélectionnés dans un champ Arbre (notamment dans le cas d'un champ Arbre multisélection). |
| [&lt;Arbre&gt;.SelectPlus](../WDLang1/1000023770.md) | Sélectionne un élément dans un champ Arbre. |
| [&lt;Arbre&gt;.Supprime](../WDLang1/1000023771.md) | Supprime un élément et l'ensemble de ses "fils" d'un champ Arbre. |
| [&lt;Arbre&gt;.SupprimeFils](../WDLang1/1000023772.md) | Supprime les fils d'un élément d'un champ Arbre sans supprimer l'élément lui-même. |
| [&lt;Arbre&gt;.SupprimeTout](../WDLang1/1000023773.md) | Supprime l'ensemble des éléments d'un champ Arbre. |
| [&lt;Arbre&gt;.Trie](../WDLang1/1000023774.md) | Trie les éléments présents dans un champ Arbre. |
| [&lt;Arbre&gt;.TypeElément](../WDLang1/1000023775.md) | Teste l'existence d'un élément d'un champ Arbre et renvoie son type (noeud ou feuille) si l'élément existe. |

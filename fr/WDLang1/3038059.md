


## Constantes de gestion des menus déroulants
			



[Afficher uniquement les constantes utilisées par les syntaxes préfixées](#" onclick="$('.objet').show();$('.pas_objet').hide();)

[Afficher uniquement les constantes utilisées par les syntaxes non préfixées](#" onclick="$('.objet').hide();$('.pas_objet').show();)

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les constantes de gestion des champs Menus déroulants sont les suivantes : 

| Constante | Description | Utilisée par les fonctions |
| --- | --- | --- |
| **menuAjouteFin** | Ajoute l'élément cloné à la fin du menu. | [MenuClone](../WDLang1/1000017257.md) |
| **menuInsèreAvantSéparateur** | Insère l'élément cloné avant le prochain séparateur. | [MenuClone](../WDLang1/1000017257.md) |
| **menuPopupBasCentréMenu** | Ouvre la popup sous le menu en la centrant sur la largeur totale du menu (deux popups de tailles identiques et ouvertes par deux options différentes s'affichent au même endroit avec cette option). | [MenuAjoutePopup](../WDLang1/1000020583.md) |
| **menuPopupBasDroite** | Ouvre la popup sous l'option de menu en l'alignant sur le bord droit de l'option de menu (la popup s'affiche vers la gauche). | [MenuAjoutePopup](../WDLang1/1000020583.md) |
| **menuPopupBasGauche** | Ouvre la popup sous l'option de menu en l'alignant sur le bord gauche de l'option de menu. | [MenuAjoutePopup](../WDLang1/1000020583.md) |
| **menuPopupHautDroite** | Ouvre la popup à droite de l'option de menu en l'alignant sur le bord haut de l'option de menu (cette constante est principalement utilisée avec les menus verticaux). | [MenuAjoutePopup](../WDLang1/1000020583.md) |
| **menuPopupHautGauche** | Ouvre la popup à gauche de l'option de menu en l'alignant sur le bord haut de l'option de menu (cette constante est principalement utilisée avec les menus verticaux). | [MenuAjoutePopup](../WDLang1/1000020583.md) |




## Fonctions de gestion du champ Traitement de texte (syntaxe préfixée)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions de gestion des champs Traitement de texte (via une syntaxe préfixée) sont les suivantes :



|   |   |
| --- | --- |
| [&lt;Document&gt;.RechercheElémentParNom](../WDLang1/1410088861.md) | Recherche un élément par son nom dans l'ensemble d'un document de type Traitement de texte. |
| [&lt;Traitement de texte&gt;.Ajoute](../WDLang1/1000024032.md) | Ajoute à la fin d'un champ Traitement de texte : un élément, un document, un paragraphe, un texte, une image ou un fragment. |
| [&lt;Traitement de texte&gt;.DébutModification](../WDLang1/1000023405.md) | Signale le début du regroupement d'un ensemble d'opérations sur un champ Traitement de texte en un seul événement (gestion du "undo/redo"). |
| [&lt;Traitement de texte&gt;.EntreEdition](../WDLang1/1000024263.md) | Entre en édition dans le champ Traitement de texte dans une partie spécifique du document : corps, en-tête ou pied de page. |
| [&lt;Traitement de texte&gt;.Ferme](../WDLang1/1000024264.md) | Ferme le fichier docx associé au champ Traitement de texte et libère le document. |
| [&lt;Traitement de texte&gt;.FinModification](../WDLang1/1000024265.md) | Signale la fin du regroupement d'un ensemble d'opérations sur un champ Traitement de texte en un seul événement (gestion du "undo/redo"). |
| [&lt;Traitement de texte&gt;.Imprime](../WDLang1/1000024266.md) | Imprime le contenu d'un champ Traitement de texte au format "docx". |
| [&lt;Traitement de texte&gt;.InfoXY](../WDLang1/1000025346.md) | Renvoie pour une position donnée dans un champ Traitement de texte (coordonnées d'un point du champ) :<br><br>- soit le numéro de la page correspondante, <br><br>- soit la position X (en mm) correspondante, <br><br>- soit la position Y (en mm) correspondante.<br><br><br> |
| [&lt;Traitement de texte&gt;.Insère](../WDLang1/1000024267.md) | Insère un objet dans un champ Traitement de texte. Les objets manipulables sont : <br><br>- un élément,<br><br>- un document, <br><br>- un paragraphe, <br><br>- un texte, <br><br>- une image, <br><br>- un fragment.<br><br><br> |
| [&lt;Traitement de texte&gt;.InsèreTableau](../WDLang1/1000024268.md) | Insère un tableau dans un champ Traitement de texte. |
| [&lt;Traitement de texte&gt;.Ouvre](../WDLang1/1000024269.md) | Ouvre un fichier Traitement de texte (format docx) en mode Lecture/Ecriture. Le fichier docx est ouvert et bloqué jusqu'à sa fermeture. |

| [&lt;Traitement de texte&gt;.Recherche](../WDLang1/1000024271.md) | Recherche une chaîne de caractères dans l'ensemble d'un champ Traitement de texte. |
| [&lt;Traitement de texte&gt;.RechercheElémentParNom](../WDLang1/1410088862.md) | Recherche un élément par son nom dans l'ensemble d'un document de type Traitement de texte. |
| [&lt;Traitement de texte&gt;.Redo](../WDLang1/1000024390.md) | Rétablit la dernière action annulée sur un champ Traitement de texte. |
| [&lt;Traitement de texte&gt;.RedoOccurrence](../WDLang1/1000024391.md) | Renvoie le nombre d'actions annulées qui peuvent être refaites sur un champ Traitement de texte. |

| [&lt;Traitement de texte&gt;.Sélectionne](../WDLang1/1000024394.md) | Sélectionne un fragment dans un champ Traitement de texte. |
| [&lt;Traitement de texte&gt;.SortEdition](../WDLang1/1000024395.md) | Sort de l'édition en cours. L'entrée en édition a été effectuée avec la fonction [&lt;Traitement de texte&gt;.EntreEdition](../WDLang1/1000024263.md). |


## ParamètreFAA (Fonction)

***En anglais : ConfigureAAF***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de spécifier le comportement de certaines Fonctionnalités Automatiques de vos Applications (FAA). Il est par exemple possible de paramétrer l'option d'export vers Excel utilisée par [le menu automatique des tables](../Editeurs/9500102.md).

Remarque : cette fonction est identique à la fonction [FAAParamètre](../WDLang1/1000022108.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Paramètre le mode d'exportation du contenu du champ Table vers Excel
ParamètreFAA(faaTableVersExcel, taAvecMiseEnForme)

// Avec combinaison d'options : 
ParamètreFAA(faaTableVersExcel, taTitreColonnes + taColonneOrdreAffiché + taAvecMiseEnForme)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = ParamètreFAA(<FAA concernée> , <Paramètre>)`
---

**`<Résultat> : (Variant)`**

Ancienne valeur du paramétrage de la FAA.

**`<FAA concernée> : (Constante de type Entier ou combinaison de constantes de type Entier)`**

Constante correspondant à la FAA à paramétrer : 



|   |   |
| --- | --- |
| *faaDuréeToastRestaureFiltre* | Paramètre la durée d'affichage du toast indiquant la restauration des filtres mémorisés par l'utilisateur sur un champ Table. <br>Rappel : L'utilisateur peut mémoriser les filtres définis sur une colonne d'un champ Table via l'option "Mémoriser le filtre" du menu contextuel.<br><br>Élément concerné : Champ Table |
| *faaImprimeEtatSurTable* | Paramètre l'impression de l'état sur un champ Table. La mise en forme des cellules (couleurs, ...) définie dans le champ Table est conservée lors de l'utilisation de la fonction [TableImprime](../WDLang1/1000017237.md). <br><br>Élément concerné : Champ Table |
| *faaRechercheInternet* | Paramètre le moteur utilisé pour effectuer la recherche sur Internet. Par défaut, le moteur de recherche utilisé est Google. <br><br>Élément concerné : <br><br>- Champ de saisie<br><br>- Champs Table, Table hiérarchique, Tableur et Tableau Croisé dynamique<br><br><br> |
| *faaRépertoireExportTable* | Paramètre le répertoire par défaut dans lequel le fichier d'export sera créé. |
| *faaSélectionColonne* | Sélection des colonnes visibles par le menu contextuel du champ Table.<br><br>Élément concerné : Champ Table |
| *faaTableVersExcel* | Exporte le contenu du champ Table ou la sélection réalisée dans le champ vers Excel.<br><br>Élément concerné : Champ Table |
| *faaTableVersWord* | Exporte le contenu du champ Table ou la sélection réalisée dans le champ vers Word.<br><br>Élément concerné : Champ Table |
| *faaTableVersXML* | Exporte le contenu du champ Table ou la sélection réalisée dans le champ vers XML.<br><br>Élément concerné : Champ Table |



**`<Paramètre> : (Chaîne de caractères ou Constante de type Entier)`**

Paramétrage de la FAA. 

- **Pour paramétrer l'export vers Excel, Word ou XML**, les constantes utilisables sont les suivantes : 
	


|   |   |
| --- | --- |
| *taAvecColonnesInvisibles* | Exporte aussi les colonnes invisibles du champ Table. |
| *taAvecMiseEnForme* | Exporte la couleur et la police du champ Table. |
| *taColonneOrdreAffiché* | Exporte les colonnes selon l'ordre des colonnes actuellement affiché dans le champ Table (et non selon l'ordre des colonnes défini sous l'éditeur). |
| *taFusion* | **Uniquement si le fichier XLS existe déjà.** <br><br>Fusionne le contenu existant avec le nouveau contenu :<br><br>	- Les formules, la mise en forme, ... des cellules existantes sont conservées.<br><br>	- Les données existantes ne sont pas conservées. Seules les données des colonnes présentes en plus dans le fichier XLS existant sont conservées.<br><br><br> |
| *taLigneSélectionnées* | Exporte uniquement les lignes sélectionnées (par défaut, toutes les lignes sont exportées). N'exporte pas les totaux. |
| *taSansTitre* | Seules les données sont copiées dans le fichier d'export. |
| *taSansTotaux* | N'exporte pas les lignes de totaux, moyenne et comptage automatique (par défaut, ces lignes sont exportées). |
| *taTitreColonnes* | Le titre des colonnes est inséré dans le fichier d'export. |



- **Pour paramétrer la sélection des colonnes visibles**, &lt;Paramètre&gt; est un entier indiquant le nombre de colonnes du champ Table à partir duquel le menu contextuel reste affiché après une sélection de l'utilisateur. Par défaut, ce paramètre vaut 4.

- **Pour paramétrer le répertoire d'export**, &lt;Paramètre&gt; est une chaîne correspondant au chemin du répertoire.

- **Pour paramétrer le moteur de recherche utilisé pour la recherche Internet**, &lt;Paramètre&gt; est une chaîne correspondant à la ligne de commande du moteur de recherche. Quelques exemples : 

	- Google : "http://www.google.com/#q=%1"

	- Bing : "http://www.bing.com/search?q=%1"

	- Yahoo : "http://fr.search.yahoo.com/search?p=%1"


Si &lt;Paramètre&gt; = "" (chaîne vide), le moteur utilisé sera le moteur par défaut (Google). 

- **Pour paramétrer l'impression de l'état** (et notamment la conservation de la mise en forme des cellules),  &lt;Paramètre&gt; est un entier qui vaut 1.

- **Pour paramétrer le temps d'affichage du message furtif (filtres dans les champs Table)**, &lt;Paramètre&gt; indique le temps d'affichage exprimé en centièmes de seconde. Par défaut, cette durée est de 7 secondes. Ce paramètre peut correspondre à : 

	- un entier correspondant au nombre de centièmes de seconde, 

	- une variable de type Durée, 

	- l'indication directe de la durée (par exemple '1 s' ou '10 cs'). 

	- -1 pour utiliser la durée par défaut.

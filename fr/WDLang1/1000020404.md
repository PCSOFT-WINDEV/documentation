


## TimelineSupprimeTout (Fonction)

***En anglais : TimelineDeleteAll***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime tous les événements et toutes les pistes d'un champ TimeLine.

Remarques : 

- Dans le cas d'un champ TimeLine par programmation, tous les événements et toutes les pistes sont supprimés. 

- Dans le cas d'un champ TimeLine fichier ou sur variable, les événements et les pistes sont uniquement supprimés dans le champ TimeLine. Les événements et les pistes sont conservés dans la source de données liée au champ. 



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Suppression des événements et de toutes les pistes du champ TimeLine par programmation
TimeLineSupprimeTout(TL_TimeLine)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`TimelineSupprimeTout(<Champ TimeLine>)`
---

**`<Champ TimeLine> : (Nom de champ)`**

Nom du champ TimeLine à manipuler.


## FinAmortissement (Fonction)

***En anglais : FinRedemption***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Crée une matrice d'amortissement pour le montant d'un emprunt à un taux d'intérêt constant pour une durée définie.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple
<a class="notetitle" target="_blank" href="$DOC$=1000003050061&name=finamortissement_fonction&product=WD">Voir des exemples supplémentaires</a>

```wl
// Matrice d'amortissement pour un emprunt de type "In Fine" de 25 000 Euros
// à un taux d'intérêt constant de 5% sur une durée de 6 ans
ResAmortissement = FinAmortissement("MatriceAmort", finInFine, 25000, 5, 6)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = FinAmortissement(<Nom de la matrice> , <Type d'amortissement> [, <Capital> [, <Taux> [, <Durée>]]])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la matrice d'amortissement a été créée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour connaître le libellé de l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.




**`<Nom de la matrice> : ()`**

Nom de la matrice d'amortissement. Cette matrice est automatiquement créée lors du calcul de l'amortissement. Cette matrice est composée de 5 colonnes. Chaque colonne correspond à une information particulière :



|   |   |
| --- | --- |
| Colonne 1 | Année |
| Colonne 2 | Reste à rembourser |
| Colonne 3 | Intérêt |
| Colonne 4 | Amortissement |
| Colonne 5 | Annuités |



**`<Type d'amortissement> : (Constante)`**

Indique le type d'amortissement :



|   |   |
| --- | --- |
| *finAmortissementConstant* | ***Amortissement constant :*** Chaque année, la valeur de l'amortissement réalisé est fixe; les autres paramètres sont calculés en conséquence. |
| *finAnnuitéConstante* | ***Annuité constante :*** Chaque année, l'emprunteur verse au prêteur la même somme, incluant le principal et l'intérêt. |
| *finInFine* | ***Remboursement "In Fine" :*** Pendant toute la durée de l'emprunt, l'emprunteur verse uniquement les intérêts dus, le principal étant remboursé la dernière année. |



**`<Capital> : (Réel ou monétaire optionnel)`**

Capital total emprunté (0 par défaut).

**`<Taux> : (Réel optionnel)`**

Taux d'intérêt de l'emprunt exprimé en pourcentage (1 par défaut). Par exemple, pour un taux d'emprunt de 5%, ce paramètre sera égal à 5.

**`<Durée> : (Réel optionnel)`**

Durée du remboursement de l'emprunt exprimée en années (1 par défaut).



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
L'existence de la matrice d'amortissement n'est pas vérifiée. Si la matrice existait auparavant, cette matrice est écrasée. Pour vérifier l'existence d'une matrice, utilisez la fonction [MatExiste](../WDLang1/3050076.md).

Le contenu de la matrice d'amortissement peut ensuite être lu grâce aux fonctions suivantes :

- [MatLit](../WDLang1/3050074.md) : Lit la valeur d'un élément d'une matrice.

- [MatLitColonne](../WDLang1/3050070.md) : Lit la valeur de tous les éléments d'une colonne d'une matrice.

- [MatLitLigne](../WDLang1/3050032.md) : Lit la valeur de tous les éléments d'une ligne d'une matrice.

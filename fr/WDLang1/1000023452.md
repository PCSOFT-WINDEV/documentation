


## LecteurPDFOuvre (Fonction)

***En anglais : PDFReaderOpen***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ouvre un fichier PDF dans un champ Lecteur PDF.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
LecteurPDFOuvre(PDF_MonLecteurPDF, "C:\MonDocument.pdf")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = LecteurPDFOuvre(<Champ Lecteur PDF> , <Chemin> [, <Mot de passe>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'ouverture du fichier PDF a été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Le document actuellement affiché dans le champ n'est pas modifié. Pour connaître le libellé de l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.




**`<Champ Lecteur PDF> : (Nom de champ)`**

Nom du champ Lecteur PDF à manipuler. 

**`<Chemin> : (Chaîne de caractères)`**

Chemin absolu ou relatif du fichier PDF à ouvrir. 

**`<Mot de passe> : (Chaîne de caractères optionnelle)`**

Mot de passe nécessaire pour ouvrir le fichier (fichier PDF protégé en ouverture). 

- Si ce paramètre n'est pas spécifié et si le fichier PDF est protégé par mot de passe en ouverture, une fenêtre sera affichée pour permettre à l'utilisateur de saisir le mot de passe. 
	

- Si le mot de passe spécifié n'est pas valide pour le document, la fonction renverra une erreur. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Pour fermer le document PDF affiché dans le champ Lecteur PDF, utilisez la fonction [LecteurPDFFerme](../WDLang1/1000023453.md).

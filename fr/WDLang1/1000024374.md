
## Constantes de gestion des champs Lecteur PDF et des fichiers PDF
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les constantes utilisées par les fonctions de gestion des champs Lecteur PDF sont les suivantes :

| Constante | Détail | Fonction ou type de variable |
| --- | --- | --- |
| **affAvecCouverture** | Mode d'affichage de la page dans le lecteur PDF utilisé : Considère la première page du fichier PDF comme étant une couverture. La première page est affichée à droite lors d'un affichage en mode 2 pages. | Type [pdfParamètre](../WDLang6/1410087857.md) |
| **affAvecDéfilement** | Mode d'affichage de la page dans le lecteur PDF utilisé : Affichage avec défilement : Affiche les pages selon une colonne verticale ininterrompue occupant la largeur d'une page. Le défilement peut être activé pour un affichage 1 page ou 2 pages. | Type [pdfParamètre](../WDLang6/1410087857.md) |
| **affDéfaut** | Mode d'affichage de la page dans le lecteur PDF utilisé : Affichage par défaut. | Type [pdfParamètre](../WDLang6/1410087857.md) |
| **affDeuxPages** | Mode d'affichage de la page dans le lecteur PDF utilisé : Affichage de deux pages à la fois, côte à côte du ficher PDF dans le lecteur PDF. | Type [pdfParamètre](../WDLang6/1410087857.md) |
| **affUnePage** | Mode d'affichage de la page dans le lecteur PDF utilisé : Affichage d'une seule page à la fois du fichier PDF dans le lecteur PDF. Les parties adjacentes ne sont pas affichées. | Type [pdfParamètre](../WDLang6/1410087857.md) |
| **pdfCoordonnéeX** | Abscisse de la position spécifiée sur la page (exprimée en millimètres). | [LecteurPDFInfoXY](../WDLang1/1000025332.md) |
| **pdfCoordonnéeY** | Ordonnée de la position spécifiée sur la page (exprimée en millimètres). | [LecteurPDFInfoXY](../WDLang1/1000025332.md) |
| **pdfCouleur** | Gestion de la couleur dans le fichier PDF : Création d'un fichier PDF en couleurs. | Type [pdfParamètre](../WDLang6/1410087857.md) |
| **pdfNiveauGris** | Gestion de la couleur dans le fichier PDF : Création d'un fichier PDF en noir et blanc. | Type [pdfParamètre](../WDLang6/1410087857.md) |
| **pdfNuméroPage** | Numéro de la page contenant la position spécifiée. | [LecteurPDFInfoXY](../WDLang1/1000025332.md) |
| **pdfOrigineEcran** | Permet de changer l'origine pour le calcul des informations correspondant à une position dans le champ Lecteur PDF. | [LecteurPDFInfoXY](../WDLang1/1000025332.md) |
| **rotationAntihoraire** | Effectue une rotation à -90 degrés de la page. | [LecteurPDFTournePage](../WDLang1/1000023940.md) |
| **rotationHoraire** | Effectue une rotation à 90 degrés de la page. | [LecteurPDFTournePage](../WDLang1/1000023940.md)<br><br> |
| **zoomContenu** | Zoom appliqué par défaut au fichier PDF à son ouverture : Zoom au contenu. | Type [pdfParamètre](../WDLang6/1410087857.md) |
| **zoomDéfaut** | Zoom appliqué par défaut au fichier PDF à son ouverture : Zoom par défaut. | Type [pdfParamètre](../WDLang6/1410087857.md) |
| **zoomPageEntière** | Zoom appliqué par défaut au fichier PDF à son ouverture : Zoom permettant d'afficher une page entière. | Type [pdfParamètre](../WDLang6/1410087857.md) |
| **zoomPleineHauteur** | Zoom appliqué par défaut au fichier PDF à son ouverture : Zoom permettant d'afficher la page entière dans sa hauteur. | Type [pdfParamètre](../WDLang6/1410087857.md) |
| **zoomPleineLargeur** | Zoom appliqué par défaut au fichier PDF à son ouverture : Zoom permettant d'afficher la page entière dans sa largeur. | Type [pdfParamètre](../WDLang6/1410087857.md) |
| **zoomTailleRéelle** | Zoom appliqué par défaut au fichier PDF à son ouverture : Zoom à 100%/ | Type [pdfParamètre](../WDLang6/1410087857.md) |




## &lt;Tableur&gt;.SelectFeuille (Fonction)

***En anglais : &lt;Spreadsheet&gt;.SelectWorksheet***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Sélectionne une feuille de calcul (identifiée par son nom) dans un champ Tableur.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Bouton [Aller sur Feuille]
TBLR_MonTableur.SelectFeuille(SAI_NomFeuille_A_Sélectionner)
SI ErreurDétectée = Vrai ALORS
	Erreur(ErreurInfo())
SINON
	LIB_MaFeuille = TBLR_MonTableur.FeuilleEncours
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Champ Tableur>.SelectFeuille(<Feuille>)`
---

**`<Champ Tableur> : (Nom de champ)`**

Nom du champ Tableur à manipuler.

**`<Feuille> : (Chaîne de caractères)`**

Nom de la feuille de calcul à sélectionner.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- L'indice de la feuille en cours est disponible via la propriété [FeuilleEncours](../Proprietes/1000021640.md). 

- Le nom de la feuille en cours est connu via la propriété [NomFeuille](../Proprietes/1000021646.md). 

- Si la feuille n'existe pas, la fonction lève une erreur non fatale : la variable [ErreurDétectée](../WDLang1/3087001.md) est à <u><u><u><u>Vrai</u></u></u></u> et la fonction [ErreurInfo](../WDLang1/3013008.md) permet de connaître la cause de l'erreur. 

- Cette fonction est utilisable uniquement sur un champ Tableur présent dans une fenêtre.

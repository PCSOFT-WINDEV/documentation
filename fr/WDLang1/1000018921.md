
## OrthoAjouteDictionnaire (Fonction)

***En anglais : SpellAddDictionary***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ajoute un mot au dictionnaire orthographique de OpenOffice : 

- soit pour l'instance de l'application en cours. 

- soit pour toutes les applications.




Après cet ajout, le mot sera considéré comme juste par la fonction [OrthoVérifie](../WDLang1/1000018923.md). 

**Remarque** : La plupart des champs d'une application WINDEV proposent d'utiliser le correcteur orthographique (option "Vérifier l'orthographe" dans l'onglet "Détail" de la description du champ). 

**Conseil** : Il est conseillé d'utiliser la fonction **OrthoAjouteDictionnaire** dans l'événement "Initialisation" du projet. Ainsi, certains mots tels que le nom de l'application, des termes spécifiques à l'application, ne seront pas considérés comme faux par le correcteur orthographique de OpenOffice.  
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
OrthoAjouteDictionnaire("WINDEV")
OrthoAjouteDictionnaire("WEBDEV")
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`OrthoAjouteDictionnaire(<Mot à ajouter> [, <Portée>])`
---

**`<Mot à ajouter> : (Chaîne de caractères)`**

Mot à ajouter au dictionnaire de Open Office. 
**Attention** : La langue du dictionnaire à prendre en compte doit être paramétrée avec la fonction [ParamètreOrthographe](../WDLang1/3025052.md).

**`<Portée> : (Constante de type Entier)`**

Portée de l'ajout dans le dictionnaire. Ce paramètre correspond à l'une des constantes suivantes : 



|   |   |
| --- | --- |
| *orthPortéePersistant* | Le mot est ajouté dans le dictionnaire de OpenOffice. <br>Cette constante a le même effet que l'option "Ajouter au dictionnaire" du menu contextuel des FAA avec correction orthographique. |
| *orthPortéeProcessus*<br>(Valeur par défaut) | Le mot n'est pas ajouté dans le dictionnaire de OpenOffice.<br>Il continuera donc à être affiché comme "faux" si l'application est relancée. |







<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Pour savoir si le correcteur d'orthographe de OpenOffice est installé sur le poste en cours, utilisez la fonction [OrthoDisponible](../WDLang1/1000019019.md).




## &lt;Tableau croisé dynamique&gt;.EnrouleTout (Fonction)

***En anglais : &lt;Pivot table&gt;.CollapseAll***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Enroule tous les entêtes de lignes et/ou de colonnes d'un champ Tableau croisé dynamique.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Charge le fichier des données
TCD_Statistiques.Charge(sFichier)
// Enroule tous les entêtes du champ Tableau croisé dynamique
TCD_Statistiques.EnrouleTout()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Champ Tableau croisé dynamique>.EnrouleTout([<Type de l'entête>])`
---

**`<Champ Tableau croisé dynamique> : (Nom de champ)`**

Nom du champ Tableau croisé dynamique à manipuler. Ce champ Tableau croisé dynamique doit avoir été rempli soit avec la fonction [&lt;Tableau croisé dynamique&gt;.CalculeTout](../WDLang1/1000022180.md), soit avec la fonction [&lt;Tableau croisé dynamique&gt;.Charge](../WDLang1/1000022181.md).

**`<Type de l'entête> : (Constante ou combinaison de constantes optionnelle de type Entier)`**

Type de l'entête à enrouler : 


|   |   |
| --- | --- |
| *tcdEntêteColonne* | Enroule les entêtes de colonnes. |
| *tcdEntêteLigne* | Enroule les entêtes de lignes. |


Par défaut, si ce paramètre n'est pas spécifié, les entêtes de lignes **ET** de colonnes sont enroulés.

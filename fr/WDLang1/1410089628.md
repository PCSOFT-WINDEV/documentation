


## &lt;Kanban&gt;.SupprimeListe (Fonction)

***En anglais : &lt;Kanban&gt;.DeleteList***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime une liste de Kanban d'un champ Kanban.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Suppression de la liste 1
KANBAN_Liste.SupprimeListe(1)
// Insertion de 2 autres listes à la place
KANBAN_Liste.InsèreListe(1, "A voir")
KANBAN_Liste.InsèreListe(2, "A faire")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Supprimer une liste identifiée par son indice

`<Champ Kanban>.SupprimeListe(<Indice>)`
---

**`<Champ Kanban> : (Nom de champ)`**

Nom du champ Kanban manipulé.

**`<Indice> : (Entier)`**

Indice de la liste à supprimer. 



<a name="SYNTAXE2"></a>

### Supprimer une liste identifiée par son nom

`<Champ Kanban>.SupprimeListe(<Liste de Kanban>)`
---

**`<Champ Kanban> : (Nom de champ)`**

Nom du champ Kanban manipulé.

**`<Liste de Kanban> : (Nom de champ)`**

Nom du champ Liste de Kanban à supprimer. 




<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- La liste de Kanban et toutes ses cartes sont supprimées du champ Kanban. 

- Si le champ Kanban est lié à un fichier de données, les données du fichier de données ne sont pas modifiées. 

- En cas d'échec de la suppression, la variable [ErreurDétectée](../WDLang1/3087001.md) est à <u><u><u><u>Vrai</u></u></u></u>. Pour plus de détails sur l'erreur, il est possible d'utiliser la fonction [ErreurInfo](../WDLang1/3013008.md).




## &lt;Variable File&gt;.Enfile (Fonction)

***En anglais : &lt;Queue variable&gt;.Enqueue***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Enfile un élément dans une file. L'élément est ajouté à la fin de la file.  




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MaFile est une File d'entiers

// On enfile les valeurs 1, 2 et 3
MaFile.Enfile(1)
MaFile.Enfile(2)
MaFile.Enfile(3)

// Affichage de la trace : 1, 2, 3
x est un entier
TANTQUE MaFile.Défile(x)
	Trace(x)
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<File>.Enfile(<Valeur>)`
---

**`<File> : (Variable de type File)`**

Nom de la variable de type [File](../WDLang1/1514021.md) à manipuler.

**`<Valeur> : (Type d'éléments manipulés par la file)`**

Valeur à ajouter à la fin de la file. Tous les types de variables peuvent être utilisés SAUF les tableaux, les tableaux associatifs, les files, les piles et les listes.




## GanttAjouteTâche (Fonction)

***En anglais : GanttAddTask***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ajoute une tâche dans une colonne d'un champ Table (ou Table hiérarchique) de type Diagramme de Gantt.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
NouvelleTâche est un TâcheGantt

// Paramètre la tâche
NouvelleTâche.ID = DonneIdentifiant()
NouvelleTâche.Ligne = 1
NouvelleTâche.Avancement = 50
NouvelleTâche.DateDébut = DateDuJour()
NouvelleTâche.DuréeEnJour = 1
NouvelleTâche.Titre = "Nouvelle tâche"

// Ajoute la tâche
GanttAjouteTâche(COL_Gantt, NouvelleTâche)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`GanttAjouteTâche(<Colonne Gantt> , <Tâche>)`
---

**`<Colonne Gantt> : (Nom de champ)`**

Nom de la colonne (d'un champ Table ou Table hiérarchique) de type Diagramme de Gantt à manipuler.

**`<Tâche> : (Variable de type TâcheGantt)`**

Nom de la variable de type [TâcheGantt](../WDLang1/1000020881.md) décrivant les caractéristiques de la tâche à ajouter.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- La nouvelle tâche est ajoutée puis affichée.

- La tâche peut être récupérée dans une variable de type [TâcheGantt](../WDLang1/1000020881.md) en utilisant la fonction [GanttRécupèreTâche](../WDLang1/1000021026.md) : 
	
	```txt
	<Nom de la variable> <- GanttRécupèreTâche(<Colonne Gantt>, <Identifiant>)
	```




où Identifiant est l'identifiant de la tâche à récupérer. 
Par exemple : 

```wl
// Modification d'une tâche
MaTâche est un TâcheGantt
MaTâche <- GanttRécupèreTâche(COL_Gantt, "ID1")
MaTâche.Avancement = 70
MaTâche.CouleurAvancement = VertClair
```

Comme la tâche est récupérée par l'opérateur **<-**, les modifications effectuées sur la variable de type [TâcheGantt](../WDLang1/1000020881.md) seront automatiquement reportées dans le diagramme de Gantt. 
Il est également possible de récupérer une tâche en utilisant la syntaxe suivante : 

```txt
<Nom de la variable> <- <Colonne Gantt>["<Identifiant>"]
```

Par exemple : 

```wl
t <- COL_Gantt["ID1"]
t.CouleurFond = BleuClair
```


- Si la tâche n'a pas d'identifiant ou si cet identifiant existe déjà, une erreur WLangage est affichée.




## EnModeEtatsEtRequêtes (Fonction)

***En anglais : InReportsAndQueriesMode***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Identifie si le code est exécuté lors du test d'un état ou d'une requête sous le logiciel Etats & Requêtes.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Code exécuté lors du test d'un état ou d'une 
// requête sous le logiciel Etats & Requêtes ?
SI EnModeEtatsEtRequêtes() = Vrai ALORS
	// Mode test sous Etats & Requêtes
	// Traitement à effectuer
	...
SINON
	// Mode différent
	// Traitement à effectuer
	...
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = EnModeEtatsEtRequêtes()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le code est exécuté lors du test d'un état ou d'une requête sous le logiciel Etats & Requêtes, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.  








<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### GO depuis le logiciel Etats & Requêtes
<a name="depuis_logiciel_etats_requetes_ELTPARAGRAPHE000099"></a>Quand un test d'un état est lancé depuis le logiciel "Etats & requêtes" (bouton "GO"), les fonctions [EnModeTest](../WDLang1/3013011.md) et **EnModeEtatsEtRequêtes** renvoient <u><u><u><u>Vrai</u></u></u></u>.

Il est conseillé de remplacer le test traditionnel :

```wl
SI EnModeTest() = Vrai ALORS
	Trace("...")
FIN
```
Par le test suivant :

```wl
SI EnModeTest() = Vrai ET EnModeEtatsEtRequêtes() = Faux ALORS
	Trace("...")
FIN
```

<a name="NOTE0_2"></a>


### Disponibilité de la fonction
<a name="disponibilite_fonction_ELTPARAGRAPHE000248"></a>La fonction **EnModeEtatsEtRequêtes** est disponible dans tous les produits pour permettre l'écriture d'un code multi-produits. Des fonctions ou procédures utilisant la fonction **EnModeEtatsEtRequêtes** peuvent être partagées entre plusieurs produits, sans que des erreurs de compilation soient générées.

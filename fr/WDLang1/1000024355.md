


## &lt;Variable Image&gt;.ModifieOpacité (Fonction)

***En anglais : &lt;Image variable&gt;.ModifyOpacity***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie l'opacité d'une image (c'est-à-dire la couche alpha de l'image).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonImage est une Image = "Test.bmp"
MonImage est Image = "c:\test.png"
// Réduit l'opacité de 50%
MonImage.ModifieOpacité(50)
MonImage.SauveImagePNG("c:\test2.png")
IMG_MonDessin = MonImage
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`ModifieOpacité(<Image> , <Facteur>)`
---

**`<Image> : (Variable de type Image)`**

Nom de la variable de type [Image](../WDLang1/1000019650.md) à manipuler.

**`<Facteur> : (Réel)`**

Facteur de modification de l'opacité, compris entre 0.0 et 100.0. Ce facteur sera appliqué sur la couche alpha de l'image.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Cette fonction ne peut être utilisée qu'avec des images contenant une couche alpha (32 bpp).

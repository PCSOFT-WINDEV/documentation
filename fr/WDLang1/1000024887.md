


## Fonctions de gestion des masques
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions de gestion des masques de saisie sont les suivantes : 



|   |   |
| --- | --- |
| [MasqueCodePostal](../WDLang1/1000024742.md) | Récupère le masque de saisie du code postal adapté à un pays donné. |
| [MasqueFormate](../WDLang1/1000024738.md) | Formate une chaîne de caractères à l'aide d'une variable de type [MasqueDeSaisie](../WDLang1/1000024886.md). |
| [MasqueNuméroDeTéléphone](../WDLang1/1000024741.md) | Récupère le masque de saisie du numéro de téléphone adapté à un pays donné. |

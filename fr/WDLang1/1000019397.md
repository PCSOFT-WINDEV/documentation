
## CaractUnicode (Fonction)

***En anglais : CharactUnicode***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le caractère correspondant au code Unicode spécifié. 




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonCaractère est une chaîne UNICODE
MonCaractère = CaractUnicode(49)    // Retourne "1"
MonCaractère = CaractUnicode(87)    // Retourne "W"

Trace(Asc(Milieu("????", 1, 1))) // Affiche 35501
Trace(CaractUnicode(35501)) // ?
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = CaractUnicode(<Code Unicode à convertir>)`
---

**`<Résultat> : (Chaîne de caractères Unicode)`**

Caractère correspondant au code Unicode précisé.

**`<Code Unicode à convertir> : (Entier)`**

Valeur à convertir. Si la valeur n'est pas conforme, une erreur du WLangage est générée.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Éviter l'utilisation de la fonction CaractUnicode
<a name="eviter_utilisation_fonction_docparampagetitleshort_ELTPARAGRAPHE000074"></a>Les constantes prédéfinies suivantes permettent d'éviter l'utilisation de la fonction **CaractUnicode** :

| Constante | Valeur | Fonction |
| --- | --- | --- |
| ESC | Caract(27) | Touche Echap |
| ESP | Caract(160) | Espace insécable |
| RC | Caract(13) + Caract(10) | Retour Chariot |
| TAB | Caract(9) | Tabulation |


## FacteurGrandePolice (Fonction)

***En anglais : LargeFontFactor***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le facteur d'agrandissement courant d'une fenêtre WINDEV (uniquement si Windows est en mode "grandes polices").

![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png) Cette fonction est disponible uniquement pour simplifier la Webisation de projets WINDEV.  
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Agrandissement du champ de saisie "SAI_NomClient"
// selon le facteur d'agrandissement de la fenêtre
SAI_NomClient.X = SAI_NomClient.X * FacteurGrandePolice()
SAI_NomClient.Largeur = SAI_NomClient.Largeur * FacteurGrandePolice()
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = FacteurGrandePolice([<Fenêtre manipulée>])`
---

**`<Résultat> : (Réel)`**



- Facteur d'agrandissement courant de la fenêtre spécifiée.

- 1 si :

	- Windows n'est pas en mode "grandes polices",

	- la fenêtre ne s'agrandit pas en grandes polices.




- 0 en cas d'erreur. 




**`<Fenêtre manipulée> : (Nom de fenêtre optionnel)`**

Nom de la fenêtre à manipuler. Si ce paramètre n'est pas spécifié, la fenêtre manipulée sera la fenêtre en cours.




## &lt;Agenda&gt;.ListeRendezVous (Fonction)

***En anglais : &lt;Organizer&gt;.ListAppointment***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie : 

- la liste des rendez-vous compris entre deux dates données dans un champ Agenda, 

- tous les rendez-vous d'un champ Agenda,

- un rendez-vous particulier (sélectionné ou survolé).  








![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) En mode Universal Windows 10 App, cette fonction permet uniquement de manipuler les champs Agenda présents dans un état. 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Liste tous les rendez-vous de l'agenda
tabRdv est un tableau de RendezVous
tabRdv = AGD_Agenda1.ListeRendezVous()
// Ajoute les rendez-vous dans un champ Table
POUR TOUT gMonRendezVous DE tabRdv
	TABLE_Table1.AjouteLigne(gMonRendezVous.DateDébut, ...
			gMonRendezVous.DateFin, gMonRendezVous.Titre)
FIN
```
<a name="Exemple2"></a>

```wl
// Liste les rendez-vous à partir de la date du jour
tabRdv est un tableau de RendezVous
tabRdv = AGD_Agenda1.ListeRendezVous(DateSys())
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Récupérer la liste des rendez-vous entre deux dates

`<Résultat> = <Champ Agenda>.ListeRendezVous([<Date de début> [, <Date de fin>]])`
---

**`<Résultat> : (Tableau de variables de type RendezVous)`**

Tableau contenant les variables de type [RendezVous](../WDLang1/1000019244.md). 
Si un rendez-vous correspondant aux critères spécifiés existe, ce tableau contiendra un seul élément. 
Si aucun rendez-vous ne correspond aux critères spécifiés, ce tableau contiendra 0 rendez-vous.

**`<Champ Agenda> : (Nom de champ)`**

Nom du champ Agenda à manipuler. Ce champ peut correspondre : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) au champ Agenda d'une fenêtre.




**`<Date de début> : (Variable optionnelle de type DateHeure)`**

Date de début de sélection des rendez-vous.

Si ce paramètre n'est pas spécifié, tous les rendez-vous du champ Agenda seront renvoyés.

**`<Date de fin> : (Variable optionnelle de type DateHeure)`**

Date de fin de sélection des rendez-vous.

Si ce paramètre n'est pas spécifié, tous les rendez-vous à partir de &lt;Date de début&gt; seront renvoyés.


<a name="XSYNTAXE2"></a>
<a name="SYNTAXE2"></a>

### Récupérer les informations d'un rendez-vous particulier

`<Résultat> = <Champ Agenda>.ListeRendezVous(<Type de rendez-vous>)`
---

**`<Résultat> : (Tableau de variables de type RendezVous)`**

Tableau contenant la variable de type [RendezVous](../WDLang1/1000019244.md). 
Si un rendez-vous correspondant aux critères spécifiés existe, ce tableau contiendra un seul élément. 
Si aucun rendez-vous ne correspond aux critères spécifiés, ce tableau contiendra 0 rendez-vous.

**`<Champ Agenda> : (Nom de champ)`**

Nom du champ Agenda à manipuler. Ce champ peut correspondre : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) au champ Agenda d'une fenêtre.




**`<Type de rendez-vous> : (Constante de type Entier)`**

Type de rendez-vous recherché :


|   |   |
| --- | --- |
| *agRdvSélectionné* | Rendez-vous actuellement sélectionné dans le champ Agenda. Si aucun rendez-vous n'est sélectionné, &lt;Résultat&gt; correspondra à un tableau de 0 élément. |
| *agRdvSurvolé* | Rendez-vous survolé par la souris dans le champ Agenda. Si aucun rendez-vous n'est survolé, &lt;Résultat&gt; correspondra à un tableau de 0 élément.<br> |

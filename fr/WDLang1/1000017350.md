


## AjouteColonne (Fonction)

***En anglais : AddColumn***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ajoute une colonne à la fin d'une propriété de type tableau à deux dimensions (par exemple variable de type [xlsDocument](../WDLang5/1000017464.md)).






<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = AjouteColonne(<Tableau WLangage> [, <Élément ligne 1> [... [, <Élément ligne N>]]])`
---

**`<Résultat> : (Entier)`**



- Indice de la colonne ajoutée, 

- -1 en cas d'erreur.




**`<Tableau WLangage> : (Tableau)`**

Nom de la variable de type Tableau à manipuler. Ce tableau doit être un tableau à une ou deux dimensions.

**`<Élément ligne 1> : (Type des éléments du tableau, optionnel)`**

Élément de la colonne à ajouter dans le tableau spécifié. Chaque élément correspond à une ligne dans le tableau. Le type de l'élément ajouté doit être compatible avec le type de la ligne concernée.
Si aucun de ces paramètres n'est spécifié, une colonne vide est ajoutée dans le tableau. Les lignes sont initialisées avec la valeur par défaut du type des autres éléments du tableau.
Si un de ces paramètres n'est pas spécifié, la ligne correspondante est initialisée avec la valeur par défaut du type des autres éléments du tableau.



**`<Élément ligne N> : (Type des éléments du tableau, optionnel)`**

Élément de la colonne à ajouter dans le tableau spécifié. Chaque élément correspond à une ligne dans le tableau. Le type de l'élément ajouté doit être compatible avec le type de la ligne concernée.
Si aucun de ces paramètres n'est spécifié, une colonne vide est ajoutée dans le tableau. Les lignes sont initialisées avec la valeur par défaut du type des autres éléments du tableau.
Si un de ces paramètres n'est pas spécifié, la ligne correspondante est initialisée avec la valeur par défaut du type des autres éléments du tableau.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Lors de l'utilisation de la fonction **AjouteColonne** :

- la variable de type avancé doit être allouée.

- le type avancé doit avoir un énumérateur de type collection modifiable.

- le type avancé est automatiquement agrandi pour accueillir la nouvelle colonne.

- les lignes sont initialisées avec les valeurs passées en paramètre. Si aucune valeur n'est passée en paramètre, les lignes sont initialisées avec la valeur par défaut du type des éléments du tableau.

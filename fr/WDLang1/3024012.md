


## Compresse (Fonction)

***En anglais : Compress***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Compresse une chaîne de caractères ou un bloc mémoire (buffer) sous forme binaire. La décompression sera ensuite réalisée grâce à la fonction [Décompresse](../WDLang1/3024009.md).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Compresse puis décompresse une chaîne
Comp est un Buffer = Compresse("Chaîne à compresser")
Info(Décompresse(Comp))
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = Compresse(<Chaîne de caractères / Buffer> [, <Type de compression>])`
---

**`<Résultat> : (Buffer)`**



- Buffer compressé. 

- Chaîne vide ("") en cas d'erreur. Le compte-rendu détaillé de l'erreur est connu grâce à la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Chaîne de caractères / Buffer> : (Chaîne de caractères ou Buffer)`**



- Chaîne de caractères à compresser.

- Buffer à compresser.




**`<Type de compression> : (Constante optionnelle de type Entier)`**

Type de compression à effectuer : 


|   |   |
| --- | --- |
| *compresseAucun* | Aucune compression n'est effectuée.<br><br>![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Non disponible. |
| *compresseChaîneCourte* | Compression à l'aide d'un algorithme optimisé pour les chaînes de caractères de petite taille.<br><br>![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Non disponible. |
| *compresseGZIP* | Compression avec l'algorithme GZIP. |
| *compresseLZW*<br>(Valeur par défaut) | Compression avec l'algorithme LZW.<br><br>![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Non disponible. |


![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Ce paramètre est obligatoire.

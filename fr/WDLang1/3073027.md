
## SysIconeModifie (Fonction)

***En anglais : SysIconModify***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie le fichier icône et/ou le message de survol d'une icône dans la barre des tâches (en bas à droite de l'écran). Cette icône a été ajoutée dans la barre des tâches avec la fonction [SysIconeAjoute](../WDLang1/3073001.md).

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Icone.gif)
 ou ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=IconePPC.gif)


**Remarque** : Le fonctionnement de l'icône (menu contextuel, appel d'une procédure) n'est pas modifié.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Modification du message de survol de l'icône
// préalablement ajoutée dans la barre des tâches
ResModifieIcone = SysIconeModifie("", "Comparatif des ventes")
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = SysIconeModifie(<Icône> [, <Message de survol>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'icône a été modifiée dans la barre des tâches, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Icône> : (Chaîne de caractères ou variable de type Icône)`**

Correspond à un des éléments suivants : 

- Nom et chemin complet (ou relatif) du fichier icône à ajouter. Un chemin de type UNC est autorisé.

- Nom d'une variable de type [Icône](../WDLang1/1000020811.md). 


Si ce paramètre correspond à une chaîne vide (""), le fichier icône n'est pas modifié.

**`<Message de survol> : (Chaîne de caractères optionnelle)`**

Message affiché lors du survol de la souris sur l'icône. 
Si ce paramètre n'est pas spécifié ou correspond à une chaîne vide (""), le message de survol n'est pas modifié.





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Une seule icône peut être ajoutée dans une application. Il n'est pas possible d'en ajouter plusieurs.

- La fonction [SysIconeAjoute](../WDLang1/3073001.md) permet d'ajouter une icône.

- La fonction [SysIconeSupprime](../WDLang1/3073022.md) permet de supprimer l'icône. L'icône est automatiquement supprimée lors de la fermeture de l'application.

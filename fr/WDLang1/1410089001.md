
## MarkdownVersTexte (Fonction)

***En anglais : MarkdownToText***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Convertit une chaîne de caractères au format Markdown en une chaîne de caractères au format texte (sans balises).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Teste si une chaîne contient des balises Markdown
PROCÉDURE TesteMarkdown(chaîne_à_tester est chaîne)
SI  MarkdownVersTexte(chaîne_à_tester) = chaîne_à_tester ALORS
	// le texte ne contient pas de balise
	RENVOYER Faux
FIN
RENVOYER Vrai
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = MarkdownVersTexte(<Chaîne Markdown>)`
---

**`<Résultat> : (Chaîne de caractères)`**

Chaîne de caractères au format texte (sans balises). 

**`<Chaîne Markdown> : (Chaîne de caractères)`**

Chaîne de caractères au format Markdown.

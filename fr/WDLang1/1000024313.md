


## &lt;Tableau&gt;.Occurrence (Fonction)

***En anglais : &lt;Array&gt;.Count***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie :

- soit le nombre d'éléments présents dans un tableau à une dimension.

- soit le nombre de lignes présentes dans un tableau à deux dimensions.

- soit le nombre d'éléments présents dans la première dimension d'un tableau à trois dimensions ou plus.




**Remarque** : Cette fonction est utilisable uniquement sur les [tableaux WLangage](../Motscles/1514030.md).








<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonTableau est un tableau de 2 chaînes
MonTableau.Insère(1, "WINDEV")
MonTableau.Insère(2, "WEBDEV")
MonTableau.Insère(3, "WINDEV et WEBDEV")
// Nombre d'éléments ?
Trace(MonTableau.Occurrence()) // Affiche 5
```
<a name="Exemple2"></a>

```wl
MonTableau est un tableau de 2 par 3 chaînes
MonTableau.InsèreLigne(1, "A", "B", "C")
MonTableau.InsèreLigne(2, "D", "E", "F")
// Nombre de lignes ?
Trace(MonTableau.Occurrence()) // Affiche 4
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Tableau WLangage>.Occurrence()`
---

**`<Résultat> : (Entier)`**



- Nombre d'éléments présents dans le tableau si &lt;Tableau WLangage&gt; est un tableau à une dimension.

- Nombre de lignes présentes dans le tableau si &lt;Tableau WLangage&gt; est un tableau à deux dimensions.

- Nombre d'éléments présents dans la première dimension du tableau si &lt;Tableau WLangage&gt; est un tableau à trois dimensions ou plus.




**`<Tableau WLangage> : (Tableau)`**

Nom de la variable de type Tableau à manipuler.


## Belgique.VérifieBCE (Fonction)

***En anglais : Belgium.CheckCBE***



<a name="description"></a>
Teste la validité d'un numéro BCE (numéro Banque-Carrefour des Entreprises de Belgique servant à identifier une entreprise).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Test d'une variable
MonNuméroBCE est une chaîne = "0436649062"
Belgique.VérifieBCE(MonNuméroBCE)	// renvoie Vrai

// Test d'un champ
SAI_NuméroBCE = "ABC"
Belgique.VérifieBCE(SAI_NuméroBCE)	// renvoie Faux
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = Belgique.VérifieBCE(<Numéro à vérifier>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'élément spécifié contient un numéro BCE valide.

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Numéro à vérifier> : (Nom de champ ou variable de type chaîne)`**

Numéro BCE à vérifier. Ce paramètre peut correspondre :

- au nom d'une variable de type chaîne.

- au nom d'un champ.

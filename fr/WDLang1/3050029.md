
## Exp (Fonction)

***En anglais : Exp***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Calcule l'exponentielle (inverse du logarithme népérien) d'une valeur numérique.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Calcul de l'exponentielle de 5
ResExponentielle = Exp(5)
// Renvoie 148.413159103
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = Exp(<Valeur numérique>)`
---

**`<Résultat> : (Réel)`**

Exponentielle recherchée.

**`<Valeur numérique> : (Entier ou réel)`**

Valeur numérique à manipuler.

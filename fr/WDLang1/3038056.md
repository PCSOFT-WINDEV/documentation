


## FenActiveGFI (Fonction)

***En anglais : WinActivateDDW***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Active ou désactive le GFI (Grisage des Fenêtres Inaccessibles) :

- soit d'une fenêtre spécifique.

- soit de toutes les fenêtres de l'application. Dans ce cas, les fenêtres paramétrées spécifiquement avec la fonction **FenActiveGFI** ne seront pas prises en compte.


Active ou désactive le GFI (Grisage des Fenêtres Inaccessibles) d'une fenêtre spécifique.

Le GFI consiste à griser automatiquement la ou les fenêtres inaccessibles. Pour plus de détails, consultez [Grisage automatique des fenêtres](../Editeurs/9000071.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// -- Code de clic du bouton "BTN_Valider"
// Activation du GFI de la fenêtre "FEN_SaisieClient"
FenActiveGFI(FEN_SaisieClient, Vrai)
// Ouverture d'une boîte de dialogue
Dialogue("Voulez-vous enregistrer les modifications ?")
// Tant que la boîte de dialogue est affichée, 
// la fenêtre "FEN_SaisieClient" est grisée.
```


<a name="Exemple2"></a>

```wl
// -- Initialisation du projet
// Désactivation du GFI de toutes les fenêtres de l'application
FenActiveGFI(Faux)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Activer ou désactiver le GFI d'une fenêtre spécifique

`<Résultat> = FenActiveGFI(<Fenêtre> , <Activation du GFI>)`
---

**`<Résultat> : (Booléen)`**

État d'activation du GFI précédent :

- <u><u><u><u>Vrai</u></u></u></u> : le GFI était activé sur la fenêtre spécifiée.

- <u><u><u><u>Faux</u></u></u></u> : le GFI n'était pas activé sur la fenêtre spécifiée.




**`<Fenêtre> : (Nom de fenêtre)`**

Nom ou alias de la fenêtre à manipuler. Cette fenêtre doit être ouverte.

**`<Activation du GFI> : (Booléen)`**

Nouvel état d'activation du GFI :

- <u><u><u><u>Vrai</u></u></u></u> : le GFI est activé sur la fenêtre spécifiée.

- <u><u><u><u>Faux</u></u></u></u> : le GFI n'est pas activé sur la fenêtre spécifiée.  





<a name="SYNTAXE2"></a>

### Activer ou désactiver le GFI de toutes les fenêtres de l'application

`<Résultat> = FenActiveGFI(<Activation du GFI>)`
---

**`<Résultat> : (Booléen)`**

État d'activation du GFI précédent :

- <u><u><u><u>Vrai</u></u></u></u> : le GFI était activé sur les fenêtres de l'application.

- <u><u><u><u>Faux</u></u></u></u> : le GFI n'était pas activé sur les fenêtres de l'application.




**`<Activation du GFI> : (Booléen)`**

Nouvel état d'activation du GFI :

- <u><u><u><u>Vrai</u></u></u></u> : le GFI est activé sur les fenêtres de l'application.

- <u><u><u><u>Faux</u></u></u></u> : le GFI n'est pas activé sur les fenêtres de l'application.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Configuration du poste des utilisateurs finaux
<a name="configuration_poste_des_utilisateurs_finaux_ELTPARAGRAPHE000120"></a>Selon la configuration du poste des utilisateurs finaux, le GFI peut également être sans effet.

En effet, le GFI prend en compte les caractéristiques de la carte graphique, la vitesse du processeur, etc.

En cas de problème, conseillez à vos utilisateurs finaux de mettre à jour le pilote de leur carte graphique.

**Remarque** : Pour fonctionner, le poste des utilisateurs finaux doit :

- être en mode d'affichage supérieur à 256 couleurs.

- être en mode 32 ou 64 bits.

- être sous Windows 2000 ou supérieur.

- utiliser le niveau maximal d'"Accélération matérielle" de la carte graphique (propriétés avancées d'affichage du poste).



<a name="NOTE0_2"></a>


### Taux de grisage
<a name="taux_grisage_ELTPARAGRAPHE000140"></a>Par défaut, le taux de grisage des fenêtres est égal à 18 %. Pour modifier ce taux, utilisez la fonction [FenTauxGFI](../WDLang1/3038057.md).

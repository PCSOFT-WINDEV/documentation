


## Fonctions de gestion des champs Zone multiligne
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


Les fonctions de gestion des champs Zone multiligne sont les suivantes :



|   |   |
| --- | --- |
| [zmlAjouteLigne](../WDLang1/1000020199.md) | Ajoute une nouvelle ligne dynamique dans un champ Zone multiligne. |
| [zmlInsèreLigne](../WDLang1/1000020207.md) | Insère une nouvelle ligne dynamique dans un champ Zone multiligne. |
| [zmlSupprime](../WDLang1/1000020208.md) | Supprime une ligne dynamique dans un champ Zone multiligne. |
| [zmlSupprimeTout](../WDLang1/1000020209.md) | Supprime toutes les lignes dynamiques dans un champ Zone multiligne. |

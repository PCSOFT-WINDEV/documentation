
## WHelp (Fonction)

***En anglais : WHelp***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Affiche un fichier ou une page d'aide au format HLP ou CHM.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ouverture du "NotePad.hlp" sur la page de Couper/Coller
WHelp("C:\Windows\NotePad.hlp", "COLLAGE DE TEXTE")
```


<a name="Exemple2"></a>

```wl
// Ouverture du "NotePad.hlp" sur la page ayant 35 pour numéro de contexte 
// ou sur l'index si le numéro n'existe pas
WHelp("C:\Windows\NotePad.hlp", 35)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Ouvrir l'aide sur l'aide (WinHelp.hlp)

`WHelp()`
---


<a name="SYNTAXE2"></a>

### Ouvrir un fichier d'aide

`WHelp(<Nom du fichier d'aide> [, <Mot-clé>])`
---

**`<Nom du fichier d'aide> : (Chaîne de caractères)`**

Nom et chemin du fichier d'aide (.HLP ou .CHM). Si le fichier d'aide n'est pas trouvé, la fonction ne fait rien.

**`<Mot-clé> : (Chaîne de caractères optionnelle)`**

Mot-clé permettant de rechercher la page d'aide. 
Si ce paramètre correspond à une chaîne vide ("") ou n'est pas spécifié, l'index du fichier d'aide est ouvert et l'utilisateur pourra sélectionner le mot-clé voulu. Si le fichier d'aide n'a pas d'index, le fichier est ouvert sur la première page de l'aide.


<a name="SYNTAXE3"></a>

### Afficher une page d'aide spécifique

`WHelp(<Nom du fichier d'aide> , <Numéro de contexte>)`
---

**`<Nom du fichier d'aide> : (Chaîne de caractères)`**

Nom et chemin du fichier d'aide (.HLP ou .CHM). Si le fichier d'aide n'est pas trouvé, la fonction ne fait rien.

**`<Numéro de contexte> : (Entier)`**

Numéro de contexte associé à la page d'aide et au champ pour lequel la page doit s'afficher. Le numéro d'aide associé à un champ peut être connu avec la fonction [NumHelp](../WDLang1/3013027.md).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Changement de langue et fichier d'aide
<a name="changement_langue_fichier_aide_ELTPARAGRAPHE000066"></a>Sous l'éditeur d'aide de WINDEV, il est possible de construire le nom du fichier d'aide à partir d'un identifiant numérique correspondant à la fonction [Nation](../WDLang1/3054014.md).

Dans ce cas, pour lancer le fichier d'aide correspondant à la langue en cours, il suffit d'utiliser le code suivant :


```wl
// Lancement de l'aide au format CHM
WHelp("MonFichier" + NumériqueVersChaîne(Nation(), "03d") + ".chm")
```

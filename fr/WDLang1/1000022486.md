
## docStyle (Type de variable)

***En anglais : docStyle***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **docStyle** permet de connaître toutes les caractéristiques d'un style faisant partie d'un document, d'un paragraphe ou d'un fragment de document. Les caractéristiques de ce style peuvent être connues à l'aide de différentes propriétés WLangage. 

**ATTENTION : Ce type ne doit pas être utilisé directement.** Il doit être utilisé uniquement via les variables de type [Document](../WDLang1/1000022461.md) qui contiennent un tableau des styles associés au document. 

Pour associer un style à un paragraphe, il suffit d'utiliser l'identifiant du style (propriété **StyleID** disponible dans les variables de type [docParagraphe](../WDLang1/1000022483.md), [docFragment](../WDLang1/1000022482.md) et [docElémentTexte](../WDLang1/1000022493.md)). 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonDoc est un Document <- TT_SansNom1
// Récupération du fragment correspondant à la sélection en cours dans le champ
Frag est un docFragment <- MonDoc[TT_SansNom1.Curseur à TT_SansNom1.FinCurseur]

// Passage du contenu du fragment en gras
Frag.MiseEnForme.PoliceGras = Vrai
```
<a name="Exemple2"></a>

```wl
// Lister les styles d'un document
MonChampDoc est un Champ

// Récupération d'un document associé
MonDoc est un Document = MonChampDoc

// Listage de tous les styles utilisé par le document
POUR TOUT UnStyleDuDoc DE MonDoc.Style
	Trace(UnStyleDuDoc.StyleID)
FIN

// Ajouter le style "Titre1" dans un document et l'appliquer à la sélection
// La définition du style "Titre1" provient d'un autre document
MonChampDoc est un Champ = {"MonChampTT", indChamp}

// Récupération d'un document associé
MonDoc est un Document = MonChampDoc

// si le style "Titre1" n'existe pas, on va l'ajouter 
	// à partir d'un document contenant des styles prédéfinis
SI MonDoc.Style["Titre1"] = Null ALORS
	// ouvre une document contenant les styles
	MonDocContenantLesStyles est un Document = "c:\MonDocAvecMesStyles.docx"
	// récupérer le style Titre 1
	StyleTitre1 est un DocStyle <- MonDocContenantLesStyles.Style["Titre1"]
	SI StyleTitre1 <> Null ALORS
		// Ajouter le style dans le document
		Ajoute(MonDoc.Style, StyleTitre1)
	FIN
FIN

// on applique le style à la sélection
StyleTitre1PourSelection est un DocStyle <- MonDoc.Style["Titre1"]
SI StyleTitre1PourSelection <> Null ALORS
	// créer le fragment (à partir de la sélection)
	f est DocFragment(MonChampDoc.Valeur, MonChampDoc.Curseur, MonChampDoc.LongueurSélection)
	// applique le style Titre1 à la sélection
	f.StyleID = StyleTitre1PourSelection.Identifiant
FIN
```

## Remarques
<a name="NOTE0_1"></a>


### Propriétés spécifiques à la description des variables de type docStyle
<a name="proprietes_specifiques_description_des_variables_type_docstyle_ELTPARAGRAPHE000042"></a>Les propriétés suivantes peuvent être utilisées pour manipuler un style :

| Nom de la propriété | Type manipulé | Effet |
| --- | --- | --- |
| Alias | Chaîne de caractères | Nom d'affichage du style. Ce nom est utilisé si la propriété **Nom** ne correspond pas à un nom d'affichage. <br><br>Si plusieurs noms sont disponibles, ils sont séparés par une virgule (","). <br><br><br><br>**Cette propriété est disponible en lecture seulement**. |
| Invisible | Booléen | <br><br>- <u><u><u><u>Vrai</u></u></u></u> si le style doit être masqué à l'utilisateur, <br><br>- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. <br><br><br><br><br>Si la propriété **Invisible** vaut <u><u><u><u>Vrai</u></u></u></u>, ce style est un style interne (par exemple, ce style est un style Parent et il ne doit pas être proposé dans l'IHM). <br><br><br><br>**Cette propriété est disponible en lecture seulement**. |
| InvisibleSiNonUtilisé | Booléen | <br><br>- <u><u><u><u>Vrai</u></u></u></u> si le style doit être masqué à l'utilisateur sauf s'il est déjà utilisé dans le document, <br><br>- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. <br><br><br><br><br>**Cette propriété est disponible en lecture seulement**. |
| Nom | Chaîne de caractères | Nom prédéfini du style ou nom d'affichage du style. <br><br>**Cette propriété est disponible en lecture seulement**. |
| ParDéfaut | Booléen | <br><br>- <u><u><u><u>Vrai</u></u></u></u> si le style est le style par défaut, <br><br>- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. <br><br><br> |
| PrioritéAffichage | Entier | Ordre de priorité lors de l'affichage des styles dans une interface (IHM). Cet ordre de priorité est un ordre croissant. |
| StyleID | Chaîne de caractères | Identifiant unique du style dans le document. <br><br>**Cette propriété est disponible en lecture seulement**. |
| StyleIDLié | Chaîne de caractères | Identifiant d'un style lié (style de type Paragraphe ou style de type Caractère). <br><br>Si le style est de type paragraphe, cette propriété renvoie l'identifiant du type de caractères associé. <br><br><br><br>**Cette propriété est disponible en lecture seulement**. |
| StyleIDParent | Chaîne de caractères | Identifiant du style parent (c'est-à-dire sur lequel est basé le style courant). <br><br>**Cette propriété est disponible en lecture seulement**. |
| Type | Constante de type Entier | Type du style : <br><br>- *docstyleCaractère* : Style de caractères. Ce style s'applique sur une portion de texte. <br><br>- *docstyleNumérotation* : Style de numérotation. <br><br>- *docstyleParagraphe* : Style de paragraphe. Ce style s'applique sur un paragraphe entier. <br><br>- *docstyleTableau* : Style de tableau. <br><br><br><br><br>**Cette propriété est disponible en lecture seulement**. <br><br><br><br>Remarque : le changement de style d'un paragraphe ou d'un fragment applique automatiquement le style paragraphe et le style caractère liés s'ils existent. |

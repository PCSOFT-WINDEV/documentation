
## Belgique.VérifieUE (Fonction)

***En anglais : Belgium.CheckEU***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Teste la validité d'un numéro UE (numéro d'unité d'établissement identifiant un établissement d'une entreprise en Belgique).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Test d'une variable
MonNuméroUE est une chaîne = "7123654957"
Belgique.VérifieUE(MonNuméroUE)	// renvoie Vrai

// Test d'un champ
SAI_NuméroUE = "ABC"
Belgique.VérifieUE(SAI_NuméroUE)	// renvoie Faux
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = Belgique.VérifieUE(<Numéro à vérifier>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'élément spécifié contient un numéro UE valide,

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Numéro à vérifier> : (Nom de champ ou variable de type chaîne)`**

Numéro UE à vérifier. Ce paramètre peut correspondre :

- au nom d'une variable de type chaîne.

- au nom d'un champ.

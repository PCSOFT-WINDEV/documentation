


## Fonctions multimédia
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


Les fonctions Multimédia permettent de manipuler les fichiers multimédia associés aux champs Multimédia. 



|   |   |
| --- | --- |
| [MultimédiaArrêt](../WDLang1/1000017203.md) | Arrête la lecture en cours d'un fichier multimédia dans un champ Multimédia. |
| [MultimédiaContrôleVisible](../WDLang1/1000019973.md) | Gère la visibilité des boutons de contrôle (Lecture, Pause, avance rapide, ...) du champ Multimédia. |
| [MultimédiaDébut](../WDLang1/1000017204.md) | Place au début du fichier multimédia joué dans le champ Multimédia. |
| [MultimédiaDurée](../WDLang1/1000017211.md) | Renvoie la taille du fichier multimédia en cours de lecture (en millisecondes) dans le champ Multimédia. |
| [MultimédiaEtat](../WDLang1/1000017214.md) | Indique l'état du fichier multimédia joué dans le champ Multimédia. |
| [MultimédiaFerme](../WDLang1/1000017205.md) | Ferme le fichier multimédia en cours dans le champ Multimédia. |
| [MultimédiaFin](../WDLang1/1000017209.md) | Place à la fin du fichier multimédia joué dans le champ Multimédia. |
| [MultimédiaHauteur](../WDLang1/1000017212.md) | Renvoie la hauteur en pixels de l'image source du fichier multimédia joué dans le champ Multimédia. |
| [MultimédiaJoue](../WDLang1/1000017202.md) | Joue un fichier multimédia dans un champ Multimédia. |
| [MultimédiaLargeur](../WDLang1/1000017213.md) | Renvoie la largeur en pixels de l'image source du fichier multimédia joué dans le champ Multimédia. |
| [MultimédiaOuvre](../WDLang1/1000017206.md) | Ouvre un fichier multimédia dans un champ Multimédia. |
| [MultimédiaPause](../WDLang1/1000017207.md) | Met en pause la lecture d'un fichier multimédia dans un champ Multimédia. |
| [MultimédiaPosition](../WDLang1/1000017210.md) | Renvoie ou modifie la position en cours du fichier multimédia en cours dans le champ Multimédia. |
| [MultimédiaVolume](../WDLang1/1000017208.md) | Renvoie ou modifie le volume sonore du fichier multimédia joué dans un champ Multimédia. |

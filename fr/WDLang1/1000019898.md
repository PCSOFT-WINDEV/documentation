
## MotVersTableau (Fonction)

***En anglais : WordToArray***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Remplit un tableau avec les mots d'une chaîne. 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Découpage de la chaîne dans le tableau
MonTableau est un tableau de chaînes
MotVersTableau("Exemple fonction MotOccurrence", MonTableau)
// le tableau contient
//		"Exemple"
//		"fonction"
//		"MotOccurrence"
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`MotVersTableau(<Chaîne> , <Tableau>)`
---

**`<Chaîne> : (Chaîne de caractères)`**

Chaîne de caractères à manipuler. 

**`<Tableau> : (Variable de type Tableau)`**

Nom de la variable de type Tableau de chaînes qui va contenir les différents mots. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Tous les caractères autres que numériques ou alphabétiques sont considérés comme des séparateurs de mots.

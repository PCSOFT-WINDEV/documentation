
## OrthoSuggère (Fonction)

***En anglais : SpellSuggest***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la liste des mots possibles présents dans le dictionnaire de OpenOffice pouvant corriger un mot mal orthographié.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Si le mot n'est pas présent dans le dictionnaire OpenOffice, propose les mots approchants
SI OrthoDisponible() = Vrai ET OrthoVérifie(Mot) = Faux ALORS
	ListeSugg est une chaîne = OrthoSuggère(Mot) 
	Info(Mot + " ? Vous ne voulez pas dire <" + RC + ListeSugg + RC + "> plutôt ?" )
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = OrthoSuggère(<Mot à vérifier>)`
---

**`<Résultat> : (Chaîne de caractères)`**

Peut correspondre à : 

- La liste des mots corrects proches du mot à vérifier. La liste renvoyée est de la forme : 
	
	```txt
	<Mot correct 1> +  RC + <Mot correct 2>  + RC + ...
	```


- Une chaîne vide ("") en cas d'erreur ou si aucune proposition n'est disponible. En cas d'erreur, la fonction [ErreurInfo](../WDLang1/3013008.md) permet de connaître la cause de l'erreur (par exemple si le dictionnaire de OpenOffice n'est pas installé). 




**`<Mot à vérifier> : (Chaîne de caractères)`**

Mot à faire vérifier par le correcteur orthographique de OpenOffice. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Liste des mots suggérés
<a name="liste_des_mots_suggeres_ELTPARAGRAPHE000040"></a>La liste des mots suggérés correspond à la liste proposée par le dictionnaire de OpenOffice lors d'un clic droit sur une faute (par exemple dans un champ de saisie proposant la vérification de l'orthographe).  


### Vérification de la présence du dictionnaire OpenOffice
<a name="verification_presence_dictionnaire_openoffice_ELTPARAGRAPHE000045"></a>Pour savoir si le correcteur d'orthographe de OpenOffice est installé sur le poste en cours, utilisez la fonction [OrthoDisponible](../WDLang1/1000019019.md).

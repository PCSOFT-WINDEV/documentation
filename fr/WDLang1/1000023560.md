
## Chrono (Type de variable)

***En anglais : Chrono***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **Chrono** permet de définir toutes les caractéristiques avancées d'un chronomètre. Les caractéristiques de ce chronomètre peuvent être définies et modifiées à l'aide de différentes propriétés WLangage. 

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
c est un Chrono
ChronoDébut(c)
...
// Code à chronométrer
...
ChronoFin(c)
```





<a name="NOTE0"></a>

## Remarques
<a name="NOTE0_1"></a>


### Propriétés spécifiques à la description des variables de type Chrono
<a name="proprietes_specifiques_description_des_variables_type_chrono_ELTPARAGRAPHE000043"></a>Les propriétés suivantes peuvent être utilisées pour manipuler une variable de type **Chrono** :

| Nom de la propriété | Type manipulé | Effet |
| --- | --- | --- |
| Démarré | Booléen | <br><br>- <u><u><u><u>Vrai</u></u></u></u> si le chronomètre est démarré, <br><br>- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. <br><br><br><br><br>**Cette propriété est disponible en lecture seulement**. |
| Valeur | Durée | Durée chronométrée. <br><br>Remarque : Cette propriété correspond au résultat de la fonction [ChronoValeur](../WDLang1/3027033.md). <br><br>**Cette propriété est disponible en lecture seulement**. |


<a name="NOTE0_2"></a>


### Fonctions WLangage manipulant le type Chrono
<a name="fonctions_wlangage_manipulant_type_chrono_ELTPARAGRAPHE000085"></a>

- Les fonctions manipulant le type **Chrono** sont les suivantes : 
	


|   |   |
| --- | --- |
| [ChronoDébut](../WDLang1/3027031.md) | Démarre un chronomètre afin de mesurer la durée d'un traitement (en millisecondes) et permet de remettre à zéro un chronomètre existant. |
| [ChronoFin](../WDLang1/3027032.md) | Arrête un chronomètre et renvoie le temps écoulé (en millisecondes) depuis le début du chronométrage (appel de la fonction [ChronoDébut](../WDLang1/3027031.md)). |
| [ChronoPause](../WDLang1/1000018829.md) | Interrompt un chronomètre afin de mesurer la durée d'un traitement. |
| [ChronoRAZ](../WDLang1/1000020474.md) | Arrête et remet un chronomètre existant à zéro. |
| [ChronoReprend](../WDLang1/1000018830.md) | Redémarre un chronomètre précédemment arrêté par les fonctions [ChronoFin](../WDLang1/3027032.md) ou [ChronoPause](../WDLang1/1000018829.md). |
| [ChronoValeur](../WDLang1/3027033.md) | Indique le temps écoulé (en millisecondes) depuis l'appel à la fonction [ChronoDébut](../WDLang1/3027031.md). |

- Les fonctions (syntaxe préfixée) manipulant le type **Chrono** sont les suivantes : 
	


|   |   |
| --- | --- |
| [&lt;Chrono&gt;.Début](../WDLang1/1000023788.md) | Démarre un chronomètre afin de mesurer la durée d'un traitement (en millisecondes) et permet de remettre à zéro un chronomètre existant. |
| [&lt;Chrono&gt;.Fin](../WDLang1/1000023789.md) | Arrête un chronomètre et renvoie le temps écoulé (en millisecondes) depuis le début du chronométrage (appel de la fonction [&lt;Chrono&gt;.Début](../WDLang1/1000023788.md)). |
| [&lt;Chrono&gt;.Pause](../WDLang1/1000023790.md) | Interrompt un chronomètre afin de mesurer la durée d'un traitement. |
| [&lt;Chrono&gt;.RAZ](../WDLang1/1000023791.md) | Arrête et remet un chronomètre existant à zéro. |
| [&lt;Chrono&gt;.Reprend](../WDLang1/1000023792.md) | Redémarre un chronomètre précédemment arrêté par les fonctions [&lt;Chrono&gt;.Fin](../WDLang1/1000023789.md) ou [&lt;Chrono&gt;.Pause](../WDLang1/1000023790.md). |

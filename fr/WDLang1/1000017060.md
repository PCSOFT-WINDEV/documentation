
## gFondDégradé (Fonction)

***En anglais : gBackgroundGradient***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Décrit une couleur d'affichage en mode dégradé. Cette couleur pourra par exemple être utilisée pour le fond du texte d'un élément d'un champ Liste ou Combo, d'une cellule de table, ...

**Attention** : Cette fonction ne peut pas être utilisée directement pour le fond d'une fenêtre (voir exemple ci-dessous). 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ajout du mot "Jaune" sur fond dégradé rouge à vert dans la liste "LISTE_ListeCouleur"
ListeAjoute(LISTE_ListeCouleur, gFondDégradé(RougeClair, VertClair) + "Jaune")
```



```wl
// Libellé avec un texte sur fond dégradé
LIB_Elément.Libellé = gFondDégradé(RVB(166, 223, 98), RVB(87, 171, 47)) + gStylo(RougeClair) + ...
	"Libellé avec fond dégradé"
```



```wl
// Exemple permettant d'appliquer un fond de fenêtre dégradé.
// Il faut utiliser un champ Image hors écran de la taille de la fenêtre (ici IMG_Degrade)

MonImage est une chaîne
MonImage = ComplèteRep(fRepExe()) + "ImageFond.JPG"
SI fFichierExiste(MonImage) = Vrai ALORS fSupprime(MonImage)
IMG_Degrade = gFondDégradé(RougePastel, BleuPastel, 70)
dSauveImageJPEG(IMG_Degrade, MonImage, 65)
FEN_Fenêtre1.ImageFond = MonImage
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = gFondDégradé(<Couleur Début> , <Couleur Fin> [, <Angle> [, <Couleur 3> [, <Distance couleur 3> [, <Couleur 4> [, <Distance couleur 4>]]]]])`
---

**`<Résultat> : (Chaîne de caractères)`**

Chaîne de contrôle à utiliser par exemple :

- avec la propriété [StyleFond](../Proprietes/2513019.md).

- dans les fonctions [ListeAjoute](../WDLang1/3049004.md), [ListeInsère](../WDLang1/3049002.md) ou [ListeModifie](../WDLang1/3049009.md).




**`<Couleur Début> : (Entier ou constante)`**

Couleur de début du dégradé. Cette couleur peut correspondre à :

- une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)),

- une couleur TSL (obtenue avec la fonction [TSL](../WDLang1/3029057.md)),

- une couleur prédéfinie (voir [les couleurs prédéfinies du WLangage](../WDLang5/3010002.md)).




**`<Couleur Fin> : (Entier ou constante)`**

Couleur de fin du dégradé. Cette couleur peut correspondre à :

- une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)),

- une couleur TSL (obtenue avec la fonction [TSL](../WDLang1/3029057.md)),

- une couleur prédéfinie (voir [les couleurs prédéfinies du WLangage](../WDLang5/3010002.md)).




**`<Angle> : (Entier optionnel)`**

Angle à utiliser pour la sélection des couleurs dans le dégradé. La valeur de l'angle est comprise entre 0 (angle horizontal, valeur par défaut) et 360°.

**`<Couleur 3> : (Entier optionnel ou constante optionnelle)`**

Couleur à utiliser dans le dégradé. Cette couleur peut correspondre à :

- une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)),

- une couleur TSL (obtenue avec la fonction [TSL](../WDLang1/3029057.md)),

- une couleur prédéfinie (voir [les couleurs prédéfinies du WLangage](../WDLang5/3010002.md)).




**`<Distance couleur 3> : (Entier optionnel)`**

Pourcentage séparant la couleur de début et la couleur 3.

**`<Couleur 4> : (Entier optionnel ou constante optionnelle)`**

Couleur à utiliser dans le dégradé. Cette couleur peut correspondre à :

- une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)),

- une couleur TSL (obtenue avec la fonction [TSL](../WDLang1/3029057.md)),

- une couleur prédéfinie (voir [les couleurs prédéfinies du WLangage](../WDLang5/3010002.md)).




**`<Distance couleur 4> : (Entier optionnel)`**

Pourcentage séparant la couleur de début et la couleur 4.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Paramètres du dégradé
<a name="parametres_degrade_ELTPARAGRAPHE000128"></a>Les paramètres de couleur sont utilisés de la façon suivante :

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=drectangledegrade.gif)


L'angle permet de définir l'orientation du dégradé.

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=angle_degrade.gif)


Les dégradés réalisé avec 3 ou 4 couleurs sont disponibles uniquement pour un angle de 0 ou de 90 degrés.

**Remarque** : Sur certains systèmes ou sous TSE 256 couleurs, les dégradés ne sont pas disponibles. Dans ce cas, un rectangle de la couleur &lt;Couleur début&gt; est affiché.


<a name="NOTE0_2"></a>




### Couleur de fond
<a name="couleur_fond_ELTPARAGRAPHE000147"></a>

- La fonction **gFondDégradé** peut être utilisée pour définir le fond d'une zone graphique dans les barres de titre et de message d'une fenêtre ou dans un champ Libellé.

- La fonction **gFondDégradé** appelée avec la constante *Transparent* permet de superposer du texte et un graphique.

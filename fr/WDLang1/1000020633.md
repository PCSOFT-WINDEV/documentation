


## Fonctions de gestion des toasts
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions de gestion des toasts sont les suivantes : 



|   |   |
| --- | --- |
| [ToastAffiche](../WDLang1/1000019358.md) | Affiche un message de type "Toast". |
| [ToastAfficheFI](../WDLang1/1000024643.md) | Ouvre un toast dont le contenu est basé sur une fenêtre interne. |
| [ToastAffichePopup](../WDLang1/1000020625.md) | Affiche une page de type popup pendant une durée donnée pour afficher un message de type "Toast". |
| [ToastSupprimeTout](../WDLang1/1000020336.md) | Supprime tous les toasts affichés par la fonction [ToastAffiche](../WDLang1/1000019358.md). |


## Description de Propriété (Type de variable)

***En anglais : Property Description ***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **Description de Propriété** permet d'obtenir des informations sur une propriété d'une classe.



**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Récupère la description de la classe type de la variable x
MaDescription est une Définition
MaDescription = RécupèreDéfinition(x)
// Récupère la description de la propriété Prop de la classe
p est une Description de Propriété 
p = MaDescription.Propriété["Prop"]
```

<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Propriétés spécifiques à la description des variables de type Description de Propriété
<a name="proprietes_specifiques_description_des_variables_type_description_propriete_ELTPARAGRAPHE000046"></a>Les propriétés suivantes peuvent être utilisées pour manipuler une variable de type Description de Propriété :

| **Nom de la propriété** | **Type manipulé** | **Effet** |
| --- | --- | --- |
| Associé | Booléen | <br><br>- <u><u><u><u>Vrai</u></u></u></u> si la propriété provient d'une classe associée, <br><br>- <u><u><u><u>Faux</u></u></u></u> si la propriété provient d'une classe de base. <br><br><br>**Cette propriété est disponible en lecture seulement**. |
| Déclaration | [Définition](../WDLang1/1000019540.md) | Définition du type qui déclare la propriété. <br>**Cette propriété est disponible en lecture seulement**. |
| Définition | [Définition](../WDLang1/1000019540.md) | Définition du type de la propriété. <br>**Cette propriété est disponible en lecture seulement**. |
| Ecriture | Booléen | <br><br>- <u><u><u><u>Vrai</u></u></u></u> si la propriété est accessible en écriture, <br><br>- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. <br>	**Cette propriété est disponible en lecture seulement**.<br><br><br> |
| Global | Booléen | <br><br>- <u><u><u><u>Vrai</u></u></u></u> si la propriété est globale, <br><br>- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. <br><br><br>**Cette propriété est disponible en lecture seulement**. |
| Hérité | Booléen | <br><br>- <u><u><u><u>Vrai</u></u></u></u> si la propriété est héritée d'une classe de base, <br><br>- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. <br><br><br>**Cette propriété est disponible en lecture seulement**. |
| Lecture | Booléen | <br><br>- <u><u><u><u>Vrai</u></u></u></u> si la propriété est accessible en lecture, <br><br>- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. <br><br><br>**Cette propriété est disponible en lecture seulement**. |
| Nom | Chaîne de caractères | Nom de la propriété. <br>Pour les types natifs du WLangage, le nom est renvoyé dans la langue du framework utilisé (français ou anglais).<br><br>**Cette propriété est disponible en lecture seulement**. |
| Virtuel | Booléen | <br><br>- <u><u><u><u>Vrai</u></u></u></u> si la propriété est virtuelle, <br><br>- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. <br><br><br>**Cette propriété est disponible en lecture seulement**. |






<a name="NOTE0_2"></a>


### Manipulation
<a name="manipulation_ELTPARAGRAPHE000181"></a>La propriété décrite peut être manipulée grâce à l'indirection : 

- {&lt;objet&gt;, &lt;Description de Propriété&gt;}

- {&lt;objet&gt;, &lt;Description de Propriété&gt;} = ...

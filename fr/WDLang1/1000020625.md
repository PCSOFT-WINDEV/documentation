
## ToastAffichePopup (Fonction)

***En anglais : ToastDisplayPopup***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Affiche une page de type popup pendant une durée donnée pour afficher un message de type "Toast".
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
ToastAffichePopup(POPUP_Toast)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`ToastAffichePopup(<Popup> [, <Durée d'affichage> [, <Cadrage vertical> [, <Cadrage horizontal>]]])`
---

**`<Popup> : (Nom de champ)`**

Nom du champ Popup à afficher.

**`<Durée d'affichage> : (Entier optionnel)`**

Constante indiquant la durée d'affichage du message Toast : 


|   |   |
| --- | --- |
| *toastCourt*<br>(Valeur par défaut) | Le message Toast sera affiché pendant une faible période de temps. |
| *toastLong* | Le message Toast sera affiché pendant une longue période de temps. |


**Remarque** : La durée d'affichage exacte dépend de l'appareil utilisé.

**`<Cadrage vertical> : (Entier optionnel)`**

Constante indiquant le cadrage vertical du message. Ce cadrage est relatif à la page.


|   |   |
| --- | --- |
| *cvBas*<br>(Valeur par défaut) | Affichage du message Toast en bas. |
| *cvHaut* | Affichage du message Toast en haut. |
| *cvMilieu* | Affichage du message Toast au milieu. |





**`<Cadrage horizontal> : (Entier optionnel)`**

Constante indiquant le cadrage horizontal du message. Ce cadrage est relatif à la page.


|   |   |
| --- | --- |
| *chCentre*<br>(Valeur par défaut) | Affichage du message Toast au centre. |
| *chDroite* | Affichage du message Toast sur la droite. |
| *chGauche* | Affichage du message Toast sur la gauche. |







<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Fonctionnement
<a name="fonctionnement_ELTPARAGRAPHE000084"></a>Le contenu des champs de la popup (texte, couleur de fond etc.) doit être initialisé avant l'appel de la fonction **ToastAffichePopup**. 

La fonction **ToastAffichePopup** ne peut être utilisée que pour un seul toast à la fois. Un second appel à la fonction **ToastAffichePopup** masquera le précédent toast.

Cette fonction est compatible avec la fonction [ToastAffiche](../WDLang1/1000019358.md). Les toasts affichés par la fonction [ToastAffiche](../WDLang1/1000019358.md) sont affichés au-dessus des popups (qu'elles soient affichées par la fonction [PopupAffiche](../WDLang2/1000019788.md), **ToastAffichePopup**, ...).
<a name="NOTE0_2"></a>


### Caractéristiques de la popup
<a name="caracteristiques_popup_ELTPARAGRAPHE000116"></a>

- La popup est affichée à la position demandée relative à la zone d'affichage de la page.

- La popup est fixée dans la page : la popup reste à la même position même si la page défile ou est redimensionnée.

- La popup est affichée sans GFI : le reste de la page (ce qui n'est pas recouvert par la popup) reste accessible.

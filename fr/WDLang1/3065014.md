
## Constantes de gestion de la base de registre
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


| Constante | Description | Utilisée par les fonctions |
| --- | --- | --- |
| **registreChercheClé** | Effectue une recherche dans le nom de clés de la base de registre. | [RegistreCherche](../WDLang1/3065005.md) |
| **registreChercheDonnée** | Effectue une recherche dans les données de la base de registre (uniquement les données de type Chaîne, Chaîne multiple ou Chaîne extensible). | [RegistreCherche](../WDLang1/3065005.md) |
| **registreChercheMotComplet** | Effectue une recherche dans la base de registre sur le mot complet spécifié dans le paramètre &lt;Chaîne recherchée&gt;. | [RegistreCherche](../WDLang1/3065005.md) |
| **registreChercheRécursif** | Effectue une recherche récursive dans la base de registre depuis la clé de base de la recherche. | [RegistreCherche](../WDLang1/3065005.md) |
| **registreChercheValeur** | Effectue une recherche dans la base de registre dans les noms des valeurs. | [RegistreCherche](../WDLang1/3065005.md) |
| **registreModeAuto** | Mode d'accès à la base de registre : Mode automatique | [Fonctions Registrexxx](../WDLang1/3065004.md) |
| **registreMode32** | Mode d'accès à la base de registre : Mode d'accès forcé pour accéder à la base de registre comme un programme 32 bits. | [Fonctions Registrexxx](../WDLang1/3065004.md) |
| **registreMode64** | Mode d'accès à la base de registre : Mode d'accès forcé pour accéder à la base de registre comme un programme 64 bits. | [Fonctions Registrexxx](../WDLang1/3065004.md) |
| **registreTypeBinaire** | Clé de la base de registre de type Binaire | [RegistreEcrit](../WDLang1/3065006.md), [RegistreTypeValeur](../WDLang1/1000017280.md) |
| **registreTypeChaîne** | Clé de la base de registre de type Chaîne | [RegistreEcrit](../WDLang1/3065006.md), [RegistreTypeValeur](../WDLang1/1000017280.md) |
| **registreTypeChaîneEnv** | Clé de la base de registre de type Chaîne contenant des variables d'environnement. | [RegistreEcrit](../WDLang1/3065006.md), [RegistreTypeValeur](../WDLang1/1000017280.md) |
| **registreTypeEntier** | Clé de la base de registre de type Entier sur 4 octets | [RegistreEcrit](../WDLang1/3065006.md), [RegistreTypeValeur](../WDLang1/1000017280.md) |
| **registreTypeEntierBigEndian** | Clé de la base de registre de type Entier au format big-endian | [RegistreEcrit](../WDLang1/3065006.md), [RegistreTypeValeur](../WDLang1/1000017280.md) |
| **registreTypeEntier_8** | Clé de la base de registre de type Entier sur 8 octets | [RegistreEcrit](../WDLang1/3065006.md), [RegistreTypeValeur](../WDLang1/1000017280.md) |
| **registreTypeMultiChaîne** | Clé de la base de registre de type multichaîne | [RegistreEcrit](../WDLang1/3065006.md), [RegistreTypeValeur](../WDLang1/1000017280.md) |




## &lt;Tableau&gt;.Distinct (Fonction)

***En anglais : &lt;Array&gt;.Distinct***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie un tableau sans doublons.

Les tableaux suivants peuvent être manipulés :

- Tableau d'éléments simples (entier, réel, chaîne de caractères) à une dimension.

- Tableau de structures à une dimension. La suppression des doublons est réalisée en fonction de la valeur d'un ou de plusieurs membres.





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Premier tableau d'entiers
tabMonTableau1 est un tableau d'entiers
tabMonTableau1.Ajoute(1)
tabMonTableau1.Ajoute(2)
tabMonTableau1.Ajoute(1)
tabMonTableau1.Ajoute(3)

// Tableau "distinct"
tabMonDistinct est un tableau d'entiers
tabMonDistinct = tabMonTableau1.Distinct()
// le résultat est 1; 2; 3
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Tableau WLangage>.Distinct()`
---

**`<Résultat> : (Tableau)`**

Variable de type Tableau contenant le tableau sans doublons.

**`<Tableau WLangage> : (Tableau)`**

Nom de la variable de type Tableau à manipuler. Ce tableau doit être un tableau à une dimension.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
**Divers**
Il est également possible de supprimer les doublons dans un tableau grâce à la fonction [&lt;Tableau&gt;.SupprimeDoublon](../WDLang1/1000024315.md). 

- La fonction [&lt;Tableau&gt;.SupprimeDoublon](../WDLang1/1000024315.md) manipule directement le tableau passé en paramètre. 

- La fonction **&lt;Tableau&gt;.Distinct** ne modifie pas le tableau passé en paramètre et renvoie un nouveau tableau sans les doublons. 
	
	**Enchaînement de fonctions**
	Il est possible d'enchaîner les différentes fonctions de manipulation d'un tableau. 
	Les fonctions suivantes peuvent être utilisées dans un enchaînement : 

	- [&lt;Tableau&gt;.Filtre](../WDLang1/1000024968.md)

	- [&lt;Tableau&gt;.Transforme](../WDLang1/1000025418.md)

	- [Union](../WDLang1/1000025408.md)

	- [Intersection](../WDLang1/1000025407.md)

	- [Distinct](../WDLang1/1000025406.md). 


Cet enchaînement pourra être utilisé comme source d'une instruction POUR TOUT ou se terminer par une des fonctions suivantes :

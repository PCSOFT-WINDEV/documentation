


## NuméroDeJourVersLettre (Fonction)

***En anglais : DayNumberInAlpha***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le nom du jour de la semaine en fonction de son numéro.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
sWarning = "Le bureau est fermé le " + NuméroDeJourVersLettre(2)
// Affiche "Le bureau est fermé le Mardi"
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = NuméroDeJourVersLettre(<Numéro du jour>)`
---

**`<Résultat> : (Chaîne de caractères)`**

Nom du jour de la semaine correspondant au numéro spécifié : Lundi, Mardi, Mercredi, Jeudi, Vendredi, Samedi, Dimanche.

**`<Numéro du jour> : (Entier)`**

Numéro du jour de la semaine compris entre 1 et 7 (1 correspond à Lundi et 7 correspond à Dimanche).



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Les noms des jours de la semaine dépendent : 

- de la langue en cours (définie par la fonction [Nation](../WDLang1/3054014.md)),

- des noms des jours personnalisés spécifiés dans les caractéristiques du projet (onglet "Langues" de la fenêtre de description du projet). 
	Rappel : Pour afficher la description du projet, sous le volet "Projet", dans le groupe "Projet", cliquez sur "Description".

- des paramètres Windows régionaux définis pour la langue en cours.

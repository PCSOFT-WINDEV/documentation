
## eDifférenceArrondi (Fonction)

***En anglais : eRoundDifference***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la différence d'arrondi entre :

- la valeur affichée d'un champ de type "Monétaire + Euro",

- le champ associé permettant un double affichage.



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Récupération de la différence d'arrondi entre le champ "SAI_PrixFranc"
// qui affiche les valeurs en Francs et le champ "SAI_PrixEuro" qui affiche
// les mêmes valeurs converties en Euros
ResDifférenceArrondi = eDifférenceArrondi(SAI_PrixFranc)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = eDifférenceArrondi(<Champ manipulé>)`
---

**`<Résultat> : (Monétaire)`**



- Différence d'arrondi existant entre le champ spécifié et le champ associé par double affichage. 
	Cette différence est exprimée dans la devise qui a la plus faible valeur monétaire. Cette valeur monétaire est fonction du taux de change et du nombre de décimales.

- 0 si aucune différence d'arrondi n'est constatée ou si le champ spécifié n'a pas de champ associé permettant un double affichage.




**`<Champ manipulé> : (Nom de champ)`**

Nom du champ de saisie ou colonne de type "Monétaire + Euro".




## Caract (Fonction)

***En anglais : Charact***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le caractère correspondant au code ASCII spécifié. Le code ASCII utilisé correspond à la norme ANSI utilisée par Windows.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple
<a class="notetitle" target="_blank" href="$DOC$=1000003024003&name=caract_fonction&product=WD">Voir des exemples supplémentaires</a>

```wl
CaractASCII = Caract(49)    // Retourne "1"
CaractASCII = Caract(87)    // Retourne "W"
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = Caract(<Code ASCII à convertir>)`
---

**`<Résultat> : (Chaîne de caractères)`**

Caractère correspondant au code ASCII précisé.

**`<Code ASCII à convertir> : (Entier)`**

Valeur à convertir (comprise entre 0 et 255). Si la valeur n'est pas conforme, une erreur du WLangage est générée. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Éviter l'utilisation de la fonction **Caract**
<a name="eviter_utilisation_fonction_docparampagetitleshort_ELTPARAGRAPHE000064"></a>Les constantes prédéfinies suivantes permettent d'éviter l'utilisation de la fonction **Caract** :

| Constante | Valeur | Fonction |
| --- | --- | --- |
| RC | Caract(13) + Caract(10) | Retour Chariot |
| TAB | Caract(9) | Tabulation |
| ESC | Caract(27) | Touche Echap |
| ESP | Caract(160) | Espace insécable |




## &lt;Traitement de texte&gt;.Ferme (Fonction)

***En anglais : &lt;Word Processing&gt;.Close***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ferme le fichier docx associé au champ Traitement de texte et libère le document. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
TT_MonDocument.Ferme()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Champ Traitement de texte>.Ferme()`
---

**`<Champ Traitement de texte> : (Nom de champ)`**

Nom du champ Traitement de texte à manipuler.

Le fichier physique associé au champ (s'il existe) est fermé. Les modifications réalisées dans le fichier ne sont pas sauvées avant sa fermeture (utilisez la fonction [&lt;Traitement de texte&gt;.Sauve](../WDLang1/1000024393.md) si nécessaire).

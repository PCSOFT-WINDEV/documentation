


## &lt;Table&gt;.CelluleSelectOccurrence (Fonction)

***En anglais : &lt;Table&gt;.CellSelectOccurrence***




	<B><B>Avertissement</B></B>

	A partir de la version <B><B>16 (56F)</B></B>,  cette fonction est conservée par compatibilité. Cette fonction est remplacée par la fonction [&lt;Table&gt;.SelectOccurrence](../WDLang1/1000024286.md).

<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le nombre de cellules sélectionnées dans un champ Table. 

**Remarque** : Le champ Table doit autoriser le mode de sélection à la cellule. Cette option est paramétrable dans l'onglet "IHM" de la description du champ Table.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
NbCelluleSel est un entier
Total est un réel
NumLigne est un entier
NumCol est un entier

NbCelluleSel = TABLE_TABLE1.CelluleSelectOccurrence()
POUR i = 1 A NbCelluleSel
	NumLigne = TABLE_TABLE1.CelluleSelect(i, tcsLigne)
	NumCol = TABLE_TABLE1.CelluleSelect(i, tcsColonne)
	Total = Total + TABLE_TABLE1[NumLigne, NumCol] 
	
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Champ Table>.CelluleSelectOccurrence()`
---

**`<Résultat> : (Entier)`**



- Nombre de cellules sélectionnées, 

- 0 si le mode de sélection à la cellule n'est pas sélectionné pour le champ Table (onglet "IHM" de la fenêtre de description du champ),

- 1 si le champ Table autorise uniquement la sélection d'une cellule.




**`<Champ Table> : (Nom de champ)`**

Nom du champ Table à manipuler.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Il est ensuite possible de récupérer la liste de ces cellules avec la fonction [&lt;Table&gt;.CelluleSelect](../WDLang1/1000023841.md).

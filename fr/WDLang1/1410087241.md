


## &lt;Variable Certificat&gt;.ExtraitValeurExtension (Fonction)

***En anglais : &lt;Certificate variable&gt;.ExtractExtensionValue***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la valeur d'une extension du certificat.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Extraction de la valeur de l'extension "1.3.6.1.4.1.311.10.3.4" du certificat
cert est un Certificat = CertificatSélecteur()

bufValeur est un Buffer = cert.ExtraitValeurExtension("1.3.6.1.4.1.311.10.3.4")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Certificat>.ExtraitValeurExtension(<OID>)`
---

**`<Résultat> : (Buffer)`**

Valeur de l'extension. Si l'extension n'existe pas, le buffer renvoyé est vide.

**`<Certificat> : (Variable de type Certificat)`**

Nom de la variable de type [Certificat](../WDLang1/1000019287.md) correspondant au certificat dans lequel l'extension doit être cherchée.

**`<OID> : (Chaîne de caractères)`**

OID de l'extension.

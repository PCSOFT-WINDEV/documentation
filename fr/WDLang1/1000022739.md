
## RuptureSupprimeTout (Fonction)

***En anglais : BreakDeleteAll***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime toutes les ruptures présentes dans un champ Table ou un champ Zone répétée. Ces ruptures ont été précédemment ajoutées via la fonction [RuptureAjoute](../WDLang1/1000022737.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Regroupe les clients par Code postal (CP)
// 1. Suppression des anciennes ruptures
RuptureSupprimeTout(TABLE_Client)
// 2. Création de la nouvelle rupture
RuptureAjoute(TABLE_Client.CP, "Rup_CP")
TableTrie(TABLE_Client, "CP")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`RuptureSupprimeTout(<Champ à manipuler>)`
---

**`<Champ à manipuler> : (Nom de champ)`**

Nom du champ à manipuler : 

- champ Table.

- champ Zone répétée. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- La fonction **RuptureSupprimeTout** supprime toutes les ruptures créées avec la fonction [RuptureAjoute](../WDLang1/1000022737.md). Si aucune rupture n'a été créée, la fonction n'a aucun effet. 

- Pour supprimer une seule rupture, utilisez la fonction [RuptureSupprime](../WDLang1/1000022738.md).

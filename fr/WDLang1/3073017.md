
## SysIconisé (Fonction)

***En anglais : SysIconize***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de savoir si une fenêtre est iconisée. Cette fenêtre n'a pas été créée par WINDEV.

**Remarque** : Pour savoir si une fenêtre WINDEV est iconisée, utilisez la fonction [FenTaille](../WDLang1/3038020.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Savoir si la fenêtre "Microsoft Word - Ventes.DOC" est iconisée
ResHandle = SysFenHandle("Microsoft Word - Ventes.DOC")
ResIconiseFen = SysIconisé(ResHandle)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = SysIconisé(<Handle de la fenêtre>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la fenêtre spécifiée est iconisée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Handle de la fenêtre> : (Entier)`**

Handle de la fenêtre à tester. Ce handle peut être obtenu avec la fonction [SysFenHandle](../WDLang1/3073007.md).

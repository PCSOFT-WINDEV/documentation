
## SysMétrique (Fonction)

***En anglais : SysMetric***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la résolution d'un élément spécifique.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Largeur du curseur
ResMétrique = SysMétrique(metCxCurseur)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = SysMétrique(<Élément>)`
---

**`<Résultat> : (Entier)`**

Résolution de l'élément spécifié.

**`<Élément> : (Constante de type entier)`**

Indique l'élément pour lequel la résolution doit être déterminée :


|   |   |
| --- | --- |
| *metAligneMenu* | Menu d'alignement des "Popup". Si cette valeur est égale à zéro, le coté gauche du menu de la popup est aligné avec le coté gauche de l'objet correspondant dans la barre de menu. Si cette valeur est différente de zéro, le coté gauche du menu de la "popup" est aligné avec le coté droit de l'objet correspondant dans la barre de menu.<br><br> |
| *metCxBord* | Largeur d'une bordure fixe de fenêtre. |
| *metCxBordEpais* | Largeur du cadre quand la fenêtre a pour style WS_DLGFRAME. |
| *metCxBtTitre* | Largeur des boutons contenus dans la barre de titre d'une fenêtre.<br><br> |
| *metCxClientMax* | Largeur de la zone client dans une fenêtre plein écran. |
| *metCxCurseur* | Largeur du curseur. |
| *metCxDefil* | Largeur de la case de défilement (en pouces) dans la barre de défilement horizontale.<br><br> |
| *metCxDim* | Largeur du cadre de la fenêtre qui peut être modifié.<br><br> |
| *metCxDoubleClic* | Largeur du rectangle entourant le lieu du premier clic lors d'un double clic. Le second clic doit avoir lieu dans cette zone pour que le système considère l'ensemble des deux clics comme un double clic. |
| *metCxEcran* | Largeur de l'écran. |
| *metCxFenMin* | Largeur minimale de la fenêtre.<br><br> |
| *metCxHScroll* | Largeur de la flèche présente sur une barre de déplacement horizontale. |
| *metCxIcone* | Largeur d'icônes. |
| *metCxIconeRect* | Largeur des rectangles utilisés pour positionner les icônes les unes à côté des autres. |
| *metCxMinDim* | Largeur minimale en redimensionnement par l'utilisateur d'une fenêtre.<br><br> |
| *metCxVScroll* | Largeur de l'image de la flèche sur la barre de défilement verticale.<br><br> |
| *metCyBord* | Hauteur d'une bordure fixe de fenêtre. |
| *metCyBordEpais* | Hauteur du cadre quand la fenêtre a pour style WS_DLGFRAME. |
| *metCyBtTitre* | Hauteur des images contenues dans la barre de titre.<br><br> |
| *metCyClientMax* | Hauteur de la zone client dans une fenêtre plein écran (équivalent à la hauteur de l'écran moins la hauteur de la barre de titre). |
| *metCyCurseur* | Hauteur du curseur. |
| *metCyDefil* | Hauteur de la case de défilement (en pouces) dans la barre de défilement horizontale. |
| *metCyDim* | Hauteur du cadre de la fenêtre qui peut être modifié.<br><br> |
| *metCyDoubleClic* | Hauteur du rectangle entourant le lieu du premier clic lors d'un double clic. Le second clic doit avoir lieu dans cette zone pour que le système considère l'ensemble des deux clics comme un double clic. |
| *metCyEcran* | Hauteur de l'écran. |
| *metCyFenMin* | Hauteur minimale de la fenêtre.<br><br> |
| *metCyHScroll* | Hauteur de la flèche présente sur une barre de déplacement horizontale. |
| *metCyIcone* | Hauteur d'icônes. |
| *metCyIconeRect* | Hauteur des rectangles utilisés pour positionner les icônes les unes à côté des autres. |
| *metCyKanji* | Hauteur de la fenêtre "Kanji".<br><br> |
| *metCyMenu* | Hauteur de la barre de menu déroulant. Cette hauteur correspond à la hauteur du menu plus la hauteur du cadre de la fenêtre qui ne peut être redimensionné. |
| *metCyMinDim* | Hauteur minimale en redimensionnement par l'utilisateur d'une fenêtre.<br><br> |
| *metCyTitre* | Hauteur du titre de la fenêtre. Cette hauteur correspond à la hauteur du titre plus la hauteur du cadre de la fenêtre qui ne peut être redimensionné. |
| *metCyVScroll* | Largeur (en pixels) des ascenseurs verticaux. |
| *metDBCSActif* | Différent de zéro si la version en cours de Windows utilise des caractères codés sur 2 bits; sinon, renvoie la valeur zéro.<br><br> |
| *metInverseBoutons* | Différent de zéro si les boutons gauche et droit de la souris sont inversés.<br><br> |
| *metPenWindows* | Handle (identifiant) de la librairie de liens dynamiques (DLL) du "Pen Windows" si celui-ci est installé.<br><br> |
| *metSouris* | Différent de zéro si le driver de la souris est installé. |

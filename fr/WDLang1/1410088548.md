
## diagPointDePassage (Type de variable)

***En anglais : diagWaypoint***
			
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **diagPointDePassage** permet de définir toutes les caractéristiques avancées d'un point de passage pour un lien. Les caractéristiques de ce point de passage peuvent être définies et modifiées à l'aide de différentes propriétés WLangage. 

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
Forme est un diagForme
dLien est un diagLien
dPassage est un diagPointDePassage
dPassage.X = 100
dPassage.Y = 50
dLien.PointDePassage.Ajoute(dPassage)
```

<a name="PROPS"></a>

## Propriétés


### Propriétés spécifiques à la description des variables de type diagPointDePassage 
<a name="proprietes_specifiques_description_des_variables_type_diagpointdepassage_ELTPARAGRAPHE000025"></a>Les propriétés suivantes peuvent être utilisées pour manipuler un point de passage :

| Nom de la propriété | Type manipulé | Effet |
| --- | --- | --- |
| Lien | [diagLien](../WDLang1/1410088454.md) | Lien manipulé. |
| X | Réel | Abscisse (X) du point de passage dans le diagramme. Cette position est exprimée en pixels. |
| Y | Réel | Ordonnée (Y) du point de passage dans le diagramme. Cette position est exprimée en pixels. |




## &lt;Tableur&gt;.InsèreLigne (Fonction)

***En anglais : &lt;Spreadsheet&gt;.InsertRow***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Insère une ligne vide dans un champ Tableur présent dans une fenêtre.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Insère plusieurs lignes
POUR I = 2 A 5
	TBLR_Tableur.InsèreLigne(I)
 FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Champ Tableur>.InsèreLigne([<Ligne>])`
---

**`<Champ Tableur> : (Nom de champ)`**

Nom du champ Tableur à manipuler.

**`<Ligne> : (Entier optionnel)`**

Numéro de la ligne avant laquelle la nouvelle ligne sera insérée. Par exemple, si ce paramètre vaut 2, la nouvelle ligne sera insérée à la position 2. Les lignes existantes seront décalées vers le bas et les formules seront mises à jour. 
Si ce paramètre n'est pas précisé et si une ligne est sélectionnée dans le champ, l'insertion sera réalisée avant la ligne sélectionnée. Dans le cas contraire (aucune ligne sélectionnée), la fonction ne fait rien.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- L'opération d'insertion est ajoutée dans la liste des opérations à annuler : l'utilisateur peut annuler cette opération via le menu contextuel ou via la combinaison de touches Ctrl + Z.

- Cette fonction est utilisable uniquement sur un champ Tableur présent dans une fenêtre.


## SysVersionWindows (Fonction)

***En anglais : SysWindowsVersion***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie des informations sur la version de Windows utilisée sur le poste en cours.



![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Renvoie la version du système.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) 
```wl
// Numéro de version de Windows utilisé sur le poste en cours
ResVersion = SysVersionWindows(SysVersionNuméro)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = SysVersionWindows([<Type d'information>])`
---

**`<Résultat> : (Chaîne de caractères)`**



- Information demandée sur la version Windows, 

- Chaîne vide ("") en cas d'erreur.




**`<Type d'information> : (Constante optionnelle)`**

Indique l'information à demander : 



|   |   |
| --- | --- |
| Paramètre non spécifié | Version de Windows :<br><br>- "11" pour Windows 11<br><br>- "10" pour Windows 10<br><br>- "8" pour Windows 8<br><br>- "7" pour Windows 7<br><br>- "VISTA"<br><br>- "LINUX"<br><br>- "2022S" pour Windows 2022 Server<br><br>- "2019S" pour Windows 2019 Server<br><br>- "2016S" pour Windows 2016 Server<br><br>- "2012S" pour Windows 2012 Server<br><br>- "2008S" pour Windows 2008 Server<br><br>- "2003S" pour Windows 2003 Server<br><br>- "XP" pour Windows XP<br><br>- "ME" pour Windows Millenium<br><br>- "NT 4", "NT 5" (pour Windows 2000 par exemple)<br><br>- "NT 3.5"<br><br>- "98"<br><br>- "95"<br><br>- "32s" pour Windows 3.1<br><br><br>Chaîne vide si la version de Windows est inconnue.<br><br><br><br><br><br> |
| *sysProcesseur* | Mode d'exécution du programme dans l'environnement en cours :<br><br>- "32" : le programme fonctionne en 32 bits dans un environnement 32 bits.<br><br>- "64" : le programme fonctionne en 32 bits ou en 64 bits dans un environnement 64 bits.<br><br><br><br><br>![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Cette constante n'est pas disponible. |
| *sysVersionCompil* | Numéro de compilation de Windows.<br>Par exemple : "1381".<br><br>![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Cette constante n'est pas disponible. |
| *sysVersionInfo* | Informations spécifiques sur la version de Windows. Par exemple :<br><br>- "Service Pack x" en NT<br><br>- "" en Windows 95A (chaîne vide)<br><br>- "B" en Windows 95B<br><br>- " " en Windows 98 (un espace).<br><br>- "Pro" pour Windows XP Pro<br><br>- "Home Edition" pour Windows XP familial.<br><br><br><br><br>![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Cette constante n'est pas disponible. |
| *sysVersionNuméro* | Numéro de version du noyau de Windows.<br>Par exemple :<br><br>- "3.51" en NT 3.51.<br><br>- "10" en Windows 11 21H2...<br><br><br><br><br>![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Cette constante n'est pas disponible. |
| *sysVersionPlateForme* | Plateforme du système :<br><br>- "Win32s" pour WIN32 sous 3.11<br><br>- "Windows" pour Windows 95/98/ME<br><br>- "NT" pour Windows NT 4.0/2000/XP/2003 Serveur<br><br>- "UNIX" pour Linux.<br><br><br><br><br>![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Cette constante n'est pas disponible. |


![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Ce paramètre n'est pas disponible.

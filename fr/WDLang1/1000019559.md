
## RécupèreDéfinitionType (Fonction)

***En anglais : GetTypeDefinition***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Récupère la description d'un type connu par son nom.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
PROCEDURE Proc(NomClasse est une chaîne)

// Récupère la définition de la classe connue par son nom
MaDéfinition est une Définition
MaDéfinition = RécupèreDéfinitionType(NomClasse)
```



```wl
Description_état est une Définition = RécupèreDéfinitionType(ETAT_CommandesClients)
Description_syntaxe est une Description de Syntaxe = Description_état.SyntaxeDéclaration
// Parcours des paramètres de l'état
POUR TOUT x DE Description_syntaxe.Paramètre
	Trace(x.Nom)
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = RécupèreDéfinitionType(<Nom de l'élément>)`
---

**`<Résultat> : (Définition)`**

Variable de type [Définition](../WDLang1/1000019540.md). 

**`<Nom de l'élément> : (Chaîne de caractères)`**



- Nom de la classe ou de la structure pour lequel la définition doit être récupérée.

- Nom de la fenêtre, de la page ou de l'état pour lequel la définition doit être récupérée.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Récupération de la définition
<a name="recuperation_definition_ELTPARAGRAPHE000066"></a>Si la classe ou la structure est inconnue, la fonction **RécupèreDéfinitionType** renvoie une définition Null accompagnée de l'erreur non fatale "Définition invalide". 

Si l'élément n'est pas alloué, la fonction **RécupèreDéfinitionType** renvoie une définition Null accompagnée de l'erreur non fatale "Elément pas alloué".

Il est possible de traiter les cas d'erreur :

- soit en comparant la description obtenue à Null.

- soit en testant la valeur de la variable [ErreurDétectée](../WDLang1/3087001.md).





<a name="NOTE0_2"></a>


### Limites
<a name="limites_ELTPARAGRAPHE000090"></a>

- Dans un projet, il n'est pas possible de récupérer la définition d'une classe ou d'une structure provenant : 

	- d'un composant externe.

	- d'un composant interne.




- Dans un composant externe, il n'est pas possible de récupérer la définition d'une classe ou d'une structure provenant du projet hôte.


## Constantes de gestion des masques
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les constantes de gestion des masques sont les suivantes : 

| Constante | Description | Utilisée par les fonctions |
| --- | --- | --- |
| **mcpBloqueSortie** | Interdit de sortir du champ tant que la valeur saisie n'est pas valide (par exemple, 5 chiffre pour le code postal de la France). | [MasqueCodePostal](../WDLang1/1000024742.md) |
| **mcpSortieLibre** | Autorise la sortie du champ même si le masque de saisie n'est pas vérifié. | [MasqueCodePostal](../WDLang1/1000024742.md) |
| **mtelAvecOuSansPréfixe** | Caractéristique du numéro de téléphone : Numéro de téléphone avec ou sans préfixe international. | [MasqueNuméroDeTéléphone](../WDLang1/1000024741.md) |
| **mtelForcePréfixe** | Caractéristique du numéro de téléphone : Numéro de téléphone avec toujours le préfixe international. Exemple : +33 4 67 03 20 32 | [MasqueNuméroDeTéléphone](../WDLang1/1000024741.md) |
| **mtelSansPréfixe** | Caractéristique du numéro de téléphone : 	Numéro de téléphone sans préfixe international. Exemple : 04 67 03 20 32. | [MasqueNuméroDeTéléphone](../WDLang1/1000024741.md) |

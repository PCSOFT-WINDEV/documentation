


## TDBSupprimeTout (Fonction)

***En anglais : DashDeleteAll***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime tous les Widgets affichés dans un champ Tableau de bord.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Vérifie qu'il existe au moins un widget
SI TDBOccurrence(TDB_TableauDeBord, toTotal) <= 0 ALORS
	Info("Il n'y a aucun widget à supprimer")
	RETOUR
FIN

// Supprime tous les widgets
TDBSupprimeTout(TDB_TableauDeBord)
ToastAffiche("Tous les widgets ont été supprimés", toastCourt, cvMilieu, chCentre)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`TDBSupprimeTout(<Champ Tableau de bord>)`
---

**`<Champ Tableau de bord> : (Nom de champ)`**

Nom du champ Tableau de bord à manipuler.

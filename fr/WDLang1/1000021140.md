


## &lt;Diagramme de Gantt&gt;.SupprimeTout (Fonction)

***En anglais : &lt;Gantt chart&gt;.DeleteAll***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime toutes les tâches dans une colonne d'un champ Table (ou Table hiérarchique) de type Diagramme de Gantt.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Bouton "Tout supprimer"
SI OuiNon("Voulez-vous supprimer toutes les tâches ?") ALORS
	COL_Gantt1.SupprimeTout()
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Colonne Gantt>.SupprimeTout()`
---

**`<Colonne Gantt> : (Nom de champ)`**

Nom de la colonne (d'un champ Table ou Table hiérarchique) de type Diagramme de Gantt à manipuler.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Toutes les tâches sont supprimées.

- Le champ Table qui contient la colonne est vidé (équivalent à la fonction [&lt;Table&gt;.SupprimeTout](../WDLang1/1000024292.md)).

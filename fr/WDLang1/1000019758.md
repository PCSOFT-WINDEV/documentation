


## OrgaSupprime (Fonction)

***En anglais : OrgDelete***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime un élément et l'ensemble de ses "fils" d'un champ Organigramme.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Suppression d'un élément et de l'ensemble de ses "fils" dans le champ Organigramme
NumElément est un entier 
NumElément = OrgaInfoXY(ORGA_MonOrganigramme, SourisPosX(), SourisPosY())
OrgaSupprime(ORGA_MonOrganigramme, NumElément)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`OrgaSupprime(<Champ Organigramme> , <Indice>)`
---

**`<Champ Organigramme> : (Nom de champ)`**

Nom du champ Organigramme à manipuler.

**`<Indice> : (Entier)`**

Indice de l'élément à supprimer. Cet indice est renvoyé par exemple par la fonction [OrgaInfoXY](../WDLang1/1000019759.md).

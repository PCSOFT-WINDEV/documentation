
## gFond (Fonction)

***En anglais : gBackground***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Construit une séquence de caractères permettant de définir la couleur d'affichage (couleur de fond d'un texte, d'un cercle, d'un rectangle, ...). 

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Dans une fenêtre, cette séquence de caractères permet de définir la couleur de fond d'une texte ou d'une figure (rectangle ou cercle) : 

- dans un élément d'un champ Liste ou Combo.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) dans la barre de titre et/ou de message d'une fenêtre. 

- dans un champ Libellé. 




**Remarque** : Pour utiliser une couleur dégradée, utilisez la fonction [gFondDégradé](../WDLang1/1000017060.md).




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 
```wl
// Ajout du mot "Jaune" sur fond jaune dans la liste "LISTE_ListeCouleur"
ListeAjoute(LISTE_ListeCouleur, gFond(JauneClair) + "Jaune")
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = gFond(<Couleur>)`
---

**`<Résultat> : (Chaîne de caractères)`**

Séquence de caractères définissant la couleur de fond du texte, du cercle, du rectangle, ... 

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Cette séquence de caractères peut être : 

- utilisée dans les fonctions [ListeAjoute](../WDLang1/3049004.md), [ListeInsère](../WDLang1/3049002.md) ou [ListeModifie](../WDLang1/3049009.md).

- affectée à un champ Libellé. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) utilisée dans la barre de titre ou de message d'une fenêtre. 




**`<Couleur> : (Entier ou constante)`**

Couleur d'affichage du fond du texte ou de la partie intérieure d'une figure (rectangle ou cercle). Cette couleur peut correspondre à :

- une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)),

- une couleur TSL (obtenue avec la fonction [TSL](../WDLang1/3029057.md)),

- [une couleur prédéfinie du WLangage](../WDLang5/3010002.md).  

- la constante *Transparent* (permet par exemple de superposer du texte et un graphique).

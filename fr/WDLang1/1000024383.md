


## &lt;Variable SignalAutomatique&gt;.Ferme (Fonction)

***En anglais : &lt;AutomaticEvent variable&gt;.Close***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ferme un signal de synchronisation entre plusieurs threads.


<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Signal>.Ferme()`
---

**`<Signal> : (Variable de type SignalAutomatique)`**

Nom d'une variable de type [SignalAutomatique](../WDLang1/1000021270.md) à fermer.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Cette fonction est équivalente à la syntaxe suivante de la fonction [SignalModifie](../WDLang1/3077017.md) : 
	
	```txt
	SignalModifie(<Signal>, signalFermé)
	```


- **Cas d'un signal automatique** : Le signal reste fermé. 
	**Nouveaux threads** : Si d'autres threads arrivent, ils sont mis en attente.

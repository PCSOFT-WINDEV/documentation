


## Fonctions de gestion des certificats (syntaxe préfixée)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions suivantes permettent de manipuler des certificats Windows (via une syntaxe préfixée) :



|   |   |
| --- | --- |
| [&lt;Variable Certificat&gt;.ExtraitValeurExtension](../WDLang1/1410087241.md) | Renvoie la valeur d'une extension du certificat. |
| [&lt;Variable Certificat&gt;.OuvrePropriétés](../WDLang1/1000024378.md) | Ouvre la fenêtre Windows permettant de connaître les propriétés d'un certificat. |

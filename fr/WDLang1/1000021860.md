
## gImageExiste (Fonction)

***En anglais : gImageExists***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Vérifie si une image a bien été chargée en mémoire. 

**Rappel** : Les fonctions [ChargeImage](../WDLang1/3029022.md) ou [ChargePlancheImage](../WDLang1/1000018826.md) permettent de charger des images en mémoire. Ces images peuvent être numérotées de 1 à 9. La fonction **gImageExiste** permet de savoir si l'image N a bien été chargée en mémoire. Cette image pourra être ensuite utilisée par la fonction [gImage](../WDLang1/3029042.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Charge l'image 3 si elle n'a pas encore été chargée
SI PAS gImageExiste(3) ALORS
	ChargeImage(3, "Logo.png")
FIN
LIB_TITRE = "Notre logo : " = gImage(3)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = gImageExiste(<Identifiant de l'image>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'image a été chargée en mémoire. 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Identifiant de l'image> : (Entier)`**

Identifiant de l'image (compris entre 1 et 9) recherchée en mémoire. Cet identifiant est donné par les fonctions [ChargeImage](../WDLang1/3029022.md) ou [ChargePlancheImage](../WDLang1/1000018826.md).

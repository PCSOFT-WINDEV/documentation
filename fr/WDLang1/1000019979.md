


## &lt;Organigramme&gt;.Occurrence (Fonction)

***En anglais : &lt;Organization Chart&gt;.Occurrence ***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le nombre d'éléments présents dans un champ Organigramme.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Recherche de l'élément selon son nom
NbElement est un entier = ORGA_Direction.Occurrence()
POUR i = 1 A NbElement
	SI ORGA_Direction[i].Titre = "Boris" ALORS
		ORGA_Direction[i].SousTitre = "Employé du mois"
	FIN 
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Champ Organigramme>.Occurrence()`
---

**`<Résultat> : (Entier)`**

Nombre d'éléments présents dans le champ Organigramme. Ce nombre correspond : 

- au nombre d'éléments ajoutés par les fonctions [&lt;Organigramme&gt;.Ajoute](../WDLang1/1000019975.md) et [&lt;Organigramme&gt;.AjouteFils](../WDLang1/1000019976.md). 

- au nombre d'éléments ajoutés grâce à la liaison fichier si le champ est lié à une source de données. Dans ce cas, ces éléments ont été affichés grâce à la fonction [&lt;Organigramme&gt;.Affiche](../WDLang1/1000019974.md).




**`<Champ Organigramme> : (Nom de champ)`**

Nom du champ Organigramme à manipuler.

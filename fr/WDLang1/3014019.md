


## Instance (Fonction)

***En anglais : Instance***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le "Handle système" (HINSTANCE) du programme en cours. Il est ensuite possible d'utiliser ce handle dans les API système qui nécessitent ce paramètre.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Récupération de l'instance du programme
hInstance est un entier = Instance()
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = Instance()`
---

**`<Résultat> : (Entier)`**

Handle système du programme.

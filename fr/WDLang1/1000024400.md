


## &lt;Traitement de texte&gt;.VersHTML (Fonction)

***En anglais : &lt;Word Processing&gt;.ToHTML***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Convertit le contenu d'un champ Traitement de texte en un fichier HTML.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple

<a name="Exemple2"></a>

```wl
// Export en HTML du contenu d'un champ Traitement de texte
// Code de clic du bouton "Export HTML" 
TT_MonDoc.VersHTML(fRepExe() + [fSep] + "tt.htm")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Champ Traitement de texte>.VersHTML([<Chemin>] [, <Gestion des styles CSS>])`
---

**`<Résultat> : (Booléen ou Buffer)`**



- Si &lt;Chemin&gt; est précisé : 

	- <u><u><u><u>Vrai</u></u></u></u> si le fichier a été généré, 

	- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




- Si &lt;Chemin&gt; n'est pas précisé, &lt;Résultat&gt; correspond à un buffer contenant le code HTML.




**`<Champ Traitement de texte> : (Nom de champ)`**

Nom du champ Traitement de texte à manipuler.

**`<Chemin> : (Chaîne de caractères optionnelle)`**

Chemin du fichier HTML à générer. Si ce chemin n'est pas précisé, le code HTML est renvoyé sous forme de buffer.

**`<Gestion des styles CSS> : (Constante optionnelle)`**

Attributs de style à utiliser :



|   |   |
| --- | --- |
| *dvhAvecCSS*<br>(Valeur par défaut) | Applique les styles CSS **externes**<br>Le code CSS est généré dans un fichier ".CSS" externe à la page HTML. Ce fichier est créé dans le même répertoire que la page HTML générée. Ce fichier porte le même nom que la page HTML avec l'extension ".CSS". Un lien vers ce fichier est ajouté dans le code HTML de la page.<br><br>Si le document contient des images, celles-ci sont sauvegardées dans un sous-répertoire portant le même nom que le fichier HTML généré. |
| *dvhSansCSS* | Applique les styles CSS **inline** : le code CSS est généré et intégré dans le code HTML de la page générée. Chaque balise contient les styles CSS qui lui sont appliqués. <br><br>Si le document contient des images, celles-ci sont intégrées et sont encodées en base 64. <br><br>Cette constante permet par exemple de réaliser une génération HTML adaptée aux nouvelles messageries emails. <br><br>Remarque : si une image contenue dans un email fait plus de 100 Ko, elle sera généralement affichée comme une pièce jointe de l'email, même si elle est encodée en base 64 dans l'email. |

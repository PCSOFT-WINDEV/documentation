


## zmlSupprimeTout (Fonction)

***En anglais : mlzDeleteAll***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime toutes les lignes dynamiques dans un champ Zone multiligne. 




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Réinitialisation de ZM_Réseau
zmlSupprimetout(ZM_Réseau)
zmlAjouteLigne(ZM_Réseau)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`zmlSupprimeTout(<Champ Zone multiligne>)`
---

**`<Champ Zone multiligne> : (Nom de champ)`**

Nom du champ Zone multiligne à manipuler.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Tous les lignes ajoutées par les fonctions [zmlAjouteLigne](../WDLang1/1000020199.md) et [zmlInsèreLigne](../WDLang1/1000020207.md) sont supprimées.

- Les lignes statiques définies sous l'éditeur de fenêtres ne sont pas impactées.

- La taille du champ Zone multiligne est modifiée pour prendre en compte les lignes supprimées : le champ se réduit par le bas. Si des champs sont situés sous le champ Zone multiligne, ils sont déplacés vers le haut. La hauteur utile de la fenêtre est diminuée en conséquence.

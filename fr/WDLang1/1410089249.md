
## Brésil.VérifieCNPJ (Fonction)

***En anglais : Brazil.CheckCNPJ***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Teste la validité d'un numéro CNPJ (numéro d'identification d'entreprise au Brésil).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Test d'une variable
MonNuméroCNPJ est une chaîne = "43337004000172"
Brésil.VérifieCNPJ(MonNuméroCNPJ)	// renvoie Vrai

// Test d'un champ
SAI_NuméroCNPJ = "ABC"
Brésil.VérifieCNPJ(SAI_NuméroCNPJ)	// renvoie Faux
```



```wl
// Test d'une variable
MonNuméroCNPJ est une chaîne
MonRésultat est un booléen
MonRésultat = Saisie("Entrez le numéro CNPJ : ", MonNuméroCNPJ)
SI Brésil.VérifieCNPJ(MonNuméroCNPJ) = Faux ALORS
	Info("Numéro CNPJ invalide")
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = Brésil.VérifieCNPJ(<Numéro à vérifier>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'élément spécifié contient un numéro CNPJ valide.

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Numéro à vérifier> : (Nom de champ ou variable de type chaîne)`**

Numéro CNPJ à vérifier. Ce paramètre peut correspondre : 

- au nom d'une variable de type chaîne.

- au nom d'un champ.

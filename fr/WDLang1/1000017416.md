


## &lt;Liste&gt;.SupprimeTout (Fonction)

***En anglais : &lt;List Box&gt;.DeleteAll***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Vide de tous ses éléments un champ Liste par programmation (y compris Liste image) ou un champ Combo par programmation.

**Remarque** : La fonction **&lt;Liste&gt;.SupprimeTout** peut être utilisée indifféremment sur les champs Liste monosélection et multisélection.











<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Suppression de tous les éléments présents dans le champ "LISTE_ListeClient"
LISTE_ListeClient.SupprimeTout()
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Champ Liste>.SupprimeTout()`
---

**`<Champ Liste> : (Nom de champ)`**

Nom du champ Liste (ou Combo) à manipuler.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
<a name="NOTE0_2"></a>


### Divers
<a name="divers_ELTPARAGRAPHE000261"></a>

- Pour supprimer l'ensemble des lignes d'un champ Table, utilisez la fonction [&lt;Table&gt;.SupprimeTout](../WDLang1/1000024292.md).

- La fonction **&lt;Liste&gt;.SupprimeTout** peut être utilisée sur une colonne d'un champ Table de type "Combo".

- Dans le cas des champs Combo avec saisie, la zone de saisie est vidée.

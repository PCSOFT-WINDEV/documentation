


## DateHeureVersRéel (Fonction)

***En anglais : DateTimeToReal***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Convertit une date et une heure en un réel sur 8 octets. Ce réel correspond  :

- soit au numéro du jour depuis le 1er janvier 1900 (compatible Excel, commence à 1). 

- soit au numéro du jour depuis le 1er janvier 1904 (compatible Excel avec l'option "Utiliser le calendrier depuis 1904", commence à 0). 




Remarque : L'option "Utiliser le calendrier depuis 1904" est configurable dans les options avancées de MS Excel. Sous Open Office, cette option est disponible dans les options de Open Office Calc (rubrique "Calculer", option "Date"). 




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affiche "3.25"
Info(DateHeureVersRéel("190001030600"))
```
<a name="Exemple2"></a>

```wl
MonJour est un DateHeure
MonJour = "190001030600"
// Affiche "3.25"
Info(DateHeureVersRéel(MonJour))
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = DateHeureVersRéel(<DateHeure à convertir> [, <Depuis 1904>])`
---

**`<Résultat> : (Réel)`**



- Numéro du jour correspondant à la variable de type DateHeure passée en paramètre. 

- -1 en cas d'erreur (date et heure invalides). 




**`<DateHeure à convertir> : (Variable de type DateHeure ou chaîne de caractères)`**

Valeur de la date et de l'heure à convertir. Ce paramètre peut correspondre à : 

- une variable de type DateHeure. 

- une chaîne de caractères au format : 

	- AAAAMMJJHHmmSSCCC

	- AAAAMMJJHHmmSSCC

	- AAAAMMJJHHmmSS

	- AAAAMMJJHHmm

	- AAAAMMJJHH

	- AAAAMMJJ







**`<Depuis 1904> : (Booléen optionnel)`**

Mode de calcul : 

- <u><u><u><u>Vrai</u></u></u></u> si la date est calculée à partir de 1904. 

- <u><u><u><u>Faux</u></u></u></u> (valeur par défaut) si la date est calculée à partir de 1900. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Mode de calcul
<a name="mode_calcul_ELTPARAGRAPHE000121"></a>La partie entière du résultat donne le numéro du jour de la partie date du paramètre : 

- la numérotation commence à 1 pour le 01/01/1900. 

- la numérotation commence à 0 pour le 01/01/1904 si le paramètre &lt;Depuis 1904&gt; vaut <u><u><u><u>Vrai</u></u></u></u>. 


La partie décimale donne la fraction de jour de la partie heure du paramètre (par exemple 06h00 donne une partie décimale de 0.25).

Toute conversion de type peut engendrer une perte de précision.
<a name="NOTE0_2"></a>


### Dates/heures valides
<a name="datesheures_valides_ELTPARAGRAPHE000141"></a>Le paramètre &lt;DateHeure à convertir&gt; est considéré invalide : 

- si la date est invalide ou antérieure à la date de début de numérotation. La date doit être comprise entre le 01/01/1900 ou 01/01/1904 et le 31/12/999. La validité de la date peut être vérifiée avec la fonction [DateValide](../WDLang1/3027003.md).

- si l'heure est invalide. La validité de l'heure peut être vérifiée avec la fonction [HeureValide](../WDLang1/3027021.md).

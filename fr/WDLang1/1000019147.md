


## AgendaSelectMoins (Fonction)

***En anglais : OrganizerSelectMinus***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Désélectionne un rendez-vous dans un champ Agenda.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ajout d'un rendez-vous dans un champ Agenda
nRangRendezVous est un entier
nRangRendezVous = AgendaAjouteRendezVous(AGD_MonAgenda, ...
				"Réunion équipe commerciale", ...
				DateSys() + "1400", DateSys() + "1600")
// Sélection du rendez-vous ajouté
AgendaSelectPlus(AGD_MonAgenda, nRangRendezVous)
...
// Clic sur le champ Bouton "Désélectionner"
AgendaSelectMoins(AGD_MonAgenda)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Désélectionner un rendez-vous identifié par son indice

`AgendaSelectMoins(<Champ Agenda> [, <Indice du rendez-vous>])`
---

**`<Champ Agenda> : (Nom de champ)`**

Nom du champ Agenda à manipuler. Ce champ peut correspondre : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) au champ Agenda d'une fenêtre.




**`<Indice du rendez-vous> : (Entier optionnel)`**

Indice du rendez-vous à désélectionner. Les indices des rendez-vous commencent à 1.
Si ce paramètre n'est pas précisé, le rendez-vous actuellement sélectionné est désélectionné.
Si ce paramètre est précisé et ne correspond pas à l'indice du rendez-vous actuellement sélectionné dans le champ Agenda, la fonction est sans effet.


<a name="SYNTAXE2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 
### Désélectionner un rendez-vous identifié par une variable de type RendezVous

`AgendaSelectMoins(<Champ Agenda> [, <Rendez-vous à désélectionner>])`
---

**`<Champ Agenda> : (Nom de champ)`**

Nom du champ Agenda à manipuler (champ d'une fenêtre). 

**`<Rendez-vous à désélectionner> : (Variable de type RendezVous optionnelle)`**

Nom de la variable de type [RendezVous](../WDLang1/1000019244.md) correspondant au rendez-vous à désélectionner. 
Si ce paramètre n'est pas précisé, le rendez-vous actuellement sélectionné est désélectionné.
Si ce paramètre est précisé et ne correspond pas au rendez-vous actuellement sélectionné dans le champ Agenda, la fonction est sans effet.




## MultimédiaVolume (Fonction)

***En anglais : MultimediaVolume***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie ou modifie le volume sonore du fichier multimédia joué dans un champ Multimédia. 








<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) 
```wl
MonFichier est une chaîne = "C:\Ma musique\mysong.mp3"
MultimédiaJoue(MM_Multimédia1, MonFichier)
// Fixe le son
MultimédiaVolume(MM_Multimédia1, 50)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le volume sonore

`<Résultat> = MultimédiaVolume(<Champ Multimédia>)`
---

**`<Résultat> : (Entier)`**

Volume sonore actuel (compris entre 0 et 100, 0 pour muet).

**`<Champ Multimédia> : (Nom de champ)`**

Nom du champ Multimédia à manipuler.  


<a name="SYNTAXE2"></a>

### Modifier le volume sonore

`<Résultat> = MultimédiaVolume(<Champ Multimédia> , <Nouveau volume>)`
---

**`<Résultat> : (Entier)`**

Volume sonore avant modification (compris entre 0 et 100, 0 pour muet).

**`<Champ Multimédia> : (Nom de champ)`**

Nom du champ Multimédia à manipuler.

**`<Nouveau volume> : (Entier)`**

Volume sonore souhaité (compris entre 0 et 100, 0 pour muet).

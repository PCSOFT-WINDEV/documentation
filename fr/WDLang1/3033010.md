


## Fonctions Euro
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions permettant de gérer l'Euro sont les suivantes :



|   |   |
| --- | --- |
| [eChangeDevise](../WDLang1/3033012.md) | Convertit une devise en une autre devise. |
| [eDeviseAffParDéfaut](../WDLang1/3033009.md) | Identifie ou modifie la devise d'affichage utilisée par défaut. |
| [eDeviseMemParDéfaut](../WDLang1/3033016.md) | Identifie ou modifie la devise de mémorisation utilisée par défaut. |
| [eDifférenceArrondi](../WDLang1/3033015.md) | Renvoie la différence d'arrondi entre :<br><br>- la valeur affichée d'un champ de type "Monétaire + Euro",<br><br>- le champ associé permettant un double affichage.<br><br><br> |
| [eInitDevise](../WDLang1/3033003.md) | Ajoute ou modifie les caractéristiques d'une devise dans la table des changes des devises. |
| [eNomDevise](../WDLang1/3033006.md) | Renvoie le libellé d'une devise. |
| [eTauxDeChange](../WDLang1/3033004.md) | Renvoie la valeur du taux de change d'une devise. |
| [eVersDevise](../WDLang1/3033001.md) | Convertit une valeur exprimée en Euro en une autre devise. |
| [eVersEuro](../WDLang1/3033005.md) | Convertit en Euros une valeur exprimée dans une devise spécifique. |

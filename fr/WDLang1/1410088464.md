


## DiagrammeInfoXY (Fonction)

***En anglais : DiagramInfoXY***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Indique la présence d'une forme à une position donnée dans le champ Editeur de diagrammes. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
soit nIndiceForme = DiagrammeInfoXY(EDIAG_Diagramme, SourisPosX, SourisPosY)
gDiag est un Diagramme <- EDIAG_Diagramme.Valeur
POUR i = 1 _À_ gDiag.Forme.Occurrence 
	// Modification de l'opacité de la forme survolée
	SI nIndiceForme = i ALORS 
		gDiag.Forme[nIndiceForme].Opacité = 50 
	SINON 
		gDiag.Forme[nIndiceForme].Opacité = 100
	FIN
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = DiagrammeInfoXY(<Editeur de diagrammes> , <X> , <Y>)`
---

**`<Résultat> : (Entier)`**



- Indice de la forme si une forme a été trouvée la position spécifiée, 

- 0 dans le cas contraire.




**`<Editeur de diagrammes> : (Nom de champ)`**

Nom du champ Editeur de diagrammes à manipuler.

**`<X> : (Entier)`**

Abscisse de la position recherchée. 

**`<Y> : (Entier)`**

Ordonnée de la position recherchée.

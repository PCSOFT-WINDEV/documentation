
## Description de Valeur (Type de variable)

***En anglais : Value description***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Une variable de type **Description de Valeur** permet de manipuler par programmation la description d'une valeur d'énumération.

Les caractéristiques de ce type peuvent être connues à l'aide de différentes propriétés WLangage. 

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Récupère la définition du type de d
d est une Définition
d = RécupèreDéfinition(Poste1)
// Affiche le nom du type de x
Trace(d.Nom)
//  Lister les valeurs de l'énumération
SI d.Type = wlEnumération ALORS
	POUR TOUT MonElément DE d.Valeur
		Trace(MonElément.Nom)	
	FIN
FIN
// Autre possibilité : Utiliser une description de valeur
SI d.Type = wlEnumération ALORS
	D1 est une Description de Valeur
	POUR TOUT D1 DE d.Valeur
		Trace(D1.Nom)	
	FIN
FIN
```

## Remarques
<a name="NOTE0_1"></a>


### Propriétés spécifiques à la description des variables de type Description de valeur
<a name="proprietes_specifiques_description_des_variables_type_description_valeur_ELTPARAGRAPHE000031"></a>Les propriétés suivantes peuvent être utilisées pour manipuler une description d'une valeur d'une énumération :

| Nom de la propriété | Type manipulé | Effet |
| --- | --- | --- |
| Alias | Booléen | <br><br>- <u><u><u><u>Vrai</u></u></u></u> si la valeur est un alias d'une valeur existante, <br><br>- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. <br><br><br>**Cette propriété est disponible en lecture seulement**. |
| Déclaration | Variable de type [Définition](../WDLang1/1000019540.md) | Déclaration de la valeur. <br><br>**Cette propriété est disponible en lecture seulement**. |
| Hérité | Booléen | <br><br>- <u><u><u><u>Vrai</u></u></u></u> si la valeur est héritée d'une énumération étendue, <br><br>- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. <br><br><br>**Cette propriété est disponible en lecture seulement**. |
| Nom | Chaîne de caractères | Nom de la valeur. <br><br>**Cette propriété est disponible en lecture seulement**. |
| Valeur | Tous types | Valeur. <br><br>**Cette propriété est disponible en lecture seulement**. |
| ValeurAssociée | Tous types | Valeur associée. <br><br>**Cette propriété est disponible en lecture seulement**. |

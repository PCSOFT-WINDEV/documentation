
## Belgique.VérifieNISS (Fonction)

***En anglais : Belgium.CheckNISS***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Teste la validité d'un numéro NISS (numéro d'identification de la sécurité sociale de Belgique).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Test d'une variable
MonNuméroNISS est une chaîne = "85073003328"
Belgique.VérifieNISS(MonNuméroNISS)	// renvoie Vrai

// Test d'un champ
SAI_NuméroNISS = "ABC"
Belgique.VérifieNISS(SAI_NuméroNISS)	// renvoie Faux
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = Belgique.VérifieNISS(<Numéro à vérifier>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'élément spécifié contient un numéro NISS valide,

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Numéro à vérifier> : (Nom de champ ou variable de type chaîne)`**

Numéro NISS à vérifier. Ce paramètre peut correspondre :

- au nom d'une variable de type chaîne.

- au nom d'un champ.




## docElémentLien (Type de variable)

***En anglais : docElementLink***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **docElémentLien** permet de connaître toutes les caractéristiques avancées d'un élément de type Lien présent dans une variable de type [docElément](../WDLang1/1000022484.md). 

Les caractéristiques du type de variable **docElémentLien** peuvent être connues et modifiées à l'aide de différentes propriétés WLangage. 

**ATTENTION : Ce type ne doit pas être utilisé directement. Il doit être utilisé uniquement via le type [docElément](../WDLang1/1000022484.md).**
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Chargement du document
TT_MonDoc = ""

// Création d'une élément de type lien
elt est un docElément
elt.Type = typeDocElémentLien
elt.Lien.Lien = "https://www.pcsoft.com"
elt.Lien.Type = typeDocLienURL

// Création d'un fragment de document lié au champ utilisé
f est un docFragment(TT_MonDoc, 1)

// Ajout du lien dans le document
DocInsère(f, elt)

// Recherche le lien pour ajouter le texte et formater le style
POUR TOUT eltInsere DE f.Elément
	soit parent = eltInsere.Parent
	parent.Lien.Contenu.Texte = "Site de PC SOFT"
	parent.Lien.Contenu.MiseEnForme.CouleurTexte = BleuFoncé
	parent.Lien.Contenu.MiseEnForme.CouleurSouligné = BleuFoncé
	parent.Lien.Contenu.MiseEnForme.PoliceSoulignée = Vrai
	SORTIR
FIN
```

## Remarques
<a name="NOTE0_1"></a>


### Propriétés spécifiques à la description des variables de type docElémentLien
<a name="proprietes_specifiques_description_des_variables_type_docelementlien_ELTPARAGRAPHE000033"></a>Les propriétés suivantes peuvent être utilisées pour manipuler un lien dans un document : 


| Nom de la propriété | Type manipulé | Effet |
| --- | --- | --- |
| Bulle | Chaîne de caractères | Contenu de la bulle associée au lien. |
| Contenu | Variable de type [docFragment](../WDLang1/1000022482.md) | Texte associé au lien.  |
| Lien | Chaîne de caractères | Contenu du lien (chemin du fichier, nom du site, ...). Le libellé du lien ne peut pas être connu. |
| Type | Entier | Type de lien : <br><br>- *typeDocLienSignet* : Lien de type signet. <br><br>- *typeDocLienURL* : Lien de type URL permettant d'ouvrir un document.<br><br><br> |

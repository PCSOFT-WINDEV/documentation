
## RegistreCherche (Fonction)

***En anglais : RegistrySeek***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Recherche une chaîne de caractères dans la base de registre.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Récupération de toutes les valeurs "FullScreen" 
// de la clé "HKEY_CURRENT_USER\Console"
Clé est une chaîne = "HKEY_CURRENT_USER\Console"
TANTQUE Clé <> ""
	Clé = RegistreCherche(Clé, "FullScreen")
	Info(Clé)
FIN
```

```wl
// Récupération de toutes les valeurs "FullScreen" 
// de la clé "HKEY_CURRENT_USER\Console"
// Mode d'accès forcé "Application 64 bits"
Clé est une chaîne = "HKEY_CURRENT_USER\Console"
TANTQUE Clé <> ""
	Clé = RegistreCherche(registreMode64, Clé, "FullScreen")
	Info(Clé)
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = RegistreCherche([<Mode d'accès>, ] <Chemin de base de la recherche> , <Chaîne recherchée> [, <Options>])`
---

**`<Résultat> : (Chaîne de caractères)`**

Résultat de la recherche (chaîne vide en cas d'erreur). Le résultat est de la forme : 

```txt
<Chemin de la clé> + TAB + <Sous-clé> [ + TAB + <Valeur> [ + TAB + <Donnée>]]
```


**`<Mode d'accès> : (Constante de type Entier)`**

Mode d'accès à la base de registre :



|   |   |
| --- | --- |
| *registreMode32* | Mode d'accès forcé pour accéder à la base de registre comme un programme 32 bits. |
| *registreMode64* | Mode d'accès forcé pour accéder à la base de registre comme un programme 64 bits. |
| *registreModeAuto*<br>(Valeur par défaut) | Mode d'accès à la base de registre automatique : <br><br>- une application 32 bits exécutée sur un système 32 bits manipule la base de registre comme un programme 32 bits.<br><br>- une application 32 bits exécutée sur un système 64 bits manipule la base de registre à partir de la branche suivante :<br>	<br><pre><code>HKEY_LOCAL_MACHINE\SOFTWARE\Wow6432Node</code></pre><br><br><br>- une application 64 bits exécutée sur un système 64 bits manipule la base de registre comme un programme 64 bits.<br><br><br> |



**`<Chemin de base de la recherche> : (Chaîne de caractères)`**

Chemin de la clé à partir de laquelle la recherche est effectuée. Ce chemin est de la forme : 

```txt
<Chemin de la clé>[ + TAB + <Sous-clé> [ + TAB + <Valeur> [ + TAB + <Donnée>]]]
```


**`<Chaîne recherchée> : (Chaîne de caractères)`**

Chaîne de caractères recherchée dans la base de registre.

**`<Options> : (Constante ou combinaison de constante optionnelle de type Entier)`**

Mode de recherche. Le mode de recherche par défaut correspond à : 
RegistreChercheClé + RegistreChercheValeur + RegistreChercheDonnée + RegistreChercheRécursif.



|   |   |
| --- | --- |
| *registreChercheClé* | Effectue une recherche dans le nom des clés. |
| *registreChercheDonnée* | Effectue une recherche dans les données (uniquement les données de type Chaîne, Chaîne multiple ou Chaîne extensible). |
| *registreChercheMotComplet* | Effectue une recherche sur le mot complet spécifié dans le paramètre &lt;Chaîne recherchée&gt;. |
| *registreChercheRécursif* | Effectue une recherche récursive depuis la clé de base de la recherche. |
| *registreChercheValeur* | Effectue une recherche dans les noms des valeurs. |




## InsèreColonne (Fonction)

***En anglais : InsertColumn***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Insère une colonne à une position donnée dans une propriété tableau d'un type avancé à deux dimensions (xlsDocument, ...). 






<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`InsèreColonne(<Tableau WLangage> , <Indice d'insertion> [, <Élément ligne 1> [... [, <Élément ligne N>]]])`
---

**`<Tableau WLangage> : (Tableau)`**

Nom de la variable de type Tableau à manipuler. Cette variable doit correspondre à un tableau à deux dimensions.

**`<Indice d'insertion> : (Entier)`**

Indice du tableau où la colonne doit être insérée. Toutes les colonnes à partir de cet indice seront décalées de 1 colonne. 
Si &lt;Indice d'insertion&gt; est égal au nombre de colonnes du tableau + 1, la colonne est ajoutée à la fin du tableau (équivalent à la fonction [AjouteColonne](../WDLang1/1000017350.md)). 
Si &lt;Indice d'insertion&gt; est supérieur au nombre de colonnes du tableau + 1, une erreur WLangage est générée.

**`<Élément ligne 1> : (Type des éléments du tableau, optionnel)`**

Élément de la colonne à insérer dans le tableau spécifié. Chaque élément correspond à une ligne dans le tableau. Le type de l'élément ajouté doit être compatible avec le type de la ligne concernée.
Si aucun de ces paramètres n'est spécifié, une colonne vide est insérée dans le tableau. Les lignes sont initialisées avec la valeur par défaut du type des autres éléments du tableau.
Si un de ces paramètres n'est pas spécifié, la ligne correspondante est initialisée avec la valeur par défaut du type des autres éléments du tableau.



**`<Élément ligne N> : (Type des éléments du tableau, optionnel)`**

Élément de la colonne à insérer dans le tableau spécifié. Chaque élément correspond à une ligne dans le tableau. Le type de l'élément ajouté doit être compatible avec le type de la ligne concernée.
Si aucun de ces paramètres n'est spécifié, une colonne vide est insérée dans le tableau. Les lignes sont initialisées avec la valeur par défaut du type des autres éléments du tableau.
Si un de ces paramètres n'est pas spécifié, la ligne correspondante est initialisée avec la valeur par défaut du type des autres éléments du tableau.

## Remarques


### Insertion d'une colonne dans une propriété tableau d'un type avancé
<a name="insertion_une_colonne_dans_une_propriete_tableau_type_avance_ELTPARAGRAPHE000073"></a>Lors de l'utilisation de la fonction **InsèreColonne** :

- la variable de type avancé doit être allouée.

- le type avancé doit avoir un énumérateur de type collection modifiable.

- le type avancé est automatiquement agrandi pour accueillir la nouvelle colonne.

- les lignes sont initialisées avec les valeurs passées en paramètre. Si aucune valeur n'est passée en paramètre, les lignes sont initialisées avec la valeur par défaut du type des éléments du tableau.

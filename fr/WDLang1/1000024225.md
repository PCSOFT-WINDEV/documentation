
## Constantes utilisées par les fonctions de gestion des champs Panneau dockable
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les constantes de gestion des champs Panneau dockable sont les suivantes :

| Constante | Détail | Fonction |
| --- | --- | --- |
| **padoBas** | Attache le champ Panneau dockable en bas. | [PanneauAttache](../WDLang1/1000023990.md),<br><br>[PanneauCrée](../WDLang1/1000023989.md) |
| **padoDroite** | Attache le champ Panneau dockable à droite. | [PanneauAttache](../WDLang1/1000023990.md),<br><br>[PanneauCrée](../WDLang1/1000023989.md) |
| **padoFlottant** | Le champ Panneau dockable est flottant. | [PanneauAttache](../WDLang1/1000023990.md),<br><br>[PanneauCrée](../WDLang1/1000023989.md) |
| **padoGauche** | Attache le champ Panneau dockable à gauche. | [PanneauAttache](../WDLang1/1000023990.md),<br><br>[PanneauCrée](../WDLang1/1000023989.md) |
| **padoHaut** | Attache le champ Panneau dockable en haut. | [PanneauAttache](../WDLang1/1000023990.md),<br><br>[PanneauCrée](../WDLang1/1000023989.md) |

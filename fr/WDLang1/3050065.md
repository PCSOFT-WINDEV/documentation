
## ArcTang (Fonction)

***En anglais : ArcTan***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie l'arc tangente d'une valeur numérique (entière ou réelle).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Récupération de l'arc tangente de 12
ResTang = ArcTang(12)
// Renvoie 85,2363583093
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = ArcTang(<Valeur numérique>)`
---

**`<Résultat> : (Réel)`**

Arc tangente (angle en degré de la tangente) de la valeur numérique spécifiée, compris dans l'intervalle [-90, 90] bornes exclues.

**`<Valeur numérique> : (Entier ou réel)`**

Valeur numérique (tangente) à manipuler.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Comment retrouver un angle à partir de sa tangente ?
<a name="comment_retrouver_angle_partir_tangente_ELTPARAGRAPHE000070"></a>Dans un triangle rectangle, la tangente d'un angle est égale au rapport du coté opposé sur le coté adjacent. Comment obtenir l'angle ? Il suffit d'utiliser la fonction **ArcTang**. 

Exemple : Le code suivant : 

```wl
Trace("ArcTang(1) = " + ArcTang(1))
Trace("ArcTang(3/4) = " + ArcTang(3/4))
```
affiche dans la fenêtre de trace : 

```txt
ArcTang(1) = 45
ArcTang(3/4) = 36.86989764584
```

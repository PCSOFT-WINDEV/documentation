


## Fonctions de gestion des champs Organigramme
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions de gestion des champs Organigramme sont les suivantes : 



|   |   |
| --- | --- |
| [OrgaAffiche](../WDLang1/1000019764.md) | Réinitialise le contenu d'un champ Organigramme à partir de sa source de données (fichier de données HFSQL par exemple). |
| [OrgaAjoute](../WDLang1/1000019729.md) | Ajoute un nouvel élément "racine" dans un champ Organigramme. |
| [OrgaAjouteFils](../WDLang1/1000019763.md) | Ajoute un élément fils dans un champ Organigramme. |
| [OrgaInfoXY](../WDLang1/1000019759.md) | Renvoie l'indice de l'élément se trouvant à une position donnée dans le champ Organigramme. |
| [OrgaModifie](../WDLang1/1000019921.md) | Modifie un élément dans un champ Organigramme. |
| [OrgaOccurrence](../WDLang1/1000019971.md) | Renvoie le nombre d'éléments présents dans un champ Organigramme. |
| [OrgaSauveImage](../WDLang1/1000019972.md) | Crée une image représentant la totalité d'un champ Organigramme. |
| [OrgaSelect](../WDLang1/1000019762.md) | Renvoie l'indice de l'élément sélectionné dans un champ Organigramme. |
| [OrgaSelectMoins](../WDLang1/1000019761.md) | Dé-sélectionne un élément dans un champ Organigramme. |
| [OrgaSelectPlus](../WDLang1/1000019760.md) | Sélectionne un élément dans un champ Organigramme. |
| [OrgaSupprime](../WDLang1/1000019758.md) | Supprime un élément et l'ensemble de ses "fils" d'un champ Organigramme. |
| [OrgaSupprimeTout](../WDLang1/1000019749.md) | Supprime tous les éléments d'un champ Organigramme. |






- ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=WDOrganigramme.gif) ***Exemples didactiques (WINDEV)*** : **WD Organigramme** <br>Cet exemple permet de créer des organigrammes.<br>Un organigramme sert à indiquer la répartition des responsables à l'intérieur d'une entreprise, et les relations de commandement qui existent entre eux.




## &lt;Diagramme de Gantt&gt;.SupprimeTâche (Fonction)

***En anglais : &lt;Gantt chart&gt;.DeleteTask***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime une tâche dans une colonne d'un champ Table (ou Table hiérarchique) de type Diagramme de Gantt.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Bouton "Supprime"
SI OuiNon("Voulez-vous supprimer cette tâche ?") ALORS
	COL_Gantt.SupprimeTâche()
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Colonne Gantt>.SupprimeTâche([<Identifiant>])`
---

**`<Colonne Gantt> : (Nom de champ)`**

Nom de la colonne (d'un champ Table ou Table hiérarchique) de type Diagramme de Gantt à manipuler.

**`<Identifiant> : (Chaîne de caractères optionnelle)`**

Identifiant de la tâche à supprimer. Cet identifiant correspond à la propriété **ID** définie dans la variable de type [TâcheGantt](../WDLang1/1000020881.md).

Si cet identifiant n'existe pas, une erreur WLangage est affichée. 

Si ce paramètre n'est pas précisé et si un élément est sélectionné, l'élément est supprimé. 

Si ce paramètre n'est pas précisé et si aucun élément n'est sélectionné, la fonction n'a aucune action

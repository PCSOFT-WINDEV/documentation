
## TitreSuivant (Fonction)

***En anglais : NextTitle***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie le titre de la prochaine fenêtre à ouvrir.

**Rappel** : Par défaut, le titre de la fenêtre est celui défini sous l'éditeur.












<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) 
```wl
// Modification du titre de la prochaine fenêtre à ouvrir
TitreSuivant("Modification d'une fiche client")
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`TitreSuivant(<Nouveau titre>)`
---

**`<Nouveau titre> : (Chaîne de caractères)`**

Nouveau titre de la prochaine fenêtre ou boîte de dialogue à ouvrir.





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Titre des boîtes de dialogues
<a name="titre_des_boites_dialogues_ELTPARAGRAPHE000091"></a>Par défaut, les fenêtres ouvertes par les fonctions [Avertissement](../WDLang1/3021009.md), [Confirmer](../WDLang1/3021007.md), [Dialogue](../WDLang1/3021015.md), [Erreur](../WDLang1/3021013.md), [ErreurInfo](../WDLang1/3013008.md), [Info](../WDLang1/3021011.md), [OKAnnuler](../WDLang1/3021004.md) ou [OuiNon](../WDLang1/3021005.md) ont le même titre que la fenêtre précédente. La fonction **TitreSuivant** permet de modifier ce titre. 


<a name="NOTE0_2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) 

### Présentation du titre
<a name="presentation_titre_ELTPARAGRAPHE000149"></a>Les [fonctions de dessin](../WDLang1/3029033.md) dans les listes (fonctions commençant par "g") permettent de :

- ajouter des dessins au titre,

- modifier la mise en forme d'un titre (couleur de fond, couleur de texte, etc.) avec les fonctions [gFond](../WDLang1/3029037.md) et [gStylo](../WDLang1/3029039.md).




Par exemple : pour modifier la couleur de texte et la couleur de fond d'un titre :


```wl
TitreSuivant(EOT + gFond(RVB(0, 255, 255)) + gStylo(RVB(255, 0, 0)) + "Test de couleur")
```


**Cas particulier sous Windows Vista et supérieur** : Les fenêtres utilisant le look "Aero system" ne peuvent pas avoir de titre graphique : la partie graphique est ignorée. Si "Aero" n'est pas actif, le titre graphique sera affiché normalement.

### Fenêtre iconisée
<a name="fenetre_iconisee_ELTPARAGRAPHE000176"></a>Quand une fenêtre est iconisée, le titre de cette fenêtre apparaît à côté de l'icône dans la barre de tâches.

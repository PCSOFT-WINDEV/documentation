
## NONBinaire (Fonction)

***En anglais : BinaryNOT***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le résultat du NON logique (bit à bit) d'une valeur.

**Rappel** : L'opération NON logique est équivalente au "Complément à un". La négation logique intervient sur TOUS les bits de la valeur.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Récupération du NON logique
ResNONBinaire = NONBinaire(6)  // Renvoie -7
ResNONBinaire = NONBinaire(2)  // Renvoie -3
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = NONBinaire(<Valeur>)`
---

**`<Résultat> : (Entier)`**

Résultat du NON logique.

**`<Valeur> : (Entier)`**

Valeur à tester.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Calcul bit à bit
<a name="calcul_bit_bit_ELTPARAGRAPHE000077"></a>**NONBinaire** effectue le calcul bit à bit de &lt;Valeur&gt; selon la table suivante :


|   |   |   |
| --- | --- | --- |
| **Valeur** | 0 | 1 |
| **Résultat** | 1 | 0 |




<a name="NOTE0_2"></a>




### Opérateur binaire
<a name="operateur_binaire_ELTPARAGRAPHE000102"></a>La fonction **NONBinaire** est équivalente à l'opérateur binaire **~**. Pour plus de détails, consultez [Opérateurs binaires](../Motscles/1512009.md).




## Fonctions de gestion des champs Panneau dockable
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions de gestion des champs Panneau dockable sont les suivantes : 



|   |   |
| --- | --- |
| [PanneauActive](../WDLang1/1000024396.md) | Rend un panneau actif lorsque plusieurs panneaux sont dockés entre eux. |
| [PanneauAttache](../WDLang1/1000023990.md) | Attache ou change la position d'attache d'un champ Panneau dockable. |
| [PanneauChargeConfiguration](../WDLang1/1000023995.md) | Restaure la configuration des champs Panneau dockable d'une fenêtre. |
| [PanneauCrée](../WDLang1/1000023989.md) | Crée un nouveau champ Panneau dockable dans la fenêtre en cours. |
| [PanneauDétache](../WDLang1/1000023991.md) | Détache un champ Panneau dockable pour le transformer en une fenêtre flottante. |
| [PanneauEnumère](../WDLang1/1000023993.md) | Enumère les champs Panneau dockable d'une fenêtre. |
| [PanneauPosition](../WDLang1/1000023992.md) | Récupère la position d'attache d'un champ Panneau dockable. |
| [PanneauSauveConfiguration](../WDLang1/1000023994.md) | Sauve la configuration courante des champs Panneau dockable d'une fenêtre. |

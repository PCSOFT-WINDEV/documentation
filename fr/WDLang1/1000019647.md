
## fSep (Fonction)

***En anglais : fSep***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le séparateur de répertoire selon la plateforme en cours ('\\' ou '/'). Il est ainsi possible de construire des chemins de fichiers sans se préoccuper de la plateforme d'exécution (Windows ou Linux par exemple).

 



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Renvoie le nom complet du fichier en utilisant le séparateur de la plateforme en cours
sNomComplet = sMonRepertoire + [fSep()] + "fichier.txt"
// En Windows : sMonrépertoire + "\fichier.txt"
// En Linux :  sMonrépertoire + "/fichier.txt"
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = fSep()`
---

**`<Résultat> : (Chaîne de caractères)`**

Séparateur utilisé pour la plateforme en cours : '\\' ou  '/'.

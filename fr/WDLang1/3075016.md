


## TableauDéplaceLigne (Fonction)

***En anglais : ArrayMoveLine***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet :

- soit de déplacer une ligne dans un tableau à deux dimensions ou dans une propriété tableau d'un type avancé. Les autres lignes du tableau sont décalées.

- soit d'échanger deux lignes dans un tableau à deux dimensions ou dans une propriété tableau d'un type avancé.




**Remarque** : Cette fonction est équivalente à la fonction [DéplaceLigne](../WDLang1/1000017359.md).






<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonTableau est un tableau de 5 par 2 chaînes
MonTableau[1,1] = "A"
MonTableau[1,2] = "a"
MonTableau[2,1] = "B"
MonTableau[2,2] = "b"
MonTableau[3,1] = "C"
MonTableau[3,2] = "c"
MonTableau[4,1] = "D"
MonTableau[4,2] = "d"
MonTableau[5,1] = "E"
MonTableau[5,2] = "e"
// Déplacer la ligne 2 à la 4° position
TableauDéplaceLigne(MonTableau, 2, 4)
// Le tableau contient les éléments :
// "A", "a"
// "C", "c"
// "D", "d"
// "B", "b"
// "E", "e"
```
<a name="Exemple2"></a>

```wl
MonTableau est un tableau de 5 par 2 chaînes
MonTableau[1,1] = "A"
MonTableau[1,2] = "a"
MonTableau[2,1] = "B"
MonTableau[2,2] = "b"
MonTableau[3,1] = "C"
MonTableau[3,2] = "c"
MonTableau[4,1] = "D"
MonTableau[4,2] = "d"
MonTableau[5,1] = "E"
MonTableau[5,2] = "e"
// Échanger la ligne 2 avec la ligne 4 
TableauDéplaceLigne(MonTableau, 2, 4, tdEchange)
// Le tableau contient les éléments :
// "A", "a"
// "D", "d"
// "C", "c"
// "B", "b"
// "E", "e"
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Déplacer une ligne dans un tableau WLangage ou dans une propriété tableau d'un type avancé 

`TableauDéplaceLigne(<Tableau WLangage> , <Indice départ> , <Indice arrivée> [, <tdDéplace>])`
---

**`<Tableau WLangage> : (Tableau)`**

Nom de la variable de type Tableau à manipuler. Ce tableau doit être un tableau à deux dimensions.

**`<Indice départ> : (Entier)`**

Indice de la ligne à déplacer.

**`<Indice arrivée> : (Entier)`**

Nouvel indice de la ligne déplacée.

**`<tdDéplace> : (Constante optionnelle)`**

*tdDéplace* : Constante permettant de déplacer une ligne dans un tableau.  


<a name="SYNTAXE2"></a>

### Échanger deux lignes dans un tableau WLangage ou dans une propriété tableau d'un type avancé 

`TableauDéplaceLigne(<Tableau WLangage> , <Indice de la 1° ligne> , <Indice de la 2° ligne> , <tdEchange>)`
---

**`<Tableau WLangage> : (Tableau)`**

Nom de la variable de type Tableau à manipuler. Ce tableau doit être un tableau à deux dimensions.

**`<Indice de la 1° ligne> : (Entier)`**

Indice de la première ligne à échanger.

**`<Indice de la 2° ligne> : (Entier)`**

Indice de la seconde ligne à échanger.

**`<tdEchange> : (Constante)`**

*tdEchange* : Constante permettant d'échanger deux lignes dans un tableau.


## PiloteConfirmer (Fonction)

***En anglais : EmulateConfirm***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Pilote le prochain appel de la fonction [Confirmer](../WDLang1/3021007.md), puis clique automatiquement sur le bouton passé en paramètre.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Fermeture automatique de la prochaine boîte de dialogue Confirmer
// La réponse sur lequel le clic aura lieu est Oui
PiloteConfirmer(Oui)

// Traitement 
// ...

SELON Confirmer("Souhaitez-vous enregistrer les modifications ?")
	CAS Oui
	// Enregistrement des modifications
	CAS Annuler
	// Annulation des modifications
FIN
// Le message est fermé automatiquement, par un clic sur Oui
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`PiloteConfirmer(<Bouton>)`
---

**`<Bouton> : (Constante de type entier)`**

Indique le bouton de la fenêtre confirmer sur lequel le clic doit avoir lieu :


|   |   |
| --- | --- |
| *Annuler* | Clic sur le bouton [ANNULER] |
| *Non* | Clic sur le bouton [NON] |
| *Oui* | Clic sur le bouton [OUI] |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Vérification de la fonction
<a name="verification_fonction_ELTPARAGRAPHE000064"></a>L'existence d'une fonction de pilotage ([PiloteAvertissement](../WDLang1/1000009001.md), [PiloteErreur](../WDLang1/1000009005.md), [PiloteInfo](../WDLang1/1000009006.md), ...) est vérifiée lors de l'appel de la fonction de dialogue associée ([Avertissement](../WDLang1/3021009.md), [Erreur](../WDLang1/3021013.md), [Info](../WDLang1/3021011.md), ...).


<a name="NOTE0_2"></a>




### Différence de type de message
<a name="difference_type_message_ELTPARAGRAPHE000091"></a>L'ordre des fonctions [PiloteXXX](../WDLang1/1000009002.md) doit respecter l'ordre d'affichage des fonctions [Erreur](../WDLang1/3021013.md), [Info](../WDLang1/3021011.md), ... Dans le cas contraire, une erreur fatale sera déclenchée.

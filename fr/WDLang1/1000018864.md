
## UUDécode (Fonction)

***En anglais : UUDecode***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Décode une chaîne de caractères au format UUEncode.

**Remarque** : Cette chaîne a été encodée auparavant avec la fonction [UUEncode](../WDLang1/1000018862.md). 




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Encodage d'une chaîne
Res = UUEncode("Le numéro de mon compte est 0123456789")

// ...

// Décodage de la chaîne
ChaîneDécodée est une chaîne
ChaîneDécodée = UUDécode(Res)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = UUDécode(<Chaîne à décoder>)`
---

**`<Résultat> : (Chaîne de caractères)`**

Chaîne décodée selon l'algorithme UUEncode. 

**`<Chaîne à décoder> : (Chaîne de caractères)`**

Texte à décoder. Ce texte a été encodé avec la fonction [UUEncode](../WDLang1/1000018862.md).

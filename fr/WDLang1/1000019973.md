
## MultimédiaContrôleVisible (Fonction)

***En anglais : MultimediaControlVisible***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie ou modifie la visibilité des boutons de contrôle (Lecture, Pause, Avance rapide, ...) du champ Multimédia.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI MultimédiaContrôleVisible(MM_Vidéo) = Faux ALORS
	MultimédiaContrôleVisible(MM_Vidéo) = Vrai
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître l'état de visibilité des boutons de contrôle

`<Résultat> = MultimédiaContrôleVisible(<Champ Multimédia>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si les boutons de contrôle sont actuellement affichés, 

- <u><u><u><u>Faux</u></u></u></u> si les boutons de contrôle sont actuellement masqués.




**`<Champ Multimédia> : (Nom de champ)`**

Nom du champ Multimédia à manipuler.


<a name="SYNTAXE2"></a>

### Modifier l'état de visibilité des boutons de contrôle

`<Résultat> = MultimédiaContrôleVisible(<Champ Multimédia> , <Visible> [, <Délai>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si les boutons étaient visibles avant modification.

- <u><u><u><u>Faux</u></u></u></u> si les boutons étaient invisibles avant modification.




**`<Champ Multimédia> : (Nom de champ)`**

Nom du champ Multimédia à manipuler.

**`<Visible> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> pour afficher les boutons de contrôle, 

- <u><u><u><u>Faux</u></u></u></u> pour masquer les boutons de contrôle. 




**`<Délai> : (Entier optionnel ou Durée optionnelle)`**

Délai d'inactivité (en secondes) à partir duquel les boutons de contrôle doivent être automatiquement masqués. 

- Si ce délai vaut 0, les boutons de contrôle resteront visibles en permanence (ou jusqu'à ce que l'utilisateur les masque manuellement). 

- Si ce délai n'est pas spécifié, le délai d'inactivité avant masquage sera de 3 secondes.


Ce paramètre durée peut correspondre à : 

- un entier correspondant au nombre de secondes, 

- une variable de type Durée, 

- l'indication directe de la durée (par exemple : '1 s'). 


**Attention** : Ce paramètre est pris en compte uniquement si &lt;Visible&gt; vaut <u><u><u><u>Vrai</u></u></u></u>.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Si le champ Multimédia n'a pas de boutons de contrôle (option paramétrable dans la description du champ), la fonction est sans effet.

- Il est possible de définir les boutons de contrôle affichés dans le champ dans la description du champ Multimédia (onglet "Général").




## &lt;Variable Document&gt;.Redo (Fonction)

***En anglais : &lt;Document variable&gt;.Redo***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Rétablit la dernière action annulée sur un document de type [Document](../WDLang1/1000022461.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonDocument est un Document

// Refait la dernière action annulée sur le document
MonDocument.Redo()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Document>.Redo()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la dernière action annulée a été rétablie, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Document> : (Variable de type Document)`**

Nom de la variable de type [Document](../WDLang1/1000022461.md) à manipuler.


## FIListeAffiche (Fonction)

***En anglais : IWListDisplay***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Rafraîchit l'affichage d'un champ "Fenêtre interne" avec parcours automatique sur un fichier de données ou une requête.

Remarque : Le champ "Fenêtre interne" doit autoriser le changement de contenu par balayage. 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
FIListeAffiche(FI_ChampFenêtreInterne, fiDébut)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`FIListeAffiche(<Champ Fenêtre Interne> [, <Position>])`
---

**`<Champ Fenêtre Interne> : (Nom de champ)`**

Nom du champ Fenêtre Interne à manipuler.

**`<Position> : (Constante optionnelle)`**

Position de départ du rafraîchissement. Ce paramètre peut correspondre à : 


|   |   |
| --- | --- |
| *fiCourant*<br>(Valeur par défaut) | Rafraîchit le contenu du champ à partir de l'enregistrement actuellement affiché dans le champ. |
| *fiDébut* | Rafraîchit le contenu du champ en se positionnant sur le premier enregistrement. |

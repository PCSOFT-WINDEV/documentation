


## EnModeVGA (Fonction)

***En anglais : InVGAMode***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Indique si l'application s'exécute sur un mobile en mode VGA. 

**Remarque** : Cette fonction fonctionne uniquement en WINDEV Mobile (exécutable ou go en mode simulateur). 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Si on est en mode VGA
SI EnModeVGA() = Vrai ALORS
	// Ouvrir la fenêtre adaptée pour le mode VGA
	Ouvre(FEN_ModeVGA)
SINON
	// Ouvrir la fenêtre adaptée pour le mode NON VGA
	Ouvre(FEN_ModeNonVGA)
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = EnModeVGA()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si les conditions suivantes sont réunies : 

	- l'application est exécutée sur un mobile VGA.

	- l'application tourne en mode VGA (et non pas en émulation non VGA).




- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.

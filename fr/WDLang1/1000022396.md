
## NoteSauveTout (Fonction)

***En anglais : NoteSave***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Sauvegarde les notes redimensionnables de la fenêtre en cours dans un fichier. Toutes les notes repositionnables sont sauvegardées : 

- Champ Note repositionnable. 

- Note repositionnable créée via la FAA (Fonctionnalité Automatique de vos Applications). 

- Note repositionnable créée par programmation (fonctions [NoteCrée](../WDLang1/1000022395.md), [FenEdCréeNote](../WDLang1/1000022270.md), [ChampCrée](../WDLang1/1000020876.md)). 



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Clic sur [Exporter mes notes]
NoteSauveTout()
SI ErreurDétectée = Vrai ALORS
	Erreur(ErreurInfo())
FIN

// Clic sur [Importer mes notes]
NoteChargeTout()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`NoteSauveTout([<Fichier>])`
---

**`<Fichier> : (Chaîne de caractères optionnelle)`**

Nom du fichier contenant les notes repositionnables à sauvegarder. 

Si ce paramètre n'est pas précisé, les notes repositionnables sont sauvegardées dans le fichier : &lt;Nom fenêtre&gt;.wdnote présent dans le répertoire des données. Le fichier de sauvegarde correspond à : 
```txt
fRepDonnées()\<MaFenetre..Nom>.wdnote
```




<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Cette fonction est équivalente à la FAA (Fonctionnalités Automatiques de vos Applications) d'exportation des notes sans l'affichage du sélecteur de fichiers. Pour plus de détails, consultez [Note repositionnable (FAA)](../Editeurs/1000021820.md). 

- Les éléments sauvegardés sont : la position de la note, son contenu, sa couleur de fond et les caractéristiques de la police utilisée.

- Les notes repositionnables sauvegardées pourront ensuite être importées : 

	- Dans l'éditeur (par simple Drag and Drop, ou ouverture du fichier des notes). 

	- Par programmation via la fonction [NoteChargeTout](../WDLang1/1000022397.md). 




- Si aucune note n'a été créée, la fonction ne fait rien.

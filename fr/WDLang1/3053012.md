
## MemRécupère (Fonction)

***En anglais : MemRetrieve***

<a name="Utilisation"></a>
<a name="description"></a>
Récupère la valeur de retour d'un élément présent dans une zone mémoire.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Récupération de la valeur de retour de l'élément 7
// présent dans la zone mémoire "MaZoneMémoire"
ResRécupère = MemRécupère("MaZoneMémoire", 7)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = MemRécupère(<Zone mémoire> , <Indice>)`
---

**`<Résultat> : (Chaîne de caractères)`**



- Valeur de retour de l'élément spécifié (définie par les fonctions [MemAjoute](../WDLang1/3053002.md) ou [MemModifie](../WDLang1/3053007.md)), 

- Chaîne vide ("") si l'élément spécifié n'a pas été trouvé.




**`<Zone mémoire> : (Chaîne de caractères)`**

Nom de la zone mémoire, défini par la fonction [MemCrée](../WDLang1/3053005.md).

**`<Indice> : (Entier)`**

Indice de l'élément à manipuler. L'indice d'un élément peut être obtenu par les fonctions [MemEnCours](../WDLang1/3053008.md) ou [MemRecherche](../WDLang1/3053015.md).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Existence de la zone mémoire
<a name="existence_zone_memoire_ELTPARAGRAPHE000234"></a>L'existence de la zone mémoire n'est pas vérifiée.

Si la zone mémoire spécifiée n'existe pas, une erreur du WLangage est générée lors de l'exécution de la fonction **MemRécupère**. 

Pour vérifier l'existence d'une zone mémoire, utilisez la fonction [MemExiste](../WDLang1/3053006.md).
<a name="NOTE0_2"></a>




### Positionnement dans la zone mémoire
<a name="positionnement_dans_zone_memoire_ELTPARAGRAPHE000104"></a>La fonction **MemRécupère** ne se positionne pas sur l'élément manipulé. Pour se positionner sur l'élément &lt;Indice&gt;, utilisez la fonction [MemPositionne](../WDLang1/3053018.md).

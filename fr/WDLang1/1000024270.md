


## &lt;Traitement de texte&gt;.Positionne (Fonction)

***En anglais : &lt;Word Processing&gt;.Position***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Provoque le défilement du champ Traitement de texte afin que le texte voulu apparaisse entièrement. Il est possible de définir la position de ce texte par rapport aux bords du champ.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affiche la sélection si elle n'était pas visible
TT_MonDoc.Positionne()

// Positionne la sélection en haut du champ
TT_MonDoc.Positionne(0)

// Positionne la sélection en bas du champ centrée horizontalement
TT_MonDoc.Positionne(100, 50)

// Positionne la partie du document située entre les caractères 123 et 158 
// à 30% de l'espace restant en hauteur et 40% en largeur
TT_MonDoc.Positionne(123, 158, 30, 40)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Afficher le texte sélectionné à la position voulue

`<Champ Traitement de texte>.Positionne([<Positionnement vertical> [, <Positionnement horizontal>]])`
---

**`<Champ Traitement de texte> : (Nom de champ)`**

Nom du champ Traitement de texte à manipuler.

**`<Positionnement vertical> : (Entier optionnel)`**

Position verticale du texte sélectionné. Cette position correspond au pourcentage en hauteur de la place restante (une fois décomptée la hauteur du texte sélectionné) dans le champ Traitement de texte. ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=DocPositionne_Hauteur1.gif)


Si ce paramètre n'est pas précisé (ou correspond à la constante *dpDéfaut*), la fonction provoquera le défilement minimum pour afficher le texte sélectionné. 

Correspond à : 

- une valeur comprise entre 0 et 100. Quelques exemples : ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=DocPositionne_Hauteur2.gif)


- une des constantes suivantes :
	


|   |   |
| --- | --- |
| *dpBas* | Le texte sélectionné est affiché en bas du champ Traitement de texte. |
| *dpCentre* | Le texte sélectionné est affiché au centre vertical du champ Traitement de Texte. |
| *dpDéfaut* <br>(Valeur par défaut) | Défilement vertical minimum pour faire apparaître le texte sélectionné dans le champ Traitement de texte. |
| *dpHaut* | Le texte sélectionné est affiché en haut du champ Traitement de texte. |






**`<Positionnement horizontal> : (Entier optionnel)`**

Position horizontale du texte sélectionné. Cette position correspond au pourcentage en largeur de la place restante (une fois décomptée la largeur du texte sélectionné) dans le champ Traitement de texte. ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=DocPositionne_Largeur1.gif)


Si ce paramètre n'est pas précisé (ou correspond à la constante *dpDéfaut*), la fonction provoquera le défilement minimum pour afficher le texte sélectionné. 

Correspond à : 

- une valeur comprise entre 0 et 100. Quelques exemples :![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=DocPositionne_Largeur2.gif)


- une des constantes suivantes :
	


|   |   |
| --- | --- |
| *dpCentre* | Le texte sélectionné est affiché au centre horizontal du champ Traitement de texte. |
| *dpDéfaut*<br>(Valeur par défaut) | Défilement horizontal minimum pour faire apparaître le texte sélectionné. |
| *dpDroite* | Le texte sélectionné est affiché à droite du champ Traitement de texte. |
| *dpGauche* | Le texte sélectionné est affiché à gauche du champ Traitement de texte. |







<a name="SYNTAXE2"></a>

### Afficher une partie de texte à la position voulue

`<Champ Traitement de texte>.Positionne(<Début du texte à afficher> , <Fin du texte à afficher> , <Positionnement vertical> [, <Positionnement horizontal>])`
---

**`<Champ Traitement de texte> : (Nom de champ)`**

Nom du champ Traitement de texte à manipuler.

**`<Début du texte à afficher> : (Entier)`**

Position (en nombre de caractères) du début de la partie de texte à afficher.

**`<Fin du texte à afficher> : (Entier)`**

Position (en nombre de caractères) de la fin de la partie de texte à afficher.

**`<Positionnement vertical> : (Entier optionnel)`**

Si ce paramètre n'est pas précisé (ou correspond à la constante *dpDéfaut*), la fonction provoquera le défilement minimum pour afficher le texte sélectionné. 

Correspond à : 

- une valeur comprise entre 0 et 100. Quelques exemples : ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=DocPositionne_Hauteur2.gif)


- une des constantes suivantes :
	


|   |   |
| --- | --- |




## Fonctions de gestion des services
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions de gestion des services sont les suivantes :



|   |   |
| --- | --- |
| [FinService](../WDLang1/1000018711.md) | Termine l'exécution du service en cours. |
| [ServiceActualise](../WDLang1/1000017107.md) | Demande à un service de relire ses informations de configuration. |
| [ServiceArrête](../WDLang1/1000017111.md) | Arrête un service. |
| [ServiceAttendEtat](../WDLang1/1000017101.md) | Attend qu'un service soit dans un état spécifique. |
| [ServiceContrôle](../WDLang1/1000017108.md) | Envoie un message de contrôle à un service. |
| [ServiceDémarre](../WDLang1/1000017110.md) | Démarre un service. |
| [ServiceDésinstalle](../WDLang1/1000017098.md) | Désinstalle un service. |
| [ServiceEcritEvénementJournal](../WDLang1/1000018807.md) | Ecrit un événement dans le journal des événements Windows. |
| [ServiceEtat](../WDLang1/1000017102.md) | Renvoie l'état actuel d'un service. |
| [ServiceExiste](../WDLang1/1000018936.md) | Teste si un service est installé. |
| [ServiceInfo](../WDLang1/1000017103.md) | Renvoie des informations concernant la configuration d'un service. |
| [ServiceInstalle](../WDLang1/1000017099.md) | Installe un service Windows ou un daemon Linux grâce aux informations données dans les membres de la [variable Service](../WDLang1/1000017118.md). |
| [ServiceListe](../WDLang1/1000017104.md) | Liste les services présents sur le poste en cours ou sur le poste distant indiqué. |
| [ServiceModifie](../WDLang1/1000017100.md) | Modifie la configuration d'un service grâce aux informations données dans les membres de la [variable Service](../WDLang1/1000017118.md). |
| [ServicePause](../WDLang1/1000017109.md) | Met un service en pause. |
| [ServiceRedémarre](../WDLang1/1000017112.md) | Redémarre un service (déjà démarré). |
| [ServiceRelance](../WDLang1/1000017105.md) | Relance un service en pause. |
| [ServiceTemporise](../WDLang1/1000018712.md) | Temporise le service en cours pour la durée indiquée. |

- ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=LesfonctionsService.gif) ***Exemples unitaires (WINDEV)*** : **Les fonctions Service** <br>Utilisation des fonctions ServiceXXX pour : <br>- lister les services installés<br>- démarrer un service<br>- arrêter un service<br>- mettre en pause et relancer un service.<br><br>Les fonctions services sont disponibles pour le poste local et également sur un poste distant.<br>Note : Pour utiliser les services d'un poste distant, il est nécessaire de posséder les droits d'administrateur sur ce poste.

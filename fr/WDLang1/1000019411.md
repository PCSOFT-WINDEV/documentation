
## fEstUneImage (Fonction)

***En anglais : fIsImage***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de savoir si un fichier présent sur disque ou un fichier contenu dans un buffer correspond à un format d'image reconnu par WINDEV, WEBDEV ou WINDEV Mobile.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI fEstUneImage(MonBuffer) = Vrai ALORS
	Info("La variable correspond à une image.")
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Analyser un fichier externe

`<Résultat> = fEstUneImage(<Nom et chemin du fichier>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le fichier est une image pouvant être chargée par WINDEV, WEBDEV ou WINDEV Mobile,

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 




**`<Nom et chemin du fichier> : (Chaîne de caractères Ansi ou Unicode)`**

Nom et chemin complet (ou relatif) du fichier à identifier. Il est possible d'utiliser un chemin UNC. 


<a name="SYNTAXE2"></a>

### Analyser une variable de type Buffer

`<Résultat> = fEstUneImage(<Nom de la variable>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le buffer correspond à une image, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Nom de la variable> : (Variable de type Buffer ou mémo binaire)`**

Nom de la variable (de type buffer ou mémo binaire) contenant l'image à analyser. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Astuce
<a name="astuce_ELTPARAGRAPHE000071"></a>La fonction **fEstUneImage** peut être relativement lente car elle tente de charger l'image. Pour vérifier rapidement si un fichier est une image, il est également possible de vérifier directement l'extension du fichier.


## sCalculeCrc16 (Fonction)

***En anglais : sComputeCrc16***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Contrôle une chaîne de caractères avant et après transmission entre applications WINDEV / entre sites WEBDEV.

Cette fonction permet de vérifier si une chaîne a été correctement transmise lors de transmissions comportant des risques de perte d'information (par modem par exemple) entre des applications WINDEV / entre des sites WEBDEV. Pour cela, il suffit de calculer le "Crc16" avant la transmission de la chaîne, puis de vérifier sa valeur à la réception de la chaîne : les deux valeurs doivent être identiques.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
nCrcLu est un entier	// Caractère de contrôle lu
LigneLue est une chaîne	// Chaîne lue
...
SI nCrcLu <> sCalculeCrc16(LigneLue) ALORS
	Info("Erreur lors de la récupération de la chaîne")
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = sCalculeCrc16(<Chaîne>)`
---

**`<Résultat> : (Entier)`**

Valeur de contrôle de la chaîne.

**`<Chaîne> : (Chaîne de caractères)`**

Chaîne de caractères quelconque.  






## Remarques
![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Cette fonction peut renvoyer un résultat différent dans la version WINDEV. Pour améliorer la portabilité du code, préférez la fonction [sCalculeCrc32](../WDLang1/3062015.md).

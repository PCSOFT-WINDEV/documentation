
## ConstruitEntier (Fonction)

***En anglais : MakeInteger***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Construit un entier de 4 octets à partir de deux entiers de 2 octets.






<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
Valeur est un entier = 5000
Faible est un entier sur 2 = PoidsFaible(Valeur)
Fort est un entier sur 2 = PoidsFort(Valeur)
// Reconstruit la valeur d'après ces octets de poids fort et de poids faible
ValeurConstruite est un entier = ConstruitEntier(Fort, Faible)
// ValeurConstruite est égal à Valeur
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = ConstruitEntier(<Poids Fort> , <Poids Faible>)`
---

**`<Résultat> : (Entier)`**

Valeur entière (sur 4 octets) construite par concaténation de &lt;Poids Fort&gt; et de &lt;Poids Faible&gt;.

**`<Poids Fort> : (Entier sur 2 octets)`**

Valeur qui déterminera les 2 octets de poids fort de l'entier &lt;Résultat&gt;.

**`<Poids Faible> : (Entier sur 2 octets)`**

Valeur qui déterminera les 2 octets de poids faible de l'entier &lt;Résultat&gt;.

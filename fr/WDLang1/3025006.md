
## ColonneEnCours (Fonction)

***En anglais : CurrentColumn***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie :

- soit l'indice de la colonne indicée en cours dans un champ Table (Table par programmation ou Table fichier).

- soit le numéro de la colonne en cours de saisie dans le champ Table indiqué (Table par programmation ou Table fichier).







<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Identification de la colonne en cours de saisie
ResChamp = ColonneEnCours(TABLE_Produit)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe
![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Syntaxe non disponible en Java et Android
### Indice de la colonne indicée en cours

`<Résultat> = ColonneEnCours()`
---

**`<Résultat> : (Entier)`**



- Indice de la colonne en cours si cette colonne est indicée, 

- 1 si la colonne en cours n'est pas indicée,

- -1 si le champ en cours n'est pas un champ Table.  





<a name="SYNTAXE2"></a>

### Numéro de la colonne en cours de saisie dans le champ Table

`<Résultat> = ColonneEnCours(<Champ Table>)`
---

**`<Résultat> : (Entier)`**



- Numéro de la colonne en cours de saisie (si le champ Table spécifié est en cours de saisie), 

- -1 dans le cas contraire.




**`<Champ Table> : (Nom de champ)`**

Nom du champ Table à manipuler. Ce champ Table doit être en saisie.

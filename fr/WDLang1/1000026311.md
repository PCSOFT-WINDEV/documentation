


## &lt;Champ d'affichage HTML&gt;.ValideFormulaire (Fonction)

***En anglais : &lt;HTML Display control&gt;.ValidateForm***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Valide le formulaire d'une page HTML affichée dans un champ d'affichage HTML.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
HTM_Affichage.ValeurChamp("LOGIN", sLogin)
HTM_Affichage.ValeurChamp("PASSWD", sMotDePasse)
HTM_Affichage.ValideFormulaire()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Champ d'affichage HTML>.ValideFormulaire([<Formulaire>])`
---

**`<Champ d'affichage HTML> : (Nom de champ)`**

Nom du champ d'affichage HTML manipulé.

**`<Formulaire> : (Chaîne de caractères optionnelle)`**

Nom du formulaire à valider. Si ce paramètre n'est pas précisé, le premier formulaire de la page affichée dans le champ est validé.

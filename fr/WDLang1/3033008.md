


## Propriétés liées à la gestion de l'Euro
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les propriétés suivantes permettent de gérer l'Euro dans :

- Les champs de saisie et les colonnes de tables de type "Monétaire + Euro" des fenêtres et des pages.

- Les champs rubrique et les champs calculés de type "Monétaire + Euro" des états.





|   |   |
| --- | --- |
| [DeviseAffichage](../Proprietes/2510030.md) | Permet de connaître et de modifier la devise utilisée lors de l'affichage et de la saisie d'une valeur dans un champ de saisie de type "Monétaire + Euro" |
| [DeviseMémoire](../Proprietes/2510013.md) | Permet de connaître et modifier la devise utilisée lors de la manipulation du champ de saisie de type "Monétaire + Euro" par programmation |

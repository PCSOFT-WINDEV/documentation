
## SysYRes (Fonction)

***En anglais : SysYRes***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la résolution verticale de l'écran.






<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Résolution verticale de l'écran
ResResVert = SysYRes()
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = SysYRes()`
---

**`<Résultat> : (Entier)`**

Résolution verticale de l'écran.

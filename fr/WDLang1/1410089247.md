
## Italie.VérifieIVA (Fonction)

***En anglais : Italy.CheckIVA***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Teste la validité d'un numéro IVA (numéro d'identification d'entreprise en Italie).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Test d'une variable
MonNuméroIVA est une chaîne = "IT07643520567"
Italie.VérifieIVA(MonNuméroIVA)	// renvoie Vrai

// Test d'un champ
SAI_NuméroIVA = "ABC"
Italie.VérifieIVA(SAI_NuméroIVA)	// renvoie Faux
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = Italie.VérifieIVA(<Numéro à vérifier>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'élément spécifié contient un numéro IVA valide.

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Numéro à vérifier> : (Nom de champ ou variable de type chaîne)`**

Numéro IVA à vérifier. Ce paramètre peut correspondre : 

- au nom d'une variable de type chaîne.

- au nom d'un champ.

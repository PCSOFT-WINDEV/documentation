
## IndiceSuivant (Fonction)

***En anglais : NextIndex***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le numéro de la ligne suivante affichée dans le champ Table en cours.

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Cette fonction permet également de renvoyer l'indice du champ indicé suivant.






<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// -- Code de sortie de ligne
NumLigne est un entier = IndiceSuivant()
Trace("Numéro de la ligne suivante : " + NumLigne)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = IndiceSuivant()`
---

**`<Résultat> : (Entier)`**

Numéro de la ligne suivante affichée dans le champ Table en cours. La fonction **IndiceSuivant** doit être utilisée dans le code de sortie de ligne. Si cette fonction est utilisée dans un autre traitement, le résultat n'a aucun sens.

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Indice du champ indicé suivant. La fonction **IndiceSuivant** doit être utilisée dans le code de sortie du champ. Si cette fonction est utilisée dans un autre traitement, le résultat n'a aucun sens.

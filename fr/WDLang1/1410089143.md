


## KanbanSupprimeToutCarte (Fonction)

***En anglais : KanbanDeleteAllCard***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime toutes les cartes d'un champ Kanban ou d'un champ Liste de Kanban.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Supprime toutes les cartes du champ Kanban
KanbanSupprimeToutCarte(KANBAN_MonKanban)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`KanbanSupprimeToutCarte(<Champ manipulé>)`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ manipulé : 

- Nom d'un champ Kanban.

- Nom d'un champ Liste de Kanban.




## &lt;Type Chaîne&gt;.Insère (Fonction)

***En anglais : &lt;String type&gt;.Insert***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Insère une chaîne dans une autre chaîne de caractères à une position donnée. 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// On ajoute Sol à SiLaFaMiReDo
sNotes est une chaîne = "SiLaFaMiReDo"
sNotesComplet est une chaîne = sNotes.Insère("Sol", 5)
// sNotesComplet contient donc "SiLaSolFaMiReDo"
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Chaîne initiale>.Insère(<Chaîne à insérer> , <Position d'insertion>)`
---

**`<Résultat> : (Chaîne de caractères)`**

Chaîne de caractères dans laquelle la chaîne a été insérée à la position spécifiée.

**`<Chaîne initiale> : (Chaîne de caractères)`**

Chaîne de caractère dans laquelle l'insertion va être réalisée. 
ATTENTION : Cette chaîne ne sera pas modifiée par l'insertion.

**`<Chaîne à insérer> : (Chaîne de caractères ou Buffer)`**

Chaîne à insérer dans la &lt;Chaîne initiale&gt;.

**`<Position d'insertion> : (Entier)`**

Position à laquelle &lt;Chaîne à insérer&gt; va être ajoutée.

- Si la position de l'ajout est négative, la chaîne sera ajoutée au début de la chaîne initiale.

- Si la position de l'ajout est supérieure à la taille de la chaîne initiale, la chaîne sera ajoutée à la fin de la chaîne initiale.

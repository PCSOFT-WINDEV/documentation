


## &lt;Traitement de texte&gt;.VersTexte (Fonction)

***En anglais : &lt;Word Processing&gt;.ToText***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Crée une chaîne de caractères avec les données d'un champ Traitement de texte.

Cette chaîne pourra ensuite par exemple être sauvegardée dans un format spécifique.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MaChaîne est une chaîne
MaChaîne = TT_MonDocument.VersTexte()
// Sauvegarde dans un fichier texte
fSauveTexte("C:\temp\Contenu.txt", MaChaîne)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Champ Traitement de texte>.VersTexte()`
---

**`<Résultat> : (Chaîne de caractères)`**



- Chaîne de caractères contenant les données. 

- Chaîne vide ("") en cas d'erreur ou si il n'y a pas de données à extraire.




**`<Champ Traitement de texte> : (Nom de champ)`**

Nom du champ Traitement de texte à manipuler.


## DDEDéconnecte (Fonction)

***En anglais : DDEDisconnect***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime une connexion entre le programme en cours et un destinataire.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MaConnexion = DDEConnecte("Excel", Sujet)		// Connexion
Compteur = 3			// Répète 3 fois si nécessaire
BOUCLE				// Envoie les données
	SI DDEEnvoie(MaConnexion, Item, Data, 10) ALORS SORTIR
	Compteur --
	SI Compteur = 0 ALORS
		Erreur("L'information n'a pas pu être envoyée à Excel")
		SORTIR
	FIN
FIN
DDEDéconnecte(MaConnexion)		// Déconnexion
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = DDEDéconnecte([<Identifiant connexion>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la déconnexion a été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> en cas d'erreur (la fonction [DDEErreur](../WDLang1/3028006.md) permet d'obtenir le compte-rendu de l'erreur).




**`<Identifiant connexion> : (Entier optionnel)`**

Identifiant de la connexion renvoyé par la fonction [DDEConnecte](../WDLang1/3028015.md). Si ce paramètre n'est pas précisé, la dernière connexion manipulée est déconnectée.

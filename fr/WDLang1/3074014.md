


## TableFixeColonne (Fonction)

***En anglais : TableFixColumn***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Fixe une colonne à gauche ou libère une colonne fixée à gauche. Cette fonction permet également de savoir si la colonne est fixée à gauche ou non.

**Remarques** : 

- Les colonnes fixées à gauche ne scrollent pas avec l'ascenseur horizontal. 

- L'appel de cette fonction est équivalente à : 

	- l'usage du cadenas par l'utilisateur.

	- l'usage de l'option "Fixée à gauche" du menu contextuel du titre de la colonne.


 Pour plus de détails, consultez [FAA : Fixer les colonnes d'un champ Table](../WDChamp/9500109.md)



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Fixe/Libère une colonne avec un bouton
// Clic sur le bouton
SI TableFixeColonne(COL_COLONNE) ALORS
	// La colonne était fixée
	TableFixeColonne(COL_COLONNE, Faux) // On la libère
SINON
	// La colonne était libre
	TableFixeColonne(COL_COLONNE, Vrai) // On la fixe
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = TableFixeColonne(<Colonne> [, <Fixer / Libérer>])`
---

**`<Résultat> : (Booléen)`**

Etat précédent de la colonne : 

- <u><u><u><u>Vrai</u></u></u></u>, colonne fixée, 

- <u><u><u><u>Faux</u></u></u></u>, colonne libre.




**`<Colonne> : (Nom de champ)`**

Nom de la colonne à manipuler. 
Si plusieurs champs Table possèdent des colonnes de même nom, le nom de la colonne doit être préfixé du nom du champ Table : &lt;Champ Table&gt;.&lt;Colonne&gt;.

**`<Fixer / Libérer> : (Booléen optionnel)`**



- <u><u><u><u>Vrai</u></u></u></u> : la colonne doit être fixée

- <u><u><u><u>Faux</u></u></u></u> : la colonne doit être libérée. 


Si ce paramètre n'est pas précisé, la fonction **TableFixeColonne** renvoie l'état actuel de la colonne.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Si la fonction **TableFixeColonne** est utilisée pour fixer une colonne déjà fixée, cette fonction n'a aucun effet.

- Cette fonction ne peut pas être utilisée avec des colonnes indicées.


## SysEspace (Fonction)

***En anglais : SysSpace***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie l'espace mémoire physique du poste en cours.






<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connaître l'espace mémoire physique
ResEspMémoire = SysEspace()
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = SysEspace([<Type de mémoire>])`
---

**`<Résultat> : (Entier)`**

Espace demandé (en octets) du poste en cours (ou du poste serveur).



**Remarque** : La fonction **SysEspace** retourne un entier sur 8 octets si la taille demandée est supérieure à 2 Go.

**`<Type de mémoire> : (Constante optionnelle de type entier)`**

Type de mémoire à renvoyer :


|   |   |
| --- | --- |
| *sysEspaceProgramme*<br>(Valeur par défaut) | Espace mémoire physique.<br><br> |
| *sysEspaceProgrammeLibre* | Espace mémoire physique inutilisé.<br><br> |
| *sysEspaceStockage* | Espace utilisé pour le stockage<br><br><br>![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Cette constante n'est pas disponible. |
| *sysEspaceStockageLibre* | Espace libre pour le stockage<br><br><br>![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Cette constante n'est pas disponible. |




## &lt;Editeur HTML&gt;.VersPDF (Fonction)

***En anglais : &lt;HTML editor&gt;.ToPDF***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Exporte le contenu d'un champ Editeur HTML vers un fichier PDF.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Exporte le contenu du champ Editeur HTML vers le fichier facture.pdf
EDHTML_MonEditeurHTML.VersPDF("c:\mes documents\facture.pdf")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Champ Editeur HTML>.VersPDF(<Fichier PDF>)`
---

**`<Champ Editeur HTML> : (Nom de champ)`**

Nom du champ Editeur HTML manipulé.

**`<Fichier PDF> : (Chaîne de caractères)`**

Nom et chemin complet du fichier PDF à créer. Si le fichier existe déjà, il est remplacé par le nouveau fichier.

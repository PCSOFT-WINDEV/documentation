


## &lt;Liste&gt;.SelectOccurrence (Fonction)

***En anglais : &lt;List Box&gt;.SelectCount***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le nombre d'éléments sélectionnés dans un champ Liste, Liste image ou Combo.

**Rappel** : L'élément sélectionné correspond à l'élément en cours sur lequel le bandeau de sélection est affiché.

**Remarque** : La fonction **&lt;Liste&gt;.SelectOccurrence** peut être utilisée indifféremment sur :

- les champs Liste fichier ou Liste par programmation,

- les champs Liste monosélection ou multisélection.











<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Nombre d'éléments sélectionnés dans le champ "LISTE_ListeClient"
ResElementSelect = LISTE_ListeClient.SelectOccurrence()
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Champ Liste>.SelectOccurrence()`
---

**`<Résultat> : (Entier)`**

Nombre d'éléments sélectionnés dans le champ Liste (ou Combo) spécifiée. Ce paramètre :

- peut être supérieur à 1 uniquement si le champ Liste est multisélection,

- est égal à 1 si le champ Combo est sans saisie ou si le champ Combo est en saisie et qu'un élément est sélectionné,

- est égal à 0 si le champ Combo est en saisie et qu'aucun élément n'est sélectionné.




**`<Champ Liste> : (Nom de champ)`**

Nom du champ Liste (ou Combo) à manipuler.




## TableauDistinct (Fonction)

***En anglais : ArrayDistinct***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie un tableau sans doublons.

Les tableaux suivants peuvent être manipulés :

- Tableau d'éléments simples (entier, réel, chaîne de caractères) à une dimension.

- Tableau de structures à une dimension. La suppression des doublons est réalisée en fonction de la valeur d'un ou de plusieurs membres.





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Premier tableau d'entiers
tabMonTableau1 est un tableau d'entiers
TableauAjoute(tabMonTableau1, 1)
TableauAjoute(tabMonTableau1, 2)
TableauAjoute(tabMonTableau1, 1)
TableauAjoute(tabMonTableau1, 3)

// Tableau "distinct"
tabMonDistinct est un tableau d'entiers
tabMonDistinct = TableauDistinct(tabMonTableau1)
// le résultat est 1; 2; 3
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = TableauDistinct(<Tableau WLangage>)`
---

**`<Résultat> : (Tableau)`**

Variable de type Tableau contenant le tableau sans doublons.

**`<Tableau WLangage> : (Tableau)`**

Nom de la variable de type Tableau à manipuler. Ce tableau doit être un tableau à une dimension. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Il est également possible de supprimer les doublons dans un tableau grâce à la fonction [TableauSupprimeDoublon](../WDLang1/1000019422.md). 

- La fonction [TableauSupprimeDoublon](../WDLang1/1000019422.md) manipule directement le tableau passé en paramètre. 

- La fonction **TableauDistinct** ne modifie pas le tableau passé en paramètre et renvoie un nouveau tableau sans les doublons.

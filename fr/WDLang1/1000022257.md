
## BulleLargeurMax (Fonction)

***En anglais : TooltipMaxWidth***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie la largeur maximale des bulles d'aide. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Largeur des bulles définie à 300 pixels
BulleLargeurMax(300)

// Affichage de bulles très larges
BulleLargeurMax(1280)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`BulleLargeurMax(<Largeur>)`
---

**`<Largeur> : (Entier)`**

Nouvelle largeur maximale de la bulle en pixels. 
Par défaut, cette largeur est de 640. Cette largeur ne peut pas être inférieure à 100.

**Remarque** : En version 21, la largeur maximale des bulles était de 300 pixels. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Cette fonction permet de paramétrer aussi bien les bulles d'aide standard que les bulles d'aide personnalisées.


## PoliceListe (Fonction)

***En anglais : FontList***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la liste des polices valides installées sur le poste en cours (TrueType, Postscript, ...).

![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Seules les polices supportées en Java sont listées.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Liste des polices dans une fenêtre d'information
Info(PoliceListe())
// Liste des polices dans un champ Liste
ListeAjoute(LISTE_ListePolice, PoliceListe())
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = PoliceListe([<Type de police>])`
---

**`<Résultat> : (Chaîne de caractères)`**

Liste des polices sous la forme : 

```txt
<Police 1> + RC + ... + <Police N>
```
Où &lt;Police 1&gt; .. &lt;Police N&gt; correspond au nom de la police.

**`<Type de police> : (Constante optionnelle de type Entier)`**

Type de police à lister : 



|   |   |
| --- | --- |
| *policeDéfaut*<br>(Valeur par défaut) | Liste les polices du driver d'affichage. |
| *policeImprimante* | Liste les polices du driver d'impression. |
| *policeParFamille* | Liste les familles de polices disponibles. Les familles de polices sont listées par ordre alphabétique. Les variantes de langues ne sont pas listées. <br><br>Remarque : Cette constante peut être combinée avec les constantes *policeDéfaut* et *policeImprimante*. |


![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Ce paramètre n'est pas disponible. Seules les polices supportées en Java sont listées.




## &lt;Document&gt;.RechercheElémentParNom (Fonction)

***En anglais : &lt;Document&gt;.FindElementByName***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Recherche un élément par son nom dans l'ensemble d'une variable de type [Document](../WDLang1/1000022461.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonDocument est un Document

// Supprime l'image appelée "Image logo" du document
ElémentTrouvé est un docElément <- MonDocument.RechercheElémentParNom("Image Logo")
SI ElémentTrouvé <> Null ALORS
	DocSupprime(ElémentTrouvé)
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Document>.DocRechercheElementParNom(<Nom de l'élément>)`
---

**`<Résultat> : (Variable de type docElément)`**



- Variable de type [docElément](../WDLang1/1000022484.md) correspondant à l'élément cherché, 

- Null si aucun élément n'a été trouvé.




**`<Document> : (Variable de type Document)`**

Nom de la variable de type [Document](../WDLang1/1000022461.md) à manipuler.

**`<Nom de l'élément> : (Chaîne de caractères)`**

Texte à rechercher dans le document.

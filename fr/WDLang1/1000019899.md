
## PositionVersLigne (Fonction)

***En anglais : PositionToLine***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le numéro de la ligne à laquelle appartient un caractère connu par sa position dans un bloc de texte.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Initialisation d'une chaîne multiligne
s est une chaîne = [
AA
BB
CC
]

PositionVersLigne(s, 1) // Renvoie 1
PositionVersLigne(s, 2) // Renvoie 1
PositionVersLigne(s, 3) // Renvoie 1 à cause du RC
PositionVersLigne(s, 5) // Renvoie 2
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = PositionVersLigne(<Chaîne> , <Position>)`
---

**`<Résultat> : (Entier)`**

Numéro de la ligne à laquelle appartient le caractère spécifié. 

**`<Chaîne> : (Chaîne de caractères)`**

Chaîne de caractères à manipuler. 

**`<Position> : (Entier)`**

Position du caractère. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Le séparateur de lignes est le retour-chariot (RC).

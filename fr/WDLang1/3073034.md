
## SysRepCarteStockage (Fonction)

***En anglais : SysDirStorageCard***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le chemin du répertoire racine de la zone de stockage externe primaire.

**A partir de Android 11**, de nouvelles restrictions apparaissent pour les emplacements accessibles aux applications. Pour plus de détails sur les emplacements autorisés, consulter [Android 11 : Modifications du comportement des applications](../Editeurs/9000201.md).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Récupération des différentes cartes de stockage du Terminal Mobile en cours
ListeAjoute(LISTE_CarteStock, SysRepCarteStockage())
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = SysRepCarteStockage()`
---

**`<Résultat> : (Chaîne de caractères)`**



- Chemin de la SDCard interne de l'appareil (attention, ce chemin ne correspond pas à celui d'une SDCard externe connectée à l'appareil). Il est possible de connaître l'état de la carte avec la fonction [SysEtatCarteStockage](../WDLang1/1000019554.md). 

- Chaîne vide ("") en cas d'erreur.

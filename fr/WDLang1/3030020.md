
## MCUActif (Fonction)

***En anglais : UMCActive***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de connaître et de modifier l'activation de la gestion du MCU (Macro-Code Utilisateur).
<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Savoir si la gestion du MCU est activée

`<Résultat> = MCUActif()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le MCU est activé, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.  





<a name="SYNTAXE2"></a>

### Modifier le mode de gestion du MCU

`<Résultat> = MCUActif(<Activer>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le MCU est activé, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Activer> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> pour activer le MCU, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Mode d'activation du MCU
<a name="mode_activation_mcu_ELTPARAGRAPHE000055"></a>Les effets de la fonction **MCUActif** sont différents en fonction de la configuration du MCU dans le projet et dans les fenêtres.


|   |   |   |   |   |   |   |
| --- | --- | --- | --- | --- | --- | --- |
|   | **MCU autorisé dans le projet** | **MCU interdit dans le projet** |
| Fonction MCUActif non utilisée | MCUActif(Vrai) | MCUActif(Faux) | Fonction MCUActif non utilisée | MCUActif(Vrai) | MCUActif(Faux) |
| MCU autorisé dans la fenêtre | MCU possible | MCU possible | MCU impossible | MCU impossible | MCU possible | MCU impossible |
| MCU interdit dans la fenêtre | MCU impossible | MCU impossible | MCU impossible | MCU impossible | MCU impossible | MCU impossible |




## &lt;Tableur&gt;.Cherche (Fonction)

***En anglais : &lt;Spreadsheet&gt;.Seek***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Recherche une valeur dans les cellules d'un champ Tableur présent dans une fenêtre.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
sCell est chaîne = TBLR_SansNom1.Cherche("%NOMCLIENT%")
// Traite le résultat de la recherche
SI sCell <> "" ALORS
	 // Remplace %NOMCLIENT% par Durant
	 TBLR_SansNom1[sCell] = "Durant"
FIN
```

```wl
// Parcours de toutes les cellules avec "TVA"
sCell est chaîne = TBLR_SansNom1.Cherche("TVA")
TANTQUE  sCell <> ""
	// Remplace TVA par TBLR_SansNom1[sCell] = 20.6
	Trace(sCell)
	// Cellule suivante
	sCell = TBLR_SansNom1.Cherche("TVA", sCell)
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Champ Tableur>.Cherche(<Valeur recherchée> [, <Début> [, <Options>]])`
---

**`<Résultat> : (Chaîne de caractères)`**



- Nom de la cellule trouvée (par exemple "A1"). Si plusieurs cellules sont trouvées, &lt;Résultat&gt; contient le nom de la première cellule trouvée. 

- "" (chaîne vide) si aucune cellule n'est trouvée.




**`<Champ Tableur> : (Nom de champ)`**

Nom du champ Tableur à manipuler.

**`<Valeur recherchée> : (Type de la valeur recherchée)`**

Valeur à rechercher dans les cellules du champ Tableur. Cette valeur peut correspondre à un nombre ou à une chaîne de caractères.

**`<Début> : (Chaîne de caractères optionnelle)`**

Nom de la cellule à partir de laquelle la recherche doit être effectuée. **Cette cellule est exclue de la recherche.** 
Remarque : ce paramètre permet de parcourir toutes les cellules contenant une valeur. 

Si ce paramètre n'est pas précisé ou correspond à chaîne vide (""), la recherche est effectuée à partir de la cellule "A1".

**`<Options> : (Constante de type Entier optionnelle)`**

Options de recherche : 


|   |   |
| --- | --- |
| *tblrRechAvecCasse* | Recherche avec prise en compte de la casse. |
| *tblrRechContient* | Recherche de type "Contient". |
| *tblrRechDansFormule* | Recherche uniquement dans les formules. |
| *tblrRechExact* | Recherche de type "Exact". |
| *tblrRechFeuilleEnCours* | Recherche dans la feuille en cours. |
| *tblrRechSansCasse* | Recherche sans prendre en compte la casse. |
| *tblrRechTouteFeuille* | Recherche dans toutes les feuilles du classeur. |


Par défaut, la recherche se fait selon la combinaison de constantes suivante : *tblrRechAvecCasse* + *tblrRechFeuilleEnCours* + *tblrRechExact*



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- La recherche est effectuée dans l'ordre suivant : A1, B1, C1, ..., A2, B2, C2, ...

- Cette fonction est utilisable uniquement sur un champ Tableur présent dans une fenêtre.

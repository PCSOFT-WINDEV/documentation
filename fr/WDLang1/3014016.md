
## InstalleIcone (Fonction)

***En anglais : IconInstall***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Crée une icône (associée à un programme) dans un groupe de programme de Windows, avec la ligne de commande associée.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Crée un raccourci pour un programme
InstalleIcone("Exemples", "Exemple WD", "D:\WinDev\Exemples\GESCOM\GESCOM.EXE")
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = InstalleIcone(<Nom du groupe> , <Libellé> , <Ligne de commande>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'icône a été installée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Nom du groupe> : (Chaîne de caractères)`**

Nom du groupe dans lequel l'icône doit être créée. Il est impossible de créer des sous-groupes. Si le groupe spécifié n'existe pas, il est créé.

**`<Libellé> : (Chaîne de caractères)`**

Libellé apparaissant sous l'icône ou dans le menu "Démarrer". Si le libellé contient des espaces, il doit être mis entre doubles quotes.

**`<Ligne de commande> : (Chaîne de caractères)`**

Ligne de commande associée à l'icône. Contient le chemin complet de l'application associée à l'icône et les paramètres de lancement (si nécessaire).



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Pour une gestion plus avancée des icônes de programmes, utilisez la fonction [CréeRaccourci](../WDLang1/3014008.md).

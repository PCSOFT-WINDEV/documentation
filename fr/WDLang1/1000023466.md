


## Fonctions de gestion du champ Lecteur PDF
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions de gestion des champs Lecteur PDF sont les suivantes : 



|   |   |
| --- | --- |
| [LecteurPDFAfficheSignet](../WDLang1/1410089202.md) | Positionne le document PDF affiché dans le champ Lecteur PDF sur le signet demandé. |
| [LecteurPDFFerme](../WDLang1/1000023453.md) | Ferme et libère le fichier PDF affiché dans un champ Lecteur PDF. |
| [LecteurPDFInfoXY](../WDLang1/1000025332.md) | Renvoie pour une position donnée dans un champ Lecteur PDF (coordonnées d'un point du champ) : <br><br>- soit le numéro de la page pour la position indiquée, <br><br>- soit la position X (en mm) pour la position indiquée, <br><br>- soit la position Y (en mm) pour la position indiquée.<br><br><br> |
| [LecteurPDFOuvre](../WDLang1/1000023452.md) | Ouvre un fichier PDF dans un champ Lecteur PDF. |
| [LecteurPDFRecherchePrécédent](../WDLang1/1000023411.md) | Positionne sur le résultat précédent d'une recherche initiée par la fonction [LecteurPDFRecherchePremier](../WDLang1/1000023409.md) dans le contenu d'un champ Lecteur PDF. |
| [LecteurPDFRecherchePremier](../WDLang1/1000023409.md) | Initialise la recherche d'un texte dans le fichier PDF affiché par le champ Lecteur PDF et positionne sur la première occurrence trouvée (à partir de la page en cours). |
| [LecteurPDFRechercheSuivant](../WDLang1/1000023410.md) | Positionne sur le résultat suivant d'une recherche dans le contenu d'un champ Lecteur PDF. |
| [LecteurPDFSauve](../WDLang1/1000024898.md) | Sauve le contenu d'un champ Lecteur PDF dans un fichier. |
| [LecteurPDFSurligne](../WDLang1/1000023544.md) | Cherche un texte dans le document affiché par un champ Lecteur PDF et surligne les occurrences de ce texte dans le champ. |
| [LecteurPDFTournePage](../WDLang1/1000023940.md) | Effectue une rotation d'une page affichée dans le champ Lecteur PDF. |



Remarque : La fonction [iImprimePDF](../WDLang5/1000020519.md) peut être utilisée pour imprimer le contenu d'un champ Lecteur PDF.


## DonneNomPays (Fonction)

***En anglais : GetCountryName***

<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le libellé d'un pays correspondant au code ISO du pays à la norme ISO 3166-1.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Libellé France
sLibelléFrance est une chaîne = DonneNomPays("FRA")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = DonneNomPays(<Code ISO>)`
---

**`<Résultat> : (Chaîne de caractères)`**

Nom du pays.

**`<Code ISO> : (Chaîne de caractères)`**

Code ISO du pays. Ce code ISO peut être le code ISO alphabétique (sur 2 ou 3 caractères) ou le code ISO numérique du pays.




## &lt;Tableau croisé dynamique&gt;.InfoXY (Fonction)

***En anglais : &lt;Pivot table&gt;.InfoXY***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie les valeurs d'entêtes correspondant à une position en pixels dans un champ Tableau croisé dynamique.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
pos est un tcdPosition de TCD_Statistiques
pos = TCD_Statistiques.InfoXY(SourisPosX(), SourisPosY())
SI pos.EnDehors = Vrai ALORS 
	LIB_Libellé  = ""
SINON
	LIB_Libellé = "Année =" + pos.Année + ...
		"Produit = " + pos.Produit + ...
		"Ventes  = " + TCD_Statistiques.VAL_Ventes[pos] 
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Champ Tableau croisé dynamique>.InfoXY(<X> , <Y>)`
---

**`<Résultat> : (Variable de type tcdPosition)`**

Variable de type [tcdPosition](../WDLang1/1000020726.md) contenant les caractéristiques de la cellule sélectionnée. 

Remarques : 

- Si la position ne correspond à aucune cellule du tableau croisé dynamique (-1,-1), la propriété **EnDehors** de la variable  [tcdPosition](../WDLang1/1000020726.md) vaut <u><u><u><u>Vrai</u></u></u></u>.

- La position (0,0) correspond à la 1ère cellule en haut à gauche.




**`<Champ Tableau croisé dynamique> : (Nom de champ)`**

Nom du champ Tableau croisé dynamique à manipuler.

**`<X> : (Entier)`**

Coordonnée X (en pixels) à analyser. Cette coordonnée est relative à la zone cliente du champ.

**`<Y> : (Entier)`**

Coordonnée Y (en pixels) à analyser. Cette coordonnée est relative à la zone cliente du champ.

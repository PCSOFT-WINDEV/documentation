


## &lt;Agenda&gt;.Mode (Fonction)

***En anglais : &lt;Organizer&gt;.Mode***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le mode d'affichage d'un champ Agenda : jour, semaine, mois, ...

Remarque : La fonction [&lt;Agenda&gt;.ChangeMode](../WDLang1/1000022879.md) permet de modifier le mode d'affichage du champ Agenda. 

![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) En mode Universal Windows 10 App, cette fonction permet uniquement de manipuler les champs Agenda présents dans un état. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Alterne entre l'affichage par jour et par mois
SI AGD_MonAgenda.Mode() = agzJour ALORS
	AGD_MonAgenda.ChangeMode(agzMois)
SINON
	AGD_MonAgenda.ChangeMode(agzJour)
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Champ Agenda>.Mode()`
---

**`<Résultat> : (Constante de type Entier)`**

Plage de temps affichée par le champ Agenda :


|   |   |
| --- | --- |
| *agzJour* | Le champ Agenda affiche une journée. |
| *agzMois* | Le champ Agenda affiche un mois complet. |
| *agzNbJourPersonnalisé* | Affiche le champ Agenda sur le nombre de jours précisé grâce à la fonction [&lt;Agenda&gt;.ChangeMode](../WDLang1/1000022879.md).<br><br> |
| *agzSemaine* | Le champ Agenda affiche une semaine complète (ou partielle si un nombre de jours a été précisé grâce à la fonction [&lt;Agenda&gt;.ChangeMode](../WDLang1/1000022879.md)). |
| *agzSemaineSur5* | Le champ Agenda affiche une semaine sur 5 jours (sans le week-end). |



**`<Champ Agenda> : (Nom de champ)`**

Nom du champ Agenda à manipuler. Ce champ peut correspondre : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) au champ Agenda d'une fenêtre.

- au champ Agenda d'un état.

## Remarques


### Equivalences
<a name="equivalences_ELTPARAGRAPHE000260"></a>Les lignes de code suivantes sont équivalentes : 

```wl
AGD_Agenda.ChangeMode(agzSemaine, 5)
```
et

```wl
AGD_Agenda.ChangeMode(agzSemaineSur5)
```
 

Dans les 2 cas, la fonction **&lt;Agenda&gt;.Mode** renvoie la constante *agzSemaineSur5*.

- ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=WDAgenda.gif) ***Exemples complets (WINDEV)*** : **WD Agenda** <br>Cet exemple permet de synchroniser des rendez-vous entre vos agendas Outlook, Lotus Notes et Google.<br><br>Basé sur le champ Agenda, vous pouvez créer, modifier, déplacer, supprimer des rendez-vous.<br>Il est également possible de classer les rendez-vous par catégorie et de les lier à plusieurs agendas externes.

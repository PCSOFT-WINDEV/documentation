
## HandleParentSuivant (Fonction)

***En anglais : HandleParentNext***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Force le parent de la prochaine fenêtre à ouvrir. Cette fonction doit être utilisée juste avant la fonction [Ouvre](../WDLang1/3038035.md).

Cette fonction permet à une fenêtre WINDEV :

- d'être fille d'une autre application.

- d'être fille d'une fenêtre non WINDEV.




**Attention : Cette fonction est d'un usage avancé.**
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Code d'ouverture de projet de l'application
HandleParent est un entier système = LigneCommande("HPARENT")
SI HandleParent <> 0 ALORS
	Trace(HandleParent)
	HandleParentSuivant(HandleParent)
FIN
Ouvre(FEN_Fenêtre1)
FinProgramme()

// Code de l'appelant
Parent est un entier système = Handle("")
LanceAppli("MonApplication.exe" + " " + "/HPARENT=" + Parent)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`HandleParentSuivant(<Handle de la fenêtre>)`
---

**`<Handle de la fenêtre> : (Entier)`**

Handle de la fenêtre parente à utiliser.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Rappel : Le handle d'une fenêtre peut être récupéré par les fonctions [Handle](../WDLang1/3015005.md) ou [SysFenHandle](../WDLang1/3073007.md).

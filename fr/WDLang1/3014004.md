
## Constantes utilisées par la fonction ExécuteActionPrédéfinie
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
La fonction [ExécuteActionPrédéfinie](../WDLang1/3014028.md) utilise les constantes suivantes :

| Constante | Détail |
| --- | --- |
| **actColler** | Coller le contenu du presse-papiers |
| **actCopier** | Copier le contenu sélectionné dans le presse-papiers |
| **actCouper** | Couper le contenu sélectionné dans le presse-papiers |
| **actEffacerUnDisqueRW** | Effacer le contenu d'un répertoire sur un CD-RW/DVD-RW |
| **actEnvoieMail** | Envoyer un email |
| **actExplorateurSurRépertoire** | Ouvrir un explorateur sur un répertoire |
| **actGraverUnRépertoire** | Graver le contenu d'un répertoire sur un CD-R/DVD-R |
| **actIconiser** | Iconiser la fenêtre en cours |
| **actImageFenVersPressePapier** | Copier l'image de la fenêtre en cours dans le presse-papiers |
| **actLanceAideContextuelle** | Passer en mode "Aide contextuelle" |
| **actLanceSiteInternet** | Ouvrir un navigateur Internet sur une URL |
| **actLitDernier** | Lire le dernier enregistrement (du fichier de données actuellement manipulé). |
| **actLitPrécédent** | Lire l'enregistrement précédent (du fichier de données actuellement manipulé). |
| **actLitPremier** | Lire le premier enregistrement (du fichier de données actuellement manipulé). |
| **actLitSuivant** | Lire l'enregistrement suivant (du fichier de données actuellement manipulé). |
| **actOuvreEtat** | Ouvrir l'aperçu avant impression d'un état |

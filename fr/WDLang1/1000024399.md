


## &lt;Traitement de texte&gt;.UndoRedoSupprimeTout (Fonction)

***En anglais : &lt;Word Processing&gt;.UndoRedoDeleteAll***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime tout l'historique du mécanisme d'Undo/Redo d'un champ Traitement de texte.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Supprime l'historique des actions effectuées sur le champ
TT_MonDocument.UndoRedoSupprimeTout()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Champ Traitement de texte>.UndoRedoSupprimeTout()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la suppression a bien été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire




**`<Champ Traitement de texte> : (Nom de champ)`**

Nom du champ Traitement de texte à manipuler.

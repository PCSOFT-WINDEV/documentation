
## Procédure WLangage appelée par la fonction OuvreDocumentAsynchrone



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Procédure WLangage (appelée "callback") appelée par la fonction [OuvreDocumentAsynchrone](../WDLang1/1410088182.md) après la fermeture de la fenêtre d'édition. Cette procédure peut être une procédure locale, globale ou interne.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
PROCEDURE INTERNE OuvreDocumentAsynchrone_Callback(sDoc, nCompteRendu)
	SI nCompteRendu = odOuvertModifié ALORS
		Info(sDoc + " a été modifié.")
	FIN
FIN

SI PAS OuvreDocumentAsynchrone(sDocument, "Mon document", ... 
		odOptionAucune, "", OuvreDocumentAsynchrone_Callback) ALORS
	Erreur("Impossible d'ouvrir le document.")
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`OuvreDocumentAsynchrone_Callback(<Document> , <Compte-rendu>)`
---

**`<Document> : (Chaîne de caractères)`**

Document ouvert par la fonction [OuvreDocumentAsynchrone](../WDLang1/1410088182.md). 

**`<Compte-rendu> : (Entier)`**

Compte-rendu de l'ouverture du document : 



|   |   |
| --- | --- |
| *odEchec* | Le document n'a pas pu être ouvert. |
| *odOuvertModifié* | Le document a été ouvert et a été modifié. |
| *odOuvertNonModifié* | Le document a été ouvert mais il n'a pas été modifié. |


## Description de procédure (Type de variable)

***En anglais : Procedure description***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **Description de procédure** permet d'obtenir des informations sur une méthode de classe. 

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Récupère la définition de la classe type de la variable x
MaDéfinition est une Définition
MaDéfinition = RécupèreDéfinition(x)

// Récupère la description de la méthode Calcul
MaProcédure est une description de procédure 
MaProcédure = MaDéfinition.Procédure["Calcul"]
```

<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Propriétés spécifiques à la description des variables de type Description de procédure
<a name="proprietes_specifiques_description_des_variables_type_description_procedure_ELTPARAGRAPHE000046"></a>

Les propriétés suivantes peuvent être utilisées pour manipuler une variable de type Description de procédure :


|   |   |   |
| --- | --- | --- |
| **Nom de la propriété** | **Type manipulé** | **Effet** |
| Associé | Booléen | <br><br>- <u><u><u><u>Vrai</u></u></u></u> si la méthode provient d'une classe associée, <br><br>- <u><u><u><u>Faux</u></u></u></u> si la méthode provient d'une classe de base. <br><br><br> |
| Déclaration | Définition | Définition de la classe qui contient la méthode. <br>**Cette propriété est disponible en lecture seulement**. |
| Global | Booléen | <br><br>- <u><u><u><u>Vrai</u></u></u></u> si la méthode est globale, <br><br>- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. <br><br><br>**Cette propriété est disponible en lecture seulement**. |
| Hérité | Booléen | <br><br>- <u><u><u><u>Vrai</u></u></u></u> si la méthode est héritée d'une classe de base, <br><br>- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. <br><br><br>**Cette propriété est disponible en lecture seulement**. |
| Nom | Chaîne de caractères | Nom de la méthode. <br>**Cette propriété est disponible en lecture seulement**. |
| Virtuel | Booléen | <br><br>- <u><u><u><u>Vrai</u></u></u></u> si la méthode est virtuelle, <br><br>- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. <br><br><br>**Cette propriété est disponible en lecture seulement**. |


<a name="NOTE0_2"></a>


### Appel de méthode
<a name="appel_methode_ELTPARAGRAPHE000136"></a>La méthode dont on a récupéré la description peut être appelée grâce à la fonction [ExécuteTraitement](../WDLang1/3013031.md).

- Appel d'une méthode statique
	
	```txt
	p est une Description de procédure
	p = ...
	ExécuteTraitement(Null, p, <param1>, <param2>, ...)
	```


- Appel d'une méthode d'instance
	
	```txt
	p est une Description de procédure
	p = ...
	ExécuteTraitement(<variable objet>, p, <param1>, <param2>, ...)
	```

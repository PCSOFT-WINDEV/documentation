
## gARécupérer (Fonction)

***En anglais : gToRetrieve***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Décrit la valeur texte à afficher et à retourner lors de la récupération de la valeur d'un élément d'un champ Liste ou d'un champ Combo.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
ListeSupprimeTout(LISTE_Annonce)
ListeAjoute(LISTE_Annonce, gStylo(Noir) + "Voiture")
// LISTE_Annonce[1] retourne la chaîne de caractères "Voiture"
ListeAjoute(LISTE_Annonce, gARécupérer("Bateau") + gStylo(Noir) + " à vendre")
// LISTE_Annonce[2] retourne la chaîne de caractères "Bateau"
ListeAjoute(LISTE_Annonce, gARécupérer(gImage(2, 25) + "ABC") + "DEF")
// LISTE_Annonce[3] retourne la chaîne de caractères "2,25" + ESC + "ABC"
// Les fonctions gRectangle, gCercle, gLigne sont ignorées
ListeAjoute(LISTE_Annonce, gARécupérer(gRectangle(0, 0, 50, 18)))
// LISTE_Annonce[4] retourne une chaîne vide
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = gARécupérer(<Chaîne à retourner>)`
---

**`<Résultat> : (Chaîne de caractères)`**

Chaîne de contrôle à utiliser dans les fonctions [ListeAjoute](../WDLang1/3049004.md), [ListeInsère](../WDLang1/3049002.md) ou [ListeModifie](../WDLang1/3049009.md). Chaque élément retourné est séparé par la constante *ESC*.

**`<Chaîne à retourner> : (Chaîne de caractères)`**

Valeur texte à retourner. Seul ce paramètre sera retourné au langage. Ce paramètre sera affiché dans la ligne.

Ce paramètre peut contenir des fonctions de dessin dans les listes (fonctions commençant par la lettre "g"). Il est ainsi possible de :

- ajouter des dessins au texte,

- modifier la mise en forme du texte (couleur de fond, couleur du texte, etc.).




Les dessins créés seront également retournés au langage.

Le contenu de la &lt;Chaîne à retourner&gt; peut être recherché grâce à la fonction [ListeCherche](../WDLang1/3049003.md).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Éléments distincts
<a name="elements_distincts_ELTPARAGRAPHE000064"></a>Sont considérés comme deux éléments distincts :

- deux textes séparés par les fonctions [gStylo](../WDLang1/3029039.md) ou [gCoord](../WDLang1/3029034.md),

- un texte et une image.


Chaque élément retourné est séparé par la constante *ESC*.
<a name="NOTE0_2"></a>


### Colonne de type combo
<a name="colonne_type_combo_ELTPARAGRAPHE000082"></a>Pour les colonnes de type combo, le tri est fonction du contenu de la combo et non pas de l'indice de la combo. Ainsi, pour les combos dans les tables, le contenu de la combo qui va servir de critère de tri doit être décrit par la fonction **gARécupérer**.

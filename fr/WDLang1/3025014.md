
## BulleActive (Fonction)

***En anglais : ShowToolTip***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Active ou désactive le mécanisme d'affichage des bulles d'aide sur les champs.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Désactive l'utilisation des bulles d'aide
EtatPrécédent = BulleActive(Faux) // EtatPrécédent vaut Vrai
```
<a name="Exemple2"></a>

```wl
// Active l'utilisation des bulles d'aide
EtatPrécédent = BulleActive(Vrai) // EtatPrécédent vaut Faux
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = BulleActive(<Activation>)`
---

**`<Résultat> : (Booléen)`**

Etat précédent d'activation des bulles d'aide :

- <u><u><u><u>Vrai</u></u></u></u> : les bulles d'aide étaient affichées.

- <u><u><u><u>Faux</u></u></u></u> : les bulles d'aide n'étaient pas affichées.




**`<Activation> : (Booléen)`**

Nouvel état d'activation des bulles d'aide :

- <u><u><u><u>Vrai</u></u></u></u> : les bulles d'aide des champs (si elles existent) seront affichées. 

- <u><u><u><u>Faux</u></u></u></u> : les bulles d'aide des champs ne seront pas affichées. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Par défaut, les bulles d'aide des champs sont activées.

- L'état indiqué par la fonction **BulleActive** sera actif dans tout le projet jusqu'au prochain appel de la fonction **BulleActive**.

- Si les bulles d'aide ont été personnalisées pour le projet, le mécanisme d'affichage des bulles d'aide utilisera les bulles d'aide personnalisées. Pour plus de détails, consultez [Bulles d'aide personnalisées](../Editeurs/9500233.md).

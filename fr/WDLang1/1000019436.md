
## Constantes Capteur

<a name="NOTE1_1"></a>






| Constante | Description | Utilisée par les fonctions |
| --- | --- | --- |
| **cptAzimut** | Notification lors du changement d'orientation autour de l'axe vertical (Z). | [CapteurDétecteChangementOrientation](../WDLang1/1000019361.md), [CapteurRécupèreOrientation](../Editeurs/2030039.md) |
| **cptFréquenceJeu** | Fréquence de notification nécessaire lors de l'utilisation de la fonction dans un jeu. | [CapteurDétecteChangementAccélération](../WDLang1/1000019362.md), [CapteurDétecteChangementOrientation](../WDLang1/1000019361.md) |
| **cptFréquenceNormal** | Fréquence de notification moyenne. | [CapteurDétecteChangementAccélération](../WDLang1/1000019362.md), [CapteurDétecteChangementOrientation](../WDLang1/1000019361.md) |
| **cptFréquenceRapide** | Fréquence de notification la plus rapide. | [CapteurDétecteChangementAccélération](../WDLang1/1000019362.md), [CapteurDétecteChangementOrientation](../WDLang1/1000019361.md) |
| **cptLatéral** | Notification lors de l'accélération de l'appareil sur l'axe horizontal. | [CapteurDétecteChangementAccélération](../WDLang1/1000019362.md) |
| **cptLongitudinal** | Notification lors de l'accélération de l'appareil sur l'axe longitudinal. | [CapteurDétecteChangementAccélération](../WDLang1/1000019362.md) |
| **cptPitch** | Notification lors du changement d'orientation autour de l'axe horizontal (X). | [CapteurDétecteChangementOrientation](../WDLang1/1000019361.md), [CapteurRécupèreOrientation](../Editeurs/2030039.md) |
| **cptRoll** | Notification lors du changement d'orientation autour de l'axe longitudinal (Y). | [CapteurDétecteChangementOrientation](../WDLang1/1000019361.md), [CapteurRécupèreOrientation](../Editeurs/2030039.md) |
| **cptSensibilitéElevée** | Sensibilité élevée aux secousses. Même de faibles secousses de l'appareil déclencheront la notification. | [CapteurDétecteDébutSecousses](../WDLang1/1000019363.md), [CapteurDétecteFinSecousses](../WDLang1/1000019364.md) |
| **cptSensibilitéFaible** | Sensibilité faible aux secousses. Seules de fortes secousses de l'appareil déclencheront la notification. | [CapteurDétecteDébutSecousses](../WDLang1/1000019363.md), [CapteurDétecteFinSecousses](../WDLang1/1000019364.md) |
| **cptSensibilitéNormale** | Sensibilité normale aux secousses. | [CapteurDétecteDébutSecousses](../WDLang1/1000019363.md), [CapteurDétecteFinSecousses](../WDLang1/1000019364.md) |
| **cptVertical** | Notification lors de l'accélération de l'appareil sur l'axe vertical. | [CapteurDétecteChangementAccélération](../WDLang1/1000019362.md) |

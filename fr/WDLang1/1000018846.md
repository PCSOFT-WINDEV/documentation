
## AppliActiveVersion (Fonction)

***En anglais : AppActivateVersion***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Active une des versions de l'application disponible dans l'historique des versions sur le serveur d'installation de référence.

**Remarque** : Cette fonction n'est utilisable qu'avec une installation réseau sur un réseau local. Elle ne peut être employée avec une installation par IP.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Crée la liste les versions disponibles
ListeVersions est une chaîne
ListeVersions = AppliListeVersion()
// Extrait la dernière version
DernièreVersion est une chaîne 
DernièreVersion = ExtraitChaine(ListeVersions, 1, RC, DepuisFin)
// Activation de la version
SI AppliActiveVersion(DernièreVersion) = Faux ALORS
	Erreur("Erreur à l'activation de la version " + DernièreVersion + ...
		RC + ErreurInfo())
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = AppliActiveVersion(<Version> [, <Version forcée> [, <Exécutable>]])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'activation a réussi, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).
	Remarque : les causes d'erreurs les plus courantes sont : 

	- l'historique des versions n'est pas activé. 

	- la version demandée n'existe pas. 







**`<Version> : (Chaîne de caractères)`**

Numéro de version à activer. 

**`<Version forcée> : (Booléen optionnel)`**



- <u><u><u><u>Vrai</u></u></u></u> pour forcer la version à activer. Tous les postes seront mis à jour avec la version indiquée dans le paramètre &lt;Version&gt;, y compris les postes utilisant une version plus récente. 

- <u><u><u><u>Faux</u></u></u></u> (valeur par défaut) dans le cas contraire. 




**`<Exécutable> : (Chaîne de caractères optionnelle)`**

Chemin complet de l'exécutable correspondant à l'application à mettre à jour. Si ce paramètre n'est pas précisé, l'application en cours est mise à jour.

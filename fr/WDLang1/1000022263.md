


## Fonctions de cryptage / compression
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions de gestion du cryptage et de la compression sont les suivantes :



|   |   |
| --- | --- |
| [Compresse](../WDLang1/3024012.md) | Compresse une chaîne de caractères ou un bloc mémoire (buffer) sous forme binaire. |
| [Crypte](../WDLang1/3024014.md) | Crypte une chaîne de caractères sous forme binaire ou sous forme ASCII. |
| [CrypteAsymétrique](../WDLang1/1000024049.md) | Crypte un buffer en utilisant un algorithme de chiffrement asymétrique (RSA) nécessitant une clé publique et une clé privée. |
| [CrypteAsymétriqueEnveloppé](../WDLang1/1000024057.md) | Réalise un cryptage asymétrique d'une enveloppe. |
| [CrypteGénèreCléRSA](../WDLang1/1000024052.md) | Génère un couple clé privée/clé publique au format RSA. |
| [CrypteStandard](../WDLang1/1000021293.md) | Crypte une chaîne de caractères ou un buffer binaire en utilisant un algorithme de chiffrement symétrique (AES, DES, etc.). |
| [Décode](../WDLang1/1000022259.md) | Décode une chaîne de caractères précédemment encodée avec la fonction [Encode](../WDLang1/1000022258.md). |
| [Décompresse](../WDLang1/3024009.md) | Décompresse un bloc mémoire (buffer) préalablement compressé par la fonction [Compresse](../WDLang1/3024012.md). |
| [Décrypte](../WDLang1/3024035.md) | Décrypte une chaîne de caractères préalablement cryptée par la fonction [Crypte](../WDLang1/3024014.md). |
| [DécrypteAsymétrique](../WDLang1/1000024050.md) | Décrypte un buffer avec un algorithme clé privée/publique selon le cryptage RSA. |
| [DécrypteAsymétriqueEnveloppé](../WDLang1/1000024058.md) | Réalise le décryptage asymétrique d'une enveloppe encryptée avec la fonction [CrypteAsymétriqueEnveloppé](../WDLang1/1000024057.md). |
| [DécrypteStandard](../WDLang1/1000021294.md) | Décrypte un message préalablement crypté avec un algorithme de chiffrement symétrique (AES, DES, etc.). |
| [Encode](../WDLang1/1000022258.md) | Encode une chaîne de caractères ou un buffer. Pour décoder le résultat, il suffit d'utiliser la fonction [Décode](../WDLang1/1000022259.md). |
| [LuhnCalcule](../WDLang1/1410087276.md) | Calcule le prochain chiffre de contrôle de Luhn pour une suite donnée. |
| [LuhnVérifie](../WDLang1/1410087277.md) | Vérifie que la suite donnée respecte bien la formule de Luhn. |
| [UUDécode](../WDLang1/1000018864.md) | Décode une chaîne de caractères au format UUEncode. |
| [UUEncode](../WDLang1/1000018862.md) | Encode une chaîne de caractères selon l'algorithme UUEncode. |

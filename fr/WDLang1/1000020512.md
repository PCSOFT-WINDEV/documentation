
## SexagésimalVersDécimal (Fonction)

***En anglais : SexagesimalToDecimal***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie l'angle décimal correspondant à un angle sexagésimal.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Cherche l'angle en degrés correspondant à la latitude 45°53'36"
sLatitude est une chaîne = "45°53'36''"
rAngleLatitude est un réel = SexagésimalVersDecimal(sLatitude)
Info("La latitude " + sLatitude + " correspond à " + rAngleLatitude + " °")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = SexagésimalVersDécimal(<Angle sexagésimale>)`
---

**`<Résultat> : (Réel)`**

Résultat de la conversion au format décimal. 

**`<Angle sexagésimale> : (Chaîne de caractères)`**

Angle au format sexagésimal. Cette chaîne est de la forme 
```txt
XXX°XX'XX''
```

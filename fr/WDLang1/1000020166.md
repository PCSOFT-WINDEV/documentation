
## fRepRessource (Fonction)

***En anglais : fResourceDir***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le chemin des ressources en lecture seule de l'application.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// La fonction renvoie le contenu du fichier
fChargeBuffer(fRepRessource() + ["\"] + "data.bin")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = fRepRessource()`
---

**`<Résultat> : (Chaîne de caractères)`**

Chemin complet des ressources en lecture seule de l'application. Correspond au répertoire ".app" de l'application. Il n'est pas possible d'écrire dans ce répertoire.

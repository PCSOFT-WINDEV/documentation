


## ChaîneInverse (Fonction)

***En anglais : StringReverse***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Inverse les caractères d'une chaîne. 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
Trace(ChaîneInverse("ABC"))
// La fenêtre de trace affiche "CBA"
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = ChaîneInverse(<Chaîne>)`
---

**`<Résultat> : (Chaîne de caractères Ansi, Unicode ou Binaire)`**

Chaîne de caractère inversée. 

**`<Chaîne> : (Chaîne de caractères Ansi, Unicode ou Binaire)`**

Chaîne de caractères à inverser.

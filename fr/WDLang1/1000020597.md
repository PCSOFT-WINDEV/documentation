


## InterrupteurInsère (Fonction)

***En anglais : CheckBoxInsert***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Insère une nouvelle option dans un champ Interrupteur.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Insère une option à la position 2 
InterrupteurInsère(INT_Options, 2, "Lire un livre sur une tablette")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = InterrupteurInsère(<Champ Interrupteur> , <Indice de l'élément> , <Libellé> [, <Sous-libellé> [, <Image>]])`
---

**`<Résultat> : (Entier)`**



- Indice de l‘option insérée. 

- -1 en cas d'erreur. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Champ Interrupteur> : (Nom de champ)`**

Nom du champ Interrupteur dans lequel l'option doit être insérée.

**`<Indice de l'élément> : (Entier)`**

Indice de l'élément à insérer. 

Si ce paramètre :

- est supérieur au nombre d'options présentes dans l'interrupteur, l'option est ajoutée après la dernière option du champ Interrupteur. 

- est égal à 1 ou 0, l'élément est ajouté avant la première option du champ Interrupteur.




**`<Libellé> : (Chaîne de caractères)`**

Libellé de l'option à ajouter.

**`<Sous-libellé> : (Chaîne de caractères optionnelle)`**

Sous-libellé de l'option à insérer. 
Si ce paramètre n'est pas spécifié, l'option ajoutée n'a pas de sous-libellé. 
Si ce paramètre est spécifié, le sous-libellé sera affiché même si l'interrupteur utilise un mode d'affichage sans sous-libellé.


**`<Image> : (Chaîne de caractères optionnelle)`**

Image associée à l'option à insérer. 
Si ce paramètre n'est pas spécifié, l'option ajoutée n'a pas d'image.
Si ce paramètre est spécifié, l'image sera affichée uniquement si l'interrupteur utilise un mode d'affichage "Style Vista" (avec image).




<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Les caractéristiques de l'option insérée peuvent également être modifiées grâce aux propriétés suivantes utilisées sur l'option : 

	- Propriété [Libellé](../Proprietes/2510053.md) pour modifier le libellé. 

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Propriété [SousLibellé](../Proprietes/1000017232.md) pour modifier le sous-libellé. 

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Propriété [Image](../Proprietes/2510034.md) pour modifier l'image. 




- Lors de l'insertion d'options dans un champ Interrupteur, la taille du champ n'est pas modifiée. Si le champ n'est pas suffisamment grand, les options ajoutées pourront ne pas être visibles. Il est possible d'augmenter la taille du champ avec les propriétés [Largeur](../Proprietes/2510059.md) et [Hauteur](../Proprietes/2510050.md).

- Pour ajouter une option à la suite des options existantes, utilisez la fonction [InterrupteurAjoute](../WDLang1/1000020595.md). 

- Lors de l'insertion d'une option : 

	- l'option insérée a pour indice &lt;Indice de l'élément&gt;. 

	- les indices des options suivant l'option insérée sont automatiquement renumérotés.

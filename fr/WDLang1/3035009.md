
## ExeListeDLL (Fonction)

***En anglais : ExeListDLL***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Retourne la liste des librairies (fichiers ".DLL") utilisées par une des applications en cours d'exécution. Cette application peut correspondre à une application WINDEV, WINDEV Mobile ou à une autre application.

**Remarque** : Seules les librairies en mémoire sont listées.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// -- Code d'initialisation du champ "LISTE_Appli"
// Ajout des applications en cours d'exécution dans le champ "LISTE_Appli"
ListeAjoute(LISTE_Appli, ExeListeProcessus(exePID, exeNomCourt))

// -- Code de sélection d'une ligne de "LISTE_Appli"
// Extraction de l'identifiant de l'application sélectionnée dans la liste
InfoAppli est une chaîne = LISTE_Appli[LISTE_Appli]
IDAppli est une chaîne = ExtraitChaîne(InfoAppli, 1)
// Affichage des DLL en mémoire dans le champ "LISTE_DLL"
ListeAjoute(LISTE_DLL, ExeListeDLL(Val(IDAppli)))
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = ExeListeDLL([<Identifiant de l'application>])`
---

**`<Résultat> : (Chaîne de caractères)`**

Liste des librairies (fichiers ".DLL") utilisées par l'application spécifiée sous la forme :

```txt
<1° librairie> + RC + <2° librairie> + RC + ... + <N° librairie>
```
Où &lt;X librairie&gt; correspond au nom et chemin complet de la librairie X.

**`<Identifiant de l'application> : (Constante optionnelle de type Entier)`**

Identifiant de l'application à manipuler. Cet identifiant peut être connu grâce à la fonction [ExeListeProcessus](../WDLang1/3035008.md).
  
Si ce paramètre n'est pas spécifié, &lt;Résultat&gt; correspondra à la liste des librairies du processus en cours. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
La fonction **ExeListeDLL** ne retourne pas de code d'erreur. Pour savoir si cette fonction a généré une erreur, utilisez la variable [ErreurDétectée](../WDLang1/3087001.md). En cas d'erreur, il est possible de connaître le détail de l'erreur grâce à la fonction [ErreurInfo](../WDLang1/3013008.md).

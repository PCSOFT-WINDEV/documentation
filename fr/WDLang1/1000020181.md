
## fCopieFichierRépertoireWeb (Fonction)

***En anglais : fCopyFileWebFolder***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Copie un fichier image depuis le répertoire des données de l'application (ou un de ses sous-répertoires) vers le répertoire "_WEB" de l'application (ou un de ses sous-répertoires).

**Attention** : Pour utiliser cette fonction, l'option "Autoriser les fonctions fCopieFichierRépertoireWeb et fSupprimeFichierRépertoireWeb." doit être cochée dans l'administrateur WEBDEV (option présente dans l'onglet "Avancé" de l'administrateur WEBDEV). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
fCopieFichierRépertoireWeb("ImageTemporaire.png", "images/produit.png")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = fCopieFichierRépertoireWeb(<Fichier source> , <Fichier destination>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le fichier a été copié, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Fichier source> : (Chaîne de caractères)`**

Fichier à copier. Ce paramètre peut correspondre à : 

- Un nom de fichier avec son chemin complet. 

- Un nom de fichier avec son chemin relatif au répertoire des données de l'application. Le répertoire des données de l'application peut être connu grâce à la fonction [fRepDonnées](../WDLang1/3036001.md). 




**`<Fichier destination> : (Chaîne de caractères)`**

Fichier destination de la copie. Ce paramètre peut correspondre à : 

- Un nom de fichier avec son chemin complet. 

- Un nom de fichier avec son chemin relatif au répertoire "_WEB" de l'application. Le répertoire "_WEB" de l'application peut être connu grâce à la fonction [fRepWeb](../WDLang1/3036052.md). 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- La fonction **fCopieFichierRépertoireWeb** copie uniquement un seul fichier à la fois. 

- Les caractères jokers (\*, ?) sont interdits dans le nom du fichier.

- Seuls les fichiers image peuvent être copiés (les extensions acceptées sont jpg, jpeg, png, gif, bmp, tif, tiff et  svg).

- Si le répertoire destination est un sous-répertoire du répertoire "_WEB", ce sous-répertoire est créé s'il n'existe pas.

- **Attention** : La copie est prise en charge par l'administrateur de WEBDEV sur le serveur (WD280ADMIN.EXE). Le compte Windows qui l'exécute doit disposer de droits suffisants sur l'emplacement cible de la copie.

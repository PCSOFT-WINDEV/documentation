


## &lt;Variable TâcheParallèle&gt;.DemandeAnnulation (Fonction)

***En anglais : &lt;ParallelTask variable&gt;.RequestCancelation***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Demande l'annulation d'une tâche parallèle.

<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Déclenche l'exécution d'une tâche
t est une TâcheParallèle = Proc.Exécute()
...
// Annule l'exécution de la tâche
t.DemandeAnnulation()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Demander l'annulation d'une tâche parallèle

`<Tâche>.DemandeAnnulation()`
---

**`<Tâche> : (Variable de type TâcheParallèle)`**

Nom de la variable de type [TâcheParallèle](../WDLang1/1000021220.md) à manipuler.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Une demande d'annulation est transmise à la tâche. Le code de la tâche doit gérer la demande d'annulation grâce à la syntaxe suivante : 
	
	```txt
	MaTâcheParallèle.Annulée
	```


- L'appel de la fonction **&lt;Variable TâcheParallèle&gt;.DemandeAnnulation** est non bloquant. La tâche est considérée comme annulée dès l'appel à la fonction.

- L'état de la tâche passe à *tpeAnnulationDemandée*. Tout appel à la propriété [ValeurRenvoyée](../Proprietes/2510134.md) provoquera une erreur fatale.

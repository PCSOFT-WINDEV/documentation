
## PlanningPosition (Fonction)

***En anglais : SchedulerPosition***






	<B><B>Avertissement</B></B>

	A partir de la version <B><B>17</B></B>,  cette fonction est gardée par compatibilité en WINDEV. En WINDEV, il est conseillé d'utiliser la fonction [PlanningPositionDateHeure](../WDLang1/1000019998.md). 



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Affiche un champ Planning à partir d'une date spécifiée ou renvoie la date du premier jour (ou dernier jour) affiché dans un champ Planning. 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Positionne le planning sur la date du jour
PlanningPosition(PLN_Planning, DateDuJour())

// Positionne le planning au 20 décembre 2016
PlanningPosition(PLN_Planning, "20161220")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Positionner le planning sur une date spécifiée

`PlanningPosition(<Champ Planning> , <Date de début>)`
---

**`<Champ Planning> : (Nom de champ)`**

Nom du champ Planning à manipuler. Ce champ peut correspondre : 

- au champ Planning d'une fenêtre, 

- au champ Planning d'une page,

- au champ Planning d'un état.




**`<Date de début> : (Chaîne de caractères ou variable de type Date)`**

Date de début de l'affichage.


<a name="SYNTAXE2"></a>

### Connaître la première date ou la dernière date affichée par le champ Planning

`<Résultat> = PlanningPosition(<Champ Planning> [, <Date recherchée>])`
---

**`<Résultat> : (Chaîne de caractères)`**

Date affichée par le champ Planning.

**`<Champ Planning> : (Nom de champ)`**

Nom du champ Planning à manipuler. Ce champ peut correspondre : 

- au champ Planning d'une fenêtre,

- au champ Planning d'une page,

- au champ Planning d'un état.




**`<Date recherchée> : (Constante optionnelle de type Entier)`**

Type de date recherchée : 


|   |   |
| --- | --- |
| *plnDateDébut* <br>(Valeur par défaut) | Première date affichée dans le champ Planning. |
| *plnDateFin* | Dernière date affichée dans le champ Planning. |
| *plnMoisEnCours* | Premier jour du mois affiché dans le champ Planning. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Positionnement dans le champ Planning
<a name="positionnement_dans_champ_planning_ELTPARAGRAPHE000079"></a>Lors de l'utilisation de la fonction **PlanningPosition** pour afficher une date, le planning conserve vos préférences d'affichage. 

Par exemple, si le planning est affiché à la semaine commençant un Lundi, la fonction **PlanningPosition** utilisée sur la date du vendredi affiche le planning du Lundi au Dimanche pour la semaine contenant le vendredi spécifié.

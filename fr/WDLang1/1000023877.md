
## Fonctions de gestion du champ Lecteur PDF (syntaxe préfixée)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions de gestion des champs Lecteur PDF (syntaxe préfixée) sont les suivantes : 



|   |   |
| --- | --- |
| [&lt;Lecteur PDF&gt;.AfficheSignet](../WDLang1/1410089536.md) | Positionne le document PDF affiché dans le champ Lecteur PDF sur le signet demandé. |
| [&lt;Lecteur PDF&gt;.Ferme](../WDLang1/1000023695.md) | Ferme et libère le fichier PDF affiché dans un champ Lecteur PDF. |
| [&lt;Lecteur PDF&gt;.InfoXY](../WDLang1/1000025359.md) | Renvoie pour une position donnée dans un champ Lecteur PDF (coordonnées d'un point du champ) : <br><br>- soit le numéro de la page pour la position indiquée, <br><br>- soit la position X (en mm) pour la position indiquée, <br><br>- soit la position Y (en mm) pour la position indiquée.<br><br><br> |
| [&lt;Lecteur PDF&gt;.Ouvre](../WDLang1/1000023776.md) | Ouvre un fichier PDF dans un champ Lecteur PDF. |
| [&lt;Lecteur PDF&gt;.RecherchePrécédent](../WDLang1/1000023777.md) | Positionne sur le résultat précédent d'une recherche initiée par la fonction [&lt;Lecteur PDF&gt;.RecherchePremier](../WDLang1/1000023778.md) dans le contenu d'un champ Lecteur PDF. |
| [&lt;Lecteur PDF&gt;.RecherchePremier](../WDLang1/1000023778.md) | Initialise la recherche d'un texte dans le fichier PDF affiché par le champ Lecteur PDF et positionne sur la première occurrence trouvée (à partir de la page en cours). |
| [&lt;Lecteur PDF&gt;.RechercheSuivant](../WDLang1/1000023779.md) | Positionne sur le résultat suivant d'une recherche dans le contenu d'un champ Lecteur PDF. |
| [&lt;Lecteur PDF&gt;.Sauve](../WDLang1/1000024900.md) | Sauve le contenu d'un champ Lecteur PDF dans un fichier. |
| [&lt;Lecteur PDF&gt;.Surligne](../WDLang1/1000023545.md) | Cherche un texte dans le document affiché par un champ Lecteur PDF et surligne les occurrences de ce texte dans le champ. |
| [&lt;Lecteur PDF&gt;.TournePage](../WDLang1/1000024412.md) | Effectue une rotation d'une page affichée dans le champ Lecteur PDF. |





Remarque : La fonction [iImprimePDF](../WDLang5/1000020519.md) peut être utilisée pour imprimer le contenu d'un champ Lecteur PDF.

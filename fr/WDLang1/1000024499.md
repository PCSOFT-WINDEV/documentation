
## &lt;Variable picCalque&gt;.PixelOpacité (Fonction)

***En anglais : &lt;picLayer variable&gt;.PixelOpacity***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Identifie l'opacité d'un pixel d'une image (présente dans une variable de type [picCalque](../WDLang1/1000024604.md)) gérant la couche Alpha.

![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Les dessins](../WDLang1/3029035.md). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonImageWDPic est un WDPic = "Test.bmp"
MonCalque est un picCalque
// Identification de l'opacité du pixel (15, 25)
// sur le calque
ResOpacité = MonCalque.PixelOpacité(15, 25)
IMG_MonDessin = MonImageWDPic
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Image picCalque>.PixelOpacité(<Abscisse> , <Ordonnée>)`
---

**`<Résultat> : (Entier)`**

Opacité du pixel spécifié (comprise entre 0 pour transparent, et 255 pour une opacité complète).

**`<Image picCalque> : (Variable de type picCalque)`**

Nom de la variable de type [picCalque](../WDLang1/1000024604.md) à manipuler.

**`<Abscisse> : (Entier)`**

Position horizontale du pixel à manipuler.

**`<Ordonnée> : (Entier)`**

Position verticale du pixel à manipuler.

## Remarques


### Coordonnées
<a name="coordonnees_ELTPARAGRAPHE000348"></a>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=dpixel.gif)
Les coordonnées sont indiquées en pixels par rapport au coin haut gauche de l'image (coordonnées (0,0)).


<a name="NOTE0_2"></a>
<a name="NOTE0_3"></a>


### Dessin dans une image avec opacité
<a name="dessin_dans_une_image_avec_opacite_ELTPARAGRAPHE000449"></a>Pour spécifier que le dessin dans l'image gère l'opacité (couche Alpha), utilisez la fonction [&lt;Variable picCalque&gt;.DébutDessin](../WDLang1/1000024470.md) en spécifiant la constante ***dAvecOpacité***.




## ChronoPause (Fonction)

***En anglais : ChronoPause***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Interrompt un chronomètre afin de mesurer la durée d'un traitement. Le chronométrage pourra être relancé par la fonction [ChronoReprend](../WDLang1/1000018830.md).






<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
DuréeTraitement est une Durée
ChronoDébut()
MaFonction1()
DuréeTraitement = ChronoPause()
Info("Le traitement MaFonction1 a duré " + DuréeTraitement) 
ChronoReprend() 
MaFonction2() 
DuréeTraitement = ChronoFin()
Info("Les traitements MaFonction1 et MaFonction2 ont duré " + DuréeTraitement)
```

## Syntaxe

### Interrompre le chronométrage réalisé avec un numéro de chronomètre

`<Résultat> = ChronoPause([<Numéro du chronomètre>])`
---

**`<Résultat> : (Durée)`**

Temps passé (en millisecondes) depuis le dernier appel à la fonction [ChronoDébut](../WDLang1/3027031.md) pour le numéro de chronomètre spécifié.

**`<Numéro du chronomètre> : (Entier optionnel)`**

Identifiant du chronomètre utilisé. Si ce paramètre n'est pas précisé, sa valeur par défaut est 1.

### Interrompre le chronométrage réalisé avec une variable de type Chrono

`<Résultat> = ChronoPause(<Chronomètre>)`
---

**`<Résultat> : (Durée)`**

Temps passé (en millisecondes) depuis le dernier appel à la fonction [ChronoDébut](../WDLang1/3027031.md) pour le chronomètre spécifié.

**`<Chronomètre> : (Variable de type Chrono)`**

Nom de la variable de type [Chrono](../WDLang1/1000023560.md) correspondant au chronomètre à utiliser. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Divers
<a name="divers_ELTPARAGRAPHE000171"></a>Si la fonction **ChronoPause** est utilisée sur un chronomètre déjà interrompu, cette fonction n'a aucun effet. 
<a name="NOTE0_2"></a>


### Erreur
<a name="erreur_ELTPARAGRAPHE000183"></a>Si la fonction [ChronoDébut](../WDLang1/3027031.md) n'a pas été appelée avant la fonction **ChronoPause**, une erreur WLangage est affichée.
<a name="NOTE0_3"></a>


### Threads
<a name="threads_ELTPARAGRAPHE000198"></a>Les fonctions de chronométrage (Chronoxxx) concernant un même chronomètre doivent être utilisées dans un même thread.

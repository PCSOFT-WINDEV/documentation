
## OUExclusifBinaire (Fonction)

***En anglais : BinaryXOR***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le résultat du OU Exclusif logique (bit à bit) entre deux valeurs.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Récupération du OU Exclusif logique
ResOUEXBinaire = OUExclusifBinaire(6, 6)  // Renvoie 0
ResOUEXBinaire = OUExclusifBinaire(6, 2)  // Renvoie 4
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = OUExclusifBinaire(<Valeur 1> , <Valeur 2>)`
---

**`<Résultat> : (Entier)`**

Résultat du OU Exclusif logique.

**`<Valeur 1> : (Entier)`**

Première valeur à tester.

**`<Valeur 2> : (Entier)`**

Seconde valeur à tester.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Calcul bit à bit
<a name="calcul_bit_bit_ELTPARAGRAPHE000066"></a>La fonction **OUExclusifBinaire** effectue le calcul bit à bit entre &lt;Valeur 1&gt; et &lt;Valeur 2&gt; selon la table suivante :


|   |   |   |
| --- | --- | --- |
|   | **0** | **1** |
| **0** | 0 | 1 |
| **1** | 1 | 0 |




<a name="NOTE0_2"></a>




### Opérateur binaire
<a name="operateur_binaire_ELTPARAGRAPHE000095"></a>

- La fonction **OUExclusifBinaire** est équivalente à l'opérateur binaire **||** pour les entiers sur 4 octets. Pour plus de détails, consultez [Opérateurs binaires](../Motscles/1512009.md).

- Le résultat de la fonction **OUExclusifBinaire** est exact sur des entiers sur 4 octets. Son utilisation avec des entiers sur 8 octets provoque un débordement de capacité.

- L'opérateur binaire **||** permet de faire des opérations sur des entiers sur 8 octets.

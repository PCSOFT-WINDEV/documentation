


## ArbreDérouleTout (Fonction)

***En anglais : TreeExpandAll***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Déroule un noeud et tous ses fils dans un champ Arbre.

Exemple de champ Arbre :
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Arbre_Vocabulaire.gif)
**Rappel** : En exécution, un noeud est déroulé lorsque le signe "-" précède le nom du noeud.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Déroule le noeud "Desserts" dans le champ Arbre "ARBRE_RecetteTV"
// Tous les éléments fils sont également déroulés
Res = ArbreDérouleTout(ARBRE_RecetteTV, "Recettes" + TAB + "Desserts")
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = ArbreDérouleTout(<Champ Arbre> [, <Noeud à dérouler>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le noeud et ses fils ont été déroulés, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Champ Arbre> : (Nom de champ)`**

Nom du champ Arbre à manipuler. 

**`<Noeud à dérouler> : (Chaîne de caractères optionnelle)`**

Chemin du noeud à dérouler (ainsi que ses fils). Ce paramètre est de la forme :

```txt
"<Nom de la racine>" + TAB + ["<Nom du 1er noeud>" + TAB + ...
["<Nom du 2ème noeud>" + TAB + [...]]]"<Nom de la feuille>"
```


Si ce paramètre n'est pas précisé, le champ Arbre est déroulé depuis la racine.

En cas de doublons sur un élément du chemin, ce chemin peut contenir l'identifiant de l'élément (indiqué à la suite du nom grâce à la fonction [ArbreID](../WDLang1/3018027.md)).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Noeud à dérouler
<a name="noeud_derouler_ELTPARAGRAPHE000071"></a>Si le noeud à dérouler :

- est introuvable, la fonction **ArbreDérouleTout** renvoie <u><u><u><u>Faux</u></u></u></u>.

- n'est pas précisé, la fonction **ArbreDérouleTout** renvoie <u><u><u><u>Vrai</u></u></u></u> et le champ Arbre est déroulé depuis sa racine.

- est une feuille, la fonction **ArbreDérouleTout** renvoie <u><u><u><u>Vrai</u></u></u></u>.

- est déjà déroulé, la fonction **ArbreDérouleTout** renvoie <u><u><u><u>Vrai</u></u></u></u>, et déroule les éléments fils si nécessaire.

- n'est pas visible, la fonction **ArbreDérouleTout** rend ce noeud visible. Ses "parents" et ses "frères" sont également affichés. Les éléments "parents" sont déroulés.


Lors de l'appel à la fonction **ArbreDérouleTout**, si la branche spécifiée et/ou ses fils possèdent une procédure associée par la propriété [AjouteFilsDifféré](../Proprietes/1000020927.md), cette procédure sera exécutée. 
<a name="NOTE0_2"></a>


### Bandeau de sélection
<a name="bandeau_selection_ELTPARAGRAPHE000127"></a>La fonction **ArbreDérouleTout** ne modifie pas la position du bandeau de sélection.

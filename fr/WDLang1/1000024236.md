


## &lt;Variable Image&gt;.DessineRectangle (Fonction)

***En anglais : &lt;Image variable&gt;.DrawRectangle***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Dessine un rectangle dans une variable de type [Image](../WDLang1/1000019650.md).



![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Les dessins](../WDLang1/3029035.md). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonImage est une Image = "Test.bmp"

// Dessin d'un rectangle dont le fond est jaune clair
// et le bord bleu clair
MonImage.DessineRectangle(10, 10, 50, 50, JauneClair, BleuClair)

IMG_MonDessin = MonImage
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Dessiner un rectangle en spécifiant les coordonnées du rectangle

`<Variable Image>.DessineRectangle(<X1> , <Y1> , <X2> , <Y2> [, <Fond> [, <Trait>]])`
---

**`<Variable Image> : (Variable de type Image)`**

Nom de la variable de type [Image](../WDLang1/1000019650.md) à manipuler.

**`<X1> : (Entier)`**

Abscisse en pixels du coin haut gauche du rectangle.

**`<Y1> : (Entier)`**

Ordonnée en pixels du coin haut gauche du rectangle.

**`<X2> : (Entier)`**

Abscisse en pixels du coin bas droit du rectangle.

**`<Y2> : (Entier)`**

Ordonnée en pixels du coin bas droit du rectangle.

**`<Fond> : (Entier, constante ou variable de type Fond (optionnel))`**



- Couleur du fond du rectangle. Cette couleur peut être :

	- une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)),

	- une couleur TSL (obtenue avec la fonction [TSL](../WDLang1/3029057.md)),

	- [une couleur prédéfinie du WLangage](../WDLang5/3010002.md), 

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) une variable de type [Couleur](../WDLang1/1000019661.md), 

	- transparente si ce paramètre n'est pas précisé et si la fonction [&lt;Variable Image&gt;.Fond](../WDLang1/1000024056.md) n'a pas été utilisée auparavant. 




- Caractéristiques du fond : 

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) variable de type [Fond](../WDLang1/1000022038.md). Dans ce cas, le dégradé défini dans la variable n'est pas pris en compte. 
			

	- caractéristiques spécifiées dans le dernier appel à la fonction [&lt;Variable Image&gt;.Fond](../WDLang1/1000024056.md) si ce paramètre n'est pas précisé. 







Pour utiliser une couleur de fond dégradée, utilisez la fonction [&lt;Variable Image&gt;.DessineRectangleDégradé](../WDLang1/1000024238.md).

**`<Trait> : (Entier, constante ou variable de type Cadre (optionnel))`**



- Couleur du trait du rectangle. Cette couleur peut être :

	- une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)),

	- une couleur TSL (obtenue avec la fonction [TSL](../WDLang1/3029057.md)),

	- [une couleur prédéfinie du WLangage](../WDLang5/3010002.md),

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) une variable de type [Couleur](../WDLang1/1000019661.md), 

	- identique à la couleur utilisée pour le fond si ce paramètre n'est pas précisé et si la fonction [&lt;Variable Image&gt;.Stylo](../WDLang1/1000024240.md) n'a pas été utilisée.  




- Caractéristiques du trait : 

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) variable de type [Cadre](../Motscles/15140100.md). Dans ce cas, seules les propriétés suivantes sont prises en compte : 

		- Couleur : si la propriété correspond à la constante *CouleurIndéfinie*, la couleur définie lors du dernier appel à la fonction [&lt;Variable Image&gt;.Stylo](../WDLang1/1000024240.md) est utilisée. Si cette fonction n'a pas été utilisée, la couleur de fond du rectangle est utilisée. 

		- Epaisseur : si la propriété correspond à la constante *EpaisseurIndéfinie*, l'épaisseur définie lors du dernier appel à la fonction [&lt;Variable Image&gt;.Stylo](../WDLang1/1000024240.md) est utilisée. Si cette fonction n'a pas été utilisée, l'épaisseur vaut 1. 

		- Trait : si la propriété correspond à *TraitIndéfini*, le type de trait défini lors du dernier appel à la fonction [&lt;Variable Image&gt;.Stylo](../WDLang1/1000024240.md) est utilisé. Si cette fonction n'a pas été utilisée, le trait est un trait continu.  




- caractéristiques précisées dans le dernier appel à la fonction [&lt;Variable Image&gt;.Stylo](../WDLang1/1000024240.md) si ce paramètre n'est pas précisé.

### Dessiner un rectangle via une variable de type Rectangle

Nom de la variable de type [Image](../WDLang1/1000019650.md) à manipuler.

**`<Rectangle> : (Variable de type Rectangle)`**

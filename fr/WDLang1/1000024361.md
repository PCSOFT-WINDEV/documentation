


## &lt;Variable Image&gt;.DessineOmbre (Fonction)

***En anglais : &lt;Image variable&gt;.DrawShadow***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ajoute un effet d'ombre sur une image contenue dans une variable de type [Image](../WDLang1/1000019650.md). 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonImage est une Image
MonImage = "Image.png"
// Ajoute une ombre
MonImage.DessineOmbre()

// Ajoute une ombre jaune
MonImage.DessineOmbre(JauneBulle)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Variable Image>.DessineOmbre([<Couleur de l'ombre> [, <Opacité de l'ombre> [, <Détourer>]]])`
---

**`<Variable Image> : (Variable de type Image)`**

Nom de la variable de type [Image](../WDLang1/1000019650.md) correspondant à l'image à manipuler.

**`<Couleur de l'ombre> : (Entier optionnel)`**

Couleur de l'ombre à ajouter à l'image. Cette couleur peut correspondre à :

- une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)),

- une couleur TSL (obtenue avec la fonction [TSL](../WDLang1/3029057.md)),

- [une couleur prédéfinie du WLangage](../WDLang5/3010002.md).




Par défaut, l'ombre est noire.

**`<Opacité de l'ombre> : (Entier optionnel)`**

Pourcentage d'opacité de l'ombre à ajouter (pour le mixage avec les pixels existants). Par défaut, cette valeur est de 100 : aucune opacité n'est appliquée.

**`<Détourer> : (Booléen optionnel)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'ombre doit être détourée, 

- <u><u><u><u>Faux</u></u></u></u> (valeur par défaut) dans le cas contraire.


## HaspHLInfo (Fonction)

***En anglais : HaspHLInfo***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Récupère des informations sur la clé électronique Hasp.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
Trace("Informations sur la clé :")
Trace(HaspHLInfo(SAI_ID_Session, haspInfoClé))
Trace("Informations sur la session :")
Trace(HaspHLInfo(SAI_ID_Session, haspInfoSession))
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = HaspHLInfo(<Identifiant de la session> , <Information> [, <Identifiant du fichier>])`
---

**`<Résultat> : (Chaîne de caractères ou Entier (selon l'information demandée))`**

Information récupérée.

**`<Identifiant de la session> : (Entier)`**

Identifiant de la session Hasp renvoyé par la fonction [HaspHLConnecte](../WDLang1/1000018891.md).

**`<Information> : (Constante de type Entier)`**

Information à récupérer :


|   |   |
| --- | --- |
| *haspInfoClé* | &lt;Résultat&gt; est une chaîne de caractères contenant des informations sur la clé électronique en XML. |
| *haspInfoSession* | &lt;Résultat&gt; est une chaîne de caractères contenant des informations sur la session Hasp en cours en XML. |
| *haspInfoTailleFichier* | &lt;Résultat&gt; est un entier correspondant à la taille en octets du fichier dont l'identifiant correspond au paramètre &lt;Identifiant du fichier&gt;. |



**`<Identifiant du fichier> : (Entier optionnel ou constante optionnelle de type Entier)`**

Identifiant du fichier dont la taille doit être calculée. Cet identifiant peut correspondre :

- à un entier. 

- à une des constantes suivantes :
	


|   |   |
| --- | --- |
| *haspFichierLicence* | Fichier "zone de licence" de la clé. |
| *haspFichierPrincipal* | Fichier "zone de mémoire principale" de la clé. |

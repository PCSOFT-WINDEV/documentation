
## gCoord (Fonction)

***En anglais : gCoord***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Construit une séquence de caractères permettant de définir la position du curseur d'écriture d'un texte. 

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Dans une fenêtre, cette séquence de caractères permet de définir la position du curseur d'écriture d'un texte : 

- dans un élément d'un champ Liste ou Combo.

- dans la barre de titre et/ou de message d'une fenêtre.

- dans un champ Libellé.







<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 
```wl
// Ajout en relief du mot "Relief" dans le champ "LISTE_TypeEcriture"
ListeAjoute(LISTE_TypeEcriture, gCoord(2,2) + "Relief" + 18 + gCoord(1,1) + "Relief")
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = gCoord(<X> , <Y>)`
---

**`<Résultat> : (Chaîne de caractères)`**

Séquence de caractères définissant la position d'écriture.

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Cette séquence de caractères peut être : 

- utilisée dans les fonctions [ListeAjoute](../WDLang1/3049004.md), [ListeInsère](../WDLang1/3049002.md) ou [ListeModifie](../WDLang1/3049009.md).

- affectée à un champ Libellé. 

- utilisée dans la barre de titre ou de message d'une fenêtre. 




**`<X> : (Réel)`**

Abscisse (en pixels) de la nouvelle position du curseur pour le prochain texte à écrire. Si ce paramètre correspond à une chaîne vide (""), l'abscisse correspondant à la position du curseur ne sera pas modifiée.

**`<Y> : (Réel)`**

Ordonnée (en pixels) de la nouvelle position du curseur pour le prochain texte à écrire. Si ce paramètre correspond à une chaîne vide (""), l'ordonnée correspondant à la position du curseur ne sera pas modifiée.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Toutes les coordonnées sont exprimées en pixels par rapport au coin haut gauche de l'élément manipulé (par exemple le coint haut gauche de la ligne du champ Liste ou Combo).




## &lt;Table&gt;.SelectMoins (Fonction)

***En anglais : &lt;Table&gt;.SelectMinus***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Dé-sélectionne une ligne dans un champ Table ou Table hiérarchique. La ligne sélectionnée correspond à la ligne en cours sur laquelle le bandeau de sélection est affiché.

<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Dé-sélectionne la ligne 5 dans le champ Table "TABLE_Produit"
TABLE_Produit.SelectMoins(5)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Champ Table>.SelectMoins([<Indice 1> [... [, <Indice N>]]])`
---

**`<Champ Table> : (Nom de champ)`**

Nom du champ à manipuler. Ce champ peut correspondre à : 

- un champ Table. 

- un champ Table hiérarchique.




**`<Indice 1> : (Entier optionnel)`**

Indice de la première ligne à dé-sélectionner. Si ce paramètre n'est pas précisé, toutes les lignes du champ spécifié sont dé-sélectionnées. Si ce paramètre correspond à une ligne non sélectionnée, la fonction **&lt;Table&gt;.SelectMoins** est sans effet. L'indice de la ligne sélectionnée peut être obtenu avec la fonction [&lt;Table&gt;.Select](../WDLang1/1000024284.md).
Si ce paramètre correspond à -1, la fonction n'a aucun effet.




**`<Indice N> : (Entier optionnel)`**

Indice de la Nième ligne à dé-sélectionner. Si ce paramètre n'est pas précisé, toutes les lignes du champ Table spécifié sont dé-sélectionnées. Si ce paramètre correspond à une ligne non sélectionnée, la fonction **&lt;Table&gt;.SelectMoins** est sans effet. L'indice de la ligne sélectionnée peut être obtenu avec la fonction [&lt;Table&gt;.Select](../WDLang1/1000024284.md).




<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Conditions d'utilisation
<a name="conditions_utilisation_ELTPARAGRAPHE000238"></a>La fonction **&lt;Table&gt;.SelectMoins** peut être utilisée indifféremment sur :

- un champ Table ou Table hiérarchique fichier.

- un champ Table ou Table hiérarchique par programmation.

- un champ monosélection ou multisélection.




## AnalyseurFin (Fonction)

***En anglais : ProfilerEnd***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Arrête la "collecte d'informations" pour l'[analyseur de performances](../Editeurs/2030035.md).

Le fichier WPF contenant l'analyse de tous les traitements effectués est enregistré. Ce fichier pourra être ouvert avec l'[analyseur de performances](../Editeurs/2030035.md) ou directement sous l'éditeur, via ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_ouvrir.gif)
 parmi les boutons d'accès rapide.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
AnalyseurDébut("Test_ProcédureLancement.wpf")
ProcédureLancement()
AnalyseurFin()
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`AnalyseurFin()`
---



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Si la fonction [AnalyseurDébut](../WDLang1/3013012.md) n'a pas été appelée avant la fonction **AnalyseurFin**, l'appel de la fonction **AnalyseurFin** est sans effet.




## TableFilsOccurrence (Fonction)

***En anglais : TableChildCount***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le nombre de fils directs d'un élément dans un champ Table hiérarchique.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// On veut compter le nombre de desserts présents dans la carte du restaurant
nNombreDessert est un entier 
nNombreDessert = TableFilsOccurrence(TABLEH_Carte, "Desserts")
Info("Il y a actuellement : " + nNombreDessert + " disponibles.")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Fils d'une ligne identifiée par son numéro

`<Résultat> = TableFilsOccurrence(<Champ Table hiérarchique> , <Numéro de la ligne>)`
---

**`<Résultat> : (Entier)`**



- Nombre de fils de l'élément.

- -1 si le numéro de la ligne spécifée n'existe pas.  




**`<Champ Table hiérarchique> : (Nom de champ)`**

Nom du champ Table hiérarchique à manipuler. 

**`<Numéro de la ligne> : (Entier)`**

Numéro de la ligne pour laquelle le nombre de fils doit être calculé. 


<a name="SYNTAXE2"></a>

### Fils d'une ligne identifiée par son chemin

`<Résultat> = TableFilsOccurrence(<Champ Table hiérarchique> , <Nom de la branche>)`
---

**`<Résultat> : (Entier)`**

Nombre de fils de l'élément

**`<Champ Table hiérarchique> : (Nom de champ)`**

Nom du champ Table hiérarchique à manipuler. 

**`<Nom de la branche> : (Chaîne de caractères)`**

Nom de la branche pour laquelle le nombre de fils doit être calculé. Ce paramètre est de la forme : 

```txt
"<Nom de la racine>" + TAB + ["<Nom du 1er noeud>" + TAB + ...
["<Nom du 2ème noeud>" + TAB + [...]]]"<Nom de la feuille>"
```
Si ce paramètre ne correspond pas à une branche existante, une erreur WLangage est générée.


## PressePapierLargeur (Fonction)

***En anglais : ClipboardWidth***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Calcule la largeur de l'image (au format Bitmap) contenue dans le presse-papiers.

**Remarque** : Cette fonction peut être utilisée uniquement si le presse-papiers contient une image au format Bitmap.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI PressePapierFormat(ppfBitmap) = Vrai ALORS
	IMG_Image1..Largeur = PressePapierLargeur()
	IMG_Image1..Hauteur = PressePapierHauteur()
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = PressePapierLargeur()`
---

**`<Résultat> : (Entier)`**



- Largeur de l'image (exprimée en pixels), 

- -1 en cas d'erreur (le presse-papiers ne contient pas d'image).  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- La fonction [PressePapier](../WDLang1/3063004.md) lit l'information de type texte ou image contenue dans le presse-papiers.

- La fonction [VersPressePapier](../WDLang1/3063005.md) permet d'écrire dans le presse-papiers.

- Pour savoir si le presse-papiers contient une image, il est possible d'utiliser la fonction [PressePapierFormat](../WDLang1/3063002.md).

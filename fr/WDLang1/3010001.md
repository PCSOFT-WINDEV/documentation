
## Constantes diverses
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les constantes suivantes peuvent être utilisées dans les traitements sans être déclarées. Elles ont une valeur prédéfinie.


| Constante | Utilisation | Valeur |
| --- | --- | --- |
| **CRLF** | Passage à la ligne suivante<br><br> | Caract(13) + Caract(10) |
| **EOT** | Caractère présent à la fin des fichiers externes | Caract(4) |
| **ESC** | Caractère correspondant à la touche "Echap".<br><br> | Caract(27) |
| **ESP** | Espace insécable.<br><br> | Caract(160) |
| **Faux** | Booléen | 0 |
| **NULL** | Entier initialisé à 0. Par exemple, il est possible d'utiliser un paramètre "null" dans une fonction API (fonction [API](../WDLang1/3014005.md) ou [AppelDLL32](../WDLang1/3014003.md)) | 0 |
| **RC** | Caractère "Retour Chariot" : passage à la ligne suivante (utilisé dans les chaînes de caractères) | Caract(13) + Caract(10) |
| **TAB** | Caractère "Tabulation" (utilisé dans les chaînes de caractères) | Caract(9) |
| **ValPI** | Valeur approximative de Pi.<br><br> | 3,14159265358979323 |
| **Vrai** | Booléen | 1 |


## Variable Gesture
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000180"></a>
La variable Gesture permet de récupérer les paramètres de gesture (balayage) : direction, vitesse, distance, ...

Cette variable peut être utilisée notamment : 

- dans l'événement de balayage des éléments d'un champ Zone répétée, pour connaître le sens du balayage. 

- dans les événements du multitouch (zoom, scroll et balayage) pour éviter de déclarer des paramètres dans les événements. 






<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Variable Gesture
<a name="variable_gesture_ELTTEXTE000204"></a>
La variable Gesture est composée des éléments suivants :


|   |   |
| --- | --- |
|   | Réel<br><br>Angle (exprimé en degrés) d'azimut d'un Apple Pencil par rapport au repère du champ courant.<br><br>Cette variable est renseignée uniquement dans l'événement "Touché / Bouton gauche enfoncé". |
|   | Constante de type Entier<br><br>Direction de la gesture. Peut correspondre à une des constantes suivantes :<br><br>- *BasEnHaut* : Balayage de bas en haut.<br><br>- *DroiteAGauche* : Balayage de droite à gauche.<br><br>- *GaucheADroite* : Balayage de gauche à droite.<br><br>- *HautEnBas* : Balayage de haut en bas.<br><br><br>Cette variable est renseignée dans les événements de type Scroll et Balayage. |
|   | Dans le cas d'un scroll : valeur du déplacement (en pixels) dans la direction indiquée par Gesture.Direction.<br><br>Dans le cas d'un zoom au doigt : valeur de l'écartement (en pixels) entre les deux points de contact.<br><br>Cette variable est renseignée dans les événements de type Scroll et Zoom au doigt. |
|   | Réel<br><br>Angle (exprimé en degrés) correspondant à l'inclinaison d'un Apple Pencil par rapport au plan de la tablette.<br><br>Cette variable est renseignée uniquement dans l'événement "Touché / Bouton gauche enfoncé". |
|   | Réel<br><br>Pression exercée lors d'un appui.<br><br>Cette pression peut prendre une valeur comprise entre 0,0 et 1,0.<br><br>Cette variable est renseignée uniquement dans l'événement "Touché / Bouton gauche enfoncé". |
|   | Réel<br><br>Pression maximale pouvant être exercée lors d'un appui.<br><br>Cette pression peut prendre une valeur comprise entre 0,0 et 1,0.<br><br>Cette variable est renseignée uniquement dans l'événement "Touché / Bouton gauche enfoncé". |
|   | Entier<br><br>Vitesse de déplacement dans la direction indiquée par Gesture.Direction. <br>Cette vitesse est exprimée en pixels par seconde.<br><br>Cette variable est renseignée dans les événements de type Balayage. |

## Exemple
<a name="exemple_ELTTEXTE000228"></a>


### Récupération des informations de gesture
<a name="recuperation_des_informations_gesture_ELTPARAGRAPHE000129"></a>


```wl
// Evénement Scroll au doigt de IMG_Image
SELON Gesture.Direction
	CAS DroiteAGauche : SEL_Direction = 1
	CAS GaucheADroite : SEL_Direction = 2
	CAS HautEnBas : SEL_Direction = 3
	CAS BasEnHaut : SEL_Direction = 4
FIN

SAI_Vitesse = Gesture.Vitesse
SAI_Distance = Gesture.Distance
```

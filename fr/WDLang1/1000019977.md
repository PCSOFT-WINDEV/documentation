


## &lt;Organigramme&gt;.InfoXY (Fonction)

***En anglais : &lt;Organization Chart&gt;.InfoXY***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie l'indice de l'élément se trouvant à une position donnée dans le champ Organigramme.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Récupère l'indice d'un élément à la position du curseur de la souris dans le champ Organigramme
NumElement est un entier = ORGA_MonOrganigramme.InfoXY(SourisPosX(), SourisPosY())
SI NumElement <> -1 ALORS
	MaFenêtre.Titre = ORGA_MonOrganigramme[NumElement].Titre
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Champ Organigramme>.InfoXY(<X> , <Y>)`
---

**`<Résultat> : (Entier)`**



- Indice de l'élément trouvé. 

- -1 si aucun élément n'existe aux coordonnées fournies.




**`<Champ Organigramme> : (Nom de champ)`**

Nom du champ Organigramme à manipuler.

**`<X> : (Entier)`**

Coordonnée X (en pixels) à analyser.

**`<Y> : (Entier)`**

Coordonnée Y (en pixels) à analyser.

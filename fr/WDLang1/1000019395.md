
## ArrondiInférieur (Fonction)

***En anglais : RoundFloor***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie : 

- la valeur arrondie d'une valeur numérique à l'entier inférieur le plus proche.

- la durée arrondie à l'intervalle inférieur le plus proche. 
	

- la date et heure arrondies à l'intervalle inférieur le plus proche. 
	





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
ResArrondi = ArrondiInférieur(-16.238167,2) // Renvoie -16.24
ResArrondi = ArrondiInférieur(3.9) // Renvoie 3 (pas de décimales)
ResArrondi = ArrondiInférieur(3.999,2) // Renvoie 3.99
ResArrondi = ArrondiInférieur(1.9) // Renvoie 1
ResArrondi = ArrondiInférieur(-1.9) // Renvoie -2
```



```wl
// Arrondi d'une Durée
ResArrondiDurée est une durée
ResArrondiDurée = ArrondiInférieur(13 min, 5 min) // Renvoie 10 min (0001000000)
// La durée est découpée en périodes de 5 minutes. 
// Le résultat correspond au début de la période qui contient la durée 13 min.
```



```wl
// Arrondi d'une DateHeure
dhMonDateHeure est une DateHeure = "202211041524"
Trace("dhMonDateHeure : " + dhMonDateHeure)
Trace("ArrondiInférieur : " + ArrondiInférieur(dhMonDateHeure, 7 min)) 
// Affiche 20221104152400000

// La journée est découpée en périodes de 7 min. 
// Dans cet exemple, la variable dhMonDateHeure correspond à l'arrondi inférieur
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Calculer l'arrondi inférieur d'une valeur numérique

`<Résultat> = ArrondiInférieur(<Valeur numérique> [, <Nombre de décimales>])`
---

**`<Résultat> : (Entier ou réel)`**

Valeur arrondie au plus grand nombre inférieur. 

**`<Valeur numérique> : (Réel ou monétaire)`**

Valeur numérique à arrondir. 

**`<Nombre de décimales> : (Entier optionnel)`**

Nombre de décimales à retourner. Si ce paramètre n'est pas précisé, aucune décimale n'est retournée.


## GénèreMotDePasse (Fonction)

***En anglais : GeneratePassword***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Génère automatiquement un mot de passe avec des caractéristiques spécifiques (taille, caractères utilisés, ...). 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Génération d'un mot de passe de 8 caractères
sMdp est une chaîne = GénèreMotDePasse(8)

// Génération d'un mot de passe ne contenant que des voyelles
sMdp2 est une chaîne = GénèreMotDePasse(8, "aeiouyAEIOUY")

// Génération d'un mot de passe contenant tous les caractères de l'alphabet + les chiffres
MonJeu est une chaîne = "abcdefghijklmnopqrstuvwxyz0123456789"
sMdp3 est une chaîne = GénèreMotDePasse(8, MonJeu)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = GénèreMotDePasse(<Longueur> [, <Jeu de caractères>])`
---

**`<Résultat> : (Chaîne de caractères)`**



- Mot de passe généré. 

- Chaîne vide ("") si la longueur du mot de passe correspond à 0.




**`<Longueur> : (Entier)`**

Taille du mot de passe. Si ce paramètre vaut 0, le mot de passe correspond à une chaîne vide ("").

**`<Jeu de caractères> : (Chaîne de caractères optionnelle)`**

Caractères souhaités dans le mot de passe. 

Si ce paramètre n'est pas spécifié, le mot de passe généré est alors "prononçable". Ce mot de passe est composé de caractères : 

- compris entre a et z.

- compris entre A et Z. 

- compris entre 0 et 9.

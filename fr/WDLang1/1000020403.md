


## TimelineSupprimeEvénement (Fonction)

***En anglais : TimelineDeleteEvent***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime un événement dans un champ TimeLine. Si le champ TimeLine est lié à une source de données, l'événement est également supprimé dans la source de données. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Supprime l'événement sélectionné dans le champ TimeLine
TimeLineSupprimeEvénement(TL_TimeLine)

// Supprime le premier événement ajouté
TimeLineSupprimeEvénement(TL_TimeLine, 1)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Supprimer un événement identifié par son indice

`TimeLineSupprimeEvénement(<Champ TimeLine> [, <Indice de l'événement>])`
---

**`<Champ TimeLine> : (Nom de champ)`**

Nom du champ TimeLine à manipuler. 

**`<Indice de l'événement> : (Entier optionnel)`**

Indice de l'événement à supprimer. Le premier événement a pour indice 1. Si ce paramètre n'est pas précisé, l'événement sélectionné sera supprimé.


<a name="SYNTAXE2"></a>

### Supprimer un événement (variable de type EvénementTimeline)

`TimeLineSupprimeEvénement(<Champ TimeLine> , <Evénement>)`
---

**`<Champ TimeLine> : (Nom de champ)`**

Nom du champ TimeLine à manipuler. 

**`<Evénement> : (Variable de type EvénementTimeline)`**

Nom de la variable de type [EvénementTimeline](../WDLang1/1000020612.md) décrivant les caractéristiques de l'événement à supprimer.


<a name="SYNTAXE3"></a>

### Supprimer un ensemble d'événements 

`TimeLineSupprimeEvénement(<Champ TimeLine> , <Tableau d'événements>)`
---

**`<Champ TimeLine> : (Nom de champ)`**

Nom du champ TimeLine à manipuler. 

**`<Tableau d'événements> : (Tableau de variables de type EvénementTimeline)`**

Nom du tableau de variables de type [EvénementTimeline](../WDLang1/1000020612.md) décrivant les événements à supprimer.




## Fonctions multimédia (syntaxe préfixée)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions Multimédia (syntaxe préfixée) permettent de manipuler les fichiers multimédia associés aux champs Multimédia.



|   |   |
| --- | --- |
| [&lt;Multimédia&gt;.Arrêt](../WDLang1/1000019987.md) | Arrête la lecture en cours d'un fichier multimédia dans un champ Multimédia. |
| [&lt;Multimédia&gt;.Début](../WDLang1/1000019988.md) | Place au début du fichier multimédia joué dans le champ Multimédia. |
| [&lt;Multimédia&gt;.Durée](../WDLang1/1000019989.md) | Renvoie la taille du fichier multimédia en cours de lecture (en millisecondes) dans le champ Multimédia. |
| [&lt;Multimédia&gt;.Etat](../WDLang1/1000019990.md) | Indique l'état du fichier multimédia joué dans le champ Multimédia. |
| [&lt;Multimédia&gt;.Fin](../WDLang1/1000019991.md) | Place à la fin du fichier multimédia joué dans le champ Multimédia. |
| [&lt;Multimédia&gt;.Hauteur](../WDLang1/1000019999.md) | Renvoie la hauteur en pixels de l'image source du fichier multimédia joué dans le champ Multimédia. |
| [&lt;Multimédia&gt;.Joue](../WDLang1/1000020089.md) | Joue un fichier multimédia dans un champ Multimédia. |
| [&lt;Multimédia&gt;.Largeur](../WDLang1/1000020093.md) | Renvoie la largeur en pixels de l'image source du fichier multimédia joué dans le champ Multimédia. |
| [&lt;Multimédia&gt;.Pause](../WDLang1/1000020094.md) | Met en pause la lecture d'un fichier multimédia dans un champ Multimédia. |
| [&lt;Multimédia&gt;.Position](../WDLang1/1000020095.md) | Renvoie ou modifie la position en cours du fichier multimédia en cours dans le champ Multimédia. |
| [&lt;Multimédia&gt;.Volume](../WDLang1/1000020096.md) | Renvoie ou modifie le volume sonore du fichier multimédia joué dans un champ Multimédia. |

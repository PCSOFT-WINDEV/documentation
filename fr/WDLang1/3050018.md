
## Ln (Fonction)

***En anglais : Ln***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Calcule le logarithme népérien (inverse de l'exponentielle) d'une valeur numérique.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Calcul du logarithme népérien de 12
ResLogNépérien = Ln(12)
// Renvoie 2.48490664979
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = Ln(<Valeur numérique>)`
---

**`<Résultat> : (Réel)`**

Logarithme népérien recherché.

**`<Valeur numérique> : (Entier ou réel)`**

Valeur numérique à manipuler. Si ce paramètre est un nombre négatif ou nul, une erreur du WLangage est générée.


## LangueVersAlphabet (Fonction)

***En anglais : LanguageToCharset***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie l'alphabet correspondant à une langue.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connaître l'alphabet correspondant
// à la langue arménienne
ResAlphabet est un entier
ResAlphabet = LangueVersAlphabet(langueArménien) 
SELON ResAlphabet
	CAS alphabetArabe :
	 ...
	CAS alphabetChinois :
	 ...
	CAS alphabetEuropeEst :
	 ...
FIN
```


<a name="Exemple2"></a>



```wl
// Connaître l'alphabet correspondant
// à la langue chinoise à Taiwan
ResAlphabet est un entier
ResAlphabet = LangueVersAlphabet(langueChinois, 1)
SELON ResAlphabet
	CAS alphabetChinois :
	 ...
	CAS alphabetChinoisTraditionnel :
	 ...
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = LangueVersAlphabet(<Langue> [, <Sous-langue>])`
---

**`<Résultat> : (Constante de type Entier)`**

Constante identifiant l'alphabet de la langue spécifiée. Pour plus de détails sur ces constantes, consultez [Correspondance entre les langues, les sous-langues, les alphabets et les nations](../WDLang1/3054022.md).

**`<Langue> : (Constante de type Entier)`**

Langue dont on veut connaître l'alphabet. Pour plus de détails sur ces constantes, consultez [Correspondance entre les langues, les sous-langues, les alphabets et les nations](../WDLang1/3054022.md).

**`<Sous-langue> : (Entier optionnel)`**

Numéro de la sous-langue dont on veut connaître l'alphabet. Pour plus de détails sur ces sous-langues, consultez [Correspondance entre les langues, les sous-langues, les alphabets et les nations](../WDLang1/3054022.md).
Si ce paramètre n'est pas spécifié, l'alphabet standard de la &lt;Langue&gt; spécifiée est donné.

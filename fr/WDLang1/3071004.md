
## CurseurCapture (Fonction)

***En anglais : CursorCapture***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Dirige tous les événements associés à la souris vers une fenêtre spécifique ou un champ, quel que soit l'objet de l'écran où intervient l'événement souris.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple
<a class="notetitle" target="_blank" href="$DOC$=1000003071004&name=curseurcapture_fonction&product=WD">Voir des exemples supplémentaires</a>

```wl
// Tous les événement souris sont dirigés vers la fenêtre "FEN_NomFenêtre"
CurseurCapture(Vrai, Handle(FEN_NomFenêtre))
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = CurseurCapture([<Redirection>, ] <Identifiant de l'objet>)`
---

**`<Résultat> : (Entier)`**



- Identifiant (handle) de la fenêtre recevant précédemment tous les messages souris, 

- NULL si aucune fenêtre particulière ne recevait les événements souris (mode classique).




**`<Redirection> : (Booléen optionnel)`**

Indique le mode de gestion des événements souris :

- <u><u><u><u>Vrai</u></u></u></u> (valeur par défaut) : les messages sont dirigés vers une fenêtre ou un champ.

- <u><u><u><u>Faux</u></u></u></u> : gestion des événements classique.




**`<Identifiant de l'objet> : (Entier)`**

Numéro de "handle" de la fenêtre ou du champ. Cet identifiant peut être obtenu avec la fonction [Handle](../WDLang1/3015005.md).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
L'utilisation de la fonction **CurseurCapture** dans le code d'initialisation de la fenêtre n'a aucun effet.

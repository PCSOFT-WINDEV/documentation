
## AutoLanceAjoute (Fonction)

***En anglais : AutoRunAdd***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de lancer automatiquement une application WINDEV lors d'un événement précis. L'application à lancer peut correspondre ou non à l'application en cours.

**Remarque** : La fonction [AutoLanceSupprime](../WDLang1/3019001.md) permet d'annuler le lancement automatique d'une application.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Lancement automatique de l'application "MonAppli"
// au démarrage
AutoLanceAjoute(alDémarrage, "MonAppli", fRepExe() + "\GestionClients.exe")
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`AutoLanceAjoute(<Événement du lancement> , <Application> , <Exécutable>)`
---

**`<Événement du lancement> : (Constante de type entier)`**

Événement provoquant le lancement de l'application :


|   |   |
| --- | --- |
| *alDémarrage* | Lancement de l'application lors du démarrage du poste PC en cours. |



**`<Application> : (Chaîne de caractères)`**

Nom de l'application à lancer. Ce nom sera utilisé dans la fonction [AutoLanceSupprime](../WDLang1/3019001.md).

**`<Exécutable> : (Chaîne de caractères)`**

Nom et chemin complet de l'exécutable de l'application à lancer. Cet exécutable est présent sur le poste en cours.
Ce paramètre peut contenir la ligne de commande à passer à l'exécutable.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Nom et répertoire de l'exécutable
<a name="nom_repertoire_executable_ELTPARAGRAPHE000049"></a>

- La fonction [ExeInfo](../WDLang1/3035001.md), utilisée avec la constante *exeNom*, retourne le nom complet de l'exécutable de l'application en cours.

- La fonction [fRepExe](../WDLang1/3036049.md) renvoie le répertoire contenant l'exécutable de l'application en cours.



<a name="NOTE0_2"></a>


### Gestion des erreurs
<a name="gestion_des_erreurs_ELTPARAGRAPHE000065"></a>**Attention** : La fonction **AutoLanceAjoute** ne retourne pas de code d'erreur. Pour savoir si cette fonction a généré une erreur, utilisez la variable [ErreurDétectée](../WDLang1/3087001.md). En cas d'erreur, il est possible de connaître le détail de l'erreur grâce à la fonction [ErreurInfo](../WDLang1/3013008.md).

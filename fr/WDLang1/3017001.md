
## AnimationChangeType (Fonction)

***En anglais : AnimationSetMode***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Change le type d'animation :

- d'une image dans un champ Image.

- d'un texte dans un champ Libellé.




Les paramètres spécifiques au nouveau type peuvent être modifiés avec la fonction [AnimationChangeParamètre](../WDLang1/3017002.md).

![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Dans cette version, il est possible uniquement de changer le type d'animation d'un texte dans un champ Libellé.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Change l'animation du champ IMG_Image1
IMG_Image1.Animation = Vrai
AnimationChangeType(IMG_Image1, actDéfilement)
```


<a name="Exemple2"></a>



```wl
// Changement des paramètres d'animation d'un champ Libellé
IMG_Image1.Animation = Vrai
AnimationChangeType(LIB_MonLibellé, actClignotement)
AnimationChangeParamètre(LIB_MonLibellé, acpDurée, 100)
AnimationChangeParamètre(LIB_MonLibellé, acpCouleur, BleuFoncé)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`AnimationChangeType(<Champ manipulé> , <Type d'animation>)`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ Image ou Libellé à manipuler.

**`<Type d'animation> : (Constante ou chaîne de caractères)`**

Animation à réaliser.



|   |   |
| --- | --- |
| *actAnimationWinDev* (ou "Image à animer") | L'image est découpée en plusieurs dessins. Chaque dessin est affiché successivement, donnant une impression de vidéo. <br>Cette constante est utilisable uniquement avec les champs Image.<br>![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Cette constante n'est pas disponible. |
| *actClignotement* (ou "Clignotement") | Le libellé s'affiche alternativement dans sa couleur par défaut et dans sa couleur de clignotement. |
| *actDéfilement* (ou "Défilement") | L'image (ou le libellé) apparaît progressivement, affichée à partir de la droite, de la gauche, ...<br><br>![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Cette constante est utilisable uniquement avec les champs Libellé. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Changer le type de l'animation
<a name="changer_type_animation_ELTPARAGRAPHE000075"></a>Si le nouveau type correspond au type utilisé actuellement, les paramètres spécifiques à ce type sont ré-initialisés avec les paramètres par défaut.


<a name="NOTE0_2"></a>




### Arrêt ou reprise d'une animation
<a name="arret_reprise_une_animation_ELTPARAGRAPHE000084"></a>L'animation peut être activée ou désactivée grâce à la propriété [Animation](../Proprietes/2510006.md).


<a name="NOTE0_3"></a>
![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) 

### Images utilisant un format animé
<a name="images_utilisant_format_anime_ELTPARAGRAPHE000097"></a>La fonction **AnimationChangeType** n'a aucun effet sur les images utilisant un format animé. Ainsi, pour une image au format GIF animé, les paramètres sont lus dans le fichier et ne peuvent être modifiés.

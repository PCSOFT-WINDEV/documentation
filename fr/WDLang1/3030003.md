 

## DnDImpose (Fonction)

***En anglais : DnDForce***
	

<a name="XUtilisation"></a>
<a name="Utilisation"></a>
|  |   |
| --- | --- |



|  
 | ButIndiquer l'action exécutée par la source (glisser/déplacer).SyntaxeDnDImpose (&lt;Action&gt;)<br><br>&lt;Action&gt; est un entier qui peut prendre pour valeur :<br><br>- *dndCopier* (les données seront copiées).<br><br>- *dndDéplacer* (les données seront déplacées).<br><br>- *dndDéfaut* (les données seront déplacées ou copiées selon les manipulations).<br><br><br>Détail**DnDImpose** indique si le champ source permet :<br><br>- aucune action<br><br>- de déplacer les données (les données sont coupées de la source)<br><br>- de copier les données (les données sont copiées, elles restent dans la source)<br><br><br>Notes**DnDImpose** permet de forcer la copie ou le déplacer des données du champ source.<br><br>**DnDAccepte** indique l'action acceptée dans la cible.<br><br>Les variables suivantes peuvent être utilisées :<br><br>- **_DND.Action** pour connaître l'action du glisser/déplacer<br><br>- **_DND.ChampCible** pour connaître le nom du champ cible<br><br>- **_DND.ChampSource** pour connaître le nom du champ source dans la fenêtre<br><br>- **_DND.CtrlEnfoncé** pour connaître l'état de la touche contrôle Ctrl<br><br>- **_DND.FenSource** pour connaître le nom de la fenêtre source<br><br>- **_DND.SourisPosX** pour connaître l'abscisse de la souris lors de l'événement<br><br>- **_DND.SourisPosY** pour connaître l'ordonnée de la souris lors de l'événement<br><br><br><br><br> |
| --- | --- |




## DiagrammeVersImage (Fonction)

***En anglais : DiagramToImage***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Exporte un diagramme vers une une variable de type [Image](../WDLang1/1000019650.md). L'image générée peut ensuite être sauvegardée ou affichée dans un autre champ.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Enregistre un diagramme au format PNG
ImageDiag est une Image
ImageDiag = DiagrammeVersImage(EDIAG_MonDiagramme.Valeur)
SI ErreurDétectée ALORS 
	Erreur()
	RETOUR
FIN
dSauveImagePNG(ImageDiag, "exportDuDiagramme.png")
```

```wl
MonDiagramme est un Diagramme <- EDIAG_MonDiagramme

// Récupère l'image générée
MonImage est une Image <- DiagrammeVersImage(MonDiagramme)

// Ouvre l'image
OuvreDocument(MonImage)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = DiagrammeVersImage(<Diagramme> [, <Couleur de fond>])`
---

**`<Résultat> : (Variable de type Image)`**

Variable de type [Image](../WDLang1/1000019650.md) correspondant à l'image du diagramme. 

**`<Diagramme> : (Nom de champ ou variable de type Diagramme)`**

Diagramme à manipuler. Ce paramètre correspond : 

- au nom d'un champ Editeur de diagrammes. 

- au nom d'une variable de type [Diagramme](../WDLang1/1410088055.md). 




**`<Couleur de fond> : (Entier optionnel)`**

Couleur de fond de l'image. Cette couleur peut correspondre à :

- une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)),

- une couleur TSL (obtenue avec la fonction [TSL](../WDLang1/3029057.md)),

- [une couleur prédéfinie du WLangage](../WDLang5/3010002.md),

- une variable de type [Couleur](../WDLang1/1000019661.md). 


Si la couleur de fond n'est pas précisée, le diagramme sera exporté tel quel, avec la couleur de fond qui a éventuellement déjà été définie.

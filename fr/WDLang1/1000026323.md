


## Fonctions de gestion des champs d'affichage HTML (syntaxe préfixée)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions suivantes permettent de gérer le champ d'affichage HTML (syntaxe préfixée) : 




|   |   |
| --- | --- |
| [&lt;Champ d'affichage HTML&gt;.DepuisEmail](../WDLang1/1410087278.md) | Importe le contenu HTML d'un email et ses images dans un champ d'affichage HTML. Les images contenues dans l'email sont également affichées dans le champ. |
| [&lt;Champ d'affichage HTML&gt;.ExécuteProcédureNavigateur](../WDLang1/1000026265.md) | Exécute une procédure WLangage navigateur WEBDEV dans un champ d'affichage HTML avec une page créée par WEBDEV. |
| [&lt;Champ d'affichage HTML&gt;.ExécuteTraitementChamp](../WDLang1/1000026266.md) | Lance l'exécution d'un événement WLangage associé à un champ présent dans une page HTML affichée par une champ d'affichage HTML. |
| [&lt;Champ d'affichage HTML&gt;.ValeurChamp](../WDLang1/1000026268.md) | Permet de connaître ou modifier un champ d'un formulaire HTML présent dans un champ d'affichage HTML. |
| [&lt;Champ d'affichage HTML&gt;.ValideFormulaire](../WDLang1/1000026311.md) | Valide le formulaire d'une page HTML affichée dans un champ d'affichage HTML. |
| [&lt;Champ d'afichage HTML&gt;.SurErreurJS](../WDLang1/1000026267.md) | Indique la procédure à appeler en cas d'erreur dans l'exécution du Javascript dans un champ d'affichage HTML. |
| [&lt;Html&gt;.Navigue](../WDLang1/1000023633.md) | Lance une opération de navigation dans un champ d'affichage HTML. |

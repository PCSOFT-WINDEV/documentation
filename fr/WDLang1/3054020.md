
## LangueVersNom (Fonction)

***En anglais : LanguageToName***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le nom d'une langue.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connaître le nom de la langue arménienne
ResNom est une chaîne
ResNom = LangueVersNom(langueArménien)
```


<a name="Exemple2"></a>



```wl
// Connaître le nom de la langue anglaise en Australie
ResNom est une chaîne
ResNom = LangueVersNom(langueAnglais, 3)
// Renvoie "Australien"
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = LangueVersNom(<Langue> [, <Sous-langue>])`
---

**`<Résultat> : (Chaîne de caractères)`**



- Nom de la langue spécifiée, 

- Chaîne vide ("") si la langue spécifiée n'est pas connue.




**`<Langue> : (Constante de type Entier)`**

Langue dont on veut connaître le nom. Pour plus de détails sur ces constantes, consultez [Correspondance entre les langues, les sous-langues, les alphabets et les nations](../WDLang1/3054022.md).

**`<Sous-langue> : (Entier optionnel)`**

Sous-langue dont on veut connaître le nom. Pour plus de détails sur ces sous-langues, consultez [Correspondance entre les langues, les sous-langues, les alphabets et les nations](../WDLang1/3054022.md).
Si ce paramètre n'est pas spécifié, le nom standard de la &lt;Langue&gt; spécifiée est donnée.

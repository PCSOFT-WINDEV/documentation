## Présentation
<a name="presentation_ELTTEXTE000160"></a>
Lors de la saisie d'un nouveau message destiné à l'utilisateur final, WINDEV ou WINDEV Mobile affiche automatiquement la fenêtre suivante.<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Assistant_question.gif)<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Assistant_Saisie.gif)
Cette fenêtre permet de configurer :

1. L'image de la boîte de dialogue. Cette image peut être une image prédéfinie ou une image spécifique, accessible depuis votre poste.

2. Le texte de la question, dans les différentes langues du projet.

3. Les différents boutons affichés par la boîte de dialogue.

	- Le bouton "Ajouter" permet d'ajouter un bouton.

	- Le bouton "Supprimer" permet de supprimer le bouton sélectionné.

	- Le bouton "Rechercher" permet de rechercher une réponse parmi les boutons réponse déjà saisis.

	- Les boutons fléchés permettent de déplacer le bouton sélectionné.




4. Les différentes caractéristiques du bouton sélectionné :

	- Libellé du bouton dans les différentes langues : Par défaut, le libellé des boutons reprend automatiquement le verbe présent dans la question.

	- Valeur de retour. Cette valeur est utilisée pour savoir quelle action faire si le bouton est cliqué.

	- Bouton Validation : Le bouton validera la boîte de dialogue

	- Bouton Annulation : Le bouton annulera la boîte de dialogue.  




5. Lors de la saisie d'une interrogation directive, il est également possible de définir les caractéristiques du champ de saisie.

## Base des messages
<a name="base_des_messages_ELTTEXTE000214"></a>
WINDEV et WINDEV Mobile proposent une liste de messages par défaut. Ces messages ne peuvent pas être supprimés.

Lors de la création d'un message, ce message est automatiquement ajouté à la base de messages. Par défaut, la base de messages est présente dans le répertoire "Personnel\\Messages" de WINDEV ou WINDEV Mobile. 
Ce répertoire peut être modifié dans les options du produit utilisé : 

1. Sous le volet "Accueil", dans le groupe "Environnement", déroulez "Options" et sélectionnez "Options générales de xxx". 

2. Affichez l'onglet "Répertoires".




**Pour supprimer un message de la base de messages** :

1. Affichez l'éditeur de code.

2. Sous le volet "Code", dans le groupe "Langues", déroulez "Traduire les chaînes" et sélectionnez "Interrogations directives".

3. Sélectionnez le message à supprimer et cliquez sur ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=IconeSupprimerPropriete.gif).

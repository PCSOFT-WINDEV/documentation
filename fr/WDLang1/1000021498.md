| **tblrChargeDéfaut** | Mode de chargement du fichier XLSX : Le fichier XLSX chargé dans le champ Tableur est ouvert en écriture. | [TableurCharge](../WDLang1/1000021340.md) |
| **tblrChargeDétaché** | Mode de chargement du fichier XLSX : Le fichier XLSX chargé dans le champ Tableur est chargé en mémoire uniquement. Le fichier ".xlxs" peut être ouvert avec une autre application. | [TableurCharge](../WDLang1/1000021340.md) |
| **tblrChargeGardeMdp** | Mode de chargement du fichier XLSX : Le mot de passe du fichier XLSX chargé dans le champ Tableur est conservé en mémoire. | [TableurCharge](../WDLang1/1000021340.md) |
| **tblrCouleurFond** | Attribut à connaître/modifier : Couleur de fond du texte | [TableurStyleSélection](../WDLang1/1000021338.md) |
| **tblrCouleurTexte** | Attribut à connaître/modifier : Couleur du texte | [TableurStyleSélection](../WDLang1/1000021338.md) |
| **tblrEcrasementAutorisé** | Mode de sauvegarde du fichier : Si le fichier XLSX existe déjà et n'est pas ouvert (dans le champ Tableur ou une autre application), il est écrasé. | [TableurSauve](../WDLang1/1000021341.md) |
| **tblrfFormule** | Personnalisation de la barre de formule : Le champ permet la saisie de la formule pour la cellule en cours dans le champ Tableur. | [TableurChampFormule](../WDLang1/1000021647.md) |

| **tblrRechContient** | Option de recherche : Recherche de type "Contient". | [TableurCherche](../WDLang1/1000021610.md), [TableurRemplace](../WDLang1/1000023436.md) |

| **tblrRechSansCasse** | Option de recherche : Recherche sans prendre en compte la casse. | [TableurCherche](../WDLang1/1000021610.md), [TableurRemplace](../WDLang1/1000023436.md) |
| **tblrRechTouteFeuille** | Option de recherche : Recherche dans toutes les feuilles du classeur. | [TableurCherche](../WDLang1/1000021610.md), [TableurRemplace](../WDLang1/1000023436.md) |
| **tblrRetourALaLigneAutomatique** | Attribut à connaître/modifier : Gestion du retour à la ligne automatique. | [TableurStyleSélection](../WDLang1/1000021338.md) |
| **tblrSelMultiCell** | Type de sélection : Plusieurs cases sont sélectionnées. | Variable de type [tblrSélection](../WDLang1/1000023498.md) |
| **tblrSelMultiEntête** | Type de sélection : Plusieurs entêtes sont sélectionnés.  | Variable de type [tblrSélection](../WDLang1/1000023498.md) |

| **tblrSelSimpleEntête** | Type de sélection : Un seul entête est sélectionné.  | Variable de type [tblrSélection](../WDLang1/1000023498.md) |

| **tblrTaillePolice** | Attribut à connaître/modifier : Taille de la police | [TableurStyleSélection](../WDLang1/1000021338.md) |


## Tiff (Type de variable)

***En anglais : Tiff***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **Tiff** permet de définir toutes les caractéristiques avancées d'une image TIFF. Les caractéristiques de ce type peuvent être définies et modifiées à l'aide de différentes propriétés WLangage. 

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Chargement du fichier
MonTiff est un Tiff = "MonFichier.tiff"
// Remplacement de la première page
MonTiff.Image[1] = "C:\MesImages\sigle48.bmp"
// Sauvegarde du fichier
dSauveImageTiff(MonTiff, "MonFichierModifié.tiff")
```

<a name="NOTE0"></a>

## Remarques
<a name="NOTE0_1"></a>


### Propriétés spécifiques à la description des variables de type Tiff
<a name="proprietes_specifiques_description_des_variables_type_tiff_ELTPARAGRAPHE000028"></a>Les propriétés suivantes peuvent être utilisées pour manipuler une image au format Tiff :

| **Nom de la propriété** | **Type manipulé** | **Effet** |
| --- | --- | --- |
| Image | Tableau de [Image](../WDLang1/1000019650.md) | Tableau contenant les images de chaque page du fichier TIFF. |
| Valeur | Image (mémo) | Valeur de l'image. Cette propriété permet de charger une image Tiff à partir d'un buffer, d'un fichier ou d'un mémo. |

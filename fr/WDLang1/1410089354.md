
## Suisse.VérifieIDE (Fonction)

***En anglais : Switzerland.CheckUID***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Teste la validité d'un numéro IDE (numéro d'identification des entreprises en Suisse).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Test d'une variable
MonNuméroIDE est une chaîne = "CHE320488267"
Suisse.VérifieIDE(MonNuméroIDE)		// renvoie Vrai

// Test d'un champ
SAI_NuméroIDE = "ABC"
Suisse.VérifieIDE(SAI_NuméroIDE)	// renvoie Faux
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = Suisse.VérifieIDE(<Numéro à vérifier>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'élément spécifié contient un numéro IDE valide.

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Numéro à vérifier> : (Nom de champ ou variable de type chaîne)`**

Numéro IDE à vérifier. Ce paramètre peut correspondre :

- au nom d'une variable de type chaîne.

- au nom d'un champ.

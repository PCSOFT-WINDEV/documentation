
## EntierVersHexa (Fonction)

***En anglais : IntToHexa***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la chaîne hexadécimale d'une valeur numérique.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
EntierVersHexa(31) 		// Renvoie "0000001F"
EntierVersHexa(31, 2) 	// Renvoie "00 00 00 1F"
EntierVersHexa(-1, 2) 	// Renvoie "FF FF FF FF"
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = EntierVersHexa(<Valeur> [, <Taille des mots>])`
---

**`<Résultat> : (Chaîne de caractères)`**

Résultat de la conversion au format hexadécimal. 

**`<Valeur> : (Entier)`**

Nombre à convertir en chaîne de caractères.

**Remarque** : Les entiers sur 1 et 2 octets sont traités de la même façon que les entiers sur 4 octets. 

**`<Taille des mots> : (Entier optionnel)`**

Taille des mots séparés par des espaces. Un mot est un "groupe" de chiffres hexadécimaux. Par exemple, un octet est un mot de 2 chiffres hexadécimaux.

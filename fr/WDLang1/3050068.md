
## FinAmortissementDégressif (Fonction)

***En anglais : FinDecreasingRedemption***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Calcule la valeur de l'amortissement d'un bien pour une période spécifique, grâce à la formule de l'amortissement décroissant à taux fixe.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple
<a class="notetitle" target="_blank" href="$DOC$=1000003050068&name=finamortissementdegressif_fonction&product=WD">Voir des exemples supplémentaires</a>

```wl
// Valeur d'amortissement d'un bien acheté 25 000 Euros dont la valeur
// résiduelle sera de 15 000 Euros après une durée de 6 ans.
ResAmortissement = FinAmortissementDégressif(25000, 15000, 6)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = FinAmortissementDégressif(<Coût d'acquisition> , <Valeur résiduelle> , <Durée> [, <Période> [, <Nombre de mois pour l'année 1>]])`
---

**`<Résultat> : (Réel ou monétaire)`**

Valeur de l'amortissement pour la &lt;Période&gt; donnée ou 0 si l'amortissement est nul.

**`<Coût d'acquisition> : (Réel ou monétaire)`**

Coût d'acquisition du bien. Ce coût ne peut pas être nul.

**`<Valeur résiduelle> : (Réel ou monétaire)`**

Valeur résiduelle du bien au terme de la durée d'amortissement.

**`<Durée> : (Réel)`**

Nombre de périodes durant lesquelles le bien est amorti. Cette durée ne peut pas être nulle.

**`<Période> : (Entier optionnel)`**

Période pour laquelle l'amortissement doit être calculé (1 par défaut).

**`<Nombre de mois pour l'année 1> : (Entier optionnel)`**

Nombre de mois de la première année d'amortissement (1 par défaut). Ce nombre doit être compris entre 0 et 12.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Amortissement à taux fixe
<a name="amortissement_taux_fixe_ELTPARAGRAPHE000078"></a>La méthode de l'amortissement à taux fixe consiste à calculer l'amortissement d'un bien à l'aide d'un taux constant sur les différentes périodes d'amortissement.

Ce taux est appliqué pour chaque période à la valeur actuelle du bien. La valeur actuelle du bien correspond à :


```txt
<Coût d'acquisition> - <Total amortissement>
```


où &lt;Total amortissement&gt; représente les amortissements cumulés des périodes précédentes.

Ce taux est calculé en fonction de la valeur résiduelle, du coût d'acquisition du bien et de la durée de l'amortissement.

Le calcul de la valeur de l'amortissement pour la première année comme celui pour la dernière année est spécifique :

- pour la première année, seul le nombre de mois amortis est pris en compte (&lt;Nombre de mois pour l'année 1&gt;);

- pour la dernière année, le nombre de mois amortis correspond à :
	
	```txt
	12 - <Nombre de mois pour l'année 1>
	```





**Remarque** : Si le nombre de mois de la première année est égal à 12, cette année est comprise dans la durée.


<a name="NOTE0_2"></a>




### Gestion des erreurs
<a name="gestion_des_erreurs_ELTPARAGRAPHE000104"></a>**Attention** : La fonction **FinAmortissementDégressif** ne retourne pas de code d'erreur. Pour savoir si cette fonction a généré des erreurs, utilisez la fonction [FinErreur](../WDLang1/3050071.md). Le libellé de l'erreur peut être connu grâce à la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.




## &lt;Editeur HTML&gt;.InsèreTableau (Fonction)

***En anglais : &lt;HTML editor&gt;.InsertTable***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Insère un tableau à la position du curseur dans un champ Editeur HTML.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Insérer un tableau de 4 x 3
EDHTML_MonEditeurHTML.InsèreTableau(4, 3, "border=2")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Champ Editeur HTML>.InsèreTableau(<Nombre de colonnes> , <Nombre de lignes> [, <Style CSS>])`
---

**`<Champ Editeur HTML> : (Nom de champ)`**

Nom du champ Editeur HTML manipulé.

**`<Nombre de colonnes> : (Entier)`**

Nombre de colonnes du tableau à insérer.

**`<Nombre de lignes> : (Entier)`**

Nombre de lignes du tableau à insérer.

**`<Style CSS> : (Chaîne de caractères optionnelle)`**

Nom CSS du style à appliquer au tableau inséré. Ce paramètre peut également correspondre à un style intégré à une balise &lt;table&gt; (par exemple : "border=1").

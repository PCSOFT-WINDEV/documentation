
## docEntêtePiedDePage (Type de variable)

***En anglais : docHeaderFooter***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **docEntêtePiedDePage** permet de définir le texte des différents paragraphes présents dans un entête ou un pied de page. Les caractéristiques de cet entête ou pied de page peuvent être définies et modifiées à l'aide de différentes propriétés WLangage. 

**ATTENTION : Ce type ne doit pas être utilisé directement.** Il doit être utilisé uniquement via les variables de type [docSection](../WDLang1/1000022685.md). 

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonDoc est un Document
// Création de l'entête de page
s est une docSection = MonDoc.Section
s.Entête.PageImpaireHéritée = Vrai
d est un docFragment(s.Entête.PremièrePage, 1)
d.MiseEnForme.PoliceGras = Vrai
d.MiseEnForme.PoliceTaille = 24
d.Texte = "ENTETE"
MonPara est un docParagraphe = d.Paragraphe[1]
MonPara.MiseEnPage.Alignement = chCentre

//Insertion d'une image dans l'entête
d3 est un docFragment(s.Entête.PremièrePage, 1)
Img est une Image = fRepExe() + "\imgtest.jpg"
MonElementImage est un docElément
MonElementImage.Type = typeDocElémentImage
MonElementImage.Image.BufferImage = Img 
MonElementImage.Image.Largeur = 20
MonElementImage.Image.Hauteur = 20
DocInsère(d3, MonElementImage)


// Création d'un pied de page
sPied est un docSection = MonDoc.Section
dPied est un docFragment(sPied.PiedDePage.PagePaire, 1)
MonPara2 est un docParagraphe = dPied.Paragraphe[1]
MonPara2.MiseEnPage.Alignement = chDroite

// Insertion d'une formule dans le pied de page
MonElémentFormule est un docElément
MonElémentFormule.Type = typeDocElémentFormule
MonElémentFormule.Formule.Instruction = "PAGE \* Arabic \* MERGEFORMAT"
DocInsère(dPied, MonElémentFormule)

TT_SansNom1 = MonDoc
```

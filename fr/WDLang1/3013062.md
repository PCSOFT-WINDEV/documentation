
## ConstruitVariantAutomation (Fonction)

***En anglais : BuildAutomationVariant***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Construit un paramètre de type variant Automation. Ce paramètre peut être utilisé lors de l'appel à une méthode d'un objet Automation nécessitant un variant Automation spécifique.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
oObjet est un objet Automation "ObjetAutomation"
tabTableau est un tableau de 5 entiers
// Force le passage du tableau par un variant Automation 
// plutôt qu'un passage direct du tableau
oObjet>>MaMethode(ConstruitVariantAutomation(tabTableau))
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = ConstruitVariantAutomation(<Paramètre à convertir>)`
---

**`<Résultat> : (Variant Automation)`**

Valeur du paramètre convertie en variant Automation. Ce résultat doit être passé directement à une méthode d'un objet Automation. Ce résultat ne peut pas être stocké dans une variable du WLangage.

**`<Paramètre à convertir> : (Tout type)`**

Elément WLangage à convertir en variant automation (constante, variable, ...). Cet élément est converti dans un variant Automation. Ce variant automation contient l'équivalent Automation de l'élément WLangage.

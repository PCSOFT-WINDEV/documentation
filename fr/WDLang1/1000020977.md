
## DateHeureLocaleVersFuseau (Fonction)

***En anglais : DateTimeLocalToTimeZone***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Convertit une date et une heure locales en une date et une heure d'un fuseau horaire.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
dhLocale est un DateHeure = Maintenant()
dhNewYork est un DateHeure = DateHeureLocaleVersFuseau(dhLocale, "America/New_York")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = DateHeureLocaleVersFuseau(<DateHeure> , <Fuseau>)`
---

**`<Résultat> : (Chaîne de caractères)`**

Date et heure au format "AAAAMMJJHHmmSSCCC" exprimées dans le fuseau horaire spécifié.

**`<DateHeure> : (DateHeure ou chaîne de caractères au format "AAAAMMJJHHmmSSCCC")`**

Date et heure locales à convertir. 

**`<Fuseau> : (Chaîne de caractères)`**

Nom du fuseau de la base de données "tz" (ou base de données Olson) à utiliser pour convertir le paramètre &lt;DateHeure&gt;. 
Ce nom doit correspondre à un nom complet du fuseau définit par l'IANA Time Zone Database. Par exemple : "Europe/Paris", "Europe/London", "America/New_York", ...

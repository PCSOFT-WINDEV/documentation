
## gValeurMémoriséeActive (Fonction)

***En anglais : gStoredValueEnable***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Autorise ou non la récupération de la valeur définie par la fonction [gValeurMémorisée](../WDLang1/1410088363.md) dans un champ Liste ou dans un champ Combo.

Remarque : Cette fonction est identique à la fonction [gLienActive](../WDLang1/3029036.md). 






<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Le champ Combo "COMBO_Civilité" affiche les titres de civilité :
// "Monsieur", "Madame" et "Mademoiselle".
// Un entier représentant une civilité sera retourné au langage.
ListeAjoute(COMBO_Civilité, "Monsieur" + gValeurMémorisée("1"))
// Seule la chaîne de caractères "1" est retournée au langage.
// Cette chaîne de caractères retournée n'est pas affichée dans le champ Combo.
// Autoriser la récupération de la valeur définie par la fonction gValeurMémorisée.
gValeurMémoriséeActive(COMBO_Civilité, Vrai)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = gValeurMémoriséeActive(<Champ Liste ou Combo> [, <Active>])`
---

**`<Résultat> : (Booléen)`**

Correspond à l'ancien mode de gestion : 

- <u><u><u><u>Vrai</u></u></u></u> si la valeur définie par la fonction [gValeurMémorisée](../WDLang1/1410088363.md) a été récupérée (valeur par défaut), 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Champ Liste ou Combo> : (Nom de champ)`**

Nom du champ Liste ou Combo à manipuler.

**`<Active> : (Booléen optionnel)`**



- <u><u><u><u>Vrai</u></u></u></u> si la valeur définie par la fonction [gValeurMémorisée](../WDLang1/1410088363.md) doit être récupérée (valeur par défaut), 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Activation/Désactivation
<a name="activationdesactivation_ELTPARAGRAPHE000108"></a>

- Si la récupération des valeurs définies avec la fonction [gValeurMémorisée](../WDLang1/1410088363.md) est activée sur le champ COMBO_COMBO1 (&lt;Active&gt; = Vrai) :
	COMBO_COMBO1.Valeur renvoie le numéro d'ordre dans le champ COMBO_COMBO1.
	COMBO_COMBO1[COMBO_COMBO1] renvoie la valeur de la fonction [gValeurMémorisée](../WDLang1/1410088363.md).

	- Si le champ COMBO_COMBO1 est liée à une rubrique :

		- La fonction [FichierVersEcran](../WDLang4/3044210.md) sélectionne la ligne dont la valeur de la fonction [gValeurMémorisée](../WDLang1/1410088363.md) est celle de la rubrique liée.

		- La fonction [EcranVersFichier](../WDLang4/3044146.md) affecte la rubrique HFSQL liée avec la valeur de la fonction [gValeurMémorisée](../WDLang1/1410088363.md).




- Si la récupération des valeurs définies avec la fonction [gValeurMémorisée](../WDLang1/1410088363.md) est désactivée sur le champ COMBO_COMBO1 (&lt;Active&gt; = Faux) : 
	COMBO_COMBO1.Valeur renvoie le numéro d'ordre dans le champ COMBO_COMBO1.
	COMBO_COMBO1[COMBO_COMBO1] renvoie la valeur affichée en cours.

	- Si le champ COMBO_COMBO1 est lié à une rubrique :

		- La fonction [FichierVersEcran](../WDLang4/3044210.md) sélectionne la ligne dont le rang est égal à la rubrique liée.

		- La fonction [EcranVersFichier](../WDLang4/3044146.md) affecte la rubrique HFSQL liée avec le numéro de l'élément sélectionné, c'est-à-dire ListeSelect(COMBO_COMBO1).








<a name="NOTE0_2"></a>




### Fonction ListeCherche
<a name="fonction_listecherche_ELTPARAGRAPHE000175"></a>

- Si **gValeurMémoriséeActive**()=Vrai, la fonction [ListeCherche](../WDLang1/3049003.md) tient compte des valeurs ajoutées par la fonction [gValeurMémorisée](../WDLang1/1410088363.md).

- Si **gValeurMémoriséeActive**()=Faux, la fonction [ListeCherche](../WDLang1/3049003.md) ne tient pas compte des valeurs ajoutées par la fonction [gValeurMémorisée](../WDLang1/1410088363.md). La recherche est effectuée sur la valeur affichée. 



<a name="NOTE0_3"></a>




### Combos auto-alimentées
<a name="combos_autoalimentees_ELTPARAGRAPHE000202"></a>La fonction **gValeurMémoriséeActive** est utilisée pour gérer les champs de type "Combo auto-alimentée".

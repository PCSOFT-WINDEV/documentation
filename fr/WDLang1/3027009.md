
## EntierVersDate (Fonction)

***En anglais : IntegerToDate***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Transforme un entier en une date au format AAAAMMJJ.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
EntierVersDate(73773)       // Retourne "20011225"
EntierVersDate(73773, "A")  // Retourne "2001"
EntierVersDate(73773, "M")  // Retourne "12"
EntierVersDate(73773, "J")  // Retourne "25"

// Calcul de la date du lendemain
soit Demain = EntierVersDate(DateVersEntier(DateSys()) + 1)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = EntierVersDate(<Date> [, <Format>])`
---

**`<Résultat> : (Chaîne de caractères ou variable de type Date)`**

Date au format AAAAMMJJ.

**`<Date> : (Entier)`**

Nombre de jours écoulés depuis le 1er janvier 1800 + 1 (le 1er janvier 1800 porte le numéro 1).

**`<Format> : (Chaîne de caractères optionnelle)`**

Format de la date à récupérer :

- A pour récupérer l'année au format AAAA,

- M pour récupérer le mois au format MM,

- J pour récupérer le jour au format JJ.




Si ce paramètre n'est pas spécifié, le format utilisé par défaut est AAAAMMJJ (AAAA correspond à l'année, MM au mois et JJ au jour).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Format AAAAMMJJ
<a name="format_aaaammjj_ELTPARAGRAPHE000080"></a>Pour obtenir la date sous un format différent de AAAAMMJJ, il faut formater la chaîne avec les fonctions [DateVersChaîne](../WDLang1/3027025.md), [Droite](../WDLang1/3024016.md), [Gauche](../WDLang1/3024001.md), etc.

Le format AAAAMMJJ a été choisi car il permet de tester la chronologie de deux dates selon ce même format, alors que cela ne serait pas possible avec une date au format AAAAJJMM ou JJMMAAAA par exemple.

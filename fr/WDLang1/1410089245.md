
## Brésil.VérifieCPF (Fonction)

***En anglais : Brazil.CheckCPF***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Teste la validité d'un numéro CPF (numéro d'identification personnel au Brésil).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Test d'une variable
MonNuméroCPF est une chaîne = "16650440396"
Brésil.VérifieCPF(MonNuméroCPF)	// renvoie Vrai

// Test d'un champ
SAI_NuméroCPF = "ABC"
Brésil.VérifieCPF(SAI_NuméroCPF)	// renvoie Faux
```



```wl
// Test d'une variable
MonNuméroCPF est une chaîne
MonRésultat est un booléen
Saisie("Entrez le numéro CPF : ", MonNuméroCPF)
SI MonRésultat = Faux ALORS
	Info("Numéro CPF invalide")
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = Brésil.VérifieCPF(<Numéro à vérifier>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'élément spécifié contient un numéro CPF valide,

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Numéro à vérifier> : (Nom de champ ou variable de type chaîne)`**

Numéro CPF à vérifier. Ce paramètre peut correspondre : 

- au nom d'une variable de type chaîne.

- au nom d'un champ.

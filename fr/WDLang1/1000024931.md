
## PaysRécupère (Fonction)

***En anglais : CountryGet***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la variable de type Pays correspondant à un code ISO ou un libellé de pays à la norme ISO 3166-1.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Récupération des informations sur la France 
oFrance est un Pays = PaysRécupère("FR")
// Affichage du libellé
LIB_Pays = oFrance.Nom
// Affichage du préfixe de numéro de téléphone
SAI_Téléphone = oFrance.PréfixeNuméroDeTéléphone
// Affichage drapeau
IMG_Drapeau = oFrance.Drapeau
// Affichage du continent
LIB_Continent = oFrance.Continent.Nom
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = PaysRécupère([<Libellé ou code ISO du pays>])`
---

**`<Résultat> : (Variable de type Pays)`**

Variable de type [Pays](../WDLang1/1000024921.md) correspondant au libellé ou code ISO spécifié.

**`<Libellé ou code ISO du pays> : (Chaîne de caractères optionnelle)`**



- Libellé (en français, anglais ou espagnol) du pays recherché.

- Code ISO du pays recherché.


Si ce paramètre n'est pas précisé, le pays sera celui de la machine.

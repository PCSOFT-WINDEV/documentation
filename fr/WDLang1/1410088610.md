
## diagMiseEnPage (Type de variable)

***En anglais : diagLayout***
			
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **diagMiseEnPage** permet de définir toutes les caractéristiques de mise en page d'un diagramme. Les caractéristiques de cette mise en page peuvent être définies et modifiées à l'aide de différentes propriétés WLangage. **Cette mise en page est utilisée lors de l'impression du diagramme et l'affichage du quadrillage.**

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
monDiagramme est un Diagramme

// Mise en page pour du A4
maMiseEnPage est un diagMiseEnPage <- monDiagramme.MiseEnPage
maMiseEnPage.LargeurPage = 210.0
maMiseEnPage.HauteurPage = 297.0
```

<a name="PROPS"></a>

## Propriétés


### Propriétés spécifiques à la description des variables de type diagMiseEnPage
<a name="proprietes_specifiques_description_des_variables_type_diagmiseenpage_ELTPARAGRAPHE000033"></a>Les propriétés suivantes peuvent être utilisées pour manipuler la mise en page d'un diagramme :

| Nom de la propriété | Type manipulé | Effet |
| --- | --- | --- |
| HauteurPage | Réel | Hauteur de la page (en millimètres). |
| LargeurPage | Réel | Largeur de la page (en millimètres). |




<a name="NOTE0"></a>

## Remarques
<a name="NOTE0_1"></a>
Si les propriétés **HauteurPage** et **LargeurPage** correspondent à 0, une mise en page infinie est utilisée.

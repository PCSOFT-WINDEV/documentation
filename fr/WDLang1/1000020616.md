


## Fonctions de gestion des champs Sélecteur
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


Les fonctions permettant de gérer les champs Sélecteur sont les suivantes :



|   |   |
| --- | --- |
| [SélecteurAjoute](../WDLang1/1000020598.md) | Ajoute une nouvelle option dans un champ Sélecteur. |
| [SélecteurInsère](../WDLang1/1000020599.md) | Insère une nouvelle option dans un champ Sélecteur. |
| [SélecteurOccurrence](../WDLang1/1000021184.md) | Renvoie le nombre d'options présentes dans un champ Sélecteur. |
| [SélecteurSupprime](../WDLang1/1000020600.md) | Supprime une option d'un champ Sélecteur. |

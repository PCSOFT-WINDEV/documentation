
## HaspHLEcrit (Fonction)

***En anglais : HaspHLWrite***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ecrit des données (chaînes de caractères ou buffers) dans la mémoire interne de la clé Hasp.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Génère une clé initiale d'activation
CléInitiale est une chaîne = CléGénèreCléInitiale()

// Conserve la clé initiale d'activation dans la mémoire de la clé Hasp
HaspHLEcrit(SAI_ID_Session, haspFichierPrincipal, CléInitiale)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = HaspHLEcrit(<Identifiant de la session> , <Identifiant du fichier> , <Position> , <Chaîne à écrire>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'écriture a réussi, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. La fonction [ErreurInfo](../WDLang1/3013008.md) permet de connaître le détail de l'erreur.




**`<Identifiant de la session> : (Entier)`**

Identifiant de la session Hasp renvoyé par la fonction [HaspHLConnecte](../WDLang1/1000018891.md).

**`<Identifiant du fichier> : (Entier ou constante de type Entier)`**

Identifiant du fichier dans lequel les données doivent être écrites. Cet identifiant peut correspondre :

- à un entier. 

- à une des constantes suivantes :
	


|   |   |
| --- | --- |
| *haspFichierLicence* | Ecrit dans le fichier "zone de licence" de la clé. |
| *haspFichierPrincipal* | Ecrit dans le fichier "zone de mémoire principale" de la clé. |






**`<Position> : (Entier)`**

Position en octets à laquelle les données seront écrites dans le fichier.

**Attention** : Le début du fichier commence à la position 0.

**`<Chaîne à écrire> : (Chaîne de caractères ou Buffer)`**

Données à écrire dans le fichier.

**Attention** : La taille des fichiers dans les clés Hasp est fixe. Si la taille des données à écrire est plus grande que la taille disponible dans le fichier, la fonction échouera.

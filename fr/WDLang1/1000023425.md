
## HasardCryptoBuffer (Fonction)

***En anglais : RandomCryptoBuffer***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie un buffer de la taille spécifiée rempli avec des données aléatoires, utilisable avec les technologies cryptographiques.

Remarque : Le buffer renvoyé peut par exemple servir de clé pour les fonctions de cryptage. 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Génération d'une clé aléatoire de 32 octets (256 bits)
CléHasard est un Buffer = HasardCryptoBuffer(32)
// Cryptage avec la clé générée
MessageCrypé est un Buffer = CrypteStandard(MonMessage, CléHasard)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = HasardCryptoBuffer([<Taille du buffer>])`
---

**`<Résultat> : (Buffer)`**



- Buffer rempli avec des données aléatoires, 

- Buffer vide en cas d'erreur. 




**`<Taille du buffer> : (Entier optionnel)`**

Taille (en nombre d'octets) du buffer à initialiser (32 par défaut). 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Le premier appel à la fonction peut être long (de l'ordre de la seconde ou plus si la taille du buffer à initialiser est grande). En effet, l'initialisation du mécanisme aléatoire est très complexe. 

- La fonction [InitHasard](../WDLang1/3013030.md) n'a aucun effet sur le résultat de la fonction **HasardCryptoBuffer**.

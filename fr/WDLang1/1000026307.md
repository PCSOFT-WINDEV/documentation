


## &lt;Editeur HTML&gt;.Sélectionne (Fonction)

***En anglais : &lt;HTML editor&gt;.Select***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Sélectionne une portion de texte dans un champ Editeur HTML.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Passe les 5 premiers caractères en gras
EDHTML_MonEditeurHTML.Selectionne(1, 5)
EDHTML_MonEditeurHTML.FormateSelection(htmlGras)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Champ Editeur HTML>.Sélectionne(<Début de la sélection> , <Fin de la sélection>)`
---

**`<Champ Editeur HTML> : (Nom de champ)`**

Nom du champ Editeur HTML manipulé.

**`<Début de la sélection> : (Entier)`**

Premier caractère à sélectionner dans le champ Editeur HTML.

**`<Fin de la sélection> : (Entier)`**

Dernier caractère de la sélection.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- La sélection peut être récupérée par la fonction [&lt;Editeur HTML&gt;.RécupèreSélection](../WDLang1/1000026305.md). 

- Le format de la sélection peut ensuite être modifié par la fonction [&lt;Editeur HTML&gt;.FormateSelection](../WDLang1/1000026298.md).

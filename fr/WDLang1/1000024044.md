


## &lt;Saisie&gt;.InsèreJeton (Fonction)

***En anglais : &lt;Edit&gt;.InsertToken***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Insère un jeton dans un champ de saisie de type "Jetons de texte".
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Clic sur [Insérer le robot de test] 
SAI_Destinataire.InsèreJeton(1, "ROBOT", "robotmail@adresse.test.fr")
```

```wl
// Syntaxe utilisant le type Jeton
MonJeton est un Jeton
MonJeton.ValeurAffichée = "ROBOT"
MonJeton.ValeurMémorisée = "robotmail@adresse.test.fr"
SAI_Destinataire.InsèreJeton(1, MonJeton)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Insérer un jeton directement

`<Résultat> = <Champ de saisie>.InsèreJeton(<Position> , <Valeur> [, <Libellé>])`
---

**`<Résultat> : (Entier)`**



- Indice effectif du jeton inséré, 

- 0 en cas d'erreur :

	- Si le champ de saisie n'est pas un champ de type "Jetons de texte" (propriété [JetonActif](../Proprietes/1000023782.md) vaut <u><u><u><u>Faux</u></u></u></u>). 

	- Si un doublon a été détecté et si la propriété [JetonAutoriseDoublon](../Proprietes/1000023784.md) vaut <u><u><u><u>Faux</u></u></u></u>.







**`<Champ de saisie> : (Nom de champ)`**

Nom du champ de saisie de type "Jetons de texte".

**`<Position> : (Entier)`**

Indice d'insertion du jeton. 

- Si &lt;Position&gt; est supérieur au nombre de jetons dans le champ, le jeton est ajouté à la fin des jetons existants (équivalent à la fonction [&lt;Saisie&gt;.AjouteJeton](../WDLang1/1000024043.md)).

- Si &lt;Position&gt; vaut 0, le jeton est inséré en première position. 

- Si &lt;Position&gt; vaut -1, &lt;Résultat&gt; vaut 0.




**`<Valeur> : (Chaîne de caractères)`**

Valeur du jeton.

**`<Libellé> : (Chaîne de caractères optionnelle)`**

Libellé du jeton. Si cette valeur n'est pas précisée, elle correspondra à la valeur du jeton.


<a name="SYNTAXE2"></a>

### Insérer un jeton via le type Jeton

`<Résultat> = <Champ de saisie>.InsèreJeton(<Position> , <Jeton>)`
---

**`<Résultat> : (Entier)`**



- Indice effectif du jeton inséré, 

- 0 en cas d'erreur :

	- Si le champ de saisie n'est pas un champ de type "Jetons de texte" (propriété [JetonActif](../Proprietes/1000023782.md) vaut <u><u><u><u>Faux</u></u></u></u>). 

	- Si un doublon a été détecté et si la propriété [JetonAutoriseDoublon](../Proprietes/1000023784.md) vaut <u><u><u><u>Faux</u></u></u></u>.







**`<Champ de saisie> : (Nom de champ)`**

Nom du champ de saisie de type "Jetons de texte".

**`<Position> : (Entier)`**

Indice d'insertion du jeton. 

- Si &lt;Position&gt; est supérieur au nombre de jetons dans le champ, le jeton est ajouté à la fin des jetons existants (équivalent à la fonction [&lt;Saisie&gt;.AjouteJeton](../WDLang1/1000024043.md)).

- Si &lt;Position&gt; vaut 0, le jeton est inséré en première position. 

- Si &lt;Position&gt; vaut -1, &lt;Résultat&gt; vaut 0.




**`<Jeton> : (Variable de type Jeton)`**

Nom de la variable de type [Jeton](../WDLang1/1000023942.md) décrivant le jeton à ajouter.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Pour ajouter un jeton à la position du curseur de saisie, utilisez la fonction [&lt;Saisie&gt;.AjouteJeton](../WDLang1/1000024043.md). 

- La propriété [ValeurMémorisée](../Proprietes/2510127.md) du champ de saisie renvoie les valeurs mémorisées des jetons du champ séparées par le caractère &lt;RC&gt; (Retour Chariot).




## SysIconeAfficheBulle (Fonction)

***En anglais : SysIconDisplayTooltip***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Affiche une bulle pour une icône présente dans la barre des tâches. Cette icône doit avoir préalablement été ajoutée dans la barre des tâches avec la fonction [SysIconeAjoute](../WDLang1/3073001.md).

Remarque : A partir de Windows 10, la bulle est affichée sous forme d'une nouvelle notification par le centre de notifications intégré à Windows.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Prévient de la fin du traitement
SysIconeAfficheBulle("Le traitement est terminé.")
```

```wl
// Affichage d'un avertissement avec une procédure de clic
SysIconeAfficheBulle("Le traitement est relativement long. " + ...
	"Cliquez sur la bulle pour voir le détails des opérations.", ...
	sysTypeBulleAvertissement, "", ProcédureDétail)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = SysIconeAfficheBulle(<Message> [, <Titre> [, <Type> [, <Icône> [, <Procédure WLangage>]]]])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la bulle a été affichée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Message> : (Chaîne de caractères)`**

Texte affiché dans la bulle (maximum 255 caractères).

**`<Titre> : (Chaîne de caractères optionnelle)`**

Titre de la bulle (maximum 63 caractères).

**`<Type> : (Constante optionnelle de type Entier)`**

Type de la bulle : 



|   |   |
| --- | --- |
| *sysTypeBulleAvertissement* | Bulle de type Avertissement. Une icône d'avertissement est affichée dans la bulle (panneau jaune). |
| *sysTypeBulleDéfaut*<br>(Valeur par défaut) | Bulle affichée par défaut (c'est-à-dire la bulle définie par la fonction [SysIconeAjoute](../WDLang1/3073001.md)). |
| *sysTypeBulleErreur* | Bulle de type Erreur. Une icône d'erreur est affichée dans la bulle (croix blanche sur fond rouge). |
| *sysTypeBulleInfo* | Bulle de type Info. Une icône d'information est affichée dans la bulle ("i" blanc sur fond bleu). |
| *sysTypeBullePerso* | Bulle personnalisée : le paramètre &lt;Icône&gt; permet de définir l'icône affichée dans la bulle. |





**`<Icône> : (Chaîne de caractères optionnelle)`**

Icône 16x16 affichée dans la bulle si le paramètre &lt;Type&gt; vaut *sysTypeBullePerso*. 
Correspond :

- soit au nom et chemin complet (ou relatif) du fichier icône à ajouter. Un chemin de type UNC est autorisé.

- soit un nom d'une variable de type Icône.


Par défaut, ce paramètre vaut chaîne vide (""). 

**`<Procédure WLangage> : (Nom de procédure optionnel)`**

Procédure WLangage appelée lors du clic sur la bulle (procédure sans paramètre). Ce paramètre est pris en compte uniquement à partir de Windows XP.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Lors du clic sur la bulle, la bulle se ferme.

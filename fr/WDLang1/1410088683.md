
## PicCaptureLasso (Fonction)

***En anglais : PicRectangularSnip***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Effectue une capture d'écran via le lasso et récupère cette capture dans une variable de type [Image](../WDLang1/1000019650.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MaCapture est une Image
MaCapture = PicCaptureLasso(Vrai)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = PicCaptureLasso([<Iconise>])`
---

**`<Résultat> : (Variable de type Image)`**

Variable de type [Image](../WDLang1/1000019650.md) contenant le résultat de la capture effectuée. 

**`<Iconise> : (Booléen optionnel)`**



- <u><u><u><u>Vrai</u></u></u></u> pour iconiser l'application lors de la capture, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.

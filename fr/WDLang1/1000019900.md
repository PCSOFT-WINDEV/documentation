
## RelanceProgramme (Fonction)

***En anglais : RestartProgram***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Termine l'application et la relance. 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Relance l'application
RelanceProgramme()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`RelanceProgramme([<Exécution du code>])`
---

**`<Exécution du code> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> pour exécuter le code de terminaison du projet. 

- <u><u><u><u>Faux</u></u></u></u> (valeur par défaut) pour ne pas exécuter le code de fermeture du projet.

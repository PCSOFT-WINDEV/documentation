
## OrthoVérifie (Fonction)

***En anglais : SpellCheck***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Indique si un mot est correctement orthographié (c'est-à-dire s'il est connu du dictionnaire OpenOffice installé sur le poste).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI OrthoDisponible() = Vrai ET OrthoVérifie(Mot) = Faux ALORS
	Erreur("Mot Incorrect")
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = OrthoVérifie(<Mot à vérifier>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> dans les cas suivants :  

	- le mot à vérifier est trouvé dans le dictionnaire OpenOffice,

	- le mot à vérifier a été ajouté avec la fonction [OrthoAjouteDictionnaire](../WDLang1/1000018921.md), 

	- le dictionnaire OpenOffice n'est pas installé. Il est possible de savoir si le correcteur orthographique est installé grâce à la fonction [OrthoDisponible](../WDLang1/1000019019.md). 




- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. La fonction [ErreurInfo](../WDLang1/3013008.md) permet de connaître le détail de l'erreur.




**`<Mot à vérifier> : (Chaîne de caractères)`**

Mot à rechercher dans le dictionnaire de OpenOffice.

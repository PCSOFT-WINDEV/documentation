
## CurseurAffiche (Fonction)

***En anglais : CursorDisplay***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Affiche ou masque le curseur de la souris. 

Remarque : Le caret de la souris (utilisé lors de la saisie) n'est pas modifié.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// -- Evénement "Entrée" associé au champ de saisie
// On masque le curseur de la souris (la flèche par défaut)
// lorsque que le champ de saisie a le focus
CurseurAffiche(Faux)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`CurseurAffiche(<Curseur affiché>)`
---

**`<Curseur affiché> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le curseur de la souris doit être affiché, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- La fonction **CurseurAffiche** n'a aucun effet si aucune souris n'est installée.

- Seul le curseur de l'application en cours est affecté par cette fonction : les fenêtres des autres applications conservent l'état d'affichage de leur curseur.

- Le curseur, même caché, reste actif : il est toujours possible de cliquer et de déplacer la souris.


## TélémétrieEstActive (Fonction)

***En anglais : TelemetryIsEnabled***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de savoir si la télémétrie est activée. 

<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI TélémétrieEstActive() = Faux ALORS TélémétrieActive()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = TélémétrieEstActive()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la télémétrie est activée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.

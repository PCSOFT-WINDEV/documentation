


## TCDValeurEntête (Fonction)

***En anglais : PVTHeaderValue***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la valeur affichée dans un entête du champ Tableau croisé dynamique.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
i est entier = 1
s est chaîne = tcdValeurEntête(TCD_Stat, "DTCD_Famille", i)
TANTQUE s <> "" 
	Trace(s)
	s = TCDValeurEntête(TCD_Stat, "DTCD_Famille", i)
	i++
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = TCDValeurEntête(<Champ Tableau croisé dynamique> , <Entête> , <Indice>)`
---

**`<Résultat> : (Chaîne de caractères)`**



- Valeur de l'entête spécifié.

- Chaîne vide ("") si l'indice spécifié est invalide. 




**`<Champ Tableau croisé dynamique> : (Nom de champ)`**

Nom du champ Tableau croisé dynamique à manipuler

**`<Entête> : (Chaîne de caractères)`**

Nom d'un entête du champ Tableau croisé dynamique.

**`<Indice> : (Entier)`**

Indice de la valeur à récupérer.

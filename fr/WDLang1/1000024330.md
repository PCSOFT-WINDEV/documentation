


## &lt;Image&gt;.ModifieLuminosité (Fonction)

***En anglais : &lt;Image&gt;.ModifyLightness***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie la luminosité d'une image contenue dans un champ Image.

![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Les dessins](../WDLang1/3029035.md). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Change la luminosité de l'image contenue dans le champ IMG_Image1
IMG_Image1.ModifieLuminosité(50)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Champ Image>.ModifieLuminosité(<Variation de Luminosité> [, <Image destination>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le changement a été effectué, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Champ Image> : (Nom de champ)`**

Nom du champ Image à manipuler.

**`<Variation de Luminosité> : (Entier)`**

Valeur comprise entre -100 et +100. Cette valeur sera ajoutée à la luminosité actuelle de chaque pixel pour obtenir une nouvelle luminosité.

**`<Image destination> : (Chaîne de caractères optionnelle ou variable de type Image)`**

Image dans laquelle l'image transformée sera dessinée. Cette image peut correspondre : 

- au nom d'un champ Image. 

- au nom d'une variable de type [Image](../WDLang1/1000019650.md).
	




Si ce paramètre n'est pas spécifié, la transformation s'effectuera sur &lt;Champ Image&gt;.

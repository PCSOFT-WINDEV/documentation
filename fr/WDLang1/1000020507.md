


## TCDCharge (Fonction)

***En anglais : PVTLoad***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Charge le résultat du calcul d'un champ Tableau croisé dynamique depuis une sauvegarde (fichier présent sur le disque ou sauvegarde dans un buffer). Ce calcul a été précédemment sauvé grâce à la fonction [TCDSauve](../WDLang1/1000020471.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
sFichier est une chaîne
// Construit le nom du fichier
sFichier = SysRep(srAppData) + ["\"] + "StatsParPays.tcd"

// Le fichier existe ?
SI fFichierExiste(sFichier) ALORS
	// Oui, on le charge 
	TCDCharge(TCD_Statistiques, sFichier)
	SI OuiNon("Voulez-vous tout recalculer ?") ALORS TCDCalculeTout(TCD_Statistiques)
SINON
	// Le fichier n'existe pas, on le calcule
	TCDCalculeTout(TCD_Statistiques)
FIN
TCDSauve(TCD_Statistiques, sFichier)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = TCDCharge(<Champ Tableau croisé dynamique> , <Support de sauvegarde> [, <Mot de passe>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le contenu du champ Tableau croisé dynamique a été chargé. 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire




**`<Champ Tableau croisé dynamique> : (Nom de champ)`**

Nom du champ Tableau croisé dynamique à manipuler.

**`<Support de sauvegarde> : (Chaîne de caractères ou Buffer)`**



- Sauvegarde dans un fichier : Nom et chemin complet du fichier à charger. Ce fichier correspond à un fichier précédemment sauvé avec la fonction [TCDSauve](../WDLang1/1000020471.md).

- Sauvegarde en mémoire : Buffer de sauvegarde renvoyé par la fonction [TCDSauve](../WDLang1/1000020471.md). 




**`<Mot de passe> : (Chaîne de caractères optionnelle)`**

Mot de passe associé à la sauvegarde du champ Tableau croisé dynamique. Ce mot de passe a été spécifié lors de la sauvegarde du champ Tableau croisé dynamique avec la fonction [TCDSauve](../WDLang1/1000020471.md).

## Remarques


### Sauvegarde et chargement du contenu d'un champ Tableau croisé dynamique
<a name="sauvegarde_chargement_contenu_champ_tableau_croise_dynamique_ELTPARAGRAPHE000080"></a>Lorsque le champ Tableau croisé dynamique est calculé avec la fonction [TCDCalculeTout](../WDLang1/1000020539.md), le résultat peut être sauvé avec la fonction [TCDSauve](../WDLang1/1000020471.md). Il est ensuite possible de le recharger avec la fonction **TCDCharge** : dans ce cas, le contenu du champ Tableau croisé dynamique n'est pas recalculé.


## EnumèreMenu (Fonction)

***En anglais : EnumMenu***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie :

- le nom de la nième option ou sous-option du menu principal de la fenêtre. 

- le nom de la nième option ou sous-option d'un menu contextuel personnalisé (créé avec WINDEV ou WINDEV Mobile).




Utilisée dans une boucle, cette fonction permet d'énumérer les options de menu d'une fenêtre, comme les sous-options d'une option de menu.






<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
i est un entier = 1
NomMenu est une chaîne
NomMenu = EnumèreMenu(MaFenêtre, 1) // Lecture du 1er menu de la fenêtre
TANTQUE NomMenu <> ""
	Trace(NomMenu)
	j est un entier = 1
	NomSousMenu est une chaîne
	NomSousMenu = EnumèreMenu(NomMenu, j) // Lecture de la 1ère option du menu
	TANTQUE NomSousMenu <> ""
			Trace(" " + NomSousMenu)
			j++ // Option suivante
			NomSousMenu = EnumèreMenu(NomMenu, j)
	FIN
	i++ // Menu suivant
	NomMenu = EnumèreMenu(MaFenêtre, i)
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = EnumèreMenu(<Objet Parent> , <Numéro de l'option>)`
---

**`<Résultat> : (Chaîne de caractères)`**



- Nom de l'option, 

- Chaîne vide ("") si &lt;Objet Parent&gt; correspond à une option de menu ou si l'énumération est terminée. 


Le nom de l'option a été défini sous l'éditeur de fenêtres lors de la création du menu.

**`<Objet Parent> : (Chaîne de caractères)`**



- Nom de la fenêtre contenant les menus à lister. Cette fenêtre doit être ouverte lors de l'utilisation de cette fonction.

- Nom du menu dont les options doivent être listées.




**`<Numéro de l'option> : (Entier)`**

Numéro de l'option dont le nom est recherché.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Une erreur du WLangage est affichée dans les cas suivants :

- &lt;Numéro de l'option&gt; est un entier négatif. 

- &lt;Numéro de l'option&gt; est supérieur au nombre d'options du menu + 1. 

- &lt;Objet Parent&gt; n'est ni une fenêtre, ni un menu, ni une option de menu. 

- &lt;Objet Parent&gt; est une fenêtre non ouverte. Pour lister un menu d'une fenêtre non ouverte, utilisez la fonction [EnumèreSousElément](../WDLang1/3064002.md).

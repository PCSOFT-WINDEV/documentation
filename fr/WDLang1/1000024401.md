


## &lt;Traitement de texte&gt;.VersImage (Fonction)

***En anglais : &lt;Word Processing&gt;.ToImage***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Exporte une page d'un champ Traitement de texte sous forme d'une image.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
d est une Image = TT_MonDocument.VersImage(1)
dSauveImageJPEG(d, "test")
LanceAppliAssociée("test.jpg")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Champ Traitement de texte>.VersImage(<Page>)`
---

**`<Résultat> : (Variable de type Image)`**

Nom de la variable de type [Image](../WDLang1/1000019650.md) correspondant à la page du document Traitement de texte.

**`<Champ Traitement de texte> : (Nom de champ)`**

Nom du champ Traitement de texte à manipuler.

**`<Page> : (Entier)`**

Numéro de la page à exporter sous forme d'une image.

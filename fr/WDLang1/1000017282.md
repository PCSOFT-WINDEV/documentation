


## CalendrierSelectOccurrence (Fonction)

***En anglais : CalendarSelectCount***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le nombre de jours sélectionnés dans un champ Calendrier.  








<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Parcours des dates sélectionnées dans le champ Calendrier
POUR i = 1 _A_ CalendrierSelectOccurrence(CAL_Calendrier1)
	Trace(CalendrierSelect(CAL_Calendrier1, i))
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = CalendrierSelectOccurrence(<Champ Calendrier>)`
---

**`<Résultat> : (Entier)`**

Nombre d'éléments sélectionnés dans le champ Calendrier. Ce paramètre :

- peut être supérieur à 1 uniquement si le champ Calendrier est multisélections,

- est égal à 1 si le champ Calendrier n'est pas multisélections.




**`<Champ Calendrier> : (Nom de champ)`**

Nom du champ Calendrier manipulé.

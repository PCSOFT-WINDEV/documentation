


## EnModeService (Fonction)

***En anglais : InServiceMode***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Identifie si le code est exécuté dans un service. 


<a name="sample_code"></a>

## Exemple
<a class="notetitle" target="_blank" href="$DOC$=1000020701&name=enmodeservice_fonction&product=WD">Voir des exemples supplémentaires</a>
<a name="Exemple1"></a>

```wl
// Code exécuté dans un service ? 
SI EnModeService() = Vrai ALORS
	// Traitement à effectuer dans le service
	...
SINON
	// Traitement à effectuer dans une application normale
	...
FIN
```
<a name="Exemple2"></a>

```wl
// Tentative d'exécuter l'exécutable d'un service
SI EnModeService() = Faux ALORS Erreur("Ce programme est un service," + ...
		" merci de l'installer par la procédure adéquate")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = EnModeService()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le code est exécuté dans un service, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Disponibilité de la fonction
<a name="disponibilite_fonction_ELTPARAGRAPHE000217"></a>La fonction **EnModeService** est disponible dans tous les produits pour permettre l'écriture d'un code multi-produits. Des fonctions ou procédures utilisant la fonction **EnModeService** peuvent être partagées entre plusieurs produits, sans que des erreurs de compilation soient générées.
<a name="NOTE0_2"></a>


### Exécution des traitements contenant des erreurs de compilation
<a name="execution_des_traitements_contenant_des_erreurs_compilation_ELTPARAGRAPHE000233"></a>Par défaut, le mécanisme de sécurité du WLangage se déclenche uniquement si la ligne contenant l'erreur de compilation est exécutée.

Par exemple, si une fonction interdite dans le produit en cours est présente dans une boucle SI, le mécanisme de sécurité du WLangage se déclenchera uniquement si la ligne contenant cette fonction s'exécute.


## FenEdSelectMoins (Fonction)

***En anglais : WinEdSelectMinus***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Dé-sélectionne un champ lors de la modification de l'interface par l'utilisateur final (mode "Modification de l'UI par l'utilisateur final" activé). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Désélectionne le champ en cours
X est un Champ
X <- FenEdSelect(FEN_Formulaire) 
FenEdSelectMoins(FEN_Formulaire, X)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`FenEdSelectMoins(<Fenêtre> , <Champ manipulé>)`
---

**`<Fenêtre> : (Nom de fenêtre)`**

Nom de la fenêtre ou de la fenêtre interne à manipuler. Cette fenêtre doit être en mode modification (fonction [FenEdActive](../WDLang1/1000021843.md)).  

**`<Champ manipulé> : (Champ optionnel)`**

Champ à dé-sélectionner dans la fenêtre. Le champ sélectionné peut être connu avec la fonction [FenEdSelect](../WDLang1/1000021871.md). 

- Si la sélection multiple est autorisée, le champ spécifié est enlevé de la sélection. 

- Si le champ spécifié n'est pas sélectionné, la fonction n'a aucun effet. 




Si ce paramètre n'est pas spécifié : 

- si la sélection multiple est autorisée, tous les champs de la fenêtre sont désélectionnés.

- si la sélection multiple n'est pas autorisée, désélectionne le champ sélectionné.

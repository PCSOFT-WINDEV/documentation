
## VersPressePapier (Fonction)

***En anglais : ToClipboard***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Écrit des informations de type texte ou image dans le presse-papiers du système.

Pour écrire des informations de type Texte RTF, utilisez la fonction [VersPressePapierRTF](../WDLang1/1000017199.md).

![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Il est possible uniquement d'écrire des informations de type texte dans le presse-papiers de Windows.










<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
Texte est une chaîne
Res est un entier
// Le contenu du champ est copié dans le presse-papiers
Texte = SAI_SAISIE1
Res = VersPressePapier(Texte)
SI Res = Faux ALORS
	Erreur("Erreur lors de l'écriture dans le presse-papiers")
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Écrire du texte dans le presse-papiers

`<Résultat> = VersPressePapier(<Texte>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le texte a été copié dans le presse-papiers, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Texte> : (Chaîne de caractères)`**

Texte à écrire dans le presse-papiers. 


<a name="SYNTAXE2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) 
### Copier une image dans le presse-papiers

`<Résultat> = VersPressePapier(<Image> , <Transparence>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'image a été copiée dans le presse-papiers, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Image> : (Chaîne de caractères ou variable de type Image)`**



- Nom du champ Image à copier dans le presse-papiers. Si le champ Image n'appartient pas à la fenêtre en cours, il est possible d'utiliser la notation &lt;Nom Fenêtre&gt;.&lt;Nom champ Image&gt;.  

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Nom de la variable de type [Image](../WDLang1/1000019650.md) à copier dans le presse-papiers. 




**`<Transparence> : (Booléen)`**

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 

- <u><u><u><u>Vrai</u></u></u></u> (valeur par défaut) pour conserver la transparence de l'image (si possible) lors de la copie vers le presse-papiers, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- La fonction **VersPressePapier** peut être utilisée pour gérer le couper/copier/coller d'informations de type texte ou de type image (BMP).

- La fonction [PressePapier](../WDLang1/3063004.md) permet de lire le contenu du presse-papiers. Les informations autres que celles de type texte ou image (binaire, etc.) contenues dans le presse-papiers ne peuvent pas être lues par la fonction [PressePapier](../WDLang1/3063004.md).

- La gestion d'informations de type texte au format RTF est réalisée avec les fonctions [VersPressePapierRTF](../WDLang1/1000017199.md) et [PressePapierRTF](../WDLang1/1000017200.md).


## EtatsUnis.VérifieEIN (Fonction)

***En anglais : UnitedStates.CheckEIN***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Teste la validité d'un numéro EIN (numéro d'identification d'entreprise aux Etats-Unis).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Test d'une variable
MonNuméroEIN est une chaîne = "322124444"
EtatsUnis.VérifieEIN(MonNuméroEIN)	// renvoie Vrai

// Test d'un champ
SAI_NuméroEIN = "ABC"
EtatsUnis.VérifieEIN(SAI_NuméroEIN)	// renvoie Faux
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = EtatsUnis.VérifieEIN(<Numéro à vérifier>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'élément spécifié contient un numéro EIN valide.

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Numéro à vérifier> : (Nom de champ ou variable de type chaîne)`**

Numéro EIN à vérifier. Ce paramètre peut correspondre : 

- au nom d'une variable de type chaîne.

- au nom d'un champ.


## FenIntHaut (Fonction)

***En anglais : WinInHeight***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la hauteur interne d'une fenêtre (sans la barre de titre ni le cadre).

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=FenExtInt.gif)

<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Hauteur interne de la fenêtre "FEN_SaisieClient"
ResHauteurInterne = FenIntHaut(FEN_SaisieClient)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = FenIntHaut([<Fenêtre>])`
---

**`<Résultat> : (Entier)`**

Hauteur interne de la fenêtre (en pixels).

**`<Fenêtre> : (Nom de fenêtre optionnel)`**

Nom ou alias de la fenêtre à manipuler. Si ce paramètre n'est pas précisé ou correspond à une chaîne vide (""), la hauteur interne de la fenêtre en cours est retournée.




## EditeurHTMLInsère (Fonction)

***En anglais : HTMLEditorInsert***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Insère du texte ou du code HTML à la position du curseur dans un champ Editeur HTML. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Insère la date du jour en gras
EditeurHTMLInsère(EDHTML_MonEditeurHTML, "<B>" + DateVersChaîne(DateSys()) + "</B>")

// Insère une émoticône sourire
EditeurHTMLInsère(EDHTML_MonEditeurHTML, "<p style=""font-size:64px"">&#128512;</p>")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`EditeurHTMLInsère(<Champ Editeur HTML> , <HTML à insérer>)`
---

**`<Champ Editeur HTML> : (Nom de champ)`**

Nom du champ Editeur HTML manipulé. 

**`<HTML à insérer> : (Chaîne de caractères)`**

Texte ou code HTML à insérer à la position courante du curseur. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Si une sélection est en cours dans le champ Editeur HTML, cette sélection est supprimée. 

- L'insertion peut être annulée par l'utilisateur en annulant l'action (undo) ou via la combinaison de touches Ctrl + Z. 

- La position courante d'insertion peut être connue par la propriété [Sélection](../Proprietes/1000023033.md) ou la propriété [Curseur](../Proprietes/2510025.md).

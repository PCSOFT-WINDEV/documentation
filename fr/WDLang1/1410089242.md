
## Espagne.VérifieDNI (Fonction)

***En anglais : Spain.CheckDNI***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Teste la validité d'un numéro DNI (numéro d'identification personnel en Espagne).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Test d'une variable
MonNuméroDNI est une chaîne = "12345678Z"
Espagne.VérifieDNI(MonNuméroDNI)	// renvoie Vrai

// Test d'un champ
SAI_NuméroDNI = "ABC"
Espagne.VérifieDNI(SAI_NuméroDNI)	// renvoie Faux
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = Espagne.VérifieDNI(<Numéro à vérifier>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'élément spécifié contient un numéro DNI valide.

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Numéro à vérifier> : (Nom de champ ou variable de type chaîne)`**

Numéro DNI à vérifier. Ce paramètre peut correspondre : 

- au nom d'une variable de type chaîne.

- au nom d'un champ.

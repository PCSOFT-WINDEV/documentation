


## GanttOrganiseTâche (Fonction)

***En anglais : GanttOrganizeTask***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Réorganise les tâches d'un diagramme de Gantt selon les contraintes (dépendances et ressources). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Réorganise les tâches
GanttOrganiseTâche(COL_Gantt)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`GanttOrganiseTâche(<Colonne Gantt>)`
---

**`<Colonne Gantt> : (Nom de champ)`**

Nom de la colonne (d'un champ Table ou Table hiérarchique) de type Diagramme de Gantt à manipuler.

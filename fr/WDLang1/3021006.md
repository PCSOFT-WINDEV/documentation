
## Message (Fonction)

***En anglais : Message***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Affiche (ou efface) un message dans la barre de message de la fenêtre ou du navigateur en cours.

**Remarque** : La fenêtre ou la fenêtre du navigateur doit posséder une barre de message.

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Si la barre de message est divisée en différentes cases, il est possible de spécifier la case dans laquelle le message est affiché.






<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
Message("Traitement en cours...")
MonTraitement() // Procédure permettant de réaliser le traitement voulu
Message("Prêt")
```




<a name="Exemple2"></a>


![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) 
```wl
// Message dans la case "CBM_InfoUtil"
Message(CBM_InfoUtil, "Saisissez votre nom")
```


<a name="Exemple3"></a>

## Syntaxe

### Afficher ou effacer un message dans la barre de message simple

`Message([<Texte>])`
---

**`<Texte> : (Chaîne de caractères optionnelle)`**

Texte à afficher (127 caractères maximum).
Si ce paramètre n'est pas précisé, le message précédemment affiché dans la barre de message est effacé.
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Si la barre de message est divisée en différentes cases, ce texte sera affiché dans la première case.



<a name="SYNTAXE2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) 
### Afficher ou effacer un message dans une case de la barre de message

`Message(<Nom de la case> , <Texte>)`
---

**`<Nom de la case> : (Chaîne de caractères)`**

Nom de la case à manipuler. Si la case a été créée :

- sous l'éditeur de fenêtres, le nom de la case peut être indiqué sans guillemets.

- avec la fonction [MessageAjouteCase](../WDLang1/3021001.md), le nom de la case doit être indiqué entre guillemets (il est également possible d'utiliser le mot-clé [Externe](../Motscles/1511013.md)).




**`<Texte> : (Chaîne de caractères)`**

Texte à afficher (127 caractères maximum).
Si ce paramètre n'est pas précisé, le message précédemment affiché dans la barre de message est effacé.

## Remarques


### Caractéristiques du message
<a name="caracteristiques_message_ELTPARAGRAPHE000120"></a>Le message est cadré à gauche.

Le texte affiché dans la barre de message simple ou dans la première case d'une barre de message multi-cases est automatiquement effacé lorsque l'utilisateur sélectionne un champ associé à un message d'aide : le message d'aide de ce champ s'affichera dans la barre de message.


<a name="NOTE0_2"></a>




### Erreur générée
<a name="erreur_generee_ELTPARAGRAPHE000131"></a>Une erreur WLangage est générée si la fonction **Message** est appelée depuis un traitement du "Code du projet" (initialisation du projet, fermeture du projet ou fermeture de session).


<a name="NOTE0_3"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) 

### Message et fenêtre Fille MDI
<a name="message_fenetre_fille_mdi_ELTPARAGRAPHE000144"></a>La fonction **Message** utilisée depuis une fenêtre fille MDI sans barre de message utilise la barre de message de la fenêtre mère.

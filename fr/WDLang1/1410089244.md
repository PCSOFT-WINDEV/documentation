
## Allemagne.VérifieIdNr (Fonction)

***En anglais : Germany.CheckIdNr***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Teste la validité d'un numéro IdNr (numéro d'identification personnel en Allemagne).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Test d'une variable
MonNuméroIdNr est une chaîne = "36574261809"
Allemagne.VérifieIdNr(MonNuméroIdNr)	// renvoie Vrai

// Test d'un champ
SAI_NuméroIdNr = "ABC"
Allemagne.VérifieIdNr(SAI_NuméroIdNr)	// renvoie Faux
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = Allemagne.VérifieIdNr(<Numéro à vérifier>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'élément spécifié contient un numéro IdNr valide.

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Numéro à vérifier> : (Nom de champ ou variable de type chaîne)`**

Numéro IdNr à vérifier. Ce paramètre peut correspondre : 

- au nom d'une variable de type chaîne.

- au nom d'un champ.

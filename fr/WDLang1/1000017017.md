
## SysNomEcranPrincipal (Fonction)

***En anglais : SysNameMainScreen***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le nom de l'écran principal de l'ordinateur.  
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affiche le nom de l'écran principal
Info(SysNomEcranPrincipal()) 
// Affiche par exemple : \\.\DISPLAY2\Monitor2 Écran Plug-and-Play
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = SysNomEcranPrincipal()`
---

**`<Résultat> : (Chaîne de caractères)`**



- Nom de l'écran principal, 

- Chaîne vide ("") en cas d'erreur.




## &lt;Caméra&gt;.Arrête (Fonction)

***En anglais : &lt;Camera&gt;.Stop***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Arrête l'affichage ou la sauvegarde d'une séquence vidéo dans un champ Caméra.






<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Arrête l'affichage de la séquence vidéo actuellement retransmise
// dans le champ Caméra "CAM_Cuisine"
ResArrêt = CAM_Cuisine.Arrête(viArrêteAffichage)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Champ Caméra>.Arrête([<Action à arrêter>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'affichage et/ou la sauvegarde dans le champ Caméra est correctement arrêtée, 

- <u><u><u><u>Faux</u></u></u></u> en cas d'erreur. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Champ Caméra> : (Nom de champ)`**

Nom du champ Caméra à manipuler.

**`<Action à arrêter> : (Constante optionnelle de type entier)`**

Type d'action à arrêter :


|   |   |
| --- | --- |
| *viArrêteAffichage* | Arrête l'affichage de la vidéo. Le champ Caméra spécifié devient vide. |
| *viArrêteCapture* | Arrête l'enregistrement de la séquence vidéo. Le fichier généré est disponible. |
| *viArrêteTout*<br>(Valeur par défaut) | Arrête l'affichage et l'enregistrement de la séquence vidéo. Le fichier généré est disponible. Le champ Caméra spécifié devient vide. |

## Remarques


### Conditions d'utilisation
<a name="conditions_utilisation_ELTPARAGRAPHE000229"></a>

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png) Pour utiliser cette fonction, une "caméra" doit être connectée au poste en cours.



<a name="NOTE0_2"></a>


### Cas particuliers
<a name="cas_particuliers_ELTPARAGRAPHE000251"></a>

- Pour sauvegarder la vidéo actuellement retransmise par une "web caméra", utilisez la fonction [&lt;Caméra&gt;.Capture](../WDLang1/1000022382.md).

- Pour afficher dans un champ Caméra la séquence vidéo actuellement retransmise par une "web caméra", utilisez la fonction [&lt;Caméra&gt;.Affiche](../WDLang1/1000022380.md).

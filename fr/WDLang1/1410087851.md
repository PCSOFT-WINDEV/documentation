
## Procédure WLangage appelée par la fonction CaméraPhoto



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Procédure WLangage (également nommée "callback") appelée lorsque la photo est enregistrée via la fonction [CaméraPhoto](../WDLang1/1410087225.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
CaméraPhoto(CAM_MaCaméra, CaméraPhoto_Callback)

PROCÉDURE INTERNE CaméraPhoto_Callback(bRéussite est un booléen, CheminPhoto est une chaîne)
	SI bRéussite = Vrai ALORS
		IMG_PHOTO = CheminPhoto
		LIB_CHEMIN_PHOTO = CheminPhoto
		MaFenêtre.Plan = 3
	SINON
		ToastAffiche("Erreur lors de la prise de photo." + RC + ErreurInfo())
	FIN
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`CaméraPhoto_Callback(<Réussite> , <Chemin de la photo>)`
---

**`<Réussite> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la photo a été enregistrée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Dans ce cas, la fonction [ErreurInfo](../WDLang1/3013008.md) permet d'obtenir le détail de l'erreur. 




**`<Chemin de la photo> : (Chaîne de caractères)`**

Chemin de la photo qui vient d'être enregistrée.


## AppliListeVersion (Fonction)

***En anglais : AppListVersion***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Liste les versions disponibles sur le serveur d'installation de référence. Ces versions sont disponibles dans l'historique de l'installation de référence. 

**Remarque** : Cette fonction n'est utilisable qu'avec une installation réseau sur un réseau local. Elle ne peut être employée avec une installation par IP.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
Res est une chaîne = AppliListeVersion()
IV est un InfoVersion

AfficheInfo est une chaîne = [
Numéro = %1
Date = %2
Taille = %3 octets
Commentaire = %4
Actif = %5
Forcé = %6
Interdite = %7
]

POUR TOUTE CHAINE MaChaîne DE Res SEPAREE PAR RC
	IV = AppliInfoVersion(MaChaîne)
	Trace(ChaîneConstruit(AfficheInfo, ...
		IV.NuméroVersion, IV.Date, ...
		IV.Taille, IV.Commentaire, ...
		IV.Activé, IV.Forcé,IV.Interdit))
	Trace("===============================================")
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = AppliListeVersion([<Chemin de l'application>])`
---

**`<Résultat> : (Chaîne de caractères)`**

Liste des versions disponibles séparées par des RC (Retour Chariot). Cette liste est de la forme : 

```txt
<Numéro de version 1> + RC + <Numéro de version 2> + RC + ... 
+ RC + <Numéro de version N>
```


**`<Chemin de l'application> : (Chaîne de caractères optionnelle)`**

Chemin complet de l'exécutable à manipuler. Si ce paramètre n'est pas précisé, l'application manipulée est l'application en cours.

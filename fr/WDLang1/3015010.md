
## FinEvénement (Fonction)

***En anglais : EndEvent***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Annule l'interception d'un événement Windows sur un champ ou une fenêtre WINDEV (mis en place grâce à la fonction [Evénement](../WDLang1/3015011.md)). La procédure associée à un événement Windows ne sera plus appelée.

**Remarque** : L'interception des événements est automatiquement arrêtée lorsque la fenêtre associée est fermée.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Capture du clic souris sur un champ de saisie
WM_LBUTTONDOWN est un entier = 513
IDEve est entier
IDEve = Evénement("ClicSouris", "SAI_SAISIE1", WM_LBUTTONDOWN)
...
// Arrêt de l'interception de WM_LBUTTONDOWN sur le champ "SAI_SAISIE1"
FinEvénement(IDEve)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = FinEvénement(<Numéro de l'événement>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'interception de l'événement a été annulée, 

- <u><u><u><u>Faux</u></u></u></u> en cas de problème (numéro d'événement incorrect par exemple).




**`<Numéro de l'événement> : (Entier)`**

Numéro de l'événement à annuler (renvoyé par la fonction [Evénement](../WDLang1/3015011.md)).

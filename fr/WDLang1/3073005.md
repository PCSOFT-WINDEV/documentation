
## SysFenActive (Fonction)

***En anglais : SysWinActive***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le handle de la fenêtre active ou active une fenêtre. Cette fenêtre n'a pas été créée par WINDEV.

**Remarques** :

- Pour connaître le handle d'une fenêtre WINDEV, utilisez la fonction [Handle](../WDLang1/3015005.md).

- Pour activer une fenêtre WINDEV, utilisez la fonction [DonneFocusEtRetourUtilisateur](../WDLang1/1410088107.md).



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Identifier la fenêtre active
ResActive = SysFenActive()
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Identifier la fenêtre active

`<Résultat> = SysFenActive()`
---

**`<Résultat> : (Entier)`**



- Handle de la fenêtre active.

- Constante *NULL* en cas d'erreur.  





<a name="SYNTAXE2"></a>

### Activer une fenêtre

`<Résultat> = SysFenActive(<Handle de la fenêtre>)`
---

**`<Résultat> : (Entier)`**



- Handle de la fenêtre active précédente. 

- Constante *NULL* en cas d'erreur.




**`<Handle de la fenêtre> : (Entier)`**

Handle de la fenêtre à activer. Ce handle peut être obtenu avec la fonction [SysFenHandle](../WDLang1/3073007.md).

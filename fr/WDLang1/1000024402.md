


## &lt;Traitement de texte&gt;.VersPDF (Fonction)

***En anglais : &lt;Word Processing&gt;.ToPDF***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Convertit un document de type Traitement de texte en un fichier PDF. 

<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Export en PDF du contenu d'un champ Traitement de texte
// Code de clic du bouton "Export PDF" 
TT_MonDoc.VersPDF(fRepExe() + [fSep] + "tt.pdf")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Champ Traitement de texte>.VersPDF(<Nom du fichier PDF>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le fichier PDF a été créé, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Champ Traitement de texte> : (Nom de champ)`**

Nom du champ Traitement de texte à manipuler.

**`<Nom du fichier PDF> : (Chaîne de caractères)`**

Nom et chemin complet du fichier PDF à créer. Si le fichier existe déjà, il est remplacé par le nouveau fichier.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Les polices "otf" (OpenType) ne sont pas gérées par la génération PDF. 

- Les filigranes ne sont pas gérés par la génération PDF.

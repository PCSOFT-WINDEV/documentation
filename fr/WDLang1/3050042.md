
## FinIntérêtPériode (Fonction)

***En anglais : FinPeriodInterest***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Évalue pour une période donnée le montant des intérêts dus pour un emprunt remboursé par versements périodiques constants avec un taux d'intérêt constant.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple
<a class="notetitle" target="_blank" href="$DOC$=1000003050042&name=fininteretperiode_fonction&product=WD">Voir des exemples supplémentaires</a>

```wl
// Montant des intérêts dus pour la deuxième année d'un emprunt.
// Cet emprunt est de 25 000 Euros avec un taux d'intérêt de 5% sur 6 ans.
ResIntérêtPériode = FinIntérêtPériode(25000, 5, 2, 6)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = FinIntérêtPériode([<Capital> [, <Taux> [, <Période> [, <Durée>]]]])`
---

**`<Résultat> : (Réel ou monétaire)`**

Montant des intérêts recherchés. Ce montant peut être :

- négatif : un paiement (ou décaissement) doit alors être effectué;

- positif : un encaissement doit alors être effectué.




**`<Capital> : (Réel ou monétaire optionnel)`**

Montant de l'emprunt (0 par défaut).

**`<Taux> : (Réel optionnel)`**

Taux d'emprunt exprimé en pourcentage (1 par défaut). Par exemple, pour un taux d'emprunt de 5%, ce paramètre sera égal à 5.

**`<Période> : (Entier optionnel)`**

Période pour laquelle le montant des intérêts doit être calculé (1 par défaut).

**`<Durée> : (Réel optionnel)`**

Nombre de périodes de l'emprunt (1 par défaut).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Unité temporelle
<a name="unite_temporelle_ELTPARAGRAPHE000079"></a>Les calculs peuvent être réalisés en prenant en compte des mois ou des années. Il suffit que les paramètres &lt;Taux&gt;, &lt;Durée&gt; et &lt;Période&gt; correspondent à la même unité temporelle.

Ainsi, pour obtenir le montant des intérêts dus au cinquième mois d'un emprunt de 3 ans, avec un taux d'intérêt annuel de 5%, il est possible d'utiliser :

- soit des unités exprimées en mois (pour un remboursement mensuel par exemple) :

	- &lt;Durée&gt; = 3 x 12 mois = 36

	- &lt;Période&gt; = 5

	- &lt;Taux&gt; = 5% / 12




- soit des unités exprimées en années (pour un remboursement annuel par exemple) :

	- &lt;Durée&gt; = 3 ans

	- &lt;Période&gt; = 5/36

	- &lt;Taux&gt; = 5%






<a name="NOTE0_2"></a>




### Gestion des erreurs
<a name="gestion_des_erreurs_ELTPARAGRAPHE000100"></a>**Attention** : La fonction **FinIntérêtPériode** ne retourne pas de code d'erreur. Pour savoir si cette fonction a généré des erreurs, utilisez la fonction [FinErreur](../WDLang1/3050071.md). Le libellé de l'erreur peut être connu grâce à la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.

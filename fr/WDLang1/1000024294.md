


## &lt;Table&gt;.Fin (Fonction)

***En anglais : &lt;Table&gt;.End***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Arrête le parcours manuel du fichier de données lié à un champ Table fichier. Après l'exécution de la fonction **&lt;Table&gt;.Fin**, les enregistrements situés après l'enregistrement en cours dans le champ Table ne seront pas lus.

<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Arrêt du parcours du fichier lié au champ Table "TABLE_TableProduit"
TABLE_TableProduit.Fin()
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Champ Table>.Fin()`
---

**`<Champ Table> : (Nom de champ)`**

Nom du champ Table à manipuler.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Conditions d'utilisation
<a name="conditions_utilisation_ELTPARAGRAPHE000194"></a>

- La fonction **&lt;Table&gt;.Fin** peut être utilisée indifféremment sur un champ Table monosélection ou multisélection.

- La fonction **&lt;Table&gt;.Fin** ne doit pas être utilisée sur les champs Table fichier utilisant un parcours automatique.





<a name="NOTE0_2"></a>


### Evénements du champ Table
<a name="evenements_champ_table_ELTPARAGRAPHE000215"></a>La fonction **&lt;Table&gt;.Fin** peut être utilisée uniquement dans les événements du champ Table suivants :

- Initialisation.

- Lecture de l'enregistrement précédent.

- Lecture de l'enregistrement suivant.



<a name="NOTE0_3"></a>


### Réinitialisation du champ Table
<a name="reinitialisation_champ_table_ELTPARAGRAPHE000239"></a>Après l'appel de la fonction **&lt;Table&gt;.Fin** dans un événement d'un champ Table, ré-initialisez l'affichage des données du champ Table. Par exemple :

```wl
TABLE_TABLE1.Affiche(taInit)
```

<a name="NOTE0_4"></a>


### Visualisation des enregistrements entre deux valeurs
<a name="visualisation_des_enregistrements_entre_deux_valeurs_ELTPARAGRAPHE000258"></a>Pour visualiser dans un champ Table les enregistrements correspondant à un intervalle de valeurs, il est conseillé d'utiliser la fonction [&lt;Source&gt;.Filtre](../WDLang4/1000024960.md) (et non la fonction **&lt;Table&gt;.Fin**).




## VidéoListeParamètre (Fonction)

***En anglais : VideoListParameter***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Liste les valeurs supportées par la caméra de l'appareil pour un paramètre donné.


	<B><B><B>Avertissement</B></B></B>

	 cette fonctionnalité n'est plus disponible



**Remarques** : 

- Pour modifier ou connaître la valeur actuelle d'un paramètre de la caméra de l'appareil, utilisez la fonction [VidéoParamètre](../WDLang1/1000019694.md).

- La fonction **VidéoListeParamètre** peut être utilisée dans l'émulateur Android mais pas dans le simulateur.





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Utilisation de la résolution maximale pour prendre une photo
sListeRésolution est une chaîne 
sListeRésolution = VidéoListeParamètre(vipRésolutionPhoto)
SI ErreurDétectée = Faux ALORS
	VidéoParamètre(vipRésolutionPhoto, ExtraîtChaîne(sListeRésolution, rangDernier, RC))
SINON
	Erreur("Erreur durant la récupération des résolutions supportées par la caméra.")
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = VidéoListeParamètre(<Paramètre>)`
---

**`<Résultat> : (Chaîne de caractères)`**



- Liste des valeurs supportées par la caméra pour le &lt;Paramètre&gt; spécifié. Cette liste est de la forme : 
	
	```txt
	<Valeur 1>+ RC + <Valeur 2> + RC + ... + <Valeur N>
	```


- Chaîne vide ("") si le paramètre n'est pas supporté par la caméra ou en cas d'erreur lors de la récupération des paramètres.
	En cas d'erreur, la variable [ErreurDétectée](../WDLang1/3087001.md) est à <u><u><u><u>Vrai</u></u></u></u>. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Paramètre> : (Constante de type chaîne de caractères ou chaîne de caractères)`**

Paramètre pour lequel les valeurs supportées doivent être récupérées. Ce paramètre peut correspondre : 

- soit à une des constantes prédéfinies. Ces constantes correspondent aux paramètres les plus courants. 

- soit à une Chaîne de caractères correspondant au nom du paramètre voulu. 
	**Attention** : le nom du paramètre est sensible à la casse et à l'accentuation. Consultez la documentation du SDK Android ou de l'appareil pour connaître le nom des paramètres supplémentaires supportés. Si le nom spécifié ne correspond pas à un paramètre supporté par la caméra, la fonction retournera chaîne vide.


Les constantes disponibles sont les suivantes : 



|   |   |
| --- | --- |
| *vipBalance* | Modes de balance des blancs supportés par la caméra : automatique, lumière du jour, nuageux, ...<br> |
| *vipCaméra* | Indices et types des caméras disponibles sur l'appareil. La liste retournée est au format suivant : <br><br><pre><code><Indice><TAB><Type></code></pre><br>où :<br><br>- &lt;Indice&gt; est un entier compris entre 1 et le nombre de caméras présentes sur l'appareil,<br><br>- &lt;Type&gt; est une des constantes suivantes :<br><br>	- *vipCaméraDorsale* : Caméra dorsale située sur la face opposée à l'écran.<br><br>	- *vipCaméraFrontale* : Caméra frontale située sur la même face que l'écran.<br><br><br><br><br><br> |
| *vipEffet* | Effets de couleurs : noir et blanc, poster, sépia, ...<br> |
| *vipFlash* | Modes de flash supportés par la caméra : automatique, sans, anti yeux rouges, ... |
| *vipFocus* | Modes de focus supportés par la caméra : automatique, macro, fixe, ...<br> |
| *vipISO* | Sensibilités ISO supportées par la caméra : automatique, 100, 200, 400, ...<br> |
| *vipRésolutionPhoto* | Résolutions supportées par la caméra pour la prise de photo. <br>La liste renvoyée est triée par ordre croissant de la plus faible résolution à la plus importante.<br>Les valeurs retournées sont au format : <br><br><pre><code><Largeur><TAB><Hauteur></code></pre><br>Exemple : <br><br><pre><code>640<TAB>480<RC>1024<TAB>760<RC>2048<TAB>1240</code></pre><br> |
| *vipScène* | Types de scènes supportés par la caméra : automatique, nuit, sport, ...<br> |
| *vipScintillement* | Ajustements du scintillement (antibanding) supportés par la caméra : automatique, aucun, 50Hz, 60Hz, ...<br> |
| *vipZoom* | Valeurs de zoom supportées par la caméra.<br>Les valeurs retournées sont de la forme :<br><br>- 100 (Zoom 1x = pas de zoom)<br><br>- 200 (zoom 2x)<br><br>- 300 (zoom 3x)<br><br>- 350 (zoom 3,5x)<br><br>- ...<br><br><br>La première valeur retournée est toujours 100 et la dernière correspond au zoom maximal. <br>Exemple : <br><br><pre><code>100<RC>200<RC>300<RC>350</code></pre><br>**Remarque** : La fonction retournera une chaîne vide ("") si la caméra de l'appareil ne supporte pas le zoom.<br> |

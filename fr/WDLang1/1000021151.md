
## PiloteSimulateur (Fonction)

***En anglais : EmulateSimulator***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Pilote la fenêtre du simulateur mobile (Android ou iOS).

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Cette fonction est disponible sous WINDEV uniquement pour pouvoir être utilisée en mode simulateur. Utilisée dans une application WINDEV, cette fonction n'a aucun effet. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Change l'orientation du simulateur
PiloteSimulateur(psTourneEcran)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`PiloteSimulateur(<Action>)`
---

**`<Action> : (Constante de type entier)`**

Action à réaliser : 


|   |   |
| --- | --- |
| *psFerme* | Ferme le simulateur. |
| *psMenu* | Simule un appui sur le bouton Menu.<br><br> |
| *psRetour* | Simule un appui sur le bouton Retour.<br><br> |
| *psTourneEcran* | Tourne l'écran : passe du mode paysage au mode portrait ou inversement. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Cette fonction doit être utilisée uniquement dans le code de tests automatiques effectués sur le simulateur Android ou iOS.




## DocChangeAltitude (Fonction)

***En anglais : DocChangeZOrder***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie de façon intelligente l'altitude d'un élément de type image ou zone de texte d'un document docx.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
f est docFragment <- TT_Document.Sélection.Fragment
e est un docElément <- f.Elément[1]
SI e.Image <> Null ALORS
	DocChangeAltitude(e, docAltitudePremierPlan)
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`DocChangeAltitude(<Elément> , <Modification>)`
---

**`<Elément> : (Variable de type docElément)`**

Nom de la variable de type [docElément](../WDLang1/1000022484.md) à manipuler. Cet élément doit être de type "Image" ou "Zone de texte". Dans le cas contraire, une erreur fatale est affichée. 

**`<Modification> : (Constante de type Entier)`**

Nouvelle altitude :


|   |   |
| --- | --- |
| *docAltitudeArrièrePlan* | Passe l'élément en arrière-plan. |
| *docAltitudeAvancer* | Passe l'élément au plan supérieur. |
| *docAltitudePremierPlan* | Passe l'élément au premier plan. |
| *docAltitudeReculer* | Passe l'élément au plan inférieur. |

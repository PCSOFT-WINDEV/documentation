
## VoletDuChamp (Fonction)

***En anglais : ControlPane***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le nom du volet en cours (statique ou dynamique) d'un champ Onglet associé à un champ. 

Remarque : Dans le cas de volets d'onglets dynamiques, cette fonction peut être utilisée dans le code de la fenêtre interne pour connaître le nom du volet d'onglet associé. 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Clic sur bouton "BTN _Nouveau"
MonVolet est un Champ 
MonVolet <- VoletDuChamp(MoiMême)
MonVolet.Libellé = "Nouveau document.txt"
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = VoletDuChamp(<Champ>)`
---

**`<Résultat> : (Chaîne de caractères)`**



- Nom du volet d'onglet auquel le champ appartient (alias). 

- "" (chaîne vide) si le &lt;Champ&gt; n'est pas dans un volet d'onglet. 




**`<Champ> : (Nom de champ)`**

Nom d'un champ présent dans un volet statique ou dynamique d'un champ Onglet.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- L'onglet du champ peut être connu avec la fonction [OngletDuChamp](../WDLang1/3025028.md).

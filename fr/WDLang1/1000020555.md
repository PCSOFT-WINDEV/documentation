
## SysVersionIOS (Fonction)

***En anglais : SysIOSVersion***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Retourne des informations sur la version d'iOS utilisée par l'application.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI SysVersionIOS() = "6.0" ALORS
	// Traitement spécifique à la version 6.0
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = SysVersionIOS()`
---

**`<Résultat> : (Chaîne de caractères)`**

Version courante du système exprimée la forme "X.Y".

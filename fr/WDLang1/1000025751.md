
## StyleDessinLibelléGrisé (Fonction)

***En anglais : StyleDrawCaptionGrayed***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Personnalise le style de dessin des libellés, dans les champs qui sont grisés. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI EnModeGabaritClair() ALORS
	StyleDessinLibelléGrisé(sdlgParDéfaut)
SINON
	StyleDessinLibelléGrisé(sdlgMonochrome, GrisClair)
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`StyleDessinLibelléGrisé(<Style> [, <Couleur>])`
---

**`<Style> : (Constante de type Entier)`**

Style à utiliser : 



|   |   |
| --- | --- |
| *sdlgMonochrome* | Les libellés sont dessinés avec la couleur spécifiée avec le paramètre &lt;Couleur&gt;. |
| *sdlgParDéfaut* | Les libellés sont dessinés avec le style par défaut (relief enfoncé gris). |





**`<Couleur> : (Entier optionnel)`**

Couleur à utiliser pour les libellés si la constante *sdlgMonochrome* est précisée. Cette couleur peut correspondre à :

- une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)),

- une couleur TSL (obtenue avec la fonction [TSL](../WDLang1/3029057.md)),

- [une couleur prédéfinie du WLangage](../WDLang5/3010002.md).

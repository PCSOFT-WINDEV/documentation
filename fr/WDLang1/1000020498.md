


## fCherche (Fonction)

***En anglais : fFind***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Cherche une chaîne de caractères ou un buffer dans un fichier ouvert avec la fonction [fOuvre](../WDLang1/3036036.md).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonBuffer est un Buffer
IDFichier est un entier 
nPosition est un entier 

MonBuffer = fChargeBuffer("C:\tmp\Page1.html")
IDFichier = fOuvre("C:\test\MonFichier.html")
nPosition = fCherche(IDFichier, MonBuffer)

SI nPosition <> -1 ALORS
	Info("Le contenu de Page1 a été trouvé dans MonFichier à la position " + nPosition)
FIN
```

```wl
MonBuffer est un Buffer
MonFichier est un FichierDisque 
nPosition est un entier 

MonBuffer = fChargeBuffer("C:\tmp\Page1.html")
MonFichier = fOuvre("C:\test\MonFichier.html")
nPosition = fCherche(MonFichier, MonBuffer)

SI nPosition <> -1 ALORS
	Info("Le contenu de Page1 a été trouvé dans MonFichier à la position " + nPosition)
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = fCherche(<Fichier manipulé> , <Elément recherché> [, <Position de départ>])`
---

**`<Résultat> : (Entier)`**



- Position de l'élément recherché dans le fichier.

- -1 si aucun élément ne correspond à la recherche. 




**`<Fichier manipulé> : (Entier ou variable de type FichierDisque)`**



- Identifiant du fichier à manipuler, défini avec la fonction [fOuvre](../WDLang1/3036036.md) ou [fCrée](../WDLang1/3036017.md).

- Variable de type [FichierDisque](../WDLang1/1410088883.md) initialisée avec la fonction [fOuvre](../WDLang1/3036036.md) ou [fCrée](../WDLang1/3036017.md).




**`<Elément recherché> : (Buffer ou chaîne de caractères)`**

Buffer ou chaîne de caractères dont le contenu doit être recherché dans le fichier texte.

**`<Position de départ> : (Entier optionnel)`**

Position de départ de la recherche. Par défaut, la recherche est effectuée à partir du début du fichier.

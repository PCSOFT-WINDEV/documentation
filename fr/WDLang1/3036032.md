


## fNomCourt (Fonction)

***En anglais : fShortName***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le nom court d'un fichier ou d'un répertoire.

<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple
<a class="notetitle" target="_blank" href="$DOC$=1000003036032&name=fnomcourt_fonction&product=WD">Voir des exemples supplémentaires</a>

```wl
// Récupération du nom court d'un fichier
ResNomCourt = fNomCourt("C:\MesRépertoires\Compte-Rendu Mars 2001.doc")
// fNomCourt renvoie "Compte~1.doc"
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = fNomCourt(<Chemin du fichier> [, <Options>])`
---

**`<Résultat> : (Chaîne de caractères)`**



- Nom court du fichier ou du répertoire. Seul le nom court du fichier ou du répertoire est renvoyé.

- Chaîne vide ("") en cas d'erreur (fichier ou répertoire inexistant). Pour connaître le libellé de l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.




**`<Chemin du fichier> : (Chaîne de caractères)`**

Nom et chemin complet (ou relatif) du fichier ou du répertoire (260 caractères maximum). Un chemin de type UNC peut être utilisé.  
![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png)![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) Ce paramètre peut être au format Ansi ou Unicode.

**`<Options> : (Constante optionnelle de type Entier)`**

Permet de spécifier le format du résultat de la fonction. Par défaut, **fNomCourt** renvoie une chaîne de caractères Ansi. 


|   |   |
| --- | --- |
| *fCheminUNICODE* | &lt;Résultat&gt; sera une chaîne de caractères au format Unicode. |


**Remarque** : Le résultat de la fonction **fNomCourt** est toujours une chaîne de caractères au format Ansi, la constante *fCheminUNICODE* permet uniquement de stocker le résultat dans une chaîne de caractères Unicode.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Noms longs
<a name="noms_longs_ELTPARAGRAPHE000069"></a>La fonction **fNomCourt** renvoie :

- *le nom court du fichier ou du répertoire* si le &lt;Chemin du fichier&gt; est exprimé en noms longs (noms de plus de 8 caractères).

- *le nom du fichier spécifié* si le nom du fichier dans &lt;Chemin du fichier&gt; est :

	- soit exprimé en nom court (nom de 8 caractères et moins, comprenant une tilde "~"),

	- soit composé de 8 caractères ou moins.




- *le nom du répertoire spécifié* si le nom du répertoire dans &lt;Chemin du fichier&gt; est :

	- soit exprimé en nom court (nom de 8 caractères et moins, comprenant une tilde "~"),

	- soit composé de 8 caractères ou moins.

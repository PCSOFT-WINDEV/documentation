
## SansEspace (Fonction)

***En anglais : NoSpace***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie une chaîne de caractères sans les espaces : 

- situés à gauche et à droite.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) situés à l'intérieur de la chaîne.







<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SansEspace("   3.75 Euros   ")   // Renvoie "3.75 Euros"
```


<a name="Exemple2"></a>

```wl
SansEspace("Abra ca da bra", sscIntérieur)   // Renvoie "Abracadabra"
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = SansEspace(<Chaîne initiale> [, <Position>])`
---

**`<Résultat> : (Chaîne de caractères)`**

Chaîne de caractères sans les espaces.

**`<Chaîne initiale> : (Chaîne de caractères)`**

Chaîne de caractères initiale dont les espaces à droite et à gauche doivent être supprimés. Cette chaîne de caractères n'est pas modifiée.

**`<Position> : (Constante de type Entier)`**

Position des espaces à supprimer. Cette position peut correspondre à une des constantes suivantes : 


|   |   |
| --- | --- |
| *sscDroite* | Supprime les espaces situés à droite de la chaîne initiale. |
| *sscExtérieur*<br>(Valeur par défaut) | Supprime les espaces situés à l'extérieur de la chaîne initiale. <br>Equivalent à *sscDroite* + *sscGauche*. |
| *sscGauche* | Supprime les espaces situés à gauche de la chaîne initiale. |
| *sscIntérieur* | Supprime les espaces présents à l'intérieur de la chaîne initiale. |
| *sscTout* | Supprime tous les espaces présents dans la chaîne initiale. <br>Equivalent à *sscDroite* + *sscGauche* + *sscIntérieur*. |




![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Ce paramètre n'est pas disponible. 





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Cas particuliers
<a name="cas_particuliers_ELTPARAGRAPHE000135"></a>

- La fonction [Gauche](../WDLang1/3024001.md) permet également de supprimer les espaces à droite.

- La fonction [Droite](../WDLang1/3024016.md) permet également de supprimer les espaces à gauche.

- **Espaces insécables** : Les espaces insécables (constante *ESP* ou caractère &lt;160&gt;) ne sont pas supprimés par la fonction **SansEspace**. Pour les supprimer, il est possible d'utiliser la fonction [SansCaractèreDroite](../WDLang1/1000020479.md) avec la constante *ESP*.





<a name="NOTE0_2"></a>




### Fonction SansEspace et UNICODE
<a name="fonction_docparampagetitleshort_unicode_ELTPARAGRAPHE000158"></a>Si la chaîne passée en paramètre est au format ANSI, le résultat sera au format ANSI.

Si la chaîne passée en paramètre est au format UNICODE, le résultat sera au format UNICODE.

**Remarque** : Si le résultat de la fonction **SansEspace** utilisée sur une chaîne ANSI est affecté à une chaîne UNICODE (et inversement), la conversion ANSI/UNICODE sera implicitement réalisée.

Pour plus de détails, consultez [Gestion de l'UNICODE](../WDLang1/3024038.md).


## docMiseEnForme (Type de variable)

***En anglais : docFormatting***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **docMiseEnForme** permet de définir les caractéristiques de mise en forme d'un élément d'un document de type Traitement de texte. Les caractéristiques de mise en forme peuvent être définies et modifiées à l'aide de différentes propriétés WLangage. 

**ATTENTION : Ce type ne doit pas être utilisé directement. Il doit être utilisé uniquement via** : 

- le type [docElémentFormule](../WDLang1/1000022937.md). 

- le type [docElémentTexte](../WDLang1/1000022493.md). 







<a name="NOTE0"></a>

## Remarques
<a name="NOTE0_1"></a>


### Propriétés spécifiques à la description des variables de type docMiseEnForme
<a name="proprietes_specifiques_description_des_variables_type_docmiseenforme_ELTPARAGRAPHE000030"></a>Les propriétés suivantes peuvent être utilisées pour manipuler la mise en forme  :

| Nom de la propriété | Type manipulé | Effet |
| --- | --- | --- |
| AlignementVertical | Constante de type Entier | Alignement vertical du texte par rapport à la ligne de base : <br><br>- *alignementBaseline* : Le texte est positionné sur la ligne de base. <br><br>- *alignementExposant* : Le texte est positionné en exposant. <br><br>- *alignementIndice* : Le texte est positionné en indice. <br><br><br> |
| CouleurSouligné | Entier | Couleur utilisée pour souligner le texte. Cette couleur peut correspondre à :<br><br>- une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)).<br><br>- une couleur TSL (obtenue avec la fonction [TSL](../WDLang1/3029057.md)),<br><br>- [une couleur prédéfinie](../WDLang5/3010002.md) du WLangage.<br><br><br> |
| CouleurSurlignage | Entier | Couleur utilisée pour surligner le texte. Seules les couleurs suivantes sont disponibles : <br><br>- *Transparent*<br><br>- *Noir*<br><br>- *BleuClair*<br><br>- *CyanClair*<br><br>- *VertClair*<br><br>- *MagentaClair*<br><br>- *RougeClair*<br><br>- *JauneClair*<br><br>- *Blanc*<br><br>- *BleuFoncé*<br><br>- *CyanFoncé*<br><br>- *VertFoncé*<br><br>- *MagentaFoncé*<br><br>- *RougeFoncé*<br><br>- *JauneFoncé*<br><br>- *GrisClair*<br><br>- *GrisFoncé*<br><br><br> |
| CouleurTexte | Entier | Couleur du texte. Cette couleur peut correspondre à :<br><br>- une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)).<br><br>- une couleur TSL (obtenue avec la fonction [TSL](../WDLang1/3029057.md)),<br><br>- [une couleur prédéfinie](../WDLang5/3010002.md) du WLangage.<br><br><br> |
| PoliceBarrée | Booléen | <br><br>- <u><u><u><u>Vrai</u></u></u></u> si la police du texte est barrée, <br><br>- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. <br><br><br> |
| PoliceGras | Booléen | <br><br>- <u><u><u><u>Vrai</u></u></u></u> si la police du texte est en gras,<br><br>- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. <br><br><br> |
| PoliceItalique | Booléen | <br><br>- <u><u><u><u>Vrai</u></u></u></u> si la police du texte est en italique,<br><br>- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. <br><br><br> |
| PoliceNom | Chaîne de caractères | Nom de la police utilisée pour le texte. |
| PoliceSoulignée | Constante de type Entier | Type de soulignement : <br><br>- *soulignéAucun* : Aucun soulignement<br><br>- *soulignéDouble* : Soulignement double<br><br>- *soulignéEpais* : Soulignement épais<br><br>- *soulignéMots* : Soulignement des mots uniquement<br><br>- *soulignéPointillés* : Soulignement en pointillés<br><br>- *soulignéPointillésEpais* : Soulignement en pointillés épais<br><br>- *soulignéPointPointTiret* : Soulignement de type '..-' (point point tiret)<br><br>- *soulignéPointPointTiretEpais* : Soulignement de type '..-' (point point tiret) épais<br><br>- *soulignéPointTiret* : Soulignement de type '.-' (point tiret)<br><br>- *soulignéPointTiretEpais* : Soulignement de type '.-' (point tiret) épais<br><br>- *soulignéSimple* : Soulignement simple<br><br>- *soulignéTiret* : Soulignement avec des tirets<br><br>- *soulignéTiretEpais* : Soulignement avec des tirets épais<br><br><br> |
| PoliceTaille | Réel | Taille de la police. |

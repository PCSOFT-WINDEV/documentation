
## FAAImporteParamètre (Fonction)

***En anglais : AAFImportParameter***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Charge toutes les personnalisations réalisées via les FAA (Fonctionnalités Automatiques de vos Applications) ayant été précédemment sauvegardées dans un fichier. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Clic sur "Importer mes paramètres"
NomFichier est une chaîne 
NomFichier = fSélecteur("", "", "Sélectionnez un fichier...", "paramètres" + ...
	TAB + "*.param" + RC + "Tous les fichiers (*.*)" + ...
	TAB + "*.*", "*.*", fselExiste)
SI NomFichier = "" ALORS RETOUR
FAAImporteParamètre(NomFichier)
SI ErreurDétectée ALORS
	Erreur()
SINON
	ToastAffiche("Paramètres importés avec succès.")
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`FAAImporteParamètre(<Fichier à importer>)`
---

**`<Fichier à importer> : (Chaîne de caractères)`**

Nom et chemin complet du fichier dans lequel les personnalisations des FAA ont été précédemment sauvées.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Personnalisations sauvegardées
<a name="personnalisations_sauvegardees_ELTPARAGRAPHE000032"></a>Toutes les personnalisations réalisées par l'utilisateur sont sauvegardées notamment : 

- Les tailles et positions des fenêtres, 

- Les valeurs mémorisées des champs,

- L'historique des saisies, 

- Les perso-notes, 

- La personnalisation des champs Tableau de bord, 

- La personnalisation des panneaux dockables, ...




Ces paramètres peuvent ensuite être restaurées par la fonction **FAAImporteParamètre** ou via la Fonctionnalité automatique d'import des paramètres (option disponible dans le menu contextuel d'une fenêtre de l'application).

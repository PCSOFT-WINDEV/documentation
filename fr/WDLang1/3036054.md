
## fTailleDécompacté (Fonction)

***En anglais : fSizeUncompressed***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la taille d'un fichier avant compression.

Lors de la décompression d'un fichier avec [fDécompacte](../WDLang1/3036021.md), il est conseillé de comparer la taille non-compressée du fichier et l'espace disque disponible.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple
<a class="notetitle" target="_blank" href="$DOC$=1000003036054&name=ftailledecompacte_fonction&product=WD">Voir des exemples supplémentaires</a>

```wl
// Taille avant compression d'un fichier
ResTailleDécompacté = fTailleDécompacté("C:\MesRépertoires\Fichier.doc")
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = fTailleDécompacté(<Chemin du fichier>)`
---

**`<Résultat> : (Entier)`**



- Taille du fichier avant compression (en octets),

- -1 en cas d'erreur. Pour connaître le libellé de l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.

- -2 si le fichier n'a pas été compressé avec [fCompacte](../WDLang1/3036016.md) ou si le fichier a été compressé avec WINDEV version 4 (4.0 ou 4.1).




**`<Chemin du fichier> : (Chaîne de caractères)`**

Nom et chemin complet (ou relatif) du fichier compressé (260 caractères maximum). Un chemin de type UNC peut être utilisé. Ce fichier doit avoir été compressé avec [fCompacte](../WDLang1/3036016.md). Les caractères jokers (\*, ?) sont interdits.
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png) Ce paramètre peut être au format Ansi ou Unicode.

## Remarques




### Gestion des erreurs
<a name="gestion_des_erreurs_ELTPARAGRAPHE000071"></a>La fonction **fTailleDécompacté** génère une erreur dans les cas suivants :

- le fichier spécifié n'existe pas,

- le nom du fichier spécifié est invalide,

- le fichier est bloqué par une autre application.

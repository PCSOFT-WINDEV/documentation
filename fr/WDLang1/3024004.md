


## Asc (Fonction)

***En anglais : Asc***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le code ASCII du caractère spécifié. Le code ASCII utilisé correspond à la norme ANSI de Windows.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple
<a class="notetitle" target="_blank" href="$DOC$=1000003024004&name=asc_fonction&product=WD">Voir des exemples supplémentaires</a>

```wl
CodeASCII = Asc("A")    // Retourne 65
CodeASCII = Asc("Barbier")   // Retourne 66
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = Asc(<Chaîne de caractères à traiter>)`
---

**`<Résultat> : (Entier)`**

Code ASCII recherché (compris entre 0 et 255).

**`<Chaîne de caractères à traiter> : (Caractère, chaîne de caractères ou buffer)`**

Caractère dont le code ASCII doit être calculé. Si ce paramètre correspond à : 

- une chaîne ou un buffer vide, &lt;Résultat&gt; sera égal à 0.

- une chaîne de caractères, &lt;Résultat&gt; sera égal au code ASCII du premier caractère.

- un buffer, &lt;Résultat&gt; sera égal au code ASCII du premier octet.

<a name="NOTE0_1"></a>

## Remarques


### Extraction de chaîne
<a name="extraction_chaine_ELTPARAGRAPHE000072"></a>L'extraction de chaîne (par les opérateurs [[ et ]]) peut être utilisée pour convertir un caractère présent à une position quelconque dans une chaîne.

Par exemple : "Asc(Ch[[N]])" retourne la valeur ASCII du Nième caractère de la chaîne "Ch".


## ExceptionAffiche (Fonction)

***En anglais : ExceptionDisplay***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Affiche la fenêtre standard des exceptions avec le contenu de l'exception courante.






<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
QUAND EXCEPTION DANS
	Proc()
FAIRE
	// Affichage de l'exception
	ExceptionAffiche()
	// Code spécifique après l'affichage
	...
	// Fin de l'application
	FinProgramme()
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = ExceptionAffiche()`
---

**`<Résultat> : (Constante de type Entier)`**

Manipulation effectuée par l'utilisateur sur la fenêtre : 


|   |   |
| --- | --- |
| *eaFinProgramme* | L'utilisateur n'a pas coché l'option "Relancer l'application". |
| *eaRelanceProgramme* | L'utilisateur a coché l'option "Relancer l'application". |


![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) La fenêtre standard des exceptions est affichée uniquement en mode test. Lors de l'exécution du site, la fenêtre n'est pas affichée. Dans ce cas, &lt;Résultat&gt; vaut *eaFinProgramme*.

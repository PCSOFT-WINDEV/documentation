


## TableurAjouteFeuille (Fonction)

***En anglais : SpreadsheetAddWorksheet***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ajoute une feuille de calcul dans un champ Tableur présent dans une fenêtre. 

Remarque : Pour utiliser cette fonction, le champ Tableur doit autoriser la gestion de plusieurs feuilles de calcul : 

- Sous l'éditeur de fenêtres : Dans l'onglet "Détail" de la fenêtre de description du champ, cochez l'option "Autoriser la gestion de plusieurs feuilles de calcul". 

- Par programmation, utilisez la propriété [MultiFeuille](../Proprietes/1000021639.md). 



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ajout d'une feuille de calcul
nFeuille est entier 
nFeuille = TableurAjouteFeuille(TBLR_MonTableur)
// Sélection de la feuille de calcul
TBLR_MonTableur.FeuilleEnCours = nFeuille
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = TableurAjouteFeuille(<Champ Tableur> [, <Nom de la feuille>])`
---

**`<Résultat> : (Entier)`**

Numéro de la feuille de calcul créée dans le champ Tableur. 

**`<Champ Tableur> : (Nom de champ)`**

Nom du champ Tableur à manipuler.

**`<Nom de la feuille> : (Chaîne de caractères optionnelle)`**

Nom de la feuille de calcul à créer. Si ce paramètre n'est pas précisé, la feuille de calcul est créée avec un nom par défaut ("FeuilleX"). Ce nom : 

- apparaît dans l'onglet permettant de sélectionner la feuille (en bas du champ Tableur). 

- est utilisé pour désigner la feuille dans les formules. Par exemple, une formule utilisant une cellule de la "Feuille2" peut être : "=SOMME(Feuille2!B4:D4)". 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- La fonction échoue et affiche une erreur WLangage dans les cas suivants : 

	- Le champ Tableur ne gère pas plusieurs feuilles de calcul. 

	- Le nom de la feuille de calcul spécifié est invalide ou déjà utilisé par une autre feuille. 




- Cette fonction est utilisable uniquement sur un champ Tableur présent dans une fenêtre.

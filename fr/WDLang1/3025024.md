
## RTFAjoute (Fonction)

***En anglais : RTFAdd***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ajoute une chaîne de caractères (au format RTF ou non) à la fin du contenu : 

- d'un champ de saisie de type RTF, 

- d'un champ RTF dans un état (état WINDEV uniquement)

- d'une chaîne de caractères contenant un texte au format RTF.




**Attention** : Quelle que soit la syntaxe utilisée, cette fonction est déconseillée dans un thread. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ajoute le contenu de SAI_SAISIE2 à la fin de SAI_SAISIE1
RTFAjoute(SAI_SAISIE1, SAI_SAISIE2)
```


<a name="Exemple2"></a>



```wl
// Ajoute "(c) 2003 MaSociete" à la fin de SAI_SAISIE1
RTFAjoute(SAI_SAISIE1, "(c) 2003 MaSociété")
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Ajouter une chaîne de caractères à la fin d'un champ RTF

`RTFAjoute(<Champ RTF> , <Nouvelle chaîne de caractères>)`
---

**`<Champ RTF> : (Chaîne de caractères)`**

Nom du champ RTF (champ de saisie de type RTF dans une fenêtre, champ RTF d'un état WINDEV).

**`<Nouvelle chaîne de caractères> : (Chaîne de caractères)`**

Chaîne de caractères (au format RTF ou non) à ajouter au contenu du champ de type RTF.  


<a name="SYNTAXE2"></a>

### Ajouter une chaîne de caractères à la fin d'une chaîne RTF

`<Résultat> = RTFAjoute(<Chaîne RTF> , <Nouvelle chaîne de caractères>)`
---

**`<Résultat> : (Chaîne de caractères)`**

Nouvelle chaîne de caractères dans laquelle l'ajout a été effectué. La chaîne initiale n'est pas modifiée.

**`<Chaîne RTF> : (Chaîne de caractères)`**

Nom de la chaîne de caractères contenant des caractères au format RTF.

**`<Nouvelle chaîne de caractères> : (Chaîne de caractères)`**

Chaîne de caractères (au format RTF ou non) à ajouter au contenu de la chaîne RTF.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Cette fonction pilote la gestion RTF de WIndows. Selon la version de Windows utilisée, il se peut qu'une fin de paragraphe soit ajoutée à la fin de la ligne.

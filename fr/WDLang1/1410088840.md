
## diagActionAutorisée (Type de variable)

***En anglais : diagActionAllowed***
			
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **diagActionAutorisée** permet de définir tous les droits de l'utilisateur final sur une forme d'un diagramme présent dans le champ Editeur de diagrammes. Les caractéristiques de ces droits peuvent être définies et modifiées à l'aide de différentes propriétés WLangage. 

Attention : Ce type de variable peut être utilisé uniquement sur les variables dérivant du type [diagForme](../WDLang1/1410088082.md) : 

- Variable de type [diagChemin](../WDLang1/1410088384.md),

- Variable de type [diagGroupe](../WDLang1/1410088127.md),

- Variable de type [diagImage](../WDLang1/1410088414.md),

- Variable de type [diagLien](../WDLang1/1410088454.md),

- Variable de type [diagOvale](../WDLang1/1410088135.md),

- Variable de type [diagPolygone](../WDLang1/1410088137.md),

- Variable de type [diagRectangle](../WDLang1/1410088083.md).




**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Définition des droits sur une forme de type Image
fond est un diagImage
fond.Image = "fond.png"
fond.ActionAutorisée.Déplacer = Faux
fond.ActionAutorisée.Redimensionner = Faux
fond.ActionAutorisée.Rotation = Faux
x est un Diagramme <- EDIAG_SansNom1
x.Forme.Ajoute(fond)
```

## Propriétés


### Propriétés spécifiques à la description des variables de type diagActionAutorisée
<a name="proprietes_specifiques_description_des_variables_type_diagactionautorisee_ELTPARAGRAPHE000060"></a>Les propriétés suivantes peuvent être utilisées pour manipuler une action autorisée sur une forme :

| Nom de la propriété | Type manipulé | Effet |
| --- | --- | --- |
| Altitude | Booléen | <br><br>- <u><u><u>Vrai</u></u></u> pour autoriser le changement d'altitude de la forme, <br><br>- <u><u><u>Faux</u></u></u> pour l'interdire.  <br><br><br> |
| ChangerProportion | Booléen | Si la propriété **Redimensionner** vaut <u><u><u>Vrai</u></u></u> : <br><br>- <u><u><u>Vrai</u></u></u> pour autoriser le changement de la hauteur et de la largeur de la forme indépendamment, <br><br>- <u><u><u>Faux</u></u></u> pour l'interdire.  <br><br><br> |
| Copier | Booléen | <br><br>- <u><u><u>Vrai</u></u></u> pour autoriser la copie et la duplication d'une forme, <br><br>- <u><u><u>Faux</u></u></u> pour l'interdire.  <br><br><br> |
| Déplacer | Booléen | <br><br><br><br>- <u><u><u>Vrai</u></u></u> pour autoriser le déplacement d'une forme, <br><br>- <u><u><u>Faux</u></u></u> pour l'interdire.  <br><br><br> |
| EditerLibellé | Booléen | <br><br>- <u><u><u>Vrai</u></u></u> pour autoriser l'édition du libellé d'une forme, <br><br>- <u><u><u>Faux</u></u></u> pour l'interdire.  <br><br><br><br><br> |
| EditerStyle | Booléen | <br><br>- <u><u><u>Vrai</u></u></u> pour autoriser le changement de style d'une forme (couleur, cadre, police, ...), <br><br>- <u><u><u>Faux</u></u></u> pour l'interdire.  <br><br><br> |
| PointDAttache | Booléen | <br><br>- <u><u><u>Vrai</u></u></u> pour autoriser le changement des points d'attache d'une forme (liens uniquement), <br><br>- <u><u><u>Faux</u></u></u> pour l'interdire.  <br><br><br> |
| PointDePassage | Booléen | <br><br>- <u><u><u>Vrai</u></u></u> pour autoriser le changement, l'ajoute et la suppression de points d'attache d'une forme (liens uniquement), <br><br>- <u><u><u>Faux</u></u></u> pour l'interdire.  <br><br><br> |
| Redimensionner | Booléen | <br><br>- <u><u><u>Vrai</u></u></u> pour autoriser le changement de taille d'une forme, <br><br>- <u><u><u>Faux</u></u></u> pour l'interdire.  <br><br><br> |
| Rotation | Booléen | <br><br>- <u><u><u>Vrai</u></u></u> pour autoriser de faire tourner une forme, <br><br>- <u><u><u>Faux</u></u></u> pour l'interdire.  <br><br><br> |
| Sélectionner | Booléen | <br><br>- <u><u><u>Vrai</u></u></u> pour autoriser la sélection d'une forme, <br><br>- <u><u><u>Faux</u></u></u> pour l'interdire.  <br><br><br> |
| Supprimer | Booléen | <br><br>- <u><u><u>Vrai</u></u></u> pour autoriser la suppression d'une forme, <br><br>- <u><u><u>Faux</u></u></u> pour l'interdire.  <br><br><br> |


## fDécoupe (Fonction)

***En anglais : fSplit***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Découpe un fichier en plusieurs fichiers. Les fichiers créés pourront être rassemblés en un seul fichier grâce à la fonction [fRegroupe](../WDLang1/1000021020.md). 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Découpe le fichier Video.avi
fDécoupe("Video.avi", 100*1024*1024, "VideoDécoupée")
SI ErreurDétectée ALORS
	// Affiche le message d'erreur
	Erreur(ErreurInfo())
	RETOUR
FIN

// Des fichiers "VideoDécoupée.001", "VideoDécoupée.002" ont été créés
// Suppression du fichier source "Video.avi"
fSupprime("Video.avi")

// Utilise les fichiers "VideoDécoupée.001", "VideoDécoupée.002" 
// pour recréer le fichier "Video.avi" 
fRegroupe("VideoDécoupée", "Video.avi")
SI ErreurDétectée ALORS
	// Affiche le message d'erreur
	Erreur(ErreurInfo())
	RETOUR
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = fDécoupe(<Nom du fichier> , <Taille maximale> [, <Fichier de sortie> [, <Option> [, <Champ Jauge>]]])`
---

**`<Résultat> : (Entier)`**



- Nombre de fichiers créés, 

- 0 en cas d'erreur. Pour connaître le libellé de l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.




**`<Nom du fichier> : (Chaîne de caractères)`**

Nom complet du fichier à découper. 

**`<Taille maximale> : (Entier)`**

Taille maximale exprimée en octets de chaque fichier créé. Cette taille doit être supérieure à 0. 

**`<Fichier de sortie> : (Chaîne de caractères optionnelle)`**

Chemin complet des fichiers à créer. Les fichiers créés correspondront au nom indiqué suivi d'une extension ".xxx" où xxx correspond à l'indice du fichier créé.
Par défaut, &lt;Fichier de sortie&gt; correspond à &lt;Nom du fichier&gt;.

**`<Option> : (Constante de type Entier optionnelle)`**

Par défaut, le fichier source est conservé lors du découpage. Il est possible de spécifier l'option de découpage suivante : 


|   |   |
| --- | --- |
| *fSupprimeSource* | Le fichier source sera supprimé au fur et à mesure du découpage.<br><br>Attention : Si une erreur survient, le fichier source sera PERDU et ne pourra pas être retrouvé. |



**`<Champ Jauge> : (Nom de champ optionnel)`**

Nom du champ Jauge permettant de suivre la progression du découpage du fichier. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- La fonction [fOuvre](../WDLang1/3036036.md) ne peut pas être utilisée sur un fichier multi-parties. Avant d'utiliser la fonction [fOuvre](../WDLang1/3036036.md), il faut recréer le fichier original avec la fonction [fRegroupe](../WDLang1/1000021020.md). 
	Remarque : La fonction [fOuvre](../WDLang1/3036036.md) utilisée sur une partie permet uniquement d'accéder au sous-fichier correspondant à la partie. 

- Si un fichier existant a le même nom que celui d'un des sous-fichiers à créer, la fonction renverra une erreur.

- Le nombre de fichiers pouvant être créés est limité à 10 000.


## Description d'Option (Type de variable)

***En anglais : Option Description***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Une variable de type **Description d'Option** permet de manipuler par programmation les caractéristiques d'une option de combinaison.

Les caractéristiques de ce type peuvent être connues à l'aide de différentes propriétés WLangage. 

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Récupère la définition du type de d
d est une Définition
d = RécupèreDéfinition(ActionFichier)
// Affiche le nom du type de d
Trace(d.Nom)

// Solution 1 : lister les options de la combinaison
SI d.Type = wlCombinaison ALORS
	POUR TOUT MonElément DE d.Option
		Trace(MonElément.Nom)	
	FIN
FIN
// Solution 2 : lister les options de la combinaison 
// (en utilisant le type Description d'Option)
SI d.Type = wlCombinaison ALORS
	D1 est une Description d'Option
	POUR TOUT D1 DE d.Option
		Trace(D1.Nom)	
	FIN
FIN
```

## Remarques
<a name="NOTE0_1"></a>


### Propriétés spécifiques à la description des variables du type Description d'Option
<a name="proprietes_specifiques_description_des_variables_type_description_option_ELTPARAGRAPHE000031"></a>Les propriétés suivantes peuvent être utilisées pour manipuler une option de combinaison :

| Nom de la propriété | Type manipulé | Effet |
| --- | --- | --- |
| Alias | Booléen | <br><br>- <u><u><u><u>Vrai</u></u></u></u> si l'option est un alias d'une option ou d'une combinaison d'options existante, <br><br>- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. <br><br><br>**Cette propriété est disponible en lecture seulement**. |
| Déclaration | Variable de type [Définition](../WDLang1/1000019540.md) | Déclaration de l'option. <br><br>**Cette propriété est disponible en lecture seulement**. |
| Hérité | Booléen | <br><br>- <u><u><u><u>Vrai</u></u></u></u> si la combinaison est héritée d'une combinaison étendue, <br><br>- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. <br><br><br>**Cette propriété est disponible en lecture seulement**. |
| Nom | Chaîne de caractères | Nom de l'option. <br><br>**Cette propriété est disponible en lecture seulement**. |
| Valeur | Tous types | Valeur de l'option. <br><br>**Cette propriété est disponible en lecture seulement**. |
| ValeurAssociée | Tous types | Valeur associée de l'option. <br><br>**Cette propriété est disponible en lecture seulement**. |

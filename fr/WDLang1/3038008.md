


## MenuMarquage (Fonction)

***En anglais : MenuIsMarked***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de savoir si la marque ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Marque.gif) est présente devant l'option de menu (menu contextuel ou déroulant).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple
<a class="notetitle" target="_blank" href="$DOC$=1000003038008&name=menumarquage_fonction&product=WD">Voir des exemples supplémentaires</a>

```wl
// L'option "OPT_BarreIcone" est-elle marquée ?
ResMarquage = MenuMarquage(OPT_BarreIcone)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = MenuMarquage([<Fenêtre>, ] <Option de menu>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'option de menu spécifiée est marquée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Fenêtre> : (Nom de fenêtre optionnel)`**

Nom de la fenêtre contenant l'option de menu. 
Si ce paramètre n'est pas spécifié, la fonction **MenuMarquage** manipulera la fenêtre en cours.

**`<Option de menu> : (Nom d'option)`**

Nom de l'option de menu à manipuler. Ce nom est défini dans la fenêtre de description de l'option de menu. 
Ce nom doit être spécifié sans guillemets. Cette option peut également correspondre à la chaîne des lettres d'appel à utiliser pour accéder à l'option.

<a name="NOTE0_1"></a>

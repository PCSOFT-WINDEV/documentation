
## SysNotificationCompatible (Fonction)

***En anglais : SysNotificationCompatible***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Vérifie si le système d'exploitation supporte les notifications interactives.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Vérifie si le système d'exploitation supporte les notifications interactives
SI SysNotificationCompatible() ALORS
	Info("Le système supporte les notifications interactives.")
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = SysNotificationCompatible()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le système supporte les notifications interactives.

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.







- ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=WDGestionNotificationsWindows.gif) ***Exemples didactiques (WINDEV)*** : **WD Gestion Notifications Windows** <br>A partir de Windows 10, les « toast » notifications (celles qui apparaissent à droite dans le bureau) sont devenues interactives, avec possibilité d’y mettre des boutons, des zones de saisie et des combos. <br>Cet exemple présente la gestion de ces notifications.

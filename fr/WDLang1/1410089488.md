


## &lt;Liste de Kanban&gt;.SupprimeCarte (Fonction)

***En anglais : &lt;Kanban list&gt;.DeleteCard***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime une carte d'un champ Kanban ou d'un champ Liste de Kanban.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
PROCEDURE SupprimeCarteAvecToast(Carte est un kbCarte)
LISTE_A_Faire.SupprimeCarte(Carte.Priorité)
ToastAffiche("Tâche supprimée de la liste A")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Supprimer une carte identifiée par son indice

`<Champ Liste de Kanban>.SupprimeCarte(<Indice>)`
---

**`<Champ Liste de Kanban> : (Nom de champ)`**

Nom du champ Liste de Kanban à manipuler.

**`<Indice> : (Entier)`**

Indice de la carte à supprimer.


<a name="SYNTAXE2"></a>

### Supprimer une carte identifié par une variable kbCarte

`<Champ manipulé>.SupprimeCarte(<Carte>)`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom d'un champ Liste de Kanban. 

**`<Carte> : (Variable de type kbCarte)`**

Nom de la variable de type [kbCarte](../WDLang1/1410089167.md) correspondant à la carte à supprimer.

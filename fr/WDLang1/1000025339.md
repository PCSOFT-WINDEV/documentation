
## Procédure WLangage appelée par la fonction AjouteLien



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Procédure (également appelée "Callback") appelée par la fonction [AjouteLien](../WDLang1/1000019038.md). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// == Déclarations globales de la fenêtre == 
tabCibleLien est un tableau de chaînes
```

```wl
// == Clic sur le champ Bouton "Ajouter une note" ==
RangLien est un entier
Destination est une chaîne

// Demande une note à associer au lien
SI Saisie("Saisissez une note associée à ce lien : ", Destination) = 1 ALORS

	// Conserve la note dans un tableau
	RangLien = Ajoute(tabCibleLien, Destination)

	// Ajoute le lien dans le champ de saisie
	AjouteLien(SAI_Saisie, SAI_Saisie.Curseur, ...
		SAI_Saisie.FinCurseur, ClicSurLien, RangLien)
FIN
```

```wl
// == Procédure ClicSurLien ==
PROCEDURE ClicSurLien(RangLien est un entier)
InfoConstruit("La note associée au lien %1 est : %2", RangLien, ...
		tabCibleLien[RangLien])
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`AjouteLien_Callback(<Paramètre de la procédure>)`
---

**`<Paramètre de la procédure> : (Variant)`**

Paramètre passé à la procédure lors d'un clic sur le lien. Ce paramètre a été spécifié dans la syntaxe d'appel de la fonction [AjouteLien](../WDLang1/1000019038.md).




## MenuEtat (Fonction)

***En anglais : MenuState***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Identifie l'état d'une option de menu (menu contextuel ou déroulant) : active, inactive ou invisible.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Etat de l'option "OPT_BarreIcone" 
ResEtat = MenuEtat(OPT_BarreIcone)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = MenuEtat([<Fenêtre>, ] <Option de menu>)`
---

**`<Résultat> : (Constante)`**

Etat actuel de l'option de menu :


|   |   |
| --- | --- |
| *Actif* | Option de menu active. |
| *Grisé* | Option de menu grisée. |
| *Invisible* | Option de menu invisible. |



**`<Fenêtre> : (Nom de fenêtre optionnel)`**

Nom de la fenêtre contenant l'option de menu. 
Si ce paramètre n'est pas spécifié, la fonction **MenuEtat** manipulera la fenêtre en cours.

**`<Option de menu> : (Nom d'option)`**

Nom de l'option de menu à manipuler. Ce nom est défini dans la fenêtre de description de l'option de menu. Ce nom doit être spécifié sans guillemets.

<a name="NOTE0_1"></a>

## Remarques

```txt
MenuEtat(<Option de menu>)
```
est équivalent à la propriété [Etat](../Proprietes/2510039.md) : 

```txt
<Option de menu>.Etat
```

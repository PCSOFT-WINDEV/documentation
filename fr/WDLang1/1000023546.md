
## TableurSupprimeImage (Fonction)

***En anglais : SpreadsheetDeleteImage***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime une image dans la feuille courante d'un champ Tableur. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ajoute l'image logo.png entre A2 et B5
Id est chaîne = TableurAjouteImage(TBLR_Tableur, "c:\temp\logo.png", "A2", "B5") 

// Surprime l'image logo.png 
TableurSupprimeImage(TBLR_Tableur, Id)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`TableurSupprimeImage(<Champ Tableur> , <Identifiant>)`
---

**`<Champ Tableur> : (Nom de champ)`**

Nom du champ Tableur à manipuler.

**`<Identifiant> : (Chaîne de caractères)`**

Identifiant de l'image à supprimer dans la feuille courante. Cet identifiant est connu via les fonctions [TableurAjouteImage](../WDLang1/1000023335.md) et [TableurListeImage](../WDLang1/1000023547.md).

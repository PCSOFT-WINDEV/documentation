


## &lt;Champ d'affichage HTML&gt;.ValeurChamp (Fonction)

***En anglais : &lt;HTML Display control&gt;.ControlValue***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de connaître ou modifier un champ d'un formulaire HTML présent dans un champ d'affichage HTML.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI HTM_Affichage.ValeurChamp("LOGIN") <> "" ALORS
	HTM_Affichage.ValeurChamp("LOGIN", RéseauUtilisateur())
FIN
```

```wl
HTM_Sites.ValeurChamp("input4")
SI ErreurDétectée ALORS
	Trace(ErreurInfo()) // ex: Le champ <"input4"> n'existe pas dans la page.
	RETOUR
SINON
	HTM_Sites.ValeurChamp("input4", sValeurChamp)
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître la valeur d'un champ présent dans un formulaire HTML

`<Résultat> = <Champ d'affichage HTML>.ValeurChamp(<Champ de la page>)`
---

**`<Résultat> : (Chaîne de caractères)`**



- Valeur du champ présent dans le champ d'affichage HTML, 

- Chaîne vide ("") en cas d'erreur (par exemple si le champ n'est pas trouvé dans la page). Dans ce cas, la variable [ErreurDétectée](../WDLang1/3087001.md) vaut <u><u><u><u>Vrai</u></u></u></u>. Il est possible de connaître le détail de l'erreur grâce à la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Champ d'affichage HTML> : (Nom de champ)`**

Nom du champ d'affichage HTML manipulé.

**`<Champ de la page> : (Chaîne de caractères)`**

Nom du champ présent dans la page HTML affichée par le champ d'affichage HTML.


<a name="SYNTAXE2"></a>

### Modifier la valeur d'un champ présent dans un formulaire HTML

`<Champ d'affichage HTML>.ValeurChamp(<Champ de la page> , <Valeur>)`
---

**`<Champ d'affichage HTML> : (Nom de champ)`**

Nom du champ d'affichage HTML manipulé.

**`<Champ de la page> : (Chaîne de caractères)`**

Nom du champ présent dans la page HTML affichée par le champ d'affichage HTML. Si le champ n'est pas trouvé, une erreur non fatale est levée.

**`<Valeur> : (Chaîne de caractères)`**

Valeur du champ à modifier.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- **Important** : Pour que cette fonction puisse s'exécuter dans le contexte de la page HTML affichée dans le champ d'affichage HTML, il est nécessaire que le champ soit accessible en Javascript dans la page. Il ne doit pas y avoir de restrictions d'accès sur cette page.

- Cette fonction permet de piloter la page Web affichée dans un champ d'affichage HTML.

- Le champ est recherché par son nom puis par son ID. Le nom du champ (ou son ID) peut être connu en éditant le code source de la page HTML.

- Attention : les noms HTML sont sensibles à la casse.

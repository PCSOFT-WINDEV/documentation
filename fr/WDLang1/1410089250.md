
## EtatsUnis.VérifieFEI (Fonction)

***En anglais : UnitedStates.CheckFEI***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Teste la validité d'un numéro FEI (numéro d'identification d'établissement d'entreprise aux Etats-Unis).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Test d'une variable
MonNuméroFEI est une chaîne = "3003591836"
EtatsUnis.VérifieFEI(MonNuméroFEI)	// renvoie Vrai

// Test d'un champ
SAI_NuméroFEI = "ABC"
EtatsUnis.VérifieFEI(SAI_NuméroFEI)	// renvoie Faux
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = EtatsUnis.VérifieFEI(<Numéro à vérifier>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'élément spécifié contient un numéro FEI valide.

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Numéro à vérifier> : (Nom de champ ou variable de type chaîne)`**

Numéro FEI à vérifier. Ce paramètre peut correspondre : 

- au nom d'une variable de type chaîne.

- au nom d'un champ.

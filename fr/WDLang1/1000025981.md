


## EditeurHTMLRécupère (Fonction)

***En anglais : HTMLEditorGet***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Récupère en mode asynchrone la valeur d'une des propriétés d'un champ Editeur HTML. 

**Remarque** : Cette fonction est notamment utile pour les applications Android et iOS pour lesquelles les propriétés WLangage [Body](../Proprietes/1000025914.md), [Valeur](../Proprietes/2510130.md) et [TexteSansFormat](../Proprietes/1000019118.md) ne sont pas disponibles (car utilisables uniquement en mode synchrone). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Sauvegarde la saisie dans un fichier
EditeurHTMLRécupère(EDHTML_MonEditeur, ehrValeur, EditeurHTMLRécupère_Callback)
PROCEDURE INTERNE EditeurHTMLRécupère_Callback(valeur)
	fSauveTexte("sauvegarde.html", valeur)
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`EditeurHTMLRécupère(<Champ Editeur HTML> , <Propriété> , <Procédure WLangage>)`
---

**`<Champ Editeur HTML> : (Nom de champ)`**

Nom du champ Editeur HTML manipulé. 

**`<Propriété> : (Constante de type Entier)`**

Propriété pour laquelle la valeur doit être récupérée ; 



|   |   |
| --- | --- |
| *ehrBody* | Permet d'obtenir le code HTML du body.<br><br>Cette constante est équivalente à la propriété [Body](../Proprietes/1000025914.md). |
| *ehrTexteSansFormat* | Permet d'obtenir le texte contenu dans le champ sans les balises HTML. <br><br>Cette constante est équivalente à la propriété [TexteSansFormat](../Proprietes/1000019118.md). |
| *ehrValeur* | Permet d'obtenir la valeur du champ. <br><br>Cette constante est équivalente à la propriété [Valeur](../Proprietes/2510130.md). |



**`<Procédure WLangage> : (Nom de procédure)`**

Procédure WLangage (également nommée "callback") exécutée pour obtenir la valeur recherchée. Cette procédure est de la forme : 

```txt
PROCEDURE <Nom de la procédure>(Valeur)
```
où &lt;Valeur&gt; est la valeur recherchée. Dans ce paramètre, les caractères sont encodés en UTF8.

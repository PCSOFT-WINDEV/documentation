
## FinValRemboursement (Fonction)

***En anglais : FinRepayVal***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le montant de chaque remboursement périodique pour un investissement à remboursement et à taux d'intérêt constants.

**Remarque** : La valeur renvoyée par **FinValRemboursement** comprend le remboursement principal et les intérêts.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple
<a class="notetitle" target="_blank" href="$DOC$=1000003050039&name=finvalremboursement_fonction&product=WD">Voir des exemples supplémentaires</a>

```wl
// Montant des remboursements périodiques. Le capital initial est 100 000 Euros.
// Les remboursement sont effectués chaque mois pendant 2 ans.
// Le taux d'intérêt est de 8%.
ResValRemboursement = FinValRemboursement(100000, 24, 8)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = FinValRemboursement([<Capital initial> [, <Nombre de périodes> [, <Taux>]]])`
---

**`<Résultat> : (Réel ou monétaire)`**

Montant des remboursements périodiques.

**`<Capital initial> : (Réel ou monétaire optionnel)`**

Capital initial (0 par défaut). Ce paramètre indique le montant sur lequel le calcul doit être effectué.

Dans le cas "classique" d'un emprunteur, ce capital est emprunté et correspond donc à une valeur négative.

Si ce paramètre est positif, le calcul est réalisé pour un prêteur. La valeur retournée est négative : cette valeur correspond à la somme à verser.

**`<Nombre de périodes> : (Entier optionnel)`**

Nombre total de périodes pour l'opération (1 par défaut).

**`<Taux> : (Réel optionnel)`**

Taux d'intérêt de l'investissement exprimé en pourcentage (1 par défaut). Par exemple, pour un taux d'investissement de 5%, ce paramètre sera égal à 5.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Unité temporelle
<a name="unite_temporelle_ELTPARAGRAPHE000083"></a>Les calculs peuvent être réalisés en prenant en compte des mois ou des années. Il suffit que les paramètres &lt;Taux&gt; et &lt;Période&gt; correspondent à la même unité temporelle.

Ainsi, pour des remboursements mensuels d'un emprunt sur une durée de quatre ans au taux d'intérêt annuel de 12%, utilisez :

- &lt;Taux&gt; = 12% / 12,

- &lt;Nombre de périodes&gt; = 4\*12




Si, pour le même emprunt, vos remboursements sont annuels, utilisez :

- &lt;Taux&gt; = 12%,

- &lt;Nombre de période&gt; = 4




Pour tous les paramètres, les décaissements sont représentés par un nombre négatif alors que les encaissements sont représentés par un nombre positif.


<a name="NOTE0_2"></a>




### Gestion des erreurs
<a name="gestion_des_erreurs_ELTPARAGRAPHE000104"></a>**Attention** : La fonction **FinValRemboursement** ne retourne pas de code d'erreur. Pour savoir si cette fonction a généré des erreurs, utilisez la fonction [FinErreur](../WDLang1/3050071.md). Le libellé de l'erreur peut être connu grâce à la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.




## EnModeEmulateurWindowsPhone (Fonction)

***En anglais : InWindowsPhoneEmulatorMode***

<DIV class="specObsolete">
	<IMG style="float:left;margin-right:10px;" src="https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=warning_big.png"><B>Avertissement</B><BR>
	A partir de la version <B>21</B>,  cette fonctionnalité est conservée par compatibilité. 
</DIV><a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Identifie si le test est exécuté grâce à l'émulateur Windows Phone ou non.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI EnModeEmulateurWindowsPhone() = Vrai ALORS
	// Code spécifique à l'émulateur Windows Phone
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = EnModeEmulateurWindowsPhone()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le code est exécuté dans l'émulateur Windows Phone, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Exécution des traitements contenant des erreurs de compilation
<a name="execution_des_traitements_contenant_des_erreurs_compilation_ELTPARAGRAPHE000212"></a>Par défaut, le mécanisme de sécurité du WLangage se déclenche uniquement si la ligne contenant l'erreur de compilation est exécutée.

Par exemple, si une fonction interdite dans le produit en cours est présente dans une boucle SI, le mécanisme de sécurité du WLangage se déclenchera uniquement si la ligne contenant cette fonction s'exécute.
<a name="NOTE0_2"></a>


### Disponibilité de la fonction
<a name="disponibilite_fonction_ELTPARAGRAPHE000224"></a>La fonction **EnModeEmulateurWindowsPhone** est disponible dans tous les produits pour permettre l'écriture d'un code multi-produits. Des fonctions ou procédures utilisant la fonction **EnModeEmulateurWindowsPhone** peuvent être partagées entre plusieurs produits, sans que des erreurs de compilation soient générées.

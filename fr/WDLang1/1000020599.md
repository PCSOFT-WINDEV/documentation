


## SélecteurInsère (Fonction)

***En anglais : RadioButtonInsert***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Insère une nouvelle option dans un champ Sélecteur.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Insère une option à la position 2 
SélecteurInsère(SEL_Options, 2, "Lire un livre sur une tablette")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = SélecteurInsère(<Champ Sélecteur> , <Indice de l'élément> , <Libellé> [, <Sous-libellé> [, <Image>]])`
---

**`<Résultat> : (Entier)`**



- Indice de l‘option insérée. 

- -1 en cas d'erreur. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Champ Sélecteur> : (Nom de champ)`**

Nom du champ Sélecteur dans lequel l'option doit être insérée.

**`<Indice de l'élément> : (Entier)`**

Indice de l'élément à insérer. 

Si ce paramètre :

- est supérieur au nombre d'options présentes dans le sélecteur, l'option est ajoutée après la dernière option du champ Sélecteur. 

- est égal à 1 ou 0, l'élément est ajouté avant la première option du champ Sélecteur.




**`<Libellé> : (Chaîne de caractères)`**

Libellé de l'option à ajouter.

**`<Sous-libellé> : (Chaîne de caractères optionnelle)`**

Sous libellé de l'option à ajouter. 
Si ce paramètre n'est pas spécifié, l'option ajoutée n'a pas de sous-libellé. 
Si ce paramètre est spécifié, le sous-libellé sera affiché même si le sélecteur utilise un mode d'affichage sans sous-libellé.


**`<Image> : (Chaîne de caractères optionnelle)`**

Image associée à l'option à ajouter. 
Si ce paramètre n'est pas spécifié, l'option ajoutée n'a pas d'image.
Si ce paramètre est spécifié, l'image sera affichée uniquement si le sélecteur utilise un mode d'affichage "Style Vista" (avec image).




<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Les caractéristiques de l'option ajoutée peuvent également être modifiées grâce aux propriétés suivantes utilisées sur l'option : 

	- [Libellé](../Proprietes/2510053.md) pour modifier le libellé. 

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) [SousLibellé](../Proprietes/1000017232.md) pour modifier le sous-libellé. 

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) [Image](../Proprietes/2510034.md) pour modifier l'image. 




- Lors de l'insertion d'options dans un champ Sélecteur, la taille du champ n'est pas modifiée. Si le champ n'est pas suffisamment grand, les options ajoutées pourront ne pas être visibles. Il est possible d'augmenter la taille du champ avec les propriétés [Largeur](../Proprietes/2510059.md) et [Hauteur](../Proprietes/2510050.md).

- Pour ajouter une option à la suite des options existantes, utilisez la fonction [SélecteurAjoute](../WDLang1/1000020598.md). 

- Lors de l'insertion d'une option : 

	- l'option insérée a pour indice &lt;Indice de l'élément&gt;. 

	- si le sélecteur utilise la numérotation des options, les indices des options suivant l'option insérée sont automatiquement renumérotées. 




- **Utilisation des valeurs renvoyées** : Si le sélecteur utilise les valeurs renvoyées, il est possible de définir la valeur renvoyée par l'option insérée grâce à la propriété [ValeurRenvoyée](../Proprietes/2510134.md). 
	Par exemple : 
	
	```wl
	nIndice est un entier
	nIndice = SélecteurInsère(SEL_Option, 2, "Lire un livre sur une tablette")
	SEL_Option[nIndice].ValeurRenvoyée = "Tablette"
	// Sélection de la nouvelle option
	SEL_Option = "Tablette"
	```

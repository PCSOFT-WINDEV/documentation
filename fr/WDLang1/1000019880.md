
## fConstruitChemin (Fonction)

***En anglais : fBuildPath***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Construit un chemin de fichier complet à partir du répertoire et du nom court d'un fichier. Cette fonction permet notamment d'obtenir un nom de fichier valide sans se préoccuper de savoir si des séparateurs sont présents dans le chemin ('/' ou '\\') ou de leur sens selon la plateforme. 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
sLog est une chaîne
sLog = fConstruitChemin(fRepExe(), CLIENT.NOM, ".log")

Trace(fConstruitChemin("C:\Mes documents", "Monfichier", "fic"))
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = fConstruitChemin(<Chemin> , <Nom court> [, <Extension>])`
---

**`<Résultat> : (Chaîne de caractères)`**

Chemin de fichier valide. 

**`<Chemin> : (Chaîne de caractères)`**

Répertoire du fichier. Ce répertoire peut avoir ou non des caractères '\\' dans le chemin. 

**`<Nom court> : (Chaîne de caractères)`**

Nom court du fichier. 

**`<Extension> : (Chaîne de caractères optionnelle)`**

Extension du fichier. 
Si le '.' n'est pas précisé au début de l'extension, celui-ci est automatiquement ajouté. 
Si ce paramètre n'est pas précisé, l'extension est considérée comme étant incluse dans le paramètre &lt;Nom court&gt;.

## Remarques


### Equivalence
<a name="equivalence_ELTPARAGRAPHE000080"></a>Le code suivant : 

```txt
fConstruitChemin(<Répertoire>, <Fichier>, <Extension>)
```


remplace le code suivant : 


```txt
ComplèteRep(<Répertoire>) + <Fichier> + ["."] + <Extension>
```

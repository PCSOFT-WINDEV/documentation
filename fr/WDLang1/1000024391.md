


## &lt;Traitement de texte&gt;.RedoOccurrence (Fonction)

***En anglais : &lt;Word Processing&gt;.RedoCount***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le nombre d'actions annulées qui peuvent être refaites sur un champ Traitement de texte.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affiche le nombre d'actions annulées et qui peuvent être refaites
Info(TT_MonDocument.RedoOccurrence())
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Champ Traitement de texte>.RedoOccurrence()`
---

**`<Résultat> : (Entier)`**

Nombre d'actions annulées qui peuvent être refaites.

**`<Champ Traitement de texte> : (Nom de champ)`**

Nom du champ Traitement de texte à manipuler.

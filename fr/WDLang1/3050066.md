
## ArcSinus (Fonction)

***En anglais : ArcSin***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie l'arc sinus d'une valeur numérique (entière ou réelle).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Récupération de l'arc sinus de 0.5
ResArcSinus = ArcSinus(0.5)
// Renvoie 30
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = ArcSinus(<Valeur numérique>)`
---

**`<Résultat> : (Réel)`**

Arc sinus (angle en degré du sinus) de la valeur numérique spécifiée, compris dans l'intervalle [-90,90].

**`<Valeur numérique> : (Entier ou réel)`**

Valeur numérique (sinus) à manipuler. Cette valeur doit être comprise dans l'intervalle [-1,1]. Dans le cas contraire, une erreur du WLangage est générée.

Les constantes de gestion des champs Onglet sont les suivantes : 

| Constante | Description | Utilisée par les fonctions |
| --- | --- | --- |
| **voletActif** | Le volet existe et il est attaché à son onglet. | [OngletEtat](../WDLang1/1000021637.md) |
| **voletFlottant** | Le volet existe et il est détaché. | [OngletEtat](../WDLang1/1000021637.md) |
| **voletInexistant** | Le volet n'existe pas ou il a été fermé. | [OngletEtat](../WDLang1/1000021637.md) |

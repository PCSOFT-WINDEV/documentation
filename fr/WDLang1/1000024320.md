
## CertificatDeConfianceAjoute (Fonction)

***En anglais : TrustedCertificateAdd***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ajoute un certificat dans la liste des autorités racine de confiance utilisées pour valider les chaînes de certification. 

Attention : Cet ajout est réalisé pour l'application en cours. Aucun ajout n'est réalisé dans les magasins de certificats de Windows.

Remarque : Cette fonction impacte : 

- Les requêtes HTTPS utilisant des variables de type [httpRequête](../WDLang3/1000021158.md),

- La gestion des fichiers par FTPS, 

- Les sockets SSL, 

- MQTT via SSL,

- la vérification des signatures (2ème syntaxe de la fonction [CertificatVérifieChaîne](../WDLang1/1000019298.md)).



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
CertificatDeConfianceAjoute("MonCA.pem")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = CertificatDeConfianceAjoute(<Certificat>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'ajout a été effectué, 

- <u><u><u><u>Faux</u></u></u></u> en cas d'erreur. 




**`<Certificat> : (Chaîne de caractères)`**

Chemin du fichier de certificat à ajouter dans la liste des autorités racine de confiance. 

Remarque : Ce fichier peut être intégré dans la librairie de l'application (fichier WDL).



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Les certificats au format cer, pem, der et p12 sont supportés.

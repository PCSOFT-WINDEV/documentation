


## SaisieAssistéeParamètre (Fonction)

***En anglais : AssistedInputConfigure***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de paramétrer le filtre et l'ouverture de la saisie assistée.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SaisieAssistéeSupprimeTout(SAI_Direction)
SaisieAssistéeAjoute(SAI_Direction, "Gauche")
SaisieAssistéeAjoute(SAI_Direction, "Droite")
SaisieAssistéeAjoute(SAI_Direction, "Centre")

// Pour désactiver le filtre 
SaisieAssistéeParamètre(SAI_Direction, saFiltre, filtreAucun)

// Activer le filtre "Contient" qui s'appliquera à chaque frappe 
// sur les 3 valeurs précédemment ajoutées
SaisieAssistéeParamètre(SAI_Direction, saFiltre, filtreContient)

// La saisie assistée ne s'ouvrira qu'au prochain appel à la fonction SaisieAssistéeOuvre
SaisieAssistéeParamètre(SAI_Direction, saOuvertureAutomatique, Faux)
SaisieAssistéeOuvre(SAI_Direction)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`SaisieAssistéeParamètre(<Champ de saisie> , <Paramètre> , <Valeur>)`
---

**`<Champ de saisie> : (Nom de champ)`**

Nom du champ de saisie à manipuler. L'option "Saisie assistée sur liaison fichier" est automatiquement activée sur ce champ (onglet "Détail" de la fenêtre de description du champ).

**`<Paramètre> : (Constante de type Entier)`**

Paramètre à modifier : 


|   |   |
| --- | --- |
| *saDélai* | Délai avant le déclenchement de la complétion. Ce délai est exprimé en millisecondes. Par défaut, le délai est de 200 millisecondes. <br><br>![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Cette constante n'est pas disponible. |
| *saFiltre* | Filtre à utiliser pour la saisie assistée. Par défaut, le filtre utilisé est de type "Commence par". <br><br>Ce filtre peut correspondre à une des valeurs suivantes : <br><br><br>\   \   \<br>\ --- \ --- \<br>\ *filtreAucun* \ Désactive le filtre automatique utilisé par la saisie assistée. Le filtre doit être effectué manuellement. \<br>\ *filtreCommencePar* \ Filtre de type "Commence par". \<br>\ *filtreContient* \ Filtre de type "Contient". \<br>\ *filtreDifférent* \ Filtre de type "Est différent de". <br><br>![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Cette constante n'est pas disponible. \<br>\ *filtreEgal* \ Filtre de type "Est égal à".<br><br>![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Cette constante n'est pas disponible. \<br>\ *filtreInférieur* \ Filtre de type "Est inférieur".<br><br>![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Cette constante n'est pas disponible. \<br>\ *filtreInférieurOuEgal* \ Filtre de type "Est inférieur ou égal".<br><br>![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Cette constante n'est pas disponible. \<br>\ *filtreNeCommencePasPar* \ Filtre de type "Ne commence pas par".<br><br>![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Cette constante n'est pas disponible. \<br>\ *filtreNeContientPas* \ Filtre de type "Ne contient pas".<br><br>![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Cette constante n'est pas disponible. \<br>\ *filtreNeTerminePasPar* \ Filtre de type "Ne se termine pas par".<br><br>![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Cette constante n'est pas disponible. \<br>\ *filtreSupérieur* \ Filtre de type "Est supérieur".<br><br>![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Cette constante n'est pas disponible. \<br>\ *filtreSupérieurOuEgal* \ Filtre de type "Est supérieur ou égal".<br><br>![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Cette constante n'est pas disponible. \<br>\ *filtreTerminePar* \ Filtre de type "Se termine par".<br><br>![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Cette constante n'est pas disponible. \<br><br> |
| *saLargeurListe* | Largeur de la liste des choix (exprimée en pixels). Par défaut, cette largeur correspond à la largeur de la zone de saisie du champ.<br><br> |
| *saOuvertureAutomatique* | Paramètre l'ouverture automatique de la liste des choix : <br><br>- <u><u><u><u>Vrai</u></u></u></u> pour ouvrir la liste automatiquement, <br><br>- <u><u><u><u>Faux</u></u></u></u> pour ne pas ouvrir la liste automatiquement. Dans ce cas, la liste peut être ouverte par la fonction [SaisieAssistéeOuvre](../WDLang1/1000020048.md). <br><br><br>Par défaut, la liste s'ouvre automatiquement. |
| *saSansCasse* | Prise en compte de la casse et des accents : <br><br>- <u><u><u><u>Vrai</u></u></u></u> pour ne pas prendre en compte la casse et les accents. <br><br>- <u><u><u><u>Faux</u></u></u></u> pour prendre en compte la casse et les accents. <br><br><br>Par défaut, la casse et les accents ne sont pas pris en compte. |
| *saTailleMin* | Nombre de caractères en saisie avant d'ouvrir la liste des choix. Par défaut, la liste des choix est ouverte après la saisie du premier caractère. |



**`<Valeur> : (Type correspondant à la valeur à modifier)`**

Nouvelle valeur du paramètre. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Si la fonction **SaisieAssistéeParamètre** est appelée sur un champ de saisie, l'option "Saisie assistée sur liaison fichier" est automatiquement activée sur ce champ (onglet "Détail" de la fenêtre de description du champ).

- La liste avec les choix ne s'ouvre pas immédiatement. Il faut : 

	- soit que l'utilisateur ait saisi une valeur correspondant au début d'un des choix de la liste, 

	- soit appeler la fonction [SaisieAssistéeOuvre](../WDLang1/1000020048.md). 




- Si le champ de saisie est lié à un fichier de données : 

	- seul le filtre par défaut (filtre de type "Commence par") est disponible. 

	- la rubrique liée doit être clé (optimisation des performances du parcours).

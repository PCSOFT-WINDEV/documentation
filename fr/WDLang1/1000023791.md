


## &lt;Chrono&gt;.RAZ (Fonction)

***En anglais : &lt;Chrono&gt;.Reset***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Arrête et remet un chronomètre existant à zéro. La valeur de la durée écoulée est perdue.  




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonChrono est un chrono
...

// -- Code de clic du bouton BTN_Reinit

// Réinitialise le chronomètre MonChrono
MonChrono.RAZ()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Chronomètre>.RAZ()`
---

**`<Chronomètre> : (Variable de type Chrono)`**

Nom de la variable de type [Chrono](../WDLang1/1000023560.md) correspondant au chronomètre à utiliser.

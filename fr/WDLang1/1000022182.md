


## &lt;Tableau croisé dynamique&gt;.Déroule (Fonction)

***En anglais : &lt;Pivot table&gt;.Expand***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Déroule toutes les données d'un champ Tableau croisé dynamique selon un entête de ligne ou de colonne.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Déroule tous les trimestres
TCD_Ventes.Déroule(COL_DateCommande_Trimestre)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Champ Tableau croisé dynamique>.Déroule(<Entête à dérouler>)`
---

**`<Champ Tableau croisé dynamique> : (Nom de champ)`**

Nom du champ Tableau croisé dynamique à manipuler.

**`<Entête à dérouler> : (Chaîne de caractères)`**

Nom de l'entête de ligne ou de colonne à dérouler.

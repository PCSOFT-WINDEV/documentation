


## TableauInsèreRéférence (Fonction)

***En anglais : ArrayInsertReference***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ajoute la référence d'un objet à la position donnée dans un tableau à une dimension.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
UnTableau est un tableau de UneClasse
UnObjet est un UneClasse
TableauInsèreRéférence(UnTableau, IndiceInsertion, UnObjet)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`TableauInsèreRéférence(<Tableau WLangage> , <Indice> , <Objet>)`
---

**`<Tableau WLangage> : (Tableau)`**

Nom de la variable de type Tableau à manipuler. Ce tableau doit être un tableau à une dimension.

**`<Indice> : (Entier)`**

Indice du tableau où l'élément doit être inséré. 

**`<Objet> : (Objet)`**

Nom de l'objet pour lequel une référence est ajoutée dans le tableau.

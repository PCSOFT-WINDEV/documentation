
## Pâques (Fonction)

***En anglais : Easter***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la date de Pâques pour l'année spécifiée dans le calendrier chrétien occidental.

**Rappel** : La date de Pâques correspond au premier dimanche qui suit la première pleine lune de printemps.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
Pâques(2006) // Renvoie "20060416"
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = Pâques(<Année>)`
---

**`<Résultat> : (Chaîne de caractères)`**



- Date de Pâques pour l'année spécifiée au  format AAAAMMJJ, 

- Chaîne vide ("") en cas d'erreur.




**`<Année> : (Chaîne de caractères, variable de type Date ou entier)`**

Année pour laquelle la date de Pâques doit être calculée. L'année de calcul doit être supérieure à 1582, année d'introduction dans le calendrier grégorien et inférieure à 9999.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
La date calculée n'est pas valable pour le calendrier orthodoxe.

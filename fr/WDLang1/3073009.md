
## SysHandleValide (Fonction)

***En anglais : SysValidHandle***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Teste la validité d'un handle de fenêtre.

**Remarque** : Un handle n'est plus valide si la fenêtre correspondante a été fermée par exemple.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Validité du handle de la fenêtre "Microsoft Word - Ventes 2000.DOC"
ResHandle = SysFenHandle("Microsoft Word - Ventes 2000.DOC")
ResHandleValide = SysHandleValide(ResHandle)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = SysHandleValide(<Handle de la fenêtre>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le handle spécifié est valide, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Handle de la fenêtre> : (Entier)`**

Handle de la fenêtre à tester.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Handle d'une fenêtre
<a name="handle_une_fenetre_ELTPARAGRAPHE000043"></a>Le handle d'une fenêtre est obtenu :

- avec la fonction [Handle](../WDLang1/3015005.md) pour une fenêtre WINDEV.

- avec la fonction [SysFenHandle](../WDLang1/3073007.md) pour une fenêtre non créée avec WINDEV.

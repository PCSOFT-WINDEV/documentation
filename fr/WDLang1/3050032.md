
## MatLitLigne (Fonction)

***En anglais : MatReadLine***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Lit la valeur de tous les éléments d'une ligne d'une matrice.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple
<a class="notetitle" target="_blank" href="$DOC$=1000003050032&name=matlitligne_fonction&product=WD">Voir des exemples supplémentaires</a>

```wl
// Lecture des éléments de la ligne 5
ResLectureLigne = MatLitLigne("MaMatrice", 5)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = MatLitLigne(<Matrice> , <Ligne à lire> [, <Taille totale d'un élément> [, <Nombre de décimales> [, <Format d'un élément>]]])`
---

**`<Résultat> : (Chaîne de caractères)`**

Valeur de tous les éléments de la ligne spécifiée, sous la forme :

```txt
<Valeur Elément 1> + TAB + <Valeur Elément 2> + TAB + ...
+ TAB + <Valeur Elément N>
```


**`<Matrice> : (Chaîne de caractères)`**

Nom de la matrice à lire, défini avec la fonction [MatCrée](../WDLang1/3050058.md).

**`<Ligne à lire> : (Entier)`**

Numéro de la ligne à lire.

**`<Taille totale d'un élément> : (Entier optionnel)`**

Taille totale d'un élément à manipuler. Cette taille comprend le signe, le séparateur des milliers, le point décimal, l'exposant, les parties entières et décimales de la valeur de l'élément.

- Si l'élément a plus de &lt;Taille totale d'un élément&gt; chiffres, la &lt;Taille totale d'un élément&gt; est augmentée.

- Si l'élément a moins de &lt;Taille totale élément&gt; chiffres, la chaîne de caractères &lt;Résultat&gt; est complétée par des espaces.

- Si la &lt;Taille totale d'un élément&gt; n'est pas précisée, la valeur de l'élément lu aura la taille de sa partie significative.

- La &lt;Taille totale d'un élément&gt; ne doit pas dépasser 100.

- Si la &lt;Taille totale d'un élément&gt; vaut 0, la valeur de l'élément aura la taille de sa partie significative.




**`<Nombre de décimales> : (Entier optionnel)`**

Nombre de décimales à lire. Ce paramètre commence toujours par un point ("."). La dernière décimale sera arrondie. Ce paramètre n'a d'effet que pour les notations flottantes ("f") et exponentielles ("e" et "E"). Si le &lt;Nombre de décimales&gt; est plus grand que la &lt;Taille totale d'un élément&gt;, le paramètre &lt;Taille totale d'un élément&gt; n'est pas pris en compte.

| *&lt;Nombre de décimales&gt;* | *Type "d" (entier)* | *Type "e" ou "E" (notation exponentielle) et type "f" (notation flottante)* |
| --- | --- | --- |
| .0 | - | 0 |
| .N | - | N (si N est supérieur aux nombres de décimales, la partie décimale sera complétée par des zéros) |
| (pas précisé) | - | 6 |



**`<Format d'un élément> : (Caractère optionnel)`**

Format de présentation d'un élément :

- d : la valeur de l'élément est sous la forme d'un entier signé. Les chiffres après la virgule sont tronqués et le paramètre &lt;Nombre de décimales&gt; n'est pas pris en compte.

- e ou E : notation exponentielle avec 1 chiffre avant le point décimal, &lt;Nombre de décimales&gt; chiffres après le point décimal et au moins 2 chiffres dans l'exposant.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Existence de la matrice
<a name="existence_matrice_ELTPARAGRAPHE000121"></a>L'existence de la matrice n'est pas vérifiée. Pour vérifier l'existence d'une matrice, utilisez la fonction [MatExiste](../WDLang1/3050076.md).


<a name="NOTE0_2"></a>




### Taille de &lt;Résultat&gt;
<a name="taille_ELTPARAGRAPHE000133"></a>**Attention** : La taille de la chaîne de caractères obtenue dans &lt;Résultat&gt; est limitée à 260 caractères. Si la ligne de la matrice contient plus de caractères, la chaîne contenue dans &lt;Résultat&gt; ne contiendra que les 260 premiers caractères.


<a name="NOTE0_3"></a>




### Extraction d'une valeur d'un élément de &lt;Résultat&gt;
<a name="extraction_une_valeur_element_ELTPARAGRAPHE000142"></a>Pour extraire une ou plusieurs valeurs d'éléments de la chaîne de caractères &lt;Résultat&gt;, utilisez la fonction [ExtraitChaîne](../WDLang1/3024017.md).


<a name="NOTE0_4"></a>




### Gestion des erreurs
<a name="gestion_des_erreurs_ELTPARAGRAPHE000154"></a>**Attention** : La fonction **MatLitLigne** ne retourne pas de code d'erreur. Pour savoir si des erreurs ont été générées par la lecture des éléments d'une ligne, utilisez la fonction [MatErreur](../WDLang1/3050044.md). Le libellé de l'erreur peut être connu grâce à la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.




## DuréeVersChaîne (Fonction)

***En anglais : DurationToString***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Convertit une durée en une chaîne de caractères au format spécifié. 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Heures de début et de fin
HeureDébut est une Heure = "125324"
HeureFin est une Heure = "140852"
// Calcul de la durée
MaDurée est une Durée = HeureFin - HeureDébut
// Formatage de la durée pour obtenir un résultat au même format que DateHeureDifférence
Res = DuréeVersChaîne(MaDurée, duréeCentième) // Res = "001152800"
// Formatage personnalisé
Res = DuréeVersChaîne(MaDurée, "J-HH:MM")     // Res = "0-01:15"
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = DuréeVersChaîne(<Durée> , <Format>)`
---

**`<Résultat> : (Chaîne de caractères)`**

Chaîne de caractères au format spécifié.

**`<Durée> : (Variable de type Durée)`**



- Variable de type Durée contenant la durée au format "+JHHMMSSCCC" (J correspond au nombre de jours, HH aux heures, MM aux minutes, SS aux secondes et CCC aux millisecondes).

- Indication directe de la durée (par exemple 1 j 2 h 3 min 10 s 10 ms). 




**`<Format> : (Chaîne de caractères ou constante)`**

Format à appliquer à la durée. Ce paramètre peut correspondre :

- soit à une chaîne de caractères (une phrase par exemple). Les caractères spécifiques représentant les différents éléments de la durée (J, HH, MM, SS ou CCC) seront automatiquement remplacés par leur valeur identifiée dans la chaîne &lt;Durée&gt;. Dans cette chaîne, les caractères utilisés sont les suivants :

	- + permet de prendre en compte les chiffres négatifs,

	- J représente le nombre de jours,

	- HH représente le nombre d'heures,

	- MM représente le nombre de minutes,

	- SS représente le nombre de secondes,

	- CCC représente le nombre de millisecondes.




- soit à une des constantes suivantes :
	


|   |   |
| --- | --- |
| *duréeCentième* | Format utilisé : "+JHHMMSSCC". Ce format permet de récupérer le résultat dans le format utilisé par la fonction [DateHeureDifférence](../WDLang1/3027020.md). |
| *duréeMillième* | Format utilisé : "+JHHMMSSCCC" |








<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
L'affectation directe &lt;Chaîne&gt; = &lt;Durée&gt; est équivalente à l'utilisation de la fonction **DuréeVersChaîne** en utilisant le format "+JHHMMSSCCC".




## ChaîneVersHeure (Fonction)

***En anglais : StringToTime***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Transforme une heure sous forme de chaîne de caractères en une heure au format HHMMSSCC, utilisable par une variable de type [Heure](../Motscles/1514068.md).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonHeure est une Heure
MonHeure = ChaîneVersHeure("12:10:25:78")	 // MonHeure = "12102578"
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = ChaîneVersHeure(<Chaîne> [, <Format>])`
---

**`<Résultat> : (Chaîne de caractères ou variable de type Heure)`**



- Heure au format HHMMSSCC (HH correspond aux heures, MM aux minutes, SS aux secondes et CC aux centièmes de seconde).

- Chaîne vide ("") en cas de problème.




**`<Chaîne> : (Chaîne de caractères)`**

Heure au format spécifié dans le paramètre &lt;Format&gt;.

**`<Format> : (Chaîne de caractères optionnelle)`**

Format de la chaîne de caractères à convertir en heure. Il est possible d'utiliser :

- Les caractères spécifiques représentant les différents éléments de l'heure (HH, MM, SS, CC ou AP) seront automatiquement remplacés par leur valeur identifiée dans la chaîne &lt;Chaîne&gt;. 
	Dans cette chaîne :

	- HH représente le nombre d'heures,

	- MM représente le nombre de minutes,

	- SS représente le nombre de secondes,

	- CC représente le nombre de centièmes de seconde,

	- AP représente "AM" ou "PM" (si "AP" est présent dans le format, l'heure est affichée sur 12 heures).




- La constante suivante :
	


|   |   |
| --- | --- |
| *maskHeureSystème* | Permet d'utiliser directement le masque utilisé par l'heure système. Ce masque dépend des options de configuration du projet. <br>Pour afficher les options de configuration des heures pour le projet : <br><br>	1. Sous le volet "Projet", dans le groupe "Projet", cliquez sur "Description". La fenêtre de description du projet s'affiche. <br><br>	2. Dans l'onglet "Langues", sélectionnez l'onglet "Heure". Il est possible d'utiliser les options de Windows ou des paramètres spécifiques. <br><br><br> |




Si ce paramètre n'est pas spécifié, le format utilisé par défaut est HH:MM:SS:CC.

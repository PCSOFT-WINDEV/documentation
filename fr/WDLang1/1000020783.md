


## Fonctions de gestion des champs TimeLine (syntaxe préfixée)
			



Les fonctions de gestion des champs TimeLine (via une syntaxe préfixée) sont les suivantes :



|   |   |
| --- | --- |
| [&lt;TimeLine&gt;.Affiche](../WDLang1/1000020775.md) | Réinitialise le contenu d'un champ TimeLine à partir de sa source de données (fichier de données HFSQL par exemple). |
| [&lt;TimeLine&gt;.AjouteEvénement](../WDLang1/1000020776.md) | Ajoute un événement dans un champ TimeLine. |
| [&lt;TimeLine&gt;.AjoutePiste](../WDLang1/1000020777.md) | Ajoute une nouvelle piste dans un champ TimeLine. |
| [&lt;TimeLine&gt;.ChangeMode](../WDLang1/1000020778.md) | Modifie la période (c'est-à-dire le nombre de secondes, millisecondes, microsecondes, ...) affichée par un champ TimeLine. |
| [&lt;TimeLine&gt;.ListeEvénement](../WDLang1/1000020779.md) | Renvoie : <br><br>- la liste des événements compris entre deux moments donnés dans un champ TimeLine, <br><br>- tous les événements d'un champ TimeLine,<br><br>- un événement particulier (sélectionné ou survolé). <br><br><br> |
| [&lt;TimeLine&gt;.SupprimeEvénement](../WDLang1/1000020780.md) | Supprime un événement dans un champ TimeLine. |
| [&lt;TimeLine&gt;.SupprimePiste](../WDLang1/1000020781.md) | Supprime une piste dans un champ TimeLine. |
| [&lt;TimeLine&gt;.SupprimeTout](../WDLang1/1000020782.md) | Supprime tous les événements et toutes les pistes d'un champ TimeLine. |

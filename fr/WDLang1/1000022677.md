
## ActionUndo (Fonction)

***En anglais : ActionUndo***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Annule la dernière action effectuée par l'utilisateur final. Cette fonction est équivalente à un "Undo" (Ctrl + Z) de l'utilisateur.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Bouton [Annuler]
ActionUndo()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`ActionUndo([<Fenêtre à manipuler>])`
---

**`<Fenêtre à manipuler> : (Nom de fenêtre optionnel)`**

Nom de la fenêtre ou de la fenêtre interne à manipuler. Si ce paramètre n'est pas précisé, la fenêtre en cours est utilisée.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Conditions d'utilisation
<a name="conditions_utilisation_ELTPARAGRAPHE000032"></a>

- Si aucune action n'est disponible pour l'annulation, la fonction **ActionUndo** ne fait rien.

- Si la dernière action est une action personnalisée exécutée par la fonction [ActionDo](../WDLang1/1000022676.md), la procédure définie par la fonction [ActionDéclareUndo](../WDLang1/1000022678.md) est exécutée.

- L'action annulée peut être rétablie : 

	- avec la combinaison de touches Ctrl + Y, 

	- par programmation grâce à la fonction [ActionRedo](../WDLang1/1000022608.md).


## ChaîneFormate (Fonction)

***En anglais : StringFormat***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Formate une chaîne de caractères selon les options choisies.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MaChaîne1 est une chaîne = "Le développement facile"
MaChaîne1 = ChaîneFormate(MaChaîne1, ccMajuscule)
// MaChaîne1 contient "LE DÉVELOPPEMENT FACILE"
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = ChaîneFormate(<Chaîne> , <Options>)`
---

**`<Résultat> : (Chaîne de caractères)`**

Chaîne formatée selon les critères choisis.

**`<Chaîne> : (Chaîne de caractères)`**

Chaîne de caractères à formater.


![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Il est possible d'utiliser une chaîne de caractères au format Unicode.

![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) Il n'est pas possible d'utiliser des chaînes de caractères au format Unicode.

**`<Options> : (Constante ou combinaison de constantes)`**

Options de formatage :               


|   |   |
| --- | --- |
| *ccMajuscule* | Passe la chaîne de caractères en majuscules (y compris les caractères accentués). |
| *ccMinuscule* | Passe la chaîne de caractères en minuscules. |
| *ccNormal* | Aucun formatage n'est effectué. |
| *ccSansAccent* | Format sans accent. |
| *ccSansEspace* | Les caractères suivants sont supprimés au début et à la fin de la chaîne de caractères :<br><br>- espace (caractère 32)<br><br>- tabulation (caractère 9)<br><br>- retour chariot (caractère 13) et saut de ligne (caractères 10)<br><br>- caractères de contrôle 11 et 12<br><br><br> |
| *ccSansEspaceIntérieur* | Les caractères suivants sont supprimés à l'intérieur de la chaîne de caractères :<br><br>- espace (caractère 32)<br><br>- tabulation (caractère 9)<br><br>- retour chariot (caractère 13) et saut de ligne (caractères 10)<br><br>- caractères de contrôle 11 et 12<br><br><br><br><br>![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Cette constante n'est pas disponible. |
| *ccSansPonctuationNiEspace* | Enlève la ponctuation et les espaces. |

## Remarques




### Fonction ChaîneFormate et Unicode
<a name="fonction_docparampagetitleshort_unicode_ELTPARAGRAPHE000147"></a>Si la chaîne passée en paramètre est au format ANSI, le résultat sera au format ANSI.

Si la chaîne passée en paramètre est au format UNICODE, le résultat sera au format UNICODE.

**Remarque** : Si le résultat de la fonction **ChaîneFormate** utilisée sur une chaîne ANSI est affecté à une chaîne UNICODE (et inversement), la conversion ANSI/UNICODE sera implicitement réalisée.

Pour plus de détails, consultez [Gestion de l'UNICODE](../WDLang1/3024038.md).

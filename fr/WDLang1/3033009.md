
## eDeviseAffParDéfaut (Fonction)

***En anglais : eDefaultViewCurrency***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Identifie ou modifie la devise d'affichage utilisée par défaut.

**Remarques** :

- La fonction **eDeviseAffParDéfaut** détermine la devise d'affichage uniquement pour les champs "Monétaire + Euro" dont la devise d'affichage correspond à l'option "Devise par défaut" sous l'éditeur de fenêtres / pages.

- Par défaut, la devise d'affichage utilisée est l'Euro.





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Modification de la devise d'affichage utilisée par défaut
ResDeviseAffParDéfaut = eDeviseAffParDéfaut(eDEM)
// La devise affichée par défaut est à présent le Mark Allemand
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Modifier la devise d'affichage utilisée par défaut

`<Résultat> = eDeviseAffParDéfaut(<Nouvelle devise affichée par défaut>)`
---

**`<Résultat> : (Entier)`**

Identifiant de la devise d'affichage utilisée par défaut avant modification.

**`<Nouvelle devise affichée par défaut> : (Entier ou constante)`**

Identifiant ou constante associée à la nouvelle devise affichée par défaut. 

Cette devise sera prise en compte : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) lors de la prochaine ouverture de fenêtre (réalisée avec les fonctions [Ouvre](../WDLang1/3038035.md) ou [Utilise](../WDLang1/3038044.md)).





<a name="SYNTAXE2"></a>

### Identifier la devise d'affichage utilisée par défaut

`<Résultat> = eDeviseAffParDéfaut()`
---

**`<Résultat> : (Entier)`**

Identifiant de la devise d'affichage utilisée par défaut.

## Remarques




### Table des changes
<a name="table_des_changes_ELTPARAGRAPHE000085"></a>Une ***table des changes ***est conservée en mémoire. A chaque monnaie sont associés :

- Un numéro de devise permettant d'identifier la devise par programmation.

- Une constante permettant d'identifier la devise par programmation.

- Un libellé (7 caractères maximum) : ce libellé est affiché dans le champ "Monétaire + Euro" si le masque de saisie le permet (le signe $ est indiqué).

- Un taux de change. Ce taux de change correspond au taux de change de la monnaie par rapport à l'Euro.

La première devise (ayant pour identifiant 0) est la monnaie prise pour pivot lors des conversions.


Dans cette table des changes, il est possible :

- d'ajouter de nouvelles devises en modifiant si nécessaire les devises personnalisables ("devises utilisateur" de 16 à 25) avec la fonction [eInitDevise](../WDLang1/3033003.md),

- de spécifier le taux de change de chaque devise grâce à la fonction [eInitDevise](../WDLang1/3033003.md).




**Attention** : Les monnaies ayant un taux de change nul n'apparaîtront pas dans le menu contextuel du champ de type "Monétaire + Euro". Pour définir les taux de change de ces devises, vous devez les redéfinir dans le code d'initialisation du projet avec la fonction [eInitDevise](../WDLang1/3033003.md).


## MarkdownVersPDF (Fonction)

***En anglais : MarkdownToPDF***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Convertit une chaîne (ou un buffer) au format Markdown en un buffer contenant un document PDF.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
sTexteMarkdown est une chaîne
// Convertit le texte au format Markdown en un document PDF 
DocPDF est un Buffer = MarkdownVersPDF(sTexteMarkdown)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = MarkdownVersPDF(<Texte au format Markdown> [, <Format page> [, <Paramètres du fichier PDF>]])`
---

**`<Résultat> : (Buffer)`**



- Buffer contenant le document PDF résultat de la conversion.

- Buffer vide en cas d'erreur. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Texte au format Markdown> : (Chaîne de caractères)`**

Texte au format Markdown à convertir.

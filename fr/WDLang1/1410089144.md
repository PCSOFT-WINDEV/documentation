


## KanbanDéplaceCarte (Fonction)

***En anglais : KanbanMoveCard***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Déplace une carte dans un champ Kanban.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Déplace la première carte de la liste de Kanban LISTE_A_Revoir
// à la fin de la liste LISTE_A_Revoir 
Carte1 est un kbCarte <- LISTE_A_Revoir.Carte[1]
KanbanDéplaceCarte(KANBAN_MonKanban, Carte1, LISTE_A_Revoir.Carte.Occurrence)

// Déplace la première carte de la liste LISTE_A_Revoir dans une autre liste
Carte2 est un kbCarte <- LISTE_A_Revoir.Carte[1]
KanbanDéplaceCarte(KANBAN_MonKanban, Carte2, LISTE_Terminé)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Déplacer une carte à une position donnée de la même liste de Kanban

`<Résultat> = KanbanDéplaceCarte(<Champ Kanban> , <Carte> , <Position>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la carte est déplacée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 




**`<Champ Kanban> : (Nom de champ)`**

Nom du champ Kanban à manipuler. 

**`<Carte> : (Variable de type kbCarte)`**

Nom de la variable de type [kbCarte](../WDLang1/1410089167.md) correspondant à la carte à déplacer. 

**`<Position> : (Entier)`**

Nouvel indice de la carte dans la liste de Kanban. La carte est déplacée vers le haut ou vers le bas dans sa liste. 


<a name="SYNTAXE2"></a>

### Déplacer une carte à une position donnée dans une liste de Kanban spécifique

`<Résultat> = KanbanDéplaceCarte(<Champ Kanban> , <Carte> , <Liste de Kanban> [, <Position>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la carte est déplacée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.  




**`<Champ Kanban> : (Nom de champ)`**

Nom du champ Kanban à manipuler.

**`<Carte> : (Variable de type kbCarte)`**

Nom de la variable de type [kbCarte](../WDLang1/1410089167.md) correspondant à la carte à déplacer. 

**`<Liste de Kanban> : (Nom de champ)`**

Nom de la liste de Kanban dans laquelle la carte doit être déplacée. 

**`<Position> : (Entier optionnel)`**

Nouvel indice de la carte. Si ce paramètre n'est pas précisé, la carte est positionnée à la fin de la liste de Kanban spécifiée.

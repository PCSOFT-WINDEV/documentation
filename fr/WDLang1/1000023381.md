


## PicSauve (Fonction)

***En anglais : PicSave***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Sauve l'image présente : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) dans un champ Editeur d'images.

- dans une variable de type [WDPic](../WDLang1/1000023503.md). 





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 
```wl
// Demande de sauvegarde de l'image présente dans un champ Editeur d'images
SI PicSauve(EI_MonEditeur) = Faux ALORS
	Erreur(ErreurInfo())
FIN
```

```wl
MonImageWDPic est un WDPic
...
// Demande de sauvegarde de l'image
SI PicSauve(MonImageWDPic, "C:\temp\test.png") = Faux ALORS
	Erreur(ErreurInfo())
FIN
```



<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = PicSauve(<Image> [, <Fichier>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'image a été créée,

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire (erreur, annulation du sélecteur de fichier, ...). En cas d'erreur, la variable [ErreurDétectée](../WDLang1/3087001.md) est à <u><u><u><u>Vrai</u></u></u></u> et la fonction [ErreurInfo](../WDLang1/3013008.md) permet de connaître le détail de l'erreur.  




**`<Image> : (Nom de champ ou variable de type WDPic)`**

Image à sauver. Cette image peut correspondre : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) au nom du champ Editeur d'images à manipuler.

- au nom d'une variable de type [WDPic](../WDLang1/1000023503.md) à manipuler.




**`<Fichier> : (Chaîne de caractères optionnelle)`**

Nom et chemin du fichier image à sauver. L'extension du fichier détermine le type de l'image à sauver.
Si ce paramètre n'est pas précisé : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Dans le cas d'un champ Editeur d'images : l'action effectuée correspond à la fonctionnalité d'enregistrement de l'éditeur d'images (clic sur la disquette dans le ruban).
	Si l'image n'a jamais été enregistrée, le sélecteur de fichier s'affiche. 

- Dans le cas d'une variable de type [WDPic](../WDLang1/1000023503.md), une erreur fatale est affichée. 







- ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=LetypeWDPIC.gif) ***Exemples unitaires (WINDEV)*** : **Le type WDPIC** <br>Cet exemple montre l'utilisation du type WLangage WDPic.<br>Ce type WLangage permet de modifier par programmation les images sources sauvegardées avec l'éditeur d'images.<br>Il permet notamment de manipuler les calques.

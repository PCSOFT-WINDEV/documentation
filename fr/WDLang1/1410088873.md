


## DiagrammeVersSVG (Fonction)

***En anglais : DiagramToSVG***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Exporte un diagramme vers une image vectorielle au format SVG.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
soit bufSVG = DiagrammeVersSVG(EDIAG_MonDiagramme)
SI ErreurDétectée ALORS
	Erreur()
	RETOUR
FIN
fSauveBuffer("exportDuDiagramme.svg", bufSVG)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = DiagrammeVersSVG(<Diagramme>)`
---

**`<Résultat> : (Buffer)`**

Image vectorielle au format SVG (chaîne encodée en UTF-8). 

**`<Diagramme> : (Champ Editeur de diagramme ou variable de type Diagramme)`**

Diagramme à manipuler. Ce document correspond :

- au nom d'un champ Editeur de diagrammes.

- à une variable de type [Diagramme](../WDLang1/1410088055.md).






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
La variable de type Buffer renvoyée peur être sauvegardée dans un fichier (fonction [fSauveBuffer](../WDLang1/1000019412.md)) ou directement affichée dans un autre champ,


## France.VérifieSIRET (Fonction)

***En anglais : France.CheckSIRET***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Vérifie la validité d'un numéro SIRET (système d'identification du répertoire des établissements).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Test d'une variable
MonNuméroSIRET est une chaîne = "33031827000064"
France.VérifieSIRET(MonNuméroSIRET) // Renvoie Vrai

// Test d'un champ
SAI_NuméroSIRET = "ABC"
France.VérifieSIRET(SAI_NuméroSIRET) // Renvoie Faux
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = France.VérifieSIRET(<Numéro à vérifier>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'élément spécifié contient un numéro SIRET valide.

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Numéro à vérifier> : (Nom de champ ou variable de type chaîne)`**

Numéro SIRET à vérifier. Ce paramètre peut correspondre : 

- au nom d'une variable de type chaîne.

- au nom d'un champ.

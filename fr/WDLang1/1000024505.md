


## &lt;Variable docFragment&gt;.Ajoute (Fonction)

***En anglais : &lt;docFragment variable&gt;.Add***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ajoute à la fin d'une variable de type [docFragment](../WDLang1/1000022482.md) : 

- un élément, 

- un texte, 

- une image.



<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE2"></a>

`<Résultat> = <Fragment>.Ajoute(<Objet à ajouter>)`
---

**`<Résultat> : (Variable de type docFragment)`**

Variable de type [docFragment](../WDLang1/1000022482.md) contenant le fragment ajouté. 

**`<Fragment> : (Variable de type docFragment)`**

Nom de la variable de type [docFragment](../WDLang1/1000022482.md) à manipuler.

**`<Objet à ajouter> : (Type de l'objet ajouté)`**

Objet à ajouter : 

- Elément : Nom de la variable de type [docElément](../WDLang1/1000022484.md) correspondant à l'élément à ajouter.

- Texte sous forme de chaîne de caractères : Texte à ajouter.
	Il est possible d'utiliser les constantes suivantes dans le texte à ajouter pour gérer les différents sauts possibles : 
	


|   |   |
| --- | --- |
| *docSautDeColonne* | Ajoute un saut de colonne dans une section multicolonne. Si la section n'est pas multicolonne, un saut de page est ajouté. |
| *docSautDeLigne* | Ajoute un saut de ligne. |
| *docSautDePage* | Ajoute un saut de page. |
| *docSautDeParagraphe* | Ajoute un saut de paragraphe. |



- Image : Nom de la variable de type [Image](../WDLang1/1000019650.md) à ajouter.

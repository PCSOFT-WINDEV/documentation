
## MatInverse (Fonction)

***En anglais : MatInvert***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Inverse une matrice carrée.

**Remarque** : La matrice inverse ne peut être calculée que :

- à partir d'une matrice carrée (nombre de lignes égal au nombre de colonnes)

- si le déterminant (renvoyé par la fonction [MatDéterminant](../WDLang1/3050045.md)) n'est pas nul.







<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple
<a class="notetitle" target="_blank" href="$DOC$=1000003050075&name=matinverse_fonction&product=WD">Voir des exemples supplémentaires</a>

```wl
// Inversion d'une matrice carrée
ResInversion = MatInverse("MaMatrice", "MaMatriceInversée")
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = MatInverse(<Matrice à inverser> , <Matrice inversée>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'inversion a été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour connaître le libellé de l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.




**`<Matrice à inverser> : (Chaîne de caractères)`**

Nom de la matrice à inverser, défini avec la fonction [MatCrée](../WDLang1/3050058.md).

**`<Matrice inversée> : (Chaîne de caractères)`**

Nom de la matrice inversée, automatiquement créée lors de l'inversion.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Existence de la matrice inversée
<a name="existence_matrice_inversee_ELTPARAGRAPHE000098"></a>L'existence de la matrice n'est pas vérifiée. Si la matrice existait auparavant, cette matrice est écrasée. Pour vérifier l'existence d'une matrice, utilisez la fonction [MatExiste](../WDLang1/3050076.md).


<a name="NOTE0_2"></a>




### Matrices identiques
<a name="matrices_identiques_ELTPARAGRAPHE000110"></a>Une erreur du WLangage est générée si le paramètre &lt;Matrice à inverser&gt; et le paramètre &lt;Matrice inversée&gt; sont identiques.

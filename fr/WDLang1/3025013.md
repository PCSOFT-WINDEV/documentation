
## ChampCauseEntréeSortie (Fonction)

***En anglais : ControlInOutReason***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Indique l'origine de l'entrée dans le champ.

**Attention** : Cette fonction doit être utilisée uniquement dans le code d'entrée ou de sortie du champ.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// -- Code d'entrée du champ SAI_SAISIE1
// Si entrée dans le champ par le clavier
SI ChampCauseEntréeSortie() = causeClavier ALORS
	// Le champ est remis à vide
	MoiMême = ""
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = ChampCauseEntréeSortie()`
---

**`<Résultat> : (Constante de type entier)`**

Origine de l'entrée dans le champ en cours de saisie :


|   |   |
| --- | --- |
| *causeClavier* | Entrée dans le champ par la touche TAB, un raccourci clavier ou navigation dans un champ Table. |
| *causeClic* | Entrée dans le champ par un clic de souris. |
| *causeCode* | Entrée dans le champ suite à l'utilisation de la fonction [DonneFocusEtRetourUtilisateur](../WDLang1/1410088107.md) ou équivalent. |
| *causeOuverture* | Entrée dans le champ après ouverture de la fenêtre. |




## dRotation (Fonction)

***En anglais : dRotation***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Effectue une rotation : 

- d'un champ Image, 

- dans une variable de type [Image](../WDLang1/1000019650.md).
	

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) dans une variable de type [picCalque](../WDLang1/1000024604.md).


![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=dRotation.gif)


![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Les dessins](../WDLang1/3029035.md). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Rotation du champ IMG_Image1 de 45 degrés
Résultat1 est un booléen
Résultat1 = dRotation(IMG_Image1, 45)
SI Résultat1 = Faux ALORS Erreur("L'opération a échoué")
```

<a name="SYNTAXE1"></a>

## Syntaxe

### Effectuer une rotation simple

`<Résultat> = dRotation(<Image> , <Angle de rotation> [, <Options>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la rotation a été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Image> : (Nom de champ, variable de type Image ou variable de type picCalque)`**

Image à manipuler. Cette image peut correspondre : 

- au nom d'un champ Image. 

- au nom d'une variable de type [Image](../WDLang1/1000019650.md).
	

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) au nom d'une variable de type [picCalque](../WDLang1/1000024604.md).




La rotation est effectuée par rapport au centre de l'image. L'image est directement modifiée.

**`<Angle de rotation> : (Réel)`**

Angle de rotation (en degrés). 

- Si l'angle est positif, l'image est tournée dans le sens horaire. 

- Si l'angle est négatif, l'image est tournée dans le sens anti-horaire.




**`<Options> : (Constante de type Entier)`**

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png)![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) Options de rotation : 


|   |   |
| --- | --- |
| *drAdapte* | L'image est tournée avec un agrandissement. Elle est ensuite réduite pour correspondre à la taille initiale de l'image. |
| *drDéfaut* | Agrandissement si nécessaire de l'image pour correspondre aux dimensions de l'image tournée.<br><br>**Constante utilisée par défaut pour les variables de type [Image](../WDLang1/1000019650.md) ou [picCalque](../WDLang1/1000024604.md)**. |
| *drSansAgrandissement* | L'image est tournée mais n'est pas agrandie : l'image peut être tronquée.<br><br>**Constante utilisée par défaut pour les champs Image**. |


![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Ce paramètre n'est pas disponible.

### Effectuer une rotation en modifiant le centre de rotation

`<Résultat> = dRotation(<Champ Image> , <Angle de rotation> , <X du centre de rotation> , <Y du centre de rotation> [, <Options>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la rotation a été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Champ Image> : (Nom de champ)`**

Nom du champ Image à manipuler. La rotation est effectuée par rapport au centre du champ Image. L'image présente dans le champ Image est modifiée.

**`<Angle de rotation> : (Réel)`**

Angle de rotation (en degrés).

**`<X du centre de rotation> : (Entier)`**

Abscisse en pixels du nouveau centre de rotation.

**`<Y du centre de rotation> : (Entier)`**

Ordonnée en pixels du nouveau centre de rotation. 

**`<Options> : (Constante de type Entier)`**

Options de rotation : 


|   |   |
| --- | --- |
| *drDéfaut* | Agrandissement si nécessaire de l'image pour correspondre aux dimensions de l'image tournée.<br><br>**Constante utilisée par défaut pour les variables de type [Image](../WDLang1/1000019650.md) ou [picCalque](../WDLang1/1000024604.md)**. |
| *drSansAgrandissement* | L'image est tournée mais n'est pas agrandie.<br><br>**Constante utilisée par défaut pour les champs Image**. |


![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Ce paramètre n'est pas disponible.

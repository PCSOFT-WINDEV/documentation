


## WDPic (Type de variable)

***En anglais : WDPic***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **WDPic** permet de définir toutes les caractéristiques avancées d'une image au format ".wdpic". Les caractéristiques de cette image peuvent être définies et modifiées à l'aide de différentes propriétés WLangage. 

Ce type permet notamment de : 

- rendre visibles/invisibles des calques par programmation (fonction [ImageCalqueVisible](../WDLang2/1000023504.md)). 

- manipuler les images au format WDPic ainsi que leurs calques.




**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affiche l'image d'un produit dans un champ Image
PROCEDURE AfficheProduit(ChampImage est un Champ, sNomWDPic est une chaîne, bPromo est un booléen)

MonImage est un WDPic
MonImage.NomFichier = sNomWDPic
ImageCalqueVisible(MonImage, "Calque Promo", bPromo) 

ChampImage.Valeur = MonImage
```

## Remarques
<a name="NOTE0_1"></a>


### Propriétés spécifiques à la description des variables de type WDPic
<a name="proprietes_specifiques_description_des_variables_type_wdpic_ELTPARAGRAPHE000046"></a>Les propriétés suivantes peuvent être utilisées pour manipuler une image au format "wdpic" :

| Nom de la propriété | Type manipulé | Effet |
| --- | --- | --- |
| Calque | Tableau de [picCalque](../WDLang1/1000024604.md) | Calques définis sur l'image au format "wdpic". Les calques sont identifiés par leurs noms. |
| Hauteur | Entier | Hauteur de l'image exprimée en pixels. <br><br>Par défaut, lors de l'affectation d'une image à la variable WDPic, correspond à la hauteur de l'image.<br><br>Cette propriété est disponible en lecture et en écriture. |
| Image |   | Image associée à la variable. <br><br>Cette image est mise à jour par exemple lors de l'affectation de la variable avec un champ Image.<br><br>Cette propriété est disponible en lecture et en écriture. |
| Largeur | Entier | Largeur de l'image exprimée en pixels. <br><br>Par défaut, lors de l'affectation d'une image à la variable WDPic, correspond à la largeur de l'image. <br><br>Cette propriété est disponible en lecture et en écriture. |
| NomFichier | Chaîne de caractères | Nom du fichier ".wdpic". |


## FenDonneHauteurAdaptée (Fonction)

***En anglais : WinGiveSuitableHeight***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la hauteur d'une fenêtre, adaptée au contenu des champs. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affiche le champ Table
TableAffiche(FEN_Prix.TABLE_Prix)
// Calcule la hauteur adaptée
nHauteurOK est un entier
nHauteurOK = FenDonneHauteurAdaptée(FEN_Prix)
// Redimensionne la fenêtre
SI nHauteurOK < 500 ALORS
	FEN_Prix..Hauteur = nHauteurOK
SINON
	FEN_Prix..Hauteur = 500
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = FenDonneHauteurAdaptée([<Fenêtre>])`
---

**`<Résultat> : (Entier)`**

Hauteur adaptée au contenu des champs pour optimiser l'affichage des champs dans la fenêtre. Cette hauteur est exprimée en pixels. 

**`<Fenêtre> : (Nom de fenêtre optionnel)`**

Nom de la fenêtre à manipuler. Si ce paramètre n'est pas précisé, la fenêtre en cours sera manipulée.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Le calcul de la hauteur adaptée d'une fenêtre est possible uniquement si la fenêtre contient un des champs suivants ancré en hauteur :

- Champ Liste,

- Champ Table,

- Champ Zone répétée,

- Champ Arbre,

- Champ Liste image,

- Champ de saisie multiligne.

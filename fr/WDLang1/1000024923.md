


## &lt;Variable WDPic&gt;.Sauve (Fonction)

***En anglais : &lt;WDPic variable&gt;.Save***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Sauve l'image présente dans une variable de type [WDPic](../WDLang1/1000023503.md). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonImageWDPic est un WDPic
...
// Demande de sauvegarde de l'image
SI MonImageWDPic.Sauve("C:\temp\test.png") = Faux ALORS
	Erreur(ErreurInfo())
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Image WDPic>.Sauve(<Fichier>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'image a été créée,

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire (erreur, annulation du sélecteur de fichier, ...). En cas d'erreur, la variable [ErreurDétectée](../WDLang1/3087001.md) est à <u><u><u><u>Vrai</u></u></u></u> et la fonction [ErreurInfo](../WDLang1/3013008.md) permet de connaître le détail de l'erreur.




**`<Image WDPic> : (Variable de type WDPic)`**

Nom de la variable de type [WDPic](../WDLang1/1000023503.md) contenant l'image à manipuler.

**`<Fichier> : (Chaîne de caractères)`**

Nom et chemin du fichier image à sauver. L'extension du fichier détermine le type de l'image à sauver.

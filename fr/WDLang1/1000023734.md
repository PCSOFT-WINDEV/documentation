


## &lt;Tableur&gt;.Sauve (Fonction)

***En anglais : &lt;Spreadsheet&gt;.Save***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Sauve le contenu d'un champ Tableur dans un fichier au format XLSX.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Sauvegarde du fichier en cours
TBLR_Tableur.Sauve("Classeur1.XLSX")
```

```wl
// Chargement du fichier
TBLR_SansNom1.Charge("test.xlsx")
// Modification du fichier
TBLR_SansNom1["A","1"] = "modifié"
// Sauvegarde sous un autre nom
// Le fichier "text1.xlsx" existe déjà
SI TBLR_SansNom1.Sauve("test1.xlsx") = Faux ALORS
	// Avant la version 200051, le fichier xlsx existant est écrasé. 
	// A partir de la version 200051, le fichier xlsx existant n'est pas modifié. 
	// La fonction renvoie Faux et une erreur 
	// Pour écraser le fichier, utilisez la constante tblrEcrasementAutorisé
	Erreur("Fichier non sauvegardé " + ErreurInfo()
SINON
	Info("Fichier sauvegardé")
FIN
```


## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Champ Tableur>.Sauve(<Fichier à sauver> [, <Option> [, <Mot de passe>]])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le fichier du champ Tableur a été sauvé,

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. La fonction [ErreurInfo](../WDLang1/3013008.md) permet d'obtenir plus de détails sur l'erreur.




**`<Champ Tableur> : (Nom de champ)`**

Nom du champ Tableur à manipuler.

**`<Fichier à sauver> : (Chaîne de caractères)`**

Nom et chemin complet du fichier XLSX dans lequel le contenu du champ Tableur sera sauvé.

**`<Option> : (Constante optionnelle de type Entier)`**

Mode d'enregistrement du fichier si le fichier existe déjà : 


|   |   |
| --- | --- |
| 0 (ou rien)<br>(Comportement par défaut) | Si le fichier XLSX existe déjà et n'est pas ouvert (dans le champ Tableur ou une autre application), il n'est pas sauvé. |
| *tblrEcrasementAutorisé* | Si le fichier XLSX existe déjà et n'est pas ouvert (dans le champ Tableur ou une autre application), il est écrasé. |



**`<Mot de passe> : (Chaîne de caractères optionnelle)`**

Mot de passe à utiliser pour protéger le fichier XLSX en ouverture. Ce mot de passe devra être indiqué pour ouvrir le fichier XLSX.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Pour ouvrir le fichier ".xlsx" sauvé avec une autre application, il est nécessaire de le fermer avec la fonction [&lt;Tableur&gt;.Ferme](../WDLang1/1000023722.md). 

- Le fichier sauvé pourra être rechargé dans un champ Tableur grâce aux fonctions [&lt;Tableur&gt;.Charge](../WDLang1/1000023719.md) et [xlsOuvre](../WDLang5/3080017.md).

- Cette fonction est utilisable uniquement sur un champ Tableur présent dans une fenêtre. 

- La fonction **&lt;Tableur&gt;.Sauve** met à jour la propriété [CheminFichier](../Proprietes/1000023104.md) du champ Tableur en associant le nom du fichier chargé au champ Tableur.

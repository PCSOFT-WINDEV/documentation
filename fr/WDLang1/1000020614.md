


## &lt;Menu&gt;.Supprime (Fonction)

***En anglais : &lt;Menu&gt;.Delete***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime un menu ou une option de menu.

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Il est possible de supprimer des options de menu principal ou des options de menu contextuel.

![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) Seuls les menus, options et séparateurs ajoutés par programmation (avec les fonctions [&lt;Menu&gt;.AjouteMenu](../WDLang1/1000020592.md), [&lt;Menu&gt;.AjouteOption](../WDLang1/1000020593.md) et [&lt;Menu&gt;.AjouteSéparateur](../WDLang1/1000020603.md)) peuvent être supprimés.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Suppression de l'option "OPT_Quitter" du menu "MenuFichier"
MenuFichier.OPT_Quitter.Supprime()
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Supprimer un menu

`<Nom du menu>.Supprime()`
---

**`<Nom du menu> : (Nom de menu)`**

Nom du menu à supprimer.


<a name="SYNTAXE2"></a>

### Supprimer une option de menu

`<Nom de l'option>.Supprime()`
---

**`<Nom de l'option> : (Nom de menu)`**

Nom de l'option de menu à supprimer, au format suivant "&lt;Nom du menu&gt;.&lt;Nom de l'option&gt;".

Cette option sera définitivement supprimée du menu.


<a name="SYNTAXE3"></a>

### Supprimer une option de menu par son indice

`<Nom du menu>.Supprime(<Rang de l'option>)`
---

**`<Nom du menu> : (Nom de menu)`**

Nom du menu contenant l'option à supprimer. Il peut s'agir d'un menu principal ou d'un menu contextuel.

**`<Rang de l'option> : (Entier)`**

Rang de l'option à supprimer dans le menu (les rangs commencent à 1).

Cette option sera définitivement supprimée du menu.




## KanbanIndiceListe (Fonction)

***En anglais : KanbanListIndex***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie l'indice d'une liste de Kanban présente dans un champ Kanban. 

Remarque : La visibilité des listes n'est pas prise en compte. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
Indice est un entier 
Indice = KanbanIndiceListe(LISTE_A_revoir)
SI Indice = 1 ALORS
	KanbanAjouteListe(KANBAN_MonKanban, "A commencer")
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Récupérer l'indice d'une liste de Kanban (en précisant le nom du champ Kanban)

`<Résultat> = KanbanIndiceListe(<Champ Kanban> , <Liste de Kanban>)`
---

**`<Résultat> : (Entier)`**



- Indice de la liste de Kanban dans le champ Kanban, 

- 0 si la liste de Kanban n'existe pas.


Remarque : Cet indice correspond à l'indice d'édition.  

**`<Champ Kanban> : (Nom de champ)`**

Nom du champ Kanban manipulé. 

**`<Liste de Kanban> : (Nom de champ)`**

Nom du champ Liste de Kanban à manipuler. Ce nom peut correspondre : 

- au nom du champ Liste de Kanban défini sous l'éditeur, 

- au nom du champ Liste de Kanban défini par programmation.  





<a name="SYNTAXE2"></a>

### Récupérer l'indice d'une liste de Kanban

`<Résultat> = KanbanIndiceListe(<Liste de Kanban>)`
---

**`<Résultat> : (Entier)`**



- Indice de la liste de Kanban dans le champ Kanban, 

- 0 si la liste de Kanban n'existe pas. 


Remarque : Cet indice correspond à l'indice d'édition.

**`<Liste de Kanban> : (Nom de champ)`**

Nom du champ Liste de Kanban à manipuler. Ce nom peut correspondre : 

- au nom du champ Liste de Kanban défini sous l'éditeur, 

- au nom du champ Liste de Kanban défini par programmation.

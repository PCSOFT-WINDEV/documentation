


## &lt;Panneau dockable&gt;.Active (Fonction)

***En anglais : &lt;Dockable panel&gt;.Activate***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Rend un panneau actif lorsque plusieurs panneaux sont dockés entre eux.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Tableau permettant de connaître le nombre d'erreurs
tabErreurs est un tableau d'entier
... 
// Force l'affichage du panneau des erreurs si au moins 1 erreur
SI tabErreurs.Occurrence > 0 ALORS
	PADO_Erreurs.Active()
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Champ Panneau dockable>.Active()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le panneau a été affiché, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire (le panneau est déjà actif ou le panneau n'est pas docké avec d'autres panneaux).




**`<Champ Panneau dockable> : (Nom de champ)`**

Champ Panneau dockable à manipuler.




## &lt;Champ d'affichage HTML&gt;.ExécuteTraitementChamp (Fonction)

***En anglais : &lt;HTML Display control&gt;.ExecuteControlProcess***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Lance l'exécution d'un événement WLangage associé à un champ présent dans une page HTML affichée par une champ d'affichage HTML. Cette fonction permet de piloter une page Web, par exemple en déclenchant un clic sur un bouton dans un formulaire HTML. 

Remarque : L'événement est exécuté en Javascript.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
HTM_Affichage.ValeurChamp("LOGIN", sLogin)
HTM_Affichage.ValeurChamp("PASSWD", sMotDePasse)
HTM_Affichage.ExécuteTraitementChamp("BTN_OK", trtClic)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Champ d'affichage HTML>.ExécuteTraitementChamp(<Champ dans la page HTML> , <Evénement>)`
---

**`<Champ d'affichage HTML> : (Nom de champ)`**

Nom du champ d'affichage HTML manipulé.

**`<Champ dans la page HTML> : (Chaîne de caractères)`**

Nom du champ à manipuler dans la page HTML. Si le champ n'est pas trouvé, une erreur non fatale est déclenchée. 
Remarque : Seuls les attributs "name" et "id" sont pris en compte.

**`<Evénement> : (Chaîne de caractères ou constante de type Entier)`**

Nom de l'événement (ou traitement) à exécuter. Ce nom peut correspondre : 

- à une chaîne de caractères contenant le nom du code à exécuter en Javascript (par exemple : 'onclick', 'onchange', 'onfocus', ...). 

- à une des constantes suivantes : 
	


|   |   |
| --- | --- |
| *trtClic* | Code de clic. |
|  *trtModification* | Code de modification. |

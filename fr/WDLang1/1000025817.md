
## GénèreLorem (Fonction)

***En anglais : GenerateLorem***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Génère un texte aléatoire de style "Lorem ipsum", utilisé par exemple pour remplir temporairement un champ avec du texte. Ce texte ne possède pas d'accents. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Remplissage avec un "Lorem ipsum" de 200 mots d'un champ de saisie 
// qui doit afficher la 4eme de couverture d'un livre

// Génération du texte de 200 mots
sTexteCouverture est une chaîne = GénèreLorem(200)

// Affichage dans le champ de saisie
SAI_couverture = sTexteCouverture
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = GénèreLorem(<Nombre de mots> [, <Graine>])`
---

**`<Résultat> : (Chaîne de caractères)`**



- Texte généré. 

- Chaîne vide ("") si le nombre de mots renseigné est de 0.




**`<Nombre de mots> : (Entier)`**

Nombre de mots à générer. Si ce nombre est égal à 0, le texte renvoyé correspond à une chaîne vide ("").

**`<Graine> : (Entier)`**

Permet de changer le texte généré. 
Pour chaque valeur de ce paramètre, un texte unique est généré. 
Si ce paramètre n'est pas précisé, le texte généré sera différent à chaque appel de la fonction.

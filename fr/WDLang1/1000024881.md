
## TableurListeCommentaire (Fonction)

***En anglais : SpreadsheetListComment***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Liste tous les commentaires de la feuille courante d'un champ Tableur. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Rendre tous les commentaires visibles
POUR TOUT Commentaire DE TableurListeCommentaire(TBLR_MonTableur)
	Commentaire..Visible = Vrai
FIN
```

```wl
// Supprimer tous les commentaires
POUR TOUT Commentaire DE TableurListeCommentaire(TBLR_MonTableur)
	Commentaire..Texte = ""
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = TableurListeCommentaire(<Champ Tableur>)`
---

**`<Résultat> : (Tableau)`**

Tableau contenant les différents commentaires avec leurs caractéristiques. 

**`<Champ Tableur> : (Nom de champ)`**

Nom du champ Tableur à manipuler. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- La modification d'un commentaire présent dans le tableau modifie également le commentaire dans le champ Tableur. 

- L'ajout ou la suppression d'un commentaire dans le tableau n'a aucun effet sur les commentaires présents dans le champ Tableur. 

- Si aucun commentaire n'existe dans la feuille, le tableau est vide.

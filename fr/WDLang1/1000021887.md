
## PremierJourDeLAnnée (Fonction)

***En anglais : FirstDayOfYear***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la date du premier jour de l'année.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affiche les clients créés depuis le début de l'année
HLitPremier(Client, Nom)
TANTQUE HEnDehors() = Faux
	// Traitement de l'enregistrement
	SI (Client.DateCreation >= PremierJourDeLAnnée()) ALORS
		Trace(Client.Nom)
	FIN
	HLitSuivant(Client, Nom)
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Obtenir le premier jour de l'année à partir d'une date

`<Résultat> = PremierJourDeLAnnée([<Date>])`
---

**`<Résultat> : (Chaîne de caractères ou variable de type Date)`**

Premier jour de l'année (au format AAAAMMJJ) pour la date spécifiée.

**`<Date> : (Chaîne de caractères, variable de type Date ou variable de type DateHeure optionnelle)`**

Date pour laquelle le premier jour de l'année doit être calculé. Cette date peut correspondre à : 

- une chaîne de caractères au format AAAAMMJJ (AAAA correspond à l'année, MM au mois et JJ au jour).  

- une chaîne de caractères au format AAAAMMJJHHmmSSCC (AAAA correspond à l'année, MM au mois, JJ au jour, HH aux heures, mm aux minutes, SS aux secondes et CC aux centièmes de seconde (utiliser CCC pour les millièmes de seconde)). Dans ce cas, seule la date est prise en compte. 

- une variable de type Date. 

- une variable de type DateHeure. Dans ce cas, seule la date est prise en compte.


Si ce paramètre n'est pas spécifié, la date du jour est utilisée par défaut.


<a name="SYNTAXE2"></a>

### Obtenir le premier jour de l'année à partir d'une année

`<Résultat> = PremierJourDeLAnnée(<Année>)`
---

**`<Résultat> : (Chaîne de caractères ou variable de type Date)`**

Premier jour de l'année (au format AAAAMMJJ).

**`<Année> : (Entier)`**

Année pour laquelle le premier jour est recherché.

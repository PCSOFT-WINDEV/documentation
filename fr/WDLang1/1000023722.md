


## &lt;Tableur&gt;.Ferme (Fonction)

***En anglais : &lt;Spreadsheet&gt;.Close***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ferme le document XLSX ouvert précédemment par la fonction [&lt;Tableur&gt;.Charge](../WDLang1/1000023719.md).
<a name="sample_code"></a>

## Exemple


```wl
TBLR_Tableur.Charge("doc1.xlsx")
TBLR_Tableur.Ferme()
LanceAppliAssociée("doc1.xlsx")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Champ Tableur>.Ferme()`
---

**`<Champ Tableur> : (Nom de champ)`**

Nom du champ Tableur à manipuler.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Par défaut, la fonction [&lt;Tableur&gt;.Charge](../WDLang1/1000023719.md) laisse le fichier XLSX ouvert en écriture. Lors de l'enregistrement, seules les cellules modifiées dans le champ Tableur sont écrasées. 

- Pour ouvrir le fichier ".xlxs" avec une autre application, il est nécessaire de le fermer avec la fonction **&lt;Tableur&gt;.Ferme**.

- Si aucun fichier ".xlsx" n'a été ouvert, la fonction **&lt;Tableur&gt;.Ferme** ne fait rien.

- Cette fonction est utilisable uniquement sur un champ Tableur présent dans une fenêtre.

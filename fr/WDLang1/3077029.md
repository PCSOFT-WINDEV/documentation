
## ThreadPause (Fonction)

***En anglais : ThreadPause***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Temporise le thread en cours pour la durée indiquée.






<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
ThreadPause(100)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`ThreadPause(<Temporisation>)`
---

**`<Temporisation> : (Entier)`**

Temporisation du thread (en centièmes de seconde). Ce paramètre peut correspondre à : 

- un entier correspondant au nombre de centièmes de seconde, 

- une variable de type Durée, 

- l'indication de la durée en clair (par exemple '1s' ou '10cs'). 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Il est conseillé d'utiliser la fonction **ThreadPause** à la place des fonctions [Multitâche](../WDLang1/3015004.md) ou [Temporisation](../WDLang1/3015012.md) lorsque plusieurs threads sont utilisés (y compris pour le thread principal de l'application) si la temporisation n'a pas besoin de traiter les actions de l'utilisateur.

- Dans une application de type Service, la fonction **ThreadPause** doit être remplacée par la fonction [ServiceTemporise](../WDLang1/1000018712.md).

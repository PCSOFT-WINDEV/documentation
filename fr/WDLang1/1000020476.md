
## Tronque (Fonction)

***En anglais : Truncate***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Tronque la partie droite d'une chaîne ou d'un buffer. Il est possible de préciser : 

- soit le nombre de caractères du résultat tronqué. 

- soit le nombre de caractères à tronquer. 





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
sChaîne est une chaîne = " Turlututu"

// On tronque la chaîne à 9 caractères
Trace(Tronque(sChaine, 9))	// Affiche " Turlutut"
Trace(Tronque(sChaine, 9, nombreDeCaractèresAConserver))	 // Affiche " Turlutut"

// On tronque la chaîne de 4 caractères
Trace(Tronque(sChaine, 4, nombreDeCaractèresASupprimer))	
// Affiche " Turlu"
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = Tronque(<Source> , <Nombre de caractères> [, <Type de troncature>])`
---

**`<Résultat> : (Chaîne de caractères)`**



- Résultat de la troncature.

- EOT si le paramètre &lt;Nombre de caractères&gt; est négatif. Une erreur du WLangage est générée (erreur non fatale). Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Source> : (Chaîne de caractères ou Buffer)`**

Elément à manipuler : Chaîne de caractères ou buffer. Cet élément n'est pas modifié.

**`<Nombre de caractères> : (Entier)`**

Correspond : 

- au nombre de caractères à conserver dans la chaîne &lt;Résultat&gt; (la constante *nombreDeCaractèresAConserver* est utilisée). 

- au nombre de caractères / octets à supprimer de &lt;Source&gt; (la constante *nombreDeCaractèresASupprimer* est utilisée). 




**`<Type de troncature> : (Constante optionnelle de type Booléen)`**

Type de troncature à effectuer : 


|   |   |
| --- | --- |
| *nombreDeCaractèresAConserver*<br>(Valeur par défaut) | Le paramètre &lt;Nombre de caractères&gt; correspond au nombre de caractères à conserver dans la chaîne &lt;Résultat&gt;. |
| *nombreDeCaractèresASupprimer* | Le paramètre &lt;Nombre de caractères&gt; correspond au nombre de caractères à supprimer de &lt;Source&gt;. |

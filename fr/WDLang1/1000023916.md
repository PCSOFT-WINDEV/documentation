
## DonneUUID256 (Fonction)

***En anglais : GetUUID256***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Calcule un identifiant unique universel (nommé "UUID"), ayant pour taille 256 bits. 

Cet identifiant est unique dans le monde quelle que soit la machine sur laquelle il a été produit.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonUUID256 est un UUID sur 256
MonUUID256 = DonneUUID256()
// Exemple du UUID256 calculé : cc0ca42fd00f354e9f0ed1386590e0fd5d75a90f28ec822d138c00ba89021956
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = DonneUUID256()`
---

**`<Résultat> : (UUID sur 256)`**

Identifiant généré.
Le UUID renvoyé  est formaté sans séparateurs.
Par exemple : "cc0ca42fd00f354e9f0ed1386590e0fd5d75a90f28ec822d138c00ba89021956".

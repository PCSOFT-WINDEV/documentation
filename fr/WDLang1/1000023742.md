


## &lt;Tableur&gt;.SupprimeLigne (Fonction)

***En anglais : &lt;Spreadsheet&gt;.DeleteRow***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime une ligne dans un champ Tableur.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Supprime les lignes 2 et 3
TBLR_MonTableur.SupprimeLigne(2)
TBLR_MonTableur.SupprimeLigne(3)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Champ Tableur>.SupprimeLigne(<Ligne> [, <Feuille>])`
---

**`<Champ Tableur> : (Nom de champ)`**

Nom du champ Tableur à manipuler.

**`<Ligne> : (Entier)`**

Numéro de la ligne à supprimer, par exemple : 1. 
Le numéro des lignes commence à 1. 
Si ce paramètre est invalide, la fonction n'a aucun effet.

**`<Feuille> : (Entier optionnel)`**

Numéro de la feuille de calcul à manipuler. Si ce paramètre n'est pas spécifié, la feuille de calcul en cours est manipulée. 
Si ce paramètre est invalide, la fonction n'a aucun effet.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Après l'exécution de la fonction : 

- les lignes en-dessous de la ligne supprimée sont décalées vers le haut. 

- le contenu des formules est renommé.

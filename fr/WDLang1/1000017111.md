
## ServiceArrête (Fonction)

***En anglais : ServiceStop***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Arrête un service. Ce service peut être un service présent sur le poste local ou sur un poste distant.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Arrête le service présent dans la table TABLE_Service
SI ServiceArrête(COL_Nom[TABLE_Service]) = Vrai ALORS
	SI ServiceAttendEtat(COL_Nom[TABLE_Service], serviceEtatArrêté, 10) = -1 ALORS
		 Erreur(ErreurInfo())
	FIN
SINON
	Erreur(ErreurInfo())
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = ServiceArrête(<Nom du service> [, <Poste distant>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la demande d'arrêt du service a été transmise au gestionnaire de services, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Nom du service> : (Chaîne de caractères)`**

Nom du service à arrêter.

**`<Poste distant> : (Chaîne de caractères optionnelle)`**

Adresse IP du poste distant au format xxx.xxx.xxx.xxx (obtenu par la fonction [NetAdresseIP](../WDLang3/3056023.md)) ou nom du poste distant (obtenu par la fonction [NetNomMachine](../WDLang3/3056014.md)). Ce paramètre permet d'arrêter un service sur un autre poste. Pour utiliser cette option, l'utilisateur doit être administrateur de la machine distante. 
Si ce paramètre n'est pas précisé ou correspond à une chaîne vide, le service est manipulé en local.

### Mode de fonctionnement
<a name="mode_fonctionnement_ELTPARAGRAPHE000056"></a>La fonction **ServiceArrête** demande au gestionnaire de services d'arrêter le service. Cette opération peut ne pas être effectuée immédiatement. Pour connaître l'état du service, utilisez la fonction [ServiceAttendEtat](../WDLang1/1000017101.md).

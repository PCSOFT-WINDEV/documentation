


## PicCrée (Fonction)

***En anglais : PicCreate***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Crée une nouvelle image dans un champ Editeur d'images.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Création de l'image dans le champ Editeur d'images
SI PicCrée(EI_MonEditeur, picImage, 48, 24) = Faux ALORS
	Erreur(ErreurInfo())
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = PicCrée(<Champ Editeur d'images> [, <Mode de création> [, <Largeur> [, <Hauteur>]]])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'image a été créée,

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour connaître le libellé de l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.




**`<Champ Editeur d'images> : (Nom de champ)`**

Nom du champ Editeur d'images à manipuler. 

**`<Mode de création> : (Constante optionnelle de type Entier)`**

Mode de création de l'image : 


|   |   |
| --- | --- |
| *picIcône* | Création d'une nouvelle icône. Dans ce cas, les paramètres &lt;Largeur&gt; et &lt;Hauteur&gt; ne doivent pas être spécifiés. Dans le cas contraire, une erreur fatale est affichée. |
| *picImage*<br>(Valeur par défaut) | Création d'une nouvelle image. Dans ce cas, les paramètres &lt;Largeur&gt; et &lt;Hauteur&gt; doivent être spécifiés. |



**`<Largeur> : (Entier optionnel)`**

Largeur de l'image à créer, exprimée en pixels. Par défaut, cette largeur est de 800 pixels. 
Attention : Ce paramètre doit être spécifié uniquement lors de l'utilisation de la constante *picImage*. Si ce paramètre est spécifié avec la constante *picIcône*, une erreur fatale est affichée. 

**`<Hauteur> : (Entier optionnel)`**

Hauteur de l'image à créer, exprimée en pixels. Par défaut, cette hauteur est de 600 pixels. 
Attention : Ce paramètre doit être spécifié uniquement lors de l'utilisation de la constante *picImage*. Si ce paramètre est spécifié avec la constante *picIcône*, une erreur fatale est affichée.

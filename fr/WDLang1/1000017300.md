
## CalendrierSélecteur (Fonction)

***En anglais : CalendarPicker***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ouvre un dialogue modal avec un calendrier pour saisir une date. Une fenêtre s'ouvre permettant à l'utilisateur de sélectionner une date dans un calendrier. Cette fonction ne nécessite ni champ de saisie de type Date, ni champ de type Calendrier.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
DateRésultat est une chaîne 
DateRésultat = CalendrierSélecteur()
```


<a name="Exemple2"></a>



```wl
DateRésultat est une chaîne 
DateRésultat = CalendrierSélecteur("20090101", "Choisissez une date :")
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = CalendrierSélecteur([<Date initiale> [, <Titre de la fenêtre>]])`
---

**`<Résultat> : (Chaîne de caractères ou variable de type Date)`**



- Date au format AAAAMMJJ sélectionnée par l'utilisateur,

- Chaîne vide ("") si l'utilisateur a fermé la fenêtre par le bouton "X".




**`<Date initiale> : (Chaîne de caractères optionnelle ou variable de type Date)`**

Date affichée par défaut dans le calendrier (au format AAAAMMJJ). 
Si ce paramètre n'est pas spécifié ou correspond à une chaîne vide (""), la date du jour est sélectionnée par défaut.

**`<Titre de la fenêtre> : (Chaîne de caractères optionnelle)`**

Titre affiché dans la barre de titre de la fenêtre de sélection de la date.




## TreeMapSupprimeTout (Fonction)

***En anglais : TreeMapDeleteAll***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime tous les éléments d'un champ TreeMap.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Supprime tous les éléments du champ TreeMap
TreeMapSupprimeTout(TMAP_Vin)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`TreeMapSupprimeTout(<Champ TreeMap>)`
---

**`<Champ TreeMap> : (Nom de champ)`**

Nom du champ TreeMap manipulé.
Si ce paramètre correspond à une chaîne vide (""), le champ manipulé sera le champ TreeMap auquel appartient l'événement en cours.

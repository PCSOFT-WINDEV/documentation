


## Restaure (Fonction)

***En anglais : Restore***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Affiche une fenêtre WINDEV dans sa taille initiale.

**Remarque** : La fonction **Restaure** réalise la même action que le bouton de restauration présent dans le coin haut droit des fenêtres.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affichage de la fenêtre "FEN_SaisieClient" dans sa taille initiale
Restaure(FEN_SaisieClient)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`Restaure([<Fenêtre>])`
---

**`<Fenêtre> : (Nom de fenêtre optionnel)`**

Nom de la fenêtre à afficher dans sa taille initiale.
Si ce paramètre n'est pas précisé ou correspond à une chaîne vide (""), la fenêtre en cours est affichée dans sa taille initiale.

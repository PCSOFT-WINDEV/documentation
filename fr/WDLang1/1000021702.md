


## &lt;Onglet&gt;.Ferme (Fonction)

***En anglais : &lt;Tab&gt;.Close***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ferme un volet dynamique dans un champ Onglet. L'événement "Fermeture d'un volet" n'est pas appelé.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Clic sur le champ Bouton Fermer
ONG_OngletDyn.Ferme(ONG_OngletDyn.Valeur)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Fermer un volet d'onglet dynamique spécifique

`<Champ Onglet>.Ferme(<Alias>)`
---

**`<Champ Onglet> : (Nom de champ)`**

Nom du champ Onglet à manipuler.

**`<Alias> : (Chaîne de caractères)`**

Alias du volet d'onglet dynamique à fermer. Cet alias peut être récupéré grâce à : 

- la propriété [Valeur](../Proprietes/2510130.md). 

- la propriété [Alias](../Proprietes/2510004.md). 

- la fonction [&lt;Onglet&gt;.Ouvre](../WDLang1/1000021704.md). 


Si ce paramètre ne correspond à aucun volet dynamique, la fonction ne fait rien et renvoie une erreur non fatale.


<a name="SYNTAXE2"></a>

### Fermer le volet d'onglet dynamique en cours

`<Champ Onglet>.Ferme()`
---

**`<Champ Onglet> : (Nom de champ)`**

Nom du champ Onglet à manipuler. Le volet dynamique en cours sera fermé.

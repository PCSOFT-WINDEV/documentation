


## Fonctions de gestion de la corbeille
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions de gestion de la corbeille sont les suivantes :



|   |   |
| --- | --- |
| [CorbeilleListeFichier](../WDLang1/1000017162.md) | Liste les fichiers contenus dans la corbeille du système. |
| [CorbeilleRestaure](../WDLang1/1000017159.md) | Restaure un fichier placé dans la corbeille système. |
| [CorbeilleSupprime](../WDLang1/1000017158.md) | Affiche un message de confirmation pour la suppression d'un fichier dans la corbeille. |
| [CorbeilleVide](../WDLang1/1000017157.md) | Vide la corbeille du système. |

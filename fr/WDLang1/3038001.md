


## Maximise (Fonction)

***En anglais : Maximize***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Affiche une fenêtre WINDEV dans sa taille maximale.

**Remarque** : La fonction **Maximise** réalise la même action que le bouton d'agrandissement présent dans le coin haut droit des fenêtres.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affichage de la fenêtre "FEN_SaisieClient" dans sa taille maximale
Maximise(FEN_SaisieClient)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`Maximise([<Fenêtre>])`
---

**`<Fenêtre> : (Nom de fenêtre)`**

Nom de la fenêtre à afficher dans sa taille maximale.
Si ce paramètre n'est pas précisé ou correspond à une chaîne vide (""), la fenêtre en cours est affichée dans sa taille maximale.

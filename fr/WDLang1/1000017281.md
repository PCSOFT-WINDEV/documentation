


## CalendrierPosition (Fonction)

***En anglais : CalendarPosition***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le mois affiché dans un champ Calendrier.



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affiche le mois du calendrier en toutes lettres
Info("Le mois affiché dans le champ Calendrier est : " + ...
	DateVersMoisEnLettre(CalendrierPosition(CAL_Calendrier1)))
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = CalendrierPosition(<Champ Calendrier>)`
---

**`<Résultat> : (Chaîne de caractères ou variable de type Date)`**

Premier jour du mois affiché. Ce paramètre est de la forme "AAAAMM01". Par exemple, si le mois affiché est le mois de Novembre 2008, &lt;Résultat&gt; vaut "20081101".

**Attention** : &lt;Résultat&gt; correspond au premier jour du mois affiché dans le calendrier, et non le premier jour affiché dans le calendrier pour le mois sélectionné (pour le mois de Novembre 2008, le premier jour affiché est le Lundi 27 Octobre 2008).

**`<Champ Calendrier> : (Nom de champ)`**

Nom du champ Calendrier manipulé.

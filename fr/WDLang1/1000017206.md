
## MultimédiaOuvre (Fonction)

***En anglais : MultimediaOpen***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ouvre un fichier multimédia dans un champ Multimédia. Le fichier est chargé en mémoire. Pour jouer le fichier, il est nécessaire d'utiliser la fonction [MultimédiaJoue](../WDLang1/1000017202.md).




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) 
```wl
MonFichier est une chaîne = "C:\Ma musique\mysong.mp3"
MultimédiaOuvre(MM_Multimédia1, MonFichier)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = MultimédiaOuvre(<Champ Multimédia> , <Fichier à ouvrir>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'ouverture du ficher a été correctement réalisée, 

- <u><u><u><u>Faux</u></u></u></u> en cas d'erreur. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Champ Multimédia> : (Nom de champ)`**

Nom du champ Multimédia qui va jouer le fichier multimédia.

**`<Fichier à ouvrir> : (Chaîne de caractères)`**

Nom et chemin du fichier multimédia à ouvrir.

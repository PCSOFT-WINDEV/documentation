


## GanttSupprimeLien (Fonction)

***En anglais : GanttDeleteLink***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime des liens dans une colonne d'un champ Table (ou Table hiérarchique) de type Diagramme de Gantt.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Supprime les liens de la tâche T2
GanttSupprimeLien(COL_Gantt, "T2")
```



```wl
MonLienGantt est un LienGantt
...
GanttSupprimeLien(COL_Gantt, MonLienGantt)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Supprimer tous les liens vers une tâche

`GanttSupprimeLien(<Colonne Gantt> , <Identifiant Tâche>)`
---

**`<Colonne Gantt> : (Nom de champ)`**

Nom de la colonne (d'un champ Table ou Table hiérarchique) de type Diagramme de Gantt à manipuler.

**`<Identifiant Tâche> : (Chaîne de caractères)`**

Identifiant de la tâche pour laquelle tous les liens préalables doivent être supprimés. Cet identifiant correspond à la propriété **ID** définie dans la variable de type [TâcheGantt](../WDLang1/1000020881.md).

Si l'identifiant n'existe pas, la fonction ne fait rien. 


<a name="SYNTAXE2"></a>

### Supprime les liens d'une tâche vers une autre tâche

`GanttSupprimeLien(<Colonne Gantt> , <Identifiant Tâche Source> , <Identifiant Tâche Destination>)`
---

**`<Colonne Gantt> : (Nom de champ)`**

Nom de la colonne (d'un champ Table ou Table hiérarchique) de type Diagramme de Gantt à manipuler.

**`<Identifiant Tâche Source> : (Chaîne de caractères)`**

Identifiant de la tâche de départ des liens à supprimer. Cet identifiant correspond à la propriété **ID** définie dans la variable de type [TâcheGantt](../WDLang1/1000020881.md).

Si l'identifiant n'existe pas, la fonction ne fait rien. 

**`<Identifiant Tâche Destination> : (Chaîne de caractères)`**

Identifiant de la tâche de destination des liens à supprimer. Cet identifiant correspond à la propriété **ID** définie dans la variable de type [TâcheGantt](../WDLang1/1000020881.md).

Si l'identifiant n'existe pas, la fonction ne fait rien. 


<a name="SYNTAXE3"></a>

### Supprimer le lien spécifié

`GanttSupprimeLien(<Colonne Gantt> , <Lien à supprimer>)`
---

**`<Colonne Gantt> : (Nom de champ)`**

Nom de la colonne (d'un champ Table ou Table hiérarchique) de type Diagramme de Gantt à manipuler.

**`<Lien à supprimer> : (Variable de type LienGantt)`**

Nom de la variable de type [LienGantt](../WDLang1/1000021076.md) à supprimer.

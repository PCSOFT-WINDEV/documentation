


## ListeInfo (Fonction)

***En anglais : ListInfo***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Récupère les caractéristiques d'une liste WLangage : types des éléments et nombre d'éléments.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Liste dont les éléments sont des instances de MaClasse
MaListe est une liste de MaClasse
// Récupère la définition du type des éléments
MaDéfinition est une Définition 
MaDéfinition = ListeInfo(MaListe, tiDéfinitionElément)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = ListeInfo(<Liste WLangage> , <Information>)`
---

**`<Résultat> : (Variant)`**

Information demandée. 

**`<Liste WLangage> : (Liste)`**

Nom de la variable de type [Liste](../WDLang1/1514023.md) à manipuler.

**`<Information> : (Constante de type Entier)`**

Information à récupérer :


|   |   |
| --- | --- |
| *tiDéfinitionElément* | &lt;Résultat&gt; est une variable de type [Définition](../WDLang1/1000019540.md) et contient la définition du type des éléments.<br><br>![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Cette constante n'est pas disponible. |
| *tiNombreTotal* | &lt;Résultat&gt; est une variable de type Entier et contient le nombre total d'éléments dans la liste. |
| *tiTypeElément* | &lt;Résultat&gt; est une variable de type Entier et contient le type des éléments de la liste. Ce résultat est compatible avec la fonction [TypeVar](../WDLang1/3013056.md). |

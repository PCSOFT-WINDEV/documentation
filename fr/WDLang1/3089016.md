
## Gestion du planificateur Windows
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000151"></a>
Windows propose en standard la possibilité de planifier des tâches automatiques grâce au gestionnaire de tâches planifiées. Vous pouvez ainsi lancer un programme tous les lundis à 10h, par exemple.

WINDEV et WEBDEV permettent de créer et de configurer directement des tâches planifiées depuis une application grâce aux fonctions de gestion du planificateur.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Principe d'utilisation
<a name="principe_utilisation_ELTTEXTE000175"></a>


### Présentation
<a name="presentation_ELTPARAGRAPHE000028"></a>Une tâche planifiée est constituée de deux éléments :

- la tâche en elle-même : programme lancé, ...

- les horaires de la tâche, c'est-à-dire les événements planifiés qui vont déclencher la tâche (par exemple, tâche exécutée tous les Lundi à 15 heures)




Une même tâche peut être associée à plusieurs horaires.

Pour manipuler ces deux éléments, WINDEV met à votre disposition :

- deux structures ([TâchePlanifiée](../WDLang1/3089004.md) et [HoraireTâchePlanifiée](../WDLang1/3089002.md)).

- plusieurs [fonctions WLangage](../WDLang1/3089015.md) (commençant par Planificateur).




Par défaut, lors de sa création une tâche planifiée s'exécute tous les lundis à 9h.


<a name="NOTE2_2"></a>


### Manipulation d'une tâche planifiée
<a name="manipulation_une_tache_planifiee_ELTPARAGRAPHE000057"></a>Pour ajouter ou modifier une tâche planifiée, il est nécessaire de manipuler les structures associées.

Par exemple, pour ajouter une tâche planifiée, il faut :

1. Définir les caractéristiques de la tâche grâce à la structure [TâchePlanifiée](../WDLang1/3089004.md).

2. Ajouter la tâche grâce à la fonction [PlanificateurAjouteTâche](../WDLang1/3089003.md).

3. Définir les caractéristiques de chaque horaire associé à la tâche grâce à la structure [HoraireTâchePlanifiée](../WDLang1/3089002.md).

4. Ajouter l'horaire ou les horaires de la tâche grâce à la fonction [PlanificateurAjouteHoraire](../WDLang1/3089001.md).




**Remarques** :

- Pour remplir la structure TâchePlanifiée avec les caractéristiques d'une tâche planifiée, utilisez la fonction [PlanificateurPropriétésTâche](../WDLang1/3089011.md).

- Pour remplir la structure HoraireTâchePlanifiée avec les caractéristiques d'une tâche planifiée, utilisez la fonction [PlanificateurPropriétésHoraire](../WDLang1/3089010.md).

- Pour ré-initialiser une ou plusieurs structures, utilisez la fonction [PlanificateurRAZ](../WDLang1/3089012.md).

- Pour créer une tâche planifiée sous Windows Vista et supérieur, il est nécessaire de posséder les droits "Administrateur" sur le poste où la tâche planifiée va être exécutée.

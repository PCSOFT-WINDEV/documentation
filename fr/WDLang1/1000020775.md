


## &lt;TimeLine&gt;.Affiche (Fonction)

***En anglais : &lt;TimeLine&gt;.Display***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Réinitialise le contenu d'un champ TimeLine à partir de sa source de données (fichier de données HFSQL par exemple). Tous les événements sont supprimés du champ TimeLine et sont recréés à partir de la source de données.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ajout d'un événement dans un fichier de données HFSQL
HAjoute(Fic_Evenement)
// Réinitialisation du champ TimeLine
TL_Musique.Affiche()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Champ TimeLine>.Affiche([<Options>])`
---

**`<Champ TimeLine> : (Nom de champ)`**

Nom du champ TimeLine à manipuler. Le champ TimeLine doit être lié à une source de données. Dans le cas contraire (par exemple si le champ est rempli par programmation), la fonction **&lt;TimeLine&gt;.Affiche** n'a aucun effet.

**`<Options> : (Constante de type Entier)`**

Option d'affichage du champ TimeLine :


|   |   |
| --- | --- |
| *plnSansExécuteRequête* | Permet de ne pas ré-exécuter automatiquement la requête liée au champ TimeLine (cas d'un champ TimeLine lié à une requête). |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Si la source de données liée au champ TimeLine est une requête, cette requête est réinitialisée. 

- La position dans le champ TimeLine n'est pas modifiée.

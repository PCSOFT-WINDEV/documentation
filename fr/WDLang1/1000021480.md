


## TableurCadreSélection (Fonction)

***En anglais : SpreadsheetBorderSelection***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de connaître et de modifier les caractéristiques du cadre des cellules sélectionnées dans un champ Tableur présent dans une fenêtre. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Définit le cadre à dessiner
xCadre est un Cadre
xCadre.Epaisseur = 2
xCadre.Couleur = Noir

// Sélectionne les cellules à encadrer
TableurSelectPlus(TBLR_Tableur, "A3", "B12")
// Trace le cadre autour du groupe de cellules
TableurCadreSélection(TBLR_Tableur, xCadre, Faux)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Modifier les caractéristiques du cadre

`TableurCadreSélection(<Champ Tableur> , <Cadre> [, <Sélection>])`
---

**`<Champ Tableur> : (Nom de champ)`**

Nom du champ Tableur manipulé. 

**`<Cadre> : (Variable de type Cadre)`**

Nom de la variable de type [Cadre](../Motscles/15140100.md) contenant les caractéristiques du cadre à utiliser. 

**`<Sélection> : (Booléen optionnel)`**



- <u><u><u><u>Vrai</u></u></u></u> si le cadre doit être appliqué à chacune des cellules sélectionnées, 

- <u><u><u><u>Faux</u></u></u></u> (valeur par défaut) si le cadre doit être appliqué à l'ensemble des cellules sélectionnées. 





<a name="SYNTAXE2"></a>

### Connaître les caractéristiques du cadre

`<Cadre> = TableurCadreSélection(<Champ Tableur>)`
---

**`<Cadre> : (Variable de type Cadre)`**

Nom de la variable de type [Cadre](../Motscles/15140100.md) contenant les caractéristiques du cadre. 

**`<Champ Tableur> : (Nom de champ)`**

Nom du champ Tableur manipulé. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- L'opération de modification est ajoutée dans la liste des opérations à annuler : l'utilisateur peut annuler cette opération via le menu contextuel ou via la combinaison de touches Ctrl + Z. 

- Cette fonction est utilisable uniquement sur un champ Tableur présent dans une fenêtre.

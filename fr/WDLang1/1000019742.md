


## &lt;TreeMap&gt;.Cherche (Fonction)

***En anglais : &lt;TreeMap&gt;.Seek***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Recherche l'indice d'un élément dans un champ TreeMap à partir du chemin de l'élément. Ce chemin a été fourni dans la fonction [&lt;TreeMap&gt;.Ajoute](../WDLang1/1000019740.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Le champ TMAP_Vin est rempli à partir d'un champ Table.
// Recherche de l'élément correspondant à l'élément sélectionné dans le champ Table
CheminSélection est une chaîne
CheminSélection = COL_Typevin[TABLE_VIN] + "\" + COL_Proprietaire[TABLE_VIN]
IndiceElement = TMAP_Vin.Cherche(CheminSélection)
// Sélection de l'élément dans le champ TreeMap
SI IndiceElement <> - 1 ALORS
	TMAP_Vin = IndiceElement
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Champ TreeMap>.Cherche(<Chemin de l'élément> [, <Indice de départ>])`
---

**`<Résultat> : (Entier)`**



- Indice de l'élément trouvé, 

- -1 si aucun élément ne correspond à la recherche. 


**Remarque** : L'élément trouvé peut être manipulé directement grâce à son indice. Il est par exemple possible de modifier les propriétés de cet élément ou de le sélectionner dans le champ TreeMap.

**`<Champ TreeMap> : (Nom de champ)`**

Nom du champ TreeMap manipulé.

**`<Chemin de l'élément> : (Chaîne de caractères)`**

Chemin et libellé de l'élément recherché. Ce chemin doit être identique à celui utilisé lors de l'ajout de l'élément dans le champ TreeMap avec la fonction [&lt;TreeMap&gt;.Ajoute](../WDLang1/1000019740.md). 
Le séparateur des différents éléments du chemin peut être :

- soit le caractère "\\". Par exemple : "MonRépertoire\\MonFichier". 

- soit le caractère "TAB". Par exemple : "Desserts"+TAB+"Crème brulée".




**`<Indice de départ> : (Entier optionnel)`**

Indice de l'élément à partir duquel la recherche doit être effectuée. Si ce paramètre n'est pas spécifié, la recherche est effectuée dans tout le champ TreeMap. Ce paramètre permet de rechercher toutes les occurrences d'un chemin.

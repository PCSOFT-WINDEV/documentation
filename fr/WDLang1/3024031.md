


## Taille (Fonction)

***En anglais : Length***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie :

- la taille d'une chaîne de caractères, c'est-à-dire le nombre de caractères composant la chaîne de caractères (y compris les caractères espace et 0 binaire).

- la taille d'un buffer, c'est-à-dire le nombre d'octets composant le buffer.







<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
Taille("Quasimodo")      // Renvoie 9
Taille(" Quasimodo ")    // Renvoie 11
Taille("A" + Caract(0))  // Renvoie 2
Taille("")               // Renvoie 0
Taille(" ")              // Renvoie 1
```

## Syntaxe

### Connaître la taille d'une chaîne de caractères

`<Résultat> = Taille(<Chaîne initiale>)`
---

**`<Résultat> : (Entier)`**

Nombre **de caractères** composant la chaîne.

**`<Chaîne initiale> : (Chaîne de caractères)`**

Chaîne de caractères dont la taille doit être calculée.  


<a name="SYNTAXE2"></a>

### Connaître la taille d'un buffer

`<Résultat> = Taille(<Buffer initial>)`
---

**`<Résultat> : (Entier)`**

Nombre d' **octets** composant le buffer.

**`<Buffer initial> : (Buffer)`**

Buffer dont la taille doit être calculée. 
![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) En UNICODE, chaque caractère occupe quatre octets de mémoire. La taille d'un buffer contenant une chaîne UNICODE correspond à quatre fois le nombre de caractères réellement contenu dans la chaîne sous Windows.




## FenTauxGFI (Fonction)

***En anglais : WinRateDDW***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de connaître ou de modifier le taux de grisage des fenêtres utilisant le GFI (Grisage des Fenêtres Inaccessibles).

Le GFI consiste à griser automatiquement la ou les fenêtres inaccessibles. Pour plus de détails, consultez [Grisage automatique des fenêtres](../Editeurs/9000071.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// -- Initialisation du projet
// Modification du taux de grisage
FenTauxGFI(40)
// Activation du GFI de toutes les fenêtres de l'application
FenActiveGFI(Vrai)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le taux de grisage

`<Résultat> = FenTauxGFI()`
---

**`<Résultat> : (Entier)`**

Taux de grisage actuel (compris entre 0 et 100). Le taux de grisage par défaut est 18%.  


<a name="SYNTAXE2"></a>

### Modifier le taux de grisage

`<Résultat> = FenTauxGFI(<Taux de grisage>)`
---

**`<Résultat> : (Entier)`**

Taux de grisage avant modification (compris entre 0 et 100).

**`<Taux de grisage> : (Entier)`**

Nouveau taux de grisage (compris entre 0 et 100). Si une valeur supérieure à 100 est donnée, la valeur 100 sera utilisée. Si une valeur inférieure à 0 est donnée, la fonction **FenTauxGFI** sera sans effet. Le taux de grisage par défaut est 18%.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Configuration du poste des utilisateurs finaux
<a name="configuration_poste_des_utilisateurs_finaux_ELTPARAGRAPHE000046"></a>Selon la configuration du poste des utilisateurs finaux, le GFI peut également être sans effet.

En effet, le GFI prend en compte les caractéristiques de la carte graphique, la vitesse du processeur, ...

En cas de problème, conseillez à vos utilisateurs finaux de mettre à jour le pilote de leur carte graphique.

**Remarque** : Pour fonctionner, le poste des utilisateurs finaux doit :

- être en mode d'affichage supérieur à 256 couleurs.

- être en mode 32 ou 64 bits.

- être sous Windows 2000 ou supérieur.

- utilisé le niveau maximal d'"Accélération matérielle" de la carte graphique (propriétés avancées d'affichage du poste).



<a name="NOTE0_2"></a>


### Activer le grisage (GFI)
<a name="activer_grisage_gfi_ELTPARAGRAPHE000064"></a>Par activer ou désactiver le GFI, utilisez la fonction [FenActiveGFI](../WDLang1/3038056.md).

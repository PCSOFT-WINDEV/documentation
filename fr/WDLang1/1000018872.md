
## SysMiniatureSupprimeBouton (Fonction)

***En anglais : SysThumbnailDeleteButton***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime un des boutons présents dans la miniature de l'application dans la barre des tâches.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Supprime le bouton au rang 3
SysMiniatureSupprimeBouton(3)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = SysMiniatureSupprimeBouton(<Numéro du bouton>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le bouton a été supprimé,

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 




**`<Numéro du bouton> : (Entier)`**

Rang du bouton à supprimer (compris entre 1 et 7). Lors de l'ajout d'un bouton, la fonction [SysMiniatureAjouteBouton](../WDLang1/1000018871.md) renvoie le rang du bouton ajouté.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Lorsqu'un bouton est supprimé, les rangs des boutons suivants sont modifiés. Par exemple, si le bouton de rang 4 est supprimé, le bouton de rang 5 devient après cette suppression le bouton de rang 4. 

- Il est possible de supprimer tous les boutons liés à une miniature grâce à la fonction [SysMiniatureSupprimeTout](../WDLang1/1000018873.md).

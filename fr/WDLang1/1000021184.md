


## SélecteurOccurrence (Fonction)

***En anglais : RadioButtonCount***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le nombre d'options présentes dans un champ Sélecteur.

Remarque : Cette fonction renvoie le même résultat que la propriété [Occurrence](../Proprietes/2510097.md) utilisée sur un champ Sélecteur. 




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Nombre d'options présentes dans le champ "SEL_Options"
ResNombreOptions = SélecteurOccurrence(SEL_Options)

// Identique à :
// ResNombreOptions = SEL_Options.Occurrence
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = SélecteurOccurrence(<Champ Sélecteur>)`
---

**`<Résultat> : (Entier)`**

Nombre d'options présentes dans le champ Sélecteur spécifié.

**`<Champ Sélecteur> : (Nom de champ)`**

Nom du champ Sélecteur à manipuler.

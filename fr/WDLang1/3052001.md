
## MDIEnumèreFille (Fonction)

***En anglais : MDIEnumChild***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie l'alias de la fenêtre fille MDI spécifiée.

Utilisée dans une boucle, la fonction **MDIEnumèreFille** peut permettre d'énumérer les alias des fenêtres filles MDI.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
NomAlias est une chaîne
i est un entier = 1
NomAlias = MDIEnumèreFille(i)
TANTQUE NomAlias <> ""
	Trace("Fille MDI n° " + i + " : " + NomAlias)
	// Fenêtre fille suivante
	i++
	NomAlias = MDIEnumèreFille(i)
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = MDIEnumèreFille(<Numéro de la fenêtre fille>)`
---

**`<Résultat> : (Chaîne de caractères)`**



- Alias de la fille MDI spécifiée 

- Chaîne vide ("") en cas d'erreur.




**`<Numéro de la fenêtre fille> : (Entier)`**

Numéro de la fenêtre fille permettant de rechercher son alias. Si ce paramètre est supérieur au nombre de filles MDI ouvertes, &lt;Résultat&gt; correspondra à une chaîne vide ("").

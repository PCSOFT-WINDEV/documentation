


## Répétition (Type de variable)

***En anglais : Repetition ***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **Répétition** permet de décrire et de modifier les caractéristiques de répétition d'un rendez-vous à l'aide de différentes propriétés WLangage. Le rendez-vous sera ensuite affiché dans un champ Agenda ou dans un champ Planning.

Ce type de variable est utilisé par le type [RendezVous](../WDLang1/1000019244.md).

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Définit un rendez-vous pour le matin
MonRendezVous est un RendezVous
MonRendezVous.DateDébut = DateSys() + "090000"
MonRendezVous.DateFin = DateSys() + "093000"
MonRendezVous.Titre = "Réunion de l'équipe de développement"
MonRendezVous.Invité = "Jacques, Marc, Emilie"
MonRendezVous.Contenu = "Point quotidien sur l'avancement des projets."
MonRendezVous.AvecRépétition = Vrai

// Paramètre une répétition quotidienne du rendez-vous
ParamètreRépétition est un Répétition
ParamètreRépétition.Type = plnTousLesJours
ParamètreRépétition.TypeFin = plnPasDeFin
MonRendezVous.Répétition = ParamètreRépétition

AgendaAjouteRendezVous(AGD_Agenda1, MonRendezVous)
```

## Remarques


### Propriétés spécifiques à la description des variables de type Répétition
<a name="proprietes_specifiques_description_des_variables_type_repetition_ELTPARAGRAPHE000045"></a>Les propriétés suivantes peuvent être utilisées pour manipuler une variable **Répétition** :

| **Nom de la propriété** | **Type manipulé** | **Effet** |
| --- | --- | --- |
| DateFin | Chaîne de caractères | Date de fin de la répétition. Cette propriété est utilisée si la propriété **TypeFin** est égale à la constante *plnDateFin*. |
| Heure | Chaîne de caractères | Heure de la répétition au format UTC (temps universel). Les valeurs possibles sont :<br><br>- un chiffre compris entre 0 et 23.<br><br>- une liste de nombres séparés par des virgules afin de spécifier plusieurs heures.<br><br>- un intervalle. Par exemple "9-13".<br><br>- une étoile "\*" pour indiquer toute la plage.<br><br>- un intervalle avec un pas. par exemple "\*/2" pour répéter le rendez-vous toutes les heures paires.<br><br><br> |
| JourDeLaSemaine | Chaîne de caractères | Numéro du jour de la semaine auquel le rendez-vous a lieu (par exemple "2"). Ce nombre est compris entre 1 et 7 (1 = Lundi, 2 = Mardi, ...).<br>Cette propriété peut également correspondre à :<br><br>- une liste de nombres séparés par des virgules afin de spécifier plusieurs jours.<br><br>- un intervalle. Par exemple "1-3" (le rendez-vous est répété Lundi, Mardi et Mercredi).<br><br>- une étoile "\*" pour indiquer toute la plage.<br><br>- un intervalle avec un pas. Par exemple "\*/2" pour répéter le rendez-vous tous les jours de la semaine pairs.<br><br>- le jour de la semaine en toutes lettres ("lundi", "mardi", "mercredi", "jeudi", "vendredi").<br><br><br><br><br> |
| JourDuMois | Chaîne de caractères | Numéro du jour du mois auquel le rendez-vous a lieu (par exemple "31"). Ce nombre est compris entre 0 et 31.<br>Cette propriété peut également correspondre à :<br><br>- une liste de nombres séparés par des virgules afin de spécifier plusieurs mois.<br><br>- un intervalle. Par exemple "9-13" (le rendez-vous est répétée du 9 au 13 du mois).<br><br>- une étoile "\*" pour indiquer toute la plage.<br><br>- un intervalle avec un pas. Par exemple "\*/2" pour répéter le rendez-vous tous les jours pairs du mois.<br><br><br><br><br> |
| JourDuMoisOuDeLaSemaine | Booléen | <br><br>- <u><u><u><u>Vrai</u></u></u></u> pour répéter le rendez-vous tous les lundis ET le premier jour du mois (conditions cumulées).<br><br>- <u><u><u><u>Faux</u></u></u></u> pour répéter le rendez-vous uniquement les lundis 1er (conditions exclusives).<br><br><br><br><br> |
| Minute | Chaîne de caractères | Minutes de l'heure de la répétition du rendez-vous. Les valeurs possibles sont :<br><br>- un chiffre compris entre 0 et 59.<br><br>- une liste de nombres séparés par des virgules afin de spécifier plusieurs minutes.<br><br>- un intervalle. Par exemple "9-13".<br><br>- une étoile "\*" pour indiquer toute la plage.<br><br>- un intervalle avec un pas. Par exemple "\*/2" pour répéter le rendez-vous à toutes les minutes paires.<br><br><br> |
| Mois | Chaîne de caractères | Numéro du mois de la répétition du rendez-vous. Les valeurs possibles sont :<br><br>- un chiffre compris entre 1 et 12.<br><br>- une liste de nombres séparés par des virgules afin de spécifier plusieurs mois.<br><br>- un intervalle. Par exemple, "9-11".<br><br>- une étoile "\*" pour indiquer toute la plage.<br><br>- un intervalle avec un pas. Par exemple "\*/2" pour répéter le rendez-vous tous les mois pairs.<br><br><br> |
| NbOccurrence | Entier | Nombre maximum de répétitions. Cette propriété est prise en compte lorsque la propriété **TypeFin** est égale à la constante *plnOccurrenceMax*.<br><br> |
| Type | Constante de type Entier | Type de répétition. <br>Cette propriété correspond à l'une des constantes suivantes :<br><br>- *plnTousLesJours* : le rendez-vous sera répété tous les jours à la même heure.<br><br>- *plnToutesLesSemaines* : le rendez-vous sera répété toutes les semaines, le même jour à la même heure.<br><br>- *plnTousLesMois* : le rendez-vous sera répété tous les mois, le même jour à la même heure.<br><br>- *plnTousLesAns* : le rendez-vous sera répété tous les ans, le même jour à la même heure.<br><br><br> |
| TypeFin | Constante de type Entier | Condition d'arrêt de la répétition. <br>Cette propriété correspond à l'une des constantes suivantes :<br><br>- *plnDateFin* : le rendez-vous cessera d'être répété à partir de la date indiquée dans la propriété **DateFin**.<br><br>- *plnOccurrenceMax* : le rendez-vous cessera d'être répété après le nombre de répétitions indiqué dans la propriété **NbOccurrence**. <br>	<br><br>- *plnPasDeFin* : le rendez-vous sera répété à l'infini.<br><br><br> |

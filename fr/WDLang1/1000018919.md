
## FenInitialisée (Fonction)

***En anglais : WinInitialized***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de savoir si le code de fin d'initialisation d'une fenêtre a été exécuté.


Cette fonction peut être utilisée pour ne pas remplir certains champs dont le code de modification est exécuté avant la fin de l'ouverture (par exemple à cause de la restauration de valeurs mémorisées).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Code de modification de l'interrupteur INT_Filtrage
SI FenInitialisée() = Vrai ALORS
	RemplirListe(INT_Filtrage)
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = FenInitialisée()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le code de fin d'initialisation de la fenêtre a été exécuté, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.

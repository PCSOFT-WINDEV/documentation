


## Création de l'exécutable et du programme d'installation
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000116"></a>
La gestion des langues gérées par l'application est prise en compte :

- au niveau de la création de l'exécutable. En effet, il est possible de créer un exécutable contenant uniquement certaines langues spécifiées dans le projet.

- au niveau de l'installation de l'exécutable. En effet, le programme d'installation peut être multilingue.




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Création de l'exécutable : langues prises en compte
<a name="creation_executable_langues_prises_compte_ELTTEXTE000140"></a>
La langue d'exécution du projet correspond à la langue utilisée par défaut définie dans le projet : 

- Sous le volet "Projet", dans le groupe "Projet", cliquez sur "Description".

- Cliquez sur l'onglet "Langues" : la langue est identifiée par le numéro 1.




Lors de la création de l'exécutable, il est possible de choisir les langues à intégrer dans l'exécutable.

Cette option permet de créer une bibliothèque (.WDL) plus petite en prenant en compte uniquement la ou les langues choisie(s). Cette option est intéressante dans plusieurs cas :

- le projet est associé à plusieurs langues dont certaines traductions n'ont pas été réintégrées. Les traductions n'étant pas disponibles, les langues correspondantes n'ont pas à être intégrées dans l'exécutable.

- le projet gère plusieurs langues, mais l'exécutable n'est livré que dans une seule langue. Les autres langues n'ont pas à être intégrées dans l'exécutable.


Pour plus de détails sur la création de l'exécutable, consultez [Créer un exécutable WINDEV ou WINDEV Mobile](../Editeurs/2025002.md). 

**Remarque** : Les configurations de projet peuvent également être utilisées pour gérer les langues d'un projet. Il est par exemple possible de créer une configuration spécifique pour générer une bibliothèque FR + US pour un projet contenant 5 langues, dont 3 non traduites. Pour plus de détails, consultez [Configuration de projet](../Editeurs/9000030.md).

## Installation de l'application
<a name="installation_application_ELTTEXTE000164"></a>
Par défaut, la procédure d'installation des applications créées avec WINDEV est fournie en anglais et en français.

Il est possible de proposer une procédure d'installation dans une autre langue : il suffit de traduire le programme d'installation proposé par WINDEV.

Ce programme est livré parmi les exemples WINDEV, sous la forme d'un projet (de nom WDSETUP). Ainsi, il est possible de traduire tous les messages de ce projet (interface, message du code, ...).

Pour plus de détails, consultez [Gérer des langues supplémentaires dans le programme d'installation](../Editeurs/2028037.md).




## &lt;Traitement de texte&gt;.UndoOccurrence (Fonction)

***En anglais : &lt;Word Processing&gt;.UndoCount***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le nombre d'actions annulables sur un champ Traitement de texte.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affiche le nombre d'actions annulables sur le champ
Info(TT_MonDocument.UndoOccurrence())
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Champ Traitement de texte>.UndoOccurrence()`
---

**`<Résultat> : (Entier)`**

Nombre d'actions annulables.

**`<Champ Traitement de texte> : (Nom de champ)`**

Nom du champ Traitement de texte à manipuler.

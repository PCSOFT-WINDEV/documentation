
## iCalendarVersPlanning (Fonction)

***En anglais : iCalendarToScheduler***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Importe des données au format iCalendar dans un champ Planning.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
siCal est une chaîne ANSI
siCal = fChargeBuffer("calendar.ics")

iCalendarVersPlanning(PLN_MonPlanning, "user", siCal)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`iCalendarVersPlanning(<Champ Planning> , <Ressource> , <Données iCalendar>)`
---

**`<Champ Planning> : (Nom de champ)`**

Nom du champ Planning à manipuler. Ce champ peut correspondre :

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) au champ Planning d'une fenêtre,

- au champ Planning d'un état.




**`<Ressource> : (Chaîne de caractères)`**

Nom de la ressource associée aux rendez-vous. 

**`<Données iCalendar> : (Chaîne de caractères ANSI au format UTF-8)`**

Données au format iCalendar.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Pour exporter les données d'un champ Planning au format iCalendar, utilisez la fonction [PlanningVersiCalendar](../WDLang1/1000021920.md). 

- ![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) Cette fonction est disponible uniquement dans les sites WEBDEV pour Linux. Cette fonction n'est pas disponible dans les applications WINDEV pour Linux.

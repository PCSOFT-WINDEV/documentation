
## ActionRedo (Fonction)

***En anglais : ActionRedo***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Exécute la dernière action annulée. Cette fonction est équivalente à un "Redo" (Ctrl + Y) de l'utilisateur. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Bouton [Rétablir]
ActionRedo()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`ActionRedo([<Fenêtre à manipuler>])`
---

**`<Fenêtre à manipuler> : (Nom de fenêtre optionnel)`**

Nom de la fenêtre ou de la fenêtre interne à manipuler.

- Si ce paramètre n'est pas précisé, l'action est exécutée dans la fenêtre en cours. 

- Si ce paramètre est précisé, la fonction **ActionRedo** est équivalente à la fonction [FenEdRedo](../WDLang1/1000021905.md).






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Conditions d'utilisation
<a name="conditions_utilisation_ELTPARAGRAPHE000041"></a>

- Si aucune action n'a été annulée, la fonction **ActionRedo** ne fait rien. 

- Si la dernière action est une action personnalisée exécutée par la fonction [ActionDo](../WDLang1/1000022676.md), la procédure déclarée dans cette fonction est exécutée.

- L'action rétablie peut être à nouveau annulée avec la combinaison de touches Ctrl + Z ou par programmation grâce à la fonction [ActionUnDo](../WDLang1/1000022677.md).


## SupprimeCommandeVocale (Fonction)

***En anglais : DeleteVoiceCommand***






	<B><B>Avertissement</B></B>

	A partir de la version <B><B>17</B></B>,  cette fonction est conservée par compatibilité. Cette fonction est remplacée par la fonction [ReconnaissanceVocaleSupprimeCommande](../WDLang1/1000020017.md). 



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime une commande vocale précédemment ajoutée par la fonction [AjouteCommandeVocale](../WDLang1/1000017106.md).

**Remarque** : un moteur de reconnaissance vocale doit être installé sur le poste d'exécution. Windows Vista est fourni par défaut avec un moteur de reconnaissance vocale. Pour les autres systèmes, il est nécessaire d'installer le logiciel "Dragon Naturally Speaking" (tests réalisés avec la version 9 "Preferred" ou "Standard").
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Lance la procédure "Proc_Anim_Fenêtre" si la commande vocale est reconnue
IDVoix1 est un entier
IDVoix1 = AjouteCommandeVocale("Proc_Anim_Fenêtre", "Animer la fenêtre")
...
// Supprime la commande vocale
SupprimeCommandeVocale(IDVoix1)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`SupprimeCommandeVocale(<Identifiant de la commande vocale>)`
---

**`<Identifiant de la commande vocale> : (Entier)`**

Identifiant de la commande vocale à supprimer. Cet identifiant a été donné par la fonction [AjouteCommandeVocale](../WDLang1/1000017106.md).

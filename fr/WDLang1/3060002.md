
## PoliceCrée (Fonction)

***En anglais : FontCreate***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Crée une définition de police. Cette police pourra être utilisée :

- pour écrire du texte dans des dessins (fonction [dPolice](../WDLang1/3029020.md)).

- pour sélectionner la police utilisée dans les graphes (fonctions [grPoliceEtiquette](../WDLang3/3042012.md), [grPoliceLégende](../WDLang3/3042037.md), [grPoliceTitre](../WDLang3/3042048.md), [grPoliceTitreAxe](../WDLang3/3042059.md)).

- pour sélectionner une police d'impression (fonction [iPolice](../WDLang5/3046014.md)).

- pour définir la police utilisée dans les champs d'une fenêtre, d'une page ou d'un état (propriété [Police](../Proprietes/2515045.md)).





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Création d'une police
MaPolice est une police 
MaPolice = PoliceCrée("Arial", 12, iGras, JauneClair)
// Utilisation de cette police dans le champ "NomClient"
NomClient.Police = MaPolice
```




<a name="Exemple2"></a>


![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) 
```wl
// Création d'une police normale
PNormal est une police = PoliceCrée("Arial", 12)
// Création d'une police en gras
PGras est une police = PoliceCrée("Arial", 12, iGras)
// Préparation de l'impression
iDestination(iPDF)
// Parcours d'un tableau
i est un entier
POUR i = 1 A 100
	// Utilisation de la police normale pour les 50 premiers éléments du tableau
	SI i<=50 ALORS
		iPolice(PNormal)
	SINON
		// Utilisation de la police en gras pour les 50 derniers éléments du tableau
		iPolice(PGras)
	FIN
FIN
// Lancement de l'impression
iFinImprime()
```

## Syntaxe

`<Résultat> = PoliceCrée(<Nom de la police> [, <Taille> [, <Style> [, <Couleur> [, <Angle>]]]])`
---

**`<Résultat> : (Variable de type Police)`**

Variable de type [Police](../Motscles/1514045.md) pouvant être utilisée dans les différentes fonctions de dessin, de graphe ou d'impression utilisant une police.

**`<Nom de la police> : (Chaîne de caractères)`**

Nom de la police utilisée. Ce nom doit correspondre à une des polices installées sur le poste en cours.

**`<Taille> : (Entier optionnel)`**

Taille de la police exprimée en points pica (1 point Pica = 0,353 mm). Par défaut, la taille de la police est 12.

**`<Style> : (Constante ou combinaison de constantes optionnelle de type entier)`**

Attributs de la police :


|   |   |
| --- | --- |
| *iBarre* | Police barrée |
| *iCondense* | Caractères condensés |
| *iEtendu* | Caractères étendus |
| *iGras* | Police en gras |
| *iItalique* | Police italique |
| *iLarge* | Caractères larges |
| *iNormal*<br>(valeur par défaut) | Police normale, sans attributs |
| *iSouligné* | Police soulignée |



**`<Couleur> : (Entier optionnel)`**

Couleur utilisée pour la police. Si ce paramètre n'est pas précisé, la couleur utilisée est le noir. 
Cette couleur peut correspondre à :

- une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)).

- une couleur TSL (obtenue avec la fonction [TSL](../WDLang1/3029057.md)),

- une [couleur prédéfinie dans le WLangage](../WDLang5/3010002.md).




**`<Angle> : (Entier optionnel)`**

Angle (en degrés) selon lequel le texte sera imprimé. L'angle est calculé à partir d'une ligne horizontale, dans le sens inverse des aiguilles d'une montre (sens trigonométrique).

## Remarques




### Différences entre la fonction PoliceCrée et la fonction iCréePolice
<a name="differences_entre_fonction_docparampagetitleshort_fonction_icreepolice_ELTPARAGRAPHE000142"></a>La fonction **PoliceCrée** permet de créer une définition de police utilisable dans tous les éléments d'une application WINDEV ou WEBDEV. Cette police est mémorisée grâce au type de [variable Police](../Motscles/1514045.md) et pourra être utilisée :

- pour écrire du texte dans des dessins (fonction [dPolice](../WDLang1/3029020.md)).

- pour sélectionner la police utilisée dans les graphes (fonctions [grPoliceEtiquette](../WDLang3/3042012.md), [grPoliceLégende](../WDLang3/3042037.md), [grPoliceTitre](../WDLang3/3042048.md), [grPoliceTitreAxe](../WDLang3/3042059.md)).

- pour sélectionner une police d'impression (fonction [iPolice](../WDLang5/3046014.md)).

- pour définir la police utilisée dans les champs d'une fenêtre (propriété [Police](../Proprietes/2515045.md) des champs d'une fenêtre).

- pour définir la police utilisée dans les champs d'une page (propriété [Police](../Proprietes/2515045.md) des champs d'une page).

- pour définir la police utilisée dans les champs d'un état (propriété [Police](../Proprietes/2515045.md) des champs d'un état).




La fonction [iCréePolice](../WDLang5/3046045.md) permet uniquement de créer une police utilisable par les fonctions d'impression.




## FenAgencementEnCours (Fonction)

***En anglais : WinCurrentLayout***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie l'indice de l'agencement en cours : 

- dans la fenêtre en cours.

- dans la fenêtre ou dans la fenêtre interne spécifiée.





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI FenAgencementEnCours(FEN_Détail) = 2 ALORS
	LIB_DETAIL = CLIENT.DETAIL
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître l'agencement de la fenêtre en cours

`<Résultat> = FenAgencementEnCours()`
---

**`<Résultat> : (Entier)`**



- Indice de l'agencement en cours.

- 0 si la fenêtre ne possède aucun agencement. 





<a name="SYNTAXE2"></a>

### Connaître l'agencement d'une fenêtre spécifique

`<Résultat> = FenAgencementEnCours(<Fenêtre manipulée>)`
---

**`<Résultat> : (Entier)`**



- Indice de l'agencement en cours.

- 0 si la fenêtre interne ne possède aucun agencement. 




**`<Fenêtre manipulée> : (Nom de fenêtre)`**

Nom de la fenêtre ou de la fenêtre interne manipulée. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Les agencements sont numérotés dans leur ordre de création en édition.

- Si le terminal change d'orientation ou si la fonction [FenChangeAgencement](../WDLang1/1000021056.md) est exécutée, l'agencement change.

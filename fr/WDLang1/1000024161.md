


## &lt;Type DateHeure&gt;.VersEpoch (Fonction)

***En anglais : &lt;DateTime type&gt;.ToEpoch***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Convertit une date-heure locale vers le format Epoch (nombre d'unités de temps depuis le 01/01/1970 à 00h00 UTC).




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonJour est un DateHeure
MonJour = "20170726100941"
// Affiche "1501056581" (en France)
Info(MonJour.VersEpoch())
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Donnée à convertir>.VersEpoch([<Options de conversion>])`
---

**`<Résultat> : (Entier sur 8 octets)`**



- Date et heure au format Epoch (nombre d'unités de temps depuis le 01/01/1970 à 00h00 UTC).

- -1 en cas d'erreur (date et heure invalides).




**`<Donnée à convertir> : (Variable de type DateHeure)`**

Variable de type DateHeure correspondant à la date et heure locale à convertir. Ce paramètre sera considéré invalide dans les cas suivants : 

- si la date est invalide ou antérieure au 01/01/1970. La validité de la date peut être vérifiée avec la fonction [DateValide](../WDLang1/3027003.md).

- si l'heure est invalide. La validité de l'heure peut être vérifiée avec la fonction [HeureValide](../WDLang1/3027021.md).


Remarque : Il est possible d'utiliser des dates au-delà de l'année 2038. 

**`<Options de conversion> : (Constante optionnelle de type Entier)`**

Options de conversion de la date et de l'heure au format Epoch :



|   |   |
| --- | --- |
| *epochUnixMilliseconde* | Le résultat sera exprimé en nombre de millisecondes depuis le 01/01/1970 à 00h00 UTC (temps 0 sous Unix). |
| *epochUnixSeconde*<br>(valeur par défaut) | Le résultat sera exprimé en nombre de secondes depuis le 01/01/1970 à 00h00 UTC (temps 0 sous Unix). |


## Suisse.VérifieREE (Fonction)

***En anglais : Switzerland.CheckBER***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Teste la validité d'un numéro REE (numéro dans le registre des entreprises et des établissements identifiant un établissement d'une entreprise en Suisse).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Test d'une variable
MonNuméroREE est une chaîne = "73462600"
Suisse.VérifieREE(MonNuméroREE)		// renvoie Vrai

// Test d'un champ
SAI_NuméroREE = "ABC"
Suisse.VérifieREE(SAI_NuméroREE)	// renvoie Faux
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = Suisse.VérifieREE(<Numéro à vérifier>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'élément spécifié contient un numéro REE valide.

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Numéro à vérifier> : (Nom de champ ou variable de type chaîne)`**

Numéro REE à vérifier. Ce paramètre peut correspondre :

- au nom d'une variable de type chaîne.

- au nom d'un champ.

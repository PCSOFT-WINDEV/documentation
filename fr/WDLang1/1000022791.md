
## FAAPossible (Fonction)

***En anglais : AAFPossible***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Indique si une FAA (Fonctionnalité Automatique de vos Applications) peut être exécutée sur un champ spécifique.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Grise le champ Bouton "BTN_Annuler" si le Undo n'est pas possible
SI FAAPossible(TBLR_MonTableur, faaUndo) ALORS
	BTN_Annuler.Grisé = Faux
SINON
	BTN_Annuler.Grisé = Vrai
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = FAAPossible(<Champ> , <Fonctionnalité à utiliser>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la FAA (Fonctionnalité Automatique de vos Applications) spécifiée peut être utilisée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 




**`<Champ> : (Nom de champ)`**

Nom du champ à manipuler, pour lequel la disponibilité d'une FAA (Fonctionnalité Automatique de vos Applications) est recherchée. Ce paramètre peut correspondre à : 

- un champ Tableur. 

- une fenêtre interne en mode Edition (en exécution). 




**`<Fonctionnalité à utiliser> : (Constante de type Entier)`**

Fonctionnalité Automatique de vos Applications à manipuler : 


|   |   |
| --- | --- |
| *faaColler* | Permet de savoir si le "Coller" est disponible pour le champ. <br><br>Champ manipulé : Champ Tableur. |
| *faaCopier* | Permet de savoir si le "Copier" est disponible pour le champ. <br><br>Champ manipulé : Champ Tableur. |
| *faaCouper* | Permet de savoir si le "Couper" est disponible pour le champ. <br><br>Champ manipulé : Champ Tableur. |
| *faaEnregistrerSous* | Permet de savoir si le contenu du champ peut être enregistré sous un autre nom.  |
| *faaRedo* | Permet de savoir si le "redo" (refaire la dernière action) est disponible pour le champ. |
| *faaUndo* | Permet de savoir si le "undo" (annuler la dernière action) est disponible pour le champ. |


Si une autre constante *faaxxx* est utilisée, &lt;Résultat&gt; vaut <u><u><u><u>Faux</u></u></u></u>.

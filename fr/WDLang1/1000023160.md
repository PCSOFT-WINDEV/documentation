


## docColonne (Type de variable)

***En anglais : docColumn***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **docColonne** permet de définir toutes les caractéristiques avancées d'une colonne d'un tableau présent dans un document de type Traitement de texte. Les caractéristiques de cette colonne de tableau peuvent être définies et modifiées à l'aide de différentes propriétés WLangage. 

**ATTENTION : Ce type ne doit pas être utilisé directement.** Il doit être utilisé uniquement via les variables de type [docParagraphe](../WDLang1/1000022483.md) correspondant à un tableau. 


<a name="NOTE0"></a>

## Remarques
<a name="NOTE0_1"></a>


### Propriétés spécifiques à la description des variables de type docColonne
<a name="proprietes_specifiques_description_des_variables_type_doccolonne_ELTPARAGRAPHE000028"></a>Les propriétés suivantes peuvent être utilisées pour manipuler une variable de type **docColonne** :

| Nom de la propriété | Type manipulé | Effet |
| --- | --- | --- |
| LargeurMillimètres | Réel | Largeur de la colonne (en millimètres). |
| ModeLargeur | Constante de type Entier | Mode de calcul de la largeur de la colonne : <br><br>- *docModeLargeurAucune* : Cas particulier. <br><br>- *docModeLargeurAutomatique* : La largeur de la colonne est calculée automatiquement en fonction du texte présent dans la colonne en cours et de celui présent dans les autres colonnes. <br><br>- *docModeLargeurNonDéfini* : Le mode de calcul de la largeur de la colonne n'est pas défini, car des cellules de la colonne ont des largeurs différentes). <br><br>- *docModeLargeurPourcent* : Le mode de calcul de la largeur de la colonne est effectué en pourcentage de la largeur totale du tableau. <br><br>- *docModeLargeurValeur* : Le mode de calcul de la largeur des colonnes est effectué selon une largeur fournie en millimètres. Si la propriété **LargeurMillimètres** est renseignée, ce mode est automatiquement sélectionné. <br><br><br><br><br>Cette propriété est disponible en lecture seulement. |

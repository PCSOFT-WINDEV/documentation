
## Constantes de débogage
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les constantes de gestion du débogage sont les suivantes :

| Constante | Description | Utilisée par les fonctions |
| --- | --- | --- |
| **AuditActif** | L'audit dynamique est immédiatement actif. | [dbgActiveAudit](../WDLang1/1000018835.md), [dbgEtatAudit](../WDLang1/1000018836.md) |
| **AuditAssertion** | Les assertions sont inscrites dans l'audit dynamique. | [dbgActiveAudit](../WDLang1/1000018835.md), [dbgOptionAudit](../WDLang1/1000018837.md) |
| **AuditCommentaire** | Les commentaires d'audit sont inscrits dans l'audit dynamique. | [dbgActiveAudit](../WDLang1/1000018835.md), [dbgOptionAudit](../WDLang1/1000018837.md) |
| **AuditErreur** | Les erreurs non fatales non traitées sont inscrites dans l'audit dynamique. | [dbgActiveAudit](../WDLang1/1000018835.md), [dbgOptionAudit](../WDLang1/1000018837.md) |
| **AuditException** | Les erreurs fatales, traitées par QUAND EXCEPTION ou non traitées, sont inscrites dans l'audit dynamique. | [dbgActiveAudit](../WDLang1/1000018835.md), [dbgOptionAudit](../WDLang1/1000018837.md) |
| **AuditInactif** | L'audit dynamique n'a pas été activé. | [dbgEtatAudit](../WDLang1/1000018836.md) |
| **AuditPause** | L'audit dynamique est en pause. | [dbgActiveAudit](../WDLang1/1000018835.md), [dbgEtatAudit](../WDLang1/1000018836.md) |
| **AuditTout** | Tous les événements d'audit sont inscrits dans l'audit dynamique. | [dbgActiveAudit](../WDLang1/1000018835.md) |
| **AuditWarningAnomalie** | Les warnings d'exécution concernant les anomalies détectées sont inscrits dans l'audit dynamique. | [dbgActiveAudit](../WDLang1/1000018835.md), [dbgOptionAudit](../WDLang1/1000018837.md) |
| **AuditWarningPerformance** | Les warnings d'exécution concernant les performances sont inscrits dans l'audit dynamique. | [dbgActiveAudit](../WDLang1/1000018835.md), [dbgOptionAudit](../WDLang1/1000018837.md) |
| **AuditWarningUtilisateur** | Les warnings d'exécution personnalisés sont inscrits dans l'audit dynamique. | [dbgActiveAudit](../WDLang1/1000018835.md), [dbgOptionAudit](../WDLang1/1000018837.md) |
| **dbgActiveDumpSystème** | Information de débogage : Active ou désactive la génération de dump système pour faire remonter certains problèmes au Support Technique. | [dbgInfo](../WDLang1/3014024.md) |
| **dbgActiveDumpUtilisateur** | Information de débogage : Active ou désactive le dump mémoire utilisateur (Ctrl + Pause). | [dbgInfo](../WDLang1/3014024.md) |
| **dbgDébogueurActif** | Information de débogage : Permet de savoir si le débogueur est actif ou non. | [dbgInfo](../WDLang1/3014024.md) |
| **dbgElément** | Information de débogage : l'information concerne le traitement en cours d'exécution. | [dbgInfo](../WDLang1/3014024.md) |
| **dbgLigne** | Information de débogage : Numéro de la ligne en cours. | [dbgInfo](../WDLang1/3014024.md) |
| **dbgNomCompletComposant** | Information de débogage : Nom complet du composant en cours d'exécution. | [dbgInfo](../WDLang1/3014024.md) |
| **dbgNomCourtElément** | Information de débogage : <br><br>- Nom du champ associé à l'événement WLangage en cours d'exécution, <br><br>- Nom de la procédure en cours d'exécution. <br><br><br> | [dbgInfo](../WDLang1/3014024.md) |
| **dbgNomType** | Information de débogage : nom du type en toutes lettres. | [dbgInfo](../WDLang1/3014024.md) |
| **dbgPile** | Information de débogage : pile des appels. | [dbgInfo](../WDLang1/3014024.md) |
| **dbgRépertoireDumpSystème** | Information de débogage : Répertoire de génération des dumps système. | [dbgInfo](../WDLang1/3014024.md) |
| **dbgTraitement** | Information de débogage : information concernant le traitement. | [dbgInfo](../WDLang1/3014024.md) |
| **dbgTraitementAppelant** | Information de débogage : information concernant le traitement appelant. | [dbgInfo](../WDLang1/3014024.md) |
| **dbgTraitementEnCours** | Information de débogage : information concernant le traitement en cours. | [dbgInfo](../WDLang1/3014024.md) |
| **LogActif** | Le log d'exécution est actif. | [dbgActiveLog](../WDLang1/1000017137.md), [dbgEtatLog](../WDLang1/1000017138.md) |
| **LogFonctionsWLangage** | Les fonctions WLangage sont inscrites dans le log d'exécution. | [dbgActiveLog](../WDLang1/1000017137.md), [dbgOptionLog](../WDLang1/1000017136.md) |
| **LogInactif** | Le log d'exécution n'a pas été activé. | [dbgEtatLog](../WDLang1/1000017138.md) |
| **LogParamètres** | Les paramètres des fonctions sont inscrits dans le log d'exécution. | [dbgActiveLog](../WDLang1/1000017137.md), [dbgOptionLog](../WDLang1/1000017136.md) |
| **LogPause** | Le log d'exécution n'est pas actif. | [dbgActiveLog](../WDLang1/1000017137.md), [dbgEtatLog](../WDLang1/1000017138.md) |
| **LogTout** | Le log d'exécution prend en compte toutes les options : fonctions WLangage, paramètres des fonctions, traitements. | [dbgActiveLog](../WDLang1/1000017137.md), [dbgOptionLog](../WDLang1/1000017136.md) |
| **LogTraitementUniquement** | Le log d'exécution récupère uniquement les informations des traitements. | [dbgActiveLog](../WDLang1/1000017137.md), [dbgOptionLog](../WDLang1/1000017136.md) |

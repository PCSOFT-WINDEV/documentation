
## ArrondiAuMultiple (Fonction)

***En anglais : RoundToMultiple***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie une valeur numérique arrondie au multiple le plus proche d'une autre valeur numérique.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
ArrondiAuMultiple(10, 9)
// Renvoie 9 

ArrondiAuMultiple(-10, -3)
// Renvoie -9

ArrondiAuMultiple(1.3, 0.2)
// Renvoie 1.4

ArrondiAuMultiple(2.64, 0.25)
// Renvoie 2.75
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = ArrondiAuMultiple(<Valeur numérique> , <Valeur multiple>)`
---

**`<Résultat> : (Variant)`**

Valeur de &lt;Valeur numérique&gt; arrondie au multiple le plus proche de &lt;Valeur multiple&gt;.

**`<Valeur numérique> : (Numérique)`**

Valeur numérique à arrondir.

**`<Valeur multiple> : (Numérique)`**

Valeur dont le multiple le plus proche doit être calculé.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Les arrondis avec le type "Réel" ne sont pas précis. En effet, les opérations avec le type "réel" ne sont pas précises du fait de la représentation informatique des réels. Pour obtenir un arrondi exact, il est conseillé d'utiliser le type Monétaire (ou Numérique) qui utilise une représentation mémoire exacte. Pour plus de détails, consultez [le type Réel](../Motscles/1514048.md).

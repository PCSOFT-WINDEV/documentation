
## TexteVersHTML (Fonction)

***En anglais : TextToHTML***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Convertit une chaîne texte en chaîne HTML (les retours chariots sont remplacés par des &lt;BR&gt;, les accents sont remplacés par leur équivalent en HTML, ...).




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// SAI_HTML est un champ de saisie
SAI_HTML = TexteVersHTML("WINDEV est un outil formidable" + RC + ...
	"Le développement devient un jeu d'enfant")
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = TexteVersHTML(<Texte> [, <Alphabet utilisé>])`
---

**`<Résultat> : (Chaîne de caractères)`**

Texte au format HTML. Attention : les balises &lt;html&gt; et &lt;/html&gt; ne sont pas intégrées dans le résultat. Si le résultat doit être affiché dans un champ HTML, il est nécessaire de les ajouter.

**`<Texte> : (Chaîne de caractères)`**

Nom du champ de type texte ou bien chaîne de caractères au format texte.  

**`<Alphabet utilisé> : (Constante optionnelle de type Entier)`**

Constante identifiant l'alphabet utilisé pour l'écriture du &lt;Texte&gt;. Par défaut, l'alphabet en cours de WINDEV/WEBDEV est utilisé (constante *alphabetCourant*). 
Pour plus de détails sur ces constantes, consultez [Correspondance entre les langues, les sous-langues, les alphabets et les nations](../WDLang1/3054022.md).





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Pour convertir une chaîne RTF au format HTML, utilisez la fonction [RTFVersHTML](../WDLang1/3025049.md).

- Pour convertir une chaîne RTF au format Texte, utilisez la fonction [RTFVersTexte](../WDLang1/3025032.md).


## MotOccurrence (Fonction)

***En anglais : WordCount***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le nombre de mots d'une chaîne. 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// La fonction renvoie 3
ResOccurrence = MotOccurrence("Exemple d'utilisation")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = MotOccurrence(<Chaîne>)`
---

**`<Résultat> : (Entier)`**

Nombre de mots dans la chaîne de caractères à analyser. 

**`<Chaîne> : (Chaîne de caractères)`**

Chaîne de caractères à manipuler. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Tous les caractères autres que numériques ou alphabétiques sont considérés comme des séparateurs de mots.

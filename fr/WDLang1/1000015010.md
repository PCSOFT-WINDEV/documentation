


## &lt;Xaml&gt;.Propriété (Fonction)

***En anglais : &lt;Xaml&gt;.Property***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de connaître ou de modifier la valeur d'une propriété .Net d'un champ Xaml ou d'un champ Carrousel.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// -- Code d'initialisation du champ Xaml DOTNET_Xaml_OK
MaChaîne est une chaîne = [
<Button Name="Test" Width=20>OK</Button>
]
DOTNET_Xaml_OK = MaChaîne

// Valeur de deux propriétés
Info("Nom du contrôle : ", DOTNET_Xaml_OK.Propriété("Test", "Name"))
Info("Taille de la police : ", DOTNET_Xaml_OK.Propriété("Test", "FontSize"))
// Erreur ?
SI ErreurDétectée = Vrai ALORS
	Erreur(ErreurInfo())
FIN
```
<a name="Exemple2"></a>

```wl
// -- Code d'initialisation du champ Xaml DOTNET_Xaml_OK
MaChaîne est une chaîne = [
<Button Name="Test" Width=20>OK</Button>
]
DOTNET_Xaml_OK = MaChaîne

// Modification d'une propriété
DOTNET_Xaml_OK.Propriété("Test", "FontSize", 15)
// Erreur ?
SI ErreurDétectée = Vrai ALORS
	Erreur(ErreurInfo())
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître la valeur d'une propriété .Net

`<Résultat> = <Champ Xaml>.Propriété(<Nom du contrôle .Net> , <Nom de la propriété>)`
---

**`<Résultat> : (Type dépendant de la propriété spécifiée)`**

Valeur de la propriété indiquée.

**`<Champ Xaml> : (Nom de champ)`**

Nom du champ Xaml.

**`<Nom du contrôle .Net> : (Chaîne de caractères)`**

Nom du contrôle .Net dont on veut connaître une propriété.

Ce nom est sensible à la casse.

**`<Nom de la propriété> : (Chaîne de caractères)`**

Nom de la propriété .Net dont on veut connaître la valeur. Pour plus de détails sur les propriétés .Net, consultez la documentation .Net.

Ce nom est sensible à la casse.


<a name="SYNTAXE2"></a>

### Modifier la valeur d'une propriété .Net

`<Champ Xaml>.Propriété(<Nom du contrôle .Net> , <Nom de la propriété> , <Nouvelle valeur>)`
---

**`<Champ Xaml> : (Nom de champ)`**

Nom du champ Xaml.

**`<Nom du contrôle .Net> : (Chaîne de caractères)`**

Nom du contrôle .Net dont on veut modifier une propriété.

Ce nom est sensible à la casse.

**`<Nom de la propriété> : (Chaîne de caractères)`**

Nom de la propriété .Net à modifier. Pour plus de détails sur les propriétés .Net, consultez la documentation .Net.

Ce nom est sensible à la casse.

**`<Nouvelle valeur> : (Type dépendant de la propriété spécifiée)`**

Nouvelle valeur de la propriété indiquée.

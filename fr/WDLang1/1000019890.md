
## FenForceGFI (Fonction)

***En anglais : WinForceDDW***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Force le GFI (Grisage des Fenêtres Inaccessibles) sur une fenêtre même si cette fenêtre n'est pas inaccessible.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
Sablier(Vrai)
// Force le GFI sur la fenêtre principale
FenForceGFI(FEN_Principale, Vrai)
// Lance un calcul important
CalculImportant()
// Arrête le GFI sur la fenêtre principale
FenForceGFI(FEN_Principale, Faux)
Sablier(Faux)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`FenForceGFI(<Fenêtre> , <Activation du GFI>)`
---

**`<Fenêtre> : (Nom de fenêtre)`**

Nom de la fenêtre à manipuler. 

**`<Activation du GFI> : (Booléen)`**

Nouvel état du GFI :

- <u><u><u><u>Vrai</u></u></u></u> : le GFI est activé immédiatement sur la fenêtre spécifiée. La fenêtre s'affiche en grisé. 

- <u><u><u><u>Faux</u></u></u></u> : le GFI est désactivé immédiatement sur la fenêtre spécifiée. La fenêtre se réaffiche normalement (sauf si elle est inactive : elle reste alors grisée).


## MDIMenuFenêtre (Fonction)

***En anglais : MDIWindowMenu***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie l'agencement des fenêtres filles MDI.

**Remarque** : La fonction **MDIMenuFenêtre** est utilisable uniquement pour les fenêtres définies comme "filles MDI" sous l'éditeur de fenêtres.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Iconiser toutes les fenêtres filles MDI ouvertes
MDIMenuFenêtre(mdiIconiseTout)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`MDIMenuFenêtre(<Action>)`
---

**`<Action> : (Constante de type chaîne de caractères)`**

Indique l'action à effectuer :



|   |   |
| --- | --- |
| *mdiArrange* | Aligne les icônes des fenêtres filles MDI iconisées. |
| *mdiCascade* | Affiche les fenêtres filles MDI ouvertes en cascade. |
| *mdiFermeTout* | Ferme toutes les fenêtres filles MDI ouvertes. |
| *mdiIconiseTout* | Iconise toutes les fenêtres filles MDI ouvertes. |
| *mdiMosaïque* | Affiche les fenêtres filles MDI ouvertes en mosaïque. |
| *mdiMosaïqueH* | Affiche les fenêtres filles MDI ouvertes en mosaïque horizontale. |
| *mdiRestaureTout* | Affiche les fenêtres filles MDI ouvertes dans leur taille initiale. |

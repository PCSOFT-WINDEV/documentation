
## FinValActuelleNette (Fonction)

***En anglais : FinNetCurrentVal***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la valeur actuelle nette d'un investissement. Cet investissement est basé sur des flux financiers non constants.

**Rappel** : La valeur actuelle nette correspond à la valeur à la date du jour d'une série de décaissements (valeurs négatives) et d'encaissements (valeurs positives) futurs.

**Remarque** : D'un point de vue financier, un investissement sera estimé rentable si sa VAN (Valeur Actuelle Nette) est positive.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple
<a class="notetitle" target="_blank" href="$DOC$=1000003050053&name=finvalactuellenette_fonction&product=WD">Voir des exemples supplémentaires</a>

```wl
// Valeur actuelle nette d'un investissement. Le taux d'intérêt est de 8%.
ResValActuelleNette = FinValActuelleNette(8, "Matrice")
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = FinValActuelleNette(<Taux> [, <Nom de la matrice> [, <Numéro de la ligne/colonne> [, <Ligne/Colonne>]]])`
---

**`<Résultat> : (Réel ou monétaire)`**

Valeur actuelle nette de l'investissement.

**`<Taux> : (Réel)`**

Taux d'intérêt de l'investissement exprimé en pourcentage (1 par défaut). Par exemple, pour un taux d'investissement de 5%, ce paramètre sera égal à 5. Ce taux ne peut pas être nul.

**`<Nom de la matrice> : (Chaîne de caractères optionnelle)`**

Nom de la matrice (défini avec la fonction [MatCrée](../WDLang1/3050058.md)) contenant les encaissements et les décaissements entraînés par l'investissement.

**`<Numéro de la ligne/colonne> : (Entier optionnel)`**

Numéro de la ligne ou de la colonne contenant les données (1 par défaut).

**`<Ligne/Colonne> : (Booléen optionnel)`**



- <u><u><u><u>Vrai</u></u></u></u> (par défaut) si la série de valeurs correspond à une colonne de matrice, 

- <u><u><u><u>Faux</u></u></u></u> si la série de valeurs correspond à une ligne de matrice.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Élément de la matrice
<a name="element_matrice_ELTPARAGRAPHE000090"></a>La ligne ou la colonne de matrice analysée par la fonction **FinValActuelleNette** doit être composée des éléments suivants :


|   |   |
| --- | --- |
| Numéro de ligne/colonne | Contenu |
| 1 | Montant investi (décaissé) : négatif |
| 2 | Revenu de l'année 1 : encaissé ou décaissé |
| 3 | Revenu de l'année 2 : encaissé ou décaissé |
| ..... |   |
| N-1 | Revenu de l'année N-2 : encaissé ou décaissé |
| N | Revenu de l'année N-1 : encaissé ou décaissé |




<a name="NOTE0_2"></a>




### Gestion des erreurs
<a name="gestion_des_erreurs_ELTPARAGRAPHE000129"></a>**Attention** : La fonction **FinValActuelleNette** ne retourne pas de code d'erreur. Pour savoir si cette fonction a généré des erreurs, utilisez la fonction [FinErreur](../WDLang1/3050071.md). Le libellé de l'erreur peut être connu grâce à la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.


## DernierJourDeLaSemaine (Fonction)

***En anglais : LastDayOfWeek***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la date du dernier jour de la semaine correspondant à une date donnée. Cette fonction permet de trouver facilement à quelle date correspond le dimanche pour une date donnée. 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Clic sur le champ Bouton "Reste de la semaine"
SAI_DATE_FIN = DernierJourDeLaSemaine(SAI_DATE_DEBUT)
```
<a name="Exemple2"></a>

```wl
NbJourAvantWe = DateDifférence(DernierJourDeLaSemaine(MaDate), MaDate)-2
```
<a name="Exemple3"></a>

```wl
DernierJourDeLaSemaine("20111103") // Renvoie 20110611
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = DernierJourDeLaSemaine(<Date>)`
---

**`<Résultat> : (Chaîne de caractères ou Date)`**

Date au format AAAAMMJJ correspondant au dernier jour de la semaine.  

**`<Date> : (Chaîne de caractères ou Date)`**

Date au format AAAAMMJJ pour laquelle le dernier jour de la semaine est recherché.




## &lt;Fenêtre&gt;.Maximise (Fonction)

***En anglais : &lt;Window&gt;.Maximize***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Affiche une fenêtre WINDEV dans sa taille maximale.

**Remarque** : La fonction **&lt;Fenêtre&gt;.Maximise** réalise la même action que le bouton d'agrandissement présent dans le coin haut droit des fenêtres.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affichage de la fenêtre "FEN_SaisieClient" dans sa taille maximale
FEN_SaisieClient.Maximise()
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Fenêtre>.Maximise()`
---

**`<Fenêtre> : (Nom de fenêtre)`**

Nom de la fenêtre à afficher dans sa taille maximale.




## MenuInvisible (Fonction)

***En anglais : MenuInvisible***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Rend une option de menu invisible (menu contextuel ou déroulant). Les options situées après l'option modifiée sont décalées d'une ligne vers le haut.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// L'option "OPT_BarreIcone" devient invisible
MenuInvisible(OPT_BarreIcone)
```
<a name="Exemple2"></a>

```wl
// Rend invisible la sous-option "Sauvegarder" 
// du menu "Fichier" dans la fenêtre "FEN_FenMenu" 
// à partir de la fenêtre "Client"
MenuInvisible(FEN_FenMenu, Fichier.Sauvegarder)
// équivalent à FEN_FenMenu.Fichier.Sauvegarder..Visible = Faux
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`MenuInvisible([<Fenêtre>, ] <Option de menu>)`
---

**`<Fenêtre> : (Nom de fenêtre optionnel)`**

Nom de la fenêtre contenant l'option de menu. 
Si ce paramètre n'est pas spécifié, la fonction **MenuInvisible** manipulera la fenêtre en cours.

**`<Option de menu> : (Nom d'option)`**

Nom de l'option de menu à manipuler. Ce nom est défini dans la fenêtre de description de l'option de menu. 
Ce nom doit être spécifié sans guillemets.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Equivalence : La syntaxe suivante : 
	
	```txt
	MenuInvisible(<Option de menu>)
	```

	est équivalent à la propriété [Visible](../Proprietes/2510138.md) : 
	
	```txt
	<Option de menu>.Visible
	```


- Lorsqu'une option de menu est invisible, le raccourci clavier associé n'est plus utilisable.


## RVB (Fonction)

***En anglais : RGB***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Décrit une couleur à partir de ses composantes Rouge, Vert et Bleu.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Coloriage d'une zone en bleu clair
dRemplissage(10, 50, RVB(0, 255, 255))
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = RVB(<Composante rouge> , <Composante verte> , <Composante bleue>)`
---

**`<Résultat> : (Entier)`**

Couleur (24 bits) relative aux composantes fournies. Cette couleur est le résultat de la formule suivante :

Couleur = 65536 \* Composante Bleu + 256 \* Composante Verte + Composante Rouge.



**`<Composante rouge> : (Entier (de 0 à 255))`**

Portion de rouge dans la couleur.

**`<Composante verte> : (Entier (de 0 à 255))`**

Portion de vert dans la couleur.

**`<Composante bleue> : (Entier (de 0 à 255))`**

Portion de bleu dans la couleur.

<a name="NOTE0_3"></a>


### Divers
<a name="divers_ELTPARAGRAPHE000125"></a>

- Les couleurs demandées avec la fonction **RVB** peuvent être modifiées selon les capacités du mode écran en cours (VGA, 32 couleurs, etc.).

- **Important** : Il est conseillé, pour indiquer les couleurs, d'utiliser les constantes de couleurs définies dans le WLangage (voir [les couleurs prédéfinies du WLangage](../WDLang5/3010002.md)).

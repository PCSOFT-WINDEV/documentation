


## &lt;Html&gt;.Navigue (Fonction)

***En anglais : &lt;HTML&gt;.Navigate***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Lance une opération de navigation dans un champ d'affichage HTML.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Code du bouton Suivant
HTM_ChampHTML1.Navigue(htmlSuivant)
```
<a name="Exemple2"></a>

```wl
// Code du bouton Précédent
HTM_ChampHTML1.Navigue(htmlPrécédent)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Champ HTML>.Navigue(<Option de navigation>)`
---

**`<Champ HTML> : (Nom de champ)`**

Nom du champ d'affichage HTML à manipuler.

**`<Option de navigation> : (Constante)`**

Sens de la navigation :


|   |   |
| --- | --- |
| *htmlActualise* | Actualise la page en cours (équivalent au bouton "Actualiser" du navigateur). |
| *htmlDémarrage* | Affiche la page de démarrage (équivalent au bouton "Home" du navigateur). Pour obtenir la valeur initiale du champ d'affichage HTML, utilisez la propriété [ValeurInitiale](../Proprietes/2510128.md).<br> |
| *htmlPrécédent* | Affiche la page précédente (équivalent au bouton "BACK/Précédent" du navigateur). |
| *htmlStop* | Arrête le chargement des ressources utilisées par la page. L'événement "Chargement de ressources" du champ n'est pas exécuté. |
| *htmlSuivant* | Affiche la page suivante (équivalent au bouton "SUIVANT" du navigateur). |


## Palette (Type de variable)

***En anglais : Palette***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **Palette** permet de définir toutes les caractéristiques avancées d'une palette de couleurs. Les caractéristiques de cette palette peuvent être définies et modifiées à l'aide de différentes propriétés WLangage. 

Ce type de variable est utilisé par le type [Icône](../WDLang1/1000020811.md).  

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Récupération de la première couleur de la palette de la première image d'une icône
MonIcône est une Icône = "MonIcone.ico"
Palette1 est une Palette = MonIcône.Palette[1]
Couleur1 est une Couleur = Palette1[1]
```

## Remarques
<a name="NOTE0_1"></a>

| --- | --- | --- |




## ListeSupprime (Fonction)

***En anglais : ListDelete***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime un élément d'un champ Liste par programmation (y compris Liste image) ou d'un champ Combo par programmation.

**Remarque** : La fonction **ListeSupprime** peut être utilisée indifféremment sur les champs Liste monosélection et multisélection.











<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple
<a class="notetitle" target="_blank" href="$DOC$=1000003049016&name=listesupprime_fonction&product=WD">Voir des exemples supplémentaires</a>

```wl
// Suppression de l'élément sélectionné dans le champ "LISTE_ListeClient"
ListeSupprime(LISTE_ListeClient)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`ListeSupprime(<Champ Liste> [, <Indice de l'élément>])`
---

**`<Champ Liste> : (Nom de champ)`**

Nom du champ Liste (ou Combo) à manipuler. Si ce paramètre correspond à une chaîne vide (""), le champ manipulé est le champ Liste (ou Combo) auquel appartient l'événement en cours.


**`<Indice de l'élément> : (Entier optionnel)`**

Indice de l'élément à supprimer du champ Liste (ou Combo). Si ce paramètre n'est pas précisé :

- dans un champ Liste monosélection, l'élément en cours est supprimé.

- dans un champ Liste multisélection, les éléments sélectionnés dans le champ Liste sont supprimés.

- dans un champ Combo, l'élément en cours est supprimé. Si ce champ est un champ Combo est en saisie, le champ de saisie est mis à vide.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Suppression en chaîne
<a name="suppression_chaine_ELTPARAGRAPHE000111"></a>Dans le cas de la suppression en chaîne d'éléments, les éléments situés après chaque élément supprimé sont déplacés. Il est conseillé d'effectuer les suppressions en chaîne en partant du dernier élément.

Par exemple :

```wl
Indice est un entier
POUR Indice = ListeOccurrence(LISTE_ListeClient) A 1 PAS -1
	ListeSupprime(LISTE_ListeClient, Indice)
FIN
```

<a name="NOTE0_2"></a>
<a name="NOTE0_3"></a>


### Divers
<a name="divers_ELTPARAGRAPHE000164"></a>

- Pour supprimer un élément dans un champ Table, utilisez la fonction [TableSupprime](../WDLang1/3074032.md).

- La fonction **ListeSupprime** peut être utilisée sur une colonne d'un champ Table de type "Combo".

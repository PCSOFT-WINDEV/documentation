


## Pays (Type de variable)

***En anglais : Country***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **Pays** permet de connaître les caractéristiques d'un pays. Les caractéristiques de ce pays peuvent être : 

- récupérées avec les fonctions [PaysRécupère](../WDLang1/1000024931.md) ou [PaysListe](../WDLang1/1000024930.md). 

- connues à l'aide de différentes propriétés WLangage. 




**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Récupération des informations sur la France 
oFrance est un Pays = PaysRécupère("FR")
// Affichage du libellé
LIB_Pays = oFrance.Nom
// Affichage du préfixe de numéro de téléphone
SAI_Téléphone = oFrance.PréfixeNuméroDeTéléphone
// Affichage drapeau
IMG_Drapeau = oFrance.Drapeau
// Affichage du continent
LIB_Continent = oFrance.Continent.Nom
```



```wl
// Trinité et Tobago en FR et en US
MonPays est un Pays = PaysRécupère("TT")
// Affichage du pays en FR
Trace(MonPays.Nom) // Trinité-et-Tobago
// Affichage du pays en US
// Changement de la langue du framework
NationFramework(nationAnglais)
Trace(MonPays.Nom) // Trinidad and Tobago
```

## Remarques
<a name="NOTE0_1"></a>


### Propriétés spécifiques à la description des variables de type Pays
<a name="proprietes_specifiques_description_des_variables_type_pays_ELTPARAGRAPHE000050"></a>Les propriétés suivantes peuvent être utilisées pour manipuler une variable de type **Pays** :

| Nom de la propriété | Type manipulé | Effet |
| --- | --- | --- |
| CodeISO2 | Chaîne de caractères | Code ISO sur 2 caractères (norme ISO 3166-1). <br><br>**Cette propriété est disponible en lecture seulement**. |
| CodeISO3 | Chaîne de caractères | Code ISO sur 3 caractères (norme ISO 3166-1). <br><br>**Cette propriété est disponible en lecture seulement**. |
| CodeISONum | Chaîne de caractères | Code ISO numérique (norme ISO 3166-1). <br><br>**Cette propriété est disponible en lecture seulement**. |
| Continent | Variable de type [Continent](../WDLang1/1000024919.md) | Continent du pays. <br><br>**Cette propriété est disponible en lecture seulement**. |
| Drapeau | Image | Drapeau du pays<br><br>![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Cette propriété n'est pas disponible.<br><br>**Cette propriété est disponible en lecture seulement**. |
| Nation | Entier | <br><br>- Entier correspondant à la constante du pays pour la fonction [Nation](../WDLang1/3054014.md).<br><br>- 0 si le  pays n'a pas de constante correspondante pour la fonction [Nation](../WDLang1/3054014.md). <br><br><br>**Cette propriété est disponible en lecture seulement**. |
| Nom | Chaîne de caractères | Nom du pays (dans la langue du framework). <br><br>**Cette propriété est disponible en lecture seulement**. |
| PréfixeNuméroDeTéléphone | Chaîne de caractères | Préfixe d'un numéro de téléphone du pays. <br><br>**Cette propriété est disponible en lecture seulement**. |

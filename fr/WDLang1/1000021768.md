
## fRepTemp (Fonction)

***En anglais : fTempDir***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le nom du répertoire utilisé pour stocker les fichiers temporaires.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Récupération du nom du répertoire temporaire
ResRepTemp = fRepTemp()
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = fRepTemp()`
---

**`<Résultat> : (Chaîne de caractères)`**



- Chemin du répertoire utilisé pour stocker les fichiers temporaires (260 caractères maximum). Ce chemin se termine par le caractère "\\". 
	

- Chaîne vide ("") en cas d'erreur. Pour connaître le libellé de l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.   


**Remarque** : Le résultat de la fonction **fRepTemp** est toujours une chaîne de caractères au format Ansi. Il est cependant possible de stocker le résultat dans une chaîne de caractères Unicode.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Répertoire temporaire
<a name="repertoire_temporaire_ELTPARAGRAPHE000043"></a>Si le répertoire temporaire géré par le système n'existe pas sur le poste en cours, la fonction **fRepTemp** renvoie : 

- avec Windows 98 ou Me : le répertoire en cours sur le poste. 

- avec Windows 2000, NT ou CE : le nom du répertoire temporaire géré par défaut par le système. Aucun test n'est réalisé sur l'existence du répertoire temporaire.




**Attention** : Sous Windows Vista (ou supérieur) avec le [système de l'UAC](../Editeurs/2025014.md) activé, le répertoire temporaire utilisé est différent si l'utilisateur à les privilèges administrateur ou non.


### Equivalence
<a name="equivalence_ELTPARAGRAPHE000059"></a>La fonction **fRepTemp** est équivalente à la fonction [fRépertoireTemp](../WDLang1/3036044.md).

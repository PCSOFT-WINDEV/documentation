


## MenuDémarque (Fonction)

***En anglais : MenuUnMark***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Enlève la marque ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Marque.gif) placée devant l'option de menu (menu contextuel ou déroulant).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Démarquage de l'option "OPT_BarreIcone"
MenuDémarque(OPT_BarreIcone)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`MenuDémarque([<Fenêtre>, ] <Option de menu>)`
---

**`<Fenêtre> : (Nom de fenêtre optionnel)`**

Nom de la fenêtre contenant l'option de menu à démarquer. 
Si ce paramètre n'est pas spécifié, la fonction **MenuDémarque** manipulera la fenêtre en cours.

**`<Option de menu> : (Nom d'option)`**

Nom de l'option de menu à démarquer. Ce nom est défini dans la fenêtre de description de l'option de menu. 
Remarque : Ce nom doit être spécifié sans guillemets.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Cette fonction est équivalente à la propriété [Cochée](../Proprietes/2510019.md).

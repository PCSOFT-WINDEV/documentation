
## TexteVersRTF (Fonction)

***En anglais : TextToRTF***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Convertit une chaîne texte en chaîne au format RTF. 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
sTexteEnRTF est une chaîne
sTexteEnRTF = TexteVersRTF("WINDEV est un outil formidable" + RC + ...
		"Le développement devient un jeu d'enfant")

// sTexteEnRTF contient la chaîne avec les balises RTF : 
// {\rtf1\ansi\ansicpg1252\deff0{\fonttbl
// {\f0\froman\fcharset0 Times New Roman;} }
// {\*\generator WinDev 19.0.20.0;}
// {WINDEV est un outil formidable\line 
// Le développement devient un jeu d'enfant} }
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = TexteVersRTF(<Texte>)`
---

**`<Résultat> : (Chaîne de caractères)`**

Texte au format RTF. 

**`<Texte> : (Chaîne de caractères)`**

Chaîne de caractères au format texte à convertir en RTF. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Cas particulier
<a name="cas_particulier_ELTPARAGRAPHE000054"></a>Dans un projet Unicode, une chaîne Unicode contenant des caractères utilisant plusieurs alphabets ne sera pas correctement transformée en RTF. Seuls les caractères de l'alphabet courant seront transformés en RTF. Les autres apparaîtront avec des '?'.

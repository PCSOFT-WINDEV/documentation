## Syntaxe

`<Résultat> = fDisqueInfo(<Chemin du disque> , <Information>)`
---

**`<Résultat> : (Tous types)`**



- Information demandée.

- 0 en cas d'erreur. Pour connaître le libellé de l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.




**`<Chemin du disque> : (Chaîne de caractères)`**

Chemin du disque à manipuler de la forme :

- &lt;lettre du lecteur&gt;**:**. Par exemple : "**A:**". La lettre du lecteur est en majuscule.

- chemin UNC. Par exemple : "\\\\&lt;NomServeur&gt;\\&lt;NomPartage&gt;".




**`<Information> : (Constante)`**

Indique le type d'information à rechercher :


|   |   |
| --- | --- |
| *fdEspaceLibre* | Espace disque sur le disque (en octets) |
| *fdEspaceTotal* | Taille totale du disque (en octets) |
| *fdNbCluster* | Nombre de clusters du disque<br><br> |
| *fdNbSecteurParCluster* | Nombre de secteurs par cluster<br><br> |
| *fdNomVolume* | Nom du volume<br><br> |
| *fdNuméroDeSérie* | Numéro de série du disque, mis en place lorsque le disque est formaté (et non le numéro de série donné par le constructeur). <br><br> |
| *fdTailleNomFichier* | Taille maximale d'un nom de fichier sur le disque (en caractères)<br><br> |
| *fdTailleSecteur* | Taille d'un secteur (en octets)<br><br> |
| *fdTypeDisque* | Type de disque (voir tableau ci-dessous)<br><br> |

### Type de disque
<a name="type_disque_ELTPARAGRAPHE000112"></a>Les types de disques sont des constantes WLangage. Les différentes valeurs possibles sont :

| Constante | Valeur | Type de disque |
| --- | --- | --- |
| *fdInconnu* | 32 | Type inconnu |
| *fdAmovible* | 1 | Disque amovible (clé USB, zip, etc.) |
| *fdFixe* | 2 | Disque fixe (disque dur) |
| *fdInvalide* | 0 | Le chemin fourni ne correspond pas à un disque |
| *fdReseau* | 4 | Disque réseau |
| *fdCDRom* | 8 | CD-ROM (et graveurs, DVD, etc.) |
| *fdRAM* | 16 | Disque virtuel |

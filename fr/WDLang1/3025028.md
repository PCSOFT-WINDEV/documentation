
## OngletDuChamp (Fonction)

***En anglais : ControlTab***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le nom du champ Onglet qui contient le champ spécifié.








<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// SAI_Nom1 est sur le champ ONG_OngletInfo
Info(OngletDuChamp(SAI_Nom1))       // Renvoie "ONG_OngletInfo"
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = OngletDuChamp(<Champ>)`
---

**`<Résultat> : (Chaîne de caractères)`**



- Nom de l'onglet associé au champ spécifié. 

- Chaîne vide ("") si le champ n'est associé à aucun onglet.




**`<Champ> : (Nom de champ)`**

Nom du champ à manipuler. Si l'objet est inconnu, une erreur WLangage est affichée.

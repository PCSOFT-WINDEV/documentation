


## &lt;Tableau&gt;.TousVrai (Fonction)

***En anglais : &lt;Array&gt;.AllTrue***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie <u><u><u>Vrai</u></u></u> si tous les éléments d'un tableau sont <u><u><u>Vrai</u></u></u>.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
TabProduit est un tableau de Produit
// Renvoie Vrai si aucun produit n'est en rupture de stock
SI TabProduit.TousVrai(UnProduit => UnProduit.Stock > 0) ALORS
	// Tout va bien
SINON
	// Alerte
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Source>.TousVrai([<Opération>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si tous les éléments du tableau spécifié valent <u><u><u><u>Vrai</u></u></u></u>, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Source> : (Variable de type Tableau)`**

Nom de la variable de type Tableau à vérifier.

**`<Opération> : (Procédure WLangage)`**

Procédure WLangage à exécuter : 

- Si ce paramètre n'est pas spécifié, le paramètre &lt;Source&gt; doit correspondre à un tableau de booléens. 

- Si ce paramètre est spécifié, la procédure WLangage est appelée pour chaque élément du tableau et doit renvoyer un booléen. 


Ce paramètre peut correspondre à une procédure locale ou globale WLangage, une procédure interne ou une procédure lambda.

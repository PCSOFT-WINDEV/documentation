
## PiloteDnD (Fonction)

***En anglais : EmulateDnD***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Pilote un "Glisser-Déposer" durant les tests automatiques. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
PiloteSouris("FEN_Diag.EDIAG_Place.__WinDevFAA_BarreEditDiagramme_1.ZR_FormesPrédéfinies", ...
	psEnfonceBoutonGauche, CoordonnéeEditeurVersEcran(107), CoordonnéeEditeurVersEcran(61))
PiloteDnD("FEN_Diag.EDIAG_Place.__WinDevFAA_BarreEditDiagramme_1" + ...
		".ZR_FormesPrédéfinies.IMG_VignetteForme",
		29, 27, dndDébutGlisser, 3)
PiloteDnD(FEN_Diag.EDIAG_Place, 1498, 840, dndEntréeChamp, 1)
PiloteDnD(FEN_Diag.EDIAG_Place, 1910, 957, dndLâcher, 0)
PiloteDnD("FEN_Diag.EDIAG_Place.__WinDevFAA_BarreEditDiagramme_1" + ...
		".ZR_FormesPrédéfinies.IMG_VignetteForme",
		29, 27, dndFinGlisser, 3)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`PiloteDnD(<Champ> , <Position X> , <Position Y> , <Action à effectuer> , <Etat>)`
---

**`<Champ> : (Nom de champ)`**

Champ cible ou source de l'action "Glisser-Déposer". 

**`<Position X> : (Entier)`**

Position correspondant à l’abscisse à laquelle l'action est effectuée. 

**`<Position Y> : (Entier)`**

Position correspondant à l'ordonnée à laquelle l'action est effectuée. 

**`<Action à effectuer> : (Constante de type Entier)`**

Action à effectuer : 


|   |   |
| --- | --- |
| *dndDébutGlisser* | Début d'un glisser/déplacer dans un champ source. |
| *dndEntréeChamp* | Entrée dans un champ cible. |
| *dndFinGlisser* | Fin d'un glisser/déplacer. |
| *dndLâcher* | Lâcher du bouton gauche de la souris sur un champ cible. |
| *dndSortieChamp* | Sortie d'un champ cible. |
| *dndSurvol* | Survol d'un champ cible. |



**`<Etat> : (Entier)`**

Permet de savoir si le clavier ou la souris ont été utilisés lors du glisser/déplacer. Les valeurs sont les suivantes : 

- 1 : Bouton gauche de la souris

- 2 : Bouton droit de la souris. 

- 3 : Touche SHIFT du clavier. 

- 4 : Touche CONTROL du clavier. 

- 5 : Bouton du milieu de la souris.

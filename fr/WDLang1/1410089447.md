
## FAALanceAide (Fonction)

***En anglais : AAFOpenHelp***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ouvre le site d'aide des FAA (Fonctionnalités Automatiques de vos Applications) : [https://faa.pcsoft.fr/](https://faa.pcsoft.fr/)
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Lance l'aide sur toutes les FAA du champ Table
FAALanceAide(TABLE_CLIENT)

// Lance le site général des FAA : https://faa.pcsoft.fr/
FAALanceAide() 

// Lance l'aide sur une FAA spécifique
FAALanceAide(faaAfficherMotDepasse)
```

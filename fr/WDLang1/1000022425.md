


## DocRedo (Fonction)

***En anglais : DocRedo***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Rétablit la dernière action annulée sur un document de type Traitement de texte.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Refait la dernière action annulée sur le document
DocRedo(TT_MonDocument)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = DocRedo(<Document>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la dernière action annulée a été rétablie, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 




**`<Document> : (Nom de champ ou variable de type Document)`**

Document à manipuler. Ce document peut correspondre à : 

- une variable de type [Document](../WDLang1/1000022461.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) un champ Traitement de texte.

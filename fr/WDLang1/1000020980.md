
## AppliOuvreFiche (Fonction)

***En anglais : AppOpenForm***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ouvre : 

- la fiche d'une application sur le market Android (Play Store), iOS (App Store) ou Windows (Windows Store) pour que l'utilisateur puisse la noter et/ou la commenter.

## Syntaxe
<a name="SYNTAXE1"></a>

`AppliOuvreFiche([<Application> [, <Type de fiche>]])`
---

**`<Application> : (Chaîne de caractères optionnelle)`**



![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Identifiant de l'application. Pour obtenir cet identifiant :  
**1.** Rendez vous sur le store Windows. 
**2.** Recherchez l'application voulue et affichez sa fiche. Le numéro se trouve alors dans l'URL fournie. 

**`<Type de fiche> : (Constante optionnelle de type Entier)`**

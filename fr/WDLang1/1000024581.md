


## &lt;Variable picCalque&gt;.DessineRectangleDégradé (Fonction)

***En anglais : &lt;picLayer variable&gt;.DrawRectangleGradient***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Dessine un rectangle dans une variable de type [picCalque](../WDLang1/1000024604.md).

![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Les dessins](../WDLang1/3029035.md). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonImageWDPic est un WDPic = "Test.bmp"
MonCalque est un picCalque

// Dessin d'un rectangle dont le fond passe du rouge au bleu
// en passant par le jaune
MonCalque.DessineRectangleDégradé(10, 20, 100, 150, RougeClair, BleuClair, 0, JauneClair, 30)
IMG_MonDessin = MonImageWDPic
```

<a name="SYNTAXE1"></a>

## Syntaxe

### Dessiner un rectangle en spécifiant les points du rectangle et le dégradé

`<Image picCalque>.DessineRectangleDégradé(<X1> , <Y1> , <X2> , <Y2> , <Couleur Début> , <Couleur Fin> [, <Angle> [, <Couleur 3> [, <Distance couleur 3> [, <Couleur 4> [, <Distance couleur 4>]]]]])`
---

**`<Image picCalque> : (Variable de type picCalque)`**

Nom de la variable de type [picCalque](../WDLang1/1000024604.md) à manipuler.

**`<X1> : (Entier)`**

Abscisse en pixels du coin haut gauche du rectangle.

**`<Y1> : (Entier)`**

Ordonnée en pixels du coin haut gauche du rectangle.

**`<X2> : (Entier)`**

Abscisse en pixels du coin bas droit du rectangle.

**`<Y2> : (Entier)`**

Ordonnée en pixels du coin bas droit du rectangle.

**`<Couleur Début> : (Entier ou constante)`**

Couleur de début du dégradé. Cette couleur peut correspondre à :

- une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)),

- une couleur TSL (obtenue avec la fonction [TSL](../WDLang1/3029057.md)),

- une couleur prédéfinie (voir [les couleurs prédéfinies du WLangage](../WDLang5/3010002.md)).




**`<Couleur Fin> : (Entier ou constante)`**

Couleur de fin du dégradé. Cette couleur peut correspondre à :

- une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)),

- une couleur TSL (obtenue avec la fonction [TSL](../WDLang1/3029057.md)),

- une couleur prédéfinie (voir [les couleurs prédéfinies du WLangage](../WDLang5/3010002.md)).




**`<Angle> : (Entier optionnel)`**

Angle à utiliser pour la sélection des couleurs dans le dégradé. La valeur de l'angle est comprise entre 0 (angle horizontal, valeur par défaut) et 360°.
![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Seuls les angles suivants sont utilisables : 0,90,180,270,360.

**`<Couleur 3> : (Entier optionnel ou constante optionnelle)`**

Couleur à utiliser dans le dégradé. Cette couleur peut correspondre à :

- une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)),

- une couleur TSL (obtenue avec la fonction [TSL](../WDLang1/3029057.md)),

- une couleur prédéfinie (voir [les couleurs prédéfinies du WLangage](../WDLang5/3010002.md)).


![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Ce paramètre n'est pas disponible.

**`<Distance couleur 3> : (Entier optionnel)`**

Pourcentage séparant la couleur de début et la couleur 3.
![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Ce paramètre n'est pas disponible.

**`<Couleur 4> : (Entier optionnel ou constante optionnelle)`**

Couleur à utiliser dans le dégradé. Cette couleur peut correspondre à :

- une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)),

- une couleur TSL (obtenue avec la fonction [TSL](../WDLang1/3029057.md)),

- une couleur prédéfinie (voir [les couleurs prédéfinies du WLangage](../WDLang5/3010002.md)).


![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Ce paramètre n'est pas disponible.

**`<Distance couleur 4> : (Entier optionnel)`**

Pourcentage séparant la couleur de début et la couleur 4.
![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Ce paramètre n'est pas disponible.

## Remarques


### Paramètres du dégradé
<a name="parametres_degrade_ELTPARAGRAPHE000813"></a>Les paramètres de couleur sont utilisés de la façon suivante :<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=drectangledegrade.gif)
L'angle permet de définir l'orientation du dégradé.<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=angle_degrade.gif)
Les dégradés réalisés avec 3 ou 4 couleurs sont disponibles uniquement pour un angle de 0 ou de 90 degrés.

**Remarque** : Sur certains systèmes (Windows 98 ou Me par exemple) ou sous TSE 256 couleurs, les dégradés ne sont pas disponibles. Dans ce cas, un rectangle de la couleur &lt;Couleur début&gt; est affiché.

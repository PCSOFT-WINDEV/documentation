
## dCopieImageFenêtre (Fonction)

***En anglais : dCopyWindowImage***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Copie l'image d'une fenêtre WINDEV dans une variable de type [Image](../WDLang1/1000019650.md) ou dans un champ Image.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Copie l'image de la fenêtre en cours dans une variable image
MonImage est une Image = dCopieImageFenetre("")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = dCopieImageFenêtre(<Fenêtre> [, <Gestion ombre>])`
---

**`<Résultat> : (Variable de type Image ou Buffer binaire)`**



- Nom de la variable de type [Image](../WDLang1/1000019650.md) qui contiendra la copie de l'écran.

- Buffer binaire correspondant à l'image. Ce buffer peut être affecté directement à un champ Image. 




**`<Fenêtre> : (Chaîne de caractères)`**

Nom de la fenêtre (WINDEV ou WINDEV Mobile) dont l'image doit être copiée. Si ce paramètre correspond à une chaîne vide (""), la fenêtre en cours est copiée. 

**`<Gestion ombre> : (Constante de type Entier)`**

Mode de gestion de l'ombre en bordure de la fenêtre (notamment sous Windows 10) : 



|   |   |
| --- | --- |
| *dAvecOmbre*<br>(Valeur par défaut) | L'ombre de la fenêtre est conservée lors de la copie de la fenêtre. |
| *dSansOmbre* | L'ombre de la fenêtre n'est pas conservée lors de la copie de la fenêtre. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- La fenêtre à copier doit être affichée à l'écran. 

- Si la fenêtre est détourée, toute la zone correspondant à la fenêtre est copiée : les éléments apparaissant en fond de la fenêtre détourée sont donc également copiés. 

- Attention : Si l'option "Autoriser la copie d'écran de la fenêtre" (onglet "Détail" de la fenêtre de description) n'est pas cochée, l'image de la fenêtre correspondra à une image entièrement noire. Cette option peut être modifiée par programmation grâce à la propriété [CopieEcranAutorisée](../Proprietes/1000025336.md).

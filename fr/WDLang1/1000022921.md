


## &lt;Planning&gt;.PositionRessource (Fonction)

***En anglais : &lt;Scheduler&gt;.PositionResource***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Affiche une ressource donnée dans un champ Planning ou renvoie la position la première ressource visible dans un champ Planning. 

<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
PLN_SansNom1.AjouteRendezVous("Salle ABC", "RDV 1", DateSys() + "14000", DateSys() + "16000")
PLN_SansNom1.PositionRessource("Salle ABC")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître le nom de la première ressource visible

`<Résultat> = <Champ Planning>.PositionRessource()`
---

**`<Résultat> : (Chaîne de caractères)`**



- Nom de la première ressource visible dans le champ Planning (ressource positionnée en haut ou à gauche). 

- Chaîne vide ("") si le planning est vide.




**`<Champ Planning> : (Nom de champ)`**

Nom du champ Planning à manipuler (présent dans une fenêtre).


<a name="SYNTAXE2"></a>

### Afficher un planning à partir de la ressource spécifiée

`<Champ Planning>.PositionRessource(<Ressource>)`
---

**`<Champ Planning> : (Nom de champ)`**

Nom du champ Planning à manipuler (présent dans une fenêtre).

**`<Ressource> : (Chaîne de caractères)`**

Nom de la ressource à afficher en premier.

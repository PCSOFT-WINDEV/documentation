


## &lt;Variable File&gt;.Info (Fonction)

***En anglais : &lt;Queue variable&gt;.Info***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Récupère les caractéristiques d'une file : types des éléments et nombre d'éléments.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// File dont les éléments sont des chaînes
MaFile est une file de chaînes
// Récupère la définition du type des éléments
MaDéfinition est une Définition
MaDéfinition = MaFile.Info(tiDéfinitionElément)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <File>.Info(<Information>)`
---

**`<Résultat> : (Variant)`**

Information demandée.

**`<File> : (Variable de type File)`**

Nom de la variable de type [File](../WDLang1/1514021.md) à manipuler.

**`<Information> : (Constante de type Entier)`**

Information à récupérer :


|   |   |
| --- | --- |
| *tiDéfinitionElément* | &lt;Résultat&gt; est une variable de type [Définition](../WDLang1/1000019540.md) et contient la définition du type des éléments.<br><br>![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Cette constante n'est pas disponible. |
| *tiNombreTotal* | &lt;Résultat&gt; est une variable de type Entier et contient le nombre total d'éléments dans la file. |
| *tiTypeElément* | &lt;Résultat&gt; est une variable de type Entier et contient le type des éléments de la file. Ce résultat est compatible avec la fonction [TypeVar](../WDLang1/3013056.md). |

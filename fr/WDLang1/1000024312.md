


## &lt;Tableau&gt;.Mélange (Fonction)

***En anglais : &lt;Array&gt;.Shuffle***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Mélange les éléments d'un tableau WLangage.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Initialisation du mélange aléatoire
InitHasard()

// Remplissage du tableau
PlayList est un tableau de chaînes
...

// Mélange des éléments du tableau
PlayList.Mélange()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Tableau WLangage>.Shuffle()`
---

**`<Tableau WLangage> : (Variable de type Tableau)`**

Tableau WLangage à manipuler. Ce tableau peut avoir de 1 à N dimensions.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Le mélange des éléments du tableau est basé sur la génération de nombres aléatoires. Il faut utiliser la fonction [InitHasard](../WDLang1/3013030.md) pour réinitialiser le mélange.

- Cette fonction ne peut pas être utilisée sur les tableaux associatifs. 

- Si le tableau possède plusieurs dimensions, le mélange est réalisé à travers les différentes dimensions.




## &lt;Fenêtre&gt;.Iconise (Fonction)

***En anglais : &lt;Window&gt;.Iconize***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Iconise une fenêtre WINDEV.

**Remarque** : La fonction **&lt;Fenêtre&gt;.Iconise** réalise la même action que le bouton de réduction présent dans le coin haut droit des fenêtres.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Iconise la fenêtre "FEN_SaisieClient"
FEN_SaisieClient.Iconise()
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Fenêtre>.Iconise()`
---

**`<Fenêtre> : (Nom de fenêtre)`**

Nom de la fenêtre à icôniser.
Remarque : Si la fenêtre spécifiée correspond à une fenêtre fille, la fenêtre mère est également icônisée.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Pour savoir si une fenêtre est icônisée, utilisez la fonction [FenTaille](../WDLang1/3038020.md).

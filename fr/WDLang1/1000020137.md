


## Fonctions de gestion des champs Ruban
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions de gestion des champs Ruban sont les suivantes :



|   |   |
| --- | --- |
| [RubanAdapteTaille](../WDLang1/1000020761.md) | Adapte la taille des champs présents dans un champ Ruban en fonction de la taille de leur libellé. |
| [RubanFermeRegroupement](../WDLang1/1000020136.md) | Ferme le regroupement actuellement ouvert dans un champ Ruban. |
| [RubanFinModeDéplié](../WDLang1/1000020378.md) | Replie un ruban enroulé et temporairement déplié. |
| [RubanSurligneVolet](../WDLang1/1000020530.md) | Surligne ou non un volet d'un champ Ruban. |

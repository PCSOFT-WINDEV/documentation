
## SysNombreEcran (Fonction)

***En anglais : SysNumberScreen***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le nombre d'écrans actuellement connectés au poste en cours.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Renvoyer le nombre d'écrans connectés
Info(SysNombreEcran())
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = SysNombreEcran()`
---

**`<Résultat> : (Entier)`**

Nombre d'écrans actuellement connectés.

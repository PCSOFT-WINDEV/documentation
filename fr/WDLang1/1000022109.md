


## Fonctions de gestion des FAA  (Fonctionnalités Automatiques de vos Applications)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions de gestion des FAA (Fonctionnalités Automatiques de vos Applications) sont les suivantes : 



|   |   |
| --- | --- |
| [FAAActive](../WDLang1/1000024767.md) | Réactive une Fonctionnalité Automatique de vos Applications (FAA) précédemment désactivée par la fonction [FAADésactive](../WDLang1/1000022018.md). |
| [FAAChangeLibellé](../WDLang1/1000022100.md) | Personnalise un libellé d'une option de menu pour les menus contextuels correspondant à une FAA (Fonctionnalité Automatique de vos Applications). |
| [FAADésactive](../WDLang1/1000022018.md) | Désactive une Fonctionnalité Automatique d'une Application (FAA) sur un champ, sur une fenêtre ou sur l'application en cours. |
| [FAAExécute](../WDLang1/1000022099.md) | Exécute une Fonctionnalité Automatique d'une Application (FAA) sur un champ ou une fenêtre. Par exemple, il est possible de lancer la correction orthographique sur un champ de saisie. |
| [FAAExporteParamètre](../WDLang1/1000024375.md) | Sauve toutes les personnalisations réalisées via les FAA (Fonctionnalités Automatiques de vos Applications) dans un fichier. |
| [FAAImporteParamètre](../WDLang1/1000024377.md) | Charge toutes les personnalisations réalisées via les FAA (Fonctionnalités Automatiques de vos Applications) ayant été précédemment sauvegardées dans un fichier. |
| [FAALanceAide](../WDLang1/1410089447.md) | Ouvre le site d'aide des FAA (Fonctionnalités Automatiques de vos Applications). |
| [FAAMenuContextuelVersChaîne](../WDLang1/1000022467.md) | Renvoie la liste des options du menu contextuel effectivement affiché par le champ. |
| [FAAParamètre](../WDLang1/1000022108.md) | Permet de spécifier le comportement de certaines fonctionnalités automatiques de l'application (FAA). Il est par exemple possible de paramétrer l'option d'export vers Excel utilisée par [le menu automatique des tables](../Editeurs/9500102.md). |
| [FAAPossible](../WDLang1/1000022791.md) | Indique si une FAA (Fonctionnalité Automatique de l'Application) peut être exécutée sur un champ spécifique. |
| [FAASécuritéExport](../WDLang1/1000022481.md) | Sécurise l'export de données confidentielles dans les FAA (Fonctionnalités Automatiques de vos Applications) qui permettent l'export, l'impression ou l'enregistrement de données. |




Remarques : Ces fonctions sont identiques aux fonctions suivantes, utilisables avant la version 21 : 



|   |   |
| --- | --- |
| [DésactiveFAA](../WDLang1/3013061.md) | Désactive une Fonctionnalité Automatique de vos Applications (FAA) sur un champ, sur une fenêtre ou sur l'application en cours. |
| [ExécuteFAA](../WDLang1/3013060.md) | Exécute une Fonctionnalité Automatique de vos Applications (FAA) sur un champ ou une fenêtre. |
| [ParamètreFAA](../WDLang1/3014032.md) | Permet de spécifier le comportement de certaines Fonctionnalités Automatiques de vos Applications (FAA). |


## MarkdownVersDOCX (Fonction)

***En anglais : MarkdownToDOCX***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Convertit une chaîne de caractères au format Markdown en un document au format DOCX.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonDocument est un Document
MaChaîneInitiale est une chaîne = "*Texte en italique*"

MonDocument = MarkdownVersDOCX(MaChaîneInitiale)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = MarkdownVersDOCX(<Chaîne Markdown>)`
---

**`<Résultat> : (Variable de type Document)`**

Variable de type [Document](../WDLang1/1000022461.md) contenant le résultat de la conversion. 

**`<Chaîne Markdown> : (Chaîne de caractères)`**

Chaîne de caractères au format Markdown à manipuler. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Le format de page est celui par défaut d'une variable de type [Document](../WDLang1/1000022461.md) (format A4).

- Si le texte Markdown contient : 

	- du texte sans balise Markdown, le style de ce texte est celui par défaut d'une variable de type [Document](../WDLang1/1000022461.md) (style "Normal"). 

	- du texte avec balises markdown, le style de ce texte sera celui du texte par défaut surchargé par les attributs du texte Markdown correspondant (gras, souligné, etc.).




- Le texte issu de texte Markdown dans une balise de style (titre de niveau 1, titre de niveau 2, etc.) aura un style équivalent ("Titre 1", "Titre 2") dans le document résultat.

- Il est possible de modifier toutes les caractéristiques du document résultat (format de page, style, etc.) en utilisant les fonctions et propriétés liées aux variables de type [Document](../WDLang1/1000022461.md).

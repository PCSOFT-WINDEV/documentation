


## Fonctions Gantt
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions permettant de manipuler une colonne d'un champ Table (ou Table hiérarchique) de type Diagramme de Gantt sont les suivantes : 



|   |   |
| --- | --- |
| [GanttAjouteCongé](../WDLang1/1000021116.md) | Déclare une nouvelle période de congés pour une ligne dans une colonne d'un champ Table (ou Table hiérarchique) de type Diagramme de Gantt. |
| [GanttAjouteLien](../WDLang1/1000021075.md) | Crée un lien de type "Fin à Début" entre deux tâches affichées dans une colonne d'un champ Table (ou Table hiérarchique) de type Diagramme de Gantt. |
| [GanttAjouteTâche](../WDLang1/1000020904.md) | Ajoute une tâche dans une colonne d'un champ Table (ou Table hiérarchique) de type Diagramme de Gantt. |
| [GanttChangeMode](../WDLang1/1000021030.md) | Modifie la période (c'est-à-dire le nombre de jours) affichée dans une colonne d'un champ Table (ou Table hiérarchique) de type Diagramme de Gantt. |
| [GanttHeureAffichée](../WDLang1/1000021081.md) | Définit la plage horaire affichée dans une colonne d'un champ Table (ou Table hiérarchique) de type Diagramme de Gantt. |
| [GanttHeureOuvrable](../WDLang1/1000021117.md) | Surcharge les heures ouvrables d'une ligne dans une colonne d'un champ Table (ou Table hiérarchique) de type Diagramme de Gantt. |
| [GanttJourSemaineOuvré](../WDLang1/1000021035.md) | Indique si un jour de la semaine est ouvré dans une colonne d'un champ Table (ou Table hiérarchique) de type Diagramme de Gantt. |
| [GanttListeLien](../WDLang1/1000021079.md) | Liste les liens présents dans une colonne d'un champ Table (ou Table hiérarchique) de type Diagramme de Gantt. |
| [GanttListeTâche](../WDLang1/1000021073.md) | Renvoie : <br><br>- la liste des tâches comprises entre deux dates données dans une colonne de type Diagramme de Gantt, <br><br>- toutes les tâches d'une colonne de type Diagramme de Gantt,<br><br>- une tâche spécifique (sélectionnée ou survolée). <br><br><br> |
| [GanttOrganiseTâche](../WDLang1/1000020890.md) | Réorganise les tâches d'un diagramme de Gantt selon les contraintes (dépendances et ressources). |
| [GanttPositionDateHeure](../WDLang1/1000021036.md) | Paramètre le premier jour et la première heure affichés dans une colonne (d'un champ Table ou Table hiérarchique) de type Diagramme de Gantt. |
| [GanttRécupèreTâche](../WDLang1/1000021026.md) | Récupère une tâche à partir de son identifiant dans une colonne (d'un champ Table ou Table hiérarchique) de type Diagramme de Gantt. |
| [GanttSupprimeLien](../WDLang1/1000021080.md) | Supprime des liens dans une colonne d'un champ Table (ou Table hiérarchique) de type Diagramme de Gantt. |
| [GanttSupprimeTâche](../WDLang1/1000020923.md) | Supprime une tâche dans une colonne d'un champ Table (ou Table hiérarchique) de type Diagramme de Gantt. |
| [GanttSupprimeTout](../WDLang1/1000020924.md) | Supprime toutes les tâches dans une colonne d'un champ Table (ou Table hiérarchique) de type Diagramme de Gantt. |



Deux types de variables sont disponibles pour manipuler les éléments des diagrammes de Gantt : 



|   |   |
| --- | --- |
| [LienGantt](../WDLang1/1000021076.md) | Le type **LienGantt** permet de définir toutes les caractéristiques avancées d'une liaison entre deux tâches d'un diagramme de Gantt. |
| [TâcheGantt](../WDLang1/1000020881.md) | Le type **TâcheGantt** permet de définir toutes les caractéristiques avancées d'une tâche pour un Diagramme de Gantt. |

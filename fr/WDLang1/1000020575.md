
## fSurveilleStop (Fonction)

***En anglais : fTrackStop***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Arrête toutes les surveillances en cours sur un fichier ou un répertoire. Ces surveillances ont pu être lancées par la fonction [fSurveilleFichier](../WDLang1/1000020522.md) ou par la fonction [fSurveilleRépertoire](../WDLang1/1000020523.md). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Arrêt de toutes les surveillances en cours sur le répertoire "C:\Test\Dossiers importants"
SI PAS fSurveilleStop("C:\Test\Dossiers importants") ALORS
	Erreur("Aucune surveillance n'était active")
SINON
	Info("Surveillance arrêtée")
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = fSurveilleStop(<Nom de l'élément>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la surveillance précédemment activée a été arrêtée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire (surveillance inexistante).




**`<Nom de l'élément> : (Chaîne de caractères)`**

Nom complet du fichier ou du répertoire à manipuler.

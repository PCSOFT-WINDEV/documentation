
## ChampSurvol (Fonction)

***En anglais : ControlOver***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Identifie le champ survolé par le curseur de la souris.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Afficher le nom des champs survolés dans une trace

// -- Code d'initialisation de la fenêtre
// Prise en compte de la fenêtre et des champs de la fenêtre
Evénement(Survol, "*.*", 512)
```

```wl
// -- Procédure "Survol"
PROCEDURE Survol()
Trace(ChampSurvol())
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = ChampSurvol([<Champs invisibles> [, <Fenêtre survolée>]])`
---

**`<Résultat> : (Chaîne de caractères)`**



- Nom du champ, de la colonne ou de la fenêtre survolé par le curseur de la souris. 

- Chaîne vide ("") si aucun champ n'est survolé par le curseur de la souris.




**`<Champs invisibles> : (Booléen optionnel)`**



- <u><u><u><u>Vrai</u></u></u></u> si les champs invisibles doivent être pris en compte.

- <u><u><u><u>Faux</u></u></u></u> (par défaut) si les champs invisibles ne doivent pas être pris en compte. 




**`<Fenêtre survolée> : (Booléen optionnel)`**



- <u><u><u><u>Vrai</u></u></u></u> si le nom de la fenêtre doit être renvoyé lorsque aucun champ n'est survolé, 

- <u><u><u><u>Faux</u></u></u></u> (par défaut) si une chaîne vide doit être renvoyée lorsque aucun champ n'est survolé.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Champ survolé par le curseur de la souris
<a name="champ_survole_par_curseur_souris_ELTPARAGRAPHE000053"></a>

- Si plusieurs champs superposés sont survolés, le nom du champ qui a l'altitude la plus haute est renvoyé (c'est-à-dire le nom du champ au premier plan).

- Si une colonne de table est survolée, le nom de la colonne est renvoyé.

- Les champs qui ne sont pas présents dans le plan actif sont considérés comme invisibles.

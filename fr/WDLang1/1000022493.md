
## docElémentTexte (Type de variable)

***En anglais : docElementText***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **docElémentTexte** permet de manipuler les caractéristiques d'un élément de type Texte présent dans une variable de type [docElément](../WDLang1/1000022484.md).

Les caractéristiques de ce type de variable peuvent être manipulées à l'aide de différentes propriétés WLangage. 

**ATTENTION : Ce type ne doit pas être utilisé directement. Il doit être utilisé uniquement via le type  [docElément](../WDLang1/1000022484.md)**.


<a name="NOTE0"></a>

## Remarques
<a name="NOTE0_1"></a>


### Propriétés spécifiques à la description des variables de type docElémentTexte
<a name="proprietes_specifiques_description_des_variables_type_docelementtexte_ELTPARAGRAPHE000030"></a>Les propriétés suivantes peuvent être utilisées pour manipuler une variable de type **docElémentTexte** :

   | --- |
| --- | --- | --- |
| Nom de la propriété | Type manipulé | Effet |
| MiseEnForme | Variable de type [docMiseEnForme](../WDLang1/1000022817.md) | Informations de mise en forme de cet élément. |
| Valeur | Chaîne de caractères | Texte de l'élément. |




## Contient (Fonction)

***En anglais : Contains***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de savoir si une sous-chaîne est contenue dans une chaîne.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
sMaChaîne est une chaîne = "Antananarivo"
SI Contient(sMaChaîne, "NA") = Faux ALORS
	Info("Cette chaîne ne contient pas la syllabe NA")
FIN
```


<a name="Exemple2"></a>

```wl
tabChaine est un tableau de chaînes = ["WINDEV","WEBDEV"]
sTest est une chaîne = "WEBDEV 28"

SI Contient(sTest, tabChaine) ALORS
	Info("Cette chaîne contient WINDEV ou WEBDEV")
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Savoir si une chaîne contient une sous-chaîne

`<Résultat> = Contient(<Chaîne à analyser> , <Sous-chaîne> [, <Options>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la chaîne contient la sous-chaîne. 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 




**`<Chaîne à analyser> : (Chaîne de caractères Ansi, Unicode ou Binaire)`**

Chaîne de caractères à manipuler. 

**`<Sous-chaîne> : (Chaîne de caractères Ansi, Unicode ou Binaire)`**

Sous-chaîne à rechercher. 

**`<Options> : (Constante de type Entier)`**

Mode de recherche de la sous-chaîne : 



|   |   |
| --- | --- |
| *MotComplet* | Recherche un mot complet (entouré par des caractères de ponctuation ou des espaces). |
| *SansCasse* | Recherche sans tenir compte de la casse (différences majuscules / minuscules) ou des accents. |


Par défaut, la recherche de la sous-chaîne est sensible à la casse et cherche toutes les chaînes spécifiées (même si elles font partie d'un mot). 


<a name="SYNTAXE2"></a>

### Savoir si une chaîne contient une des chaînes contenue dans un tableau

`<Résultat> = Contient(<Chaîne à analyser> , <Tableau de chaînes à rechercher> [, <Options>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la chaîne contient une des chaînes présente dans le tableau. 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 




**`<Chaîne à analyser> : (Chaîne de caractères Ansi, Unicode ou Binaire)`**

Chaîne de caractères à manipuler. 

**`<Tableau de chaînes à rechercher> : (Tableau)`**

Nom du tableau contenant les chaînes de caractères à rechercher. 

**`<Options> : (Constante de type Entier)`**

Mode de recherche de la sous-chaîne : 



|   |   |
| --- | --- |
| *MotComplet* | Recherche un mot complet (entouré par des caractères de ponctuation ou des espaces). |
| *SansCasse* | Recherche sans tenir compte de la casse (différences majuscules / minuscules) ou des accents. |


Par défaut, la recherche de la sous-chaîne est sensible à la casse et cherche toutes les chaînes spécifiées (même si elles font partie d'un mot).

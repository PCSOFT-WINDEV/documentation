


## EnModeProcédureStockée (Fonction)

***En anglais : InStoredProcedureMode***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Identifie si le code est exécuté sur un serveur HFSQL.



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Code exécuté sur le serveur HFSQL ?
SI EnModeProcédureStockée() = Vrai ALORS
	// Mode procédure stockée
	// Traitement à effectuer
	...
SINON
	// Mode différent
	// Traitement à effectuer
	...
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = EnModeProcédureStockée()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'exécution du code en cours se fait sur un serveur HFSQL, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Disponibilité de la fonction
<a name="disponibilite_fonction_ELTPARAGRAPHE000230"></a>La fonction **EnModeProcédureStockée** est disponible dans tous les produits pour permettre l'écriture d'un code multi-produits. Des fonctions ou procédures utilisant la fonction **EnModeProcédureStockée** peuvent être partagées entre plusieurs produits, sans que des erreurs de compilation soient générées.

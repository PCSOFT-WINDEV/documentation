


## &lt;Table&gt;.Occurrence (Fonction)

***En anglais : &lt;Table&gt;.Count***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie : 

- soit le nombre de colonnes présentes dans un champ Table ou Table hiérarchique.

- soit le nombre de lignes présentes dans un champ Table ou Table hiérarchique. 





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Nombre de lignes présentes dans le champ "TABLE_Produit"
ResNombreLigne = TABLE_Produit.Occurrence()
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Champ Table>.Occurrence([<Ligne/Colonne>])`
---

**`<Résultat> : (Entier)`**

Nombre de lignes ou de colonnes présentes dans le champ spécifié.

**`<Champ Table> : (Nom de champ)`**

Nom du champ à manipuler. Ce champ peut correspondre à : 

- un champ Table.

- un champ Table hiérarchique.




**`<Ligne/Colonne> : (Constante de type Entier (optionnel))`**

Permet d'indiquer le type du résultat renvoyé : 

- les constantes *toAffichée*, *toTotal* et *toVisible* permettent d'obtenir un nombre de lignes. 

- la constante *toColonne* permet d'obtenir le nombre de colonnes du champ. 




|   | Champ Table/Table hiérarchique fichier | Champ Table/Table hiérarchique par programmation |
| --- | --- | --- |
| *toAffichée*<br> | Nombre de lignes non vides visibles dans le champ.<br> | Nombre de lignes non vides dans la partie visible du champ.<br><br>**Remarque** : Les lignes masquées (grâce à la propriété [Visible](../Proprietes/2510138.md)) sont tout de même comptabilisées. |
| *toColonne* | Nombre de colonnes présentes dans le champ. | Nombre de colonnes présentes dans le champ. |
| *toTotal*<br>(Valeur par défaut)<br> | <br><br>- Nombre total d'enregistrements (option "Ascenseur proportionnel" cochée) : <br><br>	- du fichier lié (s'il n'y a pas de filtre).<br><br>	- filtrés du fichier lié (s'il y a un filtre).<br><br>	- de la requête (si le champ est lié à une requête).<br><br><br><br><br>- Nombre d'enregistrements visibles si le champ possède un ascenseur à rebond (option "Ascenseur proportionnel" non cochée)<br><br><br>**Attention** : Le code de fin d'initialisation n'est exécuté que lorsque le total est calculé.<br> | Nombre de lignes en mémoire (lignes ajoutées par l'utilisateur ou lignes ajoutées par les fonctions [&lt;Table&gt;.Ajoute](../WDLang1/1000023600.md), [&lt;Table&gt;.AjouteLigne](../WDLang1/1000023404.md), [&lt;Table&gt;.Insère](../WDLang1/1000024260.md) et [&lt;Table&gt;.InsèreLigne](../WDLang1/1000023407.md)).<br><br>**Remarques** : <br><br>- Les lignes masquées (grâce à la propriété [Visible](../Proprietes/2510138.md)) sont tout de même comptabilisées.<br><br>- Les lignes filtrées par la fonction [TableActiveFiltre](../WDLang1/1000017149.md) ou dans l'interface par l'utilisateur ne sont pas comptabilisées.<br><br><br> |
| *toVisible*<br> | Nombre de lignes visibles (y compris les lignes vides ou affichées partiellement). Correspond au nombre maximum de lignes que peut afficher le champ.<br> | Nombre de lignes visibles (y compris les lignes vides ou affichées partiellement). Correspond au nombre maximum de lignes que peut afficher le champ.<br><br>**Remarque** : Les lignes masquées (grâce à la propriété [Visible](../Proprietes/2510138.md)) sont tout de même comptabilisées. |

## Remarques


### Conditions d'utilisation
<a name="conditions_utilisation_ELTPARAGRAPHE000338"></a>La fonction **&lt;Table&gt;.Occurrence** peut être utilisée indifféremment sur :

- un champ Table ou Table hiérarchique fichier.

- un champ Table ou Table hiérarchique par programmation.

- un champ monosélection ou multisélection.

### Équivalence fonction &lt;Table&gt;.Occurrence et propriété Taille
<a name="equivalence_fonction_docparampagetitleshort_propriete_taille_ELTPARAGRAPHE000378"></a>La fonction **&lt;Table&gt;.Occurrence** et la propriété [Taille](../Proprietes/2510125.md) sont équivalentes :

```txt
ResNbColonne = <Champ Table>.Occurrence(toColonne)

ResNbColonne = <Champ Table>.Taille
```

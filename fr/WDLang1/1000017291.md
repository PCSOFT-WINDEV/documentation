


## CalendrierOuvre (Fonction)

***En anglais : CalendarOpen***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Dans un champ de saisie de type Date, ouvre une fenêtre popup affichant un calendrier. Le champ de saisie prend alors le focus.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affiche le calendrier associé au champ de saisie
CalendrierOuvre(SAI_Date)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`CalendrierOuvre(<Champ de saisie>)`
---

**`<Champ de saisie> : (Nom de champ)`**

Nom du champ de saisie de type Date pour lequel le calendrier doit être affiché dans une fenêtre popup.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- La fonction **CalendrierOuvre** peut être utilisée sur des champs de saisie de type Date n'ayant pas de bouton calendrier ou pour lesquels l'option de menu contextuel "Calendrier" a été désactivée.

- La fonction **CalendrierOuvre** peut être utilisée sur des colonnes des champs Table de type Date : il est nécessaire que le champ Table ainsi que la colonne soient en saisie. Dans le cas contraire, la fonction **CalendrierOuvre** n'a aucun effet. 

- La fonction **CalendrierOuvre** peut être utilisée uniquement sur les champs de saisie de type Date. Elle ne peut pas être utilisée sur les champs de saisie de type Date affichant également une heure.

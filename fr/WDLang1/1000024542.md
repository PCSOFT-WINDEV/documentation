


## &lt;Variable Document&gt;.VersImage (Fonction)

***En anglais : &lt;Document variable&gt;.ToImage***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Exporte une page contenue dans une variable de type [Document](../WDLang1/1000022461.md) sous forme d'une image.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonDoc est un Document <- TT_MonDocument
d est une Image = Mondoc.VersImage(1)
dSauveImageJPEG(d, "test")
LanceAppliAssociée("test.jpg")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Document>.VersImage(<Page>)`
---

**`<Résultat> : (Variable de type Image)`**

Nom de la variable de type [Image](../WDLang1/1000019650.md) correspondant à la page du document Traitement de texte.

**`<Document> : (Variable de type Document)`**

Nom de la variable de type [Document](../WDLang1/1000022461.md) à manipuler.

**`<Page> : (Entier)`**

Numéro de la page à exporter sous forme d'une image.

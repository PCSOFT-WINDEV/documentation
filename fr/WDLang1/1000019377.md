


## TableCelluleSelectOccurrence (Fonction)

***En anglais : TableCellSelectOccurrence***




	<B><B>Avertissement</B></B>

	A partir de la version <B><B>16 (56F)</B></B>,  cette fonction est conservée par compatibilité. Cette fonction est remplacée par la fonction [TableSelectOccurrence](../WDLang1/3074030.md).

<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le nombre de cellules sélectionnées dans un champ Table. 

**Remarque** : Le champ Table doit autoriser le mode de sélection à la cellule. Cette option est paramétrable dans l'onglet "IHM" de la description du champ Table. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
NbCelluleSel est un entier
Total est un réel
NumLigne est un entier
NumCol est un entier

NbCelluleSel = TableCelluleSelectOccurrence(TABLE_TABLE1)
POUR i = 1 A NbCelluleSel
	NumLigne = TableCelluleSelect(TABLE_TABLE1, i, tcsLigne)
	NumCol = TableCelluleSelect(TABLE_TABLE1, i, tcsColonne)
	Total = Total + TABLE_TABLE1[NumLigne, NumCol] 
	
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = TableCelluleSelectOccurrence(<Champ Table>)`
---

**`<Résultat> : (Entier)`**



- Nombre de cellules sélectionnées, 

- 0 si le mode de sélection à la cellule n'est pas sélectionné pour le champ Table (onglet "IHM" de la fenêtre de description du champ),

- 1 si le champ Table autorise uniquement la sélection d'une cellule. 




**`<Champ Table> : (Nom de champ)`**

Nom du champ Table à manipuler. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Il est ensuite possible de récupérer la liste de ces cellules avec la fonction [TableCelluleSelect](../WDLang1/1000019378.md).

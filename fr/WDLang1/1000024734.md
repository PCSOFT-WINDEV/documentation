
## Transforme (Fonction)

***En anglais : Map***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie un tableau WLangage contenant les résultats d'une procédure exécutée pour chaque élément d'un tableau source.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
tabValeur est un tableau d'entiers = [1,2,3,4,5]
tabTriple est un tableau d'entiers

// Construit un tableau avec les valeurs triples
tabTriple = Transforme(tabValeur, UneValeur => UneValeur * 3)
// tabTriple contient [3, 6, 9, 12, 15]
```



```wl
// -> mettre toutes les chaînes en minuscules dans le tableau
// (Syntaxe préfixée)
tab1 est un tableau de chaînes = ["A", "b", "C"]
tab1 = tab1.Transforme( X => Minuscule(X))
// tab1 = ["a", "b", "c"]
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = Transforme(<Tableau source> , <Opération à effectuer>)`
---

**`<Résultat> : (Tableau WLangage)`**

Tableau contenant le résultat de la procédure exécutée pour chaque élément du &lt;Tableau source&gt;. 

**`<Tableau source> : (Tableau WLangage)`**

Nom de la variable de type Tableau à manipuler. Ce tableau doit être un tableau à une dimension.

**`<Opération à effectuer> : (Procédure WLangage)`**

Nom de la procédure WLangage à exécuter pour chaque élément du &lt;Tableau source&gt;. Cette procédure peut être : 

- une procédure globale ou locale, 

- une procédure interne. 


Il est également possible d'utiliser directement une procédure lambda. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Fonctionnement
<a name="fonctionnement_ELTPARAGRAPHE000072"></a>La procédure &lt;Opération à effectuer&gt; est appelée pour chaque élément du tableau source en passant cet élément en paramètre.

Les résultats de tous les appels sont stockés dans le tableau &lt;Résultat&gt;.



**Enchaînement de fonctions**
Il est possible d'enchaîner les différentes fonctions de manipulation d'un tableau. 
Les fonctions suivantes peuvent être utilisées dans un enchaînement : 

- [Filtre](../WDLang1/1000024845.md)

- [Transforme](../WDLang1/1000024734.md)

- [Union](../WDLang1/1000025408.md)

- [Intersection](../WDLang1/1000025407.md)

- [Distinct](../WDLang1/1000025406.md). 


Cet enchaînement pourra être utilisé comme source d'une instruction POUR TOUT ou se terminer par une des fonctions suivantes : 

- [Applique](../WDLang1/1000024856.md)

- [Agrège](../WDLang1/1000024744.md)

- [Min](../WDLang1/3050004.md)

- [Max](../WDLang1/3050005.md)

- [Somme](../WDLang1/1000019892.md)

- [Moyenne](../WDLang1/1000019891.md)

- [VérifieTousFaux](../WDLang1/1000024858.md), [VérifieTousVrai](../WDLang1/1000024857.md), [VérifieUnFaux](../WDLang1/1000024860.md), [VérifieUnVrai](../WDLang1/1000024859.md). 




Exemple : 

```wl
gnMoyenne = gtabUser.Filtre(cbFiltre).Transforme(cbTransforme).Moyenne()
```

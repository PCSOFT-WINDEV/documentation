


## &lt;Variable Diagramme&gt;.VersSVG (Fonction)

***En anglais : &lt;Diagram variable&gt;.ToSVG***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Exporte un diagramme vers une image vectorielle au format SVG.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonDiagramme est un Diagramme <- EDIAG_MonDiagramme
soit bufSVG = MonDiagramme.VersSVG()
SI ErreurDétectée ALORS
	Erreur()
	RETOUR
FIN
fSauveBuffer("exportDuDiagramme.svg", bufSVG)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Diagramme>.VersSVG()`
---

**`<Résultat> : (Buffer)`**

Image vectorielle au format SVG (chaîne encodée en UTF-8).

**`<Diagramme> : (Variable de type Diagramme)`**

Variable de type [Diagramme](../WDLang1/1410088055.md) à manipuler



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
La variable de type Buffer renvoyée peur être sauvegardée dans un fichier (fonction [fSauveBuffer](../WDLang1/1000019412.md)) ou directement affichée dans un autre champ,

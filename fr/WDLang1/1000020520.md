


## ChampTypeInfo (Fonction)

***En anglais : ControlTypeInfo***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie l'icône correspondant au type de champ spécifié.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) 
```wl
nType est un entier = SAI_Nom.Type
ImgChamp est une Image = ChampTypeInfo(nType)
IMG_Champ = ImgChamp
```
<a name="Exemple2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) 
```wl
ImgChamp est une chaîne = ChampTypeInfo(typCalendrier)
IMG_Champ = ImgChamp
```

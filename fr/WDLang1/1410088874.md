
## MarkdownVersHTML (Fonction)

***En anglais : MarkdownToHTML***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Convertit une chaîne de caractères au format Markdown en un buffer au format HTML (encodé en UTF8). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonBuffer est un buffer
MaChaîneInitiale est une chaîne = "*Texte en italique*"

MonBuffer = MarkdownVersHTML(MaChaîneInitiale)
Trace(MonBuffer)
// Affiche <em>Texte en italique</em>
```



```wl
// Enregistrement dans un fichier HTML
Fichier_html est chaîne ANSI = [
<html>
    <head><meta charset="UTF-8"> </head>
<body>	
	]
Fichier_html += RC + MarkdownVersHTML(LIB_LibelléMarkdown) + "</body></html>"
fSauveTexte("C:\temp\monFichier.html", Fichier_html)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = MarkdownVersHTML(<Chaîne Markdown> , <Style CSS>)`
---

**`<Résultat> : (Buffer)`**

Buffer au format HTML (encodé en UTF8). 

**`<Chaîne Markdown> : (Chaîne de caractères)`**

Chaîne de caractères au format Markdown. 

**`<Style CSS> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> : la valeur renvoyée contiendra le style nécessaire pour assurer un rendu uniforme (en précisant les règles CSS habituelles de présentation dans un navigateur pour les balises HTML).  

- <u><u><u><u>Faux</u></u></u></u> (par défaut) : la valeur renvoyée ne contiendra pas de styles CSS.

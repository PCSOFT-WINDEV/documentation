


## &lt;Planning&gt;.SelectMoins (Fonction)

***En anglais : &lt;Scheduler&gt;.SelectMinus***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Désélectionne un rendez-vous dans un champ Planning.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ajout d'un rendez-vous dans un champ Planning
nRangRendezVous est un entier
nRangRendezVous = PLN_Planning.AjouteRendezVous("Françoise", "Réunion équipe commerciale", ...
				DateSys() + "1400", DateSys() + "1600")
// Sélection du rendez-vous ajouté
PLN_Planning.SelectPlus(nRangRendezVous)
...
// Clic sur le bouton "Désélectionner"
PLN_Planning.SelectMoins()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Désélectionner un rendez-vous identifié par son indice

`<Champ Planning>.PlanningSelectMoins([<Indice du rendez-vous>])`
---

**`<Champ Planning> : (Nom de champ)`**

Nom du champ Planning à manipuler.

**`<Indice du rendez-vous> : (Entier optionnel)`**

Indice du rendez-vous à désélectionner. Les indices des rendez-vous commencent à 1.
Si ce paramètre n'est pas précisé, le rendez-vous actuellement sélectionné est désélectionné.
Si ce paramètre est précisé et ne correspond pas à l'indice du rendez-vous actuellement sélectionné dans le planning, la fonction est sans effet.


<a name="SYNTAXE2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 
### Désélectionner un rendez-vous identifié par une variable de type RendezVous

`PlanningSelectMoins(<Champ Planning> , <Rendez-vous manipulé>)`
---

**`<Champ Planning> : (Nom de champ)`**

Nom du champ Planning à manipuler.

**`<Rendez-vous manipulé> : (Variable de type RendezVous)`**

Variable de type [RendezVous](../WDLang1/1000019244.md) correspondant au rendez-vous à désélectionner.

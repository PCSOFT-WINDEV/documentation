


## Fonctions Xaml
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions permettant de gérer les champs Xaml sont les suivantes :



|   |   |
| --- | --- |
| [XamlDésactive](../WDLang1/1000015004.md) | Simule l'absence du framework DotNet 3.0. |
| [XamlEvénement](../WDLang1/1000015001.md) | Associe une procédure écrite en WLangage à un événement d'un champ Xaml. |
| [XamlPropriété](../WDLang1/1000015002.md) | Permet de connaître ou de modifier la valeur d'une propriété .Net d'un champ Xaml ou d'un champ Carrousel. |




## &lt;Kanban&gt;.InsèreListe (Fonction)

***En anglais : &lt;Kanban&gt;.InsertList***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Insère une liste de Kanban dans un champ Kanban.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Suppression de la liste 1
KANBAN_Liste.SupprimeListe(1)
// Insertion de 2 autres listes à la place
KANBAN_Liste.InsèreListe(1, "A voir")
KANBAN_Liste.InsèreListe(2, "A faire")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Champ Kanban>.InsèreListe(<Position> , <Nom de la liste>)`
---

**`<Résultat> : (Champ)`**

Champ de type Liste de kanban.

**`<Champ Kanban> : (Nom de champ)`**

Nom du champ Kanban manipulé.

**`<Position> : (Entier)`**

Position d'insertion. Si ce paramètre est inférieur ou égal à 1, la liste est insérée à la position 1. Si ce paramètre est supérieur ou égal au nombre de listes existantes, la liste est insérée après la dernière liste de Kanban.

**`<Nom de la liste> : (Chaîne de caractères)`**

Nom de la nouvelle liste de Kanban.

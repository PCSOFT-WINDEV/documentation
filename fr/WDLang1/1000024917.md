
## SaisieInvalideAfficheIcone (Fonction)

***En anglais : InvalidInputShowIcon***

<a name="Utilisation"></a>
<a name="description"></a>
Affiche une icône d'erreur à côté d'un champ de saisie pour indiquer que la saisie est invalide.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Sortie de SAI_CodePin
SI Taille(SAI_CodePin)<4 ALORS
	SaisieInvalideAfficheIcone("Le code PIN doit faire au moins 4 caractères.", 
		icoAGauche + icoAuDessus)
	DonneFocusEtRetourUtilisateur(SAI_CodePin) 
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`SaisieInvalideAfficheIcone(<Texte de la bulle> [, <Position> [, <Icône>]])`
---

**`<Texte de la bulle> : (Chaîne de caractères)`**

Contenu de la bulle de l'icône affichée lors du survol de l'icône ou au clic sur l'icône. 

**`<Position> : (Entier optionnel)`**

Position de l'icône affichée : 


|   |   |
| --- | --- |
| *icoADroite*<br>(Valeur par défaut) | L'icône est affichée à droite du champ. |
| *icoAGauche* | L'icône est affichée à gauche du champ. |
| *icoAuDessus* | L'icône est affichée au-dessus du champ. |
| *icoEnDessous* | L'icône est affichée en dessous du champ. |



**`<Icône> : (Chaîne de caractères optionnelle)`**

Nom complet de l'image à afficher. Ce paramètre peut correspondre à n'importe quel type d'image. Elle sera affichée sous forme d'une icône de 15 x 15. 
Si ce paramètre n'est pas spécifié, l'icône par défaut (! dans un rond rouge) est affichée. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- La fonction **SaisieInvalideAfficheIcone** doit être appelée depuis un champ de saisie, sinon la fonction est sans effet.

- Si la valeur du champ est modifiée, l'icône est effacée.


## Définition (Type de variable)

***En anglais : Definition***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **Définition** permet d'obtenir les caractéristiques d'un type : 

- type simple, 

- type avancé, 

- classe,

- structure, 

- combinaison,

- énumération.




**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Récupère la définition du type de x
d est une Définition
d = RécupèreDéfinition(x)
// Affiche le nom du type de x
Trace(d.Nom)
```

## Remarques


### Propriétés spécifiques à la description des variables de type Définition
<a name="proprietes_specifiques_description_des_variables_type_definition_ELTPARAGRAPHE000057"></a>Les propriétés suivantes peuvent être utilisées pour manipuler une variable de type Définition :

| **Nom de la propriété** | **Type manipulé** | **Effet** |
| --- | --- | --- |
| AttributMapping | Chaîne de caractères | <br><br>- Valeur de l'attribut d'extension &lt;Mapping&gt; pour une classe ou une structure,<br><br>- "" (chaîne vide) si l'attribut est absent (ou pour les autres types).<br><br><br>**Cette propriété est disponible en lecture seulement**. |
| AttributNote | Chaîne de caractères | Contenu de l'attribut d'extension &lt;Note&gt;.<br><br>**Cette propriété est disponible en lecture seulement**. |
| AttributSérialise | Chaîne de caractères | Nom de sérialisation du type en fonction de son nom et de l'attribut &lt;Sérialise&gt; : <br><br>- Nom de la classe, si l'attribut est absent,<br><br>- Contenu de la chaîne si l'attribut contient une chaîne,<br><br>- "" (chaîne vide) si l'attribut contient Faux ou si le type n'est pas une classe.<br><br><br><br><br>**Cette propriété est disponible en lecture seulement**. |
| DéfinitionAssociation | Tableau de variables de type [Définition](../WDLang1/1000019540.md) | Définition des types associés. <br><br>Dans ce cas, la propriété **Associé** des variables de type [Description de Propriété](../WDLang1/1000019324.md), [Description de Variable](../WDLang1/1000019560.md) et [Description de procédure](../WDLang1/1000019317.md) renvoie <u><u><u><u>Vrai</u></u></u></u>. |
| DéfinitionBase | Tableau de variables de type [Définition](../WDLang1/1000019540.md) | Définition des types de base. |
| FamilleDéfinition | Entier | Définition des types de base. <br>Cette propriété est accessible en lecture uniquement. |
| Nom | Chaîne de caractères | Nom du type. <br>Cette propriété est accessible en lecture uniquement. |
| Option | Tableau de [Description d'Option](../WDLang1/1000023072.md) | Description des options pour une définition de combinaison.<br><br>Remarque : Information disponible aussi bien pour les combinaisons définies dans l'éditeur d'analyses que pour les combinaisons définies en WLangage. |
| Procédure | Tableau de [Description de procédure](../WDLang1/1000019317.md) | Méthodes de la classe si le type défini est une classe.<br><br>La propriété **Associé** des variables de type [Description de procédure](../WDLang1/1000019317.md) renvoie <u><u><u><u>Faux</u></u></u></u> dans le cas des méthodes de la classe de base. |
| Propriété | Tableau de [Description de Propriété](../WDLang1/1000019324.md) | Propriétés de la classe si le type défini est une classe.<br><br>La propriété **Associé** des variables de type [Description de Propriété](../WDLang1/1000019324.md) renvoie <u><u><u><u>Faux</u></u></u></u> dans le cas des propriétés de la classe de base. |
| SyntaxeDéclaration | [Description de Syntaxe](../WDLang1/1410089154.md) | Description du code de déclaration des globales pour les fenêtres, pages et états. <br><br>**Cette propriété est disponible en lecture seulement**. |
| Type | Entier | Type WLangage. <br>Cette propriété est accessible en lecture uniquement. Les différentes valeurs disponibles sont identiques à celles renvoyées par la fonction [TypeVar](../WDLang1/3013056.md). |
| Valeur | Tableau de [Description de Valeur](../WDLang1/1000023061.md) | Description des valeurs pour une définition d'énumération.<br><br>Remarque : Information disponible aussi bien pour les énumérations définies dans l'éditeur d'analyses que pour les énumérations définies en WLangage.  |
| Variable | Tableau de [Description de Variable](../WDLang1/1000019560.md) | Membres de la classe ou de la structure si le type défini est une classe ou une structure.<br><br>La propriété **Associé** des variables de type [Description de Variable](../WDLang1/1000019560.md) renvoie <u><u><u><u>Faux</u></u></u></u> dans le cas des membres de la classe de base. |

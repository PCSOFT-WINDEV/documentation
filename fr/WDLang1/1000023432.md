


## &lt;Éditeur d'images&gt;.Ouvre (Fonction)

***En anglais : &lt;Image Editor&gt;.Open***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ouvre une image dans un champ Editeur d'images.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
sFichier est une chaîne 

// Ouvre le sélecteur d'images
sFichier = fSélecteurImage("", "", "Sélectionnez une image...")

SI sFichier = "" ALORS
	RETOUR
FIN

// Ouverture de l'image dans le champ Editeur d'images
SI EI_MonEditeur.Ouvre(sFichier) = Faux ALORS
	Erreur(ErreurInfo())
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Champ Editeur d'images>.Ouvre(<Image> [, <Mode d'ouverture>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'image a été ouverte,

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour connaître le libellé de l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.




**`<Champ Editeur d'images> : (Nom de champ)`**

Nom du champ Editeur d'images utilisé pour afficher l'image.

**`<Image> : (Chaîne de caractères ou variable de type Image)`**

Image à afficher dans le champ Editeur d'images. Ce paramètre peut correspondre à : 

- une variable de type [Image](../WDLang1/1000019650.md). 

- le chemin d'un fichier image.




**`<Mode d'ouverture> : (Constante optionnelle de type Entier)`**

Mode d'ouverture de l'image : 



|   |   |
| --- | --- |
| *odOptionAucune*<br>(valeur par défaut) | L'image est affichée dans le champ Editeur d'images en mode lecture/écriture. |
| *odOptionLectureSeule* | L'image est affichée dans l'éditeur d'images en mode lecture seule.  <br><br>Dans ce mode, seules les options d'affichage et l'OCR seront accessibles. L'utilisateur pourra également imprimer et faire un "Enregistrer sous".  |
| *odOptionVoletInvisible* | L'image est affichée dans l'éditeur d'images en mode lecture/écriture et le volet de droite (permettant par exemple d'afficher les caractéristiques de couleur) n'est pas affiché par défaut.  |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Cette fonction est équivalente aux syntaxes suivantes : 

```txt
<Champ Editeur d'images> = <Chemin de l'image>
<Champ Editeur d'images> = <Variable de type Image>
```

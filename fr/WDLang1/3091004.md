


## Fonctions Exif
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions permettant de manipuler les marqueurs Exif sont les suivantes : 



|   |   |
| --- | --- |
| [ImageInfoFerme](../WDLang1/3091001.md) | Ferme le fichier image (au format JPEG ou TIFF) utilisé par la fonction [ImageInfoValeur](../WDLang1/3091003.md) et/ou la fonction [ImageInfoListe](../WDLang1/3091002.md) et libère l'image de la mémoire. |
| [ImageInfoListe](../WDLang1/3091002.md) | Renvoie la liste des marqueurs Exif disponibles pour l'image manipulée. |
| [ImageInfoModifie](../WDLang1/3091005.md) | Modifie la valeur d'un marqueur Exif pour l'image manipulée. |
| [ImageInfoSauve](../WDLang1/3091006.md) | Enregistre les modifications effectuées sur les marqueurs Exif d'une image à l'aide de la fonction [ImageInfoModifie](../WDLang1/3091005.md). |
| [ImageInfoValeur](../WDLang1/3091003.md) | Renvoie la valeur d'un marqueur Exif pour l'image manipulée. |

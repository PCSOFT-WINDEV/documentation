


## EnModeSessionPrélancée (Fonction)

***En anglais : InPrelaunchedSessionMode***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de savoir si le site est lancé en mode "session prélancée" de WEBDEV. 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI EnModeSessionPrélancée() = Vrai ALORS
	Info("Site en mode session prélancée")
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = EnModeSessionPrélancée()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le site est exécuté en mode session prélancée. 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Disponibilité de la fonction
<a name="disponibilite_fonction_ELTPARAGRAPHE000203"></a>La fonction **EnModeSessionPrélancée** est disponible dans tous les produits pour permettre l'écriture d'un code multi-produits. Des fonctions ou procédures utilisant la fonction **EnModeSessionPrélancée** peuvent être partagées entre plusieurs produits, sans que des erreurs de compilation soient générées.

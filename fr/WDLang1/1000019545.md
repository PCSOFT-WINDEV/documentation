


## AgendaMode (Fonction)

***En anglais : OrganizerMode***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le mode d'affichage d'un champ Agenda : jour, semaine, mois, ...

Remarque : La fonction [AgendaChangeMode](../WDLang1/1000019497.md) permet de modifier le mode d'affichage du champ Agenda. 

![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) En mode Universal Windows 10 App, cette fonction permet uniquement de manipuler les champs Agenda présents dans un état. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Alterne entre l'affichage par jour et par mois
SI AgendaMode(AGD_MonAgenda) = agzJour ALORS
	AgendaChangeMode(AGD_MonAgenda, agzMois)
SINON
	AgendaChangeMode(AGD_MonAgenda, agzJour)
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = AgendaMode(<Champ Agenda>)`
---

**`<Résultat> : (Constante de type Entier)`**

Plage de temps affichée par le champ Agenda :


|   |   |
| --- | --- |
| *agzJour* | Le champ Agenda affiche une journée. |
| *agzMois* | Le champ Agenda affiche un mois complet. |
| *agzNbJourPersonnalisé* | Affiche le champ Agenda sur le nombre de jours précisé grâce à la fonction [AgendaChangeMode](../WDLang1/1000019497.md).<br><br> |
| *agzSemaine* | Le champ Agenda affiche une semaine complète (ou partielle si un nombre de jours a été précisé grâce à la fonction [AgendaChangeMode](../WDLang1/1000019497.md)). |
| *agzSemaineSur5* | Le champ Agenda affiche une semaine sur 5 jours (sans le week-end). |



**`<Champ Agenda> : (Nom de champ)`**

Nom du champ Agenda à manipuler. Ce champ peut correspondre : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) au champ Agenda d'une fenêtre.

- au champ Agenda d'un état.

## Remarques


### Compatibilité
<a name="compatibilite_ELTPARAGRAPHE000140"></a>La fonction **AgendaMode** remplace la fonction [AgendaZoom](../WDLang1/1000018884.md) (conservée par compatibilité).



### Equivalences
<a name="equivalences_ELTPARAGRAPHE000153"></a>Les lignes de code suivantes sont équivalentes : 

```wl
AgendaChangeMode(AGD_Agenda, agzSemaine, 5)
```
et

```wl
AgendaChangeMode(AGD_Agenda, agzSemaineSur5)
```
 

Dans les 2 cas, la fonction **AgendaMode** renvoie la constante *agzSemaineSur5*.

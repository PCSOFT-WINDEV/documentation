
## MemRecherche (Fonction)

***En anglais : MemSeek***

<a name="Utilisation"></a>
<a name="description"></a>
Recherche un élément dans une zone mémoire :

- soit sur la valeur ajoutée par la fonction [MemAjoute](../WDLang1/3053002.md) ou sur la valeur modifiée par la fonction [MemModifie](../WDLang1/3053007.md),

- soit sur la valeur de retour.




**Remarque** : Cette recherche est une recherche à l'identique.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple
<a class="notetitle" target="_blank" href="$DOC$=1000003053015&name=memrecherche_fonction&product=WD">Voir des exemples supplémentaires</a>

```wl
// Recherche la première valeur égale à "M" dans la zone mémoire "ZoneMem"
ResRecherche = MemRecherche("ZoneMem", "M")
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = MemRecherche(<Zone mémoire> , <Valeur recherchée> [, <Élément>])`
---

**`<Résultat> : (Tout type)`**



- Si le paramètre &lt;Élément&gt; est à <u><u><u><u>Vrai</u></u></u></u> (par défaut) : 

	- Valeur du premier élément trouvé,

	- Chaîne vide ("") si aucun élément n'a été trouvé.




- Si le paramètre &lt;Élément&gt; est à <u><u><u><u>Faux</u></u></u></u> : 

	- Indice du premier élément trouvé, 

	- -1 si aucun élément n'a été trouvé.







**`<Zone mémoire> : (Chaîne de caractères)`**

Nom de la zone mémoire, défini par la fonction [MemCrée](../WDLang1/3053005.md).

**`<Valeur recherchée> : (Chaîne de caractères)`**

Valeur recherchée dans la zone mémoire. Dans le cas d'une recherche sur la valeur ajoutée, cette valeur doit être du même type que celui de la valeur ajoutée avec la fonction [MemAjoute](../WDLang1/3053002.md).

**`<Élément> : (Booléen optionnel)`**



- <u><u><u><u>Vrai</u></u></u></u> (valeur par défaut) : la recherche est effectuée sur la valeur ajoutée par la fonction [MemAjoute](../WDLang1/3053002.md) ou sur la valeur modifiée par la fonction [MemModifie](../WDLang1/3053007.md).

- <u><u><u><u>Faux</u></u></u></u> : la recherche est effectuée sur la valeur de retour.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Parcours sur une zone mémoire
<a name="parcours_sur_une_zone_memoire_ELTPARAGRAPHE000127"></a>La fonction **MemRecherche** permet d'amorcer un parcours sur la zone mémoire.

Après l'exécution de la fonction **MemRecherche** :

- la fonction [MemSuivant](../WDLang1/3053017.md) se positionnera sur le prochain élément recherché.

- la fonction [MemPrécédent](../WDLang1/3053011.md) se positionnera sur le précédent élément recherché.




Pour reprendre un parcours normal, il est nécessaire d'utiliser les fonctions [MemPremier](../WDLang1/3053014.md) ou [MemDernier](../WDLang1/3053004.md).


<a name="NOTE0_2"></a>




### Erreur
<a name="erreur_ELTPARAGRAPHE000161"></a>Une erreur du WLangage est générée lors de l'exécution de la fonction **MemRecherche** :

- si la zone mémoire &lt;Zone mémoire&gt; n'existe pas. Pour vérifier l'existence d'une zone mémoire, utilisez la fonction [MemExiste](../WDLang1/3053006.md).

- si la zone mémoire &lt;Zone mémoire&gt; n'a pas été préalablement triée. Pour trier une zone mémoire, utilisez la fonction [MemTrie](../WDLang1/3053020.md).




## &lt;Fenêtre&gt;.Repeint (Fonction)

***En anglais : &lt;Window&gt;.WinRedraw***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Redessine immédiatement les champs de la fenêtre nécessitant une mise à jour graphique.

**Remarque** : Cette fonction est identique à la fonction [MultitâcheRepeint](../WDLang1/3038004.md).








<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
i est un entier 
POUR i = 1 A 1000
	TraitementLong()
	FEN_Client.Repeint()
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Fenêtre>.Repeint()`
---

**`<Fenêtre> : (Nom de fenêtre)`**

Nom de la fenêtre à redessiner.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Comparaison avec la fonction Multitâche
<a name="comparaison_avec_fonction_multitache_ELTPARAGRAPHE000207"></a>Contrairement à la fonction [Multitâche](../WDLang1/3015004.md), l'appel de la fonction **&lt;Fenêtre&gt;.Repeint** n'interrompt pas le code en cours (les fonctions Timer ne sont pas appelées par exemple).

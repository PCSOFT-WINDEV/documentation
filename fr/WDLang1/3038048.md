


## MenuAjoute (Fonction)

***En anglais : MenuAdd***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ajoute un menu contextuel existant dans le menu principal de la fenêtre en cours.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ajout du menu "MENU_MenuContextuel1" dans l'option de menu "OPT_Fichier"
MenuAjoute(OPT_Fichier, MENU_MenuContextuel1)

// Ajout du menu "MENU_MenuContextuel2" dans le menu principal "_Menu"
MenuAjoute(_Menu, MENU_MenuContextuel2)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`MenuAjoute(<Menu destination> , <Menu contextuel> [, <Position>])`
---

**`<Menu destination> : (Nom de menu)`**

Nom de l'option de menu où l'insertion doit être réalisée.
Si ce paramètre correspond à un menu, le menu contextuel est ajouté aux options de ce menu.
Si ce paramètre correspond à une option de menu, cette option de menu affichera les options du menu contextuel.
Si ce paramètre correspond à une chaîne vide (""), le menu contextuel est ajouté dans le menu principal.  

**`<Menu contextuel> : (Nom de menu)`**

Nom du menu contextuel à insérer.

**`<Position> : (Entier optionnel)`**

Position à laquelle le menu doit être inséré. 
Si ce paramètre n'est pas précisé ou vaut -1, le menu contextuel est ajouté à la fin. 
Si ce paramètre est égal à 1, le menu contextuel est inséré au début.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Si la fenêtre ne possède pas de menu principal, ce menu est créé.

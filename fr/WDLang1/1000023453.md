


## LecteurPDFFerme (Fonction)

***En anglais : PDFReaderClose***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ferme et libère le fichier PDF affiché dans un champ Lecteur PDF.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
LecteurPDFFerme(PDF_MonLecteurPDF)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`LecteurPDFFerme(<Champ Lecteur PDF>)`
---

**`<Champ Lecteur PDF> : (Nom de champ)`**

Nom du champ Lecteur PDF contenant le fichier PDF à fermer.




## &lt;Variable Image&gt;.Police (Fonction)

***En anglais : &lt;Image variable&gt;.Font***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Déclare la police de caractères qui sera utilisée par la fonction [&lt;Variable Image&gt;.DessineTexte](../WDLang1/1000024242.md) dans une variable de type [Image](../WDLang1/1000019650.md).



![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Les dessins](../WDLang1/3029035.md). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonImage est une Image = "Test.bmp"
// Déclaration de la police de caractères :
// Arial, taille 12, gras
MonImage.Police("Arial", 12, iGras)

IMG_MonDessin = MonImage
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Utiliser une police spécifique pour les dessins

`<Variable Image>.Police(<Police> [, <Taille de la police> [, <Attributs de la police> [, <Inclinaison du texte>]]])`
---

**`<Variable Image> : (Variable de type Image)`**

Nom de la variable de type [Image](../WDLang1/1000019650.md) à manipuler. 

**`<Police> : (Chaîne de caractères)`**

Nom de la police utilisée lors du dessin du texte par [&lt;Variable Image&gt;.DessineTexte](../WDLang1/1000024242.md). Ce paramètre ne doit pas correspondre à une famille de polices. La liste des polices installées sur le poste en cours peut être connue avec la fonction [PoliceListe](../WDLang1/3060003.md). 

![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Si le nom de la police n'est pas trouvé, Windows recherche une police dont le nom est le plus proche. Pour cela, la fonction [&lt;Variable Image&gt;.DébutDessin](../WDLang1/1000024349.md) doit avoir été utilisée.

**`<Taille de la police> : (Entier optionnel)`**

Taille de la police (en points). La valeur par défaut est 12.

**`<Attributs de la police> : (Constante (ou combinaison de constantes) optionnelle)`**

Attributs de police à utiliser :


|   |   |
| --- | --- |
| *iBarre* | Barre le texte dessiné. <br> |
| *iGras* | Met le texte dessiné en gras. |
| *iItalique* | Met le texte dessiné en italique. |
| *iNormal*<br>(Valeur par défaut) | Normalise le texte dessiné |
| *iSouligne* | Souligne le texte dessiné. <br> |



**`<Inclinaison du texte> : (Entier optionnel)`**

Indique l'inclinaison (en degrés) du texte. La valeur par défaut est 0.

### Utiliser une police créée par programmation

`<Variable Image>.Police(<Police utilisée>)`
---

**`<Variable Image> : (Variable de type Image)`**

Nom de la variable de type [Image](../WDLang1/1000019650.md) à manipuler. 

**`<Police utilisée> : (Police)`**

Police à utiliser pour les dessins. Ce paramètre correspond à une variable de type [Police](../Motscles/1514045.md). Les caractéristiques de la police ont été définies grâce à la fonction [PoliceCrée](../WDLang1/3060002.md) ou grâce aux [propriétés de gestion des polices](../Proprietes/2515045.md).

### Polices
<a name="polices_ELTPARAGRAPHE000607"></a>La fonction [PoliceListe](../WDLang1/3060003.md) permet de récupérer la liste des polices valides (TrueType, PostScript) installées sur la machine en cours.

Pour obtenir un résultat satisfaisant lors de l'inclinaison du texte dessiné, la &lt;Police&gt; utilisée doit être de type "True Type".


## ChampSupprime (Fonction)

***En anglais : ControlDelete***

<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime définitivement un champ d'une fenêtre, d'une page ou d'un état. 

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Dans une fenêtre ou un état, ce champ peut avoir été créé sous l'éditeur ou par la fonction [ChampClone](../WDLang1/3025017.md).







![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) La fonction **ChampSupprime** permet uniquement de supprimer les champs créés avec la fonction [ChampClone](../WDLang1/3025017.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Suppression d'un ensemble de champs
POUR I = 1 A 10
	s est une chaîne = "SAI_" + I
	SI ChampExiste(s) = Vrai ALORS
		ChampSupprime(s)
	FIN
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`ChampSupprime(<Champ manipulé>)`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ ou de la colonne (d'un champ Table) à supprimer. Si ce champ n'existe pas, une erreur WLangage est générée.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Suppression d'une colonne d'un champ Table
<a name="suppression_une_colonne_champ_table_ELTPARAGRAPHE000105"></a>Lors de la suppression d'une colonne d'un champ Table, le contenu du champ Table (fichier ou par programmation) associé à cette colonne est supprimé.


<a name="NOTE0_2"></a>




### Limites
<a name="limites_ELTPARAGRAPHE000114"></a>Un champ supprimé ne peut pas être ré-utilisé par la suite comme source d'un clonage de champ (avec la fonction [ChampClone](../WDLang1/3025017.md)).

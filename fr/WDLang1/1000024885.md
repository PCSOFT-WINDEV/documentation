
## DocMiseAJourTableDesMatières (Fonction)

***En anglais : DocUpdateTableOfContents***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Met à jour la table des matières d'un document. Cette table des matières peut être contenue dans : 

- une variable de type Document,

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) un champ Traitement de texte, 

- un fragment. 



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonDocument est un Document
...
DocMiseAJourTableDesMatières(MonDocument)
```
<a name="Exemple2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 
```wl
DocMiseAJourTableDesMatières(TT_Fichier)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Mettre à jour la table des matières d'une variable Document

`DocMiseAJourTableDesMatières(<Document>)`
---

**`<Document> : (Variable de type Document)`**

Nom de la variable de type [Document](../WDLang1/1000022461.md) à manipuler.


<a name="SYNTAXE2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 
### Mettre à jour la table des matières d'un champ Traitement de texte

`DocMiseAJourTableDesMatières(<Champ Traitement de texte>)`
---

**`<Champ Traitement de texte> : (Nom de champ)`**

Nom du champ Traitement de texte à manipuler.


<a name="SYNTAXE3"></a>

### Mettre à jour la table des matière d'une variable Fragment

`DocMiseAJourTableDesMatières(<Fragment>)`
---

**`<Fragment> : (Variable de type docFragment)`**

Nom de la variable de type [docFragment](../WDLang1/1000022482.md) contenant la table des matières.

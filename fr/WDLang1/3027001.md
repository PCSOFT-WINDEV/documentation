
## DateDifférence (Fonction)

***En anglais : DateDifference***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Calcule la différence de jours entre deux dates.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Calcule le nombre de jours écoulés depuis le 01/01/1998
nbJours est un entier
nbJours = DateDifférence("19980101", DateDuJour())
// Affiche le résultat
Info(nbJours + " jours écoulés")
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = DateDifférence(<Date de début> , <Date de fin>)`
---

**`<Résultat> : (Entier)`**

Nombre de jours écoulés entre les deux dates. Ce nombre de jours est issu de l'opération : 

```txt
<Date de fin> - <Date de début>
```

Le nombre de jours est négatif si la &lt;Date de début&gt; est postérieure à la &lt;Date de fin&gt;.

**`<Date de début> : (Chaîne de caractères ou variable de type Date)`**

Date de début du calcul au format AAAAMMJJ (AAAA correspond à l'année, MM au mois et JJ au jour).

**`<Date de fin> : (Chaîne de caractères ou variable de type Date)`**

Date de fin du calcul au format AAAAMMJJ (AAAA correspond à l'année, MM au mois et JJ au jour).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
La validité des dates passées en paramètre est vérifiée. Un message est affiché en cas de date invalide. Il est possible de tester la validité d'une date avec la fonction [DateValide](../WDLang1/3027003.md).

Il n'est pas possible d'effectuer des différences de dates antérieures au 14 octobre 1582 (le passage du calendrier Julien au calendrier grégorien ne sera pas pris en compte).

Le format de stockage des dates permet de stocker des dates comprises entre le 01/01/0001 et le 31/12/9999.

Les calculs sur les dates réalisés avec les [fonctions WLangage](../WDLang1/3027029.md) ou les [propriétés WLangage](../Proprietes/2514010.md) sont fiables à partir du 1er Janvier 1583.

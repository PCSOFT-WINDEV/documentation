
## OemVersAnsi (Fonction)

***En anglais : OemToAnsi***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Convertit une chaîne de caractères au format OEM (DOS) en une chaîne de caractères au format ANSI (Windows). Les caractères accentués sont différents dans les deux normes. La conversion consiste à modifier les caractères accentués.

**Rappel** :

- ANSI est la norme des caractères au standard Windows.

- OEM est la norme des caractères au standard DOS (ou des fenêtres "console" de Windows).





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
Ch est une chaîne
Ch = Caract(130) + "t" + Caract(130)  // "été" en OEM
Info(OemVersAnsi(Ch))
// Affiche "été" en ANSI dans la fenêtre d'information Windows
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = OemVersAnsi(<Chaîne de caractères OEM>)`
---

**`<Résultat> : (Chaîne de caractères Ansi)`**

Chaîne de caractères au format ANSI.

**`<Chaîne de caractères OEM> : (Chaîne de caractères)`**

Chaîne de caractères au format OEM à convertir.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- La fonction **OemVersAnsi** ne convertit pas les caractères ayant un code ASCII compris entre 0 et 31 (caractères de contrôle).

- **Attention** : Ne pas utiliser la fonction **OemVersAnsi** avec une chaîne de caractères contenant des caractères spécifiques ANSI (caractères accentués) : ces caractères seront transformés et le résultat sera incorrect.

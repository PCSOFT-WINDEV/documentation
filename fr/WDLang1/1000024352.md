


## &lt;Variable Image&gt;.Fusionne (Fonction)

***En anglais : &lt;Image variable&gt;.AlphaBlend***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Fusionne une image dans une autre image. Il est ainsi possible de créer des effets de fondu entre 2 images.



![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Les dessins](../WDLang1/3029035.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonImage est une Image = "Test.bmp"
// Déclaration de l'image de promo
MonImagePromo est une Image = "ImagePromo.png"
// Fusion de l'image de promo dans l'image du produit avec une opacité à 80 %
MonImage.Fusionne(MonImagePromo, 80)
IMG_MonDessin = MonImage
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Image Destination>.Fusionne(<Image Source> [, <Opacité>])`
---

**`<Image Destination> : (Variable de type Image)`**

Nom de la variable de type [Image](../WDLang1/1000019650.md) dans laquelle la fusion doit être effectuée. Lors de la fusion, cette image sera considérée comme l'image d'arrière-plan.

**`<Image Source> : (Champ Image ou variable de type Image, WDPic ou picCalque)`**

Image à fusionner. Cette image peut correspondre : 

- au nom d'un champ Image. 

- au nom d'une variable de type [Image](../WDLang1/1000019650.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) au nom d'une variable de type [WDPic](../WDLang1/1000023503.md). Seul le calque d'arrière-plan sera manipulé. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) au nom d'une variable de type [picCalque](../WDLang1/1000024604.md).




**`<Opacité> : (Réel optionnel)`**

Pourcentage d'opacité à appliquer sur &lt;Image Source&gt; avant d'effectuer la fusion (100% par défaut, aucune opacité n'est appliquée).



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- La taille de l'image destination correspond à la taille de référence.

- La fonction **&lt;Variable Image&gt;.Fusionne** effectue la fusion en conservant la couche alpha de l'image destination. La couche alpha de l'image source n'est donc pas utilisée.

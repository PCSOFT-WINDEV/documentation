


## EntierVersJourEnLettre (Fonction)

***En anglais : IntegerToDayInAlpha***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le jour de la semaine en toutes lettres correspondant à la date spécifiée (cette date est un entier).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
EntierVersJourEnLettre(DateVersEntier("20011225")) // Renvoie "Mardi"
EntierVersJourEnLettre(73773)      // Renvoie "Mardi"
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = EntierVersJourEnLettre(<Date>)`
---

**`<Résultat> : (Chaîne de caractères)`**

Jour de la semaine correspondant à la date spécifiée : Lundi, Mardi, Mercredi, Jeudi, Vendredi, Samedi, Dimanche.

**`<Date> : (Entier)`**

Nombre de jours écoulés depuis le 1er janvier 1800 + 1 (peut être obtenu grâce à la fonction [DateVersEntier](../WDLang1/3027014.md)).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Cas particuliers
<a name="cas_particuliers_ELTPARAGRAPHE000067"></a>

- Il n'est pas possible de récupérer le jour de la semaine des dates antérieures au 14 octobre 1582 (le passage du calendrier Julien au calendrier grégorien ne sera pas pris en compte).

- La fonction **EntierVersJourEnLettre** tient compte des options linguistiques spécifiées dans la description du projet : le nom du jour sera dans la langue spécifiée dans l'onglet "Langues" de la description du projet. 
	Rappel :  Pour afficher la description du projet, sous le volet "Projet", dans le groupe "Projet", cliquez sur "Description".

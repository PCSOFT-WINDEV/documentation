


## TableurStyleSélection (Fonction)

***En anglais : SpreadsheetStyleSelection***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie ou modifie le style des cellules sélectionnées dans un champ Tableur présent dans une fenêtre.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// CLIC sur [GRAS]
TableurStyleSélection(TBLR_Tableur, tblrGras, Vrai)

// Met en gras la cellule B1
TableurSelectPlus(TBLR_Tableur, "B1")
TableurStyleSélection(TBLR_Tableur, tblrGras, Vrai)
```

```wl
sRes est un booléen = TableurStyleSélection(TBLR_Tableur, tblrGras)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître le style de la sélection

`<Résultat> = TableurStyleSélection(<Champ Tableur> , <Attribut>)`
---

**`<Résultat> : (Type du résultat attendu)`**

Attribut demandé. 

**`<Champ Tableur> : (Nom de champ)`**

Nom du champ Tableur à manipuler. 

**`<Attribut> : (Constante de type Entier)`**

Attribut à connaître : 


|   |   |
| --- | --- |
| *tblrAlignementH* | Alignement horizontal. <br><br>Dans ce cas, &lt;Résultat&gt; peut correspondre à : <br><br>- *chCentre* : Alignement centé. <br><br>- *chDroite* : Alignement à droite. <br><br>- *chGauche* : Alignement à gauche.<br><br><br> |
| *tblrAlignementV* | Alignement vertical. <br><br>Dans ce cas, &lt;Résultat&gt; peut correspondre à : <br><br>- *cvBas* : Alignement en bas. <br><br>- *cvHaut* : Alignement en haut. <br><br>- *cvMilieu* : Alignement au milieu.<br><br><br> |
| *tblrAnglePolice* | Angle de rotation du texte. <br><br>Dans ce cas, &lt;Résultat&gt; peut correspondre à un entier compris entre -180 et +180. |
| *tblrBarré* | Caractère barré. <br><br>Dans ce cas, &lt;Résultat&gt; peut correspondre à :<br><br>- <u><u><u><u>Vrai</u></u></u></u> pour barrer, <br><br>- <u><u><u><u>Faux</u></u></u></u> pour ne pas barrer.<br><br><br> |
| *tblrCouleurFond* | Couleur de fond du texte. <br><br>Dans ce cas, &lt;Résultat&gt; peut correspondre à : <br><br>- une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)),<br><br>- une couleur TSL (obtenue avec la fonction [TSL](../WDLang1/3029057.md)),<br><br>- [une couleur prédéfinie du WLangage](../WDLang5/3010002.md).<br><br><br> |
| *tblrCouleurTexte* | Couleur du texte. <br><br>Dans ce cas, &lt;Résultat&gt; peut correspondre à : <br><br>- une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)),<br><br>- une couleur TSL (obtenue avec la fonction [TSL](../WDLang1/3029057.md)),<br><br>- [une couleur prédéfinie du WLangage](../WDLang5/3010002.md).<br><br><br> |
| *tblrGras* | Caractère gras. <br><br>Dans ce cas, &lt;Résultat&gt; peut correspondre à :<br><br>- <u><u><u><u>Vrai</u></u></u></u> pour mettre en gras, <br><br>- <u><u><u><u>Faux</u></u></u></u> pour ne pas mettre en gras.<br><br><br> |
| *tblrItalique* | Caractère italique. <br><br>Dans ce cas, &lt;Résultat&gt; peut correspondre à :<br><br>- <u><u><u><u>Vrai</u></u></u></u> pour mettre en italique, <br><br>- <u><u><u><u>Faux</u></u></u></u> pour ne pas mettre en italique.<br><br><br> |
| *tblrNomPolice* | Nom de la police utilisée. <br><br>Dans ce cas, &lt;Résultat&gt; peut correspondre à une chaîne de caractères contenant le nom de la police. |
| *tblrRetourALaLigneAutomatique* | Gestion du retour à la ligne automatique. <br><br>Dans ce cas, &lt;Résultat&gt; peut correspondre à : <br><br>- <u><u><u><u>Vrai</u></u></u></u> pour utiliser le retour à la ligne automatique, <br><br>- <u><u><u><u>Faux</u></u></u></u> pour ne pas utiliser le retour à la ligne automatique. <br><br><br> |
| *tblrSouligné* | Caractère souligné. <br><br>Dans ce cas, &lt;Résultat&gt; peut correspondre à :<br><br>- <u><u><u><u>Vrai</u></u></u></u> pour souligner, <br><br>- <u><u><u><u>Faux</u></u></u></u> pour ne pas souligner.<br><br><br> |
| *tblrTaillePolice* | Taille de la police. <br><br>Dans ce cas, &lt;Résultat&gt; correspond à la taille de la police (entier). |

## Remarques


- L'opération de modification est ajoutée dans la liste des opérations à annuler : l'utilisateur peut annuler cette opération via le menu contextuel ou via la combinaison de touches Ctrl + Z.

- Cette fonction est utilisable uniquement sur un champ Tableur présent dans une fenêtre.

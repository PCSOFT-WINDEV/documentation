
## FenEdActive (Fonction)

***En anglais : WinEdActive***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet : 

- de savoir si le mode "Modification de l'UI par l'utilisateur final" est activé dans une fenêtre (ou une fenêtre interne).

- d'activer ou non le mode "Modification de l'UI par l'utilisateur final" dans une fenêtre (ou une fenêtre interne).



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Active le mode "Modification de l'UI par l'utilisateur final"
// Permet à l'utilisateur final de déplacer tous les champs sauf le champ Bouton OK
FEN_FenêtrePrésentation.FenEditionMode = fePosition
BTN_OK.FenEditionAutorisée = feAucun
FenEdActive(FEN_FenêtrePrésentation, Vrai)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Savoir si le mode "Modification de l'UI par l'utilisateur final" est activé

`<Résultat> = FenEdActive(<Fenêtre>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le mode "Modification de l'UI par l'utilisateur final" est activé, 

- <u><u><u><u>Faux</u></u></u></u> si le mode "Modification de l'UI par l'utilisateur final" est désactivé.  




**`<Fenêtre> : (Nom de fenêtre)`**

Nom de la fenêtre ou de la fenêtre interne à manipuler. 


<a name="SYNTAXE2"></a>

### Activer ou non le mode "Modification de l'UI par l'utilisateur final" 

`FenEdActive(<Fenêtre> , <Activer>)`
---

**`<Fenêtre> : (Nom de fenêtre)`**

Nom de la fenêtre ou de la fenêtre interne à manipuler. 

**`<Activer> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> pour activer le mode "Modification de l'UI par l'utilisateur final". Les autorisations de modifications doivent être définies avec les propriétés [FenEditionMode](../Proprietes/1000021824.md) et [FenEditionAutorisée](../Proprietes/1000021850.md). 

- <u><u><u><u>Faux</u></u></u></u> pour désactiver le mode "Modification de l'UI par l'utilisateur final".

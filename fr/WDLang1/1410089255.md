


## &lt;Variable picCalque&gt;.DessineTexteMarkdown (Fonction)

***En anglais : &lt;picLayer variable&gt;.DrawTextMarkdown***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Dessine un texte au format Markdown dans une variable de type [picCalque](../WDLang1/1000024604.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonImage est un WDPic = "Test.bmp"
MonCalque est un picCalque
// Dessin d'un texte au format Markdown en 8,10 dans la variable MonCalque
s est chaîne = [
# Titre
Ligne 2 avec du **gras**
*Italique* sur la ligne 3
]
MonCalque.Police("Arial", 16)
MonCalque.Stylo(VertClair) // texte en vert clair
MonCalque.DessineTexteMarkdown(8, 10, s, MonCalque.Largeur-16)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Image>.DessineTexteMarkdown(<Abscisse> , <Ordonnée> , <Texte Markdown> [, <Couleur du texte>])`
---

**`<Image> : (Variable de type picCalque)`**

Nom de la variable de type [picCalque](../WDLang1/1000024604.md) à manipuler.

**`<Abscisse> : (Entier)`**

Coordonnées (abscisse) du texte à dessiner (en pixels).

**`<Ordonnée> : (Entier)`**

Coordonnées (ordonnée) du texte à dessiner (en pixels).

**`<Texte Markdown> : (Type correspondant au texte à dessiner)`**

Texte Markdown à dessiner.

**`<Couleur du texte> : (Entier ou constante (optionnel))`**

Couleur du texte Markdown à utiliser. Cette couleur peut correspondre à :

- une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)),

- une couleur TSL (obtenue avec la fonction [TSL](../WDLang1/3029057.md)),

- [une couleur prédéfinie du WLangage](../WDLang5/3010002.md).




Si ce paramètre n'est pas précisé, la couleur du texte :

- est Noire si la fonction [&lt;Image&gt;.Stylo](../WDLang1/1000024016.md) n'a pas été utilisée auparavant,

- correspond à la couleur précisée dans le dernier appel à la fonction [&lt;Image&gt;.Stylo](../WDLang1/1000024016.md).

## Remarques


### Police
<a name="police_ELTPARAGRAPHE000347"></a>Par défaut :

- La police utilisée est la police "MS Sans Serif" (taille 13).

- Le sens d'écriture est horizontal (de gauche à droite).

- L'angle haut gauche de l'image a les coordonnées (0,0).

- Le caractère "&" n'est pas interprété.




Pour modifier ces paramètres, utilisez la fonction [&lt;Variable Image&gt;.Police](../WDLang1/1000024232.md).
![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) La police par défaut peut être différente selon le système sur lequel est exécutée l'application.
<a name="NOTE0_2"></a>


### Couleur de fond
<a name="couleur_fond_ELTPARAGRAPHE000372"></a>Par défaut, la couleur de fond d'un champ Image est transparente. Pour modifier la couleur de fond d'un champ Image, utilisez la propriété [CouleurFond](../Proprietes/2510022.md).
<a name="NOTE0_3"></a>


### Coordonnées
<a name="coordonnees_ELTPARAGRAPHE000385"></a>Les coordonnées sont indiquées par rapport au coin haut gauche du champ Image ou de l'image (coordonnées (0,0)).

Les coordonnées correspondent au point haut gauche du rectangle dans lequel le texte sera dessiné. L'emplacement précis du texte dans ce rectangle dépend de la taille, de la forme et des attributs de la police.

### Limitations
<a name="limitations_ELTPARAGRAPHE000468"></a>En mode anti-aliasing (fonction [&lt;Variable Image&gt;.ChangeMode](../WDLang1/1000024039.md)) ou en mode opacité (fonction [&lt;Variable picCalque&gt;.DébutDessin](../WDLang1/1000024470.md)), la fonction **&lt;Variable picCalque&gt;.DessineTexteMarkdown** est disponible uniquement pour les polices TrueType.

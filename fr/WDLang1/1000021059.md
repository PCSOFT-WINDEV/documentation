Les différentes constantes de gestion des champs Tableau de bord sont les suivantes :

| Constante | Description | Utilisée par les fonctions |
| --- | --- | --- |
| **tdbDéplacement** | Passage en mode édition : l'utilisateur peut déplacer les Widgets dans le champ Tableau de bord. | [TDBMode](../WDLang1/1000020956.md) |
| **tdbRedimensionnement** | Passage en mode édition : l'utilisateur peut redimensionner les Widgets dans le champ Tableau de bord. | [TDBMode](../WDLang1/1000020956.md) |
| **tdbSansModif** | Fermeture du mode édition : l'utilisateur ne peut pas modifier les Widgets dans le champ Tableau de bord. | [TDBMode](../WDLang1/1000020956.md) |
| **tdbSansRéorganisation** | Change la taille du widget sans réorganiser les widgets adjacents. | [TDBChangeTailleWidget](../WDLang1/1000021286.md),<br>[TDBDéplaceWidget](../WDLang1/1000021289.md) |
| **tdbSuppression** | Passage en mode édition : l'utilisateur peut masquer les Widgets dans le champ Tableau de bord. | [TDBMode](../WDLang1/1000020956.md) |
| **tdbUnitéCase** | L'unité utilisée pour la taille ou le déplacement du widget est un nombre de cases. | [TDBChangeTailleWidget](../WDLang1/1000021286.md),<br>[TDBDéplaceWidget](../WDLang1/1000021289.md)<br>[TDBRécupèreWidgetPosition](../WDLang1/1000022498.md) |
| **tdbUnitéPixel** | L'unité utilisée pour la taille ou le déplacement du widget est un nombre de pixels. | [TDBChangeTailleWidget](../WDLang1/1000021286.md),<br>[TDBDéplaceWidget](../WDLang1/1000021289.md)<br>[TDBRécupèreWidgetPosition](../WDLang1/1000022498.md) |

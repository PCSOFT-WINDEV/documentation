
## fRepWeb (Fonction)

***En anglais : fWebDir***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le nom physique complet du répertoire contenant les images, les fichiers Javascript et Java du site WEBDEV. Ce répertoire correspond au répertoire &lt;Nom du projet&gt;_WEB. Ce répertoire est présent sur le poste de développement et sur le poste de déploiement.

Il est ainsi possible de lire, d'écrire ou de modifier les fichiers présents dans ce répertoire. Attention les modifications ne seront possibles que si l'hébergeur accepte de donner les droits d'écriture (problèmes de sécurité).



Remarque : Pour atteindre le répertoire "&lt;Nom projet&gt;_WEB", il est possible d'utiliser : 

- la fonction [RépertoireWeb](../WDLang2/3012015.md) : cette fonction renvoie un chemin relatif destiné à être interprété par le navigateur. 

- la fonction **fRepWeb** : cette fonction renvoie un chemin absolu à utiliser depuis le serveur avec les [fonctions de gestion des fichiers externes (fxxx)](../WDLang1/3036034.md).





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Répertoire de lancement
Info("Fichiers images présents dans le répertoire : " + fRepWeb())
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = fRepWeb()`
---

**`<Résultat> : (Chaîne de caractères)`**

Nom physique complet du répertoire contenant les images, les fichiers Javascript et Java du site WEBDEV. Ce résultat est de la forme "C:\\Mes Projets\\Planning\\Planning_WEB". Il n'y a pas de caractère "\\" à la fin de ce résultat.

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
[Afficher uniquement les constantes utilisées par les syntaxes préfixées](#" onclick="$('.objet').show();$('.pas_objet').hide();)

[Afficher uniquement les constantes utilisées par les syntaxes non préfixées](#" onclick="$('.objet').hide();$('.pas_objet').show();)Les différentes constantes de gestion des champs Tableau croisé dynamique sont les suivantes :

| Constante | Détail | Fonction |
| --- | --- | --- |
| **tcdAvecUI** | Sauvegarde la configuration du champ Tableau croisé dynamique :<br><br>- taille des colonnes,<br><br>- taille des lignes,<br><br>- entête de colonnes et de lignes enroulées ou déroulées.<br><br><br> | [TCDSauve](../WDLang1/1000020471.md) |
| **tcdEntêteColonne** | Manipule les entêtes de colonnes. | [TCDDérouleTout](../WDLang1/1000020623.md), [TCDEnrouleTout](../WDLang1/1000020622.md) |
| **tcdEntêteLigne** | Manipule les entêtes de lignes. | [TCDDérouleTout](../WDLang1/1000020623.md), [TCDEnrouleTout](../WDLang1/1000020622.md) |
| **tcdEvolutionAbsolue** | Affiche l'évolution entre les périodes en valeur absolue. | [TCDComparePériode](../WDLang1/1000024869.md) |
| **tcdEvolutionPourcentage** | Affiche l'évolution entre les périodes en pourcentage. | [TCDComparePériode](../WDLang1/1000024869.md) |
| **tcdSansEvolution** | N'affiche pas l'évolution entre les périodes. | [TCDComparePériode](../WDLang1/1000024869.md) |

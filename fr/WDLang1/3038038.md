
## FenImageFond (Fonction)

***En anglais : WinBackgroundImage***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie l'image de fond d'une fenêtre en précisant le mode d'affichage de cette image.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) 
```wl
// Modification de l'image de fond de la fenêtre "FEN_SaisieImage"
// La nouvelle image est "C:\Documents\Fond.GIF"
// Cette image sera centrée dans la fenêtre
Res = FenImageFond(FEN_SaisieImage, "C:\Documents\Fond.GIF", imgCentré)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = FenImageFond(<Fenêtre manipulée> , <Nom de l'image> , <Affichage>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'image de fond a été modifiée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Fenêtre manipulée> : (Nom de fenêtre)`**

Nom de la fenêtre à manipuler.
Si ce paramètre correspond à une chaîne vide (""), la fenêtre en cours est manipulée.

**`<Nom de l'image> : (Chaîne de caractères)`**

Nom et chemin complet (ou relatif) de l'image à afficher dans la fenêtre. Un chemin de type UNC peut être utilisé. Si ce paramètre ne correspond pas à une image existante, aucune image n'est affichée dans la fenêtre.



**`<Affichage> : (Constante)`**

Indique le mode d'affichage de l'image :


|   |   |
| --- | --- |
| *imgCentré* | Centré : Image centrée dans la fenêtre.<br><br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=IMG_Centre.GIF)<br> |
| *imgEtiré* | Etiré : Image déformée pour occuper toute la surface de la fenêtre.<br><br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=IMG_Etiree.GIF)<br> |
| *imgH* | Homothétique : Image agrandie ou rétrécie en respectant les proportions de l'image d'origine pour être affichée entièrement dans la fenêtre.<br><br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=IMG_HOMO.GIF)<br> |
| *imgHC* | Homothétique centré : Image agrandie ou rétrécie en respectant les proportions de l'image d'origine et centrée dans la fenêtre.<br><br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=IMG_HomoCentre.GIF)<br> |
| *imgHE* | Homothétique étendu : Image agrandie ou rétrécie en respectant les proportions de l'image d'origine pour que le plus petit côté de l'image soit affiché entièrement dans la fenêtre.<br><br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=IMG_HOMOETENDU.GIF)<br> |
| *imgHEC* | Homothétique étendu centré : Image agrandie ou rétrécie en respectant les proportions de l'image d'origine pour que le plus petit côté de l'image soit affiché entièrement dans la fenêtre et centrée dans la fenêtre.<br><br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=IMG_HomoCentreEtendu.GIF)<br> |
| *imgHSA* | Homothétique sans agrandissement : Si l'image peut être affichée entièrement dans la fenêtre, elle est affichée à 100%. Sinon, l'image est automatiquement réduite proportionnellement pour être affichée entièrement. |
| *imgHSAC* | Homothétique sans agrandissement centrée : Si l'image peut être affichée entièrement dans la fenêtre, elle est affichée à 100% et centrée dans la fenêtre. Sinon, l'image est automatiquement réduite proportionnellement pour être affichée entièrement au centre de la fenêtre. |
| *imgMosaique* | Répété : Image affichée à 100% et répétée pour occuper tout l'espace de la fenêtre (en mosaïque).<br><br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=IMG_Mosaique.GIF)<br> |
| *imgNormal* | 100% : Image affichée à 100% en haut à gauche de la fenêtre.<br><br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=IMG_100.GIF)<br> |

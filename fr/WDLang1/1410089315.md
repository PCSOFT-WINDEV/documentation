
## France.VérifieINSEE (Fonction)

***En anglais : INSEEValid***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Teste la validité d'un numéro INSEE (numéro d'inscription au répertoire des personnes physiques attribué par l'Institut National de la Statistique et des Etudes Economiques).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Test d'une variable
MonNuméroINSEE est une chaîne = "285063410315510"
France.VérifieINSEE(MonNuméroINSEE)		// renvoie Vrai

// Test d'un champ
SAI_NuméroINSEE = "ABC"
France.VérifieINSEE(SAI_NuméroINSEE)	// renvoie Faux
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = France.VérifieINSEE(<Numéro à vérifier>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'élément spécifié contient un numéro INSEE valide.

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Numéro à vérifier> : (Nom de champ ou variable de type chaîne)`**

Numéro INSEE à vérifier. Ce paramètre peut correspondre :

- au nom d'une variable de type chaîne.

- au nom d'un champ.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Un numéro INSEE est considéré valide même sans sa clé de contrôle finale sur 2 chiffres.


## fRepTaille (Fonction)

***En anglais : fDirSize***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la taille d'un répertoire en octets.  

**Attention** : Le temps d'exécution de cette fonction peut être relativement long selon la taille du répertoire à analyser.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) 
```wl
ResTaille est un entier
// Taille du répertoire "C:\MesDocuments"
ResTaille = fRepTaille("C:\MesDocuments")
// Affichage lisible
Info("La taille du répertoire est : " + TailleVersChaîne(ResTaille))
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = fRepTaille(<Chemin du répertoire> [, <Options>])`
---

**`<Résultat> : (Entier)`**



- Taille du répertoire en octets,

- -1 en cas d'erreur ou d'interruption. Pour connaître le libellé de l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.




**Remarque** : les fichiers cachés sont pris en compte dans le calcul de la taille du répertoire.

**`<Chemin du répertoire> : (Chaîne de caractères)`**

Nom et chemin complet (ou relatif) du répertoire à manipuler (260 caractères maximum). Un chemin de type UNC peut être utilisé. Les caractères jockers (\*,?) sont interdits.

![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png)![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) Ce paramètre peut être au format Ansi ou Unicode.





**`<Options> : (Constante ou combinaison de constantes (optionnelle) de type entier)`**

Type de parcours effectué pour les fichiers du répertoire :


|   |   |
| --- | --- |
| *frIgnoreErreur* | Si un répertoire ne peut pas être parcouru, il est ignoré. Si cette constante n'est pas précisée, **fRepTaille** renvoie une erreur. |
| *frInterruptible* | Le parcours peut être interrompu en utilisant la touche ECHAP. Le résultat renvoyé sera alors le nom des répertoires listés jusqu'à l'interruption.<br><br><br>![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) Cette constante n'a aucun effet. |
| *frNonRécursif* | Le parcours est non récursif. Les sous-répertoires ne sont pas pris en compte. |
| *frRécursif*<br>(Valeur par défaut) | Le parcours est récursif. Les sous-répertoires sont automatiquement pris en compte. |







<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Gestion des erreurs
<a name="gestion_des_erreurs_ELTPARAGRAPHE000135"></a>La fonction **fRepTaille** génère une erreur dans les cas suivants :

- le répertoire spécifié n'existe pas,

- le nom du répertoire spécifié est invalide,

- l'utilisateur a interrompu l'opération.

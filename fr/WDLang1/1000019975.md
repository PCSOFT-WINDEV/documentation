


## &lt;Organigramme&gt;.Ajoute (Fonction)

***En anglais : &lt;Organization Chart&gt;.Add***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ajoute un nouvel élément "racine" dans un champ Organigramme. Si le champ Organigramme est lié à une source de données, l'élément est également ajouté dans la source de données.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ajout d'un élément "Racine" dans le champ Organigramme (Syntaxe 1)
Boss est un entier 
Boss = ORGA_MonOrganigramme.Ajoute("Matthieu Boss", "Big boss", "Goldbar.gif")
// Modification de la couleur de fond de cet élément
ORGA_MonOrganigramme[Boss].CouleurFond = iJauneClair
```
<a name="Exemple2"></a>

```wl
// Ajout d'éléments dans le champ Organigramme (syntaxe 2)
// Elément Racine
ElémentPDG est un OrgaElément
ElémentPDG.Titre = "Big Boss"
ORGA_DIRECTION.Ajoute(ElémentPDG)

// Eléments fils
Elément2 est un OrgaElément
Elément2.Titre = "Production"
ORGA_DIRECTION.AjouteFils(1, Elément2)

Elément3 est un OrgaElément
Elément3.Titre = "Vente"
ORGA_DIRECTION.AjouteFils(1, Elément3)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Ajouter un élément "racine" en précisant ses caractéristiques

`<Résultat> = <Champ Organigramme>.Ajoute(<Titre> [, <Contenu> [, <Image>]])`
---

**`<Résultat> : (Entier)`**



- Indice de l'élément ajouté. 

- -1 en cas d'erreur. La fonction [ErreurInfo](../WDLang1/3013008.md) permet d'identifier l'erreur.




**`<Champ Organigramme> : (Nom de champ)`**

Nom du champ Organigramme dans lequel l'élément doit être ajouté.

**`<Titre> : (Chaîne de caractères)`**

Titre de l'élément.

**`<Contenu> : (Chaîne de caractères optionnelle)`**

Contenu de l'élément.

**`<Image> : (Chaîne de caractères optionnelle)`**

Image associée à l'élément. Cette image s'affiche à gauche du titre de l'élément. Ce paramètre correspond au nom et au chemin complet (ou relatif) de l'image. Un chemin de type UNC peut être utilisé.


<a name="SYNTAXE2"></a>

### Ajouter un élément "racine" en utilisant le type OrgaElément

`<Résultat> = <Champ Organigramme>.Ajoute(<Élément>)`
---

**`<Résultat> : (Entier)`**



- Indice de l'élément ajouté.

- -1 en cas d'erreur. La fonction [ErreurInfo](../WDLang1/3013008.md) permet d'identifier l'erreur.




**`<Champ Organigramme> : (Nom de champ)`**

Nom du champ Organigramme dans lequel l'élément doit être ajouté.

**`<Élément> : (Variable de type OrgaElément)`**

Nom de la variable de type [OrgaElément](../WDLang1/1000019713.md) décrivant les caractéristiques de l'élément à ajouter.

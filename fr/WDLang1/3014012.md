
## SupprimeRaccourci (Fonction)

***En anglais : DeleteShortCut***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime un raccourci précédemment créé avec la fonction [CréeRaccourci](../WDLang1/3014008.md) sur le bureau de Windows, dans le menu "Démarrer" de Windows ou dans un répertoire spécifique.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Crée le raccourci "Test" du programme C:\Test\Test.exe"
// sur le bureau, avec Ctrl-Alt-A comme raccourci clavier 
CréeRaccourci(raccourciBureau, "Test", "C:\Test\Test.exe", "", sfaNormal, "A")
...
SupprimeRaccourci(raccourciBureau, "Test")
```
<a name="Exemple2"></a>

```wl
// Créer le raccourci dans "Démarrer/Programmes/Accessoires"
CréeRaccourci(raccourciProgramme, "Test", "C:\Test\Test.exe", "Accessoires", sfaNormal, "A")
...
SupprimeRaccourci(raccourciProgramme, "Test")
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = SupprimeRaccourci(<Emplacement> , <Nom> [, <Chemin>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le raccourci a été supprimé, 

- <u><u><u><u>Faux</u></u></u></u> en cas de problème.




**`<Emplacement> : (Constante de type entier)`**

Emplacement relatif du chemin du raccourci :


|   |   |
| --- | --- |
| *raccourciBureau*<br>(Valeur par défaut) | Raccourci sur le bureau de Windows.<br>Sous Windows Vista, l'application supprimant le raccourci doit avoir un manifeste administrateur. |
| *raccourciBureauUtilisateur* | Raccourci sur le bureau de Windows de l'utilisateur courant du poste. |
| *raccourciDémarrage* | Raccourci dans le menu "Démarrer .. Programmes .. Démarrage" de Windows pour tous les utilisateurs du poste. |
| *raccourciDémarrageUtilisateur* | Raccourci dans le menu "Démarrer .. Programmes .. Démarrage" de Windows pour l'utilisateur courant du poste. |
| *raccourciDémarrer* | Raccourci dans le menu "Démarrer" de Windows pour tous les utilisateurs du poste. |
| *raccourciDémarrerUtilisateur* | Raccourci dans le menu "Démarrer" de Windows pour l'utilisateur courant du poste. |
| *raccourciLancementRapide* | Raccourci dans la barre de lancement rapide (Windows 98 et version supérieures). |
| *raccourciLibre* | Raccourci dans le répertoire indiqué par le paramètre &lt;Chemin&gt;. |
| *raccourciProgramme* | Raccourci dans le menu "Démarrer .. Programmes" de Windows.<br>Sous Windows Vista (et supérieur), l'application supprimant le raccourci doit avoir un manifeste administrateur. |
| *raccourciProgrammeUtilisateur* | Raccourci dans le menu "Démarrer .. Programmes" de Windows pour l'utilisateur courant du poste. |



**`<Nom> : (Chaîne de caractères)`**

Nom du raccourci.

**`<Chemin> : (Chaîne de caractères optionnelle)`**

Répertoire relatif à l'&lt;Emplacement&gt; d'où le raccourci doit être supprimé. Si &lt;Emplacement&gt; = *raccourciLibre*, le chemin est absolu.

Si ce paramètre n'est pas précisé ou correspond à une chaîne vide (""), le raccourci sera supprimé directement dans l'&lt;Emplacement&gt; indiqué.




## KanbanDéplaceListe (Fonction)

***En anglais : KanbanMoveList***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Déplace une liste de Kanban dans un champ Kanban.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Déplace la liste LISTE_A_VOIR vers la position 2
KanbanDéplaceListe(KANBAN_Planning, KanbanIndiceListe(LISTE_A_VOIR), 2)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`KanbanDéplaceListe(<Champ Kanban> , <Position avant déplacement> , <Position après déplacement>)`
---

**`<Champ Kanban> : (Nom de champ)`**

Nom du champ Kanban manipulé. 

**`<Position avant déplacement> : (Entier)`**

Position de la liste de Kanban avant déplacement. Cette position peut être connue avec la fonction [KanbanIndiceListe](../WDLang1/1410089317.md). 

**`<Position après déplacement> : (Entier)`**

Nouvelle position de la liste de Kanban. 

- Si cette position est inférieure ou égale à 1, la liste de Kanban est déplacée en première position. 

- Si cette position est supérieure ou égale au nombre de listes de Kanban dans le champ, la liste de Kanban est déplacée en dernière position. 

- Si &lt;Position avant déplacement&gt; = &lt;Position après déplacement&gt;, aucun déplacement n'est effectué.

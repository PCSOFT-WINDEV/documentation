
## tcdPosition (Type de variable)

***En anglais : pvtPosition***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **tcdPosition** permet de manipuler directement une cellule d'un champ Tableau croisé dynamique. Ce type de variable permet notamment de récupérer la valeur présente dans une cellule ou de modifier les caractéristiques de la cellule (la couleur de fond par exemple). 

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Déclaration
x est un tcdPosition de TCD_Ventes

// Spécifie les valeurs des entêtes de la cellule 
x.Année = "2020"
x.Trimestre = "2020T1"
x.Mois = "202001"
x.Produit.Cumul = Vrai 

// Afficher le total des ventes de tous les produits en janvier 2020
LIB_Resultat = TCD_Ventes.Valeur1[x]
```

<a name="NOTE0"></a>

## Remarques
<a name="NOTE0_1"></a>


### Propriétés spécifiques à la description des variables de type tcdPosition
<a name="proprietes_specifiques_description_des_variables_type_tcdposition_ELTPARAGRAPHE000028"></a>Les propriétés suivantes peuvent être utilisées pour manipuler une variable de type **tcdPosition** :

| **Nom de la propriété** | **Type manipulé** | **Effet** |
| --- | --- | --- |
| EnDehors | Booléen | <br><br>- <u><u><u><u>Vrai</u></u></u></u> si la position indiquée est en dehors du tableau croisé dynamique. <br><br>- <u><u><u><u>Faux</u></u></u></u> si la position est valide. <br><br><br>**Cette propriété est disponible en lecture seulement**.<br><br>Cette propriété est notamment utile pour savoir si la position indiquée avec la fonction [TCDInfoXY](../WDLang1/1000020735.md) existe. |




<a name="NOTE0_2"></a>


### Fonctionnement
<a name="fonctionnement_ELTPARAGRAPHE000063"></a>Pour manipuler une variable de type **tcdPosition**, il suffit de : 

- spécifier les valeurs des entêtes de la cellule voulue. Par exemple : 
	
	```wl
	MaPosition est un tcdPosition de TCD_Statistiques
	// Indique les entêtes
	MaPosition.COL_Continent = "Europe"
	MaPosition.COL_DateCommande_Année = "2012"
	```


- accéder à une valeur du tableau croisé dynamique par la syntaxe suivante : 
	
	```txt
	<Nom Tableau croisé dynamique>.<Nom valeur>[<variable tcdPosition>]
	```

	Par exemple : 
	
	```wl
	// Récupère la valeur "VAL_Qté"
	nQuantité = TCD_Statistiques.VAL_Qté[MaPosition]
	```





Remarques : 

- Si la position ne correspond pas à une cellule affichée (par exemple parce qu'elle est repliée), il ne sera pas possible de récupérer la valeur de la cellule ou de modifier sa couleur. 

- Si la position ne correspond pas à une valeur qui existe dans le résultat (par exemple un produit qui ne s'est pas vendu ce jour-là), la valeur renvoyée est 0. 

- Si la position possède un nom d'entête qui n'existe pas dans le tableau croisé dynamique, une erreur WLangage est affichée.

- Si la position est en-dehors du tableau croisé dynamique, une erreur WLangage est affichée. 

- Pour obtenir un cumul sur un entête, il suffit de ne pas spécifier l'entête ou d'utiliser la propriété [.Cumul](../Proprietes/1000020702.md) à Vrai.

- La propriété **.EnDehors** est utile pour savoir si la position renvoyée par la fonction [TCDInfoXY](../WDLang1/1000020735.md) existe. 



<a name="NOTE0_3"></a>


### Cas des tableaux croisés dynamiques avec filtre
<a name="cas_des_tableaux_croises_dynamiques_avec_filtre_ELTPARAGRAPHE000096"></a>Les variables de type **tcdPosition** manipulent les données du tableau croisé dynamique en entier : les données non affichées sont également prises en compte. En cas d'utilisation de filtres (fonction [TCDFiltre](../WDLang1/1000020531.md)), il est nécessaire de déclarer ce filtre dans la variable **tcdPosition**. 

Exemple : 


```wl
MaPosition est un tcdPosition de TCD_Statistiques

// Indique le filtre
MaPosition.COL_FILTRE_Produit = COMBO_Produit

// Indique les entêtes
MaPosition.COL_Continent = "Europe"
MaPosition.COL_DateCommande_Année = "2012"

// Récupère la valeur "VAL_Qté"
nQuantité = TCD_Statistiques.VAL_Qté[MaPosition]
```

<a name="NOTE0_4"></a>


### Modifier les caractéristiques d'une cellule
<a name="modifier_les_caracteristiques_une_cellule_ELTPARAGRAPHE000110"></a>Pour mettre en avant une cellule à l'affichage (valeur maximale, minimale, ...), il est possible d'utiliser une variable de type **tcdPosition**.  

Exemple : 


```wl
MaPosition est un tcdPosition de TCD_Statistiques

// Indique les entêtes
MaPosition.COL_Continent = "Europe"
MaPosition.COL_DateCommande_Année = "2012"

// Modifie la couleur de fond de la valeur "VAL_Qté"
VAL_Qté[MaPosition].CouleurFond = RougeClair
```

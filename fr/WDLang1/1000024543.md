


## &lt;Variable Document&gt;.VersTexte (Fonction)

***En anglais : &lt;Document variable&gt;.ToText***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Crée une chaîne de caractères avec les données d'une variable de type [Document](../WDLang1/1000022461.md).

Cette chaîne pourra ensuite par exemple être sauvegardée dans un format spécifique.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonDocument est un Document

MaChaîne est une chaîne
MaChaîne = MonDocument.VersTexte()
// Sauvegarde dans un fichier texte
fSauveTexte("C:\temp\Contenu.txt", MaChaîne)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Document>.VersTexte()`
---

**`<Résultat> : (Chaîne de caractères)`**



- Chaîne de caractères contenant les données. 

- Chaîne vide ("") en cas d'erreur ou si il n'y a pas de données à extraire.




**`<Document> : (Variable de type Document ou Chaîne de caractères)`**

Nom de la variable de type [Document](../WDLang1/1000022461.md) à manipuler.




## &lt;Tableau&gt;.Info (Fonction)

***En anglais : &lt;Array&gt;.Info***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie les caractéristiques d'un tableau : type des éléments, nombre de dimensions, dimensions, ...

**Remarque** : Cette fonction est utilisable uniquement sur les [tableaux WLangage](../Motscles/1514030.md). 










<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Nombre total d'éléments dans un tableau à une dimension
MonTableau est un tableau de 5 entiers
...
nbDim est un entier
nbDim = MonTableau.Info(tiNombreTotal)
// nbDim vaut 5
```
<a name="Exemple2"></a>

```wl
// Dimensions d'un tableau
MonTableau est un tableau de 5 par 2 entiers
nbDimLigne est un entier 
nbDimLigne = MonTableau.Info(tiNombreLignes)
// NbDimLigne vaut 5
nbDimColonne est un entier 
nbDimColonne = MonTableau.Info(tiNombreColonnes)
// NbDimColonne vaut 2
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Tableau WLangage>.Info(<Type d'information> [, <Paramètre supplémentaire>])`
---

**`<Résultat> : (Entier ou booléen)`**

Information demandée.

Remarque : La fonction ne retourne aucun résultat sur un tableau dynamique qui n'a pas été alloué.

**`<Tableau WLangage> : (Tableau)`**

Nom de la variable de type Tableau à manipuler.

**`<Type d'information> : (Constante de type Entier)`**

Type d'information à récupérer sur le tableau :


|   |   |
| --- | --- |
| *tiAssociatifAvecDoublon* | Booléen.<br><br>- <u><u><u><u>Vrai</u></u></u></u> si le tableau associatif gère les doublons<br><br>- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.<br><br><br> |
| *tiDéfinitionElément* | Variable de type [Définition](../WDLang1/1000019540.md). Définition du type des éléments.<br> |
| *tiDimension* | Entier. Nombre d'éléments dans une dimension du tableau.<br>&lt;Paramètre supplémentaire&gt; doit contenir l'indice de la dimension dont on veut connaître le nombre d'éléments. <br>Valeur par défaut de &lt;Paramètre supplémentaire&gt; : 1<br>Tableau associatif : Nombre d'éléments dans le tableau. |
| *tiDynamique* | Booléen.<br><br>- <u><u><u><u>Vrai</u></u></u></u> si le tableau est un tableau dynamique.<br><br>- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.<br><br><br> |
| *tiNombreColonnes* | Entier. Nombre de colonnes d'un tableau à 2 dimensions. Le nombre de colonnes correspond à la 2ème dimension du tableau. <br>Si le tableau n'a pas 2 dimensions, tiNombreColonnes = 0.<br>Tableau associatif : 0. |
| *tiNombreDimensions* | Entier. Nombre de dimensions du tableau (entre 1 et 10)<br>Tableau associatif : 1. |
| *tiNombreLignes* | Entier. Nombre de lignes d'un tableau à 2 dimensions. Le nombre de lignes correspond à la 1ère dimension du tableau. <br>Si le tableau n'a pas 2 dimensions, tiNombreLignes = 0.<br>Tableau associatif : 0. |
| *tiNombreTotal* | Entier. Nombre total d'éléments dans le tableau. |
| *tiTailleElément* | Entier. Taille (en octets) d'un élément du tableau.<br> |
| *tiTailleTotale* | Entier. Taille totale (en octets) du tableau.<br> |
| *tiTypeClé* | Entier. Type des clés du tableau associatif. &lt;Résultat&gt; est un entier correspondant aux constantes de la fonction [TypeVar](../WDLang1/3013056.md).<br>Tableau non associatif : &lt;Résultat&gt; vaut 0. |
| *tiTypeElément* | Entier. Type des éléments du tableau. &lt;Résultat&gt; est un entier correspondant aux constantes de la fonction [TypeVar](../WDLang1/3013056.md).<br> |



**`<Paramètre supplémentaire> : (Optionnel)`**

Paramètre supplémentaire à indiquer en fonction de l'information recherchée.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Equivalences
<a name="equivalences_ELTPARAGRAPHE000356"></a>

- La constante *tiNombreLignes* est équivalente à la constante *tiDimension*, avec &lt;Paramètre supplémentaire&gt; = 1. Les codes suivants sont identiques :
	
	```wl
	MonTableau.Info(tiNombreLignes)
	MonTableau.Info(tiDimension, 1)
	```


- La constante *tiNombreColonnes* est équivalente à la constante *tiDimension*, avec &lt;Paramètre supplémentaire&gt; = 2.
	
	```wl
	MonTableau.Info(tiNombreColonnes)
	MonTableau.Info(tiDimension, 2)
	```




<a name="NOTE0_2"></a>


### Divers
<a name="divers_ELTPARAGRAPHE000381"></a>Cette fonction peut être utilisée avec :

- les tableaux simples.

- les tableaux fixes.

- les tableaux dynamiques.

- les tableaux associatifs.

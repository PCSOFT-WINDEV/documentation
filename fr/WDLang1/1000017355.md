


## SupprimeLigne (Fonction)

***En anglais : DeleteLine***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime une ligne à une position donnée :

- dans un tableau à deux dimensions.

- dans la propriété tableau d'un type avancé (tableau des événements de gglAgenda, ...)


La taille du tableau est automatiquement réduite.

**Remarque** : Cette fonction est équivalente à la fonction [TableauSupprimeLigne](../WDLang1/3075015.md).








<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonTableau est un tableau de 2 par 3 chaînes
InsèreLigne(MonTableau, 1, "A", "B", "C")
InsèreLigne(MonTableau, 2, "a", "b", "c")
InsèreLigne(MonTableau, 3, "D", "E", "F")
// Suppression de la 1° ligne
SupprimeLigne(MonTableau, 1) // Supprime "A B C"
// Affichage de la 1° ligne
Trace(MonTableau[1,1], MonTableau[1,2], MonTableau[1,3])
// Affiche "a b c"
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`SupprimeLigne(<Tableau WLangage> , <Indice de la ligne>)`
---

**`<Tableau WLangage> : (Tableau)`**

Nom de la variable de type Tableau à manipuler. Ce tableau doit être un tableau à deux dimensions.

**`<Indice de la ligne> : (Entier)`**

Indice de la ligne à supprimer. Si ce paramètre est supérieur au nombre de lignes dans le tableau, une erreur WLangage est générée.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Suppression de la dernière ligne
<a name="suppression_derniere_ligne_ELTPARAGRAPHE000086"></a>Si la dernière ligne est supprimée, la taille du tableau passe à 0. Le tableau n'est pas libéré : Il est possible de faire de nouveaux ajouts sans ré-allouer le tableau.
<a name="NOTE0_2"></a>


### Divers
<a name="divers_ELTPARAGRAPHE000093"></a>Cette fonction ne peut pas être utilisée avec les tableaux fixes.




## &lt;Table&gt;.SelectVersFichier (Fonction)

***En anglais : &lt;Table&gt;.SelectToFile***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Positionne sur l'enregistrement du fichier de données correspondant à une ligne sélectionnée dans le champ Table fichier multisélection.

**Remarque** : Sur un champ Table fichier monosélection, l'enregistrement en cours dans le fichier de données correspond toujours à l'élément sélectionné dans le champ Table.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
i est un entier
POUR i = 1 A TABLE_Produit.SelectOccurrence()
	TABLE_Produit.SelectVersFichier(i)
	Trace(Produit.IDProduit)
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Champ Table>.SelectVersFichier(<Rang>)`
---

**`<Champ Table> : (Nom de champ)`**

Nom du champ Table fichier à manipuler. 
Si ce paramètre ne correspond pas à un champ Table fichier, une erreur WLangage est affichée.

**`<Rang> : (Entier)`**

Rang de la sélection. Si ce paramètre ne correspond pas à un rang existant, une erreur WLangage est affichée.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Il est possible d'utiliser cette fonction avec un champ Table fichier monosélection si et seulement si le paramètre &lt;Rang&gt; vaut 1.

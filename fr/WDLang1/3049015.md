


## ListePosition (Fonction)

***En anglais : ListPosition***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Affiche un champ Liste (y compris champ Liste image) à partir d'un élément spécifié ou renvoie l'indice du premier élément affiché dans un champ Liste (lorsque l'ascenseur du champ a été utilisé).

**Remarque** : La fonction **ListePosition** peut être utilisée indifféremment sur :

- les champs Liste fichier ou les champs Liste par programmation,

- les champs Liste monosélection ou multisélection.













<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Indice du premier élément affiché dans le champ Liste "LISTE_ListeClient"
ResPosition = ListePosition(LISTE_ListeClient)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Afficher un champ Liste à partir d'un élément spécifié

`ListePosition(<Champ Liste> , <Indice de l'élément>)`
---

**`<Champ Liste> : (Nom de champ)`**

Nom du champ Liste à manipuler.
Si ce paramètre correspond à une chaîne vide (""), l'élément est affiché au début du champ Liste auquel appartient l'événement en cours.

**`<Indice de l'élément> : (Entier)`**

Indice de l'élément à afficher au début du champ Liste. 
Si ce paramètre est supérieur au nombre d'éléments présents dans le champ Liste (connu avec la fonction [ListeOccurrence](../WDLang1/3049014.md)), une erreur WLangage est affichée.

Si ce paramètre est égal à -1, le champ Liste est repositionnée au début.  


<a name="SYNTAXE2"></a>

### Connaître l'indice du premier élément affiché dans un champ Liste ou Combo

`<Résultat> = ListePosition(<Champ Liste>)`
---

**`<Résultat> : (Entier)`**

Indice du premier élément affiché dans le champ Liste spécifié.

**`<Champ Liste> : (Nom de champ)`**

Nom du champ Liste à manipuler.
Si ce paramètre correspond à une chaîne vide (""), l'indice du premier élément du champ Liste auquel appartient l'événement en cours est renvoyé.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Afficher un élément au début d'un champ Liste (syntaxe 1)
<a name="afficher_element_debut_champ_liste_syntaxe_1_ELTPARAGRAPHE000132"></a>L'élément affiché au début d'un champ Liste peut être différent de l'élément demandé. En effet, la fonction **ListePosition** place au début du champ l'élément permettant de visualiser un maximum de lignes dans le champ. S'il n'y a pas assez d'éléments affichés en-dessous de l'élément spécifié, cet élément ne pourra pas être affiché au début du champ.

Par exemple, l'élément "Durand Jacques" ne peut pas être affiché au début du champ Liste : il n'y a pas assez d'éléments affichés en-dessous de cet élément.<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=LISTEPOSITION.gif)

<a name="NOTE0_2"></a>


### Limitations
<a name="limitations_ELTPARAGRAPHE000147"></a>La fonction **ListePosition** ne fonctionne pas pour les champs Liste fichier en mode "manuel" qui utilisent un ascenseur à rebond.

Rappel :

- Un champ Liste fichier en mode "manuel" est un champ Liste relié par programmation à un fichier de données.

- Avec un ascenseur à rebond, il est possible de faire défiler l'ascenseur en gardant le bouton de la souris enfoncé sur le bouton de l'ascenseur ou en gardant la touche flèche haut ou flèche bas enfoncée.



<a name="NOTE0_3"></a>


### Divers
<a name="divers_ELTPARAGRAPHE000164"></a>

- Pour afficher un élément au début d'un champ Table ou pour connaître l'indice du premier élément affiché dans un champ Table, utilisez la fonction [TablePosition](../WDLang1/3074022.md).

- La fonction **ListePosition** peut être utilisée sur une colonne d'un champ Table de type "Combo".

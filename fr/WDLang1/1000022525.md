


## &lt;Tableau de bord&gt;.ConfigurationInitiale (Fonction)

***En anglais : &lt;Dashboard&gt;.InitialConfiguration***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie la configuration d'un champ Tableau de bord (définie sous l'éditeur) avant son premier affichage. Il est ainsi possible de configurer et de prendre en compte les Widgets déclarés en programmation par la fonction [&lt;Tableau de bord&gt;.AjouteWidget](../WDLang1/1000022502.md).




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 
```wl
// -- Initialisation de TDB_TableauDeBord
// Ajoute des informations dans la configuration initiale :
// - chiffres clés des 3 derniers mois
nX = 1 ; nY = 3
POUR i = 1 A 3
	// Ajoute le widget
	nIndice = TDB_TableauDeBord.AjouteWidget(FI_Widget_ChiffreClé, ...
			ChaîneConstruit("Chiffres de %1", DateVersChaîne(dDate, "Mmm AAAA")), ...
			dDate)
	// Configure le widget (calé à gauche)
	TDB_TableauDeBord.ConfigurationInitiale(nIndice, nX, nY)
	// Le prochain sera plus bas
	nY++
	dDate.Mois--
FIN
// - alerte de stock
// Ajoute le widget
nIndice = TDB_TableauDeBord.AjouteWidget(FI_Widget_AlerteStock, "Alerte stock")
// Configure le widget (calé à gauche) 
TDB_TableauDeBord.ConfigurationInitiale(nIndice, 5, 1)
```

## Syntaxe
<a name="SYNTAXE1"></a>

### Configuration initiale : manipuler le Widget par son indice

`<Champ Tableau de bord>.ConfigurationInitiale(<Indice du Widget> [, <X> [, <Y> [, <Largeur> [, <Hauteur>]]]])`
---

**`<Champ Tableau de bord> : (Nom de champ)`**

Nom du champ Tableau de bord à manipuler.

**`<Indice du Widget> : (Entier)`**

Indice du Widget à manipuler. Cet indice correspond : 

- au numéro d'ajout du Widget défini sous l'éditeur de fenêtres ou de pages. 

- au numéro d'ajout du Widget renvoyé par la fonction [&lt;Tableau de bord&gt;.AjouteWidget](../WDLang1/1000022502.md).




**`<X> : (Entier optionnel)`**

Position horizontale du Widget en nombre de cases.

Si les paramètres [&lt;X&gt;, &lt;Y&gt;] ne sont pas précisés ou ont pour valeur [0,0], le Widget est positionné sur la première place libre. 

La première case est la case [1, 1].

**`<Y> : (Entier optionnel)`**

Position verticale du Widget en nombre de cases.

Si les paramètres [&lt;X&gt;, &lt;Y&gt;] ne sont pas précisés ou ont pour valeur [0,0], le Widget est positionné sur la première place libre. 

La première case est la case [1, 1].

**`<Largeur> : (Entier optionnel)`**

Largeur du Widget en nombre de cases.

Si les paramètres [&lt;Largeur&gt;, &lt;Hauteur&gt;] ne sont pas précisés ou ont pour valeur [0, 0] : 

- au premier appel de la fonction **&lt;Tableau de bord&gt;.ConfigurationInitiale**, la taille du Widget correspond à la taille de la fenêtre ou page interne source (au moins 1 case).

- aux appels suivants de la fonction **&lt;Tableau de bord&gt;.ConfigurationInitiale**, la taille du Widget correspond à la taille définie au premier appel.




**`<Hauteur> : (Entier optionnel)`**

Hauteur du Widget en nombre de cases.

Si les paramètres [&lt;Largeur&gt;, &lt;Hauteur&gt;] ne sont pas précisés ou ont pour valeur [0, 0] : 

- au premier appel de la fonction **&lt;Tableau de bord&gt;.ConfigurationInitiale**, la taille du Widget correspond à la taille de la fenêtre ou page interne source (au moins 1 case).

- aux appels suivants de la fonction **&lt;Tableau de bord&gt;.ConfigurationInitiale**, la taille du Widget correspond à la taille définie au premier appel.

<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Si la fonction **&lt;Tableau de bord&gt;.ConfigurationInitiale** est appelée après l'initialisation du champ Tableau de bord, une erreur est générée.

- Les Widgets configurés par la fonction **&lt;Tableau de bord&gt;.ConfigurationInitiale** seront chargés et rendus visibles par défaut après l'exécution des traitements d'initialisation.

- La fonction **&lt;Tableau de bord&gt;.ConfigurationInitiale** n'est pas prise en compte dans les cas suivants : 

	- L'option "Mémoriser la configuration des Widgets" est cochée dans la fenêtre de description du champ Tableau de bord (onglet "Détail") et l'utilisateur rend le Widget invisible. 

	- L'utilisateur a rendu le Widget invisible et la configuration du champ Tableau de bord a été sauvegardée puis rechargée.

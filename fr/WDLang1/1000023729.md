


## &lt;Tableur&gt;.RecalculeTout (Fonction)

***En anglais : &lt;Spreadsheet&gt;.RecalculateAll***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Recalcule toutes les formules d'un champ Tableur présent dans une fenêtre (y compris les formules utilisant des fonctions/procédures WLangage).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
HLitSuivant(FicChiffre)
TBLR_Tableur.RecalculeTout()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Champ Tableur>.RecalculeTout()`
---

**`<Champ Tableur> : (Nom de champ)`**

Nom du champ Tableur à manipuler.

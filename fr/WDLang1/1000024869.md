
## TCDComparePériode (Fonction)

***En anglais : PVTCompareDateRange***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Calcule et affiche une comparaison de périodes dans un champ Tableau croisé dynamique. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Comparer les 1er trimestres 2018 à 2020
sPériode est une chaîne 
POUR nAnnée = 2018 A 2020
	// du 1er janvier (0101) au 31 mars (0331)
	sPériode += [RC] + "" + nAnnée + "0101" + TAB + nAnnée + "0331"
FIN
TCDComparePériode(TCD_Ventes, sPériode, tcdEvolutionAbsolue + tcdEvolutionPourcentage)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`TCDComparePériode(<Champ Tableau croisé dynamique> , <Période> , <Mode d'affichage>)`
---

**`<Champ Tableau croisé dynamique> : (Nom de champ)`**

Nom du champ Tableau croisé dynamique à manipuler.

**`<Période> : (Chaîne de caractères)`**

Périodes à comparer. Ce paramètre est de la forme : 

```txt
<Date Min Période 1> + TAB + <Date Max Période 1> + RC + ...
<Date Min Période N> + TAB + <Date Max Période N>
```
 où : 

- &lt;Date Min Période&gt; correspond à la date minimale de la période, 

- &lt;Date Max Période&gt; correspond à la date maximale de la période. 


L'ordre des périodes doit être croissant ou décroissant comme le sens de la dimension des dates.

**`<Mode d'affichage> : (Constante de type Entier)`**

Indique si l'évolution doit être affichée en pourcentage et/ou en valeur absolue :


|   |   |
| --- | --- |
| *tcdEvolutionAbsolue* | Affiche l'évolution entre les périodes en valeur absolue. <br><br>Cette constante peut être combinée avec la constante *tcdEvolutionPourcentage*. |
| *tcdEvolutionPourcentage* | Affiche l'évolution entre les périodes en pourcentage. <br><br>Cette constante peut être combinée avec la constante *tcdEvolutionAbsolue*. |
| *tcdSansEvolution*<br>(Valeur par défaut) | N'affiche pas l'évolution entre les périodes. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Cette fonction permet d'effectuer une comparaison de périodes. Elle est équivalente à la Fonctionnalité Automatique de vos Applications (FAA) "Comparaison de périodes" disponible dans le menu contextuel du champ Tableau croisé dynamique. Pour plus de détails, consultez [Fonctionnalités automatiques des champs Tableau croisé dynamique](../WDChamp/1000029008.md). 

- Le champ Tableau croisé dynamique ne doit pas être vide sinon la fonction est sans effet.

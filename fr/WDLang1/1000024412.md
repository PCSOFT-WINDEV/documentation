


## &lt;Lecteur PDF&gt;.TournePage (Fonction)

***En anglais : &lt;PDF Reader&gt;.RotatePage***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Effectue une rotation d'une page affichée dans le champ Lecteur PDF. 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Tourne la page 4 du PDF affiché dans le champ PDF_Lecteur
PDF_Lecteur.TournePage(4, rotationHoraire)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Champ Lecteur PDF>.TournePage(<Page> , <Sens>)`
---

**`<Champ Lecteur PDF> : (Nom de champ)`**

Nom du champ Lecteur PDF à manipuler.

**`<Page> : (Entier)`**

Numéro de la page à tourner.

**`<Sens> : (Constante de type Entier)`**

Sens de rotation de la page : 


|   |   |
| --- | --- |
| *rotationAntihoraire* | Effectue une rotation à -90 degrés de la page. |
| *rotationHoraire* | Effectue une rotation à 90 degrés de la page. |

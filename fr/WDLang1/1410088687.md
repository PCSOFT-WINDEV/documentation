


## &lt;Éditeur de diagrammes&gt;.Sauve (Fonction)

***En anglais : &lt;Diagram editor&gt;.Save***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Sauve un diagramme dans un fichier "diag".
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
EDIAG_MonDiagramme.Sauve("diagramme1.wddiag")
SI ErreurDétectée ALORS
	Erreur("Fichier non sauvegardé : " + ErreurInfo())
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Editeur de diagrammes>.Sauve(<Fichier>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la sauvegarde a été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. La fonction [ErreurInfo](../WDLang1/3013008.md) permet d'obtenir plus de détails sur l'erreur.




**`<Editeur de diagrammes> : (Nom de champ)`**

Nom du champ Editeur de diagrammes à manipuler. 

**`<Fichier> : (Chaîne de caractères)`**

Chemin complet du fichier "diag".

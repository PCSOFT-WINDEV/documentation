
## sysNotificationEntrée (Type de variable)

***En anglais : sysNotificationInput***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **sysNotificationEntrée** permet de définir toutes les caractéristiques avancées d'un champ d'entrée utilisateur (saisie de texte ou combo) dans une notification interactive. Les caractéristiques de ce champ peuvent être définies et modifiées à l'aide de différentes propriétés WLangage. 

Le type **sysNotificationEntrée** est utilisé par les variables de type [sysNotification](../WDLang1/1410087893.md). 

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Création d'une notification permettant de rentrer un nom, un prénom et un service
MaNotif est une sysNotification
MaNotif.Titre = "s'inscrire"

MaNotif.Entrées[1].Type = sysNotifEntréeTexte
MaNotif.Entrées[1].Nom = "Nom_Utilisateur"
MaNotif.Entrées[1].Libellé = "Nom"

MaNotif.Entrées[2].Type = sysNotifEntréeTexte
MaNotif.Entrées[2].Nom = "Prénom_Utilisateur"
MaNotif.Entrées[2].Libellé = "Prénom"

MaNotif.Entrées[3].Type = sysNotifEntréeCombo
MaNotif.Entrées[3].Nom = "Service_Utilisateur"

MaNotif.Entrées[3].ElémentsCombo = ["1" + TAB + "Développement", ...
				"2" + TAB + "Direction", ...
				"3" + TAB + "Traduction"]
								
MaNotif.Boutons[1].Libellé = "S'inscrire"
MaNotif.Boutons[1].Note = "btn_inscrire"

MaNotif.Boutons[2].Libellé = "Décliner"
MaNotif.Boutons[2].Type = sysNotifBoutonFerme

SysNotificationEnvoie(MaNotif)
```





<a name="NOTE0"></a>

## Remarques
<a name="NOTE0_1"></a>


### Propriétés spécifiques à la description des variables de type sysNotificationEntrée
<a name="proprietes_specifiques_description_des_variables_type_sysnotificationentree_ELTPARAGRAPHE000034"></a>Les propriétés suivantes peuvent être utilisées pour manipuler un champ d'entrée utilisateur :

| Nom de la propriété | Type manipulé | Effet |
| --- | --- | --- |
| ElémentsCombo | Tableau de chaînes de caractères | Eléments du champ combo. Chaque chaîne dans le tableau doit être de la forme :<br><br><pre><code>Identifiant + TAB + Libellé</code></pre><br> où : <br><br>- Identifiant : valeur récupérée par l'application.<br><br>- Libellé : libellé de l'élément dans la combo.<br><br><br><br><br>Cette propriété est nécessaire uniquement si la propriété **Type** correspond à la constante *sysNotifEntréeCombo*. |
| IndiceParDéfaut | Entier | Numéro de la ligne affichée par défaut dans le champ Combo. <br><br>Cette propriété est nécessaire uniquement si la propriété **Type** correspond à la constante *sysNotifEntréeCombo*. |
| Libellé | Chaîne de caractères | Libellé du champ.<br><br>Cette propriété est nécessaire uniquement si la propriété **Type** correspond à la constante *sysNotifEntréeTexte*. |
| Nom | Chaîne de caractères | Nom récupéré par l'application pour déterminer le contenu du champ. |
| Type | Constante de type entier | Type du champ permettant une interaction avec l'utilisateur :<br><br>- *sysNotifEntréeCombo* : champ Combo.<br><br>- *sysNotifEntréeTexte* : champ de saisie texte.<br><br><br> |

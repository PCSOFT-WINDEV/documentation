
## FenRectangleEcran (Fonction)

***En anglais : WinScreenRectangle***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie les coordonnées de l'écran qui contient une fenêtre. Il est ensuite possible de gérer les positions ou les tailles des fenêtres dans un environnement mono ou multi écrans. Il est par exemple possible de savoir si une fenêtre dépasse sur un des côtés de l'écran ou si la fenêtre est centrée dans l'écran.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple
<a class="notetitle" target="_blank" href="$DOC$=1000020350&name=fenrectangleecran_fonction&product=WD">Voir des exemples supplémentaires</a>

```wl
sRect est une chaîne = FenRectangleEcran(FEN_Fenêtre1, écranSansBarreDesTâches)
nYMax est un entier = ExtraîtChaîne(sRect, 4)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = FenRectangleEcran([<Fenêtre manipulée> [, <Options>]])`
---

**`<Résultat> : (Chaîne de caractères)`**

Dimensions de l'écran où est affichée la fenêtre. Ces coordonnées sont renvoyées sous la forme suivante : 

```txt
<X1> + TAB + <Y1> + TAB + <X2> + TAB + <Y2>
```


- Les coordonnées (X1, Y1) correspondent au point haut gauche de l'écran.

- Les coordonnées (X2, Y2) correspondent au point bas droite de l'écran.




Si 2 écrans sont utilisés et si la fenêtre est affichée sur le second écran, les coordonnées (X1, Y1) correspondent aux coordonnées du deuxième écran (par exemple 1280,0).

**`<Fenêtre manipulée> : (Nom de fenêtre optionnel)`**

Nom ou alias de la fenêtre à manipuler. 
Si ce paramètre n'est pas précisé ou correspond à une chaîne vide (""), la fenêtre appelante est utilisée. 
Si aucune fenêtre n'est ouverte, l'écran principal est utilisé.

**`<Options> : (Constante de type Entier optionnel)`**

Éléments pris en compte pour la taille de l'écran :


|   |   |
| --- | --- |
| *écranSansBarreDesTâches* | Si la barre des tâches est active sur l'écran, sa taille est retranchée. |
| *écranTout* <br>(Valeur par défaut) | Taille totale de l'écran. |

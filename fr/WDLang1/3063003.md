
## Constantes de gestion du presse-papiers
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les constantes utilisées par les fonctions de gestion du presse-papiers sont les suivantes : 

| Constante | Description | Détail | Utilisée par la fonction |
| --- | --- | --- | --- |
| **ppfBitmap** | Format contenu dans le presse-papiers | Format image | [PressePapierFormat](../WDLang1/3063002.md) |
| **ppfRtf** | Format contenu dans le presse-papiers | Format RTF | [PressePapierFormat](../WDLang1/3063002.md) |
| **ppfTexte** | Format contenu dans le presse-papiers | Format texte | [PressePapierFormat](../WDLang1/3063002.md) |
| **ppfTexteUnicode** | Format contenu dans le presse-papiers | Format texte Unicode | [PressePapierFormat](../WDLang1/3063002.md) |

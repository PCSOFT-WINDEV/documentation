


## ListeSupprimeTout (Fonction)

***En anglais : ListDeleteAll***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Vide de tous ses éléments un champ Liste par programmation (y compris Liste image) ou un champ Combo par programmation.

**Remarque** : La fonction **ListeSupprimeTout** peut être utilisée indifféremment sur les champs Liste monosélection et multisélection.











<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Suppression de tous les éléments présents dans le champ "LISTE_ListeClient"
ListeSupprimeTout(LISTE_ListeClient)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`ListeSupprimeTout(<Champ Liste>)`
---

**`<Champ Liste> : (Nom de champ)`**

Nom du champ Liste (ou Combo) à manipuler. Si ce paramètre correspond à une chaîne vide (""), tous les éléments du champ Liste (ou Combo) auquel appartient l'événement en cours sont effacés.





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
<a name="NOTE0_2"></a>


### Divers
<a name="divers_ELTPARAGRAPHE000136"></a>

- Pour supprimer l'ensemble des lignes d'un champ Table, utilisez la fonction [TableSupprimeTout](../WDLang1/3074034.md).

- La fonction **ListeSupprimeTout** peut être utilisée sur une colonne d'un champ Table de type "Combo".

- Dans le cas des champs Combo avec saisie, la zone de saisie est vidée.

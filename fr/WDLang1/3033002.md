


## Gérer l'Euro
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000090"></a>
Depuis le 1er janvier 2002, l'Euro est la monnaie unique dans la majorité des pays européens. Depuis le 28 février 2002, le franc (comme toutes les autres monnaies Européennes) n'existe plus. 

WINDEV, WINDEV Mobile et WEBDEV permettent de prendre en compte cette nouvelle monnaie très simplement dans toutes vos applications.

<a name="NOTE2_1"></a>


## Comment gérer l'Euro dans une application ou un site ?
<a name="comment_gerer_euro_dans_une_application_site_ELTTEXTE000114"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 

### Les outils à la disposition de l'utilisateur
<a name="les_outils_disposition_utilisateur_ELTPARAGRAPHE000022"></a>Quelques exemples de gestion de l'Euro :

- L'utilisateur peut simplement choisir la monnaie dans laquelle il va saisir son prix grâce à un **menu contextuel** présentant la valeur saisie dans toutes les devises associées à l'Euro. 

- Grâce au **double affichage**, la somme saisie par l'utilisateur est automatiquement convertie et affichée dans un champ "Suiveur". Ainsi, une somme saisie en Francs est automatiquement convertie et affichée en Euro dans un second champ de saisie. 

- Une **bulle d'aide** affiche automatiquement la conversion en francs de la somme saisie en Euro. 



<a name="NOTE2_2"></a>


### Les solutions pour le développeur
<a name="les_solutions_pour_developpeur_ELTPARAGRAPHE000033"></a>Plusieurs outils permettent de simplifier la gestion de l'Euro dans vos applications :

- Un type de champ spécifique : le type "Monétaire + Euro". 
	Ce type de champ est utilisable aussi bien dans les fenêtres (champs de saisie, colonnes de table) que dans les états (champ Libellé, champ Calculé). Ce type de champ permet de gérer bulle d'aide, double affichage et menu contextuel.

- Des propriétés du WLangage utilisables sur les champs de type "Monétaire + Euro".

- Des fonctions du WLangage permettant de gérer les différentes devises et les conversions.

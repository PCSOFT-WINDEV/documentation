
## ExécuteActionPrédéfinie (Fonction)

***En anglais : ExecutePresetAction***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Exécute une action prédéfinie de WINDEV.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) 
```wl
// Ouvrir un explorateur sur le répertoire "C:\Mes documents\"
ExécuteActionPrédéfinie(actExplorateurSurRépertoire, "C:\Mes documents\")
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`ExécuteActionPrédéfinie(<Action> [, <Paramètre 1> [... [, <Paramètre N>]]])`
---

**`<Action> : (Constante de type entier)`**

Action prédéfinie à exécuter. Les constantes disponibles sont les suivantes :

| Constante | Action | Paramètres |
| --- | --- | --- |
| *actColler* | Coller le contenu du presse-papiers | &lt;aucun&gt; |
| *actCopier* | Copier le contenu sélectionné dans le presse-papiers | &lt;aucun&gt; |
| *actCouper* | Couper le contenu sélectionné dans le presse-papiers | &lt;aucun&gt; |
| *actEffacerUnDisqueRW* | Effacer le contenu d'un répertoire sur un CD-RW/DVD-RW<br><br>![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Constante non disponible | Constante permettant de spécifier le type d'effacement : <br><br>- *graveurEffacementComplet* : effacement complet du support (peut prendre un certain temps). <br><br>- *graveurEffacementRapide* : effacement rapide du support. <br><br><br> |
| *actEnvoieMail* | Envoyer un email | 1: Serveur SMTP<br>2: Login<br>3: Mot de passe<br>4: Expéditeur<br>5: Destinataire<br>6: Sujet<br>7: Contenu<br>8: Numéro de port |
| *actExplorateurSurRépertoire* | Ouvrir un explorateur sur un répertoire<br><br>![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Constante non disponible. | 1: Chemin du répertoire |
| *actGraverUnRépertoire* | Graver le contenu d'u répertoire sur un CD-R/DVD-R<br><br>![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Constante non disponible. | 1: Chemin du répertoire |
| *actIconiser* | Iconiser la fenêtre en cours | &lt;aucun&gt; |
| *actImageFenVersPressePapier* | Copier l'image de la fenêtre en cours dans le presse-papiers | &lt;aucun&gt; |
| *actLanceAideContextuelle* | Passer en mode "Aide contextuelle"<br><br>![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Constante non disponible. | &lt;aucun&gt; |
| *actLanceSiteInternet* | Ouvrir un navigateur Internet sur une URL<br><br>![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Constante non disponible. | 1: URL à afficher |
| *actLitDernier* | Lire le dernier enregistrement (du fichier de données actuellement manipulé). | &lt;aucun&gt; |
| *actLitPrécédent* | Lire l'enregistrement précédent (du fichier de données actuellement manipulé). | &lt;aucun&gt; |
| *actLitPremier* | Lire le premier enregistrement (du fichier de données actuellement manipulé). | &lt;aucun&gt; |
| *actLitSuivant* | Lire l'enregistrement suivant (du fichier de données actuellement manipulé). | &lt;aucun&gt; |
| *actOuvreEtat* | Ouvrir l'aperçu avant impression d'un état<br><br>![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Constante non disponible. | 1: Nom de l'état |




**`<Paramètre 1> : (Chaîne de caractères ou entier (optionnel))`**

Premier paramètre associé à l'action prédéfinie.



**`<Paramètre N> : (Chaîne de caractères ou entier (optionnel))`**

Nième paramètre associé à l'action prédéfinie.

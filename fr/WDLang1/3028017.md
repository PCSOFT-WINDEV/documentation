
## DDERécupère (Fonction)

***En anglais : DDERetrieve***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Récupère une donnée envoyée par un programme (le destinataire de la connexion pour l'objet spécifié).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple
<a class="notetitle" target="_blank" href="$DOC$=1000003028017&name=dderecupere_fonction&product=WD">Voir des exemples supplémentaires</a>

```wl
DDEExécute(NumConn, "EXE1")
SI DDERécupère(NumDDE, "Item1") ALORS
	Info("La donnée récupérée est : " + DDEChaîne())
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = DDERécupère(<Identifiant connexion> , <Objet> [, <Temporisation>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si les informations ont été récupérées, 

- <u><u><u><u>Faux</u></u></u></u> en cas d'erreur (la fonction [DDEErreur](../WDLang1/3028006.md) permet de connaître le compte-rendu de l'erreur).




**`<Identifiant connexion> : (Entier)`**

Identifiant de la connexion renvoyé par la fonction [DDEConnecte](../WDLang1/3028015.md).

**`<Objet> : (Chaîne de caractères)`**

Objet de la récupération.

**`<Temporisation> : (Entier optionnel ou Durée optionnelle)`**

Durée maximale d'attente (en secondes). Si l'ordre n'a pas pu être exécuté :

- Si &lt;Temporisation&gt; est précisé, il y a une attente de &lt;Temporisation&gt; secondes avant de quitter la fonction **DDERécupère**.

- Si &lt;Temporisation&gt; n'est pas précisé, il y a une attente infinie. 


Remarque : Ce paramètre peut correspondre à : 

- un entier correspondant au nombre de secondes, 

- une variable de type Durée, 

- la durée en clair (par exemple '1s').

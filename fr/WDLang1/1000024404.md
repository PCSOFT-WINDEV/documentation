


## &lt;Panneau dockable&gt;.Attache (Fonction)

***En anglais : &lt;Dockable panel&gt;.Attach***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Attache ou change la position d'attache d'un champ Panneau dockable.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Crée un configuration avec 2 panneaux en bas, l'un à coté de l'autre
PADO_Gauche.Attache(padoGauche)
PADO_Droite.Attache(padoHaut + padoDroite)


// Crée un configuration avec 2 panneaux en bas dans la même zone 
PADO_Detail.Attache(padoDroite)
PADO_Infos.Attache(padoDroite)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Champ Panneau dockable>.Attache([<Champ Panneau de référence>] [, <Position>])`
---

**`<Champ Panneau dockable> : (Nom de champ)`**

Nom du champ Panneau dockable à manipuler.

**`<Champ Panneau de référence> : (Nom de champ optionnel)`**

Nom du champ Panneau dockable à utiliser comme référence pour la position. Si ce paramètre est spécifié : 

- la position indiquée avec le paramètre &lt;Position&gt; sera relative à ce champ.  

- si le paramètre &lt;Position&gt; n'est pas précisé, &lt;Champ Panneau dockable&gt; sera positionné dans le même onglet que &lt;Champ Panneau de référence&gt;.




**`<Position> : (Constante ou combinaison de constantes optionnelle de type Chaîne de caractères)`**

Décrit la position où le champ Panneau dockable doit être attaché : 


|   |   |
| --- | --- |
| *padoBas* | Attache le champ Panneau dockable en bas. |
| *padoDroite* | Attache le champ Panneau dockable à droite. |
| *padoGauche* | Attache le champ Panneau dockable à gauche. |
| *padoHaut* | Attache le champ Panneau dockable en haut. |


Les constantes sont combinables 2 à 2, une constante donnant la position Gauche/Droite avec une constante donnant la position Bas/Haut. 
Si ce paramètre n'est pas précisé et si le champ Panneau dockable était flottant, il est attaché à sa dernière position connue. 
Si la position est déjà occupée, le champ Panneau dockable est ajouté dans la zone est un volet est automatiquement créé et affiché.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- La position du champ Panneau dockable peut être connue avec la fonction [&lt;Panneau&gt;.Position](../WDLang1/1000024408.md). 

- La fonction **&lt;Panneau dockable&gt;.Attache** provoque une erreur non fatale si la position est invalide.

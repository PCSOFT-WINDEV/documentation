
## diagTexte (Type de variable)

***En anglais : diagText***
			
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **diagTexte** permet de définir toutes les caractéristiques avancées d'un texte associé à une forme dans un diagramme. Les caractéristiques de ce texte peuvent être définies et modifiées à l'aide de différentes propriétés WLangage.

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).

ATTENTION : Ce type correspond à la propriété **Texte** d'une variable de type [diagForme](../WDLang1/1410088082.md) ou de type [diagSélection](../WDLang1/1410088172.md). 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Crée une forme rectangle
MaForme est un diagRectangle

// Définit les dimensions, la position et la couleur de fond
MaForme.X = 100
MaForme.Y = 100
MaForme.Largeur = 350
MaForme.Hauteur = 150
MaForme.Fond.Couleur = RougePastel
// Texte 
MaForme.Texte.Libellé = "Rectangle"
MaForme.Texte.CadrageHorizontal = chCentre
MaForme.Texte.CadrageVertical = cvMilieu
MaForme.Texte.Couleur = JauneOr
MaForme.Texte.Police.Gras = Vrai

// Obtient une référence sur le diagramme affiché
MonDiagramme est un Diagramme <- EDIAG_MonDiagramme

// Ajoute la forme rectangle dans le diagramme
Ajoute(MonDiagramme.Forme, MaForme)
```





<a name="PROPS"></a>

## Propriétés


### Propriétés spécifiques à la description des variables de type diagTexte 
<a name="proprietes_specifiques_description_des_variables_type_diagtexte_ELTPARAGRAPHE000035"></a>Les propriétés suivantes peuvent être utilisées pour manipuler le texte d'une forme :

| Nom de la propriété | Type manipulé | Effet |
| --- | --- | --- |
| CadrageHorizontal | Constante de type Entier | Cadrage horizontal du texte : <br><br>- *chCentre* : Le texte est centré.<br><br>- *chDroite* : Le texte est aligné à droite.<br><br>- *chGauche* : Le texte est aligné à gauche.<br><br><br> |
| CadrageVertical | Constante de type Entier | Cadrage vertical du texte : <br><br>- *cvBas* : Le texte est placé en bas de la forme. <br><br>- *cvHaut* : Le texte est placé en haut de la forme. <br><br>- *cvMilieu* : Le texte est placé au milieu de la forme. <br><br><br> |
| Couleur | [Couleur](../WDLang1/1000019661.md) | Couleur du texte. |
| CouleurFond | [Couleur](../WDLang1/1000019661.md) | Couleur affichée sous le texte. |
| Libellé | Chaîne de caractères | Texte affiché.<br><br>Remarque : Cette propriété ne doit pas être utilisée en même temps que la propriété **LibelléHTML**. |
| LibelléHTML | Chaîne de caractères | Texte affiché au format HTML. Ce texte peut contenir des balises de mise en forme HTML. <br><br>Remarque : Cette propriété ne doit pas être utilisée en même temps que la propriété **Libellé**. |
| Marge | Réel | Marge autour du texte exprimée en pixels. |
| Police | [Police](../Motscles/1514045.md) | Caractéristiques de la police : gras, italique, ... |

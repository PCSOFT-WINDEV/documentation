
## MCIMsgErr (Fonction)

***En anglais : MCIMsgErr***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le libellé de la dernière erreur survenue sur une fonction MCI (Media Control Interface).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Libellé de la dernière erreur
ResMsgErreur = MCIMsgErr()
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = MCIMsgErr()`
---

**`<Résultat> : (Chaîne de caractères)`**

Libellé de la dernière erreur survenue sur une fonction MCI.

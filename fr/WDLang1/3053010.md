***En anglais : MemCount***

<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le nombre d'éléments présents dans une zone de mémoire.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Nombre d'éléments présents dans la zone mémoire "MaZoneMémoire"
ResOccurrence = MemOccurrence("MaZoneMémoire")
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = MemOccurrence(<Zone mémoire>)`
---

**`<Résultat> : (Entier)`**

Nombre d'éléments présents dans la zone mémoire spécifiée.

**`<Zone mémoire> : (Chaîne de caractères)`**

Nom de la zone mémoire, défini par la fonction [MemCrée](../WDLang1/3053005.md).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Existence de la zone mémoire
<a name="existence_zone_memoire_ELTPARAGRAPHE000201"></a>L'existence de la zone mémoire n'est pas vérifiée.

Si la zone mémoire spécifiée n'existe pas, une erreur du WLangage est générée lors de l'exécution de la fonction **MemOccurrence**. 

Pour vérifier l'existence d'une zone mémoire, utilisez la fonction [MemExiste](../WDLang1/3053006.md).

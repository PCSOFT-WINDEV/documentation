
## Constantes de gestion des projets
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


Les constantes de gestion des projets sont les suivantes :

| Constante | Description | Utilisée par les fonctions |
| --- | --- | --- |
| **enumBoiteOutils** | Énumère les volets d'une boîte à outils. | [EnumèreSousElément](../WDLang1/3064002.md) |
| **enumComposantExterne** | Énumère les éléments publics d'un composant externe intégré au projet. | [EnumèreElément](../WDLang1/3064001.md) |
| **enumEtat** | Énumère tous les états du projet. | [EnumèreElément](../WDLang1/3064001.md) |
| **enumFenêtre** | Énumère toutes les fenêtres du projet. | [EnumèreElément](../WDLang1/3064001.md) |
| **enumFenêtreInterne** | Énumère toutes les fenêtres internes du projet. | [EnumèreElément](../WDLang1/3064001.md) |
| **enumGPU** | Constante réservée pour le Groupware utilisateur. | [EnumèreElément](../WDLang1/3064001.md) |
| **enumGroupe** | Énumère les groupes d'une fenêtre. | [EnumèreSousElément](../WDLang1/3064002.md) |
| **enumMenuContextuel** | Énumère les menus contextuels d'une fenêtre. | [EnumèreSousElément](../WDLang1/3064002.md) |
| **enumMenuPrincipal** | Énumère les menus déroulant d'une fenêtre. | [EnumèreSousElément](../WDLang1/3064002.md) |
| **enumModèleEtat** | Enumère les modèles d'états du projet. | [EnumèreElément](../WDLang1/3064001.md) |
| **enumModèleFenêtre** | Enumère les modèles de fenêtres du projet. | [EnumèreElément](../WDLang1/3064001.md) |
| **enumModèlePage** | Enumère les modèles de pages du projet. | [EnumèreElément](../WDLang1/3064001.md) |
| **enumPage** | Énumère toutes les pages du projet. | [EnumèreElément](../WDLang1/3064001.md) |
| **enumPageDynamique** | Énumère toutes les pages dynamiques du projet. | [EnumèreElément](../WDLang1/3064001.md) |
| **enumPageInterne** | Énumère toutes les pages internes du projet. | [EnumèreElément](../WDLang1/3064001.md) |
| **enumPremier** | Initialise l'énumération. | [EnumèreSousElément](../WDLang1/3064002.md) |
| **enumRequête** | Énumère toutes les requêtes du projet. | [EnumèreElément](../WDLang1/3064001.md) |
| **enumTout** | Énumère tous les éléments du projet. | [EnumèreElément](../WDLang1/3064001.md) |
| **enumVolet** | Énumère les volets des onglets. | [EnumèreSousElément](../WDLang1/3064002.md) |
| **piAdresseInternet** | Type d'information à renvoyer : Adresse Internet définie lors de la création de l'exécutable, de la bibliothèque ou du composant. | [ProjetInfo](../WDLang1/3064004.md) |
| **piAdresseInternetApplication** | Type d'information à renvoyer : Adresse Internet définie lors de la création de l'exécutable, de la bibliothèque ou du composant. | [ProjetInfo](../WDLang1/3064004.md) |
| **piAnalyse** | Type d'information à renvoyer : Nom physique sans extension du fichier .WDD utilisé par le projet ou le composant. | [ProjetInfo](../WDLang1/3064004.md) |
| **piCheminDonnéesCC** | Type d'information à renvoyer : Chemin du répertoire de données des Centres de Contrôle défini lors de la création de l'exécutable, de la bibliothèque ou du composant.<br>Si cette constante est utilisée dans le projet, donne le chemin du répertoire des données correspondant au projet. Si cette constante est utilisée dans un composant, donne le chemin du répertoire des données correspondant au composant. | [ProjetInfo](../WDLang1/3064004.md) |
| **piCheminDonnéesCCApplication** | Type d'information à renvoyer : Chemin du répertoire de données des Centres de Contrôle défini lors de la création de l'exécutable, de la bibliothèque ou du composant.<br>Si cette constante est utilisée dans un composant, permet de connaître le chemin du répertoire des données de l'application qui l'utilise. | [ProjetInfo](../WDLang1/3064004.md) |
| **piConfiguration** | Type d'information à renvoyer : Nom de la configuration de projet qui a permis de générer l'exécutable ou le composant en cours d'exécution. | [ProjetInfo](../WDLang1/3064004.md) |
| **piCopyright** | Type d'information à renvoyer : Copyright défini lors de la création de l'exécutable. | [ProjetInfo](../WDLang1/3064004.md) |
| **piDateGénération** | Type d'information à renvoyer : Date de génération de la bibliothèque (WDL) de l'application ou du composant. | [ProjetInfo](../WDLang1/3064004.md) |
| **piDescription** | Type d'information à renvoyer : Description de la société définie lors de la création de l'exécutable. | [ProjetInfo](../WDLang1/3064004.md) |
| **piEmail** | Type d'information à renvoyer : Email défini lors de la création de l'exécutable, de la bibliothèque ou du composant. | [ProjetInfo](../WDLang1/3064004.md) |
| **piEmailApplication** | Type d'information à renvoyer : Email défini lors de la création de l'exécutable, de la bibliothèque ou du composant. | [ProjetInfo](../WDLang1/3064004.md) |
| **piNomEXE** | Type d'information à renvoyer : Nom de l'exécutable spécifié lors de la création de l'exécutable. | [ProjetInfo](../WDLang1/3064004.md) |
| **piNomProjet** | Type d'information à renvoyer : Nom du projet. | [ProjetInfo](../WDLang1/3064004.md) |
| **piNomProjetCourant** | Type d'information à renvoyer : Nom du projet dans lequel est appelée la fonction ProjetInfo.<br>Si cette constante est utilisée dans le code du projet principal, le résultat correspond au nom du projet principal.<br>Si cette constante est utilisée dans le code d'un composant, le résultat correspond au nom du projet du composant. | [ProjetInfo](../WDLang1/3064004.md) |
| **piNomSiteDéployé** | Type d'information à renvoyer : Nom réel du site déployé (cas d'un site WEBDEV lors d'un déploiement multi-site). | [ProjetInfo](../WDLang1/3064004.md) |
| **piNouveautésContexteAide** | Type d'information à renvoyer : Numéro de contexte de la page à afficher dans le fichier d'aide des nouveautés. | [ProjetInfo](../WDLang1/3064004.md) |
| **piNouveautésNomAide** | Type d'information à renvoyer : Nom du fichier d'aide des nouveautés. | [ProjetInfo](../WDLang1/3064004.md) |
| **piRegistre** | Type d'information à renvoyer : Chemin dans la base de registre dans lequel il est possible de sauver des informations spécifiques au projet. | [ProjetInfo](../WDLang1/3064004.md) |
| **piSociété** | Type d'information à renvoyer : Nom de la société spécifié lors de la création de l'exécutable. | [ProjetInfo](../WDLang1/3064004.md) |


## TableauVersCSV (Fonction)

***En anglais : ArrayToCSV***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Convertit un tableau à 1 ou 2 dimensions en chaîne de caractères au format CSV.

**Remarque** : Cette fonction est utilisable uniquement sur les [tableaux WLangage](../Motscles/1514030.md).




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Tableau à une dimension
sMesValeurs est une chaîne 
// Tableau initial
MonTableau est un tableau de 3 chaînes
MonTableau[1] = "A"
MonTableau[2] = "B"
MonTableau[3] = "C"
sMesValeurs = TableauVersCSV(MonTableau)
// sMesValeurs contient : "A" + RC + "B" + RC + "C"
```


<a name="Exemple2"></a>



```wl
// Tableau à deux dimensions
sMesValeurs est une chaîne 
// Tableau initial
MonTableau est un tableau de 2 par 3 chaînes
MonTableau[1,1] = "A"; MonTableau[1,2] = "B"
MonTableau[1,3] = "C"; MonTableau[2,1] = "D"
MonTableau[2,2] = "E"; MonTableau[2,3] = "F"

sMesValeurs = TableauVersCSV(MonTableau)
// sMesValeurs contient : "A;B;C"+RC+"D;E;F"
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = TableauVersCSV(<Tableau WLangage> [, <Séparateur de colonne>])`
---

**`<Résultat> : (Chaîne de caractères)`**

Chaîne de caractères contenant les différentes valeurs du tableau.

**Pour un tableau à une dimension**, les éléments du tableau sont convertis en chaîne de caractères, puis ajoutés dans la chaîne en étant séparés par RC (Retour Chariot).

**Pour un tableau à 2 dimensions**, les éléments du tableau sont convertis en chaîne de caractères, puis ajoutés dans la chaîne. Les lignes sont séparées par RC (Retour Chariot) et les colonnes sont séparées par &lt;Séparateur de colonne&gt;.

**`<Tableau WLangage> : (Variable de type Tableau)`**

Nom du tableau à une ou 2 dimensions à convertir en chaîne de caractères. Ce tableau doit être alloué. 

**`<Séparateur de colonne> : (Chaîne de caractères optionnelle)`**

Séparateur utilisé pour séparer les valeurs des différentes colonnes. Par défaut, le séparateur de colonnes utilisé est ";".



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Le séparateur utilisé dans le &lt;Résultat&gt; pour séparer les valeurs des différentes lignes est RC (Retour Chariot).

- Les tableaux de type structure gérés sont uniquement des tableaux à une dimension.

- Les tableaux contenant des chaînes UNICODE ou des variants ne sont pas gérés.

- Pour le type Durée, la valeur est convertie en millièmes de seconde.

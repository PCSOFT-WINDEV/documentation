
## dbgFinEnregistrement (Fonction)

***En anglais : dbgEndRecording***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Arrête l'enregistrement du scénario en cours. L'enregistrement a été commencé grâce à la fonction [dbgDébutEnregistrement](../WDLang1/1000017160.md).

**Remarque** : Cette fonction est à utiliser dans le cadre de la [création d'un test automatique par l'utilisateur final](../Editeurs/2019031.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
dbgFinEnregistrement()
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`dbgFinEnregistrement()`
---



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
La fonction **dbgFinEnregistrement** est sans effet si aucun enregistrement de scénario n'est en cours.

**Note** : L'enregistrement du scénario peut être également démarré grâce au fichier ".wx" placé à côté de l'exécutable. Pour plus de détails, consultez [Scénario créé par l'utilisateur](../Editeurs/2019031.md).

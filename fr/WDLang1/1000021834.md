


## &lt;Table&gt;.DérouleTout (Fonction)

***En anglais : &lt;Table&gt;.ExpandAll***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Déroule :

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) toute la hiérarchie (tous les noeuds) d'un champ Table hiérarchique.

- toutes les ruptures d'un champ Table. 




![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Les ruptures ne sont pas disponibles sur les champs Table dans cette version.






<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 
```wl
// Déroule tous les noeuds du champ Table hiérarchique "TABLEH_Planning"
TABLEH_Planning.DérouleTout()
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Champ Table>.DérouleTout()`
---

**`<Champ Table> : (Nom de champ)`**

Nom du champ à manipuler. Ce nom peut correspondre :

- au nom du champ Table hiérarchique pour lequel tous les noeuds doivent être déroulés.

- au nom du champ Table pour lequel toutes les ruptures doivent être déroulées.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Conditions d'utilisation
<a name="conditions_utilisation_ELTPARAGRAPHE000223"></a>La fonction **&lt;Table&gt;.DérouleTout** peut être utilisée sur : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) un champ Table hiérarchique.

- un champ Table par programmation. 

- un champ Table fichier chargé en mémoire.  




Cette fonction ne peut pas être utilisée sur les champs Table fichier (accès direct).
<a name="NOTE0_2"></a>


### Champ Table hiérarchique
<a name="champ_table_hierarchique_ELTPARAGRAPHE000252"></a>Lors de l'appel à la fonction **&lt;Table&gt;.DérouleTout**, si la branche spécifiée et/ou ses fils possèdent une procédure associée par la propriété [AjouteFilsDifféré](../Proprietes/1000020927.md), cette procédure sera exécutée.

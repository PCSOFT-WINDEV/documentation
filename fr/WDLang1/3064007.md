
## ComposantListe (Fonction)

***En anglais : ComponentList***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Retourne la liste des composants externes (fichiers ".WDK") utilisés par l'application WINDEV, WINDEV Mobile ou par le site WEBDEV en cours.

**Remarque** : Tous les composants externes sont listés, qu'ils soient chargés en mémoire ou non.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Récupération des composants utilisés
MaListeComposant est une chaîne = ComposantListe()
// Parcours de cette liste
Composant est une chaîne
Composant = ExtraitChaîne(MaListeComposant, rangPremier, RC)
TANTQUE Composant <> EOT
	// Récupération du composant (noms logique et physique)
	Trace(ExtraitChaîne(Composant, 1, TAB) + " - " + ExtraitChaîne(Composant, 2, TAB))
	// Composant suivant
	Composant = ExtraitChaîne(MaListeComposant, rangSuivant, RC)
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = ComposantListe()`
---

**`<Résultat> : (Chaîne de caractères)`**

Liste des composants externes (fichiers ".DLL") utilisés par l'application en cours, sous la forme :

```txt
<Nom 1° composant> + TAB + <Chemin 1° composant> + RC + 
<Nom 2° composant> + TAB + <Chemin 2° composant> + RC + ... + 
<Nom N° composant> + TAB + <Chemin N° composant>
```


où :

- &lt;Nom X composant&gt; correspond au nom logique du composant externe X.

- &lt;Chemin X composant&gt; correspond au nom et chemin complet du composant externe X.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
La fonction **ComposantListe** ne retourne pas de code d'erreur. Pour savoir si cette fonction a généré une erreur, utilisez la variable [ErreurDétectée](../WDLang1/3087001.md). En cas d'erreur, il est possible de connaître le détail de l'erreur grâce à la fonction [ErreurInfo](../WDLang1/3013008.md).

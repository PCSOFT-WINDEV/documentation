
## TraceFin (Fonction)

***En anglais : TraceEnd***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ferme la fenêtre de trace si cette fenêtre a été ouverte ou arrête l'enregistrement des informations de trace dans un fichier texte. 

**Remarque** : A la fermeture du projet, la fenêtre de trace est automatiquement fermée et l'enregistrement dans le fichier texte est automatiquement arrêté.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Fermeture de la fenêtre de trace
TraceFin()
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`TraceFin()`




## TDBSupprime (Fonction)

***En anglais : DashDelete***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime un Widget affiché dans un champ Tableau de bord. Le champ Tableau de bord est mis à jour et un emplacement vide remplace le Widget supprimé.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Vérifie qu'il existe au moins un widget
SI TDBOccurrence(TDB_TableauDeBord, toTotal) <= 0 ALORS
	Info("Il n'y a aucun widget à supprimer")
	RETOUR
FIN

// Supprime le premier widget
TDBSupprime(TDB_TableauDeBord, 1)
ToastAffiche("Le widget d'indice 1 a été supprimé", toastCourt, cvMilieu, chCentre)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`TDBSupprime(<Champ Tableau de bord> , <Indice>)`
---

**`<Champ Tableau de bord> : (Nom de champ)`**

Nom du champ Tableau de bord à manipuler.

**`<Indice> : (Entier)`**

Indice du Widget à supprimer.

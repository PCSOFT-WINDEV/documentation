
## Fonctions de gestion des champs Agenda (syntaxe préfixée)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>




Les fonctions permettant de manipuler le champ Agenda via une syntaxe préfixée sont : 



|   |   |
| --- | --- |
| [&lt;Agenda&gt;.Affiche](../WDLang1/1000022880.md) | Réinitialise le contenu d'un champ Agenda à partir de sa source de données (fichier de données HFSQL par exemple). |
| [&lt;Agenda&gt;.AffichePériodePrécédente](../WDLang1/1000022878.md) | Déplace la zone temporelle affichée sur la zone précédente (par exemple, semaines précédentes) dans un champ Agenda. |
| [&lt;Agenda&gt;.AffichePériodeSuivante](../WDLang1/1000022896.md) | Déplace la zone temporelle affichée sur la zone suivante (par exemple, semaines suivantes) dans un champ Agenda. |
| [&lt;Agenda&gt;.AjouteRendezVous](../WDLang1/1000022894.md) | Ajoute un nouveau rendez-vous dans un champ Agenda. |
| [&lt;Agenda&gt;.ChangeMode](../WDLang1/1000022879.md) | Modifie la plage temporelle affichée par un champ Agenda (Jour/Semaine/Mois). |
| [&lt;Agenda&gt;.CouleurFond](../WDLang1/1000022895.md) | Fixe une couleur de fond sur une plage horaire d'un champ Agenda. |
| [&lt;Agenda&gt;.DateVisible](../WDLang1/1000022915.md) | Rend visible ou invisible un jour dans un champ Agenda. |
| [&lt;Agenda&gt;.HeureAffichée](../WDLang1/1000022892.md) | Définit la plage horaire affichée par le champ Agenda en mode Jour ou Semaine. |
| [&lt;Agenda&gt;.HeureOuvrable](../WDLang1/1000022916.md) | Permet d'indiquer les heures ouvrables spécifiques par jour dans un champ Agenda. |
| [&lt;Agenda&gt;.InfoXY](../WDLang1/1000022889.md) | Renvoie pour une position donnée dans le champ Agenda (coordonnées d'un point de l'agenda) : <br><br>- soit l'indice du rendez vous pour la position indiquée. <br><br>- soit la date correspondant à la position indiquée.<br><br><br> |
| [&lt;Agenda&gt;.ListeRendezVous](../WDLang1/1000022886.md) | Renvoie : <br><br>- la liste des rendez-vous compris entre deux dates données dans un champ Agenda, <br><br>- tous les rendez-vous d'un champ Agenda,<br><br>- un rendez-vous particulier (sélectionné ou survolé).<br><br><br> |
| [&lt;Agenda&gt;.Mode](../WDLang1/1000022885.md) | Renvoie le mode d'affichage par un champ Agenda : jour, semaine, mois, ... |
| [&lt;Agenda&gt;.PlageSélectionnée](../WDLang1/1000022883.md) | Renvoie la date et l'heure de début ou de fin de la plage horaire sélectionnée dans un champ Agenda. |
| [&lt;Agenda&gt;.Position](../WDLang1/1000022897.md) | Affiche un champ Agenda à partir d'une date spécifiée ou renvoie la date du premier jour (ou dernier jour) affiché dans un champ Agenda. |
| [&lt;Agenda&gt;.SelectMoins](../WDLang1/1000022882.md) | Désélectionne un rendez-vous dans un champ Agenda. |
| [&lt;Agenda&gt;.SelectPlus](../WDLang1/1000022881.md) | Sélectionne un rendez-vous dans un champ Agenda. |
| [&lt;Agenda&gt;.SupprimeRendezVous](../WDLang1/1000022877.md) | Supprime un rendez-vous dans un champ Agenda. |
| [&lt;Agenda&gt;.SupprimeTout](../WDLang1/1000022898.md) | Supprime tous les rendez-vous d'un champ Agenda. |
| [&lt;Agenda&gt;.VersiCalendar](../WDLang1/1000022899.md) | Exporte les rendez-vous d'un champ Agenda au format iCalendar. |

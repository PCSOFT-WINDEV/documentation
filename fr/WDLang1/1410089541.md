
## kbActionAutorisée (Type de variable)

***En anglais : kbActionAllowed***
			
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **kbActionAutorisée** permet de définir toutes autorisations données à l'utilisateur final pour manipuler une carte d'un champ Kanban. Ces autorisations peuvent être définies et modifiées à l'aide de différentes propriétés WLangage. 

Attention : Ce type de variable peut être utilisé uniquement sur les variables de type [kbCarte](../WDLang1/1410089167.md). 

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Interdire de supprimer une tâche
laTache est un KbCarte
laTache.Titre = "IMPORTANT"
laTache.ActionAutorisée.Supprimer = Faux
KanbanAjouteCarte(LISTE_TODO, laTache)
```





<a name="PROPS"></a>

## Propriétés


### Propriétés spécifiques à la description des variables de type kbActionAutorisée :
<a name="proprietes_specifiques_description_des_variables_type_kbactionautorisee_ELTPARAGRAPHE000031"></a>Les propriétés suivantes peuvent être utilisées pour manipuler les actions autorisées sur une carte d'un champ Kanban :

| Nom de la propriété | Type manipulé | Effet |
| --- | --- | --- |
| AfficherDétail | Booléen | <br><br>- <u><u><u>Vrai</u></u></u> pour autoriser l'exécution de l'événement "Affichage du détail de la carte" lors d'un clic ou double-clic sur mobile, <br><br>- <u><u><u>Faux</u></u></u> pour l'interdire. <br><br><br> |
| ChangerListe | Booléen | <br><br>- <u><u><u>Vrai</u></u></u> pour autoriser l'utilisateur final à déplacer la carte d'une liste à l'autre, <br><br>- <u><u><u>Faux</u></u></u> pour l'interdire. <br><br><br> |
| Déplacer | Booléen | <br><br>- <u><u><u>Vrai</u></u></u> pour autoriser l'utilisateur final à déplacer la carte dans la liste,  <br><br>- <u><u><u>Faux</u></u></u> pour l'interdire. <br><br><br> |
| Supprimer | Booléen | <br><br>- <u><u><u>Vrai</u></u></u> pour autoriser l'utilisateur final à supprimer la carte par un Drag and Drop vers le pied de la liste, <br><br>- <u><u><u>Faux</u></u></u> pour l'interdire.  <br><br><br> |




## &lt;Composant&gt;.Exécute (Fonction)

***En anglais : &lt;Component&gt;.Execute***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Exécute une procédure globale présente dans un composant externe. Ce composant externe a été précédemment chargé par la fonction [&lt;Composant&gt;.Charge](../WDLang1/1410089475.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
un_composant_plugin est un Composant
un_composant_plugin.Charge(nom_physique_composant_plugin)
interface_principale_plugin est ...
	IInterfacePrincipale <- un_composant_plugin.Exécute("GetInterfacePrincipale")
```

```wl
MonComposant est un Composant
MonComposant.Charge("C:\MesComposants\MonComposant.wdk")
MonComposant.Exécute("MesProcédures.Hello", ("World"))
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Composant>.Exécute(<Procédure> [, <Paramètre 1> [... [, <Paramètre N>]]])`
---

**`<Résultat> : (Type attendu)`**

Valeur de retour de la procédure.

**`<Composant> : (Variable de type Composant)`**

Nom de la variable de type [Composant](../WDLang1/1410089120.md) correspondant au composant dont une fonction doit être exécutée.

**`<Procédure> : (Chaîne de caractères)`**

Nom de la procédure du composant externe à exécuter.

**`<Paramètre 1> : (Paramètres optionnels (le type correspond au type du paramètre))`**

Paramètre optionnel attendu par la procédure pour son exécution. Ce paramètre doit être spécifié entre parenthèses.



**`<Paramètre N> : (Paramètres optionnels (le type correspond au type du paramètre))`**

Paramètre optionnel attendu par la procédure pour son exécution. Ce paramètre doit être spécifié entre parenthèses.

<a name="NOTE0_1"></a>

Si l’appel échoue, la fonction génère une erreur fatale.

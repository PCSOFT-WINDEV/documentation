
## NombreFenêtre (Fonction)

***En anglais : WindowCount***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Calcule le nombre de fenêtres de l'application en cours actuellement ouvertes.








<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI NombreFenêtre() >= 1 ALORS
	Info("Il y a " + NombreFenêtre() + "fenêtres ouvertes")
SINON
	Info("Aucune fenêtre n'est ouverte")
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = NombreFenêtre()`
---

**`<Résultat> : (Entier)`**



- Nombre de fenêtres actuellement ouvertes, 

- 0 si la fonction est exécutée dans le code d'initialisation de projet.

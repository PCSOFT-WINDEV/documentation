
## TraceSupprimeTout (Fonction)

***En anglais : TraceDeleteAll***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Vide le contenu de la fenêtre de trace ouverte en parallèle de la fenêtre en cours.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Vide le contenu de la fenêtre de trace
TraceSupprimeTout()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`TraceSupprimeTout()`
---



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Fichier de trace
<a name="fichier_trace_ELTPARAGRAPHE000029"></a>La trace peut être effectuée dans un fichier dans les cas suivants : 

- sous Linux, 

- via la fonction [TraceDébut](../WDLang1/3013049.md) avec la constante *trFichier*. 


Dans ce cas, la fonction **TraceSupprimeTout** supprime le fichier de trace.

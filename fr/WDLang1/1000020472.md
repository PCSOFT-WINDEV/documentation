
## PropriétéExiste (Fonction)

***En anglais : PropertyExist***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de savoir si une propriété existe sur un champ ou une instance de classe.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
PROCEDURE ChangeLibellé(NomChamp, NouveauLibellé)

SI PropriétéExiste(NomChamp, "Libellé") ALORS
	NomChamp.Libellé = NouveauLibellé
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = PropriétéExiste(<Elément> , <Propriété>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la propriété existe, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 




**`<Elément> : (Chaîne de caractères ou classe)`**

Nom du champ ou de l'instance de classe à manipuler. 

- Si ce paramètre correspond au nom d'un champ, la propriété est recherchée parmi les propriétés standard du WLangage.

- Si ce paramètre correspond à une variable de type Classe, la propriété est recherchée parmi les propriétés définies pour la classe. 




**`<Propriété> : (Chaîne de caractères)`**

Nom de la propriété dont l'existence doit être testée.

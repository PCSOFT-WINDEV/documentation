


## TableEnumèreRupture (Fonction)

***En anglais : TableEnumBreak***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Enumère les ruptures d'un champ Table.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Liste les ruptures de TABLE_MaTable
sNomRupture est chaîne
bPosition est un booléen
nNumCol est un entier

soit i = 1
sDesc est une chaîne = TableEnumèreRupture(TABLE_MaTable, i)
TANTQUE sDesc <> "" 
	sNomRupture = ExtraitChaîne(sDesc, 1, TAB)
	bPosition = ExtraitChaîne(sDesc, 2, TAB)
	nNumCol = ExtraitChaîne(sDesc, 3, TAB)
   	SI {sNomRupture, indChamp}..Visible ALORS
		Trace(sNomRupture, bPosition, nNumCol)
	FIN
	// Rupture suivante
	i++
	sDesc = TableEnumèreRupture(TABLE_MaTable, i)

FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = TableEnumèreRupture(<Champ Table> , <Indice de la rupture>)`
---

**`<Résultat> : (Chaîne de caractères)`**



- Description de la rupture pour l'indice spécifié. Cette description est de la forme : 
	
	```txt
	<Rupture> + TAB + <Position de la rupture> + TAB + <Numéro de la colonne>
	```
où : 

	- &lt;Rupture&gt; : Nom de la rupture. 

	- &lt;Position de la rupture&gt; : 1 pour un haut de rupture et 0 pour un bas de rupture. 

	- &lt;Numéro de la colonne&gt; : Numéro de la colonne qui provoque la rupture. 




- Chaîne vide ("") si le champ Table ne contient pas de rupture ou si le paramètre &lt;Indice de la rupture&gt; est égal au nombre de ruptures du champ Table + 1.




**`<Champ Table> : (Nom de champ)`**

Nom du champ Table à manipuler.

Si ce paramètre correspond à une chaîne vide (""), le champ manipulé est le champ Table auquel le traitement en cours appartient.

**`<Indice de la rupture> : (Entier)`**

Indice de la rupture recherchée, compris entre 1 et le nombre de ruptures du champ Table + 1.
Si ce paramètre est supérieur au nombre de ruptures du champ Table, la fonction renvoie une chaîne vide ("").

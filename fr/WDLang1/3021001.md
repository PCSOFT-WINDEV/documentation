
## MessageAjouteCase (Fonction)

***En anglais : StatusBarAddCell***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ajoute une nouvelle case dans la barre de message de la fenêtre en cours.

**Remarque** : La fenêtre doit posséder une barre de message.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ajoute la case "MaCase" dans la barre de message de la fenêtre en cours
MessageAjouteCase("MaCase", 100, RougeClair)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`MessageAjouteCase(<Nom de la case> [, <Taille de la case> [, <Couleur de la case>]])`
---

**`<Nom de la case> : (Chaîne de caractères)`**

Nom de la case à créer. Ce nom sera ensuite utilisé en programmation pour manipuler la case (avec les propriétés par exemple). La première case a pour nom "". Si la case existe déjà, sa description sera automatiquement modifiée.

**`<Taille de la case> : (Entier optionnel)`**

Taille de la case en pixels. Si ce paramètre correspond à -1, la case occupera la largeur restante de la barre de message.

**`<Couleur de la case> : (Constante ou entier optionnel)`**

Couleur de fond de la case. Cette couleur peut correspondre à :

- une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)),

- une couleur TSL (obtenue avec la fonction [TSL](../WDLang1/3029057.md)),

- [une couleur prédéfinie du WLangage](../WDLang5/3010002.md).






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Utiliser les propriétés WLangage avec les cases des barres de message
<a name="utiliser_les_proprietes_wlangage_avec_les_cases_des_barres_message_ELTPARAGRAPHE000051"></a>De nombreuses propriétés WLangage peuvent être utilisées sur les cases des barres de message.

Attention : Pour utiliser une propriété sur une case d'une barre de message, la syntaxe à utiliser est la suivante :

```txt
{"Nom de la case"}..<Nom de la propriété>
```


Pour plus de détails, consultez [Liste des propriétés utilisables sur les cases de la barre de message](../WDChamp/1010035.md).

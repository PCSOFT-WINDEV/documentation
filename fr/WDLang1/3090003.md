
## USBPropriété (Fonction)

***En anglais : USBProperty***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Récupère la valeur d'une propriété d'un périphérique USB.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
sDescription est une chaîne
sListe est une chaîne = USBListePériphérique()
SI sListe <> "" ALORS
	sDescription = USBPropriété(ExtraitChaîne(sListe, rangPremier, RC), USBPropriétéDescription)
	SI sDescription <> "" ALORS
		Info("Premier contrôleur USB trouvé : " + sDescription)
	FIN
FIN
```


<a name="Exemple2"></a>



```wl
// On commence l'exploration sans spécifier d'identifiant
Explore("")
```

```wl
// ----------------------------------------------------------- 
// Exploration complète de l'arborescence
PROCEDURE Explore(sDébut)
sListe est une chaîne
sListe = USBListePériphérique(sDébut)
SI sListe <> "" ALORS
	sPérif est une chaîne
	POUR TOUTE CHAINE sPerif de sListe SEPAREE PAR RC
		Trace(USBPropriété(sPerif, USBPropriétéDescription))
		Explore(sPerif)
	FIN
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = USBPropriété(<Identifiant USB> , <Propriété>)`
---

**`<Résultat> : (Type correspondant à la propriété)`**



- Chaîne ou entier contenant le résultat demandé. Peut également correspondre à une chaîne multiple : les différents éléments de la chaîne sont séparés par Caract(0).

- Chaîne vide ("") si la propriété n'est pas disponible (par exemple si le pilote du périphérique ne fournit pas la propriété demandée).




**`<Identifiant USB> : (Chaîne de caractères)`**

Identifiant du périphérique USB à manipuler. Cet identifiant est obtenu grâce à la fonction [USBListePériphérique](../WDLang1/3090002.md).

**`<Propriété> : (Constante de type Entier)`**

Option de détail du résultat :


|   |   |
| --- | --- |
| *USBPropriétéClasse* | Classe de périphérique |
| *USBPropriétéDescription* | Description du périphérique |
| *USBPropriétéEmplacement* | Emplacement physique du périphérique |
| *USBPropriétéFabricant* | Nom du fabricant |
| *USBPropriétéNomExplicite* | Nom explicite du périphérique |

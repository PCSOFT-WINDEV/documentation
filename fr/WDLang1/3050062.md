
## Cosinus (Fonction)

***En anglais : Cos***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Calcule le cosinus d'un angle.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Calcul du cosinus de 60
ResCosinus = Cosinus(60)
// Renvoie 0.5
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = Cosinus(<Angle>)`
---

**`<Résultat> : (Réel)`**

Cosinus de l'angle spécifié.

**`<Angle> : (Entier ou réel)`**

Angle (en degrés) à manipuler. Ce paramètre peut avoir une valeur quelconque : le calcul du cosinus est effectué "modulo 360".

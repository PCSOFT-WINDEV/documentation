
## EchangeLigne (Fonction)

***En anglais : SwapLine***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet d'échanger deux lignes dans un tableau à deux dimensions ou dans une propriété tableau d'un type avancé.

**Remarque** : Cette fonction est équivalente à la fonction [TableauEchangeLigne](../WDLang1/1000020483.md).




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonTableau est un tableau de 5 par 2 chaînes
MonTableau[1,1] =  "A"
MonTableau[1,2] =  "a"
MonTableau[2,1] =  "B"
MonTableau[2,2] =  "b"
MonTableau[3,1] =  "C"
MonTableau[3,2] =  "c"
MonTableau[4,1] =  "D"
MonTableau[4,2] =  "d"
MonTableau[5,1] =  "E"
MonTableau[5,2] =  "e"
// Échanger la ligne 2 avec la ligne 4 
EchangeLigne(MonTableau, 2, 4)
// Le tableau contient les éléments :
// "A", "a"
// "D", "d"
// "C", "c"
// "B", "b"
// "E", "e"
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`EchangeLigne(<Tableau WLangage> , <Indice de la 1° ligne> , <Indice de la 2° ligne>)`
---

**`<Tableau WLangage> : (Tableau)`**

Nom de la variable de type Tableau à manipuler. Ce tableau doit être un tableau à deux dimensions.

**`<Indice de la 1° ligne> : (Entier)`**

Indice de la première ligne à échanger.

**`<Indice de la 2° ligne> : (Entier)`**

Indice de la seconde ligne à échanger.




## JourFérié (Fonction)

***En anglais : BankHoliday***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Indique si un jour est férié ou non. Les dates correspondant à des jours fériés sont précisées avec la fonction [JourFériéAjoute](../WDLang1/1000017304.md).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Définition des jours fériés
JourFériéAjoute("0714")
// Vérification d'un jour férié
SI JourFérié("19700714") = Vrai ALORS
	Info(DateVersChaîne("19700714", maskDateSystème) + " est un jour férié")
FIN
// Vérification d'une date présente dans un champ de saisie
SI JourFérié(SAI_Date) = Vrai ALORS
	Info(DateVersChaîne(SAI_Date, maskDateSystème) + " est un jour férié")
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = JourFérié(<Date à vérifier>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la date indiquée correspond à un jour férié défini avec la fonction [JourFériéAjoute](../WDLang1/1000017304.md), 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 




**`<Date à vérifier> : (Chaîne de caractères ou variable de type Date)`**

Date à vérifier. Si cette date correspond à une chaîne, elle doit être au format AAAAMMJJ (AAAA correspond à l'année, MM au mois et JJ au jour).

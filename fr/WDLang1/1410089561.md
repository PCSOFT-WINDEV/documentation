


## &lt;Kanban&gt;.InsèreCarte (Fonction)

***En anglais : &lt;Kanban&gt;.InsertCard***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Insère une carte dans une liste de Kanban.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Insère une carte à la première position de la liste de Kanban LISTE_Matière_à_Revoir
// Cette liste est définie sous l'éditeur
maCarte est un kbCarte
maCarte.Titre = "Anglais"
KANBAN_Révision.InsèreCarte(LISTE_Matière_à_Revoir, 1, maCarte)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Champ Kanban>.InsèreCarte(<Liste de Kanban> , <Position> , <Carte>)`
---

**`<Résultat> : (Entier)`**

Indice de l'élément inséré.

**`<Champ Kanban> : (Nom de champ)`**

Nom du champ Kanban manipulé.

**`<Liste de Kanban> : (Nom de champ)`**

Nom du champ Liste de Kanban dans lequel la carte doit être insérée. Ce nom peut correspondre : 

- au nom du champ Liste de Kanban défini sous l'éditeur, 

- au nom du champ Liste de Kanban défini par programmation. 


Si la liste de Kanban n'existe pas, elle est ajoutée au champ Kanban.

**`<Position> : (Entier)`**

Position à laquelle la carte doit être insérée.

**`<Carte> : (Variable de type kbCarte)`**

Nom de la variable de type [kbCarte](../WDLang1/1410089167.md) correspondant à la carte à insérée. La carte est insérée à la position spécifiée dans la liste de Kanban.

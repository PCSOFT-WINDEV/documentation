
## ServiceRedémarre (Fonction)

***En anglais : ServiceRestart***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Redémarre un service (déjà démarré). Ce service peut être un service présent sur le poste local ou sur un poste distant. Les options spécifiées lors du précédent démarrage du service sont automatiquement prises en compte (par exemple la ligne de commande).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Redémarre le service présent dans le champ TABLE_Service
SI ServiceRedémarre(COL_Nom[TABLE_Service]) = Faux ALORS
	Erreur(ErreurInfo(errMessage))
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = ServiceRedémarre(<Nom du service> [, <Poste distant>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le redémarrage a été demandé, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Nom du service> : (Chaîne de caractères)`**

Nom du service à redémarrer.

**`<Poste distant> : (Chaîne de caractères optionnelle)`**

Adresse IP du poste distant au format xxx.xxx.xxx.xxx (obtenu par la fonction [NetAdresseIP](../WDLang3/3056023.md)) ou nom du poste distant (obtenu par la fonction [NetNomMachine](../WDLang3/3056014.md)). Ce paramètre permet de redémarrer un service sur un autre poste. Pour utiliser cette option, l'utilisateur doit être administrateur de la machine distante. 
Si ce paramètre n'est pas précisé ou correspond à une chaîne vide (""), le service est manipulé en local.

### Mode de fonctionnement
<a name="mode_fonctionnement_ELTPARAGRAPHE000056"></a>La fonction **ServiceRedémarre** envoie au gestionnaire de services une demande de redémarrage d'un service. Cette opération peut ne pas être effectuée immédiatement. Pour connaître l'état du service, utilisez la fonction [ServiceAttendEtat](../WDLang1/1000017101.md).

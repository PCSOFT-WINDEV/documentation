
## PlanificateurModifieTâche (Fonction)

***En anglais : SchedulerModifyTask***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie les paramètres d'une tâche planifiée. Ces paramètres sont ceux saisis dans la structure [TâchePlanifiée](../WDLang1/3089004.md).

**Remarque** : Pour remplir la structure TâchePlanifiée avec les paramètres d'une tâche, utilisez la fonction [PlanificateurPropriétésTâche](../WDLang1/3089011.md).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Initialisation des valeurs par défaut
PlanificateurRAZ()
// Description de la tâche à ajouter
TâchePlanifiée.Application = "Calc.exe"  // Chemin du programme
TâchePlanifiée.Utilisateur = "Janis"     // Utilisateur de la session
TâchePlanifiée.MotDePasse = SAI_MDP          // Mot de passe de la session
TâchePlanifiée.Commentaire = "Programme de calcul"      // Commentaire de la tâche
TâchePlanifiée.SuppressionAutomatique = Faux      // Booléen à vrai si la tâche doit être supprimée 
// Ajout de la tâche préalablement décrite
PlanificateurAjouteTâche("Calculatrice")

...

// Modification du commentaire
TâchePlanifiée.Commentaire = "Calculette"
PlanificateurModifieTâche("Calculatrice")
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = PlanificateurModifieTâche(<Nom de la tâche>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> en cas de succès, 

- <u><u><u><u>Faux</u></u></u></u> en cas d'erreur.




**`<Nom de la tâche> : (Chaîne de caractères)`**

Nom de la tâche à manipuler. Ce nom a été défini lors de la création de la tâche avec la fonction [PlanificateurAjouteTâche](../WDLang1/3089003.md).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Paramètres concernant l'utilisateur
<a name="parametres_concernant_utilisateur_ELTPARAGRAPHE000061"></a>Si la variable **TâchePlanifiée.Application** est modifiée, il est nécessaire de renseigner à nouveau le mot de passe de l'utilisateur (variable TâchePlanifiée.MotDePasse).

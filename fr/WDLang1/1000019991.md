


## &lt;Multimédia&gt;.Fin (Fonction)

***En anglais : &lt;Multimedia&gt;.End***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Place à la fin du fichier multimédia joué dans le champ Multimédia.








<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) 
```wl
MonFichier est une chaîne = "C:\Ma musique\mysong.mp3"

MM_Multimédia1.Ouvre(MonFichier)
MM_Multimédia1.Fin()
...
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Champ Multimédia>.Fin()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le positionnement a été réalisé, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Champ Multimédia> : (Nom de champ)`**

Nom du champ Multimédia à manipuler.

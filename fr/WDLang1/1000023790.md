


## &lt;Chrono&gt;.Pause (Fonction)

***En anglais : &lt;Chrono&gt;.Pause***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Interrompt un chronomètre afin de mesurer la durée d'un traitement. Le chronométrage pourra être relancé par la fonction [&lt;Chrono&gt;.Reprend](../WDLang1/1000023792.md).






<a name="sample_code"></a>

## Exemple

<a name="Exemple1"></a>


![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png)![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) 
```wl
DuréeTraitement est une Durée 
MonChrono est un Chrono 
MonChrono.Début()
MaFonction1()
DuréeTraitement = MonChrono.Pause()
Info("Le traitement MaFonction1 a duré " + DuréeTraitement)
MonChrono.Reprend()
MaFonction2()
DuréeTraitement = MonChrono.Fin()
Info("Les traitements MaFonction1 et MaFonction2 ont duré " + DuréeTraitement)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Chronomètre>.Pause()`
---

**`<Résultat> : (Durée)`**

Temps passé (en millisecondes) depuis le dernier appel à la fonction [&lt;Chrono&gt;.Début](../WDLang1/1000023788.md) pour le chronomètre spécifié.

**`<Chronomètre> : (Variable de type Chrono)`**

Nom de la variable de type [Chrono](../WDLang1/1000023560.md) correspondant au chronomètre à utiliser.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Divers
<a name="divers_ELTPARAGRAPHE000300"></a>Si la fonction **&lt;Chrono&gt;.Pause** est utilisée sur un chronomètre déjà interrompu, cette fonction n'a aucun effet.
<a name="NOTE0_2"></a>


### Erreur
<a name="erreur_ELTPARAGRAPHE000313"></a>Si la fonction [&lt;Chrono&gt;.Début](../WDLang1/1000023788.md) n'a pas été appelée avant la fonction **&lt;Chrono&gt;.Pause**, une erreur WLangage est affichée.
<a name="NOTE0_3"></a>


### Threads
<a name="threads_ELTPARAGRAPHE000331"></a>Les fonctions de chronométrage (Chronoxxx) concernant un même chronomètre doivent être utilisées dans un même thread.

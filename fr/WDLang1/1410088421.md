
## DiagrammeGroupe (Fonction)

***En anglais : DiagramGroup***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Crée un groupe à partir des éléments sélectionnés dans le champ Editeur de diagrammes. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Crée un groupe avec la sélection
SI DiagrammeGroupe(EDIAG_MonDiagramme) ALORS
	ToastAffiche("Groupement créé - CTRL+Z pour annuler")
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = DiagrammeGroupe(<Diagramme>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le groupe a été créé, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. La fonction [ErreurInfo](../WDLang1/3013008.md) permet d'obtenir plus de détails sur l'erreur.




**`<Diagramme> : (Nom de champ ou variable de type Diagramme)`**

Diagramme à manipuler. Ce paramètre correspond : 

- au nom d'un champ Editeur de diagrammes. 

- au nom d'une variable de type [Diagramme](../WDLang1/1410088055.md).






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Une forme de type "Groupe" est créée et ajoutée dans le diagramme. 

- Cette forme est créée à partir de la sélection en cours. 

- Si la sélection comprend moins de 2 formes, la fonction ne crée pas le groupe et renvoie <u><u><u><u>Faux</u></u></u></u>. 

- La création d'un groupe est annulable par l’utilisateur final (via le "undo") si le diagramme est en cours d’édition.


## fDisquePrêt (Fonction)

***En anglais : fDriveReady***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de savoir :

- si le lecteur de carte, CD ou DVD est disponible,

- si le disque existe.



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple
<a class="notetitle" target="_blank" href="$DOC$=1000003036047&name=fdisquepret_fonction&product=WD">Voir des exemples supplémentaires</a>

```wl
// Lecteur disponible ?
ResDisponible = fDisquePrêt("E")
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = fDisquePrêt(<Chemin du disque>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le lecteur est prêt, accessible et disponible, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour connaître le libellé de l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.




**`<Chemin du disque> : (Chaîne de caractères)`**

Chemin du disque à manipuler de la forme :

- &lt;lettre du lecteur&gt;. Par exemple : "a", "a:" ou "a:\\".

- chemin UNC. Par exemple : "\\\\&lt;NomServeur&gt;\\&lt;NomPartage&gt;".




Si le lecteur est un lecteur de disquette, de carte, de CD ou de DVD, le lecteur est prêt uniquement s'il contient une disquette, une carte, un CD ou un DVD.

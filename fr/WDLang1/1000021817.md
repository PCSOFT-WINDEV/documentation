
## SysMémoireStockageExterne (Fonction)

***En anglais : SysMemoryExternalStorage***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la quantité de mémoire disponible sur la zone de stockage externe de l'appareil.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Récupère la quantité de mémoire disponible sur l'espace de stockage externe
nEspaceLibre est un entier sur 8 
nEspaceLibre = SysMémoireStockageExterne(1)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = SysMémoireStockageExterne(<Indice> [, <Type>])`
---

**`<Résultat> : (Entier)`**



- Quantité de mémoire (en octets), 

- -1 en cas d'erreur.




**`<Indice> : (Entier)`**

Indice de l'espace de stockage externe à utiliser (si l'appareil possède plusieurs espaces de stockage externe). 
Remarque : La fonction [SysNbStockageExterne](../WDLang1/1000021816.md) permet de connaître le nombre d'espaces de stockage externe de l'appareil.

**`<Type> : (Constante optionnelle de type Entier)`**

Type de mémoire à calculer : 


|   |   |
| --- | --- |
| *sysEspaceStockage* | Espace total disponible pour le stockage. |
| *sysEspaceStockageLibre*<br>(Valeur par défaut) | Espace libre pour le stockage. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Un même appareil peut contenir plusieurs zones de stockage externe (par exemple, une zone correspond à la partition "externe" de la mémoire permanente et une autre zone correspond à la SDCard).

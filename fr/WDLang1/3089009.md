
## PlanificateurOuvrePropriétésTâche (Fonction)

***En anglais : SchedulerOpenTaskProperties***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ouvre la page de propriétés d'une tâche planifiée. Il est ainsi possible de configurer la tâche planifiée avec le planificateur de tâches de Windows.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ouverture de la page de propriétés de la tâche MaTâche
PlanificateurOuvrePropriétésTâche("MaTache")
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`PlanificateurOuvrePropriétésTâche(<Nom de la tâche>)`
---

**`<Nom de la tâche> : (Chaîne de caractères)`**

Nom de la tâche à manipuler.

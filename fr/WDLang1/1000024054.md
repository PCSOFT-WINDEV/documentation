


## &lt;Table&gt;.ColonnesTriées (Fonction)

***En anglais : &lt;Table&gt;.SortedColumn***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la liste des colonnes triées d'un champ Table. 

Ces colonnes ont été triées : 

- directement par l'utilisateur. 

- par la fonction [&lt;Table&gt;.Trie](../WDLang1/1000024293.md). 







<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
ListeColonnes est une chaîne
ListeColonnes = TABLE_Table1.ColonnesTriées()
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Champ Table>.ColonnesTriées()`
---

**`<Résultat> : (Chaîne de caractères)`**

Liste des différentes colonnes triées avec leur sens de tri. Cette liste est de la forme : 

```txt
<Sens><Colonne1> + TAB + <Sens><Colonne2> + TAB + ...
```


- &lt;Sens&gt; est le sens du tri. Lorsque le tri est décroissant, ce paramètre correspond à "-". 

- &lt;Colonne&gt; est le nom de la colonne.




**`<Champ Table> : (Nom de champ)`**

Nom du champ Table à manipuler.

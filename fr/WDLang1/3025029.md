


## OuvreMenuContextuel (Fonction)

***En anglais : OpenContextMenu***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ouvre automatiquement un menu contextuel pour le champ, la fenêtre ou la page en cours.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// -- Code de clic du bouton BTN_Action
OuvreMenuContextuel(MenuChoixDate)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`OuvreMenuContextuel([<Nom du menu contextuel> [, <X> , <Y >]])`
---

**`<Nom du menu contextuel> : (Chaîne de caractères optionnelle)`**

Nom du menu contextuel à ouvrir pour le champ en cours. 
Si ce paramètre n'est pas spécifié, le menu contextuel par défaut associé au champ est ouvert.


**`<X> : (Entier (optionnel))`**

Abscisse de la position d'ouverture du menu contextuel. Cette abscisse est indiquée en pixels, par rapport au coin haut gauche de la fenêtre. Si les paramètres &lt;X&gt; et &lt;Y&gt; ne sont pas spécifiés, le menu contextuel est ouvert à la position de la souris.


**`<Y > : (Entier (optionnel))`**

Ordonnée de la position d'ouverture du menu contextuel. Cette ordonnée est indiquée en pixels, par rapport au coin haut gauche de la fenêtre. Si les paramètres &lt;X&gt; et &lt;Y&gt; ne sont pas spécifiés, le menu contextuel est ouvert à la position de la souris.




<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Cette fonction doit être exécutée dans un code associé à un champ, une fenêtre ou une page. 
	Il n'est pas conseillé d'utiliser cette fonction dans le code de redimensionnement d'une fenêtre.

- Cette fonction est bloquante jusqu'à ce que le menu contextuel soit fermé.
	

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Dans le cas d'une fenêtre fille MDI, il peut être intéressant d'ouvrir un menu contextuel défini dans la fenêtre mère MDI. Dans ce cas, le paramètre &lt;Nom du menu contextuel&gt; doit être de la forme : 
	
	```txt
	NomDeLaFenêtre.NomDuMenuContextuel
	```




## &lt;Liste de Kanban&gt;.InsèreCarte (Fonction)

***En anglais : &lt;Kanban list&gt;.InsertCard***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Insère une carte dans une liste de Kanban.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Crée dynamiquement une liste de Kanban
LISTE_Liste_a_revoir est un Champ
LISTE_Liste_a_revoir <- KANBAN_Révision.AjouteListe("Revues")

// Insère une carte à la première position 
// dans la liste de Kanban créée dynamiquement 
// (Liste de Kanban manipulée par son nom)
maCarte1 est un kbCarte
maCarte1.Titre = "Géographie"
LISTE_Liste_a_revoir.InsèreCarte(1, maCarte1)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Liste de Kanban>.InsèreCarte(<Position> , <Carte>)`
---

**`<Résultat> : (Entier)`**

Indice de l'élément inséré.

**`<Liste de Kanban> : (Nom de champ)`**

Nom du champ Liste de Kanban dans lequel la carte doit être insérée. Ce nom peut correspondre : 

- au nom du champ Liste de Kanban défini sous l'éditeur, 

- au nom du champ Liste de Kanban défini par programmation.




**`<Position> : (Entier)`**

Position à laquelle la carte doit être insérée.

**`<Carte> : (Variable de type kbCarte)`**

Nom de la variable de type [kbCarte](../WDLang1/1410089167.md) correspondant à la carte à insérée. La carte est insérée à la position spécifiée dans la liste de Kanban.

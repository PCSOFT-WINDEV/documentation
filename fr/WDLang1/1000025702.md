
## ErreurRAZ (Fonction)

***En anglais : ErrorReset***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Réinitialise la détection des erreurs non fatales : 

- Le résultat de la fonction [ErreurInfo](../WDLang1/3013008.md) est réinitialisé. 

- La variable [ErreurDétectée](../WDLang1/3087001.md) revient à <u><u><u>Faux</u></u></u>.





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// dans une boucle
POUR TOUT ...
	// Réinitialise la détection des erreurs non fatales
	ErreurRAZ()
	// Exécute l'opération
	bExécutionOK = UneOpération()
	// Traite les erreurs 
	SI bExécutionOK = Faux ALORS
		// Ecriture de l'erreur dans un log
		LogErreur(ErreurInfo())
	FIN
FIN
```

## Syntaxe

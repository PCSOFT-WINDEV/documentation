
## AscenseurAffiche (Fonction)

***En anglais : ScrollBarShow***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Affiche ou non un ascenseur dans un champ WINDEV (champ de saisie ou liste), même si cet ascenseur n'a pas été sélectionné lors de la description du champ.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
AscenseurAffiche(LISTE_MaListe, ascHorz, Faux)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`AscenseurAffiche(<Champ manipulé> [, <Position ascenseur> [, <Visible>]])`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ manipulé : 

- Champ de saisie, 

- Champ Liste.




**`<Position ascenseur> : (Constante optionnelle)`**

Position de l'ascenseur dans le champ :


|   |   |
| --- | --- |
| *ascHorz*<br>(Valeur par défaut) | Ascenseur horizontal. |
| *ascTous* | Ascenseur vertical et horizontal. |
| *ascVert* | Ascenseur vertical. |



**`<Visible> : (Booléen optionnel)`**



- <u><u><u><u>Vrai</u></u></u></u> (par défaut) pour rendre visible l'ascenseur ajouté, 

- <u><u><u><u>Faux</u></u></u></u> pour rendre invisible l'ascenseur ajouté.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

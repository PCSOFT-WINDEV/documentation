
## Canada.VérifieNE15 (Fonction)

***En anglais : Canada.CheckBN15***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Teste la validité d'un numéro NE15 (numéro d'entreprise sur 15 caractères identifiant un établissement d'une entreprise au Canada).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Test d'une variable
MonNuméroNE15 est une chaîne = "887471886RT0001"
Canada.VérifieNE15(MonNuméroNE15)		// renvoie Vrai

// Test d'un champ
SAI_NuméroNE15 = "ABC"
Canada.VérifieNE15(SAI_NuméroNE15)	// renvoie Faux
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = Canada.VérifieNE15(<Numéro à vérifier>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'élément spécifié contient un numéro NE15 valide.

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Numéro à vérifier> : (Nom de champ ou variable de type chaîne)`**

Numéro NE15 à vérifier. Ce paramètre peut correspondre :

- au nom d'une variable de type chaîne.

- au nom d'un champ.

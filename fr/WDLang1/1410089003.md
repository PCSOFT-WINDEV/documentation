


## &lt;Image&gt;.ModifieContraste (Fonction)

***En anglais : &lt;Image&gt;.ModifyContrast***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie le contraste d'une image contenue dans un champ Image.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Modification du contraste
IMG_Image1.ModifieContraste(50)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Champ Image>.ModifieContraste(<Variation du contraste> [, <Image destination>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le contraste a été modifié, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Champ Image> : (Nom de champ)`**

Nom du champ Image à manipuler.

**`<Variation du contraste> : (Réel)`**

Valeur comprise entre -100 et +100. Les valeurs négatives diminuent le contraste, les valeurs positives l'augmentent. Cette valeur sera ajoutée au contraste actuel pour obtenir un nouveau contraste.

**`<Image destination> : (Nom de champ optionnel ou variable de type Image optionnelle)`**

Image dans laquelle l'image transformée sera dessinée. Cette image peut correspondre : 

- au nom d'un champ Image. 

- au nom d'une variable de type [Image](../WDLang1/1000019650.md).


Si ce paramètre n'est pas spécifié, la transformation s'effectuera sur &lt;Champ Image&gt;.

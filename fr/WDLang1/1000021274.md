


## InterrupteurOccurrence (Fonction)

***En anglais : CheckBoxCount***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le nombre d'options présentes dans un champ Interrupteur.

Remarque : Cette fonction renvoie le même résultat que la propriété [Occurrence](../Proprietes/2510097.md) utilisée sur un champ Interrupteur. 




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Nombre d'options présentes dans le champ "INT_Options"
ResNombreOptions = InterrupteurOccurrence(INT_Options)

// Identique à : 
// ResNombreOptions = INT_Options.Occurrence
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = InterrupteurOccurrence(<Champ Interrupteur>)`
---

**`<Résultat> : (Entier)`**

Nombre d'options présentes dans le champ spécifié.

**`<Champ Interrupteur> : (Nom de champ)`**

Nom du champ Interrupteur à manipuler.

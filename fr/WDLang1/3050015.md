
## EstImpair (Fonction)

***En anglais : IsOdd***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Identifie un nombre impair.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Nombre impair ?
ResImpair = EstImpair(65)      // Renvoie Vrai
ResImpair = EstImpair(5 + 11)  // Renvoie Faux
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = EstImpair(<Nombre>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le nombre spécifié est impair, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Nombre> : (Expression numérique entière ou entier)`**

Expression numérique entière ou entier à identifier.

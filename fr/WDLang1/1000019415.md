
## MoisEnCours (Fonction)

***En anglais : CurrentMonth***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le numéro du mois en cours.






<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
Res est un entier
Res = MoisEnCours()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = MoisEnCours()`
---

**`<Résultat> : (Entier)`**

Numéro du mois en cours.

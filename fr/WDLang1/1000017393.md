


## &lt;Liste&gt;.SelectMoins (Fonction)

***En anglais : &lt;List Box&gt;.SelectMinus***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Désélectionne un élément dans un champ Liste, Liste image ou Combo.

**Rappel** : L'élément sélectionné correspond à l'élément en cours sur lequel le bandeau de sélection est affiché.

**Remarque** : La fonction **&lt;Liste&gt;.SelectMoins** peut être utilisée indifféremment sur :

- les champs Liste fichier ou Liste par programmation,

- les champs Liste monosélection ou multisélection.













<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Désélection de l'élément dont l'indice est 7 dans le champ "LISTE_ListeClient"
LISTE_ListeClient.SelectMoins(7)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Champ Liste>.SelectMoins([<Indice 1> [... [, <Indice N>]]])`
---

**`<Champ Liste> : (Nom de champ)`**

Nom du champ Liste (ou Combo) à manipuler.

**`<Indice 1> : (Entier optionnel)`**

Indice du premier élément à désélectionner. 
Si aucun indice n'est précisé, tous les éléments du champ Liste (ou Combo) spécifié sont désélectionnés. 
Si ces indices correspondent à des éléments non sélectionnés, la fonction **&lt;Liste&gt;.SelectMoins** est sans effet.



**`<Indice N> : (Entier optionnel)`**

Indice du Nième élément à désélectionner. 
Si aucun indice n'est précisé, tous les éléments du champ Liste (ou Combo) spécifié sont désélectionnés. 
Si ces indices correspondent à des éléments non sélectionnés, la fonction **&lt;Liste&gt;.SelectMoins** est sans effet.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Champ Combo
<a name="champ_combo_ELTPARAGRAPHE000247"></a>Si la fonction **&lt;Liste&gt;.SelectMoins** est utilisée avec un champ Combo :

- aucun élément du champ Combo n'est affiché,

- si le champ Combo est en saisie, le champ de saisie est mis à vide.



<a name="NOTE0_2"></a>


### Divers
<a name="divers_ELTPARAGRAPHE000263"></a>

- Pour désélectionner un élément dans un champ Table, utilisez la fonction [&lt;Table&gt;.SelectMoins](../WDLang1/1000024285.md).

- La fonction **&lt;Liste&gt;.SelectMoins** peut être utilisée sur une colonne d'un champ Table de type "Combo".

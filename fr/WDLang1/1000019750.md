
## GesteNbPointeur (Fonction)

***En anglais : GestureNbPointer***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Retourne le nombre de pointeurs en contact avec l'écran. 

La valeur renvoyée varie selon le mode d'utilisation de la fonction :

- Utilisée dans un événement lié au mouvement du pointeur (clic, déplacement, enfoncé, relâché, ...), retourne le nombre de pointeurs (doigt ou stylet) actuellement en contact avec l'écran

- Utilisée dans un autre événement ou traitement, retourne le dernier nombre de points de contacts connu.





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// -- Code de clic sur un champ Image
nbPointsContact est un entier
nbPointsContact = GesteNbPointeur()

ToastAffiche("Vous utilisez actuellement " + nbPointsContact + " doigt(s) sur l'écran")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = GesteNbPointeur()`
---

**`<Résultat> : (Entier)`**

Nombre de pointeurs (doigts ou stylets) en contact avec l'écran.

## Remarques

### Limitations
<a name="limitations_ELTPARAGRAPHE000071"></a>



<a name="NOTE0_3"></a>


### Divers
<a name="divers_ELTPARAGRAPHE000096"></a>

- Pour connaître la position verticale (Y) d'un pointeur, utilisez la fonction [GestePosY](../WDLang1/1000019745.md).

- Pour connaître la position horizontale (X) d'un pointeur, utilisez la fonction [GestePosX](../WDLang1/1000019741.md). 

- Pour connaître l'indice du pointeur ayant déclenché l'exécution d'un événement lié à un mouvement sur l'écran, utilisez la fonction [GestePointeurEncours](../WDLang1/1000019751.md).

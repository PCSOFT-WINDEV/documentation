


## &lt;Menu&gt;.InsèreSéparateur (Fonction)

***En anglais : &lt;Menu&gt;.InsertSeparator***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Insère un séparateur dans un menu. Pour insérer un séparateur à la fin d'un menu, utilisez la fonction [&lt;Menu&gt;.AjouteSéparateur](../WDLang1/1000020603.md).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ajoute un nouveau menu
_Menu.AjouteMenu("MNU_Planning", "Planning")
// Ajoute des options dans le nouveau menu
MNU_Planning.AjouteOption("MNU_Créer", "Créer", Créer_Planning)
MNU_Planning.AjouteOption("MNU_Voir", "Voir", Voir_Planning)
// Insère un séparateur
MNU_Planning.InsèreSéparateur("MNU_Créer")
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Insérer un séparateur (position définie par un nom d'option de menu)

`<Menu d'origine>.InsèreSéparateur(<Option d'insertion>)`
---

**`<Menu d'origine> : (Nom de menu)`**

Nom du menu dans lequel le séparateur doit être inséré.

**`<Option d'insertion> : (Chaîne de caractères)`**

Nom de l'option de menu utilisée comme position de base pour l'insertion. Le nouveau séparateur sera inséré avant cette position.


<a name="SYNTAXE2"></a>

### Insérer un séparateur (position définie par un indice)

`<Menu d'origine>.InsèreSéparateur(<Indice>)`
---

**`<Menu d'origine> : (Nom de menu)`**

Nom du menu dans lequel le séparateur doit être inséré.

**`<Indice> : (Entier)`**

Indice de l'option de menu utilisée comme position de base pour l'insertion. Le nouveau séparateur sera inséré avant cette position.

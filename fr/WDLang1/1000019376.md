


## ArbreSelectOccurrence (Fonction)

***En anglais : TreeSelectCount***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le nombre d'éléments sélectionnés dans un champ Arbre (notamment dans le cas d'un champ Arbre multisélection). Les lignes sélectionnées correspondent aux lignes en cours sur lesquelles le bandeau de sélection est affiché.

**Remarque** : La fonction **ArbreSelectOccurrence** peut être utilisée indifféremment sur :

- un champ Arbre fichier, Arbre par programmation ou Arbre sur variable. 

- un champ Arbre monosélection ou multisélection. 



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
nbOccurrences est un entier 
nbOccurrences = ArbreSelectOccurrence(ARBRE_Arbre1)
POUR i = 1 A nbOccurrences 
	Trace("Noeud sélectionné :" + ArbreSelect(ARBRE_Arbre1, Faux, i))  
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = ArbreSelectOccurrence(<Champ Arbre>)`
---

**`<Résultat> : (Entier)`**



- Nombre de lignes sélectionnées dans le champ Arbre spécifié. Ce nombre peut être supérieur à 1 pour les champs Arbre multisélection. 

- 0 si aucune ligne n'est sélectionnée.




**`<Champ Arbre> : (Nom de champ)`**

Nom du champ Arbre à manipuler.
Si ce paramètre ne correspond pas à un nom de champ Arbre, une erreur WLangage est générée.
Si ce paramètre correspond à une chaîne vide (""), le champ Arbre auquel appartient l'événement en cours est manipulé.

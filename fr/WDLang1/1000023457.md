


## EnModeArrièrePlan (Fonction)

***En anglais : InBackgroundMode***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Identifie si le code est exécuté alors que l'application est en arrière-plan.

Rappel : Une application est considérée comme étant en arrière-plan lorsqu'aucune de ses fenêtres n'est visible à l'écran.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Si l'application n'est pas en arrière-plan, on peut afficher une boîte de dialogue, 
// sinon on effectue une notification
SI PAS EnModeArrièrePlan() ALORS
	Info("Traitement terminé")
SINON
	n est une Notification
	n..Message = "Traitement terminé"
	NotifAjoute(n)
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = EnModeArrièrePlan()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le code s'exécute alors que l'application est en arrière-plan,

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.








<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Disponibilité de la fonction
<a name="disponibilite_fonction_ELTPARAGRAPHE000187"></a>La fonction **EnModeArrièrePlan** est disponible dans tous les produits pour permettre l'écriture d'un code multi-produits. Des fonctions ou procédures utilisant la fonction **EnModeArrièrePlan** peuvent être partagées entre plusieurs produits, sans que des erreurs de compilation soient générées.
<a name="NOTE0_2"></a>


### Exécution des traitements contenant des erreurs de compilation
<a name="execution_des_traitements_contenant_des_erreurs_compilation_ELTPARAGRAPHE000203"></a>Par défaut, le mécanisme de sécurité du WLangage se déclenche uniquement si la ligne contenant l'erreur de compilation est exécutée.

Par exemple, si une fonction interdite dans le produit en cours est présente dans une boucle SI, le mécanisme de sécurité du WLangage se déclenchera uniquement si la ligne contenant cette fonction s'exécute.

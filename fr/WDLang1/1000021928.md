
## Constantes de gestion de la modification des fenêtres par l'utilisateur
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


| Constante | Description | Utilisée par les propriétés/fonctions |
| --- | --- | --- |
| **feAjouteFil** | Autorisation de modification : L'utilisateur final peut ajouter un fil entre plusieurs champs. | [FenEditionAutorisée](../Proprietes/1000021850.md), [FenEditionMode](../Proprietes/1000021824.md) |
| **feAjouteNoteRepositionnable** | Autorisation de modification : L'utilisateur final peut ajouter une note repositionnable dans la fenêtre. | [FenEditionAutorisée](../Proprietes/1000021850.md), [FenEditionMode](../Proprietes/1000021824.md) |
| **feAucun** | Autorisation de modification : Aucune modification n'est autorisée. | [FenEditionAutorisée](../Proprietes/1000021850.md), [FenEditionMode](../Proprietes/1000021824.md) |
| **feChangeBulle** | Autorisation de modification : L'utilisateur final peut modifier les bulles d'aide des champs. | [FenEditionAutorisée](../Proprietes/1000021850.md), [FenEditionMode](../Proprietes/1000021824.md) |
| **feChangeFil** | Autorisation de modification : L'utilisateur final peut modifier les fils entre les champs. | [FenEditionAutorisée](../Proprietes/1000021850.md), [FenEditionMode](../Proprietes/1000021824.md) |
| **feDéplaceChamp** | Autorisation de modification : L'utilisateur final peut déplacer les champs. | [FenEditionAutorisée](../Proprietes/1000021850.md), [FenEditionMode](../Proprietes/1000021824.md) |
| **feModeComplet** | Type de sauvegarde à réaliser : Sauvegarde de tous les champs et de leurs propriétés courantes. Cette constante peut être utilisée uniquement sur une fenêtre interne. | [FenEdSauve](../WDLang1/1000021848.md) |
| **feModeDifférentiel** | Type de sauvegarde à réaliser : Sauvegarde de toutes les opérations de modification, y compris les opérations personnalisées (fonction [FenEdDo](../WDLang1/1000021897.md)). Ces opérations seront toutes rejouées lors du rechargement (comme pour un journal). | [FenEdSauve](../WDLang1/1000021848.md) |
| **feModeNoteRepositionnable** | Type de sauvegarde à réaliser : Sauvegarde uniquement des notes repositionnables créées par l'utilisateur. Cette constante est équivalente à la fonctionnalité "Exporter les notes" présente dans le menu contextuel des notes repositionnables (FAA). | [FenEdSauve](../WDLang1/1000021848.md) |
| **feMultiSélection** | Autorisation de modification : L'utilisateur final peut sélectionner plusieurs champs à la fois. | [FenEditionAutorisée](../Proprietes/1000021850.md), [FenEditionMode](../Proprietes/1000021824.md) |
| **fePosition** | Autorisation de modification : L'utilisateur final peut déplacer les champs. | [FenEditionAutorisée](../Proprietes/1000021850.md), [FenEditionMode](../Proprietes/1000021824.md) |
| **feRedimensionneChamp** | Autorisation de modification : L'utilisateur final peut redimensionner les champs. | [FenEditionAutorisée](../Proprietes/1000021850.md), [FenEditionMode](../Proprietes/1000021824.md) |
| **feSélection** | Autorisation de modification : L'utilisateur final peut sélectionner les champs. | [FenEditionAutorisée](../Proprietes/1000021850.md), [FenEditionMode](../Proprietes/1000021824.md) |
| **feTaille** | Autorisation de modification : L'utilisateur final peut modifier la taille des champs. | [FenEditionAutorisée](../Proprietes/1000021850.md), [FenEditionMode](../Proprietes/1000021824.md) |
| **feTout** | Autorisation de modification : Toutes les modifications sont autorisées. | [FenEditionAutorisée](../Proprietes/1000021850.md), [FenEditionMode](../Proprietes/1000021824.md) |

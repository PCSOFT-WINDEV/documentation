


## TableFinRemplissage (Fonction)

***En anglais : TableEndFilling***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Indique la fin du remplissage d'un champ Table (ou Table hiérarchique) par programmation.

Rappel : Le début du remplissage a été indiqué avec la fonction [TableDébutRemplissage](../WDLang1/1000022368.md). L'avancement du remplissage est représenté :

- soit par une jauge infinie affichée au bas du champ.

- soit par une fenêtre interne.




Attention : Avant d'utiliser cette fonction, vous devez activer le mécanisme de gestion du remplissage dans la fenêtre de description du champ. Pour plus de détails, consultez [Caractéristiques des champs Table dans une fenêtre](../WDChamp/1013200.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
TableDébutRemplissage(TABLE_Mem) 

POUR i = 1 A 10000
	TableAjouteLigne(TABLE_Mem)	
FIN

TableFinRemplissage(TABLE_Mem) // Remplissage terminé
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`TableFinRemplissage(<Champ Table>)`
---

**`<Champ Table> : (Nom de champ)`**

Nom du champ à manipuler. Ce champ peut correspondre à : 

- un champ Table.

- un champ Table hiérarchique. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) un champ Combo avec table.


Si ce paramètre correspond à une chaîne vide (""), le champ Table manipulé sera celui auquel appartient l'événement en cours.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Cette fonction est sans effet sur les champs Table reliés à une source de données de type fichier ou requête.

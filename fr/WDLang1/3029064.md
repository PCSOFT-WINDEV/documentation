
## gPoliceTaille (Fonction)

***En anglais : gFontSize***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Construit une séquence de caractères permettant d'indiquer la taille de la police d'affichage du texte d'une chaîne graphique.

**Remarque** : Tout le texte écrit **à la suite** de la fonction est affecté.

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Dans une fenêtre, cette séquence de caractères permet d'indiquer la taille de la police :  

- dans un élément d'un champ Liste ou Combo.

- dans la barre de titre et/ou de message d'une fenêtre. 

- dans un champ Libellé. 







<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 
```wl
// Affichage du titre "WINDEV, AGL n°1 en France." de la prochaine fenêtre
TitreSuivant(gPoliceTaille(12) + "WINDEV" + gPoliceTaille(10) + ", AGL n°1 en France.")
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = gPoliceTaille(<Taille de la police>)`
---

**`<Résultat> : (Chaîne de caractères)`**

Séquence de caractères définissant la taille de la police pour la suite du texte.

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Cette séquence de caractères peut être : 

- utilisée dans les fonctions [ListeAjoute](../WDLang1/3049004.md), [ListeInsère](../WDLang1/3049002.md) ou [ListeModifie](../WDLang1/3049009.md).

- affectée à un champ Libellé. 

- utilisée dans la barre de titre ou de message d'une fenêtre. 




**`<Taille de la police> : (Entier)`**

Nouvelle taille de la police pour le texte suivant la fonction **gPoliceTaille**.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- La police par défaut du champ manipulé est rétablie à la fin de l'appel des fonctions [gPoliceXXX](../WDLang1/3029028.md).

- Toutes les autres caractéristiques de la police sont conservées (Italique, ...).

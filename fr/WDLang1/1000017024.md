
## Abandonne (Fonction)

***En anglais : Abandon***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ferme la fenêtre en cours en exécutant l'événement "Clic" du champ Bouton de type ABANDON.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Clic sur Bouton BTN_ANNULER
Abandonne()
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`Abandonne()`
---



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Fonctionnement
<a name="fonctionnement_ELTPARAGRAPHE000056"></a>Si la fenêtre contient un champ Bouton de type ABANDON, l'événement "Clic" du champ Bouton est exécuté. Attention :

- Si l'événement "Clic" du champ Bouton de type ABANDON ferme la fenêtre, la fenêtre est fermée.

- Si l'événement "Clic" du champ Bouton de type ABANDON ne ferme pas la fenêtre, la fenêtre n'est pas fermée.




Si la fenêtre ne contient pas de champ Bouton de type ABANDON, la fenêtre est simplement fermée. Aucune valeur de retour n'est renvoyée.

Si la fonction **Abandonne** est appelée depuis un champ Bouton de type ABANDON, la fenêtre est fermée (même si l'événement "Clic" du champ Bouton n'utilise pas la fonction [Ferme](../WDLang1/3038018.md)). Si une valeur était renvoyée (avec la fonction [Ferme](../WDLang1/3038018.md) par exemple) dans l'événement "Clic" du champ Bouton, cette valeur n'est pas renvoyée.

L'appel de la fonction **Abandonne** est équivalent à la combinaison de touches Alt + F4.

**Remarque** : si la fenêtre possède plusieurs champs Bouton de type ABANDON, le fonctionnement est indéterminé.


<a name="NOTE0_2"></a>




### Rappel : Les champs Bouton devant être utilisés dans une fenêtre
<a name="rappel_les_champs_bouton_devant_etre_utilises_dans_une_fenetre_ELTPARAGRAPHE000088"></a>

- **Si votre fenêtre ne comporte pas de champs de saisie**, tous les champs Bouton peuvent être de type NORMAL.

- **Si votre fenêtre comporte au moins un champ de saisie**, les types des champs Bouton à utiliser sont :

	- un champ Bouton VALIDATION,

	- un champ Bouton ABANDON,

	- un champ Bouton de type AIDE (si les champs ont des fenêtres d'aide associées),

	- tous les autres champs Bouton de type INTERRUPTION.

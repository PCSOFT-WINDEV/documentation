


## Fonctions de gestion du champ Action Bar
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions de gestion de l'Action Bar sont les suivantes : 



|   |   |
| --- | --- |
| [ActionBarRechercheVisible](../WDLang1/1000021599.md) | Permet de connaître ou modifier l'état de visibilité du champ de recherche associé à l'Action Bar de la fenêtre courante. |
| [ActionBarSupprimeHistoriqueRecherche](../WDLang1/1000021600.md) | Supprime l'historique du champ de recherche de l'action bar d'une fenêtre. |

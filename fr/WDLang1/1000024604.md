
## picCalque (Type de variable)

***En anglais : picLayer***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **picCalque** permet de définir toutes les caractéristiques avancées d'un calque d'une image chargée dans une variable WDPic. Les caractéristiques de ce calque peuvent être définies et modifiées à l'aide de différentes propriétés WLangage. 

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ouverture d'une image source
MonImage est un WDPic
MonImage.NomFichier = fRepExe() + ["\"] + "Source.png"

// Ajout d'un calque contenant un rectangle
CalqueRectangle est un picCalque
CalqueRectangle.Nom = "Rectangle"
CalqueRectangle.Largeur = 100
CalqueRectangle.Hauteur = 100
dRectangle(CalqueRectangle, 1, 1, 50, 50, VertFoncé, BleuFoncé)

// Rotation du calque
dRotation(CalqueRectangle, 45, drAdapte)

CalqueRectangle.X = nXCalque
CalqueRectangle.Y = nYCalque
Ajoute(MonImage.Calque, CalqueRectangle)


// Sauvegarde du résultat
PicSauve(MonImage, fRepExe() + ["\"] + "Resultat.png")
```

## Remarques
<a name="NOTE0_1"></a>


### Propriétés spécifiques à la description des variables de type picCalque
<a name="proprietes_specifiques_description_des_variables_type_piccalque_ELTPARAGRAPHE000029"></a>Les propriétés suivantes peuvent être utilisées pour manipuler un calque d'une image de type WDPic :

| Nom de la propriété | Type manipulé | Effet |
| --- | --- | --- |
| Hauteur | Entier | Hauteur du calque exprimée en pixels. |
| Image | <br><br> | Image associée au calque.<br><br>Si cette propriété est renseignée, la propriété **Type** correspond à la constante *typeCalqueImage*. |
| Largeur | Entier | Largeur du calque exprimée en pixels. |
| Nom | Chaîne de caractères | Nom du calque. |
| RTF | Variable de type [picRTF](../WDLang1/1000025183.md) | Texte au format RTF. <br><br> |
| Texte | Variable de type [picTexte](../WDLang1/1000025178.md) | Texte du calque. Les propriétés de la variable [picTexte](../WDLang1/1000025178.md) permettent de définir et de connaître les options du texte (police, opacité, cadrage, ...).  <br><br> |
| Type | Constante de type Entier | Type du calque : <br><br>- *typeCalqueImage* : Calque de type Image. La propriété **Image** permet de connaître et de modifier l'image associée au calque. <br><br>- *typeCalqueRTF* : Calque de type RTF. La propriété **RTF** permet de connaître et de modifier le contenu du calque. <br><br>- *typeCalqueTexte* : Calque de type Texte. La propriété **Texte** permet de connaître et de modifier le texte associé au calque. <br><br><br><br><br><br><br> |
| Visible | Booléen | <br><br>- <u><u><u><u>Vrai</u></u></u></u> si le calque est visible, <br><br>- <u><u><u><u>Faux</u></u></u></u> si le calque est invisible. <br><br><br> |
| X | Entier | Position horizontale du calque sur l'image, exprimée en pixels. |
| Y | Entier | Position verticale du calque sur l'image, exprimée en pixels. |

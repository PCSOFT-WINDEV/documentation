
## FinErreur (Fonction)

***En anglais : FinError***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le numéro de la dernière erreur due à une fonction Finxxx (voir la liste des [fonctions financières](../WDLang1/3050054.md)).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple
<a class="notetitle" target="_blank" href="$DOC$=1000003050071&name=finerreur_fonction&product=WD">Voir des exemples supplémentaires</a>

```wl
// Identification du type d'erreur
ResErreur = FinErreur()
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = FinErreur()`
---

**`<Résultat> : (Entier)`**

Numéro de la dernière erreur due à une fonction financière :

| Numéro | Type d'erreur | Fonctions concernées |
| --- | --- | --- |
| 0 | Pas d'erreur | Toutes les fonctions |
| 1 | La durée ne peut pas être nulle | [FinAmortissementDégressif](../WDLang1/3050068.md),<br>[FinAmortissementLinéaire](../WDLang1/3050046.md) |
| 2 | La matrice est vide | [FinValActuelleNette](../WDLang1/3050053.md) |
| 4 | La période ne peut pas être plus grande que la durée | [FinIntérêtPériode](../WDLang1/3050042.md) |
| 5 | Le type d'amortissement n'est pas correct | [FinAmortissement](../WDLang1/3050061.md) |
| 6 | La somme de remboursement est nulle | [FinNbVersement](../WDLang1/3050041.md) |
| 7 | Incohérence dans les données | [FinTauxIntérêt](../WDLang1/3050047.md),<br>[FinNbVersement](../WDLang1/3050041.md) |
| 8 | Le coût d'acquisition ne peut pas être nul | [FinAmortissementDégressif](../WDLang1/3050068.md), |
| 9 | La matrice n'existe pas | [FinValActuelleNette](../WDLang1/3050053.md) |
| 10 | Valeurs maximales dépassées | Toutes les fonctions |
| 11 | Calcul impossible (division par zéro) | [FinAmortissement](../WDLang1/3050061.md),<br>[FinIntérêtPériode](../WDLang1/3050042.md),<br>[FinValActuelle](../WDLang1/3050055.md),<br>[FinValActuelleNette](../WDLang1/3050053.md),<br>[FinValRemboursement](../WDLang1/3050039.md) |
| 12 | Nombre de remboursements infini | [FinNbVersement](../WDLang1/3050041.md) |
| 13 | Le taux n'a pas été trouvé | [FinTauxIntérêt](../WDLang1/3050047.md) |
| 14 | La période est supérieure à la durée | [FinAmortissementDégressif](../WDLang1/3050068.md) |
| 15 | Nombre de mois de la première année non compris entre 0 et 12 | [FinAmortissementDégressif](../WDLang1/3050068.md) |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Le libellé de l'erreur peut être connu grâce à la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.

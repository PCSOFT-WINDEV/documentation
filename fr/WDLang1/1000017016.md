
## SysListeCarteGraphique (Fonction)

***En anglais : SysListVideoCard***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le nom des cartes graphiques connectées au poste en cours.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Récupération des différentes cartes graphiques du PC en cours
ListeAjoute(LISTE_Ecran, SysListeCarteGraphique())
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = SysListeCarteGraphique()`
---

**`<Résultat> : (Chaîne de caractères)`**

Liste des cartes graphiques du poste en cours, chaîne vide ("") en cas d'erreur. La fonction [ErreurInfo](../WDLang1/3013008.md) permet d'obtenir plus de renseignements sur le problème rencontré.

Cette liste est du type :

```txt
<Nom de l'écran 1> + TAB + <Nom de la carte graphique 1> + RC + 
<Nom de l'écran 2> + TAB + <Nom de la carte graphique 2> + RC + ... 
+ RC + <Nom de l'écran N>+ TAB + <Nom de la carte graphique N>
```


Par exemple : 

```txt
\\.\DISPLAY1<TAB>NVIDIA GeForce 210 <\r><\n>\\.\DISPLAY2<TAB>NVIDIA GeForce 210
```




## NationDroiteAGauche (Fonction)

***En anglais : NationRightToLeft***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de savoir si la langue courante s'écrit de droite à gauche. Cette information est renseignée dans les informations de langue du projet (onglet "Langues" de la fenêtre de description du projet, sous-onglet "Divers") : <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=NationDroiteAGauche%20-%20HC%20N%B0001.gif)



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI NationDroiteAGauche() ALORS
	// Code spécifique aux langues qui s'écrivent de droite à gauche
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = NationDroiteAGauche()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la langue courante s'écrit de droite à gauche, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




## Fonctions de gestion des réseaux
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions de gestion des réseaux sont les suivantes :



|   |   |
| --- | --- |
| [RéseauConnecte](../WDLang1/3066002.md) | Associe un lecteur à un répertoire réseau partagé. |
| [RéseauDéconnecte](../WDLang1/3066003.md) | Déconnecte un lecteur réseau. |
| [RéseauNomDomaine](../WDLang1/1000018859.md) | Renvoie le nom du domaine auquel est associée la machine. |
| [RéseauNomRep](../WDLang1/3066005.md) | Identifie le répertoire partagé associé à un lecteur réseau. |
| [RéseauUtilisateur](../WDLang1/3066004.md) | Identifie le nom de l'utilisateur connecté. |

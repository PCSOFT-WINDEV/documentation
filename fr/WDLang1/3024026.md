
## Phonétique (Fonction)

***En anglais : Phonetic***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la phonétique correspondant à la chaîne passée en paramètre. Il est ainsi possible de détecter si deux chaînes de caractères sont phonétiquement semblables (selon la phonétique française).




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Comparaison de deux chaînes de caractères
MaChaîne1 est une chaîne = Phonétique("volant") 
MaChaîne2 est une chaîne = Phonétique("volent")

SI MaChaîne1 = MaChaîne2 ALORS
	Trace("Phonétique comparable")
SINON
	Trace("Phonétique différente")
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = Phonétique(<Chaîne initiale>)`
---

**`<Résultat> : (Chaîne de caractères)`**

Chaîne de caractères en pseudo-phonétique correspondant à &lt;Chaîne initiale&gt;.

**`<Chaîne initiale> : (Chaîne de caractères)`**

Chaîne de caractères à traduire en phonétique. Cette chaîne n'est pas modifiée.


## NoteSupprime (Fonction)

***En anglais : NoteDelete***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime une note repositionnable présente dans la fenêtre en cours. Cette note repositionnable peut être un champ ou une note créée par programmation avec les fonctions [FenEdCréeNote](../WDLang1/1000022270.md), [NoteCrée](../WDLang1/1000022395.md) ou [ChampCrée](../WDLang1/1000020876.md)). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Clic sur [Nouvelle note]
gMaNote <- NoteCrée()

// Clic sur [Supprimer]
SI gMaNote <> NULL ALORS
	NoteSupprime(gMaNote)
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`NoteSupprime(<Note à manipuler>)`
---

**`<Note à manipuler> : (Chaîne de caractères)`**

Nom de la note repositionnable à supprimer. Ce nom peut correspondre : 

- au nom d'un champ Note repositionnable présent dans la fenêtre. 

- à une variable de type Champ associée à une note repositionnable créée avec une des fonctions suivantes : [FenEdCréeNote](../WDLang1/1000022270.md), [NoteCrée](../WDLang1/1000022395.md) ou [ChampCrée](../WDLang1/1000020876.md).

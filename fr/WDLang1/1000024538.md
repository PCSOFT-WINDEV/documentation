


## &lt;Variable Document&gt;.Undo (Fonction)

***En anglais : &lt;Document variable&gt;.Undo***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Annule la dernière modification effectuée sur une variable de type [Document](../WDLang1/1000022461.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonDocument est un Document

// Annule la dernière action effectuée sur le document
MonDocument.Undo()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Document>.Undo()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la dernière modification a été annulée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Document> : (Variable de type Document)`**

Nom de la variable de type [Document](../WDLang1/1000022461.md) à manipuler.

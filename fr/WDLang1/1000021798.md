
## SysEtatStockageExterne (Fonction)

***En anglais : SysStatusExternalStorage***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie l'état de disponibilité de l'espace de stockage externe de l'appareil. 

Cette fonction doit être utilisée avant les opérations de lecture/écriture sur l'espace de stockage pour s'assurer que celui-ci est bien disponible.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Récupère le chemin du répertoire à utiliser pour le stockage des fichiers de l'application 
// sur l'espace de stockage externe primaire de l'application
SI SysEtatStockageExterne(1) = sseDisponible  ALORS
	RENVOYER SysRepStockageExterne(1, sseAppFichier)
SINON
	RENVOYER ""
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = SysEtatStockageExterne(<Indice>)`
---

**`<Résultat> : (Constante de type Entier)`**

Etat actuel de la carte de stockage sur l'appareil :


|   |   |
| --- | --- |
| *sseDisponible* | Il est possible de lire et d'écrire sur l'espace de stockage externe. |
| *sseIndisponible* | Il n'est pas possible de lire et d'écrire sur l'espace de stockage externe. |
| *sseLectureSeule* | Il est seulement possible de lire sur l'espace de stockage externe. |



**`<Indice> : (Entier)`**

Indice de l'espace de stockage externe à utiliser (si l'appareil possède plusieurs espaces de stockage externe). 
Remarque : La fonction [SysNbStockageExterne](../WDLang1/1000021816.md) permet de connaître le nombre d'espaces de stockage externe de l'appareil.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Un même appareil peut contenir plusieurs zones de stockage externe (par exemple, une zone correspond à la partition "externe" de la mémoire permanente et une autre zone correspond à la SDCard).

- La fonction [SysStockageExterneAmovible](../WDLang1/1000021818.md) permet de savoir si un espace de stockage correspond à de la mémoire amovible (SDCard) ou non.

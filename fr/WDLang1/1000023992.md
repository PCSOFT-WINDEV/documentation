


## PanneauPosition (Fonction)

***En anglais : PanelPosition***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Récupère la position d'attache d'un champ Panneau dockable.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Récupère la position du Champ Panneau dockable "PADO_TEST"
SELON PanneauPosition(PADO_TEST)
	CAS padoGauche
	// Le champ Panneau dockable est attaché à gauche.
	CAS padoDroite
	// Le champ Panneau dockable est attaché à droite.	
	CAS padoHaut
	// Le champ Panneau dockable est attaché en haut.
	CAS padoBas
	// Le champ Panneau dockable est attaché en bas.
	CAS padoFlottant
	// Le champ Panneau dockable est flottant. 
FIN
```

```wl
// Si le champ Panneau dockable est à gauche, le mettre à droite
SI PanneauPosition(PADO_Test) = padoGauche ALORS
	PanneauAttache(PADO_Test, padoDroite)
FIN

// Crée un configuration avec 2 champs Panneau dockable en bas dans la même zone
PanneauAttache(PADO_Détail, padoBas)
PanneauAttache(PADO_Infos, PanneauPosition(PADO_Détail))
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = PanneauPosition(<Champ Panneau dockable>)`
---

**`<Résultat> : (Chaîne de caractères ou constante)`**



- Chaîne vide ("") si le champ Panneau dockable est un panneau flottant.

- Position du champ Panneau dockable. Peut correspondre à une des constantes suivantes ou à une combinaison des constantes suivantes : 
	


|   |   |
| --- | --- |
| *padoBas* | Le champ Panneau dockable est attaché en bas |
| *padoDroite* | Le champ Panneau dockable est attaché à droite. |
| *padoFlottant* | Le champ Panneau dockable est flottant. |
| *padoGauche* | Le champ Panneau dockable est attaché à gauche. |
| *padoHaut* | Le champ Panneau dockable est attaché en haut. |





Les constantes sont combinables.

**`<Champ Panneau dockable> : (Nom de champ)`**

Nom du champ Panneau dockable à manipuler. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- La position peut être modifiée avec la fonction [PanneauAttache](../WDLang1/1000023990.md). 

- Si le champ Panneau dockable est flottant, sa position peut être connue via les propriétés [X](../Proprietes/2510135.md) et [Y](../Proprietes/2510137.md).


## fListeRessource (Fonction)

***En anglais : fListResource***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Liste les ressources intégrées à l'application.

**Remarques** : Les ressources ont été incluses dans l'application générée : 

- via [l'assistant de génération Android](../Editeurs/9000109.md),

- via [l'assistant de génération iPhone/iPad](../Editeurs/1000019908.md).







<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Extraction de tous les fichiers HFSQL de l'application
sListeRessources est une chaîne = fListeRessource()

POUR TOUTE CHAÎNE sNomRessource DE sListeRessources SEPAREE PAR RC
	sExtension est une chaîne = Minuscule(fExtraitChemin(sNomRessource, fExtension))
	SI sExtension DANS (".fic", ".mmo", ".ndx", ".ftx") ALORS
		fExtraitRessource(sNomRessource, "bdd" + [fSep()] + sNomRessource)
	FIN
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = fListeRessource()`
---

**`<Résultat> : (Chaîne de caractères)`**

Noms des ressources intégrées dans l'application, séparés par des Retours Chariots (RC).

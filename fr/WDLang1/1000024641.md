


## &lt;Variable WDPic&gt;.SymétrieVerticale (Fonction)

***En anglais : &lt;WDPic variable&gt;.VerticalSymmetry***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Retourne l'image contenue dans une variable de type [WDPic](../WDLang1/1000023503.md) (sur le calque d'arrière-plan) par rapport à un axe vertical (symétrie par rapport à un axe vertical).![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=dSymetrieVerticale.gif)


![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Les dessins](../WDLang1/3029035.md). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonImageWDPic est un WDPic = "Test.bmp"
// Symétrie verticale de l'image contenue dans la variable WDPic
MonRésultat est un booléen
MonRésultat = MonImage.SymétrieVerticale()
SI MonRésultat = Faux ALORS Erreur("L'opération a échoué")
IMG_MonDessin = MonImage
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Image WDPic>.SymétrieVerticale()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la symétrie a été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Image WDPic> : (Variable de type WDPic)`**

Nom de la variable de type [WDPic](../WDLang1/1000023503.md) à manipuler. Seul le calque d'arrière-plan sera manipulé. 

La symétrie est réalisée par rapport à l'axe vertical médian de l'image. L'image est directement modifiée.

<a name="NOTE0_1"></a>

## Remarques


### Fin du dessin
<a name="fin_dessin_ELTPARAGRAPHE000347"></a>Si la fonction [&lt;Variable WDPic&gt;.FinDessin](../WDLang1/1000024472.md) est appelée dans le même traitement que la fonction **&lt;Variable WDPic&gt;.SymétrieVerticale**, la transformation de l'image ne sera pas affichée.




## &lt;Menu&gt;.Existe (Fonction)

***En anglais : &lt;Menu&gt;.Exist***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Indique si une option de menu existe dans un menu.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Si l'option n'existe pas encore
SI MonMenu.Existe("OuvreClient" + ID) = Faux ALORS
	// Ajout de l'option
	MonMenu.AjouteOption("OuvreClient" + ID, "Ouvrir la fiche " + ID, OuvreFiche, ID)
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Menu>.Existe(<Option de menu>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'option de menu existe, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 




**Attention** : La recherche n'est pas récursive : si le menu contient un sous-menu qui contient l'option, &lt;Résultat&gt; vaut <u><u><u><u>Faux</u></u></u></u>.

**`<Menu> : (Nom de menu)`**

Nom du menu ou du menu contextuel dans lequel la recherche doit être effectuée.

**`<Option de menu> : (Chaîne de caractères)`**

Nom de l'option de menu à rechercher.


## MatRemplit (Fonction)

***En anglais : MatFill***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Initialise tous les éléments d'une matrice d'une taille donnée. Si la matrice est plus grande, seuls les éléments spécifiés seront remplis.

Par exemple :

- Initialisation de tous les éléments d'une matrice avec la valeur 5. Cette matrice est composée de 2 lignes et de 3 colonnes :
	
	```wl
	ResRemplissage = MatRemplit("MaMatrice", 5, 2, 3)
	```


<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=MATREMPLIT1.gif)


- Initialisation des éléments d'une matrice de la ligne 1 à 2 et de la colonne 1 à 3 avec la valeur 5. Seule une partie de la matrice est initialisée : 
	```wl
	ResRemplissage = MatRemplit("MaMatrice", 5, 2, 3)
	```




<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=MATREMPLIT2.gif)





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple
<a class="notetitle" target="_blank" href="$DOC$=1000003050057&name=matremplit_fonction&product=WD">Voir des exemples supplémentaires</a>

```wl
// Remplissage d'une matrice de 2 lignes, 3 colonnes avec le nombre 5
ResRemplissage = MatRemplit("MaMatrice", 5, 2, 3)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = MatRemplit(<Matrice> , <Valeur> , <Nombre de lignes> , <Nombre de colonnes>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le remplissage a été effectué, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour connaître le libellé de l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.




**`<Matrice> : (Chaîne de caractères)`**

Nom de la matrice à remplir, défini avec la fonction [MatCrée](../WDLang1/3050058.md).

**`<Valeur> : (Réel)`**

Valeur donnée à l'ensemble des éléments de la matrice.

**`<Nombre de lignes> : (Entier)`**

Nombre de lignes à remplir.

**`<Nombre de colonnes> : (Entier)`**

Nombre de colonnes à remplir.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Une matrice doit exister afin de pouvoir être remplie. Pour vérifier l'existence d'une matrice, utilisez la fonction [MatExiste](../WDLang1/3050076.md).

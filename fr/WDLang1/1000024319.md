


## Fonctions de gestion des champs Table (syntaxe préfixée)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions (syntaxe préfixée) de gestion des champs Table sont les suivantes :



|   |   |
| --- | --- |
| [&lt;Table&gt;.Affiche](../WDLang1/1000023505.md) | Rafraîchit l'affichage d'un champ Table ou Table hiérarchique à partir d'une position donnée :<br><br>- pour un champ lié à un fichier : les modifications effectuées sur le fichier de données lié sont répercutées dans le champ.<br><br>- pour un champ manipulé par programmation : les colonnes calculées sont re-calculées.<br><br><br> |
| [&lt;Table&gt;.Ajoute](../WDLang1/1000023600.md) | Ajoute une ligne dans : <br><br>- un champ Table, <br><br>- un champ Table hiérarchique, <br><br>- un champ "Combo avec table" (champ Table affiché dans un champ Combo).<br><br><br> |
| [&lt;Table&gt;.AjouteColonne](../WDLang1/1000023607.md) | Ajoute des éléments à droite d'un champ Table horizontale. |
| [&lt;Table&gt;.AjouteLigne](../WDLang1/1000023404.md) | Ajoute une ligne dans : <br><br>- un champ Table, <br><br>- un champ Table hiérarchique. <br><br><br> |
| [&lt;Table&gt;.AjouteLigneAvecAnimation](../WDLang1/1000023678.md) | Ajoute une ligne dans un champ Table ou un champ Table hiérarchique avec une animation. |
| [&lt;Table&gt;.AjoutEnCours](../WDLang1/1000023838.md) | Modifie ou renvoie l'état de visibilité de la fenêtre interne de patience lors de l'ajout d'éléments supplémentaires dans un champ Table infinie. |
| [&lt;Table&gt;.AjouteSurEntête](../WDLang1/1000023839.md) | Ajoute un sur-entête de colonnes dans un champ Table ou dans un champ Table hiérarchique d'une fenêtre. |
| [&lt;Table&gt;.Ajuste](../WDLang1/1000023840.md) | Ajuste le titre et la taille des colonnes d'un champ Table ou Table hiérarchique. |
| [&lt;Table&gt;.CelluleSelect](../WDLang1/1000023841.md) | Renvoie les coordonnées (numéro de ligne et numéro de colonne) d'une cellule sélectionnée. Cette fonction est conservée par compatibilité. Il est conseillé d'utiliser la fonction TableSelect. |
| [&lt;Table&gt;.CelluleSelectOccurrence](../WDLang1/1000023842.md) | Renvoie le nombre de cellules sélectionnées dans un champ Table. Cette fonction est conservée par compatibilité. Il est conseillé d'utiliser la fonction TableSelectOccurrence. |
| [&lt;Table&gt;.CherchePartout](../WDLang1/1410088374.md) | Recherche une valeur dans toutes les colonnes des champs suivants : <br><br>- champ Table,<br><br>- champ Table hiérarchique,<br><br>- champ Table affiché dans un champ Combo.<br><br><br> |
| [&lt;Table&gt;.ColonnesFiltrées](../WDLang1/1000024053.md) | Renvoie la liste des colonnes filtrées par l'utilisateur dans un champ Table ou Table hiérarchique. |
| [&lt;Table&gt;.ColonnesTriées](../WDLang1/1000024054.md) | Renvoie la liste des colonnes triées d'une table. |
| [&lt;Table&gt;.DébutRemplissage](../WDLang1/1000024055.md) | Indique le début du remplissage d'un champ Table (ou Table hiérarchique) par programmation. |
| [&lt;Table&gt;.DéplaceColonne](../WDLang1/1000024244.md) | Déplace une colonne d'un champ Table ou Table hiérarchique. |
| [&lt;Table&gt;.DéplaceLigne](../WDLang1/1000024245.md) | Déplace une ligne ou échange deux lignes dans un champ Table. |
| [&lt;Table&gt;.DésactiveFiltre](../WDLang1/1000024246.md) | Annule un filtre utilisateur sur une colonne ou toutes les colonnes d'un champ Table ou Table hiérarchique. |
| [&lt;Table&gt;.DonneFocus](../WDLang1/1000024247.md) | Donne le focus à une ligne d'un champ Table ou Table hiérarchique : le champ passe en saisie pour la ligne et la colonne spécifiée. |
| [&lt;Table&gt;.EchangeLigne](../WDLang1/1000024248.md) | Echange : <br><br>- deux lignes dans un champ Table par programmation. <br><br>- deux branches dans un champ Table hiérarchique par programmation.<br><br><br> |
| [&lt;Table&gt;.Enregistre](../WDLang1/1000024249.md) | Modifie ou ajoute l'enregistrement associé à la ligne en cours dans le champ Table fichier ou Table hiérarchique fichier. |
| [&lt;Table&gt;.EnrouleDéroule](../WDLang1/1000024802.md) | Enroule ou déroule un élément d'un champ Table ou Table hiérarchique. |
| [&lt;Table&gt;.EnumèreColonne](../WDLang1/1000024250.md) | Renvoie le nom d'une colonne d'un champ Table ou Table hiérarchique. |
| [&lt;Table&gt;.EnumèreRupture](../WDLang1/1000024251.md) | Enumère les ruptures d'un champ Table. |
| [&lt;Table&gt;.Fin](../WDLang1/1000024294.md) | Arrête le parcours manuel du fichier de données lié à un champ Table fichier. |

| [&lt;Table&gt;.IndiceRupture](../WDLang1/1000024258.md) | Renvoie l'indice du haut et du bas de rupture d'une rupture donnée dans un champ Table. |
| [&lt;Table&gt;.InfoXY](../WDLang1/1000024259.md) | Renvoie pour une position donnée dans un champ Table ou Table hiérarchique (coordonnées d'un point du champ) :<br><br>- soit le nom de la colonne affichée pour la position indiquée.<br><br>- soit l'indice de la ligne ou de la colonne pour la position indiquée.<br><br><br> |
| [&lt;Table&gt;.Insère](../WDLang1/1000024260.md) | Insère une ligne dans : <br><br>- un champ Table, <br><br>- un champ Table hiérarchique,<br><br>- une table affichée dans un champ Combo.<br><br><br> |
| [&lt;Table&gt;.InsèreColonne](../WDLang1/1000024261.md) | Insère des éléments sous forme de colonne dans ***un champ Table horizontale***. |

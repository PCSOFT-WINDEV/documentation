
## AgendaZoom (Fonction)

***En anglais : OrganizerZoom***






	<B><B>Avertissement</B></B>

	A partir de la version <B><B>17</B></B>,  cette fonction est conservée par compatibilité. Cette fonction est remplacée par la fonction [AgendaMode](../WDLang1/1000019545.md).



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la plage temporelle affichée par un champ Agenda : jour, semaine, mois, ...

**Remarque** : La fonction [AgendaChangeZoom](../WDLang1/1000018883.md) permet de modifier la plage temporelle affichée par le champ Agenda.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Alterne entre l'affichage par jour et par mois
SI AgendaZoom(AGD_MonAgenda) = agzJour ALORS
	AgendaChangeZoom(AGD_MonAgenda, agzMois)
SINON
	AgendaChangeZoom(AGD_MonAgenda, agzJour)
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = AgendaZoom(<Champ Agenda>)`
---

**`<Résultat> : (Constante de type Entier)`**

Plage de temps affichée par le champ Agenda :


|   |   |
| --- | --- |
| *agzJour* | L'agenda affiche une journée. |
| *agzMois* | L'agenda affiche un mois complet. |
| *agzSemaine* | L'agenda affiche une semaine complète (ou partielle si un nombre de jours a été précisé grâce à la fonction [AgendaChangeZoom](../WDLang1/1000018883.md)). |
| *agzSemaineSur5* | L'agenda affiche une semaine sur 5 jours (sans le week-end). |



**`<Champ Agenda> : (Nom de champ)`**

Nom du champ Agenda à manipuler. Ce champ peut correspondre : 

- au champ Agenda d'une fenêtre ou d'une page, 

- au champ Agenda d'un état.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Les lignes de code suivantes sont équivalentes : 


```wl
AgendaZoom(AGD_Agenda, agzSemaine, 5)
```


et 


```wl
AgendaZoom(AGD_Agenda, agzSemaineSur5)
```


Dans les 2 cas, la fonction **AgendaZoom** renvoie la constante *agzSemaineSur5*.

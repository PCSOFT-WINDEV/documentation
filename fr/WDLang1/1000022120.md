
## FenEdSauveNote (Fonction)

***En anglais : WinEdSaveNote***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Exporte les notes repositionnables créées par l'utilisateur dans un fichier. Ces notes peuvent être ensuite importées : 

- directement sous l'éditeur. 

- via la fonction [FenEdCharge](../WDLang1/1000021849.md). 



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Clic sur [Exporter mes notes]
SI FenEdSauveNote("Export.wdnote") = Faux ALORS
	ErreurInfo()
FIN

// Clic sur [Importer mes notes]
FenEdCharge(FEN_FenêtreNote, "Export.wdnote")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = FenEdSauveNote(<Fichier de sauvegarde> [, <Fenêtre>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si les notes repositionnables ont été sauvegardées, 

- <u><u><u><u>Faux</u></u></u></u> en cas d'erreur ou si aucune note n'a été créée. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).


Remarque : Si aucune note n'a été créée, la variable [ErreurDétectée](../WDLang1/3087001.md) vaut <u><u><u><u>Faux</u></u></u></u>. 

**`<Fichier de sauvegarde> : (Chaîne de caractères)`**

Nom du fichier dans lequel la sauvegarde doit être réalisée. Ce fichier a pour extension "wdnote". 

**`<Fenêtre> : (Nom de fenêtre)`**

Nom de la fenêtre ou de la fenêtre interne contenant les notes à enregistrer. Si ce paramètre n'est pas précisé, toutes les notes créées par l'utilisateur dans l'application sont exportées. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
La fonction **FenEdSauveNote** est équivalente à la fonctionnalité automatique d'export de la note (FAA). La seule différence est que le sélecteur de fichiers n'est pas ouvert.

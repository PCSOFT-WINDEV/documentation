


## &lt;Caméra&gt;.DémarreDécodage (Fonction)

***En anglais : &lt;Camera&gt;.StartDecoding***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Démarre le décodage des codes-barres dans un champ Caméra. Il est possible de spécifier le type des codes-barres à décoder.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Démarre le décodage des codes-barres de type QRCode dans le champ Caméra
CAM_Caméra1.DémarreDécodage(CB_QRCODE)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Champ Caméra>.DémarreDécodage([<Type de code-barres>])`
---

**`<Champ Caméra> : (Nom de champ)`**

Nom du champ Caméra à manipuler.

**`<Type de code-barres> : (Constante de type Entier optionnelle)`**

Type de codes-barres pouvant être décodé : 


|   |   |
| --- | --- |
| *CB_AZTEC* | Code-barres de type Aztec. <br> |
| *CB_CODE128* | Code-barres de type Code128.<br> |
| *CB_CODE39* | Code-barres de type Code39.<br> |
| *CB_CODE93* | Code-barres de type Code93.<br> |
| *CB_DATAMATRIX* | Code-barres de type DATAMATRIX.<br> |
| *CB_EAN13* | Code-barres de type EAN13.<br> |
| *CB_EAN8* | Code-barres de type EAN8.<br> |
| *CB_INTER2OF5* | Code-barres de type Interleaved 2 of 5.<br> |
| *CB_PDF417* | Code-barres de type PDF417.<br> |
| *CB_QRCODE* | Code-barres de type QR Code. |
| *CB_UPCA* | Code-barres de type UPC-A.<br> |
| *CB_UPCE* | Code-barres de type UPC-E.<br> |
| *CB_TOUS* | Tous les types de codes-barres supportés par cette fonction pourront être décodés. |


Si ce paramètre n'est pas spécifié : 

- les codes-barres spécifiés lors du dernier appel à la fonction **&lt;Caméra&gt;.DémarreDécodage** seront décodés. 

- les codes-barres spécifiés dans la fenêtre de description du champ Caméra seront décodés.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Le champ Caméra doit avoir été configuré pour autoriser le décodage des codes-barres (dans la fenêtre de description). Si ce n'est pas le cas, l'appel de la fonction **&lt;Caméra&gt;.DémarreDécodage** déclenchera une erreur fatale.

- Lorsque le champ Caméra est configuré pour autoriser le décodage des codes-barres (dans la fenêtre de description), le décodage démarre automatiquement à l'ouverture de la fenêtre. Pour empêcher ce comportement, il est possible d'appeler la fonction [&lt;Caméra&gt;.ArrêteDécodage](../WDLang1/1000022474.md) dans l'événement "Initialisation" du champ Caméra.

- Si la fonction **&lt;Caméra&gt;.DémarreDécodage** est appelée alors qu'un décodage est déjà en cours dans le champ Caméra, le décodage est arrêté puis relancé pour prendre en compte le nouveau type de code-barres à décoder. Si le type de code-barres est identique, la fonction est sans effet.

- Lorsqu'un code-barres est détecté, l'événement "Décodage d'un code-barres" du champ Caméra est appelé avec en paramètre une variable de type [CodeBarres](../WDLang5/1000019451.md) correspondant au code-barres détecté. Si cet événement renvoie <u><u><u><u>Vrai</u></u></u></u>, le décodage de code-barres sera automatiquement arrêté.

- Cette fonction est équivalente à la fonction [CaméraDémarreDécodage](../WDLang1/1000022454.md).

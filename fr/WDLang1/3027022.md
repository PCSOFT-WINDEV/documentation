
## Manipuler des dates et des heures 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000128"></a>
WINDEV, WEBDEV et WINDEV Mobile mettent à votre disposition de nombreux moyens pour manipuler les dates et les heures dans vos applications :

- Les champs de saisie de type Date ou de type Heure.

- Les rubriques de fichiers de données de type Date ou de type Heure.

- Les variables de type [Date](../Motscles/1514066.md), [Heure](../Motscles/1514068.md), [DateHeure](../Motscles/1514070.md) et [Durée](../Motscles/1514069.md).

- Les propriétés sur les variables de type Date, Heure, DateHeure et Durée.

- Les [fonctions du WLangage de gestion des dates et des heures](../WDLang1/3027029.md).






<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Différentes façons de manipuler les dates et les heures
<a name="differentes_facons_manipuler_les_dates_les_heures_ELTTEXTE000152"></a>
Selon le type de manipulation effectuée sur les dates et les heures, il peut être plus efficace d'utiliser les variables ou les fonctions du WLangage.

Les fonctions du WLangage sont conseillées :

- Pour le formatage des dates et des heures ainsi que pour les opérations de conversion (conversion d'une date en entier, ...).

- Pour obtenir des informations sur les dates et les heures (validité de la date ou de l'heure, obtenir le jour en toutes lettres, ...).




L'utilisation des variables est conseillée : 

- Pour tous les calculs effectués sur les dates et les heures (différence, calcul de durée, ...).

- Pour les extractions de partie de date ou d'heure (jour, mois, année, ...).





<a name="NOTE2_2"></a>


### Manipulation et validité des dates
<a name="manipulation_validite_des_dates_ELTPARAGRAPHE000063"></a>Le format de stockage des dates permet de stocker des dates comprises entre le 01/01/0001 et le 31/12/9999.

Les calculs sur les dates réalisés avec les [fonctions WLangage](../WDLang1/3027029.md) ou les [propriétés WLangage](../Proprietes/2514010.md) sont fiables à partir du 1er Janvier 1583.

## Manipuler les dates ou les heures présentes dans des champs de saisie
<a name="manipuler_les_dates_les_heures_presentes_dans_des_champs_saisie_ELTTEXTE000192"></a>
Il est possible de manipuler les valeurs présentes dans les champs de saisie avec les variables de type Date ou Heure.

Il suffit d'affecter la variable avec le contenu du champ.


```wl
DateDébut est une Date
// Affectation de la variable avec le contenu du champ
DateDébut = SAI_ChampDate
```

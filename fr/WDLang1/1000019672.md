


## dIncrusteCouleur (Fonction)

***En anglais : dEncrustColor***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Incruste une couleur dans une image contenue dans une variable de type [Image](../WDLang1/1000019650.md). La couleur est appliquée au contenu de l'image. L'utilisation de l'opacité permet d'obtenir des effets spécifiques. 

![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Les dessins](../WDLang1/3029035.md). 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Utilisation d'une variable de type Image
MonImage est une Image
MonImage = "Image.png"
// Incruste la couleur Jaune bulle avec 5% d'opacité
dIncrusteCouleur(MonImage, JauneBulle, 5)
// Affiche l'image dans un champ Image
IMG_ImagePaysage = MonImage
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`dIncrusteCouleur(<Image> , <Couleur à incruster> [, <Opacité>])`
---

**`<Image> : (Variable de type Image)`**

Variable de type [Image](../WDLang1/1000019650.md) correspondant à l'image à manipuler. 

**`<Couleur à incruster> : (Entier)`**

Couleur à incruster dans l'image. Cette couleur peut correspondre à :

- une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)),

- une couleur TSL (obtenue avec la fonction [TSL](../WDLang1/3029057.md)),

- [une couleur prédéfinie du WLangage](../WDLang5/3010002.md).




**`<Opacité> : (Réel optionnel)`**

Pourcentage d'opacité utilisée pour la couleur incrustée (100% par défaut, aucune opacité n'est appliquée).

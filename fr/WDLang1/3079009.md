


## VidéoConfigure (Fonction)

***En anglais : VideoConfigure***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de configurer le driver de capture vidéo. Cette configuration peut être effectuée par l'utilisateur (à l'aide d'une boîte de dialogue) ou directement par programmation (configuration de la résolution).

Attention : Cette fonction ne peut être utilisée que avec les champs Caméra en mode compatible (option "Utiliser les anciennes API 'Video for Windows' (compatibilité)" cochée dans la description du champ). 

Remarque : Cette fonction est équivalente à la fonction [CaméraConfigure](../WDLang1/1000022452.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affiche la boîte de dialogue standard permettant la configuration du driver de capture vidéo 
// utilisé pour le champ "CAM_MaVidéo"
VidéoConfigure(CAM_MaVidéo)
```
<a name="Exemple2"></a>

```wl
// Spécifie une résolution de 640 par 480
VidéoConfigure(CAM_VIDEO1, viResolution, 640, 480)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = VidéoConfigure(<Champ Caméra> [, <Type de configuration> [, <Résolution Largeur> , <Résolution Hauteur>]])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la boîte de dialogue standard s'est affichée, 

- <u><u><u><u>Faux</u></u></u></u> en cas d'erreur. Le message d'erreur correspondant peut être obtenu avec la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Champ Caméra> : (Nom de champ)`**

Nom du champ Caméra à manipuler.

**`<Type de configuration> : (Constante optionnelle de type entier)`**

Type de configuration à effectuer :


|   |   |
| --- | --- |
| *viConfigureFormat* | Affichage d'une boîte de dialogue (si elle existe) permettant à l'utilisateur de spécifier la résolution de la caméra, le nombre de couleurs utilisées, ...![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=FormatVideo.gif)<br> |
| *viConfigureSource*<br>(valeur par défaut) | Affichage d'une boîte de dialogue (si elle existe) permettant à l'utilisateur de spécifier la caméra à utiliser et les paramètres de cette caméra (saturation par exemple).<br>Cette boîte de dialogue dépend du driver utilisé par le champ Caméra.![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=SourceVideo.gif)<br> |
| *viRésolution* | Permet de spécifier une résolution pour la caméra, à l'aide des paramètres &lt;Résolution Largeur&gt; et &lt;Résolution Hauteur&gt;. |



**`<Résolution Largeur> : (Entier optionnel)`**

Largeur en pixels de la résolution de la caméra. Ce paramètre doit être spécifié si &lt;Type de configuration&gt; vaut *viRésolution*.

**`<Résolution Hauteur> : (Entier optionnel)`**

Hauteur en pixels de la résolution de la caméra. Ce paramètre doit être spécifié si &lt;Type de configuration&gt; vaut *viRésolution*.

<a name="NOTE0_1"></a>

## Remarques


### Fonctionnement du champ Caméra pendant l'affichage des boîtes de dialogue
<a name="fonctionnement_champ_camera_pendant_affichage_des_boites_dialogue_ELTPARAGRAPHE000112"></a>

- Si le panneau de configuration de format est affiché, le champ Caméra est figé.

- Si le panneau de configuration est affiché par la constante *viConfigureSource*, le champ Caméra continue à fonctionner. Les modifications de paramètres effectuées sont prises en compte immédiatement et se voient dans le champ Caméra en "direct live".

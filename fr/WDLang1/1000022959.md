


## dCouleurDominanteImage (Fonction)

***En anglais : dDominantImageColor***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la couleur dominante (c'est-à-dire la plus fréquente) d'une image présente dans une variable de type [Image](../WDLang1/1000019650.md) ou dans un champ Image.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MaCouleur est une Couleur = dCouleurDominanteImage(IMG_MonImage)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = dCouleurDominanteImage(<Image>)`
---

**`<Résultat> : (Entier)`**

Couleur dominante de l'image. 

**`<Image> : (Chaîne de caractères ou variable de type Image)`**

Image à analyser. Ce paramètre peut correspondre :

- soit au nom d'un champ Image,

- soit au nom d'une variable de type [Image](../WDLang1/1000019650.md).






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Les pixels transparents ou semi-transparents de l'image sont ignorés.

- La fonction **dCouleurDominanteImage** ne fonctionne pas sur des images vectorielles (SVG, PDF, ...).




## TreeMapInfoXY (Fonction)

***En anglais : TreeMapInfoXY***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie l'indice de l'élément affiché à une position donnée dans un champ TreeMap.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// -- Code optionnel de survol souris pour le champ TMAP_TreeMap1
ÉlémentSurvolé est un entier
ÉlémentSurvolé = TreeMapInfoXY(TMAP_TreeMap1, SourisPosX(), SourisPosY())
// Change la bulle d'aide de l'élément
SI ÉlémentSurvolé > 0 ALORS
	TMAP_TreeMap1[ÉlémentSurvolé].Bulle = ...
		"Elément survolé : " + TMAP_TreeMap1[ÉlémentSurvolé].Libellé
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = TreeMapInfoXY(<Champ TreeMap> , <X> , <Y>)`
---

**`<Résultat> : (Entier)`**



- Indice de l'élément affiché à la position spécifiée. 

- -1 si aucun élément ne correspond à la position spécifiée.




**`<Champ TreeMap> : (Nom de champ)`**

Nom du champ TreeMap à manipuler.
Si ce paramètre correspond à une chaîne vide (""), le champ TreeMap manipulé sera le champ TreeMap auquel appartient l'événement en cours.

**`<X> : (Entier)`**

Coordonnée X (en pixels) à analyser. Cette coordonnée est relative au champ.

**`<Y> : (Entier)`**

Coordonnée Y (en pixels) à analyser. Cette coordonnée est relative au champ.

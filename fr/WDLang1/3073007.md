
## SysFenHandle (Fonction)

***En anglais : SysWinHandle***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le numéro de handle d'une fenêtre identifiée par son titre. Cette fenêtre n'a pas été créée par WINDEV.

**Remarque** : Pour connaître le numéro de handle d'une fenêtre WINDEV, utilisez la fonction [Handle](../WDLang1/3015005.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connaître le numéro de handle de la fenêtre "Ventes 2011 - Microsoft Word.DOC"
ResHandleFen est un entier
ResHandleFen = SysFenHandle("Ventes 2011 - Microsoft Word.DOC")

SI ResHandleFen = Null ALORS Trace("Fenêtre non trouvée")
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = SysFenHandle(<Titre de la fenêtre>)`
---

**`<Résultat> : (Entier)`**



- Handle de la fenêtre spécifiée. 

- Constante Null ou 0 si la fenêtre n'a pas été trouvée.




**`<Titre de la fenêtre> : (Chaîne de caractères)`**

Titre de la fenêtre à manipuler. Ce paramètre doit être identique au titre apparaissant dans la barre de titre de la fenêtre.

<a name="NOTE0_1"></a>

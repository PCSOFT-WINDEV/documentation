
## PanneauChargeConfiguration (Fonction)

***En anglais : PanelLoadConfiguration***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Restaure la configuration des champs Panneau dockable d'une fenêtre. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Sauve la configuration courante
sConf est une chaîne = PanneauSauveConfiguration()


// Restaure la configuration sauvée dans sConf
PanneauChargeConfiguration(sConf)
SI ErreurDétectée ALORS ToastAffiche(ErreurInfo())
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = PanneauChargeConfiguration(<Configuration>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la configuration a été chargée.

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Configuration> : (Chaîne de caractères)`**

Configuration à restaurer. Cette configuration est le résultat de la fonction [PanneauSauveConfiguration](../WDLang1/1000023994.md).

Attention : Si le format de ce paramètre est invalide ou ne peut pas être appliqué au champ Panneau dockable, &lt;Résultat&gt; vaut <u><u><u><u>Faux</u></u></u></u>.

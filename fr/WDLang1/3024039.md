
## Majuscule (Fonction)

***En anglais : Upper***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Convertit une chaîne de caractères en majuscules en fonction des paramètres de localisation définis dans Windows.

**Remarque** : La fonction **Majuscule** est conservée par compatibilité et fonctionne uniquement avec les alphabets latins. Pour paramétrer de manière plus précise le formatage d'une chaîne ou manipuler d'autres alphabets, utilisez la fonction [ChaîneFormate](../WDLang1/3024019.md).






<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
Majuscule("abcd")   // Renvoie "ABCD"
Majuscule("pépère")  // Renvoie "PEPERE"
Majuscule("ça !")   // Renvoie "ÇA !"
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = Majuscule(<Chaîne de caractères à convertir>)`
---

**`<Résultat> : (Chaîne de caractères)`**

Chaîne de caractères en majuscules.

**`<Chaîne de caractères à convertir> : (Chaîne de caractères)`**

Chaîne de caractères à passer en majuscules.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Règles de conversion
<a name="regles_conversion_ELTPARAGRAPHE000089"></a>

- Les lettres minuscules non accentuées sont converties en majuscules non accentuées.

- Les lettres minuscules accentuées sont converties en majuscules non accentuées. 
	

- Les lettres majuscules non accentuées restent en majuscules non accentuées.

- Les lettres majuscules accentuées restent en majuscules accentuées.




**Remarque** : La fonction **Majuscule** est conservée par compatibilité et fonctionne uniquement avec les alphabets latins. Pour paramétrer de manière plus précise le formatage d'une chaîne ou manipuler d'autres alphabets, utilisez la fonction [ChaîneFormate](../WDLang1/3024019.md).


<a name="NOTE0_2"></a>




### Divers
<a name="divers_ELTPARAGRAPHE000107"></a>

- Les "blancs" sont conservés.

- Le moteur HFSQL effectue automatiquement des recherches sans tenir compte de la casse. La fonction **Majuscule** est donc inutile pour effectuer ce type d'opération.

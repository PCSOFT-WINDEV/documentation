


## Fonctions de gestion des champs TreeMap
			



Les fonctions de gestion des champs TreeMaps sont les suivantes : 



|   |   |
| --- | --- |
| [TreeMapAjoute](../WDLang1/1000018814.md) | Ajoute un élément dans un champ TreeMap. |
| [TreeMapCherche](../WDLang1/1000019739.md) | Recherche l'indice d'un élément dans un champ TreeMap à partir du chemin de l'élément. |
| [TreeMapInfoXY](../WDLang1/1000018899.md) | Renvoie l'indice de l'élément affiché à une position donnée dans un champ TreeMap. |
| [TreeMapOccurrence](../WDLang1/1000018928.md) | Renvoie le nombre d'éléments présents dans un champ TreeMap. |
| [TreeMapRecherche](../WDLang1/1000018901.md) | Recherche l'indice d'un élément dans un champ TreeMap à partir du chemin de l'élément. |
| [TreeMapSelectGroupe](../WDLang1/1000018902.md) | Sélectionne visuellement un groupe d'éléments dans un champ TreeMap. |
| [TreeMapSupprime](../WDLang1/1000018903.md) | Supprime un élément dans un champ TreeMap. |
| [TreeMapSupprimeTout](../WDLang1/1000018914.md) | Supprime tous les éléments d'un champ TreeMap. |






- ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=LechampTreemap.gif) ***Exemples unitaires (WINDEV)*** : **Le champ Treemap** <br>Utilisation du champ Treemap.<br>Le champ TreeMap permet de représenter dynamiquement des données, éventuellement arborescentes, de manière pondérée.<br>Ce type de champ est très utile notamment pour permettre une visualisation intuitive de volumes de données. <br><br>Par exemple, dans l'exemple, il permet de visualiser le contenu d'un répertoire et d'analyser la taille des fichiers et sous-répertoires qu'il contient.

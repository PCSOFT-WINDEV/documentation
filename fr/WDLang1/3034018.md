
## Constantes de gestion des erreurs et des exceptions
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


| Constante | Description | Utilisée par les fonctions |
| --- | --- | --- |
| **eaFinProgramme** | Manipulation effectuée par l'utilisateur sur la fenêtre standard des exceptions : L'utilisateur n'a pas coché l'option "Relancer l'application". | [ExceptionAffiche](../WDLang1/1000019408.md) |
| **eaRelanceProgramme** | Manipulation effectuée par l'utilisateur sur la fenêtre standard des exceptions : L'utilisateur a coché l'option "Relancer l'application". | [ExceptionAffiche](../WDLang1/1000019408.md) |
| **epAfficheMessage** | Affiche le message d'erreur dans une fenêtre en proposant à l'utilisateur les choix suivants "Réessayer", "Annuler" et "Arrêter". | [ErreurChangeParamètre](../WDLang1/3034009.md) |
| **epAfficheMessageFinProgramme** | Affiche le message d'erreur dans une fenêtre en proposant à l'utilisateur les choix "Arrêter" et "Relancer l'application" (Arrêter par défaut). | [ErreurChangeParamètre](../WDLang1/3034009.md) |
| **epAfficheMessageFinTraitement** | Affiche le message d'erreur dans une fenêtre en proposant à l'utilisateur les choix "Annuler" et "Arrêter". | [ErreurChangeParamètre](../WDLang1/3034009.md) |
| **epAfficheMessageRelanceProgramme** | Affiche le message d'erreur dans une fenêtre en proposant à l'utilisateur les choix "Arrêter" et "Relancer l'application" (Relancer par défaut). | [ErreurChangeParamètre](../WDLang1/3034009.md) |
| **epCode** | Mode actuel : l'erreur est traitée par programmation (test de la valeur de retour des fonctions, utilisation de [ErreurDétectée](../WDLang1/3087001.md), ...)<br>Fonctionnement classique des exceptions. | [ErreurChangeParamètre](../WDLang1/3034009.md),<br>[ExceptionChangeParamètre](../WDLang1/3034007.md) |
| **epExécuteProcédure** | Exécute la procédure indiquée en paramètre. | [ErreurChangeParamètre](../WDLang1/3034009.md),<br>[ExceptionChangeParamètre](../WDLang1/3034007.md) |
| **epExécuteProcédureGénérale** | Exécute la procédure indiquée en paramètre. La procédure est appelée si aucun traitement d'exception local n'est actif.<br>La procédure est active au niveau du contexte d'exécution (projet, composant externe autonome, composant interne autonome). S'il n'y a pas de procédure dans le contexte courant, la procédure est recherchée dans le contexte parent et ceci récursivement jusqu'au niveau de l'application. | [ExceptionChangeParamètre](../WDLang1/3034007.md) |
| **epGotoCasErreur** | Exécute le code défini par l'étiquette : CAS ERREUR: | [ErreurChangeParamètre](../WDLang1/3034009.md) |
| **epGotoCasException** | Exécute le code défini par l'étiquette : CAS EXCEPTION: | [ExceptionChangeParamètre](../WDLang1/3034007.md) |
| **epRenvoyer** | Valide les erreurs. Sort du traitement en cours en renvoyant la valeur indiquée.<br>Sort du traitement en cours en renvoyant la valeur spécifiée en paramètre. L'exception n'est pas propagée. | [ErreurChangeParamètre](../WDLang1/3034009.md),<br>[ExceptionChangeParamètre](../WDLang1/3034007.md) |
| **epRenvoyerErreur** | Valide les erreurs. Sort du traitement en cours en renvoyant la valeur indiquée et en propageant l'erreur.<br>L'erreur est alors traitée par le niveau supérieur de manière automatique (si le traitement d'erreur est activé) ou grâce à un SI PAS xxx ou un SI [ErreurDétectée](../WDLang1/3087001.md). | [ErreurChangeParamètre](../WDLang1/3034009.md) |
| **epRetour** | Valide les erreurs. Sort du traitement en cours.<br>Si la gestion automatique des erreurs est activée dans le code appelant, l'erreur n'est pas remontée.<br>Termine le traitement en cours sans propager l'exception. | [ErreurChangeParamètre](../WDLang1/3034009.md),<br>[ExceptionChangeParamètre](../WDLang1/3034007.md) |
| **epRetourErreur** | Valide les erreurs. Sort du traitement en cours en propageant l'erreur. L'erreur est alors traitée par le niveau supérieur de manière automatique (si le traitement d'erreur est activé) ou grâce à un SI PAS xxx ou un SI [ErreurDétectée](../WDLang1/3087001.md).<br>Si la gestion automatique des erreurs est activée dans le code appelant, l'erreur est remontée. | [ErreurChangeParamètre](../WDLang1/3034009.md) |
| **errCode** | Code de l'erreur | [ErreurInfo](../WDLang1/3013008.md), [ExceptionInfo](../WDLang1/3034006.md) |
| **errCodeSystème** | Numéro de l'erreur système (si elle existe) | [ErreurInfo](../WDLang1/3013008.md), [ExceptionInfo](../WDLang1/3034006.md) |
| **errComplet** | Toutes les informations de l'erreur (ligne de l'erreur, traitement en cours, ...) | [ErreurInfo](../WDLang1/3013008.md), [ExceptionInfo](../WDLang1/3034006.md) |
| **errDateHeure** | Date et heure de l'erreur | [ErreurInfo](../WDLang1/3013008.md), [ExceptionInfo](../WDLang1/3034006.md) |
| **errElément** | Nom de l'élément associé au traitement en cours lors de l'erreur | [ErreurInfo](../WDLang1/3013008.md), [ExceptionInfo](../WDLang1/3034006.md) |
| **errExceptionDotNet** | Objet de l'exception DotNet | [ExceptionInfo](../WDLang1/3034006.md) |
| **errFIC** | Nom complet du fichier de données (.FIC) associé à l'erreur | [ErreurInfo](../WDLang1/3013008.md), [ExceptionInfo](../WDLang1/3034006.md) |
| **errFichier** | Nom logique du fichier associé à l'erreur | [ErreurInfo](../WDLang1/3013008.md), [ExceptionInfo](../WDLang1/3034006.md) |
| **errInfo** | Informations complémentaires | [ErreurInfo](../WDLang1/3013008.md), [ExceptionInfo](../WDLang1/3034006.md) |
| **errInfoDébug** | Informations spécifiques au débogage du problème rencontré. Ces informations ne doivent être affichées et communiquées qu'en cas de demande expresse de PC SOFT. | [ErreurInfo](../WDLang1/3013008.md) |
| **errLiaison** | Nom de la liaison associée à l'erreur | [ErreurInfo](../WDLang1/3013008.md), [ExceptionInfo](../WDLang1/3034006.md) |
| **errLigne** | Numéro de la ligne de l'erreur | [ErreurInfo](../WDLang1/3013008.md), [ExceptionInfo](../WDLang1/3034006.md) |
| **errMessage** | Message complet de l'erreur | [ErreurInfo](../WDLang1/3013008.md), [ExceptionInfo](../WDLang1/3034006.md) |
| **errMessageSystème** | Message d'erreur système (si elle existe) | [ErreurInfo](../WDLang1/3013008.md), [ExceptionInfo](../WDLang1/3034006.md) |
| **errMMO** | Nom complet du fichier mémo (.MMO) associé à l'erreur | [ErreurInfo](../WDLang1/3013008.md), [ExceptionInfo](../WDLang1/3034006.md) |
| **errNDX** | Nom complet du fichier d'index (.NDX) associé à l'erreur | [ErreurInfo](../WDLang1/3013008.md), [ExceptionInfo](../WDLang1/3034006.md) |
| **errNombreSousErreur** | Nombre de sous-erreurs de l'erreur | [ErreurInfo](../WDLang1/3013008.md), [ExceptionInfo](../WDLang1/3034006.md) |
| **errNomCompletComposant** | Nom du composant en cours d'exécution :<br><br>- "" aucun composant.<br><br>- "Composant1" : Composant "Composant1" utilisé directement dans le projet.<br><br>- "Composant1.Composant2" : Composant "Composant2" utilisé depuis le "Composant1".<br><br><br> | [ErreurInfo](../WDLang1/3013008.md), [ExceptionInfo](../WDLang1/3034006.md) |
| **errNumEnr** | Numéro de l'enregistrement associé à l'erreur | [ErreurInfo](../WDLang1/3013008.md), [ExceptionInfo](../WDLang1/3034006.md) |
| **errOrigine** | HFSQL Client/Serveur : permet d'accéder directement à la première erreur qui s'est déclenchée sur le serveur HFSQL. | [ExceptionInfo](../WDLang1/3034006.md) |
| **errPendant** | Nom de la fonction WLangage qui a provoqué l'erreur | [ErreurInfo](../WDLang1/3013008.md), [ExceptionInfo](../WDLang1/3034006.md) |

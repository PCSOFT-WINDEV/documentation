


## PanneauDétache (Fonction)

***En anglais : PanelDetach***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Détache un champ Panneau dockable pour le transformer en une fenêtre flottante. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Détacher le champ Panneau dockable et le met en haut à gauche de l'écran
PanneauDétache(PADO_Test)
PADO_Test.X = 0
PADO_Test.Y = 0
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`PanneauDétache(<Champ Panneau dockable>)`
---

**`<Champ Panneau dockable> : (Nom de champ)`**

Nom du champ Panneau dockable à manipuler. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Si le champ Panneau dockable est déjà flottant, la fonction **PanneauDétache** n'a aucun effet. 

- Le champ Panneau dockable peut être réattaché à sa fenêtre avec la fonction [PanneauAttache](../WDLang1/1000023990.md).

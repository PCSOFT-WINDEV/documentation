
## FIListeSupprimeTout (Fonction)

***En anglais : IWListDeleteAll***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime toutes les fenêtres internes ajoutées par programmation dans la liste des fenêtres internes parcourue par un champ Fenêtre Interne..

**Remarque** : Si une fenêtre interne est définie en édition pour le champ Fenêtre Interne, cette fenêtre s'affichera après l'appel à cette fonction.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
FIListeSupprimeTout(FI_ChampFenêtreInterne)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`FIListeSupprimeTout(<Champ Fenêtre Interne>)`
---

**`<Champ Fenêtre Interne> : (Nom de champ)`**

Nom du champ Fenêtre Interne à manipuler.

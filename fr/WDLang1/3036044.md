
## fRépertoireTemp (Fonction)

***En anglais : fTempPath***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le nom du répertoire utilisé pour stocker les fichiers temporaires.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple
<a class="notetitle" target="_blank" href="$DOC$=1000003036044&name=frepertoiretemp_fonction&product=WD">Voir des exemples supplémentaires</a>

```wl
// Récupération du nom du répertoire temporaire
ResRepTemp = fRépertoireTemp()
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = fRépertoireTemp()`
---

**`<Résultat> : (Chaîne de caractères)`**



- Chemin du répertoire utilisé pour stocker les fichiers temporaires (260 caractères maximum). Ce chemin se termine par le caractère "\\". 
	

- Chaîne vide ("") en cas d'erreur. Pour connaître le libellé de l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.   


**Remarque** : Le résultat de la fonction **fRépertoireTemp** est toujours une chaîne de caractères au format Ansi. Il est cependant possible de stocker le résultat dans une chaîne de caractères Unicode.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Répertoire temporaire
<a name="repertoire_temporaire_ELTPARAGRAPHE000072"></a>Sous Windows Vista (ou supérieur) avec le [système de l'UAC](../Editeurs/2025014.md) activé, le répertoire temporaire utilisé est différent si l'utilisateur a les privilèges administrateur ou non.


### Equivalence
<a name="equivalence_ELTPARAGRAPHE000082"></a>La fonction **fRépertoireTemp** est équivalente à la fonction [fRepTemp](../WDLang1/1000021768.md).

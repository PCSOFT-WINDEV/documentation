


## MenuLibellé (Fonction)

***En anglais : MenuLabel***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Identifie ou modifie le libellé d'une option de menu (menu contextuel ou déroulant).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connaître le libellé de l'option "OPT_BarreIcone" 
ResLibellé = MenuLibellé(OPT_BarreIcone)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Identifier le libellé d'une option de menu

`<Résultat> = MenuLibellé([<Fenêtre>, ] <Option de menu>)`
---

**`<Résultat> : (Chaîne de caractères)`**

Libellé actuel de l'option de menu.

**`<Fenêtre> : (Nom de fenêtre)`**

Nom de la fenêtre contenant l'option de menu. 
Si ce paramètre n'est pas spécifié, la fonction **MenuLibellé** manipulera la fenêtre en cours.

**`<Option de menu> : (Nom d'option)`**

Nom de l'option de menu à manipuler. Ce nom est défini dans la fenêtre de description de l'option de menu. 
Ce nom doit être spécifié sans guillemets.  


<a name="SYNTAXE2"></a>

### Modifier le libellé d'une option de menu

`<Résultat> = MenuLibellé([<Fenêtre>, ] <Option de menu> , <Nouveau libellé>)`
---

**`<Résultat> : (Chaîne de caractères)`**

Ancien libellé de l'option de menu.

**`<Fenêtre> : (Chaîne de caractères optionnelle)`**

Nom de la fenêtre contenant l'option de menu. 
Si ce paramètre n'est pas spécifié, la fonction **MenuLibellé** manipulera la fenêtre en cours.

**`<Option de menu> : (Option de menu)`**

Nom de l'option de menu à manipuler. Ce nom est défini dans la fenêtre de description de l'option de menu. 
Ce nom doit être spécifié sans guillemets.

**`<Nouveau libellé> : (Chaîne de caractères)`**

Nouveau libellé de l'option de menu. Il est possible d'ajouter un message d'aide à l'option de menu. Ce paramètre est alors de la forme : "&lt;Nouveau libellé&gt;:&lt;Message d'aide&gt;.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Caractères spéciaux
<a name="caracteres_speciaux_ELTPARAGRAPHE000068"></a>

- Pour que le caractère ":" ne soit pas considéré comme un séparateur d'aide, ce caractère ":" doit être précédé par un "antislash" ("\\").

- La chaîne de caractères "TAB" permet d'aligner les caractères suivants à gauche ("\\t" dans l'éditeur de menu).

- Le caractère "Caract (8)" permet d'aligner les caractères suivants à droite ("\\a" dans l'éditeur de menu).

- Le caractère "&" placé devant une lettre permet de décrire la lettre suivante en tant que lettre d'appel de l'option de menu.


![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Les caractères spéciaux sont ignorés. Les raccourcis clavier sur les options de menu ne sont pas gérés
<a name="NOTE0_2"></a>


### Equivalence
<a name="equivalence_ELTPARAGRAPHE000082"></a>La fonction **MenuLibellé** est équivalente à la propriété [Libellé](../Proprietes/2510053.md).

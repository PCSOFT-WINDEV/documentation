


## MenuMarque (Fonction)

***En anglais : MenuMark***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Positionne la marque ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Marque.gif)
 devant l'option de menu (menu contextuel ou déroulant).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Marquage de l'option "OPT_Option1" du menu contextuel "MenuContextuel1"
MenuMarque(OPT_Option1)
// Equivalent à : MenuMarque(MenuContextuel1.OPT_Option1)
```
<a name="Exemple2"></a>

```wl
// Marquage de l'option "OPT_BarreIcone"
MenuMarque(OPT_BarreIcone)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`MenuMarque([<Fenêtre>, ] <Option de menu>)`
---

**`<Fenêtre> : (Nom de fenêtre optionnel)`**

Nom de la fenêtre contenant l'option de menu à marquer. 
Si ce paramètre n'est pas spécifié, la fonction **MenuMarque** manipulera la fenêtre en cours.

**`<Option de menu> : (Nom d'option)`**

Nom de l'option de menu à marquer. Ce nom est défini dans la fenêtre de description de l'option de menu. 
Ce nom doit être spécifié sans guillemets.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Afficher une marque dans une option associée à une image
<a name="afficher_une_marque_dans_une_option_associee_une_image_ELTPARAGRAPHE000051"></a>Si une marque est associée à une option de menu avec image, l'image n'est pas remplacée par la marque.
<a name="NOTE0_2"></a>


### Equivalence
<a name="equivalence_ELTPARAGRAPHE000058"></a>Cette fonction est équivalente à la propriété [Cochée](../Proprietes/2510019.md).

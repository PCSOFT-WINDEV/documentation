
## gLigne (Fonction)

***En anglais : gLine***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Construit une séquence de caractères contenant une ligne. 

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Cette séquence de caractères permet de dessiner une ligne :  

- dans un élément d'un champ Liste ou d'un champ Combo.

- dans la barre de titre et/ou de message d'une fenêtre. 

- dans un champ Libellé. 







<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 
```wl
// Tracé d'une ligne verte dans un élément du champ "LISTE_Figure"
ListeAjoute(LISTE_Figure, gLigne(0, 0, 25, 18, VertClair))
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = gLigne(<X1> , <Y1> , <X2> , <Y2> [, <Couleur>])`
---

**`<Résultat> : (Chaîne de caractères)`**

Séquence de caractères contenant une ligne.  

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Cette séquence de caractères peut être : 

- utilisée dans les fonctions [ListeAjoute](../WDLang1/3049004.md), [ListeInsère](../WDLang1/3049002.md) ou [ListeModifie](../WDLang1/3049009.md).

- affectée à un champ Libellé. 

- utilisée dans la barre de titre ou de message d'une fenêtre. 




**`<X1> : (Réel)`**

Abscisse (en pixels) du point de départ de la ligne.

**`<Y1> : (Réel)`**

Ordonnée (en pixels) du point de départ de la ligne.

**`<X2> : (Réel)`**

Abscisse (en pixels) du point d'arrivée de la ligne. Ce point d'arrivée n'est pas dessiné.

**`<Y2> : (Réel)`**

Ordonnée (en pixels) du point d'arrivée de la ligne. Ce point d'arrivée n'est pas dessiné.

**`<Couleur> : (Entier ou constante (optionnel))`**

Couleur de la ligne. Cette couleur peut correspondre à :

- une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)),

- une couleur TSL (obtenue avec la fonction [TSL](../WDLang1/3029057.md)),

- [une couleur prédéfinie du WLangage](../WDLang5/3010002.md).




Si ce paramètre n'est pas précisé, la couleur du trait correspond à la couleur du texte en cours (définie par la fonction [gStylo](../WDLang1/3029039.md)).

## Remarques




### Coordonnées
<a name="coordonnees_ELTPARAGRAPHE000118"></a>Toutes les coordonnées sont exprimées en pixels par rapport au coin haut gauche de l'élément manipulé (par exemple par rapport au coin haut gauche de la ligne du champ Liste ou du champ Combo).

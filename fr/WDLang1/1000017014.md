
## SysListeEcran (Fonction)

***En anglais : SysListScreen***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la liste des écrans connectés sur une ou plusieurs cartes graphiques.  
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Récupération des différents écrans du PC en cours
ListeAjoute(LISTE_ListeEcran, SysListeEcran())
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = SysListeEcran([<Nom de la carte graphique>])`
---

**`<Résultat> : (Chaîne de caractères)`**



- Liste des noms des écrans séparés par RC. Cette liste est du type :
	
	```txt
	<Nom Ecran 1> + RC + <Nom Ecran 2> + RC + ... + RC + <Nom Ecran N>
	```

	Si la fonction système n'a pas pu être manipulée, &lt;Résultat&gt; correspond à une chaîne vide ("").

- Chaîne vide ("") en cas d'erreur. La fonction [ErreurInfo](../WDLang1/3013008.md) permet d'obtenir plus de renseignements sur le problème rencontré.




**`<Nom de la carte graphique> : (Chaîne de caractères optionnelle)`**

Nom de la carte graphique pour laquelle les écrans doivent être listés. Le nom des cartes graphiques peut être connu grâce à la fonction [SysListeCarteGraphique](../WDLang1/1000017016.md). Si ce paramètre n'est pas spécifié, les écrans de l'ensemble des cartes graphiques sont listés.

Si le nom spécifié n'existe pas, &lt;Résultat&gt; correspond à chaîne vide ("").

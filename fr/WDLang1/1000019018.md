| Constante | Description | Utilisée par les fonctions ou les types de variables |
| --- | --- | --- |

| **agDateHeure1erVisible** | Type de date recherchée : Première date et heure affichée dans le champ Agenda ou dans le champ Planning. | <br><br>- [AgendaPosition](../WDLang1/1000018889.md)<br><br>- [PlanningPositionDateHeure](../WDLang1/1000019998.md)<br><br><br> |
| **agiDateHeure** | Type d'information demandée : Date et heure de du rendez-vous correspondant aux coordonnées fournies. | [AgendaInfoXY](../WDLang1/1000018887.md) |
| **agiOrigineEcran** | Utiliser des coordonnées relatives à l'écran. | [AgendaInfoXY](../WDLang1/1000018887.md) |

| **agRdvSélectionné** | Type de rendez-vous recherché : Rendez-vous actuellement sélectionné dans le champ Agenda. | [AgendaListeRendezVous](../WDLang1/1000018880.md) |
| **agRdvSurvolé** | Type de rendez-vous recherché : Rendez-vous survolé par la souris dans le champ Agenda. | [AgendaListeRendezVous](../WDLang1/1000018880.md) |
| **agzJour** | Mode d'affichage de l'agenda : Agenda affiché sur une journée. | [AgendaChangeMode](../WDLang1/1000019497.md), [AgendaMode](../WDLang1/1000019545.md) |
| **agzMois** | Mode d'affichage de l'agenda : Agenda affiché sur un mois. | [AgendaChangeMode](../WDLang1/1000019497.md), [AgendaMode](../WDLang1/1000019545.md) |
| **agzNbJourPersonnalisé** | Mode d'affichage de l'agenda : Agenda affiché sur un nombre de jours personnalisé. | [AgendaChangeMode](../WDLang1/1000019497.md), [AgendaMode](../WDLang1/1000019545.md) |

| **agzSemaineSur5** | Mode d'affichage de l'agenda : L'agenda affiche les cinq premiers jours de la semaine. | [AgendaChangeMode](../WDLang1/1000019497.md), [AgendaMode](../WDLang1/1000019545.md) |

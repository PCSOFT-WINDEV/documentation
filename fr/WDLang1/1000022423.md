


## DocFinModification (Fonction)

***En anglais : DocEndModification***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Signale la fin du regroupement d'un ensemble d'opérations sur un document de type Traitement de texte en un seul événement (gestion du "undo/redo").


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
DocDébutModification(TT_MonDocument)
DocRemplace(TT_MonDocument, "%NomClient%", Client.Nom)
DocRemplace(TT_MonDocument, "%PrénomClient%", Client.Prénom)
DocFinModification(TT_MonDocument)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`DocFinModification(<Document>)`
---

**`<Document> : (Variable de type Document ou chaîne de caractères)`**

Document à manipuler. Ce document correspond : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) au nom d'un champ Traitement de texte.

- à une variable de type [Document](../WDLang1/1000022461.md).






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Les modifications effectuées sur le document sont regroupées à partir de l'appel de la fonction [DocDébutModification](../WDLang1/1000022620.md). Si plusieurs modifications annulables sont effectuées, il suffit d'un seul "Ctrl + Z" pour annuler toutes les opérations effectuées.

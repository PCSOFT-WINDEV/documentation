
## Log (Fonction)

***En anglais : Log***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Calcule le logarithme décimal d'une valeur numérique.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Calcul du logarithme de 10
ResLogarithme = Log(10)
// Renvoie 1
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = Log(<Valeur numérique>)`
---

**`<Résultat> : (Réel)`**

Logarithme décimale recherché.

**`<Valeur numérique> : (Entier ou réel)`**

Valeur numérique à manipuler. Si ce paramètre est un nombre négatif ou nul, une erreur du WLangage est générée.

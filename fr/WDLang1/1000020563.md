
## FIListeOccurrence (Fonction)

***En anglais : IWListCount***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le nombre de fenêtres internes actuellement présentes dans la liste des fenêtres internes parcourue par un champ Fenêtre Interne.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
FIListeOccurrence(FI_ChampFenêtreInterne)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = FIListeOccurrence(<Champ Fenêtre Interne>)`
---

**`<Résultat> : (Entier)`**

Nombre de fenêtres internes actuellement associées au champ Fenêtre Interne spécifié.

**`<Champ Fenêtre Interne> : (Nom de champ)`**

Nom du champ Fenêtre Interne à manipuler.

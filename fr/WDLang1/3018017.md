


## ArbreSupprimeFils (Fonction)

***En anglais : TreeDeleteChild***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime les fils d'un élément d'un champ Arbre sans supprimer l'élément lui-même.

Exemple de champ Arbre :
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Arbre_Vocabulaire.gif)

<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Suppression des fils du noeud "Desserts" du champ "ARBRE_RecetteTV"
Res = ArbreSupprimeFils(ARBRE_RecetteTV, "Recettes" + TAB + "Desserts")
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = ArbreSupprimeFils(<Champ Arbre> , <Chemin de l'élément>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la suppression a été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> si l'élément spécifié n'existe pas.




**`<Champ Arbre> : (Nom de champ)`**

Nom du champ Arbre à manipuler.

**`<Chemin de l'élément> : (Chaîne de caractères)`**

Chemin complet de l'élément dont les fils doivent être supprimés. Ce paramètre est de la forme :

```txt
"<Nom de la racine>" + TAB + ["<Nom du 1er noeud>" + TAB + ...
["<Nom du 2ème noeud>" + TAB + [...]]]"<Nom de la feuille>"
```
En cas de doublons sur un élément du chemin, ce chemin peut contenir l'identifiant de l'élément (indiqué à la suite du nom grâce à la fonction [ArbreID](../WDLang1/3018027.md)).

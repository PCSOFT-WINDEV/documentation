


## Fonctions financières
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


Les fonctions financières sont les suivantes :



|   |   |
| --- | --- |
| [FinAmortissement](../WDLang1/3050061.md) | Crée une matrice d'amortissement pour le montant d'un emprunt à un taux d'intérêt constant pour une durée définie. |
| [FinAmortissementDégressif](../WDLang1/3050068.md) | Calcule la valeur de l'amortissement d'un bien pour une période spécifique, grâce à la formule de l'amortissement décroissant à taux fixe. |
| [FinAmortissementLinéaire](../WDLang1/3050046.md) | Calcule la valeur de l'amortissement linéaire d'un bien pour une période donnée. |
| [FinErreur](../WDLang1/3050071.md) | Renvoie le numéro de la dernière erreur due à une fonction Finxxx ([fonctions financières](../WDLang1/3050054.md)). |
| [FinIntérêtPériode](../WDLang1/3050042.md) | Évalue pour une période donnée le montant des intérêts dus pour un emprunt remboursé par versements périodiques constants avec un taux d'intérêt constant. |
| [FinNbVersement](../WDLang1/3050041.md) | Renvoie le nombre de versements nécessaires pour rembourser un capital selon un taux. |
| [FinTauxIntérêt](../WDLang1/3050047.md) | Évalue le taux d'intérêt pour un emprunt donné sur une période avec des versements constants. |
| [FinValActuelle](../WDLang1/3050055.md) | Renvoie la valeur actuelle d'un investissement. |
| [FinValActuelleNette](../WDLang1/3050053.md) | Renvoie la valeur actuelle nette d'un investissement. |
| [FinValFuture](../WDLang1/3050056.md) | Renvoie la valeur future d'un investissement. |
| [FinValRemboursement](../WDLang1/3050039.md) | Renvoie le montant de chaque remboursement périodique pour un investissement à remboursement et à taux d'intérêt constants. |

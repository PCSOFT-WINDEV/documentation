


## &lt;Variable picCalque&gt;.Rotation (Fonction)

***En anglais : &lt;picLayer variable&gt;.Rotation***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Effectue une rotation d'une image contenue dans une variable de type [picCalque](../WDLang1/1000024604.md).![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=dRotation.gif)


![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Les dessins](../WDLang1/3029035.md). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonImageWDPic est un WDPic = "Test.bmp"
MonCalque est un picCalque

// Rotation du calque de 45 degrés
Résultat1 est un booléen
Résultat1 = MonCalque.Rotation(45)
SI Résultat1 = Faux ALORS Erreur("L'opération a échoué")
IMG_MonDessin = MonImageWDPic
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Effectuer une rotation simple

`<Résultat> = <Image picCalque>.Rotation(<Angle de rotation> [, <Options>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la rotation a été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Image picCalque> : (Variable de type picCalque)`**

Nom de la variable de type [picCalque](../WDLang1/1000024604.md) à manipuler.

**`<Angle de rotation> : (Réel)`**

Angle de rotation (en degrés). 

- Si l'angle est positif, l'image est tournée dans le sens horaire. 

- Si l'angle est négatif, l'image est tournée dans le sens anti-horaire.




**`<Options> : (Constante de type Entier)`**

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png)![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) Options de rotation : 


|   |   |
| --- | --- |
| *drAdapte* | L'image est tournée avec un agrandissement. Elle est ensuite réduite pour correspondre à la taille initiale de l'image. |
| *drDéfaut* | Agrandissement si nécessaire de l'image pour correspondre aux dimensions de l'image tournée.<br><br>**Constante utilisée par défaut pour les variables de type [picCalque](../WDLang1/1000024604.md)**. |
| *drSansAgrandissement* | L'image est tournée mais n'est pas agrandie : l'image peut être tronquée.<br><br> |


![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Ce paramètre n'est pas disponible.

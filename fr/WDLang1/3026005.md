


## &lt;Combo&gt;.Ouvre (Fonction)

***En anglais : &lt;Combo box&gt;.Open***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Déroule la liste d'un champ Combo (champ Combo simple ou champ Combo à plusieurs colonnes).

**Remarques** : 

- Le champ Combo dont la liste est déroulée prend le focus et le code d'entrée de cette combo est exécuté.

- Si un autre champ Combo est déroulé, la liste de ce champ est d'abord fermée.






![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Cette fonction ne peut pas être utilisée sur les combos natives (option "Combo native
(Windows Store Apps)" cochée dans l'onglet "Détail" de la fenêtre de description du champ).




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Déroule la liste du champ "COMBO_VilleFrance"
COMBO_VilleFrance.Ouvre()
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Champ Combo>.Ouvre()`
---

**`<Champ Combo> : (Nom de champ)`**

Nom du champ Combo à manipuler.


## FichierDisque (Type de variable)

***En anglais : DiskFile***
			
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **FichierDisque** permet de manipuler un fichier texte présent sur disque. Les caractéristiques de ce fichier peuvent être définies et modifiées à l'aide de différentes propriétés WLangage. 

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ecriture dans un fichier texte
MonFichier est un FichierDisque = fOuvre("Fichier.txt")
MonFichier.Ecrit("Compte-rendu")
```

## Propriétés


### Propriétés spécifiques à la description des variables de type **FichierDisque**
<a name="proprietes_specifiques_description_des_variables_type_fichierdisque_ELTPARAGRAPHE000026"></a>Les propriétés suivantes peuvent être utilisées pour manipuler un fichier texte :

| Nom de la propriété | Type manipulé | Effet |
| --- | --- | --- |
| DateCréation | DateHeure | Date et heure de création du fichier.<br><br>![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) <br>**Cette propriété est disponible en lecture seulement**.<br><br>![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Cette propriété est disponible à partir de Android 8 uniquement. Dans les versions précédentes, la lecture de cette propriété renverra une chaîne vide. |
| DateDernierAccès | DateHeure | Date et heure du dernier accès au fichier.<br><br>![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Cette propriété est disponible à partir de Android 8 uniquement. Dans les versions précédentes, la lecture de cette propriété renverra une chaîne vide et son écriture sera sans effet. |
| DateModification | DateHeure | Date et heure de dernière modification du fichier.<br><br>![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Cette propriété est disponible à partir de Android 8 uniquement. Dans les versions précédentes, la lecture de cette propriété renverra une chaîne vide et son écriture sera sans effet. |
| Handle | Entier système | <br><br>- Handle système du fichier, <br><br>- -1 si pas de fichier associé.<br><br><br>**Cette propriété est disponible en lecture seulement**.<br><br>![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Cette propriété n'est pas disponible. |


Remarque : Le fichier associé à une variable de type **FichierDisque** est automatiquement fermé à la fin de la portée de la dernière variable associée au fichier. 
![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Le fichier associé à une variable de type **FichierDisque** est automatiquement fermé lorsque la dernière variable pointant sur le fichier sera éligible à la libération par le système Android.

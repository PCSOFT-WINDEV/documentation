
## RubanSurligneVolet (Fonction)

***En anglais : RibbonHighlightPane***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Surligne ou non un volet d'un champ Ruban. Le surlignage consiste à afficher un nouvel onglet au-dessus de l'onglet spécifié, avec le texte et la couleur spécifiés.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Clic sur une image
RubanSurligneVolet(RUBAN_Menu, 5, "Outils image", MagentaClair)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`RubanSurligneVolet([<Champ Ruban>, ] <Indice> , <Texte> [, <Couleur>])`
---

**`<Champ Ruban> : (Nom de champ optionnel)`**

Nom du champ Ruban à manipuler. Si ce paramètre n'est pas spécifié, manipule le champ Ruban de la fenêtre en cours. 

**`<Indice> : (Entier)`**

Indice du volet à manipuler. 

**`<Texte> : (Chaîne de caractères)`**

Texte à afficher lors du surlignage. Si ce paramètre correspond à une chaîne vide (""), le surlignage sera supprimé. 

**`<Couleur> : (Entier ou constante de type Entier (optionnel))`**

Couleur de surlignage. Cette couleur peut correspondre à :

- une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)),

- une couleur TSL (obtenue avec la fonction [TSL](../WDLang1/3029057.md)),

- une couleur prédéfinie (voir [les couleurs prédéfinies du WLangage](../WDLang5/3010002.md)).  


Cette couleur est appliquée à l'ensemble de l'onglet supplémentaire. Si ce paramètre n'est pas spécifié, la couleur utilisée correspond à celle de l'onglet. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Le volet surligné n'est pas sélectionné. 

- Un champ Ruban doit être présent dans la fenêtre en cours.

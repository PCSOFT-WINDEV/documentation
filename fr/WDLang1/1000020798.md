


## dSauveImageICO (Fonction)

***En anglais : dSaveImageICO***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Sauve une image au format icône soit dans un fichier, soit en mémoire.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Création d'une icône
MonIcône est une Icône
MonIcône.Image[1].Image = "C:\MesImages\sigle48.bmp"
MonIcône.CouleurTransparence[1] = MagentaClair
MonIcône.Format[1] = icoFormatBMP
MonIcône.Image[2].Image = "C:\MesImages\sigle256.png"
MonIcône.CouleurTransparence[2] = Transparent
MonIcône.Format[2] = icoFormatPNG
dSauveImageICO(MonIcône, "Sigle.ICO")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Sauvegarder une icône dans un fichier au format ICO

`<Résultat> = dSauveImageICO(<Icône> , <Nom du fichier>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'image a été sauvegardée,

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour connaître le libellé de l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.




**`<Icône> : (Variable de type Icône)`**

Nom de la variable de type [Icône](../WDLang1/1000020811.md) à manipuler.

**`<Nom du fichier> : (Chaîne de caractères)`**

Nom et chemin complet (ou relatif) du fichier ICO à créer. Un chemin de type UNC peut être utilisé. L'extension du fichier n'est pas obligatoire.

**Attention** :

- Pour que l'image soit sauvée, le répertoire de destination doit exister.

- L'existence du fichier ICO n'est pas vérifiée. Si ce fichier existe, il est automatiquement écrasé.





<a name="SYNTAXE2"></a>

### Sauvegarder une icône en mémoire

`<Résultat> = dSauveImageICO(<Icône> , <enMémoire>)`
---

**`<Résultat> : (Buffer)`**

Chaîne d'octets contenant l'image au format ICO. Cette chaîne d'octets peut être par la suite affectée à un champ Image, enregistrée dans un fichier (fonction [fSauveTexte](../WDLang1/3036046.md)), ...

**`<Icône> : (Variable de type Icône)`**

Nom de la variable de type [Icône](../WDLang1/1000020811.md) à manipuler.

**`<enMémoire> : (Constante)`**

*enMémoire* : Constante permettant de sauvegarder une image en mémoire.

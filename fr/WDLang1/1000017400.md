
## AnimationMinFPS (Fonction)

***En anglais : AnimationMinFPS***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Paramètre le nombre minimal d'images par seconde pour que les animations dans les champs (effets de transitions) soient activées.

Si la machine n'est pas assez rapide pour afficher le nombre d'images minimum par seconde, les effets d'animation dans les champs ne seront pas affichés afin d'éviter un effet saccadé. Le champ passe alors directement de l'état initial à l'état final.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Animation minimale : 25 images par secondes
AnimationMinFPS(25)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`AnimationMinFPS(<FPS>)`
---

**`<FPS> : (Entier)`**

Nombre minimum d'images par secondes.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Cette fonction concerne uniquement l'animation des champs d'une fenêtre. Les animations d'images (GIF animés) et les animations de fenêtres ne sont pas modifiées par cette fonction.

- Le terme FPS signifie "Frame per second".




## DiagrammeCharge (Fonction)

***En anglais : DiagramLoad***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Charge un diagramme dans une variable de type Diagramme à partir d'un fichier ".wddiag", d'un mémo ou d'un buffer binaire.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonDiagramme est un Diagramme
MonDiagramme <- EDIAG_MonDiagramme
DiagrammeCharge(MonDiagramme, "diagramme1.wddiag")
SI ErreurDétectée ALORS
	Erreur("Fichier non chargé : " + ErreurInfo())
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = DiagrammeCharge(<Diagramme> , <Fichier à charger>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le fichier a été chargé, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. La fonction [ErreurInfo](../WDLang1/3013008.md) permet d'obtenir plus de détails sur l'erreur.




**`<Diagramme> : (Variable de type Diagramme)`**

Nom d'une variable de type [Diagramme](../WDLang1/1410088055.md) dans laquelle le fichier doit être chargé.  

**`<Fichier à charger> : (Chaîne de caractères)`**

Fichier diagramme à charger. Ce fichier peut correspondre :

- au chemin complet d'un fichier ".wddiag",

- au nom d'un fichier ".wddiag" (si le fichier est dans le répertoire de l'exécutable),

- à une rubrique mémo binaire présente dans un fichier de données,

- à un buffer binaire contenant les données complètes d'un fichier ".wddiag" (résultat de la fonction [DiagrammeSauve](../WDLang1/1410088122.md)).




## PlanningSupprimeRendezVous (Fonction)

***En anglais : SchedulerDeleteAppointment***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime un rendez-vous dans un champ Planning. Si le champ Planning est lié à une source de données, le rendez-vous est également supprimé dans la source de données. 



![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) En mode Universal Windows 10 App, cette fonction permet uniquement de manipuler les champs Planning présents dans un état. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Supprime le rendez-vous sélectionné dans le champ Planning PLN_Planning
PlanningSupprimeRendezVous(PLN_Planning)

// Supprime le premier rendez-vous ajouté
PlanningSupprimeRendezVous(PLN_Planning, 1)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Supprimer un rendez-vous identifié par son indice

`PlanningSupprimeRendezVous(<Champ Planning> [, <Indice du rendez-vous>])`
---

**`<Champ Planning> : (Nom de champ)`**

Nom du champ Planning à manipuler. Ce champ peut correspondre : 

- au champ Planning d'une fenêtre, 

- au champ Planning d'une page,

- au champ Planning d'un état.




**`<Indice du rendez-vous> : (Entier optionnel)`**

Indice du rendez-vous à supprimer. Le premier rendez-vous a pour indice 1. Si ce paramètre n'est pas précisé, le rendez-vous sélectionné sera supprimé.


<a name="SYNTAXE2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 
### Supprimer un rendez-vous identifié par une variable de type RendezVous

`PlanningSupprimeRendezVous(<Champ Planning> , <Rendez-vous manipulé>)`
---

**`<Champ Planning> : (Nom de champ)`**

Nom du champ Planning à manipuler. Ce champ peut correspondre : 

- au champ Planning d'une fenêtre, 

- au champ Planning d'une page,

- au champ Planning d'un état.




**`<Rendez-vous manipulé> : (Variable de type RendezVous)`**

Variable de type [RendezVous](../WDLang1/1000019244.md) correspondant au rendez-vous à supprimer.

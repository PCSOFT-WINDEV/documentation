


## Fonctions Vidéo
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


Les fonctions permettant de gérer les vidéos sont les suivantes :



|   |   |
| --- | --- |
| [VidéoAffiche](../WDLang1/3079004.md) | Affiche dans un champ Caméra la séquence vidéo actuellement retransmise par une "web caméra" ou par la caméra de l'appareil mobile.<br><br> |
| [VidéoArrête](../WDLang1/3079001.md) | Arrête l'affichage ou l'enregistrement d'une séquence vidéo affichée dans un champ Caméra.<br><br> |
| [VidéoArrêteDécodage](../WDLang1/1000022164.md) | Arrête le décodage d'un code-barres dans un champ Caméra. |
| [VidéoCapture](../WDLang1/3079007.md) | Capture la séquence vidéo actuellement retransmise par une "caméra" :<br><br>- sous forme d'une image : prend une photo.<br><br>- sous forme de vidéo : enregistre une vidéo.<br><br><br><br><br> |
| [VidéoChangeDriver](../WDLang1/3079010.md) | Connecte le champ "Caméra" à un driver de capture vidéo spécifique. |
| [VidéoConfigure](../WDLang1/3079009.md) | Permet de configurer le driver de capture vidéo. |
| [VidéoDemandePermission](../WDLang1/1000022355.md) | Demande la permission d'utiliser la caméra de l'appareil. |
| [VidéoDémarreDécodage](../WDLang1/1000022163.md) | Démarre le décodage des codes-barres dans un champ Caméra. |
| [VidéoEtat](../WDLang1/3079006.md) | Renvoie l'état d'un champ Caméra.<br><br> |
| [VidéoGénèreMiniature](../WDLang1/1000020268.md) | Génère une miniature de la photo ou de la vidéo spécifiée. |
| [VidéoLanceAppli](../WDLang1/1000019368.md) | Lance l'application caméra native de l'appareil afin d'enregistrer une vidéo ou de prendre une photo. |
| [VidéoListeDriver](../WDLang1/3079008.md) | Liste les drivers de capture vidéo installés sur le poste en cours. |
| [VidéoListeParamètre](../WDLang1/1000019695.md) | Liste les valeurs supportées par la caméra de l'appareil pour un paramètre donné.<br><br> |
| [VidéoParamètre](../WDLang1/1000019694.md) | Renvoie ou modifie la valeur d'un paramètre de la caméra de l'appareil.<br><br> |





<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


### Remarques : 
<a name="remarques_ELTPARAGRAPHE000036"></a>

- Les fonctions Vidéoxxx et les fonctions Caméraxxx sont identiques. 

- La fonction [dSauveImageJpeg](../WDLang1/3029038.md) permet de sauver l'image contenue dans un champ Caméra :

	- soit dans un fichier au format JPEG.

	- soit en mémoire.




- La fonction [dSauveImageBmp](../WDLang1/3029026.md) permet de sauver l'image contenue dans un champ Caméra :

	- soit dans un fichier au format BMP.

	- soit en mémoire.

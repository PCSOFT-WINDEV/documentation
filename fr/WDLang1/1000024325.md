


## &lt;Image&gt;.CouleurDominanteImage (Fonction)

***En anglais : &lt;Image&gt;.DominantImageColor***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la couleur dominante (c'est-à-dire la plus fréquente) d'une image présente dans un champ Image.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MaCouleur est une Couleur = IMG_MonImage.CouleurDominanteImage()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Champ Image>.CouleurDominanteImage()`
---

**`<Résultat> : (Entier)`**

Couleur dominante de l'image.

**`<Champ Image> : (Nom de champ)`**

Nom du champ Image à manipuler.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Les pixels transparents ou semi-transparents de l'image sont ignorés.

- La fonction **&lt;Image&gt;.CouleurDominanteImage** ne fonctionne pas sur des images vectorielles (SVG, PDF, ...).

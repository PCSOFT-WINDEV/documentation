


## Fonctions de gestion des champs RTF
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions suivantes sont spécifiques à la gestion des champs de saisie de type RTF :



|   |   |
| --- | --- |
| [RTFAjoute](../WDLang1/3025024.md) | Ajoute une chaîne de caractères (au format RTF ou non) à la fin du contenu d'un champ de saisie de type RTF, d'un champ RTF dans un état ou d'une chaîne de caractères contenant un texte au format RTF. |
| [RTFAjouteLien](../WDLang1/1000021062.md) | Ajoute un lien sur une portion de texte dans un champ de saisie RTF. |
| [RTFCharge](../WDLang1/3025041.md) | Charge un fichier au format RTF dans un champ RTF (dans une fenêtre ou un état). |
| [RTFInsère](../WDLang1/3025025.md) | Insère une chaîne de caractères au format RTF dans le contenu d'un champ RTF (dans une fenêtre ou un état), ou dans le contenu d'une chaîne de caractères au format RTF. |
| [RTFRecherche](../WDLang1/3025043.md) | Recherche une chaîne de caractères dans un champ RTF (d'une fenêtre ou d'un état) ou dans une chaîne de caractères contenant du texte au format RTF. |
| [RTFRemplace](../WDLang1/3025038.md) | Remplace ou insère une chaîne de caractères dans un champ RTF (dans une fenêtre ou un état) ou dans une chaîne au format RTF (contenant un texte au format RTF). |
| [RTFSélection](../WDLang1/3025023.md) | Change ou liste les attributs du texte actuellement sélectionné dans un champ RTF (d'une fenêtre ou d'un état). |
| [RTFVersHTML](../WDLang1/3025049.md) | Renvoie la chaîne HTML correspondant à la chaîne RTF spécifiée. |
| [RTFVersTexte](../WDLang1/3025032.md) | Renvoie le texte sans les caractéristiques de formatage propres au RTF. |


## ServiceContrôle (Fonction)

***En anglais : ServiceControl***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Envoie un message de contrôle à un service. Ce service peut être un service présent sur le poste local ou sur un poste distant.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Met en pause le service présent dans la table TABLE_Service
SI ServiceContrôle(COL_Nom[TABLE_Service], serviceCmdPause) = Faux ALORS
	Erreur(ErreurInfo(errMessage))
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = ServiceContrôle(<Nom du service> , <Commande à envoyer> [, <Poste distant>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la commande a été envoyée au gestionnaire de services, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Nom du service> : (Chaîne de caractères)`**

Nom du service à contrôler.

**`<Commande à envoyer> : (Constante de type Entier ou entier)`**

Commande à envoyer au service :


|   |   |
| --- | --- |
| *serviceCmdArrête* | Arrête le service (équivalent à la fonction [ServiceArrête](../WDLang1/1000017111.md)). |
| *serviceCmdPause* | Met le service en pause (équivalent à la fonction [ServicePause](../WDLang1/1000017109.md)). |
| *serviceCmdRelance* | Sort un service de la pause (équivalent à la fonction [ServiceRelance](../WDLang1/1000017105.md)). |
| *serviceCmdRelireConfig* | Demande au service de relire ses paramètres de configuration (équivalent à la fonction [ServiceActualise](../WDLang1/1000017107.md)). |


Il est également possible d'utiliser un entier compris entre 128 et 255. Cette plage de valeurs est réservée pour les commandes définies par le service lui-même (Consultez la documentation du service pour savoir quelles sont les commandes acceptées et leur action).

**`<Poste distant> : (Chaîne de caractères optionnelle)`**

Adresse IP du poste distant au format xxx.xxx.xxx.xxx (obtenu par la fonction [NetAdresseIP](../WDLang3/3056023.md)) ou nom du poste distant (obtenu par la fonction [NetNomMachine](../WDLang3/3056014.md)). Ce paramètre permet de contrôler un service sur un autre poste. Pour utiliser cette option, l'utilisateur doit être administrateur de la machine distante. 
Si ce paramètre n'est pas précisé ou correspond à une chaîne vide, le service est manipulé en local.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Mode de fonctionnement
<a name="mode_fonctionnement_ELTPARAGRAPHE000100"></a>La fonction **ServiceContrôle** envoie au gestionnaire de services une commande. Cette opération peut ne pas être effectuée immédiatement. Pour connaître l'état du service, utilisez la fonction [ServiceAttendEtat](../WDLang1/1000017101.md).

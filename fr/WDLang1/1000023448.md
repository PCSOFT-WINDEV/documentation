
## TableurListeNom (Fonction)

***En anglais : SpreadsheetListName***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la liste des noms qui identifient une cellule d'un champ Tableur.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
tabNom est un tableau de chaînes
tabNom = TableurListeNom(TBLR_MonTableur) 
POUR TOUT sDesc1Nom DE tabNom
	sNom est chaîne  = ExtraitChaîne(sDesc1Nom, 1, TAB)
	sCellule est chaîne  = ExtraitChaîne(sDesc1Nom, 2, TAB)
	sPortée est une chaîne = ExtraitChaîne(sDesc1Nom, 3, TAB)
	sCommentaire est chaîne = ExtraitChaîne(sDesc1Nom, 4, TAB)
	Trace("Nom : " + sNom + "; " + "Cellule : " + sCellule + "; " + ...
	 "Portée : " + sPortée + ";" + "Commentaire : " + sCommentaire)
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = TableurListeNom(<Champ Tableur>)`
---

**`<Résultat> : (Tableau)`**

Tableau de chaînes contenant la description des différents noms de cellules présent dans le champ. Chaque élément du tableau est de la forme : 

```txt
<Nom>+TAB+<Cellule>+TAB+<Portée>+TAB+<Commentaire>
```
 où : 

- &lt;Nom&gt; correspond au nom de la cellule. 

- &lt;Cellule&gt; correspond à la cellule identifiée par le nom. 

- &lt;Portée&gt; correspond à la portée du nom de cellule : 

	- 0 si la portée correspond au classeur, 

	- 1 si la portée correspond à la feuille. 




- &lt;Commentaire&gt; correspond au commentaire associé au nom de la cellule. 




**`<Champ Tableur> : (Nom de champ)`**

Nom du champ Tableur à manipuler.

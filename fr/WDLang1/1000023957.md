


## SaisieSupprimeToutJeton (Fonction)

***En anglais : EditDeleteAllToken***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime tous les jetons d'un champ de saisie de type "Jetons de texte".
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SaisieSupprimeToutJeton(SAI_Destinataire)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`SaisieSupprimeToutJeton(<Champ de saisie>)`
---

**`<Champ de saisie> : (Nom de champ)`**

Nom du champ de saisie de type "Jetons de texte".


## URLExtraitChemin (Fonction)

***En anglais : URLExtractPath***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie les différents éléments d'une URL : protocole, utilisateur, mot de passe, domaine, port, chemin de la ressource, nom de la ressource, extension de la ressource, requête et fragment.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Récupération du chemin complet de la ressource de l'URL
Ressource = URLExtraitChemin(...
		"http://user1:mdpuser1@www.site.com:80/images/image.gif", ...
		urlCheminRessource + urlNomRessource + urlExtensionRessource)
// Ressource = "/images/image.gif"
```
<a name="Exemple2"></a>

```wl
// Récupération de l'adresse du serveur
lsServeur = URLExtraitChemin(...
		"http://user1:mdpuser1@www.site.com:80/images/image.gif", ...
		urlProtocole + urlDomaine + urlPort)
// lsServeur = "http://www.site.com:80"
// Test si l'adresse répond
// httpRequête(lsServeur)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = URLExtraitChemin(<URL> , <Portion à extraire>)`
---

**`<Résultat> : (Chaîne de caractères)`**



- Portion de l'URL demandée. 

- Chaîne vide ("") en cas d'erreur. Pour connaître le libellé de l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.




**`<URL> : (Chaîne de caractères)`**

URL complète ou relative à manipuler. 

**`<Portion à extraire> : (Constante ou combinaison de constantes de type Entier)`**

Portion de l'URL à extraire : 



|   |   |
| --- | --- |
| *urlCheminRessource* | Chemin de la ressource de l'URL.<br><br>Exemple : pour l'URL suivante "http://user1:mdpuser1@www.site.com:80/pages/page1.htm?id=5#partie1", correspond à "/pages/". |
| *urlDomaine* | Domaine de l'URL.<br><br>Exemple : pour l'URL suivante "http://user1:mdpuser1@www.site.com:80/pages/page1.htm?id=5#partie1", correspond à "www.site.com". |
| *urlExtensionRessource* | Extension de la ressource dans l'URL.<br><br>Exemple : pour l'URL suivante "http://user1:mdpuser1@www.site.com:80/pages/page1.htm?id=5#partie1", correspond à ".htm". |
| *urlFragment* | Fragment de l'URL (ancre dans un document).<br><br>Exemple : pour l'URL suivante "http://user1:mdpuser1@www.site.com:80/pages/page1.htm?id=5#partie1", correspond à "partie1". |
| *urlMotDePasse* | Mot de passe de l'utilisateur dans l'URL.<br><br>Exemple : pour l'URL suivante "http://user1:mdpuser1@www.site.com:80/pages/page1.htm?id=5#partie1", correspond à "mdpuser1". |
| *urlNomRessource* | Nom de la ressource de l'URL.<br><br>Exemple : pour l'URL suivante "http://user1:mdpuser1@www.site.com:80/pages/page1.htm?id=5#partie1", correspond à "page1". |
| *urlPort* | Port de l'URL.<br><br>Exemple : pour l'URL suivante "http://user1:mdpuser1@www.site.com:80/pages/page1.htm?id=5#partie1", correspond à "80". |
| *urlProtocole* | Protocole de l'URL.<br><br>Exemple : pour l'URL suivante "http://user1:mdpuser1@www.site.com:80/pages/page1.htm?id=5#partie1", correspond à "http". |
| *urlRequête* | Requête de l'URL.<br><br>Exemple : pour l'URL suivante "http://user1:mdpuser1@www.site.com:80/pages/page1.htm?id=5#partie1", correspond à "id=5". |
| *urlUtilisateur* | Nom de l'utilisateur dans l'URL.<br><br>Exemple : pour l'URL suivante "http://user1:mdpuser1@www.site.com:80/pages/page1.htm?id=5#partie1", correspond à "user1". |

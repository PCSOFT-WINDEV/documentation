


## CaméraVidéoDémarre (Fonction)

***En anglais : CameraVideoStart***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Démarre l'enregistrement d'une vidéo via un champ Caméra dans une application Android ou iOS. Les différentes options précisées pour le champ Caméra (flash, caméra dorsale ou frontale, ...) sont prises en compte.  

Pour arrêter l'enregistrement de la vidéo, utilisez la fonction [CameraVidéoArrête](../WDLang1/1410087229.md). La vidéo est sauvée dans le répertoire par défaut de l'application ou dans le répertoire spécifié avec la propriété [RépertoireDeTravail](../Proprietes/1000026169.md).

Remarque : Cette fonction est utilisable uniquement avec un champ Caméra en version 27 ou supérieure (le champ ne doit pas avoir l'option "Mode compatible 26" cochée dans l'onglet "Général" de la fenêtre de description du champ).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Code de clic sur le champ BTN_Démarre_video
ChronoDébut()

TimerSys(TimerSys_Callback, 1s)
PROCÉDURE INTERNE TimerSys_Callback()
	dDurée est une Durée = ChronoValeur()
	LIB_CPT_VIDEO = DuréeVersChaîne(dDurée, "HH:MM:SS")
FIN

CaméraVidéoDémarre(CAM_Caméra, CaméraVidéoDémarre_Callback)


PROCÉDURE INTERNE CaméraVidéoDémarre_Callback(bRéussite est un booléen, CheminVidéo est une chaîne)
	ChronoFin()
	FinTimerSys()
	LIB_CPT_VIDEO = "00:00:00"
	SI bRéussite ALORS
		LIB_CHEMIN_VIDEO = CheminVidéo
		MaFenêtre.Plan = 4
	SINON
		ToastAffiche("Erreur lors de l'enregistrement de la vidéo." + RC + ErreurInfo())
	FIN
FIN
```

```wl
// Arrêt de la vidéo 
CaméraVidéoArrête(CAM_Caméra)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`CaméraVidéoDémarre(<Champ Caméra> , <Procédure WLangage>)`
---

**`<Champ Caméra> : (Nom de champ)`**

Nom du champ Caméra qui effectue l'enregistrement de la vidéo. 

**`<Procédure WLangage> : (Nom de procédure)`**

Nom de la procédure WLangage (également nommée "callback") appelée lorsque l'enregistrement de la vidéo est terminé (la fonction [CameraVidéoArrête](../WDLang1/1410087229.md) a été appelée).
Pour plus de détails sur cette procédure, consultez [Paramètres de la procédure utilisée par la fonction CaméraVidéoDémarre](../WDLang1/1410087850.md).

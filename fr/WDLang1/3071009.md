


## Fonctions de gestion de la souris
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


Les fonctions de gestion de la souris sont les suivantes : 



|   |   |
| --- | --- |
| [ChampSurvol](../WDLang1/3025021.md) | Identifie le champ survolé par le curseur de la souris. |
| [CurseurAffiche](../WDLang1/3071003.md) | Affiche ou masque le curseur de la souris. |
| [CurseurCapture](../WDLang1/3071004.md) | Dirige tous les événements associés à la souris vers une fenêtre spécifique ou un champ, quel que soit l'objet de l'écran où intervient l'événement souris. |
| [CurseurPos](../WDLang1/3071006.md) | Renvoie et modifie la position du curseur de souris. |
| [Sablier](../WDLang1/3071011.md) | Transforme le curseur souris (curseur du stylet) en sablier et inversement.<br>Sous Android et iOS, affiche une jauge sans fin dans une fenêtre. |
| [SablierRéseau](../WDLang1/1000020797.md) | Affiche ou masque une jauge sans fin : <br><br>- dans la barre de statut d'une application iOS. <br><br>- dans la barre de titre des fenêtres d'une application Android.<br><br><br> |
| [SourisPosX](../WDLang1/3071007.md) | Utilisée dans un événement lié à la souris ou au stylet (code de clic, code de survol, code de bouton gauche ou droit enfoncé, ...) renvoie la position horizontale (X) du curseur de souris par rapport au champ ou à la fenêtre considérée. |
| [SourisPosY](../WDLang1/3071008.md) | Utilisée dans un événement lié à la souris ou au stylet (code de clic, code de survol, code de bouton gauche ou droit enfoncé, ...) renvoie la position verticale (Y) du curseur de souris par rapport au champ ou à la fenêtre considérée. |

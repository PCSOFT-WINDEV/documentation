


## &lt;Tableur&gt;.AjouteImage (Fonction)

***En anglais : &lt;Spreadsheet&gt;.AddImage***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ajoute une image dans la feuille en cours d'un champ Tableur.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ajoute l'image logo.png entre A2 et B5
TBLR_Tableur.AjouteImage("c:\temp\logo.png", "A2", "B5") 

// Ajoute l'image banniere.png à la position C6
TBLR_Tableur.AjouteImage("c:\temp\banniere.png", "C6")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Champ Tableur>.AjouteImage(<Image> , <Cellule d'insertion> [, <Cellule de fin>])`
---

**`<Résultat> : (Chaîne de caractères)`**

Identifiant de l'image ajoutée.

**`<Champ Tableur> : (Nom de champ)`**

Nom du champ Tableur à manipuler.

**`<Image> : (Chaîne de caractères)`**

Chemin du fichier image à ajouter dans le champ.

**`<Cellule d'insertion> : (Chaîne de caractères)`**

Coordonnée de la cellule dans laquelle le coin haut gauche de l'image doit être positionné.

**`<Cellule de fin> : (Chaîne de caractères)`**

Coordonnée de la cellule dans laquelle le coin bas droit de l'image doit être positionné. L'image recouvre la cellule. Si ce paramètre n'est pas précisée, l'image est insérée avec sa taille à 100%.

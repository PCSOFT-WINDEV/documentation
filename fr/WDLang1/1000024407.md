


## &lt;Panneau&gt;.Détache (Fonction)

***En anglais : &lt;Panel&gt;.Detach***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Détache un champ Panneau dockable pour le transformer en une fenêtre flottante.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Détacher le champ Panneau dockable et le met en haut à gauche de l'écran
PADO_Test.Détache()
PADO_Test.X = 0
PADO_Test.Y = 0
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Champ Panneau dockable>.Détache()`
---

**`<Champ Panneau dockable> : (Nom de champ)`**

Nom du champ Panneau dockable à manipuler.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Si le champ Panneau dockable est déjà flottant, la fonction **&lt;Panneau&gt;.Détache** n'a aucun effet. 

- Le champ Panneau dockable peut être réattaché à sa fenêtre avec la fonction [&lt;Panneau dockable&gt;.Attache](../WDLang1/1000024404.md).




## SélecteurAjoute (Fonction)

***En anglais : RadioButtonAdd***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ajoute une nouvelle option dans un champ Sélecteur. L'option est ajoutée à la suite des options existantes. 




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ajout de 2 options 
SélecteurAjoute(SEL_Options, "Lire un livre sur une tablette")
SélecteurAjoute(SEL_Options, "Lire un livre de poche")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = SélecteurAjoute(<Champ Sélecteur> , <Libellé> [, <Sous-libellé> [, <Image>]])`
---

**`<Résultat> : (Entier)`**



- Indice de l‘option ajoutée. 

- -1 en cas d'erreur. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Champ Sélecteur> : (Nom de champ)`**

Nom du champ Sélecteur dans lequel l'option doit être ajoutée. L'option est ajoutée à la suite des options existantes.

**`<Libellé> : (Chaîne de caractères)`**

Libellé de l'option à ajouter.

**`<Sous-libellé> : (Chaîne de caractères optionnelle)`**

Sous-libellé de l'option à ajouter. 
Si ce paramètre n'est pas spécifié, l'option ajoutée n'a pas de sous-libellé. 
Si ce paramètre est spécifié, le sous-libellé sera affiché même si le sélecteur utilise un mode d'affichage sans sous-libellé.


**`<Image> : (Chaîne de caractères optionnelle)`**

Image associée à l'option à ajouter. 
Si ce paramètre n'est pas spécifié, l'option ajoutée n'a pas d'image.
Si ce paramètre est spécifié, l'image sera affichée uniquement si le sélecteur utilise un mode d'affichage "Style Vista" (avec image).




<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Les caractéristiques de l'option ajoutée peuvent également être modifiées grâce aux propriétés suivantes utilisées sur l'option : 

	- [Libellé](../Proprietes/2510053.md) pour modifier le libellé. 

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) [SousLibellé](../Proprietes/1000017232.md) pour modifier le sous-libellé. 

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) [Image](../Proprietes/2510034.md) pour modifier l'image. 




- Lors de l'ajout d'options dans un champ Sélecteur, la taille du champ n'est pas modifiée. Si le champ n'est pas suffisamment grand, les options ajoutées pourront ne pas être visibles. Il est possible d'augmenter la taille du champ avec les propriétés [Largeur](../Proprietes/2510059.md) et [Hauteur](../Proprietes/2510050.md).

- Pour ajouter une option à une position spécifique, utilisez la fonction [SélecteurInsère](../WDLang1/1000020599.md).  

- **Utilisation des valeurs renvoyées** : Si le sélecteur utilise les valeurs renvoyées, il est possible de définir la valeur renvoyée par l'option ajoutée grâce à la propriété [ValeurRenvoyée](../Proprietes/2510134.md). 
	Par exemple : 
	
	```wl
	nIndice est un entier
	nIndice = SélecteurAjoute(SEL_Option, "Lire un livre sur une tablette")
	SEL_Option[nIndice].ValeurRenvoyée = "Tablette"
	// Sélection de la nouvelle option
	SEL_Option = "Tablette"
	```

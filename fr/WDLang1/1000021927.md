
## Utilisation de la modification d'UI lors d'un test de projet ou de fenêtre
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000105"></a>
L'utilisateur final peut modifier directement l'interface de son application (selon les options précisées par le développeur). 

Ces modifications sont également disponibles lors du test d'une fenêtre ou d'un projet : 

- Création d'une note repositionnable, 

- Création d'un champ Fil, 

- Modification de la bulle d'aide d'un champ, 

- Déplacement d'un ou de plusieurs champs, ...




Au retour du mode GO, si des modifications d'interface ont été effectuées, l'éditeur de fenêtres propose automatiquement de récupérer ces modifications dans l'interface de la fenêtre. 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Exemple d'utilisation
<a name="exemple_utilisation_ELTTEXTE000129"></a>
Pour récupérer les modifications effectuées en GO de projet ou de fenêtre : 

1. Lancez le GO du projet (![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ICO_GO_Projet_WD.gif)) ou de la fenêtre (![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ICO_GO_Fenetre_WD_bl.gif)). 

2. Affichez les fenêtres voulues.

3. Modifiez l'UI (interface) des fenêtres voulues. Par exemple : 

	- Créez un champ Note repositionnable. 

	- Passez en mode modification et : 

		- Déplacez les champs, 

		- Créez un champ Fil.




- Modifiez le texte de la bulle d'aide.

4. Terminez le test. 

5. Lors du retour sous l'éditeur, une fenêtre spécifique apparaît. Cette fenêtre permet de récupérer les modifications d'UI réalisées en mode GO. 

<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=IMH_Modifiable_3%20-%20HC%20N%B0001.gif)


Cette fenêtre présente, pour chaque fenêtre modifiée, les modifications qui peuvent être récupérées : 

- Modification de taille et position : la taille et/ou la position de un ou plusieurs champs ont été modifiées. 

- Création d'un champ : un et/ou plusieurs champs ont été créés (champ Fil, champ Note). 

- Modification de propriétés : une et/ou plusieurs propriétés des champs ont été modifiées. Les modifications de la bulle d'aide sont prises en compte dans cette catégorie. 




Les boutons "Tous" et "Aucun" permettent de sélectionner les modifications à appliquer ou non. En déroulant le bouton, il est possible de sélectionner le type de modification à prendre en compte : <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=IMH_Modifiable_3%20-%20HC%20N%B0002.gif)


**Remarque** : Cette fenêtre est également affichée lors de l'importation sous l'éditeur d'un fichier "\*.wdmodif". Ce fichier est généré lors de l'export des modifications d'interface (option "Personnaliser l'interface .. Exporter la personnalisation de cette fenêtre" dans le menu contextuel de la fenêtre). Pour plus de détails, consultez [UI modifiable par l'utilisateur final](../WDLang1/1000021853.md).


## MultimédiaFerme (Fonction)

***En anglais : MultimediaClose***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ferme le fichier multimédia en cours dans le champ Multimédia. La mémoire allouée au fichier multimédia est libérée.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) 
```wl
MonFichier est une chaîne = "C:\Ma musique\mysong.mp3"

MultimédiaJoue(MM_Multimédia1, MonFichier)
...
MultimédiaFerme(MM_Multimédia1)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = MultimédiaFerme(<Nom du champ Multimédia>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le fichier joué dans le champ Multimédia est correctement fermé, 

- <u><u><u><u>Faux</u></u></u></u> en cas d'erreur. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).






**`<Nom du champ Multimédia> : (Chaîne de caractères)`**

Nom du champ Multimédia à manipuler.

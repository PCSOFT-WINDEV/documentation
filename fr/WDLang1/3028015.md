
## DDEConnecte (Fonction)

***En anglais : DDEConnect***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ouvre une connexion DDE entre le programme en cours et le destinataire selon un sujet donné.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple
<a class="notetitle" target="_blank" href="$DOC$=1000003028015&name=ddeconnecte_fonction&product=WD">Voir des exemples supplémentaires</a>

```wl
// Ouverture d'une connexion DDE
NumConnexion = DDEConnecte(NomServeur, "MonSujet")
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = DDEConnecte(<Nom du destinataire> , <Sujet>)`
---

**`<Résultat> : (Entier)`**



- Identifiant de la connexion (utilisé dans les fonctions DDE), 

- 0 si la connexion a échoué ([DDEErreur](../WDLang1/3028006.md) contient le compte-rendu de l'erreur).




**`<Nom du destinataire> : (Chaîne de caractères)`**

Nom du programme destinataire (sans le chemin ni l'extension). Le programme destinataire doit être accessible (dans le répertoire courant ou dans un répertoire accessible par le PATH).

**`<Sujet> : (Chaîne de caractères)`**

Sujet de la connexion.




## SauveParamètre (Fonction)

***En anglais : SaveParameter***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Sauve une valeur persistante : 

- dans la base de registre ou le fichier équivalent sur la plateforme d'exécution.

	- ![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Sous Universal Windows 10 App, les valeurs persistantes sont sauvegardées dans les paramètres de l'application ("Isolated Storage Settings").

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) Dans une application Linux, les valeurs persistantes sont sauvegardées dans le dossier $HOME/.local/share.




- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) dans le fichier spécifié avec la fonction [InitParamètre](../WDLang1/3025044.md).




Cette valeur pourra être lue grâce à la fonction [ChargeParamètre](../WDLang1/3025047.md) lors du prochain lancement de l'application.






<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Sauvegarde
SauveParamètre("LOGIN", SAI_Saisie1)

// Restauration
SAI_Saisie1 = ChargeParamètre("LOGIN","")
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`SauveParamètre(<Nom du paramètre> , <Valeur du paramètre>)`
---

**`<Nom du paramètre> : (Chaîne de caractères)`**

Nom du paramètre à sauvegarder. Ce paramètre ne doit pas correspondre à une chaîne vide.

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Si le paramètre a été sauvegardé dans la base de registre, il est possible de donner un chemin complet. Les séparateurs acceptés sont ".","/" et "\\". Le chemin complet d'un champ peut être donné grâce à la propriété [NomComplet](../Proprietes/2514005.md). Pour plus de détails, consultez [les remarques](#NOTE0_1).

![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Si le mode d'initialisation choisi est la base de registre, la fonction **SauveParamètre** n'est sensible ni à la casse, ni à l'accentuation du nom du paramètre ("Paramètre" et "parametre" sont équivalents).

**`<Valeur du paramètre> : (Chaîne de caractères ou entier)`**

Valeur à sauver.

## Remarques


### Nom du paramètre dans une base de registre
<a name="nom_parametre_dans_une_base_registre_ELTPARAGRAPHE000130"></a>

- **Cas 1 : le nom du paramètre est "Param1"** : 
	Ce paramètre est enregistré dans la base de registre en utilisant le chemin suivant :
	HKEY_CURRENT_USER\\Software\\&lt;Société&gt;\\&lt;Application&gt;\\Param1

- **Cas 2 : le nom du paramètre est "FEN_Fenêtre1\\SAI_Saisie2\\Param1"** : 
	Ce paramètre est enregistré dans la base de registre en utilisant le chemin suivant :
	HKEY_CURRENT_USER\\Software\\&lt;Société&gt;\\&lt;Application&gt;\\FEN_Fenêtre1\\SAI_Saisie2\\Param1

- **Cas 3 : le nom du paramètre est "FEN_Fenêtre1.SAI_Saisie2.Param1"** : 
	Ce paramètre est enregistré dans la base de registre en utilisant le chemin suivant :
	HKEY_CURRENT_USER\\Software\\&lt;Société&gt;\\&lt;Application&gt;\\FEN_Fenêtre1\\SAI_Saisie2\\Param1




**Rappel** : Pour connaître le chemin de la base de registre où sont stockées les informations concernant l'application, utilisez la fonction [ProjetInfo](../WDLang1/3064004.md) avec la constante *piRegistre*.

&lt;Société&gt; correspond à la société indiquée lors de la création de l'exécutable. Si ce nom n'est pas renseigné, &lt;Société&gt; correspond à "Applications WinDev".

**Remarque** : En mode Test, le chemin utilisé dans la base de registre est le suivant : HKEY_CURRENT_USER\\SOFTWARE\\PC SOFT\\WinDev\\28.0\\ModeTest\\&lt;projet&gt;.




## &lt;Type Date&gt;.VersJourEnLettre (Fonction)

***En anglais : &lt;Date type&gt;.ToDayInAlpha***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le jour de la semaine en toutes lettres correspondant à la date spécifiée.








<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MaDate est une Date
MaDate = "20011225"
MaDate.VersJourEnLettre() // Renvoie "mardi"
MaDate est une Date = "20011225"
MaDate.VersJourEnLettre()      // Renvoie "mardi"
```
<a name="Exemple2"></a>

```wl
MaDate est une Date = DateSys()
MonHeure est une Heure = HeureSys()

SAI_Date = MaDate.VersJourEnLettre()+ " " + MaDate.Jour + ...
	" " + MaDate.VersMoisEnLettre() + " à " + ...
	MonHeure.VersChaîne("HH:MM:SS")
// Le champ SAI_Date contient par exemple : lundi 13 Août à 14:44:17
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Date>.VersJourEnLettre()`
---

**`<Résultat> : (Chaîne de caractères)`**

Jour de la semaine correspondant à la date spécifiée : lundi, mardi, mercredi, jeudi, vendredi, samedi, dimanche.
**Remarque** : Le jour de la semaine dépend des options linguistiques spécifiées pour les dates dans l'onglet "Langues" de la description du projet (sous le volet "Projet", dans le groupe "Projet", cliquez sur "Description") :

- Si le projet utilise les options linguistiques de Windows pour les dates, le jour sera renvoyé dans la langue de Windows.

- Si le projet utilise des paramètres spécifiques pour les dates (pour une ou plusieurs langues), le jour renvoyé correspondra à celui indiqué dans la description du projet.




**`<Date> : (Variable de type Date ou variable de type DateHeure)`**

Date à manipuler. Cette date peut correspondre à : 

- une variable de type Date. 

- une variable de type DateHeure. Dans ce cas, seule la date est prise en compte.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Le format de stockage des dates permet de stocker des dates comprises entre le 01/01/0001 et le 31/12/9999.

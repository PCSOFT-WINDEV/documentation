
## Handle (Fonction)

***En anglais : Handle***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le "Handle" système (HWND) d'un champ ou d'une fenêtre WINDEV. Il est ainsi possible d'appeler certaines fonctions API avec ce paramètre (avec les fonctions [API](../WDLang1/3014005.md) ou [AppelDLL32](../WDLang1/3014003.md), [SendMessage](../WDLang1/3015003.md) ou [PostMessage](../WDLang1/3015001.md)).

L'utilisation de ce "Handle" est rare avec les fonctions du WLangage.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Copier le contenu d'un champ de saisie dans le presse-papiers
SendMessage(Handle(SAI_SAISIE1), WM_COPY, 0, 0)
```


<a name="Exemple2"></a>


![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) 
```wl
// Rafraîchissement immédiat du dessin de CHAMP1
hwnd est un entier = Handle(CHAMP1)

```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Récupérer le handle d'un champ ou d'une fenêtre

`<Résultat> = Handle([<Nom de l'objet>])`
---

**`<Résultat> : (Pointeur)`**

Handle demandé.

**`<Nom de l'objet> : (Chaîne de caractères optionnelle)`**

Nom du champ ou de la fenêtre. Si ce paramètre n'est pas précisé, la fonction **Handle** utilise la fenêtre en cours. Si ce paramètre correspond à une colonne de table, le handle de son champ de saisie sera renvoyé (NULL si la colonne n'est pas en saisie).  


<a name="SYNTAXE2"></a>

### Récupérer le handle d'un élément spécifique d'un champ

`<Résultat> = Handle([<Nom de l'objet>, ] <Type de l'objet>)`
---

**`<Résultat> : (Entier)`**

Handle demandé.

**`<Nom de l'objet> : (Chaîne de caractères optionnelle)`**

Nom du champ.

**`<Type de l'objet> : (Constante de type Entier)`**

Élément de la combo dont le handle est recherché.


|   |   |
| --- | --- |
| *HandleAscenseurH* | Renvoie le handle de l'ascenseur horizontal du champ. |
| *HandleAscenseurV* | Renvoie le handle de l'ascenseur vertical du champ. |
| *HandleListeCombo* | Renvoie le handle de la liste déroulante associée à une combo (le champ doit être un champ Combo). |
| *HandleListeSaisieAssistée* | Renvoie le handle de la liste déroulante associée à un champ proposant la saisie assistée. |
| *HandleSaiseCombo* | Renvoie le handle du champ de saisie associé à une combo (le champ doit être un champ Combo). |

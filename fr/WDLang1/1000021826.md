
## AscenseurPositionMax (Fonction)

***En anglais : ScrollbarMaxPosition***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Récupère la position maximale d'un des ascenseurs (horizontal ou vertical) d'un champ dans une application WINDEV.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affiche la fin des éléments du champ Table
AscenseurPosition(TABLE_MaTable, ascVert, AscenseurPositionMax(TABLE_MaTable))
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = AscenseurPositionMax(<Champ manipulé> [, <Orientation>])`
---

**`<Résultat> : (Entier)`**



- Position maximale de la cage de l'ascenseur (en unités logiques).

- 0 si l'ascenseur n'existe pas ou si l'ascenseur est grisé.




**`<Champ manipulé> : (Nom de champ)`**

Nom du champ manipulé.

**`<Orientation> : (Constante optionnelle de type Entier)`**

Orientation de l'ascenseur pour lequel la position doit être récupérée : 


|   |   |
| --- | --- |
| *ascHorz* | La position est calculée pour l'ascenseur horizontal du champ. |
| *ascVert*<br>(Valeur par défaut) | La position est calculée pour l'ascenseur vertical du champ. |

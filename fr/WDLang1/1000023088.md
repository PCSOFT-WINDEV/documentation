


## DocSortEdition (Fonction)

***En anglais : DocExitEdit***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Sort de l'édition en cours. L'entrée en édition a été effectuée avec la fonction [DocEntreEdition](../WDLang1/1000023077.md). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
DocSortEdition(TT_ChampMonDoc)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`DocSortEdition(<Champ Traitement de texte>)`
---

**`<Champ Traitement de texte> : (Nom de champ)`**

Nom du champ Traitement de texte à manipuler.

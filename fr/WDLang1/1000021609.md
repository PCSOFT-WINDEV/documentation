


## TableurPosition (Fonction)

***En anglais : SpreadsheetPosition***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Affiche la cellule spécifiée dans un champ Tableur présent dans une fenêtre: 

- soit la zone visible du champ Tableur est déplacée pour afficher la cellule. 

- soit la cellule est affichée en haut à gauche de la zone visible du champ Tableur. 



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
sCell est chaîne = TableurCherche(TBLR_TABLEUR1, "ATTENTION")
// Cas où la cellule est trouvée
SI sCell <> "" ALORS
	TableurPosition(TBLR_TABLEUR1, sCell, tblrVisible)
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`TableurPosition(<Champ Tableur> , <Cellule> , <Type de positionnement>)`
---

**`<Champ Tableur> : (Nom de champ)`**

Nom du champ Tableur à manipuler. 

**`<Cellule> : (Chaîne de caractères)`**

Nom de cellule à manipuler. Par exemple : "A4". 

**`<Type de positionnement> : (Entier)`**

Type de positionnement à effectuer : 


|   |   |
| --- | --- |
| *tblrHautGauche* | L'affichage du champ Tableur est modifié pour que la cellule spécifiée soit positionnée en haut à gauche de la zone visible du champ. |
| *tblrVisible* | L'affichage du champ Tableur est modifié pour que la cellule spécifiée soit visible dans la champ Tableur. Si la cellule spécifiée est déjà visible dans le champ Tableur, l'affichage du champ n'est pas modifié. |

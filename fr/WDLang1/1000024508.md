


## &lt;Variable Document&gt;.Ferme (Fonction)

***En anglais : &lt;Document variable&gt;.Close***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ferme le fichier docx associé à la variable [Document](../WDLang1/1000022461.md) et libère le document. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonDocument est un Document

MonDocument.Ferme()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Document>.Ferme()`
---

**`<Document> : (Variable de type Document)`**

Nom de la variable de type [Document](../WDLang1/1000022461.md) à manipuler.

Le fichier physique associé à la variable (s'il existe) est fermé. Les modifications réalisées dans le fichier ne sont pas sauvées avant sa fermeture (utilisez la fonction [&lt;Variable Document&gt;.Sauve](../WDLang1/1000024537.md) si nécessaire).

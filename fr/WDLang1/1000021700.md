


## &lt;Onglet&gt;.Etat (Fonction)

***En anglais : &lt;Tab&gt;.Status***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie l'état d'un volet d'onglet dynamique (identifié par son alias).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Récupère la ligne sélectionnée dans un champ Table
nIndice est un entier = TABLE_Client.Select()
SI nIndice = -1 ALORS
	ToastAffiche("Vous devez sélectionner un client dans le champ Table")
	RETOUR
FIN

// Récupère l'identifiant du client
nNumClient est un entier = TABLE_Client.IDCLIENT

// Définit l'alias du volet
sAlias est une chaîne = gtaOnglet[nNumClient]
gtaOnglet est un tableau associatif de chaînes

// Vérifie si la fiche client n'est pas déjà ouverte
SI sAlias = "" _OU_ ONG_MonOnglet.Etat(sAlias) = voletInexistant ALORS
	// Ouvre la fenêtre en modification	
	gtaOnglet[nNumClient] = ...
		ONG_MonOnglet.Ouvre(ChaîneConstruit("Fiche client de : %1", ...
			TABLE_Client.Nom[nIndice]), FI_Fiche, nNumClient)
SINON
	ToastAffiche("Fiche déjà ouverte")
	// Active le volet
	ONG_MonOnglet = sAlias
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Champ Onglet>.Etat(<Alias>)`
---

**`<Résultat> : (Constante de type Entier)`**

Etat du volet d'onglet dynamique : 


|   |   |
| --- | --- |
| *voletActif* | Le volet d'onglet existe et il est attaché à son onglet. |
| *voletFlottant* | Le volet d'onglet existe et il est détaché. |
| *voletInexistant* | Le volet d'onglet n'existe pas ou il a été fermé. |



**`<Champ Onglet> : (Nom de champ)`**

Nom du champ Onglet à manipuler.

**`<Alias> : (Chaîne de caractères)`**

Alias du volet d'onglet dynamique à manipuler. Cet alias peut être récupéré grâce à : 

- la propriété [Valeur](../Proprietes/2510130.md). 

- la propriété [Alias](../Proprietes/2510004.md). 

- la fonction [&lt;Onglet&gt;.Ouvre](../WDLang1/1000021704.md). 


Si ce paramètre ne correspond à aucun volet dynamique, la fonction ne fait rien et renvoie une erreur non fatale.

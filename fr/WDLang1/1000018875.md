
## SysIconeSupprimeImage (Fonction)

***En anglais : SysIconDeleteImage***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime l'icône en surimpression de l'icône de l'application dans la barre des tâches. Cette icône a pu être affichée grâce à la fonction [SysIconeAjouteImage](../WDLang1/1000018874.md). 

**Important** : La barre des tâches doit être configurée pour afficher le mode "Grandes icônes".
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Pendant le traitement, ajoute une icône de sablier sur l'icône de l'application
SysIconeAjouteImage("sablier.ico")
Sablier()

Traitement_Long()

Sablier(Faux)

// Supprime l'icône à la fin du traitement
SysIconeSupprimeImage()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = SysIconeSupprimeImage()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'icône en surimpression a été supprimée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.

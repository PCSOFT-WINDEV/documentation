


## &lt;Multimédia&gt;.Joue (Fonction)

***En anglais : &lt;Multimedia&gt;.Play***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Joue un fichier multimédia dans un champ Multimédia.








<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) 
```wl
MonFichier est une chaîne = "C:\Ma musique\mysong.mp3"
SI MM_Multimédia1.Joue(MonFichier) = Faux ALORS
	Erreur("Erreur de lecture du fichier")
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Champ Multimédia>.Joue([<Nom du fichier à jouer>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la lecture du fichier a réussi, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Champ Multimédia> : (Nom de champ)`**

Nom du champ Multimédia qui va jouer le fichier multimédia.

**`<Nom du fichier à jouer> : (Chaîne de caractères optionnelle)`**

Nom et chemin du fichier multimédia à jouer. 

Si ce paramètre est précisé, le fichier multimédia indiqué est chargé en mémoire puis joué. La lecture du fichier multimédia reprend toujours au début. 

Si ce paramètre n'est pas précisé ou correspond à une chaîne vide (""), le fichier associé au champ (par exemple avec la fonction [MultimédiaOuvre](../WDLang1/1000017206.md)) est automatiquement utilisé.




<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Utilisation de la fonction MultimédiaJoue
<a name="utilisation_fonction_multimediajoue_ELTPARAGRAPHE000273"></a>La fonction **&lt;Multimédia&gt;.Joue** peut être utilisée :

- pour charger en mémoire et jouer un fichier multimédia.

- pour jouer un fichier multimédia chargé avec la fonction [MultimédiaOuvre](../WDLang1/1000017206.md).

- pour relancer un fichier mis en pause avec la fonction [&lt;Multimédia&gt;.Pause](../WDLang1/1000020094.md).
	Remarque : Pour faire des pauses dans la lecture d'un fichier multimédia, la lecture du fichier multimédia doit avoir été lancée via la fonction [MultimédiaOuvre](../WDLang1/1000017206.md) suivi de la fonction &lt;Multimédia&gt;.Joue en précisant uniquement le nom du champ.

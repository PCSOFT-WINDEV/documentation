


## &lt;Fenêtre&gt;.EdCharge (Fonction)

***En anglais : &lt;Window&gt;.EdLoad***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Charge : 

- les modifications faites par l'utilisateur final dans une fenêtre. Ces modifications ont été préalablement sauvegardées en mémoire ou dans un fichier. 

- les notes créées par l'utilisateur finale dans une fenêtre. 




Remarque : Cette fonction peut être utilisée lorsque le mode "Modification de l'UI par l'utilisateur final" est autorisé pour les fenêtres de l'application.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Clic sur [Sauver]
FEN_Présentation.EdSauve("Mes modifications.wdmodif")

// Clic sur [Charger]
FEN_Présentation.EdCharge("Mes modifications.wdmodif")
SI ErreurDétectée = Vrai ALORS
	Info("Chargement impossible" + RC + ErreurInfo()) 
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Charger les modifications enregistrées dans un buffer

`<Résultat> = <Fenêtre>.EdCharge(<Buffer>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si les modifications ont été chargées, 

- <u><u><u><u>Faux</u></u></u></u> en cas d'erreur. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Fenêtre> : (Nom de fenêtre)`**

Nom de la fenêtre ou de la fenêtre interne à manipuler.

**`<Buffer> : (Variable de type Buffer)`**

Buffer contenant les modifications (les modifications ont été précédemment enregistrées avec la fonction [FenEdSauve](../WDLang1/1000021848.md)).


<a name="SYNTAXE2"></a>

### Charger les modifications enregistrées dans un fichier

`<Résultat> = <Fenêtre>.EdCharge(<Fichier>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si les modifications ont été chargées, 

- <u><u><u><u>Faux</u></u></u></u> en cas d'erreur. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Fenêtre> : (Nom de fenêtre)`**

Nom de la fenêtre ou de la fenêtre interne à manipuler.

**`<Fichier> : (Chaîne de caractères)`**



- Nom et chemin complet ou relatif du fichier ".wdmodif" contenant les modifications effectuées (les modifications ont été précédemment enregistrées avec la fonction [FenEdSauve](../WDLang1/1000021848.md)). 

- Nom et chemin complet ou relatif du fichier ".wdnote" contenant les notes créées par l'utilisateur (les notes ont été précédemment enregistrées avec la fonction [FenEdSauveNote](../WDLang1/1000022120.md)).




## &lt;Variable Tiff&gt;.SauveTIFF (Fonction)

***En anglais : &lt;Tiff variable&gt;.SaveTIFF***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Sauve une image au format TIFF soit dans un fichier, soit en mémoire.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Chargement du fichier
MonTiff est un Tiff = "MonFichier.tiff"
// Remplacement de la première page
MonTiff.Image[1] = "C:\MesImages\sigle48.bmp"
// Sauvegarde du fichier
MonTiff.SauveTIFF("MonFichierModifié.tiff")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Sauvegarder une image TIFF dans un fichier

`<Résultat> = <Image Tiff>.SauveTIFF(<Nom du fichier>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'image a été sauvegardée,

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour connaître le libellé de l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.




**`<Image Tiff> : (Variable de type Tiff)`**

Nom de la variable de type [Tiff](../WDLang1/1000021025.md) à manipuler.

**`<Nom du fichier> : (Chaîne de caractères)`**

Nom et chemin complet (ou relatif) du fichier TIFF à créer. Un chemin de type UNC peut être utilisé. L'extension du fichier n'est pas obligatoire.

**Attention** :

- Pour que l'image soit sauvée, le répertoire de destination doit exister.

- L'existence du fichier TIFF n'est pas vérifiée. Si ce fichier existe, il est automatiquement écrasé.





<a name="SYNTAXE2"></a>

### Sauvegarder une image TIFF en mémoire

`<Résultat> = <Image Tiff>.Sauve(<enMémoire>)`
---

**`<Résultat> : (Buffer)`**

Chaîne d'octets contenant l'image au format TIFF. Cette chaîne d'octets peut être par la suite affectée à un champ Image, enregistrée dans un fichier (fonction [fSauveTexte](../WDLang1/3036046.md)), ...

**`<Image Tiff> : (Variable de type Tiff)`**

Nom de la variable de type [Tiff](../WDLang1/1000021025.md) à manipuler.

**`<enMémoire> : (Constante)`**

*enMémoire* : Constante permettant de sauvegarder une image en mémoire.


## PiloteDialogue (Fonction)

***En anglais : EmulateDialog***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Pilote le prochain appel de la fonction [Dialogue](../WDLang1/3021015.md), puis clique automatiquement sur le bouton passé en paramètre.

**Attention** : Cette fonction ne peut être utilisée que dans le cadre d'un test automatique. Dans ce cas, le test vérifie que le dialogue est appelé et répond automatiquement. En dehors des tests automatiques, cette fonction ne fait rien.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Fermeture automatique du prochaine message de type Dialogue
// La réponse sur laquelle le clic aura lieu est "Enregistrer" (valeur de retour = 1)
PiloteDialogue(1)

// Traitement 
// ... 
SELON Dialogue("Souhaitez-vous enregistrer les modifications ?")
	// Enregistrer
	CAS 1
	// ...

	// Ne pas enregistrer
	CAS 2
	// ...
FIN
// Le message est fermé automatiquement, par un clic sur "Enregistrer"
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`PiloteDialogue(<Valeur de retour>)`
---

**`<Valeur de retour> : (Entier)`**

Valeur du bouton de la fenêtre Dialogue sur lequel le clic doit avoir lieu.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Vérification de la fonction
<a name="verification_fonction_ELTPARAGRAPHE000038"></a>L'existence d'une fonction de pilotage ([PiloteAvertissement](../WDLang1/1000009001.md), [PiloteErreur](../WDLang1/1000009005.md), [PiloteInfo](../WDLang1/1000009006.md), ...) est vérifiée lors de l'appel de la fonction de dialogue associée ([Avertissement](../WDLang1/3021009.md), [Erreur](../WDLang1/3021013.md), [Info](../WDLang1/3021011.md), ...).


<a name="NOTE0_2"></a>




### Différence de type de message
<a name="difference_type_message_ELTPARAGRAPHE000065"></a>L'ordre des fonctions [PiloteXXX](../WDLang1/1000009002.md) doit respecter l'ordre d'affichage des fonctions [Erreur](../WDLang1/3021013.md), [Info](../WDLang1/3021011.md), ... Dans le cas contraire, une erreur fatale sera déclenchée.

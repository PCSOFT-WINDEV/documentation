


## DocVersTexte (Fonction)

***En anglais : DocToText***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Crée une chaîne de caractères avec les données : 

- d'un champ Traitement de texte ou d'un document Traitement de texte. 

- d'un paragraphe d'un champ Traitement de texte.




Cette chaîne pourra ensuite par exemple être sauvegardée dans un format spécifique.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MaChaîne est une chaîne
MaChaîne = DocVersTexte(TT_MonDocument)
// Sauvegarde dans un fichier texte
fSauveTexte("C:\temp\Contenu.txt", MaChaîne)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = DocVersTexte(<Document>)`
---

**`<Résultat> : (Chaîne de caractères)`**



- Chaîne de caractères contenant les données. 

- Chaîne vide ("") en cas d'erreur ou si il n'y a pas de données à extraire.




**`<Document> : (Variable de type Document ou nom de champ)`**

Document à manipuler. Ce document correspond : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) au nom d'un champ Traitement de texte. 

- à une variable de type [Document](../WDLang1/1000022461.md). 

- à une variable de type [docParagraphe](../WDLang1/1000022483.md).




## &lt;Organigramme&gt;.AjouteFils (Fonction)

***En anglais : &lt;Organization Chart&gt;.AddChild***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ajoute un élément fils dans un champ Organigramme.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ajout d'un élément "Racine" dans le champ Organigramme 
Boss est un entier 
Boss = ORGA_MonOrganigramme.Ajoute("Matthieu Boss", "big boss", "Goldbar.gif")
// Ajout d'un élément fils
ORGA_MonOrganigramme.AjouteFils(Boss, "Production")
```
<a name="Exemple2"></a>

```wl
// Ajout d'éléments dans le champ Organigramme (syntaxe 2)
// Elément Racine
ElémentPDG est un OrgaElément
ElémentPDG.Titre = "Big Boss"
ORGA_DIRECTION.Ajoute(ElémentPDG)

// Eléments fils
Elément2 est un OrgaElément
Elément2.Titre = "Production"
ORGA_DIRECTION.AjouteFils(1, Elément2)

Elément3 est un OrgaElément
Elément3.Titre = "Vente"
ORGA_DIRECTION.AjouteFils(1, Elément3)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Ajouter un élément fils en précisant ses caractéristiques

`<Résultat> = <Champ Organigramme>.AjouteFils(<Indice Parent> , <Titre> [, <Contenu> [, <Image>]])`
---

**`<Résultat> : (Entier)`**



- Indice de l'élément ajouté. 

- -1 en cas d'erreur. La fonction [ErreurInfo](../WDLang1/3013008.md) permet d'identifier l'erreur.




**`<Champ Organigramme> : (Nom de champ)`**

Nom du champ Organigramme dans lequel l'élément fils doit être ajouté.

**`<Indice Parent> : (Entier)`**

Indice de l'élément "père" de l'élément à ajouter. 
Remarque : Pour ajouter un élément à la racine du champ Organigramme, ce paramètre doit être égal à 0.

**`<Titre> : (Chaîne de caractères)`**

Titre de l'élément.

**`<Contenu> : (Chaîne de caractères optionnelle)`**

Contenu de l'élément.

**`<Image> : (Chaîne de caractères optionnelle)`**

Image associée à l'élément. Cette image  s'affiche à gauche du titre de l'élément. Ce paramètre correspond au nom et au chemin complet (ou relatif) de l'image. Un chemin de type UNC peut être utilisé.


<a name="SYNTAXE2"></a>

### Ajouter un élément fils en utilisant le type OrgaElement

`<Résultat> = <Champ Organigramme>.AjouteFils(<Indice Parent> , <Élément>)`
---

**`<Résultat> : (Entier)`**



- Indice de l'élément ajouté.

- -1 en cas d'erreur. La fonction [ErreurInfo](../WDLang1/3013008.md) permet d'identifier l'erreur.




**`<Champ Organigramme> : (Nom de champ)`**

Nom du champ Organigramme dans lequel l'élément fils doit être ajouté.

**`<Indice Parent> : (Entier)`**

Indice de l'élément "père" de l'élément à ajouter. 
Remarque : Pour ajouter un élément à la racine du champ Organigramme, ce paramètre doit être égal à 0.

**`<Élément> : (Variable de type OrgaElément)`**

Nom de la variable de type [OrgaElément](../WDLang1/1000019713.md) décrivant les caractéristiques de l'élément fils à ajouter.

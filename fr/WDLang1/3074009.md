


## Fonctions de gestion des champs Table
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions de gestion des champs Table sont les suivantes :



|   |   |
| --- | --- |
| [ConstruitTableFichier](../WDLang1/3074002.md) | Crée un champ Table à partir des rubriques présentes dans un fichier de données (une vue ou une requête). |
| [FichierVersTableMémoire](../WDLang1/3074005.md) | Remplit un champ de type "Table par programmation" (également appelé "Table mémoire") avec tous les enregistrements d'un fichier de données, d'une vue HFSQL ou d'une requête (requête créée avec l'éditeur de requêtes ou avec la fonction [HExécuteRequêteSQL](../WDLang4/3044084.md)). |
| [TableActiveFiltre](../WDLang1/1000017149.md) | Ajoute un filtre utilisateur sur une colonne d'un champ Table ou Table hiérarchique. |
| [TableAffiche](../WDLang1/3074003.md) | Rafraîchit l'affichage d'un champ Table ou Table hiérarchique à partir d'une position donnée :<br><br>- pour un champ lié à un fichier : les modifications effectuées sur le fichier de données lié sont répercutées dans le champ.<br><br>- pour un champ manipulé par programmation : les colonnes calculées sont re-calculées.<br><br><br> |
| [TableAjoute](../WDLang1/3074017.md) | Ajoute une ligne dans : <br><br>- un champ Table, <br><br>- un champ Table hiérarchique, <br><br>- un champ "Combo avec table" (champ Table affiché dans un champ Combo).<br><br><br> |
| [TableAjouteColonne](../WDLang1/1000017168.md) | Ajoute des éléments à droite d'un champ Table horizontale. |
| [TableAjouteLigne](../WDLang1/3074006.md) | Ajoute une ligne dans : <br><br>- un champ Table, <br><br>- un champ Table hiérarchique. <br><br><br> |
| [TableAjouteLigneAvecAnimation](../WDLang1/1000022402.md) | Ajoute une ligne dans un champ Table ou un champ Table hiérarchique avec une animation. |
| [TableAjoutEnCours](../WDLang1/1000021966.md) | Modifie ou renvoie l'état de visibilité de la fenêtre interne de patience lors de l'ajout d'éléments supplémentaires dans un champ Table infinie. |
| [TableAjouteSurEntête](../WDLang1/1000020585.md) | Ajoute un sur-entête de colonnes dans un champ Table ou dans un champ Table hiérarchique d'une fenêtre. |
| [TableAjuste](../WDLang1/3074004.md) | Ajuste le titre et la taille des colonnes d'un champ Table ou Table hiérarchique. |
| [TableCelluleSelect](../WDLang1/1000019378.md) | Renvoie les coordonnées (numéro de ligne et numéro de colonne) d'une cellule sélectionnée. Cette fonction est conservée par compatibilité. Il est conseillé d'utiliser la fonction TableSelect. |
| [TableCelluleSelectOccurrence](../WDLang1/1000019377.md) | Renvoie le nombre de cellules sélectionnées dans un champ Table. Cette fonction est conservée par compatibilité. Il est conseillé d'utiliser la fonction TableSelectOccurrence. |
| [TableCherche](../WDLang1/3074015.md) | Effectue une recherche dans : <br><br>- un champ Table, <br><br>- un champ Table hiérarchique, <br><br>- une table affichée dans un champ Combo.<br><br><br> |
| [TableCherchePartout](../WDLang1/1410088362.md) | Recherche une valeur dans toutes les colonnes des champs suivants : <br><br>- champ Table,<br><br>- champ Table hiérarchique,<br><br>- champ Table affiché dans un champ Combo.<br><br><br> |
| [TableColonneSelect](../WDLang1/1000019381.md) | Renvoie le numéro de colonne correspondant à l'une des colonnes sélectionnées dans le champ Table. Cette fonction est conservée par compatibilité. Il est conseillé d'utiliser la fonction TableSelect. |
| [TableColonneSelectOccurrence](../WDLang1/1000019380.md) | Renvoie le nombre de colonnes sélectionnées d'une table. Cette fonction est conservée par compatibilité. Il est conseillé d'utiliser la fonction TableSelectOccurrence. |
| [TableColonnesFiltrées](../WDLang1/1000020517.md) | Renvoie la liste des colonnes filtrées par l'utilisateur dans un champ Table ou Table hiérarchique. |
| [TableColonnesTriées](../WDLang1/3074010.md) | Renvoie la liste des colonnes triées d'un champ Table. |
| [TableDébutRemplissage](../WDLang1/1000022368.md) | Indique le début du remplissage d'un champ Table (ou Table hiérarchique) par programmation. |
| [TableDéplaceColonne](../WDLang1/3074011.md) | Déplace une colonne d'un champ Table ou Table hiérarchique. |
| [TableDéplaceLigne](../WDLang1/1000017247.md) | Déplace une ligne ou échange deux lignes dans un champ Table. |
| [TableDésactiveFiltre](../WDLang1/1000017150.md) | Annule un filtre utilisateur sur une colonne ou toutes les colonnes d'un champ Table ou Table hiérarchique. |
| [TableDonneFocus](../WDLang1/1000022319.md) | Donne le focus à une ligne d'un champ Table ou Table hiérarchique : le champ passe en saisie pour la ligne et la colonne spécifiée. |
| [TableEchangeLigne](../WDLang1/1000020627.md) | Echange : <br><br>- deux lignes dans un champ Table par programmation. <br><br>- deux branches dans un champ Table hiérarchique par programmation.<br><br><br> |
| [TableEnregistre](../WDLang1/3074008.md) | Modifie ou ajoute l'enregistrement associé à la ligne en cours dans le champ Table fichier ou Table hiérarchique fichier. |
| [TableEnrouleDéroule](../WDLang1/1000024474.md) | Enroule ou déroule un élément d'un champ Table ou Table hiérarchique. |
| [TableEnumèreColonne](../WDLang1/3074012.md) | Renvoie le nom d'une colonne d'un champ Table ou Table hiérarchique. |
| [TableEnumèreRupture](../WDLang1/1000022418.md) | Enumère les ruptures d'un champ Table. |
| [TableFin](../WDLang1/3074013.md) | Arrête le parcours manuel du fichier de données lié à un champ Table fichier. |
| [TableFinRemplissage](../WDLang1/1000022369.md) | Indique la fin du remplissage d'un champ Table (ou Table hiérarchique) par programmation. |
| [TableFixeColonne](../WDLang1/3074014.md) | Fixe une colonne à gauche ou libère une colonne fixée à gauche. |
| [TableFormuleAjoute](../WDLang1/1000022361.md) | Ajoute une ligne calculée dans un champ Table en fournissant des procédures de calculs personnalisées. |
| [TableFormuleSupprimeTout](../WDLang1/1000022362.md) | Supprime toutes les formules ajoutées avec la fonction [TableFormuleAjoute](../WDLang1/1000022361.md). |
| [TableImprime](../WDLang1/1000017237.md) | Imprime directement le contenu du champ Table spécifié sans avoir réalisé au préalable un état sur le champ Table. |
| [TableIndiceColonne](../WDLang1/3074016.md) | Renvoie l'indice d'une colonne dans un champ Table ou Table hiérarchique. |
| [TableIndiceRupture](../WDLang1/1000018758.md) | Renvoie l'indice du haut et du bas de rupture d'une rupture donnée dans un champ Table. |
| [TableInfoXY](../WDLang1/3074007.md) | Renvoie pour une position donnée dans un champ Table ou Table hiérarchique (coordonnées d'un point du champ) :<br><br>- soit le nom de la colonne affichée pour la position indiquée.<br><br>- soit l'indice de la ligne ou de la colonne pour la position indiquée.<br><br><br> |
| [TableInsère](../WDLang1/3074025.md) | Insère une ligne dans : <br><br>- un champ Table, <br><br>- un champ Table hiérarchique,<br><br>- une table affichée dans un champ Combo.<br><br><br> |
| [TableInsèreColonne](../WDLang1/1000017178.md) | Insère des éléments sous forme de colonne dans ***un champ Table horizontale***. |
| [TableInsèreLigne](../WDLang1/3074026.md) | Insère une ligne dans : <br><br>- un champ Table, <br><br>- un champ Table hiérarchique.<br><br><br> |
| [TableInsèreLigneAvecAnimation](../WDLang1/1000022401.md) | Insère une ligne dans un champ Table ou un champ Table hiérarchique avec une animation. |
| [TableMode](../WDLang1/3074001.md) | La fonction TableMode est conservée uniquement pour compatibilité. |
| [TableModifie](../WDLang1/3074019.md) | Modifie une ligne dans : <br><br>- un champ Table, <br><br>- un champ Table hiérarchique, <br><br>- une table affichée dans un champ Combo.<br><br><br> |
| [TableModifieColonne](../WDLang1/1000017179.md) | Modifie une colonne dans un champ Table horizontale. |
| [TableModifieLigne](../WDLang1/3074028.md) | Modifie une ligne dans : <br><br>- un champ Table, <br><br>- un champ Table hiérarchique. <br><br><br> |
| [TableNumEnr](../WDLang1/3074027.md) | Renvoie le numéro de l'enregistrement en cours dans un champ Table fichier. |
| [TableOccurrence](../WDLang1/3074021.md) | Renvoie : <br><br>- soit le nombre de colonnes présentes dans un champ Table ou Table hiérarchique.<br><br>- soit le nombre de lignes présentes dans un champ Table ou Table hiérarchique. <br><br><br> |
| [TablePosition](../WDLang1/3074022.md) | Affiche un champ Table ou Table hiérarchique à partir d'une ligne spécifiée ou renvoie l'indice de la première ligne affichée dans un champ Table ou Table hiérarchique. |
| [TableRafraîchissementVisible](../WDLang1/1000021196.md) | Modifie ou renvoie l'état de visibilité du bandeau de rafraîchissement d'un champ Table ou Table hiérarchique. |
| [TableRestaurePositionEtSélection](../WDLang1/1000017413.md) | Restaure la position de l'ascenseur d'un champ Table ou Table hiérarchique ainsi que le ou les éléments sélectionnés. |
| [TableSaisieEnCours](../WDLang1/3074023.md) | Indique si un champ Table ou Table hiérarchique est en cours de saisie. |
| [TableSaisieLoupe](../WDLang1/3074024.md) | Passe en saisie la zone de recherche présente dans le titre de colonne du champ Table ou Table hiérarchique. |
| [TableSauvePositionEtSélection](../WDLang1/1000017412.md) | Sauve : <br><br>- la position de l'ascenseur d'un champ Table ou d'un champ Table hiérarchique, <br><br>- le ou les éléments sélectionnés dans le champ.<br><br><br> |
| [TableSelect](../WDLang1/3074020.md) | Renvoie l'indice de l'élément sélectionné dans le champ Table ou Table hiérarchique. |
| [TableSelectMoins](../WDLang1/3074029.md) | Dé-sélectionne une ligne dans un champ Table ou Table hiérarchique. |
| [TableSelectOccurrence](../WDLang1/3074030.md) | Renvoie le nombre d'éléments sélectionnés dans un champ Table ou Table hiérarchique (notamment dans le cas d'un champ multi-sélections). |
| [TableSelectPlus](../WDLang1/3074031.md) | Sélectionne une ligne dans un champ Table ou Table hiérarchique. |
| [TableSelectVersFichier](../WDLang1/3074038.md) | Positionne sur l'enregistrement du fichier de données correspondant à une ligne sélectionnée dans le champ Table fichier multisélection. |
| [TableSupprime](../WDLang1/3074032.md) | Supprime une ligne dans : <br><br>- un champ Table, <br><br>- un champ Table hiérarchique, <br><br>- une table affichée dans un champ Combo.<br><br><br> |
| [TableSupprimeAvecAnimation](../WDLang1/1000022400.md) | Supprime une ligne dans un champ Table, un champ Table hiérarchique (ou une table affichée dans un champ Combo) avec une animation. |
| [TableSupprimeSelect](../WDLang1/1000020506.md) | Supprime la ou les lignes sélectionnées dans un champ Table ou Table hiérarchique. |
| [TableSupprimeTout](../WDLang1/3074034.md) | Supprime toutes les lignes dans : <br><br>- un champ Table par programmation, <br><br>- un champ Table hiérarchique par programmation, <br><br>- une table affichée dans un champ Combo.<br><br><br> |
| [TableTrie](../WDLang1/3074035.md) | Trie un champ Table ou Table hiérarchique sur une ou plusieurs colonnes (tri lexicographique). |
| [TableVersExcel](../WDLang1/3074033.md) | Crée un fichier Excel avec les données d'un champ Table ou Table hiérarchique. |
| [TableVersPressePapier](../WDLang1/1000017062.md) | Copie le contenu d'un champ Table ou Table hiérarchique dans le presse-papiers. |
| [TableVersTexte](../WDLang1/1000017061.md) | Crée une chaîne de caractères avec les données d'un champ Table ou Table hiérarchique. |
| [TableVersWord](../WDLang1/3074036.md) | Crée un fichier Word (.RTF) avec les données d'un champ Table ou Table hiérarchique. |
| [TableVersXML](../WDLang1/3074037.md) | Crée un fichier XML avec les données d'un champ Table ou Table hiérarchique. |

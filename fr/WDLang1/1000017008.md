
## ChaîneVersTableau (Fonction)

***En anglais : StringToArray***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Remplit un tableau à 1 ou 2 dimensions du type spécifié avec la chaîne de caractères passée en paramètre. Le tableau est vidé avant d'être rempli.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Tableau à une dimension
sMesValeurs est une chaîne = "A" + RC + "B" + RC + "C" + RC + "D" + RC + "D"
// Tableau initial
MonTableau est un tableau de 3 chaînes
ChaîneVersTableau(sMesValeurs, MonTableau)
// Le tableau contient : "A" , "B" , "C" , "D" , "D"
```

```wl
// Tableau à une dimension
sChaîne est une chaîne = "A" + RC + "B" + ";" + "C" + RC + "D" + "," + E + RC + "F"
// Tableau initial
MonTableau est un tableau de 6 chaînes
ChaîneVersTableau(sChaîne, MonTableau, [RC, ";", ","]) 
// Le tableau contient : "A" , "B" , "C" , "D" , "E" , "F"
```


<a name="Exemple2"></a>

```wl
// Tableau à deux dimensions
sMesValeurs est une chaîne = "A" + TAB + "B" + RC + "C" +TAB + "D" + RC + "E" + TAB + "F"
// Tableau initial
MonTableau est un tableau de 2 par 3 chaînes
ChaîneVersTableau(sMesValeurs, MonTableau)
// Le tableau contient : "A;B" , "C;D" , "E;F"
```

```wl
// Tableau à deux dimensions
sChaîne est une chaîne = "A" + TAB + "B" + RC + "C" + "/" + "D" + "*" + E + "\" + "F" 
// Tableau initial    
MonTableau est un tableau de 2 par 3 chaînes
ChaîneVersTableau(sChaîne, MonTableau, [RC, "*"], [TAB, "/", "\"]) 
// Le tableau contient : "A;B" , "C;D" , "E;F"
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Tableau à une dimension

`ChaîneVersTableau(<Chaîne> , <Nom du tableau> [, <Séparateur d'élément>])`
---

**`<Chaîne> : (Chaîne de caractères)`**

Contient les différents éléments du tableau. La chaîne sera découpée selon le séparateur d'éléments spécifié.
Si ce paramètre correspond à une chaîne vide (""), la dimension du tableau est ramenée à une dimension et une seule ligne.

**`<Nom du tableau> : (Variable de type Tableau)`**

Nom du tableau à 1 dimension dans lequel les éléments contenus dans la chaîne doivent être ajoutés. Ce tableau doit être alloué dans un type simple ou un type structure.

**`<Séparateur d'élément> : (Chaîne de caractères optionnelle ou Tableau optionnel)`**

Séparateur utilisé dans la &lt;Chaîne&gt; pour séparer les valeurs des différents éléments du tableau. Ce séparateur peut correspondre à : 

- une chaîne de caractères (par exemple : ";").

- un tableau de chaînes (par exemple : [RC,";"]).
	![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Non disponible.


Par défaut, le séparateur d'élément utilisé est RC (Retour Chariot).  


<a name="SYNTAXE2"></a>

### Tableau à 2 dimensions

`ChaîneVersTableau(<Chaîne> , <Nom du tableau> [, <Séparateur de ligne> [, <Séparateur de colonne>]])`
---

**`<Chaîne> : (Chaîne de caractères)`**

Contient les différents éléments du tableau. 
La chaîne sera découpée selon le séparateur de ligne spécifié, puis suivant le séparateur de colonnes. Les éléments seront ajoutés ligne par ligne. 

- Il y aura autant de lignes que de &lt;Séparateur de ligne&gt; trouvés. 

- Des colonnes seront ajoutées si le nombre maximal de &lt;Séparateur de colonne&gt; trouvé dépasse le nombre de colonnes du tableau. 

- A chaque colonne doit correspondre une valeur pour la ligne. 

- Le nombre de colonnes du tableau résultat est le nombre d'éléments de la première ligne.

- Si une ligne contient plus d'éléments que la première ligne, les éléments en trop sont concaténés dans le dernier élément. 

- Si une ligne contient moins d'éléments que la première ligne, les cases du tableau non affectées conservent la valeur par défaut.

- Si ce paramètre correspond à une chaîne vide (""), la dimension du tableau est ramenée à une dimension et une seule ligne.




**`<Nom du tableau> : (Variable de type Tableau)`**

Nom du tableau à 2 dimensions dans lequel les éléments contenus dans la chaîne doivent être ajoutés. Ce tableau doit être alloué dans un type simple ou un type structure.

**`<Séparateur de ligne> : (Chaîne de caractères optionnelle ou Tableau optionnel)`**

Séparateur utilisé dans la &lt;Chaîne&gt; pour séparer les valeurs des différentes lignes. Ce séparateur peut correspondre à : 

- une chaîne de caractères (par exemple : ";").

- un tableau de chaînes (par exemple : [RC,"\*"]).
	![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Non disponible.


Par défaut, le séparateur de lignes utilisé est RC (Retour Chariot).

**`<Séparateur de colonne> : (Chaîne de caractères optionnelle ou Tableau optionnel)`**

Séparateur utilisé dans la &lt;Chaîne&gt; pour séparer les valeurs des différentes colonnes. Ce séparateur peut correspondre à : 

- une chaîne de caractères (par exemple : "TAB").

- un tableau de chaînes (par exemple : ["/", "\\"]).
	![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Non disponible.


Par défaut, le séparateur de colonnes utilisé est TAB.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- La dimension du tableau sera modifiée si nécessaire : des lignes et des colonnes pourront être ajoutées selon les valeurs contenues dans la &lt;Chaîne&gt;.

- Les tableaux de type structure gérés sont uniquement des tableaux à une dimension.

- Les tableaux contenant des chaînes UNICODE ou des variants ne sont pas gérés.

- Pour le type Durée, la valeur doit être fournie en millièmes de seconde. La valeur 0 n'est pas valide. Il faut utiliser "0000000000".


## StatErreur (Fonction)

***En anglais : StatError***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Calcule le numéro de la dernière erreur due à une fonction Statxx ([fonctions statistiques](../WDLang1/3050027.md)).




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple
<a class="notetitle" target="_blank" href="$DOC$=1000003050019&name=staterreur_fonction&product=WD">Voir des exemples supplémentaires</a>

```wl
// Identification du type d'erreur
ResErreur = StatErreur()
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = StatErreur()`
---

**`<Résultat> : (Entier)`**

Numéro de la dernière erreur due à une fonction statistique :


|   |   |   |
| --- | --- | --- |
| 0 | Pas d'erreur | Toutes les fonctions |
| 1 | Matrice vide | Toutes les fonctions |
| 2 | Deux valeurs minimum sont nécessaires dans la ligne ou la colonne de la matrice | [StatVariance](../WDLang1/3050033.md),<br>[StatEcartMoyen](../WDLang1/3050024.md) |
| 3 | Moyenne géométrique négative | [StatMoyenne](../WDLang1/3050036.md),<br>[StatEcartType](../WDLang1/3050025.md) |
| 4 | Division par zéro | [StatMoyenne](../WDLang1/3050036.md),<br>[StatCorrélation](../WDLang1/3050022.md) |
| 5 | La matrice n'existe pas ou les valeurs limites sont dépassées | Toutes les fonctions |







<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Le libellé de l'erreur peut être connu grâce à la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.

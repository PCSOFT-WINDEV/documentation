


## PlanningSupprimeRessource (Fonction)

***En anglais : SchedulerDeleteResource***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime une ressource dans un champ Planning. 

**Attention** : Si le champ Planning est lié à une source de données, la suppression est réalisée uniquement dans le champ : les rendez-vous liés à la ressource ne sont pas supprimés de la source de données. 



![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) En mode Universal Windows 10 App, cette fonction permet uniquement de manipuler les champs Planning présents dans un état. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ajoute un rendez-vous pour ce soir
MonTitre est une chaîne
DébutRDV est un DateHeure
FinRDV est un DateHeure
NBressource est un entier

MonTitre = "Réunion commerciale"
DébutRDV = DateSys() + "17000000"
FinRDV = DateSys() + "17300000"

NBressource = PlanningAjouteRessource(PLN_Planning1, "Florence" + gValeurMémorisée("Flo"))
PlanningAjouteRendezVous(PLN_Planning1, "Flo", MonTitre, DébutRDV)

MonTitre = "Réunion 2"
DébutRDV = DateSys() + "17300000"
FinRDV = DateSys() + "18000000"
PlanningAjouteRessource(PLN_Planning1, "Emmanuel" + gValeurMémorisée("Manu"))
PlanningAjouteRendezVous(PLN_Planning1, "Manu", MonTitre, DébutRDV)

// Supprime une des ressources
ResSup est un booléen
ResSup = PlanningSupprimeRessource(PLN_Planning1, "Flo")
SI ResSup = Vrai ALORS
	Info("Ressource supprimée")
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = PlanningSupprimeRessource(<Champ Planning> , <Ressource>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si une ressource a été supprimée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 




**`<Champ Planning> : (Nom de champ)`**

Nom du champ Planning dans lequel la ressource doit être supprimée. Ce champ peut correspondre : 

- au champ Planning d'une fenêtre, 

- au champ Planning d'une page,

- au champ Planning d'un état.




**`<Ressource> : (Chaîne de caractères)`**

Nom de la ressource à supprimer.  

**Remarque** : Si les ressources du champ Planning ont été créées avec la fonction [PlanningAjouteRessource](../WDLang1/1000019279.md) en utilisant la fonction [gValeurMémorisée](../WDLang1/1410088363.md), ce paramètre doit correspondre à la valeur précisée dans la fonction [gValeurMémorisée](../WDLang1/1410088363.md) pour cette ressource.

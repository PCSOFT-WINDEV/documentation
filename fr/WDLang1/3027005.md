


## EntierVersMoisEnLettre (Fonction)

***En anglais : IntegerToMonthInAlpha***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le mois en toutes lettres correspondant à la date spécifiée (cette date est un entier).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
Trace(EntierVersMoisEnLettre(DateVersEntier("20131225"))) // Renvoie "Décembre"
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = EntierVersMoisEnLettre(<Date>)`
---

**`<Résultat> : (Chaîne de caractères)`**

Mois correspondant à la date spécifiée : Janvier, Février, Mars, Avril, Mai, Juin, Juillet, Août, Septembre, Octobre, Novembre, Décembre.

**`<Date> : (Entier)`**

Nombre de jours écoulés depuis le 1er janvier 1800 + 1 (peut être obtenu grâce à la fonction [DateVersEntier](../WDLang1/3027014.md)).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Il n'est pas possible de récupérer le mois des dates antérieures au 14 octobre 1582 (le passage du calendrier Julien au calendrier grégorien ne sera pas pris en compte). 

- La fonction **EntierVersMoisEnLettre** tient compte des options linguistiques spécifiées dans la description du projet : le nom du mois sera dans la langue spécifiée dans l'onglet "Langues" de la description du projet. 
	Rappel : Pour afficher la description du projet, sous le volet "Projet", dans le groupe "Projet", cliquez sur "Description".

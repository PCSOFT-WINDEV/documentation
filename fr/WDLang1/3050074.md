
## MatLit (Fonction)

***En anglais : MatRead***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Lit la valeur d'un élément d'une matrice.

**Rappel** : Un élément d'une matrice est identifié par sa ligne et sa colonne.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple
<a class="notetitle" target="_blank" href="$DOC$=1000003050074&name=matlit_fonction&product=WD">Voir des exemples supplémentaires</a>

```wl
// Lecture de l'élément (ligne 3, colonne 5)
ResLecture = MatLit("MaMatrice", 3, 5)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = MatLit(<Matrice> , <Ligne de l'élément> , <Colonne de l'élément>)`
---

**`<Résultat> : (Réel)`**

Valeur de l'élément lu.

**`<Matrice> : (Chaîne de caractères)`**

Nom de la matrice à lire, défini avec la fonction [MatCrée](../WDLang1/3050058.md).

**`<Ligne de l'élément> : (Entier)`**

Numéro de la ligne de l'élément à lire.

**`<Colonne de l'élément> : (Entier)`**

Numéro de la colonne de l'élément à lire.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Existence de la matrice
<a name="existence_matrice_ELTPARAGRAPHE000084"></a>L'existence de la matrice n'est pas vérifiée. Pour vérifier l'existence d'une matrice, utilisez la fonction [MatExiste](../WDLang1/3050076.md).


<a name="NOTE0_2"></a>




### Gestion des erreurs
<a name="gestion_des_erreurs_ELTPARAGRAPHE000096"></a>**Attention** : La fonction **MatLit** ne retourne pas de code d'erreur. 

Pour savoir si des erreurs ont été générées par la lecture d'un élément, utilisez la fonction [MatErreur](../WDLang1/3050044.md). Le libellé de l'erreur peut être connu grâce à la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.

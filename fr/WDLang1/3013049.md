
## Constantes utilisées par la fonction TraceDébut
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
La fonction [TraceDébut](../WDLang1/3013052.md) utilise les constantes suivantes :

| Constante | Détail |
| --- | --- |
| **trCréation** | Vide la fenêtre de trace |
| **trDessusTout** | Fenêtre de trace affichée au dessus de toutes les fenêtres système |
| **trDésactive** | Tous les appels à la fonction [Trace](../WDLang1/3013050.md) seront ignorés. |
| **trFenêtre** | La fenêtre de Trace s'affiche en haut à gauche de l'écran.<br><br><br><br><br><br><br> |
| **trFichier** | La fenêtre de Trace ne s'affichera plus et les informations contenues dans la fenêtre de Trace seront enregistrées dans le fichier &lt;Fichier de trace&gt;.<br>![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) Seule cette constante est disponible. Par défaut, ce fichier est créé dans le répertoire temporaire d'UNIX, sous le nom "Trace_&lt;Projet&gt;.txt". |
| **trFenêtre** + **trFichier** | La fenêtre de trace s'affichera et le contenu de la fenêtre de trace sera également enregistré dans le fichier &lt;Fichier de Trace&gt; |




## &lt;Chrono&gt;.Fin (Fonction)

***En anglais : &lt;Chrono&gt;.End***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Arrête un chronomètre et renvoie le temps écoulé (en millisecondes) depuis le début du chronométrage (appel de la fonction [&lt;Chrono&gt;.Début](../WDLang1/1000023788.md)). Ce chronomètre pourra si nécessaire être relancé avec la fonction [&lt;Chrono&gt;.Reprend](../WDLang1/1000023792.md). 








<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple






```wl
MonChrono1 est un Chrono
MonChrono2 est un Chrono
// Démarrage du premier chronomètre 
MonChrono1.Début()
// Démarrage du second chronomètre 
MonChrono2.Début()
// Arrêt premier chronomètre
DuréeChronoDébut est une Durée = MonChrono1.Fin()
MaFonction()
// Arrêt du second chronomètre
DuréeMaFonction est une Durée = MonChrono2.Fin()
Info("Le traitement MaFonction() a duré " + DuréeMaFonction, ...
	"Le traitement complet a duré " + DuréeChronoDébut)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Terminer le chronométrage réalisé avec une variable de type Chrono

`<Résultat> = <Chronomètre>.Fin()`
---

**`<Résultat> : (Durée)`**

Temps passé (en millisecondes) depuis le dernier appel à la fonction [&lt;Chrono&gt;.Début](../WDLang1/1000023788.md) pour le chronomètre spécifié.

**`<Chronomètre> : (Variable de type Chrono)`**

Nom de la variable de type [Chrono](../WDLang1/1000023560.md) correspondant au chronomètre à utiliser.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Divers
<a name="divers_ELTPARAGRAPHE000303"></a>Si la fonction **&lt;Chrono&gt;.Fin** est appelée deux fois sur le même chronomètre sans avoir appelé la fonction [&lt;Chrono&gt;.Début](../WDLang1/1000023788.md), la durée renvoyée sera identique lors des deux appels (le chronomètre est arrêté lors du premier appel à la fonction **&lt;Chrono&gt;.Fin**).
<a name="NOTE0_2"></a>


### Erreur
<a name="erreur_ELTPARAGRAPHE000324"></a>Si la fonction [&lt;Chrono&gt;.Début](../WDLang1/1000023788.md) n'a pas été appelée avant la fonction **&lt;Chrono&gt;.Fin**, une erreur WLangage est affichée (erreur de type "le chronomètre n'est pas démarré").
<a name="NOTE0_3"></a>


### Threads et tâches parallèles
<a name="threads_taches_paralleles_ELTPARAGRAPHE000342"></a>Chaque thread (et chaque tâche parallèle) possède ses propres chronomètres. 

Deux threads distincts (ou deux tâches parallèles distinctes) peuvent réaliser des chronométrages séparés en utilisant un même numéro de chronomètre. 

L'erreur WLangage "le chronomètre n'est pas démarré" sera affichée uniquement si la fonction [&lt;Chrono&gt;.Début](../WDLang1/1000023788.md) et la fonction **&lt;Chrono&gt;.Fin** ne sont pas appelées dans le même thread.

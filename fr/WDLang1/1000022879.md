


## &lt;Agenda&gt;.ChangeMode (Fonction)

***En anglais : &lt;Organizer&gt;.ChangeMode***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie la plage temporelle affichée par un champ Agenda (Jour/Semaine/Mois).







![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) En mode Universal Windows 10 App, cette fonction permet uniquement de manipuler les champs Agenda présents dans un état. 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affiche les quatre premiers jours de la semaine
AGD_MonAgenda.ChangeMode(agzSemaine, 4)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Champ Agenda>.ChangeMode(<Plage d'affichage> [, <Nombre de jours>])`
---

**`<Champ Agenda> : (Nom de champ)`**

Nom du champ Agenda à manipuler. Ce champ peut correspondre : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) au champ Agenda d'une fenêtre.

- au champ Agenda d'un état.




**`<Plage d'affichage> : (Constante de type Entier)`**

Plage de temps à afficher dans le champ Agenda :


|   |   |
| --- | --- |
| *agzJour* | Affiche l'agenda sur une journée. |
| *agzMois* | Affiche l'agenda sur un mois. |
| *agzNbJourPersonnalisé* | Affiche l'agenda sur le nombre de jours indiqués dans le paramètre &lt;Nombre de jours&gt;.<br><br> |
| *agzSemaine* | Affiche l'agenda sur le nombre de jours indiqués dans le paramètre &lt;Nombre de jours&gt;. |
| *agzSemaineSur5* | Affiche l'agenda sur 5 jours (le week-end n'est pas affiché). |



**`<Nombre de jours> : (Entier optionnel)`**

Nombre de jours affichés. Si le paramètre  &lt;Plage d'affichage&gt; est égal à  : 

- *agzSemaine* : Nombre de jours affichés dans la semaine. Si ce paramètre n'est pas précisé, l'agenda sera affiché sur une semaine complète. 

- *agzNbJourPersonnalisé* : Nombre de jours affichés dans le champ Agenda. 


Dans les autres cas, une erreur WLangage est affichée.

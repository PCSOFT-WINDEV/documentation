


## &lt;Arbre&gt;.Déroule (Fonction)

***En anglais : &lt;TreeView&gt;.Expand***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Déroule un noeud dans un champ Arbre. Les éléments "fils" (noeuds et feuilles) de ce noeud sont alors visibles.

Exemple de champ Arbre :
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Arbre_Vocabulaire.gif)
**Rappel** : En exécution, un noeud est déroulé lorsque le signe "-" précède le nom du noeud.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Déroule le noeud "Desserts" dans le champ Arbre "ARBRE_RecetteTV"
Res = ARBRE_RecetteTV.Déroule("Recettes" + TAB + "Desserts")
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Champ Arbre>.Déroule(<Noeud à dérouler>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le noeud a été déroulé, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Champ Arbre> : (Nom de champ)`**

Nom du champ Arbre à manipuler.

**`<Noeud à dérouler> : (Chaîne de caractères)`**

Chemin du noeud à dérouler. Ce paramètre est de la forme :

```txt
"<Nom de la racine>" + TAB + ["<Nom du 1er noeud>" + TAB + ...
["<Nom du 2ème noeud>" + TAB + [...]]]"<Nom de la feuille>"
```
En cas de doublons sur un élément du chemin, ce chemin peut contenir l'identifiant de l'élément (indiqué à la suite du nom grâce à la fonction [ArbreID](../WDLang1/3018027.md)).



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Noeud à dérouler
<a name="noeud_derouler_ELTPARAGRAPHE000192"></a>Si le noeud à dérouler :

- est introuvable, la fonction **&lt;Arbre&gt;.Déroule** renvoie <u><u><u><u>Faux</u></u></u></u>.

- est une feuille, la fonction **&lt;Arbre&gt;.Déroule** renvoie <u><u><u><u>Vrai</u></u></u></u>.

- est déjà déroulé, la fonction **&lt;Arbre&gt;.Déroule** renvoie <u><u><u><u>Vrai</u></u></u></u>.

- n'est pas visible, la fonction **&lt;Arbre&gt;.Déroule** rend ce noeud visible. Ses "parents" et ses "frères" sont également affichés. Les éléments "parents" sont déroulés.


Lors de l'appel à la fonction **&lt;Arbre&gt;.Déroule**, si la branche spécifiée possède une procédure associée par la propriété [AjouteFilsDifféré](../Proprietes/1000020927.md), cette procédure sera exécutée.
<a name="NOTE0_2"></a>


### Bandeau de sélection
<a name="bandeau_selection_ELTPARAGRAPHE000241"></a>La fonction **&lt;Arbre&gt;.Déroule** ne modifie pas la position du bandeau de sélection.

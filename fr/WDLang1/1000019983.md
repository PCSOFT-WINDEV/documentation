


## &lt;Organigramme&gt;.SelectPlus (Fonction)

***En anglais : &lt;Organization Chart&gt;.SelectPlus***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Sélectionne un élément dans un champ Organigramme. Si l'élément à sélectionner ne se trouve pas dans la zone visible du champ Organigramme, un scroll est automatiquement réalisé.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Récupère l'indice de l'élément sélectionné dans un champ Organigramme
NumElément est un entier 
NumElément = ORGA_MonOrganigramme.InfoXY(SourisPosX(), SourisPosY())
SI NumElément <> -1 ALORS
	ORGA_MonOrganigramme.SelectPlus(NumElément)
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Champ Organigramme>.SelectPlus(<Indice>)`
---

**`<Champ Organigramme> : (Nom de champ)`**

Nom du champ Organigramme à manipuler.

**`<Indice> : (Entier)`**

Indice de l'élément à sélectionner. Cet indice est renvoyé par exemple par la fonction [&lt;Organigramme&gt;.InfoXY](../WDLang1/1000019977.md).


## SysInfoAppareil (Fonction)

***En anglais : SysDeviceInfo***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Récupère des informations sur l'appareil mobile sur lequel s'exécute l'application
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
soit sNomModèle = SysInfoAppareil(sysModèle)
Trace(sNomModèle)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = SysInfoAppareil(<Type d'information>)`
---

**`<Résultat> : (Chaîne de caractères)`**

Information demandée. 

**`<Type d'information> : (Constante de type Entier)`**

Type d'information demandée :



|   |   |
| --- | --- |
| *sysFabricant* | Fabricant de l'appareil. |
| *sysModèle* | Modèle de l'appareil. |




## &lt;Variable Document&gt;.RedoOccurrence (Fonction)

***En anglais : &lt;Document variable&gt;.RedoCount***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le nombre d'actions annulées qui peuvent être refaites sur une variable de type [Document](../WDLang1/1000022461.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonDocument est un Document

// Affiche le nombre d'actions annulées et qui peuvent être refaites
Info(MonDocument.RedoOccurrence())
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Document>.RedoOccurrence()`
---

**`<Résultat> : (Entier)`**

Nombre d'actions annulées qui peuvent être refaites.

**`<Document> : (Variable de type Document)`**

Nom de la variable de type [Document](../WDLang1/1000022461.md) à manipuler.

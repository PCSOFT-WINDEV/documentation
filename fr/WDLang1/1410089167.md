
## kbCarte (Type de variable)

***En anglais : kbCard***
			
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **kbCarte** permet de définir toutes les caractéristiques avancées d'une carte utilisée dans un champ Kanban. Les caractéristiques de cette carte peuvent être définies et modifiées à l'aide de différentes propriétés WLangage. 

Les cartes correspondent aux éléments affichés dans les listes d'un champ Kanban. Ces cartes peuvent : 

- être ajoutées dans une liste d'un champ Kanban (fonction [KanbanAjouteCarte](../WDLang1/1410089140.md)). 

- être passées en paramètre d'événements WLangage (par exemple, événement "Déplacement d'une carte" du champ Kanban). 




**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MaCarte est un kbCarte
MaCarte.Titre = "Dossier"
MaCarte.Texte = "Gestion des dossiers clients"
MaCarte.Extra.ID = DonneIdentifiant()
KanbanAjouteCarte(KANBAN_Planning.LISTE_AFaire, MaCarte)
```





<a name="PROPS"></a>

## Propriétés


### Propriétés spécifiques à la description des variables de type kbCarte
<a name="proprietes_specifiques_description_des_variables_type_kbcarte_ELTPARAGRAPHE000034"></a>Les propriétés suivantes peuvent être utilisées pour manipuler une carte d'un champ Kanban :

| Nom de la propriété | Type manipulé | Effet |
| --- | --- | --- |
| ActionAutorisée | [kbActionAutorisée](../WDLang1/1410089541.md) | Action autorisée sur la carte.<br><br> |
| Date | DateHeure | Date et heure de la carte. Par défaut, cette propriété est initialisée avec les date et heure de création. |
| Extra | Tableau associatif | Tableau associatif libre pour enregistrer des informations utilisateur. |
| FenêtreInterne | Chaîne de caractères | Nom de la fenêtre interne à utiliser pour afficher la carte. Si ce paramètre correspond à chaîne vide (""), la carte utilisée sera la carte par défaut du champ Kanban. |
| Indice | Entier | Priorité de la carte. Correspond à l'ordre dans la liste. <br>Pour modifier la priorité d'une carte, utilisez la fonction [KanbanDéplaceCarte](../WDLang1/1410089144.md). <br><br>**Cette propriété est disponible en lecture seulement.** |
| Intervenant | Chaîne de caractères | Nom de l'intervenant associé à la carte. |
| NomListe | Chaîne de caractères | Nom de la liste qui contient la carte. <br>Pour modifier la liste associée à une carte, utilisez la fonction [KanbanDéplaceCarte](../WDLang1/1410089144.md). <br><br>**Cette propriété est disponible en lecture seulement.** |
| PageInterne | Chaîne de caractères | Nom de la page interne à utiliser pour afficher la carte. Si ce paramètre correspond à chaîne vide (""), la carte utilisée sera la carte par défaut du champ Kanban. |
| Texte | Chaîne de caractères | Texte de la carte. Par défaut, ce texte est au format Markdown. |
| Titre | Chaîne de caractères | Titre de la carte. |
| Visible | Booléen | <br><br>- <u><u><u>Vrai</u></u></u> si la carte doit être visible, <br><br>- <u><u><u>Faux</u></u></u> dans le cas contraire. <br><br><br><br><br> |

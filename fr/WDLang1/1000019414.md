
## HTMLVersRVB (Fonction)

***En anglais : HTMLToRGB***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Convertit la couleur HTML en couleur RVB.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
Trace(HTMLVersRVB("AliceBlue"))	// Renvoie 16775408
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = HTMLVersRVB(<Couleur>)`
---

**`<Résultat> : (Entier)`**

Couleur RVB correspondant à la couleur HTML spécifiée. Cette couleur est le résultat de la formule suivante : 
Couleur = 65536 \* Composante Bleue + 256 \* Composante Verte + Composante Rouge.

**`<Couleur> : (Chaîne de caractères)`**

Couleur HTML exprimée dans un des formats suivants :  

- notation hexadécimale (par exemple #f00 ou #ff0000).

- notation RVB (par exemple RVB(100%,0%,0%)).

- nom HTML de la couleur (voir liste ci-dessous).




## XamlDésactive (Fonction)

***En anglais : XamlDisable***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Simule l'absence du framework DotNet 3.0.

La présence du framework DotNet 3.0 est nécessaire à l'utilisation du champ Carrousel, du champ Cube et du champ Xaml. Lors du développement d'une application contenant un champ Carrousel, un champ Cube et/ou un champ Xaml, il peut être utile de savoir comment réagira l'application sur une machine ne disposant pas du framework DotNet 3.0 ou supérieur. La fonction **XamlDésactive** permet d'effectuer ce test.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Simuler l'absence du framework DotNet 3.0
SI EnModeTest() = Vrai ALORS
	XamlDésactive()
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`XamlDésactive()`
---



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
L'utilisation de la fonction **XamlDésactive** est équivalente au renommage de la clé de registre :

```txt
"HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\.NETFramework\v3.0"
```


## TableurSelectCommentaire (Fonction)

***En anglais : SpreadsheetSelectComment***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de : 

- savoir si un commentaire est sélectionné dans un champ Tableur, 

- sélectionner ou désélectionner un commentaire dans un champ Tableur. 



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Sélectionner le commentaire de la cellule A1
TableurSelectCommentaire(TBLR_MonTableur, "A1")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Savoir si un commentaire est sélectionné

`<Résultat> = TableurSelectCommentaire(<Champ Tableur>)`
---

**`<Résultat> : (Chaîne de caractères)`**



- Nom de la cellule qui a un commentaire sélectionné (par exemple "A1"), 

- Chaîne vide ("") si aucun commentaire n'est sélectionné.  




**`<Champ Tableur> : (Nom de champ)`**

Nom du champ Tableur à manipuler. 


<a name="SYNTAXE2"></a>

### Sélectionner ou désélectionner un commentaire

`TableurSelectCommentaire(<Champ Tableur> , <Position>)`
---

**`<Champ Tableur> : (Nom de champ)`**

Nom du champ Tableur à manipuler. 

**`<Position> : (Chaîne de caractères)`**



- Nom de la cellule (par exemple "A1") dont le commentaire doit être sélectionné, forcé visible et en saisie. Si la cellule n'a pas de commentaire, le commentaire est créé. 

- Chaîne vide (""). Dans ce cas, si un commentaire était sélectionné, il est désélectionné et la saisie effectuée dans le commentaire est validée. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Un seul commentaire peut être sélectionné à la fois dans un champ Tableur.

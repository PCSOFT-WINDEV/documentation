


## &lt;Type Date&gt;.VersEntier (Fonction)

***En anglais : &lt;Date type&gt;.ToInteger***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Transforme une date en un entier. Cet entier correspond :

- soit au nombre de jours écoulés entre le 1er janvier 1800 et la date spécifiée.

- soit au nombre de jours écoulés entre le 1er janvier de la date spécifiée et la date spécifiée.











<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MaDate est une Date
// Nombre de jours écoulés entre le 01/01/1800 et le 25/12/2001
MaDate = "20011225"
MaDate.VersEntier()   // Retourne 73773

// Nombre de jours écoulés entre le 01/01/1800 et le 03/01/1800
MaDate = "18000103"
MaDate.VersEntier()   // Retourne 3
```

## Syntaxe

`<Résultat> = <Date>.VersEntier([<Depuis début année>])`
---

**`<Résultat> : (Entier)`**

Nombre de jours écoulés entre : 

- soit le 1er janvier de la date spécifiée et la date spécifiée.

- soit le 1er janvier 1800 et la date spécifiée




**`<Date> : (Variable de type Date ou DateHeure)`**

Date à convertir. Cette date peut correspondre à : 

- une variable de type Date. 

- une variable de type DateHeure. Dans ce cas, seule la date est prise en compte.




**`<Depuis début année> : (Booléen optionnel)`**



- <u><u><u><u>Vrai</u></u></u></u> pour obtenir le nombre de jours écoulés entre le 1er janvier de la date spécifiée et la date spécifiée, 

- <u><u><u><u>Faux</u></u></u></u> (valeur par défaut) pour obtenir le nombre de jours écoulés entre le 1er janvier 1800 et la date spécifiée.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- La validité de la date passée en paramètre est vérifiée. Un message d'erreur est affiché en cas de date invalide. Il est possible de tester la validité d'une date avec la fonction [DateValide](../WDLang1/3027003.md).

- Les fonction **&lt;Type Date&gt;.VersEntier** et [DateDifférence](../WDLang1/3027001.md) peuvent être utilisées pour calculer le nombre de jours écoulés entre deux dates. 

- Le format de stockage des dates permet de stocker des dates comprises entre le 01/01/0001 et le 31/12/9999. Les calculs sur les dates réalisés avec les [fonctions WLangage](../WDLang1/1000023943.md) ou les [propriétés WLangage](../Proprietes/2514010.md) sont fiables à partir du 1er Janvier 1583.

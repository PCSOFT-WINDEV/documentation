


## TableurSelectPlus (Fonction)

***En anglais : SpreadsheetSelectPlus***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Sélectionne une ou plusieurs cellules dans un champ Tableur présent dans une fenêtre. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
TableurSelectMoins(TBLR_Tableur)
TableurSelectPlus(TBLR_Tableur, "A2", "B5")
TableurStyleSélection(TBLR_Tableur, tblrCouleurFond, RougeClair)
```
<a name="Exemple2"></a>

```wl
// Sélectionner une plage de cellules pour les fusionner
TableurSelectPlus(TBLR_Tableur, "B1", "D1")
// Fusion des cellules 
FAAExécute(TBLR_Tableur, faaTableurFusion)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`TableurSelectPlus(<Champ Tableur> [, <Cellule début> [, <Cellule fin>]])`
---

**`<Champ Tableur> : (Nom de champ)`**

Nom du champ Tableur à manipuler. 

**`<Cellule début> : (Chaîne de caractères optionnelle)`**

Nom de la cellule à sélectionner. Ce paramètre est au format &lt;Lettre de la colonne&gt;&lt;Numéro de la ligne&gt;. Par exemple : "A32". 

Si ce paramètre correspond : 

- au nom d'une colonne (par exemple "A"), la colonne spécifiée est entièrement sélectionnée. 

- au numéro d'une ligne (par exemple "2"), la ligne spécifiée est entièrement sélectionnée. 

- à chaîne vide (""), toutes les cellules du champ Tableur sont sélectionnées.


Si ce paramètre n'est pas précisé, toutes les cellules et les entêtes de lignes et de colonnes sont sélectionnés. 

Si le paramètre &lt;Cellule fin&gt; est précisé, &lt;Cellule début&gt; correspond à la première cellule à sélectionner d'une plage de cellules.  

Si le paramètre &lt;Cellule fin&gt; n'est pas précisé, seule &lt;Cellule début&gt; est sélectionnée. 

**`<Cellule fin> : (Chaîne de caractères optionnelle)`**

Nom de la cellule correspondant à la fin de la plage de cellules à sélectionner. Ce paramètre est au format &lt;Lettre de la colonne&gt;&lt;Numéro de la ligne&gt;. Par exemple : "A32".  

Si ce paramètre n'est pas précisé, seule la cellule &lt;Cellule début&gt; est sélectionnée. 

Si le paramètre &lt;Cellule fin&gt; correspond à une cellule positionnée avant &lt;Cellule début&gt;, les coordonnées sont inversées et la sélection est réalisée. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- La sélection réalisée est ajoutée à la sélection courante. 

- La sélection est manipulée par les fonctions [TableurCadreSélection](../WDLang1/1000021480.md), [TableurTypeEtMasqueSélection](../WDLang1/1000021349.md), [TableurStyleSélection](../WDLang1/1000021338.md). Il est également possible d'utiliser la fonction [FAAExécute](../WDLang1/1000022099.md) (par exemple, avec la constante *faaTableurFusion* pour fusionner les cellules sélectionnées).

- Pour sélectionner une seule cellule et désélectionner toutes les autres, il suffit d'utiliser la syntaxe suivante :
	
	```wl
	TBLR_Tableur = "A3"
	```


- Cette fonction est utilisable uniquement sur un champ Tableur présent dans une fenêtre.

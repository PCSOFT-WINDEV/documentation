
## MessageSupprimeCase (Fonction)

***En anglais : StatusBarDeleteCell***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime une case de la barre de message de la fenêtre en cours.

**Remarque** : La fenêtre doit posséder une barre de message.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ajoute la case "MaCase" dans la barre de message de la fenêtre en cours
MessageAjouteCase("MaCase", 100, RougeClair)
...
MessageSupprimeCase("MaCase")
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`MessageSupprimeCase(<Nom de la case>)`
---

**`<Nom de la case> : (Chaîne de caractères)`**

Nom de la case à supprimer. Ce nom peut correspondre à une case de la barre de message créée sous l'éditeur ou par programmation (fonction [MessageAjouteCase](../WDLang1/3021001.md)). Si la case n'existe pas, une erreur WLangage est affichée.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Supprimer une case d'une barre de message
<a name="supprimer_une_case_une_barre_message_ELTPARAGRAPHE000038"></a>Il est conseillé de rendre la case invisible plutôt que de la supprimer.


<a name="NOTE0_2"></a>




### Utiliser les propriétés WLangage avec les cases des barres de message
<a name="utiliser_les_proprietes_wlangage_avec_les_cases_des_barres_message_ELTPARAGRAPHE000047"></a>De nombreuses propriétés WLangage peuvent être utilisées sur les cases des barres de message.

**Attention** : Pour utiliser une propriété sur une case d'une barre de message, la syntaxe à utiliser est la suivante :

```txt
{"Nom de la case"}..<Nom de la propriété>
```


Pour plus de détails, consultez [Liste des propriétés utilisables sur les cases de barre de message](../WDChamp/1010035.md).

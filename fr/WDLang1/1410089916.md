


## Fonctions de gestion des chaînes de caractères (syntaxe préfixée)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions permettant de manipuler des chaînes de caractères via une syntaxe préfixée sont les suivantes :




|   |   |
| --- | --- |
| [&lt;Type Buffer&gt;.AnsiVersUnicode](../WDLang1/1000026033.md) | Convertit un buffer contenant une chaîne de caractères au format ANSI (Windows) en un buffer contenant une chaîne de caractères au format UNICODE. |
| [&lt;Type Chaîne Ansi&gt;.AnsiVersOem](../WDLang1/1000024891.md) | Convertit une chaîne de caractères au format ANSI (Windows) en une chaîne de caractères au format OEM (DOS). |
| [&lt;Type Chaîne Ansi&gt;.Complète](../WDLang1/1410089823.md) | Renvoie une chaîne de caractères spécifique de taille déterminée. |
| [&lt;Type Chaîne Ansi&gt;.Contient](../WDLang1/1410089824.md) | Permet de savoir si une sous-chaîne est contenue dans une chaîne. |
| [&lt;Type Chaîne Ansi&gt;.Droite](../WDLang1/1410089825.md) | Extrait la partie droite (c'est-à-dire les derniers caractères) d'une chaîne de caractères ou d'un buffer. |
| [&lt;Type Chaîne Ansi&gt;.SupprimeDoublon](../WDLang1/1410087983.md) | Supprime les doublons parmi les sous-chaînes d'une chaîne de caractères en fonction d'un ou plusieurs séparateurs. |
| [&lt;Type Chaîne Ansi&gt;.VersNumérique](../WDLang1/1410087984.md) | Renvoie la valeur numérique d'une chaîne de caractères. Il est ainsi possible d'effectuer des calculs numériques sur des chaînes de caractères. |
| [&lt;Type Chaîne&gt;.AnsiVersUnicode](../WDLang1/1000024892.md) | Convertit une chaîne de caractères au format ANSI (Windows) en une chaîne de caractères au format UNICODE. |
| [&lt;Type Chaîne&gt;.BufferVersHexa](../WDLang1/1000024894.md) | Convertit une chaîne Ansi ou Unicode en une chaîne hexadécimale affichable (par exemple : "4A 5B 00"). |
| [&lt;Type Chaîne&gt;.CaractOccurrenceType](../WDLang1/1000024895.md) | Renvoie le nombre de caractères vérifiant les informations de type données. |
| [&lt;Type Chaîne&gt;.CommencePar](../WDLang1/1000024901.md) | Vérifie si une chaîne de caractères commence : <br><br>- soit par une chaîne de caractères spécifique. <br><br>- soit par une des chaînes de caractères présentes dans un tableau.<br><br><br> |
| [&lt;Type Chaîne&gt;.Découpe](../WDLang1/1000024902.md) | Extrait toutes les sous-chaînes d'une chaîne de caractères en fonction d'un ou plusieurs séparateurs de chaînes. |
| [&lt;Type Chaîne&gt;.ExtraitChaîne](../WDLang1/1000024876.md) | Permet de :<br><br>- extraire une sous-chaîne d'une chaîne de caractères en fonction d'un séparateur de chaîne donné.<br><br>- effectuer un parcours de sous-chaînes dans une chaîne de caractères en fonction d'un séparateur de chaîne donné.<br><br><br> |
| [&lt;Type Chaîne&gt;.FinitPar](../WDLang1/1000024903.md) | Vérifie si une chaîne de caractères se termine : <br><br>- soit par une chaîne de caractères spécifique.<br><br>- soit par une des chaînes de caractères présentes dans un tableau.<br><br><br> |
| [&lt;Type Chaîne&gt;.Formate](../WDLang1/1000024877.md) | Formate une chaîne de caractères selon les options choisies. |
| [&lt;Type Chaîne&gt;.Incrémente](../WDLang1/1000024905.md) | Incrémente un nombre situé en fin ou début d'une chaîne de caractères. |
| [&lt;Type Chaîne&gt;.Insère](../WDLang1/1000024906.md) | Insère une chaîne dans une autre chaîne de caractères à une position donnée. |
| [&lt;Type Chaîne&gt;.Inverse](../WDLang1/1000024908.md) | Inverse les caractères d'une chaîne. |
| [&lt;Type Chaîne&gt;.Remplace](../WDLang1/1000024890.md) | Remplace toutes les occurrences d'un mot présent dans une chaîne par un autre mot. |
| [&lt;Variable JSON&gt;.VersChaîne](../WDLang1/1410089810.md) | Convertit le contenu d'une variable de type JSON en une chaîne au format JSON. |


## FinValFuture (Fonction)

***En anglais : FinFutureVal***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la valeur future d'un investissement. Les remboursements sont périodiques et constants. Le taux d'intérêt est constant.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple
<a class="notetitle" target="_blank" href="$DOC$=1000003050056&name=finvalfuture_fonction&product=WD">Voir des exemples supplémentaires</a>

```wl
// Valeur future d'un investissement. Le capital initial est 100 000 Euros.
// 3 000 Euros sont versés chaque mois pendant 2 ans. Le taux d'intérêt est de 8%.
ResValFuture = FinValFuture(-100000, -3000, 24, 8)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = FinValFuture([<Capital initial> [, <Dépôt périodique> [, <Nombre de périodes> [, <Taux> [, <Début>]]]]])`
---

**`<Résultat> : (Réel ou monétaire)`**

Valeur future de l'investissement.

**`<Capital initial> : (Réel ou monétaire optionnel)`**

Capital initial (0 par défaut).

**`<Dépôt périodique> : (Réel ou monétaire optionnel)`**

Dépôt périodique (0 par défaut). Cette valeur est fixe pendant toute la durée de l'opération. En principe, ce montant comprend le capital et les intérêts.

**`<Nombre de périodes> : (Entier optionnel)`**

Nombre total de périodes pour l'opération (1 par défaut).

**`<Taux> : (Réel optionnel)`**

Taux d'intérêt de l'investissement exprimé en pourcentage (1 par défaut). Par exemple, pour un taux d'investissement de 5%, ce paramètre sera égal à 5. Ce taux ne peut pas être nul.

**`<Début> : (Booléen optionnel)`**



- <u><u><u><u>Vrai</u></u></u></u> si les paiements doivent être effectués en début de période (valeur par défaut), 

- <u><u><u><u>Faux</u></u></u></u> si les paiements doivent être effectués en fin de période.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Unité temporelle
<a name="unite_temporelle_ELTPARAGRAPHE000086"></a>Les calculs peuvent être réalisés en prenant en compte des mois ou des années. Il suffit que les paramètres &lt;Taux&gt; et &lt;Période&gt; correspondent à la même unité temporelle.

Ainsi, pour des remboursements mensuels d'un emprunt sur durée de cinq ans au taux d'intérêt annuel de 8%, utilisez :

- &lt;Taux&gt; = 8% / 12,

- &lt;Nombre de périodes&gt; = 5\*12




Si, pour le même emprunt, vos remboursements sont annuels, utilisez :

- &lt;Taux&gt; = 8%,

- &lt;Nombre de périodes&gt; = 5




Pour tous les paramètres, les décaissements sont représentés par un nombre négatif alors que les encaissements sont représentés par un nombre positif.


<a name="NOTE0_2"></a>




### Gestion des erreurs
<a name="gestion_des_erreurs_ELTPARAGRAPHE000107"></a>**Attention** : La fonction **FinValFuture** ne retourne pas de code d'erreur. Pour savoir si cette fonction a généré des erreurs, utilisez la fonction [FinErreur](../WDLang1/3050071.md). Le libellé de l'erreur peut être connu grâce à la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.

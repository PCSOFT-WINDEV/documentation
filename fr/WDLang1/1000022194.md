


## Fonctions de gestion des champs Tableau croisé dynamique (syntaxe préfixée)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Ces fonctions (syntaxe préfixée) permettent de manipuler les champs de type Tableau croisé dynamique :



|   |   |
| --- | --- |
| [&lt;Tableau croisé dynamique&gt;.AxeXY](../WDLang1/1000022178.md) | Permet de : <br><br>- Spécifier les entêtes de lignes et de colonnes affichés dans un champ Tableau croisé dynamique. <br><br>- Connaître les entêtes de lignes et de colonnes affichés dans un champ Tableau croisé dynamique.<br><br><br> |
| [&lt;Tableau croisé dynamique&gt;.CalculeMiseAJour](../WDLang1/1000022179.md) | Met à jour une partie des données d'un champ Tableau croisé dynamique à partir d'une date. |
| [&lt;Tableau croisé dynamique&gt;.CalculeTout](../WDLang1/1000022180.md) | Calcule ou recalcule entièrement un champ Tableau croisé dynamique. |
| [&lt;Tableau croisé dynamique&gt;.Charge](../WDLang1/1000022181.md) | Charge le résultat du calcul d'un champ Tableau croisé dynamique depuis une sauvegarde (fichier présent sur le disque ou sauvegarde dans un buffer). |
| [&lt;Tableau croisé dynamique&gt;.Déroule](../WDLang1/1000022182.md) | Déroule toutes les données d'un champ Tableau croisé dynamique selon un entête de ligne ou de colonne. |
| [&lt;Tableau croisé dynamique&gt;.DérouleTout](../WDLang1/1000022183.md) | Déroule tous les entêtes de lignes et/ou de colonnes d'un champ Tableau croisé dynamique. |
| [&lt;Tableau croisé dynamique&gt;.Enroule](../WDLang1/1000022184.md) | Enroule toutes données d'un champ Tableau croisé dynamique selon un entête de ligne ou de colonne. |
| [&lt;Tableau croisé dynamique&gt;.EnrouleTout](../WDLang1/1000022185.md) | Enroule tous les entêtes de lignes et/ou de colonnes d'un champ Tableau croisé dynamique. |
| [&lt;Tableau croisé dynamique&gt;.Filtre](../WDLang1/1000022186.md) | Active ou désactive un filtre sur un champ Tableau croisé dynamique. |
| [&lt;Tableau croisé dynamique&gt;.InfoXY](../WDLang1/1000022187.md) | Renvoie les valeurs d'entêtes correspondant à une position en pixels dans un champ Tableau croisé dynamique. |
| [&lt;Tableau croisé dynamique&gt;.ListePositionEntête](../WDLang1/1000022188.md) | Renvoie toutes les valeurs associées à un entête de ligne ou de colonne d'un champ Tableau croisé dynamique. |
| [&lt;Tableau croisé dynamique&gt;.Sauve](../WDLang1/1000022189.md) | Sauve le résultat du calcul d'un champ Tableau croisé dynamique (dans un fichier ou en mémoire). |
| [&lt;Tableau croisé dynamique&gt;.Select](../WDLang1/1000022190.md) | Permet de connaître la position des cellules sélectionnées et de sélectionner des cellules dans un champ Tableau croisé dynamique. |
| [&lt;Tableau croisé dynamique&gt;.SelectOccurrence](../WDLang1/1000022191.md) | Renvoie le nombre de cellules sélectionnées dans un champ Tableau croisé dynamique. |
| [&lt;Tableau croisé dynamique&gt;.ValeurEntête](../WDLang1/1000022192.md) | Renvoie la valeur affichée dans un entête du champ Tableau croisé dynamique. |
| [&lt;Tableau croisé dynamique&gt;.VersExcel](../WDLang1/1000022193.md) | Crée un fichier Excel avec les données d'un champ Tableau croisé dynamique. |

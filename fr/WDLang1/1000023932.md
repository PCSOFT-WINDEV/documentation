


## TableauAjouteRéférence (Fonction)

***En anglais : ArrayAddReference***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ajoute la référence d'un objet en dernière position dans un tableau à une dimension.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
UnTableau est un tableau de UneClasse
UnObjet est un UneClasse
TableauAjouteRéférence(UnTableau, UnObjet)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = TableauAjouteRéférence(<Tableau WLangage> , <Objet>)`
---

**`<Résultat> : (Entier)`**

Indice du tableau dans lequel l'élément a été ajouté. 

**`<Tableau WLangage> : (Tableau)`**

Nom de la variable de type Tableau à manipuler. Ce tableau doit être un tableau à une dimension.

**`<Objet> : (Objet)`**

Nom de l'objet pour lequel une référence est ajoutée dans le tableau.

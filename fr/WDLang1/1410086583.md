


## Télémétrie : Visualisation des données
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000199"></a>
Pour visualiser les résultats des données de télémétrie pour vos applications, WINDEV et WINDEV Mobile proposent un éditeur de télémétrie. 

**Pour afficher l'éditeur de télémétrie** : 

1. Sous le volet "Projet", dans le groupe "Audit et performances", cliquez sur "Télémétrie". 

2. Indiquez les paramètres de connexion au serveur de télémétrie : 

	- Serveur : adresse du serveur, nom et mot de passe de l'utilisateur. 
			Remarque : Le nom et le mot de passe de l'utilisateur ont été fournis lors de l'installation du serveur de télémétrie. 

	- Plateforme CLOUD : nom de la plateforme, nom et mot de passe de l'utilisateur.




3. Testez la connexion si nécessaire. 

4. Validez.

## Editeur de télémétrie
<a name="editeur_telemetrie_ELTTEXTE000223"></a>


### Interface générale
<a name="interface_generale_ELTPARAGRAPHE000042"></a>L'éditeur de télémétrie permet d'afficher les statistiques sur toutes les applications envoyant leurs données à un même serveur de télémétrie. 

Plusieurs onglets permettent de regrouper les statistiques pour chaque application. La plupart des statistiques sont consultables : 

- pour chaque application (ou chaque plateforme d'une application si le projet contient plusieurs plateformes déployées)

- que le projet soit ouvert sous l'éditeur ou non (un message est affiché si la consultation de certaines données nécessite que le projet soit ouvert). 




Il est possible de consulter les onglets suivants : 

- **Synthèse** : synthèse de l'utilisation de l'application pour une période donnée : nombre d'utilisateurs, éléments les plus utilisés, les moins utilisés, ....

- **Fenêtres** : statistiques d'utilisation sur les fenêtres pour une période donnée : éléments les plus utilisés, les moins utilisés, traitements les plus exécutés, les plus longs, ....

- **Etats/Requêtes** : statistiques d'utilisation sur les états et les requêtes pour une période donnée : éléments les plus utilisés, les moins utilisés, traitements les plus exécutés, les plus longs, ....

- **Utilisateurs** : statistiques concernant le nombre d'utilisateurs de l'application. 

- **Matériel PC** : statistiques concernant le matériel sur lequel l'application est déployée. Cet onglet concerne les applications WINDEV déployées sur PC.  

- **Matériel Mobile** : statistiques concernant le matériel mobile sur lequel l'application est déployée. Cet onglet concerne les applications WINDEV Mobile déployées sur mobile.  

- **Anomalies** : statistiques concernant les anomalies rencontrées lors de l'utilisation de l'application (erreurs WLangage les plus fréquentes, ...). 

- onglets correspondants à la [télémétrie personnalisée](../WDLang1/1410086591.md).




**Remarque** : Les éléments affichés dans ces différents onglets peuvent être réorganisés ou masqués à votre convenance. Il suffit de : 

- Afficher le menu contextuel (clic droit). 

- Sélectionner l'option "Mode Modification". 

- Il est possible de masquer des zones, de les déplacer, ...

- Pour revenir en mode "Normal", sélectionnez à nouveau "Mode Modification" dans le menu contextuel. 





### Options disponibles si le projet correspondant est ouvert sous l'éditeur de WINDEV ou de WINDEV Mobile 
<a name="options_disponibles_projet_correspondant_est_ouvert_sous_editeur_windev_windev_mobile_ELTPARAGRAPHE000077"></a>**Lorsque le projet correspondant à la télémétrie est ouvert sous l'éditeur de WINDEV ou de WINDEV Mobile**, il est possible de visualiser directement la télémétrie d'un élément du projet. Par exemple, le menu contextuel des fenêtres les plus utilisées permet de visualiser la télémétrie de la fenêtre (option "Voir la télémétrie"). 

Lorsque la télémétrie sur un élément est affichée, il est possible de faire un GO de cette fenêtre en mode télémétrie : sous le volet "Télémétrie", dans le groupe "Général", cliquez sur "GO". 

Le "GO Télémétrie" permet de lancer l'élément en affichant directement les statistiques d'utilisation sur la fenêtre grâce à des zones colorées. Il est possible de visualiser : 

- les champs les plus utilisés. Un potentiomètre permet de fixer le seuil du nombre d'utilisations. 

- les champs non utilisés, 

- les erreurs WLangage, 

- le temps moyen d'exécution des traitements.

## Options spécifiques de l'éditeur de télémétrie
<a name="options_specifiques_editeur_telemetrie_ELTTEXTE000253"></a>


### Connexion
<a name="connexion_ELTPARAGRAPHE000103"></a>L'option "Connexion" (présente sous le volet "Télémétrie", dans le groupe "Général") permet de changer la connexion en cours. Il est ainsi possible de changer le serveur de télémétrie visualisé. Il suffit d'indiquer les caractéristiques du nouveau serveur de télémétrie. 


### Paramètres
<a name="parametres_ELTPARAGRAPHE000116"></a>L'option "Paramètres" (présente sous le volet "Télémétrie", dans le groupe "Général") permet d'effectuer des paramétrages globaux pour toutes les applications utilisant le serveur de télémétrie en cours. 

Ce paramétrage concerne : 

- **La conservation des données agrégées** : Les données sont reçues régulièrement sur le serveur de télémétrie et sont agrégées toutes les deux heures pour proposer des statistiques complètes. 
	Des statistiques à la journée, à la semaine, au mois et à l'année sont ainsi effectuées. Il est possible de spécifier la durée de conservation des données : 

	- pour les données à la journée : Ces données sont conservées pendant 1 an par défaut. 

	- pour les données à la semaine et au mois. Ces données sont conservées pendant 3 ans par défaut. 




- **Le serveur HFSQL utilisé pour stocker les données** : Ces informations correspondent au serveur HFSQL utilisé pour stocker la base de données de la télémétrie. Si ce paramètre est modifié, la base de données sera modifiée pour toutes les applications utilisant la télémétrie sur ce serveur. 

- **La modification du mot de passe de la télémétrie**. Le nom d'utilisateur et son mot de passe ont été fournis lors de l'installation du serveur de télémétrie. Il est possible de modifier ces informations. 
	Attention : La modification de ces informations n'a aucune incidence sur les applications déployées. La modification de ces informations entraîne la modification de ces informations dans les applications utilisant ce serveur de télémétrie lors de leur prochaine mise à jour : le nouveau login et le mot de passe seront demandés lors de la création de l'exécutable. 





### Statistiques d'un élément
<a name="statistiques_element_ELTPARAGRAPHE000142"></a>L'option "Statistiques d'un élément" (présente sous le volet "Télémétrie", dans le groupe "Général") permet d'afficher la fenêtre des statistiques pour un élément du projet en cours. Il suffit de sélectionner l'élément voulu.

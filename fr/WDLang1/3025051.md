
## BulleDélai (Fonction)

***En anglais : ToolTipDelay***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de modifier :

- le délai à attendre avant l'affichage des bulles d'aide.

- la durée d'affichage des bulles d'aide des champs. Si l'utilisateur déplace le curseur de la souris pendant ce laps de temps, la bulle d'aide en cours se ferme automatiquement.




Ces modifications sont effectuées pour l'application en cours.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Les bulles d'aide des champs s'afficheront uniquement 
// s'ils sont survolés plus de 1 seconde
BulleDélai(bulleDuréeAvantOuverture, 100)
// Les bulles d'aide des champs s'afficheront pendant 5 secondes
BulleDélai(bulleDuréeAffichage, 500)
```


<a name="Exemple2"></a>





```wl
// Restaurer le délai à attendre par défaut avant 
// l'affichage des bulles d'aide des champs
BulleDélai(bulleDuréeAvantOuverture, -1)
// Restaurer la durée d'affichage par défaut 
// des bulles d'aide des champs
BulleDélai(bulleDuréeAffichage, -1)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`BulleDélai(<Type de délai> , <Nouveau délai>)`
---

**`<Type de délai> : (Constante de type Booléen)`**

Type de délai à modifier :


|   |   |
| --- | --- |
| *bulleDuréeAffichage* | Durée d'affichage des bulles d'aide des champs. |
| *bulleDuréeAvantOuverture* | Délai à attendre avant l'affichage des bulles d'aide. |



**`<Nouveau délai> : (Entier ou Durée)`**



- Nouveau délai (en centièmes de seconde). Ce paramètre peut correspondre à : 

	- un entier correspondant au nombre de centièmes de seconde, 

	- une variable de type Durée, 

	- l'indication directe de la durée (par exemple 1 s ou 10 ms). 




- -1 pour restaurer les valeurs par défaut (gérées par Windows).  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Par défaut, les bulles d'aide des champs sont activées. Pour les désactiver, utilisez la fonction [BulleActive](../WDLang1/3025014.md).

- La durée maximale de l'affichage des bulles d'aide est de 32765 ms soit 32,6s. Astuce pour utiliser une durée supérieure : utiliser un champ Libellé qui sera rendu visible ou invisible. 

- Cette fonction permet de paramétrer aussi bien les bulles d'aide standard que les bulles d'aide personnalisées.

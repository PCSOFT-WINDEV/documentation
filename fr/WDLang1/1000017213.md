


## MultimédiaLargeur (Fonction)

***En anglais : MultimediaWidth***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la largeur en pixels de l'image source du fichier multimédia joué dans le champ Multimédia. Cette fonction est utilisable par exemple pour les fichiers vidéo.




<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = MultimédiaLargeur(<Champ Multimédia>)`
---

**`<Résultat> : (Entier)`**



- Largeur en pixels de l'image du fichier multimédia, 

- 0 en cas d'erreur. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Champ Multimédia> : (Nom de champ)`**

Nom du champ Multimédia à manipuler.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- La fonction [MultimédiaHauteur](../WDLang1/1000017212.md) permet de connaître la hauteur d'un fichier multimédia. 

- La fonction **MultimédiaLargeur** permet de connaître la largeur d'un fichier multimédia.

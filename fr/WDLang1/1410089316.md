


## KanbanInsèreCarte (Fonction)

***En anglais : KanbanInsertCard***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Insère une carte dans une liste de Kanban. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Insère une carte à la première position de la liste de Kanban LISTE_Matière_à_Revoir
// Cette liste est définie sous l'éditeur
maCarte est un kbCarte
maCarte.Titre = "Anglais"
KanbanInsèreCarte(KANBAN_Révision, LISTE_Matière_à_Revoir, 1, maCarte)
```
<a name="Exemple2"></a>

```wl
// Crée dynamiquement une liste de Kanban
LISTE_Liste_a_revoir est un Champ
LISTE_Liste_a_revoir <- KanbanAjouteListe(KANBAN_Révision, "Revues")

// Insère une carte à la première position 
// dans la liste de Kanban créée dynamiquement 
// (Liste de Kanban manipulée par son libellé)
maCarte2 est un kbCarte
maCarte2.Titre = "Espagnol"
KanbanAjouteCarte(KANBAN_Révision, "Revues", maCarte2)
// Insère une carte en première position
maCarte2b est un kbCarte
maCarte2b.Titre = "Japonais"
KanbanInsèreCarte(KANBAN_Révision, "Revues", 1, maCarte2b)

// Insère une carte à la première position 
// dans la liste de Kanban créée dynamiquement 
// (Liste de Kanban manipulée par son nom)
maCarte1 est un kbCarte
maCarte1.Titre = "Géographie"
KanbanInsèreCarte(LISTE_Liste_a_revoir, 1, maCarte1)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Insérer une carte dans une liste de Kanban en précisant le nom du champ Kanban

`<Résultat> = KanbanInsèreCarte(<Champ Kanban> , <Liste de Kanban> , <Position> , <Carte>)`
---

**`<Résultat> : (Entier)`**

Indice de l'élément inséré. 

**`<Champ Kanban> : (Nom de champ)`**

Nom du champ Kanban manipulé. 

**`<Liste de Kanban> : (Nom de champ)`**

Nom du champ Liste de Kanban dans lequel la carte doit être insérée. Ce nom peut correspondre : 

- au nom du champ Liste de Kanban défini sous l'éditeur, 

- au nom du champ Liste de Kanban défini par programmation. 


Si la liste de Kanban n'existe pas, elle est ajoutée au champ Kanban. 

**`<Position> : (Entier)`**

Position à laquelle la carte doit être insérée. 

**`<Carte> : (Variable de type kbCarte)`**

Nom de la variable de type [kbCarte](../WDLang1/1410089167.md) correspondant à la carte à insérée. La carte est insérée à la position spécifiée dans la liste de Kanban.


<a name="SYNTAXE2"></a>

### Insérer une carte dans une liste de Kanban

`<Résultat> = KanbanInsèreCarte(<Liste de Kanban> , <Position> , <Carte>)`
---

**`<Résultat> : (Entier)`**

Indice de l'élément inséré. 

**`<Liste de Kanban> : (Nom de champ)`**

Nom du champ Liste de Kanban dans lequel la carte doit être insérée. Ce nom peut correspondre : 

- au nom du champ Liste de Kanban défini sous l'éditeur, 

- au nom du champ Liste de Kanban défini par programmation. 




**`<Position> : (Entier)`**

Position à laquelle la carte doit être insérée.

**`<Carte> : (Variable de type kbCarte)`**

Nom de la variable de type [kbCarte](../WDLang1/1410089167.md) correspondant à la carte à insérée. La carte est insérée à la position spécifiée dans la liste de Kanban.

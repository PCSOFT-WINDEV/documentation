
## SysNotificationSupprimeTout (Fonction)

***En anglais : SysNotificationClear***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime toutes les notifications interactives précédemment envoyées par l'application en cours.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// -- Code de fermeture de l'application 
// Efface toutes les notifications de l'application avant sa fermeture
SysNotificationSupprimeTout()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`SysNotificationSupprimeTout()`
---



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
La suppression des notifications de l'application fait disparaître toutes ces notifications de l'écran de l'utilisateur et du centre d'action de Windows.


- ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=WDGestionNotificationsWindows.gif) ***Exemples didactiques (WINDEV)*** : **WD Gestion Notifications Windows** <br>A partir de Windows 10, les « toast » notifications (celles qui apparaissent à droite dans le bureau) sont devenues interactives, avec possibilité d’y mettre des boutons, des zones de saisie et des combos. <br>Cet exemple présente la gestion de ces notifications.

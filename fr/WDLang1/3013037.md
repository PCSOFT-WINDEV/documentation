
## Hasard (Fonction)

***En anglais : Random***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie un nombre aléatoire.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Récupération d'un réel compris entre 0 et 1
ResHasard = Hasard()
// Récupération d'un entier compris entre 1 et 33
ResHasard = Hasard(33)
// Récupération d'un entier compris entre 7 et 21
ResHasard = Hasard(7, 21)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Récupérer un nombre aléatoire compris entre 0 et 1

`<Résultat> = Hasard()`
---

**`<Résultat> : (Réel)`**

Nombre aléatoire compris entre 0 et 1.  


<a name="SYNTAXE2"></a>

### Récupérer un nombre aléatoire compris entre 1 et une valeur

`<Résultat> = Hasard(<Borne maximale>)`
---

**`<Résultat> : (Entier)`**

Nombre aléatoire compris entre 1 et une valeur spécifiée incluse.

**`<Borne maximale> : (Entier)`**

Borne maximale du nombre retourné (valeur comprise entre 1 et 2 147 483 647).  


<a name="SYNTAXE3"></a>

### Récupérer un nombre aléatoire compris entre deux valeurs

`<Résultat> = Hasard(<Borne minimale> , <Borne maximale>)`
---

**`<Résultat> : (Entier)`**

Nombre aléatoire compris entre deux valeurs spécifiées incluses.

**`<Borne minimale> : (Entier)`**

Borne minimale du nombre retourné (valeur comprise entre 0 et 2 147 483 647).

**`<Borne maximale> : (Entier)`**

Borne maximale du nombre retourné (valeur comprise entre 0 et 2 147 483 647). 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Noyau aléatoire
<a name="noyau_aleatoire_ELTPARAGRAPHE000090"></a>Le "noyau" dans lequel le nombre aléatoire est "choisi" peut être initialisé par la fonction [InitHasard](../WDLang1/3013030.md).

Si la fonction [InitHasard](../WDLang1/3013030.md) n'est pas appelée, les valeurs aléatoires générées par la fonction **Hasard** seront les mêmes à chaque exécution du programme.

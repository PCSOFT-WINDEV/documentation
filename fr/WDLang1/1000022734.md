


## DocUndoRedoSupprimeTout (Fonction)

***En anglais : DocUndoRedoDeleteAll***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime tout l'historique du mécanisme d'Undo/Redo d'un document de type Traitement de texte.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Supprime l'historique des actions effectuées sur le document
DocUndoRedoSupprimeTout(TT_MonDocument)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = DocUndoRedoSupprimeTout(<Document>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la suppression a bien été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire




**`<Document> : (Nom de champ ou variable de type Document)`**

Document à manipuler. Ce document peut correspondre à : 

- une variable de type [Document](../WDLang1/1000022461.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) un champ Traitement de texte.


## SysCouleur (Fonction)

***En anglais : SysColor***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Identifie ou modifie la couleur d'un élément Windows.

![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Dans cette version, il est possible uniquement d'identifier la couleur d'un élément Windows.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Identifier la couleur de la barre de titre active
ResCouleur = SysCouleur(scBrTitreActif)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Identifier la couleur d'un élément Windows

`<Résultat> = SysCouleur(<Élément Windows>)`
---

**`<Résultat> : (Entier)`**

Couleur de l'élément spécifié sous la forme RVB. Pour récupérer les composantes de la couleur, utilisez les fonctions [RVBBleu](../WDLang1/3029049.md), [RVBRouge](../WDLang1/3029050.md) et [RVBVert](../WDLang1/3029051.md). 

**`<Élément Windows> : (Constante)`**

Élément Windows à manipuler :


|   |   |
| --- | --- |
| *scAscenseur* | Barre de défilement.<br><br>![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Cette constante n'est pas disponible. |
| *scBordActif* | Bordure de la fenêtre active.<br><br>![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Cette constante n'est pas disponible. |
| *scBordBtn* | Bordure d'un bouton. |
| *scBordInactif* | Bordure d'une fenêtre inactive.<br><br>![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Cette constante n'est pas disponible. |
| *scBrTitreActif* | Barre de titre active. |
| *scBrTitreInactif* | Barre de titre inactive. |
| *scBtnSélect* | Bouton sélectionné (correspond aux boutons 3D de Windows 95). |
| *scBtnTexte* | Texte des boutons texte. |
| *scEspaceApp* | Espace de travail des applications MDI.<br><br>![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Cette constante n'est pas disponible. |
| *scFenêtre* | Fond de la fenêtre. |
| *scFond* | Bureau. |
| *scFondObj* | Fond de l'objet sélectionné.<br><br>![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Cette constante n'est pas disponible. |
| *scMenu* | Fond des menus.<br><br>![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Cette constante n'est pas disponible. |
| *scOmbreBtn* | Ombre des boutons.<br><br>![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Cette constante n'est pas disponible. |
| *scTexteFen* | Texte d'une fenêtre. |
| *scTexteGrisé* | Texte grisé (envoie zéro si le driver d'affichage ne comprend pas de niveaux de gris). |
| *scTexteMenu* | Texte des menus.<br><br>![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Cette constante n'est pas disponible. |
| *scTexteSélect* | Texte de l'objet sélectionné. |
| *scTitre* | Texte de la barre de titre. |

### Modifier la couleur d'un élément Windows

`SysCouleur(<Élément Windows> , <Couleur>)`
---

**`<Élément Windows> : (Constante)`**

Élément Windows à manipuler :


|   |   |
| --- | --- |
| *scAscenseur* | Barre de défilement. |
| *scBordActif* | Bordure de la fenêtre active. |
| *scBordBtn* | Bordure d'un bouton. |
| *scBordInactif* | Bordure d'une fenêtre inactive. |
| *scBrTitreActif* | Barre de titre active. |
| *scBrTitreInactif* | Barre de titre inactive. |
| *scBtnSelect* | Bouton sélectionné (correspond aux boutons 3D de Windows 95). |
| *scBtnTexte* | Texte des boutons texte. |
| *scEspaceApp* | Espace de travail des applications MDI. |
| *scFenêtre* | Fond de la fenêtre. |
| *scFond* | Bureau. |
| *scFondObj* | Fond de l'objet sélectionné. |
| *scMenu* | Fond des menus. |
| *scOmbreBtn* | Ombre des boutons. |
| *scTexteFen* | Texte d'une fenêtre. |
| *scTexteGrisé* | Texte grisé (envoie zéro si le driver d'affichage ne comprend pas de niveaux de gris). |
| *scTexteMenu* | Texte des menus. |
| *scTexteSelect* | Texte de l'objet sélectionné. |
| *scTitre* | Texte de la barre de titre. |
| *scTitreInactif* | Barre de titre inactive. |
| *scTourFen* | Bordure de la fenêtre. |



**`<Couleur> : (Entier)`**

Nouvelle couleur de l'élément spécifié. Cette couleur peut correspondre à :

- une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)),

- [une couleur prédéfinie du WLangage](../WDLang5/3010002.md).

<a name="NOTE0_1"></a>

## Remarques




### Couleur
<a name="couleur_ELTPARAGRAPHE000289"></a>La couleur dépend du nombre de couleurs utilisées par le système et de la résolution de l'écran. Ce nombre de couleurs peut être connu grâce à la fonction [SysCouleurRes](../WDLang1/3073012.md).

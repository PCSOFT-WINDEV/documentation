


## &lt;Organigramme&gt;.SelectMoins (Fonction)

***En anglais : &lt;Organization Chart&gt;.SelectMinus***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Dé-sélectionne un élément dans un champ Organigramme.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Récupère l'indice de l'élément sélectionné dans un champ Organigramme
NumElément est un entier 
NumElément = ORGA_MonOrganigramme.InfoXY(SourisPosX(), SourisPosY())
SI NumElément <> -1 ALORS
	ORGA_MonOrganigramme.SelectMoins(NumElément)
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Champ Organigramme>.SelectMoins(<Indice>)`
---

**`<Champ Organigramme> : (Nom de champ)`**

Nom du champ Organigramme à manipuler.

**`<Indice> : (Entier)`**

Indice de l'élément à dé-sélectionner. 

Rappel : L'indice de l'élément sélectionné est renvoyé par la fonction [&lt;Organigramme&gt;.Select](../WDLang1/1000019981.md).
Si l'indice est invalide, une erreur WLangage est affichée.


## Gestion des exécutables (Fonctions WLangage)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000073"></a>
WINDEV, WEBDEV et WINDEV Mobile proposent différentes fonctions pour gérer des exécutables directement depuis une application WINDEV, WINDEV Mobile ou un site WEBDEV. Ces exécutables peuvent aussi bien être créés avec WINDEV qu'avec une autre application.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Fonctions de manipulation des exécutables
<a name="fonctions_manipulation_des_executables_ELTTEXTE000097"></a>
Les fonctions suivantes permettent de manipuler et d'obtenir des renseignements sur les exécutables accessibles ou lancés sur le poste en cours :



|   |   |
| --- | --- |
| [AppliOuvreFiche](../WDLang1/1000020980.md) | Ouvre : <br><br>- la fiche d'une application sur le market Android (Play Store), iOS (App Store) ou Windows (Windows Store) pour que l'utilisateur puisse la noter et/ou la commenter.<br><br>- la fiche de paramétrage des achats In-App d'une application. <br><br><br> |
| [AutoLanceAjoute](../WDLang1/3019002.md) | Permet de lancer automatiquement une application WINDEV lors d'un événement précis. |
| [AutoLanceSupprime](../WDLang1/3019001.md) | Permet d'arrêter le lancement automatiquement d'une application WINDEV. |
| [ExeDonnePID](../WDLang1/1000017239.md) | Renvoie des informations sur le processus courant. |
| [ExeInfo](../WDLang1/3035001.md) | Récupère les informations spécifiées sur la version d'un exécutable ou d'une DLL. |
| [ExeLancé](../WDLang1/3035003.md) | Permet de savoir si une application 32 bits est déjà lancée. |
| [ExeListeDLL](../WDLang1/3035009.md) | Retourne la liste des librairies (fichiers ".DLL") utilisées par une des applications en cours d'exécution. |
| [ExeListeProcessus](../WDLang1/3035008.md) | Retourne la liste des applications en cours d'exécution. |
| [ExePriorité](../WDLang1/3035010.md) | Renvoie ou modifie la priorité d'une application en cours d'exécution. |
| [ExeTermine](../WDLang1/3035004.md) | Termine l'exécution d'une application 32 bits sur le poste en cours : toutes les instances de l'exécutable sont "tuées". |
| [LanceAppli](../WDLang1/3035006.md) | Lance l'exécution d'un programme (exécutable par exemple) depuis l'application en cours. |
| [LanceAppliAssociée](../WDLang1/3035007.md) | Ouvre directement un document dans son application associée. |
| [LanceAppliDistante](../WDLang1/1000018695.md) | Lance depuis l'application en cours l'exécution d'un programme à distance. |
| [LancePartage](../WDLang1/1000020620.md) | Ouvre une popup permettant de sélectionner l'application à utiliser pour partager une ressource (texte ou document). |
| [RelanceProgramme](../WDLang1/1000019900.md) | Termine l'application et la relance. |





D'autres fonctions sont également utilisables : 



|   |   |
| --- | --- |
| [DonneIdentifiant](../WDLang1/3013028.md) | Calcule un identifiant unique de type entier pour un exécutable ou une session donné. |
| [ListeDLL](../WDLang1/3064006.md) | Retourne :<br><br>- **en exécution**: la liste des librairies (fichiers ".DLL") utilisées par l'application WINDEV, WINDEV Mobile ou par le site WEBDEV en cours. Seules les librairies en mémoire sont listées.<br><br>- **en mode test** : toutes les librairies installées sur le poste en cours.<br><br><br> |

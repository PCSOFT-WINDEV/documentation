


## NationVersLangue (Fonction)

***En anglais : NationToLanguage***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la langue correspondant à une nation.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connaître la langue correspondant à la nation italienne
ResLangue est un entier
ResLangue = NationVersLangue(nationItalien) 
SELON ResLangue
	CAS langueAllemand :
	 ...
	CAS langueItalien :
	 ...
FIN
```

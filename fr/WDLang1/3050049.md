
## MatAdditionne (Fonction)

***En anglais : MatAdd***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Additionne deux matrices de mêmes dimensions (même nombre de lignes et même nombre de colonnes).
<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Matadd0.gif)





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple
<a class="notetitle" target="_blank" href="$DOC$=1000003050049&name=matadditionne_fonction&product=WD">Voir des exemples supplémentaires</a>

```wl
// Addition de deux matrices
ResAddition = MatAdditionne("MaMatrice1", "MaMatrice2", "MaMatriceRésultat")
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = MatAdditionne(<1ère matrice> , <2ème matrice> , <Matrice résultat>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'addition a été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour connaître le libellé de l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.




**`<1ère matrice> : (Chaîne de caractères)`**

Nom de la première matrice à additionner, défini avec la fonction [MatCrée](../WDLang1/3050058.md).

**`<2ème matrice> : (Chaîne de caractères)`**

Nom de la seconde matrice à additionner, défini avec la fonction [MatCrée](../WDLang1/3050058.md).

**`<Matrice résultat> : (Chaîne de caractères)`**

Nom de la matrice contenant le résultat de l'addition. Cette matrice est automatiquement créée lors de l'addition des deux matrices.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Existence de la matrice Résultat
<a name="existence_matrice_resultat_ELTPARAGRAPHE000097"></a>L'existence de la matrice n'est pas vérifiée. Si la matrice existait auparavant, cette matrice est écrasée. Pour vérifier l'existence d'une matrice, utilisez la fonction [MatExiste](../WDLang1/3050076.md).


<a name="NOTE0_2"></a>




### Matrices identiques
<a name="matrices_identiques_ELTPARAGRAPHE000109"></a>Si le paramètre &lt;1ère matrice&gt; est identique au paramètre &lt;2ème matrice&gt;, la matrice résultat contient le double de la matrice d'origine.


<a name="NOTE0_3"></a>




### Soustraction de matrices
<a name="soustraction_matrices_ELTPARAGRAPHE000118"></a>Pour soustraire une matrice à une autre matrice, il suffit de :

- multiplier la matrice à soustraire par -1 avec la fonction [MatRéelMultiplie](../WDLang1/3050040.md),

- additionner la matrice obtenue à la première matrice.

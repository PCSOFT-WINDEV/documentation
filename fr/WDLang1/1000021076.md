


## LienGantt (Type de variable)

***En anglais : GanttLink***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **LienGantt** permet de définir toutes les caractéristiques avancées d'une liaison entre deux tâches d'un diagramme de Gantt. Les caractéristiques de cette liaison peuvent être définies et modifiées à l'aide de différentes propriétés WLangage. 

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Récupération de tous les liens qui pointent sur "ID1"
tabLien est un Tableau de LienGantt
tabLien = GanttListeLien(COL_Gantt, "ID1")
Lien est un LienGantt
POUR TOUT Lien de tabLien
	Trace("Lié à " + Lien.IDSource)
FIN
```

<a name="NOTE0"></a>

## Remarques
<a name="NOTE0_1"></a>


### Propriétés spécifiques à la description des variables de type LienGantt
<a name="proprietes_specifiques_description_des_variables_type_liengantt_ELTPARAGRAPHE000028"></a>Les propriétés suivantes peuvent être utilisées pour manipuler un lien entre deux tâches Gantt :

| **Nom de la propriété** | **Type manipulé** | **Effet** |
| --- | --- | --- |
| IDDestination | Chaîne de caractères | Identifiant de la tâche destination. Cet identifiant correspond à la propriété **ID** définie dans la variable de type [TâcheGantt](../WDLang1/1000020881.md). |
| IDSource | Chaîne de caractères | Identifiant de la tâche source. Cet identifiant correspond à la propriété **ID** définie dans la variable de type [TâcheGantt](../WDLang1/1000020881.md). |
| Type | Constante de type Entier | Type du lien. Les types de lien disponibles sont les suivants : <br><br>- *tlgDébutADébut* : Lien de type "Début à Début" : La tâche destination ne peut pas commencer avant que la tâche source ne soit commencée.<br><br>- *tlgDébutAFin* : Lien de type "Début à Fin" : La tâche destination ne peut pas se terminer avant que la tâche source ne soit commencée (cas rare).<br><br>- *tlgFinADébut* (valeur par défaut) : Lien de type "Fin à Début" : La tâche destination ne peut pas commencer avant que la tâche source ne soit terminée. <br><br>- *tlgFinAFin* : Lien de type "Fin à Fin" : La tâche destination ne peut pas se terminer avant que la tâche source ne soit terminée.<br><br><br> |




<a name="NOTE0_2"></a>


### Notes
<a name="notes_ELTPARAGRAPHE000072"></a>

- Un lien entre 2 tâches est représenté par une flèche.

- Ce type de variable : 

	- sert à ajouter un lien via la fonction [GanttAjouteLien](../WDLang1/1000021075.md), 

	- peut être renvoyé par la fonction [GanttListeLien](../WDLang1/1000021079.md). 




- Un lien de type "Fin à Début" est créé par défaut entre les deux tâches : la tâche correspondant à la propriété **IDSource** doit être finie pour que la tâche **IDDestination** puisse commencer. 

- Selon leurs types, les liens sont dessinés sur les bords de début ou de fin des tâches. 



<a name="NOTE0_3"></a>


### Utilisation des variables de type LienGantt
<a name="utilisation_des_variables_type_liengantt_ELTPARAGRAPHE000091"></a>Les variables de type **LienGantt** sont utilisées par les fonctions suivantes : 



|   |   |
| --- | --- |
| [GanttAjouteLien](../WDLang1/1000021075.md) | Crée un lien de type "Fin à Début" entre deux tâches affichées dans une colonne d'un champ Table (ou Table hiérarchique) de type Diagramme de Gantt. |
| [GanttListeLien](../WDLang1/1000021079.md) | Liste les liens présents dans une colonne d'un champ Table (ou Table hiérarchique) de type Diagramme de Gantt. |
| [GanttSupprimeLien](../WDLang1/1000021080.md) | Supprime des liens dans une colonne d'un champ Table (ou Table hiérarchique) de type Diagramme de Gantt. |

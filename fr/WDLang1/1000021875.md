
## TexteEllipse (Fonction)

***En anglais : TextEllipsis***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le texte avec ellipse affiché dans un champ de saisie, un champ Combo ou un champ Libellé d'une fenêtre.

**Attention** : 

- Cette fonction ne permet pas d'afficher le texte dans le champ. 

- Cette fonction est disponible uniquement pour les champs d'une fenêtre. Cette fonction n'est pas utilisable pour les champs d'un état. 



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
sTexteAvecEllipse est une chaîne = TexteEllipse(SAI_Saisie1, "Je vous souhaite la bienvenue")
// sTexteAvecEllipse vaut : "Je vous souhaite ..."
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = TexteEllipse(<Nom du champ> , <Texte> [, <Gestion de l'ellipse>])`
---

**`<Résultat> : (Chaîne de caractères)`**

Texte avec ellipse correspondant au texte affiché dans le champ spécifié (en fonction du style du champ).

**`<Nom du champ> : (Chaîne de caractères)`**

Nom du champ de saisie, du champ Libellé ou du champ Combo utilisé pour calculer l'ellipse du texte. Ce champ est un champ présent dans une fenêtre.  

**`<Texte> : (Chaîne de caractères)`**

Texte dont le texte avec ellipse doit être calculé.

**`<Gestion de l'ellipse> : (Constante optionnelle de type Entier)`**

Mode de gestion de l'ellipse.


|   |   |
| --- | --- |
| *ellipseChemin* | Les caractères "..." remplacent les caractères au milieu du texte. Cette constante est conseillée pour afficher un chemin vers un fichier. |
| *ellipseFin*<br>(Valeur par défaut) | Les caractères "..." remplacent les caractères qui ne peuvent pas être affichés à la fin du champ. |
| *ellipseInactive* | Les caractères "..." ne sont pas utilisés. L'ellipse n'est pas effectuée. |

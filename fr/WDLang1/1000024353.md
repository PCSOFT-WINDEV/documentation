


## &lt;Variable Image&gt;.InverseCouleur (Fonction)

***En anglais : &lt;Image variable&gt;.InvertColor***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Inverse les couleurs dans une variable de type [Image](../WDLang1/1000019650.md).![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=dInverseCouleur.gif)
![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Les dessins](../WDLang1/3029035.md). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonImage est une Image = "Test.bmp"
// Inverse les couleurs de la variable MonImage
Résultat1 est un booléen
Résultat1 = MonImage.InverseCouleur()
SI Résultat1 = Faux ALORS Erreur("L'opération a échoué")
IMG_MonDessin = MonImage
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Variable Image>.InverseCouleur()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'inversion des couleurs a été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Variable Image> : (Variable de type Image)`**

Nom de la variable de type [Image](../WDLang1/1000019650.md) à manipuler. 

Chaque pixel de l'image prend la couleur opposée (négation bit à bit) de la couleur d'origine. L'image manipulée est directement modifiée.

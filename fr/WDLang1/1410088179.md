
## diagBordure (Type de variable)

***En anglais : diagBorder***
			
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **diagBordure** permet de définir toutes les caractéristiques avancées d'une bordure (ou d'un cadre) d'une forme utilisée dans un diagramme. Les caractéristiques de cette bordure peuvent être définies et modifiées à l'aide de différentes propriétés WLangage.

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).

**ATTENTION : Ce type ne doit pas être utilisé directement. Il doit être utilisé uniquement via le type [diagForme](../WDLang1/1410088082.md) (ou un de ses types dérivés).**
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Crée une forme rectangle
MaForme est un diagRectangle

// Définit les dimensions, la position et la couleur de fond
MaForme.X = 100
MaForme.Y = 100
MaForme.Largeur = 350
MaForme.Hauteur = 150
MaForme.Fond.Couleur = RougePastel
MaForme.Bordure.Couleur = Noir
MaForme.Bordure.Epaisseur = 2
MaForme.Bordure.Type = TraitTiret

// Obtient une référence sur le diagramme affiché
MonDiagramme est un Diagramme <- EDIAG_MonDiagramme

// Ajoute la forme rectangle dans le diagramme
Ajoute(MonDiagramme.Forme, MaForme)
```

Dans ce cas, l'opacité spécifiée dans la variable est prise en compte.<br><br><br> |
| Epaisseur | Réel | Epaisseur du trait de la bordure. |
| Type | Entier | Type du trait de la bordure : <br><br>- *TraitContinu* : ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=traitcontinu.gif)<br><br><br>- *TraitPointillé* : ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=TraitPointille.gif)<br><br><br>- *TraitTiret* : ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=TraitTiret.gif)<br><br><br><br> |

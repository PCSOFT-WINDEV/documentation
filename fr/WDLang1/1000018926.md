
## ComposantChargeTout (Fonction)

***En anglais : ComponentLoadAll***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Charge en mémoire de manière récursive tous les composants externes utilisés dans le projet.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
ComposantChargeTout(cctRécursif)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = ComposantChargeTout([<Options>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si tous les composants externes ont été chargés sans erreur, 

- <u><u><u><u>Faux</u></u></u></u> si une erreur s'est produite lors du chargement des composants externes. Les informations détaillées de l'erreur peuvent être récupérées grâce à la fonction [ErreurInfo](../WDLang1/3013008.md). 




**`<Options> : (Constante optionnelle de type Entier)`**

Options de chargement des composants externes :


|   |   |
| --- | --- |
| *cctParent* | Tous les composants externes du projet sont chargés depuis un composant interne. |
| *cctProjet* | Tous les composants externes utilisés par le projet sont chargés. Les composants externes utilisés par les composants eux-mêmes ne sont pas chargés. |
| *cctRécursif*<br>(Valeur par défaut) | Tous les composants externes utilisés par le projet ainsi que tous les composants externes utilisés par les composants eux-mêmes sont chargés. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Exécution des traitements
<a name="execution_des_traitements_ELTPARAGRAPHE000059"></a>Les traitements d'initialisation (initialisation du projet, déclaration des classes, déclarations des collections, ...) des composants externes chargés sont exécutés.
<a name="NOTE0_2"></a>


### Erreurs au chargement des composants externes
<a name="erreurs_chargement_des_composants_externes_ELTPARAGRAPHE000066"></a>Les erreurs fatales apparaissant pendant le chargement des composants externes sont transformées en erreurs non fatales et la fonction renvoie <u><u><u><u>Faux</u></u></u></u>.


## dHalo (Fonction)

***En anglais : dHalo***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ajoute un effet de halo sur une image présente dans une variable de type [Image](../WDLang1/1000019650.md). 

![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Les dessins](../WDLang1/3029035.md). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonImage est une Image
MonImage = "Image.png"

// Ajoute un halo jaune de 5 pixels de large
dHalo(MonImage, JauneClair, 5)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`dHalo(<Image> , <Couleur du halo> [, <Epaisseur> [, <Détourer>]])`
---

**`<Image> : (Variable de type Image)`**

Nom de la variable de type [Image](../WDLang1/1000019650.md) correspondant à l'image à manipuler. 

**`<Couleur du halo> : (Entier)`**

Couleur du halo à ajouter à l'image. Cette couleur peut correspondre à :

- une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)),

- une couleur TSL (obtenue avec la fonction [TSL](../WDLang1/3029057.md)),

- [une couleur prédéfinie du WLangage](../WDLang5/3010002.md).




**`<Epaisseur> : (Entier optionnel)`**

Epaisseur du halo en pixels (2 pixels par défaut). 

**`<Détourer> : (Booléen optionnel)`**



- <u><u><u><u>Vrai</u></u></u></u> si le halo doit être détouré, 

- <u><u><u><u>Faux</u></u></u></u> (valeur par défaut) dans le cas contraire.


## FichierAide (Fonction)

***En anglais : HelpFile***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie ou modifie le nom du fichier d'aide utilisé par l'aide contextuelle d'une fenêtre.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Utiliser le fichier d'aide "D:\WINDEV\Aide\Aide.chm"
// pour l'aide contextuelle de la fenêtre "FEN_Fenêtre1"
FichierAide(FEN_Fenêtre1, "D:\WINDEV\Aide\Aide.chm")
```


<a name="Exemple2"></a>



```wl
// Utiliser le fichier d'aide "Aide.hlp"
// pour l'aide contextuelle de la fenêtre en cours
FichierAide("", "Aide.hlp")
```


<a name="Exemple3"></a>



```wl
// Connaître le fichier d'aide utilisé
// pour l'aide contextuelle de la fenêtre en cours
ResFicAide = FichierAide("")
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le fichier d'aide utilisé pour l'aide contextuelle

`<Résultat> = FichierAide(<Nom de la fenêtre>)`
---

**`<Résultat> : (Chaîne de caractères)`**

Nom et chemin du fichier d'aide (.HLP ou .CHM) associé à la fenêtre pour l'aide contextuelle.

**`<Nom de la fenêtre> : (Chaîne de caractères)`**

Nom de la fenêtre dont on recherche le fichier d'aide associé. Si ce paramètre correspond à une chaîne vide (""), la fenêtre considérée est la fenêtre qui possède le traitement en cours.


<a name="SYNTAXE2"></a>

### Modifier le fichier d'aide utilisé pour l'aide contextuelle

`<Résultat> = FichierAide(<Nom de la fenêtre> [, <Nom du fichier d'aide>])`
---

**`<Résultat> : (Chaîne de caractères)`**

Nom et chemin du fichier d'aide (.HLP ou .CHM) associé à la fenêtre pour l'aide contextuelle **avant la modification**.

**`<Nom de la fenêtre> : (Chaîne de caractères)`**

Nom de la fenêtre manipulée. Si ce paramètre correspond à une chaîne vide (""), la fenêtre considérée est la fenêtre qui possède le traitement en cours.

**`<Nom du fichier d'aide> : (Chaîne de caractères optionnelle)`**

Nom et chemin du fichier d'aide (.HLP ou .CHM) utilisé pour l'aide contextuelle. Ce chemin peut être un chemin relatif ou un chemin de type UNC.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Aide contextuelle
<a name="aide_contextuelle_ELTPARAGRAPHE000053"></a>L'aide contextuelle est affichée lorsque l'utilisateur utilise l'icône "Qu'est-ce que c'est" de la fenêtre (icône ?). ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=aidecontext.gif)

Cette aide contextuelle est basée :

- Sur le style de la fenêtre (option "Icone d'aide contextuelle" cochée).

- Sur le fichier d'aide contextuelle défini : 

	- soit sous l'éditeur de fenêtres (onglet "Aide" de la fenêtre de description de la fenêtre). <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=fichierAide%20-%20HC%20N%B0001.gif&type=thumb)


	- soit par la fonction **FichierAide**. 

- Sur le numéro d'aide défini dans la description du champ.<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=fichierAide%20-%20HC%20N%B0002.gif&type=thumb)

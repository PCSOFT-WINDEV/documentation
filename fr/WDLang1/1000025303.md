
## fInsèreTexteAuDébut (Fonction)

***En anglais : fInsertTextAtBeginning***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Insère un texte au début d'un fichier sans écraser le texte déjà présent dans le fichier.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Chemin du fichier
soit sCheminDuFichier = "C:\MesRépertoires\Fichier.txt"
// Chaîne à insérer
sChaîneAEcrire est une chaîne
sChaîneAEcrire = "Voici le texte à insérer en début de fichier"
// Insertion au début
soit bInsertion_réussie = fInsèreTexteAuDébut(sCheminDuFichier, sChaîneAEcrire)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = fInsèreTexteAuDébut(<Chemin du fichier> , <Texte>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'insertion a été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 




**`<Chemin du fichier> : (Chaîne de caractères)`**

Nom et chemin complet (ou relatif) du fichier (260 caractères maximum). Un chemin de type UNC peut être utilisé.

Si le fichier n'existe pas, il est automatiquement créé. 



**`<Texte> : (Chaîne de caractères)`**

Texte à insérer au début du fichier.


## AnnéeBissextile (Fonction)

***En anglais : LeapYear***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Indique si l'année spécifiée est bissextile ou non.

**Rappel** : une année bissextile est une année comportant 366 jours au lieu de 365 jours pour une année non bissextile. Le jour supplémentaire est le 29 février.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Indique si on est dans une année bissextile
SI AnnéeBissextile() ALORS
	Info("Nous somme dans une année bissextile.")
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = AnnéeBissextile([<Date>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'année est bissextile,

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Date> : (Chaîne de caractères ou entier optionnels)`**

Date dont on veut savoir si elle est bissextile. Ce paramètre peut correspondre à : 

- une date ou une année sous forme de chaîne de caractères, 

- une date, une DateHeure ou année sous forme d'entier.


Si ce paramètre n'est pas précisé, l'année manipulée est l'année en cours.  

Remarque : L'année doit être comprise entre 0 et 9999.




## USBRecherche (Fonction)

***En anglais : USBFind***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Recherche un périphérique USB d'après certains mots-clés.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
sPériphériques est une chaîne 
sPériphériques = USBRecherche(USBPropriétéFabricant, "PC Soft")
SI sPériphériques <> "" ALORS
	Info("Identifiants des périphériques PC Soft :", sPériphériques)
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = USBRecherche(<Propriété> , <Mot 1> [, <Mot 2> [... [, <Mot N>]]])`
---

**`<Résultat> : (Chaîne de caractères)`**

Chaîne contenant le ou les identifiants des périphériques USB trouvés. Si plusieurs périphériques USB sont trouvés, les identifiants sont séparés par des RC (Retour Chariot). Ces identifiants peuvent être utilisés dans les [fonctions USB](../WDLang1/3090005.md).

**`<Propriété> : (Constante de type Entier)`**

Option de détail du résultat :


|   |   |
| --- | --- |
| *USBLecteur* | Identifiant USB correspondant à un nom de lecteur (exemple, f:) |
| *USBPropriétéClasse* | Classe de périphérique |
| *USBPropriétéDescription* | Description du périphérique |
| *USBPropriétéEmplacement* | Emplacement physique du périphérique |
| *USBPropriétéFabricant* | Nom du fabricant |
| *USBPropriétéNomExplicite* | Nom explicite du périphérique |



**`<Mot 1> : (Chaîne de caractères)`**

Chaîne à rechercher dans le propriété spécifiée.

**`<Mot 2> : (Chaîne de caractères optionnelle)`**

Chaîne à rechercher dans le propriété spécifiée.



**`<Mot N> : (Chaîne de caractères optionnelle)`**

Chaîne à rechercher dans le propriété spécifiée.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Condition de recherche
<a name="condition_recherche_ELTPARAGRAPHE000066"></a>Pour qu'un périphérique corresponde au critère de recherche, il faut que la &lt;Propriété&gt; contienne tous les &lt;Mot 1&gt;, &lt;Mot 2&gt;, &lt;Mot N&gt; indiqués. La recherche n'est pas sensible à la casse.

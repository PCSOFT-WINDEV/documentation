


## Fonctions de gestion du champ Editeur HTML
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions WLangage permettant de gérer les champs Editeur HTML sont les suivantes : 



|   |   |
| --- | --- |
| [EditeurHTMLDepuisEmail](../WDLang1/1000026172.md) | Importe le contenu HTML d'un email et ses images dans un champ Editeur HTML. Les images contenues dans l'email sont également affichées dans le champ. |
| [EditeurHTMLExécuteJS](../WDLang1/1000026260.md) | Exécute du code Javascript dans un champ Editeur HTML. |
| [EditeurHTMLFormateSélection](../WDLang1/1000025996.md) | Modifie le style de la sélection dans un champ Editeur HTML. |
| [EditeurHTMLInsère](../WDLang1/1000025994.md) | Insère du texte ou du code HTML à la position du curseur dans un champ Editeur HTML. |
| [EditeurHTMLInsèreTableau](../WDLang1/1000025917.md) | Insère un tableau à la position du curseur dans un champ Editeur HTML. |
| [EditeurHTMLRecherche](../WDLang1/1000025918.md) | Recherche une chaîne de caractères dans un champ Editeur HTML. |
| [EditeurHTMLRécupère](../WDLang1/1000025981.md) | Récupère en mode asynchrone la valeur d'une des propriétés d'un champ Editeur HTML. |
| [EditeurHTMLRécupèreDocument](../WDLang1/1000025993.md) | Récupère en mode asynchrone le document HTML en cours dans un champ Editeur HTML. |
| [EditeurHTMLRécupèreSélection](../WDLang1/1000025992.md) | Récupère en mode asynchrone la sélection effectuée dans un champ Editeur HTML. |
| [EditeurHTMLRemplace](../WDLang1/1000025995.md) | Remplace une chaîne de caractères par une autre dans un champ Editeur HTML. |
| [EditeurHTMLSélectionne](../WDLang1/1000025926.md) | Sélectionne une portion de texte dans un champ Editeur HTML. |
| [EditeurHTMLVersEmail](../WDLang1/1000026171.md) | Exporte le contenu d'un champ Editeur HTML vers une variable de type Email (pour envoyer l'email). |
| [EditeurHTMLVersPDF](../WDLang1/1000025927.md) | Exporte le contenu d'un champ Editeur HTML vers un fichier PDF. |

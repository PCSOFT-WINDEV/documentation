
## VersionDémo (Fonction)

***En anglais : TrialVersion***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de savoir si l'exécution courante de l'application est en mode "version d'évaluation". 

**Remarque** : Microsoft Store propose un mode d'exécution correspondant à une version d'évaluation des applications. Il est ainsi possible de mettre en place une version d'évaluation gratuite d'une application en limitant l'accès à certaines fonctionnalités grâce à la fonction **VersionDémo**. 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI VersionDémo() = Vrai ALORS
	Info("Cette fonctionnalité n'est pas disponible dans la version d'évaluation")
	RETOUR
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = VersionDémo()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'application est exécutée en mode "version d'évaluation", 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.

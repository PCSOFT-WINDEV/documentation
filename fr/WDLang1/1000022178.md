


## &lt;Tableau croisé dynamique&gt;.AxeXY (Fonction)

***En anglais : &lt;Pivot table&gt;.AxisXY***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de : 

- Spécifier les entêtes de lignes et de colonnes affichés dans un champ Tableau croisé dynamique. 

- Connaître les entêtes de lignes et de colonnes affichés dans un champ Tableau croisé dynamique. 



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Spécifie les entêtes de lignes et de colonnes affichés
SI INT_InverseAxe = Vrai ALORS
	TCD_Stats.AxeXY("COL_Date" + RC + "COL_Produit", "COL_Vendeur")
SINON
	TCD_Stats.AxeXY("COL_Vendeur", "COL_Date" + RC + "COL_Produit")
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Spécifier les entêtes de lignes et de colonnes affichés

`<Champ Tableau croisé dynamique>.AxeXY(<Liste entêtes de colonnes> , <Liste entêtes de lignes>)`
---

**`<Champ Tableau croisé dynamique> : (Nom de champ)`**

Nom du champ Tableau croisé dynamique à manipuler.

**`<Liste entêtes de colonnes> : (Chaîne de caractères)`**

Liste des entêtes de colonnes (affichés sur l'axe X). Cette liste contient les différents entêtes de colonnes séparés par des RC (Retours Chariot). 
Cette liste peut utiliser une des syntaxes suivantes : 

- 
	```txt
	<Nom entête 1>..Nom + RC + ... + RC + <Nom entête N>..Nom
	```
Exemple : 
	
	```txt
	COL_Date..Nom + RC + COL_Produit..Nom
	```


- 
	```txt
	"<Nom entête 1>" + RC + ... + RC + "<Nom entête N>"
	```
Exemple : 
	
	```txt
	"COL_Date" + RC + "COL_Produit"
	```





**`<Liste entêtes de lignes> : (Chaîne de caractères)`**

Liste des entêtes de lignes (affichés sur l'axe Y). Cette liste contient les différents entêtes de lignes séparés par des RC (Retours Chariot). Cette liste peut utiliser une des syntaxes suivantes : 

- 
	```txt
	<Nom entête 1>..Nom + RC + ... + RC + <Nom entête N>..Nom
	```
Exemple : 
	```txt
	COL_Date..Nom + RC + COL_Produit..Nom
	```


- 
	```txt
	"<Nom entête 1>" + RC + ... + RC + "<Nom entête N>"
	```
Exemple : 
	```txt
	"COL_Date" + RC + "COL_Produit"
	```






<a name="SYNTAXE2"></a>

### Connaître les entêtes de lignes et de colonnes affichés

`<Résultat> = <Champ Tableau croisé dynamique>.AxeXY()`
---

**`<Résultat> : (Chaîne de caractères)`**

Listes des entêtes de lignes et de colonnes affichés dans le champ Tableau croisé dynamique. Cette liste est de la forme : 

```txt
<Nom Entête colonne 1> + RC + ...+ RC + <Nom Entête colonne N> + EOT + 
<Nom Entête ligne 1> + RC + ...+ RC + <Nom Entête ligne N> + EOT + 
<Nom Filtre 1> + RC + ... RC + <Nom Filtre N>
```
Par exemple : 

```txt
COL_DateCommande_Année + RC + COL__DateCommande_Trimestre + RC + 
COL__DateCommande_Mois + EOT + COL_Pays + RC + COL_Ville + EOT + DTCD_Libellé
```


**`<Champ Tableau croisé dynamique> : (Nom de champ)`**

Nom du champ Tableau croisé dynamique à manipuler.

<a name="NOTE0_1"></a>

## Remarques


### Spécifier les entêtes
<a name="specifier_les_entetes_ELTPARAGRAPHE000215"></a>

- Le champ Tableau croisé dynamique n'a pas besoin d'être recalculé et l'affichage est immédiat.

- Les entêtes spécifiés doivent correspondre aux entêtes existants. Il est par exemple possible : 

	- d'inverser les entêtes de ligne et de colonne. 

	- de spécifier moins d'entêtes que ceux définis dans le champ.


## France.VérifieSIREN (Fonction)

***En anglais : France.CheckSIREN***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Vérifie la validité d'un numéro SIREN (système d'identification du répertoire des entreprises).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Test d'une variable
MonNuméroSIREN est une chaîne = "330318270"
France.VérifieSIREN(MonNuméroSIREN) // Renvoie Vrai

// Test d'un champ
SAI_NuméroSIREN = "ABC"
France.VérifieSIREN(SAI_NuméroSIREN) // Renvoie Faux
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = France.VérifieSIREN(<Numéro à vérifier>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'élément spécifié contient un numéro SIREN valide.

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Numéro à vérifier> : (Nom de champ ou variable de type chaîne)`**

Numéro SIREN à vérifier. Ce paramètre peut correspondre : 

- au nom d'une variable de type chaîne.

- au nom d'un champ.

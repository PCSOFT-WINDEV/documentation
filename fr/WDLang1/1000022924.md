


## docTableau (Type de variable)

***En anglais : docTable***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **docTableau** permet de définir toutes les caractéristiques avancées d'un paragraphe de type Tableau présent dans un document de type Traitement de texte. Les caractéristiques de ce tableau peuvent être définies et modifiées à l'aide de différentes propriétés WLangage. 

**ATTENTION : Ce type ne doit pas être utilisé directement.** Il doit être utilisé uniquement via les variables de type [docParagraphe](../WDLang1/1000022483.md) correspondant à un tableau. 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Récupération du document
MonDocument est un Document <- TT_Tableau
// Récupération du tableau, ici le paragraphe 1
pTableau est un docParagraphe <- MonDocument.Paragraphe[1]

// Parcours les lignes
POUR i = 1 _A_ pTableau.Tableau.Lignes.Occurrence
	// Parcours les colonnes
	POUR j = 1 _A_ pTableau.Tableau.Colonnes.Occurrence
		Trace("Valeur de la cellule [[%i%]][[%j%]] = " + 
		pTableau.Tableau.Cellules[i,j].Contenu.Texte)
	FIN
FIN
```

## Remarques
<a name="NOTE0_1"></a>


### Propriétés spécifiques à la description des variables de type docTableau
<a name="proprietes_specifiques_description_des_variables_type_doctableau_ELTPARAGRAPHE000033"></a>Les propriétés suivantes peuvent être utilisées pour manipuler une variable de type **docTableau** :

   | --- |
| --- | --- | --- |
| Nom de la propriété | Type manipulé | Effet |
| Cellules[Ligne,Colonne] | Tableau des cellules du tableau ([docCellule](../WDLang1/1000022927.md)) | Accès aux cellules d'un paragraphe de type tableau. Les cellules du tableau sont de type [docCellule](../WDLang1/1000022927.md). |
| Colonnes |   | Variable de type [docColonne](../WDLang1/1000023160.md) | Permet de manipuler les colonnes du tableau. <br><br>Par exemple, pour insérer une colonne à la 3ème colonne d'un tableau : <br><br><pre><code>Insère(pTableau.Tableau.Colonnes, 3)</code></pre><br> |
|   | Colonnes.LargeurMillimètres | Réel | Largeur de la ligne (en millimètres). |
|   | Colonnes.ModeLargeur | Constante de type Entier | Mode de calcul de la largeur de la colonne : <br><br>- *docModeLargeurAucune* : Cas particulier. <br><br>- *docModeLargeurAutomatique* : La largeur de la colonne est calculée automatiquement en fonction du texte présent dans la colonne en cours et de celui présent dans les autres colonnes. <br><br>- *docModeLargeurNonDéfini* : Le mode de calcul de la largeur de la colonne n'est pas défini car des cellules de la colonne ont des largeurs différentes. <br><br>- *docModeLargeurPourcent* : Le mode de calcul de la largeur de la colonne est effectué en pourcentage de la largeur totale du tableau. <br><br>- *docModeLargeurValeur* : Le mode de calcul de la largeur des colonnes est effectué selon une largeur fournie en millimètres. Si la propriété Colonnes.LargeurMillimètres est renseignée, ce mode est automatiquement sélectionné. <br><br><br><br><br>Cette propriété est disponible en lecture seulement. |
| Lignes |   | Variable de type [docLigne](../WDLang1/1000022925.md) | Permet de manipuler les lignes du tableau. <br><br>Par exemple, pour insérer une ligne à la 3ème ligne d'un tableau : <br><br><pre><code>Insère(pTableau.Tableau.Lignes, 3)</code></pre><br> |
|   | Lignes.Hauteur | Entier | Hauteur de la ligne (en millimètres). |
|   | Lignes.ModeHauteur | Constante de type Entier | Mode de calcul de la hauteur de la ligne : <br><br>- *docModeHauteurAuMoins* : La hauteur de la ligne sera au moins celle définie par la propriété **Hauteur**. <br><br>- *docModeHauteurAutomatique* : La hauteur de la ligne est calculée automatiquement en fonction du texte présent dans les cellules de la ligne. <br><br>- *docModeHauteurExact* : La hauteur de la ligne sera celle définie par la propriété **Hauteur**. <br><br>- *docModeHauteurNonDéfini* : Le mode de calcul de la hauteur de la ligne n'est pas défini (par défaut, le mode de calcul automatique est effectué). <br><br><br> |
|   | Lignes.NombreCellules | Entier | Nombre de cellules de la ligne. <br><br>Cette propriété est disponible en lecture seulement. |




## Constantes EXE
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les différentes constantes de gestion des exécutables sont les suivantes : 


| Constante | Description | Utilisée par les fonctions |
| --- | --- | --- |
| **alDémarrage** | Lancement de l'application lors du démarrage du poste PC en cours | [AutoLanceAjoute](../WDLang1/3019002.md), [AutoLanceSupprime](../WDLang1/3019001.md) |
| **exeActif** | Mode de lancement du programme : programme lancé actif | [LanceAppli](../WDLang1/3035006.md) |
| **exeBloquant** | Délai de reprise du programme en cours : le programme en cours reprend son exécution lorsque le programme lancé est terminé. | [LanceAppli](../WDLang1/3035006.md) |
| **exeCopyright** | Information à récupérer : copyright du programme | [ExeInfo](../WDLang1/3035001.md) |
| **exeDeepLink** |   | [ExeInfo](../WDLang1/3035001.md) |
| **exeDémarrage** |   | [ExeInfo](../WDLang1/3035001.md) |
| **exeDescription** | Information à récupérer : description du programme | [ExeInfo](../WDLang1/3035001.md) |
| **exeEnArrièrePlan** |   | [ExeInfo](../WDLang1/3035001.md) |
| **exeGéoSuivi** | <br><br>Remarque : Cette constante est conservée par compatibilité à partir de la version 230050. Elle est identique à la constante *exeEnArrièrePlan*. | [ExeInfo](../WDLang1/3035001.md) |
| **exeIconise** | Mode de lancement du programme : programme lancé inactif et réduit en icône | [LanceAppli](../WDLang1/3035006.md) |
| **exeInactif** | Mode de lancement du programme : programme lancé inactif | [LanceAppli](../WDLang1/3035006.md) |
| **exeLancement** |   | [ExeInfo](../WDLang1/3035001.md) |
| **exeLigneCommande** | Ligne de commande de l'exécutable. | [ExeListeProcessus](../WDLang1/3035008.md) |
| **exeMaximise** | Mode de lancement du programme : programme lancé actif, "prenant la main" sur le programme en cours (la fenêtre d'exécution a la taille maximale autorisée) | [LanceAppli](../WDLang1/3035006.md) |
| **exeMémoire** | Consommation actuelle de l'application | [ExeListeProcessus](../WDLang1/3035008.md) |
| **exeMémoireMax** | Consommation maximale atteinte par l'application | [ExeListeProcessus](../WDLang1/3035008.md) |
| **exeMode** | Information à récupérer : mode de compilation de l'exécutable (16, 32 ou 64 bits ou DOS) | [ExeInfo](../WDLang1/3035001.md) |
| **exeNom** | Information à récupérer : nom complet de l'exécutable | [ExeInfo](../WDLang1/3035001.md) |
| **exeNomCourt** | Nom et extension de l'exécutable de l'application | [ExeListeProcessus](../WDLang1/3035008.md) |
| **exeNomLong** | Nom et chemin complet de l'exécutable de l'application | [ExeListeProcessus](../WDLang1/3035008.md) |
| **exeNonBloquant** | Délai de reprise du programme en cours : le programme en cours et le programme lancé s'exécutent en parallèle | [LanceAppli](../WDLang1/3035006.md) |
| **exeNotificationPush** | Mode de démarrage de l'application : l'application est lancée automatiquement par le système suite à la réception d'une notification push. | [ExeInfo](../WDLang1/3035001.md) |
| **exePID** | Identifiant du processus de l'application.<br>Cet identificateur peut être utilisé par exemple dans certaines API. | [ExeDonnePID](../WDLang1/1000017239.md), [LanceAppli](../WDLang1/3035006.md), [ExeListeProcessus](../WDLang1/3035008.md) |
| **exePIDParent** | PID du parent (c'est-à-dire le processus qui a lancé le processus en cours). | [ExeDonnePID](../WDLang1/1000017239.md), [ExeListeProcessus](../WDLang1/3035008.md) |
| **exePrioritéBasse** | Priorité de l'application : Basse | [ExePriorité](../WDLang1/3035010.md) |
| **exePrioritéHaute** | Priorité de l'application : Priorité haute | [ExePriorité](../WDLang1/3035010.md) |
| **exePrioritéInférieureNormale** | Priorité de l'application : Priorité supérieure à *exePrioritéBasse* et inférieure à *exePrioritéNormale*. | [ExePriorité](../WDLang1/3035010.md) |
| **exePrioritéNormale** | Priorité de l'application : Priorité supérieure à *exePrioritéInférieureNormale*et inférieure à *exePrioritéSupérieureNormale*. | [ExePriorité](../WDLang1/3035010.md) |
| **exePrioritéSupérieureNormale** | Priorité de l'application : Priorité supérieure à *exePrioritéNormale*et inférieure à *exePrioritéTempsRéel*. | [ExePriorité](../WDLang1/3035010.md) |
| **exePrioritéTempsRéel** | Priorité de l'application : Priorité temps réel. | [ExePriorité](../WDLang1/3035010.md) |
| **exeRépertoire** | Information à récupérer : répertoire de l'exécutable (sans le \\ final, mais avec les ":"). | [ExeInfo](../WDLang1/3035001.md) |
| **exeSansHéritageHandle** | Permet d'indiquer que l'exécutable ne doit pas hériter des "Handles" du parent (attention, s'applique à TOUS les handles, fichiers compris). L'exécutable est indépendant de l'application qui le lance.  | [DDELance](../WDLang1/3028013.md)<br>[LanceAppli](../WDLang1/3035006.md) |
| **exeSociété** | Information à récupérer : nom saisi pour la société lors de la création de l'exécutable. | [ExeInfo](../WDLang1/3035001.md) |
| **exeTéléchargementFin** |   | [ExeInfo](../WDLang1/3035001.md) |
| **exeTID** | Identificateur du thread courant. | [ExeDonnePID](../WDLang1/1000017239.md) |
| **exeValeurRetour** | Le programme en cours reprend son exécution lorsque le programme lancé est terminé. &lt;Résultat&gt; correspond à la valeur de retour de l'exécutable. | [LanceAppli](../WDLang1/3035006.md) |
| **exeVersion** | Information à récupérer : version du programme | [ExeInfo](../WDLang1/3035001.md) |
| **exeVersionFormatWindows** | Information à récupérer : version du programme au format Windows standard. | [ExeInfo](../WDLang1/3035001.md) |
| **ficheAchatInApp** | Type de fiche à ouvrir : Ouvre la fenêtre de paramétrage des achats In-App de l'application. | [AppliOuvreFiche](../WDLang1/1000020980.md) |
| **ficheApplication** | Type de fiche à ouvrir : Ouvre la fiche de l'application. | [AppliOuvreFiche](../WDLang1/1000020980.md) |
| **partageDocument** | Partage d'un ou de plusieurs documents (image, vidéo, pdf, ...). Ces documents sont identifiés par leur chemin complet, précisé en paramètre. | [LancePartage](../WDLang1/1000020620.md) |
| **partageTexte** | Partage de texte. | [LancePartage](../WDLang1/1000020620.md) |

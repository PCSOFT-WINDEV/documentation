
## RéelVersDateHeure (Fonction)

***En anglais : RealToDateTime***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Convertit un réel sur 8 octets en une date et une heure. Ce réel correspond :

- soit au numéro du jour depuis le 1er janvier 1900 (compatible Excel, commence à 1)

- soit au numéro du jour depuis le 1er janvier 1904 (compatible Excel avec l'option "Utiliser le calendrier depuis 1904", commence à 0).




Remarque : L'option "Utiliser le calendrier depuis 1904" est configurable dans les options avancées de MS Excel. Sous Open Office, cette option est disponible dans les options de Open Office Calc (rubrique "Calculer", option "Date"). 






<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affiche "19000103060000000"
Info(RéelVersDateHeure(3.25))
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = RéelVersDateHeure(<Valeur> [, <Depuis 1904>])`
---

**`<Résultat> : (Variable de type DateHeure ou chaîne de caractères)`**

Valeur de la date et de l'heure. Peut correspondre à : 

- une variable de type DateHeure. 

- une chaîne de caractères au format AAAAMMJJHHmmSSCCC (AAAA correspond à l'année, MM au mois, JJ au jour, HH à l'heure, mm aux minutes, SS aux secondes et CCC aux millisecondes). 

- une chaîne vide ("") en cas d'erreur. 




**`<Valeur> : (Réel)`**

Valeur à convertir. 

**`<Depuis 1904> : (Booléen optionnel)`**

Mode de calcul : 

- <u><u><u><u>Vrai</u></u></u></u> si la date est calculée à partir de 1904. 

- <u><u><u><u>Faux</u></u></u></u> (valeur par défaut) si la date est calculée à partir de 1900. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Mode de calcul
<a name="mode_calcul_ELTPARAGRAPHE000094"></a>La partie entière du réel donne le numéro de jour de la partie date du résultat : 

- la numérotation commence à 1 pour le 01/01/1900 

- la numérotation commence à 0 pour le 01/01/1904 si le paramètre &lt;Depuis 1904&gt; vaut <u><u><u><u>Vrai</u></u></u></u>. 


La partie décimale donne la fraction de jour de la partie heure du résultat (par exemple 0.25 donne une partie heure "060000000").

Attention : Toute conversion de type peut engendrer une perte de précision.

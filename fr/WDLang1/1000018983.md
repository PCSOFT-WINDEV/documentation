
## COMObjet (Type de variable)

***En anglais : COMObject***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **COMObjet** permet de représenter une interface d'un objet COM. Ce type de variable est initialisé grâce à la fonction [COMCréeInstance](../WDLang1/1000018831.md).

Il est ensuite possible d'appeler des méthodes de cette interface avec la fonction [COMAppelleMéthode](../WDLang1/1000018832.md).

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md)


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Initialisation d'une chaîne Unicode de 260 caractères
PapierPeint est une chaîne Unicode
PapierPeint = AnsiVersUnicode(Répète(" ", 260))

// Déclaration d'une variable de type COMObjet
ActiveDesktop est un COMObjet

// Récupération de l'interface de manipulation du bureau
ActiveDesktop = COMCréeInstance("75048700-EF1F-11D0-9888-006097DEACF9", ...
				"F490EB00-1240-11D1-9888-006097DEACF9")
SI ErreurDétectée = Faux ALORS
	// Récupération de l'image de papier peint du bureau
	COMAppelleMéthode(ActiveDesktop, 4, &PapierPeint, 260, 1)
	Info("L'image du bureau est : " + UnicodeVersAnsi(PapierPeint))
FIN
```

## Remarques


### Propriétés associées au type COMObjet 
<a name="proprietes_associees_type_comobjet_ELTPARAGRAPHE000036"></a>

Les propriétés suivantes peuvent être utilisées pour manipuler une variable de type **COMObjet** :

| Nom | Type manipulé | Effet |
| --- | --- | --- |
| Adresse | Entier système | Adresse en mémoire du pointeur d'interface de l'objet COM.<br><br>**Remarque** : Cette propriété est d'un usage avancé uniquement. Il est fortement déconseillé de la manipuler directement. |




<a name="NOTE0_3"></a>


### Fonctions utilisant les variables de type COMObjet
<a name="fonctions_utilisant_les_variables_type_comobjet_ELTPARAGRAPHE000059"></a>


|   |   |
| --- | --- |
| [COMAppelleMéthode](../WDLang1/1000018832.md) | Exécute une méthode d'une interface d'un objet COM instancié grâce à la fonction [COMCréeInstance](../WDLang1/1000018831.md). |
| [COMCréeInstance](../WDLang1/1000018831.md) | Instancie un objet COM (Component Object Model). |
| [COMRécupèreInterface](../WDLang1/1000019784.md) | Appelle la méthode QueryInterface de l'objet COM spécifié pour obtenir une interface spécifique de cet objet afin de l'utiliser avec la fonction [COMAppelleMéthode](../WDLang1/1000018832.md). |




## &lt;Variable Image&gt;.DessineCorde (Fonction)

***En anglais : &lt;Image variable&gt;.DrawChord***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Dessine une corde de cercle (intersection d'une ellipse et d'une droite) dans une variable de type [Image](../WDLang1/1000019650.md).



![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Les dessins](../WDLang1/3029035.md). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonImage est une Image = "Test.bmp"
// Dessin dans une variable Image d'une corde de cercle dont le 
// fond est jaune clair et le bord bleu clair
MonImage.DessineCorde(10, 10, 50, 50, 20, 20, 40, 50, JauneClair, BleuClair)

IMG_MonDessin = MonImage
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Variable Image>.DessineCorde(<X1> , <Y1> , <X2> , <Y2> , <X3> , <Y3> , <X4> , <Y4> [, <Couleur du fond> [, <Couleur du trait>]])`
---

**`<Variable Image> : (Variable de type Image)`**

Nom de la variable de type [Image](../WDLang1/1000019650.md) à manipuler.

**`<X1> : (Entier)`**

Abscisse du coin haut gauche du rectangle dans lequel le cercle est inscrit. Ces coordonnées sont exprimées en pixels.

**`<Y1> : (Entier)`**

Ordonnée du coin haut gauche du rectangle dans lequel le cercle est inscrit. Ces coordonnées sont exprimées en pixels.

**`<X2> : (Entier)`**

Abscisse du coin bas droit du rectangle dans lequel le cercle est inscrit. Ces coordonnées sont exprimées en pixels.

**`<Y2> : (Entier)`**

Ordonnée du coin bas droit du rectangle dans lequel le cercle est inscrit. Ces coordonnées sont exprimées en pixels.

**`<X3> : (Entier)`**

Abscisse du point de départ de la corde de cercle. Ces coordonnées sont exprimées en pixels.

**`<Y3> : (Entier)`**

Ordonnée du point de départ de la corde de cercle. Ces coordonnées sont exprimées en pixels.

**`<X4> : (Entier)`**

Abscisse du point d'arrivée de la corde de cercle. Ces coordonnées sont exprimées en pixels.

**`<Y4> : (Entier)`**

Ordonnée du point d'arrivée de la corde de cercle. Ces coordonnées sont exprimées en pixels.

**`<Couleur du fond> : (Entier ou constante (optionnel))`**

Couleur du fond de la corde de cercle. Cette couleur peut correspondre à :

- une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)),

- une couleur TSL (obtenue avec la fonction [TSL](../WDLang1/3029057.md)),

- [une couleur prédéfinie du WLangage](../WDLang5/3010002.md),

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) une variable de type [Couleur](../WDLang1/1000019661.md).




Si ce paramètre n'est pas précisé, la couleur du fond :

- est *Transparent* si la fonction [&lt;Variable Image&gt;.Fond](../WDLang1/1000024056.md) n'a pas été utilisée auparavant,

- correspond à la couleur précisée dans le dernier appel à la fonction [&lt;Variable Image&gt;.Fond](../WDLang1/1000024056.md).




**`<Couleur du trait> : (Entier ou constante (optionnel))`**

Couleur du trait de la corde du cercle. Cette couleur peut correspondre à :

- une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)),

- une couleur TSL (obtenue avec la fonction [TSL](../WDLang1/3029057.md)),

- [une couleur prédéfinie du WLangage](../WDLang5/3010002.md),

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) une variable de type [Couleur](../WDLang1/1000019661.md).




Si ce paramètre n'est pas précisé, la couleur du trait :

- correspond à la couleur précisée dans le dernier appel à la fonction [&lt;Variable Image&gt;.Stylo](../WDLang1/1000024240.md),

- est identique à la couleur utilisée pour le fond si la fonction [&lt;Variable Image&gt;.Stylo](../WDLang1/1000024240.md) n'a pas été utilisée.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Limites
<a name="limites_ELTPARAGRAPHE000496"></a>

- La fonction **&lt;Variable Image&gt;.DessineCorde** n'est pas disponible en mode Anti-aliasing (pour plus de détails, consultez l'aide de la fonction [&lt;Variable Image&gt;.ChangeMode](../WDLang1/1000024039.md)).

- La fonction **&lt;Variable Image&gt;.DessineCorde** n'est pas disponible en cas de dessin avec gestion du canal alpha (pour plus de détails, consultez l'aide de la fonction [&lt;Variable Image&gt;.DébutDessin](../WDLang1/1000024349.md)). Cette fonction n'a aucun effet.



<a name="NOTE0_2"></a>


### Coordonnées
<a name="coordonnees_ELTPARAGRAPHE000523"></a>Si ***les points de départ et d'arrivée*** ne sont pas situés sur le bord du cercle (ou de l'ellipse), le point pris en compte est l'intersection du cercle et de la droite joignant le point précisé et le centre du rectangle (c'est-à-dire le centre du cercle).<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=RbCorde.gif)
Les coordonnées sont indiquées par rapport au coin haut gauche du champ Image (coordonnées (0,0)).
<a name="NOTE0_3"></a>


### Figure pleine
<a name="figure_pleine_ELTPARAGRAPHE000535"></a>La fonction **&lt;Variable Image&gt;.DessineCorde** permet uniquement de dessiner une corde : contour uniquement ou zone de la corde. Pour dessiner une figure pleine (portion délimitée par le centre du cercle et les deux points sur le cercle), utilisez la fonction [&lt;Variable Image&gt;.DessinePortion](../WDLang1/1000024235.md).

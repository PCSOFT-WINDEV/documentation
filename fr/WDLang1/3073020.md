
## SysModeErreur (Fonction)

***En anglais : SysErrorMode***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie le mode d'affichage des erreurs.

**Attention : Cette fonction est à manipuler avec précaution.**
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Les erreurs critiques ne s'afficheront plus
ResModeErreur = SysModeErreur(smeErreurCritique)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = SysModeErreur(<Mode d'affichage>)`
---

**`<Résultat> : (Constante de type Entier)`**

Ancien mode d'affichage des erreurs : 


|   |   |
| --- | --- |
| *smeEPG* | Pas de panneau blanc lors d'une erreur de protection générale. |
| *smeErreurCritique* | Pas de boîte de dialogue pour les erreurs critiques. |
| *smeErreurFichier* | Pas de boîte de dialogue pour les erreurs fichier. |



**`<Mode d'affichage> : (Constante ou combinaison de constantes)`**

Indique le nouveau mode d'affichage des erreurs : 



|   |   |
| --- | --- |
| *smeEPG* | Pas de panneau blanc lors d'une erreur de protection générale. |
| *smeErreurCritique* | Pas de boîte de dialogue pour les erreurs critiques. |
| *smeErreurFichier* | Pas de boîte de dialogue pour les erreurs fichier. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Mode d'affichage initial
<a name="mode_affichage_initial_ELTPARAGRAPHE000064"></a>Pour récupérer le mode d'affichage initial :

- Conservez le mode d'affichage initial renvoyé par la fonction **SysModeErreur** (lors de la première modification du mode d'affichage).

- Initialisez le mode d'affichage des erreurs avec la valeur du mode d'affichage initial.




Par exemple :


```wl
// Modification du mode d'affichage des erreurs
// Conservation de l'ancien mode d'affichage des erreurs
ResModeErreurInit = SysModeErreur(smeErreurCritique)
...
// Récupération du mode d'affichage initial
SysModeErreur(ResModeErreurInit)
```

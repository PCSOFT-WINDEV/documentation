
## FenAdapteHauteur (Fonction)

***En anglais : WinAdaptHeight***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Adapte la hauteur de la fenêtre au contenu des champs. Les champs sont affichés de manière optimale dans la fenêtre (les ascenseurs inutiles sont supprimés si possible). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affiche le champ TABLE_Prix
TableAffiche(FEN_Prix.TABLE_Prix)
// Adapte la hauteur de la fenêtre
FenAdapteHauteur(FEN_Prix)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`FenAdapteHauteur([<Fenêtre>])`
---

**`<Fenêtre> : (Nom de fenêtre optionnel)`**

Nom de la fenêtre à redimensionner. Si ce paramètre n'est pas précisé, la fenêtre en cours sera manipulée. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Le redimensionnement en hauteur d'une fenêtre est possible uniquement si la fenêtre contient un des champs suivants ancré en hauteur : 

- Champ Liste.

- Champ Table.

- Champ Zone répétée.

- Champ Arbre.

- Champ Liste image

- Champ de saisie multiligne.

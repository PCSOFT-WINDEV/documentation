
## Constantes du champ Editeur HTML
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les constantes utilisées par les fonctions de gestion des champs Editeur HTML sont les suivantes :

| Constante | Détail | Fonction / Type |
| --- | --- | --- |
| **edthtmlSelMultiple** | Type de la sélection : Plusieurs éléments HTML sont sélectionnés. | Type [edtHTMLSélection](../WDLang1/1000026093.md) |
| **edthtmlSelSimple** | Type de la sélection : Un seul élément HTML est sélectionné. | Type [edtHTMLSélection](../WDLang1/1000026093.md) |
| **edthtmlSelVide** | Type de la sélection : Aucune sélection n'est réalisée. | Type [edtHTMLSélection](../WDLang1/1000026093.md) |
| **ehrBody** | Permet d'obtenir le code HTML du body. | [EditeurHTMLRécupère](../WDLang1/1000025981.md) |
| **ehrTexteSansFormat** | Permet d'obtenir le texte contenu dans le champ sans les balises HTML. | [EditeurHTMLRécupère](../WDLang1/1000025981.md) |
| **ehrValeur** | Permet d'obtenir la valeur du champ. | [EditeurHTMLRécupère](../WDLang1/1000025981.md) |
| **emailOptionRépondreAprès** | Options d'affichage de l'email : Le contenu de l'email est affiché en décalé, la partie réponse se trouve après le contenu. | [EditeurHTMLDepuisEmail](../WDLang1/1000026172.md) |
| **emailOptionRépondreAvant** | Options d'affichage de l'email : Le contenu de l'email est affiché en décalé, la partie réponse se trouve avant le contenu. | [EditeurHTMLDepuisEmail](../WDLang1/1000026172.md) |
| **emailOptionTransféré** | Options d'affichage de l'email : Ajoute un entête de réponse au début de l'email. | [EditeurHTMLDepuisEmail](../WDLang1/1000026172.md) |
| **htmlAlignement** | Aligne horizontalement la sélection. | [EditeurHTMLFormateSélection](../WDLang1/1000025996.md) |
| **htmlBarré** | Barre la sélection. | [EditeurHTMLFormateSélection](../WDLang1/1000025996.md) |
| **htmlCouleur** | Change la couleur de la sélection. | [EditeurHTMLFormateSélection](../WDLang1/1000025996.md) |
| **htmlCouleurFond** | Change la couleur de fond de la sélection. | [EditeurHTMLFormateSélection](../WDLang1/1000025996.md) |
| **htmlCréeLien** | Crée un lien sur l'URL spécifiée dans le paramètre . | [EditeurHTMLFormateSélection](../WDLang1/1000025996.md) |
| **htmlExposant** | Passe la sélection en mode "exposant". | [EditeurHTMLFormateSélection](../WDLang1/1000025996.md) |
| **htmlGras** | Passe la sélection en gras. | [EditeurHTMLFormateSélection](../WDLang1/1000025996.md) |
| **htmlIndice** | Passe la sélection en mode "indice". | [EditeurHTMLFormateSélection](../WDLang1/1000025996.md) |
| **htmlItalique** | Passe la sélection en italique. | [EditeurHTMLFormateSélection](../WDLang1/1000025996.md) |
| **htmlMotComplet** | Recherche le mot complet (entouré par des caractères de ponctuation ou des espaces). | [EditeurHTMLRemplace](../WDLang1/1000025995.md) |
| **htmlNomPolice** | Modifie le nom de la police utilisée pour la sélection. | [EditeurHTMLFormateSélection](../WDLang1/1000025996.md) |
| **htmlNomStyleCss** | Modifie le style CSS utilisé par la sélection. | [EditeurHTMLFormateSélection](../WDLang1/1000025996.md) |
| **htmlSensibleCasse** | Recherche en prenant en compte de la casse (majuscules/minuscules) et des accents. | [EditeurHTMLRemplace](../WDLang1/1000025995.md) |
| **htmlSouligné** | Souligne la sélection. | [EditeurHTMLFormateSélection](../WDLang1/1000025996.md) |
| **htmlSupprimeLien** | Supprime le lien de la sélection. | [EditeurHTMLFormateSélection](../WDLang1/1000025996.md) |
| **htmlTaillePolice** | Modifie la taille de la police utilisée dans la sélection. | [EditeurHTMLFormateSélection](../WDLang1/1000025996.md) |
| **htmlTaillePoliceMoins** | Réduit la taille de la police de la sélection. | [EditeurHTMLFormateSélection](../WDLang1/1000025996.md) |
| **htmlTaillePolicePlus** | Augmente la taille de la police de la sélection. | [EditeurHTMLFormateSélection](../WDLang1/1000025996.md) |
| **typehtmlAutre** | Type de la ressource à télécharger : Type inconnu | Evénement "[Chargement de ressources](../WDChamp/1410087152.md)" du champ Editeur HTML. |
| **typehtmlResCss** | Type de la ressource à télécharger : Style CSS | Evénement "[Chargement de ressources](../WDChamp/1410087152.md)" du champ Editeur HTML. |
| **typehtmlResImage** | Type de la ressource à télécharger : Image | Evénement "[Chargement de ressources](../WDChamp/1410087152.md)" du champ Editeur HTML. |
| **typehtmlResScript** | Type de la ressource à télécharger : Script | Evénement "[Chargement de ressources](../WDChamp/1410087152.md)" du champ Editeur HTML. |
| **typehtmlResXMLHttpRequest** | Type de la ressource à télécharger : Document XML généré en réponse à une requête HTTP | Evénement "[Chargement de ressources](../WDChamp/1410087152.md)" du champ Editeur HTML. |

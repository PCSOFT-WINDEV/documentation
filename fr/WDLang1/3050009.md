
## Sinus (Fonction)

***En anglais : Sin***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le sinus d'un angle.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Sinus de 30
ResSinus = Sinus(30)
// Renvoie 0.5
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = Sinus(<Angle>)`
---

**`<Résultat> : (Réel)`**

Sinus de l'angle spécifié.

**`<Angle> : (Entier ou réel)`**

Angle (en degrés) à manipuler.

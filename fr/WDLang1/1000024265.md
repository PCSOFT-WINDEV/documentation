


## &lt;Traitement de texte&gt;.FinModification (Fonction)

***En anglais : &lt;Word Processing&gt;.EndModification***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Signale la fin du regroupement d'un ensemble d'opérations sur un champ Traitement de texte en un seul événement (gestion du "undo/redo").
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
TT_MonDocument.DébutModification()
TT_MonDocument.Remplace("%NomClient%", Client.Nom)
TT_MonDocument.Remplace("%PrénomClient%", Client.Prénom)
TT_MonDocument.FinModification()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Champ Traitement de texte>.FinModification()`
---

**`<Champ Traitement de texte> : (Nom de champ)`**

Nom du champ Traitement de texte à manipuler.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Les modifications effectuées sur le document sont regroupées à partir de l'appel de la fonction [&lt;Traitement de texte&gt;.DébutModification](../WDLang1/1000023405.md). Si plusieurs modifications annulables sont effectuées, il suffit d'un seul "Ctrl + Z" pour annuler toutes les opérations effectuées.

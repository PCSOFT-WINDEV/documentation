


## TCDEnroule (Fonction)

***En anglais : PVTCollapse***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Enroule toutes données d'un champ Tableau croisé dynamique selon un entête de ligne ou de colonne. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Enroule tous les trimestres
TCDEnroule(TCD_Ventes, COL_DateCommande_Trimestre)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`TCDEnroule(<Champ Tableau croisé dynamique> , <Entête à enrouler>)`
---

**`<Champ Tableau croisé dynamique> : (Nom de champ)`**

Nom du champ Tableau croisé dynamique à manipuler.

**`<Entête à enrouler> : (Chaîne de caractères)`**

Nom de l'entête de ligne ou de colonne à enrouler.




## EnModeAJAX (Fonction)

***En anglais : InAJAXMode***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Identifie si le code est exécuté depuis :

- un traitement en mode AJAX.

- une procédure appelée grâce aux fonctions [AJAXExécute](../WDLang2/3055114.md) ou [AJAXExécuteAsynchrone](../WDLang2/3055115.md).




Pour plus de détails sur AJAX, consultez [Présentation de AJAX](../WDLang2/3055101.md).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Code exécuté depuis un traitement AJAX ?
SI EnModeAJAX() = Vrai ALORS
	// Mode AJAX
	// Traitements à effectuer
	...
SINON
	// Mode non AJAX
	// Traitements à effectuer
	...
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = EnModeAJAX()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le code est exécuté depuis un traitement en mode AJAX ou depuis une procédure appelée grâce aux fonctions [AJAXExécute](../WDLang2/3055114.md) ou [AJAXExécuteAsynchrone](../WDLang2/3055115.md), 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




Si le traitement en cours n'est pas en mode AJAX mais s'il a été appelé à partir d'un traitement en mode AJAX (grâce à la fonction [ExécuteTraitement](../WDLang1/3013031.md) par exemple), la fonction **EnModeAJAX** retourne <u><u><u><u>Vrai</u></u></u></u>.  





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
<a name="NOTE0_2"></a>
<a name="NOTE0_3"></a>


### Disponibilité de la fonction
<a name="disponibilite_fonction_ELTPARAGRAPHE000297"></a>La fonction **EnModeAJAX** est disponible dans tous les produits pour permettre l'écriture d'un code multi-produits. Des fonctions ou procédures utilisant la fonction **EnModeAJAX** peuvent être partagées entre plusieurs produits, sans que des erreurs de compilation soient générées.

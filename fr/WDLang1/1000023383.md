


## Fonctions de gestion du champ Editeur d'images et des variables WDPic
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions de gestion du champ Editeur d'images sont : 



|   |   |
| --- | --- |
| [PicCaptureLasso](../WDLang1/1410088683.md) | Effectue une capture d'écran via le lasso et récupère cette capture dans une variable de type [Image](../WDLang1/1000019650.md). |
| [PicCaptureLassoOCR](../WDLang1/1410088839.md) | Lance une capture d'écran par le lasso pour en extraire le texte via une analyse avec le système OCR intégré. |
| [PicCrée](../WDLang1/1000023380.md) | Crée une image dans le champ Editeur d'images. |
| [PicFerme](../WDLang1/1000023382.md) | Ferme l'image se trouvant dans un champ Editeur d'images. |
| [PicFusionneCalque](../WDLang1/1000024865.md) | Fusionne les calques d'une image au format WDPic. |
| [PicOuvre](../WDLang1/1000023379.md) | Ouvre une image dans un champ Editeur d'images. |
| [PicRedimensionne](../WDLang1/1000024635.md) | Redimensionne une image au format WDPic. |
| [PicRotation](../WDLang1/1000024633.md) | Effectue une rotation d'une image au format WDPic. |
| [PicSauve](../WDLang1/1000023381.md) | Sauve l'image présente : <br><br>- dans un champ Editeur d'images.<br><br>- dans une variable de type WDPic. <br><br><br> |


## TableurListeImage (Fonction)

***En anglais : SpreadsheetListImage***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Liste les images présentes dans la feuille courante dans un champ Tableur. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Supprime toutes les images
Id est une chaîne 
POUR TOUS Id DE TableurListeImage(TBLR_Tableur) 
	// Suppression de l'image 
	TableurSupprimeImage(TBLR_Tableur, Id) 
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = TableurListeImage(<Champ Tableur>)`
---

**`<Résultat> : (Tableau)`**

Tableau de chaîne de caractères contenant les identifiants des images présentes dans le champ Tableur. Cet identifiant peut être utilisé dans la fonction [TableurSupprimeImage](../WDLang1/1000023546.md). 

**`<Champ Tableur> : (Nom de champ)`**

Nom du champ Tableur à manipuler.

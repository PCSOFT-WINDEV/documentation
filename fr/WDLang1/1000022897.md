


## &lt;Agenda&gt;.Position (Fonction)

***En anglais : &lt;Organizer&gt;.Position***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Affiche un champ Agenda à partir d'une date spécifiée ou renvoie la date du premier jour (ou dernier jour) affiché dans un champ Agenda.





![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) En mode Universal Windows 10 App, cette fonction permet uniquement de manipuler les champs Agenda présents dans un état. 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Positionne l'agenda sur la date du jour
AGD_Agenda1.Position(DateDuJour())
// Positionne l'agenda au 20 décembre 2012
AGD_Agenda1.Position("20121220")
```
<a name="Exemple2"></a>

```wl
// Obtenir la première date affichée dans l'agenda
DateAgenda est une Date
DateAgenda = AGD_MonAgenda.Position()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Positionner l'agenda sur une date spécifiée

`<Champ Agenda>.Position(<Date de début>)`
---

**`<Champ Agenda> : (Nom de champ)`**

Nom du champ Agenda à manipuler. Ce champ peut correspondre : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) au champ Agenda d'une fenêtre.

- au champ Agenda d'un état.




**`<Date de début> : (Chaîne de caractères ou variable de type Date)`**

Date de début de l'affichage.


<a name="SYNTAXE2"></a>

### Connaître la première date ou la dernière date affichée par le champ Agenda

`<Résultat> = <Champ Agenda>.Position(<Date recherchée>)`
---

**`<Résultat> : (Chaîne de caractères ou variable de type Date)`**

Date affichée par le champ Agenda.

**`<Champ Agenda> : (Nom de champ)`**

Nom du champ Agenda à manipuler. Ce champ peut correspondre : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) au champ Agenda d'une fenêtre.

- au champ Agenda d'un état.




**`<Date recherchée> : (Constante optionnelle de type Entier)`**

Type de date recherchée :


|   |   |
| --- | --- |
| *agDateDébut* <br>(Valeur par défaut) | Première date affichée dans le champ Agenda.<br><br> |
| *agDateFin* | Dernière date affichée dans le champ Agenda. |
| *agDateHeure1erVisible* | Première date et heure affichée dans le champ Agenda (en fonction de la position de l'ascenseur). |
| *agMoisEnCours* | Premier jour du mois affiché dans le champ Agenda. |


**Remarque** : Si le champ Agenda est en mode d'affichage "par jour", les constantes *agDateDébut* et *agDateFin* sont équivalentes.


## Description de Variable (Type de variable)

***En anglais : Variable Description ***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **Description de Variable** permet d'obtenir des informations sur un membre de classe ou de structure.

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Récupère la définition de la classe ou de la structure type de la variable x
MaDéfinition est une Définition
MaDéfinition = RécupèreDéfinition(x)
// Récupère la description du membre nValeur de la classe ou de la structure
m est une Description de Variable 
m = MaDéfinition.Variable["nValeur"]
```

<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Propriétés spécifiques à la description des variables de type Description de Variable
<a name="proprietes_specifiques_description_des_variables_type_description_variable_ELTPARAGRAPHE000046"></a>Les propriétés suivantes peuvent être utilisées pour manipuler une variable de type **Description de Variable** :

| **Nom de la propriété** | **Type manipulé** | **Effet** |
| --- | --- | --- |
| Associé | Booléen | <br><br>- <u><u><u><u>Vrai</u></u></u></u> si la variable provient d'un élément associé, <br><br>- <u><u><u><u>Faux</u></u></u></u> si la variable provient d'un élément de base. <br><br><br>**Cette propriété est disponible en lecture seulement**. |
| AttributCléUnique | Booléen | <br><br>- <u><u><u><u>Vrai</u></u></u></u> si l'attribut &lt;Clé unique&gt; est présent, <br><br>- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.  <br><br><br>**Cette propriété est disponible en lecture seulement**. |
| AttributMapping | Chaîne de caractères | <br><br>- Valeur de l'attribut d'extension &lt;Mapping&gt; pour une classe ou une structure, <br><br>- "" (chaîne vide) si l'attribut est absent (ou pour les autres types). <br><br><br>**Cette propriété est disponible en lecture seulement**. |
| AttributNote | Chaîne de caractères | <br><br>- Valeur de l'attribut d'extension &lt;Note&gt; pour une classe ou une structure, ou pour leurs membres.<br><br>- "" (chaîne vide) si l'attribut est absent (ou pour les autres types). <br><br><br><br><br>**Cette propriété est disponible en lecture seulement**. |
| AttributSérialise | Chaîne de caractères | Nom de sérialisation du type en fonction de son nom et de l'attribut &lt;Sérialise&gt; : <br><br>- Nom de la classe, si l'attribut est absent, <br><br>- Contenu de la chaîne si l'attribut contient une chaîne, <br><br>- "" (chaîne vide) si l'attribut contient Faux ou si le type n'est pas une classe. <br><br><br><br><br>**Cette propriété est disponible en lecture seulement**. |
| Déclaration | [Définition](../WDLang1/1000019540.md) | Définition de la classe ou de la structure qui déclare le membre. <br>**Cette propriété est disponible en lecture seulement**. |
| Définition | [Définition](../WDLang1/1000019540.md) | Définition du type du membre. <br>**Cette propriété est disponible en lecture seulement**. |
| Global | Booléen | <br><br>- <u><u><u><u>Vrai</u></u></u></u> si le membre est global, <br><br>- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. <br><br><br>**Cette propriété est disponible en lecture seulement**. |
| Hérité | Booléen | <br><br>- <u><u><u><u>Vrai</u></u></u></u> si le membre est hérité d'une classe de base, <br><br>- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.<br><br><br>**Cette propriété est disponible en lecture seulement**. |
| Masqué | Booléen | <br><br>- <u><u><u><u>Vrai</u></u></u></u> si le membre est masqué par un membre d'une classe dérivée, <br><br>- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.<br><br><br>**Cette propriété est disponible en lecture seulement**.<br><br> |
| Nom | Chaîne de caractères | Nom du membre. <br>**Cette propriété est disponible en lecture seulement**. |


<a name="NOTE0_2"></a>


### Manipulation
<a name="manipulation_ELTPARAGRAPHE000205"></a>Le membre décrit peut être manipulé grâce à l'indirection : 

- {&lt;objet&gt;, &lt;Description de Variable&gt;}

- {&lt;objet&gt;, &lt;Description de Variable&gt;} = ...


## BitcoinAdresseValide (Fonction)

***En anglais : BitcoinValidAddress***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Vérifie si une adresse d'un portefeuille Bitcoin est valide.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Vérification de l'adresse du portefeuille Bitcoin 
SI PAS BitcoinAdresseValide(SAI_Adresse) ALORS
	Erreur()
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = BitcoinAdresseValide(<Adresse>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'adresse est valide, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. La variable [ErreurDétectée](../WDLang1/3087001.md) vaut <u><u><u><u>Vrai</u></u></u></u> et le message d'erreur correspondant peut être connu avec la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Adresse> : (Chaîne de caractères)`**

Adresse du portefeuille Bitcoin à tester. Cette adresse peut être de n'importe quel type :

- P2PKH (Pay To Public Key Hash, commence par "1"),

- P2SH (Pay To Script Hash, commence par "3"),

- P2WSH (Pay To Witness Script Hash, commence par "3"),

- Segwit v0 ou P2WPKH (Pay To Witness Public Key Hash, commence par "bc1q"),

- Segwit v1 ou P2TR (Pay to Taproot, commence par "bc1p"). 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Cette fonction teste uniquement la validité d'une adresse d'un portefeuille Bitcoin et non pas la validité d'une clé privée ou publique d'un portefeuille Bitcoin.

- La fonction ne permet pas de vérifier si l'adresse correspond à un portefeuille Bitcoin existant.

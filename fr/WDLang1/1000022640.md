


## DocImprime (Fonction)

***En anglais : DocPrint***

<DIV class="specObsolete">
	<IMG style="float:left;margin-right:10px;" src="https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=warning_big.png"><B>Avertissement</B><BR>
	A partir de la version <B>24</B>,  cette fonction est conservée par compatibilité. Il est conseillé d'utiliser la fonction [iImprimeDoc](../WDLang5/1000022390.md) qui offre plus de possibilités.
</DIV><a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Imprime un document au format "docx". 

Remarque : Cette fonction est équivalente à la fonction [iImprimeDoc](../WDLang5/1000022390.md). 




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonDocument est un Document = "c:\temp\propale.docx"
iDestination(iVisualisateur)
DocImprime(MonDocument)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`DocImprime(<Document>)`
---

**`<Document> : (Variable de type Document ou nom de champ)`**

Document à imprimer. Ce document correspond :

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) au nom d'un champ Traitement de texte.

- à une variable de type [Document](../WDLang1/1000022461.md).

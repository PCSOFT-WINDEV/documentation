
## PlanificateurListeTâche (Fonction)

***En anglais : SchedulerTaskList***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Récupère la liste des tâches planifiées.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Remplir la liste LISTE_ListeTache avec le nom des différentes tâches planifiées
ListeAjoute(LISTE_ListeTache, PlanificateurListeTâche())
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = PlanificateurListeTâche()`
---

**`<Résultat> : (Chaîne de caractères)`**



- Liste des noms de tâches séparés par des Retour Chariot (RC), 

- Chaîne vide ("") s'il n'y a pas de tâches planifiées ou en cas d'erreur.


## gVersTexte (Fonction)

***En anglais : gToText***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime les éléments graphiques d'une chaîne de caractères utilisant les fonctions [gValeurMémorisée](../WDLang1/1410088363.md), [gImage](../WDLang1/3029042.md), ...


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
s est une chaîne = gImage("Blog.bmp") + "Test" + gValeurMémorisée("3")
ListeAjoute(LISTE_Liste1, s)

SI gVersTexte(s) = "Test" ALORS
	Info("Test")
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = gVersTexte(<Chaîne à traiter>)`
---

**`<Résultat> : (Chaîne de caractères)`**

Chaîne de caractères sans les éléments graphiques. 

**`<Chaîne à traiter> : (Chaîne de caractères)`**

Chaîne de caractères contenant ou pas des éléments graphiques (fonctions [gValeurMémorisée](../WDLang1/1410088363.md), [gImage](../WDLang1/3029042.md), [gRectangle](../WDLang1/3029045.md), ...).

Si la chaîne ne contient pas d'éléments graphiques, &lt;Résultat&gt; correspondra à la chaîne à traiter.

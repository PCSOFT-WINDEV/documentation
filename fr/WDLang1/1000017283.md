


## CalendrierSelect (Fonction)

***En anglais : CalendarSelect***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la date sélectionnée dans un champ Calendrier.








<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affiche la date sélectionnée dans une fenêtre de trace
Trace(CalendrierSelect(CAL_Calendrier1))
```
<a name="Exemple2"></a>

```wl
// Récupère la date dans un champ de saisie
SAI_Date = CalendrierSelect(CAL_Calendrier1)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = CalendrierSelect(<Champ Calendrier> [, <Indice de l'élément sélectionné>])`
---

**`<Résultat> : (Chaîne de caractères ou variable de type date)`**

Jour sélectionné dans le champ Calendrier. Ce paramètre est de la forme "AAAAMMJJ". Par exemple, si le jour sélectionné est le 12 novembre 2008, &lt;Résultat&gt; vaut "20081112".

**`<Champ Calendrier> : (Nom de champ)`**

Nom du champ Calendrier manipulé.

**`<Indice de l'élément sélectionné> : (Entier optionnel)`**

Rang de la sélection dans le cas d'un calendrier multisélection. 
Si ce paramètre n'est pas spécifié, la date correspondant au premier élément sélectionné sera renvoyée.


## RTFCharge (Fonction)

***En anglais : RTFLoad***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Charge un fichier au format RTF dans un champ RTF (dans une fenêtre ou un état WINDEV).

**Attention** : Il n'est pas possible de récupérer les tableaux et les images présents dans le document RTF.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Charge le fichier "C:\Temp\FichierExemple.RTF" dans le champ SAI_Saisie1
RTFCharge(SAI_Saisie1, "C:\Temp\FichierExemple.RTF")
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = RTFCharge(<Nom du champ RTF> , <Nom du fichier RTF>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le fichier a été chargé, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Le compte-rendu de l'erreur peut être obtenu grâce à la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Nom du champ RTF> : (Chaîne de caractères)`**

Nom du champ RTF à manipuler.

**`<Nom du fichier RTF> : (Chaîne de caractères)`**

Nom du fichier RTF à ouvrir. Ce fichier peut être présent sur un disque accessible depuis le poste en cours ou dans la bibliothèque de l'application (WDL).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Limitation du format RTF
<a name="limitation_format_rtf_ELTPARAGRAPHE000049"></a>Le format des fichiers RTF gérés par WINDEV est le même que le format RTF géré par "Wordpad" (jusqu'à Windows XP). Quelques limites supplémentaires sont à noter lors de l'export d'un état vers un document RTF (par exemple, les tableaux ne sont pas gérés).

Si le fichier RTF est généré par un autre outil (Word, ...), afin de vérifier la compatibilité du RTF, il est conseillé d'ouvrir le fichier avec Wordpad. L'affichage du fichier sous Wordpad donnera un aperçu plus réel de ce qui sera imprimé.

Dans tous les cas il est conseillé de ré-enregistrer le fichier avec Wordpad pour forcer le format du RTF.
 
La fonction **RTFCharge** permet de forcer le format du RTF directement avec WINDEV. Pour cela, il suffit d'utiliser la fonction **RTFCharge** pour affecter le champ RTF.


## CoTang (Fonction)

***En anglais : CoTan***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Calcule la co-tangente d'un angle.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Calcul de la co-tangente de 20
ResCoTangente = CoTang(20)
// Renvoie 2.74747741945
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = CoTang(<Angle>)`
---

**`<Résultat> : (Réel)`**

Co-tangente de l'angle spécifié.

**`<Angle> : (Entier ou réel)`**

Angle (en degrés) à manipuler. Si ce paramètre est égal à 0°, 180° ou -180°, une erreur du WLangage est générée.

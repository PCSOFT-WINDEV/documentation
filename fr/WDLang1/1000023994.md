
## PanneauSauveConfiguration (Fonction)

***En anglais : PanelSaveConfiguration***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Sauve la configuration courante des champs Panneau dockable d'une fenêtre. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Sauve la configuration courante
sConf est une chaîne = PanneauSauveConfiguration()


// Restaure la configuration sauvée dans sConf
PanneauChargeConfiguration(sConf)
SI ErreurDétectée ALORS ToastAffiche(ErreurInfo())
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = PanneauSauveConfiguration()`
---

**`<Résultat> : (Chaîne de caractères)`**

Configuration de tous les champs Panneau dockable de la fenêtre en cours.

Cette configuration peut être sauvée avec les fonctions [fSauveTexte](../WDLang1/3036046.md) ou [SauveParamètre](../WDLang1/3025046.md).

Cette configuration pourra être restaurée grâce à la fonction [PanneauChargeConfiguration](../WDLang1/1000023995.md). 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Les informations suivantes sont sauvées pour chaque champ Panneau dockable de la fenêtre : 

- Taille, 

- Position, 

- Attaché / détaché, 

- Visible / invisible

- Replié / déplié. 




Pour sauver/restaurer la position d'un champ Panneau dockable de la fenêtre, utilisez les fonctions [PanneauPosition](../WDLang1/1000023992.md) et [PanneauAttache](../WDLang1/1000023990.md).

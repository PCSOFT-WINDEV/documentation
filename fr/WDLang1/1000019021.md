


## Fonctions de gestion des champs Agenda
			







Les fonctions de gestion des champs Agenda sont les suivantes :



|   |   |
| --- | --- |
| [AgendaAffiche](../WDLang1/1000019425.md) | Réinitialise le contenu d'un champ Agenda à partir de sa source de données (fichier de données HFSQL par exemple). |
| [AgendaAffichePériodePrécédente](../WDLang1/1000018886.md) | Déplace la zone temporelle affichée sur la zone précédente (par exemple, semaines précédentes) dans un champ Agenda. |
| [AgendaAffichePériodeSuivante](../WDLang1/1000018885.md) | Déplace la zone temporelle affichée sur la zone suivante (par exemple, semaines suivantes) dans un champ Agenda. |
| [AgendaAjouteRendezVous](../WDLang1/1000018877.md) | Ajoute un nouveau rendez-vous dans un champ Agenda. |
| [AgendaChangeMode](../WDLang1/1000019497.md) | Modifie la plage temporelle affichée par un champ Agenda (Jour/Semaine/Mois). |
| [AgendaChangeZoom](../WDLang1/1000018883.md) | Modifie la plage temporelle affichée par un champ Agenda (Jour/Semaine/Mois). Cette fonction est conservée par compatibilité. |
| [AgendaCouleurFond](../WDLang1/1000020004.md) | Fixe une couleur de fond sur une plage horaire d'un champ Agenda. |
| [AgendaDateVisible](../WDLang1/1000022875.md) | Rend visible ou invisible un jour dans un champ Agenda. |
| [AgendaHeureAffichée](../WDLang1/1000018888.md) | Définit la plage horaire affichée par le champ Agenda en mode Jour ou Semaine. |
| [AgendaHeureOuvrable](../WDLang1/1000022530.md) | Permet d'indiquer les heures ouvrables spécifiques par jour dans un champ Agenda. |
| [AgendaInfoXY](../WDLang1/1000018887.md) | Renvoie pour une position donnée dans le champ Agenda (coordonnées d'un point de l'agenda) : <br><br>- soit l'indice du rendez vous pour la position indiquée. <br><br>- soit la date correspondant à la position indiquée.<br><br><br> |
| [AgendaListeRendezVous](../WDLang1/1000018880.md) | Renvoie : <br><br>- la liste des rendez-vous compris entre deux dates données dans un champ Agenda, <br><br>- tous les rendez-vous d'un champ Agenda,<br><br>- un rendez-vous particulier (sélectionné ou survolé).<br><br><br> |
| [AgendaMode](../WDLang1/1000019545.md) | Renvoie le mode d'affichage par un champ Agenda : jour, semaine, mois, ... |
| [AgendaPlageSélectionnée](../WDLang1/1000022399.md) | Renvoie la date et l'heure de début ou de fin de la plage horaire sélectionnée dans un champ Agenda. |
| [AgendaPosition](../WDLang1/1000018889.md) | Affiche un champ Agenda à partir d'une date spécifiée ou renvoie la date du premier jour (ou dernier jour) affiché dans un champ Agenda. |
| [AgendaSelectMoins](../WDLang1/1000019147.md) | Désélectionne un rendez-vous dans un champ Agenda. |
| [AgendaSelectPlus](../WDLang1/1000019003.md) | Sélectionne un rendez-vous dans un champ Agenda. |
| [AgendaSupprimeRendezVous](../WDLang1/1000018881.md) | Supprime un rendez-vous dans un champ Agenda. |
| [AgendaSupprimeTout](../WDLang1/1000018882.md) | Supprime tous les rendez-vous d'un champ Agenda. |
| [AgendaVersiCalendar](../WDLang1/1000021919.md) | Exporte les rendez-vous d'un champ Agenda au format iCalendar. |
| [AgendaZoom](../WDLang1/1000018884.md) | Renvoie la plage temporelle affichée par un champ Agenda. |
| [iCalendarVersAgenda](../WDLang1/1000021918.md) | Importe des données au format iCalendar dans un champ Agenda. |

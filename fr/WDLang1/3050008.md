
## PartieEntière (Fonction)

***En anglais : IntegerPart***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la partie entière d'un nombre. La partie entière d'un nombre est le plus grand entier relatif inférieur ou égal au nombre. Par exemple :

- la partie entière de 3,5 est 3.

- la partie entière de -3,5 est -4.









<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Partie entière de 16.999
ResPartieEntière = PartieEntière(16.999)
// Renvoie 16
// Pour obtenir 17, utiliser la fonction Arrondi
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = PartieEntière(<Nombre>)`
---

**`<Résultat> : (Réel)`**

Partie entière recherchée. Correspond au plus grand entier inférieur ou égal à &lt;Nombre&gt;.

**`<Nombre> : (Réel)`**

Nombre à manipuler.


## CaméraArrête (Fonction)

***En anglais : CameraStop***



<a name="XUtilisation"></a>


<a name="Utilisation"></a>
<a name="description"></a>
Arrête l'affichage ou l'enregistrement d'une séquence vidéo affichée dans un champ Caméra.






<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Arrête l'affichage de la séquence vidéo actuellement retransmise
// dans le champ Caméra "CAM_Cuisine"
ResArrêt = CaméraArrête(CAM_Cuisine, viArrêteAffichage)
```

<a name="XSYNTAXE"></a>


<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = CaméraArrête(<Champ Caméra> [, <Action à arrêter>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'affichage et/ou l'enregistrement de la séquence vidéo dans le champ Caméra est correctement arrêté, 

- <u><u><u><u>Faux</u></u></u></u> en cas d'erreur. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Champ Caméra> : (Nom de champ)`**

Nom du champ Caméra à manipuler.

**`<Action à arrêter> : (Constante optionnelle de type entier)`**

Type d'action à arrêter :


|   |   |
| --- | --- |
| *viArrêteAffichage* | Arrête l'affichage de la vidéo. Le champ Caméra spécifié devient vide. |
| *viArrêteCapture* | Arrête l'enregistrement de la séquence vidéo. Le fichier généré est disponible. |
| *viArrêteTout*<br>(Valeur par défaut) | Arrête l'affichage et l'enregistrement de la séquence vidéo. Le fichier généré est disponible. Le champ Caméra spécifié devient vide. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Conditions d'utilisation
<a name="conditions_utilisation_ELTPARAGRAPHE000065"></a>

- ![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png) Pour utiliser cette fonction, une "caméra" doit être connectée au poste en cours.

- **Remarque** : Cette fonction est équivalente à la fonction [VidéoArrête](../WDLang1/3079001.md). 



<a name="NOTE0_2"></a>


### Cas particuliers
<a name="cas_particuliers_ELTPARAGRAPHE000084"></a>

- Pour enregistrer la séquence vidéo actuellement retransmise par une "web caméra", utilisez la fonction [CaméraCapture](../WDLang1/1000022450.md).

- Pour afficher dans un champ Caméra la séquence vidéo actuellement retransmise par une "web caméra", utilisez la fonction [CaméraAffiche](../WDLang1/1410086595.md).




## Tests automatiques : Fonctions de pilotage
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions de pilotage permettent de piloter les différents types de fenêtres pouvant apparaître lors d'un test automatique. Ces fonctions, utilisées dans le code du test, permettent d'effectuer une action. Elles évitent ainsi le blocage du test. 

Les fonctions de pilotage sont les suivantes :



|   |   |
| --- | --- |
| [PiloteActionBar](../WDLang1/1000021199.md) | Pilote une Action Bar pendant les tests automatiques exécutés sur mobile (Android ou iOS). |
| [PiloteAvertissement](../WDLang1/1000009001.md) | Pilote le prochain appel de la fonction [Avertissement](../WDLang1/3021009.md) et clique automatiquement sur le bouton "OK". |
| [PiloteCombo](../WDLang1/1000025309.md) | Pilote une des actions suivantes sur un champ Combo : <br><br>- clic sur le bouton permettant de dérouler la liste du champ Combo. <br><br>- sélection d'un élément dans un champ Combo.<br><br><br> |
| [PiloteConfirmer](../WDLang1/1000009003.md) | Pilote le prochain appel de la fonction [Confirmer](../WDLang1/3021007.md), puis clique automatiquement sur le bouton passé en paramètre. |
| [PiloteDialogue](../WDLang1/1000009004.md) | Pilote le prochain appel de la fonction [Dialogue](../WDLang1/3021015.md), puis clique automatiquement sur le bouton passé en paramètre. |
| [PiloteDnD](../WDLang1/1410088663.md) | Pilote un "Glisser-Déposer" durant les tests automatiques. |
| [PiloteErreur](../WDLang1/1000009005.md) | Pilote le prochain appel de la fonction [Erreur](../WDLang1/3021013.md), puis clique automatiquement sur le bouton "OK". |
| [PiloteFenêtre](../WDLang1/1000009009.md) | Indique le test qui sera utilisé lors de la prochaine ouverture d'une fenêtre. |
| [PiloteInfo](../WDLang1/1000009006.md) | Pilote le prochain appel de la fonction [Info](../WDLang1/3021011.md), puis clique automatiquement sur le bouton "OK". |
| [PiloteInterrupteur](../WDLang1/1000021200.md) | Pilote la sélection d'une option dans un champ Interrupteur. |
| [PiloteMenu](../WDLang1/1000009010.md) | Pilote le prochain menu qui sera ouvert, puis exécute automatiquement une option de ce menu indiquée en paramètre. |
| [PiloteOKAnnuler](../WDLang1/1000009007.md) | Pilote le prochain appel de la fonction [OKAnnuler](../WDLang1/3021004.md), puis clique automatiquement sur le bouton passé en paramètre. |
| [PiloteOuiNon](../WDLang1/1000009008.md) | Pilote le prochain appel de la fonction [OuiNon](../WDLang1/3021005.md), puis clique automatiquement sur le bouton indiqué en paramètre. |
| [PiloteSaisie](../WDLang1/1000009012.md) | Effectue une saisie dans un champ par pilotage, en vérifiant qu'une saisie dans ce champ est possible. |
| [PiloteSélecteur](../WDLang1/1000021201.md) | Pilote la sélection d'une option dans un champ Sélecteur. |
| [PiloteSimulateur](../WDLang1/1000021151.md) | Pilote la fenêtre du simulateur mobile (Android ou iOS). |
| [PiloteSouris](../WDLang1/1000009011.md) | Simule un événement de la souris (clic gauche, clic droit, double-clic, ...). |
| [PiloteVitesse](../WDLang1/1000022075.md) | Règle la vitesse d'exécution des fonctions de pilotage. |
| [PiloteVolet](../WDLang1/1000021198.md) | Pilote la sélection d'un volet d'un champ Onglet, Ruban ou Boîte à outils. |
| [PiloteVoletAttache](../WDLang1/1000021582.md) | Pilote la réintégration d'un volet détaché dans le champ Onglet dynamique associé. |
| [PiloteVoletDétache](../WDLang1/1000021581.md) | Pilote le détachement d'un volet d'un champ Onglet dynamique. |

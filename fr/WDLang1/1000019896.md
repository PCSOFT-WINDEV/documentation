
## LigneVersPosition (Fonction)

***En anglais : LineToPosition***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la position du premier caractère d'une ligne dans un bloc de texte.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Initialisation d'une chaîne multiligne
s est une chaîne = [
AA
BB
CC
]

// la fonction renvoie 1
LigneVersPosition(s, 1)
// la fonction renvoie 5
LigneVersPosition(s, 2)
// la fonction renvoie 9
LigneVersPosition(s, 3)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = LigneVersPosition(<Chaîne> , <Ligne>)`
---

**`<Résultat> : (Entier)`**

Position du premier caractère de la ligne spécifiée. 

**`<Chaîne> : (Chaîne de caractères)`**

Chaîne de caractères à manipuler. 

**`<Ligne> : (Entier)`**

Numéro de la ligne recherchée. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Le séparateur de lignes est le retour chariot (RC). Il correspond à 2 caractères. 

- Le ou les caractères constituant le séparateur de ligne sont considérés comme appartenant à la ligne précédente.




## &lt;Type Chaîne&gt;.CaractOccurrenceType (Fonction)

***En anglais : &lt;String type&gt;.CharactTypeCount***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le nombre de caractères vérifiant les informations de type données.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
NbCaract est un entier
MaChaîne est une chaîne = "AbcdeF"
NbCaract = MaChaîne.CaractOccurrenceType(ctMajuscule)
// NbCaract = 2
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Chaîne>.CaractOccurrenceType(<Type>)`
---

**`<Résultat> : (Entier)`**

Nombre de caractères de la chaîne correspondant au type recherché.

**`<Chaîne> : (Chaîne de caractères)`**

Chaîne de caractères à manipuler.

**`<Type> : (Constante ou combinaison de constantes de type Entier)`**

Type des caractères à rechercher : 


|   |   |
| --- | --- |
| *ctAccent* | Caractère accentué ou diacritique. |
| *ctEspace* | Espace. |
| *ctLettre* | Lettre. |
| *ctMajuscule* | Caractère majuscule. |
| *ctMinuscule* | Caractère minuscule. |
| *ctNumérique* | Caractère numérique. |
| *ctPonctuation* | Caractère de ponctuation. |


Remarques : 

- La combinaison de constantes *ctMajuscule* + *ctAccent* permet de rechercher les majuscules accentuées. 

- La combinaison de constantes *ctMinuscule* + *ctAccent* permet de rechercher les minuscules accentuées.

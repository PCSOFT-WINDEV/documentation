
## ServiceInfo (Fonction)

***En anglais : ServiceInfo***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie des informations concernant la configuration d'un service. La configuration du service est affectée à la [variable Service](../WDLang1/1000017118.md).

Ce service peut être un service présent sur le poste local ou sur un poste distant.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Obtenir les caractéristiques du service présent dans la table TABLE_Service
SI ServiceInfo(COL_Nom[TABLE_Service]) = Faux ALORS
	Erreur(ErreurInfo(errMessage))
SINON
	Info("Variable Service mise à jour")
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = ServiceInfo(<Nom du service> [, <Poste distant>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si les informations ont été demandées, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md). La variable Service est alors remplie avec les informations concernant la configuration du service.




**`<Nom du service> : (Chaîne de caractères)`**

Nom du service à manipuler.

**`<Poste distant> : (Chaîne de caractères optionnelle)`**

Adresse IP du poste distant au format xxx.xxx.xxx.xxx (obtenu par la fonction [NetAdresseIP](../WDLang3/3056023.md)) ou nom du poste distant (obtenu par la fonction [NetNomMachine](../WDLang3/3056014.md)). Ce paramètre permet d'obtenir des informations sur un service sur un autre poste. Pour utiliser cette option, l'utilisateur doit être administrateur de la machine distante. 
Si ce paramètre n'est pas précisé ou correspond à une chaîne vide (""), le service est manipulé en local.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Variable Service
<a name="variable_service_ELTPARAGRAPHE000060"></a>La variable Service permet de connaître les éléments suivants. Pour plus de détails, consultez la page d'aide concernant la [variable Service](../WDLang1/1000017118.md). 


|   |   |
| --- | --- |
| Service.Démarrage | Mode de démarrage du service |
| Service.Description | Description du service |
| Service.ErreurDémarrage | Traitement par Windows des erreurs lors du démarrage d'un service (cas des services avec démarrage automatique). |
| Service.LigneCommande | Ligne de commande complète du service (chemin de l'exécutable et paramètres). |
| Service.Type | Type du service |
| Service.Utilisateur | Compte utilisateur qui exécutera le service. |


## Puissance (Fonction)

***En anglais : Power***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Élever un nombre à une puissance.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// 8 puissance 5
ResPuissance = Puissance(8, 5)
// Renvoie 32768
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = Puissance(<Nombre à élever> , <Exposant>)`
---

**`<Résultat> : (Réel)`**

Résultat du &lt;Nombre à élever&gt; à la puissance &lt;Exposant&gt;.

**`<Nombre à élever> : (Entier ou réel)`**

Nombre à élever à la puissance.

**`<Exposant> : (Entier ou réel)`**

Exposant.

### Equivalence
<a name="equivalence_ELTPARAGRAPHE000078"></a>La fonction **Puissance** est équivalente à l'opérateur arithmétique **^**. La syntaxe de cet opérateur est la suivante : 


```txt
<Nombre à élever> ^ <Exposant>
```


## ChampContextuel (Fonction)

***En anglais : ControlContextMenu***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Identifie le champ sur lequel le menu contextuel a été ouvert.

**Attention** : La fonction **ChampContextuel** doit être utilisée uniquement dans les événements WLangage associés aux menus contextuels et dans les procédures éventuelles qu'ils appellent.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// -- Code de l'option menu
// Identification du champ sur lequel le menu contextuel a été ouvert
ResChamp = ChampContextuel()
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = ChampContextuel()`
---

**`<Résultat> : (Chaîne de caractères)`**



- Nom du champ (en majuscules) sur lequel le menu contextuel a été ouvert, 

- Chaîne vide ("") si aucun menu contextuel n'a été ouvert.




Si le champ appartient à un superchamp, &lt;Résultat&gt; est de la forme &lt;Nom du superchamp&gt;.&lt;Nom du champ&gt;.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Lorsqu'un même menu contextuel est affecté à plusieurs champs, la fonction **ChampContextuel** permet d'identifier le champ sur lequel le menu a été déroulé afin d'exécuter le traitement approprié.

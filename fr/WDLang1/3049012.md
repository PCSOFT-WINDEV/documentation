


## ListeSelectPlus (Fonction)

***En anglais : ListSelectPlus***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Sélectionne un ou plusieurs éléments dans un champ Liste, Liste image ou Combo.

L'élément est sélectionné mais n'est pas affiché s'il n'est pas visible. Pour rendre l'élément sélectionné visible, utilisez la fonction [ListePosition](../WDLang1/3049015.md). 

**Rappel** : L'élément sélectionné correspond à l'élément en cours sur lequel le bandeau de sélection est affiché.

**Remarque** : La fonction **ListeSelectPlus** peut être utilisée indifféremment sur :

- les champs Liste fichier ou Liste par programmation,

- les champs Liste monosélection ou multisélection.











<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Sélection de l'élément dont l'indice est 7 dans le champ "LISTE_ListeClient"
ListeSelectPlus(LISTE_ListeClient, 7)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`ListeSelectPlus(<Champ Liste> [, <Indice 1> [... [, <Indice N>]]])`
---

**`<Champ Liste> : (Nom de champ)`**

Nom du champ Liste (ou Combo) à manipuler.
Si ce paramètre correspond à une chaîne vide (""), l'élément spécifié selon son indice est sélectionné dans le champ Liste (ou Combo) auquel appartient l'événement en cours.



**`<Indice 1> : (Entiers optionnels)`**

Indice du premier élément à sélectionner. 
Si aucun indice n'est précisé, tous les éléments du champ Liste (ou Combo) spécifié sont sélectionnés. 
Si ces paramètres correspondent à des éléments déjà sélectionnés, la fonction **ListeSelectPlus** est sans effet.



**`<Indice N> : (Entier optionnel)`**

Indice du Nième élément à sélectionner. 
Si aucun indice n'est précisé, tous les éléments du champ Liste (ou Combo) spécifié sont sélectionnés. 
Si ces paramètres correspondent à des éléments déjà sélectionnés, la fonction **ListeSelectPlus** est sans effet.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Dernier élément
<a name="dernier_element_ELTPARAGRAPHE000126"></a>Pour se positionner sur le dernier élément d'un champ Liste, la syntaxe est :

```txt
ListeSelectPlus(<Champ Liste>, <Champ Liste>.Occurrence)
```
ou

```txt
<Champ Liste> = <Champ Liste>.Occurrence
```
ou

```txt
<Champ Liste> = ListeOccurrence(<Champ Liste>)
```

<a name="NOTE0_2"></a>


### Équivalence
<a name="equivalence_ELTPARAGRAPHE000147"></a>Dans le cas d'un champ Liste monosélection sans l'utilisation de la fonction [gValeurMémorisée](../WDLang1/1410088363.md), les deux syntaxes suivantes sont équivalentes :

```txt
ListeSelectPlus(<Champ Liste>, <Indice>)
```


et

```txt
<Champ Liste> = <Indice>
```

<a name="NOTE0_3"></a>


### Divers
<a name="divers_ELTPARAGRAPHE000168"></a>

- Pour sélectionner un élément dans un champ Table, utilisez la fonction [TableSelectPlus](../WDLang1/3074031.md).

- La fonction **ListeSelectPlus** peut être utilisée sur une colonne d'un champ Table de type "Combo".

- La fonction [ListeSelectMoins](../WDLang1/3049008.md) permet de désélectionner les éléments d'un champ Liste ou d'un champ Combo.

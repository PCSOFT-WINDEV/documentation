


## Fonctions statistiques
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


Les fonctions statistiques sont les suivantes :



|   |   |
| --- | --- |
| [StatCorrélation](../WDLang1/3050022.md) | Calcule le coefficient de corrélation entre deux séries de valeurs. |
| [StatCovariance](../WDLang1/3050020.md) | Calcule la covariance entre deux séries de valeurs. |
| [StatEcartMoyen](../WDLang1/3050024.md) | Calcule la moyenne des écarts absolus des valeurs par rapport à leur moyenne arithmétique. |
| [StatEcartType](../WDLang1/3050025.md) | Calcule l'écart type d'une série de valeurs. |
| [StatEcartTypeP](../WDLang1/3050026.md) | Calcule l'écart type d'une série complète de valeurs. |
| [StatErreur](../WDLang1/3050019.md) | Calcule le numéro de la dernière erreur due à une fonction Statxx ([fonctions statistiques](../WDLang1/3050027.md)). |
| [StatMax](../WDLang1/3050034.md) | Calcule la valeur maximale d'une série de valeurs. |
| [StatMin](../WDLang1/3050037.md) | Calcule la valeur minimale d'une série de valeurs. |
| [StatMoyenne](../WDLang1/3050036.md) | Calcule la moyenne arithmétique, géométrique ou harmonique d'une série de valeurs. |
| [StatSomme](../WDLang1/3050035.md) | Calcule le cumul d'une série de valeurs. |
| [StatVariance](../WDLang1/3050033.md) | Calcule la variance d'une série de valeurs. |
| [StatVarianceP](../WDLang1/3050030.md) | Calcule la variance d'une série complète de valeurs. |

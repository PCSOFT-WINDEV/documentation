
## FenColleFormulaire (Fonction)

***En anglais : WinPasteForm***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Colle le contenu d'un formulaire stocké dans le presse-papiers.

**Remarque** : Le contenu du formulaire a été préalablement copié : 

- soit par la fonction [FenCopieFormulaire](../WDLang1/1000018992.md).

- soit par la FAA (Fonctionnalité Automatique de vos Applications) de copie de formulaire.



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Copie les valeurs de formulaire
FenColleFormulaire() // équivalent à FenColleFormulaire(MaFenêtre)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`FenColleFormulaire([<Nom de la fenêtre>])`
---

**`<Nom de la fenêtre> : (Chaîne de caractères optionnelle)`**

Nom ou alias de la fenêtre à manipuler. Si ce paramètre n'est pas précisé ou correspond à une chaîne vide (""), la fenêtre en cours est manipulée.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Types de champs pris en compte
<a name="types_champs_pris_compte_ELTPARAGRAPHE000040"></a>Les types de champ supportés sont :

- Champs de saisie,

- Sélecteurs/Interrupteurs,

- Potentiomètres, potentiomètres rotatifs,

- Calendriers,

- Combos en saisie.




## &lt;Type Date&gt;.VersNuméroDeSemaine (Fonction)

***En anglais : &lt;Date type&gt;.ToWeekNumber***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le numéro de la semaine dans l'année correspondant à la date spécifiée.








<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MaDate est une Date = DateDuJour()
MaDate.VersNuméroDeSemaine()
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Date>.VersNuméroDeSemaine()`
---

**`<Résultat> : (Entier)`**

Numéro de la semaine (entre 0 et 53) correspondant à la date spécifiée.

**`<Date> : (Variable de type Date ou variable de type DateHeure)`**

Date à manipuler. Cette date peut correspondre à : 

- une variable de type Date. 

- une variable de type DateHeure. Dans ce cas, seule la date est prise en compte.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Caractéristiques des semaines
<a name="caracteristiques_des_semaines_ELTPARAGRAPHE000239"></a>Une semaine commence un lundi et se termine un dimanche.

Pour les premiers jours de l'année :

- Si le premier jour de l'année est un vendredi, samedi ou dimanche, le numéro de semaine sera 0. La semaine numéro 1 correspondra au premier lundi de l'année.

- Si le premier jour de l'année est un lundi, mardi, mercredi ou jeudi, le numéro de semaine sera 1.

- Les jours compris entre le dimanche de la 52ème semaine et le 1er janvier sont dans la semaine 53.




Remarque : Pour obtenir un numéro de semaine correspondant à la norme ISO 8601, utilisez la fonction [NuméroDeSemaine](../WDLang1/3027027.md).
<a name="NOTE0_2"></a>


### Cas particulier
<a name="cas_particulier_ELTPARAGRAPHE000260"></a>Le format de stockage des dates permet de stocker des dates comprises entre le 01/01/0001 et le 31/12/9999.

Les calculs sur les dates réalisés avec les [fonctions WLangage](../WDLang1/1000023943.md) ou les [propriétés WLangage](../Proprietes/2514010.md) sont fiables à partir du 1er Janvier 1583.

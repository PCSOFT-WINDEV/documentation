
## &lt;Tableur&gt;.FusionneCellule (Fonction)

***En anglais : &lt;Spreadsheet&gt;.MergeCell***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Fusionne les cellules spécifiées ou les cellules sélectionnées dans un champ Tableur.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Fusionne les cellules du rectangle défini par B2 en haut-gauche et D4 en 
// bas-droite, la donnée de B2 est centrée dans la nouvelle cellule fusionnée
TBLR_monTableur["B2"] = "Titre"
TBLR_monTableur.FusionneCellule("B2", "D4")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Fusionner les cellules spécifiées

`<Champ Tableur>.FusionneCellule(<Cellule début> , <Cellule fin> [, <Type de fusion>])`
---

**`<Champ Tableur> : (Nom de champ)`**

Nom du champ Tableur à manipuler.

**`<Cellule début> : (Chaîne de caractères optionnelle)`**

Nom de la première cellule définissant le rectangle des cellules à fusionner (cellule en haut à gauche du rectangle des cellules à fusionner). Ce paramètre est au format &lt;Lettre de la colonne&gt;&lt;Numéro de la ligne&gt;. Par exemple : "A32".

**`<Cellule fin> : (Chaîne de caractères optionnelle)`**

Nom de la dernière cellule définissant le rectangle des cellules à fusionner (cellule en bas à droite du rectangle des cellules à fusionner). Ce paramètre est au format &lt;Lettre de la colonne&gt;&lt;Numéro de la ligne&gt;. Par exemple : "A32".

**`<Type de fusion> : (Constante optionnelle de type Entier)`**

Type de fusion à effectuer : 


|   |   |
| --- | --- |
| *fusionCelluleCentrée*<br>(Valeur par défaut) | Fusionne les cellules en centrant le contenu. |
| *fusionCelluleSimple* | Fusionne les cellules sans centrer le contenu. |




<a name="SYNTAXE2"></a>

### Fusionner les cellules de la sélection en cours

`<Champ Tableur>.FusionneCellule([<Type de fusion>])`
---

**`<Champ Tableur> : (Nom de champ)`**

Nom du champ Tableur à manipuler.

**`<Type de fusion> : (Constante optionnelle de type Entier)`**

Type de fusion à effectuer : 


|   |   |
| --- | --- |
| *fusionCelluleCentrée*<br>(Valeur par défaut) | Fusionne les cellules en centrant le contenu. |
| *fusionCelluleSimple* | Fusionne les cellules sans centrer le contenu. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Il est également possible de fusionner des cellules sélectionnées d'un champ Tableur avec la fonction [FAAExécute](../WDLang1/1000022099.md) (utilisation de la constante *faaTableurFusion*).

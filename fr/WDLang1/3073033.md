
## SysNumSérie (Fonction)

***En anglais : SysSerialNum***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le numéro IMEI de l'appareil en cours.

Cette fonction est équivalente à la fonction [SysIMEI](../WDLang1/3073029.md). 

Attention : Cette information peut être récupérée jusqu'à Android 9. A partir d'Android 10, cette information n'est plus accessible : cette fonction provoque une erreur fatale.  


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI Val(SysVersionAndroid(sysVersionNuméro))<10 ALORS  
	// Récupération du numéro de série du téléphone en cours
	Info("Le numéro de série de ce téléphone est " + SysNumSérie())
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = SysNumSérie()`
---

**`<Résultat> : (Chaîne de caractères)`**



- Numéro de série du téléphone Android en cours, 

- Chaîne vide ("") en cas d'erreur.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Cas particuliers
<a name="cas_particuliers_ELTPARAGRAPHE000054"></a>

- La fonction [SysIMEI](../WDLang1/3073029.md) permet d'obtenir le numéro IMEI d'un téléphone. 

- Tous les appareils qui n'ont pas d'emplacement de carte SIM n'ont pas de numéro IMEI (cas de la majorité des tablettes).

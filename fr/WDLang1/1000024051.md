


## &lt;Type Date&gt;.VersMoisEnLettre (Fonction)

***En anglais : &lt;Date type&gt;.ToMonthInAlpha***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le mois en toutes lettres correspondant à la date spécifiée.










<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MaDate est une Date = DateSys()
SAI_Date = MaDate.VersJourEnLettre() + " " + MaDate.Jour + ...
	" " + MaDate.VersMoisEnLettre() + " à " + ...
	HeureVersChaîne(HeureSys(), "HH:MM:SS")
// Le champ SAI_Date contient par exemple : Lundi 13 Août à 14:44:17
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Date>.VersMoisEnLettre()`
---

**`<Résultat> : (Chaîne de caractères)`**

Mois correspondant à la date spécifiée : Janvier, Février, Mars, Avril, Mai, Juin, Juillet, Août, Septembre, Octobre, Novembre, Décembre.
**Remarque** : Le mois renvoyé dépend des options linguistiques spécifiées pour les dates dans l'onglet "Langues" de la description du projet (sous le volet "Projet", dans le groupe "Projet", cliquez sur "Description") :

- Si le projet utilise les options linguistiques de Windows pour les dates, le mois sera renvoyé dans la langue de Windows.

- Si le projet utilise des paramètres spécifiques pour les dates (pour une ou plusieurs langues), le mois renvoyé correspondra à celui indiqué dans la description du projet.




**`<Date> : (Variable de type Date ou variable de type DateHeure)`**

Date à manipuler. Cette date peut correspondre à : 

- une variable de type Date. 

- une variable de type DateHeure. Dans ce cas, seule la date est prise en compte.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Le format de stockage des dates permet de stocker des dates comprises entre le 01/01/0001 et le 31/12/9999.

Les calculs sur les dates réalisés avec les [fonctions WLangage](../WDLang1/1000023943.md) ou les [propriétés WLangage](../Proprietes/2514010.md) sont fiables à partir du 1er Janvier 1583.




## &lt;Tableur&gt;.Remplace (Fonction)

***En anglais : &lt;Spreadsheet&gt;.Replace***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Recherche et remplace une valeur dans un champ Tableur. 

Remarque : Cette opération peut être annulée par l'utilisateur via la combinaison de touches Ctrl + Z.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Remplacement de TVA par Taxe V.A
nRempl est entier = TBLR_MonTableur.Remplace("TVA" , "Taxe V.A.")
SI nRempl = 0 ALORS
	Info("Pas trouvé")
SINON
	Info("[%nRempl%] éléments ont été remplacés.")
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Champ Tableur>.Remplace(<Valeur recherchée> , <RemplacerPar> [, <Cellule> [, <Options>]])`
---

**`<Résultat> : (Entier)`**



- Nombre de remplacements effectués.

- 0 si aucun remplacement n'a été effectué.




**`<Champ Tableur> : (Nom de champ)`**

Nom du champ Tableur à manipuler.

**`<Valeur recherchée> : (Réel ou chaîne de caractères)`**

Valeur à rechercher dans le champ Tableur pour être remplacée.

**`<RemplacerPar> : (Réel ou chaîne de caractères)`**

Nouvelle valeur qui doit remplacer la valeur recherchée.

**`<Cellule> : (Chaîne de caractères optionnelle)`**

Nom de la cellule à modifier. 

- Si ce paramètre est spécifié, seules les valeurs correspondantes dans cette cellule sont remplacées. 

- Si ce paramètre n'est pas spécifié, toutes les valeurs correspondantes (dans la feuille en cours ou dans toutes les feuilles, selon le paramètre &lt;Options&gt;) sont modifiées.




**`<Options> : (Constante de type Entier optionnelle)`**

Options de recherche : 



|   |   |
| --- | --- |
| *tblrRechAvecCasse* | Recherche avec prise en compte de la casse. |
| *tblrRechContient* | Recherche de type "Contient". |
| *tblrRechDansFormule* | Recherche uniquement dans les formules. |
| *tblrRechExact* | Recherche de type "Exact". |
| *tblrRechFeuilleEnCours* | Recherche dans la feuille en cours. |
| *tblrRechSansCasse* | Recherche sans prendre en compte la casse. |
| *tblrRechTouteFeuille* | Recherche dans toutes les feuilles du classeur. |


Par défaut, la recherche se fait selon la combinaison de constantes suivante : *tblrRechAvecCasse* + *tblrRechFeuilleEnCours* + *tblrRechExact*

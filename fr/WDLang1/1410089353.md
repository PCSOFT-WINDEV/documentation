
## Canada.VérifieNE9 (Fonction)

***En anglais : Canada.CheckBN9***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Teste la validité d'un numéro NE9 (numéro d'entreprise à 9 chiffres servant à identifier une entreprise au Canada).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Test d'une variable
MonNuméroNE9 est une chaîne = "887471886"
Canada.VérifieNE9(MonNuméroNE9)		// renvoie Vrai

// Test d'un champ
SAI_NuméroNE9 = "ABC"
Canada.VérifieNE9(SAI_NuméroNE9)	// renvoie Faux
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = Canada.VérifieNE9(<Numéro à vérifier>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'élément spécifié contient un numéro NE9 valide.

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Numéro à vérifier> : (Nom de champ ou variable de type chaîne)`**

Numéro NE9 à vérifier. Ce paramètre peut correspondre :

- au nom d'une variable de type chaîne.

- au nom d'un champ.

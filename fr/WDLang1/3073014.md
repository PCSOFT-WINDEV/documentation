
## SysEnvironnement (Fonction)

***En anglais : SysEnvironment***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie les variables d'environnement du système d'exploitation du poste en cours.



**Remarque** : Pour visualiser et/ou modifier les variables d'environnement du poste en cours, ouvrez le panneau de configuration et cliquez sur l'icône "Système".


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connaître toutes les variables d'environnement du système d'exploitation
ResEnvironnement = SysEnvironnement()
```
<a name="Exemple2"></a>

```wl
// Connaître les variables d'environnement du système d'exploitation
Trace(SysEnvironnement("homepath"))
// renvoie par exemple : 
// homepath=\Users\doc
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = SysEnvironnement([<Variable d'environnement>])`
---

**`<Résultat> : (Chaîne de caractères)`**



- Environnement complet du poste en cours (ou du poste serveur) si aucune &lt;Variable d'environnement&gt; n'est spécifiée. Les lignes de l'environnement sont séparées par des RC.

- Ligne de l'environnement dont le premier mot correspond au paramètre &lt;Variable d'environnement&gt;.

- Une chaîne vide ("") en cas de problème.




**`<Variable d'environnement> : (Chaîne de caractères optionnelle)`**

Variable à rechercher dans l'environnement du poste en cours (par exemple, "PATH" ou "USERNAME").  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Recherche
<a name="recherche_ELTPARAGRAPHE000052"></a>

- La recherche selon le paramètre &lt;Variable d'environnement&gt; est générique. Ainsi, SysEnvironnement("PR") retourne la ligne concernant le prompt (si elle existe dans l'environnement).

- Si plusieurs lignes peuvent répondre à la recherche, seule la première est retournée.





<a name="NOTE0_2"></a>




### Équivalence
<a name="equivalence_ELTPARAGRAPHE000062"></a>La fonction **SysEnvironnement** retourne le même résultat que la commande SET de la ligne de commande (DOS).

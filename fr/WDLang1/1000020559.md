
## FIListeSupprime (Fonction)

***En anglais : IWListDelete***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime une fenêtre interne présente dans la liste des fenêtres internes parcourue par un champ Fenêtre Interne.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Supprime la première fenêtre interne ajoutée dans le champ
FIListeSupprime(FI_ChampFenêtreInterne, 1)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = FIListeSupprime(<Champ Fenêtre Interne> [, <Position>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la fenêtre interne a été supprimée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Champ Fenêtre Interne> : (Nom de champ)`**

Nom du champ Fenêtre Interne à manipuler.

**`<Position> : (Entier optionnel)`**

Position de la fenêtre interne à supprimer. Si ce paramètre n'est pas spécifié, la fenêtre interne en cours d'affichage sera supprimée.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Si la fenêtre interne supprimée correspond à la fenêtre interne actuellement affichée dans le champ, la fenêtre interne suivante sera affichée à sa place (ou la précédente si la fenêtre interne supprimée était la dernière). 

- Lors de la suppression d'une fenêtre interne dans la liste des fenêtres internes, les fenêtres internes suivant la fenêtre supprimée sont renumérotées.

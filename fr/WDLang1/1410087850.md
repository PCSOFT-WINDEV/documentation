
## Procédure WLangage appelée par la fonction CaméraVidéoDémarre



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Procédure WLangage (également nommée "callback") appelée lorsque l'enregistrement de la vidéo est terminé. L'enregistrement de la vidéo a été démarré via la fonction [CaméraVidéoDémarre](../WDLang1/1410087227.md). La fonction [CaméraVidéoArrête](../WDLang1/1410087229.md) doit avoir été appelée pour arrêter l'enregistrement de la vidéo. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Code de clic sur le champ BTN_Démarre_video
ChronoDébut()

TimerSys(TimerSys_Callback, 1s)
PROCÉDURE INTERNE TimerSys_Callback()
	dDurée est une Durée = ChronoValeur()
	LIB_CPT_VIDEO = DuréeVersChaîne(dDurée, "HH:MM:SS")
FIN

CaméraVidéoDémarre(CAM_Caméra, CaméraVidéoDémarre_Callback)


PROCÉDURE INTERNE CaméraVidéoDémarre_Callback(bRéussite est un booléen, CheminVidéo est une chaîne)
	ChronoFin()
	FinTimerSys()
	LIB_CPT_VIDEO = "00:00:00"
	SI bRéussite ALORS
		LIB_CHEMIN_VIDEO = CheminVidéo
		MaFenêtre.Plan = 4
	SINON
		ToastAffiche("Erreur lors de l'enregistrement de la vidéo." + RC + ErreurInfo())
	FIN
FIN
```

```wl
// Arrêt de la vidéo 
CaméraVidéoArrête(CAM_Caméra)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`CaméraVidéoDémarre_Callback(<Réussite> , <Chemin de la vidéo>)`
---

**`<Réussite> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la vidéo a été enregistrée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Dans ce cas, la fonction [ErreurInfo](../WDLang1/3013008.md) permet d'obtenir le détail de l'erreur. 




**`<Chemin de la vidéo> : (Chaîne de caractères)`**

Chemin de la vidéo qui vient d'être enregistrée.

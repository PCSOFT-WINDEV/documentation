
## SansAccent (Fonction)

***En anglais : NoAccent***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Transforme les caractères accentués d'une chaîne de caractères en caractères non accentués.






<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SansAccent("pépère")    // Renvoie "pepere"
SansAccent("où ça")   // Renvoie "ou ca"
SansAccent("")          // Renvoie ""
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = SansAccent(<Chaîne initiale>)`
---

**`<Résultat> : (Chaîne de caractères)`**

Chaîne de caractères dans laquelle tous les caractères accentués ont été remplacés par des caractères non accentués.

**`<Chaîne initiale> : (Chaîne de caractères)`**

Chaîne de caractères initiale.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Fonction SansAccent et UNICODE
<a name="fonction_docparampagetitleshort_unicode_ELTPARAGRAPHE000069"></a>Si la chaîne passée en paramètre est au format ANSI, le résultat sera au format ANSI.

Si la chaîne passée en paramètre est au format UNICODE, le résultat sera au format UNICODE.

**Remarque** : Si le résultat de la fonction **SansAccent** utilisée sur une chaîne ANSI est affecté à une chaîne UNICODE (et inversement), la conversion ANSI/UNICODE sera implicitement réalisée.

Pour plus de détails, consultez [Gestion de l'UNICODE](../WDLang1/3024038.md).

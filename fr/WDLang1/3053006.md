
## MemExiste (Fonction)

***En anglais : MemExist***

<a name="Utilisation"></a>
<a name="description"></a>
Teste l'existence d'une zone de mémoire.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Teste l'existence de la zone mémoire "MaZoneMémoire"
ResExiste = MemExiste("MaZoneMémoire")
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = MemExiste(<Zone mémoire>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la zone mémoire existe, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Zone mémoire> : (Chaîne de caractères)`**

Nom de la zone mémoire, défini par la fonction [MemCrée](../WDLang1/3053005.md).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Pour exister, une zone mémoire doit avoir été créée avec la fonction [MemCrée](../WDLang1/3053005.md). Une zone mémoire peut être détruite avec la fonction [MemSupprimeTout](../WDLang1/3053019.md).

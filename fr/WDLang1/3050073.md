
## MatNbColonne (Fonction)

***En anglais : MatNbColumn***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le nombre de colonnes d'une matrice. Le nombre de colonnes d'une matrice varie dynamiquement. Il correspond au nombre de colonnes initialisées. Le nombre de colonnes peut varier après l'utilisation des fonctions :

- [MatEcrit](../WDLang1/3050077.md),

- [MatRemplit](../WDLang1/3050057.md).





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple
<a class="notetitle" target="_blank" href="$DOC$=1000003050073&name=matnbcolonne_fonction&product=WD">Voir des exemples supplémentaires</a>

```wl
// Nombre de colonnes ?
ResNbColonne = MatNbColonne("MaMatrice")
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = MatNbColonne(<Matrice>)`
---

**`<Résultat> : (Entier)`**

Nombre de colonnes dans la matrice spécifiée.

**`<Matrice> : (Chaîne de caractères)`**

Nom de la matrice à manipuler, défini avec la fonction [MatCrée](../WDLang1/3050058.md).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Existence de la matrice
<a name="existence_matrice_ELTPARAGRAPHE000084"></a>L'existence de la matrice n'est pas vérifiée. Pour vérifier l'existence d'une matrice, utilisez la fonction [MatExiste](../WDLang1/3050076.md).


<a name="NOTE0_2"></a>




### Gestion des erreurs
<a name="gestion_des_erreurs_ELTPARAGRAPHE000096"></a>**Attention** : La fonction **MatNbColonne** ne retourne pas de code d'erreur. Pour savoir si des erreurs ont été générées par le renvoi du nombre de colonnes, utilisez la fonction [MatErreur](../WDLang1/3050044.md). Le libellé de l'erreur peut être connu grâce à la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.

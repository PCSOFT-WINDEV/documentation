


## &lt;Saisie&gt;.SupprimeToutJeton (Fonction)

***En anglais : &lt;Edit&gt;.DeleteAllToken***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime tous les jetons d'un champ de saisie de type "Jetons de texte".
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SAI_Destinataire.SupprimeToutJeton()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Champ de saisie>.SupprimeToutJeton()`
---

**`<Champ de saisie> : (Nom de champ)`**

Nom du champ de saisie de type "Jetons de texte".

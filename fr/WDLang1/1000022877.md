


## &lt;Agenda&gt;.SupprimeRendezVous (Fonction)

***En anglais : &lt;Organizer&gt;.DeleteAppointment***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime un rendez-vous dans un champ Agenda. Si le champ Agenda est lié à une source de données, le rendez-vous est également supprimé dans la source de données. 





![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) En mode Universal Windows 10 App, cette fonction permet uniquement de manipuler les champs Agenda présents dans un état. 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Supprime le rendez-vous sélectionné dans le champ Agenda 
AGD_Agenda1.SupprimeRendezVous()
```
<a name="Exemple2"></a>

```wl
// Supprime le premier rendez-vous ajouté
AGD_Agenda1.SupprimeRendezVous(1)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Champ Agenda>.SupprimeRendezVous([<Indice du rendez-vous>])`
---

**`<Champ Agenda> : (Nom de champ)`**

Nom du champ Agenda à manipuler. Ce champ peut correspondre : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) au champ Agenda d'une fenêtre.

- au champ Agenda d'un état.




**`<Indice du rendez-vous> : (Entier optionnel)`**

Indice du rendez-vous à supprimer. Le premier rendez-vous a pour indice 1. Si ce paramètre n'est pas précisé, le rendez-vous sélectionné sera supprimé.


<a name="SYNTAXE2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 
### Supprimer un rendez-vous identifié par une variable de type RendezVous

`<Champ Agenda>.SupprimeRendezVous([<Rendez-vous à supprimer>])`
---

**`<Champ Agenda> : (Nom de champ)`**

Nom du champ Agenda à manipuler. Ce champ peut correspondre : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) au champ Agenda d'une fenêtre. 

- au champ Agenda d'un état.




**`<Rendez-vous à supprimer> : (Variable de type RendezVous)`**

Nom de la variable de type [RendezVous](../WDLang1/1000019244.md) correspondant au rendez-vous à supprimer. 
Si ce paramètre n'est pas précisé, le rendez-vous sélectionné sera supprimé.


<a name="SYNTAXE3"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 
### Supprimer plusieurs rendez-vous

`<Champ Agenda>.SupprimeRendezVous([<Rendez-vous à supprimer>])`
---

**`<Champ Agenda> : (Nom de champ)`**

Nom du champ Agenda à manipuler. Ce champ peut correspondre : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) au champ Agenda d'une fenêtre. 

- au champ Agenda d'un état.




**`<Rendez-vous à supprimer> : (Tableau de variables de type RendezVous)`**

Nom de la variable de type Tableau de [RendezVous](../WDLang1/1000019244.md) correspondant aux rendez-vous à supprimer. 
Si ce paramètre n'est pas précisé, le rendez-vous sélectionné sera supprimé.




## fNomLong (Fonction)

***En anglais : fLongName***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le nom long d'un fichier ou d'un répertoire.

<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple
<a class="notetitle" target="_blank" href="$DOC$=1000003036033&name=fnomlong_fonction&product=WD">Voir des exemples supplémentaires</a>

```wl
// Récupération du nom long d'un fichier
ResNomLong = fNomLong("C:\MesRép~1\Compte~1.doc")
// fNomLong renvoie "Compte-Rendu Mars 2001.doc"
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = fNomLong(<Chemin du fichier> [, <Options>])`
---

**`<Résultat> : (Chaîne de caractères)`**



- Nom long du fichier ou du répertoire. Seul le nom long du fichier ou du répertoire est renvoyé. 

- Chaîne vide ("") en cas d'erreur (fichier ou répertoire inexistant). Pour connaître le libellé de l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.




**`<Chemin du fichier> : (Chaîne de caractères)`**

Nom et chemin complet (ou relatif) du fichier ou du répertoire (260 caractères maximum). Un chemin de type UNC peut être utilisé.

**`<Options> : (Constante optionnelle de type Entier)`**

Permet de spécifier le format du résultat de la fonction. Par défaut, **fNomLong** renvoie une chaîne de caractères Ansi. 


|   |   |
| --- | --- |
| *fCheminUNICODE* | &lt;Résultat&gt; sera une chaîne de caractères au format Unicode. |


**Remarque** : Le résultat de la fonction **fNomLong** est toujours une chaîne de caractères au format Ansi, la constante *fCheminUNICODE* permet uniquement de stocker le résultat dans une chaîne de caractères Unicode.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Noms longs
<a name="noms_longs_ELTPARAGRAPHE000064"></a>La fonction **fNomLong** renvoie :

- *le nom long du fichier ou du répertoire* si le &lt;Chemin du fichier&gt; est exprimé en noms courts (noms de 8 caractères ou moins, comprenant une tilde "~").

- *le nom du fichier spécifié* si le nom du fichier dans &lt;Chemin du fichier&gt; est :

	- soit exprimé en nom long (nom de plus de 8 caractères),

	- soit composé de 8 caractères ou moins et ne comprend pas une tilde ("~").




- *le nom du répertoire spécifié* si le nom du répertoire dans &lt;Chemin du fichier&gt; est :

	- soit exprimé en nom long (nom de plus de 8 caractères),

	- soit composé de 8 caractères ou moins et ne comprend pas une tilde ("~").

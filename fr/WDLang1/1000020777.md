


## &lt;TimeLine&gt;.AjoutePiste (Fonction)

***En anglais : &lt;TimeLine&gt;.AddTrack***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ajoute une nouvelle piste dans un champ TimeLine.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ajout de 2 pistes 
TL_General.AjoutePiste("Son")
TL_General.AjoutePiste("Image")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Champ TimeLine>.AjoutePiste(<Piste> [, <Image>])`
---

**`<Résultat> : (Entier)`**



- Indice de la piste ajoutée, 

- -1 en cas d'erreur. La fonction [ErreurInfo](../WDLang1/3013008.md) permet d'identifier l'erreur.




**`<Champ TimeLine> : (Nom de champ)`**

Nom du champ TimeLine dans lequel la piste doit être ajoutée.

**`<Piste> : (Chaîne de caractères)`**

Nom de la piste ajoutée. Il est possible d'ajouter une valeur invisible avec la fonction [gValeurMémorisée](../WDLang1/1410088363.md).

**`<Image> : (Chaîne de caractères optionnelle)`**

Chemin et nom de l'image de fond du titre de la piste. Cette image est affichée sous l'entête de la ligne contenant le nom de la piste.

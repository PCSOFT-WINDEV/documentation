<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ferme un fichier externe.

## Syntaxe

`<Résultat> = fFerme(<Fichier manipulé>)`
---

**`<Résultat> : (Entier)`**



- 0 si le fichier externe est fermé, 

- -1 en cas d'erreur. Pour connaître le libellé de l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*. La fonction **fFerme** génère une erreur si le fichier n'est pas ouvert.




**`<Fichier manipulé> : (Entier ou variable de type FichierDisque)`**



- Identifiant du fichier à manipuler, défini avec la fonction [fOuvre](../WDLang1/3036036.md) ou [fCrée](../WDLang1/3036017.md).

- Nom de la variable de type [FichierDisque](../WDLang1/1410088883.md) initialisée avec la fonction [fOuvre](../WDLang1/3036036.md) ou [fCrée](../WDLang1/3036017.md).

## Remarques


### Fermeture d'un fichier ouvert plusieurs fois
<a name="fermeture_fichier_ouvert_plusieurs_fois_ELTPARAGRAPHE000155"></a>Il est possible d'ouvrir un même fichier plusieurs fois en même temps. Chaque version du fichier doit être fermée individuellement par son propre identifiant.

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) 

### Fermeture d'un fichier bloqué
<a name="fermeture_fichier_bloque_ELTPARAGRAPHE000166"></a>Un fichier bloqué sera automatiquement débloqué lors de sa fermeture.

**Rappel** : La fonction [fBloque](../WDLang1/3036030.md) permet de bloquer un fichier externe. La fonction [fDébloque](../WDLang1/3036020.md) permet de débloquer un fichier externe.


## Développer des applications multilingues
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000081"></a>
Une application multilingue est une application qui pourra être diffusée dans plusieurs langues. WINDEV, WEBDEV et WINDEV Mobile prennent en compte les différentes langues de l'application à toutes les étapes du développement d'une application.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Principe de développement d'une application multilingue
<a name="principe_developpement_une_application_multilingue_ELTTEXTE000105"></a>
Pour développer une application multilingue, les étapes sont les suivantes : 

1. [Choix des langues gérées par le projet et l'analyse.](../WDLang1/3054010.md)

2. [Traduction des différents éléments du projet (fenêtre, messages du code, ...) dans les différentes langues du projet.](../WDLang1/3054009.md)

3. [Détermination de la langue du projet par programmation](../WDLang1/3054012.md).

4. [Gestion des alphabets spécifiques dans les fichiers de données](../WDLang1/3054017.md).

5. [Création de l'exécutable et du programme d'installation](../WDLang1/3054006.md).


## MasqueFormate (Fonction)

***En anglais : MaskFormat***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Formate une chaîne de caractères à l'aide d'une variable de type [MasqueDeSaisie](../WDLang1/1000024886.md). La chaîne de caractères est formatée comme pour l'affichage dans un champ de saisie utilisant ce masque.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonMasque est un MasqueDeSaisie = MasqueNuméroDeTéléphone("FR")
Trace(MasqueFormate(MonMasque, "+33467032032"))
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = MasqueFormate(<Masque à appliquer> , <Chaîne>)`
---

**`<Résultat> : (Chaîne de caractères)`**



- Chaîne formatée en fonction du masque spécifié.

- Chaîne vide ("") en cas d'erreur. 




**`<Masque à appliquer> : (Variable de type MasqueDeSaisie)`**

Nom de la variable de type [MasqueDeSaisie](../WDLang1/1000024886.md) à utiliser pour formater la chaîne de caractères. 

**`<Chaîne> : (Chaîne de caractères)`**

Chaîne de caractères à formater. si ce paramètre est invalide, &lt;Résultat&gt; correspond à chaîne vide ("").

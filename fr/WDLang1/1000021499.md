


## TableurRécupèreSélection (Fonction)

***En anglais : SpreadsheetGetSelection***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Récupère la liste des cellules sélectionnées dans un champ Tableur présent dans une fenêtre. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Calcule la somme de toutes les cellules sélectionnées
// (Cellules de type Numérique)
Total est un entier
POUR TOUT sNomCell De TableurRécupèreSélection(TBLR_Tableur)
	Total += TBLR_Tableur[sNomCell]
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = TableurRécupèreSélection(<Champ Tableur>)`
---

**`<Résultat> : (Tableau de chaînes)`**



- Tableau de chaînes contenant les noms des cellules sélectionnées dans le champ Tableur.

- Chaîne vide ("") si aucune cellule n'est sélectionnée. 




**`<Champ Tableur> : (Nom de champ)`**

Nom du champ Tableur à manipuler.


## AgendaChangeZoom (Fonction)

***En anglais : OrganizerChangeZoom***






	<B><B>Avertissement</B></B>

	A partir de la version <B><B>17</B></B>,  cette fonction est conservée par compatibilité. Cette fonction est remplacée par la fonction [AgendaChangeMode](../WDLang1/1000019497.md).



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie le mode d'affichage d'un champ Agenda. La plage temporelle affichée peut être le jour, la semaine ou le mois.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affiche les quatre premiers jours de la semaine
AgendaChangeZoom(AGD_MonAgenda, agzSemaine, 4)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`AgendaChangeZoom(<Champ Agenda> , <Plage d'affichage> [, <Nombre de jours>])`
---

**`<Champ Agenda> : (Nom de champ)`**

Nom du champ Agenda. Ce champ peut correspondre : 

- au champ Agenda d'une fenêtre ou d'une page, 

- au champ Agenda d'un état.




**`<Plage d'affichage> : (Constante de type Entier)`**

Plage de temps à afficher dans le champ Agenda :


|   |   |
| --- | --- |
| *agzJour* | Affiche l'agenda sur une journée. |
| *agzMois* | Affiche l'agenda sur un mois. |
| *agzSemaine* | Affiche l'agenda sur le nombre de jours indiqués dans le paramètre &lt;Nombre de jours&gt;. |
| *agzSemaineSur5* | Affiche l'agenda sur 5 jours (le week-end n'est pas affiché). |





**`<Nombre de jours> : (Entier optionnel)`**

Nombre de jours affichés dans la semaine. Si ce paramètre n'est pas précisé, l'agenda sera affiché sur une semaine complète. Ce paramètre est pris en compte uniquement si le paramètre  &lt;Plage d'affichage&gt; est égal à *agzSemaine*.




## DocRechercheElémentParNom (Fonction)

***En anglais : DocFindElementByName***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Recherche un élément par son nom dans l'ensemble d'un document de type Traitement de texte. Cette fonction permet notamment de rechercher des images dans un document.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Supprime l'image appelée "Image logo" du document
ElémentTrouvé est un docElément <- DocRechercheElémentParNom(TT_MonDocument, "Image Logo")
SI ElémentTrouvé <> Null ALORS
	DocSupprime(ElémentTrouvé)
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = DocRechercheElementParNom(<Document> , <Nom de l'élément>)`
---

**`<Résultat> : (Variable de type docElément)`**



- Variable de type [docElément](../WDLang1/1000022484.md) correspondant à l'élément cherché, 

- Null si aucun élément n'a été trouvé. 




**`<Document> : (Nom de champ ou variable de type Document)`**

Elément dans lequel la recherche doit être effectuée. Cet élément peut correspondre : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) au nom d'un champ Traitement de texte.

- à une variable de type [Document](../WDLang1/1000022461.md).




**`<Nom de l'élément> : (Chaîne de caractères)`**

Nom de l'élément à rechercher.




## &lt;Table&gt;.VersPressePapier (Fonction)

***En anglais : &lt;Table&gt;.ToClipboard***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Copie le contenu d'un champ Table ou Table hiérarchique dans le presse-papiers. Le contenu du champ est copié dans un format texte : il est converti sous forme de chaîne de caractères et cette chaîne est copiée dans le presse-papiers.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
TABLE_TABLE1.VersPressePapier(taSansTitre, ";")
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Champ Table>.VersPressePapier([<Options> [, <Séparateur> [, <Ligne début> [, <Ligne fin>]]]])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le contenu du champ a été mis dans le presse-papiers, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour obtenir plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Champ Table> : (Nom de champ)`**

Nom du champ à manipuler. Ce champ peut correspondre à : 

- un champ Table. 

- un champ Table hiérarchique.




**`<Options> : (Constante optionnelle)`**

Paramètre les informations à prendre en compte dans la chaîne copiée dans le presse-papiers.


|   |   |
| --- | --- |
| *taAvecColonnesInvisibles* | Exporte aussi les colonnes invisibles du champ.<br>![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Les colonnes définies comme non imprimables et non exportables sont exportées. Ces colonnes sont définies : <br><br>- soit par programmation : propriété [VisibleEnExportEtImpression](../Proprietes/1000022320.md) à <u><u><u><u>Faux</u></u></u></u>. <br><br>- soit dans l'onglet "Général" de la fenêtre de description des colonnes : option "Export et impression" à "Jamais".<br><br><br> |
| *taColonneOrdreAffiché* | Exporte les colonnes selon l'ordre des colonnes actuellement affiché dans le champ (et non selon l'ordre des colonnes défini sous l'éditeur). |
| *taLigneSélectionnes* | Exporte uniquement les lignes sélectionnées (par défaut, toutes les lignes sont exportées). |
| *taSansTitre*<br>(valeur par défaut) | Seules les données sont copiées. |
| *taSansTotaux* | N'exporte pas : <br><br>- les lignes de totaux, moyenne et comptage automatique (par défaut, ces lignes sont exportées).<br><br>- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) les lignes de calculs personnalisés (par défaut, ces lignes sont exportées).<br><br><br> |
| *taTitreColonnes* | Le titre des colonnes est inséré. |



**`<Séparateur> : (Chaîne de caractères optionnelle)`**

Séparateur utilisé entre les colonnes. Par défaut, le séparateur utilisé est la tabulation (TAB).

**`<Ligne début> : (Entier optionnel)`**

Numéro de la ligne de début de l'exportation. Si ce paramètre n'est pas précisé, la ligne de début est la première ligne du champ.

**`<Ligne fin> : (Entier optionnel)`**

Numéro de la ligne de fin de l'exportation. Si ce paramètre n'est pas précisé, la ligne de fin est la dernière ligne du champ.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Conditions d'utilisation
<a name="conditions_utilisation_ELTPARAGRAPHE000256"></a>La fonction **&lt;Table&gt;.VersPressePapier** peut être utilisée indifféremment sur : 

- un champ Table ou Table hiérarchique fichier.

- un champ Table ou Table hiérarchique par programmation.

- un champ monosélection ou multisélection.



<a name="NOTE0_2"></a>


### Limite
<a name="limite_ELTPARAGRAPHE000273"></a>Les colonnes fusionnées sont ignorées.

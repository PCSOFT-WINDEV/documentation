
## Tang (Fonction)

***En anglais : Tangent***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la tangente d'un angle.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Tangente de 45
ResTangente = Tang(45)
// Renvoie 1
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = Tang(<Angle>)`
---

**`<Résultat> : (Réel)`**

Tangente de l'angle spécifié.

**`<Angle> : (Entier ou réel)`**

Angle (en degrés) à manipuler. 
Si ce paramètre est égal à 90°, -90°, 270° ou -270°, une erreur du WLangage est générée.

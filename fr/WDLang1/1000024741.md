
## MasqueNuméroDeTéléphone (Fonction)

***En anglais : MaskPhoneNumber***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Récupère le masque de saisie du numéro de téléphone adapté à un pays donné.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Met le masque téléphonique espagnol sur le champ SAI_téléphone
SAI_Téléphone.MasqueSaisie = MasqueNuméroDeTéléphone("ES")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = MasqueNuméroDeTéléphone(<Pays> [, <Option>])`
---

**`<Résultat> : (Variable de type MasqueDeSaisie)`**

Variable de type [MasqueDeSaisie](../WDLang1/1000024886.md) contenant les caractéristiques du masque de saisie du téléphone pour le pays spécifié. Ce résultat peut être affecté directement à la propriété [MasqueSaisie](../Proprietes/2510033.md).  

**`<Pays> : (Chaîne de caractères ou variable de type Pays)`**

Pays pour lequel le masque de saisie du numéro de téléphone doit être récupéré. Ce paramètre peut correspondre à : 

- Le code ISO alphabétique ou numérique du pays voulu.

- Une variable de type [Pays](../WDLang1/1000024921.md). 




**`<Option> : (Constante optionnelle de type Entier)`**

Caractéristiques du numéro de téléphone :


|   |   |
| --- | --- |
| *mtelAvecOuSansPréfixe*<br>(Valeur par défaut) | Numéro de téléphone avec ou sans préfixe international. |
| *mtelForcePréfixe* | Numéro de téléphone avec toujours le préfixe international. <br>Exemple : +33 4 67 03 20 32. |
| *mtelSansPréfixe* | Numéro de téléphone sans préfixe international. <br>Exemple : 04 67 03 20 32. |

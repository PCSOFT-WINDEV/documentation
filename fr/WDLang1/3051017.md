
## MCIOuvre (Fonction)

***En anglais : MCIOpen***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ouvre un fichier multimédia.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ouverture du fichier "C:\MesFichiers\Fichier.AVI"
// dans le champ Image "ImageAVI"
ResExécuteMCI = MCIOuvre("C:\MesFichiers\Fichier.AVI")
```


<a name="Exemple2"></a>



```wl
// Lancer une vidéo "C:\MesFichiers\Fichier.AVI" en mode répétition
// Ouverture de la vidéo
MCIOuvre("C:\MesFichiers\Fichier.AVI", "AVI", MciStyleFille, IMG_Vidéo, MciModeNormal)
// Exécution de la vidéo
MCIExécute("Play AVI repeat")
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = MCIOuvre(<Fichier multimédia> [, <Nom Alias> [, <Type de l'objet contenant> [, <Fenêtre contenant> [, <Mode d'affichage> [, <Paramètres> [, <Nom de la fenêtre de messages>]]]]]])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'ouverture du fichier a été réalisée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour connaître le numéro de l'erreur, utilisez la fonction [MCIErr](../WDLang1/3051011.md). 
	Pour connaître le libellé de l'erreur, utilisez la fonction [MCIMsgErr](../WDLang1/3051021.md) ou la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.
	Le compte-rendu de la commande peut être connu avec la fonction [MCIRécupère](../WDLang1/3051022.md).




**`<Fichier multimédia> : (Chaîne de caractères ou constante)`**

Nom du fichier multimédia à ouvrir (AVI, MP3, ...) ou type du périphérique à ouvrir :


|   |   |
| --- | --- |
| *MciCDAudio* | Initialise le CD Audio |
| *MciDigitalVidéo* | Initialise le lecteur de vidéo numérique |
| *MciDisqueVidéo* | Initialise le lecteur de disque vidéo |
| *MciSéquenceur* | Initialise la table de mixage MIDI |
| *MciWaveAudio* | Initialise le lecteur de .WAV |

**Remarque** : tous les formats gérés par le système sont gérés (c'est-à-dire les CODEC installés sur le poste).

**`<Nom Alias> : (Chaîne de caractères optionnelle)`**

Nom permettant de manipuler le fichier multimédia. Ce nom peut être utilisé dans toutes les fonctions MCI.

**`<Type de l'objet contenant> : (Chaîne de caractères optionnelle)`**

Type de la fenêtre ou du champ (fenêtre fille, popup, ...) dans lequel sera affiché le fichier multimédia.


|   |   |
| --- | --- |
| *MciStyleFille* | Fenêtre fille de la fenêtre en cours |
| *MciStylePopup* | Fenêtre affichée en mode Popup |
| *MciStyleStandard* | Fenêtre standard |



**`<Fenêtre contenant> : (Entier ou chaîne de caractères optionnel)`**

Nom ou identifiant (Handle) de la fenêtre ou du champ où sera affiché le fichier. 
**Remarque** : Le handle peut être obtenu avec la fonction [Handle](../WDLang1/3015005.md).

**`<Mode d'affichage> : (Constante optionnelle)`**

Mode d'affichage du fichier dans la fenêtre :


|   |   |
| --- | --- |
| *MciModeCentré* | Image centrée dans la fenêtre |
| *MciModeEtiré* | Image étirée pour occuper toute la surface |
| *MciModeHomothétiqueCentré*<br>(Valeur par défaut) | Image étirée proportionnellement |
| *MciModeNormal* | Affichage à 100% |



**`<Paramètres> : (Chaîne de caractères optionnelle)`**

Paramètres complémentaires si nécessaire. Ces paramètres peuvent être connus en consultant la documentation de la fonction MCI Windows "Open".
Il est possible d'ajouter la constante *MciAttente* à cette chaîne. Cette constante permet d'indiquer que le programme doit attendre la fin de la commande MCI pour exécuter la suite de l'application. 

**`<Nom de la fenêtre de messages> : (Chaîne de caractères optionnelle)`**

Nom de la fenêtre permettant d'intercepter et de gérer les différents messages envoyés par MMSYSTEM.DLL. Ces messages sont du type MM_xxxxxx et sont récupérables grâce à la fonction [Evénement](../WDLang1/3015011.md). Il est par exemple possible d'obtenir des messages concernant la gestion du lecteur de CD ROM ou du Joystick.




## &lt;Thread&gt;.DemandeArrêt (Fonction)

***En anglais : &lt;Thread&gt;.RequestStop***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Envoie une demande d'arrêt à un thread.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple

<a name="Exemple2"></a>

```wl
// Construit un thread
t est un Thread(Proc, ("Premier paramètre", 2))
// Déclenche l'exécution du thread
ThreadExécute(t)
...
t.DemandeArrêt()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE2"></a>

### Demander l'arrêt d'un thread identifié par une variable de type Thread

`<Thread>.DemandeArrêt()`
---

**`<Thread> : (Variable de type Thread)`**

Nom de la variable de type [Thread](../WDLang1/1000024879.md) correspondant au thread à arrêter.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Le thread ne sera pas arrêté automatiquement. C'est au code du thread de vérifier si l'arrêt est demandé grâce à la fonction [ThreadArrêtDemandé](../WDLang1/1000021219.md) et de se terminer proprement.

- La fonction est non bloquante et n'attend pas l'arrêt du thread. Il est nécessaire d'utiliser la fonction [&lt;Variable Thread&gt;.Attend](../WDLang1/1000024880.md) pour attendre l'arrêt effectif du thread.

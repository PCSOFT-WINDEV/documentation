
## TableActiveFiltre (Fonction)

***En anglais : TableEnableFilter***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ajoute un filtre utilisateur sur une colonne d'un champ Table ou Table hiérarchique. Ce filtre pourra être désactivé par l'utilisateur ou grâce à la fonction [TableDésactiveFiltre](../WDLang1/1000017150.md).

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) **Remarque** : Lorsqu'un filtre utilisateur est défini sur un champ Table ou Table hiérarchique, l'icône suivante est affichée dans l'entête de la colonne :<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=TableActiveFiltre_entonnoir.gif)


L'utilisateur peut supprimer le filtre : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) grâce à l'option "Supprimer le filtre" du menu contextuel de l'icône "entonnoir". Les filtres utilisateur font partie des FAA (Fonctionnalités Automatiques de vos Applications) disponibles sur les champs Table et Table hiérarchique. Pour plus de détails, consultez [FAA sur les colonnes d'un champ Table](../WDChamp/9500107.md). 





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Filtre la colonne Société pour afficher les sociétés commençant par "Fl"
TableActiveFiltre(TABLE_Client.COL_Societe, filtreCommencePar, "Fl")
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`TableActiveFiltre(<Champ Table.Colonne> , <Type du filtre> , <Valeur du filtre>)`
---

**`<Champ Table.Colonne> : (Chaîne de caractères)`**

Nom de la colonne où le filtre utilisateur doit être activé. Ce nom est de la forme : 

```txt
<Champ Table>.<Champ Colonne>
```


&lt;Champ Table&gt; peut correspondre au nom d'un champ Table ou Table hiérarchique. 

Exemple : 
```txt
TABLE_TableClient.COL_NomSociété
```


**`<Type du filtre> : (Constante de type Entier)`**

Type de filtre à appliquer. Ce filtre peut être choisi parmi les valeurs suivantes :


|   |   |
| --- | --- |
| *filtreCommencePar* | Affiche les lignes dont la valeur de la colonne spécifiée "Commence par" les caractères indiqués dans la &lt;Valeur du filtre&gt;. |
| *filtreContient* | Affiche les lignes dont la valeur de la colonne spécifiée "Contient" les caractères indiqués dans la &lt;Valeur du filtre&gt;. |
| *filtreDifférent* | Affiche les lignes dont la valeur de la colonne spécifiée "Est différente" des caractères indiqués dans la &lt;Valeur du filtre&gt;. |
| *filtreEgal* | Affiche les lignes dont la valeur de la colonne spécifiée "Est égale" aux caractères indiqués dans la &lt;Valeur du filtre&gt;. |
| *filtreInférieur* | Affiche les lignes dont la valeur de la colonne spécifiée "Est inférieure" aux caractères indiqués dans la &lt;Valeur du filtre&gt;. |
| *filtreInférieurOuEgal* | Affiche les lignes dont la valeur de la colonne spécifiée "Est inférieure ou égale" aux caractères indiqués dans la &lt;Valeur du filtre&gt;. |
| *filtreNeCommencePasPar* | Affiche les lignes dont la valeur de la colonne spécifiée "Ne commence pas par" les caractères indiqués dans la &lt;Valeur du filtre&gt;. |
| *filtreNeContientPas* | Affiche les lignes dont la valeur de la colonne spécifiée "Ne contient pas" les caractères indiqués dans la &lt;Valeur du filtre&gt;. |
| *filtreNeTerminePasPar* | Affiche les lignes dont la valeur de la colonne spécifiée "Ne se termine pas par" les caractères indiqués dans la &lt;Valeur du filtre&gt;. |
| *filtreSupérieur* | Affiche les lignes dont la valeur de la colonne spécifiée "Est supérieure" aux caractères indiqués dans la &lt;Valeur du filtre&gt;. |
| *filtreSupérieurOuEgal* | Affiche les lignes dont la valeur de la colonne spécifiée "Est supérieure ou égale" aux caractères indiqués dans la &lt;Valeur du filtre&gt;. |
| *filtreTerminePar* | Affiche les lignes dont la valeur de la colonne spécifiée "Se termine par" les caractères indiqués dans la &lt;Valeur du filtre&gt;. |



**`<Valeur du filtre> : (Type correspondant à la colonne filtrée)`**

Valeur à donner au filtre. Si ce paramètre correspond à une chaîne vide (""), le filtre est désactivé.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Conditions d'utilisation
<a name="conditions_utilisation_ELTPARAGRAPHE000148"></a>Cette fonction est disponible sur : 

- un champ Table ou Table hiérarchique fichier.

- un champ Table ou Table hiérarchique par programmation.









<a name="NOTE0_2"></a>




### Avantage des filtres utilisateur
<a name="avantage_des_filtres_utilisateur_ELTPARAGRAPHE000178"></a>Les filtres utilisateur sont modifiables par l'utilisateur final (à la différence d'un filtre mis en place par la fonction [HFiltre](../WDLang4/3044100.md) ou par la propriété [Filtre](../Proprietes/2510042.md)).


<a name="NOTE0_3"></a>




### Cas particuliers
<a name="cas_particuliers_ELTPARAGRAPHE000193"></a>

- Si un filtre utilisateur est déjà activé sur la colonne spécifiée, la fonction **TableActiveFiltre** annule le filtre existant et le remplace par le nouveau filtre.

- La propriété [Filtre](../Proprietes/2510042.md) ne permet pas de connaître le filtre utilisateur utilisé.

- La fonction [TableColonnesFiltrées](../WDLang1/1000020517.md) permet de connaître les filtres utilisateurs (définis directement par l'utilisateur ou par la fonction **TableActiveFiltre**).




## &lt;Tableur&gt;.AjouteNom (Fonction)

***En anglais : &lt;Spreadsheet&gt;.AddName***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Spécifie un nom pour identifier une cellule d'un champ Tableur.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Crée une formule qui référence A1 et A2 avec des noms plus clairs
TBLR_TABLEUR1.AjouteNom("PrixHT", "$A$1")
TBLR_TABLEUR1.AjouteNom("TVA", "$A$2")
TBLR_TABLEUR1.AjouteFormule("A3", "PrixHT*TVA")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Champ Tableur>.AjouteNom(<Nom de la cellule> , <Cellule> [, <Portée> [, <Commentaire>]])`
---

**`<Champ Tableur> : (Nom de champ)`**

Nom du champ Tableur à manipuler.

**`<Nom de la cellule> : (Chaîne de caractères)`**

Nom à donner à la cellule.

Attention : 

- Ce paramètre ne doit pas correspondre à un nom de cellule (par exemple "A1") ou à un nom de fonction standard (par exemple "SOMMME").

- Ce paramètre ne doit pas contenir d'espaces. 

- Si le nom spécifié est invalide ou existe déjà, la fonction **&lt;Tableur&gt;.AjouteNom** échoue : la variable [ErreurDétectée](../WDLang1/3087001.md) est à <u><u><u><u>Vrai</u></u></u></u> et la fonction [ErreurInfo](../WDLang1/3013008.md) permet de connaître la cause de l'erreur.




**`<Cellule> : (Chaîne de caractères)`**

Cellule à nommer. 
Attention : Il est nécessaire d'utiliser des coordonnées absolues pour désigner la cellule (par exemple : "$A$1"). 
En effet, dans un nom de cellule, les coordonnées sont relatives. L'utilisation de la notation "A1", "A2", ... peut entraîner des erreurs de références circulaires.

**`<Portée> : (Entier optionnel)`**

Numéro de la feuille dans laquelle le nom est utilisable. Si ce paramètre est omis ou vaut 0, le nom est utilisable dans toutes les feuilles.

**`<Commentaire> : (Chaîne de caractères optionnelle)`**

Commentaire associé au nom de la cellule (255 caractères maximum). 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Les noms ne sont pas sensibles à la casse (majuscules/minuscules).


## TâcheParallèleParamètre (Fonction)

***En anglais : ParallelTaskConfigure***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie les paramètres des tâches parallèles.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Limite à 8 le nombre maximal de tâches simultanées
TâcheParallèleParamètre(tppNombreMaximalThread, 8)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître la valeur d'un paramètre

`<Résultat> = TâcheParallèleParamètre(<Paramètre>)`
---

**`<Résultat> : (Type correspondant au paramètre recherché)`**

Valeur du paramètre.

**`<Paramètre> : (Constante de type Entier)`**

Paramètre dont la valeur est recherchée : 


|   |   |
| --- | --- |
| *tppNombreMaximalThread* | Nombre maximal de threads pouvant être exécutés en même temps.<br><br>![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Si le nombre de threads spécifié est inférieur au nombre minimum de threads du spool, la valeur spécifiée sera ignorée. Le nombre de threads minimum du spool dépend de l'implémentation de la JVM. |




<a name="SYNTAXE2"></a>

### Modifier la valeur d'un paramètre

`<Résultat> = TâcheParallèleParamètre(<Paramètre> , <Valeur>)`
---

**`<Résultat> : (Type correspondant au paramètre recherché)`**

Valeur du paramètre avant la modification.

**`<Paramètre> : (Constante de type Entier)`**

Paramètre dont la valeur doit être modifiée : 


|   |   |
| --- | --- |
| *tppNombreMaximalThread* | Nombre maximal de threads pouvant être exécutés en même temps. |



**`<Valeur> : (Type correspondant au paramètre)`**

Nouvelle valeur du paramètre.

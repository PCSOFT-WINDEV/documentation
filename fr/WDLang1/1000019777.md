
## dbgVérifieDifférence (Fonction)

***En anglais : dbgCheckDifferent***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Vérifie si les expressions spécifiées sont différentes en programmation défensive et en mode test automatique.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// La fonction ne devrait pas renvoyer 0 
dbgVérifieDifférence(FonctionCalcul(), 0)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`dbgVérifieDifférence(<Expression 1> , <Expression 2> [, <Commentaires> [, <Paramètre 1> [... [, <Paramètre N>]]]])`
---

**`<Expression 1> : (Chaîne de caractères)`**

Expression quelconque. 

**`<Expression 2> : (Chaîne de caractères)`**

Expression quelconque.

**`<Commentaires> : (Chaîne de caractères)`**

Chaîne de caractères au format défini par la fonction [ChaîneConstruit](../WDLang1/3024006.md). 

**`<Paramètre 1> : (Tout type)`**

Premier paramètre attendu par le paramètre &lt;Commentaires&gt;.



**`<Paramètre N> : (Tout type)`**

Dernier paramètre attendu par le paramètre &lt;Commentaires&gt;.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Fonctionnement
<a name="fonctionnement_ELTPARAGRAPHE000081"></a>Lorsque les expressions sont égales : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) **en mode test automatique**, une erreur est ajoutée dans le compte-rendu du test (équivalent de la fonction [TestEcritRésultat](../WDLang1/1000013001.md)).

- **en mode test manuel**, une assertion est affichée (équivalent de la fonction [dbgAssertion](../WDLang1/3014022.md)).

- **en mode exécution**, une assertion est affichée si les assertions ont été activées (équivalent de la fonction [dbgAssertion](../WDLang1/3014022.md)).




En exécution d'une application, si les assertions ne sont pas activées, l'expression n'est pas évaluée.

La fonction [dbgSurErreur](../WDLang1/1000020722.md) permet de lancer automatiquement une procédure WLangage. Cette procédure  permet de récupérer le commentaire lié à l'erreur.

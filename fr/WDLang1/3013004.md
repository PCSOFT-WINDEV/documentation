
## Bip (Fonction)

***En anglais : Beep***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Émet un bip sonore. Ce son correspond au son système par défaut de Windows (réglable dans le panneau de configuration ou en choisissant un thème de bureau).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI HTrouve() = Faux ALORS
	Message("Non trouvé")
	Bip()
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`Bip()`
---



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Si aucune carte son n'est présente (ou installée), le haut-parleur de la carte mère est utilisé pour émettre un bip.

- La fonction n'est pas bloquante : dès que le son est lancé, la suite du programme s'exécute. Le programme n'attend pas la fin du son.

- L'utilisateur peut désactiver l'émission de tout son système via le panneau de configuration.

- Si plusieurs appels à la fonction **Bip** sont effectués à la suite dans le même traitement, seul un seul son sera entendu.

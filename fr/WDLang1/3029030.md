


## dSymétrieVerticale (Fonction)

***En anglais : dVerticalSymmetry***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Retourne l'image par rapport à un axe vertical (symétrie par rapport à un axe vertical). Cette image peut correspondre à : 

- un champ Image, 

- une variable de type [Image](../WDLang1/1000019650.md).
	

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [WDPic](../WDLang1/1000023503.md) (sur le calque d'arrière-plan),

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [picCalque](../WDLang1/1000024604.md).


![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=dSymetrieVerticale.gif)


![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Les dessins](../WDLang1/3029035.md). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Symétrie verticale du champ IMG_Image1
MonRésultat est un booléen
MonRésultat = dSymétrieVerticale(IMG_Image1)
SI MonRésultat = Faux ALORS Erreur("L'opération a échoué")
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = dSymétrieVerticale(<Image>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la symétrie a été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Image> : (Nom de champ ou variable de type Image, WDPic ou picCalque)`**

Image à manipuler. Cette image peut correspondre : 

- au nom d'un champ Image. 

- au nom d'une variable de type [Image](../WDLang1/1000019650.md).
	

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) au nom d'une variable de type [WDPic](../WDLang1/1000023503.md). Seul le calque d'arrière-plan sera manipulé. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) au nom d'une variable de type [picCalque](../WDLang1/1000024604.md).




La symétrie est réalisée par rapport à l'axe vertical médian du champ ou de l'image. L'image est directement modifiée.

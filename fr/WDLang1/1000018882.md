


## AgendaSupprimeTout (Fonction)

***En anglais : OrganizerDeleteAll***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime tous les rendez-vous d'un champ Agenda.

**Remarques** : 

- Dans le cas d'un champ Agenda par programmation, tous les rendez-vous sont supprimés. 

- Dans le cas d'un champ Agenda fichier ou sur variable, les rendez-vous sont uniquement supprimés dans le champ Agenda. Les rendez-vous sont conservés dans la source de données liée au champ. 








![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) En mode Universal Windows 10 App, cette fonction permet uniquement de manipuler les champs Agenda présents dans un état. 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Suppression des rendez-vous du champ Agenda AGD_MonAgenda
AgendaSupprimeTout(AGD_MonAgenda)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`AgendaSupprimeTout(<Champ Agenda>)`
---

**`<Champ Agenda> : (Nom de champ)`**

Nom du champ Agenda à manipuler. Ce champ peut correspondre : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) au champ Agenda d'une fenêtre.

- au champ Agenda d'un état.

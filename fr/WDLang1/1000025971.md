
## PhraseVersBuffer (Fonction)

***En anglais : PhraseToBuffer***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le buffer encodé dans une phrase mnémotechnique BIP39.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Décode la phrase mnémotechnique pour retrouver le buffer

sPhraseFR est une chaîne = "abreuver barrage espèce associer pointe véhicule"
Trace(BufferVersHexa(PhraseVersBuffer(sPhraseFR)))
// contient 01 23 45 67 89 AB CD EF

sPhraseEN est une chaîne = "abuse boss fly battle rubber waste"
Trace(BufferVersHexa(PhraseVersBuffer(sPhraseEN)))
// contient 01 23 45 67 89 AB CD EF

sPhraseES est une chaîne = "abuelo azul filtro arte prueba vengar"
Trace(BufferVersHexa(PhraseVersBuffer(sPhraseES)))
// contient 01 23 45 67 89 AB CD EF
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = PhraseVersBuffer(<Phrase> [, <Dictionnaire>])`
---

**`<Résultat> : (Buffer)`**

Buffer encodé dans la phrase.

**`<Phrase> : (Chaîne de caractères)`**

Phrase générée par un algorithme BIP39. Les mots dans cette phrase doivent être séparés par des espaces. Une phrase BIP39 doit contenir un nombre de mots multiple de 3 et inférieur ou égal à 24.

**`<Dictionnaire> : (Chaîne de caractères ou constante de type Entier optionnelle)`**

Chemin du fichier correspondant au dictionnaire BIP39 à utiliser. Ce dictionnaire permet d'identifier les mots de la phrase et de connaître leur indice. 
Si ce paramètre n'est pas spécifié, la fonction déterminera seule le dictionnaire à utiliser parmi les dictionnaires internes au produit (français, anglais ou espagnol).

<a name="NOTE0_1"></a>

## Remarques


### Dictionnaires BIP39
<a name="dictionnaires_bip39_ELTPARAGRAPHE000048"></a>Les dictionnaires BIP39 ne sont pas équivalents à des dictionnaires de langues classique. En effet, leur construction suit des règles précises. Il est conseillé d'utiliser les dictionnaires BIP39 internes au produit (français, anglais, espagnol).


## DDEErreur (Fonction)

***En anglais : DDEError***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Retourne le compte-rendu d'exécution d'une fonction de gestion du dialogue DDE.

En effet, la plupart des fonctions DDE renvoient <u><u><u>Vrai</u></u></u> en cas de succès et <u><u><u>Faux</u></u></u> en cas de problème. La fonction **DDEErreur** permet alors d'identifier l'erreur survenue. Il est également possible d'utiliser la fonction [ErreurInfo](../WDLang1/3013008.md) pour connaître le code de l'erreur.

**Attention** : Les fonctions **DDEErreur** et [ErreurInfo](../WDLang1/3013008.md) ne renvoient pas les mêmes codes d'erreur en cas d'erreur DDE. En effet, **DDEErreur** renvoie le code d'erreur Windows et [ErreurInfo](../WDLang1/3013008.md) renvoie un code d'erreur spécifique à WINDEV ou à WEBDEV.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connexion
MaConnexion = DDEConnecte("Excel", Sujet)
// Erreur rencontrée ?
SI MaConnexion = 0 ALORS
	SELON DDEErreur()
		CAS 1 : Erreur("Erreur d'initialisation")
		CAS 6 : Erreur("Le destinataire est inconnu")
	FIN
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = DDEErreur([<Identifiant connexion>])`
---

**`<Résultat> : (Entier)`**

Code de la dernière erreur DDE survenue sur la connexion DDE spécifiée.


|   |   |
| --- | --- |
| -1 | Erreur de syntaxe (client = \*, sujet = \*, etc.) |
| 1 | Erreur d'initialisation |
| 2 | Le programme est déjà terminé |
| 3 | Le programme en cours d'exécution est lancé alors qu'il est toujours enregistré auprès de WINDEV/WEBDEV (le programme n'a pas appelé la fonction [DDEDéconnecte](../WDLang1/3028003.md) lors d'une précédente exécution) |
| 4 | Trop de programmes sont enregistrés auprès de WDDE.DLL (50 programmes maximum) |
| 5 | Trop de messages sont en mémoire (400 messages maximum) |
| 6 | Destinataire inconnu |
| 7 | Aucun message n'est à récupérer : la file des messages est vide pour le sujet demandé |
| 8 | Pas assez de mémoire pour lancer le serveur |
| 9 | Programme appelé inconnu (chemin incorrect) |
| 10 | Programme appelé déjà lancé |
| 11 | Erreur de lancement du programme |
| 12 | Erreur lors d'une allocation mémoire |
| 13 | Erreur lors de l'initialisation DDE |
| 14 | Erreur lors de la connexion à l'application (serveur) |
| 15 | Erreur lors de la transaction DDE |
| 16 | Erreur lors de la récupération d'une donnée DDE |



**`<Identifiant connexion> : (Entier optionnel)`**

Identifiant de la connexion renvoyé par la fonction [DDEConnecte](../WDLang1/3028015.md). Si ce paramètre n'est pas précisé, **DDEErreur** utilise la dernière connexion manipulée.


## FAAMenuContextuelVersChaîne (Fonction)

***En anglais : AAFContextMenuToString***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la liste des options du menu contextuel effectivement affiché par le champ. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
sTout est chaîne = FAAMenuContextuelVersChaîne(TABLE_SansNom1)	

POUR TOUTE CHAÎNE sOption DE sTout SEPAREE PAR RC
	sLibellé est chaîne = ExtraitChaîne(sOption, 1, TAB)
	nID est un entier = ExtraitChaîne(sOption, 2, TAB)
	sPos est une chaîne = ExtraitChaîne(sOption, 3, TAB)
	Trace(sLibellé + " - " + nID + " - " + sPos)
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = FAAMenuContextuelVersChaîne(<Champ>)`
---

**`<Résultat> : (Chaîne de caractères)`**

Liste des options du menu contextuel du champ. Cette liste est de la forme : 

```txt
<Libellé 1> + TAB + <Identifiant 1> + TAB + <Position 1> + RC + ... 
+ RC + <Libellé N> + TAB + <Identifiant N> + TAB + <Position N>
```
où : 

- Libellé : Libellé de l'option. 

- Identifiant : Identifiant de l'option. 
	Remarque : Cet identifiant peut être utilisé avec la fonction [FAAExécute](../WDLang1/1000022099.md) pour exécuter directement l'option.

- Position : Indice de l'option dans le menu. Si l'option est dans un sous-menu, cet indice est de la forme : 
	
	```txt
	<Numéro de l'option principale>.<Numéro de l'option du sous menu>
	```
Par exemple, "4.5" pour l'option 5 du sous-menu 4. 




**`<Champ> : (Nom de champ)`**

Nom du champ manipulé. 







<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Toutes les options du menu contextuel sont prises en compte : 

	- Options de menu système. 

	- Options de menu affichées spécifiques aux FAA (Fonctionnalités Automatiques de vos Applications). 

	- Options de menu contextuel personnalisées. 




- Seules les options cliquables sont listées. 

- Si le champ peut afficher plusieurs menus, le dernier menu affiché est renvoyé. 
	Par exemple, sur un champ Table : 

	- Par défaut, les options du menu du champ Table sont listées. 

	- Si le menu contextuel d'une colonne est affiché, la fonction renverra les options du menu contextuel de la colonne.

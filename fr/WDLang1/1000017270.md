


## MenuAjouteSéparateur (Fonction)

***En anglais : MenuAddSeparator***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ajoute un nouveau séparateur dans un menu (menu déroulant ou contextuel). Pour insérer un séparateur à un emplacement spécifique, utilisez la fonction [MenuInsèreSéparateur](../WDLang1/1000017271.md).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 
```wl
// Ajoute un nouveau menu dans le menu MENU_MonMenu
MenuAjouteMenu(MENU_MonMenu, "OPT_Planning", "Planning")
// Ajoute une option dans le nouveau menu OPT_Planning
MenuAjouteOption("OPT_Planning", "OPT_Créer", "Créer", Créer_Planning)
// Associe une image à l'option de menu OPT_Créer
{"OPT_Créer", indChamp}..Image = "planning.png"
// Ajoute un séparateur
MenuAjouteSéparateur("OPT_Planning")
// Ajoute une nouvelle option (OPT_Voir)
MenuAjouteOption("OPT_Planning", "OPT_Voir", "Voir", Voir_planning)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = MenuAjouteSéparateur(<Menu d'origine>)`
---

**`<Résultat> : (Entier)`**

Position du séparateur dans le menu (déroulant ou contextuel). Si l'ajout du séparateur n'a pas été effectué, une erreur fatale est affichée.

**`<Menu d'origine> : (Nom de menu)`**

Nom du menu dans lequel le séparateur doit être ajouté. Le séparateur sera ajouté après la dernière option de ce menu. 
Remarque : Ce nom de menu peut correspondre à un menu contextuel.

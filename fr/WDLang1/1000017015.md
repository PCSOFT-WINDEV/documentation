


## TestAjouteItération (Fonction)

***En anglais : TestAddIteration***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ajoute une itération dans le test en cours et donne les valeurs des paramètres du test pour cette itération.

**Remarque** : Cette fonction peut être utilisée uniquement dans le code d'un test automatique. Elle doit être utilisée dans l'événement "Avant le test".


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Le test en cours fait une addition , il est décrit avec 3 paramètres p1,p2 et res
// Le code du scénario de test est res=p1+p2
// Dans le code d'avant test, on ajoute les données de test
TestAjouteItération(1, 0, 1)
TestAjouteIteration(2, 2, 4)
TestAjouteItération("a", "b", "ab")
// Ainsi, le scénario de test sera exécuté 3 fois, et vérifiera que
// 1 + 0 = 1
// 2 + 2 = 4
// "a" + "b "= "ab"
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`TestAjouteItération(<Paramètre 1> [, <Paramètre 2> [... [, <Paramètre N>]]])`
---

**`<Paramètre 1> : (Variant)`**

Premier paramètre à passer au test. Ce paramètre a été défini dans l'événement "Scénario du test".

**`<Paramètre 2> : (Variant optionnel)`**

Second paramètre à passer au test. Ce paramètre a été défini dans l'événement "Scénario du test".



**`<Paramètre N> : (Variant optionnel)`**

Nième paramètre à passer au test. Ce paramètre a été défini dans l'événement "Scénario du test".

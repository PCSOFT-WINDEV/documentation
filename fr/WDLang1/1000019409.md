
## Factorielle (Fonction)

***En anglais : Factorial***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la factorielle d'un nombre entier.

En mathématiques, la factorielle d'un entier naturel n (notée n!) est le produit des nombres entiers strictement positifs inférieurs ou égaux à n. 




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
Res = Factorielle(5) // Renvoie 120
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = Factorielle(<n>)`
---

**`<Résultat> : (Entier ou réel)`**

Factorielle du nombre recherché. 

**`<n> : (Entier)`**

Nombre entier dont la factorielle doit être calculée.

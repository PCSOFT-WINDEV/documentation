
## Icône (Type de variable)

***En anglais : Icon***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **Icône** permet de définir toutes les caractéristiques avancées d'une icône. Les caractéristiques de cette icône peuvent être définies et modifiées à l'aide de différentes propriétés WLangage.

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Chargement d'une icône
MonIcône est une Icône = "Sigle.ico"
// Modification de la première image de l'icône
MonIcône.Image[1].Image = "C:\MesImages\sigle48.bmp"
MonIcône.CouleurTransparence[1] = MagentaClair
MonIcône.Format[1] = icoFormatBMP
// Sauvegarde de l'icône modifiée
dSauveImageICO(MonIcône, "Sigle.ico")
```

<a name="NOTE0"></a>

## Remarques
<a name="NOTE0_1"></a>


### Propriétés spécifiques à la description des variables de type Icône
<a name="proprietes_specifiques_description_des_variables_type_icone_ELTPARAGRAPHE000027"></a>Les propriétés suivantes peuvent être utilisées pour manipuler une icône :

| **Nom de la propriété** | **Type manipulé** | **Effet** |
| --- | --- | --- |
| BitParPixel | Tableau d'entiers | Tableau contenant pour chaque image de l'icône le nombre de bits par pixel de l'image. |
| CouleurTransparence | Tableau de [Couleur](../WDLang1/1000019661.md) | Tableau contenant la couleur de transparence de chaque image de l'icône. |
| Format | Tableau | Tableau contenant le format de chaque image de l'icône. Les formats supportés sont : <br><br>- BMP : la constante correspondante est *icoFormatBMP*. <br><br>- PNG : la constante correspondante est *icoFormatPNG*.<br><br><br> |
| Image | Tableau de [Image](../WDLang1/1000019650.md) | Tableau contenant les images de chaque format de l'icône. <br><br>Les dimensions des images fournies seront conservées. Le système choisira par défaut l'image la plus adaptée en fonction de ses dimensions et de son nombre de couleurs (il prendra la première image trouvée si deux images ont les mêmes dimensions et nombre de couleurs). |
| Palette | Tableau de [Palette](../WDLang1/1000021347.md) | Palette de couleurs de chaque image de l'icône. |
| Valeur | Image (mémo) | Valeur de l'icône. Cette propriété permet de charger une icône à partir d'un buffer, d'un fichier ou d'un mémo. |




## Fonctions de vérification de l'orthographe
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions permettant de gérer la correction orthographique sont les suivantes :



|   |   |
| --- | --- |
| [OrthoAjouteDictionnaire](../WDLang1/1000018921.md) | Ajoute un mot au dictionnaire orthographique de OpenOffice : <br><br>- soit pour l'instance de l'application en cours. <br><br>- soit pour toutes les applications.<br><br><br> |
| [OrthoDisponible](../WDLang1/1000019019.md) | Indique si un dictionnaire est disponible pour la vérification orthographique (vérification effectuée dans les champs utilisant la correction intégrée ou vérification par les fonctions WLangage). |
| [OrthoSuggère](../WDLang1/1000018920.md) | Renvoie la liste des mots possibles présents dans le dictionnaire de OpenOffice pouvant corriger un mot mal orthographié. |
| [OrthoVérifie](../WDLang1/1000018923.md) | Indique si un mot est correctement orthographié (c'est-à-dire s'il est connu du dictionnaire OpenOffice installé sur le poste). |


## TélémétrieActive (Fonction)

***En anglais : TelemetryEnable***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Active manuellement la télémétrie. Cette fonction doit être utilisée si l'activation manuelle de la télémétrie a été spécifiée dans les options de télémétrie. 

**Remarque** : Les caractéristiques de la télémétrie sont configurées dans la description de la plateforme. La description de la plateforme est accessible : 

- Depuis la fenêtre de description du projet : dans l'onglet "Projet", sélectionnez l'option "Description de la configuration courante".

- Depuis le ruban de WINDEV ou WINDEV Mobile : sous le volet "Projet", dans le groupe "Configuration de projet", option "Configuration courante". 





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
TélémétrieActive()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Activation manuelle de la télémétrie

`TélémétrieActive()`
---


<a name="SYNTAXE2"></a>

### Activation manuelle de la télémétrie en modifiant le nom du serveur

`TélémétrieActive(<Serveur>)`
---

**`<Serveur> : (Chaîne de caractères)`**

Serveur de télémétrie à utiliser. Ce paramètre peut correspondre : 

- au nom du serveur de télémétrie, 

- à l'adresse IP du serveur de télémétrie (par exemple "192.54.12.8").




## Fonctions MDI
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions MDI sont les suivantes :



|   |   |
| --- | --- |
| [MDIActive](../WDLang1/3052002.md) | Identifie ou place une fenêtre fille MDI en avant-plan. |
| [MDIEnumèreFille](../WDLang1/3052001.md) | Renvoie l'alias de la fenêtre fille MDI spécifiée. |
| [MDIMenuFenêtre](../WDLang1/3052004.md) | Modifie l'agencement des fenêtres filles MDI. |
| [MDIMère](../WDLang1/3052006.md) | Identifie le nom de la fenêtre mère MDI. |
| [MDIOuvre](../WDLang1/3052005.md) | Ouvre et active une fenêtre fille MDI. |

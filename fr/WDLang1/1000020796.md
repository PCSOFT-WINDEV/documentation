
## SysIconeBadge (Fonction)

***En anglais : SysIconBadge***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie ou modifie la valeur affichée dans le badge (ou pastille) associé à l'icône d'une application iOS ou Universal Windows 10 App.

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=sysIconeBadge.gif)



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Incrémente le badge pour notifier l'utilisateur de l'arrivée d'un email
SysIconeBadge(SysIconeBadge() + 1)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Modifier la valeur affichée dans le badge

`SysIconeBadge(<Valeur>)`
---

**`<Valeur> : (Entier)`**

Nouvelle valeur à afficher dans le badge associé à l'icône de l'application. Si cette valeur vaut 0, le badge n'est pas affiché. 


<a name="SYNTAXE2"></a>

### Connaître la valeur affichée dans le badge

`<Résultat> = SysIconeBadge()`
---

**`<Résultat> : (Entier)`**

Valeur actuelle affichée dans le badge de l'application. Si cette valeur vaut 0, le badge n'est pas affiché.


## SysEtatCarteStockage (Fonction)

***En anglais : SysStatusStorageCard***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie l'état de disponibilité de la carte de stockage de l'appareil (zone de stockage externe primaire). Cette fonction doit être utilisée avant les opérations de lecture/écriture sur la carte de stockage pour s'assurer que celle-ci est bien disponible. 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
nEtat est un entier
nEtat = SysEtatCarteStockage()
SI nEtat = sysCarteDisponible ALORS
	sRep est une chaîne 
	sRep = SysRepCarteStockage()
	// Création d'un fichier sur la carte de stockage
	fCree(CompleteRep(sRep) + "monFichier.txt")
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = SysEtatCarteStockage()`
---

**`<Résultat> : (Constante de type Entier)`**

Etat actuel de la carte de stockage sur l'appareil : 


|   |   |
| --- | --- |
| *sysCarteDisponible* | Il est possible de lire et d'écrire sur la carte de stockage. |
| *sysCarteIndisponible* | Il n'est pas possible de lire ou d'écrire sur la carte de stockage. |
| *sysCarteLectureSeule* | Il est uniquement possible de lire sur la carte de stockage. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Gestion de la carte de stockage
<a name="gestion_carte_stockage_ELTPARAGRAPHE000053"></a>

- Le répertoire de la carte de stockage externe de l'appareil est connu avec la fonction [SysRepCarteStockage](../WDLang1/3073034.md). 

- Le répertoire de la carte de stockage externe de l'appareil peut être inaccessible en lecture et/ou en écriture pour différentes raisons : 

	- l'appareil est connecté au PC, 

	- la carte de stockage a été retirée, problème matériel, ...

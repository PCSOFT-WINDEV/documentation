
## CorbeilleRestaure (Fonction)

***En anglais : RecycleBinRestore***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Restaure un fichier placé dans la corbeille système.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// -- Bouton de remplissage d'un champ Liste
// Affiche les éléments de la corbeille dans un champ Liste
sMaListeCorbeille est une chaîne
ListeSupprimeTout(LISTE_Corbeille)
sMaListeCorbeille = CorbeilleListeFichier()
ListeAjoute(LISTE_Corbeille, sMaListeCorbeille)

// -- Bouton de restauration de l'élément sélectionné dans le champ Liste
CorbeilleRestaure(LISTE_Corbeille[LISTE_Corbeille])
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = CorbeilleRestaure(<Nom du fichier>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la restauration a été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Nom du fichier> : (Chaîne de caractères)`**

Nom complet d'origine du fichier (chemin complet et nom du fichier). Si le fichier est présent en plusieurs exemplaires dans la corbeille, tous les fichiers seront restaurés (sous des noms différents). 
Pour restaurer uniquement un fichier, il est nécessaire de faire suivre le nom du fichier par l'heure d'effacement en utilisant le format suivant : 
&lt;Nom du fichier&gt; + TAB + &lt;Heure d'effacement&gt;
Ce format est renvoyé par la fonction [CorbeilleListeFichier](../WDLang1/1000017162.md).

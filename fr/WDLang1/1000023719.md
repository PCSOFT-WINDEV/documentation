


## &lt;Tableur&gt;.Charge (Fonction)

***En anglais : &lt;Spreadsheet&gt;.Load***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Charge un fichier au format XLSX dans un champ Tableur présent dans une fenêtre.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Charge la seconde feuille du fichier XLSX
TBLR_Tableur.Charge("Classeur1.XLSX", 2)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Champ Tableur>.Charge(<Fichier à charger> [, <Numéro de feuille> [, <Option> [, <Mot de passe>]]])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le fichier a été chargé dans le champ Tableur, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Champ Tableur> : (Nom de champ)`**

Nom du champ Tableur manipulé.

**`<Fichier à charger> : (Chaîne de caractères)`**

Nom et chemin complet du fichier à charger. Les formats supportés sont : XLSX, XLSM, XLTX, XLTM.

Attention : Il n'est pas possible de charger un fichier XLSX contenant plus de 2 millions de cellules.

**`<Numéro de feuille> : (Entier optionnel)`**



- Numéro de la feuille à charger. 

- -1 pour utiliser le comportement par défaut : 

	- Si le champ Tableur est en mode multifeuille, toutes les feuilles de calcul seront chargées.

	- Si le champ Tableur n'est pas en mode multifeuille, seule la première feuille de calcul sera chargée.







**`<Option> : (Constante de type Entier)`**

Mode d'ouverture du fichier au format XLSX à charger : 


|   |   |
| --- | --- |
| *tblrChargeDéfaut*<br>(Valeur par défaut) | Le fichier XLSX chargé dans le champ Tableur est ouvert en écriture. <br><br>Dans ce mode, pour ouvrir le fichier ".xlxs" avec une autre application, il est nécessaire de le fermer avec la fonction [&lt;Tableur&gt;.Ferme](../WDLang1/1000023722.md). |
| *tblrChargeDétaché* | Le fichier XLSX chargé dans le champ Tableur est chargé en mémoire uniquement. Le fichier ".xlxs" peut être ouvert avec une autre application. <br><br>Pour sauvegarder les données modifiées, il est nécessaire d'utiliser la fonction [&lt;Tableur&gt;.Sauve](../WDLang1/1000023734.md) avec la constante *tblrEcrasementAutorisé*. Dans ce cas, les données modifiées par une autre application seront écrasées. |
| *tblrChargeGardeMdp* | Le mot de passe du fichier XLSX chargé dans le champ Tableur est conservé en mémoire. <br><br>Lors de la sauvegarde du fichier ("Enregistrer sous" ou [&lt;Tableur&gt;.Sauve](../WDLang1/1000023734.md)), il ne sera pas nécessaire de préciser à nouveau le mot de passe du fichier. Le mot de passe mémorisé sera conservé. |



**`<Mot de passe> : (Chaîne de caractères optionnelle)`**

Mot de passe d'ouverture du fichier XLSX à manipuler (si le fichier est protégé par un mot de passe d'ouverture). 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Le contenu du fichier est chargé dans le champ Tableur.

- L'ancien contenu est supprimé.

- Les formules sont calculées.

- Certaines fonctionnalités d'Excel ne sont pas supportées.

- La taille des fichiers XLSX pouvant être chargés est limitée à 2 millions de cellules. 

- Cette fonction est utilisable uniquement sur un champ Tableur présent dans une fenêtre. 

- La fonction **&lt;Tableur&gt;.Charge** met à jour la propriété [CheminFichier](../Proprietes/1000023104.md) du champ Tableur en associant le nom du fichier chargé au champ Tableur.

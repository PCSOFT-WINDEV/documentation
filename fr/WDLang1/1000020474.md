


## ChronoRAZ (Fonction)

***En anglais : ChronoReset***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Arrête et remet un chronomètre existant à zéro. La valeur de la durée écoulée est perdue.  




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// -- Code de clic du bouton BTN_Reinit
// Réinitialise le chronomètre en cours
ChronoRAZ()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Interrompre et réinitialiser le chronométrage réalisé avec un numéro de chronomètre

`ChronoRAZ([<Numéro du chronomètre>])`
---

**`<Numéro du chronomètre> : (Entier optionnel)`**

Identifiant du chronomètre à réinitialiser. Si ce paramètre n'est pas précisé, sa valeur par défaut est 1.

### Interrompre et réinitialiser le chronométrage réalisé avec une variable de type Chrono

`ChronoRAZ(<Chronomètre>)`
---

**`<Chronomètre> : (Variable de type Chrono)`**

Nom de la variable de type [Chrono](../WDLang1/1000023560.md) correspondant au chronomètre à utiliser.

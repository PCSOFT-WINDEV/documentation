
## ActionBarRechercheVisible (Fonction)

***En anglais : ActionBarSearchVisible***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de connaître ou modifier l'état de visibilité du champ de recherche associé à l'Action Bar de la fenêtre courante.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
ActionBarRechercheVisible(Vrai)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Modifier l'état de visibilité du champ de recherche associé à l'Action Bar

`ActionBarRechercheVisible(<Visible> [, <Indication>])`
---

**`<Visible> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le champ de recherche doit devenir visible. 

- <u><u><u><u>Faux</u></u></u></u> si le champ de recherche doit devenir invisible. 




**`<Indication> : (Chaîne de caractères optionnelle)`**

Texte d'indication à afficher dans le champ de recherche.


<a name="SYNTAXE2"></a>

### Connaître l'état de visibilité du champ de recherche associé à l'Action Bar

`<Résultat> = ActionBarRechercheVisible()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le champ de recherche est visible, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Suivant les options du champ Action Bar, une auto complétion d'historique de recherche sera aussi affichée.

- Cette fonction s'applique sur la fenêtre courante (c'est-à-dire active au moment de l'appel). 

- Pour que cette fonction puisse s'appliquer, la fenêtre courante doit posséder un champ Action Bar ayant l'option "Autoriser la recherche dans l'Action Bar" cochée. Une erreur fatale est affichée dans le cas contraire. Pour plus de détails, consultez [Description du champ Action Bar](../WDChamp/1000025005.md).


## SysLibèreDC (Fonction)

***En anglais : SysReleaseDC***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Libère le DC (Device Context) en cours récupéré par la fonction [SysRécupèreDC](../WDLang1/3073023.md).

**Remarque** : Si un DC (Device Context) a été récupéré pendant le déroulement d'une application et qu'aucune libération n'a été faite, le DC (Device Context) en cours est automatiquement libéré à la fin de l'application.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Libération du DC
SysLibèreDC()
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`SysLibèreDC()`

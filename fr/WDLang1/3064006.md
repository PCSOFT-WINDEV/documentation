
## ListeDLL (Fonction)

***En anglais : ListDLL***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Retourne :

- **en exécution** : la liste des librairies (fichiers ".DLL") utilisées par l'application WINDEV, WINDEV Mobile ou par le site WEBDEV en cours. Seules les librairies en mémoire sont listées.

- **en mode test** : toutes les librairies installées sur le poste en cours.



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Récupération des librairies utilisées
ListeLibrairie est une chaîne = ListeDLL()
// Parcours de cette liste
Librairie est une chaîne
Librairie = ExtraitChaîne(ListeLibrairie, rangPremier, RC)
TANTQUE Librairie <> EOT
	// Récupération de la librairie
	Trace(ExtraitChaîne(Librairie, 1, TAB))
	// Librairie suivante
	Librairie = ExtraitChaîne(ListeLibrairie, rangSuivant, RC)
FIN
```


<a name="Exemple2"></a>

```wl
// Récupération des librairies utilisées
ListeLibrairie est une chaîne = ListeDLL()
// Parcours de cette liste
POUR TOUTE CHAINE CheminLibrairie de ListeLibrairie SEPAREE PAR TAB + RC
	// Récupération de la librairie
	Trace(CheminLibrairie)
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = ListeDLL()`
---

**`<Résultat> : (Chaîne de caractères)`**

Liste des librairies (fichiers ".DLL") utilisées par l'application en cours ou installées sur le poste en cours, sous la forme :

```txt
<1° librairie> + TAB + RC + <2° librairie> + TAB + RC + ...
+ <X librairie>
```
Où  &lt;X librairie&gt; correspond au nom et chemin complet de la librairie X.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Gestion des erreurs
<a name="gestion_des_erreurs_ELTPARAGRAPHE000041"></a>La fonction **ListeDLL** ne retourne pas de code d'erreur. Pour savoir si cette fonction a généré une erreur, utilisez la variable [ErreurDétectée](../WDLang1/3087001.md). En cas d'erreur, il est possible de connaître le détail de l'erreur grâce à la fonction [ErreurInfo](../WDLang1/3013008.md).

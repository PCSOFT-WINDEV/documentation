


## &lt;Planning&gt;.DateVisible (Fonction)

***En anglais : &lt;Scheduler&gt;.DateVisible ***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Rend visible ou invisible un jour dans un champ Planning. 

Remarque : Le jour rendu invisible n'apparaîtra plus si le champ Planning est affiché à la semaine.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ne pas afficher le mercredi
CeMercredi est Date = DateSys()
CeMercredi.Jour += 3 - DateVersJour(CeMercredi) 
PLN_MonPlanning.DateVisible(CeMercredi, Faux)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Rendre un jour visible ou invisible

`<Champ Planning>.DateVisible(<Date> , <Visible>)`
---

**`<Champ Planning> : (Nom de champ)`**

Nom du champ Planning à manipuler.

**`<Date> : (Chaîne de caractères ou variable de type Date)`**

Date à rendre visible ou invisible. Cette date est au format "AAAAMMJJ".

**`<Visible> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> pour rendre la date visible, 

- <u><u><u><u>Faux</u></u></u></u> pour rendre la date invisible.





<a name="SYNTAXE2"></a>

### Savoir si un jour est visible ou invisible

`<Résultat> = <Champ Planning>.DateVisible(<Date>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la date est visible, 

- <u><u><u><u>Faux</u></u></u></u> si la date est invisible.




**`<Champ Planning> : (Nom de champ)`**

Nom du champ Planning à manipuler.

**`<Date> : (Chaîne de caractères ou variable de type Date)`**

Date à manipuler. Cette date est au format "AAAAMMJJ".



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Les jours invisibles sont "sautés" dans le dessin du champ Planning.
	Par exemple si un mercredi est invisible, le champ planning affiche le mardi puis directement le jeudi. 

- L'affichage est mis à jour immédiatement ou lors du changement de période : 

	- Si le champ Planning affiche actuellement un seul jour et si ce jour devient invisible, le jour est rendu invisible lors du changement de jour ou de période. 

	- Si le champ Planning affiche 1 semaine ou un mois, les jours invisibles ne sont pas affichés mais le champ Planning n'affiche pas plus de jours.

	- Si le champ Planning affiche X jours, les jours invisibles ne sont pas comptés dans les X jours. Par exemple, si le champ Planning affiche 4 jours avec le mercredi invisible, les 4 jours affichés seront : Lundi, Mardi, Jeudi, Vendredi.  




- Si un rendez-vous dure sur plusieurs jours (dont 1 jour invisible), il paraîtra visuellement plus court.

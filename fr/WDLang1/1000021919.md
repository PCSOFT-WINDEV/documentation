


## AgendaVersiCalendar (Fonction)

***En anglais : OrganizerToiCalendar***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Exporte les rendez-vous d'un champ Agenda au format iCalendar.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
siCal est une chaîne ANSI
siCal = AgendaVersiCalendar(AGD_Agenda, "20150101", "20151231")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = AgendaVersiCalendar(<Champ Agenda> [, <Date de début> [, <Date de fin>]])`
---

**`<Résultat> : (Chaîne de caractères au format UTF-8)`**



- Liste des rendez-vous du champ Agenda au format iCalendar. 

- Chaîne vide ("") en cas d'erreur. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md). 




**`<Champ Agenda> : (Nom de champ)`**

Nom du champ Agenda à manipuler. Ce champ peut correspondre : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) au champ Agenda d'une fenêtre.

- au champ Agenda d'un état.




**`<Date de début> : (Chaîne de caractères ou variable de type DateHeure optionnelle)`**

Date de début des éléments pris en compte pour l'export.

**`<Date de fin> : (Chaîne de caractères ou variable de type DateHeure optionnelle)`**

Date de fin des éléments pris en compte pour l'export.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Pour importer des données au format iCalendar dans un champ Agenda, utilisez la fonction [iCalendarVersAgenda](../WDLang1/1000021918.md).

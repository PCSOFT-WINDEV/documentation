
## CouleurSélection (Fonction)

***En anglais : SelectionColor***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Change les caractéristiques du bandeau de sélection dans un champ Table, un champ Combo ou un champ Liste. Les nouvelles caractéristiques sont immédiatement prises en compte.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Change la couleur du bandeau de sélection du champ Table "TABLE_Commande"
CouleurSélection(TABLE_Commande, RVB(255,200,200), VertClair)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`CouleurSélection(<Champ manipulé> , <Couleur du texte> , <Couleur de fond>)`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler : champ Table, Liste ou Combo. La couleur du bandeau de sélection sera modifiée pour ce champ.

**`<Couleur du texte> : (Entier)`**

Nouvelle couleur du texte des éléments sélectionnés dans le champ. Cette couleur peut correspondre à :

- une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)),

- une couleur TSL (obtenue avec la fonction [TSL](../WDLang1/3029057.md)),

- [une couleur prédéfinie du WLangage](../WDLang5/3010002.md).




**`<Couleur de fond> : (Entier)`**

Nouvelle couleur de fond du bandeau de sélection du champ. Cette couleur peut correspondre à :

- une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)),

- une couleur TSL (obtenue avec la fonction [TSL](../WDLang1/3029057.md)),

- [une couleur prédéfinie du WLangage](../WDLang5/3010002.md).






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Erreur
<a name="erreur_ELTPARAGRAPHE000069"></a>Une erreur WLangage apparaît à l'exécution si le nom du champ ne correspond ni à un champ Table, ni à un champ Liste, ni à un champ Combo.


<a name="NOTE0_2"></a>




### Divers
<a name="divers_ELTPARAGRAPHE000078"></a>La fonction **CouleurSélection** peut être utilisée pour définir les caractéristiques du bandeau de sélection d'un champ Combo présent dans un champ Table.




## &lt;Traitement de texte&gt;.Redo (Fonction)

***En anglais : &lt;Word Processing&gt;.Redo***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Rétablit la dernière action annulée sur un champ Traitement de texte.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Refait la dernière action annulée sur le champ
TT_MonDocument.Redo()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Champ Traitement de texte>.Redo()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la dernière action annulée a été rétablie, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Champ Traitement de texte> : (Nom de champ)`**

Nom du champ Traitement de texte à manipuler.

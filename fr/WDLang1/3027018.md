


## HeureDifférence (Fonction)

***En anglais : TimeDifference***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Calcule la différence en centièmes de seconde entre deux heures.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affiche le temps écoulé depuis 12h15
Diff est un entier 
Signe_temps est une chaîne
MonRésultat est une chaîne 

Diff = HeureDifférence("1215", HeureSys())
MonRésultat = EntierVersHeure(Abs(Diff))
SI Diff < 0 ALORS Signe_temps = "-"
Info("Temps écoulé : " + Signe_Temps + Gauche(MonRésultat, 2) + " heures " + ...
	Milieu(MonRésultat, 3, 2) + " minutes " + ...
	Milieu(MonRésultat, 5, 2) + " secondes " + ...
	Milieu(MonRésultat, 7, 2) + " centièmes de seconde")
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = HeureDifférence(<Heure de début> , <Heure de fin>)`
---

**`<Résultat> : (Entier)`**

Nombre de centièmes de seconde écoulés entre les deux heures. Ce nombre est issu de l'opération : 
&lt;Heure de fin&gt; - &lt;Heure de début&gt;.
Ce résultat sera négatif si l'&lt;Heure de début&gt; est postérieure à l'&lt;Heure de fin&gt;.

**`<Heure de début> : (Chaîne de caractères ou variable de type Heure)`**

Heure de début du calcul au format :

- HHMMSSCC

- HHMMSS

- HHMM

- HH




**`<Heure de fin> : (Chaîne de caractères ou variable de type Heure)`**

Heure de fin du calcul au format :

- HHMMSSCC

- HHMMSS

- HHMM

- HH

## Remarques


### Format du résultat
<a name="format_resultat_ELTPARAGRAPHE000095"></a>Pour obtenir la durée dans un format (par exemple HHMMSSCC), utilisez la fonction [EntierVersHeure](../WDLang1/3027006.md).

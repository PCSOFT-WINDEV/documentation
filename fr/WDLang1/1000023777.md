


## &lt;Lecteur PDF&gt;.RecherchePrécédent (Fonction)

***En anglais : &lt;PDF Reader&gt;.SeekPrevious***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Positionne sur le résultat précédent d'une recherche initiée par la fonction [&lt;Lecteur PDF&gt;.RecherchePremier](../WDLang1/1000023778.md) dans le contenu d'un champ Lecteur PDF. 

Remarque : La fonction [&lt;Lecteur PDF&gt;.RechercheSuivant](../WDLang1/1000023779.md) permet de trouver l'élément suivant correspondant à la recherche. 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Recherche un texte dans un champ Lecteur PDF et affiche le texte
PDF_MonLecteurPDF.RecherchePremier("société")
// Recherche l'élément suivant
SI PDF_MonLecteurPDF.RechercheSuivant() = Faux ALORS
	Info("Recherche terminée")
FIN

// Recherche l'élément précédent
PDF_MonLecteurPDF.RecherchePrécédent()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Champ Lecteur PDF>.RecherchePrécédent()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le texte spécifié dans la fonction [&lt;Lecteur PDF&gt;.RecherchePremier](../WDLang1/1000023778.md) a été trouvé dans le document en cours dans le champ Lecteur PDF, 

- <u><u><u><u>Faux</u></u></u></u> si le texte n'a pas été trouvé.




**`<Champ Lecteur PDF> : (Nom de champ)`**

Nom du champ Lecteur PDF à manipuler.

<a name="NOTE0_1"></a>

## Remarques


- Le texte et les options de recherche ont été spécifiés dans la fonction [&lt;Lecteur PDF&gt;.RecherchePremier](../WDLang1/1000023778.md).




## FenEdSelectOccurrence (Fonction)

***En anglais : WinEdSelectOccurrence***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Indique le nombre de champs sélectionnés lors de la modification de l'interface par l'utilisateur final (mode "Modification de l'UI par l'utilisateur final" activé). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Elargir tous les champs de 10 pixels
POUR i = 1 _A_ FenEdSelectOccurrence(FEN_Formulaire)
	X est un Champ <- FenEdSelect(FEN_Formulaire)
	X..Largeur += 10
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = FenEdSelectOccurrence(<Fenêtre>)`
---

**`<Résultat> : (Entier)`**



- Nombre de champs sélectionnés par l'utilisateur final (mode "Modification de l'UI par l'utilisateur final"), 

- 0 si aucun champ est sélectionné ou si la multisélection n'est pas activée. 




**`<Fenêtre> : (Nom de fenêtre)`**

Nom de la fenêtre ou de la fenêtre interne à manipuler. Cette fenêtre doit être en mode modification (fonction [FenEdActive](../WDLang1/1000021843.md)). 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Activer le mode "multisélection" pour l'utilisateur final
<a name="activer_mode_multiselection_pour_utilisateur_final_ELTPARAGRAPHE000037"></a>Le mode "multisélection" pour l'utilisateur final peut être activé : 

- sous l'éditeur : 

	- via les options de paramétrage spécifiques du projet : 

		- sous le volet "Projet", dans le groupe "Projet", cliquez sur "Description". 

		- dans l'onglet "Avancé", cliquez sur le bouton "Options" pour définir les différentes options autorisées. 




- via les options de paramétrage spécifiques de la fenêtre : 

	- Affichez la fenêtre de description de la fenêtre (option "Description" du menu contextuel de la fenêtre).  

	- Dans l'onglet "Détail", cochez l'option "Modification de l'UI par l'utilisateur final". 

	- Le bouton "Options" permet de définir les différentes options autorisées. 

- par programmation via la propriété [FenEditionMode](../Proprietes/1000021824.md).

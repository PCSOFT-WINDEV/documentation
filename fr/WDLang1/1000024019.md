
## &lt;Variable Image&gt;.DessineCadre (Fonction)

***En anglais : &lt;Image variable&gt;.DrawBorder***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Dessine un cadre dans une variable de type [Image](../WDLang1/1000019650.md).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Dessine un rectangle avec bords bleus, fond vert et coins arrondis
// dans une variable de type Image
MonImage est une Image = "Test.bmp"
Rect est un Rectangle
Rect.X = 10
Rect.Y = 15
Rect.Largeur = 100
Rect.Hauteur = 150
StyleCadre est un Cadre
StyleCadre.Couleur  = BleuClair
StyleCadre.Arrondi = 30
StyleCadre.Epaisseur = 1
StyleFond est un Fond
StyleFond.Couleur = VertClair
MonImage.DessineCadre(Rect, StyleCadre, StyleFond)
IMG_MonDessin = MonImage
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Variable Image>.DessineCadre(<Cadre> [, <Style du cadre> [, <Style du fond>]])`
---

**`<Variable Image> : (Variable de type Image)`**

Nom de la variable de type [Image](../WDLang1/1000019650.md) à manipuler.

**`<Cadre> : (Variable de type Rectangle)`**

Nom de la variable de type [Rectangle](../Motscles/1514079.md) spécifiant les limites du cadre à dessiner.

**`<Style du cadre> : (Variable de type Cadre optionnelle)`**

Nom de la variable de type [Cadre](../Motscles/15140100.md) décrivant le style du cadre. 

Si ce paramètre n'est spécifié ou si les propriétés **Couleur**, **Epaisseur** et **Trait** de la variable de type [Cadre](../Motscles/15140100.md) sont indéfinies, les informations du style courant (définies par la fonction [&lt;Variable Image&gt;.Stylo](../WDLang1/1000024240.md)) sont utilisées. Si la fonction [&lt;Variable Image&gt;.Stylo](../WDLang1/1000024240.md) n'a pas été utilisée : 

- Couleur du cadre : correspond à la couleur de fond du rectangle. 

- Epaisseur :  1 

- Trait : correspond à la constante *TraitContinu*.




**`<Style du fond> : (Variable de type Fond optionnelle)`**

Nom de la variable de type [Fond](../WDLang1/1000022038.md) décrivant le style du fond. 

Si ce paramètre n'est pas précisé, les informations de la brosse courante sont utilisées.

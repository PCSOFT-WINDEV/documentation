
## KanbanPositionListe (Fonction)

***En anglais : KanbanPositionList***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de faire défiler une liste d'un champ Kanban afin d'afficher une carte donnée. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
PositionCarte est un entier
MaCarte est un kbCarte
MaCarte.Titre = "Anglais"
PositionCarte = KanbanInsèreCarte(KANBAN_Révision, LISTE_Matière_à_Revoir, 3, MaCarte)

// Sélectionne la carte insérée
MaCarteKanban est un kbCarte
MaCarteKanban <- LISTE_Matière_à_Revoir[PositionCarte]

// Affiche la carte insérée
KanbanPositionListe(LISTE_Matière_à_Revoir, MaCarteKanban)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`KanbanPositionListe(<Liste de Kanban> , <Carte>)`
---

**`<Liste de Kanban> : (Nom de champ)`**

Nom du champ Liste de Kanban à manipuler. 

**`<Carte> : (Variable de type kbCarte)`**

Nom de la variable de type [kbCarte](../WDLang1/1410089167.md) correspondant à la carte à afficher. Si la carte n'est pas présente dans la liste de Kanban, aucune action n'est réalisée.

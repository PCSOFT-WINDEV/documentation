
## EtatsUnis.VérifieSSN (Fonction)

***En anglais : UnitedStates.CheckSSN***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Teste la validité d'un numéro SSN (numéro d'identification personnel aux Etats-Unis).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Test d'une variable
MonNuméroSSN est une chaîne = "721074426"
EtatsUnis.VérifieSSN(MonNuméroSSN)		// renvoie Vrai

// Test d'un champ
SAI_NuméroSSN = "ABC"
EtatsUnis.VérifieSSN(SAI_NuméroSSN)	// renvoie Faux
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = EtatsUnis.VérifieSSN(<Numéro à vérifier>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'élément spécifié contient un numéro SSN valide.

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Numéro à vérifier> : (Nom de champ ou variable de type chaîne)`**

Numéro SSN à vérifier. Ce paramètre peut correspondre : 

- au nom d'une variable de type chaîne.

- au nom d'un champ.

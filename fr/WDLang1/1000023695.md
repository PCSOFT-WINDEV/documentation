


## &lt;Lecteur PDF&gt;.Ferme (Fonction)

***En anglais : &lt;PDF Reader&gt;.Close***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ferme et libère le fichier PDF affiché dans un champ Lecteur PDF.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
PDF_MonLecteurPDF.Ferme()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Champ Lecteur PDF>.Ferme()`
---

**`<Champ Lecteur PDF> : (Nom de champ)`**

Nom du champ Lecteur PDF contenant le fichier PDF à fermer.

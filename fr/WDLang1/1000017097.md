
## CSVVersTableau (Fonction)

***En anglais : CSVToArray***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Remplit un tableau à 1 ou 2 dimensions du type spécifié avec la chaîne de caractères au format CSV passée en paramètre. Le tableau est vidé avant d'être rempli.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Tableau à une dimension
sMesValeurs est une chaîne = "A" + RC + "B" + RC + "C" + RC + "D" + RC + "D"
// Tableau initial
MonTableau est un tableau de 3 chaînes
MonTableau[1] = "Y"
MonTableau[2] = "X"
CSVVersTableau(sMesValeurs, MonTableau)
// Le tableau contient : "A" , "B" , "C" , "D" , "D"
```


<a name="Exemple2"></a>





```wl
// Tableau à deux dimensions
sMesValeurs est une chaîne = "A" + TAB + "B" + RC + "C" +TAB + "D" + RC + "E" + TAB + "F"
// Tableau initial
MonTableau est un tableau de 2 par 3 chaînes
MonTableau[1,1] = "Y"
MonTableau[1,2] = "Z"
CSVVersTableau(sMesValeurs, MonTableau)
// Le tableau contient : "A;B" , "C;D" , "E;F"
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`CSVVersTableau(<Chaîne> , <Tableau> [, <Séparateur de colonne>])`
---

**`<Chaîne> : (Chaîne de caractères)`**

Contient les différents éléments du tableau.

**Pour un tableau à 1 dimension**, la chaîne sera découpée selon les Retours Chariots (RC).

**Pour un tableau à 2 dimensions**, la chaîne sera découpée selon les Retours Chariots (RC), puis suivant le séparateur de colonnes. Les éléments seront ajoutés ligne par ligne. Il y aura autant de lignes que de Retours Chariots trouvés. Des colonnes seront ajoutées si le nombre maximal de &lt;Séparateur de colonne&gt; trouvé dépasse le nombre de colonnes du tableau. 
A chaque colonne doit correspondre une valeur pour la ligne.

Attention : ce paramètre ne doit pas contenir de guillemets. 

**`<Tableau> : (Variable de type Tableau)`**

Nom du tableau à 1 ou 2 dimensions dans lequel les éléments contenus dans la chaîne doivent être ajoutés. Ce tableau doit être alloué dans un type simple ou un type structure.

**`<Séparateur de colonne> : (Chaîne de caractères optionnelle)`**

Séparateur utilisé pour séparer les valeurs des différentes colonnes. Par défaut, le séparateur de colonnes utilisé correspond au paramètre "séparateur de liste" des options régionales de Windows.

![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) Le séparateur est le point virgule ";".



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- La dimension du tableau sera modifiée si nécessaire : des lignes et des colonnes pourront être ajoutées selon les valeurs contenues dans la &lt;Chaîne&gt;.

- Les tableaux de type structure gérés sont uniquement des tableaux à une dimension.

- Les tableaux contenant des chaînes UNICODE ou des variants ne sont pas gérés.

- Pour le type Durée, la valeur doit être fournie en millièmes de seconde. La valeur 0 n'est pas valide. Il faut utiliser "0000000000".




## docSection (Type de variable)

***En anglais : docSection***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **docSection** permet de définir toutes les caractéristiques avancées d'une section d'un document ou d'un paragraphe. 

Au niveau du document : 

- le type **docSection** permet de connaître les informations générale du document.

- le type **docSection** permet de modifier toutes les informations de toutes les sections du document.




Au niveau du paragraphe : 

- le type **docSection** permet de connaître les informations sur le ou les paragraphes appartenant à la même section.

- le type **docSection** permet de modifier les informations sur le ou les paragraphes de la même section.




Les caractéristiques de cette section peuvent être définies et modifiées à l'aide de différentes propriétés WLangage. 

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Récupération du champ Traitement de texte
MonDoc est un Document <- TT_MonDoc
// Récupération du l'orientation du document
Info(MonDoc.Section.Orientation)
// Passage de l'orientation du document en paysage
MonDoc.Section.Orientation = docPaysage
```


## RVBVert (Fonction)

***En anglais : RGBGreen***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la composante verte d'une couleur RVB.






<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
RVBVert(RVB(128, 38, 15)) // Renvoie 38
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = RVBVert(<Couleur RVB>)`
---

**`<Résultat> : (Entier)`**

Composante verte comprise entre 0 et 255 de la couleur RVB.

**`<Couleur RVB> : (Entier)`**

Couleur (24 bits). Cette couleur est le résultat de la formule suivante :

Couleur = 65536 \* Composante Bleu + 256 \* Composante Verte + Composante Rouge.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Divers
<a name="divers_ELTPARAGRAPHE000081"></a>

- La fonction [RVB](../WDLang1/3029012.md) permet d'obtenir l'identifiant de la couleur décrite à partir de ses composantes Bleu, Verte, Rouge.

- La fonction [RVBRouge](../WDLang1/3029050.md) permet de connaître la composante rouge d'une couleur RVB. La fonction [RVBBleu](../WDLang1/3029049.md) permet de connaître la composante bleue d'une couleur RVB.

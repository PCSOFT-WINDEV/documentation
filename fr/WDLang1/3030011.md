


## Fonctions de gestion du Drag and Drop
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions de gestion du Drag and Drop sont les suivantes :



|   |   |
| --- | --- |
| [DnDAccepte](../WDLang1/3030005.md) | Indique l'action acceptée par la cible du glisser/déplacer et gère le curseur de souris lors du glisser/déplacer. |
| [DnDCurseur](../WDLang1/3030004.md) | Gère le curseur souris et indique l'action autorisée lors du glisser/déplacer. |
| [DnDDonne](../WDLang1/3030001.md) | Indique le type des données et les données à copier/déplacer lors du glisser/déplacer. |
| [DnDDonnéeDisponible](../WDLang1/3030002.md) | Teste si des données d'un type spécifié sont disponibles lors du glisser/déplacer. |
| [DnDDonneElémentTDB](../WDLang1/1000020971.md) | Ajoute un Widget dans un champ Tableau de bord lors d'un "Drag and Drop". |
| [DnDEvénement](../WDLang1/3030006.md) | Indique la procédure à exécuter lors d'un événement du glisser/déplacer. |
| [DnDImpose](../WDLang1/3030003.md) | La fonction DnDImpose est conservée uniquement pour compatibilité. |
| [DnDRécupère](../WDLang1/3030008.md) | Récupère les données d'un certain type lors du glisser/déplacer. |
| [ExplorerAccepte](../WDLang1/3030015.md) | Active ou désactive la possibilité de faire du "Drag and Drop" (Glisser/Déplacer) depuis l'explorateur Windows vers une fenêtre ou un champ WINDEV. |
| [ExplorerRécupère](../WDLang1/3030009.md) | Récupère le nombre et le nom des fichiers "droppés" depuis l'explorateur. |


## VersionWINDEV (Fonction)

***En anglais : WINDEVVersion***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le numéro de version de la DLL wdxxxvm.

![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Renvoie le numéro de version de l'archive wdxxxjava.jar.

**Remarque** : Cette fonction est équivalente à la fonction [ExeInfo](../WDLang1/3035001.md) avec la constante *exeVersion*.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) 
```wl
Info("Version WINDEV : " + VersionWINDEV())
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = VersionWINDEV([<Version>])`
---

**`<Résultat> : (Chaîne de caractères ou constante)`**



- Numéro de version de l'élément demandé.

- Constante correspondant à la langue du framework utilisé. 




**`<Version> : (Constante optionnelle de type Entier)`**

Version recherchée :


|   |   |
| --- | --- |
| *versionFramework* | Renvoie le numéro de version du framework PC SOFT (par exemple, "01F280039g").<br><br> |
| *versionLangue* | Renvoie une constante correspondant à la langue du framework : <br><br>- *langueAnglais* : Framework en anglais.<br><br>- *langueChinois* : Framework en chinois. <br><br>- *langueFrançais* : Framework en français. <br><br><br><br><br>![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Non disponible. |
| *versionVM*<br>(valeur par défaut) | Renvoie le numéro de version de la DLL wdxxxvm (par exemple, "28.01Dn").<br><br>![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Renvoie le numéro de version de l'archive wdxxxjava.aar.<br>Pour connaître la version de cette archive, il suffit de :<br><br>1. Ouvrir l'archive ".jar" ou ".aar" (il s'agit d'une archive au format zip),<br><br>2. Accéder au dossier "\\res\\raw\\",<br><br>3. Ouvrir le fichier "build_info.properties". Ce fichier texte contient (par exemple) : <br>	<br><pre><code>major.number=25<br>	minor.number=0<br>	build.number=341<br>	revision.number=8</code></pre><br><br><br><br><br><br>Si une application Android APK / AAB est générée avec cette version du framework, la fonction WLangage **VersionWINDEV** retourne "25.0.341.8".<br><br> |

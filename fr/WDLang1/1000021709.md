
## Fonctions de gestion des champs Onglet (syntaxe préfixée)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions permettant de manipuler les champs Onglet (via la syntaxe préfixée) sont les suivantes : 



|   |   |
| --- | --- |
| [&lt;Onglet&gt;.ChargeConfiguration](../WDLang1/1000021699.md) | Restaure la configuration des volets dynamiques d'un champ Onglet. |
| [OngletDuChamp](../WDLang1/3025028.md) | Renvoie le nom du champ Onglet qui contient le champ spécifié. |
| [&lt;Onglet&gt;.Etat](../WDLang1/1000021700.md) | Renvoie l'état d'un volet d'onglet dynamique (identifié par son alias). |
| [&lt;Onglet&gt;.Ferme](../WDLang1/1000021702.md) | Ferme un volet dynamique dans un champ Onglet. |
| [&lt;Onglet&gt;.FermeTout](../WDLang1/1000021703.md) | Ferme tous les volets dynamiques d'un champ Onglet. |
| [&lt;Onglet&gt;.Ouvre](../WDLang1/1000021704.md) | Ouvre un nouveau volet dans un champ Onglet dynamique. |
| [&lt;Onglet&gt;.SauveConfiguration](../WDLang1/1000021708.md) | Sauvegarde la configuration des volets dynamiques d'un champ Onglet sous forme d'une chaîne de caractères. |
| [VoletDuChamp](../WDLang1/1000021625.md) | Renvoie le nom du volet (statique ou dynamique) d'un champ Onglet associé à un champ. |

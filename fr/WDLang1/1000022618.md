


## Fonctions de gestion du champ Traitement de texte
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions permettant de gérer les champs Traitements de texte sont les suivantes : 



|   |   |
| --- | --- |
| [DocAjoute](../WDLang1/1000022919.md) | Ajoute : <br><br>- à la fin d'un document de type Traitement de texte : un élément, un document, un paragraphe, un texte, une image ou un fragment.<br><br>- à la fin d'un fragment : un élément, un texte ou une image.<br><br><br> |
| [DocChangeAltitude](../WDLang1/1000023970.md) | Modifie l'altitude d'un élément de type Image ou de type Zone de texte dans un document DocX. |
| [DocDébutModification](../WDLang1/1000022620.md) | Signale le début du regroupement d'un ensemble d'opérations sur un document en un seul événement (gestion du "undo/redo"). |
| [DocEntreEdition](../WDLang1/1000023077.md) | Entre en édition dans le champ Traitement de texte dans une partie spécifique du document : corps, en-tête ou pied de page. |
| [DocFerme](../WDLang1/1000022621.md) | Ferme un fichier docx et libère le document. |
| [DocFinModification](../WDLang1/1000022423.md) | Signale la fin du regroupement d'un ensemble d'opérations en un seul événement (gestion du "undo/redo"). |
| [DocImprime](../WDLang1/1000022640.md) | Imprime un document au format "docx". |
| [DocInfoXY](../WDLang1/1000025333.md) | Renvoie pour une position donnée dans un champ Traitement de texte (coordonnées d'un point du champ) :<br><br>- soit le numéro de la page correspondante, <br><br>- soit la position X (en mm) correspondante, <br><br>- soit la position Y (en mm) correspondante.<br><br><br> |
| [DocInsère](../WDLang1/1000022732.md) | Insère un objet dans un document de type Traitement de texte ou remplace le contenu du fragment en cours. Les objets manipulables sont : <br><br>- un élément,<br><br>- un document, <br><br>- un paragraphe, <br><br>- un texte, <br><br>- une image, <br><br>- un fragment.<br><br><br> |
| [DocInsèreTableau](../WDLang1/1000022967.md) | Insère un tableau dans un document de type Traitement de texte. |
| [DocInsèreTableDesMatières](../WDLang1/1000024884.md) | Insère une table des matières dans un document de type Traitement de texte. |
| [DocMiseAJourFormule](../WDLang1/1000024897.md) | Met à jour les formules d'un document. |
| [DocMiseAJourTableDesMatières](../WDLang1/1000024885.md) | Met à jour la table des matières d'un document. |
| [DocOuvre](../WDLang1/1000022622.md) | Ouvre un fichier Traitement de texte (format docx) en mode Lecture/Ecriture. Le fichier docx est ouvert et bloqué jusqu'à sa fermeture. |
| [DocPositionDébutPage](../WDLang1/1410088921.md) | Renvoie la position du premier caractère d'une page donnée dans un champ Traitement de texte. |
| [DocPositionne](../WDLang1/1000022743.md) | Provoque le défilement du champ Traitement de texte afin que le texte voulu apparaisse entièrement. |
| [DocRecherche](../WDLang1/1000022388.md) | Recherche une chaîne de caractères : <br><br>- dans l'ensemble d'un document de type Traitement de texte.<br><br>- dans un fragment.<br><br><br> |
| [DocRechercheElémentParNom](../WDLang1/1410088857.md) | Recherche un élément par son nom dans l'ensemble d'un document de type Traitement de texte. |
| [DocRedo](../WDLang1/1000022425.md) | Rétablit la dernière action annulée sur un document de type Traitement de texte. |
| [DocRedoOccurrence](../WDLang1/1000022427.md) | Renvoie le nombre d'actions annulées qui peuvent être refaites sur un document de type Traitement de texte. |
| [DocRemplace](../WDLang1/1000022496.md) | Recherche et remplace toutes les occurrences d'une chaîne de caractères par une autre chaîne : <br><br>- dans l'ensemble d'un document de type Traitement de texte.<br><br>- dans un fragment.<br><br><br> |
| [DocSauve](../WDLang1/1000022387.md) | Sauve un document au format "docx". |
| [DocSélectionne](../WDLang1/1000022391.md) | Sélectionne un fragment dans un champ Traitement de texte. |
| [DocSortEdition](../WDLang1/1000023088.md) | Sort de l'édition en cours. L'entrée en édition a été effectuée avec la fonction [DocEntreEdition](../WDLang1/1000023077.md). |
| [DocSupprime](../WDLang1/1000024172.md) | Supprime un élément d'un document. Cet élément est supprimé du document auquel il est associé. |
| [DocSupprimeTableDesMatières](../WDLang1/1000024916.md) | Supprime une table des matières dans un document. |
| [DocUndo](../WDLang1/1000022424.md) | Annule la dernière modification effectuée sur un document de type Traitement de texte. |
| [DocUndoOccurrence](../WDLang1/1000022426.md) | Renvoie le nombre d'actions annulables sur un document. |
| [DocUndoRedoSupprimeTout](../WDLang1/1000022734.md) | Supprime tout l'historique du mécanisme d'Undo/Redo d'un document de type Traitement de texte. |
| [DocVersHTML](../WDLang1/1000022393.md) | Convertit un document de type Traitement de texte en un fichier HTML. |
| [DocVersImage](../WDLang1/1000022808.md) | Exporte une page d'un document de type Traitement de texte sous forme d'une image. |
| [DocVersPDF](../WDLang1/1000022531.md) | Convertit un document de type Traitement de texte en un fichier PDF. |
| [DocVersTexte](../WDLang1/1000022389.md) | Crée une chaîne de caractères avec les données : <br><br>- d'un champ Traitement de texte ou d'un document Traitement de texte. <br><br>- d'un paragraphe d'un champ Traitement de texte.<br><br><br> |

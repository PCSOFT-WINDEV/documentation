


## TableColonneSelect (Fonction)

***En anglais : TableColumnSelect***




	<B><B>Avertissement</B></B>

	A partir de la version <B><B>16 (56)</B></B>,  cette fonctionnalité est conservée par compatibilité. Il est conseillé d'utiliser la fonction [TableSelect](../WDLang1/3074020.md).

<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le numéro de colonne correspondant à l'une des colonnes sélectionnées dans un champ Table. 

**Remarques** :  

- Le champ Table doit autoriser le mode de sélection à la colonne. Cette option est paramétrable dans l'onglet "IHM" de la description du champ Table. 

- Pour connaître le nombre de colonnes sélectionnées, utilisez la fonction [TableColonneSelectOccurrence](../WDLang1/1000019380.md).



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
NbColSélection est entier 
NumCol est entier 

NbColSélection = TableColonneSelectOccurrence(TABLE_TABLE1)
Trace("TableColonneSelectOccurrence() = " + NbColSélection)

POUR i = 1 A NbColSélection
	NumCol = TableColonneSelect(TABLE_TABLE1,i)
	Trace("La colonne " + NumCol + " est sélectionnée")
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = TableColonneSelect(<Champ Table> , <Indice>)`
---

**`<Résultat> : (Entier)`**



- Numéro de la colonne sélectionnée. Le numéro de la première colonne est 1. 

- -1 en cas d'erreur. 




**`<Champ Table> : (Nom de champ)`**

Nom du champ Table à manipuler. 

**`<Indice> : (Entier)`**

Indice de la sélection. L'indice du premier élément sélectionné est 1. Si &lt;Indice&gt; est supérieur au nombre de colonnes sélectionnées, &lt;Résultat&gt; vaut -1.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Pour autoriser la sélection d'une colonne, il est nécessaire d'activer dans les caractéristiques du champ Table : 

- le mode de sélection de type "Colonnes" (sélection simple ou multiple) dans l'onglet "IHM" de la fenêtre de description du champ Table. 

- le mode de sélection de type "Cellules" (sélection multiple) dans l'onglet "IHM" de la fenêtre de description du champ Table. 




L'utilisateur final peut sélectionner les colonnes en cliquant dans l'en-tête de la colonne (comme pour les trier).

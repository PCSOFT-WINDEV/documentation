
## ETBinaire (Fonction)

***En anglais : BinaryAND***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le résultat du ET logique (bit à bit) entre deux valeurs.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Récupération du ET logique
ResETBinaire = ETBinaire(6, 6)  // Renvoie 6
ResETBinaire = ETBinaire(6, 2)  // Renvoie 2
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = ETBinaire(<Valeur 1> , <Valeur 2>)`
---

**`<Résultat> : (Entier)`**

Résultat du ET logique.

**`<Valeur 1> : (Entier)`**

Première valeur à tester.

**`<Valeur 2> : (Entier)`**

Seconde valeur à tester.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Calcul bit à bit
<a name="calcul_bit_bit_ELTPARAGRAPHE000076"></a>**ETBinaire** effectue le calcul bit à bit entre &lt;Valeur1&gt; et &lt;Valeur2&gt; selon la table suivante :


|   |   |   |
| --- | --- | --- |
|   | 0 | 1 |
| 0 | 0 | 0 |
| 1 | 0 | 1 |




<a name="NOTE0_2"></a>




### Opérateur binaire
<a name="operateur_binaire_ELTPARAGRAPHE000105"></a>

- La fonction **ETBinaire** est équivalente à l'opérateur binaire **&**. Pour plus de détails, consultez [Opérateurs binaires](../Motscles/1512009.md).

- Le résultat de la fonction **ETBinaire** est exact sur des entiers sur 4 octets. Son utilisation avec des entiers sur 8 octets provoque un débordement de capacité.

- L'opérateur binaire **&** permet de faire des opérations sur des entiers sur 8 octets.


## MatMultiplie (Fonction)

***En anglais : MatMultiply***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Multiplie deux matrices.

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Matmul0.gif)





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple
<a class="notetitle" target="_blank" href="$DOC$=1000003050072&name=matmultiplie_fonction&product=WD">Voir des exemples supplémentaires</a>

```wl
// Multiplication de deux matrices
ResMultiplie = MatMultiplie("MaMatrice1", "MaMatrice2", "MaMatriceRésultat")
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = MatMultiplie(<1ère matrice> , <2ème matrice> , <Matrice résultat>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la multiplication a été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour connaître le libellé de l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.




**`<1ère matrice> : (Chaîne de caractères)`**

Nom de la première matrice à multiplier, défini avec la fonction [MatCrée](../WDLang1/3050058.md).

**`<2ème matrice> : (Chaîne de caractères)`**

Nom de la seconde matrice à multiplier, défini avec la fonction [MatCrée](../WDLang1/3050058.md).

**`<Matrice résultat> : (Chaîne de caractères)`**

Nom de la matrice contenant le résultat de la multiplication. Cette matrice est automatiquement créée lors de la multiplication des deux matrices.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Multiplication de matrices
<a name="multiplication_matrices_ELTPARAGRAPHE000097"></a>Pour multiplier deux matrices, le nombre de colonnes de la première matrice doit être égal au nombre de lignes de la seconde matrice.

L'élément (ligne **i**, colonne **j**) de la matrice résultat est obtenu en multipliant la **i**ème ligne de la première matrice par la **j**ème colonne de la seconde matrice.

Il est possible de multiplier une matrice par elle-même. Pour cela, utilisez le même nom de matrice pour le paramètre &lt;1ère matrice&gt; et le paramètre  &lt;2ème matrice&gt;.


<a name="NOTE0_2"></a>




### Existence de la matrice résultat
<a name="existence_matrice_resultat_ELTPARAGRAPHE000110"></a>L'existence de la matrice n'est pas vérifiée. Si la matrice existait auparavant, cette matrice est écrasée. Pour vérifier l'existence d'une matrice, utilisez la fonction [MatExiste](../WDLang1/3050076.md).


## SaisieCurseur (Fonction)

***En anglais : EditCaret***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie le curseur de survol standard de tous les champs de saisie de l'application.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SaisieCurseur("MonCurseur.cur")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`SaisieCurseur(<Curseur>)`
---

**`<Curseur> : (Chaîne de caractères)`**



- Nom d'un fichier de type curseur (fichier de type .cur ou .ani). 

- Chaîne vide ("") pour réactiver le curseur standard.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- La propriété [CurseurSouris](../Proprietes/2510026.md) permet de modifier le curseur de survol d'un champ. Si le curseur de survol a été modifié avec la propriété [CurseurSouris](../Proprietes/2510026.md), la fonction **SaisieCurseur** n'aura pas d'effet sur le curseur du champ. 

- Sous l'éditeur, il est possible de modifier le curseur de survol utilisé par un champ d'une fenêtre dans l'onglet "UI" de la fenêtre de description du champ.


## &lt;Tableau&gt;.Min (Fonction)

***En anglais : &lt;Array&gt;.Min***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la plus petite des valeurs.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Plus petite valeur ?
montableau est un tableau de 5 entiers
montableau = [1,2,3,4,5]
Trace(montableau.Min())
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Calcule le minimum des valeurs du tableau

`<Résultat> = <Tableau WLangage>.Min()`
---

**`<Résultat> : (Chaîne de caractères, entier ou réel)`**

La plus petite des valeurs passées en paramètre.

**`<Tableau WLangage> : (Tableau de chaînes de caractères, d'entiers ou de réels)`**

Nom de la variable de type Tableau à une dimension contenant les valeurs à comparer.

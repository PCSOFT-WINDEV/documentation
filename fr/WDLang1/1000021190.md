


## Fonctions de gestion des champs Sélecteur (syntaxe préfixée)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions permettant de gérer les champs Sélecteur via une syntaxe préfixée sont les suivantes :



|   |   |
| --- | --- |
| [&lt;Sélecteur&gt;.Ajoute](../WDLang1/1000021186.md) | Ajoute une nouvelle option dans un champ Sélecteur. |
| [&lt;Sélecteur&gt;.Insère](../WDLang1/1000021187.md) | Insère une nouvelle option dans un champ Sélecteur. |
| [&lt;Sélecteur&gt;.Occurrence](../WDLang1/1000021188.md) | Renvoie le nombre d'options présentes dans un champ Sélecteur. |
| [&lt;Sélecteur&gt;.Supprime](../WDLang1/1000021189.md) | Supprime une option d'un champ Sélecteur. |

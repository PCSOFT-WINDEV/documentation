
## Constantes de gestion de la correction orthographique
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


| Constante | Description | Utilisée par les fonctions |
| --- | --- | --- |
| **orthPortéePersistant** | Le mot est ajouté dans le dictionnaire de OpenOffice.<br>Cette constante a le même effet que l'option "Ajouter au dictionnaire" du menu contextuel des FAA avec correction orthographique. | [OrthoAjouteDictionnaire](../WDLang1/1000018921.md) |
| **orthPortéeProcessus** | Le mot n'est pas ajouté dans le dictionnaire de OpenOffice.<br>Il continuera donc à être affiché comme "faux" si l'application est relancée. | [OrthoAjouteDictionnaire](../WDLang1/1000018921.md) |

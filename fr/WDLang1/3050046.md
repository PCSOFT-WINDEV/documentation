
## FinAmortissementLinéaire (Fonction)

***En anglais : FinLinearRedemption***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Calcule la valeur de l'amortissement linéaire d'un bien pour une période donnée.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple
<a class="notetitle" target="_blank" href="$DOC$=1000003050046&name=finamortissementlineaire_fonction&product=WD">Voir des exemples supplémentaires</a>

```wl
// Valeur d'amortissement d'un bien acheté 25 000 Euros dont la
// valeur résiduelle sera de 15 000 Euros après une durée de 6 ans.
ResAmortissement = FinAmortissementLinéaire(25000, 15000, 6)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = FinAmortissementLinéaire(<Coût d'acquisition> , <Valeur résiduelle> , <Durée>)`
---

**`<Résultat> : (Réel ou monétaire)`**



- Valeur de l'amortissement, 

- 0 si l'amortissement est nul.




**`<Coût d'acquisition> : (Réel ou monétaire)`**

Coût d'acquisition du bien.

**`<Valeur résiduelle> : (Réel ou monétaire)`**

Valeur résiduelle du bien au terme de la durée d'amortissement.

**`<Durée> : (Réel)`**

Nombre de périodes durant lesquelles le bien est amorti. Cette durée ne peut pas être nulle.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Amortissement linéaire
<a name="amortissement_lineaire_ELTPARAGRAPHE000075"></a>Lors d'un amortissement linéaire, la valeur de l'amortissement est la même pour chaque période d'amortissement du bien.


<a name="NOTE0_2"></a>




### Gestion des erreurs
<a name="gestion_des_erreurs_ELTPARAGRAPHE000084"></a>**Attention** : La fonction **FinAmortissementLinéaire** ne retourne pas de code d'erreur. Pour savoir si cette fonction a généré des erreurs, utilisez la fonction [FinErreur](../WDLang1/3050071.md). Le libellé de l'erreur peut être connu grâce à la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.

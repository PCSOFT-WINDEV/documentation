


## Fonctions de gestion des chaînes de caractères
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


Les fonctions de gestion des chaînes de caractères sont les suivantes :



|   |   |
| --- | --- |
| [AnsiVersOem](../WDLang1/3024007.md) | Convertit une chaîne de caractères au format ANSI (Windows) en une chaîne de caractères au format OEM (DOS). |
| [AnsiVersUnicode](../WDLang1/3024002.md) | Convertit :<br><br>- une chaîne de caractères au format ANSI (Windows) en une chaîne de caractères au format UNICODE.<br><br>- un buffer contenant une chaîne de caractères au format ANSI (Windows) en un buffer contenant une chaîne de caractères au format UNICODE.<br><br><br> |
| [Asc](../WDLang1/3024004.md) | Renvoie le code ASCII du caractère spécifié. |
| [BufferVersEntier](../WDLang1/1000020508.md) | Extrait un entier présent dans un buffer binaire à une position donnée. |
| [BufferVersHexa](../WDLang1/1000019916.md) | Convertit un buffer en une chaîne hexadécimale affichable (par exemple : "4A 5B 00"). |
| [BufferVersPhrase](../WDLang1/1000025938.md) | Renvoie une phrase mnémotechnique à partir d'un buffer en utilisant l'algorithme BIP39. |
| [BufferVersRéel](../WDLang1/1000020509.md) | Extrait un réel présent dans un buffer binaire à une position donnée. |
| [Caract](../WDLang1/3024003.md) | Renvoie le caractère correspondant au code ASCII spécifié. |
| [CaractOccurrenceType](../WDLang1/1000019894.md) | Renvoie le nombre de caractères vérifiant les informations de type données. |
| [CaractType](../WDLang1/1000019895.md) | Renvoie des informations sur le type d'un caractère. |
| [CaractUnicode](../WDLang1/1000019397.md) | Renvoie le caractère correspondant au code Unicode spécifié. |
| [ChaîneCommencePar](../WDLang1/1000018827.md) | Vérifie si une chaîne de caractères commence : <br><br>- soit par une chaîne de caractères spécifique. <br><br>- soit par une des chaînes de caractères présentes dans un tableau.<br><br><br> |
| [ChaîneCompare](../WDLang1/3024011.md) | Compare caractère par caractère deux chaînes :<br><br>- selon l'ordre des caractères ASCII.<br><br>- selon l'ordre lexicographique.<br><br><br> |
| [ChaîneConstruit](../WDLang1/3024006.md) | Construit une chaîne de caractères en fonction d'une chaîne initiale et de paramètres. |
| [ChaîneDécoupe](../WDLang1/1000022322.md) | Extrait toutes les sous-chaînes d'une chaîne de caractères en fonction d'un ou plusieurs séparateurs de chaînes. |
| [ChaîneFinitPar](../WDLang1/1000018828.md) | Vérifie si une chaîne de caractères se termine : <br><br>- soit par une chaîne de caractères spécifique.<br><br>- soit par une des chaînes de caractères présentes dans un tableau.<br><br><br> |
| [ChaîneFormate](../WDLang1/3024019.md) | Formate une chaîne de caractères selon les options choisies. |
| [ChaîneIncrémente](../WDLang1/1000020773.md) | Incrémente un nombre situé en fin ou début d'une chaîne de caractères. |
| [ChaîneInsère](../WDLang1/1000020477.md) | Insère une chaîne dans une autre chaîne de caractères à une position donnée. |
| [ChaîneInverse](../WDLang1/1000019901.md) | Inverse les caractères d'une chaîne. |
| [ChaîneOccurrence](../WDLang1/3024008.md) | Calcule : <br><br>- le nombre d'occurrences d'une chaîne de caractères spécifique (en respectant les critères de recherche) dans une autre chaîne de caractères.<br><br>- le nombre d'occurrences d'un ensemble de chaînes présentes dans un tableau. <br><br><br> |
| [ChaîneRécupère](../WDLang1/3024005.md) | Récupère une chaîne de caractères à partir d'un format externe à PC Soft (chaîne de caractères résultat d'une fonction de l'API Windows par exemple). |
| [ChaîneSupprime](../WDLang1/1000019917.md) | Supprime : <br><br>- toutes les occurrences d'une sous-chaîne à l'intérieur d'une chaîne. <br><br>- une portion de chaîne à l'intérieur d'une chaîne. La portion de chaîne est identifiée par sa position et sa longueur. <br><br><br> |
| [ChaîneSupprimeDoublon](../WDLang1/1410087412.md) | Supprime les doublons parmi les sous-chaînes d'une chaîne de caractères en fonction d'un ou plusieurs séparateurs. |
| [ChaîneVersJSON](../WDLang1/1000024531.md) | Convertit une chaîne au format JSON en une variable de type JSON. |
| [ChaîneVersNumérique](../WDLang1/1410087852.md) | Renvoie la valeur numérique d'une chaîne de caractères. Il est ainsi possible d'effectuer des calculs numériques sur des chaînes de caractères. |
| [ChaîneVersUTF8](../WDLang1/3024042.md) | Convertit une chaîne ANSI ou UNICODE au format UTF8. |
| [Complète](../WDLang1/3024010.md) | Renvoie une chaîne de caractères spécifique de taille déterminée. |
| [ComplèteRep](../WDLang1/3024018.md) | Ajoute, si nécessaire, le caractère "\\" à la fin d'une chaîne de caractères. |
| [Contient](../WDLang1/1000019893.md) | Permet de savoir si une sous-chaîne est contenue dans une chaîne. |
| [DécomplèteRep](../WDLang1/3024044.md) | Enlève le caractère '/' ou '\\' à la fin d'une chaîne si ce caractère est présent. |
| [Droite](../WDLang1/3024016.md) | Extrait la partie droite (c'est-à-dire les derniers caractères) d'une chaîne de caractères ou d'un buffer. |
| [EntierSystèmeVersEntier](../WDLang1/1000022256.md) | Convertit un entier système en un entier sur 4 octets. |
| [EntierVersHexa](../WDLang1/1000019494.md) | Renvoie la chaîne hexadécimale d'une valeur numérique. |
| [ExtraitChaîne](../WDLang1/3024017.md) | Permet de :<br><br>- extraire une sous-chaîne d'une chaîne de caractères en fonction d'un séparateur de chaîne donné.<br><br>- effectuer un parcours de sous-chaînes dans une chaîne de caractères en fonction d'un séparateur de chaîne donné.<br><br><br> |
| [ExtraitChaîneEntre](../WDLang1/1000024868.md) | Permet de :<br><br>- extraire d'une chaîne de caractères une sous-chaîne entre deux séparateurs de chaînes donnés.<br><br>- effectuer un parcours de sous-chaînes entre deux séparateurs de chaîne donnés dans une chaîne de caractères.<br><br><br> |
| [ExtraitLigne](../WDLang1/1000020480.md) | Extrait une ligne spécifique présente dans une chaîne de caractères. |
| [Gauche](../WDLang1/3024001.md) | Extrait la partie gauche (c'est-à-dire les premiers caractères) d'une chaîne de caractères ou d'un buffer. |
| [HexaVersBuffer](../WDLang1/1000019914.md) | Convertit une chaîne représentant des octets en hexadécimal en un buffer binaire. |
| [HexaVersEntier](../WDLang1/1000019413.md) | Renvoie la valeur numérique d'une chaîne hexadécimale. |
| [HTMLVersRTF](../WDLang1/1000017040.md) | Convertit une chaîne HTML ou un buffer HTML en chaîne au format RTF. |
| [HTMLVersTexte](../WDLang1/1000017039.md) | Convertit une chaîne HTML ou un buffer HTML en chaîne texte. |
| [Inverse](../WDLang1/3024028.md) | Renvoie pour chaque caractère d'une chaîne de caractères le complément à 255. |
| [JSONValide](../WDLang1/1410089147.md) | Teste la validité d'une chaîne au format JSON. |
| [JSONVersChaîne](../WDLang1/1000024530.md) | Convertit le contenu d'une variable de type JSON en une chaîne au format JSON. |
| [JSONVersVariant](../WDLang1/1000021052.md) | Convertit une chaîne au format JSON en une variable de type Variant. |
| [LigneVersPosition](../WDLang1/1000019896.md) | Renvoie la position du premier caractère d'une ligne dans un bloc de texte. |
| [Majuscule](../WDLang1/3024039.md) | Convertit une chaîne de caractères en majuscules en fonction des paramètres de localisation définis dans Windows. |
| [MarkdownVersDOCX](../WDLang1/1410089153.md) | Convertit une chaîne de caractères au format Markdown en un document au format DOCX. |
| [MarkdownVersHTML](../WDLang1/1410088874.md) | Convertit une chaîne de caractères au format Markdown en un buffer au format HTML (encodé en UTF8). |
| [MarkdownVersPDF](../WDLang1/1410089119.md) | Convertit une chaîne (ou un buffer) au format Markdown en un buffer contenant un document PDF. |
| [MarkdownVersTexte](../WDLang1/1410089001.md) | Convertit une chaîne de caractères au format Markdown en une chaîne de caractères au format texte (sans balises). |
| [Milieu](../WDLang1/3024023.md) | Extrait : <br><br>- soit une sous-chaîne d'une chaîne à partir d'une position donnée.<br><br>- soit une portion d'un buffer à partir d'une position donnée.<br><br><br> |
| [Minuscule](../WDLang1/3024030.md) | Convertit une chaîne de caractères en minuscules en fonction des paramètres de localisation définis dans Windows. |
| [MotOccurrence](../WDLang1/1000019897.md) | Renvoie le nombre de mots d'une chaîne. |
| [MotVersTableau](../WDLang1/1000019898.md) | Remplit un tableau avec les mots d'une chaîne. |
| [NombreEnLettres](../WDLang1/1000020632.md) | Convertit un nombre en une chaîne de caractères correspondant au nombre "écrit en toutes lettres". |
| [NumériqueVersChaîne](../WDLang1/3024024.md) | Transforme un numérique (entier, réel ou monétaire) en une chaîne de caractères selon le format spécifié. |
| [NumériqueVersEcritureFinancièreChinoise](../WDLang1/1000021282.md) | Transforme un numérique (entier, réel ou monétaire) en une chaîne au format d'écriture financière chinoise. |
| [OemVersAnsi](../WDLang1/3024021.md) | Convertit une chaîne de caractères au format OEM (DOS) en une chaîne de caractères au format ANSI (Windows). |
| [Phonétique](../WDLang1/3024026.md) | Renvoie la phonétique correspondant à la chaîne passée en paramètre. |
| [PhraseVersBuffer](../WDLang1/1000025971.md) | Renvoie le buffer encodé dans une phrase mnémotechnique BIP39. |
| [Position](../WDLang1/3024027.md) | Recherche la position d'une chaîne de caractères dans une autre chaîne de caractères. |
| [PositionOccurrence](../WDLang1/3024029.md) | Recherche la Xème position d'une chaîne de caractères dans une chaîne de caractères. |
| [PositionVersLigne](../WDLang1/1000019899.md) | Renvoie le numéro de la ligne à laquelle appartient un caractère connu par sa position dans un bloc de texte. |
| [PremièreLettreEnMajuscule](../WDLang1/1410088702.md) | Convertit en majuscule la première lettre de tous les mots d'une chaîne de caractères. |
| [RegexCherche](../WDLang1/1000026274.md) | Cherche toutes les parties d'une chaîne de caractères qui respectent un format donné. |
| [RegexRemplace](../WDLang1/1000026275.md) | Remplace toutes les parties d'une chaîne qui respectent un format donné. |
| [RegexVérifie](../WDLang1/1000026273.md) | Vérifie si une chaîne respecte un format défini par une expression régulière. |
| [Remplace](../WDLang1/3024022.md) | Remplace toutes les occurrences d'un mot présent dans une chaîne par un autre mot. |
| [Répète](../WDLang1/3024036.md) | Concatène N répétitions d'une même chaîne de caractères ou d'un même buffer. |
| [SansAccent](../WDLang1/3024033.md) | Transforme les caractères accentués d'une chaîne de caractères en caractères non accentués. |
| [SansCaractère](../WDLang1/1000019391.md) | Renvoie une chaîne de caractères sans les caractères spécifiés situés à gauche et à droite. |
| [SansCaractèreDroite](../WDLang1/1000020479.md) | Renvoie une chaîne de caractères sans les caractères spécifiés situés à droite de la chaîne initiale. |
| [SansCaractèreGauche](../WDLang1/1000020478.md) | Renvoie une chaîne de caractères sans les caractères spécifiés situés à gauche de la chaîne initiale. |
| [SansEspace](../WDLang1/3024034.md) | Renvoie une chaîne de caractères sans les espaces : <br><br>- situés à gauche et à droite.<br><br>- situés à l'intérieur de la chaîne.<br><br><br> |
| [Taille](../WDLang1/3024031.md) | Renvoie :<br><br>- la taille d'une chaîne de caractères, c'est-à-dire le nombre de caractères composant la chaîne de caractères (y compris les caractères espace et 0 binaire).<br><br>- la taille d'un buffer, c'est-à-dire le nombre d'octets composant le buffer.<br><br><br> |
| [TailleCommune](../WDLang1/3024045.md) | Renvoie le nombre de caractères communs à deux chaînes de caractères, en partant du début ou de la fin. |
| [TailleVersChaîne](../WDLang1/1000017038.md) | Convertit une taille donnée en octets en une chaîne de caractères formatée en kilo-octets, méga-octets ou tera-octets. |
| [TexteVersHTML](../WDLang1/3025050.md) | Convertit une chaîne texte en chaîne HTML (les retours chariots sont remplacés par des &lt;BR&gt;, les accents sont remplacés par leur équivalent en HTML, ...). |
| [TexteVersRTF](../WDLang1/1000020514.md) | Convertit une chaîne texte en chaîne au format RTF. |
| [Tronque](../WDLang1/1000020476.md) | Tronque la partie droite d'une chaîne ou d'un buffer. |
| [UnicodeVersAnsi](../WDLang1/3024025.md) | Convertit :<br><br>- une chaîne de caractères au format UNICODE en une chaîne de caractères au format ANSI (Windows).<br><br>- un buffer contenant une chaîne de caractères au format UNICODE en un buffer contenant une chaîne de caractères au format ANSI (Windows).<br><br><br> |
| [URLDécode](../WDLang1/3024041.md) | Décode une URL encodée (c'est-à-dire avec des caractères sous la forme de %xx). |
| [URLEncode](../WDLang1/3024040.md) | Encode une URL avec uniquement un sous-ensemble de caractères ASCII. |
| [URLExtraitChemin](../WDLang1/1000020965.md) | Renvoie les différents éléments d'une URL : protocole, utilisateur, mot de passe, domaine, port, chemin de la ressource, nom de la ressource, extension de la ressource, requête et fragment. |
| [UTF8VersAnsi](../WDLang1/1000020525.md) | Convertit une chaîne au format UTF8 en une chaîne Ansi. |
| [UTF8VersChaîne](../WDLang1/3024043.md) | Convertit une chaîne au format UTF8 en une chaîne ANSI ou une chaîne UNICODE. |
| [UTF8VersUnicode](../WDLang1/1000020481.md) | Convertit une chaîne au format UTF8 en une chaîne Unicode. |
| [UTFConvertit](../WDLang1/1410088734.md) | Convertit une chaîne ou un buffer vers le format spécifié : UTF, Ansi, Unicode. |
| [Val](../WDLang1/3024037.md) | Renvoie la valeur numérique d'une chaîne de caractères. |
| [VariantVersJSON](../WDLang1/1000021051.md) | Convertit le contenu d'une variable de type Variant au format JSON.  |
| [VérifieExpressionRégulière](../WDLang1/3024032.md) | Vérifie si une chaîne de caractères correspond à un format spécifique et récupère les différentes sous-chaînes constituant le format. |

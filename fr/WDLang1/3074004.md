


## TableAjuste (Fonction)

***En anglais : TableAdjust***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ajuste le titre et la taille des colonnes d'un champ Table ou Table hiérarchique. Il est possible de définir : 

- le titre d'une colonne,

- la largeur d'une colonne,

- le cadrage des éléments d'une colonne,

- la taille de saisie de la colonne.





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Modification des titres et de la largeur des deux
// premières colonnes du champ "TABLE_TableProduit"
TableAjuste(TABLE_TableProduit, "Réf" + TAB + "50" + TAB + ...
	"C" + TAB + "12" + RC + "Désignation" + TAB + "200" + ...
	TAB + "G" + TAB + "12")
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`TableAjuste(<Champ Table> [, <Format>])`
---

**`<Champ Table> : (Nom de champ)`**

Nom du champ à manipuler. Ce champ peut correspondre : 

- à un champ Table, 

- à un champ Table hiérarchique.


Si ce paramètre correspond à une chaîne vide (""), le champ manipulé sera le champ auquel appartient l'événement en cours.

**`<Format> : (Chaîne de caractères optionnelle)`**

Ce paramètre est de la forme : 

```txt
<Titre de la colonne 1> + TAB + <Taille de la colonne 1> + TAB + ...
<Cadrage des éléments> + TAB + <Taille de saisie> + RC + ...
...
<Titre de la colonne N> + TAB + <Taille de la colonne N> + TAB + ...
<Cadrage des éléments> + TAB + <Taille de saisie>
```
où :


|   |   |   |
| --- | --- | --- |
| &lt;Titre de la colonne&gt; | Chaîne de caractères | Modifie le titre de la colonne. |
| &lt;Taille de la colonne&gt; | Entier | Modifie la taille de la colonne (en caractères). |
| &lt;Cadrage des éléments&gt; | Caractère | Modifie le cadrage des éléments de la colonne :<br><br>- "C" : Centre les éléments dans la colonne<br><br>- "G" : Cadre les éléments à gauche<br><br>- "D" : Cadre les éléments à droite<br><br><br> |
| &lt;Taille de saisie&gt; | Entier | Modifie le nombre de caractères pouvant être saisis dans la colonne (en nombre de caractères et non en pixels). <br>Remarque : La largeur d'un caractère correspond à la largeur du caractère 'E' dans la police du titre du champ. |


Si ce paramètre n'est pas spécifié, la largeur des colonnes est ajustée à la plus petite largeur possible pour que l'ensemble des données soit visible.
![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Ce paramètre est obligatoire.

## Remarques


### Conditions d'utilisation
<a name="conditions_utilisation_ELTPARAGRAPHE000113"></a>La fonction **TableAjuste** peut être utilisée indifféremment sur :

- un champ Table ou Table hiérarchique fichier.

- un champ Table ou Table hiérarchique par programmation.

- un champ monosélection ou multisélection.



<a name="NOTE0_2"></a>


### Création/Suppression de colonnes
<a name="creationsuppression_colonnes_ELTPARAGRAPHE000129"></a>La fonction **TableAjuste** ne crée pas et ne supprime pas de colonnes. Les colonnes du champ doivent être préalablement décrites sous l'éditeur de fenêtres. Assurez-vous qu'un nombre suffisant de colonnes existe avant de les manipuler avec la fonction **TableAjuste**.

Si le nombre de colonnes présentes dans le champ est supérieur au nombre de colonnes manipulées avec la fonction **TableAjuste**, les dernières colonnes seront sans titre et de taille nulle.
<a name="NOTE0_3"></a>


### Colonnes invisibles
<a name="colonnes_invisibles_ELTPARAGRAPHE000149"></a>**Si les colonnes manipulées avec la fonction TableAjuste** sont invisibles, ces colonnes deviennent automatiquement visibles.
<a name="NOTE0_4"></a>


### Équivalence de la fonction TableAjuste avec les propriétés des champs
<a name="equivalence_fonction_docparampagetitleshort_avec_les_proprietes_des_champs_ELTPARAGRAPHE000161"></a>Les propriétés [Titre](../Proprietes/2510090.md), [Largeur](../Proprietes/2510059.md), [Taille](../Proprietes/2510125.md) et [CadrageHorizontal](../Proprietes/2510023.md) ont le même effet que la fonction **TableAjuste**.

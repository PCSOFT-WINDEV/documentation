


## PlanningChangeMode (Fonction)

***En anglais : SchedulerChangeMode***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie la période (c'est-à-dire le nombre de jours) affichée par un champ Planning.



![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) En mode Universal Windows 10 App, cette fonction permet uniquement de manipuler les champs Planning présents dans un état. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Clic sur BTN_Jour
PlanningChangeMode(PLN_Planning, plnModeJournée)

// Clic sur BTN_Semaine
PlanningChangeMode(PLN_Planning, plnModeSemaine)

// Clic sur BTN_Mois
PlanningChangeMode(PLN_Planning, plnModeMois)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`PlanningChangeMode(<Champ Planning> , <Mode d'affichage> [, <Nombre de jours>])`
---

**`<Champ Planning> : (Nom de champ)`**

Nom du champ Planning manipulé. Ce champ peut correspondre : 

- au champ Planning d'une fenêtre, 

- au champ Planning d'une page,

- au champ Planning d'un état.




**`<Mode d'affichage> : (Constante de type Entier)`**

Plage de temps à afficher dans le champ Planning : 


|   |   |
| --- | --- |
| *plnModeJournée* | Affiche le champ Planning sur une journée. |
| *plnModeMois* | Affiche le champ Planning sur un mois. |
| *plnModeNbJourPersonnalisé* | Affiche le champ Planning sur le nombre de jours indiqué dans le paramètre &lt;Nombre de jours&gt;. |
| *plnModeSelonRdv* | Affiche le champ Planning en utilisant un nombre de jours permettant de voir tous les rendez-vous disponibles.<br><br> |
| *plnModeSemaine* | Affiche le champ Planning sur une semaine de 5 jours ou sur une semaine de 7 jours. Le nombre de jours est indiqué dans le paramètre &lt;Nombre de jours&gt;. Le premier jour affiché est le premier jour de la semaine défini dans la fenêtre de description du champ Planning. |



**`<Nombre de jours> : (Entier optionnel)`**

Nombre de jours affichés. Ce paramètre est pris en compte uniquement si le paramètre &lt;Mode d'affichage&gt; correspond à la constante *plnModeNbJourPersonnalisé* ou à la constante *plnModeSemaine*. Dans le cas contraire, une erreur WLangage est affichée.

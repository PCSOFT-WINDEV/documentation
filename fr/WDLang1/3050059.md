
## MatRéelAdditionne (Fonction)

***En anglais : MatFloatAdd***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Additionne une valeur à chaque élément d'une matrice. Cette addition est réalisée directement dans la matrice spécifiée.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple
<a class="notetitle" target="_blank" href="$DOC$=1000003050059&name=matreeladditionne_fonction&product=WD">Voir des exemples supplémentaires</a>

```wl
// Addition de la valeur 3 à une matrice
MatRéelAdditionne("MaMatrice", 3)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`MatRéelAdditionne(<Matrice> , <Valeur>)`
---

**`<Matrice> : (Chaîne de caractères)`**

Nom de la matrice à manipuler, défini avec la fonction [MatCrée](../WDLang1/3050058.md).

**`<Valeur> : (Réel)`**

Valeur à ajouter à la matrice.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Existence de la matrice
<a name="existence_matrice_ELTPARAGRAPHE000077"></a>L'existence de la matrice n'est pas vérifiée. Pour vérifier l'existence d'une matrice, utilisez la fonction [MatExiste](../WDLang1/3050076.md).


<a name="NOTE0_2"></a>




### Gestion des erreurs
<a name="gestion_des_erreurs_ELTPARAGRAPHE000089"></a>**Attention** : La fonction **MatRéelAdditionne** ne retourne pas de code d'erreur. Pour savoir si des erreurs ont été générées, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.

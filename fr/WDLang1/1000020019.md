
## HeureSélecteur (Fonction)

***En anglais : TimePicker***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Affiche le sélecteur d'heure du système.
<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=HeureSelecteur.gif)


**Remarque** : Pour sélectionner une heure à l'aide du rouleau, il est conseillé d'utiliser directement le **masque de saisie de type "Rouleau Heure"**pour le champ de saisie de type "Heure". Ce masque de saisie est également disponible pour les applications iPhone/iPad. 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
h est une Heure = HeureSélecteur()
```



```wl
HeureDébut est une Heure = "0730"
HeureSélectionnée est une Heure = HeureSélecteur(HeureDébut, Vrai, "Saisissez l'heure d'arrivée")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = HeureSélecteur([<Heure initiale> [, <Mode 24 heures> [, <Titre>]]])`
---

**`<Résultat> : (Heure ou chaîne de caractères)`**



- Heure sélectionnée dans le sélecteur, 

- Chaîne vide ("") si la sélection a été annulée par l'utilisateur.




**`<Heure initiale> : (Heure ou chaîne de caractères optionnelle)`**

Heure affichée à l'ouverture du sélecteur. Si cette heure n'est pas spécifiée, le sélecteur affichera l'heure courante.

**`<Mode 24 heures> : (Booléen optionnel)`**



- <u><u><u><u>Vrai</u></u></u></u> (valeur par défaut) pour afficher l'heure au format 24 heures.

- <u><u><u><u>Faux</u></u></u></u> pour afficher l'heure au format AM/PM. 




**`<Titre> : (Chaîne de caractères optionnelle)`**

Titre à afficher dans le sélecteur. Si ce paramètre n'est pas spécifié, le titre affiché sera une représentation textuelle de l'heure affichée par le sélecteur.

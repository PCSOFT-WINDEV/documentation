
## AppliDéclencheMAJ (Fonction)

***En anglais : AppInstallUPD***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Déclenche la mise à jour automatique de l'application en cours.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI AppliMAJDisponible() = appMAJDispoObligatoire ALORS
	SI OuiNon("Une mise à jour de l'application est disponible.", ...
		  "Voulez-vous effectuer cette mise à jour ?") = Oui ALORS
			  AppliDéclencheMAJ()
	FIN
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = AppliDéclencheMAJ()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la mise à jour est déclenchée, 

- <u><u><u><u>Faux</u></u></u></u> si la mise à jour n'a pas été déclenchée. Pour plus de détails sur le problème rencontré, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md). 




Si la mise à jour se déclenche, l'application se termine puis la mise à jour de l'application est exécutée.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Lors de l'exécution de la fonction **AppliDéclencheMAJ** :

- l'application en cours est fermée. Le code de fermeture du projet est exécuté.

- l'application est relancée (sans ligne de commande). La mise à jour est détectée et exécutée.

- l'application est relancée une nouvelle fois à la fin de la mise à jour, sans ligne de commande.

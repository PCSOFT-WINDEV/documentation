


## TableIndiceColonne (Fonction)

***En anglais : TableColumnIndex***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie l'indice d'une colonne dans un champ Table ou Table hiérarchique.

<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) 
```wl
// Indice de la colonne "Réf" du champ Table "TABLE_Produit"
// L'ordre d'affichage est pris en compte
ResIndiceColonne = TableIndiceColonne(TABLE_Produit, "Réf", ticAffichage)
```


<a name="Exemple2"></a>

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = TableIndiceColonne(<Champ Table> , <Colonne> , <Ordre>)`
---

**`<Résultat> : (Entier)`**

Indice de la colonne : 

- La première colonne possède l'indice 1. 

- Les colonnes invisibles sont également listées.




**`<Champ Table> : (Nom de champ)`**

Nom du champ Table ou Table hiérarchique à manipuler.

Si ce paramètre correspond à une chaîne vide (""), la fonction **TableIndiceColonne** manipule le champ auquel appartient l'événement en cours.

**`<Colonne> : (Chaîne de caractères)`**

Nom de la colonne à manipuler.

**`<Ordre> : (Constante de type Entier)`**

Indique l'ordre des colonnes considérées :


|   |   |
| --- | --- |
| *ticAffichage* | Ordre d'affichage des colonnes en exécution. |
| *ticCréation* | Ordre de création des colonnes lors de la description du champ Table. |

## Remarques


### Conditions d'utilisation
<a name="conditions_utilisation_ELTPARAGRAPHE000134"></a>La fonction **TableIndiceColonne** peut être utilisée indifféremment sur :

- un champ Table ou Table hiérarchique fichier.

- un champ Table ou Table hiérarchique par programmation.

- un champ monosélection ou multisélection.


## FenEdSelectPlus (Fonction)

***En anglais : WinEdSelectPlus***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Sélectionne un champ lors de la modification de l'interface par l'utilisateur final (mode "Modification de l'UI par l'utilisateur final" activé).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ajout d'un champ et sélection du champ
X est un Champ
X -> ChampCrée("LIB" + DonneIdentifiant(), typLibellé, 10, 10, 100, 24, Vrai)
X.Libellé = "Nouveau libellé"
FenEdSelectPlus(FEN_Formulaire, X)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`FenEdSelectPlus(<Fenêtre> [, <Champ manipulé>])`
---

**`<Fenêtre> : (Nom de fenêtre)`**

Nom de la fenêtre ou de la fenêtre interne à manipuler. Cette fenêtre doit être en mode modification (fonction [FenEdActive](../WDLang1/1000021843.md)). 

**`<Champ manipulé> : (Champ optionnel)`**

Champ à sélectionner dans la fenêtre. 

- Si la sélection multiple n'est pas activée, la sélection en cours est remplacée par le champ spécifié. 

- Si la sélection multiple est activée, le champ spécifié est ajouté à la sélection. 


Si le champ spécifié n'est pas sélectionné, la fonction n'a aucun effet.

Si ce paramètre n'est pas spécifié : 

- si la sélection multiple est autorisée, tous les champs de la fenêtre sont sélectionnés.

- si la sélection multiple n'est pas autorisée, la fonction n'a aucun effet.   






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Le champ sélectionné peut être obtenu à l'aide de la fonction [FenEdSelect](../WDLang1/1000021871.md).  

- Si la multisélection est autorisée, il est possible de connaître le nombre de champs sélectionnés via la fonction [FenEdSelectOccurrence](../WDLang1/1000021870.md) et de les parcourir à l'aide de la fonction [FenEdSelect](../WDLang1/1000021871.md).

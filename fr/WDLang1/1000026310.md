


## Fonctions de gestion du champ Editeur HTML (syntaxe préfixée)



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions WLangage permettant de gérer les champs Editeur HTML (syntaxe préfixée) sont les suivantes : 



|   |   |
| --- | --- |
| [&lt;Editeur HTML&gt;.DepuisEmail](../WDLang1/1000026296.md) | Importe le contenu HTML d'un email et ses images dans un champ Editeur HTML. Les images contenues dans l'email sont également affichées dans le champ. |
| [&lt;Editeur HTML&gt;.ExécuteJS](../WDLang1/1000026297.md) | Exécute du code Javascript dans un champ Editeur HTML. |
| [&lt;Editeur HTML&gt;.FormateSelection](../WDLang1/1000026298.md) | Modifie le style de la sélection dans un champ Editeur HTML. |
| [&lt;Editeur HTML&gt;.Insère](../WDLang1/1000026299.md) | Insère du texte ou du code HTML à la position du curseur dans un champ Editeur HTML. |
| [&lt;Editeur HTML&gt;.InsèreTableau](../WDLang1/1000026300.md) | Insère un tableau à la position du curseur dans un champ Editeur HTML. |
| [&lt;Editeur HTML&gt;.Recherche](../WDLang1/1000026301.md) | Recherche une chaîne de caractères dans un champ Editeur HTML. |
| [&lt;Editeur HTML&gt;.Récupère](../WDLang1/1000026302.md) | Récupère en mode asynchrone la valeur d'une des propriétés d'un champ Editeur HTML. |
| [&lt;Editeur HTML&gt;.RécupèreDocument](../WDLang1/1000026304.md) | Récupère en mode asynchrone le document HTML en cours dans un champ Editeur HTML. |
| [&lt;Editeur HTML&gt;.RécupèreSélection](../WDLang1/1000026305.md) | Récupère en mode asynchrone la sélection effectuée dans un champ Editeur HTML. |
| [&lt;Editeur HTML&gt;.Remplace](../WDLang1/1000026306.md) | Remplace une chaîne de caractères par une autre dans un champ Editeur HTML. |
| [&lt;Editeur HTML&gt;.Sélectionne](../WDLang1/1000026307.md) | Sélectionne une portion de texte dans un champ Editeur HTML. |
| [&lt;Editeur HTML&gt;.VersEmail](../WDLang1/1000026308.md) | Exporte le contenu d'un champ Editeur HTML vers une variable de type Email (pour envoyer l'email). |
| [&lt;Editeur HTML&gt;.VersPDF](../WDLang1/1000026309.md) | Exporte le contenu d'un champ Editeur HTML vers un fichier PDF. |

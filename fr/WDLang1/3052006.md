
## MDIMère (Fonction)

***En anglais : MDIMother***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Identifie le nom de la fenêtre mère MDI.

**Remarque** : La fonction **MDIMère** est utilisable uniquement pour les fenêtres définies comme "mères MDI" sous l'éditeur de fenêtres.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Identification de la fenêtre mère MDI
ResFenêtreMère = MDIMère()
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = MDIMère()`
---

**`<Résultat> : (Chaîne de caractères)`**



- Nom de la fenêtre mère MDI, 

- Chaîne vide ("") si aucune fenêtre mère MDI n'existe.

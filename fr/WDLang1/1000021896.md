
## SysRécupèreAssociationExt (Fonction)

***En anglais : SysGetExtLink***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la commande associée à une extension. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Associe l'extension ".txt" à l'exécutable "app.exe"
SysAssocieExtension(".txt", "C:\Program Files\MyApp\app.exe ""%1""", sysUtilisateurCourant)
...
SysRécupèreAssociationExt(".txt")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = SysRécupèreAssociationExt(<Extension> [, <Utilisateur>])`
---

**`<Résultat> : (Chaîne de caractères)`**



- Commande associée à l'extension, 

- Chaîne vide ("") si aucune commande n'est associée à l'extension. 




**`<Extension> : (Chaîne de caractères)`**

Extension à manipuler. 

**`<Utilisateur> : (Constante optionnelle de type Entier)`**

L'association entre l'application et l'extension est enregistrée dans la base de registres. Deux options sont possibles : 


|   |   |
| --- | --- |
| *sysTousUtilisateurs* | L'association est recherchée pour tous les utilisateurs du poste en cours. <br><br>Remarque : Il est nécessaire de posséder les droits "Administrateur" pour exécuter cette opération. |
| *sysUtilisateurCourant*<br>(Valeur par défaut) | L'association est recherchée uniquement pour l'utilisateur courant. |

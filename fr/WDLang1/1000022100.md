
## FAAChangeLibellé (Fonction)

***En anglais : AAFChangeCaption***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Personnalise un libellé d'une option de menu pour les menus contextuels correspondant à une FAA (Fonctionnalité Automatique de vos Applications). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
FAAChangeLibellé(faaTableVersExcel, "Enregistrer dans un fichier Excel")

FAAChangeLibellé(ONG_DOC, faaOngletNouveau, "Nouveau document")
FAAChangeLibellé(ONG_CLIENT, faaOngletNouveau, "Nouveau client")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

---

**`<Champ ou fenêtre> : (Chaîne de caractères optionnelle)`**

Nom d'un champ ou d'une fenêtre ouverte. 

Si ce paramètre n'est pas précisé, la modification est réalisée pour tous les champs et toutes les fenêtres.


## SupprimeParamètre (Fonction)

***En anglais : DeleteParameter***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime un paramètre ou une série de paramètres sauvés : 

- soit avec la fonction [SauveParamètre](../WDLang1/3025046.md), 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) soit automatiquement grâce à la [persistance des données](../WDChamp/9000019.md) dans les champs.




**Attention** : Cette opération est irréversible et ne peut pas être annulée.






<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Suppression d'un paramètre (syntaxe 1)
SupprimeParamètre("MonParamètre")
```


<a name="Exemple2"></a>


![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) 
```wl
// Suppression d'un paramètre (syntaxe 1)
SupprimeParamètre("FEN_Fenêtre1\SAI_Saisie1")
```


<a name="Exemple3"></a>


![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) 
```wl
// Suppression des paramètres de la fenêtre "FEN_Fen1" (syntaxe 2)
SupprimeParamètre(FEN_Fen1)
// équivalent à : SupprimeParamètre(FEN_Fen1..CheminComplet)
```

<a name="SYNTAXE1"></a>

## Syntaxe

### Supprimer un paramètre sauvé

`<Résultat> = SupprimeParamètre(<Nom du paramètre>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la fonction a réussi, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. En cas d'erreur, il est possible de connaître le détail de l'erreur grâce à la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Nom du paramètre> : (Chaîne de caractères)`**

Nom du paramètre à supprimer. Ce paramètre est recherché : 

- soit dans la base de registre (ou le fichier équivalent sur la plateforme d'exécution),

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) soit dans le fichier de paramètre spécifié avec la fonction [InitParamètre](../WDLang1/3025044.md). 


Ce nom peut être une valeur ou un chemin complet (les séparateurs acceptés sont le point (.) et le slash (\\)). Dans le cas d'un chemin complet, toutes les valeurs situées "en dessous" de ce chemin seront supprimées (voir exemple dans les notes).

![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Si le mode d'initialisation choisi est la base de registre, la fonction **SupprimeParamètre** n'est sensible ni à la casse, ni à l'accentuation du nom du paramètre (Paramètre et parametre sont équivalents).


<a name="SYNTAXE2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) 
### Supprimer tous les paramètres sauvés pour l'élément spécifié

`<Résultat> = SupprimeParamètre(<Nom de l'élément>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la fonction a réussi, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. En cas d'erreur, il est possible de connaître le détail de l'erreur grâce à la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Nom de l'élément> : (Chaîne de caractères)`**

Nom du champ ou de la fenêtre dont les paramètres sauvegardés doivent être supprimés. Cet élément est recherché dans le fichier de paramètre spécifié avec la fonction [InitParamètre](../WDLang1/3025044.md) (par défaut, dans la base de registre).  


<a name="SYNTAXE3"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) 
### Supprimer tous les paramètres sauvegardés

`<Résultat> = SupprimeParamètre()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la fonction a réussi, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. En cas d'erreur, il est possible de connaître le détail de l'erreur grâce à la fonction [ErreurInfo](../WDLang1/3013008.md). 




Tous les paramètres sauvegardés pour l'application sont effacés. Au prochain lancement, l'application se comportera comme à son premier lancement.
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) Le fichier de paramètre de l'application spécifié avec la fonction [InitParamètre](../WDLang1/3025044.md) est entièrement vidé. Si les paramètres ont été sauvegardés dans un fichier INI ou XML, ce fichier est supprimé.




## Fonctions Hasp
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000083"></a>
Les clés électroniques de la famille Hasp (Hasp3, Hasp4, NetHasp, MemoHasp et Hasp HL) permettent de protéger les applications développées avec WINDEV ou WEBDEV.

L'utilisation des fonctions Hasp permet de restreindre l'utilisation d'une application aux seuls détenteurs d'une clé électronique valide.

Une autre méthode de protection consiste à utiliser [les fonctions de gestion des clés d'activation](../WDLang1/1000018998.md).

**Remarques** : 

- Pour utiliser les fonctions Hasp, il faut connaître le fonctionnement des clés Hasp et avoir leur documentation.

- La fonction [Hasp](../WDLang1/3013038.md) n'est pas disponible en 64 bits, utilisez plutôt la famille de fonctions HaspHLxxx.

<a name="NOTE2_1"></a>


## Fonctions Hasp
<a name="fonctions_hasp_ELTTEXTE000107"></a>
Les fonctions hasp sont les suivantes :



|   |   |
| --- | --- |
| [Hasp](../WDLang1/3013038.md) | Interroge une clé électronique HASP de type HASP-3, HASP-4, MemoHasp ou NetHasp. |
| [HaspHLConnecte](../WDLang1/1000018891.md) | Connecte l'application à une clé Hasp. |
| [HaspHLCrypte](../WDLang1/1000018893.md) | Crypte une chaîne de caractères en utilisant les algorithmes de la clé Hasp spécifiée. |
| [HaspHLDéconnecte](../WDLang1/1000018892.md) | Ferme la connexion de l'application à une clé Hasp. |
| [HaspHLDécrypte](../WDLang1/1000018894.md) | Décrypte une chaîne de caractères en utilisant les algorithmes de la clé Hasp. |
| [HaspHLEcrit](../WDLang1/1000018897.md) | Ecrit des données (chaînes de caractères ou buffer) dans la mémoire interne de la clé Hasp. |
| [HaspHLInfo](../WDLang1/1000018895.md) | Récupère des informations sur la clé électronique Hasp. |
| [HaspHLLit](../WDLang1/1000018896.md) | Lit des données (chaînes de caractères ou buffer) dans la mémoire interne de la clé Hasp. |

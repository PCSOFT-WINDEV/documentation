


## Fonctions de gestion du multilingue
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions permettant de gérer les applications multilingues sont les suivantes :  



|   |   |
| --- | --- |
| [ChangeAlphabet](../WDLang1/3054001.md) | Récupère et/ou change l'alphabet utilisé :<br><br>- par toutes les polices écran.<br><br>- par toutes les opérations sur les chaînes de caractères (fonctions et/ou opérateurs).<br><br><br> |
| [ChangeClavier](../WDLang1/3054002.md) | Récupère et/ou modifie la langue associée au clavier. |
| [ChangeSéparateur](../WDLang1/3054004.md) | Change la valeur d'un des séparateurs (de décimales, ou de milliers) dans les masques des numériques pour l'internationalisation. |
| [ChargeErreur](../WDLang1/3054003.md) | Charge en mémoire le fichier contenant les messages d'erreur dans la langue souhaitée. |
| [LangueVersAlphabet](../WDLang1/3054018.md) | Renvoie l'alphabet correspondant à une langue. |
| [LangueVersNation](../WDLang1/3054019.md) | Renvoie la nation correspondant à une langue. |
| [LangueVersNom](../WDLang1/3054020.md) | Renvoie le nom d'une langue. |
| [MessTraduit](../WDLang1/3054015.md) | Récupère un message présent dans le code d'un programme dans la langue en cours. |
| [Nation](../WDLang1/3054014.md) | Définit ou identifie la langue d'affichage (libellés, messages, ...) utilisée pour les fenêtres, les pages, les états et les champs. |
| [NationDroiteAGauche](../WDLang1/1000020820.md) | Permet de savoir si la langue courante s'écrit de droite à gauche. |
| [NationFramework](../WDLang1/1000024376.md) | Définit ou identifie la langue du framework. |
| [NationVersLangue](../WDLang1/3054021.md) | Renvoie la langue correspondant à une nation. |
| [NationVersNom](../WDLang1/1000022324.md) | Renvoie le nom de la langue correspondant à une nation. |
| [NationVersSousLangue](../WDLang1/1000022209.md) | Renvoie la sous-langue par défaut correspondant à une nation. |






- ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=WDListeChainee.gif) ***Exemples didactiques (WINDEV)*** : **WD ListeChaînée** <br>Cet exemple fournit les outils pour construire des listes chaînées à chaînage simple. Une liste à chaînage simple permet de stocker et parcourir de manière ordonnée une collection de valeurs quelconques.<br><br>Dans l'exemple, qui est autoexplicatif, la liste chaînée permet de stocker des boutons. <br>- La liste est ensuite utilisée pour masquer/déplacer les boutons dans l'écran, créant un menu dynamique de type Sidebar.<br>- La liste est aussi utilisée pour traduire les boutons - à la volée - c'est à dire sans besoin de recharger la fenêtre. Pour cela, on stocke aussi pour chaque bouton le libellé français et le libellé anglais. L'avantage de procéder ainsi est de conserver l'état de la fenêtre, notamment le menu dans son aspect déroulé, le texte explicatif, etc .<br><br>Particularité de cet exemple:<br>- il est totalement bilingue (français / anglais).

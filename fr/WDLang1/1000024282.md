


## &lt;Table&gt;.SaisieLoupe (Fonction)

***En anglais : &lt;Table&gt;.InputSearch***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Passe en saisie la zone de recherche présente dans le titre de colonne du champ Table ou Table hiérarchique.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Sous WINDEV : ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=TableSaisieLoupe1.gif)![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=TableSaisieLoupe2.gif)




Cette fonction a la même action qu'un clic de l'utilisateur sur l'icône représentant une loupe. Elle peut lancer une recherche ou un filtre sur le champ.

Pour annuler un filtre lancé avec la fonction **&lt;Table&gt;.SaisieLoupe**, il suffit d'utiliser la fonction [&lt;Table&gt;.DésactiveFiltre](../WDLang1/1000024246.md).

**Remarque** : Cette fonction est ignorée si le champ Table spécifié utilise un ascenseur non proportionnel (option "Ascenseur proportionnel" décochée dans l'onglet "Détail" de la fenêtre de description du champ). Pour plus de détails sur le fonctionnement de la loupe, consultez [Caractéristiques des colonnes d'un champ Table](../WDChamp/1013333.md).




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Passe la colonne COL_Nom en mode saisie
COL_NOM.SaisieLoupe()
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Colonne>.SaisieLoupe([<Type de filtre ou de recherche> [, <Valeur initiale>]])`
---

**`<Colonne> : (Nom de champ)`**

Nom de la colonne à passer en mode de recherche. Si l'opération ne peut pas être réalisée sur cette colonne (colonne non clé par exemple), la fonction ne fait rien.

Le nom de la colonne peut être donné directement ou sous la forme &lt;Champ Table&gt;.&lt;Colonne&gt;.
![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) La colonne manipulée doit être de type "Avec recherche" (onglet "Général" de la description de la colonne).

**`<Type de filtre ou de recherche> : (Constante optionnelle de type Entier)`**

Type de filtre à appliquer ou de recherche à effectuer :


|   |   |
| --- | --- |
| *filtreCommencePar* | Affiche les lignes dont la valeur de la colonne spécifiée "Commence par" les caractères indiqués dans la &lt;Valeur initiale&gt;. |
| *filtreContient* | Affiche les lignes dont la valeur de la colonne spécifiée "Contient" les caractères indiqués dans la &lt;Valeur initiale&gt;. |
| *filtreDifférent* | Affiche les lignes dont la valeur de la colonne spécifiée "est différente" des caractères indiqués dans la &lt;Valeur initiale&gt;. |
| *filtreEgal* | Affiche les lignes dont la valeur de la colonne spécifiée "est égale" aux caractères indiqués dans la &lt;Valeur initiale&gt;. |
| *filtreInférieur* | Affiche les lignes dont la valeur de la colonne spécifiée "est inférieure" aux caractères indiqués dans la &lt;Valeur initiale&gt;. |
| *filtreInférieurOuEgal* | Affiche les lignes dont la valeur de la colonne spécifiée "est inférieure ou égale" aux caractères indiqués dans la &lt;Valeur initiale&gt;. |
| *filtreNeCommencePasPar* | Affiche les lignes dont la valeur de la colonne spécifiée "Ne commence pas par" les caractères indiqués dans la &lt;Valeur initiale&gt;. |
| *filtreNeContientPas* | Affiche les lignes dont la valeur de la colonne spécifiée "ne contient pas" les caractères indiqués dans la &lt;Valeur initiale&gt;. |
| *filtreNeTerminePasPar* | Affiche les lignes dont la valeur de la colonne spécifiée "ne termine pas par" les caractères indiqués dans la &lt;Valeur initiale&gt;. |
| *filtreSupérieur* | Affiche les lignes dont la valeur de la colonne spécifiée "est supérieure" aux caractères indiqués dans la &lt;Valeur initiale&gt;. |
| *filtreSupérieurOuEgal* | Affiche les lignes dont la valeur de la colonne spécifiée "est supérieure ou égale" aux caractères indiqués dans la &lt;Valeur initiale&gt;. |
| *filtreTerminePar* | Affiche les lignes dont la valeur de la colonne spécifiée "termine par" les caractères indiqués dans la &lt;Valeur initiale&gt;. |
| *rechercheCommencePar* | Trie la table selon la colonne spécifiée, passe la colonne en mode recherche et rempli le champ de saisie de la colonne avec les caractères indiqués dans la &lt;Valeur initiale&gt;. Affiche les lignes à partir de celles pour lesquelles la valeur de la colonne spécifiée "commence par" les caractères indiqués dans la &lt;Valeur initiale&gt;. |
| *rechercheDéfaut*<br>(Valeur par défaut) | Trie la table selon la colonne spécifiée et passe la colonne en mode recherche. |


![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Ce paramètre n'est pas disponible.

**`<Valeur initiale> : (Chaîne de caractères optionnelle)`**

Valeur initiale à indiquer dans la zone de saisie du mode recherche.

![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Ce paramètre n'est pas disponible.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Effet de la fonction
<a name="effet_fonction_ELTPARAGRAPHE000299"></a>Cette fonction n'aura aucun effet si le passage en mode de recherche ne peut pas être effectué. Par exemple, le mode de recherche ne sera pas activé :

- Si la colonne n'a pas été définie en tant que colonne avec recherche (fenêtre de description du champ).

- Si la colonne n'est pas associée à une rubrique de type clé.



<a name="NOTE0_2"></a>


### Conditions d'utilisation
<a name="conditions_utilisation_ELTPARAGRAPHE000312"></a>Cette fonction est disponible sur : 

- un champ Table ou Table hiérarchique fichier.

- un champ Table ou Table hiérarchique par programmation.

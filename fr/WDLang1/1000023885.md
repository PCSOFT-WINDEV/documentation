


## &lt;Fenêtre&gt;.ImageFenêtre (Fonction)

***En anglais : &lt;Window&gt;.WindowBitmap***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Crée l'image de la fenêtre spécifiée dans un fichier BMP.

**Attention** : L'image de la fenêtre correspond à la fenêtre telle qu'elle apparaît sous l'éditeur de fenêtres. Pour réaliser une image de la fenêtre en exécution, utilisez la fonction [dCopieImageFenêtre](../WDLang1/1000019912.md) et le type [Image](../WDLang1/1000019650.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Création des images de la fenêtre "FEN_SaisieClient"
// dans le fichier "C:\Images\SaisieClient"
Res = FEN_SaisieClient.ImageFenêtre("C:\Images\SaisieClient")
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Fenêtre>.ImageFenêtre(<Fichier image>)`
---

**`<Résultat> : (Entier)`**

Nombre d'images créées. Pour la fenêtre spécifiée, la fonction **&lt;Fenêtre&gt;.ImageFenêtre** crée autant d'images (fichiers BMP) que la fenêtre possède de plans. Si la fenêtre possède des onglets, une seule image est créée.

**`<Fenêtre> : (Nom de fenêtre)`**

Nom de la fenêtre à manipuler.

**`<Fichier image> : (Chaîne de caractères)`**

Nom et chemin complet (ou relatif) du fichier résultat (sans préciser l'extension). Un chemin de type UNC peut être utilisé.

Si un seul fichier image est créé, son extension sera ".BMP".

Si plusieurs fichiers images sont créés, ils auront pour extension ".B01", ".B02", ...



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
L'image est créée avec les informations système en cours : couleurs, polices, ...

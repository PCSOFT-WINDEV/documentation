
## VariantConvertit (Fonction)

***En anglais : VariantConvert***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Convertit la valeur stockée dans une variable de [type Variant](../Motscles/1514019.md).










<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
Procédure MaProcédure(Paramètre)
// On récupère le paramètre dans un variant
vVariant est un variant = Paramètre
// On convertit le variant en chaîne
VariantConvertit(vVariant, wlChaîne)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = VariantConvertit(<Variant> , <Type>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'opération a été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> en cas d'erreur. Pour plus de détails sur le problème rencontré, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Variant> : (Variant)`**

Nom de la variable de type variant.

**`<Type> : (Constante de type entier)`**

Identifie le type dans lequel le contenu de la variable variant doit être converti.


|   |   |
| --- | --- |
| *wlBooléen* | Booléen |
| *wlCaractère* | Caractère |
| *wlChaîne* | Chaîne de caractères |
| *wlChaîneBinaire* | Chaîne binaire |
| *wlDate* | Date |
| *wlDateHeure* | DateHeure |
| *wlDurée* | Durée |
| *wlEntier* | Entier signé sur 4 octets |
| *wlHeure* | Heure |
| *wlMonétaire* | Monétaire |
| *wlRéel* | Réel sur 8 octets |

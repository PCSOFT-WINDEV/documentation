


## Fonctions Tableur (syntaxe préfixée)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions (syntaxe préfixée) permettant de manipuler le champ Tableur sont les suivantes : 



|   |   |
| --- | --- |
| [&lt;Tableur&gt;.AjouteDonnée](../WDLang1/1000023711.md) | Ajoute une donnée dans un champ Tableur présent dans une fenêtre. |
| [&lt;Tableur&gt;.AjouteFeuille](../WDLang1/1000023712.md) | Ajoute une feuille de calcul dans un champ Tableur présent dans une fenêtre. |
| [&lt;Tableur&gt;.AjouteFormule](../WDLang1/1000023713.md) | Ajoute une formule de calcul dans un champ Tableur présent dans une fenêtre. |
| [&lt;Tableur&gt;.AjouteImage](../WDLang1/1000023714.md) | Ajoute une image dans la feuille en cours d'un champ Tableur. |
| [&lt;Tableur&gt;.AjouteNom](../WDLang1/1000023715.md) | Spécifie un nom pour identifier une cellule d'un champ Tableur. |
| [&lt;Tableur&gt;.AutoriseProcédure](../WDLang1/1000023716.md) | Autorise l'utilisation d'une fonction/procédure WLangage dans une formule d'un champ Tableur. |
| [&lt;Tableur&gt;.CadreSélection](../WDLang1/1000023717.md) | Modifie le cadre des cellules sélectionnées dans un champ Tableur présent dans une fenêtre. |
| [&lt;Tableur&gt;.ChampFormule](../WDLang1/1000023718.md) | Permet d'utiliser un champ pour réaliser une barre de formule personnalisée pour le champ Tableur. |
| [&lt;Tableur&gt;.Charge](../WDLang1/1000023719.md) | Charge un fichier au format XLSX dans un champ Tableur présent dans une fenêtre. |
| [&lt;Tableur&gt;.Cherche](../WDLang1/1000023720.md) | Recherche une valeur dans les cellules d'un champ Tableur présent dans une fenêtre. |
| [&lt;Tableur&gt;.DébutModification](../WDLang1/1000023721.md) | Signale le début du regroupement d'un ensemble d'opérations sur un champ Tableur en un seul événement (gestion du "undo/redo"). |
| [&lt;Tableur&gt;.DéfusionneCellule](../WDLang1/1410089570.md) | Supprime la fusion de cellule pour la cellule spécifiée ou pour la cellule sélectionnée dans un champ Tableur. |
| [&lt;Tableur&gt;.DéplaceFeuille](../WDLang1/1410087262.md) | Déplace une feuille de calcul dans un champ Tableur présent dans une fenêtre. |
| [&lt;Tableur&gt;.Ferme](../WDLang1/1000023722.md) | Ferme le document XLSX ouvert précédemment par la fonction [&lt;Tableur&gt;.Charge](../WDLang1/1000023719.md). |
| [&lt;Tableur&gt;.Fige](../WDLang1/1000023723.md) | Permet de figer l'affichage d'une ligne ou d'une colonne dans un champ Tableur. |
| [&lt;Tableur&gt;.FinModification](../WDLang1/1000023724.md) | Signale la fin du regroupement d'un ensemble d'opérations sur un champ Tableur en un seul événement (gestion du "undo/redo"). |
| [&lt;Tableur&gt;.FusionneCellule](../WDLang1/1000026322.md) | Fusionne les cellules spécifiées ou les cellules sélectionnées dans un champ Tableur. |
| [&lt;Tableur&gt;.InsèreColonne](../WDLang1/1000023725.md) | Insère une colonne vide dans un champ Tableur présent dans une fenêtre. |
| [&lt;Tableur&gt;.InsèreFeuille](../WDLang1/1000023726.md) | Insère une feuille de calcul dans un champ Tableur présent dans une fenêtre. |
| [&lt;Tableur&gt;.InsèreLigne](../WDLang1/1000023727.md) | Insère une ligne vide dans un champ Tableur présent dans une fenêtre. |
| [&lt;Tableur&gt;.Position](../WDLang1/1000023728.md) | Affiche la cellule spécifiée dans un champ Tableur présent dans une fenêtre: <br><br>- soit la zone visible du champ Tableur est déplacée pour afficher la cellule. <br><br>- soit la cellule est affichée en haut à gauche de la zone visible du champ Tableur.<br><br><br> |
| [&lt;Tableur&gt;.RecalculeTout](../WDLang1/1000023729.md) | Recalcule toutes les formules d'un champ Tableur présent dans une fenêtre (y compris les formules utilisant des fonctions/procédures WLangage). |
| [&lt;Tableur&gt;.RécupèreFormule](../WDLang1/1000023730.md) | Récupère la formule de calcul d'une cellule présente dans un champ Tableur d'une fenêtre. |
| [&lt;Tableur&gt;.RécupèreFusion](../WDLang1/1000023731.md) | Permet de savoir si une cellule est fusionnée avec d'autres cellules et d'obtenir le nom des cellules fusionnées. |
| [&lt;Tableur&gt;.RécupèreSélection](../WDLang1/1000023732.md) | Récupère la liste des cellules sélectionnées dans un champ Tableur présent dans une fenêtre. |
| [&lt;Tableur&gt;.Remplace](../WDLang1/1000023898.md) | Recherche et remplace une valeur dans un champ Tableur. |
| [&lt;Tableur&gt;.RestaurePositionEtSélection](../WDLang1/1000023733.md) | Restaure : <br><br>- la position de l'ascenseur d'un champ Tableur, <br><br>- la feuille en cours, <br><br>- le ou les éléments sélectionnés dans la feuille en cours du champ Tableur. <br><br><br>Ces éléments ont été précédemment sauvegardés par la fonction [&lt;Tableur&gt;.SauvePositionEtSélection](../WDLang1/1000023735.md). |
| [&lt;Tableur&gt;.Sauve](../WDLang1/1000023734.md) | Sauve le contenu d'un champ Tableur dans un fichier au format XLSX. |
| [&lt;Tableur&gt;.SauvePositionEtSélection](../WDLang1/1000023735.md) | Sauve : <br><br>- la position de l'ascenseur d'un champ Tableur pour la feuille en cours, <br><br>- la feuille en cours dans le champ Tableur,<br><br>- le ou les éléments sélectionnés dans le champ Tableur pour la feuille en cours.<br><br><br> |
| [&lt;Tableur&gt;.SelectFeuille](../WDLang1/1000023736.md) | Sélectionne une feuille de calcul (identifiée par son nom) dans un champ Tableur. |
| [&lt;Tableur&gt;.SelectMoins](../WDLang1/1000023737.md) | Désélectionne une ou plusieurs cellules dans un champ Tableur présent dans une fenêtre. |
| [&lt;Tableur&gt;.SelectPlus](../WDLang1/1000023738.md) | Sélectionne une ou plusieurs cellules dans un champ Tableur présent dans une fenêtre. |
| [&lt;Tableur&gt;.StyleSélection](../WDLang1/1000023739.md) | Renvoie ou modifie le style des cellules sélectionnées dans un champ Tableur présent dans une fenêtre. |
| [&lt;Tableur&gt;.SupprimeColonne](../WDLang1/1000023740.md) | Supprime une colonne dans un champ Tableur. |
| [&lt;Tableur&gt;.SupprimeFeuille](../WDLang1/1000023741.md) | Supprime une feuille de calcul dans un champ Tableur présent dans une fenêtre. |
| [&lt;Tableur&gt;.SupprimeLigne](../WDLang1/1000023742.md) | Supprime une ligne dans un champ Tableur. |
| [&lt;Tableur&gt;.SupprimeTout](../WDLang1/1000023743.md) | Supprime toutes les données d'un champ Tableur présent dans une fenêtre (cellules, formules, taille et mises en forme d'entêtes). |
| [&lt;Tableur&gt;.TrieSélection](../WDLang1/1000023744.md) | Trie les valeurs sélectionnées dans un champ Tableur. |
| [&lt;Tableur&gt;.TypeEtMasqueSélection](../WDLang1/1000023745.md) | Renvoie ou modifie le type et le masque des cellules sélectionnées dans un champ Tableur. |




## &lt;Variable Image&gt;.CouleurDominanteImage (Fonction)

***En anglais : &lt;Image variable&gt;.DominantImageColor***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la couleur dominante (c'est-à-dire la plus fréquente) d'une image présente dans une variable de type [Image](../WDLang1/1000019650.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonImage est une Image = "Test.bmp"
MaCouleur est une Couleur = MonImage.CouleurDominanteImage()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Variable Image>.CouleurDominanteImage()`
---

**`<Résultat> : (Entier)`**

Couleur dominante de l'image.

**`<Variable Image> : (Variable de type Image)`**

Nom de la variable de type [Image](../WDLang1/1000019650.md) à manipuler.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Les pixels transparents ou semi-transparents de l'image sont ignorés.

- La fonction **&lt;Variable Image&gt;.CouleurDominanteImage** ne fonctionne pas sur des images vectorielles (SVG, PDF, ...).

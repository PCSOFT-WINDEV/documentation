
## EntierSystèmeVersEntier (Fonction)

***En anglais : SystemIntegerToInteger***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Convertit un entier système en un entier sur 4 octets.

Remarque : L'entier système est codé sur 4 ou 8 octets en fonction du mode de compilation du projet (32 ou 64 bits). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
NumHandle est un entier système

GetModuleHandle est une Description d'API
GetModuleHandle.NomDLL = "kernel32"
GetModuleHandle.NomFonction = "GetModuleHandleA"
GetModuleHandle.TypeRetour = apiEntierSystème
GetModuleHandle.Paramètre[1].Type = apiChaîne

// Récupère le handle d'une fenêtre par un API
NumHandle = API(GetModuleHandle, "MaFenetre")

Num est un entier
Num = EntierSystèmeVersEntier(NumHandle)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = EntierSystèmeVersEntier(<Valeur>)`
---

**`<Résultat> : (Entier)`**

Entier codé sur 4 octets quelque soit le mode de compilation.

**`<Valeur> : (Entier système)`**

Entier système à convertir. 
Rappel : 

- Un entier système est codé sur 4 octets en mode de compilation 32 bits.

- Un entier système est codé sur 8 octets en mode de compilation 64 bits.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Si la valeur de l'entier système est trop grande pour être convertie en entier, une erreur fatale sera générée. Dans ce cas, il faut traiter une exception.

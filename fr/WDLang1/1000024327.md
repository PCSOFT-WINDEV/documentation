


## &lt;Image&gt;.FinDessin (Fonction)

***En anglais : &lt;Image&gt;.EndDrawing***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Efface tous les dessins réalisés dans le champ Image depuis l'exécution de la fonction [&lt;Image&gt;.DébutDessin](../WDLang1/1000024326.md). 
**Remarque** : L'appel de la fonction **&lt;Image&gt;.FinDessin** est optionnel. Les ressources sont automatiquement libérées au moment de la fermeture de la fenêtre ou de la page.



![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Les dessins](../WDLang1/3029035.md). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Destruction de l'image présente dans le champ Image "IMG_Dessin"
IMG_Dessin.FinDessin()
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Champ Image>.FinDessin()`
---

**`<Champ Image> : (Nom de champ)`**

Nom du champ Image à manipuler.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Utilisation dans le cadre de la syntaxe préfixée
<a name="utilisation_dans_cadre_syntaxe_prefixee_ELTPARAGRAPHE000382"></a>La fonction [&lt;Image&gt;.DébutDessin](../WDLang1/1000024326.md) doit être appelée avant l'utilisation des fonctions de dessin. La fonction [&lt;Image&gt;.DébutDessin](../WDLang1/1000024326.md) permet de définir les options de dessin utilisées sur le champ Image. 

Dans le cas d'un champ Image : 

- Les fonctions de dessin travaillent sur une copie ("bitmap") de l'image. Vous pouvez utiliser les fonctions de dessin de Windows (grâce à la fonction [API](../WDLang1/3014005.md) ou [AppelDLL32](../WDLang1/3014003.md)) mais ces fonctions Windows doivent utiliser le DC (Device Context) renvoyé par la fonction [&lt;Image&gt;.DébutDessin](../WDLang1/1000024326.md).

- La fonction **&lt;Image&gt;.FinDessin** ne doit pas être appelée dans le même traitement que la fonction [&lt;Image&gt;.DébutDessin](../WDLang1/1000024326.md).
	Si la fonction **&lt;Image&gt;.FinDessin** est appelée dans le même traitement que la fonction [&lt;Image&gt;.DébutDessin](../WDLang1/1000024326.md), le dessin n'apparaîtra pas : il sera automatiquement effacé.

### Fonctions de dessin, Zoom et ascenseurs automatiques
<a name="fonctions_dessin_zoom_ascenseurs_automatiques_ELTPARAGRAPHE000552"></a>L'option "Ascenseurs et zoom automatiques" disponible sur un champ Image est incompatible avec la fonction [&lt;Image&gt;.DébutDessin](../WDLang1/1000024326.md). Si cette option est utilisée sur le champ Image, la fonction [&lt;Image&gt;.DébutDessin](../WDLang1/1000024326.md) entraîne un affichage figé dans le champ. Le zoom et les ascenseurs sont également figés. Le comportement redevient normal lors de l'appel de la fonction **&lt;Image&gt;.FinDessin**.


## DernierJourDeLAnnée (Fonction)

***En anglais : LastDayOfYear***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la date du dernier jour de l'année.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
dateAbonnement est une Date = "20190125"
// Détermine le dernier jour de l'année
dateFinDAnnée est une Date = DernierJourDeLAnnée(dateAbonnement)


nNombreJoursAvantFinAnnée est un entier
// Calcule le nombre de jours entre le 25 janvier 2019 et la fin de l'année 2019 (31 décembre)
nNombreJoursAvantFinAnnée = DateDifférence(dateAbonnement, dateFinDAnnée)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Obtenir le dernier jour de l'année à partir d'une date

`<Résultat> = DernierJourDeLAnnée([<Date>])`
---

**`<Résultat> : (Chaîne de caractères)`**

Dernier jour de l'année (au format AAAAMMJJ) pour la date spécifiée.

**`<Date> : (Chaîne de caractères, variable de type Date ou variable de type DateHeure optionnelle)`**

Date pour laquelle le dernier jour de l'année doit être calculé. Cette date peut correspondre à : 

- une chaîne de caractères au format AAAAMMJJ (AAAA correspond à l'année, MM au mois et JJ au jour).  

- une chaîne de caractères au format AAAAMMJJHHmmSSCC (AAAA correspond à l'année, MM au mois, JJ au jour, HH aux heures, mm aux minutes, SS aux secondes et CC aux centièmes de seconde (utiliser CCC pour les millièmes de seconde)). Dans ce cas, seule la date est prise en compte. 

- une variable de type Date. 

- une variable de type DateHeure. Dans ce cas, seule la date est prise en compte.


Si ce paramètre n'est pas spécifié, la date du jour est utilisée par défaut.


<a name="SYNTAXE2"></a>

### Obtenir le dernier jour de l'année à partir d'une année

`<Résultat> = DernierJourDeLAnnée(<Année>)`
---

**`<Résultat> : (Chaîne de caractères ou variable de type Date)`**

Dernier jour de l'année (au format AAAAMMJJ).

**`<Année> : (Entier)`**

Année pour laquelle le dernier jour est recherché.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Cette fonction permet de calculer simplement des fourchettes de dates.


## DocSupprimeTableDesMatières (Fonction)

***En anglais : DocDeleteTableOfContents***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime une table des matières dans un document de type Traitement de texte. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
DocSupprimeTableDesMatières(TT_SansNom1.Sélection.Fragment)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`DocSupprimeTableDesMatières(<Fragment>)`
---

**`<Fragment> : (Variable de type docFragment)`**

Nom de la variable de type [docFragment](../WDLang1/1000022482.md) correspondant à la table des matières à supprimer.

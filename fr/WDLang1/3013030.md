
## InitHasard (Fonction)

***En anglais : InitRandom***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Initialise le générateur de nombres aléatoires.






<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Initialisation du générateur de nombres aléatoires
InitHasard()
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`InitHasard(<n>)`
---

**`<n> : (Entier)`**

Nombre de base du générateur de nombres aléatoires. 
Si ce paramètre n'est pas spécifié, le générateur de nombres aléatoires se base sur l'heure système (centièmes de seconde).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Si la fonction **InitHasard** n'est pas appelée, les valeurs aléatoires générées par la fonction [Hasard](../WDLang1/3013037.md) seront les mêmes à chaque exécution du programme.

- Des appels à la fonction **InitHasard** très rapprochés dans le temps donneront des séquences identiques car la fonction **InitHasard** utilise l'heure système pour générer des suites de nombres aléatoires.

- L'utilisation du paramètre &lt;n&gt; ne garantit pas d'obtenir des séquences différentes.

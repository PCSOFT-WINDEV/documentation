
## Procédure WLangage appelée par la fonction DialogueAsynchrone



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Procédure (également appelée "Callback") appelée par la fonction [DialogueAsynchrone](../WDLang1/1000025310.md) lorsque l'utilisateur clique sur l'un des boutons de la boîte de dialogue. 
Cette procédure permet de connaître la valeur du bouton cliqué et permet d'exécuter si nécessaire un traitement spécifique. 
Remarque : Cette procédure peut être une procédure locale, globale, interne ou lambda. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
nBouton est un entier = Hasard(2)
DialogueAsynchrone("Cliquez sur le bouton %1 pour lancer l'application.", ...
		MaProcédure_Dialogue, nBouton)

PROCEDURE INTERNE MaProcédure_Dialogue(nParam1 entier)
	SI nBouton <> nParam1 ALORS
		FinProgramme()
	FIN
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`DialogueAsynchrone_Callback(<Réponse sélectionnée>)`
---

**`<Réponse sélectionnée> : (Entier)`**

Identifie la réponse sélectionnée par l'utilisateur. 
Les valeurs possibles ont été définies dans l'assistant de création de la fonction [DialogueAsynchrone](../WDLang1/1000025310.md). 
Remarque : Les différentes valeurs correspondant aux différentes réponses sont automatiquement intégrées en commentaires dans l'éditeur de code lors de la création du dialogue avec l'assistant.

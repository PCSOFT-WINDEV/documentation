
## docNiveauNumérotation (Type de variable)

***En anglais : docNumberingLevel***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **docNiveauNumérotation** permet de définir toutes les caractéristiques avancées d'un niveau de numérotation utilisable dans un document de type traitement de texte. Les caractéristiques de ce niveau de numérotation peuvent être définies et modifiées à l'aide de différentes propriétés WLangage. 

Ce type est utilisé par les variables de type [docNumérotation](../WDLang1/1000022794.md). 

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).






<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonDoc est un Document = TT_SansNom1
MaNumérotation est un docNumérotation
Niveau est un docNiveauNumérotation

// Définition de la numérotation

// 1er niveau de la numérotation
Niveau.Texte = "%1) "
Niveau.Texte = "%1) "
Niveau.ValeurInitiale = 1
Niveau.MiseEnPage.RetraitPremièreLigne = 10
Niveau.Format = fnumRomainMajuscule
Ajoute(MaNumérotation.Niveau, Niveau)

// Second niveau de la numérotation
Niveau.Texte = "%2] "
Niveau.ValeurInitiale = 1
Niveau.MiseEnPage.RetraitPremièreLigne = 20
Niveau.Format = fnumLettreMajuscule
Ajoute(MaNumérotation.Niveau, Niveau)

// Troisième niveau de la numérotation
Niveau.Texte = "%3\ "
Niveau.ValeurInitiale = 1
Niveau.MiseEnPage.RetraitPremièreLigne = 30
Niveau.Format = fnumLettreMinuscule
Ajoute(MaNumérotation.Niveau, Niveau)


// Ajout de la numérotation au document
soit id = Ajoute(MonDoc.Numérotation, MaNumérotation)

// Définition de la numérotation associée au paragraphe 1
MonDoc.Paragraphe[1].Numérotation.Identifiant = id
MonDoc.Paragraphe[1].Numérotation.Niveau = 1

// Affichage du document dans le champ Traitement de texte
TT_SansNom1 = MonDoc
```

## Remarques
<a name="NOTE0_1"></a>


### Propriétés spécifiques à la description des variables de type docNiveauNumérotation
<a name="proprietes_specifiques_description_des_variables_type_docniveaunumerotation_ELTPARAGRAPHE000040"></a>Les propriétés suivantes peuvent être utilisées pour manipuler un niveau de numérotation :

   | --- |
| --- | --- | --- |
| **Nom de la propriété** | Type manipulé | Effet |
| Alignement | Constante de type Entier | Position de la numérotation par rapport à la position de début du paragraphe : <br><br>- *chCentre* : La numérotation est centrée par rapport au début du paragraphe. ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=TT_num_aligne_centre.gif)<br><br><br>- *chDroite* : La numérotation est positionnée à droite par rapport au début du paragraphe. ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=TT_num_aligne_droite.gif)<br><br><br>- *chGauche* : La numérotation est positionnée à gauche par rapport au début du paragraphe. ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=TT_num_aligne_gauche.gif)<br><br><br><br> |
| Format | Constante de type Entier | Formatage de la numérotation de ce niveau (chiffres, chiffres romains, lettres, ...) : <br><br>- *fnumLettreMajuscule* : Numérotation en lettres majuscules : A, B, C, D, ...<br><br>- *fnumLettreMinuscule* : Numérotation en lettres minuscules : a, b, c, d, ...<br><br>- *fnumNumérique* : Numérotation en chiffres : 1, 2, 3, 4, ...<br><br>- *fnumNumériqueEnLettres* : Numérotation en chiffres écrits en lettres : un, deux, trois, quatre, ...<br><br>- *fnumOrdinal* : Numérotation ordonnée : 1er, 2nd, 3ème, ...<br><br>- *fnumOrdinalEnLettres* : Numérotation ordonnée en lettres : premier, second, troisième, ...<br><br>- *fnumPuce* : Numérotation sous forme de puces. Seules les puces représentées par un caractère d'une police sont disponibles. Il n'est pas possible de créer une puce associée à une image. <br><br>- *fnumRomainMajuscule* : Numérotation en chiffres romains majuscules : I, II, III, ...<br><br>- *fnumRomainMinuscule* : Numérotation en chiffres romains minuscules : i, ii, iii, ...<br><br><br> |
| MiseEnForme | Variable de type [docMiseEnForme](../WDLang1/1000022817.md) | Paramètres de mise en forme de la numérotation. |
| MiseEnPage |   | Mise en page du paragraphe numéroté. |
|   | MiseEnPage.Alignement | Constante de type Entier | Alignement horizontal utilisé pour le paragraphe numéroté : <br><br>- *chCentre* : Centré<br><br>- *chDroite* : Aligné à droite<br><br>- *chGauche* : Aligné à gauche<br><br>- *chJustifié* : Justifié. <br><br><br> |
|   | MiseEnPage.Cadre | Variable de type [Cadre](../Motscles/15140100.md) | Caractéristiques du cadre utilisé pour le paragraphe numéroté. <br><br>Remarque : les coins arrondis ne sont pas gérés. |
|   | MiseEnPage.CouleurFond | Entier | Couleur de fond du paragraphe numéroté. Cette couleur peut correspondre à :<br><br>- une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)),<br><br>- [une couleur prédéfinie](../WDLang5/3010002.md). <br><br><br> |
|   | MiseEnPage.EspacementAprès | Réel | Espacement après le paragraphe numéroté (exprimé en millimètres). <br><br>Par défaut, cette propriété vaut 0. |
|   | MiseEnPage.EspacementAvant | Réel | Espacement avant le paragraphe numéroté (exprimé en millimètres). <br><br>Par défaut, cette propriété vaut 0. |
|   | MiseEnPage.Interligne | Réel | Interligne (exprimé en millimètres). <br><br>Utilisé si la propriété **TypeInterligne** vaut *interligneExact* ou *interligneMinimum*. <br><br><br><br>Par défaut, cette propriété vaut 0. |
|   | MiseEnPage.RetraitDroit | Réel | Retrait à droite du paragraphe numéroté (exprimé en millimètres). <br><br>Par défaut, cette propriété vaut 0. |
|   | MiseEnPage.RetraitGauche | Réel | Retrait à gauche du paragraphe numéroté (exprimé en millimètres). <br><br>Par défaut, cette propriété vaut 0. |
|   | MiseEnPage.RetraitPremièreLigne | Réel | Retrait de la première ligne du paragraphe numéroté (exprimé en millimètres). <br><br>Par défaut, cette propriété vaut 0. |
|   | MiseEnPage.TabulationPerso | Tableau de [docTabulation](../WDLang1/1000022573.md) | Tabulations personnalisées du paragraphe numéroté. |
|   | MiseEnPage.TrameFond | Variable de type [Fond](../WDLang1/1000022038.md) | Caractéristiques de fond du paragraphe numéroté. <br><br> |
|   | MiseEnPage.TypeInterligne | Constante de type Entier | Type d'interligne utilisé : <br><br>- *interligneExact* : L'interligne correspond à la valeur de la propriété **Interligne**. <br><br>- *interligneMinimum* : L'interligne correspond au minimum à la valeur de la propriété **Interligne**. Si l'interligne par défaut de la police de caractères de la ligne précédente est supérieur à la propriété **Interligne**, l'interligne est celui par défaut de la police. <br><br>- *interligneMultiple* (valeur par défaut) : L'interligne est obtenu en multipliant la propriété **Interligne** par l'interligne par défaut de la police de caractères de la ligne précédente et en divisant le tout par 240. <br><br><br> |
| RecommencerAprès | Entier | Niveau après lequel la numérotation doit être réinitialisée. Cette propriété peut correspondre à : <br><br>- 0 : la numérotation n'est jamais réinitialisée. <br><br>- -1 : la numérotation est réinitialisée dès qu'un niveau supérieur est incrémenté. <br><br>- une valeur x supérieure à 0 : la numérotation est réinitialisée dès que le niveau x est incrémenté. <br><br><br> |
| Séparateur | Constante de type Entier | Type de séparateur utilisé pour réaliser l'espacement entre la numérotation du paragraphe et la suite du texte : <br><br>- *sepnumAucun* : Pas de séparateur. <br><br>- *sepnumEspace* : Le séparateur est un espace. <br><br>- *sepnumTabulation* : Le séparateur est une tabulation. <br><br><br> |
| Texte | Chaîne de caractères | Numérotation à utiliser. Le symbole % suivi d'un chiffre contiendra la valeur de la numérotation au niveau défini par ce chiffre. <br><br>Exemples : <br><br><br><br>- "%1 )" utilise la numérotation du niveau 1 suivie d'une parenthèse. <br><br>- "%1. %2" affiche la numérotation du niveau 1 suivie de la numérotation du niveau 2. <br><br><br><br><br>Pour une liste à puces, caractères représentant la puce voulue. Ce caractère sera affiché dans la police choisie pour la mise en forme de la numérotation. <br><br><br><br>Attention : Le numéro des niveaux précédents sera formaté comme défini par le niveau précédent sauf si la propriété **TousEnChiffre** est à Vrai. |
| TousEnChiffre | Booléen | <br><br>- <u><u><u><u>Vrai</u></u></u></u> pour que tout les numéros de niveau (niveau courant et niveaux précédents) soient écrits en chiffres arabes, <br><br>- <u><u><u><u>Faux</u></u></u></u> pour conserver les caractéristiques de chaque niveau. <br><br><br> |
| Tplc | Chaîne de caractères | Identifiant spécifique pour MS Word. <br>**Cette propriété est disponible en lecture seulement**. |
| ValeurInitiale | Entier | Valeur de début de la numérotation (doit être un entier positif). |

### Ajouter une description de numérotation à un document
<a name="ajouter_une_description_numerotation_document_ELTPARAGRAPHE000278"></a>Pour ajouter une description de numérotation à un document, il est possible d'utiliser la fonction [Ajoute](../WDLang1/1000017348.md) via la syntaxe suivante : 

```txt
Ajoute(MonDocument.Numérotation, MaNumérotation)
```
 où : 

- MonDocument est une variable de type [Document](../WDLang1/1000022461.md). 

- MaNumérotation est une variable de type [docNumérotation](../WDLang1/1000022794.md). 





### Ajouter un niveau de numérotation à un document
<a name="ajouter_niveau_numerotation_document_ELTPARAGRAPHE000298"></a>Pour ajouter un niveau de numérotation à un document, il est possible d'utiliser la fonction [Ajoute](../WDLang1/1000017348.md) via la syntaxe suivante : 

```txt
Ajoute(Numérotation.Niveau, MonNiveau)
```
 où : 

- Numérotation est une variable de type [docNumérotation](../WDLang1/1000022794.md). 

- MonNiveau est une variable de type **docNiveauNumérotation**.

Elles apparaissent dans le ruban, dans le groupe "Paragraphes", en déroulant l'option ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Champ_TT_Puces%20-%20HC%20N%B0001%201.gif)
, dans le groupe "Numérotation du document".

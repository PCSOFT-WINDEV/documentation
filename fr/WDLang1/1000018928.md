


## TreeMapOccurrence (Fonction)

***En anglais : TreeMapCount***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le nombre d'éléments présents dans un champ TreeMap.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Nombre d'éléments présents dans le champ TreeMap TMAP_Fichier
ResNombreElement = TreeMapOccurrence(TMAP_Fichier)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = TreeMapOccurrence(<Champ TreeMap>)`
---

**`<Résultat> : (Entier)`**

Nombre d'éléments présents dans le champ TreeMap (0 si le champ est vide). 
**Remarque** : Tous les éléments sont pris en compte (même les éléments de poids 0). 

**`<Champ TreeMap> : (Nom de champ)`**

Nom du champ TreeMap à manipuler.
Si ce paramètre correspond à une chaîne vide (""), le champ TreeMap manipulé sera le champ TreeMap auquel appartient l'événement en cours.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
La fonction **TreeMapOccurrence** et la propriété [Occurrence](../Proprietes/2510097.md) sont équivalentes :

```txt
<Nb éléments> = TreeMapOccurrence(<Champ TreeMap>)
```

```txt
<Nb éléments> = <Champ TreeMap>.Occurrence
```

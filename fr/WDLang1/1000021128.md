


## &lt;Diagramme de Gantt&gt;.ListeLien (Fonction)

***En anglais : &lt;Gantt chart&gt;.ListLink***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Liste les liens présents dans une colonne d'un champ Table (ou Table hiérarchique) de type Diagramme de Gantt.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Liste tous les liens qui pointent sur la tâche "ID1"
tabLiens est un Tableau de LienGantt
tabLiens = COL_Gantt.ListeLien(ganttLienDestination, "ID1")
Lien est un LienGantt
POUR TOUT Lien de tabLiens
	Trace("Lié à " + Lien.IDSource)
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Colonne Gantt>.ListeLien(<Filtre> [, <Identifiant Tâche Destination>])`
---

**`<Résultat> : (Tableau de LienGantt)`**

Nom de la variable de type Tableau de [LienGantt](../WDLang1/1000021076.md) contenant tous les liens trouvés.

**`<Colonne Gantt> : (Nom de champ)`**

Nom de la colonne (d'un champ Table ou Table hiérarchique) de type Diagramme de Gantt à manipuler.

**`<Filtre> : (Constante de type Entier)`**

Type de liens à récupérer : 


|   |   |
| --- | --- |
| *ganttLienDestination* | Liste tous les liens qui pointent sur la tâche &lt;Identifiant Tâche Destination&gt;. |
| *ganttLienTout* | Liste tous les liens. |



**`<Identifiant Tâche Destination> : (Chaîne de caractères optionnelle)`**

Identifiant de la tâche Destination dans le cas de l'utilisation de la constante *ganttLienDestination*. 

Cet identifiant correspond à la propriété **ID** définie dans la variable de type [TâcheGantt](../WDLang1/1000020881.md).

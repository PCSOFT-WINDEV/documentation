***En anglais : UMCOpen***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ouvre la fenêtre d'édition du Macro-Code Utilisateur pour la fenêtre en cours.
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=MCUOuvre%20-%20HC%20N%B0001.gif)

<a name="sample_code"></a>

## Exemple


```wl
// -- Code de clic Bouton 
MCUOuvre()
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`MCUOuvre()`
---



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Pour utiliser cette fonction, le MCU doit être activé au niveau du projet : 

1. Sous le volet "Projet", dans le groupe "Projet", cliquez sur "Description". 

2. Affichez l'onglet "Avancé". 

3. Cliquez sur le bouton "Macro Code Utilisateur". 

4. Sélectionnez l'option "Oui : Autoriser la saisie de Macro Code Utilisateur (MCU)". 

5. Validez.

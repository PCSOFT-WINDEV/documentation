
## Jeton (Type de variable)

***En anglais : Token***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **Jeton** permet de définir toutes les caractéristiques avancées d'un jeton affiché dans un champ de saisie. Les caractéristiques de ce jeton peuvent être définies et modifiées à l'aide de différentes propriétés WLangage. 

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ajout d'un jeton dans un champ de saisie
MonJeton est un Jeton 
MonJeton.Libellé = Client.Nom
MonJeton.Valeur = Client.IdClient
SaisieAjouteJeton(SAI_Client, MonJeton)

// Modification de la couleur de fond d'un jeton
POUR TOUT MonJeton DE SAI_Client.Jeton
	SI MonJeton.Valeur = Client.IdClient ALORS
		MonJeton.CouleurFond = RougeClair
	FIN
FIN
```





<a name="NOTE0"></a>

## Remarques
<a name="NOTE0_1"></a>


### Propriétés spécifiques à la description des variables de type Jeton
<a name="proprietes_specifiques_description_des_variables_type_jeton_ELTPARAGRAPHE000033"></a>Les propriétés suivantes peuvent être utilisées pour manipuler un jeton :

| Nom de la propriété | Type manipulé | Effet |
| --- | --- | --- |
| Bulle | Chaîne de caractères | Bulle affichée au survol du jeton. |
| Couleur | Variable de type [Couleur](../WDLang1/1000019661.md) | Couleur du texte du jeton.<br><br> |
| CouleurFond | Variable de type [Couleur](../WDLang1/1000019661.md) | Couleur de fond du jeton.<br><br> |
| Libellé | Chaîne de caractères | Libellé du jeton tel qu'il sera affiché dans le champ de saisie. Si cette propriété n'est pas spécifiée, le libellé du jeton correspondra à sa valeur mémorisée.<br><br> |
|   | Chaîne de caractères | Code HTML du libellé du jeton tel qu'il sera affiché dans le champ de saisie. Si cette propriété n'est pas spécifiée, le libellé spécifié avec la propriété **Libellé** est utilisé. |
| Valeur | Chaîne de caractères | Valeur du jeton. |
| ValeurMémorisée | Chaîne de caractères | Valeur mémorisée du jeton. Permet de stocker n'importe quelle valeur. |




<a name="NOTE0_2"></a>


### Manipulation des jetons
<a name="manipulation_des_jetons_ELTPARAGRAPHE000110"></a>

- Pour manipuler les jetons d'un champ de saisie, il est possible d'utiliser : 

	- soit les [fonctions de gestion des jetons](../WDLang1/1000024042.md). 

	- soit la propriété [Jeton](../Proprietes/1000023976.md) du champ de saisie qui retourne une liste modifiable des jetons affichés dans le champ.




- Pour réinitialiser le contenu d'une variable de type Jeton, il est possible d'utiliser la fonction [VariableRAZ](../WDLang1/3013069.md).

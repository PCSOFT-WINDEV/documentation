


## ChaîneIncrémente (Fonction)

***En anglais : StringIncrement***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Incrémente un nombre situé en fin ou début d'une chaîne de caractères.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affiche "Chaîne2"
Info(ChaîneIncrémente("Chaîne1"))
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = ChaîneIncrémente(<Chaîne initiale> [, <Position>])`
---

**`<Résultat> : (Chaîne de caractères)`**

Chaîne initiale avec le nombre (ou le chiffre) incrémenté.

**`<Chaîne initiale> : (Chaîne de caractères)`**

Chaîne à incrémenter. 

**`<Position> : (Entier optionnel)`**

Position où se situe le nombre (ou le chiffre) à incrémenter : 


|   |   |
| --- | --- |
| *DepuisDébut* | Le nombre en début de chaîne est incrémenté.<br><br>Si la chaîne ne commence pas par un nombre, le chiffre 1 sera ajouté en début de chaîne. |
| *DepuisFin* | Le nombre en fin de chaîne est incrémenté.<br><br>Si la chaîne ne se termine pas par un nombre, le chiffre 1 sera ajouté en fin de chaîne. |


Si ce paramètre n'est pas spécifié, trois cas peuvent se présenter :

- la chaîne se termine par un nombre : dans ce cas, c'est le nombre à la fin qui sera incrémenté, 

- la chaîne commence par un nombre : dans ce cas, c'est le nombre au début qui sera incrémenté, 

- la chaîne ne commence ou ne se termine pas par un nombre : dans ce cas, le chiffre 1 sera ajouté en fin de chaîne.




## Fonctions de gestion de la reconnaissance vocale
			



Les fonctions permettant de gérer la reconnaissance vocale sont les suivantes :



|   |   |
| --- | --- |
| [ReconnaissanceVocaleAjouteCommande](../WDLang1/1000020014.md) | Ajoute une commande vocale dans la fenêtre en cours. |
| [ReconnaissanceVocaleDéclenche](../WDLang1/1000020013.md) | Déclenche le service de reconnaissance vocale de l'appareil. |
| [ReconnaissanceVocaleSupprimeCommande](../WDLang1/1000020017.md) | Supprime une commande vocale précédemment ajoutée par la fonction [ReconnaissanceVocaleAjouteCommande](../WDLang1/1000020014.md). |







![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png) **Remarque** : Par compatibilité avec les versions précédentes, les fonctions suivantes sont également disponibles : 



|   |   |
| --- | --- |
| [AjouteCommandeVocale](../WDLang1/1000017106.md) | Ajoute une commande vocale dans la fenêtre en cours. A partir de la version 17, cette fonction est remplacée par la fonction [ReconnaissanceVocaleAjouteCommande](../WDLang1/1000020014.md). |
| [SupprimeCommandeVocale](../WDLang1/1000017187.md) | Supprime une commande vocale précédemment ajoutée par la fonction [AjouteCommandeVocale](../WDLang1/1000017106.md). A partir de la version 17, cette fonction est remplacée par la fonction [ReconnaissanceVocaleSupprimeCommande](../WDLang1/1000020017.md). |






- ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Lesfonctionsdereconnaissancevocale.gif) ***Exemples unitaires (WINDEV)*** : **Les fonctions de reconnaissance vocale** <br>Utilisation des fonctions de reconnaissance vocale.<br>Les commandes vocales de Windows Vista (ou de Windows XP, avec un logiciel tiers) permettent de piloter des applications WINDEV à la voix.<br>Par défaut, une application WINDEV répond déjà à la voix, lorsque l'on dicte dans un champ de saisie, ou lorsque l'on énonce le libellé d'un bouton.<br>Il est également possible de programmer une commande vocale spécifique pour effectuer une action particulière.
- ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=AndroidSyntheseVocale.gif) ***Android (WINDEV Mobile)*** : **Android Synthèse Vocale** <br>Cet exemple didactique montre comment gérer la reconnaissance et la synthèse vocale sous Android.<br>La synthèse vocale se fait en utilisant soit les fonctions WLangage, soit un fichier JAR externe intégré dans le projet WINDEV Mobile.


## FIListePrécédent (Fonction)

***En anglais : IWListPrevious***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Affiche la fenêtre interne précédente de la liste des fenêtres internes parcourue par un champ Fenêtre Interne.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
FIListePrécédent(FI_ChampFenêtreInterne)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = FIListePrécédent(<Champ Fenêtre Interne>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la fenêtre interne précédente a été affichée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Champ Fenêtre Interne> : (Nom de champ)`**

Nom du champ Fenêtre Interne à manipuler.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Le changement de fenêtre interne avec les fonctions [FIListeSuivant](../WDLang1/1000020564.md) et **FIListePrécédent** utilise une animation. Pour effectuer un changement sans animation, utilisez la fonction [FIListePosition](../WDLang1/1000020562.md).

![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Les animations ne sont pas disponibles.

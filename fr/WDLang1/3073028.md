
## SysVersion (Fonction)

***En anglais : SysVersion***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie des renseignements sur la version de PHP utilisée sur le poste serveur en cours.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affichage de la version PHP
 Info("Version PHP " + SysVersion())
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = SysVersion()`
---

**`<Résultat> : (Chaîne de caractères)`**



- Information demandée sur la version PHP, 

- Chaîne vide ("") en cas d'erreur.

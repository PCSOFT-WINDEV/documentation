
## CombinaisonDepuisValeur (Fonction)

***En anglais : CombinationFromValue***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie une combinaison d'une ou plusieurs options connues par leurs valeurs associées.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Description de la combinaison
UneCombinaison est une combinaison
	Option1 = 1
	Option2 = 2
FIN

o est UneCombinaison

// o est affecté avec l'option "Option1"
o = CombinaisonDepuisValeur(UneCombinaison, 1)

// o est affecté avec la combinaison des options "Option1" et "Option2"
o est UneCombinaison = CombinaisonDepuisValeur(UneCombinaison, 3)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = CombinaisonDepuisValeur(<Combinaison> , <Valeur>)`
---

**`<Résultat> : (Variable de type Combinaison)`**

Variable de type [Combinaison](../Motscles/1514074.md) contenant la combinaison d'options. 

**`<Combinaison> : (Variable de type Combinaison)`**

Nom de la variable de type [Combinaison](../Motscles/1514074.md) à manipuler.

**`<Valeur> : (Chaîne de caractères ou Entier)`**



- **Si ce paramètre est une chaîne de caractères**, les options sont créées en vérifiant la présence des valeurs associées à chaque option dans la chaîne grâce à une opération de recherche respectant la casse.

- **Si ce paramètre est un entier**, les options sont créées en vérifiant la présence des bits correspondant aux valeurs associées à chaque option grâce à une option de Et binaire.

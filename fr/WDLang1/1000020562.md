
## FIListePosition (Fonction)

***En anglais : IWListPosition***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Affiche la fenêtre interne correspondant à la position spécifiée (dans la liste des fenêtres internes parcourues par un champ Fenêtre Interne) ou renvoie la position de la fenêtre interne actuellement affichée (dans la liste des fenêtres internes parcourues par un champ Fenêtre Interne).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affiche la première fenêtre interne associée au champ 
FIListePosition(CFI_ChampFenêtreInterne, 1)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Afficher les fenêtres internes à partir d'une position

`FIListePosition(<Champ Fenêtre Interne> , <Position>)`
---

**`<Champ Fenêtre Interne> : (Nom de champ)`**

Nom du champ Fenêtre Interne à manipuler.

**`<Position> : (Entier)`**

Indice de la fenêtre interne à afficher dans le champ.


<a name="SYNTAXE2"></a>

### Connaître la position de la fenêtre interne actuellement affichée

`<Résultat> = FIListePosition(<Champ Fenêtre Interne>)`
---

**`<Résultat> : (Entier)`**



- Position de la fenêtre interne actuellement affichée dans le champ "Fenêtre interne". 

- -1 si aucune fenêtre interne n'est associée au champ.




**`<Champ Fenêtre Interne> : (Nom de champ)`**

Nom du champ Fenêtre interne à manipuler.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Le changement de la fenêtre interne affichée dans le champ avec la fonction **FIListePosition** n'utilise pas d'animation. Pour utiliser une animation lors du changement de fenêtre interne, utilisez les fonctions [FIListeSuivant](../WDLang1/1000020564.md) et [FIListePrécédent](../WDLang1/1000020565.md). 

![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Les animations ne sont pas disponibles.




## &lt;Tableau de bord&gt;.SauveConfiguration (Fonction)

***En anglais : &lt;Dashboard&gt;.SaveConfiguration***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Sauvegarde la configuration d'un champ Tableau de bord sous forme d'une chaîne. Cette configuration est constituée des éléments suivants : 

- position des fenêtres/pages internes du champ Tableau de bord.

- taille des fenêtres/pages internes du champ Tableau de bord.

- visibilité des fenêtres/pages internes du champ Tableau de bord.




Remarque : Pour sauver ou restaurer automatiquement la configuration utilisateur, cochez l'option "Mémoriser la configuration des Widgets" dans l'onglet "Détail" de la fenêtre de description du champ.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 
```wl
// Demande à l'utilisateur le fichier de sauvegarde
sFichier est une chaîne

// Ouvre le sélecteur de fichiers
sFichier = fSélecteur("", "", "Sélectionnez un fichier de configuration...", ...
			"Configuration du tableau de bord" + TAB + "*.tdbconf", "*.tdbconf", ...
			fselCrée + fselExiste)
// Si un fichier est sélectionné
SI sFichier <> "" ALORS
	// Récupère la configuration courante dans une chaîne
	sConfiguration est chaîne = TDB_TableauDeBord.SauveConfiguration()
	// Sauve la configuration
	SI fSauveTexte(sFichier, sConfiguration) ALORS
		ToastAffiche("La configuration a été sauvée", toastCourt, cvMilieu, chCentre)
	SINON
		Erreur("Erreur en sauvegarde du fichier de configuration", ErreurInfo())
	FIN
FIN
```

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Champ Tableau de bord>.SauveConfiguration()`
---

**`<Résultat> : (Chaîne de caractères)`**

Configuration du champ Tableau de bord. 

Cette configuration peut être sauvée avec les fonctions [fSauveTexte](../WDLang1/3036046.md) ou [SauveParamètre](../WDLang1/3025046.md).

Cette configuration pourra être restaurée grâce à la fonction [&lt;Tableau de bord&gt;.ChargeConfiguration](../WDLang1/1000022506.md).

**`<Champ Tableau de bord> : (Nom de champ)`**

Nom du champ Tableau de bord à manipuler.

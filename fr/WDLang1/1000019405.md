


## EnModeWebservice (Fonction)

***En anglais : InWebserviceMode***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de savoir si l'exécution est en cours dans le moteur AWWS.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Code exécuté depuis un Webservice ? 
SI EnModeWebservice() = Vrai ALORS
	// Traitements à effectuer
	...
SINON
	// Traitements à effectuer
	...
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = EnModeWebservice()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le code est exécuté dans le moteur AWWS, 

- <u><u><u><u>Faux</u></u></u></u> dans tous les autres cas. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Disponibilité de la fonction
<a name="disponibilite_fonction_ELTPARAGRAPHE000204"></a>La fonction **EnModeWebservice** est disponible dans tous les produits pour permettre l'écriture d'un code multi-produits. Des fonctions ou procédures utilisant la fonction **EnModeWebservice** peuvent être partagées entre plusieurs produits, sans que des erreurs de compilation soient générées.

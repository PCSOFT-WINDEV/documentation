
## docFormatMulticolonne (Type de variable)

***En anglais : docFormatMulticolumn***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **docFormatMulticolonne** permet de définir toutes les caractéristiques avancées d'une section multicolonne définie dans une variable de type [docSection](../WDLang1/1000022685.md). 

Les caractéristiques de ce type de variable **docFormatMulticolonne** peuvent être connues et modifiées à l'aide de différentes propriétés WLangage. 

**ATTENTION : Ce type ne doit pas être utilisé directement. Il doit être utilisé uniquement via le type [docSection](../WDLang1/1000022685.md).**


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Récupération du champ Traitement de texte
MonDoc est un Document <- TT_SansNom1

// Met en page le docx avec 3 colonnes
// un espacement de 20 entre chaque colonne et le séparateur visible
MonDoc.Section.Multicolonne.NombreColonne = 3
MonDoc.Section.Multicolonne.EspaceModeIdentique = 20
MonDoc.Section.Multicolonne.Séparateur = Vrai
```





<a name="NOTE0"></a>

## Remarques
<a name="NOTE0_1"></a>


### Propriétés spécifiques à la description des variables de type docFormatMulticolonne
<a name="proprietes_specifiques_description_des_variables_type_docformatmulticolonne_ELTPARAGRAPHE000035"></a>Les propriétés suivantes peuvent être utilisées pour manipuler une section de type multicolonne :

   | --- |
| --- | --- | --- |
| Nom de la propriété | Type manipulé | Effet |
| Colonnes[n] |   | Tableau de docFormatColonne | Configuration de chaque colonne lors de l'utilisation d'un mode multicolonne avec une largeur de colonne variable. |
|   | Colonnes[n].Espace | Réel | Espace avec la colonne suivante (exprimé en millimètres). |
|   | Colonnes[n].Largeur | Réel | Largeur de la colonne (exprimé en millimètres). |
| EspaceModeIdentique | Réel | Espace entre les différentes colonnes dans le cas d'utilisation d'un mode multicolonne avec une largeur des colonnes identique. Cet espace est exprimé en millimètres.  |
| Mode | Constante de type Entier | Type de gestion du multicolonne : <br><br>- *docMulticolonneAucun* : Pas de gestion du multicolonne. <br><br>- *docMulticolonneIdentique* : Chaque colonne a une largeur identique. <br><br>- *docMulticolonneVariable* ; Chaque colonne a une largeur variable.<br><br><br> |
| NombreColonne | Entier | Nombre de colonne de la section. |
| Séparateur | Booléen | <br><br>- <u><u><u><u>Vrai</u></u></u></u> pour afficher le séparateur de colonnes. <br><br>- <u><u><u><u>Faux</u></u></u></u> pour ne pas afficher le séparateur de colonnes.  <br><br><br> |


## Constantes service
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les constantes de gestion des services sont les suivantes :

| Constante | Description | Utilisée par les fonctions |
| --- | --- | --- |
| **CompteServiceLocal** | Compte local qui a des droits recommandés pour exécuter un service. Sur le réseau, ce compte sera considéré comme invité. | [Variable Service](../WDLang1/1000017118.md) |
| **CompteServiceRéseau** | Compte local qui a des droits recommandés pour exécuter un service. Sur le réseau, ce compte aura les droits de l'ordinateur en cours. | [Variable Service](../WDLang1/1000017118.md) |
| **CompteSystèmeLocal** | Compte local représentant le système d'exploitation. Ce compte a tout les droits. Sur le réseau, ce compte aura les droits de l'ordinateur en cours. | [Variable Service](../WDLang1/1000017118.md) |
| **fsArrêt** | Le service s'arrête normalement, sans provoquer d'erreur. Les différents traitements de terminaison sont exécutés puis le service s'arrête. | [FinService](../WDLang1/1000018711.md) |
| **fsDéfaillance** | Le service s'arrête en provoquant une défaillance. Les différents traitements de terminaison ne sont pas exécutés. | [FinService](../WDLang1/1000018711.md) |
| **ejAvertissement** | Evénement d'avertissement | [ServiceEcritEvénementJournal](../WDLang1/1000018807.md) |
| **ejErreur** | Evénement d'erreur | [ServiceEcritEvénementJournal](../WDLang1/1000018807.md) |
| **ejInformation** | Evénement d'information | [ServiceEcritEvénementJournal](../WDLang1/1000018807.md) |
| **serviceCmdArrête** | Arrête le service (équivalent à la fonction [ServiceArrête](../WDLang1/1000017111.md)). | [ServiceContrôle](../WDLang1/1000017108.md) |
| **serviceCmdPause** | Met le service en pause (équivalent à la fonction [ServicePause](../WDLang1/1000017109.md)). | [ServiceContrôle](../WDLang1/1000017108.md) |
| **serviceCmdRelance** | Sort un service de la pause (équivalent à la fonction [ServiceRelance](../WDLang1/1000017105.md)). | [ServiceContrôle](../WDLang1/1000017108.md) |
| **serviceCmdRelireConfig** | Demande au service de relire ses paramètres de configuration (équivalent à la fonction [ServiceActualise](../WDLang1/1000017107.md)). | [ServiceContrôle](../WDLang1/1000017108.md) |
| **serviceDémarrageAutomatique** | Démarrage automatique après l'initialisation du système | [Variable Service](../WDLang1/1000017118.md) |
| **serviceDémarrageDésactivé** | Le service est désactivé. Toute tentative de démarrage provoque une erreur. | [Variable Service](../WDLang1/1000017118.md) |
| **serviceDémarrageManuel** | Démarrage manuel | [Variable Service](../WDLang1/1000017118.md) |
| **serviceDémarrageSystème** | Démarrage automatique en même temps que le système d'exploitation (utilisé par les pilotes). | [Variable Service](../WDLang1/1000017118.md) |
| **serviceErreurCritique** | Le système inscrit l'échec de démarrage du service dans le journal de l'observateur d'événement et redémarre en mode "Dernière bonne configuration connue". | [Variable Service](../WDLang1/1000017118.md) |
| **serviceErreurGrave** | Le système inscrit l'échec de démarrage du service dans le journal de l'observateur d'événement et redémarre en mode "Dernière bonne configuration connue" s'il n'est pas déjà dans ce mode là. | [Variable Service](../WDLang1/1000017118.md) |
| **serviceErreurIgnore** | Le système ne fait rien de spécial si le service ne démarre pas. | [Variable Service](../WDLang1/1000017118.md) |
| **serviceErreurLog** | Le système inscrit l'échec de démarrage du service dans le journal de l'observateur d'événement. | [Variable Service](../WDLang1/1000017118.md) |
| **serviceEtatArrêté** | Le service est arrêté. | [ServiceAttendEtat](../WDLang1/1000017101.md), [ServiceEtat](../WDLang1/1000017102.md) |
| **serviceEtatAttenteArrêt** | Le service a reçu une commande d'arrêt mais il n'est pas encore arrêté. | [ServiceAttendEtat](../WDLang1/1000017101.md), [ServiceEtat](../WDLang1/1000017102.md) |
| **serviceEtatAttenteDémarrage** | Le service a reçu une commande de démarrage mais il n'est pas encore démarré. | [ServiceAttendEtat](../WDLang1/1000017101.md), [ServiceEtat](../WDLang1/1000017102.md) |
| **serviceEtatAttentePause** | Le service a reçu une commande de pause mais il n'est pas encore en pause. | [ServiceAttendEtat](../WDLang1/1000017101.md), [ServiceEtat](../WDLang1/1000017102.md) |
| **serviceEtatAttenteRelancement** | Le service a reçu une commande de relancement mais il n'est pas encore relancé. | [ServiceAttendEtat](../WDLang1/1000017101.md), [ServiceEtat](../WDLang1/1000017102.md) |
| **serviceEtatDémarré** | Le service est en cours de fonctionnement. | [ServiceAttendEtat](../WDLang1/1000017101.md), [ServiceEtat](../WDLang1/1000017102.md) |
| **serviceEtatEnPause** | Le service est en pause. | [ServiceAttendEtat](../WDLang1/1000017101.md), [ServiceEtat](../WDLang1/1000017102.md) |
| **serviceEtatTimeout** | Le service n'a pas atteint l'état demandé à l'issue du timeout. | [ServiceAttendEtat](../WDLang1/1000017101.md) |
| **serviceTypeInteractifPartagé** | Service autorisé à interagir avec le bureau et partageant son processus avec d'autres services. | [Variable Service](../WDLang1/1000017118.md) |
| **serviceTypeInteractifSimple** | Service autorisé à interagir avec le bureau et fonctionnant seul dans son processus. | [Variable Service](../WDLang1/1000017118.md) |
| **serviceTypePartagé** | Service standard partageant son processus avec d'autres services. | [Variable Service](../WDLang1/1000017118.md) |
| **serviceTypePiloteNoyau** | Pilote du noyau. | [Variable Service](../WDLang1/1000017118.md) |
| **serviceTypePiloteSystèmeFichier** | Pilote de système de fichier. | [Variable Service](../WDLang1/1000017118.md) |
| **serviceTypeSimple** | Service standard fonctionnant seul dans son processus. | [Variable Service](../WDLang1/1000017118.md) |

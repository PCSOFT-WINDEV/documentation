


## Iconise (Fonction)

***En anglais : Iconize***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Iconise une fenêtre WINDEV.

**Remarque** : La fonction **Iconise** réalise la même action que le bouton de réduction présent dans le coin haut droit des fenêtres.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Iconise la fenêtre "FEN_SaisieClient"
Iconise(FEN_SaisieClient)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`Iconise([<Fenêtre>])`
---

**`<Fenêtre> : (Nom de fenêtre)`**

Nom de la fenêtre à icôniser. 

- Si ce paramètre n'est pas précisé ou correspond à une chaîne vide (""), la fenêtre en cours est icônisée.

- Si la fenêtre spécifiée correspond à une fenêtre fille, la fenêtre mère est également icônisée.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Pour savoir si une fenêtre est icônisée, utilisez la fonction [FenTaille](../WDLang1/3038020.md).

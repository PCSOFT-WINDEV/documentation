
## ActionBarSupprimeHistoriqueRecherche (Fonction)

***En anglais : ActionBarDeleteSearchHistory***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime l'historique du champ de recherche de l'Action Bar d'une fenêtre. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Supprime l'historique de recherche de la fenêtre courante
ActionBarSupprimeHistoriqueRecherche()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`ActionBarSupprimeHistoriqueRecherche([<Fenêtre>])`
---

**`<Fenêtre> : (Nom de fenêtre)`**

Nom de la fenêtre pour laquelle l'historique de recherche doit être supprimé. Si ce paramètre n'est pas précisé ou correspond à une chaîne vide, la suppression s'effectue sur la fenêtre active. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Si la fenêtre ne possède pas de champ Action Bar, une erreur fatale est affichée.

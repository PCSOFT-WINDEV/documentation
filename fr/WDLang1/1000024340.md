


## &lt;Image&gt;.SymétrieVerticale (Fonction)

***En anglais : &lt;Image&gt;.VerticalSymmetry***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Retourne l'image contenue dans un champ Image par rapport à un axe vertical (symétrie par rapport à un axe vertical).![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=dSymetrieVerticale.gif)


![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Les dessins](../WDLang1/3029035.md). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Symétrie verticale du champ IMG_Image1
MonRésultat est un booléen
MonRésultat = IMG_Image1.SymétrieVerticale()
SI MonRésultat = Faux ALORS Erreur("L'opération a échoué")
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Champ Image>.SymétrieVerticale()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la symétrie a été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Champ Image> : (Nom de champ)`**

Nom du champ Image à manipuler.

La symétrie est réalisée par rapport à l'axe vertical médian du champ. L'image est directement modifiée.

<a name="NOTE0_1"></a>

## Remarques


### Fin du dessin
<a name="fin_dessin_ELTPARAGRAPHE000347"></a>Si la fonction [&lt;Image&gt;.FinDessin](../WDLang1/1000024327.md) est appelée dans le même traitement que la fonction **&lt;Image&gt;.SymétrieVerticale**, la transformation de l'image ne sera pas affichée.

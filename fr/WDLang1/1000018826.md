
## ChargePlancheImage (Fonction)

***En anglais : LoadImageSheet***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Charge une planche d'images en mémoire pour une utilisation ultérieure par la fonction [gImage](../WDLang1/3029042.md). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// L'image MULTI.BMP est composée de 130 composantes
// Le cadre autour de chacune de ces composantes ne sera pas affiché
ChargePlancheImage(2, "MULTI.BMP", 16, 16)
// Affichage des images dans une liste graphique
POUR Indice = 1 A 10*13
	ListeAjoute(LISTE_ListeGrahique, gImage(2, Indice))
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = ChargePlancheImage(<Identifiant de l'image> , <Nom de l'image> , <Largeur d'une image> , <Hauteur d'une image> [, <Cadre>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la planche d'images a été chargée en mémoire, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Identifiant de l'image> : (Entier)`**

Identifiant de l'image (compris entre 1 et 9) chargée en mémoire. Si cet identifiant a déjà été utilisé, l'image chargée remplace l'image précédente. Cet identifiant pourra être utilisé par la fonction [gImage](../WDLang1/3029042.md).

**`<Nom de l'image> : (Chaîne de caractères)`**

Nom et chemin complet (ou relatif) de l'image à charger. Un chemin de type UNC peut être utilisé. 

**`<Largeur d'une image> : (Entier)`**

Largeur d'une image présente dans la planche d'images. 

**`<Hauteur d'une image> : (Entier)`**

Hauteur d'une image présente dans la planche d'images.

**`<Cadre> : (Caractère optionnel)`**

"C" si le cadre entourant chaque dessin ne doit pas être affiché. Par défaut, un cadre de 1 pixel d'épaisseur entourant chaque dessin de la planche d'images est affiché.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Dessins numérotés
<a name="dessins_numerotes_ELTPARAGRAPHE000054"></a>La planche d'images est composée de plusieurs dessins disposés en lignes et en colonnes. Tous ces dessins doivent avoir la même taille et peuvent être entourés d'un cadre de 1 pixel.

Les dessins sont numérotés de 1 à &lt;Nombre de lignes&gt; \* &lt;Nombre de colonnes&gt; de la gauche vers la droite et de haut en bas.

Remarques : 

- A la différence de la fonction [ChargeImage](../WDLang1/3029022.md), si le nombre de dessins change dans l'image, les images affichées ne seront pas tronquées et il ne sera pas nécessaire de modifier le code correspondant. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) La fonction [gImageExiste](../WDLang1/1000021860.md) permet de savoir si une image a été chargée en mémoire par la fonction [ChargeImage](../WDLang1/3029022.md) ou **ChargePlancheImage**.



<a name="NOTE0_2"></a>


### Limites
<a name="limites_ELTPARAGRAPHE000084"></a>La fonction **ChargePlancheImage** ne doit pas être utilisée dans le code du projet. En effet, cette fonction doit être utilisée dans la fenêtre où l'image doit être manipulée.

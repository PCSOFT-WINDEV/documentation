


## &lt;Variable Document&gt;.FinModification (Fonction)

***En anglais : &lt;Document variable&gt;.EndModification***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Signale la fin du regroupement d'un ensemble d'opérations sur une variable de type [Document](../WDLang1/1000022461.md) en un seul événement (gestion du "undo/redo").
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonDocument est un Document

MonDocument.DébutModification()
MonDocument.Remplace("%NomClient%", Client.Nom)
MonDocument.Remplace("%PrénomClient%", Client.Prénom)
MonDocument.FinModification()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Document>.FinModification()`
---

**`<Document> : (Variable de type Document)`**

Nom de la variable de type [Document](../WDLang1/1000022461.md) à manipuler.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Les modifications effectuées sur le document sont regroupées à partir de l'appel de la fonction [&lt;Variable Document&gt;.DébutModification](../WDLang1/1000024507.md). Si plusieurs modifications annulables sont effectuées, il suffit d'un seul "Ctrl + Z" pour annuler toutes les opérations effectuées.

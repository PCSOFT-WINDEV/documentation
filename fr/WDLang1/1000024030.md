


## &lt;Type Date&gt;.VersJour (Fonction)

***En anglais : &lt;Date type&gt;.ToDay***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le jour correspondant à une date donnée.










<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MaDate est une Date
MaDate = "20011225"
MaDate.VersJour() // Retourne 2

MaDate = DateSys
SI MaDate.VersJour() > 5 ALORS
	Info("C'est le week-end")
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Date>.VersJour()`
---

**`<Résultat> : (Entier)`**

Numéro du jour de la semaine correspondant à la date spécifiée :
1 - Lundi
2 - Mardi
3 - Mercredi
4 - Jeudi
5 - Vendredi
6 - Samedi
7 - Dimanche

**`<Date> : (Variable de type Date ou variable de type DateHeure)`**

Date à manipuler. Cette date peut correspondre à : 

- une variable de type Date. 

- une variable de type DateHeure. Dans ce cas, seule la date est prise en compte.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Le format de stockage des dates permet de stocker des dates comprises entre le 01/01/0001 et le 31/12/9999.

Les calculs sur les dates réalisés avec les [fonctions WLangage](../WDLang1/1000023943.md) ou les [propriétés WLangage](../Proprietes/2514010.md) sont fiables à partir du 1er Janvier 1583.

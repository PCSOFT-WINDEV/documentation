|   |   |

| [&lt;Variable FichierDisque&gt;.Cherche](../WDLang1/1410089039.md) | Cherche une chaîne de caractères ou un buffer dans un fichier ouvert avec la fonction [fOuvre](../WDLang1/3036036.md). |

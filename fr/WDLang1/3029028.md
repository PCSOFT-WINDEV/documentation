


## Fonctions de manipulation de chaînes graphiques
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions de gestion des chaînes graphiques permettent notamment d'afficher des chaînes graphiques dans : 

- les lignes des champs Liste ou Combo, 

- les champs Libellé. 

- la barre de titre ou de message d'une fenêtre. 




Les différentes fonctions permettant de créer et manipuler des chaînes graphiques sont les suivantes : 



|   |   |
| --- | --- |
| [ChargeImage](../WDLang1/3029022.md) | Charge une image en mémoire pour une utilisation ultérieure par la fonction [gImage](../WDLang1/3029042.md). |
| [ChargePlancheImage](../WDLang1/1000018826.md) | Charge une planche d'images en mémoire pour une utilisation ultérieure par la fonction [gImage](../WDLang1/3029042.md). |
| [gARécupérer](../WDLang1/3029025.md) | Décrit la valeur texte à afficher et à retourner lors de la récupération de la valeur d'un élément d'un champ Liste ou d'un champ Combo. |
| [gCadrage](../WDLang1/3029046.md) | Construit une séquence de caractères contenant un texte cadré ou une image cadrée. |
| [gCercle](../WDLang1/3029041.md) | Construit une séquence de caractères contenant un cercle ou une ellipse. |
| [gCoord](../WDLang1/3029034.md) | Construit une séquence de caractères permettant de définir la position du curseur d'écriture d'un texte. |
| [gFond](../WDLang1/3029037.md) | Construit une séquence de caractères permettant de définir la couleur d'affichage (couleur de fond d'un texte, d'un cercle, d'un rectangle, ...). |
| [gFondDégradé](../WDLang1/1000017060.md) | Décrit une couleur d'affichage en mode dégradé. |
| [gImage](../WDLang1/3029042.md) | Construit une séquence de caractères contenant une image. |
| [gImageExiste](../WDLang1/1000021860.md) | Vérifie si une image a bien été chargée en mémoire. |
| [gLien](../WDLang1/3029043.md) | Ajoute une valeur "invisible" dans un élément d'un champ Liste ou Combo.<br><br>Fonction est conservée par compatibilité. Il est conseillé d'utiliser la fonction [gValeurMémorisée](../WDLang1/1410088363.md). |
| [gLienActive](../WDLang1/3029036.md) | Autorise ou non la récupération de la valeur définie par la fonction [gLien](../WDLang1/3029043.md) dans un champ Liste ou dans un champ Combo.<br><br>Fonction est conservée par compatibilité. Il est conseillé d'utiliser la fonction [gValeurMémoriséeActive](../WDLang1/1410088364.md). |
| [gLienCliquable](../WDLang1/1000021774.md) | Ajoute un lien cliquable dans une chaîne graphique. |
| [gLigne](../WDLang1/3029044.md) | Construit une séquence de caractères contenant une ligne. |
| [gPolice](../WDLang1/3029047.md) | Construit une séquence de caractères décrivant la police d'affichage à utiliser pour le texte. |
| [gPoliceBarrée](../WDLang1/1000023596.md) | Construit une séquence de caractères permettant d'indiquer si le texte suivant l'appel de cette fonction est barré ou non. |
| [gPoliceGras](../WDLang1/3029061.md) | Construit une séquence de caractères permettant d'indiquer si le texte suivant l'appel de cette fonction est en gras ou non. |
| [gPoliceItalique](../WDLang1/3029062.md) | Construit une séquence de caractères permettant d'indiquer si le texte suivant l'appel de cette fonction est en italique ou non. |
| [gPoliceSoulignée](../WDLang1/3029063.md) | Construit une séquence de caractères permettant d'indiquer si le texte suivant l'appel de cette fonction est souligné ou non. |
| [gPoliceTaille](../WDLang1/3029064.md) | Construit une séquence de caractères permettant d'indiquer la taille de la police d'affichage du texte d'une chaîne graphique. |
| [gRectangle](../WDLang1/3029045.md) | Construit une séquence de caractères permettant de dessiner un rectangle. |
| [gStylo](../WDLang1/3029039.md) | Construit une séquence de caractères permettant de décrire la couleur d'affichage du texte et/ou du contour d'une figure (rectangle ou cercle). |
| [gValeurMémorisée](../WDLang1/1410088363.md) | Ajoute une valeur "invisible" dans un élément d'un champ Liste, Combo ou Planning. |
| [gValeurMémoriséeActive](../WDLang1/1410088364.md) | Autorise ou non la récupération de la valeur définie par la fonction [gValeurMémorisée](../WDLang1/1410088363.md) dans un champ Liste ou dans un champ Combo. |
| [gVersTexte](../WDLang1/1000020524.md) | Supprime les éléments graphiques d'une chaîne de caractères utilisant les fonctions [gValeurMémorisée](../WDLang1/1410088363.md), [gImage](../WDLang1/3029042.md), ... |


## FenEdDésélectionneTout (Fonction)

***En anglais : WinEdDeselectAll***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Désélectionne tous les champs lors de la modification de l'interface par l'utilisateur final (mode "Modification de l'UI par l'utilisateur final" activé). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Active l'édition sur la fenêtre 
FenEdActive(FEN_Formulaire, Vrai)
FenEdDésélectionneTout(FEN_Formulaire)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`FenEdDésélectionneTout(<Fenêtre>)`
---

**`<Fenêtre> : (Nom de fenêtre)`**

Nom de la fenêtre ou de la fenêtre interne à manipuler. Cette fenêtre doit être en mode modification (fonction [FenEdActive](../WDLang1/1000021843.md)).

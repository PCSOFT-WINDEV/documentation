
## DocPositionDébutPage (Fonction)

***En anglais : DocPositionTopPage***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la position du premier caractère d'une page donnée dans un champ Traitement de texte.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Insérer une image sur la page 2
nPosPage2 est entier = DocPositionDébutPage(TT_MonDocument, 2)
DocInsère(TT_MonDocument, nPosPage2, ElémentEntête)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = DocPositionDébutPage(<Champ Traitement de texte> , <Numéro de page>)`
---

**`<Résultat> : (Entier)`**

Position du premier caractère de la page spécifiée. Cet position peut être utilisée avec la fonction [DocInsère](../WDLang1/1000022732.md) pour insérer un élément dans le champ Traitement de texte. 

**`<Champ Traitement de texte> : (Nom de champ)`**

Nom du champ Traitement de texte à manipuler.

**`<Numéro de page> : (Entier)`**

Numéro de la page pour laquelle la position est recherchée.

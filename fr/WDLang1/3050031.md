
## MatSupprime (Fonction)

***En anglais : MatDelete***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime une matrice existante. La suppression d'une matrice inutilisée (ou nulle) dans une application permet de libérer la place mémoire utilisée par cette matrice.

**Remarque** : Toutes les matrices utilisées dans une application sont systématiquement supprimées à la fermeture de l'application.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple
<a class="notetitle" target="_blank" href="$DOC$=1000003050031&name=matsupprime_fonction&product=WD">Voir des exemples supplémentaires</a>

```wl
// Suppression d'une matrice
MatSupprime("MaMatrice")
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`MatSupprime(<Matrice>)`
---

**`<Matrice> : (Chaîne de caractères)`**

Nom de la matrice à supprimer, défini avec la fonction [MatCrée](../WDLang1/3050058.md).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Existence de la matrice à supprimer
<a name="existence_matrice_supprimer_ELTPARAGRAPHE000077"></a>L'existence de la matrice n'est pas vérifiée. Pour vérifier l'existence d'une matrice, utilisez la fonction [MatExiste](../WDLang1/3050076.md). Si la matrice n'existe pas, la fonction **MatSupprime** n'a aucune action.

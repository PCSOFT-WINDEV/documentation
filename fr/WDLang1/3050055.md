
## FinValActuelle (Fonction)

***En anglais : FinCurrentVal***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la valeur actuelle d'un investissement.

**Rappel** : La valeur actuelle correspond à la somme que représente aujourd'hui un ensemble de remboursements futurs. Par exemple, lorsque vous faites un emprunt, le montant de l'emprunt représente la valeur actuelle pour le prêteur.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple
<a class="notetitle" target="_blank" href="$DOC$=1000003050055&name=finvalactuelle_fonction&product=WD">Voir des exemples supplémentaires</a>

```wl
// Valeur actuelle d'un investissement. La valeur de remboursement est de 5 000 Euros.
// Cet investissement doit rapporter 50 000 Euros par an pendant 10 ans.
// Le taux d'intérêt est de 8%.
ResValActuelle = FinValActuelle(-5000, 50000, 10, 8)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = FinValActuelle([<Valeur d'un remboursement> [, <Valeur future> [, <Nombre de périodes> [, <Taux> [, <Début>]]]]])`
---

**`<Résultat> : (Réel ou monétaire)`**

Valeur actuelle de l'investissement.

**`<Valeur d'un remboursement> : (Réel ou monétaire optionnel)`**

Valeur du remboursement pour chaque période (0 par défaut). Ce remboursement est identique pour toute la durée de l'opération. Cette valeur doit être négative.

**`<Valeur future> : (Réel ou monétaire optionnel)`**

Valeur future (valeur capitalisée). Il s'agit du montant après le dernier paiement (0 par défaut).

**`<Nombre de périodes> : (Entier optionnel)`**

Nombre total de périodes de l'investissement (1 par défaut).

**`<Taux> : (Réel optionnel)`**

Taux d'intérêt de l'investissement exprimé en pourcentage (1 par défaut). Par exemple, pour un taux d'investissement de 5%, ce paramètre sera égal à 5.

**`<Début> : (Booléen optionnel)`**



- <u><u><u><u>Vrai</u></u></u></u> si les paiements doivent être effectués en début de période (valeur par défaut), 

- <u><u><u><u>Faux</u></u></u></u> si les paiements doivent être effectués en fin de période.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Unité temporelle
<a name="unite_temporelle_ELTPARAGRAPHE000087"></a>Les calculs peuvent être réalisés en prenant en compte des mois ou des années. Il suffit que les paramètres &lt;Taux&gt; et &lt;Période&gt; correspondent à la même unité temporelle.

Ainsi, pour obtenir la valeur actuelle d'un emprunt sur 4 ans, remboursable mensuellement, avec un taux d'intérêt annuel de 10%, il est possible d'utiliser des unités exprimées en mois :

- &lt;Période&gt; = 4\*12,

- &lt;Taux&gt; = 10% / 12





<a name="NOTE0_2"></a>




### Gestion des erreurs
<a name="gestion_des_erreurs_ELTPARAGRAPHE000101"></a>**Attention** : La fonction **FinValActuelle** ne retourne pas de code d'erreur. Pour savoir si cette fonction a généré des erreurs, utilisez la fonction [FinErreur](../WDLang1/3050071.md). Le libellé de l'erreur peut être connu grâce à la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.

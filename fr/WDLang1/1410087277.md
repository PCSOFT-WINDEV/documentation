
## LuhnVérifie (Fonction)

***En anglais : LuhnCheck***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Vérifie que la suite donnée respecte bien la formule de Luhn. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
sNuméroSiret est une chaîne = SAI_SaisieSIRET
SI LuhnVérifie(sNuméroSiret) ALORS
	Info("Numéro valide")
SINON
	Info("Numéro invalide")
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = LuhnVérifie(<Identifiant>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la suite respecte la formule de Luhn, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 




**`<Identifiant> : (Chaîne de caractères)`**

Suite impérativement composée uniquement de chiffres et d'espaces (les espaces ne sont pas déterminants pour la formule).

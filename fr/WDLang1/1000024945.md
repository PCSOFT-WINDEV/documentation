


## Fonctions de gestion des continents et des pays
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions de gestion des continents et des pays sont les suivantes :



|   |   |
| --- | --- |
| [ContinentListe](../WDLang1/1000024932.md) | Renvoie la liste des continents. |
| [ContinentRécupère](../WDLang1/1000024933.md) | Renvoie la variable de type [Continent](../WDLang1/1000024919.md) correspondant à une des valeurs suivantes : <br><br>- une constante continent (code continent sur 2 caractères), <br><br>- un libellé de continent.<br><br><br> |
| [PaysListe](../WDLang1/1000024930.md) | Renvoie la liste de tous les pays ou des pays d'un continent donné selon la norme ISO 3166-1. |
| [PaysRécupère](../WDLang1/1000024931.md) | Renvoie la variable de type Pays correspondant à un code ISO ou un libellé de pays à la norme ISO 3166-1. |



Ces fonctions manipulent les types de variables suivants : 



|   |   |
| --- | --- |
| [Continent](../WDLang1/1000024919.md) | Le type **Continent** permet de connaître les caractéristiques d'un continent. |
| [Pays](../WDLang1/1000024921.md) | Le type **Pays** permet de connaître les caractéristiques d'un pays. |

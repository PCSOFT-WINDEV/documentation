


## PlanningSelectPlus (Fonction)

***En anglais : SchedulerSelectPlus***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Sélectionne un rendez-vous dans un champ Planning.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Sélectionne le premier rendez-vous du champ Planning
PlanningSelectPlus(PLN_Planning, 1)

// Récupère la description de ce rendez-vous
taRendezVous est un tableau de RendezVous
taRendezVous = PlanningListeRendezVous(PLN_Planning, plnRdvSélectionné)
```
<a name="Exemple2"></a>

```wl
// Ajoute un rendez-vous dans le champ Planning puis sélectionne ce rendez-vous
MonRendezVous est un RendezVous

IndiceAjout est un entier 

IndiceAjout = PlanningAjouteRendezVous(PLN_Planning, MonRendezVous)
PlanningSelectPlus(PLN_Planning, IndiceAjout)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Sélectionner un rendez-vous identifié par son indice

`PlanningSelectPlus(<Champ Planning> , <Indice du rendez-vous>)`
---

**`<Champ Planning> : (Nom de champ)`**

Nom du champ Planning à manipuler.

**`<Indice du rendez-vous> : (Entier)`**

Indice du rendez-vous à sélectionner. Les indices des rendez-vous commencent à 1.


<a name="SYNTAXE2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 
### Sélectionner un rendez-vous identifié par une variable de type RendezVous

`PlanningSelectPlus(<Champ Planning> , <Rendez-vous manipulé>)`
---

**`<Champ Planning> : (Nom de champ)`**

Nom du champ Planning à manipuler.

**`<Rendez-vous manipulé> : (Variable de type RendezVous)`**

Variable de type [RendezVous](../WDLang1/1000019244.md) correspondant au rendez-vous à sélectionner.

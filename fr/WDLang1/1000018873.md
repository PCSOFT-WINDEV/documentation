
## SysMiniatureSupprimeTout (Fonction)

***En anglais : SysThumbnailDeleteAll***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime tous les boutons associés à la miniature de l'application dans la barre des tâches. Il est ensuite possible d'ajouter de nouveaux boutons avec la fonction [SysMiniatureAjouteBouton](../WDLang1/1000018871.md). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Suppression de tous les boutons liés à la miniature de l'application
SysMiniatureSupprimeTout()
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = SysMiniatureSupprimeTout()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si toutes les boutons associés à la miniature ont été supprimés, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.


## InfoVersion (Type de variable)

***En anglais : InfoVersion***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **InfoVersion** permet de connaître les caractéristiques d'une version spécifique d'une application. Les caractéristiques de cette version peuvent être connues à l'aide de différentes propriétés WLangage.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
sRes est une chaîne = AppliListeVersion()
IV est un InfoVersion

sAffiche est une chaîne = [
Numéro = %1
Date = %2
Taille = %3 octets
Commentaire = %4
Actif = %5
Forcé = %6
Interdite = %7
]

POUR TOUTE CHAINE MaChaîne DE sRes SEPAREE PAR RC
	IV = AppliInfoVersion(MaCHaîne)
	Trace(ChaîneConstruit(sAffiche, ...
		IV.NuméroVersion, IV.Date, ...
		IV.Taille, IV.Commentaire, ...
		IV.Activé, IV.Forcé, IV.Interdit))
	Trace("===============================================")
FIN
```

## Remarques


### Propriétés spécifiques à la description des variables de type InfoVersion
<a name="proprietes_specifiques_description_des_variables_type_infoversion_ELTPARAGRAPHE000040"></a>

Les propriétés suivantes peuvent être utilisées pour manipuler une variable de type **InfoVersion** :


| Nom de la propriété | Type manipulé | Effet |
| --- | --- | --- |
| Activé | Booléen | <br><br>- <u><u><u><u>Vrai</u></u></u></u> si la version est activée (c'est-à-dire visible par les postes client),<br><br>- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.<br><br><br>**Cette propriété est disponible en lecture seulement**.<br><br> |
| Commentaire | Chaîne de caractères | Informations concernant les nouveautés de la version. Ces informations ont été renseignées par le développeur. Le contenu de cette propriété peut être : <br><br>- un texte, <br><br>- un fichier texte, RTF ou CHM,<br><br>- un "lien" vers un page d'aide (aide des nouveautés générée à la création de l'exécutable). <br><br><br>**Cette propriété est disponible en lecture seulement**. |
| Date | DateHeure | Date de mise en ligne de la version. <br>**Cette propriété est disponible en lecture seulement**. |
| Forcé | Booléen | <br><br>- <u><u><u><u>Vrai</u></u></u></u> si la version est forcée, <br><br>- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. <br><br><br>**Cette propriété est disponible en lecture seulement**.<br><br> |
| Interdit | Booléen | <br><br>- <u><u><u><u>Vrai</u></u></u></u> si cette version est interdite, <br><br>- <u><u><u><u>Faux</u></u></u></u> si la version est autorisée. <br><br><br>**Cette propriété est disponible en lecture seulement**.<br><br> |
| NuméroVersion | Chaîne de caractères | Numéro de la version. <br>**Cette propriété est disponible en lecture seulement**. |
| Taille | Entier | Taille de la version sur le disque (exprimée en octets). <br>**Cette propriété est disponible en lecture seulement**. |

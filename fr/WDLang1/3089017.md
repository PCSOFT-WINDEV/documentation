
## Constantes de gestion du planificateur Windows
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


Les différentes constantes de gestion du planificateur Windows :

| Constante | Description | Utilisée par la structure / fonction |
| --- | --- | --- |
| **htAoût** | Mois d'exécution de la tâche : Août | [HoraireTâchePlanifiée](../WDLang1/3089002.md) |
| **htAvril** | Mois d'exécution de la tâche : Avril | [HoraireTâchePlanifiée](../WDLang1/3089002.md) |
| **htConnexion** | Type de tâche : Exécution à l'ouverture d'une session | [HoraireTâchePlanifiée](../WDLang1/3089002.md) |
| **htDécembre** | Mois d'exécution de la tâche : Décembre | [HoraireTâchePlanifiée](../WDLang1/3089002.md) |
| **htDémarrage** | Type de tâche : Exécution au démarrage du système | [HoraireTâchePlanifiée](../WDLang1/3089002.md) |
| **htDernier** | Semaine d'exécution de la tâche : Dernière semaine du mois | [HoraireTâchePlanifiée](../WDLang1/3089002.md) |
| **htDeuxième** | Semaine d'exécution de la tâche : Deuxième semaine du mois | [HoraireTâchePlanifiée](../WDLang1/3089002.md) |
| **htDimanche** | Jour d'exécution de la tâche : dimanche | [HoraireTâchePlanifiée](../WDLang1/3089002.md) |
| **htFévrier** | Mois d'exécution de la tâche : Février | [HoraireTâchePlanifiée](../WDLang1/3089002.md) |
| **htHebdomadaire** | Type de tâche : Planification à la semaine | [HoraireTâchePlanifiée](../WDLang1/3089002.md) |
| **htInactivité** | Type de tâche : Exécution lorsque le système est inactif | [HoraireTâchePlanifiée](../WDLang1/3089002.md) |
| **htJanvier** | Mois d'exécution de la tâche : Janvier | [HoraireTâchePlanifiée](../WDLang1/3089002.md) |
| **htJeudi** | Jour d'exécution de la tâche : jeudi | [HoraireTâchePlanifiée](../WDLang1/3089002.md) |
| **htJuillet** | Mois d'exécution de la tâche : Juillet | [HoraireTâchePlanifiée](../WDLang1/3089002.md) |
| **htJuin** | Mois d'exécution de la tâche : Juin | [HoraireTâchePlanifiée](../WDLang1/3089002.md) |
| **htLundi** | Jour d'exécution de la tâche : lundi | [HoraireTâchePlanifiée](../WDLang1/3089002.md) |
| **htMai** | Mois d'exécution de la tâche : Mai | [HoraireTâchePlanifiée](../WDLang1/3089002.md) |
| **htMardi** | Jour d'exécution de la tâche : mardi | [HoraireTâchePlanifiée](../WDLang1/3089002.md) |
| **htMars** | Mois d'exécution de la tâche : Mars | [HoraireTâchePlanifiée](../WDLang1/3089002.md) |
| **htMensuel** | Type de tâche : Planification au mois | [HoraireTâchePlanifiée](../WDLang1/3089002.md) |
| **htMercredi** | Jour d'exécution de la tâche : mercredi | [HoraireTâchePlanifiée](../WDLang1/3089002.md) |
| **htNovembre** | Mois d'exécution de la tâche : Novembre | [HoraireTâchePlanifiée](../WDLang1/3089002.md) |
| **htOctobre** | Mois d'exécution de la tâche : Octobre | [HoraireTâchePlanifiée](../WDLang1/3089002.md) |
| **htPremier** | Semaine d'exécution de la tâche : Première semaine du mois | [HoraireTâchePlanifiée](../WDLang1/3089002.md) |
| **htQuatrième** | Semaine d'exécution de la tâche : Quatrième semaine du mois | [HoraireTâchePlanifiée](../WDLang1/3089002.md) |
| **htQuotidien** | Type de tâche : Planification par jour | [HoraireTâchePlanifiée](../WDLang1/3089002.md) |
| **htSamedi** | Jour d'exécution de la tâche : samedi | [HoraireTâchePlanifiée](../WDLang1/3089002.md) |
| **htSeptembre** | Mois d'exécution de la tâche : Septembre | [HoraireTâchePlanifiée](../WDLang1/3089002.md) |
| **htTroisième** | Semaine d'exécution de la tâche : Troisième semaine du mois | [HoraireTâchePlanifiée](../WDLang1/3089002.md) |
| **htUneFois** | Type de tâche : Exécution une seule fois | [HoraireTâchePlanifiée](../WDLang1/3089002.md) |
| **htVendredi** | Jour d'exécution de la tâche : vendredi | [HoraireTâchePlanifiée](../WDLang1/3089002.md) |
| **planificateurRAZHoraire** | Structure à ré-initialise : HoraireTâchePlanifiée | [PlanificateurRAZ](../WDLang1/3089012.md) |
| **planificateurRAZTâche** | Structure à ré-initialise : TâchePlanifiée | [PlanificateurRAZ](../WDLang1/3089012.md) |
| **tâchePlanifiéeActive** | Action à effectuer : Active une tâche | [PlanificateurEtatTâche](../WDLang1/3089005.md) |
| **tâchePlanifiéeAucunHoraire** | État de la tâche : La tâche n'a aucun horaire ou les horaires de la tâche sont tous invalides ou inactifs | [PlanificateurEtatTâche](../WDLang1/3089005.md) |
| **tâchePlanifiéeDésactivée** | État de la tâche : La tâche est désactivée (elle ne sera jamais exécutée jusqu'à sa ré-activation) | [PlanificateurEtatTâche](../WDLang1/3089005.md) |
| **tâchePlanifiéeDésactive** | Action à effectuer : Désactive la tâche spécifiée | [PlanificateurEtatTâche](../WDLang1/3089005.md) |
| **tâchePlanifiéeErreur** | État de la tâche : La tâche spécifiée n'existe pas | [PlanificateurEtatTâche](../WDLang1/3089005.md) |
| **tâchePlanifiéeExécute** | Action à effectuer : Exécute immédiatement la tâche indépendamment de sa planification | [PlanificateurEtatTâche](../WDLang1/3089005.md) |
| **tâchePlanifiéeExécution** | État de la tâche : La tâche planifiée est en cours d'exécution | [PlanificateurEtatTâche](../WDLang1/3089005.md) |
| **tâchePlanifiéeNonPlanifiée** | État de la tâche : Les propriétés de la tâche ne permettent pas de l'exécuter | [PlanificateurEtatTâche](../WDLang1/3089005.md) |
| **tâchePlanifiéePasExécutée** | État de la tâche : La tâche n'a pas encore été exécutée | [PlanificateurEtatTâche](../WDLang1/3089005.md) |
| **tâchePlanifiéePlusExécutée** | État de la tâche : La tâche n'a plus d'exécutions planifiées | [PlanificateurEtatTâche](../WDLang1/3089005.md) |
| **tâchePlanifiéePrête** | État de la tâche : La tâche est prête à être exécutée, en attente dans le planificateur | [PlanificateurEtatTâche](../WDLang1/3089005.md) |
| **tâchePlanifiéeTerminée** | État de la tâche : La dernière exécution de la tâche a été terminée par l'utilisateur | [PlanificateurEtatTâche](../WDLang1/3089005.md) |
| **tâchePlanifiéeTermine** | Action à effectuer : Si la tâche est dans l'état tâchePlanifiéeExécution, termine son exécution | [PlanificateurEtatTâche](../WDLang1/3089005.md) |

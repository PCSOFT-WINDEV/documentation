
## IndiceEnCours (Fonction)

***En anglais : CurrentIndex***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie selon le champ spécifié :

- le numéro de la ligne en cours affichée dans le champ Table en cours,

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) le numéro de l'option d'un champ Interrupteur ayant le focus,

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) l'indice du champ indicé en cours.









<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
NumLigne est un entier = IndiceEnCours()
Trace("Numéro de la ligne en cours: " + NumLigne)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = IndiceEnCours()`
---

**`<Résultat> : (Entier)`**



- Numéro de la ligne en cours affichée dans le champ Table en cours.

- Numéro de l'option du champ Interrupteur en cours.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Indice du champ indicé en cours.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Utilisation dans les événements d'un champ Table
<a name="utilisation_dans_les_evenements_champ_table_ELTPARAGRAPHE000079"></a>Lors de l'initialisation d'un champ Table, l'événement "Affichage d'une ligne" est exécuté pour chaque ligne affichée dans le champ Table. 

Lors de l'initialisation d'un champ Table, si la fonction IndiceEnCours est utilisée dans l'événement "Affichage d'une ligne", la fonction **IndiceEnCours** renvoie 0.






### Limites
<a name="limites_ELTPARAGRAPHE000094"></a>La fonction **IndiceEnCours** peut être utilisée avec : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) les champs Table, les champs Interrupteur et les champs Indicés.


## OuvreMenuPrincipal (Fonction)

***En anglais : OpenMainMenu***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ouvre le menu principal de la fenêtre en cours. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
OuvreMenuPrincipal()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`OuvreMenuPrincipal()`
---



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- S'il n'y a pas de fenêtre en cours lors de l'appel de la fonction ou si la fenêtre en cours n'a pas de menu principal, la fonction est sans effet.

- A partir de la version 3.0 d'Android, le bouton de menu ne sera pas systématiquement affiché dans la barre de navigation de l'appareil.
	Pour afficher le menu principal de la fenêtre, il est conseillé :

	- soit d'utiliser une [Action Bar](../WDChamp/1000025001.md) dans la fenêtre : dans ce cas le menu pourra être ouvert depuis l'Action Bar.

	- soit d'appeler la fonction **OuvreMenuPrincipal** depuis un bouton de la fenêtre (par exemple).




- Cette fonction peut être utilisée dans le simulateur.

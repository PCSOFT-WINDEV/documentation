


## MenuAjoutePopup (Fonction)

***En anglais : MenuAddPopup***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Transforme une option de menu d'une page pour que cette option ouvre une popup.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MenuAjoutePopup(MENU_Principal.OPT_Produits, POPUP_Produits)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`MenuAjoutePopup(<Nom de l'option> , <Popup à afficher> [, <Position>])`
---

**`<Nom de l'option> : (Nom d'option)`**

Nom de l'option de menu qui doit ouvrir une popup. Cette option de menu doit être une option de menu simple. Elle ne doit pas ouvrir un sous-menu ou être une option de type séparateur. 

**`<Popup à afficher> : (Nom de popup)`**

Nom de la popup qui sera affichée par l'option.

**`<Position> : (Constante de type entier optionnel)`**

Position d'ouverture de la popup :



|   |   |
| --- | --- |
| *menuPopupBasCentréMenu* | Ouvre la popup sous le menu en la centrant sur la largeur totale du menu (deux popups de tailles identiques et ouvertes par deux options différentes s'affichent au même endroit avec cette option). |
| *menuPopupBasDroite* | Ouvre la popup sous l'option de menu en l'alignant sur le bord droit de l'option de menu (la popup s'affiche vers la gauche). |
| *menuPopupBasGauche*<br>(Valeur par défaut) | Ouvre la popup sous l'option de menu en l'alignant sur le bord gauche de l'option de menu. |
| *menuPopupHautDroite* | Ouvre la popup à droite de l'option de menu en l'alignant sur le bord haut de l'option de menu (cette constante est principalement utilisée avec les menus verticaux). |
| *menuPopupHautGauche* | Ouvre la popup à gauche de l'option de menu en l'alignant sur le bord haut de l'option de menu (cette constante est principalement utilisée avec les menus verticaux). |

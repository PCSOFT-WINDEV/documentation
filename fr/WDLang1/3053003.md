 

## MemTri (Fonction)

***En anglais : MemSort_55***
	

<a name="XUtilisation"></a>
<a name="Utilisation"></a>
|  |   |
| --- | --- |



|  
 | ButEffectuer le tri d'une une zone mémoire selon l'argument de tri donné dans MemAjoute.**Syntaxe**MemTri(&lt;Zone_Mémoire&gt; [, &lt;sens&gt;])<br><br>- &lt;Zone_Mémoire&gt; est une chaîne de caractères contenant le nom de la zone mémoire (20 caractères maximum)<br><br>- &lt;sens&gt; indique le sens du tri<br><br><br>DétailLa zone mémoire est triée :<br><br>- dans l'ordre croissant, si &lt;Sens&gt; est à Vrai (valeur par défaut)<br><br>- dans l'ordre décroissant, si &lt;Sens&gt; est à Faux.<br><br><br><br><br>Si &lt;Zone_Mémoire&gt; n'existe pas, une erreur du WLangage est générée lors de l'exécution. Avant de manipuler une zone mémoire, elle doit être créée avec la fonction MemCrée. Son existence est vérifiée avec la fonction MemExiste.NotesLe tri est utilisé par les fonctions de parcours MemPremier, MemDernier, MemSuivant et MemPrécédent et MemRecherche.<br><br>L'argument de tri est indiqué dans MemAjoute et MemModifie.<br><br>Après un ou plusieurs ajouts ou modifications dans la zone mémoire, il faut demander de nouveau de tri avec MemTri.<br><br> |




## &lt;Diagramme de Gantt&gt;.OrganiseTâche (Fonction)

***En anglais : &lt;Gantt chart&gt;.OrganizeTask***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Réorganise les tâches d'un diagramme de Gantt selon les contraintes (dépendances et ressources).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Réorganise les tâches
COL_Gantt.OrganiseTâche()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Colonne Gantt>.OrganiseTâche()`
---

**`<Colonne Gantt> : (Nom de champ)`**

Nom de la colonne (d'un champ Table ou Table hiérarchique) de type Diagramme de Gantt à manipuler.

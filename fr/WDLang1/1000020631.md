


## Fonctions de gestion des menus (syntaxe préfixée)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions de gestion des menus (syntaxe préfixée) sont les suivantes : 




|   |   |
| --- | --- |
| [&lt;Menu&gt;.Ajoute](../WDLang1/1000020591.md) | Ajoute un menu contextuel existant dans le menu principal de la fenêtre en cours. |
| [&lt;Menu&gt;.AjouteMenu](../WDLang1/1000020592.md) | Ajoute un nouveau menu dans un menu existant (déroulant ou contextuel) d'une fenêtre ou d'une page. |
| [&lt;Menu&gt;.AjouteOption](../WDLang1/1000020593.md) | Ajoute une nouvelle option de menu à la suite de la dernière option d'un menu (déroulant ou contextuel). |
| [&lt;Menu&gt;.AjouteOptionURL](../WDLang1/1000020594.md) | Ajoute une nouvelle option de menu à la suite de la dernière option d'un menu (déroulant ou contextuel). Cette option de menu permet d'afficher la page correspondant à l'URL indiquée. |
| [&lt;Menu&gt;.AjoutePopup](../WDLang1/1000020602.md) | Transforme une option de menu d'une page pour que cette option ouvre une popup. |
| [&lt;Menu&gt;.AjouteSéparateur](../WDLang1/1000020603.md) | Ajoute un nouveau séparateur dans un menu (menu déroulant ou contextuel). |
| [&lt;Menu&gt;.Clone](../WDLang1/1000020604.md) | Clone un menu ou une option de menu ainsi que le code associé. |
| [&lt;Menu&gt;.Existe](../WDLang1/1000020605.md) | Indique si une option de menu existe dans un menu. |
| [&lt;Menu&gt;.InsèreMenu](../WDLang1/1000020606.md) | Insère un menu avant un autre menu dans une fenêtre. |
| [&lt;Menu&gt;.InsèreOption](../WDLang1/1000020609.md) | Insère une nouvelle option à une position spécifique. |
| [&lt;Menu&gt;.InsèreSéparateur](../WDLang1/1000020610.md) | Insère un séparateur dans un menu. |
| [&lt;Menu&gt;.Supprime](../WDLang1/1000020614.md) | Supprime un menu ou une option de menu. |





Les fonctions suivantes permettent également de gérer les menus : 


|   |   |
| --- | --- |
| [&lt;Graphe&gt;.Menu](../WDLang3/1000024094.md) | Active ou désactive le menu contextuel d'un graphe. |
| [ChampContextuel](../WDLang1/3025012.md) | Identifie le champ sur lequel le menu contextuel a été ouvert. |
| [EnumèreMenu](../WDLang1/3038039.md) | Utilisée dans une boucle, cette fonction permet d'énumérer les options de menu d'une fenêtre, comme les sous options d'une option de menu. |
| [EnumèreSousElément](../WDLang1/3064002.md) | Utilisée dans une boucle, cette fonction permet d'énumérer les menus déroulants ou les menus contextuels d'une fenêtre ou d'un champ. |
| [OuvreMenuContextuel](../WDLang1/3025029.md) | Ouvre automatiquement un menu contextuel pour le champ ou la fenêtre en cours. |
| [PiloteMenu](../WDLang1/1000009010.md) | Pilote le prochain menu qui sera ouvert, puis exécute automatiquement une option de ce menu indiquée en paramètre. |


## CaméraListeDriver (Fonction)

***En anglais : CameraListDriver***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Liste les drivers de capture vidéo installés sur le poste en cours.

Remarque : Cette fonction est équivalente à la fonction [VidéoListeDriver](../WDLang1/3079008.md). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connaître les drivers Vidéo installés
// Affichage dans la liste LISTE_ListeDriver
ListeAjoute(LISTE_ListeDriver, CaméraListeDriver(viDirectShow))
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = CaméraListeDriver([<Technologie utilisée>])`
---

**`<Résultat> : (Chaîne de caractères)`**

Liste des drivers vidéo installés sur le poste, sous la forme :

```txt
<Nom du driver 1> + TAB + <Version du driver 1> + RC + ... 
<Nom du driver N> + TAB + <Version du driver N>
```


Où : 

- &lt;Nom du driver 1&gt; ... &lt;Nom du driver N&gt; correspondent au nom des drivers vidéos, 

- &lt;Version du driver 1&gt; ... &lt;Version du driver N&gt; correspondent à la version du driver.




**`<Technologie utilisée> : (Constante de type Entier)`**

Type de technologie utilisée pour lister les drivers : 


|   |   |
| --- | --- |
| *vidDirectShow* | Technologie utilisée : Direct Show. <br><br>Cette constante doit être utilisée pour sélectionner un driver utilisé dans un champ Conférence (fonction [FluxChangeDriver](../WDLang3/1000021613.md)). |
| *vidVideoForWindows*<br>(Valeur par défaut) | Technologie utilisée : Video for Windows |

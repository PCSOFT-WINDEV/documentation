
## Fonctions de gestion des champs Barre d'outils (syntaxe préfixée)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions permettant de manipuler le champ Barre d'outils via une syntaxe préfixée sont les suivantes :



|   |   |
| --- | --- |
| [&lt;Barre d'outils&gt;.Attache](../WDLang1/3020006.md) | Ancre un champ Barre d'outils à la fenêtre ou change la position d'ancrage d'un champ Barre d'outils ancré. |
| [&lt;Barre d'outils&gt;.Détache](../WDLang1/3020008.md) | Transforme un champ Barre d'outils ancré en champ Barre d'outils flottant. |
| [&lt;Barre d'outils&gt;.Position](../WDLang1/3020007.md) | Indique le mode d'affichage d'un champ Barre d'outils. |


## TableurListeFeuille (Fonction)

***En anglais : SpreadsheetListWorksheet***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la liste des feuilles de calcul présentes dans un champ Tableur.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
TabNom est un tableau de chaînes
TabNom = TableurListeFeuille(TBLR_Tableur1) 
POUR TOUT sNomFeuille DE TabNom
	ListeAjoute(LISTE_MesFeuilles, sNomFeuille)	
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = TableurListeFeuille(<Champ Tableur>)`
---

**`<Résultat> : (Tableau de chaînes)`**

Tableau contenant le nom de chaque feuille trouvée dans le champ Tableur. 

**`<Champ Tableur> : (Nom de champ)`**

Nom du champ Tableur à manipuler.

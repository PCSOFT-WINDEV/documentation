


## Fonctions de gestion des champs Interrupteur
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


Les fonctions permettant de gérer les champs Interrupteur sont les suivantes :



|   |   |
| --- | --- |
| [InterrupteurAjoute](../WDLang1/1000020595.md) | Ajoute une nouvelle option dans un champ Interrupteur. |
| [InterrupteurInsère](../WDLang1/1000020597.md) | Insère une nouvelle option dans un champ Interrupteur. |
| [InterrupteurOccurrence](../WDLang1/1000021274.md) | Renvoie le nombre d'options présentes dans un champ Interrupteur. |
| [InterrupteurSupprime](../WDLang1/1000020596.md) | Supprime une option d'un champ Interrupteur. |

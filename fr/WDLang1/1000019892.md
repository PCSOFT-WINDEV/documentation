
## Somme (Fonction)

***En anglais : Sum***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Calcule la somme des éléments d'un tableau. 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Somme des éléments d'un tableau d'entiers
MonTableau est un tableau d'entiers = [1,2,3]
MaSomme est un numérique
MaSomme = Somme(MonTableau) // MaSomme = 6
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Calculer la somme des éléments d'un tableau à une dimension (types numériques)

`<Résultat> = Somme(<Tableau>)`
---

**`<Résultat> : (Numérique)`**

Somme des éléments numériques du tableau. 

**`<Tableau> : (Variable de type Tableau)`**

Nom du tableau à une dimension de types numériques contenant les éléments dont la somme doit être calculée. 


<a name="SYNTAXE2"></a>

### Calculer la somme des éléments d'un tableau à une dimension (types structurés)

`<Résultat> = Somme(<Tableau> , <Membre>)`
---

**`<Résultat> : (Numérique)`**

Somme des valeurs des membres. 

**`<Tableau> : (Variable de type Tableau)`**

Nom du tableau à une dimension de types structurés. Les types structurés autorisés sont les suivants : classe, classe dynamique, structure, structure dynamique.

Remarque : Dans un tableau de classes dynamiques ou de structures dynamiques, si l'élément n'est pas alloué, il est ignoré. 

**`<Membre> : (Chaîne de caractères)`**

Nom du membre contenant les valeurs à utiliser pour le calcul de la somme. 


<a name="SYNTAXE3"></a>

### Calculer la somme des éléments d'une colonne d'un tableau à 2 dimensions

`<Résultat> = Somme(<Tableau> , <Colonne>)`
---

**`<Résultat> : (Numérique)`**

Somme des valeurs de la colonne. 

**`<Tableau> : (Variable de type Tableau)`**

Nom du tableau à 2 dimensions de type numérique. 

**`<Colonne> : (Entier)`**

Indice de la colonne du tableau contenant les valeurs à utiliser pour le calcul de la somme. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Types gérés
<a name="types_geres_ELTPARAGRAPHE000120"></a>Les types gérés pour les tableaux de numériques et pour les membres des tableaux de types structurés sont les suivants : 

- tous les types entier (sur 1, 2, 4 ou 8 octets, signés ou non signés + entier système).

- les types réel (sur 4 ou 8 octets), monétaire et numérique.

- le type Durée.





### Somme de durées
<a name="somme_durees_ELTPARAGRAPHE000131"></a>Si la fonction **Somme** est utilisée pour calculer une somme de durées, tous les paramètres doivent être de type Durée.



**Enchaînement de fonctions**

- [Distinct](../WDLang1/1000025406.md).




## DocRedoOccurrence (Fonction)

***En anglais : DocRedoCount***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le nombre d'actions annulées qui peuvent être refaites sur un document de type Traitement de texte.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affiche le nombre d'actions annulées et qui peuvent être refaites
Info(DocRedoOccurrence(TT_MonDocument))
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = DocRedoOccurrence(<Document>)`
---

**`<Résultat> : (Entier)`**

Nombre d'actions annulées qui peuvent être refaites.

**`<Document> : (Nom de champ ou variable de type Document)`**

Document à manipuler. Ce document peut correspondre à : 

- une variable de type [Document](../WDLang1/1000022461.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) un champ Traitement de texte.




## Fonctions Tableur
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions permettant de manipuler le champ Tableur sont les suivantes : 



|   |   |
| --- | --- |
| [TableurAjouteDonnée](../WDLang1/1000021365.md) | Ajoute une donnée dans un champ Tableur présent dans une fenêtre. |
| [TableurAjouteFeuille](../WDLang1/1000021642.md) | Ajoute une feuille de calcul dans un champ Tableur présent dans une fenêtre. |
| [TableurAjouteFormule](../WDLang1/1000021373.md) | Ajoute une formule de calcul dans un champ Tableur présent dans une fenêtre. |
| [TableurAjouteImage](../WDLang1/1000023335.md) | Ajoute une image dans la feuille en cours d'un champ Tableur. |
| [TableurAjouteNom](../WDLang1/1000022523.md) | Spécifie un nom pour identifier une cellule d'un champ Tableur. |
| [TableurAutoriseProcédure](../WDLang1/1000021473.md) | Autorise l'utilisation d'une fonction/procédure WLangage dans une formule d'un champ Tableur. |
| [TableurCadreSélection](../WDLang1/1000021480.md) | Modifie le cadre des cellules sélectionnées dans un champ Tableur présent dans une fenêtre. |
| [TableurChampFormule](../WDLang1/1000021647.md) | Permet d'utiliser un champ pour réaliser une barre de formule personnalisée pour le champ Tableur. |
| [TableurCharge](../WDLang1/1000021340.md) | Charge un fichier au format XLSX dans un champ Tableur présent dans une fenêtre. |
| [TableurCherche](../WDLang1/1000021610.md) | Recherche une valeur dans les cellules d'un champ Tableur présent dans une fenêtre. |
| [TableurDébutModification](../WDLang1/1000023460.md) | Signale le début du regroupement d'un ensemble d'opérations sur un champ Tableur en un seul événement (gestion du "undo/redo"). |
| [TableurDéfusionneCellule](../WDLang1/1410089203.md) | Supprime la fusion de cellule pour la cellule spécifiée ou pour la cellule sélectionnée dans un champ Tableur. |
| [TableurDéplaceFeuille](../WDLang1/1410087257.md) | Déplace une feuille de calcul dans un champ Tableur présent dans une fenêtre. |
| [TableurFerme](../WDLang1/1000021482.md) | Ferme le document XLSX ouvert précédemment par la fonction [TableurCharge](../WDLang1/1000021340.md). |
| [TableurFige](../WDLang1/1000022519.md) | Permet de figer l'affichage d'une ligne ou d'une colonne dans un champ Tableur. |
| [TableurFinModification](../WDLang1/1000023461.md) | Signale la fin du regroupement d'un ensemble d'opérations sur un champ Tableur en un seul événement (gestion du "undo/redo"). |
| [TableurFusionneCellule](../WDLang1/1000026181.md) | Fusionne les cellules spécifiées ou les cellules sélectionnées dans un champ Tableur. |
| [TableurInsèreColonne](../WDLang1/1000021518.md) | Insère une colonne vide dans un champ Tableur présent dans une fenêtre. |
| [TableurInsèreFeuille](../WDLang1/1000021648.md) | Insère une feuille de calcul dans un champ Tableur présent dans une fenêtre. |
| [TableurInsèreLigne](../WDLang1/1000021456.md) | Insère une ligne vide dans un champ Tableur présent dans une fenêtre. |
| [TableurListeCommentaire](../WDLang1/1000024881.md) | Liste tous les commentaires de la feuille courante d'un champ Tableur. |
| [TableurListeFeuille](../WDLang1/1000023450.md) | Renvoie la liste des feuilles de calcul présentes dans un champ Tableur. |
| [TableurListeImage](../WDLang1/1000023547.md) | Liste les images présentes dans la feuille courante dans un champ Tableur. |
| [TableurListeNom](../WDLang1/1000023448.md) | Renvoie la liste des noms qui identifient une cellule d'un champ Tableur. |
| [TableurModifieNom](../WDLang1/1000023464.md) | Modifie le nom utilisé pour identifier une cellule d'un champ Tableur. |
| [TableurPosition](../WDLang1/1000021609.md) | Affiche la cellule spécifiée dans un champ Tableur présent dans une fenêtre: <br><br>- soit la zone visible du champ Tableur est déplacée pour afficher la cellule. <br><br>- soit la cellule est affichée en haut à gauche de la zone visible du champ Tableur.<br><br><br> |
| [TableurRecalculeTout](../WDLang1/1000021457.md) | Recalcule toutes les formules d'un champ Tableur présent dans une fenêtre (y compris les formules utilisant des fonctions/procédures WLangage). |
| [TableurRécupèreFormule](../WDLang1/1000021507.md) | Récupère la formule de calcul d'une cellule présente dans un champ Tableur d'une fenêtre. |
| [TableurRécupèreFusion](../WDLang1/1000021740.md) | Permet de savoir si une cellule est fusionnée avec d'autres cellules et d'obtenir le nom des cellules fusionnées. |
| [TableurRécupèreInfoTri](../WDLang1/1000023462.md) | Renvoie différentes informations sur le tri effectué dans un champ Tableur. |
| [TableurRécupèreSélection](../WDLang1/1000021499.md) | Récupère la liste des cellules sélectionnées dans un champ Tableur présent dans une fenêtre. |
| [TableurRemplace](../WDLang1/1000023436.md) | Recherche et remplace une valeur dans un champ Tableur. |
| [TableurRestaurePositionEtSélection](../WDLang1/1000023334.md) | Restaure : <br><br>- la position de l'ascenseur d'un champ Tableur, <br><br>- la feuille en cours, <br><br>- le ou les éléments sélectionnés dans la feuille en cours du champ Tableur. <br><br><br>Ces éléments ont été précédemment sauvegardés par la fonction [TableurSauvePositionEtSélection](../WDLang1/1000023333.md). |
| [TableurSaisieFonction](../WDLang1/1000023412.md) | Insère un nom de fonction à la position du curseur dans la cellule en cours de saisie d'un champ Tableur présent dans une fenêtre. |
| [TableurSauve](../WDLang1/1000021341.md) | Sauve le contenu d'un champ Tableur dans un fichier au format XLSX. |
| [TableurSauvePositionEtSélection](../WDLang1/1000023333.md) | Sauve : <br><br>- la position de l'ascenseur d'un champ Tableur pour la feuille en cours, <br><br>- la feuille en cours dans le champ Tableur,<br><br>- le ou les éléments sélectionnés dans le champ Tableur pour la feuille en cours.<br><br><br> |
| [TableurSelectCommentaire](../WDLang1/1000024883.md) | Permet de : <br><br>- savoir si un commentaire est sélectionné dans un champ Tableur, <br><br>- sélectionner ou désélectionner un commentaire dans un champ Tableur.<br><br><br> |
| [TableurSelectFeuille](../WDLang1/1000021649.md) | Sélectionne une feuille de calcul (identifiée par son nom) dans un champ Tableur. |
| [TableurSelectMoins](../WDLang1/1000021494.md) | Désélectionne une ou plusieurs cellules dans un champ Tableur présent dans une fenêtre. |
| [TableurSelectPlus](../WDLang1/1000021493.md) | Sélectionne une ou plusieurs cellules dans un champ Tableur présent dans une fenêtre. |
| [TableurStyleSélection](../WDLang1/1000021338.md) | Renvoie ou modifie le style des cellules sélectionnées dans un champ Tableur présent dans une fenêtre. |
| [TableurSupprimeColonne](../WDLang1/1000022406.md) | Supprime une colonne dans un champ Tableur. |
| [TableurSupprimeFeuille](../WDLang1/1000021643.md) | Supprime une feuille de calcul dans un champ Tableur présent dans une fenêtre. |
| [TableurSupprimeImage](../WDLang1/1000023546.md) | Supprime une image dans la feuille courante d'un champ Tableur. |
| [TableurSupprimeLigne](../WDLang1/1000022405.md) | Supprime une ligne dans un champ Tableur. |
| [TableurSupprimeNom](../WDLang1/1000023447.md) | Supprime un des noms permettant d'identifier une cellule d'un champ Tableur. |
| [TableurSupprimeTout](../WDLang1/1000021495.md) | Supprime toutes les données d'un champ Tableur présent dans une fenêtre (cellules, formules, taille et mises en forme d'entêtes). |
| [TableurTrieSélection](../WDLang1/1000023463.md) | Trie les valeurs sélectionnées dans un champ Tableur. |
| [TableurTypeEtMasqueSélection](../WDLang1/1000021349.md) | Renvoie ou modifie le type et le masque des cellules sélectionnées dans un champ Tableur. |



    
D'autres fonctions permettent également de manipuler les coordonnées d'une cellule : 



|   |   |
| --- | --- |
| [xlsCoordonnéesVersNom](../WDLang5/1000021500.md) | Récupère le nom d'une cellule d'une feuille XLS ou d'un champ Tableur en fonction de ses coordonnées. |
| [xlsNomVersCoordonnées](../WDLang5/1000021501.md) | Renvoie les coordonnées d'une cellule d'un fichier XLS ou d'un champ Tableur en fonction de son nom. |


## StatEcartType (Fonction)

***En anglais : StatStandardDeviation***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Calcule l'écart type d'une série de valeurs.

**Rappel** : L'écart type est une mesure de la dispersion des valeurs par rapport à la valeur moyenne.

**Remarque** : Ce calcul s'effectue sur un échantillon de valeurs. Pour calculer l'écart type sur une série complète de valeurs, utilisez la fonction [StatEcartTypeP](../WDLang1/3050026.md).




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple
<a class="notetitle" target="_blank" href="$DOC$=1000003050025&name=statecarttype_fonction&product=WD">Voir des exemples supplémentaires</a>

```wl
// Calcul de l'écart type des éléments de la ligne 1 d'une matrice
ResEcartType = StatEcartType("MaMatrice", 1, Faux)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = StatEcartType(<Matrice> [, <Numéro de la ligne/colonne> [, <Ligne/Colonne>]])`
---

**`<Résultat> : (Réel)`**



- Écart type des valeurs. 

- 0 si l'écart type est nul.




**`<Matrice> : (Chaîne de caractères)`**

Nom de la matrice contenant les données, défini avec la fonction [MatCrée](../WDLang1/3050058.md).

**`<Numéro de la ligne/colonne> : (Entier optionnel)`**

Numéro de la ligne ou de la colonne contenant les données (1 par défaut).

**`<Ligne/Colonne> : (Booléen optionnel)`**



- <u><u><u><u>Vrai</u></u></u></u> (par défaut) si la série de valeurs correspond à une colonne de la matrice, 

- <u><u><u><u>Faux</u></u></u></u> si la série de valeurs correspond à une ligne de la matrice.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Nombre d'éléments d'une ligne ou d'une colonne de la matrice
<a name="nombre_elements_une_ligne_une_colonne_matrice_ELTPARAGRAPHE000100"></a>Le nombre d'éléments présents dans une ligne ou dans une colonne de la matrice est pris comme référence. Ce nombre d'éléments correspond :

- au nombre total de lignes de la matrice, connu avec la fonction [MatNbLigne](../WDLang1/3050060.md) (si le calcul s'effectue sur une colonne),

- au nombre total de colonnes de la matrice, connu avec la fonction [MatNbColonne](../WDLang1/3050073.md) (si le calcul s'effectue sur une ligne).




Si les séries comportent moins d'éléments que la matrice, les valeurs manquantes sont automatiquement complétées par des 0. Ces valeurs sont prises en compte lors des calculs.


<a name="NOTE0_2"></a>




### Gestion des erreurs
<a name="gestion_des_erreurs_ELTPARAGRAPHE000120"></a>**Attention** : La fonction **StatEcartType** ne retourne pas de code d'erreur. Pour savoir si le calcul de l'écart type a généré des erreurs, utilisez la fonction [StatErreur](../WDLang1/3050019.md). Le libellé de l'erreur peut être connu grâce à la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.

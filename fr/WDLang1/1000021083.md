<a name="NOTE1_1"></a>
Cette page présente les constantes utilisées dans les différentes fonctions de gestion des diagrammes de Gantt (syntaxe non préfixée). 

| Constante | Détail | Utilisée par la fonction |
| --- | --- | --- |
| **ganttLienDestination** | Liste tous les liens qui pointent sur la tâche spécifiée. | [GanttListeLien](../WDLang1/1000021079.md) |
| **ganttLienTout** | Liste tous les liens. | [GanttListeLien](../WDLang1/1000021079.md) |
| **ganttModeEnJour** | Affiche le diagramme de Gantt en fonction de la largeur de la colonne. Dans ce cas, il est nécessaire de spécifier le nombre de jours à afficher. | [GanttChangeMode](../WDLang1/1000021030.md) |
| **ganttModeSelonTâche** | Affiche le diagramme de Gantt de la première à la dernière tâche. | [GanttChangeMode](../WDLang1/1000021030.md) |
| **ganttTâcheSélectionnée** | Tâche actuellement sélectionnée dans le champ Diagramme de Gantt. | [GanttListeTâche](../WDLang1/1000021073.md) |
| **ganttTâcheSurvolée** | Tâche survolée par la souris dans le champ Diagramme de Gantt. | [GanttListeTâche](../WDLang1/1000021073.md) |
| **tlgDébutADébut** | Lien de type "Début à Début" : La tâche destination ne peut pas commencer avant que la tâche source ne soit commencée. | [GanttAjouteLien](../WDLang1/1000021075.md), <br>variable de type [LienGantt](../WDLang1/1000021076.md) |
| **tlgDébutAFin** | Lien de type "Début à Fin" : La tâche destination ne peut pas se terminer avant que la tâche source ne soit commencée (cas rare). | [GanttAjouteLien](../WDLang1/1000021075.md), <br>variable de type [LienGantt](../WDLang1/1000021076.md) |
| **tlgFinADébut** | Lien de type "Fin à Début" : La tâche destination ne peut pas commencer avant que la tâche source ne soit terminée. | [GanttAjouteLien](../WDLang1/1000021075.md), <br>variable de type [LienGantt](../WDLang1/1000021076.md) |
| **tlgFinAFin** | Lien de type "Fin à Fin" : La tâche destination ne peut pas se terminer avant que la tâche source ne soit terminée. | [GanttAjouteLien](../WDLang1/1000021075.md), <br>variable de type [LienGantt](../WDLang1/1000021076.md) |


## FenEdRestaure (Fonction)

***En anglais : WinEdRestore***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Annule toutes les modifications effectuées par l'utilisateur final (mode "Modification de l'UI par l'utilisateur final" activé). La fenêtre est restaurée dans son état initial. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Clic sur le bouton [Annuler toutes les modifications]
FenEdRestaure(FEN_Formulaire)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`FenEdRestaure(<Fenêtre>)`
---

**`<Fenêtre> : (Nom de fenêtre)`**

Nom de la fenêtre ou de la fenêtre interne à manipuler. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Cette fonction est sans effet : 

- Si la fenêtre spécifiée n'est pas éditable. 

- Si la fenêtre spécifiée n'a pas été modifiée par l'utilisateur final.

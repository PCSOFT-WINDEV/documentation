


## ParamètreOrthographe (Fonction)

***En anglais : ConfigureSpellCheck***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Paramètre le correcteur orthographique pour les champs de saisie et les colonnes de champ Table de toute l'application. Il est possible de spécifier dans quelle langue la correction orthographique doit être effectuée.  
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Active la vérification d'orthographe pour toute l'application
ParamètreOrthographe(orthographeActif, Vrai)
// Sélectionne le dictionnaire anglais
ParamètreOrthographe(orthographeNation, nationAnglais)
```
<a name="Exemple2"></a>

```wl
// Savoir si la vérification d'orthographe est activée pour toute l'application
SI ParamètreOrthographe(orthographeActif) = Vrai ALORS
	Info("Le correcteur est activé")
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Paramétrer le correcteur orthographique

`ParamètreOrthographe(<Option> , <Valeur>)`
---

**`<Option> : (Constante de type Entier)`**

Option de paramétrage du correcteur orthographique de OpenOffice : 


|   |   |
| --- | --- |
| *orthographeActif* | Si &lt;Valeur&gt; correspond à <u><u><u><u>Vrai</u></u></u></u>, le correcteur orthographique est activé pour tous les champs de saisie de l'application. <br>**Attention** : Il est nécessaire que l'option "Vérifier l'orthographe" ait été cochée sous l'éditeur.<br>Si &lt;Valeur&gt; correspond à <u><u><u><u>Faux</u></u></u></u>, le correcteur orthographique est désactivé pour tous les champs de saisie de l'application. |
| *orthographeNation* | Indique la langue du dictionnaire à utiliser. &lt;Valeur&gt; peut correspondre à une des constantes permettant d'identifier les langues (voir paragraphe [Remarques](#NOTE0_2)). Si le dictionnaire n'est pas installé, la correction automatique ne sera pas proposée.<br><br>Remarque : Cette constante est prise en compte uniquement pour une vérification orthographique avec OpenOffice. |
| *orthographeTypeCorrecteur* | Permet de sélectionner le type de correcteur utilisé. <br>Si &lt;Valeur&gt; correspond à : <br><br>- *orthographeCorrecteurIntégré* : le correcteur orthographique intégré sera utilisé pour tous les champs de saisie de l'application.<br><br>- *orthographeCorrecteurOpenOffice* : le correcteur orthographique de OpenOffice sera utilisé pour tous les champs de saisie de l'application.<br><br><br> |



**`<Valeur> : (Variant)`**

Valeur de l'option paramétrée (voir le paramètre &lt;Options&gt;).  


<a name="SYNTAXE2"></a>

### Connaître le paramétrage du correcteur orthographique

`<Résultat> = ParamètreOrthographe(<Option>)`
---

**`<Résultat> : (Variant)`**

Valeur de l'option demandée.

**`<Option> : (Entier)`**

Option de paramétrage du correcteur orthographique de OpenOffice :


|   |   |
| --- | --- |
| *orthographeActif* | Si &lt;Résultat&gt; correspond à <u><u><u><u>Vrai</u></u></u></u>, le correcteur orthographique de OpenOffice est activé pour tous les champs de saisie de l'application<br>Si &lt;Résultat&gt; correspond à <u><u><u><u>Faux</u></u></u></u>, le correcteur orthographique de OpenOffice est désactivé pour tous les champs de saisie de l'application. |
| *orthographeNation* | Indique la langue du dictionnaire à utiliser. &lt;Résultat&gt; correspond à un identifiant de langue (voir paragraphe Notes). |
| *orthographeTypeCorrecteur* | Si &lt;Résultat&gt; correspond à : <br><br>- *orthographeCorrecteurIntégré* : le correcteur orthographique intégré est utilisé pour tous les champs de saisie de l'application.<br><br>- *orthographeCorrecteurOpenOffice* : le correcteur orthographique de OpenOffice est utilisé pour tous les champs de saisie de l'application.<br><br><br><br><br> |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Notes
<a name="notes_ELTPARAGRAPHE000112"></a>

- Il est également possible d'activer le correcteur orthographique de Word avec la fonction [FAAExécute](../WDLang1/1000022099.md) et de le désactiver avec la fonction [FAADésactive](../WDLang1/1000022018.md).

- L'utilisation du correcteur orthographique de OpenOffice demande nécessite l'installation des dictionnaires de OpenOffice. Pour plus de détails, consultez [Vérifiez l'orthographe avec OpenOffice](../Editeurs/1013146.md).

- L'utilisation du correcteur orthographique intégré nécessite l'utilisation des dictionnaires Hunspell. Pour plus de détails, consultez [Vérification d'orthographe basée sur les dictionnaires Hunspell](../WDChamp/1410087030.md).


## CaméraArrêteDécodage (Fonction)

***En anglais : CameraStopDecoding***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Arrête le décodage des codes-barres via un champ Caméra.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Arrête le décodage des codes-barres dans le champ Caméra
CaméraArrêteDécodage(CAM_Caméra1)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`CaméraArrêteDécodage(<Champ Caméra>)`
---

**`<Champ Caméra> : (Nom de champ)`**

Nom du champ Caméra à manipuler.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Lorsque le champ Caméra est configuré pour autoriser le décodage de code-barres (dans la fenêtre de description), le décodage démarre automatiquement à l'ouverture de la fenêtre. Pour empêcher ce comportement, il est possible d'appeler la fonction **CaméraArrêteDécodage** dans l'événement "Initialisation" du champ Caméra.

- Lorsque le décodage des codes-barres est arrêté, la zone de visée n'est plus affichée dans le champ Caméra.

- Si le champ Caméra n'a pas été configuré pour autoriser le décodage des codes-barres (dans la fenêtre de description) ou s'il n'y a pas décodage en cours, la fonction est sans effet.

- Cette fonction est équivalente à la fonction [VidéoArrêteDécodage](../WDLang1/1000022164.md).

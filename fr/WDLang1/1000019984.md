


## &lt;Organigramme&gt;.Supprime (Fonction)

***En anglais : &lt;Organization Chart&gt;.Delete***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime un élément et l'ensemble de ses "fils" d'un champ Organigramme.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Suppression d'un élément et de l'ensemble de ses "fils" dans le champ Organigramme
NumElément est un entier 
NumElément = ORGA_MonOrganigramme.InfoXY(SourisPosX(), SourisPosY())
ORGA_MonOrganigramme.Supprime(NumElément)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Champ Organigramme>.Supprime(<Indice>)`
---

**`<Champ Organigramme> : (Nom de champ)`**

Nom du champ Organigramme à manipuler.

**`<Indice> : (Entier)`**

Indice de l'élément à supprimer. Cet indice est renvoyé par exemple par la fonction [&lt;Organigramme&gt;.InfoXY](../WDLang1/1000019977.md).

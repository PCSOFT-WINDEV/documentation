


## &lt;Tableur&gt;.Fige (Fonction)

***En anglais : &lt;Spreadsheet&gt;.Freeze***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de figer l'affichage d'une ligne ou d'une colonne dans un champ Tableur. Les lignes et les colonnes figées resteront visibles lors d'un déplacement dans la feuille de calcul. 

Il est possible de : 

- figer une ou plusieurs colonnes à gauche (ou de les libérer). 

- figer une ou plusieurs lignes en haut (ou de les libérer).



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Fige la 1ère ligne
TBLR_MonTableur.Fige(1, 0)

// Fige la 1ère colonne
TBLR_MonTableur.Fige(0, 1)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Champ Tableur>.Fige(<Ligne> , <Colonne>)`
---

**`<Champ Tableur> : (Nom de champ)`**

Nom du champ Tableur à manipuler.

**`<Ligne> : (Entier)`**



- Numéro de la dernière ligne à figer. Toutes les lignes indiquées (de la première au numéro indiqué) seront figées. 

- 0 pour libérer les lignes figées.




**`<Colonne> : (Entier)`**



- Numéro de la dernière colonne à figer. Toutes les colonnes indiquées (de la première au numéro indiqué) seront figées. 

- 0 pour libérer les colonnes figées.

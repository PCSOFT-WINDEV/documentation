
## MemTrie (Fonction)

***En anglais : MemSort***

	<IMG style="float:left;margin-right:10px;" src="https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=warning_big.png"><B>Avertissement</B><BR>

<a name="Utilisation"></a>
<a name="description"></a>
Trie les éléments présents dans une zone mémoire. Ce tri est effectué sur l'argument de tri spécifié avec les fonctions [MemAjoute](../WDLang1/3053002.md) ou [MemModifie](../WDLang1/3053007.md).




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Tri croissant des éléments dans la zone mémoire "ZoneMémoire"
MemTrie("ZoneMémoire", Vrai)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`MemTrie(<Zone mémoire> [, <Sens>])`
---

**`<Zone mémoire> : (Chaîne de caractères)`**

Nom de la zone mémoire, défini par la fonction [MemCrée](../WDLang1/3053005.md).

**`<Sens> : (Booléen optionnel)`**



- <u><u><u><u>Vrai</u></u></u></u> (valeur par défaut) si le tri doit être croissant, 

- <u><u><u><u>Faux</u></u></u></u> si le tri doit être décroissant.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Existence de la zone mémoire
<a name="existence_zone_memoire_ELTPARAGRAPHE000238"></a>L'existence de la zone mémoire n'est pas vérifiée.

Si la zone mémoire spécifiée n'existe pas, une erreur du WLangage est générée lors de l'exécution de la fonction **MemTrie**. 

Pour vérifier l'existence d'une zone mémoire, utilisez la fonction [MemExiste](../WDLang1/3053006.md).
<a name="NOTE0_2"></a>




### Tri des éléments
<a name="tri_des_elements_ELTPARAGRAPHE000099"></a>Le tri est utilisé par les fonctions de parcours [MemPremier](../WDLang1/3053014.md), [MemDernier](../WDLang1/3053004.md), [MemSuivant](../WDLang1/3053017.md) et [MemPrécédent](../WDLang1/3053011.md), ainsi que par la fonction [MemRecherche](../WDLang1/3053015.md).

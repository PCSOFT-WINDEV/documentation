


## &lt;Variable WDPic&gt;.FusionneCalque (Fonction)

***En anglais : &lt;WDPic variable&gt;.MergeLayer***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Fusionne les calques d'une image au format WDPic.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonImage est un WDPic = sNomFichier

// Création d'un calque avec un rectangle
NouveauCalque est un picCalque
NouveauCalque.Nom = "Rectangle"
NouveauCalque.Largeur = 100
NouveauCalque.Hauteur = 100
NouveauCalque.DessineRectangle(1, 1, 99, 99, RougeFoncé, BleuFoncé)
MonImage.FusionneCalque(1, 2)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Fusionner tous les calques d'une image au format WDPic

`<Résultat> = <Image WDPic>.FusionneCalque()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la fusion a été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. En cas d'erreur, la variable [ErreurDétectée](../WDLang1/3087001.md) est à <u><u><u><u>Vrai</u></u></u></u> et la fonction [ErreurInfo](../WDLang1/3013008.md) permet de connaître le détail de l'erreur.




**`<Image WDPic> : (Variable de type WDPic)`**

Nom de la variable de type [WDPic](../WDLang1/1000023503.md) contenant les calques à fusionner.


<a name="SYNTAXE2"></a>

### Fusionner deux calques d'une image au format WDPic

`<Résultat> = <Image WDPic>.FusionneCalque(<Indice 1> , <Indice 2>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la fusion a été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. En cas d'erreur, la variable [ErreurDétectée](../WDLang1/3087001.md) est à <u><u><u><u>Vrai</u></u></u></u> et la fonction [ErreurInfo](../WDLang1/3013008.md) permet de connaître le détail de l'erreur.




**`<Image WDPic> : (Variable de type WDPic)`**

Nom de la variable de type [WDPic](../WDLang1/1000023503.md) contenant les calques à fusionner.

**`<Indice 1> : (Entier)`**

Indice du premier calque à fusionner.

**`<Indice 2> : (Entier)`**

Indice du second calque à fusionner.


<a name="SYNTAXE3"></a>

### Fusionner les calques indiqués pour une image au format WDPic

`<Résultat> = <Image WDPic>.FusionneCalque(<Indices des calques>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la fusion a été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. En cas d'erreur, la variable [ErreurDétectée](../WDLang1/3087001.md) est à <u><u><u><u>Vrai</u></u></u></u> et la fonction [ErreurInfo](../WDLang1/3013008.md) permet de connaître le détail de l'erreur.




**`<Image WDPic> : (Variable de type WDPic)`**

Nom de la variable de type [WDPic](../WDLang1/1000023503.md) contenant les calques à fusionner.

**`<Indices des calques> : (Tableau d'entiers)`**

Nom du tableau contenant les indices des calques à fusionner.

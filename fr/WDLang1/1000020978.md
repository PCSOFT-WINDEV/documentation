
## FenCoulissanteVisible (Fonction)

***En anglais : WinSlidingVisible***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Affiche ou masque une fenêtre coulissante (sliding menu). 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Clic sur MENU_Visible
FenCoulissanteVisible(fcGauche, Vrai) 

// Clic sur Fermer_le_menu
FenCoulissanteVisible(fcGauche, Faux)
```



```wl
MaFenêtreCoulissanteGauche.ConnecteAprès(Faux)
FenCoulissanteVisible(fcGauche, Vrai)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Rendre une fenêtre coulissante visible ou non

`FenCoulissanteVisible(<Fenêtre coulissante> , <Nouvel état>)`
---

**`<Fenêtre coulissante> : (Constante de type Entier)`**

Fenêtre coulissante à manipuler. Cette fenêtre coulissante est définie dans la fenêtre de description de la fenêtre principale : 


|   |   |
| --- | --- |
| *fcDroite* | La fenêtre coulissante de droite est manipulée. |
| *fcGauche* | La fenêtre coulissante de gauche est manipulée. |



**`<Nouvel état> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> pour rendre la fenêtre coulissante visible, 

- <u><u><u><u>Faux</u></u></u></u> pour rendre la fenêtre coulissante invisible.





<a name="SYNTAXE2"></a>

### Savoir si une fenêtre coulissante est visible ou non

`<Résultat> = FenCoulissanteVisible(<Fenêtre coulissante>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la fenêtre coulissant est visible. 

- <u><u><u><u>Faux</u></u></u></u> si la fenêtre coulissante est invisible. 




**`<Fenêtre coulissante> : (Constante de type Entier)`**

Fenêtre coulissante à manipuler. Cette fenêtre coulissante est définie dans la fenêtre de description de la fenêtre principale : 


|   |   |
| --- | --- |
| *fcDroite* | La fenêtre coulissante de droite est manipulée. |
| *fcGauche* | La fenêtre coulissante de gauche est manipulée. |







<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- La fenêtre coulissante doit avoir été préalablement définie sous l'éditeur de fenêtres (fenêtre de description de la fenêtre principale).

- Par défaut, les fenêtres coulissantes s'affichent lors d'une gesture de défilement effectuée sur le bord droit ou gauche de la fenêtre. Si ce fonctionnement est désactivé dans l'éditeur, cette fonction permet d'ouvrir les fenêtres coulissantes par un bouton par exemple.

- Un appel à la fonction [Ferme](../WDLang1/3038018.md) dans la fenêtre interne affichée dans une fenêtre coulissante ferme la fenêtre parente ET la fenêtre coulissante.

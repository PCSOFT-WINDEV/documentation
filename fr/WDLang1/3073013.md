
## SysArrête (Fonction)

***En anglais : SysShutdown***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Arrête le système ou ferme la session Windows en cours.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Arrête le système et coupe l'alimentation
SysArrête(sysEteint)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = SysArrête(<Mode d'arrêt>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si aucun problème n'a été rencontré, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. La fonction [ErreurInfo](../WDLang1/3013008.md) permet d'obtenir plus de détails.




Le résultat est immédiatement renvoyé. Le processus d'arrêt est ensuite effectué en tâche de fond.

**`<Mode d'arrêt> : (Constante)`**

Indique le mode de fermeture souhaité :


|   |   |
| --- | --- |
| *sysEteint* | Arrête le système et coupe l'alimentation. |
| *sysFermeSession* | Termine la session en cours |
| *sysRedémarre* | Arrête et redémarre le système. |
| *sysVerrouilleSession* | Verrouille la session en cours. |







<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Arrêt par logiciel impossible
<a name="arret_par_logiciel_impossible_ELTPARAGRAPHE000067"></a>Si le système ne supporte pas l'extinction par logiciel, la fonction **SysArrête** se contente d'arrêter Windows et de placer la machine dans l'état affichant le message : "Vous pouvez éteindre votre ordinateur en toute sécurité".

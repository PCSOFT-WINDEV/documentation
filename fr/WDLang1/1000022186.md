


## &lt;Tableau croisé dynamique&gt;.Filtre (Fonction)

***En anglais : &lt;Pivot table&gt;.Filter***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Active ou désactive un filtre sur un champ Tableau croisé dynamique. 

Attention : Seul un filtre défini dans l'onglet "Contenu" de la fenêtre de description du champ Tableau croisé dynamique peut être utilisé. Lorsque la fonction **&lt;Tableau croisé dynamique&gt;.Filtre** est utilisée, seul l'affichage du champ Tableau croisé dynamique est modifié, les différents totaux ne sont pas mis à jour.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// -- Sélection d'une ligne de COMBO_Filtrer_par_famille
// Filtre le champ Tableau croisé dynamique sur la famille des produits dont on doit tenir compte
TCD_Statistiques.Filtre(COL_Libellé, MoiMême)

SI TCD_Statistiques.Occurrence = 0 ALORS
	LIB_AUCUNE_DONNEE.Visible = Vrai
SINON
	LIB_AUCUNE_DONNEE.Visible = Faux
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Champ Tableau croisé dynamique>.Filtre(<Filtre d'affichage> , <Valeur>)`
---

**`<Champ Tableau croisé dynamique> : (Nom de champ)`**

Nom du champ Tableau croisé dynamique à manipuler.

**`<Filtre d'affichage> : (Chaîne de caractères)`**

Nom du filtre d'affichage à utiliser. Ce filtre a été défini dans l'onglet "Contenu" de la fenêtre de description du champ Tableau croisé dynamique.

**`<Valeur> : (Type correspondant au filtre)`**



- Valeur du filtre à mettre en place. 

- Chaîne vide ("") pour supprimer le filtre.

<a name="NOTE0_1"></a>

## Remarques


### Mise en place
<a name="mise_place_ELTPARAGRAPHE000178"></a>Pour mettre en place un filtre dans un champ Tableau croisé dynamique, il faut : 

1. Créer le champ Tableau croisé dynamique. 

2. Dans la fenêtre de description du champ Tableau croisé dynamique, dans l'onglet "Contenu", ajouter un ou plusieurs filtres d'affichage. 

3. Créer un champ permettant à l'utilisateur de sélectionner la valeur du filtre. La sélection de la valeur exécutera la fonction **&lt;Tableau croisé dynamique&gt;.Filtre**. 




**Exemple** :   

Un champ Tableau croisé dynamique est défini pour afficher des statistiques de ventes : ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=TCD_ExempleFiltre.gif)


Un filtre est défini sur les libellés de la famille de produits. Le filtre par famille permet de ne tenir compte que de la famille spécifiée dans le calcul des chiffres d'affaires. ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=TCDFiltre.gif)


La fenêtre propose à l'utilisateur une combo permettant de choisir la famille de produits à filtrer : ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=TCD_ExempleFiltreExe.gif)

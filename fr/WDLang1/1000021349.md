


## TableurTypeEtMasqueSélection (Fonction)

***En anglais : SpreadsheetTypeAndMaskSelection***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie ou modifie le type et le masque des cellules sélectionnées dans un champ Tableur présent dans une fenêtre.

**Remarque** : Cette fonction permet par exemple de réaliser une barre de mise en forme pour un champ Tableur. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// CLIC sur [NOMBRE] - Changement du masque 
TableurTypeEtMasqueSélection(TBLR_Tableur, typSaisieNum, "999 999 999,99")

// CLIC sur [DATE Longue]. 
// Changement du masque : par exemple 19 -> "Dimanche 19 Janvier 1900"
TableurTypeEtMasqueSélection(TBLR_Tableur, typSaisieDate, "JJJ J MMMM AAAA")
```

```wl
// Renvoie le masque de la sélection
sRes est une chaîne 
nType est un entier
sMasque est une chaîne
sRes = TableurTypeEtMasqueSélection(TBLR_Tableur)
nType = Val(ExtraitChaîne(sRes, 1, TAB))
sMasque = Val(ExtraitChaîne(sRes, 2, TAB))
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître le type et le masque des cellules sélectionnées

`<Résultat> = TableurTypeEtMasqueSélection(<Champ Tableur>)`
---

**`<Résultat> : (Chaîne de caractères)`**

Masque des cellules sélectionnées. Ce masque est de la forme : 

```txt
<Type> + TAB + <Masque>
```

## Remarques


### Masques disponibles
<a name="masques_disponibles_ELTPARAGRAPHE000130"></a>Les masques disponibles dépendent du type de masque. Tous les masques utilisables avec la propriété [MasqueSaisie](../Proprietes/2510033.md) sont disponibles. Pour plus de détails, consultez l'aide de la propriété [MasqueSaisie](../Proprietes/2510033.md). 

Quelques exemples : 

- sur une date : il est possible d'utiliser le masque "JJ/MM/AAAA". 

- sur un numérique : il est possible d'utiliser le masque "999 999".




Remarques : 

- Le couple (typSaisieTexte, "") signifie "aucun masque". 

- Le masque défini par la fonction **TableurTypeEtMasqueSélection** peut être modifié par la propriété [MasqueSaisie](../Proprietes/2510033.md). 
	Attention : seul le masque peut être modifié, le type de la cellule n'est pas modifié. 



<a name="NOTE0_2"></a>


### Définir un masque sur une colonne
<a name="definir_masque_sur_une_colonne_ELTPARAGRAPHE000163"></a>Pour définir un masque sur une colonne, il suffit de sélectionner la colonne et d'appliquer le masque voulu. Exemple : 

```wl
TableurSelectPlus(TBLR_SansNom1,"A")
TableurTypeEtMasqueSélection(TBLR_SansNom1, typSaisieDate, "JJ-MM-AA")
```

### Divers

- Cette fonction est utilisable uniquement sur un champ Tableur présent dans une fenêtre.




## &lt;Éditeur d'images&gt;.Ferme (Fonction)

***En anglais : &lt;Image Editor&gt;.Close***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ferme l'image se trouvant dans un champ Editeur d'images.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
EI_MonChamp.Ferme()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Champ Editeur d'images>.Ferme()`
---

**`<Champ Editeur d'images> : (Nom de champ)`**

Nom du champ Editeur d'images à manipuler.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Si aucune image n'est ouverte dans le champ Editeur d'images, la fonction n'a aucun effet.


- ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=LetypeWDPIC.gif) ***Exemples unitaires (WINDEV)*** : **Le type WDPIC** <br>Cet exemple montre l'utilisation du type WLangage WDPic.<br>Ce type WLangage permet de modifier par programmation les images sources sauvegardées avec l'éditeur d'images.<br>Il permet notamment de manipuler les calques.

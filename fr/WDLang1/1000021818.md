
## SysStockageExterneAmovible (Fonction)

***En anglais : SysRemovableExternalStorage***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de savoir si l'espace de stockage externe correspond à de la mémoire amovible (SD Card).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Copie du fichier dans le répertoire de l'application sur la SDCard du téléphone
POUR i = 1 _A_ SysNbStockageExterne()
	SI SysStockageExterneAmovible(i) _ET_ SysEtatStockageExterne(i) = sseDisponible ALORS
		fCopieFichier(sFichier, SysRepStockageExterne(i, sseAppFichier))
		SORTIR
	FIN
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = SysStockageExterneAmovible(<Indice>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'espace de stockage externe considéré correspond à de la mémoire amovible, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 




**`<Indice> : (Entier)`**

Indice de l'espace de stockage externe à utiliser (si l'appareil possède plusieurs espaces de stockage externe). 
Remarque : La fonction [SysNbStockageExterne](../WDLang1/1000021816.md) permet de connaître le nombre d'espaces de stockage externe de l'appareil.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Un même appareil peut contenir plusieurs zones de stockage externe (par exemple, une zone correspond à la partition "externe" de la mémoire permanente et une autre zone correspond à la SDCard).

- Avant d'accéder à un répertoire situé dans la zone de stockage externe de l'appareil, il est conseillé de tester sa disponibilité avec la fonction [SysEtatCarteStockage](../WDLang1/1000019554.md).

- Attention : Cette fonction est disponible uniquement à partir de Android 5.0.

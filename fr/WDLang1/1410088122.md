


## DiagrammeSauve (Fonction)

***En anglais : DiagramSave***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Sauve un diagramme dans un fichier "diag". 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
DiagrammeSauve(EDIAG_MonDiagramme, "diagramme1.wddiag")
SI ErreurDétectée ALORS
	Erreur("Fichier non sauvegardé : " + ErreurInfo())
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = DiagrammeSauve(<Diagramme> , <Fichier>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la sauvegarde a été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. La fonction [ErreurInfo](../WDLang1/3013008.md) permet d'obtenir plus de détails sur l'erreur.




**`<Diagramme> : (Nom de champ ou variable de type Diagramme)`**

Diagramme à sauvegarder. Ce paramètre correspond : 

- au nom d'un champ Editeur de diagrammes. 

- au nom d'une variable de type [Diagramme](../WDLang1/1410088055.md).




**`<Fichier> : (Chaîne de caractères)`**

Chemin complet du fichier "diag".

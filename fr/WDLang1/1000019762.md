


## OrgaSelect (Fonction)

***En anglais : OrgSelect***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie l'indice de l'élément sélectionné dans un champ Organigramme.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Récupère l'indice de l'élément sélectionné dans un champ Organigramme
NumElementSel est un entier 
NumElementSel = OrgaSelect(ORGA_MonOrganigramme)
SI NumElementSel <> -1 ALORS
	 MaFenêtre.Titre = ORGA_MonOrganigramme[NumElementSel].Titre
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = OrgaSelect(<Champ Organigramme>)`
---

**`<Résultat> : (Entier)`**



- Indice de l'élément sélectionné dans le champ Organigramme spécifié, 

- -1 si aucun élément n'est sélectionné.




**`<Champ Organigramme> : (Nom de champ)`**

Nom du champ Organigramme à manipuler.

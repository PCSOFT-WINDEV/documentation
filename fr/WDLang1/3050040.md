
## MatRéelMultiplie (Fonction)

***En anglais : MatFloatMultiply***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Multiplie chaque élément d'une matrice par une valeur. Cette multiplication est réalisée directement dans la matrice spécifiée.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple
<a class="notetitle" target="_blank" href="$DOC$=1000003050040&name=matreelmultiplie_fonction&product=WD">Voir des exemples supplémentaires</a>

```wl
// Multiplication des éléments de la matrice par la valeur 3
MatRéelMultiplie("MaMatrice", 3)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`MatRéelMultiplie(<Matrice> , <Valeur>)`
---

**`<Matrice> : (Chaîne de caractères)`**

Nom de la matrice à manipuler, défini avec la fonction [MatCrée](../WDLang1/3050058.md).

**`<Valeur> : (Réel)`**

Valeur utilisée pour la multiplication.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Diviser tous les éléments d'une matrice
<a name="diviser_tous_les_elements_une_matrice_ELTPARAGRAPHE000078"></a>Pour diviser tous les éléments d'une matrice par une valeur donnée, multipliez la matrice par : 1/&lt;Valeur&gt;.


<a name="NOTE0_2"></a>




### Existence de la matrice
<a name="existence_matrice_ELTPARAGRAPHE000087"></a>L'existence de la matrice n'est pas vérifiée. Pour vérifier l'existence d'une matrice, utilisez la fonction [MatExiste](../WDLang1/3050076.md).


<a name="NOTE0_3"></a>




### Gestion des erreurs
<a name="gestion_des_erreurs_ELTPARAGRAPHE000099"></a>**Attention** : La fonction **MatRéelMultiplie** ne retourne pas de code d'erreur. Pour savoir si des erreurs ont été générées par la multiplication, utilisez la fonction [MatErreur](../WDLang1/3050044.md). Le libellé de l'erreur peut être connu grâce à la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.

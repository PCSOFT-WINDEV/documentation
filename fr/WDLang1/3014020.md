


## dbgSortieStandard (Fonction)

***En anglais : dbgStandardOutput***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Écrit une information dans le flux de sortie standard "stdout" (également appelé "console").

**Remarques** : 

- ![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) Sous Linux, les informations écrites avec cette fonction sont directement affichées car les exécutables créés sont des exécutables "console".

- ![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png) Sous Windows, les exécutables ne sont pas des exécutables "console" : les informations écrites avec cette fonction ne sont pas directement affichées. Cependant, si la sortie standard est redirigée dans un fichier (via "&gt; FichierSortie.txt" par exemple), les informations écrites sont disponibles.




![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Cette fonction est utilisable quel que soit le système sur lequel l'application est lancée (Windows, Unix, Linux, ...).










<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
dbgSortieStandard("Opération réussie")
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = dbgSortieStandard(<Information>)`
---

**`<Résultat> : (Entier)`**

Taille des informations écrites. 
Remarque : La fonction **dbgSortieStandard** rajoute automatiquement un RC (Retour Chariot) en fin de chaîne. La taille obtenue est donc supérieure de 1 ou 2 caractères à la taille de l'information.

**`<Information> : (Chaîne de caractères)`**

Information qui sera écrite dans le flux de sortie standard.




## Fonctions de gestion des matrices
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


Les fonctions de gestion des matrices sont les suivantes :



|   |   |
| --- | --- |
| [MatAdditionne](../WDLang1/3050049.md) | Additionne deux matrices de mêmes dimensions (même nombre de lignes et même nombre de colonnes). |
| [MatCompresse](../WDLang1/3050043.md) | Compresse la place mémoire occupée par une matrice. |
| [MatCopie](../WDLang1/3050048.md) | Crée la copie d'une matrice. |
| [MatCrée](../WDLang1/3050058.md) | Crée une matrice de 0 ligne et 0 colonne. |
| [MatDéterminant](../WDLang1/3050045.md) | Calcule le déterminant d'une matrice carrée (nombre de lignes égal au nombre de colonnes). |
| [MatEcrit](../WDLang1/3050077.md) | Écrit un élément dans une matrice. |
| [MatErreur](../WDLang1/3050044.md) | Identifie le type de la dernière erreur due à une fonction Matxxx ([fonctions de gestion des matrices](../WDLang1/3050051.md)). |
| [MatExiste](../WDLang1/3050076.md) | Teste l'existence d'une matrice en mémoire. |
| [MatInverse](../WDLang1/3050075.md) | Inverse une matrice carrée. |
| [MatLit](../WDLang1/3050074.md) | Lit la valeur d'un élément d'une matrice. |
| [MatLitColonne](../WDLang1/3050070.md) | Lit la valeur de tous les éléments d'une colonne d'une matrice. |
| [MatLitLigne](../WDLang1/3050032.md) | Lit la valeur de tous les éléments d'une ligne d'une matrice. |
| [MatMultiplie](../WDLang1/3050072.md) | Multiplie deux matrices. |
| [MatNbColonne](../WDLang1/3050073.md) | Renvoie le nombre de colonnes d'une matrice. |
| [MatNbLigne](../WDLang1/3050060.md) | Renvoie le nombre de lignes d'une matrice. |
| [MatRéelAdditionne](../WDLang1/3050059.md) | Additionne une valeur à chaque élément d'une matrice. |
| [MatRéelMultiplie](../WDLang1/3050040.md) | Multiplie chaque élément d'une matrice par une valeur. |
| [MatRemplit](../WDLang1/3050057.md) | Initialise tous les éléments d'une matrice d'une taille donnée. |
| [MatSupprime](../WDLang1/3050031.md) | Supprime une matrice existante. |
| [MatTranspose](../WDLang1/3050001.md) | Calcule la transposée d'une matrice. |

### Rappel : Mode AWP
<a name="rappel_mode_awp_ELTPARAGRAPHE000045"></a>Dès le début de ce cours, nous avons présenté les différences entre un site WEBDEV en mode Session et un site WEBDEV en mode AWP.

Seul le mode AWP permet d'avoir des URL de page indépendantes, fixes et directement adressables.

Un site Internet, pour être référencé, doit obligatoirement être développé en mode AWP.
<a name="NOTE2_3"></a>


### Les méthodes à votre disposition
<a name="les_methodes_votre_disposition_ELTPARAGRAPHE000056"></a>Pour vous aider à optimiser le référencement de vos pages AWP, WEBDEV propose plusieurs outils :

- Un assistant de référencement.

- Les propriétés des pages.

- Le plan de navigation.


Nous allons détailler ces différentes méthodes.

<a name="NOTE3_1"></a>


## Assistant de référencement
<a name="assistant_referencement_ELTTEXTE000346"></a>
WEBDEV est livré avec un assistant d'aide au référencement.



- Pour lancer cet assistant, dans le ruban, sous le volet "Projet", dans le groupe "Web", déroulez "Référencement" et sélectionnez "Optimiser le référencement". 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=P5_R%E9f%E9rencement%20-%20HC%20N%B0001.jpg&type=thumb)





L'assistant propose des conseils pour optimiser le référencement du site. Les conseils les plus importants sont les suivants :

- **Titre des pages** : Le titre des pages doit refléter le contenu et contenir les mots-clés importants. Le titre d'une page se définit dans la fenêtre de description de la page, pour chaque langue.

- **Mots-clés des pages**: Il est conseillé d'associer des mots-clés à une page. Pour définir ces mots-clés :

	- Affichez la fenêtre de description de la page "PAGE_Liste_des_nouveautes" : affichez le menu contextuel et sélectionnez l'option "Description".

	- Affichez l'onglet "Détail".

	- Dans la partie "Référencement", cliquez sur "Editer les expressions et les mots-clés". 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=P5_R%E9f%E9rencement%20-%20HC%20N%B0002.jpg&type=thumb)
La fenêtre qui s'affiche permet de saisir les mots-clé voulus. Fermez les différentes fenêtres de description. 




- **Texte alternatif** : Les textes alternatifs sont les textes qui sont substitués aux images lorsque celles-ci ne sont pas affichées (que ce soit parce que l'internaute a explicitement demandé à ne pas afficher les images, parce qu'elles ne sont pas encore chargées ou encore parce que l'internaute utilise un navigateur en mode texte ou un lecteur d'écran pour malvoyant).
	Les textes alternatifs décrivent l'image et sont indexés par les moteurs de recherche. A ce titre, il convient de les choisir avec attention.
	Par exemple, le texte alternatif d'une image se définit dans la fenêtre de description du champ Image, dans l'onglet "Aide". 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=P5_R%E9f%E9rencement%20-%20HC%20N%B0005%20bak.jpg&type=thumb)





- Fermez si nécessaire les différentes fenêtres de description ouvertes ainsi que l'assistant de référencement.

## Propriétés des pages
<a name="proprietes_des_pages_ELTTEXTE000370"></a>
Comme nous venons de le voir, l'assistant de référencement indique plusieurs modifications pouvant être effectuées au niveau de la fenêtre de description des pages :

- Titre de la page,

- Mots-clés des pages, ...




D'autres propriétés influençant le référencement peuvent être définies dans la fenêtre de description de la page :

- Fréquence de mise à jour,

- Importance dans le site.



<a name="NOTE4_2"></a>


### Fréquence de mise à jour
<a name="frequence_mise_jour_ELTPARAGRAPHE000131"></a>La fréquence de mise à jour indique au moteur de recherche à quelle fréquence la page doit être réindexée. 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=P5_R%E9f%E9rencement%20-%20HC%20N%B0002%201.jpg&type=thumb)


Dans la mesure du possible, il faut indiquer une fréquence proche de la réalité :

- Si la page change plus souvent que ce qui est annoncé, certaines versions pourraient ne pas être indexées.

- Si la page change moins souvent, le passage du moteur de recherche va induire une charge inutile sur le serveur et l'indexation pourrait être pénalisée.



<a name="NOTE4_3"></a>


### Importance dans le site
<a name="importance_dans_site_ELTPARAGRAPHE000144"></a>Ce critère permet au moteur de recherche de savoir quelle page proposer en priorité à l'internaute quand plusieurs pages du site correspondent à une recherche.

<a name="NOTE5"></a>
<a name="NOTE5_1"></a>


## Plan de navigation
<a name="plan_navigation_ELTTEXTE000406"></a>
Le plan de navigation définit l'organisation de votre site. En référencement, il a deux utilisations :

- permettre d'afficher le champ "Plan du site" pour guider les internautes à l'intérieur du site.

- générer le fichier sitemap.xml.




- Pour définir le plan de navigation : 

	1. Dans le ruban, sous le volet "Projet", dans le groupe "Web", cliquez sur "Plan de navigation".

	2. Une fenêtre s'ouvre pour vous permettre de construire l'arborescence de votre site. Un mode de construction automatique peut vous aider à faire une ébauche d'arborescence en détectant automatiquement les liens entre les pages. Vous pouvez personnaliser le plan de navigation par la suite.







- Le fichier sitemap est construit automatiquement par WEBDEV à partir du plan de navigation. Il est généré dans le répertoire _WEB de votre site et se nomme _sitemap.xml.




Le fichier sitemap.xml doit être fourni aux outils pour webmasters des moteurs de recherche afin de leur permettre d'indexer le contenu de votre site.

<a name="NOTE6_1"></a>


## Nommage physique des pages
<a name="nommage_physique_des_pages_ELTTEXTE000430"></a>
Pour améliorer encore le référencement, il est possible de fixer le nom physique des pages AWP pour qu'il soit différent dans chaque langue (et différent du nom logique également).

Par exemple, une page affichant un tableau de bord des ventes dont le nom logique (le nom utilisé en programmation) serait PAGE_TDBVentes peut être générée en français dans le fichier ‘tableau-de-bord-ventes.awp' et en anglais dans le fichier ‘sales-dashboard.awp'.

- Pour configurer le nom de génération d'une page :

	1. Affichez la fenêtre de description de la page "PAGE_Liste_des_nouveautes".

	2. Dans l'onglet "Général", cliquez sur le bouton "Langue" (![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=P5_MultiLangue%20-%20HC%20N%B0006%201.jpg)
) à droite du champ "Fichier généré".

	3. Une fenêtre permettant de configurer le nom des pages générées s'affiche : 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=P5_R%E9f%E9rencement%20-%20HC%20N%B0004.jpg&type=thumb)


	4. Vous pouvez changer le nom de la page et valider.

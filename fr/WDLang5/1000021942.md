


## &lt;Variable fbSession&gt;.Etat (Fonction)

***En anglais : &lt;fbSession variable&gt;.Status***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Vérifie l'état d'une session Facebook. 




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MaSession est une fbSession

SI MaSession.Etat() <> fbSessionOuverte ALORS
	Info("Pas de session ouverte")
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Session>.Etat()`
---

**`<Résultat> : (Constante de type Entier)`**

Etat de la session : 



|   |   |
| --- | --- |
| *fbSessionFermée* | Session Facebook fermée. |
| *fbSessionOuverte* | Session Facebook ouverte. |
| *fbSessionPermissionIncorrecte* | Session Facebook ouverte avec des droits différents de ceux de la session passée en paramètre. |



**`<Session> : (Variable de type fbSession)`**

Nom de la variable de type [fbSession](../WDLang5/1000021851.md) correspondant à la connexion à manipuler.

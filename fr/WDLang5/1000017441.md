
## gglLieu (Type de variable)

***En anglais : gglLocation ***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **gglLieu** permet de décrire et de modifier le lieu d'un événement d'un agenda Google.

Ce type de variable est utilisé par les variables de type [gglEvénement](../WDLang5/1000017440.md).

**Remarques** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connexion à Google
Cnx est un gglConnexion
...
// Récupération du tableau des agendas disponibles
tabAgendas est un tableau de gglAgenda
tabAgendas = GglRécupèreAgenda(Cnx)
// Récupération des événements du premier agenda
GglRemplitAgenda(Cnx, tabAgendas[1])
// Affichage des lieux du premier événement
MonAgenda est un gglAgenda = tabAgendas[1]
Lieu est un gglLieu
POUR TOUT Lieu DE MonAgenda
	Trace(Lieu.Valeur)
FIN
```


## iImprimeZoneMarkdown (Fonction)

***En anglais : iPrintAreaMarkdown***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Imprime un texte au format Markdown dans une zone rectangulaire. L'impression n'est pas lancée (seule la fonction [iFinImprime](../WDLang5/3046053.md) permet de lancer l'impression).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
iDestination(iVisualisateur)

// Récupération de la hauteur de la zone
dHauteur est un réel
dHauteur = iHauteurZone(sTexteMarkdown, 100, iMarkdown)

// Trace un cadre
iCadre(0,0, 200, dHauteur*2)

// Impression du texte Markdown dans le cadre
iImprimeZoneMarkdown(sTexteMarkdown, 0, 0, 200, dHauteur*2)

iFinImprime()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = iImprimeZoneMarkdown(<Texte Markdown> , <X1> , <Y1> , <X2> , <Y2> [, <Début> [, <Fin>]])`
---

**`<Résultat> : (Entier)`**

Indice du dernier élément Markdown imprimé dans la zone rectangulaire (comptage spécifique de WINDEV/WEBDEV). 

**`<Texte Markdown> : (Chaîne de caractères)`**

Chaîne de caractères au format Markdown à imprimer dans une zone spécifique.

**`<X1> : (Réel)`**

Coordonnée horizontale (colonne ou abscisse) du coin haut gauche de la zone d'impression (en millimètres). 
Valeur comprise entre 0 et la largeur de la page (donnée par [iLargeurPage](../WDLang5/3046036.md)).

**`<Y1> : (Réel)`**

Coordonnée verticale (ligne ou ordonnée) du coin haut gauche de la zone d'impression (en millimètres). 
Valeur comprise entre 0 et la hauteur de la page (donnée par [iHauteurPage](../WDLang5/3046052.md)).

**`<X2> : (Réel)`**

Coordonnée horizontale (colonne ou abscisse) du coin bas droit de la zone d'impression (en millimètres). 
Valeur comprise entre 0 et la largeur de la page (donnée par [iLargeurPage](../WDLang5/3046036.md)).

**`<Y2> : (Réel)`**

Coordonnée verticale (ligne ou ordonnée) du coin bas droit de la zone d'impression (en millimètres). 
Valeur comprise entre 0 et la hauteur de la page (donnée par [iHauteurPage](../WDLang5/3046052.md)).

**`<Début> : (Entier optionnel)`**

Indice du premier élément Markdown du &lt;Texte Markdown&gt; à imprimer. Si ce paramètre n'est pas indiqué, le &lt;Texte Markdown&gt; est imprimé depuis le premier élément Markdown.

**`<Fin> : (Entier optionnel)`**

Indice du dernier élément Markdown du &lt;Texte Markdown&gt; à imprimer. Si ce paramètre n'est pas indiqué, le &lt;Texte Markdown&gt; est imprimé jusqu'au dernier élément Markdown.

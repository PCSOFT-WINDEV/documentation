
## iChangeSousEtatSource (Fonction)

***En anglais : iChangeSubReportSource***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie par programmation l'état associé à un sous-état dans un état composite. 

**Remarque** : Cette fonction peut être utilisée uniquement durant l'exécution de l'impression d'un état composite.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Code de déclaration des globales d'un état composite
iChangeSousEtatSource(SETAT_SousEtat1, ETAT_Fiche)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`iChangeSousEtatSource(<Sous-état> , <Etat cible>)`
---

**`<Sous-état> : (Chaîne de caractères)`**

Nom du sous-état à modifier. 

**`<Etat cible> : (Chaîne de caractères)`**

Nom de l'état à associer au sous-état. Cet état sera effectivement imprimé lors de l'impression du sous-état. Ce nom doit être valide. Si ce paramètre correspond à une chaîne vide (""), le sous-état ne sera pas imprimé.


## Type XLSCellule : Propriété MasqueWD
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000161"></a>
La propriété MasqueWD du type xlsDocument permet de définir le masque d'affichage au format WINDEV ou WEBDEV à utiliser dans la cellule Excel (par exemple "JJ/MM/AAAA").

Le masque spécifié est automatiquement converti au format Excel. Les centièmes de seconde présents dans les masques ne sont pas gérés.

Cette page d'aide présente les différents masques pouvant être utilisés.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>

- "999 999"

- "999 999 999"

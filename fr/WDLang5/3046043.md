
## iStatusImpressionEtat (Fonction)

***En anglais : iReportPrintingStatus***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie l'état de l'impression en cours.

Cette fonction doit être utilisée dans le code de fermeture de l'état.



**Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Imprimer sous Linux](../Editeurs/9000078.md). 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Fin de l'impression
SELON iStatusImpressionEtat() 
	CAS -2 : Trace("Impression abandonnée")
	CAS 0 : Trace("Impression en cours")
	CAS 1 : Trace("Etat vide : aucune données à imprimer")
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = iStatusImpressionEtat()`
---

**`<Résultat> : (Entier)`**

Etat de l'impression en cours :

- 1 : Il n'y a pas de données à imprimer

- 0 : Impression en cours

- -1 : Aucune impression en cours

- -2 : Abandon




## HTMLDésencadreTousLesFils (Fonction)

***En anglais : HTMLUnwrapAllChildren***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime le noeud spécifié dans un document HTML sans supprimer ses fils, qui prennent sa place. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
hDoc est un htmlDocument = "
<html>
	<head/>
	<body>
		<h1></h1><div id="div">
			<p>Texte1</p>
			<p>Texte2</p>
		</div><h3></h3>
	</body>
</html>
"
hNoeud est un htmlNoeud <- HTMLChercheElémentParID(hDoc, "div")
hNoeudInsertion est un htmlNoeud

hNoeudInsertion..NomBalise = "h2"
HTMLEncadre(hNoeud, hNoeudInsertion)
// body : <h1/><h2><div id="div"><p>Texte1</p><p>Texte2</p></div></h2><h3/>
Trace(hDoc)
hNoeudInsertion..NomBalise = "span"
HTMLEncadreTousLesFils(hNoeud, hNoeudInsertion)
// body : <h1/><h2><div id="div"><span><p>Texte1</p><p>Texte2</p></span></div></h2><h3/>
Trace(hDoc)
HTMLDésencadreTousLesFils(hNoeud)
// body : <h1/><h2><span><p>Texte1</p><p>Texte2</p></span></h2><h3/>
Trace(hDoc)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`HTMLDésencadreTousLesFils(<Noeud>)`
---

**`<Noeud> : (Variable de type htmlNoeud)`**

Nom de la variable de type [htmlNoeud](../WDLang5/1000026081.md) correspondant au noeud à supprimer.


## iHautBasPageImbriqué (Fonction)

***En anglais : iNestedHeaderFooter***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Imprime ou non les hauts et bas de page de l'état imbriqué.

Cette fonction peut être utilisée :

- soit dans le code d'ouverture de l'état principal : elle sera prise en compte pour tous les états imbriqués qui seront imprimés depuis l'état principal.

- soit directement avant l'impression de l'état imbriqué : elle sera prise en compte uniquement pour l'état imbriqué suivant.




![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Imprimer sous Linux](../Editeurs/9000078.md). 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Impression de l'état "ETAT_Imb2" en conservant son haut et bas de page
iHautBasPageImbriqué(Vrai)
IImprimeEtat(ETAT_Imb2)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`iHautBasPageImbriqué(<Haut et bas de page Etat imbriqué>)`
---

**`<Haut et bas de page Etat imbriqué> : (Booléen)`**



- <u><u><u><u>Faux</u></u></u></u> (valeur par défaut) si l'état principal doit être imprimé en partageant son haut et bas de page avec l'état imbriqué. Le haut et bas de page de l'état imbriqué ne seront pas imprimés.

- <u><u><u><u>Vrai</u></u></u></u> si le haut de page et le bas de page de l'état imbriqué doivent être imprimés.

### Astuce
<a name="astuce_ELTPARAGRAPHE000104"></a>Lors de l'utilisation de la fonction **iHautBasPageImbriqué** avec le paramètre <u><u><u><u>Vrai</u></u></u></u>, il est conseillé d'utiliser la fonction [iTerminePage](../WDLang5/3046038.md) avant la fonction [iImprimeEtat](../WDLang5/3046032.md). Il sera ainsi possible par exemple d'imprimer successivement deux états dans la même fenêtre d'aperçu avant impression.






### Changement d'orientation
<a name="changement_orientation_ELTPARAGRAPHE000124"></a>Si un état imbriqué n'a pas la même orientation que l'état principal, cette fonction est sans effet : les blocs de haut et bas de page ne seront **pas** partagés.

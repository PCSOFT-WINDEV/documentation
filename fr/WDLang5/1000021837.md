


## FBRécupèreInformationUtilisateur (Fonction)

***En anglais : FBGetUserInformation***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Récupère les informations Facebook de l'utilisateur connecté. 






<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MaSession est une fbSession
...
sEmail est chaîne = FBRécupèreInformationUtilisateur(MaSession, fbEmail)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = FBRécupèreInformationUtilisateur(<Session> , <Information>)`
---

**`<Résultat> : (Chaîne de caractères ou Buffer)`**

Information demandée. 

**`<Session> : (Variable de type fbSession)`**

Nom de la variable de type [fbSession](../WDLang5/1000021851.md) correspondant à la connexion Facebook à manipuler. 

**`<Information> : (Constante de type chaîne de caractères)`**

Information demandée : 


|   |   |
| --- | --- |
| *fbEmail* | Email de l'utilisateur. |
| *fbIdentifiant* | Identifiant Facebook de l'utilisateur. |
| *fbImage* | Image de l'utilisateur. |
| *fbNom* | Nom de l'utilisateur. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Lors de la connexion, les droits (permissions) correspondant à l'information demandée devront avoir été accordés. Dans le cas contraire, la fonction échouera.

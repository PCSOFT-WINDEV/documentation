


## gglEvénement (Type de variable)

***En anglais : gglEvent ***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **gglEvénement** permet de décrire et de modifier un événement d'un agenda. L'agenda est décrit grâce au type [gglAgenda](../WDLang5/1000017418.md). Cet agenda est utilisé par le service Agenda Google Calendar. 

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connexion à Google
Cnx est un gglConnexion
Cnx.Email = "balthazar@gmail.com"
Cnx.MotDePasse = "azerty"
Cnx.NomApplication = "MonAppli-01"
SI GglConnecte(Cnx) = Faux ALORS
	Erreur(ErreurInfo())
FIN
...
// Récupération du tableau des agendas disponibles
tabAgendas est un tableau de gglAgenda
tabAgendas = GglListeAgenda(Cnx)
// Récupération de tous les événements du premier agenda
GglRemplitAgenda(Cnx, tabAgendas[1])
// Parcours des événements
Evt est un gglEvénement
POUR TOUT Evt DE tabAgendas[1]
	Trace(Evt.Titre)
	Trace(Evt.Couleur)
	Trace(Evt.Notification.Défaut)
	Trace(Evt.Notification.Surcharge.Occurrence)
	Trace(Evt.Visibilité)
	Trace(Evt.Disponibilté)
	Trace(Evt.PièceJointe.Occurrence)
	POUR TOUT UneNotif DE Evt.Notification.Surcharge
		Trace(UneNotif.Méthode)
		Trace(UneNotif.NbMinute)
	FIN
	POUR TOUT UnFichier DE Evt.PièceJointe
		Trace(UnFichier.IDFichier)
		Trace(UnFichier.Titre)
		Trace(UnFichier.URLFichier)
		Trace(UnFichier.URLIcone)
		Trace(UnFichier.TypeMIME)
	FIN
FIN
```

### Propriétés spécifiques à la description des variables de type gglEvénement
<a name="proprietes_specifiques_description_des_variables_type_gglevenement_ELTPARAGRAPHE000033"></a>Les propriétés suivantes peuvent être utilisées pour manipuler un événement d'un agenda grâce au type **gglEvénement** :

| Nom de la propriété | Type manipulé | Effet |
| --- | --- | --- |
| Auteur | [gglAuteur](../WDLang5/1000017746.md) | Auteur de l'agenda. Cette propriété est accessible en lecture uniquement. |
| Contenu | Chaîne de caractères Unicode | Description de l'événement. |
| Couleur | Chaîne de caractères Unicode | Identifiant de la couleur de l'événement. Les couleurs disponibles sont les suivantes :![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ggleEvenement_Couleur.gif)<br> |
| DateDébut | DateHeure | Date de début de l'événement.<br>**Remarques** :<br><br>- Si l'événement correspond à une journée entière, la date de début de l'événement est celle du jour à 0h00.<br><br>- La date est celle du fuseau horaire de la machine locale.<br><br><br> |
| DateFin | DateHeure | Date de fin de l'événement.<br>**Remarques** :<br><br>- Si l'événement correspond à une journée entière, la date de fin de l'événement est celle du jour suivant à 0h00.<br><br>- La date est celle du fuseau horaire de la machine locale.<br><br><br> |
| DateMiseAJour | DateHeure | Date de mise à jour de l'événement. Cette propriété est accessible en lecture uniquement.<br>**Remarque** : La date est celle du fuseau horaire de la machine locale. |
| Disponibilité | Constante de type chaîne | Disponibilité de l'événement. Cette propriété peut correspondre aux constantes suivantes : <br><br>- *gglOpaque* : Occupé. <br><br>- *gglTransparent* : Disponible. <br><br><br><br><br> |
| Identifiant | Chaîne de caractères Unicode | Identifiant Google de l'événement. |

| JournéeEntière | <br><br>- <u><u><u><u>Vrai</u></u></u></u> si l'événement correspond à des journées entières,<br><br>- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.<br><br><br> |

### Gestion des dates et de leur fuseau horaire
<a name="gestion_des_dates_leur_fuseau_horaire_ELTPARAGRAPHE000191"></a>Suite à l'utilisation de l'authentification "OAuth 2.0", la gestion des dates a été modifiée. Toutes les dates renvoyées par les fonctions [GglRécupèreAgenda](../WDLang5/1000017410.md), [GglListeAgenda](../WDLang5/1000017330.md), ... dans les variables de type [gglEvénement](../WDLang5/1000017440.md) utilisent le fuseau horaire de la machine locale. Elles sont donc utilisables directement dans un champ Calendrier avec le fuseau horaire en cours. 

Pour les convertir dans un fuseau horaire particulier, il est possible d'utiliser les fonctions [DateHeureLocaleVersFuseau](../WDLang1/1000020977.md) ou [DateHeureLocaleVersUTC](../WDLang1/3027036.md).

Il est possible de récupérer le fuseau horaire de l'agenda accédé via la propriété **FuseauHoraire** des variables de type [gglAgenda](../WDLang5/1000017418.md). La valeur renvoyée correspond au nom du fuseau, issu de la base de données "tz" (ou base de données Olson). Ce nom est directement utilisable avec la fonction [DateHeureLocaleVersFuseau](../WDLang1/1000020977.md).

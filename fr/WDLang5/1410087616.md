


## HTMLInsèreDernierFils (Fonction)

***En anglais : HTMLInsertLastChild***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ajoute un fils après les fils du noeud courant dans un document HTML.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
hDoc est un htmlDocument = "
<html>
	<head/>
	<body>
		<h2><span>
			<p>Texte</p>
			</span></h2><h3></h3>
	</body>
</html>
"

hNoeud est un htmlNoeud
// body : <div id="div"><p>Texte</p></div>
hNoeudInsertion est un htmlNoeud


hNoeudRecherche est un tableau de htmlNoeuds <- HTMLChercheElémentParBalise(hDoc, "h3")
hNoeud <- hNoeudRecherche[1]
hNoeudInsertion..NomBalise = "div"
HTMLInsèrePremierFils(hNoeud, hNoeudInsertion)
// body : <h1/><h2><span><p>Texte</p></span></h2><h3><div/></h3>
hNoeudInsertion..NomBalise = "span"
HTMLInsèreDernierFils(hNoeud, hNoeudInsertion)
// body : <h1/><h2><span><p>Texte</p></span></h2><h3><div/><span/></h3>
Trace(hDoc)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`HTMLInsèreDernierFils(<Noeud source> , <Noeud à insérer>)`
---

**`<Noeud source> : (Variable de type htmlNoeud)`**

Nom de la variable de type [htmlNoeud](../WDLang5/1000026081.md) correspondant au noeud de référence pour l'insertion. 

**`<Noeud à insérer> : (Variable de type htmlNoeud)`**

Nom de la variable de type [htmlNoeud](../WDLang5/1000026081.md) correspondant au noeud fils à insérer après le dernier fils de &lt;Noeud source&gt;.

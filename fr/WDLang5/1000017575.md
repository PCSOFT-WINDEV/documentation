
## gglAdressePostale (Type de variable)

***En anglais : gglPostalAddress ***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **gglAdressePostale** permet de décrire et de modifier une adresse postale d'un contact.

Ce type de variable est utilisé par le type [gglContact](../WDLang5/1000017434.md).

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connexion
Cnx est un gglConnexion
Cnx.Email = "balthazar@gmail.com"
Cnx.MotDePasse = "azerty"
Cnx.NomApplication = "MonAppli-01"
SI GglConnecte(Cnx) = Faux ALORS
	Erreur(ErreurInfo())
FIN
...
// Récupération du tableau des contacts
tabContacts est un tableau de gglContact
tabContacts = GglListeContact(Cnx)
// Affichage des adresses du premier contact
UnContact est un gglContact = tabContacts[1]
UneAdresse est un gglAdressePostale dynamique
POUR TOUT UneAdresse DE UnContact.Adresse
	Trace(UneAdresse.Libellé + " : " + UneAdresse.Valeur)
FIN
```

## Remarques




### Propriétés spécifiques à la description des variables de type gglAdressePostale
<a name="proprietes_specifiques_description_des_variables_type_ggladressepostale_ELTPARAGRAPHE000037"></a>Les propriétés suivantes peuvent être utilisées pour manipuler une variable de type **gglAdressePostale** :

| Nom | Type manipulé | Effet |
| --- | --- | --- |
| Adresse | Chaîne de caractères | Adresse postale du contact. |
| Libellé | Chaîne de caractères | Libellé de description de l'adresse. |
| Principal | Booléen | <br><br>- <u><u><u><u>Vrai</u></u></u></u> si l'adresse correspond à l'adresse principale du contact,<br><br>- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.<br><br><br> |
| Type | Constante de type entier | Type de l'adresse correspondant à une des variables suivantes :<br><br>- *gglAutre* : Autre type d'adresse.<br><br>- *gglBureau* : Adresse professionnelle.<br><br>- *gglDomicile* : Adresse du domicile.<br><br><br> |

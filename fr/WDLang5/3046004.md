
## iMMVersPica (Fonction)

***En anglais : iMMToPica***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Convertit les coordonnées d'un point (exprimées en millimètres) en coordonnées système.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
Largeur est un réel
LargeurPica est un réel
Largeur = iLargeurImage("Fleur.BMP")
LargeurPica = iMMversPica(Largeur)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = iMMVersPica(<Position en MM> [, <Sens>])`
---

**`<Résultat> : (Réel)`**

Position recherchée (horizontale ou verticale) du point en coordonnées système.

**`<Position en MM> : (Réel)`**

Position horizontale ou verticale du point exprimée en millimètres.

**`<Sens> : (Booléen optionnel)`**

Permet de paramétrer le sens de la conversion :

- <u><u><u><u>Vrai</u></u></u></u> (valeur par défaut) : conversion pour une position horizontale.

- <u><u><u><u>Faux</u></u></u></u> : conversion pour une position verticale.

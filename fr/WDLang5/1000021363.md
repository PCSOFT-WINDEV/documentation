


## Fonctions MATLAB
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


Les fonctions d'accès à MATLAB sont les suivantes : 



|   |   |
| --- | --- |
| [mlChargeEspaceDeTravail](../WDLang5/1000021358.md) | Charge l'ensemble des variables d'un fichier MATLAB dans la session. |
| [mlEcritVariable](../WDLang5/1000021355.md) | Ecrit la valeur d'une variable de la session MATLAB. |
| [mlExécute](../WDLang5/1000021353.md) | Evalue et exécute une commande sur une session MATLAB. |
| [mlInitialise](../WDLang5/1000021351.md) | Initialise une nouvelle session MATLAB indépendante avec un espace de travail vide pour exécuter des fonctions. |
| [mlListeVariable](../WDLang5/1000021357.md) | Liste les variables présentes dans la session MATLAB. |
| [mlRAZEspaceDeTravail](../WDLang5/1000021360.md) | Supprime l'ensemble des variables de la session MATLAB. |
| [mlRécupèreVariable](../WDLang5/1000021354.md) | Récupère une valeur d'une variable de la session MATLAB. |
| [mlSauveEspaceDeTravail](../WDLang5/1000021359.md) | Sauve l'espace de travail de la session dans un fichier MATLAB. |
| [mlSupprimeVariable](../WDLang5/1000021356.md) | Supprime une variable de la session MATLAB. |
| [mlTermine](../WDLang5/1000021352.md) | Termine une session MATLAB. Cette session a été initialisée avec la fonction [mlInitialise](../WDLang5/1000021351.md). |

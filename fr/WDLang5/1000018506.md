
## sfRequêteFusionne (Type de variable)

***En anglais : sfMergeRequest ***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **sfRequêteFusionne** permet de décrire les éléments à fusionner lors de l'appel de la fonction [SFFusionne](../WDLang5/1000018509.md). Les caractéristiques des éléments à fusionner peuvent être définies et modifiées à l'aide de différentes propriétés WLangage.

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).

<a name="NOTE0_1"></a>

## Remarques




### Propriétés spécifiques à la description des variables de type sfRequêteFusionne
<a name="proprietes_specifiques_description_des_variables_type_sfrequetefusionne_ELTPARAGRAPHE000039"></a>Les propriétés suivantes peuvent être utilisées pour manipuler un élément à fusionner :

| Nom | Type manipulé | Effet |
| --- | --- | --- |
| EnregistrementMaître | [sfObjet](../WDLang5/1000018439.md) | Enregistrement dans lequel vont être fusionnées les données. |
| IDEnregistrementAFusionner | Chaîne de caractères | Liste des identifiants Salesforce des enregistrements dont les données vont être fusionnées dans l'enregistrement maître. Les différents identifiants sont séparés par des RC (Retour-chariot). |

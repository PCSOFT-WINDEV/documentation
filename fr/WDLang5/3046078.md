
## iRepImageHTML (Fonction)

***En anglais : iDirImageHTML***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de sélectionner le répertoire dans lequel les images doivent être générées lors d'une impression au format HTML.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Génération des images dans le sous-répertoire "Temp" du répertoire "_WEB"
iRepImageHTML(iRépertoireWeb, "Temp")

// Impression au format HTML
iDestination(iHTML)

// Impression de l'état
iImprimeEtat(ETAT_MonEtat)

// Ouverture du fichier HTML généré dans le navigateur
FichierAffiche(iDernierFichier())
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`iRepImageHTML(<Répertoire> [, <Sous-Répertoire>])`
---

**`<Répertoire> : (Constante de type Entier)`**

Répertoire dans lequel les images d'une impression au format HTML doivent être générées :


|   |   |
| --- | --- |
| *iRépertoireHTML* | Répertoire dans lequel le fichier HTML est généré. Ce répertoire est sélectionné grâce à la fonction [iDestination](../WDLang5/3046074.md).<br>Les images générées dans ce répertoire **ne** pourront **pas** être affichées grâce à la fonction [FichierAffiche](../WDLang2/3012005.md). |
| *iRépertoireWeb* | Répertoire "&lt;MonProjet&gt;_WEB" du site. Ce répertoire doit être accessible en lecture et en écriture.<br>Les images générées dans ce répertoire pourront être affichées grâce à la fonction [FichierAffiche](../WDLang2/3012005.md). |



**`<Sous-Répertoire> : (Chaîne de caractères optionnelle)`**

Sous-répertoire relatif au répertoire spécifié dans &lt;Répertoire&gt;. Si aucun sous-répertoire n'est spécifié, les images seront générées directement dans le répertoire spécifié dans &lt;Répertoire&gt;.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Pour plus de détails sur l'affichage d'un état au format HTML, consultez [Afficher un état au format HTML dans le navigateur de l'internaute](../WDChamp/1011051.md).

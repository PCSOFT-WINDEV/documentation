
## iParamètreXLS (Fonction)

***En anglais : iParameterXLS***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Fixe les options du fichier XLS généré lors de l'impression. Il est possible de prendre en compte la mise en forme actuelle (couleur et police) ou non.

Ce paramétrage est pris en compte si une impression est réalisée dans un fichier XLS (fichier XLS généré à partir de l'aperçu avant impression ou généré directement à l'aide de la fonction [iAperçu](../WDLang5/3046069.md) ou [iDestination](../WDLang5/3046074.md)).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Export des couleurs des cellules dans le fichier XLS imprimé
iParamètreXLS(iAvecMiseEnForme)
// Création du fichier XLS
iAperçu(iXLS, "C:\Temp\MonFichierXLS.XLS")
// Code d'impression
iCréePolice(1, 16, iGras, iRoman)
ImprimeTitre()
ImprimeTexte()
// Fin de l'impression et fermeture du fichier XLS créé 
iFinImprime()
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`iParamètreXLS(<Options>)`
---

**`<Options> : (Constante de type Entier)`**

Configure la mise en forme du fichier créé.


|   |   |
| --- | --- |
| *iAvecMiseEnForme* | Conserve la couleur et la police lors de l'export au format XLS. |







<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- La fonction **iParamètreXLS** n'est pas prise en compte au milieu d'une impression.

- Le paramétrage réalisée avec la fonction **iParamètreXLS** est actif :

	- soit jusqu'à la fin de l'application ou du site,

	- soit jusqu'à la prochaine utilisation de la fonction **iParamètreXLS**,

	- soit jusqu'à la prochaine utilisation de la fonction [iRAZ](../WDLang5/3046067.md).




## &lt;Variable htmlDocument&gt;.ConstruitChaîne (Fonction)

***En anglais : &lt;htmlDocument variable&gt;.BuildString***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Génère une chaîne de caractères au format UTF8 contenant un document HTML.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Chargement du document
d est un htmlDocument = HTMLOuvre("..\Modèles\Modèle1.html", depuisFichier)

// Modifications du document
// ...

// Récupère le document modifié
DocumentRésultat est une chaîne = d.ConstruitChaîne()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Document HTML>.ConstruitChaîne()`
---

**`<Résultat> : (Chaîne de caractères)`**

Chaîne de caractères ANSI au format UTF8 contenant le document HTML.

**`<Document HTML> : (Variable de type htmlDocument)`**

Nom de la variable de type [htmlDocument](../WDLang5/1000025910.md) correspondant au document HTML à manipuler.

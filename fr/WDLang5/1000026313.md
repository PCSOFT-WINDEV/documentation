


## &lt;Variable htmlNoeud&gt;.ChercheElémentParBalise (Fonction)

***En anglais : &lt;htmlNode variable&gt;.FindElementByTag***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Cherche les éléments d'un noeud HTML correspondant à une balise.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
d est un htmlNoeud
// ...
t est un tableau de htmlNoeud <- d.ChercheElémentParBalise("form")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Elément à manipuler>.ChercheElémentParBalise([<Elément recherché>])`
---

**`<Résultat> : (Tableau)`**

Tableau d'éléments de type [htmlNoeud](../WDLang5/1000026081.md) correspondant à la recherche.

**`<Elément à manipuler> : (Variable de type htmlNoeud)`**

Nom de la variable de type [htmlNoeud](../WDLang5/1000026081.md) dans laquelle la recherche doit être effectuée.

**`<Elément recherché> : (Chaîne de caractères optionnelle)`**

Nom de la balise à rechercher. Si ce paramètre n'est pas précisé ou correspond à une chaîne vide (""), &lt;Résultat&gt; contiendra tous les éléments.

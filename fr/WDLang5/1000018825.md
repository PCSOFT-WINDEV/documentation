
## iNumColonne (Fonction)

***En anglais : iColumnNum***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le numéro de la colonne en cours dans un état multicolonne.

**Remarque** : Cette fonction peut uniquement être utilisée avec des états créés avec l'éditeur d'états. 




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Changement de la couleur de fond d'un bloc selon le numéro de colonne 
// (cas d'un état à 2 colonnes)
// Dans l'événement "Avant impression du bloc"

MaColonneCourante est un entier 
MaColonneCourante = iNumColonne()
SELON MaColonneCourante
	CAS 1:
		MoiMême.CouleurFond = RougeClair
	CAS 2:
		MoiMême.CouleurFond = VertClair
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = iNumColonne()`
---

**`<Résultat> : (Entier)`**



- Numéro de la colonne en cours d'impression, 

- -1 en cas d'erreur. 

- 1 si l'état en cours n'est pas un état multicolonne.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Le numéro des colonnes commence à 1.

- La propriété [NbColonne](../Proprietes/1000018810.md) permet de connaître le nombre de colonnes de l'état en cours.


## HTMLOuvre (Fonction)

***En anglais : HTMLOpen***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Charge un document HTML dans une variable de type [htmlDocument](../WDLang5/1000025910.md). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
d est un htmlDocument = HTMLOuvre("https://www.pcsoft.fr", depuisURL)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = HTMLOuvre(<Source HTML> [, <Type de source>])`
---

**`<Résultat> : (Variable de type htmlDocument)`**

Variable de type [htmlDocument](../WDLang5/1000025910.md) contenant la source spécifiée. 

**`<Source HTML> : (Chaîne de caractères)`**

Document HTML à charger. Le contenu de ce document dépend du paramètre &lt;Type de source&gt;. 

**`<Type de source> : (Constante optionnelle de type Entier)`**

Mode d'interprétation de la source. Les constantes disponibles sont les suivantes :


|   |   |
| --- | --- |
| *depuisChaîne* | Le paramètre &lt;Source HTML&gt; contient directement le contenu HTML. |
| *depuisFichier* | Le paramètre &lt;Source HTML&gt; est un chemin de fichier. Le contenu HTML est chargé depuis ce fichier. |
| *depuisURL* | Le paramètre &lt;Source HTML&gt; est une URL. Le contenu HTML est récupéré depuis cette URL. |


Si ce paramètre n'est pas spécifié, le type de la source est automatiquement déduit selon le contenu du paramètre &lt;Source HTML&gt; :

- Si le paramètre &lt;Source HTML&gt; commence par "http://", la constante utilisée est *depuisURL*. 

- Si le paramètre &lt;Source HTML&gt; commence par une lettre de lecteur ou un chemin UNC, la constante utilisée est *depuisFichier*. 

- Dans tous les autres cas, la constante utilisée est *depuisChaîne*.

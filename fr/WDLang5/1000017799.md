
## rssImage (Type de variable)

***En anglais : rssImage ***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **rssImage** permet de décrire l'image d'un flux RSS. Les caractéristiques de cette image peuvent être définies et modifiées à l'aide de différentes propriétés WLangage.  

Ce type de variable est utilisé par le [type rssFlux](../WDLang5/1000017795.md).

**Notes** :

- Le plus souvent, la déclaration d'une variable de type **rssImage** n'est pas nécessaire. En effet, les propriétés du [type rssFlux](../WDLang5/1000017795.md) permettent de manipuler directement les propriétés du type rssImage (notamment la propriété .Image).

- Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).

## Remarques




### Propriétés spécifiques à la description des variables de type rssImage
<a name="proprietes_specifiques_description_des_variables_type_rssimage_ELTPARAGRAPHE000058"></a>Les propriétés suivantes peuvent être utilisées pour manipuler une variable de type **rssImage** :

| Nom | Type manipulé | Effet |
| --- | --- | --- |
| Description | Chaîne de caractères | Description de l'image. |
| Hauteur | Entier | Hauteur en pixels de l'image. Si cette propriété n'est pas indiquée, la hauteur est déduite de l'image. |
| Largeur | Entier | Largeur en pixels de l'image. Si cette propriété n'est pas indiquée, la largeur est déduite de l'image. |
| Lien | Chaîne de caractères | Lien associé à l'image (généralement un site WEB). |
| Titre | Chaîne de caractères | Valeur texte de l'image (utilisée quand l'image n'est pas affichable). |
| URL | Chaîne de caractères | URL de l'image du canal. |




## &lt;Etat&gt;.InitSousEtat (Fonction)

***En anglais : &lt;Report&gt;.InitSubReport***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Initialise les paramètres d'un sous-état d'un état composite. Cette fonction doit être utilisée lorsqu'un sous-état demande des paramètres tels que des bornes minimales ou maximales ou des paramètres spécifiés par programmation.

**Attention** : Cette fonction peut être appelée uniquement durant l'exécution de l'impression d'un état composite.

<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Code d'ouverture d'un état composite
SETAT_SousEtat1.InitSousEtat(nBorneMin, nBorneMax)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Sous-état manipulé>.InitSousEtat([<Paramètre 1> [... [, <Paramètre N>]]])`
---

**`<Sous-état manipulé> : (Nom d'état)`**

Nom du sous-état à initialiser.

**`<Paramètre 1> : (Type du paramètre (optionnel))`**

Premier paramètre à passer au sous-état.



**`<Paramètre N> : (Nième du paramètre (optionnel))`**

Dernier paramètre à passer au sous-état.




## XMLNamespaceURI (Fonction)

***En anglais : XMLNamespaceURI***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie l'URI de définition de l'espace de nommage de la balise courante dans un document XML.








<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonDoc est une chaîne
MonDoc = fChargeTexte("Exemple.xml")
XMLDocument("exemple", MonDoc)
SI ErreurDétectée = Vrai ALORS
	Erreur("Le fichier n'est pas au format XML")
	RETOUR
FIN
// On se positionne sur la première balise du fichier
SI XMLPremier() = Vrai ALORS
	SI XMLNamespace() <> "" ALORS
		Info("Le préfixe utilisé pour l'espace de nommage est : " + XMLNamespace(), ...
			"L'URI correspondante est : " + XMLNamespaceURI())
	SINON
		SI ErreurDétectée = Vrai ALORS 
			Info("L'élément lu n'est pas une balise")
	  	SINON
			Info("La balise ne contient pas d'espace de nommage")
	  	FIN
   	FIN
FIN

XMLTermine()
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = XMLNamespaceURI([<Document XML>])`
---

**`<Résultat> : (: Chaîne de caractères)`**



- URI de définition de l'espace de nommage associé à la balise en cours, 

- Chaîne vide ("") : 

	- si la balise ne contient pas de préfixe de nommage. 

	- si le document XML n'est pas positionné sur une balise. La variable [ErreurDétectée](../WDLang1/3087001.md) est à <u><u><u><u>Vrai</u></u></u></u>.







**`<Document XML> : (: Chaîne de caractères optionnelle)`**

Nom du document XML à manipuler. Ce document contient le code XML et a été créé avec la fonction [XMLDocument](../WDLang5/3081008.md). Si ce paramètre n'est pas précisé, la fonction **XMLNamespaceURI**s'applique au dernier document XML manipulé dans le thread en cours.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Manipulation d'une balise associée à un espace de nommage
<a name="manipulation_une_balise_associee_espace_nommage_ELTPARAGRAPHE000088"></a>

- Une balise associée à un espace de nommage peut être manipulée grâce aux fonctions [XMLDonnée](../WDLang5/3081004.md), [XMLExtraitChaîne](../WDLang5/3081017.md), ...

- Le préfixe de l'espace de nommage associé à une balise peut être connu par la fonction [XMLNamespace](../WDLang5/3081039.md).



<a name="NOTE0_2"></a>


### Fonctions XML et threads
<a name="fonctions_xml_threads_ELTPARAGRAPHE000232"></a>Si votre application utilise des threads, le document XML est partagé entre tous ces threads. Pour plus de détails sur la gestion des threads, consultez [Gestion des threads](../WDLang1/3077026.md).


## sfRésultatVideCorbeille (Type de variable)

***En anglais : sfEmptyRecycleBinResult ***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **sfRésultatVideCorbeille** permet de connaître le résultat de la fonction [SFVideCorbeille](../WDLang5/1000018466.md). Les caractéristiques de ce résultat peuvent être connues à l'aide de différentes propriétés  

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connexion à Salesforce
Cnx est une  sfConnexion
Cnx.Login = "balthazar@gmail.com"
Cnx.MotDePasse = "azerty" + "XXB12VCZ54"
SI SFConnecte(Cnx) = Faux ALORS
	Erreur("La connexion a échoué")
	RETOUR
FIN
ResRequête est un sfRésultatExécuteRequête 
ResRequête = SFExécuteRequête(Cnx, "SELECT Date__c,Miles__c,Contact__c FROM Mileage__c")

MonEnregistrement est un sfObjet
POUR TOUT MonEnregistrement DE ResRequête.Objet
	Trace("---- Type : " + MonEnregistrement.Type+" | ID : " + MonEnregistrement.ID)
	Trace(MonEnregistrement.Date__c + " / " + MonEnregistrement.Miles__c + " / " + ...
		MonEnregistrement.Contact__c)
FIN

// Suppression d'un objet
tabSuppr est un tableau de chaînes
Ajoute(tabSuppr, ResRequête.Objet[1].ID)

ResSuppr est un tableau de SFRésultatSupprime
ResSuppr = SFSupprime(Cnx, tabSuppr)
POUR TOUT res DE ResSuppr
	Trace("ID : " + res.ID + " - Suppression = " + res.Succès)
FIN

ResVideCorb est un tableau de SFRésultatVideCorbeille
ResVideCorb = SFVideCorbeille(Cnx, TabSuppr)
POUR TOUT res DE ResVideCorb
	Trace("ID : " + res.ID + "- VideCorbeille  = " + res.Succès)
FIN
```

## Remarques




### Propriétés spécifiques à la description des variables de type sfRésultatVideCorbeille
<a name="proprietes_specifiques_description_des_variables_type_sfresultatvidecorbeille_ELTPARAGRAPHE000045"></a>Les propriétés suivantes peuvent être utilisées pour manipuler le résultat de la fonction [SFVideCorbeille](../WDLang5/1000018466.md) :

| Nom de la propriété | Type manipulé | Effet |
| --- | --- | --- |
| Erreur | Tableau de [sfErreur](../WDLang5/1000018455.md) | Compte-rendu d'erreurs. |
| ID | Chaîne de caractères | Identifiant Salesforce de l'enregistrement supprimé de la corbeille.<br>**Cette propriété est disponible en lecture seulement**. |
| Succès | Booléen | <br><br>- <u><u><u><u>Vrai</u></u></u></u> si la suppression de la corbeille a été effectuée,<br><br>- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.<br><br><br>**Cette propriété est disponible en lecture seulement**. |




## &lt;Variable sfConnexion&gt;.Recherche (Fonction)

***En anglais : &lt;sfConnection variable&gt;.Search***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Effectue une recherche parmi les enregistrements d'une base de données Salesforce. La recherche est effectuée grâce au langage SOSL (Salesforce Object Search Language).




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connexion à Salesforce
Cnx est une sfConnexion
Cnx.Login = "balthazar@gmail.com"
Cnx.MotDePasse = "azerty" + "XXB12VCZ54"
SI Cnx.Connecte() = Faux ALORS
	Erreur("La connexion a échoué")
	RETOUR
FIN

Res est un sfRésultatRecherche
Res = Cnx.Recherche("FIND {John Smith} IN NAME FIELDS")
MonEnregistrement est un sfObjet
POUR TOUT MonEnregistrement de Res..Objet
	Trace(MonEnregistrement..ID)
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Connexion Salesforce>.Recherche(<Conditions de recherche>)`
---

**`<Résultat> : (Variable de type sfRésultatRecherche)`**

Nom de la variable de type [sfRésultatRecherche](../WDLang5/1000018525.md) contenant les données résultat.

**`<Connexion Salesforce> : (Variable de type sfConnexion)`**

Nom de la variable de type [sfConnexion](../WDLang5/1000017579.md) à utiliser.

**`<Conditions de recherche> : (Chaîne de caractères)`**

Code correspondant à la condition de recherche. Cette condition de recherche est écrite dans le langage SOSL (Salesforce Object Search Language). Pour plus de détails sur ce langage, consultez la documentation Salesforce.

<a name="NOTE0_1"></a>

## Remarques


### Cas d'erreurs possibles : 
<a name="cas_erreurs_possibles_ELTPARAGRAPHE000195"></a>

- La connexion Internet n'est pas valide.

- L'authentification n'a pas pu être réalisée correctement.

- La réponse XML du serveur n'est pas lisible.




Pour savoir si une erreur est survenue, utilisez la variable [ErreurDétectée](../WDLang1/3087001.md).

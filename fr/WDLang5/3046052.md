
## iHauteurPage (Fonction)

***En anglais : iPageHeight***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Calcule la hauteur de la page imprimable (en millimètres) en tenant compte des marges (hautes ou bases) définies.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple
<a class="notetitle" target="_blank" href="$DOC$=1000003046052&name=ihauteurpage_fonction&product=WD">Voir des exemples supplémentaires</a>

```wl
// Imprime les dimensions de la feuille (en mm)
iImprime("Largeur de la page : " + iLargeurPage() + "mm")
iImprime("Hauteur de la page : " + iHauteurPage() + "mm")
iFinImprime()
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = iHauteurPage(<Identifiant de la police>)`
---

**`<Résultat> : (Réel)`**

Hauteur de la page :

- en millimètres si le paramètre &lt;Identifiant de la police&gt; n'est pas précisé,

- en nombre de lignes imprimables avec la police &lt;Identifiant de la police&gt;.




**`<Identifiant de la police> : (Chaîne de caractères)`**

Identifiant de la police utilisée, renvoyé par la fonction [iPolice](../WDLang5/3046014.md) ou précisé lors de la création de la police avec la fonction [iCréePolice](../WDLang5/3046045.md). Si ce paramètre n'est pas spécifié, la dernière police utilisée sera sélectionnée.

### Utilisation dans le code d'un état
<a name="utilisation_dans_code_etat_ELTPARAGRAPHE000073"></a>Si la fonction **iHauteurPage** est utilisée :

- après le code d'ouverture de l'état, le résultat correspond à la hauteur imprimable dans l'état (hauteur de la page - les marges de haut et bas).

- dans le code d'ouverture de l'état, le résultat correspond à la hauteur imprimable physiquement (hauteur de la page - les marges physiques de haut et bas définies pour l'imprimante).


## iForceComplément (Fonction)

***En anglais : iForceComplement***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Force l'impression d'un bloc de complément de corps. Le complément de corps s'imprime sur la hauteur de la page restante, c'est-à-dire jusqu'au bas de page.

Cette fonction permet par exemple de forcer l'impression d'un complément de corps : 

- soit après une fin de rupture.

- soit sur la dernière page d'un état.




**Remarque** : Si cette fonction n'est pas utilisée, le complément de corps s'imprime après le dernier bloc de corps de chaque page. Le complément de corps ne s'imprime que si un bloc corps est imprimé dans la page.

![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Imprimer sous Linux](../Editeurs/9000078.md). 




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
iForceComplément()
```

`iForceComplément()`

<a name="NOTE0_1"></a>

## Remarques
La fonction **iForceComplément** pourrait être remplacée par l'impression d'un bloc (fonction [iImprimeBloc](../WDLang5/3046030.md)) redimensionné avec la propriété [Hauteur](../Proprietes/2510050.md).

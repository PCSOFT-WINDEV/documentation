
## iTermineColonne (Fonction)

***En anglais : iColumnEnd***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Force un saut de colonne dans un état multicolonne.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Code après impression d'un bas de rupture
iTermineColonne()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`iTermineColonne()`
---



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Dans le cas d'un état avec une seule colonne, cette fonction se comporte comme la fonction [iTerminePage](../WDLang5/3046038.md).

- Si cette fonction est appelée sur la dernière colonne de l'état, un saut de page est effectué.

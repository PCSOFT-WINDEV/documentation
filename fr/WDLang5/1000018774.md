


## &lt;Variable sfConnexion&gt;.RechercheSupprimé (Fonction)

***En anglais : &lt;sfConnection variable&gt;.SeekDeleted***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Recherche et récupère les enregistrements (objets Salesforce) supprimés pendant la période spécifiée.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connexion à Salesforce
Cnx est une sfConnexion
Cnx.Login = "balthazar@gmail.com"
Cnx.MotDePasse = "azerty" + "XXB12VCZ54"
SI SFConnecte(Cnx) = Faux ALORS
	Erreur("La connexion a échoué")
	RETOUR
FIN

ResRecupSuppr est un sfRésultatRechercheSupprimé
ResRecupSuppr = SFRechercheSupprimé(Cnx, "Mileage__c", "20090101", "20090131")

MonEnregistrement est un sfEnregistrementSupprimé
POUR TOUT MonEnregistrement DE ResRecupSuppr.EnregistrementSupprimé
	Trace("--- ID : " + MonEnregistrement.ID)
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Connexion Salesforce>.RechercheSupprimé(<Fichier Salesforce> , <Date de début> , <Date de fin>)`
---

**`<Résultat> : (Variable de type sfRésultatRechercheSupprimé)`**

Variable de type [sfRésultatRechercheSupprimé](../WDLang5/1000018562.md) contenant les données résultat.

**`<Connexion Salesforce> : (Variable de type sfConnexion)`**

Nom de la variable de type [sfConnexion](../WDLang5/1000017579.md) à utiliser

**`<Fichier Salesforce> : (Chaîne de caractères)`**

Nom du fichier Salesforce à traiter. Ce nom doit correspondre au nom défini sous l'interface Salesforce comme le "Nom API". Dans le cas d'un fichier personnalisé, ce nom correspond au nom du fichier suivi par 2 underscores puis par la lettre "C". Par exemple : "MonCompte__C".

**`<Date de début> : (DateHeure)`**

Date de début de la période pour laquelle le nombre d'objets supprimés est recherché. Cette date ne peut excéder 30 jours avant la date courante. En effet, les éléments supprimés sont conservés uniquement 30 jours dans la corbeille. Ils sont ensuite effacés définitivement par Salesforce. 
Ce paramètre doit être au format UTC (utilisez si nécessaire la fonction [DateHeureLocaleVersUTC](../WDLang1/3027036.md) pour réaliser la conversion).

**`<Date de fin> : (DateHeure)`**

Date de fin de la période pour laquelle le nombre d'objets supprimés est recherché. Ce paramètre doit être au format UTC (utilisez si nécessaire la fonction [DateHeureLocaleVersUTC](../WDLang1/3027036.md) pour réaliser la conversion).

## Remarques


### Cas d'erreurs possibles : 
<a name="cas_erreurs_possibles_ELTPARAGRAPHE000206"></a>

- La connexion Internet n'est pas valide.

- L'authentification n'a pas pu être réalisée correctement.

- La réponse XML du serveur n'est pas lisible.




Pour savoir si une erreur est survenue, utilisez la variable [ErreurDétectée](../WDLang1/3087001.md).

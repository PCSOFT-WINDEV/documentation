
## sfConversionPiste (Type de variable)

***En anglais : sfLeadConvert ***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **sfConversionPiste** permet de décrire les paramètres de conversion d'une piste ou "lead". Les caractéristiques de ce type de variable peuvent être définies et modifiées à l'aide de différentes propriétés WLangage.

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).

## Remarques




### Propriétés spécifiques à la description des variables de type sfConversionPiste
<a name="proprietes_specifiques_description_des_variables_type_sfconversionpiste_ELTPARAGRAPHE000036"></a>Les propriétés suivantes peuvent être utilisées pour manipuler la description de la conversion d'une piste :

| Nom | Type manipulé | Effet |
| --- | --- | --- |
| EcraseSourcePiste | Booléen | <br><br>- <u><u><u><u>Vrai</u></u></u></u> pour écraser la rubrique "Origine de la piste" du contact cible avec le contenu de la rubrique "Origine de la piste" de la piste à convertir,<br><br>- <u><u><u><u>Faux</u></u></u></u> pour ne pas écraser cette information.<br><br><br> |
| EnvoiNotificationEmail | Booléen | <br><br>- <u><u><u><u>Vrai</u></u></u></u> si une notification de conversion doit être envoyée par email au responsable de la piste,<br><br>- <u><u><u><u>Faux</u></u></u></u> pour ne rien faire.<br><br><br> |
| IDCompte | Chaîne de caractères | Identifiant du compte dans lequel la piste sera fusionnée. Si cette propriété est renseignée, le compte spécifié sera mis à jour. Si cette propriété correspond à une chaîne vide, un nouveau compte sera créé.<br>**Important** : Si la propriété "IDCompte" est renseignée, la propriété "IDContact" ne doit pas être renseignée. |
| IDContact | Chaîne de caractères | Identifiant du contact dans lequel la piste sera fusionnée. Si cette propriété est renseignée, le contact spécifié sera mis à jour. Si cette propriété correspond à une chaîne vide, un nouveau contact sera créé. |
| IDPiste | Chaîne de caractères | Identifiant de la piste à convertir. Cette propriété est obligatoire. |
| IDPropriétaire | Chaîne de caractères | Identifiant de la personne possédant le nouveau compte, contact ou opportunité. Par défaut, le propriétaire est celui de la piste. |
| NePasCréerOpportunité | Booléen | <br><br>- <u><u><u><u>Vrai</u></u></u></u> pour ne pas créer d'opportunité pendant la conversion,<br><br>- <u><u><u><u>Faux</u></u></u></u> pour créer une opportunité.<br><br><br>Par défaut, une opportunité est automatiquement créée. |
| NomOpportunité | Chaîne de caractères | Nom de l'opportunité à créer. Si cette propriété correspond à une chaîne vide, le nom de l'entreprise associée à la piste est utilisée. |
| StatutConverti | Chaîne de caractères | Valeur du statut valide pour la piste convertie. Les statuts sont définis par l'administrateur. Pour être proposé, le statut doit être coché pour la colonne "Converti".<br>Cette propriété est obligatoire.<br>**Remarque** : Les valeurs possibles peuvent être connues par programmation en exécutant la requête suivante avec la fonction [SFExécuteRequête](../WDLang5/1000018438.md) : "SELECT Id, MasterLabel FROM LeadStatus WHERE IsConverted=true"). |

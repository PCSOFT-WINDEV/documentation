
## sfRequêteTraitementTâche (Type de variable)

***En anglais : sfProcessWorkItemRequest ***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **sfRequêteTraitementTâche** permet de décrire la requête de traitement d'une tâche à soumettre à la fonction [SFExécuteTraitement](../WDLang5/1000018523.md). Les caractéristiques de ce type peuvent être modifiées à l'aide de différentes propriétés WLangage. 

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).

## Remarques




### Propriétés spécifiques à la description des variables de type sfRequêteTraitementTâche
<a name="proprietes_specifiques_description_des_variables_type_sfrequetetraitementtache_ELTPARAGRAPHE000039"></a>Les propriétés suivantes peuvent être utilisées pour manipuler la requête de traitement d'une tâche à soumettre à la fonction [SFExécuteTraitement](../WDLang5/1000018523.md) :

| Nom | Type manipulé | Effet |
| --- | --- | --- |
| Action | Chaîne de caractères | Action à réaliser sur la tâche Salesforce. Cette action peut correspondre à une des chaînes de caractères suivantes :<br><br>- Approve<br><br>- Reject<br><br>- Remove<br><br><br> |
| Commentaire | Chaîne de caractères | Commentaire à ajouter à l'historique du traitement de la tâche. |
| IDApprobateurSuivant | Chaîne de caractères | Identifiants des utilisateurs qui devront effectuer la prochaine opération. |
| IDTâche | Chaîne de caractères | Identifiant de la tâche à traiter. |

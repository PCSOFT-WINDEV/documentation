
## rssCatégorie (Type de variable)

***En anglais : rssCategory ***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **rssCatégorie** permet de manipuler la catégorie d'un flux RSS ou d'une entrée de flux. Les caractéristiques de ce type de variable peuvent être définies et modifiées à l'aide de différentes propriétés WLangage.

Ce type de variable est utilisé par :

- [le type rssFlux](../WDLang5/1000017795.md).

- [le type rssEntrée](../WDLang5/1000017792.md).




**Notes** :

- Le plus souvent, la déclaration d'une variable de type **rssCatégorie** n'est pas nécessaire. En effet, les propriétés du [type rssFlux](../WDLang5/1000017795.md) ou du [type rssEntrée](../WDLang5/1000017792.md) permettent de manipuler directement les propriétés du type **rssCatégorie** (notamment la propriété **Catégorie**).

- Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).

## Remarques




### Propriétés spécifiques à la description des variables de type rssCatégorie
<a name="proprietes_specifiques_description_des_variables_type_rsscategorie_ELTPARAGRAPHE000066"></a>Les propriétés suivantes peuvent être utilisées pour manipuler une variable de type **rssCatégorie** :

| Nom | Type manipulé | Effet |
| --- | --- | --- |
| Domaine | Chaîne de caractères | URL qui décrit le domaine sur lequel s'applique la catégorie. |
| Valeur | Chaîne de caractères | Valeur arbitraire décrivant la catégorie de l'entrée. Cette valeur est utilisée par les programmes qui agrègent des flux XML. |

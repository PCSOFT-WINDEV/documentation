


## Fonctions Salesforce (syntaxe préfixée)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions de gestion des bases de données Salesforce (syntaxe préfixée) sont les suivantes :



|   |   |
| --- | --- |
| [&lt;Variable sfConnexion&gt;.Ajoute](../WDLang5/1000018753.md) | Crée un enregistrement dans un fichier d'une base de données Salesforce. |
| [&lt;Variable sfConnexion&gt;.AvanceRequête](../WDLang5/1000018754.md) | Continue d'exécuter une requête SOQL (Salesforce Object Query Language) sur les enregistrements d'une base de données Salesforce. |
| [&lt;Variable sfConnexion&gt;.Connecte](../WDLang5/1000018756.md) | Permet de s'authentifier auprès de la plateforme Salesforce. |
| [&lt;Variable sfConnexion&gt;.ConvertitPiste](../WDLang5/1000018757.md) | Convertit des pistes, également appelées LEAD. |
| [&lt;Variable sfConnexion&gt;.Déconnecte](../WDLang5/1000018760.md) | Déconnecte : <br><br>- la plateforme Salesforce. <br><br>- la session Salesforce en cours.<br><br><br> |
| [&lt;Variable sfConnexion&gt;.ExécuteRequête](../WDLang5/1000018762.md) | Exécute une requête SOQL (Salesforce Object Query Language) sur une base de données Salesforce. |
| [&lt;Variable sfConnexion&gt;.ExécuteTraitement](../WDLang5/1000018766.md) | Demande l'exécution d'un traitement spécifique. |
| [&lt;Variable sfConnexion&gt;.Fusionne](../WDLang5/1000018769.md) | Fusionne des enregistrements Salesforce (appelés également objets Salesforce). |
| [&lt;Variable sfConnexion&gt;.Lit](../WDLang5/1000018770.md) | Récupère des enregistrements Salesforce à partir de leurs identifiants. |
| [&lt;Variable sfConnexion&gt;.Modifie](../WDLang5/1000018771.md) | Modifie les enregistrements d'un fichier défini dans Salesforce. |
| [&lt;Variable sfConnexion&gt;.Recherche](../WDLang5/1000018772.md) | Effectue une recherche parmi les enregistrements d'une base de données Salesforce. |
| [&lt;Variable sfConnexion&gt;.RechercheModifié](../WDLang5/1000018773.md) | Recherche et récupère les enregistrements (objets Salesforce) modifiés pendant la période spécifiée. |
| [&lt;Variable sfConnexion&gt;.RechercheSupprimé](../WDLang5/1000018774.md) | Recherche et récupère les enregistrements (objets Salesforce) supprimés pendant la période spécifiée. |
| [&lt;Variable sfConnexion&gt;.Restaure](../WDLang5/1000018775.md) | Restaure des enregistrements Salesforce précédemment effacés. |
| [&lt;Variable sfConnexion&gt;.Supprime](../WDLang5/1000018776.md) | Supprime des enregistrements d'une base de données Salesforce. |
| [&lt;Variable sfConnexion&gt;.VideCorbeille](../WDLang5/1000018778.md) | Vide les enregistrements spécifiés de la corbeille Salesforce. |


## Etape 2 : Création des polices d'impression
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000260"></a>
Cette étape permet de définir et de choisir les différentes polices qui seront utilisées dans le document imprimé. Il est possible de choisir et de définir :

- des polices spécifiques à l'impression : ces polices ne pourront être utilisées que lors de l'impression avec les fonctions du WLangage.

- des polices dynamiques, pouvant être aussi bien utilisées à l'écran (dans des fenêtres, des dessins, ...) que dans des impressions réalisées en WLangage.




**Cette étape est facultative**.

Si aucune police n'est définie pour l'impression, la police par défaut est utilisée.

- Fonte : Don't care,

- Police numéro 0,

- taille : 12 points pica,

- style normal,

- couleur : noir.




**Attention** : Il est possible d'imprimer uniquement avec les polices de l'imprimante. Si la police utilisée n'est pas disponible, la police la plus proche sera utilisée.

![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) Pour plus de détails sur la gestion des polices, consultez [Imprimer en Linux](../Editeurs/9000078.md).

## Police d'impression
<a name="police_impression_ELTTEXTE000284"></a>
Les polices d'impression sont créées avec la fonction [iCréePolice](../WDLang5/3046045.md). Cette fonction permet de choisir :

- la fonte (nom et famille)
	Constantes pour la fonte : 
	


|   |   |
| --- | --- |
| *iCourier* | Fonte de style Courrier (largeur de caractère fixe) |
| *iDéfaut* | Fonte quelconque ("Don't care") |
| *iRoman* | Fonte Roman (police souvent exotique) |
| *iSwiss* | Fonte Swiss (police de qualité) |



- la taille (des caractères)

- les attributs (gras, italique, souligné, ...)
	Constantes pour les attributs : 
	


|   |   |
| --- | --- |
| *iBarre* | Police barrée |
| *iCondense* | Caractères condensés |
| *iEtendu* | Caractères étendus |
| *iGras* | Police en gras |
| *iItalique* | Police italique |
| *iLarge* | Caractères larges |
| *iNormal*<br>(valeur par défaut) | Police normale, sans attributs |
| *iSouligné* | Police soulignée |



- la couleur (voir les [constantes prédéfinies](../WDLang5/3010002.md))




Les différents paramètres de la police sont définis à l'aide de plusieurs constantes.

Chaque police est associée à un numéro. Pour choisir la police d'impression, il suffit d'utiliser la fonction [iPolice](../WDLang5/3046014.md).

## Police dynamique
<a name="police_dynamique_ELTTEXTE000308"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png) Il est également possible de créer des polices dynamiques. Une variable de type "Police" peut être utilisée pour créer une police qui pourra être utilisée dans tous les objets du projet : fenêtres, états, impression, graphes, dessins, ...

Les polices dynamiques permettent par exemple d'imprimer un texte orienté à 90° (grâce à la propriété [Angle](../Proprietes/1410086632.md) de la police dynamique).

Pour définir une police dynamique :

1. Créez une variable de type [Police](../Motscles/1514045.md).

2. Définissez les caractéristiques de la police :

	- soit en utilisant la fonction [PoliceCrée](../WDLang1/3060002.md).

	- soit grâce aux [propriétés de police](../Proprietes/2515045.md) : [Angle](../Proprietes/1410086632.md), [Barré](../Proprietes/2513005.md), [Condensé](../Proprietes/2513006.md), [Couleur](../Proprietes/2510071.md), [Etendu](../Proprietes/2513002.md), [Gras](../Proprietes/2513001.md), [Italique](../Proprietes/2513007.md), [Large](../Proprietes/2513012.md), [Nom](../Proprietes/2510082.md), [Souligné](../Proprietes/2513013.md) et [Taille](../Proprietes/2510125.md).







Pour choisir une police d'impression, il suffit d'utiliser la fonction [iPolice](../WDLang5/3046014.md).

## Police d'imprimante
<a name="police_imprimante_ELTTEXTE000332"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png) Les constructeurs d'imprimante installent des polices sur les imprimantes. Ces polices sont appelées police "imprimante".

Au contraire des polices logicielles (telles que "True Type"), ces polices sont directement accessibles par l'imprimante, elles n'ont pas besoin d'être téléchargées.

Dans les listes des polices, les polices imprimantes sont identifiées par une icône "imprimante" présente devant le nom de la police.

Attention : l'utilisation de polices d'imprimante peut entraîner des effets d'affichage en aperçu avant impression.

Si vous utilisez des polices d'imprimante, ces polices ne sont interprétées que par l'imprimante. Lors de l'aperçu avant impression, l'affichage des polices est géré par Windows. Ne sachant pas gérer ces polices spécifiques, Windows utilise la police écran la plus proche.

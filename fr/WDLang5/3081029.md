


## XMLVersTexte (Fonction)

***En anglais : XMLToText***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Convertit une chaîne de caractères au format XML en une chaîne de caractères standard.








<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Convertir une chaîne de caractères au format XML
// en une chaîne de caractères au format ANSI
sChaîneAnsi est une chaîne
sChaîneAnsi = XMLVersTexte("X est &gt; que Y")
// sChaîneAnsi contient "X est > que Y"
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = XMLVersTexte(<Chaîne à convertir>)`
---

**`<Résultat> : (Chaîne de caractères)`**

Chaîne de caractères. Les caractères réservés suivants sont automatiquement remplacés :

- & devient &

- ' devient '

- &gt; devient &gt; 

- &lt; devient &lt; 

- " devient "




**`<Chaîne à convertir> : (Chaîne de caractères)`**

Chaîne de caractères au format XML à convertir.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Pour extraire une partie d'un code XML contenu dans une balise XML, utilisez la fonction [XMLExtraitChaîne](../WDLang5/3081017.md).

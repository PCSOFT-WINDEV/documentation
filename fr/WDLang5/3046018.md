
## iLien (Fonction)

***En anglais : iLink***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ajoute un lien sur une adresse URL dans la page HTML en cours d'impression.

**Remarque** : La fonction **iLien** n'a aucun effet si l'impression n'est pas réalisée dans un fichier HTML.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
iDestination(iHTMLCSS, "MaPage")
iImprime("Sur la ligne suivante, un lien vers une page")
iLien("Tout savoir sur WINDEV", "http://www.pcsoft.fr")
iFinImprime()
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = iLien(<Libellé du lien> , <Adresse URL> [, <Navigateur en cours>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le lien a été imprimé, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Libellé du lien> : (Chaîne de caractères)`**

Texte du lien. Ce texte sera affiché dans la page HTML sous forme de lien.

**`<Adresse URL> : (Chaîne de caractères)`**

Adresse de la page à lancer. Cette page sera automatiquement ouverte lorsque l'utilisateur cliquera sur le &lt;Libellé du lien&gt;.

**`<Navigateur en cours> : (Booléen optionnel)`**



- <u><u><u><u>Vrai</u></u></u></u> (valeur par défaut) : la page est ouverte dans le navigateur en cours.

- <u><u><u><u>Faux</u></u></u></u> : la page est ouverte dans un nouveau navigateur.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Impression d'une URL
<a name="impression_une_url_ELTPARAGRAPHE000058"></a>Un saut de ligne est automatiquement généré après le libellé du lien. Le curseur est positionné sur la ligne suivante. Le passage à la ligne suivante tient compte de la hauteur de la ligne courante. Cette hauteur dépend de la hauteur de la police utilisée.

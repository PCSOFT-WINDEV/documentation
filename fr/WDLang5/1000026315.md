


## &lt;Variable htmlNoeud&gt;.ChercheElémentParClasse (Fonction)

***En anglais : &lt;htmlNode variable&gt;.FindElementByClass***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Cherche les éléments d'un noeud HTML pour lesquels l'attribut "class" correspond à une valeur spécifique.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
d est un htmlNoeud
e est un tableau de htmlNoeud <- d.ChercheElémentParClasse("Red")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Elément à manipuler>.ChercheElémentParClasse(<Attribut Class recherché>)`
---

**`<Résultat> : (Tableau)`**

Tableau d'éléments de type [htmlNoeud](../WDLang5/1000026081.md) correspondant à la recherche.

**`<Elément à manipuler> : (Variable de type htmlNoeud)`**

Nom de la variable de type [htmlNoeud](../WDLang5/1000026081.md) dans laquelle la recherche doit être effectuée.

**`<Attribut Class recherché> : (Chaîne de caractères)`**

Valeur de l'attribut "Class" recherché.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
La fonction **&lt;Variable htmlNoeud&gt;.ChercheElémentParClasse** respecte les règles de base de "sélection" des éléments HTML par leur(s) classe(s) :

- recherche des différentes classes séparées par des espaces, toutes les classes demandées doivent être présentes (recherche "ET").

- recherche sensible à la casse (de manière générale, les classes sont toujours en minuscules).

- recherche en ignorant les doublons.


## xmlLecteur (Type de variable)

***En anglais : xmlReader***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **xmlLecteur** permet de lire un document XML sans le charger en mémoire. Les caractéristiques de ce document XML peuvent être lues à l'aide de différentes propriétés WLangage. 

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonNoeud est un xmlNoeud
MonNoeud..Type = XMLBalise
MonNoeud..Nom = "MonNoeud"
MonNoeud..Texte = "Le texte du noeud"

Lecteur est un xmlLecteur = xmlOuvreLecteur("fichier.xml")

POUR TOUT Lecteur EN PROFONDEUR
	Trace(Lecteur.Type)
FIN

VariableRAZ(Lecteur) // Fermeture du fichier

Lecteur2 est un xmlLecteur = xmlOuvreLecteur("fichier.xml")

POUR TOUT x DE Lecteur2
	SI Lecteur2.Type = xmlLecteurBaliseDébut _OU_ Lecteur2.Type = xmlLecteurBaliseVide ALORS
		SI Lecteur2.Nom = "config" ALORS
			POUR TOUT y DE x
				Trace(x.Nom)
			FIN
		FIN
	FIN
FIN

VariableRAZ(Lecteur2) // Fermeture du fichier
```

## Remarques
<a name="NOTE0_1"></a>


### Propriétés spécifiques à la description des variables de type xmlLecteur
<a name="proprietes_specifiques_description_des_variables_type_xmllecteur_ELTPARAGRAPHE000029"></a>Les propriétés suivantes peuvent être utilisées pour manipuler un document XML :

| Nom de la propriété | Type manipulé | Effet |
| --- | --- | --- |
| Attribut | Tableau de [xmlAttribut](../WDLang5/1000018799.md) | Tableau des attributs XML associés au noeud. |
| Namespace | Variable de type [xmlNamespace](../WDLang5/1000018803.md) | Espace de nommage du noeud.<br><br>**Cette propriété est disponible en lecture seulement**. |
| NamespaceDéclaré | Tableau de [xmlNamespace](../WDLang5/1000018803.md) | Tableau des namespaces XML déclarés par le noeud. |
| Nom | Chaîne de caractères Unicode | Nom du noeud. <br><br>**Cette propriété est disponible en lecture seulement**. |
| Profondeur | Entier | Profondeur de l'élément courant dans l'arbre XML. <br><br>**Cette propriété est disponible en lecture seulement**. |
| Type | Entier | Type du noeud XML : <br><br>- *XMLLecteurAucun* : L'élément en cours ne correspond pas à un noeud. <br><br>- *XMLLecteurBaliseDébut* : Début de balise. <br><br>- *XMLLecteurBaliseFin* : Fin de balise. <br><br>- *XMLLecteurBaliseVide* : Balise vide. <br><br>- *XMLLecteurCDATA* : Section CDATA.<br><br>- *XMLLecteurCommentaire* : Commentaires. <br><br>- *XMLLecteurEntité* : Déclaration d'une entité. <br><br>- *XMLLecteurInconnu* : Type inconnu. <br><br>- *XMLLecteurProcessingInstruction* : Processing instruction<br><br>- *XMLLecteurRéférenceEntité* : Référence à une entité. <br><br>- *XMLLecteurTexte* : Texte. <br><br>- *XMLLecteurTypeDocument* : Déclaration DOCTYPE. <br><br><br><br><br>**Cette propriété est disponible en lecture seulement**. |
| Valeur | Chaîne de caractères Unicode | Valeur du noeud. <br><br>Remarque : Les références à une entité sont automatiquement interprétées. <br><br>**Cette propriété est disponible en lecture seulement**. |

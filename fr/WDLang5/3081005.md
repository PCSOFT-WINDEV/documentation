## Principe de manipulation d'un document XML
<a name="principe_manipulation_document_xml_ELTTEXTE000120"></a>


Deux méthodes peuvent être utilisées pour manipuler un document XML : 

1. [Utilisation d'une variable de type xmlDocument](../WDLang5/3081018.md) (**Méthode conseillée**). Cette méthode permet d'utiliser un fichier de type XSD. 
	 Plusieurs types de variables spécifiques sont disponibles : 



|   |   |
| --- | --- |
| [xmlAttribut](../WDLang5/1000018799.md) | Le type **xmlAttribut** permet de manipuler les attributs d'un noeud dans un document XML. |
| [xmlDocument](../WDLang5/1000018782.md) | Le type **xmlDocument** permet de manipuler simplement un fichier XML. |
| [xmlNamespace](../WDLang5/1000018803.md) | Le type **xmlNamespace** permet de manipuler l'espace de nommage XML d'un attribut de noeud dans un document XML. |
| [xmlNoeud](../WDLang5/1000018786.md) | Le type **xmlNoeud** permet de manipuler un noeud d'un document XML. |

2. [Utilisation d'une variable de type chaîne pour stocker le document XML](../WDLang5/3081018.md). Le document XML peut être ensuite manipulé grâce aux différentes fonctions XML. 
	Les différentes fonctions de gestion des documents XML sont les suivantes :
	


|   |   |
| --- | --- |
| [JSONVersXML](../WDLang5/1410087330.md) | Convertit une chaîne de caractères au format JSON en une chaîne de caractères au format XML. |
| [TexteVersXML](../WDLang5/3081006.md) | Convertit une chaîne de caractères au format ANSI en une chaîne de caractères au format XML. |
| [XMLAjouteAttribut](../WDLang5/3081011.md) | Ajoute un attribut dans un document XML. |
| [XMLAjouteFils](../WDLang5/3081002.md) | Ajoute une balise fille dans un document XML. |
| [XMLAnnuleRecherche](../WDLang5/3081001.md) | Annule la recherche effectuée par la fonction [XMLRecherche](../WDLang5/3081025.md). |
| [XMLChercheNamespaceParNom](../WDLang5/1000019167.md) | Recherche un namespace (espace de nommage) XML à partir de son nom dans un noeud XML et dans les parents de ce noeud. |
| [XMLChercheNamespaceParURI](../WDLang5/1000019168.md) | Recherche un namespace (espace de nommage) XML à partir de son URI (Universal Resource Identifier) dans un noeud et dans les parents de ce noeud. |
| [XMLConstruitChaîne](../WDLang5/3081016.md) | Récupère et met en forme le contenu d'un document XML (créé avec la fonction [XMLDocument](../WDLang5/3081008.md) et modifié grâce aux fonctions XML). |
| [XMLDernier](../WDLang5/3081013.md) | Positionne sur le dernier élément (balise ou attribut) du niveau en cours de l'arborescence. |
| [XMLDocument](../WDLang5/3081008.md) | Crée un nouveau document XML. |
| [XMLDocumentValide](../WDLang5/1000019272.md) | Valide un document XML à partir d'un schéma XSD. |
| [XMLDonnée](../WDLang5/3081004.md) | Renvoie la valeur de l'élément en cours (balise ou attribut XML) ou d'un de ses attributs. |

| [XMLEnDehors](../WDLang5/3081009.md) | Permet de savoir si l'élément (balise ou attribut) sur lequel on veut se positionner est en dehors du document XML. |
| [XMLExécuteXPath](../WDLang5/3081015.md) | Exécute une requête XPATH : <br><br>	- dans un document XML.<br><br>	- dans une variable de type xmlDocument.<br><br><br> |

| [XMLExtraitDocument](../WDLang5/3081010.md) | Crée un nouveau document XML à partir d'une partie d'un document XML existant. |
| [XMLFils](../WDLang5/3081012.md) | Positionne sur le premier élément fils de l'élément en cours. |
| [XMLFilsExiste](../WDLang5/3081038.md) | Indique si la balise courante a des fils correspondant au type recherché (balise, attribut, ...). |
| [XMLInsèreDocument](../WDLang5/3081014.md) | Insère la balise XML en cours présente dans un document XML dans un autre document XML. |
| [XMLInsèreElément](../WDLang5/3081007.md) | Insère un élément (balise ou attribut) XML dans un document XML. |
| [XMLLit](../WDLang5/1000017030.md) | Renvoie la valeur d'un élément XML ou la valeur de l'attribut d'un élément XML. |
| [XMLModifie](../WDLang5/3081037.md) | Modifie le contenu de l'élément courant dans un document XML. |
| [XMLNamespace](../WDLang5/3081039.md) | Renvoie le préfixe de l'espace de nommage de la balise courante dans un document XML. |
| [XMLNamespaceURI](../WDLang5/3081040.md) | Renvoie l'URI de définition de l'espace de nommage de la balise courante dans un document XML. |
| [XMLNoeudValide](../WDLang5/1000019280.md) | Valide un noeud XML, ses fils et ses attributs à partir de sa description dans le schéma XSD lié au document XML. |
| [XMLNomElément](../WDLang5/3081026.md) | Renvoie le nom de l'élément en cours (balise ou attribut). |
| [XMLNomParent](../WDLang5/3081035.md) | Renvoie le nom de la balise parente de l'élément (balise ou attribut) en cours. |

| [XMLSupprime](../WDLang5/3081036.md) | Supprime l'élément courant d'un document XML, ainsi que toute la branche qui en découle. |
| [XMLTermine](../WDLang5/3081034.md) | Ferme un document XML créé avec [XMLDocument](../WDLang5/3081008.md). |

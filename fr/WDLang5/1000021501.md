


## xlsNomVersCoordonnées (Fonction)

***En anglais : xlsNameToCoordinates***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie les coordonnées d'une cellule d'un fichier XLS ou d'un champ Tableur en fonction de son nom. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
CoordonnéeLigneCellule est un entier 
CoordonnéeLigneCellule = xlsNomVersCoordonnées("C1", xlsNumLigne)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = xlsNomVersCoordonnées(<Nom> , <Type>)`
---

**`<Résultat> : (Entier)`**



- Coordonnée recherchée.

- Chaîne vide en cas d'erreur (coordonnées invalides). 




**`<Nom> : (Chaîne de caractères)`**

Nom de la cellule (par exemple "C1"). 

**`<Type> : (Constante de type Entier)`**

Type de la coordonnée à renvoyer : 


|   |   |
| --- | --- |
| *xlsNumColonne* | &lt;Résultat&gt; correspondra au numéro de colonne. |
| *xlsNumLigne* | &lt;Résultat&gt; correspondra au numéro de ligne. |




## FBFermeSession (Fonction)

***En anglais : FBStopSession***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ferme la session Facebook ouverte avec la fonction [FBOuvreSession](../WDLang5/1000021835.md).




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MaSession est une fbSession
// Connexion à Facebook
...
// Déconnexion
FBFermeSession(MaSession)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`FBFermeSession(<Session>)`
---

**`<Session> : (Variable de type fbSession)`**

Nom de la variable de type [fbSession](../WDLang5/1000021851.md) correspondant à la connexion à fermer. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Par défaut, la session est automatiquement fermée à la fermeture de l'application. 

- Pour utiliser de nouveau les services Facebook, il sera nécessaire de se reconnecter avec la fonction [FBOuvreSession](../WDLang5/1000021835.md).

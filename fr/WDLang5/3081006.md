
## TexteVersXML (Fonction)

***En anglais : TextToXML***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Convertit une chaîne de caractères en une chaîne de caractères au format XML.

Les caractères réservés ("&gt;", "&lt;", ...) sont automatiquement transformés selon la norme XML.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Convertir une chaîne de caractères en une chaîne de caractères au format XML
sChaîneXML est une chaîne 
sChaîneXML = TexteVersXML("L'appareil est > à celui-ci")
// sChaîneXML contient "L&amp;apos;appareil est &amp;gt; à celui-ci"
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = TexteVersXML(<Chaîne de caractères à convertir>)`
---

**`<Résultat> : (Chaîne de caractères)`**

Chaîne de caractères au format XML. Les caractères réservés suivants sont automatiquement remplacés :

- & devient &

- ' devient '

- &gt; devient &gt;

- &lt; devient &lt;

- " devient "




**`<Chaîne de caractères à convertir> : (Chaîne de caractères)`**

Chaîne de caractères à convertir.

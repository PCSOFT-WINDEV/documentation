


## &lt;Variable htmlDocument&gt;.ChercheElémentParID (Fonction)

***En anglais : &lt;htmlDocument variable&gt;.FindElementByID***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Cherche l'élément d'un document HTML dont l'attribut "ID" correspond à une valeur spécifique.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
d est un htmlDocument = HTMLOuvre("https://www.pcsoft.fr", depuisURL)
e est un htmlNoeud <- d.ChercheElémentParID("MonID")
SI e <> Null ALORS
	Info(e.NomBalise)
SINON
	Info("Elément non trouvé")
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Elément à manipuler>.ChercheElémentParID(<Identifiant>)`
---

**`<Résultat> : (Variable de type htmlNoeud)`**

Variable de type [htmlNoeud](../WDLang5/1000026081.md) correspondant à la recherche.

**`<Elément à manipuler> : (Variable de type htmlDocument)`**

Nom de la variable de type [htmlDocument](../WDLang5/1000025910.md) dans laquelle la recherche doit être effectuée.

**`<Identifiant> : (Chaîne de caractères)`**

Identifiant de l'élément à chercher.

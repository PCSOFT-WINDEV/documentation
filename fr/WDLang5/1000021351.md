
## mlInitialise (Fonction)

***En anglais : mlInitialize***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Initialise une nouvelle session MATLAB indépendante avec un espace de travail vide pour exécuter des fonctions. Lors de cet initialisation, le shell de MATLAB est affiché pendant quelques secondes.

Remarque : Cette fonction nécessite MATLAB version 7.3 ou supérieur.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
gnIdSession est un entier
gnIdSession = mlInitialise()

SI gnIdSession = -1 ALORS
	Erreur(ErreurInfo(errComplet))	
SINON
	LIB_SessionML = "MATLAB initialisé"
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = mlInitialise()`
---

**`<Résultat> : (Entier)`**



- Identifiant de la session MATLAB. Cet identifiant sera utilisé dans les différentes fonctions de gestion de MATLAB. 

- -1 en cas d'erreur. Pour connaître le libellé de l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.

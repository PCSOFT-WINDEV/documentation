
## iHauteurRestantePage (Fonction)

***En anglais : iPageHeightLeft***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Calcule la hauteur disponible (en millimètres) restant sur la page en cours en tenant compte des marges (hautes ou basses) définies.

**Remarque** : Cette fonction est utilisable uniquement lors de l'impression d'un état. Dans le cas contraire, la fonction **iHauteurRestantePage** renvoie toujours 0.

![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Imprimer sous Linux](../Editeurs/9000078.md). 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Saut de page si le bloc ne rentre pas
SI Bloc..Hauteur > iHauteurRestantePage() ALORS
	iTerminePage()
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = iHauteurRestantePage()`
---

**`<Résultat> : (Réel)`**

Hauteur de la page disponible pour l'impression (en millimètres).

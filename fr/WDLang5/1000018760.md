


## &lt;Variable sfConnexion&gt;.Déconnecte (Fonction)

***En anglais : &lt;sfConnection variable&gt;.Disconnect***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Déconnecte :

- la plateforme Salesforce.

- la session Salesforce en cours.

- les sessions Salesforce spécifiées.







<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connexion à Salesforce
Cnx est une sfConnexion
Cnx.Login = "balthazar@gmail.com"
Cnx.MotDePasse = "azerty" + "XXB12VCZ54"
SI Cnx.Connecte() = Faux ALORS
	Erreur("La connexion a échoué")
	RETOUR
FIN
...
// Déconnexion (première syntaxe)
SI PAS Cnx.Déconnecte() ALORS
	Erreur("La déconnexion a échoué")
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Déconnecter la plateforme Salesforce

`<Résultat> = <Connexion Salesforce>.Déconnecte()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la déconnexion a été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 


Si la session a été déjà déconnectée ou si il n'y a pas eu de connexion, &lt;Résultat&gt; vaut <u><u><u><u>Vrai</u></u></u></u>.

**`<Connexion Salesforce> : (Variable de type sfConnexion)`**

Nom de la variable de type [sfConnexion](../WDLang5/1000017579.md) à utiliser.


<a name="SYNTAXE2"></a>

### Déconnecter une ou plusieurs sessions

`<Résultat> = <Connexion Salesforce>.Déconnecte(<Identifiant de session> [, <Compte-rendu>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si les déconnexions ont été effectuées, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire (par exemple si une des déconnexions n'a pas pu s'effectuer).




**`<Connexion Salesforce> : (Variable de type sfConnexion)`**

Nom de la variable de type [sfConnexion](../WDLang5/1000017579.md) à utiliser.

**`<Identifiant de session> : (Variable de type Tableau)`**

Tableau de chaînes contenant les identifiants des sessions à fermer.

**`<Compte-rendu> : (Variable optionnelle de type tableau)`**

Tableau de variables de type [sfRésultatDéconnecte](../WDLang5/1000018560.md). Ce tableau contient après l'exécution de la fonction le compte-rendu de chaque déconnexion.

<a name="NOTE0_1"></a>

## Remarques


### Cas d'erreurs possibles : 
<a name="cas_erreurs_possibles_ELTPARAGRAPHE000244"></a>

- La connexion Internet n'est pas valide.

- L'authentification n'a pas pu être réalisée correctement.

- La réponse XML du serveur n'est pas lisible.

- Une des déconnexions n'a pas pu s'effectuer.




Pour savoir si une erreur est survenue, utilisez la variable [ErreurDétectée](../WDLang1/3087001.md).

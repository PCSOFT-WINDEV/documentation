


## &lt;Variable sfConnexion&gt;.Fusionne (Fonction)

***En anglais : &lt;sfConnection variable&gt;.Merge***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Fusionne des enregistrements Salesforce (appelés également objets Salesforce).




<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Connexion Salesforce>.Fusionne(<Type de fusion>)`
---

**`<Résultat> : (Variable de type Tableau)`**

Tableau de variables de type [sfRésultatFusionne](../WDLang5/1000018499.md) contenant les données résultat, notamment les identifiants des enregistrements fusionnés.

**`<Connexion Salesforce> : (Variable de type sfConnexion)`**

Nom de la variable de type [sfConnexion](../WDLang5/1000017579.md) à utiliser

**`<Type de fusion> : (Variable de type Tableau)`**

Tableau de variables de type [sfRequêteFusionne](../WDLang5/1000018506.md) correspondant aux enregistrements (objets Salesforce) à fusionner.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Cas d'erreurs possibles : 
<a name="cas_erreurs_possibles_ELTPARAGRAPHE000168"></a>

- La connexion Internet n'est pas valide.

- L'authentification n'a pas pu être réalisée correctement.

- La réponse XML du serveur n'est pas lisible.




Pour savoir si une erreur est survenue, utilisez la variable [ErreurDétectée](../WDLang1/3087001.md).

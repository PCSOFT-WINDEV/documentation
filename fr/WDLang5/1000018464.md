


## Fonctions Salesforce
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions de gestion des bases de données Salesforce sont les suivantes :



|   |   |
| --- | --- |
| [SFAjoute](../WDLang5/1000018463.md) | Crée un enregistrement dans un fichier d'une base de données Salesforce. |
| [SFAvanceRequête](../WDLang5/1000018483.md) | Continue d'exécuter une requête SOQL (Salesforce Object Query Language) sur les enregistrements d'une base de données Salesforce. |
| [SFConnecte](../WDLang5/1000017583.md) | Permet de s'authentifier auprès de la plateforme Salesforce. |
| [SFConvertitPiste](../WDLang5/1000018498.md) | Convertit des pistes, également appelées LEAD. |
| [SFDéconnecte](../WDLang5/1000017585.md) | Déconnecte : <br><br>- la plateforme Salesforce. <br><br>- la session Salesforce en cours.<br><br><br> |
| [SFExécuteRequête](../WDLang5/1000018438.md) | Exécute une requête SOQL (Salesforce Object Query Language) sur une base de données Salesforce. |
| [SFExécuteTraitement](../WDLang5/1000018523.md) | Demande l'exécution d'un traitement spécifique. |
| [SFFusionne](../WDLang5/1000018509.md) | Fusionne des enregistrements Salesforce (appelés également objets Salesforce). |
| [SFLit](../WDLang5/1000018524.md) | Récupère des enregistrements Salesforce à partir de leurs identifiants. |
| [SFModifie](../WDLang5/1000018478.md) | Modifie les enregistrements d'un fichier défini dans Salesforce. |
| [SFRecherche](../WDLang5/1000018555.md) | Effectue une recherche parmi les enregistrements d'une base de données Salesforce. |
| [SFRechercheModifié](../WDLang5/1000018554.md) | Recherche et récupère les enregistrements (objets Salesforce) modifiés pendant la période spécifiée. |
| [SFRechercheSupprimé](../WDLang5/1000018475.md) | Recherche et récupère les enregistrements (objets Salesforce) supprimés pendant la période spécifiée. |
| [SFRestaure](../WDLang5/1000018527.md) | Restaure des enregistrements Salesforce précédemment effacés. |
| [SFSupprime](../WDLang5/1000018465.md) | Supprime des enregistrements d'une base de données Salesforce. |
| [SFVideCorbeille](../WDLang5/1000018466.md) | Vide les enregistrements spécifiés de la corbeille Salesforce. |

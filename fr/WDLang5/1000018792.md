


## XMLOuvre (Fonction)

***En anglais : XMLOpen***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Charge un document XML à partir d'un fichier, d'une URL ou d'une chaîne dans une variable de type [xmlDocument](../WDLang5/1000018782.md).






<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Chargement du flux RSS des derniers articles publiés sur le forum pcsoft.fr.windev 
Flux est un xmlDocument 
Flux = XMLOuvre("http://forum.pcsoft.fr/rss.awp?groupid=pcsoft.fr.windev&v=2", depuisURL)
SI ErreurDetectée = Vrai ALORS
	Erreur("Impossible d'ouvrir le flux RSS. Vérifiez votre connexion Internet.")
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Charger un document XML depuis un fichier ou une URL

`<Résultat> = XMLOuvre(<Source du document> [, <Type de source>])`
---

**`<Résultat> : (Variable de type xmlDocument)`**

Variable de type [xmlDocument](../WDLang5/1000018782.md) contenant le résultat du chargement et de l'analyse du document XML.
En cas de problème, la variable [ErreurDétectée](../WDLang1/3087001.md) est à <u><u><u><u>Vrai</u></u></u></u>. 

**`<Source du document> : (Chaîne de caractères)`**

Correspond :

- au chemin du fichier contenant le document XML (dans le cas d'une source fichier).

- à l'URL du document XML (dans le cas d'une source URL).




**`<Type de source> : (Constante optionnelle de type Entier)`**

Type de source à utiliser :


|   |   |
| --- | --- |
| *depuisFichier* | Le paramètre &lt;Source du document&gt; est le chemin d'un fichier XML. |
| *depuisURL* | Le paramètre &lt;Source du document&gt; est l'URL d'un fichier XML. |




<a name="SYNTAXE2"></a>

### Charger un document XML contenu dans une chaîne

`<Résultat> = XMLOuvre(<Document XML> , <depuisChaîne>)`
---

**`<Résultat> : (Variable de type xmlDocument)`**

Variable de type [xmlDocument](../WDLang5/1000018782.md) contenant le résultat du chargement et de l'analyse du document XML.
En cas de problème, la variable [ErreurDétectée](../WDLang1/3087001.md) est à <u><u><u><u>Vrai</u></u></u></u>. 

**`<Document XML> : (Chaîne de caractères)`**

Code XML à charger.

**`<depuisChaîne> : (Constante de type Entier)`**

Type de source à utiliser :


|   |   |
| --- | --- |
| *depuisChaîne* | Le paramètre &lt;Document XML&gt; est une chaîne contenant directement du code XML. |






## Remarques
Si l'encodage n'est pas précisé dans le document XML, l'encodage UTF-8 est utilisé par défaut.

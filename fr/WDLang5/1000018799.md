
## xmlAttribut (Type de variable)

***En anglais : xmlAttribute (Type of variable)***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **xmlAttribut** permet de manipuler les attributs d'un noeud dans un document XML.

Ce type de variable est utilisé par : 

- le type [xmlNoeud](../WDLang5/1000018786.md).

- le type [xpathRésultat](../WDLang5/1000021967.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) le type [xmlLecteur](../WDLang5/1000023392.md).




**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).


<a name="NOTE0_1"></a>

## Remarques


### Propriétés spécifiques à la description des variables de type xmlAttribut
<a name="proprietes_specifiques_description_des_variables_type_xmlattribut_ELTPARAGRAPHE000066"></a>Les propriétés suivantes peuvent être utilisées pour manipuler une variable de type **xmlAttribut** :


| Nom | Type manipulé | Effet |
| --- | --- | --- |
| Namespace | Variable de type [xmlNamespace](../WDLang5/1000018803.md) | Espace de nommage dans lequel se trouve l'attribut.<br><br>Deux attributs de même nom peuvent être définis sur un noeud de document XML s'ils proviennent d'espaces de nommage différents. |
| Nom | Chaîne de caractères Unicode | Nom de l'attribut.<br><br> |
| Valeur | Chaîne de caractères Unicode | Valeur de l'attribut. |
| Vide | Booléen | <br><br>- <u><u><u><u>Vrai</u></u></u></u> si l'attribut est défini dans un noeud donné,<br><br>- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.<br><br><br>Cette propriété est accessible en lecture seulement. |





![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Attention : L'ordre d'itération n'est pas nécessairement l'ordre de déclaration des attributs dans le document (généralement les attributs sont listés dans l'ordre alphabétique).

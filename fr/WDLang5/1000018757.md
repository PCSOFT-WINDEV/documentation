


## &lt;Variable sfConnexion&gt;.ConvertitPiste (Fonction)

***En anglais : &lt;sfConnection variable&gt;.ConvertLead***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Convertit des pistes, également appelées LEAD. Ces pistes peuvent être converties en compte, en contact ou en opportunité.  

**Remarque** : Une piste est un prospect ou une opportunité potentielle. Pour plus de détails, consultez l'aide de Salesforce.




<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Connexion Salesforce>.ConvertitPiste(<Conversion à effectuer>)`
---

**`<Résultat> : (Variable de type Tableau)`**

Tableau des variables de type [sfRésultatConvertitPiste](../WDLang5/1000018739.md) contenant les résultats de conversion.

**`<Connexion Salesforce> : (Variable de type sfConnexion)`**

Nom de la variable de type [sfConnexion](../WDLang5/1000017579.md) à utiliser.

**`<Conversion à effectuer> : (Variable de type Tableau)`**

Tableau des variables de type [sfConversionPiste](../WDLang5/1000018742.md) contenant les différentes conversions à effectuer.

## Remarques


### Cas d'erreurs possibles : 
<a name="cas_erreurs_possibles_ELTPARAGRAPHE000170"></a>

- La connexion Internet n'est pas valide.

- L'authentification n'a pas pu être réalisée correctement.

- La réponse XML du serveur n'est pas lisible.




Pour savoir si une erreur est survenue, utilisez la variable [ErreurDétectée](../WDLang1/3087001.md).

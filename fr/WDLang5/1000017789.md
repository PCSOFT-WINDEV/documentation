
## rssDocumentJoint (Type de variable)

***En anglais : rssEnclosure ***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **rssDocumentJoint** permet de décrire et de manipuler la pièce jointe d'une entrée de flux RSS. Les caractéristiques de ce type de variable peuvent être définies et modifiées à l'aide de différentes propriétés WLangage. 

Ce type de variable est utilisé par le [type rssEntrée](../WDLang5/1000017792.md).  

**Notes** :

- Le plus souvent, la déclaration d'une variable de type **rssDocumentJoint** n'est pas nécessaire. En effet, les propriétés du [type rssEntrée](../WDLang5/1000017792.md) permettent de manipuler directement les propriétés du type **rssDocumentJoint**.

- Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).

## Remarques




### Propriétés spécifiques à la description des variables de type rssDocumentJoint
<a name="proprietes_specifiques_description_des_variables_type_rssdocumentjoint_ELTPARAGRAPHE000059"></a>Les propriétés suivantes peuvent être utilisées pour manipuler une variable de type **rssDocumentJoint** :

| Propriété | Type manipulé | Effet |
| --- | --- | --- |
| Taille (\*) | Entier | Taille du document en octets. |
| Type (\*) | Chaîne de caractères | Type MIME du document. Plus de 150 types sont définis dans la norme de communication MIME (disponible sur Internet). Les valeurs les plus courantes sont :<br><br>- "application/pdf" : document au format PDF (\*.pdf)<br><br>- "text/html" : page HTML (\*.htm, \*.html)<br><br>- "text/plain": fichier texte (\*.txt)<br><br>- "image/gif" : image au format GIF (\*.gif)<br><br>- "image/jpeg" : image au format JPEG (\*.jpg, \*.jpeg)<br><br>- "video/mpeg" : vidéo au format MPEG (\*.mpg, \*.mpeg)<br><br>- "application/unknown" : permet d'afficher une boîte de dialogue pour proposer le téléchargement du fichier sur le poste de l'internaute.<br><br>- "application/msword" : fichier Word.<br><br>- "application/vnd.ms-excel" : fichier Excel.<br><br><br> |
| URL (\*) | Chaîne de caractères | URL permettant d'accéder au document. |


**Remarques** :

- Les propriétés suivies d'une astérisque (\*) sont obligatoires.

- Toutes les propriétés de cette liste sont disponibles en lecture et en écriture.

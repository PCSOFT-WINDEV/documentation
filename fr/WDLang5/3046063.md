
## iListeImprimante (Fonction)

***En anglais : iListPrinter***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la liste des imprimantes installées sur le poste en cours.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
sListeImprimante est une chaîne
sListeImprimante = iListeImprimante()
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = iListeImprimante()`
---

**`<Résultat> : (Chaîne de caractères)`**

Liste des imprimantes sous la forme :

```txt
<Imprimante 1> + RC + ... + <Imprimante N>
```

Où &lt;Imprimante1&gt; ... &lt;Imprimante N&gt; correspondent au nom des imprimantes.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Pour afficher la liste des imprimantes dans un champ Liste ou un champ Combo, utilisez la syntaxe :


```txt
ListeAjoute(<Nom Champ Liste ou Combo>, <Résultat>)
```

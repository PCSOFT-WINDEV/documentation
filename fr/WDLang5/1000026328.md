
## Constantes HTML
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les constantes de gestion des fichiers HTML sont les suivantes :

| Constante | Détail | Fonction / Type de variable |
| --- | --- | --- |
| **htmlNoeudCData** | Type du noeud dans le document HTML : Noeud CDATA (compatibilité XHTML). | Type [htmlNoeud](../WDLang5/1000026081.md) |
| **htmlNoeudCommentaire** | Type du noeud dans le document HTML : Noeud de type commentaire. | Type [htmlNoeud](../WDLang5/1000026081.md) |
| **htmlNoeudElément** | Type du noeud dans le document HTML : Noeud de type élément, une balise du HTML. | Type [htmlNoeud](../WDLang5/1000026081.md) |
| **htmlNoeudTexte** | Type du noeud dans le document HTML : Noeud de type texte. | Type [htmlNoeud](../WDLang5/1000026081.md) |

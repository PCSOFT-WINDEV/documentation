
## iMagentaTransparent (Fonction)

***En anglais : iTransparentMagenta***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie la gestion de la couleur Magenta afin que cette couleur soit considérée comme transparente ou non dans toutes les prochaines impressions.

**Rappel** : Par défaut, la couleur Magenta est imprimée en tant que Magenta aussi bien dans les états réalisés en WLangage que dans les états créés avec l'éditeur d'Etats.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Imprime Image1.BMP : le magenta est transparent
iMagentaTransparent(Vrai)
iImprimeImage("Image1.BMP")
// Imprime Image2.BMP : le magenta est utilisé normalement
iMagentaTransparent(Faux)
iImprimeImage("Image2.BMP")
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`iMagentaTransparent([<Actif>])`
---

**`<Actif> : (Booléen optionnel)`**



- <u><u><u><u>Vrai</u></u></u></u> (valeur par défaut) : la couleur Magenta est considérée comme transparente. 

- <u><u><u><u>Faux</u></u></u></u> : la couleur Magenta est imprimée telle quelle.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- A partir de la première utilisation de iMagentaTransparent(Vrai), la couleur Magenta devient transparente dans toutes les impressions réalisées jusqu'à la prochaine utilisation de cette fonction.

- Il est possible d'utiliser plusieurs fois la fonction **iMagentaTransparent** dans le même traitement.

- Rappel : la couleur Magenta utilisée pour la gestion de la transparence correspond à RVB(255, 0, 255) où Rouge = 255, Vert = 0 et Bleu = 255.


## sfErreur (Type de variable)

***En anglais : sfError ***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **sfErreur** permet de connaître les caractéristiques d'une erreur renvoyée par Salesforce. Les caractéristiques de ces erreurs peuvent être connues à l'aide de différentes propriétés WLangage. 

Ce type de variable est utilisé par les types de variables suivants :

- [sfRésultatConvertitPiste](../WDLang5/1000018739.md).

- [sfRésultatDéconnecte](../WDLang5/1000018560.md).

- [sfRésultatExécuteTraitement](../WDLang5/1000018559.md).

- [sfRésultatFusionne](../WDLang5/1000018499.md).

- [sfRésultatRestaure](../WDLang5/1000018526.md).

- [sfRésultatSauvegarde](../WDLang5/1000018460.md).

- [sfRésultatSupprime](../WDLang5/1000018546.md).

- [sfRésultatVideCorbeille](../WDLang5/1000018467.md).




**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).

## Remarques




### Propriétés spécifiques à la description des variables de type sfErreur
<a name="proprietes_specifiques_description_des_variables_type_sferreur_ELTPARAGRAPHE000063"></a>Les propriétés suivantes peuvent être utilisées pour manipuler une erreur Salesforce :

| Nom de la propriété | Type manipulé | Effet |
| --- | --- | --- |
| CodeErreur | Chaîne de caractères | Code de l'erreur. Ce code permet de connaître l'erreur rencontrée. Pour plus de détails, consultez directement la documentation de l'API Salesforce (mot-clé StatusCode).<br>**Cette propriété est disponible en lecture seulement**. |
| Message | Chaîne de caractères | Texte du message d'erreur.<br>**Cette propriété est disponible en lecture seulement**. |
| Rubrique | Chaîne de caractères | Liste des rubriques concernées par l'erreur. Les noms des rubriques sont séparés par des RC (retour-chariot).<br>**Cette propriété est disponible en lecture seulement**. |

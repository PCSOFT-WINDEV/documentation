
## mlExécute (Fonction)

***En anglais : mlExecute***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Evalue et exécute une commande sur une session MATLAB.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI gnIdSession = -1 ALORS	
	ToastAffiche("MATLAB non initialisé")
	RETOUR
FIN

sChaîneAExécuter est une chaîne = [
a = [ 1 2 3; 2 3 4; 1 2 5]
b = [ 2 1 3 ; 5 0 -2; 2 3 -1]
prod = a * b
]

soit sRésultat = mlExécute(gnIdSession, sChaîneAExécuter)

SAI_Résultat = sRésultat
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = mlExécute(<Identifiant de la session> , <Commande>)`
---

**`<Résultat> : (Chaîne de caractères)`**

Contenu de la sortie standard renvoyée par l'exécution de la commande.

**`<Identifiant de la session> : (Entier)`**

Identifiant de la session MATLAB. Cet identifiant est renvoyé par la fonction [mlInitialise](../WDLang5/1000021351.md). 

**`<Commande> : (Chaîne de caractères)`**

Commande MATLAB à exécuter.




## iListeEtatsImbriqués (Fonction)

***En anglais : iListNestedReports***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie : 

- la liste des états imbriqués en cours d'impression (états lancés par la fonction [iImprimeEtat](../WDLang5/3046032.md) dans un des traitements de l'état en cours d'impression). 

- la liste des états enchaînés en cours d'impression (fonction [iEnchaînementAjoute](../WDLang5/1000020059.md)).




**Remarque** : Cette fonction permet de connaître : 

- le nom des différents états imprimés à partir de l'état en cours d'impression (appelés états imbriqués). Cette liste est construite au fur et à mesure de l'impression des états imbriqués.

- le nom des différents états enchaînés. La liste des états enchaînés est connus dès l'impression du premier état.




**Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Imprimer sous Linux](../Editeurs/9000078.md). 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
ListeEtat est une chaîne = iListeEtatsImbriqués()
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = iListeEtatsImbriqués()`
---

**`<Résultat> : (Chaîne de caractères)`**



- Liste des noms logiques des états imbriqués déjà imprimés.

- Liste des noms logiques des états enchaînés.




Chaque nom est séparé du nom suivant par des tabulations (caractère TAB).

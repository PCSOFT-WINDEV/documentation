<a name="Utilisation"></a>
<a name="description"></a>
Récupère la liste des groupes de contacts associés au compte Google spécifié.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connexion
Cnx est un gglConnexion
...
// Récupère les groupes de contacts
tabGroupes est un tableau de gglGroupeContact
tabGroupes = GglListeGroupeContact(Cnx) 
// Parcours des groupes
Grp est un gglGroupeContact dynamique
POUR TOUT Grp DE tabGroupes
	Trace(Grp.Nom)
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Récupérer les groupes d'un contact

`<Résultat> = GglListeGroupeContact(<Connexion Google> , <Contact>)`
---

**`<Résultat> : (Tableau de variables gglGroupeContact)`**

Tableau de variables de type [gglGroupeContact](../WDLang5/1000017576.md).

**`<Connexion Google> : (Variable de type gglConnexion)`**

Nom de la variable de type [gglConnexion](../WDLang5/1000018009.md) à utiliser. Cette connexion a été validée avec la fonction [GglConnecte](../WDLang5/1000017332.md).

**`<Contact> : (Variable de type gglContact)`**

Nom de la variable de type [gglContact](../WDLang5/1000017434.md) dont on veut connaître les groupes associés.  


<a name="SYNTAXE2"></a>

### Récupérer les groupes de la connexion

`<Résultat> = GglListeGroupeContact(<Connexion Google>)`
---

**`<Résultat> : (Tableau de variables gglGroupeContact)`**

Tableau de variables de type [gglGroupeContact](../WDLang5/1000017576.md).

**`<Connexion Google> : (Variable de type gglConnexion)`**

Nom de la variable de type [gglConnexion](../WDLang5/1000018009.md) à utiliser. Cette connexion a été validée avec la fonction [GglConnecte](../WDLang5/1000017332.md).

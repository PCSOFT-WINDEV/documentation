
## iFenêtreAbandon (Fonction)

***En anglais : iWindowCancel***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Paramètre l'affichage de la fenêtre permettant l'abandon de l'impression en cours :
<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=IFENETREABANDON.gif)


Il est possible :

- d'afficher ou non la fenêtre d'abandon,

- de modifier le titre de la fenêtre d'abandon,

- de modifier le message de la fenêtre d'abandon.




**Remarque** : Cette fonction doit être appelée **avant** le début d'une impression (par exemple, avant la fonction [iImprime](../WDLang5/3046031.md) ou [iImprimeEtat](../WDLang5/3046032.md), ...).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) 
```wl
// Change le titre de la fenêtre et affiche une ligne
iFenêtreAbandon(Vrai, "Patientez", "Impression de la page#P")
iImprimeEtat(ETAT_NomEtat)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`iFenêtreAbandon(<Affichage> , <Titre> , <Texte>)`
---

**`<Affichage> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> pour afficher la fenêtre d'abandon de l'impression, 

- <u><u><u><u>Faux</u></u></u></u> pour ne pas afficher la fenêtre d'abandon de l'impression.




**`<Titre> : (Chaîne de caractères)`**

Nouveau titre de la fenêtre d'abandon d'impression. 
Si ce paramètre n'est pas précisé, le titre par défaut de la fenêtre est conservé.

**`<Texte> : (Chaîne de caractères)`**

Message à afficher dans la fenêtre d'abandon d'impression. 
Si ce paramètre n'est pas précisé, le texte par défaut est affiché. 

Pour préciser la page en cours d'édition dans ce message, utilisez la chaîne "#P". Cette chaîne sera remplacée à l'exécution par le numéro de la page en cours.

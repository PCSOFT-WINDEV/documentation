
## iRAZ (Fonction)

***En anglais : iReset***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ré-initialise les paramètres d'impression (marges, police, format du papier, orientation, ...) mémorisés dans la librairie d'impression.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
iRAZ()
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`iRAZ()`
---



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
La fonction **iRAZ** doit être utilisée pour modifier les paramètres d'impression en cours de programme lorsque les fonctions [iConfigure](../WDLang5/3046072.md) et [iParamètre](../WDLang5/3046008.md) ne peuvent pas être utilisées.




## &lt;Variable htmlDocument&gt;.Sauve (Fonction)

***En anglais : &lt;htmlDocument variable&gt;.Save***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Sauve le contenu d'une variable de type [htmlDocument](../WDLang5/1000025910.md) dans un fichier HTML.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Chargement du document
d est un htmlDocument = HTMLOuvre("..\Modèles\Modèle1.html", depuisFichier)

// Modifications du document
// ...

// Sauvegarde du document modifié
d.Sauve("..\Génération\" + NomDocument)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Document>.Sauve([<Nom du fichier>])`
---

**`<Document> : (Variable de type htmlDocument)`**

Nom de la variable de type [htmlDocument](../WDLang5/1000025910.md) correspondant au document à manipuler.

**`<Nom du fichier> : (Chaîne de caractères optionnelle)`**

Nom du fichier à enregistrer. Si le nom du fichier n'est pas précisé et que le document avait été ouvert à partir d'un fichier, le fichier d'origine est écrasé.

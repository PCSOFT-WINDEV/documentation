


## xlsRecalculeTout (Fonction)

***En anglais : xlsRecalculateAll***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Recalcule toutes les formules d'un document Excel au format XLSX.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonDocument est un xlsDocument

// Modification du document XLSX
MonDocument[1,"A"] = 1
// Définition de la formule
MonDocument[1,"B"].Formule = "A1"
// Recalcule toutes les formules
xlsRecalculeTout(MonDocument)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`xlsRecalculeTout(<Document Excel>)`
---

**`<Document Excel> : (Variable de type xlsDocument)`**

Nom de la variable de type [xlsDocument](../WDLang5/1000017464.md) manipulée. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- La modification de la valeur ou de la formule d'une cellule ne déclenche pas automatiquement le calcul des formules du classeur. 

- La fonction **xlsRecalculeTout** doit être appelée après avoir modifié la valeur ou la formule d'une ou plusieurs cellules. Elle permet de mettre à jour les valeurs des cellules du classeur ayant une formule.

- Cette fonction ne peut pas être utilisée sur un document Excel au format XLS.

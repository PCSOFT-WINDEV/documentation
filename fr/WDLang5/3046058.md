
## Etape 4 : Lancement de l'impression
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000127"></a>
Pour lancer l'impression effective, il suffit d'utiliser la fonction [iFinImprime](../WDLang5/3046053.md).



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Impression par infrarouge
<a name="impression_par_infrarouge_ELTTEXTE000151"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png) Si l'ordinateur et l'imprimante associée sont équipés d'un port Infrarouge, l'impression est réalisée lors de l'appel de la fonction [iFinImprime](../WDLang5/3046053.md).

**Rappel** : L'imprimante utilisée a été préalablement sélectionnée grâce à la fonction [iDestination](../WDLang5/3046074.md), [iParamètre](../WDLang5/3046008.md) ou [iConfigure](../WDLang5/3046072.md).





<a name="NOTE4"></a>
<a name="NOTE4_1"></a>


## Impression grâce à un fichier PCL
<a name="impression_grace_fichier_pcl_ELTTEXTE000175"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png) WINDEV et WEBDEV permettent aussi d'imprimer par l'intermédiaire d'un fichier PCL (Printer Control Language).

Pour réaliser ce type d'impression :

1. Spécifiez le fichier PCL à créer (fonction [iDestination](../WDLang5/3046074.md)).
	Par exemple :  
	
	```wl
	iDestination(iPCL, "C:\MesDocuments\MonFichier.PCL")
	```


2. Copiez le fichier PCL du poste PC vers l'imprimante à utiliser. 
	Par exemple :

	- à l'aide de la fonction [fCopieFichier](../WDLang1/3036004.md).

	- en saisissant une ligne de commande (dans la boîte de dialogue "Démarrer .. Exécuter" de Windows).
			
		```txt
		Copy "<FichierPCL>" "<ImprimanteUtilisée>"
		```








**Remarque** : A partir de la fenêtre de configuration d'une imprimante (fonction [iConfigure](../WDLang5/3046072.md) ou [iParamètre](../WDLang5/3046008.md)), l'utilisateur peut également créer un fichier PCL.

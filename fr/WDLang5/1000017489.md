
## gglCoordonnée (Type de variable)

***En anglais : gglCoordinate ***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **gglCoordonnée** permet de décrire et de manipuler un point géographique d'une carte. Les caractéristiques de ce type de variable peuvent être définies et modifiées à l'aide de différentes propriétés WLangage.

Ce type de variable est utilisé par :

- le type [gglParamètreCarte](../WDLang5/1000017505.md).

- le type [gglChemin](../WDLang5/1000017433.md).




**Remarques** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).

## Remarques




### Propriétés de gestion
<a name="proprietes_gestion_ELTPARAGRAPHE000050"></a>Les propriétés suivantes peuvent être utilisées pour manipuler une variable de type **gglCoordonnée** :

| Nom de la propriété | Type manipulé | Effet |
| --- | --- | --- |
| Latitude | Réel | Latitude du point. |
| Longitude | Réel | Longitude du point. |

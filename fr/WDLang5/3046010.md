
## iPicaVersMM (Fonction)

***En anglais : iPicaToMM***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Convertit les coordonnées d'un point (exprimées en coordonnées système) en millimètres.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
PosX est un entier = iPicaVersMM(50)
PosY est un entier = iPicaVersMM(50, Faux)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = iPicaVersMM(<Position PICA> [, <Sens>])`
---

**`<Résultat> : (Réel)`**

Position recherchée (horizontale ou verticale) du point en millimètres.

**`<Position PICA> : (Réel)`**

Position horizontale ou verticale du point exprimée en coordonnées système.

**`<Sens> : (Booléen optionnel)`**

Permet de paramétrer le sens de la conversion :

- <u><u><u><u>Vrai</u></u></u></u> (valeur par défaut) : conversion pour une position horizontale.

- <u><u><u><u>Faux</u></u></u></u> : conversion pour une position verticale.




## xlsMsgErreur (Fonction)

***En anglais : xlsMsgError***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le libellé de la dernière erreur due à une fonction XLS.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple
<a class="notetitle" target="_blank" href="$DOC$=1000003080016&name=xlsmsgerreur_fonction&product=WD">Voir des exemples supplémentaires</a>

```wl
// Ouverture d'un fichier XLS
IDFichierXLS = xlsOuvre("C:\MesRépertoires\Fichier.XLS")
SI IDFichierXLS <> -1 ALORS
	...
SINON
	// Erreur survenue sur le fichier XLS
	ResErreur = xlsMsgErreur(IDFichierXLS)
	...
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Syntaxe utilisant le type xlsDocument

`<Résultat> = xlsMsgErreur(<Document XLS>)`
---

**`<Résultat> : (Chaîne de caractères)`**



- Libellé de la dernière erreur survenue sur le fichier XLS, 

- Chaîne vide ("") si aucune erreur n'est survenue.




**`<Document XLS> : (Variable de type xlsDocument)`**

Nom de la variable de type [xlsDocument](../WDLang5/1000017464.md) correspondant au fichier XLS en cours.  


<a name="SYNTAXE2"></a>

### Syntaxe conservée par compatibilité avec la version 12

`<Résultat> = xlsMsgErreur(<Identifiant du fichier XLS>)`
---

**`<Résultat> : (Chaîne de caractères)`**



- Libellé de la dernière erreur survenue sur le fichier XLS, 

- Chaîne vide ("") si aucune erreur n'est survenue.




**`<Identifiant du fichier XLS> : (Entier)`**

Identifiant du fichier XLS en cours, renvoyé par la fonction [xlsOuvre](../WDLang5/3080017.md).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Différentes erreurs
<a name="differentes_erreurs_ELTPARAGRAPHE000059"></a>Les erreurs pouvant survenir sur l'ensemble des fonctions XLS sont :

- Le fichier XLS n'a pas été trouvé, n'existe pas, n'est pas ouvert, n'est pas valide ou ne peut être ouvert.

- La ligne et/ou la colonne spécifiée sont vides ou n'existent pas.

- Pas assez de mémoire pour ouvrir le fichier.

- La version d'Excel utilisée est inconnue ou trop ancienne (les fonctions XLS ne peuvent pas lire les fichiers au format Excel version 2 ou inférieure).



<a name="NOTE0_2"></a>


### Fonction équivalente
<a name="fonction_equivalente_ELTPARAGRAPHE000071"></a>La fonction [ErreurInfo](../WDLang1/3013008.md) (utilisée avec la constante *errMessage*) permet également de renvoyer le libellé de la dernière erreur survenue.

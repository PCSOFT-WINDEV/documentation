
## mlSauveEspaceDeTravail (Fonction)

***En anglais : mlSaveWorkspace***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Sauve l'espace de travail de la session dans un fichier MATLAB.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
gnIdSession est un entier = mlInitialise()
SI ErreurDetectée ALORS
	ErreurInfo(errComplet)
	RETOUR
FIN

sNomEspace est une chaîne = "EspTest.mat"
sNomVariable, sListeVariables est une chaîne

mlExécute(gnIdSession, "maVar = 1")
Trace("1 - Variables : " + mlListeVariable(gnIdSession))
mlSauveEspaceDeTravail(gnIdSession, sNomEspace)

mlRAZEspaceDeTravail(gnIdSession)
Trace("2 - Variables : " + mlListeVariable(gnIdSession))
mlExécute(gnIdSession, "maVarEffacee = 2")
mlChargeEspaceDeTravail(gnIdSession, sNomEspace, Faux)
sListeVariables = mlListeVariable(gnIdSession)
Trace("3 - Variables : " + sListeVariables)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = mlSauveEspaceDeTravail(<Identifiant de la session> , <Nom du fichier>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la sauvegarde a réussi.

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire (par exemple si aucune variable n'est définie dans la session). Pour connaître le libellé de l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.




**`<Identifiant de la session> : (Entier)`**

Identifiant de la session MATLAB. Cet identifiant est renvoyé par la fonction [mlInitialise](../WDLang5/1000021351.md). 

**`<Nom du fichier> : (Chaîne de caractères)`**

Chemin et le nom du fichier Mat-File à manipuler.

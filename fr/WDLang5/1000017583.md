


## SFConnecte (Fonction)

***En anglais : SFConnect***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de s'authentifier auprès de la plateforme Salesforce.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connexion à Salesforce
Cnx est une sfConnexion

Cnx.Login = "balthazar@gmail.com"
Cnx.MotDePasse = "azerty" + "XXB12VCZ54"
Cnx.URLServeurLogin = "https://login.salesforce.com/services/Soap/u/42.0"

SI SFConnecte(Cnx) = Faux ALORS
	Erreur("La connexion a échoué." + RC + ErreurInfo())
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = SFConnecte(<Connexion Salesforce>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'authentification a été réalisée, 

- <u><u><u><u>Faux</u></u></u></u> en cas de problème. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Connexion Salesforce> : (Variable de type sfConnexion)`**

Nom de la variable de type [sfConnexion](../WDLang5/1000017579.md) à utiliser. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Cas d'erreurs possibles : 
<a name="cas_erreurs_possibles_ELTPARAGRAPHE000079"></a>

- La connexion Internet n'est pas valide.

- L'authentification n'a pas pu être réalisée correctement.

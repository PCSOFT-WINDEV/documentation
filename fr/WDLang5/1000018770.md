


## &lt;Variable sfConnexion&gt;.Lit (Fonction)

***En anglais : &lt;sfConnection variable&gt;.Read***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Récupère des enregistrements Salesforce à partir de leurs identifiants. Ces enregistrements peuvent provenir d'un fichier prédéfini Salesforce ou d'un fichier personnalisé.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connexion à Salesforce
Cnx est une  sfConnexion
Cnx..Login = "balthazar@gmail.com"
Cnx..MotDePasse = "azerty" + "XXB12VCZ54"
SI Cnx.Connecte() = Faux ALORS
	Erreur("La connexion a échoué")
	RETOUR
FIN

Identifiants est un tableau de chaînes
Ajoute(Identifiants, "123546789")
Ajoute(Identifiants, "987654321")
Res est un tableau de sfObjet 
Res = Cnx.Lit("Date__c,Miles__c,Contact__c", "Mileage__c", Identifiants)

MonEnregistrement est un sfObjet
POUR TOUT MonEnregistrement DE Res
	Trace("---- Type : " + MonEnregistrement..TypeObjet + ...
			" | ID : " + MonEnregistrement..ID)
	Trace(MonEnregistrement..Date__c + " / " + MonEnregistrement..Miles__c + ...
			" / "+  MonEnregistrement..Contact__c)
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Connexion Salesforce>.Lit(<Liste des rubriques> , <Fichier Salesforce> , <Identifiants>)`
---

**`<Résultat> : (Variable de type Tableau)`**

Tableau de variables de type [sfObjet](../WDLang5/1000018439.md) correspondant aux enregistrements Salesforce à récupérer.

**`<Connexion Salesforce> : (Variable de type sfConnexion)`**

Nom de la variable de type [sfConnexion](../WDLang5/1000017579.md) à utiliser.

**`<Liste des rubriques> : (Chaîne de caractères)`**

Liste des rubriques dont la valeur doit être récupérée. Les noms des rubriques sont séparées par des virgules.

**`<Fichier Salesforce> : (Chaîne de caractères)`**

Nom du fichier Salesforce à lire. Ce nom doit correspondre au nom défini sous l'interface Salesforce comme le "Nom API". Dans le cas d'un fichier personnalisé, ce nom correspond au nom du fichier suivi par 2 underscores puis par la lettre "C". Par exemple : "MonCompte__C".

**`<Identifiants> : (Variable de type Tableau)`**

Tableau de chaînes de caractères contenant les identifiants Salesforce des enregistrements à récupérer.

## Remarques


### Cas d'erreurs possibles : 
<a name="cas_erreurs_possibles_ELTPARAGRAPHE000203"></a>

- La connexion Internet n'est pas valide.

- L'authentification n'a pas pu être réalisée correctement.

- La réponse XML du serveur n'est pas lisible.




Pour savoir si une erreur est survenue, utilisez la variable [ErreurDétectée](../WDLang1/3087001.md).

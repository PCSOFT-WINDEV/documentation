


## Fonctions de gestion des flux RSS
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


Les fonctions de gestion des flux RSS sont les suivantes :



|   |   |
| --- | --- |
| [rssAffiche](../WDLang5/1000017397.md) | Construit un flux RSS et renvoie le contenu du flux RSS au client. |
| [rssConstruitChaîne](../WDLang5/1000017395.md) | Construit le flux RSS et renvoie le résultat au format XML dans une chaîne de caractères. |
| [rssInitialise](../WDLang5/1000017388.md) | Charge en mémoire un flux RSS. |
| [rssSauve](../WDLang5/1000017396.md) | Construit le flux RSS et sauvegarde le flux RSS dans un fichier XML. |

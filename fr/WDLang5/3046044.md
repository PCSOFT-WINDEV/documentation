
## iCadre (Fonction)

***En anglais : iBorder***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Imprime un cadre aux coordonnées spécifiées.

**Remarque** : L'origine (0,0) est située dans le coin supérieur gauche de la feuille. Elle tient compte des marges physiques de l'imprimante.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple
<a class="notetitle" target="_blank" href="$DOC$=1000003046044&name=icadre_fonction&product=WD">Voir des exemples supplémentaires</a>

```wl
iImprime(iPosX(50) + CL.NOMCLI)
// Encadre le texte avec un trait de 2 points Pica
// Le fond du cadre est d'un gris à 50%
iCadre(0, 0, iLargeurPage(), 10, 2, 50)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`iCadre(<X1> , <Y1> , <X2> , <Y2> [, <Épaisseur> [, <Remplissage> [, <Couleur> [, <Trait> [, <Brosse> [, <Bord>]]]]]])`
---

**`<X1> : (Réel)`**

Coordonnée horizontale (colonne ou abscisse) du coin haut gauche du cadre (en millimètres). Ce coin définit le bord extérieur du cadre. 
Valeur comprise entre 0 et la largeur de la page (donnée par la fonction [iLargeurPage](../WDLang5/3046036.md)).

**`<Y1> : (Réel)`**

Coordonnée verticale (ligne ou ordonnée) du coin haut gauche du cadre (en millimètres). Ce coin définit le bord extérieur du cadre. 
Valeur comprise entre 0 et la hauteur de la page (donnée par la fonction [iHauteurPage](../WDLang5/3046052.md)).

**`<X2> : (Réel)`**

Coordonnée horizontale (colonne ou abscisse) du coin bas droit du cadre (en millimètres). Ce coin définit le bord extérieur du cadre. 
Valeur comprise entre 0 et la largeur de la page (donnée par la fonction [iLargeurPage](../WDLang5/3046036.md)).

**`<Y2> : (Réel)`**

Coordonnée verticale (ligne ou ordonnée) du coin bas droit du cadre (en millimètres). Ce coin définit le bord extérieur du cadre. 
Valeur comprise entre 0 et la hauteur de la page (donnée par la fonction [iHauteurPage](../WDLang5/3046052.md)).

**`<Épaisseur> : (Réel optionnel)`**

Épaisseur du trait du cadre (en points pica). Valeur par défaut = 1 (un point Pica = 0,353 mm).

**Attention** : Selon la valeur de ce paramètre, l'épaisseur est obtenue en ajoutant plusieurs lignes les unes à côté des autres de l'extérieur vers l'intérieur.

![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Ce paramètre doit correspondre à une valeur de type entier. Si ce n'est pas le cas, l'épaisseur sera arrondie à la valeur entière la plus proche.

**`<Remplissage> : (Entier optionnel (compris entre 1 et 200))`**

Taux de la couleur du bord et du fond du cadre. Si ce paramètre est :

- Égal à 0 (valeur par défaut): aucun remplissage.

- Égal à 100 : remplissage total avec la couleur spécifiée.

- Inférieur à 100 : plus ce paramètre se rapproche de 0 et plus la couleur s'éclaircit.

- Supérieur à 100 : plus ce paramètre se rapproche de 200 et plus la couleur s'assombrit.




**`<Couleur> : (Entier optionnel)`**

Couleur utilisée pour le fond et le bord du cadre. Si ce paramètre n'est pas précisé, la couleur utilisée est le noir. 
Cette couleur peut correspondre à :

- une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)),

- une couleur TSL (obtenue avec la fonction [TSL](../WDLang1/3029057.md)),

- [une couleur prédéfinie du WLangage](../WDLang5/3010002.md).




**`<Trait> : (Constante optionnelle de type entier)`**

Type du trait.



|   |   |
| --- | --- |
| *iContinu*<br>(Valeur par défaut) | Trait continu |
| *iMixte* | Trait mixte (-.-.) |
| *iPointillé* | Trait pointillé (....) |
| *iTiret* | Trait tiret (-----) |





**`<Brosse> : (Constante optionnelle de type entier)`**

Motif de remplissage du cadre.



|   |   |
| --- | --- |
| *iCroisée* | Lignes croisées. |
| *iCroiséeDiagonale* | Lignes croisées en diagonale. |
| *iDiagonaleDescendante* | Lignes diagonales descendantes. |
| *iDiagonaleMontante* | Lignes diagonales ascendantes. |
| *iHorizontale* | Lignes horizontales. |
| *iPleine*<br>(valeur par défaut) | Remplissage complet. |
| *iVerticale* | Lignes verticales |





![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Ce paramètre est ignoré.

**`<Bord> : (Constante optionnelle de type entier)`**

Bords du cadre à dessiner :



|   |   |
| --- | --- |
| *iBordAucun* | Le cadre n'a pas de bords |
| *iBordBas* | Le bord bas est dessiné. |
| *iBordDroit* | Le bord droit est dessiné |
| *iBordGauche* | Le bord gauche est dessiné |
| *iBordHaut* | Le bord haut est dessiné |
| *iBordTous*<br>(Valeur par défaut) | Tous les bords du cadre sont dessinés |

<a name="NOTE0_1"></a>

## Remarques




### Luminosité de la couleur noire
<a name="luminosite_couleur_noire_ELTPARAGRAPHE000213"></a>Dans le cas de la couleur noire, le taux correspond à un taux de gris dont les valeurs s'échelonnent entre 1 (blanc) et 100 (noir complet). Dans ce cas, les valeurs supérieures à 100 seront traitées de la même façon que la valeur 100.


<a name="NOTE0_2"></a>




### Divers
<a name="divers_ELTPARAGRAPHE000222"></a>

- La position du curseur n'est pas modifiée par le tracé du trait.

- L'utilisation du type de trait pointillé (*iPointillé*) ou mixte (*iMixte*) combinée avec un cadre épais peut provoquer des effets disgracieux.

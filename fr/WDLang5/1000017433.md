
## gglChemin (Type de variable)

***En anglais : gglPath ***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **gglChemin** permet de définir les caractéristiques du chemin à dessiner sur une carte. Les caractéristiques de ce chemin peuvent être définies et modifiées à l'aide de différentes propriétés WLangage. Le tracé du chemin utilise les points indiqués (et non pas les rues présentes sur la carte).

Ce type de variable est utilisé par le type [gglParamètreCarte](../WDLang5/1000017505.md). 

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).

## Remarques




### Propriétés spécifiques à la description des variables de type gglChemin
<a name="proprietes_specifiques_description_des_variables_type_gglchemin_ELTPARAGRAPHE000044"></a>Les propriétés suivantes peuvent être utilisées pour manipuler une variable de type **gglChemin** :

| Nom de la propriété | Type manipulé | Effet |
| --- | --- | --- |
| Couleur | Entier | Couleur du tracé. Cette couleur peut correspondre à :<br><br>- une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)),<br><br>- [une couleur prédéfinie](../WDLang5/3010002.md).<br><br><br> |
| Epaisseur | Entier | Epaisseur du trait (en pixels). |
| Opacité | Entier | Opacité du trait (en pourcentage). |
| Point[n] | Tableau de [gglCoordonnée](../WDLang5/1000017489.md) | Points du chemin. Le nombre de points maximum pouvant être utilisés est 100. |
| Point[n].Latitude  | Réel | Latitude du point. |
| Point[n].Longitude | Réel | Longitude du point. |

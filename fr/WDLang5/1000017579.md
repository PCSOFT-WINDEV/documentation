
## sfConnexion (Type de variable)

***En anglais : sfConnection***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **sfConnexion** permet de décrire une connexion à la plateforme Salesforce et de gérer l'authentification de l'utilisateur. Les caractéristiques de ce type de variable peuvent être définies et modifiées à l'aide de différentes propriétés WLangage.

La fonction [SFConnecte](../WDLang5/1000017583.md) permet de vérifier la validité des informations de connexion.

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connexion à Salesforce
Cnx est une sfConnexion

Cnx.Login = "balthazar@gmail.com"
Cnx.MotDePasse = "azerty" + "XXB12VCZ54"
Cnx.URLServeurLogin = "https://login.salesforce.com/services/Soap/u/42.0"

SI SFConnecte(Cnx) = Faux ALORS
	Erreur("La connexion a échoué")
FIN
```

## Remarques




### Propriétés spécifiques à la description des variables de type sfConnexion
<a name="proprietes_specifiques_description_des_variables_type_sfconnexion_ELTPARAGRAPHE000048"></a>Les propriétés suivantes peuvent être utilisées pour manipuler une connexion Salesforce :

| Nom de la propriété | Type manipulé | Effet |
| --- | --- | --- |
| InfoSession | [sfInfoSession](../WDLang5/1000018416.md) | Données sur la session en cours. Cette propriété est renseignée après authentification.<br>**Cette propriété est disponible en lecture seulement**. |
| Login | Chaîne de caractères | Login de l'utilisateur. |
| ModeBacASable | Booléen | <br><br>- <u><u><u><u>Vrai</u></u></u></u> si le mode "Bac à sable" est activé (connexion au serveur de tests)<br><br>- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.<br><br><br> |
| MotDePasse | Chaîne de caractères | Mot de passe de connexion. Il est nécessaire d'ajouter à ce mot de passe un jeton de sécurité fourni par Salesforce. |
| URLServeurLogin | Chaîne de caractères | URL du serveur à utiliser. Cette propriété est optionnelle. Si cette propriété est renseignée, elle est utilisée pour l'authentification. |

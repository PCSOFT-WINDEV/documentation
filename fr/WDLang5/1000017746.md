
## gglAuteur (Type de variable)

***En anglais : gglAuthor ***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **gglAuteur** permet de connaître l'auteur d'un élément Google. Ces informations sont disponibles en lecture uniquement : elles sont renseignées par Google à la création (en se basant sur les informations de connexion) et ne sont pas modifiables.

Ce type de variable est utilisé par :

- Les variables de type [gglAgenda](../WDLang5/1000017418.md).

- Les variables de type [gglEvénement](../WDLang5/1000017440.md).




**Remarques** :

- Le plus souvent, la déclaration d'une variable de type **gglAuteur** n'est pas nécessaire. En effet, les propriétés du type [gglAgenda](../WDLang5/1000017418.md) ou du type [gglEvénement](../WDLang5/1000017440.md) permettent de manipuler directement les propriétés du type **gglAuteur** (notamment la propriété **Auteur**).

- Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connexion à Google
Cnx est un gglConnexion
Cnx.Email = "balthazar@gmail.com"
Cnx.MotDePasse = "azerty"
Cnx.NomApplication = "MonAppli-01"
SI GglConnecte(Cnx) = Faux ALORS
	Erreur(ErreurInfo())
FIN
...

// Récupération du tableau des agendas disponibles
tabAgendas est un tableau de gglAgenda
tabAgendas = GglListeAgenda(Cnx)

// Récupération des événements du premier agenda
GglRemplitAgenda(Cnx, tabAgendas[1])

// Cas 1 : Utilisation directe du nom de l'auteur
Trace("Auteur de l'agenda : " + tabAgendas[1].Auteur.Nom)

// Cas 2 : Utilisation en passant par une variable de type gglAuteur
AuteurAgenda est un gglAuteur
AuteurAgenda = tabAgendas[1].Auteur
Trace("Auteur de l'agenda : " + AuteurAgenda.Nom)
```

## Remarques




### Propriétés spécifiques à la description des variables de type gglAuteur
<a name="proprietes_specifiques_description_des_variables_type_gglauteur_ELTPARAGRAPHE000052"></a>Les propriétés suivantes peuvent être utilisées pour manipuler une variable de type **gglAuteur** :

| Nom de la propriété | Type manipulé | Effet |
| --- | --- | --- |
| Email | Chaîne de caractères | Email de l'auteur. Cette propriété correspond à l'email de l'utilisateur qui a créé l'élément (nom utilisé dans les informations de connexion). <br>**Cette propriété est disponible en lecture seulement**. |
| Nom | Chaîne de caractères | Nom de l'auteur. Cette propriété correspond au nom de l'utilisateur qui a créé l'élément (nom utilisé dans les informations de connexion). <br>**Cette propriété est disponible en lecture seulement**. |

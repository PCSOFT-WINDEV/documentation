
## iLargeurPage (Fonction)

***En anglais : iPageWidth***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Calcule la largeur de la page imprimable (en millimètres) en tenant compte des marges (gauches ou droites) définies.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple
<a class="notetitle" target="_blank" href="$DOC$=1000003046036&name=ilargeurpage_fonction&product=WD">Voir des exemples supplémentaires</a>

```wl
// Imprime les dimensions de la feuille (en mm)
iImprime("Largeur de la page : " + iLargeurPage() + "mm")
iImprime("Hauteur de la page : " + iHauteurPage() + "mm")
iFinImprime()
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = iLargeurPage(<Identifiant de la police>)`
---

**`<Résultat> : (Réel)`**

Largeur de la page :

- en millimètres si le paramètre &lt;Identifiant de la police&gt; n'est pas précisé.

- en nombre de caractères imprimables avec la police &lt;Identifiant de la police&gt;. Si la police &lt;Identifiant de la police&gt; est une police proportionnelle, le nombre retourné correspond au nombre moyen de caractères imprimables.




![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Largeur de la page en millimètres.

**`<Identifiant de la police> : (Chaîne de caractères)`**

Identifiant de la police utilisée, renvoyé par la fonction [iPolice](../WDLang5/3046014.md) ou précisé lors de la création de la police avec la fonction [iCréePolice](../WDLang5/3046045.md). Si ce paramètre n'est pas spécifié, la largeur de la page sera renvoyée en millimètres.

![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Ce paramètre n'est pas disponible.

### Utilisation dans le code d'un état
<a name="utilisation_dans_code_etat_ELTPARAGRAPHE000079"></a>Si la fonction **iLargeurPage** est utilisée :

- après le code d'ouverture de l'état, le résultat correspond à la largeur imprimable dans l'état (largeur de la page - les marges de gauche et droite).

- dans le code d'ouverture de l'état, le résultat correspond à la largeur imprimable physiquement (largeur de la page - les marges physiques de gauche et droite définies pour l'imprimante).

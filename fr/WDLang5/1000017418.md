
## gglAgenda (Type de variable)

***En anglais : gglCalendar ***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **gglAgenda** permet de décrire et de modifier un agenda utilisé par le service Agenda Google Calendar. Les caractéristiques de ce type de variable peuvent être définies et modifiées à l'aide de différentes propriétés WLangage.

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connexion à Google
Cnx est un gglConnexion
..
// Récupération du tableau des agendas disponibles
tabAgendas est un tableau de gglAgenda
tabAgendas = GglListeAgenda(Cnx)
// Récupération des événements du premier agenda
GglRemplitAgenda(Cnx, tabAgendas[1])
Trace("Description de l'agenda : " + tabAgendas[1].Description)
```

## Remarques




### Propriétés spécifiques à la description des variables de type gglAgenda
<a name="proprietes_specifiques_description_des_variables_type_gglagenda_ELTPARAGRAPHE000032"></a>Les propriétés suivantes peuvent être utilisées pour manipuler une variable de type **gglAgenda** :

| Nom de la propriété | Type manipulé | Effet |
| --- | --- | --- |
| Description | Chaîne de caractères | Description de l'agenda. |
| Evénement[n] | Tableau de [gglEvénement](../WDLang5/1000017440.md) | Événements de l'agenda.<br>Un événement ayant de nombreuses caractéristiques, il est conseillé de déclarer une variable de type [gglEvénement](../WDLang5/1000017440.md). |
| FuseauHoraire | Chaîne de caractères | Fuseau horaire utilisé par l'agenda. Cette propriété renvoie une chaîne du type "America/Los Angeles".<br>**Cette propriété est disponible en lecture seulement**. |
| Identifiant | Chaîne de caractères | Identifiant de l'agenda. **Cette propriété est disponible en lecture seulement**. |
| Titre | Chaîne de caractères | Titre de l'agenda. |

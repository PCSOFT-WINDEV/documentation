
## iSauteLigne (Fonction)

***En anglais : iSkipLine***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Force un ou plusieurs sauts de ligne. Le curseur d'impression est positionné aux coordonnées (0, ligne en cours + 1). Par défaut, le saut de ligne est réalisé en fonction de la taille (hauteur) de la dernière ligne imprimée.

L'impression n'est pas lancée (seule la fonction [iFinImprime](../WDLang5/3046053.md) permet de lancer l'impression).

**Remarque** : Cette fonction ne peut pas être utilisée dans les états créés avec l'éditeur d'états.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Impression d'un texte
iImprime("Bonjour")
// Saut de ligne
iSauteLigne()
// Impression d'un texte
iImprime("Nous sommes le" + DateSys())
// Lancement de l'impression
iFinImprime()
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`iSauteLigne([<Nombre de lignes à sauter>])`
---

**`<Nombre de lignes à sauter> : (Entier optionnel)`**

Nombre de lignes à sauter lors de l'impression. Par défaut, ce nombre correspond à 1.

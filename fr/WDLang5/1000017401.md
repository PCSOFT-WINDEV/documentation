


## GglEcrit (Fonction)

***En anglais : GglWrite***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Crée et/ou met à jour des données sur le serveur Google (ajout ou modification de données). Les données créées ou mises à jour sont contenues dans une variable correspondant au service manipulé :

- Variable de type [gglAgenda](../WDLang5/1000017418.md) pour créer/mettre à jour l'agenda.

- Variable de type [gglGroupeContact](../WDLang5/1000017576.md) pour créer/mettre à jour les groupes de contacts.

- Variable de type [gglContact](../WDLang5/1000017434.md) pour créer/mettre à jour les contacts.


**Remarques** : 

- En cas d'ajouts multiples, la fonction [VariableRAZ](../WDLang1/3013069.md) permet de réinitialiser ces variables entre les appels à la fonction **GglEcrit**.

- En cas de modification, il est nécessaire de récupérer les données avant de les modifier.





	<B><B><B>Avertissement</B></B></B>

	 A partir du 15 juin 2021, l'API Contacts de Google utilisée par cette fonctionnalité pour gérer les contacts et les groupes de contacts n'est plus disponible ([https://developers.google.com/contacts/v3/announcement](https://developers.google.com/contacts/v3/announcement)). 
Pour utiliser les contacts de Google, consultez l'API People proposée en remplacement par Google :  [https://developers.google.com/people](https://developers.google.com/people). 
La fonction [RESTEnvoie](../WDLang3/1000021476.md) simplifie l'intégration d'une API REST dans vos développements.

<a name="sample_code"></a>

## Exemple


```wl
// Connexion 
Cnx est une gglConnexion
...

// Récupération de l'agenda nommé "Travail"
Agenda est un gglAgenda = GglRécupèreAgenda(Cnx, "Travail")
// Modification du premier événement de l'agenda
Agenda[1].Titre = "Rdv patron"
Agenda[1].DateFin = "200810131530"
// Suppression du deuxième événement de l'agenda
Supprime(Agenda.Evénement, 2)
// Mise à jour effective des changements sur le serveur
GglEcrit(Cnx, Agenda)
```

<a name="SYNTAXE1"></a>

## Syntaxe

### Créer et/ou mettre à jour un élément Google

`<Résultat> = GglEcrit(<Connexion Google> , <Elément Google>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la mise à jour a été réalisée, 

- <u><u><u><u>Faux</u></u></u></u> en cas d'erreur. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Connexion Google> : (Variable de type gglConnexion)`**

Nom de la variable de type [gglConnexion](../WDLang5/1000018009.md) à utiliser. Cette connexion a été validée avec la fonction [GglConnecte](../WDLang5/1000017332.md).

**`<Elément Google> : (Variable correspondant au type à mettre à jour)`**

Elément à mettre à jour. Ce paramètre peut correspondre à :

- Nom de la variable de type [gglAgenda](../WDLang5/1000017418.md) correspondant à l'agenda à créer/modifier.

- Nom de la variable de type [gglContact](../WDLang5/1000017434.md) correspondant au contact à créer/modifier.

- Nom de la variable de type [gglGroupeContact](../WDLang5/1000017576.md) correspondant au groupe de contacts à créer/modifier.

## Remarques


### Cas d'erreurs possibles :

- La connexion Internet n'est pas valide.

- L'authentification n'a pas pu être réalisée correctement.

- La réponse XML du serveur n'est pas lisible.

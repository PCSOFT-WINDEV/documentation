


## GglDéconnecte (Fonction)

***En anglais : GglDisconnect***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Déconnecte l'utilisateur des services Google utilisés. Lors de la prochaine reconnexion, les autorisations liées aux services Google seront redemandées. 

**Remarque** : Cette fonction n'est pas obligatoire. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
Cnx est une gglConnexion
// ...
// utilisation de la connexion Google
// ...
GglDéconnecte(Cnx)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`GglDéconnecte(<Connexion> [, <Service Google>])`
---

**`<Connexion> : (Variable de type gglConnexion)`**

Nom de la variable de type [gglConnexion](../WDLang5/1000018009.md) correspondant à la connexion à manipuler. 

**`<Service Google> : (Constante ou combinaison de constantes de type Entier)`**

Service Google à déconnecter : 


|   |   |
| --- | --- |
| *gglServiceAgenda* | Service "Agenda Google Calendar" : Agenda Google |
| *gglServiceContacts* | Service "Google Contacts" : Gestion des contacts Google |
| *gglServiceDocuments* | Service "Google Documents List" : Gestion des documents |
| *gglServicePicasa* | Service "Picasa" |


Si ce paramètre n'est pas spécifié, tous les services seront déconnectés.

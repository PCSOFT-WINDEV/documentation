<a name="Utilisation"></a>
<a name="description"></a>
Le type **gglGroupeContact** permet de décrire et de modifier un groupe de contacts utilisé par le service Google Contacts.

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connexion à Google
Cnx est un gglConnexion
Cnx.Email = "balthazar@gmail.com"
Cnx.MotDePasse = "azerty"
Cnx.NomApplication = "MonAppli-01"
SI GglConnecte(Cnx) = Faux ALORS
	Erreur(ErreurInfo())
FIN
...

// Récupère les groupes de contacts
tabGroupes est un tableau de gglGroupeContact
tabGroupes = GglListeGroupeContact(Cnx) 

// Parcours des groupes
Grp est un gglGroupeContact dynamique
POUR TOUT Grp DE tabGroupes
	Trace(Grp.Nom)
FIN
```


<a name="Exemple2"></a>



```wl
// Création d'un groupe de contacts
Cnx est un gglConnexion
Cnx.Email = "MonAdresse@gmail.com"
Cnx.MotDePasse = "MotdePasseAdresseGmail"
Cnx.NomApplication = ""
grpGoogle est un gglGroupeContact

SI GglConnecte(Cnx) = Faux ALORS
	Erreur(ErreurInfo())
SINON		
	// Ajout du groupe
	grpGoogle.Nom = "Mon Nouveau Groupe"
	SI PAS GglEcrit(Cnx, grpGoogle) ALORS 
		Erreur(ErreurInfo())
	FIN	
FIN
```

## Remarques




### Propriétés spécifiques à la description des variables de type gglGroupeContact
<a name="proprietes_specifiques_description_des_variables_type_gglgroupecontact_ELTPARAGRAPHE000051"></a>Les propriétés suivantes peuvent être utilisées pour manipuler une variable de type **gglGroupeContact** :

| Nom | Type manipulé | Effet |
| --- | --- | --- |
| DateMiseAJour | DateHeure | Date de la dernière mise à jour. Cette information est en lecture uniquement.<br>**Remarque** : La date est une date UTC (en temps universel). |
| Identifiant | Chaîne de caractères | Identifiant du groupe. Cette information est en lecture uniquement. |
| Nom | Chaîne de caractères | Nom du groupe. |

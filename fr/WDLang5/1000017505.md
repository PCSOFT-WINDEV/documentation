
## gglParamètreCarte (Type de variable)

***En anglais : gglMapParameter ***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **gglParamètreCarte** permet de définir toutes les caractéristiques avancées d'une carte : points, chemin, ... Les caractéristiques de cette carte peuvent être définies et modifiées à l'aide de différentes propriétés WLangage.

La carte correspondante est ensuite obtenue grâce à la fonction [GglRécupèreCarte](../WDLang5/1000017404.md).

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple
<a class="notetitle" target="_blank" href="$DOC$=1000017570&name=gglparametrecarte_type_variable&product=WD">Voir des exemples supplémentaires</a>

```wl
// Paramétrage d'une carte
Params est un gglParamètreCarte
// Définition des marqueurs
M1 est un gglMarqueur
M1.Latitude = 47.022356
M1.Longitude = 52.423654
M1.Couleur = 0x0000FF // bleu
M1.Taille = gglPetit
// Ajout des marqueurs
Ajoute(Params.Marqueur, M1)
// Paramètres divers
Params.Format = gglJPG
Params.Cadre = Vrai
```

## Remarques




### Propriétés spécifiques à la description des variables de type gglParamètreCarte
<a name="proprietes_specifiques_description_des_variables_type_gglparametrecarte_ELTPARAGRAPHE000052"></a>Les propriétés suivantes peuvent être utilisées pour manipuler une variable de type **gglParamètreCarte** :

| Nom de la propriété | Type manipulé | Effet |
| --- | --- | --- |
| Cadre | Booléen | <br><br>- <u><u><u><u>Vrai</u></u></u></u> pour ajouter un cadre autour de l'image,<br><br>- <u><u><u><u>Faux</u></u></u></u> pour ne pas utiliser de cadre.<br><br><br>Les caractéristiques du cadre sont les suivantes : cadre bleu de 5 pixels de large, avec une opacité de 55%.<br><br>**Remarque** : depuis le passage en version 2 du service de Google, ce paramètre est devenu obsolète et est ignoré. |
| Chemin | [gglChemin](../WDLang5/1000017433.md) | Chemin à tracer dans la carte. Le tracé du chemin utilise les points indiqués (et non pas les rues présentes sur la carte). Les points sont reliés par des lignes. |
| Echelle | Entier | Modifie le nombre de pixels renvoyés. Cette propriété est nécessaire pour un affichage sur un écran haute résolution. <br><br>Si la propriété **Echelle** vaut 2, le nombre de pixels renvoyé est deux fois plus important que si la propriété **Echelle** vaut 1, tout en conservant la même zone de couverture et le même niveau de détail (le contenu de la carte reste inchangé). <br><br>Valeurs possibles : <br><br>- 1 (valeur par défaut), <br><br>- 2,<br><br>- 4 (disponible uniquement en possédant une licence Google Maps APIs Premium Plan). <br><br><br> |
| Format | Constante de type chaîne | Format de l'image reçue. Ce paramètre peut correspondre aux constantes suivantes :<br><br>- *gglGIF* (valeur par défaut) : image au format GIF<br><br>- *gglJPG* : image au format JPG.<br><br>- *gglJPGNonProgressif* : image au format JPG non progressif.<br><br>- *gglPNG8* : image au format PNG 8<br><br>- *gglPNG32* : image au format PNG 32<br><br><br> |
| Langue | Chaîne de caractères | Langue dans laquelle seront affichées les informations sur la carte. Ce paramètre peut correspondre par exemple à "fr", "en", "de", ... |
| Marqueur | Tableau de [gglMarqueur](../WDLang5/1000017442.md) | Marqueurs à afficher sur la carte. Le nombre maximum de marqueurs  pouvant être utilisé est 50. Ce nombre diminue si des points sont ajoutés (en effet, la taille de l'URL attendue par Google limite le nombre maximum de marqueurs). |
| PériphériqueMobile | Booléen | <br><br>- <u><u><u><u>Vrai</u></u></u></u> pour récupérer une carte optimisée pour l'affichage sur mobile,<br><br>- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.<br><br><br> |
| Zone | [gglCoordonnée](../WDLang5/1000017489.md) | Spécifie l'écart en latitude et longitude que doit afficher la carte. Pour que ce paramètre soit pris en compte, le paramètre &lt;Zoom&gt; de la fonction [GglRécupéreCarte](../WDLang5/1000017404.md) doit correspondre à la constante *gglZoomAuto*.<br><br> |

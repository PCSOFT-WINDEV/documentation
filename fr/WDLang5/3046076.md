
## iConfigureEtat (Fonction)

***En anglais : iConfigureReport***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Configure l'imprimante en ouvrant directement la fenêtre de configuration (WINDEV et WINDEV Mobile). La fenêtre de configuration est initialisée avec les paramètres choisis lors du développement (orientation de l'état, marges, ...) de l'état spécifié.



![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Imprimer sous Linux](../Editeurs/9000078.md). 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI iConfigureEtat(ETAT_MonEtat) = Vrai ALORS
	iImprimeEtat(ETAT_MonEtat)
SINON  
	Info("Configuration de l'état abandonnée")
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = iConfigureEtat(<Nom de l'état>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'utilisateur a validé les informations saisies dans la boîte de dialogue, 

- <u><u><u><u>Faux</u></u></u></u> si l'utilisateur a abandonné.




**`<Nom de l'état> : (Chaîne de caractères)`**

Les caractéristiques de cet état (choisies lors du développement de l'état) seront utilisées pour initialiser la fenêtre de configuration (orientation, ...).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Configuration prise en compte lors de l'impression d'un état
<a name="configuration_prise_compte_lors_impression_etat_ELTPARAGRAPHE000064"></a>Seule la dernière configuration est prise en compte lors de l'impression d'un état.

Il est déconseillé d'utiliser plusieurs fois les fonctions **iConfigureEtat** et [iParamètreEtat](../WDLang5/3046077.md) sans imprimer d'état.


<a name="NOTE0_2"></a>




### Ré-initialisation de la configuration
<a name="reinitialisation_configuration_ELTPARAGRAPHE000081"></a>Après l'appel à la fonction **iConfigureEtat** (ou [iParamètreEtat](../WDLang5/3046077.md)), tout nouvel appel à cette fonction ré-initialisera la fenêtre de configuration avec les paramètres utilisés lors du premier appel à la fonction.


## rssCloud (Type de variable)

***En anglais : rssCloud ***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **rssCloud** permet de décrire la manière de s'abonner pour recevoir les notifications de modification du flux RSS. Les caractéristiques de ce type de variable peuvent être définies et modifiées à l'aide de différentes propriétés WLangage. 

Ce type de variable est utilisé par le [type rssFlux](../WDLang5/1000017795.md).

**Notes** :

- Le plus souvent, la déclaration d'une variable de type **rssCloud** n'est pas nécessaire. En effet, les propriétés du [type rssFlux](../WDLang5/1000017795.md) permettent de manipuler directement les propriétés du type **rssCloud** (notamment la propriété **Cloud**).

- Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).

## Remarques




### Propriétés spécifiques à la description des variables de type rssCloud
<a name="proprietes_specifiques_description_des_variables_type_rsscloud_ELTPARAGRAPHE000058"></a>Les propriétés suivantes peuvent être utilisées pour manipuler une variable de type **rssCloud** :

| Nom | Type manipulé | Effet |
| --- | --- | --- |
| Chemin | Chaîne de caractères | Chemin d'accès au service. |
| Domaine | Chaîne de caractères | Nom de l'hôte qui héberge le serveur. |
| Port | Entier | Numéro du port TCP sur lequel le service écoute. |
| Procédure | Chaîne de caractères | Nom de la procédure du service WEB à appeler pour obtenir les notifications. |
| Protocole | Chaîne de caractères | Type de protocole utilisé par le service WEB. |

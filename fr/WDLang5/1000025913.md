


## HTMLChercheElémentParNom (Fonction)

***En anglais : HTMLFindElementByName***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Cherche les éléments d'un document HTML (ou d'un noeud HTML) dont l'attribut "name" correspond à une valeur spécifique.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
d est un htmlDocument = HTMLOuvre("https://www.pcsoft.fr", depuisURL)
e est un tableau de htmlNoeud <- HTMLChercheElémentParNom(d, "Couleur1")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = HTMLChercheElémentParNom(<Elément à manipuler> , <Nom>)`
---

**`<Résultat> : (Tableau)`**

Tableau d'éléments de type [htmlNoeud](../WDLang5/1000026081.md) correspondant à la recherche. 

**`<Elément à manipuler> : (Variable de type htmlDocument ou de type htmlNoeud)`**

Elément dans lequel la recherche doit être effectuée : 

- Nom de la variable de type [htmlDocument](../WDLang5/1000025910.md). 

- Nom de la variable de type [htmlNoeud](../WDLang5/1000026081.md).




**`<Nom> : (Chaîne de caractères)`**

Valeur de l'attribut "name" recherché.

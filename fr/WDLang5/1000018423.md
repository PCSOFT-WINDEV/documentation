
## sfInfoUtilisateur (Type de variable)

***En anglais : sfUserInfo ***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **sfInfoUtilisateur** permet d'obtenir des informations sur un utilisateur. Les caractéristiques de ce type de variable peuvent être connues à l'aide de différentes propriétés WLangage.

Ce type est utilisé par les variables de type [sfInfoSession](../WDLang5/1000018416.md). Les propriétés de ce type sont remplies après l'appel à la fonction [SFConnecte](../WDLang5/1000017583.md).

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connexion à Salesforce
Cnx est une  sfConnexion
Cnx.Login = "balthazar@gmail.com"
Cnx.MotDePasse = "azerty" + "XXB12VCZ54"
SI SFConnecte(Cnx) = Faux ALORS
	Erreur("La connexion a échoué")
SINON
	Trace("Nom de l'utilisateur connecté : " +  ...
			  Cnx.InfoSession.InfoUtilisateur.NomComplet)
FIN
```

### Propriétés spécifiques à la description des variables de type sfInfoUtilisateur
<a name="proprietes_specifiques_description_des_variables_type_sfinfoutilisateur_ELTPARAGRAPHE000052"></a>Les propriétés suivantes peuvent être utilisées pour manipuler les informations concernant un utilisateur de Salesforce :

| Nom de la propriété | Type manipulé | Effet |
| --- | --- | --- |
| CodeISODeviseDéfautUtilisateur | Chaîne de caractères | Code ISO de la devise par défaut de l'utilisateur.<br>**Cette propriété est disponible en lecture seulement**. |
| CodeISODeviseDéfautSociété | Chaîne de caractères | Code ISO de la devise par défaut de la société.<br>**Cette propriété est disponible en lecture seulement**. |
| Email | Chaîne de caractères | Email de l'utilisateur.<br>**Cette propriété est disponible en lecture seulement**. |
| FuseauHoraire | Chaîne de caractères | Fuseau horaire de l'utilisateur.<br>**Cette propriété est disponible en lecture seulement**. |
| IDProfil | Chaîne de caractères | Identifiant du profil de l'utilisateur.<br>**Cette propriété est disponible en lecture seulement**. |
| IDRôle | Chaîne de caractères | Identifiant du poste (rôle) occupé par l'utilisateur dans la société.<br>**Cette propriété est disponible en lecture seulement**. |
| IDSociété | Chaîne de caractères | Identifiant de la société.<br>**Cette propriété est disponible en lecture seulement**. |
| IDUtilisateur | Chaîne de caractères | Identifiant de l'utilisateur.<br>**Cette propriété est disponible en lecture seulement**. |
| Langue | Chaîne de caractères | Langue de l'utilisateur. Ce paramètre définit la langue utilisée pour les libellés des champs dans l'application. Les deux premiers caractères représentent le code ISO de la langue. Ces caractères sont suivis d'un caractère underscore (_) puis du code ISO du pays.<br><br>- Exemple pour les USA : "en_US".<br><br>- Exemple pour le Canada Français : "fr_CA".<br><br><br>**Cette propriété est disponible en lecture seulement**. |
| Login | Chaîne de caractères | Login de l'utilisateur.<br>**Cette propriété est disponible en lecture seulement**. |
| ModeAccessibilité | Booléen | <br><br>- <u><u><u><u>Vrai</u></u></u></u> si le mode d'accessibilité pour les utilisateurs malvoyants est activé, <br><br>- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.<br><br><br>**Cette propriété est disponible en lecture seulement**. |
| MultiDeviseSociété | Booléen | <br><br>- <u><u><u><u>Vrai</u></u></u></u> si la société utilise plusieurs devises,<br><br>- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.<br><br><br>**Cette propriété est disponible en lecture seulement**. |
| NomComplet | Chaîne de caractères | Nom complet de l'utilisateur.<br>**Cette propriété est disponible en lecture seulement**. |
| NomSociété | Chaîne de caractères | Nom de l'organisation de l'utilisateur ou de sa société.<br>**Cette propriété est disponible en lecture seulement**. |
| ParamètresRégionaux | Chaîne de caractères | Langue utilisée pour définir les paramètres régionaux (format des dates, symboles des monnaies). Les deux premiers caractères représentent le code ISO de la langue. Ces caractères sont suivis d'un caractère underscore (_) puis du code ISO du pays.<br><br>- Exemple pour les USA : "en_US".<br><br>- Exemple pour le Canada Français : "fr_CA".<br><br><br>**Cette propriété est disponible en lecture seulement**.<br><br>**Attention** : Avant la version 170078, cette propriété s'appelait "Région". Elle a été renommée à partir de la version 170078. |
| Région | Chaîne de caractères | **Propriété conservée par compatibilité. A partir de la version 170078, il est conseillé d'utiliser la propriété ParamètresRégionaux**.<br><br>Langue utilisée pour définir les paramètres régionaux (format des dates, symboles des monnaies). Les deux premiers caractères représentent le code ISO de la langue. Ces caractères sont suivis d'un caractère underscore (_) puis du code ISO du pays.<br><br>- Exemple pour les USA : "en_US".<br><br>- Exemple pour le Canada Français : "fr_CA".<br><br><br>**Cette propriété est disponible en lecture seulement**. |
| SymboleDevise | Chaîne de caractères | Symbole de la devise utilisée.<br>**Cette propriété est disponible en lecture seulement**. |
| ThèmeUI | Chaîne de caractères | Thème utilisé pour l'interface. Peut correspondre à :<br><br>- "Theme1" si l'utilisateur utilise l'interface nommée "Salesforce Classic".<br><br>- "Theme2" si l'utilisateur utilise l'interface nommée "Salesforce".<br><br><br>**Cette propriété est disponible en lecture seulement**.<br><br>Remarque : Dans les versions précédentes, cette propriété avait pour nom **ThèmeIHM**. |
| TypeUtilisateur | Chaîne de caractères | Type de licence utilisateur attribuée au profil associé à l'utilisateur.<br>**Cette propriété est disponible en lecture seulement**. |

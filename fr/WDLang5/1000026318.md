


## &lt;Variable htmlDocument&gt;.ChercheElémentParNom (Fonction)

***En anglais : &lt;htmlDocument variable&gt;.FindElementByName***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Cherche les éléments d'un document HTML dont l'attribut "name" correspond à une valeur spécifique.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
d est un htmlDocument = HTMLOuvre("https://www.pcsoft.fr", depuisURL)
e est un tableau de htmlNoeud <- d.ChercheElémentParNom("Couleur1")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Elément à manipuler>.ChercheElémentParNom(<Nom>)`
---

**`<Résultat> : (Tableau)`**

Tableau d'éléments de type [htmlNoeud](../WDLang5/1000026081.md) correspondant à la recherche.

**`<Elément à manipuler> : (Variable de type htmlDocument)`**

Nom de la variable de type [htmlDocument](../WDLang5/1000025910.md) dans laquelle la recherche doit être effectuée.

**`<Nom> : (Chaîne de caractères)`**

Valeur de l'attribut "name" recherché.

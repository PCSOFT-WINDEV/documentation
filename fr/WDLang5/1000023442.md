
## XML : Manipulations de fichiers de plusieurs gigaoctets
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000093"></a>
Depuis la version 23, un parseur XML est intégré dans WINDEV, WEBDEV et WINDEV Mobile afin de manipuler les fichiers XML de plusieurs Gigas : par exemple, données de Open Street Map ou de Open Data France.

La vitesse de lecture et la consommation mémoire sont nettement améliorées sur les fichiers XML de taille importante. En effet, le fichier n'est pas chargé en mémoire : il est simplement parcouru. 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000117"></a>


### Méthode
<a name="methode_ELTPARAGRAPHE000018"></a>Pour lire un fichier XML de taille importante avec le parseur XML : 

1. Ouvrez le fichier XML à lire avec la fonction [XMLOuvreLecteur](../WDLang5/1000023396.md). Cette fonction permet d'initialiser une variable de type [xmlLecteur](../WDLang5/1000023392.md). 
	Par exemple : 
	
	```wl
	Lecteur est un xmlLecteur = XMLOuvreLecteur("MonFichier.xml")
	```


2. Parcourez le XML en utilisant une des syntaxes de [POUR TOUT](../Motscles/1510022.md).

3. Les propriétés de la variable [xmlLecteur](../WDLang5/1000023392.md) permettent de connaître les caractéristiques de l'élément lu. 





### Exemples
<a name="exemples_ELTPARAGRAPHE000043"></a>L'exemple ci-dessous présente un fichier xml didactique et différents modes de parcours avec la liste des éléments parcourus. 

```xml
< ?xml version=”1.0”?>
<aa>
	<bb>
		Text1
        
		<cc>
		</cc>
	</bb>
	<bb>
		<cc>
		</cc>
	</bb>
</aa>
```



```wl
// Parcours sur un niveau
// Lecteur est la variable correspondant au fichier XML
POUR TOUT Lecteur
	// Lecture des balises début aa et fin aa
	POUR TOUT Lecteur
		// Lecture des balises début bb, fin bb, début bb, fin bb
		POUR TOUT Lecteur
		// Lecture de Text1 et des balises début cc, fin cc
		// début cc, fin cc
		FIN
	FIN
FIN
```



```wl
// Lecteur est la variable correspondant au fichier XML
POUR TOUT Lecteur EN PROFONDEUR
	// Lecture des balises début aa, début bb, text1, 
	// début cc, fin cc, fin bb, début bb,
	// début cc, fin cc, fin bb, fin aa
FIN
```

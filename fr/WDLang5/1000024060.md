
## CBVersImage (Fonction)

***En anglais : BCToImage***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Génère l'image d'un code-barres.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Génère l'image du code-barres
MonCodeBarres est un CodeBarres
MonCodeBarres.Contenu = "012345678901"
MonCodeBarres.TypeCodeBarres = CB_EAN13
ImageCodeBarres est une Image = CBVersImage(MonCodeBarres)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = CBVersImage(<Code-barres> [, <Largeur maximale> [, <Hauteur maximale> [, <Couleur de fond> [, <Couleur du code-barres>]]]])`
---

**`<Résultat> : (Variable de type Image)`**

Variable de type [Image](../WDLang1/1000019650.md) correspondant au code-barres.

**`<Code-barres> : (Variable de type CodeBarres)`**

Nom de la variable de type [CodeBarres](../WDLang5/1000019451.md) correspondant à l'image à générer.

**`<Largeur maximale> : (Entier optionnel)`**

Largeur maximale de l'image. 

- Si la largeur maximale n'est pas précisée ou correspond à la constante *cbLargeurDéfaut*, la largeur sera la largeur minimale nécessaire à l'affichage du code-barres. 

- Si la largeur précisée est inférieure à la largeur minimale nécessaire à l'affichage du code-barres, une erreur sera générée.




**`<Hauteur maximale> : (Entier optionnel)`**

Hauteur maximale de l'image. 

- Si la hauteur maximale n'est pas précisée ou correspond à la constante *cbHauteurDéfaut*, la hauteur sera la hauteur minimale nécessaire à l'affichage du code-barres. 

- Si la hauteur précisée est inférieure à la hauteur minimale nécessaire à l'affichage du code-barres, une erreur sera générée.




**`<Couleur de fond> : (Entier optionnel)`**

Couleur de fond de l'image. Cette couleur peut correspondre à :

- une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)),

- une couleur TSL (obtenue avec la fonction [TSL](../WDLang1/3029057.md)),

- [une couleur prédéfinie du WLangage](../WDLang5/3010002.md),

- une variable de type [Couleur](../WDLang1/1000019661.md). 


Si la couleur de fond n'est pas précisée ou correspond à la constante *Transparent*, le fond sera transparent.

**`<Couleur du code-barres> : (Entier optionnel)`**

Couleur du code-barres. Cette couleur peut correspondre à :

- une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)),

- une couleur TSL (obtenue avec la fonction [TSL](../WDLang1/3029057.md)),

- [une couleur prédéfinie du WLangage](../WDLang5/3010002.md),

- une variable de type [Couleur](../WDLang1/1000019661.md). 


Si la couleur du code-barres n'est pas précisée, le code-barres sera noir.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Taille de l'image
<a name="taille_image_ELTPARAGRAPHE000100"></a>L'image aura la taille occupée par le dessin du code-barres. Par exemple, pour un code-barres de type QR-Code carré, si la largeur indiquée est suffisante et si la hauteur demandée est supérieure à la largeur, l'image renvoyée sera carrée (la hauteur sera égale à la largeur demandée).

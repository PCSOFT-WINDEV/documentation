
## gglDocument (Type de variable)

***En anglais : gglDocument ***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **gglDocument** permet de manipuler un document Google. Ce document a été créé par la fonction [GglEnvoieDocument](../WDLang5/1000017368.md). Les caractéristiques de ce document peuvent être connues à l'aide de différentes propriétés WLangage.  

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connexion
Cnx est un gglConnexion
Cnx.Email = "balthazar@gmail.com"
Cnx.MotDePasse = "azerty"
Cnx.NomApplication = "MonAppli-03"
SI GglConnecte(Cnx) = Faux ALORS
	Erreur(ErreurInfo())
SINON
	// Récupération du tableau des documents
	tabDocs est un tableau de gglDocument = GglListeDocument(Cnx)
	// Parcours des documents
	Doc est un gglDocument
	POUR TOUT Doc DE tabDocs
		Trace(Doc.Titre)
	FIN
FIN
```

## Remarques




### Propriétés spécifiques à la description des variables de type gglDocument
<a name="proprietes_specifiques_description_des_variables_type_ggldocument_ELTPARAGRAPHE000035"></a>Les propriétés suivantes peuvent être utilisées pour manipuler un document Google (ces propriétés sont disponibles en lecture uniquement) :

| Nom de la propriété | Type manipulé | Effet |
| --- | --- | --- |
| DateMiseAJour | DateHeure | Date de la dernière mise à jour du document.<br><br>**Remarque** : La date est une date UTC (en temps universel). |
| Identifiant | Chaîne de caractères | Identifiant du document. |
| Suivi | Booléen | <br><br>- <u><u><u><u>Vrai</u></u></u></u> si le document est marqué comme "Suivi",<br><br>- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.<br><br><br> |
| Titre | Chaîne de caractères | Titre du document. |
| TypeMIME | Chaîne de caractères | Type MIME du document. <br><br>Remarque : Pour plus de détails sur les types MIME autorisés, consultez <br><br>- Types standards : [http://fr.wikipedia.org/wiki/Type_MIME](http://fr.wikipedia.org/wiki/Type_MIME)<br><br>- Types spécifiques Google Drive : [https://developers.google.com/drive/web/mime-types](https://developers.google.com/drive/web/mime-types)<br><br><br> |

***En anglais : GglGetLicense***

<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le texte de la licence d'utilisation des librairies Google.

**Attention** : 

- Depuis le début de l'année 2018, Google ne permet plus de récupérer le texte de la licence Google des librairies utilisées. Cette licence d'utilisation des librairies Google ne doit plus être affichée dans l'application. Cette licence est accessible par l'utilisateur final dans les paramètres du téléphone (Paramètres -> Google -> Menu "Licences Open Source"). 

- Cette fonction ne renvoie plus la licence Google quelle que soit la version de WINDEV Mobile utilisée. 



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
sLicence est une chaîne 
sLicence = gglRécupèreLicence(gglPlayServices)
SI sLicence <> "" ALORS
	Info(sLicence)
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = GglRécupèreLicence(<Librairie>)`
---

**`<Résultat> : (Chaîne de caractères)`**

Chaîne vide (""). 

**`<Librairie> : (Constante de type Entier)`**

Librairie Google pour laquelle la licence doit être récupérée : 


|   |   |
| --- | --- |
| *gglPlayServices* | Librairie Google Play Services. Cette librairie est utilisée sous Android pour les champs Carte basés sur l'API Google Maps v2. |

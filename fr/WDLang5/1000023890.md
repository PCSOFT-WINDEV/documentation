


## &lt;Variable xmlNoeud&gt;.ChercheNamespaceParURI (Fonction)

***En anglais : &lt;xmlNode variable&gt;.FindNamespaceByURI***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Recherche un namespace (espace de nommage) XML à partir de son URI (Universal Resource Identifier) dans un noeud et dans les parents de ce noeud.






<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
gcDocumentXML est un xmlDocument
gcDocumentXML = XMLOuvre("c:\temp\fichier.xml", depuisFichier)

cNoeud est un xmlNoeud = gcDocumentXML..NoeudRacine[1]
cNamespace est un xmlNamespace = cNoeud.ChercheNamespaceParURI("http://schemas.xmlsoap.org/wsdl/")

SI ErreurDétectée = Vrai ALORS
	Info("Le namespace n'a pas été trouvé")
SINON
	Info("Namespace trouvé : ", cNamespace.Nom)
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Noeud de départ>.ChercheNamespaceParURI(<URI>)`
---

**`<Résultat> : (Variable de type xmlNamespace)`**

Variable de type [xmlNamespace](../WDLang5/1000018803.md) correspondant au namespace de l'URI recherché.

**`<Noeud de départ> : (Variable de type xmlNoeud)`**

Nom de la variable de type [xmlNoeud](../WDLang5/1000018786.md) correspondant au noeud du document XML à partir duquel le namespace doit être recherché.

**`<URI> : (Chaîne de caractères Unicode)`**

URI du namespace à rechercher.

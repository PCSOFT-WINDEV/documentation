
## iImpressionAbandonnée (Fonction)

***En anglais : iDocumentCanceled***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de savoir si l'utilisateur a demandé l'abandon de l'impression du document en cours.

**Remarque** : Si l'utilisateur abandonne une impression en cours, les fonctions d'impression n'ont aucun effet jusqu'au prochain appel à la fonction [iFinImprime](../WDLang5/3046053.md). Le code WLangage continue cependant de s'exécuter.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple
<a class="notetitle" target="_blank" href="$DOC$=1000003046006&name=iimpressionabandonnee_fonction&product=WD">Voir des exemples supplémentaires</a>

```wl
SI iImpressionAbandonnée() = Vrai ALORS
	Info("Vous avez annulé l'impression en cours")
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = iImpressionAbandonnée()`
---

**`<Résultat> : (Booléen)`**

<u><u><u><u>Vrai</u></u></u></u> si l'utilisateur :

- a fermé le visualisateur de rapports avant la fin des impressions,

- a cliqué sur le bouton ***Annuler*** de la fenêtre d'impression en cours.


## &lt;Variable xlsDocument&gt;.RécupèreFusion (Fonction)

***En anglais : &lt;xlsDocument variable&gt;.GetMerge***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de savoir si une cellule est fusionnée avec d'autres cellules et d'obtenir la plage des cellules fusionnées correspondante.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonDocumentXLS est un xlsDocument
 
sFusion est une chaîne = MonDocumentXLS.RécupèreFusion("B1")
SI sFusion = "" ALORS
	Info("Aucune fusion")
SINON
	sMin est chaîne = sFusion.ExtraitChaîne(1, ":")
	sMax est chaîne = sFusion.ExtraitChaîne(2, ":")
	Info("Fusion entre " + sMin + " et " +  sMax)
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Récupérer la plage de cellules fusionnées contenant une cellule identifiée par sa ligne et colonne

`<Résultat> = <Document Excel>.RécupèreFusion(<Ligne cellule> , <Colonne cellule>)`
---

**`<Résultat> : (Chaîne de caractères)`**



- Chaîne vide ("") si la cellule spécifiée n'est pas fusionnée,

- Plage de cellules fusionnées au format : "&lt;Nom cellule haut gauche&gt;:&lt;Nom cellule bas droite&gt;". 
	Par exemple : "A2:C5".




**`<Document Excel> : (Variable de type xlsDocument)`**

Nom de la variable de type [xlsDocument](../WDLang5/1000017464.md) à utiliser.

**`<Ligne cellule> : (Entier)`**

Numéro de la ligne de la cellule à manipuler.

**`<Colonne cellule> : (Entier)`**

Numéro de la colonne de la cellule à manipuler.


<a name="SYNTAXE2"></a>

### Récupérer la plage de cellules fusionnées contenant une cellule identifiée par son nom

`<Résultat> = <Document Excel>.RécupèreFusion(<Cellule>)`
---

**`<Résultat> : (Chaîne de caractères)`**



- Chaîne vide ("") si la cellule spécifiée n'est pas fusionnée,

- Plage de cellules fusionnées au format : "&lt;Nom cellule haut gauche&gt;:&lt;Nom cellule bas droite&gt;".
	Par exemple : "A2:C5"




**`<Document Excel> : (Variable de type xlsDocument)`**

Nom de la variable de type [xlsDocument](../WDLang5/1000017464.md) à utiliser.

**`<Cellule> : (Chaîne de caractères)`**

Nom de cellule à manipuler. Par exemple : "A4".

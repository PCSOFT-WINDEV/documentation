


## XMLOuvreLecteur (Fonction)

***En anglais : XMLOpenReader***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Crée un lecteur permettant de lire un document XML sans le charger en mémoire.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple
<a class="notetitle" target="_blank" href="$DOC$=1000023780&name=xmlouvrelecteur_fonction&product=WD">Voir des exemples supplémentaires</a>

```wl
MonNoeud est un xmlNoeud
MonNoeud..Type = XMLBalise
MonNoeud..Nom = "MonNoeud"
MonNoeud..Texte = "Le texte du noeud"

Lecteur est un xmlLecteur = xmlOuvreLecteur("fichier.xml")

POUR TOUT Lecteur EN PROFONDEUR
	Trace(Lecteur.Type)
FIN

VariableRAZ(Lecteur) // Fermeture du fichier

Lecteur2 est un xmlLecteur = xmlOuvreLecteur("fichier.xml")

POUR TOUT x DE Lecteur2
	SI Lecteur2.Type = xmlLecteurBaliseDébut _OU_ Lecteur2.Type = xmlLecteurBaliseVide ALORS
		SI Lecteur2.Nom = "config" ALORS
			POUR TOUT y DE x
				Trace(x.Nom)
			FIN
		FIN
	FIN
FIN

VariableRAZ(Lecteur2) // Fermeture du fichier
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = XMLOuvreLecteur(<Fichier>)`
---

**`<Résultat> : (Variable de type xmlLecteur)`**

Variable de type [xmlLecteur](../WDLang5/1000023392.md) permettant de lire un document XML sans le charger en mémoire. 

**`<Fichier> : (Chaîne de caractères)`**

Chemin du fichier contenant le document XML à lire.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Cette fonction permet de lire des fichiers XML de taille importante. Pour lire des petits fichiers XML, utilisez la fonction [XMLOuvre](../WDLang5/1000018792.md). 

- Fermeture du lecteur : La fonction [VariableRAZ](../WDLang1/3013069.md) appliquée à la variable de type [xmlLecteur](../WDLang5/1000023392.md) permet de fermer le fichier XML manipulé (voir exemple).




## xlsCoordonnéesVersNom (Fonction)

***En anglais : xlsCoordinatesToName***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Récupère le nom d'une cellule d'une feuille XLS ou d'un champ Tableur en fonction de ses coordonnées. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
NomCellule est une chaîne = xlsCoordonnéesVersNom(1,1)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = xlsCoordonnéesVersNom(<Colonne> , <Ligne>)`
---

**`<Résultat> : (Chaîne de caractères)`**

Nom de la cellule. Ce nom est du type "A1". 

**`<Colonne> : (Entier)`**

Numéro de la colonne identifiant la cellule. 

**`<Ligne> : (Entier)`**

Numéro de la ligne identifiant la cellule.


## xlsLigne (Type de variable)

***En anglais : xlsRow ***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **xlsLigne** permet de décrire et de modifier une ligne d'un document Excel. Les caractéristiques de cette ligne peuvent être définies et modifiées à l'aide de différentes propriétés WLangage.

Ce type de variable est utilisé par le type [xlsDocument](../WDLang5/1000017464.md). 

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).








<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ouverture d'un fichier Excel
sFichierExcel est une chaîne
sFichierExcel = fSélecteur("", "", "Sélectionnez un fichier Excel...", "Excel" + TAB + ...
	"*.xlsx" + RC + "Tous les fichiers (*.*)" + TAB + "*.*", "*.xlsx")
SI sFichierExcel = "" ALORS RETOUR

// Ouverture du document Excel
MonDocument est un xlsDocument
MonDocument = xlsOuvre(sFichierExcel)

// Police de caractères
MaPolice est une Police
MaPolice.Nom = "Arial"
MaPolice.Taille = 12
MaPolice.Gras = Vrai
MaPolice.Couleur = Blanc

// Cellule par défaut
MaCelluleDéfaut est un xlsCellule
MaCelluleDéfaut.CouleurFond = VertFoncé
MaCelluleDéfaut.Police = MaPolice

// Variable xlsLigne
MaLigne est un xlsLigne
MaLigne..Hauteur = 200
MaLigne..CelluleDéfaut = MaCelluleDéfaut

// Toutes les cellules de la ligne 2 ont 
// les caractéristiques de la cellule (couleur, taille, police, ...)
MonDocument..Ligne[2] = MaLigne

// Sauvegarde les modifications.
xlsSauve(MonDocument)
xlsFerme(MonDocument)
```

## Remarques




### Propriétés spécifiques à la description des variables de type xlsLigne
<a name="proprietes_specifiques_description_des_variables_type_xlsligne_ELTPARAGRAPHE000050"></a>Les propriétés suivantes peuvent être utilisées pour manipuler une variable de type **xlsLigne** :

| Nom | Type manipulé | Effet |
| --- | --- | --- |
| Cellule | Tableau de [xlsCellule](../WDLang5/1000017472.md) | Cellules de la colonne. |
| CelluleDéfaut | xlsCellule | Le style par défaut de la ligne correspondra au style de la cellule spécifiée. |
| Hauteur | Réel double | Hauteur de la ligne en pixels (entre 0 et 545 pixels). Si cette propriété est initialisée à -1, la hauteur par défaut des lignes du document Excel sera utilisée. |

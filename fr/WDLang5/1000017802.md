
## rssSaisieTexte (Type de variable)

***En anglais : rssTextInput ***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **rssSaisieTexte** permet de décrire la zone de saisie d'un flux RSS. Les caractéristiques de ce type de variable peuvent être définies et modifiées à l'aide de différentes propriétés WLangage. 

Ce type de variable est utilisé par le [type rssFlux](../WDLang5/1000017795.md).

**Notes** :

- Le plus souvent, la déclaration d'une variable de type **rssSaisieTexte** n'est pas nécessaire. En effet, les propriétés des variables de type [rssFlux](../WDLang5/1000017795.md) permettent de manipuler directement les propriétés du type **rssSaisieTexte** (notamment la propriété **SaisieTexte**).

- Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).

<a name="proprietes_specifiques_description_des_variables_type_rsssaisietexte_ELTPARAGRAPHE000058"></a>Les propriétés suivantes peuvent être utilisées pour manipuler une variable de type **rssSaisieTexte** :

| Nom de la propriété | Type manipulé | Effet |
| --- | --- | --- |
| Description | Chaîne de caractères | Libellé du champ de saisie. |
| Lien | Chaîne de caractères | URL vers laquelle les données seront envoyées. |
| Nom | Chaîne de caractères | Nom utilisé dans le POST pour transmettre la valeur. |
| Titre | Chaîne de caractères | Titre de validation de la zone de saisie. |




## xlsFerme (Fonction)

***En anglais : xlsClose***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ferme un fichier XLS.

<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple
<a class="notetitle" target="_blank" href="$DOC$=1000003080015&name=xlsferme_fonction&product=WD">Voir des exemples supplémentaires</a>

```wl
MonFichierXLS est un xlsDocument

// Ouverture d'un fichier XLS
MonFichierXLS = xlsOuvre("C:\MesRépertoires\Fichier.XLS")
...
// Fermeture du fichier XLS
xlsFerme(MonFichierXLS)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Syntaxe utilisant le type xlsDocument

`xlsFerme(<Document XLS>)`
---

**`<Document XLS> : (Variable de type xlsDocument)`**

Nom de la variable de type [xlsDocument](../WDLang5/1000017464.md) correspondant au fichier XLS à fermer. Le fichier physique associé à cette variable (s'il existe) est fermé. Les modifications réalisées dans le fichier ne sont pas sauvées avant sa fermeture (utilisez la fonction [xlsSauve](../WDLang5/1000018057.md) si nécessaire).  


<a name="SYNTAXE2"></a>

### Syntaxe conservée par compatibilité avec la version 12

`xlsFerme(<Identifiant du fichier XLS>)`
---

**`<Identifiant du fichier XLS> : (Entier)`**

Identifiant du fichier XLS à fermer, renvoyé par la fonction [xlsOuvre](../WDLang5/3080017.md).

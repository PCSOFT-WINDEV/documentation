


## SFRechercheModifié (Fonction)

***En anglais : SFSeekModified***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Recherche et récupère les enregistrements (objets Salesforce) modifiés pendant la période spécifiée.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connexion à Salesforce
Cnx est une sfConnexion
Cnx..Login = "balthazar@gmail.com"
Cnx..MotDePasse = "azerty" + "XXB12VCZ54"
SI SFConnecte(Cnx) = Faux ALORS
	Erreur("La connexion a échoué")
	RETOUR
FIN

ResRecupModif est un sfRésultatRechercheModifié
ResRecupModif = SFRechercheModifié(Cnx, "Mileage__c", "20090101", "20090131")

Identifiant est une chaîne
POUR TOUTE CHAINE Identifiant DE ResRecupModif..Id SEPAREE PAR RC
		Trace("--- ID : " + Identifiant)
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = SFRechercheModifié(<Connexion Salesforce> , <Fichier Salesforce> , <Date de début> , <Date de fin>)`
---

**`<Résultat> : (Variable de type sfRésultatRechercheModifié)`**

Nom de la variable de type [sfRésultatRechercheModifié](../WDLang5/1000018561.md) contenant les données résultat.

**`<Connexion Salesforce> : (Variable de type sfConnexion)`**

Nom de la variable de type [sfConnexion](../WDLang5/1000017579.md) à utiliser.

**`<Fichier Salesforce> : (Chaîne de caractères)`**

Nom du fichier Salesforce à traiter. Ce nom doit correspondre au nom défini sous l'interface Salesforce comme le "Nom API". Dans le cas d'un fichier personnalisé, ce nom correspond au nom du fichier suivi par 2 underscores puis par la lettre "C". Par exemple : "MonCompte__C".

**`<Date de début> : (DateHeure)`**

Date de début de la période pour laquelle le nombre d'objets modifiés est recherché. Ce paramètre doit être au format UTC (utilisez si nécessaire la fonction [DateHeureLocaleVersUTC](../WDLang1/3027036.md) pour réaliser la conversion). 

**`<Date de fin> : (DateHeure)`**

Date de fin de la période pour laquelle le nombre d'objets modifiés est recherché. Ce paramètre doit être au format UTC (utilisez si nécessaire la fonction [DateHeureLocaleVersUTC](../WDLang1/3027036.md) pour réaliser la conversion).

## Remarques


### Cas d'erreurs possibles : 
<a name="cas_erreurs_possibles_ELTPARAGRAPHE000090"></a>

- La connexion Internet n'est pas valide.

- L'authentification n'a pas pu être réalisée correctement.

- La réponse XML du serveur n'est pas lisible.




Pour savoir si une erreur est survenue, utilisez la variable [ErreurDétectée](../WDLang1/3087001.md).

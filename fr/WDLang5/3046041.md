
## iFermeEtat (Fonction)

***En anglais : iCloseReport***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Arrête immédiatement l'impression de l'état en cours. Le code WLangage situé après l'appel à la fonction est exécuté. Cette fonction doit être utilisée dans un des traitements de l'état.

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Si l'état est imprimé dans le visualisateur de rapports, la fenêtre d'aperçu est automatiquement fermée.



**Remarque** : Cette fonction peut uniquement être utilisée dans les traitements des états créés avec l'éditeur d'états.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Fin de l'impression 
SI MoiMême..Source.NomClient = "MOULIN" ALORS
	iFermeEtat()
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`iFermeEtat()`
---



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Pour ne pas exécuter le code WLangage présent après la fonction **iFermeEtat**, utilisez par exemple le mot-clé [RETOUR](../Motscles/1510009.md).

Exemple :


```wl
SI TraitementFini() = Vrai ALORS
	iFermeEtat()
	RETOUR
FIN
Info("Coucou")
```

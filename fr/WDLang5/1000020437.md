


## &lt;Variable gglConnexion&gt;.RécupèreDocument (Fonction)

***En anglais : &lt;gglConnection variable&gt;.GetDocument***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Télécharge un document depuis le service Google Docs.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connexion
Cnx est un gglConnexion
...
SI Cnx.Connecte() = Faux ALORS
	Erreur(ErreurInfo())
FIN
...
tabDoc est un tableau de gglDocument 
tabDoc = Cnx.ListeDocument()
Cnx.RécupèreDocument(tabDoc[1], "c:\temp\document.pdf")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Connexion Google>.RécupèreDocument(<Document à télécharger> , <Chemin destination>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le document a été récupéré, 

- <u><u><u><u>Faux</u></u></u></u> en cas de problème. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Connexion Google> : (Variable de type gglConnexion)`**

Nom de la variable de type [gglConnexion](../WDLang5/1000018009.md) à utiliser. Cette connexion a été validée avec la fonction [&lt;Variable gglConnexion&gt;.Connecte](../WDLang5/1000021455.md).

**`<Document à télécharger> : (Variable de type gglDocument)`**

Variable de type [gglDocument](../WDLang5/1000018001.md) correspondant au document à télécharger.

**`<Chemin destination> : (Chaîne de caractères Unicode)`**

Nom et chemin complet correspondant au fichier à créer à partir du document récupéré. Si l'extension du fichier est précisée, le fichier sera converti dans ce format lors de son téléchargement. 
Par défaut, si l'extension n'est pas spécifiée, le format du fichier original est conservé.

## Remarques


### Formats de conversion possibles selon le format d'origine
<a name="formats_conversion_possibles_selon_format_origine_ELTPARAGRAPHE000198"></a>Les formats de sortie supportés dépendent du type du fichier d'origine. Pour plus de détails, consultez la [documentation Google](http://code.google.com/intl/fr-FR/apis/documents/docs/3.0/developers_guide_protocol.html#DownloadingDocsAndFiles). 
Quelques exemples : 

- Les formats valides pour un document texte sont les suivants : 

	- doc

	- html

	- jpeg

	- odt 

	- pdf 

	- png

	- rtf

	- txt

	- zip. Dans ce cas, l'archive Zip contient les images (si nécessaire) utilisées dans le document aussi bien que les fichiers HTML contenant le texte du document. 




- Les formats valides pour des dessins sont les suivants : 

	- jpeg

	- pdf

	- png

	- svg




- Les formats valides pour les présentations sont les suivants : 

	- pdf

	- png

	- ppt

	- txt




- Les formats valides pour les feuilles de calcul sont les suivants :  

	- xls

	- csv : seule la première feuille de calcul est renvoyée au format CSV. 

	- pdf

	- ods

	- tsv : seule la première feuille de calcul est renvoyée au format TSV.

	- html

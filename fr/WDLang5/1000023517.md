
## iImprimeXLS (Fonction)

***En anglais : iPrintXLS***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Imprime la feuille courante d'un document au format "xlsx" ou "xls". 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonXLSX est une chaîne = "c:\temp\Comptes.xlsx"
iDestination(iVisualisateur)
iImprimeXLS(MonXLSX)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Imprimer le contenu d'un champ Tableur ou d'une variable xlsDocument

`<Résultat> = iImprimeXLS(<Document>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le document a été envoyé au spooler d'impression, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour connaître le libellé de l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.




**`<Document> : (Type correspondant)`**

Document XLS ou XLSX à imprimer. Ce document correspond :

- à une variable de type [xlsDocument](../WDLang5/1000017464.md).

- au chemin d'un fichier de type Tableur (fichier XLS ou XLSX).

- à un mémo HFSQL correspondant à un document de type  XLS ou XLSX.





<a name="SYNTAXE2"></a>

### Imprimer le contenu d'un fichier "xlsx" ou "xls"

`<Résultat> = iImprimeXLS(<Chemin> [, <Mot de passe>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le document a été envoyé au spooler d'impression, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour connaître le libellé de l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.




**`<Chemin> : (Chaîne de caractères)`**

Chemin d'un fichier de type Tableur (fichier XLS ou XLSX).

**`<Mot de passe> : (Chaîne de caractères optionnelle)`**

Mot de passe d'ouverture du fichier XLS ou XLSX à imprimer. 





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Seule la feuille courante du document XLS ou XLSX est imprimée. Pour imprimer d'autres feuilles, il est nécessaire d'utiliser une variable de type [xlsDocument](../WDLang5/1000017464.md) et de changer le numéro de la feuille courante via la propriété **Feuille**.

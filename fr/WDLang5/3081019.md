


## XMLRacine (Fonction)

***En anglais : XMLRoot***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Positionne sur la balise racine d'un document XML.








<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI XMLDocument("MonDocXML", CodeXML) = Vrai ALORS
	// Recherche des liens 
	XMLRecherche("MonDocXML", "a", XMLBalise)
	TANTQUE XMLTrouve("MonDocXML") = Vrai
		// Traitement
		// ...
		// Elément suivant dans la recherche
		XMLSuivant("MonDocXML")
	FIN
	// Positionnement sur la racine
	XMLRacine("MonDocXML")
	// Nouvelle recherche
	XMLRecherche("MonDocXML", "Lien", XMLBalise)
	TANTQUE XMLTrouve("MonDocXML") = Vrai
		// Traitement
		// ...
		// Elément suivant dans la recherche
		XMLSuivant("MonDocXML")
	FIN
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = XMLRacine(<Document XML>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si un élément a été trouvé, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. En cas d'erreur : 

	- la position courante reste inchangée. 

	- la variable [ErreurDétectée](../WDLang1/3087001.md) est à <u><u><u><u>Vrai</u></u></u></u>. Pour obtenir plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).







**`<Document XML> : (Chaîne de caractères)`**

Nom du document XML manipulé. Ce document contient le code XML à analyser et a été créé avec la fonction [XMLDocument](../WDLang5/3081008.md).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Manipulation des éléments dans le code XML
<a name="manipulation_des_elements_dans_code_xml_ELTPARAGRAPHE000090"></a>Après la fonction **XMLRacine**, l'élément en cours peut être manipulé avec les fonctions [XMLNomElément](../WDLang5/3081026.md), [XMLNomParent](../WDLang5/3081035.md), [XMLTypeElément](../WDLang5/3081031.md) et [XMLDonnée](../WDLang5/3081004.md).
<a name="NOTE0_2"></a>


### Utilisation de la fonction XMLRacine lors d'une recherche
<a name="utilisation_fonction_docparampagetitleshort_lors_une_recherche_ELTPARAGRAPHE000112"></a>Si une recherche est en cours (fonction [XMLRecherche](../WDLang5/3081025.md)), la fonction **XMLRacine** annule cette recherche.
<a name="NOTE0_3"></a>


### Fonctions XML et threads
<a name="fonctions_xml_threads_ELTPARAGRAPHE000263"></a>Si votre application utilise des threads, le document XML est partagé entre tous ces threads. Pour plus de détails sur la gestion des threads, consultez [Gestion des threads](../WDLang1/3077026.md).
Si la position courante dans un document XML est modifiée dans un thread, la position courante dans ce document XML est modifiée pour tous les threads.


## iSautePage (Fonction)

***En anglais : iSkipPage***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Génère un saut de page : le code correspondant est envoyé dans la file d'impression de l'imprimante. Le curseur d'impression est positionné aux coordonnées (0,0) de la page suivante.

L'impression n'est pas lancée (seule la fonction [iFinImprime](../WDLang5/3046053.md) permet de lancer l'impression).

**Remarque** : Il est fortement déconseillé d'utiliser cette fonction dans les états créés avec l'éditeur d'états. Utilisez à la place la fonction [iTerminePage](../WDLang5/3046038.md).




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Impression d'un texte
iImprime("Bonjour")
// Saut de page
iSautePage()
// Impression d'un texte
iImprime("Nous sommes le" + DateSys())
// Lancement de l'impression
iFinImprime()
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`iSautePage()`

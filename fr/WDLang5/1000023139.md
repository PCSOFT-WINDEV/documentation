
## xlsMiseEnPage (Type de variable)

***En anglais : xlsLayout***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **xlsMiseEnPage** permet de définir toutes les caractéristiques de mise en page pour l'impression d'une feuille d'un document XLSX présente dans un champ Tableur ou dans une variable de type [xlsDocument](../WDLang5/1000017464.md). Les caractéristiques de cette mise en page peuvent être définies et modifiées à l'aide de différentes propriétés WLangage. 

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Feuille en cours [SansMarges]
xMiseEnPage est un xlsMiseEnPage  = TBLR_ChampTableur.MiseEnPage
xMiseEnPage.MargeGauche = 0
xMiseEnPage.MargeDroite = 0
TBLR_ChampTableur.MiseEnPage = xMiseEnpage
```

## Remarques
<a name="NOTE0_1"></a>


### Propriétés spécifiques à la description des variables de type xlsMiseEnPage
<a name="proprietes_specifiques_description_des_variables_type_xlsmiseenpage_ELTPARAGRAPHE000032"></a>Les propriétés suivantes peuvent être utilisées pour manipuler une mise en page :

| Nom de la propriété | Type manipulé | Effet |
| --- | --- | --- |
| EntêteImprimé | Booléen | <br><br>- <u><u><u><u>Vrai</u></u></u></u> (valeur par défaut) pour imprimer l’entête de la feuille.<br><br>- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.<br><br><br> |
| EntêteVisible | Booléen | <br><br>- <u><u><u><u>Vrai</u></u></u></u> (valeur par défaut) pour visualiser l'entête de la feuille dans le champ Tableur.<br><br>- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.<br><br><br> |
| HautDePage | Réel | Hauteur du haut de page dans la feuille imprimée, exprimée en millimètres. |
| HauteurPage | Réel | Hauteur de la page imprimée, exprimée en millimètres. |
| LargeurPage | Réel | Largeur de la page imprimée, exprimée en millimètres. |
| MargeBasse | Réel double | Marge basse de la page imprimée, exprimée en millimètres. |
| MargeDroite | Réel | Marge droite de la page imprimée, exprimée en millimètres. |
| MargeGauche | Réel | Marge gauche de la page imprimée, exprimée en millimètres. |
| MargeHaute | Réel | Marge haute de la page imprimée, exprimée en millimètres. |
| ModeAjustement | Constante de type entier | <br><br>- *modeAjustementAucun* : Aucune mise à l'échelle n'est effectuée. <br><br>- *modeAjustementColonnes* : Toutes les colonnes sont ajustées à la largeur du champ. <br><br>- *modeAjustementLignes* : Toutes les lignes sont ajustées à la hauteur du champ. <br><br>- *modeAjustementLignesEtColonnes* : La feuille de calcul est ajustée aux dimensions du champ Tableur. <br><br><br> |
| OrientationPaysage | Booléen | <br><br>- <u><u><u><u>Vrai</u></u></u></u> (valeur par défaut) pour une orientation en mode paysage,<br><br>- <u><u><u><u>Faux</u></u></u></u> pour une orientation en mode portrait.<br><br><br> |
| PiedDePage | Réel | Hauteur du pied de page dans la page imprimée, exprimée en millimètres. |
| QuadrillageImprimé | Booléen | <br><br>- <u><u><u><u>Vrai</u></u></u></u> (valeur par défaut) pour imprimer le quadrillage autour des cellules,<br><br>- <u><u><u><u>Faux</u></u></u></u> pour imprimer sans quadrillage.<br><br><br> |
| QuadrillageVisible | Booléen | <br><br>- <u><u><u><u>Vrai</u></u></u></u> (valeur par défaut) pour visualiser dans le champ Tableur le quadrillage autour des cellules, <br><br>- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.<br><br><br> |

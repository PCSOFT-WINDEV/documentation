
## gglMarqueur (Type de variable)

***En anglais : gglMarker ***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **gglMarqueur** permet de définir les caractéristiques d'un marqueur à placer sur une carte. Les caractéristiques de ce marqueur peuvent être définies et modifiées à l'aide de différentes propriétés WLangage.

Ce type de variable est utilisé par le type [gglParamètreCarte](../WDLang5/1000017505.md). 

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).

## Remarques




### Propriétés spécifiques à la description des variables de type gglMarqueur
<a name="proprietes_specifiques_description_des_variables_type_gglmarqueur_ELTPARAGRAPHE000044"></a>Les propriétés suivantes peuvent être utilisées pour manipuler une variable de type **gglMarqueur** :

| Nom | Type manipulé | Effet |
| --- | --- | --- |
| Couleur | Constante de type entier | Couleur du marqueur :<br><br>- *Blanc* : Blanc<br><br>- *Noir* : Noir<br><br>- *BleuClair* : Bleu<br><br>- *GrisClair* : Gris<br><br>- *JauneClair* : Jaune<br><br>- *RougeClair* : Rouge<br><br>- *VertClair* : Vert<br><br><br> |
| Latitude | Réel | Latitude du marqueur. |
| Lettre | Caractère Ansi | Lettre du marqueur. Seuls les marqueurs de taille *gglGrand* ou *gglMoyen* peuvent accueillir une lettre. |
| Longitude | Réel | Longitude du marqueur. |
| Taille | Constante de type chaîne de caractères | Taille du marqueur. Les constantes disponibles sont : *gglMini*, *gglPetit*, *gglMoyen* ou *gglGrand* (*gglGrand* est utilisé par défaut). |

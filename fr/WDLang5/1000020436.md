


## &lt;Variable gglConnexion&gt;.ListeDocument (Fonction)

***En anglais : &lt;gglConnection variable&gt;.ListDocument***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Récupère la liste des documents disponibles sur le serveur Google pour l'utilisateur en cours.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connexion
Cnx est un gglConnexion
...
SI Cnx.Connecte() = Faux ALORS
	Erreur(ErreurInfo())
FIN
...

// Liste de tous les documents
tabDocuments est un tableau de 0 gglDocument dynamique

// Liste les documents contenant le texte "Test"
tabDocuments = Cnx.ListeDocument("fullText contains 'Test'")

// Affiche le titre de chaque document listé
Doc est un gglDocument dynamique
POUR TOUT Doc DE tabDocuments
	 Trace(Doc.Titre)
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE2"></a>

## Syntaxe

`<Résultat> = <Connexion Google>.ListeDocument([<Recherche à effectuer>])`
---

**`<Résultat> : (Tableau de variables de type gglDocument)`**

Tableau de variables de type [gglDocument](../WDLang5/1000018001.md) : chaque élément du tableau contient les caractéristiques d'un document présent sur le serveur Google.

**`<Connexion Google> : (Variable de type gglConnexion)`**

Nom de la variable de type [gglConnexion](../WDLang5/1000018009.md) à utiliser. Cette connexion a été validée avec la fonction [&lt;Variable gglConnexion&gt;.Connecte](../WDLang5/1000021455.md).

**`<Recherche à effectuer> : (Chaîne de caractères)`**

Formule de recherche. Pour connaître les paramètres utilisables, consultez la documentation de l'API "Drive" : [https://developers.google.com/drive/web/search-parameters](https://developers.google.com/drive/web/search-parameters)

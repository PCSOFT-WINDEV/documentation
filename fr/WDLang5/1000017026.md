
## iParamètreExport (Fonction)

***En anglais : iParameterExport***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de paramétrer les exportations réalisées depuis le visualisateur de rapports :

- paramètres pour l'envoi d'emails.

- paramètres pour la création des différents fichiers.



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Fixe l'adresse email et le sujet du message pour l'export
iParamètreExport(iExportEmailDestinataire, "Wam@CrazyMail.com; Celia.Wat@ZMail.fr")
iParamètreExport(iExportEmailExpéditeur, "Cémoy@Berrymail.com")
iParamètreExport(iExportEmailSujet, "Résultat des ventes de l'année")
// Lance le visualisateur de rapports
iDestination(iVisualisateur)
iImprimeEtat(ETAT_ResultatVente)
```


<a name="Exemple2"></a>



```wl
// Fixe le nom et le répertoire des fichiers créés
iParamètreExport(iExportRépertoire, fRepExe() + [fSep] + DateSys())
iParamètreExport(iExportFichier, "StatMensuelle")
iParamètreExport(iExportOption, ieDéfaut)
// Lance le visualisateur de rapports
iDestination(iVisualisateur)
iImprimeEtat(ETAT_ResultatVente)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`iParamètreExport(<Option à paramétrer> , <Valeur à donner>)`
---

**`<Option à paramétrer> : (Constante de type Entier)`**

Elément à paramétrer :


|   |   |
| --- | --- |
| *iExportEmailDestinataire* | Adresse(s) email des destinataires. |
| *iExportEmailExpéditeur* | Adresse email de l'expéditeur. |
| *iExportEmailSujet* | Sujet du mail. |
| *iExportFichier* | Nom du fichier généré (sans extension). |
| *iExportOption* | Options supplémentaires concernant les fichiers générés. |
| *iExportRépertoire* | Chemin du répertoire où le fichier généré doit être créé. |



**`<Valeur à donner> : (Chaîne de caractères)`**

Valeur à donner à l'option spécifiée.

- Pour indiquer plusieurs destinataires, les adresses email doivent être séparées par des ";".

- Pour les options supplémentaires, les valeurs pouvant être utilisées sont les suivantes (ces constantes sont combinables) :
	


|   |   |
| --- | --- |
| *ieConfirmationAucune* | Aucune confirmation n'est demandée. |
| *ieConfirmationEcrasementFichier* | Si le nom du fichier a été défini, une confirmation est demandée par le visualisateur de rapports avant l'écrasement du fichier. |
| *ieConfirmationOuvertureFichier* | Si l'ouverture du fichier créé a été demandée (constante *ieOuvertureFichier*), une confirmation est demandée par le visualisateur de rapports avant d'ouvrir le fichier. |
| *ieDéfaut* | Combinaison de constantes utilisée par défaut : *ieOuvreFichier* + *ieConfirmationOuvertureFichier* + *ieConfirmationEcrasementFichier* |
| *ieOuvertureFichier* | Ouvre le fichier créé dans l'application associée. |








<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Le paramétrage réalisé est actif jusqu'à :

- la fermeture de l'application,

- la prochaine utilisation de la fonction [iRAZ](../WDLang5/3046067.md),

- la prochaine utilisation de la fonction **iParamètreExport**.

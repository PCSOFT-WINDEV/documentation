


## &lt;Variable sfConnexion&gt;.ExécuteRequête (Fonction)

***En anglais : &lt;sfConnection variable&gt;.ExecuteQuery***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Exécute une requête SOQL (Salesforce Object Query Language) sur une base de données Salesforce.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connexion à Salesforce
Cnx est une  sfConnexion
Cnx..Login = "balthazar@gmail.com"
Cnx..MotDePasse = "azerty" + "XXB12VCZ54"
SI Cnx.Connecte() = Faux ALORS
	Erreur("La connexion a échoué")
	RETOUR
FIN

ResRequête est un sfRésultatExécuteRequête
ResRequête = Cnx.ExécuteRequête("SELECT Date__c,Miles__c,Contact__c FROM Mileage__c")

UnEnregistrement est un sfObjet
POUR TOUT UnEnregistrement DE ResRequête..Objet
	 Trace("---- Type : " + UnEnregistrement..TypeObjet + " | ID : " + UnEnregistrement..ID)
	 Trace(UnEnregistrement..Date__c + " / " + UnEnregistrement..Miles__c + " / " + ...
			UnEnregistrement..Contact__c)
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Connexion Salesforce>.ExécuteRequête(<Requête à exécuter> [, <Enregistrements supprimés>])`
---

**`<Résultat> : (Variable de type sfRésultatExécuteRequête)`**

Variable de type [sfRésultatExécuteRequête](../WDLang5/1000018558.md) contenant les données résultat, notamment les enregistrements Salesforce correspondant au résultat de la requête.

**`<Connexion Salesforce> : (Variable de type sfConnexion)`**

Nom de la variable de type [sfConnexion](../WDLang5/1000017579.md) à utiliser.

**`<Requête à exécuter> : (Chaîne de caractères)`**

Code de la requête à exécuter. Cette requête est une requête SOQL (Salesforce Object Query Language). Pour plus de détails sur ce langage, consultez la documentation Salesforce.

**`<Enregistrements supprimés> : (Booléen optionnel)`**



- <u><u><u><u>Vrai</u></u></u></u> pour inclure les enregistrements supprimés dans le résultat de la requête, 

- <u><u><u><u>Faux</u></u></u></u> (valeur par défaut) pour ne pas inclure les enregistrements supprimés dans le résultat de la requête.

## Remarques


### Cas d'erreurs possibles : 
<a name="cas_erreurs_possibles_ELTPARAGRAPHE000206"></a>

- La connexion Internet n'est pas valide.

- L'authentification n'a pas pu être réalisée correctement.

- La réponse XML du serveur n'est pas lisible.




Pour savoir si une erreur est survenue, utilisez la variable [ErreurDétectée](../WDLang1/3087001.md).

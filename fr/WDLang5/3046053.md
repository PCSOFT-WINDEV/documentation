
## iFinImprime (Fonction)

***En anglais : iEndPrinting***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Signale la fin du document à imprimer et lance l'impression effective des données actuellement stockées dans la file d'attente de l'imprimante par les fonctions [iImprime](../WDLang5/3046031.md), [iImprimeImage](../WDLang5/3046017.md), [iLigne](../WDLang5/3046035.md), ... L'ensemble de ces données constitue le document à imprimer.

**Remarque** : Si la fonction **iFinImprime** n'est pas exécutée, l'ensemble de l'édition sera perdue. Cette fonction ne doit être utilisée qu'avec les états créés par programmation. Cette fonction ne doit pas être appelée dans les traitements d'un état créé avec l'éditeur d'états de WINDEV.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
iImprime("Liste des clients")
iSauteLigne()
// Traitement
iFinImprime()
```

<a name="SYNTAXE1"></a>

## Remarques




### Action de iFinImprime
<a name="action_ifinimprime_ELTPARAGRAPHE000071"></a>La fonction **iFinImprime** annule :

- le paramétrage de [iMarge](../WDLang5/3046011.md),

- les polices créées précédemment.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) le mode d'impression "aperçu" en cours.

### Traitements à interrompre pendant une impression
<a name="traitements_interrompre_pendant_une_impression_ELTPARAGRAPHE000092"></a>Pendant une impression, il faut interrompre les traitements suivants :

- timer (avec la fonction [FinTimer](../WDLang1/3015007.md)),

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) gestion des liens DDE (avec la fonction [DDEFinLien](../WDLang1/3028005.md)),

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) événement (avec la fonction [FinEvénement](../WDLang1/3015010.md)).

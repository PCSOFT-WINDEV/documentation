


## HTMLInsèreAvant (Fonction)

***En anglais : HTMLInsertBefore***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ajoute un noeud avant le noeud spécifié dans un document HTML. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
hDoc est un htmlDocument = "
	<html><head/><body><div id="div"><p>Texte</p></div></body></html>
"
hNoeud est un htmlNoeud <- HTMLChercheElémentParID(hDoc, "div")
// body : <div id="div"><p>Texte</p></div>
hNoeud1 est un htmlNoeud
hNoeud1..NomBalise = "h1"
hNoeud1..Texte = "texte_h1"
HTMLInsèreAvant(hNoeud, hNoeud1)
// <html><head><body><h1>texte_h1</h1><div id="div"><p>Texte</p></div></body></html>
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`HTMLInsèreAvant(<Noeud source> , <Noeud à insérer>)`
---

**`<Noeud source> : (Variable de type htmlNoeud)`**

Nom de la variable de type [htmlNoeud](../WDLang5/1000026081.md) correspondant au noeud de référence pour l'insertion.

**`<Noeud à insérer> : (Variable de type htmlNoeud)`**

Nom de la variable de type [htmlNoeud](../WDLang5/1000026081.md) correspondant au noeud à insérer avant &lt;Noeud source&gt;.

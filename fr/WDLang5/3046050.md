
## iTrait (Fonction)

***En anglais : iLine***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Imprime un trait aux coordonnées spécifiées. Contrairement à la fonction [iLigne](../WDLang5/3046035.md), le bout du trait est carré (et non arrondi).

L'impression n'est pas lancée (seule la fonction [iFinImprime](../WDLang5/3046053.md) permet de lancer l'impression).

**Remarque** : L'origine (0,0) est située dans le coin supérieur gauche de la feuille. Elle tient compte des marges physiques de l'imprimante.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Trace un trait sur toute la largeur de la page
// Épaisseur : 2.5 points Pica
iTrait(10, 10, 30, 25, 2.5)
iFinImprime()
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = iTrait(<X1> , <Y1> , <X2> , <Y2> [, <Épaisseur> [, <Couleur> [, <Style>]]])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le trait a été imprimé, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<X1> : (Réel)`**

Coordonnée horizontale (colonne ou abscisse) du début du trait (en millimètres). Valeur comprise entre 0 et la largeur de la page (donnée par la fonction [iLargeurPage](../WDLang5/3046036.md)).



**`<Y1> : (Réel)`**

Coordonnée verticale (ligne ou ordonnée) du début du trait (en millimètres). Valeur comprise entre 0 et la hauteur de la page (donnée par la fonction [iHauteurPage](../WDLang5/3046052.md)).



**`<X2> : (Réel)`**

Coordonnée horizontale (colonne ou abscisse) de la fin du trait (en millimètres). Valeur comprise entre 0 et la largeur de la page (donnée par la fonction [iLargeurPage](../WDLang5/3046036.md)).



**`<Y2> : (Réel)`**

Coordonnée verticale (ligne ou ordonnée) de la fin du trait (en millimètres). Valeur comprise entre 0 et la hauteur de la page (donnée par la fonction [iHauteurPage](../WDLang5/3046052.md)).



**`<Épaisseur> : (Réel optionnel)`**

Épaisseur du trait (en points pica). Valeur par défaut = 1 (un point Pica = 0,353 mm).
Pour obtenir une épaisseur de 1 pixel, &lt;Épaisseur&gt; doit être égale à 0. Lorsque l'épaisseur augmente, elle est répartie proportionnellement des deux côtés du trait.



**`<Couleur> : (Entier optionnel)`**

Couleur utilisée pour le trait. Si ce paramètre n'est pas précisé, la couleur utilisée est le noir. 
Cette couleur peut correspondre à :

- une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)),

- une couleur TSL (obtenue avec la fonction [TSL](../WDLang1/3029057.md)),

- [une couleur prédéfinie du WLangage](../WDLang5/3010002.md).




**`<Style> : (Constante optionnelle de type entier)`**

Type du trait.


|   |   |
| --- | --- |
| *iContinu*<br>(valeur par défaut) | Trait continu |
| *iMixte* | Trait mixte (-.-.) |
| *iPointillé* | Trait pointillé (....) |
| *iTiret* | Trait tiret (-----) |


![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) **Attention** : Si &lt;Épaisseur&gt; est supérieure à 1, le type du trait sera toujours continu. Ce paramètre peut être ignoré si le driver d'imprimante ne gère pas ce paramètre.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- La position du curseur n'est pas modifiée par le tracé du trait.

- L'utilisation du type de trait pointillé (*iPointillé*) ou mixte (*iMixte*) combinée avec un trait épais peut provoquer des effets disgracieux.

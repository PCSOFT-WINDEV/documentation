
## htmlAttribut (Type de variable)

***En anglais : htmlAttribute***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **htmlAttribut** permet de définir toutes les caractéristiques avancées d'un attribut d'un élément d'un document HTML. Les caractéristiques de cet attribut peuvent être définies et modifiées à l'aide de différentes propriétés WLangage. 

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// ce code permet de manipuler un champ de saisie 
// destiné à la saisie d'un code à usage unique

noeud_html est un htmlNoeud
noeud_html..NomBalise = "input"
attribut_html est un htmlAttribut
attribut_html.Nom = "autocomplete"
attribut_html.Valeur = "one-time-code"
TableauAjoute(noeud_html..Attribut, attribut_html)
```

### Propriétés spécifiques à la description des variables de type htmlAttribut
<a name="proprietes_specifiques_description_des_variables_type_htmlattribut_ELTPARAGRAPHE000029"></a>Les propriétés suivantes peuvent être utilisées pour manipuler un attribut d'un document HTML :

| Nom de la propriété | Type manipulé | Effet |
| --- | --- | --- |
| Existe | Booléen | <br><br>- <u><u><u><u>Vrai</u></u></u></u> si l'élément existe dans le document, <br><br>- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. <br><br><br>**Cette propriété est disponible en lecture seulement**. |
| Nom | Chaîne de caractères | Nom de l'attribut. |
| Valeur | Tous types | Valeur de l'attribut. |

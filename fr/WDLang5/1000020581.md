
## iGroupeParamètre (Fonction)

***En anglais : iGroupConfigure***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie les paramètres d'un état présent dans un groupe d'états. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Préparation d'un groupe d'états
nIndice est un entier = iGroupeAjoute(ETAT_CA_Annee_A4_Portrait, 2012)
iGroupeParamètre(nIndice, "Statistiques 2012 - Portrait", "stat2012.png")
nIndice = iGroupeAjoute(ETAT_CA_Annee_A4_Portrait, 2011)
iGroupeParamètre(nIndice, "Statistiques 2011 - Portrait", "stat2011.png")
iGroupeImprime()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Modifier les paramètres d'un état identifié par son nom

`iGroupeParamètre(<Nom de l'état> , <Libellé> [, <Image>])`
---

**`<Nom de l'état> : (Chaîne de caractères)`**

Nom de l'état à manipuler. Cet état a été ajouté dans le groupe par la fonction [iGroupeAjoute](../WDLang5/1000020580.md). 

**`<Libellé> : (Chaîne de caractères)`**

Libellé associé à l'état spécifié. Ce libellé sera affiché dans la combo présentant les états du groupe. 

**`<Image> : (Chaîne de caractères)`**

Nom de l'image associée à l'état. Cette image sera affichée dans la combo présentant les états du groupe. Cette image peut correspondre à une image présente dans la bibliothèque de l'application ou au chemin complet d'un fichier image. 
Remarque : Il est conseillé d'utiliser une image de 48 x 48 pixels. Si l'image utilisée est plus grande, sa taille sera homothétiquement adaptée.
Remarque : il n'est pas possible d'utiliser une variable de type Image.  


<a name="SYNTAXE2"></a>

### Modifier les paramètres d'un état identifié par son indice dans le groupe

`iGroupeParamètre(<Indice de l'état> , <Libellé> [, <Image>])`
---

**`<Indice de l'état> : (Entier)`**

Indice de l'état à manipuler. Cet indice a été renvoyé par la fonction [iGroupeAjoute](../WDLang5/1000020580.md) lors de l'ajout de l'état dans le groupe. 

**`<Libellé> : (Chaîne de caractères)`**

Libellé associé à l'état spécifié. Ce libellé sera affiché dans la combo présentant les états du groupe. 

**`<Image> : (Chaîne de caractères)`**

Nom de l'image associée à l'état. Cette image sera affichée dans la combo présentant les états du groupe. Cette image peut correspondre à une image présente dans la bibliothèque de l'application ou au chemin complet d'un fichier image. 
Remarque : Il est conseillé d'utiliser une image de 48 x 48 pixels. Si l'image utilisée est plus grande, sa taille sera homothétiquement adaptée. 
Remarque : il n'est pas possible d'utiliser une variable de type Image.

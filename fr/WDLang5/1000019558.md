


## Fonctions de gestion des Codes-Barres
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions de gestion des codes-barres sont les suivantes : 



|   |   |
| --- | --- |
| [CBCalculeCaractèreDeVérification](../WDLang5/1000019909.md) | Retourne le code de contrôle d'un code-barres. |
| [CBCapture](../WDLang5/1000019456.md) | Décode les informations stockées dans un code-barres en utilisant la caméra de l'appareil (Android, iPhone, iPad). |
| [CBDécode](../WDLang5/1000019633.md) | Décode les informations stockées dans un code-barres de type QR Code. |
| [CBVersImage](../WDLang5/1000024060.md) | Génère l'image d'un code-barres. |





Il est également possible d'imprimer par programmation des codes-barres grâce à la fonction suivante : 



|   |   |
| --- | --- |
| [iImprimeCodeBarre](../WDLang5/3046020.md) | Imprime un code-barres (à l'intérieur d'un rectangle). |


## gglTéléphone (Type de variable)

***En anglais : gglPhoneNumber ***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **gglTéléphone** permet de décrire et de modifier le numéro de téléphone d'un contact.

Ce type de variable est utilisé par le type [gglContact](../WDLang5/1000017434.md).

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connexion à Google
Cnx est un gglConnexion
Cnx.Email = "balthazar@gmail.com"
Cnx.MotDePasse = "azerty"
Cnx.NomApplication = "MonAppli-01"
SI GglConnecte(Cnx) = Faux ALORS
	Erreur(ErreurInfo())
FIN

...

// Récupération du tableau des contacts
tabContacts est un tableau de gglContact
tabContacts = GglListeContact(Cnx)

// Affichage des téléphones du premier contact
MonContact est un gglContact = tabContacts[1]
MonTéléphoneContact est un gglTéléphone dynamique
POUR TOUT MonTéléphoneContact DE MonContact.Téléphone
	Trace(MonTéléphoneContact.Libellé + " : " + MonTéléphoneContact.Numéro)
FIN
```

## Remarques




### Propriétés spécifiques à la description des variables de type gglTéléphone
<a name="proprietes_specifiques_description_des_variables_type_ggltelephone_ELTPARAGRAPHE000037"></a>Les propriétés suivantes peuvent être utilisées pour manipuler une variable de type **gglTéléphone** :

| Nom | Type manipulé | Effet |
| --- | --- | --- |
| Libellé | Chaîne de caractères | Description du numéro de téléphone. |
| Numéro | Chaîne de caractères | Numéro de téléphone. |
| Principal | Chaîne de caractères | <br><br>- <u><u><u><u>Vrai</u></u></u></u> si le numéro correspond au numéro de téléphone principal du contact,<br><br>- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.<br><br><br> |
| Type | Constante de type entier | Type du numéro de téléphone correspondant à une des variables suivantes :<br><br>- *gglAutre* : Autre numéro.<br><br>- *gglBureau* : Numéro professionnel.<br><br>- *gglDomicile* : Numéro du domicile.<br><br>- *gglFax* : Numéro de fax.<br><br>- *gglFaxDomicile* : Numéro de fax du domicile.<br><br>- *gglFaxTravail* : Numéro de fax professionnel.<br><br>- *gglPager* : Numéro de pager / bippeur.<br><br>- *gglPériphériqueMobile* : Numéro du téléphone portable.<br><br><br> |

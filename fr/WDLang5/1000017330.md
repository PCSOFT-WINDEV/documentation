


## GglListeAgenda (Fonction)

***En anglais : GglListCalendar***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Récupère la liste des agendas Google associés au compte Google spécifié. Cette fonction utilise le service Agenda Google Calendar.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple
<a class="notetitle" target="_blank" href="$DOC$=1001000017330&name=ggllisteagenda_fonction&product=WD">Voir des exemples supplémentaires</a>

```wl
// Connexion
Cnx est un gglConnexion
...
SI GglConnecte(Cnx) = Faux ALORS
	Erreur(ErreurInfo())
	RETOUR
FIN
tabAgendas est un tableau de gglAgenda 
tabAgendas = GglListeAgenda(Cnx, gglTous) 
// Affichage du titre de chaque agenda
Agenda est un gglAgenda
POUR TOUT Agenda DE tabAgendas
	Trace(Agenda.Titre)
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = GglListeAgenda(<Connexion Google> [, <Filtre>])`
---

**`<Résultat> : (Tableau de variables de type gglAgenda)`**

Liste des agendas Google du compte spécifié.

**`<Connexion Google> : (Variable de type gglConnexion)`**

Nom de la variable de type [gglConnexion](../WDLang5/1000018009.md) à utiliser. Cette connexion a été validée avec la fonction [GglConnecte](../WDLang5/1000017332.md).

**`<Filtre> : (Constante de type Entier optionnelle)`**

Indique le type d'agenda à lister :


|   |   |
| --- | --- |
| *gglTous*<br>(Valeur par défaut) | Liste tous les agendas souscrits (partagés) et créés par l'utilisateur en cours. |
| *gglUtilisateur* | Liste uniquement les agendas créés par l'utilisateur en cours. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Cas particuliers
<a name="cas_particuliers_ELTPARAGRAPHE000068"></a>Les agendas renvoyés ne contiennent pas leurs événements. Pour obtenir les événements des agendas, utilisez les fonctions [GglRemplitAgenda](../WDLang5/1000017337.md) ou [GglRemplitAgendaParRequête](../WDLang5/1000017347.md).

Pour obtenir l'agenda avec les événements associés, utilisez la fonction [GglRécupèreAgenda](../WDLang5/1000017410.md).
<a name="NOTE0_2"></a>


### Cas d'erreurs possibles : 
<a name="cas_erreurs_possibles_ELTPARAGRAPHE000088"></a>

- La connexion Internet n'est pas valide.

- L'authentification n'a pas pu être réalisée correctement.

- La réponse XML du serveur n'est pas lisible.




Pour savoir si une erreur est survenue, utilisez la variable [ErreurDétectée](../WDLang1/3087001.md).
<a name="NOTE0_3"></a>


### Limite
<a name="limite_ELTPARAGRAPHE000104"></a>Seuls les 100 premiers agendas sont renvoyés.

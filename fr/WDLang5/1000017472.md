
## xlsCellule (Type de variable)

***En anglais : xlsCell ***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Une variable de type **xlsCellule** permet de décrire et de modifier une cellule d'un document Excel.

Ce type de variable est utilisé : 

- par les variables de type [xlsDocument](../WDLang5/1000017464.md).

- par les variables de type [xlsColonne](../WDLang5/1000017471.md).

- par les variables de type [xlsLigne](../WDLang5/1000017325.md).




**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ouverture d'un fichier Excel
sFichierExcel est une chaîne
sFichierExcel = fSélecteur("", "", "Sélectionnez un fichier Excel...", "Excel" + ...
	TAB + "*.xlsx" + RC + "Tous les fichiers (*.*)" + TAB + "*.*", "*.xlsx")
SI sFichierExcel = "" ALORS RETOUR

// Ouverture du document Excel
MonDocument est un xlsDocument
MonDocument = xlsOuvre(sFichierExcel)

// Création d'une variable xlsCellule avec fond rouge.
Cellule est un xlsCellule
Cellule = "WINDEV est formidable !"
Cellule.CouleurFond = RougeClair

// Renseigne la cellule A6 
MonDocument..Cellule[6,"A"] = Cellule

// Sauvegarde les modifications
xlsSauve(MonDocument)
xlsFerme(MonDocument)
```

## Remarques




### Propriétés spécifiques à la description des variables de type xlsCellule
<a name="proprietes_specifiques_description_des_variables_type_xlscellule_ELTPARAGRAPHE000059"></a>Les propriétés suivantes peuvent être utilisées pour manipuler une variable de type **xlsCellule** :

| Nom | Type manipulé | Effet |
| --- | --- | --- |
| AlignementH | Constante de type Entier | Alignement horizontal de la cellule. Les valeurs possibles sont les suivantes :<br><br>- *chCentre* : centré.<br><br>- *chDroite* : alignement à droite.<br><br>- *chGauche* : alignement à gauche.<br><br><br> |
| AlignementV | Constante de type Entier | Alignement vertical de la cellule. Les valeurs possibles sont les suivantes :<br><br>- *cvHaut* : alignement en haut.<br><br>- *cvMilieu* : alignement au milieu.<br><br>- *cvBas* : alignement en bas.<br><br><br> |
| Cadre | [Variable de type Cadre](../Motscles/15140100.md) | Caractéristiques du cadre de la cellule. |
| CouleurFond | Entier ou constante de type Entier | Couleur de fond de la cellule. Cette couleur peut correspondre à :<br><br>- une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)),<br><br>- [une couleur prédéfinie](../WDLang5/3010002.md).<br><br><br><br><br> |
| Formule | Chaîne de caractères | Formule d'une cellule. <br><br>Attention : La modification de la formule d'une cellule ne déclenche pas automatiquement le calcul des formules du classeur. <br>Il faut appeler la fonction [xlsRecalculeTout](../WDLang5/1410089617.md) après avoir modifié la formule d'une ou plusieurs cellules pour mettre à jour les valeurs des cellules du classeur ayant une formule.<br><br>Cette propriété fonctionne uniquement sur un document XLSX (elle ne fonctionne pas sur un document XLS). |
| Masque | Chaîne de caractères | Masque d'affichage dans Excel. Ce masque doit correspondre au masque reconnu par Excel. Pour plus de détails sur les masques disponibles, consultez la documentation de Excel. |
| MasqueWD | Chaîne de caractères | Masque d'affichage au format WINDEV ou WEBDEV à utiliser dans la cellule Excel (par exemple "JJ/MM/AAAA"). Le masque spécifié est automatiquement converti au format Excel. Les centièmes de seconde présents dans les masques ne sont pas gérés.<br>Pour plus de détails, consultez [Masques gérés](../WDLang5/1000017473.md).<br>Cette propriété est disponible en écriture uniquement. |
| Multiligne | Booléen | <br><br>- <u><u><u><u>Vrai</u></u></u></u> si la cellule est multiligne,<br><br>- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.<br><br><br> |
| Police | [Variable de type Police](../Motscles/1514045.md) | Police de la cellule. |
| Type | Constante de type Entier | Type Excel de la cellule. Les valeurs possibles sont les suivantes :<br><br>- *xlsTypeChaîne* : Colonne "chaîne de caractères" ou colonne contenant des données de différents types.<br><br>- *xlsTypeDate* : Colonne "Date".<br><br>- *xlsTypeDateHeure* : Colonne "DateHeure".<br><br>- *xlsTypeDouble* : Colonne "Numérique".<br><br>- *xlsTypeDurée* : Colonne "Durée".<br><br>- *xlsTypeErreur* : En cas d'erreur. Pour connaître le libellé de l'erreur, utilisez la fonction [xlsMsgErreur](../WDLang5/3080016.md) ou la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.<br><br>- *xlsTypeHeure* : Colonne "Heure".<br><br>- *xlsTypeMonétaire* : Colonne "Monétaire".<br><br><br>**Cette propriété est disponible en lecture seulement**. |
| Valeur | Type de la valeur de la cellule | Valeur de la cellule.<br>Dans le cas de la définition du style d'une colonne (xlsColonne.CelluleDéfaut), cette propriété est ignorée en écriture et initialisée à vide en lecture). |

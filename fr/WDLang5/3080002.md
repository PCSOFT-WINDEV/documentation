
## Constantes de gestion des fichiers XLS
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


| Constante | Description | Détail | Utilisée par les fonctions |
| --- | --- | --- | --- |
| **xlsCompatible** | Mode d'ouverture d'un fichier XLS | Ouverture en mode compatible avec les versions précédentes de WINDEV et WEBDEV (gestion uniquement de la première feuille du classeur) | [xlsOuvre](../WDLang5/3080017.md) |
| **xlsDonnéeErreur** | Erreur (constante conservée par compatibilité) | Constante retournée en cas d'erreur | [xlsDonnée](../WDLang5/3080021.md) |
| **xlsEcriture** | Mode d'ouverture d'un fichier XLS | Ouvre le fichier Excel en mode "Lecture / Ecriture" : le fichier XLS est ouvert et bloqué jusqu'à sa fermeture. | [xlsOuvre](../WDLang5/3080017.md) |
| **xlsFusionCelluleCentrée** | Mode de fusion des cellules | Fusionne les cellules en centrant le contenu. | [xlsFusionneCellule](../WDLang5/1410089537.md) |
| **xlsFusionCelluleSimple** | Mode d'ouverture d'un fichier XLS | Fusionne les cellules sans centrer le contenu. | [xlsFusionneCellule](../WDLang5/1410089537.md) |
| **xlsNumColonne** | Type de coordonnée à renvoyer | Numéro de colonne. | [xlsNomVersCoordonnées](../WDLang5/1000021501.md) |
| **xlsNumLigne** | Type de coordonnée à renvoyer | Numéro de ligne. | [xlsNomVersCoordonnées](../WDLang5/1000021501.md) |
| **xlsTitreErreur** | Erreur | Constante retournée en cas d'erreur | [xlsTitreColonne](../WDLang5/3080012.md) |
| **xlsTypeChaîne** | Type des données saisies | Colonne "chaîne de caractères" ou colonne contenant des données de différents types | [xlsTypeColonne](../WDLang5/3080019.md), variable de type [xlsCellule](../WDLang5/1000017472.md). |
| **xlsTypeDate** | Type des données saisies | Colonne "date" | [xlsTypeColonne](../WDLang5/3080019.md), variable de type [xlsCellule](../WDLang5/1000017472.md). |
| **xlsTypeDateHeure** | Type des données saisies | Colonne "dateheure" | [xlsTypeColonne](../WDLang5/3080019.md), variable de type [xlsCellule](../WDLang5/1000017472.md). |
| **xlsTypeDouble** | Type des données saisies | Colonne "numérique" | [xlsTypeColonne](../WDLang5/3080019.md), variable de type [xlsCellule](../WDLang5/1000017472.md). |
| **xlsTypeDurée** | Type des données saisies | Colonne "durée" | [xlsTypeColonne](../WDLang5/3080019.md), variable de type [xlsCellule](../WDLang5/1000017472.md). |
| **xlsTypeErreur** | Erreur | Constante retournée en cas d'erreur | [xlsTypeColonne](../WDLang5/3080019.md), variable de type [xlsCellule](../WDLang5/1000017472.md). |
| **xlsTypeHeure** | Type des données saisies | Colonne "heure" | [xlsTypeColonne](../WDLang5/3080019.md), variable de type [xlsCellule](../WDLang5/1000017472.md). |
| **xlsTypeMonétaire** | Type des données saisies | Colonne "monétaire" | [xlsTypeColonne](../WDLang5/3080019.md), variable de type [xlsCellule](../WDLang5/1000017472.md). |
| **xlsTypeVide** | Type des données saisies | Colonne entièrement vide | [xlsTypeColonne](../WDLang5/3080019.md) |

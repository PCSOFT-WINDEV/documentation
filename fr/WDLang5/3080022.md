


## xlsFeuilleEnCours (Fonction)

***En anglais : xlsCurrentWorksheet***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de connaître ou de modifier la feuille courante d'un fichier XLS.

**Remarques** :

- En mode multi-feuilles, les fonctions de gestion des fichiers XLS travaillent toujours sur la feuille courante. Lors de l'ouverture du ficher XLS, la 1ère feuille du classeur est la feuille courante. 

- En mode compatible, cette fonction n'a aucun effet.





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ouverture d'un fichier XLS
IdentifiantFichier est un entier 
IdentifiantFichier = xlsOuvre("C:\Mes Documents\Test.XLS")

// Manipulations sur le fichier XLS
// ...

// Récupération de la feuille courante
Feuille est un entier = xlsFeuilleEnCours(IdentifiantFichier)
Info("La feuille courante est " + Feuille)
```
<a name="Exemple2"></a>

```wl
// Ouverture d'un fichier XLS
IdentifiantFichier est un entier = xlsOuvre("C:\Mes Documents\Test.XLS")

// Positionnement sur la dernière feuille du fichier
NbFeuille est un entier = xlsNbFeuille(IdentifiantFichier)
SI xlsFeuilleEnCours(IdentifiantFichier, NbFeuille) = Vrai ALORS
	Info("La feuille courante a été modifiée.")
FIN
```
<a name="Exemple3"></a>

```wl
cMonFicXLS est xlsDocument
sMonFichier est une chaîne = "Fichier.xls"	// Nom complet du fichier à ouvrir

cMonFicXLS = xlsOuvre(sMonFichier, xlsEcriture)
SI ErreurDétectée = Faux ALORS
	Azz est un entier = xlsFeuilleEnCours(cMonFicXLS)
	Info("Feuille en cours dans le fichier Excel : " + Azz)
	// Changement de feuille
	SI xlsFeuilleEnCours(cMonFicXLS, 2) = Vrai ALORS
		Info("La feuille courante a été modifiée.")
		// Lecture d'une cellule pour test
		Info(xlsDonnée(cMonFicXLS, 11, 2))
	SINON
		Erreur("Le fichier sélectionné ne comporte pas de seconde feuille !")
	FIN 
SINON
	Erreur(ErreurInfo(errComplet))
FIN
xlsFerme(cMonFicXLS)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître la feuille en cours

`<Résultat> = xlsFeuilleEnCours(<Document XLS>)`
---

**`<Résultat> : (Entier)`**

Indice de la feuille en cours. Cette valeur est comprise entre 1 et [xlsNbFeuille](../WDLang5/3080023.md).

**`<Document XLS> : (Variable de type xlsDocument)`**

Nom de la variable de type [xlsDocument](../WDLang5/1000017464.md) à manipuler.  


<a name="SYNTAXE2"></a>

### Modifier la feuille en cours

`<Résultat> = xlsFeuilleEnCours(<Document XLS> , <Numéro de la feuille>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la feuille courante a été modifiée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire (par exemple si la feuille spécifiée dans le paramètre &lt;Numéro de la feuille&gt; n'existe pas).




**`<Document XLS> : (Variable de type xlsDocument)`**

Nom de la variable de type [xlsDocument](../WDLang5/1000017464.md) à manipuler.

**`<Numéro de la feuille> : (Entier)`**

Indice de la nouvelle feuille. Cette valeur est comprise entre 1 et [xlsNbFeuille](../WDLang5/3080023.md).  


<a name="SYNTAXE3"></a>

### Connaître la feuille en cours (syntaxe conservée par compatibilité avec la version 12)

`<Résultat> = xlsFeuilleEnCours(<Identifiant du fichier XLS>)`
---

**`<Résultat> : (Entier)`**

Indice de la feuille en cours. Cette valeur est comprise entre 1 et [xlsNbFeuille](../WDLang5/3080023.md).

**`<Identifiant du fichier XLS> : (Entier)`**

Identifiant du fichier XLS à manipuler, renvoyé par la fonction [xlsOuvre](../WDLang5/3080017.md).  


<a name="SYNTAXE4"></a>

---

- <u><u><u><u>Vrai</u></u></u></u> si la feuille courante a été modifiée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire (par exemple si la feuille spécifiée dans le paramètre &lt;Numéro de la feuille&gt; n'existe pas).




**`<Identifiant du fichier XLS> : (Entier)`**

**`<Numéro de la feuille> : (Entier)`**




## &lt;Variable xlsDocument&gt;.Ferme (Fonction)

***En anglais : &lt;xlsDocument variable&gt;.Close***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ferme un fichier XLS.

<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonFichierXLS est un xlsDocument

// Ouverture d'un fichier XLS
MonFichierXLS = xlsOuvre("C:\MesRépertoires\Fichier.XLS")
...
// Fermeture du fichier XLS
MonFichierXLS.Ferme()
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Document XLS>.Ferme()`
---

**`<Document XLS> : (Variable de type xlsDocument)`**

Nom de la variable de type [xlsDocument](../WDLang5/1000017464.md) correspondant au fichier XLS à fermer. Le fichier physique associé à cette variable (s'il existe) est fermé. Les modifications réalisées dans le fichier ne sont pas sauvées avant sa fermeture (utilisez la fonction [&lt;Variable xlsDocument&gt;.Sauve](../WDLang5/1000021511.md) si nécessaire).




## XMLRenomme (Fonction)

***En anglais : XMLRename***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie le nom de l'élément courant (balise ou attribut) dans un document XML.






<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = XMLRenomme(<Document XML> , <Nouveau nom de l'élément>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la modification est effectuée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire (la variable [ErreurDétectée](../WDLang1/3087001.md) est alors à <u><u><u><u>Vrai</u></u></u></u>). Pour obtenir plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Document XML> : (Chaîne de caractères)`**

Nom du document XML dans lequel la modification doit être effectuée.

**`<Nouveau nom de l'élément> : (Chaîne de caractères)`**

Nouveau nom de l'élément en cours :

- nom de la balise si l'élément en cours est une balise.

- nom de l'attribut si l'élément en cours est un attribut.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Lecture du nom de l'élément courant
<a name="lecture_nom_element_courant_ELTPARAGRAPHE000086"></a>Pour lire le nom de l'élément courant, utilisez la fonction [XMLNomElément](../WDLang5/3081026.md).
<a name="NOTE0_2"></a>


### Enregistrer les modifications dans le fichier XML
<a name="enregistrer_les_modifications_dans_fichier_xml_ELTPARAGRAPHE000096"></a>Pour enregistrer les modifications effectuées dans un fichier XML, vous devez :

1. Utiliser la fonction [XMLConstruitChaîne](../WDLang5/3081016.md). Cette fonction récupère et met en forme le contenu d'un document XML.

2. Enregistrer la chaîne de caractères générée par la fonction [XMLConstruitChaîne](../WDLang5/3081016.md) dans un fichier XML. Pour cela, il suffit d'utiliser la fonction [fSauveTexte](../WDLang1/3036046.md).




Exemple :

```wl
// Après modification du document XML 
SourceXML = XMLConstruitChaîne("DocXML")
// Enregistrement du fichier XML
fSauveTexte("ExampleModifié.xml", SourceXML)
```

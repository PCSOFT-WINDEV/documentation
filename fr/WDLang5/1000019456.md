
## CBCapture (Fonction)

***En anglais : BCCapture***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Décode les informations stockées dans un code-barres en utilisant la caméra de l'appareil (Android, iPhone, iPad).

**Conseil : Préférez le champ Caméra** : 









<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Décodage d'un code-barres QR Code
cb est un CodeBarres
cb = CBCapture()
SI ErreurDétectée = Faux _ET_ cb.TypeContenu = cbTypeEmail ALORS
    	Info("Adresse email : " + cb.ValeurBrute)
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = CBCapture([<Type code-barres> [, <Activer le flash>]])`
---

**`<Résultat> : (Variable de type CodeBarres)`**

Variable de type [CodeBarres](../WDLang5/1000019451.md) initialisée avec les informations lues dans le code-barres.



**`<Type code-barres> : (Constante de type Entier optionnelle)`**

Type de code-barres pouvant être décodé : 


|   |   |
| --- | --- |
| *CB_AZTEC* | Code-barres de type Aztec. <br><br> |
| *CB_CODE128* | Code-barres de type Code128.<br><br> |
| *CB_CODE39* | Code-barres de type Code39.<br><br> |
| *CB_CODE93* | Code-barres de type Code93.<br><br> |
| *CB_DATAMATRIX* | Code-barres de type DATAMATRIX.<br><br> |
| *CB_EAN128* | Code-barres de type EAN128.<br><br> |
| *CB_EAN13* | Code-barres de type EAN13.<br><br> |
| *CB_EAN8* | Code-barres de type EAN8.<br><br> |
| *CB_INTER2OF5* | Code-barres de type Interleaved 2 of 5.<br><br> |
| *CB_PDF417* | Code-barres de type PDF417.<br><br> |
| *CB_QRCODE* | Code-barres de type QR Code. |
| *CB_UPCA* | Code-barres de type UPC-A.<br><br> |
| *CB_UPCE* | Code-barres de type UPC-E.<br><br> |
| *CB_TOUS* | Tous les types de codes-barres supportés par cette fonction pourront être décodés. |


Si ce paramètre n'est pas spécifié, tous les types de codes-barres supportés par cette fonction pourront être décodés. 



**`<Activer le flash> : (Booléen optionnel)`**

### Gestion des erreurs
<a name="gestion_des_erreurs_ELTPARAGRAPHE000264"></a>Pour savoir si la récupération du code-barres a échoué, utilisez la variable [ErreurDétectée](../WDLang1/3087001.md).

La fonction échouera dans les cas suivants : 

- si elle est utilisée dans le simulateur, 

- si elle est utilisée dans l'émulateur, 

- si elle est utilisée sur un appareil ne disposant pas d'une caméra.

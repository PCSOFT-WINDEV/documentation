


## XMLSigne (Fonction)

***En anglais : XMLSign***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Signe un document XML.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonDoc est un xmlDocument
NoeudASigner est un xmlNoeud, description="schemaXSDDuProjet.NoeudRacine"
MonCertificat est un Certificat
	
MaSignature est un xmlSignature
MaSignature.AlgorithmeNormalisation = atC14NExcluAvecCommentaire
MaSignature.AlgorithmeSignature = asRSA_SHA_160
MaSignature.Clé = MonCertificat

SI XMLSigne(MaSignature, MonDoc.NoeudASigner) = Vrai ALORS 
	Trace("Document signé")
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = XMLSigne(<Signature> , <Noeud d'insertion>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le document XML a été signé, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Signature> : (Variable de type xmlSignature)`**

Nom de la variable de type [xmlSignature](../WDLang3/1000020736.md) correspondant à la signature XML à utiliser. 

**`<Noeud d'insertion> : (Variable de type xmlNoeud)`**

Nom de la variable de type [xmlNoeud](../WDLang5/1000018786.md) correspondant à l'emplacement où la signature doit être insérée.

## Remarques


### Cas particulier
<a name="cas_particulier_ELTPARAGRAPHE000059"></a>La signature du XML n'est pas possible pour des "smart card". Dans ce cas, la fonction échoue.


## mlRAZEspaceDeTravail (Fonction)

***En anglais : mlResetWorkspace***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime l'ensemble des variables de la session MATLAB.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
tRésultat est un tableau  de 1 par 1 réel = [[123]]

mlRAZEspaceDeTravail(gnIdSession)
tRésultat = mlRécupèreVariable(gnIdSession, "maRac")
SI ErreurDétectée ALORS
	Trace("Erreur : " + ErreurInfo())
FIN

Trace("Racine = " + tRésultat[1,1])
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = mlRAZEspaceDeTravail(<Identifiant de la session>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'espace de travail est réinitialisé, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. En cas d'erreur, la variable [ErreurDétectée](../WDLang1/3087001.md) est à <u><u><u><u>Vrai</u></u></u></u>. Pour connaître le libellé de l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.




**`<Identifiant de la session> : (Entier)`**

Identifiant de la session MATLAB. Cet identifiant est renvoyé par la fonction [mlInitialise](../WDLang5/1000021351.md).

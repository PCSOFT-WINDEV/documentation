 



## xlsNbLignes (Fonction)

***En anglais : xlsNbRows***
	

<a name="XUtilisation"></a>
<a name="Utilisation"></a>
|  |   |
| --- | --- |

|  
 | **But**<br><br>Obtenir le nombre de lignes d'un fichier XLS.<br><br>**Syntaxe (utilisant le type xlsDocument)**<br><br>&lt;Résultat&gt; = xlsNbLignes(&lt;Document XLS&gt;)<br><br>- &lt;Résultat&gt; est un entier. <br><br>- &lt;Document XLS&gt; est le nom de la variable de type [xlsDocument](../WDLang5/1000017464.md) correspondant au fichier XLS à manipuler.<br><br><br><br><br>**Syntaxe (compatibilité version 12)**<br><br>&lt;Résultat&gt; = xlsNbLignes(&lt;NumFichier&gt;)<br><br>- &lt;Résultat&gt; est un entier.<br><br>- &lt;NumFichier&gt; est un entier contenant l'identifiant du fichier fourni par xlsOuvre.<br><br><br><br><br>**Détail**<br><br>**xlsNbLignes** retourne :<br><br>- le nombre de lignes du fichier XLS : Les lignes vides ne sont pas prises en compte.<br><br>- -1 en cas d'erreur : la fonction xlsErreur permet de connaître le libellé de l'erreur<br><br><br> |
| --- | --- |


## iTerminePage (Fonction)

***En anglais : iPageEnd***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Force le passage à la page suivante au cours de l'impression d'un état.

**Remarques** :

- Pour forcer le saut de page, il est aussi possible de sélectionner l'option "Saut de page après le bloc" dans la description d'un bloc (onglet "UI").

- Cette fonction peut être utilisée uniquement avec des états créés avec l'éditeur d'états.






**Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Imprimer sous Linux](../Editeurs/9000078.md). 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Saut de page
iTerminePage()
```

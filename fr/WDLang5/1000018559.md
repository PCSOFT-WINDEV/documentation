
## sfRésultatExécuteTraitement (Type de variable)

***En anglais : sfExecuteProcessResult***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **sfRésultatExécuteTraitement** permet de connaître le résultat de la fonction [SFExécuteTraitement](../WDLang5/1000018523.md). Les caractéristiques de ce résultat peuvent être connues à l'aide de différentes propriétés WLangage.

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).

<a name="NOTE0_1"></a>

## Remarques




### Propriétés spécifiques à la description des variables de type sfRésultatExécuteTraitement
<a name="proprietes_specifiques_description_des_variables_type_sfresultatexecutetraitement_ELTPARAGRAPHE000039"></a>Les propriétés suivantes peuvent être utilisées pour manipuler le résultat de la fonction [SFExécuteTraitement](../WDLang5/1000018523.md) :

| Nom | Type manipulé | Effet |
| --- | --- | --- |
| Erreur | Tableau de [sfErreur](../WDLang5/1000018455.md) | Compte-rendu d'erreurs. |
| IDActeur | Chaîne de caractères | Identifiants des acteurs actuellement assignés à l'étape de validation.<br>**Cette propriété est disponible en lecture seulement**. |
| IDEntité | Chaîne de caractères | Identifiant de l'objet Salesforce en cours de traitement.<br>**Cette propriété est disponible en lecture seulement**. |
| IDInstance | Chaîne de caractères | Identifiant de l'instance de traitement associé à l'objet Salesforce.<br>**Cette propriété est disponible en lecture seulement**. |
| IDNouvelleTâche | Chaîne de caractères | Identifiants des nouvelles tâches liées au traitement. Ces identifiants sont séparés par des RC (Retour-chariot).<br>**Cette propriété est disponible en lecture seulement**. |
| StatutInstance | Chaîne de caractères | Statut de l'instance du traitement. Cette propriété peut par exemple correspondre à "Approved", "Rejected", "Remove", ou "Pending".<br>**Cette propriété est disponible en lecture seulement**. |
| Succès | Booléen | <br><br>- <u><u><u><u>Vrai</u></u></u></u> si le traitement s'est correctement déroulé,<br><br>- <u><u><u><u>Faux</u></u></u></u> en cas d'erreur.<br><br><br>**Cette propriété est disponible en lecture seulement**. |

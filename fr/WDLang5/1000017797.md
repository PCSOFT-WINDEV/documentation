
## rssIdentifiant (Type de variable)

***En anglais : rssIdentifier ***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **rssIdentifiant** permet de décrire l'identifiant unique d'une entrée de flux RSS. Les caractéristiques de ce type de variable peuvent être définies et modifiées à l'aide de différentes propriétés WLangage.  

Ce type de variable est utilisé par le [type rssEntrée](../WDLang5/1000017792.md).  

**Notes** :

- Le plus souvent, la déclaration d'une variable de type **rssIdentifiant** n'est pas nécessaire. En effet, les propriétés du [type rssEntrée](../WDLang5/1000017792.md) permettent de manipuler directement les propriétés du type rssIdentifiant (notamment la propriété .Identifiant).

- Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).

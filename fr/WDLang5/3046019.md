
## iImprimeComplémentCorps (Fonction)

***En anglais : iPrintBodyComplement***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Force l'impression d'un bloc de complément de corps en spécifiant la hauteur du bloc de complément.

Cette fonction permet par exemple de forcer l'impression d'un complément de corps :

- soit après une fin de rupture

- soit sur la dernière page d'un état




**Remarque** : Si cette fonction n'est pas utilisée, le complément de corps s'imprimera à chaque page après le dernier bloc corps et prendra toute la hauteur restante sur la page. Le complément de corps ne s'imprime que si un bloc corps est imprimé dans la page.

![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Imprimer sous Linux](../Editeurs/9000078.md). 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Code avant impression du Haut de document
iImprimeComplémentCorps(iHauteurRestantePage() - MoiMême.Hauteur)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`iImprimeComplémentCorps(<Hauteur du bloc complément>)`
---

**`<Hauteur du bloc complément> : (Réel)`**

Hauteur du bloc lors de l'impression, exprimée en millimètres. Si cette hauteur est supérieure à la place restante sur la page, le complément de corps s'imprimera sur 2 pages.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
La fonction **iImprimeComplémentCorps** pourrait être remplacée par l'impression d'un bloc (fonction [iImprimeBloc](../WDLang5/3046030.md)) redimensionné avec la propriété [Hauteur](../Proprietes/2510050.md).

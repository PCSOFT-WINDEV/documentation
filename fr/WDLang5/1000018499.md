
## sfRésultatFusionne (Type de variable)

***En anglais : sfMergeResult ***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **sfRésultatFusionne** permet de connaître le résultat de la fonction [SFFusionne](../WDLang5/1000018509.md). Les caractéristiques de ce résultat peuvent être connues à l'aide de différentes propriétés WLangage.

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).


<a name="XSYNTAXE"></a>


<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Propriétés spécifiques à la description des variables de type sfRésultatFusionne
<a name="proprietes_specifiques_description_des_variables_type_sfresultatfusionne_ELTPARAGRAPHE000039"></a>Les propriétés suivantes peuvent être utilisées pour manipuler le résultat de la fonction [SFFusionne](../WDLang5/1000018509.md) :

| Nom | Type manipulé | Effet |
| --- | --- | --- |
| Erreur | Tableau de [sfErreur](../WDLang5/1000018455.md) | Compte-rendu d'erreurs. |
| ID | Chaîne de caractères | Identifiant de l'objet fusionné.<br>**Cette propriété est disponible en lecture seulement**. |
| IDEnregistrementApparentéMisAJour | Chaîne de caractères | Liste des identifiants Salesforce des enregistrements qui ont été modifiés suite à la fusion (enregistrements liés aux enregistrements fusionnés). Les différents identifiants sont séparés par des RC (Retour-chariot). <br>**Cette propriété est disponible en lecture seulement**. |
| IDEnregistrementFusionné | Chaîne de caractères | Liste des identifiants Salesforce des enregistrements fusionnés. Les différents identifiants sont séparés par des RC (Retour-chariot). <br>**Cette propriété est disponible en lecture seulement**. |
| Succès | Booléen | <br><br>- <u><u><u><u>Vrai</u></u></u></u> si la fusion s'est correctement déroulée,<br><br>- <u><u><u><u>Faux</u></u></u></u> en cas d'erreur.<br><br><br>**Cette propriété est disponible en lecture seulement**. |

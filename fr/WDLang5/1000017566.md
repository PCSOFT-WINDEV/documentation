


## Fonctions de gestion des cartes Google
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions suivantes permettent de gérer des cartes par le service de cartographie "Google Maps" :


|   |   |
| --- | --- |
| [GglAdresseVersCoordonnées](../WDLang5/1000017568.md) | Récupère les coordonnées (couple latitude/longitude) d'une adresse. |
| [GglRécupèreCarte](../WDLang5/1000017404.md) | Récupère la carte d'un lieu précis par le service de cartographie Google Maps. |
| [GglRequête](../WDLang5/1000017345.md) | Effectue une requête de communication (requête HTTP) à un service Google. |


**Avertissement** : Avant d'utiliser une fonctionnalité reliée à des services Google, nous vous conseillons vivement de vous reporter à la licence d'utilisation de ce service. Certaines restrictions peuvent s'appliquer. Le contenu des licences peut varier avec le temps.

PC SOFT dégage toute responsabilité de l'usage qui peut être fait des fonctions d'accès natif. Il vous appartient de vérifier que l'usage que vous faites respecte la licence du fournisseur du service.

**Attention** : Si vous utilisez un proxy pour accéder à Internet, il est nécessaire de configurer le proxy (fonction [Proxy](../WDLang3/3043002.md)) pour utiliser les fonctions Google.

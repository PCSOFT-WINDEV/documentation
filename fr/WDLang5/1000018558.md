
## sfRésultatExécuteRequête (Type de variable)

***En anglais : sfExecuteQueryResult***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **sfRésultatExécuteRequête** permet de connaître le résultat des fonctions [SFAvanceRequête](../WDLang5/1000018483.md) et [SFExécuteRequête](../WDLang5/1000018438.md). Les caractéristiques de ce résultat peuvent être connues à l'aide de différentes propriétés WLangage.

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connexion à Salesforce
Cnx est une  sfConnexion
Cnx.Login = "balthazar@gmail.com"
Cnx.MotDePasse = "azerty" + "XXB12VCZ54"
SI SFConnecte(Cnx) = Faux ALORS
	Erreur("La connexion a échoué")
	RETOUR
FIN

// Récupération de données
ResRequête est un sfRésultatExécuteRequête
ResRequête = SFExécuteRequête(Cnx, "SELECT Date__c, Miles__c, Contact__c FROM Mileage__c")

UnEnregistrement est un sfObjet
POUR TOUT UnEnregistrement DE ResRequête.Objet
	 Trace("---- Type : " + UnEnregistrement.TypeObjet + " | ID : " + UnEnregistrement.ID)
	 Trace(UnEnregistrement.Date__c + " / " + UnEnregistrement.Miles__c + " / " + ...
		UnEnregistrement.Contact__c)
FIN
```

## Remarques




### Propriétés spécifiques à la description des variables de type sfRésultatExécuteRequête
<a name="proprietes_specifiques_description_des_variables_type_sfresultatexecuterequete_ELTPARAGRAPHE000048"></a>Les propriétés suivantes peuvent être utilisées pour manipuler le résultat des fonctions [SFAvanceRequête](../WDLang5/1000018483.md) et [SFExécuteRequête](../WDLang5/1000018438.md) :

| Nom | Type manipulé | Effet |
| --- | --- | --- |
| Fini | Booléen | <br><br>- <u><u><u><u>Vrai</u></u></u></u> si la requête a été complètement exécutée,<br><br>- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.<br><br><br>Cette propriété est utile lors de l'exécution de la fonction [SFAvanceRequête](../WDLang5/1000018483.md).<br>**Cette propriété est disponible en lecture seulement**. |
| LocalisationRequête | Chaîne de caractères | Localisation de la requête. Cette propriété est utile lors de l'exécution de la fonction [SFAvanceRequête](../WDLang5/1000018483.md).<br>**Cette propriété est disponible en lecture seulement**. |
| Objet | Tableau de [sfObjet](../WDLang5/1000018439.md) | Enregistrements Salesforce correspondant au résultat de la requête. |
| Taille | Entier | Nombre d'enregistrements correspondant au résultat.<br>**Cette propriété est disponible en lecture seulement**. |

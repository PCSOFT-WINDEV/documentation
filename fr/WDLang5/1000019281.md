
## XMLDocument

***En anglais : ***
	

<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le mot-clé **xmlDocument** peut correspondre :

- à la fonction **XMLDocument** qui permet de créer un nouveau document XML. Pour plus de détails, consultez [XMLDocument](../WDLang5/3081008.md).

- à une variable de type **xmlDocument**. Pour plus de détails, consultez [xmlDocument](../WDLang5/1000018782.md).

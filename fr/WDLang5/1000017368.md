


## GglEnvoieDocument (Fonction)

***En anglais : GglUploadDocument***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Envoie un document sur le serveur Google.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
Cnx est un gglConnexion
...
// Connexion
SI GglConnecte(Cnx) = Faux ALORS
	Erreur(ErreurInfo())
SINON
	// Envoi d'un fichier Word
	GglEnvoieDocument(Cnx, "c:\Docs\fichier.doc") 
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = GglEnvoieDocument(<Connexion Google> , <Chemin du document> [, <Langue source> [, <Langue destination>]])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'envoi du fichier s'est correctement déroulé, 

- <u><u><u><u>Faux</u></u></u></u> en cas de problème. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Connexion Google> : (Variable de type gglConnexion)`**

Nom de la variable de type [gglConnexion](../WDLang5/1000018009.md) à utiliser. Cette connexion a été validée avec la fonction [GglConnecte](../WDLang5/1000017332.md).

**`<Chemin du document> : (Chaîne de caractères)`**

Nom et chemin complet (ou relatif) du fichier (260 caractères maximum). Un chemin de type UNC peut être utilisé.

**`<Langue source> : (Chaîne de caractères)`**

Langue actuelle du document. Ce paramètre permet de définir la langue source pour la traduction du document lors de l'envoi du fichier. Ce paramètre correspond au code pays à la norme ISO639-1. La liste des codes pays à la norme ISO639-1 est présente à l'adresse suivante : [http://www.loc.gov/standards/iso639-2/php/code_list.php](http://www.loc.gov/standards/iso639-2/php/code_list.php)
Si ce paramètre correspond à une chaîne vide (""), Google tente de détecter automatiquement la langue du document. 
Attention : Avec l'utilisation de l'API "Drive", ce paramètre n'est plus pris en compte.

**`<Langue destination> : (Chaîne de caractères)`**

Langue du document présent sur le serveur. Ce paramètre permet de traduire le document lors de l'envoi du fichier. Ce paramètre correspond au code pays à la norme ISO639-1. 
La liste des codes pays à la norme ISO639-1 est présente à l'adresse suivante : [http://www.loc.gov/standards/iso639-2/php/code_list.php](http://www.loc.gov/standards/iso639-2/php/code_list.php). 
Attention : Avec l'utilisation de l'API "Drive", ce paramètre n'est plus pris en compte.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Cas d'erreurs possibles : 
<a name="cas_erreurs_possibles_ELTPARAGRAPHE000094"></a>

- La connexion Internet n'est pas valide.

- L'authentification n'a pas pu être réalisée correctement.

- Le chemin du document n'existe pas, le format du document n'est pas pris en charge ou le document est trop volumineux.



<a name="NOTE0_2"></a>


### Types de fichiers pris en charge par Google
<a name="types_fichiers_pris_charge_par_google_ELTPARAGRAPHE000105"></a>

- Fichiers de présentation PowerPoint (extensions ppt et pps). Limite de taille : 10 Mo. 

- Fichiers HTML, texte brut, Word (extension doc), OpenDocument Text (extension odt), RTF, Star Office (extension sxw) : 500 Ko.

- Fichiers correspondants à des feuilles de calculs (extensions csv, xls et ods) : 1 Mo.



<a name="NOTE0_3"></a>


### Cas particuliers
<a name="cas_particuliers_ELTPARAGRAPHE000116"></a>

- Les caractéristiques d'un document présent sur le serveur Google sont gérées grâce aux variables de type [gglDocument](../WDLang5/1000018001.md). Pour obtenir les caractéristiques de tous documents présents sur un serveur Google, utilisez la fonction [GglListeDocument](../WDLang5/1000017367.md) : cette fonction remplit un tableau de variables de type gglDocument.

- L'interface WEB permet de :

	- placer un document dans un dossier.

	- donner un titre au document.

	- marquer les documents comme "suivis".


 Ces éléments ne sont pas gérés par programmation dans cette version.

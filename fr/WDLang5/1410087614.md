


## HTMLSupprime (Fonction)

***En anglais : HTMLRemove***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime le noeud spécifié et son arborescence dans un document HTML. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
hDoc est un htmlDocument = "
<html>
	<head/>
	<body>
		<h2><span>
			<p>Texte</p>
			</span></h2><h3></h3>
	</body>
</html>
"
hNoeud est un htmlNoeud
// body : <div id="div"><p>Texte</p></div>
hNoeudInsertion est un htmlNoeud
hNoeudRecherche est un tableau de htmlNoeuds <- HTMLChercheElémentParBalise(hDoc, "h3")
hNoeud <- hNoeudRecherche[1]
hNoeudInsertion..NomBalise = "div"
HTMLInsèrePremierFils(hNoeud, hNoeudInsertion)
// body : <h1/><h2><span><p>Texte</p></span></h2><h3><div/></h3>
hNoeudInsertion..NomBalise = "span"
HTMLInsèreDernierFils(hNoeud, hNoeudInsertion)
// body : <h1/><h2><span><p>Texte</p></span></h2><h3><div/><span/></h3>
Trace(hDoc)
HTMLSupprime(hNoeud)
// body : <h2><span><p>Texte</p></span></h2>
Trace(hDoc)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`HTMLSupprime(<Noeud à supprimer>)`
---

**`<Noeud à supprimer> : (Variable de type htmlNoeud)`**

Nom de la variable de type [htmlNoeud](../WDLang5/1000026081.md) correspondant au noeud à supprimer. Toute son arborescence sera supprimée.


## iParamètreEtat (Fonction)

***En anglais : iParameterReport***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Configure l'imprimante en ouvrant directement la fenêtre de configuration. La fenêtre de configuration est initialisée avec les paramètres choisis lors du développement (orientation de l'état, marges, ...) de l'état spécifié.





![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Attention** : L'utilisation de cette fonction sous Linux nécessite une configuration spécifique. Pour plus de détails, consultez [Imprimer sous Linux](../Editeurs/9000078.md). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI iParamètreEtat(ETAT_MonEtat) = Vrai ALORS
	iImprimeEtat(ETAT_MonEtat)
SINON  
	 Info("Configuration de l'état abandonnée")
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = iParamètreEtat(<Nom de l'état>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'utilisateur a validé les informations saisies dans la boîte de dialogue, 

- <u><u><u><u>Faux</u></u></u></u> si l'utilisateur a abandonné.




**`<Nom de l'état> : (Chaîne de caractères)`**

Nom de l'état utilisé comme référence. Les caractéristiques de cet état (choisies lors du développement de l'état) seront utilisées pour initialiser la fenêtre de configuration (orientation, ...).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Configuration prise en compte lors de l'impression d'un état
<a name="configuration_prise_compte_lors_impression_etat_ELTPARAGRAPHE000064"></a>Seule la dernière configuration est prise en compte lors de l'impression d'un état.

Il est déconseillé d'utiliser plusieurs fois les fonctions **iParamètreEtat** et [iConfigureEtat](../WDLang5/3046076.md) sans imprimer d'état.


<a name="NOTE0_2"></a>




### Ré-initialisation de la configuration
<a name="reinitialisation_configuration_ELTPARAGRAPHE000081"></a>Après l'appel à la fonction **iParamètreEtat** (ou [iConfigureEtat](../WDLang5/3046076.md)), tout nouvel appel à cette fonction ré-initialisera la fenêtre de configuration avec les paramètres utilisés lors du premier appel à la fonction.
<a name="NOTE0_3"></a>




### Nombre de copies
<a name="nombre_copies_ELTPARAGRAPHE000095"></a>Si un nombre de copies est spécifié par l'utilisateur (fonction [iParamètre](../WDLang5/3046008.md) ou fonction **iParamètreEtat**) avant de lancer l'impression de l'état, ce nombre de copies sera prioritaire sur celui spécifié avec la propriété [NbCopie](../Proprietes/2511042.md).




## &lt;Variable gglConnexion&gt;.ListeAgenda (Fonction)

***En anglais : &lt;gglConnection variable&gt;.ListCalendar***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Récupère la liste des agendas Google associés au compte Google spécifié. Cette fonction utilise le service Agenda Google Calendar.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connexion
Cnx est un gglConnexion
...
SI Cnx.Connecte() = Faux ALORS
	Erreur(ErreurInfo())
	RETOUR
FIN
tabAgendas est un tableau de gglAgenda 
tabAgendas = Cnx.ListeAgenda(gglTous) 
// Affichage du titre de chaque agenda
Agenda est un gglAgenda
POUR TOUT Agenda DE tabAgendas
	Trace(Agenda.Titre)
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Connexion Google>.ListeAgenda([<Filtre>])`
---

**`<Résultat> : (Tableau de variables de type gglAgenda)`**

Liste des agendas Google du compte spécifié.

**`<Connexion Google> : (Variable de type gglConnexion)`**

Nom de la variable de type [gglConnexion](../WDLang5/1000018009.md) à utiliser. Cette connexion a été validée avec la fonction [&lt;Variable gglConnexion&gt;.Connecte](../WDLang5/1000021455.md).

**`<Filtre> : (Constante de type Entier optionnelle)`**

Indique le type d'agenda à lister :


|   |   |
| --- | --- |
| *gglTous*<br>(Valeur par défaut) | Liste tous les agendas souscrits (partagés) et créés par l'utilisateur en cours. |
| *gglUtilisateur* | Liste uniquement les agendas créés par l'utilisateur en cours. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Cas particuliers
<a name="cas_particuliers_ELTPARAGRAPHE000199"></a>Les agendas renvoyés ne contiennent pas leurs événements. Pour obtenir les événements des agendas, utilisez les fonctions [&lt;Variable gglConnexion&gt;.RemplitAgenda](../WDLang5/1000017749.md) ou [&lt;Variable gglConnexion&gt;.RemplitAgendaParRequête](../WDLang5/1000017750.md).

Pour obtenir l'agenda avec les événements associés, utilisez la fonction [&lt;Variable gglConnexion&gt;.RécupèreAgenda](../WDLang5/1000017748.md).
<a name="NOTE0_2"></a>


### Cas d'erreurs possibles : 
<a name="cas_erreurs_possibles_ELTPARAGRAPHE000226"></a>

- La connexion Internet n'est pas valide.

- L'authentification n'a pas pu être réalisée correctement.

- La réponse XML du serveur n'est pas lisible.




Pour savoir si une erreur est survenue, utilisez la variable [ErreurDétectée](../WDLang1/3087001.md).
<a name="NOTE0_3"></a>


### Limite
<a name="limite_ELTPARAGRAPHE000243"></a>Seuls les 100 premiers agendas sont renvoyés.

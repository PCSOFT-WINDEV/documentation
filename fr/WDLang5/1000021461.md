


## &lt;Variable gglConnexion&gt;.Supprime (Fonction)

***En anglais : &lt;gglConnection variable&gt;.Delete***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime des données sur le serveur Google. Les données créées ou mises à jour sont contenues dans une variable correspondant au service manipulé :

- Variable de type [gglAgenda](../WDLang5/1000017418.md) pour supprimer un agenda.

- Variable de type [gglContact](../WDLang5/1000017434.md) pour supprimer un contact.

- Variable de type [gglGroupeContact](../WDLang5/1000017576.md) pour supprimer un groupe de contacts.





	<B><B><B>Avertissement</B></B></B>

	 A partir du 15 juin 2021, l'API Contacts de Google utilisée par cette fonctionnalité pour gérer les contacts et les groupes de contacts n'est plus disponible ([https://developers.google.com/contacts/v3/announcement](https://developers.google.com/contacts/v3/announcement)). 
Pour utiliser les contacts de Google, consultez l'API People proposée en remplacement par Google :  [https://developers.google.com/people](https://developers.google.com/people). 
La fonction [&lt;Variable restRequête&gt;.Envoie](../WDLang3/1410089811.md) simplifie l'intégration d'une API REST dans vos développements.

<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connexion
Cnx est un gglConnexion
...
SI Cnx.Connecte() = Faux ALORS 
	Erreur(ErreurInfo())
FIN

// Récupération de l'agenda "Travail"
tabAgenda est un tableau de 0 gglAgenda
tabAgenda = Cnx.ListeAgenda(gglUtilisateur)
// Affichage du titre de chaque agenda
Agenda est un gglAgenda
POUR TOUT Agenda DE tabAgenda
	SI Agenda.Titre = "TRAVAIL" ALORS
	 	// Suppression de l'agenda
	 	Cnx.Supprime(Agenda)
	FIN
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Connexion Google>.Supprime(<Elément Google>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la suppression a été réalisée, 

- <u><u><u><u>Faux</u></u></u></u> en cas d'erreur. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Connexion Google> : (Variable de type gglConnexion)`**

Nom de la variable de type [gglConnexion](../WDLang5/1000018009.md) à utiliser. Cette connexion a été validée avec la fonction [&lt;Variable gglConnexion&gt;.Connecte](../WDLang5/1000021455.md).

**`<Elément Google> : (Variable correspondant au type à supprimer)`**

Elément à supprimer. Ce paramètre peut correspondre à :

- Nom de la variable de type [gglAgenda](../WDLang5/1000017418.md) correspondant à l'agenda à supprimer.

- Nom de la variable [gglContact](../WDLang5/1000017434.md) correspondant au contact à supprimer.

- Nom de la variable de type [gglGroupeContact](../WDLang5/1000017576.md) correspondant au groupe de contacts à supprimer.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Suppression d'agenda
<a name="suppression_agenda_ELTPARAGRAPHE000232"></a>Pour supprimer un agenda, cet agenda doit avoir été récupéré auparavant avec la fonction [&lt;Variable gglConnexion&gt;.ListeAgenda](../WDLang5/1000017747.md) avec la constante *gglUtilisateur*. Dans le cas contraire, le serveur renvoie une erreur "InternalServerError".

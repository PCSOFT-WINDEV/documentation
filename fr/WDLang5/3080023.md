


## xlsNbFeuille (Fonction)

***En anglais : xlsNbWorksheet***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le nombre de feuilles d'un fichier XLS.

**Remarque** : En mode compatible, le nombre de feuilles est toujours égal à 1.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
IdentifiantFichier est un entier

// Ouverture d'un fichier XLS
IdentifiantFichier = xlsOuvre("C:\Mes documents\MonFichier.XLS")
SI IdentifiantFichier <> -1 ALORS
	// Récupération du nombre de feuilles du fichier
	Info("Le fichier contient " + xlsNbFeuille(IdentifiantFichier) + " feuille(s).")
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Syntaxe utilisant le type xlsDocument

`<Résultat> = xlsNbFeuille(<Document XLS>)`
---

**`<Résultat> : (Entier)`**



- Nombre de feuilles du fichier XLS spécifié, 

- -1 en cas d'erreur. Pour connaître le libellé de l'erreur, utilisez la fonction [xlsMsgErreur](../WDLang5/3080016.md) ou la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.




**`<Document XLS> : (Variable de type xlsDocument)`**

Nom de la variable de type [xlsDocument](../WDLang5/1000017464.md) correspondant au fichier XLS à fermer. Le fichier physique associé à cette variable (s'il existe) est fermé. Les modifications réalisées dans le fichier ne sont pas sauvées avant fermeture (utilisez la fonction [xlsSauve](../WDLang5/1000018057.md) si nécessaire).  


<a name="SYNTAXE2"></a>

### Syntaxe conservée par compatibilité avec la version 12

`<Résultat> = xlsNbFeuille(<Identifiant du fichier XLS>)`
---

**`<Résultat> : (Entier)`**



- Nombre de feuilles du fichier XLS spécifié, 

- -1 en cas d'erreur. Pour connaître le libellé de l'erreur, utilisez la fonction [xlsMsgErreur](../WDLang5/3080016.md) ou la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.




**`<Identifiant du fichier XLS> : (Entier)`**

Identifiant du fichier XLS à manipuler, renvoyé par la fonction [xlsOuvre](../WDLang5/3080017.md).

## Remarques


### Gestion des feuilles de type Graphique
<a name="gestion_des_feuilles_type_graphique_ELTPARAGRAPHE000076"></a>Les feuilles Excel de type "Graphiques" sont ignorées. En effet, ces feuilles ne contiennent pas de cellules et ne sont donc pas accessibles.

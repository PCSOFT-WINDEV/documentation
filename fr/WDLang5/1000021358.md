
## mlChargeEspaceDeTravail (Fonction)

***En anglais : mlLoadWorkspace***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Charge l'ensemble des variables d'un fichier MATLAB dans la session. L'espace de travail (workspace) de la session MatLab est initialisé en chargeant l'ensemble des variables d'un fichier (fichier au format Mat-File d'extension .MAT).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
nIdSession est un entier = mlInitialise()
SI ErreurDétectée ALORS
	ErreurInfo(errComplet)
	RETOUR
FIN
mlChargeEspaceDeTravail(nIdSession, "C:\temp\workspace.mat")
sListeVar est une chaîne = mlListeVariable(nIdSession)
mlTermine(nIdSession)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = mlChargeEspaceDeTravail(<Identifiant de la session> , <Nom du fichier> , <Variables conservées>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le chargement a réussi.

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. En cas d'erreur, la variable [ErreurDétectée](../WDLang1/3087001.md) est à <u><u><u><u>Vrai</u></u></u></u>. Pour connaître le libellé de l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.




**`<Identifiant de la session> : (Entier)`**

Identifiant de la session MATLAB. Cet identifiant est renvoyé par la fonction [mlInitialise](../WDLang5/1000021351.md). 

**`<Nom du fichier> : (Chaîne de caractères)`**

Chemin et le nom du fichier Mat-File à manipuler. 

**`<Variables conservées> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> pour conserver les variables existantes. La variable existante est conservée sauf si elle est redéfinie dans le fichier. Dans ce cas, la variable de la session est écrasée.

- <u><u><u><u>Faux</u></u></u></u> pour ne pas les conserver (valeur par défaut). Les variables présentes dans la session sont supprimées avant le chargement.

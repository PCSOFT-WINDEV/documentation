


## iHauteurTexte (Fonction)

***En anglais : iTextHeight***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Calcule la hauteur de la police du texte à imprimer (en millimètres).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
x est un entier = iHauteurTexte(iPolice(2) + CL.NOMCLI)
iImprimeMot("La hauteur du nom du client est égale à : ")
iImprime(x)
iFinImprime()
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = iHauteurTexte([<Identifiant de la police> + ] <Texte à imprimer>)`
---

**`<Résultat> : (Réel)`**



- Hauteur (en millimètres) du texte spécifié si l'impression est réalisée, 

- -1 si l'impression a été abandonnée avant l'exécution de la fonction.




**`<Identifiant de la police> : (Chaîne de caractères optionnelle)`**

Identifiant de la police utilisée, renvoyé par la fonction [iPolice](../WDLang5/3046014.md). Si ce paramètre n'est pas spécifié, la dernière police utilisée sera sélectionnée.

**`<Texte à imprimer> : (Chaîne de caractères)`**

Chaîne de caractères dont la hauteur est recherchée.

Attention : une hauteur est renvoyée même si le texte à imprimer correspond à une chaîne vide.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Calculer la hauteur d'un texte écrit avec des polices de tailles différentes
<a name="calculer_hauteur_texte_ecrit_avec_des_polices_tailles_differentes_ELTPARAGRAPHE000065"></a>Il est nécessaire d'utiliser la fonction **iHauteurTexte** pour chaque texte utilisant une police différente.


### Calculer la hauteur d'un texte contenant des retours chariot (RC)
<a name="calculer_hauteur_texte_contenant_des_retours_chariot_ELTPARAGRAPHE000073"></a>La fonction **iHauteurTexte** ne tient pas compte des retours chariot (RC) présents dans le texte. Il est nécessaire de compter le nombre de RC afin d'ajouter autant de fois la hauteur du texte pour obtenir la hauteur totale.

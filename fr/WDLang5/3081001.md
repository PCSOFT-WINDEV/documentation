


## XMLAnnuleRecherche (Fonction)

***En anglais : XMLCancelSearch***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Annule la recherche effectuée par la fonction [XMLRecherche](../WDLang5/3081025.md).








<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Recherche de la balise "<prix>"
XMLRecherche("DocXML", "prix", XMLBalise)
TANTQUE XMLTrouve("DocXML") = Vrai
	Info("Balise trouvée " + XMLNomElément("DocXML"))
	XMLSuivant("DocXML")
FIN

// Annule la recherche en cours
XMLAnnuleRecherche("DocXML")

// Reprise d'un parcours standard
XMLPremier("DocXML")
TANTQUE XMLEnDehors("DocXML") = Faux
	// Opérations
	XMLSuivant("DocXML")
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`XMLAnnuleRecherche(<Document XML>)`
---

**`<Document XML> : (Chaîne de caractères)`**

Nom du document XML dans lequel la recherche doit être annulée. Ce document contient le code XML et a été créé avec la fonction [XMLDocument](../WDLang5/3081008.md).

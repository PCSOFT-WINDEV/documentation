


## &lt;Variable gglConnexion&gt;.ListeContactParRequête (Fonction)

***En anglais : &lt;gglConnection variable&gt;.ListContactByRequest***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Récupère une liste de contacts à partir de paramètres personnalisés.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connexion
Cnx est un gglConnexion
...
SI Cnx.Connecte() = Faux ALORS
	Erreur(ErreurInfo())
FIN
...
// Récupère tous les contacts mis à jour depuis le 08/08/2008
tabContacts est un tableau de 0 gglContact
tabContacts = Cnx.ListeContactParRequête("updated-min=2008-08-08")
// Récupère les 10 premiers contacts mis à jour depuis le 14 juillet 1789
// par ordre décroissant
tabContacts = Cnx.ListeContactParRequête(...
		"updated-min=1789-07-14&start-index=1&max-results=10")
// Récupère les contacts d'un groupe (spécifié par son identifiant)
// par ordre décroissant de date de mise à jour
Identifiant est une chaîne
Identifiant = "http://www.google.com/feeds/contacts/groups/jo%40gmail.com/base/1234a"
tabContacts = Cnx.ListeContactParRequête("sortorder=descending&group=" + Identifiant)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Connexion Google>.ListeContactParRequête(<Paramètres>)`
---

**`<Résultat> : (Tableau de variables gglContact)`**

Tableau de variables de type [gglContact](../WDLang5/1000017434.md).

**`<Connexion Google> : (Variable de type gglConnexion)`**

Nom de la variable de type [gglConnexion](../WDLang5/1000018009.md) à utiliser. Cette connexion a été validée avec la fonction [&lt;Variable gglConnexion&gt;.Connecte](../WDLang5/1000021455.md).

**`<Paramètres> : (Chaîne de caractères)`**

Paramètres à passer à la requête HTTP. Les paramètres sont séparés par le caractère &. Pour plus de détails sur les paramètres possibles, consultez la documentation sur le service Google Contacts.

## Remarques


### Cas d'erreurs possibles : 
<a name="cas_erreurs_possibles_ELTPARAGRAPHE000181"></a>

- La connexion Internet n'est pas valide.

- L'authentification n'a pas pu être réalisée correctement.

- La réponse XML du serveur n'est pas lisible.




Pour savoir si une erreur est survenue, utilisez la variable [ErreurDétectée](../WDLang1/3087001.md).


## xlsColonne (Type de variable)

***En anglais : xlsColumn ***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **xlsColonne** permet de décrire et de modifier une colonne d'un document Excel. Les caractéristiques de cette colonne peuvent être définies et modifiées à l'aide de différentes propriétés WLangage.

Ce type de variable est utilisé par le type [xlsDocument](../WDLang5/1000017464.md). 

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).








<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ouverture d'un fichier Excel.
sFichierExcel est une chaîne
sFichierExcel = fSélecteur("", "", "Sélectionnez un fichier Excel...", ...
	"Excel" + TAB + "*.xlsx" + RC + "Tous les fichiers (*.*)" + TAB + "*.*", "*.xlsx")
SI sFichierExcel = "" ALORS RETOUR

// Ouverture du document Excel.
MonDocument est un xlsDocument
MonDocument = xlsOuvre(sFichierExcel)

// Cellule par défaut.
Cellule est un xlsCellule
Cellule.CouleurFond = RougeClair

// Variable xlsColonne
Colonne est un xlsColonne
Colonne.Largeur = 200
Colonne.CelluleDéfaut = Cellule

// La variable Colonne est utilisée pour la colonne 2 du document.
MonDocument..Colonne[2] = Colonne

// Sauvegarde les modifications.
xlsSauve(MonDocument)
xlsFerme(MonDocument)
```

## Remarques




### Propriétés spécifiques à la description des variables de type xlsColonne
<a name="proprietes_specifiques_description_des_variables_type_xlscolonne_ELTPARAGRAPHE000050"></a>Les propriétés suivantes peuvent être utilisées pour manipuler une variable de type **xlsColonne** :

| Nom | Type manipulé | Effet |
| --- | --- | --- |
| Cellule | Tableau de [xlsCellule](../WDLang5/1000017472.md) | Cellules de la colonne. |
| CelluleDéfaut | xlsCellule | Le style par défaut de la colonne correspondra au style de la cellule spécifiée. |
| Largeur | Réel | Largeur de la colonne en pixels (entre 0 et 1789 pixels). Si cette propriété est initialisée à -1, la largeur par défaut des colonnes du document Excel sera utilisée. |




## XMLPosition (Fonction)

***En anglais : XMLPath***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la position en cours dans le document sous la forme XPATH.








<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
Res est une chaîne
Res = XMLPosition("MonDocument")
// Res est de la forme : /COMMANDE/LIGNECDE[2]/@NUMERO sur un attribut
// Res est de la forme : /COMMANDE/LIGNECDE[1] sur une balise fille
// Res est de la forme : /COMMANDE sur une balise 
// Attention, la fonction XMLPosition peut également retourner une chaîne
// de cette forme (valide syntaxiquement) : /*/*[8]/*[1]
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = XMLPosition(<Document XML>)`
---

**`<Résultat> : (Chaîne de caractères)`**

Position de l'élément en cours, sous la forme XPATH.

**`<Document XML> : (Chaîne de caractères)`**

Nom du document XML manipulé. Ce document contient le code XML à analyser et a été créé avec la fonction [XMLDocument](../WDLang5/3081008.md).

## Remarques


### Fonctions XML et threads
<a name="fonctions_xml_threads_ELTPARAGRAPHE000198"></a>Si votre application utilise des threads, le document XML est partagé entre tous ces threads. Pour plus de détails sur la gestion des threads, consultez [Gestion des threads](../WDLang1/3077026.md).

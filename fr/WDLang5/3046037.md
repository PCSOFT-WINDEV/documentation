


## Etape 1 : Configuration des paramètres d'impression
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000176"></a>
Cette étape permet de choisir selon le cas :

- les paramètres d'impression du document (qualité d'impression, nombre de copies, ...).

- les paramètres de configuration de l'imprimante (driver imprimante, orientation, format papier, ...).




**Cette étape est facultative**.

Cette étape doit être effectuée uniquement si les paramètres par défaut (paramètres définis dans le panneau de configuration de Windows) ne conviennent pas pour l'impression à lancer.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Les différentes manipulations possibles
<a name="les_differentes_manipulations_possibles_ELTTEXTE000200"></a>
Les différentes manipulations possibles sont les suivantes :

**1. Sélection des paramètres d'impression du document** (fonction [iParamètre](../WDLang5/3046008.md)). Il est possible :

- soit d'ouvrir la fenêtre de paramétrage de l'impression.

- soit de configurer un à un les paramètres de l'impression (par exemple la hauteur et la largeur de la page, le nombre de copies, ...).






**2. Choix de l'imprimante utilisée pour l'impression** (fonction [iConfigure](../WDLang5/3046072.md)). Il est possible :

- soit d'ouvrir la fenêtre de sélection de l'imprimante.

- soit de modifier directement l'imprimante utilisée pour l'impression en cours. Cette modification peut être temporaire (uniquement pour le programme en cours) ou définitive.




![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) La fonction [iConfigure](../WDLang5/3046072.md) n'est pas disponible.

**3. Re-définition des marges d'impression** (fonction [iMarge](../WDLang5/3046011.md)). Il est possible de modifier les marges basse, haute, gauche et droite. Si cette fonction n'est pas appelée, les marges par défaut (différentes selon l'imprimante sélectionnée) sont utilisées.

**Attention** : L'ordre des manipulations présentées ci-dessus doit être respecté. Si une impression est en cours, les fonctions [iParamètre](../WDLang5/3046008.md) et [iConfigure](../WDLang5/3046072.md) provoquent la fermeture de l'impression en cours.

## Fonctions permettant de configurer la destination de l'impression
<a name="fonctions_permettant_configurer_destination_impression_ELTTEXTE000224"></a>
Les fonctions suivantes permettent de configurer la destination de l'impression, et de connaître la configuration en cours :


|   |   |
| --- | --- |
| [iAperçu](../WDLang5/3046069.md) / [iDestination](../WDLang5/3046074.md) | Paramètre la destination de l'impression. |
| [iConfigure](../WDLang5/3046072.md) | Configure l'imprimante. |
| [iInfoImprimante](../WDLang5/3046022.md) | Récupère les caractéristiques de l'imprimante en cours ou de l'imprimante par défaut. |
| [iListeImprimante](../WDLang5/3046063.md) | Renvoie la liste des imprimantes installées sur le poste en cours. |
| [iParamètre](../WDLang5/3046008.md) | Paramètre l'impression. |

La configuration et le paramétrage réalisé par les fonctions **iConfigure** et **iParametre** ré-initialisent le module d'impression. En conséquence, ces fonctions doivent être les premières fonctions d'impression appelées. Toutes les fonctions d'impression appelées avant ces deux fonctions ne seront pas prises en compte.

![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) L'impression peut être réalisée uniquement sur une imprimante.

![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) L'impression peut être réalisée uniquement au format PDF.

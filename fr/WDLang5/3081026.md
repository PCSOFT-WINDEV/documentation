


## XMLNomElément (Fonction)

***En anglais : XMLElementName***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le nom de l'élément en cours (balise ou attribut). Pour sélectionner un élément, utilisez les fonctions de recherche et de parcours XML ([XMLPremier](../WDLang5/3081024.md), [XMLDernier](../WDLang5/3081013.md), [XMLFils](../WDLang5/3081012.md), ...).








<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
NomBalise est une chaîne
// Récupère le nom de l'élément en cours
NomBalise = XMLNomElément("DocXML")

// Positionné sur un élément ?
SI NomBalise = "" ALORS
	Erreur("Aucune balise en cours")
SINON
	Info("Le nom de l'élément courant est : " + NomBalise)
FIN

// Autre possibilité : 

// Positionné sur un élément ?
SI ErreurDétectée = Vrai ALORS
	Erreur("Aucune balise en cours")
SINON
	Info("Le nom de l'élément courant est : " + NomBalise)
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = XMLNomElément(<Document XML>)`
---

**`<Résultat> : (Chaîne de caractères)`**



- Nom de l'élément en cours.

- Chaîne vide ("") si aucun élément n'est en cours. Dans ce cas, la variable [ErreurDétectée](../WDLang1/3087001.md) est à <u><u><u><u>Vrai</u></u></u></u>. Pour obtenir plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Document XML> : (Chaîne de caractères)`**

Nom du document XML manipulé. Ce document contient le code XML à analyser et a été créé avec la fonction [XMLDocument](../WDLang5/3081008.md).

## Remarques


### Connaître l'élément courant 
<a name="connaitre_element_courant_ELTPARAGRAPHE000090"></a>

- Le type de l'élément en cours peut être connu avec la fonction [XMLTypeElément](../WDLang5/3081031.md).

- La valeur de l'élément en cours peut être connue avec la fonction [XMLDonnée](../WDLang5/3081004.md).



<a name="NOTE0_2"></a>


### Fonctions XML et threads
<a name="fonctions_xml_threads_ELTPARAGRAPHE000231"></a>Si votre application utilise des threads, le document XML est partagé entre tous ces threads. Pour plus de détails sur la gestion des threads, consultez [Gestion des threads](../WDLang1/3077026.md).

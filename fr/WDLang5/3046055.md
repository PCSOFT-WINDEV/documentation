
## iSelectPolice (Fonction)

***En anglais : iSelectFont***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Crée une nouvelle police d'impression depuis la fenêtre standard de sélection des polices de Windows :

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=iselectpolice.gif)


Cette police pourra être utilisée dans les différents traitements. Cette police est référencée par un numéro de police, choisi lors du développement. Pour choisir cette police et l'utiliser dans les diverses fonctions d'impression, il suffira ensuite de la sélectionner avec la fonction [iPolice](../WDLang5/3046014.md).

**Remarque** : Cette fonction doit IMPÉRATIVEMENT être appelée APRÈS la fonction [iAperçu](../WDLang5/3046069.md). Dans le cas contraire, les aperçus seront ignorés.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
NumPol est un entier
NumPol = iSelectPolice(1, Vrai, "Arial", 8, iGras + iSouligné, RougeClair)
SI NumPol = 0 ALORS
	TraiteErreur()
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = iSelectPolice(<Numéro de police> [, <Options avancées> [, <Police pré-sélectionnée> [, <Hauteur pré-sélectionnée>]]] , <Style> [, <Couleur>])`
---

**`<Résultat> : (Entier)`**



- Identifiant de la police sélectionnée, 

- 0 en cas d'échec (annulation de la sélection par exemple).




**`<Numéro de police> : (Entier)`**

Identifiant de la police. Cet identifiant sera utilisé pour sélectionner une police grâce à la fonction [iPolice](../WDLang5/3046014.md).

**`<Options avancées> : (Booléen optionnel)`**



- <u><u><u><u>Vrai</u></u></u></u> (option par défaut) : active l'affichage du choix du soulignement et du choix des couleurs. 

- <u><u><u><u>Faux</u></u></u></u> : ne prend pas en compte les options avancées.




**`<Police pré-sélectionnée> : (Chaîne de caractères optionnelle)`**

Nom de la police sélectionnée par défaut dans la fenêtre de sélection des polices. Si ce paramètre n'est pas spécifié, aucune police n'est pré-sélectionnée.

**`<Hauteur pré-sélectionnée> : (Entier optionnel)`**

Hauteur de la police sélectionnée par défaut dans la fenêtre de sélection des polices. Si ce paramètre n'est pas spécifié, aucune hauteur n'est sélectionnée par défaut.

**`<Style> : (Constante ou combinaison de constantes de type entier)`**

Attributs de la police : 


|   |   |
| --- | --- |
| *iBarre* | Police barrée |
| *iCondense* | Caractères condensés |
| *iEtendu* | Caractères étendus |
| *iGras* | Police en gras |
| *iItalique* | Police italique |
| *iLarge* | Caractères larges |
| *iNormal* <br>(valeur par défaut) | Police normale, sans attributs |
| *iSouligné* | Police soulignée |



**`<Couleur> : (Entier optionnel)`**

Couleur utilisée pour la police. Si ce paramètre n'est pas précisé, la couleur utilisée est le noir. 
Cette couleur peut correspondre à :

- une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)),

- une couleur TSL (obtenue avec la fonction [TSL](../WDLang1/3029057.md)),

- [une couleur prédéfinie du WLangage](../WDLang5/3010002.md).  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Police par défaut
<a name="police_par_defaut_ELTPARAGRAPHE000117"></a>La police par défaut a les caractéristiques suivantes : 

- Police numéro 0, 

- Taille : 12, 

- Style normal.





<a name="NOTE0_2"></a>




### Obtenir la liste des polices installées sur un poste
<a name="obtenir_liste_des_polices_installees_sur_poste_ELTPARAGRAPHE000130"></a>Pour obtenir la liste des polices installées sur une machine :

1. Dans le gestionnaire des programmes de Windows, choisissez le groupe principal.

2. Cliquez sur l'icône "Panneau de configuration", puis choisissez "Polices".

3. Dans la liste des polices installées, notez les noms sans tenir compte des attributs ni des types de police.




Remarques :

- La fonction [PoliceListe](../WDLang1/3060003.md) permet d'obtenir la liste des polices installées sur un poste.

- La fonction [iSelectPolice](../WDLang5/3046055.md) permet de créer une police d'impression à partir de la fenêtre standard de sélection des polices de Windows.





<a name="NOTE0_3"></a>




### Divers
<a name="divers_ELTPARAGRAPHE000154"></a>

- La fonction [iCréePolice](../WDLang5/3046045.md) permet également de créer des polices.

- Il est nécessaire de créer autant de polices que le texte comporte de couleurs.

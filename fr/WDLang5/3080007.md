 

## xlsNbColonnes (Fonction)

***En anglais : xlsNbColumns***
	

<a name="XUtilisation"></a>
<a name="Utilisation"></a>
|  |   |
| --- | --- |



|  
 | **But**<br><br>Obtenir le nombre de colonnes du fichier XLS.<br><br>**Syntaxe (utilisant le type xlsDocument)**<br><br>&lt;Résultat&gt; = xlsNbColonnes(&lt;Document XLS&gt;)<br><br>- &lt;Résultat&gt; est un entier, correspondant au nombre de colonnes.<br><br>- &lt;Document XLS&gt; est le nom de la variable de type [xlsDocument](../WDLang5/1000017464.md) correspondant au fichier XLS à manipuler.<br><br><br><br><br>**Syntaxe (compatibilité version 12)**<br><br>&lt;Résultat&gt; = xlsNbColonnes(&lt;NumFichier&gt;)<br><br>- &lt;Résultat&gt; est un entier, correspondant au nombre de colonnes. <br><br>- &lt;NumFichier&gt; est un entier contenant l'identifiant du fichier fourni par xlsOuvre<br><br><br><br><br>**Détail**<br><br>xlsNbColonnes retourne le nombre de colonnes du fichier XLS. Les colonnes vides ne sont pas prises en compte. Par exemple, dans le cas suivant, la fonction xlsNbColonnes retourne 4 :<br><br><br>\   \   \   \   \   \<br>\ --- \ --- \ --- \ --- \ --- \<br>\ Nom \ Prénom \   \ Age \ Profession \<br>\ Durand \ Paul \   \ 18 \ Etudiant \<br>\ Dupond \ Jacques \   \ 19 \ Etudiant \<br>\ Martin \ Jean \   \ 20 \ Etudiant \<br><br><br><br><br><br>**Notes**<br><br>Cette fonction suppose que le fichier XLS possède une structure de type base de données, c'est-à-dire qu'à chaque ligne correspond un nombre de colonnes fixe.<br><br> |

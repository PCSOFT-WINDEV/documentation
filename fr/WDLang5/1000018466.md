


## SFVideCorbeille (Fonction)

***En anglais : SFEmptyRecycleBin***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Vide les enregistrements spécifiés de la corbeille Salesforce.

**Remarque** : Au bout de 30 jours, les éléments présents dans la corbeille Salesforce sont automatiquement supprimés de la corbeille.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connexion à Salesforce
Cnx est une sfConnexion
Cnx..Login = "balthazar@gmail.com"
Cnx..MotDePasse = "azerty" + "XXB12VCZ54"
SI SFConnecte(Cnx) = Faux ALORS
	Erreur("La connexion a échoué")
	RETOUR
FIN
ResRequête est un sfRésultatExécuteRequête 
ResRequête = SFExécuteRequête(Cnx, "SELECT Date__c,Miles__c,Contact__c FROM Mileage__c")

MonEnregistrement est un sfObjet
POUR TOUT MonEnregistrement DE ResRequête..Objet
	Trace("---- Type : " + MonEnregistrement..Type + " | ID : " + MonEnregistrement..ID)
	Trace(MonEnregistrement..Date__c + " / " + MonEnregistrement..Miles__c + " / " + ...
		MonEnregistrement..Contact__c)
FIN

// Suppression d'un objet
tabSuppr est un tableau de chaînes
Ajoute(tabSuppr, ResRequête..Objet[1]..ID)

tabResSuppr est un tableau de SFRésultatSupprime
tabResSuppr = SFSupprime(Cnx, tabSuppr)
POUR TOUT res DE tabResSuppr
	Trace("ID : " + res..ID + " - Suppression = " + res..Succès)
FIN

tabResVideCorb est un tableau de SFRésultatVideCorbeille
tabResVideCorb = SFVideCorbeille(Cnx, tabSuppr)
POUR TOUT res DE tabResVideCorb
	Trace("ID : " + res..ID + "- VideCorbeille  = " + res..Succès)
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = SFVideCorbeille(<Connexion Salesforce> , <Enregistrements à supprimer>)`
---

**`<Résultat> : (Tableau de sfRésultatVideCorbeille)`**

Tableau de variables de type [sfRésultatVideCorbeille](../WDLang5/1000018467.md) contenant les enregistrements supprimés de la corbeille.

**`<Connexion Salesforce> : (Variable de type sfConnexion)`**

Nom de la variable de type [sfConnexion](../WDLang5/1000017579.md) à utiliser.

**`<Enregistrements à supprimer> : (Variable de type Tableau)`**

Tableau de chaînes de caractères contenant les identifiants Salesforce des enregistrements à supprimer de la corbeille.

## Remarques


### Cas d'erreurs possibles : 
<a name="cas_erreurs_possibles_ELTPARAGRAPHE000078"></a>

- La connexion Internet n'est pas valide.

- L'authentification n'a pas pu être réalisée correctement.

- La réponse XML du serveur n'est pas lisible.




Pour savoir si une erreur est survenue, utilisez la variable [ErreurDétectée](../WDLang1/3087001.md).

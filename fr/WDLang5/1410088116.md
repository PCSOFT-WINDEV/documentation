


## &lt;Variable htmlNoeud&gt;.SupprimeFils (Fonction)

***En anglais : &lt;htmlNode variable&gt;.RemoveChild***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime un noeud donné dans le noeud courant d'un document HTML.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
hDoc est un htmlDocument = "
<html>
	<head/>
	<body>
			<h1/>
			<h2>
			<span>
			<p>Texte</p>
			</span>
			</h2>
	</body>
</html>
"

hNoeud est un htmlNoeud
hNoeudRecherche est un tableau de htmlNoeuds 

hNoeudRecherche <- hDoc.ChercheElémentParBalise("body")
hNoeud <- hNoeudRecherche[1]
hNoeudRecherche <- hNoeud.ChercheElémentParBalise("h1")
hNoeudFils est un htmlNoeud <- hNoeudRecherche[1]
hNoeud.SupprimeFils(hNoeudFils)
// body : <h2><span><p>Texte</p></span></h2>
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Noeud source>.SupprimeFils(<Fils à supprimer>)`
---

**`<Noeud source> : (Variable de type htmlNoeud)`**

Nom de la variable de type [htmlNoeud](../WDLang5/1000026081.md) correspondant au noeud de référence pour la suppression du fils.

**`<Fils à supprimer> : (Variable de type htmlNoeud)`**

Nom de la variable de type [htmlNoeud](../WDLang5/1000026081.md) correspondant au noeud fils à supprimer.

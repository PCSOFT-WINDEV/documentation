
## rssAffiche (Fonction)

***En anglais : rssDisplay***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Construit un flux RSS et renvoie le contenu du flux RSS au client. Le résultat est affiché sur le navigateur de l'internaute. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonFluxRSS est un rssFlux
MonFluxRSS = rssInitialise("http://www.monsite.fr/feeds/rss/actualites/", depuisURL)
...
rssAffiche(MonFluxRSS)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`rssAffiche(<Flux RSS à afficher> [, <Encodage>])`
---

**`<Flux RSS à afficher> : (Variable de type rssFlux)`**

Nom de la variable de type [rssFlux](../WDLang5/1000017795.md) à utiliser.

**`<Encodage> : (Constante optionnelle de type Entier)`**

Indique l'encodage à utiliser dans le fichier XML correspondant au flux :


|   |   |
| --- | --- |
| *XMLEncodageAucun* | Aucun encodage |
| *XMLEncodageIso8859_1* | Encodage selon l'alphabet latin |
| *XMLEncodageIso8859_2* | Encodage selon Iso8859_2 |
| *XMLEncodageIso8859_3* | Encodage selon Iso8859_3 |
| *XMLEncodageIso8859_4* | Encodage selon Iso8859_4 |
| *XMLEncodageIso8859_5* | Encodage selon Iso8859_5 |
| *XMLEncodageIso8859_6* | Encodage selon Iso8859_6 |
| *XMLEncodageIso8859_7* | Encodage selon Iso8859_7 |
| *XMLEncodageIso8859_8* | Encodage selon Iso8859_8 |
| *XMLEncodageIso8859_9* | Encodage selon Iso8859_9 |
| *XMLEncodageIso8859_15* | Encodage selon l'alphabet latin-9 |
| *XMLEncodageUTF8*(valeur par défaut) | Encodage UNICODE 8 |
| *XMLEncodageUTF16* | Encodage UNICODE |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Le type MIME utilisé est "application/xml".

- L'encodage utilisé pour la création du fichier XML est automatiquement écrit dans les entêtes HTTP.

- Si votre projet utilise les sessions prélancées, cette fonction ne doit pas être utilisée dans l'événement "Initialisation" du projet. Cette fonction doit être utilisée dans l'événement "Initialisation du projet après connexion au site".


## rssEntrée (Type de variable)

***En anglais : rssEntry ***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **rssEntrée** permet de manipuler une entrée d'un flux RSS. Le flux RSS est décrit grâce au type [rssFlux](../WDLang5/1000017795.md).

Les caractéristiques d'une entrée du flux peuvent être définies et modifiées à l'aide de différentes propriétés WLangage.  

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonFlux est un rssFlux
MonCanal est un rssCanal
MonEntrée est un rssEntrée
MonChemin est une chaîne

// Renseignements du canal
MonCanal.Titre = "Mon site"
MonCanal.Description = "Exemple de flux RSS 2.0"
MonCanal.DateMiseAJour = DateSys() + HeureSys()
MonCanal.Lien = "http://www.example.org"
 
// Renseignements des entrées du canal 1
MonEntrée.Auteur = "pcsoft@pcsoft.fr"
MonEntrée.Description = "Première actualité"
MonEntrée.Titre = "Actualité N°1"

// Ajoute l'entrée dans le canal
Ajoute(MonCanal.Entrée, MonEntrée)

// Ajoute le canal dans le flux
Ajoute(MonFlux.Canal, MonCanal)


// Sauve le flux dans un fichier
MonChemin = ComplèteRep(fRepEnCours()) + "Rss.xml"
rssSauve(MonFlux, MonChemin)

// Affiche le fichier XML du flux
SI PAS LanceAppliAssociée(MonChemin) ALORS
	Erreur("LanceAppliAssociée('" + MonChemin + "'). '" + ErreurInfo() + "'")
FIN
```

## Remarques




### Propriétés spécifiques à la description des variables de type rssEntrée
<a name="proprietes_specifiques_description_des_variables_type_rssentree_ELTPARAGRAPHE000059"></a>Les propriétés suivantes peuvent être utilisées pour manipuler une entrée d'un flux RSS grâce à la variable de type **rssEntrée** :

   | --- |
| --- | --- | --- |
| Propriété | Type manipulé | Effet |
| Auteur | Chaîne de caractères | Adresse email de l'auteur de l'entrée. |
| Catégorie | [rssCatégorie](../WDLang5/1000017782.md) | Catégorie à laquelle appartient l'entrée. Peut être utilisée pour agréger des éléments sur le même sujet. |
| Commentaires | Chaîne de caractères | URL de la page contenant les commentaires sur l'entrée. |
| DatePublication | DateHeure | Date de publication du contenu de l'entrée. <br>Les dates lues sont automatiquement converties en date UTC. La valeur écrite dans le canal est toujours en UTC. |
| Description (\*) | Chaîne de caractères | Contenu de l'entrée. <br>Les balises HTML sont autorisées. |
| DocumentJoint | [rssDocumentJoint](../WDLang5/1000017789.md) | Décrit un document externe (généralement multimédia) associé à une entrée. |
| Identifiant | [rssIdentifiant](../WDLang5/1000017797.md) | Identifiant (potentiellement unique et permanent) de l'entrée. Souvent utilisé pour identifier les entrées déjà lues / traitées. |
| Lien | Chaîne de caractères | Lien externe vers l'élément décrit par l'entrée. |
| Source | [rssSource](../WDLang5/1000017803.md) | Description du flux de publication original de l'entrée. En cas d'agrégation, cette valeur est différente de l'URL du canal qui contient actuellement l'entrée. |
| Titre (\*) | Chaîne de caractères | Titre du Canal. <br>Il est possible de connaître et de modifier la valeur de cette propriété. |


**Remarques** : 

- Les propriétés suivies d'une astérisque (\*) sont obligatoires.

- Toutes les propriétés de cette liste sont disponibles en lecture et en écriture.

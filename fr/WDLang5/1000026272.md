


## HTMLChercheElémentParBalise (Fonction)

***En anglais : HTMLFindElementByTag***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Cherche les éléments d'un document HTML (ou d'un noeud HTML) correspondant à une balise.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
d est un htmlDocument = HTMLOuvre("https://www.pcsoft.fr", depuisURL)
// ...
t est un tableau de htmlNoeud <- HTMLChercheElémentParBalise(d, "form")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = HTMLChercheElémentParBalise(<Elément à manipuler> [, <Elément recherché>])`
---

**`<Résultat> : (Tableau)`**

Tableau d'éléments de type [htmlNoeud](../WDLang5/1000026081.md) correspondant à la recherche. 

**`<Elément à manipuler> : (Variable de type htmlDocument ou de type htmlNoeud)`**

Elément dans lequel la recherche doit être effectuée : 

- Nom de la variable de type [htmlDocument](../WDLang5/1000025910.md). 

- Nom de la variable de type [htmlNoeud](../WDLang5/1000026081.md).




**`<Elément recherché> : (Chaîne de caractères optionnelle)`**

Nom de la balise à rechercher. Si ce paramètre n'est pas précisé ou correspond à une chaîne vide (""), &lt;Résultat&gt; contiendra tous les éléments.

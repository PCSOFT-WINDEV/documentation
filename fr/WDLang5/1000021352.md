
## mlTermine (Fonction)

***En anglais : mlEnd***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Termine une session MATLAB. Cette session a été initialisée avec la fonction [mlInitialise](../WDLang5/1000021351.md). 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
nIdSession est un entier = mlInitialise()
SI ErreurDétectée() ALORS
	ErreurInfo(errComplet)
	RETOUR
FIN
mlExécute(nIdSession, "A = cosd(180)")
... 
mlTermine(nIdSession)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`mlTermine(<Identifiant de la session>)`
---

**`<Identifiant de la session> : (Entier)`**

Identifiant de la session MATLAB à manipuler. Cet identifiant est renvoyé par la fonction [mlInitialise](../WDLang5/1000021351.md).

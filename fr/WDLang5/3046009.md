
## iEllipse (Fonction)

***En anglais : iEllipse***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Imprime une ellipse (à l'intérieur d'un rectangle).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple
<a class="notetitle" target="_blank" href="$DOC$=1000003046009&name=iellipse_fonction&product=WD">Voir des exemples supplémentaires</a>

```wl
// Impression d'une ellipse
iEllipse(50, 50, 150, 130, 1, 100, RVB(0,0,0), iPointillé, iCroisée)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`iEllipse(<X1> , <Y1> , <X2> , <Y2> [, <Épaisseur> [, <Luminosité> [, <Couleur> [, <Trait> [, <Brosse>]]]]])`
---

**`<X1> : (Réel)`**

Coordonnée horizontale (colonne ou abscisse) du coin haut gauche du rectangle dans lequel l'ellipse va être dessinée (en millimètres). 
Valeur comprise entre 0 et la largeur de la page (donnée par la fonction [iLargeurPage](../WDLang5/3046036.md)).

**`<Y1> : (Réel)`**

Coordonnée verticale (ligne ou ordonnée) du coin haut gauche du rectangle dans lequel l'ellipse va être dessinée (en millimètres). 
Valeur comprise entre 0 et la hauteur de la page (donnée par la fonction [iHauteurPage](../WDLang5/3046052.md)).

**`<X2> : (Réel)`**

Coordonnée horizontale (colonne ou abscisse) du coin bas droit du rectangle dans lequel l'ellipse va être dessinée (en millimètres). 
Valeur comprise entre 0 et la largeur de la page (donnée par la fonction [iLargeurPage](../WDLang5/3046036.md)).

**`<Y2> : (Réel)`**

Coordonnée verticale (ligne ou ordonnée) du coin bas droit du rectangle dans lequel l'ellipse va être dessinée (en millimètres).
Valeur comprise entre 0 et la hauteur de la page (donnée par la fonction [iHauteurPage](../WDLang5/3046052.md)).

**`<Épaisseur> : (Réel optionnel)`**

Épaisseur du trait de l'ellipse (en points pica). Valeur par défaut = 1 (un point Pica = 0,353 mm).
**Attention** : Selon la valeur de ce paramètre, l'épaisseur est obtenue en ajoutant plusieurs lignes les unes à côté des autres de l'extérieur vers l'intérieur.
![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Ce paramètre doit correspondre à une valeur de type entier. Si ce n'est pas le cas, l'épaisseur sera arrondie à la valeur entière la plus proche.

**`<Luminosité> : (Entier optionnel (compris entre 1 et 200))`**

Taux de la couleur du bord et du fond de l'ellipse. Si ce paramètre est :

- Égal à 0 (valeur par défaut): aucun remplissage.

- Égal à 100 : remplissage total avec la couleur spécifiée.

- Inférieur à 100 : plus ce paramètre se rapproche de 0 et plus la couleur s'éclaircit.

- Supérieur à 100 : plus ce paramètre se rapproche de 200 et plus la couleur s'assombrit.




**`<Couleur> : (Entier optionnel)`**

Couleur utilisée pour le fond et le bord de l'ellipse. Si ce paramètre n'est pas précisé, la couleur utilisée est le noir. 
Cette couleur peut correspondre à :

- une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)),

- une couleur TSL (obtenue avec la fonction [TSL](../WDLang1/3029057.md)),

- [une couleur prédéfinie du WLangage](../WDLang5/3010002.md).




**`<Trait> : (Constante optionnelle de type entier)`**

Type du trait.



|   |   |
| --- | --- |
| *iContinu*<br>(valeur par défaut) | Trait continu |
| *iMixte* | Trait mixte (-.-.) |
| *iPointillé* | Trait pointillé (....) |
| *iTiret* | Trait tiret (-----) |



**`<Brosse> : (Constante optionnelle de type entier)`**

Motif de remplissage du cadre.



|   |   |
| --- | --- |
| *iCroisée* | Brosse croisée |
| *iCroiséeDiagonale* | Brosse croisée en diagonale |
| *iDiagonaleDescendante* | Brosse diagonale descendante |
| *iDiagonaleMontante* | Brosse diagonale ascendante |
| *iHorizontale* | Brosse horizontale |
| *iPleine*<br>(valeur par défaut) | Brosse pleine |
| *iVerticale* | Brosse verticale |





![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Ce paramètre est ignoré.

<a name="NOTE0_1"></a>

## Remarques


### Luminosité de la couleur noire
<a name="luminosite_couleur_noire_ELTPARAGRAPHE000171"></a>Dans le cas de la couleur noire, le taux correspond à un taux de gris dont les valeurs s'échelonnent entre 1 (blanc) et 100 (noir complet). Dans ce cas, les valeurs supérieures à 100 seront traitées de la même façon que la valeur 100.


<a name="NOTE0_2"></a>


### Divers
<a name="divers_ELTPARAGRAPHE000179"></a>

- La position du curseur n'est pas modifiée par le tracé du trait.

- L'utilisation du type de trait pointillé (*iPointillé*) ou mixte (*iMixte*) combinée avec un cadre épais peut provoquer des effets disgracieux.

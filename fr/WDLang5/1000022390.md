


## iImprimeDoc (Fonction)

***En anglais : iPrintDoc***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Imprime un document au format "docx". 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonDocument est un Document = "c:\temp\propale.docx"
iDestination(iVisualisateur)
iImprimeDoc(MonDocument)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Imprimer le contenu d'un champ Traitement de texte ou d'une variable de type Document

`<Résultat> = iImprimeDoc(<Document>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le document est valide pour l'impression, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 




**`<Document> : (Variable de type Document ou nom de champ)`**

Document à imprimer. Ce document correspond : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) au nom d'un champ Traitement de texte.

- à une variable de type [Document](../WDLang1/1000022461.md).





<a name="SYNTAXE2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 
### Imprimer le contenu d'un champ Traitement de texte avec redimensionnement

`<Résultat> = iImprimeDoc(<Champ Traitement de texte> [, <Dimensionnement>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le document est valide pour l'impression, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 




**`<Champ Traitement de texte> : (Nom de champ)`**

Nom du champ Traitement de texte à manipuler affichant le document à imprimer. 

**`<Dimensionnement> : (Constante de type Entier)`**

Taille d'impression : 


|   |   |
| --- | --- |
| *iHomothétique* | Agrandissement ou réduction homothétique des pages du document afin qu'elles s'adaptent au format de sortie en ignorant les marges physiques. |
| *iHomothétiqueSansAgrandissement* | Réduction homothétique des pages du document afin qu'elles s'adaptent au format de sortie en ignorant les marges physiques. Les pages utilisant un format inférieur conservent leur format et sont centrées. |
| *iHomothétiqueSansAgrandissementZoneImprimable* | Réduction homothétique des pages du document afin qu'elles s'adaptent au format de sortie en prenant en compte les marges physiques. Les pages utilisant un format inférieur conservent leur format et sont centrées. |
| *iTailleRéelle* | Impression à la taille réelle. Les pages utilisant un format : <br><br>- plus petit que le format de sortie sont centrées dans la page. <br><br>- plus grand que le format de sortie sont tronquées et sont centrées dans la page. <br><br><br> |




<a name="SYNTAXE3"></a>

### Imprimer le contenu d'un fichier de type "docx", d'une variable de type Buffer ou d'un mémo binaire

`<Résultat> = iImprimeDoc(<Elément à imprimer> [, <Mot de passe>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le document est valide pour l'impression, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 




**`<Elément à imprimer> : (Chaîne de caractères, Buffer ou mémo binaire)`**



- Nom et chemin du fichier de type "docx" à imprimer. 

- Nom de la variable de type Buffer à imprimer. 

- Nom de la rubrique de type Mémo binaire à imprimer. 




**`<Mot de passe> : (Chaîne de caractères optionnelle)`**



- Mot de passe d'ouverture de l'élément à imprimer (fichier de type "docx", contenu de la variable de type Buffer, contenu de la variable de type Mémo binaire). 

- Chaîne vide ("") si l'élément à imprimer n'a pas de mot de passe.

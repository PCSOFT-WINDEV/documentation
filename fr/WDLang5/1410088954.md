
## iInfoSortie (Type de variable)

***En anglais : iOutputInfo***
			
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **iInfoSortie** permet d'obtenir des informations sur une impression (ou sur un export) d'un état ou sur une impression programmée effectuée depuis le visualisateur de rapports. Les caractéristiques de cette impression peuvent être connues à l'aide de différentes propriétés WLangage. 

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
iDestination(iVisualisateur)
iImprimeEtat(MonEtat)

tabInfoSortie est un tableau de iInfoSortie
tabInfoSortie = iImpressionDemandée(iListeSortie)

POUR TOUTE InfoSortie DE tabInfoSortie
	Trace("----------------------------------------")
	Trace("Destination = " + InfoSortie.Destination)
	Trace("Imprimante = " + InfoSortie.Imprimante)
	Trace("NomFichier = " + InfoSortie.NomFichier)
	Trace("FormatPapier = " + InfoSortie.FormatPapier)
	Trace("LargeurPapier = " + InfoSortie.LargeurPapier)
	Trace("HauteurPapier = " + InfoSortie.HauteurPapier)
	Trace("Orientation = " + InfoSortie.Orientation)
FIN
```


## iFichierEnCours (Fonction)

***En anglais : iCurrentFile***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le nom du fichier en cours de génération.

Remarque : Cette fonction permet de connaître le nom du fichier en cours de génération dans l'événement "Avant réimpression de l'état" ou dans l'événement "Fermeture" de l'état. Cette fonction est disponible dans les états programmés et dans les états créés avec l'éditeur d'états. 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
sNomFichier est une chaîne
sNomFichier = iFichierEnCours()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = iFichierEnCours()`
---

**`<Résultat> : (Chaîne de caractères)`**



- Nom complet du fichier en cours de génération, 

- Chaîne vide ("") dans les cas suivants : 

	- l'impression réalisée est une impression directe sur l'imprimante, 

	- l'export (PDF, HTML, ...) n'a pas encore généré de fichier, 

	- lors d'une impression programmée, l'export est réalisé via le visualisateur avant impression.

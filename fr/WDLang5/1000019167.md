


## XMLChercheNamespaceParNom (Fonction)

***En anglais : XMLFindNamespaceByName***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Recherche un namespace (espace de nommage) XML à partir de son nom dans un noeud XML et dans les parents de ce noeud.






<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
gcDocumentXML est un xmlDocument
gcDocumentXML = XMLOuvre("c:\temp\fichier.xml", depuisFichier)

cNoeud est un xmlNoeud = gcDocumentXML..NoeudRacine[1]
cNamespace est un xmlNamespace = XMLChercheNamespaceParNom(cNoeud, "monnamepace")

SI ErreurDétectée = Vrai ALORS
	Info("Le namespace n'a pas été trouvé.")
SINON
	Info("Namespace trouvé : ", cNamespace.URI)
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = XMLChercheNamespaceParNom(<Noeud XML> , <Nom du namespace>)`
---

**`<Résultat> : (Variable de type xmlNamespace)`**

Namespace correspondant au nom recherché.

**`<Noeud XML> : (Variable de type xmlNoeud)`**

Nom de la variable de type [xmlNoeud](../WDLang5/1000018786.md) correspondant au noeud du document XML à partir duquel la recherche est effectuée.

**`<Nom du namespace> : (Chaîne de caractères Unicode)`**

Nom du namespace à rechercher.

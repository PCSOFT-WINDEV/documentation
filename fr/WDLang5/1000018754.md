


## &lt;Variable sfConnexion&gt;.AvanceRequête (Fonction)

***En anglais : &lt;sfConnection variable&gt;.ForwardQuery***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Continue d'exécuter une requête SOQL (Salesforce Object Query Language) sur les enregistrements d'une base de données Salesforce. La requête a été exécutée avec la fonction [&lt;Variable sfConnexion&gt;.ExécuteRequête](../WDLang5/1000018762.md).




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connexion à Salesforce
Cnx est une sfConnexion
Cnx.Login = "balthazar@gmail.com"
Cnx.MotDePasse = "azerty" + "XXB12VCZ54"
SI Cnx.Connecte() = Faux ALORS
	Erreur("La connexion a échoué")
	RETOUR
FIN

ResRequête est un sfRésultatExécuteRequête
ResRequête = Cnx.ExécuteRequête("SELECT Date__c,Miles__c,Contact__c FROM Mileage__c")

BOUCLE
	MonEnregistrement est un sfObjet
	POUR TOUT MonEnregistrement DE ResRequête..Objet
		Trace("---- Type : " + MonEnregistrement..TypeObjet + " | ID : " + MonEnregistrement..ID)
		Trace(MonEnregistrement..Date__c + " / " + ...
		MonEnregistrement..Miles__c + " / " + MonEnregistrement..Contact__c)
	FIN
	ResRequête = Cnx.AvanceRequête(ResRequête..LocalisationRequête)
A FAIRE TANTQUE PAS ResRequête..Fini

POUR TOUT MonEnregistrement DE ResRequête..Objet
	Trace("---- Type : " + MonEnregistrement..TypeObjet + ...
		" | ID : " + MonEnregistrement..ID)
	Trace(MonEnregistrement..Date__c + " / " + ...
		MonEnregistrement..Miles__c + " / " + MonEnregistrement..Contact__c)
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Connexion Salesforce>.AvanceRequête(<Localisation de la requête>)`
---

**`<Résultat> : (Variable de type sfRésultatExécuteRequête)`**

Variable de type [sfRésultatExécuteRequête](../WDLang5/1000018558.md) contenant les données résultat, notamment les enregistrements Salesforce correspondant au résultat de la requête.

**`<Connexion Salesforce> : (Variable de type sfConnexion)`**

Nom de la variable de type [sfConnexion](../WDLang5/1000017579.md) à utiliser.

**`<Localisation de la requête> : (Chaîne de caractères Unicode)`**

Chaîne permettant d'identifiant la requête. Cette chaîne est obtenue grâce à la propriété **LocalisationRequête** de la dernière variable [sfRésultatExécuteRequête](../WDLang5/1000018558.md) reçue.

## Remarques


### Cas d'erreurs possibles : 
<a name="cas_erreurs_possibles_ELTPARAGRAPHE000203"></a>

- La connexion Internet n'est pas valide.

- L'authentification n'a pas pu être réalisée correctement.

- La réponse XML du serveur n'est pas lisible.




Pour savoir si une erreur est survenue, utilisez la variable [ErreurDétectée](../WDLang1/3087001.md).


## rssSource (Type de variable)

***En anglais : rssSource***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **rssSource** permet de décrire la source d'une entrée d'un flux RSS. Les caractéristiques de ce type de variable peuvent être définies et modifiées à l'aide de différentes propriétés WLangage.  

Ce type de variable est utilisé par le [type rssEntrée](../WDLang5/1000017792.md).  

**Notes** :

- Le plus souvent, la déclaration d'une variable de type **rssSource** n'est pas nécessaire. En effet, les propriétés d'une variable de [type rssEntrée](../WDLang5/1000017792.md) permettent de manipuler directement les propriétés d'une variable de type **rssSource** (notamment la propriété .Source).

- Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).

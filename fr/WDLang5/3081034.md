


## XMLTermine (Fonction)

***En anglais : XMLClose***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ferme un document XML créé avec la fonction [XMLDocument](../WDLang5/3081008.md). Toutes les ressources allouées lors de la création et de la manipulation du document XML sont libérées.








<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
XMLDocument("DocXML", CodeXML)
// Traitement et analyse du document XML
// ...

XMLTermine("DocXML")
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = XMLTermine(<Document XML>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le document existe et est fermé, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Document XML> : (Chaîne de caractères)`**

Nom du document XML manipulé. Ce document contient le code XML et a été créé avec la fonction [XMLDocument](../WDLang5/3081008.md).

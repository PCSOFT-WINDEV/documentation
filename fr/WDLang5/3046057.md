


## iHauteurImage (Fonction)

***En anglais : iImageHeight***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Calcule la hauteur de l'image à imprimer (en millimètres). Ce calcul est réalisé par rapport à la résolution de l'imprimante sélectionnée (et non par rapport à la résolution de l'écran). Aucun facteur d'agrandissement n'est appliqué.

![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png)  Seules les images au format JPEG peuvent être manipulées.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Imprime les dimensions de l'image (en mm)
iImprime("Largeur de l'image : " + iLargeurImage("WINDEV.JPG") + "mm")
iImprime("Hauteur de l'image : " + iHauteurImage("WINDEV.JPG") + "mm")
iFinImprime()
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Calculer la hauteur d'une image présente dans un fichier

`<Résultat> = iHauteurImage(<Nom de l'image>)`
---

**`<Résultat> : (Réel)`**

Hauteur de l'image (en millimètres).

**`<Nom de l'image> : (Chaîne de caractères)`**

Nom et chemin (relatif ou complet) du fichier image.

![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) Seules des images au format JPEG peuvent être manipulées.

### Calculer la hauteur d'une image présente dans une variable de type Image

`<Résultat> = iHauteurImage(<Image>)`
---

**`<Résultat> : (Réel)`**

Hauteur de l'image (en millimètres).

**`<Image> : (Variable de type Image)`**

Nom d'une variable de type [Image](../WDLang1/1000019650.md).

<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- La fonction [iLargeurImage](../WDLang5/3046061.md) permet de connaître la largeur d'une image à imprimer.

- La fonction **iHauteurImage** ne permet pas de connaître la hauteur d'une image au format WMF.

- Les dimensions d'une image peuvent être connues grâce à la fonction [InfoBitmap](../WDLang1/3029015.md).

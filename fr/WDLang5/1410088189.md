
## iImprimeDiagramme (Fonction)

***En anglais : iPrintDiagram***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Imprime un diagramme provenant : 

- d'un champ Editeur de diagrammes, 

- d'un fichier "wddiag",

- d'une variable de type [Diagramme](../WDLang1/1410088055.md).  



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonDiagramme est un Diagramme = "c:\temp\propale.wddiag"
iDestination(iVisualisateur)
iImprimeDiagramme(MonDiagramme)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Imprimer le contenu d'un champ Editeur de diagrammes ou d'une variable de type Diagramme

`<Résultat> = iImprimeDiagramme(<Diagramme>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le diagramme est valide pour l'impression, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 




**`<Diagramme> : (Variable de type Diagramme ou nom de champ)`**

Diagramme à imprimer. Ce diagramme correspond : 

- au nom d'un champ Editeur de diagrammes.

- à une variable de type [Diagramme](../WDLang1/1410088055.md).





<a name="SYNTAXE2"></a>

### Imprimer le contenu d'un fichier de type "wddiag", d'une variable de type Buffer ou d'un mémo binaire

`<Résultat> = iImprimeDoc(<Elément à imprimer>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le diagramme est valide pour l'impression, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 




**`<Elément à imprimer> : (Chaîne de caractères, Buffer ou mémo binaire)`**



- Nom et chemin du fichier de type "wddiag" à imprimer. 

- Nom de la variable de type Buffer à imprimer. 

- Nom de la rubrique de type Mémo binaire à imprimer.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
La mise en page du diagramme à imprimer peut être paramétrée grâce à la propriété [MiseEnPage](../Proprietes/1000023138.md).

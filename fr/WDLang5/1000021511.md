


## &lt;Variable xlsDocument&gt;.Sauve (Fonction)

***En anglais : &lt;xlsDocument variable&gt;.Save***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Sauve un document XLS/XLSX. Le document sauvegardé reste ouvert. Pour le fermer, il est nécessaire d'utiliser la fonction [&lt;Variable xlsDocument&gt;.Ferme](../WDLang5/1000021510.md).  


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MaFeuille est un xlsDocument
NomFichier est une chaîne
NomFichier = fSélecteur(fRepExe(), "", ...
	"Sélectionnez un fichier...", ...
	"Tous fichiers (*.*)" + TAB + "*.*", "*", fselOuvre + fselExiste)
SI NomFichier = "" ALORS RETOUR
MaFeuille = xlsOuvre(NomFichier, xlsEcriture)

// On va mettre OK dans la case A 25
MaFeuille[25,"A"] = "OK"
// Sauvegarde
MaFeuille.Sauve()
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Document XLS>.Sauve([<Chemin du fichier> [, <Mot de passe>]])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la sauvegarde a été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Document XLS> : (Variable de type xlsDocument)`**

Nom de la variable de type [xlsDocument](../WDLang5/1000017464.md) à utiliser.

**`<Chemin du fichier> : (Chaîne de caractères optionnelle)`**

Chemin complet et nom du fichier XLS.

**Si ce paramètre est précisé,** le fichier est sauvegardé à l'emplacement indiqué (le fichier existant est écrasé si nécessaire). 

- Si la sauvegarde est réussie, la propriété **Fichier** de la variable de type [xlsDocument](../WDLang5/1000017464.md) manipulée contient &lt;Chemin du fichier&gt;. Le fichier XLS reste ouvert. 

- Si le document XLS est enregistré avec un chemin différent de celui d'origine (correspond à un "Enregistrer sous"), le fichier d'origine est fermé, le document est enregistré dans le nouveau fichier et le nouveau fichier XLS reste ouvert.  


**Si ce paramètre n'est pas précisé**, le chemin pris en compte est le chemin spécifié dans la propriété **Fichier** de la variable de type [xlsDocument](../WDLang5/1000017464.md) manipulée.

**`<Mot de passe> : (Chaîne de caractères optionnelle)`**

Mot de passe à utiliser pour protéger le fichier XLS en ouverture. Ce mot de passe devra être indiqué pour ouvrir le fichier XLS.


## iImpressionDemandée (Fonction)

***En anglais : iDocumentPrinted***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de : 

- savoir si l'utilisateur a demandé l'impression du document depuis le visualisateur de rapports.

- obtenir les informations détaillées concernant l'impression effectuée. 



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
iDestination(iVisualisateur)
iImprime("Texte")
iFinImprime()
SI iImpressionDemandée() = Vrai ALORS
	Info("Vous avez demandé l'impression du document")
FIN
```



```wl
iDestination(iVisualisateur)
iImprimeEtat(MonEtat)

tabInfoSortie est un tableau de iInfoSortie
tabInfoSortie = iImpressionDemandée(iListeSortie)

POUR TOUTE InfoSortie DE tabInfoSortie
	Trace("----------------------------------------")
	Trace("Destination = " + InfoSortie.Destination)
	Trace("Imprimante = " + InfoSortie.Imprimante)
	Trace("NomFichier = " + InfoSortie.NomFichier)
	Trace("FormatPapier = " + InfoSortie.FormatPapier)
	Trace("LargeurPapier = " + InfoSortie.LargeurPapier)
	Trace("HauteurPapier = " + InfoSortie.HauteurPapier)
	Trace("Orientation = " + InfoSortie.Orientation)
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Savoir si l'utilisateur a demandé une impression ou un export depuis le visualisateur de rapports

`<Résultat> = iImpressionDemandée()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'utilisateur a demandé l'impression du document depuis le visualisateur de rapports, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**Attention** : Cette fonction ne permet pas de savoir si l'impression s'est bien passée. En cas de problème physique de l'imprimante, la fonction renverra toujours le même résultat.  


<a name="SYNTAXE2"></a>

### Obtenir les informations détaillées de l'impression ou de l'export réalisé

`<Résultat> = iImpressionDemandée(<Options>)`
---

**`<Résultat> : (Tableau de iInfoSortie)`**

Tableau de variables de type [iInfoSortie](../WDLang5/1410088954.md) contenant les différentes impressions réalisées depuis le visualisateur de rapports. 

**`<Options> : (Constante de type Entier)`**

Type de liste :


|   |   |
| --- | --- |
| *iListeSortie* | Liste des différentes impressions ou exports réalisés depuis le visualisateur de rapports. |

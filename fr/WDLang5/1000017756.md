
## gglSociété (Type de variable)

***En anglais : gglOrganization ***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **gglSociété** permet de décrire les sociétés et organisations auxquelles appartiennent un contact.

Ce type de variable est utilisé par le type [gglContact](../WDLang5/1000017434.md).

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connexion à Google
Cnx est un gglConnexion
Cnx.Email = "balthazar@gmail.com"
Cnx.MotDePasse = "azerty"
Cnx.NomApplication = "MonAppli-01"
SI GglConnecte(Cnx) = Faux ALORS
	Erreur(ErreurInfo())
FIN
...
// Récupération du tableau des contacts
tabContacts est un tableau de gglContact
tabContacts = GglListeContact(Cnx)
// Affichage des sociétés du premier contact
MonContact est un gglContact = tabContacts[1]
Org est un gglSociété dynamique
POUR TOUT Org DE MonContact.Société
	Trace(Org.Nom + " : " + Org.Libellé)
FIN
```

## Remarques




### Propriétés spécifiques à la description des variables de type gglSociété
<a name="proprietes_specifiques_description_des_variables_type_gglsociete_ELTPARAGRAPHE000037"></a>Les propriétés suivantes peuvent être utilisées pour manipuler une variable de type **gglSociété** :

| Nom | Type manipulé | Effet |
| --- | --- | --- |
| Libellé | Chaîne de caractères | Description de la société. |
| Nom | Chaîne de caractères | Nom de la société. |
| Principal | Booléen | <br><br>- <u><u><u><u>Vrai</u></u></u></u> si la société correspond à la société principale du contact,<br><br>- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.<br><br><br> |
| Titre | Chaîne de caractères | Titre du contact dans la société. |
| Type | Constante de type entier | Type de la société correspondant à une des variables suivantes :<br><br>- *gglAutre* : Société à laquelle le contact est lié d'une manière non professionnelle.<br><br>- *gglBureau* : Société à laquelle le contact est lié d'une manière professionnelle.<br><br><br> |

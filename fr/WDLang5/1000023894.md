


## &lt;Variable xmlNoeud&gt;.NoeudValide (Fonction)

***En anglais : &lt;xmlNode variable&gt;.ValidNode***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Valide un noeud XML, ses fils et ses attributs à partir de sa description dans le schéma XSD lié au document XML. 






<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Validation à partir du schéma défini lors de la déclaration
cMonNoeud est un xmlNoeud, description="schemaXSDDuProjet.NoeudRacine"
cMonNoeud..Texte = "21"
cMonNoeud..NoeudFils = "221"
SI cMonNoeud.NoeudValide() = Faux ALORS
	Erreur(ErreurInfo())
SINON
	Info("Le noeud XML est valide") 
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Noeud XML>.NoeudValide()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la validation réussie, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. En cas d'erreur, il est possible de connaître le libellé de l'erreur avec la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Noeud XML> : (Variable de type xmlNoeud)`**

Nom de la variable de type [xmlNoeud](../WDLang5/1000018786.md) correspondant au noeud à valider en utilisant le schéma XSD. Le schéma XSD utilisé correspond : 

- soit à la description utilisée au moment de la déclaration du noeud (par exemple : cMonNoeud est un xmlNoeud, description="monXSD.MonNoeud")

- soit à la description utilisée au moment de la déclaration du document XML auquel est associé le noeud.

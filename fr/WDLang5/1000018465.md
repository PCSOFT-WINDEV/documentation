


## SFSupprime (Fonction)

***En anglais : SFDelete***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime des enregistrements d'une base de données Salesforce.



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connexion à Salesforce
Cnx est une sfConnexion
Cnx..Login = "balthazar@gmail.com"
Cnx..MotDePasse = "azerty" + "XXB12VCZ54"
SI SFConnecte(Cnx) = Faux ALORS
	Erreur("La connexion a échoué")
	RETOUR
FIN

ResRequête est un sfRésultatExécuteRequête
ResRequête = SFExécuteRequête(Cnx, "SELECT Date__c,Miles__c,Contact__c FROM Mileage__c")

MonEnregistrement est un sfObjet
POUR TOUT MonEnregistrement DE ResRequête..Objet
	 Trace("---- Type : " + MonEnregistrement..TypeObjet + " | ID : " + MonEnregistrement..ID)
	 Trace(MonEnregistrement..Date__c + " / " + MonEnregistrement..Miles__c + ...
			 " / " + MonEnregistrement..Contact__c)
FIN

//Suppression d'un objet
tabSuppr est un tableau de chaînes
Ajoute(tabSuppr, ResRequête..Objet[1]..ID)

ResSuppr est un tableau de SFRésultatSupprime
ResSuppr = SFSupprime(Cnx, tabSuppr)
POUR TOUT res DE ResSuppr
	Trace("ID : " + res..ID + " - Suppression = " + res..Succès)
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = SFSupprime(<Connexion Salesforce> , <Enregistrements à supprimer>)`
---

**`<Résultat> : (Variable de type Tableau)`**

Tableau de variables de type [sfRésultatSupprime](../WDLang5/1000018546.md) contenant les données supprimées.

**`<Connexion Salesforce> : (Variable de type sfConnexion)`**

Nom de la variable de type [sfConnexion](../WDLang5/1000017579.md) à utiliser.

**`<Enregistrements à supprimer> : (Variable de type Tableau)`**

Tableau de chaînes de caractères contenant les identifiants Salesforce des enregistrements à supprimer.

## Remarques


### Cas d'erreurs possibles : 
<a name="cas_erreurs_possibles_ELTPARAGRAPHE000075"></a>

- La connexion Internet n'est pas valide.

- L'authentification n'a pas pu être réalisée correctement.

- La réponse XML du serveur n'est pas lisible.




Pour savoir si une erreur est survenue, utilisez la variable [ErreurDétectée](../WDLang1/3087001.md).

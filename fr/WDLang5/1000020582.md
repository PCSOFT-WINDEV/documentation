


## iGroupeImprime (Fonction)

***En anglais : iGroupPrint***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Lance l'impression d'un état présent dans un groupe d'états. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Préparation d'un groupe d'états
nIndice est un entier = iGroupeAjoute(ETAT_CA_Annee_A4_Portrait, 2019)
iGroupeParamètre(nIndice, "Statistiques 2019")
nIndice = iGroupeAjoute(ETAT_CA_Annee_A4_Portrait, 2018)
iGroupeParamètre(nIndice, "Statistiques 2018")
iGroupeImprime()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Imprimer un état d'un groupe identifié par son nom

`<Résultat> = iGroupeImprime([<Etat à imprimer>])`
---

**`<Résultat> : (Type du résultat retourné par l'état (optionnel))`**

Valeur renvoyée par l'état lors de son impression. Cette valeur est renvoyée : 

- soit par le mot-clé [RENVOYER](../Motscles/1510007.md) présent dans le code de fermeture de l'état. 

- soit par la propriété [ValeurRevoyée](../Proprietes/2510134.md) utilisée avant la fermeture de l'état.




**`<Etat à imprimer> : (Nom de l'état)`**

Nom de l'état à imprimer. Dans le visualisateur de rapports, une coche sera affichée devant le libellé de cet état. Cet état a été ajouté dans le groupe par la fonction [iGroupeAjoute](../WDLang5/1000020580.md).

Si ce paramètre n'est pas spécifié, l'état imprimé correspond : 

- soit au dernier état du groupe choisi par l'utilisateur dans le visualisateur de rapports. 

- soit au premier état du groupe (selon l'ordre d'ajout avec la fonction [iGroupeAjoute](../WDLang5/1000020580.md)) si le visualisateur de rapports n'a jamais été lancé.





<a name="SYNTAXE2"></a>

### Imprimer un état d'un groupe identifié par son identifiant

`<Résultat> = iGroupeImprime([<Indice de l'état>])`
---

**`<Résultat> : (Type du résultat retourné par l'état (optionnel))`**

Valeur renvoyée par l'état lors de son impression. Cette valeur est renvoyée : 

- soit par le mot-clé [RENVOYER](../Motscles/1510007.md) présent dans le code de fermeture de l'état. 

- soit par la propriété [ValeurRevoyée](../Proprietes/2510134.md) utilisée avant la fermeture de l'état.




**`<Indice de l'état> : (Entier optionnel)`**

Indice de l'état à imprimer. Cet indice a été renvoyé par la fonction [iGroupeAjoute](../WDLang5/1000020580.md) lors de l'ajout de l'état dans le groupe. Dans le visualisateur de rapports, une coche sera affichée devant le libellé de cet état.

Si ce paramètre n'est pas spécifié, l'état imprimé correspond : 

- soit au dernier état choisi par l'utilisateur dans le visualisateur de rapports. 

- soit au premier état du groupe (selon l'ordre d'ajout avec la fonction [iGroupeAjoute](../WDLang5/1000020580.md)) si le visualisateur de rapports n'a jamais été lancé.

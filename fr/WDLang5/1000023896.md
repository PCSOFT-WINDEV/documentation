## Principe de manipulation d'un document XML
<a name="principe_manipulation_document_xml_ELTTEXTE000065"></a>


Pour manipuler un document XML, il est conseillé d'[Utiliser une variable de type xmlDocument](../WDLang5/3081018.md). Cette méthode permet d'utiliser un fichier de type XSD. 

Plusieurs types de variables spécifiques sont disponibles : 



|   |   |
| --- | --- |
| [xmlAttribut](../WDLang5/1000018799.md) | Le type **xmlAttribut** permet de manipuler les attributs d'un noeud dans un document XML. |
| [xmlDocument](../WDLang5/1000018782.md) | Le type **xmlDocument** permet de manipuler simplement un fichier XML. |
| [xmlNamespace](../WDLang5/1000018803.md) | Le type **xmlNamespace** permet de manipuler l'espace de nommage XML d'un attribut de noeud dans un document XML. |
| [xmlNoeud](../WDLang5/1000018786.md) | Le type **xmlNoeud** permet de manipuler un noeud d'un document XML. |





Plusieurs fonctions spécifiques permettent de manipuler les variables de type xmlDocument



|   |   |
| --- | --- |
| [&lt;Variable JSON&gt;.VersXML](../WDLang5/1410087332.md) | Convertit une chaîne de caractères au format JSON en une chaîne de caractères au format XML. |
| [&lt;Variable xmlDocument&gt;.ConstruitChaîne](../WDLang5/1000023891.md) | Récupère et met en forme le contenu d'un document XML. |
| [&lt;Variable xmlDocument&gt;.DocumentValide](../WDLang5/1000023892.md) | Valide un document XML à partir d'un schéma XSD. |
| [&lt;Variable xmlDocument&gt;.ExécuteXPath](../WDLang5/1000023893.md) | Exécute une requête XPATH dans une variable de type xmlDocument. |
| [&lt;Variable xmlDocument&gt;.Sauve](../WDLang5/1000023895.md) | Sauvegarde un document XML dans un fichier. |
| [&lt;Variable xmlNoeud&gt;.ChercheNamespaceParNom](../WDLang5/1000023889.md) | Recherche un namespace (espace de nommage) XML à partir de son nom dans un noeud XML et dans les parents de ce noeud. |
| [&lt;Variable xmlNoeud&gt;.ChercheNamespaceParURI](../WDLang5/1000023890.md) | Recherche un namespace (espace de nommage) XML à partir de son URI (Universal Resource Identifier) dans un noeud et dans les parents de ce noeud. |
| [&lt;Variable xmlNoeud&gt;.NoeudValide](../WDLang5/1000023894.md) | Valide un noeud XML, ses fils et ses attributs à partir de sa description dans le schéma XSD lié au document XML. |
| [&lt;xmlDocument&gt;.VersJSON](../WDLang5/1410087334.md) | Convertit une chaîne de caractères au format XML en une chaîne de caractères au format JSON. |
| [&lt;xmlNoeud&gt;.VersJSON](../WDLang5/1410087333.md) | Convertit une chaîne de caractères au format XML en une chaîne de caractères au format JSON. |






Certaines fonctions sont également liées à la gestion du XML : 



|   |   |

| [&lt;Source&gt;.ImporteXML](../WDLang4/1000024994.md) | Importe un fichier XML dans un fichier de données au format HFSQL Classic ou Client/Serveur (fichier de données décrit sous l'éditeur d'analyses ou déclaré par les fonctions [&lt;Source&gt;.Déclare](../WDLang4/1000024213.md), [&lt;Source&gt;.DéclareExterne](../WDLang4/1000024214.md) ou [&lt;Source&gt;.DécritFichier](../WDLang4/1000024215.md)). |
| [J2EEExécuteXML](../WDLang3/3047008.md) | Exécute une procédure sur un serveur de services Web XML J2EE. |

| [&lt;Table&gt;.VersXML](../WDLang1/1410088378.md) | Crée un fichier XML avec les données d'un champ Table ou Table hiérarchique. |
| [&lt;Zone répétée&gt;.VersXML](../WDLang2/1000023707.md) | Crée un fichier XML avec les données d'un champ Zone répétée (champ Zone répétée fichier ou champ Zone répétée par programmation). |

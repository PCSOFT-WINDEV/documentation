## Présentation
<a name="presentation_ELTTEXTE000139"></a>
WDStatistique est un utilitaire permettant d'effectuer des statistiques de fréquentation de vos sites WEBDEV dynamiques (classiques et AWP) et des Webservices déployés sur un Serveur d'Application WEBDEV.

Ces statistiques sont réalisées à partir de fichiers journal (extension ".LOG") générés par le Serveur d'Application WEBDEV. Ces fichiers peuvent être enregistrés pour l'ensemble des sites ou pour un site spécifique.

Quelques exemples des statistiques automatiquement réalisées par WDStatistique :

- Liste des connexions : détail de toutes les connexions effectuées sur le serveur Web (adresse IP, date et heure, navigateur utilisé, système d'exploitation, etc.).

- Connexions par site : nombre de connexions par site WEBDEV et durée de connexion.

- Liste des pages visitées par un internaute.

- Nombre de connexions réalisées pour un type de navigateur (Internet Explorer, etc.).

- Nombre de connexions réalisées par plage horaire.

- Nombre de connexions réalisées par système d'exploitation.

- Liste des sites sur lesquels les internautes se trouvaient avant d'arriver sur votre site.

- Liste des mot-clés utilisés sur les moteurs de recherche permettant d'arriver sur votre site.

- Statistiques sur l'origine géographique des visiteurs (par pays).




Ces différentes statistiques sont présentées sous forme de :

- Tableau récapitulatif. Ces tableaux peuvent être enregistrés au format Word, Excel ou XML (clic droit sur le tableau).

- Graphique. Ces graphiques peuvent être personnalisés (clic droit sur le graphique). Il est ainsi possible de changer le type de graphe, d'ajouter des légendes, de modifier la police utilisée, etc.

- Rapport imprimable. Il est possible de choisir le type de rapport à imprimer ainsi que la période voulue. Ces rapports peuvent être imprimés, enregistrés au format PDF, envoyés par email, etc.




WDStatistique peut être lancé directement en double-cliquant sur le programme "WDStatistique.EXE".

WDStatistique est un outil **non redistribuable**. WDStatistique ne peut pas être installé sur le serveur Web où votre site est déployé.

Pour utiliser WDStatistique, WEBDEV version Développement doit obligatoirement être installé sur le poste en cours.

Remarque : L'outil WDStatistique est également disponible en ligne, à partir de l'administrateur distant. Pour plus de détails, consultez [Visualiser les statistiques dans l'administrateur distant](../WDAdminWeb/6466.md).

## Utilisation de WDStatistique
<a name="utilisation_wdstatistique_ELTTEXTE000163"></a>
Pour connaître les statistiques de fréquentation de vos sites dynamiques, il suffit de :

1. [Configurer l'administrateur WEBDEV](../WDStatistique/3536003.md) afin de générer le(s) fichiers journal(aux) (extension ".LOG").

2. [Importer le(s) fichier(s) journal(aux)](../WDStatistique/3536002.md) dans WDStatistique.

<a name="NOTE2_2"></a>




### Fichier journal
<a name="fichier_journal_ELTPARAGRAPHE000071"></a>Les statistiques de fréquentation de vos sites dynamiques sont calculées à partir de fichiers de type journal (extension ".LOG"). Ces fichiers sont générés par le moteur WEBDEV.

Il est possible de créer ce type de fichier :

- Soit ***pour chacun des sites WEBDEV dynamiques*** installés sur le serveur Web. Dans ce cas, un seul fichier journal sera créé par jour dans un répertoire spécifique à chaque site. Ce fichier contiendra l'historique des manipulations effectuées uniquement dans le site WEBDEV dynamique concerné.

- Soit ***pour l'ensemble des sites WEBDEV dynamiques*** installés sur le serveur Web. Dans ce cas, un seul fichier journal sera créé par jour dans un répertoire commun. Ce fichier contiendra l'historique des manipulations effectuées dans l'ensemble des sites WEBDEV dynamiques installés sur le serveur Web.




Ces fichiers de type journal sont enregistrés en continu dès que le(s) site(s)associé(s) au fichier sont utilisés.

Le format du nom de ces fichiers de type journal est le suivant : "WDSession_&lt;AnnéeEnregistrement&gt;&lt;MoisEnregistrement&gt;&lt;JourEnregistrement&gt;"


<a name="NOTE2_3"></a>




### Fichiers de données
<a name="fichiers_donnees_ELTPARAGRAPHE000089"></a>Les fichiers de données de WDStatistique sont présents dans le répertoire "C:\\Documents and Settings\\Utilisateur\\Application Data\\PC SOFT\\WDSTATISTIQUE\\". Il est possible de transférer ces fichiers d'un poste à un autre.

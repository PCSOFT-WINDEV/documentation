
## WDStatistique : Importation des fichiers journal
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000117"></a>
Pour visualiser les statistiques de fréquentation de vos sites grâce à WDStatistique, il est nécessaire d'importer les fichiers journal sur le poste en cours.

Cette importation peut être :

- **manuelle** :
	Cette importation est conseillée :

	- pour visualiser occasionnellement les statistiques de fréquentation de vos sites.

	- quand le répertoire contenant les fichiers journal est directement accessible depuis le poste en cours (site Intranet par exemple).




- **automatique** :
	Cette importation est conseillée pour réaliser un suivi régulier de la fréquentation de vos sites.




Avant de lancer une importation de fichiers journal, il est nécessaire de configurer les informations sur le site dont on souhaite obtenir les statistiques.
Pour configurer un site, il suffit de lancer WDStatistique et de cliquer sur le bouton "Gestion des sites".

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Importation manuelle des fichiers journal
<a name="importation_manuelle_des_fichiers_journal_ELTTEXTE000141"></a>
Pour importer manuellement un fichier journal :

1. Lancez WDStatistique (double-cliquez sur le programme "WDStatistique.EXE"). 

2. Configurez la manière d'importer les fichiers journal en cliquant sur le bouton "Gestion des sites" (via WebDep, FTP ou directement un chemin). 

3. Cliquez sur le bouton "Importation". La fenêtre permettant de paramétrer l'importation s'affiche.

4. Sélectionnez le site dont il faut importer les fichiers journal.

5. Saisissez la date à partir de laquelle vous souhaitez importer les fichiers journal.

6. Cliquez sur le bouton "Lancer maintenant une importation".

7. WDStatistique affiche les statistiques concernant le (ou les) fichier(s) journal importé(s).

## Importation automatique des fichiers journal
<a name="importation_automatique_des_fichiers_journal_ELTTEXTE000165"></a>


### Configurer l'importation automatique de fichiers journal
<a name="configurer_importation_automatique_fichiers_journal_ELTPARAGRAPHE000050"></a>Pour configurer l'importation automatique de fichiers journal :

1. Lancez WDStatistique (double-cliquez sur le programme "WDStatistique.EXE").

2. Cliquez sur le bouton "Importation". La fenêtre de paramétrage de l'importation s'affiche.

3. Sélectionnez le site dont vous souhaitez automatiser l'importation des fichiers journal.

4. Cochez la case "Planifier l'importation des logs".

5. Sélectionnez la fréquentation d'importation (occasionnelle, journalière, hebdomadaire ou mensuelle).

6. Spécifiez les informations de lancement (date et heure).

7. Valider la fenêtre.




### Remarque importante
<a name="remarque_importante_ELTPARAGRAPHE000063"></a>L'importation automatique des fichiers journal s'appuie sur le mécanisme des tâches planifiées de Microsoft Windows.
Il est donc très important de renseigner correctement les informations concernant l'utilisateur Windows de la machine et le mot de passe associé au compte.
Sans ces informations, il est impossible de planifier automatiquement l'importation des fichiers journal.

## Présentation
<a name="presentation_ELTTEXTE000133"></a>
Après avoir importé les fichiers journal, l'utilisation de WDStatistique peut commencer.

La fenêtre principale de WDStatistique est la suivante :

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=WDStatistique%20-%20HC%20N%B0001%201.gif)



|   |   |
| --- | --- |
| ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CERCLE1.gif)<br> | Nom du site actuellement observé<br><br>Sélectionnez le nom du site à observer ainsi que la période souhaitée. |
| ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CERCLE2.gif)<br> | Types de statistiques à afficher<br><br>Cliquez sur le type de statistiques à afficher. |
| ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CERCLE3.gif)<br> | Statistiques demandées. <br>Ces différentes statistiques sont présentées sous forme de :<br><br>- **tableaux récapitulatifs**. <br>	Ces tableaux peuvent être enregistrés au format Word, Excel ou XML (clic droit sur le tableau).<br><br>- **graphiques**. Ces graphiques peuvent être personnalisés (clic droit sur le graphique). Il est ainsi possible de changer le type de graphe, d'ajouter des légendes, de modifier la police utilisée, ...<br><br><br> |

## Les statistiques disponibles
<a name="les_statistiques_disponibles_ELTTEXTE000157"></a>
Les statistiques proposées par WDStatistique sont les suivantes :


|   |   |
| --- | --- |
| <br><br>- ***Tableau de bord :***<br><br><br> | Tableau de bord récapitulant :<br><br>- le nombre de visiteurs.<br><br>- le nombre de pages vues.<br><br>- le nombre de pages moyen par visiteur.<br><br>- le nombre moyen de visiteurs par jour.<br><br>- le nombre moyen de pages vues par jour.<br><br>- le nombre maximum de connexions.<br><br>- le nombre minimum de connexions.<br><br>- la date de dernière importation des fichiers journal.<br><br>- le fuseau horaire du site.<br><br>- le type de site (Classique ou AWP).<br><br><br> |
| <br><br>- ***Réseaux de recherche :***<br><br><br> | Liste des mot-clés utilisés sur les moteurs de recherche permettant d'arriver sur le site. |
| <br><br>- ***Données techniques :***<br><br><br> | Liste des systèmes d'exploitation des internautes et des navigateurs. |
| <br><br>- ***Fréquentation :***<br><br><br> | Graphe récapitulant le nombre de visites selon différentes répartitions temporelles. |
| <br><br>- ***Vue Géographique :***<br><br><br> | Répartition de chaque visiteur selon le pays. |
| <br><br>- ***Parcours des visiteurs :***<br><br><br> | Parcours effectués pour chaque visiteur sur le site (temps de parcours et durée passée sur chaque page). |


## WDStatistique : Configuration de l'administrateur WEBDEV
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000116"></a>
Pour créer les fichiers journal nécessaires aux calculs des statistiques de fréquentation de vos sites, un paramétrage spécifique est nécessaire dans l'administrateur WEBDEV.

Ce paramétrage peut être réalisé :

- soit depuis l'administrateur WEBDEV Distant accessible directement depuis le poste en cours (conseillé).

- soit depuis l'administrateur WEBDEV installé chez votre hébergeur (livré avec le serveur d'application WEBDEV).






<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Configuration de l'administrateur WEBDEV Distant
<a name="configuration_administrateur_webdev_distant_ELTTEXTE000140"></a>
Pour créer les fichiers journal pour chacun des sites WEBDEV dynamiques installés sur le serveur Web : 

1. Affichez la page générale de l'administrateur WEBDEV Distant.
	Rappel : L'administrateur WEBDEV Distant peut être lancé à partir de n'importe quel poste possédant un navigateur Internet. Il suffit d'utiliser l'adresse suivante : http://&lt;ServeurWeb&gt;/WD280AWP/WD280AWP.EXE/CONNECT/WDAdminWeb

2. Sélectionnez le site WEBDEV voulu.

3. Cochez l'option "Générer le fichier journal".

4. Spécifiez le répertoire dans lequel les fichiers journal doivent être créés.




**Remarque** : L'administrateur WEBDEV Distant ne permet pas de configurer la création d'un fichier journal commun pour l'ensemble des sites WEBDEV dynamiques installés sur le serveur Web.



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Configuration de l'administrateur WEBDEV (Serveur d'application WEBDEV)
<a name="configuration_administrateur_webdev_serveur_application_webdev_ELTTEXTE000164"></a>


### Créer les fichiers journal pour chacun des sites WEBDEV dynamiques installés sur le serveur Web
<a name="creer_les_fichiers_journal_pour_chacun_des_sites_webdev_dynamiques_installes_sur_serveur_web_ELTPARAGRAPHE000045"></a>Pour créer les fichiers journal pour chacun des sites WEBDEV dynamiques installés sur le serveur Web : 

1. Affichez l'onglet "Sites" de l'administrateur WEBDEV (version utilisée sur le serveur d'application).

2. Sélectionnez le site WEBDEV voulu.

3. Cochez l'option "Paramètres personnalisés".

4. Cochez l'option "Générer un fichier journal particulier à ce site pour les statistiques de fréquentation (.log)".

5. Spécifiez le répertoire dans lequel les fichiers journal doivent être créés.

6. Cliquez sur le bouton "Appliquer".

7. Recommencez ces opérations pour chacun de vos sites (si nécessaire).





<a name="NOTE3_2"></a>


### Créer les fichiers journal commun pour l'ensemble des sites WEBDEV dynamiques installés sur le serveur Web
<a name="creer_les_fichiers_journal_commun_pour_ensemble_des_sites_webdev_dynamiques_installes_sur_serveur_web_ELTPARAGRAPHE000061"></a>Pour créer les fichiers journal commun pour l'ensemble des sites WEBDEV dynamiques installés sur le serveur Web : 

1. Affichez l'onglet "Configuration" de l'administrateur WEBDEV (version utilisée avec le serveur d'application).

2. Cochez l'option "Générer un fichier journal pour les statistiques de fréquentation (.log)".

3. Spécifiez le répertoire dans lequel les fichiers journal doivent être créés.

4. Cliquez sur le bouton "Appliquer".

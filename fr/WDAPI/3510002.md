


## WDAPI : Aide sur les API
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000135"></a>
WDAPI permet de simplifier l'utilisation des API Windows dans une application WINDEV. WD API génère le code et les structures WLangage permettant d'utiliser une API Windows donnée.

WDAPI ne documente PAS le fonctionnement de chaque fonction API Windows du système !

Cependant, WDAPI offre la possibilité de lancer directement un site ou un document (que vous aurez choisi) correspondant à la documentation des API Windows. Si une fonction, une structure ou une constante est sélectionnée sous WDAPI, il est possible d'afficher l'aide correspondante.

**Remarque** : La liste des API Windows fournie avec WDAPI n'est PAS exhaustive.

**Rappel** : WDAPI peut être lancé depuis WDOutil : sous le volet "Outils", dans le groupe "Utilitaires", cliquez sur "WDOutil".

## Obtenir de l'aide sur une fonctionnalité de l'API Windows
<a name="obtenir_aide_sur_une_fonctionnalite_api_windows_ELTTEXTE000159"></a>


### Configurer WDAPI pour obtenir de l'aide sur une fonction, structure ou constante de l'API Windows
<a name="configurer_wdapi_pour_obtenir_aide_sur_une_fonction_structure_constante_api_windows_ELTPARAGRAPHE000038"></a>Pour configurer WDAPI pour obtenir de l'aide sur une fonction, structure ou constante de l'API Windows : 

1. Sélectionnez le volet "Sites préférés".

2. Indiquez l'adresse correspondant au site proposant l'aide sur l'élément voulu (fonction, structure ou constante). Il est possible :

	- Soit de sélectionner une adresse existante dans la liste.

	- Soit d'ajouter une nouvelle adresse. Pour cela, il suffit de saisir la nouvelle adresse et de cliquer sur le bouton "Ajouter".







L'adresse peut correspondre :

- soit à une adresse Internet (par exemple : http://www.monsitefavori.fr).

- soit à un chemin complet vers un document (par exemple : C:\\Mes Documents\\InfoAPI.doc).

- soit à une application (par exemple : \\\\PosteAPI\\Utilitaires\\ApiPlus.exe). 




**Cette adresse peut directement ouvrir le site sur l'aide de l'élément sélectionné (fonction, constante ou structure).**

Si l'adresse originale utilise le nom de l'élément, remplacez dans l'adresse originale le nom de la fonction, constante ou structure par le mot-clé [%FONCTION%]. 
Par exemple :

```txt
http://www.fonctionsapi.com/trouve.asp?fonction=[%FONCTION%]
```


Si l'adresse originale utilise la première lettre de l'élément (cas d'un index alphabétique par exemple), remplacez dans l'adresse originale la lettre par le mot [%DEBUTFONCT%].
Par exemple :

```txt
http://www.fonctionsapi.com/index.asp?indexfonction=[%DEBUTFONCT%]
```

### Obtenir de l'aide sur une fonction, structure ou constante de l'API Windows
<a name="obtenir_aide_sur_une_fonction_structure_constante_api_windows_ELTPARAGRAPHE000081"></a>Pour obtenir de l'aide sur une fonction, structure ou constante de l'API Windows : 

1. Sélectionnez une fonction (ou une structure ou encore une constante) sous WDAPI.

2. Cliquez sur le lien "Lancer le lien vers mon site d'explication des API".


Le site sélectionné dans le volet "Sites préférés" est automatiquement lancé.

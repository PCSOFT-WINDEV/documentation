


## WDAPI : Convertir une déclaration de fonction ou de structure C
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000108"></a>
WDAPI permet de convertir une déclaration de fonction ou de structure C en une déclaration de fonction ou de structure WLangage. Il devient très simple de reprendre un code C existant et de le convertir en WLangage.

**Rappel** : WDAPI peut être lancé depuis WDOutil : sous le volet "Outils", dans le groupe "Utilitaires", cliquez sur "WDOutil".

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Convertir une déclaration de fonction ou de structure C en une déclaration de structure WLangage
<a name="convertir_une_declaration_fonction_structure_une_declaration_structure_wlangage_ELTTEXTE000132"></a>


### Convertir une déclaration de fonction ou de structure C en une déclaration de structure WLangage
<a name="convertir_une_declaration_fonction_structure_une_declaration_structure_wlangage_ELTPARAGRAPHE000032"></a>Pour convertir une déclaration de fonction ou de structure C en une déclaration de structure WLangage :

1. Sélectionnez le volet "Conversion C en WLangage".

2. Pour convertir une déclaration de fonction C, saisissez le code de cette fonction dans le "Prototype de la fonction en C".
	Pour convertir une déclaration de structure C, saisissez le code de cette structure dans le champ "Structure en C".

3. Cliquez sur la flèche (![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=fleche.gif)) placée sous le champ correspondant.

4. Le code WLangage correspondant est automatiquement généré. Ce code peut alors être copié dans votre programme WLangage (voir paragraphe suivant).


**Cas rare :** Il se peut que certaines particularités liées à une structure ou une fonction de l'API Windows ne soit pas interprétées correctement par WDAPI. Vous devrez donc effectuer la conversion par vous-même.
<a name="NOTE2_2"></a>


### Copier le code de déclaration de la structure C converti en WLangage
<a name="copier_code_declaration_structure_converti_wlangage_ELTPARAGRAPHE000050"></a>Pour copier le code de déclaration de la structure C converti en WLangage :

1. Effectuez la conversion de code.

2. Cliquez sur le bouton "Copier" associé au code de déclaration de la syntaxe ou de la structure WLangage.

3. Dans votre projet WINDEV, positionnez-vous sur le traitement ou l'événement voulu :

	- initialisation du projet (conseillé).

	- déclaration des globales de la fenêtre.

	- déclaration des globales de la collection de procédures dans laquelle l'appel à la fonction API est réalisé.




4. Dans l'éditeur de code, sélectionnez l'option "Coller" du menu contextuel (ou appuyez sur les touches Ctrl + V).

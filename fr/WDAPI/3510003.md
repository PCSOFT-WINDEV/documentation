## Présentation
<a name="presentation_ELTTEXTE000143"></a>
WDAPI permet de simplifier l'utilisation des API Windows dans une application WINDEV. WDAPI génère le code et les structures WLangage permettant d'utiliser une API Windows donnée.

**Attention** : WDAPI ne permet pas de rechercher une API selon une fonctionnalité donnée. Pour utiliser WDAPI, il est nécessaire de connaître le nom de l'API à utiliser. 
WDAPI ne documente PAS le fonctionnement de chaque fonction API Windows du système ! Cependant, WDAPI offre la possibilité de lancer directement un site ou un document (que vous aurez choisi) correspondant à la documentation des API Windows.

WDAPI ne vous formera pas sur l'utilisation des quelques 1600 fonctions API Windows, mais vous aidera énormément dans la phase de génération du code. En effet, vous pourrez utiliser dans votre code la fonction API que vous aurez sélectionnée dans WDAPI.

**Remarque** : La liste des API Windows fournie avec WDAPI n'est PAS exhaustive.

## Lancement de WDAPI
<a name="lancement_wdapi_ELTTEXTE000167"></a>


### Lancement
<a name="lancement_ELTPARAGRAPHE000026"></a>WDAPI peut être lancé via une des méthodes suivantes :

- en exécutant le fichier "WDAPI.EXE" présent dans le répertoire d'installation de WINDEV.

- depuis le centre de maintenance de WINDEV : sous le volet "Outils", dans le groupe "Utilitaires", cliquez sur "WDOutil".



<a name="NOTE2_2"></a>


### Conditions d'utilisation
<a name="conditions_utilisation_ELTPARAGRAPHE000048"></a>WDAPI n'est pas redistribuable avec vos applications. WDAPI ne peut pas être installé avec les applications développées avec WINDEV.

La licence d'utilisation de WINDEV s'applique intégralement.

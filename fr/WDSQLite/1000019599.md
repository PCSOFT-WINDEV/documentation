
## Connecteur Natif SQLite
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000121"></a>
WINDEV et WEBDEV 28 proposent un Connecteur Natif (également appelé module d'Accès Natif) à SQLite. Ce Connecteur Natif permet d'accéder à une base SQLite sans drivers externes depuis un programme en WLangage.

SQLite est un système de gestion de fichiers indexés. 

Le Connecteur Natif à SQLite est disponible : 

- sous Windows 32 et 64 bits.

- sous Linux 32 et 64 bits. 






<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Configuration nécessaire
<a name="configuration_necessaire_ELTTEXTE000145"></a>


### Conditions nécessaires à l'utilisation du Connecteur Natif SQLite
<a name="conditions_necessaires_utilisation_connecteur_natif_sqlite_ELTPARAGRAPHE000033"></a>

- SQLite version 3



<a name="NOTE3_2"></a>


### Fichiers nécessaires
<a name="fichiers_necessaires_ELTPARAGRAPHE000040"></a>L'utilisation du Connecteur Natif SQLite nécessite le fichier wd280sqlite.dll. Ce fichier est nécessaire pour une connexion native à une base SQLite. 



<a name="NOTE4"></a>
<a name="NOTE4_1"></a>


## Utilisation
<a name="utilisation_ELTTEXTE000175"></a>


### Utiliser le Connecteur Natif SQLite dans vos applications WINDEV ou vos sites WEBDEV
<a name="utiliser_connecteur_natif_sqlite_dans_vos_applications_windev_vos_sites_webdev_ELTPARAGRAPHE000052"></a>Pour utiliser SQLite en natif, il suffit de programmer :

- soit avec les fonctions "SQL" du WLangage (voir [Programmation à l'aide des fonctions SQL](../WDSQLite/1000019634.md)).

- soit avec les fonctions "HFSQL" du WLangage (voir [Programmation à l'aide des fonctions HFSQL](../WDSQLite/1000019624.md)).




**Remarques** : 

- N'oubliez pas de consulter les [limitations concernant l'utilisation du Connecteur Natif SQLite](../WDSQLite/1000019635.md).

- Pour déployer des sites utilisant le Connecteur Natif SQLite, il est nécessaire d'installer le Connecteur Natif SQLite sur le poste Serveur (Serveur d'Application WEBDEV).

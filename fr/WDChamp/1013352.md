
## Variante de mise en forme de texte
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000078"></a>
Un texte présenté dans une fenêtre peut être personnalisé au niveau de l'approche des caractères, de la taille, de l'interligne.
Ce paramétrage peut s’effectuer "à la main" en saisissant la combinaison de paramètres dans la fenêtre de description du champ Libellé.
La version 28 propose une visualisation automatique de combinaisons de paramètres prédéfinis harmonieux.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000102"></a>
Pour accéder aux variantes de mise en forme d'un texte présent dans un champ Libellé : 

1. Sélectionnez le champ Libellé. 

2. Affichez le menu contextuel du champ et sélectionnez l'option "Variante de mise en forme".<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Variantes_mise_en_forme%20-%20HC%20N%B0002.gif&type=thumb)


3. La fenêtre qui s'affiche présente les différentes variantes de mise en forme possibles. <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Variantes_mise_en_forme%20-%20HC%20N%B0001.gif&type=thumb)
Cette fenêtre vous permet de parcourir les différentes possibilités avec une visualisation directe des effets sur votre texte. 

4. Sélectionnez la variante voulue et validez avec la touche Entrée. La variante sélectionnée est automatiquement appliquée à votre libellé. 




Remarques : 

- En WEBDEV, cette option est également disponible sur les champs Zone de texte riche. 

- Différentes touches permettent de se déplacer dans les différentes propositions et de les modifier si nécessaire.


## Menu contextuel système
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000088"></a>
Les menus contextuels système sont gérés par le système (Android, Windows, ...). Ces menus contextuels s'affichent sur certains types de champs (ActiveX, ...).

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Les menus contextuels système par défaut
<a name="les_menus_contextuels_systeme_par_defaut_ELTTEXTE000112"></a>
Certains types de champs peuvent avoir un menu contextuel système. Ce menu contextuel peut correspondre :

- ***soit au menu contextuel affiché par Windows***. C'est le cas par exemple pour les menus contextuels des champs ActiveX, multimédia ou HTML :

	- un champ HTML : ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=MenuCtxChpHTML.gif)
Remarque : Le menu contextuel système affiché est celui d'Internet Explorer. Il dépend de l'objet sélectionné dans la page (image, texte, lien, ...).

	- un champ ActiveX : ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=MenuCtxActiveX.gif)
Remarque : Le menu contextuel système affiché dépend de l'ActiveX associé au champ.




- ***soit au menu contextuel des FAA géré par WINDEV***.

	- un champ de saisie : ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=MenuCtxChpSaisie.gif)


	- un champ Table : ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Champ_Table_MenuCtx.gif)
Pour plus de détails, consultez [Menu contextuel d'un champ Table](../WDChamp/1013207.md).




Le menu contextuel des FAA peut être configuré (par exemple pour afficher uniquement certaines options) et traduit. Pour plus de détails, consultez [Configurer le menu contextuel des FAA](../Editeurs/2010040.md).

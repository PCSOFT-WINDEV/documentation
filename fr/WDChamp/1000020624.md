
## Les graphes de type "Surface"
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000140"></a>
Le graphe de type Surface permet de représenter des données analogiques corrélées, selon 3 axes. Quelques exemples : 

- représenter une résistance d'un matériau en fonction de la tension et de la température. 

- pour un moteur, représenter le temps d'ouverture des injecteurs en fonction du volume d'air en entrée et du régime moteur. 

- pour une photo, représenter les variations de luminosité. 




Il est également possible de représenter une fonction mathématique en 3 dimensions.

## Caractéristiques d'un graphe de type Surface
<a name="caracteristiques_graphe_type_surface_ELTTEXTE000164"></a>

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Graphe_Surface.gif&type=thumb)


Dans ce graphe :

**1. Titre du graphe**. 
Les caractéristiques du titre du graphe sont saisies dans l'onglet "Détail" de la description du champ (bouton "Editer le titre du graphe"). 
Il est possible de définir le titre du graphe, sa position et son style.

**2. Quadrillage**. 
Il est possible d'afficher le quadrillage sur un ou plusieurs plans du graphe dans l'onglet "Général" de la description du champ.
Pour afficher le quadrillage par programmation, utilisez la fonction [grSurfaceQuadrillage](../WDLang3/1000020428.md). 

**3. Graphe**. 
Le graphe peut : 

- Afficher un maillage ou non (option "Afficher un maillage sur la surface" de l'onglet "Général" de la description du champ ou fonction [grSurfaceMaillage](../WDLang3/1000020430.md)). 

- Être orienté : Il suffit de définir l'angle de rotation selon les différents axes dans l'onglet "Général" de la fenêtre de description du champ. Il est également possible d'orienter le graphe par programmation grâce à la fonction [grOrientation](../WDLang3/3042043.md). 

- Être orienté par l'utilisateur : Il suffit de cocher l'option "Rotation à la souris 
(graphes de moins de 10 000 points)" dans l'onglet "Général" de la fenêtre de description du champ. 

- Avoir des couleurs : Il suffit d'utiliser la fonction [grSurfaceCouleurAltitude](../WDLang3/1000020427.md) pour définir les couleurs à utiliser. 




**4. Graduations**. 
Les graduations selon les axes peuvent être définies dans l'onglet "Général" de la fenêtre de description du champ. 

Il est possible de définir le libellé des graduations grâce à la fonction [grSurfaceLibelléGraduation](../WDLang3/1000020617.md). 

La fenêtre de description du champ permet également de définir le repère utilisé. Il est possible d'utiliser un repère orthonormé ou non.

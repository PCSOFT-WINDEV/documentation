
## Evénements associés aux champs d'un état
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000112"></a>
Les événements associés aux champs d'un état sont les suivants :

| Evénement | Conditions d'exécution |
| --- | --- |
| **Initialisation** | Exécuté lors de l'initialisation de l'état. L'ordre d'exécution des événements au lancement d'un état (y compris lors d'un test) est le suivant :<br><br>- événement "Ouverture" de l'état.<br><br>- événement "Initialisation" des blocs de l'état.<br><br>- événement "Initialisation" des champs.<br><br><br> |
| **Avant impression** | Exécuté avant l'impression du champ. |
| **Entrée dans** | Exécuté lors de l'entrée du curseur de saisie dans le champ. Cet événement est disponible uniquement pour les champs de saisie utilisés dans le visualisateur de rapports. |
| **Sortie de** | Exécuté lors du changement de champ (touche TAB, changement de champ par la souris, ...). Cet événement est disponible uniquement pour les champs de saisie utilisés dans le visualisateur de rapports. |
| **Modification de** | Exécuté à chaque modification du contenu du champ (saisie ou effacement d'une lettre par exemple). Cet événement est disponible uniquement pour les champs de saisie utilisés dans le visualisateur de rapports. |
| **Clic sur** | Exécuté lors du clic sur le champ. Le clic sur le champ est disponible uniquement dans le visualisateur de rapports. Une étoile permet à l'utilisateur d'identifier simplement les champs cliquables. |


Dans ces événements, vous pouvez saisir du code en WLangage. Pour plus de détails sur l'enchaînement de ces événements, consultez [Enchaînement des événements associés aux états](../WDChamp/1011004.md).

## Visualiser et/ou modifier les événements d'un champ
<a name="visualiser_etou_modifier_les_evenements_champ_ELTTEXTE000136"></a>
Pour visualiser et/ou modifier les événements d'un champ, vous pouvez utiliser une des méthodes suivantes :

- sélectionnez le champ et appuyez sur la touche F2.

- sélectionnez le champ et affichez le volet "[Visualisation contextuelle du code](../Editeurs/2027033.md)". 

- sélectionnez l'option "Code" du menu contextuel du champ (clic droit sur le champ).

- cliquez sur l'icône "Code" de la description du champ.

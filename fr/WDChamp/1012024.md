


## Comment ne pas afficher l'URL des pages WEBDEV dans le navigateur ? 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000100"></a>
Par défaut, lorsque l'utilisateur se connecte à un site développé avec WEBDEV (site dynamique en mode session), la barre d'adresse du navigateur contient une adresse de la forme :  

- url courte : www.mondomaine.fr/MonAppli

- ou url complète si le serveur web n'accepte pas une url courte : www.mondomaine.fr/WD280AWP/WD280AWP.exe/CONNECT/MonAppli




Cette adresse peut aisément être masquée à l'utilisateur en l'encapsulant dans une iFrame d'une page statique.

**Remarques** : 

- Cette fonctionnalité concerne uniquement les sites dynamiques en mode session. Cette fonctionnalité ne concerne pas les sites AWP, PHP ou les sites statiques.

- Pour les sites AWP, il est possible de masquer le nom du serveur d'application WEBDEV dans l'URL des pages grâce au mécanisme d'[URL Rewriting](../Editeurs/2030054.md).




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000124"></a>


### Principe
<a name="principe_ELTPARAGRAPHE000035"></a>Pour afficher une URL inchangée pendant l'utilisation d'un site WEBDEV, il suffit d'afficher le site dans une iFrame d'une page statique. L'adresse affichée dans le navigateur est celle de la page statique et le site WEBDEV est affiché dans une iFrame.
<a name="NOTE2_2"></a>


### Mise en place
<a name="mise_place_ELTPARAGRAPHE000042"></a>La création du frameset statique accueillant le site dynamique en mode session peut être réalisée :

- soit par un éditeur quelconque.

- soit avec WEBDEV. Dans ce cas, il est conseillé d'utiliser un projet différent du projet dynamique en mode session à afficher.




Voici un exemple de code HTML du frameset statique à réaliser :

```html
<html>
<head><meta http-equiv="Content-Type" content="text/html; charset=UTF-8">
<title>Titre du site</title>
<iframe src="/sitealancer" name="MonSite" style="border:0;overflow:auto;width:100%;height:100%;
position:absolute;top:0;left:0;"></iframe>
</html>
```
Cette page HTML sera généralement la page définie par défaut au niveau du serveur Web (IIS, Apache, ...)

Contactez l'administrateur de votre serveur Internet pour rediriger l'adresse http://www.mondomaine.fr vers cette page.

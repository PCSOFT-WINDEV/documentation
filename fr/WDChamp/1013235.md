## Présentation du Champ Réglette
<a name="presentation_champ_reglette_ELTTEXTE000146"></a>
Le champ Réglette permet d'accéder rapidement aux enregistrements d'un champ Table ou d'un champ Zone répétée affiché sur plusieurs pages. L'internaute n'a qu'à cliquer sur le numéro de la page pour afficher les enregistrements correspondants. 


Lors de la création d'un champ Table ou d'un champ Zone répétée (fichier ou par programmation) non Ajax, une réglette est automatiquement associée au champ dès que le nombre maximum de lignes (ou de répétitions) par page est supérieur au nombre de lignes (ou de répétitions) du champ. 

Il est également possible de créer un champ Réglette et de l'associer par la suite à un champ Table ou à un champ Zone répétée.


### Création d'un champ de type Réglette
<a name="creation_champ_type_reglette_ELTPARAGRAPHE000019"></a>Pour créer un champ de type Réglette : 

1. Sous le volet "Création", dans le groupe "Navigation", cliquez sur "Réglette".

2. Cliquez dans la page à la position où le champ doit être créé. Le champ apparaît en création.




Pour afficher les caractéristiques du champ, sélectionnez l'option "Description" dans le menu contextuel du champ.

Pour associer la réglette à un champ Table ou à un champ Zone répétée existant : 

1. Affichez la fenêtre de description du champ Réglette. 

2. Sélectionnez le champ Table ou Zone répétée dans l'onglet "Général".

## Notes
<a name="notes_ELTTEXTE000176"></a>


### Envoyer la valeur des champs au serveur
<a name="envoyer_valeur_des_champs_serveur_ELTPARAGRAPHE000050"></a>L'option "Envoyer la valeur des champs au serveur" (onglet "Général" de la fenêtre de description) permet d'effectuer les opérations suivantes lors de l'utilisation du champ Réglette :

- valider les champs de la page en cours.

- renvoyer la valeur de ces champs au serveur.




Le serveur effectue alors l'action spécifiée.
<a name="NOTE2_2"></a>


### Description d'un champ Table et d'un champ Zone répétée
<a name="description_champ_table_champ_zone_repetee_ELTPARAGRAPHE000062"></a>Lors de la description d'un champ Table ou Zone répétée, il est possible de spécifier le nombre maximum de lignes (ou de répétitions) par page :

- Champ Table : onglet "Détail" des caractéristiques du champ Table.

- Champ Zone répétée : onglet "Général" des caractéristiques du champ Zone répétée.




Si le nombre de lignes (ou de répétitions) affichées dans la page est supérieur au nombre maximum spécifié, une réglette est automatiquement associée au champ Table ou Zone répétée. Le champ Réglette est affiché sous le champ Table et/ou Zone répétée. 

**Remarque** : Cette option est disponible uniquement pour les champs Table non Ajax. Les champs Table en mode "Serveur + AJAX" utilisent un ascenseur.
<a name="NOTE2_3"></a>


### Réglette AWP
<a name="reglette_awp_ELTPARAGRAPHE000078"></a>La réglette en mode AWP gère automatiquement le passage de paramètres, sans programmation. 

Ce type de réglette permet le référencement de toutes les pages liées et évite l'usage de cookies pour mémoriser la recherche en cours.

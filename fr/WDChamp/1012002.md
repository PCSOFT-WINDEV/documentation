## Créer un frameset vierge
<a name="creer_frameset_vierge_ELTTEXTE000240"></a>


Pour créer un frameset vierge :

1. Créez une page vierge : 

	- Cliquez sur ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_nouveau.gif) parmi les boutons d'accès rapide. La fenêtre de création d'un nouvel élément s'affiche : cliquez sur "Page" puis sur "Page". L'assistant de création d'une page se lance.

	- Cliquez sur "Page" puis sur "Vierge".

	- Validez.




2. Sous le volet "Page", dans le groupe "Edition", déroulez "Frame" et sélectionnez "Nouveau FrameSet vierge".

3. Ajoutez si nécessaire des frames à ce frameset : 

	- Sous le volet "Page", dans le groupe "Edition", déroulez "Frame" et sélectionnez "Vertical" pour réaliser un fractionnement vertical. 

	- Sous le volet "Page", dans le groupe "Edition", déroulez "Frame" et sélectionnez "Horizontal" pour réaliser un fractionnement horizontal.




4. Enregistrez le frameset :  sous le volet "Page", dans le groupe "Edition", déroulez "Frame" et sélectionnez "Enregistrer le frameset". Spécifiez le nom de chacune des pages contenues dans le frameset ainsi que le nom du frameset.
	Les pages et le frameset correspondent à des fichiers ".WWH". Par défaut, ces fichiers sont créés dans le répertoire principal du projet.

5. Associez les frames à des pages. Pour plus de détails, consultez [Associer une frame à une page](../WDChamp/1012012.md).




Remarque : Il est important de spécifier le type du frameset (statique ou dynamique) en fonction du type de pages affichées (onglet "Général" de la description du frameset). Pour plus de détails, consultez [Frameset statique et frameset dynamique](../WDChamp/1012007.md).

## Créer un frameset prédéfini
<a name="creer_frameset_predefini_ELTTEXTE000264"></a>
Pour créer un frameset prédéfini :

1. Cliquez sur ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_nouveau.gif) parmi les boutons d'accès rapide. 

	- La fenêtre de création d'un nouvel élément s'affiche : cliquez sur "Page" puis sur "Page".

	- L'assistant de création d'une page se lance.




2. Dans l'assistant, cliquez sur le modèle prédéfini "Frameset".  

3. Choisissez le type de frameset voulu.

4. Validez la création du frameset. Le frameset est créé.

5. Enregistrez le frameset : sous le volet "Page", dans le groupe "Edition", déroulez "Frame" et sélectionnez "Enregistrer le frameset". Spécifiez le nom de chacune des pages contenues dans le frameset ainsi que le nom du frameset. 
	Les pages et le frameset correspondent à des fichiers ".WWH". Par défaut, ces fichiers seront créés dans le répertoire principal du projet.

6. Associez les frames à des pages. Pour plus de détails, consultez [Associer une frame à une page](../WDChamp/1012012.md).




Remarque : Il est important de spécifier le type du frameset (statique ou dynamique) en fonction du type de pages affichées (onglet "Général" de la description du frameset). Pour plus de détails, consultez [Frameset statique et frameset dynamique](../WDChamp/1012007.md).


## Personnalisation du visualisateur de rapports : Fenêtres disponibles
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Fenêtres utilisées par le visualisateur de rapports
<a name="fenetres_utilisees_par_visualisateur_rapports_ELTTEXTE000165"></a>
Par défaut, ces fenêtres sont les suivantes :



|   |   |
| --- | --- |
| <br>![Fenêtre "Aperçu"](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=FenParam_apercu20.gif)<br>Cette fenêtre correspond à l'aperçu avant impression. |
| <br>![Fenêtre "WD_Filigrane"](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=FenParamFiligrane.gif)<br>Cette fenêtre permet de gérer les filigranes. |
| <br>![Fenêtre "WD_Imprimer_Etat"](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=FenParamImpr1.gif)<br>Cette fenêtre est utilisée lors de l'impression d'un état WINDEV depuis la fenêtre d'aperçu avant impression.<br>Cette fenêtre permet de paramétrer :<br><br>- les pages à imprimer.<br><br>- le nombre de copies à réaliser.<br><br>- la destination de l'impression (imprimante).<br><br>- les paramètres de l'imprimante, notamment l'orientation du papier.<br><br><br><br>Cette fenêtre permet également de connaître l'état de l'imprimante et son type. | <br>![Fenêtre "WD_Imprimer"](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=FenParamImpr2.gif)<br>Cette fenêtre est utilisée lors d'une impression réalisée en WLangage (fonction [iFinImprime](../WDLang5/3046053.md) utilisée après la fonction [iAperçu](../WDLang5/3046069.md)). <br>Cette fenêtre permet de paramétrer :<br><br>- les pages à imprimer.<br><br>- le nombre de copies à réaliser.<br><br><br>Cette fenêtre permet également de connaître :<br><br>- le nom de l'imprimante utilisée,<br><br>- l'état de l'imprimante utilisée<br><br>- le type de l'imprimante utilisée.<br><br><br> |
| ![Fenêtre "WD_Patience"](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=FenPatienceImpr.gif) |
| <br>![Fenêtre "WD_Installe_EtatsEtRequêtes"](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=FenParam_EtatsR20.gif)<br>Cette fenêtre est affichée par exemple lorsque l'utilisateur tente de modifier un état sans avoir installé le logiciel "Etats et requêtes". |
| <br>![Fenêtre "WD_PopupSelCouleur"](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=FEN_POP_Couleur.gif)<br>Cette fenêtre permet de sélectionner les couleurs utilisées pour les annotations. | <br>![Fenêtre "WD_PopupSelEpaisseur"](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=FEN_POP_Epaisseur.gif)<br>Cette fenêtre permet de sélectionner l'épaisseur du trait utilisé pour les annotations. |
| <br>![Fenêtre "WD_PopupGroupeEtat"](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=FenParam_format.gif)<br>Cette fenêtre permet de sélectionner le type d'état à imprimer (cas d'un groupe d'états ou d'états avec ancrage). | <br>![Fenêtre "WD_PopupImprimante"](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=FenParam_ListeImp.gif)<br>Cette fenêtre permet de sélectionner l'imprimante à utiliser. |
| <br>![Fenêtre "WD_PopupOptionMire"](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=fenparam_mire.gif)<br>Cette fenêtre permet de définir les options de la mire de l'aperçu. |
| <br>![Fenêtre "WD_SelImprimante"](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=fenparam_imp.gif)<br>Cette fenêtre permet d'afficher le message provenant de l'imprimante. |

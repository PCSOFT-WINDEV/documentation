


## Tableau HTML
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000198"></a>
Le tableau HTML est un champ qui permet de faire des présentations spécifiques dans des pages HTML. Le tableau HTML ne permet pas de positionner des champs mais permet de présenter les informations.

Par exemple, grâce aux tableaux HTML il est possible de faire des énumérations avec des bordures et des colonnes.
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Tableau_HTML.gif)

<a name="NOTE2_1"></a>


## Création d'un tableau HTML
<a name="creation_tableau_html_ELTTEXTE000222"></a>
**Pour créer un tableau HTML** : 

- sous le volet "Création", dans le groupe "Conteneurs", cliquez sur "Tableau HTML". Dans l'assistant qui se lance, sélectionnez les caractéristiques du tableau HTML à créer : 

	- Nombre de lignes.

	- Nombre de colonnes.

	- Style du tableau et alternance des lignes et des colonnes.




- sous le volet "Création", dans le groupe "Conteneurs", cliquez sur "Tableau HTML" et sélectionnez directement le nombre de lignes et de colonnes du tableau. Dans l'assistant qui se lance, sélectionnez les caractéristiques de style du tableau et l'alternance des lignes et des colonnes. 




Lorsque le tableau HTML est créé dans la page, il est possible de :

- le déplacer grâce à l'icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Tableau_HTML_deplacement.gif)
.

- manipuler sa description grâce au menu contextuel du champ. Le menu contextuel de ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Tableau_HTML_deplacement.gif)
 permet de manipuler le tableau, le menu contextuel d'une cellule permet de manipuler les cellules du tableau.

## Utilisation d'un tableau HTML
<a name="utilisation_tableau_html_ELTTEXTE000246"></a>
Dans chaque cellule d'un tableau HTML, il est possible de mettre un champ. Le champ et la cellule ont la même taille. Le champ et la cellule ont les mêmes propriétés de bordure et de fond.

Il n'est pas possible de mettre plusieurs champs dans une cellule d'un tableau HTML.

Cependant, il est possible de mettre une cellule HTML contenant plusieurs champs dans une cellule d'un tableau HTML.

## Manipulation d'un tableau HTML
<a name="manipulation_tableau_html_ELTTEXTE000270"></a>


### Sélectionner une cellule
<a name="selectionner_une_cellule_ELTPARAGRAPHE000079"></a>Pour sélectionner une cellule vide, il suffit de :

1. Survoler la cellule avec la souris.

2. Cliquer dans la cellule lorsque le curseur de souris se transforme en flèche noir en diagonale. La cellule est alors sélectionnée : des hachures apparaissent dans la cellule.![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Tableau_HTML_Cellule_select.gif)



Dans ce cas, l'option "Description" du menu contextuel du tableau permettra d'accéder à la description de la cellule.

Pour sélectionner une cellule associée à un champ, il suffit de sélectionner le champ.
Dans ce cas, l'option "Description" du menu contextuel du tableau, permettra d'accéder à la description du champ. 
Pour obtenir la description de la cellule, il suffit de sélectionner l'option "Tableau HTML .. Description" du menu contextuel.
<a name="NOTE4_2"></a>


### Sélectionner une colonne ou une ligne
<a name="selectionner_une_colonne_une_ligne_ELTPARAGRAPHE000098"></a>Pour sélectionner une colonne ou une ligne, il suffit de :

1. Positionner la souris à l'extrémité de la colonne ou de la ligne.

2. Cliquer lorsque le curseur de souris se transforme en flèche noire verticale (pour une colonne) ou horizontale (pour une ligne). La ligne ou la colonne est alors sélectionnée : des hachures apparaissent.




Il est alors possible grâce au menu contextuel de :

- fractionner verticalement ou horizontalement l'élément sélectionné

- fusionner les différentes parties de l'élément sélectionné.



<a name="NOTE4_3"></a>


### Supprimer un tableau HTML
<a name="supprimer_tableau_html_ELTPARAGRAPHE000113"></a>Pour supprimer un tableau HTML, il suffit de sélectionner l'option "Tableau HTML .. Supprimer le tableau" du menu contextuel.

## Modifier la taille d'un tableau HTML
<a name="modifier_taille_tableau_html_ELTTEXTE000306"></a>


### Modifier le nombre de cellules du tableau
<a name="modifier_nombre_cellules_tableau_ELTPARAGRAPHE000122"></a>Pour modifier le nombre de cellules d'un tableau HTML, il est possible :

- d'ajouter des lignes ou des colonnes une par une grâce au menu contextuel (option "Tableau HTML .. Insérer une ligne" ou "Tableau HTML .. Insérer une colonne")

- d'ajouter plusieurs lignes ou colonnes grâce au menu contextuel (option "Tableau HTML .. Agrandir le tableau")



<a name="NOTE5_2"></a>


### Modifier la taille du tableau
<a name="modifier_taille_tableau_ELTPARAGRAPHE000132"></a>La taille du tableau peut être modifiée de deux façons différentes :

- En agrandissant ou en réduisant proportionnellement toutes les lignes / colonnes du tableau : il suffit de réduire ou agrandir le tableau en utilisant la poignée du milieu de la dernière ligne ou colonne du tableau.![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Tableau_HTML_Agrandissement.gif)


- en agrandissant ou en réduisant uniquement la dernière ligne / colonne du tableau : il suffit de réduire ou d'agrandir le tableau en cliquant sur le bord de la dernière ligne ou colonne du tableau.![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Tableau_HTML_Agrandis_col.gif)







## Editer le style d'un tableau HTML
<a name="editer_style_tableau_html_ELTTEXTE000336"></a>
Il est possible d'éditer séparément les caractéristiques du bord et du fond du tableau HTML et celles de chacune des cellules du tableau HTML. Les propriétés de style du bord et de style du fond sont les mêmes que pour les autres champs. 

Le style de chaque élément du tableau HTML peut également être modifié. Pour cela :

1. Affichez la fenêtre de description du tableau HTML.

2. Dans l'onglet "Style", sélectionnez l'élément à personnaliser dans la combo "Elément".

3. Les propriétés configurables pour l'élément sélectionné sont affichées.


**Remarque** : L'onglet "Style" permet également de lancer un assistant pour créer un style automatique pour le tableau HTML. Il suffit de cliquer sur le lien "Lancer l'assistant de création de style".

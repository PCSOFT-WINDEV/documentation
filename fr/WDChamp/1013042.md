
## Manipuler un champ d'affichage formaté par programmation
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000115"></a>
WEBDEV permet de manipuler un champ d'affichage formaté directement par programmation. Il suffit d'utiliser directement la variable du champ d'affichage formaté dans le code.

La variable du champ d'affichage formaté correspond au nom du champ d'affichage formaté. Le type de la variable dépend du type du champ (numérique, texte, monétaire, ...). 



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Initialiser un champ d'affichage formaté (code serveur)
<a name="initialiser_champ_affichage_formate_code_serveur_ELTTEXTE000139"></a>
Pour initialiser un champ d'affichage formaté, une simple affectation de la variable du champ suffit. Cette initialisation doit être réalisée en code serveur. Par exemple :


```wl
// Initialisation du champ d'affichage formaté "LIBF_Date"
LIBF_Date = 19990506 
// Autre possibilité : Date.Valeur = "19990506"
```


**Remarque** : il est également possible d'utiliser la propriété [Valeur](../Proprietes/2510130.md) ou la propriété [Libellé](../Proprietes/2510053.md).



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Récupérer la valeur d'un champ d'affichage formaté (code serveur)
<a name="recuperer_valeur_champ_affichage_formate_code_serveur_ELTTEXTE000163"></a>
Pour récupérer le contenu d'un champ d'affichage formaté (statique ou dynamique) dans un code serveur :

- réalisez une simple affectation. Par exemple :
	
	```wl
	ContenuActuelAff = LIBF_FORMATE1
	```


- manipulez directement le champ d'affichage formaté. Par exemple :
	
	```wl
	Info(LIBF_FORMATE1)
	```


- utilisez la propriété [Valeur](../Proprietes/2510130.md) ou la propriété [Libellé](../Proprietes/2510053.md).

## Exemple d'utilisation
<a name="exemple_utilisation_ELTTEXTE000187"></a>


### Comment afficher dans une page le libellé "date de dernière mise à jour : jj/mm/aaaa" :
<a name="comment_afficher_dans_une_page_libelle_date_derniere_mise_jour_jjmmaaaa_ELTPARAGRAPHE000061"></a>

1. Créer un champ d'affichage formaté.

2. Dans le code d'initialisation du champ, saisissez le code suivant :
	
	```wl
	MoiMême = fDate(ComplèteRep(fRepExe()) + "MonSite.WDL", "", fModification)
	```
Où "MonSite.WDL" correspond à la WDL de votre site.

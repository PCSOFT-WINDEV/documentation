## Présentation du Champ Web Caméra
<a name="presentation_champ_web_camera_ELTTEXTE000104"></a>


### Champ d'une page
<a name="champ_une_page_ELTPARAGRAPHE000011"></a>Le Champ Web Caméra permet de visualiser un fichier rafraîchi à intervalle régulier. Par exemple, ce fichier peut correspondre à une image retransmise par une "web caméra".


### Création du champ
<a name="creation_champ_ELTPARAGRAPHE000016"></a>Pour créer un champ Web Caméra :

1. Sous le volet "Création", dans le groupe "Champs usuels", déroulez "Image" et cliquez sur "Web Caméra".

2. Cliquez dans la page à la position où le champ doit être créé. Le champ apparaît en création.


Pour afficher les caractéristiques du champ, sélectionnez l'option "Description" dans le menu contextuel du champ.

## Configuration nécessaire
<a name="configuration_necessaire_ELTTEXTE000134"></a>


### Récupérer une image retransmise par une "Web caméra"
<a name="recuperer_une_image_retransmise_par_une_web_camera_ELTPARAGRAPHE000042"></a>Pour récupérer une image retransmise par une "Web caméra" :

1. **Sur le poste relié à la "web caméra"** : paramétrez l'enregistrement de l'image retransmise par une "web caméra" (nom du fichier, cadence de rafraîchissement, "web caméra" utilisée, ...). Ce paramétrage peut être réalisé :

	- soit avec le logiciel livré avec la "web caméra".

	- soit avec l'application "WD WebCam.exe" disponible sur le site de [PC SOFT](https://www.pcsoft.fr/st/telec/index.html) (page de téléchargement, paragraphe "Autres logiciels").
			Le fichier image doit être accessible depuis n'importe quel poste utilisant le site WEBDEV.
			**Remarque** : Le code source de "WD WebCam" est disponible dans les exemples fournis avec WINDEV.




2. **Sur le poste de développement** : indiquez le fichier image relié au champ Web Caméra (onglet "Général" de la fenêtre de description du champ).

## Evénements gérés par défaut
<a name="evenements_geres_par_defaut_ELTTEXTE000076"></a>
WEBDEV gère par défaut les événements suivants (ordre d'apparition sous l'éditeur de code) :

| Evénement | Condition d'exécution |
| --- | --- |
| Initialisation (Code serveur) | Exécuté à l'ouverture de la page. |
| Clic sur le champ (Code navigateur) | Exécuté lorsque l'internaute clique sur le champ. |
| Clic sur le champ (Code serveur) | Exécuté lorsque l'internaute clique sur le champ. |


## Manipuler un champ Notation par programmation
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000164"></a>
WINDEV, WEBDEV et WINDEV Mobile permettent de manipuler un champ Notation directement par programmation. Il suffit d'utiliser directement la variable du champ Notation dans le code.

La variable du champ Notation :

- correspond au nom du champ Notation.

- est initialisée avec la note affichée par le champ Notation.






<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Récupérer la note affichée dans le champ
<a name="recuperer_note_affichee_dans_champ_ELTTEXTE000188"></a>
Pour récupérer la note, utilisez la syntaxe suivante :


```txt
<Note> = <Champ Notation>
```


&lt;Note&gt; est une variable de type entier ou réel, selon la granularité définie pour les notes dans la fenêtre de description du champ. 

**Remarques** : 

- Il est également possible d'utiliser la propriété [Valeur](../Proprietes/2510130.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) La &lt;Note&gt; prend en compte la valeur par étoile spécifiée (soit dans la description du champ, soit par la propriété [ValeurParEtoile](../Proprietes/1000021996.md)). 






<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Modifier la note affichée dans le champ
<a name="modifier_note_affichee_dans_champ_ELTTEXTE000212"></a>
Pour modifier la note, utilisez la syntaxe suivante :


```txt
<Champ Notation> = <Note>
```


&lt;Note&gt; est une valeur comprise entre la note minimale et la note maximale du champ Notation. Ces valeurs ont été décrites dans la fenêtre de description du champ ou par programmation grâce aux propriétés [BorneMin](../Proprietes/2510008.md) et [BorneMax](../Proprietes/2510009.md).

**Remarques** : 

- Il est également possible d'utiliser la propriété [Valeur](../Proprietes/2510130.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Attention : si une valeur par étoile est spécifiée (soit dans la description du champ, soit par la propriété [ValeurParEtoile](../Proprietes/1000021996.md)), la &lt;Note&gt; doit prendre en compte cette valeur (valeur d'une étoile multipliée par le nombre d'étoiles affichées).

## Propriétés spécifiques à la gestion d'une notation
<a name="proprietes_specifiques_gestion_une_notation_ELTTEXTE000236"></a>
Les propriétés suivantes sont spécifiques à la gestion des caractéristiques d'un champ Notation par programmation.


|   |   |
| --- | --- |
| [BorneMax](../Proprietes/2510009.md) | Permet de connaître et de modifier la borne maximale d'un champ Notation. |
| [BorneMin](../Proprietes/2510008.md) | Permet de connaître et de modifier la borne minimale d'un champ Notation. Cette propriété est notamment utile pour les champs Notation utilisés par exemple pour donner un avis du type "J'aime / J'aime pas".<br><br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Notation_Jaime.gif)<br> |
| [ValeurParEtoile](../Proprietes/1000021996.md) | Permet de connaître et de modifier la valeur d'une étoile dans un champ Notation. |


Pour connaître la liste complète des propriétés du WLangage pouvant être utilisées avec un champ de type Notation, consultez [Propriétés associées au champ Notation](../WDChamp/1000019744.md).

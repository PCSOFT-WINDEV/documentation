## Présentation du Champ Caméra
<a name="presentation_champ_camera_ELTTEXTE000172"></a>


### Champ d'une fenêtre
<a name="champ_une_fenetre_ELTPARAGRAPHE000011"></a>Le Champ Caméra permet de : 

- ![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png) visualiser la séquence vidéo actuellement retransmise par une caméra connectée au poste en cours.




Il est possible d'enregistrer cette séquence vidéo ou une photo de cette séquence vidéo.

**Cette page d'aide présente l'utilisation du champ Caméra dans une fenêtre WINDEV. Sous WINDEV Mobile, consultez [Champ Caméra (application mobile)](../WDChamp/1013349.md).**




### Créer un champ Caméra dans une fenêtre WINDEV
<a name="creer_champ_camera_dans_une_fenetre_windev_ELTPARAGRAPHE000055"></a>Pour créer un champ Caméra :

1. Sous le volet "Création", dans le groupe "Champs graphiques", déroulez "Vidéo et capture" et sélectionnez "Caméra".

2. Cliquez dans la fenêtre à la position où le champ doit être créé. Le champ apparaît en création.




Pour afficher les caractéristiques du champ, sélectionnez l'option "Description" dans le menu contextuel du champ.

Le champ Caméra peut être manipulé par programmation grâce aux [fonctions Caméra](../WDLang1/1410086597.md). 
Remarque : Ces fonctions sont également disponibles avec une syntaxe préfixée. Pour plus de détails, consultez [Fonctions Caméra (Syntaxe préfixée)](../WDLang1/1000022386.md).

**Remarques** : 

- Le champ Caméra fonctionne sous toutes les versions de Windows, quels que soient les modèles de pilote installés, grâce à la technologie DirectShow. Le comportement des versions précédentes peut être retrouvé en cochant l'option "Utiliser les anciennes API 'Video for Windows' (compatibilité)". 
	Remarque : Il est déconseillé d'utiliser un champ Caméra utilisant la technologie DirectShow et un champ Caméra compatible dans une même fenêtre.

- Pour récupérer une image retransmise par une Caméra, indiquez le driver vidéo utilisé (onglet "Général" de la fenêtre de description du champ). Ce driver peut être défini par programmation grâce à la propriété [Driver](../Proprietes/2510061.md) ou grâce à la fonction [CaméraChangeDriver](../WDLang1/1000022451.md).

## Visualiser une séquence vidéo
<a name="visualiser_une_sequence_video_ELTTEXTE000202"></a>
Seule la séquence vidéo actuellement retransmise peut être visualisée dans un champ Caméra.

Pour visualiser :

- une image capturée avec la fonction [CaméraCapture](../WDLang1/1000022450.md), utilisez un [champ Image](../WDChamp/1013218.md).

- une séquence vidéo capturée avec la fonction [CaméraCapture](../WDLang1/1000022450.md), utilisez un [champ Multimédia](../WDChamp/9500200.md).




## Programmer une barre de message
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000150"></a>
WINDEV permet de manipuler une barre de message et une case d'une barre de message directement par programmation. Il suffit d'utiliser directement la variable de la barre de message ou de la case dans le code.

La variable d'une barre de message correspond au nom de la barre de message. La variable d'une case d'une barre de message correspond au nom de la case.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Ajouter une case dans la barre de message de la fenêtre en cours
<a name="ajouter_une_case_dans_barre_message_fenetre_cours_ELTTEXTE000174"></a>


### Comment le faire ? 
<a name="comment_faire_ELTPARAGRAPHE000020"></a>Pour ajouter une case par programmation dans une barre de message, il suffit d'utiliser la fonction [MessageAjouteCase](../WDLang1/3021001.md).

Cette case peut ensuite être manipulée par diverses propriétés.
 
**Attention** : Le nom d'une case créée avec la fonction [MessageAjouteCase](../WDLang1/3021001.md) doit être entouré d'opérateurs d'indirection ({ }) pour être manipulée avec les propriétés WLangage.
<a name="NOTE2_2"></a>


### Exemple
<a name="exemple_ELTPARAGRAPHE000037"></a>L'exemple suivant permet d'ajouter une case dans la barre de message de la fenêtre en cours.

Cette case affiche les coordonnées du curseur de la souris. Le fond de cette case est rouge. La police utilisée dans cette case est en gras.

```wl
// -- Code d'initialisation de la fenêtre
// Ajoute la case "MaCase" dans la barre de message de la fenêtre en cours
MessageAjouteCase("MaCase", 100, RougeClair) 
// Modifie la graisse des informations affichées dans la case "MaCase"
// Les opérateurs d'indirection sont nécessaires avec une case créée 
// avec la fonction MessageAjouteCase
{"MaCase"}..PoliceGras = Vrai
```



```wl
// -- Code optionnel de survol de la souris
// Affiche les coordonnées du curseur de la souris dans la case "MaCase"
Message("MaCase", SourisPosX() + " , " + SourisPosY())
```

## Fonctions WLangage associées aux barres de message
<a name="fonctions_wlangage_associees_aux_barres_message_ELTTEXTE000204"></a>
Les différents fonctions associées aux barres de message sont les suivantes :


|   |   |
| --- | --- |
| [Jauge](../WDLang1/3021008.md) | Affiche une jauge dans la barre de message de la fenêtre en cours. |
| [Message](../WDLang1/3021006.md) | Affiche (ou efface) un message dans la barre de message de la fenêtre en cours. |
| [MessageAjouteCase](../WDLang1/3021001.md) | Ajoute une case dans la barre de message de la fenêtre en cours. |
| [MessageSupprimeCase](../WDLang1/3021002.md) | Supprime une case de la barre de message de la fenêtre en cours. |


**Remarque** : Le nom d'une case créée avec la fonction [MessageAjouteCase](../WDLang1/3021001.md) doit être entouré d'opérateurs d'indirection pour être manipulée avec les propriétés WLangage.

<a name="NOTE4_1"></a>


## Propriétés WLangage permettant de gérer les barres de messages
<a name="proprietes_wlangage_permettant_gerer_les_barres_messages_ELTTEXTE000228"></a>
Les principales propriétés permettant de gérer les barres de messages sont les suivantes :



|   |   |
| --- | --- |
| [BarreDeMessage](../Proprietes/2510007.md) | Permet de savoir si la barre de messages de la fenêtre est activée et permet de l'activer si nécessaire. |
| [Message](../WDLang1/3021006.md) | Permet de connaître et de modifier le message d'aide associé au champ ou à la fenêtre. |


Des propriétés spécifiques sont également disponibles sur chaque case d'une barre de message. Pour plus de détails, consultez [Propriétés associées aux cases des barres de message](../WDChamp/1010035.md).

## Présentation des fenêtres
<a name="presentation_des_fenetres_ELTTEXTE000195"></a>
Les fenêtres constituent l'interface principale d'une application WINDEV ou WINDEV Mobile. En effet, c'est grâce aux fenêtres que les utilisateurs vont pouvoir consulter et manipuler des informations. Ces informations peuvent être dynamiques et provenir par exemple de fichiers, requêtes ou être statiques, saisies directement dans les champs de la fenêtre.

## Créer une fenêtre
<a name="creer_une_fenetre_ELTTEXTE000219"></a>
Pour créer une fenêtre : 

1. Cliquez sur ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_nouveau.gif) parmi les boutons d'accès rapide : 

	- La fenêtre de création d'un nouvel élément s'affiche : cliquez sur "Fenêtre" puis sur "Fenêtre".

	- L'assistant de création d'une fenêtre se lance.




2. Sélectionnez le type de fenêtre à créer :

	- l'onglet "Standard" permet notamment de créer des fenêtres vierges, sans aucun champ.

		- ![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) L'option "Vierge" permet de créer des fenêtres maximisées ou non maximisées. 




- l'onglet "RAD" permet de créer des [fenêtres de type RAD](../Editeurs/2031003.md). Ces fenêtres permettent d'afficher et de saisir des données. Ce type de fenêtre peut être basé sur un fichier de données, une requête, ...

- l'onglet "Fenêtre interne" permet de créer des fenêtres pouvant être utilisées par un champ [Fenêtre interne](../WDChamp/1013255.md).

- l'onglet "Basée sur un modèle" permet de créer une fenêtre utilisant un [modèle de fenêtres](../WDChamp/9000086.md) livré ou créé précédemment.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) l'onglet "Import" permet de créer une fenêtre à partir de [l'importation d'une fenêtre non WINDEV existante](../Editeurs/2026003.md).




**Pour afficher les caractéristiques de la fenêtre**, sélectionnez l'option "Description" dans le menu contextuel de la fenêtre.


## Les graphes de type "Aire Polaire"
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000104"></a>
Le graphe de type Aire Polaire est similaire aux graphes de type Secteur, avec un angle identique pour chaque part. La valeur n’est pas représentée par la largeur de la part mais par sa longueur.

Ce type de graphe est principalement utilisé pour comparer des objectifs ou des parts de marché, ou illustrer une progression.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Caractéristiques d'un graphe Aire Polaire
<a name="caracteristiques_graphe_aire_polaire_ELTTEXTE000128"></a>

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=graphe_Polar%20-%20HC%20N%B0001.gif&type=thumb)
Dans ce graphe :

**1. Titre du graphe**. 
Les caractéristiques du titre du graphe sont saisies dans l'onglet "Détail" de la description du champ (bouton "Editer le titre du graphe").
Il est possible de définir le titre du graphe, sa position et son style.

**2. Valeur du graphe**
L'angle de la part représentant la valeur est identique pour toutes les valeurs du graphe. Seule la longueur de la part varie selon la valeur de la série. La couleur de parts est définie automatiquement. Pour modifier la couleur des parts, utilisez la fonction [grCouleurSérie](../WDLang3/3042013.md).

**3. Etiquette de catégorie**. 
Les étiquettes de catégories sont affichées dans la légende. Les étiquettes de catégorie peuvent être définies dans l'onglet "Séries" de la description du champ. La source des étiquettes de catégorie peut correspondre à une colonne de table, un tableau WLangage, une liste, etc. Pour afficher les étiquettes de catégorie, il suffit de cocher l'option "Afficher les étiquettes des catégories" dans l'onglet "Général" de la fenêtre de description du graphe.

**4. Légende**. 
Les caractéristiques de la légende sont saisies dans l'onglet "Détail" de la description du champ (bouton "Editer la légende du graphe"). 
Il est possible de définir sa position et son style.

**5. Graduation**. 
La graduation est calculée automatiquement.

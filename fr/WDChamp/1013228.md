<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000210"></a>
Le champ Sélecteur est un champ permettant à l'utilisateur de sélectionner une option parmi plusieurs.

Un sélecteur peut afficher jusqu'à 256 options, disposées sur plusieurs lignes et plusieurs colonnes. L'utilisateur peut sélectionner uniquement une option. Le sélecteur renvoie le numéro de l'option sélectionnée.

## Création d'un champ Sélecteur
<a name="creation_champ_selecteur_ELTTEXTE000234"></a>


### Créer un champ de type sélecteur
<a name="creer_champ_type_selecteur_ELTPARAGRAPHE000063"></a>Pour créer un champ de type sélecteur :

1. Sous le volet "Création", dans le groupe "Champs usuels", cliquez sur "Sélecteur".

2. Cliquez dans la fenêtre ou dans la page à la position où le champ doit être créé. Le champ apparaît en création.




![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) **Astuce** : Il est également possible de créer directement un sélecteur "pré-formaté" en déroulant "Sélecteur". La liste des champs disponibles avec leur aperçu au gabarit du projet apparaît. Il suffit de cliquer sur le champ voulu pour le créer dans l'éditeur.

Pour afficher les caractéristiques du champ, sélectionnez l'option "Description" dans le menu contextuel du champ.

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) **Remarque** : Il est possible de gérer soit une bulle d'aide générale pour le champ, soit une bulle d'aide pour chacune des options du champ.
<a name="NOTE2_2"></a>
![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) 

### Limites en Java
<a name="limites_java_ELTPARAGRAPHE000097"></a>En Java, les caractéristiques suivantes ne sont pas gérées :

- colonnes proportionnelles (si les options du sélecteur sont affichées sur plusieurs colonnes).

- libellé des options du sélecteur multiligne.

- texte des options du sélecteur avec mise en forme (R. T. F.).

- numéro d'aide associé à chaque option du sélecteur.

## Les différents modes d'affichage proposés
<a name="les_differents_modes_affichage_proposes_ELTTEXTE000270"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) 
WINDEV et WINDEV Mobile proposent différents types d'affichage des sélecteurs :

- sélecteur standard,

- sélecteur standard avec sous-libellés, 

- sélecteur au look Vista (avec possibilité d'utiliser une image sur chaque option), 

- sélecteur au look Vista avec sous-libellés (avec possibilité d'utiliser une image sur chaque option). 


Le mode d'affichage du sélecteur est sélectionné dans l'onglet "Général" de la description du champ. L'onglet "Contenu" permet de saisir les caractéristiques propres au mode d'affichage (image, sous-libellé, hauteur de ligne, ...) dans les différentes langues de l'application. Ces éléments sont modifiables par programmation.

Selon le type d'affichage choisi, le mode de programmation peut varier. Pour plus de détails, consultez [Manipuler un sélecteur par programmation](../WDChamp/1013254.md).

![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Seuls les modes standard et standard avec sous-libellés sont proposés.

## Permuter
<a name="permuter_ELTTEXTE000294"></a>
WINDEV, WINDEV Mobile et WEBDEV permettent de transformer automatiquement un champ Sélecteur en un champ Interrupteur et inversement : 

1. Sélectionnez le champ à transformer. 

2. Sous le volet "Modification", dans le groupe "Transformations", déroulez "Refactoring et permutations" et sélectionnez "Sélecteur / Interrupteur".

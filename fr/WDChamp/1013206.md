
## Manipuler un champ OLE par programmation
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000096"></a>
WINDEV permet de manipuler un champ OLE directement par programmation. Il suffit d'utiliser directement la variable du champ dans le code.

La variable du champ OLE correspond au nom du champ OLE.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Connaître le nom du fichier associé au champ OLE
<a name="connaitre_nom_fichier_associe_champ_ole_ELTTEXTE000120"></a>
Pour connaître le nom du fichier OLE associé à un champ, la syntaxe est la suivante :


```wl
OLE_OLE1 = NomFichier // OLE_OLE1 est le nom du champ
```




<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Modifier le nom du fichier associé à un champ OLE
<a name="modifier_nom_fichier_associe_champ_ole_ELTTEXTE000144"></a>
Pour modifier le nom du fichier associé à un champ, la syntaxe est la suivante :


```wl
OLE_OLE1 = NomFichier // OLE_OLE1 est le nom du champ
// Cette ligne équivaut à : OLECharge("OLE_OLE1", NomFichier)
```




<a name="NOTE4"></a>
<a name="NOTE4_1"></a>


## Fonctions de programmation des champs OLE
<a name="fonctions_programmation_des_champs_ole_ELTTEXTE000168"></a>
WINDEV permet de manipuler simplement les champs OLE grâce à des fonctions spécifiques du WLangage. Pour plus de détails, consultez [Fonctions OLE du WLangage](../WDLang3/3057007.md).

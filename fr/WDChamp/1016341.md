
## Manipuler un Range Slider par programmation
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000246"></a>
WINDEV permet de manipuler un champ Range Slider directement par programmation. Il suffit d'utiliser directement la variable du champ Range Slider dans le code.

La variable du champ Range Slider :

- correspond au nom du champ Range Slider.

- est initialisée avec la valeur du champ Range Slider.






<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Récupérer les caractéristiques du champ Range Slider
<a name="recuperer_les_caracteristiques_champ_range_slider_ELTTEXTE000270"></a>




### Caractéristiques récupérables par programmation
<a name="caracteristiques_recuperables_par_programmation_ELTPARAGRAPHE000024"></a>Les caractéristiques récupérables par programmation sont les suivantes : 

- [la valeur du champ Range Slider](#NOTE2_2),

- [les bornes du champ (minimale et maximale)](#NOTE2_3),

- [l'étendue totale du champ Range Slider](#NOTE2_4),

- [l'étendue visible du champ Range Slider](#NOTE2_4).



<a name="NOTE2_2"></a>


### Valeur du champ Range Slider
<a name="valeur_champ_range_slider_ELTPARAGRAPHE000044"></a>La valeur du champ Range Slider correspond à : 

- la position de l'extrémité gauche de la cage d'un champ Range Slider horizontal. 

- la position de l'extrémité haute de la cage d'un champ Range Slider vertical. 




Pour récupérer la valeur du champ Range Slider, utilisez la syntaxe suivante :


```txt
<Position> = <Nom Range Slider>
```


&lt;Position&gt; est une variable de type entier.

**Remarque** : Il est également possible d'utiliser la propriété [Valeur](../Proprietes/2510130.md).
<a name="NOTE2_3"></a>


### Connaître les bornes du champ Range Slider
<a name="connaitre_les_bornes_champ_range_slider_ELTPARAGRAPHE000065"></a>Les propriétés [BorneMax](../Proprietes/2510009.md) et [BorneMin](../Proprietes/2510008.md) permettent de connaître les bornes maximales et minimales du champ Rang Slider. 

La **borne minimale** correspond à la valeur de l'extrémité du champ Range Slider : 

- extrémité gauche pour un champ Range Slider horizontal

- extrémité haute pour un champ Range Slider vertical. 




La **borne maximale** correspond à la valeur maximale que peut prendre le champ Range Slider avec la configuration actuelle de la cage du champ Range Slider. 

Exemple : 

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=RangeSlider_Bornes.gif)


Dans cet exemple, en position 1 : 

- la valeur du champ Range Slider est 21. 

- la borne minimale est 1. 

- la borne maximale est 40. Cette borne maximale correspond à la valeur du champ Range Slider lorsque la cage du champ Range Slider est positionnée à l'extrémité maximale. Ce cas correspond à la position 2. Cette borne correspond au calcul suivant : Borne minimale + (Etendue totale - Etendue visible). 



<a name="NOTE2_4"></a>


### Connaître l'étendue totale et l'étendue visible du champ Range Slider
<a name="connaitre_etendue_totale_etendue_visible_champ_range_slider_ELTPARAGRAPHE000095"></a>Les propriétés [EtendueTotale](../Proprietes/1000019066.md) et [EtendueVisible](../Proprietes/1000019067.md) permettent de connaître les étendues totales et visibles du champ Range Slider.

## Modifier la valeur du champ Range Slider
<a name="modifier_valeur_champ_range_slider_ELTTEXTE000312"></a>
Pour modifier la valeur du champ Range Slider, utilisez la syntaxe suivante :


```txt
<Nom Range Slider> = <Valeur>
```


&lt;Valeur&gt; est une valeur comprise entre la valeur minimale et la valeur maximale du champ Range Slider. Ces valeurs ont été décrites dans la fenêtre de description du champ ou par programmation, grâce aux propriétés [BorneMin](../Proprietes/2510008.md) et [BorneMax](../Proprietes/2510009.md) (la borne maximale dépendant de l'étendue visible et de l'étendue totale). 

Par exemple, pour un champ Range Slider dont les caractéristiques sont les suivantes : 

- BorneMin = 1

- EtendueTotale = 5

- EtendueVible   = 2


La borne maximale sera 4 (1 + 5 – 2)

Les valeurs possibles seront 1, 2, 3 et 4

**Remarque** : Il est également possible d'utiliser la propriété [Valeur](../Proprietes/2510130.md) pour modifier la valeur du champ Range Slider.

## Propriétés spécifiques à la gestion d'un champ Range Slider
<a name="proprietes_specifiques_gestion_champ_range_slider_ELTTEXTE000336"></a>
Les propriétés suivantes sont spécifiques à la gestion des caractéristiques d'un champ Range Slider par programmation.


|   |   |
| --- | --- |
| [Avance](../Proprietes/1000021574.md) | Permet de connaître et de modifier la valeur du pas d'avancement d'un champ Range Slider. |
| [AvanceRapide](../Proprietes/2510011.md) | Permet de connaître et de modifier la valeur du pas d'incrémentation d'un champ Range Slider. |
| [BorneMax](../Proprietes/2510009.md) | Permet de connaître et de modifier la borne maximale d'un champ Range Slider. |
| [BorneMin](../Proprietes/2510008.md) | Permet de connaître et de modifier la borne minimale d'un champ Range Slider. |
| [EtendueTotale](../Proprietes/1000019066.md) | Permet de connaître et de modifier le nombre total de données représenté par le champ Range Slider. |
| [EtendueVisible](../Proprietes/1000019067.md) | Permet de connaître et de modifier la largeur de la cage d'ascenseur du champ Range Slider. |


Pour connaître la liste complète des propriétés du WLangage pouvant être utilisées avec un champ de type Range Slider, consultez [Propriétés associées au champ Range Slider](../WDChamp/1016342.md).

## Exemple de code permettant d'utiliser un champ Range Slider pour zoomer sur une image
<a name="exemple_code_permettant_utiliser_champ_range_slider_pour_zoomer_sur_une_image_ELTTEXTE000360"></a>
Le code suivant est appelé à chaque modification d'un champ Range Slider et permet d'effectuer un zoom sur une zone d'image déterminée par l'étendue visible du champ Range Slider. 


```wl
PROCEDURE AfficherImage()

// On copie la zone correspondant à la barre de zoom des deux champs Range Slider
// (1 horizontal et 1 vertical) dans la totalité du champ Image de destination
dCopieImage(IMG_SOURCE, IMG_DESTINATION, copieSrcCopie, ...		// Copie "directe"
		// Borne de départ du Range Slider
		RGS_HORIZONTAL.Valeur, RGS_VERTICAL.Valeur, ...
		// Toute la largeur de la zone de zoom
		RGS_VERTICAL.EtendueVisible, RGS_HORIZONTAL.EtendueVisible, ... 
		// Copie dans le champ destination en haut à gauche
		1, 1, ...
		// Copie dans le champ destination sur toute la surface
		IMG_DESTINATION.Hauteur, IMG_DESTINATION.Largeur)
```

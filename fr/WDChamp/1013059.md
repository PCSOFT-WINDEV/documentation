## Présentation du champ Rubrique
<a name="presentation_champ_rubrique_ELTTEXTE000129"></a>


### Champ d'un état
<a name="champ_etat_ELTPARAGRAPHE000014"></a>Le champ "Rubrique" permet d'éditer le contenu d'une rubrique d'un fichier de données. La rubrique peut être une rubrique du fichier de données de base de l'état ou d'un fichier de données accessible par une liaison. Ce champ peut également être affecté par programmation.



**Pour créer un champ "Rubrique"** :

1. Sous le volet "Création", dans le groupe "Champs usuels", cliquez sur "Rubrique".

2. Cliquez dans l'état à la position où le champ doit être créé.

## Comment lier un champ Rubrique à une rubrique ?
<a name="comment_lier_champ_rubrique_une_rubrique_ELTTEXTE000153"></a>
Pour définir la rubrique liée au champ "Rubrique" : 

1. Affichez l'onglet "Général" de la fenêtre de description du champ (option "Description" du menu contextuel du champ).

2. **Pour lier le champ avec une rubrique appartenant à la source de données de l'état**, sélectionnez directement cette rubrique dans la combo "Rubrique liée".

3. **Pour lier le champ avec une rubrique n'appartenant pas à la source de données de l'état**, cliquez sur le bouton "Autres" et sélectionnez la rubrique liée au champ en cours.

## Notes
<a name="notes_ELTTEXTE000177"></a>


### Champ Rubrique RTF : Transformer un champ "Rubrique" en champ "RTF"
<a name="champ_rubrique_rtf_transformer_champ_rubrique_champ_rtf_ELTPARAGRAPHE000049"></a>Pour afficher du texte au format RTF dans un champ "Rubrique" : 

1. Affichez la fenêtre de description du champ (option "Description" du menu contextuel du champ). 

2. Dans l'onglet "Détail", cochez l'option "Format RTF".

3. Validez la fenêtre de description. 




Le champ "Rubrique" se transforme alors en champ "RTF". Toutes les caractéristiques du champ "Rubrique" sont conservées.


<a name="NOTE3_2"></a>


### Champ "Rubrique" spécifique : "Monétaire + Euro"
<a name="champ_rubrique_specifique_monetaire_euro_ELTPARAGRAPHE000063"></a>Si le champ "Rubrique" est de type "Monétaire + Euro", il est possible de gérer :

- la **devise mémorisée** : devise des données présentes dans la source de données de l'état. Si le champ est manipulé par programmation, la devise correspondante sera la devise mémorisée.

- la **devise affichée** : devise utilisée pour afficher/imprimer la valeur dans l'état. Cette devise peut être différente de la devise mémorisée. La conversion est automatiquement réalisée.

- le **double-affichage** : il est possible de relier deux champs de type "Monétaire + Euro" entre eux. La valeur affichée dans le champ de base est automatiquement convertie dans le champ suiveur.




Ce type de champ peut également être utilisé dans une fenêtre. Pour plus de détails sur ce type de champ, consultez [Le type "Monétaire + Euro"](../WDLang1/3033007.md).




### Champ Rubrique et chaîne de caractères de type UNICODE
<a name="champ_rubrique_chaine_caracteres_type_unicode_ELTPARAGRAPHE000078"></a>Les rubriques HFSQL de type "Chaîne UNICODE" sont affichées automatiquement dans les champs Rubrique des états. Aucune opération n'est nécessaire pour réaliser l'affichage.

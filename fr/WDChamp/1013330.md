


## Paramétrage des titres des colonnes (champ Table)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000367"></a>
Le titre des colonnes est un élément important d'un champ Table. En effet, le titre des colonnes permet d'intervenir aussi bien au niveau du look (avec la possibilité d'intégrer des images) qu'au niveau des fonctionnalités (filtre, recherche).

Cette page d'aide présente les différentes fonctionnalités disponibles : 

- [Titre des colonnes](#NOTE2_1).

- [Icônes affichées dans les colonnes](#NOTE3_1).

- [Bandeau de sélection](#NOTE4_1).






<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Titre des colonnes
<a name="titre_des_colonnes_ELTTEXTE000391"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 

### Image du titre des colonnes
<a name="image_titre_des_colonnes_ELTPARAGRAPHE000043"></a>Il est possible d'afficher une image dans le titre d'une colonne. Si un libellé est affiché dans le titre de la colonne, cette image sera positionnée à gauche du libellé. Cette image peut être insérée sous l'éditeur ou par programmation.

Pour insérer une image dans le titre d'une colonne sous l'éditeur : 

1. Affichez la fenêtre de description de la colonne (option "Description de la colonne" du menu contextuel).

2. Dans l'onglet "Général", indiquez l'image du titre de la colonne en cliquant le bouton ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Menu_Image_Editeur%20-%20HC%20N%B0001.gif) : 

	- L'option "Parcourir" permet de sélectionner un fichier image disponible depuis votre poste. 

	- L'option "Catalogue" permet de choisir une image présente dans le catalogue d'images de WINDEV.




3. Validez. L'image apparaît dans le titre de la colonne sous l'éditeur.




Pour afficher une image dans un titre de colonne par programmation, utilisez la propriété [ImageTitre](../Proprietes/1000018306.md).
<a name="NOTE2_2"></a>


### Titre de colonne multiligne
<a name="titre_colonne_multiligne_ELTPARAGRAPHE000066"></a>Par défaut, le titre de colonne est affiché sur plusieurs lignes. 

Pour paramétrer le mode de gestion du titre des colonnes (mono-ligne ou multi-ligne)  : 

1. Affichez la fenêtre de description du champ Table (option "Description de la table" du menu contextuel).

2. Dans l'onglet "Détail", cochez ou décochez l'option "Forcer le titre en monoligne (8)".

3. Validez.




Remarques :

- Le titre des colonnes supporte également le RTF et les fonctions graphiques (fonctions [gXXX](../WDLang1/3029028.md)).

- Dans l'éditeur de fenêtres, la hauteur des titres de colonnes peut être modifiée à la souris.

- Le titre des colonnes peut être modifié par programmation grâce à la propriété [Titre](../Proprietes/2510090.md) ou à la propriété [Libellé](../Proprietes/2510053.md).  

- La hauteur des titres des colonnes peut être modifié par programmation grâce à la propriété [HauteurTitre](../Proprietes/1000021058.md). 



<a name="NOTE2_3"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 

### Inclinaison du titre des colonnes
<a name="inclinaison_titre_des_colonnes_ELTPARAGRAPHE000110"></a>Dans un champ Table, il est possible de fixer un angle d'écriture des titres (en degrés). Il est ainsi possible de créer des titres de colonne inclinés. 

Pour incliner le titre des colonnes : 

1. Affichez la fenêtre de description du champ Table (option "Description" du menu contextuel).

2. Dans l'onglet "Détail", saisissez la valeur de l'option "Angle du titre des colonnes" (en degrés). Le bouton fléché à côté du champ permet de sélectionner de manière précise l'angle voulu. <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Angle_titre_col.gif)


3. Validez.




Pour incliner le titre des colonnes par programmation, utilisez la propriété [AngleTitre](../Proprietes/1000019371.md).  
<a name="NOTE2_4"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 

### Image de fond de la zone de titre de la colonne
<a name="image_fond_zone_titre_colonne_ELTPARAGRAPHE000135"></a>Il est possible d'inclure une image dans chaque cellule de titre.<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=TitreColonneTable.GIF)
Pour personnaliser l'image de la zone de titre :

1. Affichez l'onglet "Style" de la fenêtre de description du champ Table (option "Description" du menu contextuel).

2. Sélectionnez l'élément "Cadre titre de colonne".

3. Ouvrez la combo "Cadre" et sélectionnez l'option "Image (Mode 9 images)".

4. Cliquez sur le bouton "Flèche droite" situé à droite de la combo "Cadre". Une fenêtre de paramétrage de l'image du titre des colonnes s'ouvre.

5. Sélectionnez l'image du cadre à utiliser en cliquant le bouton ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Menu_Image_Editeur%20-%20HC%20N%B0001.gif) : 

	- L'option "Parcourir" permet de sélectionner un fichier image disponible depuis votre poste. 

	- L'option "Catalogue" permet de choisir une image présente dans le catalogue d'images de WINDEV. Le catalogue d'images propose uniquement les images pouvant être utilisées comme titre de colonne.




6. Si l'image sélectionnée contient plusieurs états, spécifiez ce nombre d'états. 

7. Fermez la fenêtre de paramétrage de l'image des cellules de titre.

8. Validez.




**Remarque** : L'élément "Cadre titre de colonne" de l'onglet "Style" permet également de paramétrer la hauteur de la zone de titre des colonnes.

### Sur-entête de colonnes
<a name="surentete_colonnes_ELTPARAGRAPHE000164"></a>Il est possible de définir un sur-entête de colonnes. Un sur-entête de colonnes est un titre supplémentaire affiché au-dessus du titre d'une ou plusieurs colonnes.<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=TableAjouteSurEntete.gif)
Pour définir un sur-entête de colonnes :

1. Affichez l'onglet "Détail" de la fenêtre de description du champ Table (option "Description" du menu contextuel).

2. Cliquez sur le bouton "Éditer les sur-entêtes de colonnes".

3. Dans la fenêtre qui s'ouvre, les différentes colonnes de la table sont affichées.<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=TableAjouteSurEntete_Edit.gif)
Dans cette fenêtre, il est possible de : 

	- 1 : Ajouter des sur-entêtes (bouton "+") ou supprimer des sur-entêtes (bouton "-"). 

	- 2 : Redimensionner et fusionner les différentes cellules des sur-entêtes en survolant les séparateurs. 

	- 3 : Renommer les cellules des sur-entêtes en cliquant dessus. 




4. Validez.




Remarques : 

- Il est possible de créer des sur-entêtes de colonnes par programmation grâce à la fonction [TableAjouteSurEntête](../WDLang1/1000020585.md). 

- La propriété [Fusion](../Proprietes/1000017095.md) permet de fusionner : 

	- les cellules d'un champ Table. 

	- les titres des colonnes. 




- Si les sur-entêtes de colonnes sont activés, la modification du libellé des colonnes via la fenêtre de description de la colonne n'est pas prise en compte : le libellé doit être modifié dans la fenêtre d'édition des sur-entêtes de colonnes.  

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) La propriété [Surentête](../Proprietes/1000022744.md) permet de :

	- Connaître et modifier le titre et l'aspect d'un sur-entête de colonne de table.

	- Connaître le nombre de sur-entêtes d'une colonne de table.

## Icônes affichées dans les colonnes
<a name="icones_affichees_dans_les_colonnes_ELTTEXTE000439"></a>


### Affichage de l'icône de tri
<a name="affichage_icone_tri_ELTPARAGRAPHE000215"></a>Par défaut, dans une colonne triable, l'icône de tri est affichée si l'utilisateur clique dans la colonne. L'icône de tri indique alors le sens de tri de la colonne.

Il est également possible d'afficher tout le temps une icône de tri dans la colonne. Dans ce cas, lorsque la colonne n'est pas triée, une icône indique à l'utilisateur qu'un tri peut être effectué sur cette colonne.

Pour afficher l'icône de tri : 

1. Affichez l'onglet "Style" de la fenêtre de description du champ Table (option "Description" du menu contextuel).

2. Sélectionnez l'élément "Cadre titre de colonne".

3. Sélectionnez l'option "Afficher l'icône de tri".



<a name="NOTE3_2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 

### Personnalisation des icônes "Loupe", tri, menu contextuel et filtre
<a name="personnalisation_des_icones_loupe_tri_menu_contextuel_filtre_ELTPARAGRAPHE000232"></a>Dans une colonne, plusieurs icônes permettent d'indiquer à l'utilisateur si la colonne est triable, si la colonne est triée et dans quel sens, si la colonne permet une recherche ou un filtre.

Pour personnaliser le look de ces éléments :

1. Affichez l'onglet "Style" de la fenêtre de description du champ Table (option "Description" du menu contextuel).

2. Sélectionnez l'élément "Cadre titre de colonne".

3. Cliquez sur le bouton "Image Tri/Loupe" et sélectionnez l'image à utiliser pour la loupe via le bouton ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Menu_Image_Editeur%20-%20HC%20N%B0001.gif) (option "Parcourir" ou "Catalogue").
	Cette image doit avoir pour taille 53 x 33 pixels et doit contenir les éléments suivants :

	- Image de colonne triable : 9 x 8 pixels

	- Images de colonne triée : triée vers le bas et vers le haut (22 x 9 pixels chaque dessin)

	- Images de la loupe : grisée, actif et enfoncée (33 x 11 pixels chaque dessin)

	- Image du menu contextuel (10 x 14 pixels)

	- Image du filtre : grisée, actif et enfoncée (33 x 11 pixels chaque dessin)







Par exemple :<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Loupe_perso.gif)

### Position des icônes de tri et de recherche
<a name="position_des_icones_tri_recherche_ELTPARAGRAPHE000260"></a>Par défaut, les différentes icônes affichées dans le titre de la colonne ont une position précise :

- les icônes permettent d'indiquer à l'utilisateur si la colonne est triable, si la colonne est triée et dans quel sens sont affichés à gauche.

- les icônes permettant d'indiquer si la colonne permet une recherche ou un filtre sont affichées à droite.




Pour personnaliser la position de ces éléments :

1. Affichez la fenêtre de description de la colonne (option "Description" du menu contextuel).

2. Dans l'onglet "Style", sélectionnez l'élément "Cadre titre de colonne".

3. Cliquez sur le bouton "Position Image Tri/Loupe" et indiquez la position des différentes icônes par rapport au cadre du titre de la colonne.



<a name="NOTE3_4"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 

### Mode de recherche par défaut
<a name="mode_recherche_par_defaut_ELTPARAGRAPHE000278"></a>Une colonne d'un champ Table peut permettre à l'utilisateur de réaliser une recherche ou un filtre. Il est possible de choisir le fonctionnement par défaut des colonnes d'un champ Table :

- **Mode loupe = Recherche** : Lorsque l'utilisateur saisira des caractères dans l'entête de colonne, une recherche sera automatiquement effectuée sur le contenu de la colonne. Les lignes dont le contenu commence par les caractères saisis seront affichées.

- **Mode loupe = Filtre &lt;Contient&gt;** : Lorsque l'utilisateur saisira des caractères dans l'entête de colonne, une recherche sera automatiquement effectuée sur le contenu de la colonne. Les lignes dont le contenu contient les caractères saisis seront affichées.




Pour paramétrer le mode de recherche par défaut des colonnes sous l'éditeur : 

1. Affichez la fenêtre de description du champ Table (option "Description" du menu contextuel).

2. Dans l'onglet "Général", indiquez le "Mode loupe" voulu.

3. Validez.




**Pour paramétrer le mode de recherche par défaut des colonnes par programmation**, utilisez la propriété [ModeLoupe](../Proprietes/1000018158.md). Cette propriété permet d'utiliser tous les modes disponibles.

## Bandeau de sélection
<a name="bandeau_selection_ELTTEXTE000481"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 

### Affichage ou non du bandeau de sélection sur la colonne
<a name="affichage_non_bandeau_selection_sur_colonne_ELTPARAGRAPHE000303"></a>Dans un champ Table, le bandeau de sélection permet de visualiser rapidement la ou les lignes sélectionnées. Dans certains cas, il est possible de vouloir afficher le bandeau de sélection uniquement sur certaines colonnes du champ Table.

Pour ne pas afficher le bandeau de sélection sur certaines colonnes sous l'éditeur : 

1. Affichez la fenêtre de description de la colonne (option "Description" du menu contextuel).

2. Dans l'onglet "Détail", décochez l'option "Afficher le bandeau sur cette colonne".

3. Validez.




**Pour ne pas afficher le bandeau de sélection sur certaines colonnes par programmation**, utilisez la propriété [BandeauVisible](../Proprietes/1000017411.md).

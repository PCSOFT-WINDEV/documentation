


## Modifier les caractéristiques des champs d'un état
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000505"></a>
Les différentes caractéristiques d'un ou de plusieurs champs peuvent être visualisées et/ou modifiées dans :

- la fenêtre de description de ce champ.

- le modifieur. 




L'éditeur d'états permet également directement de :

- modifier le libellé d'un champ.

- modifier le nom d'un champ.




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Fenêtre de description des champs d'un état
<a name="fenetre_description_des_champs_etat_ELTTEXTE000529"></a>
Pour afficher la fenêtre de description d'un champ, utilisez une des méthodes suivantes :

- double-cliquez sur le champ (conseillé).

- sélectionnez l'option "Description" du menu contextuel du champ (clic droit de la souris).

- sélectionnez le champ et sous le volet "Modification", cliquez sur l'icône de regroupement ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_regroup.gif)
 du groupe "Edition rapide" (combinaison de touches Alt + Entrée).

- sous le volet "Affichage", cliquez sur l'icône de regroupement ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_regroup.gif)
 du groupe "Options" (touche F12). Sélectionnez ensuite le champ et cliquez sur le bouton "Description".




**Remarque** : Il est possible d'afficher la fenêtre de description pour une sélection de champs. Seules les caractéristiques communes aux différents champs sélectionnés seront affichées.
<a name="NOTE2_2"></a>


### Caractéristiques par onglet
<a name="caracteristiques_par_onglet_ELTPARAGRAPHE000060"></a>La fenêtre de description regroupe les différentes caractéristiques du ou des champs sélectionnés par onglet :

| Onglet | Caractéristiques |
| --- | --- |
| **Général** | Permet de spécifier le nom du champ et toutes les caractéristiques d'affichage du champ. Par exemple :<br><br>- le libellé du champ (ce libellé peut être multilingue).<br><br>- le type d'information affiché dans le champ (texte, numérique, ...).<br><br>- le masque d'affichage utilisé pour le champ (ce masque peut être multilingue). <br><br>- l'image affichée dans un champ Image, le type de code-barres utilisé dans un champ Code-barres, le type de graphe utilisé dans un champ Graphe, etc. Pour plus de détails sur les caractéristiques d'un champ, consultez directement l'aide sur le type de ce champ.<br><br><br> |
| **UI** | Permet de définir :<br><br>- les différents paramètres d'impression du champ (visible, sécable, ...). <br><br>- la taille du champ.<br><br>- la position du champ dans l'état et dans le bloc auquel le champ appartient.<br><br>- l'ancrage du champ.<br><br>- le groupe de champs associé. <br><br>- le plan associé. <br><br><br> |
| **Liaison** | Permet de définir la rubrique ou la variable présente dans la source de données à laquelle le champ est lié. La valeur de cette rubrique ou de cette variable sera affichée dans le champ.<br><br>Pour lier le champ avec une rubrique ou une variable appartenant à la source de données de l'état, sélectionnez directement cette rubrique dans la liste  "Rubrique reliée".<br><br><br><br>Pour lier le champ avec une rubrique n'appartenant pas à la source de données de l'état, sélectionnez l'option "Multi-fichier" puis sélectionnez la rubrique liée au champ en cours. |
| **Note** | Permet de décrire le fonctionnement détaillé du champ.<br>Ces informations seront imprimées dans la documentation technique (icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_imprimer.gif)<br> sous le volet "Accueil", dans le groupe "Général").<br><br>Ces informations peuvent être remplies et connues par programmation grâce aux propriétés [Note](../Proprietes/2510085.md) et [TitreNote](../Proprietes/2510121.md). |
| **Style** | Permet de définir :<br><br>- l'aspect du champ (couleur de fond, cadre, ...).<br><br>- la police utilisée dans le champ.<br><br><br>**Remarque** : La police utilisée dans le champ peut également être définie grâce aux icônes du groupe "Edition rapide" dans le volet "Modification". |


Remarque : Toutes les options de la fenêtre de description de l'éditeur d'états proposent des bulles d'aide complètes, affichant par exemple, en plus de l'aide sur l'option, la propriété WLangage correspondante, la disponibilité par plateforme, les valeurs multilingues, ...

## Principaux paramètres d'impression des champs d'un état
<a name="principaux_parametres_impression_des_champs_etat_ELTTEXTE000559"></a>
Les paramètres d'impression d'un champ sont les suivants (définis dans l'onglet "UI" de la fenêtre de description) :

| Paramètre | Description | Disponible pour les champs suivants |
| --- | --- | --- |
| ***Visible*** | Le champ peut être affiché ou non. Cette caractéristique peut être modifiée en programmation grâce à la propriété [Visible](../Proprietes/2510138.md). | [Libellé](../WDChamp/1013179.md), [Rubrique](../WDChamp/1013059.md), [Champ calculé](../WDChamp/1013212.md), [Image](../WDChamp/1013218.md), [Code-Barres](../WDChamp/1013010.md), [Champ RTF](../WDChamp/1013056.md), [Champ prédéfini](../WDChamp/1013058.md), [Graphe](../WDChamp/1013006.md), [Forme](../WDChamp/1013102.md), [Champ Etat Interne](../WDChamp/1013044.md) |
| ***Auto-Extensible*** | La taille du champ définie sous l'éditeur d'états peut augmenter en fonction du contenu du champ. Les champs suivants le champ "auto-extensible" sont automatiquement déplacés. <br>**Remarques** :<br><br>- Si un champ n'est pas auto-extensible, son contenu peut être tronqué si la taille du champ est trop petite.<br>	Pour connaître par programmation la hauteur d'un champ auto-extensible lorsque ce champ sera agrandi, utilisez la propriété [HauteurFuture](../Proprietes/2511061.md).<br><br>- Si un champ d'un bloc est auto-extensible, le bloc associé sera automatiquement auto-extensible. Un bloc contenant des champs auto-extensible est auto-extensible. Un bloc contenant aucun champ auto-extensible n'est pas auto-extensible.<br><br>- Les champs présents dans un bas de page ne peuvent pas être auto-extensibles.<br><br><br>**Limite** : Dans un état en mode Etiquette, les champs ne peuvent pas être auto-extensibles. | [Libellé](../WDChamp/1013179.md), [Rubrique](../WDChamp/1013059.md), [Champ calculé](../WDChamp/1013212.md), [Forme](../WDChamp/1013102.md), [Code-Barres](../WDChamp/1013010.md), [Champ Etat Interne](../WDChamp/1013044.md) |
| ***Multi-ligne*** | Le contenu du champ peut être affiché sur plusieurs lignes. Cette caractéristique peut être modifiée en programmation grâce à la propriété [Multiligne](../Proprietes/2510105.md).<br>**Remarques** :<br><br>- La hauteur du champ doit être suffisante pour visualiser plusieurs lignes.<br><br>- Pour modifier par programmation la hauteur d'un champ, utilisez la propriété [Hauteur](../Proprietes/2510050.md).<br><br>- Si un angle d'orientation est appliqué à un champ multiligne, les retours chariot automatiques ne sont pas pris en compte. Seuls les retours à la ligne manuels sont pris en compte.<br><br><br> | [Libellé](../WDChamp/1013179.md), [Rubrique](../WDChamp/1013059.md), [Champ calculé](../WDChamp/1013212.md), [Champ prédéfini](../WDChamp/1013058.md) |
| ***Sécable*** | Le contenu d'un champ sécable peut être imprimé sur plusieurs pages lorsque l'espace restant pour imprimer la totalité du champ sur une page est insuffisant.<br>**Remarques** :<br><br>- Pour qu'un champ soit sécable, le bloc contenant le champ doit également être sécable.<br><br>- S'il n'y a pas assez de place sur la page en cours pour imprimer complètement un champ non-sécable, tout le champ sera totalement imprimé sur la page suivante.<br><br><br> | [Libellé](../WDChamp/1013179.md), [Rubrique](../WDChamp/1013059.md), [Champ calculé](../WDChamp/1013212.md), [Image](../WDChamp/1013218.md), [Code-Barres](../WDChamp/1013010.md), [Champ RTF](../WDChamp/1013056.md), [Champ prédéfini](../WDChamp/1013058.md), [Graphe](../WDChamp/1013006.md), [Forme](../WDChamp/1013102.md) |
| ***Mise à blanc si zéro*** | La valeur du champ est affichée uniquement si la valeur du champ est différente de 0.<br>Si la valeur du champ est égale à 0, le champ affiche une chaîne vide (""). | [Libellé](../WDChamp/1013179.md), [Rubrique](../WDChamp/1013059.md) |
| ***Ellipse (tronquer les libellés trop longs par '...')*** | L'ellipse consiste à indiquer qu'un libellé est trop long pour être affiché en entier. Les caractères suivants sont alors ajoutés au libellé : "...". Ces caractères peuvent être ajoutés en fin du libellé ou au milieu du texte (conseillé pour l'affichage de chemins de fichier). | [Rubrique](../WDChamp/1013059.md), [Champ calculé](../WDChamp/1013212.md) |
| ***Affiché lorsque la valeur change*** | La valeur du champ est affichée uniquement si la valeur de la rubrique liée change lors du passage à l'enregistrement suivant dans la source de données de l'état. | [Rubrique](../WDChamp/1013059.md), [Champ calculé](../WDChamp/1013212.md) |
| ***Remis à 0 après impression*** | La valeur du champ est automatiquement remise à 0 après son impression.<br>Ce paramètre est souvent utilisé pour les calculs sur rupture. | [Champ calculé](../WDChamp/1013212.md) |
| ***Format RTF*** | La valeur du champ pourra être au format RTF.<br>**Rappel** : Le format RTF permet de prendre en compte les attributs des caractères (gras, italique, souligné, ...) d'un texte. | [Libellé](../WDChamp/1013179.md), [Rubrique](../WDChamp/1013059.md), [Champ RTF](../WDChamp/1013056.md) |
| ***Utiliser les Infos Partagées en exécution*** | Le champ affichera les "infos partagées" de la rubrique liée au champ (et non les indices correspondant à ces "infos partagées").<br>Par exemple, les "Infos RAD" de la rubrique "Civilité" correspondent à la combo suivante : ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=UtiliserInfosRad.gif)<br>Dans la base de données, seuls les indices des "Infos partagées" sont conservés. | [Rubrique](../WDChamp/1013059.md) |
| ***Quadrillage horizontal / vertical*** | Le quadrillage horizontal et/ou vertical d'un graphe peut être affiché ou non. | [Graphe](../WDChamp/1013006.md) |

## Utilisation du modifieur
<a name="utilisation_modifieur_ELTTEXTE000583"></a>


### Présentation
<a name="presentation_ELTPARAGRAPHE000400"></a>Le modifieur permet de visualiser et/ou de modifier directement depuis l'éditeur d'états les principales caractéristiques :

- du ou des champs sélectionnés.

- de l'état ou du bloc sélectionné.




Pour plus de détails, consultez le [Modifieur](../Editeurs/2027006.md).
<a name="NOTE4_2"></a>


### Visualiser et/ou modifier les caractéristiques d'un ou de plusieurs champs grâce au modifieur
<a name="visualiser_etou_modifier_les_caracteristiques_plusieurs_champs_grace_modifieur_ELTPARAGRAPHE000415"></a>Pour visualiser et/ou modifier les caractéristiques d'un ou de plusieurs champs grâce au modifieur :

1. Sélectionnez le ou les champs à modifier :

	- soit directement dans l'état (voir [Sélectionner un ou plusieurs champs](../Editeurs/2026007.md)).

	- soit grâce à la liste déroulante du modifieur. Dans ce cas, seul un champ à la fois peut être sélectionné. Les poignées du champ sélectionné s'affichent automatiquement.
			Si plusieurs champs sont sélectionnés, seules les informations communes aux champs sélectionnés sont affichées dans le modifieur.




2. Effectuez les modifications nécessaires. Ces modifications sont réalisées sur tous les champs. Vous pouvez :

	- soit sélectionner la nouvelle valeur dans une liste déroulante.

	- soit sélectionner la nouvelle valeur dans un écran (modification d'une couleur par exemple).

	- soit modifier directement la valeur.

## Modifier le libellé d'un champ
<a name="modifier_libelle_champ_ELTTEXTE000613"></a>
Pour modifier le libellé d'un champ directement dans l'état en création :

1. Sélectionnez le champ à modifier.

2. Cliquez une nouvelle fois sur le champ ou appuyez sur le touche Entrée. Le curseur de la souris se transforme en curseur de saisie. ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=LibelleModifDirectetat.gif)


3. Saisissez le nouveau libellé du champ.

4. Validez la saisie du libellé en appuyant sur la touche Entrée.

## Modifier le nom d'un champ
<a name="modifier_nom_champ_ELTTEXTE000637"></a>
Il est possible de modifier le nom d'un champ directement depuis la barre de messages de l'éditeur d'états.

Pour modifier le nom d'un champ :

1. Sélectionnez le champ à modifier.

2. Saisissez directement dans la barre de messages le nouveau nom du champ. ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=NomChampModifEtat.gif)

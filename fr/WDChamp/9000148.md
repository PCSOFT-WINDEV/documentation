


## Fenêtres non maximisées sous iOS
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000134"></a>
Une fenêtre non maximisée peut occuper seulement une partie de l'écran. Le plus souvent, les fenêtres non maximisées sont utilisées pour afficher des messages d'information.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Création d'une fenêtre non maximisée
<a name="creation_une_fenetre_non_maximisee_ELTTEXTE000158"></a>
Pour créer une fenêtre non maximisée : 

1. Cliquez sur ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_nouveau.gif) parmi les boutons d'accès rapide. 

	- La fenêtre de création d'un nouvel élément s'affiche : cliquez sur "Fenêtre" puis sur "Fenêtre".

	- L'assistant de création d'une fenêtre se lance. 

	- Sélectionnez "Vierge pour iPad" ou "Vierge pour iPhone" dans l'assistant et validez.




2. La fenêtre est créée sous l'éditeur. 

3. Affichez la fenêtre de description (option "Description" du menu contextuel). 

4. Dans l'onglet "Style", sélectionnez "Fenêtre non maximisée". 
	Remarques : 

	- L'option "Fenêtre avec flèche" permet de créer une [fenêtre avec flèche](../Editeurs/9000146.md).

	- Ce type de fenêtre est disponible uniquement pour iPad.




5. Validez.

## Caractéristiques des fenêtres non maximisées
<a name="caracteristiques_des_fenetres_non_maximisees_ELTTEXTE000182"></a>


### Taille de la fenêtre
<a name="taille_fenetre_ELTPARAGRAPHE000047"></a>La taille de la fenêtre à l'ouverture peut être modifiée dans l'onglet "Détail" de la fenêtre de description. 

Par défaut, la fenêtre est centrée par rapport à l'écran. 

Remarque : Il est possible de sélectionner les hauteurs et largeurs minimales et maximales de la fenêtre dans l'onglet "Style" de la fenêtre de description (bouton "Taille min/max"). 
<a name="NOTE2_2"></a>


### Ouverture d'une fenêtre non maximisée
<a name="ouverture_une_fenetre_non_maximisee_ELTPARAGRAPHE000058"></a>L'ouverture d'une fenêtre non maximisée est réalisée avec la fonction [OuvreFille](../WDLang1/3038021.md) ou la fonction [OuvreFenêtreMobile](../WDLang1/1000021018.md).
 
Par exemple :

```wl
OuvreFenêtreMobile(FEN_Non_Max)
```

<a name="NOTE2_3"></a>


### Fenêtre détourée
<a name="fenetre_detouree_ELTPARAGRAPHE000217"></a>Une fenêtre non maximisée peut être détourée (fenêtre non rectangulaire). Il suffit pour cela d'utiliser une image au format PNG. 

Pour détourer une fenêtre non maximisée :

- Dans l'onglet "Style" de la description de la fenêtre, sélectionnez la couleur de fond "Transparent".

- Dans l'onglet "Image" de la description de la fenêtre, sélectionnez : 

	- une image de fond au format PNG, 

	- le mode de transparence "Selon l'image".
			Remarque : La transparence de l'image est gérée par le format PNG.






<a name="NOTE2_4"></a>


### Remarque
<a name="remarque_ELTPARAGRAPHE000082"></a>La barre de titre d'une fenêtre non maximisée est affichée uniquement pour les fenêtres de type [Fenêtre à flèche](../Editeurs/9000146.md).

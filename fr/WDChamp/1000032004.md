
## Manipuler un champ Zone de Texte Riche par programmation
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000130"></a>
Pour manipuler un champ Zone de Texte Riche par programmation, il suffit d'utiliser directement la variable du champ Zone de Texte Riche dans le code.

La variable du champ Zone de Texte Riche correspond au nom du champ. 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Initialiser un champ Zone de Texte Riche
<a name="initialiser_champ_zone_texte_riche_ELTTEXTE000154"></a>
Pour initialiser un champ Zone de Texte Riche, il est possible : 

- d'initialiser le champ directement avec le code HTML. Il est ainsi possible de mettre en forme le contenu du champ par programmation.
	Par exemple : 
	
	```wl
	// Modification de la Zone de Texte Riche
	ZTR_MODIFIABLE = "<B>Cette portion a été " + ...
	 	"modifi&#233;e</B>"
	```


- d'utiliser la propriété [Valeur](../Proprietes/2510130.md) ou [LibelléHTML](../Proprietes/1000020922.md). 



<a name="NOTE2_3"></a>


### Modifier un ou plusieurs éléments d'un champ Zone de Texte Riche
<a name="modifier_plusieurs_elements_champ_zone_texte_riche_ELTPARAGRAPHE000037"></a>Un champ Zone de Texte Riche peut contenir des liens ou des images. Pour ne modifier que les informations concernant un lien ou une image contenue dans le champ, il suffit de manipuler directement par programmation l'élément voulu. 

Par exemple, un champ Zone de Texte Riche contient un lien et une image. Le libellé et l'URL du lien varient selon si l'internaute est connecté ou non. De même, l'image affichée n'est pas la même. Dans ce cas, il suffit simplement de manipuler directement les champs Lien et Image. 


```wl
// Modification du lien et de l'image contenus dans le champ ZTR_Bienvenue
SI bConnecté = Vrai ALORS
	LIEN_Connexion.Libellé = "Se déconnecter"
	LIEN_Connexion.URL = "www.monsite.fr"
	IMG_Logo = "Logo_User.gif"
SINON
	LIEN_Connexion.Libellé = "Se connecter"
	LIEN_Connexion.URL = "www.monsite.fr\Login.htm"
	IMG_Logo = "Logo.gif"
FIN
```




<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Récupérer la valeur d'un champ Zone de Texte Riche
<a name="recuperer_valeur_champ_zone_texte_riche_ELTTEXTE000184"></a>
Pour récupérer le contenu d'un champ Zone de Texte Riche :

- réalisez une simple affectation. Par exemple :
	
	```wl
	ContenuActuel = ZTR_Présentation
	```


- manipulez directement le champ Zone de Texte Riche. Par exemple :
	
	```wl
	Trace(ZTR_Présentation)
	```


- utilisez la propriété [Valeur](../Proprietes/2510130.md) ou [LibelléHTML](../Proprietes/1000020922.md).




Remarque : Le texte récupéré correspond au code HTML du champ.

## Propriétés spécifiques à la gestion d'un champ Zone de Texte Riche
<a name="proprietes_specifiques_gestion_champ_zone_texte_riche_ELTTEXTE000208"></a>
Les propriétés suivantes sont spécifiques à la gestion des caractéristiques d'un champ Zone de Texte Riche par programmation : 



|   |   |
| --- | --- |
| [LibelléHTML](../Proprietes/1000020922.md) | La propriété **LibelléHTML** permet de manipuler le libellé d'un champ sous forme de code HTML. |





Pour connaître la liste complète des propriétés du WLangage pouvant être utilisées avec un champ de type Zone de Texte Riche, consultez [Propriétés associées à un champ Zone de Texte Riche](../WDChamp/1000032003.md).

## Présentation
<a name="presentation_ELTTEXTE000152"></a>
L'éditeur d'états offre la possibilité de créer des états de type "Étiquette". L'état est alors constitué uniquement d'un bloc *Corps* dans lequel une ou plusieurs étiquettes sont imprimées.

## Créer un état étiquette
<a name="creer_etat_etiquette_ELTTEXTE000176"></a>


### Création
<a name="creation_ELTPARAGRAPHE000036"></a>

1. Pour créer un état de type "Étiquette", cliquez sur ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_nouveau.gif)
 parmi les boutons d'accès rapide : 

	- La fenêtre de création d'un nouvel élément s'affiche : cliquez sur "Etat" puis sur "Etat".

	- L'assistant de création d'un état se lance. Dans l'assistant, sélectionnez "Etiquette".




2. Dans cet assistant, il est nécessaire de spécifier :

	- la source de données de l'état (fichier de données, requête, ...). Pour plus de détails, consultez [Source de données d'un état](../WDChamp/1011055.md).

	- les informations spécifiques à la création des étiquettes.






<a name="NOTE2_2"></a>


### Informations spécifiques à la création des étiquettes
<a name="informations_specifiques_creation_des_etiquettes_ELTPARAGRAPHE000055"></a>Les informations spécifiques à la création des étiquettes sont les suivantes :

- le nombre d'exemplaires identiques à imprimer pour chaque étiquette.

- le format des étiquettes et/ou de la planche d'étiquettes. Ce format peut correspondre :

	- soit à un **format personnalisé** :

		1. Sélectionnez l'élément "&lt;Personnalisé&gt;" dans la combo "Marque".

		2. Spécifiez le format de la planche d'étiquettes sur laquelle les étiquettes seront imprimées.

		3. Spécifiez le format des étiquettes.

		4. Spécifiez le nombre d'étiquettes à imprimer sur la largeur d'une page.




- soit à un **format prédéfini** :

	1. Sélectionnez la marque des étiquettes utilisées dans la combo "Marque".

	2. Sélectionnez le modèle des étiquettes utilisées dans la combo "Modèle".

	3. Les informations spécifiques aux étiquettes utilisées (libellé du modèle, taille des étiquettes et nombre d'étiquettes imprimées sur la largeur d'une planche d'étiquettes) s'affichent. 
						Remarque : Cochez l'option "Pouce" pour afficher la taille des étiquettes en pouces.



<a name="NOTE2_3"></a>


### Modifier les caractéristiques des étiquettes
<a name="modifier_les_caracteristiques_des_etiquettes_ELTPARAGRAPHE000079"></a>Lorsque l'état "Étiquette" est créé, il est possible à tout moment de modifier les caractéristiques des étiquettes : sous le volet "Modification", dans le groupe "Etat", cliquez sur "Format et étiquettes".

## Propriétés du WLangage associées aux états étiquette
<a name="proprietes_wlangage_associees_aux_etats_etiquette_ELTTEXTE000212"></a>
Les différentes propriétés des états "Étiquette" sont les suivantes. Ces propriétés permettent de paramétrer dynamiquement (par programmation) le format et le mode d'impression des étiquettes.

**Remarque** : Ces propriétés peuvent uniquement être utilisées sur des états de type "Étiquette".



|   |   |
| --- | --- |
| [EspaceH](../Proprietes/2511067.md) | La propriété **EspaceH** permet de connaître et de modifier l'espacement horizontal entre les étiquettes d'un état. Cet espace a été défini lors de la création de l'état au format étiquette. |
| [EspaceV](../Proprietes/2511065.md) | La propriété **EspaceV** permet de connaître et de modifier l'espacement vertical entre les étiquettes d'un état. Cet espace a été défini lors de la création de l'état au format étiquette. |
| [HauteurEtiquette](../Proprietes/2511069.md) | La propriété **HauteurEtiquette** permet de :<br><br>- Connaître la hauteur des étiquettes d'un état de type Étiquette.<br><br>- Modifier la hauteur des étiquettes d'un état de type Étiquette.<br><br><br> |
| [LargeurEtiquette](../Proprietes/2511055.md) | La propriété **LargeurEtiquette** permet de :<br><br>- Connaître la largeur des étiquettes d'un état de type "Étiquette".<br><br>- Modifier la largeur des étiquettes d'un état de type "Étiquette".<br><br><br> |
| [NbEtiquette](../Proprietes/2511050.md) | La propriété **NbEtiquette** permet de :<br><br>- Connaître le nombre d'étiquettes imprimées sur la largeur d'une page.<br><br>- Modifier le nombre d'étiquettes imprimées sur la largeur d'une page.<br><br><br> |
| [NbExemplaireEtiquette](../Proprietes/2511059.md) | La propriété **NbExemplaireEtiquette** permet de :<br><br>- Connaître le nombre d'exemplaires d'étiquettes identiques imprimés.<br><br>- Modifier le nombre d'exemplaires d'étiquettes identiques à imprimer.<br><br><br> |
| [NumEtiquetteDeDépart](../Proprietes/2511032.md) | La propriété **NumEtiquetteDeDépart** permet de :<br><br>- Connaître le numéro de l'étiquette sur la planche d'étiquettes à partir de laquelle l'impression est effectuée.<br><br>- Modifier le numéro de l'étiquette à partir de laquelle l'impression est effectuée sur la planche d'étiquettes.<br><br><br> |

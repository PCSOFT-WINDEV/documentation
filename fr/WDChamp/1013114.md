


## Manipuler un champ Boîte à outils par programmation
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000170"></a>
WINDEV et WINDEV Mobile permettent de manipuler un champ Boîte à outils directement par programmation. Il suffit d'utiliser directement la variable du champ Boîte à outils dans le code.

La variable du champ Boîte à outils correspond au numéro du volet actif. Cette variable est de type entier.

**Remarque** : WINDEV et WINDEV Mobile gèrent le passage automatiquement lorsque l'utilisateur clique sur un volet.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Manipuler un volet du champ Boîte à outils
<a name="manipuler_volet_champ_boite_outils_ELTTEXTE000194"></a>
Pour manipuler un volet, utilisez la syntaxe suivante :

```txt
<Champ Boîte à outils>[Numéro du volet].NomPropriété = ValeurPropriété
```
Par exemple :

```wl
BAO_MaBoîte[2].Etat = Grisé
```


<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Changer le volet actif
<a name="changer_volet_actif_ELTTEXTE000218"></a>
Par défaut, le volet 1 (le volet en haut) est activé. Il est possible de changer le volet en cours par programmation.

Pour activer un volet de boîte à outils :

- Affectez l'onglet avec le numéro du volet à activer :
	
	```txt
	<Champ Boîte à outils> = <Numéro du volet>
	```


- Utilisez la propriété [Valeur](../Proprietes/2510130.md).




Le numéro est attribué lors de la description des volets: dans la liste des volets, le premier a le numéro 1, le deuxième a le numéro 2...

Remarques : 

- Si le volet indiqué n'existe pas, le volet affiché n'est pas modifié.

- Sur une même boîte à outils, un seul volet peut être affiché à la fois.

- Lors du changement de volet :

	- le code de sortie du champ qui a le focus est exécuté.

	- le code d'entrée du premier champ en saisie dans le volet est activé.

## Connaître le volet actif
<a name="connaitre_volet_actif_ELTTEXTE000242"></a>
Pour récupérer le numéro du volet actif :

- réalisez une simple affectation. Par exemple :
	
	```txt
	<Numéro du volet actif> = <Champ Boîte à outils>
	```


- utilisez la propriété [Valeur](../Proprietes/2510130.md).

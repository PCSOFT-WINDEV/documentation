


## Texte riche, liens et images dans les champs WEBDEV
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000162"></a>
Un texte riche est un texte qui comporte des enrichissements : police, graisse, taille, couleur, attribut, ...

Il est possible d'utiliser du texte riche dans de nombreux champs :

- Saisie

- Combo

- Liste

- Interrupteur

- Sélecteur

- Lien




Le texte riche peut correspondre : 

- au libellé du champ. 

- au libellé des options (par exemple pour les champs Sélecteur et Interrupteur). 




Exemple :

## Comment le faire ?
<a name="comment_faire_ELTTEXTE000186"></a>


### Saisir des libellés en texte riche
<a name="saisir_des_libelles_texte_riche_ELTPARAGRAPHE000035"></a>Pour utiliser du texte riche dans le libellé d'un champ : 

1. Sélectionnez le champ présent dans la page. 

2. Affichez le menu contextuel du champ (clic droit) et sélectionnez l'option "Editer le libellé". 

3. Le libellé du champ passe alors en mode "Edition" : un cadre jaune apparaît autour du libellé sélectionné. 

4. Il est alors possible de personnaliser l'enrichissement grâce au volet "Texte" qui apparaît dans le ruban : 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Texte_Riche_2.gif&type=thumb)
Il est possible par exemple de : 

	- Personnaliser la mise en forme du texte. 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Texte_Riche_1.gif)


	- Ajouter un lien. 

- Ajouter une image. 

5. Pour valider, cliquez ailleurs dans la page. 




Quelques exemples :

- Un champ Libellé contenant un lien : 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=WB-ZTR-Lien%20dans%20texte.jpg)


- Un champ de saisie dont le libellé contient un lien : 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=WB-ZTR-Saisie.jpg)


- Un champ Interrupteur dont le titre est en gras et dont le libellé d'option contient un lien : 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=WB-ZTR-Lien%20dans%20interrupteur.jpg)


- Un champ Sélecteur dont le titre est gras et dont un libellé d'option contient un lien : 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=WB-ZTR-S%E9lecteur.jpg)


- Une demande d'email, avec étoile rouge : 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=WB%20-%20Texte%20riche%20dans%20les%20champs.jpg)





**Remarques** : 

- Pour passer le libellé en édition, il est également possible : 

	- depuis l'éditeur, de cliquer deux fois sur le champ. 

	- depuis la fenêtre de description du champ, dans l'onglet "Général", de cliquer sur le lien "Editer avec mise en forme". 




- Pour éditer une option d'un champ Sélecteur ou Interrupteur : 

	- depuis l'éditeur, sélectionnez le champ et cliquez sur l'option voulue : l'option passe en édition. 

	- depuis la fenêtre de description du champ, dans l'onglet "Contenu", de sélectionner l'option voulue et de cliquer sur le lien "Editer avec mise en forme".

### Editer le code HTML des libellés
<a name="editer_code_html_des_libelles_ELTPARAGRAPHE000087"></a>Il est possible d'éditer le code source HTML des libellés depuis WINDEV. Il est ainsi possible de modifier le code HTML généré, d'ajouter son propre code HTML par exemple pour proposer un effet spécifique, ...

Pour éditer le code HTML d'un libellé : 

1. Sélectionnez le champ présent dans la page. 

2. Affichez la fenêtre de description du champ (option "Description" du menu contextuel). 

3. Dans l'onglet "Général", cliquez sur "Editer le code HTML". 

4. L'éditeur de code HTML apparaît. Vous pouvez modifier le code HTML proposé. 




Remarques : 

- Dans la mesure du possible, les modifications effectuées sous l'éditeur de code HTML sont visibles dans l'éditeur de pages en WYSIWYG. Les codes HTML avancés (type animation) ne sont pas visibles. 

- L'éditeur de code HTML permet de saisir du code HTML contenant des fonctions Javascript. Pour plus de détails sur les possibilités de l'éditeur de code HTML, consultez Editeur HTML.

## Présentation
<a name="presentation_ELTTEXTE000177"></a>
Pour intégrer des graphes dans vos applications ou sites, WINDEV, WEBDEV et WINDEV Mobile mettent à votre disposition : 

- un [champ Graphe](../WDChamp/1013006.md) (utilisable dans l'éditeur de fenêtres, de pages et d'états). 

- des [fonctions WLangage](../WDLang3/3042003.md) qui permettent de créer et de manipuler des graphes.




Les graphes permettent une visualisation rapide et concrète des données numériques.

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **Utilisation des graphes en WEBDEV Linux et en WINDEV Linux**
Pour utiliser les graphes sous Linux, il est nécessaire d'installer la librairie QT (Core et GUI), version 5 minimale. 

**Sous Ubuntu**, cette librairie peut être installée avec les commandes suivantes : 

```txt
sudo apt-get install libqtcore5
sudo apt-get install libqtgui5
```

## Comment créer des graphes ?
<a name="comment_creer_des_graphes_ELTTEXTE000336"></a>


### Créer des graphes dans les éditeurs
<a name="creer_des_graphes_dans_les_editeurs_ELTPARAGRAPHE000083"></a>Les éditeurs de fenêtres, de pages et d'états disposent d'un champ Graphe. Lors de la création de ce champ, les assistants vous posent différentes questions afin de décrire le graphe qui va être créé. La fenêtre de description du champ permet de modifier à tout moment ces options.

Les fonctions WLangage permettent également de modifier les graphes affichés dans les champs Graphe. Il suffit d'indiquer le nom du champ Graphe dans les différentes fonctions.



Pour plus de détails sur ce type de champ, consultez [Le champ Graphe](../WDChamp/1013006.md).
<a name="NOTE3_2"></a>


### Créer des graphes par programmation
<a name="creer_des_graphes_par_programmation_ELTPARAGRAPHE000101"></a>Pour manipuler des graphes, WINDEV, WEBDEV et WINDEV Mobile disposent d'une cinquantaine de fonctions. Ces fonctions permettent de créer les graphes, de les modifier, de changer des options d'affichage ...

Pour plus de détails, consultez : 

- [Programmation des graphes](../WDLang3/1000021344.md). 

- [Fonctions de gestion des graphes](../WDLang3/3042003.md).




## Etat multicolonne
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000137"></a>
L'éditeur d'états offre la possibilité de créer des états de type "Multicolonne". L'état est alors constitué d'un bloc *Corps* dans lequel une ou plusieurs colonnes sont imprimées.

## Créer un état multicolonne
<a name="creer_etat_multicolonne_ELTTEXTE000161"></a>


### Création d'un nouvel état
<a name="creation_nouvel_etat_ELTPARAGRAPHE000043"></a>Pour créer un état de type "Multicolonnes" : 

1. Cliquez sur ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_nouveau.gif) parmi les boutons d'accès rapide.

	- La fenêtre de création d'un nouvel élément s'affiche : cliquez sur "Etat" puis sur "Etat".

	- L'assistant de création d'un état se lance.




2. Sélectionnez "Etat multicolonne". 

3. Dans cet assistant, il est nécessaire de spécifier :

	- Le nombre de colonnes

	- Le type de l'état (Fiche ou Tableau)

	- La source de données de l'état (fichier de données, requête, ...). Pour plus de détails, consultez [Source de données d'un état](../WDChamp/1011055.md).

	- Les informations spécifiques aux tris et aux ruptures.

	- Les rubriques à imprimer.

	- Les calculs automatiques sur les rubriques (Somme, Moyenne, Compteur).

	- Le format du papier et l'orientation (portrait, paysage).

	- Le gabarit à appliquer.

	- Le titre et le nom de l'état.








### Modification d'un état existant
<a name="modification_etat_existant_ELTPARAGRAPHE000070"></a>Il est possible de transformer un état existant en état multicolonne. Pour cela :

1. Ouvrez la fenêtre de description de l'état.

2. Sélectionnez l'onglet "Format".

3. Dans la zone "Multicolonne", sélectionnez le nombre de colonnes de l'état.




**Remarque** : Il sera nécessaire de repositionner les champs de l'état pour les adapter à la nouvelle taille de l'état.



## Propriétés du WLangage associées aux états multicolonnes
<a name="proprietes_wlangage_associees_aux_etats_multicolonnes_ELTTEXTE000191"></a>
Pour savoir par programmation si un état est un état multicolonne, la propriété [NbColonne](../Proprietes/1000018810.md) peut être utilisée. 

Pour connaître la liste des propriétés disponibles sur un état, consultez [Propriétés associées à un état](../WDChamp/1011029.md).

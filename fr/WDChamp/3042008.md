


## Les graphes de type "Nuage de points"
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000167"></a>
Un nuage de points (XY) représente la relation entre deux valeurs numériques dans deux séries de données. Les séries impaires permettent d'identifier les abscisses des points, et les séries paires, les ordonnées des mêmes points. Les nuages de points mettent en évidence des intervalles irréguliers, ils permettent aussi de grouper les données.

Les graphes "nuage de points" sont couramment utilisés pour représenter des données scientifiques.

## Caractéristiques d'un nuage de points
<a name="caracteristiques_nuage_points_ELTTEXTE000191"></a>

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Gennuage.gif&type=thumb)
Dans ce graphe :

**1. Titre de l'axe vertical**. 
Ce paramètre est affiché si le titre de l'axe a été défini dans l'onglet "Général" de la description du champ Graphe. Ce paramètre peut être défini également par programmation grâce à la fonction [grTitreAxe](../WDLang3/3042057.md). Il est également possible de définir un titre pour l'axe horizontal.

**2. Valeur X du point**. 
Cette valeur est une des données de la première série.

**3. Valeur Y du point**. 
Cette valeur est une des données de la seconde série.

**4. Titre du graphe**. 
Les caractéristiques du titre du graphe sont saisies dans l'onglet "Détail" de la description du champ (bouton "Editer le titre du graphe").
Il est possible de définir le titre du graphe, sa position et son style.

**5. Légende**. 
Les caractéristiques de la légende sont saisies dans l'onglet "Détail" de la description du champ (bouton "Editer la légende du graphe").
Il est possible de définir sa position et son style.

**6. Etiquettes de séries**. 
Le libellé de la légende correspond au libellé de la première série du graphe (définie dans l'onglet "Séries").

**7. Couleur de série**. 
Pour chaque série, il est possible de définir une couleur dans l'onglet "Séries" de la description du champ. La couleur prise en compte sera celle définie pour la première série. Par défaut la couleur utilisée est automatique.

**8. Graduation**. 
Par défaut, la graduation est calculée automatiquement. L'onglet "Général" de la description du champ permet de :

- définir une graduation spécifique.

- définir une échelle des abscisses en fonction du temps. Cette option fonctionne uniquement avec des données de type Date ou DateHeure provenant d'un champ Table ou d'une rubrique HFSQL. Cette option ne peut pas être utilisée avec une série remplie par programmation.

- définir une échelle des ordonnées logarithmique.

- indiquer l'orientation des graduations.




**9. Graduations et titre de l'axe horizontal**.

**10. Ascenseurs permettant de zoomer**. 
Il est possible d'autoriser le zoom dans un graphe dans l'onglet "Détail" de la description du champ. Cette option est disponible uniquement en WINDEV.
**Remarque** : l'utilisateur peut également activer le zoom avec le [menu contextuel du graphe](../WDChamp/1013007.md).

**Autres fonctionnalités**

- Les points d'un nuage de points peuvent être reliés ou déliés (fonction [grNuageReliePoint](../WDLang3/3042040.md)). 
	Cette fonction permet également de relier ou délier les points d'une seule série.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Les graphes de type "Nuages de points" peuvent être affichés en temps réel. Pour plus de détails, consultez [Graphe avec affichage en temps réel](../WDChamp/1000021037.md).

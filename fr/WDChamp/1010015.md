
## Gestion des options de menu dans les fenêtres MDI
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000165"></a>
Les fenêtres MDI doivent respecter certaines contraintes spécifiques aux applications MDI. Trois types de fenêtres sont disponibles : Fenêtre mère MDI, fille MDI ou soeur MDI.

La fenêtre ouvrant une autre fenêtre est appelée **fenêtre mère**; les fenêtres ouvertes par la fenêtre mère sont appelées **fenêtres filles**. Toutes les fenêtres filles ouvertes par la même fenêtre mère sont des **fenêtres soeurs**.

Pour plus de détails, consultez [Les différents types de fenêtres](../WDChamp/1010024.md).

Les manipulations des options de menu sont identiques que le menu appartienne à une fenêtre MDI ou non. Cependant, des fonctionnalités supplémentaires sont disponibles dans les fenêtres MDI :

- utilisation des [groupes d'options de menu](../WDChamp/1010015.md) pour afficher certaines options de menu dans la fenêtre mère MDI.

- affichage de la [liste des différentes fenêtres fille MDI ouvertes](../WDChamp/1010015.md) dans le menu principal de la fenêtre mère MDI.




Remarque : Pour plus de détails sur les menus et sur les options de menu, consultez :

- [Gestion des menus contextuels dans une application WINDEV](../WDChamp/1010011.md).

- [Gestion des menus principaux des fenêtres](../WDChamp/1010010.md).

- [Manipuler les options de menu sous l'éditeur](../WDChamp/1010008.md).

- [Manipuler les options de menu par programmation](../WDChamp/1010007.md).

## Groupes d'options de menu
<a name="groupes_options_menu_ELTTEXTE000189"></a>


### Présentation
<a name="presentation_ELTPARAGRAPHE000052"></a>WINDEV permet d'afficher certaines options de menu dans la fenêtre mère MDI uniquement lorsqu'une fenêtre fille MDI est ouverte. Il suffit de :

- associer les options de menu à afficher à un groupe.

- associer la fenêtre fille MDI à ce même groupe.




Exemple : Les groupes d'options de menu permettent de gérer les fonctionnalités proposées aux utilisateurs selon leur niveau d'accès.

Remarques :

- Une même option de menu peut être affichée à l'ouverture de plusieurs fenêtres filles. Il suffit d'associer cette option à plusieurs groupes.

- Pour qu'une option soit affichée quelle que soit la fenêtre fille ouverte, aucun groupe ne doit être associé à cette option.





<a name="NOTE2_2"></a>


### Associer une option de menu à un groupe
<a name="associer_une_option_menu_groupe_ELTPARAGRAPHE000070"></a>Pour associer une option de menu à un groupe : 

1. Affichez l'onglet "UI" de la description de l'option (option "Description de l'option" du menu contextuel de l'option).

2. Décochez l'option "Visible". L'option sera automatiquement visible lors de l'ouverture de la fenêtre fille MDI. En effet, la visibilité de l'option est automatiquement gérée par WINDEV.

3. Cliquez sur le bouton "Groupe". La liste des groupes existant s'affiche.

4. Créez si nécessaire un nouveau groupe.

5. Associez l'option au groupe en cochant la case correspondante. <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=GroupeOption.gif)






<a name="NOTE2_3"></a>


### Associer une fenêtre fille MDI à un groupe
<a name="associer_une_fenetre_fille_mdi_groupe_ELTPARAGRAPHE000085"></a>Pour associer une fenêtre fille MDI à un groupe : 

1. Ouvrez la fenêtre fille MDI.

2. Affichez l'onglet "Détail" de la description de cette fenêtre (option "Description" du menu contextuel).

3. Saisissez le nom du groupe à associer dans l'option "Groupe MDI".<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=OptionGroupeMDI.gif)

## Connaître les fenêtres fille MDI ouvertes
<a name="connaitre_les_fenetres_fille_mdi_ouvertes_ELTTEXTE000225"></a>


### Présentation
<a name="presentation_ELTPARAGRAPHE000099"></a>WINDEV permet d'afficher le titre des différentes fenêtres fille MDI ouvertes dans le menu principal de la fenêtre mère MDI. Le titre de la fenêtre en cours est précédé d'une coche. Pour afficher une des fenêtres ouvertes, il suffit de sélectionner son titre.

Par exemple : <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=FenFilleMDIOuvertepsd.gif)



<a name="NOTE3_2"></a>


### Afficher la liste des fenêtres MDI ouvertes
<a name="afficher_liste_des_fenetres_mdi_ouvertes_ELTPARAGRAPHE000110"></a>Pour afficher la liste des fenêtres MDI ouvertes : 

1. Sélectionnez une des options principales du menu. La liste des fenêtres fille MDI ouvertes apparaîtra après la dernière sous-option de cette option.

2. Affichez l'onglet "UI" de la description de cette option (option "Description" du menu contextuel).

3. Cochez l'option "Menu MDI".<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=OptionMenuMDI.gif)

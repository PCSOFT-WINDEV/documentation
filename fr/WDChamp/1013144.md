


## Manipuler un champ de saisie par programmation
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000297"></a>
Il est possible de manipuler un champ de saisie directement par programmation. Il suffit d'utiliser directement la variable du champ de saisie dans le code.

La variable du champ de saisie correspond au nom du champ de saisie. Le type de la variable dépend du type du champ de saisie (numérique, texte, monétaire, ...).
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Cas particulier : [Manipuler un champ de saisie de type RTF par programmation](../WDChamp/1013190.md)

## Afficher une information dans un champ de saisie
<a name="afficher_une_information_dans_champ_saisie_ELTTEXTE000321"></a>
Pour afficher une information dans un champ de saisie, une simple affectation suffit.

Quelques exemples :

- affectation directe :
	
	```wl
	SAI_NomClient = "Dupont"
	```


- affectation par variable :
	
	```wl
	SAI_NOM = Nom_Client
	```


- affectation par calcul :
	
	```wl
	SAI_NOM = SansEspace(CLIENT.NOM) + TAB + SansEspace(CLIENT.PRENOM)
	```



**Remarque** : Si le champ est associé à une rubrique d'un fichier de données, l'affectation est automatiquement réalisée en utilisant : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) la fonction [FichierVersEcran](../WDLang4/3044210.md).


Il est aussi possible d'utiliser la notation NomClient = CLIENT.NOM. Pour plus de détails, consultez [Lier un champ de saisie à une rubrique](../WDChamp/1013159.md).

## Récupérer la valeur saisie dans un champ de saisie
<a name="recuperer_valeur_saisie_dans_champ_saisie_ELTTEXTE000345"></a>
Pour récupérer le contenu d'un champ de saisie :

- réalisez une simple affectation. 
	Par exemple :
	
	```wl
	ContenuActuelSaisie = SAI_SAISIE1
	```


- récupérez une partie de l'information. 
	Par exemple :
	
	```wl
	Initiale = Gauche(SAI_Nom, 1) 
	Initiale = SAI_Nom[[1]]
	```


- utilisez la propriété [Valeur](../Proprietes/2510130.md)




**Remarque** : Si le champ est associé à une rubrique d'un fichier de données, la récupération de la valeur saisie est automatiquement réalisée en utilisant : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) la fonction [EcranVersFichier](../WDLang4/3044146.md).


Il est aussi possible d'utiliser la notation CLIENT.NOM = Saisie1. Pour plus de détails, consultez [Lier un champ de saisie à une rubrique](../WDChamp/1013159.md).

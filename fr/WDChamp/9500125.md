
## FAA : Tri/Recherche/Filtre sur les colonnes des champs Table
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000096"></a>
Dans les champs Table (champ Table fichier et par programmation), les colonnes avec recherche (loupe affichée dans la colonne) affichent un menu contextuel qui permet de spécifier les caractéristiques de tri, de filtre ou de recherche à effectuer sur la colonne. 

Il suffit de faire un clic droit sur la loupe pour voir le menu contextuel de la colonne s'afficher : <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Table_menuTriColonne.gif)


Ce menu contextuel propose :

- d'effectuer une recherche (option "Rechercher"): 
	Il suffit d'indiquer les premières lettres correspondant à l'élément recherché dans la colonne. Le champ Table affiche alors tous les éléments correspondant aux premières lettres saisies. 

- de poser un filtre (option "Filtrer") : 
	Il suffit de sélectionner une condition et de saisir la valeur voulue. Le contenu du champ Table est alors filtré selon la condition spécifiée.
	Remarque : La condition "Est vide" permet de filtrer la colonne sur les éléments vides. 

- la suppression d'un filtre précédemment posé sur la colonne (option "Supprimer le filtre"). 

- la mémorisation des filtres (option "Mémoriser les filtres").
	Si cette option est activée, les filtres posés à la fermeture de la fenêtre seront automatiquement exécutés lors de la prochaine ouverture de la fenêtre. Un message furtif sera affiché pour prévenir l'utilisateur et lui indiquer le(s) filtre(s) mis en place. 

- la mémorisation des tris (option "Mémoriser les tris").
	Si cette option est activée, les tris effectués à la fermeture de la fenêtre seront automatiquement exécutés lors de la prochaine ouverture de la fenêtre. 

- de poser un filtre avec une des valeurs contenues dans la colonne.




**Remarques** : 

- Le menu contextuel affiché lors d'un clic droit sur le titre de la colonne et celui affiché lors d'un clic droit sur la loupe est différent. 

- Ces options sont également disponibles pour les champs Table hiérarchique. 

- Pour plus de détails sur les autres options de ce menu contextuel, consultez [Menu contextuel des colonnes](../WDChamp/9500125.md).

## Présentation du Champ Jauge
<a name="presentation_champ_jauge_ELTTEXTE000255"></a>


### Champ d'une fenêtre ou d'une page dynamique
<a name="champ_une_fenetre_une_page_dynamique_ELTPARAGRAPHE000011"></a>

Les différents types de jauges proposés sont :

- Les jauges linéaires simples (horizontales ou verticales).
	

- Les jauges linéaires symétriques (horizontales ou verticales) : la jauge est séparée en deux parties (négative et positive) permettant d'afficher la progression.<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=jauge_exemple_lineaire.gif)
![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Ce type de jauge n'est pas disponible.

- Les jauges circulaires à aiguilles : une aiguille se déplace pour indiquer la valeur de la jauge.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Les jauges circulaires "LCD" : une "barre LCD" se remplit pour indiquer la valeur de la jauge.<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Jauge_exemple_circulaire.gif)

## Créer un champ Jauge
<a name="creer_champ_jauge_ELTTEXTE000279"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) 

### Créer un champ de type Jauge dans une fenêtre
<a name="creer_champ_type_jauge_dans_une_fenetre_ELTPARAGRAPHE000070"></a>Pour créer un champ de type Jauge : 

1. Sous le volet "Création", dans le groupe "Champs graphiques", déroulez l'option "Jauge" et sélectionnez une jauge linéaire ou circulaire. <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Champ_Jauge20.gif)

	Remarque : l'option "Jauge infinie" permet de créer un champ de type [Jauge infinie](../WDChamp/1000034001.md). 

2. Cliquez dans la fenêtre à la position où le champ doit être créé. Le champ apparaît en création.




Pour afficher les caractéristiques du champ, sélectionnez l'option "Description" dans le menu contextuel du champ.

**Remarques** : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Un grand choix de jauges circulaires prédéfinies est livré en standard. Ces jauges sont directement accessibles dans le volet "Champs prédéfinis" de l'éditeur.

- ![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Seules les jauges linéaires simples sont disponibles.

### Caractéristiques des jauges dans une fenêtre : fenêtre à 7 onglets
<a name="caracteristiques_des_jauges_dans_une_fenetre_fenetre_7_onglets_ELTPARAGRAPHE000143"></a>L'onglet "Général" permet de configurer :

- Le type de jauge : linéaire (horizontale ou verticale) ou circulaire (à aiguille ou LCD).

- Les différentes valeurs de la jauge. Ces valeurs peuvent également être spécifiées par programmation. Pour plus de détails, consultez [Manipuler une jauge par programmation](../WDChamp/1013141.md). 




L'onglet "Détail" permet de configurer :

- Les différentes images affichées dans la jauge (pour les jauges linéaires)
	Il est possible d'utiliser le mode 9 images sur les images affichées dans les jauges.

- Si la jauge est symétrique.
	![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Cette option n'est pas disponible.

- Si la jauge utilise une image fixe.

- Si le pourcentage de progression doit être affiché dans le champ Jauge. Il est possible d'indiquer si ce pourcentage doit être mis à blanc si la valeur affichée est zéro. Il est également possible de définir la position du pourcentage affiché. Cette position peut être modifiée par programmation grâce à la propriété [CadrageHorizontal](../Proprietes/2510023.md). La mise en forme de ce pourcentage peut être effectuée par programmation grâce à la propriété [MasqueSaisie](../Proprietes/2510033.md).
	![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Cette option n'est pas disponible.




L'onglet "Style" permet de configurer tous les paramètres de présentation de la jauge (y compris les images et leur position pour les jauges circulaires).

**Remarque** : Ces informations sont également disponibles pour les jauges affichées dans les champs Table.

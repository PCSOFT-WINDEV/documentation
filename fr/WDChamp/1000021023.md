
## Les graphes de type "Pyramide"
			



## Présentation
<a name="presentation_ELTTEXTE000138"></a>
Le graphe de type Pyramide permet de visualiser des données décroissantes. Chaque étage représente un pourcentage de l'étage inférieur. Par exemple le premier étage représente le pourcentage d'internautes qui remplissent leur panier, le 2ème étage représente le pourcentage de ces internautes qui ont validé leur panier et qui saisissent leurs coordonnées, ...

Comme son nom l'indique, il est représenté sous la forme d'une pyramide.

## Caractéristiques d'un graphe de type Pyramide
<a name="caracteristiques_graphe_type_pyramide_ELTTEXTE000162"></a>

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Graphe_pyramide%20-%20HC%20N%B0001.gif&type=thumb)


Dans ce graphe :

**1. Titre du graphe**. Les caractéristiques du titre du graphe sont saisies dans l'onglet "Détail" de la description du champ (bouton "Editer le titre du graphe").

Il est possible de définir le titre du graphe, sa position et son style.

**2. Première valeur**. Il est possible d'afficher ou non la valeur totale dans le graphe Pyramide. Pour ne pas afficher cette valeur, il suffit de cocher l'option "Masquer la première valeur" (onglet "Général" de la fenêtre de description).  

**3. Légende**. Les caractéristiques de la légende sont saisies dans l'onglet "Détail" de la description du champ (bouton "Editer la légende du graphe").

Il est possible de définir sa position et son style.

**4. Etiquette de catégorie**. Les étiquettes de catégories sont affichées dans la légende. Les étiquettes de catégorie peuvent être définies dans l'onglet "Séries" de la description du champ. La source des étiquettes de catégorie peut correspondre à une colonne de table, un tableau WLangage, une liste, ... 

Les étiquettes de catégorie peuvent être également affichées directement dans chaque portion de la pyramide. Il suffit de sélectionner l'option "Afficher les étiquettes des catégories" dans l'onglet "Général" de la fenêtre de description du champ. 

**5. Couleur des portions**. La couleur des portions est définie automatiquement. Pour modifier la couleur des portions, utilisez la fonction [grCouleurSérie](../WDLang3/3042013.md) en précisant le numéro de la portion. 

**6. Couleurs dégradées**. Les caractéristiques d'apparence du graphe sont définies dans l'onglet "Détail" de la fenêtre de description du graphe. Les options suivantes sont disponibles :

- Adoucir les effets d'escalier (Antialiasing).

- Couleurs de bordure automatiques.

- Couleurs dégradées (avec possibilité de dégradé horizontal).




**Autres caractéristiques** : 

- **Espacement entre les portions**. L'espacement entre les portions peut être défini par l'option "Espacement entre les portions " (onglet "Général" de la fenêtre de description du champ) ou par la fonction [grHistoEspacement](../WDLang3/3042014.md). 

- **Taille des portions**. L'onglet "Général" de la fenêtre de description du champ ou la fonction [grParamètre](../WDLang3/1000018337.md)) permet de choisir la taille des portions : 

	- Proportionnelle à l'aire : La valeur est symbolisée par l'aire de la portion de la pyramide.

	- Proportionnelle à la hauteur : La valeur est symbolisée par la hauteur de la portion de la pyramide. 




- Le graphe peut être orienté : Il suffit de définir l'angle de rotation selon les différents axes dans l'onglet "Général" de la fenêtre de description du champ. Il est également possible d'orienter le graphe par programmation grâce à la fonction [grOrientation](../WDLang3/3042043.md).

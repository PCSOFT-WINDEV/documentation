
## Valider une page différente de la page en cours

	A partir de la version <B>18</B>,  les framesets sont disponibles par compatibilité pour les anciens sites. ## Présentation
<a name="presentation_ELTTEXTE000082"></a>
Dans certains cas, il peut être nécessaire de proposer la validation d'une page par un bouton (ou un lien) présent dans une frame différente.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000106"></a>
Pour valider une page différente de la page en cours (c'est-à-dire présente dans une autre frame) :

1. Créez un bouton (ou un lien) dans la page contenant les informations à valider (page contenant le tableau dans l'exemple). Ce bouton (ou ce lien) peut être invisible. Seul le code de clic serveur de ce bouton sera exécuté.

2. Dans le code du bouton (ou du lien) permettant la validation de la page (lien "Valider les corrections" dans l'exemple), utilisez la fonction [PageValide](../WDLang2/3058020.md) en précisant :

	- le nom de la page à valider (page contenant le tableau dans l'exemple).

	- le nom du bouton (ou de lien) permettant la validation de la page (champ créé à l'étape n°1).

	- la destination (le nom de la frame) où le résultat de la validation doit être affiché.


## Manipuler un champ Ascenseur par programmation
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000144"></a>
WINDEV permet de manipuler un champ Ascenseur directement par programmation. Il suffit d'utiliser directement la variable du champ ascenseur dans le code.

La variable du champ Ascenseur :

- correspond au nom de l'ascenseur.

- est initialisée avec la position de la cage de l'ascenseur.






<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Récupérer la position du curseur
<a name="recuperer_position_curseur_ELTTEXTE000168"></a>
Pour récupérer la position du curseur dans un champ Ascenseur, utilisez la syntaxe suivante :


```txt
<Position> = <Nom Ascenseur>
```


&lt;Position&gt; est une variable de type entier

**Remarque** : Il est également possible d'utiliser la propriété [Valeur](../Proprietes/2510130.md).



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Modifier la position du curseur
<a name="modifier_position_curseur_ELTTEXTE000192"></a>
Pour modifier la position du curseur dans un champ Ascenseur, utilisez la syntaxe suivante :


```txt
<Nom Ascenseur> = <valeur>
```


&lt;Valeur&gt; est une valeur comprise entre la valeur minimale et la valeur maximale de l'ascenseur. Ces valeurs ont été décrites dans la fenêtre de description du champ, ou par programmation, grâce aux propriétés [BorneMin](../Proprietes/2510008.md) et [BorneMax](../Proprietes/2510009.md).

**Remarque** : Il est également possible d'utiliser la propriété [Valeur](../Proprietes/2510130.md).

## Propriétés spécifiques à la gestion d'un champ Ascenseur
<a name="proprietes_specifiques_gestion_champ_ascenseur_ELTTEXTE000216"></a>
Les propriétés suivantes sont spécifiques à la gestion des caractéristiques d'un champ Ascenseur par programmation.


|   |   |
| --- | --- |
| [Avance](../Proprietes/1000021574.md) | Permet de connaître et de modifier le pas d'avancement d'un champ Ascenseur. |
| [AvanceRapide](../Proprietes/2510011.md) | Permet de connaître et de modifier la valeur de l'avance rapide d'un champ Ascenseur. |
| [BorneMax](../Proprietes/2510009.md) | Permet de connaître et de modifier la borne maximale d'un champ Ascenseur. |
| [BorneMin](../Proprietes/2510008.md) | Permet de connaître et de modifier la borne minimale d'un champ Ascenseur. |


Pour connaître la liste complète des propriétés du WLangage pouvant être utilisées avec un champ de type Ascenseur, consultez [Propriétés associées au champ Ascenseur](../WDChamp/1013112.md).

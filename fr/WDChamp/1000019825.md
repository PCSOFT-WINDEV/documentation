## Présentation
<a name="presentation_ELTTEXTE000091"></a>
WINDEV, WEBDEV et WINDEV Mobile permettent de manipuler un champ Carte directement par programmation. Il suffit d'utiliser directement la variable du champ Carte dans le code.

La variable du champ Carte correspond au nom du champ Carte.

Cette page d'aide détaille comment manipuler un champ Carte par programmation. 

**Remarque** : De nombreuses fonctions du WLangage sont spécifiques à la gestion des champs de type Carte. Pour plus de détails, consultez [les fonctions de gestion des champs Carte](../WDLang3/1000022419.md).

## Initialiser une carte
<a name="initialiser_une_carte_ELTTEXTE000155"></a>


### Afficher une carte
<a name="afficher_une_carte_ELTPARAGRAPHE000178"></a>A l'ouverture de la fenêtre contenant un champ Carte, une carte du monde est automatiquement affichée : la carte est centrée sur l'Europe avec un niveau de zoom minimal.

Il est possible d'afficher la dernière position mémorisée : il suffit de sélectionner l'option "Mémoriser la dernière position affichée" dans l'onglet "Général" de la fenêtre de description du champ Carte. Dans ce cas, la dernière position est affichée avec le même niveau de zoom. 

Pour afficher une carte correspondant à un lieu spécifique, utilisez : 

- soit la fonction [&lt;Carte&gt;.AffichePosition](../WDLang3/1000022394.md) : Cette fonction peut être utilisée avec une position spécifique (variable de type [géoPosition](../WDLang3/1000019191.md)) ou sur une ville ou un monument. 
	Par exemple : 
	
	```wl
	// Centre la carte affichée par le champ "CARTE_Position" 
	// sur une position géographique 
	MaPosition est une geoPosition
	CARTE_Position.AffichePosition(MaPosition)
	
	// Centre la carte affichée par le champ "CARTE_Position" sur une ville 
	// à partir de son nom
	CARTE_Position.AffichePosition("Montpellier, France")
	
	// Centre la carte affichée par le champ "CARTE_Position" sur un monument 
	// à partir de son nom
	CARTE_Position.AffichePosition("Tour Eiffel")
	```


- soit l'affectation directe avec une position spécifique. Dans ce cas, la position est affichée sans animation. 
	Exemple : 
	
	```wl
	// Centre la carte affichée par le champ "CARTE_Position" 
	// sur une position géographique
	MaPosition est une geoPosition
	CARTE_Position = MaPosition
	```

## Fonctions WLangage utilisables avec les champs Carte
<a name="fonctions_wlangage_utilisables_avec_les_champs_carte_ELTTEXTE000275"></a>
Les fonctions WLangage utilisables avec les champs Carte peuvent être classées en plusieurs catégories : 

- Les fonctions permettant de gérer les positions sur la carte : 
	


|   |   |
| --- | --- |
| [&lt;Carte&gt;.AffichePosition](../WDLang3/1000022394.md) | Centre la carte affichée dans un champ Carte sur une position géographique ou un lieu donné. |
| [&lt;Carte&gt;.InfoPosition](../WDLang3/1000022447.md) | Renvoie les coordonnées en pixel du point d'un champ Carte correspondant à une position géographique (latitude et longitude). |
| [&lt;Carte&gt;.InfoXY](../WDLang3/1000022448.md) | Renvoie la position géographique (latitude et longitude) correspondant à un point d'un champ Carte. |
| [&lt;Carte&gt;.RécupèrePosition](../WDLang3/1000022463.md) | Renvoie la position géographique du point situé au centre de la carte actuellement affiché dans un champ Carte. |

- Les fonctions permettant de suivre le déplacement de l'appareil : 
	


|   |   |
| --- | --- |
| [&lt;Carte&gt;.FinDéplacement](../WDLang3/1000022437.md) | Arrête le suivi du déplacement de l'appareil dans un champ Carte. |
| [&lt;Carte&gt;.SuitDéplacement](../WDLang3/1000022464.md) | Affiche la position courante de l'appareil dans un champ Carte et met à jour cette position durant son déplacement. |

- Les fonctions permettant de gérer les marqueurs sur la carte. Ces fonctions utilisent des variables de type [Marqueur](../WDLang3/1000019940.md) permettant de définir toutes les caractéristiques des marqueurs. 
	


|   |   |
| --- | --- |
| [&lt;Carte&gt;.AjouteMarqueur](../WDLang3/1000022422.md) | Ajoute un nouveau marqueur sur une carte affichée dans un champ Carte. |
| [&lt;Carte&gt;.ModifieMarqueur](../WDLang3/1000022383.md) | Modifie un marqueur affiché dans un champ Carte. |
| [&lt;Carte&gt;.SupprimeMarqueur](../WDLang3/1000022466.md) | Supprime un marqueur d'une carte affichée par un champ Carte. |
| [&lt;Carte&gt;.SupprimeTout](../WDLang3/1000022472.md) | Supprime dans un champ Carte : <br><br>	- tous les marqueurs, <br><br>	- toutes les formes,<br><br>	- toutes les images.<br><br><br> |

- Les fonctions permettant de gérer les itinéraires sur la carte. 
	


|   |   |
| --- | --- |
| [&lt;Carte&gt;.AjouteItinéraire](../WDLang3/1000022420.md) | Ajoute un itinéraire sur un champ Carte. |
| [&lt;Carte&gt;.SupprimeItinéraire](../WDLang3/1000022465.md) | Supprime un itinéraire d'un champ Carte. |

- Les fonctions permettant de manipuler des options supplémentaires de l'API Google de gestion des cartes : 
	


|   |   |
| --- | --- |
| [CarteConfigureAPIGgl](../WDLang3/1000021611.md) | Déclare des paramètres supplémentaires pour le champ Carte qui seront transmis à l'API Google. |
| [&lt;Carte&gt;.ExécuteJS](../WDLang3/1000022384.md) | Exécute du code natif de l'API Google Maps dans un champ Carte en Javascript. |
| [&lt;Carte&gt;.RécupèreObjetJS](../WDLang3/1000022385.md) | Retourne l'instance de l'API d'un champ Carte. |

## Propriétés spécifiques aux champs Carte
<a name="proprietes_specifiques_aux_champs_carte_ELTTEXTE000484"></a>
Les propriétés suivantes sont spécifiques à la gestion d'un champ Carte par programmation : 



|   |   |
| --- | --- |
| [ModeCarte](../Proprietes/1000019956.md) | Renvoie ou modifie le mode d'affichage de la carte dans le champ Carte. |
| [Valeur](../Proprietes/2510130.md) | Permet de centrer la carte sur la position correspondant à la variable de type [géoPosition](../WDLang3/1000019191.md) spécifiée. |
| [Zoom](../Proprietes/1000017201.md) | Renvoie ou modifie le niveau de zoom utilisé pour l'affichage de la carte. |


Pour connaître la liste complète des propriétés du WLangage pouvant être utilisées avec un champ Carte, consultez [Propriétés associées au champ Carte](../WDChamp/1000019814.md).

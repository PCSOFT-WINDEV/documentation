
## Evénements associés à un champ Bandeau défilant
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Evénements gérés par défaut
<a name="evenements_geres_par_defaut_ELTTEXTE000101"></a>
WEBDEV gère par défaut les événements suivants (ordre d'apparition sous l'éditeur de code) :

| **Evénement** | **Condition d'exécution** |
| --- | --- |
| Initialisation (code serveur) | Exécuté à l'ouverture de la page. |
| A chaque changement de plan (code navigateur) | Exécuté à chaque changement de plan (par programmation, puce ou bouton). |
| A chaque changement de plan (code serveur) | Exécuté à chaque changement de plan (par programmation, puce ou bouton). |
| Retour de traitement AJAX après chaque changement de plan (code navigateur) | Exécuté quand le traitement Ajax sur le serveur a fini de s'exécuter. |
| Clic (code navigateur) | Exécuté lors d'un clic sur le champ Bandeau défilant. <br><br>Remarque : Cet événement est disponible uniquement si l'option "Activer le clic en fond de bandeau" a été cochée (option disponible dans l'onglet "Détail" de la fenêtre de description du champ). |
| Clic (code serveur) | Exécuté lors d'un clic sur le champ Bandeau défilant. <br><br>Remarque : Cet événement est disponible uniquement si l'option "Activer le clic en fond de bandeau" a été cochée (option disponible dans l'onglet "Détail" de la fenêtre de description du champ). |


## Menu contextuel des champs Diagramme de Gantt (FAA)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000161"></a>
Chaque champ Diagramme de Gantt présent dans une fenêtre propose par défaut un menu à l'utilisateur.

Ce menu, disponible sans programmation, permet des actions utiles :

- Modifier la durée d'une tâche, 

- Supprimer des tâches,

- Ajouter des tâches,

- Lier des tâches, 

- Imprimer le diagramme de Gantt.




![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=FAA_Gantt.gif)


**Remarque** : Il est possible d'ajouter des choix à ce menu par programmation. Il suffit d'utiliser la propriété [MenuContextuel](../Proprietes/2510077.md).



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Paramétrage
<a name="parametrage_ELTTEXTE000185"></a>


### Paramétrage du menu automatique dans un champ Diagramme de Gantt
<a name="parametrage_menu_automatique_dans_champ_diagramme_gantt_ELTPARAGRAPHE000032"></a>Le menu contextuel du champ Diagramme de Gantt peut bien évidemment être désactivé. Pour paramétrer l'affichage du menu contextuel du champ Diagramme de Gantt, vous devez :

1. Afficher la fenêtre de description du champ Table ou Table hiérarchique.

2. Sélectionner la colonne de type "Diagramme de Gantt". 

3. Sélectionner l'onglet "UI".

4. Choisir le menu désiré dans la combo "Menu contextuel". Il est possible de : 

	- "Afficher le menu des FAA (Système)" : Dans ce cas, le menu contextuel affiché sera le menu automatique du champ Diagramme de Gantt. Pour plus de détails sur la configuration de ce menu, consultez [Configurer le menu des FAA](../Editeurs/2010040.md).

	- "Ajouter un menu contextuel" : Si cette option est cochée, vous pouvez sélectionner le menu contextuel personnalisé à afficher. ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=FAA_Menu_Agenda.gif)








Remarques : 

- Si les deux options sont cochées, le menu contextuel personnalisé peut être ajouté avant ou après le menu des FAA. 

- Si aucune option n'est sélectionnée, aucun menu contextuel ne sera sélectionné. L'option &lt;Désactivé&gt; sera affichée dans la fenêtre de description. 

- La propriété [MenuContextuel](../Proprietes/2510077.md) permet également de supprimer le menu contextuel spécifique aux FAA et de retrouver le menu Windows classique pour un champ spécifique.

- Par programmation, la fonction [FAADésactive](../WDLang1/1000022018.md) permet de supprimer certaines options du menu contextuel des FAA spécifique aux champs Diagramme de Gantt. Différentes constantes peuvent être utilisées selon l'option à désactiver : 

	- *faaGanttAjouteDépendance* : permet de désactiver : 

		- l'ajout d'une tâche préalable,

		- la suppression de toutes les tâches préalables. 




- *faaGanttAjouteTâche* : permet de désactiver l'ajout d'une tâche. 

- *faaGanttEditeTâche* : permet de désactiver l'édition d'une tâche. 

- *faaGanttSupprimeTâche* : permet de désactiver la suppression d'une tâche. 

- *faaImprimeEtatSurTable* : permet de désactiver l'impression du diagramme de Gantt.

## Impression d'un diagramme de Gantt
<a name="impression_diagramme_gantt_ELTTEXTE000209"></a>
<a name="NOTE3_1"></a>
L'option "Imprimer" permet d'imprimer le contenu de la colonne de type Diagramme de Gantt.   

Les caractéristiques de l'impression proposée sont les suivantes :

- Le titre de l'impression correspond au libellé du champ Table (ou Table hiérarchique) affichant le champ Diagramme de Gantt. Si aucun libellé n'a été spécifié, l'état n'affiche aucun titre.

- L'impression respecte les couleurs utilisées dans le champ Diagramme de Gantt.

- La date du jour est imprimée en haut à droite.

- Le numéro de la page courante et le nombre total de pages sont imprimés en bas à droite.

- L'impression réalisée prend en compte le sens d'écriture spécifié avec la propriété [SensEcriture](../Proprietes/2513020.md) au niveau de la fenêtre et/ou au niveau du champ Diagramme de Gantt.




**Remarques** : 

- L'option "Imprimer" du menu contextuel des colonnes Diagramme de Gantt nécessite la livraison des librairies d'impression du framework WINDEV avec l'exécutable.

- Pour ne pas proposer l'option "Imprimer" dans le menu contextuel du champ Diagramme de Gantt, utilisez la fonction [FAADésactive](../WDLang1/1000022018.md) avec la constante *faaImprimeEtatSurTable*.

- Pour personnaliser l'état associé à un champ Diagramme de Gantt spécifique, il faut créer un nouvel état de type Diagramme de Gantt basé sur ce champ Diagramme de Gantt. Cet état sera automatiquement associé au champ Diagramme de Gantt et imprimé par l'option "Imprimer". Pour plus de détails, consultez [Etat de type Diagramme de Gantt](../WDChamp/1011084.md).

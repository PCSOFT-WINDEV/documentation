## Présentation du champ Calculé
<a name="presentation_champ_calcule_ELTTEXTE000128"></a>


### Champ d'un état
<a name="champ_etat_ELTPARAGRAPHE000011"></a>Le champ "Calculé" permet de réaliser des calculs dans un état. Ces calculs peuvent être réalisés :

- sur une rubrique de la source de données.

- sur un champ présent dans l'état.

- par programmation.






<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Créer un champ calculé
<a name="creer_champ_calcule_ELTTEXTE000152"></a>
Pour créer un champ "Calculé" :

1. Sous le volet "Création", dans le groupe "Texte", cliquez sur "Calcul".

2. Cliquez dans l'état à la position où le champ doit être créé.

3. Dans l'assistant de création du champ Calculé, spécifiez : 

	- si le champ calculé effectue un comptage d'enregistrements. 

	- si le champ calculé effectue un calcul sur une rubrique ou un champ de l'état. Dans ce cas, sélectionnez le type de calcul (somme, moyenne, ...) et la source du calcul (nom du champ ou nom de la rubrique).

	- si le champ calculé affiche un calcul réalisé par programmation. Dans ce cas, sélectionnez le masque à utiliser. 







A tout moment, ces informations peuvent être modifiées dans la fenêtre de description du champ (option "Description" du menu contextuel).
<a name="NOTE2_2"></a>


### Important : Différence entre calcul et calcul sur rupture
<a name="important_difference_entre_calcul_calcul_sur_rupture_ELTPARAGRAPHE000066"></a>Dans les calculs sur rupture, le champ est ré-initialisé à chaque rupture.

## Notes
<a name="notes_ELTTEXTE000182"></a>


### Champ "Calculé" spécifique : "Monétaire + Euro"
<a name="champ_calcule_specifique_monetaire_euro_ELTPARAGRAPHE000075"></a>Si le champ "Calculé" est de type "Monétaire + Euro", il est possible de gérer :

- la ***devise mémorisée*** : devise des données présentes dans la source de données de l'état. Si le champ est manipulé par programmation, la devise de la valeur sera la devise mémorisée.

- la ***devise affichée*** : devise utilisée pour afficher la valeur dans l'état. Cette devise peut être différente de la devise mémorisée. La conversion est automatiquement réalisée.

- le ***double-affichage*** : il est possible de relier deux champs de type "Monétaire + Euro" entre eux. La valeur affichée dans le champ de base est automatiquement convertie dans le champ suiveur.

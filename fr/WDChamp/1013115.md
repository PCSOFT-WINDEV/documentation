
## Caractéristiques d'un champ Boîte à outils
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000139"></a>
Les principales caractéristiques d'un champ Boîte à outils sont configurées :

- dans l'onglet "Général" de la description du champ : il est possible de configurer chaque volet du champ Boîte à outils.

- dans l'onglet "UI" de la description du champ :  il est possible de configurer la gestion des volets.

- dans l'onglet "Détail" de la description du champ : il est possible de configurer l'image de fond des volets.




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Configuration des volets du champ Boîte à outils
<a name="configuration_des_volets_champ_boite_outils_ELTTEXTE000163"></a>
L'onglet "Général" de la fenêtre de description du champ Boite à outils permet de :

- ajouter, supprimer, insérer de nouveaux onglets.

- configurer pour le volet sélectionné dans l'aperçu :

	- l'état initial du volet.

	- son libellé.

	- l'image associée au libellé du volet. 
			**Attention** : Cette image ne correspond pas à l'image de fond du volet mais à une image affichée à gauche du libellé.







**Remarque** : Il n'y a pas d'ascenseurs automatiques dans les volets des champs Boîte à outils. Si la totalité des champs ne peut être affichée, les champs sont tronqués.



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Gestion des volets
<a name="gestion_des_volets_ELTTEXTE000187"></a>
L'onglet "UI" de la fenêtre de description du champ Boîte à outils permet entre autres de gérer l'affichage des volets du champ.

Il est ainsi possible de configurer :

- l'animation du changement de volet.

- la mémorisation du volet actif. Si cette option est sélectionnée, le volet actif sera mémorisé à la fermeture de la fenêtre. Ce volet sera automatiquement sélectionné lors de la prochaine ouverture de la fenêtre.






<a name="NOTE4"></a>
<a name="NOTE4_1"></a>


## Gestion de l'image de fond des volets
<a name="gestion_image_fond_des_volets_ELTTEXTE000211"></a>
L'onglet "Détail" de la description du champ Boîte à outils permet de gérer l'image de fond des volets. Cette image est utilisée par tous les boutons de volets. Cette image peut contenir les différents états du bouton associé au volet.

**Astuce** : Pour obtenir des images originales, consultez le catalogue d'images de WINDEV/WINDEV Mobile et sélectionnez une image de bouton à 5 états.
<a name="NOTE4_2"></a>


### Etats des volets
<a name="etats_des_volets_ELTPARAGRAPHE000061"></a>Il est possible de gérer les différents états du volet grâce à l'image associée au volet.

Chaque état correspond à un dessin spécifique. Tous les dessins sont enregistrées dans une seule image. Cette image est associée au volet : il suffit de préciser le nombre d'états (c'est-à-dire de dessins) présents dans le fichier image.

Exemple : fichier image voletBleu.gif
<br>![Volet au repos enfoncé grisé sélectionné survolé](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Bleu0208.gif)


Les formats d'images supportés sous Windows sont :

- Bitmap (\*.BMP)

- Graphics Interchange Format (\*.gif)

- Joint Picture Experts Group (\*.jpg ; \*.jpeg)

- Kodak Photo CD (\*.pcd)

- PaintBrush (\*.pcx)

- Adobe Photoshop Format (\*.psd)

- TrueVision TARGA (\*.tga)

- Tagged Image File Format (\*.tif ; \*.tiff)
	Remarque : l'option "tiff mosaïque" de la norme tiff 6.0 n'est pas supportée. Dans ce cas, il est conseillé d'enregistrer l'image à la norme tiff 5.0. 

- Portable Network Graphics (\*.png)

- Windows MetaFiles (\*.emf ; \*.wmf)

- Icônes (\*.ico ; \*.icw)
	Remarques : 

	- Affichage HQ (Haute Qualité) disponible.

	- Gestion des planches au format BMP transparent (32 bits).




- Curseurs (\*.cur)

- Portable Document Format (\*.pdf)

- Scalable Vector Graphics (\*.svg)

- Images WINDEV (générées par l'éditeur d'images, \*.WDPIC).





<a name="NOTE4_3"></a>


### Cas particulier : image des volets à trois images
<a name="cas_particulier_image_des_volets_trois_images_ELTPARAGRAPHE000078"></a>L'image des volets sera découpée en 3 images automatiquement par WINDEV.

La règle de découpage est la suivante :

- l'image est découpée en trois parties

- la partie 1 correspond à la zone de gauche

- la partie 3 correspond à la zone de droite

- la partie 2 correspond à la zone du milieu. La taille de cette partie est égale à 16 pixels. Si le volet d'onglet est proportionnel au libellé, cette partie sera répétée autant de fois que nécessaire pour contenir le libellé.



<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=BoiteOutil3Image.gif)


**Remarque** : Si la partie 2 de votre image est inférieure à 16 pixels, il est conseillé de ne pas utiliser un volet proportionnel au libellé et d'intégrer si nécessaire les images supplémentaires ainsi que le libellé à l'image.

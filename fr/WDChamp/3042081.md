


## Champ Graphe : Onglet "Séries"
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000227"></a>
L'onglet "Séries" permet de définir les différentes séries utilisées dans le champ Graphe ainsi que la source des étiquettes de catégories. 



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Source des étiquettes de catégories
<a name="source_des_etiquettes_categories_ELTTEXTE000251"></a>
Les étiquettes de catégories sont les étiquettes représentant les catégories affichées dans le graphe. 

Les étiquettes de catégories sont affichées :

- sous les graduations horizontales pour les graphes de type courbe ou les graphes boursiers de type chandelier japonais et BarCharts,

- sous l'axe des X pour les histogrammes ou les graphes de type boursier Minimum/Maximum.

<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=EtiquetteCat.gif)
Il est possible de choisir : 

- la source : 

	- Remplie par programmation, 

	- Parcours d'un fichier de données ou d'une requête, 

	- Colonne de table, 
			

	- Champ Liste, 
			

	- Tableau WLangage. 
			




- les données. Les informations à donner dépendent de la source sélectionnée. 




Remarque : Dans un état, les sources de données disponibles sont : 

- Remplie par programmation, 

- Parcours d'un fichier de données ou d'une requête,  

- Tableau WLangage, 

- Source de données de l'état. 




<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Source des séries
<a name="source_des_series_ELTTEXTE000275"></a>
La source des séries correspond au mode de stockage des séries prises en compte par le graphe. 

Il est possible : 

- d'ajouter une série dans le tableau via le bouton "+". 

- de modifier le libellé, la source et les données dans le tableau. 




Les sources disponibles et les données associées sont : 

| Source de données | Données attendues |
| --- | --- |
| Remplie par programmation | Valeurs initiales de la série. Les valeurs doivent être séparées par un retour chariot ou un ";". |
| Parcours d'un fichier ou d'une requête | Rubrique source, choisie parmi les fichiers de données décrits dans l'analyse. |
| Colonne de table | Nom de la colonne source. <br><br> |
| Champ Liste | Nom du champ Liste. <br><br> |
| Tableau WLangage | Nom du tableau WLangage, type du tableau et nom de la colonne ou du membre à parcourir. <br><br> |
| ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Courbe de tendance | Type de courbe à définir dans les options de la série. Il est possible d'utiliser : <br><br>- une courbe de tendance de type constante. Il suffit de préciser la valeur dans les options de la série sélectionnée. <br><br>- une courbe de tendance de type moyenne. Il suffit de préciser la série de référence. <br><br>- une courbe de tendance de type régression linéaire. Il suffit de préciser la série de référence. <br>	Attention : Ce type de courbe de tendance fonctionne uniquement avec une seule série. <br><br>- une courbe de tendance de type régression quadratique. ## Options de la série sélectionnée
<a name="options_serie_selectionnee_ELTTEXTE000299"></a>
Pour chaque série sélectionnée dans le tableau des série, il est possible de définir des options spécifiques : 

- **Type de série** : Type de graphe associé à la série. Par défaut, ce paramètre correspond au type de graphe affiché dans le champ (défini dans l'onglet "Général"). 
	![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Pour créer des graphes composites, il est possible de choisir des types de graphes différents pour chaque série.

- **Couleur** : Couleur de la série. 

- **Opacité** : Pourcentage d'opacité de la série. 

- **Type de point** : Forme du point (cercle, carré, ...). Il est possible de ne pas dessiner le point. 

- **Parcourir la source des données à chaque dessin pour mettre à jour les valeurs.** : Cette option permet de mettre à jour les données du graphe à chaque nouveau dessin (affichage) du graphe. 

- **Utiliser l'échelle secondaire** : La série sera affichée en utilisant l'échelle secondaire. 




En fonction du type de série sélectionné (Courbe, Aire, Nuage de points et Histogramme), des propriétés peuvent être définies ou surchargées par rapport au type de graphe défini dans l'onglet "Général".

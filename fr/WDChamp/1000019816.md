
## Organigramme lié à un fichier de données
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000091"></a>
Il est possible de créer des champs Organigrammes directement liés aux données. Vous pouvez créer un **champ Organigramme Fichier** : le champ Organigramme sera lié à un fichier de données ou à une requête. Dans ce cas, le fichier de données contient les caractéristiques des éléments affichés dans le champ. 






## Comment le faire ?
<a name="comment_faire_ELTTEXTE000115"></a>
Pour créer un champ Organigramme fichier, il est nécessaire de modifier la description du champ Organigramme.

**Pour modifier les caractéristiques d'un champ Organigramme existant :** 

1. Affichez la fenêtre de description du champ (option "Description" du menu contextuel). 

2. Dans l'onglet "Contenu", sélectionnez l'option "Fichier/Requête" et sélectionnez la source de données utilisée. Cette source de données peut être : 

	- un fichier de données défini dans l'analyse,

	- une requête du projet,

	- une requête intégrée à la fenêtre. 




3. Indiquez les différents éléments à prendre en compte pour réaliser la hiérarchie des éléments de l'organigramme : 

	- la rubrique de parcours

	- l'identifiant du parent (présent dans l'enregistrement de la source) et l'élément auquel il est lié. 




4. Associez les différentes rubriques de la source de données aux caractéristiques de l'élément. Les éléments précédés d'une étoile sont obligatoires. 

5. Validez la fenêtre de description du champ Organigramme. Le champ Organigramme est un organigramme fichier. 














## Manipuler un champ Organigramme Fichier
<a name="manipuler_champ_organigramme_fichier_ELTTEXTE000139"></a>
Un champ Organigramme Fichier se manipule très simplement. L'utilisateur peut manipuler les éléments via une fenêtre spécifique. Il suffit de modifier directement le fichier de données lié à l'organigramme. Les données affichées seront automatiquement mises à jour. Il est également possible d'utiliser la fonction [OrgaAffiche](../WDLang1/1000019764.md) pour remettre à jour les données affichées dans le champ.




## Gestion des menus contextuels dans une application WINDEV ou WINDEV Mobile
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000145"></a>
Un menu contextuel présent dans une fenêtre peut être ouvert par l'utilisateur : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) grâce à un clic droit de la souris.

- ![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) par un appui long sur le champ. 


Ce menu permet aux utilisateurs d'accéder plus facilement aux fonctionnalités spécifiques d'une fenêtre ou d'un champ d'une fenêtre.

WINDEV et WINDEV Mobile permettent de gérer plusieurs types de menus contextuels :

- **Les menus contextuels personnalisés**. Ces menus sont créés sous l'éditeur de fenêtres et peuvent être manipulés par programmation. 
	Pour plus de détails, consultez [Menu contextuel personnalisé](../WDChamp/1010017.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) **Les menus contextuels des FAA** (Fonctionnalités Automatiques de vos Applications) gérés par WINDEV. Ces menus contextuels s'affichent lors d'un clic droit sur certains types de champs (Champ de saisie, Graphe, Table, ...). Ces menus contextuels permettent à l'utilisateur final d'utiliser des fonctionnalités disponibles automatiquement dans l'application. 
	Pour plus de détails sur les FAA, consultez [Fonctionnalités Automatiques de vos Applications](../Editeurs/9000022.md).
	![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) Les menus contextuels des FAA sont disponibles uniquement sur les champs Table.

- **Les menus contextuels système** gérés par le système (Android, Windows, ...). Ces menus contextuels s'affichent sur certains types de champs (ActiveX, ...). 
	Pour plus de détails, consultez [Menu contextuel système](../WDChamp/1010023.md).
	![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Les menus contextuels système ne sont pas disponibles.




Exemple de menu contextuel personnalisé : ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=MenuContextuel.gif)


Remarque : Dans une page WEBDEV, il est également possible de gérer des menus contextuels grâce au champ [Menu contextuel](../WDChamp/1410087217.md).

## Menu contextuel des champs Monétaire + Euro
<a name="menu_contextuel_des_champs_monetaire_euro_ELTTEXTE000169"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Dans les champs de type "Monétaire WINDEV + Euro" (champ de saisie, colonne d'un champ Table, ...), il est possible de proposer à l'utilisateur un menu contextuel permettant de choisir la devise d'affichage. Ce menu contextuel présente le résultat de la conversion de la valeur saisie dans les différentes devises gérées.
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=MenuctxMonetaireeuro.gif)
Si ce menu contextuel est proposé aux utilisateurs, le menu contextuel associé au champ ne sera pas pris en compte.

Pour activer ce menu contextuel système : 

1. Affichez la fenêtre de description du champ de saisie de type "Monétaire + Euro". 

2. Dans l'onglet "Détail", dans la zone "Gestion de l'Euro", cliquez sur le bouton "Options". 

3. Dans la fenêtre qui s'affiche, cochez l'option "Devise modifiable par menu contextuel". Validez la fenêtre. 

4. Validez la fenêtre de description du champ.




## Manipulations des blocs d'un état
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000168"></a>
Les différentes manipulations possibles sur les blocs d'un état sont les suivantes :

- [Créer un bloc](#NOTE2_1).

- [Supprimer un bloc](#NOTE3_1).

- [Réordonner les blocs](#NOTE4_1).


Remarque : Lors du clic sur le nom d'un bloc ou dans une zone de l'état, les délimiteurs du bloc sont mis en surbrillance.  



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Créer un bloc dans un état
<a name="creer_bloc_dans_etat_ELTTEXTE000192"></a>
Pour créer un bloc dans un état, réalisez une des manipulations suivantes :

- sous le volet "Création", dans le groupe "Structure", déroulez "Blocs".

- sélectionnez l'option "Ajouter un bloc" du menu contextuel de l'état (clic droit sur l'état en cours de création).

- cliquez sur le bouton "Ajouter un bloc" dans l'onglet "Blocs" de la description de l'état.




**Remarque** : Les blocs *Haut et Bas de rupture* sont indissociables. Il est cependant possible de rendre un de ces blocs invisible (dans la fenêtre de description du bloc ou par la propriété [Visible](../Proprietes/2510138.md)).

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Supprimer un bloc d'un état
<a name="supprimer_bloc_etat_ELTTEXTE000216"></a>
Pour supprimer un bloc d'un état, utilisez :

- soit l'option "Supprimer le bloc / la rupture" du menu contextuel du bloc à supprimer (clic droit de la souris sur l'intitulé du bloc).

- soit le bouton "Supprimer un bloc" dans l'onglet "Blocs" de la description de l'état.

- soit la touche Suppr du clavier (le bloc à supprimer étant sélectionné).




**Remarque** : Le bloc *Corps* ne peut pas être supprimé. Il est possible de rendre ce bloc invisible (dans la fenêtre de description du bloc ou par la propriété [Visible](../Proprietes/2510138.md)).

## Réordonner les blocs d'un état
<a name="reordonner_les_blocs_etat_ELTTEXTE000240"></a>


### Réordonner des blocs
<a name="reordonner_des_blocs_ELTPARAGRAPHE000088"></a>L'éditeur d'états permet de réordonner les blocs présents dans un état. Lors de l'impression de l'état, les différents blocs seront imprimés selon l'ordre spécifié sous l'éditeur d'états. L'ordre d'exécution des traitements des blocs correspond à l'ordre d'impression des blocs.
Pour réordonner les blocs d'un état :

1. Affichez la fenêtre de description de l'état (option "Description de l'état" du menu contextuel de l'état). 

2. Affichez l'onglet "Blocs".

3. Modifiez l'ordre d'impression des blocs : 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Etat_Onglet_Blocs.gif)



**Remarque** : Il est également possible de modifier l'ordre d'impression des blocs dans la fenêtre de description de chaque bloc (option "Description du bloc" du menu contextuel du bloc).


### Cas particulier : réordonner les blocs d'itération
<a name="cas_particulier_reordonner_les_blocs_iteration_ELTPARAGRAPHE000103"></a>Par défaut, l'ordre d'affichage des blocs d'itération dans un état correspond à l'ordre de création de ces blocs. 

Pour réordonner les blocs d'itération d'un état :

1. Affichez la fenêtre de description de l'état (option "Description de l'état" du menu contextuel de l'état). 

2. Affichez l'onglet "Blocs".

3. Sélectionnez le nom du bloc d'itération à modifier et utilisez les boutons "Monter le bloc" et "Descendre le bloc". 
Remarque : Ces options sont également disponibles dans le menu contextuel du bloc.

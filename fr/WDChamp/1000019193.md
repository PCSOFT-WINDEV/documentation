
## Les graphes de type "Entonnoir"
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000146"></a>
Le graphe en entonnoir affiche sous forme d'entonnoir des données. Une fois totalisées, ces données sont égales à 100 %. Ce type de graphe est un graphe à série unique. Il représente les données comme des parties de 100 %. Ce type de graphe n'utilise pas d'axe.

Les graphes de type Entonnoir sont un type de graphe souvent utilisé pour représenter les étapes d'un processus de vente et indiquer le montant des recettes potentielles ou de concrétisation attendue pour chaque étape. 

Le graphique en entonnoir est particulièrement destiné à visualiser la représentation des opportunités d'affaires. Ainsi, les affaires les plus proches de la signature sont en bas de l'entonnoir, les affaires les moins avancées sont en haut. 

C'est une solution utile pour assister les décideurs lors de la phase d'instruction du processus de prise de décision.

## Caractéristiques d'un graphe de type Entonnoir
<a name="caracteristiques_graphe_type_entonnoir_ELTTEXTE000170"></a><br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=genEntonnoir.gif&type=thumb)


Dans ce graphe :

**1. Titre du graphe**. Les caractéristiques du titre du graphe sont saisies dans l'onglet "Détail" de la description du champ (bouton "Editer le titre du graphe").

Il est possible de définir le titre du graphe, sa position et son style.

**2. Embout de l'entonnoir**. Les caractéristiques de l'embout de l'entonnoir sont saisies dans l'onglet "Général" de la description du champ : 

- Largeur de l'embout (exprimée en pourcentage). 

- Hauteur de l'embout (exprimée en pourcentage). 




**3. Première valeur**. Il est possible d'afficher ou non la valeur totale dans le graphe Entonnoir. Pour ne pas afficher cette valeur, il suffit de cocher l'option "Masquer la première valeur" (onglet "Général" de la fenêtre de description).  

**4. Légende**. Les caractéristiques de la légende sont saisies dans l'onglet "Détail" de la description du champ (bouton "Editer la légende du graphe").

Il est possible de définir sa position et son style.

**5. Etiquette de catégorie**. Les étiquettes de catégories sont affichées dans la légende. Les étiquettes de catégorie peuvent être définies dans l'onglet "Séries" de la description du champ. La source des étiquettes de catégorie peut correspondre à une colonne de table, un tableau WLangage, une liste, ... 

Les étiquettes de catégorie peuvent être également affichées directement dans chaque portion de l'entonnoir. Il suffit de sélectionner l'option "Afficher les étiquettes des catégories" dans l'onglet "Général" de la fenêtre de description du champ. 

**6. Couleur des portions**. La couleur des portions est définie automatiquement. Pour modifier la couleur des portions, utilisez la fonction [grCouleurSérie](../WDLang3/3042013.md) en précisant le numéro de la portion. 

**7. Couleurs dégradées**. Les caractéristiques d'apparence du graphe sont définies dans l'onglet "Détail" de la fenêtre de description du graphe. Les options suivantes sont disponibles :

- Adoucir les effets d'escalier.

- Couleurs de bordure automatiques.

- Couleurs dégradées (avec possibilité de dégradé horizontal).




**Autres caractéristiques** : 

- **Espacement entre les portions**. L'espacement entre les portions peut être défini par l'option "Espacement entre les portions " (onglet "Général" de la fenêtre de description du champ) ou par la fonction [grHistoEspacement](../WDLang3/3042014.md). 

- **Taille des portions**. L'onglet "Général" de la fenêtre de description du champ ou la fonction [grParamètre](../WDLang3/1000018337.md)) permet de choisir la taille des portions : 

	- Proportionnelle à l'aire : La valeur est symbolisée par l'aire de la portion de l'entonnoir.

	- Proportionnelle à la hauteur : La valeur est symbolisée par la hauteur de la portion de l'entonnoir.

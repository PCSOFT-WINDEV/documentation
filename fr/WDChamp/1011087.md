


## Visualisateur de rapports : Volet "Rechercher"
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000096"></a>
Lors de la prévisualisation d'un état, une fonctionnalité de recherche est automatiquement disponible, via le volet "Recherche". Cette recherche est effectuée dans toutes les pages affichées dans le visualisateur de rapports.

Cette fonctionnalité est disponible dans les impressions réalisées à l'aide de l'éditeur d'états (ou avec le logiciel "Etats & Requêtes") et dans les impressions programmées (réalisées avec les fonctions WLangage).

Rappel : Le visualisateur de rapports est disponible : 

- sous l'éditeur d'états de WINDEV, WEBDEV, WINDEV Mobile. 

- dans le logiciel "Etats et Requêtes".

- lors de l'impression d'états depuis une application WINDEV. 




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000120"></a>
Pour lancer la recherche depuis le visualisateur de rapports : 

1. Affichez le volet "Rechercher" du visalisateur de rapports : 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Apercu_Volet_Rechercher.gif)


2. Saisissez les caractéristiques de la recherche :

	- Le texte recherché.

	- Si la recherche doit être effectuée sur le mot entier (option "Mot complet uniquement").

	- Si la recherche doit respecter la casse (option "Respecter la casse").




3. Cliquez sur le bouton "Loupe" pour lancer la recherche. L'élément trouvé est surligné en vert.
	Les boutons fléchés "Suivant" et "Précédent" permettent de se déplacer entre les éléments trouvés.




**Remarque**

- Si le visualisateur de rapports affiche un fichier RTF avec des images, la recherche ne fonctionne pas.

- La fonction [iParamètreVisualisateur](../WDLang5/1000024634.md)permet de désactiver si nécessaire le volet de recherche (constante *iBoutonRecherche*).

## Créer, ouvrir un frameset
<a name="creer_ouvrir_frameset_ELTTEXTE000291"></a>


### Créer un frameset
<a name="creer_frameset_ELTPARAGRAPHE000049"></a>WEBDEV propose plusieurs méthodes pour créer un frameset :

- Créer un frameset vierge.

- Utiliser un frameset prédéfini.

- Construire automatiquement un site basé sur des framesets à partir de l'analyse.




Pour plus de détails, consultez [Créer un frameset](../WDChamp/1012002.md).


<a name="NOTE2_2"></a>


### Ouvrir un frameset
<a name="ouvrir_frameset_ELTPARAGRAPHE000066"></a>Pour ouvrir un frameset : 

1. Sous le volet "Accueil", dans le groupe "Général", déroulez "Ouvrir" et sélectionnez "Ouvrir" (Ctrl + O).

2. Sélectionnez le frameset à ouvrir.




Autres manipulations :

- Double-cliquez sur le nom du frameset affiché dans le volet "Explorateur de projet", option "Pages".

- Double-cliquez sur la représentation du frameset dans le graphe du projet, sous l'éditeur de projet.

## Associer une frame à une page, associer une frame dans un frameset
<a name="associer_une_frame_une_page_associer_une_frame_dans_frameset_ELTTEXTE000321"></a>


### Associer une frame à une page
<a name="associer_une_frame_une_page_ELTPARAGRAPHE000089"></a>Pour associer une frame à une page : 

1. Dans le volet "Explorateur de projet", sélectionnez la page à associer.

2. Réalisez un "Drag and Drop" (Glisser / Déplacer) de la page présente dans le volet "Explorateur de projet" vers la frame où la page doit être affichée.

3. Enregistrez le frameset : sous le volet "Page", dans le groupe "Edition", déroulez "Frame" et sélectionnez "Enregistrer le frameset".




Remarques :

- Pour associer une frame à une page, il est également possible de cliquer à l'intérieur d'une frame et d'ouvrir la page à associer à la frame en cours. La page ouverte apparaît dans la frame en cours.

- Lors de la création et de l'enregistrement du frameset, l'état initial du frameset est automatiquement défini. Pour plus de détails sur l'état initial d'un frameset, consultez le paragraphe [Définir l'état initial d'un frameset](#NOTE6_1).

## Ajouter ou supprimer une frame dans un frameset
<a name="ajouter_supprimer_une_frame_dans_frameset_ELTTEXTE000345"></a>


### Ajouter une frame dans un frameset
<a name="ajouter_une_frame_dans_frameset_ELTPARAGRAPHE000121"></a>Pour ajouter une frame dans un frameset : 

1. Sous le volet "Page", dans le groupe "Edition", déroulez "Frame" et sélectionnez : 

	- "Vertical" pour réaliser un fractionnement vertical. 

	- "Horizontal" pour réaliser un fractionnement horizontal.




2. Sélectionnez la position de la nouvelle frame (à droite, à gauche, en haut ou en bas) par rapport à la frame en cours.


La frame sélectionnée se dédouble. Il est possible de redimensionner les deux nouvelles frames. Pour plus de détails, consultez le paragraphe [Modifier les dimensions d'une frame](#NOTE5_1).


<a name="NOTE4_2"></a>


### Supprimer une frame d'un frameset
<a name="supprimer_une_frame_frameset_ELTPARAGRAPHE000150"></a>Pour supprimer une frame d'un frameset : 

1. Sélectionnez la frame à supprimer. Le contour de la frame sélectionnée apparaît en rouge.

2. Sous le volet "Page", dans le groupe "Edition", déroulez "Frame" et sélectionnez "Supprimer la frame". Les frames restantes se redimensionnent automatiquement pour occuper l'espace libéré.

## Modifier les dimensions d'une frame
<a name="modifier_les_dimensions_une_frame_ELTTEXTE000375"></a>
Pour modifier les dimensions d'une frame : 

1. Cliquez à l'intérieur de la frame à modifier. Le contour de la frame sélectionnée apparaît en rouge.

2. Survolez le bord à redimensionner. Lorsque le curseur de la souris se transforme en double-flèche, maintenez le bouton gauche de la souris enfoncé et redimensionnez la frame.

3. Relâchez le bouton gauche de la souris. Le redimensionnement est effectué.


Pour éviter l'affichage intempestif des ascenseurs dans les frames, les champs présents dans les pages associées aux frames doivent être visibles dans le frameset.

## Définir les pages initiales du frameset
<a name="definir_les_pages_initiales_frameset_ELTTEXTE000399"></a>
Il est nécessaire de définir les pages initiales d'un frameset : ces pages seront ouvertes par défaut à l'ouverture du frameset sous le navigateur de l'internaute.

Pour définir les pages initiales du frameset :

1. Associez les pages voulues aux différentes frames.

2. Sous le volet "Page", dans le groupe "Edition", déroulez "Frame" et sélectionnez "Définir l'état initial".




**Rappel** : Un frameset permet d'afficher plusieurs pages dans une même fenêtre d'un navigateur. Le fichier HTML correspondant au frameset contient :

- La définition du découpage des différentes frames dans le navigateur.

- Le nom des pages ouvertes lors de l'affichage du frameset dans le navigateur.





<a name="NOTE6_2"></a>


### Imbriquer les framesets
<a name="imbriquer_les_framesets_ELTPARAGRAPHE000213"></a>Par défaut, un frameset permet d'afficher plusieurs pages dans une même fenêtre du navigateur. Il est possible qu'une des pages affichées dans le frameset soit elle-même un autre frameset : on parle de "Framesets imbriqués". Les framesets imbriqués peuvent être utilisés par exemple pour réaliser des mises en page originales dans vos sites.

Pour plus de détails, consultez [Imbriquer des framesets](../WDChamp/1012006.md)

## Conditions d'utilisation
<a name="conditions_utilisation_ELTTEXTE000123"></a>
**Le champ Publicité utilise la librairie Google Play Service de Android.**
Pour pouvoir utiliser un champ Publicité, il est nécessaire : 

- d'installer Google Play Service dans le SDK Android. 
	Pour cela, lancez le SDK Manager et vérifiez si "Google Play Services", présent dans les extra, est bien installé. 

- [d'indiquer l'identifiant du groupe d'annonces lors de la génération de l'application Android](#NOTE3_1). 




**Attention** : L'affichage des publicités AdMob dans une application est soumise à un certain nombre de règles (nombre de publicités visibles à la fois, emplacement des publicités, ...). Pour plus de détails, consultez la page [https://support.google.com/admob/bin/answer.py?hl=fr&answer=1307237&topic=1307235&ctx=topic](https://support.google.com/admob/bin/answer.py?hl=fr&answer=1307237&topic=1307235&ctx=topic)

## Identifiant AdMob du groupe d'annonces
<a name="identifiant_admob_groupe_annonces_ELTTEXTE000147"></a>


### Présentation
<a name="presentation_ELTPARAGRAPHE000032"></a>Pour que le champ Publicité puisse afficher des publicités, il est nécessaire de définir un groupe d'annonces sur le site de "AdMob", afin de définir le type de publicités à obtenir. Lors de cet enregistrement, un identifiant est associé au groupe d'annonces. Cet identifiant doit être indiqué dans l'assistant de génération de l'application Android. 


### Enregistrement de l'application sur le site AdMob
<a name="enregistrement_application_sur_site_admob_ELTPARAGRAPHE000037"></a>Attention : Ces manipulations sont données à titre indicatif. L'interface a pu être modifiée depuis l'écriture de ce document. L'interface utilisée dans cette page de documentation correspond à l'interface disponible en Janvier 2014 (nouvelle interface AdMob de Google). 

**Pour enregistrer une nouvelle application Android sur AdMob** : 

1. Connectez-vous au site AdMob : [http://fr.admob.com/](http://fr.admob.com/) (un compte Google ou AdMob est nécessaire). 

2. Créez si nécessaire un compte AdMob. 

3. Lorsque le compte est créé, vous pouvez ajouter votre site ou application.

4. Renseignez si besoin toutes les informations nécessaires au paiement. 

5. Dans la page d'accueil, cliquez sur "Monétiser" ou sur le bouton "Monétiser une nouvelle application". <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Admob_1.gif)


6. Sélectionnez ou créez l'application à monétiser. <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=admob_2.gif)
Vous pouvez : 

	- Rechercher une application existante. 

	- Ajouter une application manuellement. 




7. Renseignez les informations concernant l'application : 

	- le nom de l'application. Ce nom peut être différent du nom final de votre application. 

	- la plateforme de l'application : Android




8. Après validation de votre application, vous pouvez créer une publicité. <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=admob_3.gif)


9. Après validation du groupe d'annonces, l'identifiant du bloc d'annonces est fourni. <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=admob_4.gif)
Cet identifiant doit être saisi dans l'assistant de création de l'application Android. 







### Saisie de l'identifiant dans l'assistant de génération
<a name="saisie_identifiant_dans_assistant_generation_ELTPARAGRAPHE000070"></a>Dans l'assistant de génération de l'application Android, indiquez l'identifiant du bloc d'annonces. 

Remarque : Une erreur de chargement d'AdMob peut survenir tant que le bloc de publicité n'est pas encore disponible. Cette opération peut prendre plusieurs heures.

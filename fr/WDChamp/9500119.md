## Présentation
<a name="presentation_ELTTEXTE000113"></a>
Un dispositif "multi-touch", appelé en français multi-point ou multi-tactile, est une technique permettant à une personne d'interagir avec un matériel informatique à l'aide de plusieurs points de contact (le plus souvent, ce sont les doigts qui sont utilisés). 

L'une des applications les plus courante du multi-touch est la manipulation d'images. La taille de l'affichage sur un téléphone étant souvent réduite, il est souvent nécessaire de zoomer et / ou de se déplacer dans une image. 

Il est ainsi possible de réaliser un zoom sur une image avec le contact de 2 doigts qui s'écartent. 

WINDEV Mobile permet la gestion du multi-touch dans les applications Android et iPhone/iPad grâce à : 

- [Des options spécifiques disponibles pour les champs Image](../WDChamp/9500131.md).

- [Des fonctions WLangage spécifiques](#NOTE2_1).

## Multi-touch : Fonctions WLangage spécifiques
<a name="multitouch_fonctions_wlangage_specifiques_ELTTEXTE000137"></a>
Plusieurs fonctions WLangage permettent de gérer le multi-touch dans les traitements suivants des champs : 

- Touché / Bouton gauche enfoncé (traitement optionnel)

- Relâché / Bouton gauche relâché (traitement optionnel)

- Tape / Clic

- Double tap / Bouton gauche double-clic (traitement optionnel)

- Déplacement (traitement optionnel). 




Ces fonctions sont les suivantes : 



|   |   |
| --- | --- |
| [GesteNbPointeur](../WDLang1/1000019750.md) | Retourne le nombre de pointeurs en contact avec l'écran. |
| [GestePointeurEncours](../WDLang1/1000019751.md) | Retourne l'indice du pointeur (doigt ou stylet) ayant déclenché l'appel d'un traitement spécifique au multi-touch dans un champ Image. |
| [GestePosX](../WDLang1/1000019741.md) | Retourne la position horizontale du pointeur (doigt ou stylet). |
| [GestePosY](../WDLang1/1000019745.md) | Retourne la position verticale du pointeur (doigt ou stylet). |

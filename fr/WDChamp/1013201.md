
## Calculs dans les colonnes d'un champ Table
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000150"></a>
Lorsqu'un champ Table contient des données numériques, il est souvent intéressant de connaître le total d'une colonne ou le comptage d'un nombre d'éléments.

WINDEV permet de réaliser et d'afficher automatiquement différents calculs pour les colonnes : 

- somme, 

- moyenne, 

- comptage, 

- maximum,

- minimum.




Quelques exemples d'utilisation des calculs dans les colonnes d'un champ Table :

- le total des pièces commandées,

- le nombre de personnes ayant une adresse email,

- le montant moyen dépensé par chaque client.




Par programmation, il est également possible de définir des formules de calcul spécifiques et d'afficher le résultat en bas de la colonne voulue. Pour plus de détails, consultez [Programmation de calculs personnalisés dans les tables](../WDChamp/1013331.md). 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Définir les calculs à réaliser
<a name="definir_les_calculs_realiser_ELTTEXTE000174"></a>
Pour réaliser un ou plusieurs calculs sur une colonne : 

1. Affichez la fenêtre de description du champ Table (option "Description" du menu contextuel).

2. Sélectionnez le nom d'une colonne pour laquelle un calcul doit être réalisé.

3. Affichez l'onglet "Détail" de cette colonne (dans la partie basse de l'écran).

4. Sélectionnez les différents calculs à réaliser : 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=TotauxTable.gif)


	- somme des cellules de la colonne,

- moyenne des cellules de la colonne,

- compteur des valeurs non nulles de la colonne. Si le champ Table contient des cellules fusionnées, elles sont considérées comme valeur nulle.

- minimum des valeurs des cellules de la colonne.

- maximum des valeurs des cellules de la colonne. Remarque : Les calculs disponibles dépendent du type de la colonne. Le compteur est disponible sur tous les types de colonnes. 

5. Répétez les opérations 2 à 4 pour toutes les colonnes pour lesquelles un calcul doit être effectué.




**Remarque** : il est possible de manipuler les différents calculs par programmation grâce à des constantes spécifiques. Pour plus de détails, consultez [Programmation des calculs dans les champs Table](../WDChamp/1013248.md).

## Configurer la position des calculs
<a name="configurer_position_des_calculs_ELTTEXTE000198"></a>
Pour configurer la position des calculs : 

1. Affichez l'onglet "Général" du champ Table (pour cela, sélectionnez le nom du champ Table et cliquez sur l'onglet "Général").

2. Indiquez la position des totaux dans la zone "Afficher les résultats". Pour afficher le ou les résultats :

	- dans une ou plusieurs lignes ajoutées directement après la dernière ligne du champ, sélectionnez l'option "Après la dernière ligne, dans le champ Table".
			Par exemple :![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=calculTableLignesAjoutees.gif)
Ces lignes sont visibles uniquement :

		- au bas du champ Table, si le champ ne possède pas d'ascenseur vertical.

		- quand l'ascenseur vertical est tout en bas, si le champ Table possède un ascenseur vertical.




- sous le champ Table, sélectionnez l'option "Sous le champ Table".
			Par exemple :![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CalculTableSousTable.gif)
Ces lignes sont toujours visibles.

- il est également possible de réaliser des calculs automatiques sans les afficher (option "Sans affichage").

3. Modifiez si nécessaire le libellé affiché devant les différents calculs (bouton "Modifier les libellés").

4. Validez la fenêtre de description du champ Table.

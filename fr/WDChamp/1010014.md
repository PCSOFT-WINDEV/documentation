
## Manipuler un groupe de champs par programmation
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000112"></a>
WINDEV, WEBDEV et WINDEV Mobile permettent de manipuler un groupe de champs directement par programmation. Il suffit d'utiliser directement la variable correspondant au nom du groupe de champs dans le code.

La variable du groupe de champs correspond au nom du groupe de champs.

Exemple : 

```wl
// Manipulation du groupe de champs GR_Avis
// Rend le groupe de champs invisible
GR_Avis.Etat = Invisible
```


La manipulation des groupes de champs est globale à la fenêtre en cours d'exécution : elle inclut tous les groupes de champs de la fenêtre, y compris les groupes de champs présents dans une fenêtre interne, une page interne, un modèle de champs ou un superchamp. Dans ce cas là, le nom du groupe de champs ne doit pas être préfixé par le nom de l'élément le contenant (fenêtre interne, page interne, modèle de champs ou superchamp par exemple).

Exemple : 

```wl
// Manipulation du groupe de champs GR_Avis 
// présent dans la fenêtre interne FI_Utilisateur
// Rend le groupe de champs invisible
GR_Avis.Etat = Invisible
```


Remarque : Il est possible de connaître par programmation les champs qui composent un groupe de champs grâce à la fonction [EnumèreChamp](../WDLang1/3025005.md).

## Propriétés spécifiques aux groupes de champs
<a name="proprietes_specifiques_aux_groupes_champs_ELTTEXTE000136"></a>
Les propriétés suivantes sont spécifiques à la gestion des caractéristiques d'un groupe de champs par programmation.



|   |   |
| --- | --- |
| [Etat](../Proprietes/2510039.md) | Permet de connaître et de modifier l'état d'affichage d'un groupe de champs. |
| [MenuContextuel](../Proprietes/2510077.md) | Permet de connaître et de changer le menu contextuel associé à un groupe de champs. |
| [Modifié](../Proprietes/2510080.md) | Permet de savoir si un champ d'un groupe de champs a été modifié par l'utilisateur (saisie au clavier ou à la souris). |
| [Occurrence](../Proprietes/2510097.md) | Permet de connaître le nombre de champs présents dans un groupe de champs. |


Pour connaître la liste complète des propriétés du WLangage pouvant être utilisées avec un groupe de champs, consultez [Propriétés associées à un groupe de champs](../WDChamp/1010018.md).

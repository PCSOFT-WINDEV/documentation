## Présentation du champ Potentiomètre
<a name="presentation_champ_potentiometre_ELTTEXTE000134"></a>


### Champ d'une page
<a name="champ_une_page_ELTPARAGRAPHE000011"></a>Un potentiomètre est un champ qui permet de sélectionner une valeur entre deux bornes. Son fonctionnement ressemble à celui d'un ascenseur. La valeur courante est identifiée par un curseur que l'utilisateur déplace à l'aide de la souris.

WEBDEV propose deux types de potentiomètres :

- **potentiomètre linéaire** : Composé d'une règle et d'un curseur, ce champ est très utile pour proposer à l'internaute la saisie d'une valeur dans un intervalle donné. ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Potentio_Lin_WB.gif)


- **potentiomètre d'intervalle** : Ce potentiomètre est un potentiomètre linéaire à 2 curseurs. Il permet à l'internaute de définir un intervalle de valeurs. Par exemple, sur un site de vente, ce champ permet à l'internaute de définir une fourchette de prix pour filtrer les produits affichés. ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Potentio_Intervalle_WB.gif)

## Créer un champ Potentiomètre
<a name="creer_champ_potentiometre_ELTTEXTE000158"></a>
Pour créer un champ de type Potentiomètre : 

1. Sous le volet "Création", dans le groupe "Champs graphiques", déroulez "Potentiomètre" et sélectionnez le type de potentiomètre à créer : 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Potentio_Creation_WB.gif)


2. Cliquez dans la fenêtre à la position où le champ doit être créé. Le champ apparaît en création.




Pour afficher les caractéristiques du champ, sélectionnez l'option "Description" dans le menu contextuel du champ.

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Remarque : WINDEV propose également un champ Potentiomètre. Pour plus de détails, consultez [Champ Potentiomètre](../WDChamp/1013241.md).

## Caractéristiques du champ Potentiomètre
<a name="caracteristiques_champ_potentiometre_ELTTEXTE000182"></a>
L'onglet "Général" du champ Potentiomètre permet de spécifier les principales caractéristiques du champ : 

- Orientation : Il est possible d'utiliser un potentiomètre horizontal ou vertical. 

- Borne minimale : Borne de départ des valeurs du potentiomètre. 

- Borne maximale : Borne extrême des valeurs du potentiomètre.  




**Uniquement pour le potentiomètre linéaire** : 

- Valeur initiale : Position de départ du curseur du potentiomètre. 




**Uniquement pour le potentiomètre d'intervalle** : 

- Valeur inférieure : Position de départ du curseur inférieur du potentiomètre. 

- Valeur supérieure : Position de départ du curseur supérieur du potentiomètre. 





### Orientation du champ Potentiomètre
<a name="orientation_champ_potentiometre_ELTPARAGRAPHE000075"></a>Un champ Potentiomètre peut être horizontal ou vertical. 

Pour modifier l'orientation d'un champ Potentiomètre : 

1. Sélectionnez le champ Potentiomètre dans la page. 

2. Dans le menu contextuel du champ, sélectionnez l'option "Choisir un style WEBDEV". 

3. Dans la fenêtre qui s'affiche, sélectionnez un  style de potentiomètre vertical. 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=WB_Style_Potentiometre.gif&type=thumb)


4. Validez.

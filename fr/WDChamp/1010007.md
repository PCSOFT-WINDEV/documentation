


## Manipuler les options de menu par programmation
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000166"></a>
Les options de menu peuvent être manipulées :

- depuis l'éditeur de fenêtres. Pour plus de détails, consultez [Manipuler les options de menu sous l'éditeur](../WDChamp/1010008.md).

- par programmation.






<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Principe
<a name="principe_ELTTEXTE000190"></a>
La manipulation d'une option de menu est identique que le menu soit un menu contextuel ou un menu principal.

Pour manipuler par programmation une option de menu, utilisez la notation :

- ** "&lt;Nom de l'option&gt;"** si l'option de menu est manipulée depuis la fenêtre.

- **"&lt;Nom de la fenêtre&gt;.&lt;Nom du menu&gt;.&lt;Nom de l'option&gt;"** si l'option est manipulée depuis une autre fenêtre.




**Remarque** : il est également possible de manipuler les menus contextuels par programmation (et non uniquement ses options). Pour plus de détails, consultez [Menu contextuel personnalisé](../WDChamp/1010017.md).

| [MenuAjoute](../WDLang1/3038048.md) | Ajoute un menu contextuel existant dans le menu principal de la fenêtre en cours. |

| [MenuAjouteOption](../WDLang1/1000017262.md) | Ajoute une nouvelle option de menu à la suite de la dernière option d'un menu (déroulant ou contextuel). |
| [MenuAjouteSéparateur](../WDLang1/1000017270.md) | Ajoute un nouveau séparateur dans un menu (menu déroulant ou contextuel). |
| [MenuClone](../WDLang1/1000017257.md) | Clone un menu ou une option de menu ainsi que le code associé. |
| [MenuDémarque](../WDLang1/3038002.md) | Enlève la marque placée devant l'option de menu (menu contextuel ou déroulant). |
| [MenuEtat](../WDLang1/3038003.md) | Identifie l'état d'une option de menu (menu contextuel ou déroulant) : active, inactive ou invisible. |
| [MenuExiste](../WDLang1/1000019365.md) | Indique si une option de menu existe dans un menu. |
| [MenuInsèreMenu](../WDLang1/1000017267.md) | Insère un menu avant un autre menu dans une fenêtre. |
| [MenuInsèreOption](../WDLang1/1000017265.md) | Insère une nouvelle option à une position spécifique. |
| [MenuInsèreSéparateur](../WDLang1/1000017271.md) | Insère un séparateur dans un menu. |
| [MenuInvisible](../WDLang1/3038022.md) | Rend une option de menu invisible (menu contextuel ou déroulant). |
| [MenuLibellé](../WDLang1/3038009.md) | Identifie ou modifie le libellé d'une option de menu (menu contextuel ou déroulant). |

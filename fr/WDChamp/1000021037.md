
## Graphe avec affichage en temps réel
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000136"></a>
Ce type de graphe permet de représenter en temps réel des flux de données permanents. Le graphe se déplace seul en fonction des données reçues en temps réel. 

Un graphe en temps réel est une option spécifique des graphes de type Aire, Courbe, Histogramme ou Nuage de points. 

Pour qu'un graphe deviennent un graphe avec affichage en temps réel, il est nécessaire : 

- dans la fenêtre de description du champ, d'indiquer que le graphe utilise une échelle des abscisses espacée en fonction du temps et de spécifier l'intervalle utilisé pour un graphe en temps réel. 

- d'ajouter les données dans le graphe par programmation via la fonction [grAjouteDonnéeTemporelle](../WDLang3/1000023532.md). Cette fonction permet de mettre à jour le graphe. 
	Remarque : Il est conseillé d'utiliser une procédure automatique (de type Timer) pour mettre à jour régulièrement les données du graphe.

## Caractéristiques des graphes en temps réel
<a name="caracteristiques_des_graphes_temps_reel_ELTTEXTE000160"></a>

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Graphe%20temps%20r%E9el%20-%20HC%20N%B0001.gif&type=thumb)


Dans ce graphe :

**1. Titre du graphe**. 
Les caractéristiques du titre du graphe sont saisies dans l'onglet "Détail" de la description du champ (bouton "Editer le titre du graphe").

Il est possible de définir le titre du graphe, sa position et son style.

**2. Titre des axes**. 
Le titre des axes horizontal et vertical est affiché si le titre a été défini dans l'onglet "Général" de la description du champ Graphe. Ce paramètre peut être défini également par programmation grâce à la fonction [grTitreAxe](../WDLang3/3042057.md).

**3. Abscisse**. 
Dans un graphe en temps réel, l'échelle des abscisses est espacée en fonction du temps. Dans l'onglet "Général" de la fenêtre de description du champ Graphe, il est possible de définir les éléments suivants : 

- Mode d'affichage du temps  : Il est possible : 

	- d'utiliser un affichage standard : la durée écoulée commence à 0.  

	- d'afficher la durée écoulée depuis l'ajout le plus récent : la durée est recalculée à chaque affichage. 0 correspond au dernier ajout effectué dans le graphe. 




- Intervalle affiché pour un graphe temps réel (ms) : Cet intervalle est utilisé pour définir l'intervalle de temps affiché dans le graphe. Si l'ntervalle de temps est de 30000ms, le graphe affichera les données sur un intervalle de 30 secondes. Par défaut, cet intervalle de temps vaut 0 (équivalent à 60s).  




**4. Légende**. 
Les caractéristiques de la légende sont saisies dans l'onglet "Détail" de la description du champ (bouton "Editer la légende du graphe").

Il est possible de définir sa position et son style.

**5. Etiquettes de séries**. 
Le libellé de la légende correspond au libellé des différentes séries affichées dans le graphe (définie dans l'onglet "Séries").

**6. Couleur de série**. 
Pour chaque série, il est possible de définir une couleur dans l'onglet "Séries" de la description du champ. Par défaut la couleur utilisée est automatique.

**7. Etiquette de catégorie**. 
Les étiquettes de catégorie peuvent être définies dans l'onglet "Séries" de la description du champ.

**8. Quadrillage**. 
Il est possible de définir le quadrillage horizontal et vertical dans l'onglet "Général" de la description du champ.

## Présentation
<a name="presentation_ELTTEXTE000277"></a>
Les champs Table affichés dans une fenêtre offrent de nombreuses possibilités de manipulations à l'utilisateur final. Certaines fonctionnalités sont automatiques, certaines fonctionnalités demandent uniquement quelques modifications simples dans l'application pour les obtenir.

Cette page d'aide présente les Fonctionnalités Automatiques de vos Applications (FAA) disponibles sur les colonnes des champs Table : 

- [Modifier la taille des colonnes](#NOTE2_1).

- [Ajuster automatiquement la taille des colonnes](#NOTE2B_1). 

- [Changer l'ordre des colonnes](#NOTE3_1).

- [Fixer les colonnes](#NOTE4_1). 

- [Trier des colonnes](../WDChamp/9500126.md). 

- [Rechercher dans les colonnes](#NOTE6_1).

- [Filtrer le contenu des colonnes](../WDChamp/9500107.md).

- [Afficher le contenu de la colonne sur plusieurs lignes](#NOTE7_1).

- [Copier la colonne](#NOTE8_1).




Remarque : Ces Fonctionnalités Automatiques de vos Applications (FAA) sont aussi disponibles sur les champs Table hiérarchique.

## Modifier la taille des colonnes
<a name="modifier_taille_des_colonnes_ELTTEXTE000301"></a>
A tout moment, l'utilisateur peut modifier la taille des colonnes d'un champ Table.

Pour modifier la taille des colonnes d'un champ Table :

1. Survolez le séparateur de colonnes.

2. Lorsque le curseur de redimensionnement apparaît, maintenez le bouton gauche de la souris enfoncé.

3. Déplacez le séparateur de colonnes afin d'obtenir la dimension voulue.




Exemple : <br>![La taille de la première colonne est trop petite 
pour visualiser le nom complet des sociétés.](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Table_Taille_avant.gif)

<br>![La première colonne a été agrandie.](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Table_Taille_apres.gif)

<a name="NOTE2_2"></a>


### Astuce
<a name="astuce_ELTPARAGRAPHE000074"></a>Pour ajuster automatiquement la taille de la colonne au texte le plus long présent dans cette colonne, il suffit de double-cliquer sur le séparateur de colonnes (à droite du titre de la colonne à ajuster).

**Remarque** : Pour que l'ajustement automatique soit disponible, il est nécessaire de cocher l'option "Redimensionnable" dans l'onglet "Général" de la description de la colonne.

## Ajuster automatiquement la taille des colonnes
<a name="ajuster_automatiquement_taille_des_colonnes_ELTTEXTE000331"></a>
A tout moment, l'utilisateur peut ajuster la taille des colonnes d'un champ Table à leur contenu.

Pour ajuster au contenu la taille des colonnes d'un champ Table :

1. Cliquez sur le titre de la colonne avec le bouton droit de la souris. Un menu contextuel apparaît. Ce menu contextuel concerne la colonne en cours. 

2. Sélectionnez l'option "Ajuster toutes les colonnes". 

3. Les colonnes sont automatiquement ajustées à leur contenu. 




**Remarque** : Pour que l'ajustement automatique soit disponible, il est nécessaire de cocher l'option "Redimensionnable" dans l'onglet "Général" de la description de la colonne.

## Changer l'ordre des colonnes
<a name="changer_ordre_des_colonnes_ELTTEXTE000355"></a>
A tout moment, l'utilisateur final peut réorganiser les colonnes d'un champ Table. Il suffit de réaliser les opérations suivantes :

1. Cliquez sur le titre de la colonne à déplacer et maintenez le bouton gauche de la souris enfoncé.

2. Déplacez la colonne jusqu'à sa nouvelle position puis relâchez le bouton gauche de la souris.

## Fixer les colonnes
<a name="fixer_les_colonnes_ELTTEXTE000385"></a>
Il est possible de fixer une colonne pour en visualiser le contenu : la colonne est fixée au bord du champ Table et si nécessaire, l'ascenseur horizontal permet d'accéder aux autres colonnes du champ Table. 

Il est possible de fixer ou non les colonnes d'un table : 

- à l'aide de la souris :  

	1. Survolez la partie gauche du titre de la dernière colonne à fixer.

	2. Lorsque le curseur de souris se transforme en cadenas ouvert, cliquez avec le bouton gauche de la souris. La colonne sélectionnée et toutes les colonnes à gauche de cette colonne seront alors fixées. Ces colonnes ne peuvent plus être déplacées.


 Pour ne plus fixer des colonnes, il suffit d'effectuer la même opération.

- à l'aide du menu contextuel de la colonne : 

	1. Cliquez sur le titre de la colonne à fixer avec le bouton droit de la souris. Un menu contextuel apparaît. Ce menu contextuel concerne la colonne en cours. 

	2. Pour fixer la colonne, sélectionnez l'option "Fixée à gauche". Une coche apparaît devant l'option. La colonne sélectionnée et toutes les colonnes à gauche de cette colonne seront alors fixées. Ces colonnes ne peuvent plus être déplacées.. 

	3. Pour ne plus fixer les colonnes, sélectionnez l'option "Fixée à gauche" d'une colonne fixée. La coche disparaît de l'option. La colonne sélectionnée et toutes les colonnes à droite de cette colonne ne seront plus fixées. 







Exemple : <br>![La colonne Nom a été fixée. Les colonnes Société et Nom ne sont plus déplaçables](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Table_Col_Fixee.gif)


**Astuce** : Pour ajouter cette fonctionnalité, il est nécessaire de cocher l'option "Redimensionnable" dans l'onglet "Général" de la description de la colonne.

## Recherche dans les colonnes
<a name="recherche_dans_les_colonnes_ELTTEXTE000409"></a>
Pour effectuer une recherche dans un champ Table, il suffit de :

1. Cliquer sur la loupe présente dans le titre de la colonne. Une zone de saisie apparaît.

2. Saisir les premières lettres de l'élément cherché. Le premier élément correspondant est automatiquement sélectionné dans le champ Table.




Remarques :

- Pour ajouter cette fonctionnalité, il est nécessaire de cocher l'option "Recherche (loupe/filtre)" dans l'onglet "Général" de la fenêtre de description de la colonne.

- Il est également possible de filtrer les éléments affichés dans un champ Table. Pour plus de détails, consultez [Filtre dans les colonnes d'un champ Table](../WDChamp/9500107.md).



<a name="NOTE6_2"></a>


### Astuce
<a name="astuce_ELTPARAGRAPHE000173"></a>Il est également possible de faire une recherche en saisissant directement les premières lettres recherchées. Il suffit de :

1. Trier la colonne où la recherche doit être effectuée (la loupe doit être affichée dans cette colonne).

2. Saisir les premières lettres de l'élément cherché. Le premier élément correspondant est automatiquement sélectionné dans la table.

## Afficher le contenu de la colonne sur plusieurs lignes
<a name="afficher_contenu_colonne_sur_plusieurs_lignes_ELTTEXTE000439"></a>
Dans certains cas, les colonnes de tables sont trop étroites pour afficher toutes les informations disponibles. Il est alors nécessaire d'agrandir la colonne. Mais dans certains cas, la taille de la colonne devient aussi grande que la taille de la table (sinon plus) ! 

Pour éviter ces manipulations, il suffit d'utiliser le menu contextuel de la colonne :

1. Cliquez sur le titre de la colonne avec le bouton droit de la souris. Un menu contextuel apparaît. Ce menu contextuel concerne la colonne en cours. 

2. Sélectionnez l'option "Afficher sur plusieurs lignes". 

3. Le contenu de la colonne est automatiquement affiché sur plusieurs lignes.  




**Remarques** :

- Le contenu de la colonne de la table doit contenir des espaces pour être affiché sur plusieurs lignes. 

- Il est possible d'afficher toutes les colonnes texte sous forme de colonnes multilignes en cochant l'option "Colonnes texte multilignes" dans l'onglet "Détail" de la fenêtre de description du champ Table. 

- Comme tous les menus contextuels proposés par les FAA, il est possible de configurer les options proposées par défaut par le champ Table. Pour plus de détails, consultez [Configurer les menus FAA](../Editeurs/2010040.md). 

- En programmation : 

	- la fonction [FAADésactive](../WDLang1/1000022018.md) utilisée avec la constante *faaAfficherSurPlusieursLignes* supprime l'option de menu.

	- la propriété [Multiligne](../Proprietes/2510105.md) utilisée sur une colonne permet de savoir si la colonne est affichée en mode multiligne ou non.

## Copier la colonne
<a name="copier_colonne_ELTTEXTE000463"></a>
Pour copier les données présentes dans une colonne d'un champ Table : 

1. Cliquez sur le titre de la colonne avec le bouton droit de la souris. Un menu contextuel apparaît. Ce menu contextuel concerne la colonne en cours. 

2. Sélectionnez l'option "Copier la colonne". 

3. Le contenu de la colonne est automatiquement copié dans le presse-papiers et peut être collé dans n'importe quelle application.  




**Remarque** : Comme tous les menus contextuels proposés par les FAA, il est possible de configurer les options proposées par défaut par le champ Table. Pour plus de détails, consultez [Configurer les menus FAA](../Editeurs/2010040.md).




## Manipuler un champ Barre de navigation par programmation
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000090"></a>
Le champ Barre de navigation peut être manipulé directement par programmation. Il suffit d'utiliser directement la variable du champ Barre de navigation dans le code. La variable du champ Barre de navigation correspond au nom du champ Barre de navigation. Le type de la variable est une chaîne de caractères.

Remarque : Les différentes zones du champ Barre de navigation ne peuvent pas être manipulées par programmation indépendamment. 



## Manipuler le champ Barre de navigation par programmation
<a name="manipuler_champ_barre_navigation_par_programmation_ELTTEXTE000114"></a>


### Dérouler la zone enroulée
<a name="derouler_zone_enroulee_ELTPARAGRAPHE000017"></a>La propriété [Enroulé](../Proprietes/2513014.md) utilisée en code navigateur sur le champ Barre de navigation permet d'enrouler ou de dérouler la zone centrale de la barre de navigation. 
Il est par exemple possible d'utiliser la propriété [Enroulé](../Proprietes/2513014.md) dans le code de survol d'un champ. 

La propriété [Enroulé](../Proprietes/2513014.md) a un effet uniquement si le champ Barre de navigation est en mode compact. 

Exemple : 

```wl
// -- Code de chargement de la page (navigateur) 
// Variable utile pour masquer le menu déroulé en cas de clic en dehors
gbClicSurBurger est un booléen = Faux

// -- Clic sur la page (code navigateur)
SI PAS gbClicSurBurger ALORS
	// On masque le menu
	NAV_Bandeau.Enroulé = Vrai
FIN
gbClicSurBurger = Faux
```




## Le champ Table sur mobile (Android et iOS)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000276"></a>
Le champ Table est disponible pour les applications mobiles Android et iOS. Ce champ reprend les fonctionnalités du champ Table de WINDEV.

Le déplacement (scrolling vertical et horizontal) s'effectue au doigt.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Fonctionnalités disponibles
<a name="fonctionnalites_disponibles_ELTTEXTE000300"></a>
Le champ Table propose les fonctionnalités suivantes pour les applications Android et iOS :

- Remplissage : 

	- par programmation ([Table par programmation](../WDChamp/1013183.md)).

	- liaison fichier ([Table fichier](../WDChamp/1013177.md) avec data binding). 

	- liaison variable ([Table sur variable](../WDChamp/1013306.md)).




- Redimensionnement de colonne, directement effectué au doigt. 

- Ancrage de colonne (particulièrement utile quand la fenêtre passe de portrait à paysage). 

- Sélection simple ou sélection multiple. 

- Sur-entête de colonne. 

- Menu contextuel. 

- Saisie de données.

- ...


Les types de colonnes suivants sont proposés : Texte, Numérique, Monétaire, Date, Heure, Libellé, Image, Jauge, Notation.

Remarque : La fonctionnalité "Tirer pour rafraîchir" (Pull to Refresh) est disponible sur le champ Table sous Android et iOS. Pour plus de détails, consultez [Pull to Refresh sur les champs Table et Zone répétée (Android/IOS)](../WDChamp/9000156.md). 
<a name="NOTE2_2"></a>


### Saisie dans un champ Table
<a name="saisie_dans_champ_table_ELTPARAGRAPHE000054"></a>La saisie dans les champs Table sur mobile est gérée automatiquement.

Pour effectuer une saisie dans une cellule du champ Table : 

- Cliquez sur la cellule dans laquelle la saisie doit être effectuée. 

- Une zone de saisie s'ouvre en haut du champ Table. Si la cellule contient un texte, ce texte est affiché.![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Android_ChampTable_Saisie.gif)


- Modifiez ou ajoutez le texte voulu. 

- Validez.  



<a name="NOTE2_3"></a>


### Programmation du champ Table
<a name="programmation_champ_table_ELTPARAGRAPHE000069"></a>Les fonctions WLangage de la famille [Table](../WDLang1/3074009.md) sont supportées en Android et iOS.

## FAA du champ Table
<a name="faa_champ_table_ELTTEXTE000336"></a>
Les FAA disponibles pour le champ Table sur mobile sont :

- [Tri](#NOTE3_2).

- [Recherche grâce à la loupe](#NOTE3_3).

- [Filtre](#NOTE3_4).

- [Redimensionnement de colonne](#NOTE3_5).


Les FAA sont accessibles "au doigt" par l'utilisateur final.
<a name="NOTE3_2"></a>


### Tri
<a name="tri_ELTPARAGRAPHE000101"></a>Les colonnes triables sont identifiées par l'icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Android_ChampTable_ico_tri.gif)
. 

Pour trier une colonne, il suffit de taper avec le doigt sur l'entête de colonne. Plusieurs cas peuvent se présenter : 

- La colonne est de type Texte, Numérique, ... : un menu contextuel apparaît.![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Android_ChampTable_Mnu_tri.gif)
Cliquez sur les icônes de tri pour choisir le sens de tri.

- La colonne est de type Notation ou Image, le tri est fait lors dès le touché du doigt. 


Dans tous les cas, une fois la colonne triée, une flèche indique le sens de tri dans l'entête de la colonne. 
<a name="NOTE3_3"></a>


### Recherche grâce à la loupe
<a name="recherche_grace_loupe_ELTPARAGRAPHE000119"></a>Pour effectuer une recherche de type "Commence par" dans un champ Table : 

- Tapez avec le doigt sur l'entête de colonne pour afficher le menu contextuel. 

- Sélectionnez la loupe : ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Android_ChampTable_Loupe.gif)


- Dans la zone de saisie qui s'affiche en haut du champ Table, saisissez les premières lettres de l'élément voulu. ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Android_TableSaisieLoupe.gif)


- Le premier élément correspondant à la recherche est sélectionné et affiché dans le champ Table.  


Attention : Le clic sur la croix présente dans la zone de saisie annule la recherche. 
<a name="NOTE3_4"></a>


### Filtre des données : filtre de type "Contient"
<a name="filtre_des_donnees_filtre_type_contient_ELTPARAGRAPHE000135"></a>Pour filtrer les éléments affichés dans un champ Table (filtre de type "Contient") : 

- Tapez avec le doigt sur l'entête de colonne pour afficher le menu contextuel. 

- Sélectionnez l'entonnoir : ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Android_Table_Filtre.gif)


- Dans la zone de saisie qui s'affiche en haut du champ Table, saisissez le filtre voulu. ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Android_TableSaisieFiltre.gif)


- Seules les données du champ Table correspondant au filtre saisi sont affichées. 


Attention : Le clic sur la croix présente dans la zone de saisie annule le filtre. 

Remarque : Filtres spécifiques sur les colonnes numériques
Dans les colonnes numériques, il est possible de réaliser des filtres spécifiques : 

- Tapez avec le doigt sur l'entête de la colonne numérique pour afficher le menu contextuel. 

- Sélectionnez le filtre à réaliser : ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Android_Table_filtrenum.gif)


	- égal.

- différent.

- inférieur.

- supérieur.

- Dans la zone de saisie qui s'affiche en haut du champ Table, saisissez la valeur de comparaison.

- Seules les données du champ Table correspondant au filtre saisi sont affichées. 



<a name="NOTE3_5"></a>


### Redimensionnement de colonnes
<a name="redimensionnement_colonnes_ELTPARAGRAPHE000166"></a>Pour redimensionner une colonne : 

- Tapez avec le doigt sur l'extrémité de l'entête de la colonne à redimensionner. 

- Maintenez le doigt appuyé et redimensionnez la colonne. 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Android_Table_RedimCol.gif&type=thumb)


- Relâchez le doigt lorsque la taille voulue est atteinte.




## Choisir le type d'image : dynamique, statique, générée, ... 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000189"></a>
Les images sont souvent utilisées pour illustrer les pages des sites.

**Plusieurs types d'images sont disponibles en WEBDEV pour une page dynamique**. Chaque type correspond à une utilisation spécifique de l'image.

Le choix du type de l'image est réalisé dans l'onglet "Général" de la description de l'image.

Quatre types d'images sont proposés :

- Image statique.

- Image dynamique.

- Image générée.

- Depuis une base de données.


Pour utiliser les images générées et stockées dans une base de données sous Linux, il est nécessaire d'installer la librairie QT (Core et GUI), version 5 minimale.

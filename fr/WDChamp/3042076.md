## Champ Graphe dans une fenêtre
<a name="champ_graphe_dans_une_fenetre_ELTTEXTE000299"></a>
Les caractéristiques du champ Graphe sont saisies directement dans la fenêtre de description du champ Graphe :

- Type de graphe (onglet "Général").

- Caractéristiques de l'axe des X et des Y (onglet "Général").

- Affichage du quadrillage (onglet "Général").

- Apparence et zoom (onglet "Détail").

- Affichage de la mire (onglet "Détail"). Pour plus de détails, consultez [Mire dans un graphe](../WDChamp/3042078.md). 

- Titre et légende du graphe (onglet "Détail"). Remarque : L'éditeur de fenêtres permet également de positionner directement sous l'éditeur les différents éléments du graphe. Pour plus de détails, consultez [Manipuler un graphe sous l'éditeur](../WDChamp/1013292.md).

- Image de fond (onglet "Détail").

- Animation 3D (onglet "Détail").

- Animation 2D (onglet "Détail"), uniquement pour la majorité des graphes (Aire, Bulle, Courbe, Donut, Hémicycle, Histogramme, MinMax, Radar, Secteur, Waterfall). Pour plus de détails, consultez [Animation d'un champ Graphe](../WDChamp/1013296.md).

- Caractéristiques des séries (onglet "Séries"). Pour plus de détails, consultez [Mode d'alimentation des champs Graphe](../WDChamp/1013009.md).

- Apparence des différents éléments du graphe : libellé, couleur des axes, couleur du quadrillage, ... (onglet "Style"). 




Pour plus de détails sur les différentes options présentes dans les différents onglets, consultez :

- [Champ Graphe : Onglet Séries](../WDChamp/3042081.md)

## Champ Graphe dans un état
<a name="champ_graphe_dans_etat_ELTTEXTE000353"></a>


### Champ Graphe dans un état
<a name="champ_graphe_dans_etat_ELTPARAGRAPHE000185"></a>Un graphe imprimé dans un état peut représenter :

- l'ensemble des données affichées dans l'état : l'état ne contient qu'un seul graphe.

- les données par rupture : l'état contient autant de graphes que de ruptures.

- les sommes globales des différentes ruptures : l'état ne contient qu'un seul graphe.

- des données différentes de celles affichées dans l'état. Ces données peuvent être définies par programmation, provenir d'un tableau WLangage, être définies dans la fenêtre de description du champ, ...




**A partir de la version 17**, les champs Graphe ont évolué et se rapprochent des champs disponibles dans les fenêtres et les pages. Pour prendre en compte cette évolution, sous l'éditeur d'états, sélectionnez l'option "Mettre à niveau le champ Graphe" du menu contextuel du champ.


Il est également possible d'imprimer uniquement un graphe (sans imprimer les données sources). Dans ce cas, le bloc contenant les données n'est pas affiché (option "Visible" décochée dans l'onglet "Détail" de la description du bloc).

Les caractéristiques du champ Graphe sont saisies directement dans la fenêtre de description du champ Graphe :

- Type de graphe (onglet "Général")

- Caractéristiques de l'axe des X et des Y (onglet "Général")

- Affichage du quadrillage (onglet "Général")

- Apparence (onglet "Détail")

- Titre et légende du graphe (onglet "Détail")

- Image de fond (onglet "Détail")

- Caractéristiques des séries (onglet "Séries"). Pour plus de détails, consultez [Mode d'alimentation des champs Graphe](../WDChamp/1013009.md).




Pour plus de détails sur les différentes options présentes dans les différents onglets, consultez :

## Présentation
<a name="presentation_ELTTEXTE000209"></a>


### Champ d'une fenêtre WINDEV et WINDEV Mobile
<a name="champ_une_fenetre_windev_windev_mobile_ELTPARAGRAPHE000011"></a>Le champ Multimédia permet de jouer des fichiers multimédia.

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) **Sur un PC**, il est nécessaire d'avoir les éléments suivants installés : Windows Media Player, une version récente de DirectX ainsi que les codecs nécessaires à la lecture des fichiers vidéo. Si les codecs nécessaires ne sont pas installés, la lecture des vidéos ne pourra pas être effectuée.





<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Créer un champ Multimédia
<a name="creer_champ_multimedia_ELTTEXTE000233"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Pour créer un champ Multimédia dans une fenêtre : 

1. Sous le volet "Création", dans le groupe "Champs graphiques" : 

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) déroulez "Vidéo et capture" et sélectionnez "Multimédia".




2. Cliquez dans la fenêtre à la position où le champ doit être créé. Le champ apparaît en création.




Pour afficher les caractéristiques du champ, sélectionnez l'option "Description" dans le menu contextuel du champ.

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Manipuler un champ Multimédia
<a name="manipuler_champ_multimedia_ELTTEXTE000257"></a>
Un champ Multimédia peut être manipulé par programmation grâce :

- aux [fonctions Multimédia du WLangage](../WDLang1/1000015005.md). Ces fonctions permettent par exemple d'associer un ficher multimédia au champ et de le jouer.

- aux [propriétés du WLangage](../WDChamp/9500201.md). La propriété [Valeur](../Proprietes/2510130.md) permet notamment d'associer un fichier multimédia au champ et de le jouer.





### Remarques
<a name="remarques_ELTPARAGRAPHE000136"></a>

- Il n'est pas possible d'utiliser le champ Multimédia dans un champ Zone répétée.

- Il n'est pas possible de capturer l'image d'une vidéo jouée dans un champ Multimédia.


## Les différents types de sites
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000107"></a>
WEBDEV permet de créer des sites dynamiques et / ou des sites statiques. La grande différence entre ces deux types de sites est la suivante : 
les sites dynamiques gèrent des données de manière interactive alors que les sites statiques ne gèrent que des données fixes.

Un site dynamique peut très bien avoir une partie statique (par exemple la présentation de la société).



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Différences principales entre un site statique et un site dynamique
<a name="differences_principales_entre_site_statique_site_dynamique_ELTTEXTE000131"></a>


| Site statique | Site dynamique |
| --- | --- |
| Le contenu des pages du site est fixe, déterminé une fois pour toutes.<br><br>Un site statique ne peut pas interagir avec des données. | Les données affichées dans les pages varient. En général, les pages permettent de :<br><br>- réaliser des traitements et/ou des calculs par programmation.<br><br>- afficher des données stockées dans une base de données.<br><br>- afficher des images et du texte interactif.<br><br><br><br><br>WEBDEV propose plusieurs types de sites dynamiques :<br><br>- Site WEBDEV en mode Session.<br><br>- Site WEBDEV en mode AWP.<br><br>- Site PHP.<br><br><br> |
| **Site statique WEBDEV** : le Serveur d'Application WEBDEV n'est pas nécessaire. | **Site dynamique WEBDEV** : le Serveur d'Application WEBDEV ou le langage PHP est nécessaire. |

## Quelques exemples
<a name="quelques_exemples_ELTTEXTE000175"></a>


| Site statique | Site dynamique |
| --- | --- |
| Site de présentation d'une société, d'une région | Catalogue produit avec gestion en temps réel des commandes |
| Site d'une association | Site de gestion des réservations de voyages |
| Site d'un journal (quotidien) | Site de gestion des stock d'une entreprise (Intranet) |

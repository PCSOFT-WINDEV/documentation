
## Manipulations d'un champ Graphe dans un site WEBDEV
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000144"></a>
Lors de l'exécution d'un champ Graphe dans une page d'un site WEBDEV, l'internaute dispose d'une barre d'outils permettant de modifier la présentation du graphe affiché.

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Graphe_BarreWeb_20.gif)




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Fonctionnalités de la barre d'outils
<a name="fonctionnalites_barre_outils_ELTTEXTE000168"></a>
La barre d'outils d'un champ Graphe permet de choisir le type de graphe à afficher. Il est possible de :

- afficher le graphe sous forme d'un graphe en secteur : 

	- Secteur, 

	- Hémicycle,

	- Donut, 

	- Entonnoir.




- afficher le graphe sous forme d'histogramme : 

	- Histogramme groupé, empilé, horizontal groupé, horizontal empilé, 

	- Waterfall.




- afficher le graphe sous forme de : 

	- courbe, 

	- nuage de points, 

	- aire, 

	- radar, 

	- bulles.




- afficher le graphe sous forme de graphe boursier : 

	- chandelier japonais, 

	- BarCharts, 

	- Minimum Maximum. 







Cette barre d'outils permet également :

- d'afficher ou non le quadrillage horizontal ou vertical.

- d'afficher ou non la légende et de gérer sa position. 

- de gérer les effets de lissage, dégradé, relief, crénelage et transparence.

- d'afficher au choix une bulle en survol des valeurs, une mire ou une mire multiple.

- d'afficher ou non les animations. 

- d'activer ou non l'interactivité du graphe.

- d'enregistrer ou d'imprimer le graphe.

## Mettre en place la barre d'outils
<a name="mettre_place_barre_outils_ELTTEXTE000192"></a>
Par défaut, lors de la création d'un champ Graphe, la barre d'outils est affichée en mode automatique. 

Pour configurer l'affichage de la barre d'outils : 

1. Affichez la fenêtre de description du champ Graphe. 

2. Dans l'onglet "Détail", dans la zone "Apparence et barre d'outils", il est possible de sélectionner : 

	- le mode d'affichage de la barre d'outils : jamais, automatique ou toujours visible. 

	- les options affichées par la barre d'outils (bouton "..." placé à droite du mode d'affichage). 




3. Validez la fenêtre de description du champ. 




Remarque : si vous avez choisi une barre d'outils toujours visible, la barre d'outils apparaît dans la page en cours d'édition.

## Manipulation d'un champ Arbre par l'utilisateur
<a name="manipulation_champ_arbre_par_utilisateur_ELTTEXTE000079"></a>
Lorsque le champ Arbre est en saisie, l'utilisateur peut enrouler ou dérouler les différents niveaux de l'arborescence par un simple CLIC de souris sur les éléments du champ Arbre.

Pour dérouler un niveau, il suffit :

- soit de cliquer sur le signe "+" présent devant le nom de l'élément.

- soit de double-cliquer directement sur le noeud à dérouler.




Pour enrouler un niveau, il suffit :

- soit de cliquer sur le signe "-" présent devant le nom de l'élément.

- soit de double-cliquer directement sur le noeud à enrouler.

## Evénements associés
<a name="evenements_associes_ELTTEXTE000103"></a>
A chaque manipulation des éléments du champ Arbre par l'utilisateur, plusieurs événements sont exécutés. Pour plus de détails sur les événements liés à un champ Arbre, consultez [Evénements associés à un champ Arbre](../WDChamp/1013110.md).

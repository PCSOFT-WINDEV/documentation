
## Manipuler un champ Jauge par programmation
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000231"></a>
Un champ Jauge peut être manipulé directement par programmation. Il suffit d'utiliser directement la variable du champ Jauge dans le code.

La variable du champ Jauge :

- correspond au nom du champ Jauge.

- est initialisée avec la position de la jauge.

## Récupérer la position de la jauge
<a name="recuperer_position_jauge_ELTTEXTE000291"></a>
Pour récupérer la position d'une jauge, utilisez la syntaxe suivante : 

```txt
<Position> = <Champ Jauge>
```


&lt;Position&gt; est une variable de type entier. 

**Remarque** : Il est également possible d'utiliser la propriété [Valeur](../Proprietes/2510130.md).



<a name="NOTE4"></a>
<a name="NOTE4_1"></a>


## Modifier la position de la jauge
<a name="modifier_position_jauge_ELTTEXTE000315"></a>
Pour modifier la position d'une jauge, utilisez la syntaxe suivante : 

```txt
<Champ Jauge> = <Valeur>
```


&lt;Valeur&gt; est une valeur comprise entre la valeur minimale et la valeur maximale de la jauge. Ces valeurs ont été décrites dans la fenêtre de description du champ ou par programmation, grâce aux propriétés [BorneMin](../Proprietes/2510008.md) et [BorneMax](../Proprietes/2510009.md).

**Remarque** : Il est également possible d'utiliser la propriété [Valeur](../Proprietes/2510130.md).

## Propriétés spécifiques à la gestion d'un champ Jauge
<a name="proprietes_specifiques_gestion_champ_jauge_ELTTEXTE000339"></a>
Les propriétés suivantes sont spécifiques à la gestion des caractéristiques d'un champ Jauge par programmation.


|   |   |
| --- | --- |
| [BorneMax](../Proprietes/2510009.md) | Permet de connaître et de modifier la borne maximale d'un champ Jauge. |
| [BorneMin](../Proprietes/2510008.md) | Permet de connaître et de modifier la borne minimale d'un champ Jauge. |
| [CadrageHorizontal](../Proprietes/2510023.md) | Permet de connaître et de modifier le cadrage horizontal du pourcentage de progression affiché dans le champ Jauge. |
| [Image](../Proprietes/2510034.md) | Permet de connaître et de modifier l'image de progression d'un champ Jauge. |
| [ImageFond](../Proprietes/2510056.md) | Permet de connaître et de modifier l'image de fond d'un champ Jauge. |
| [TexteJauge](../Proprietes/1000017246.md) | Permet de connaître ou de modifier le texte affiché dans un champ Jauge (au lieu du pourcentage). |
| [Vertical](../Proprietes/2510133.md) | Permet de savoir si  le champ Jauge correspond à une jauge horizontale ou verticale. |


Pour connaître la liste complète des propriétés du WLangage pouvant être utilisées avec un champ de type Jauge, consultez [Propriétés associées au champ Jauge](../WDChamp/1013137.md).

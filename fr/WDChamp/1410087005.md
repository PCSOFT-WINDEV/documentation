
## Evénements associés à un champ Barre de navigation
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Evénement géré par défaut
<a name="evenement_gere_par_defaut_ELTTEXTE000072"></a>
Dans une page dynamique, WEBDEV gère par défaut l'événement suivant :


| Evénement | Condition d'exécution |
| --- | --- |
| Initialisation (Code serveur) | Exécuté à l'ouverture de la page. |


Dans une page statique, aucun code n'est associé au champ Barre de navigation.


## Multi-touch dans un champ Graphe
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000121"></a>
L'une des applications les plus courante du multi-touch est la manipulation d'images. La taille de l'affichage sur un téléphone étant souvent réduite, il est souvent nécessaire de zoomer et/ou de se déplacer dans une image. 

Les graphes sont un type spécifique d'image. 

Il est ainsi possible de réaliser un zoom sur un champ Graphe avec le contact de 2 doigts qui s'écartent. 

Le multi-touch dans un champ Graphe permet à l'utilisateur : 

- d'agrandir ou de diminuer le graphe affiché avec deux doigts. 

- d'afficher une bulle d'aide spécifique lors de l'appui du doigt sur une valeur du graphe. Cette bulle d'aide contient pas défaut la valeur correspondante. 

- d'exécuter un événement spécifique lors de l'appui sur une valeur du graphe. 




WINDEV Mobile permet la gestion du multi-touch dans les champs Graphe des applications grâce à : 

- [Différentes options spécifiques disponibles dans la description du champ](#NOTE2_1).




Remarque : La gestion du multi-touch est également disponible pour les champs Image. Pour plus de détails, consultez [Multi-touch dans un champ Image](../WDChamp/9500131.md).

## Description du champ Graphe : Options spécifiques au multi-touch
<a name="description_champ_graphe_options_specifiques_multitouch_ELTTEXTE000145"></a>
L'onglet "Détail" de la fenêtre de description du champ Graphe permet de gérer les actions effectuées lors de l'utilisation des doigts.  

Il est possible de gérer le zoom grâce aux options suivantes : 

- Autoriser le zoom horizontal. 

- Autoriser le zoom vertical. 




Dans ce cas : 

- l'écartement de deux doigts en contact avec l'écran permet de zoomer dans le graphe. Le zoom sera centré sur le point situé au milieu de la ligne tracée par les deux points de contact.

- le resserrement de deux doigts en contact avec l'écran permet de dé-zoomer dans le graphe. 




**Remarque** : Si la fenêtre possède un ascenseur, il n'est pas possible d'utiliser les fonctionnalités de multi-touch du champ Image.

## Présentation du Champ Barre d'outils
<a name="presentation_champ_barre_outils_ELTTEXTE000119"></a>
Le champ Barre d'outils est un type de champ avancé, lié à une fenêtre. Une barre d'outils contient plusieurs champs (bouton, champ de saisie, combo, ...) permettant à l'utilisateur de lancer rapidement certaines fonctionnalités de son application.

Une barre d'outils peut être :

- **Ancrée** : la barre d'outils est "collée" à un des bords de la fenêtre. 
	<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=champ%20barre%20outils%20-%20HC%20N%B0002.gif)


- **Flottante** : la barre d'outils est une fenêtre indépendante.
	<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=champ%20barre%20outils%20-%20HC%20N%B0001.gif)





L'utilisateur peut changer l'état de la barre d'outils en utilisant la souris.

## Créer un champ Barre d'outils
<a name="creer_champ_barre_outils_ELTTEXTE000143"></a>


### Pour créer un champ de type Barre d'outils :
<a name="pour_creer_champ_type_barre_outils_ELTPARAGRAPHE000029"></a>

1. Sous le volet "Fenêtre", dans le groupe "Barres et menus", déroulez "Autres barres" et sélectionnez "Barre d'outils".

2. Cliquez dans la fenêtre à la position où le champ doit être créé. Le champ apparaît en création.




Pour afficher les caractéristiques du champ, sélectionnez l'option "Description" dans le menu contextuel du champ. Il est possible de configurer dans l'onglet "Détail" les différentes positions autorisées de la barre d'outils (ancrée ou flottante).

**Attention : Ne placez pas de champs à côté d'une barre d'outils** : Si des champs sont positionnés à côté de la barre d'outils en création (à gauche d'une barre horizontale ou en dessous d'une barre verticale), ces champs seront automatiquement déplacés (et risquent de disparaître) lorsque l'utilisateur modifiera la position de la barre d'outils. ### Associer des champs à la barre d'outils
<a name="associer_des_champs_barre_outils_ELTPARAGRAPHE000053"></a>**Pour associer des champs existants à la barre d'outils** :
**Solution 1** : Réalisez un "Drag and Drop" du champ vers le champ "Barre d'outils".
**Solution 2** : Dans la description du champ (onglet "Général"), cliquez sur le bouton "Ajouter" et sélectionnez le champ à ajouter à la barre d'outils. Le champ sélectionné est automatiquement créé dans la barre d'outils.

**Pour associer des nouveaux champs à la barre d'outils** :
**Solution 1** : Réalisez un "Drag and Drop" du champ vers le champ "Barre d'outils".
**Solution 2** : Dans la description du champ (onglet "Général"), cliquez sur le bouton "Ajouter" et sélectionnez le type de champ à ajouter à la barre d'outils. Le champ sélectionné est automatiquement créé dans la barre d'outils. Pour le modifier, utilisez le bouton "Editer" : la description du nouveau champ apparaît.

**Remarque** : L'espace entre les champs de la barre d'outils peut être configuré dans [l'onglet "Détail" de la description de la barre d'outils](../WDChamp/1013119.md).

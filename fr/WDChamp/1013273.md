
## Menu contextuel des champs Arbre (FAA)
			



## Présentation
<a name="presentation_ELTTEXTE000123"></a>
Chaque champ Arbre présent dans une fenêtre propose par défaut un menu contextuel à l'utilisateur. ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=FAA_Arbre.gif)


Ce menu, disponible sans programmation, permet des actions utiles :

- copier (l'élément, la ligne, la colonne, tout). La copie de la colonne permet de copier l'intégralité du contenu des différentes branches de l'arbre dans le presse-papiers. Les différents éléments sont séparés par des Retours Chariot. 

- déplier ou replier tout : Cette option permet de déplier ou replier tous les éléments de la branche en cours. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) exporter vers Word, Excel, XML : Cette option permet d'exporter le contenu du champ Arbre dans un fichier au format sélectionné. 




**Remarques** : 

- Le même menu contextuel est disponible pour la partie arbre d'une table hiérarchique. 

- Il est possible d'ajouter des choix à ce menu par programmation. Il suffit d'utiliser la propriété [MenuContextuel](../Proprietes/2510077.md).

## Paramétrage
<a name="parametrage_ELTTEXTE000147"></a>


### Paramétrage du menu automatique dans un champ Arbre
<a name="parametrage_menu_automatique_dans_champ_arbre_ELTPARAGRAPHE000034"></a>Le menu contextuel du champ Arbre peut bien évidemment être désactivé. Pour paramétrer l'affichage du menu contextuel du champ Arbre, vous devez :

1. Afficher la fenêtre de description du champ Arbre.

2. Sélectionner l'onglet "UI".

3. Choisir le menu désiré dans la combo "Menu contextuel". Il est possible de : 

	- "Afficher le menu des FAA (Système)" : Dans ce cas, le menu contextuel affiché sera le menu automatique du champ Arbre. Pour plus de détails sur la configuration de ce menu, consultez [Configurer le menu des FAA](../Editeurs/2010040.md).

	- "Ajouter un menu contextuel" : Si cette option est cochée, vous pouvez sélectionner le menu contextuel personnalisé à afficher. 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=FAA_Menu_Agenda.gif)








**Remarques** : 

- Si les deux options sont cochées, le menu contextuel personnalisé peut être ajouté avant ou après le menu des FAA. 

- Si aucune option n'est sélectionnée, aucun menu contextuel ne sera sélectionné. L'option &lt;Désactivé&gt; sera affichée dans la fenêtre de description. 

- La propriété [MenuContextuel](../Proprietes/2510077.md) permet également de supprimer le menu contextuel spécifique aux FAA et de retrouver le menu Windows classique pour un champ spécifique.

- Par programmation, la fonction [FAADésactive](../WDLang1/1000022018.md) permet de supprimer certaines options du menu contextuel des FAA spécifiques aux champs Arbre. Par exemple, les constantes *faaTableVersExcel*, *faaTableVersWord* et *faaTableVersXML* permettent de désactiver les fonctionnalités d'export.


## Associer un identifiant à un élément d'un champ Arbre
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Identifiant d'un élément
<a name="identifiant_element_ELTTEXTE000093"></a>
WINDEV, WINDEV Mobile et WEBDEV donnent la possibilité d'associer un identifiant à chaque élément d'un champ Arbre. Cet identifiant peut être une chaîne de caractères ou un entier.

Cet identifiant peut être utilisé pour mémoriser une information spécifique (numéro de l'enregistrement correspondant à l'élément sélectionné dans le champ Arbre par exemple).

**Remarque** : Il est également possible d'associer une image à un élément. Pour plus de détails, consultez [Associer une image aux éléments](../WDChamp/1013034.md). 



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Associer un identifiant à un élément
<a name="associer_identifiant_element_ELTTEXTE000117"></a>
Pour associer un identifiant à un élément d'un champ Arbre, il suffit de préciser cet identifiant dans les fonctions de programmation suivantes :



|   |   |
| --- | --- |
| [ArbreAjoute](../WDLang1/3018012.md) | Ajoute un élément "feuille" (et tous les noeuds nécessaires) dans un champ Arbre. |
| [ArbreInsère](../WDLang1/3018023.md) | Insère une feuille à une position spécifique dans un champ Arbre. |
| [ArbreModifie](../WDLang1/3018013.md) | Modifie un élément (noeud ou feuille) dans un champ Arbre. |





Ces fonctions sont également disponibles en syntaxe préfixée : 



|   |   |
| --- | --- |
| [&lt;Arbre&gt;.Ajoute](../WDLang1/1000023422.md) | Ajoute un élément "feuille" (et tous les noeuds nécessaires) dans un champ Arbre. |
| [&lt;Arbre&gt;.Insère](../WDLang1/1000023761.md) | Insère une feuille à une position spécifique dans un champ Arbre. |
| [&lt;Arbre&gt;.Modifie](../WDLang1/1000023763.md) | Modifie un élément (noeud ou feuille) dans un champ Arbre. |







<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Lire l'identifiant d'un élément
<a name="lire_identifiant_element_ELTTEXTE000181"></a>
Pour connaître l'identifiant d'un élément, il suffit d'utiliser la fonction suivante :



|   |   |
| --- | --- |
| [ArbreRécupèreIdentifiant](../WDLang1/3018004.md) | Renvoie l'identifiant associé à un élément (noeud ou feuille) dans un champ Arbre. |





Cette fonction est également disponible en syntaxe préfixée : 



|   |   |
| --- | --- |
| [&lt;Arbre&gt;.RécupèreIdentifiant](../WDLang1/1000023766.md) | Renvoie l'identifiant associé à un élément (noeud ou feuille) dans un champ Arbre. |


## Manipuler les options de menu WEBDEV
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000237"></a>
Le menu contextuel des options de menu permet de réaliser toutes les manipulations nécessaires pour définir : 

- un menu déroulant.




Pour manipuler les options d'un menu déroulant : 

- Sélectionnez le menu. 

- Cliquez sur le menu sélectionné : un cadre jaune apparaît. 

- Le menu est en édition. Le menu contextuel des options de menu permet d'ajouter directement les options dans le menu en édition.


![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Menu_creation_Option.gif)


Pour manipuler les options d'un menu contextuel : 

- Affichez le champ Menu contextuel par une des méthodes suivantes : 

	- double-cliquez sur son nom dans le volet "Explorateur de projet". 

	- sous le volet "Page", dans le groupe "Edition", déroulez "Menus contextuels" et sélectionnez "&lt;NomMenuContextuel&gt;".




- Sélectionnez une option. Le menu contextuel des options de menu permet d'ajouter directement les options dans le menu en édition.




**Remarques** : 

- Pour ne plus être dans le menu en édition, utilisez la touche Echap (le cadre jaune doit disparaître).

- Pour éditer le libellé d'une option de menu (menu déroulant ou menu contextuel), il suffit de sélectionner cette option et d'appuyer sur la touche Espace (ou de sélectionner l'option "Editer le libellé de l'option" du menu contextuel) : un cadre jaune apparaît.

- Pour associer une action à une option de menu, vous pouvez utiliser la fenêtre de description de l'option de menu, ou directement le code associé à l'option. Pour plus de détails, consultez [Programmer un menu déroulant](../WDChamp/1010010.md).

- Pour créer un menu déroulant popup, il suffit d'associer l'option de menu à une popup présente dans la page. Pour plus de détails, consultez [Options de menu de type Popup](#NOTE3_1). 

- Il est possible d'ajouter par programmation des options à un menu existant (menu contextuel ou menu déroulant) grâce aux fonctions suivantes : 
	


|   |   |
| --- | --- |
| [MenuAjouteMenu](../WDLang1/1000017266.md) | Ajoute un nouveau menu dans un menu existant (déroulant ou contextuel) d'une fenêtre ou d'une page. |
| [MenuAjouteOption](../WDLang1/1000017262.md) | Ajoute une nouvelle option de menu à la suite de la dernière option d'un menu (déroulant ou contextuel). |
| [MenuAjouteOptionURL](../WDLang1/1000018529.md) | Ajoute une nouvelle option de menu à la suite de la dernière option d'un menu (déroulant ou contextuel). Cette option de menu permet d'afficher la page correspondant à l'URL indiquée. |
| [MenuAjoutePopup](../WDLang1/1000020583.md) | Transforme une option de menu d'une page pour que cette option ouvre une popup. |
| [MenuAjouteSéparateur](../WDLang1/1000017270.md) | Ajoute un nouveau séparateur dans un menu (menu déroulant ou contextuel). |

## Les différentes manipulations possibles
<a name="les_differentes_manipulations_possibles_ELTTEXTE000298"></a>


### Changer le libellé d'une option de menu
<a name="changer_libelle_une_option_menu_ELTPARAGRAPHE000073"></a>Pour changer le libellé d'une option de menu :

1. Sélectionnez l'option.

2. Appuyez sur la touche Espace.

3. Saisissez le nouveau libellé et validez par la touche Entrée.





<a name="NOTE2_2"></a>


### Ajouter une option de menu
<a name="ajouter_une_option_menu_ELTPARAGRAPHE000085"></a>Pour ajouter une option de menu :

1. Pour ajouter une option de menu après l'option de menu sélectionnée, affichez le menu contextuel de l'option de menu en cours (clic droit) et sélectionner l'option "Ajouter après".

2. Appuyez sur la touche Entrée.

3. Saisissez le libellé de la nouvelle option et validez par la touche Entrée.


Remarque : La touche Echap permet d'annuler l'ajout d'une option de menu.


<a name="NOTE2_3"></a>


### Ajouter un séparateur
<a name="ajouter_separateur_ELTPARAGRAPHE000099"></a>Pour ajouter un séparateur :

1. Sélectionnez l'option de sous-menu après laquelle le séparateur doit être ajouté.

2. Affichez le menu contextuel de l'option de menu en cours (clic droit).

3. Sélectionnez l'option "Ajouter un séparateur".





<a name="NOTE2_4"></a>


### Ajouter un sous-menu
<a name="ajouter_sousmenu_ELTPARAGRAPHE000111"></a>Pour ajouter un sous-menu :

1. Sélectionnez l'option de menu à laquelle une sous-option doit être ajoutée.

2. Affichez le menu contextuel de l'option de menu en cours (clic droit).

3. Sélectionnez l'option "Insérer un sous-menu".

4. Appuyez sur la touche Espace.

5. Saisissez le libellé de la première option du sous-menu créé.

6. Appuyez sur la touche Entrée pour valider l'ajout.




Remarques : 

- L'ajout de sous-options dans un sous-menu se fait de façon identique à l'ajout d'options dans un menu.





<a name="NOTE2_5"></a>


### Supprimer une option de menu
<a name="supprimer_une_option_menu_ELTPARAGRAPHE000140"></a>Pour supprimer une option de menu :

1. Sélectionnez l'option de menu à supprimer.

2. Affichez le menu contextuel de l'option de menu en cours (clic droit).

3. Sélectionnez l'option "Supprimer".


Remarque : Il est également possible de supprimer l'option de menu en cours en appuyant sur la touche Suppr.

## Paramétrer les options d'un menu Popup (menus déroulants uniquement)
<a name="parametrer_les_options_menu_popup_menus_deroulants_uniquement_ELTTEXTE000346"></a>


### Présentation
<a name="presentation_ELTPARAGRAPHE000157"></a>Dans un menu "Popup", le survol d'une option de la barre de menu permet d'ouvrir une page Popup totalement personnalisable. 

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Menu_popup_WB.gif)


La page popup permet de réaliser simplement des menus graphiques. Il est possible d'insérer n'importe que type de champ dans la popup. 
<a name="NOTE3_2"></a>


### Créer les options d'un menu popup
<a name="creer_les_options_menu_popup_ELTPARAGRAPHE000168"></a>Pour créer les options d'un menu popup : 

1. Affichez le menu en mode édition : le menu doit être encadré d'un trait jaune. 

2. Sélectionnez l'option de menu devant afficher la page popup. 

3. Affichez le menu contextuel de l'option (clic droit) et sélectionnez l'option "Dérouler une page popup". 

4. L'onglet "UI" de la fenêtre de description de l'option de menu est affichée. Dans cette fenêtre : 

	- Sélectionnez la page popup à afficher. Si la page popup n'existe pas, cliquez sur "&lt;Nouvelle popup&gt;" pour la créer. 

	- Sélectionnez la position de la popup par rapport à l'option (en dessous, à droite, à gauche, ...). 




5. Validez la fenêtre de description. Si vous avez choisi une nouvelle popup, la popup apparaît en création sous l'éditeur. 

6. Il est possible de visualiser la popup associée à l'option directement sous l'éditeur : 

	- affichez si nécessaire le menu en mode édition (avec le cadre jaune). 

	- cliquez sur l'option de menu associée à la popup. 





Remarque : Il est également possible de transformer une option de menu existante en option ouvrant une popup : 

1. Affichez le menu en mode édition : le menu doit être encadré d'un trait jaune. 

2. Affichez le menu contextuel de l'option (clic droit) et sélectionnez l'option "Description de l'option".

3. Dans l'onglet "UI", sélectionnez la fenêtre popup à associer à l'option.

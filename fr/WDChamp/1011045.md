


## Bloc Entête de page (Haut de page)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000139"></a>
Le bloc **Entête de page** est un bloc d'un état créé sous l'éditeur d'états. Le bloc **Entête de page** est imprimé à chaque nouvelle page. Ce bloc est exécuté après le bloc *Début de document* s'il existe.

Le bloc **Entête de page** contient des informations qui doivent figurer en haut de chaque page, par exemple le titre ou un champ prédéfini indiquant le numéro de page.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Création du bloc Entête de page
<a name="creation_bloc_entete_page_ELTTEXTE000163"></a>
Pour créer un bloc Entête de page :

1. Effectuez une des manipulations suivantes sous l'éditeur d'états :

	- sous le volet "Création", dans le groupe "Structure", déroulez "Blocs".

	- sélectionnez "Ajouter un bloc" du menu contextuel de l'état (clic droit sur l'état en cours de création).

	- cliquez sur le bouton "Ajouter un bloc" dans l'onglet "Blocs" de la description de l'état.




2. Sélectionnez le type "Entête de page".




<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Impression du bloc Entête de page
<a name="impression_bloc_entete_page_ELTTEXTE000187"></a>


### Imprimer le bloc En-tête de page avant ou après le bloc Début de document
<a name="imprimer_bloc_entete_page_avant_apres_bloc_debut_document_ELTPARAGRAPHE000054"></a>Par défaut, le bloc **En-tête de page** est imprimé après le bloc *Début de document* sur la première page de l'état. L'éditeur d'états permet de modifier cet ordre d'impression par défaut.
Pour imprimer le bloc En-tête de page avant le bloc Début de document sur la première page de l'état :

- Méthode 1 : 

	1. Affichez la fenêtre de description du bloc "Début de document" (option "Description du bloc" du menu contextuel).

	2. Dans l'onglet "Général", sélectionnez l'option "Imprimer le bloc "Début de document" APRES le bloc "Entête de page" sur la première page".




- Méthode 2 : 

	1. Affichez la fenêtre de description de l'état (option "Description de l'état" du menu contextuel).

	2. Dans l'onglet "Blocs", dans la section "Début de document", cochez l'option "APRES le bloc "Entête de page"".






<a name="NOTE3_2"></a>


### Visibilité du bloc En-tête de page
<a name="visibilite_bloc_entete_page_ELTPARAGRAPHE000080"></a>Par défaut, le bloc **En-tête de page** est visible sur toutes les pages de l'état. L'éditeur d'états permet de spécifier les pages de l'état sur lesquelles le bloc *En-tête de page* doit être affiché.

Pour spécifier les pages sur lesquelles le bloc En-tête de page doit être affiché :

1. Affichez l'onglet "UI" de la fenêtre description du bloc (option "Description du bloc" du menu contextuel).

2. Spécifiez la visibilité du bloc :

	- **Toujours visible** : Le bloc et ses champs seront affichés sur toutes les pages de l'état.

	- **Toujours invisible** : Le bloc et ses champs ne seront pas affichés dans l'état.

	- **Visible uniquement sur la première page** : Le bloc et ses champs seront affichés uniquement sur la première page de l'état.

	- **Invisible uniquement sur la première page** : Le bloc et ses champs seront affichés sur toutes les pages de l'état sauf sur la première page.

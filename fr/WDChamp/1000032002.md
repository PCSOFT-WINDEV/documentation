
## Evénements associés à un champ Zone de Texte Riche

<a name="NOTE1_1"></a>


## Evénements gérés par défaut
<a name="evenements_geres_par_defaut_ELTTEXTE000096"></a>
WEBDEV gère par défaut les événements suivants (ordre d'apparition sous l'éditeur de code) :

| **Evénement** | **Condition d'exécution** |
| --- | --- |
| Initialisation (code serveur) | Exécuté à l'ouverture de la page. \* |
| Clic (code navigateur) | Exécuté lors du clic dans le champ Zone de Texte Riche. |
| Souris déplacée (code navigateur) | Exécuté lors du déplacement de la souris au-dessus du champ Zone de Texte Riche. |


(\*) Par défaut, l'événement "Initialisation" de chaque champ est exécuté selon l'ordre de création des différents champs dans la page. Pour modifier cet ordre d'exécution : 

1. Sous le volet "Page", dans le groupe "Edition", déroulez "Navigation" et sélectionnez "Editer". 

2. Dans la fenêtre qui s'affiche, utilisez les flèches situées sur la droite pour modifier l'ordre d'initialisation des champs.




Remarque : Les champs contenus dans le champ Zone de Texte Riche ont leurs propres événements.

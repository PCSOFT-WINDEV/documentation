
## Le champ de saisie RTF : Barre de mise en forme automatique
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000132"></a>
Dans les champs de saisie de type RTF, une barre de mise en forme peut apparaître automatiquement dès l'entrée en saisie dans le champ. La gestion des options de formatage devient ainsi automatique.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000156"></a>


### Activer la barre de mise en forme pour les champs RTF
<a name="activer_barre_mise_forme_pour_les_champs_rtf_ELTPARAGRAPHE000019"></a>Pour activer la barre de mise en forme pour les champs RTF :

1. Affichez la description d'un champ de saisie RTF.

2. Dans l'onglet "Général", indiquez si le champ utilise une barre d'outils RTF : <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=RTF_ConfigBarre.gif)


3. Validez.




**Attention** : Une seule barre d'outils RTF peut être affichée dans une fenêtre. Si plusieurs champs sont en mode "Toujours visible", le champ qui est en cours de saisie est prioritaire et affichera la barre d'outils RTF.
<a name="NOTE2_2"></a>


### Fonctionnalité disponible en FAA
<a name="fonctionnalite_disponible_faa_ELTPARAGRAPHE000033"></a>Dans un champ de saisie RTF, l'utilisateur final peut activer la barre de mise en forme grâce à l'option "Barre de mise en forme RTF" du menu contextuel du champ de saisie.


<a name="NOTE2_3"></a>


### Police utilisée par défaut dans la barre d'outils RTF
<a name="police_utilisee_par_defaut_dans_barre_outils_rtf_ELTPARAGRAPHE000041"></a>Lors de l'exécution, la barre d'outils RTF est initialisée avec la police, la taille et les attributs de police donnés dans la description du champ RTF. Il est possible de modifier cette police par programmation grâce [aux propriétés de gestion des polices](../Proprietes/2515045.md). 


<a name="NOTE2_4"></a>


### Désactiver la barre de mise en forme RTF
<a name="desactiver_barre_mise_forme_rtf_ELTPARAGRAPHE000052"></a>Pour ne pas afficher l'option "Barre de mise en forme RTF" dans le menu contextuel du champ de saisie, utilisez la fonction [FAADésactive](../WDLang1/1000022018.md) avec la constante *faaBarreRTF*.

**Attention** : Cette commande désactivera également la [barre de mise en forme des champs HTML](../WDChamp/9500113.md).

## Utilisation de la barre de mise en forme RTF
<a name="utilisation_barre_mise_forme_rtf_ELTTEXTE000198"></a>
La barre de mise en forme RTF est la suivante : <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=RTF_Barre.gif)
Les raccourcis clavier pouvant être utilisés pour mettre en forme le texte sont les suivants :

- Italique : Ctrl + I

- Gras : Ctrl + G

- Souligné : Ctrl + U ou Ctrl + _ (underscore, sous la touche 8)

- Barré : Ctrl + - (tiret, sous la touche 6)

- Surligné : Ctrl + H 




Lorsque le texte sélectionné correspond à une option de la barre RTF (texte souligné par exemple), le bouton correspondant dans la barre est enfoncé.

Le bouton "Couleur" ne reflète pas la couleur de la sélection : il correspond à la dernière valeur choisie par l'utilisateur.

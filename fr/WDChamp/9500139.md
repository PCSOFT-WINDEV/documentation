


## Colorer une ligne, une colonne, une cellule d'un champ (FAA)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000126"></a>
Pour faciliter la lecture dans les champs permettant de visualiser des volumes importants de données, l'utilisateur final a la possibilité de colorer les données contenues dans le champ. 

Les données affichées dans les champs suivants peuvent être colorées : 

- Champ Table : cellule, ligne, colonne 

- Champ Table hiérarchique : cellule, ligne, colonne 

- Champ Tableau Croisé Dynamique : cellule ou multisélection de cellules.  




**Remarque** : Cette option est disponible uniquement pour les champs "par programmation" ou les champs "fichier chargé en mémoire". Cette option n'est pas disponible par exemple pour les champs Table fichier à accès direct. 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Colorer un élément
<a name="colorer_element_ELTTEXTE000150"></a>
**Pour colorer une ligne, colonne, cellule, ...** : 

1. Sélectionnez l'élément à colorer dans le champ. 

2. Dans le menu contextuel de l'élément, sélectionnez l'option "Couleur de fond" puis sélectionnez la couleur voulue parmi la liste proposée. <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=FAA_Couleur_Table.gif)



L'option "Couleur par défaut" permet de retrouver la couleur initiale. 

**Remarque** : La couleur choisie est disponible uniquement durant l'affichage du champ. A la fermeture de la fenêtre, cette information ne sera pas conservée.

## Programmation
<a name="programmation_ELTTEXTE000174"></a>


### Programmer la couleur des cellules, lignes, ...
<a name="programmer_couleur_des_cellules_lignes_ELTPARAGRAPHE000041"></a>Les couleurs d'un champ Table (fond des lignes, des cellules) peuvent être modifiées par programmation grâce aux propriétés :

- [Couleur](../Proprietes/2510071.md) : pour modifier la couleur du texte.

- [CouleurFond](../Proprietes/2510022.md) : pour modifier la couleur de fond.


Pour plus de détails sur les syntaxes à utiliser, consultez [Gérer les couleurs dans les champs Table](../WDChamp/1013231.md).


### Désactiver la coloration des éléments d'un champ
<a name="desactiver_coloration_des_elements_champ_ELTPARAGRAPHE000058"></a>Pour ne pas autoriser la coloration des éléments d'un champ, il suffit d'utiliser la fonction [FAADésactive](../WDLang1/1000022018.md) avec la constante *faaChangeCouleurFond*. L'option de menu ne sera pas proposée. 

Par exemple : 

```wl
// Pas de changement de la couleur de fond possible
FAADésactive(TABLE_Client, faaChangeCouleurFond)
```



### Modifier le libellé de l'option "Couleur de fond" du menu contextuel
<a name="modifier_libelle_option_couleur_fond_menu_contextuel_ELTPARAGRAPHE000071"></a>Par défaut, l'option de menu permettant à l'utilisateur de modifier la couleur de fond a pour libellé "Couleur de fond". Pour modifier ce libellé, utilisez la fonction [FAAChangeLibellé](../WDLang1/1000022100.md) avec la constante *faaChangeCouleurFond*. 

Par exemple : 

```wl
// Pas de changement de la couleur de fond possible
FAAChangeLibellé(TABLE_Client, faaChangeCouleurFond, "Choisir une couleur de fond")
```

## Présentation
<a name="presentation_ELTTEXTE000176"></a>
WINDEV et WEBDEV permettent de réaliser automatiquement des impressions pour un certain nombre d'imprimantes PCL spécifiques.

**Si une de ces imprimantes est sélectionnée**, l'impression des éléments spécifiques (traits, images, couleurs, ...) seront gérés en fonction de cette imprimante.

**Si l'imprimante utilisée n'est pas présente dans cette liste**, il est possible de créer un fichier PCL générique, reconnu par la plupart des imprimantes PCL. Dans certains cas, l'impression des traits et des images pourra ne pas fonctionner correctement.

Vous pouvez également envoyer les commandes d'impression directement sur votre imprimante (développez votre propre driver). Dans ce cas, vous ne devez pas utiliser les fonctions d'impression du WLangage ou les états : pour imprimer, vous devrez utiliser les fonctions que vous aurez écrites.

Consultez la documentation de votre imprimante pour plus d'informations.

Exemple :

```wl
// Ici "Proc_Imprimante" est une collection de procédures qui contient 
// le code d'impression spécifique à une imprimante
// A vous d'écrire le code WLangage qui correspond à chacune de vos imprimantes
Proc_Imprimante.InitialiseConnexionIR()
Proc_Imprimante.ImprimeLigne("Titre de l'impression.") 
Proc_Imprimante.SauteLigne()
Proc_Imprimante.ImprimeMot("Une partie de phrase sans saut de ligne")
Proc_Imprimante.ImprimeMot(" pour que la suite s'imprime sur la même ligne.")
Proc_Imprimante.SautePage()
Proc_Imprimante.ImprimeMot("Page 2, et Fin")
Proc_Imprimante.TermineImpression()
```

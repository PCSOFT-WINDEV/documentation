


## Les graphes de type "Courbe"
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000181"></a>
Dans les graphes de type Courbe, les données sont interprétées comme des ordonnées successives de points. Les points de chaque série sont ensuite reliés entre eux par des traits.

Les graphes de type Courbe permettent de mettre en évidence les tendances des données.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Caractéristiques des courbes
<a name="caracteristiques_des_courbes_ELTTEXTE000205"></a><br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Genligne.gif&type=thumb)
Dans ce graphe :

**1. Titre du graphe**. 
Les caractéristiques du titre du graphe sont saisies dans l'onglet "Détail" de la description du champ (bouton "Editer le titre du graphe").
Il est possible de définir le titre du graphe, sa position et son style.

**2. Titre des axes**. 
Le titre des axes horizontal et vertical est affiché si le titre a été défini dans l'onglet "Général" de la description du champ Graphe. Ce paramètre peut être défini également par programmation grâce à la fonction [grTitreAxe](../WDLang3/3042057.md).

**3. Graduation de l'axe vertical**. 
Par défaut, la graduation est calculée automatiquement. L'onglet "Général" de la description du champ permet de définir :

- une graduation spécifique.

- une échelle des abscisses en fonction du temps. Cette option fonctionne uniquement avec des données de type Date ou DateHeure provenant d'un champ Table ou d'une rubrique HFSQL. Cette option ne peut pas être utilisée avec une série remplie par programmation. 

- une échelle des ordonnées logarithmique.




Il est également possible de demander à utiliser l'échelle secondaire pour une ou plusieurs séries (onglet "Séries").

**4. Légende**. 
Les caractéristiques de la légende sont saisies dans l'onglet "Détail" de la description du champ (bouton "Editer la légende du graphe").
Il est possible de définir sa position et son style.

**5. Etiquettes de séries**. 
Le libellé de la légende correspond au libellé des différentes séries affichées dans le graphe (définie dans l'onglet "Séries").

**6. Couleur de série**. 
Pour chaque série, il est possible de définir une couleur dans l'onglet "Séries" de la description du champ. Par défaut la couleur utilisée est automatique.

**7. Etiquette de catégorie**. 
Les étiquettes de catégorie peuvent être définies dans l'onglet "Séries" de la description du champ.

**8. Quadrillage**. 
Il est possible de définir le quadrillage horizontal et vertical dans l'onglet "Général" de la description du champ.

### Autres fonctionnalités
<a name="autres_fonctionnalites_ELTPARAGRAPHE000104"></a>Il est également possible **d'afficher les valeurs des données**. Ce paramètre est affiché si les deux conditions suivantes sont réunies :

- si le point de la série est dessiné (option "Type de point" dans l'onglet "Séries" de la fenêtre de description du champ).  

- si l'option "Afficher les valeurs" est cochée dans l'onglet "Détail" de la description du champ Graphe.


Une **image de fond** peut également être choisie pour le champ Graphe dans l'onglet "Détail" de la fenêtre de description du champ.

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Les graphes de type "Courbe" peuvent être affichés avec une animation. Pour animer ce type de graphe, il suffit de cocher l'option "Activer les animations" dans l'onglet "Détail" de la description du champ.

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Les graphes de type "Courbe" peuvent être affichés en temps réel. Pour plus de détails, consultez [Graphe avec affichage en temps réel](../WDChamp/1000021037.md).

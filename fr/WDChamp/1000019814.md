
## Propriétés associées au champ Carte
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000137"></a>


Un champ Carte peut être utilisé :

- dans une fenêtre,

- dans une page. 




Les propriétés associées au champ varient en fonction du lieu d'utilisation.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>

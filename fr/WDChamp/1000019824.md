
## Le champ Carte : Fonctionnalités Automatiques de vos Applications (FAA)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000083"></a>
Le champ Carte propose de nombreuses fonctionnalités accessibles directement à l'utilisateur final. Ces manipulations s'effectuent de manière intuitive à la souris comme sous le logiciel Google Maps.  

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Les fonctionnalités disponibles
<a name="les_fonctionnalites_disponibles_ELTTEXTE000107"></a>
Avec le champ Carte de WINDEV, l'utilisateur final peut :

- Se déplacer sur la carte. (1)
	Il suffit d'utiliser les boutons de déplacement (situés en haut à gauche du champ) ou la souris. 

- Zoomer et dé-zoomer. (2)
	Il suffit d'utiliser les boutons de zoom et l'ascenseur disponibles en haut à gauche du champ. 

- Passer en mode StreetView (vue depuis la chaussée) (3)
	Il suffit de déplacer le personnage vers la zone à afficher en mode StreetView. Pour fermer le mode StreetView, il suffit de cliquer sur la croix de fermeture en haut à droite du champ. 

- Consulter des Points d'Intérêts (POI) présents sur la carte. (4) 
	Lorsqu'un Point d'intérêt est présent sur la carte, il suffit de cliquer dessus pour afficher sa description. 
	Remarque : Lorsque la description d'un Point d'intérêt est affiché sur la carte, un lien permet de passer en mode StreetView. Pour fermer le mode StreetView, il suffit de cliquer sur la croix de fermeture en haut à droite du champ. 

- Afficher la carte en mode plan ou en mode satellite (5).




![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Champ_Carte_FAA.gif)


Il n'y a rien à programmer dans l'application pour que l'utilisateur final bénéficie de ces fonctionnalités.

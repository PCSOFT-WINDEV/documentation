


## Animation d'un champ Graphe
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000182"></a>
L'animation du champ Graphe permet de proposer un affichage dynamique dans vos applications. 

Par exemple, au premier affichage d'un graphe de type Histogramme, chaque barre semble jaillir de l'axe des abscisses. L'animation est également effectuée lorsqu'une valeur est modifiée. 

L'animation du champ Graphe est définie sous l'éditeur dans l'onglet ["Détail"](../WDChamp/3042080.md) de la fenêtre de description du champ. 


<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Mise en place
<a name="mise_place_ELTTEXTE000206"></a>


### Animer un champ Graphe
<a name="animer_champ_graphe_ELTPARAGRAPHE000042"></a>Pour animer un champ Graphe : 

1. Affichez la fenêtre de description du champ Graphe.

2. Dans l'onglet "Détail" : 

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Dans la zone "Animations des graphes 2D", cochez l'option "Activer les animations". 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Graphe_Animation_WD%20-%20HC%20N%B0001.gif)





3. Validez la fenêtre de description du champ. 




Remarques : 

- Cette option est disponible pour les types de graphes suivants : 

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Aire, Bulle, Courbe, Donut, Histogramme, MinMax, Radar, Secteur,  Waterfall. 




- L'animation effectuée sur les champs Graphe dépend du type de graphe affiché. Cette animation ne peut pas être paramétrée. 

- Par défaut, l'animation d'un champ Graphe dure 30 millisecondes. 

- Si un champ Graphe est animé, l'animation sera également effectuée lors du dessin du graphe en cas de changement d'une valeur (fonction [grDessine](../WDLang3/3042026.md)).  

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Par défaut, les champs Graphe créés à partir de la version 19 sont animés. 



<a name="NOTE2_2"></a>


### Modifier ou supprimer l'animation par programmation
<a name="modifier_supprimer_animation_par_programmation_ELTPARAGRAPHE000121"></a>La propriété [DuréeAnimation](../Proprietes/1000021069.md) permet de : 

- Connaître et modifier la durée de l'animation effectuée dans un champ Graphe. 
	
	```wl
	// Augmentation du délai
	GRF_MonGraphe.DuréeAnimation = 2000
	```


- Annuler l'animation d'un champ Graphe. Il suffit pour cela de changer la durée de l'animation en lui donnant une valeur nulle. 
	
	```wl
	// Arrête l'animation temporairement
	GRF_MonGraphe.DuréeAnimation = 0
	```

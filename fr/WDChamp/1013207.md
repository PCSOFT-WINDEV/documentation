


## Menu contextuel des champs Table et de leurs colonnes (FAA)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000184"></a>
Les applications WINDEV proposent automatiquement un menu contextuel sur les champs Table et Table hiérarchique ainsi que sur leurs colonnes. 

Ce menu contextuel, disponible sans programmation, présente de nombreuses fonctionnalités utiles pour l'utilisateur : ces fonctionnalités font partie des Fonctionnalités Automatiques de vos Applications (FAA). 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Les FAA disponibles dans le menu contextuel des champs Table et Table hiérarchique
<a name="les_faa_disponibles_dans_menu_contextuel_des_champs_table_table_hierarchique_ELTTEXTE000208"></a>
Le menu contextuel système (appelé également "Menu des FAA") associé par défaut aux champs Table et Tables hiérarchique propose les options suivantes : <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Champ_Table_MenuCtx.gif)


- **Copie** de la cellule, de la ligne ou du contenu du champ. Pour plus de détails, consultez [Manipulations des colonnes d'un champ Table en exécution (FAA)](../WDChamp/9500109.md). 

- **Mémoriser l'état des ruptures** (si le champ Table contient des ruptures). Pour plus de détails, consultez [Mémoriser les ruptures dans les champs Table (FAA)](../WDChamp/9500138.md). 

- **Affichage des informations sur plusieurs lignes** (pour les colonnes de type Texte). Pour plus de détails, consultez [Manipulations des colonnes d'un champ Table en exécution (FAA)](../WDChamp/9500109.md). 

- **Calculs automatiques** (somme, moyenne, ...) pour certains types de colonnes. Pour plus de détails, consultez [Calculs automatiques dans un champ Table (FAA)](../WDChamp/9500141.md). 

- **Export de la table** : 

	- vers Word par défaut ou vers OpenOffice Writer s'il est installé.

	- vers Excel par défaut ou vers OpenOffice Calc s'il est installé.

	- vers XML.


Pour plus de détails, consultez [Export de données (FAA)](../WDChamp/9500142.md). 

- **Graphique** : Création d'un graphique de représentation des données (sur les colonnes de type numériques). Pour plus de détails, consultez [Graphe sur une colonne de table (FAA)](../WDChamp/9500143.md). 

- **Sélection de plusieurs colonnes** du champ Table.
	Pour simplifier le menu, seules les colonnes ayant un titre sont proposées.
	Cette option permet à l'utilisateur d'afficher ou masquer les colonnes (pour éviter d'avoir un menu trop important, seules les 32 premières colonnes du champ Table sont affichées).

- **Impression** du contenu du champ Table (impression directe ou impression via le logiciel "Etats & Requêtes"). Pour plus de détails, consultez [Imprimer le contenu d'une table (FAA)](../WDChamp/9500132.md). 

- Définition de la **couleur de fond**. Pour plus de détails, consultez [Colorer une ligne, une colonne, une cellule d'un champ (FAA)](../WDChamp/9500139.md).




Grâce à ce menu, l'utilisateur gagne en autonomie, et peut réaliser des traitements qui n'étaient pas forcément prévus dans l'application

**Remarques** : 

- Les champs Table et Table hiérarchique proposent de nombreuses autres FAA.  

- Il est possible d'ajouter des choix à ce menu : 

	- depuis l'éditeur (voir ci-dessous). 

	- par programmation. Il suffit d'utiliser la propriété [MenuContextuel](../Proprietes/2510077.md).

## Paramétrage du menu contextuel automatique des champs Table et Table hiérarchique
<a name="parametrage_menu_contextuel_automatique_des_champs_table_table_hierarchique_ELTTEXTE000232"></a>


### Paramétrage du menu automatique dans un champ Table/Table hiérarchique
<a name="parametrage_menu_automatique_dans_champ_tabletable_hierarchique_ELTPARAGRAPHE000090"></a>Le menu contextuel du champ peut bien évidemment être désactivé. Pour paramétrer l'affichage du menu contextuel du champ :

1. Affichez la fenêtre de description du champ.

2. Sélectionnez l'onglet "UI".

3. Choisissez le menu désiré dans la combo "Menu contextuel" :<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Fonctionnalites_auto_Menu2.gif)





Vous pouvez choisir d'afficher : 

- le menu système (menu des FAA). Il est possible de paramétrer les options proposées avec le bouton "Paramètres". Ce menu peut être affiché ou non. 

- un menu contextuel personnalisé. Ce menu peut être affiché seul, ou en plus du menu système. Dans ce cas, il est possible d'indiquer si ce menu doit être affiché avant ou après le menu système. Pour plus de détails, consultez [Gestion des menus contextuels](../WDChamp/1010011.md).

- aucun menu. 




**Remarque** : La même configuration peut être effectuée au niveau des colonnes du champ Table. Par défaut, le menu contextuel du champ Table est associé à toutes les colonnes du champ.  
<a name="NOTE3_2"></a>


### Paramétrage de l'option de menu "Sélectionner les colonnes..."
<a name="parametrage_option_menu_selectionner_les_colonnes_ELTPARAGRAPHE000119"></a>

Dans certains cas, il peut être gênant de donner la possibilité à l'utilisateur de masquer des colonnes. Vous pouvez pour cela activer ou désactiver l'option de menu "Sélectionner les colonnes..." :

1. Affichez la fenêtre de description du champ.

2. Sélectionnez l'onglet "Détail".

3. Cochez ou décochez l'option "Menu 'Sélectionner les colonnes...'".<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Fonctionnalites_auto_Menu3.gif)

### Paramétrer certaines options par programmation : Fonctions de gestion des Fonctionnalités automatiques
<a name="parametrer_certaines_options_par_programmation_fonctions_gestion_des_fonctionnalites_automatiques_ELTPARAGRAPHE000133"></a>

Plusieurs fonctions WLangage permettent de gérer l'affichage ou non de certaines options du menu contextuel système du champ Table ou Table hiérarchique :



|   |   |
| --- | --- |
| [FAADésactive](../WDLang1/1000022018.md) | Désactive une Fonctionnalité Automatique d'une Application (FAA) sur un champ, sur une fenêtre ou sur l'application en cours. |
| [FAAExécute](../WDLang1/1000022099.md) | Exécute une Fonctionnalité Automatique d'une Application (FAA) sur un champ ou une fenêtre. Par exemple, il est possible de lancer la correction orthographique sur un champ de saisie. |


## Manipuler un champ Libellé par programmation
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000193"></a>
WINDEV, WINDEV Mobile et WEBDEV permettent de manipuler un champ Libellé directement par programmation. Il suffit d'utiliser directement la variable du champ Libellé dans le code.

La variable du champ Libellé correspond au nom du libellé. Un champ Libellé est toujours de type Texte.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Initialiser un champ Libellé
<a name="initialiser_champ_libelle_ELTTEXTE000217"></a>
Pour initialiser un champ Libellé, une simple affectation de la variable du champ Libellé suffit.

Par exemple : 


```wl
// Affectation d'un champ Libellé avec le nom et le prénom lus dans le fichier de données Client
LIB_MonLibellé = Client.Nom + "  " + Client.Prenom
```






<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Récupérer la valeur d'un champ Libellé
<a name="recuperer_valeur_champ_libelle_ELTTEXTE000241"></a>
Pour récupérer le contenu d'un champ Libellé :

- réalisez une simple affectation. Par exemple :
	
	```wl
	ContenuActuelLibellé = LIB_LIBELLE1
	```


- manipulez directement le champ Libellé. Par exemple :
	
	```wl
	Info(LIB_LIBELLE1)
	```


- utilisez la propriété [Valeur](../Proprietes/2510130.md) ou la propriété [Libellé](../Proprietes/2510053.md).

## Propriétés spécifiques à la gestion d'un champ Libellé
<a name="proprietes_specifiques_gestion_champ_libelle_ELTTEXTE000289"></a>
Les propriétés suivantes sont spécifiques à la gestion des caractéristiques d'un champ Libellé par programmation.


|   |   |
| --- | --- |
| [Animation](../Proprietes/2510006.md) | Permet de connaître l'état actuel de l'animation dans un champ Libellé et de lancer ou d'arrêter l'animation d'un champ Libellé animé. |
| [AnimationInitiale](../Proprietes/2510012.md) | Permet de connaître l'état initial de l'animation dans un champ Libellé. |
| [Ellipse](../Proprietes/2510051.md) | Permet de connaître et modifier le mode de gestion de l'ellipse dans un champ Libellé. |
| [Libellé](../Proprietes/2510053.md) | Permet de connaître et de modifier le libellé du champ. |
| [PoliceTailleMin](../Proprietes/1000024918.md) | Permet de connaître et de modifier la taille minimale de la police utilisée pour un champ Libellé avec l'option "Réduire la police". |
| [RetraitBas](../Proprietes/1000022891.md) | Permet de connaître et de modifier la marge basse d'un champ Libellé. |
| [RetraitDroit](../Proprietes/1000021110.md) | Permet de connaître et de modifier la marge droite d'un champ Libellé. |
| [RetraitGauche](../Proprietes/2513018.md) | Permet de connaître et de modifier la marge gauche d'un champ Libellé. |
| [RetraitHaut](../Proprietes/1000022890.md) | Permet de connaître et de modifier la marge haute d'un champ Libellé. |
| [RichEdit](../Proprietes/2510108.md) | Permet de connaître et de modifier la caractéristique RTF (Rich Text Format) d'un champ Libellé. |


Pour connaître la liste complète des propriétés du WLangage pouvant être utilisées avec un champ de type Libellé, consultez [Propriétés associées à un champ Libellé](../WDChamp/1013002.md).

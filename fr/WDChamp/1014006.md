## Créer un champ dans une page
<a name="creer_champ_dans_une_page_ELTTEXTE000579"></a>


### Créer un champ dans une page
<a name="creer_champ_dans_une_page_ELTPARAGRAPHE000045"></a>Pour créer un champ dans une page :

1. Affichez le volet "Création" de l'éditeur de pages. Ce volet présente tous les types de champs pouvant être créés dans une page. 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Page_Creation_champ.gif&type=thumb)
Selon le type de champ à créer : 

	- cliquez sur son icône. 

	- si une flèche est affichée à côté du nom du champ, déroulez la liste et sélectionnez le type de champ à créer.




2. Cliquez dans la page à la position où le champ doit être créé.




**Remarque** : Pour créer un nouvelle colonne dans un champ Table depuis l'éditeur de pages :

1. Sélectionnez le champ Table.

2. Affichez le menu contextuel (clic droit) : 

	- sélectionnez l'option "Ajouter une colonne". 

	- sélectionnez le type de la colonne à créer.






<a name="NOTE2_2"></a>


### Créer un champ à partir d'un champ d'un dictionnaire
<a name="creer_champ_partir_champ_dictionnaire_ELTPARAGRAPHE000068"></a>WEBDEV permet de créer un champ à partir d'un champ présent dans un dictionnaire (fichier ".DIC"). Un dictionnaire permet de conserver des éléments (champs, pages, ...) en prévision d'une prochaine ré-utilisation dans de nouveaux projets. Pour plus de détails, consultez le [Dictionnaire](../Editeurs/2023002.md).

Pour créer un champ à partir d'un champ d'un dictionnaire :

1. Affichez le [volet "Dictionnaire"](../Editeurs/2027014.md).

2. Ouvrez si nécessaire le dictionnaire (option "Ouvrir" du menu contextuel).

3. Sélectionnez le champ original présent dans le dictionnaire (dans la catégorie "Champs d'une page").

4. Effectuez un glisser/déplacer (Drag and Drop) du champ présent dans le dictionnaire vers la future position du champ dans la page.

5. Abonnez si nécessaire le champ créé au dictionnaire. Si le champ créé est abonné au dictionnaire, les modifications apportées au champ présent dans le dictionnaire pourront être automatiquement reportées sur le champ créé. Pour plus de détails, consultez [Abonnement d'un élément au dictionnaire](../Editeurs/2023001.md).



<a name="NOTE2_3"></a>


### Créer un champ avec le contenu du presse-papiers
<a name="creer_champ_avec_contenu_pressepapiers_ELTPARAGRAPHE000092"></a>L'éditeur de pages permet de créer certains types de champ en fonction du contenu du presse-papiers :

| Contenu du presse-papiers | Type du champ créé |
| --- | --- |
| Texte, copié depuis un éditeur de texte par exemple. | Champ Libellé |
| Fichier image, copié depuis l'explorateur de Windows. | Champ Image |
| Image, copiée depuis une page HTML par exemple. | Champ Image |
| Description d'un fichier de données, copiée depuis l'éditeur d'analyses de WEBDEV ou depuis le [volet "Analyse"](../Editeurs/2027012.md). | Table fichier |
| Rubrique d'un fichier de données, copiée depuis le [volet "Analyse"](../Editeurs/2027012.md). | Champ créé par le RAD selon le type de champ spécifié dans l'onglet "RAD" de la description de rubrique. |


Pour créer un champ avec le contenu du presse-papiers :

1. Copiez l'élément dans le presse-papiers (option "Édition .. Copier" ou Ctrl + C).

2. Sous l'éditeur de pages, sous le volet "Accueil", dans le groupe "Edition", cliquez sur "Coller" (ou Ctrl + V).




**Remarque** : Il est possible de choisir explicitement le format dans lequel le contenu du presse-papiers doit être collé dans l'éditeur de pages : sous le volet "Accueil", dans le groupe "Edition", cliquez sur "Coller" et sélectionnez "Collage spécial". Pour plus de détails, consultez le [Collage spécial](../Editeurs/2026001.md).
<a name="NOTE2_4"></a>


### Créer un champ Image à partir d'un fichier image par "Drag and Drop"
<a name="creer_champ_image_partir_fichier_image_par_drag_and_drop_ELTPARAGRAPHE000165"></a>WEBDEV permet de créer un champ Image par un simple glisser/déplacer (Drag and Drop) d'un fichier image présent dans l'explorateur Windows vers une page WEBDEV.

Pour créer un champ Image à partir d'un fichier image :

1. Sélectionnez le fichier image voulu dans l'explorateur Windows.

2. Maintenez le bouton gauche de la souris enfoncé et déplacez le fichier image vers une page WEBDEV ouverte sous l'éditeur.

3. Relâchez le bouton de la souris. Un champ Image associé au fichier image est automatiquement créé.


Les formats d'images supportés sous Windows sont :

- Bitmap (\*.BMP)

- Graphics Interchange Format (\*.gif)

- Joint Picture Experts Group (\*.jpg ; \*.jpeg)

- Kodak Photo CD (\*.pcd)

- PaintBrush (\*.pcx)

- Adobe Photoshop Format (\*.psd)

- TrueVision TARGA (\*.tga)

- Tagged Image File Format (\*.tif ; \*.tiff)
	Remarque : l'option "tiff mosaïque" de la norme tiff 6.0 n'est pas supportée. Dans ce cas, il est conseillé d'enregistrer l'image à la norme tiff 5.0. 

- Portable Network Graphics (\*.png)

- Windows MetaFiles (\*.emf ; \*.wmf)

- Icônes (\*.ico ; \*.icw)
	Remarques : 

	- Affichage HQ (Haute Qualité) disponible.

	- Gestion des planches au format BMP transparent (32 bits).




- Curseurs (\*.cur)

- Portable Document Format (\*.pdf)

- Scalable Vector Graphics (\*.svg)

- Images WINDEV (générées par l'éditeur d'images, \*.WDPIC).

## Associer un champ à une rubrique d'un fichier de données
<a name="associer_champ_une_rubrique_fichier_donnees_ELTTEXTE000652"></a>


### Comment le faire
<a name="comment_faire_ELTPARAGRAPHE000182"></a>Pour associer un champ existant à une rubrique d'un fichier de données :

1. Sélectionnez le champ voulu.

2. Affichez l'onglet "Liaison" de la fenêtre de description du champ (option "Description" du menu contextuel).

3. Renseignez les informations sur la rubrique reliée.




**Remarque** : Il est également possible de créer directement :

- un champ de saisie à partir d'une rubrique d'un fichier de données :

	1. Dans le [volet "Analyse"](../Editeurs/2027012.md), sélectionnez la rubrique à associer au champ à créer.

	2. Effectuez un glisser/déplacer (Drag and Drop) de la rubrique vers la future position du champ de saisie lié.




- un champ Table à partir d'un fichier de données :

	1. Dans le [volet "Analyse"](../Editeurs/2027012.md), sélectionnez le fichier à associer au champ Table à créer.

	2. Effectuez un glisser/déplacer (Drag and Drop) de la rubrique vers la future position du champ Table lié. L'assistant de création d'un champ Table s'affiche automatiquement.






<a name="NOTE3_2"></a>


### Attention
<a name="attention_ELTPARAGRAPHE000210"></a>Les champs suivants ne peuvent pas être associés à une rubrique d'un fichier de données :

- les boutons,

- les réglettes,

- les cellules,

- les lignes,

- les champs Applet Java,

- les champs Flash,

- les champs Web Caméra,

- les champs HTML.

## Sélectionner, regrouper et supprimer les champs d'une page
<a name="selectionner_regrouper_supprimer_les_champs_une_page_ELTTEXTE000682"></a>


### Sélectionner un ou plusieurs champs dans une page
<a name="selectionner_plusieurs_champs_dans_une_page_ELTPARAGRAPHE000228"></a>WEBDEV permet de sélectionner un ou plusieurs champs d'une page à l'aide du clavier ou de la souris. Pour plus de détails, consultez [Sélectionner un ou plusieurs champs](../Editeurs/2026007.md).

**Cas particulier : Champs groupés**

- Un clic sur un champ groupé sélectionne tous les champs groupés.

- Deux clics successifs sur un champ groupé sélectionnent uniquement ce champ.



<a name="NOTE4_2"></a>


### Regrouper des champs dans une page
<a name="regrouper_des_champs_dans_une_page_ELTPARAGRAPHE000243"></a>Plusieurs champs peuvent être regroupés. Ce regroupement simplifie par exemple le déplacement de plusieurs champs

Rappel : Il existe deux types de groupes :

- le regroupement de champs pour manipuler plusieurs champs sous l'éditeur de pages.

- les [groupes de champs](../WDChamp/1010016.md) pour manipuler plusieurs champs par programmation.




Pour regrouper des champs :

1. Sélectionnez les champs à regrouper (voir [Sélectionner un ou plusieurs champs](../Editeurs/2026007.md)).

2. Sous le volet "Modification", dans le groupe "Profondeur et Alignement", cliquez sur "Grouper la sélection" (Ctrl + G).




Pour dissocier des champs groupés :

1. Sélectionnez un champ appartenant au regroupement. WEBDEV sélectionne automatiquement l'ensemble des champs groupés.

2. Sous le volet "Modification", dans le groupe "Profondeur et Alignement", cliquez sur "Grouper la sélection" et sélectionnez "Dissocier la sélection" (Ctrl + Maj + G).



<a name="NOTE4_3"></a>


### Supprimer un ou plusieurs champs d'une page
<a name="supprimer_plusieurs_champs_une_page_ELTPARAGRAPHE000295"></a>Pour supprimer un ou plusieurs champs d'une page :

1. Sélectionnez le ou les champs à supprimer (voir [Sélectionner un ou plusieurs champs](../Editeurs/2026007.md)).

2. Appuyez sur la touche Suppr.

## Copier les champs d'une page
<a name="copier_les_champs_une_page_ELTTEXTE000718"></a>


### Présentation
<a name="presentation_ELTPARAGRAPHE000310"></a>Un ou plusieurs champs peuvent être copiés dans une même page ou d'une page vers une autre.

La description et le code du champ copié seront identiques à ceux du champ d'origine.
<a name="NOTE5_2"></a>


### Copier les champs dans une même page
<a name="copier_les_champs_dans_une_meme_page_ELTPARAGRAPHE000319"></a>Pour copier un champ dans une même page :

- **1ère méthode** :

	- Sélectionnez le champ à copier.

	- Sous le volet "Accueil", dans le groupe "Edition", cliquez sur "Copier".
			Remarque : Il est également possible d'utiliser le raccourci clavier Ctrl + C.

	- Sous le volet "Accueil", dans le groupe "Edition", cliquez sur "Coller".
			Remarque : Il est également possible d'utiliser le raccourci clavier Ctrl + V.

	- Positionnez le champ à l'aide de la souris (le contour du champ apparaît en pointillé et permet de positionner le nouveau champ).




- **2ème méthode** :

	- Sélectionnez le champ à copier.

	- Cliquez avec le bouton droit de la souris sur le champ à copier et maintenez ce bouton enfoncé.

	- Déplacez le champ à l'aide de la souris (le contour du champ apparaît en pointillé et permet de positionner le champ à coller).

	- Relâchez le bouton de la souris.

	- Sélectionnez l'option "Copier ici".

	- Un nouveau nom est automatiquement attribué au champ copié. Ce nom peut être modifié dans la description du champ copié (option "Description" du menu contextuel du champ).






<a name="NOTE5_3"></a>


### Copier un champ d'une page dans une autre page
<a name="copier_champ_une_page_dans_une_autre_page_ELTPARAGRAPHE000369"></a>Pour copier un champ d'une page dans une autre page :

- **1ère méthode** :

	- Dans la première page, sélectionnez le champ à copier.

	- Sous le volet "Accueil", dans le groupe "Edition", cliquez sur "Copier".
			Remarque : Il est également possible d'utiliser le raccourci clavier Ctrl + C.

	- Affichez la page de destination sous l'éditeur de pages.

	- Sous le volet "Accueil", dans le groupe "Edition", cliquez sur "Coller".
			Remarque : Il est également possible d'utiliser le raccourci clavier Ctrl + V.

	- Positionnez le champ à l'aide de la souris (le contour du champ apparaît en pointillé et permet de positionner le nouveau champ).




- **2ème méthode** : (les deux pages doivent être affichées sous l'éditeur)

	- Dans la première page, sélectionnez le champ à copier.

	- Maintenez le bouton gauche de la souris enfoncé et faites glisser le champ dans la page destination.

	- Relâchez la souris une fois le champ positionné dans la page destination (le contour du champ en pointillé permet de positionner rapidement le champ collé).






<a name="NOTE5_4"></a>


### Copier le style d'un champ
<a name="copier_style_champ_ELTPARAGRAPHE000417"></a>La copie du style d'un champ consiste à recopier les caractéristiques du style d'un champ sur un autre champ. Seul le style du champ est copié (et non le contenu du champ). Pour plus de détails, consultez [Copie du style d'un champ](../Editeurs/2021001.md).

## Déplacer, verrouiller et aligner les champs d'une page
<a name="deplacer_verrouiller_aligner_les_champs_une_page_ELTTEXTE000760"></a>


### Déplacer un ou plusieurs champs d'une page
<a name="deplacer_plusieurs_champs_une_page_ELTPARAGRAPHE000429"></a>Pour déplacer un ou plusieurs champs d'une page :

1. Sélectionnez le ou les champs à déplacer.

2. Maintenez le bouton gauche de la souris enfoncé et déplacez les champs avec la souris. Le correcteur d'interface en temps réel permet d'aligner les champs déplacés (voir le paragraphe suivant).




**Remarque** : Le déplacement des champs sélectionnés peut également s'effectuer avec les flèches du clavier. Pour accélérer le déplacement avec les flèches, il faut appuyer sur la touche Ctrl en même temps que la flèche.
<a name="NOTE6_2"></a>


### Verrouiller des champs
<a name="verrouiller_des_champs_ELTPARAGRAPHE000441"></a>Dans certains cas, il peut être nécessaire de verrouiller les champs en position fixe : la modification de la position du champ devient impossible et le champ n'est pas sélectionné lors d'une sélection au lasso. Cette fonctionnalité est souvent utile pour les champs de décor.

Pour verrouiller un champ : 

1. Sélectionnez le champ.

2. Utilisez une des méthodes suivantes :

	- sous le volet "Affichage", dans le groupe "Champ en cours", cochez la case "Verrouillé".

	- utilisez la combinaison de touches Alt + V.






<a name="NOTE6_4"></a>


### Aligner les champs d'une page
<a name="aligner_les_champs_une_page_ELTPARAGRAPHE000468"></a>Pour aligner correctement les champs d'une page, vous disposez :

- des [options d'alignement](../Editeurs/2022004.md).

- du [correcteur d'interface en temps réel (positionnement automatique)](../Editeurs/2022003.md).

- des [règles et les repères](../Editeurs/2022001.md).

- de la [grille magnétique](../Editeurs/2022002.md).

- du [zoom](../Editeurs/2021007.md).



<a name="NOTE6_5"></a>


### Redimensionner les champs d'une page
<a name="redimensionner_les_champs_une_page_ELTPARAGRAPHE000496"></a>Pour redimensionner un champ dans une page, il est possible d'utiliser : 

- la souris (utilisation des poignées de redimensionnement du champ) 

- les raccourcis clavier suivants sur le champ sélectionné : 

	- Alt flèche droite : Agrandissement du champ par la droite. 

	- Alt flèche gauche : Agrandissement du champ par la gauche.

	- Alt flèche haut : Agrandissement du champ par le haut.  

	- Alt flèche bas : Agrandissement du champ par le bas.

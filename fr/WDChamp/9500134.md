


## Contenu d'un champ Table
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000096"></a>
Un champ Table peut être un champ Table fichier, Table sur variable ou Table par programmation : 

- Un champ Table fichier permet d'afficher des données provenant d'une source de données spécifique (fichier de données, requête, ...).

- Un champ Table sur variable permet d'afficher les données provenant d'une variable.

- Un champ Table par programmation permet d'afficher des données définies par programmation.






<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Contenu d'un champ Table
<a name="contenu_champ_table_ELTTEXTE000120"></a>
Le contenu du champ est défini à la création du champ et peut être modifié dans la description du champ Table (onglet "Contenu").
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Champ_Table_Contenu.gif)
Cet écran permet de définir si le champ est un champ de type : 

- Table fichier (option "Fichier/Requête"),

- Table par programmation (option "Par programmation"), 

- Table sur variable (option "Par variable"). 




**Dans le cas d'un champ Table sur variable**, il est possible de définir la source parcourue et la variable mémorisée. Pour plus de détails sur ce type de champ, consultez [Le champ Table sur variable](../WDChamp/1013306.md). 

**Dans le cas d'un champ Table fichier**, il est possible de déterminer les caractéristiques du parcours de la source de données. Pour plus de détails sur les champs de type Table fichier, consultez [Liaison d'un champ Table à un fichier de données](../WDChamp/1013236.md).

## Présentation du champ Image
<a name="presentation_champ_image_ELTTEXTE000250"></a>


### Champ d'une fenêtre, d'une page, d'un état
<a name="champ_une_fenetre_une_page_etat_ELTPARAGRAPHE000011"></a>Le champ Image permet de :

- Afficher des images statiques ou animées dans vos applications.

- Dessiner.

- Afficher des graphes réalisés avec le grapheur. Cependant, il est conseillé d'utiliser un [champ Graphe](../WDChamp/1013006.md).

- Exécuter des traitements spécifiques lors d'un clic de l'utilisateur sur l'image (équivalent aux zones de cliquages de WINDEV 5.5).

- Afficher des vidéos grâce [aux fonctions MCI](../WDLang1/3051010.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) [Afficher des fichiers PDF](../WDChamp/9500147.md). 
	![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Cette fonctionnalité nécessite le [framework GDI+](../Editeurs/2020007.md).

## Créer un champ Image
<a name="creer_champ_image_ELTTEXTE000274"></a>


### Dans une fenêtre ou une page
<a name="dans_une_fenetre_une_page_ELTPARAGRAPHE000105"></a>Pour créer un champ de type Image dans une fenêtre ou dans une page : 

1. Sous le volet "Création", dans le groupe "Champs usuels", cliquez sur "Image".

2. Cliquez dans la fenêtre ou la page à la position où le champ doit être créé. Le champ apparaît en création.




**Pour afficher les caractéristiques du champ**, sélectionnez l'option "Description" dans le menu contextuel du champ. Les principales options disponibles sont présentées :

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) dans la page [Champ Image dans une fenêtre](../WDChamp/9500147.md). 




**Pour éditer l'image présente dans un champ Image**, utilisez l'option "Editer l'image" du menu contextuel. L'éditeur d'images se lance automatiquement. Pour plus de détails, consultez [Editeur d'images](../Editeurs/1410086603.md). 
<a name="NOTE2_3"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 

### Dans un état
<a name="dans_etat_ELTPARAGRAPHE000149"></a>Pour créer un champ de type Image :

1. Sous le volet "Création", dans le groupe "Visuel", cliquez sur "Image".

2. Cliquez dans l'état à la position où le champ doit être créé. Le champ apparaît en création.




Pour afficher les caractéristiques du champ, sélectionnez l'option "Description" dans le menu contextuel du champ. 

Pour sélectionner l'image (ou le fichier PDF) affichée par le champ, utilisez le bouton ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Menu_Image_Editeur%20-%20HC%20N%B0001.gif), option "Parcourir".

Pour plus de détails sur le mode d'affichage et de transparence de l'image, consultez [Choisir le mode d'affichage de l'image](../WDChamp/1013134.md).

## Images multilingues
<a name="images_multilingues_ELTTEXTE000304"></a>
Si votre application est multilingue, il est possible de sélectionner une image différente pour chaque langue. Cette option est intéressante par exemple lorsque les images contiennent du texte.

Pour gérer des images multilingues : 

1. Affichez la description du champ Image, onglet "Général". 

2. Cliquez sur le bouton ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Menu_Image_Editeur%20-%20HC%20N%B0001.gif), option "Multilingue". La fenêtre de gestion des images multilingues apparaît. Par défaut, le même fichier est associé à toutes les langues gérées par la fenêtre, la page ou l'état.

3. Pour chaque langue, cliquez sur le bouton ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Menu_Image_Editeur%20-%20HC%20N%B0001.gif), option "Parcourir" et sélectionnez l'image correspondant à la langue. En exécution, en fonction de la langue choisie par la fonction [Nation](../WDLang1/3054014.md), l'image correspondante sera affichée.




**Attention** : Seuls les fichiers image sont gérés par le multilingue et non les caractéristiques d'affichage de l'image.

**Remarque** : Si une image multilingue a été définie, le message "&lt;Valeur Multilingue&gt;" apparaît dans la zone correspondant au nom du fichier image dans la fenêtre de description du champ Image.

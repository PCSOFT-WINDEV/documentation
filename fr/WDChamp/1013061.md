
## Propriétés associées au champ Etat Interne
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les différentes propriétés WLangage utilisables avec les champs de type "État interne" sont les suivantes :



|   |   |
| --- | --- |
| [Altitude](../Proprietes/2510002.md) | La propriété **Altitude** permet de gérer l'altitude d'un champ. |
| [Ancrage](../Proprietes/2510001.md) | La propriété **Ancrage** permet de gérer l'ancrage d'un champ dans une fenêtre, une page ou un état. |
| [Cadre](../Proprietes/1000018157.md) | La propriété **Cadre** permet de connaître et de modifier les caractéristiques du cadre : <br><br>- d'un champ d'un état ou d'un bloc d'un état.<br><br>- d'un champ d'une fenêtre.<br><br>- d'une cellule d'un champ Table. <br><br><br> |
| [Etat](../Proprietes/2510039.md) | La propriété **Etat** permet de connaître et de modifier l'état d'affichage d'un élément. |
| [Groupe](../Proprietes/2510049.md) | La propriété **Groupe** permet de : <br><br>- savoir si le champ appartient ou non à un groupe de champs. Si le champ appartient à un ou plusieurs groupes, la propriété **Groupe** permet d'identifier ces groupes de champs.<br><br>- associer un champ à un groupe de champs. <br><br><br> |
| [Hauteur](../Proprietes/2510050.md) | La propriété **Hauteur** permet de connaître et modifier la hauteur :<br><br>- d'un champ ou d'un bloc dans un état.<br><br>- d'un champ (présent dans une fenêtre), d'une fenêtre, d'une ligne d'un champ Table ou Liste (y compris Liste image), d'une option d'un champ Sélecteur ou Interrupteur, d'un haut ou d'un bas de rupture d'un champ Table ou d'un champ Zone répétée.<br><br>- d'un champ (présent dans une page), d'un haut ou bas de rupture d'un champ Table ou Zone répétée. Le champ peut être superposable ou non.<br><br>- d'une cellule de rupture d'un champ Table ou d'un champ Zone répétée.<br><br><br> |
| [HauteurPixel](../Proprietes/2511064.md) | La propriété **HauteurPixel** permet de connaître la hauteur d'un champ d'un état en pixels. |
| [Largeur](../Proprietes/2510059.md) | La propriété **Largeur** permet de connaître et modifier la largeur :<br><br>- d'un champ ou d'un bloc dans un état.<br><br>- d'un champ (présent dans une fenêtre), d'une fenêtre, d'une colonne de table ou d'une colonne de liste (y compris liste image).<br><br>- d'un champ (présent dans une page), d'une colonne de table ou d'une liste. Le champ peut être superposable ou non.<br><br><br> |
| [LargeurInitiale](../Proprietes/2510067.md) | La propriété **LargeurInitiale** permet de :<br><br>- Connaître la largeur de création d'un champ ou d'une fenêtre.<br><br>- Modifier la largeur de base utilisée pour l'ancrage des champs "en largeur".<br><br>- Connaître la largeur de création d'un champ d'un état.<br><br><br> |
| [LargeurPixel](../Proprietes/2511052.md) | La propriété **LargeurPixel** permet de connaître la largeur d'un champ d'un état en pixels. |
| [NiveauFlou](../Proprietes/1410088348.md) | La propriété **NiveauFlou** permet de connaître et modifier le pourcentage de flou utilisé lors du floutage d'un champ d'un état. |
| [Nom](../Proprietes/2510082.md) | La propriété **Nom** permet de connaître :<br><br>- le nom de la police associée à une variable de type Police.<br><br>- le nom d'un champ, d'un état ou d'un bloc d'un état.<br><br>- le nom d'un champ, d'un groupe de champs ou d'une fenêtre.<br><br>- le nom d'un champ, d'un groupe de champs ou d'une page.<br><br><br> |
| [NomEtatInterne](../Proprietes/2511037.md) | La propriété **NomEtatInterne** permet de :<br><br>- Connaître le nom de l'état imprimé dans un champ de type "[Etat Interne](../WDChamp/1013044.md)".<br><br>- Changer l'état imprimé dans un champ de type "[Etat Interne](../WDChamp/1013044.md)".<br><br><br> |
| [Note](../Proprietes/2510085.md) | La propriété **Note** permet de :<br><br>- Connaître les notes associées à un champ, à une fenêtre, à une page ou un état.<br><br>- Modifier les notes associées à un champ, à une fenêtre, à une page ou un état.<br><br><br> |
| [Occulté](../Proprietes/1410087964.md) | La propriété **Occulté** permet de : <br><br>- Savoir si un champ d'un état est occulté. <br><br>- Occulter un champ d'un état en le floutant ou en le biffant.<br><br><br> |
| [Plan](../Proprietes/2510094.md) | La propriété **Plan** permet de :<br><br>- Connaître : <br><br>	- le plan actif d'une fenêtre (ou d'un état) ou le plan auquel appartient un champ.<br><br>	- le plan actif d'un élément WEBDEV.<br><br><br><br><br>- Modifier : <br><br>	- le plan actif d'une fenêtre (ou d'un état) ou associer un champ à un autre plan.<br><br>	- le plan actif d'un élément WEBDEV.<br><br><br><br><br><br> |
| [TauxAncrageBas](../Proprietes/2510113.md) | La propriété **TauxAncrageBas** permet de :<br><br>- Connaître le taux d'ancrage en déplacement d'un champ sur l'axe vertical.<br><br>- Modifier le taux d'ancrage en déplacement d'un champ sur l'axe vertical.<br><br><br> |
| [TauxAncrageDroite](../Proprietes/2510114.md) | La propriété **TauxAncrageDroite** permet de :<br><br>- Connaître le taux d'ancrage en déplacement d'un champ sur l'axe horizontal.<br><br>- Modifier le taux d'ancrage en déplacement d'un champ sur l'axe horizontal.<br><br><br> |
| [TauxAncrageHauteur](../Proprietes/2510119.md) | La propriété **TauxAncrageHauteur** permet de :<br><br>- Connaître le taux d'ancrage en hauteur d'un champ.<br><br>- Modifier le taux d'ancrage en hauteur d'un champ.<br><br><br> |
| [TauxAncrageLargeur](../Proprietes/2510120.md) | La propriété **TauxAncrageLargeur** permet de :<br><br>- Connaître le taux d'ancrage en agrandissement d'un champ (ou d'une colonne de table) sur l'axe horizontal.<br><br>- Modifier le taux d'ancrage en agrandissement d'un champ (ou d'une colonne de table) sur l'axe horizontal.<br><br><br> |
| [TitreNote](../Proprietes/2510121.md) | La propriété **TitreNote** permet de :<br><br>- Connaître le titre des notes utilisé pour la documentation technique d'un champ, d'une fenêtre, d'une page ou d'un état. <br><br>- Modifier le titre des notes utilisé pour la documentation technique d'un champ, d'une fenêtre, d'une page ou d'un état.<br><br><br> |
| [Traitement](../Proprietes/1000020878.md) | La propriété **Traitement** permet de modifier les traitements à exécuter par les événements disponibles sur un champ, une fenêtre, une page, un état, ... |
| [ValeurRenvoyée](../Proprietes/2510134.md) | La propriété **ValeurRenvoyée** permet de :<br><br>- Connaître et modifier la valeur renvoyée par une fenêtre, une page ou un état.<br><br>- Connaître et modifier la valeur renvoyée par une option d'un champ Sélecteur. <br><br><br> |
| [Visible](../Proprietes/2510138.md) | La propriété **Visible** permet de :<br><br>- Savoir si un élément est visible.<br><br>- Rendre un élément visible ou invisible.<br><br><br> |
| [X](../Proprietes/2510135.md) | La propriété **X** permet de :<br><br>- Connaître l'abscisse d'un champ, d'une colonne d'un champ Table ou d'une fenêtre (position sur l'axe des X).<br><br>- Modifier l'abscisse d'un champ ou d'une fenêtre (position sur l'axe des X).<br><br><br> |
| [XInitial](../Proprietes/2510140.md) | La propriété **XInitial** permet de :<br><br>- Connaître l'abscisse : <br><br>	- d'un champ (position sur l'axe des X) lors de l'ouverture de la fenêtre ou de la page.<br><br>	- d'une fenêtre (position sur l'axe des X) lors de l'ouverture de la fenêtre. <br><br><br><br><br>- Modifier l'abscisse d'un champ ancré "à droite" (position sur l'axe des X) lors de l'ouverture de la fenêtre.<br><br>- Connaître l'abscisse d'un champ par rapport à son bloc (position en millimètres sur l'axe des X) lors de l'ouverture de l'état.<br><br><br> |
| [Y](../Proprietes/2510137.md) | La propriété **Y** permet de :<br><br>- Connaître l'ordonnée d'un champ ou d'une fenêtre (position sur l'axe des Y).<br><br>- Modifier l'ordonnée d'un champ ou d'une fenêtre (position sur l'axe des Y).<br><br><br> |
| [YInitial](../Proprietes/2510087.md) | La propriété **YInitial** permet de :<br><br>- Connaître l'ordonnée : <br><br>	- d'un champ (position sur l'axe des Y) lors de l'ouverture de la fenêtre ou de la page.<br><br>	- d'une fenêtre (position sur l'axe des Y) lors de l'ouverture de la fenêtre.<br><br><br><br><br>- Modifier l'ordonnée d'un champ ancré "en bas" (position sur l'axe des Y) lors de l'ouverture de la fenêtre.<br><br>- Connaître l'ordonnée d'un champ par rapport à son bloc (position en millimètres sur l'axe des Y) lors de l'ouverture de l'état.<br><br><br> |

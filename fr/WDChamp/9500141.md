
## Calculs automatiques dans un champ Table (FAA)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000133"></a>
Dans les champs Table et Table hiérarchique, l'utilisateur final a la possibilité d'ajouter des calculs automatiques. Ces calculs sont disponibles sur tous les types de colonnes. 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Afficher ou non un calcul automatique
<a name="afficher_non_calcul_automatique_ELTTEXTE000157"></a>


### Afficher un calcul automatique
<a name="afficher_calcul_automatique_ELTPARAGRAPHE000016"></a>Pour afficher un calcul automatique dans une colonne de table : 

1. Affichez le menu contextuel de la colonne voulue. 

2. Sélectionnez l'option "Calcul automatique (somme, moyenne, ...)" puis sélectionnez le type de calcul voulu : 

	- Somme

	- Nombre d'éléments

	- Moyenne

	- Minimum

	- Maximum




3. Le calcul est automatiquement réalisé et ajouté en dessous de la table. Il est toujours visible. 




Attention : 

- Le menu contextuel permettant d'ajouter ou de supprimer des calculs automatiques n'est pas disponible sur tous les types de colonnes. 

- Le menu contextuel permettant d'ajouter ou de supprimer des calculs automatiques n'est pas disponible sur les champs Table de type "Accès direct" affichant plus de 1000 lignes (pour optimiser les temps de calcul). 

- L'exécution des calculs automatiques du champ Table (haut ou bas de champ et ruptures) est un traitement synchrone.





### Supprimer un calcul automatique
<a name="supprimer_calcul_automatique_ELTPARAGRAPHE000041"></a>Pour supprimer un calcul automatique dans une colonne de table : 

1. Affichez le menu contextuel de la colonne voulue. 

2. Sélectionnez l'option "Calcul automatique (somme, moyenne, ...)" puis sélectionnez le type de calcul à ne plus afficher : 

	- Somme

	- Nombre d'éléments

	- Moyenne

	- Minimum

	- Maximum




3. Le calcul est automatiquement supprimé pour la colonne sélectionnée. 
	Attention : Si le calcul existe pour d'autres colonnes, la ligne de calculs sera toujours affichée sous la table. 




Attention : Le menu contextuel permettant d'ajouter ou de supprimer des calculs automatiques n'est pas disponible sur tous les types de colonnes.

## Programmation
<a name="programmation_ELTTEXTE000187"></a>


### Proposer un calcul automatique
<a name="proposer_calcul_automatique_ELTPARAGRAPHE000069"></a>Les calculs automatiques sont proposés par défaut dans le menu contextuel des colonnes de tables (pour les types de colonnes les supportant). 

Rappels : 

- Il est possible d'afficher un calcul automatique dans un champ Table ou Table hiérarchique. Il suffit de définir le calcul à effectuer dans la fenêtre de description de la colonne. Pour plus de détails, consultez [Calculs dans les colonnes d'un champ Table](../WDChamp/1013201.md). 

- Des calculs automatiques personnalisés peuvent également être ajoutés dans les colonnes d'un champ Table. Pour plus de détails, consultez [Programmation de calculs personnalisés dans les tables](../WDChamp/1013331.md).

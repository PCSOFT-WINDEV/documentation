
## FAA : Tri sur les colonnes d'un champ Table
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000152"></a>
WINDEV permet à l'utilisateur de trier très simplement les colonnes d'un champ Table. 

Il est également possible de réaliser des tris par programmation. 

Remarque : Ces manipulations sont également disponibles sur un champ Table hiérarchique. 



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Trier les colonnes (FAA)
<a name="trier_les_colonnes_faa_ELTTEXTE000176"></a>


### Tri simple
<a name="tri_simple_ELTPARAGRAPHE000023"></a>Pour trier les colonnes d'un champ Table, l'utilisateur doit juste cliquer sur le titre de la colonne.

Les colonnes triables sont identifiées par une double-flèche.

Lorsque le tri est actif, une petite flèche indique le sens de tri de la colonne.
<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Triez%20les%20donn%E9es%20affich%E9es%20dans%20vos%20tables%20en%20un%20clic.jpg)


**Astuce** : Pour ajouter cette fonctionnalité, il est nécessaire de cocher l'option "Triable" dans l'onglet "Général" de la fenêtre de description de la colonne.


<a name="NOTE2_2"></a>


### Tri multiple
<a name="tri_multiple_ELTPARAGRAPHE000039"></a>Pour trier le contenu d'un champ Table sur plusieurs colonnes, l'utilisateur doit :

1. Maintenir la touche Ctrl enfoncée.

2. Cliquer sur le titre des colonnes à prendre en compte dans le tri. Il est nécessaire de sélectionner les colonnes selon l'ordre voulu. Par exemple, pour trier sur la société puis le nom, il est nécessaire de cliquer en premier sur la colonne "Société" puis ensuite sur la colonne "Nom" .




**Remarque** : Le tri multiple est disponible uniquement sur les champs Table mémoire.


<a name="NOTE2_3"></a>


### Mémoriser les tris
<a name="memoriser_les_tris_ELTPARAGRAPHE000054"></a>Lorsque l'utilisateur a effectué un ou un ensemble de tris sur des colonnes d'un champ Table, il peut demander à mémoriser ces tris afin que ces tris soient appliqués par défaut. 

Pour activer cette fonctionnalité, il suffit de : 

1. Afficher le menu contextuel de la colonne de tri du champ Table. Pour plus de détails, consultez [Menu contextuel des colonnes](../WDChamp/9500125.md). 

2. Sélectionner l'option "Mémoriser les tris".

## Développement et programmation
<a name="developpement_programmation_ELTTEXTE000212"></a>
**Lors de la création d'un champ Table** :
Pour rendre une colonne triable, il suffit de sélectionner l'option "Triable" dans l'onglet "Général" de la fenêtre de description de la colonne. 

**En programmation** : 

- Pour trier le contenu d'un champ Table, il suffit d'utiliser la fonction [TableTrie](../WDLang1/3074035.md). 

- La fonction [TableColonnesTriées](../WDLang1/3074010.md) permet d'obtenir la liste des colonnes triées dans le champ Table. 

- La propriété [MémoriseTriFAA](../Proprietes/1000020069.md) permet de savoir et d'indiquer si le tri effectué par l'utilisateur sur un champ Table ou Table hiérarchique est conservé entre 2 lancements de l'application.

- La propriété [OptionTri](../Proprietes/1000017044.md) permet de connaître et de modifier les options de tri utilisées dans une colonne d'un champ Table. 

- La propriété [Triable](../Proprietes/2510116.md) permet de savoir si une colonne est triable et de rendre une colonne triable ou non. 

- Pour ne pas proposer les options "Mémoriser les filtres" et "Mémoriser les tris" dans le menu contextuel, il suffit d'utiliser la fonction [FAADésactive](../WDLang1/1000022018.md) (ou [DésactiveFAA](../WDLang1/3013061.md)) avec la constante *faaSauveFiltreEtTri*.

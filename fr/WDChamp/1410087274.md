## Présentation du champ Flexbox
<a name="presentation_champ_flexbox_ELTTEXTE000123"></a>
Flexbox est une méthode de mise en page standard et déterministe des pages Web. A partir de la version 28, le champ Flexbox est disponible dans WEBDEV. 

Le principe est simple : 

- Vous posez les champs de l'UI dans le champ Flexbox.

- Le champ Flexbox se charge alors de positionner les champs automatiquement en fonction des paramètres souhaités.




Le champ Flexbox permet par exemple de : 

- Gérer le positionnement des champs lorsque le navigateur change de taille. Pour plus de détails, consultez [Exemple Flexbox : disposition automatique des champs](../WDChamp/1410087285.md). 

- Mettre en place des ancrages proportionnels sur les champs. Pour plus de détails, consultez [Exemple Flexbox : ancrage proportionnel sur les champs](../WDChamp/1410087275.md).

## Créer un champ Flexbox
<a name="creer_champ_flexbox_ELTTEXTE000147"></a>
Pour créer un champ de type Flexbox :

1. Sous le volet "Création", dans le groupe "Conteneurs", cliquez sur "Flexbox". 

2. Cliquez dans la page à la position où le champ doit être créé. Le champ apparaît en création.




Remarque : En déroulant l'option "Flexbox" du menu, plusieurs champs Flexbox pré-définis sont proposés, notamment : 

- Flexbox en colonnes,

- Flexbox en lignes,

- Cartes flexbox. 




Pour afficher les caractéristiques du champ : 

1. Sélectionnez le champ. 

2. Sélectionnez l'option "Description" dans le menu contextuel du champ.


## Champ TimeLine lié à un fichier de données ou une variable
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000130"></a>
Il est possible de créer des champs TimeLine directement liés aux données. Vous pouvez créer : 

- un **champ TimeLine fichier** : le champ TimeLine sera lié à un fichier de données ou à une requête. Dans ce cas, le fichier de données contient les caractéristiques des événements. 

- un **champ TimeLine sur variable** : le champ TimeLine sera lié à une variable (le plus souvent une instance de classe). Dans ce cas, la classe liste les événements avec leurs caractéristiques. 




La programmation des champs TimeLine est grandement simplifiée. 





## Comment le faire ?
<a name="comment_faire_ELTTEXTE000154"></a>


### Créer un champ TimeLine fichier
<a name="creer_champ_timeline_fichier_ELTPARAGRAPHE000020"></a>Pour créer un champ TimeLine fichier : 

1. Affichez la fenêtre de description du champ (option "Description" du menu contextuel). 

2. Dans l'onglet "Contenu", sélectionnez l'option "Fichier/Requête" et sélectionnez la source de données utilisée. Cette source de données peut être : 

	- un fichier de données défini dans l'analyse,

	- une requête du projet,

	- une requête intégrée à la fenêtre. 




3. Associez les différentes rubriques de la source de données aux caractéristiques de l'événement. Les éléments précédés d'une étoile sont obligatoires. 
	**Remarque** : Si le fichier de données associé ne contient pas directement la rubrique correspondante, il est possible de réaliser une liaison multi-fichiers. 

4. Validez la fenêtre de description du champ TimeLine. Le champ TimeLine est un timeline fichier. 







### Créer un champ TimeLine sur variable
<a name="creer_champ_timeline_sur_variable_ELTPARAGRAPHE000037"></a>Pour créer un champ TimeLine sur variable : 

1. Affichez la fenêtre de description du champ (option "Description" du menu contextuel). 

2. Dans l'onglet "Contenu", sélectionnez l'option "Par variable" et sélectionnez la source de données utilisée (classe, ...). 

3. Associez les différents éléments (membres de la classe par exemple) de la source de données aux caractéristiques de l'événement. Les éléments précédés d'une étoile sont obligatoires. 

4. Validez la fenêtre de description du champ TimeLine. Le champ TimeLine est un timeline sur variable. 










## Manipuler un champ TimeLine fichier ou un champ TimeLine sur variable
<a name="manipuler_champ_timeline_fichier_champ_timeline_sur_variable_ELTTEXTE000184"></a>


### Champ TimeLine fichier
<a name="champ_timeline_fichier_ELTPARAGRAPHE000050"></a>Un champ TimeLine fichier se manipule très simplement : 

- Si l'utilisateur manipule directement le timeline via le menu contextuel du champ, les ajouts, suppressions, modifications des événements sont automatiquement reportés dans le fichier de données lié au champ.

- Si l'utilisateur manipule les événements via une fenêtre spécifique, il suffit de modifier directement le fichier de données lié au timeline. Les données affichées seront automatiquement mises à jour. Il est également possible d'utiliser la fonction [TimelineAffiche](../WDLang1/1000020683.md) pour remettre à jour les données affichées dans le champ. 




**Remarque** : Si le champ est lié à une source de données, la suppression réalisée dans le champ est effectuée en même temps dans la source de données liée. 


### Champ TimeLine sur variable
<a name="champ_timeline_sur_variable_ELTPARAGRAPHE000066"></a>Un champ TimeLine sur variable se manipule très simplement : 

- Si l'utilisateur manipule directement le timeline via le menu contextuel du champ, les ajouts, suppressions, modifications des événements sont automatiquement reportés dans la source de données du champ.

- Si l'utilisateur manipule les événements via une fenêtre spécifique, il suffit de modifier directement les éléments lié au timeline (membres de classe par exemple). Les données affichées seront automatiquement mises à jour. Il est également possible d'utiliser la fonction [TimelineAffiche](../WDLang1/1000020683.md) pour remettre à jour les données affichées dans le champ. 




**Remarque** : Si le champ est lié à une source de données, la suppression réalisée dans le champ est effectuée en même temps dans la source de données liée.

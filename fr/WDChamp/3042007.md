


## Les graphes de type "Secteur", "Beignet" ou "Hémicycle"
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000197"></a>
Plusieurs types de graphes font partie d'une même famille :

- Graphe en secteurs appelé également camembert : ce type de graphe représente un cercle découpé en secteurs.

- Graphe en beignet : ce type de graphe représente un cercle découpé en secteurs avec un trou en son centre.

- Graphe en hémicycle : ce type de graphe représente un demi-cercle découpé en secteurs.




La taille de chaque secteur est proportionnelle à une donnée de la série.

Un graphe en secteur, beignet ou hémicycle n'affiche jamais plus d'une série de données.

Il permet de mettre en évidence un élément particulièrement important.

![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Seuls les graphes de type Secteur sont disponibles.

## Caractéristiques d'un graphe de type Secteur, Beignet ou Hémicycle
<a name="caracteristiques_graphe_type_secteur_beignet_hemicycle_ELTTEXTE000221"></a>

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Gencam.gif&type=thumb)
Dans ce graphe :

**1. Angle de départ**. 
Angle utilisé pour le départ de la première part. Ce paramètre peut être défini dans l'onglet "Général" de la fenêtre de description du champ Graphe (option "Angle de départ (en degrés) "). Il est également possible de décaler les parts du graphe (option "Excentrage" de l'onglet "Général").

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Pour un graphe de type Beignet, il est également possible de spécifier la taille du trou.

**2. Titre du graphe**. 
Les caractéristiques du titre du graphe sont saisies dans l'onglet "Détail" de la description du champ (bouton "Editer le titre du graphe").
Il est possible de définir le titre du graphe, sa position et son style.

**3. Légende**. 
Les caractéristiques de la légende sont saisies dans l'onglet "Détail" de la description du champ (bouton "Editer la légende du graphe").
Il est possible de définir sa position et son style.

**4. Etiquette de catégorie**. 
Les étiquettes de catégories sont affichées dans la légende. Les étiquettes de catégorie peuvent être définies dans l'onglet "Séries" de la description du champ. La source des étiquettes de catégorie peut correspondre à une colonne de table, un tableau WLangage, une liste, ... 

**5. Couleur de secteur**. 
La couleur de parts est définie automatiquement. Pour modifier la couleur des secteurs, utilisez la fonction [grCouleurSérie](../WDLang3/3042013.md) en précisant le numéro du secteur. 

**6. Relief**. 
Les caractéristiques de relief peuvent être spécifiées dans l'onglet "Général" de la fenêtre de description du champ Graphe.



**7. Libellé de secteur**. 
Le libellé des secteurs peut contenir :

- la valeur correspondant au secteur : option "Afficher les valeurs" de l'onglet "Détail" de la fenêtre de description du champ Graphe.

- les étiquettes de catégorie (option "Afficher les étiquettes des catégories" de l'onglet "Général" de la fenêtre de description du champ Graphe). 

- les pourcentages (option "Afficher les pourcentages" de l'onglet "Général" de la fenêtre de description du champ Graphe).




Il est possible de choisir si les étiquettes seront affichées à l'intérieur ou à l'extérieur des portions du graphe (option "Afficher les étiquettes à l'extérieur des portions" de l'onglet "Détail" de la fenêtre de description du champ Graphe).

**8. Couleurs dégradées**. 
Les caractéristiques d'apparence du graphe sont définies dans l'onglet "Détail" de la fenêtre de description du graphe.
Les options suivantes sont disponibles :

- Adoucir les effets d'escalier.

- Couleurs de bordure automatiques.

- Couleurs dégradées (avec possibilité de dégradé horizontal).

- Effet de lumière réaliste (avec possibilité d'indiquer le pourcentage du biseau)

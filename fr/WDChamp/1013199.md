
## Gestion de la saisie obligatoire dans WEBDEV
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000098"></a>
Pour obliger l'internaute à renseigner un champ, il suffit d'utiliser le mécanisme de saisie obligatoire. Grâce à ce mécanisme, l'internaute ne peut pas valider la page tant que toutes les informations obligatoires ne sont pas renseignées.

**Remarque** : Le mécanisme de saisie obligatoire concerne uniquement les champs de saisie.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Mécanisme de saisie obligatoire
<a name="mecanisme_saisie_obligatoire_ELTTEXTE000122"></a>
Lors de la validation d'une page, une vérification est effectuée. Si un ou plusieurs champs de type "Saisie obligatoire" ne sont pas renseignés :

- un boîte de dialogue indique les champs obligatoires.

- la validation de la page est annulée.

- le premier champ non renseigné dans l'ordre de tabulation reprend automatiquement le focus.




Par exemple, la page suivante contient plusieurs champs de type "Saisie obligatoire" :

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Formulaire.gif)


Lors de la validation de cette page, l'internaute a oublié de renseigner plusieurs champs obligatoires. La validation de la page s'annule et la boîte de dialogue suivante s'affiche :

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=BoiteDialogueSaisieOblig.gif)


Après validation de cette boîte de dialogue, le premier champ non renseigné dans l'ordre de tabulation (ici, le champ "Code postal") reprend automatiquement le focus :

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Formulaire2.gif)




<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Mise en place
<a name="mise_place_ELTTEXTE000146"></a>
Il suffit de spécifier les différents champs de saisie obligatoire en cochant l'option "Saisie obligatoire" (onglet "Détail" de la fenêtre de description du champ de saisie).


![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=SaisieObligatoireOption.gif)

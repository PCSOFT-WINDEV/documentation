


## Champ Lecteur PDF : Utiliser le ruban et les volets
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000347"></a>
Le champ Lecteur PDF propose de nombreuses fonctionnalités accessibles via : 

- Le ruban. Plusieurs volets sont disponibles : 

	- Le volet "Affichage" qui permet d'effectuer les manipulations de base sur le fichier PDF affiché. 

	- Le volet "Rechercher" qui permet d'effectuer une recherche dans le fichier PDF affiché. 

	- Le volet "Modification" qui permet de modifier le contenu du fichier PDF affiché. 




- Le volet de l'éditeur, qui permet de visualiser les vignettes des différentes pages. 




Remarque : Une bulle d'aide est affichée sur l'ascenseur du champ Lecteur PDF permettant de connaître à tout moment le numéro de la page en cours et le nombre total de pages.

## Ruban : Volet "Affichage"
<a name="ruban_volet_affichage_ELTTEXTE000371"></a>
Le volet "Affichage" permet d'effectuer toutes les manipulations de base sur un fichier PDF. 

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Champ_LecteurPDF_Ruban%20-%20HC%20N%B0001%201.gif&type=thumb)
Ce volet propose les groupes suivants : 

- **Général** : Permet d'ouvrir, d'imprimer ou d'enregistrer un document PDF.

- **Presse-papiers** : Permet de gérer le presse-papiers (copier). 

- **Zoom** : Permet de gérer le zoom sur le document PDF.

- **Mode d'affichage** : Permet de gérer les options d'affichage.  

- **Pages** : Permet de se déplacer entre les différentes pages du document PDF. 

- **Outils** : Permet de choisir l'action effectuée par la souris.  





### Groupe "Général"
<a name="groupe_general_ELTPARAGRAPHE000043"></a>Le groupe **Général** permet d'ouvrir, d'imprimer ou d'enregistrer un document PDF. ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Champ_LecteurPDF_Ruban%20-%20HC%20N%B0001%202.gif)
Il est possible de : 

- **Ouvrir un fichier PDF existant**. 

	- A l'ouverture d'un fichier PDF, les droits appliqués sur ce fichier sont respectés (droits de copie, droits d'impression). 

	- Si le fichier PDF est protégé par un mot de passe, ce mot de passe est demandé à l'utilisateur. 




- **Imprimer le fichier PDF en cours**. La fenêtre de configuration de l'impression est affichée (avec un aperçu du document imprimé). 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Champ_Lecteur_PDF_Imprimer%20-%20HC%20N%B0001.gif&type=thumb)
Il est possible de paramétrer : 

	- les propriétés de l'imprimante : Nom de l'imprimante, orientation, mode d'impression, nombre de copies. 

	- les pages à imprimer : toutes les pages, page courante ou une sélection de pages (pages séparées par un "-" pour une sélection, pages séparées par ',' pour des pages indépendantes). 

	- le dimensionnement des pages : réduction des pages hors format, ajustement de la page ou utilisation de la taille réelle. Il est également possible de choisir la source du papier selon le format de la page. 


 Il est également possible de :

- visualiser l'aperçu des différentes pages à imprimer. 

- réaliser un export sous forme d'email avec le fichier PDF attaché (bouton "Exporter"). 

- **Enregistrer le fichier PDF en cours**. Cette option permet d'enregistrer le fichier PDF avec ses modifications (notamment le changement d'orientation des pages). Il est possible d'enregistrer le fichier actuel ou de faire un "Enregistrer sous". 
	**Remarque** : Cette fonctionnalité est disponible à partir de la version 250064 (Update 2). Si vous utilisez la fenêtre par défaut, la modification est automatiquement effectuée lors de la recompilation du projet. Si vous avez personnalisé la fenêtre interne, il est nécessaire de faire une comparaison des différences entre la fenêtre interne "FI_WindevFAA_RubanLecteurPDF" du composant WDFAA et celle présente dans le répertoire "Programmes\\Donnees\\Fenêtres prédéfinies\\FAA". Pour plus de détails, consultez [Comparateur de projets](../Editeurs/2030024.md).    




Remarque : Si le fichier PDF ouvert est un fichier signé, l'onglet "Affichage" affiche une information spécifique. 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Champ_Lecteur_PDF_Signature%20-%20HC%20N%B0004.gif)
 Pour plus de détails, consultez [Signature de documents](#NOTE4_4). 


### Groupe "Presse-papiers"
<a name="groupe_pressepapiers_ELTPARAGRAPHE000087"></a>Le groupe **Presse-papiers** permet de copier le contenu de la sélection. ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Champ_LecteurPDF_Ruban%20-%20HC%20N%B0001%203.gif)
Ce contenu pourra ensuite être collé dans une autre application (bloc-notes, ...). 

Remarques : 

- La copie est disponible uniquement si le fichier PDF ouvert autorise la copie. 

- Pour sélectionner une partie du document PDF, il faut être en "Mode Sélection". 

- La copie peut également être effectuée via le menu contextuel de la sélection. 





### Groupe "Zoom"
<a name="groupe_zoom_ELTPARAGRAPHE000100"></a>Le groupe **Zoom** permet de gérer le zoom d'un document PDF. ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Champ_LecteurPDF_Ruban%20-%20HC%20N%B0001%204.gif)
Il est possible de : 

- Afficher le document PDF à 100%. 

- Afficher le document PDF pour que la largeur de la page soit affichée en entier.

- Afficher le document PDF sous forme de pages entières à l'écran. 

- Personnaliser le zoom. 





### Groupe "Mode d'affichage"
<a name="groupe_mode_affichage_ELTPARAGRAPHE000112"></a>Le groupe **Mode d'affichage** permet de gérer les options d'affichage du document PDF en cours. ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Champ_LecteurPDF_Ruban%20-%20HC%20N%B0001%205.gif)
Il est possible de : 

- Afficher une seule page. 

- Afficher deux pages. Dans ce cas, il est possible de gérer une page de couverture. La première page sera affichée à droite.  
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Champ_Lecteur_PDF_Imprimer%20-%20HC%20N%B0002.gif&type=thumb)



Remarque : l'affichage d'une couverture modifie automatiquement le zoom pour que la page entière soit affichée. 


### Groupe "Pages"
<a name="groupe_pages_ELTPARAGRAPHE000125"></a>Le groupe **Pages** permet de se déplacer dans un document PDF. ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Champ_LecteurPDF_Ruban%20-%20HC%20N%B0001%206.gif)
Il est possible de : 

- Afficher ou non le volet des miniatures. Ce volet permet de visualiser les différentes pages du document PDF sous forme de miniatures et de se déplacer parmi les différentes pages. 

- Connaître le nombre total de pages du document. 

- Connaître le numéro de la page actuelle. 

- Se déplacer entre les différentes pages via les boutons fléchés.  





### Groupe "Outils"
<a name="groupe_outils_ELTPARAGRAPHE000137"></a>Le groupe **Outils** permet de choisir l'action effectuée par la souris. ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Champ_LecteurPDF_Ruban%20-%20HC%20N%B0002.gif)


- Si l'outil "Déplacement" est sélectionné, la souris se transforme en une main et permet de parcourir rapidement les différentes pages du fichier PDF. 

- Si l'outil "Sélection" est sélectionné, la souris permet de sélectionner du texte et le met en surbrillance. Selon les droits accordés sur le fichier, il est possible de copier par la suite le contenu de la sélection. 

- Si l'outil "OCR" est sélectionné, la souris permet de sélectionner un texte à reconnaître via un lasso. Le contenu de la sélection réalisée avec le lasso est analysé et le texte correspondant est copié dans le presse-papiers.

## Ruban : Volet "Rechercher"
<a name="ruban_volet_rechercher_ELTTEXTE000431"></a>
Le volet "Rechercher" permet d'effectuer des recherches dans le document PDF en cours. 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Champ_LecteurPDF_Ruban%20-%20HC%20N%B0003.gif)
Il suffit de saisir le texte à rechercher dans la zone de recherche pour que la recherche soit lancée. Il est possible de spécifier des options de recherche : 

- **Mot complet uniquement** : Cherche uniquement le mot spécifié. Les mots contenant le mot spécifié ne seront pas trouvés. Par exemple, si la recherche est effectuée sur "séjour", avec l'option "Mot complet uniquement", "séjours" ne sera pas trouvé. 

- **Respecter la casse** : Respecte la casse utilisée dans le champ de recherche.  Par exemple, si la recherche est effectuée sur "séjour", avec l'option "Respecter la casse", "Séjour" ne sera pas trouvé. 


Les boutons fléchés permettent de se déplacer dans les différents résultats de la recherche. Si la recherche est terminée, un message est affiché.

## Ruban : Volet "Modification"
<a name="ruban_volet_modification_ELTTEXTE000455"></a>
Le volet "Modification" permet de modifier le document PDF en cours.![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Champ%20Lecteur%20PDF_Volet_modification%20-%20HC%20N%B0003%201.gif)


Ce volet propose les groupes suivants : 

- **Rotation**, qui permet d'effectuer une rotation des pages du document PDF. 

- **Pages**, qui permet de supprimer ou d'importer des pages.

- **Insertion**, qui permet d'ajouter un filigrane ou une signature.



<a name="NOTE4_2"></a>


### Groupe "Rotation"
<a name="groupe_rotation_ELTPARAGRAPHE000182"></a>Le groupe "Rotation" permet d'effectuer une rotation de pages dans le document PDF. ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Champ%20Lecteur%20PDF_Volet_modification%20-%20HC%20N%B0003%202.gif)
Il est possible d'effectuer une rotation de toutes les pages du document affiché dans le lecteur PDF. 
Remarques : 

- Pour effectuer la rotation de la page en cours, il suffit d'afficher le menu contextuel de la page (clic droit) et de sélectionner l'option voulue. ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Champ_LecteurPDF_Menu%20-%20HC%20N%B0001.gif)


- Il est possible d'enregistrer le document en prenant en compte les rotations effectuées (bouton "Enregistrer" du groupe "Général").



<a name="NOTE4_3"></a>


### Groupe "Pages"
<a name="groupe_pages_ELTPARAGRAPHE000197"></a>Le groupe "Pages" permet d'insérer ou supprimer des pages dans le document PDF affiché. ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Champ%20Lecteur%20PDF_Volet_modification%20-%20HC%20N%B0003%203.gif)


Il est possible de : 

- Supprimer la page en cours. 
	Remarque : La suppression de la page en cours peut également être effectuée depuis le menu contextuel de la page (option "Supprimer la page"). 

- Supprimer un ensemble de pages. Dans ce cas, une fenêtre s'affiche permettant d'indiquer : ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Champ%20Lecteur%20PDF_Volet_modification%20-%20HC%20N%B0004.gif)


	- le numéro de la ou des pages à supprimer (séparés par un point-virgule), 

- l'intervalle de pages à supprimer : il suffit d'indiquer la première et la dernière page à supprimer. 

- l'intervalle de pages à conserver : dans ce cas, toutes les pages seront supprimées SAUF l'intervalle indiqué. 

- Importer des pages : La fenêtre de configuration de l'importation permet de : 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Champ%20Lecteur%20PDF_Volet_modification%20-%20HC%20N%B0005.gif&type=thumb)


	- sélectionner le fichier PDF contenant les pages à importer. 

- indiquer les pages à importer. Il est possible d'importer une ou plusieurs pages, un intervalle de pages ou toutes les pages. 

- indiquer la position des pages importées : au début du document, à la fin du document ou avant la page courante.  




Il est possible d'enregistrer le document en prenant en compte les modifications effectuées (bouton "Enregistrer" du groupe "Général").
<a name="NOTE4_4"></a>


### Groupe "Insertion"
<a name="groupe_insertion_ELTPARAGRAPHE000227"></a>Le groupe "Insertion" permet d'insérer un filigrane ou une signature dans le document PDF affiché. ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Champ%20Lecteur%20PDF_Volet_modification%20-%20HC%20N%B0003%204.gif)


Il est possible de : 

- Ajouter un filigrane. 

- Ajouter une ou plusieurs signatures manuscrites. 

- Ajouter une ou plusieurs signatures numériques. 




**Ajout d'un filigrane**
Lors du clic sur l'option "Ajouter un filigrane", une fenêtre s'affiche permettant d'indiquer : 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Champ_Lecteur_PDF_Signature%20-%20HC%20N%B0001.gif)


- le positionnement du filigrane.

- le texte du filigrane.

- la police, la taille et la couleur du texte.
	Attention : la police utilisée doit déjà être présente dans le fichier PDF. Dans le cas contraire, la police "Arial" sera automatiquement utilisée.  

- les options de transparence du filigrane.




**Ajout d'une ou de plusieurs signatures manuscrites**
Pour gérer les signatures manuscrites, déroulez l'option "Signature manuscrite". 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Champ_Lecteur_PDF_Signature%20-%20HC%20N%B0002.gif)
Vous pouvez : 

- Ajouter une nouvelle signature.

- Utiliser une signature existante.  

- Supprimer une signature existante. 




Lors de l'ajout d'une signature, une fenêtre spécifique est affichée. Cette fenêtre permet de : 

- Sélectionner le fichier image qui contient la signature.

- Dessiner la signature à l'aide de la souris (ou d'un stylet). 
	Dans ce cas, la couleur du stylo ne peut pas être modifiée. 

- Ecrire la signature dans une zone de texte, en choisissant la police, la taille, ...




Lors de la sélection d'une signature existante, cette signature apparaît dans la page en cours dans le champ Lecteur PDF. Elle peut être déplacée et positionnée.  
Il est ainsi possible de définir plusieurs signatures manuscrites, par exemple une signature complète et un paraphe. Le paraphe pourra être positionné en bas à droite des différentes pages.

**Ajouter une signature numérique**
Dans ce cas, il est nécessaire de posséder un certificat permettant de signer un fichier PDF. 
Lors de l'ajout d'une signature numérique, une fenêtre spécifique est affichée. 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Champ_Lecteur_PDF_Signature%20-%20HC%20N%B0003.gif)
Cette fenêtre permet de : 

- Sélectionner le certificat permettant de signer le fichier PDF. 

- Sélectionner la signature manuscrite à associer à la signature numérique. 




Attention : 

- Une signature numérique nécessite obligatoirement un certificat de signature et une signature manuscrite. 

- Pour qu'un fichier PDF signé soit reconnu par Acrobat, l'organisme émetteur du certificat doit être membre de la liste AATL (Adobe Authorized Trust List) ou du programme CDS (Certified Document Services). 




Lors de l'ouverture d'un fichier PDF signé numériquement, un message est affiché en haut à gauche du champ Lecteur PDF. Un clic sur ce message permet de consulter les caractéristiques de la signature.

**Enregistrement d'un document contenant des signatures**
Il est possible d'enregistrer le document en prenant en compte les modifications effectuées (bouton "Enregistrer" du groupe "Général"). 

**Ouverture d'un document contenant des signatures numériques** : 
Lors de l'ouverture d'un document PDF signé numériquement, une nouvelle zone apparaît sous l'onglet "Affichage" indiquant que le document est signé. 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Champ_Lecteur_PDF_Signature%20-%20HC%20N%B0004.gif)
Il est possible de cliquer sur le lien "signature" pour voir le détail de la signature. 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Champ_Lecteur_PDF_Signature%20-%20HC%20N%B0005.gif&type=thumb)
Dans cette fenêtre, un clic sur le lien correspondant au certificat permettra de consulter les informations sur du certificat.

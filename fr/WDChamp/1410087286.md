
## Description du champ Flexbox : Onglet Général
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000108"></a>
L'onglet "Général" de la fenêtre de description du champ Flexbox permet de connaître et de paramétrer les différents champs intégrés dans le champ Flexbox. 

Pour chaque champ, il est possible de définir les paramètres d'ancrage du champ correspondant à son comportement dans le flexbox. 

Remarque : Ces paramètres sont également affichés lors de l'affichage des paramètres d'ancrage du champ présent dans le champ Flexbox.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Options de l'onglet "Général"
<a name="options_onglet_general_ELTTEXTE000132"></a>
Les éléments paramétrables pour chaque champ du champ Flexbox sont les suivants : 

- **En largeur** : Répartition de l'espaces disponible en largeur en "parts".<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Flexbox_General%20-%20HC%20N%B0001%201.gif)


	- **si la largeur de la flexbox s'agrandit**, la largeur des champs sera agrandie en fonction de la place disponible en largeur (flex-grow). Les valeurs possibles sont les suivantes : 

	- 0 : pas d'agrandissement

	- 1 : agrandissement par défaut, les champs seront agrandis en proportion égale, en fonction de l'espace disponible en largeur. 

	- autre valeur : les champs seront agrandis proportionnellement à ce coefficient en fonction de l'espace disponible en largeur.

- **En hauteur** : Définition de l'alignement et de l'ancrage vertical (align-self). <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Flexbox_General%20-%20HC%20N%B0002.gif)
Les valeurs possibles sont : 

	- non défini, le champ va s'étendre verticalement (stretch)

	- auto, défini par le champ Flexbox (ancrage du champ flexbox)

	- au début (flex-start)

	- centré (center)

	- à la fin (flex-end)

	- aligner les textes \* (baseline) : Alignement selon la ligne de base du texte. 
			Remarque : "\*" indique que cette option n'est pas prise en compte sous l'éditeur. 


 Par défaut, le champ va s'étendre verticalement (stretch).

- **Ordre d'affichage** : Les champs sont affichés en fonction de l'ordre défini par ce paramètre. Si aucune valeur n'est indiqué (non défini ou 0), le champ est positionné à la droite du précédent champ. Attention : un champ qui possède un numéro d'ordre ne peut plus être déplacé dans l'éditeur : il reste à la position donnée par son ordre.

- **Ancrage personnalisé au format CSS** : Cette option permet de saisir le code CSS personnalisé indiquant l'ancrage souhaité. Ce code CSS surcharge les paramètres précédemment décrits. Par exemple : 
	
	```txt
	flex 20 1 100px;
	```
correspond à :

	- flex-grow de 20

	- flex-shrink de 1

	- flex-basis de 100px

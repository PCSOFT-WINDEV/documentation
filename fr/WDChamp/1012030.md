


## Options de génération des pages HTML
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000089"></a>
Les options de génération des pages HTML dans un projet WEBDEV peuvent être définies :

- soit dans l'onglet "Avancé" de la description du projet (sous le volet "Projet", dans le groupe "Projet", cliquez sur "Description"). Les options de génération sont prises en compte pour toutes les pages du projet.

- soit lors de la description des pages (option "Description" du menu contextuel de la page sous l'éditeur, onglet "Avancé"). Les options de génération sélectionnées sont prises en compte uniquement pour la page en cours.




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Options de génération du code HTML
<a name="options_generation_code_html_ELTTEXTE000113"></a>
Il est possible de générer :

- un code HTML compatible avec tous les navigateurs existants.

- un code HTML optimisé pour les navigateurs récents. Dans ce cas, le bouton "Avancé" permet de spécifier les différentes optimisations pouvant être effectuées.


**Remarque** : Certaines optimisations ne sont pas sélectionnées par défaut, car elles peuvent impliquer des modifications au niveau du code WLangage existant. Il est conseillé de tester l'application après chaque type d'optimisation effectuée.

WEBDEV offre la possibilité de générer un second jeu de pages "Compatible tout navigateur". Toutes les pages du projet sont alors automatiquement générées en double. Le second jeu de pages est préfixé par le caractère "_".

WEBDEV offre également la possibilité d'encoder les pages HTML en UTF8.

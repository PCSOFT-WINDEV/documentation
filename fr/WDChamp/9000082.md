
## eYe magnet : mise en forme conditionnelle des champs
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000137"></a>
Le "eYe magnet" (appelé également mise en forme conditionnelle) consiste à modifier l'aspect visuel d'un champ afin d'attirer l'attention de l'utilisateur sur ce champ. WINDEV permet de modifier cet aspect visuel automatiquement en fonction d'une série de conditions. 

Par exemple, un champ dont la saisie est obligatoire s'affichera sur un fond rouge tant que son contenu sera vide.

Le eYe magnet s'applique aux champs suivants :

- Champs de saisie.

- Champs Combo.

- Colonnes de champs Table en saisie.

- Colonnes de champs Table hiérarchique en saisie.

- Cellules d'un champ Tableau croisé dynamique.

## Comment le faire ?
<a name="comment_faire_ELTTEXTE000161"></a>
<a name="NOTE2_2"></a>


### Définir un nouvel eYe magnet
<a name="definir_nouvel_eye_magnet_ELTPARAGRAPHE000031"></a>Pour définir un nouvel eYe magnet (mise en forme conditionnelle) :

1. Affichez la fenêtre de description du champ (option "Description" du menu contextuel).

2. Dans l'onglet "UI", cliquez sur le lien "eYe magnet: &lt;Aucune mise en forme&gt;" ou sur la description de l'eYe magnet actuel (si un eYe magnet a déjà été défini). 
	Remarque : Pour les cellules d'un champ Tableau croisé dynamique, dans l'onglet "Contenu", cliquez sur la valeur voulue ("Valeur à afficher dans le tableau"). Dans la zone "Affichage" des caractéristiques, cliquez sur le lien "eYe magnet: &lt;Aucune mise en forme&gt;" ou sur la description de l'eYe magnet actuel (si un eYe magnet a déjà été défini). 

3. Si le champ n'a pas encore de mise en forme définie, la fenêtre de création d'une mise en forme conditionnelle s'ouvre directement. 
	Si une ou plusieurs mises en forme existent déjà pour ce champ, une fenêtre présentant la liste des eYe magnets s'affiche. 

4. Cliquez sur le bouton "Ajouter un eYe magnet" pour ouvrir la fenêtre de description d'un eYe magnet.




La fenêtre de description des conditions et des actions est la suivante : 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=eyemagnet_fen_edition.gif)


- Les boutons ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=eyemagnet_btn_plus.gif)
 et ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=eyemagnet_btn_supprimer.gif)
 permettent d'ajouter ou de supprimer des conditions ou des actions.

- Lorsque toutes les conditions sont réalisées simultanément, toutes les actions associées sont automatiquement effectuées.

- Il est possible de définir plusieurs eYe magnets sur le même champ pour réaliser des actions différentes en fonction des conditions.

- Le eYe magnet est appliqué après les traitements du champ (Entrée en saisie, A chaque modification, etc.). Il n'est pas possible de surcharger ses effets par programmation.





<a name="NOTE2_3"></a>


### Modifier un eYe magnet existant
<a name="modifier_eye_magnet_existant_ELTPARAGRAPHE000062"></a>Pour modifier un eYe magnet existant :

1. Affichez la fenêtre de description du champ (option "Description" du menu contextuel).

2. Dans l'onglet "UI", cliquez sur la description de l'eYe magnet actuel pour ouvrir la fenêtre d'édition des eYe magnets.

3. Cliquez sur le bouton ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=eyemagnet_btn_modif.gif)
 pour ouvrir la fenêtre d'édition.

4. Modifiez les caractéristiques du eYe magnet. Pour plus de détails, consultez [Définir un nouvel eYe magnet](#NOTE2_2).



<a name="NOTE2_4"></a>


### Supprimer un eYe magnet
<a name="supprimer_eye_magnet_ELTPARAGRAPHE000079"></a>Pour supprimer un eYe magnet existant : 

1. Affichez la fenêtre de description du champ (option "Description" du menu contextuel).

2. Dans l'onglet "UI", cliquez sur la description de l'eYe magnet actuel pour ouvrir la fenêtre d'édition des eYe magnets.

3. Cliquez sur le bouton ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=eyemagnet_btn_supprimer.gif)
 de l'eYe magnet à supprimer.

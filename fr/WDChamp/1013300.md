<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000139"></a>


### Champ d'un site
<a name="champ_site_ELTPARAGRAPHE000011"></a>Le champ Plan du site permet à l'internaute :

- d'avoir une vision d'ensemble des pages présentes dans le site.

- d'accéder directement à une page spécifique.




Le champ Plan du site permet également d'améliorer le référencement d'un site.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Création d'un champ Plan du site
<a name="creation_champ_plan_site_ELTTEXTE000163"></a>


### Créer un champ Plan du site
<a name="creer_champ_plan_site_ELTPARAGRAPHE000025"></a>Pour créer un champ Plan du site :

1. Sous le volet "Création", dans le groupe "Navigation", cliquez sur "Plan du site".

2. Cliquez dans la page à la position où le champ doit être créé. Le champ apparaît en création. Si un chemin de navigation est défini pour le site en cours, le champ Plan du site affiche les données correspondantes.




Pour afficher les caractéristiques du champ, sélectionnez l'option "Description" dans le menu contextuel du champ.
<a name="NOTE2_2"></a>


### Caractéristiques d'un champ Plan du site
<a name="caracteristiques_champ_plan_site_ELTPARAGRAPHE000049"></a>La fenêtre de description d'un champ "Plan du site" permet de :

- Construire si nécessaire le plan de navigation du site (onglet "Général").
	Si votre site ne contient pas de champ de type "Chemin de navigation", il est possible d'éditer et de construire [le plan de navigation du site](../WDChamp/1013301.md).

- Indiquer les différentes parties du plan du site à afficher (onglet "Général").
	En effet, il est possible d'utiliser plusieurs champs de type "Plan du site" dans une même page : il est ainsi possible de les mettre côte à côte et d'utiliser des modes de présentation différents. Si aucun élément n'est sélectionné, tout le plan de navigation sera affiché dans le champ (dans la limite du nombre de niveaux visibles).

- Indiquer le nombre de colonnes pour chaque niveau ainsi que l'indentation de chaque niveau (onglet "Détail").

- Indiquer les styles utilisés pour chaque niveau du plan du site (onglet "Style").

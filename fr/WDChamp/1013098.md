## Présentation
<a name="presentation_ELTTEXTE000083"></a>


### Champ d'une page dynamique ou statique
<a name="champ_une_page_dynamique_statique_ELTPARAGRAPHE000016"></a>Le champ Flash permet d'afficher une animation Flash® dans une page WEBDEV.

**Pour créer un champ de type Flash :**

1. Sous le volet "Création", dans le groupe "Intégration", déroulez "Plug-in" puis cliquez sur "Champ Flash".

2. Cliquez dans la page à la position où le champ doit être créé. Le champ apparaît en création.

3. Dans la description du champ (onglet "Général"), associez l'animation Flash® (fichier ".swf") au champ Flash.




**Remarque** : il est conseillé de placer l'animation Flash® dans le répertoire des images (répertoire &lt;Nom_Projet&gt;_WEB).

Pour afficher les caractéristiques du champ, sélectionnez l'option "Description" dans le menu contextuel du champ.

## Présentation du Champ Indicé
<a name="presentation_champ_indice_ELTTEXTE000114"></a>
Un champ est indicé si il est composé de plusieurs champs du même nom. Chaque champ a le même nom, mais a un indice différent. Cet indice permet de manipuler simplement le champ voulu par programmation.

**Remarque** : Les champs indicés permettent de gérer simplement les rubriques texte de type Tableau définies sous l'éditeur d'analyses.

Quels champs peuvent être indicés ? Les champs de saisie et les colonnes des champs Table (numérique, texte, monétaire, date et heure).

**Attention** :

- Les colonnes indicées doivent toutes se trouver dans le même champ Table.

- Les champs indicés ne sont utilisables que dans l'éditeur de fenêtres. Les champs indicés ne sont pas utilisables dans l'éditeur d'états.

## Création d'un Champ Indicé
<a name="creation_champ_indice_ELTTEXTE000138"></a>


### Créer un champ de saisie indicé
<a name="creer_champ_saisie_indice_ELTPARAGRAPHE000027"></a>Pour créer un champ de saisie indicé :

1. Créez un nouveau champ de saisie. 

2. Donnez-lui le nom d'un champ de saisie existant. Les champs sont alors indicés.



<a name="NOTE2_2"></a>


### Pour créer une colonne indicée
<a name="pour_creer_une_colonne_indicee_ELTPARAGRAPHE000037"></a>Il suffit de créer une nouvelle colonne dans un champ Table avec le même nom qu'une colonne existante. 

- Le nom des deux colonnes doit se terminer par un chiffre (par exemple MaColonne1). 

- Lors de la création de la seconde colonne, WINDEV propose de créer un champ indicé. Si vous acceptez, l'indice apparaît dans la fenêtre de description du champ Table.



<a name="NOTE2_3"></a>


### Champ indicé et rubrique tableau
<a name="champ_indice_rubrique_tableau_ELTPARAGRAPHE000047"></a>Après sa création, le champ indicé peut être lié à un des éléments d'une rubrique tableau.

## Notes
<a name="notes_ELTTEXTE000174"></a>


### Champ indicé et rubrique tableau
<a name="champ_indice_rubrique_tableau_ELTPARAGRAPHE000056"></a>Lors de l'utilisation d'une rubrique tableau, la présence de tous les champs indicés correspondant à la rubrique dans la même fenêtre n'est pas obligatoire (contrairement à WINDEV 5.5). Si la rubrique tableau contient 6 éléments, seuls trois éléments peuvent se trouver dans la fenêtre.
<a name="NOTE3_2"></a>
Dans une fenêtre, un champ indicé peut être placé :

- dans une fenêtre (au premier plan).

- sur un plan.

- sur un onglet.

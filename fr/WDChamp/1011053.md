


## Bloc Début de document
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000114"></a>
Le bloc **Début de document** est imprimé une seule fois au début de l'exécution de l'état. Ce bloc contient généralement le titre général du document, la date du jour et éventuellement l'heure d'édition.

Dans l'événement associé à ce type de bloc, il est possible par exemple d'afficher une fenêtre avertissant du début de l'impression.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Création du bloc Début de document
<a name="creation_bloc_debut_document_ELTTEXTE000138"></a>
Pour créer un bloc Début de document :

1. Réalisez une des manipulations suivantes sous l'éditeur d'états :

	- sous le volet "Création", dans le groupe "Structure", déroulez "Blocs".

	- sélectionnez l'option "Ajouter un bloc" du menu contextuel de l'état (clic droit sur l'état en cours de création).

	- cliquez sur le bouton "Ajouter un bloc" dans l'onglet "Blocs" de la description de l'état.




2. Sélectionnez le type "Début de document".




<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Impression du bloc Début de document
<a name="impression_bloc_debut_document_ELTTEXTE000162"></a>


### Imprimer le bloc Début de document avant ou après le bloc Haut de page
<a name="imprimer_bloc_debut_document_avant_apres_bloc_haut_page_ELTPARAGRAPHE000056"></a>Par défaut, le bloc **Début de document** est imprimé avant le bloc *En-tête de page* sur la première page de l'état. L'éditeur d'états permet de modifier cet ordre d'impression par défaut.

- Méthode 1 : 

	1. Affichez la fenêtre de description du bloc "Début de document" (option "Description du bloc" du menu contextuel).

	2. Dans l'onglet "Général", sélectionnez l'option "Imprimer le bloc "Début de document" APRÈS le bloc "En-tête de page" sur la première page".




- Méthode 2 : 

	1. Affichez la fenêtre de description de l'état (option "Description de l'état" du menu contextuel).

	2. Dans l'onglet "Blocs", dans la section "Début de document", cochez l'option "APRES le bloc "En-tête de page"".

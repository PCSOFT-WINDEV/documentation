## Présentation du champ Bouton
<a name="presentation_champ_bouton_ELTTEXTE000229"></a>


### Champ d'une fenêtre ou d'une page (statique ou dynamique)
<a name="champ_une_fenetre_une_page_statique_dynamique_ELTPARAGRAPHE000011"></a>Le champ Bouton permet à l'utilisateur ou l'internaute de lancer une action spécifique par un simple clic de souris.

### Champ Bouton dans une fenêtre
<a name="champ_bouton_dans_une_fenetre_ELTPARAGRAPHE000061"></a>Le champ Bouton permet à l'utilisateur de lancer une action spécifique par un simple clic de souris.

Un champ Bouton peut être composé :

- de texte, ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=BOUTON_TEXTE.gif)
![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Les boutons ne peuvent pas avoir un libellé multiligne.

- d'image, ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Btn_img.gif)


- de texte et d'image.![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=BTN_IMG_TXT.gif)





WINDEV et WINDEV Mobile offrent de nombreuses possibilités pour personnaliser les boutons, leur look ou leur effet. 

Il est ainsi possible de : 

- définir le [type du bouton](../WDChamp/1013093.md). Chaque type de bouton permet de réaliser une action spécifique.

- créer des [boutons poussoir](../WDChamp/1013073.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) créer des [boutons à flèche](../WDChamp/1013163.md).

- définir des images pour illustrer le bouton ou personnaliser sa forme. Pour plus de détails, consultez [Images et boutons](../WDChamp/1013075.md). 

- créer des boutons animés. Pour plus de détails, consultez [Images et boutons](../WDChamp/1013075.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) utiliser du RTF pour mettre en forme le libellé du bouton. 
	**Remarque** : Pour utiliser du RTF dans un champ Bouton, cochez l'option "RTF" dans l'onglet "Général" de la description du champ. La barre de configuration RTF apparaît alors au-dessus du libellé du champ pour le configurer facilement. Il est également possible de copier directement un code RTF dans le libellé du champ.

### Champ Bouton dans une page
<a name="champ_bouton_dans_une_page_ELTPARAGRAPHE000120"></a>Un champ Bouton peut avoir différents aspects :


|   |   |
| --- | --- |
| <br><br>- texte<br><br><br> | ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=BOUTON_TEXTE.gif)<br> |
| <br><br>- Image fixe ou avec survol<br><br><br> | Une ou deux images sont associées au bouton :<br><br>- une image pour le bouton dans son état ***"normal"***,<br><br>- une image pour son état ***"survolé"*** (lors du passage du curseur de la souris).<br><br><br>Dans ce cas, des images indépendantes sont utilisées. |
| <br><br>- Image à plusieurs états<br><br><br> | Des dessins différents sont associés au bouton. Ces dessins sont regroupés dans une image (appelée "planche d'images"). Il est possible d'utiliser jusqu'à 5 dessins. Les dessins correspondent aux états suivants du bouton : normal / enfoncé / grisé / focus / survol. |


L'aspect du champ Bouton est défini dans l'onglet "Style" de la fenêtre de description du champ, dans l'élément "Bouton (Images)".

Plusieurs informations permettent de déterminer l'action du bouton et les différents événements exécutés lors du clic sur le champ Bouton :

- [le type du champ Bouton](../WDChamp/1013074.md).

- [l'action définie pour le champ Bouton](../WDChamp/1013071.md).

## Créer un champ de type Bouton
<a name="creer_champ_type_bouton_ELTTEXTE000301"></a>
Pour créer un champ de type Bouton : 

1. Sous le volet "Création", dans le groupe "Champs usuels", cliquez sur "Bouton" (ou cliquez sur ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_bouton.gif)
).

2. Cliquez dans la fenêtre ou dans la page à la position où le champ doit être créé. Le champ apparaît en création.




**Astuce** : Il est également possible de créer directement un champ Bouton "pré-formaté" (qui peut contenir du code si nécessaire) en déroulant "Bouton". La liste des champs Bouton disponibles avec leur aperçu au gabarit du projet apparaît. Il suffit de cliquer sur le champ Bouton voulu pour le créer dans l'éditeur. 

**Pour afficher les caractéristiques du champ**, sélectionnez l'option "Description" dans le menu contextuel du champ.

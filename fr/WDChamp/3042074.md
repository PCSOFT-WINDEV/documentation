


## Les graphes de type "Aire"
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000191"></a>
Le graphe de type Aire permet de mettre en évidence l'amplitude d'un changement.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Caractéristiques d'un graphe de type Aire
<a name="caracteristiques_graphe_type_aire_ELTTEXTE000215"></a><br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=GenAire.gif&type=thumb)
Dans ce graphe :

**1.** **Point avec sa valeur.**
La valeur du point est affichée si  les conditions suivantes sont réunies :

- l'option "Afficher les valeurs" est cochée dans l'onglet "Détail" de la description du champ Graphe.

- le type de point est défini pour chaque série dans l'onglet "Séries" de la fenêtre de description du point. Si le type de point est "Non dessiné", le point ne sera pas mis en valeur et sa valeur ne sera pas affichée.




La taille des points est définie dans l'onglet "Général" de la fenêtre de description du champ.  

**2. Titre des axes**. 
Le titre des axes horizontal et vertical est affiché si le titre a été défini dans l'onglet "Général" de la description du champ Graphe. Ce paramètre peut être défini également par programmation grâce à la fonction [grTitreAxe](../WDLang3/3042057.md).

**3. Titre du graphe**. 
Les caractéristiques du titre du graphe sont saisies dans l'onglet "Détail" de la description du champ (bouton "Editer le titre du graphe").
Il est possible de définir le titre du graphe, sa position et son style.

**4. Légende**. 
Les caractéristiques de la légende sont saisies dans l'onglet "Détail" de la description du champ (bouton "Editer la légende du graphe").
Il est possible de définir sa position et son style.

**5. Etiquettes de séries**. 
Le libellé de la légende correspond au libellé des différentes séries affichées dans le graphe (définie dans l'onglet "Séries").

**6. Couleur de série**. 
Pour chaque série, il est possible de définir une couleur dans l'onglet "Séries" de la description du champ. Par défaut, la couleur utilisée est automatique.

**7. Etiquette de catégorie et graduations**.
Les étiquettes de catégorie peuvent être définies dans l'onglet "Séries" de la description du champ.

Par défaut, la graduation est calculée automatiquement. L'onglet "Général" de la description du champ permet :

- de définir une graduation spécifique.

- de définir une échelle des abscisses en fonction du temps. Cette option fonctionne uniquement avec des données de type Date ou DateHeure provenant d'un champ Table ou d'une rubrique HFSQL. Cette option ne peut pas être utilisée avec une série remplie par programmation.

- de définir une échelle des ordonnées logarithmique.




Il est également possible de demander à utiliser l'échelle secondaire pour une ou plusieurs séries (onglet "Séries").

**8. Quadrillage**. 
Il est possible d'afficher le quadrillage grâce aux options "Quadrillage" et "Quadrillage" dans l'onglet "Général" de la description du champ.

**9. Ascenseurs permettant de zoomer**. 
Il est possible d'autoriser le zoom dans un graphe dans l'onglet "Détail" de la description du champ. Cette option est disponible uniquement en WINDEV.

**Autres caractéristiques** : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Les graphes de type "Aire" peuvent être affichés avec une animation. Pour animer ce type de graphe, il suffit de cocher l'option "Activer les animations" dans l'onglet "Détail" de la description du champ. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Les graphes de type "Aire" peuvent être affichés en temps réel. Pour plus de détails, consultez [Graphe avec affichage en temps réel](../WDChamp/1000021037.md).

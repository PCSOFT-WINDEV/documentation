


## Modes d'impression d'un état
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000168"></a>
<a name="Modes_impression"></a>
Plusieurs modes d'impression sont disponibles pour un état. Ces différents modes sont présentés dans le tableau suivant :


|   |   |   |   |
| --- | --- | --- | --- |
|   | [Impression lors du test d'un état sous l'éditeur](../WDChamp/1011027.md) | [Impression d'un état depuis le visualisateur de rapports](../WDChamp/1011085.md)<br>(WINDEV uniquement) | Impression d'un état par programmation (fonctions [iDestination](../WDLang5/3046074.md) et [iImprimeEtat](../WDLang5/3046032.md)) |
| Impression papier | X | X | X |
| Aperçu avant impression<br>Visualisateur de rapports | X | - | X (WINDEV uniquement) |
| Mini-aperçu |   |   | X (WINDEV uniquement) |
| Fichier HTML (avec feuille de styles) | X | X | X |
| Fichier HTML (sans feuille de styles) |   |   | X |
| Fichier RTF | X | X | X |
| Fichier XML |   | X | X |
| Fichier PDF | X | X | X |
| Fichier texte |   |   | X |
| Envoi d'un fax |   |   | X |
| Fichier PCL |   |   | X |
| Fichier XLS |   |   | X |


![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) **WINDEV Mobile** : Lors du test d'un état sous l'éditeur, toutes les options sont disponibles. Cependant, lors de l'impression d'un état par programmation dans une application iOS, Android ou en mode Universal Windows App, seules les impressions sur un fichier PDF sont disponibles. **Remarque** : Par défaut, le mode d'impression de l'état est automatiquement demandé lors du test de chaque état dans une fenêtre spécifique. Pour ne pas afficher cette fenêtre lors du test de l'état, cochez l'option "Ne plus afficher" dans la fenêtre du choix du mode d'impression. Pour plus de détails, consultez [Ne plus afficher un message](../Editeurs/2020011.md).


- ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=WDEtats.gif) ***Exemples didactiques (WINDEV)*** : **WD Etats** <br>Cet exemple montre les différentes méthodes pour réaliser un état :<br> <br>- impressions basées sur différentes sources de données (requêtes, variables, ...)<br>- impressions basées sur des champs (Table, Tableur, TCD, ...)<br>- impression d'états composés<br>- impressions spécifiques (portrait / paysage, état avec filigrane, état avec code-barres, ...)




## Manipuler un champ iFrame par programmation
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000127"></a>
WEBDEV permet de manipuler un champ IFrame directement par programmation. Il suffit d'utiliser directement la variable du champ IFrame dans le code.

La variable du champ IFrame correspond au nom du champ IFrame. Le type de la variable est une chaîne de caractères.

**Important** : Un champ IFrame est programmable en code serveur.

<a name="NOTE2_1"></a>


## Afficher une page dans un champ IFrame
<a name="afficher_une_page_dans_champ_iframe_ELTTEXTE000151"></a>


### Changer la cible du champ IFrame sous l'éditeur
<a name="changer_cible_champ_iframe_sous_editeur_ELTPARAGRAPHE000022"></a>Le changement de cible d'une champ IFrame peut se faire très simplement par exemple par l'intermédiaire d'un bouton ou d'un lien.

Ce changement peut être réalisé uniquement si la cible du champ IFrame appartient au site en cours.

Il suffit de créer un bouton ou un lien avec les caractéristiques suivantes :

- Opération sur les champs : "Aucune".

- Action : "Afficher la page XXX". Il suffit de sélectionner la page à afficher.

- Destination : "Dans un IFrameXXX". Il suffit de sélectionner le nom du champ IFrame.




**Remarque** : Si la cible du champ IFrame n'appartient pas au site en cours, le navigateur Internet ne permet pas de modifier cette cible (pour des raisons de sécurité).
<a name="NOTE2_2"></a>


### Changer la cible du champ IFrame par programmation
<a name="changer_cible_champ_iframe_par_programmation_ELTPARAGRAPHE000039"></a>Pour changer la cible du champ IFrame par programmation, il suffit d'utiliser :

- la fonction WLangage [PageAffiche](../WDLang2/3058008.md) en lui spécifiant l'alias du champ IFrame dans lequel la page doit être affichée (code navigateur).

- la fonction WLangage [ChangeDestination](../WDLang2/3058005.md) en lui spécifiant l'alias du champ IFrame dans lequel la page doit être affichée (code navigateur) puis la fonction [PageAffiche](../WDLang2/3058008.md) en code Serveur.




Ce changement peut être réalisé uniquement si la cible du champ IFrame appartient au site en cours.

**Remarque** : Si la cible du champ IFrame n'appartient pas au site en cours, le navigateur Internet ne permet pas de modifier cette cible (pour des raisons de sécurité).

## Propriétés spécifiques associées au champ IFrame
<a name="proprietes_specifiques_associees_champ_iframe_ELTTEXTE000181"></a>
Les propriétés suivantes sont spécifiques à la gestion des caractéristiques d'un champ IFrame par programmation.


|   |   |
| --- | --- |
| [Type](../Proprietes/2510131.md) | Permet de connaître le type d'un objet |
| [ValeurInitiale](../Proprietes/2510128.md) | Permet de connaître la valeur initiale d'un champ. Pour un champ IFrame, cette propriété correspond soit au nom de la page du site, soit à une chaîne vide si le champ est associé à une page d'un autre site. |

Pour connaître la liste complète des propriétés du WLangage pouvant être utilisées avec un champ IFrame, consultez [Propriétés associées à un champ IFrame](../WDChamp/1013271.md).

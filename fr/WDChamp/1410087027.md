## Evénements gérés par défaut
<a name="evenements_geres_par_defaut_ELTTEXTE000121"></a>
WEBDEV gère par défaut les événements suivants (ordre d'apparition sous l'éditeur de code) :

| Evénement | Condition d'exécution |
| --- | --- |
| Action (Navigateur) | Code pouvant être appelé directement depuis le Composant Web. Permet d'exécuter une action via le code HTML ou Javascript du champ. |
| Action (Serveur) | Code pouvant être appelé directement depuis le Composant Web. Permet d'exécuter une action via le code HTML ou Javascript du champ. |
| Affectation de la propriété Valeur (Navigateur) | Code permettant d'affecter une valeur au champ Composant Web. |
| Chargement (onload) (Navigateur) | Exécuté lors du chargement du composant. Par défaut, cet événement contient la ligne de code : <br><br><pre><code>ExécuteTraitement(<Nom du champ>, trtAffectationValeur)</code></pre><br>Cette ligne de code permet d'exécuter l'événement "Affectation de la propriété Valeur" du champ. <br>Il est ainsi possible par exemple de donner une valeur au champ. |
| Déclaration globales (Serveur) | Déclaration des variables globales liées au champ. |
| Formulaire envoyé (onsubmit)  (Navigateur) | Exécuté : <br><br>- lors de la validation de la page (submit). <br><br>- lors de l'exécution d'une action serveur du champ. <br><br><br><br><br>Par défaut, cet événement contient la ligne de code : <br><br><pre><code>ExécuteTraitement(<Nom du champ>, trtRécupérationValeur)</code></pre><br>Cette ligne de code permet d'exécuter l'événement "Récupération de la propriété Valeur" du champ. <br>Il est ainsi possible par exemple de mémoriser la valeur au champ. |
| Initialisation (Serveur) | Exécuté à l'ouverture de la page. |
| Récupération de la propriété Valeur (Navigateur) | Code permettant de définir le contenu de la propriété Valeur du champ Composant Web. |

## Evénements supplémentaires
<a name="evenements_supplementaires_ELTTEXTE000145"></a>
Plusieurs événements supplémentaires sont affichés sous l'éditeur de code : 

- Evénement CSS : "Déclaration des styles CSS" : Code correspondant au code CSS du composant Web. Ce code a été saisi lors de la création du champ. 

- Evénement JS : "Initialisation du composant Web" : Code JS du composant Web. Ce code a été saisi lors de la création du champ.




## FAA : Augmenter la hauteur des lignes des champs Table ou Zone répétée
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000102"></a>
Dans les champs Table, l'utilisateur peut redimensionner la hauteur des lignes, en affichage ou en saisie.<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=FAA_Redim-hauteur-ligne.gif)


Cette fonctionnalité est également disponible : 

- dans les champs Zone répétée.

- dans les champs Table hiérarchique. 




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000126"></a>


### Autoriser l'utilisateur à redimensionner la hauteur des lignes d'un champ Table (ou Table hiérarchique)
<a name="autoriser_utilisateur_redimensionner_hauteur_des_lignes_champ_table_table_hierarchique_ELTPARAGRAPHE000024"></a>Pour autoriser l'utilisateur à redimensionner la hauteur des lignes d'un champ Table (ou Table hiérarchique) : 

1. Affichez la description du champ.

2. Dans l'onglet "Détail" de la description du champ, cochez l'option "Hauteur des lignes redimensionnable".

3. Validez.



<a name="NOTE2_2"></a>


### Autoriser l'utilisateur à redimensionner la hauteur des répétitions d'un champ Zone répétée
<a name="autoriser_utilisateur_redimensionner_hauteur_des_repetitions_champ_zone_repetee_ELTPARAGRAPHE000039"></a>Pour autoriser l'utilisateur à redimensionner la hauteur des répétitions d'un champ Zone répétée : 

1. Affichez la description du champ Zone répétée.

2. Dans l'onglet "Détail" de la description du champ Zone répétée, cochez l'option "Redimensionnable" pour la hauteur des lignes.

3. Validez.



<a name="NOTE2_3"></a>


### Désactiver le redimensionnement des lignes
<a name="desactiver_redimensionnement_des_lignes_ELTPARAGRAPHE000050"></a>Pour ne pas permettre à l'utilisateur de redimensionner la hauteur des lignes d'un champ Table, Table hiérarchique ou Zone répétée, utilisez la fonction [FAADésactive](../WDLang1/1000022018.md) avec la constante *faaRedimHauteurLigne*.

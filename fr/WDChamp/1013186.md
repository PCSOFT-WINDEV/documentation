## Présentation du champ Liste image
<a name="presentation_champ_liste_image_ELTTEXTE000207"></a>
Le champ Liste image permet d'afficher une liste d'éléments graphiques (images, logo, ...) et de sélectionner un ou plusieurs éléments dans cette liste.

Plusieurs modes d'affichage sont disponibles : 

- **Mode Liste image** : La liste image est déroulée, c'est-à-dire que plusieurs éléments de la liste sont affichés dans la fenêtre.<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ExempleChampListeImage.gif)


- **Mode Liste classique** : Les images ne sont pas affichées dans la liste image. Seuls les libellés sont affichés. 

- ![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png) **Mode Panorama** : Les images apparaissent comme si elles étaient présentes sur un mur incurvé. <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Listeimage_panorama.gif)





Ces éléments peuvent :

- être déterminés par programmation.

- provenir d'un fichier de données ou d'une requête.




**Remarque** : Une liste image peut être utilisée pour réaliser un "Drag and Drop" automatique. Pour plus de détails, consultez [Gestion du "Drag and Drop"](../WDLang1/3030007.md).

## Création du champ Liste image
<a name="creation_champ_liste_image_ELTTEXTE000231"></a>
Pour créer un champ de type Liste image :

1. Sous le volet "Création", dans le groupe "Données", déroulez "Table et Liste" et sélectionnez "Liste Image".

2. Cliquez dans la fenêtre à la position où le champ doit être créé. L'assistant de création d'un champ Liste image se lance automatiquement.




Pour afficher les caractéristiques du champ, sélectionnez l'option "Description" dans le menu contextuel du champ.
![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png) 

### Spécificités du mode Panorama
<a name="specificites_mode_panorama_ELTPARAGRAPHE000064"></a>

- Le champ Liste image peut afficher au maximum 20 images. Si la liste contient plus d'éléments, les éléments supplémentaires sont présents mais ne sont pas visibles par l'utilisateur. 

- Plusieurs fonctionnalités ne sont pas disponibles : gestion des ascenseurs, ...

## Vocabulaire
<a name="vocabulaire_ELTTEXTE000261"></a>
Pour manipuler simplement un champ Liste image, il est nécessaire de connaître le vocabulaire suivant :

| Nom | Définition |
| --- | --- |
| Élément | Un champ Liste image est composé de plusieurs éléments. Chaque élément est constitué d'une image et d'un libellé. |
| Mode d'affichage | Un champ Liste image peut être affiché :<br><br>- en mode "Liste image" : le champ Liste image affichera des images et des libellés.<br><br>- en mode "Liste classique" : le champ Liste image affichera uniquement des libellés.<br><br>- ![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png) en mode "Panorama" : le champ Liste image affichera les images avec un effet incurvé.<br><br><br> |
| Bandeau de sélection | Permet de visualiser l'élément sélectionné. Le plus souvent, une couleur spécifique est utilisée. |
| Liste image Fichier | Le champ Liste image est basé sur un fichier de données ou une requête. Les éléments du champ Liste image correspondent à des enregistrements du fichier de données.<br>Deux types de champ Liste image fichier sont disponibles :<br><br>- ***Champ Liste image fichier "Chargé en mémoire"*** : le contenu du champ Liste image est chargé en mémoire puis affiché. L'accès aux différents éléments du champ Liste image est plus rapide. Ce mode est réservé pour des fichiers de données de moins de 100 000 enregistrements.<br><br>- ***Champ Liste image fichier "Accès direct (sans limite)"*** : Le champ Liste image affiche directement le contenu du fichier de données ou de la requête. L'enregistrement en cours correspond à l'élément sélectionné. Chaque déplacement dans le champ Liste image entraîne un accès au fichier de données lié. Ce type de champ Liste image est réservé aux fichiers de données de plus de 100 000 enregistrements.<br><br><br> |
| Liste image par programmation | Le champ Liste image n'est basé ni sur un fichier de données ni sur une requête. Les éléments affichés dans le champ Liste image sont déterminés par le développeur par programmation. |
| Liste image monosélection | L'utilisateur peut uniquement sélectionner un élément dans le champ Liste image. |
| Liste image multisélection | L'utilisateur peut sélectionner en même temps plusieurs éléments dans le champ Liste image (il suffit d'utiliser la touche Ctrl). |
| Défilement Vertical / Horizontal | Sens de défilement du champ Liste image.<br>Si le nombre d'éléments affichés dans le champ Liste image dépasse le nombre d'éléments visibles, un ascenseur vertical apparaît automatiquement. <br>Si la taille des éléments est supérieure à la largeur du champ Liste image, un ascenseur horizontal apparaît automatiquement. |
| Liste image sur variable | Le champ Liste image est basée sur une variable (globale au projet, à la fenêtre ou à la page, membre de classe, ...). |

## Permuter
<a name="permuter_ELTTEXTE000285"></a>


### Champ Liste image / champ Zone répétée
<a name="champ_liste_image_champ_zone_repetee_ELTPARAGRAPHE000141"></a>Il est possible de transformer automatiquement un champ Liste image en un champ Zone répétée : 

1. Sélectionnez le champ Liste image à transformer. 

2. Sous le volet "Modification", dans le groupe "Transformations", déroulez "Refactoring et permutations" et sélectionnez "Liste Image vers Zone répétée".





- ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=LechampListeImage.gif) ***Exemples unitaires (WINDEV)*** : **Le champ Liste Image** <br>Utilisation du champ Liste Image de WINDEV.<br>Le champ Liste Image permet d'afficher le contenu d'une liste sous forme de vignettes.<br>Tout comme les champs Liste ou Table, le champ Liste Image peut être "mémoire" (rempli par programmation) ou "fichier" (lié à une base de données).

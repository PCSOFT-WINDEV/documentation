


## Gestion des polices d'icônes
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les **icônes de police** permettent d'améliorer la présentation d'un titre, d'un libellé ou d'un texte. Il est possible en effet d'inclure des images (icônes) dans le texte d'un libellé HTML.

Tous les champs WEBDEV incluant un libellé possèdent par défaut un libellé au format HTML : libellé, bouton, ... Ce libellé utilise un éditeur HTML intégré. Cet éditeur HTML permet de personnaliser le texte (police, taille, couleur, etc.) grâce aux options du volet "Texte" présent dans le ruban : <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Gestion_des_polices_icones%20-%20HC%20N%B0001.gif)
L'éditeur HTML intégré inclut également un bouton permettant d'ajouter une icône dans le texte.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Ajouter une icône
<a name="ajouter_une_icone_ELTTEXTE000110"></a>
Pour ajouter une icône dans un libellé HTML : 

1. Positionnez le curseur de la souris dans le texte HTML à modifier.

2. Cliquez sur le bouton **Icônes de police** ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Gestion_des_polices_icones%20-%20HC%20N%B0001%201.gif). 

3. La fenêtre suivante s'affiche. <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Gestion_des_polices_icones%20-%20HC%20N%B0002.gif&type=thumb)


4. Sélectionnez la famille d'icônes dans la liste des polices.

5. Double-cliquez sur l'icône voulue ou sélectionnez l'icône et validez la fenêtre. L'icône est ajoutée dans le libellé HTML à la position du curseur.




Attention : La configuration de projet courante doit gérer les chaînes Unicode en exécution. Cette option est disponible dans la fenêtre de description de la configuration.

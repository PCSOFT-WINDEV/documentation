
## Proprietés associées à un champ Agenda
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000142"></a>
Il est possible d'utiliser des propriétés WLangage sur un champ Agenda :

- dans une fenêtre.

- dans une page.

- dans un état.

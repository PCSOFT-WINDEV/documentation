## Présentation du champ Réseau social
<a name="presentation_champ_reseau_social_ELTTEXTE000196"></a>
Le champ "Réseau social" est une barre d'outils composée de pictogrammes faisant le lien vers les principaux réseaux sociaux du Web :

- Facebook, 

- Tweeter, 

- Linkedln, 

- Tumblr,

- Pinterest.

- Instagram.

- TikTok.

- WhatsApp.




Ce champ est en général inséré dans un site ou en marge d'un article. 

Ce champ permet deux types d'actions pour l'internaute : 

- Partager son expérience avec la communauté de son réseau social. 

- Evaluer une page ou un site à l'aide des fonctionnalités "J'aime" de Facebook ou "+1" de Google+. 




**Remarque** : Le champ Réseau social n'est pas disponible dans les zones répétées Ajax et navigateur. Le champ Réseau Social est disponible uniquement dans les zones répétées classiques.

## Créer un champ Réseau social
<a name="creer_champ_reseau_social_ELTTEXTE000220"></a>
Pour créer un champ de type Réseau social :

1. Sous le volet "Création", dans le groupe "Intégration", déroulez "Réseaux sociaux" et sélectionnez le type de réseau social à utiliser. <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Champ%20R%E9seau%20social%20-%20HC%20N%B0001.gif)


2. Cliquez dans la page à la position où le champ doit être créé. Le champ apparaît en création.




Pour afficher les caractéristiques du champ : 

1. Sélectionnez le champ correspondant au réseau social voulu. 

2. Sélectionnez l'option "Description" dans le menu contextuel du champ.




Remarque : Pour créer un champ Réseau social "WhatsApp" : 

1. Cliquez directement sur "Réseaux sociaux". 

2. Cliquez dans la page à la position où le champ doit être créé.

3. Sélectionnez le type de réseau social à gérer dans la fenêtre de description du champ.

## Caractéristiques du champ Réseau social
<a name="caracteristiques_champ_reseau_social_ELTTEXTE000244"></a>
L'onglet "Général" de la description du champ permet de définir : 

- Le **type** du lien à afficher. Pour chaque réseau social, un ou plusieurs types de boutons peuvent être affichés : <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Champ%20R%E9seau%20social%20-%20HC%20N%B0002.gif&type=thumb)


- La **cible** de l'action sur le réseau social. Cette cible peut être : 

	- le domaine. Le clic sur le champ Réseau social créera un lien vers le domaine actuel du site en cours. 

	- le site. Le clic sur le champ Réseau social créera un lien vers le site en cours (c'est-à-dire l'adresse actuelle sans le nom de la page). 

	- la page courante. Le clic sur le champ Réseau social créera un lien vers la page du site WEBDEV courante. Si le site WEBDEV est un site dynamique, le lien sera effectué vers le domaine. 

	- une cible personnalisée. Le clic sur le champ Réseau social créera un lien vers un autre site que le site en cours. Ce site peut être indiqué dans le champ "URL ou compte cible". 
			Pour plus de détails, consultez [Cibles disponibles pour chaque type de lien](../WDChamp/1000028004.md).




- **URL cible** : Adresse URL à utiliser lorsque la cible est personnalisée (par exemple pour un lien de type "Tweeter").

- **Compte cible** : Adresse du compte à utiliser lorsque la cible est personnalisée (par exemple pour un lien de type "Suivre").
	Pour plus de détails, consultez [Cibles disponibles pour chaque type de lien](../WDChamp/1000028004.md).




Selon le type du lien, il est possible de : 

- **Utiliser le bouton natif conseillé par le réseau social**. Dans ce cas, certaines options peuvent être configurées : 

	- **Afficher le nombre de partages dans une bulle** : Si cette option est cochée, le nombre de partages sera affiché dans une bulle à côté du champ Réseau social. <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Champ_Lien_Social_Bulle.gif)


	- **Demander au réseau social un affichage dans la langue courante de la page** : Si cette option est cochée, les caractéristiques du bouton seront affichées dans la langue de la page (par exemple, le libellé du bouton "Recommander" sera traduit automatiquement). 

- **Utiliser un lien image personnalisé**. Dans ce cas, il suffit de sélectionner l'image à utiliser. 
	**Attention** : Vous devez respecter les conditions d'utilisation des réseaux sociaux et en particulier les restrictions relatives aux images dans les liens qui dirigent vers leurs services. Utilisez les images officielles fournies par les sites.

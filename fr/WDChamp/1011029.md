


## Propriétés associées à un état
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


Les différentes propriétés utilisables sur les états sont les suivantes :



|   |   |
| --- | --- |
| [BasPageCollé](../Proprietes/2511074.md) | La propriété **BasPageCollé** permet de :<br><br>- Savoir si le bas de page doit être collé au dernier bloc avant le bas de page.<br><br>- Modifier la gestion du bloc de pas de page.<br><br><br> |
| [BorneMax](../Proprietes/2510009.md) | La propriété **BorneMax** permet de :<br><br>- connaître et de modifier la borne maximale d'un état basé sur un fichier de données. <br><br>- connaître la borne maximale du filtre en cours.<br><br>- connaître la valeur maximale d'une série d'un graphe.<br><br>- connaître et modifier la valeur maximale de certains types de champs.<br><br><br> |
| [BorneMin](../Proprietes/2510008.md) | La propriété **BorneMin** permet de :<br><br>- connaître et de modifier la borne minimale d'un état basé sur un fichier de données. <br><br>- connaître la borne minimale du filtre en cours.<br><br>- connaître la valeur minimale d'une série d'un graphe.<br><br>- connaître et modifier la valeur minimale de certains types de champs. <br><br><br> |
| [ConserverHiérarchieVisible](../Proprietes/1000021055.md) | La propriété **ConserverHiérarchieVisible** permet de :<br><br>- Savoir quels éléments de la table hiérarchique sont imprimés : impression de la hiérarchie complète ou impression de la hiérarchie visible dans le champ.<br><br>- Modifier le mode d'impression de la hiérarchie : impression de la hiérarchie complète ou impression de la hiérarchie visible dans le champ.<br><br><br> |
| [DroiteAGauche](../Proprietes/1000017228.md) | La propriété **DroiteAGauche** permet de connaître le sens d'écriture réel, effectif sur un champ, une colonne, une fenêtre ou un état. |
| [EspaceH](../Proprietes/2511067.md) | La propriété **EspaceH** permet de connaître et de modifier l'espacement horizontal entre les étiquettes d'un état. Cet espace a été défini lors de la création de l'état au format étiquette. |
| [EspaceV](../Proprietes/2511065.md) | La propriété **EspaceV** permet de connaître et de modifier l'espacement vertical entre les étiquettes d'un état. Cet espace a été défini lors de la création de l'état au format étiquette. |
| [HauteurEtiquette](../Proprietes/2511069.md) | La propriété **HauteurEtiquette** permet de :<br><br>- Connaître la hauteur des étiquettes d'un état de type Étiquette.<br><br>- Modifier la hauteur des étiquettes d'un état de type Étiquette.<br><br><br> |
| [HTML](../Proprietes/2511040.md) | La propriété **HTML** permet de : <br><br>- savoir si l'état en cours d'impression s'imprime en mode HTML. <br><br>- gérer le type HTML d'un champ Libellé présent dans un état.<br><br><br> |
| [ImageFond](../Proprietes/2510056.md) | La propriété **ImageFond** permet de connaître et de modifier l'image de fond d'un élément : fenêtre, page, champ ou état. |
| [ImageFondImprimée](../Proprietes/1000021350.md) | La propriété **ImageFondImprimée** permet de : <br><br>- Savoir si l'image de fond d'un état (formulaire ou non) doit être imprimée.<br><br>- Imprimer ou non l'image de fond d'un état. <br><br><br> |
| [ImpressionAChaquePage](../Proprietes/2511057.md) | La propriété **ImpressionAChaquePage** permet de :<br><br>- Demander l'impression d'un état page par page. Chaque page de l'état sera envoyée séparément au spooler de l'imprimante. <br><br>- Connaître le mode d'impression de l'état (page par page ou en totalité).<br><br><br> |
| [InterfaceModifiableAvecEtatsEtRequêtes](../Proprietes/2511075.md) | La propriété **InterfaceModifiableAvecEtatsEtRequêtes** permet de savoir si l' **interface** d'un état est modifiable sous le logiciel Etats & Requêtes. |
| [LargeurEtiquette](../Proprietes/2511055.md) | La propriété **LargeurEtiquette** permet de :<br><br>- Connaître la largeur des étiquettes d'un état de type "Étiquette".<br><br>- Modifier la largeur des étiquettes d'un état de type "Étiquette".<br><br><br> |
| [MargeBasse](../Proprietes/2511051.md) | La propriété **MargeBasse** permet de :<br><br>- Connaître la marge basse d'un état.<br><br>- Modifier la marge basse d'un état.<br><br><br> |
| [MargeDroite](../Proprietes/2511053.md) | La propriété **MargeDroite** permet de :<br><br>- Connaître la marge de droite d'un état.<br><br>- Modifier la marge de droite d'un état.<br><br><br> |
| [MargeGauche](../Proprietes/2511048.md) | La propriété **MargeGauche** permet de :<br><br>- Connaître la marge de gauche d'un état.<br><br>- Modifier la marge de gauche d'un état.<br><br><br> |
| [MargeHaute](../Proprietes/2511043.md) | La propriété **MargeHaute** permet de :<br><br>- Connaître la marge haute d'un état.<br><br>- Modifier la marge haute d'un état.<br><br><br> |
| [MsgTriSel](../Proprietes/2511045.md) | La propriété **MsgTriSel** permet de :<br><br>- Connaître le message d'attente affiché lors du tri et de la sélection des enregistrements de l'état.<br><br>- Modifier le message d'attente affiché lors du tri et de la sélection des enregistrements de l'état.<br><br><br> |
| [NbColonne](../Proprietes/1000018810.md) | La propriété **NbColonne** permet de connaître le nombre de colonnes d'un état. |
| [NbCopie](../Proprietes/2511042.md) | La propriété **NbCopie** permet de connaître et de modifier le nombre d'exemplaires à imprimer d'un état. |
| [NbEnrImprimé](../Proprietes/2511058.md) | La propriété **NbEnrImprimé** permet de connaître le nombre d'enregistrements déjà imprimés. |
| [NbEnrLu](../Proprietes/2511041.md) | La propriété **NbEnrLu** permet de connaître le nombre d'enregistrements lus : sélectionnés à la fois par la source de l'état ET par le code de filtre (s'il existe). |
| [NbEnrTotal](../Proprietes/2511047.md) | La propriété **NbEnrTotal** permet de :<br><br>- Connaître le nombre d'enregistrements à imprimer.<br><br>- Modifier le nombre d'enregistrements à imprimer.<br><br><br> |
| [NbEtiquette](../Proprietes/2511050.md) | La propriété **NbEtiquette** permet de :<br><br>- Connaître le nombre d'étiquettes imprimées sur la largeur d'une page.<br><br>- Modifier le nombre d'étiquettes imprimées sur la largeur d'une page.<br><br><br> |
| [NbExemplaireEtiquette](../Proprietes/2511059.md) | La propriété **NbExemplaireEtiquette** permet de :<br><br>- Connaître le nombre d'exemplaires d'étiquettes identiques imprimés.<br><br>- Modifier le nombre d'exemplaires d'étiquettes identiques à imprimer.<br><br><br> |
| [Nom](../Proprietes/2510082.md) | La propriété **Nom** permet de connaître :<br><br>- le nom de la police associée à une variable de type Police.<br><br>- le nom d'un champ, d'un état ou d'un bloc d'un état.<br><br>- le nom d'un champ, d'un groupe de champs ou d'une fenêtre.<br><br>- le nom d'un champ, d'un groupe de champs ou d'une page.<br><br><br> |
| [NomSource](../Proprietes/2511035.md) | La propriété **NomSource** permet de :<br><br>- Connaître le nom de la source de données de l'état.<br><br>- Changer la source de données de l'état.<br><br><br> |
| [NomVue](../Proprietes/2511030.md) | La propriété **NomVue** permet de :<br><br>- connaître le nom de la vue HFSQL, source de données d'un état<br><br>- modifier le nom de la vue HFSQL, source de données d'un état.<br><br><br> |
| [NomZoneMem](../Proprietes/2511029.md) | La propriété **NomZoneMem** permet de :<br><br>- connaître le nom de la zone mémoire utilisée comme source de données d'un état.<br><br>- modifier le nom de la zone mémoire utilisée comme source de données d'un état.<br><br><br> |
| [NuméroPage](../Proprietes/2510106.md) | La propriété **NuméroPage** permet de : <br><br>- connaître et de modifier le numéro de la page affichée dans un champ Image (fichiers image de type "multi-pages")<br><br>- connaître et de modifier le numéro de la page d'un fichier PDF affiché dans un champ image d'une fenêtre<br><br>- connaître et de modifier le numéro de la page d'un fichier PDF affiché en fond d'un état. <br><br><br> |
| [NumEtiquetteDeDépart](../Proprietes/2511032.md) | La propriété **NumEtiquetteDeDépart** permet de :<br><br>- Connaître le numéro de l'étiquette sur la planche d'étiquettes à partir de laquelle l'impression est effectuée.<br><br>- Modifier le numéro de l'étiquette à partir de laquelle l'impression est effectuée sur la planche d'étiquettes.<br><br><br> |
| [ODBCCodeSQL](../Proprietes/2511028.md) | La propriété **ODBCCodeSQL** permet de :<br><br>- Connaître le code SQL de la "requête réalisée sur une connexion" source de l'état. <br><br>- Modifier le code SQL de la "requête réalisée sur une connexion" source de l'état.<br><br><br> |
| [ODBCMotDePasse](../Proprietes/2511031.md) | La propriété **ODBCMotDePasse** permet de :<br><br>- Connaître le mot de passe utilisé pour se connecter à une base de données via un driver ODBC ou via un Connecteur Natif. <br><br>- Préciser le mot de passe utilisé pour se connecter à une base de données via un driver ODBC ou via un Connecteur Natif.<br><br><br> |
| [ODBCNomBase](../Proprietes/2511034.md) | La propriété **ODBCNomBase** permet de :<br><br>- Connaître le nom de la base utilisée lors de l'impression d'un état basé sur une "requête réalisée sur une connexion".<br><br>- Préciser le nom de la base sur laquelle porte l'impression d'un état basé sur une "requête réalisée sur une connexion".<br><br><br> |
| [ODBCNomConnexion](../Proprietes/2511033.md) | La propriété **ODBCNomConnexion** permet de :<br><br>- Connaître le nom de la connexion ODBC utilisée lors de l'impression d'un état basé sur une "requête réalisée sur une connexion".<br><br>- Préciser le nom de la connexion ODBC sur laquelle porte l'impression d'un état basé sur une "requête réalisée sur une connexion".<br><br><br> |
| [ODBCNomUtilisateur](../Proprietes/2511019.md) | La propriété **ODBCNomUtilisateur** permet de :<br><br>- Connaître le nom de l'utilisateur utilisé pour se connecter à une base de données via un driver ODBC ou via un Connecteur Natif. <br><br>- Préciser le nom d'utilisateur utilisé pour se connecter à une base de données via un driver ODBC ou via un Connecteur Natif. <br><br><br> |
| [Plan](../Proprietes/2510094.md) | La propriété **Plan** permet de :<br><br>- Connaître : <br><br>	- le plan actif d'une fenêtre (ou d'un état) ou le plan auquel appartient un champ.<br><br>	- le plan actif d'un élément WEBDEV.<br><br><br><br><br>- Modifier : <br><br>	- le plan actif d'une fenêtre (ou d'un état) ou associer un champ à un autre plan.<br><br>	- le plan actif d'un élément WEBDEV.<br><br><br><br><br><br> |
| [RubriqueParcourue](../Proprietes/2510111.md) | La propriété **RubriqueParcourue** permet de :<br><br>- Connaître la rubrique utilisée pour le parcours automatique des champs Table, Zone répétée, Liste (y compris Liste image), Combo, état sur fichier ou des superchamps.<br><br>- Modifier la rubrique utilisée pour le parcours automatique des champs Table, Zone répétée, Liste (y compris Liste image), Combo, état sur fichier ou des superchamps. Le parcours est alors réinitialisé.<br><br><br> |
| [SélectionSeule](../Proprietes/2511017.md) | La propriété **SélectionSeule** permet de :<br><br>- Savoir quels enregistrements du champ Table ou Table hiérarchique sont imprimés : impression de tous les enregistrements ou uniquement des enregistrements sélectionnés.<br><br>- Modifier le mode d'impression des lignes du champ Table ou Table hiérarchique : impression de tous les enregistrements ou uniquement des enregistrements sélectionnés.<br><br><br> |
| [SensEcriture](../Proprietes/2513020.md) | La propriété **SensEcriture** permet de connaître et de modifier le sens d'écriture utilisé dans une fenêtre, un état ou dans un champ (utile pour la gestion des langues de type arabe ou hébreux). |
| [ToutModifiableAvecEtatsEtRequêtes](../Proprietes/2511076.md) | La propriété **ToutModifiableAvecEtatsEtRequêtes** permet de savoir si un état est modifiable sous le logiciel Etats & Requêtes. |
| [Traitement](../Proprietes/1000020878.md) | La propriété **Traitement** permet de modifier les traitements à exécuter par les événements disponibles sur un champ, une fenêtre, une page, un état, ... |
| [ValeurRenvoyée](../Proprietes/2510134.md) | La propriété **ValeurRenvoyée** permet de :<br><br>- Connaître et modifier la valeur renvoyée par une fenêtre, une page ou un état.<br><br>- Connaître et modifier la valeur renvoyée par une option d'un champ Sélecteur. <br><br><br> |


## Les graphes de type "Waterfall"
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000143"></a>
Le graphe "Waterfall" est un graphe en barres permettant de représenter les additions (positives ou négatives) de plusieurs catégories. 

Principalement utilisé en finances, ce graphe permet par exemple de visualiser l'évolution d'une valeur dans le temps ou de voir l'impact de différentes composantes sur cette valeur. 



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Caractéristiques d'un graphe de type Waterfall
<a name="caracteristiques_graphe_type_waterfall_ELTTEXTE000167"></a>
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Graphe_Waterfall.gif)


Dans ce graphe : 

**1.** Valeurs correspondant à un total. Un total correspond à la somme de toutes les catégories précédentes à partir du début du graphe. La valeur est affichée si l'option "Afficher les valeurs" est cochée dans l'onglet "Détail" de la description du champ Graphe.

**2.** Valeurs positives d'une catégorie. La valeur est affichée si l'option "Afficher les valeurs" est cochée dans l'onglet "Détail" de la description du champ Graphe.

**3.** Valeurs négatives d'une catégorie. La valeur est affichée si l'option "Afficher les valeurs" est cochée dans l'onglet "Détail" de la description du champ Graphe.

**4.** Valeur correspondant à un sous-total : un sous-total correspond à la somme de toutes les catégories précédentes à partir du dernier total ou sous-total. La valeur est affichée si l'option "Afficher les valeurs" est cochée dans l'onglet "Détail" de la description du champ Graphe.

**5.** **Quadrillage** :  Il est possible d'afficher le quadrillage grâce aux options "Quadrillage" et "Quadrillage" dans l'onglet "Général" de la description du champ.

**6.** **Titre du graphe** : 
Les caractéristiques du titre du graphe sont saisies dans l'onglet "Détail" de la description du champ (bouton "Editer le titre du graphe"). 
Il est possible de définir le titre du graphe, sa position et son style.

**7.** **Titre des axes** : 
Le titre des axes horizontal et vertical est affiché si le titre a été défini dans l'onglet "Général" de la description du champ Graphe. Ce paramètre peut être défini également par programmation grâce à la fonction [grTitreAxe](../WDLang3/3042057.md).

**8.** Libellé de la valeur (catégorie). 

**9. Ascenseurs permettant de zoomer**. Il est possible d'autoriser le zoom dans un graphe dans l'onglet "Détail" de la description du champ. Cette option est disponible uniquement en WINDEV.

**Autres caractéristiques** : 

- Les graphes de type "Waterfall" peuvent être affichés avec une animation. Pour animer ce type de graphe, il suffit de cocher l'option "Activer les animations" dans l'onglet "Détail" de la description du champ. 

- Pour personnaliser les couleurs des catégories, utilisez la fonction [grWaterfallCouleur](../WDLang3/1000020957.md).

- Pour définir les types des catégories, utilisez la fonction [grWaterfallTypeCatégorie](../WDLang3/1000020930.md).

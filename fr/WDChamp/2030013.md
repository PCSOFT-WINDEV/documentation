


## Première page du projet en mode Session
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000139"></a>
La première page du projet est la première page en mode session ouverte sous le navigateur de l'internaute lors du lancement d'un site WEBDEV en mode session. Idem pour un frameset.

**Remarques** :

- Si une page d'accueil est définie dans votre site, il est nécessaire de lancer la première page en mode session à partir de cette page d'accueil.

- La première page en mode session du projet est automatiquement affichée lors du test d'un site en mode session ou de la partie en mode session d'un site (cliquez sur l'icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_GO_Projet_WB_bl.gif) parmi les boutons d'accès rapide).




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000163"></a>
WEBDEV propose plusieurs méthodes pour définir et choisir la première page en mode session d'un projet :

- Par le volet "Explorateur de projet".

- Dans la description de la configuration.




**Remarque** : La première page en mode session du projet est également demandée lors du test d'un projet en mode session, si aucune page n'a été définie. Pour plus de détails, consultez [Tester un projet](../Editeurs/2019003.md).
<a name="NOTE2_2"></a>


### Définir la première page du projet dans le volet "Explorateur de projet"
<a name="definir_premiere_page_projet_dans_volet_explorateur_projet_ELTPARAGRAPHE000038"></a>Pour définir la première page du projet :

1. Sélectionnez la page qui va être la première page en mode session dans le volet "Explorateur de projet". Cette page doit être une page en mode session de votre site.

2. Affichez le menu contextuel de la page (clic droit) et sélectionnez l'option "Première page du projet en mode session". Cette option n'est disponible que si la page sélectionnée est une page en mode session. 
	L'icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=IconePremPage.gif) permet d'identifier la première page du projet en mode session dans le volet "Explorateur de projet".

<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=PremPageVoletWWTV.gif)

<a name="NOTE2_4"></a>


### Définir la première page en mode session dans la description de la configuration
<a name="definir_premiere_page_mode_session_dans_description_configuration_ELTPARAGRAPHE000054"></a>Pour définir la première page en mode session dans la description de la configuration : 

1. Affichez la description de la configuration courante : sous le volet "Projet", dans le groupe "Configuration de projet", cliquez sur "Configuration courante".

2. La fenêtre de description de la configuration courante s'affiche. 

3. Dans l'onglet "Général", sélectionnez la première page en mode session. <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=PremPageGraphe.gif&type=thumb)


4. Validez la description de la configuration.

5. Validez. 




Remarque : Il est également possible d'afficher la description de la configuration courante via la fenêtre de description du projet : 

1. Sous le volet "Projet", dans le groupe "Projet", cliquez sur "Description". 

2. Dans l'onglet "Projet", cliquez sur "Description de la configuration courante".

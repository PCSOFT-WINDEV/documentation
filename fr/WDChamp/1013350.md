
## Colonne de type Attribut
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000136"></a>
WINDEV et WEBDEV proposent d'utiliser dans les champs Table les colonnes de type "Attribut". Ce type de colonne permet notamment de : 

- stocker une valeur qui ne sera pas affichée dans le champ Table. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) simplifier l'utilisation et la programmation des colonnes Conteneur, en permettant d'associer un attribut à un champ de la colonne conteneur.

## Colonne Attribut : Stocker une valeur qui ne sera pas affichée
<a name="colonne_attribut_stocker_une_valeur_qui_sera_pas_affichee_ELTTEXTE000160"></a>


### Créer une colonne de type Attribut pour stocker une valeur
<a name="creer_une_colonne_type_attribut_pour_stocker_une_valeur_ELTPARAGRAPHE000021"></a>Pour créer une colonne d'un champ Table de type Attribut :

1. Créez si nécessaire un champ Table (Table fichier, Table par programmation ou Table sur variable).

2. Créez une colonne de type Attribut (soit par le menu contextuel du champ Table, soit depuis la fenêtre de description du champ Table).

3. Validez si nécessaire la fenêtre de description du champ. 

4. La colonne de type Attribut n'apparaît pas le champ Table sous l'éditeur. C'est une colonne automatiquement invisible, manipulable uniquement par programmation. 




Remarques : 

- Cette colonne de type Attribut ne doit pas être associée à un champ. 

- Cette colonne de type Attribut est disponible pour les champs Table (Table fichier, Table par programmation ou Table sur variable) et les champs Table hiérarchiques, 





### Manipuler la colonne de type Attribut par programmation
<a name="manipuler_colonne_type_attribut_par_programmation_ELTPARAGRAPHE000036"></a>La colonne de type Attribut peut être manipulée comme les autres types de colonne par programmation. La colonne et sa valeur ne seront jamais affichées dans le champ. 

La colonne de type Attribut est prise en compte lors de l'utilisation des fonctions [TableAjoute](../WDLang1/3074017.md), [TableAjouteLigne](../WDLang1/3074006.md), [TableInsère](../WDLang1/3074025.md) et [TableInsèreLigne](../WDLang1/3074026.md). 

Elle peut être manipulée directement avec la notation : 

```txt
<Nom_colonne>[Numéro_ligne}
```

## Colonne Attribut : Associer un attribut à un champ (colonne conteneur)
<a name="colonne_attribut_associer_attribut_champ_colonne_conteneur_ELTTEXTE000190"></a>


### Créer une colonne de type Attribut associée à un champ
<a name="creer_une_colonne_type_attribut_associee_champ_ELTPARAGRAPHE000064"></a>Pour créer une colonne d'un champ Table de type Attribut associée à un champ d'une colonne de type Conteneur :

1. Créez si nécessaire un champ Table (Table fichier, Table par programmation ou Table sur variable).

2. Créez une colonne de type Attribut (soit par le menu contextuel du champ Table, soit depuis la fenêtre de description du champ Table).

3. Dans l'onglet "Général" de la fenêtre de description de la colonne : 

	- Sélectionnez le champ associé. Ce champ doit être un champ présent dans une colonne conteneur du champ Table. 

	- Sélectionnez la propriété du champ associée à la colonne Attribut : le contenu de la colonne correspondra à la valeur de cette propriété. 




4. Validez la fenêtre de description du champ. 

5. La colonne de type Attribut n'apparaît pas le champ Table sous l'éditeur. C'est une colonne automatiquement invisible, manipulable uniquement par programmation. 




Remarques : 

- Cette colonne de type Attribut est disponible pour les champs Table (Table fichier, Table par programmation ou Table sur variable) et les champs Table hiérarchiques, 





### Manipuler la colonne de type Attribut associée à un champ par programmation
<a name="manipuler_colonne_type_attribut_associee_champ_par_programmation_ELTPARAGRAPHE000082"></a>La colonne de type Attribut associée à un champ peut être manipulée comme les autres types de colonne par programmation. La colonne et sa valeur ne seront jamais affichées dans le champ. 

Cette colonne Attribut peut alors simplifier la programmation des colonnes Conteneur. 

En effet, il devient possible de modifier une caractéristique d'un élément d'une colonne Conteneur en modifiant uniquement la valeur correspondante dans la colonne de type Attribut associée.

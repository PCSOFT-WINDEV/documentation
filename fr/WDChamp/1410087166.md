
## Champ Editeur d'images WEBDEV : Fonctionnalités

## Présentation
<a name="presentation_ELTTEXTE000128"></a>
Le champ Editeur d'images WEBDEV propose de nombreuses fonctionnalités accessibles via plusieurs rubans, en haut et en bas du champ.<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Editeur_images_WB%20-%20HC%20N%B0001.gif)

## Détail des fonctionnalités
<a name="detail_des_fonctionnalites_ELTTEXTE000152"></a>


### Ouvrir une image sous l'éditeur
<a name="ouvrir_une_image_sous_editeur_ELTPARAGRAPHE000017"></a>Pour ouvrir une image dans le champ Editeur d'images, il est possible : 

- soit de dropper directement un fichier image dans le champ, 

- soit de dropper un image contenue dans la page. 

- soit de cliquer sur le bouton "+" dans le ruban du bas de l'éditeur. <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Editeur_images_WB%20-%20HC%20N%B0002.gif)
Le menu permet alors : 

	- d'ouvrir une image depuis le disque : il suffit de préciser dans la fenêtre qui s'affiche le chemin du fichier (ou de dropper le fichier voulu). <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Editeur_images_WB%20-%20HC%20N%B0003.gif)


	- d'ouvrir une image depuis une URL : il suffit de préciser dans la fenêtre qui s'affiche l'URL correspondante. <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Editeur_images_WB%20-%20HC%20N%B0004.gif)





### Dessiner sur l'image
<a name="dessiner_sur_image_ELTPARAGRAPHE000033"></a>Le champ Editeur d'images permet de dessiner librement sur l'image affichée, via le crayon, le feutre ou le surligneur. <br>![Ajout d'un dessin](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Editeur_images_WB%20-%20HC%20N%B0005.gif)
Lorsqu'une de ces options est sélectionnée, une nouvelle barre d'outils apparaît en bas du champ permettant de configurer la couleur et l'épaisseur du trait. <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Editeur_images_WB%20-%20HC%20N%B0012.gif)
Il est également possible de supprimer le trait en cours. 


### Ajouter une forme sur l'image
<a name="ajouter_une_forme_sur_image_ELTPARAGRAPHE000043"></a>Le champ Editeur d'images permet d'ajouter : 

- une ligne ou une flèche sur l'image affichée. <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Editeur_images_WB%20-%20HC%20N%B0006.gif)


- une forme (triangle, rectangle ou cercle). <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Editeur_images_WB%20-%20HC%20N%B0007.gif)





Lorsqu'une de ces options est sélectionnée, une nouvelle barre d'outils apparaît en bas du champ permettant de configurer la couleur et l'épaisseur du trait. <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Editeur_images_WB%20-%20HC%20N%B0012.gif)
Il est également possible de supprimer la forme en cours. 


### Ajouter un texte sur l'image
<a name="ajouter_texte_sur_image_ELTPARAGRAPHE000057"></a>Le champ Editeur d'images permet d'ajouter un texte sur l'image. <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Editeur_images_WB%20-%20HC%20N%B0008.gif)
Lorsqu'une de ces options est sélectionnée, le texte modifiable apparait dans l'image dans une zone de texte, et une nouvelle barre d'outils apparaît en bas du champ permettant de configurer la police, la couleur, .... <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Editeur_images_WB%20-%20HC%20N%B0010.gif)
Il est également possible de supprimer le texte en cours. 


### Manipuler l'image
<a name="manipuler_image_ELTPARAGRAPHE000066"></a>Le champ Editeur d'images permet de manipuler l'image. <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Editeur_images_WB%20-%20HC%20N%B0011.gif)
Il est possible de recadrer, agrandir et faire pivoter l'image. La barre d'outils en bas du champ s'adapte pour proposer les différentes options : <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Editeur_images_WB%20-%20HC%20N%B0009.gif)


Pour zoomer l'image, il est possible d'utiliser : 

- soit la roulette de la souris, 

- soit les boutons + et - situés dans la barre du haut de l'éditeur. 




A tout moment, il est possible de sauver l'image en la téléchargeant via le bouton fléché (![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Editeur_images_WB%20-%20HC%20N%B0010%201.gif)) présent dans la barre du haut de l'éditeur.

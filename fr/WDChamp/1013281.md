
## Manipuler un champ Page Interne par programmation
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000078"></a>
Le champ Page interne permet d'afficher une page interne existante. Différentes manipulations sont possibles sur le champ Page interne en programmation.

**Attention : Ne confondez pas "Champ Page Interne" et "Page interne" : le champ "Page interne" est un type de champ, et la "Page interne" est un type de page.**



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Programmation en WLangage
<a name="programmation_wlangage_ELTTEXTE000102"></a>


### Passer des paramètres à une page interne
<a name="passer_des_parametres_une_page_interne_ELTPARAGRAPHE000021"></a>Pour passer des paramètres à une page interne, il est conseillé de :

1. Créer une procédure d'initialisation dans la page interne.

2. Appeler cette procédure dans la page qui intègre le champ "Page interne".




Exemple :


```wl
// -- Code d'initialisation de PAGE_Page1
// MaPageInterne est un champ de PAGE_Page1
// qui est relié à une page interne 
// Cette page interne possède une 
// procédure Init()
MaPageInterne.Init(Param1, Param2)
```

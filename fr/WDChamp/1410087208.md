
## Manipuler un champ Kanban par programmation
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000137"></a>
WINDEV, WEBDEV et WINDEV Mobile permettent de manipuler un champ Kanban directement par programmation. Il suffit d'utiliser directement la variable du champ Kanban dans le code. 

Pour manipuler un champ Kanban par programmation, WINDEV, WEBDEV et WINDEV Mobile proposent les [Fonctions de gestion des champs Kanban](../WDLang1/1410089375.md). 

Cette page d'aide détaille comment manipuler un champ Kanban par programmation. 







<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Manipuler les listes et les cartes d'un champ Kanban
<a name="manipuler_les_listes_les_cartes_champ_kanban_ELTTEXTE000161"></a>


### Manipuler les listes de Kanban
<a name="manipuler_les_listes_kanban_ELTPARAGRAPHE000033"></a>Les listes des champs Kanban peuvent être créées directement en édition (dans la fenêtre de description du champ) ou par programmation. 

Les différentes listes correspondent à un tableau de listes. La propriété [Liste](../Proprietes/1410089296.md) permet de manipuler les différentes listes d'un champ Kanban, en précisant l'indice de la liste manipulée. 

Il est possible d'utiliser la syntaxe suivante pour manipuler une liste donnée : 

```txt
<Champ Kanban>.Liste[<Numéro de la liste>]
```

Il est ensuite possible d'utiliser par exemple : 

- les propriétés s'appliquant aux listes de Kanban pour modifier leurs caractéristiques. Pour plus de détails sur ces propriétés, consultez [Propriétés associées aux listes de Kanban](../WDChamp/1410087207.md). 

- les [fonctions WLangage](../WDLang1/1410089375.md) permettant de manipuler des listes de Kanban. 





### Manipuler les cartes de Kanban
<a name="manipuler_les_cartes_kanban_ELTPARAGRAPHE000058"></a>Les cartes de Kanban correspondent à une variable de type [kbCarte](../WDLang1/1410089167.md). 

Les cartes sont créées et affichées dans les listes des champs Kanban. La propriété [Carte](../Proprietes/1410089204.md) permet de manipuler les différentes cartes associées à une liste de Kanban. Il suffit de préciser l'indice de la carte manipulée. 

Il est possible d'utiliser la syntaxe suivante pour manipuler une carte d'une liste donnée : 

```txt
<Champ Kanban>.Liste[<Numéro de la liste>].Carte[<Numéro de la carte>]
```


Exemple : Procédure permettant d'afficher le détail de la carte Kanban sélectionnée : 

```wl
// Affichage du détail d'une carte de KANBAN_TACHES
PROCÉDURE DétailCarte(Carte est une kbCarte)

oListe est un Champ <- KANBAN_TACHES.Liste[KANBAN_TACHES.IndiceListe(Carte.NomListe)]
FEN_KBCARTE_MOBILE.OuvreFenêtreMobile(oListe, Carte, Faux)
```

## Propriétés spécifiques au champ Kanban
<a name="proprietes_specifiques_champ_kanban_ELTTEXTE000191"></a>
Les propriétés suivantes sont spécifiques au champ Kanban et aux listes de Kanban : 



|   |   |
| --- | --- |
| [Carte](../Proprietes/1410089204.md) | La propriété **Carte** permet de manipuler les différentes cartes associées à une liste de Kanban. Il est ainsi possible par exemple de parcourir, ajouter, modifier ou supprimer les cartes d'une liste de Kanban. |
| [Liste](../Proprietes/1410089296.md) | La propriété **Liste** permet de manipuler les différentes listes d'un champ Kanban. Il est ainsi possible par exemple de parcourir, ajouter, modifier ou supprimer les listes d'un champ Kanban. |
| [ModeAffichage](../Proprietes/1000023378.md) | La propriété **ModeAffichage** permet de connaître et de modifier : <br><br>- le mode d'affichage d'un champ Traitement de texte, <br><br>- le mode d'affichage d'un champ Lecteur PDF<br><br>- le mode d'affichage d'un champ Editeur HTML, <br><br>- le mode d'affichage d'un champ Kanban.<br><br><br> |





Pour connaître la liste complète des propriétés du WLangage pouvant être utilisées avec un champ de type Kanban, consultez [Propriétés associées au champ Kanban](../WDChamp/1410087207.md).

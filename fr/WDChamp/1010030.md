


## Plateforme d'exécution
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000154"></a>
WINDEV Mobile vous permet de développer des applications pour différentes familles de produits :   

- Android,

- iPhone et/ou iPad,

- Universal Windows.




Voyons en détail chaque plateforme : 




- ![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) **Universal Windows 10 App** : Pour les plateformes Universal Windows 10 App, sélectionnez l'appareil à utiliser pour l'édition de l'application. Il est également possible d'indiquer le répertoire du SDK Windows 10. 




L'ensemble de ces caractéristiques constituent la plateforme d'exécution.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Quand est-ce que la plateforme d'exécution est prise en compte ?
<a name="quand_estce_que_plateforme_execution_est_prise_compte_ELTTEXTE000178"></a>
La plateforme d'exécution sélectionnée est prise en compte :

- **pour la création des fenêtres** : les dimensions sélectionnées seront automatiquement proposées pour les nouvelles fenêtres.

- **pour la génération de l'application**. En effet, l'application générée correspond à une plateforme spécifique.

- **lors du lancement de l'application en mode test**. La plateforme sélectionnée permet d'utiliser le simulateur correspondant. 




<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Comment modifier la plateforme d'exécution ?
<a name="comment_modifier_plateforme_execution_ELTTEXTE000202"></a>
La plateforme d'exécution peut être modifiée :

- lors de la création d'un projet.

- dans la description du projet : 

	- Sous le volet "Projet", dans le groupe "Projet", cliquez sur "Description". 

	- Dans l'onglet "Projet", cliquez sur le bouton "Description de la plateforme courante".







La plateforme sélectionnée est valide pour tout le projet en cours.


## Bouton à flèche
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000184"></a>
Le bouton à flèche est un champ Bouton qui permet de dérouler : 

- un menu contextuel. 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Bouton_fleche_Menu.gif)


- une fenêtre "Popup".
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=bouton_fleche_popup.gif)





Ce type de champ Bouton permet de proposer un ensemble d'actions dans un seul bouton. 

- Le clic sur le bouton exécute le choix par défaut. 

- Le clic sur la flèche déroule le menu ou la fenêtre popup.

## Mise en place
<a name="mise_place_ELTTEXTE000208"></a>


### Créer un champ Bouton à flèche
<a name="creer_champ_bouton_fleche_ELTPARAGRAPHE000028"></a>Pour créer un champ Bouton à flèche :

1. Créez un champ Bouton.

2. Affichez l'onglet "UI" de la fenêtre de description du champ et sélectionnez le type de flèche. Les options disponibles sont : 

	- Aucune flèche (valeur par défaut) : le champ Bouton est un bouton standard. 

	- Bouton flèche : le bouton et la flèche exécutent la même action. 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=bouton_fleche.gif)
Il est possible de choisir : 

		- d'exécuter le code de clic du champ Bouton. 

		- de dérouler le menu contextuel associé au champ Bouton. 

		- d'afficher une des fenêtres de l'application (par exemple fenêtre popup présentant un menu spécifique). 




- Bouton avec flèche distincte : le champ Bouton et la flèche exécutent deux actions distinctes. 

	- Le bouton exécute son code. 

	- La flèche exécute : 
						- le code de clic sur la flèche. 
						- l'action spécifiée : 
						
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=bouton_fleche_distinct.gif)
Il est possible de choisir : 

		- d'exécuter le code de clic du champ Bouton. 

		- de dérouler le menu contextuel associé au champ Bouton. 
										Dans ce cas, il est possible de cocher l'option "Le clic sur le bouton exécutera automatiquement la dernière option de menu sélectionnée via la flèche". 

		- d'afficher une des fenêtres de l'application (par exemple fenêtre popup présentant un menu spécifique). 

3. Validez la fenêtre de description.

## Caractéristiques et fonctionnement des boutons à flèche
<a name="caracteristiques_fonctionnement_des_boutons_fleche_ELTTEXTE000232"></a>


### Positionnement de la flèche
<a name="positionnement_fleche_ELTPARAGRAPHE000072"></a>Selon la taille, la position du libellé dans le champ Bouton et la présence ou non d'une icône dans le champ Bouton, la flèche peut être positionnée à droite ou en-dessous du champ Bouton.  


### Bouton à flèche avec ouverture d'une fenêtre popup
<a name="bouton_fleche_avec_ouverture_une_fenetre_popup_ELTPARAGRAPHE000077"></a>Si votre bouton doit ouvrir une fenêtre, il est conseillé d'utiliser une fenêtre : 

- sans titre,

- sans menu,

- de taille réduite,

- utilisant un cadre simple. 




Remarque : Il est possible de créer une fenêtre popup directement (option "Vierge pour popup" dans l'onglet "Standard" de l'assistant de création de fenêtres). 


### Bouton avec flèche distincte ouvrant un menu et avec mémorisation de l'option sélectionnée
<a name="bouton_avec_fleche_distincte_ouvrant_menu_avec_memorisation_option_selectionnee_ELTPARAGRAPHE000089"></a>Lors de la création d'un bouton avec flèche distincte ouvrant un menu, si vous avez coché l'option "Le clic sur le bouton exécutera automatiquement la dernière option de menu sélectionnée via la flèche", le fonctionnement du champ Bouton sera le suivant : 

- à la première ouverture de la fenêtre contenant le bouton, le libellé du bouton contient la première option du menu contextuel (image associée à l'option et libellé de l'option). 

- lorsque l'utilisateur déroule le menu contextuel et sélectionne une option : 

	- le libellé de l'option et l'image associée apparaissent automatiquement dans le bouton.

	- le code de l'option sélectionnée est exécuté. 







Remarque : si l'option de menu contient une image, WINDEV gère automatiquement en exécution les différents états sur l'image affichée dans le bouton. 




### Bouton à flèche utilisé dans un champ Ruban
<a name="bouton_fleche_utilise_dans_champ_ruban_ELTPARAGRAPHE000107"></a>Si un champ Bouton est présent dans un regroupement d'un champ Ruban, l'onglet "UI" du champ Bouton permet de configurer le mode de gestion du bouton lorsque le ruban est réduit. 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ruban_Champ_Bouton.gif)


Il est possible au choix : 

- de ne pas réduire le champ. 

- de le réduire pour afficher au minimum l'image du bouton et son libellé. 

- de le réduire pour afficher un minimum l'image du bouton. 


Les deux dernières options sont intéressantes dans le cas de boutons affichant une icône spécifique et un libellé. 


### Look du bouton à flèche
<a name="look_bouton_fleche_ELTPARAGRAPHE000121"></a>L'onglet "Style" de la fenêtre de description d'un bouton à flèche permet notamment de définir les caractéristiques du séparateur. Il est notamment possible de spécifier : 

- l'opacité utilisée, 

- le type de la flèche (chevron ou flèche), 

- la couleur du séparateur. 
	La couleur "Transparente" permet d'obtenir un bouton fléché sans séparateur. 


Ces options peuvent être définies pour un seul état du bouton ou pour tous les états du bouton.

## Présentation du champ Range Slider
<a name="presentation_champ_range_slider_ELTTEXTE000128"></a>


### Champ d'une fenêtre
<a name="champ_une_fenetre_ELTPARAGRAPHE000011"></a>Le champ Range Slider permet la sélection facile d'intervalle ou de zoom. 

Un champ Range Slider est constitué des éléments suivants : <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=RangeSlider_voc.gif)


Les boutons situés à l'extrémité de la cage du Range Slider permettent de définir l'étendue visible. 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Créer un champ Range Slider
<a name="creer_champ_range_slider_ELTTEXTE000152"></a>
Pour créer un champ de type Range Slider :

1. Sous le volet "Création", dans le groupe "Ajuster", déroulez "Ascenseur" et cliquez sur "Range Slider".

2. Cliquez dans la fenêtre à la position où le champ doit être créé. Le champ apparaît en création.




Pour afficher les caractéristiques du champ, sélectionnez l'option "Description" dans le menu contextuel du champ. Vous pourrez ainsi choisir : 

- les valeurs du Range Slider.

- l'orientation du Range Slider.




Lorsque le champ est créé, il peut être utilisé pour gérer la sélection d'intervalle ou de zoom par programmation.

<a name="NOTE3_1"></a>


## Vocabulaire associé au champ Range Slider
<a name="vocabulaire_associe_champ_range_slider_ELTTEXTE000176"></a>
Pour manipuler simplement un Range Slider, il est nécessaire de connaître le vocabulaire suivant :

| Nom | Définition |
| --- | --- |
| Avance rapide | Pas du Range Slider utilisé lorsque l'utilisateur clique dans la zone située entre la cage du Range Slider et les boutons spin. |
| Etendue totale | Taille totale du champ Range Slider. |
| Etendue visible | Taille de la cage du Range Slider. |
| Valeur | Position du début de la cage du Range Slider. |
| Valeur minimale | Valeur du Range Slider lorsque la cage du Range Slider est placée à l'origine (1 par défaut). Cette valeur correspond à la borne minimale. |

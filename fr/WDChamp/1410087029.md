## Interagir avec un champ Composant Web
<a name="interagir_avec_champ_composant_web_ELTTEXTE000128"></a>


### Procédures associées au champ Composant Web
<a name="procedures_associees_champ_composant_web_ELTPARAGRAPHE000019"></a>Pour interagir avec un champ Composant Web, plusieurs méthodes sont disponibles. 

Il est par exemple possible de créer des procédures Serveur et/ou Navigateur liées au champ Composant Web (onglet "Détail" de la description du champ). 

Ces procédures peuvent permettre par exemple de modifier des données présentes dans le champ Composant Web, en leur affectant une valeur provenant de la page WEBDEV. 

Le champ Composant Web peut appeler ces procédures directement depuis le code HTML du composant Web. L'éditeur de code HTML propose via la complétion sur les actions les différentes procédures disponibles. 

Par exemple, le code HTML suivant permet d'appeler le code "Action" du composant lors du clic sur le bouton "Commander" : 

```html
<button onclick="CMP_Test()" class="w3-aqua">
	Commander
</button>
```


L'éditeur de code HTML propose les différentes procédures et actions disponibles : <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CompoWeb_code.gif)


Il est également possible d'appeler les procédures du champ Composant Web directement depuis des champs WEBDEV. 




### Traitements d'affectation et de récupération de la propriété Valeur
<a name="traitements_affectation_recuperation_propriete_valeur_ELTPARAGRAPHE000040"></a>Les traitements d'affectation et de récupération de la propriété Valeur permettent notamment d'initialiser la valeur du composant Web lors du chargement de la page et de récupérer sa valeur. 











## Manipuler le champ Composant Web par programmation
<a name="manipuler_champ_composant_web_par_programmation_ELTTEXTE000158"></a>
Les propriétés spécifiques au champ Composant Web sont les suivantes : 



|   |   |
| --- | --- |
| [CodeCSS](../Proprietes/1000022365.md) | La propriété **CodeCSS** permet de lire et modifier le code CSS associé à un champ Composant Web. |
| [CodeHTML](../Proprietes/1000022364.md) | La propriété **CodeHTML** permet de lire et modifier le code HTML associé à un champ Composant Web. |
| [CodeJS](../Proprietes/1000022366.md) | La propriété **CodeJS** permet de lire et modifier le code Javascript associé à un champ Composant Web. |





Pour connaître la liste complète des propriétés du WLangage pouvant être utilisées avec un champ de type Composant Web, consultez [Propriétés associées au champ Composant Web](../WDChamp/1410087028.md).

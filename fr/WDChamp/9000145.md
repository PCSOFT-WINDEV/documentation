
## Ancrage des champs superposés (dans une page)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000107"></a>
WEBDEV permet de définir des champs superposables : les champs superposables peuvent être affichés par dessus d'autres champs. 

Un champ superposable peut être ancré par rapport à son conteneur ou par rapport à un autre champ de la page. 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000131"></a>


### Définir un ancrage sur des champs superposables
<a name="definir_ancrage_sur_des_champs_superposables_ELTPARAGRAPHE000018"></a>Pour définir un ancrage sur des champs superposables :

1. Sélectionnez le champ superposable.  
	Rappel : Pour définir un champ superposable, cochez l'option "Le champ peut être superposé" dans l'onglet "UI" de la fenêtre de description du champ. 

2. Définissez l'ancrage de l'élément :

	- soit dans l'onglet "UI" de la fenêtre de description de l'élément.

	- soit grâce à l'option "Ancrage" du menu contextuel de l'élément voulu.  




3. La fenêtre suivante s'affiche : <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=WB_Ancrage_superpose_nv.gif&type=thumb)
Dans cette fenêtre, les options sont les suivantes : 

	- **Position** : Dans l'option "Se déplacer comme le champ de référence suivant", sélectionnez le champ de référence du champ superposable. Le champ superposable se déplacera de la même façon. Ce champ de référence peut être : 

		- Le conteneur associé au champ (par exemple un champ Cellule ou la page contenant le champ). 

		- Un autre champ du même parent (par exemple un autre champ associé à la cellule).  

		- Un autre champ de la page (non superposable). 


 Sélectionnez ensuite si nécessaire la position à appliquer au champ superposable (à appliquer après le déplacement éventuel). 

- **Ancrage en largeur** : permet de définir le comportement du champ en largeur lors du redimensionnement du champ de référence. Le champ superposé peut :

	- conserver sa largeur,

	- s'adapter au contenu,

	- suivre le comportement du navigateur.<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Ancrage_Largeur_WB.gif)

4. Validez. Le champ superposable est alors ancré.




## Barre de message des fenêtres
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000171"></a>


### Champ d'une fenêtre
<a name="champ_une_fenetre_ELTPARAGRAPHE000011"></a>La barre de message d'une fenêtre permet d'afficher plusieurs types d'informations. Par exemple :

- un message d'information lié au champ en cours. 
	Ce message est saisi dans l'onglet "Aide" de la fenêtre de description du champ. 
	Le message sera affiché dans la barre de message lorsque l'utilisateur sélectionnera le champ.

- un message d'information spécifique. Ce message est affiché par programmation grâce à la fonction [Message](../WDLang1/3021006.md).

- une jauge de progression. Cette jauge est affichée par programmation grâce à la fonction [Jauge](../WDLang1/3021008.md).




**Pour afficher simultanément diverses informations**, il est possible de diviser la barre de message en plusieurs cases.

<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000195"></a>


### Afficher une barre de message
<a name="afficher_une_barre_message_ELTPARAGRAPHE000036"></a>Pour afficher une barre de message dans la fenêtre en cours, différentes manipulations sont possibles : 

- sélectionnez l'option "Afficher la barre de message" dans l'onglet "Style" de la description de la fenêtre.

- sous le volet "Fenêtre", dans le groupe "Barres et menus", déroulez "Barre de message" et sélectionnez "Avec barre de message".

- par programmation, utilisez la propriété [BarreDeMessage](../Proprietes/2510007.md).



<a name="NOTE2_2"></a>


### Afficher une barre de message multi-cases
<a name="afficher_une_barre_message_multicases_ELTPARAGRAPHE000066"></a>Pour proposer une barre de message multi-cases dans une fenêtre, il est possible d'ajouter des cases en utilisant une des solutions suivantes :

- depuis la fenêtre de description de la barre de message : sous le volet "Fenêtre", dans le groupe "Barres et menus", déroulez "Barre de message" et sélectionnez "Editer".

- par le menu contextuel de la barre de message (option "Insérer une case").

- par programmation, grâce à la fonction [MessageAjouteCase](../WDLang1/3021001.md).




Chaque case peut afficher un type d'information spécifique. Pour plus de détails, consultez [Informations affichées dans une barre de message multicases](../WDChamp/1010037.md).

## Fenêtre de description de la barre de message
<a name="fenetre_description_barre_message_ELTTEXTE000225"></a>
La fenêtre de description d'une barre de message permet de :

- ajouter et/ou supprimer des cases, modifier l'ordre des cases.

- spécifier pour chaque case, ses caractéristiques (pour plus de détails, consultez [Informations affichées dans une barre de message multicases](../WDChamp/1010037.md)).



![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=FenDescriptionBarreMessage.gif)
Pour afficher la fenêtre de description d'une barre de message, vous pouvez au choix :

- double-cliquer sur la barre de message.

- sélectionner l'option "Description" dans le menu contextuel de la barre de message.

- sous le volet "Fenêtre", dans le groupe "Barres et menus", déroulez "Barre de message" et sélectionnez "Editer".

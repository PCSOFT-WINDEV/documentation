
## Rechercher le texte sélectionné sur Internet (FAA)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000098"></a>
Le menu contextuel d'un champ de saisie propose une option "Rechercher sur Internet" sur le terme sélectionné dans le champ. 

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=FAA_Recherche_Internet.gif)


Cette option est également disponible dans les colonnes des champs Table, Table hiérarchique, Tableur et Tableau Croisé dynamique. Si la colonne ne permet pas la saisie, le contenu de la cellule est sélectionné. 

Lorsque l'option "Rechercher sur Internet" est sélectionnée, une recherche sur la sélection est automatiquement effectuée dans Google. 



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000122"></a>


### Mise en place
<a name="mise_place_ELTPARAGRAPHE000026"></a>L'option "Rechercher xxx sur Internet" est automatiquement mise en place pour les champs suivants : 

- champs de saisie. 

- champs Table et Table hiérarchique, 

- champs Tableur, 

- champs Tableau Croisé dynamique. 




L'affichage de cette option est configurable (comme toutes les options proposées par les FAA). Pour plus de détails, consultez [Configurer les menus FAA](../Editeurs/2010040.md). 

En programmation : 

- la fonction [FAADésactive](../WDLang1/1000022018.md) utilisée avec la constante *faaRechercheInternet* permet de supprimer l'option dans le menu contextuel. 

- la fonction [FAAParamètre](../WDLang1/1000022108.md) permet de spécifier le moteur de recherche utilisé (Google par défaut).


## Caractéristiques d'un champ indicé
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Description d'un Champ Indicé
<a name="description_champ_indice_ELTTEXTE000096"></a>
Pour chaque champ d'un champ indicé, une fenêtre de description est disponible. Le nom du champ est le même. Seul varie l'indice du champ. Il est possible de définir des options différentes pour chaque champ d'un champ indicé.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Différences avec WINDEV 5.5
<a name="differences_avec_windev_55_ELTTEXTE000120"></a>
**En version 5.5**, l'indice apparaissait uniquement dans la barre d'état de l'éditeur de fenêtres. La fenêtre de description était la même pour tous les champs indicés. Chaque modification effectuée dans la fenêtre de description (7 onglets) était reportée sur tous les champs indicés.

**A partir de la version 9**, l'indice apparaît dans la fenêtre de description. Chaque champ d'un champ indicé a sa propre description (fenêtre à 7 onglets), et ses propres caractéristiques. Un champ d'un champ indicé peut être vert et un autre champ du même champ indicé peut être bleu. De même deux champs d'un même champ indicé peuvent utiliser des menus contextuels différents.

***Pour modifier les caractéristiques de tous les champs indicés en une seule opération***, il suffit :

1. De sélectionner tous les champs indicés (avec le lasso de la souris par exemple)

2. D'afficher la fenêtre de description et d'effectuer les modifications voulues.






<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Code d'un champ indicé
<a name="code_champ_indice_ELTTEXTE000144"></a>
Tous les champs indicés de même nom sont associés aux mêmes traitements.

Par exemple le code d'initialisation du champ indicé Saisie1[1] est le même que le code d'initialisation du champ indicé Saisie1[5].

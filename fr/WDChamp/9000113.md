## Présentation
<a name="presentation_ELTTEXTE000204"></a>
Une fenêtre maximisée occupe tout l'écran du mobile. Différentes options sont paramétrables dans l'assistant de création d'une fenêtre. Pour lancer cet assistant : 

- Cliquez sur ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_nouveau.gif) parmi les boutons d'accès rapide. 

- La fenêtre de création d'un nouvel élément s'affiche : cliquez sur "Fenêtre" puis sur "Fenêtre".

- Dans l'assistant de création d'une fenêtre, sélectionnez "Assistant" :

## Caractéristiques des fenêtres maximisées
<a name="caracteristiques_des_fenetres_maximisees_ELTTEXTE000228"></a>


### Barre de titre
<a name="barre_titre_ELTPARAGRAPHE000026"></a>La barre de titre permet d'afficher le titre de la fenêtre. Cette barre de titre peut être affichée ou non grâce à l'option "Avec barre de titre" de l'onglet "Style" de la fenêtre de description. 
<a name="NOTE2_2"></a>


### Action Bar
<a name="action_bar_ELTPARAGRAPHE000035"></a>Grâce à Android 3, il est désormais possible d'insérer une Action Bar dans vos applications. 

Les "Action bars" remplacent la barre de titre d'une fenêtre et permettent d'afficher plusieurs informations :

- l'icône de l'application,

- le titre de la fenêtre,

- certaines options de menu,

- ...


Pour plus de détails, consultez [Action Bar d'une fenêtre](../WDChamp/1000025001.md).
<a name="NOTE2_3"></a>


### Onglets
<a name="onglets_ELTPARAGRAPHE000054"></a>Les onglets permettent d'afficher différentes informations sur différents volets. L'utilisateur n'a qu'à choisir l'onglet voulu.

Pour plus de détails sur ce type de champ, consultez [Champ Onglet](../WDChamp/1013194.md).
<a name="NOTE2_4"></a>


### Ascenseur
<a name="ascenseur_ELTPARAGRAPHE000066"></a>Les ascenseurs seront automatiquement affichés si la fenêtre a une taille supérieure à la résolution de l'écran du mobile Android utilisé.

Cette option peut être choisie à tout moment dans la description de la fenêtre (onglet "Détail").

**Attention** : En mode plein écran, les ascenseurs automatiques ne sont jamais affichés dans une fenêtre maximisée.
<a name="NOTE2_5"></a>


### Menu
<a name="menu_ELTPARAGRAPHE000077"></a>Dans une application pour Android, le menu se trouve en bas des fenêtres. Un menu déroulant permet à l'utilisateur d'accéder rapidement à une fonctionnalité de l'application.

En édition, les menus des fenêtres Android sont présentés comme de simples menus déroulants.

A l'exécution, ces menus s'afficheront de façon différente en fonction du nombre d'options :

- Si le menu comporte moins de 6 options, il s'affiche sous forme de gros boutons en bas de l'écran.

- Si le menu comporte plus de 6 options, les 5 premières options s'affichent sous forme de gros boutons et un sixième bouton automatique "Plus" permet à l'utilisateur de voir la suite du menu sous forme d'un menu déroulant.




Les menus des applications Android ne supportent pas d'avoir plus d'un niveau de sous-menus. Au-delà, une erreur fatale se produit lors de l'exécution de l'application.

L'ajout ou la suppression du menu déroulant dans une fenêtre peut être effectué à tout moment :

- soit sous le volet "Fenêtre", dans le groupe "Barres et menus", déroulez "Menu principal" et sélectionnez "Ajouter le menu principal".

- soit dans l'onglet "Style" de la description de la fenêtre (option "Activer le menu principal").

## Taille d'une fenêtre maximisée
<a name="taille_une_fenetre_maximisee_ELTTEXTE000276"></a>
La taille réellement utilisable d'une fenêtre maximisée dépend des différents éléments pris en compte dans la fenêtre (barre de titre visible ou non, ...).

WINDEV Mobile permet de paramétrer tous ces éléments dans vos fenêtres pour Android : 

- L'onglet "Style" de la description de la fenêtre permet de paramétrer : 

	- L'affichage de la barre de titre 

	- L'affichage de la barre système




- L'onglet "Détail" de la description de la fenêtre permet de paramétrer la taille de la fenêtre :

## Options supplémentaires
<a name="options_supplementaires_ELTTEXTE000300"></a>
L'onglet "Détail" permet également de paramétrer la gestion de l'orientation de la fenêtre. En effet, lorsque l'utilisateur modifie l'orientation de son portable Android, la fenêtre peut : 

- suivre l'orientation du poste. Dans ce cas, les ancrages permettent d'obtenir une fenêtre utilisable en mode portrait comme en mode paysage. 

- être bloquée en mode portrait.

- être bloquée en mode paysage.




**Remarque** : Lors du changement du mode d'affichage (portrait à paysage ou inversement), la nouvelle taille de la fenêtre est matérialisée par une ligne en pointillés sous l'éditeur de fenêtres. Il ne reste plus qu'à redimensionner la fenêtre grâce aux poignées et à positionner les champs.

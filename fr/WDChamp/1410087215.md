
## Manipuler un champ Editeur d'images pour mobile par programmation
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000124"></a>
WINDEV Mobile permet de manipuler directement un champ Editeur d'images mobile par programmation. 

La variable du champ Editeur d'images mobile permet de manipuler l'image affichée dans le champ.  

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Afficher une image dans un champ Editeur d'images
<a name="afficher_une_image_dans_champ_editeur_images_ELTTEXTE000148"></a>
Pour afficher une image dans un champ Editeur d'images, une simple affectation suffit. Il est possible d'utiliser le nom d'un fichier ou une variable de type [Image](../WDLang1/1000019650.md). 

Quelques exemples :

- affectation directe : 
	
	```wl
	EI_MonImage = "C:\Temp\MonImage.bmp"
	```


- affectation par variable :
	
	```wl
	CheminImage = fRepEnCours() + "\MonImage.bmp"
	EI_MonImage = CheminImage
	```


- affectation par variable de type [Image](../WDLang1/1000019650.md) : 
	
	```wl
	MonImage est une Image
	MonImage.Largeur = 500
	MonImage.Hauteur = 100
	// Chargement à partir d'un fichier
	MonImage = dChargeImage("image.png")
	// Dessin dans l'image
	dDébutDessin(MonImage)
	// Ecriture d'un texte
	dPolice("Arial", 12, iNormal, 0)
	dTexte(4, 0, "J'écris mon texte dans un champ Image !")
	// Fin du dessin
	dFinDessin(MonImage)
	
	// Affectation à un champ Editeur d'images mobile
	EI_MonImage = MonImage
	```






Remarque : Un champ Editeur d'images mobile peut aussi être initialisé à partir d'une variable de type Buffer ou de type texte contenant la valeur de l'image en binaire.

## Récupérer l'image affichée dans le champ Editeur d'images
<a name="recuperer_image_affichee_dans_champ_editeur_images_ELTTEXTE000172"></a>
L'image affichée et manipulée dans le champ Editeur d'images peut être récupérée dans une variable de type [Image](../WDLang1/1000019650.md). 

Par exemple : 

```wl
MonImageModifiée est une Image
MonImageModifiée = EI_MonImage
```


Remarque : L'image récupérée contient uniquement les modifications validées par l'utilisateur. Si des modifications sont en cours dans le champ et n'ont pas été validées, elles ne seront pas prises en compte dans la variable [Image](../WDLang1/1000019650.md) renvoyée. 

L'image peut ensuite être sauvegardée (fonction [dSauveImagePNG](../WDLang1/3029027.md), fonction [AlbumSauve](../WDLang3/1000020191.md) par exemple).




## Raccourci clavier et lettre d'appel
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000159"></a>
Un champ d'une fenêtre (ou d'une page) peut être associé à un caractère alphanumérique, à une touche du clavier ou à une combinaison de touches du clavier, appelé "raccourci clavier".

Ce raccourci clavier permet un accès direct par le clavier à ce champ (sélection du champ ou clic sur le champ).

**Remarque** : Par défaut, il est possible de définir un raccourci clavier de type "Lettre d'appel" à partir du libellé du champ (caractère '&' devant la lettre du libellé à définir comme lettre d'appel). Pour plus de détails, consultez le paragraphe ["Caractère &"](#NOTE3_1).

### Définir le raccourci clavier d'un champ d'une fenêtre
<a name="definir_raccourci_clavier_champ_une_fenetre_ELTPARAGRAPHE000035"></a>Pour définir le raccourci clavier d'un champ d'une fenêtre : 

1. Affichez la fenêtre de description du champ (option "Description" du menu contextuel).

2. Affichez l'onglet "UI" et cliquez sur le bouton situé à droite du champ "Raccourci clavier".

3. Sélectionnez la ou les touches composant la lettre d'appel voulue. 
	Le bouton "Détection" permet de détecter la touche ou la combinaison de touches utilisée au clavier.

4. Validez.

## Caractère &
<a name="caractere_ELTTEXTE000213"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 

### Gestion du caractère & dans les champs des fenêtres
<a name="gestion_caractere_dans_les_champs_des_fenetres_ELTPARAGRAPHE000072"></a>Par défaut, il est possible d'utiliser une lettre d'appel comme raccourci clavier. Cette lettre d'appel est définie à partir du libellé du champ (caractère '&' devant la lettre du libellé à définir comme lettre d'appel).

Le raccourci clavier associé au champ sera alors Alt + Lettre suivant le &.

Pour utiliser le caractère '&' dans un libellé, il est nécessaire de le doubler. Par exemple :

- le libellé "Guerre&Paix" sera "Guerre<u><u><u><u>P</u></u></u></u>aix" (avec Alt + P comme lettre d'appel).

- le libellé "Guerre&&Paix" sera "Guerre&Paix".




Pour modifier le mode d'interprétation du caractère '&', utilisez la propriété [InterprèteEtCommercial](../Proprietes/2510151.md).

Pour que le caractère '&' soit automatiquement inséré devant le libellé des nouveaux champs : 

1. Affichez les options de l'éditeur de fenêtres : sous le volet "Accueil", dans le groupe "Environnement", déroulez "Options" et sélectionnez "Options de l'éditeur de fenêtres". 

2. Dans l'onglet "Champs", cochez l'option "Lettre d'appel automatique lors de la création d'un champ".

3. Validez.

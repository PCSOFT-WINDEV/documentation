
## Description d'un champ Tableau croisé dynamique
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000236"></a>
La fenêtre de description d'un champ Tableau croisé dynamique permet de paramétrer de nombreux éléments du champ. 

Cette page d'aide présente les différents paramétrages possibles suivants les onglets de la fenêtre de description du champ. 

Rappel : Pour afficher la fenêtre de description du champ Tableau croisé dynamique, sélectionnez l'option "Description" dans le menu contextuel du champ. 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Principales caractéristiques du champ Tableau croisé dynamique
<a name="principales_caracteristiques_champ_tableau_croise_dynamique_ELTTEXTE000260"></a>


### Onglet "Général"
<a name="onglet_general_ELTPARAGRAPHE000020"></a>L'onglet "Général" de la description du champ Tableau croisé dynamique permet de définir : 

- la **position des valeurs affichées** : Plusieurs valeurs peuvent être affichées dans le champ Tableau croisé dynamique. Ces valeurs peuvent être affichées : 

	- côte à côte : <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=TCD_Donnee_cote.gif)


	- les unes en dessous des autres : <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=TCD_Donnee_sous.gif)

- la **position des totaux** : Le champ Tableau croisé dynamique peut effectuer des totaux des valeurs affichées. Les totaux effectués peuvent : 

	- être affichés en bas et à droite (en fin de ligne et de colonne). 

	- ne pas être affichés. 

	- être affichés en bas uniquement (en fin de colonne). 

	- être affichés à droite uniquement (en fin de ligne). 


L'option "Totaux toujours visibles" permet d'afficher les totaux en fin de ligne et/ou de colonne même si la ligne ou la colonne n'est pas entièrement affichée. Les totaux sont alors toujours visibles. 



<a name="NOTE2_2"></a>


### Onglet "UI"
<a name="onglet_ELTPARAGRAPHE000045"></a>L'onglet "UI" permet principalement de : 

- définir l'état initial du champ. 

- définir le menu contextuel associé au champ (menu des FAA, menu personnalisé, ...). 

- définir le **mode de sélection**. L'option "Autoriser la sélection multiple des lignes, colonnes et cellules"  permet à l'utilisateur de sélectionner plusieurs colonnes, lignes ou cellules en même temps. Les options proposées par le menu des FAA s'appliquent alors à l'ensemble des cellules sélectionnées (copie, couleur de fond, ...). 
	Il est également possible de définir le mode de sélection par programmation grâce à la propriété [Multisélection](../Proprietes/1000017054.md). 





<a name="NOTE2_2B"></a>


### Onglet "Détail"
<a name="onglet_detail_ELTPARAGRAPHE000068"></a>L'onglet "Détail" permet principalement de : 

- définir l'apparition des ascenseurs et des bulles d'aide (avec la possibilité d'afficher une bulle d'aide sur les éléments tronqués). 

- autoriser le retour à la ligne dans les entêtes de ligne. Il est également possible de paramétrer cette option par programmation à l'aide de la propriété [RetourALaLigneAuto](../Proprietes/1000021868.md). 

- mémoriser les paramètres définis par l'utilisateur. Ces paramètres seront restaurés automatiquement à la prochaine ouverture de la fenêtre. 



<a name="NOTE2_3"></a>


### Onglet "Contenu"
<a name="onglet_contenu_ELTPARAGRAPHE000084"></a>L'onglet "Contenu" permet de visualiser et configurer les différents éléments du champ Tableau croisé dynamique. Cet onglet est divisé en plusieurs zones : <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=TCD_Contenu.gif&type=thumb)


**1. Fichier source du champ Tableau croisé dynamique**. Ce fichier correspond au fichier de données contenant la valeur utilisée pour le calcul. Par exemple, le fichier de données LigneCde pour calculer les ventes effectuées.

**2. Filtres d'affichage** (utilisation avancée). Cette liste permet de visualiser le ou les filtres d'affichage du champ Tableau croisé dynamique. Ces filtres pourront être utilisés avec la fonction [TCDFiltre](../WDLang1/1000020531.md). 
Pour ajouter ou modifier les filtres, il suffit de cliquer dans la zone. Le détail des filtres est affiché. 
Pour créer un nouveau filtre : 

1. Cliquez sur "Ajouter". 

2. Définissez la liaison depuis le fichier source pour atteindre la rubrique qui servira de filtre et validez. 

3. Dans la partie droite de l'écran, définissez les caractéristiques de la valeur affichée. 

4. Pour revenir à l'onglet "Contenu" initial, cliquez sur la flèche en haut de l'écran (![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=TCD_Fleche_retour.gif)). 




**3. Lignes**. Cette liste permet de visualiser les rubriques affichées sur les entêtes de lignes du champ Tableau croisé dynamique. Ces rubriques permettent de regrouper les données. 
Pour ajouter ou modifier les entêtes de lignes, il suffit de cliquer dans la zone. Le détail des entêtes de lignes est affiché. 
Pour chaque entête de ligne, les informations suivantes sont affichées : 

- dans le tableau, la liaison entre le fichier source et l'entête. 

- à droite, les caractéristiques de la valeur affichée dans l'entête de ligne : 

	- Source affichée. La flèche rouge permet de définir le tri sur la source. 

	- Filtre appliqué sur la source affichée. Pour plus de détails, consultez [Filtrer les entêtes](../WDChamp/1000029005.md). 

	- Nom de l'entête (pour être manipulé par programmation)

	- Masque d'affichage de l'entête

	- Etat de l'entête par défaut (enroulé ou déroulé). 





Pour ajouter un nouvel entête de ligne : 

1. Cliquez sur "Ajouter". 

2. Définissez la liaison depuis le fichier source pour atteindre la rubrique affichée en entête et validez. 

3. Dans la partie droite de l'écran, définissez les caractéristiques de la valeur affichée. 


Pour revenir à l'onglet "Contenu" initial, cliquez sur la flèche en haut de l'écran (![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=TCD_Fleche_retour.gif)). 

**4. Colonnes**. Cette liste permet de visualiser les rubriques affichées sur les entêtes de colonnes du champ Tableau croisé dynamique. Ces rubriques permettent de regrouper les données. 
Pour ajouter ou modifier les entêtes de colonnes, il suffit de cliquer dans la zone. Le détail des entêtes de colonnes est affiché. 
Pour chaque entête de colonne, les informations suivantes sont affichées : 

- dans le tableau, la liaison entre le fichier source et l'entête. 

- à droite, les caractéristiques de la valeur affichée dans l'entête de colonne : 

	- Source affichée. La flèche rouge permet de définir le tri sur la source. 

	- Filtre appliqué sur la source affichée. Pour plus de détails, consultez [Filtrer les entêtes](../WDChamp/1000029005.md). 

	- Nom de l'entête (pour être manipulé par programmation)

	- Masque d'affichage de l'entête

	- Etat de l'entête par défaut (enroulé ou déroulé). 





Pour ajouter un nouvel entête de colonne : 

1. Cliquez sur "Ajouter". 

2. Définissez la liaison depuis le fichier source pour atteindre la rubrique affichée en entête et validez. 

3. Dans la partie droite de l'écran, définissez les caractéristiques de la valeur affichée. 


Pour revenir à l'onglet "Contenu" initial, cliquez sur la flèche en haut de l'écran (![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=TCD_Fleche_retour.gif)). 

**5. Valeurs à afficher dans le champ Tableau croisé dynamique**. Cette liste permet de visualiser les valeurs affichées dans les cellules du champ Tableau croisé dynamique. 
Pour ajouter, modifier, supprimer ou réorganiser ces valeurs, il suffit de cliquer dans la zone. Le détail des valeurs est affiché.  

Les caractéristiques des valeurs sont regroupées en 3 thèmes : 

- Général : Permet de définir : 

	- le nom de la valeur. Ce nom permet de manipuler la valeur par programmation. 

	- le libellé de chaque donnée affichée dans la cellule. <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=WD%20-%20TCD%20Libelle%20Cellules.gif)
Ce libellé peut également être modifié par programmation grâce à la propriété [Libellé](../Proprietes/2510053.md). 




- Calcul : Permet de définir le calcul effectué, la source utilisée et le masque d'affichage de la valeur dans le tableau croisé dynamique. 

- Affichage : Permet de définir les options d'affichage de la valeur : cadrage, mise à blanc, libellé du total (en ligne ou en colonne), ...
	Il est également possible de définir le "eye-magnet" (mise en forme conditionnelle) sur les valeurs : il est par exemple possible d'afficher en rouge toutes les cellules d'un champ Tableau Croisé Dynamique dont la valeur est négative.




Pour revenir à l'onglet "Contenu" initial, cliquez sur la flèche en haut de l'écran (![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=TCD_Fleche_retour.gif)).


## Manipuler les options de menu via le menu contextuel
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000193"></a>
WINDEV et WINDEV Mobile gèrent deux types de menus déroulants :

- **les menus principaux des fenêtres**. 
	Un menu principal est obligatoirement associé à une fenêtre. Ce menu peut être affiché sous la barre de titre de la fenêtre. Pour plus de détails, consultez [Gestion des menus principaux des fenêtres](../WDChamp/1010010.md).
	

- **les menus contextuels personnalisés**.
	Un menu contextuel peut être ouvert par l'utilisateur grâce à un clic droit de la souris. 
	Pour plus de détails sur les menus contextuels personnalisés, consultez [Menu contextuel personnalisé](../WDChamp/1010017.md).




Un menu déroulant (contextuel personnalisé ou principal) est composé d'une ou de plusieurs options et sous-options. Chaque option permet d'exécuter un code WLangage.

Par exemple :


|   |   |

Les manipulations des options de menu sous l'éditeur sont identiques que le menu soit un menu principal ou un menu contextuel personnalisé. Pour manipuler les options de menu, il suffit d'éditer le menu concerné.

**Pour éditer (ou afficher sous l'éditeur)** :

- ![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png)![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) Le menu principal d'une fenêtre :

	- ouvrez la fenêtre concernée sous l'éditeur de fenêtres.

	- sélectionnez une des options de menu. Les options et les sous-options s'affichent automatiquement.




- Un menu contextuel personnalisé : sous le volet "Fenêtre", dans le groupe "Barres et menus", déroulez "Menus contextuels" et sélectionnez un des menus contextuels existants.




Remarque :  WINDEV et WINDEV Mobile permettent également de [manipuler les options de menu par programmation](../WDChamp/1010007.md).

Cette page d'aide présente la manipulation des options de menu (ajout, suppression, ...) sous l'éditeur via le menu contextuel des options. Pour plus de détails, consultez également :

- [Caractéristiques des options de menu (fenêtre de description)](../WDChamp/1010009.md).

- [Gestion des options de menu dans les fenêtres MDI](../WDChamp/1010015.md).

## Manipulation des options de menu sous l'éditeur
<a name="manipulation_des_options_menu_sous_editeur_ELTTEXTE000288"></a>


### Ajouter une option de menu standard
<a name="ajouter_une_option_menu_standard_ELTPARAGRAPHE000033"></a>Pour ajouter une option de menu standard : 

- **Après** l'option de menu sélectionnée, vous pouvez :

	- soit afficher le menu contextuel de l'option de menu en cours (clic droit) et sélectionner l'option "Ajouter après".

	- soit appuyer sur la touche Entrée.




- **Avant**l'option de menu sélectionnée, vous pouvez :

	- soit afficher le menu contextuel de l'option de menu en cours (clic droit) et sélectionner l'option "Ajouter avant".

	- soit appuyer sur la touche Inser.




- Saisissez le libellé de la nouvelle option de menu.

- Appuyez sur la touche Entrée pour valider l'ajout.




**Remarques** :

- La touche Echap permet d'annuler l'ajout d'une option de menu.

- Pour passer d'une option de menu à la suivante, utilisez les touches fléchées (haut et bas).

- ![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Si une fenêtre contient plus de 2 menus, une flèche permet d'afficher les menus suivants sous l'éditeur de fenêtres.



<a name="NOTE2_3"></a>


### Ajouter un séparateur
<a name="ajouter_separateur_ELTPARAGRAPHE000059"></a>Pour ajouter un séparateur : 

1. Sélectionnez l'option de menu avant laquelle le séparateur doit être ajouté.

2. Affichez le menu contextuel de l'option de menu en cours (clic droit).

3. Sélectionnez l'option "Insérer un séparateur".


Remarque : Il est également possible :

- d'ajouter un séparateur avant l'option de menu sélectionnée en appuyant sur la touche Pause.

- d'ajouter un séparateur en donnant à l'option de menu le libellé "-" ou "--" ou "---". A la validation de la saisie, l'option est transformée en séparateur. 




![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Les séparateurs ne sont pas disponibles.
<a name="NOTE2_2"></a>


### Lier une option de menu à une action prédéfinie
<a name="lier_une_option_menu_une_action_predefinie_ELTPARAGRAPHE000079"></a>Pour lier une option de menu à une action prédéfinie : 

1. Sélectionnez l'option de menu voulue.

2. Affichez le menu contextuel de l'option de menu en cours (clic droit).

3. Sélectionnez l'option "Action prédéfinie...". La fenêtre de sélection d'une action prédéfinie s'affiche.

4. Sélectionnez l'action voulue et validez.


![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Les actions prédéfinies ne sont pas disponibles.
<a name="NOTE2_4"></a>


### Ajouter un sous-menu
<a name="ajouter_sousmenu_ELTPARAGRAPHE000095"></a>Pour ajouter un sous-menu : 

1. Sélectionnez l'option de menu à laquelle une sous-option doit être ajoutée.

2. Affichez le menu contextuel de l'option de menu en cours (clic droit).

3. Sélectionnez l'option "Transformer pour dérouler un sous-menu".

4. Saisissez le libellé de la première option du sous-menu créé.

5. Appuyez sur la touche Entrée pour valider l'ajout.




Remarques :

- L'ajout de sous-options dans un sous-menu se fait de façon identique à l'ajout d'options dans un menu.

- Il est également possible d'ajouter un sous-menu à l'option de menu sélectionnée en appuyant sur la touche Tab.

### Modifier la couleur de fond des menus et des options de menu
<a name="modifier_couleur_fond_des_menus_des_options_menu_ELTPARAGRAPHE000119"></a>Pour modifier la couleur de fond des menus et des options de menu : 

1. Sélectionnez le menu ou une de ses options.

2. Affichez la fenêtre de description (option "Description de l'option" du menu contextuel).

3. Dans l'onglet "Style", configurez :

	- la couleur de la barre de menu au repos ou lors du survol.

	- la couleur des options au repos ou lors du survol.





La couleur "Automatique" permet d'obtenir la couleur par défaut gérée par Windows.

Remarque : Pour que les couleurs définies soient prises en compte, décochez l'option "Afficher le menu au look XP" dans l'onglet "Style" de la description de la fenêtre.
<a name="NOTE2_6"></a>


### Supprimer une option de menu
<a name="supprimer_une_option_menu_ELTPARAGRAPHE000137"></a>Pour supprimer une option de menu : 

1. Sélectionnez l'option de menu à supprimer.

2. Affichez le menu contextuel de l'option de menu en cours (clic droit).

3. Sélectionnez l'option "Supprimer".


Remarque : Il est également possible de supprimer l'option de menu en cours en appuyant sur la touche Suppr.

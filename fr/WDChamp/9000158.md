## Présentation
<a name="presentation_ELTTEXTE000177"></a>
Un des éléments importants du style d'un champ et auquel on ne pense pas assez est **son cadre**. De nombreux paramétrages sont disponibles : 

- arrondi des coins, 

- épaisseur de la bordure, 

- ...




L'éditeur d'états et l'éditeur de fenêtres permettent de définir des cadres simplement, directement sous l'éditeur ou via la fenêtre de description des champs.

La mise en forme des cadres personnalisés est disponible : 

- sous l'éditeur d'états pour tous les types de champs et pour les blocs.  

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) sous l'éditeur de fenêtres pour les champs Libellé et les champs Image.




Remarques : 

- Il est également possible de définir un cadre personnalisé par programmation grâce à la propriété [Cadre](../Proprietes/1000018157.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Il est également possible de définir un cadre personnalisé pour **les cellules des champs Table** (par programmation uniquement, via la propriété [Cadre](../Proprietes/1000018157.md)).

## Comment le faire ?
<a name="comment_faire_ELTTEXTE000201"></a>


### Définir un cadre sous l'éditeur
<a name="definir_cadre_sous_editeur_ELTPARAGRAPHE000051"></a>Pour définir les options d'un cadre sous l'éditeur : 

1. Sélectionnez le champ voulu. 

2. Appuyez sur la touche Alt du clavier. Le picto ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Style_Cadre.gif)
 apparaît en haut à droite du champ. 

3. Maintenez la touche Alt enfoncée et cliquez sur le picto ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Style_Cadre.gif)
. Les options de configuration apparaissent : 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Style_Cadre_Etat_bl.gif&type=thumb)


4. Il est possible de configurer : 

	- l'arrondi. La zone "Arrondi" permet de définir la taille de l'arrondi, ainsi que le coin sur lequel l'appliquer. 

	- l'épaisseur, la couleur et le type de la bordure, ainsi que le côté sur lequel l'appliquer. 




5. Pour valider, cliquez ailleurs dans l'éditeur. 




**Pour configurer l'arrondi**, vous pouvez : 

- Sélectionner le ou les coins à prendre en compte soit en cliquant sur le carré "Arrondi", soit en sélectionnant les points oranges correspondant aux coins dans le champ (il est possible de sélectionner plusieurs coins avec la touche Ctrl).  

- Configurer la forme de l'arrondi soit en modifiant la largeur et la hauteur, soit en déplaçant directement le point orange sous l'éditeur. 


Remarque : Lorsque l'arrondi est défini pour un coin, il est possible de reporter la configuration sur un des autres coins grâce au bouton "Reporter sur ...". 

**Pour configurer la bordure**, vous pouvez : 

- Sélectionner le ou les traits à prendre en compte soit en cliquant sur le carré "Epaisseur", soit en sélectionnant directement les côtés du champ (il est possible de sélectionner plusieurs côtés avec la touche Ctrl). 

- Configurer les caractéristiques du trait : en sélectionnant l'épaisseur, la couleur et le type du trait.  
	


Remarque : Il est possible de reporter la configuration d'un côté sur un des autres côtés grâce au bouton "Reporter sur ...". 


### Définir un cadre sous la fenêtre de description
<a name="definir_cadre_sous_fenetre_description_ELTPARAGRAPHE000090"></a>Pour définir les options d'un cadre dans la fenêtre de description du champ : 

1. Sélectionnez le champ voulu. 

2. Affichez la fenêtre de description (option "Description" du menu contextuel). 

3. Affichez l'onglet "Style".

	- Sous l'éditeur de fenêtres, sélectionnez l'élément "Cadre extérieur". Dans la zone "Cadre", sélectionnez "Personnalisé" puis cliquez sur la flèche. L'éditeur de cadre apparaît.

	- Sous l'éditeur d'états, sélectionnez l'élément "Cadre externe". 




4. Il est possible de configurer : 

	- l'arrondi directement dans la zone d'aperçu ou dans la zone "Arrondi". 

	- le style de la bordure. 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Style_Cadre_Etat_Desc.gif&type=thumb)
*Fenêtre de description sous l'éditeur d'états*




5. Validez la fenêtre de description. 




**Pour configurer l'arrondi**, vous pouvez : 

- Sélectionner le ou les coins à prendre en compte soit en cliquant sur le carré "Arrondi", soit en sélectionnant les points oranges correspondant aux coins dans l'aperçu (il est possible de sélectionner plusieurs coins avec la touche Ctrl).  

- Configurer la forme de l'arrondi soit en modifiant la largeur et la hauteur dans la zone "Arrondi", soit en déplaçant directement le point orange dans l'aperçu. 


Remarque : Lorsque l'arrondi est défini pour un coin, il est possible de reporter la configuration sur un des autres coins grâce au bouton "Reporter sur...". 

**Pour configurer la bordure**, vous pouvez : 

- Sélectionner la ou les bordures à prendre en compte soit en cliquant sur le carré "Style de bordure", soit en sélectionnant directement les côtés du champ dans l'aperçu (il est possible de sélectionner plusieurs côtés avec la touche Ctrl). 

- Configurer les caractéristiques de la bordure en sélectionnant l'épaisseur, la couleur et le type du trait.  
	


Remarque : Il est possible de reporter la configuration d'un côté sur un des autres côtés grâce au bouton "Reporter sur...".

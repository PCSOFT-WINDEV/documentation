


## Fenêtres non maximisées pour Android
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000148"></a>
Une fenêtre non maximisée est une fenêtre occupant seulement une partie de l'écran.

Une fenêtre non maximisée peut également être détourée (fenêtre non rectangulaire).

Les fenêtres non maximisées peuvent être créées via l'assistant de création de fenêtres. Pour lancer cet assistant : 

- Cliquez sur ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_nouveau.gif) parmi les boutons d'accès rapide. 

- La fenêtre de création d'un nouvel élément s'affiche : cliquez sur "Fenêtre" puis sur "Fenêtre".

- L'assistant de création d'une fenêtre se lance. 

- Sélectionnez "Assistant". 

- Dans l'assistant, sélectionnez "Non maximisée".




Le plus souvent, les fenêtres non maximisées sont utilisées pour afficher des messages d'information.

Les informations demandées par l'assistant de création d'une fenêtre sont : 

- [la taille de la fenêtre à l'ouverture](#NOTE2_1). 

- [les boutons à insérer dans la fenêtre](#NOTE2_2). 

- le nom de la fenêtre. 




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Caractéristiques des fenêtres non maximisées
<a name="caracteristiques_des_fenetres_non_maximisees_ELTTEXTE000172"></a>


### Taille de la fenêtre
<a name="taille_fenetre_ELTPARAGRAPHE000046"></a>La taille de la fenêtre à l'ouverture peut être modifiée dans l'onglet "Détail" de la fenêtre de description. 

Par défaut, la fenêtre est centrée par rapport à l'écran. 
Remarques : Positionnement des fenêtres non maximisées : 

- La position des fenêtres non maximisées peut être modifiée avec la fonction [FenTaille](../WDLang1/3038020.md). 

- Il est possible de sélectionner la position d'ouverture "relatif à l'écran" (onglet "UI" de la fenêtre de description des fenêtres).



<a name="NOTE2_2"></a>


### Boutons à afficher dans la fenêtre
<a name="boutons_afficher_dans_fenetre_ELTPARAGRAPHE000064"></a>Lors de la création d'une fenêtre non maximisée à l'aide de l'assistant de création d'une fenêtre, il est possible de choisir les boutons à insérer dans cette fenêtre : 

- Bouton OK et bouton Annuler,

- Bouton Réessayer et bouton Abandonner,

- Bouton Ignorer et bouton Fermer,

- Bouton Quitter.




Dans ce cas, les boutons sélectionnés (et leur code) sont automatiquement ajoutés dans la fenêtre. Selon le type de bouton demandé, le code de clic du bouton renvoie un booléen (Oui pour le bouton "OK", Non pour le bouton "Abandonner", "Annuler"). Cette valeur de retour peut être traitée avec la fonction [OuvreFenêtreMobile](../WDLang1/1000021018.md) (ou la fonction [OuvreFille](../WDLang1/3038021.md)). Par exemple :

```wl
SI OuvreFenêtreMobile(FEN_Non_Max) = Oui ALORS Info("Vous avez cliqué sur OK")
```

<a name="NOTE2_3"></a>


### Fenêtre détourée
<a name="fenetre_detouree_ELTPARAGRAPHE000089"></a>Une fenêtre non maximisée peut être détourée (fenêtre non rectangulaire). Il suffit pour cela d'utiliser une image au format PNG. 

Pour détourer une fenêtre non maximisée :

- Dans l'onglet "Style" de la description de la fenêtre, sélectionnez la couleur de fond "Transparent".

- Dans l'onglet "Image" de la description de la fenêtre, sélectionnez : 

	- une image de fond au format PNG, 

	- le mode de transparence "Selon l'image".
			Remarque : La transparence de l'image est gérée par le format PNG.

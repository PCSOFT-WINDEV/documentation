


## Propriétés associées à un menu ou une option de menu
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000084"></a>
Il est possible d'utiliser des propriétés WLangage sur un menu présent dans une fenêtre ou une page.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>

| --- | --- |

| [Couleur](../Proprietes/2510071.md) | La propriété **Couleur** permet de connaître ou modifier la couleur de la police de :<br><br>- une variable de type Police.<br><br>- un texte d'un champ Libellé ou calculé d'un état.<br><br>- un texte d'un champ d'une fenêtre.<br><br>- un texte d'un champ d'une page.<br><br><br> |

| [Etat](../Proprietes/2510039.md) | La propriété **Etat** permet de connaître et de modifier l'état d'affichage d'un élément. |
| [FormatAffichage](../Proprietes/1410088879.md) | La propriété **FormatAffichage** permet de connaître et de modifier le format attendu pour l'affichage d'un libellé : texte simple, texte au format RTF ou Markdown. |
| [Grisé](../Proprietes/1000019648.md) | La propriété **Grisé** permet de : <br><br>- Savoir si un champ, un groupe de champs ou une fenêtre est grisé ou non. <br><br>- Griser ou dégriser un champ, un groupe de champs ou une fenêtre.<br><br><br> |
| [Groupe](../Proprietes/2510049.md) | La propriété **Groupe** permet de : <br><br>- savoir si le champ appartient ou non à un groupe de champs. Si le champ appartient à un ou plusieurs groupes, la propriété **Groupe** permet d'identifier ces groupes de champs.<br><br>- associer un champ à un groupe de champs. <br><br><br> |
| [Image](../Proprietes/2510034.md) | La propriété **Image** permet de connaître et de modifier l'image de fond d'un champ ou d'un bloc. |
| [ImageEtat](../Proprietes/2510055.md) | La propriété **ImageEtat** permet de connaître et de modifier le nombre de dessins présents dans :<br><br>- L'image associée à un bouton.<br><br>- L'image du curseur dans un potentiomètre.<br><br>- L'image associée au bouton d'un volet de boîte à outils.<br><br>- L'image associée au bouton d'un volet d'onglet.<br><br>- L'image associée au champ Notation.<br><br>- L'image associée au regroupement d'un champ Ruban.<br><br>- L'image associée à une option de menu.<br><br><br> |
| [Libellé](../Proprietes/2510053.md) | **Dans une base de données**, la propriété **Libellé** permet de :<br><br>- Connaître le libellé associé à une rubrique d'un fichier de données. <br><br>- Connaître le libellé associé à un fichier de données. <br><br>- Connaître et modifier le libellé d'une connexion. <br><br><br><br><br>**Dans un état**, la propriété **Libellé** permet de :<br><br>- Connaître le texte d'un champ Libellé ou Lien.<br><br>- Modifier le texte d'un champ Libellé ou Lien.<br><br><br><br><br>**Dans une fenêtre ou une page**, la propriété **Libellé** permet également de :<br><br>- Connaître et modifier le libellé d'un champ présent dans une fenêtre ou une page.<br><br>- Connaître et modifier le libellé d'une option de menu.<br><br>- Connaître et modifier le titre d'une fenêtre ou d'une page.<br><br>- Connaître le contenu d'un champ chemin de navigation.<br><br><br><br><br> |
| [Message](../Proprietes/2510078.md) | La propriété **Message** permet de :<br><br>- Connaître le message d'aide associé à un champ ou une option de menu d'une fenêtre.<br><br>- Modifier le message d'aide associé à un champ ou une option de menu d'une fenêtre.<br><br><br> |
| [Nom](../Proprietes/2510082.md) | La propriété **Nom** permet de connaître :<br><br>- le nom de la police associée à une variable de type Police.<br><br>- le nom d'un champ, d'un état ou d'un bloc d'un état.<br><br>- le nom d'un champ, d'un groupe de champs ou d'une fenêtre.<br><br>- le nom d'un champ, d'un groupe de champs ou d'une page.<br><br><br> |
| [NomComplet](../Proprietes/2514005.md) | La propriété **NomComplet** permet de connaître le nom complet :<br><br>- d'un champ.<br><br>- d'une fenêtre.<br><br>- d'une option de menu.<br><br>- d'une colonne de table.<br><br><br><br><br> |
| [NuméroAide](../Proprietes/2510141.md) | La propriété **NuméroAide** permet de connaître et de modifier le numéro d'aide associé au champ. |
| [Occurrence](../Proprietes/2510097.md) | La propriété **Occurrence** permet de connaître le nombre d'éléments présents dans un ensemble. |
| [Police](../Proprietes/2510095.md) | La propriété **Police** permet de :<br><br>- Connaître l'identifiant de la police utilisée dans un champ d'un état, d'une page ou d'une fenêtre.<br><br>- Modifier la police utilisée dans un champ d'un état, d'une page ou d'une fenêtre.<br><br><br> |
| [PoliceAlphabet](../Proprietes/2510089.md) | La propriété **PoliceAlphabet** permet de :<br><br>- Connaître l'alphabet actuellement utilisé par les polices écran d'un champ.<br><br>- Modifier l'alphabet utilisé par les polices écran d'un champ.<br><br><br> |
| [PoliceBarrée](../Proprietes/2510091.md) | La propriété **PoliceBarrée** permet de :<br><br>- Savoir si le texte affiché est barré ou non.<br><br>- Barrer ou non le texte affiché dans un champ.<br><br><br> |
| [PoliceCondensée](../Proprietes/2510096.md) | La propriété **PoliceCondensée** permet de :<br><br>- Savoir si les caractères du texte affiché sont condensés ou non.<br><br>- Condenser ou non les caractères du texte affiché.<br><br><br> |
| [PoliceEtendue](../Proprietes/2510104.md) | La propriété **PoliceEtendue** permet de :<br><br>- Savoir si les caractères du texte affiché sont étendus ou non.<br><br>- Étendre ou non les caractères du texte affiché.<br><br><br> |
| [PoliceGras](../Proprietes/2510103.md) | La propriété **PoliceGras** permet de :<br><br>- Savoir si le texte affiché est en gras ou non.<br><br>- Modifier la graisse du texte.<br><br><br> |
| [PoliceItalique](../Proprietes/2510102.md) | La propriété **PoliceItalique** permet de :<br><br>- Savoir si le texte affiché est en italique ou non.<br><br>- Modifier la mise en forme italique du texte.<br><br><br> |
| [PoliceLarge](../Proprietes/2510101.md) | La propriété **PoliceLarge** permet de :<br><br>- Savoir si les caractères du texte affiché sont élargis ou non.<br><br>- Élargir ou non les caractères du texte affiché.<br><br><br> |
| [PoliceNom](../Proprietes/2510100.md) | La propriété **PoliceNom** permet de :<br><br>- Connaître la police utilisée dans un champ.<br><br>- Modifier la police utilisée dans un champ.<br><br><br> |
| [PoliceSoulignée](../Proprietes/2510099.md) | La propriété **PoliceSoulignée** permet de :<br><br>- Savoir si le texte affiché est souligné ou non.<br><br>- Souligner ou non le texte.<br><br><br> |
| [PoliceTaille](../Proprietes/2510072.md) | La propriété **PoliceTaille** permet de :<br><br>- Connaître la taille de la police utilisée dans un champ.<br><br>- Modifier la taille de la police utilisée dans un champ.<br><br><br> |
| [StyleCoche](../Proprietes/1410088605.md) | La propriété **StyleCoche** permet de connaître et de modifier le style de la coche d'une option de menu. Cette coche apparaît uniquement si l'option de menu est cochée. |
| [Traitement](../Proprietes/1000020878.md) | La propriété **Traitement** permet de modifier les traitements à exécuter par les événements disponibles sur un champ, une fenêtre, une page, un état, ... |
| [Type](../Proprietes/2510131.md) | La propriété **Type** permet de connaître :<br><br>- le type d'un élément décrit par programmation (description de rubrique ou de fichier) ou non (fichier ou rubrique défini sous l'éditeur d'analyses, vue ou requête).<br><br>- le type d'un objet.<br><br>- le type de la valeur stockée dans un variant.<br><br><br> |
| [Valeur](../Proprietes/2510130.md) | La propriété **Valeur** permet de :<br><br>- Connaître et modifier la valeur d'un champ d'un état.<br><br>- Connaître et modifier la valeur d'un champ ou le titre d'une fenêtre.<br><br>- Connaître l'identifiant d'un flux. <br><br>- Connaître et modifier la valeur d'un champ ou le titre d'une page.<br><br><br> |
| [Visible](../Proprietes/2510138.md) | La propriété **Visible** permet de :<br><br>- Savoir si un élément est visible.<br><br>- Rendre un élément visible ou invisible.<br><br><br> |

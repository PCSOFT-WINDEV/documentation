
## Graphe Heatmap
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000108"></a>
Le graphe de type Heatmap permet de représenter en 2 dimensions des données qui ont 3 axes de valeurs.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Caractéristiques d'un graphe heatmap
<a name="caracteristiques_graphe_heatmap_ELTTEXTE000132"></a>
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Graphe_Heatmap%20-%20HC%20N%B0001.gif)


**1. Titre du graphe**
Les caractéristiques du titre du graphe sont saisies dans l'onglet "Détail" de la description du champ (bouton "Editer le titre du graphe").

Il est possible de définir le titre du graphe, sa position et son style.

**2. Valeur de la données**

**3. Titre de l'axe vertical**
 
Ce paramètre est affiché si le titre de l'axe a été défini dans l'onglet "Général" de la description du champ Graphe. Ce paramètre peut être défini également par programmation grâce à la fonction [grTitreAxe](../WDLang3/3042057.md). Il est également possible de définir un titre pour l'axe horizontal.

**4. Graduation et titre de l'axe horizontal**

Autres caractéristiques : 

- Il est possible de réaliser un affichage des données par palier (fonction [grParamètre](../WDLang3/1000018337.md)). 

- Les fonctions grHMxxx permettent d'effectuer des manipulations spécifiques sur les graphes de type Heatmap.

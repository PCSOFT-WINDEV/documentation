
## Etat Tableur
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000146"></a>
L'éditeur d'états offre la possibilité de créer des états de type "Tableur". 

L'état est alors constitué uniquement d'un champ Tableur. 

Remarque : Ce type de champ est créé lors de la création d'un état Tableur et peut également être créé indépendamment de cet état. 

Exemple :

## Créer un état Tableur
<a name="creer_etat_tableur_ELTTEXTE000170"></a>


### Création de l'état
<a name="creation_etat_ELTPARAGRAPHE000025"></a>Pour créer un état de type "Tableur" : 

1. Cliquez sur ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_nouveau.gif)
 parmi les boutons d'accès rapide. 

	- La fenêtre de création d'un nouvel élément s'affiche : cliquez sur "Etat" puis sur "Etat". 

	- L'assistant de création d'un état se lance. 




2. Sélectionnez un état de type "Tableur". 

3. Indiquez dans la suite de l'assistant :

	- la source de données du champ Tableur créé dans l'état. Cette source peut être : 

		- une rubrique d'un fichier de données,

		- un champ Tableur présent dans une fenêtre. 

		- aucune, si le champ Tableur est affecté par programmation. 




- les options de mise à l'échelle du champ Tableur présent dans l'état : 

	- Pas de mise à l'échelle : La feuille du tableur est imprimée à ses dimensions. L'impression peut être effectuée sur plusieurs pages en largeur et en hauteur si nécessaire. 

	- Ajuster la feuille de calcul aux dimensions du champ : La feuille du tableur sera réduite pour entrer entièrement dans la zone du champ Tableur. 

	- Ajuster toutes les colonnes à la largeur du champ : Les colonnes de la feuille du tableur seront réduites ou agrandies pour que toutes les colonnes de la feuille du tableur soient affichées dans la largeur du champ. 

	- Ajuster toutes les lignes à la hauteur du champ : Les lignes de la feuille du tableur seront réduites ou agrandies pour que toutes les lignes de la feuille du tableur soient affichées dans la hauteur du champ.

## Caractéristiques d'un état Tableur et d'un champ Tableur
<a name="caracteristiques_etat_tableur_champ_tableur_ELTTEXTE000194"></a>
Il est possible de modifier les caractéristiques suivantes d'un état Tableur : 

- Par défaut, l'état Tableur n'est pas lié à une source de données. La source de données de l'état Tableur peut être modifiée dans l'onglet "Données" de la fenêtre de description de l'état (option "Description de l'état" dans le menu contextuel de l'état). 

- L'onglet "Général" de la fenêtre de description du champ Tableur permet de spécifier les caractéristiques de mise à l'échelle du champ Tableur. 

- L'onglet "Contenu" de la fenêtre de description du champ Tableur permet de lier le champ Tableur à une source de données : 

	- Rubrique mémo d'un fichier de données ou d'une requête, ... 

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Champ Tableur présent dans une fenêtre. 

	- Variable de type [xlsDocument](../WDLang5/1000017464.md). 




- L'onglet "Style" de la fenêtre de description du champ Tableur permet de spécifier le style du quadrillage et du texte d'entête affichés dans le champ Tableur. 




Pour plus de détails, consultez [Champ Tableur dans un état](../WDChamp/1000033009.md).

## Manipulation d'un champ Tableur dans un état par programmation
<a name="manipulation_champ_tableur_dans_etat_par_programmation_ELTTEXTE000218"></a>
Il est possible d'associer un champ Tableur d'un état : 

- à une variable de type [xlsDocument](../WDLang5/1000017464.md). Il suffit d'utiliser la ligne de code : 
	
	```txt
	<Champ Tableur> = <Variable xlsDocument>
	```


- à un fichier au format xls/xlsx. Il suffit d'utiliser la ligne de code : 
	
	```txt
	<Champ Tableur> = <Chemin du fichier xls/xlsx>
	```





Attention : Les fonctions de gestion des champs Tableur ne peuvent pas être utilisées sur le champ Tableur d'un état.

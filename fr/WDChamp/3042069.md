
## Graphe : Gestion des légendes
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000248"></a>
Les graphes permettent de gérer plusieurs types de légendes. Selon leurs positions, ces légendes ont des noms différents : 

- **Etiquette de séries**![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=EtiquetteSerie.gif)
L'ensemble des étiquettes de séries correspond à la **légende du graphe**.  

- **Etiquette de catégories**![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=EtiquetteCat.gif)


- **Libellé du graphe**![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Libelle_Graphe.gif)
Le libellé du graphe peut être constitué de trois éléments :

	- les étiquettes de catégories

	- la valeur des catégories

	- le pourcentage des catégories




- **Titre du graphe**![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Titre.gif)


- **Titre des axes**![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Titreaxe.gif)

## Comment manipuler les différents types de légendes ?
<a name="comment_manipuler_les_differents_types_legendes_ELTTEXTE000272"></a>


### Manipulation des étiquettes de catégories
<a name="manipulation_des_etiquettes_categories_ELTPARAGRAPHE000037"></a>Les étiquettes de catégories sont affichées :

- sous les graduations horizontales pour les graphes de type courbe ou les graphes boursiers de type chandelier japonais et BarCharts,

- sous l'axe des X pour les histogrammes ou les graphes de type boursier Minimum/Maximum.




![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=EtiquetteCat.gif)


Les étiquettes de catégories ne sont pas utilisables avec les graphes de type nuage de points.

La définition des étiquettes de catégories est réalisée avec la fonction [grEtiquetteCatégorie](../WDLang3/3042015.md).

Il est possible de modifier la police de l'étiquette grâce à la fonction [grPoliceEtiquette](../WDLang3/3042012.md).

Les étiquettes de catégories définies avec la fonction [grEtiquetteCatégorie](../WDLang3/3042015.md) peuvent être affichées dans le libellé du graphe de type Secteur (constante *grAfficheEtiquette*utilisée avec la fonction [grLibellé](../WDLang3/3042046.md)).

**Cas particulier : les étiquettes de catégories de type date ou date-heure**
Un champ Graphe peut afficher des étiquettes de catégories de type Date. L'option "Echelle des abscisses espacée en fonction du temps" (onglet "Général" de la fenêtre de description du champ Graphe) permet d'obtenir une bonne répartition des étiquettes. 

Si le champ Graphe affiche des abscisses au format DateHeure, un zoom de l'utilisateur sur le graphe permet d'obtenir une échelle temporelle à plusieurs niveaux, pour chaque unité de temps. 

![L'affichage et la présentation de l'année, des mois et des jours est automatique](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=073-abscisses_temporelles.jpg)


Les niveaux possibles sont les suivants : 

- Millisecondes,

- Secondes,

- Minutes,

- Heures,

- Jours, 

- Mois, 

- Années. 




Par défaut : 

- Le niveau minimum à afficher est déterminé en fonction de la plage de données.

- Le niveau maximum est déterminé en utilisant deux paramètres :

	- Le nombre de niveaux maximum à afficher dans le graphe (par défaut :3). 

	- Le niveau maximum autorisé pour l'affichage (par défaut : année).







Les fonctions suivantes permettent de configurer les niveaux affichés : 



|   |   |
| --- | --- |
| [grNiveauTemporelMasque](../WDLang3/1000023017.md) | Définit le masque d'affichage d'un niveau de l'échelle temporelle d'un graphe. |
| [grNiveauTemporelMax](../WDLang3/1000023016.md) | Définit le nombre de niveaux à afficher dans l'échelle temporelle d'un graphe ainsi que le niveau maximum. |

### Manipulation des étiquettes de séries
<a name="manipulation_des_etiquettes_series_ELTPARAGRAPHE000102"></a>L'ensemble des étiquettes de séries correspond à la légende du graphe. ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=EtiquetteSerie.gif)


Les étiquettes de série ne sont pas utilisables avec les graphes de type secteur.

La définition des étiquettes de séries est réalisée avec la fonction [grEtiquetteSérie](../WDLang3/3042024.md). 

Il est possible de modifier la police de la légende grâce à la fonction [grPoliceLégende](../WDLang3/3042037.md).  

La fonction [grLégende](../WDLang3/3042047.md) permet d'indiquer la présence et la position de la légende (c'est-à-dire de l'ensemble des étiquettes de séries).

**Remarque** : Pour définir des étiquettes de série dans des graphes de type "Nuage de points", il est conseillé d'utiliser la fonction [grNuageEtiquetteSérie](../WDLang3/3042038.md).


<a name="NOTE2_3"></a>


### Manipulation du libellé du graphe
<a name="manipulation_libelle_graphe_ELTPARAGRAPHE000133"></a>Le libellé du graphe peut être constitué de trois éléments :

- les étiquettes de catégories,

- la valeur des catégories,

- le pourcentage des catégories.




![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Libelle_Graphe.gif)


Le libellé du graphe peut être affiché uniquement pour les graphes en secteur, histogrammes et histogrammes groupés.

La fonction [grLibellé](../WDLang3/3042046.md) permet d'afficher les différents éléments du libellé du graphe. Les éléments affichés diffèrent selon le type de graphe.

Les étiquettes de catégories affichées correspondent à celles définies avec la fonction [grEtiquetteCatégorie](../WDLang3/3042015.md).

La valeur des catégories dépendent des données du graphe.

Le pourcentage des catégories est automatiquement calculé.

L'affichage de ces informations peut être mis en forme grâce à la fonction [grMasque](../WDLang3/3042034.md).


<a name="NOTE2_4"></a>


### Manipulation du titre du graphe
<a name="manipulation_titre_graphe_ELTPARAGRAPHE000168"></a>Le titre du graphe peut être affiché au-dessous ou au-dessus du graphe. ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Titre.gif)


Le contenu du titre du graphe est défini par la fonction [grTitre](../WDLang3/3042056.md). Cette fonction permet également de positionner ce titre.

La police utilisée pour le titre du graphe peut être modifiée grâce à la fonction [grPoliceTitre](../WDLang3/3042048.md).


<a name="NOTE2_5"></a>


### Manipulation du titre des axes du graphe
<a name="manipulation_titre_des_axes_graphe_ELTPARAGRAPHE000187"></a>Le titre des axes du graphe peut être affiché le long des différents axes. ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Titreaxe.gif)


Le contenu du titre des axes du graphe est défini par la fonction [grTitreAxe](../WDLang3/3042057.md). Cette fonction permet également de positionner ce titre selon l'axe voulu. Il n'est pas possible de définir le positionnement et / ou l'orientation du titre par rapport à l'axe.

La police utilisée pour le titre des axes du graphe peut être modifiée grâce à la fonction [grPoliceTitreAxe](../WDLang3/3042059.md).

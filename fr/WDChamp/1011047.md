


## Bloc Complément de corps
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000150"></a>
Le bloc **Complément de corps** est un bloc d'un état créé avec l'éditeur d'état. Le bloc **Complément de corps** est lié au bloc *Corps*. Ce bloc permet de compléter l'édition jusqu'en bas de page, s'il n'y a plus assez de lignes à éditer issues du bloc *Corps*.

Ce bloc s'imprime dès qu'il existe un espace libre entre le bloc *Corps* et le bloc *Pied de page*.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Création du bloc "Complément de corps"
<a name="creation_bloc_complement_corps_ELTTEXTE000174"></a>
Pour créer un bloc Complément de corps :

1. Effectuez une des manipulations suivantes sous l'éditeur d'états :

	- sous le volet "Création", dans le groupe "Structure", déroulez "Blocs".

	- sélectionnez l'option "Ajouter un bloc" du menu contextuel de l'état (clic droit sur l'état en cours de création).

	- cliquez le bouton "Ajouter un bloc" dans l'onglet "Blocs" de la description de l'état.




2. Sélectionnez le type "Complément de corps".




Remarque : Pour plus de détails sur les différentes options disponibles sur ce type de bloc, consultez [Description des blocs d'un état](../WDChamp/1011049.md).

<a name="NOTE3_1"></a>


## Impression du bloc Complément de corps
<a name="impression_bloc_complement_corps_ELTTEXTE000198"></a>


### Position d'impression du bloc Complément de corps
<a name="position_impression_bloc_complement_corps_ELTPARAGRAPHE000057"></a>Le bloc **Complément de corps** peut s'imprimer :

- après le bloc Corps : sélectionnez l'option "Juste après le bloc "Corps"" dans l'onglet "Général" de la description du bloc.

- avant le bloc Pied de page : sélectionnez l'option "Avant le bloc "Pied de page"" dans l'onglet "Général" de la description du bloc.




Par exemple, votre état table imprime des totaux dans le bloc *Bas de rupture*. Il est possible d'afficher ce bloc :

<a name="NOTE3_2"></a>


### Fréquence d'impression du bloc Complément de corps
<a name="frequence_impression_bloc_complement_corps_ELTPARAGRAPHE000088"></a>Le bloc **Complément de corps** peut s'imprimer :

- uniquement s'il y a un bloc Corps dans la page : sélectionnez l'option "Uniquement s'il y a un bloc "Corps" dans la page" dans l'onglet "Général" de la description du bloc.

- à chaque page de l'état : sélectionnez l'option "A chaque page" dans l'onglet "Général" de la description du bloc.
	Cette option peut être utile si la dernière page de votre état ne contient pas de bloc *Corps*.




## Visualisateur de rapports : Copie de texte
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000109"></a>
Le visualisateur de rapports permet de copier du texte présent dans l'état pour le coller dans un fichier d'une autre application.

Rappel : Le visualisateur de rapports est disponible : 

- sous l'éditeur d'états de WINDEV, WEBDEV, WINDEV Mobile. 

- dans le logiciel "Etats et Requêtes".

- lors de l'impression d'états depuis une application WINDEV. 




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000133"></a>
Pour copier un texte :

1. Dans le visualisateur de rapports, sous le volet " Aperçu   ", dans le groupe "Mode", cliquez sur "Sélection" (![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Apercu_ico_Copier.gif)).

2. Sélectionnez le texte à copier : sélectionnez la première lettre de la phrase ou de l'expression et faites glisser jusqu'à la dernière lettre. 
	Remarque : Le texte à sélectionner peut être présent sur plusieurs pages. 

3. Copiez le texte sélectionné. Vous pouvez : 

	- Utiliser la combinaison de touches Ctrl + C. 

	- Utiliser l'option "Copier" du menu contextuel de la sélection (clic droit). 




4. Collez le texte dans le logiciel voulu (par exemple en utilisant la combinaison de touches Ctrl + V). 
	Remarque : Si le logiciel cible le gère, les couleurs du texte copié sont respectées.  



<a name="NOTE2_2"></a>


### Cas particuliers
<a name="cas_particuliers_ELTPARAGRAPHE000055"></a>

- Les graphiques présents dans la sélection ne sont pas copiés. 

- Si l'état contient du texte au format RTF, pensez à utiliser la constante *iRTFAvecImagesEtTableaux* (fonction [iParamètre](../WDLang5/3046008.md)) pour optimiser la sélection de texte.

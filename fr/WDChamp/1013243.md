


## Personnaliser l'interface d'un champ Table
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000188"></a>
Dans un champ Table, il est possible de personnaliser les éléments suivants :

- [la couleur des lignes paires et des lignes impaires](#NOTE2_1).

- [le bandeau de sélection et le bandeau de survol des lignes](#NOTE2_2).

- [le titre des colonnes (images, icônes de tri, loupe, filtre, …)](../WDChamp/1013330.md)

- [les ascenseurs verticaux et horizontaux](#NOTE2_3).

- [le cadre extérieur et interne de la table](#NOTE2_4).




**Remarque** : La personnalisation de ces différents éléments permettent par exemple d'obtenir un style "Vista" sur des postes n'utilisant pas Windows Vista. Les différents gabarits proposés avec WINDEV contiennent des exemples d'images réutilisables et modifiables.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Personnalisation
<a name="personnalisation_ELTTEXTE000212"></a>


### Couleur de fond des lignes paires et des lignes impaires
<a name="couleur_fond_des_lignes_paires_des_lignes_impaires_ELTPARAGRAPHE000046"></a>Il est possible de différencier les lignes d'un champ Table en indiquant une couleur spécifique pour les lignes paires et les lignes impaires.

Pour indiquer la couleur de fond des lignes :

1. Affichez l'onglet "Style" de la fenêtre de description du champ Table (option "Description" du menu contextuel).

2. Sélectionnez l'élément "Couleurs lignes paires" ou "Couleurs lignes impaires".

3. Sélectionnez la couleur de fond à utiliser (combo "Couleur fond").



<a name="NOTE2_2"></a>


### Image du bandeau de sélection et du bandeau de survol des lignes
<a name="image_bandeau_selection_bandeau_survol_des_lignes_ELTPARAGRAPHE000059"></a>Il est possible d'utiliser une image pour visualiser correctement la ligne sélectionnée et la ligne survolée.
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=BandeauTable.gif)
Pour utiliser une image pour le bandeau de sélection et pour le bandeau de survol :

1. Affichez l'onglet "Style" de la fenêtre de description de la table (option "Description" du menu contextuel).

2. Sélectionnez l'élément "Ligne sélectionnée" ou "Ligne survolée".

3. Ouvrez la combo "Cadre" et sélectionnez l'option "Image (Mode 9 images)".

4. Cliquez sur le bouton "Flèche droite" situé à droite de la combo "Cadre". Une fenêtre de paramétrage de l'image du bandeau s'ouvre.

5. Sélectionnez l'image à utiliser en cliquant le bouton ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Menu_Image_Editeur%20-%20HC%20N%B0001.gif)
 : 

	- L'option "Parcourir" permet de sélectionner un fichier image disponible depuis votre poste. 

	- L'option "Catalogue" permet de choisir une image présente dans le catalogue d'images de WINDEV.




6. Si l'image sélectionnée contient plusieurs états, spécifiez ce nombre d'états. 

7. Fermez la fenêtre de paramétrage de l'image du bandeau.


**Remarque** : Le bandeau de sélection peut être affiché uniquement sur certaines colonnes d'une table. Pour plus de détails, consultez [Caractéristiques des colonnes de table](../WDChamp/1013330.md).

### Image des ascenseurs
<a name="image_des_ascenseurs_ELTPARAGRAPHE000094"></a>Il est possible d'utiliser une image pour représenter l'ascenseur vertical et l'ascenseur horizontal.

Pour utiliser une image pour les ascenseurs :

1. Affichez l'onglet "Style" de la fenêtre de description de la table (option "Description" du menu contextuel).

2. Sélectionnez l'élément "Ascenseur".

3. Ouvrez la combo "Cadre" et sélectionnez l'option "Image (Mode 9 images)".

4. Cliquez sur le bouton "Flèche droite" situé à droite de la combo "Cadre". Une fenêtre de paramétrage de l'image des ascenseurs s'ouvre.

5. Sélectionnez les images à utiliser en cliquant le bouton ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Menu_Image_Editeur%20-%20HC%20N%B0001.gif)
 : 

	- L'option "Parcourir" permet de sélectionner un fichier image disponible depuis votre poste. 

	- L'option "Catalogue" permet de choisir une image présente dans le catalogue d'images de WINDEV.




6. Ces images doivent respecter le format suivant :

	- Ascenseur vertical : Taille de l'image : 60 x 128 pixels. ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Ascenseur_V_Table.gif)


	- Ascenseur horizontal : Taille de l'image : 384 x 47 pixels. ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Ascenseur_H_Table.gif)

7. Fermez la fenêtre de paramétrage de l'image des ascenseurs.

### Image des cadres du champ Table
<a name="image_des_cadres_champ_table_ELTPARAGRAPHE000123"></a>Il est possible d'utiliser une image spécifique pour les cadres du champ Table.

Pour utiliser une image pour les cadres du champ Table :

1. Affichez l'onglet "Style" de la fenêtre de description du champ (option "Description" du menu contextuel).

2. Sélectionnez l'élément "Cadre extérieur" ou "Cadre interne".

3. Ouvrez la combo "Cadre" et sélectionnez l'option "Image (Mode 9 images)".

4. Cliquez sur le bouton "Flèche droite" situé à droite de la combo "Cadre". Une fenêtre de paramétrage de l'image du cadre s'ouvre.

5. Sélectionnez l'image à utiliser en cliquant le bouton ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Menu_Image_Editeur%20-%20HC%20N%B0001.gif)
 : 

	- L'option "Parcourir" permet de sélectionner un fichier image disponible depuis votre poste. 

	- L'option "Catalogue" permet de choisir une image présente dans le catalogue d'images de WINDEV.


 Pour augmenter l'épaisseur du cadre, il suffit d'augmenter son débordement.

6. Si l'image sélectionnée contient plusieurs états, spécifiez ce nombre d'états. 

7. Fermez la fenêtre de paramétrage de l'image du cadre.




## Options d'édition d'un état
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000377"></a>
L'éditeur d'états permet de :

- [Proposer plusieurs modes d'impression lors du test de l'état](#NOTE2_1).

- [Visualiser le nom des champs](#NOTE3_1).

- [Visualiser les marges de l'état](#NOTE3_2).

- [Visualiser le contenu des fichiers de données dans les champs en création](#NOTE3_3).

- [Visualiser la page utile de l'état](#NOTE3_4).

- [Afficher le nom des blocs complètement ou en abrégé](#NOTE4_1).

- [Afficher la langue associée à chaque police](#NOTE5_1).

- [Encadrer les champs et les blocs](#NOTE6_1).

- [Définir l'unité de mesure utilisée en affichage](#NOTE7_1). 




Ces fonctionnalités s'appliquent :

- soit uniquement à l'état en cours,

- soit uniquement à la prochaine ouverture des états.


La plupart de ces fonctionnalités sont accessibles :

- soit depuis le volet "Affichage",

- soit depuis la fenêtre de gestion des options d'édition de l'éditeur d'états. Pour afficher cette fenêtre, sous le volet "Affichage", cliquez sur l'icône de regroupement ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_regroup.gif)
 du groupe "Options".

## Proposer plusieurs modes d'impression lors du test d'un état
<a name="proposer_plusieurs_modes_impression_lors_test_etat_ELTTEXTE000401"></a>
Par défaut, lors du test d'un état, l'éditeur d'états propose plusieurs modes d'impression : 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=GO_Etat_choix.gif)


Pour plus de détails sur ces différents modes, consultez [Modes d'impression d'un état](../WDChamp/1011031.md).

Pour ne pas afficher cette fenêtre, il suffit de cocher l'option "Ne plus afficher", présente dans la fenêtre. Dans ce cas, l'état testé sera directement visualisé dans un aperçu avant impression.

Remarque : Il est possible de réactiver l'affichage de cette fenêtre. Pour plus de détails, consultez [Ne plus afficher un message](../Editeurs/2020011.md).

## Visualiser
<a name="visualiser_ELTTEXTE000425"></a>


### Visualiser le nom des champs
<a name="visualiser_nom_des_champs_ELTPARAGRAPHE000084"></a>Pour visualiser ou non le nom des champs sous l'éditeur d'états, sous le volet "Affichage", dans le groupe "Visualisation", cochez l'option "Noms des champs" (ou appuyez sur la touche F6).

Par exemple : 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=NOMCHAMPETAT.GIF)


Autre solution :

1. Sous le volet "Affichage", cliquez sur l'icône de regroupement ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_regroup.gif)
 du groupe "Options".

2. Affichez si nécessaire l'onglet "Affichage". 

3. Dans l'option "Nom des champs", sélectionnez : 

	- "Afficher" pour afficher le nom des champs,

	- "Désactivé" pour ne rien afficher.






<a name="NOTE3_2"></a>


### Visualiser les marges de l'état
<a name="visualiser_les_marges_etat_ELTPARAGRAPHE000121"></a>Les marges des états peuvent être :


|   |   |
| --- | --- |
| <br><br>- soit visibles.<br><br><br><br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=MARGEETATAFFICHE.GIF)<br> | <br><br>- soit invisibles.<br><br><br><br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=MARGEETATNONAFFICHE.GIF)<br> |

Pour modifier cet affichage dans l'état en cours, sous le volet "Affichage", dans le groupe "Visualisation", cochez ou décochez l'option "Marges logiques" (touche de raccourci F11).

Autre solution :

1. Sous le volet "Affichage", cliquez sur l'icône de regroupement ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_regroup.gif)
 du groupe "Options".

2. Affichez si nécessaire l'onglet "Affichage". 

3. Dans l'option "Marges de l'état", sélectionnez "Voir les marges" ou "Ne pas voir les marges".



<a name="NOTE3_3"></a>


### Visualiser le contenu des fichiers de données
<a name="visualiser_contenu_des_fichiers_donnees_ELTPARAGRAPHE000163"></a>L'éditeur d'états permet de visualiser le contenu des fichiers de données dans les champs d'un état en création. Il est ainsi possible de configurer simplement la taille d'un champ en fonction des données pouvant être affichées dans le champ.

Cette fonctionnalité s'appelle le "Live Data". Pour plus de détails, consultez [Live Data](../Editeurs/2021005.md). 
<a name="NOTE3_4"></a>


### Visualiser la page utile
<a name="visualiser_page_utile_ELTPARAGRAPHE000175"></a>L'éditeur d'états permet d'afficher l'état en cours tel qu'il sera lors de son impression :

- les marges de l'état sont invisibles,

- le nom des blocs de l'état est affiché en abrégé.


Pour visualiser l'état tel qu'il sera lors de son impression, utilisez la touche de raccourci F5.

## Afficher le nom des blocs complètement ou en abrégé
<a name="afficher_nom_des_blocs_completement_abrege_ELTTEXTE000467"></a>
Le nom des blocs est affiché à gauche des états. Ces noms peuvent être affichés :


|   |   |
| --- | --- |
| <br><br>- soit complètement.<br><br><br><br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=NOMBLOCCOMPLET.GIF)<br> | <br><br>- soit en abrégé.<br><br><br><br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=NOMBLOcABReGe.GIF)<br> |

Pour modifier cet affichage dans l'état en cours, sous le volet "Affichage", dans le groupe "Visualisation", cochez ou décochez l'option "Nom complet des blocs" (touche de raccourci F10).

Autre solution :

1. Sous le volet "Affichage", cliquez sur l'icône de regroupement ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_regroup.gif)
 du groupe "Options".

2. Affichez si nécessaire l'onglet "Affichage". 

3. Dans l'option "Voir le nom des blocs", sélectionnez "Complètement" ou "En abrégé".

## Afficher la langue associée à chaque police
<a name="afficher_langue_associee_chaque_police_ELTTEXTE000491"></a>
Pour modifier la police des champs d'un état, il suffit d'utiliser :

- Soit la combo "Police" de l'onglet "Style" de la description des champs.

- Soit la combo "Police" de la barre contextuelle de l'éditeur d'états.


Par défaut, dans cette combo "Police", seul le nom des polices installées sur le poste de développement est affiché.
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Police_Langues%20-%20HC%20N%B0001.gif)


Sous Windows, une même police peut être associée à plusieurs groupes de langues. Pour compléter chaque nom de police par la langue associée :

1. Dans l'éditeur d'états, sous le volet "Affichage", cliquez sur l'icône de regroupement ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_regroup.gif)
 du groupe "Options".

2. Affichez si nécessaire l'onglet "Préférences". 

3. Cochez l'option "Afficher toutes les langues disponibles".
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Police_Langues%20-%20HC%20N%B0003.gif)





Dans l'onglet "Style" d'un champ, la liste des polices deviendra : 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Police_Langues%20-%20HC%20N%B0004.GIF)


**Remarque** : Cette option sera prise en compte uniquement à la prochaine ouverture d'un état.

## Visualiser les champs et les blocs grâce à un cadre
<a name="visualiser_les_champs_les_blocs_grace_cadre_ELTTEXTE000515"></a>
L'éditeur d'états permet de visualiser rapidement l'ensemble des champs et des blocs présents dans l'état en cours. 

Il est possible : 

- d'encadrer d'un simple trait les champs et les blocs. Ce trait apparaît uniquement en édition. Ce trait n'est pas visible en exécution.
	Cet encadrement permet de visualiser plus facilement la zone occupée par les champs et les blocs présents dans un état (notamment pour les champs image).

- de visualiser uniquement les coins des champs et des blocs. Ce mode de visualisation plus léger qu'un trait permet de visualiser rapidement les différentes zones.

- de n'utiliser aucun mode de visualisation spécifique. 


Pour changer de mode d'affichage dans l'état en cours, sous le volet "Affichage", dans le groupe "Visualisation", sélectionnez le mode d'affichage voulu : 

- Coins

- Cadre complet

- Aucun


La touche F7 permet de modifier dynamiquement le mode d'affichage.

Autre solution :

1. Sous le volet "Modification", cliquez sur l'icône de regroupement ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_regroup.gif)
 du groupe "Options".

2. Affichez si nécessaire l'onglet "Affichage". 

3. Dans l'option "Encadrement des champs", sélectionnez "Aucun", "Coins seulement" ou "Cadre complet".

## Unité de mesure
<a name="unite_mesure_ELTTEXTE000539"></a>
L'unité de mesure utilisée sous l'éditeur d'états peut être soit les millimètres, soit les pouces. 

Pour choisir l'unité de mesure utilisée sous l'éditeur d'états : 

1. Sous le volet "Affichage", cliquez sur l'icône de regroupement ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_regroup.gif)
 du groupe "Options".

2. Affichez si nécessaire l'onglet "Préférences". 

3. Dans la section "Unité de mesure", sélectionnez l'unité utilisée en affichage : millimètres ou pouces.

4. Validez. L'unité de mesure sélectionnée sera utilisée pour les prochains états ouverts sous l'éditeur d'états.

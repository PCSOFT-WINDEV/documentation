
## FAA : Zoomer le contenu d'un champ (Table, Tableur, Liste, Tableau croisé dynamique, ...)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000144"></a>
Pour faciliter la lecture dans les champs permettant de visualiser des volumes importants de données, l'utilisateur final a la possibilité de zoomer les données contenues dans le champ. 

Les données affichées peuvent être zoomées dans les champs suivants : 

- Champ Table, 

- Champ Table présent dans un champ Combo,

- Champ Table hiérarchique, 

- Champ Tableau Croisé Dynamique,

- Champ Tableur,

- Champ Liste.

## Zoomer/Dézoomer le contenu d'un champ Table
<a name="zoomerdezoomer_contenu_champ_table_ELTTEXTE000168"></a>
**Pour zoomer le contenu du champ** : 
L'utilisateur peut zoomer le contenu d'un champ Table en utilisant la touche Ctrl et la roulette de la souris. 

**Pour revenir à l'affichage initial**, il suffit d'utiliser l'option "Annuler le zoom" du menu contextuel du champ (ou la combinaison de touches Ctrl + 0). 
<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=FAA_Zoom_Table.gif&type=thumb)



### Conserver le zoom
<a name="conserver_zoom_ELTPARAGRAPHE000037"></a>

- **Sur les champs Table et Table hiérarchique**, le zoom est automatiquement sauvegardé et réappliqué au prochain affichage du champ si l'option "Mémoriser la config. des colonnes" est cochée (onglet "Détail" de la fenêtre de description du champ). 

- **Sur le champ Tableau croisé dynamique**, le zoom est automatiquement sauvegardé.

## Programmation
<a name="programmation_ELTTEXTE000198"></a>


### Programmer le zoom
<a name="programmer_zoom_ELTPARAGRAPHE000053"></a>Il est possible de programmer le zoom à appliquer au champ (champ Table, champ Table hiérarchique, champ Tableau croisé dynamique, champ Tableur, ...) en utilisant la propriété [Zoom](../Proprietes/1000017201.md) sur le champ. 

Par exemple : 


```wl
// Agrandissement à 150%
TABLE_Client.Zoom = 150
```


Si le zoom est réalisé par programmation, le menu contextuel du champ permet d'annuler le zoom. 


### Désactiver le zoom
<a name="desactiver_zoom_ELTPARAGRAPHE000067"></a>Pour ne pas autoriser le zoom sur un champ, il suffit d'utiliser la fonction [FAADésactive](../WDLang1/1000022018.md) avec la constante *faaZoom*. 

Par exemple : 


```wl
// Pas de zoom possible
FAADésactive(TABLE_Client, faaZoom)
```



### Annuler le zoom
<a name="annuler_zoom_ELTPARAGRAPHE000079"></a>Pour annuler le zoom sur un champ Table (et revenir à l'état initial), il suffit d'utiliser la fonction [FAAExécute](../WDLang1/1000022099.md) avec la constante *faaZoom*.


### Modifier le libellé de l'option permettant d'annuler le zoom (par programmation)
<a name="modifier_libelle_option_permettant_annuler_zoom_par_programmation_ELTPARAGRAPHE000087"></a>Pour annuler le zoom, l'utilisateur final peut afficher le menu contextuel du champ (clic droit) et sélectionner l'option "Annuler le zoom". 

Pour modifier le libellé de cette option, il suffit d'utiliser la fonction [FAAChangeLibellé](../WDLang1/1000022100.md) avec la constante *faaZoom*.

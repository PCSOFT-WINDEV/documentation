


## Les graphes de type "Radar"
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000162"></a>
Le graphe de type Radar permet de mettre en valeur une série par rapport à une autre.

Dans un graphe de type Radar, il y autant d'axes que de catégories. Tous les axes partent du point central.

Les catégories sont indiquées autour du graphique. Les valeurs des séries sont affichées à l'intérieur du graphe.





<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Caractéristiques d'un radar
<a name="caracteristiques_radar_ELTTEXTE000186"></a>

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=GenRadar.gif&type=thumb)
Dans ce graphe :

**1. Point du radar avec sa valeur.**
La valeur du point est affichée si  les conditions suivantes sont réunies :
l'option "Afficher les valeurs" est cochée dans l'onglet "Détail" de la description du champ Graphe.

- le type de point est défini pour chaque série dans l'onglet "Séries" de la fenêtre de description du point. Si le type de point est "Non dessiné", le point ne sera pas mis en valeur et sa valeur ne sera pas affichée.


La taille des points est définie dans l'onglet "Général" de la fenêtre de description du champ.  

**2. Titre du graphe**. 
Les caractéristiques du titre du graphe sont saisies dans l'onglet "Détail" de la description du champ (bouton "Editer le titre du graphe").
Il est possible de définir le titre du graphe, sa position et son style.

**3. Etiquette de catégorie**. 
Les étiquettes de catégorie peuvent être définies dans l'onglet "Séries" de la description du champ. Pour afficher les étiquettes de catégorie, il suffit de cocher l'option "Afficher les étiquettes des catégories" dans l'onglet "Général" de la fenêtre de description du graphe.

**4. Légende**. 
Les caractéristiques de la légende sont saisies dans l'onglet "Détail" de la description du champ (bouton "Editer la légende du graphe").
Il est possible de définir sa position et son style.

**5. Etiquettes de séries**. 
Le libellé de la légende correspond au libellé des différentes séries affichées dans le graphe (définie dans l'onglet "Séries").

**6. Couleur de série**. 
Pour chaque série, il est possible de définir une couleur dans l'onglet "Séries" de la description du champ. Par défaut la couleur utilisée est automatique.

**7. Quadrillage**. 
Il est possible d'afficher le quadrillage grâce à l'option "Afficher le quadrillage" dans l'onglet "Général" de la description du champ.

**8. Graduation**. 
La graduation est calculée automatiquement. Par conséquent, la fonction [grOrigine](../WDLang3/3042039.md) sera sans effet sur ce type de graphe.

Remarque : Le zoom n'est pas supporté sur un graphe radar.


## Dessinez des graphes dans un champ Image
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000110"></a>
WINDEV, WINDEV Mobile et WEBDEV mettent à votre disposition des fonctions qui permettent de créer et de manipuler des graphes.

Il est possible de réaliser plusieurs types de graphes : Secteurs, Histogramme, Courbe, Nuage de points, ... Pour chacun des types de graphes, des options d'affichage permettent d'avoir des rendus différents.

Ces graphes peuvent être dessinés :

- dans un fichier image,

- dans un champ Image. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) dans une fenêtre,

- dans une variable de type [Image](../WDLang1/1000019650.md).




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment dessiner ?
<a name="comment_dessiner_ELTTEXTE000134"></a>
Pour dessiner un graphe dans un champ Image, quelques règles doivent être suivies :

1. Créez le graphe avec la fonction [grCrée](../WDLang3/3042005.md).

2. Initialisez les données du graphe, avec les fonctions [grAjouteDonnée](../WDLang3/3042023.md) et [grNuageAjouteDonnéeXY](../WDLang3/3042032.md).

3. Choisissez un champ Image comme destination du graphe (fonction [grDestinationChamp](../WDLang3/3042020.md))

4. Dessinez le graphe avec la fonction [grDessine](../WDLang3/3042026.md).




Exemple :


```wl
grCrée("Graphe1",grSecteur)
grAjouteDonnée("Graphe1",1,50)
grAjouteDonnée("Graphe1",1,90)
grAjouteDonnée("Graphe1",1,500)
grDestinationChamp("Graphe1", IMG_Image1)
grDessine("Graphe1")
```

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000188"></a>
Un menu principal est obligatoirement associé à une fenêtre. Ce menu est affiché sous la barre de titre de la fenêtre. En règle générale, les applications contiennent une fenêtre menu. Cette fenêtre contient uniquement un menu principal et permet d'accéder aux autres fenêtres de l'application.

## Comment créer, éditer et supprimer un menu principal ?
<a name="comment_creer_editer_supprimer_menu_principal_ELTTEXTE000236"></a>


### Créer un menu principal
<a name="creer_menu_principal_ELTPARAGRAPHE000039"></a>Pour créer un menu principal :

1. Sous l'éditeur de fenêtres, ouvrez la fenêtre dans laquelle un menu principal doit être ajouté.

2. Sous le volet "Fenêtre", dans le groupe "Barres et menus", déroulez "Menu principal" et sélectionnez "Ajouter le menu principal". La première option de menu apparaît dans la barre de menu. Par défaut, le nom du menu principal créé est "_Menu" et le libellé de cette première option est "Menu".
	![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Le menu est automatiquement créé dans le champ Action Bar.

3. Pour modifier le nom du menu principal : 

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) affichez l'onglet "Général" de la description de la fenêtre (option "Description" du menu contextuel de la fenêtre) et saisissez le nouveau nom du menu principal dans l'option "Nom du menu principal".




4. Ajoutez des options de menu directement dans la fenêtre en édition. Pour plus de détails, consultez [Manipuler les options de menu](../WDChamp/1010008.md).




**Astuce** : Pour ajouter des images dans les options du menu, utilisez le catalogue d'images (option "Catalogue" du bouton ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Menu_Image_Editeur%20-%20HC%20N%B0001.gif)
). Les images sélectionnées seront automatiquement adaptées à la taille et au format d'image attendu par la plateforme d'exécution utilisée.

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png) **Remarque : Menu dans les fenêtres détourées** : Il est possible de spécifier la position du menu principal pour l'adapter au mieux à la fenêtre. Pour positionner le menu :

1. Affichez la description de la fenêtre (option "Description" du menu contextuel de la fenêtre).

2. Dans l'onglet "Général", saisissez les coordonnées X et Y du menu.



<a name="NOTE3_2"></a>


### Editer un menu principal
<a name="editer_menu_principal_ELTPARAGRAPHE000088"></a>Pour éditer un menu principal : 

1. Ouvrez une fenêtre avec un menu principal sous l'éditeur de fenêtres.

2. Sélectionnez une des options de menu. Les options et les sous-options s'affichent automatiquement.



<a name="NOTE3_3"></a>


### Supprimer un menu principal
<a name="supprimer_menu_principal_ELTPARAGRAPHE000098"></a>Pour supprimer un menu principal :

1. Sous l'éditeur de fenêtres, ouvrez la fenêtre dans laquelle le menu principal doit être supprimé.

2. Sous le volet "Fenêtre", dans le groupe "Barres et menus", déroulez "Menu principal" et sélectionnez "Supprimer le menu principal". Toutes les options de menu et le code associé à ce menu principal sont supprimés.

## Manipuler un menu principal par programmation
<a name="manipuler_menu_principal_par_programmation_ELTTEXTE000278"></a>
Il n'est pas possible de manipuler directement un menu principal par programmation. Cependant, il est possible de manipuler par programmation les options d'un menu principal. Pour plus de détails, consultez [Manipuler les options de menu par programmation](../WDChamp/1010007.md).


- ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=WDMenuAnime.gif) ***Exemples didactiques (WINDEV)*** : **WD Menu Animé** <br>L'exemple "WD Menu Animé" propose différentes idées de design de menu avec animations. <br><br>Ces animations permettent de rendre le menu principal plus attractif et dynamique à l'ouverture de la fenêtre : <br>- apparition en fondu des boutons de menu,<br>- déplacement des boutons,<br>- etc. <br><br>Et également, il présente un menu animé rempli dynamiquement (dont les options sont "extraites" d'un fichier HFSQL).
- ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=WDOSD.gif) ***Exemples didactiques (WINDEV)*** : **WD OSD** <br>OSD signifie : On-Screen Display. C'est le nom donné aux interfaces qui apparaissent sur les écrans d'ordinateur ou des téléviseurs par exemple. Ils servent souvent à effectuer des réglages.<br>Ces types de menus ont comme particularité d'être affichés par dessus tout ce qui se trouve à l'écran.<br>Cet exemple montre comment réaliser ce type d'interface sous WINDEV.




## Manipuler un champ Graphe par programmation
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000133"></a>
WINDEV, WINDEV Mobile et WEBDEV permettent de manipuler un champ Graphe directement par programmation. Il suffit d'utiliser directement la variable du champ Graphe dans le code.

La variable du champ Graphe correspond au nom du champ Graphe. Cette variable est de type entier.

Le WLangage propose également de nombreuses fonctions pour manipuler un champ Graphe. Pour plus de détails, consultez les [fonctions de gestion des graphes](../WDLang3/3042003.md).  



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Manipuler les séries d'un champ Graphe
<a name="manipuler_les_series_champ_graphe_ELTTEXTE000157"></a>
Pour manipuler une série d'un champ Graphe, utilisez la syntaxe suivante :

```txt
<Nom du graphe>[Indice de la série].<Nom de la propriété>
```


<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Propriétés spécifiques aux graphes
<a name="proprietes_specifiques_aux_graphes_ELTTEXTE000181"></a>
Les propriétés suivantes sont spécifiques à la gestion des champs Graphe par programmation.


|   |   |
| --- | --- |
| [DuréeAnimation](../Proprietes/1000021069.md) | Permet de connaître et de modifier la durée de l'animation d'un champ Graphe. Cette propriété permet également de supprimer l'animation d'un champ Graphe. |
| [Occurrence](../WDLang1/3025009.md) | Permet de connaître le nombre de séries présentes dans un graphe. |
| [Valeur](../Proprietes/2510130.md) | Permet de connaître le nom d'un champ Graphe. |
| [Vide](../Proprietes/2510126.md) | Permet de savoir si un champ Graphe ne contient aucune série. |


Pour connaître la liste complète des propriétés du WLangage pouvant être utilisées avec un champ Graphe, consultez [Propriétés associées à un champ Graphe](../WDChamp/1013008.md).

Pour connaître la liste complète des propriétés du WLangage pouvant être utilisées avec une série d'un champ Graphe, consultez [Propriétés associées aux séries d'un champ Graphe](../WDChamp/1013011.md).

<a name="NOTE4"></a>
<a name="NOTE4_1"></a>


## Remarques
<a name="remarques_ELTTEXTE000205"></a>
Un champ Graphe peut être manipulé dans un seul thread. Un champ Graphe peut être manipulé uniquement dans le thread qui l'a créé, c'est-à-dire dans le thread principal de la fenêtre. 

Le champ Graphe est manipulable uniquement dans le code qui est exécuté directement (ou indirectement) depuis les traitements de la fenêtre et de ses champs.

**Conséquences** : Deux threads ne peuvent pas manipuler le même champ Graphe.

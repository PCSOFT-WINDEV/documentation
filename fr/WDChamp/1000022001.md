## Présentation du champ Popup
<a name="presentation_champ_popup_ELTTEXTE000244"></a>
Le champ Popup est un champ permettant d'associer une popup à une page. Cette popup sera affichée à partir de la page dans laquelle elle est insérée. Cette popup permet de dialoguer avec l'utilisateur d'une manière simplifiée. 

Jusqu'en version 17, ce mode de dialogue était réalisé avec un [champ Cellule](../WDChamp/1013096.md). 

L'utilisation du champ Popup permet de manipuler sous l'éditeur la fenêtre de dialogue comme un élément différent de la page, tout en étant intégré à la page. Par rapport à l'utilisation des cellules, sous l'éditeur, l'utilisation de popups ne perturbe pas la mise en page.

## Créer un champ Popup
<a name="creer_champ_popup_ELTTEXTE000268"></a>


### Créer un champ de type Popup
<a name="creer_champ_type_popup_ELTPARAGRAPHE000025"></a>La création d'une page Popup peut être effectuée à partir du bandeau de la page : 

1. Dans le bandeau de la page en cours d'édition, cliquez sur "Pages Popup". 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=WB_Popup_creation%20-%20HC%20N%B0001.gif)


2. Dans le menu affiché, cliquez sur "Nouvelle popup". 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=WB_Popup_creation%20-%20HC%20N%B0002.gif)


3. La page popup apparaît en création sous l'éditeur. 


Pour afficher les caractéristiques du champ Popup, sélectionnez l'option "Description" dans le menu contextuel du champ.

**Remarques** : 

- Une page peut contenir une ou plusieurs popups. Pour afficher la popup voulue en exécution, il suffit d'utiliser la fonction [PopupAffiche](../WDLang2/1000019788.md). 

- Un modèle de pages peut contenir une ou plusieurs popups. Les pages utilisant ce modèle pourront utiliser ces popups. 

- Un modèle de champs peut utiliser une ou plusieurs popups (exemple, un sélecteur de couleur).

- Le menu d'une page peut utiliser des options de type Popup. Pour plus de détails, consultez [Configurer un menu Popup](../WDChamp/1010042.md). 

- Une page peut être affichée sous forme de popup grâce à la fonction [PopupAffichePage](../WDLang2/1410087258.md). 





### Autres modes de création
<a name="autres_modes_creation_ELTPARAGRAPHE000059"></a>

- **Créer une popup à partir d'une cellule existante** : 
	Pour créer une popup directement à partir d'une cellule existante : 

	1. Sélectionnez la cellule voulue. 

	2. Sous le volet "Modification", dans le groupe "Transformations", déroulez "Refactoring et permutations" et sélectionnez "Cellule/Page Popup".


 Remarque : Si la cellule n'est pas présente dans un modèle et si elle est superposable, une option du menu contextuel permet également de la transformer en popup. 

- **Créer une popup à partir d'un ensemble de champs** : 

	1. Sélectionnez les champs voulus dans la page. 

	2. Effectuez une des opérations suivantes : 

		- sélectionnez l'option "Refactoring .. Créer une page Popup avec la sélection" du menu contextuel.

		- sous le volet "Modification", dans le groupe "Transformations", déroulez "Refactoring et permutations" et sélectionnez "Créer une page Popup avec la sélection".

## Caractéristiques du champ Popup
<a name="caracteristiques_champ_popup_ELTTEXTE000298"></a>


### Fenêtre de description
<a name="fenetre_description_ELTPARAGRAPHE000144"></a>La fenêtre de description du champ Popup permet de définir notamment : 

- les différentes procédures locales (serveur et navigateur) associées à la popup (onglet "Général"). 

- la possibilité de déplacer la popup à la souris (onglet "UI"). 

- le taux de GFI (Grisage des fenêtres inaccessibles) associé à ce champ (60% par défaut)

- le look de la popup (onglet "Style"). Il est possible de sélectionner une image de fond spécifique pour la popup ou encore de définir un cadre.

## Manipuler un champ Popup sous l'éditeur
<a name="manipuler_champ_popup_sous_editeur_ELTTEXTE000322"></a>


### Fenêtre de gestion des popups
<a name="fenetre_gestion_des_popups_ELTPARAGRAPHE000158"></a>Les différentes popups associées à une page peuvent être connues : 

- dans le volet "Explorateur de projet", lors de la visualisation des informations de la page. Par exemple : 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Popup_Description.gif)


- sous le volet "Page", dans le groupe "Edition", cliquez sur "Pages popup".



![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Popup_FenDescription.gif)
Cette dernière option permet notamment de : 

- Créer une nouvelle popup.

- Supprimer une popup existante. 

- Afficher la fenêtre de description d'une popup.

- Ouvrir une popup sous l'éditeur.

- Renommer une popup. 



<a name="NOTE4_2"></a>


### Manipulation des popups associées à une page
<a name="manipulation_des_popups_associees_une_page_ELTPARAGRAPHE000190"></a>Sous l'éditeur, les popups associées à la page sont accessibles via le bandeau de la page en cours d'édition (bouton "Pages Popup"). 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=WB_Popup_creation%20-%20HC%20N%B0001.gif)
Un clic sur ce bouton permet d'afficher l'aperçu des popups associées à la page. Il est possible d'ouvrir sous l'éditeur la popup voulue en cliquant sur l'aperçu correspondant. Chaque popup ouverte sous l'éditeur est affichée dans un nouvel onglet de l'éditeur. 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=popup_barre_doc.gif)



- ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=WebApp.gif) ***Exemples didactiques (WEBDEV)*** : **WebApp** <br>Cet exemple présente une utilisation didactique des plans en WEBDEV.
- ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=WW_Newsletter.gif) ***Exemples complets (WEBDEV)*** : **WW_Newsletter** <br>WW_Newsletter permet de créer un site de gestion de newsletters. Il permet de créer, rédiger et gérer l'envoi en masse de newsletters à des abonnés. Les fonctionnalités du site sont les suivantes :<br>- inscription (double opt-in) et désinscription en ligne des abonnés<br>- consultation des newsletters publiques<br>- gestion des catégories de newsletters<br>- tableau de bord d'administration<br>- rédaction et mise en forme des newsletters en ligne<br>- création de modèles de newsletters<br>- etc.

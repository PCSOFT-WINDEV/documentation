
## Champ Image : Mode de fonctionnement d'une image cliquable
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000110"></a>
Il est possible de définir les actions de clic d'un champ Image (onglet "Général" de la fenêtre de description du champ) pour que celui-ci deviennent un champ Image cliquable.  
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=image_clic_modeFoctionnemen.gif)


<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Action lors du clic : Pas d'appel serveur
<a name="action_lors_clic_pas_appel_serveur_ELTTEXTE000134"></a>
Si l'action lors du clic correspond à "Pas d'appel serveur", le champ Image correspond à une image cliquable simple, associée uniquement à un code de clic navigateur. Il est possible d'associer une action au clic sur l'image : 

- sous l'éditeur, via le champ "Type". 

- par programmation en code navigateur uniquement. 




Cependant, il n'est pas possible de récupérer les coordonnées de la souris ou de définir des zones de clic graphiques.



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Action lors du clic : Appel serveur
<a name="action_lors_clic_appel_serveur_ELTTEXTE000158"></a>
Si l'action lors du clic correspond à "Appel serveur", l'image est une image cliquable associée à un code de clic serveur. 

Si le champ "Lors de l'action" correspond à "Envoyer les coordonnées du clic", il est possible de récupérer en code serveur les coordonnées de clic de la souris (fonction [PageParamètre](../WDLang2/3058026.md) avec le paramètre "POS" ou fonctions [ImagePosX](../WDLang2/3045003.md) et [ImagePosY](../WDLang2/3045001.md)). 

Pour plus de détails, consultez [Récupérer les coordonnées du clic](../WDChamp/1013130.md).

<a name="NOTE4"></a>
<a name="NOTE4_1"></a>


## Action lors du clic : Zones de clic définies en édition (Map area)
<a name="action_lors_clic_zones_clic_definies_edition_map_area_ELTTEXTE000182"></a>
Si l'action lors du clic correspond à "Zones de clic définies en édition (Map area)", l'image est une image cliquable avec définition de zones graphiques cliquables. Ce type d'image est plus connu sous le terme de Map Area.

Il est possible d'associer une action au clic sur l'image et sur chaque zone graphique définie.

Pour plus de détails, consultez [Définir des zones de clic graphiques](../WDChamp/1013157.md).

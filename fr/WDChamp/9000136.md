
## Animation de fenêtres mobiles
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000078"></a>
Dans les applications mobiles, les différents écrans de l'application sont affichés via une petite animation : affichage par la droite, par le haut, ...

WINDEV Mobile permet de configurer ce mode d'affichage de vos fenêtres dans vos applications mobiles.  

Remarque : Pour que l'animation soit exécutée, la fenêtre doit posséder un champ Action Bar. 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000102"></a>
Pour paramétrer l'animation d'une fenêtre :

1. Ouvrez la fenêtre à animer sous l'éditeur de fenêtres.  

2. Affichez la description de la fenêtre (option "Description" du menu contextuel).

3. Dans l'onglet "UI", choisissez l'animation à effectuer : <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Animation_fenetre_mobile%20-%20HC%20N%B0001.gif)


4. Validez.




**Remarque** : En mode automatique, l'animation système est utilisée **sauf** dans le cas de l'ouverture d'une fenêtre depuis un champ présent dans un champ Zone répétée. Dans ce cas, l'ouverture de la fenêtre est réalisée de la droite vers la gauche.

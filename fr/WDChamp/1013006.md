## Présentation du champ Graphe
<a name="presentation_champ_graphe_ELTTEXTE000177"></a>
Le champ Graphe permet de visualiser un graphe dans vos fenêtres, vos pages et vos états. 

Les types de graphes disponibles sont les suivants :

- [Graphe de type Secteur (ou Camembert), Beignet ou Hémicycle](../WDChamp/3042007.md).

- [Graphe de type Histogramme](../WDChamp/3042011.md).

- [Graphe de type Courbe](../WDChamp/3042002.md).

- [Graphe de type Nuage de points](../WDChamp/3042008.md).

- [Graphe de type Boursier](../WDChamp/3042004.md).

- [Graphe de type Radar](../WDChamp/3042073.md).

- [Graphe de type Aire](../WDChamp/3042074.md).

- [Graphe à bulles](../WDChamp/1000019190.md)
	**Remarque** : Dans l'éditeur d'états, les graphes à bulles ne sont pas disponibles.

- [Graphe de type Entonnoir](../WDChamp/1000019193.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) [Graphe de type Surface](../WDChamp/1000020624.md).

- [Graphe de type Waterfall](../WDChamp/1000021009.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) [Graphe de type Secteur multiniveau](../WDChamp/1000021010.md).

## Création d'un champ Graphe
<a name="creation_champ_graphe_ELTTEXTE000312"></a>
**Pour créer un champ de type Graphe** :

- dans une fenêtre ou une page : 

	1. Sous le volet "Création", dans le groupe "Champs graphiques", cliquez sur "Graphe". 

	2. Cliquez dans la fenêtre ou la page à la position où le champ doit être créé. L'assistant de création d'un champ Graphe se lance automatiquement.
			**Astuce** : Il est également possible de créer directement un champ Graphe en choisissant son type : il suffit de dérouler l'option "Graphe" sous le volet "Création". La liste des différents types de graphes disponibles apparaît. Il suffit de cliquer sur le type de graphe voulu pour le créer dans l'éditeur. 




- dans un état : 

	1. Sous le volet "Création", dans le groupe "Données", cliquez sur "Graphe". 

	2. Cliquez dans l'état à la position où le champ doit être créé. L'assistant de création d'un champ Graphe se lance automatiquement.





Remarque : Les dimensions du champ créé sont optimisées pour occuper l'espace disponible à la position indiquée. Si la taille du champ ne vous convient pas, utilisez la combinaison de touches Ctrl + Z : le champ retrouvera sa taille par défaut.

Pour afficher les caractéristiques du champ, sélectionnez l'option "Description" dans le menu contextuel du champ. Pour plus de détails, consultez [Champ Graphe : Caractéristiques sous l'éditeur](../WDChamp/3042076.md). 


### Echantillonnage automatique
<a name="echantillonnage_automatique_ELTPARAGRAPHE000119"></a>Dès qu'un graphe contient un nombre important de données et que celles-ci deviennent illisibles, un échantillonnage (également appelé downsampling) est automatiquement réalisé. 
![En haut : un graphe sans échantillonnage. En bas : le même graphe avec échantillonnage](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=WD-Graphe-Echatillonage-Sub-Sampling.jpg)


Remarque : l'échantillonnage est réalisé automatiquement. Il est possible de le désactiver avec la fonction [grParamètre](../WDLang3/1000018337.md).

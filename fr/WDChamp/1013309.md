


## Manipuler une barre d'outils par programmation (syntaxe préfixée)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000086"></a>
WINDEV permet de manipuler un champ Barre d'outils directement par programmation. Il suffit d'utiliser directement la variable du champ "Barre d'outils" dans le code.

La variable du champ Barre d'outils correspond au nom de la barre d'outils. Cette variable n'a aucune valeur.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Manipuler les champs présents dans un champ Barre d'outils
<a name="manipuler_les_champs_presents_dans_champ_barre_outils_ELTTEXTE000114"></a>
Les champs présents dans la barre d'outils sont accessibles :

- soit comme tous les champs, en utilisant directement leur nom.

- soit en précisant le nom du champ Barre d'outils puis le nom du champ. Par exemple : 
	
	```txt
	Barre1.Bouton1
	```





**Remarque** : les propriétés de taille et de position des champs présents dans la barre d'outils ne sont accessibles qu'en lecture.

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Comment fermer un champ Barre d'outils flottant ?
<a name="comment_fermer_champ_barre_outils_flottant_ELTTEXTE000147"></a>
Pour fermer un champ Barre d'outils flottant en utilisant la croix, il faut utiliser la ligne de code suivante dans l'événement "Fermeture" du champ Barre d'outils :

```wl
MoiMême..Visible = Faux
```
**Attention** : Il est déconseillé d'utiliser la fonction [Ferme](../WDLang1/3038018.md). En effet, cette fonction ferme la fenêtre et non le champ Barre d'outils.

## Fonctions spécifiques à la manipulation de champs Barre d'outils
<a name="fonctions_specifiques_manipulation_champs_barre_outils_ELTTEXTE000178"></a>
Les fonctions suivantes sont spécifiques à la manipulation du champ Barre d'outils par programmation :



|   |   |
| --- | --- |
| [&lt;Barre d'outils&gt;.Attache](../WDLang1/3020006.md) | Ancre un champ Barre d'outils à la fenêtre ou change la position d'ancrage d'un champ Barre d'outils ancré. |
| [&lt;Barre d'outils&gt;.Détache](../WDLang1/3020008.md) | Transforme un champ Barre d'outils ancré en champ Barre d'outils flottant. |
| [&lt;Barre d'outils&gt;.Position](../WDLang1/3020007.md) | Indique le mode d'affichage d'un champ Barre d'outils. |

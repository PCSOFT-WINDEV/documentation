## Présentation du champ Note repositionnable
<a name="presentation_champ_note_repositionnable_ELTTEXTE000139"></a>
Le champ Note repositionnable permet d'intégrer dans vos applications un petit papier jaune bien connu. 

Vous pouvez ainsi afficher dans ce champ des informations spécifiques, importantes pour l'utilisateur final. 

Plusieurs types d'utilisations sont possibles. Voici quelques exemples : 

- utilisation du champ Note repositionnable pour afficher une donnée présente dans la base de données (par exemple des commentaires sur le type de livraison pour chaque client). 

- utilisation du champ Note repositionnable pour rappeler une action à faire à l'utilisateur. Dans ce cas, le texte de la note est fixe et non lié à la base de données. 

<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Champ_Note_Go%20-%20HC%20N%B0001.gif)
Remarque : L'utilisateur final peut manipuler les notes repositionnables, les fermer, les supprimer. Il peut également en ajouter grâce à la FAA (Fonctionnalité Automatique de vos Applications) de création de notes. Pour plus de détails, consultez [Note repositionnable (FAA)](../Editeurs/1000021820.md).

## Créer un champ Note repositionnable
<a name="creer_champ_note_repositionnable_ELTTEXTE000163"></a>
Pour créer un champ de type Note repositionnable : 

1. Sous le volet "Création", dans le groupe "Autres champs", sélectionnez "Note reposi.".

2. Cliquez dans la fenêtre à la position où le champ doit être créé. Le champ apparaît en création.




Pour afficher les caractéristiques du champ, sélectionnez l'option "Description" dans le menu contextuel du champ. 

**Remarques** : 

- Lorsqu'un champ Note repositionnable est présent dans une fenêtre, aucune date n'apparaît au bas de la note. Si l'utilisateur modifie le texte de la note, la date de modification sera automatiquement ajoutée au bas de la note.  

- Pour créer un champ Note repositionnable par programmation, utilisez la fonction [ChampCrée](../WDLang1/1000020876.md) avec la constante *typNoteRepositionnable*.

## Caractéristiques du champ Note repositionnable
<a name="caracteristiques_champ_note_repositionnable_ELTTEXTE000187"></a>
La fenêtre de description du champ Note repositionnable permet de : 

- Saisir directement le contenu de la note (onglet "Général"), 

- Réduire automatiquement la note (onglet "Détail"). Si cette option est cochée, la note sera automatiquement réduite 5 secondes après l'ouverture de la fenêtre. Cette option est équivalente à la propriété [RéduitAutomatique](../Proprietes/1000021963.md). <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Champ_Note_Go%20-%20HC%20N%B0002.gif)





**Astuce** : Pour ne pas autoriser la saisie dans un champ Note repositionnable, il suffit de cocher l'option "Inactif" dans l'onglet "UI". L'utilisateur pourra entrer dans le champ mais ne pourra pas saisir.

## Fonctionnement par défaut du champ Note repositionnable
<a name="fonctionnement_par_defaut_champ_note_repositionnable_ELTTEXTE000211"></a>
Par défaut, si une note repositionnable existe sur une fenêtre, lorsque la fenêtre s'ouvre, la note repositionnable s'affiche et se réduit "en signet" sur le côté de la fenêtre au bout de 5 secondes. 

Si l'utilisateur restaure la note, la note s'ouvre de manière permanente. 

La note repositionnable peut être déplacée sur la fenêtre et tout autour de la fenêtre. 

En cliquant sur la note, le texte passe en édition. 

En cliquant sur la croix, la note repositionnable disparaît. La combinaison de touches Ctrl + Z permet de réafficher la note supprimée.

L'utilisateur a un menu contextuel complet qui lui permet de manipuler le champ Note repositionnable. Pour plus de détails, consultez l'aide sur la [FAA du champ Note](../Editeurs/1000021820.md).

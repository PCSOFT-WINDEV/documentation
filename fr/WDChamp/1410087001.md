## Présentation du champ Fil
<a name="presentation_champ_fil_ELTTEXTE000162"></a>


### Champ d'une fenêtre
<a name="champ_une_fenetre_ELTPARAGRAPHE000011"></a>Le champ Fil permet de relier visuellement 2 champs. Ce champ permet de créer des diagrammes (workflow) très simplement. <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=WD-Champ%20Fil.jpg)
Un fil peut être : 

- composé d'un ou de plusieurs segments. 

- rectiligne, à angle droit ou libre. 




Lors du déplacement d'un des champs attaché au fil, le fil suit automatiquement la position des champs reliés.

## Créer un champ Fil
<a name="creer_champ_fil_ELTTEXTE000186"></a>
**Pour créer un champ de type Fil**, plusieurs méthodes sont à votre disposition : 

- **Création à partir de la sélection de deux champs** : 

	1. Sélectionnez les deux champs à relier : utilisez le lasso de la souris ou la touche Ctrl pour sélectionner les deux champs. 

	2. Dans le ruban, sous le volet "Création", dans le groupe "Ajuster", cliquez sur "Fil entre 2 champs". 




- **Création à partir de la sélection d'un champ** : 

	1. Un champ est sélectionné dans la fenêtre. 

	2. Dans le ruban, sous le volet "Création", dans le groupe "Ajuster", cliquez sur "Fil entre 2 champs". 

	3. Un assistant se lance vous demandant d'ajouter à la sélection le champ d'arrivée. Cliquez sur la touche Ctrl puis sur le champ d'arrivée. 

	4. Cliquez sur le bouton "Créer le fil". Le champ Fil est créé. 




- **Création à partir d'aucune sélection** : 

	1. Dans le ruban, sous le volet "Création", dans le groupe "Ajuster", cliquez sur "Fil entre 2 champs". 

	2. Un assistant se lance vous demandant : 

		- de sélectionner le champ de départ. 

		- d'ajouter à la sélection le champ d'arrivée. Cliquez sur la touche Ctrl puis sur le champ d'arrivée.  




3. Cliquez sur le bouton "Créer le fil". Le champ Fil est créé.




Pour afficher les caractéristiques du champ, sélectionnez l'option "Description" dans le menu contextuel du champ.

## Caractéristiques du champ Fil
<a name="caracteristiques_champ_fil_ELTTEXTE000210"></a>
L'onglet "Général" de la fenêtre de description du champ Fil permet de décrire les principales caractéristiques du champ : 

- Caractéristiques du champ de départ du fil : 

	- Nom du champ. 

	- Forme de l'extrémité du champ Fil pour le champ de départ. Cette forme peut être par exemple une flèche, un triangle, un rond, ... 

	- Position du point d'attache du champ Fil sur le champ de départ. Le point d'attache peut être en haut, en bas, à droite ou à gauche du champ. 




- Caractéristiques du champ d'arrivée du fil : 

	- Nom du champ. 

	- Forme de l'extrémité du champ Fil pour le champ d'arrivée. Cette forme peut être par exemple une flèche, un triangle, un rond, ...

	- Position du point d'attache du champ Fil sur le champ d'arrivée. Le point d'attache peut être en haut, en bas, à droite ou à gauche du champ. 




- Dimensions et routage : 

	- Epaisseur du trait du champ Fil. Cette épaisseur est exprimée en pixels. 

	- Taille des formes des extrémités. Cette taille est exprimée en pixels. 

	- Type de routage : Deux types de routage sont disponibles : 

		- A angle droit : Le champ Fil peut être décomposé en plusieurs segments. 

		- En ligne droite : Le champ Fil sera composé d'un seul segment. 







Toutes ces caractéristiques peuvent être modifiées par programmation grâce aux propriétés WLangage. Pour plus de détails, consultez [Manipuler un champ Fil par programmation](../WDChamp/1410087004.md).

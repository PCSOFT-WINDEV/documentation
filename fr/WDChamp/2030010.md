


## La page d'accueil
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000102"></a>
La page d'accueil est la première page de votre site WEBDEV (statique ou en mode session). Cette page est toujours une page statique.

Dans un site en mode session, la page d'accueil est un des éléments permettant le référencement du site.

Il est possible de choisir une page statique existante dans votre site.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Définir la page d'accueil
<a name="definir_page_accueil_ELTTEXTE000126"></a>
WEBDEV permet de définir la page d'accueil d'un site via le volet "Explorateur de projet".

**Remarque** : Une page d'accueil est également proposée lors du test d'un projet statique, si aucune page d'accueil n'a été définie. Pour plus de détails, consultez [Tester un projet](../Editeurs/2019003.md).
<a name="NOTE2b_2"></a>


### Définir la page d'accueil dans le volet "Explorateur de projet"
<a name="definir_page_accueil_dans_volet_explorateur_projet_ELTPARAGRAPHE000032"></a>Pour définir la page d'accueil du projet :

1. Sélectionnez la page qui va être la nouvelle page d'accueil dans le volet "Explorateur de projet". Cette page doit être une page statique de votre site.

2. Dans le menu contextuel, sélectionnez l'option "Page d'accueil". Cette option est disponible uniquement si la page sélectionnée est une page statique.

3. La page HTML correspondante est automatiquement générée et la page d'accueil est définie. L'icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_accueil.gif) permet d'identifier la page d'accueil dans le volet "Explorateur de projet".

<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=page_Accueil_TV.gif)

## Fichiers créés correspondant à la page d'accueil
<a name="fichiers_crees_correspondant_page_accueil_ELTTEXTE000156"></a>
Lorsque une page d'accueil est définie pour votre site statique ou votre site en mode session, les fichiers suivants sont automatiquement créés dans le répertoire &lt;Nom Projet&gt;_WEB ainsi que dans chaque sous-répertoire de langue :

- Accueil.htm

- Default.htm

- Index.htm




Lors de l'installation du site WEBDEV, cette page sera utilisée comme première page du site. L'adresse de lancement du site devra lancer cette page.

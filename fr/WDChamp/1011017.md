
## Caractéristiques d'une impression PCL
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000109"></a>
Le résultat d'une impression PCL dépend beaucoup des possibilités de l'imprimante utilisée (gestion des images et des traits, gestion des polices, ...).

WINDEV et WEBDEV réalisent l'impression en fonction de l'imprimante sélectionnée.

Si l'imprimante sélectionnée est :

- [une des imprimantes gérées par WINDEV ou WEBDEV](../WDChamp/1011022.md), le fichier PCL généré exploite au maximum les possibilités de l'imprimante (couleurs, couleur du texte, images, cadres, polices, ...).

- **l'imprimante de type "Générique" ("Universal PCL")**, le fichier PCL généré utilisera les caractéristiques par défaut des imprimantes PCL. Les cadres, les traits et les images ne seront imprimés que si l'imprimante utilisée le permet.

- **l'imprimante "Universal PCL No Font"** : ce driver permet d'imprimer sur toute imprimante même si les polices d'imprimantes ne sont pas disponibles. Les polices du mobile sont utilisées. L'impression sur ce driver permet de ne pas utiliser les polices d'imprimante. Toutes les tailles de polices sont gérées. Cependant, l'utilisation de ce driver implique une impression plus lente. 
	**Remarque** : il est possible de copier des fichiers de police supplémentaires (fichiers .ttf) dans le répertoire "\\Windows\\Polices" du poste utilisé.

## Caractéristiques générales
<a name="caracteristiques_generales_ELTTEXTE000133"></a>
Les impressions réalisées sur une imprimante PCL ont les caractéristiques suivantes :

- Impression en mode Portrait

- Impression au format A4

- Qualité d'impression : Normale

- Résolution : 300 dpi.

- Vitesse d'impression : la vitesse d'impression dépend du contenu de l'état. Plus le nombre de traits et d'images est important, moins l'impression sera rapide.

- Gestion des polices :
	**Lors de l'impression sur une imprimante générique**, les polices utilisées sont les polices d'imprimantes. L'imprimante sélectionne automatiquement la police correspondant le mieux aux critères spécifiés. La taille maximale de la police dépend des caractéristiques de l'imprimante utilisée. Consultez la documentation de l'imprimante utilisée.
	**Lors de l'impression sur l'imprimante "Universal PCL No Font"**, vous pouvez imprimer sur toute imprimante même si les polices d'imprimantes ne sont pas disponibles. Les polices du mobile sont utilisées. Toutes les tailles de polices sont gérées. Cependant, l'utilisation de ce driver implique une impression plus lente.

- Le format RTF n'est pas pris en compte.

- Les libellés orientés ne sont pas pris en compte.

- Lors de l'impression de traits ou de cadres, les traits imprimés sont toujours des traits pleins. L'impression de cadres arrondis n'est pas supportée.

- Selon le type d'imprimante utilisé, les caractères n'existant pas dans le jeu de caractères ISO 8859-1 Latin 1 (ECMA 94 Latin 1) -(par exemple : le caractère Â€) risquent de ne pas être imprimés correctement, ou de ne pas être imprimés du tout.

## Caractéristiques d'une impression réalisée sur une imprimante PCL "Générique"
<a name="caracteristiques_une_impression_realisee_sur_une_imprimante_pcl_generique_ELTTEXTE000157"></a>
Si vous choisissez d'imprimer sur une imprimante PCL générique, les caractéristiques de l'impression sont les suivantes (ces caractéristiques s'ajoutent aux caractéristiques générales) :

- Impression en Noir et Blanc

- L'impression des traits et des cadres peut ne pas être supportée par l'imprimante sur laquelle l'impression sera effectuée.

- Les images pourront ne pas être imprimées par l'imprimante sur laquelle l'impression sera effectuée.




## Bloc Fin de document
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000197"></a>
Le bloc *Fin de document* est exécuté une seule fois à la fin de l'exécution de l'état. Le bloc *Fin de document* contient généralement des compteurs et/ou des totaux (générés lors de la création du document ou ajoutés par la suite).

Dans l'événement associé à ce type de bloc, il est par exemple possible d'afficher une fenêtre avertissant de la fin de l'impression.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Création du bloc Fin de document
<a name="creation_bloc_fin_document_ELTTEXTE000221"></a>
Pour créer un bloc Fin de document : 

1. Effectuez une des manipulations suivantes sous l'éditeur d'états :

	- sous le volet "Création", dans le groupe "Structure", déroulez "Blocs".

	- sélectionnez l'option "Ajouter un bloc" du menu contextuel de l'état (clic droit sur l'état en cours de création).

	- cliquez sur le bouton "Ajouter un bloc" dans l'onglet "Blocs" de la description de l'état.




2. Sélectionnez le type "Fin de document".




**Remarque** : Pour plus de détails sur les différentes options disponibles sur ce type de bloc, consultez [Description des blocs d'un état](../WDChamp/1011049.md).

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Impression du bloc Fin de document
<a name="impression_bloc_fin_document_ELTTEXTE000245"></a>


### Imprimer le bloc Fin de document avant ou après le bloc Pied de page
<a name="imprimer_bloc_fin_document_avant_apres_bloc_pied_page_ELTPARAGRAPHE000059"></a>Pour imprimer le bloc Fin de document après le bloc Pied de page sur la dernière page de l'état :

- Méthode 1 : 

	1. Affichez l'onglet "Général" de la fenêtre de description du bloc "Fin de document" (option "Description du bloc" du menu contextuel).

	2. Cochez l'option "Imprimer le bloc "Fin de document" APRES le bloc "Pied de page" sur la dernière page".




- Méthode 2 : 

	1. Affichez l'onglet "Blocs" de la fenêtre de description de l'état (option "Description de l'état" du menu contextuel).

	2. Dans la section "Fin de document", cochez l'option "APRES le bloc "Pied de page" ".






<a name="NOTE3_2"></a>


### Bloc Fin de document collé au bloc Bas de page
<a name="bloc_fin_document_colle_bloc_bas_page_ELTPARAGRAPHE000083"></a>Par défaut, le bloc **Fin de document** est collé au bloc *Corps* ou au bloc *Complément de corps* s'il existe. Il est possible que le bloc *Fin de document* soit obligatoirement collé au bloc *Pied de page*.

Pour coller le bloc Fin de document au bloc Pied de page :

- Méthode 1 : 

	1. Affichez l'onglet "Général" de la fenêtre de description du bloc "Pied de page" (option "Description du bloc" du menu contextuel).

	2. Cochez l'option "Le bloc "Fin de document" est collé au bloc "Pied de page"". 
			Le bloc *Fin de document* sera imprimé juste avant ou après le bloc *Pied de page*.




- Méthode 2 : 

	1. Affichez l'onglet "Blocs" de la fenêtre de description de l'état (option "Description de l'état" du menu contextuel).

	2. Dans la section "Fin de document", cochez l'option "Collé au bloc "Pied de page"". 
			Le bloc *Fin de document* sera imprimé juste avant ou après le bloc *Pied de page*.






<a name="NOTE3_3"></a>


### Bloc Fin du document à la place du bloc Bas de page sur la dernière page de l'état
<a name="bloc_fin_document_place_bloc_bas_page_sur_derniere_page_etat_ELTPARAGRAPHE000113"></a>Par défaut, le bloc *Pied de page* est imprimé sur chaque page (y compris la dernière page de l'état). L'éditeur d'états permet de remplacer le bloc *Pied de page* par le bloc **Fin de document** sur la dernière page de l'état.
Pour remplacer le bloc Pied de page par le bloc Fin de document sur la dernière page de l'état :

- Méthode 1 : 

	1. Affichez l'onglet "Général" de la fenêtre de description du bloc "Pied de page" (option "Description du bloc" du menu contextuel).

	2. Cochez l'option "Imprimer le bloc "Fin de document" A LA PLACE du bloc "Pied de page" sur la dernière page". 




- Méthode 2 : 

	1. Affichez l'onglet "Blocs" de la fenêtre de description de l'état (option "Description de l'état" du menu contextuel).

	2. Dans la section "Fin de document", cochez l'option "A LA PLACE du bloc "Pied de page"".







Dans ce cas :

- le bloc *Fin de document* remplacera le bloc *Pied de page* sur la dernière page de l'état.

- l'option "Le bloc "Fin de document" est collé au bloc "Pied de page"" sera ignorée sur la dernière page de l'état.

- l'option "Le bloc "Pied de page" ne s'imprime pas en bas de la page, il se colle aux blocs précédents" sera prise en compte pour le bloc *Fin de document* sur la dernière page de l'état.


**Remarque** : Dans ce cas, pour éviter des effets de superposition, le bloc de *Fin de document* doit avoir la même hauteur que le bloc *Pied de page*.

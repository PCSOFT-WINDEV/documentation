
## Définir le premier champ en saisie (FAA)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000087"></a>
Le menu contextuel d'un champ de saisie propose une option "Définir comme 1er champ en saisie". Cette option permet de modifier le champ en saisie à l'ouverture de la fenêtre. 
<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=FAA_PremierChampSaisie%20-%20HC%20N%B0001.gif)


<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000111"></a>


### Mise en place
<a name="mise_place_ELTPARAGRAPHE000019"></a>L'option "Définir comme 1er champ en saisie" est automatiquement mise en place pour les champs de saisie. 

L'affichage de cette option est configurable (comme toutes les options proposées par les FAA). Pour plus de détails, consultez [Configurer les menus FAA](../Editeurs/2010040.md). 

En programmation : 

- la fonction [FAADésactive](../WDLang1/1000022018.md) utilisée avec la constante *faa1erChampEnSaisie* permet de supprimer l'option dans le menu contextuel. 

- la fonction [FAAChangeLibellé](../WDLang1/1000022100.md) permet de modifier le libellé de l'option du menu contextuel.

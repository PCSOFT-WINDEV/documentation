
## Propriétés associées aux zones définies en mode Zoning
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


Les différentes propriétés utilisables avec les zones (mode zoning) sont les suivantes :

![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png)![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) **En code serveur** :



|   |   |
| --- | --- |
| [Alias](../Proprietes/2510004.md) | La propriété **Alias** permet de : <br><br>- connaître ou de modifier l'alias d'une fenêtre (WINDEV et WINDEV Mobile).<br><br>- connaître ou de modifier l'alias d'un volet d'onglet dynamique (WINDEV).<br><br>- connaître le nom interne HTML / Javascript d'un champ (WEBDEV).<br><br><br> |
| [AttributHTML](../Proprietes/1410089223.md) | La propriété **AttributHTML** permet de lire et de modifier les attributs HTML du champ. |
| [ClasseHTML](../Proprietes/1000021530.md) | La propriété **ClasseHTML** permet de lire et de modifier la valeur de l'attribut HTML "class" du champ. |
| [CouleurFond](../Proprietes/2510022.md) | La propriété **CouleurFond** permet de gérer la couleur de fond d'un champ, d'une fenêtre, d'une page ou d'un bloc d'un état. |
| [CurseurSouris](../Proprietes/2510026.md) | La propriété **CurseurSouris** permet de connaître et de modifier le curseur de survol défini pour un champ, une fenêtre ou une page. |
| [Extra](../Proprietes/1410089327.md) | La propriété **Extra** permet de :<br><br>- Connaître les informations supplémentaires associées à un champ, à une fenêtre, ou à une page.<br><br>- Modifier les informations supplémentaires associées à un champ, à une fenêtre ou à une page.<br><br><br> |
| [FichierParcouru](../Proprietes/2510040.md) | La propriété **FichierParcouru** permet de connaître et de modifier la source de données utilisée pour l'affichage des enregistrements dans les champs basés sur un fichier de données. |
| [Groupe](../Proprietes/2510049.md) | La propriété **Groupe** permet de : <br><br>- savoir si le champ appartient ou non à un groupe de champs. Si le champ appartient à un ou plusieurs groupes, la propriété **Groupe** permet d'identifier ces groupes de champs.<br><br>- associer un champ à un groupe de champs. <br><br><br> |
| [Hauteur](../Proprietes/2510050.md) | La propriété **Hauteur** permet de connaître et modifier la hauteur :<br><br>- d'un champ ou d'un bloc dans un état.<br><br>- d'un champ (présent dans une fenêtre), d'une fenêtre, d'une ligne d'un champ Table ou Liste (y compris Liste image), d'une option d'un champ Sélecteur ou Interrupteur, d'un haut ou d'un bas de rupture d'un champ Table ou d'un champ Zone répétée.<br><br>- d'un champ (présent dans une page), d'un haut ou bas de rupture d'un champ Table ou Zone répétée. Le champ peut être superposable ou non.<br><br>- d'une cellule de rupture d'un champ Table ou d'un champ Zone répétée.<br><br><br> |
| [HTMLAprès](../Proprietes/1000017002.md) | La propriété **HTMLAprès** permet de connaître et de modifier le code HTML ajouté après le champ. |
| [HTMLAvant](../Proprietes/1000017051.md) | La propriété **HTMLAvant** permet de connaître et de modifier le code HTML ajouté avant le champ. |
| [ImageFond](../Proprietes/2510056.md) | La propriété **ImageFond** permet de connaître et de modifier l'image de fond d'un élément : fenêtre, page, champ ou état. |
| [Largeur](../Proprietes/2510059.md) | La propriété **Largeur** permet de connaître et modifier la largeur :<br><br>- d'un champ ou d'un bloc dans un état.<br><br>- d'un champ (présent dans une fenêtre), d'une fenêtre, d'une colonne de table ou d'une colonne de liste (y compris liste image).<br><br>- d'un champ (présent dans une page), d'une colonne de table ou d'une liste. Le champ peut être superposable ou non.<br><br><br> |
| [LibelléHTML](../Proprietes/1000020922.md) | La propriété **LibelléHTML** permet de manipuler le libellé d'un champ sous forme de code HTML. |
| [MenuContextuel](../Proprietes/2510077.md) | La propriété **MenuContextuel** permet de :<br><br>- Connaître le menu contextuel personnalisé associé à un champ, à une fenêtre ou une page.<br><br>- Modifier le menu contextuel personnalisé associé à un champ, une fenêtre ou une page (associer un autre menu contextuel au champ, à la fenêtre ou à la page).<br><br>- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Personnaliser les menus système des champs en ajoutant des options.<br><br><br> |
| [Nom](../Proprietes/2510082.md) | La propriété **Nom** permet de connaître :<br><br>- le nom de la police associée à une variable de type Police.<br><br>- le nom d'un champ, d'un état ou d'un bloc d'un état.<br><br>- le nom d'un champ, d'un groupe de champs ou d'une fenêtre.<br><br>- le nom d'un champ, d'un groupe de champs ou d'une page.<br><br><br> |
| [NomComplet](../Proprietes/2514005.md) | La propriété **NomComplet** permet de connaître le nom complet :<br><br>- d'un champ.<br><br>- d'une fenêtre.<br><br>- d'une option de menu.<br><br>- d'une colonne de table.<br><br><br><br><br> |
| [Note](../Proprietes/2510085.md) | La propriété **Note** permet de :<br><br>- Connaître les notes associées à un champ, à une fenêtre, à une page ou un état.<br><br>- Modifier les notes associées à un champ, à une fenêtre, à une page ou un état.<br><br><br> |
| [Opacité](../Proprietes/2514006.md) | La propriété **Opacité** permet de connaître et définir le pourcentage d'opacité d'une fenêtre ou des champs présents dans une fenêtre ou une page. |
| [Plan](../Proprietes/2510094.md) | La propriété **Plan** permet de :<br><br>- Connaître : <br><br>	- le plan actif d'une fenêtre (ou d'un état) ou le plan auquel appartient un champ.<br><br>	- le plan actif d'un élément WEBDEV.<br><br><br><br><br>- Modifier : <br><br>	- le plan actif d'une fenêtre (ou d'un état) ou associer un champ à un autre plan.<br><br>	- le plan actif d'un élément WEBDEV.<br><br><br><br><br><br> |
| [RubriqueParcourue](../Proprietes/2510111.md) | La propriété **RubriqueParcourue** permet de :<br><br>- Connaître la rubrique utilisée pour le parcours automatique des champs Table, Zone répétée, Liste (y compris Liste image), Combo, état sur fichier ou des superchamps.<br><br>- Modifier la rubrique utilisée pour le parcours automatique des champs Table, Zone répétée, Liste (y compris Liste image), Combo, état sur fichier ou des superchamps. Le parcours est alors réinitialisé.<br><br><br> |
| [TitreNote](../Proprietes/2510121.md) | La propriété **TitreNote** permet de :<br><br>- Connaître le titre des notes utilisé pour la documentation technique d'un champ, d'une fenêtre, d'une page ou d'un état. <br><br>- Modifier le titre des notes utilisé pour la documentation technique d'un champ, d'une fenêtre, d'une page ou d'un état.<br><br><br> |
| [Traitement](../Proprietes/1000020878.md) | La propriété **Traitement** permet de modifier les traitements à exécuter par les événements disponibles sur un champ, une fenêtre, une page, un état, ... |
| [Type](../Proprietes/2510131.md) | La propriété **Type** permet de connaître :<br><br>- le type d'un élément décrit par programmation (description de rubrique ou de fichier) ou non (fichier ou rubrique défini sous l'éditeur d'analyses, vue ou requête).<br><br>- le type d'un objet.<br><br>- le type de la valeur stockée dans un variant.<br><br><br> |
| [Valeur](../Proprietes/2510130.md) | La propriété **Valeur** permet de :<br><br>- Connaître et modifier la valeur d'un champ d'un état.<br><br>- Connaître et modifier la valeur d'un champ ou le titre d'une fenêtre.<br><br>- Connaître l'identifiant d'un flux. <br><br>- Connaître et modifier la valeur d'un champ ou le titre d'une page.<br><br><br> |
| [ValeurInitiale](../Proprietes/2510128.md) | La propriété **ValeurInitiale** permet de connaître la valeur initiale d'un champ d'une fenêtre ou d'une page. |
| [Visible](../Proprietes/2510138.md) | La propriété **Visible** permet de :<br><br>- Savoir si un élément est visible.<br><br>- Rendre un élément visible ou invisible.<br><br><br> |
| [VisibleInitial](../Proprietes/2510070.md) | La propriété **VisibleInitial** permet de savoir si :<br><br>- un champ était visible lors du premier affichage de la fenêtre ou la page associée.<br><br>- une fenêtre était visible lors de son premier affichage.<br><br><br> |
| [X](../Proprietes/2510135.md) | La propriété **X** permet de :<br><br>- Connaître l'abscisse d'un champ, d'une colonne d'un champ Table ou d'une fenêtre (position sur l'axe des X).<br><br>- Modifier l'abscisse d'un champ ou d'une fenêtre (position sur l'axe des X).<br><br><br> |
| [XInitial](../Proprietes/2510140.md) | La propriété **XInitial** permet de :<br><br>- Connaître l'abscisse : <br><br>	- d'un champ (position sur l'axe des X) lors de l'ouverture de la fenêtre ou de la page.<br><br>	- d'une fenêtre (position sur l'axe des X) lors de l'ouverture de la fenêtre. <br><br><br><br><br>- Modifier l'abscisse d'un champ ancré "à droite" (position sur l'axe des X) lors de l'ouverture de la fenêtre.<br><br>- Connaître l'abscisse d'un champ par rapport à son bloc (position en millimètres sur l'axe des X) lors de l'ouverture de l'état.<br><br><br> |
| [Y](../Proprietes/2510137.md) | La propriété **Y** permet de :<br><br>- Connaître l'ordonnée d'un champ ou d'une fenêtre (position sur l'axe des Y).<br><br>- Modifier l'ordonnée d'un champ ou d'une fenêtre (position sur l'axe des Y).<br><br><br> |
| [YInitial](../Proprietes/2510087.md) | La propriété **YInitial** permet de :<br><br>- Connaître l'ordonnée : <br><br>	- d'un champ (position sur l'axe des Y) lors de l'ouverture de la fenêtre ou de la page.<br><br>	- d'une fenêtre (position sur l'axe des Y) lors de l'ouverture de la fenêtre.<br><br><br><br><br>- Modifier l'ordonnée d'un champ ancré "en bas" (position sur l'axe des Y) lors de l'ouverture de la fenêtre.<br><br>- Connaître l'ordonnée d'un champ par rapport à son bloc (position en millimètres sur l'axe des Y) lors de l'ouverture de l'état.<br><br><br> |

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000155"></a>
Par défaut, les événements disponibles sous l'éditeur de code correspondent aux événements les plus classiques.

Cependant, les fenêtres, les pages et leurs champs peuvent gérer des événements optionnels (ou traitements optionnels).

Il est ainsi possible de gérer par exemple selon les plateformes :

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Pour les fenêtres et leurs champs : 

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Entrée en survol et sortie du survol d'un champ par la souris

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) le survol d'un champ par la souris,

	- le bouton gauche enfoncé, relâché, avec un double-clic (sur certains champs),

	- le bouton droit enfoncé, relâché, avec double-clic  (sur certains champs),

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) la roulette de la souris, 

	- ...


Dans ces événements optionnels, il est possible de connaître les différentes actions réalisées par l'utilisateur (touche enfoncée, utilisation de la roulette de la souris, ...).

## Ajouter un événement optionnel
<a name="ajouter_evenement_optionnel_ELTTEXTE000179"></a>
Pour ajouter un événement optionnel dans une fenêtre, une page ou dans un champ d'une fenêtre ou d'une page :

1. Sélectionnez la fenêtre, la page ou le champ voulu.

2. Affichez la fenêtre de code de cette fenêtre, de cette page ou de ce champ (touche F2).

3. Cliquez sur le lien "Ajouter d'autres événements à xxx" situé à la fin de la fenêtre de code, après le dernier événement.<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Traitements_optionnels_WD_OK%20-%20HC%20N%B0001.gif)


4. La totalité des événements optionnels disponibles pour l'élément est affichée. Par exemple, pour une fenêtre : <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Traitements_optionnels_WD_OK%20-%20HC%20N%B0002.gif&type=thumb)


5. Cochez l'événement optionnel à ajouter et validez. 
	Remarque : Il est possible de sélectionner plusieurs événements optionnels. 

6. L'événement optionnel sélectionné est automatiquement ajouté dans les événements gérés par la fenêtre, la page ou par le champ.




**Attention** : Un événement optionnel est **toujours** exécuté (contrairement aux événements standard), y compris lors d'un appel à la fonction [Temporisation](../WDLang1/3015012.md) ou à la fonction [Multitâche](../WDLang1/3015004.md).




## Manipuler un champ Carrousel par programmation
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000133"></a>
WINDEV permet de manipuler un champ Carrousel directement par programmation. Il suffit d'utiliser directement le nom du champ Carrousel dans le code.

Le champ Carrousel utilise la même famille de fonctions que le champ Liste Image (fonction [ListeXXX](../WDLang1/3049001.md)).

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Remplir un champ Carrousel
<a name="remplir_champ_carrousel_ELTTEXTE000157"></a>


### Initialiser un élément
<a name="initialiser_element_ELTPARAGRAPHE000023"></a>Pour initialiser un élément d'un champ Carrousel, utilisez la fonction [ListeAjoute](../WDLang1/3049004.md). 

```wl
// Ajout d'éléments
ListeAjoute(CAR_Carrousel, "Client", "Client.gif")
ListeAjoute(CAR_Carrousel, "Commande", "Commande.gif")
```
**Remarque** : Le libellé ("Client", "Commande" dans cet exemple) est affiché en survol de l'élément dans le coin haut gauche du champ Carrousel.
<a name="NOTE2_2"></a>


### Remplir un champ Carrousel
<a name="remplir_champ_carrousel_ELTPARAGRAPHE000035"></a>Pour initialiser un champ Carrousel, il suffit d'utiliser une boucle d'itération pour initialiser tous les éléments du champ. Par exemple : 

```wl
i est un entier
POUR i = 1 a 52
	ListeAjoute(CAR_Carrousel, Nom[i], ImageCarrousel[i])
FIN
```


<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Vider un champ Carrousel
<a name="vider_champ_carrousel_ELTTEXTE000187"></a>


### Supprimer les éléments d'un champ Carrousel
<a name="supprimer_les_elements_champ_carrousel_ELTPARAGRAPHE000045"></a>Pour supprimer tous les éléments, utilisez la fonction [ListeSupprimeTout](../WDLang1/3049013.md).


```wl
// Suppression de tous les éléments du champ Carrousel
ListeSupprimeTout(CAR_Carrousel)
```

## Propriétés spécifiques aux champs Carrousel
<a name="proprietes_specifiques_aux_champs_carrousel_ELTTEXTE000211"></a>
Les propriétés suivantes sont spécifiques à la gestion des caractéristiques d'un champ Carrousel par programmation.


|   |   |
| --- | --- |
| [AngleObservateur](../Proprietes/2515046.md) | Permet de connaître ou modifier l'angle secondaire d'un champ Carrousel. Cet angle secondaire permet de faire "pencher" le champ Carrousel. |
| [Animation](../Proprietes/2510006.md) | Permet d'arrêter ou de reprendre l'animation du champ Carrousel. |
| [AnimationInitiale](../Proprietes/2510012.md) | Permet de configurer l'animation du champ Carrousel à son initialisation. |
| [PériodeAnimation](../Proprietes/2510152.md) | Permet de connaître et de modifier la vitesse de rotation d'un champ Carrousel. |


Pour connaître la liste complète des propriétés du WLangage pouvant être utilisées avec un champ Carrousel, consultez [Propriétés associées à un champ Carrousel](../WDChamp/1016003.md).

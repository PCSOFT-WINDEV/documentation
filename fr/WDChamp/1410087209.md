## Présentation du champ Editeur d'images pour mobile
<a name="presentation_champ_editeur_images_pour_mobile_ELTTEXTE000118"></a>
Le champ Editeur d'Images permet de proposer aux utilisateurs de vos applications mobiles de retoucher et modifier des images ou des photos.  

Le champ Editeur d’Images sur mobile propose les fonctionnalités attendues sur un mobile :

- recadrage,

- rotation,

- redressage d’image (changement de perspective),

- luminosité, contraste, saturation,

- annotations : dessin à main levée, surligneur, texte,

- symétrie,

- floutage de zone,

- OCR proposé sur les zones de texte détectées,

- etc.

## Créer un champ Editeur d'images pour mobile
<a name="creer_champ_editeur_images_pour_mobile_ELTTEXTE000142"></a>
**Pour créer un champ de type Editeur d'images pour mobile** : 

1. Sous le volet "Création", dans le groupe "Champs graphiques", déroulez "Bureautique" puis cliquez sur "Editeur d'images mobile". 

2. Cliquez dans la fenêtre à la position où le champ doit être créé.




Remarque : Les dimensions du champ créé sont optimisées pour occuper l'espace disponible à la position indiquée. Si la taille du champ ne vous convient pas, utilisez la combinaison de touches Ctrl + Z : le champ retrouvera sa taille par défaut.

Pour afficher les caractéristiques du champ, sélectionnez l'option "Description" dans le menu contextuel du champ.

## Caractéristiques du champ Editeur d'images pour mobile
<a name="caracteristiques_champ_editeur_images_pour_mobile_ELTTEXTE000170"></a>
Dans l'onglet "Général" de la fenêtre de description du champ Editeur d'images mobile, il est possible de paramétrer les options affichées dans la barre d'outils du champ : <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Editeur_img_mobile%20-%20HC%20N%B0001.gif)


- **Afficher les options pour le changement d'image (Ouvrir, prendre une photo, ... )** : Permet à l'utilisateur de sélectionner l'image à afficher dans le champ. Cette image peut être un fichier présent sur le mobile ou une photo prise directement. La barre d'outils correspondante est affichée en haut à droite du champ Editeur d'images mobile. <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Editeur_img_mobile%20-%20HC%20N%B0002.gif)


- **Afficher les options pour la modification (Recadrer, rotation, contraste, ... )** : Permet à l'utilisateur de modifier l'image affichée dans le champ Editeur d'images mobile. Cette option correspond aux boutons "Transformation" et "Réglages" de la barre d'outils du champ.  <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Editeur_img_mobile%20-%20HC%20N%B0003.gif)<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Editeur_img_mobile%20-%20HC%20N%B0004.gif)


- **Afficher les options pour l'annotation (Crayon, texte, surlignage, ... )** : Permet à l'utilisateur d'annoter l'image en ajoutant par exemple du texte. Cette option correspond aux boutons "Dessin" et "Texte" de la barre d'outils du champ. 

- **Afficher les options pour l'OCR (Reconnaissance de caractères)** : Permet à l'utilisateur d'utiliser la reconnaissance de caractères sur l'image affichée dans le champ Editeur d'images mobile. Cette option correspond au bouton "OCR" disponible dans les outils du champ. 





- ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=WMPhotos.gif) ***Exemples multiplateforme (WINDEV Mobile)*** : **WM Photos** <br>Cet exemple Android/iOS permet de prendre des photos, et de les sauvegarder en leur associant un titre et des mot-clés.<br>Une recherche est également disponible.<br>Les photos sont stockées dans le répertoire de l'application et sont donc privées (non visibles par d'autres applications).

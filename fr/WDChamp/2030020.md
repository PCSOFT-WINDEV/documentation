


## Les différents types de pages
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000113"></a>
WEBDEV permet de créer des sites dynamiques et / ou des sites statiques.

Ces sites peuvent combiner plusieurs types de pages :

- [pages statiques](#NOTE2_1).

- [pages dynamiques (en mode Session ou AWP)](#NOTE4_1).




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Page statique
<a name="page_statique_ELTTEXTE000137"></a>
**Caractéristiques**
Une page statique permet d'afficher du texte, des images, ... Ce type de page ne permet pas de réelles interactions avec l'internaute. 
Pour permettre de visualiser plusieurs fiches produit, il est nécessaire de créer autant de pages que de fiches produit à consulter. Toutes les données sont enregistrées dans la page.

**Développement**
Les pages statiques sont créées sous l'éditeur WEBDEV.

**Installation**
Sur le serveur, il suffit de copier la page sur le serveur Web.

## Page dynamique (en mode Session ou AWP)
<a name="page_dynamique_mode_session_awp_ELTTEXTE000161"></a>
**Caractéristiques**
Une page dynamique permet d'afficher du texte, des images, des données provenant d'une base de données, ...
Pour permettre de visualiser plusieurs fiches produit, une seule page dynamique suffit : les données de la page seront remplies en fonction du contenu de la base de données et du produit demandé par l'internaute. Aucune donnée de la base de données n'est enregistrée dans la page.

**Développement**
Les pages dynamiques sont créées sous l'éditeur WEBDEV.

**Installation**
Sur le serveur, il est nécessaire d'installer :

- le moteur WEBDEV : ce moteur permet d'afficher la page HTML avec les données présentes dans la base de données. 

- la base de données.

- les pages HTML générées par WEBDEV.

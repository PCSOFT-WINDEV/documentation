## Présentation
<a name="presentation_ELTTEXTE000142"></a>
WEBDEV permet de manipuler à l'aide des propriétés WLangage : 

- le champ Bandeau défilant. Il suffit d'utiliser la syntaxe suivante : 
	
	```txt
	<Champ Bandeau défilant>.<Nom de la propriété>
	```
Par exemple : 
	
	```wl
	BAN_MaBannière.Valeur = 3
	```
Consultez [Propriétés associées au champ Bandeau défilant](#NOTE2_1). 

- les plans associés au champ Bandeau défilant. Il suffit d'utiliser la syntaxe suivante : 
	
	```txt
	<Champ Bandeau défilant>[Numéro du plan].<Nom de la propriété>
	```
Par exemple : 
	
	```wl
	BAN_MaBannière[3].Visible = Vrai
	```
Consultez [Propriétés associées aux plans d'un champ Bandeau défilant](#NOTE3_1).

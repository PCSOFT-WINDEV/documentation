
## Option de menu : opération effectuée et action
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000164"></a>
Une option de menu déroulant ou de menu contextuel d'une page peut être associée à une action. Plusieurs éléments caractérisent cette action : 

- [Type de l'action](#NOTE2_1), 

- [Destination de l'action](#NOTE3_1), 

- [Gestion du contenu des champs](#NOTE4_1).  



<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=choisir_l_action_associee_option_menu%20-%20HC%20N%B0001.gif)

## Type de l'action : les actions disponibles
<a name="type_action_les_actions_disponibles_ELTTEXTE000188"></a>


### Actions prédéfinies
<a name="actions_predefinies_ELTPARAGRAPHE000031"></a>Différentes actions prédéfinies sont proposées par défaut dans une liste déroulante. Ces actions prédéfinies dépendent des éléments présents dans la page et présents dans le projet.

Il est ainsi possible par exemple de choisir de :

- Exécuter les codes serveur et navigateur. 

- Exécuter le code de clic navigateur uniquement

- Afficher une page du site. Dans ce cas, il est nécessaire de sélectionner la page voulue et si nécessaire d'indiquer les paramètres attendus par cette page;  

- Afficher une page d'un autre site. Dans ce cas, il est nécessaire d'indiquer : 

	- Les options de sécurité, 

	- L'URL de la page à afficher. 




- Afficher un lien personnalisé. Dans ce cas, il est nécessaire de saisir les différents éléments de l'adresse (email, FTP, téléphone, ...). 




Attention : Les actions sont effectuées après le code de clic de l'option de menu (code navigateur et code serveur).


<a name="NOTE2_2"></a>


### Actions spécifiques
<a name="actions_specifiques_ELTPARAGRAPHE000052"></a>La liste déroulante "Type" propose également le lien "Plus de choix". Cette option permet d'ajouter différentes options dans les types d'actions proposées. Il est ainsi possible de :

- faire une action sur un autre champ,

- se positionner sur une ancre,

- partager sur un réseau social,

- changer la langue courante, 

- envoyer la valeur des champs vers une page ou un script. 


Remarque : Toutes ces opérations sont également réalisables par programmation.

## La destination de l'action
<a name="destination_action_ELTTEXTE000218"></a>
Si aucune destination n'est choisie pour l'action, l'action est réalisée dans la destination par défaut définie pour la page en cours.

Les destinations possibles d'une action effectuée par une option de menu sont :


|   |   |
| --- | --- |
| **Nouveau navigateur (_blank) :**<br>Affiche le résultat dans une nouvelle fenêtre du navigateur. | <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Dest_nouveau.gif)<br> |
| **Frame en cours (_self) :**<br>Affiche le résultat dans le même cadre. | <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Dest_FrameEnCours.gif)<br> |
| **Frame parent (_parent) :**<br>Affiche le résultat dans la frame d'origine (cas de framesets imbriqués). | <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Dest_FrameParent.gif)<br> |
| **Navigateur en cours (_top) :**<br>Affiche le résultat dans la page en cours du navigateur (sans frames). | <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Dest_NavEnCours.gif)<br> |
| **Autres frames du frameset (si la page fait partie d'un frameset) :**<br>Affiche le résultat dans une frame spécifique. |   |

## La gestion du contenu des champs
<a name="gestion_contenu_des_champs_ELTTEXTE000242"></a>
Selon l'action sélectionnée, il est nécessaire de préciser si le contenu des champs doit être ou non envoyé au serveur. 

Par défaut, aucune action n'est associée à l'option de menu, mais il est possible de  : 

- Envoyer les valeurs au serveur (submit). Cette opération est nécessaire si l'option de menu permet de basculer en mode SSL grâce à la fonction [SSLActive](../WDLang2/3012014.md). 

- Effacer le contenu des champs côté navigateur (reset). Cette option, conservée par compatibilité permet de ré-initialiser les champs de la page.




## Image "cliquable" dans une fenêtre
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000123"></a>
Sous WINDEV et WINDEV Mobile, il est possible d'utiliser des images cliquables. Un image cliquable est une image (affichée dans un champ Image) sur laquelle l'utilisateur peut cliquer. Un traitement spécifique (code de clic sur l'image) est alors exécuté.

Contrairement à un bouton, aucun effet d'enfoncement n'est observé lors du clic sur une image cliquable.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Utiliser une image cliquable
<a name="utiliser_une_image_cliquable_ELTTEXTE000147"></a>
Pour utiliser une image cliquable dans une application :

1. Sous le volet "Création", dans le groupe "Champs usuels", cliquez sur "Image".

2. Affichez la description du champ (option "Description" du menu contextuel de l'image).

3. Dans l'onglet "Détail", utilisez les paramètres propres au cliquage : 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=FAA_MenuContextuelImage%20-%20HC%20N%B0001%201.gif)


	- L'option "Cette image est une zone de clic" permet de rendre l'image cliquable. L'événement "Clic sur l'image" est ajouté automatiquement aux événements WLangage associés au champ.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png) L'option "Prioritaire pour les clics" permet d'intercepter le clic de souris sur l'image, quelle que soit l'altitude du champ Image dans la fenêtre. Si cette option n'est pas cochée, le champ Image doit être au premier plan pour intercepter le clic de la souris.

- Indiquez si l'image doit être manipulable par l'utilisateur. Pour plus de détails, consultez [Menu contextuel des champs Image](../WDChamp/9500105.md).

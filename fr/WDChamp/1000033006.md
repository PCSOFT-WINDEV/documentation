Le champ Tableur propose plusieurs Fonctionnalités Automatiques de vos Applications (FAA) à l'utilisateur qui permettent de simplifier l'utilisation du champ.

## Menu contextuel du champ Tableur
<a name="menu_contextuel_champ_tableur_ELTTEXTE000302"></a>


### Présentation
<a name="presentation_ELTPARAGRAPHE000036"></a>Chaque champ Tableur présent dans une fenêtre propose par défaut un menu à l'utilisateur.

Ce menu, disponible sans programmation, permet des actions utiles :

- Annuler / Rétablir la dernière action. 

- Couper / Copier / Coller : Ces options sont applicables à une cellule, une colonne, une ligne ou une sélection de cellules. 

- Annuler le zoom : Cette option permet d'annuler le zoom effectué par exemple avec Ctrl + Roulette de la souris. 

- Insérer : Cette option permet d'insérer une ligne ou une colonne. 

- Supprimer : Cette option permet de supprimer le contenu d'une cellule, ligne ou colonne. Attention : tout le contenu est immédiatement supprimé (valeur, formule, ...). 

- Enregistrer sous : Cette option permet d'enregistrer le contenu du champ Tableur dans un nouveau fichier xlsx. 




Remarque : Il est possible d'ajouter des choix à ce menu par programmation. Il suffit d'utiliser la propriété [MenuContextuel](../Proprietes/2510077.md). 


### Paramétrage
<a name="parametrage_ELTPARAGRAPHE000055"></a>Le menu contextuel du champ Tableur peut bien évidemment être désactivé. Pour paramétrer l'affichage du menu contextuel du champ Tableur, vous devez :

1. Afficher la fenêtre de description du champ Tableur.

2. Sélectionner l'onglet "UI".

3. Choisir le menu désiré dans la combo "Menu contextuel". Il est possible de : 

	- "Afficher le menu des FAA (Système)" : Dans ce cas, le menu contextuel affiché sera le menu automatique du champ Tableur. Pour plus de détails sur la configuration de ce menu, consultez [Configurer le menu des FAA](../Editeurs/2010040.md).

	- "Ajouter un menu contextuel" : Si cette option est cochée, vous pouvez sélectionner le menu contextuel personnalisé à afficher. ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=FAA_Menu_Agenda.gif)








Remarques : 

- Si les deux options sont cochées, le menu contextuel personnalisé peut être ajouté avant ou après le menu des FAA. 

- Si aucune option n'est sélectionnée, aucun menu contextuel ne sera sélectionné. L'option &lt;Désactivé&gt; sera affichée dans la fenêtre de description. 

- La propriété [MenuContextuel](../Proprietes/2510077.md) permet également de supprimer le menu contextuel spécifique aux FAA et de retrouver le menu Windows classique pour un champ spécifique.

- Certaines options spécifiques peuvent être désactivées avec la fonction [FAADésactive](../WDLang1/1000022018.md).

## Barres d'outils et de formule
<a name="barres_outils_formule_ELTTEXTE000332"></a>
Par défaut, le champ Tableur comporte une barre d'outils et une barre de formule. 


### Barre d'outils
<a name="barre_outils_ELTPARAGRAPHE000104"></a>La barre d'outils permet à l'utilisateur final de réaliser les opérations de mise en forme les plus courantes : 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=FAA_Tableur_Barre_outils.gif)


- Annuler / Rétablir

- Nom de la police

- Taille de la police

- Zoom de la sélection

- Formatage de la cellule : Gras, Italique, Souligné, Barré

- Bordure et couleur de la bordure

- Couleur de fond

- Couleur du texte

- Alignement du texte : Aligné à gauche, Centré, Aligné à droite

- Fusion des cellules

- Format de la cellule




Il est possible : 

- d'afficher ou non cette barre d'outils. 

	- Sous l'éditeur : option "Avec barre d'outils" dans l'onglet "Détail" de la description du champ. 

	- par programmation : propriété [BarreOutilsVisible](../Proprietes/1000021333.md). 




- de personnaliser la barre d'outils. Pour plus de détails, consultez [Personnaliser la barre d'outils (champ Tableur)](../WDChamp/1000033001.md).





### Barre de formule
<a name="barre_formule_ELTPARAGRAPHE000136"></a>La barre de formule permet à l'utilisateur final de connaître la ou les cellules sélectionnées et de saisir la formule ou la valeur associée à une cellule. 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=FAA_Tableur_Barre_Formule.gif)
Deux boutons permettent de valider ou d'annuler la saisie. 

Il est possible : 

- d'afficher ou non cette barre de formule. 

	- Sous l'éditeur : option "Avec barre de formule" dans l'onglet "Détail" de la description du champ. 

	- par programmation : propriété [BarreDeFormuleVisible](../Proprietes/1000021334.md). 




- de personnaliser la barre de formule. Pour plus de détails, consultez [Personnaliser la barre de formules (champ Tableur)](../WDChamp/1000033001.md).




Remarque : Une formule commence par le signe '='. Lors de la saisie du signe '=', toutes les fonctions disponibles par défaut sont affichées dans la formule. Pour utiliser une fonction, il suffit de la sélectionner.

## Saisie des valeurs et des formules
<a name="saisie_des_valeurs_des_formules_ELTTEXTE000368"></a>
La saisie des valeurs et des formules se fait : 

- soit directement dans la cellule. 

- soit dans la barre de formule de la cellule en cours si celle-ci est visible. 




Remarque : Si le champ Tableur autorise la saisie de valeurs, l'utilisateur final pourra ajouter, modifier, supprimer les valeurs dans les cellules. L'utilisateur final pourra également ajouter des lignes et des colonnes. Pour autoriser la saisie des valeurs :  

- sous l'éditeur : option "Autoriser la saisie de valeurs" dans l'onglet "Détail" de la description du champ. 

- par programmation : propriété [SaisieValeur](../Proprietes/1000021336.md). 




Une formule commence toujours par le signe '='. Une formule peut être composée : 

- d'une fonction, 

- d'une ou plusieurs cellules provenant de la feuille en cours ou d'une autre feuille de calcul. Pour référencer une cellule dans un calcul, il suffit de cliquer sur la cellule voulue : le nom de la cellule s'affiche dans la barre de formule.

## Manipulation des cellules
<a name="manipulation_des_cellules_ELTTEXTE000392"></a>
Le champ Tableur permet de : 

- Sélectionner une cellule. 

- Sélectionner une plage de cellules. 

- Sélectionner plusieurs cellules. 


Il est possible d'appliquer à ces cellules les différentes options de la barre d'outils. 

Il est également possible : 

- de créer des formules comprenant différentes cellules. 

- d'effectuer une copie simple de cellule. Par exemple : 

	- Sélectionnez une cellule contenant une valeur ou une formule. 

	- Sélectionnez la poignée de recopie (carré en bas de la sélection) et déplacez cette poignée : La valeur initiale est recopiée dans les différentes cellules sélectionnées 




- d'effectuer une recopie incrémentale d'un groupe de cellules. Par exemple : 

	- Sélectionnez deux cellules verticales contenant deux valeurs d'une suite (par exemple "1" et "2"). 

	- Sélectionnez la poignée de recopie (carré en bas de la sélection) et déplacez cette poignée vers le bas : les différentes valeurs des cellules sélectionnées sont mises à jour.

## Gestion des feuilles de calcul
<a name="gestion_des_feuilles_calcul_ELTTEXTE000416"></a>
Le champ Tableur peut être multifeuille et gérer les différentes feuilles de calcul d'un fichier xlsx. 

Dans ce cas : 

- Les différentes feuilles de calcul apparaissent en bas à gauche du champ Tableur. 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=FAA_Tableur_multifeuille.gif)
Le bouton "+" permet d'ajouter de nouvelles feuilles de calcul. 

- L'utilisateur final dispose d'un menu contextuel permettant de gérer les différentes feuilles de calcul : 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=FAA_Tableur_multifeuilleMenu.gif)
Il est possible de : 

	- Insérer une feuille. 

	- Supprimer la feuille en cours. 

	- Renommer la feuille en cours. 
			Remarque : Les options de ce menu sont paramétrables comme le menu contextuel du champ Tableur. 







Astuce : Si plusieurs feuilles de calcul sont affichées, il est possible de faire défiler les différents onglets correspondant aux feuilles à l'aide des boutons fléchés. 

- Le raccourci Ctrl + clic sur la flèche de gauche permet d'afficher l'onglet de la première feuille de calcul. 

- Le raccourci Ctrl + clic sur la flèche de droite permet d'afficher l'onglet de la dernière feuille de calcul.


## Propriétés associées au champ Editeur d'images
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000066"></a>
Un champ Editeur d'images peut être manipulé en WLangage dans une fenêtre avec les propriétés suivantes.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Propriétés disponibles dans une fenêtre
<a name="proprietes_disponibles_dans_une_fenetre_ELTTEXTE000090"></a>
Les différentes propriétés associées aux champs Editeur d'images sont les suivantes :



|   |   |
| --- | --- |
| [AffichageActif](../Proprietes/2510015.md) | La propriété **AffichageActif** permet de :<br><br>- savoir si l'affichage d'un champ ou d'une fenêtre est activé ou non. Si l'affichage est activé, le contenu du champ est rafraîchi régulièrement.<br><br>- activer ou non l'affichage d'un champ ou d'une fenêtre.<br><br><br> |
| [Altitude](../Proprietes/2510002.md) | La propriété **Altitude** permet de gérer l'altitude d'un champ. |
| [Ancrage](../Proprietes/2510001.md) | La propriété **Ancrage** permet de gérer l'ancrage d'un champ dans une fenêtre, une page ou un état. |
| [BarreOutilsVisible](../Proprietes/1000021333.md) | La propriété **BarreOutilsVisible** permet : <br><br>- de savoir si la barre d'outils ou le ruban est affichée dans un champ.  <br><br>- d'afficher ou non la barre d'outils ou le ruban dans un champ.<br><br><br> |
| [CheminFichier](../Proprietes/1000023104.md) | La propriété **CheminFichier** permet de connaître : <br><br>- le nom du fichier xlsx associé à un champ Tableur.<br><br>- le nom du fichier associé à un champ Editeur d'images. <br><br>- le nom du fichier PDF associée à un champ Lecteur PDF. <br><br>- le nom du fichier DOCX associé à un champ Traitement de texte. <br><br>- le nom du fichier wddiag associé à un champ Editeur de diagrammes. <br><br><br> |
| [Epinglé](../Proprietes/1410087943.md) | La propriété **Epinglé** permet de : <br><br>- Epingler ou non un champ : le champ épinglé sera positionné par rapport à la zone visible de son conteneur au lieu de la zone du contenu.<br><br>- Savoir si un champ est épinglé.<br><br><br> |
| [Etat](../Proprietes/2510039.md) | La propriété **Etat** permet de connaître et de modifier l'état d'affichage d'un élément. |
| [EtatInitial](../Proprietes/2510063.md) | La propriété **EtatInitial** permet de connaître l'état initial d'un champ. |
| [Extra](../Proprietes/1410089327.md) | La propriété **Extra** permet de :<br><br>- Connaître les informations supplémentaires associées à un champ, à une fenêtre, ou à une page.<br><br>- Modifier les informations supplémentaires associées à un champ, à une fenêtre ou à une page.<br><br><br> |
| [Grisé](../Proprietes/1000019648.md) | La propriété **Grisé** permet de : <br><br>- Savoir si un champ, un groupe de champs ou une fenêtre est grisé ou non. <br><br>- Griser ou dégriser un champ, un groupe de champs ou une fenêtre.<br><br><br> |
| [Groupe](../Proprietes/2510049.md) | La propriété **Groupe** permet de : <br><br>- savoir si le champ appartient ou non à un groupe de champs. Si le champ appartient à un ou plusieurs groupes, la propriété **Groupe** permet d'identifier ces groupes de champs.<br><br>- associer un champ à un groupe de champs. <br><br><br> |
| [Hauteur](../Proprietes/2510050.md) | La propriété **Hauteur** permet de connaître et modifier la hauteur :<br><br>- d'un champ ou d'un bloc dans un état.<br><br>- d'un champ (présent dans une fenêtre), d'une fenêtre, d'une ligne d'un champ Table ou Liste (y compris Liste image), d'une option d'un champ Sélecteur ou Interrupteur, d'un haut ou d'un bas de rupture d'un champ Table ou d'un champ Zone répétée.<br><br>- d'un champ (présent dans une page), d'un haut ou bas de rupture d'un champ Table ou Zone répétée. Le champ peut être superposable ou non.<br><br>- d'une cellule de rupture d'un champ Table ou d'un champ Zone répétée.<br><br><br> |
| [HauteurImageInitiale](../Proprietes/1410088607.md) | La propriété **HauteurImageInitiale** permet de connaître ou de modifier la hauteur par défaut de l'image créée dans le champ Editeur d'images. |
| [HauteurInitiale](../Proprietes/2510043.md) | La propriété **HauteurInitiale** permet de :<br><br>- Connaître la hauteur de création d'un champ ou d'une fenêtre.<br><br>- Modifier la hauteur de base utilisée pour l'ancrage des champs "en hauteur".<br><br>- Connaître la hauteur de création d'un champ ou d'un bloc d'un état.<br><br><br> |
| [HauteurMax](../Proprietes/2510065.md) | La propriété **HauteurMax** permet de connaître et de modifier la hauteur maximale d'un champ ou d'une fenêtre. |
| [HauteurMin](../Proprietes/2510035.md) | La propriété **HauteurMin** permet de connaître et de modifier la hauteur minimale d'un champ ou d'une fenêtre. |
| [ImageType](../Proprietes/1000023516.md) | La propriété **ImageType** permet de connaître le type de l'image en cours d'édition dans un champ Editeur d'images (icône ou image). |
| [Largeur](../Proprietes/2510059.md) | La propriété **Largeur** permet de connaître et modifier la largeur :<br><br>- d'un champ ou d'un bloc dans un état.<br><br>- d'un champ (présent dans une fenêtre), d'une fenêtre, d'une colonne de table ou d'une colonne de liste (y compris liste image).<br><br>- d'un champ (présent dans une page), d'une colonne de table ou d'une liste. Le champ peut être superposable ou non.<br><br><br> |
| [LargeurImageInitiale](../Proprietes/1410088608.md) | La propriété **LargeurImageInitiale** permet de connaître ou de modifier la largeur par défaut de l'image créée dans le champ Editeur d'images. |
| [LargeurInitiale](../Proprietes/2510067.md) | La propriété **LargeurInitiale** permet de :<br><br>- Connaître la largeur de création d'un champ ou d'une fenêtre.<br><br>- Modifier la largeur de base utilisée pour l'ancrage des champs "en largeur".<br><br>- Connaître la largeur de création d'un champ d'un état.<br><br><br> |
| [LargeurMax](../Proprietes/2510068.md) | La propriété **LargeurMax** permet de connaître et de modifier la largeur maximale d'un champ ou d'une fenêtre. |
| [LargeurMin](../Proprietes/2510062.md) | La propriété **LargeurMin** permet de connaître et de modifier la largeur minimale d'un champ ou d'une fenêtre. |
| [MasquageAuScroll](../Proprietes/1410087944.md) | La propriété **MasquageAuScroll** permet de :<br><br>- Savoir si le champ épinglé est rendu invisible pendant le scroll, <br><br>- Rendre ou non le champ épinglé invisible pendant le scroll.<br><br><br> |
| [Modifié](../Proprietes/2510080.md) | La propriété **Modifié** permet de savoir si un champ ou un groupe de champs a été modifié par l'utilisateur (saisie au clavier ou à la souris). |
| [Nom](../Proprietes/2510082.md) | La propriété **Nom** permet de connaître :<br><br>- le nom de la police associée à une variable de type Police.<br><br>- le nom d'un champ, d'un état ou d'un bloc d'un état.<br><br>- le nom d'un champ, d'un groupe de champs ou d'une fenêtre.<br><br>- le nom d'un champ, d'un groupe de champs ou d'une page.<br><br><br> |
| [NomComplet](../Proprietes/2514005.md) | La propriété **NomComplet** permet de connaître le nom complet :<br><br>- d'un champ.<br><br>- d'une fenêtre.<br><br>- d'une option de menu.<br><br>- d'une colonne de table.<br><br><br><br><br> |
| [Note](../Proprietes/2510085.md) | La propriété **Note** permet de :<br><br>- Connaître les notes associées à un champ, à une fenêtre, à une page ou un état.<br><br>- Modifier les notes associées à un champ, à une fenêtre, à une page ou un état.<br><br><br> |
| [Nouveau](../Proprietes/2510073.md) | La propriété **Nouveau** permet de savoir :<br><br>- dans un champ Table, si la ligne en cours vient d'être créée grâce aux fonctions TableAjouteXXX ou TableInsèreXXX.<br><br>- dans un champ Zone répétée, si la répétition en cours vient d'être créée grâce aux fonctions ZoneRépétéeAjouteXXX ou ZoneRépétéeInsèreXXX.<br><br>- dans un champ Agenda, si un rendez-vous vient d'être créé par le menu contextuel.<br><br>- dans un champ Planning, si un rendez-vous vient d'être créé par le menu contextuel.<br><br><br> |
| [NuméroAide](../Proprietes/2510141.md) | La propriété **NuméroAide** permet de connaître et de modifier le numéro d'aide associé au champ. |
| [OutilInitial](../Proprietes/1410088609.md) | La propriété **OutilInitial** permet de connaître ou de modifier l'outil de dessin initial sélectionné dans le champ Editeur d'images. |
| [Plan](../Proprietes/2510094.md) | La propriété **Plan** permet de :<br><br>- Connaître : <br><br>	- le plan actif d'une fenêtre (ou d'un état) ou le plan auquel appartient un champ.<br><br>	- le plan actif d'un élément WEBDEV.<br><br><br><br><br>- Modifier : <br><br>	- le plan actif d'une fenêtre (ou d'un état) ou associer un champ à un autre plan.<br><br>	- le plan actif d'un élément WEBDEV.<br><br><br><br><br><br> |
| [Ruban](../Proprietes/1000023996.md) | La propriété **Ruban** permet d'accéder au champ Ruban du champ associé (champ Traitement de texte, Lecteur PDF, Tableur, Editeur d'images, Editeur HTML ou Editeur de diagrammes). |
| [TauxAncrageBas](../Proprietes/2510113.md) | La propriété **TauxAncrageBas** permet de :<br><br>- Connaître le taux d'ancrage en déplacement d'un champ sur l'axe vertical.<br><br>- Modifier le taux d'ancrage en déplacement d'un champ sur l'axe vertical.<br><br><br> |
| [TauxAncrageDroite](../Proprietes/2510114.md) | La propriété **TauxAncrageDroite** permet de :<br><br>- Connaître le taux d'ancrage en déplacement d'un champ sur l'axe horizontal.<br><br>- Modifier le taux d'ancrage en déplacement d'un champ sur l'axe horizontal.<br><br><br> |

| [TauxAncrageLargeur](../Proprietes/2510120.md) | La propriété **TauxAncrageLargeur** permet de :<br><br>- Connaître le taux d'ancrage en agrandissement d'un champ (ou d'une colonne de table) sur l'axe horizontal.<br><br>- Modifier le taux d'ancrage en agrandissement d'un champ (ou d'une colonne de table) sur l'axe horizontal.<br><br><br> |
| [TitreNote](../Proprietes/2510121.md) | La propriété **TitreNote** permet de :<br><br>- Connaître le titre des notes utilisé pour la documentation technique d'un champ, d'une fenêtre, d'une page ou d'un état. <br><br>- Modifier le titre des notes utilisé pour la documentation technique d'un champ, d'une fenêtre, d'une page ou d'un état.<br><br><br> |
| [Traitement](../Proprietes/1000020878.md) | La propriété **Traitement** permet de modifier les traitements à exécuter par les événements disponibles sur un champ, une fenêtre, une page, un état, ... |

| [Valeur](../Proprietes/2510130.md) | La propriété **Valeur** permet de :<br><br>- Connaître et modifier la valeur d'un champ d'un état.<br><br>- Connaître et modifier la valeur d'un champ ou le titre d'une fenêtre.<br><br>- Connaître l'identifiant d'un flux. <br><br>- Connaître et modifier la valeur d'un champ ou le titre d'une page.<br><br><br> |
| [ValeurInitiale](../Proprietes/2510128.md) | La propriété **ValeurInitiale** permet de connaître la valeur initiale d'un champ d'une fenêtre ou d'une page. |
| [Vide](../Proprietes/2510126.md) | La propriété **Vide** permet de savoir si :<br><br>- un champ Zone répétée, Table, Liste ou Combo est vide. Le champ peut être un champ fichier ou mémoire.<br><br>- un champ Graphe ne contient aucune série.<br><br>- un champ d'un état est vide (champ lié à une rubrique).<br><br>- un tableau (ou un tableau associatif) ne contient aucun élément.<br><br>- un élément d'un tableau associatif existe ou non.<br><br>- une file, une liste ou une pile ne contient aucun élément. <br><br><br> |
| [Visible](../Proprietes/2510138.md) | La propriété **Visible** permet de :<br><br>- Savoir si un élément est visible.<br><br>- Rendre un élément visible ou invisible.<br><br><br> |
| [VisibleAvecAnimation](../Proprietes/1410087232.md) | La propriété **VisibleAvecAnimation** permet de rendre un champ visible ou invisible en jouant l'animation "Apparition/Disparition". |
| [VisibleInitial](../Proprietes/2510070.md) | La propriété **VisibleInitial** permet de savoir si :<br><br>- un champ était visible lors du premier affichage de la fenêtre ou la page associée.<br><br>- une fenêtre était visible lors de son premier affichage.<br><br><br> |
| [Volet](../Proprietes/1000021098.md) | La propriété **Volet** permet de : <br><br>- Connaître les volets d'onglet associés à un champ. <br><br>- Définir les volets d'onglet associés à un champ.<br><br><br> |

## Présentation du champ Forme
<a name="presentation_champ_forme_ELTTEXTE000209"></a>


### Champ d'un état ou d'une fenêtre
<a name="champ_etat_une_fenetre_ELTPARAGRAPHE000011"></a>Le champ Forme permet de créer dans vos fenêtres et dans vos états des formes géométriques :

- Rectangle.

- Ellipse (ce type de forme n'est pas disponible dans les états).

- Losange (ce type de forme n'est pas disponible dans les états).

- Flèche (ce type de forme n'est pas disponible dans les états).

- Trait.




![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Le champ Forme est disponible uniquement dans l'éditeur d'états.

## Créer un champ forme

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) 

### Créer un champ de type Forme sous l'éditeur de fenêtres
<a name="creer_champ_type_forme_sous_editeur_fenetres_ELTPARAGRAPHE000050"></a>Pour créer un champ de type Forme sous l'éditeur de fenêtres : 

1. Sous le volet "Création", dans le groupe "Autres champs", cliquez sur "Forme".

2. Cliquez dans la fenêtre à la position où le champ doit être créé. L'assistant de création de forme se lance. Choisissez la forme à créer et ses caractéristiques (épaisseur de trait, ...).

3. Validez la création de la forme.




Pour afficher les caractéristiques du champ, sélectionnez l'option "Description" dans le menu contextuel du champ.

Remarques : 

- Pour créer directement une forme spécifique, sous le volet "Création", dans le groupe "Autres champs", déroulez "Forme" et choisissez la forme voulue.

- A partir de la version 12, les formes de type Flèche, Ellipse et Trait utilisent le [framework GDI+](../Editeurs/2020007.md) : les effets de "crénelage" sont ainsi diminués. Si votre fenêtre possède des champs Forme créés avec une version précédente, ce nouveau mode de dessin est automatiquement appliqué.

- Sous l'éditeur de fenêtres, il est possible de définir une image de fond pour les formes de type Rectangle, Ellipse et Losange (option "Image de fond" dans l'onglet "Détail" de la fenêtre de description du champ). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Sous l'éditeur de fenêtres, la fonction [FormePolygone](../WDLang1/1000022000.md) permet de créer une forme libre.

### Créer un champ de type Forme sous l'éditeur d'états
<a name="creer_champ_type_forme_sous_editeur_etats_ELTPARAGRAPHE000108"></a>Pour créer un champ de type Forme sous l'éditeur d'états : 

1. Sous le volet "Création", dans le groupe "Visuel", déroulez "Forme" et sélectionnez la forme voulue.

2. Cliquez dans l'état à la position où le champ doit être créé.

3. Pour modifier la forme, affichez la fenêtre de description du champ (option "Description" dans le menu contextuel du champ) et choisissez la forme voulue dans l'onglet "Général".

### Comment modifier l'angle d'une flèche sous l'éditeur de fenêtres
<a name="comment_modifier_angle_une_fleche_sous_editeur_fenetres_ELTPARAGRAPHE000135"></a>Pour modifier l'angle d'une flèche sous l'éditeur, il suffit de :

1. Sélectionner l'extrémité de la flèche avec la souris.

2. Maintenir le bouton gauche de la souris enfoncé.

3. Déplacer la souris jusqu'à obtenir l'angle voulu.



<a name="NOTE3_2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 

### Déplacer les extrémités d'une flèche par programmation
<a name="deplacer_les_extremites_une_fleche_par_programmation_ELTPARAGRAPHE000148"></a>Pour déplacer les extrémités d'une flèche, vous pouvez utiliser les fonctions [FlècheXXX](../WDLang1/3029033.md) : 

- La fonction [FlècheDébut](../WDLang1/3029059.md) permet de positionner l'extrémité d'origine de la flèche.

- La fonction [FlècheFin](../WDLang1/3029060.md) permet de positionner l'extrémité de fin de la flèche.




## Manipuler un champ HTML par programmation
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000115"></a>
Le champ HTML peut être manipulé directement par programmation. Il suffit d'utiliser directement la variable du champ HTML dans le code. La variable du champ HTML correspond au nom du champ HTML. Le type de la variable est une chaîne de caractères.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Afficher une information dans un champ HTML
<a name="afficher_une_information_dans_champ_html_ELTTEXTE000139"></a>


### Champ HTML présent dans une page (code serveur)
<a name="champ_html_present_dans_une_page_code_serveur_ELTPARAGRAPHE000018"></a>Pour afficher une information dans un champ HTML, une simple affectation suffit.

Quelques exemples :

- Affectation directe :
	
	```wl
	HTM_PageWeb = "Nom de l'internaute"
	```

	
	```wl
	soit sMaPage = [
		<a href="http://www.pcsoft.fr" target="_blank"> PC SOFT</a>
		]
	HTM_Site = sMaPage
	```


- Affectation par variable :
	
	```wl
	HTM_PageWeb = MaPage
	```


- Affectation par calcul :
	
	```wl
	HTM_Nom = "Nom du Client : " + SansEspace(CLIENT.NOM) + TAB + SansEspace(CLIENT.PRENOM)
	```



Il est également possible d'utiliser la propriété [Valeur](../Proprietes/2510130.md).

## Récupérer la valeur d'un champ HTML
<a name="recuperer_valeur_champ_html_ELTTEXTE000163"></a>
<a name="NOTE3_2"></a>


### Champ d'une page
<a name="champ_une_page_ELTPARAGRAPHE000048"></a>Pour récupérer le contenu d'un champ HTML (l'utilisateur a la possibilité de saisir dans le champ) :

- Réalisez une simple affectation. Par exemple :
	
	```wl
	ContenuHTML = HTM_ChampHTML
	```


- Utilisez la propriété [Valeur](../Proprietes/2510130.md).

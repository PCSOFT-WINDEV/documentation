## Présentation du champ Boîte à outils
<a name="presentation_champ_boite_outils_ELTTEXTE000102"></a>


### Champ d'une fenêtre
<a name="champ_une_fenetre_ELTPARAGRAPHE000011"></a>Le champ Boîte à outils est un moyen rapide d'accéder à différentes options.

Un champ Boîte à outils est constitué de plusieurs volets. Chaque volet peut contenir tous les types de champs (y compris un autre champ Boîte à outils).

## Création d'un champ Boîte à outils
<a name="creation_champ_boite_outils_ELTTEXTE000126"></a>


### Créer un champ de type Boîte à outils
<a name="creer_champ_type_boite_outils_ELTPARAGRAPHE000023"></a>Pour créer un champ de type Boîte à outils :

1. Sous le volet "Création", dans le groupe "Conteneurs", déroulez "Onglet et associés" et sélectionnez "Boîte à outils".

2. Cliquez dans la fenêtre à la position où le champ doit être créé. Le champ apparaît en création.




Pour afficher les caractéristiques du champ, sélectionnez l'option "Description" dans le menu contextuel du champ. Pour plus de détails, consultez [Caractéristiques d'une boîte à outils](../WDChamp/1013115.md).
<a name="NOTE2_2"></a>


### Associer un champ à un volet d'un champ Boîte à outils
<a name="associer_champ_volet_champ_boite_outils_ELTPARAGRAPHE000050"></a>Pour associer un champ à un volet d'une boîte à outils :

1. Affichez sous l'éditeur le volet du champ Boîte à outils dans lequel le champ doit être affiché.

2. Déplacez le champ dans le contenu du volet du champ Boîte à outils.

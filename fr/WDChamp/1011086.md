


## Visualisateur de rapports : Volet "Annoter"
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000148"></a>
Votre application permet d'imprimer de nombreux états ? L'utilisateur final a la possibilité d'ajouter des annotations depuis le visualisateur de rapports : dessin à main levée, flèche, texte formaté, ...

**Les annotations seront imprimées avec l'état et seront également affichées lors d'un export (PDF, email, etc.)**.

Cette fonctionnalité est disponible dans les impressions réalisées à l'aide de l'éditeur d'états (ou avec le logiciel "Etats & Requêtes") et dans les impressions programmées (réalisées avec les fonctions WLangage).

Rappel : Le visualisateur de rapports est disponible : 

- sous l'éditeur d'états de WINDEV, WEBDEV, WINDEV Mobile. 

- dans le logiciel "Etats et Requêtes".

- lors de l'impression d'états depuis une application WINDEV.

## Comment le faire ?
<a name="comment_faire_ELTTEXTE000172"></a>


### Insérer des annotations dans un état
<a name="inserer_des_annotations_dans_etat_ELTPARAGRAPHE000030"></a>Pour insérer des annotations, utilisez le volet "Annoter" du visualisateur de rapports : 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Volet_Apercu_Annoter.gif)
Vous pouvez :

- Annuler ou rétablir une action.

- Dessiner à main levée : 

	- Cliquez sur "Pinceau". 

	- Commencez le dessin.




- Dessiner une flèche. 

	- Cliquer sur "Flèche". 

	- Cliquez dans le visualisateur de rapports pour définir le point de départ.

	- Maintenez le bouton gauche de la souris jusqu'au point d'arrivé de la flèche. La pointe de la flèche sera dessinée à la fin du trait.




- Dessiner un cadre. 

	- Cliquez sur "Rectangle". 

	- Cliquez dans le visualisateur de rapports pour définir le coin haut gauche du cadre.

	- Maintenez le bouton gauche de la souris jusqu'au coin bas droit du cadre. 




- Insérer un texte. 

	- Cliquez sur "Texte". 

	- Définissez un cadre où le texte doit être inséré. 

	- Une barre d'outils permet de définir les caractéristiques du texte inséré (taille, couleur, ...).

	- Saisissez le texte. 




- Insérer une image. 

	- Cliquez sur "Image". 

	- Définissez la zone où l'image doit être insérée.

	- Le sélecteur de fichiers s'ouvre automatiquement pour saisir le chemin de l'image à insérer.




- Insérer un filigrane. 

	- Cliquez sur "Filigrane". 

	- Dans la fenêtre qui s'affiche, il est possible de définir : 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Fen_Filigrane.gif&type=thumb)


		- la destination du filigrane : impression et/ou duplicata. 

- le positionnement du filigrane : diagonal montant, descendant ou horizontal. 

- le texte du filigrane, sa police, sa taille, sa couleur et son opacité. 

- Flouter du texte, une image, ...

	- Cliquez sur "Flouter une zone". 

	- Définissez la zone à flouter. 

	- Dans la zone définie les différents éléments sont floutés.




- Définir la taille par défaut du trait. 

- Définir la couleur utilisée par défaut.



<a name="NOTE2_2"></a>


### Supprimer une annotation dans un état
<a name="supprimer_une_annotation_dans_etat_ELTPARAGRAPHE000086"></a>Pour supprimer une annotation dans un état :

1. Sélectionnez l'annotation voulue.

2. Supprimez l'annotation avec la touche Suppr du clavier.




**Remarque**

- La fonction [iParamètreVisualisateur](../WDLang5/1000024634.md) permet de désactiver si nécessaire le volet des annotations (constante *iBoutonAnnotation*).

- Les annotations peuvent être déplacées uniquement dans la page en cours affichée dans le visualisateur de rapports. Il n'est pas possible de déplacer une annotation d'une page à une autre.

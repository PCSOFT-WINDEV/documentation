## Présentation
<a name="presentation_ELTTEXTE000102"></a>


### Champ d'une page
<a name="champ_une_page_ELTPARAGRAPHE000011"></a>Le champ "Page cornée" permet de créer un effet visuel dans une page en présentant une image partiellement recouverte par une autre. Lors du survol du champ, l'image du dessus est progressivement repliée afin de faire apparaître l'image du dessous, à la façon d'un coin de feuille cornée sous lequel on verrait la page inférieure.<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Cornage1.gif)<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Cornage2.gif)

## Créer un champ Page cornée
<a name="creer_champ_page_cornee_ELTTEXTE000126"></a>
Pour créer un champ Page cornée :

1. Sous le volet "Création", dans le groupe "Champs usuels", déroulez "Image" et sélectionnez "Page cornée".

2. Cliquez dans la page à la position où le champ doit être créé. 




<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Caractéristiques d'un champ Page cornée
<a name="caracteristiques_champ_page_cornee_ELTTEXTE000150"></a>
La fenêtre de description du champ permet de définir : 

- l'image qui sera découverte par la page cornée (onglet "Général"). 

- l'action à effectuer à effectuer lors du clic sur le champ (onglet "Général"). Cette information est identique à celle des champs Bouton. Pour plus de détails, consultez [Choisir l'action associée au champ](../WDChamp/1013074.md) et [choisir le type d'action associée au champ](../WDChamp/1013071.md). 

- l'image utilisée pour l'effet corné (onglet "Style"). 

- le coin découvert de l'image (onglet "Style").




## Bouton poussoir (On/Off) dans une fenêtre
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000119"></a>
Le bouton poussoir est un bouton qui fonctionne comme un interrupteur.

Si l'état normal du bouton est le suivant : <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Btn_OnOff.gif)


Lorsque l'utilisateur clique une première fois sur le bouton, le bouton reste enfoncé : <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Btn_OnOff_enfonce.gif)


Si l'utilisateur clique une seconde fois sur le bouton, le bouton revient à son état normal. <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Btn_OnOff.gif)




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Mise en place
<a name="mise_place_ELTTEXTE000143"></a>


### Créer un bouton poussoir
<a name="creer_bouton_poussoir_ELTPARAGRAPHE000049"></a>Pour créer un bouton poussoir :

1. Créez un bouton.

2. Affichez l'onglet "UI" de la fenêtre de description du bouton et sélectionnez l'option "Bouton On/Off".

3. Validez la fenêtre de description.



<a name="NOTE2_2"></a>


### Programmation
<a name="programmation_ELTPARAGRAPHE000064"></a>La programmation d'un bouton poussoir est identique à celle des autres types de bouton (voir [Manipuler un bouton par programmation](../WDChamp/1013094.md)). Seule différence : la gestion de l'état du bouton poussoir est réalisée grâce à la propriété [Enfoncé](../Proprietes/2510038.md).

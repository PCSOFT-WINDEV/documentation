## Présentation
<a name="presentation_ELTTEXTE000313"></a>
Quelque soit l'éditeur (fenêtres, pages ou états), l'onglet "Détail" de la fenêtre de description du champ Graphe permet de décrire : 

- [l'apparence du graphe](#NOTE2_1),

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) [les caractéristiques du zoom et de la mire](#NOTE3_1),

- [le titre et la légende](#NOTE4_1),

- [les animations et l'image de fond](#NOTE5_1).

## Apparence du graphe
<a name="apparence_graphe_ELTTEXTE000337"></a>
Les caractéristiques d'apparence du graphe sont les suivantes : 

- **Adoucir les effets d'escalier (Antialiasing)** : permet d'utiliser l'antialiasing si le GDI+ est installé. 
	![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Cette option n'est pas disponible.

- **Effet de lumière réaliste** : permet d'utiliser un effet de lumière sur le graphe. Si cette option est choisie, le pourcentage de biseau permet de paramétrer l'inclinaison de la lumière. 
	

- **Couleurs de bordure automatiques** : Si cette option est cochée, les couleurs de bordure des séries sont calculées automatiquement. 

- **Couleurs dégradées** : Permet d'utiliser des couleurs dégradées dans l'affichage des séries. Le dégradé est par défaut vertical mais il est possible d'utiliser un dégradé horizontal. 

- **Afficher aussi les étiquettes superposées** : Cette option permet d'autoriser le chevauchement des étiquettes dans les graphes de type Secteur, Hémicycle, Beignet, ... 
	

- **Afficher les étiquettes à l'extérieur des portions** : Cette option permet de placer les étiquettes en dehors des portions dans les graphes de type Secteur, Hémicycle, Beignet, ... 
	

- **Afficher les valeurs** : Permet d'afficher les valeurs en plus des pourcentages dans les étiquettes. Il est possible de définir le masque d'affichage à utiliser. 
	L'option **Valeurs en vertical** permet d'afficher les valeurs verticalement.

## Zoom et mire (non disponible dans les états)
<a name="zoom_mire_non_disponible_dans_les_etats_ELTTEXTE000361"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 

### Gestion du Zoom
<a name="gestion_zoom_ELTPARAGRAPHE000118"></a>L'activation du Zoom permet à l'utilisateur de pouvoir zoomer sur les données affichées dans le graphe.

- **Autoriser le zoom horizontal** : permet d'activer uniquement le zoom des valeurs selon l'axe horizontal. L'ascenseur de zoom est placé sous le champ. 

- **Autoriser le zoom vertical** : permet d'activer uniquement le zoom des valeurs selon l'axe vertical. L'ascenseur de zoom est placé à droite du champ. 



![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 

### Gestion de la mire
<a name="gestion_mire_ELTPARAGRAPHE000136"></a>La mire permet à l'utilisateur de connaître avec précision la valeur pointée. Pour plus de détails, consultez [Mire dans un graphe](../WDChamp/3042078.md). 

L'activation de la mire est réalisée via l'option "Activer la mire du graphe".

L'option "Mire multiple (courbe et histo. empilés)" permet d'obtenir une mire multiple dans les graphes de type Courbe ou Histogramme.

Remarque : La mire est disponible pour les types de graphes suivants : Aire, Bulle, Courbe, Histogramme, MinMax, Radar et Waterfall.

## Titre et légende
<a name="titre_legende_ELTTEXTE000391"></a>
Le titre et la légende peuvent être paramétrés dans un champ Graphe. 

Le bouton "Editer le titre du graphe" permet d'afficher la fenêtre de description du titre du graphe. Cette fenêtre permet de : 

- Saisir si nécessaire le titre du graphe. 

- Paramétrer son positionnement. Si un positionnement manuel est choisi, il est possible de déplacer la zone du titre sous l'éditeur. 

- Paramétrer le style utilisé pour afficher le titre du graphe. 




Le bouton "Editer la légende du graphe" permet d'afficher la fenêtre de description de la légende du graphe. Cette fenêtre permet de : 

- Définir le positionnement de la légende. 
	Ce positionnement peut être automatique ou manuel. 
	Si un positionnement manuel est choisi, il est possible : 

	- de déplacer la zone de la légende sous l'éditeur. 

	- de choisir la position des libellés (en ligne ou en colonne)

	- d'adapter la taille de la légende au contenu




- Définir l'aspect de la légende : Il est possible d'utiliser un cadre arrondi pour la légende. 

- Définir le style de la légende (police, ...).

## Animations et image de fond
<a name="animations_image_fond_ELTTEXTE000415"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 

### Animations
<a name="animations_ELTPARAGRAPHE000199"></a>Certains graphes peuvent être animés. 
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Pour les graphes 2D, il est possible d'activer ou non les animations. 

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Pour les graphes 3D, plusieurs options peuvent être paramétrées : 

- Zoom avant, 

- Construction progressive, 

- Rotation, 

- Couleurs progressives. 




Pour plus de détails sur la gestion des animations dans le champ graphe, consultez [Animation d'un champ Graphe](../WDChamp/1013296.md). 


### Image de fond
<a name="image_fond_ELTPARAGRAPHE000230"></a>L'image de fond sera affichée en fond du graphe. 
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Il est possible d'utiliser le mode 9 images pour cette image.

## Présentation
<a name="presentation_ELTTEXTE000282"></a>
WINDEV permet de personnaliser :

- les fenêtres et les champs de ces fenêtres en utilisant le thème de Windows XP/Vista.

- les menus (principaux et/ou contextuels) des fenêtres en utilisant le look XP/Vista (fonctionnalité utilisable même si le poste en cours n'est pas sous Windows XP/Vista).




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Utiliser le thème de Windows XP/Vista
<a name="utiliser_theme_windows_xpvista_ELTTEXTE000306"></a>


### Présentation
<a name="presentation_ELTPARAGRAPHE000024"></a>Le thème de Windows XP/Vista correspond aux caractéristiques de l'interface d'un poste sous Windows XP/Vista. Ce thème est défini dans le panneau de configuration de Windows.

WINDEV permet de forcer l'utilisation du thème de Windows XP/Vista sur les fenêtres et les champs de ces fenêtres. Dans ce cas, dès que l'application sera ouverte sur un poste sous Windows XP/Vista, le thème de Windows XP/Vista défini sur ce poste sera appliqué.

**Remarque** : Si l'application est ouverte sur un poste utilisant un autre système d'exploitation, les caractéristiques de style (définies dans l'onglet "Style" des fenêtres de description) seront automatiquement appliquées.
<a name="NOTE2_2"></a>


### Forcer l'utilisation du thème de Windows XP/Vista sur une fenêtre
<a name="forcer_utilisation_theme_windows_xpvista_sur_une_fenetre_ELTPARAGRAPHE000035"></a>Pour forcer l'utilisation du thème de Windows XP/Vista sur une fenêtre :

1. Ouvrez la fenêtre voulue sous WINDEV (par exemple depuis l'explorateur de projet).

2. Affichez le volet "Modifieur" : sous le volet "Accueil", dans le groupe "Environnement", déroulez "Volets" puis sélectionnez "Volets" et "Modifieur".

3. Dans le modifieur, dans la catégorie "Style", passez l'option "Thème XP" à "Oui". 
	Dans l'onglet "Style" de la fenêtre de description de la fenêtre en cours, l'option "Forcer le thème actif sous Windows XP/Vista" est alors affichée et automatiquement cochée.




**Remarques** :

- Les fenêtres utilisant le thème de Windows XP/Vista conservent toutes leurs caractéristiques de style (définies dans l'onglet "Style" de la fenêtre de description).

- Si une image de fond est utilisée, cette image est conservée. Sinon, le thème XP/Vista correspondant (couleur de fond de la fenêtre) est appliqué.



<a name="NOTE2_3"></a>


### Forcer l'utilisation du thème de Windows XP/Vista sur un champ d'une fenêtre
<a name="forcer_utilisation_theme_windows_xpvista_sur_champ_une_fenetre_ELTPARAGRAPHE000065"></a>Pour forcer l'utilisation du thème de Windows XP/Vista sur un champ d'une fenêtre : 

1. Forcez l'utilisation du thème de Windows XP sur la fenêtre à laquelle le champ appartient (voir le paragraphe ci-dessus).

2. Sélectionnez le champ voulu.

3. Affichez l'onglet "Style" de la fenêtre de description de ce champ (option "Description" du menu contextuel).

4. Sélectionnez "Thème XP/Vista" dans la combo "Élément".

5. Cochez l'option "Utiliser le thème actif sous Windows XP/Vista".




Remarque : Si le thème n'a pas été forcé au niveau de la fenêtre, la coche ne sera pas présente au niveau du champ.

<a name="utiliser_look_xpvista_ELTTEXTE000348"></a>

### Utiliser le look XP/Vista dans le menu principal d'une fenêtre
<a name="utiliser_look_xpvista_dans_menu_principal_une_fenetre_ELTPARAGRAPHE000214"></a>Pour que le menu principal d'une fenêtre utilise le look XP/Vista : 

1. Ouvrez la fenêtre voulue sous WINDEV.

2. Insérez si nécessaire le menu principal : sous le volet "Fenêtre", dans le groupe "Barres et menus", déroulez "Menu principal" et sélectionnez "Ajouter le menu principal".

3. Affichez l'onglet "Style" de la fenêtre de description de la fenêtre en cours (option "Description" du menu contextuel).

4. Cochez l'option "Afficher le menu au look XP/Vista".




**Remarque** : Les menus utilisant le look XP conservent les caractéristiques de la police des options de menu (taille, couleur, ...). Ces caractéristiques ont été définies dans l'onglet "Style" de la fenêtre de description de l'option de menu.

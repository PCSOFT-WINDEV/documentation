## Présentation
<a name="presentation_ELTTEXTE000091"></a>


### Champ d'une page dynamique ou statique
<a name="champ_une_page_dynamique_statique_ELTPARAGRAPHE000014"></a>Le champ Flex permet d'afficher une animation Flash® créée par un éditeur Flex dans une page WEBDEV.

**Remarque** : Pour afficher une animation Flash® créée par un éditeur Flex, le navigateur doit posséder Adobe Flash Player version 10 minimum.


### Créer un champ de type Flex
<a name="creer_champ_type_flex_ELTPARAGRAPHE000021"></a>Pour créer un champ de type Flex : 

1. Sous le volet "Création", dans le groupe "Intégration", déroulez "Plug-in" puis cliquez sur "Champ Flex".

2. Cliquez dans la page à la position où le champ doit être créé. Le champ apparaît en création.

3. Dans la description du champ, associez l'animation Flash® (fichier ".swf") au champ Flex.




Pour afficher les caractéristiques du champ, sélectionnez l'option "Description" dans le menu contextuel du champ.


### Remarque : Localisation de l'animation
<a name="remarque_localisation_animation_ELTPARAGRAPHE000040"></a>Il est conseillé de placer l'animation Flex dans le répertoire des images (répertoire &lt;Nom_Projet&gt;_WEB). Les fichiers de dépendances de l'animation (images, fichiers texte, ...) doivent être copiés également dans le répertoire &lt;Nom_Projet&gt;_WEB.

Si l'animation utilise des fichiers Javascript (.js), il est nécessaire d'inclure ces fichiers dans la page (onglet "Avancé" de la description de la page).

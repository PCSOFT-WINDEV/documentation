## Réseau social de type Facebook
<a name="reseau_social_type_facebook_ELTTEXTE000556"></a>
Pour un champ Réseau social Facebook, plusieurs types sont disponibles. Chaque type peut être associé à une cible spécifique. 

Le tableau suivant résume pour chaque type et chaque cible les informations utilisées. 


| Type | Cible | Effet |
| --- | --- | --- |
| **J'aime ou Recommander** | Le domaine | Crée sur le mur Facebook de l'internaute connecté un lien vers le domaine du site en cours.<br><br>Attention : Une page d'index par défaut doit exister sur le serveur d'hébergement. |
|   | Le site | Crée sur le mur Facebook de l'internaute connecté un lien vers le site en cours.<br><br>Attention : Une page d'index par défaut doit exister sur le serveur d'hébergement. |
|   | La page | Crée sur le mur Facebook de l'internaute connecté un lien vers la page du site en cours. |
|   | Cible personnalisée | <br><br>L'URL doit être saisie dans le champ "URL cible".<br><br><br><br>**Cas 1 : URL d'une page Facebook** : <br><br>- Incrémente le compteur de la page Facebook aimée.<br><br>- Crée un favori sur le mur Facebook de l'internaute connecté.<br><br><br>**Cas 2 : URL d'une page ou d'un site quelconque** : <br>Crée sur le mur Facebook de l'internaute connecté un lien vers la page ou le site spécifié. |
| **Envoyer** | Le domaine | Permet d'envoyer un lien sur le domaine à un autre utilisateur Facebook. Le message envoyé est un message privé. |
|   | Le site | Permet d'envoyer un lien sur le site à un autre utilisateur Facebook. Le message envoyé est un message privé. |
|   | La page | Permet d'envoyer un lien sur la page à un autre utilisateur Facebook. Le message envoyé est un message privé. |
|   | Cible personnalisée | Permet d'envoyer un lien à un autre utilisateur Facebook. Ce lien correspond à un lien vers une page ou un site quelconque. Le message envoyé est un message privé.<br><br><br><br><br><br>L'URL du lien à envoyer doit être saisie dans le champ "URL cible". |
| **S'abonner** | Cible personnalisée | <br><br>Dans le champ "Compte cible", il est possible de saisir :<br><br>- l'URL de la page Facebook à laquelle l'internaute veut s'abonner. Par exemple : "www.facebook.com/01net" ou "http://www.facebook.com/pages/PC-SOFT/247782873866"<br><br>- l'identifiant du compte Facebook auquel l'internaute veut s'abonner. Par exemple : "01net" ou "247782873866".<br><br><br> |
| **Afficher le profil** | Cible personnalisée | Dans le champ "Compte cible", il est possible de saisir l'identifiant du compte Facebook correspondant au profil. Par exemple : "01net" ou "247782873866". |





Remarque : Pour permettre la publication sur le mur de l'internaute, la page comportant le champ Réseau social doit remplir les conditions suivantes : 

- la page doit posséder un titre. 

- la page doit posséder au moins un champ Image. 




Attention : Lors du test du site (GO en mode localhost), seules les bulles d'aide sont correctes. Il n'est pas possible de publier sur un mur.

## Réseau social de type Twitter
<a name="reseau_social_type_twitter_ELTTEXTE000580"></a>
Pour un champ Réseau social Twitter, plusieurs types sont disponibles. Chaque type peut être associé à une cible spécifique. 

Le tableau suivant résume pour chaque cible les informations utilisées. 


| Type | Cible | Effet |
| --- | --- | --- |
| **Tweeter** | Le domaine | Crée sur le mur Twitter de l'internaute connecté un message contenant un lien vers le domaine du site en cours. |
|   | Le site | Crée sur le mur Twitter de l'internaute connecté un message contenant un lien vers le site en cours. |
|   | La page | Crée sur le mur Twitter de l'internaute connecté un message contenant un lien vers la page du site en cours. |
|   | Cible personnalisée | <br><br>L'URL doit être saisie dans le champ "URL cible".<br><br><br><br>Crée sur le mur Twitter de l'internaute connecté un message contenant un lien vers l'URL indiquée. |
| **Suivre** | Cible personnalisée | <br><br>Dans le champ "URL cible", il est possible de saisir :<br><br><br><br>- l'URL de la page Twitter à laquelle l'internaute veut s'abonner. Par exemple : "https://twitter.com/PCSOFT_FR"<br><br>- l'identifiant du compte Twitter auquel l'internaute veut s'abonner. Par exemple : "PCSOFT_FR".<br><br><br> |
| **Afficher le profil** | Cible personnalisée | Dans le champ "Compte cible", il est possible de saisir l'identifiant du compte Twitter correspondant au profil voulu. Par exemple : "PCSOFT_FR". |


Remarques : 

- Les messages Twitter sont limités à 140 caractères. 

- Les champs Réseau social de type Twitter fonctionnent en mode test (mais l'URL n'est pas renseignée dans les messages).

## Réseau social de type LinkedIn
<a name="reseau_social_type_linkedin_ELTTEXTE000604"></a>
Pour un champ Réseau social LinkedIn, il est possible de partager le contenu en cours. 

Le tableau suivant résume pour chaque type et chaque cible les informations utilisées. 


| Type | Cible | Effet |
| --- | --- | --- |
| **Partager** | Le domaine | Crée un post sur les nouvelles du profil LinkedIn connecté avec un lien sur le domaine.<br><br>Attention : Une page d'index par défaut doit exister sur le serveur d'hébergement. |
|   | Le site | Crée un post sur les nouvelles du profil LinkedIn connecté avec un lien sur le site.<br><br>Attention : Une page d'index par défaut doit exister sur le serveur d'hébergement. |
|   | La page | Crée un post sur les nouvelles du profil LinkedIn connecté avec un lien sur la page. |
|   | Cible personnalisée | Crée un post sur les nouvelles du profil LinkedIn connecté avec un lien personnalisé.<br><br>L'URL du lien à envoyer doit être saisie dans le champ "URL cible". |
| **Afficher le profil** | Cible personnalisée | Affiche la page correspondant au profil LinkedIn. <br><br>Dans le champ "Compte cible", il est possible de saisir l'identifiant du compte LinkedIn correspondant au profil voulu. |


**Attention** : Lors du test du site (GO en mode localhost), il n'est pas possible de tester ce type de champ. 

Remarque : Pour renseigner la vignette qui apparaîtra dans les nouvelles, il est nécessaire de fournir cette image en utilisant une description OpenGraph. Il est nécessaire d'intégrer une balise spécifique dans l'entête de la page. Cette balise doit contenir l'image à afficher.

Par exemple : 

```txt
<meta property="og:image" content="http://mondomaine.com/monimage.jpg"/>
```

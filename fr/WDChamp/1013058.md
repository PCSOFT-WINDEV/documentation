## Présentation du champ Prédéfini
<a name="presentation_champ_predefini_ELTTEXTE000115"></a>


### Champ d'un état
<a name="champ_etat_ELTPARAGRAPHE000011"></a>Le champ "Prédéfini" permet d'afficher des champs très souvent utilisés sans avoir à les programmer. Il est ainsi possible d'insérer dans un état un champ "Pré-défini" pour afficher les informations suivantes :

- Date système.

- Heure système.

- Numéro de la page en cours.

- Numéro de la page / Nombre de pages.

- Nombre de pages total de l'état.

## Création d'un champ Prédéfini
<a name="creation_champ_predefini_ELTTEXTE000139"></a>
Pour créer un champ "prédéfini" dans un état :

1. Sous le volet "Création", dans le groupe "Texte", cliquez sur "Prédéfini".

2. Cliquez dans l'état à la position où le champ doit être créé.

3. Dans l'assistant de création de champ, sélectionnez le type de champ à créer.

4. Indiquez le contenu ou le format du champ.

5. Le champ est automatiquement créé à l'emplacement indiqué.




Remarque : Il est également possible d'utiliser les champs prédéfinis proposés par défaut. Sous le volet "Création", dans le groupe "Texte", cliquez sur "Prédéfini" et sélectionnez le type de champ à créer. Cliquez ensuite dans l'état à la position où le champ doit être créé.

## Comment modifier le type d'information affiché dans un champ "prédéfini" ?
<a name="comment_modifier_type_information_affiche_dans_champ_predefini_ELTTEXTE000163"></a>
Pour modifier le type d'information affiché dans un champ "prédéfini" :

1. Affichez l'onglet "Général" de la fenêtre de description du champ "prédéfini" (option "Description" du menu contextuel du champ).

2. Sélectionnez le type d'information affiché (combo "Type").

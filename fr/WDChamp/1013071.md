


## Choisir l'action associée au bouton (WEBDEV)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000262"></a>
Quel que soit le type du bouton, il est possible d'associer une action à ce bouton. WEBDEV propose les actions les plus courantes directement dans l'onglet "Général" de la fenêtre de description du champ.
<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Btn_Action.gif)


Pour les actions plus spécifiques, il suffit de cliquer sur le lien "Plus de choix".

Dans tous les cas, il est nécessaire de définir la destination de cette action.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Les actions disponibles
<a name="les_actions_disponibles_ELTTEXTE000286"></a>


### Actions prédéfinies
<a name="actions_predefinies_ELTPARAGRAPHE000024"></a>Différentes actions prédéfinies sont proposées par défaut dans une liste déroulante. Ces actions prédéfinies dépendent des éléments présents dans la page et présents dans le projet.

Il est ainsi possible par exemple de choisir de :

- exécuter le code de clic d'un champ Bouton (code navigateur ou code serveur et navigateur),

- afficher une page du site,

- afficher une page d'un autre site,

- réaliser une action personnalisée (envoyer un email, appeler un numéro de téléphone, se connecter à un serveur FTP, télécharger un fichier, ...). Dans ce cas, des paramètres peuvent être spécifiés. 




**Attention** : les actions sont effectuées après le code de clic du bouton (code navigateur et code serveur).
<a name="NOTE2_2"></a>


### Actions spécifiques
<a name="actions_specifiques_ELTPARAGRAPHE000040"></a>Le lien "Plus de choix" permet de proposer plus d'options. Il est ainsi possible par exemple de :

- faire une action sur un autre champ, 

- se positionner sur une ancre de la page, 

- partager un réseau social,

- changer la langue courante

- envoyer la valeur des champs de la page vers ... une page ou un script. Cette option permet par exemple de traiter le contenu d'un formulaire présent dans un site statique.




**Remarque** : Toutes ces opérations sont également réalisables par programmation.
<a name="NOTE2_3"></a>


### Cas particulier : partager sur un  réseau social
<a name="cas_particulier_partager_sur_reseau_social_ELTPARAGRAPHE000057"></a>L'action "Partager sur un réseau social" permet de créer un champ Bouton, un champ Lien ou un champ Image (avec zones de clic) permettant de partager la page en cours, le domaine en cours ou une URL spécifique sur un réseau social. Cette action peut également être effectuée grâce au champ [Réseau social](../WDChamp/1000028001.md). 

Remarque : L'utilisation d'un champ Bouton, d'un champ Lien ou d'un champ Image (avec zones de clic) permet de paramétrer entièrement le look utilisé (il est cependant conseillé de suivre les licences d'utilisation des différents réseaux sociaux). Le champ [Réseau social](../WDChamp/1000028001.md) offre un paramétrage avancé (affichage du nombre de partages, ...). 

Le tableau suivant présente pour chaque réseau social proposé, les informations utilisées : 

| Réseau social | Elément partagé | Effet |
| --- | --- | --- |
| **Facebook** | Le domaine | Crée sur le fil Facebook de l'internaute connecté un lien vers le domaine du site en cours.<br><br>Attention : Une page d'index par défaut doit exister sur le serveur d'hébergement. |
|   | La page | Crée sur le fil Facebook de l'internaute connecté un lien vers la page du site en cours. |
|   | Cible personnalisée | L'URL doit être saisie dans le champ "URL cible".<br><br>**Cas 1 : URL d'une page Facebook** : <br>Incrémente le compteur de la page Facebook aimée.<br>**Cas 2 : URL d'une page ou d'un site quelconque** : <br>Crée sur le fil Facebook de l'internaute connecté un lien vers la page ou le site spécifié. |
| **Twitter** | Le domaine | Crée sur le fil Twitter de l'internaute connecté un message contenant un lien vers le domaine du site en cours. |
|   | La page | Crée sur le fil Twitter de l'internaute connecté un message contenant un lien vers la page du site en cours. |
|   | Cible personnalisée | L'URL doit être saisie dans le champ "URL cible".<br><br>Crée sur le fil Twitter de l'internaute connecté un message contenant un lien vers l'URL indiquée. |
| **LinkedIn** | Le domaine | Crée un post sur les nouvelles du profil LinkedIn connecté avec un lien sur le domaine.<br><br>Attention : Une page d'index par défaut doit exister sur le serveur d'hébergement. |
|   | La page | Crée un post sur les nouvelles du profil LinkedIn connecté avec un lien sur la page. |
|   | Cible personnalisée | Crée un post sur les nouvelles du profil LinkedIn connecté avec un lien personnalisé.<br><br>L'URL du lien à envoyer doit être saisie dans le champ "URL cible". |
| **Tumblr** | Le domaine | Crée sur le fil Tumblr de l'internaute connecté un message contenant un lien vers le domaine du site en cours. |
|   | La page | Crée sur le fil Tumblr de l'internaute connecté un message contenant un lien vers la page du site en cours. |
|   | Cible personnalisée | L'URL doit être saisie dans le champ "URL cible".<br><br>Crée sur le fil Tumblr de l'internaute connecté un message contenant un lien vers l'URL indiquée. |
| **WhatsApp** | La page | Crée un message WhatsApp contenant un lien vers la page du site en cours. |
|   | Cible personnalisée | L'URL doit être saisie dans le champ "URL cible".<br><br>Crée un message WhatsApp contenant un lien vers l'URL indiquée. |


Le champ "Destination" permet de sélectionner le mode d'affichage de la fenêtre de connexion et de saisie du message sur le réseau social paramétré.

## La destination de l'action
<a name="destination_action_ELTTEXTE000322"></a>
Si aucune destination n'est choisie pour l'action, l'action est réalisée dans la destination par défaut définie pour la page en cours.

Les destinations possibles d'une action effectuée par un champ Bouton sont :


|   |   |
| --- | --- |
| **Nouveau navigateur (_blank)** :<br>Affiche le résultat dans une nouvelle fenêtre du navigateur. | <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Dest_nouveau.gif)<br> |
| **Frame en cours (_self)** :<br>Affiche le résultat dans le même cadre. | <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Dest_FrameEnCours.gif)<br> |
| **Frame parent (_parent)** :<br>Affiche le résultat dans la frame d'origine (cas de framesets imbriqués). | <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Dest_FrameParent.gif)<br> |
| **Navigateur en cours (_top)** :<br>Affiche le résultat dans la page en cours du navigateur (sans frames). | <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Dest_NavEnCours.gif)<br> |
| **Autres frames du frameset** (si la page fait partie d'un frameset) :<br>Affiche le résultat dans une frame spécifique. |   |
| **Champ iFrame de la page** (si la page contient au moins un champ iFrame) :<br>Affiche le résultat dans un [champ iFrame](../WDChamp/1013270.md). |   |




## Image de fond d'une fenêtre multilingue
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000081"></a>
Dans certains cas, l'image de fond d'une fenêtre peut contenir du texte (par exemple le nom d'une application, un libellé spécifique, ...).

Si l'application est destinée à être utilisée dans plusieurs langues, il est nécessaire d'utiliser une image spécifique pour chaque langue.

WINDEV et WINDEV Mobile permettent de prendre en compte très simplement ces différentes images.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Gestion multilingue
<a name="gestion_multilingue_ELTTEXTE000105"></a>
Une image de fond différente peut être affichée selon la langue d'exécution du programme.

Pour définir une image de fond multilingue : 

1. Affichez la fenêtre de description de la fenêtre.

2. Affichez l'onglet "Image".

3. Cliquez sur l'icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Menu_Image_Editeur%20-%20HC%20N%B0001.gif) (à droite du nom de l'image) et sélectionnez "Multilingue".

4. Pour chaque langue définie dans votre fenêtre, choisissez l'image appropriée.
	**Attention** : Les caractéristiques d'affichage saisies dans cette fenêtre s'appliqueront à tous les fichiers image.

5. Validez. Le libellé "Valeur multilingue" apparaît dans le champ de définition des images de fond.

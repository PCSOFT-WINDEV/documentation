
## Plan de navigation
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000098"></a>
Le plan de navigation permet de mettre en place la structure et l'enchaînement des pages de votre site. Le plan de navigation est utilisé :

- par le champ [Chemin de navigation](../WDChamp/1013265.md).

- par le champ [Plan du site](../WDChamp/1013300.md). 






<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Construire le plan de navigation
<a name="construire_plan_navigation_ELTTEXTE000122"></a>
La mise en place du plan de navigation est nécessaire dès qu'un champ permettant de naviguer dans le site est mis en place (chemin de navigation ou plan du site).


![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Plan%20de%20navigation%20-%20HC%20N%B0001.gif)


Il est possible de :

- **Construire automatiquement le plan de navigation**à partir des enchaînements réalisés dans votre site (bouton "Construction automatique"). Si un plan de navigation existe déjà, il sera supprimé. WEBDEV se base sur l'enchaînement des pages décrit sous l'éditeur ou par programmation pour construire le plan du site.

- **Construire vous-même votre plan de navigation** : il suffit de sélectionner les pages à afficher et de les ajouter dans le plan de navigation par le bouton "Ajouter". Ensuite, vous pouvez organiser l'enchaînement des pages en effectuant des "Drag and drop" des différentes pages dans la zone "Plan de navigation".

- **Ajouter ou enlever des pages**du plan de navigation (boutons "Ajouter" et "Enlever").

- ** Afficher et modifier la description de chaque page**présente dans le plan de navigation : il suffit de double-cliquer sur la page présente dans la zone "Plan de navigation". Il est possible de définir :

	- le libellé utilisé dans le chemin de navigation ou dans le plan du site (onglet "Libellé"). Par défaut, le titre de la page est utilisé.

	- la bulle d'aide associée à la page et affichée lors du survol dans le chemin de navigation ou dans le plan du site (onglet "Bulle d'aide"). Par défaut, le titre de la page est utilisé.

	- la valeur des paramètres à utiliser pour afficher la page (onglet "Paramètres"). Cette option est très intéressante par exemple pour les pages AWP.







Cette opération doit être effectuée pour chaque page.

**Remarque** : Les liens présents dans le plan de navigation utilisant des URL avec paramètres seront automatiquement pris en compte dans le [fichier Sitemaps](../Editeurs/2030053.md).


## Champ Liste image lié à un fichier de données (ou une requête)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000100"></a>
Un champ Liste image peut être utilisé pour afficher le contenu d'un fichier de données (ou d'une requête).

Lorsqu'un champ Liste image est lié à un fichier de données (ou une requête), ce champ pourra être initialisé avec le contenu du fichier de données :

- soit automatiquement : Aucun code spécifique n'est nécessaire pour initialiser le champ Liste image. Les caractéristiques du parcours du fichier de données lié au champ sont décrites dans l'onglet "Contenu" de la description du champ.

- soit par programmation : Il est nécessaire de saisir le code de parcours du fichier de données lié au champ Liste image dans le code d'initialisation du champ. Cette méthode est déconseillée.




Le contenu de deux rubriques différentes est affiché dans le champ Liste image :

- une rubrique correspondant aux libellés (rubrique de type "Texte" par exemple).

- une rubrique correspondant aux images (rubrique de type "Son, image, binaire" par exemple).




**Remarque** : Lors de la création d'un champ Liste image, l'assistant propose de créer les différents types de listes image. Dans le cas d'un champ Liste image lié à un fichier de données ou une requête, l'assistant demande les différents renseignements nécessaires (fichier de données utilisé, rubriques utilisées, ...).

<a name="NOTE2_1"></a>


## Code d'initialisation d'un champ Liste image programmé
<a name="code_initialisation_champ_liste_image_programme_ELTTEXTE000124"></a>
Si vous utilisez un champ Liste image basé sur un fichier de données avec initialisation programmée, il est nécessaire de saisir le code de parcours du fichier de données dans le code d'initialisation du champ.

**Exemple** : Initialisation d'un champ Liste image avec les photographies du fichier de données VOYAGE.


```wl
// -- Code d'initialisation du champ Liste image
HLitPremier(VOYAGE, Photo)
TANTQUE PAS HEnDehors()
	ListeAjoute(LSI_PhotoVoyage, VOYAGE.Photo)
	HLitSuivant(VOYAGE, Photo)
FIN
```

## Vignettes HFSQL
<a name="vignettes_hfsql_ELTTEXTE000148"></a>
La rubrique liée aux images affichées dans le champ Liste image peut utiliser des vignettes HFSQL. Dans ce cas, l'affichage des images sera optimisé.

Pour plus de détails, consultez [Vignettes HFSQL](../WDLang4/3044348.md).


## Manipuler un chemin de navigation par programmation
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000099"></a>
Le champ Chemin de navigation peut être manipulé par programmation :

- grâce aux fonctions [CheminNavigationXXX](../WDLang2/3083021.md) du WLangage.

- grâce aux fonctions [CheminNavigationXXX (syntaxe préfixée)](../WDLang2/3083039.md) du WLangage.

- grâce [aux propriétés du WLangage.](../WDChamp/1013267.md)




**Rappel** : Un champ chemin de navigation peut être créé entièrement sous l'éditeur de pages. Il est également possible de le personnaliser par programmation grâce aux fonctions WLangage.





<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Connaître l'indice de l'élément cliqué dans le chemin de navigation
<a name="connaitre_indice_element_clique_dans_chemin_navigation_ELTTEXTE000123"></a>
Pour connaître l'indice de l'élément cliqué dans le chemin de navigation, il suffit d'utiliser la propriété [Valeur](../Proprietes/2510130.md) (par exemple dans le code de clic du champ Chemin de navigation). 

**Exemple** :



```wl
// -- Code de clic du champ SMP_CheminNavig
// Connaître l'indice de l'élément cliqué
Trace("Indice de l'élément cliqué : " + MoiMême.Valeur)
```

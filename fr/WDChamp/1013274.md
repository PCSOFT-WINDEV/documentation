## Présentation
<a name="presentation_ELTTEXTE000180"></a>
Pour optimiser le look de vos onglets, vous pouvez utiliser des images et des couleurs. Pour plus de détails, consultez [Images et onglets](../WDChamp/1013195.md). 

Mais vous pouvez également utiliser : 

- [la position des onglets](#NOTE2_1) : en bas, en haut, à gauche ou à droite. Cette position peut être combinée avec l'orientation du libellé de l'onglet. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) [le mode d'affichage des onglets](#NOTE3_1).

## Position des onglets
<a name="position_des_onglets_ELTTEXTE000204"></a>
Par défaut, lors de la création d'un champ Onglet, les volets d'onglets sont positionnés en haut du champ. 

Pour modifier cette position : 

1. Affichez la fenêtre de description du champ Onglet.  

2. Dans l'onglet "Détail", sélectionnez la position voulue : 

	- en haut, 

	- en bas, 

	- à gauche,

	- à droite. 




3. Validez la fenêtre de description du champ. 




**Remarque** : Si vous sélectionnez une position à gauche ou à droite, par défaut le libellé du volet reste horizontal. Pour afficher ce libellé verticalement, pensez à modifier l'orientation du libellé dans l'onglet "Style" de la description du champ : 

1. Affichez la fenêtre de description du champ Onglet.  

2. Dans l'onglet "Style, sélectionnez l'élément "Libellé du volet actif". 

3. Modifiez l'orientation de le police (0° par défaut). 

4. Répétez l'opération 3 pour l'élément "Libellé du volet inactif".  
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Volet_onglet_gauche.gif)

## Mode d'affichage des onglets
<a name="mode_affichage_des_onglets_ELTTEXTE000228"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 

### Les différents modes
<a name="les_differents_modes_ELTPARAGRAPHE000075"></a>Le champ Onglet propose différents modes d'affichage des onglets :  

- le **mode standard** (par défaut) : les volets d'onglets sont affichés sur une ligne et s'il n'y a plus de place, les onglets sont affichés sur une ligne supplémentaire :  
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=onglet_mode_std.gif)


- le **mode 3 volets avec glissement** : seuls 3 volets sont affichés, et lors du clic sur le dernier ou le premier volet, celui-ci passe au milieu : 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=onglet_mode_3.gif)


- le **mode volets invisibles** : tous les volets sont invisibles. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) le **mode standard avec boutons + / -** : les volets d'onglets sont affichés sur une ligne et s'il n'y a plus de place, des boutons apparaissent permettant à l'utilisateur de faire défiler les volets. Le look de ces boutons peut être modifié dans l'onglet "Style" de la description du champ (élément "Boutons suivant/précédent"). 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=onglet_mode_boutons.gif)





Pour définir le mode d'affichage des onglets : 

1. Affichez la fenêtre de description du champ Onglet.  

2. Dans l'onglet "Détail", sélectionnez le mode d'affichage voulu. 

3. Validez la fenêtre de description du champ.

## Décalage des volets d'onglet
<a name="decalage_des_volets_onglet_ELTTEXTE000258"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Quelque soit la position des volets d'onglets, il est possible de les décaler sous l'éditeur grâce à une poignée spécifique. 

Ainsi, si les volets d'onglet sont positionnés : 

- En haut ou en bas, il est possible de les décaler vers la droite. 

- A gauche ou à droite, il est possible de les décaler vers le bas. 


L'espace ainsi libéré peut être par exemple utilisé pour afficher un logo ou un bouton déroulant des options. 

**Pour décaler la position des volets d'onglet** : 

1. Sélectionnez le champ Onglet. ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Onglet_Decalage%20-%20HC%20N%B0001.gif)


2. Une poignée spécifique apparaît au milieu du bord du premier onglet. 

3. Sélectionnez cette poignée et déplacez-la. La zone située à côté du premier onglet s'agrandit et les différents onglets sont décalés (vers la droite ou vers le bas selon l'orientation des volets).  ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Onglet_Decalage%20-%20HC%20N%B0002.gif)


4. Les volets d'onglet ont été décalés. ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Onglet_Decalage%20-%20HC%20N%B0003.gif)


## Graphe de type "Secteur multiniveau"
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000156"></a>
Le graphe de type Secteur multiniveau est un graphe Secteur contenant plusieurs niveaux de données hiérarchiques.

L'intérêt de ce type de graphe est double :

- D'une part, il permet l'affichage concis et hiérarchisé d'un très grand nombre de données. 

- D'autre part, ce type de graphe est interactif : le clic sur une part permet de rentrer dans le détail des sous-niveaux de cette part.

<a name="NOTE2_1"></a>


## Caractéristiques d'un graphe de type Secteur multiniveau
<a name="caracteristiques_graphe_type_secteur_multiniveau_ELTTEXTE000180"></a><br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Graphe_Secteur_Niveau.gif)


Dans ce graphe :

**1.** **Titre du graphe.**
Les caractéristiques du titre du graphe sont saisies dans l'onglet "Détail" de la description du champ (bouton "Editer le titre du graphe").
Il est possible de définir le titre du graphe, sa position et son style.

**2.** **Taille du trou**
Un graphe de type Secteur multiniveau peut afficher un trou en son centre. Il est possible de définir la taille du trou dans l'onglet "Général" de la description du champ. 

**3.** **Portion du graphe**
A chaque niveau, les valeurs affichées représentent une portion. 

L'utilisateur final peut cliquer sur une portion du graphe pour voir le détail de cette part : la part cliquée devient le centre du graphe et autour sont représentés tous ses sous-niveaux. Et ainsi de suite, ... Un clic au centre du graphe remonte d'un niveau.

L'affichage du détail est automatique : il est géré par le champ.

Remarques : 

- Il est possible d'afficher les libellés des portions (étiquettes de catégories) perpendiculairement : utilisez la fonction  [grParamètre](../WDLang3/1000018337.md) avec la constante *grLibelléSMNPerpendiculaireProtion*.
	

- Pour définir l'angle de départ de la première part, utilisez la fonction [grSMNAngleDépart](../WDLang3/1000021915.md).

- Pour excentrer une part (ou toutes les parts), utilisez la fonction [grSMNPartExcentrée](../WDLang3/1000021916.md).




**4.** **Bulle d'aide**
La valeur des données survolées à la souris s'affiche dans une bulle.

**Autres caractéristiques** : 

- Par défaut, un dégradé radial de couleurs par niveau est géré automatiquement. Il est possible de définir la couleur d'une portion avec la fonction [grSMNCouleurPortion](../WDLang3/1000021288.md). 

- Pour limiter le nombre de niveaux affichés, utilisez la fonction [grParamètre](../WDLang3/1000018337.md) avec la constante *grSMNNiveauMax*. 

- Les fonctions WLangage spécifiques à ce type de graphe sont les suivantes : 


|   |   |
| --- | --- |
| [grSMNAjouteDonnée](../WDLang3/1000021285.md) | Ajoute une donnée à un graphe de type Secteur multiniveau. |
| [grSMNAngleDépart](../WDLang3/1000021915.md) | Indique l'angle de départ de la première part dans un graphe de type Secteur multiniveau. |
| [grSMNCouleurPortion](../WDLang3/1000021288.md) | Change la couleur d'une portion d'un graphe de type Secteur multiniveau. |
| [grSMNPartExcentrée](../WDLang3/1000021916.md) | Excentre une part ou toutes les parts d'un graphe de type Secteur multiniveau. |
| [grSMNPortionRacine](../WDLang3/1000021298.md) | Change ou précise le chemin de la racine dans un graphe de type Secteur multiniveau. |
| [grSMNRécupèreDonnée](../WDLang3/1000021287.md) | Récupère une valeur numérique dans un graphe de type Secteur multiniveau. |




Remarque : Ce type de graphe n'accepte pas de valeurs négatives.

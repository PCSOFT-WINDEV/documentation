


## Les graphes de type "Boursier"
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000094"></a>
Les graphes de type Boursier permettent de représenter les variations du cours d'une action. Trois types de graphes Boursier sont disponibles :

- **chandelier** : ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=chandelier.gif)


- **barcharts** : ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=barcharts.gif)


- **minmax** : ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=mini_maxi.gif)





<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000118"></a>
Pour créer un graphe boursier dans un champ Graphe :

1. Créez un champ Graphe : sous le volet "Création", dans le groupe "Champs graphiques", cliquez sur "Graphe".

2. Dans l'onglet "Séries" de la fenêtre de description du champ, créez quatre séries dans l'ordre suivant :

	- Série 1 : Valeurs minimales de la séance.

	- Série 2 : Valeurs maximales de la séance.

	- Série 3 : Valeurs en début de séance.

	- Série 4 : Valeurs en fin de séance.




3. La couleur attribuée à la première série sera celle utilisée pour le graphe Boursier. Les couleurs attribuées aux autres séries ne seront pas prises en compte.

4. Pour les graphes de type Chandelier japonais, il est possible de définir la couleur d'un chandelier haussier dans l'onglet "Style" de la fenêtre de description du champ.

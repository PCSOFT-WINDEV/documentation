
## Manipuler un champ Note repositionnable par programmation
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000172"></a>
Un champ Note repositionnable peut être manipulé par programmation. Il suffit d'utiliser directement la variable du champ Note repositionnable dans le code. 

Des fonctions spécifiques permettent également de manipuler des notes repositionnables. Pour plus de détails, consultez [Fonctions Notes repositionnables](../WDLang1/1000022398.md). 

Cette page d'aide détaille comment manipuler un champ Note repositionnable par programmation. 





<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Afficher une information dans un champ Note repositionnable
<a name="afficher_une_information_dans_champ_note_repositionnable_ELTTEXTE000196"></a>
Pour afficher une information dans un champ Note repositionnable, une simple affectation suffit.

Quelques exemples :

- affectation directe :
	
	```wl
	NOTE_NoteClient = "Client important"
	```


- affectation par variable :
	
	```wl
	NOTE_NoteClient = Note_Client
	```





**Remarque** : Si le champ est associé à une rubrique d'un fichier de données, l'affectation est automatiquement réalisée en utilisant la fonction [FichierVersEcran](../WDLang4/3044210.md).
Il est aussi possible d'utiliser la notation "NOTE_NoteClient = CLIENT.NOTE". Pour plus de détails, consultez [Lier un champ Note repositionnable à une rubrique](../WDChamp/1410087012.md).

**Attention** : Le champ Note redimensionnable ne possède pas d'ascenseur. Si le texte dépasse la taille de la note repositionnable, il sera tronqué. Il est possible de modifier la taille du champ (propriétés [Hauteur](../Proprietes/2510050.md) et [Largeur](../Proprietes/2510059.md)).

## Récupérer la valeur saisie dans un champ Note repositionnable
<a name="recuperer_valeur_saisie_dans_champ_note_repositionnable_ELTTEXTE000220"></a>
Pour récupérer le contenu d'un champ Note repositionnable :

- réalisez une simple affectation. 
	Par exemple :
	
	```wl
	ContenuActuelNote = NOTE_NoteClient
	```


- utilisez la propriété [Valeur](../Proprietes/2510130.md).




**Remarque** : Si le champ est associé à une rubrique d'un fichier de données, la récupération de la valeur saisie est automatiquement réalisée en utilisant la fonction [EcranVersFichier](../WDLang4/3044146.md).
Il est aussi possible d'utiliser la notation "CLIENT.Note = NOTE_NoteClient". Pour plus de détails, consultez [Lier un champ Note repositionnable à une rubrique](../WDChamp/1410087012.md).

## Forcer l'exportation ou l'importation de notes
<a name="forcer_exportation_importation_notes_ELTTEXTE000244"></a>
L'utilisateur peut importer ou exporter les notes (de la fenêtre ou de l'application) dans un fichier d'extension "wdnotes" grâce au menu contextuel du champ Note ou de la fenêtre. 

Il est possible de forcer l'export ou l'import des notes par programmation grâce à la fonction [FAAExécute](../WDLang1/1000022099.md) : 

- la constante *faaExporterNote* permet de forcer l'export des notes de la fenêtre spécifiée. L'utilisateur doit sélectionner le fichier contenant les notes. 

- la constante *faaImporterNote* permet de forcer l'import des notes. L'utilisateur doit sélectionner le fichier contenant les notes. 




**Attention** : l'export est réalisable uniquement si le contenu de la note a été modifié. 

Astuce : Pour importer et exporter toutes les notes repositionnables d'une fenêtre, utilisez les fonctions [NoteChargeTout](../WDLang1/1000022397.md) et [NoteSauveTout](../WDLang1/1000022396.md).

## Propriétés spécifiques aux champs Note repositionnable
<a name="proprietes_specifiques_aux_champs_note_repositionnable_ELTTEXTE000268"></a>
Les propriétés suivantes sont spécifiques à la gestion des caractéristiques d'un champ Note repositionnable par programmation.




|   |   |
| --- | --- |
| [Réduit](../Proprietes/1000022005.md) | La propriété **Réduit** permet de : <br><br>- savoir si une note repositionnable est réduite, <br><br>- réduire ou restaurer une note repositionnable.<br><br><br> |
| [RéduitAutomatique](../Proprietes/1000021963.md) | La propriété **RéduitAutomatique** permet de : <br><br>- Connaître le délai avant la réduction automatique du champ Note repositionnable.<br><br>- Définir le délai avant la réduction automatique du champ Note repositionnable. <br><br><br> |





Pour connaître la liste complète des propriétés du WLangage pouvant être utilisées avec un champ de type Note repositionnable, consultez [Propriétés associées au champ Note repositionnable](../WDChamp/1410087010.md).

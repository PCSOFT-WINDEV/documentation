
## Intégration d'un superchamp dans une fenêtre ou une page
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Récupération d'un superchamp
<a name="recuperation_superchamp_ELTTEXTE000134"></a>
Dans la majorité des cas, le superchamp est disponible dans un dictionnaire (fichier .Dic). Pour intégrer le superchamp dans votre fenêtre / page, il suffit de :

1. Afficher le [volet "Dictionnaire"](../Editeurs/2027014.md).

2. Ouvrir le dictionnaire (option "Ouvrir" du menu contextuel).

3. Réaliser un "Drag and Drop" (glisser / déplacer) du champ présent dans le dictionnaire vers votre fenêtre / page. Le superchamp est automatiquement intégré à votre fenêtre / page.


**Remarque** : Diverses copies d'un même superchamp peuvent se trouver dans une même fenêtre ou page. Seul le nom du superchamp est modifié. Les noms des éléments présents dans le superchamp ne sont pas modifiés. Les traitements associés aux champs sont indépendants.

<a name="NOTE2_1"></a>


## Programmation d'un superchamp
<a name="programmation_superchamp_ELTTEXTE000158"></a>
La programmation d'un superchamp varie en fonction des possibilités données par le concepteur du superchamp.

- En règle générale, le superchamp a une valeur. Cette valeur peut être obtenue soit directement en utilisant le nom du superchamp, soit en utilisant la propriété [Valeur](../Proprietes/2510130.md).

- Pour manipuler un champ du superchamp (par exemple pour connaître sa valeur, il faut utiliser la notation suivante :
	
	```txt
	<Nom du superchamp>.<Nom du champ>
	```


- Utilisez si nécessaire les [traitemements associés au superchamp](../WDChamp/1013082.md).




**Remarque** : Il est déconseillé de modifier les procédures associées au superchamp sans l'accord du concepteur.

## Propriétés spécifiques aux superchamps
<a name="proprietes_specifiques_aux_superchamps_ELTTEXTE000182"></a>
Les propriétés suivantes sont spécifiques à la gestion des caractéristiques d'un superchamp par programmation.


|   |   |
| --- | --- |
| [Hauteur](../Proprietes/2510050.md) | Permet de connaître et de modifier la hauteur du superchamp |
| [Largeur](../Proprietes/2510059.md) | Permet de connaître et de modifier la largeur du superchamp |
| [Nom](../Proprietes/2510082.md) | Permet de connaître le nom du superchamp |
| [Valeur](../Proprietes/2510130.md) | Permet de connaître et de modifier la valeur du superchamp |


Pour connaître la liste complète des propriétés du WLangage pouvant être utilisées avec un superchamp, consultez [Propriétés associées à un superchamp](../WDChamp/1013066.md).




## Champ Image dans une page
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000196"></a>
Cette page d'aide présente les caractéristiques d'un champ Image présent dans une page. 

Rappel : Pour créer un champ de type Image : 

1. Sous le volet "Création", dans le groupe "Champs usuels", déroulez "Image" et sélectionnez "Image".

2. Cliquez dans la page à la position où le champ doit être créé. Le champ apparaît en création.




**Pour afficher les caractéristiques du champ**, sélectionnez l'option "Description" dans le menu contextuel du champ.

<a name="NOTE2_1"></a>


## Options de la fenêtre de description
<a name="options_fenetre_description_ELTTEXTE000220"></a>


### Onglet "Général"
<a name="onglet_general_ELTPARAGRAPHE000037"></a>L'onglet "Général" permet de définir l'image affichée et ses caractéristiques. 

**Source de l'image** : 


- **Image**: L'image affichée par le champ est sélectionnée via le bouton ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Menu_Image_Editeur%20-%20HC%20N%B0001.gif)
, option "Parcourir". En fonction de l'image sélectionnée, un assistant apparaît, permettant de :

	- **compresser automatiquement l'image** (cas des images dont la taille dépasse les 20 Ko). Cette option permet également de convertir l'image à un des formats adapté aux pages Internet. Si cette option n'est pas choisie :

		- Si l'image utilise un format non adapté aux pages internet, l'image sera convertie à chaque génération de la page, ce qui peut ralentir cette génération.

		- Si l'image utilise un format adapté aux pages mais est de taille trop élevée, le temps d'affichage de la page pourra être ralenti.


 Si vous choisissez de réaliser la conversion immédiatement, il est possible de sélectionner le format de l'image et sa qualité. Il est également possible d'utiliser le format ***JPEG progressif***. Ce format affiche une image progressivement, pendant que le navigateur charge la page. 

- **convertir automatiquement l'image** à un des formats adapté aux pages Internet. Si cette option n'est pas choisie, l'image sera convertie à chaque génération de la page, ce qui peut ralentir cette génération. Si vous choisissez de réaliser la conversion immédiatement, il est possible de sélectionner le format de l'image et sa qualité.

- **définir l'emplacement de l'image** : il est possible de copier l'image dans le répertoire du site (répertoire _WEB) ce qui optimise le temps de génération de la page, ou bien de laisser l'image à son emplacement (option conseillée lors du partage du fichier entre plusieurs intervenants).




**Paramètres d'affichage**
**Mode d'affichage** : Pour plus de détails sur le mode d'affichage et de transparence de l'image, consultez [Choisir le mode d'affichage de l'image](../WDChamp/1013134.md).

**Action**
Il est également possible d'associer une action à l'image. Pour plus de détails, consultez [Choisir l'action associée à un champ](../WDChamp/1013165.md).




### Onglet "UI"
<a name="onglet_ELTPARAGRAPHE000081"></a>**Transitions - Effets - Défilement automatique**

- **Transition / Effet** : Lorsque la valeur du champ Image est modifiée, une transition peut être sélectionnée comme par exemple Fondu, Recouvrement ou Balayage pour rendre le changement d'image plus harmonieux. Pour plus de détails, consultez [Animer l'affichage une image](../WDChamp/9500117.md).

- **Activer le défilement automatique d'images** : Cette option est accessible uniquement pour les images de type "dynamiques" ou de type "depuis une base de données : chemin". Pour plus de détails, consultez [Image avec défilement automatique](../WDChamp/9500118.md).









### Onglet "Détail"
<a name="onglet_detail_ELTPARAGRAPHE000097"></a>L'onglet "Détail" permet de spécifier des options avancées. 

**Localiser l'image dans le répertoire _WEB en code navigateur** : 
Par défaut, cette option est cochée lors de création d'un champ Image. Dans ce cas, en code navigateur, l'image est automatiquement recherchée dans le répertoire _WEB du site. Aucun code spécifique n'est nécessaire.
Si cette option est décochée, le chemin complet de l'image doit être indiqué pour retrouver l'image. Il faut pour cela, utiliser la fonction [RépertoireWeb](../WDLang2/3012015.md).

**Protection de l'image**  : 
Si cette option est cochée, le menu contextuel de l'image ne pourra pas être affiché dans le navigateur. Toutes les manipulations de l'image disponibles au niveau du navigateur (enregistrement, impression, ...) ne pourront pas être réalisées. Cette option est également disponible dans les champs [Image cliquable](../WDChamp/1013140.md) et dans les champs [Vignette](../WDChamp/1013275.md).

**Zoom automatique** : 
WEBDEV propose le zoom automatique d'une image dans une popup. Pour plus de détails, consultez [Zoomer une image](../WDChamp/9500127.md). 

**Glisser / Déplacer** : 
Cette option permet de configurer le champ Image pour la gestion du Drag and Drop. Pour plus de détails, consultez [Drag and Drop automatique](../WDLang1/3030012.md). 

**Chargement différé** : 
L'option "Charger l'image lorsque le champ est à l'écran" permet de différer le chargement de l'image. En effet, quand un navigateur réalise le rendu d'une page, il charge depuis le serveur la page et ses images. Dans certains cas, ce temps de chargement peut s'avérer long. Cette option permet au navigateur de charger l'image uniquement lorsque celle-ci apparaît dans la partie visible de la page. Il est possible de spécifier également l'image d'attente, utilisée pendant le chargement différé de l'image. 

**Zoom et scroll aux doigts**
Cette option permet de gérer le multi-touch sur le navigateur pour les images. Pour plus de détails, consultez [Multi-touch dans un champ Image](../WDChamp/9500131.md).




## Le champ de saisie HTML : Barre de mise en forme automatique
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000143"></a>
Dans les champs de saisie de type HTML, une barre de mise en forme peut apparaître automatiquement dès l'entrée en saisie dans le champ. La gestion des options de formatage devient ainsi automatique.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000167"></a>


### Activer la barre de mise en forme pour les champs HTML
<a name="activer_barre_mise_forme_pour_les_champs_html_ELTPARAGRAPHE000018"></a>Pour activer la barre de mise en forme pour les champs HTML :

1. Affichez la description d'un champ de saisie HTML. Le champ de saisie est de type "Texte HTML". 

2. Dans l'onglet "Général", indiquez si le champ utilise une barre d'outils HTML : <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=HTML_ConfigBarreWB%20-%20HC%20N%B0002.gif)
Le bouton "Paramétrer la barre d'outils..." permet de sélectionner les options proposées par la barre d'outils. 

3. Validez.



<a name="NOTE2_2"></a>


### Fonctionnalité disponible en FAA
<a name="fonctionnalite_disponible_faa_ELTPARAGRAPHE000035"></a>Dans un champ de saisie HTML, la barre d'outils HTML est toujours visible ou s'affiche automatiquement (selon l'option sélectionnée dans l'onglet "Général" de la fenêtre de description du champ). 
<a name="NOTE2_3"></a>


### Police utilisée par défaut dans la barre d'outils HTML
<a name="police_utilisee_par_defaut_dans_barre_outils_html_ELTPARAGRAPHE000042"></a>Lors de l'exécution, la barre d'outils est initialisée avec la police, la taille et les attributs de police donnés dans la description du champ. Il est possible de modifier cette police par programmation grâce [aux propriétés de gestion des polices](../Proprietes/2515045.md). 

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Utilisation de la barre de mise en forme HTML
<a name="utilisation_barre_mise_forme_html_ELTTEXTE000203"></a>
La barre HTML par défaut est la suivante : <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=HTML_Barre_WB%20-%20HC%20N%B0003.gif)


Cette barre permet de choisir : 

- La police utilisée et sa taille. 
	Remarque : Ces options sont également disponibles en saisie libre.

- La mise en forme logique du texte (div, h1, etc.).

- Les caractéristiques de la police (gras, souligné, italique, barré). 

- La couleur de la police.

- La couleur du surlignement.

- Les caractéristiques d'alignement du texte. 

- Les puces et la numérotation. 

- Les retraits à droite et à gauche. 

- La mise en indice ou en exposant. 

- Les caractéristiques des liens. 

- Les caractéristiques des images. 




Il est également possible : 

- d'insérer des tableaux : <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=HTML_Barre_Tableau%20-%20HC%20N%B0004.gif)


- de choisir le style du tableau inséré. <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=HTML_Barre_Tableau%20-%20HC%20N%B0006.gif&type=thumb)


- d'afficher le code source HTML (![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=HTML_Barre_WB%20-%20HC%20N%B0003%201.gif)).  




**Remarque** : 
Les couleurs proposées par défaut sont celles utilisées par l'ambiance du site. Il suffit de cliquer sur la couleur voulue. La cohérence des couleurs utilisées dans le site est ainsi assurée.<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=HTML_Barre_WBCouleur%20-%20HC%20N%B0004.gif)


## Mire dans un graphe
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000180"></a>
Dans un graphe de type Courbe, qui n'a jamais eu du mal à lire la valeur des axes pour un point éloigné de ces axes ?

Grâce à la nouvelle mire de lecture proposée en standard sur tous les graphes, l'utilisateur final connaît avec précision la valeur pointée.

Tous les graphes avec axes bénéficient automatiquement de la mire de lecture.

L'utilisateur survole un point du graphe et la mire s'affiche !

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=WD%20-%20Graphe%20Mire.jpg)


Si la mire verticale coupe plusieurs courbes, elle affiche la valeur à chacune des intersections : c'est la mire multiple. 

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=WD%20-%20Graphe%20Mire%20multiple.jpg)


Dans le cas d'une mire multiple, l'entête de la bulle d'aide est personnalisable.

## Mise en place
<a name="mise_place_ELTTEXTE000204"></a>


### Mire dans un champ Graphe
<a name="mire_dans_champ_graphe_ELTPARAGRAPHE000042"></a>Par défaut, la mire est disponible sur tous les champs Graphe créés à partir de la version 20. 

Pour afficher une mire dans un champ Graphe : 

1. Affichez la fenêtre de description du champ Graphe.

2. Dans l'onglet "Détail" : 

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) dans la zone "Zoom et mire", cochez l'option "Activer la mire du graphe". 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Mire_Graphe_Editeur.gif)
L'option "Mire multiple (courbe et histo. empilés)" permet d'obtenir une mire multiple dans les graphes de type Courbe ou Histogramme. 




3. Validez la fenêtre de description du champ. 




**Remarques** : 

- La mire est disponible pour les types de graphes suivants : Aire, Bulle, Courbe, Histogramme, MinMax, Radar et Waterfall. 



<a name="NOTE2_2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 

### Mire dans les FAA (Fonctionnalités Automatiques de vos Applications)
<a name="mire_dans_les_faa_fonctionnalites_automatiques_vos_applications_ELTPARAGRAPHE000091"></a>Si la mire n'est pas activée dans un graphe, l'utilisateur peut l'activer grâce au menu contextuel du graphe (option "Activer la mire"). 
<a name="NOTE2_3"></a>


### Modifier la mire multiple par programmation
<a name="modifier_mire_multiple_par_programmation_ELTPARAGRAPHE000098"></a>Par défaut, la mire multiple affiche : 

- en entête, le nom de la catégorie. 

- le contenu des différentes bulles d'aide des points correspondant à la mire. 




Il est possible de modifier l'entête de la bulle d'aide de la mire grâce à la fonction [grBulle](../WDLang3/3042010.md) utilisée avec la constante *grFormatBulleEntêteMire*. 

Par exemple : 

```wl
grBulle(GRF_Graphe1, grFormatBulleEntêteMire, "Catégorie : [%CATEGORIE%]")
```

<a name="NOTE2_4"></a>


### Paramétrer les options de la mire par programmation
<a name="parametrer_les_options_mire_par_programmation_ELTPARAGRAPHE000116"></a>Par défaut, la mire possède les caractéristiques suivantes : 

- couleur basée sur la couleur du quadrillage du champ Graphe. 

- épaisseur : 1 pixel. 

- opacité : mire complètement opaque. 




Il est possible de modifier ces caractéristiques grâce aux propriétés suivantes :  



|   |   |
| --- | --- |
| [CouleurMire](../Proprietes/1000021505.md) | La propriété **CouleurMire** permet de connaître et de modifier la couleur de la mire d'un champ Graphe. |
| [EpaisseurMire](../Proprietes/1000021506.md) | La propriété **EpaisseurMire** permet de connaître et de modifier l'épaisseur de la mire d'un champ Graphe. |
| [OpacitéMire](../Proprietes/1000021517.md) | La propriété **OpacitéMire** permet de connaître et de modifier l'opacité de la mire d'un champ Graphe. |

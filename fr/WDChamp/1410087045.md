
## Manipuler un champ Conteneur natif par programmation
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000203"></a>
Le champ Conteneur Natif renvoie un handle/identifiant : 

- via la propriété [Valeur](../Proprietes/2510130.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) via la fonction [Handle](../WDLang1/3015005.md).




Ce handle peut ensuite être utilisé dans le code natif pour gérer le champ. 

L'événement "Création de XXX" associé au champ doit contenir le code de création de l'objet affiché dans le champ Conteneur natif.

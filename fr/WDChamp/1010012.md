


## Look des fenêtres : image et couleur de fond
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000194"></a>
Pour optimiser le look de vos écrans, WINDEV et WINDEV Mobile permettent d'utiliser :

- des images dans vos fenêtres. Ces images sont utilisées en image de fond de la fenêtre. Vos champs sont superposés sur cette image.

- des couleurs de fond. De nombreuses couleurs standard peuvent être utilisées.




![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Il est également possible de construire des couleurs de fond dégradées. Ces couleurs dégradées peuvent être utilisées : 

- en fond de fenêtre, 

- en couleur de fond des différents champs d'une fenêtre. 




**Remarque** : sur les fenêtres avec un gabarit détouré, il n'est pas possible d'utiliser la fonctionnalité Aéro Snap de Windows. Aéro Snap impose une fenêtre standard redimensionnable.

## Images de fond des fenêtres
<a name="images_fond_des_fenetres_ELTTEXTE000218"></a>


### Image de fond
<a name="image_fond_ELTPARAGRAPHE000030"></a>Les formats d'images supportés sous Windows sont :

- Bitmap (\*.BMP)

- Graphics Interchange Format (\*.gif)

- Joint Picture Experts Group (\*.jpg ; \*.jpeg)

- Kodak Photo CD (\*.pcd)

- PaintBrush (\*.pcx)

- Adobe Photoshop Format (\*.psd)

- TrueVision TARGA (\*.tga)

- Tagged Image File Format (\*.tif ; \*.tiff)
	Remarque : l'option "tiff mosaïque" de la norme tiff 6.0 n'est pas supportée. Dans ce cas, il est conseillé d'enregistrer l'image à la norme tiff 5.0. 

- Portable Network Graphics (\*.png)

- Windows MetaFiles (\*.emf ; \*.wmf)

- Icônes (\*.ico ; \*.icw)
	Remarques : 

	- Affichage HQ (Haute Qualité) disponible.

	- Gestion des planches au format BMP transparent (32 bits).




- Curseurs (\*.cur)

- Portable Document Format (\*.pdf)

- Scalable Vector Graphics (\*.svg)

- Images WINDEV (générées par l'éditeur d'images, \*.WDPIC).




Exemple :![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Fen_Imagefond.gif)


**Remarque** : Pour personnaliser vos fenêtres, vous pouvez également utiliser un gabarit. Un gabarit correspond à une interface prédéfinie. Pour plus de détails, consultez [Gabarit](../Editeurs/2016011.md).

### Configuration des images de fond
<a name="configuration_des_images_fond_ELTPARAGRAPHE000055"></a>La configuration est effectuée dans l'onglet "Image" de la description de la fenêtre.

Pour sélectionner l'image de fond, cliquez sur le bouton ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Menu_Image_Editeur%20-%20HC%20N%B0001.gif)
 du champ "Image de fond" : 

- L'option "Parcourir" permet de sélectionner un fichier image accessible depuis votre poste.

- L'option "Catalogue" permet de sélectionner une image du catalogue d'images.




**Remarque** : Il est également possible de réaliser directement un Drag and Drop du fichier image présent dans l'explorateur vers la fenêtre WINDEV ou WINDEV Mobile.

## Couleur de fond
<a name="couleur_fond_ELTTEXTE000279"></a>


### Sélection de la couleur de fond
<a name="selection_couleur_fond_ELTPARAGRAPHE000113"></a>Pour sélectionner la couleur de fond d'une fenêtre :

1. Affichez la fenêtre de description.

2. Dans l'onglet "Style", ouvrez la liste "Couleur de fond". Une liste des différentes couleurs disponibles apparaît : 

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Couleurs disponibles : 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Fen_couleur_fond.gif)





3. Sélectionnez la couleur de fond voulue.

4. Validez la fenêtre de description.



<a name="NOTE3_2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 

### Couleur de fond dégradée
<a name="couleur_fond_degradee_ELTPARAGRAPHE000136"></a>Dans la fenêtre de sélection des couleurs de fond, plusieurs couleurs de fond dégradées sont proposées par défaut. L'option "Autres dégradés" permet d'afficher la fenêtre de paramétrage du dégradé.

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Fen_degrade.gif)


Dans cette fenêtre, il est possible de :

- Sélectionner une couleur dégradée proposée par défaut.

- Modifier une des couleurs dégradées proposée par défaut : il est possible de paramétrer les couleurs, le nombre de couleurs pris en compte par le dégradé et l'angle du dégradé. Il suffit de laisser libre cours à votre esprit créatif.




**Remarque** : Les couleurs dégradées peuvent également être utilisées en couleur de fond des champs : il suffit de modifier la couleur de fond du cadre extérieur du champ dans l'onglet "Style" de la fenêtre de description.

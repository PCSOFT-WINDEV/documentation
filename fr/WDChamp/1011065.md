
## Impression multipage
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000134"></a>
Un état peut parfois être très large, d'une largeur supérieure à une feuille de papier. C'est le cas essentiellement des états Tableau, Planning ou Diagramme de Gantt, ...

L'impression multipage permet d'imprimer un état sur plusieurs pages (2 ou plus) qui pourront être mise bord à bord. 

Cette fonctionnalité permet de rendre lisibles de grands tableaux ou des plannings. 

Sous l'éditeur, l'état affiche les différentes pages côte à côte : 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Etat_Multipage_Editeur.gif&type=thumb)


En exécution, les pages sont toutes visualisées dans le visualisateur de rapports et peuvent être imprimées : 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Etat_Multipage_Go.gif&type=thumb)




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000158"></a>
Pour imprimer un état en mode multipage, vous pouvez : 

- soit choisir l'option "Imprimer sur X pages" lors de la création de l'état. 

- soit configurer le format d'impression de l'état dans ses caractéristiques. 



<a name="NOTE2_2"></a>


### Création d'un état multipage
<a name="creation_etat_multipage_ELTPARAGRAPHE000035"></a>A la fin de l'assistant de création d'un état, si l'état est trop large pour rentrer dans le format spécifié, l'assistant propose les options suivantes : 

- Imprimer l'état sur plusieurs pages. Dans ce cas, un état multipage est automatiquement créé. 

- Utiliser le mode paysage. 

- Réduire l'état par rapport à l'original. Dans ce cas, selon le pourcentage de réduction choisi, l'état imprimé pourra devenir illisible. 



<a name="NOTE2_3"></a>


### Modification d'un état existant
<a name="modification_etat_existant_ELTPARAGRAPHE000046"></a>Il est possible de transformer un état existant en état multipage. 

Pour changer le format de l'état : 

1. Affichez la fenêtre de description de l'état (option "Description de l'état" du menu contextuel). 

2. Sélectionnez l'onglet "Format". 

3. Dans la zone "Multipage en largeur", indiquez le nombre de pages à utiliser en largeur : 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Etat_Multipage_Format.gif&type=thumb)


4. Validez. 

5. L'état apparaît sous l'éditeur avec le nombre de pages spécifié. Vous pouvez déplacer et agrandir si nécessaire les différents champs de votre état. 

6. Configurez la gestion des blocs (notamment "Entête de page" et "Pied de page") pour le mode multipage : 

	- Affichez la fenêtre de description de l'état (option "Description de l'état" du menu contextuel). 

	- Dans l'onglet "Blocs", définissez les options : les blocs peuvent être : 

		- étirés en largeur sur l'ensemble des pages.

		- répétés sur chaque page.  




- Validez. 


Remarque : Pour un état de type Tableau, le bloc d'entête est obligatoirement étiré.

<a name="NOTE3_1"></a>


## Impression de l'état multipage
<a name="impression_etat_multipage_ELTTEXTE000194"></a>
Lors de l'impression d'un état multipage, les différentes pages sont automatiquement imprimées. Cette option est prise en compte uniquement : 

- dans le visualisateur de rapports, 

- lors d'une impression sur l'imprimante, 

- lors d'une impression au format PDF. 


L'impression des pages est réalisée de gauche à droite puis vers le bas.

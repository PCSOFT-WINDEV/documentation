## Associer un champ à un volet d'onglet
<a name="associer_champ_volet_onglet_ELTTEXTE000125"></a>
Pour associer un champ à un volet d'onglet : 

- Méthode 1 : Positionnez le champ sur le volet d'onglet voulu. Le champ est automatiquement associé au volet en cours.

- Méthode 2 : 

	- Affichez le menu contextuel du champ à manipuler. 

	- Sélectionnez l'option "Associer .. à un onglet". 

	- Dans la fenêtre qui s'affiche, sélectionnez le volet d'onglet associé au champ. 

	- Validez. 







**Remarques** : 

- Si le champ doit être associé à un volet d'onglet, sans être positionné sur le volet d'onglet, choisissez la seconde méthode. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Un champ peut être associé à plusieurs volets d'onglet. Il suffit d'utiliser la seconde méthode et de sélectionner dans la fenêtre l'ensemble des volets d'onglet voulus. <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=NV_Affecter_onglet.gif)


- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Cette page d'aide concerne les volets d'onglet statiques uniquement.

## Ne pas associer un champ à un volet d'onglet
<a name="pas_associer_champ_volet_onglet_ELTTEXTE000149"></a>
Un champ positionné sur un volet d'onglet n'est pas nécessairement associé à ce volet d'onglet. C'est le cas par exemple, si le champ doit apparaître sur le volet d'onglet quel que soit l'onglet sélectionné.

Pour ne pas associer un champ à un volet d'onglet : 

1. Sélectionnez l'option "Associer .. à un onglet" du menu contextuel du champ.

2. Sélectionnez "Aucun".

## Fonctions et propriétés WLangage spécifiques à la gestion des onglets
<a name="fonctions_proprietes_wlangage_specifiques_gestion_des_onglets_ELTTEXTE000173"></a>




### Nom du champ Onglet associé à un champ
<a name="nom_champ_onglet_associe_champ_ELTPARAGRAPHE000053"></a>Pour connaître le nom du champ Onglet associé à un champ, utilisez la fonction [OngletDuChamp](../WDLang1/3025028.md). 

Exemple : 

```wl
// SAI_Nom1 est sur le champ ONG_OngletInfo
Info(OngletDuChamp(SAI_Nom1))       // Renvoie "ONG_OngletInfo"
```



![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 

### Volets d'onglet associés à un champ
<a name="volets_onglet_associes_champ_ELTPARAGRAPHE000069"></a>Pour connaître et modifier le ou les volet(s) d'onglet associé(s) à un champ, utilisez la propriété [Volet](../Proprietes/1000021098.md). 

Exemple : 


```wl
// Associer le bouton Valider aux volets 1, 2 et 4
BTN_Valider.Volet = "1,2,4"

// Connaître les volets associés à un bouton
SI BTN_Valider.Volet [=] "2" ALORS Info("Le champ est associé au volet 2")
```

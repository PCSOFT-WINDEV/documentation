## Présentation du bloc Corps
<a name="presentation_bloc_corps_ELTTEXTE000080"></a>
Le bloc **Corps** correspond au bloc principal de l'état. Ce bloc est obligatoirement présent dans chaque état. Ce bloc est imprimé autant de fois qu'il y a d'enregistrements à imprimer.

Pour les états libres, il est possible de répéter le bloc **Corps** autant de fois que nécessaire pour remplir la page (option "Répété jusqu'en bas de page" cochée dans l'onglet "UI" des caractéristiques du bloc).


**Remarque** : Le bloc **Corps** ne peut pas être supprimé. Il est possible de rendre ce bloc invisible (dans la fenêtre de description du bloc ou par la propriété [Visible](../Proprietes/2510138.md)).

Pour plus de détails sur les différentes options disponibles sur ce type de bloc, consultez [Description des blocs d'un état](../WDChamp/1011049.md).

## Présentation
<a name="presentation_ELTTEXTE000112"></a>
L'affichage d'un nouveau frameset dans un site WEBDEV peut être réalisé selon plusieurs méthodes :

- [directement depuis la description du champ qui doit ouvrir le frameset (bouton, lien, menu, image cliquable, ...)](#NOTE2_1).

- [par programmation en WLangage](#NOTE3_1).






<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Affichage d'un frameset depuis la description d'un champ
<a name="affichage_frameset_depuis_description_champ_ELTTEXTE000136"></a>
Lors de la description d'un bouton, d'un lien, d'une image cliquable, il est possible de définir directement l'action effectuée et la destination de cette action.

Pour ouvrir directement un frameset "xxx" à partir d'un bouton, lien, image, ... :

1. Affichez la description de l'élément (option "Description" du menu contextuel de l'élément)

2. Sélectionnez l'action à effectuer : "Afficher la page xxx". Cette page doit correspondre au frameset.

3. Sélectionnez la destination de l'action : page en cours, navigateur en cours, ...

4. Validez.

## Affichage d'un frameset par programmation (en WLangage)
<a name="affichage_frameset_par_programmation_wlangage_ELTTEXTE000160"></a>
Le WLangage propose plusieurs fonctions permettant d'ouvrir un frameset.



|   |   |
| --- | --- |
| [FramesetActualise](../WDLang2/3058014.md) | Actualise un frameset affiché dans le navigateur de l'internaute à partir du contexte présent sur le serveur. |
| [FramesetAffiche](../WDLang2/3058006.md) | Affiche un frameset WEBDEV dans le navigateur de l'internaute. |
| [FramesetUtilise](../WDLang2/3058012.md) | Affiche un frameset WEBDEV dans le navigateur de l'internaute et ferme tous les contextes de pages et de framesets en cours. |





Lors de l'ouverture du frameset, il est possible de passer des paramètres :

- au frameset. Il suffit d'indiquer ces paramètres dans une des fonctions d'ouverture du frameset.

- à une des pages du frameset. Il suffit d'ouvrir le contexte de la page concernée en indiquant les paramètres à passer (fonction [ContexteOuvre](../WDLang2/3058004.md)). Cette fonction doit être utilisée avant l'ouverture du frameset.




 
**Remarque** : Par défaut, lors de l'ouverture d'une page en programmation, la destination choisie est :

- soit la destination définie dans la description de l'objet lançant l'ouverture de la page (bouton, lien, ...).

- soit la destination définie pour la page en cours.

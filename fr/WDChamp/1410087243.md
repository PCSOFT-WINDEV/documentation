
## Manipuler un champ Menu contextuel par programmation
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000139"></a>
Pour manipuler un champ Menu contextuel par programmation, WEBDEV propose différentes fonctions et propriétés WLangage.

Il est également possible de manipuler le menu ou ses options directement par programmation : il suffit d'utiliser la notation "&lt;Nom de l'option&gt;" si l'option de menu est manipulée depuis la page.







## Manipuler le champ Menu contextuel par programmation
<a name="manipuler_champ_menu_contextuel_par_programmation_ELTTEXTE000163"></a>


### Associer un menu contextuel à un champ ou une page 
<a name="associer_menu_contextuel_champ_une_page_ELTPARAGRAPHE000018"></a>L'association d'un menu contextuel à un champ ou une page peut être effectué : 

- Sous l'éditeur de pages (via l'option "Menu contextuel" de l'onglet "UI" de la fenêtre de description de l'élément). 

- Par programmation : 

	- Grâce à la propriété [MenuContextuel](../Proprietes/2510077.md). 

	- Grâce à la fonction [OuvreMenuContextuel](../WDLang1/3025029.md) (code navigateur uniquement). 







Remarque : la fonction [ChampContextuel](../WDLang1/3025012.md) permet de connaître le nom du champ qui a ouvert le menu contextuel en cours. 






### Fonctions WLangage de manipulation des menus et de leurs options
<a name="fonctions_wlangage_manipulation_des_menus_leurs_options_ELTPARAGRAPHE000042"></a>Les fonctions permettant de manipuler un champ Menu contextuel sont les suivantes : 



|   |   |
| --- | --- |
| [MenuAjouteMenu](../WDLang1/1000017266.md) | Ajoute un nouveau menu dans un menu existant (déroulant ou contextuel) d'une fenêtre ou d'une page. |
| [MenuAjouteOption](../WDLang1/1000017262.md) | Ajoute une nouvelle option de menu à la suite de la dernière option d'un menu (déroulant ou contextuel). |
| [MenuAjouteOptionURL](../WDLang1/1000018529.md) | Ajoute une nouvelle option de menu à la suite de la dernière option d'un menu (déroulant ou contextuel). Cette option de menu permet d'afficher la page correspondant à l'URL indiquée. |
| [MenuAjoutePopup](../WDLang1/1000020583.md) | Transforme une option de menu d'une page pour que cette option ouvre une popup. |
| [MenuAjouteSéparateur](../WDLang1/1000017270.md) | Ajoute un nouveau séparateur dans un menu (menu déroulant ou contextuel). |
| [MenuSupprime](../WDLang1/3038058.md) | Supprime un menu ou une option de menu. |








### Propriétés WLangage
<a name="proprietes_wlangage_ELTPARAGRAPHE000049"></a>Les propriétés disponibles sur les options de menu (menus déroulants ou contextuels) sont les suivantes : 


|   |   |
| --- | --- |
| [Cochée](../Proprietes/2510019.md) | Permet de gérer une marque![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=MenuCtxMarque.gif)devant l'option de menu. |
| [Libellé](../Proprietes/2510053.md) | Permet de connaître et de modifier le libellé d'une option de menu. |
| [Nom](../Proprietes/2510082.md) | Permet de connaître le nom d'une option de menu. |
| [Sélectionnée](../Proprietes/2510112.md) | Permet de savoir si l'option de menu est sélectionnée.<br><br> |
| [Type](../Proprietes/2510131.md) | Permet de savoir si l'élément manipulé est une option de menu. |
| [Visible](../Proprietes/2510138.md) | Permet de savoir si une option de menu est visible et de rendre visible / invisible une option de menu. |





Pour connaître la liste complète des propriétés du WLangage pouvant être utilisées avec un champ de type Menu contextuel, consultez [Propriétés associées au champ Menu contextuel](../WDChamp/1410087223.md).

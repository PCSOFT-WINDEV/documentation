


## Fenêtre iPhone : assistant de création de fenêtres
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000190"></a>
L'assistant de création de fenêtres iPhone permet de créer en quelques clics une fenêtre contenant tous les éléments de base d'une fenêtre pour une application iPhone. 

Ces différents éléments peuvent être ajoutés à tout moment dans la fenêtre. 

Il est également possible de configurer différentes options dans la fenêtre de description de la fenêtre. 

A la fin de l'assistant, la fenêtre est créée sous l'éditeur avec les caractéristiques sélectionnées. Sous l'éditeur, en plus de la fenêtre, il est possible de visualiser : <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Fen%EAtre_iphone_Editeur%20-%20HC%20N%B0001.gif&type=thumb)


- la barre système,

- les zones correspondant au mode portrait ou paysage (matérialisées par des tirets), 

- la place de la barre d'accueil (à partir de iPhone X). Il est conseillé de ne placer aucun champ cliquable (bouton, ...) à cet emplacement.

## Les différentes étapes de l'assistant
<a name="les_differentes_etapes_assistant_ELTTEXTE000214"></a>


### Orientation de la fenêtre
<a name="orientation_fenetre_ELTPARAGRAPHE000031"></a>Plusieurs options sont possibles : <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=IOS_Ass_Orientation.gif)
Ces options peuvent être modifiées sous l'éditeur, dans l'onglet "Détail" de la fenêtre de description de la fenêtre (option "Orientation"). 

Il est également possible de spécifier si des ascenseurs automatiques doivent être affichés. Les ascenseurs seront automatiquement affichés si la fenêtre a une taille supérieure à la résolution de l'écran de l'appareil utilisé. <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=IOS_FenDetail.gif)



### Contenu de la fenêtre
<a name="contenu_fenetre_ELTPARAGRAPHE000041"></a><br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=IOS_Ass_Contenu.gif)
Vous pouvez créer une fenêtre : 

- avec **barre de navigation**. Cette barre permet à l'utilisateur par exemple de fermer une fenêtre grâce à un bouton "Fermer" ou "Retour".
	Pour ajouter une barre de navigation dans une fenêtre : sous le volet "Fenêtre", dans le groupe "Barres et menus", cliquez sur "Autres barres" puis sélectionnez "Pour compatibilité" et "Barre de navigation manuelle (Navigation Bar iOS)".

- avec une **zone répétée ou une zone multiligne**. Ce type de fenêtre permet par exemple de paramétrer une ou plusieurs options, de sélectionner une option, etc.

- avec une **Tab bar (barre de thème)**. En règle générale, une "Tab bar" permet d'accéder aux différentes fenêtres de l'application.
	Pour ajouter une barre de thème dans une fenêtre : sous le volet "Fenêtre", dans le groupe "Barres et menus", déroulez "Autres barres" puis sélectionnez "Pour compatibilité" et "Barre de thème manuelle (Tab Bar iOS)".

- avec une **barre d'outils**. En règle générale, la barre d'outils permet d'effectuer des actions sur la fenêtre en cours. 
	Pour ajouter une barre d'outils dans une fenêtre : sous le volet "Fenêtre", dans le groupe "Barres et menus", déroulez "Autres barres" puis sélectionnez "Pour compatibilité" et "Barre d'outils manuelle (ToolBar iOS)".




**Différence entre une barre de thème et une barre d'outils** : 

- Une barre d'outils propose des actions sur la vue courante.

- Une barre de thèmes correspond à des onglets.





### Configuration de la zone répétée ou multiligne
<a name="configuration_zone_repetee_multiligne_ELTPARAGRAPHE000099"></a>Si vous avez choisi de créer une fenêtre avec une zone répétée ou une zone multiligne, les différents plans de l'assistant permettent de configurer cet élément. 

- Plusieurs types de zones répétées prédéfinies ou de zones multilignes sont disponibles : <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=IOS_Ass_TypeZR.gif)<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=IOS_ass_typeZM.gif)


- Le plan suivant permet de paramétrer les éléments affichés dans la zone répétée ou dans la zone multiligne. <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=IOS_ass_ParamZR.gif)


- Dans le cas d'une zone répétée, il est possible de choisir le type de remplissage de la zone répétée (programmation, liaison fichier, liaison variable).




A la validation de l'assistant, la zone répétée ou la zone multiligne choisie est automatiquement créée dans la fenêtre. 

Vous pouvez saisir le code permettant de la remplir et de la faire fonctionner. 

**Dans la cas d'une zone répétée**, les principaux événements WLangage à modifier sont les suivants :

- Code d'initialisation du champ Zone répétée : cet événement permet par exemple de remplir la zone répétée. 

- Code de sélection d'une ligne : cet événement permet de gérer les actions effectuées lorsque l'utilisateur clique sur la flèche standard. 

- Code de clic de l'image IMG_Disclosure : cet événement permet de gérer le clic sur la puce personnalisée. 




Pour plus de détails, consultez : 

- Champ [Zone répétée](../WDChamp/1013099.md). 

- Champ [Zone multiligne](../WDChamp/1000024001.md). 





### Barre de thème ou barre d'outils
<a name="barre_theme_barre_outils_ELTPARAGRAPHE000135"></a>Si vous avez choisi cette option, spécifiez les paramètres de la barre : 

- Couleur,

- Style,

- Boutons affichés.




A la fin de l'assistant, la fenêtre est créée avec la barre spécifiée. Il ne reste plus qu'à saisir le code des boutons.


## Manipuler un champ Lien par programmation (Page)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000101"></a>
WEBDEV permet de manipuler un champ Lien directement par programmation. Il suffit d'utiliser directement la variable du champ Lien dans le code.

La variable du champ Lien correspond au nom du champ. Un champ Lien est toujours de type Texte.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Récupérer et modifier le libellé (code serveur)
<a name="recuperer_modifier_libelle_code_serveur_ELTTEXTE000125"></a>
Pour manipuler le libellé d'un champ Lien, il suffit d'utiliser la propriété [Libellé](../Proprietes/2510053.md).

Par exemple, la ligne de code suivante modifie le libellé du champ LIEN_Email :


```wl
LIEN_Email.Libellé = "Ecrivez-moi directement"
```


**Remarque** : Si l'option "..Valeur représente l'URL du lien" est décochée (onglet "Détail" de la description du lien), le libellé du lien peut également être connu par la propriété [Valeur](../Proprietes/2510130.md).



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Récupérer et modifier l'URL du lien (cas d'un lien de type "mailto" en code serveur)
<a name="recuperer_modifier_url_lien_cas_lien_type_mailto_code_serveur_ELTTEXTE000149"></a>
Pour manipuler l'URL d'un champ Lien de type "Mailto", il suffit d'utiliser la propriété [URL](../Proprietes/2510132.md). 

Par exemple, la ligne de code suivante modifie l'adresse email du champ LIEN_Email :


```wl
LIEN_Email.URL = "adresse@laposte.net"
```


**Remarque** : Si l'option "..Valeur représente l'URL du lien" est cochée (onglet "Détail" de la description du lien), l'URL du lien peut également être connue par la propriété [Valeur](../Proprietes/2510130.md).

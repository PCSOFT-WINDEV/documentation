


## Le champ Organigramme
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000206"></a>


### Champ d'une fenêtre
<a name="champ_une_fenetre_ELTPARAGRAPHE000011"></a>Le champ Organigramme permet de représenter des éléments hiérarchisés entre eux. 
Par exemple : 

- les collaborateurs d'une société en fonction de leur hiérarchie. 

- les différentes filiales d'un groupe avec leurs dépendances. 

- les différentes étapes d'un processus de fabrication, ...




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Créer un champ de type Organigramme
<a name="creer_champ_type_organigramme_ELTTEXTE000230"></a>
Pour créer un champ Organigramme : 

1. Sous le volet "Création", dans le groupe "Champs graphiques", cliquez sur "Organigramme".

2. Cliquez dans la fenêtre à la position où le champ doit être créé. L'assistant de création du champ Organigramme se lance. Sélectionnez les différentes caractéristiques du champ et validez. Le champ est automatiquement créé.


Remarque : Les dimensions du champ créé sont optimisées pour occuper l'espace disponible à la position indiquée. Si la taille du champ ne vous convient pas, utilisez la combinaison de touches Ctrl + Z : le champ retrouvera sa taille par défaut.

Pour afficher les caractéristiques du champ, sélectionnez l'option "Description" dans le menu contextuel du champ.

## Description d'un Organigramme
<a name="description_organigramme_ELTTEXTE000258"></a>


### Les options du champ
<a name="les_options_champ_ELTPARAGRAPHE000053"></a>Le champ Organigramme peut être paramétré directement sous l'éditeur grâce à la fenêtre de description (onglet "Général").

Les paramètres de l'organigramme sont les suivants : 

- **Orientation** : L'organigramme peut être affiché horizontalement (de gauche à droite ou de droite à gauche) ou verticalement (de haut en bas ou de bas en haut). 
	<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=orga_orientation.gif)


- **Type de flèche** : La flèche peut pointer vers les éléments fils, pointer vers l'élément parent ou encore pointer dans les 2 sens. 

- **Fenêtre interne pour les éléments** : Fenêtre interne permettant de personnaliser l'apparence des éléments de l'organigramme (voir ci-dessous). 

- **Largeur et hauteur minimale des éléments** : taille minimale des éléments du champ Organigramme. 

- **Passer sur plusieurs lignes à partir de**. Si le nombre d'éléments est supérieur au nombre indiqué, les éléments seront affichés sur plusieurs lignes (ou colonnes selon l'orientation du champ).



<a name="NOTE3_2"></a>


### Personnaliser la zone de l'élément : couleur, traduction, etc.
<a name="personnaliser_zone_element_couleur_traduction_etc_ELTPARAGRAPHE000082"></a>Dans le champ Organigramme, chaque élément est affiché dans une zone spécifique. Il est possible de traduire et de personnaliser cette zone : couleur du titre, couleur de fond de l'élément, ...

Pour personnaliser l'affichage des éléments :

1. Affichez la fenêtre de description du champ Organigramme. 

2. Dans l'onglet "Général", sélectionnez la fenêtre interne utilisée pour les éléments : 

	- "Aucune" : l'affichage par défaut sera utilisé. 

	- "Fenêtre prédéfinie" : 
			Si cette option est sélectionnée, le composant interne WDFAA est immédiatement ajouté dans le projet en cours. La fenêtre interne "FI_WinDevVueOrgaElement" est automatiquement associée à l'option "Fenêtre interne pour les éléments". 
			Remarque : Si le composant WDFAA existe déjà, seule la fenêtre est ajoutée au composant.  

	- une fenêtre interne existante de votre projet. 




3. Validez si nécessaire la fenêtre de description du champ. 




**Remarques** : 

- La fenêtre interne personnalisée est disponible dans le volet "Explorateur de projet", dans le dossier "Composants internes". Elle peut être modifiée dans l'éditeur de fenêtres.

- Si vous utilisez une fenêtre interne de votre projet, cette fenêtre interne doit gérer l'affichage des caractéristiques des éléments dans les différents champs de cette fenêtre interne. Consultez à titre d'exemple le code de déclaration des globales de la fenêtre interne prédéfinie. 




Pour plus de détails sur les personnalisations possibles, consultez [Personnaliser l'affichage d'un champ Organigramme](../WDChamp/1000019822.md).

## Imprimer un organigramme
<a name="imprimer_organigramme_ELTTEXTE000294"></a>
Pour imprimer un champ Organigramme : 

- **l'utilisateur final** dispose d'une option de menu contextuel pour enregistrer l'organigramme sous forme d'image (Fonctionnalité Automatique de vos Applications). Il pourra ainsi imprimer cette image. 

- **par programmation**, la fonction [OrgaSauveImage](../WDLang1/1000019972.md) permet de créer l'image de l'organigramme dans un fichier image. L'application peut ensuite par exemple imprimer l'image via un état, l'envoyer par mail, ...

## Fonctionnalités Automatiques de vos Applications (FAA) pour un champ Organigramme
<a name="fonctionnalites_automatiques_vos_applications_faa_pour_champ_organigramme_ELTTEXTE000318"></a>
Le champ Organigramme dispose de plu­sieurs FAA (Fonctionnalités Automatiques de l'Application). 

L'utilisateur peut en effet : 

- zoomer en utilisant le raccourci "Ctrl + roulette souris". 

- se déplacer dans l'organigramme avec la souris. 

- utiliser les "range slider" disponibles sur les côtés du champ pour zoomer et se déplacer. 

- enregistrer l'organigramme dans une image (pour l'imprimer par exemple) via le menu contextuel du champ.

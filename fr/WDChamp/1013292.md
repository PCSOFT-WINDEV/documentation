


## Manipuler un graphe sous l'éditeur
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000086"></a>
Sous l'éditeur, un champ Graphe est constitué de différentes parties :

- ***le titre du graphe***. 
	Cette zone apparaît uniquement si un titre a été défini pour le graphe. 
	Pour définir le titre, quel que soit l'éditeur, sélectionnez l'option "Editer le titre du graphe" dans l'onglet "Détail". 

- ***la légende du graphe***. 
	Cette zone apparaît uniquement si une légende a été définie pour le graphe. 
	Pour définir une légende, quel que soit l'éditeur, sélectionnez l'option "Editer la légende du graphe" dans l'onglet "Détail". 

- ***le graphe***.




Chaque partie peut être paramétrée séparément.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000110"></a>
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Graphe_Zones.gif)
Les trois zones d'un champ Graphe sont manipulables séparément. Chaque zone peut être positionnée manuellement : il suffit de sélectionner la zone voulue avec la souris et de modifier l'emplacement de cette zone.

Par exemple :
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Graphe_Zones1.gif)
Le double-clic sur chaque zone permet d'afficher directement la description de cette zone. Ainsi, pour afficher la description de la légende, il suffit de double-cliquer sur la légende sous l'éditeur.

**Remarque** : Le titre et la légende peuvent avoir un positionnement automatique ou manuel. Dès que la zone correspondante est modifiée ou déplacée à l'aide de la souris, son positionnement devient manuel. Il est possible de retrouver le positionnement initial en sélectionnant pour la zone un positionnement automatique.

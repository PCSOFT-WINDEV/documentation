


## Table de positionnement
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000122"></a>
Le positionnement des éléments d'une page est simplifié par l'utilisation possible d'une table de positionnement (table HTML). Vous pouvez ainsi visualiser la table HTML qui sera générée et la simplifier en alignant vos différents champs.

Cette table de positionnement est également très pratique pour créer des "bas de page".
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=TablePositionnement.gif)
La table de positionnement est matérialisée par les pointillés.

L'édition en mode "Zoning" permet de ne plus utiliser les tables de positionnement. Pour plus de détails, consultez [Edition en mode zoning](../WDChamp/9000130.md). 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000146"></a>


### Afficher ou masquer la table de positionnement
<a name="afficher_masquer_table_positionnement_ELTPARAGRAPHE000029"></a>Lorsque vous êtes dans l'éditeur de pages, pour afficher ou masquer la table de positionnement, vous pouvez :

- sous le volet "Affichage", dans le groupe "Aide à l'édition", cocher ou décocher "Table de positionnement".

- utiliser la combinaison de touches Ctrl + U.



<a name="NOTE2_2"></a>


### Ancrer un champ
<a name="ancrer_champ_ELTPARAGRAPHE000051"></a>L'ancrage des champs est simplifié grâce à la génération en mode zoning. Pour plus de détails, consultez [Ancrage des champs dans une page](../WDChamp/9000131.md).

Par défaut, les champs WEBDEV ont leur position réelle précisée dans la table de positionnement.

Par exemple : ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=TableP_Pixels.gif)


Pour ancrer un champ (le plus souvent vers le bas, ou vers la droite), il suffit d'utiliser un pourcentage dans la table de positionnement pour les coordonnées du champ. Pour cela, il suffit de faire un clic droit sur la coordonnée et de choisir "Extensible". Vous pouvez alors saisir la valeur voulue (en pourcentage) : ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=TableP_Pourcent.gif)

<a name="NOTE2_3"></a>


### Remarques
<a name="remarques_ELTPARAGRAPHE000071"></a>

- Lorsque la table de positionnement est affichée sous l'éditeur, pour optimiser la visualisation de la table de positionnement, la couleur de fond des cellules n'est plus affichée, ainsi que les images.

- La combinaison de touches Ctrl + I permet de visualiser les zones extensibles. La table de positionnement n'est pas dessinée et les images sont affichées.

- Les tables de positionnement ne s'appliquent pas aux tables AJAX.

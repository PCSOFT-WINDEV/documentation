
## Manipuler un champ Popup par programmation (syntaxe préfixée)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000063"></a>
La popup étant un champ de la page, il est possible d'accéder : 

- depuis la popup aux champs de la page. 

- depuis la page aux champs de la popup. Par exemple : 
	
	```wl
	SAI_NomSaisie = POPUP_MaPopup.SAI_Info_a_saisir
	```



Les variables de la page sont accessibles dans les événements associés au champ Popup. Par exemple, le code suivant permet d'afficher une popup Fiche sur l'élément sélectionné dans un champ Zone répétée : 

```wl
// Code de clic navigateur sur le champ Zone répétée ZR_Produits
POPUP_FICHE.IMG_PRODUIT = RépertoireWeb + "ui/vide.gif"
POPUP_FICHE.Anime(CELL_PAGE, animDéfilementGauche, 25)
IDPRODUIT = ATT_IDPRODUIT
ExécuteTraitement(POPUP_FICHE.ZONE_ENTETE_POPUP.BTN_Charger, trtClic)
```




## Manipuler le champ Popup par programmation
<a name="manipuler_champ_popup_par_programmation_ELTTEXTE000102"></a>


### Fonctions disponibles
<a name="fonctions_disponibles_ELTPARAGRAPHE000125"></a>Plusieurs fonctions sont spécifiques à la gestion des champs Popup : 



|   |   |
| --- | --- |
| [&lt;Popup&gt;.Affiche](../WDLang2/1000021654.md) | Affiche une popup dans la page avec un effet de GFI (Grisage des Fenêtres Inaccessibles). |
| [&lt;Popup&gt;.Anime](../WDLang2/1000021655.md) | Affiche une popup dans une cellule de la page. |
| [&lt;Popup&gt;.Ferme](../WDLang2/1000021656.md) | Masque une popup affichée dans la page via la fonction [&lt;Popup&gt;.Affiche](../WDLang2/1000021654.md). |



Remarques : 

- Les fonctions [&lt;Cellule&gt;.AfficheDialogue](../WDLang2/1000021652.md) et [&lt;Cellule&gt;.FermeDialogue](../WDLang2/1000021653.md) sont compatibles avec les champs Popup. En cas de permutation des champs Cellules en champs Popup, le code existant reste compatible et fonctionnel.  

- La fonction [&lt;Menu&gt;.AjoutePopup](../WDLang1/1000020602.md) permet de transformer une option de menu d'une page pour que cette option ouvre un champ Popup.

- Il est possible d'épingler un champ Popup grâce à la fonction [&lt;Popup&gt;.Affiche](../WDLang2/1000021654.md).

## Présentation
<a name="presentation_ELTTEXTE000139"></a>
Par défaut, un frameset permet d'afficher plusieurs pages dans une même fenêtre du navigateur. Il est possible qu'une des pages affichées dans le frameset soient elle-même un frameset : on parle de "Framesets imbriqués".

Les framesets imbriqués peuvent être utilisés par exemple pour réaliser des mises en page originales de vos sites.

## Comment le faire ?
<a name="comment_faire_ELTTEXTE000163"></a>


### Imbriquer des framesets
<a name="imbriquer_des_framesets_ELTPARAGRAPHE000030"></a>Pour imbriquer des framesets :

1. Affichez le frameset principal sous l'éditeur de pages.

2. Sélectionnez la frame où le frameset doit être affiché. Le contour de la frame sélectionnée apparaît en rouge.

3. Sous le volet "Page", dans le groupe "Edition", déroulez "Frame" et sélectionnez "En Frame support".

4. Affichez la description de la frame support (option "Description" du menu contextuel de la frame support).

5. Sélectionnez la page du projet correspondant au frameset à afficher. 
	Attention : Le frameset ne sera pas affiché sous l'éditeur, mais uniquement en exécution du projet ou de la page.

6. L'état initial du frameset est automatiquement mis à jour : le nouvel état initial du frameset prend en compte la frame support.
	Attention : La plupart des navigateurs ne supportent pas l'imbrication récursive de framesets. Il n'est pas possible de :

	- imbriquer le frameset N dans le frameset N.

	- imbriquer un frameset contenant déjà le frameset N dans le frameset N.








<a name="NOTE2_2"></a>


### Transformer une frame support en frame "normale"
<a name="transformer_une_frame_support_frame_normale_ELTPARAGRAPHE000062"></a>Pour transformer une frame support en frame "normale" : 

1. Affichez le frameset principal sous l'éditeur de pages.

2. Sélectionnez la frame support. Le contour de la frame sélectionnée apparaît en rouge.

3. Sous le volet "Page", dans le groupe "Edition", déroulez "Frame" et sélectionnez "En Frame normale".


Attention : Cette manipulation ne change pas l'état initial du frameset en cours : l'état initial contient toujours la frame support. Pour définir l'état initial du frameset, sous le volet "Page", dans le groupe "Edition", déroulez "Frame" et sélectionnez "Définir l'état initial".

## Présentation du champ d'affichage formaté
<a name="presentation_champ_affichage_formate_ELTTEXTE000162"></a>


### Champ d'une page dynamique
<a name="champ_une_page_dynamique_ELTPARAGRAPHE000011"></a>Ce champ permet par exemple d'afficher des dates, des heures, des valeurs monétaires, en respectant le format d'affichage correspondant.

Un champ d'affichage formaté est donc un libellé permettant d'afficher les types suivants :

- texte,

- numérique,

- date,

- heure,

- monétaire, ...




Les champs d'affichage formaté sont principalement utilisés dans les zones répétées.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Création d'un champ d'affichage formaté
<a name="creation_champ_affichage_formate_ELTTEXTE000186"></a>
Pour créer un champ d'affichage formaté :

1. Sous le volet "Création", dans le groupe "Champs usuels", déroulez "Texte" et sélectionnez "Champ d'affichage formaté".
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Libelle_wb%20-%20HC%20N%B0001.gif)


2. Cliquez dans la page à la position où le champ doit être créé. Le champ apparaît en création.




Pour afficher les caractéristiques du champ, sélectionnez l'option "Description" dans le menu contextuel du champ.


### Configurer un champ de type d'affichage formaté
<a name="configurer_champ_type_affichage_formate_ELTPARAGRAPHE000051"></a>Pour configurer un champ de type d'affichage formaté :

1. Affichez la fenêtre de description du champ (option "Description" dans le menu contextuel du champ).

2. Dans l'onglet "Général", spécifiez : 

	- le type des données affichées. 

	- le masque d'affichage utilisé (option "Masque de saisie", utilisé uniquement pour l'affichage dans le cas du champ d'affichage formaté). 

	- le format de programmation si nécessaire (option "Valeur retournée"). Le format de programmation permet de définir le format manipulé par le programme (format des rubriques affichées dans le champ par exemple).

## Notes
<a name="notes_ELTTEXTE000216"></a>


### Remarques
<a name="remarques_ELTPARAGRAPHE000067"></a>

- **Pour les champs d'affichage formaté de type Date**, il est possible de sélectionner le masque d'affichage "Durée relative". Ce format permet d'exprimer la durée écoulée (ou à venir) entre la date du jour et la date affichée dans le champ. Ce format est exprimé en langage naturel. Les différentes formulations utilisées sont configurables dans la description du projet pour la langue en cours : 

	1. Sous le volet "Projet", dans le groupe "Projet", cliquez sur "Description". 

	2. Dans l'onglet "Langues" : 

		- sélectionnez la langue à configurer.

		- sélectionnez l'onglet "Date". 




3. Cliquez sur le bouton "Dates et Heures en langage naturel...". 

4. Définissez les libellés personnalisés à utiliser si nécessaire. Ces libellés seront utilisés : 

	- pour le résultat de la fonction [DateVersChaîne](../WDLang1/3027025.md). 

	- pour les champs utilisant le masque d'affichage "Durée relative". 

- Un champ d'affichage formaté peut être relié ou non à une rubrique d'un fichier de données. Le champ permet alors de visualiser le contenu d'une rubrique des enregistrements du fichier de données. 

- Le contenu d'un champ d'affichage formaté peut être défini uniquement en programmation.

### Astuce : Afficher la date de mise à jour d'un site
<a name="astuce_afficher_date_mise_jour_site_ELTPARAGRAPHE000108"></a>Pour afficher la date de mise à jour d'un site :

1. Créez un champ d'affichage formaté de type Date. 

2. Saisissez le code suivant dans le traitement d'initialisation du champ :
	
	```wl
	MoiMême = fDate(ComplèteRep(fRepExe()) + "<Nom de la bibliothèque>.WDL", "", fModification)
	```

## Présentation du champ Ligne
<a name="presentation_champ_ligne_ELTTEXTE000120"></a>
Un champ Ligne permet d'afficher une ligne dans une page.

Le style, l'épaisseur et la couleur du trait sont entièrement paramétrables.

**Pour créer un champ de type Ligne** :

1. Sous le volet "Création", dans le groupe "Intégration", cliquez sur "Ligne".

2. Cliquez dans la page à la position où le champ doit être créé. Le champ apparaît en création.




Pour afficher les caractéristiques du champ, sélectionnez l'option "Description" dans le menu contextuel du champ.

Remarque : Par défaut, une ligne horizontale est créée. Pour créer une ligne verticale, sous le volet "Création", dans le groupe "Intégration", cliquez sur "Ligne" et sélectionnez le type de ligne voulu.

## Evénement géré par défaut
<a name="evenement_gere_par_defaut_ELTTEXTE000144"></a>
WEBDEV gère par défaut l'événement suivant :

| Evénement | Condition d'exécution |
| --- | --- |
| Initialisation (Code serveur) | Exécuté à l'ouverture de la page. |


**Remarque** : Dans une page statique, seuls les codes navigateur sont disponibles. Le champ n'est associé à aucun code navigateur.

## Les différentes sources de données
<a name="les_differentes_sources_donnees_ELTTEXTE000155"></a>
WINDEV, WEBDEV et WINDEV Mobile permettent de créer des états grâce à l'éditeur d'états. Les états peuvent être créés à partir de données issues :

- [d'un fichier de données](../WDChamp/1011052.md).

- [du résultat d'une requête](../WDChamp/1011013.md).

- [d'une vue sur un fichier de données HFSQL](../WDChamp/1011064.md).

- [d'une zone mémoire](../WDChamp/1011061.md).

- [d'un champ Table présent dans une fenêtre ou une page](../WDChamp/1011012.md).

- [d'un champ Zone répétée présent dans une fenêtre ou une page](../WDChamp/1011093.md).

- [d'une variable](../WDChamp/1011058.md). 

- [d'un champ Table hiérarchique présent dans une fenêtre ou une page](../WDChamp/1011082.md).

- [d'un champ Agenda présent dans une fenêtre ou une page](../WDChamp/1011081.md). 

- [d'un champ Planning présent dans une fenêtre ou une page](../WDChamp/1011080.md). 

- [d'un fichier de type texte](../WDChamp/1011011.md).

- [d'une source de données programmée](../WDChamp/1011034.md).




L'éditeur d'états permet également de créer des états associés à aucune source de données.|   |   |   |

| ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ER.png)<br> | Important | Sous [le logiciel Etats & Requêtes](../Presentation/3088004.md), il n'est pas possible de créer :<br><br>- des états sur une vue HFSQL. <br><br>- des états sur une zone mémoire. <br><br>- des états sur un champ Table. <br><br>- des états sur un champ Zone répétée. <br><br>- des états sur un champ Table hiérarchique. <br><br>- des états sur un champ Agenda. <br><br>- des états sur un champ Planning. <br><br>- des états sur une source de données programmée. <br><br><br><br><br>Pour plus de détails, consultez [Limitations du logiciel Etats & Requêtes](../Presentation/3088005.md). |

## Notes
<a name="notes_ELTTEXTE000197"></a>
Si vous sélectionnez une des sources de données ci-dessus, l'éditeur d'états se charge entièrement de la programmation du parcours des enregistrements à afficher dans l'état.

Pour programmer et sélectionner vous-même les enregistrements à afficher dans l'état, choisissez un [état sur une source programmée](../WDChamp/1011034.md).

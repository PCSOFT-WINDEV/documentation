
## Exemple Flexbox : ancrage proportionnel sur les champs
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000088"></a>
Le champ Flexbox permet de mettre en place des ancrages proportionnels sur les champs.

Avec le champ Flexbox, il est possible de définir une largeur proportionnelle à appliquer champ par champ lorsque la page s'agrandit.

Dans l'exemple suivant, un champ Zone répétée contient 4 champs de saisie.
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Flexbox_Exemple2%20-%20HC%20N%B0005.gif&type=thumb)
Quand le navigateur est agrandi en largeur, les 2 premières zones de saisie doivent rester en largeur fixe et l'espace supplémentaire doit être distribué ainsi : 20 % pour la 3ème zone (libellé complet), 80 % pour la 4ème zone (descriptif). 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Flexbox_Exemple2%20-%20HC%20N%B0004.gif&type=thumb)


Ces ancrages peuvent être définis avec un champ Flexbox.

Remarque : Cette page est basée sur l'exemple WW_Flexbox.





<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000112"></a>
Pour obtenir un ancrage spécifique des champs de saisie composant la ligne d'un champ Zone répétée : 

1. Créer un champ Flexbox. Ce champ Flexbox est présent dans la ligne du champ Zone répétée et contient les différents champ de saisie affichés par le champ Zone répétée.
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Flexbox_Exemple2%20-%20HC%20N%B0001.gif&type=thumb)


2. Dans l'onglet "Général" de la fenêtre de description du champ Flexbox, les 4 champs de saisie sont affichés. Pour obtenir un agrandissement des zones de saisie, il suffit de définir l'option "flex-grow" pour le troisième et le quatrième champ de saisie. Dans notre cas : 

	- pour le troisième champ de saisie, l'option "flex-grow" a pour valeur 20. 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Flexbox_Exemple2%20-%20HC%20N%B0003.gif&type=thumb)


	- pour le quatrième champ de saisie, l'option "flex-grow" a pour valeur 80. 

3. Dans l'onglet "Détail", aucune modification ne doit être effectuée.


## Propriétés associées au champ Panneau dockable
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000069"></a>
Un champ Panneau dockable peut être manipulé en WLangage dans une fenêtre avec les propriétés suivantes.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Propriétés disponibles dans une fenêtre

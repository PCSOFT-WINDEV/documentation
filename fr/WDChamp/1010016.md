


## Les groupes de champs
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000244"></a>
Un groupe de champs est un ensemble de champs regroupés de façon à modifier (ou consulter) leurs caractéristiques (par exemple leur état de visibilité) en une seule opération, par programmation. Une option de menu peut également être associée à un groupe de champs.

Sous l'éditeur de fenêtres, de pages ou sous l'éditeur d'états, tous les champs de tous les groupes sont affichés : certains champs peuvent donc se superposer.

Un même champ peut être associé à plusieurs groupes.

**Exemple** : Pour rendre plusieurs champs invisibles, il suffit de rendre le groupe invisible. Dans le cas d'une fiche d'état civil, certains champs sont relatifs au sexe masculin alors que d'autres sont relatifs au sexe féminin. Le regroupement des champs relatifs à chaque sexe permet de les rendre visibles ou invisibles selon le sexe saisi.

**Remarque** : Dans le cas des fenêtres MDI, il est possible d'associer une fenêtre MDI à un groupe. Pour plus de détails, consultez [Gestion des options de menu dans les fenêtres MDI](../WDChamp/1010015.md).

## Créer un groupe de champs
<a name="creer_groupe_champs_ELTTEXTE000268"></a>
Pour créer un groupe de champs : 

1. Editez les groupes de champs : 

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Sous le volet "Fenêtre", dans le groupe "Edition", déroulez "Groupes" et sélectionnez "Editer". 

	- Sous l'éditeur d'états, sous le volet "Modification", dans le groupe "Groupes", déroulez "Groupes" et sélectionnez "Editer".




2. Cliquez sur le bouton "Nouveau" (ou "+" dans un état). Un nouveau groupe est créé.

3. Modifiez le nom du groupe si nécessaire.

4. Validez.

## Visualiser les champs d'un groupe
<a name="visualiser_les_champs_groupe_ELTTEXTE000292"></a>
Pour visualiser la liste des champs d'un groupe : 

1. Editez les groupes de champs : 

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Sous le volet "Fenêtre", dans le groupe "Edition", déroulez "Groupes" et sélectionnez "Editer". 

	- Sous l'éditeur d'états, sous le volet "Modification", dans le groupe "Groupes", déroulez "Groupes" et sélectionnez "Editer".




2. Sélectionnez le groupe de champs voulu.

3. La liste des champs du groupe sélectionné s'affiche dans la liste de droite.

<a name="NOTE4_1"></a>


## Affecter un champ à un groupe
<a name="affecter_champ_groupe_ELTTEXTE000316"></a>
Pour affecter un champ à un groupe : 

- soit renseignez le nom du groupe lors de la description du champ (onglet "Détail" ou onglet "UI"). 

- soit sélectionnez le ou les champs et sélectionnez l'option "Groupe .. Associer la sélection" du menu contextuel. Choisissez le nom du groupe ou créez un nouveau groupe.

## Modifier le nom d'un groupe
<a name="modifier_nom_groupe_ELTTEXTE000340"></a>
Pour modifier le nom d'un groupe : 

1. Editez les groupes de champs : 

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Sous le volet "Fenêtre", dans le groupe "Edition", déroulez "Groupes" et sélectionnez "Editer". 

	- Sous l'éditeur d'états, sous le volet "Modification", dans le groupe "Groupes", déroulez "Groupes" et sélectionnez "Editer". 




2. Cliquez sur le nom du groupe, saisissez le nouveau nom du groupe et validez.

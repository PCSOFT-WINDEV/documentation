
## Informations sur un champ d'une fenêtre ou d'une page
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000102"></a>
L'éditeur de fenêtres et l'éditeur de pages permettent d'obtenir rapidement des renseignements sur les champs manipulés. Vous pouvez :

- visualiser des informations lors du survol d'un champ.

- visualiser et modifier des informations lors de la sélection d'un champ.






<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Informations affichées lors du survol d'un champ
<a name="informations_affichees_lors_survol_champ_ELTTEXTE000126"></a>
Lors du survol d'un champ : 

- le champ survolé est mis en évidence via un liseré léger (les coins sont affichés en rouge). 

- Une bulle d'aide apparaît. Cette bulle d'aide affiche :

	- le nom du champ survolé ("BTN_Afficher_le_TreeMap" dans l'exemple ci-dessous).

	- la position (en pixels) du champ survolé ("&lt;640, 129&gt;" dans l'exemple ci-dessous).

	- la taille (en pixels) du champ survolé ("&lt;124x24&gt;" dans l'exemple ci-dessous).

	- l'ancrage du champ survolé, si le champ est ancré ("Largeur" dans l'exemple ci-dessous).







Par exemple : ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=BulleAideChamp.gif)


**Remarque** : Si le champ survolé est invisible à l'ouverture de la fenêtre, la chaîne "(Invisible)" apparaît dans la bulle d'aide. 



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Informations affichées et modifiables lors de la sélection d'un champ
<a name="informations_affichees_modifiables_lors_selection_champ_ELTTEXTE000150"></a>
Lors de la sélection d'un champ, les informations suivantes sont affichées et modifiables dans la barre de messages de WINDEV, WEBDEV ou WINDEV Mobile :

- le nom du champ sélectionné ("LIB_AVERTISSEMENT" dans l'exemple ci-dessous).

- la position (en pixels) du champ sélectionné ("15, 222" dans l'exemple ci-dessous).

- la taille (en pixels) du champ sélectionné ("319x58" dans l'exemple ci-dessous).

- le plan auquel le champ sélectionné appartient ("Plan 0" dans l'exemple ci-dessous).

- le zoom (100% dans l'exemple ci-dessous). 




Par exemple : ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=InfoChampBarreMessage2.gif)

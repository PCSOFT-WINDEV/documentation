## Présentation du champ Onglet
<a name="presentation_champ_onglet_ELTTEXTE000224"></a>
Le champ Onglet permet d'afficher uniquement certains champs en même temps. Lorsqu'un volet de l'onglet est actif :

- les champs associés à ce volet s'affichent dans la fenêtre ou dans la page.

- les champs associés à un volet différent deviennent invisibles,

- les champs associés à aucun volet conservent leur état d'affichage.



<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Champ%20onglet.jpg)








## Création d'un champ Onglet
<a name="creation_champ_onglet_ELTTEXTE000248"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Pour créer un champ de type Onglet :

1. Sous le volet "Création", dans le groupe "Conteneurs", déroulez "Onglet et associés" et sélectionnez "Onglet".

2. Cliquez dans la fenêtre à la position où le champ doit être créé. Le champ apparaît en création.




Remarque : Les dimensions du champ créé sont optimisées pour occuper l'espace disponible à la position indiquée. Si la taille du champ ne vous convient pas, utilisez la combinaison de touches Ctrl + Z : le champ retrouvera sa taille par défaut.

Pour afficher les caractéristiques du champ, sélectionnez l'option "Description" dans le menu contextuel du champ. 

La fenêtre de description du champ permet notamment de : 

- paramétrer les options des onglets (libellé, icône, ...). 

- [nommer les onglets](../WDChamp/1013196.md). 

- sélectionner [le mode d'affichage des onglets](../WDChamp/1013274.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) utiliser des [onglets dynamiques](../WDChamp/1013299.md).

## Vocabulaire associé au champ Onglet
<a name="vocabulaire_associe_champ_onglet_ELTTEXTE000282"></a>
Pour manipuler simplement un champ Onglet, il est nécessaire de connaître le vocabulaire suivant :

| Nom | Définition |
| --- | --- |
| Onglet | Bouton du volet d'onglet permettant de rendre actif un volet d'onglet. L'onglet peut contenir une image ou un libellé. Le libellé peut être composé de plusieurs lignes séparées par des RC. |
| Volet d'onglet | Zone permettant de positionner les champs associés à l'onglet. |
| ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Onglet dynamique | ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Volet d'onglet affichant une fenêtre interne. Permet de réaliser une interface proche de celle des navigateurs Internet :<br><br>- les onglets et leurs volets peuvent être "sortis" de la fenêtre principale.<br><br>- les onglets et leurs volets peuvent être réorganisés.<br><br>- il est possible de fermer certains onglets et d'en ouvrir de nouveaux.<br><br><br>Pour plus de détails, consultez [Onglets dynamiques](../WDChamp/1013299.md). |
| ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Onglet statique | ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Volet d'onglet classique, par opposition aux volets dynamiques. |

## Limites
<a name="limites_ELTTEXTE000306"></a>
![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) 

### Limites en Java
<a name="limites_java_ELTPARAGRAPHE000164"></a>En Java, les caractéristiques suivantes ne sont pas gérées :

- image associée au libellé des onglets.

- image des volets d'onglet.

- numéro d'aide associé à l'onglet.

- position des onglets. Par défaut, l'onglet est positionné en haut du volet.

- mode d'affichage des images. 

- onglets dynamiques.

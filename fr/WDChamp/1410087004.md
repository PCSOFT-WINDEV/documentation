
## Manipuler un champ Fil par programmation
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000083"></a>
WINDEV permet de manipuler un champ Fil directement par programmation. Il suffit d'utiliser directement la variable du champ Fil dans le code.

Pour manipuler un champ Fil par programmation, WINDEV propose les différentes fonctions et propriétés. 

Cette page d'aide détaille comment manipuler un champ Fil par programmation. 





## Manipuler le champ Fil par programmation
<a name="manipuler_champ_fil_par_programmation_ELTTEXTE000107"></a>


### Les fonctions WLangage
<a name="les_fonctions_wlangage_ELTPARAGRAPHE000018"></a>Les différentes fonctions WLangage permettant de manipuler un champ Fil sont les suivantes : 



|   |   |
| --- | --- |
| [PointDAttacheAjoute](../WDLang1/1000021902.md) | Définit un point d'attache pour une des deux extrémités d'un champ Fil. |
| [PointDAttacheSupprimeTout](../WDLang1/1000021903.md) | Supprime tous les points d'attache d'un champ. |








### Propriétés spécifiques à la gestion d'un champ Fil
<a name="proprietes_specifiques_gestion_champ_fil_ELTPARAGRAPHE000026"></a>Les propriétés suivantes sont spécifiques à la gestion des caractéristiques d'un champ Fil par programmation : 



|   |   |
| --- | --- |
| [ChampArrivée](../Proprietes/1000021878.md) | La propriété **ChampArrivée** permet de connaître ou de modifier le champ d'arrivée d'un champ Fil. |
| [ChampDépart](../Proprietes/1000021877.md) | La propriété **ChampDépart** permet de connaître ou de modifier le champ de départ d'un champ Fil. |
| [FormeExtrémitéArrivée](../Proprietes/1000021890.md) | La propriété **FormeExtrémitéArrivée** permet de connaître et de modifier le type de forme utilisé pour l'extrémité d'arrivée d'un champ Fil. |
| [FormeExtrémitéDépart](../Proprietes/1000021889.md) | La propriété **FormeExtrémitéDépart** permet de connaître et de modifier le type de forme utilisé pour l'extrémité de départ d'un champ Fil. |
| [FormeExtrémitéTaille](../Proprietes/1000021960.md) | La propriété **FormeExtrémitéTaille** permet de connaître et de modifier la taille de la forme à l'extrémité de départ et d'arrivée d'un champ Fil. |
| [NbMaxSegment](../Proprietes/1000021906.md) | La propriété **NbMaxSegment** permet de connaître et de modifier le nombre maximum de segments de droite pour dessiner un champ Fil. |
| [OriginePointDePassage](../Proprietes/1000022161.md) | La propriété **OriginePointDePassage** permet de connaître et de définir l'origine des points dont les coordonnées sont spécifiées dans la propriété [PointDePassage](../Proprietes/1000021957.md). |
| [PointDAttacheArrivée](../Proprietes/1000021908.md) | La propriété **PointDAttacheArrivée** permet de connaître et de modifier la position du point d'attache d'arrivée d'un champ Fil. |
| [PointDAttacheDépart](../Proprietes/1000021907.md) | La propriété **PointDAttacheDépart** permet de connaître et de modifier la position du point d'attache de départ d'un champ Fil. |
| [PointDePassage](../Proprietes/1000021957.md) | La propriété **PointDePassage** permet de connaître et de modifier les points de passage obligatoires pour un champ Fil. |





Pour connaître la liste complète des propriétés du WLangage pouvant être utilisées avec un champ de type Fil, consultez [Propriétés associées au champ Fil](../WDChamp/1410087003.md).

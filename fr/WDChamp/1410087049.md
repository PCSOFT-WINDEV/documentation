
## Manipuler un champ Disposition par programmation
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000144"></a>
Un champ Disposition peut être manipulé par programmation. Il est également possible de manipuler par programmation : 

- les champs présents dans le champ Disposition, 

- les cellules du champ Disposition. 




Cette page d'aide détaille les différentes manipulations possibles. 





## Gérer la visibilité par programmation
<a name="gerer_visibilite_par_programmation_ELTTEXTE000168"></a>


### Rendre dynamiquement invisible un champ
<a name="rendre_dynamiquement_invisible_champ_ELTPARAGRAPHE000020"></a>Pour rendre dynamiquement un champ présent dans un champ Disposition visible ou invisible, il suffit d'utiliser la propriété [Visible](../Proprietes/2510138.md) en utilisant la syntaxe suivante : 

```txt
<Nom du champ>.Visible = <Vrai/Faux>
```

**Remarque : Fonctionnement d'un champ Disposition en mode "pousser les champs autour"** : 
La cellule du champ Disposition est automatiquement rendue invisible et les autres cellules du champ Disposition sont automatiquement repositionnées si les conditions suivantes sont réunies : 

- un champ présent dans un champ Disposition est rendu invisible,

- ce champ est seul présent dans une cellule ou tous les champs de la cellule du champ Disposition sont invisibles.  




Remarques : 

- Si tous les champs présents dans un champ Disposition sont invisibles, le champ Disposition reste visible. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) La propriété [VisibleAvecAnimation](../Proprietes/1410087232.md) permet de rendre visible ou invisible un champ Disposition en jouant une animation.





### Rendre dynamiquement invisible une cellule d'un champ Disposition 
<a name="rendre_dynamiquement_invisible_une_cellule_champ_disposition_ELTPARAGRAPHE000055"></a>Pour rendre dynamiquement une cellule d'un champ Disposition visible ou invisible, il suffit d'utiliser la propriété [Visible](../Proprietes/2510138.md) sur la cellule en utilisant la syntaxe suivante : 

```txt
<Nom du champ Disposition>[Indice de la cellule].Visible = <Vrai/Faux>
```

**Remarque : Fonctionnement d'un champ Disposition en mode "pousser les champs autour"** : 
Si une cellule d'un champ Disposition est rendu invisible, automatiquement tous les champs de la cellule sont rendus invisibles : les autres cellules du champ Disposition sont automatiquement repositionnées. 

Remarques : 

- Si toutes les cellules présentes dans un champ Disposition sont invisibles, le champ Disposition est lui-même invisible. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) La propriété [VisibleAvecAnimation](../Proprietes/1410087232.md) permet de rendre visibles ou invisibles les cellules d'un champ Disposition en jouant une animation.








## Programmation du champ Disposition
<a name="programmation_champ_disposition_ELTTEXTE000198"></a>


### Propriétés disponibles
<a name="proprietes_disponibles_ELTPARAGRAPHE000089"></a>Les propriétés spécifiques au champ Disposition sont les suivantes : 



|   |   |
| --- | --- |
| [Disposition](../Proprietes/1000023391.md) | La propriété **Disposition** permet de connaître la disposition des cellules dans un champ Disposition. |
| [HauteurAvecAnimation](../Proprietes/1410087234.md) | La propriété **HauteurAvecAnimation** permet de modifier la hauteur d'une cellule d'un champ Disposition en jouant l'animation "Changement de taille d'une cellule" définie : <br><br>- soit dans l'onglet "Avancé" de la fenêtre de description du projet (bouton "Animation des champs"). <br><br>- soit dans l'onglet "Style" du champ ou du projet.<br><br><br> |
| [LargeurAvecAnimation](../Proprietes/1410087240.md) | La propriété **LargeurAvecAnimation** permet de modifier la largeur d'une cellule d'un champ Disposition en jouant l'animation "Changement de taille d'une cellule" définie : <br><br>- soit dans l'onglet "Avancé" de la fenêtre de description du projet (bouton "Animation des champs"). <br><br>- soit dans l'onglet "Style" du champ ou du projet.<br><br><br> |
| [MargeInterCellules](../Proprietes/1000023413.md) | La propriété **MargeInterCellules** permet de : <br><br>- connaître la marge entres les lignes ou les colonnes d'un champ Disposition. <br><br>- modifier la marge entres les lignes ou les colonnes d'un champ Disposition. <br><br><br> |
| [VisibleAvecAnimation](../Proprietes/1410087232.md) | La propriété **VisibleAvecAnimation** permet de rendre un champ visible ou invisible en jouant l'animation "Apparition/Disparition". |



Pour connaître la liste complète des propriétés du WLangage pouvant être utilisées avec un champ de type Disposition, consultez [Propriétés associées au champ Disposition](../WDChamp/1410087048.md).


## Manipuler un champ Potentiomètre par programmation
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000153"></a>
WINDEV et WINDEV Mobile permettent de manipuler un champ Potentiomètre directement par programmation. Il suffit d'utiliser directement la variable du champ Potentiomètre dans le code.

La variable du champ Potentiomètre:

- correspond au nom du potentiomètre.

- est initialisé avec la position du curseur du potentiomètre.






<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Récupérer la position du curseur
<a name="recuperer_position_curseur_ELTTEXTE000177"></a>
Pour récupérer la position d'un potentiomètre, utilisez la syntaxe suivante :


```txt
<Position> = <Nom Potentiomètre>
```


&lt;Position&gt; est une variable de type entier

**Remarque** : Il est également possible d'utiliser la propriété [Valeur](../Proprietes/2510130.md).



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Modifier la position du curseur
<a name="modifier_position_curseur_ELTTEXTE000201"></a>
Pour modifier la position d'un potentiomètre, utilisez la syntaxe suivante :


```txt
<Nom Potentiomètre> = <valeur>
```


&lt;Valeur&gt; est une valeur comprise entre la valeur minimale et la valeur maximale du potentiomètre. Ces valeurs ont été décrites dans la fenêtre de description du champ, ou par programmation, grâce aux propriétés [BorneMin](../Proprietes/2510008.md) et [BorneMax](../Proprietes/2510009.md).

**Remarque** : Il est également possible d'utiliser la propriété [Valeur](../Proprietes/2510130.md).



<a name="NOTE4"></a>
<a name="NOTE4_1"></a>


## Propriétés spécifiques à la gestion d'un potentiomètre
<a name="proprietes_specifiques_gestion_potentiometre_ELTTEXTE000225"></a>
Les propriétés suivantes sont spécifiques à la gestion des caractéristiques d'un potentiomètre par programmation.


|   |   |
| --- | --- |
| [BorneMax](../Proprietes/2510009.md) | Permet de connaître et de modifier la borne maximale d'un potentiomètre. |
| [BorneMin](../Proprietes/2510008.md) | Permet de connaître et de modifier la borne minimale d'un potentiomètre. |
| [Image](../Proprietes/2510034.md) | Permet de connaître et de modifier l'image d'un curseur de potentiomètre standard. |
| [ImageEtat](../Proprietes/2510055.md) | Permet de connaître et de modifier le nombre de dessins présents dans l'image associée à l'image du curseur dans un potentiomètre standard. |
| [ImageFond](../Proprietes/2510056.md) | Permet de connaître et de modifier l'image de fond d'un potentiomètre standard. |
| [Vertical](../Proprietes/2510133.md) | Permet de savoir si le potentiomètre standard est horizontal ou vertical. |


Pour connaître la liste complète des propriétés du WLangage pouvant être utilisées avec un champ de type Potentiomètre, consultez [Propriétés associées au champ Potentiomètre](../WDChamp/1013233.md).

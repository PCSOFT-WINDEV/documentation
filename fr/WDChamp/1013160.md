
## Animer un champ Libellé
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Principe
<a name="principe_ELTTEXTE000153"></a>
WINDEV permet d'animer un champ Libellé de deux façons différentes :

- libellé défilant : le libellé défile d'un côté à l'autre du champ. Par exemple : <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=LibelleScrollant.gif)


- libellé clignotant : le libellé s'affiche alternativement de deux couleurs différentes.




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Animer un champ Libellé
<a name="animer_champ_libelle_ELTTEXTE000177"></a>


### Comment le faire ? 
<a name="comment_faire_ELTPARAGRAPHE000022"></a>Pour animer un champ Libellé : 

1. Affichez la fenêtre de description du libellé (option "Description" du menu contextuel du libellé).

2. Dans l'onglet "Détail", paramétrez les options d'animation. Choisissez le type d'animation et configurez les options associées. Les options varient selon le mode d'animation choisi.




Par exemple,** si l'animation choisie est le défilement,** il est possible d'indiquer :

- La cadence de rafraîchissement entre chaque déplacement du libellé.

- Le nombre de pixels correspondant à un déplacement du libellé.

- Le sens de défilement.




Par exemple, **si l'animation choisie est le clignotement**, il est possible d'indiquer :

- La cadence de rafraîchissement entre chaque clignotement.

- La couleur de clignotement du libellé. Pour faire clignoter un libellé, deux couleurs sont nécessaires :

	- la couleur normale du libellé.

	- la couleur de clignotement du libellé. Si cette couleur est transparente, le libellé apparaîtra une fois sur deux.








<a name="NOTE2_2"></a>


### Remarques
<a name="remarques_ELTPARAGRAPHE000047"></a>

- Il n'est possible d'animer un libellé au format RTF ou un libellé avec une "Ombre floue".

- L'animation d'un libellé est visible uniquement en exécution.

- Un libellé animé est limité à 1024 octets.

- Si les fonctions de mise en forme du texte sont utilisées ([gPoliceGras](../WDLang1/3029061.md), [gPoliceTaille](../WDLang1/3029064.md), [gPoliceItalique](../WDLang1/3029062.md), [gPoliceSoulignée](../WDLang1/3029063.md)), l'animation ne fonctionne plus.

## Programmation
<a name="programmation_ELTTEXTE000207"></a>
Différentes fonctions du WLangage permettent de gérer les libellés animés. Ces fonctions sont les suivantes :


|   |   |
| --- | --- |
| [AnimationChangeParamètre](../WDLang1/3017002.md) | Change un des paramètres d'animation d'un texte dans un champ Libellé. |
| [AnimationChangeType](../WDLang1/3017001.md) | Change le type d'animation utilisé dans un champ Libellé. |
| [AnimationLitParamètre](../WDLang1/3017003.md) | Lit la valeur d'un paramètre d'animation d'une image dans un champ Libellé. |




<a name="NOTE3_2"></a>
Différentes propriétés WLangage permettent de gérer les libellés animés. Ces propriétés sont les suivantes :


|   |   |
| --- | --- |
| [Animation](../Proprietes/2510006.md) | Permet de connaître l'état actuel de l'animation dans un champ Libellé et de lancer ou d'arrêter l'animation d'un champ Libellé animé. |
| [AnimationInitiale](../Proprietes/2510012.md) | Permet de connaître l'état initial de l'animation dans un champ Libellé. |

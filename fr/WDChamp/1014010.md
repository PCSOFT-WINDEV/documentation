


## Modifier les caractéristiques des champs d'une fenêtre
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000325"></a>
Les différentes caractéristiques d'un ou de plusieurs champs peuvent être visualisées et/ou modifiées dans :

- [la fenêtre de description de ce champ](#NOTE2_1).

- [le modifieur](#NOTE3_1).




L'éditeur de fenêtres permet également directement de :

- [modifier le libellé d'un champ](#NOTE4_1).

- [modifier le nom d'un champ](#NOTE5_1).

- [modifier la position et les dimensions d'un champ](#NOTE6_1).

- [changer l'image associée à un champ](#NOTE7_1).

<a name="NOTE2_1"></a>


## Fenêtre de description des champs d'une fenêtre
<a name="fenetre_description_des_champs_une_fenetre_ELTTEXTE000349"></a>


### Afficher la fenêtre de description d'un champ
<a name="afficher_fenetre_description_champ_ELTPARAGRAPHE000040"></a>Pour afficher la fenêtre de description d'un champ : 

- soit double-cliquez sur le champ.

- soit sélectionnez l'option "Description" du menu contextuel du champ (clic droit de la souris).

- soit sélectionnez le champ et utilisez la combinaison de touches Alt + Entrée.




Par exemple, fenêtre de description d'un bouton : 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=FenDescriptionChamp.gif)


**Remarques** :

- Il est possible d'afficher la fenêtre de description pour une sélection de champs. Seules les caractéristiques communes aux différents champs sélectionnés seront affichées.

- Plusieurs fenêtres de description peuvent être affichées simultanément. Chaque fenêtre de description affiche alors les caractéristiques d'un ou de plusieurs champs.



<a name="NOTE2_2"></a>


### Caractéristiques par onglet
<a name="caracteristiques_par_onglet_ELTPARAGRAPHE000059"></a>La fenêtre de description regroupe les différentes caractéristiques du ou des champs sélectionnés par onglet :

| Onglet | Caractéristiques |
| --- | --- |
| **Général** | Permet de spécifier le nom du champ et toutes les caractéristiques d'affichage du champ. Par exemple :<br><br>- le libellé du champ (ce libellé peut être multilingue).<br><br>- le type d'information affiché dans le champ (texte, numérique, ...).<br><br>- le masque de saisie utilisé pour le champ (ce masque peut être multilingue). Pour plus de détails, consultez les [Différents masques de saisie](../WDChamp/1014003.md).<br><br>- l'image affichée dans un champ Image, le libellé des options d'un sélecteur, les champs contenus dans une barre d'outils, etc. Pour plus de détails sur les caractéristiques d'un champ, consultez directement l'aide sur le type de ce champ.<br><br><br> |
| **UI** | Permet de définir :<br><br>- l'état initial du champ à l'ouverture de la fenêtre.<br><br>- les différents paramètres du champ (accessible par Tab, ...).<br><br>- l'action du clic droit sur le champ.<br><br>- le curseur de la souris lors du survol du champ.<br><br>- le menu contextuel associé au champ. Pour plus de détails, consultez [Menu contextuel](../WDChamp/1010011.md).<br><br>- le raccourci clavier associé au champ.<br><br>- la taille du champ.<br><br>- la position du champ dans la fenêtre.<br><br>- l'ancrage du champ. Pour plus de détails, consultez [Ancrage](../Proprietes/2510001.md).<br><br>- le plan de la fenêtre auquel le champ appartient. Pour plus de détails, consultez [Plan d'une fenêtre](../WDChamp/1013078.md).<br><br>- le groupe de champs auquel le champ appartient. Pour plus de détails, consultez [Groupe de champs](../WDChamp/1010016.md).<br><br><br> |
| **Détail** | Permet de définir les paramètres avancés des champs :<br><br>- l'action du Drag and Drop pour ce champ. Pour plus de détails, consultez [Gestion du Drag and Drop](../WDLang1/3030007.md).<br><br>- l'utilisation de la validation automatique (pour les boutons). Pour plus de détails, consultez [Validation automatique d'une fenêtre](../WDChamp/1013072.md).<br><br>- les paramètres de saisie pour les champs de saisie.<br><br>- les paramètres spécifiques aux champs Liste, Combo, Table, etc.<br><br><br>Ces paramètres avancés sont regroupés en plusieurs catégories :<br><br>- les paramètres modifiant l'apparence du champ (paramètres sur fond beige).<br><br>- les paramètres modifiant le fonctionnement et la saisie dans le champ (paramètres sur fond vert).<br><br>- les paramètres concernant la valeur de retour du champ (paramètres sur fond bleu).<br><br><br> |
| **Liaison** | Permet de sélectionner la rubrique (d'un fichier ou d'une requête) à laquelle le champ est relié. Selon l'enregistrement en cours, le contenu de la rubrique reliée sera affiché dans le champ. |
| **Contenu** | Permet de définir :<br><br>- soit le contenu initial du champ (pour les champs de saisie uniquement).<br><br>- soit la source de données utilisée pour remplir le champ (pour les champs Liste, Combo et Table uniquement).<br><br><br> |
| **Note** | Permet de décrire le fonctionnement détaillé du champ.<br>Ces informations seront imprimées dans la documentation technique (icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_imprimer.gif)<br> sous le volet "Accueil", dans le groupe "Général").<br><br>Ces informations peuvent être remplies et connues par programmation grâce aux propriétés [Note](../Proprietes/2510085.md) et [TitreNote](../Proprietes/2510121.md). |
| **Aide** | Permet d'associer de l'aide au champ. Pour plus de détails, consultez [Aider l'utilisateur](../Editeurs/2010012.md). |
| **Style** | Permet de :<br><br>- appliquer un style prédéfini au champ.<br><br>- modifier ou créer un style prédéfini pour le type du champ.<br><br>- modifier l'aspect du champ (couleur de fond, cadre, police, ...).<br><br><br> |

## Utilisation du modifieur
<a name="utilisation_modifieur_ELTTEXTE000379"></a>


### Présentation
<a name="presentation_ELTPARAGRAPHE000184"></a>Le modifieur permet de visualiser et/ou de modifier directement depuis l'éditeur de fenêtres les principales caractéristiques :

- du ou des champs sélectionnés.

- de la fenêtre sélectionnée.




Pour plus de détails, consultez le [Modifieur](../Editeurs/2027006.md).
<a name="NOTE3_2"></a>


### Visualiser et/ou modifier les caractéristiques d'un ou de plusieurs champs grâce au modifieur
<a name="visualiser_etou_modifier_les_caracteristiques_plusieurs_champs_grace_modifieur_ELTPARAGRAPHE000199"></a>Pour visualiser et/ou modifier les caractéristiques d'un ou de plusieurs champs grâce au modifieur : 

1. Sélectionnez le ou les champs à modifier :

	- soit directement dans la fenêtre (voir [Sélectionner un ou plusieurs champs](../Editeurs/2026007.md)).

	- soit grâce à la liste déroulante du modifieur. Dans ce cas, seul un champ à la fois peut être sélectionné. Les poignées du champ sélectionné s'affichent automatiquement.
			Si plusieurs champs sont sélectionnés, seules les informations communes aux champs sélectionnés sont affichées dans le modifieur.




2. Effectuez les modifications nécessaires. Ces modifications sont réalisées sur tous les champs. Vous pouvez :

	- soit sélectionner la nouvelle valeur dans une liste déroulante.

	- soit sélectionner la nouvelle valeur dans une fenêtre spécifique (modification d'une couleur par exemple).

	- soit modifier directement la valeur.

## Modifier le libellé d'un champ
<a name="modifier_libelle_champ_ELTTEXTE000409"></a>
Pour modifier le libellé d'un champ directement dans la fenêtre en création : 

1. Sélectionnez le champ à modifier.

2. Cliquez une nouvelle fois sur le champ ou appuyez sur la touche Entrée. Le curseur de la souris se transforme en curseur de saisie. 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=LibelleModifDirect.gif)


3. Saisissez le nouveau libellé du champ.

4. Validez la saisie du libellé en appuyant sur la touche Entrée.




**Remarques** : 

- L'édition directe des libellés effectue la vérification orthographique, autorise la saisie en RTF ainsi que la saisie en alphabet non latin.

- De la même manière, il est possible de modifier le libellé d'une colonne d'un champ Table.
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Lib_table_modifdirect.gif)

## Modifier le nom d'un champ
<a name="modifier_nom_champ_ELTTEXTE000433"></a>
Il est possible de modifier le nom d'un champ directement depuis la barre de messages de l'éditeur de fenêtres. 

Pour modifier le nom d'un champ :

1. Sélectionnez le champ à modifier.

2. Saisissez directement dans la barre de messages le nouveau nom du champ. 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=NomChamp.gif)

## Modifier la position et les dimensions d'un champ
<a name="modifier_position_les_dimensions_champ_ELTTEXTE000457"></a>
Il est possible de modifier la position et/ou les dimensions d'un champ directement depuis la barre de messages de l'éditeur de fenêtres.

Pour modifier la position et/ou les dimensions d'un champ :

1. Sélectionnez le champ à modifier.

2. Saisissez directement dans la barre de messages la nouvelle position et/ou les nouvelles dimensions du champ. 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=PositionDimensionChamp.gif)

## Modifier l'image associée à un champ
<a name="modifier_image_associee_champ_ELTTEXTE000481"></a>


### Changer l'image associée à un champ
<a name="changer_image_associee_champ_ELTPARAGRAPHE000270"></a>Il est possible de changer l'image associée à un champ Image ou à un bouton directement dans la fenêtre en création.

Pour modifier l'image associée à un champ directement dans la fenêtre en création :

1. Sélectionnez le champ à modifier.

2. Utilisez la combinaison de touches Ctrl + Maj + P. Un sélecteur de fichiers s'ouvre.

3. Sélectionnez le fichier image à associer au champ en cours et validez.

## Mode de sélection
<a name="mode_selection_ELTTEXTE000152"></a>
L'onglet "UI" de la fenêtre de description du champ Liste permet de définir le mode de sélection dans le champ. 

Les différents modes de sélection disponibles sont : 

- **Sélection simple** : Le bandeau de sélection s'affiche sur la ligne sélectionnée. Le code de sélection de ligne est exécuté.

- **Sélection multiple** : Le bandeau de sélection s'affiche sur les lignes sélectionnées. Pour sélectionner plusieurs lignes, il suffit de maintenir la touche Maj ou Ctrl enfoncée. A chaque ligne sélectionnée, le code de sélection d'une ligne est exécuté.

## Déplacement
<a name="deplacement_ELTTEXTE000176"></a>
L'onglet "Détail" de la fenêtre de description d'un champ Liste permet notamment de définir les options de déplacement de la liste. Il est possible de définir : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) **L'affichage de l'ascenseur horizontal / vertical**. Il peut être :

	- automatique : l'ascenseur sera affiché uniquement en cas de besoin.

	- toujours visible : l'ascenseur sera toujours affiché.

	- jamais visible : l'ascenseur ne sera jamais affiché.




- **Le mode de déplacement** : 

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Scroll au pixel : selon la position de l'ascenseur, une ligne peut ne pas être affichée complètement.

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Scroll au doigt : permet de manipuler l'ascenseur par un écran tactile.

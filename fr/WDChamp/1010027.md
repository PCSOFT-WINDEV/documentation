<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000070"></a>
Une fenêtre maximisée occupe tout l'écran du mobile. Différentes options sont paramétrables dans l'assistant de création de fenêtres.  

Rappel : Pour lancer l'assistant de création d'une fenêtre : 

- Cliquez sur ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_nouveau.gif)
 parmi les boutons d'accès rapide. 

- La fenêtre de création d'un nouvel élément s'affiche : cliquez sur "Fenêtre" puis sur "Fenêtre".

- Sélectionnez "Assistant" et validez. 

- [Options disponibles en Android](../WDChamp/9000113.md) : ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Fen_Max_Caract_Android.gif)

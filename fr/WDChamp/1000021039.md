
## Graphe de type "Circulaire de progrès"
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000104"></a>
Le graphe circulaire de progrès est un graphe en anneau utilisé pour comparer des objectifs ou pour illustrer une progression.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Caractéristiques d'un graphe circulaire de progrès
<a name="caracteristiques_graphe_circulaire_progres_ELTTEXTE000128"></a><br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Graphe_Circulaire_de_progres%20-%20HC%20N%B0001.gif)


Dans ce graphe :

**1.** **Titre du graphe.**
Les caractéristiques du titre du graphe sont saisies dans l'onglet "Détail" de la description du champ (bouton "Editer le titre du graphe").
Il est possible de définir le titre du graphe, sa position et son style.

**2.** **Taille du trou**
Un graphe de type Graphe circulaire de progrès peut afficher un trou en son centre. Il est possible de définir la taille du trou dans l'onglet "Général" de la description du champ. 

**3.** **Portion du graphe**
A chaque niveau, les valeurs affichées représentent une portion, correspondant à la valeur de la donnée. Il est possible d'afficher les étiquettes de catégorie ainsi que la valeur en pourcentage. 
Les étiquettes de catégorie peuvent être affichées à l'intérieur ou à l'extérieur de la portion. 

**4.** **Bulle d'aide**
La valeur des données survolées à la souris s'affiche dans une bulle.

**5.** **Légende du graphe.**
Les caractéristiques de la légende sont saisies dans l'onglet "Détail" de la description du champ (bouton "Editer la légende du graphe"). Il est possible de définir sa position et son style.


**Autres caractéristiques** : 

- Il est possible de définir l'angle de départ du graphe.  

- Les fonctions WLangage spécifiques à ce type de graphe sont les suivantes : 


|   |   |
| --- | --- |
| [grCPAjouteDonnée](../WDLang3/1410087792.md) | Ajoute une donnée à un graphe circulaire de progrès. |
| [grCPSensDeRotation](../WDLang3/1410087791.md) | Fixe le sens de rotation d'un graphe circulaire de progrès. |
| [grCPValeurMaximale](../WDLang3/1410087801.md) | Fixe la valeur maximale du graphe circulaire de progrès. |




Remarque : Ce type de graphe n'accepte pas de valeurs négatives.

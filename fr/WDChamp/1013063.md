## Présentation du champ Ascenseur
<a name="presentation_champ_ascenseur_ELTTEXTE000135"></a>


### Champ d'une fenêtre
<a name="champ_une_fenetre_ELTPARAGRAPHE000011"></a>Le champ Ascenseur permet de faire défiler des objets (texte, libellé, ...). Le défilement de l'objet voulu est réalisé par programmation de l'ascenseur. Un ascenseur horizontal permet un défilement horizontal et un ascenseur vertical permet un défilement vertical.

Un ascenseur est constitué des éléments suivants :<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Champ_ASCENSEUR.gif)

## Créer un champ Ascenseur
<a name="creer_champ_ascenseur_ELTTEXTE000159"></a>
Pour créer un champ de type Ascenseur : 

1. Sous le volet "Création", dans le groupe "Ajuster", déroulez "Ascenseur" et cliquez sur "Ascenseur".

2. Cliquez dans la fenêtre à la position où le champ doit être créé. Le champ apparaît en création.




Pour afficher les caractéristiques du champ, sélectionnez l'option "Description" dans le menu contextuel du champ. Vous pourrez ainsi choisir l'orientation de l'ascenseur.



## Image d'un champ Ascenseur
<a name="image_champ_ascenseur_ELTTEXTE000183"></a>
Un champ ascenseur peut être dessiné à partir des éléments système ou à l'aide d'une image.

Pour utiliser une image :

1. Ouvrez la fenêtre de description du champ.

2. Sélectionnez l'onglet "Style".

3. Choisissez l'élément "Fond et Image".

4. Sélectionnez un cadre de type "Image (Mode 9 images)".

5. Cliquez sur le bouton à côté du champ "Cadre". 

6. La popup qui s'affiche permet de choisir les deux images à utiliser (ascenseur horizontal et ascenseur vertical) via le bouton ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Menu_Image_Editeur%20-%20HC%20N%B0001.gif) : 

	- L'option "Parcourir" permet d'utiliser un fichier accessible depuis votre poste. 

	- L'option "Catalogue" permet de sélectionner une image du catalogue d'images. Le catalogue d'images propose par défaut un grand nombre d'images d'ascenseurs.

## Vocabulaire associé au champ Ascenseur
<a name="vocabulaire_associe_champ_ascenseur_ELTTEXTE000207"></a>
Pour manipuler simplement un ascenseur, il est nécessaire de connaître le vocabulaire suivant :

| Nom | Définition |
| --- | --- |
| Avance rapide | Pas de l'ascenseur utilisé lorsque l'utilisateur clique dans la zone située entre la cage de l'ascenseur et le bouton spin. |
| Répétition | Lorsque le bouton de la souris reste enfoncé sur un des boutons spin, une répétition du clic sur le bouton spin est automatiquement gérée. Il est possible de paramétrer l'intervalle de temps entre ces répétitions. |
| Valeur minimale | Valeur de l'ascenseur lorsque la cage d'ascenseur est placée à l'origine (1 par défaut). Cette valeur correspond à la borne minimale. |
| Valeur maximale | Valeur de l'ascenseur lorsque la cage de l'ascenseur est placée dans la position extrême. Cette valeur correspond à la borne maximale. |

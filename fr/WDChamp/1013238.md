
## Manipuler un champ Spin par programmation
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000134"></a>
WINDEV et WINDEV Mobile permettent de manipuler un champ Spin directement par programmation. Il suffit d'utiliser directement la variable du champ Spin dans le code.

La variable du champ Spin :

- correspond au nom du champ Spin.

- est initialisée avec la valeur du spin.






<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Récupérer la valeur du spin
<a name="recuperer_valeur_spin_ELTTEXTE000158"></a>
Pour récupérer la valeur du spin, utilisez la syntaxe suivante :


```txt
<Position> = <Nom Spin>
```


&lt;Position&gt; est une variable de type entier.

**Remarque** : Il est également possible d'utiliser la propriété [Valeur](../Proprietes/2510130.md).



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Modifier la valeur du spin
<a name="modifier_valeur_spin_ELTTEXTE000182"></a>
Pour modifier la valeur du spin, utilisez la syntaxe suivante :


```txt
<Nom Spin> = <Valeur>
```


&lt;Valeur&gt; est une valeur comprise entre la valeur minimale et la valeur maximale du spin. Ces valeurs ont été décrites dans la fenêtre de description du champ, ou par programmation, grâce aux propriétés [BorneMin](../Proprietes/2510008.md) et [BorneMax](../Proprietes/2510009.md).

**Remarque** : Il est également possible d'utiliser la propriété [Valeur](../Proprietes/2510130.md).

## Propriétés spécifiques à la gestion d'un champ Spin
<a name="proprietes_specifiques_gestion_champ_spin_ELTTEXTE000206"></a>
Les propriétés suivantes sont spécifiques à la gestion des caractéristiques d'un spin par programmation.


|   |   |
| --- | --- |
| [AvanceRapide](../Proprietes/2510011.md) | Permet de connaître et de modifier la valeur du pas d'incrémentation d'un spin. |
| [BorneMax](../Proprietes/2510009.md) | Permet de connaître et de modifier la borne maximale d'un spin. |
| [BorneMin](../Proprietes/2510008.md) | Permet de connaître et de modifier la borne minimale d'un spin. |


Pour connaître la liste complète des propriétés du WLangage pouvant être utilisées avec un champ de type Spin, consultez [Propriétés associées au champ Spin](../WDChamp/1013226.md).

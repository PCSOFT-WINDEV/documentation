## Présentation
<a name="presentation_ELTTEXTE000161"></a>


### Champ d'une fenêtre ou d'une page
<a name="champ_une_fenetre_une_page_ELTPARAGRAPHE000011"></a>Le champ Notation permet à l'utilisateur : 

- d'indiquer le niveau de popularité d'un élément sous forme de pictogrammes répétés. 

- de connaître le niveau de popularité d'un élément sous forme de pictogrammes répétés. 




Généralement, ce type de champ présente le niveau de popularité sous forme de 1 à 5 étoiles : ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Notation_Etoile.gif)


Il peut être également utilisé pour indiquer un choix de l'utilisateur : ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Notation_Jaime.gif)


WINDEV et WEBDEV offrent la possibilité : 

- d'utiliser n'importe quel nombre de pictogrammes.

- d'utiliser n'importe quelle forme de pictogramme. Par exemple, il est possible d'utiliser des images du catalogue d'images.

## Créer un champ de type Notation
<a name="creer_champ_type_notation_ELTTEXTE000185"></a>
Pour créer un champ de type Notation :

1. Sous le volet "Création", dans le groupe "Champs graphiques", cliquez sur "Notation".

2. Cliquez dans la fenêtre ou dans la page à la position où le champ doit être créé. Le champ apparaît en création.




**Astuce** : Il est également possible de créer directement un champ Notation "pré-formaté" en cliquant sur la flèche présente à droite de l'option "Notation". La liste des champs disponibles avec leur aperçu au gabarit du projet apparaît. Il suffit de cliquer sur le champ voulu pour le créer dans l'éditeur.

Pour afficher les caractéristiques du champ, sélectionnez l'option "Description" dans le menu contextuel du champ.

## Caractéristiques du champ Notation
<a name="caracteristiques_champ_notation_ELTTEXTE000209"></a>
L'onglet "Général" de la fenêtre de description du champ Notation permet de définir les caractéristiques du champ : 

- La **note maximale possible** : permet de définir le nombre de répétitions du pictogramme dans le champ Notation. Cette option est disponible uniquement en mode "Même image pour chaque note". 

- Le **nombre de notes dans l'image** : indique le nombre de pictogrammes présents dans l'image. Cette option est disponible uniquement si le découpage de la planche d'images est en mode "Image différente pour chaque note". 

- La **note initiale**, c'est-à-dire la note utilisée à l'affichage du champ. 

- La **granularité des notes**. Cette option correspond au mode de sélection des notes. Il est possible de faire évoluer la notation : 

	- par valeur entière : la note évoluera de 1 en 1

	- par demi-valeur : la note évoluera de 0,5 en 0,5. 

	- par valeur réelle : la note évoluera selon la valeur saisie par l'utilisateur. 




- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) La **valeur d'une 'étoile'** : valeur associée à une étoile (ou tout autre pictogramme). La valeur du champ Notation correspondra alors à la valeur d'une étoile multipliée par le nombre d'étoiles affichées. 

- **L'image des notes** : Image utilisée pour visualiser les notes. Si ce paramètre n'est pas renseigné, l'image d'une étoile est utilisée par défaut. Le bouton ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Menu_Image_Editeur%20-%20HC%20N%B0001.gif)
 permet de sélectionner : 

- soit une image accessible depuis votre poste (option "Parcourir"). 

- soit une image du catalogue d'images (option "Catalogue"). Dans ce cas, l'image sélectionnée est automatiquement adaptée aux options spécifiées.

- **Découpage de la planche d'images** : permet de définir le mode de découpage de la planche d'images du champ : 

	- **Même image pour chaque note** : L'image doit contenir un seul pictogramme (représenté sous plusieurs états). La répétition du pictogramme permettra de représenter les différentes valeurs possibles dans le champ. 

	- **Image différente pour chaque note** : L'image doit contenir plusieurs pictogrammes (chacun étant représenté sous plusieurs états). Chaque pictogramme permettra de représenter les différentes valeurs possibles dans le champ. Dans ce mode, la note maximale correspond au nombre de pictogrammes contenus dans l'image. 




- **Différencier graphiquement les notes inférieures à la note courante** : 

	- Lorsque ce mode est désactivé, les notes inférieures à la note en cours sont affichées de la même manière que la note en cours. 

	- Lorsque ce mode est activé, les notes inférieures à la note en cours sont dessinées en utilisant l'état 0 de la planche d'images (les notes inférieures seront invisibles si l'image ne contient qu'un seul état).

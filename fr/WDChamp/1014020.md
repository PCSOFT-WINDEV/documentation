
## Champs déplaçables dans une page
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000145"></a>
WEBDEV propose deux modes de déplacement des champs dans une page :

- **Champs déplaçables par programmation** : des propriétés de positionnement des champs sont disponibles.

- **Champs déplaçables à la souris** : il est possible d'indiquer qu'un champ peut être déplacé à la souris directement par l'utilisateur. La nouvelle position du champ pourra être sauvegardée.






<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Champs déplaçables par programmation
<a name="champs_deplacables_par_programmation_ELTTEXTE000169"></a>
Les champs des pages sont déplaçables dynamiquement par programmation.

Les propriétés [X](../Proprietes/2510135.md) et [Y](../Proprietes/2510137.md) sont disponibles en lecture et en écriture, en code serveur et en code navigateur.

Il est ainsi possible de :

- proposer à l'internaute de placer lui-même des champs ou des groupes de champs.

- modifier dynamiquement la position des champs en fonction de certains traitements et de certaines actions.




Exemple : Suivre le pointeur de la souris : 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Deplacement_Souris4.gif)

## Champs déplaçables à la souris
<a name="champs_deplacables_souris_ELTTEXTE000193"></a>
Dans une page, un champ peut directement être déplacé grâce à la souris par l'internaute. L'internaute pourra ainsi, réorganiser la page à sa guise. Il est également possible de mémoriser la nouvelle position du champ fixée par l'internaute. Ainsi, lors de la prochaine ouverture de la page, les champs apparaîtront à la position souhaitée.


![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Deplacement_souris3.gif)


Cette fonctionnalité est surtout utilisée pour permettre de déplacer simplement les champs contenus dans une cellule, une zone répétée, un superchamp, ...

**Pour rendre des champs déplaçables à la souris** : 

1. Affichez la fenêtre de description du champ.

2. Dans l'onglet "UI", cochez les options voulues dans la zone "Superposition et déplacement". 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Champ_Deplacable_Page%20-%20HC%20N%B0001.gif&type=thumb)





Si le champ est superposable, il peut être déplaçable à la souris, et dans ce cas, il est possible de mémoriser la nouvelle position du champ.

**Remarque** : il est possible de définir un champ déplaçable à la souris dans une page interne. Ce champ sera déplaçable dans la page qui accueille la page interne.


<a name="NOTE3_2"></a>


### Cas des champs Zone répétée, Superchamp, Cellule
<a name="cas_des_champs_zone_repetee_superchamp_cellule_ELTPARAGRAPHE000066"></a>Pour rendre les champs Zone répétée, Superchamp et Cellule déplaçables à la souris, il est également nécessaire d'indiquer le ou les champs qui vont permettre de déplacer le champ concerné. 

Par exemple, pour le champ Cellule : 

1. Affichez la fenêtre de description du champ.

2. Dans l'onglet "UI", dans la zone "Superposition et déplacement du champ", sélectionnez l'option "Superposable et déplaçable à la souris". 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Champ_Deplacable_Page%20-%20HC%20N%B0002.gif)


3. Cliquez sur le bouton "Paramètres" et spécifiez le ou les champs qui permettront de déplacer le champ Cellule à la souris : 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Champ_Deplacable_Page%20-%20HC%20N%B0005.gif)





**Attention** : Il n'est pas possible d'utiliser un champ iFrame, WebCam, Flash ou Applet Java pour déplacer les champs Zone répétée, Superchamp et cellule à la souris.


<a name="NOTE3_3"></a>


### Limitations
<a name="limitations_ELTPARAGRAPHE000088"></a>Seuls les champs suivants peuvent être déplaçables directement :

- Champ de saisie,

- Cellule,

- Zone répétée,

- Superchamp,

- Libellé,

- Image.

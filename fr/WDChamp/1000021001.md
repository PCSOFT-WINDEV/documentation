## Présentation du champ Ruban
<a name="presentation_champ_ruban_ELTTEXTE000195"></a>
Le champ Ruban offre une fonctionnalité de menu supplémentaire à vos applications. Il permet de présenter dans une petite surface un grand nombre de champs rangés avec logique afin d'effectuer des choix. 

Par exemple, les différents menus de l'application "Gestion Commerciale" présentés sous forme de ruban : 

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Champ_Ruban1.gif)


![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Champ_Ruban2.gif)


![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Champ_Ruban3.gif)

Dans cet exemple, les différents éléments du champ Ruban sont des boutons, mais il est possible d'utiliser n'importe quel type de champ. 

En exécution, selon la taille de la fenêtre, les éléments du ruban sont plus ou moins affichés. Si tous les éléments ne peuvent pas être affichés, une petite flèche permet de "dérouler le ruban" et d'afficher sous forme de fenêtre popup la totalité des éléments : 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Champ_Ruban4.gif)
**Remarques** : 

- La touche TAB utilisée sur un champ de saisie présent dans le champ Ruban permet de passer au champ suivant (selon l'ordre de tabulation). 

- La touche ENTREE utilisée sur un champ de saisie présent dans le champ Ruban n'a aucun effet.

## Créer un champ Ruban
<a name="creer_champ_ruban_ELTTEXTE000219"></a>
Pour créer un champ de type Ruban :

1. Sous le volet "Création", dans le groupe "Conteneurs", déroulez "Onglet et associés" et cliquez sur "Ruban".
	Autre possibilité : Sous le volet "Fenêtre", dans le groupe "Barres et menus", déroulez "Autres barres" et sélectionnez "Ruban".

2. Cliquez dans la fenêtre à la position où le champ doit être créé. Le champ apparaît en création. 

3. Définissez les caractéristiques du champ Ruban. Pour afficher les caractéristiques du champ, sélectionnez l'option "Description" dans le menu contextuel du champ.

4. Insérez dans les différents regroupements les champs voulus. Tous les types de champs peuvent être inclus dans un regroupement. 




Pour transformer le menu déroulant d'une fenêtre en champ Ruban : 

1. Ouvrez la fenêtre contenant le menu principal. 

2. Sous le volet "Modification", dans le groupe "Transformations", déroulez "Refactoring et permutations" et sélectionnez "Menu principal vers ruban".


Attention : Pour utiliser cette option, le champ Ruban ne doit ne doit pas exister dans la fenêtre.

## Caractéristiques du champ Ruban
<a name="caracteristiques_champ_ruban_ELTTEXTE000243"></a>
Le champ Ruban est constitué de volets et de regroupements. 

Chaque volet est représenté par une icône et un libellé. 

Chaque volet contient un ou plusieurs regroupements séparés par une ligne verticale. 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Ruban_Volet_regroupement.gif)



### Description d'un champ Ruban
<a name="description_champ_ruban_ELTPARAGRAPHE000103"></a>La fenêtre de description d'un champ Ruban permet de : 

- définir les différents volets du champ Ruban (onglet "Général"). Pour chaque volet, il est possible de définir : 

	- le libellé du volet, 

	- son état initial, 

	- l'image affichée à côté du libellé. 
			La position de cette image peut être modifiée dans l'onglet "Style" (éléments "Libellé du volet actif" et "Libellés des volets inactifs").  

	- la bulle d'aide du volet. 




- définir pour chaque volet, les différents regroupements affichés par le volet (onglet "Général"). Pour chaque regroupement, il est possible de définir : 

	- le libellé du regroupement. Ce libellé sera affiché en bas du regroupement lorsque celui-ci est ouvert. 

	- si le volet affiche un bouton spécifique (option "Avec bouton (traitement de clic)"). 
			Dans ce cas, le traitement de clic du regroupement sera exécuté lorsque l'utilisateur cliquera sur ce bouton. 
			Il est ainsi possible par exemple d'afficher un menu contextuel ou une fenêtre spécifique, ...

	- l'image à afficher lorsque le regroupement est réduit. Cette image doit avoir pour dimensions 32 x 32. 







**Remarques** : 

- Il est possible de personnaliser l'image utilisée pour le dessin des volets dans l'onglet "Détail" de la fenêtre de description du champ. 

- De nombreuses options permettant de personnaliser les volets et les regroupements sont disponibles dans l'onglet "Style" de la fenêtre de description du champ. 





### Utilisation de boutons dans un champ Ruban
<a name="utilisation_boutons_dans_champ_ruban_ELTPARAGRAPHE000131"></a>Si un champ Bouton est présent dans un regroupement d'un champ Ruban, l'onglet "UI" du champ Bouton permet de configurer le mode de gestion du bouton lorsque le ruban est réduit. 

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ruban_Champ_Bouton.gif)
Il est possible au choix : 

- de ne pas réduire le champ. 

- de le réduire pour afficher au minimum l'image du bouton et son libellé. 

- de le réduire pour afficher un minimum l'image du bouton. 




Les deux dernières options sont intéressantes dans le cas de boutons affichant une icône spécifique et un libellé. 

**Remarque** : Il est possible d'utiliser des [boutons à flèche](../WDChamp/1013163.md) dans un champ Ruban, permettant par exemple de hiérarchiser les actions à effectuer. 


- ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=WDOrganigramme.gif) ***Exemples didactiques (WINDEV)*** : **WD Organigramme** <br>Cet exemple permet de créer des organigrammes.<br>Un organigramme sert à indiquer la répartition des responsables à l'intérieur d'une entreprise, et les relations de commandement qui existent entre eux.

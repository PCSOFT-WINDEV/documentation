## Présentation du champ Séparateur
<a name="presentation_champ_separateur_ELTTEXTE000107"></a>
Le champ Séparateur permet de découper une fenêtre en plusieurs zones distinctes. Ces zones sont redimensionnables par l'utilisateur lors de l'exécution de la fenêtre.<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ExChampSplitter.gif)

## Créer un champ Séparateur
<a name="creer_champ_separateur_ELTTEXTE000131"></a>
Pour créer un champ de type Séparateur : 

1. Sous le volet "Création", dans le groupe "Ajuster", cliquez sur "Séparateur".

2. Cliquez dans la fenêtre à la position où le champ doit être créé.
	**Remarque** : La touche Ctrl permet de modifier automatiquement l'orientation du séparateur.

3. L'assistant de création du séparateur se lance.

4. Spécifiez l'orientation du champ et saisissez le nom du champ.

5. Validez l'assistant.




Pour afficher les caractéristiques du champ, sélectionnez l'option "Description" dans le menu contextuel du champ.
<a name="NOTE2_2"></a>


### Gestion du séparateur
<a name="gestion_separateur_ELTPARAGRAPHE000045"></a>Les différentes zones de la fenêtre sont délimitées :

- soit par un des bords de la fenêtre.

- soit par un séparateur.




Remarques :

- L'ancrage d'un champ affiché dans une zone est réalisé par rapport à cette zone et non par rapport à la fenêtre.

- Un champ Séparateur :

	- peut être horizontal ou vertical.

	- ne peut pas être associé à une rubrique.

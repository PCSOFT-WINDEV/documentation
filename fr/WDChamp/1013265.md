## Présentation du champ Chemin de navigation
<a name="presentation_champ_chemin_navigation_ELTTEXTE000163"></a>


### Champ d'une page statique, dynamique, AWP et PHP
<a name="champ_une_page_statique_dynamique_awp_php_ELTPARAGRAPHE000011"></a>Le champ "Chemin de Navigation" correspond à une série de liens permettant à l'internaute de remonter dans l'arborescence logique d'un site.

Le contenu de ce champ se construit dynamiquement au fur et à mesure des "descentes" dans un site.

Typiquement, vous pouvez construire des menus tels que : Matériel &gt;&gt; Photo &gt;&gt; Numérique &gt;&gt; Réflexe.

## Créer un champ chemin de navigation
<a name="creer_champ_chemin_navigation_ELTTEXTE000187"></a>


### Créer un champ Chemin de navigation
<a name="creer_champ_chemin_navigation_ELTPARAGRAPHE000038"></a>Pour créer un champ Chemin de navigation : 

1. Sous le volet "Création", dans le groupe "Navigation", cliquez sur "Chemin de nav.".

2. Cliquez dans la page à la position où le champ doit être créé. Le champ est automatiquement créé. Sous l'éditeur, le champ Chemin de navigation apparaît sous la forme :<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Chemin_Navig_Editeur.gif)





Pour afficher les caractéristiques du champ, sélectionnez l'option "Description" dans le menu contextuel du champ.

Dans l'onglet "Détail" de la description du champ, il est possible de :

- définir le caractère de séparation des différents éléments du chemin de navigation. 

- définir si le dernier élément d'un chemin de navigation est un lien.





<a name="NOTE2_2"></a>


### Paramétrer un champ Chemin de navigation
<a name="parametrer_champ_chemin_navigation_ELTPARAGRAPHE000069"></a>Pour paramétrer un champ Chemin de navigation :

1. Affichez la description du champ (option "Description" du menu contextuel du champ).

2. Dans l'onglet "Général", cliquez sur le bouton "Editer le plan de navigation".

3. La fenêtre de paramétrage du plan de navigation s'affiche. Pour plus de détails, consultez [Construire et modifier le plan de navigation](../WDChamp/1013301.md).

4. Validez le plan de navigation.

5. Le champ "Chemin de navigation" est paramétré pour être utilisé sur les différentes pages de votre site.




**Remarque** : Les différents éléments d'un champ "Chemin de navigation" peuvent être manipulés et modifiés en programmation grâce [aux fonctions WLangage de gestion des champs Chemin de navigation](../WDLang2/3083021.md).

## Conseil
<a name="conseil_ELTTEXTE000217"></a>


### Utilisation de modèles
<a name="utilisation_modeles_ELTPARAGRAPHE000097"></a>Si votre site utilise un modèle, il est conseillé de créer le chemin de navigation de votre site sur le modèle de pages. Ainsi, toutes les pages utilisant le modèle seront automatiquement mises à jour et utiliseront le chemin de navigation.


<a name="NOTE3_2"></a>


### Chemin de navigation et types de pages
<a name="chemin_navigation_types_pages_ELTPARAGRAPHE000105"></a>Le chemin de navigation doit contenir :

- soit uniquement des pages dynamiques WEBDEV.

- soit des pages statiques et / ou des pages AWP.

- soit des pages PHP.




Par programmation, il est possible de mélanger les types de pages.


## Propriétés associées au champ Prédéfini
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les différentes propriétés utilisables avec les champs de type Prédéfini dans l'éditeur d'états sont les suivantes :



|   |   |
| --- | --- |
| [Altitude](../Proprietes/2510002.md) | La propriété **Altitude** permet de gérer l'altitude d'un champ. |
| [Ancrage](../Proprietes/2510001.md) | La propriété **Ancrage** permet de gérer l'ancrage d'un champ dans une fenêtre, une page ou un état. |
| [CadrageHorizontal](../Proprietes/2510023.md) | La propriété **CadrageHorizontal** permet de connaître et de modifier le cadrage horizontal d'un champ. |
| [CadrageVertical](../Proprietes/2510054.md) | La propriété **CadrageVertical** permet de connaître et de modifier le cadrage vertical d'un champ. |
| [Cadre](../Proprietes/1000018157.md) | La propriété **Cadre** permet de connaître et de modifier les caractéristiques du cadre : <br><br>- d'un champ d'un état ou d'un bloc d'un état.<br><br>- d'un champ d'une fenêtre.<br><br>- d'une cellule d'un champ Table. <br><br><br> |
| [Couleur](../Proprietes/2510071.md) | La propriété **Couleur** permet de connaître ou modifier la couleur de la police de :<br><br>- une variable de type Police.<br><br>- un texte d'un champ Libellé ou calculé d'un état.<br><br>- un texte d'un champ d'une fenêtre.<br><br>- un texte d'un champ d'une page.<br><br><br> |
| [CouleurFond](../Proprietes/2510022.md) | La propriété **CouleurFond** permet de gérer la couleur de fond d'un champ, d'une fenêtre, d'une page ou d'un bloc d'un état. |
| [DroiteAGauche](../Proprietes/1000017228.md) | La propriété **DroiteAGauche** permet de connaître le sens d'écriture réel, effectif sur un champ, une colonne, une fenêtre ou un état. |
| [Etat](../Proprietes/2510039.md) | La propriété **Etat** permet de connaître et de modifier l'état d'affichage d'un élément. |
| [Groupe](../Proprietes/2510049.md) | La propriété **Groupe** permet de : <br><br>- savoir si le champ appartient ou non à un groupe de champs. Si le champ appartient à un ou plusieurs groupes, la propriété **Groupe** permet d'identifier ces groupes de champs.<br><br>- associer un champ à un groupe de champs. <br><br><br> |
| [Hauteur](../Proprietes/2510050.md) | La propriété **Hauteur** permet de connaître et modifier la hauteur :<br><br>- d'un champ ou d'un bloc dans un état.<br><br>- d'un champ (présent dans une fenêtre), d'une fenêtre, d'une ligne d'un champ Table ou Liste (y compris Liste image), d'une option d'un champ Sélecteur ou Interrupteur, d'un haut ou d'un bas de rupture d'un champ Table ou d'un champ Zone répétée.<br><br>- d'un champ (présent dans une page), d'un haut ou bas de rupture d'un champ Table ou Zone répétée. Le champ peut être superposable ou non.<br><br>- d'une cellule de rupture d'un champ Table ou d'un champ Zone répétée.<br><br><br> |
| [HauteurFuture](../Proprietes/2511061.md) | La propriété **HauteurFuture** permet de connaître la hauteur d'un champ auto-extensible dans un état lorsque ce champ sera agrandi. |
| [HauteurInitiale](../Proprietes/2510043.md) | La propriété **HauteurInitiale** permet de :<br><br>- Connaître la hauteur de création d'un champ ou d'une fenêtre.<br><br>- Modifier la hauteur de base utilisée pour l'ancrage des champs "en hauteur".<br><br>- Connaître la hauteur de création d'un champ ou d'un bloc d'un état.<br><br><br> |
| [HauteurPixel](../Proprietes/2511064.md) | La propriété **HauteurPixel** permet de connaître la hauteur d'un champ d'un état en pixels. |
| [Image](../Proprietes/2510034.md) | La propriété **Image** permet de connaître et de modifier l'image de fond d'un champ ou d'un bloc. |
| [Largeur](../Proprietes/2510059.md) | La propriété **Largeur** permet de connaître et modifier la largeur :<br><br>- d'un champ ou d'un bloc dans un état.<br><br>- d'un champ (présent dans une fenêtre), d'une fenêtre, d'une colonne de table ou d'une colonne de liste (y compris liste image).<br><br>- d'un champ (présent dans une page), d'une colonne de table ou d'une liste. Le champ peut être superposable ou non.<br><br><br> |
| [LargeurInitiale](../Proprietes/2510067.md) | La propriété **LargeurInitiale** permet de :<br><br>- Connaître la largeur de création d'un champ ou d'une fenêtre.<br><br>- Modifier la largeur de base utilisée pour l'ancrage des champs "en largeur".<br><br>- Connaître la largeur de création d'un champ d'un état.<br><br><br> |
| [LargeurPixel](../Proprietes/2511052.md) | La propriété **LargeurPixel** permet de connaître la largeur d'un champ d'un état en pixels. |
| [MasqueSaisie](../Proprietes/2510033.md) | La propriété **MasqueSaisie** permet de :<br><br>- connaître le masque de saisie.<br><br>- modifier le masque de saisie (parmi les masques disponibles pour le type).<br><br><br> |
| [Multiligne](../Proprietes/2510105.md) | Dans une fenêtre ou une page, la propriété **Multiligne** permet :<br><br>- de savoir et de modifier l'option "Multiligne" d'un champ de saisie.<br><br>- de savoir et de modifier l'option "Multiligne" d'une colonne d'un champ Table.  <br><br><br><br><br>Dans un état, la propriété **Multiligne** permet de connaître et de modifier l'option "Multiligne" d'un champ d'un état. |
| [NiveauFlou](../Proprietes/1410088348.md) | La propriété **NiveauFlou** permet de connaître et modifier le pourcentage de flou utilisé lors du floutage d'un champ d'un état. |
| [Nom](../Proprietes/2510082.md) | La propriété **Nom** permet de connaître :<br><br>- le nom de la police associée à une variable de type Police.<br><br>- le nom d'un champ, d'un état ou d'un bloc d'un état.<br><br>- le nom d'un champ, d'un groupe de champs ou d'une fenêtre.<br><br>- le nom d'un champ, d'un groupe de champs ou d'une page.<br><br><br> |
| [Note](../Proprietes/2510085.md) | La propriété **Note** permet de :<br><br>- Connaître les notes associées à un champ, à une fenêtre, à une page ou un état.<br><br>- Modifier les notes associées à un champ, à une fenêtre, à une page ou un état.<br><br><br> |
| [Occulté](../Proprietes/1410087964.md) | La propriété **Occulté** permet de : <br><br>- Savoir si un champ d'un état est occulté. <br><br>- Occulter un champ d'un état en le floutant ou en le biffant.<br><br><br> |
| [Plan](../Proprietes/2510094.md) | La propriété **Plan** permet de :<br><br>- Connaître : <br><br>	- le plan actif d'une fenêtre (ou d'un état) ou le plan auquel appartient un champ.<br><br>	- le plan actif d'un élément WEBDEV.<br><br><br><br><br>- Modifier : <br><br>	- le plan actif d'une fenêtre (ou d'un état) ou associer un champ à un autre plan.<br><br>	- le plan actif d'un élément WEBDEV.<br><br><br><br><br><br> |
| [Police](../Proprietes/2510095.md) | La propriété **Police** permet de :<br><br>- Connaître l'identifiant de la police utilisée dans un champ d'un état, d'une page ou d'une fenêtre.<br><br>- Modifier la police utilisée dans un champ d'un état, d'une page ou d'une fenêtre.<br><br><br> |
| [PoliceAlphabet](../Proprietes/2510089.md) | La propriété **PoliceAlphabet** permet de :<br><br>- Connaître l'alphabet actuellement utilisé par les polices écran d'un champ.<br><br>- Modifier l'alphabet utilisé par les polices écran d'un champ.<br><br><br> |
| [PoliceBarrée](../Proprietes/2510091.md) | La propriété **PoliceBarrée** permet de :<br><br>- Savoir si le texte affiché est barré ou non.<br><br>- Barrer ou non le texte affiché dans un champ.<br><br><br> |
| [PoliceCondensée](../Proprietes/2510096.md) | La propriété **PoliceCondensée** permet de :<br><br>- Savoir si les caractères du texte affiché sont condensés ou non.<br><br>- Condenser ou non les caractères du texte affiché.<br><br><br> |
| [PoliceEtendue](../Proprietes/2510104.md) | La propriété **PoliceEtendue** permet de :<br><br>- Savoir si les caractères du texte affiché sont étendus ou non.<br><br>- Étendre ou non les caractères du texte affiché.<br><br><br> |
| [PoliceGras](../Proprietes/2510103.md) | La propriété **PoliceGras** permet de :<br><br>- Savoir si le texte affiché est en gras ou non.<br><br>- Modifier la graisse du texte.<br><br><br> |
| [PoliceItalique](../Proprietes/2510102.md) | La propriété **PoliceItalique** permet de :<br><br>- Savoir si le texte affiché est en italique ou non.<br><br>- Modifier la mise en forme italique du texte.<br><br><br> |
| [PoliceLarge](../Proprietes/2510101.md) | La propriété **PoliceLarge** permet de :<br><br>- Savoir si les caractères du texte affiché sont élargis ou non.<br><br>- Élargir ou non les caractères du texte affiché.<br><br><br> |
| [PoliceNom](../Proprietes/2510100.md) | La propriété **PoliceNom** permet de :<br><br>- Connaître la police utilisée dans un champ.<br><br>- Modifier la police utilisée dans un champ.<br><br><br> |
| [PoliceSoulignée](../Proprietes/2510099.md) | La propriété **PoliceSoulignée** permet de :<br><br>- Savoir si le texte affiché est souligné ou non.<br><br>- Souligner ou non le texte.<br><br><br> |
| [PoliceTaille](../Proprietes/2510072.md) | La propriété **PoliceTaille** permet de :<br><br>- Connaître la taille de la police utilisée dans un champ.<br><br>- Modifier la taille de la police utilisée dans un champ.<br><br><br> |
| [SensEcriture](../Proprietes/2513020.md) | La propriété **SensEcriture** permet de connaître et de modifier le sens d'écriture utilisé dans une fenêtre, un état ou dans un champ (utile pour la gestion des langues de type arabe ou hébreux). |
| [TauxAncrageBas](../Proprietes/2510113.md) | La propriété **TauxAncrageBas** permet de :<br><br>- Connaître le taux d'ancrage en déplacement d'un champ sur l'axe vertical.<br><br>- Modifier le taux d'ancrage en déplacement d'un champ sur l'axe vertical.<br><br><br> |
| [TauxAncrageDroite](../Proprietes/2510114.md) | La propriété **TauxAncrageDroite** permet de :<br><br>- Connaître le taux d'ancrage en déplacement d'un champ sur l'axe horizontal.<br><br>- Modifier le taux d'ancrage en déplacement d'un champ sur l'axe horizontal.<br><br><br> |
| [TauxAncrageHauteur](../Proprietes/2510119.md) | La propriété **TauxAncrageHauteur** permet de :<br><br>- Connaître le taux d'ancrage en hauteur d'un champ.<br><br>- Modifier le taux d'ancrage en hauteur d'un champ.<br><br><br> |
| [TauxAncrageLargeur](../Proprietes/2510120.md) | La propriété **TauxAncrageLargeur** permet de :<br><br>- Connaître le taux d'ancrage en agrandissement d'un champ (ou d'une colonne de table) sur l'axe horizontal.<br><br>- Modifier le taux d'ancrage en agrandissement d'un champ (ou d'une colonne de table) sur l'axe horizontal.<br><br><br> |
| [TitreNote](../Proprietes/2510121.md) | La propriété **TitreNote** permet de :<br><br>- Connaître le titre des notes utilisé pour la documentation technique d'un champ, d'une fenêtre, d'une page ou d'un état. <br><br>- Modifier le titre des notes utilisé pour la documentation technique d'un champ, d'une fenêtre, d'une page ou d'un état.<br><br><br> |
| [Traitement](../Proprietes/1000020878.md) | La propriété **Traitement** permet de modifier les traitements à exécuter par les événements disponibles sur un champ, une fenêtre, une page, un état, ... |
| [Valeur](../Proprietes/2510130.md) | La propriété **Valeur** permet de :<br><br>- Connaître et modifier la valeur d'un champ d'un état.<br><br>- Connaître et modifier la valeur d'un champ ou le titre d'une fenêtre.<br><br>- Connaître l'identifiant d'un flux. <br><br>- Connaître et modifier la valeur d'un champ ou le titre d'une page.<br><br><br> |
| [Vide](../Proprietes/2510126.md) | La propriété **Vide** permet de savoir si :<br><br>- un champ Zone répétée, Table, Liste ou Combo est vide. | [Visible](../Proprietes/2510138.md) | La propriété **Visible** permet de :<br><br>- Savoir si un élément est visible.<br><br>- Rendre un élément visible ou invisible.<br><br><br> |


## Propriétés associées à une fenêtre
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les propriétés associées à une fenêtre sont les suivantes : 



|   |   |
| --- | --- |
| [AffichageActif](../Proprietes/2510015.md) | La propriété **AffichageActif** permet de :<br><br>- savoir si l'affichage d'un champ ou d'une fenêtre est activé ou non. Si l'affichage est activé, le contenu du champ est rafraîchi régulièrement.<br><br>- activer ou non l'affichage d'un champ ou d'une fenêtre.<br><br><br> |
| [Alias](../Proprietes/2510004.md) | La propriété **Alias** permet de : <br><br>- connaître ou de modifier l'alias d'une fenêtre (WINDEV et WINDEV Mobile).<br><br>- connaître ou de modifier l'alias d'un volet d'onglet dynamique (WINDEV).<br><br>- connaître le nom interne HTML / Javascript d'un champ (WEBDEV).<br><br><br> |
| [AutoriseMasquageActionBar](../Proprietes/1000021893.md) | La propriété **AutoriseMasquageActionBar** permet de gérer le masquage automatique de l'Action Bar. |
| [BarreDeMessage](../Proprietes/2510007.md) | La propriété **BarreDeMessage** permet de :<br><br>- savoir si la barre de message est affichée ou non<br><br>- afficher ou non la barre de message d'une fenêtre.<br><br><br> |
| [ClicDroit](../Proprietes/2510069.md) | La propriété **ClicDroit** permet de connaître et de modifier le nom du champ Bouton qui sera exécuté lors d'un clic droit sur un objet. |
| [CompacteBarreOutils](../Proprietes/1000018693.md) | La propriété **CompacteBarreOutils** permet de :<br><br>- savoir si les barres d'outils de la fenêtre sont compactées.<br><br>- compacter ou non les barres d'outils.<br><br><br> |
| [CopieEcranAutorisée](../Proprietes/1000025336.md) | La propriété **CopieEcranAutorisée** permet de : <br><br>- Savoir si la copie d'écran est autorisée sur une fenêtre, <br><br>- Autoriser ou non la copie d'écran sur une fenêtre.<br><br><br> |
| [Couleur](../Proprietes/2510071.md) | La propriété **Couleur** permet de connaître ou modifier la couleur de la police de :<br><br>- une variable de type Police.<br><br>- un texte d'un champ Libellé ou calculé d'un état.<br><br>- un texte d'un champ d'une fenêtre.<br><br>- un texte d'un champ d'une page.<br><br><br> |
| [CouleurBarreSystème](../Proprietes/1000022485.md) | La propriété **CouleurBarreSystème** permet de connaître ou de modifier la couleur de la barre système des fenêtres maximisées mobiles. |
| [CouleurFond](../Proprietes/2510022.md) | La propriété **CouleurFond** permet de gérer la couleur de fond d'un champ, d'une fenêtre, d'une page ou d'un bloc d'un état. |
| [CouleurTexteAutomatique](../Proprietes/1000022074.md) | La propriété **CouleurTexteAutomatique** permet de connaître et de modifier la couleur à utiliser pour tous les libellés des éléments utilisant la couleur "automatique". |
| [CurseurSouris](../Proprietes/2510026.md) | La propriété **CurseurSouris** permet de connaître et de modifier le curseur de survol défini pour un champ, une fenêtre ou une page. |
| [DéplaçableParLeFond](../Proprietes/2510149.md) | La propriété **DéplaçableParLeFond** permet de :<br><br>- savoir si la fenêtre est déplaçable par le fond,<br><br>- rendre ou non la fenêtre déplaçable par le fond.<br><br><br> |
| [DroiteAGauche](../Proprietes/1000017228.md) | La propriété **DroiteAGauche** permet de connaître le sens d'écriture réel, effectif sur un champ, une colonne, une fenêtre ou un état. |
| [Etat](../Proprietes/2510039.md) | La propriété **Etat** permet de connaître et de modifier l'état d'affichage d'un élément. |
| [EtatInitial](../Proprietes/2510063.md) | La propriété **EtatInitial** permet de connaître l'état initial d'un champ. |
| [Extra](../Proprietes/1410089327.md) | La propriété **Extra** permet de :<br><br>- Connaître les informations supplémentaires associées à un champ, à une fenêtre, ou à une page.<br><br>- Modifier les informations supplémentaires associées à un champ, à une fenêtre ou à une page.<br><br><br> |
| [FenEditionMode](../Proprietes/1000021824.md) | La propriété **FenEditionMode** permet de connaître et de modifier les actions autorisées sur la fenêtre et ses champs lorsque la fenêtre passe en mode "Modification de l'UI par l'utilisateur final". |
| [FichierParcouru](../Proprietes/2510040.md) | La propriété **FichierParcouru** permet de connaître et de modifier la source de données utilisée pour l'affichage des enregistrements dans les champs basés sur un fichier de données. |
| [Grisé](../Proprietes/1000019648.md) | La propriété **Grisé** permet de : <br><br>- Savoir si un champ, un groupe de champs ou une fenêtre est grisé ou non. <br><br>- Griser ou dégriser un champ, un groupe de champs ou une fenêtre.<br><br><br> |

Le champ peut être superposable ou non.<br><br>- d'une cellule de rupture d'un champ Table ou d'un champ Zone répétée.<br><br><br> |

| [HauteurInitiale](../Proprietes/2510043.md) | La propriété **HauteurInitiale** permet de :<br><br>- Connaître la hauteur de création d'un champ ou d'une fenêtre.<br><br>- Modifier la hauteur de base utilisée pour l'ancrage des champs "en hauteur".<br><br>- Connaître la hauteur de création d'un champ ou d'un bloc d'un état.<br><br><br> |

| [HauteurTitre](../Proprietes/1000021058.md) | La propriété **HauteurTitre** permet de connaître et de modifier : <br><br>- la hauteur du titre des colonnes d'un champ Table ou Table hiérarchique. <br><br>- la hauteur de la barre de titre d'un champ Panneau dockable.<br><br>- la hauteur de la barre de titre d'une fenêtre possédant une barre de titre personnalisée.<br><br><br> |
| [HauteurUtile](../Proprietes/1000021504.md) | La propriété **HauteurUtile** permet de connaître et de modifier la hauteur utile d'une fenêtre interne, d'une fenêtre ou d'un superchamp avec ascenseurs. |
| [Icone](../Proprietes/1000019344.md) | La propriété **Icone** permet de :<br><br>- Connaître l'icône affichée dans la barre de titre d'une fenêtre ou d'un champ Panneau dockable. <br><br>- Modifier l'icône de la barre de titre d'une fenêtre ou d'un champ Panneau dockable.<br><br><br> |

| [LargeurMax](../Proprietes/2510068.md) | La propriété **LargeurMax** permet de connaître et de modifier la largeur maximale d'un champ ou d'une fenêtre. |

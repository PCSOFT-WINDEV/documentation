
## Propriétés associées aux Action Bar
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000084"></a>
Une "Action Bar" peut être manipulée en WLangage dans une fenêtre avec les propriétés suivantes.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Propriétés disponibles dans une fenêtre
<a name="proprietes_disponibles_dans_une_fenetre_ELTTEXTE000108"></a>


### Action Bar : Propriétés disponibles dans une fenêtre
<a name="action_bar_proprietes_disponibles_dans_une_fenetre_ELTPARAGRAPHE000017"></a>Les différentes propriétés associées aux "Action Bar" sont les suivantes :



|   |   |
| --- | --- |
| [Couleur](../Proprietes/2510071.md) | La propriété **Couleur** permet de connaître ou modifier la couleur de la police de :<br><br>- une variable de type Police.<br><br>- un texte d'un champ Libellé ou calculé d'un état.<br><br>- un texte d'un champ d'une fenêtre.<br><br>- un texte d'un champ d'une page.<br><br><br> |
| [CouleurFond](../Proprietes/2510022.md) | La propriété **CouleurFond** permet de gérer la couleur de fond d'un champ, d'une fenêtre, d'une page ou d'un bloc d'un état. |
| [Extra](../Proprietes/1410089327.md) | La propriété **Extra** permet de :<br><br>- Connaître les informations supplémentaires associées à un champ, à une fenêtre, ou à une page.<br><br>- Modifier les informations supplémentaires associées à un champ, à une fenêtre ou à une page.<br><br><br> |
| [Image](../Proprietes/2510034.md) | La propriété **Image** permet de connaître et de modifier l'image de fond d'un champ ou d'un bloc. |
| [ImageFond](../Proprietes/2510056.md) | La propriété **ImageFond** permet de connaître et de modifier l'image de fond d'un élément : fenêtre, page, champ ou état. |
| [Note](../Proprietes/2510085.md) | La propriété **Note** permet de :<br><br>- Connaître les notes associées à un champ, à une fenêtre, à une page ou un état.<br><br>- Modifier les notes associées à un champ, à une fenêtre, à une page ou un état.<br><br><br> |
| [Opacité](../Proprietes/2514006.md) | La propriété **Opacité** permet de connaître et définir le pourcentage d'opacité d'une fenêtre ou des champs présents dans une fenêtre ou une page. |
| [Style](../Proprietes/1000021472.md) | La propriété **Style** permet de récupérer ou d'affecter le style complet d'un champ (y compris les surcharges). |
| [TitreNote](../Proprietes/2510121.md) | La propriété **TitreNote** permet de :<br><br>- Connaître le titre des notes utilisé pour la documentation technique d'un champ, d'une fenêtre, d'une page ou d'un état. <br><br>- Modifier le titre des notes utilisé pour la documentation technique d'un champ, d'une fenêtre, d'une page ou d'un état.<br><br><br> |
| [Traitement](../Proprietes/1000020878.md) | La propriété **Traitement** permet de modifier les traitements à exécuter par les événements disponibles sur un champ, une fenêtre, une page, un état, ... |
| [Valeur](../Proprietes/2510130.md) | La propriété **Valeur** permet de :<br><br>- Connaître et modifier la valeur d'un champ d'un état.<br><br>- Connaître et modifier la valeur d'un champ ou le titre d'une fenêtre.<br><br>- Connaître l'identifiant d'un flux. <br><br>- Connaître et modifier la valeur d'un champ ou le titre d'une page.<br><br><br> |
| [ValeurRecherche](../Proprietes/1000021602.md) | La propriété **ValeurRecherche** permet de connaître et modifier la valeur du champ de recherche dans une Action Bar. |
| [Visible](../Proprietes/2510138.md) | La propriété **Visible** permet de :<br><br>- Savoir si un élément est visible.<br><br>- Rendre un élément visible ou invisible.<br><br><br> |
| [VisibleAvecAnimation](../Proprietes/1410087232.md) | La propriété **VisibleAvecAnimation** permet de rendre un champ visible ou invisible en jouant l'animation "Apparition/Disparition". |
| [Volet](../Proprietes/1000021098.md) | La propriété **Volet** permet de : <br><br>- Connaître les volets d'onglet associés à un champ. <br><br>- Définir les volets d'onglet associés à un champ.<br><br><br> |




## Propriétés associées aux séries d'un graphe
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les différentes propriétés utilisables avec les séries d'un "Graphe" sont les suivantes :



|   |   |
| --- | --- |
| [AxeSecondaire](../Proprietes/1000019187.md) | La propriété **AxeSecondaire** permet de : <br><br>- savoir si une série d'un graphe est placée sur l'axe secondaire. <br><br>- placer une série d'un graphe sur l'axe secondaire.<br><br><br> |
| [BorneMax](../Proprietes/2510009.md) | La propriété **BorneMax** permet de :<br><br>- connaître et de modifier la borne maximale d'un état basé sur un fichier de données. <br><br>- connaître la borne maximale du filtre en cours.<br><br>- connaître la valeur maximale d'une série d'un graphe.<br><br>- connaître et modifier la valeur maximale de certains types de champs.<br><br><br> |
| [BorneMin](../Proprietes/2510008.md) | La propriété **BorneMin** permet de :<br><br>- connaître et de modifier la borne minimale d'un état basé sur un fichier de données. <br><br>- connaître la borne minimale du filtre en cours.<br><br>- connaître la valeur minimale d'une série d'un graphe.<br><br>- connaître et modifier la valeur minimale de certains types de champs. <br><br><br> |
| [CouleurSérie](../Proprietes/2510145.md) | La propriété **CouleurSérie** permet de :<br><br>- Connaître la couleur d'une série d'un graphe.<br><br>- Modifier la couleur d'une série d'un graphe. <br><br><br> |
| [Epaisseur](../Proprietes/2510144.md) | La propriété **Epaisseur** permet de :<br><br>- Connaître l'épaisseur du trait d'une série (uniquement pour les graphes de type Courbe). <br><br>- Modifier l'épaisseur du trait d'une série (uniquement pour les graphes de type Courbe).<br><br><br> |
| [Légende](../Proprietes/2510143.md) | La propriété **Légende** permet de :<br><br>- Connaître l'étiquette d'une série.<br><br>- Modifier l'étiquette d'une série.<br><br><br> |
| [MAJAutomatique](../Proprietes/2510146.md) | La propriété **MAJAutomatique** permet de :<br><br>- Savoir si les données de la séries sont automatiquement recalculées à chaque appel de la fonction [grDessine](../WDLang3/3042026.md)<br><br>- Modifier le mode de recacul des données de la série.<br><br><br> |
| [Source](../Proprietes/2510147.md) | La propriété **Source** peut être utilisée sur :<br><br>- **un champ Graphe** : dans ce cas, la propriété **Source** permet de connaître la source de la série : Colonne de table, liste de valeurs, ...<br><br>- **une connexion** : dans ce cas, la propriété **Source** permet de connaître  ou définir la source d'une connexion (HFSQL Client/Serveur, OLE DB, ...).<br><br><br><br><br> |
| [SourceDescription](../Proprietes/2510148.md) | La propriété **SourceDescription** permet de connaître le nom de l'élément utilisé comme source de données d'une série. |

## Présentation du champ IFrame
<a name="presentation_champ_iframe_ELTTEXTE000123"></a>
IFrame est un concept HTML. Ce concept indique que un contenu Web indépendant est inséré dans une zone définie d'une page. Il est ainsi possible d'afficher dans une page HTML :

- soit une page HTML d'un site extérieur au site WEBDEV en cours.

- soit une page HTML du site en cours.

## Créer un champ IFrame
<a name="creer_champ_iframe_ELTTEXTE000147"></a>


### Créer un champ IFrame
<a name="creer_champ_iframe_ELTPARAGRAPHE000021"></a>Pour créer un champ IFrame :

1. Sous le volet "Création", dans le groupe "Intégration", cliquez sur "iFrame".

2. Cliquez dans la page à la position où le champ doit être créé. Le champ apparaît en création.




Pour afficher les caractéristiques du champ, sélectionnez l'option "Description" dans le menu contextuel du champ.
<a name="NOTE2_2"></a>


### Associer une page à un champ IFrame
<a name="associer_une_page_champ_iframe_ELTPARAGRAPHE000045"></a>Pour associer une page à un champ IFrame :

1. Affichez la fenêtre de description du champ IFrame (sélectionnez l'option "Description" dans le menu contextuel du champ).

2. Dans l'onglet "Général", sélectionnez la page affichée dans le champ :

	- soit une page du projet en cours.

	- soit saisissez l'adresse d'un site ou d'une page d'un site (par exemple [www.pcsoft.fr](http://www.pcsoft.fr)) ainsi que le mode de sécurité voulu.




3. Validez.

## Manipuler un champ IFrame
<a name="manipuler_champ_iframe_ELTTEXTE000177"></a>


### Changer la cible du champ IFrame
<a name="changer_cible_champ_iframe_ELTPARAGRAPHE000064"></a>Le changement de cible d'un champ IFrame peut se faire très simplement par exemple par l'intermédiaire d'un champ Bouton ou d'un champ Lien.

Ce changement peut être réalisé uniquement si la cible du champ IFrame appartient au site en cours.

Il suffit de créer un champ Bouton ou un champ Lien avec les caractéristiques suivantes :

- Type : "Afficher une page du site". Il suffit de sélectionner la page à afficher.

- Destination : "Dans un IFrame XXX". Il suffit de sélectionner le nom du champ IFrame.

- Lors de l'action : "Ne rien envoyer au serveur". 




**Remarque** : Si la cible du champ IFrame n'appartient pas au site en cours, le navigateur Internet ne permet pas de modifier cette cible (pour des raisons de sécurité).

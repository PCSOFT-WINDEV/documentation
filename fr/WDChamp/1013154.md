
## Souris aimantée : déplacer automatiquement le curseur sur le bouton par défaut
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000083"></a>
Le curseur de la souris peut se positionner automatiquement sur le bouton par défaut lors de l'ouverture de la fenêtre.

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=PositionneSouris.GIF)


Cette fonctionnalité est très utile dans les fenêtres dans lesquelles l'utilisateur clique généralement rapidement sur le bouton proposé par défaut.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Mise en place
<a name="mise_place_ELTTEXTE000107"></a>
Pour déplacer automatiquement le curseur de la souris sur le bouton par défaut lors de l'ouverture de la fenêtre :

1. Affichez la fenêtre de description de la fenêtre (option "Description" du menu contextuel).

2. Affichez l'onglet "Détail" et sélectionnez l'option "Déplacer le pointeur de la souris sur le bouton par défaut lors de l'ouverture de la fenêtre".

3. Validez la fenêtre de description.




<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Bouton par défaut
<a name="bouton_par_defaut_ELTTEXTE000131"></a>
Le bouton sur lequel le curseur de la souris se déplace doit être de type "Validation".

Si la fenêtre contient plusieurs boutons de type "Validation", le curseur de la souris se déplacera sur le premier bouton créé.

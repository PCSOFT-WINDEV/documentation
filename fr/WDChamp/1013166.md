## Champ Lien
<a name="champ_lien_ELTTEXTE000163"></a>
Le champ Lien permet de :

- .

- créer un lien dans un état imprimé au format HTML.




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Champ Lien dans une page
<a name="champ_lien_dans_une_page_ELTTEXTE000187"></a>


### Créer un champ Lien
<a name="creer_champ_lien_ELTPARAGRAPHE000031"></a>Pour créer un champ de type Lien : 

1. Sous le volet "Création", dans le groupe "Champs usuels", cliquez sur "Lien".

2. Cliquez dans la page à la position où le champ doit être créé. Le champ apparaît en création.




Pour afficher les caractéristiques du champ, sélectionnez l'option "Description" dans le menu contextuel du champ.

<a name="NOTE3_1"></a>


## Champ Lien dans un état
<a name="champ_lien_dans_etat_ELTTEXTE000217"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 

### Présentation
<a name="presentation_ELTPARAGRAPHE000059"></a>Le champ "Lien URL" permet d'insérer des liens dans un état. Il est ainsi possible d'insérer par exemple :

- un lien vers une adresse email,

- un lien vers une adresse de site Internet,

- ...


Le champ "Lien URL" est actif uniquement si l'état est imprimé dans un fichier HTML. Dans les autres cas, le champ "Lien URL" est visible mais inactif.

### Créer un champ Lien URL
<a name="creer_champ_lien_url_ELTPARAGRAPHE000070"></a>Pour créer un champ de type Lien URL :

1. Sous le volet "Création", dans le groupe "Interactions", cliquez sur "Lien".

2. Cliquez dans l'état à la position où le champ doit être créé. Le champ apparaît en création.




Pour afficher les caractéristiques du champ, sélectionnez l'option "Description" dans le menu contextuel du champ.

<a name="NOTE3_2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 

### Action associée au champ Lien présent dans l'état
<a name="action_associee_champ_lien_present_dans_etat_ELTPARAGRAPHE000096"></a>L'action associée au champ "Lien" est défini dans la fenêtre de description du champ. Cette action peut provenir :

- soit d'une rubrique.

- soit d'une adresse URL. Dans ce cas, le bouton "..." situé à droite du champ "URL (fixe)" permet de définir une action spécifique.


![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Actionlien%20-%20HC%20N%B0003.gif)
Il est ainsi possible de :

- afficher une page d'un site,

- se connecter à un serveur FTP,

- télécharger un fichier,

- d'envoyer un mail, ...




Remarques : 

- Toutes ces opérations sont également réalisables par programmation.

- La copie d'un champ Lien présent dans l'éditeur d'états colle la balise ainsi que le texte et le champ dans le presse-papiers. Cela permet par exemple de copier directement le lien dans une page HTML.

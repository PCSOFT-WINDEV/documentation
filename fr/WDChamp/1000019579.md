
## Manipuler un champ DotNet WinForm par programmation
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000086"></a>
WINDEV permet de manipuler un champ DotNet (WinForm) par programmation. Il suffit d'utiliser directement la variable du champ dans le code.

La variable du champ DotNet (WinForm) correspond au nom du champ DotNet (WinForm).



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Utiliser des propriétés du champ
<a name="utiliser_des_proprietes_champ_ELTTEXTE000110"></a>
Lorsque le champ DotNet (WinForm) est créé, l'éditeur de code propose la complétion sur toutes les propriétés disponibles du champ fourni par l'assemblage. 


```wl
// DOTNET_Bouton_NEt1 = Nom du champ

// Modification du libellé
DOTNET_Bouton_NEt1.Text = "Bouton .NET"
DOTNET_Bouton_NEt1.BackColor = Color.FromArgb(111,87,235)
```




<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Gérer les délégués
<a name="gerer_les_delegues_ELTTEXTE000134"></a>
Pour appeler une procédure à partir du champ .NET, il suffit d'utiliser la fonction [DotNetDélégué](../WDLang3/1000017052.md). 


```wl
// DOTNET_Bouton_NEt1 = nom du champ
// Ajout d'un délégué sur le clic
DOTNET_Bouton_NEt1.add_Click(DotNetDélégué(CallbackWinDev,"System.EventHandler"))
```

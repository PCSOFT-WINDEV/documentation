


## Manipuler une barre de message sous l'éditeur de fenêtres
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000168"></a>
Cette page d'aide présente les différentes manipulations possibles d'une barre de message :

- ajouter une case.

- supprimer une case.

- modifier l'ordre des cases.




Toutes ces manipulations se font sous l'éditeur de fenêtres :

- grâce au menu contextuel de la barre de message (clic droit sur la barre de message affichée dans la fenêtre).<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=MenuCtxBarreMessage.gif)


- depuis le ruban : sous le volet "Fenêtre", dans le groupe "Barres et menus", déroulez "Barre de message" et sélectionnez "Editer".




**Rappel** : Pour afficher une barre de message dans la fenêtre en cours, effectuez une des manipulations suivantes :

- sélectionnez l'option "Barre de message" dans l'onglet "Style" de la description de la fenêtre.

- sous le volet "Fenêtre", dans le groupe "Barres et menus", déroulez "Barre de message" et sélectionnez "Avec barre de message".




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Ajouter une case
<a name="ajouter_une_case_ELTTEXTE000192"></a>
Pour ajouter une case dans la barre de message de la fenêtre en cours :

- soit sélectionnez l'option "Insérer une case" dans le menu contextuel de la barre de message.

- soit cliquez sur le bouton "Ajouter" ou "Insérer" depuis la fenêtre de description de la barre de message.


**Remarque** : Par défaut, la première case créée affiche le message d'information lié au champ en cours ou spécifié par programmation.

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Supprimer une case
<a name="supprimer_une_case_ELTTEXTE000216"></a>
Pour supprimer une case dans la barre de message de la fenêtre en cours :

1. Sélectionnez la case à supprimer.

2. Supprimez la case en effectuant une des manipulations suivantes :

	- Affichez le menu contextuel de la case et sélectionnez l'option "Supprimer une case".

	- Affichez la fenêtre de description de la barre de message : sous le volet "Fenêtre", dans le groupe "Barres et menus", déroulez "Barre de message" et sélectionnez "Editer". La case en cours est sélectionnée. 
			Cliquez sur le bouton "Supprimer".







<a name="NOTE4"></a>
<a name="NOTE4_1"></a>


## Modifier l'ordre des cases
<a name="modifier_ordre_des_cases_ELTTEXTE000240"></a>
Pour modifier l'ordre des cases dans la barre de message de la fenêtre en cours :

1. Sélectionnez la case dont la position doit être modifiée.

2. Déplacez la case :

	- soit grâce au menu contextuel de la case (option "Déplacer à gauche" ou "Déplacer à droite")

	- soit par la fenêtre de description de la barre de message. 

		- Affichez la fenêtre de description : sous le volet "Fenêtre", dans le groupe "Barres et menus", déroulez "Barre de message" et sélectionnez "Editer". La case en cours est sélectionnée.

		- Cliquez sur le bouton ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=BtnDeplaceGauche.gif) ou ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=BtnDeplaceDroite.gif).

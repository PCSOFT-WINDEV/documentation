


## Propriétés associées aux champs d'un état
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>






Les différentes propriétés utilisables sur les champs d'un état sont les suivantes :



|   |   |
| --- | --- |
| [Altitude](../Proprietes/2510002.md) | La propriété **Altitude** permet de gérer l'altitude d'un champ. |
| [Ancrage](../Proprietes/2510001.md) | La propriété **Ancrage** permet de gérer l'ancrage d'un champ dans une fenêtre, une page ou un état. |
| [CadrageHorizontal](../Proprietes/2510023.md) | La propriété **CadrageHorizontal** permet de connaître et de modifier le cadrage horizontal d'un champ. |
| [CadrageVertical](../Proprietes/2510054.md) | La propriété **CadrageVertical** permet de connaître et de modifier le cadrage vertical d'un champ. |
| [Cadre](../Proprietes/1000018157.md) | La propriété **Cadre** permet de connaître et de modifier les caractéristiques du cadre : <br><br>- d'un champ d'un état ou d'un bloc d'un état.<br><br>- d'un champ d'une fenêtre.<br><br>- d'une cellule d'un champ Table. <br><br><br> |
| [Couleur](../Proprietes/2510071.md) | La propriété **Couleur** permet de connaître ou modifier la couleur de la police de :<br><br>- une variable de type Police.<br><br>- un texte d'un champ Libellé ou calculé d'un état.<br><br>- un texte d'un champ d'une fenêtre.<br><br>- un texte d'un champ d'une page.<br><br><br> |
| [CouleurFond](../Proprietes/2510022.md) | La propriété **CouleurFond** permet de gérer la couleur de fond d'un champ, d'une fenêtre, d'une page ou d'un bloc d'un état. |
| [DateDébutPériodeAffichée](../Proprietes/1000020183.md) | La propriété **DateDébutPériodeAffichée** permet de connaître la date de début de la période affichée dans un champ Planning ou Agenda d'un état. |
| [DateFinPériodeAffichée](../Proprietes/1000020184.md) | La propriété **DateFinPériodeAffichée** permet de connaître la date de fin de la période affichée dans un champ Planning ou Agenda d'un état. |
| [DeviseAffichage](../Proprietes/2510030.md) | La propriété **DeviseAffichage** permet de connaître et modifier la devise utilisée lors de l'affichage et/ou de la saisie dans un champ de type "Monétaire + Euro". |
| [DeviseMémoire](../Proprietes/2510013.md) | La propriété **DeviseMémoire** permet de connaître et modifier la devise utilisée lors de la manipulation d'un champ de type "Monétaire + Euro" par programmation. |
| [DroiteAGauche](../Proprietes/1000017228.md) | La propriété **DroiteAGauche** permet de connaître le sens d'écriture réel, effectif sur un champ, une colonne, une fenêtre ou un état. |

| [FeuilleImprimée](../Proprietes/1000021814.md) | La propriété **FeuilleImprimée** permet de connaître et de modifier la feuille à imprimer dans un champ Tableur présent dans un état. |
| [GranularitéQuadrillage](../Proprietes/1000020107.md) | La propriété **GranularitéQuadrillage** permet de connaître et de modifier la taille de la grille utilisée pour l'affichage du quadrillage dans un champ Planning ou un champ Agenda d'un état. |
| [Groupe](../Proprietes/2510049.md) | La propriété **Groupe** permet de : <br><br>- savoir si le champ appartient ou non à un groupe de champs. Si le champ appartient à un ou plusieurs groupes, la propriété **Groupe** permet d'identifier ces groupes de champs.<br><br>- associer un champ à un groupe de champs. <br><br><br> |
| [Hauteur](../Proprietes/2510050.md) | La propriété **Hauteur** permet de connaître et modifier la hauteur :<br><br>- d'un champ ou d'un bloc dans un état.<br><br>- d'un champ (présent dans une fenêtre), d'une fenêtre, d'une ligne d'un champ Table ou Liste (y compris Liste image), d'une option d'un champ Sélecteur ou Interrupteur, d'un haut ou d'un bas de rupture d'un champ Table ou d'un champ Zone répétée.<br><br>- d'un champ (présent dans une page), d'un haut ou bas de rupture d'un champ Table ou Zone répétée. Le champ peut être superposable ou non.<br><br>- d'une cellule de rupture d'un champ Table ou d'un champ Zone répétée.<br><br><br> |
| [HauteurFuture](../Proprietes/2511061.md) | La propriété **HauteurFuture** permet de connaître la hauteur d'un champ auto-extensible dans un état lorsque ce champ sera agrandi. |
| [HauteurInitiale](../Proprietes/2510043.md) | La propriété **HauteurInitiale** permet de :<br><br>- Connaître la hauteur de création d'un champ ou d'une fenêtre.<br><br>- Modifier la hauteur de base utilisée pour l'ancrage des champs "en hauteur".<br><br>- Connaître la hauteur de création d'un champ ou d'un bloc d'un état.<br><br><br> |
| [HauteurPixel](../Proprietes/2511064.md) | La propriété **HauteurPixel** permet de connaître la hauteur d'un champ d'un état en pixels. |
| [HTML](../Proprietes/2511040.md) | La propriété **HTML** permet de : <br><br>- savoir si l'état en cours d'impression s'imprime en mode HTML. <br><br>- gérer le type HTML d'un champ Libellé présent dans un état.<br><br><br> |
| [Image](../Proprietes/2510034.md) | La propriété **Image** permet de connaître et de modifier l'image de fond d'un champ ou d'un bloc. |

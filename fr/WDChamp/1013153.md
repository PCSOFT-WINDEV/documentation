


## Manipuler un champ Interrupteur par programmation
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000353"></a>
WINDEV, WINDEV Mobile et WEBDEV permettent de manipuler un champ Interrupteur directement par programmation. Il suffit d'utiliser directement la variable du champ Interrupteur dans le code.

La variable du champ Interrupteur correspond au nom du champ Interrupteur. Cette variable est de type entier.

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) **Remarque** : Plusieurs fonctions WLangage permettent d'ajouter, d'insérer ou de supprimer des options dans un champ Interrupteur. Pour plus de détails, consultez [Fonctions de gestion des champs Interrupteur](../WDLang1/1000020613.md).

## Initialiser les différentes options d'un champ Interrupteur
<a name="initialiser_les_differentes_options_champ_interrupteur_ELTTEXTE000377"></a>
Par défaut, à l'affichage d'un champ Interrupteur, aucune option n'est cochée.

**Pour cocher une option d'un champ Interrupteur**, vous pouvez utiliser :

- l'affectation directe : 
	
	```txt
	<Champ Interrupteur>[Indice de l'option] = 1 
	
	<Champ Interrupteur>[Indice de l'option] = Vrai
	```


- la propriété [Valeur](../Proprietes/2510130.md) : 
	
	```txt
	<Champ Interrupteur>[Indice de l'option].Valeur = Vrai
	
	
	<Champ Interrupteur>[Indice de l'option].Valeur = 1
	```



**Pour décocher une option d'un champ Interrupteur**, vous pouvez utiliser :

- l'affectation directe : 
	
	```txt
	<Champ Interrupteur>[Indice de l'option] = 0 
	
	<Champ Interrupteur>[Indice de l'option] = Faux
	```


- la propriété [Valeur](../Proprietes/2510130.md). 
	
	```txt
	<Champ Interrupteur>[Indice de l'option].Valeur = 0 
	
	<Champ Interrupteur>[Indice de l'option].Valeur = Faux
	```



**Remarque** : Si le champ est associé à une rubrique d'un fichier de données, l'affectation est automatiquement réalisée en utilisant : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) la fonction [FichierVersEcran](../WDLang4/3044210.md).


Pour plus de détails, consultez [Lier un champ Interrupteur à une rubrique](../WDChamp/1013152.md).

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) **Si l'état indéterminé d'une option est géré** (onglet "Détail" de la description du champ ou propriété [TroisEtats](../Proprietes/2510124.md)), pour initialiser l'option dans le mode indéterminé, utilisez la syntaxe suivante :

```txt
<Champ Interrupteur>[Indice de l'option] = -1
```

## Récupérer la valeur d'une option d'un champ Interrupteur
<a name="recuperer_valeur_une_option_champ_interrupteur_ELTTEXTE000401"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) 

### En WINDEV et WINDEV Mobile
<a name="windev_windev_mobile_ELTPARAGRAPHE000095"></a>Pour récupérer la valeur d'une option, utilisez la syntaxe suivante :

```txt
<Valeur_Option> = <Champ Interrupteur>[Indice de l'option]
```

Dans ce cas, le paramètre &lt;Valeur_Option&gt; est un booléen qui vaut :

- Vrai (1) si la case est cochée,

- Faux (0) si la case n'est pas cochée.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png) -1 si la case est indéterminée (si cet état est géré pour l'option).




**Remarque** : Si le champ est associé à une rubrique d'un fichier de données, la récupération de la valeur saisie est automatiquement réalisée en utilisant la fonction [EcranVersFichier](../WDLang4/3044146.md). Pour plus de détails, consultez [Lier un champ Interrupteur à une rubrique](../WDChamp/1013152.md).

## Modifier les couleurs d'un champ Interrupteur par ligne
<a name="modifier_les_couleurs_champ_interrupteur_par_ligne_ELTTEXTE000431"></a>
Pour modifier les couleurs d'un champ Interrupteur par ligne, il est possible d'utiliser la syntaxe suivante :

```txt
<Champ Interrupteur>.Libellé = gStylo(<Couleur option 1>) + <Libellé option 1> + TAB + ...
gStylo(<Couleur option N>) + <Libellé option N>
```

Le paramètre &lt;Couleur option&gt; correspond :

- soit une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)),

- soit une couleur TSL (obtenue avec la fonction [TSL](../WDLang1/3029057.md)),

- soit [une couleur prédéfinie du WLangage](../WDLang5/3010002.md).

## Modifier les caractéristiques d'un libellé
<a name="modifier_les_caracteristiques_libelle_ELTTEXTE000455"></a>
La propriété [Libellé](../Proprietes/2510053.md) permet de :

- changer le libellé du champ Interrupteur :
	
	```txt
	<Champ Interrupteur>.Libellé = <Libellé du champ>
	```


- changer le libellé de toutes les options du champ Interrupteur :
	Chaque option doit être séparée des autres par le caractère TAB. Il est possible de spécifier uniquement les premières options.
	
	```txt
	<Champ Interrupteur>.Libellé = <Option 1> + TAB + <Option 2> + TAB + <Option 3> ...
	```


- changer le libellé d'une option :
	
	```txt
	<Champ Interrupteur>[Indice de l'option].Libellé = <Nouveau libellé de l'option>
	```


- récupérer le libellé d'une option :
	
	```txt
	<Libellé de l'option> = <Champ Interrupteur>[Indice de l'option].Libellé
	```




<a name="NOTE5_2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Si le champ Interrupteur utilise un mode d'affichage avec des images et/ou des sous-libellés, il est possible de : 

- connaître et modifier l'image de l'option grâce à la propriété [Image](../Proprietes/2510034.md) :
	
	```txt
	<Champ Interrupteur>[Indice de l'option].Image = <Nouvelle image>
	```


- connaître et modifier le sous-libellé de l'option grâce à la propriété [SousLibellé](../Proprietes/1000017232.md) :
	
	```txt
	<Champ Interrupteur>[Indice de l'option].SousLibellé = <Nouveau sous-libellé>
	```


- connaître et modifier la hauteur de l'option grâce à la propriété [Hauteur](../Proprietes/2510050.md) : 
	
	```txt
	<Champ Interrupteur>[Indice de l'option].Hauteur = <Nouvelle hauteur en pixels>
	```

## Propriétés spécifiques aux champs Interrupteur
<a name="proprietes_specifiques_aux_champs_interrupteur_ELTTEXTE000485"></a>
Les propriétés suivantes sont spécifiques à la gestion des caractéristiques d'un champ Interrupteur par programmation.


|   |   |
| --- | --- |
| [CadrageHorizontal](../Proprietes/2510023.md) | Permet de connaître et de modifier le cadrage horizontal d'un champ. |
| [CadrageVertical](../Proprietes/2510054.md) | Permet de connaître et de modifier le cadrage vertical d'un champ. |
| [Couleur](../Proprietes/2510071.md) | Permet de connaître et de modifier la couleur des options du champ Interrupteur. |
| [CouleurFond](../Proprietes/2510022.md) | Permet de connaître et de modifier la couleur de fond du champ Interrupteur. |
| [Hauteur](../Proprietes/2510050.md) | Permet de connaître et de modifier la hauteur de l'option (pour les champs Interrupteur utilisant un mode d'affichage avec une image et/ou un sous-libellé). |
| [Image](../Proprietes/2510034.md) | Permet de connaître et de modifier l'image associée à une option (si le champ Interrupteur utilise un mode d'affichage avec une image). |
| [Libellé](../Proprietes/2510053.md) | Permet de connaître et de modifier le libellé du champ Interrupteur ou d'une option du champ Interrupteur. |
| [NombreColonne](../Proprietes/2510083.md) | Permet de connaître le nombre de colonnes du champ Interrupteur. |
| [Occurrence](../WDLang1/3025009.md) | Permet de connaître le nombre d'options présentes dans un champ Interrupteur. |
| [SousLibellé](../Proprietes/1000017232.md) | Permet de connaître et de modifier le sous-libellé d'une option (si le champ Interrupteur utilise un mode d'affichage avec des sous-libellés). |
| [TroisEtats](../Proprietes/2510124.md) | Permet de gérer les trois états d'une option d'un champ Interrupteur (coché, décoché, indifférent). |
| [ValeurInitiale](../Proprietes/2510128.md) | Permet de connaître la valeur initiale d'un champ Interrupteur. |


Pour connaître la liste complète des propriétés du WLangage pouvant être utilisées avec un champ Interrupteur, consultez [Propriétés associées à un champ Interrupteur](../WDChamp/1013172.md).

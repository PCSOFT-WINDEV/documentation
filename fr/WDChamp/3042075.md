
## Graphe : Gestion des échelles
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000106"></a>
Dans un graphe, les échelles permettent d'obtenir une estimation rapide de la valeur des différents points d'un graphe.

WINDEV, WEBDEV et WINDEV Mobile permettent une gestion avancée des échelles affichées dans un champ Graphe.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Caractéristiques des échelles
<a name="caracteristiques_des_echelles_ELTTEXTE000130"></a>


### Caractéristiques générales
<a name="caracteristiques_generales_ELTPARAGRAPHE000028"></a>Dans la majorité des graphes disposant d'échelles, le champ Graphe permet de :

- définir le ***pas des graduations***. Par défaut, les graduations sont automatiquement calculées à partir des valeurs présentes dans les différentes séries. Pour personnaliser les graduations, il suffit d'indiquer le pas voulu dans l'option "Graduation" de l'onglet "Général" du champ.

- définir ***l'orientation des graduations***. Par défaut, l'orientation des graduation s'adapte à l'espace des graduations. Il est possible de forcer l'affichage horizontal ou vertical du texte des graduations. Pour personnaliser l'orientation des graduations, il suffit de sélectionner l'orientation voulue dans l'onglet "Général" du champ.

- définir ***la valeur minimale et maximale des graduations***. Par défaut, ces valeurs dépendent des valeurs présentes dans les différentes séries. Il est possible de forcer ces valeurs dans l'onglet "Général" de la description du champ.

- définir l'espace en pixels entre 2 graduations horizontales (option "Espacement" dans l'onglet "Général" de la description du champ).

- définir le format des graduations verticales (option "Format des graduations" dans l'onglet "Général" de la description du champ). 





### Caractéristiques spécifiques
<a name="caracteristiques_specifiques_ELTPARAGRAPHE000039"></a>Il est également possible d'utiliser des échelles spécifiques :

- **Echelle des abscisses espacée en fonction du temps**. Cette option fonctionne uniquement avec des données de type Date ou Heure provenant : 

	- d'un champ Table, 
			

	- d'une rubrique HFSQL. 

	- d'un tableau. 
			


 Cette option ne peut pas être utilisée avec une série remplie par programmation.

- **Echelles des ordonnées logarithmiques**.

- **Echelle secondaire** : Il est possible d'indiquer pour chaque série si l'échelle utilisée pour la série est l'échelle normale ou l'échelle secondaire. L'échelle secondaire est affichée à droite du graphe.




## Comparer des champs
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000102"></a>
Dans certains cas, il peut être pratique de comparer deux champs situés dans la même fenêtre ou dans deux fenêtres différentes.

Par exemple, si vous utilisez un modèle de champs, il est possible de comparer les surcharges effectuées sur les deux modèles. 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000126"></a>


### Comparer deux champs situés sur la même fenêtre
<a name="comparer_deux_champs_situes_sur_meme_fenetre_ELTPARAGRAPHE000020"></a>Pour comparer deux champs situés sur la même fenêtre : 

1. Sélectionner les deux champs à comparer. 

2. Dans le menu contextuel de la sélection (clic droit), sélectionnez l'option "Comparer les deux éléments". <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Comparer_Champs%20-%20HC%20N%B0001.gif)


3. La fenêtre de résultat s'affiche. <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Comparer_Champs%20-%20HC%20N%B0002.gif&type=thumb)
Pour visualiser les différences, cliquez sur ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Comparer_Champs%20-%20HC%20N%B0002%201.gif)

4. Le détail des éléments différents s'affiche : <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Comparer_Champs%20-%20HC%20N%B0003.gif&type=thumb)




<a name="NOTE2_2"></a>


### Comparer deux champs situés sur des fenêtres différentes
<a name="comparer_deux_champs_situes_sur_des_fenetres_differentes_ELTPARAGRAPHE000037"></a>Pour comparer deux champs situés sur des fenêtres différents : 

1. Sélectionner le premier champ à comparer. 

2. Dans le menu contextuel (clic droit), sélectionnez l'option "Comparer avec". 

3. Un assistant de comparaison se lance permettant de sélectionner la fenêtre contenant le champ à comparer. Cette fenêtre peut être : 

	- Dans le projet en cours. Il suffit de sélectionner la fenêtre voulue dans la liste proposée. 

	- Dans un autre projet. Il suffit de sélectionner le chemin du fichier physique de la fenêtre. 




4. L'assistant propose alors la liste des champs présents dans la fenêtre. Il suffit de sélectionner celui utilisé pour la comparaison. 

5. Spécifiez les options de comparaison : 

	- Ignorer les différences de ligne blanche : Si cette option est cochée, les différences de lignes vides dans le code ne sont pas prises en compte. 

	- Ignorer les différences de commentaires : Si cette option est cochée, les différences de commentaires dans le code ne sont pas prises en compte.

	- Comparer les sous-éléments : Si cette option est cochée, les sous-éléments sont comparés (par exemple, dans le cas d'un champ Table, les colonnes sont comparées). 




6. La fenêtre de résultat s'affiche. Pour visualiser les différences, cliquez sur ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Comparer_Champs%20-%20HC%20N%B0002%201.gif)

7. Le détail des éléments différents s'affiche.




## Propriétés associées au volet d'onglet
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000099"></a>
Un volet d'onglet peut être manipulé en WLangage aussi bien dans une fenêtre ou dans une page avec les propriétés suivantes.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Propriétés disponibles dans une fenêtre

### Propriétés disponibles dans une fenêtre
<a name="proprietes_disponibles_dans_une_fenetre_ELTPARAGRAPHE000020"></a>Les différentes propriétés WLangage associées à un volet d'onglet sont les suivantes :



|   |   |
| --- | --- |
| [Alias](../Proprietes/2510004.md) | La propriété **Alias** permet de : <br><br>- connaître ou de modifier l'alias d'une fenêtre (WINDEV et WINDEV Mobile).<br><br>- connaître ou de modifier l'alias d'un volet d'onglet dynamique (WINDEV).<br><br>- connaître le nom interne HTML / Javascript d'un champ (WEBDEV).<br><br><br> |
| [Bulle](../Proprietes/2510005.md) | La propriété **Bulle** permet de connaître et de modifier le texte affiché dans la bulle d'aide associée à un champ d'une fenêtre ou d'une page. |
| [Couleur](../Proprietes/2510071.md) | La propriété **Couleur** permet de connaître ou modifier la couleur de la police de :<br><br>- une variable de type Police.<br><br>- un texte d'un champ Libellé ou calculé d'un état.<br><br>- un texte d'un champ d'une fenêtre.<br><br>- un texte d'un champ d'une page.<br><br><br> |
| [CouleurFond](../Proprietes/2510022.md) | La propriété **CouleurFond** permet de gérer la couleur de fond d'un champ, d'une fenêtre, d'une page ou d'un bloc d'un état. |
| [Détaché](../Proprietes/1000026078.md) | La propriété **Détaché** permet de savoir si un volet d'un champ Onglet ou d'un champ Ruban est détaché. |
| [Etat](../Proprietes/2510039.md) | La propriété **Etat** permet de connaître et de modifier l'état d'affichage d'un élément. |
| [EtatInitial](../Proprietes/2510063.md) | La propriété **EtatInitial** permet de connaître l'état initial d'un champ. |
| [FenEditionAutorisée](../Proprietes/1000021850.md) | La propriété **FenEditionAutorisée** permet de connaître et de modifier les manipulations autorisées sur un champ lorsque la fenêtre passe en mode "Modification de l'UI par l'utilisateur final". |
| [Grisé](../Proprietes/1000019648.md) | La propriété **Grisé** permet de : <br><br>- Savoir si un champ, un groupe de champs ou une fenêtre est grisé ou non. <br><br>- Griser ou dégriser un champ, un groupe de champs ou une fenêtre.<br><br><br> |
| [Image](../Proprietes/2510034.md) | La propriété **Image** permet de connaître et de modifier l'image de fond d'un champ ou d'un bloc. |
| [Libellé](../Proprietes/2510053.md) | **Dans une base de données**, la propriété **Libellé** permet de :<br><br>- Connaître le libellé associé à une rubrique d'un fichier de données. <br><br>- Connaître le libellé associé à un fichier de données. <br><br>- Connaître et modifier le libellé d'une connexion. <br><br><br><br><br>**Dans un état**, la propriété **Libellé** permet de :<br><br>- Connaître le texte d'un champ Libellé ou Lien.<br><br>- Modifier le texte d'un champ Libellé ou Lien.<br><br><br><br><br>**Dans une fenêtre ou une page**, la propriété **Libellé** permet également de :<br><br>- Connaître et modifier le libellé d'un champ présent dans une fenêtre ou une page.<br><br>- Connaître et modifier le libellé d'une option de menu.<br><br>- Connaître et modifier le titre d'une fenêtre ou d'une page.<br><br>- Connaître le contenu d'un champ chemin de navigation.<br><br><br><br><br> |
| [Nom](../Proprietes/2510082.md) | La propriété **Nom** permet de connaître :<br><br>- le nom de la police associée à une variable de type Police.<br><br>- le nom d'un champ, d'un état ou d'un bloc d'un état.<br><br>- le nom d'un champ, d'un groupe de champs ou d'une fenêtre.<br><br>- le nom d'un champ, d'un groupe de champs ou d'une page.<br><br><br> |
| [NomComplet](../Proprietes/2514005.md) | La propriété **NomComplet** permet de connaître le nom complet :<br><br>- d'un champ.<br><br>- d'une fenêtre.<br><br>- d'une option de menu.<br><br>- d'une colonne de table.<br><br><br><br><br> |
| [NuméroAide](../Proprietes/2510141.md) | La propriété **NuméroAide** permet de connaître et de modifier le numéro d'aide associé au champ. |
| [Visible](../Proprietes/2510138.md) | La propriété **Visible** permet de :<br><br>- Savoir si un élément est visible.<br><br>- Rendre un élément visible ou invisible.<br><br><br> |
| [VisibleInitial](../Proprietes/2510070.md) | La propriété **VisibleInitial** permet de savoir si :<br><br>- un champ était visible lors du premier affichage de la fenêtre ou la page associée.<br><br>- une fenêtre était visible lors de son premier affichage.<br><br><br> |





Remarque : Selon la plateforme utilisée, certaines propriétés peuvent ne pas être disponibles. Consultez la page d'aide de la propriété pour plus de détails.

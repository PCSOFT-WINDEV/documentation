
## FAA : Graphe sur une colonne de table
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000095"></a>
Dès qu'une colonne de table est de type Numérique, l'utilisateur final a la possibilité de réaliser un graphe sur les données de la colonne, profitant ainsi de toute la puissance des graphes créés avec WINDEV. 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Afficher un graphique
<a name="afficher_graphique_ELTTEXTE000119"></a>
Pour réaliser un graphe à partir des données d'une colonne (d'une table ou d'une table hiérarchique) : 

1. Affichez le menu contextuel de la colonne voulue. 

2. Sélectionnez l'option "Graphique".

3. Le graphique réalisé à partir des données présentes dans la colonne est automatiquement affiché dans une nouvelle fenêtre. Vous pouvez utilisez les fonctionnalités du menu contextuel du graphe pour afficher les données à votre convenance. Pour plus de détails, consultez [Manipulations des champs Graphe en exécution (FAA)](../WDChamp/1013007.md).




Remarque : Il n'est pas possible de sélectionner les colonnes à intégrer dans le graphe. Pour choisir uniquement certaines colonnes du champ Table, il est conseillé de créer un nouveau graphe à partir du champ Table. Ce graphe s'affichera par exemple dans une fenêtre.

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Programmation
<a name="programmation_ELTTEXTE000143"></a>


### Désactiver le graphique
<a name="desactiver_graphique_ELTPARAGRAPHE000032"></a>Pour ne pas proposer l'option permettant de réaliser des graphiques, il suffit d'utiliser la fonction [FAADésactive](../WDLang1/1000022018.md) avec la constante *faaGraphique*.


### Afficher le graphique
<a name="afficher_graphique_ELTPARAGRAPHE000040"></a>Pour afficher le graphique d'une colonne de table par programmation, il suffit d'utiliser la fonction [FAAExécute](../WDLang1/1000022099.md) avec la constante *faaGraphique*.

Exemple : 

```wl
FAAExécute(TABLE_Stat.COL_Resultat2016, faaGraphique)
```

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000105"></a>
Voici les principaux points de la programmation des champs Zone répétée fichier.

Le WLangage met à votre disposition de nombreuses fonctions permettant de gérer par programmation les champs Zone répétée fichier.

N'hésitez pas à consultez la documentation de ces fonctions. Pour plus de détails, consultez : [Fonctions de gestions des champs Zone répétée](../WDLang2/3083007.md).

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Ajout et modification d'une répétition dans un champ Zone répétée fichier
<a name="ajout_modification_une_repetition_dans_champ_zone_repetee_fichier_ELTTEXTE000129"></a>
L'ajout et la modification d'une répétition dans le champ Zone répétée (et d'un enregistrement dans le fichier de données) se décompose en deux phases :

1. Ajout ou modification de l'enregistrement dans le fichier de données (par une fenêtre ou une page de type fiche par exemple)

2. Réaffichage du contenu du champ Zone répétée fichier (fonction [ZoneRépétéeAffiche](../WDLang2/3083001.md)).




<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Suppression d'une répétition dans un champ Zone répétée fichier
<a name="suppression_une_repetition_dans_champ_zone_repetee_fichier_ELTTEXTE000153"></a>
Pour supprimer une répétition dans un champ Zone répétée fichier : 

1. Supprimez l'enregistrement dans le fichier de données (fonction [HSupprime](../WDLang4/3044018.md)).

2. Réaffichez le contenu du champ Zone répétée fichier (fonction [ZoneRépétéeAffiche](../WDLang2/3083001.md)).


**Important** : La fonction [ZoneRépétéeNumEnr](../WDLang2/3083010.md) renvoie le numéro de l'enregistrement en cours dans un champ Zone répétée fichier.


## Connecteur Natif Informix pour WINDEV et WEBDEV
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000167"></a>
En option, WINDEV et WEBDEV proposent un Connecteur Natif à Informix (également appelé Accès Natif à Informix). Ce Connecteur Natif permet d'accéder à une base Informix sans drivers externes depuis un programme en WLangage.

Voici quelques uns des nombreux avantages de l'utilisation du Connecteur Natif Informix :

- Les fonctions HLitxxx (HLitSuivant, HModifie, ...) sont utilisables sur les bases de données Informix. C'est-à-dire que vous manipulez une base Informix avec les mêmes ordres qu'une base de données HFSQL. Le portage d'une application existante vers des bases tierces est donc grandement facilité. 

- Le RAD permet de générer du code avec les fonctions Hxxx (HLitSuivant, ...) ou SQLxxx. 

- L'accès est direct sur base Informix sans passer par une couche intermédiaire : MDAC inutile, OLE DB inutile, ODBC inutile. 

- WDMAP (le visionneur de données) est utilisable sur vos bases de données Informix (utilisation de la couche client 32 bits). 

- Le Centre de Contrôle HFSQL est utilisable sur vos bases de données Informix (CC28HF.exe pour les couches clientes 32 bits et CC28HF64.exe pour les couches clientes 64 bits). Vous pouvez ainsi réaliser directement des requêtes et visualiser les données Informix. 

- L'installation est simple : Pas de couche OLE DB ou ODBC chez le client. 

- Support des bases Informix toutes versions 

- etc.


L'utilisation de ce Connecteur Natif est fortement conseillée pour tout développement d'application Client/Serveur avec WINDEV ou WEBDEV, accédant à des bases de données Informix. Pour toute information complémentaire, n'hésitez pas à nous contacter.

**Attention** : 

- Cette version du Connecteur Natif Informix ne fonctionne qu'avec WINDEV et WEBDEV 28.

- Cette version du Connecteur Natif Informix ne fonctionne pas avec les produits suivants : Informix Standard Edition (SE), Informix OnLine et Informix C-ISAM.

## Configuration nécessaire
<a name="configuration_necessaire_ELTTEXTE000191"></a>


### Conditions nécessaires à l'utilisation du Connecteur Natif Informix
<a name="conditions_necessaires_utilisation_connecteur_natif_informix_ELTPARAGRAPHE000043"></a>

**En 32 bits** : 

- Couche client Informix 3.34.0.11061 (version testée par nos services) installée.

- La DLL **ICLIT09B.DLL** (version testée par nos services) doit être accessible :

	- à partir du poste de développement.

	- à partir du poste de déploiement.







**En 64 bits** : 

- Couche client Informix 4.10.00.15364 (version testée par nos services) installée.

- La DLL **ICLIT09B.DLL** (version testée par nos services) doit être accessible :

	- à partir du poste de développement.

	- à partir du poste de déploiement.







La DLL **ICLIT09B.DLL** est recherchée par défaut. Si cette DLL n'est pas disponible :

1. Créez un fichier "WDIFMX.INI" dans le répertoire de Windows (exemple C:\\WINDOWS).

2. Ajoutez les informations suivantes dans le fichier "WDIFMX.INI" :





```txt
[CLIENT]
DRIVER=<Nom de la DLL>
```


où &lt;Nom de la DLL&gt; correspond au chemin complet de la DLL Informix. Cette DLL peut également correspondre au driver ODBC utilisé pour accéder à Informix.
<a name="NOTE3_2"></a>


### Les fichiers nécessaires
<a name="les_fichiers_necessaires_ELTPARAGRAPHE000075"></a>Les fichiers nécessaires à l'utilisation du Connecteur Natif Informix sont :

- ***ICLIT09B.DLL*** (version testée par nos services) : Cette DLL est installée par la couche client de Informix. Pour les conditions de diffusion de cette DLL, reportez-vous à la licence de Informix.

- ***wd280ifmx.dll*** : Nécessaire au Connecteur Natif à une base Informix en 32 bits. Ce fichier est livré avec le Connecteur Natif pour Informix.

- ***wd280ifmx64.dll*** : Nécessaire au Connecteur Natif à une base Informix en 64 bits. Ce fichier est livré avec le Connecteur Natif pour Informix.

## Utilisation
<a name="utilisation_ELTTEXTE000221"></a>


### Utiliser le Connecteur Natif Informix dans vos applications WINDEV et vos sites WEBDEV
<a name="utiliser_connecteur_natif_informix_dans_vos_applications_windev_vos_sites_webdev_ELTPARAGRAPHE000094"></a>Pour utiliser Informix en natif, il suffit de programmer :

- soit avec les fonctions "SQL" du WLangage (voir [Programmation à l'aide des fonctions SQL](../WDInformix/5511006.md)).

- soit avec les fonctions "HFSQL" du WLangage (voir [Programmation à l'aide des fonctions HFSQL](../WDInformix/5511001.md)).




**Remarques** : 

- N'oubliez pas de consulter [les spécificités concernant l'utilisation du Connecteur Natif Informix](../WDInformix/5511004.md).

- L'option "Créer les fichiers de données automatiquement au moment de leur ouverture" (disponible dans l'onglet "Fichiers" de la description du projet) est prise en compte par le Connecteur Natif Informix.

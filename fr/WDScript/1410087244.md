## Présentation
<a name="presentation_ELTTEXTE000168"></a>
WDScript est un éditeur de scripts WLangage, livré avec WINDEV, WEBDEV et WINDEV Mobile. WDScript permet : 

- d'écrire des scripts en WLangage. 

- d'exécuter des scripts WLangage existants. 


Il est ainsi possible d'automatiser certaines tâches qui ne nécessitent pas d'interface et qui doivent être lancées en mode "batch" (par exemple le traitement de journaux d'erreurs).

## Utilisation
<a name="utilisation_ELTTEXTE000192"></a>


### Lancement de WDScript
<a name="lancement_wdscript_ELTPARAGRAPHE000021"></a>WDScript peut être lancé en mode **interactif** : 

- directement depuis WINDEV, WEBDEV ou WINDEV Mobile : sous le volet "Outils", dans le groupe "Utilitaires", cliquez sur "WDScript". 

- en lançant directement les fichiers suivants présents dans le sous-répertoire "Programmes" du répertoire d'installation de WINDEV, WEBDEV ou WINDEV Mobile : 

	- WDScript.exe : Lance WDScript en version 32 bits. 

	- WDScript64.exe : Lance WDScript en version 64 bits. 





Pour plus de détails, consultez [WDScript : utilisation en mode interactif](../WDScript/1410087245.md). 

WDScript peut être lancé en mode **ligne de commande** pour exécuter directement le script WLangage spécifié. 

WDScript peut être lancé en mode **"console"** grâce aux exécutables suivants :  

- WDScriptConsole.exe : Lance WDScript en mode console en version 32 bits. 

- WDScriptConsole64.exe : Lance WDScript en mode console en version 64 bits. 




Pour plus de détails, consultez [WDScript : utilisation en ligne de commande et en mode console](../WDScript/1410087267.md). 




### Conditions d'utilisation
<a name="conditions_utilisation_ELTPARAGRAPHE000062"></a>WDScript est un outil redistribuable. WDScript peut être installé avec les applications développées avec WINDEV, WEBDEV ou WINDEV Mobile. La licence d'utilisation de WINDEV s'applique intégralement.




### Installation de WDScript
<a name="installation_wdscript_ELTPARAGRAPHE000068"></a>

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) **Cas 1 : Installation avec vos applications WINDEV**
 
Lors de la création de la procédure d'installation, WDScript peut être livré avec vos applications WINDEV.
Rappel : La procédure d'installation peut être créée :

- soit à partir de l'assistant : sous le volet "Projet", dans le groupe "Génération", déroulez "Procédure d'installation" et sélectionnez "Créer la procédure d'installation".

- soit à partir de l'éditeur d'installation : sous le volet "Outils", dans le groupe "Utilitaires", cliquez sur "WDInst".




**Cas 2 : Installation indépendante**
Les fichiers nécessaires à l'installation de WDScript sont les suivants :

- WDScript.exe (32 bits) ou WDScript64.exe (64 bits) : Ce module est présent dans le répertoire "\\Programmes\\" de WINDEV, WEBDEV ou WINDEV Mobile,

- WDScriptConsole.exe (32 bits) ou WDScriptConsole64.exe (64 bits) : Ce module est présent dans le répertoire "\\Programmes\\" de WINDEV, WEBDEV ou WINDEV Mobile,

- Le Framework correspondant : comme toutes les fonctions du WLangage peuvent être utilisées, il est nécessaire d'installer le Framework complet. Il est disponible dans le répertoire :

	- "\\Programmes\\Framework\\Win32x86" pour WDScript.exe (32 bits),

	- "\\Programmes\\Framework\\Win64x86" pour WDScript.exe (64 bits).




- Le composant WDOutils.wdk : il est également présent dans le répertoire "\\Programmes\\" de WINDEV, WEBDEV ou WINDEV Mobile.

- La DLL CC280HF.DLL (32 bits) ou CC280HF64.DLL (64 bits) présente dans le répertoire "\\Programmes\\" de WINDEV, WEBDEV ou WINDEV Mobile. Ces DLL correspondent à l'éditeur de code simplifié, partagé avec le Centre de Contrôle HFSQL.


## WDScript : utilisation en ligne de commande et en mode console
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000100"></a>
Les scripts créés avec WDScript peuvent être exécutés : 

- en ligne de commande. 

- en mode console. En appelant le script depuis la console Windows, il est possible d'écrire dans la sortie standard. 




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire
<a name="comment_faire_ELTTEXTE000124"></a>


### Exécuter WDScript en mode ligne de commande
<a name="executer_wdscript_mode_ligne_commande_ELTPARAGRAPHE000019"></a>Pour exécuter WDScript en mode ligne de commande, saisissez la ligne de commande suivante : 

```txt
<Chemin de WDScript> /SCRIPT=<Chemin du script>
```
 où : 

- &lt;Chemin de WDScript&gt; correspond au chemin complet de l'exécutable de WDScript.exe (ou WDScript64.exe).

- "/SCRIPT=" est l'instruction permettant de préciser le script à lancer. 

- &lt;Chemin du script&gt; correspond au chemin complet du script à lancer.  




Exemple : 
```txt
c:\WinDev64\Programmes\WDScript64.exe /SCRIPT="C:\Temp\test.wl"
```



### Exécuter WDScript en mode console
<a name="executer_wdscript_mode_console_ELTPARAGRAPHE000034"></a>Pour exécuter WDScript en mode console : 

1. Ouvrez le mode console de Windows (raccourci "Windows R"). 

2. Saisissez "cmd" et validez. 

3. Saisissez la ligne de commande suivante : 
	
	```txt
	<Chemin de WDScriptConsole> /SCRIPT=<Chemin du script>
	```
où : 

	-  correspond au chemin complet de WDScriptConsole.exe (ou WDScriptConsole64.exe).

	- "/SCRIPT=" est l'instruction permettant de préciser le script à lancer. 

	- &lt;Chemin du script&gt; correspond au chemin complet du script à lancer. 







Exemple : 
```txt
c:\WinDev64\Programmes\WDScriptConsole.exe /SCRIPT="C:\Temp\test.wl" >"c:\temp\log.log"
```


Remarque : Le résultat de la fonction **Trace** sera affiché dans la sortie standard. Il est également possible d'utiliser dans le script la fonction WLangage **ConsoleEcrit**.


## WDScript : utilisation en mode interactif
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000143"></a>
En mode interactif, l'éditeur WDScript se décompose en plusieurs zones : 

- Le ruban qui permet d'accéder aux options les plus courantes, 

- L'éditeur de code qui permet de saisir le script en WLangage. 

- La console de sortie qui permet notamment d'afficher les différents résultats ainsi que les erreurs rencontrés en exécution.  




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Le ruban de WDScript
<a name="ruban_wdscript_ELTTEXTE000167"></a>
Le ruban de WDScript est composé de deux volets : 

- Le volet "Accueil" qui propose les différentes options permettant de manipuler les scripts WLangage. 

- Le volet "Code" qui permet de manipuler le code WLangage. 





### Volet "Accueil"
<a name="volet_accueil_ELTPARAGRAPHE000026"></a>Le volet "Accueil" permet de : <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=WD_Script%20-%20HC%20N%B0001.gif)


- Créer et manipuler un fichier script WLangage, grâce aux options "Nouveau", "Ouvrir", "Enregistrer" et "Fermer". 

- Lancer le script affiché (option "Go").  

- Modifier les options de WDScript. Il est possible de : 

	- Choisir la langue d'affichage de WDScript. 

	- Gérer l'association des fichiers "wl" générés par WDScript à WDScript ou WDScriptConsole. <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=WD_Script%20-%20HC%20N%B0003.gif)


		- Si le fichier wl est associé à WDScript, il sera automatiquement ouvert dans l'interface de WDScript lors du double-clic sur son nom. 

- Si le fichier wl est associé à WDScriptConsole, il sera automatiquement exécuté en mode console lors du double-clic sur son nom. 

- proposer l'option "Ouvrir dans WDScript" dans l'explorateur de fichiers Windows. Cette option sera proposée dans le menu contextuel du fichier sous l'explorateur de fichier Windows. 





### Volet "Code"
<a name="volet_code_ELTPARAGRAPHE000044"></a>Le volet "Code" permet de manipuler le code affiché dans l'éditeur de code de WDScript. <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=WD_Script%20-%20HC%20N%B0002.gif)
Il est possible de : 

- Commenter / Décommenter une ou plusieurs lignes de code 

- Faire une recherche dans le script affiché (rechercher un terme ou afficher un numéro de ligne). 

- Afficher l'aide en ligne de la fonction WLangage sélectionnée (via le site de documentation de PC SOFT).

## L'éditeur de code de WDScript
<a name="editeur_code_wdscript_ELTTEXTE000203"></a>
L'éditeur de code de WDScript est un éditeur de code simplifié par rapport à l'éditeur de code de WINDEV, WEBDEV ou WINDEV Mobile. Il est destiné à des scripts contenant un code plus court et plus simple que celui saisi dans l'éditeur de code de WINDEV, WEBDEV ou WINDEV Mobile. 
<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=WD_Script%20-%20HC%20N%B0004.gif)


Caractéristiques de l'éditeur de code de WDScript : 

- Coloration du code. 

- Présence des raccourcis simples. 

- Complétion simple du code. 

- Les erreurs de compilation s'affichent directement dans le code, sous forme de bulle. <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=WD_Script%20-%20HC%20N%B0005%201.gif)





Plusieurs contraintes existent : 

- Le script WLangage doit contenir un seul bloc de code. 

- Les procédures doivent être des procédures internes. 

- Les procédures internes doivent être déclarées avant leur exécution. 

- Pas de débogueur. 

- Pas d'utilisation de nom de champ, de page, de fenêtre ou d'état.

## Console de sortie de WDScript
<a name="console_sortie_wdscript_ELTTEXTE000227"></a>
La console de sortie permet de : 

- Afficher les résultats renvoyés par le script lors de l'exécution du script en mode interactif (utilisation de la fonction **Trace** par exemple). <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=WD_Script%20-%20HC%20N%B0004%201.gif)
La valeur de retour du code exécuté est également affichée. 

- Afficher les erreurs d'exécution lors de l'exécution du script en mode interactif. <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=WD_Script%20-%20HC%20N%B0005.gif)





Ce volet est détachable et peut être réduit. Les erreurs sont affichées en rouge. Le bouton ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=WD_Script%20-%20HC%20N%B0001%201.gif) permet d'effacer le contenu de la fenêtre de sortie.




## Gestion des serveurs HFSQL
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000309"></a>
Le Centre de Contrôle HFSQL propose une gestion complète des serveurs HFSQL.

Pour afficher les caractéristiques d'un serveur HFSQL : 

1. Connectez-vous si nécessaire à un serveur HFSQL. Pour plus de détails, consultez [Gestion des bases de données HFSQL Client/Serveur](../CCHF/35400013.md). 

2. Double-cliquez sur le nom du serveur HFSQL dans l'arborescence. 

3. La partie droite du Centre de Contrôle affiche un onglet au nom du serveur. Cet onglet est constitué de : 

	- différents onglets permettant de gérer les caractéristiques du serveur HFSQL.

	- un ruban permettant de configurer le serveur HFSQL. 









<a name="NOTE2"></a>


## Détail des caractéristiques d'un serveur HFSQL
<a name="detail_des_caracteristiques_serveur_hfsql_ELTTEXTE000339"></a>


### Onglet "Tableau de bord"
<a name="onglet_tableau_bord_ELTPARAGRAPHE000034"></a>Cet onglet affiche les caractéristiques générales du serveur.

Depuis le tableau de bord, de nombreuses actions sont possibles sur le serveur. 

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CC_HF_Gestion_Serveur%20-%20HC%20N%B0001.gif&type=thumb)


Cet onglet affiche les caractéristiques suivantes sous forme de Widgets : 

- Nom et état du serveur. L'heure, la quantité de RAM, le système sont également affichés. 

- Version du serveur, avec le mode de mise à jour. Pour configurer la mise à jour du serveur, cliquez sur le lien correspondant. Pour plus de détails, consultez [Mise à jour automatique du serveur](../CCHF/35400018.md). 

- Utilisateur en cours. Il est possible d'éditer le profil de l'utilisateur en cours. 

- Utilisateurs et groupes définis sur le serveur. Un clic sur les informations affichées permet d'accéder aux caractéristiques des utilisateurs ou des groupes. 

- Transactions : Affiche les transactions en cours. 

- Configuration : Affiche les principales caractéristiques de paramétrage du serveur : Calcul des statistiques sur les clés, statistiques d'activité, cache des index, gestion des logs, etc. Un clic sur chacune des options permet d'ouvrir la fenêtre de paramétrage correspondante. 

- Activité du serveur : Affiche les tâches en cours. Un clic sur le nombre de tâches en cours permet d'obtenir le détail des tâches. 

- Tâches planifiées : Affiche les différents types de tâches planifiées existantes sur le serveur. Un clic sur ces tâches permet de voir leur configuration. 

- Notifications : Affiche les notifications reçues. Il est possible de configurer directement les destinataires des notifications. 

- Réplication : Affiche les réplications des données de serveur définies. Il est possible de configurer directement la réplication de serveurs. 

- Sauvegarde : Affiche les sauvegardes planifiées. Il est possible de configurer les sauvegardes planifiées. 

- Connexions : Affiche le nombre de connexions. Le lien "Plus de détails" permet d'afficher l'onglet "Connexions". 

- Volumétrie des bases : Affiche un graphe permettant de connaître le volume de chaque base installée sur le serveur. L'option "Afficher la liste des bases" permet d'afficher l'onglet "Bases". 




Pour paramétrer les "Widgets" affichés dans le tableau de bord : 

1. Affichez le menu contextuel de l'onglet "Tableau de bord" (par exemple dans un emplacement ne contenant pas de Widgets). 

2. Sélectionnez l'option "Mode Modification". 

3. Le tableau de bord apparaît en mode "modification". Il est possible  de : 

	- Masquer certains Widgets. 

	- Afficher des Widgets précédemment masqués. 

	- Redimensionner les Widgets à l'aide de la souris. 




4. Pour sortir du mode modification et appliquer vos changements, sélectionnez à nouveau l'option "Mode Modification" du menu contextuel.

### Onglet "Connexions"
<a name="onglet_connexions_ELTPARAGRAPHE000074"></a>Cet onglet liste les différents utilisateurs connectés sur les différentes bases de données présentes sur le serveur.

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CC_HF_Gestion_Serveur%20-%20HC%20N%B0002.gif&type=thumb)
La liste des connexions actives contient les informations suivantes :

- le nom de la base de données,

- le nom de l'utilisateur,

- le nom de la machine,

- le nom de l'application qui accède à la base, etc.




Lorsque cet onglet est sélectionné, des options supplémentaires apparaissent dans le ruban. Ces options, placées dans le groupe "Connexions", permettent de : 

- interdire ou autoriser les connexions, 

- déconnecter les utilisateurs, 

- envoyer un message aux utilisateurs.



<a name="NOTE2_3"></a>


### Onglet "Bases"
<a name="onglet_bases_ELTPARAGRAPHE000094"></a>Cet onglet permet de lister les différentes bases de données présentes sur le serveur. 

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CC_HF_Gestion_Serveur%20-%20HC%20N%B0003.gif&type=thumb)
Pour chaque base de données, sont indiquées : 

- Le nombre de connexions, 

- le nombre de blocages, 

- le nombre de tables. 




Lorsque cet onglet est sélectionné, des options supplémentaires apparaissent dans le ruban. Ces options, placées dans le groupe "Bases de données", permettent de : 

- accéder à la base sélectionnée, 

- afficher le détail de la base sélectionnée ou de toutes les bases. Un ou plusieurs nouveaux onglets s'ouvrent, affichant le détail de la base. 

- supprimer la base sélectionnée du serveur. 



<a name="NOTE2_4"></a>


### Onglet "Utilisateurs"
<a name="onglet_utilisateurs_ELTPARAGRAPHE000113"></a>Cet onglet permet de gérer les utilisateurs (création, modification, suppression, recherche).

Lorsque cet onglet est sélectionné, des options supplémentaires apparaissent dans le ruban. Ces options, placées dans le groupe "Utilisateurs", permettent de : 

- créer un nouvel utilisateur, 

- éditer les caractéristiques d'un utilisateur existant, 

- supprimer un utilisateur. 




Lors de la création d'un utilisateur, l'utilisateur peut être un super utilisateur : les utilisateurs désignés comme "super utilisateur" sont autorisés à effectuer toutes les actions sur le serveur, les bases de données et sur tous les fichiers.
<a name="NOTE2_5"></a>


### Onglet "Groupe"
<a name="onglet_groupe_ELTPARAGRAPHE000128"></a>Cet onglet permet de gérer les groupes d'utilisateurs (création, modification, suppression).

Lorsque cet onglet est sélectionné, des options supplémentaires apparaissent dans le ruban. Ces options, placées dans le groupe "Groupes", permettent de : 

- créer un nouveau groupe, 

- éditer les caractéristiques d'un groupe existant, 

- supprimer un groupe.



<a name="NOTE2_6"></a>


### Onglet "Transactions"
<a name="onglet_transactions_ELTPARAGRAPHE000141"></a>Cet onglet permet de gérer les transactions en cours ou interrompues sur le serveur. Il est possible de : 

- Visualiser les transactions en cours, 

- Libérer les enregistrements en transaction. 



![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CC_HF_Gestion_Serveur%20-%20HC%20N%B0004.gif&type=thumb)

<a name="NOTE2_7"></a>


### Onglet "Eléments planifiés"
<a name="onglet_elements_planifies_ELTPARAGRAPHE000153"></a>Cet onglet permet de créer, modifier et supprimer des planifications sur le serveur : 

- sauvegardes.

- tâches (procédures stockées).

- optimisation.

- rafraîchissement de vues matérialisées. 




Lorsque cet onglet est sélectionné, des options supplémentaires apparaissent dans le ruban. Ces options, placées dans le groupe "Eléments planifiés", permettent de : 

- créer une nouvelle planification, 

- afficher le détail d'une planification,

- exécuter une planification,

- supprimer une planification. 



![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CC_HF_Gestion_Serveur%20-%20HC%20N%B0005.gif&type=thumb)
Pour plus de détails, consultez [Gestion des tâches planifiées](../CCHF/3540009.md).
<a name="NOTE2_8"></a>


### Onglet "Sauvegardes"
<a name="onglet_sauvegardes_ELTPARAGRAPHE000178"></a>Cet onglet permet de réaliser et de restaurer des sauvegardes du serveur et / ou des bases de données. Cet onglet liste les sauvegardes du serveur : 

- en cours d'exécution, 

- terminées. 




Lorsque cet onglet est sélectionné, des options supplémentaires apparaissent dans le ruban. Ces options, placées dans le groupe "Sauvegardes", permettent de : 

- créer une nouvelle sauvegarde (à chaud ou planifiée), 

- relancer une sauvegarde,

- restaurer une sauvegarde,

- supprimer une sauvegarde. 




Pour plus de détails, consultez [Sauvegardes](../CCHF/3540006.md).
<a name="NOTE2_9"></a>


### Onglet "Notifications"
<a name="onglet_notifications_ELTPARAGRAPHE000200"></a>Cet onglet liste les différentes notifications du serveur :

- Tâches planifiées en échec,

- Sauvegardes en échec,

- Serveur de réplication inaccessible,

- etc.




Lorsque cet onglet est sélectionné, des options supplémentaires apparaissent dans le ruban. Ces options, placées dans le groupe "Notifications", permettent de : 

- Supprimer les notifications sélectionnées.

- Supprimer toutes les notifications du serveur.




Pour plus de détails sur les notifications, consultez [Notifications d'un serveur HFSQL](../WDLang4/1000017311.md).

## Options du ruban
<a name="options_ruban_ELTTEXTE000411"></a>
Lorsque les caractéristiques du serveur sont affichées, le ruban propose plusieurs options : 

- des options générales. 

- des options spécifiques aux différents onglets sélectionnés. 



![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CC_HF_Gestion_Serveur%20-%20HC%20N%B0001%201.gif&type=thumb)
Les options générales sont les suivantes : 

- Dans le groupe "Requête SQL" : 

	- Nouvelle requête : Cette option permet d'ouvrir un onglet spécifique permettant de réaliser une requête.  

	- Ouvrir une requête : Cette option permet d'ouvrir un onglet spécifique correspondant à une requête précédemment réalisée. 




- Dans le groupe "Serveur" : 

	- Arrêter le serveur : Cette option permet d'arrêter le serveur. 

	- Démarrer le serveur : Cette option permet de démarrer le serveur (si celui-ci a été préalablement arrêté). 




- Dans le groupe "Base de données" : 

	- Nouvelle base de données : Cette option permet de créer une nouvelle base de données sur le serveur. 

	- Importer une base HFSQL Classic : Cette option permet d'importer une base de données HFSQL Classic sur le serveur en mode Client/Serveur. Un assistant permet de sélectionner les fichiers de données à importer. 

	- Récupérer des fichiers depuis le serveur : Cette option permet de copier en local des fichiers présents sur le serveur. Un assistant permet de sélectionner les différentes options de la copie ainsi que le répertoire de copie.




- Dans le groupe "Droits" : 

	- Gestion des droits : Cette option permet de gérer les droits sur le serveur pour l'utilisateur ou le groupe sélectionné. Il est possible de mémoriser et de restaurer une configuration de droits.
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CC_HF_Gestion_Serveur%20-%20HC%20N%B0006.gif&type=thumb)





- Dans le groupe "Paramètres", l'option "Configuration du serveur" permet de configurer le serveur. Pour plus de détails, consultez [Configuration du serveur HFSQL](../CCHF/3540004.md). 

- Dans le groupe "Logs et statistiques" : 

	- Afficher les logs du serveur : Cette option permet d'obtenir une vue synthétique sur l'utilisation de la base de données entre deux dates :

		- requêtes les plus utilisées, les plus longues,

		- applications consommatrices de ressources

		- statistiques par application, par traitement, etc.


 Cette option est disponible uniquement si les logs sont activés sur le serveur.

- Afficher les stats du serveur : Cette option est disponible uniquement si les statistiques d'activité sont activés sur le serveur.




## Gestion des fichiers HFSQL Client/Serveur
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000209"></a>
Le Centre de Contrôle HFSQL propose une gestion complète des fichiers de données HFSQL Client/Serveur.

Pour afficher les caractéristiques d'un fichier de données HFSQL Client/Serveur : 

1. Connectez-vous si nécessaire à un serveur HFSQL. Pour plus de détails, consultez [Gestion des bases de données HFSQL Client/Serveur](../CCHF/35400013.md). 

2. Sélectionnez le nom du serveur HFSQL dans l'arborescence. 

3. Sélectionnez ensuite le nom de la base de données voulue.

4. Double-cliquez ensuite sur le nom du fichier de données voulu.

5. La partie droite du Centre de Contrôle affiche un onglet au nom du fichier de données. Cet onglet est constitué de : 

	- différents onglets permettant de gérer les caractéristiques du fichier de données HFSQL. Par défaut, l'onglet "Contenu" est affiché. 

	- un ruban permettant de configurer le fichier de données HFSQL. 




<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CC_HF_Gestion_Fichier_HFCS%20-%20HC%20N%B0002.gif&type=thumb)


**Remarque** : Depuis le volet du Centre de Contrôle, le menu contextuel du fichier de données permet de :<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CC_HF_Gestion_Fichier_HFCS%20-%20HC%20N%B0001.gif)


- Optimiser et réparer le fichier de données. 

- Fermer le fichier de données. 

- Changer le mot de passe du fichier de données. 

- Supprimer le fichier de données. 

- Gérer les droits sur le fichier de données.

## Détail des caractéristiques d'un fichier de données HFSQL Client/Serveur
<a name="detail_des_caracteristiques_fichier_donnees_hfsql_clientserveur_ELTTEXTE000239"></a>


### Onglets "Description" et "Propriétés"
<a name="onglets_description_proprietes_ELTPARAGRAPHE000047"></a>L'onglet "Description" affiche la description des différentes rubriques du fichier de données : 

- Le nom de la rubrique,

- Le type de la rubrique,

- La taille de la rubrique,

- Le type de clé,

- Si la rubrique est visible sous le logiciel "Etats & Requêtes".




Si nécessaire, des informations complémentaires sont affichées.

Remarque : Par défaut, les rubriques sont affichées dans l'ordre logique. il est possible d'utiliser l'ordre alphabétique si nécessaire via les options du Centre de Contrôle HFSQL : 

1. Déroulez le menu du Centre de Contrôle HFSQL (![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CC_Suivi_Configuration%20-%20HC%20N%B0001.gif)). 

2. Sélectionnez "Options".

3. Dans la fenêtre qui s'affiche, indiquez "alphabétique" pour l'option "Ordre d'affichage des rubriques". 




L'onglet "Propriétés" affiche les propriétés du fichier de données (nom physique, format, protection, etc.).
<a name="NOTE2_2"></a>


### Onglet "Connexions"
<a name="onglet_connexions_ELTPARAGRAPHE000066"></a>Cet onglet liste les différents utilisateurs connectés sur le fichier de données.

La liste des connexions actives contient les informations suivantes :

- le nom de l'utilisateur,

- le nom de la machine,

- le nom de l'application qui accède à la base, etc.




Il est possible de gérer les connexions depuis cet onglet : déconnecter les utilisateurs ou envoyer un message aux utilisateurs.
<a name="NOTE2_3"></a>


### Onglet "Blocages"
<a name="onglet_blocages_ELTPARAGRAPHE000081"></a>Cet onglet permet de gérer les blocages.

Les informations affichées sont les suivantes : 

- le numéro de l'enregistrement bloqué, 

- le nom de l'utilisateur qui bloque, 

- le nom de la machine qui bloque,

- l'adresse IP de la machine qui bloque, 

- le nom de l'application qui bloque, 

- la date de début de blocage, 

- la durée du blocage.



<a name="NOTE2_4"></a>


### Onglet "Contenu"
<a name="onglet_contenu_ELTPARAGRAPHE000098"></a>Cet onglet permet de visualiser les différents enregistrements du fichier de données.

Le menu contextuel des enregistrements permet de manipuler l'enregistrement sélectionné : suppression, ajout, mode d'affichage des mémos texte, etc.

Il est également possible de :

- Supprimer un enregistrement : sélectionnez l'enregistrement et cliquez sur le bouton "Supprimer" (ou appuyez sur la touche Suppr).

- Modifier ou rajouter un enregistrement : la saisie se fait directement dans la table. La modification du fichier de données est faite automatiquement.




Le menu contextuel des enregistrements permet également de : 

- Supprimer un enregistrement, 

- Dupliquer un enregistrement, 

- Spécifier des options d'affichage, 

- Imprimer les enregistrement, 



<a name="NOTE2_5"></a>


### Onglet "Recherche"
<a name="onglet_recherche_ELTPARAGRAPHE000119"></a>Cet onglet permet de réaliser des recherches dans le fichier de données. Il suffit de saisir les conditions recherchées et les valeurs dans la table des conditions. 

Le code SQL correspondant à la recherche peut être copié via le bouton "Copier le code SQL" présent dans le ruban. 

Remarque : Pour effectuer une recherche SQL, cliquez sur "Nouvelle requête" dans le ruban.
<a name="NOTE2_6"></a>


### Onglet "Journaux"
<a name="onglet_journaux_ELTPARAGRAPHE000130"></a>Cet onglet permet de visualiser le contenu des journaux et d'effectuer une recherche dans les journaux.

Il est nécessaire que le fichier de données  soit journalé.
<a name="NOTE2_8"></a>


### Onglet "Transactions"
<a name="onglet_transactions_ELTPARAGRAPHE000139"></a>Cet onglet permet de libérer un enregistrement ou toutes les transactions.

## Autres options disponibles
<a name="autres_options_disponibles_ELTTEXTE000317"></a>


### Détail du Ruban
<a name="detail_ruban_ELTPARAGRAPHE000148"></a>Pour le fichier de données HFSQL Client/Serveur sélectionné, le ruban propose les actions suivantes : 

- Dans le groupe "Requête SQL" : 

	- Nouvelle requête : Cette option permet d'ouvrir un onglet spécifique permettant de réaliser une requête SQL sur la base de données en cours.  

	- Ouvrir une requête : Cette option permet d'ouvrir un onglet spécifique correspondant à une requête précédemment réalisée. 




- Dans le groupe "Fichier" :

	- Optimiser et réparer :  Cette option permet d'ouvrir la fenêtre de configuration de l'optimisation du fichier de données. 

	- Fermer le fichier : cette option permet de fermer le fichier de données et l'onglet correspondant. 




- Dans le groupe "Droits", l'option "Gestion des droits" permet de gérer les droits des utilisateurs et des groupes sur le fichier de données. 




Selon l'onglet sélectionné, des options supplémentaires peuvent apparaître dans le ruban.

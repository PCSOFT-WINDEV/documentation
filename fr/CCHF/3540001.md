


## Centre de Contrôle HFSQL : Présentation
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000185"></a>
Le Centre de Contrôle HFSQL permet de gérer les différentes bases de données HFSQL (Réseau, Mobile, Client/Serveur) ainsi que les bases tierces.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Lancement du Centre de Contrôle HFSQL
<a name="lancement_centre_controle_hfsql_ELTTEXTE000215"></a>


### Lancement du Centre de Contrôle HFSQL
<a name="lancement_centre_controle_hfsql_ELTPARAGRAPHE000020"></a>Le Centre de Contrôle HFSQL peut être lancé :

- depuis WINDEV, WEBDEV ou WINDEV Mobile : sous le volet "Outils", dans le groupe "Base de données", cliquez sur "HFSQL".

- en lançant directement le programme "CCxx0HF.EXE". 

- directement à partir d'un fichier de données (double-clic sur un fichier ".FIC"),

### Conditions d'utilisation
<a name="conditions_utilisation_ELTPARAGRAPHE000043"></a>Le Centre de Contrôle HFSQL est un outil redistribuable.

## Fonctionnalités du Centre de Contrôle HFSQL
<a name="fonctionnalites_centre_controle_hfsql_ELTTEXTE000245"></a>
Le Centre de Contrôle HFSQL permet de gérer : <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=cc_HF_Presentation%20-%20HC%20N%B0001.gif)


- Les bases de données HFSQL Client/Serveur. Il est possible d'obtenir des informations et de manipuler :

	- [le serveur](../CCHF/35400014.md) : Arrêt, redémarrage, configuration du serveur, définition des utilisateurs, tâches planifiées, sauvegardes.

	- [la base de données](../CCHF/35400015.md) : Description, recherche, sauvegarde, maintenance des fichiers de données, transactions, etc.

	- [chaque fichier de la base de données](../CCHF/35400016.md) : Description, contenu, recherche, journal, transactions, etc.




- Les [fichiers HFSQL Classic](../CCHF/35400010.md). Il est possible d'obtenir des informations et de manipuler chaque fichier de la base de données : Description, contenu, recherche, journal, transactions, etc.

- [Une analyse](../CCHF/3540003.md). Il est possible d'obtenir des informations et de manipuler l'ensemble des fichiers de données de l'analyse. Ces fichiers de données peuvent être :

	- HFSQL Client/Serveur,

	- HFSQL Classic,

	- Bases tierces




- Les bases tierces. Il est possible d'obtenir des informations et de manipuler des fichiers de données non HFSQL :

	- Oracle,

	- SQL Server

	- MySQL

	- etc.




- [Un cluster HFSQL](../CCHF/3540005.md). Il est possible d'obtenir des informations et de manipuler des fichiers de données HFSQL Cluster.

## Options du Centre de Contrôle HFSQL
<a name="options_centre_controle_hfsql_ELTTEXTE000269"></a>
Le Centre de Contrôle HFSQL propose plusieurs options générales concernant l'ensemble des fonctionnalités disponibles. Pour afficher ces options : 

1. Déroulez le menu du Centre de Contrôle HFSQL (<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CC_Suivi_Configuration%20-%20HC%20N%B0001.gif)
). 

2. Sélectionnez "Options". 

<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CC_HF_Presentation_options%20-%20HC%20N%B0001.gif)
Les options disponibles sont les suivantes : 

- **Afficher l'icône "x" permettant de visualiser les mots de passe saisis** : Permet d'afficher un oeil à côté des champs de type "Mot de passe". Il sera ainsi possible de connaître le mot de passe en cliquant sur cette icône. 

- **Afficher les mémos image uniquement si la qualité de la connexion avec le serveur HFSQL est suffisante** : Si cette option n'est pas cochée, les mémos image sont toujours affichés. Si cette option est cochée, les mémos images seront affichés uniquement si la qualité de la connexion est suffisante. Cette option permet d'optimiser les accès et l'affichage des données.

- **Afficher la base système des serveurs HFSQL (nécessite le droit de connexion à la base)** :  Permet d'afficher la base système des serveurs HFSQL. 

- **Mémoriser les volets affichés (connexions HFSQL Client/Serveur uniquement)** : Cette option permet de mémoriser les vues ouvertes (volets affichant les informations sur le serveur, la base de données, le fichier de données, la requête, etc.) lorsque le Centre de Contrôle est fermé ou lors d'une déconnexion. Les volets sont automatiquement réouverts à la prochaine connexion. Cette option est sélectionnée par défaut. 
	L'option "réinitialiser" permet de réinitialiser les vues mémorisées. 

- **Ordre d'affichage des rubriques** : Permet de spécifier l'ordre d'affichage des rubriques utilisé lors de l'affichage de la description des fichiers de données (onglet "Description" du fichier de données). Il est possible de choisir l'ordre logique (par défaut) ou l'ordre alphabétique. 

- **Activer la gestion de l'intégrité lors de l'exécution des requêtes SQL** : Permet de prendre en compte la gestion de l'intégrité lors de l'exécution des requêtes SQL depuis le Centre de Contrôle HFSQL.

- **Activer la gestion des doublons lors de l'exécution des requêtes SQL** : Permet de prendre en compte la gestion des doublons lors de l'exécution des requêtes SQL depuis le Centre de Contrôle HFSQL.

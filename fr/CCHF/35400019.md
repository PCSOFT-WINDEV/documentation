
## Centre de Contrôle HFSQL : Créer et exécuter une requête
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000097"></a>
Le Centre de Contrôle HFSQL permet de créer une requête sur les fichiers de données de la base de données actuellement affichée dans le Centre de Contrôle. 

Pour créer une requête : 

1. Dans le ruban, dans le groupe "Requête SQL", déroulez "Nouvelle requête" puis sélectionnez "Requête vierge". 

2. Un nouvel onglet s'affiche, correspondant à la requête en cours de création. 

3. Sélectionnez si nécessaire la base de données sur laquelle la requête doit être effectuée et saisissez le code SQL de la requête. 

4. Exécutez la requête via l'option "Exécuter" du ruban. 




Remarque : La requête peut également être créé via un assistant.

<a name="NOTE2_1"></a>


## Requête : Les options du ruban
<a name="requete_les_options_ruban_ELTTEXTE000121"></a>
Le ruban des requêtes propose les options suivantes : 

Les options du groupe "Requête SQL" sont les suivantes : 

- Nouvelle requête : Permet de créer une nouvelle requête, soit directement en saisissant le code SQL, soit via l'assistant. 

- Ouvrir une requête : Permet d'ouvrir une requête précédemment enregistrée ou provenant de l'historique. 

- Enregistrer : Permet d'enregistrer la requête en cours. 

- Fermer la requête : Permet de fermer la requête actuellement affichée. 




Les options du groupe "Exécution" sont les suivantes : 

- Exécuter : Permet d'exécuter la requête affichée dans l'onglet en cours. 

- Suggérer les clés : Suggère les clés permettant d'optimiser les performances de la requête. 

- Afficher le EXPLAIN : Permet d'afficher graphiquement ou sous forme XML le EXPLAIN de la requête. 




Les options du groupe "Transactions" sont les suivantes : 

- Débuter une transaction : Permet de débuter une transaction avant d'exécuter la requête. Lorsque la transaction est débutée, il est possible à tout moment de valider cette transaction via le bouton "Valider la transaction". 

- Annuler la transaction : Permet si nécessaire d'annuler la transaction en cours. 




Les options du groupe "Autres actions" sont les suivantes : 

- Historique des requêtes : permet d'obtenir la liste des requêtes effectuées depuis le Centre de Contrôle HFSQL, quelles aient été exécutées ou en échec. 

- Liste des mots-clés SQL : permet d'obtenir la liste des mots-clés SQL utilisables dans les requêtes SQL.

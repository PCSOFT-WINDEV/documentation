
## Gestion des applications HFSQL Client/Serveur
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000112"></a>
Le Centre de Contrôle HFSQL permet de gérer les serveurs HFSQL Client/Serveur et les bases de données HFSQL Client/Serveur.

Depuis le Centre de Contrôle, il est possible :

- de manipuler [un serveur HFSQL](../CCHF/35400014.md).

- de manipuler [les bases de données HFSQL Client/Serveur](../CCHF/35400015.md).

- de manipuler [les fichiers de données HFSQL Client/Serveur](../CCHF/35400016.md).

## Se connecter à un serveur HFSQL
<a name="connecter_serveur_hfsql_ELTTEXTE000142"></a>
Pour se connecter à un serveur HFSQL, il est possible de :

- spécifier une nouvelle connexion.

- utiliser une connexion récente.




**Pour indiquer une nouvelle connexion** : 

1. Lancez le Centre de Contrôle HFSQL (si ce n'est pas déjà fait).

2. Dans l'écran de bienvenue, cliquez sur "Se connecter à un serveur HFSQL". 

3. Précisez : <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=cc_HF_Presentation%20-%20HC%20N%B0001%201.gif&type=thumb)


	- le nom ou l'adresse IP du serveur, 

- le port,

- le nom de la base de données (facultatif), 

- l'utilisateur et le mot de passe. 

4. Validez. Le serveur HFSQL apparaît dans le Centre de Contrôle HFSQL. Le nom du serveur et ses différentes bases de données HFSQL Client/Serveur sont listés dans la partie gauche de l'écran.




**Pour utiliser une connexion récente** : 

1. Lancez le Centre de Contrôle HFSQL (si ce n'est pas déjà fait).

2. Dans l'écran de bienvenue, cliquez sur "Se connecter à un serveur HFSQL". 

3. Sélectionnez la connexion souhaitée (dans la zone "Connexions récentes"), puis validez. Le serveur HFSQL apparaît dans le Centre de Contrôle HFSQL. Le nom du serveur et ses différentes bases de données HFSQL Client/Serveur sont listés dans la partie gauche de l'écran.




Remarque : Pour changer la connexion en cours dans le Centre de Contrôle HFSQL : 

1. Cliquez sur ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Passage%20analyse%20en%20CS%20-%20HC%20N%B0009%201.gif). 

2. Dans le menu, sélectionnez "Se connecter à un serveur HFSQL". 

3. La fenêtre de bienvenue s'affiche. Connectez-vous au serveur voulu.

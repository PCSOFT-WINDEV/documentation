


## Configuration du serveur HFSQL
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000226"></a>
Le Centre de Contrôle HFSQL propose une gestion complète des serveurs HFSQL.

Pour afficher les caractéristiques d'un serveur HFSQL : 

1. Connectez-vous si nécessaire à un serveur HFSQL. Pour plus de détails, consultez [Gestion des bases de données HFSQL Client/Serveur](../CCHF/35400013.md). 

2. Double-cliquez sur le nom du serveur HFSQL dans l'arborescence. 

3. La partie droite du Centre de Contrôle affiche un onglet au nom du serveur. Cet onglet est constitué de : 

	- différents onglets permettant de gérer les caractéristiques du serveur HFSQL.

	- un ruban permettant de configurer le serveur HFSQL. 





Dans le ruban, dans le groupe "Paramètres", l'option "Configuration du serveur" permet de gérer toutes les options de configuration du serveur HFSQL. 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CC_HF_Gestion_Serveur%20-%20HC%20N%B0001%202.gif&type=thumb)

## Général
<a name="general_ELTTEXTE000256"></a>
Dans la fenêtre de configuration du serveur, cliquez sur "Général". Les options suivantes apparaissent : 

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CC_HF_Gestion_Serveur%20-%20HC%20N%B0007.gif&type=thumb)


Vous pouvez configurer : 

- **Emplacement des bases de données**, c'est-à-dire les fichiers de données des différentes applications HFSQL Client/Serveur. Des raccourcis peuvent être utilisés pour indiquer les répertoires spécifiques : 

	- "%%EXE%%" : Répertoire du service Manta.

	- "%%COMMONAPPDATA%%" : Répertoire des données commun à toutes les applications. 




- **Calcul des statistiques sur les clés** : L'activation de cette option permet d'optimiser automatiquement l'exécution des requêtes et des filtres. 

- **Répertoire local de stockage** : Ce répertoire contient toutes les données récupérées (logs et statistiques) afin d'effectuer des consolidations sur de longues périodes.

## Logs
<a name="logs_ELTTEXTE000304"></a>
Dans la fenêtre de configuration du serveur, cliquez sur "Logs du serveur". Les options suivantes apparaissent : 

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CC_HF_Gestion_Serveur%20-%20HC%20N%B0009.gif&type=thumb)


Il est possible : 

- d'indiquer le répertoire des fichiers de log. 

- d'activer l'enregistrement des appels au serveur dans le fichier de log. 

- d'activer l'enregistrement des paramètres des appels dans le fichier de log. 




Il est également possible d'indiquer la taille maximale du fichier de log.

## Statistiques d'activité
<a name="statistiques_activite_ELTTEXTE000328"></a>
Dans la fenêtre de configuration du serveur, cliquez sur "Statistiques d'activité". Les options suivantes apparaissent : 

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CC_HF_Gestion_Serveur%20-%20HC%20N%B0010.gif&type=thumb)


Les statistiques d'activité permettent de mesurer la charge du serveur. Il est possible : 

- d'activer les statistiques d'activité, 

- de spécifier le répertoire d'enregistrement, 

- de spécifier la taille maximale du fichier,

- de spécifier la période de calcul des performances (en secondes).

## Caches
<a name="caches_ELTTEXTE000376"></a>
Dans la fenêtre de configuration du serveur, cliquez sur "Caches". Les options suivantes apparaissent :  
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CC_HF_Gestion_Serveur%20-%20HC%20N%B0012.gif&type=thumb)


- Taille du cache des index.
	**Remarque** : en 32 bits, la taille du cache des index est limitée à 1Go.

- Cache disque (serveur Windows uniquement) : La gestion standard des caches disques de Windows en mode 64 bits (indépendamment de HFSQL) peut amener à des saturations mémoire. Cette option permet de personnaliser la gestion des caches mémoire.

## Notifications
<a name="notifications_ELTTEXTE000400"></a>
Dans la fenêtre de configuration du serveur, cliquez sur "Notifications".

Cette option permet de gérer les destinataires des notifications. 

Attention : la configuration de l'envoi des notifications est nécessaire avant d'ajouter des destinataires. 

Pour plus de détails, consultez [Notifications d'un serveur HFSQL](../WDLang4/1000017311.md).

## Version
<a name="version_ELTTEXTE000448"></a>
Dans la fenêtre de configuration du serveur, cliquez sur "Version".

Cette option permet de : 

- Connaître la version actuelle du serveur : 

	- Version majeure : Version correspondant à la version du moteur HFSQL utilisé par le serveur. 

	- Version mineure : Numéro de la mise à jour utilisée. 

	- Langue : Langue utilisée pour le serveur. 


Le lien "Plus de détails" permet d'obtenir le détail des modules utilisés. 

- Mettre en place la mise à jour automatique du serveur. Si cette option est activée, le serveur HFSQL peut être mis à jour automatiquement dès qu'une nouvelle version est disponible : 

	- sur le serveur PC SOFT. 

	- sur un serveur personnalisé. 


Pour plus de détails, consultez [Mise à jour du serveur](../CCHF/35400018.md).

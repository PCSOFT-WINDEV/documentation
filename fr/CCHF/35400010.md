
## Gestion des fichiers de données HFSQL Classic 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000198"></a>
Le Centre de Contrôle HFSQL propose une gestion complète des fichiers de données HFSQL Classic. 

**Pour afficher les caractéristiques d'un fichier de données HFSQL Classic** :

1. Affichez si nécessaire la fenêtre de bienvenue du Centre de Contrôle HFSQL. Cette fenêtre apparaît : 

	- au lancement du Centre de Contrôle HFSQL. 

	- en déroulant ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CC_HF_Gestion_Fichier_HFCS%20-%20HC%20N%B0003%201.gif)
 puis en sélectionnant "Ouvrir un fichier HFSQL Classic". 




2. La fenêtre de bienvenue du Centre de Contrôle s'affiche. 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CC_HF_FichierHFSQLClassic%20-%20HC%20N%B0001.gif&type=thumb)


3. Indiquez : 

	- le chemin du fichier de données HFSQL Classic à ouvrir, 

	- son mot de passe (si nécessaire). Il est possible de mémoriser le mot de passe. 




4. Validez. 

5. Le Centre de Contrôle HFSQL affiche le contenu du fichier de données sélectionné. 

	- Les différents onglets permettent de gérer les caractéristiques du fichier de données HFSQL Classic.

	- Le ruban propose les différentes actions possibles sur le fichier. Selon l'onglet sélectionné, les actions proposées peuvent varier.

## Détail des caractéristiques d'un fichier de données HFSQL Classic
<a name="detail_des_caracteristiques_fichier_donnees_hfsql_classic_ELTTEXTE000228"></a>


### Onglet "Description"
<a name="onglet_description_ELTPARAGRAPHE000043"></a>Cet onglet présente toutes les informations concernant les rubriques du fichier de données.


![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CCHF_Fichiers_HFClassic%20-%20HC%20N%B0001.gif&type=thumb)


Les informations affichées sont les suivantes :

- Le nom de la rubrique,

- Le type de la rubrique,

- La taille de la rubrique,

- Le type de clé,

- Les statistiques,

- Si la rubrique est visible sous le logiciel "Etats & Requêtes".




Si nécessaire, des informations complémentaires sont affichées.

Remarque : Par défaut, les rubriques sont affichées dans l'ordre logique. il est possible d'utiliser l'ordre alphabétique si nécessaire via les options du Centre de Contrôle HFSQL : 

1. Déroulez le menu du Centre de Contrôle HFSQL (![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CC_Suivi_Configuration%20-%20HC%20N%B0001.gif)
). 

2. Sélectionnez "Options".

3. Dans la fenêtre qui s'affiche, indiquez "alphabétique" pour l'option "Ordre d'affichage des rubriques". 



<a name="NOTE2_2"></a>


### Onglet "Propriétés"
<a name="onglet_proprietes_ELTPARAGRAPHE000083"></a>L'onglet "Propriétés" affiche les propriétés du fichier de données (nom physique, format, protection, ...).

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CCHF_Fichiers_HFClassic%20-%20HC%20N%B0002.gif&type=thumb)



<a name="NOTE2_3"></a>


### Onglet "Contenu"
<a name="onglet_contenu_ELTPARAGRAPHE000092"></a>Cet onglet permet de visualiser les différents enregistrements du fichier de données. 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CCHF_Fichiers_HFClassic%20-%20HC%20N%B0003.gif&type=thumb)


Le menu contextuel des enregistrements permet de manipuler l'enregistrement sélectionné : suppression, ajout, mode d'affichage des mémos texte, ...

Il est également possible de :

- Supprimer un enregistrement : sélectionnez l'enregistrement et cliquez sur le bouton "Supprimer" (ou appuyez sur la touche Suppr).

- Modifier ou ajouter un enregistrement : la saisie se fait directement dans la liste des enregistrements. Il suffit de se positionner sur la première ligne vide et de saisir les informations de l'enregistrement. La modification du fichier de données est faite automatiquement.



<a name="NOTE2_4"></a>


### Onglet "Recherche"
<a name="onglet_recherche_ELTPARAGRAPHE000107"></a>L'onglet "Recherche" permet de réaliser des recherches dans le fichier de données.

L'onglet "Recherche" permet de saisir les conditions de recherche de manière explicite (recherches QBE). Pour la rubrique voulue, il suffit de sélectionner une condition et une valeur à tester. 


![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CCHF_Fichiers_HFClassic%20-%20HC%20N%B0004.gif&type=thumb)

<a name="NOTE2_6"></a>


### Onglet "Journaux"
<a name="onglet_journaux_ELTPARAGRAPHE000118"></a>Cet onglet permet de visualiser le contenu des journaux, d'effectuer une recherche dans les journaux.

Il est nécessaire que le fichier de données soit journalé.
<a name="NOTE2_7"></a>


### Onglet "Transactions"
<a name="onglet_transactions_ELTPARAGRAPHE000127"></a>Cet onglet permet de libérer un enregistrement ou toutes les transactions.


![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CCHF_Fichiers_HFClassic%20-%20HC%20N%B0005.gif&type=thumb)

## Options proposées par le ruban
<a name="options_proposees_par_ruban_ELTTEXTE000282"></a>
Pour le fichier HFSQL Classic sélectionné, le ruban propose les actions suivantes : 

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CCHF_Fichiers_HFClassic%20-%20HC%20N%B0001%201.gif)


- Dans le groupe "Requête SQL" : 

	- Nouvelle requête : Cette option permet d'ouvrir un onglet spécifique permettant de réaliser une requête sur le fichier de données en cours.  

	- Ouvrir une requête : Cette option permet d'ouvrir un onglet spécifique correspondant à une requête précédemment réalisée. 




- Dans le groupe "Fichier" : 

	- Optimiser et réparer : Cette option permet d'afficher la fenêtre de configuration de l'optimisation et de la réparation des fichiers HFSQL. 

	- Fermer le fichier : Cette option permet de fermer le fichier et l'onglet associé.




## Centre de Contrôle HFSQL : Ouvrir une analyse
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000181"></a>
Le Centre de Contrôle HFSQL propose une gestion complète d'une analyse. Il donne accès à toutes les bases de données présentes dans l'analyse : HFSQL Classic, HFSQL Client/Serveur et bases tierces.

Les caractéristiques d'une analyse peuvent être affichées : 

- soit directement en lançant le Centre de Contrôle HFSQL depuis un projet : 

	1. Ouvrez le projet WINDEV, WINDEV Mobile ou WEBDEV. 

	2. Sous le volet "Outils", dans le groupe "Base de données", cliquez sur "HFSQL". 

	3. La fenêtre de bienvenue du Centre de Contrôle HFSQL est affichée. L'option "Ouvrir une analyse" est sélectionnée et le chemin de l'analyse du projet en cours est spécifié. <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Passage%20analyse%20en%20CS%20-%20HC%20N%B0008.gif&type=thumb)


	4. Validez. L'analyse est ouverte dans le Centre de Contrôle HFSQL. 

- soit directement depuis le Centre de Contrôle HFSQL : 

	1. Lancez le Centre de Contrôle HFSQL. 

	2. Dans la fenêtre de Bienvenue, cliquez sur "Ouvrir une analyse". 

	3. Indiquez le chemin de l'analyse (fichier WDD) et le mot de passe de l'analyse si nécessaire. 

	4. Validez. L'analyse est ouverte dans le Centre de Contrôle HFSQL.

## Caractéristiques de l'analyse
<a name="caracteristiques_analyse_ELTTEXTE000211"></a>


### Onglet "HFSQL Classic"
<a name="onglet_hfsql_classic_ELTPARAGRAPHE000056"></a>Cet onglet présente toutes les informations concernant les fichiers de données au format HFSQL Classic présents dans l'analyse.
Les informations affichées sont les suivantes :

- Nom logique du fichier de données,

- Chemin complet (s'il existe),

- Taille du fichier de données,

- Nombre d'enregistrements actifs,

- Nombre d'enregistrements supprimés,

- Nombre total d'enregistrements.





### Onglet "HFSQL C/S"
<a name="onglet_hfsql_ELTPARAGRAPHE000069"></a>Cet onglet présente toutes les informations concernant les fichiers de données au format HFSQL Client/Serveur présents dans l'analyse.
Les informations affichées sont les suivantes :

- Nom logique du fichier de données,

- Serveur,

- Base de données associée,

- Taille du fichier de données,

- Nombre d'enregistrements actifs,

- Nombre d'enregistrements supprimés,

- Nombre total d'enregistrements.





### Onglet "Base tierce"
<a name="onglet_base_tierce_ELTPARAGRAPHE000083"></a>Cet onglet présente toutes les informations concernant les fichiers de données au format non HFSQL présents dans l'analyse.
Les informations affichées sont les suivantes :

- Nom logique du fichier de données,

- Type,

- Source de données,

- Nombre total d'enregistrements.





### Onglet "Liaisons"
<a name="onglet_liaisons_ELTPARAGRAPHE000094"></a>Cet onglet présente toutes les informations concernant les liaisons entre les fichiers de données présents dans l'analyse.

Les informations affichées sont les suivantes :

- Nom de la liaison, 

- Nom du fichier de données source, 

- Cardinalités source, 

- Nom du fichier de données destination, 

- Cardinalités destination, 

- Règle d'intégrité de suppression, 

- Règle d'intégrité de modification. 




Remarque : La création et la modification de liaisons sont réalisées sous l'éditeur d'analyses.

## Options du ruban
<a name="options_ruban_ELTTEXTE000253"></a>
Lors de l'ouverture d'une analyse dans le Centre de Contrôle HFSQL, le ruban propose les options suivantes : 

- Dans le groupe "Gestion des fichiers" : 

	- **Synchronisation des fichiers** : Cette option permet de lancer la modification automatique des fichiers de données afin de synchroniser les fichiers de données et leur description dans l'analyse. Cette opération est réalisée via un assistant. 

	- **Ajouter un répertoire de déploiement** : Cette option permet de sélectionner un répertoire dans lequel les fichiers de données correspondant à l'analyse doivent être recherchés. 

	- **Gestion du .REP** : Cette option permet de lister les emplacements des fichiers de données enregistrés dans le fichier .REP lié à l'analyse. Il est possible d'ajouter, supprimer des fichiers dans le .REP. 




- Dans le groupe "Fichiers" : Ce groupe apparaît uniquement pour les onglets d'accès aux fichiers de données. Il permet notamment d'accéder directement aux fichiers de données. Pour les fichiers de données HFSQL, il est également possible de lancer une optimisation et une réparation des fichiers de données.

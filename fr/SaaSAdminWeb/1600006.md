
## Administrateur SaaS - Gérer les abonnements
			





## Présentation
<a name="presentation_ELTTEXTE000111"></a>
Les abonnements SaaS permettent la mise à disposition d'un site SaaS pour un compte client.

Pour chaque tarification définie pour un Site SaaS, un compte client peut souscrire à un abonnement et obtenir ainsi un certain nombre de licences qu'il répartira entre ses utilisateurs.

**Remarque** : La gestion des abonnements peut se faire uniquement avec le compte "admin". Les utilisateurs ayant les droits "Administrateur" peuvent toutefois changer la répartition des licences pour les sites auxquels ils sont abonnés.





## Abonner un compte client à un site SaaS
<a name="abonner_compte_client_site_saas_ELTTEXTE000135"></a>


### Abonner un compte client à un site SaaS
<a name="abonner_compte_client_site_saas_ELTPARAGRAPHE000017"></a>Pour abonner un compte client à un site SaaS depuis le site : 

1. Ouvrez la page de gestion du site SaaS (option "Sites .. Gérer un site" du menu principal de l'Administrateur SaaS).

2. Dans la boîte de dialogue, sélectionnez le nom du site SaaS concerné et validez. 

3. Dans la page de gestion du site, cliquez sur le lien "Abonner un client".

4. Une boîte de dialogue apparaît : choisissez le compte client et cliquez sur "OK".

5. La page de gestion des abonnements s'affiche.




Pour abonner un compte client à un site SaaS depuis le compte client :

1. Ouvrez la page de gestion du compte client (option "Clients .. Gérer un compte client" du menu principal de l'Administrateur SaaS).

2. Dans la table des sites SaaS de la partie "Abonnements" de la page, cliquez sur le bouton dans la colonne "Modifier".

3. La page de gestion des abonnements s'affiche.








## Gérer un abonnement
<a name="gerer_abonnement_ELTTEXTE000159"></a>


### Abonner un compte client à une tarification
<a name="abonner_compte_client_une_tarification_ELTPARAGRAPHE000036"></a>Pour abonner un compte client à une tarification :

1. Cochez la case "abonné" située en dessous de la tarification choisie.

2. Un champ de saisie apparaît. Saisissez le nombre de licences à accorder au compte client.

3. Choisissez la date de fin de l'abonnement en cliquant sur le lien en-dessous du nombre de licences. Il est possible d'indiquer une date de fin ou de choisir un abonnement à durée illimité (dans le cas d'un abonnement renouvelé automatiquement par exemple).

4. Cliquez sur le bouton "Valider" en bas de page pour enregistrer les modifications.








## Gérer les licences
<a name="gerer_les_licences_ELTTEXTE000183"></a>


### Dans le cas d'une tarification au forfait
<a name="dans_cas_une_tarification_forfait_ELTPARAGRAPHE000048"></a>Il n'y a pas de limite du nombre d'utilisateurs pouvant bénéficier de la tarification. 

Pour gérer les licences : 

1. Cochez tous les utilisateurs devant obtenir l'accès à cette tarification. 

2. Validez le bouton en bas de page.





### Dans le cas d'une tarification par utilisateur
<a name="dans_cas_une_tarification_par_utilisateur_ELTPARAGRAPHE000058"></a>Pour gérer les licences : 

1. Cochez le ou les utilisateurs devant obtenir l'accès (dans la limite du nombre de licences). 

2. Validez le bouton en bas de page.


## Administrateur SaaS - Gérer un client
			

<a name="NOTE0"></a>
<a name="NOTE0_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000165"></a>
La gestion d'un compte client SaaS permet de : 

- [Modifier les informations générales du compte client](#NOTE1_1). 

- [Modifier les abonnements du compte client](#NOTE2_1). 

- [Gérer les utilisateurs associés au compte client](#NOTE3_1).

- [Modifier la valeur des paramètres des différents sites SaaS correspondants aux abonnements du compte client](#NOTE4_1).




La suppression d'un compte client se fait par le lien "Supprimer ce compte client" en bas de la page de gestion du compte.
**Remarque** : Supprimer un compte client entraîne la suppression définitive de ses abonnements en cours, de ses utilisateurs, ainsi que de toutes ses visites.

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Modification des informations générales
<a name="modification_des_informations_generales_ELTTEXTE000189"></a>
Le nom du compte client ainsi que la référence externe, saisis lors de la création du compte peuvent être modifiés.

Il est également possible de saisir des informations diverses concernant le compte client.

Pour que ces modifications soient prises en compte, il faut cliquer sur le bouton "Valider" en bas de la page.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Modification des abonnements
<a name="modification_des_abonnements_ELTTEXTE000213"></a>
Depuis la page de gestion d'un compte client, il est possible de voir les différents sites SaaS auxquels ce compte est abonné ainsi que le nombre de licences acquises pour chaque tarification.

En cliquant sur l'icône de la colonne "Modifier", la page de gestion des abonnements s'affiche.

Pour plus de détails, consultez [Gestion des Abonnements](../SaaSAdminWeb/1600006.md).

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Gestion des utilisateurs
<a name="gestion_des_utilisateurs_ELTTEXTE000237"></a>
La page de gestion d'un compte client liste les différents utilisateurs appartenant à ce compte.

En cliquant sur le lien "Ajouter un utilisateur", il est possible de déclarer de nouveaux utilisateurs. Les informations demandées sont les suivantes :

| Elément | Rôle |
| --- | --- |
| *Nom* | Nom de l'utilisateur (paramètre optionnel). |
| *Prénom* | Prénom de l'utilisateur (paramètre optionnel). |
| *Login* | Identifiant de connexion aux sites SaaS.<br>**Note** : après sa création, il faut allouer des licences d'Abonnements à l'utilisateur pour lui permettre d'accéder aux Sites SaaS. |
| *Mot de passe* | Mot de passe de l'utilisateur |
| *Administrateur* | Si la case "Administrateur" est cochée, l'utilisateur créé sera un administrateur pour le compte client en cours.<br>Cela signifie qu'il pourra (en utilisant l'Administrateur SaaS ou la fonction [SaaSEcritParamètreSite](../WDLang3/1000019039.md)) modifier les paramètres du compte client, gérer les utilisateurs et les abonnements et modifier les paramètres des sites SaaS. |
| *Mot de passe à saisir* | Si la case "Mot de passe à saisir" est cochée, à la prochaine connexion de l'utilisateur, un message lui indiquera que son mot de passe est périmé et lui demandera de saisir un nouveau mot de passe.<br><br>Remarque : Si cette case est cochée lors de la création de l'utilisateur, il faut tout de même indiquer un mot de passe pour la première connexion de l'utilisateur. |

## Modification des paramètres
<a name="modification_des_parametres_ELTTEXTE000261"></a>
Lors de la déclaration d'un site SaaS dans l'Administrateur SaaS, il est possible de définir une série de paramètres. Ces paramètres peuvent ensuite être utilisés en programmation pour personnaliser l'interface ou le fonctionnement du site SaaS en fonction du client.

Dans la page de gestion d'un compte client, il est possible de saisir les valeurs associées aux différents paramètres des sites pour lesquels le compte client possède des abonnements.

Il suffit de saisir la valeur dans le champ en regard du nom du paramètre.

Typiquement, on retrouvera des paramètres comme "Logo" ayant pour valeur l'URL du fichier image logo du compte client.

A la fin de la saisie, confirmez les nouvelles valeurs grâce au bouton "Valider".

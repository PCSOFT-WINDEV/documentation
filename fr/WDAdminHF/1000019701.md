
## WDAdminHF : Gestion du serveur
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000160"></a>
Lors de la connexion à l'administrateur HFSQL distant, il est nécessaire de préciser le serveur HFSQL pris en compte. Les différentes options disponibles sur le serveur choisi sont alors listées. 



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Détail des options disponibles
<a name="detail_des_options_disponibles_ELTTEXTE000190"></a>


### Bases de données
<a name="bases_donnees_ELTPARAGRAPHE000019"></a>Cette option permet de sélectionner une des bases de données présente sur le serveur. Il est ainsi possible d'accéder à toutes les options de [gestion de la base de données](../WDAdminHF/1000019702.md). 


### Connexions
<a name="connexions_ELTPARAGRAPHE000027"></a>Cette option liste les différents utilisateurs connectés sur les différentes bases de données présentes sur le serveur.

La liste des connexions actives contient les informations suivantes :

- le nom de la base de données,

- le nom de l'utilisateur,

- le nom de la machine,

- le nom de l'application qui accède à la base de données, ...




Il est possible de gérer les connexions depuis cette page : interdire ou autoriser les connexions, déconnecter les utilisateurs, ou encore envoyer un message aux utilisateurs.


### Transactions
<a name="transactions_ELTPARAGRAPHE000041"></a>Cette option permet de gérer les transactions en cours ou interrompues sur le serveur.


### Utilisateurs
<a name="utilisateurs_ELTPARAGRAPHE000046"></a>Cette option permet de connaître et de gérer les utilisateurs du serveur. Il est possible de créer un utilisateur, de le modifier ou de le supprimer. 

Lors de la création / modification d'un utilisateur, l'utilisateur peut être un super utilisateur : les utilisateurs désignés comme "super utilisateur" sont autorisés à effectuer toutes les actions sur le serveur, les bases de données et sur tous les fichiers de données.


### Groupes
<a name="groupes_ELTPARAGRAPHE000053"></a>Cette option permet de connaître et de gérer les groupes d'utilisateurs du serveur. Il est possible de créer un groupe d'utilisateurs, de le modifier ou de le supprimer. 

Il est également possible d'associer des utilisateurs à un groupe. 


### Droits
<a name="droits_ELTPARAGRAPHE000060"></a>Cette option permet de gérer les droits sur le serveur pour l'utilisateur ou le groupe sélectionné.


### Informations système
<a name="informations_systeme_ELTPARAGRAPHE000065"></a>Cette option permet de connaître : 

- la plateforme du serveur.

- la version de la plateforme.

- l'architecture du moteur.

- le processeur du serveur.

- la mémoire disponible.

- la version des éléments sur le serveur.





### Configuration du serveur
<a name="configuration_serveur_ELTPARAGRAPHE000077"></a>Cette option permet de configurer le serveur. Les différents éléments pris en compte sont :

- Emplacement des bases de données sur le serveur (option "Général")

- Répertoire des journaux et de leur sauvegarde (option "Journaux")

- Fichier de log (option "Logs") : il est possible d'indiquer le répertoire des fichiers de log et d'activer certaines options d'enregistrement. Il est également possible d'indiquer la taille maximale du fichier de log.

- Paramètres système (option "Système") : port d'écoute, compte utilisateur du serveur, port de débogage à distance, ...

- Taille du cache des index (option "Caches"). 
	Remarque : en 32 bits, la taille du cache des index est limité à 1Go.

- Statistiques d'activité (option "Stat. d'activités") : Les statistiques d'activité permettent de mesurer la charge du serveur. Il est possible d'activer les statistiques d'activité et de spécifier le répertoire d'enregistrement.







### Logs
<a name="logs_ELTPARAGRAPHE000092"></a>Cette option permet de visualiser directement les logs du serveur. Cette option est disponible uniquement si les logs sont activés sur le serveur.

Il est également possible d'importer des logs. 


### Sauvegardes
<a name="sauvegardes_ELTPARAGRAPHE000099"></a>Cette option permet de réaliser et de restaurer des sauvegardes du serveur et/ou des bases de données. Pour plus de détails, consultez [Réaliser des sauvegardes avec l'administrateur distant HFSQL](../WDAdminHF/1000019704.md). 


### Tâches planifiées
<a name="taches_planifiees_ELTPARAGRAPHE000107"></a>Cette option permet de créer, modifier et supprimer des tâches planifiées sur le serveur. Pour plus de détails, consultez [Gérer les tâches planifiées avec l'administrateur distant HFSQL](../WDAdminHF/1000019705.md).

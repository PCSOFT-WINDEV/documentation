### L'administrateur HFSQL
<a name="administrateur_hfsql_ELTPARAGRAPHE000010"></a>L'administrateur HFSQL distant est un site WEBDEV installé avec le serveur d'application WEBDEV et permettant de gérer à distance les bases de données HFSQL Client/Serveur installées sur le serveur Web. 



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Lancement de l'administrateur HFSQL distant
<a name="lancement_administrateur_hfsql_distant_ELTTEXTE000143"></a>
Pour utiliser l'administrateur HFSQL distant, le responsable de sites doit connaître l'adresse de lancement de l'administrateur HFSQL distant.

Sur un serveur Windows, cette adresse est du type :

```txt
http://PosteServeur/WD260AWP/WD260AWP.EXE/CONNECT/WDAdminhf260
```


où PosteServeur correspond au nom du serveur Web où est installé le serveur d'application WEBDEV. Ce nom peut être :

- soit une adresse IP (par exemple : 192.168.1.130).

- soit le nom du poste (par exemple : PosteDoc).




Sur un serveur Linux, cette adresse est du type :

```txt
http://PosteServeur/WD260AWP/WD260AWP/CONNECT/WDAdminhf260
```
ou

```txt
http://PosteServeur/WD260AWP/WD260Awp.exe/CONNECT/WDAdminhf260
```


où PosteServeur correspond au nom du serveur Web où est installé le serveur d'application WEBDEV. Ce nom peut être :

- soit une adresse IP (par exemple : 192.168.1.130).

- soit le nom du poste (par exemple : PosteDoc).

## Interface de l'administrateur HFSQL distant
<a name="interface_administrateur_hfsql_distant_ELTTEXTE000167"></a>
Lors du lancement de l'administrateur HFSQL distant, une page de connexion est affichée. Les informations à fournir sont : 

- le nom ou l'adresse du serveur HFSQL,

- le port de connexion,

- le nom d'utilisateur et le mot de passe associé (ADMIN sans mot de passe par défaut). 




Il suffit de cliquer sur le bouton "Connexion" pour accéder à toutes les options disponibles sur le serveur spécifié. 

L'administrateur distant HFSQL permet de gérer les bases de données HFSQL Client/Serveur. Il est possible d'obtenir des informations et de manipuler : 

- [le serveur](../WDAdminHF/1000019701.md),

- [la base de données](../WDAdminHF/1000019702.md),

- [chaque fichier de la base de données](../WDAdminHF/1000019703.md).

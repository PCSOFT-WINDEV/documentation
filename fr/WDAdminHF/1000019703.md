
## WDAdminHF : Gestion des fichiers de données
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000141"></a>
L'administrateur distant HFSQL propose une gestion complète des bases de données HFSQL Client/Serveur. 

**Pour afficher les caractéristiques d'un fichier de données HFSQL Client/Serveur** : 

1. Connectez-vous au serveur HFSQL voulu à l'aide de l'administrateur HFSQL distant. 

2. Sélectionnez l'option "Bases de données". 

3. Sélectionnez la base de données voulue et cliquez sur le lien "Accéder à la base de données sélectionnée". 

4. Les options affichées sont les options disponibles sur la base de données en cours. Cliquez sur "Fichiers". 

5. Cliquez sur le nom du fichier de données à gérer. Les options disponibles sont affichées. 






<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Détail des options disponibles
<a name="detail_des_options_disponibles_ELTTEXTE000171"></a>


### Connexions
<a name="connexions_ELTPARAGRAPHE000027"></a>Cette option liste les différents utilisateurs connectés sur le fichier de données en cours.

La liste des connexions actives contient les informations suivantes :

- le nom de l'utilisateur,

- le nom de la machine,

- le nom de l'application qui accède à la base, ...




Il est possible de gérer les connexions depuis cette page : interdire ou autoriser les connexions, déconnecter les utilisateurs, ou encore envoyer un message aux utilisateurs.


### Description
<a name="description_ELTPARAGRAPHE000040"></a>Cette option permet d'afficher la description des différentes rubriques du fichier de données (nom de la rubrique, taille, type, ...).


### Propriétés
<a name="proprietes_ELTPARAGRAPHE000045"></a>Cette option permet d'afficher les propriétés du fichier de données (nom physique, format, protection, ...).


### Recherche
<a name="recherche_ELTPARAGRAPHE000050"></a>Cette option permet de réaliser une recherche dans le fichier de données. Il est possible de saisir les conditions de recherche de manière explicite.


### Requête SQL
<a name="requete_sql_ELTPARAGRAPHE000055"></a>Cette option permet de réaliser une recherche dans le fichier de données. Il est possible de saisir directement le code SQL de la requête à effectuer.


### Droits
<a name="droits_ELTPARAGRAPHE000060"></a>Cette option permet de gérer les droits sur le fichier de données pour l'utilisateur ou le groupe sélectionné.


### Blocages
<a name="blocages_ELTPARAGRAPHE000065"></a>Cette option permet de gérer les blocages.

Les informations affichées sont les suivantes :

- le numéro de l'enregistrement bloqué,

- le nom de l'utilisateur qui bloque,

- le nom de la machine qui bloque,

- l'adresse IP de la machine qui bloque,

- le nom de l'application qui bloque,

- la date de début de blocage,

- la durée du blocage.





### Journal
<a name="journal_ELTPARAGRAPHE000080"></a>Cette option permet de visualiser le contenu des journaux et d'effectuer une recherche dans les journaux.

Il est nécessaire que le fichier soit journalé.




### Maintenance
<a name="maintenance_ELTPARAGRAPHE000088"></a>Cette option permet de libérer un enregistrement ou toutes les transactions.

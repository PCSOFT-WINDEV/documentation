## Consulter les informations à votre disposition
<a name="consulter_les_informations_votre_disposition_ELTTEXTE000071"></a>


- La documentation des différents produits,

- L'aide en ligne,

- Les assistants et l'aide accessible par le guide (utilisez la recherche par mot-clé),

- Les exemples fournis,

- Le site Internet PC SOFT, qui regroupe les problèmes le plus souvent rencontrés.




Les conditions d'apparition d'un problème sont déterminantes pour la recherche d'une solution. Ces quelques critères d'apparition sont primordiaux.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Connaître votre problème
<a name="connaitre_votre_probleme_ELTTEXTE000095"></a>


- Reproductibilité - Si votre problème apparaît aléatoirement, il n'est sans doute pas suffisamment isolé et il sera très difficile de vous apporter une aide efficace.

- Isolement - Il est important de vérifier que le comportement de votre code est indépendant du contexte. Pour cela, nous vous conseillons de chercher à le reproduire dans une nouvelle fenêtre, voire un nouveau projet.

- Précision - Si un message d'erreur apparaît, vous devez en noter le texte (au mot près) ou mieux en faire une Hard Copy.

<a name="recenser_les_solutions_essayees_ELTTEXTE000119"></a>
Avant de joindre le Support Technique Gratuit, il est important de nous décrire les solutions que vous avez testées et qui ont échoués. Elles permettront de répondre efficacement et rapidement à votre problème !

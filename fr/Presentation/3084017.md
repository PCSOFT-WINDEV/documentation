## Présentation
<a name="presentation_ELTTEXTE000099"></a>
Un Tutoriel est livré avec WINDEV, WEBDEV et WINDEV Mobile. Ce tutoriel a pour objectif de vous faire découvrir le produit à l'aide d'exercices simples.

Pour découvrir ce tutoriel, vous pouvez : 

- suivre le tutoriel en ligne : Découvrez la première leçon du tuto pour le produit qui vous intéresse : 

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) [Découvrez WINDEV](../TutoWD/1410087512.md).







Si vous avez choisi d'installer les exemples du tutoriel avec WINDEV, WINDEV Mobile ou WEBDEV, les différents exemples et exercices associés sont installés dans le sous-répertoire "Tutoriel" du répertoire d'installation.

Pour réaliser les manipulations sur les projets associés au tutoriel : 

1. Affichez la page d'accueil du produit : utilisez la combinaison de touches Ctrl + &lt;. 

2. Cliquez sur "Tutoriel". 

3. Via cette page, vous pouvez accéder au cours en ligne et aux différents projets associés à chaque leçon.

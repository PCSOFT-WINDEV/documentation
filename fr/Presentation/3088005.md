
## Logiciel Etats et requêtes : Spécificités

<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000168"></a>
Cette page d'aide présente les différentes spécificités du logiciel "Etats & Requêtes" (c'est-à-dire l'éditeur d'états Utilisateur) :

- Source de données de l'état.

- Fonctions de modification des enregistrements.

- Volets verticaux.

- Gabarit personnalisé d'états.

- Modification d'états présents dans un composant.




**Remarque** : L'icône suivante ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=er.png) (présent dans l'aide en ligne) permet d'indiquer les différences entre l'éditeur d'états Utilisateur et l'éditeur d'états intégré.

<a name="NOTE2_1"></a>


## Spécificités
<a name="specificites_ELTTEXTE000192"></a>


### Source de données de l'état
<a name="source_donnees_etat_ELTPARAGRAPHE000029"></a>Sous l'éditeur d'états Utilisateur, il n'est pas possible de créer :

- ***des états sur un vue HFSQL***. Cependant, il est possible de personnaliser des états sur une vue HyperFileSQL créés sous l'éditeur d'états intégré (couleur, polices, taille, ...).

- ***des états sur un champ Table***. Cependant, il est possible de personnaliser des états sur un champ Table créés sous l'éditeur d'états intégré (couleur, polices, taille, colonne reliée, ...).

- ***des états sur un champ Agenda ou Planning***. Cependant, il est possible de personnaliser des états sur un champ Agenda ou Planning créés sous l'éditeur d'états intégré (couleur, polices, taille, colonne reliée, ...).

- ***des états sur une source de données programmée***.




**Remarque** : Les fichiers de données et les rubriques pouvant être utilisés dans un état sur un fichier de données sont définis sous l'éditeur d'analyses de WINDEV. Pour plus de détails sur la préparation nécessaire à l'utilisation d'une application WINDEV avec l'éditeur d'états Utilisateur, consultez [Diffuser le logiciel Etats & Requêtes](../Presentation/3088002.md).

Pour plus de détails sur les différentes sources de données d'un état, consultez [Sources de données d'un état](../WDChamp/1011055.md).


<a name="NOTE2_2"></a>


### Fonctions de modification des enregistrements
<a name="fonctions_modification_des_enregistrements_ELTPARAGRAPHE000054"></a>Sous l'éditeur d'états Utilisateur, l'utilisation des fonctions de modification des enregistrements ([HAjoute](../WDLang4/3044147.md), [HModifie](../WDLang4/3044042.md), [HSupprime](../WDLang4/3044018.md), ...) est interdite.


<a name="NOTE2_3"></a>


### Volets ancrables
<a name="volets_ancrables_ELTPARAGRAPHE000071"></a>Sous l'éditeur d'états Utilisateur, les volets ancrables permettent d'accéder uniquement :

- aux principales fonctionnalités de l'éditeur d'états Utilisateur.

- aux éléments accessibles (états, requêtes, ...) de votre projet.




Pour plus de détails sur les volets ancrables, consultez [Volets ancrables](../Editeurs/2027001.md).


<a name="NOTE2_4"></a>


### Gabarit personnalisé d'états
<a name="gabarit_personnalise_etats_ELTPARAGRAPHE000087"></a>Sous l'éditeur d'états Utilisateur, il n'est pas possible de créer vos propres gabarits d'états.

Pour plus de détails sur les gabarits d'états, consultez [Création d'un gabarit d'état](../Editeurs/2016005.md).


<a name="NOTE2_5"></a>


### Modification d'états présents dans un composant externe
<a name="modification_etats_presents_dans_composant_externe_ELTPARAGRAPHE000100"></a>Il n'est pas possible à l'utilisateur final de modifier un état présent dans un composant externe avec le logiciel "Etats & Requêtes".

Pour plus de détails sur les composants externes, consultez [Composant externe](../Editeurs/2014011.md).


<a name="NOTE2_6"></a>


### Modification d'états présents dans une bibliothèque externe
<a name="modification_etats_presents_dans_une_bibliotheque_externe_ELTPARAGRAPHE000113"></a>Il n'est pas possible à l'utilisateur final de modifier un état présent dans une bibliothèque externe avec le logiciel "Etats & Requêtes".

Pour plus de détails sur les composants, consultez [Bibliothèque](../Editeurs/2030022.md).

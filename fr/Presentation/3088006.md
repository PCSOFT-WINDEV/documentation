## Présentation
<a name="presentation_ELTTEXTE000134"></a>
Une requête sert à interroger une base de données pour visualiser, insérer, modifier ou supprimer des données. La structure de la requête définit les données manipulées. Pour plus de détails, consultez [Les requêtes](../Editeurs/2032052.md).

**La création et la manipulation d'une requête s'effectuent sous l'éditeur de requêtes**.

Deux versions de l'éditeur de requêtes sont disponibles :

- ***l'éditeur de requêtes directement intégré***.
	Cette version est fournie en standard avec WINDEV, WEBDEV ou WINDEV Mobile. Cette version ne peut être utilisée que lors du développement d'une application WINDEV, WINDEV Mobile ou d'un site WEBDEV. Pour plus de détails, consultez [Editeur de requêtes](../Editeurs/2032063.md).

- ***l'éditeur de requêtes Utilisateur*** (appelé également le ***logiciel Etats & Requêtes***).
	Cet éditeur est livré uniquement avec WINDEV.
	Cette version est indépendante de WINDEV et peut être utilisée par un utilisateur final. 
	Cette version est intégrée au logiciel Etats & Requêtes. Pour plus de détails, consultez [Logiciel Etats & Requêtes](../Presentation/3088003.md).
	Certaines fonctionnalités de l'éditeur de requêtes ne sont pas disponibles dans cette version. Pour connaître les limitations de l'éditeur de requêtes Utilisateur, consultez [Limitations de l'éditeur de requêtes Utilisateur](../Presentation/3088001.md).




Pour plus d'informations concernant l'éditeur de requêtes Utilisateur, contactez le service commercial de PC SOFT.

## Éditeur de requêtes Utilisateur (logiciel Etats & Requêtes)
<a name="editeur_requetes_utilisateur_logiciel_etats_requetes_ELTTEXTE000158"></a>
Avec le logiciel Etats & Requêtes, il est possible de créer et d'exécuter des requêtes depuis l'éditeur (sans avoir besoin de WINDEV).

Les requêtes créées pourront être exécutées depuis l'éditeur de requêtes ou depuis une application WINDEV (si cette application gère les requêtes créées avec le logiciel Etats & Requêtes, grâce à la fonction [HExécuteRequête](../WDLang4/3044080.md)).

Certaines fonctionnalités de l'éditeur de requêtes ne sont pas disponibles dans cette version. Pour connaître les limitations de l'éditeur de requêtes Utilisateur, consultez [Limitations de l'éditeur de requêtes Utilisateur](../Presentation/3088001.md).

L'icône suivante ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=picto_nondispo4.gif) permet d'indiquer les différences entre l'éditeur de requêtes Utilisateur et l'éditeur de requêtes intégré.

La clé électronique n'est pas nécessaire à l'utilisation de l'éditeur de requêtes Utilisateur.

Pour créer des nouvelles requêtes ou pour modifier des requêtes existantes avec le logiciel Etats & Requêtes, certaines manipulations sont nécessaires. Pour plus de détails, consultez [Diffuser le logiciel Etats & Requêtes](../Presentation/3088002.md).

## Migration des requêtes créées avec WDEtat Utilisateur
<a name="migration_des_requetes_creees_avec_wdetat_utilisateur_ELTTEXTE000182"></a>
Les requêtes créées avec WDEtat Utilisateur peuvent être ouvertes sous le logiciel Etats & Requêtes. A leur ouverture, ces requêtes sont automatiquement migrées en version 16.

Pour pouvoir utiliser ces requêtes, nous vous conseillons d'enregistrer ces requêtes avec le logiciel Etats & Requêtes (![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ico_Enregistrer.gif)).

**Remarque** : Les requêtes créées sous WDQBE ne peuvent pas être ouvertes sous le logiciel Etats & Requêtes. Il est nécessaire de recréer ces requêtes avec le logiciel Etats & Requêtes.

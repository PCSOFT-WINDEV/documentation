## Spécificités
<a name="specificites_ELTTEXTE000143"></a>


### Type de requêtes
<a name="type_requetes_ELTPARAGRAPHE000027"></a>Sous l'éditeur de requêtes Utilisateur, il n'est pas possible de créer :

- des requêtes d'insertion.

- des requêtes de modification.

- des requêtes de suppression.

- des requêtes en code SQL utilisant les ordres suivants : [INSERT](../Editeurs/2034001.md), [UPDATE](../Editeurs/2034001.md), [DELETE](../Editeurs/2034001.md)




**Rappel** : L'éditeur de requêtes Utilisateur ne permet pas de modifier la base de données du projet en cours.


<a name="NOTE2_2"></a>


### Source de données d'une requête
<a name="source_donnees_une_requete_ELTPARAGRAPHE000050"></a>Les fichiers et les rubriques pouvant être utilisés dans une requête sont définis sous l'éditeur d'analyses de WINDEV. Pour plus de détails sur la préparation nécessaire à l'utilisation d'une application WINDEV avec le logiciel Etats & Requêtes, consultez [Diffuser le logiciel Etats & Requêtes](../Presentation/3088002.md).


<a name="NOTE2_3"></a>


### Volets ancrables
<a name="volets_ancrables_ELTPARAGRAPHE000061"></a>Sous l'éditeur de requêtes Utilisateur, les volets ancrables permettent d'accéder uniquement :

- aux principales fonctionnalités de l'éditeur de requêtes Utilisateur.

- aux éléments accessibles (états, requêtes, ...) de votre projet.




Pour plus de détails sur les volets ancrables, consultez [Volets ancrables](../Editeurs/2027001.md).

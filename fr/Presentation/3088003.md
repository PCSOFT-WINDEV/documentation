## Présentation
<a name="presentation_ELTTEXTE000083"></a>
Le logiciel Etats & Requêtes permet de :

- créer, de manipuler et/ou d'exécuter des états. Pour plus de détails, consultez [Éditeur d'états Utilisateur](../Presentation/3088004.md).

- créer, de manipuler et/ou d'exécuter des requêtes. Pour plus de détails, consultez [Éditeur de requêtes Utilisateur](../Presentation/3088006.md).




**Remarque** : Le logiciel Etats & Requêtes remplace WDEtat Utilisateur.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Conditions d'utilisation
<a name="conditions_utilisation_ELTTEXTE000107"></a>
Le logiciel Etats & Requêtes est un module indépendant de WINDEV. Ce module s'adresse aux utilisateurs des applications WINDEV. Grâce à ce module, les utilisateurs peuvent créer leurs propres états et requêtes :

- soit à partir d'une application WINDEV existante.

- soit indépendamment de toute application WINDEV.

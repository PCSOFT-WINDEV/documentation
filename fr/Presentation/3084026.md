## Partage d'éléments entre WEBDEV, WEBDEV et WINDEV Mobile
<a name="partage_elements_entre_webdev_webdev_windev_mobile_ELTTEXTE000094"></a>
Un site WEBDEV peut partager de nombreux éléments avec une application WINDEV ou WINDEV Mobile.

Les éléments partageables sont les suivants :


|   |   |
| --- | --- |
| Analyse | L'analyse d'un projet WINDEV ou WINDEV Mobile peut être partagé avec un projet WEBDEV. |
| Classes | Les classes d'un projet WINDEV ou WINDEV Mobile peuvent être partagées avec un projet WEBDEV. <br>Attention : seul le code compatible avec WEBDEV doit être utilisé dans la classe partagée. |
| Collections de procédures | Les collections de procédures d'un projet WINDEV ou WINDEV Mobile peuvent être partagées avec un projet WEBDEV. <br>Attention : seul le code compatible avec WEBDEV doit être utilisé dans la collection de procédures partagée. |
| Etats | <br><br>- Les états d'un projet WINDEV peuvent être partagés avec un projet WEBDEV.<br><br>- Les états d'un projet WINDEV ne peuvent pas être partagés avec un projet WINDEV.<br><br><br> |
| Requêtes | <br><br>- Les requêtes d'un projet WINDEV ou WINDEV Mobile peuvent être partagées avec un projet WEBDEV.<br><br>- Les requêtes d'un projet WINDEV ou WINDEV Mobile peuvent être chargées dans un projet WEBDEV. <br>	Ces requêtes sont en lecture seule. Il est possible de les exécuter, et le code SQL associé est visualisable. Il n'est pas possible de visualiser le graphe de la requête.<br><br><br> |

**Remarque** : Les fenêtres d'une application WINDEV peuvent être directement importées dans un projet WEBDEV. Pour plus de détails, consultez [Importer une fenêtre WINDEV](../Editeurs/2030034.md).

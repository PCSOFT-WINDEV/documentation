
## Les différents Serveurs d'application WEBDEV
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000163"></a>
WEBDEV propose plusieurs serveurs d'application WEBDEV : 

- **Le Serveur d'application WEBDEV complet** (version déploiement) : Ce Serveur d'Application WEBDEV permet de profiter de toutes les fonctionnalités WEBDEV : 

	- Connexions illimitées pour les sites et Webservices WEBDEV, 

	- Applications spécifiques (Serveur de réplication, Serveur de télémétrie, modules BaaS, Webservices d'accès aux bases de données tierces).   

	- Gestion des Clusters WEBDEV.  

	- Administrateurs WEB : SaaS, HFSQL, Serveur d'application WEBDEV.




- **Le serveur d'application WEBDEV 10 connexions** : Ce Serveur d'application WEBDEV permet uniquement de gérer 10 connexions simultanées aux sites et Webservices WEBDEV. 

- **Le serveur Réservé** : Ce serveur d'application WEBDEV ne gère pas les sites et les Webservices WEBDEV. Seules les applications spécifiques sont disponibles (Serveur de réplication, Serveur de télémétrie, modules BaaS, Webservices d'accès aux bases de données tierces). 




Remarques : 

- Il est possible de combiner le Serveur d'application WEBDEV 10 connexions et le Serveur Réservé. 

- A partir de la version 28, l'installation du Serveur d'application WEBDEV complet et du Serveur d'application WEBDEV 10 Connexions se font via le même programme d'installation.




## Logiciel Etats & Requêtes : Fonctions WLangage non supportées
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000256"></a>
Certaines fonctions WLangage ne sont pas disponibles sous le logiciel Etats & Requêtes. Ces fonctions sont les suivantes :


|   |   |
| --- | --- |
| [HAjoute](../WDLang4/3044147.md) | Ajoute l'enregistrement présent en mémoire dans le fichier de données (la requête ou la vue). |
| [HAjouteGroupe](../WDLang4/3044280.md) | Ajoute un groupe d'utilisateurs (connexion HFSQL Client/Serveur) |
| [HAjouteLiaison](../WDLang4/3044312.md) | Ajoute une règle d'intégrité entre deux fichiers sur le serveur. |
| [HAjouteUtilisateur](../WDLang4/3044281.md) | Ajoute un utilisateur (connexion HFSQL Client/Serveur) |
| [HArrêteServeur](../WDLang4/3044317.md) | Arrête un serveur HFSQL. Pour réaliser cette action, l'utilisateur doit posséder les droits d'actions sur le serveur. |
| [HChangeRepJNL](../WDLang4/3044164.md) | Modifie l'emplacement des fichiers du journal correspondant à un fichier HFSQL. (fichier \*JNL et fichiers JournalOpération et JournalIdentification s'ils existent). |
| [HCopieFichier](../WDLang4/3044282.md) | Effectue une copie d'un fichier HFSQL du Client vers le serveur et inversement |
| [HCréation](../WDLang4/3044255.md) | Crée à vide un fichier de données (fichier d'extension ".FIC") avec le fichier d'index et le fichier mémo si nécessaire. |
| [HCréationSiInexistant](../WDLang4/3044254.md) | La fonction **HCréationSiInexistant** effectue les opérations suivantes :<br><br>- Si le fichier n'existe pas, crée à vide un fichier de données (fichier d'extension ".FIC") avec le fichier d'index et le fichier mémo si nécessaire. <br>	La fonction est équivalente à [HCréation](../WDLang4/3044255.md).<br><br>- Si le fichier existe, ouvre le fichier. <br>	La fonction est alors équivalente à [HOuvre](../WDLang4/3044120.md).  <br>	Pour optimiser l'appel à la fonction **HCréationSiInexistant**, l'ouverture du fichier peut être également réalisée uniquement au premier accès réalisé sur le fichier.<br><br><br> |
| [HDBCréation](../WDLang4/3044171.md) | Termine la description dynamique de la structure du fichier. Le fichier qui vient d'être décrit est créé sur le disque dans le chemin indiqué dans [HDBDécritFichier](../WDLang4/3044172.md). Les fichiers de données, mémos et d'index sont créés sur le disque. |
| [HEcrit](../WDLang4/3044092.md) | Écrit un enregistrement dans le fichier de données sans mettre à jour les index correspondants à toutes les clés utilisées dans le fichier. |
| [HImporteHF55](../WDLang4/3044055.md) | Importe un fichier Hyper File 5.5 dans un fichier au format HFSQL Classic. |
| [HImporteTexte](../WDLang4/3044011.md) | Importe un fichier Texte dans un fichier au format HFSQL Classic. |
| [HImporteXML](../WDLang4/3044007.md) | Importe un fichier XML dans un fichier au format HFSQL Classic. |
| [HLibère](../WDLang4/3044064.md) | Transforme les enregistrements rayés (HRaye) d'un fichier en enregistrements supprimés. La place occupée par les enregistrements rayés est ainsi libérée, et pourra être réutilisée lors des prochains ajout (HAjoute). |
| [HMigreCléComposéeReliée](../WDLang4/3044038.md) | Migre les valeurs des clés composées reliées provenant d'un fichier au format Hyper File 5.5 vers le format HFSQL Classic. |
| [HModifie](../WDLang4/3044042.md) | Modifie l'enregistrement spécifié ou l'enregistrement présent en mémoire dans le fichier de données (la requête ou la vue). |
| [HModifieDroitBaseDeDonnées](../WDLang4/3044326.md) | Modifie les droits donnés à un utilisateur ou à un groupe pour une base de données HFSQL Client/Serveur. |
| [HModifieDroitFichier](../WDLang4/3044327.md) | Modifie les droits accordés sur un fichier de données HFSQL Client/Serveur pour un utilisateur ou un groupe. |
| [HModifieDroitServeur](../WDLang4/3044328.md) | Modifie les droits accordés sur un serveur HFSQL pour un utilisateur ou un groupe. |
| [HModifieGroupe](../WDLang4/3044308.md) | Modifie les informations du groupe spécifié en fonction des informations de la structure. |
| [HModifiePropriétéBaseDeDonnées](../WDLang4/3044329.md) | Modifie les propriétés d'une base de données située sur un serveur HFSQL. |
| [HModifiePropriétéFichier](../WDLang4/3044330.md) | Modifie les propriétés d'un fichier HFSQL situé sur un serveur. |
| [HModifiePropriétéServeur](../WDLang4/3044331.md) | Modifie les propriétés d'un serveur HyperFileSQL. |
| [HModifieUtilisateur](../WDLang4/3044309.md) | Modifie les informations de l'utilisateur spécifié en fonction des informations de la structure. |
| [HRaye](../WDLang4/3044121.md) | Raye un enregistrement d'un fichier. L'enregistrement est supprimé logiquement et pourra éventuellement être récupéré. |
| [HRegénèreFichier](../WDLang4/3044122.md) | Régénère un fichier à partir de son journal. |
| [HSupprime](../WDLang4/3044018.md) | Supprime un enregistrement d'un fichier (d'une requête ou d'une vue). |
| [HSupprimeBaseDeDonnées](../WDLang4/3044332.md) | Supprime une base de données présente sur un serveur HFSQL. Toutes les connexions utilisant la base de données supprimée sont fermées. |
| [HSupprimeFichier](../WDLang4/3044294.md) | Supprime les fichiers HFSQL (fichier .fic, .ndx et .mmo s'ils existent) sur le serveur. |
| [HSupprimeGroupe](../WDLang4/3044295.md) | Supprime un groupe d'utilisateurs associé à une connexion, sur le serveur. |
| [HSupprimeLiaison](../WDLang4/3044314.md) | Supprime une règle d'intégrité entre deux fichiers sur le serveur. |
| [HSupprimeUtilisateur](../WDLang4/3044296.md) | Supprime un utilisateur associé à une connexion sur le serveur. |
| [HVueVersFichier](../WDLang4/3044021.md) | Enregistre les modifications réalisées dans une vue (grâce aux fonctions [HModifie](../WDLang4/3044042.md), [HSupprime](../WDLang4/3044018.md) ou [HRaye](../WDLang4/3044121.md)) dans le fichier correspondant. |

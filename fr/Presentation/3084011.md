## Lancer un outil redistribuable en ligne de commande
<a name="lancer_outil_redistribuable_ligne_commande_ELTTEXTE000430"></a>
Pour lancer un outil redistribuable en ligne de commande, il suffit d'exécuter le code suivant :


```wl
sChemin est une chaîne
sChemin = """" + ComplèteRep(SysRep(srProgramFilesCommun)) + ...
	"PC SOFT\<Version de l'outil>\<Nom de l'outil>\<Nom de l'outil>.exe"""
LanceAppli(sChemin)
```


Où :

- &lt;Version de l'outil&gt; : numéro de version de l'outil utilisé (par exemple, "28.0" pour un outil livré avec WINDEV 28).

- &lt;Nom de l'outil&gt; : nom de l'outil à lancer (par exemple, "WDJournal").

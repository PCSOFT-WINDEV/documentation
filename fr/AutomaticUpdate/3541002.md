<a name="NOTE1_24"></a>
<a name="NOTE1_1_24"></a>


## Présentation
<a name="presentation_ELTTEXTE000119"></a>
Automatic Update est une application permettant de mettre à jour automatiquement ou manuellement les différents produits PC SOFT que vous possédez.

Il est possible de configurer :

- Le lancement de l'application. 

- Le répertoire de téléchargement des mises à jour.

- La bande passante utilisée.

<a name="NOTE2_1_24"></a>

## Options de configuration
<a name="options_configuration_ELTTEXTE000167"></a>


### Général
<a name="general_ELTPARAGRAPHE000039"></a>L'option "Lancer au démarrage de Windows" permet de lancer automatiquement l'application "Automatic Update" au lancement de Windows. Cette option permet de s'assurer qu'une recherche automatique est réalisée à chaque démarrage. 

Remarque : Une recherche automatique des mises à jour est effectuée toutes les 24 heures. 
<a name="NOTE3_2_24"></a>


### Téléchargement
<a name="telechargement_ELTPARAGRAPHE000048"></a>Deux modes sont disponibles pour le téléchargement des mises à jour. Il est possible de : 

- Télécharger automatiquement les mises à jour : dans ce cas, vous pouvez lancer directement l'installation de la mise à jour pour le produit voulu depuis "Automatic Update" (bouton "Mise à jour"). 

- Me demander pour télécharger : dans ce cas, un message informe qu'une mise à jour est disponible et propose le téléchargement. Vous pouvez lancer le téléchargement de la mise à jour également pour le produit voulu (via le bouton "Télécharger" présent sur la ligne du produit). Une notification est également affichée en haut de l'application. Si plusieurs mises à jour sont disponibles, un lien "Tout télécharger" est affiché. 




Il est également possible de : 

- spécifier un répertoire de téléchargement des mises à jour. Par défaut, le répertoire de téléchargement des mises à jour des produits PC SOFT est le suivant : "C:\\ProgramData\\PC SOFT\\AutomaticUpdate\\Packages". Pour changer ce répertoire, il suffit de spécifier un autre chemin. 
	Remarque : Lors de l'installation d'une mise à jour, le fichier téléchargé n'est pas automatiquement effacé. 

- indiquer une limite pour l'espace de téléchargement. 

- spécifier une durée limite de conservation des téléchargements. 

- autoriser le téléchargement des mises à jour de produit en niveau 1 de validation. Dans ce cas, les mises à jour avec uniquement une première phase de validation seront également proposées. Pour plus de détails sur les différentes phases de validation, consultez [Validation d'une version](https://pcsoft.fr/st/telec/validation-vi.htm). 



<a name="NOTE3_3_24"></a>


### Bande passante
<a name="bande_passante_ELTPARAGRAPHE000070"></a>Il est possible de limiter la bande passante des téléchargements en arrière-plan comme au premier plan.

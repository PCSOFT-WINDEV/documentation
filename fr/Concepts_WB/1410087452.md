<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Au-delà des sites Web, WEBDEV vous permet également de générer de nombreux autres types de projets.


|   |   |
| --- | --- |
| <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=P3_WB%20Mode%20de%20generation%20-%20HC%20N%B0002%201.gif)<br> | **Sites**<br><br>Les sites sont le mode de génération le plus fréquemment utilisé. WEBDEV permet de générer :<br><br>- des sites statiques ou dynamiques (Session ou AWP).<br><br>- des sites PHP.<br><br><br> |
| <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=P3_WB%20Mode%20de%20generation%20-%20HC%20N%B0001%202.gif)<br> | **Bibliothèques**<br><br>Une bibliothèque est un fichier unique rassemblant plusieurs éléments d'un projet WEBDEV : des pages, des états, etc. Il est possible de générer des bibliothèques autonomes pouvant être utilisées par d'autres sites. |
| <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=P3_WB%20Mode%20de%20generation%20-%20HC%20N%B0001%201.gif)<br> | **Composants externes**<br><br>Les composants externes sont des briques applicatives permettant de partager une ou plusieurs fonctionnalités spécifiques entre différentes applications. Un composant externe généré avec WINDEV peut également être utilisé dans un projet WEBDEV ou WINDEV Mobile. |
| <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=P3_WB%20Mode%20de%20generation%20-%20HC%20N%B0002.gif)<br> | **Webservices SOAP ou REST**<br><br>A partir d'un projet WINDEV ou WEBDEV, il est possible de générer un Webservice (également appelé service Web XML). Un Webservice expose un ensemble de fonctions (généralement une ou plusieurs collections de procédures) et les rend accessibles à travers le Web (ou un réseau privé) en utilisant les protocoles standard de communication HTTP, SOAP et REST.<br><br>Remarque : Un Webservice doit être déployé sur un Serveur d'Application WEBDEV pour être utilisable. |

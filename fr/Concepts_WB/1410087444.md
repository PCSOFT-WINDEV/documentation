
## 5. Configuration de projet

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les configurations de projet permettent de créer à partir d'un même projet plusieurs "cibles" différentes.

Vous pouvez ainsi créer à partir d'un même projet par exemple :

- des sites qui ne contiennent pas les mêmes éléments, qui ont des noms différents, …

- différents composants,

- un Webservice.




A tout moment, vous pouvez travailler sur une configuration spécifique : les éléments n'appartenant pas à cette configuration apparaissent grisés sous l'éditeur de projet.
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=P3-configuration%20de%20projet.gif)
Il est possible de générer en une seule opération toutes les configurations d'un projet (ou uniquement certaines) grâce à la génération multiple.

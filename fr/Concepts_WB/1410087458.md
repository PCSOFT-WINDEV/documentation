<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Le RAD (Rapid Application Development) permet de créer des pages à partir :

- de l'analyse liée au projet,

- des patterns RAD standard ou personnalisés,

- des gabarits.




**En génération RAD**, les pages générées contiennent tout le code nécessaire à leur fonctionnement. Ces pages peuvent être testées immédiatement, avec les données présentes sur le poste de développement.

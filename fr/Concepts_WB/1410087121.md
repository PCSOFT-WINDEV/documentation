
## 10. Modèles de pages

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
WEBDEV permet de créer des modèles de pages. Un modèle est destiné à contenir des éléments communs à un ensemble de pages du site.

Les modifications effectuées dans un modèle de pages sont automatiquement reportées sur toutes les pages utilisant ce modèle.

Un modèle de pages permet, par exemple, de respecter la charte graphique définie pour un site.

Pour créer une page utilisant un modèle, sélectionnez le modèle à utiliser lors de la création de la page.

**Remarque** : La programmation associée aux éléments du modèle peut être directement réalisée dans le modèle.

Les caractéristiques des éléments peuvent être désolidarisées du modèle. Par exemple, désolidariser la position d'un champ du modèle pour positionner le champ ailleurs tout en conservant les autres évolutions sur le champ (code, style, ...). On parle alors d**'héritage**. Dans ce cas, les éléments sont identifiés par un carré bleu.


## 22. Modèles d'états

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Dans une société, bien souvent, les impressions utilisent un look et une mise en page unifiés : date en haut à droite dans un format spécifique, bas de page avec heure d'impression et nom du fichier, logo en haut à gauche, ...

Les modèles d'états permettent d'uniformiser simplement la mise en page de vos états.

Un modèle d'états peut être créé :

- directement grâce à l'icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=P1_ICO_Cr%E9er_cpt.gif)
 disponible parmi les boutons d'accès rapide. Dans la fenêtre de création d'un nouvel élément, cliquez sur "Etat" puis cliquez sur "Modèle d'états". 

- à partir de l'état en cours (sous le volet "Accueil", dans le groupe "Général", déroulez "Enregistrer" et sélectionnez "Enregistrer comme un modèle").


Pour créer un état utilisant un modèle, sélectionnez le modèle à utiliser lors de la création de l'état.

Les caractéristiques des éléments peuvent être désolidarisées du modèle. Par exemple, désolidariser la position d'un champ du modèle pour positionner le champ ailleurs tout en conservant les autres évolutions sur le champ (code, style, ...). On parle alors d**'héritage**. Dans ce cas, les éléments sont identifiés par un carré bleu.

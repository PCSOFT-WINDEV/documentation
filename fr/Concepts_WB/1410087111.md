## L'internaute
<a name="internaute_ELTTEXTE000143"></a>



|   |
| --- |
| <br><br>- une machine : PC, Mac, Unix, Smartphone, etc.<br><br><br> |
| <br><br>- un navigateur : Microsoft Edge, FireFox, Chrome, etc.<br><br><br> |
| <br><br>- un accès Internet (ou Intranet).<br><br><br> |


Aucun module ne sera à télécharger. Le fonctionnement est immédiat et rapide.




| Le serveur | La machine de développement |
| --- | --- |
| <br><br>- **1 machine serveur** : PC<br><br><br> | <br><br>- **1 machine** : PC<br><br><br> |
| <br><br>- **1 système d'exploitation serveur** (Windows ou Linux dans une version en cours de support par l'éditeur)<br><br><br> | <br><br>- **1 système d'exploitation** :<br>	Windows Vista ou supérieur.<br><br><br> |
| <br><br>- **1 logiciel serveur Web** :<br>	IIS, Apache, etc.<br><br><br> | <br><br>- **1 logiciel serveur Web** :<br>	IIS, Apache, etc.<br><br><br> |
| <br><br>- **1 Serveur d'Application WEBDEV**<br><br><br>ou<br><br>- **1 Moteur PHP**<br>	(version 5.3 minimum)<br><br><br> |   |
|   | <br><br>- **1 logiciel WEBDEV "Développement"**<br><br><br> |
| <br><br>- **le site lui-même**<br><br><br> | <br><br>- **le site en cours de développement**<br><br><br> |
| <br><br>- **les données** \*<br>	(site dynamique uniquement)<br><br><br> | <br><br>- **les données** (optionnel) \*<br><br><br> |
|   | <br><br>- **au moins un navigateur** :<br>	Microsoft Edge, Firefox, Chrome, etc.<br><br><br> |


 \* Les données peuvent se trouver sur une autre machine reliée en réseau.

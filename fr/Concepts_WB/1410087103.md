
## 1. Qu'est-ce qu'un site Web ?
			

| [Sommaire](../Concepts_WB/1410087102.md) | [Page suivante](../Concepts_WB/1410087104.md) |
| --- | --- |



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Un site Web est un ensemble de pages HTML (HyperText Markup Language) stockées sur un serveur Web. Ces pages HTML sont organisées dans un but précis (par exemple, présenter une entreprise, vendre des produits, ...). 

Un site est destiné à être utilisé par des internautes. Les internautes utilisent un simple navigateur pour accéder au site.

WEBDEV permet de créer facilement des sites Web qui gèrent ou non des données.

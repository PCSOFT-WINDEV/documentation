
## 6. Génération multiple

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les configurations de projets permettent de définir simplement les différentes "Cibles" de votre projet. Vous pouvez ainsi définir pour un même projet plusieurs sites, plusieurs bibliothèques ou plusieurs composants.

Pour générer le résultat de chaque configuration, vous pouvez bien entendu sélectionner une à une chaque configuration et générer l'élément correspondant.

Un autre moyen plus rapide existe : **la génération multiple**. Vous sélectionnez en une seule opération les configurations à générer et le résultat est immédiat.
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=P3_WB%20-%20Generation%20multiple%20-%20HC%20N%B0001.gif)
Pour lancer une génération multiple, sous le volet "Projet", dans le groupe "Génération", déroulez "Générer la configuration" et sélectionnez "Génération multiple".

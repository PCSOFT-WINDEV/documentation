
## 2. Site statique ou dynamique

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Plusieurs types de sites peuvent être réalisés :

- des sites statiques,

- des sites dynamiques.




Le tableau ci-dessous présente les principales différences entre ces types de sites :

| Site statique | Site dynamique |
| --- | --- |
| Le contenu des pages du site est fixe, déterminé une fois pour toutes.<br><br>Un site statique ne peut pas interagir avec des données. | Les données affichées dans les pages varient. En général, les pages permettent de :<br><br>- réaliser des traitements et/ou des calculs par programmation.<br><br>- afficher des données stockées dans une base de données.<br><br>- afficher des images et du texte interactif.<br><br><br><br><br>WEBDEV propose plusieurs types de sites dynamiques :<br><br>- Site WEBDEV en mode Session.<br><br>- Site WEBDEV en mode AWP.<br><br>- Site PHP.<br><br><br> |
| **Site statique WEBDEV** : le Serveur d'Application WEBDEV n'est pas nécessaire. | **Site dynamique WEBDEV** : le Serveur d'Application WEBDEV ou le langage PHP est nécessaire. |


**Remarque** : Un site dynamique peut contenir une partie statique (présentation de l'entreprise, ...).

WEBDEV permet de créer :

- des pages statiques,

- des pages dynamiques.

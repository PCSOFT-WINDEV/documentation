
## 5. HFSQL Client/Serveur

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
HFSQL Client/Serveur est un puissant système de gestion de bases de données relationnelles (SGBD/R) en mode Client/Serveur.

Les caractéristiques du mode Client/Serveur sont les suivantes :

- Un site HFSQL Client/Serveur est exécuté sur différents postes utilisateur. Pour chaque site lancé, une session est présente sur le serveur Web.

- Les fichiers de données sont présents sur un poste serveur (Serveur HFSQL). Seul le poste serveur accède physiquement aux fichiers de données.

- L'ensemble des traitements (requête, lecture/ajout dans un fichier de données, ...) est réalisé sur le serveur.

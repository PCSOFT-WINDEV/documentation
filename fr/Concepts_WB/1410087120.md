
## 9. Page interne

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Le champ Page interne permet d'inclure une page (et son code) dans une autre page. A l'exécution, la page interne sera dynamiquement fusionnée à la page hôte.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## 1. Création d'une page interne
<a name="1_creation_une_page_interne_ELTTEXTE000136"></a>
Une page interne est une page spécifique qui ne comporte ni barre de titre, ni menu.

Dans cette page, tous les types de champs peuvent être utilisés.
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=P2_Page%20interne%20-%20HC%20N%B0001.gif&type=thumb)
Dans l'exemple, une page interne permet de gérer le panier.

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## 2. Utilisation d'une page interne
<a name="2_utilisation_une_page_interne_ELTTEXTE000160"></a>
Pour utiliser une page interne, il suffit de :

- créer un champ de type page interne.

- sélectionner dans la description du champ la page interne à utiliser et valider.
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=P2_Page%20interne%20-%20HC%20N%B0002.gif&type=thumb)



**Remarques** :

- Il n'est pas possible de modifier par programmation la page interne utilisée dans le champ "Page interne".

- La zone d'accueil est rectangulaire et aucune surcharge n'est possible. Pour réaliser des surcharges, il est conseillé d'utiliser des modèles de champs.

Légende :


|   |   |   |
| --- | --- | --- |
| **.WWH : Format source des pages**<br>Ce fichier contient la description complète de la page (champ, code navigateur, code serveur). Ce fichier est utilisé par l'éditeur et reste sur la machine du développeur.<br><br> |   | **.WDL : Bibliothèque du projet**<br>Générée lors du déploiement du site WEBDEV (en mode Session ou AWP), la bibliothèque contient le code serveur des pages du projet. Ce fichier sera présent sur le serveur.<br><br> |
| **.HTML générique**<br>Ce fichier contient la description complète de la page (champ, code navigateur, code serveur). Certains paramètres de cette page sont génériques et seront remplis dynamiquement lors de l'affichage de la page.<br><br> |   | **.AWP : Page AWP**<br>Généré à partir du fichier WWH, ce fichier contient la description des champs, le code serveur, le code navigateur. Ce fichier sera présent sur le serveur.<br><br> |
| **.HTM : Page HTML du site**<br>Généré à partir du fichier WWH, ce fichier contient la description des champs en HTML et le code navigateur (si le code a été saisi en WLangage, il est automatiquement traduit en Javascript). Ce fichier sera présent sur le serveur.<br><br> |   | **.PHP : Page PHP**<br>Généré à partir du fichier WWH, ce fichier contient la description des champs, le code serveur, le code navigateur au format PHP. Ce fichier sera présent sur le serveur.<br><br> |

## Présentation
<a name="presentation_ELTTEXTE000232"></a>


### L'utilitaire WDJournal
<a name="utilitaire_wdjournal_ELTPARAGRAPHE000011"></a>Les différentes opérations pouvant être réalisées avec WDJournal sont les suivantes :

- [Sauvegarde des fichiers de données HFSQL liés à une application WINDEV, WEBDEV ou WINDEV Mobile](../WDJournal/3516006.md)
	Permet de créer une sauvegarde des fichiers de données. En cas de problème, ces fichiers de données pourront être restaurés.

- [Restauration de fichiers de données précédemment sauvegardés](../WDJournal/3516009.md)
	Permet de restaurer une sauvegarde de fichiers de données HFSQL précédemment effectuée avec WDJournal.

- [Mise à jour de fichiers de données grâce à un fichier journal](../WDJournal/3516001.md)
	Permet d'effectuer toutes les opérations présentes dans un journal sur un fichier de données.

- [Visualisation du journal](../WDJournal/3516007.md)
	Permet de visualiser les informations enregistrées dans le fichier Journal. Il est également possible d'effectuer des recherches dans le journal.




**Remarque** : Lors d'une modification de l'analyse, le fichier journal sera automatiquement sauvegardé au moment de la modification automatique des fichiers de données. Le fichier journal sera ensuite purgé.

## Utilisation
<a name="utilisation_ELTTEXTE000262"></a>


### Lancement de WDJournal
<a name="lancement_wdjournal_ELTPARAGRAPHE000041"></a>WDJournal peut être lancé :

- En mode interactif : 

	- depuis WINDEV, WEBDEV ou WINDEV Mobile : sous le volet "Outils", dans le groupe "Utilitaires", cliquez sur "WDOutil" et sélectionnez "WDJournal". 

	- en lançant directement le programme "WDJOURNAL.EXE" présent dans le sous-répertoire "Programmes" du répertoire d'installation de WINDEV, WEBDEV et WINDEV Mobile.




- Depuis une ligne de commande.




En mode interactif, les différentes options du menu déroulant permettent d'accéder à toutes les fonctionnalités de WDJournal.<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=wdjournal.gif)

### Conditions d'utilisation
<a name="conditions_utilisation_ELTPARAGRAPHE000069"></a>WDJournal est un outil redistribuable. WDJournal peut être installé avec les applications développées avec WINDEV et/ou WEBDEV. La licence d'utilisation de WINDEV, WEBDEV ou WINDEV Mobile s'applique intégralement.

Lors de la création de la procédure d'installation, il est possible d'intégrer WDJournal à vos applications.

**Rappel** : La procédure d'installation peut être créée : 

- soit à partir de l'assistant. Pour lancer l'assistant, sous le volet "Projet", dans le groupe "Génération", déroulez "Procédure d'installation" et sélectionnez "Créer la procédure d'installation". 

- soit à partir de l'éditeur d'installation. Pour lancer l'éditeur d'installation, sous le volet "Outils", dans le groupe "Utilitaires", cliquez sur "WDInst".

## Rappel : Journal d'un fichier de données
<a name="rappel_journal_fichier_donnees_ELTTEXTE000298"></a>
WINDEV, WEBDEV et WINDEV Mobile offrent la possibilité de créer un fichier journal pour chaque fichier de données. Ce fichier journal peut être :

- Un fichier des écritures seulement

- Un fichier des lectures et des écritures




Le journal contient l'historique de l'utilisation des fichiers journalés :

- Les valeurs des rubriques de l'enregistrement avant modification du fichier de données,

- Les valeurs des rubriques de l'enregistrement après modification du fichier de données,

- L'auteur de l'opération,

- La date de l'opération,

- Le poste ayant réalisé l'opération,

- Un commentaire libre,

- L'adresse IP du poste ayant réalisé l'opération,

- Le nom de l'application,

- Le nom réseau de l'utilisateur,

- La nature de l'opération (ajout, modification, suppression, lecture, ...).




La journalisation d'un fichier de données est réalisée sous l'éditeur d'analyses de WINDEV, WEBDEV ou WINDEV Mobile.

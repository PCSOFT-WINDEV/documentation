
## WDJournal : Restaurer un fichier de données à partir de son journal
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000096"></a>
WDJournal permet d'appliquer à une sauvegarde d'un fichier de données les différentes opérations enregistrées dans le journal de ce fichier.

Il est ainsi possible, à partir d'une sauvegarde d'un fichier de données de réappliquer toutes les modifications effectuées sur ce fichier (et enregistrées dans le journal) jusqu'à une date donnée.

Cette re-génération d'un fichier de données ne peut être réalisée que en mode interactif. Cette opération nécessite :

- Le fichier journal associé au fichier de données (\*JNL.FIC).

- Le mot de passe du fichier journal (si ce mot de passe existe).

- Le fichier de données sur lequel les opérations du journal doivent être ré-effectuées. Si ce fichier n'existe pas, il est possible de re-créer le fichier à partir des informations du journal.






<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Mode Interactif
<a name="mode_interactif_ELTTEXTE000126"></a>


### Pour re-générer des fichiers de données HFSQL
<a name="pour_regenerer_des_fichiers_donnees_hfsql_ELTPARAGRAPHE000028"></a>

1. Lancez WDJournal.

2. Sélectionnez l'option "Fichier .. Gestion des journaux" (ou cliquez sur le livre).

3. Cliquez sur le bouton "Restaurer un fichier à partir de son journal".

4. Indiquez :

	- Le fichier journal associé au fichier de données. 
			Le fichier de données sera mis à jour grâce à ce fichier journal. Ce fichier doit être un fichier de type &lt;NomFichier&gt;JNL.FIC.

	- Le mot de passe associé au fichier journal (s'il existe).

	- Le fichier de données sur lequel le fichier journal doit être appliqué. 
			Ce fichier doit être le dernier fichier de données correct existant. Si ce fichier n'existe pas, WDJournal peut tenter de re-créer le fichier de données à partir du journal.

	- Le répertoire dans lequel le fichier de données sera créé. 
			Ce répertoire doit être précisé et doit être différent du répertoire contenant le fichier de sauvegarde.

	- La date et l'heure limite. 
			Date et heure jusqu'auxquelles les opérations présentes dans le fichier journal seront re-jouées sur le fichier de sauvegarde. Le fichier de données sera mis à jour. Si aucune date et heure ne sont précisées, toutes les opérations présentes dans le fichier journal seront effectuées.




5. Cliquez sur le bouton "Restaurer" pour effectuer la restauration du fichier.

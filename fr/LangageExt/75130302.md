
## Projet (Langage externe)

***En anglais : Project***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ouvre et exécute un projet WINDEV. La première fenêtre du projet est affichée et la saisie de cette fenêtre est automatiquement activée.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```txt
// En C
APPELWD("Projet,Compta");
```


<a name="Exemple2"></a>



```txt
// En Pascal
APPELWD('Projet,Compta');
```


<a name="Exemple3"></a>



```txt
// En VB
call APPELWD("Projet,Compta")
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`Projet(<Nom du projet>)`
---

**`<Nom du projet> : (Chaîne de caractères)`**

Nom du projet à lancer. Lors de l'exécution du projet, la feuille de styles de la bibliothèque du projet est chargée.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Important: la fonction **Projet** doit être appelée après la fonction [Bibli,Disque](../LangageExt/7513024.md) et avant la fonction HInit.

- Si une première fenêtre a été sélectionnée dans la liste des projets, cette fenêtre sera ouverte par la fonction **Projet**. Pour n'avoir aucune fenêtre ouverte, il faut sélectionner &lt;Aucune&gt; comme première fenêtre dans la liste des fenêtres du projet.

- Un programme en C, en Pascal peut par exemple contenir uniquement la fonction de lancement du projet.

- Si une erreur est survenue lors de l'ouverture, WdTouche contient "\*E\*".

- Si une fenêtre fait appel à des variables ou procédures globales, il faut exécuter le projet par la fonction Projet avant d'ouvrir la fenêtre, sinon les variables ou procédures globales ne seront pas reconnues, ni la feuille de styles.

- Plusieurs projets peuvent être appelés dans un même programme. Pour cela, il faut fermer le projet en cours avant d'en ouvrir un autre. Pour fermer le projet en cours, il faut utiliser la syntaxe suivante :

	- Exemple en C : 
			
		```txt
		HInit(&stTabCtx[0],NBCTX);
		APPELWD("BIBLI,DISQUE,MABIBLI");
		APPELWD("PROJET,MonProjet");
		```


	- Exemple en Pascal : 
			
		```txt
		HInit;
		APPELWD('PROJET, BIBLI,DISQUE,MABIBLI ');
		APPELWD('PROJET, MonProjet ');
		```


	- Exemple en VB : 
			
		```txt
		call HInit
		call APPELWD("PROJET, BIBLI,DISQUE,MABIBLI ")
		call APPELWD("PROJET, MonProjet ")
		```

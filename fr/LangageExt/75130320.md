
## Table,Supprime (Langage externe)

***En anglais : Table,Delete***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime un élément dans un champ Table.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```txt
// En C
APPELWD("Table,Supprime,TableProd,3");
```


<a name="Exemple2"></a>



```txt
// En Pascal
APPELWD('Table,Supprime,TableProd,3');
```


<a name="Exemple3"></a>



```txt
// En VB
call APPELWD("Table,Supprime,TableProd,3")
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`Table,Supprime(<Champ Table> [, <Indice>])`
---

**`<Champ Table> : (Nom de champ)`**

Nom du champ Table à manipuler.

**`<Indice> : (Chaîne de caractères (optionnelle))`**

Indice de l'élément à supprimer. Les éléments suivants sont remontés d'un rang.

Si l'indice n'est pas spécifié, l'élément sélectionné est supprimé.

Dans le cas d'un champ Table par programmation, l'indice est relatif à la totalité du champ Table.

Dans le cas d'un champ Table fichier, l'indice est relatif aux seuls éléments visualisés.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Dans le cas d'un champ Table fichier, l'enregistrement est directement supprimé du fichier.

- Dans le cas de suppressions successives, il est conseillé d'effectuer les suppressions en partant du dernier élément à supprimer. En effet, les éléments situés après chaque suppression sont déplacés vers le haut.

- **Table,Supprime**,&lt;Champ Table&gt;,\* supprime tous les éléments d'un champ Table par programmation.

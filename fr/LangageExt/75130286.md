
## Liste,Modifie (Langage externe)

***En anglais : List,Modify***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie un élément dans une liste déroulante ou dans une liste déroulée.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```txt
// En C
APPELWD("Liste,Modifie,ListeProd,%s=%s",indice,valeur);
```


<a name="Exemple2"></a>



```txt
// En Pascal
APPELWD('Liste,Modifie,ListeProd,'+indice+'='+valeur);
```


<a name="Exemple3"></a>





```txt
// En VB
call APPELWD("Liste,Modifie,ListeProd,"+indice+"="+valeur)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`Liste,Modifie(<Nom de la liste> , <Indice> [, <Valeur>])`
---

**`<Nom de la liste> : (Chaîne de caractères)`**

Nom de la liste à modifier.

**`<Indice> : (Chaîne de caractères)`**

Indice de l'élément à modifier.

**`<Valeur> : (Chaîne de caractères (optionnelle))`**

Nouvelle valeur de l'élément.

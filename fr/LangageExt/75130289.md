
## Liste,Recupere (Langage externe)

***En anglais : List,Get***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Récupère la valeur d'un élément donné ou de l'élément choisi en cours dans une liste déroulante ou dans une liste déroulée.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```txt
// En C
APPELWD("Liste,Recupere,ListeProd,3");
strcpy(Valeur,WDChaine);
```


<a name="Exemple2"></a>



```txt
// En Pascal
APPELWD('Liste,Recupere,ListeProd,3');
Valeur:=WdChaine;
```


<a name="Exemple3"></a>



```txt
// En VB
call APPELWD("Liste,Recupere,ListeProd,3")
Valeur=WdChaine
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`Liste,Recupere(<Nom de la liste> [, <Indice>])`
---

**`<Nom de la liste> : (Chaîne de caractères)`**

Nom de la liste à manipuler.

**`<Indice> : (Chaîne de caractères  (optionnelle))`**

Indice de l'élément qui doit être récupéré.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Si &lt;Indice&gt; n'est pas précisé, la valeur de l'élément sélectionné est contenue dans WdChaine. L'indice de l'élément, s'il existe, est contenu dans WdEntier.
	Il est conseillé d'utiliser [Liste,Select,?](../LangageExt/75130290.md) pour récupérer l'indice de l'élément sélectionné.

- Si aucun élément n'est sélectionné dans la liste ou si la liste est vide, **Liste,Recupere** retourne -1 dans WdEntier.

- Si &lt;Indice&gt; est précisé, **Liste,Recupere** retourne dans WdChaine la valeur de l'élément dont l'indice est spécifié et dans WdEntier la taille de l'élément.

- Pour les combos avec saisie, **Liste,Recupere** renvoie toujours WdChaine qui contient la valeur du champ de saisie de la combo.

- WdEntier n'a pas de signification.

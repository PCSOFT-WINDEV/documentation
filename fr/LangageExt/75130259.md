
## Ecran,Courant (Langage externe)

***En anglais : Screen,Current***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Récupère le nom du champ en cours de saisie et éventuellement son indice.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```txt
// En C
APPELWD("Ecran,Courant");
strcpy(ChampCourant,WDNom);
```


<a name="Exemple2"></a>



```txt
// En Pascal
APPELWD('Ecran,Courant');
ChampCourant:=WdNom;
```


<a name="Exemple3"></a>



```txt
// En VB
call APPELWD("Ecran,Courant")
ChampCourant=WdNom
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`Ecran,Courant()`
---



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- WdNom contient le nom du champ. Si le champ est un champ indicé, WdIndice contient l'indice du champ.

- Par défaut, les espaces situés en fin de WdNom sont supprimés. La fonction Espace,Oui doit être utilisée afin que les espaces ne soient pas supprimés dans WdNom (sur 10 caractères).

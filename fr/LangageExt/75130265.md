
## Ecran,Saisie (Langage externe)

***En anglais : Screen,Input***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Lance la saisie des champs d'une fenêtre.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```txt
// En C
while (strcmp(Touche,"ESC")!=0) {
	APPELWD("Ecran,Saisie");
	strcpy(Touche,WDTouche);
```


<a name="Exemple2"></a>





```txt
// En Pascal
While Touche<>'ESC' DO
Begin
	APPELWD('Ecran,Saisie');
	Touche:=WdTouche;
	...
End;
```


<a name="Exemple3"></a>



```txt
// En VB
While Touche<>"ESC"
	call APPELWD("Ecran,Saisie")
	Touche=WdTouche
	...
WEnd
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`Ecran,Saisie(</CR> , </NONBLOQUANT>)`
---

**`</CR> : ()`**

Permet de détecter si une erreur est intervenue. Si ce paramètre est précisé, en cas d'erreur WdTouche contient "\*E\*".

**`</NONBLOQUANT> : ()`**

Permet de quitter immédiatement la saisie sans attendre forcément de résultat. Si ce paramètre est précisé : "\*N\*" est retourné dans WdTouche si aucune action n'a été effectuée.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- En sortie de saisie, la variable WdTouche contient le compte-rendu de la saisie modifié en WLangage par RenvoieWdTouche ou WdTouche=&lt;Code Touche&gt;.

- En sortie de saisie d'un menu, la variable WdTouche contient "\*M\*" et la suite des lettres d'appel des options sélectionnées est dans WdChaine.

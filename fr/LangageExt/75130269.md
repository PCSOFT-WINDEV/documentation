
## Erreur (Langage externe)

***En anglais : Error***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Affiche une boîte de dialogue contenant un sigle d'erreur et le message passé en paramètre.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```txt
// En C
APPELWD("Erreur,Erreur de lecture du fichier %s",NomFic);
```


<a name="Exemple2"></a>



```txt
// En Pascal
APPELWD('Erreur,Erreur de lecture du fichier '+NomFic);
```


<a name="Exemple3"></a>



```txt
// En VB
call APPELWD("Erreur,Erreur de lecture du fichier "+NomFic)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`Erreur(<Message>)`
---

**`<Message> : (Chaîne de caractères)`**

Message à afficher (200 caractères maximum).



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- La fenêtre sera fermée lorsque l'utilisateur valide le bouton "OK".

- Par défaut, la fenêtre a le même titre que la fenêtre en cours. Pour modifier le titre, il faut utiliser la fonction [Windows,TSuivant](../LangageExt/75130326.md).

- Pour afficher le message sur plusieurs lignes, il faut utiliser la syntaxe suivante :

	- Exemple en C :
			
		```txt
		APPELWD("Erreur,  \n ");
		```


	- Exemple en Pascal :
			
		```txt
		APPELWD('Erreur, '+Chr(10)+ ' ');
		```


	- Exemple en VB :
			
		```txt
		call APPELWD("Erreur, "+CHR(10)+"")
		```


## wWDGetTypeValeurRetour 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Utilisation
<a name="utilisation_ELTTEXTE000119"></a>
Type de la valeur de retour de la dernière instruction [nWDExecute](../LangageExt/7513015.md), [nWDExecuteEx](../LangageExt/7513020.md) ou [nWDEvalue](../LangageExt/7513017.md).



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Syntaxe
<a name="syntaxe_ELTTEXTE000143"></a>

```txt
Void      wWDGetTypeValeurRetour(void);
```




<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Notes
<a name="notes_ELTTEXTE000167"></a>


### Valeur renvoyée
<a name="valeur_renvoyee_ELTPARAGRAPHE000036"></a>

Cette fonction renvoie une des constantes suivantes :


|   |   |
| --- | --- |
| **Nom** | **Valeur** |
| WDTYPE_VOID | 0 |
| WDTYPE_BOOL | 1 |
| WDTYPE_INT | 2 |
| WDTYPE_INT64 | 3 |
| WDTYPE_DOUBLE | 4 |
| WDTYPE_TSTR | 5 |
| WDTYPE_BIN | 6 |

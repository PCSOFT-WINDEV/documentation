
## Recupere (Langage externe)

***En anglais : Get***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Récupère la valeur d'un champ.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```txt
// En C
APPELWD("Recupere,Nom");
```


<a name="Exemple2"></a>



```txt
// En Pascal
APPELWD('Recupere,Nom');
```


<a name="Exemple3"></a>





```txt
// En VB
call APPELWD("Recupere,Nom")
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`Recupere(<Nom du champ> [, <Indice>])`
---

**`<Nom du champ> : (Chaîne de caractères)`**

 Nom du champ à récupérer.

**`<Indice> : (Chaîne de caractères (optionnelle))`**

Indice dans le cas d'un champ indicé.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Recupere est équivalent à [Ecran,Recupere](../LangageExt/75130264.md).

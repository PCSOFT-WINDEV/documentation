
## Ferme (Langage externe)

***En anglais : Close***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ferme la fenêtre en cours et restitue le contexte existant avant l'ouverture de la fenêtre.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```txt
// En C
APPELWD("Ferme");
```


<a name="Exemple2"></a>





```txt
// En Pascal
APPELWD('Ferme');
```


<a name="Exemple3"></a>





```txt
// En VB
call APPELWD("Ferme")
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`Ferme()`

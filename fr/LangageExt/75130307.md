
## Style (Langage externe)

***En anglais : Style***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Charge une feuille de styles.

**Remarque** : Un projet est rattaché à soit la feuille de styles standard, soit une feuille de styles propre au projet. Lors du chargement du projet, la feuille de styles qui lui est rattachée est chargée. **Style** permet de charger une feuille de styles différente de celle par défaut.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```txt
// En C
APPELWD("STYLE,PROJET1.WDS");
```


<a name="Exemple2"></a>



```txt
// En Pascal
APPELWD('STYLE,PROJET1.WDS');
```


<a name="Exemple3"></a>





```txt
// En VB
call APPELWD("STYLE,PROJET1.WDS")
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`Style(<Nom de la feuille>)`
---

**`<Nom de la feuille> : (Chaîne de caractères)`**

Nom de la nouvelle feuille de styles à utiliser. Une feuille de styles a pour nom le nom du projet qui l'a créé avec l'extension WDS. La feuille de styles standard a pour nom WD.STY.

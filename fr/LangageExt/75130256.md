
## Confirmer (Langage externe)

***En anglais : Confirm***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Affiche une boîte de dialogue comportant un sigle, un message, les boutons "Oui", "Non" et "Annuler" et retourne le bouton validé par l'utilisateur.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```txt
// En C
APPELWD("Confirmer, Sauvegarde des informations saisies");
if WDEntier==1 Sauve();
```


<a name="Exemple2"></a>



```txt
// En Pascal
APPELWD('Confirmer, Sauvegarde des informations saisies');
if WdEntier=1 THEN Sauve();
```


<a name="Exemple3"></a>





```txt
// En VB
call APPELWD("Confirmer, Sauvegarde des informations saisies")
if WdEntier=1 THEN Call Sauve
EndIf
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`Confirmer(<Message>)`
---

**`<Message> : (Chaîne de caractères)`**

Message à afficher (200 caractères maximum).



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- WdChaine contient la réponse de l'utilisateur:

	- si la réponse est "Oui", WdChaine contient "O" et WdEntier contient 1.

	- si la réponse est "Non", WdChaine contient "N" et WdEntier contient 0.

	- si la réponse est "Annuler", WdChaine contient "A" et WdEntier contient 2.







- Pour afficher le message sur plusieurs lignes, il faut utiliser la syntaxe suivante :

	- Exemple en C :
			
		```txt
		APPELWD("Confirmer,  \n ");
		```


	- Exemple en Pascal :
			
		```txt
		APPELWD('Confirmer, '+Chr(10)+' ');
		```


	- Exemple en VB :
			
		```txt
		call APPELWD("Confirmer, "+CHR(10)+"")
		```

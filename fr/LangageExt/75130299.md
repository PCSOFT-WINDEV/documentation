
## Option,Version (Langage externe)

***En anglais : Option,Version***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Connaît le numéro de version de wdxxxwm.dll de WINDEV.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```txt
// En C
APPELWD("Option,Version");
strcpy(Version,WDChaine);
```


<a name="Exemple2"></a>



```txt
// En Pascal
APPELWD('Option,Version');
Version:=WdChaine;
```


<a name="Exemple3"></a>



```txt
// En VB
call APPELWD("Option,Version")
Version=WdChaine
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`Option,Version()`
---



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Le numéro de version est contenu dans WdChaine sous la forme 9.99X (par exemple 1.05A).

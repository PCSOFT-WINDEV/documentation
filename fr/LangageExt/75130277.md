
## HsMenu,Marquage (Langage externe)

***En anglais : HsMenu,Mark***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Sait si une option de menu est marquée ou non.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```txt
// En C
// L'option correspondant au chemin LC est-elle marquée ?
APPELWD("HsMenu,Marquage,LC");
if WDEntier==1 APPELWD("INFO, L'option est marquée");
// L'option numéro 15 est-elle marquée ?
APPELWD("HsMenu,Marquage,15");
if WDEntier==0 APPELWD("INFO, L'option n'est pas marquée");
```


<a name="Exemple2"></a>





```txt
// En Pascal
(* L'option correspondant au chemin LC est-elle marquée ? *)
APPELWD('HsMenu,Marquage,LC');
if WDEntier=1 then APPELWD('INFO, L'option est marquée');
(* L'option numéro 15 est-elle marquée ? *)
APPELWD('HsMenu,Marquage,15');
if WDEntier=0 then APPELWD('INFO, L'option n'est pas marquée');
```


<a name="Exemple3"></a>





```txt
// En VB
' L'option correspondant au chemin LC est-elle marquée ?
call APPELWD("HsMenu,Marquage,LC")
if WDEntier=1 then  
	call APPELWD("INFO, L'option est marquée")
end if
' L'option numéro 15 est-elle marquée ?
call APPELWD("HsMenu,Marquage,15")
if WDEntier=0 then  
	call APPELWD("INFO, L'option n'est pas marquée")
end if
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`HsMenu,Marquage(<Option>)`
---

**`<Option> : (Chaîne de caractères)`**

Ce paramètre peut être :

- un numéro qui correspond à l'indice de l'option,

- une chaîne de caractères qui correspond au chemin d'accès à l'option (lettres d'appels)  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
**HsMenu,Marquage** retourne dans WdEntier l'état de marquage de l'option :

- 0 si l'option n'est pas marquée

- 1 si l'option est marquée

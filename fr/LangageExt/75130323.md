
## WDTermine (Langage externe)

***En anglais : WDEnd***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Termine un programme qui utilise WINDEV.
Remarque : **WDTermine** est obligatoire dans tous les programmes WINDEV.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```txt
// En C
WDTermine();
```


<a name="Exemple2"></a>



```txt
// En Pascal
WDTermine;
```


<a name="Exemple3"></a>





```txt
// En VB
call WDTermine
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`WDTermine()`
---



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Cette fonction n'est pas appelée par APPELWD.

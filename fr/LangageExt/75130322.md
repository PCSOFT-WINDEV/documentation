
## Utilise (Langage externe)

***En anglais : Use***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ouvre la fenêtre spécifiée en fermant toutes les fenêtres précédemment ouvertes.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```txt
// En C
APPELWD("Utilise,Princip.WDW");
```


<a name="Exemple2"></a>



```txt
// En Pascal
APPELWD('Utilise,Princip.WDW');
```


<a name="Exemple3"></a>



```txt
// En VB
call APPELWD("Utilise,Princip.WDW")
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`Utilise(<Nom de la fenêtre> [, <X> [, <Y> [, </CR>]]])`
---

**`<Nom de la fenêtre> : (Chaîne de caractères)`**

 Nom de la fenêtre à ouvrir. Ce paramètre comprend le chemin, le nom et l'extension de la fenêtre (l'extension des fenêtres de WINDEV est par défaut ".WDW").

**`<X> : (Entier (optionnel))`**

 Abscisse (en pixels) d'affichage de la fenêtre (par rapport au coin haut gauche de l'écran ou de la fenêtre mère).

**`<Y> : (Entier (optionnel))`**

 Ordonnée (en pixels) d'affichage de la fenêtre (par rapport au coin haut gauche de l'écran).

**`</CR> : (Paramètre (optionnel))`**

Permet de détecter si une erreur est intervenue.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Si le paramètre /CR est précisé, en cas d'erreur le compte-rendu "\*E\*" est retourné dans WdTouche.

- Pour modifier le titre de la fenêtre, il faut utiliser les fonctions Windows,TSuivant ou Windows,TCourant.

- Lors d'un test interactif, il ne faut pas appeler la fonction **Utilise**. Un message d'erreur s'affichera alors. Il n'est pas possible de passer des paramètres à la fenêtre par cette fonction.

- Si une fenêtre fait appel à des procédures globales, il faut exécuter le projet par la fonction Projet avant d'ouvrir la fenêtre, sinon les procédures globales ne seront pas reconnues, ni la feuille de styles.

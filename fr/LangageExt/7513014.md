
## nWDInitProjet : Code d'initialisation du projet
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Utilisation
<a name="utilisation_ELTTEXTE000074"></a>
Exécution du code d'initialisation du projet.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Syntaxe
<a name="syntaxe_ELTTEXTE000098"></a>

```wl
Int      nWDInitProjet(LPCTSTR pszNomProjet)
```


**Détail des paramètres**

**pszNomProjet :** *Chaîne de caractères*

Nom du projet dont le code d'initialisation doit être exécuté. Attention : un seul code d'initialisation de projet peut être exécuté dans une application.

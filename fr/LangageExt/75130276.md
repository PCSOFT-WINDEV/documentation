
## HsMenu,Libelle (Fonction)

***En anglais : HsMenu,Label***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie le libellé d'une option de menu.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```txt
// En C
APPELWD("HSMENU,LIBELLE,%s,%s",Option,NouvOption);
```


<a name="Exemple2"></a>





```txt
// En Pascal
APPELWD('HSMENU,LIBELLE,'+Option+','+NouvOption);
```


<a name="Exemple3"></a>



```txt
// En VB
call APPELWD("HSMENU,LIBELLE,"+Option+","+NouvOption)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`HsMenu,Libelle(<Identifiant de l'option> , <Nouveau libellé>)`
---

**`<Identifiant de l'option> : (Chaîne de caractères)`**

Numéro de l'option à modifier. Le numéro de l'option est celui affecté lors de la description du menu. Il est disponible dans l'éditeur de menus et dans le dossier du projet.

**`<Nouveau libellé> : (Chaîne de caractères)`**

Nouveau libellé de l'option. Ce paramètre doit contenir le caractère "&" (précédant la lettre d'appel de l'option).

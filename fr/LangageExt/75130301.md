
## Ouvre (Langage externe)

***En anglais : Open***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ouvre une fenêtre.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```txt
// En C
APPELWD("Ouvre,FEN_Fenetre1.WDW");
```


<a name="Exemple2"></a>



```txt
// En Pascal
APPELWD('Ouvre,FEN_Fenetre1.WDW');
```


<a name="Exemple3"></a>



```txt
// En VB
call APPELWD("Ouvre,FEN_Fenetre1.WDW")
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`Ouvre(<Nom de la fenêtre> [, <"/CR"> [, <Y> [, <X> [, <"+TAB+"Paramètre 1"+TAB+"Paramètre 2"+...>]]]])`
---

**`<Nom de la fenêtre> : (Chaîne de caractères)`**

 Nom de la fenêtre à ouvrir. Ce paramètre comprend le chemin, le nom et l'extension de la fenêtre (l'extension des fenêtres de WINDEV est par défaut ".WDW").

**`<"/CR"> : (Paramètre optionnel)`**

Permet de détecter une erreur.

**`<Y> : (Entier (optionnel))`**

 Ordonnée (en pixels) d'affichage de la fenêtre (par rapport au coin haut gauche de l'écran ou de la fenêtre mère).

**`<X> : (Entier (optionnel))`**

 Abscisse (en pixels) d'affichage de la fenêtre (par rapport au coin haut gauche de l'écran).

**`<"+TAB+"Paramètre 1"+TAB+"Paramètre 2"+...> : (Paramètres optionnels)`**

 Paramètres à passer à la fenêtre lors de son ouverture.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Si le paramètre /CR est précisé, en cas d'erreur le compte-rendu "\*E\*" est retourné dans WdTouche.

- La fonction **Ouvre** doit être préférée à la fonction Utilise dans le cas où le contexte avant l'ouverture de la fenêtre doit être restitué.

- La fenêtre ouverte se superpose à la fenêtre précédente, si elle existe.

- Les champs de la fenêtre ouverte sont actifs. Les champs de la fenêtre précédente, si elle existe sont inactifs.

- Pour modifier le titre de la fenêtre, il faut utiliser les fonctions Ecran,Suivant ou Ecran,Courant.

- Si une erreur est survenue lors de l'ouverture, WdTouche contient "\*E\*".

- Si une fenêtre fait appel à des procédures globales, il faut exécuter le projet par la fonction Projet avant d'ouvrir la fenêtre, sinon les procédures globales ne seront pas reconnues, ni la feuille de styles.

- Une fenêtre peut être ouverte même si aucun projet n'est en cours. La fenêtre ne doit être alors rattachée à aucun projet.

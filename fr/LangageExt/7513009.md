
## xxWDGetValeurRetour 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Utilisation
<a name="utilisation_ELTTEXTE000131"></a>
Permet de récupérer la valeur de retour suite à l'appel d'une instruction [nWDEvalue](../LangageExt/7513017.md), [nWDExecute](../LangageExt/7513015.md) ou [nWDExecuteEx](../LangageExt/7513020.md).

Chacune des fonctions présentées ci-dessous renvoie un type différent. Pour connaître le type qui va être retourné par le code WLangage :

- soit consultez la documentation de la fonction WLangage,

- soit utilisez la commande [wWDGetTypeValeurRetour](../LangageExt/7513008.md)






<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Syntaxe
<a name="syntaxe_ELTTEXTE000155"></a>


BOOL      bWDGetValeurRetour(void);




Valeur de retour en cas d'erreur : Si la valeur n'est pas de type booléen, la fonction renvoie False.
<a name="NOTE2_2"></a>


Int       nWDGetValeurRetour(void);




Valeur de retour en cas d'erreur : Si la valeur n'est pas de type Entier, la fonction renvoie 0.


<a name="NOTE2_3"></a>


WDINT64   i64WDGetValeurRetour(void);




Valeur de retour en cas d'erreur : Si la valeur n'est pas de type entier sur 8 octets, la fonction renvoie 0.


<a name="NOTE2_4"></a>


Double    dWDGetValeurRetour(void);




Valeur de retour en cas d'erreur : Si la valeur n'est pas de type flottant, la fonction renvoie 0.0.


<a name="NOTE2_5"></a>


Double dWDGetValeurRetour(void);




Valeur de retour en cas d'erreur : Si la valeur n'est pas de type flottant, la fonction renvoie 0.0.



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Notes
<a name="notes_ELTTEXTE000203"></a>


### Valeur renvoyée
<a name="valeur_renvoyee_ELTPARAGRAPHE000081"></a>Si le type de la valeur de retour ne correspond pas à une des fonctions citées ci-dessus, une valeur par défaut sera renvoyée. Cette valeur par défaut est précisée pour chacune des fonctions (valeur de retour en cas d'erreur)

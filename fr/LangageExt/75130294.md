
## Message (Langage externe)

***En anglais : Message***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Affiche un message dans la barre des messages.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```txt
// En C
APPELWD("Message,Saisir les informations demandées");
```


<a name="Exemple2"></a>



```txt
// En Pascal
APPELWD('Message,Saisir les informations demandées');
```


<a name="Exemple3"></a>



```txt
// En VB
call APPELWD("Message,Saisir les informations demandées")
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`Message(<Contenu du message>)`
---

**`<Contenu du message> : (Chaîne de caractères)`**

Message à afficher.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Le message reste affiché jusqu'au prochain message affiché dans la barre.


## Table,Select,? (Langage externe)

***En anglais : Table,Select,?***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie l'indice de l'élément sélectionné dans un champ Table.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```txt
// En C
APPELWD("Table,Select,?,TableProd");
strcpy(indice,WDEntier);
```


<a name="Exemple2"></a>



```txt
// En Pascal
APPELWD('Table,Select,?,TableProd');
indice:=WdEntier;
```


<a name="Exemple3"></a>



```txt
// En VB
call APPELWD("Table,Select,?,TableProd")
indice=WdEntier
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`Table,Select,?(<Champ Table>)`
---

**`<Champ Table> : (Nom de champ)`**

Nom du champ Table à manipuler.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- WdEntier contient l'indice de l'élément sélectionné.

- Dans le cas d'un champ Table par programmation, l'indice est relatif à la totalité du champ Table.

- Dans le cas d'un champ Table fichier, l'indice est relatif aux seuls éléments visualisés.

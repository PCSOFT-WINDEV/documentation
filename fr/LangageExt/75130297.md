
## Option,Espace (Langage externe)

***En anglais : Option,Space***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Complète le contenu de WdChaine par des espaces à droite.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```txt
// En C
APPELWD("OPTION,ESPACE,OUI");
```


<a name="Exemple2"></a>



```txt
// En Pascal
APPELWD('OPTION,ESPACE,OUI');
```


<a name="Exemple3"></a>





```txt
// En VB
call APPELWD("OPTION,ESPACE,OUI")
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`Option,Espace(<Indicateur>)`
---

**`<Indicateur> : (Chaîne de caractères)`**

 Prend pour valeur :

- "Oui" pour compléter WdChaine par des espaces

- "Non" pour ne pas compléter WdChaine par des espaces.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- **Option,espace** est équivalente à Espace

- La fonction se comporte comme une bascule, après l'appel de Option,Espace,Oui WdChaine sera automatiquement complété par des espaces jusqu'à l'appel de Option,Espace,Non

- Si Option,Espace,Non est appelée, WdChaine ne sera pas complétée par des espaces.

- Par défaut, WdChaine ne sera pas complétée par des espaces (option "Non").

- Si Option,Espace,Oui est appelée, alors Ecran,Recupere retourne dans WdChaine le contenu du champ complété par des espaces (longueur de WdChaine = longueur du champ).
	Attention : WdChaine a une taille maximum de 200 caractères.

- Si Option,Espace,Oui est appelée, alors Ecran,Courant retourne dans WdChaine le contenu du champ complété par des espaces pour avoir 10 caractères

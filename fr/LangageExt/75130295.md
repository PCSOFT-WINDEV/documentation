
## Multitache (Langage externe)

***En anglais : Multitask***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet aux programmes en cours d'exécution de traiter les messages Windows ou permet d'exécuter une temporisation.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```txt
// En C
// Attente de 10 secondes
APPELWD("Multitache,1000");
```


<a name="Exemple2"></a>



```txt
// En Pascal
(* Attente de 10 secondes *)
APPELWD('Multitache,1000');
```


<a name="Exemple3"></a>



```txt
// En VB
' Attente de 10 secondes
call APPELWD("Multitache,1000")
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`Multitache([<Durée>])`
---

**`<Durée> : (Chaîne de caractères optionnelle)`**

 Durée de la temporisation exprimée en centièmes de seconde.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
**Multitache** peut être utilisée dans 3 contextes :

- dans une boucle, pour rendre 'la main' aux autres programmes,

- dans une boucle, pour afficher des informations,

- dans une boucle, lors de l'utilisation d'un timer.




Attention : **Multitache** supprime les événements souris et clavier de la fenêtre du programme, sans les traiter..

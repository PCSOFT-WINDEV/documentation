
## SonInit (Langage externe)

***En anglais : WaveInit***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Initialise l'interpréteur de sons. Cette fonction doit être exécutée une fois dans chaque programme. Il permet d'étalonner la vitesse d'exécution des sons. Cette fonction peut prendre quelques instants.
<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`SonInit([<Initialisation du son>])`
---

**`<Initialisation du son> : (Chaîne de caractères (optionnelle))`**

 Nom du fichier d'initialisation avec lequel il faut initialiser l'interpréteur de son.

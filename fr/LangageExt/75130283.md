
## Liste,Ajoute (Langage externe)

***En anglais : List,Add***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ajoute un élément dans une liste déroulante (champ Combo) ou dans une liste déroulée (champ Liste).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```txt
// En C
APPELWD("Liste,Ajoute,ListeProd=%s",CODE);
```


<a name="Exemple2"></a>



```txt
// En Pascal
APPELWD('Liste,Ajoute,ListeProd='+CODE);
```


<a name="Exemple3"></a>



```txt
// En VB
call APPELWD("Liste,Ajoute,ListeProd="+CODE)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`Liste,Ajoute(<Liste à manipuler> [, <Valeur>])`
---

**`<Liste à manipuler> : (Nom de champ)`**

Nom du champ Liste ou Combo à modifier.

**`<Valeur> : (Chaîne de caractères (optionnelle))`**

Valeur à ajouter.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Si le champ Liste/Combo n'est pas trié, l'élément est ajouté à la fin des éléments de la liste.

- Si le champ Liste/Combo est trié, la valeur est ajoutée à l'emplacement correspondant à l'ordre de tri.

- Dans le cas d'un champ Liste/Combo trié, **Liste,Ajoute** doit être préférée à **Liste,Insere**.

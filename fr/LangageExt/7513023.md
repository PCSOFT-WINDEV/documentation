
## Beep (Langage externe)

***En anglais : Beep***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet d'émettre un bip sonore sur le haut parleur de la machine.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```txt
// En C
APPELWD("Beep");
```


<a name="Exemple2"></a>



```txt
// En Pascal
APPELWD('Beep');
```


<a name="Exemple3"></a>





```txt
// En VB
Call APPELWD("Beep")
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`Beep()`

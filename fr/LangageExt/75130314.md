
## Table,Modifie (Langage externe)

***En anglais : Table,Modify***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie un élément dans un champ Table.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```txt
// En C
APPELWD("Table,Modifie,TableProd,%s=%s",indice,valeur);
```


<a name="Exemple2"></a>



```txt
// En Pascal
APPELWD('Table,Modifie,TableProd,'+indice+'='+valeur);
```


<a name="Exemple3"></a>





```txt
// En VB
call APPELWD("Table,Modifie,TableProd,"+indice+"="+valeur)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`Table,Modifie(<Champ Table> [, <Indice>] , <Valeur>)`
---

**`<Champ Table> : (Nom de champ)`**

Nom du champ Table à manipuler.

**`<Indice> : (Chaîne de caractères (optionnelle))`**

Indice de l'élément dans le champ Table. Si l'indice n'est pas précisé, il s'agit de l'élément en cours.

- Dans le cas d'un champ Table par programmation, l'indice est relatif à la totalité du champ Table. 

- Dans le cas d'un champ Table fichier, l'indice est relatif aux seuls éléments visualisés.




**`<Valeur> : (Chaîne de caractères)`**

Nouvelle valeur.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Note
<a name="note_ELTPARAGRAPHE000049"></a>Si le champ Table comporte plusieurs colonnes, il faut séparer chaque colonne par la valeur numérique de [TAB] (code binaire 9).

Exemple en C :

```txt
APPELWD("Table,Modifie,TabCli,14=DUPOND\tJean");
```


Exemple en Pascal :

```txt
APPELWD('Table,Ajoute,TabCli,14=DUPOND'+#9+'Jean');
```


Exemple en VB :

```txt
call APPELWD("Table,Modifie,TabCli,14=DUPOND"+CHR$(9)+"Jean")
```

<a name="NOTE0_2"></a>


### Champ Table fichier
<a name="champ_table_fichier_ELTPARAGRAPHE000074"></a>Dans le cas d'un champ Table fichier, l'enregistrement n'est pas modifié dans le fichier de données, il faut appeler la fonction [Table,Enregistre](../LangageExt/75130312.md).

Pour modifier le fichier de données associé à un champ Table, il ne faut pas utiliser la fonction **HModifie**, mais utiliser les fonctions **Table,Modifie** et [Table,Enregistre](../LangageExt/75130312.md).

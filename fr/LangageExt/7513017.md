
## nWDEvalue : Evaluation de la valeur d'un élément
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Utilisation
<a name="utilisation_ELTTEXTE000082"></a>
Evaluation de la valeur d'un élément : champ, variable, rubrique HyperFileSQL (par exemple : Fenêtre.champ, variable, fichier.rubrique, ...).

**Remarque** : les accès aux éléments indicés ne sont pas possibles (exemple : Table[1]).



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Syntaxe
<a name="syntaxe_ELTTEXTE000106"></a>

```wl
Int      nWDEvalue(LPCTSTR pszElement);
```


**Détail des paramètres**

**pszElement :**

Nom de l'élément à évaluer

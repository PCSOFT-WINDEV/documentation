
## HsMenu,Select (Langage externe)

***En anglais : HsMenu,Select***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Sélectionne ou désélectionne, rend invisible, marque ou démarque une option du menu en cours.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```txt
// En C
APPELWD("HsMenu,Select,+,FS");
```


<a name="Exemple2"></a>



```txt
// En Pascal
APPELWD('HsMenu,Select,+,FS');
```


<a name="Exemple3"></a>





```txt
// En VB
call APPELWD("HsMenu,Select,+,FS")
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`HsMenu,Select(<Code> [, <Option>])`
---

**`<Code> : (Chaîne de caractères)`**

Caractère qui peut prendre pour valeur :

- "+" pour sélectionner une option

- "-" pour désélectionner une option

- "I" pour rendre une option invisible

- "M" pour marquer une option

- "D" pour démarquer une option.




**`<Option> : (Chaîne de caractères (optionnelle))`**

Combinaison de lettres d'appel pour accéder à l'option ou numéro de l'option.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- L'accès à l'option s'effectue en fonction des lettres d'appel de l'option.

- Lors de la saisie du menu, les options désélectionnées ne pourront pas être validées, elles seront grisées.


## Ecran,Premier (Langage externe)

***En anglais : Screen,First***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Indique le nom du premier champ en saisie ou le nom du champ suivant devant être en saisie.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```txt
// En C
// Le prochain champ sera NOMCLI
APPELWD("Ecran,Premier,NOMCLI");
```


<a name="Exemple2"></a>





```txt
// En Pascal
(* Le prochain champ sera NOMCLI *
APPELWD('Ecran,Premier,NOMCLI');
```




<a name="Exemple3"></a>





```txt
// En VB
' Le prochain champ sera NOMCLI
call APPELWD("Ecran,Premier,NOMCLI")
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`Ecran,Premier(<Nom du champ> [, <Indice>])`
---

**`<Nom du champ> : (Chaîne de caractères)`**

Nom du champ dans lequel le texte doit être affiché.

**`<Indice> : (Chaîne de caractères (optionnelle))`**

Indice du champ.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Dans le cas d'une table, la ligne de la table à saisir doit être sélectionnée au préalable. 

Exemple :


```txt
Table,Select,+,<Nom de la table>,<Indice de la ligne>
Ecran,Premier,<Nom de la colonne>
```


## Handle (Langage externe)

***En anglais : Handle***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Retourne le handle d'une fenêtre ou d'un champ.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```txt
// En C
// Connaître le handle du champ IMAGE1
APPELWD("Handle,IMAGE1");
NumHand=WdEntier
```


<a name="Exemple2"></a>



```txt
// En Pascal
(* Connaître le handle du champ IMAGE1 *)
APPELWD('Handle,IMAGE1');
NumHand:=WdEntier
```


<a name="Exemple3"></a>



```txt
// En VB
' Connaître le handle du champ IMAGE1
call APPELWD("Handle,IMAGE1")
NumHand=WdEntier
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le handle d'une fenêtre

`Handle(<Nom de la fenêtre>)`
---

**`<Nom de la fenêtre> : (Chaîne de caractères)`**

 Nom de la fenêtre dont on veut connaître le handle.  


<a name="SYNTAXE2"></a>

### Connaître le handle d'un champ

`Handle(<Nom du champ> [, <Indice>])`
---

**`<Nom du champ> : (Chaîne de caractères)`**

 Nom du champ dont on veut connaître le handle.

**`<Indice> : (Chaîne de caractères (optionnelle))`**

 Indice du champ indicé dont on veut connaître le handle.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Le numéro de handle est retourné dans WdEntier et dans WdChaine.

- Dans le cas des tables, aucun indice ne peut être précisé.

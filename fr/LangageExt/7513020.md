
## nWDExecuteEx : Exécution d'une fonction du WLangage
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Utilisation
<a name="utilisation_ELTTEXTE000079"></a>
Exécution d'une fonction du WLangage par compilation dynamique sans formatage de commande.

**Remarque** : cette fonction se comporte comme la fonction [nWDExecute](../LangageExt/7513015.md) mais n'interprète pas la chaîne de caractères contenant la commande. Ceci permet une exécution plus rapide des commandes non formatées (ne contenant pas les caractères %s, %d, ...).



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Syntaxe
<a name="syntaxe_ELTTEXTE000103"></a>

```wl
Int      nWDExecuteEx(LPCTSTR pszCommande);
```


**Détail des paramètres**

**pszCommande** : 

Commande du WLangage à exécuter


## Fenetre (Langage externe)

***En anglais : Window***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le nombre de fenêtres ouvertes à un instant donné.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```txt
// En C
APPELWD("Fenetre");
strcpy(FenActive,WDChaine);
```


<a name="Exemple2"></a>



```txt
// En Pascal
APPELWD('Fenetre');
FenActive:=WdChaine;
```


<a name="Exemple3"></a>



```txt
// En VB
call APPELWD("Fenetre")
FenActive=WdChaine
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`Fenetre()`
---



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Le nombre de fenêtres est retourné dans WdEntier.

- Le nom de la fenêtre active est retourné dans WdChaine.

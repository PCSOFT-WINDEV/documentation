
## nWDExecute : Exécution d'une fonction du WLangage
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Utilisation
<a name="utilisation_ELTTEXTE000080"></a>
Exécution d'une fonction du WLangage par compilation dynamique avec formatage de commande. 



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Syntaxe
<a name="syntaxe_ELTTEXTE000104"></a>

```wl
Int      nWDExecute(LPCTSTR pszCommande, [Paramètres]);
```


**Détail des paramètres**

**pszCommande :**

Commande du WLangage à exécuter

**Paramètres :** Optionnel

Permet de passer des paramètres à la fonction du WLangage. La syntaxe à utiliser est identique à celle de la fonction C nommée printf. Exemple : 
nWDExecute("Info(\\"Le nom du client est : %s\\")",szNomClient)


## Liste,Supprime (Langage externe)

***En anglais : List,Delete***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime un élément dans une liste déroulante ou dans une liste déroulée. Les éléments suivants sont remontés d'un rang.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```txt
// En C
APPELWD("Liste,Supprime,ListeProd,3");
```
<a name="Exemple2"></a>



```txt
// En Pascal
APPELWD('Liste,Supprime,ListeProd,3');
```


<a name="Exemple3"></a>





```txt
// En VB
call APPELWD("Liste,Supprime,ListeProd,3")
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`Liste,Supprime(<Nom de la liste> , <Indice>)`
---

**`<Nom de la liste> : (Chaîne de caractères)`**

Nom de la liste à manipuler.

**`<Indice> : (Chaîne de caractères)`**

Indice de l'élément à supprimer.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Dans le cas de suppression en chaîne, il est conseillé d'effectuer les suppressions en partant du dernier élément à supprimer. En effet, les éléments situés après chaque suppression sont déplacés vers le haut.

- La syntaxe Liste,Supprime,&lt;Nom de la liste&gt;,\* supprime tous les éléments de la liste &lt;Nom de la liste&gt;.

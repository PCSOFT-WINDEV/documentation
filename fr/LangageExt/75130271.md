
## Execute (Langage externe)

***En anglais : Execute***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Exécute le code d'un champ ou d'une procédure en WLangage.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```txt
// En C
// Exécution de la procédure DESSIN de la fenêtre en cours
APPELWD("Execute,DESSIN.PRO");
// Exécution du code du bouton de validation
APPELWD("Execute,Valide.CLI");
```


<a name="Exemple2"></a>



```txt
// En Pascal
(* Exécution de la procédure DESSIN de la fenêtre en cours *)
APPELWD('Execute,DESSIN.PRO');
(* Exécution du code du bouton de validation *)
APPELWD('Execute,Valide.CLI');
```


<a name="Exemple3"></a>



```txt
// En VB
' Exécution de la procédure DESSIN de la fenêtre en cours
call APPELWD("Execute,DESSIN.PRO")
' Exécution du code du bouton de validation
call APPELWD("Execute,VALIDE.CLI")
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Exécuter le code d'un champ

`Execute(<Nom du champ> , <Type de l'événement>)`
---

**`<Nom du champ> : (Chaîne de caractères)`**

Nom du champ dont le code doit être exécuté.

**`<Type de l'événement> : (Chaîne de caractères)`**

Correspond au code du traitement en WLangage à exécuter :

- "INI" pour le code d'initialisation

- "ENT" pour le code d'entrée d'un champ

- "SOR" pour le code de sortie d'un champ

- "CLI" pour le code de clic sur un bouton

- "MOD" pour le code de modification d'un champ

- "LEN" pour le code d'entrée dans une ligne de table

- "LSO" pour le code de sortie d'une ligne de table

- "SUI" pour le code de traitement du suivant dans une table

- "PRE" pour le code de traitement du précédent dans une table

- "AFF" pour le code d'affichage d'une table.  





<a name="SYNTAXE2"></a>

### Exécuter le code d'une procédure

`Execute(<Nom de la procédure + ".PRO">)`
---

**`<Nom de la procédure + ".PRO"> : (Chaîne de caractères)`**

Nom de la procédure en WLangage à exécuter.

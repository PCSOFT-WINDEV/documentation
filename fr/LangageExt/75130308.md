
## Table,Affiche (Langage externe)

***En anglais : Table,Display***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Rafraîchit l'affichage du champ Table. Les champs calculés sont recalculés. Le champ Table est repositionné par rapport au fichier de données.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```txt
// En C
APPELWD("Table,Affiche,TabProduit,10");
```


<a name="Exemple2"></a>





```txt
// En Pascal
APPELWD('Table,Affiche,TabProduit,10');
```


<a name="Exemple3"></a>





```txt
// En VB
call APPELWD("Table,Affiche,TabProduit,10")
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`Table,Affiche(<Champ Table> , <Indice>)`
---

**`<Champ Table> : (Nom de champ)`**

Nom du champ Table à manipuler.

**`<Indice> : (Chaîne de caractères)`**

Indice de l'élément à réafficher ou caractère indiquant l'action à réaliser.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Affichage d'un champ Table par programmation
<a name="affichage_champ_table_par_programmation_ELTPARAGRAPHE000045"></a>

- Si &lt;Indice&gt; est précisé, **Table,Affiche** réaffiche la ligne dont l'indice est spécifié.

- Si &lt;Indice&gt;=P ou si &lt;Indice&gt;=T ou si &lt;Indice&gt;=\*, **Table,Affiche** réaffiche les éléments actuellement visualisés.

- **Table,Affiche** est utilisée pour recalculer les champs calculés.





<a name="NOTE0_2"></a>




### Affichage dans un champ Table fichier
<a name="affichage_dans_champ_table_fichier_ELTPARAGRAPHE000056"></a>

- Si &lt;Indice&gt; est précisé, **Table,Affiche** réaffiche la ligne dont l'indice est spécifié.

- Si &lt;Indice&gt;=P, **Table,Affiche** réaffiche les éléments actuellement visualisés dans le champ Table. Cela permet de réactualiser l'affichage (après l'écriture dans le fichier par Table,Enregistre par exemple)

- Si &lt;Indice&gt;=T, **Table,Affiche** réaffiche le champ Table à partir de l'élément en cours dans le fichier. Il est ainsi possible d'afficher le champ Table à partir d'une position donnée.




Par exemple pour afficher les enregistrements correspondant à une recherche.

1. La recherche est effectuée (HLitRecherche).

2. Le résultat est affiché avec Table,Affiche.

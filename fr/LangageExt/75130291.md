
## Liste,Select,+ (Langage externe)

***En anglais : List,Select,+***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Sélectionne un élément dans une liste déroulante ou dans une liste déroulée. Cette fonction est notamment utilisée pour manipuler les listes multisélection.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```txt
// En C
APPELWD("Liste,Select,+,ListeProd,10");
```


<a name="Exemple2"></a>



```txt
// En Pascal
APPELWD('Liste,Select,+,ListeProd,10');
```


<a name="Exemple3"></a>



```txt
// En VB
call APPELWD("Liste,Select,+,ListeProd,10")
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`Liste,Select,+(<Nom de la liste> , <Indice>)`
---

**`<Nom de la liste> : (Chaîne de caractères)`**

Nom de la liste à manipuler.

**`<Indice> : (Chaîne de caractères)`**

Indice de l'élément à sélectionner. Si l'indice passé en paramètre est supérieur au nombre d'éléments de la liste, **Liste,Select,+** n'a aucun effet.

Pour une liste à multisélection, si &lt;Indice&gt;=\*, tous les éléments de la liste sont sélectionnés.

Pour une liste monosélection, &lt;Indice&gt; ne doit pas être égal à "\*" sinon WdEntier contient -1 pour signaler une erreur.

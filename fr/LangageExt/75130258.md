
## Ecran,Coordonnee (Langage externe)

***En anglais : Screen,Coordinate***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Récupère les coordonnées du champ par rapport au bord haut gauche de la fenêtre ainsi que la taille physique d'un champ.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```txt
// En C
// Récupère la position et la taille de l'image
APPELWD("Ecran,Coordonnee,IMAGE");
strcpy(Coord,WDChaine);
```




<a name="Exemple2"></a>



```txt
// En Pascal
// Récupère la position et la taille de l'image
APPELWD('Ecran,Coordonnee,IMAGE');
Coord:=WdChaine;
```


<a name="Exemple3"></a>



```txt
// En VB
// Récupère la position et la taille de l'image
call APPELWD("Ecran,Coordonnee,IMAGE")
Coord=WdChaine
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`Ecran,Coordonnee(<Nom du champ> [, <Indice>])`
---

**`<Nom du champ> : (Chaîne de caractères)`**

Nom du champ dont les coordonnées doivent êtres récupérées.

**`<Indice> : (Chaîne de caractères (optionnelle))`**

Indice du champ.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Les coordonnées sont retournées dans WdChaine sous la forme : "&lt;ligne&gt;,&lt;colonne&gt;:&lt;hauteur&gt;,&lt;largeur&gt;"

Les paramètres &lt;ligne&gt;,&lt;colonne&gt;:&lt;hauteur&gt;,&lt;largeur&gt; sont exprimés en pixels. La taille du champ tient compte du cadre externe du champ.

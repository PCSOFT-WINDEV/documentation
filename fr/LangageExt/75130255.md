
## Bibli,Erreur (Langage externe)

***En anglais : Library,Error***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Charge le fichier qui contient les messages d'erreur gérés par WD???EXE.DLL dans la langue souhaitée. Par défaut, si aucune fonction **Bibli,Erreur** n'est appelée, les messages d'erreur sont en français.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```txt
// En C
APPELWD("BIBLI,ERREUR,ALLEMAND.WDM");
if WdEntier>0 {
	APPELWD("ERREUR, Fichier des messages non trouvé");
}
```


<a name="Exemple2"></a>



```txt
// En Pascal
APPELWD('BIBLI, ERREUR, ALLEMAND.WDM');
if WdEntier>0 then APPELWD('ERREUR,Fichier des messages non trouvé');
```


<a name="Exemple3"></a>



```txt
// En VB
call APPELWD("BIBLI, ERREUR, ALLEMAND.WDM")
if WdEntier>0 then
	call APPELWD("ERREUR,Fichier des messages non trouvé")
end if
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`Bibli,Erreur(<Nom du fichier>)`
---

**`<Nom du fichier> : (Chaîne de caractères)`**

Nom et chemin du fichier à charger. Ce fichier fourni avec WINDEV ou créé avec WDINT. Il a pour extension .WDM. WdEntier est supérieur à 0 lorsque &lt;Nom du fichier&gt; n'a pas été trouvé.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
**Bibli,Erreur** ne peut pas être utilisée pour charger un fichier qui n'a pas été créé avec WDINT.

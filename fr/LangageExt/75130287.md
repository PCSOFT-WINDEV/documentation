
## Liste,Occurrence (Langage externe)

***En anglais : List,Count***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le nombre d'éléments d'une liste.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```txt
// En C
APPELWD("Liste,Occurrence,ListeProd");
NbElem=WDEntier;
```


<a name="Exemple2"></a>



```txt
// En Pascal
APPELWD('Liste,Occurrence,ListeProd');
NbElem:=WdEntier;
```


<a name="Exemple3"></a>



```txt
// En VB
call APPELWD("Liste,Occurrence,ListeProd")
NbElem=WdEntier
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`Liste,Occurrence(<Nom de la liste>)`
---

**`<Nom de la liste> : (Chaîne de caractères)`**

Nom du champ Liste à manipuler.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
WdEntier contient le nombre d'éléments de la liste.

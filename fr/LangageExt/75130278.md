
## HsMenu,Nom (Langage externe)

***En anglais : HsMenu,Name***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ajoute un menu dans la fenêtre qui est ouverte. Le menu est celui d'une autre fenêtre.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```txt
// En C
APPELWD("HsMenu,Nom,Menu3");
```


<a name="Exemple2"></a>





```txt
// En Pascal
APPELWD('HsMenu,Nom,Menu3');
```


<a name="Exemple3"></a>



```txt
// En VB
call APPELWD("HsMenu,Nom,Menu3")
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`HsMenu,Nom(<Nom de la fenêtre menu>)`
---

**`<Nom de la fenêtre menu> : (Chaîne de caractères)`**

Nom de la fenêtre contenant le menu (sans l'extension).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- **HsMenu,Nom** permet d'utiliser dans une fenêtre, un menu qui a été créé dans une autre fenêtre.

- **HsMenu,Nom** ne peut être utilisée que pour les fenêtres pouvant comporter un menu (fenêtre avec un "Cadre fin").

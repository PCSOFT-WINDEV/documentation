
## Ecran,Suivant (Langage externe)

***En anglais : Screen,Next***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Retourne le nom du prochain champ à saisir.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```txt
// En C
APPELWD("Ecran,suivant");
strcpy(NomChamp,WdNom);
Indice=WDIndice;
```


<a name="Exemple2"></a>



```txt
// En Pascal
APPELWD('Ecran,Suivant');
NomChamp:=WdNom;
Indice:=WdIndice;
```


<a name="Exemple3"></a>



```txt
// En VB
call APPELWD("Ecran,Suivant")
NomChamp=WdNom
Indice=WdIndice
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`Ecran,Suivant()`
---



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Le nom du champ est retourné dans WdNom.

- WdIndice contient l'indice du champ dans le cas d'un champ indicé.


## Oui,Non (Langage externe)

***En anglais : YesNo***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Affiche une boîte de dialogue avec un sigle, un message et les boutons "oui" et "non". L'utilisateur doit valider un des deux boutons "Oui" ou "Non" pour fermer la fenêtre.

Si la réponse est "Oui", WdChaine contient "O" et WdEntier contient 1.

Si la réponse est "Non", WdChaine contient "N" et WdEntier contient 0.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```txt
// En C
APPELWD("OuiNon,Voulez-vous quitter le programme");
if WDEntier=1 Quitte_Programme();
```


<a name="Exemple2"></a>



```txt
// En Pascal
APPELWD('OuiNon,Voulez-vous quitter le programme');
if WdEntier=1 then Quitte_Programme;
```


<a name="Exemple3"></a>



```txt
// En VB
call APPELWD("OuiNon,Voulez-vous quitter le programme")
if WdEntier=1 then gosub Quitte_Programme
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`Oui,Non(<Message>)`
---

**`<Message> : (Chaîne de caractères)`**

Message à afficher (200 caractères maximum).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Pour afficher le message sur plusieurs lignes, il faut utiliser la syntaxe suivante:

Exemple en C :


```txt
APPELWD("OuiNon,  \n ");
```


Exemple en Pascal :


```txt
APPELWD('OuiNon, '+Chr(10)+' ');
```


Exemple en VB :


```txt
call APPELWD("OuiNon, "+CHR(10)+"")
```

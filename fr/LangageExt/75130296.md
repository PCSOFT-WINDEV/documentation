
## Nation (Langage externe)

***En anglais : Nation***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Définit la langue d'affichage des titres, libellés, messages... des fenêtres et des champs.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```txt
// En C
Francais=5
Espagnol=7;
//LANGUE est un sélecteur
APPELWD("ECRAN,RECUPERE, LANGUE");
If WdEntier==1 APPELWD("NATION,%d",Français);
If WdEntier==2 { //espagnol
	APPELWD("NATION,%d",Espagnol);
	APPELWD("BIBLI,ERREUR,ESPAGNOL.WDM");
If WdEntier>0 APPELWD("ERREUR,Fichier WDM non trouvé")
```


<a name="Exemple2"></a>



```txt
// En Pascal
Français:=5; Espagnol:=7;
{LANGUE est un sélecteur}
APPELWD('ECRAN,RECUPERE, LANGUE');
If WdEntier=1 THEN  {Français}
	APPELWD('NATION,'+ConvertChaine(Français));
End;
If WdEntier=2 THEN  {Espagnol}
Begin
	APPELWD('NATION,'+ ConvertChaine(Espagnol));
	APPELWD('BIBLI,ERREUR,ESPAGNOL.WDM');
	If WdEntier>0 THEN APPELWD('ERREUR,Fichier WDM non trouvé');
End;
```


<a name="Exemple3"></a>



```txt
// En VB
Français=5
Espagnol=7
'LANGUE est un sélecteur
CALL APPELWD("ECRAN,RECUPERE, LANGUE")
If WdEntier=1 THEN 
	CALL APPELWD("NATION,"+STR$(Français))
End If

If WdEntier=2 THEN
	CALL APPELWD("NATION,"+ STR$(Espagnol))
	CALL APPELWD("BIBLI,ERREUR,ESPAGNOL.WDM")
	If WdEntier>0 THEN
		CALL APPELWD("ERREUR,Fichier WDM non trouvé")
	End if
End if
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`Nation(<Numéro de la langue>)`
---

**`<Numéro de la langue> : (Entier)`**

 Numéro de la langue dans laquelle s'exécutera le projet. Cette valeur est définie sous l'éditeur lors du choix des langues.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Après l'appel de **Nation**, les chargements des fenêtres affichent les titres, libellés, messages... dans la langue demandée.

- **Nation** peut être utilisée pour développer des applications internationales pouvant être diffusées dans plusieurs langues


## Table,Select,+ (Langage externe)

***En anglais : Table,Select,+ ***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Sélectionne un élément dans un champ Table.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```txt
// En C
APPELWD("Table,Select,+,TableProd,10");
```


<a name="Exemple2"></a>



```txt
// En Pascal
APPELWD('Table,Select,+,TableProd,10');
```


<a name="Exemple3"></a>



```txt
// En VB
call APPELWD("Table,Select,+,TableProd,10")
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`Table,Select,+(<Champ Table> , <Indice>)`
---

**`<Champ Table> : (Nom de champ)`**

Nom du champ Table à manipuler.

**`<Indice> : (Chaîne de caractères)`**

Numéro de la ligne à sélectionner. Si ce paramètre est inférieur à 0, le champ Table fichier "remonte" pour visualiser l'enregistrement demandé. Si l'indice est supérieur au nombre de lignes affichées, le champ Table "descend" pour visualiser l'enregistrement demandé. Il est cependant conseillé d'utiliser [Table,Affiche](../LangageExt/75130308.md) précédé du positionnement dans le fichier.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Dans le cas d'un champ Table par programmation, l'indice est relatif à la totalité du champ Table.

- Dans le cas d'un champ Table fichier, l'indice est relatif aux seuls éléments visualisés.


## Liste,Insere (Langage externe)

***En anglais : List,Insert***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Insère un élément à l'indice donné d'une liste déroulante ou d'une liste déroulée.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```txt
// En C
APPELWD("Liste,Insere,ListeProd,2=%s",CODE);
```


<a name="Exemple2"></a>



```txt
// En Pascal
APPELWD("Liste,Insere,ListeProd,2="+CODE);
```


<a name="Exemple3"></a>



```txt
// En VB
call APPELWD("Liste,Insere,ListeProd,2="+CODE)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`Liste,Insere(<Nom de la liste> , <Indice> , <Valeur>)`
---

**`<Nom de la liste> : (Chaîne de caractères)`**

Nom du champ Liste à modifier.

**`<Indice> : (Chaîne de caractères)`**

Indice de l'élément dans la liste où l'élément doit être inséré.

**`<Valeur> : (Chaîne de caractères)`**

Valeur à insérer dans la liste.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Si la liste est triée, Liste,Insere positionne l'élément à l'emplacement courant sans trier la liste. Dans le cas de liste triée, Liste,Ajoute doit être préférée à Liste,Insere.

- Si l'indice spécifié est supérieur au nombre d'éléments de la liste, Liste,Insere est équivalent à Liste,Ajoute.

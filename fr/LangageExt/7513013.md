
## nWDOuvreWDL : Ouverture d'une bibliothèque
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Utilisation
<a name="utilisation_ELTTEXTE000074"></a>
Ouverture d'une bibliothèque créée par WINDEV (fichier d'extension WDL). Ce fichier contient par exemple les fenêtres, le code, l'analyse de l'application WINDEV.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Syntaxe
<a name="syntaxe_ELTTEXTE000098"></a>

```wl
Int      nWDOuvreWDL(LPCTSTR pszNomCompletWDL);
```


**Détail des paramètres**

**pszNomCompletWDL :** Chaîne de caractères

Nom et chemin complet de la bibliothèque WINDEV (.WDL) à ouvrir.

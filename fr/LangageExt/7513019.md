
## nWDGetHFContext : Récupération du contexte HyperFileSQL
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Utilisation
<a name="utilisation_ELTTEXTE000068"></a>
Récupération du contexte HyperFileSQL (en C, Pascal et Visual Basic uniquement). Cette fonction est utilisée lors de l'initialisation d'un projet utilisant une analyse HyperFileSQL.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Syntaxe
<a name="syntaxe_ELTTEXTE000092"></a>

```wl
Int      nWDGetHFContext(void** ppHFContext);
```

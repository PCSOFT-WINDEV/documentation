
## Table,Cherche (Langage externe)

***En anglais : Table,Search***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Recherche une valeur dans un champ Table par programmation. Il s'agit d'une recherche à l'identique.

**Remarque** : Cette fonction ne doit pas être utilisée pour un champ Table fichier.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```txt
// En C
APPELWD("Table,Cherche,Prod,000002");
```


<a name="Exemple2"></a>





```txt
// En Pascal
APPELWD('Table,Cherche,Prod,000002');
```


<a name="Exemple3"></a>





```txt
// En VB
call APPELWD("Table,Cherche,Prod,000002")
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`Table,Cherche(<Nom de la colonne> , <Valeur recherchée>)`
---

**`<Nom de la colonne> : (Chaîne de caractères)`**

Nom de la colonne où la recherche doit être effectuée.

**`<Valeur recherchée> : (Chaîne de caractères)`**

Valeur recherchée dans la colonne. Cette valeur doit avoir la même taille que la valeur de la colonne recherchée.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
**Table,Cherche** retourne dans WdEntier :

- l'indice de l'élément de la table correspondant à la recherche,

- -1 si la chaîne recherchée n'est pas trouvée.

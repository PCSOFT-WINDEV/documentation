
## HsMenu,Etat (Langage externe)

***En anglais : HsMenu,Status***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie l'état d'une option de menu (actif, invisible, grisé).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```txt
// En C
// Etat de l'option correspondant au chemin LC
APPELWD("HsMenu,Etat,LC");
if WDEntier==1 APPELWD("INFO, L'option n'est pas active");
// Etat de l'option numéro 15
APPELWD("HsMenu,Etat,15");
if WDEntier==2 APPELWD("INFO, L'option n'apparaît pas");
```


<a name="Exemple2"></a>



```txt
// En Pascal
(* Etat de l'option correspondant au chemin LC *)
APPELWD('HsMenu,Etat,LC');
if WDEntier=1 then 
	APPELWD('INFO, L'option n'est pas active');
(* Etat de l'option numéro 15 *)
APPELWD('HsMenu,Etat,15');
if WDEntier=2 then 
	APPELWD('INFO, L'option n'apparaît pas');
```


<a name="Exemple3"></a>



```txt
// En VB
' Etat de l'option correspondant au chemin LC
call APPELWD("HsMenu,Etat,LC")
if WDEntier=1 then 
	call APPELWD('INFO, L'option n'est pas active')
end if
' Etat de l'option numéro 15
call APPELWD("HsMenu,Etat,15")
if WDEntier=2 then  
	call APPELWD("INFO, L'option n'apparaît pas")
end if
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`HsMenu,Etat([<Option>])`
---

**`<Option> : (Chaîne de caractères (optionnelle))`**

Numéro qui correspond :

- soit à l'indice de l'option

- soit à une chaîne de caractères correspondant au chemin d'accès à l'option (lettres d'appels).  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
**HsMenu,Etat** retourne dans WdEntier l'état de l'option du menu. Cet état peut correspondre à :

- -1 si l'option n'existe pas

- 0 si l'option est active

- 1 si l'option est grisée

- 2 si l'option est invisible

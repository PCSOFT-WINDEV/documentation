
## Test (Langage externe)

***En anglais : Test***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Affiche chaque appel de fonction et chaque compte-rendu retourné par la fonction.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```txt
// En C
APPELWD("Test,Oui");
```


<a name="Exemple2"></a>



```txt
// En Pascal
APPELWD('Test,Oui');
```


<a name="Exemple3"></a>



```txt
// En VB
call APPELWD("Test,Oui")
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`Test(<Indicateur>)`
---

**`<Indicateur> : (Chaîne de caractères)`**

 Prend pour valeur :

- "oui" pour passer en mode test

- "non" pour inhiber le mode test.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Pour exécuter la fonction suivante, il faut appuyer sur une touche.

- Pour abandonner le mode test, il suffit d'appuyer sur la touche Echap (équivalent à Option,Test,Non).


## Ecran,EcranVersFichier (Langage externe)

***En anglais : Screen,ScreenToFile***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Initialise les rubriques de fichier avec la valeur des champs de la fenêtre reliés à ces rubriques.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```txt
// En C
APPELWD("Ecran,EcranVersFichier");
HAjoute("CLIENT");
```


<a name="Exemple2"></a>





```txt
// En Pascal
APPELWD('Ecran,EcranVersFichier');
HAjoute('CLIENT');
```


<a name="Exemple3"></a>



```txt
// En VB
call APPELWD("Ecran,EcranVersFichier")
call HAjoute("CLIENT")
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`Ecran,EcranVersFichier()`
---



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- **Ecran,EcranVersFichier** n'écrit pas directement dans le fichier. Il affecte les variables du fichier.




- Pour modifier l'enregistrement dans le fichier, il faut exécuter selon le cas, les fonctions HAjoute ou HModifie.

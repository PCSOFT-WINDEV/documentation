
## Table,Enregistre (Langage externe)

***En anglais : Table,Save***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie ou ajoute un enregistrement dans le fichier de donnée lié au champ Table.

**Remarque** : Cette fonction doit être utilisée uniquement avec les champs Table fichier.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```txt
// En C
APPELWD("Table,Enregistre,TableProd");
```


<a name="Exemple2"></a>



```txt
// En Pascal
APPELWD('Table,Enregistre,TableProd');
```


<a name="Exemple3"></a>



```txt
// En VB
call APPELWD("Table,Enregistre,TableProd")
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`Table,Enregistre(<Champ Table>)`
---

**`<Champ Table> : (Nom de champ)`**

Nom du champ Table fichier à manipuler.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- **Table,Enregistre**initialise les variables des rubriques du fichier avec les valeurs des champs de la ligne en cours du champ Table. L'enregistrement est écrit dans le fichier de données.

- **Table,Enregistre** ne gère pas les rubriques des fichiers reliés. Il faut écrire dans le fichier relié avec les fonctions HAjoute ou HModifie.

- Pour modifier le fichier de donnés lié directement à un champ Table, il ne faut par utiliser la fonction HAjoute (ou HModifie) : il faut utiliser les fonctions [Table,Ajoute](../LangageExt/75130309.md), [Table,Insere](../LangageExt/75130313.md) ou [Table,Modifie](../LangageExt/75130314.md) et [Table,Enregistre](../LangageExt/75130312.md).

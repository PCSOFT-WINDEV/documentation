
## Table,Position (Langage externe)

***En anglais : Table,Position***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Positionne un élément donné ou renvoie l'élément de la première ligne dans un champ Table par programmation.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```txt
// En C
APPELWD("Table,Position,TableNom,10");
```


<a name="Exemple2"></a>



```txt
// En Pascal
APPELWD('Table,Position,TableNom,10');
```


<a name="Exemple3"></a>



```txt
// En VB
call APPELWD("Table,Position,TableNom,10")
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`Table,Position(<Champ Table> [, <Indice>])`
---

**`<Champ Table> : (Nom de champ)`**

Nom du champ Table à manipuler.

**`<Indice> : (Chaîne de caractères (optionnelle))`**

Indice de l'élément dans le champ Table.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- **Table,Position** ne peut pas être utilisé avec les champs Table fichier.

- Si l'indice n'est pas précisé, WdEntier contient l'indice correspondant à la première ligne affichée.

- Si l'indice est précisé, l'élément correspondant est positionné en haut du champ Table.


## Table,Insere (Langage externe)

***En anglais : Table,Insert***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Insère un élément dans une table à une position donnée. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```txt
// En C
APPELWD("Table,Insere,TableProd,10=%s",Ligne);
```


<a name="Exemple2"></a>





```txt
// En Pascal
APPELWD('Table,Insere,TableProd,10='+Ligne);
```


<a name="Exemple3"></a>





```txt
// En VB
call APPELWD("Table,Insere,TableProd,10="+Ligne)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`Table,Insere(<Nom de la table> , <Indice> , <Valeur de la ligne>)`
---

**`<Nom de la table> : (Chaîne de caractères)`**

Nom de la table à manipuler.

**`<Indice> : (Chaîne de caractères)`**

Indice de l'élément dans la table.

**`<Valeur de la ligne> : (Chaîne de caractères)`**

Valeur de la ligne à insérer.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Notes
<a name="notes_ELTPARAGRAPHE000046"></a>

- Après la fonction **Table,Insere**, la table n'est pas en saisie. Pour pouvoir saisir dans la table, il faut appeler [Ecran,Saisie](../LangageExt/75130265.md).

- Si la table comporte plusieurs colonnes, il faut séparer chaque colonne par la valeur numérique de [TAB] (code binaire 9) :

	- Exemple en C : 
			
		```txt
		APPELWD("Table,Insere,TabCli,14=DUPOND\tJean");
		```


	- Exemple en Pascal : 
			
		```txt
		APPELWD('Table,Insere,TabCli,14=DUPOND'+#9+'Jean');
		```


	- Exemple en VB : 
			
		```txt
		call APPELWD("Table,Insere,TabCli,14=DUPOND"+CHR$(9)+"Jean")
		```









<a name="NOTE0_2"></a>




### Table fichier
<a name="table_fichier_ELTPARAGRAPHE000075"></a>

- Dans le cas d'une table fichier, l'enregistrement n'est pas inséré dans le fichier, il faut appeler la fonction [Table,Enregistre](../LangageExt/75130312.md).

- Pour modifier le fichier associé à une table, il ne faut par utiliser la fonction HAjoute : il faut utiliser les fonctions **Table,Insere** et [Table,Enregistre](../LangageExt/75130312.md).

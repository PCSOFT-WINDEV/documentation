
## Table,Ajoute (Langage externe)

***En anglais : Table,Add***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ajoute un élément dans un champ Table.

**Remarque** : Pour ajouter un élément à une position précise, il faut utiliser la fonction [Table,Insere](../LangageExt/75130313.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```txt
// En C
APPELWD("Table,Ajoute,TableProduit=%s",Ligne);
```


<a name="Exemple2"></a>



```txt
// En Pascal
APPELWD('Table,Ajoute,TableProduit='+Ligne);
```


<a name="Exemple3"></a>





```txt
// En VB
call APPELWD("Table,Ajoute,TableProduit="+Ligne)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`Table,Ajoute(<Champ Table> , <Valeur de la ligne>)`
---

**`<Champ Table> : (Nom de champ)`**

Nom du champ Table à manipuler.

**`<Valeur de la ligne> : (Chaîne de caractères)`**

Valeur de la ligne à ajouter.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Ajout dans un champ Table par programmation
<a name="ajout_dans_champ_table_par_programmation_ELTPARAGRAPHE000049"></a>&lt;Valeur de la ligne&gt; est ajoutée à la dernière ligne du champ Table par programmation, même si elle n'est pas affichée.

Après la fonction **Table,Ajoute**, le champ Table n'est pas en saisie : pour saisir dans le champ Table, il faut appeler la fonction [Ecran,Saisie](../LangageExt/75130265.md).


<a name="NOTE0_2"></a>




### Ajout dans un champ Table fichier
<a name="ajout_dans_champ_table_fichier_ELTPARAGRAPHE000063"></a>&lt;Valeur de la ligne&gt; est ajoutée à la première ligne disponible du champ Table, ou si aucune ligne n'est disponible, à la dernière ligne visible du champ Table.

Le fichier n'est pas modifié : il faut exécuter la fonction [Table,Enregistre](../LangageExt/75130312.md) pour modifier le fichier.

Après la fonction **Table,Ajoute**, le champ Table n'est pas en saisie : pour saisir dans le champ Table, il faut appeler [Ecran,Saisie](../LangageExt/75130265.md).

Dans un champ Table fichier, il faut appeler [Table,Enregistre](../LangageExt/75130312.md) ou [Table,Supprime](../LangageExt/75130320.md) après **Table,Ajoute** avant de pouvoir appeler à nouveau **Table,Ajoute**(ou [Table,Insere](../LangageExt/75130313.md)). Dans le cas contraire, le contenu de la ligne est perdu.

Si le champ Table comporte plusieurs colonnes, les valeurs d'initialisation doivent être séparées par la valeur du caractère [TAB] (code binaire 9):

- Exemple en C :
	
	```txt
	APPELWD("Table,Ajoute,TabClient=DUPOND\tJean");
	```


- Exemple en Pascal :
	
	```txt
	APPELWD('Table,Ajoute,TabClient=DUPOND'+#9+'Jean');
	```


- Exemple en VB :
	
	```txt
	call APPELWD("Table,Ajoute,TabClient=DUPOND"+CHR$(9)+"Jean")
	```

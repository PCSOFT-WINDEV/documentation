
## Sablier (Langage externe)

***En anglais : Hourglass***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Affiche un pointeur de souris en forme de sablier.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```txt
// En C
APPELWD("Sablier,Oui");
```


<a name="Exemple2"></a>



```txt
// En Pascal
APPELWD('Sablier,Oui');
```


<a name="Exemple3"></a>



```txt
// En VB
call APPELWD("Sablier,Oui")
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`Sablier(<Indicateur>)`
---

**`<Indicateur> : (Chaîne de caractères)`**

Prend pour valeur :

- "Oui" pour afficher le pointeur en sablier

- "Non" pour réafficher le pointeur de souris initial.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Par défaut la fonction Sablier,Non est appelée.

- Le sablier reprend automatiquement sa forme initiale lors de l'appel de la fonction [Ecran,Saisie](../LangageExt/75130265.md).

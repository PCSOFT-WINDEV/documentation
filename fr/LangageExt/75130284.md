
## Liste,Cherche (Langage externe)

***En anglais : List,Search***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Recherche une valeur dans une liste. La recherche effectuée est une recherche à l'identique.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```txt
// En C
APPELWD("Liste,Cherche,ListeProd,000002");
if WDEntier==-1 APPELWD("ERREUR,Element non trouvé");
```


<a name="Exemple2"></a>



```txt
// En Pascal
APPELWD('Liste,Cherche,ListeProd,000002');
if WDEntier=-1 then APPELWD('ERREUR,Element non trouvé');
```


<a name="Exemple3"></a>



```txt
// En VB
call APPELWD("Liste,Cherche,ListeProd,000002")
if WDEntier=-1 then APPELWD("ERREUR,Element non trouvé")
end if
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`Liste,Cherche(<Nom de la liste> , <Chaine recherchée>)`
---

**`<Nom de la liste> : (Chaîne de caractères)`**

Nom du champ Liste à modifier.

**`<Chaine recherchée> : (Chaîne de caractères)`**

Valeur recherchée dans la liste. Cette valeur est recherchée dans toute la liste.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Liste,Cherche
<a name="listecherche_ELTPARAGRAPHE000041"></a>

- l'indice de l'élément de la liste correspondant à la recherche,

- -1 si la chaîne recherchée n'est pas trouvée..


## Info (Langage externe)

***En anglais : Info***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Affiche une boîte de dialogue contenant un sigle d'information et le message passé en paramètre.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```txt
// En C
APPELWD("Info,Bonjour");
```


<a name="Exemple2"></a>



```txt
// En Pascal
APPELWD('Info,Bonjour');
```


<a name="Exemple3"></a>





```txt
// En VB
call APPELWD("Info,Bonjour")
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`Info(<Message>)`
---

**`<Message> : (Chaîne de caractères)`**

Message à afficher.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- La fenêtre sera fermée lorsque l'utilisateur valide le bouton "OK".

- La fenêtre a par défaut le même titre que la fenêtre en cours. Pour modifier le titre, il faut utiliser la fonction Windows,TSuivant.

- La taille maximum du message est de 200 caractères.

- Pour afficher le message sur plusieurs lignes, il faut utiliser la syntaxe suivante :

	- Exemple en C : 
			
		```txt
		APPELWD("Info,  \n ");
		```


	- Exemple en Pascal :
			
		```txt
		APPELWD('Info, <Texte1>'+Chr(10)+'<Texte2>');
		```


	- Exemple en VB : 
			
		```txt
		call APPELWD("Info, "+CHR(10)+"")
		```

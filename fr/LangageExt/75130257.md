
## Ecran,Colonne (Langage externe)

***En anglais : Screen,Column***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie l'indice de la colonne en cours dans le cas de colonne indicée dans un champ Table fichier ou dans un champ Table par programmation.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```txt
// En C
APPELWD("Ecran,Colonne");
NumCol=WDEntier;
```


<a name="Exemple2"></a>





```txt
// En Pascal
AppelWd('Ecran,Colonne');
NumCol:=WdEntier;
```


<a name="Exemple3"></a>





```txt
// En VB
call AppelWd("Ecran,Colonne")
NumCol=WdEntier
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`Ecran,Colonne()`
---



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- **Ecran,Colonne** retourne le numéro de la colonne en cours dans WdEntier.

- **Ecran,Colonne** ne peut être utilisée que sur des colonnes indicées dans des champs Table fichier ou des champs Table par programmation.

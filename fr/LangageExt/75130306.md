
## SonJoue (Langage externe)

***En anglais : WavPlay***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Joue le son demandé. L'interpréteur de son doit être initialisé avec [SonInit](../LangageExt/75130305.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```txt
// En C
APPELWD("SonJoue,DING.WAV");
```


<a name="Exemple2"></a>



```txt
// En Pascal
APPELWD('SonJoue,DING.WAV');
```


<a name="Exemple3"></a>





```txt
// En VB
call APPELWD("SonJoue,DING.WAV")
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`SonJoue(<Nom du son>)`
---

**`<Nom du son> : (Chaîne de caractères)`**

Nom du fichier son à jouer. Ce fichier est de type ".WAV" ou ".VOC".

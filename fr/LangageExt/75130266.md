
## Ecran,Selection (Langage externe)

***En anglais : Screen,Selection***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Modifie l'état d'un champ (invisible, actif, inactif, grisé)
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```txt
// En C
APPELWD("Ecran,Selection,-,Nom "); // rendre inactif champ Nom
```


<a name="Exemple2"></a>





```txt
// En Pascal
APPELWD('Ecran,Selection,-,Nom);    (* rendre inactif le champ Nom *)
```


<a name="Exemple3"></a>





```txt
// En VB
call APPELWD("Ecran,Selection,-,Nom") ' rendre inactif le champ Nom
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`Ecran,Selection(<Etat> , <Liste des champs>)`
---

**`<Etat> : (Chaîne de caractères)`**

Nouvel état des champs. Les états disponibles sont les suivants : 
+ : actif
\* : inactif
I : invisible
G : grisé.

**`<Liste des champs> : (Chaîne de caractères)`**

Liste des champs dont l'état doit être modifié.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Si le champ est un champ indicé, la modification de l'état s'opère sur tous les indices du champ.

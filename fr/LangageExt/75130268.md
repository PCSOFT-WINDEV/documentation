
## Ecran,Xvar (Langage externe)

***En anglais : Screen,Xvar***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Détermine si un champ fait partie ou non de la fenêtre en cours.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```txt
// En C
// Vérifie si le champ NOMCLI est présent dans la fenêtre
APPELWD("Ecran,XVar,%s",NomChamp);
if WDEntier==1 APPELWD("ECRAN,PREMIER,%s",NomChamp);
```


<a name="Exemple2"></a>





```txt
// En Pascal
(* Vérifie si le champ NOMCLI est présent dans la fenêtre *)
APPELWD('Ecran,XVar,NOMCLI');
if WdEntier=1 then APPELWD('ECRAN,PREMIER,'+NomChamp);
```


<a name="Exemple3"></a>



```txt
// En VB
' Vérifie si le champ NOMCLI est présent dans la fenêtre
call APPELWD("Ecran,XVar,NOMCLI")
if WdEntier=1 then 
	APPELWD("ECRAN,PREMIER,"+NomChamp)
end if
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`Ecran,Xvar(<Nom du champ> [, <Indice>])`
---

**`<Nom du champ> : (Chaîne de caractères)`**

Nom du champ dans lequel le texte doit être affiché.

**`<Indice> : (Chaîne de caractères (optionnelle))`**

Indice du champ.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
La variable WdEntier vaut :

- 0 si le champ n'est pas présent dans la fenêtre 

- 1 si le champ est présent dans la fenêtre.

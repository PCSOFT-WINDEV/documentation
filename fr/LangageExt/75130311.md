
## Table,Colonne (Langage externe)

***En anglais : Table,Col***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie l'indice en cours de la colonne.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```txt
// En C
APPELWD("Table, Colonne, NomTab");
NumLigne=WdEntier;
```


<a name="Exemple2"></a>





```txt
// En Pascal
APPELWD('Table, Colonne, NomTab');
NumLigne=WdEntier;
```


<a name="Exemple3"></a>





```txt
// En VB
call APPELWD("Table, Colonne, NomTab")
NumLigne=WdEntier
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`Table,Colonne(<Nom de la table>)`
---

**`<Nom de la table> : (Chaîne de caractères)`**

Nom de la table à manipuler.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
WdEntier contient le numéro de la ligne en cours.

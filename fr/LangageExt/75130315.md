
## Table,Occurrence (Langage externe)

***En anglais : Table,Count***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le nombre d'éléments d'un champ Table. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```txt
// En C
APPELWD("Table,Occurrence,TableProd");
```


<a name="Exemple2"></a>



```txt
// En Pascal
APPELWD('Table,Occurrence,TableProd');
```


<a name="Exemple3"></a>





```txt
// En VB
call APPELWD("Table,Occurrence,TableProd")
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`Table,Occurrence(<Champ Table>)`
---

**`<Champ Table> : (Nom de champ)`**

Nom du champ Table à manipuler.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Dans le cas d'un champ Table par programmation, WdEntier contient le nombre d'éléments du champ Table.

- Dans le cas d'un champ Table fichier, WdEntier contient le nombre d'éléments affichés dans le champ Table.


## Liste,Select,? (Langage externe)

***En anglais : List,Select,?***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Retourne l'élément sélectionné dans la liste.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```txt
// En C
APPELWD("Liste,Select,?,ListeProd,1");
strcpy(Selection,WDChaine);
strcpy(Indice,WDEntier);
```


<a name="Exemple2"></a>



```txt
// En Pascal
APPELWD('Liste,Select,?,ListeProd,1');
Selection:=WdChaine;
Indice:=WdEntier;
```


<a name="Exemple3"></a>



```txt
// En VB
call APPELWD("Liste,Select,?,ListeProd,1")
Selection=WdChaine
Indice=WdEntier
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`Liste,Select,?(<Nom de la liste> , <Rang>)`
---

**`<Nom de la liste> : (Chaîne de caractères)`**

Nom de la liste à manipuler.

**`<Rang> : (Chaîne de caractères)`**

Rang de l'élément sélectionné (liste multisélections) : 1 pour obtenir le premier élément sélectionné, 2 pour le second, ...  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- WdEntier contient l'indice de l'élément sélectionné.

- WdChaine contient la valeur de l'élément sélectionné.

- Dans le cas où aucune sélection n'a été effectuée, WdEntier est égal à -1.

- Si cette fonction est utilisée sur les combos avec saisie, la valeur retournée par WdEntier est incohérente.

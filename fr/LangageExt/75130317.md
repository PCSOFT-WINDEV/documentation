
## Table,Recupere (Langage externe)

***En anglais : Table,Get***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Récupère la valeur d'un élément donné ou de l'élément choisi dans une table.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```txt
// En C
APPELWD("Table,Recupere,TableProd");
strcpy(Valeur,WDChaine);
```


<a name="Exemple2"></a>



```txt
// En Pascal
APPELWD('Table,Recupere,TableProd');
Valeur:=WdChaine;
```


<a name="Exemple3"></a>



```txt
// En VB
call APPELWD("Table,Recupere,TableProd")
Valeur=WdChaine
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`Table,Recupere(<Nom de la table> [, <Indice>])`
---

**`<Nom de la table> : (Chaîne de caractères)`**

Nom de la table ou de la colonne.

**`<Indice> : (Chaîne de caractères  (optionnelle))`**

Indice de l'élément dans la table.

- Si ce paramètre n'est pas précisé, **Table,Recupere** retourne la valeur de l'élément courant dans la table.

- Si ce paramètre est précisé, **Table,Recupere** retourne la valeur de l'élément dont l'indice est spécifié.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
La valeur de l'élément récupéré est contenu dans WdChaine. L'indice de l'élément est contenu dans WdEntier.

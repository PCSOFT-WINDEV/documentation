
## WDADMINEXE : Caractéristiques des utilisateurs
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000112"></a>
L'onglet "Utilisateurs" de WDADMINEXE permet de connaître à partir du poste serveur :

- Les utilisateurs connectés actuellement à l'application.

- Pour chaque utilisateur connecté :

	- le poste de lancement de l'application,

	- la version de l'application utilisée.

## Notes
<a name="notes_ELTTEXTE000136"></a>


- Les données affichées proviennent des fichiers "LOKxxx.tmp" présents dans le sous-répertoire "\\INSTALL\\LOCK" du répertoire d'installation de l'application sur le poste serveur. Ces fichiers sont temporaires (présent uniquement pendant la durée de la connexion).

- Si la gestion des utilisateurs n'est pas intégrée dans le programme d'installation de l'application, l'onglet "Utilisateurs" sera grisé. Pour intégrer la gestion des utilisateurs dans le programme d'installation, consultez [Gestion des utilisateurs d'une application](../Editeurs/2028059.md).

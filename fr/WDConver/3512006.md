## Présentation
<a name="presentation_ELTTEXTE000215"></a>


### L'utilitaire WDConver
<a name="utilitaire_wdconver_ELTPARAGRAPHE000011"></a>WDConver est un utilitaire permettant de convertir vers un fichier de données HFSQL Classic, les **données** d'un fichier :

- au format Hyper File 5 ou 4.

- au format texte.

- au format XML.

- d'une base de données accédée via un provider OLE DB ou un driver ODBC (Excel, Access, Oracle, SQL Server, ...).

- d'une base de données accédée via un accès natif WINDEV (SQL Server, Oracle ou AS/400).




**Attention** : WDConver permet uniquement de convertir les **données** d'un fichier. Pour effectuer cette conversion, il est nécessaire de convertir au préalable la **structure** du fichier de données. Pour convertir la structure d'un fichier de données, sous l'éditeur d'analyses, sous le volet "Analyse", dans le groupe "Création", déroulez "Importer" et sélectionnez "Importer des descriptions de fichiers/tables". 

**Remarque** : Par programmation, les importations peuvent être réalisées grâce aux fonctions [HImporteHF55](../WDLang4/3044055.md), [HImporteTexte](../WDLang4/3044011.md), [HImporteXML](../WDLang4/3044007.md), ...

**Exemple d'utilisation** : Après une mise à jour importante des données d'un fichier externe, WDConver permet de ré-importer toutes les données dans une base de données WINDEV.

## Utilisation
<a name="utilisation_ELTTEXTE000239"></a>


### Conditions d'utilisation
<a name="conditions_utilisation_ELTPARAGRAPHE000053"></a>WDConver est un outil redistribuable. WDConver peut être installé avec les applications développées avec WINDEV. La licence d'utilisation de WINDEV s'applique intégralement.

Lors de la création de la procédure d'installation, il est possible de livrer WDConver avec vos applications.

**Rappel** : La procédure d'installation peut être créée :

- soit depuis l'assistant : sous le volet "Projet", dans le groupe "Génération", déroulez "Procédure d'installation" et sélectionnez "Créer la procédure d'installation", 

- soit depuis l'éditeur d'installation : sous le volet "Outils", dans le groupe "Utilitaires", cliquez sur "WDInst".

## Lancement de WDConver
<a name="lancement_wdconver_ELTTEXTE000269"></a>
WDConver peut être lancé :

- [**En mode interactif**](../WDConver/3512002.md), en lançant directement le programme "WDConver.EXE" ou depuis l'éditeur d'analyses : sous le volet "Analyse", dans le groupe "Création", déroulez "Importer" et sélectionnez "Importer des données d'une autre base".

- [**En mode ligne de commande**](../WDConver/3512003.md) pour exécuter une conversion directement depuis une application WINDEV.

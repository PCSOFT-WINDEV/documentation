
## Comment associer une image à un élément d'une table hiérarchique ? 
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Lors du remplissage d'un champ Table hiérarchique, une image par défaut est associée à chaque élément. Il s'agit d'un pictogramme représentant un dossier ouvert ou fermé. En effet, chaque niveau et chaque élément de la hiérarchique peut être :

- enroulé (image par défaut dossier fermé).

- déroulé (image par défaut dossier ouvert).


Ces images sont modifiables par programmation.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Méthode
<a name="methode_ELTTEXTE000101"></a>
Pour modifier les images représentant l'élément "enroulé" ou "déroulé", utilisez les propriétés [ImageEnroulée](../Proprietes/2513016.md) et [ImageDéroulée](../Proprietes/2513015.md). La syntaxe pour utiliser ces propriétés est la suivante :

```txt
<Nom Table hiérarchique>[Indice ligne].ImageEnroulée = <Nom image>
<Nom Table hiérarchique>[Indice ligne].ImageDéroulée = <Nom image>
```


Exemples de code :


```wl
TABLEH_Produit[10].ImageEnroulée = "ProduitPrixNormal.jpg"
TABLEH_Produit[11].ImageEnroulée = "ProduitEnPromo.jpg"
```


## Comment gérer une combinaison par programmation ?
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Une combinaison est un type de variable contenant une liste de valeurs. Ce type de variable permet de normaliser les valeurs possibles d'un paramètre. Par exemple, le développeur peut ainsi initialiser ce paramètre uniquement à partir des valeurs possibles imposées par la combinaison. 

La variable associée à cette combinaison peut prendre plusieurs valeurs de la liste possible en les additionnant.

Pour plus de détails, consultez [Combinaison (Type de variable)](../Motscles/1514074.md).

## Déclaration et utilisation
<a name="declaration_utilisation_ELTTEXTE000121"></a>


### Syntaxes
<a name="syntaxes_ELTPARAGRAPHE000023"></a>La syntaxe pour déclarer une combinaison est la suivante : 


```txt
NomCombinaison est une Combinaison 
	Valeur possible 1
	Valeur possible 2
	Valeur possible 3
	...
FIN
```


La syntaxe pour utiliser une combinaison précédemment déclarée est la suivante : 


```txt
NomVariable est un NomCombinaison 

NomVariable = Valeur possible 1 + Valeur possible 3
```



### Exemple
<a name="exemple_ELTPARAGRAPHE000034"></a>
```wl
ModePossible est une Combinaison 
	Gras
	Italique
	Souligné
FIN

ParamMode est un ModePossible 

ParamMode = Gras + Souligné
```

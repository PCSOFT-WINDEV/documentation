
## Comment associer une image à un élément d'un arbre ?
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Lors du remplissage d'un champ Arbre, par défaut une image est associée à chaque élément. Il s'agit d'un pictogramme représentant un dossier ouvert ou fermé. En effet, chaque niveau et chaque élément de l'arbre peut être :

- enroulé (image par défaut dossier fermé). 

- déroulé (image par défaut dossier ouvert). 


Ces images sont modifiables par programmation.

Remarque : L'image proposée par défaut pour les éléments d'un champ Arbre peut être également modifiée par programmation grâce aux propriétés [ImageEnroulée](../Proprietes/2513016.md) et [ImageDéroulée](../Proprietes/2513015.md). 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Méthode
<a name="methode_ELTTEXTE000114"></a>
Pour modifier les images représentant l'élément **enroulé** ou **déroulé**, il faut spécifier les images souhaitées dans les fonctions [ArbreAjoute](../WDLang1/3018012.md) et [ArbreInsère](../WDLang1/3018023.md).

La syntaxe pour la fonction [ArbreAjoute](../WDLang1/3018012.md) est la suivante : 

```txt
ArbreAjoute(<Nom Arbre>, <Elément>, <Nom image enroulée>, <Nom image déroulée>)
```


Exemple de code :


```wl
ArbreAjoute(ARBRE_Article, "Homme", "H1.jpg", "H2.jpg")
ArbreAjoute(ARBRE_Article, "Femme", "F1.jpg", "F2.jpg")


ArbreAjoute(ARBRE_Article, "Homme" + TAB + "Pantalon", "pantalon.JPG", "pantalon.JPG")
ArbreAjoute(ARBRE_Article, "Homme" + TAB + "Veste", "veste.JPG", "veste.JPG")
ArbreAjoute(ARBRE_Article, "Homme" + TAB + "Chaussures", "chaussH.JPG", "chaussH.JPG")
ArbreAjoute(ARBRE_Article, "Femme" + TAB + "Robe", "robe.JPG", "robe.JPG")
ArbreAjoute(ARBRE_Article, "Femme" + TAB + "Pull", "pull.JPG", "pull.JPG")
ArbreAjoute(ARBRE_Article, "Femme" + TAB + "Chaussures", "chaussF.JPG", "chaussF.JPG")
```

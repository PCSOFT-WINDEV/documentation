
## Comment gérer une énumération par programmation ?
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Une énumération est un type de variable contenant une liste de valeurs. Ce type de variable permet par exemple de normaliser les valeurs possibles d'un paramètre. Ainsi le développeur pourra initialiser ce paramètre uniquement à partir des valeurs possibles imposées par l'énumération. 

La variable associée à cette énumération ne peut prendre qu'une seule valeur de la liste possible.

Pour plus de détails sur les énumérations, consultez [Enumération (Type de variable)](../Motscles/1514067.md). 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Déclaration et utilisation
<a name="declaration_utilisation_ELTTEXTE000118"></a>


### Syntaxe
<a name="syntaxe_ELTPARAGRAPHE000023"></a>


```txt
NomEnumération est une Enumération
	Valeur possible 1
	Valeur possible 2
	...
FIN


NomVariable est un NomEnumération 

NomVariable = Valeur possible 1
```



### Exemple
<a name="exemple_ELTPARAGRAPHE000028"></a>

- Déclaration de l'énumération : 
	
	```wl
	// Déclarations globales de la fenêtre XXX
	ActionPossible est une Enumération
		Création
		Modification
		Suppression
		Impression
	FIN
	```

	```wl
	// Utilisation de l'énumération
	ParamAction est un ActionPossible
	
	ParamAction = Création
	
	Ouvre(FEN_Client, ParamAction)
	```


- Dans l'événement "Déclarations globales" de la fenêtre FEN_Client : 
	
	```wl
	PROCEDURE FEN_Client(ModeTravail est un ActionPossible)
	```

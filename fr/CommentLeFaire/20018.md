
## Comment donner le focus à une colonne d'un champ Table ?
			

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Principe
<a name="principe_ELTTEXTE000068"></a>
Pour donner le focus à une colonne d'un champ Table (c'est-à-dire se positionner en saisie dans cette colonne), il faut :

1. Se positionner sur la ligne voulue. 

2. Utiliser la fonction [DonneFocusEtRetourUtilisateur](../WDLang1/1410088107.md) pour donner le focus à la colonne voulue. 




Exemple de code : 

```wl
nNumLigne est un entier

nNumLigne = 5
TABLE_LigneCde = nNumLigne
// On se positionne dans la colonne RefProduit sur la ligne n°5
DonneFocusEtRetourUtilisateur(TABLE_LigneCde.COL_RefProduit)
```




## Fonctions de gestion du Groupware Utilisateur
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Ces fonctions permettent de personnaliser la gestion du groupware utilisateur.



|   |   |
| --- | --- |
| [gpwActiveUtilisateur](../WDLang6/1000021096.md) | Valide la création d'un nouvel utilisateur dans la base de données du groupware utilisateur en activant l'utilisateur. |
| [gpwAjouteAuthLogin](../WDLang6/1000023576.md) | Ajoute un bouton (Facebook, Google, Microsoft, ...) de connexion dans la fenêtre ou dans la page de login. |
| [gpwAjouteUtilisateur](../WDLang6/1000021094.md) | Ajoute un utilisateur dans la base de données du groupware utilisateur. |
| [gpwAjouteUtilisateurDansGroupe](../WDLang6/1000024595.md) | Ajoute un utilisateur dans un groupe de la base de données du groupware utilisateur. |
| [gpwAucun](../WDLang6/1000023557.md) | Renvoie la valeur du mot "AUCUN" selon le framework utilisé. |
| [gpwAuthLogin](../WDLang6/1000023597.md) | Permet de se connecter au Groupware Utilisateur en utilisant une authentification avec un compte de type Facebook, Google, Microsoft, etc. |
| [gpwConnecteDistant](../WDLang6/1000024855.md) | Permet de se connecter à l'application via le Webservice du groupware utilisateur. |
| [gpwConnecteUtilisateur](../WDLang6/1000021099.md) | Connecte un utilisateur au groupware utilisateur. |
| [gpwCrypteMotDePasse](../WDLang6/1000021617.md) | Renvoie le mot de passe crypté. |
| [gpwDéconnecteUtilisateur](../WDLang6/1000021100.md) | Déconnecte l'utilisateur courant du groupware utilisateur. |
| [gpwDéfaut](../WDLang6/1000023556.md) | Renvoie la valeur correspondant à "DEFAUT" selon le framework utilisé. |
| [gpwEnModeLDAP](../WDLang6/1000017056.md) | Permet de savoir si l'identification des utilisateurs dans le groupware utilisateur sera réalisée via un annuaire LDAP. |
| [gpwEnumèreChamp](../WDLang6/1000020536.md) | Enumère les champs d'une fenêtre ou d'une page. |
| [gpwImportUtilisateursLDAP](../WDLang6/1000017058.md) | Importe les utilisateurs enregistrés dans un annuaire LDAP dans le fichier des utilisateurs du groupware utilisateur. |
| [gpwInitAnalyse](../WDLang6/3041005.md) | Initialisation de l'analyse du groupware utilisateur. |
| [gpwListeOAuthParamètres](../WDLang6/1000024479.md) | Récupère la liste des paramètres OAuth définis pour le groupware utilisateur, c'est-à-dire la liste des providers (Google, Facebook, ...) auxquels il est possible de se connecter via le groupware utilisateur. Ces paramètres ont été définis via le site d'administration du groupware utilisateur. |
| [gpwLogin](../WDLang6/3041001.md) | Connecte directement un utilisateur à une application utilisant le groupware utilisateur. |
| [gpwMode](../WDLang6/1000020566.md) | Renvoie le mode d'exécution du groupware utilisateur. |
| [gpwModifieUtilisateur](../WDLang6/1000021095.md) | Modifie un utilisateur dans la base de données du groupware utilisateur. |
| [gpwMotPasseOublié](../WDLang6/1000021097.md) | Exécute une action pour récupérer ou changer le mot de passe d'un utilisateur. |
| [gpwMotsDePasseCryptés](../WDLang6/1000021616.md) | Permet de savoir si les mots de passe du groupware utilisateur sont cryptés. |
| [gpwOuvre](../WDLang6/3041002.md) | Ouvre la fenêtre ou la page de login du groupware utilisateur. |
| [gpwOuvreConfiguration](../WDLang6/3041009.md) | Ouvre la fenêtre ou la page de configuration du groupware utilisateur. |
| [gpwOuvreConnexion](../WDLang6/3041007.md) | Ouvre une connexion à une base de données spécifique Client/Serveur pour gérer les fichiers du groupware utilisateur en mode Client/Serveur. |
| [gpwOuvreSessionLDAP](../WDLang6/1000017055.md) | Indique au groupware utilisateur que l'authentification des utilisateurs sera réalisée via un annuaire LDAP. |
| [gpwOuvreSite](../WDLang6/1000020821.md) | Permet de lancer le site lors de l'utilisation du groupware utilisateur. |
| [gpwRechercheUtilisateur](../WDLang6/1000021087.md) | Récupère les caractéristiques d'un utilisateur du groupware utilisateur. |
| [gpwRecupDroits](../WDLang6/1000020728.md) | Permet de récupérer les droits d'un champ, d'une fenêtre, d'une page ou d'un état. |
| [gpwRecupInfoFichier](../WDLang6/3041008.md) | Récupère le répertoire des fichiers de données du groupware utilisateur. |
| [gpwRecupInfoLDAP](../WDLang6/1000017063.md) | Récupère des les caractéristiques du serveur LDAP utilisé pour le groupware utilisateur.  |
| [gpwRecupInfoUtilisateur](../WDLang6/3041004.md) | Récupère des informations sur l'utilisateur en cours (application utilisant le groupware utilisateur). |
| [gpwRéinitialiseMotDePasse](../WDLang6/1000024854.md) | Réinitialise le mot de passe d'un utilisateur inscrit. |
| [gpwSuperviseur](../WDLang6/1000023555.md) | Renvoie le login du compte "Superviseur" selon le framework utilisé. |
| [gpwUtilisateursEnregistrés](../WDLang6/1000023559.md) | Renvoie la valeur correspondant au mot "Utilisateurs enregistrés" selon le framework utilisé. |
| [gpwVérificationBiométrique](../WDLang6/1000024596.md) | Active la vérification biométrique pour l'utilisateur connecté ou récupère l'état actuel pour l'utilisateur connecté. |
| [gpwVérifieMotDePasse](../WDLang6/1000021618.md) | Vérifie un mot de passe (crypté ou non crypté). |
| [gpwVérifieUtilisateur](../WDLang6/1000021104.md) | Vérifie l'existence et la validité du mot de passe d'un utilisateur du groupware utilisateur. |
| [gpwVisiteur](../WDLang6/1000023558.md) | Renvoie la valeur correspondant au mot "Visiteur" selon le framework utilisé. |

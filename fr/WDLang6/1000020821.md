
## gpwOuvreSite (Fonction)

***En anglais : gpwOpenSite***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de lancer le site lors de l'utilisation du groupware utilisateur. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ouverture du site
gpwOuvreSite()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = gpwOuvreSite()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le site a été lancé, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.

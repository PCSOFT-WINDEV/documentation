
## gpwAjouteUtilisateurDansGroupe (Fonction)

***En anglais : gpwAddUserToGroup***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ajoute un utilisateur dans un groupe de la base de données du groupware utilisateur.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
gpwAjouteUtilisateurDansGroupe("Premium")
SI ErreurDétectée = Vrai ALORS
       // Message complet de l'erreur principale
        Trace(ErreurInfo())
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = gpwAjouteUtilisateurDansGroupe(<Groupe> [, <Déplacement>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'utilisateur a été ajouté au groupe, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. La fonction [ErreurInfo](../WDLang1/3013008.md) permet d'identifier l'erreur.  




**`<Groupe> : (Chaîne de caractères)`**

Nom du groupe auquel l'utilisateur en cours doit être associé. 

**`<Déplacement> : (Booléen optionnel)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'utilisateur doit être déplacé dans le groupe spécifié. Dans ce cas, l'utilisateur sera associé uniquement à ce groupe. Il sera supprimé de tous les autres groupes. 

- <u><u><u><u>Faux</u></u></u></u> (valeur par défaut) si l'utilisateur doit être associé au groupe spécifié tout en restant associé aux autres groupes.

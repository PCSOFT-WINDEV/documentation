
## gpwImportUtilisateursLDAP (Fonction)

***En anglais : gpwImportLDAPUsers***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Importe les utilisateurs enregistrés dans un annuaire LDAP dans le fichier des utilisateurs du groupware utilisateur.  

**Remarque** : Il est nécessaire d'ouvrir une session LDAP pour le groupware utilisateur avant d'utiliser cette fonctionnalité (fonction [gpwOuvreSessionLDAP](../WDLang6/1000017055.md)).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Groupware utilisateur utilisé en mode automatique
// Code du projet
gpwOuvreSessionLDAP("gemini.monserveur.com", 389, 3, Vrai)

// Importation des utilisateurs
gpwImportUtilisateursLDAP()
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = gpwImportUtilisateursLDAP()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'importation a été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) &lt;Résultat&gt; est toujours à <u><u><u><u>Faux</u></u></u></u>.







<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Attention
<a name="attention_ELTPARAGRAPHE000053"></a>En fonction des stratégies LDAP définies dans Active Directory, l'importation peut être limitée à 1 000 utilisateurs. Dans ce cas, pour supprimer cette limitation, il faut intervenir sur le paramètre LDAP MaxPageSize. Pour plus de détails, consultez [https://support.microsoft.com/kb/315071](https://support.microsoft.com/kb/315071).


## GraveurListe (Fonction)

***En anglais : BurnerList***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Liste les graveurs installés sur le poste en cours.

**Remarques** : 

- La fonction [GraveurSélectionne](../WDLang6/3061012.md) permet de sélectionner le graveur à utiliser.

- La fonction [GraveurChemin](../WDLang6/3061004.md) permet de connaître le lecteur associé au graveur en cours.



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple
<a class="notetitle" target="_blank" href="$DOC$=1000003061009&name=graveurliste_fonction&product=WD">Voir des exemples supplémentaires</a>

```wl
// Connaître les graveurs installés
ListeGraveur est une chaîne
ListeGraveur = GraveurListe()
SI ListeGraveur ~= "" ALORS
	Erreur("Aucun graveur n'a été détecté sur ce poste")
	RETOUR
FIN
// Pour visualiser cet exemple en totalité, cliquez sur le lien "Exemples"
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = GraveurListe([<Détail>])`
---

**`<Résultat> : (Chaîne de caractères)`**

Liste des graveurs installés sur le poste en cours, sous la forme :

- **liste détaillée (si le paramètre &lt;Détail&gt; correspond à <u><u><u><u>Vrai</u></u></u></u>)** :
	
	```txt
	<Identifiant du graveur 1> + TAB + <Marque du graveur 1> + TAB + 
	<Modèle du graveur 1> + TAB + <Logiciel intégré du graveur 1> + RC + ...
	<Identifiant du graveur N> + TAB + <Marque du graveur N> + TAB + 
	<Modèle du graveur N> + TAB + <Logiciel intégré du graveur N>
	```


- **liste non détaillée (si le paramètre &lt;Détail&gt; correspond à <u><u><u><u>Faux</u></u></u></u>**) :
	
	```txt
	<Identifiant du graveur 1> + RC + ... + <Identifiant du graveur N>
	```



Où : 

- &lt;Identifiant du graveur 1&gt; ... &lt;Identifiant du graveur N&gt; correspondent à l'identifiant des graveurs.

- &lt;Marque du graveur 1&gt; ... &lt;Marque du graveur N&gt; correspondent à la marque des graveurs.

- &lt;Modèle du graveur 1&gt; ... &lt;Modèle du graveur N&gt; correspondent au modèle des graveurs.

- &lt;Logiciel intégré du graveur 1&gt; ... &lt;Logiciel intégré du graveur N&gt; correspondent à la version du logiciel intégré (appelé également "firmware") des graveurs.






**`<Détail> : (Booléen optionnel)`**



- <u><u><u><u>Vrai</u></u></u></u> pour lister les informations détaillées des graveurs, 

- <u><u><u><u>Faux</u></u></u></u> (par défaut) pour lister uniquement l'identifiant des graveurs.

<a name="configuration_necessaire_ELTPARAGRAPHE000078"></a>![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) La gravure de CD est disponible uniquement sous Windows XP et supérieur. La gravure de DVD est disponible sous Windows Vista, et sous Windows XP en utilisant la mise à jour KB932716 pour Windows XP (cette mise à jour doit être téléchargée depuis le site de Microsoft et installée car elle n'est pas prise en compte par le mécanisme de Live Update).


## ScanListe (Fonction)

***En anglais : ScanList***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Liste les scanneurs connectés au poste en cours.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affiche la liste des scanneurs du poste en cours dans le champ LISTE_Scanneur
ListeAjoute(LISTE_Scanneur, ScanListe())
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = ScanListe()`
---

**`<Résultat> : (Chaîne de caractères)`**

Liste des scanneurs connectés au poste en cours, sous la forme :

```txt
<Nom du scanneur 1> + RC + ... <Nom du scanneur N>
```


où : &lt;Nom du scanneur 1&gt; ... &lt;Nom du scanneur N&gt; correspondent au nom des scanneurs connectés.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Deux scanneurs ne peuvent pas avoir le même nom : le système ajoute automatiquement "#&lt;numéro&gt;" si un scanneur de même nom existe déjà.

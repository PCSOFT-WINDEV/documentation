


## &lt;Variable pdfDocument&gt;.RécupèreDateSignature (Fonction)

***En anglais : &lt;pdfDocument variable&gt;.GetSignatureDate***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Récupère la date de l'une des signatures d'un document PDF.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = PDFRécupèreDateSignature(<Document PDF> , <Indice>)`
---

**`<Résultat> : (DateHeure)`**

Date de la signature examinée.

**`<Document PDF> : (Variable de type pdfDocument)`**



**`<Indice> : (Entier)`**

Indice de la signature dans le document (le nombre de signatures est connu par la fonction [&lt;Variable pdfDocument&gt;.NbSignature](../WDLang6/1410088154.md)).

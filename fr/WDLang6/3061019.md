
## GraveurNomVolume (Fonction)

***En anglais : BurnerVolumeName***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de connaître ou de modifier le nom du CD ou DVD à graver.

**Remarque** : Il est nécessaire d'appeler cette fonction avant d'ajouter des fichiers dans la compilation (fonctions [GraveurAjouteFichier](../WDLang6/3061001.md) ou [GraveurAjouteRépertoire](../WDLang6/3061002.md)). Dans le cas contraire, le nom du CD ne sera pas pris en compte.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Nom du CD
ResNom est un entier = GraveurNomVolume("Mon CD")
// Erreur rencontrée ?
SI ResNom = Faux ALORS
	// Affichage de l'erreur
	Erreur(ErreurInfo(errMessage))
SINON
	// Ajout de fichiers
	GraveurAjouteRépertoire("C:\MesMusiques\*.WAV")
	...
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le nom du CD ou DVD à graver

`<Résultat> = GraveurNomVolume()`
---

**`<Résultat> : (Chaîne de caractères)`**

Nom du CD ou DVD à graver.  


<a name="SYNTAXE2"></a>

### Modifier le nom du CD ou DVD à graver

`<Résultat> = GraveurNomVolume(<Nom du CD ou DVD>)`
---

**`<Résultat> : (Entier)`**



- <u><u><u><u>Vrai</u></u></u></u> si le nom du CD ou DVD a été pris en compte, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Nom du CD ou DVD> : (Chaîne de caractères)`**

Nom à donner au CD ou DVD à graver. C'est ce nom qui apparaît sous l'explorateur de fichiers pour identifier le CD ou le DVD.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) 

### Présence d'un CD/DVD dans le graveur
<a name="presence_cddvd_dans_graveur_ELTPARAGRAPHE000057"></a>Pour connaître le nom du volume dans le graveur, il est bien entendu nécessaire qu'un CD ou DVD soit présent dans le graveur. Pour modifier le nom du volume, il est nécessaire que le CD/DVD présent dans le graveur contienne assez de place pour graver une nouvelle session.  
<a name="NOTE0_2"></a>




### Configuration nécessaire 
<a name="configuration_necessaire_ELTPARAGRAPHE000065"></a>

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) La gravure de CD est disponible uniquement sous Windows XP et supérieur. La gravure de DVD est disponible sous Windows Vista, et sous Windows XP en utilisant la mise à jour KB932716 pour Windows XP (cette mise à jour doit être téléchargée depuis le site de Microsoft et installée car elle n'est pas prise en compte par le mécanisme de Live Update).

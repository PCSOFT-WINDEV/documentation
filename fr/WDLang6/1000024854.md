
## gpwRéinitialiseMotDePasse (Fonction)

***En anglais : gpwResetPassword***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Réinitialise le mot de passe d'un utilisateur inscrit.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
nRes est un entier = gpwRéinitialiseMotDePasse(sEmail)
SELON nRes
	CAS gpwOk : Info("Les informations ont été envoyées par email.")
	CAS gpwUtilisateurInconnu : Info("Oups ! Cette adresse email est inconnue.")
	AUTRE CAS : Erreur("Oups ! Une erreur s'est produite." + RC + ErreurInfo())
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = gpwRéinitialiseMotDePasse(<Email>)`
---

**`<Résultat> : (Constante de type Entier)`**

Mode de re-initialisation du mot de passe : 


|   |   |
| --- | --- |
| *gpwOk* | L'utilisateur a été identifié grâce à son adresse email et les informations de réinitialisation ont été envoyées par email. |
| *gpwUtilisateurInconnu* | L'utilisateur n'a pas été identifié grâce à son adresse email. |





**`<Email> : (Chaîne de caractères)`**

Adresse email de l'utilisateur.




## Fonctions de gravure
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions de gravure sont les suivantes : 



|   |   |
| --- | --- |
| [GraveurAjouteFichier](../WDLang6/3061001.md) | Ajoute un fichier à la compilation. |
| [GraveurAjouteRépertoire](../WDLang6/3061002.md) | Ajoute la totalité des fichiers d'un répertoire à la compilation. |
| [GraveurAnnuleGravure](../WDLang6/3061003.md) | Annule la gravure en cours de réalisation. |
| [GraveurChemin](../WDLang6/3061004.md) | Renvoie le chemin du graveur en cours. |
| [GraveurEfface](../WDLang6/3061005.md) | Efface les fichiers présents sur un CD ou DVD réinscriptible. |
| [GraveurEjecte](../WDLang6/3061014.md) | Ouvre ou ferme le tiroir du graveur en cours. |
| [GraveurEnregistre](../WDLang6/3061006.md) | Grave les fichiers présents dans la compilation sur le CD ou le DVD. |
| [GraveurEtat](../WDLang6/3061007.md) | Renvoie l'état actuel du graveur en cours. |
| [GraveurInfoCompilation](../WDLang6/3061018.md) | Récupère les caractéristiques de la compilation en cours. |
| [GraveurInfoMédia](../WDLang6/3061008.md) | Récupère les caractéristiques du CD/DVD présent dans le graveur en cours. |
| [GraveurListe](../WDLang6/3061009.md) | Liste les graveurs installés sur le poste en cours. |
| [GraveurMédiaPrésent](../WDLang6/1000019910.md) | Permet de savoir si le graveur est plein ou vide. |
| [GraveurNomVolume](../WDLang6/3061019.md) | Permet de connaître ou de modifier le nom du CD ou DVD à graver. |
| [GraveurPropriété](../WDLang6/3061011.md) | Permet de connaître ou de modifier les propriétés du graveur en cours. |
| [GraveurSélectionne](../WDLang6/3061012.md) | Permet de sélectionner le graveur utilisé par défaut. |
| [GraveurTypeMédia](../WDLang6/3061013.md) | Permet de connaître ou de modifier le format du CD ou DVD à graver. |


## OCRAjouteLangue (Fonction)

***En anglais : OCRAddLanguage***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ajoute une langue de détection à l'OCR natif. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ajoute la langue US
MonNomDeModèle est une chaîne = "en_US"
OCRAjouteLangue(MonNomDeModèle)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`OCRAjouteLangue(<Langue ISO>)`
---

**`<Langue ISO> : (Chaîne de caractères)`**

Nom de la langue à ajouter au format ISO. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Pour les plateformes autres que iOS, utilisez la fonction [OCRChargeLangue](../WDLang6/1000025790.md).

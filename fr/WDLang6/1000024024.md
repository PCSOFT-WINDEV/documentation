
## ConsoleLit (Fonction)

***En anglais : ConsoleRead***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Lit une chaîne sur le flux d'entrée standard "stdin".

Remarque : Lorsque la fonction est appelée, la console attend que l'utilisateur tape du texte. Dès que l'utilisateur tape le caractère '\\n' (en appuyant sur la touche Entrée), la chaîne renseignée est renvoyée par la fonction.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
ConsoleEcrit("Veuillez renseigner votre prénom : ")

soit sPrénom = ConsoleLit()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = ConsoleLit()`
---

**`<Résultat> : (Chaîne de caractères)`**

Caractères entrés sur le flux d'entrée standard. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Attention : Sous Windows, l'exécutable généré doit être en mode console pour permettre à l'utilisateur de saisir du texte.

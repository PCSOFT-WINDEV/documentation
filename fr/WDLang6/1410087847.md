


## PDFAjouteImage (Fonction)

***En anglais : PDFAddImage***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ajoute une image dans une page d'un fichier PDF existant.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonPDF est un pdfDocument
...
MonImage est une Image = fRepExe() + [fSep()] + "Signature.pdf"
PDFAjouteImage(MonPDF.Page[1], MonImage, 10, 10, 150, 50, drHomothétiqueCentré)
PDFSauve(MonPDF, fRepExe() + [fSep()] + "ETAT_SansNom1_Avec_Image.pdf")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`PDFAjouteImage(<Page manipulée> , <Image à ajouter> , <X> , <Y> , <Largeur> , <Hauteur> [, <Mode d'affichage>])`
---

**`<Page manipulée> : (Variable de type pdfPage)`**

Nom de la variable de type [pdfPage](../WDLang6/1000024904.md) correspondant à la page dans laquelle l'image doit être ajoutée.

**`<Image à ajouter> : (Nom de champ, de variable ou chaîne de caractères)`**

Image à ajouter dans la page du fichier PDF. Cette image peut correspondre à : 

- une variable de type [Image](../WDLang1/1000019650.md),

- le nom et chemin de l'image à ajouter dans la page,

- une rubrique de type mémo Image,

- un champ Image. 




**`<X> : (Réel)`**

Coordonnée X (en millimètres) de l'image dans la page.

**`<Y> : (Réel)`**

Coordonnée Y (en millimètres) de l'image dans la page.

**`<Largeur> : (Réel)`**

Largeur de l'image (en millimètres).

**`<Hauteur> : (Réel)`**

Hauteur de l'image (en millimètres).

**`<Mode d'affichage> : (Constante optionnelle de type Entier ou combinaison de constantes)`**

Mode d'affichage de l'image : 


|   |   |
| --- | --- |
| *drEtiré*<br>(Valeur par défaut) | L'image est étirée si nécessaire pour correspondre à la largeur et à la hauteur spécifiées. |
| *drHauteQualité* | Cette constante peut être combinée avec les options de redimensionnement de type homothétique.<br><br>En mode homothétique, cette constante permet d'améliorer la qualité de l'image lorsque celle-ci est réduite par rapport à sa taille initiale. |
| *drHomothétique* | L'image est étirée ou réduite pour occuper de manière optimale l'espace défini sans être déformée. Elle occupe :<br><br>- soit toute la hauteur spécifiée.<br><br>- soit toute la largeur spécifiée.<br><br><br> |
| *drHomothétiqueCentré* | L'image est étirée ou réduite pour occuper de manière optimale l'espace défini sans être déformée et elle est centrée. Elle occupe :<br><br>- soit toute la hauteur spécifiée : l'image est centrée horizontalement.<br><br>- soit toute la largeur spécifiée : l'image est centrée verticalement.<br><br><br> |
| *drHomothétiqueEtendu* | L'image est étirée ou réduite pour que le plus petit côté de l'image soit affiché entièrement. |
| *drRépété* | L'image est répétée pour occuper toute la nouvelle largeur et toute la nouvelle hauteur spécifiées. |

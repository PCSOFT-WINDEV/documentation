
## gpwRecupDroits (Fonction)

***En anglais : gpwGetRights***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de récupérer les droits d'un champ, d'une fenêtre, d'une page ou d'un état.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
nEtat est un entier = gpwRecupDroits("MaFenêtre", "MonChamp")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = gpwRecupDroits(<Elément parent> [, <Nom du champ> [, <Utilisateur> [, <Groupe>]]])`
---

**`<Résultat> : (Constante de type Entier)`**

Type de l'élément : 


|   |   |
| --- | --- |
| *gpwEtatDéfaut* | Le champ est dans l'état "Par défaut". |
| *gpwEtatGrisé* | Le champ est grisé. |
| *gpwEtatInactif* | Le champ est inactif. |
| *gpwEtatInvisible* | Le champ est invisible. |





**`<Elément parent> : (Chaîne de caractères)`**

Nom de la fenêtre, de la page ou de l'état contenant le champ. 

**`<Nom du champ> : (Chaîne de caractères optionnelle)`**

Nom du champ à manipuler. 

**`<Utilisateur> : (Chaîne de caractères optionnelle)`**

Nom de l'utilisateur pour lequel les droits sont recherchés. 

**`<Groupe> : (Booléen optionnel)`**



- <u><u><u><u>Vrai</u></u></u></u> pour rechercher les droits du groupe, 

- <u><u><u><u>Faux</u></u></u></u> pour rechercher les droits d'un utilisateur.

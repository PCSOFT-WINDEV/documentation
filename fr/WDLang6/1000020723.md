


## Fonctions de gestion des tuiles
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


Les fonctions de gestion des tuiles des applications Windows Store apps sont les suivantes : 



|   |   |
| --- | --- |
| [TuileAfficheBadge](../WDLang6/1000020721.md) | Affiche un badge (notification numérique ou image) sur la tuile d'une application Universal Windows 10 App. |
| [TuileAjoute](../WDLang6/1000020719.md) | Ajoute une nouvelle personnalisation de tuile pour l'application Windows Store Apps en cours. |
| [TuileSupprimeTout](../WDLang6/1000020720.md) | Supprime toutes les modifications faites à la tuile de l'application. |




## gpwOuvreConfiguration (Fonction)

***En anglais : gpwOpenConfiguration***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ouvre la fenêtre ou la page de configuration du groupware utilisateur.

Cette fonction est prise en compte uniquement si l'utilisateur en cours est un **superviseur** et s'il est connecté au groupware utilisateur (grâce à la fonction [gpwLogin](../WDLang6/3041001.md) ou depuis la page ouverte par la fonction [gpwOuvre](../WDLang6/3041002.md)).

**Remarque** : Cette fonction est disponible quel que soit le mode d'intégration du groupware utilisateur dans le site (option "Groupware utilisateur automatique" ou "Groupware utilisateur personnalisé" dans l'onglet "Intégration" de la fenêtre de description du groupware utilisateur).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ouverture de la page de configuration du groupware utilisateur 
gpwOuvreConfiguration()
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = gpwOuvreConfiguration()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la page de configuration s'est ouverte, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour connaître le libellé de l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.

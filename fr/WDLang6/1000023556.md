
## gpwDéfaut (Fonction)

***En anglais : gpwDefault***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la valeur correspondant à "DEFAUT" selon le framework utilisé. 

Remarque : Le groupware et les fichiers du groupware étant multilingues, il est ainsi possible de récupérer la valeur de la constante représentant le mot "DEFAUT".


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
sCstDéfaut est une chaîne
sCstDéfaut = gpwDéfaut()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = gpwDéfaut()`
---

**`<Résultat> : (Chaîne de caractères)`**

Valeur du mot "DEFAUT" selon le framework utilisé : 

- "DEFAUT" lorsque le framework est en français,

- "DEFAULT" lorsque le framework est en anglais.

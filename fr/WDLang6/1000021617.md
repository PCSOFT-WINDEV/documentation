
## gpwCrypteMotDePasse (Fonction)

***En anglais : gpwEncryptPassword***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le mot de passe crypté.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI gpwMotsDePasseCryptés() = Vrai ALORS
	GPU_UtilisateurMDP.MotPasseCrypte = gpwCrypteMotDePasse(sMotDePasse)
SINON
	GPU_Utilisateur.MotPasse = sMotDePasse
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = gpwCrypteMotDePasse(<Mot de passe>)`
---

**`<Résultat> : (Chaîne de caractères)`**

Mot de passe crypté. 

**`<Mot de passe> : (Chaîne de caractères)`**

Mot de passe en clair.


## ValideMotDePasse (Fonction)

***En anglais : ValidatePassword***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Indique si le mot de passe spécifié est valide pour l'utilisateur en cours. Le mot de passe peut être :

- le mot de passe d'authentification Windows

- le mot de passe du groupware utilisateur WINDEV/WEBDEV.

- le mot de passe spécifié pour le blocage des applications (mot de passe spécifié avec la fonction [VerrouAppliMotDePasse](../WDLang6/1000017010.md)).




![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png)![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) Cette fonction est disponible uniquement pour vérifier si le mot de passe est valide pour le groupware utilisateur. 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI ValideMotDePasse("MonMotDePasse") = Vrai ALORS
	Info("Utilisateur reconnu")
SINON
	Info("Mauvais mot de passe")
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = ValideMotDePasse(<Mot de passe> [, <Authentification>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le mot de passe est correct, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Mot de passe> : (Chaîne de caractères)`**

Mot de passe à valider pour l'utilisateur en cours.

**`<Authentification> : (Constante de type Entier optionnelle)`**

Méthode d'authentification du mot de passe :



|   |   |
| --- | --- |
| *mdpGpw* | Authentification réalisée par le groupware utilisateur de WINDEV/WEBDEV. |
| *mdpPersonnalisé* | Authentification réalisée par rapport au mot de passe spécifié avec la fonction [VerrouAppliMotDePasse](../WDLang6/1000017010.md).<br><br> |
| *mdpWindows* | Authentification réalisée par rapport au mot de passe Windows.<br> |


Si &lt;Authentification&gt; n'est pas précisé :

- Si la fonction [VerrouAppliMotDePasse](../WDLang6/1000017010.md) a été utilisée, ce mot de passe est vérifié.

- Si le groupware utilisateur est actif, le mot de passe est vérifié par rapport au groupware utilisateur.

- Sinon, le mot de passe est vérifié par rapport au compte Windows.

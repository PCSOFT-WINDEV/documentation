
## Scanneur (Type de variable)

***En anglais : Scanner***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **Scanneur** permet de manipuler les informations provenant d'un scanneur. Les caractéristiques de ce scanneur peuvent être définies et modifiées à l'aide de différentes propriétés WLangage.

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Scan d'une image
MonScanneur est un Scanneur = ScanSélecteur()
IMG_Scan = ScanVersImage(MonScanneur)
```

## Remarques
<a name="NOTE0_1"></a>


### Propriétés spécifiques à la description des variables de type Scanneur
<a name="proprietes_specifiques_description_des_variables_type_scanneur_ELTPARAGRAPHE000029"></a>Les propriétés suivantes peuvent être utilisées pour manipuler les caractéristiques d'un scanneur :

| Nom de la propriété | Type manipulé | Effet |
| --- | --- | --- |
| BitParPixel | Entier | <br><br>- Nombre de bits par pixel, <br><br>- Constante *scanBitsParPixelDéfaut* (valeur par défaut) si cette propriété ne doit pas être modifiée lors du prochain scan. La valeur précédente de la propriété sera conservée. <br><br><br> |
| Brillance | Entier | <br><br>- Brillance, comprise entre -1000 et 1000,<br><br>- Constante *scanBrillanceDéfaut* (valeur par défaut) si cette propriété ne doit pas être modifiée lors du prochain scan. La valeur précédente de la propriété sera conservée. <br><br><br> |
| Contraste | Entier | <br><br>- Contraste, compris entre -1000 et 1000, <br><br>- Constante *scanContrasteDéfaut* (valeur par défaut) si cette propriété ne doit pas être modifiée lors du prochain scan. La valeur précédente de la propriété sera conservée.<br><br><br> |
| FormatCouleur | Constante de type Entier | Format de couleur d'image :<br><br>- *scanFormatCouleurAuto* : Format de couleur détecté automatiquement par le périphérique (valeur par défaut).<br><br>- *scanFormatCouleurRVB* : Image RVB (24 bits par pixel).<br><br>- *scanFormatCouleurNiveauGris* : Image en niveau de gris.<br><br>- *scanFormatCouleurNoirBlanc* : Image en noir et blanc.<br><br><br> |
| Hauteur | Entier | <br><br>- Hauteur maximale (en millièmes de millimètre), <br><br>- Constante *scanHauteurDéfaut* (valeur par défaut).<br><br><br><br><br>**Cette propriété est disponible en lecture seulement**. |
| Largeur | Entier | <br><br>- Largeur maximale (en millièmes de millimètre), <br><br>- Constante *scanLargeurDéfaut* (valeur par défaut).<br><br><br><br><br>**Cette propriété est disponible en lecture seulement**. |
| Nom | Chaîne de caractères | <br><br>- Nom du scanneur, <br><br>- Chaîne vide si la variable n'a pas été initialisée (valeur par défaut).<br><br><br><br><br>**Cette propriété est disponible en lecture seulement**. |
| ProcédureAvancement | Procédure | <br><br>- Nom de la procédure WLangage appelée lors de l'avancement du scan d'une page, <br><br>- Chaîne vide pour ne pas utiliser de procédure (valeur par défaut). <br><br><br><br><br>Cette procédure est de la forme : <br><br><pre><code><Nom de la procédure>(<Numéro de page>, <br>	<Pourcentage d'avancement>)</code></pre><br>où : <br><br>- &lt;Numéro de page&gt; : Numéro de la page scannée.<br><br>- &lt;Pourcentage d'avancement&gt;	: Pourcentage d'avancement du scan de la page (compris entre 0 et 100)<br><br><br>Si la procédure renvoie <u><u><u><u>Faux</u></u></u></u>, le scan est interrompu. |
| RectoVerso | Constante de type Entier | Gestion du recto-verso : <br><br>- *scanRectoVersoDéfaut* (valeur par défaut) si cette propriété ne doit pas être modifiée lors du prochain scan. La valeur précédente de la propriété sera conservée.<br><br>- *scanRectoVersoOui* : le scan réalisé est un scan recto-verso.<br><br>- *scanRectoVersoNon* : le scan réalisé est un scan recto uniquement. <br><br><br> |
| Résolution | Entier | <br><br>- Résolution de l'image en pixels par pouce, <br><br>- Constante *scanRésolutionDéfaut* (valeur par défaut) si cette propriété ne doit pas être modifiée lors du prochain scan. La valeur précédente de la propriété sera conservée.<br><br><br> |
| ScanDepuis | Constante de type Entier | Indique la provenance du scan : <br><br>- *scanDepuisAuto* : le scan sera effectué depuis le chargeur s'il y a un chargeur et des pages dans le chargeur sinon le scan sera effectué depuis le plateau (valeur par défaut). <br><br>- *scanDepuisChargeur* : le scan est effectué depuis le chargeur. <br><br>- *scanDepuisPlateau* : le scan est effectué depuis le plateau. <br><br><br> |
| Zone | [Rectangle](../Motscles/1514079.md) | Zone à scanner (en millièmes de millimètre). Par défaut, un rectangle avec des coordonnées et des dimensions nulles (0,0,0,0) est utilisé pour scanner toute la page.<br><br>**Remarque** : Cette zone est prise en compte par la fonction ScanPropriété si la constante **scanFormatPapierPerso** est utilisée. |

### Fonctions WLangage
<a name="fonctions_wlangage_ELTPARAGRAPHE000159"></a>Le type Scanneur est utilisé par les fonctions WLangage suivantes : 



|   |   |
| --- | --- |
| [ScanAvecInterfaceUtilisateur](../WDLang6/1410087840.md) | Enregistre un document provenant d'un scanneur dans un ou plusieurs fichiers, en utilisant l'interface utilisateur du scanneur. |
| [ScanPropriété](../WDLang6/1410087853.md) | Permet de connaître ou de modifier les propriétés d'un scanneur. |
| [ScanRécupère](../WDLang6/1410087836.md) | Récupère les caractéristiques d'un scanneur dans une variable de type [Scanneur](../WDLang6/1410087808.md). |
| [ScanSélecteur](../WDLang6/1410087835.md) | Affiche une boîte de dialogue standard listant les scanneurs disponibles. |
| [ScanVersImage](../WDLang6/1410087838.md) | Récupère un document provenant d'un scanneur dans une variable de type [Image](../WDLang1/1000019650.md). |
| [ScanVersPDF](../WDLang6/1410087839.md) | Enregistre un document provenant d'un scanneur dans un fichier PDF. |

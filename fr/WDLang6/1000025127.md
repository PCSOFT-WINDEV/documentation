
## PDFFerme (Fonction)

***En anglais : PDFClose***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ferme le fichier PDF chargé en mémoire dans une variable de type [pdfDocument](../WDLang6/1000024909.md). 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonPDF est un pdfDocument = "test.pdf"

POUR TOUTE PageDocument de MonPDF.Page
	Trace(PageDocument.Hauteur)
FIN

PDFFerme(MonPDF)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`PDFFerme(<PDF à fermer>)`
---

**`<PDF à fermer> : (Variable de type pdfDocument)`**

Nom de la variable de type [pdfDocument](../WDLang6/1000024909.md) correspondant au fichier PDF à fermer.


## gpwRecupInfoLDAP (Fonction)

***En anglais : gpwGetLDAPInfo***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Récupère des les caractéristiques du serveur LDAP utilisé pour le groupware utilisateur.  
<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = gpwRecupInfoLDAP(<Type d'information>)`
---

**`<Résultat> : (Type de l'information demandée)`**

Information demandée.

![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) Correspond toujours à 0 ou chaîne vide ("").

**`<Type d'information> : (Constante de type Entier)`**

Information à récupérer :


|   |   |
| --- | --- |
| *gpwLDAPDNBase* | DN de base (chaîne de caractères au format "DC=xxx,DC=zzz,..."). |
| *gpwLDAPPort* | Port du serveur LDAP. |
| *gpwLDAPSansFichier* | Mode d'utilisation de l'annuaire LDAP. <u><u><u><u>Vrai</u></u></u></u> si le groupware utilisateur est lancé "sans fichiers de données", en mode authentification uniquement. |
| *gpwLDAPServeur* | Nom du serveur LDAP. |
| *gpwLDAPVersion* | Numéro de version LDAP. |


## GraveurEtat (Fonction)

***En anglais : BurnerStatus***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie l'état actuel du graveur en cours. Ce graveur peut être sélectionné grâce à la fonction [GraveurSélectionne](../WDLang6/3061012.md).

**Remarque** : Pour connaître l'ensemble des graveurs disponibles à partir du poste en cours, utilisez la fonction [GraveurListe](../WDLang6/3061009.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connaître l'état du graveur
ResEtat est un entier
ResEtat = GraveurEtat() 
// État du graveur 
SELON ResEtat
	// Gravure en cours
	CAS graveurGrave 
		Info("Le graveur est en cours de gravure")
	// Graveur inactif
	CAS graveurInactif 
		Info("Le graveur est inactif")
	// Graveur en cours d'utilisation
	CAS graveurUtiliséExclusivement 
		Info("Le graveur est en cours d'utilisation")
	CAS graveurPrépareEffacement
		Info("Effacement en préparation")
	CAS graveurPrépareGravure
		Info("Gravure en préparation")
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = GraveurEtat()`
---

**`<Résultat> : (Constante de type Entier)`**



- État actuel du graveur : 
	Le graveur peut être dans un des états suivants :
	


|   |   |
| --- | --- |
| *graveurGrave* | Graveur en cours de gravure |
| *graveurInactif* | Graveur inactif |
| *graveurPrépareEffacement* | Graveur en cours de préparation pour effacer des fichiers |
| *graveurPrépareGravure* | Graveur en cours de préparation pour graver des fichiers |
| *graveurUtiliséExclusivement* | Graveur en cours d'utilisation par l'application en cours (effacement de données par exemple) ou par une autre application |



- -1 en cas d'erreur. Pour connaître le libellé de l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md) avec la constante *errMessage*.

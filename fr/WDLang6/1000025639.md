
## ocrBlocTexte (Type de variable)

***En anglais : ocrTextBlock***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **ocrBlocTexte** permet de définir et de connaître toutes les caractéristiques avancées d'un bloc de texte lu par l'OCR natif. Les caractéristiques de ce bloc de texte peuvent être définies et modifiées à l'aide de différentes propriétés WLangage. 

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonOCRBlocTexte est un ocrBlocTexte
tabBlocsTexte est un tableau de ocrBlocTexte 
tabBlocsTexte = OCRExtraitBlocTexte(MonImage)
Trace("Fin de l'extraction des blocs")
POUR i = 1 _À_ tabBlocsTexte.Occurrence
	MonOCRBlocTexte = tabBlocsTexte[i]
	soit sChaîne = MonOCRBlocTexte.Texte	
	soit nIndice = MonOCRBlocTexte.IndiceConfiance
	Trace(sChaîne)
	Trace("Indice de confiance : " + nIndice)
	Trace("----------------------------------------")
FIN
```

## Remarques
<a name="NOTE0_1"></a>


### Propriétés spécifiques à la description des variables de type ocrBlocTexte
<a name="proprietes_specifiques_description_des_variables_type_ocrbloctexte_ELTPARAGRAPHE000029"></a>Les propriétés suivantes peuvent être utilisées pour manipuler un bloc de texte lu par l'OCR natif :

| Nom de la propriété | Type manipulé | Effet |
| --- | --- | --- |
| IndiceConfiance | Entier | Indice de confiance. |
| Polygone | Variable de type [Polygone](../WDLang1/1000025624.md) | Polygone du contour du texte. <br><br>Remarque : Pour l'OCR, le polygone doit être un polygone de 4 points. |
| Texte | Chaîne de caractères | Texte contenu dans le polygone. |


## OCRParamètre (Fonction)

***En anglais : OCRConfigure***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Initialise les options générales de l'OCR (Optical Character Recognition) natif. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Création des options
MesOCROptions est une ocrOption

// Ne souhaite que les nombres lors de la lecture d'une image
MesOCROptions.NombreUniquement = Vrai

OCRParamètre(MesOCROptions)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`OCRParamètre(<Options avancées>)`
---

**`<Options avancées> : (Variable de type ocrOption)`**

Nom de la variable de type [ocrOption](../WDLang6/1000025768.md) contenant les paramètres avancés pour la gestion de l'OCR.

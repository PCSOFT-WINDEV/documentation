
## ConsoleErreur (Fonction)

***En anglais : ConsoleError***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ecrit une chaîne de caractères dans le flux de sortie d'erreurs "stderr".
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
ConsoleErreur("Erreur 404")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`ConsoleErreur(<Message>)`
---

**`<Message> : (Chaîne de caractères)`**

Chaîne à écrire. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Attention : sous Windows, l'exécutable généré doit être en mode console pour pouvoir voir la chaîne écrite.

## Composante :

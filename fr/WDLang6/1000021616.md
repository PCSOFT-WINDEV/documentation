
## gpwMotsDePasseCryptés (Fonction)

***En anglais : gpwEncryptedPasswords***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de savoir si les mots de passe du groupware utilisateur sont cryptés. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI gpwMotsDePasseCryptés() ALORS
	GPU_UtilisateurMDP.MotPasseCrypte = gpwCrypteMotDePasse(sLogin, sMotDePasse)
SINON
	GPU_Utilisateur.MotPasse = sMotDePasse
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = gpwMotsDePasseCryptés()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si les mots de passe du groupware utilisateur sont cryptés, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.

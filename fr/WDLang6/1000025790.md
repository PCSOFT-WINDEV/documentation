
## OCRChargeLangue (Fonction)

***En anglais : OCRLoadLanguage***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ajoute une langue à l'OCR natif. 

Remarques : 

- La fonction **OCRChargeLangue** permet d'ajouter un modèle ".traineddata".

- Sous iOS, utilisez la fonction [OCRAjouteLangue](../WDLang6/1000025728.md) pour ajouter une langue. 



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonNomDeModèle est une chaîne = "eng"
OCRChargeLangue(MonNomDeModèle)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Ajouter une langue

`OCRChargeLangue(<Fichier modèle>)`
---

**`<Fichier modèle> : (Chaîne de caractères)`**

Nom court du modèle à ajouter.


<a name="SYNTAXE2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 
### Ajouter plusieurs langues

`OCRChargeLangue(<Tableau de fichiers modèles>)`
---

**`<Tableau de fichiers modèles> : (Tableau de chaînes de caractères)`**

Tableau contenant les noms courts des modèles à ajouter.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Le modèle contenant la langue doit se trouver dans le répertoire de l'exécutable. 

- Les modèles anglais, français et espagnol sont proposés avec le produit. Le nom court de ces modèles est : "eng", "fra" et "spa". Ils sont sélectionnés en fonction de la langue en cours de l'application.  

- L'extension du fichier modèle doit finir par ".traineddata". Cette extension n'est pas nécessaire lors de l'appel de la fonction : seul le nom court est demandé. 

- D'autres modèles sont disponibles sur le site de TESSERACT :

	- [https://github.com/tesseract-OCR/tessdata_best](https://github.com/tesseract-OCR/tessdata_best)

	- [https://github.com/tesseract-OCR/tessdata](https://github.com/tesseract-OCR/tessdata)

	- [https://github.com/tesseract-OCR/tessdata_fast](https://github.com/tesseract-OCR/tessdata_fast)


 La liste des langues est disponible à l'adresse suivante : [https://github.com/tesseract-ocr/tessdoc/blob/master/Data-Files-in-different-versions.md](https://github.com/tesseract-ocr/tessdoc/blob/master/Data-Files-in-different-versions.md)

- Plusieurs langues peuvent être combinées : 

	- les ajouts doivent se faire les uns après les autres (syntaxe 1).

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) les ajouts peuvent se faire en une seule opération via un tableau (syntaxe 2).


## gpwAucun (Fonction)

***En anglais : gpwNone***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la valeur du mot "AUCUN" selon le framework utilisé. Le mot "AUCUN" est utilisé lorsqu'un utilisateur n'appartient pas à un groupe. 

Remarque : Le groupware et les fichiers du groupware étant multilingues, il est ainsi possible de récupérer la valeur de la constante représentant le mot "AUCUN".


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
sCstAucun est une chaîne
sCstAucun = gpwAucun()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = gpwAucun()`
---

**`<Résultat> : (Chaîne de caractères)`**

Valeur du mot "AUCUN" selon le framework utilisé :   

- "&lt;AUCUN&gt;" lorsque le framework est en français

- "&lt;NONE&gt;" lorsque le framework est en anglais.

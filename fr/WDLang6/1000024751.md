
## gpwOAuthParamètres (Type de variable)

***En anglais : gpwOAuthParameters***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **gpwOAuthParamètres** permet de définir toutes les informations nécessaires pour s'authentifier au groupware utilisateur via une application utilisant le standard OAuth 2.0. Ces caractéristiques peuvent être modifiées à l'aide de différentes propriétés WLangage. 

Ce type de variable doit être passé en paramètre à la fonction [gpwAuthLogin](../WDLang6/1000023597.md). En cas de succès, cette fonction renvoie un Buffer à utiliser avec la fonction [gpwConnecteDistant](../WDLang6/1000024855.md). 

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
tabOAuthParam est un tableau de gpwOAuthParamètres
tabOAuthParam = gpwListeOAuthParamètres()
SI ErreurDétectée ALORS
	Erreur(ErreurInfo())
	RETOUR
FIN

Paramètres est un gpwOAuthParamètres = tabOAuthParam[1]
sCode est une chaîne = gpwAuthLogin(Paramètres)
SI ErreurDétectée ALORS
	Erreur(ErreurInfo())
	RETOUR
FIN

Utilisateur est un gpwUtilisateur
sToken est une chaîne
(sToken, Utilisateur) = gpwConnecteDistant(Paramètres, sCode)
SI ErreurDétectée ALORS
	Erreur(ErreurInfo())
	RETOUR
FIN
```

## Remarques
<a name="NOTE0_1"></a>


### Propriétés spécifiques à la description des variables de type gpwOAuthParamètres
<a name="proprietes_specifiques_description_des_variables_type_gpwoauthparametres_ELTPARAGRAPHE000037"></a>Les propriétés suivantes peuvent être utilisées pour manipuler les informations nécessaires à la demande d'authentification :

| Nom de la propriété | Type manipulé | Effet |
| --- | --- | --- |
| ClientID | Chaîne de caractères | Identifiant du client fourni par le service lors de l'enregistrement de l'application. |
| Identifiant | Entier | Identifiant du provider utilisé. |
| Nom | Chaîne de caractères | Nom du provider utilisé. |
| Type | Constante de type Entier | Type de connexion permettant l'authentification : <br><br>- *gpwFacebook* : Connexion utilisant un compte Facebook.<br><br>- *gpwGoogle* : Connexion utilisant un compte Google.<br><br>- *gpwMicrosoft* : Connexion utilisant un compte Microsoft MSN.<br><br>- *gpwSalesforce* : Connexion utilisant un compte Salesforce.<br><br>- *gpwYahoo* : Connexion utilisant un compte Yahoo.<br><br><br> |
| URLAuth | Chaîne de caractères | URL d'autorisation à utiliser (première URL du standard OAuth 2.0). |


## gpwEnModeLDAP (Fonction)

***En anglais : gpwInLDAPMode***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de savoir si l'identification des utilisateurs dans le groupware utilisateur sera réalisée via un annuaire LDAP.  
<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = gpwEnModeLDAP()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le groupware utilisateur utilise un annuaire LDAP pour l'authentification des utilisateurs, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) &lt;Résultat&gt; est toujours à <u><u><u><u>Faux</u></u></u></u>.

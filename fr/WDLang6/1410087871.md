## Syntaxe
<a name="SYNTAXE1"></a>

### Ajouter un filigrane dans une page d'un document PDF

`PDFAjouteFiligrane(<Page> , <Filigrane>)`
---

**`<Page> : (Variable de type pdfPage)`**

Nom de la variable de type [pdfPage](../WDLang6/1000024904.md) correspondant à la page du fichier PDF dans laquelle le filigrane doit être ajouté.

**`<Filigrane> : (Variable de type Filigrane)`**

Nom de la variable de type [Filigrane](../WDLang5/1000019922.md) correspondant au filigrane à ajouter à la page.


## paNomCompletProjet (Fonction)

***En anglais : apFullProjectName***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Récupère le nom physique complet d'un projet ouvert pendant l'exécution d'un plan d'action. 

**Attention** : Cette fonction peut être exécutée uniquement :

- depuis le code d'un plan d'action.

- depuis une procédure WLangage présente dans un projet ou une configuration de type "Plan d'action".


WINDEV, WEBDEV ou WINDEV Mobile doit avoir été lancé (par une action précédente). 

**Remarque** : Cette fonction est une fonction permettant de manipuler l'éditeur de WINDEV, WEBDEV ou WINDEV Mobile. Elle est donc disponible dans les trois produits.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Inscrit le nom du projet dans le log
paLog(paNomCompletProjet())
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = paNomCompletProjet()`
---

**`<Résultat> : (Chaîne de caractères)`**

Nom complet du projet en cours de traitement.

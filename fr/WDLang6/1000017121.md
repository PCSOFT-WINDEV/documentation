
## SAPDéconnecte (Fonction)

***En anglais : SAPDisconnect***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Déconnecte un système SAP R/3.  
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SAP.Adresse = "MINISAP.test.com"
SAP.NuméroSystème = "0"
SAP.NuméroClient = "000"
SAP.Utilisateur = "BCUSER"
SAP.MotDePasse = "MINISAP"
SAP.Langue = ""
IDSAP est un entier = SAPConnecte()
...
SAPDéconnecte(IDSAP)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`SAPDéconnecte(<Identifiant de la connexion>)`
---

**`<Identifiant de la connexion> : (Entier)`**

Identifiant de la connexion renvoyé par la fonction [SAPConnecte](../WDLang6/1000017120.md).

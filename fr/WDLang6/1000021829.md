
## PlanActionInfo (Fonction)

***En anglais : ActionPlanInfo***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Récupère une information concernant l'exécution d'une instance d'un plan d'action.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Teste l'exécution du plan d'action de compilation de mon projet
bCompilOK est un booléen = PlanActionInfo("CoordFab", "admin", ...
	"clefab", nIdPlanCompilMonProjet, paEtatTerminéOK)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = PlanActionInfo(<Coordinateur> , <Utilisateur> , <Mot de passe> , <Identifiant de l'instance du plan d'action> [, <Information>])`
---

**`<Résultat> : (Type de l'information demandée)`**

Information désirée. 

**`<Coordinateur> : (Chaîne de caractères)`**

Nom de la machine de coordination d'exécution des plans d'action.

**`<Utilisateur> : (Chaîne de caractères)`**

Nom de l'utilisateur. Cet utilisateur doit être autorisé à exécuter un plan d'action sur le coordinateur. 

**`<Mot de passe> : (Chaîne de caractères)`**

Mot de passe de l'utilisateur. 

**`<Identifiant de l'instance du plan d'action> : (Entier)`**

Identifiant de l'instance du plan d'action à manipuler. Cet identifiant est renvoyé par la fonction [PlanActionExécute](../WDLang6/1000021332.md).

**`<Information> : (Constante optionnelle de type Entier)`**

Information désirée sur le plan d'action. 



|   |   |
| --- | --- |
| *paInfoAnnulationEnCours* | &lt;Résultat&gt; est un booléen et vaut : <br><br>- <u><u><u><u>Vrai</u></u></u></u> si l'instance du plan d'action est en cours d'annulation, <br><br>- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. <br><br><br> |
| *paInfoAvancement* | &lt;Résultat&gt; est un entier correspondant au pourcentage d'avancement de l'exécution du plan d'action. |
| *paInfoDébut* | &lt;Résultat&gt; est une variable de type DateHeure correspondant à la date et à l'heure du début de l'exécution du plan d'action. |
| *paInfoEtat*<br>(Valeur par défaut) | &lt;Résultat&gt; est une constante de type Entier correspondant à l'état du plan d'action : <br><br>- *paEtatDéconnecté* : L'automate d'exécution du plan d'action ne répond plus.<br><br>- *paEtatEnAttente* : Le plan d'action est en attente d'exécution.<br><br>- *paEtatEnCours* : Le plan d'action en cours d'exécution.<br><br>- *paEtatLancement* : Le coordinateur va lancer le plan d'action.<br><br>- *paEtatTerminéEchec* : L'exécution du plan s'est terminé avec erreur.<br><br>- *paEtatTerminéOK* : L'exécution du plan s'est terminé sans erreur.<br><br><br> |
| *paInfoFin* | &lt;Résultat&gt; est une variable de type DateHeure correspondant à la date et à l'heure de la fin de l'exécution du plan d'action. |
| *paInfoJournal* | &lt;Résultat&gt; est une chaîne de caractères contenant le journal d'exécution du plan d'action. |
| *paInfoMachine* | &lt;Résultat&gt; est une chaîne de caractères contenant le nom de la machine d'exécution du plan d'action. |
| *paInfoUtilisateur* | &lt;Résultat&gt; est une chaîne de caractères contenant le nom de l'utilisateur ayant lancé l'exécution du plan d'action. |


## paLog (Fonction)

***En anglais : apLog***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Enregistre une information dans un fichier "log" pendant l'exécution d'un plan d'action. Ce fichier "log" a pour nom "&lt;Nom du plan d'action&gt;.log" et est enregistré dans le répertoire du plan d'action. Le contenu de ce fichier correspond au compte-rendu d'exécution du plan d'action. 

**Attention** : Cette fonction peut être exécutée uniquement : 

- depuis le code d'un plan d'action. 

- depuis une procédure WLangage présente dans un projet ou une configuration de type "Plan d'action". 




**Remarque** : Cette fonction est une fonction permettant de manipuler l'éditeur de WINDEV, WEBDEV ou WINDEV Mobile. Elle est donc disponible dans les trois produits.

<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Désactivation de l'action "AuditStatique"
SI paramAudit = Faux ALORS
	paDésactiveAction(AuditStatique)
	paLog("** Audit statique désactivé ***")
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`paLog(<Information>)`
---

**`<Information> : (Chaîne de caractères)`**

Information à enregistrer dans le fichier "log" du plan d'action.


## TuileAjoute (Fonction)

***En anglais : TileAdd***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ajoute une nouvelle personnalisation de tuile pour l'application Universal Windows 10 App en cours. 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MaTuile est une Tuile
MaTuile.Type = tuileLarge
MaTuile.Modèle = tuileModèleLargeImageHaut
Ajoute(MaTuile.Images, "photo.jpg")
MaTuile.Texte = "Important"
TuileSupprimeTout()
TuileAjoute(MaTuile)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Ajouter une tuile définie par une variable de type Tuile

`TuileAjoute(<Tuile>)`
---

**`<Tuile> : (Variable de type Tuile)`**

Nom de la variable de type [Tuile](../WDLang6/1000020713.md) contenant la description de la tuile à ajouter. 


<a name="SYNTAXE2"></a>

### Ajouter une tuile définie par un fichier XML

`TuileAjoute(<URI> , <Intervalle>)`
---

**`<URI> : (Chaîne de caractères)`**

Adresse URI pointant vers le contenu XML de la tuile à ajouter. 

**`<Intervalle> : (Constante de type Entier)`**

Intervalle de temps entre les mises à jour de la tuile : 


|   |   |
| --- | --- |
| *tuileTousLesJours* | La tuile est mise à jour une fois par jour. |
| *tuileToutesLesDemiHeures* | La tuile est mise à jour toutes les 30 minutes. |
| *tuileToutesLesDouzeHeures* | La tuile est mise à jour toutes les 12 heures. |
| *tuileToutesLesHeures* | La tuile est mise à jour toutes les heures. |
| *tuileToutesLesSixHeures* | La tuile est mise à jour toutes les 6 heures. |







<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Tuile ajoutée à partir du type Tuile
<a name="tuile_ajoutee_partir_type_tuile_ELTPARAGRAPHE000073"></a>La tuile associée à l'application est personnalisée. Il est ainsi possible d'ajouter jusqu'à 5 personnalisations différentes. Ces personnalisations peuvent être enlevées à l'aide de la fonction [TuileSupprimeTout](../WDLang6/1000020720.md).
<a name="NOTE0_2"></a>


### Tuile ajoutée à partir d'une URI
<a name="tuile_ajoutee_partir_une_uri_ELTPARAGRAPHE000083"></a>Une tuile ajoutée à partir d'une URI permet de créer une tuile avec mise à jour, même si l'application est fermée. Dans ce cas, la tuile ira, à chaque fin de période, chercher le contenu du fichier XML présent dans URI pour se mettre à jour.

Pour plus d'informations sur le format XML à utiliser, consultez la page [http://msdn.microsoft.com/en-us/library/windows/apps/hh779724.aspx](http://msdn.microsoft.com/en-us/library/windows/apps/hh779724.aspx).

Remarque : La propriété **XML** du type [Tuile](../WDLang6/1000020713.md) permet de récupérer un code XML valide. 

Une tuile périodique peut être supprimée à l'aide de la fonction [TuileSupprimeTout](../WDLang6/1000020720.md).
<a name="NOTE0_3"></a>


### Cas particulier
<a name="cas_particulier_ELTPARAGRAPHE000105"></a>Le premier ajout d'une tuile dans une application supprimera les personnalisations existantes s'il y en a.




## &lt;Variable Scanneur&gt;.AvecInterfaceUtilisateur (Fonction)

***En anglais : &lt;Scanner variable&gt;.WithUserInterface***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Enregistre un document provenant d'un scanneur dans un ou plusieurs fichiers, en utilisant l'interface utilisateur du scanneur.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonScanneur est un Scanneur
// Scan avec interface utilisateur
tabFichier est tableau de chaînes = MonScanneur.AvecInterfaceUtilisateur()
// Affichage des chemins des fichiers scannés
POUR TOUT NomFichier DE tabFichier
	Trace(NomFichier)
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Scanneur>.AvecInterfaceUtilisateur([<Chemin du fichier>] [, <Options>])`
---

**`<Résultat> : (Tableau)`**

Tableau de chaînes de caractères contenant les chemins des fichiers scannés. 
En cas d'erreur, la variable [ErreurDétectée](../WDLang1/3087001.md) vaut <u><u><u><u>Vrai</u></u></u></u> et le message d'erreur correspondant peut être connu avec la fonction [ErreurInfo](../WDLang1/3013008.md).

**`<Scanneur> : (Variable de type Scanneur)`**

Nom de la variable de type [Scanneur](../WDLang6/1410087808.md) initialisée avec les informations du scanneur utilisé. Seul le nom du scanneur sera pris en compte. Les autres paramètres du scan seront déterminés uniquement par les options choisies par l'utilisateur dans l'interface utilisateur du scanneur.

**`<Chemin du fichier> : (Chaîne de caractères optionnelle)`**

Nom et chemin complet (ou relatif) du fichier (260 caractères maximum). L'extension du fichier est inutile et sera ignorée car elle dépendra du type de fichier choisi par l'utilisateur. Un chemin de type UNC peut être utilisé.
Le nom de chaque fichier scanné sera indicé de manière à ne pas écraser un fichier déjà existant.
Si ce chemin n'est pas précisé, les fichiers seront générés dans le répertoire courant avec "scan" comme préfixe (un fichier "scan.bmp" sera par exemple généré si l'utilisateur a choisi de scanner dans un fichier au format bmp).

**`<Options> : (Combinaison de constantes optionnelle de type Entier)`**

Options de scan possibles en utilisant l'interface utilisateur :


|   |   |
| --- | --- |
| *scanUIOptionDéfaut*<br> (Valeur par défaut) | Utilise les options par défaut. |
| *scanUIOptionSystème* | Affiche l'interface utilisateur système, si elle est disponible, plutôt que l'interface utilisateur spécifique au scanneur. |
| *scanUIOptionUneSeuleImage* | Restreint l'interface utilisateur pour proposer uniquement le scan d'un seul fichier.<br><br>Remarque : Certains scanneurs ne gèrent pas cette option. |

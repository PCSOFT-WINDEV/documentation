
## ScanRécupère (Fonction)

***En anglais : ScanGet***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Récupère les caractéristiques d'un scanneur dans une variable de type [Scanneur](../WDLang6/1410087808.md). Cette variable pourra ensuite être utilisée pour effectuer des opérations de scan.

Remarque : Pour connaître l'ensemble des scanneurs connectés au poste en cours, utilisez la fonction [ScanListe](../WDLang6/1410087807.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Récupérer les caractéristiques d'un scanneur
MonScanneur est un Scanneur 
MonScanneur = ScanRécupère("hp scanjet 5590")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = ScanRécupère(<Nom du scanneur>)`
---

**`<Résultat> : (Variable de type Scanneur)`**

Variable de type [Scanneur](../WDLang6/1410087808.md) initialisée avec les informations du scanneur &lt;Nom du scanneur&gt;. 

En cas d'erreur, la variable [ErreurDétectée](../WDLang1/3087001.md) vaut <u><u><u><u>Vrai</u></u></u></u> et le message d'erreur correspondant peut être connu avec la fonction [ErreurInfo](../WDLang1/3013008.md).

**`<Nom du scanneur> : (Chaîne de caractères)`**

Nom du scanneur dont les caractéristiques doivent être récupérées. Ce nom doit correspondre à un des noms renvoyés par la fonction [ScanListe](../WDLang6/1410087807.md).

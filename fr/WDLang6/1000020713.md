
## Tuile (Type de variable)

***En anglais : Tile***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **Tuile** permet de définir toutes les caractéristiques avancées d'une tuile pour une application en mode Universal Windows 10 App. Les caractéristiques de cette tuile peuvent être définies et modifiées à l'aide de différentes propriétés WLangage. 

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MaTuile est une Tuile
MaTuile.Type = tuileLarge
MaTuile.Modèle = tuileModèleLargeImageHaut
Ajoute(MaTuile.Images, "photo.jpg")
MaTuile.Texte = "Important"
TuileSupprimeTout()
TuileAjoute(MaTuile)
```

## Remarques
<a name="NOTE0_1"></a>


### Propriétés spécifiques à la description des variables de type Tuile
<a name="proprietes_specifiques_description_des_variables_type_tuile_ELTPARAGRAPHE000038"></a>Les propriétés suivantes peuvent être utilisées pour manipuler une tuile :

| **Nom de la propriété** | **Type manipulé** | **Effet** |
| --- | --- | --- |
| Images | Tableau de chaînes de caractères | Liste des chemins d'accès à l'image (ou aux images) affichée dans la tuile. <br><br>Les chemins des images peuvent être :<br><br>- Des URL de type "http://" pointant sur une image. <br><br>- Un chemin relatif par rapport au bundle de l'application.<br><br>- Un chemin relatif par rapport au répertoire "appdata" de l'application. <br><br><br>Les images sont recherchées dans le bundle puis dans le répertoire "appdata".<br><br>**Attention** : Les images doivent faire moins de 1024\*1024 pixels et moins de 200 Ko (limitation de Microsoft, pour plus de détails, consultez [http://msdn.microsoft.com/en-us/library/windows/apps/hh465439.aspx](http://msdn.microsoft.com/en-us/library/windows/apps/hh465439.aspx))<br><br>Remarques : <br><br>- Si l'URL spécifiée n'est pas correcte ou si l'image ne respecte pas les limitations, la tuile ne sera pas affichée.<br><br>- Si aucune image n'est spécifiée, seul le texte (et éventuellement l'entête) sont affichés.<br><br><br> |
| Modèle | Constante de type Entier | Modèle prédéfini utilisé pour afficher la tuile. Les modèles disponibles sont les suivants : <br><br>- *tuileModèleBloc* : La tuile est affichée sous forme de bloc (mode carré et mode large) : ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=tuile_modeleBloc.gif)<br><br><br>- *tuileModèleDéfilant* : <br>	La tuile "carrée" est composée d'une image et d'un texte. Les lignes de texte défilent (4 lignes maximum). <br>	La tuile "Large" est composée d'une grande image et de 4 petites images. Les lignes de texte défilent (4 lignes maximum). ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=tuile_modeledefilant.gif)<br><br><br>- *tuileModèleLargeDéfilantImageCôté* : Spécifique aux tuiles larges. La tuile "Large" est composée d'une grande image et d'une petite image positionnée à côté du texte. Les lignes de texte défilent (4 lignes maximum). ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=tuile_modeleLargedefilantim.gif)<br><br><br>- *tuileModèleLargeImageCôté* : Spécifique aux tuiles larges. La tuile "Large" est composée d'une petite image positionnée à côté du texte. Les lignes de texte ne défilent pas.![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=tuile_modeleLargeimgcote.gif)<br><br><br>- *tuileModeleLargeImageHaut* : Spécifique aux tuiles larges. La tuile "Large" est composée d'une grande image positionnée au-dessus du texte. Les lignes de texte ne défilent pas.![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=tuile_modeleLargeimghaut.gif)<br><br><br>- *tuileModeleTexte* : La tuile contient uniquement du texte (mode carré et mode large) : ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=tuile_modeleTexte.gif)<br><br><br><br> |
| RetourALaLigne | Booléen | <br><br>- <u><u><u><u>Vrai</u></u></u></u> pour renvoyer automatiquement le texte à la ligne selon la taille de la tuile, <br><br>- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.<br><br><br> |
| Texte | Chaîne de caractères | Texte de la tuile. <br><br>Dans le cas d'une tuile personnalisée (propriété **Type** correspondant à la constante *tuilePersonnalisée*), cette propriété correspond au code XML permettant de définir la tuile. Pour plus de détails, consultez la documentation de Microsoft à l'adresse suivante : [http://msdn.microsoft.com/en-us/library/windows/apps/hh761491.aspx](http://msdn.microsoft.com/en-us/library/windows/apps/hh761491.aspx)<br><br>Remarque : Si le titre et le texte sont vides, seule l'image est affichée. |

| Type | Constante de type Entier | Type de la tuile. <a name="NOTE0_2"></a>


### Affichage du texte et/ou des images
<a name="affichage_texte_etou_des_images_ELTPARAGRAPHE000146"></a>

- Si la propriété **Image** est vide, seul le texte (et éventuellement le titre) sont affichés. 

- A l'inverse, si le titre et le texte sont vides, seule l'image est affichée.



<a name="NOTE0_3"></a>


### Tuiles carrées et rectangulaires
<a name="tuiles_carrees_rectangulaires_ELTPARAGRAPHE000154"></a>Les tuiles peuvent être à la fois carrées et larges. L'utilisateur choisit le mode d'affichage qu'il souhaite. Pour obtenir un affichage correcte de la tuile dans les deux modes, il est nécessaire de gérer ces deux modes. 

Il est alors nécessaire d'utiliser la fonction TuileAjoute pour ajouter les deux modes d'affichage de la tuile. 

Par exemple : 

```wl
// Tuile en mode rectangulaire
MaTuile est une Tuile
MaTuile.Type = tuileLarge
MaTuile.Modèle = tuileModèleLargeImageHaut
Ajoute(MaTuile.Images, "photo.jpg")
MaTuile.Texte= "Mon joli texte"
TuileAjoute(MaTuile)

// Tuile en mode carré
MaTuile.Type = tuileCarrée
MaTuile.Modèle = tuileModèleDéfilant
TuileAjoute(MaTuile)
```

<a name="NOTE0_4"></a>


### Notes
<a name="notes_ELTPARAGRAPHE000166"></a>

- Il est possible de faire défiler jusqu'à 5 tuiles différentes dans une file. 

- Les tuiles peuvent être mises à jour même lorsque l'application est fermée en spécifiant une URL dans la description XML de la tuile. 

- Il est possible d'ajouter des badges en bas de la tuile avec la fonction [TuileAfficheBadge](../WDLang6/1000020721.md).

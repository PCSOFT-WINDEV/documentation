
## Fonctions de gestion de l'OCR (Optical Character Recognition) intégré
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions de gestion de l'OCR intégré à WINDEV, WEBDEV et WINDEV Mobile sont les suivantes :



|   |   |
| --- | --- |
| [OCRAjouteLangue](../WDLang6/1000025728.md) | Ajoute une langue de détection à l'OCR natif. |
| [OCRChargeLangue](../WDLang6/1000025790.md) | Ajoute une langue à l'OCR natif. |
| [OCRDétecteZoneTexte](../WDLang6/1000025704.md) | Renvoie toutes les zones de texte d'une image sans lire le texte. |
| [OCRExtraitBlocTexte](../WDLang6/1000025703.md) | Renvoie tous les blocs texte d'une image. |
| [OCRExtraitTexte](../WDLang6/1000025699.md) | Lit le texte contenu dans une image. |
| [OCRParamètre](../WDLang6/1000025701.md) | Initialise les options générales de l'OCR (Optical Character Recognition) natif. |
| [OCRRépertoireModèle](../WDLang6/1000025700.md) | Permet de définir le répertoire contenant les modèles de langue pour l'OCR natif. |
| [OCRSupprimeToutesLangues](../WDLang6/1000025729.md) | Supprime toutes les langues de l'OCR natif. |





- ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=LesfonctionsOCR.gif) ***Exemples unitaires (WINDEV)*** : **Les fonctions OCR** <br>Cet exemple montre l'utilisation des fonctions OCR de WINDEV.<br>Ces fonctions permettent d'analyser une image afin d'en extraire le texte.<br>Il est possible de spécifier la langue du document analysé afin de détecter plus finement.<br>Il est également possible de récupérer les positions des blocs de texte trouvés.

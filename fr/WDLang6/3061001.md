
## GraveurAjouteFichier (Fonction)

***En anglais : BurnerAddFile***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ajoute un fichier à la compilation. Lorsque tous les fichiers voulus seront ajoutés à la compilation, la gravure du CD/DVD pourra être réalisée (fonction [GraveurEnregistre](../WDLang6/3061006.md)).

**Remarque** : Pour ajouter l'ensemble des fichiers d'un répertoire à la compilation, utilisez la fonction [GraveurAjouteRépertoire](../WDLang6/3061002.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple
<a class="notetitle" target="_blank" href="$DOC$=1000003061001&name=graveurajoutefichier_fonction&product=WD">Voir des exemples supplémentaires</a>

```wl
// Gravure d'un CD de type données 
GraveurTypeMédia(MédiaType_CD_ROM_XA) 
// Ajout de fichiers 
ResAjout est un booléen 
ResAjout = GraveurAjouteFichier("C:\MonRépertoire\MonFichierA.TXT", "\MesFichiers\FicA.TXT") 
ResAjout = GraveurAjouteFichier("C:\MonRépertoire\MonFichierB.TXT", "\MesFichiers\FicB.TXT") 
// Pour visualiser cet exemple en totalité, cliquez sur le lien "Exemple"
```


<a name="Exemple2"></a>



```wl
// Gravure d'un CD de type audio
GraveurTypeMédia(MédiaType_CDDA_CDROM) 
// Ajout de fichiers 
ResAjout est un booléen 
ResAjout = GraveurAjouteFichier("C:\MesMusiques\MaChansonA.WAV") 
ResAjout = GraveurAjouteFichier("C:\MesMusiques\MaChansonB.WAV") 
// Pour visualiser cet exemple en totalité, cliquez sur le lien "Exemple"
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = GraveurAjouteFichier(<Fichier à ajouter> [, <Fichier sur le CD ou DVD>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le fichier a été ajouté à la compilation, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour connaître le libellé de l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**Remarque** : Il est nécessaire qu'un CD ou DVD soit présent dans le graveur. Ce CD ou DVD doit contenir assez d'espace pour graver une nouvelle session.

**`<Fichier à ajouter> : (Chaîne de caractères)`**

Nom et chemin du fichier à ajouter dans la compilation (260 caractères maximum). Ce chemin peut être un chemin complet, relatif ou de type UNC.
Dans le cas d'une compilation de type audio, ce fichier doit correspondre à un fichier ".WAV" (44100 Hz, 16 bits).



**`<Fichier sur le CD ou DVD> : (Chaîne de caractères optionnelle)`**

Nom et chemin du fichier sur le CD ou sur le DVD. Ce chemin doit commencer par le caractère "antislash" ("\\").
Par défaut, le fichier sera ajouté à la racine du CD/DVD.
Dans le cas d'une compilation de type audio, ce paramètre n'est pas pris en compte. En effet, le fichier sera automatiquement ajouté à la racine du CD/DVD.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Compilation
<a name="compilation_ELTPARAGRAPHE000071"></a>Une compilation correspond à l'ensemble des fichiers à graver sur un CD ou un DVD. Cette compilation sera prise en compte lors du prochain appel de la fonction [GraveurEnregistre](../WDLang6/3061006.md).

Une compilation est automatiquement créée lors du premier appel de la fonction **GraveurAjouteFichier** ou de la fonction [GraveurAjouteRépertoire](../WDLang6/3061002.md). Une seule compilation à la fois peut être créée.

Une compilation est automatiquement supprimée :

- soit lors du changement du type du CD/DVD à graver (fonction [GraveurTypeMédia](../WDLang6/3061013.md)).

- soit lors de la sélection du graveur utilisé par défaut (fonction [GraveurSélectionne](../WDLang6/3061012.md)).

- soit après l'appel de la fonction [GraveurEnregistre](../WDLang6/3061006.md) (uniquement si la gravure a été lancée, &lt;Résultat&gt; à <u><u><u><u>Vrai</u></u></u></u>).

- soit lors de l'annulation de la gravure (fonction [GraveurAnnuleGravure](../WDLang6/3061003.md)).

- soit lors de la fermeture de l'application WINDEV ou du site WEBDEV en cours.





<a name="NOTE0_2"></a>




### Ajout de plusieurs fichiers identiques
<a name="ajout_plusieurs_fichiers_identiques_ELTPARAGRAPHE000115"></a>Si le même fichier est ajouté plusieurs fois à la compilation :

- dans le cas d'un CD/DVD de type audio : ce fichier sera présent sur le CD/DVD autant de fois qu'il a été ajouté.

- dans le cas d'un CD/DVD de type données : seul le dernier fichier ajouté (même nom, même répertoire sur le CD/DVD) sera présent sur le CD/DVD.





<a name="NOTE0_3"></a>




### Configuration nécessaire 
<a name="configuration_necessaire_ELTPARAGRAPHE000127"></a>

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) La gravure de CD est disponible uniquement sous Windows XP et supérieur. La gravure de DVD est disponible sous Windows Vista, et sous Windows XP en utilisant la mise à jour KB932716 pour Windows XP (cette mise à jour doit être téléchargée depuis le site de Microsoft et installée car elle n'est pas prise en compte par le mécanisme de Live Update).

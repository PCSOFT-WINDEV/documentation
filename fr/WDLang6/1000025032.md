
## PDFOuvre (Fonction)

***En anglais : PDFOpen***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ouvre un document PDF et charge ce document dans une variable de type [pdfDocument](../WDLang6/1000024909.md). 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonPDF est un pdfDocument = PDFOuvre("test.pdf")

SI ErreurDétectée() ALORS
   Erreur(ErreurInfo())
   RETOUR
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = PDFOuvre(<Source du document> [, <Mot de passe>])`
---

**`<Résultat> : (Variable de type pdfDocument)`**

Variable de type [pdfDocument](../WDLang6/1000024909.md) correspondant au document PDF spécifié. 

En cas d'erreur, la variable [ErreurDétectée](../WDLang1/3087001.md) est à <u><u><u><u>Vrai</u></u></u></u> et la fonction [ErreurInfo](../WDLang1/3013008.md) permet de connaître la cause de l'erreur.

**`<Source du document> : (Type correspondant à la source)`**

Correspond à un des éléments suivants : 

- Chemin complet ou relatif du fichier PDF à ouvrir (260 caractères maximum).

- Nom de la variable de type Buffer contenant le document PDF. 

- Nom de la rubrique mémo contenant le document PDF. Ce paramètre est de la forme : &lt;Fichier de données&gt;.&lt;Rubrique&gt;.




**`<Mot de passe> : (Chaîne de caractères optionnelle)`**

Mot de passe d'ouverture du fichier PDF à manipuler (si le fichier est protégé par un mot de passe d'ouverture).

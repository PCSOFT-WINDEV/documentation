
## GraveurChemin (Fonction)

***En anglais : BurnerPath***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le chemin du graveur en cours. Ce graveur peut être sélectionné grâce à la fonction [GraveurSélectionne](../WDLang6/3061012.md).

**Remarque** : Pour connaître l'ensemble des graveurs disponibles à partir du poste en cours, utilisez la fonction [GraveurListe](../WDLang6/3061009.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connaître les graveurs installés
ListeGraveur est une chaîne
ListeGraveur = GraveurListe()
SI ListeGraveur ~= "" ALORS
	Erreur("Aucun graveur n'a été détecté sur ce poste")
	RETOUR
FIN
// Remplissage de la table TABLE_TableGraveur avec la liste des graveurs
UnGraveur est une chaîne
CheminGraveur est une chaîne

POUR CHAQUE CHAINE UnGraveur DE ListeGraveur SEPAREE PAR RC
	// Sélection du graveur
	SI GraveurSélectionne(ExtraitChaîne(UnGraveur, 1)) = Faux ALORS
		Erreur("Echec de la sélection de graveur", ErreurInfo())
		RETOUR
	FIN
	// Chemin du graveur en cours
	CheminGraveur = GraveurChemin()
	// Ajout dans la table
	TableAjouteLigne(TABLE_TableGraveur, CheminGraveur, UnGraveur)
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = GraveurChemin()`
---

**`<Résultat> : (Chaîne de caractères)`**



- Chemin du graveur en cours. 
	Ce chemin se termine toujours par un "antislash" ("\\").

- Chaîne vide ("") en cas d'erreur. Pour connaître le libellé de l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).


## PDFMotDePasse (Fonction)

***En anglais : PDFPassword***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Protège et crypte un document PDF selon un mot de passe. Un nouveau document PDF crypté avec le mot de passe indiqué est créé. Ce mot de passe sera demandé à l'ouverture du document PDF.  


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI PAS PDFMotDePasse("clair.pdf", "crypte.pdf", iProtectionDéfaut, "motdepasse") ALORS 
	Erreur(ErreurInfo())
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = PDFMotDePasse(<Fichier PDF source> , <Fichier PDF résultat> , <Options> , <Mot de passe> [, <Mot de passe de protection>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la protection a été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> en cas de problème. La fonction [ErreurInfo](../WDLang1/3013008.md) permet de connaitre le détail de l'erreur. 




**`<Fichier PDF source> : (Chaîne de caractères)`**

Nom et chemin complet du fichier PDF à protéger. 
Remarque : Ce fichier PDF ne doit pas être déjà protégé par un mot de passe. 

**`<Fichier PDF résultat> : (Chaîne de caractères)`**

Nom et chemin complet du fichier PDF protégé.

**`<Options> : (Constante ou combinaison de constantes de type Entier)`**

Options de protection disponibles : 



|   |   |
| --- | --- |
| *iProtectionCommentaire* | Protection contre l'ajout ou la modification de commentaires. |
| *iProtectionDéfaut* | Protection par défaut : le fichier pdf est crypté avec le mot de passe sans protection supplémentaire. |
| *iProtectionImpression* | Le fichier PDF ne pourra pas être imprimé. |
| *iProtectionModification* | Le contenu du fichier PDF ne pourra pas être modifié. |
| *iProtectionSélection* | Le contenu du fichier PDF ne pourra pas être copié ou extrait du fichier. |



**`<Mot de passe> : (Chaîne de caractères)`**

Mot de passe utilisé pour l'ouverture du fichier PDF.  Ce mot de passe permet de décrypter le fichier PDF pour le lire en respectant les permissions passées dans le paramètre &lt;Options&gt;.

Ce mot de passe ne peut pas contenir de caractères Unicode. 

**`<Mot de passe de protection> : (Chaîne de caractères optionnelle)`**

Mot de passe utilisé pour permettre la modification des mesures de protection du fichier PDF. Si ce mot de passe est utilisé, les options de protection spécifiées dans le paramètre &lt;Options&gt; ne sont pas prises en compte. 

Ce mot de passe ne peut pas contenir de caractères Unicode.

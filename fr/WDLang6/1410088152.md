
## Fonctions permettant la détection d'objets dans une image (IA)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions permettant de détecter la présence d'objets dans les images sont les suivantes : 



|   |   |
| --- | --- |
| [IADétecte](../WDLang6/1410087942.md) | Détecte différents éléments sur une image : objets, visages, ... Cette détection est réalisée grâce à l'intelligence artificielle (IA). |
| [IADétecteContour](../WDLang6/1410089109.md) | Détecte les contours des formes présentes dans une image. |
| [IADétecteDessin](../WDLang6/1410089108.md) | Classifie les dessins à main levée parmi cercle, carré et triangle. |
| [IAModèleDétecte](../WDLang6/1410087982.md) | Exécute un modèle (de réseau de neurones) entrainé sur une image. Cette fonction permet d'utiliser l'intelligence artificielle pour la détection d'éléments dans une image. |



Ces fonctions utilisent les types de variables suivants : 



|   |   |
| --- | --- |
| [iaModèleRéseauDeNeurones](../WDLang6/1410088095.md) | Le type **iaModèleRéseauDeNeurones** permet de définir toutes les caractéristiques avancées d'un réseau de neurones utilisé par la fonction [IAModèleDétecte](../WDLang6/1410087982.md). |
| [iaRésultatImage](../WDLang6/1410087965.md) | Le type **iaRésultatImage** permet de récupérer le résultat de la fonction [IADétecte](../WDLang6/1410087942.md). |

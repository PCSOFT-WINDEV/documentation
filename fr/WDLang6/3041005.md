
## gpwInitAnalyse (Fonction)

***En anglais : gpwInitAnalysis***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Initialisation de l'analyse du groupware utilisateur.

**Remarque** : Cette fonction concerne le groupware utilisateur utilisé en mode compatible (conservé par compatibilité à partir de la version 18). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Initialisation de l'analyse du groupware utilisateur
SI gpwInitAnalyse() = Faux ALORS
	Erreur(ErreurInfo())
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = gpwInitAnalyse()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'initialisation s'est bien passée, 

- <u><u><u><u>Faux</u></u></u></u> en cas d'erreur. La fonction [ErreurInfo](../WDLang1/3013008.md) permet d'identifier l'erreur.

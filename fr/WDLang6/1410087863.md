


## &lt;Variable Scanneur&gt;.Propriété (Fonction)

***En anglais : &lt;Scanner variable&gt;.Property***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de connaître ou de modifier les propriétés d'un scanneur.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonScanneur est un Scanneur
// Connaître la valeur de niveau de gris 
// à partir de laquelle un pixel sera converti
// en noir ou blanc lors d'un scan monochrome
WIA_IPS_THRESHOLD est un entier = 6159
nSeuil est un entier = MonScanneur.Propriété(WIA_IPS_THRESHOLD)
```



```wl
MonScanneur est un Scanneur
soit CapacitéScanneur = MonScanneur.Propriété(scanPropriétéCapacité) 
SI CapacitéScanneur & scanCapacitéChargeur ALORS
	Info("Le scanneur a un chargeur")
FIN
SI CapacitéScanneur & scanCapacitéPlateau ALORS
	Info("Le scanneur a un plateau")
FIN
SI CapacitéScanneur & scanCapacitéRectoVerso ALORS
	Info("Le scanneur gère le recto-verso")
FIN
```

### Connaître les propriétés d'un scanneur

`<Résultat> = <Scanneur>.Propriété(<Propriété>)`
---

**`<Résultat> : (Type de la propriété)`**

Valeur de la propriété. 

En cas d'erreur, la variable [ErreurDétectée](../WDLang1/3087001.md) vaut <u><u><u><u>Vrai</u></u></u></u> et le message d'erreur correspondant peut être connu avec la fonction [ErreurInfo](../WDLang1/3013008.md).

**`<Scanneur> : (Variable de type Scanneur)`**

Nom de la variable de type [Scanneur](../WDLang6/1410087808.md) initialisée avec les informations du scanneur dont on veut connaître une propriété.

**`<Propriété> : (Entier)`**

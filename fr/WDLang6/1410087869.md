
## Scanner un document (via un driver WIA)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000092"></a>
Windows Image Acquisition (WIA) est un modèle de pilote et une API Windows permettant de numériser des documents notamment depuis un scanneur. Cette norme remplace de plus en plus la norme Twain. 

WINDEV met à votre disposition des fonctions permettant d'effectuer des scans de documents à partir de scanneurs proposant des drivers WIA. 

Les fonctions de récupération d'un document via un scanneur permettent :

- soit de programmer entièrement le scan du document.

- soit de lancer directement l'interface utilisateur du scanneur utilisé (ou l'interface système correspondante). Cette interface permet par exemple d'enregistrer le document, de modifier les paramètres par défaut du périphérique, ... Les différentes actions proposées dépendent du périphérique utilisé.




Remarque : La norme WIA 2 est supportée. 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Principe d'utilisation
<a name="principe_utilisation_ELTTEXTE000116"></a>
Le principe d'utilisation est le suivant : 

1. Définir une variable de type [Scanneur](../WDLang6/1410087808.md). Cette variable contient les caractéristiques du scanneur utilisé. 

2. Récupérer les caractéristiques du scanneur via les fonctions suivantes : 


|   |   |
| --- | --- |
| [ScanListe](../WDLang6/1410087807.md) | Liste les scanneurs connectés au poste en cours. |
| [ScanRécupère](../WDLang6/1410087836.md) | Récupère les caractéristiques d'un scanneur dans une variable de type [Scanneur](../WDLang6/1410087808.md). |
| [ScanSélecteur](../WDLang6/1410087835.md) | Affiche une boîte de dialogue standard listant les scanneurs disponibles. |

3. Scanner le document via les fonctions suivantes : 


|   |   |
| --- | --- |
| [ScanAvecInterfaceUtilisateur](../WDLang6/1410087840.md) | Enregistre un document provenant d'un scanneur dans un ou plusieurs fichiers, en utilisant l'interface utilisateur du scanneur. |
| [ScanVersImage](../WDLang6/1410087838.md) | Récupère un document provenant d'un scanneur dans une variable de type [Image](../WDLang1/1000019650.md). |
| [ScanVersPDF](../WDLang6/1410087839.md) | Enregistre un document provenant d'un scanneur dans un fichier PDF. |




Remarques : 

- Le type [Scanneur](../WDLang6/1410087808.md) est disponible en lecture et en écriture. Il est ainsi possible de modifier un ou plusieurs paramètres entre deux scans différents. Les constantes *scanxxxDéfaut* permettent de conserver la dernière valeur spécifiée pour les différentes propriétés. 

- La fonction [ScanPropriété](../WDLang6/1410087853.md) permet de connaître et de modifier les différentes propriétés du scanneur gérées par le driver (propriétés de l'API Windows).

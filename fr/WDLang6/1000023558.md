
## gpwVisiteur (Fonction)

***En anglais : gpwVisitor***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la valeur correspondant au mot "Visiteur" selon le framework utilisé. Un visiteur est une personne qui n'est pas une personne de type superviseur.

Remarque : Le groupware et les fichiers du groupware étant multilingues, il est ainsi possible de récupérer la valeur de la constante représentant le mot "Visiteur".


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
sCstVisiteur est une chaîne
sCstVisiteur = gpwVisiteur()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = gpwVisiteur()`
---

**`<Résultat> : (Chaîne de caractères)`**

Valeur du mot "Visiteur" selon le framework utilisé : 

- "Visiteur (non connecté)" lorsque le framework est en français,

- "Visitor (not connected)" lorsque le framework est en anglais.

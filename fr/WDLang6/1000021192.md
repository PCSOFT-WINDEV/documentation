


## TwainPropriété (Fonction)

***En anglais : TwainProperty***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de connaître ou de modifier les propriétés du périphérique Twain en cours. Ce périphérique Twain peut être sélectionné grâce à la fonction [TwainSelectSource](../WDLang6/3078013.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connaître la résolution du scanner
nRésolution est un entier = TwainPropriété(TwainRésolution)
```
<a name="Exemple2"></a>

```wl
// Modifier la résolution pour scanner en 200 dpi
bAppliquer est un booléen = TwainPropriété(TwainRésolution, 200)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître les propriétés du périphérique Twain en cours

`<Résultat> = TwainPropriété(<Propriété>)`
---

**`<Résultat> : (Entier)`**



- Valeur de la propriété spécifiée.

- 0 si le périphérique TWAIN ne supporte pas la modification de la propriété désirée ou la valeur de la propriété désirée.




**`<Propriété> : (Constante de type Entier)`**

Propriété dont la valeur est demandée. Cette propriété correspond à une des constantes suivantes : 


|   |   |
| --- | --- |
| *TwainBitsParPixel* | Nombre de bits par pixel. |
| *TwainBrillance* | Brillance du périphérique. <br>&lt;Valeur&gt; doit être compris entre -1000 et 1000. |
| *TwainChargeurActivé* | Mode de gestion du chargeur. &lt;Résultat&gt; vaut : <br><br>- <u><u><u><u>Vrai</u></u></u></u> si le chargeur est activé,<br><br>- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.<br><br><br> |
| *TwainContraste* | Contraste du périphérique Twain. &lt;Résultat&gt; sera compris entre -1000 et 1000. |
| *TwainDétectionBordActivée* | Mode de détection des bords. &lt;Résultat&gt; vaut : <br><br>- <u><u><u><u>Vrai</u></u></u></u> si la détection des bords est activée,<br><br>- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.<br><br><br> |
| *TwainDétectionCodeBarresActivée* | Mode de détection des codes-barres. &lt;Résultat&gt; vaut : <br><br>- <u><u><u><u>Vrai</u></u></u></u> si la détection des codes-barres est activée, <br><br>- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.<br><br><br> |
| *TwainHauteur* | Hauteur maximale (en millimètres). |
| *TwainInterfaceContrôlable* | Mode de gestion de l'interface utilisateur. &lt;Résultat&gt; vaut : <br><br>- <u><u><u><u>Vrai</u></u></u></u> si l'interface utilisateur peut être désactivé,<br><br>- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.<br><br><br> |

| *TwainRotationAutoActivée* | Mode de gestion de la rotation d'image. &lt;Résultat&gt; vaut : <br><br>- <u><u><u><u>Vrai</u></u></u></u> si la rotation automatique d'image est activée, <br><br>- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.<br><br><br> |

| *TwainDétectionBordActivée* | Mode de détection des bords. &lt;Valeur&gt; vaut : <br><br>- <u><u><u><u>Vrai</u></u></u></u> si la détection des bords doit être activée,<br><br>- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.<br><br><br> |

## Remarques


### Propriétés spécifiques
<a name="proprietes_specifiques_ELTPARAGRAPHE000365"></a>Il est possible de récupérer ou de modifier la valeur d'une propriété ne figurant pas parmi les constantes Twain ci-dessus. Dans ce cas, le paramètre &lt;Propriété&gt; correspondra à la valeur de la constante CAP_xxx ou ICAP_xxx de TWAIN.H, associée à la propriété désirée.


### Fonctionnement dans des applications 64 bits
<a name="fonctionnement_dans_des_applications_64_bits_ELTPARAGRAPHE000497"></a>Dans une application 64 bits, cette fonction utilisera automatiquement le driver Twain 64 bits. Si ce driver n'est pas installé, le périphérique TWAIN ne pourra pas être manipulé.


## Fonctions Python
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions Python sont les suivantes : 



|   |   |
| --- | --- |
| [PythonAjouteChemin](../WDLang6/1410088169.md) | Permet d'ajouter le chemin d'un répertoire ou le chemin de l'environnement virtuel pour la recherche des modules Python. |
| [PythonExécute](../WDLang6/1410088102.md) | Exécute une fonction présente dans un module Python. |
| [PythonInitialise](../WDLang6/1410088100.md) | Indique l'emplacement de la DLL Python à charger. |
| [PythonTermine](../WDLang6/1410088101.md) | Décharge la DLL Python et libère la mémoire. |

Vous pouvez installer Python 3 à partir de l’adresse suivante : http://legacy.python.org/download/


## gpwUtilisateur (Type de variable)

***En anglais : gpwUser***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **gpwUtilisateur** permet de décrire et de modifier les caractéristiques d'un utilisateur du groupware utilisateur WEBDEV ou WINDEV. 

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).

## Remarques




### Propriétés spécifiques à la description des variables de type gpwUtilisateur
<a name="proprietes_specifiques_description_des_variables_type_gpwutilisateur_ELTPARAGRAPHE000055"></a>Les propriétés suivantes peuvent être utilisées pour manipuler une variable de type **gpwUtilisateur** :

| Nom | Type manipulé | Effet |
| --- | --- | --- |
| Activé | Booléen | <br><br>- <u><u><u><u>Vrai</u></u></u></u> si l'utilisateur est activé, <br><br>- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.<br><br><br> |
| CléTOTP | Chaine de caractères | Clé TOTP liée à l'utilisateur en cas de double authentification. <br><br>Remarque : Cette propriété est prise en compte uniquement pour le groupware utilisateur en mode WDBaaS et le Store Privé. |
| DoubleAuthentification | Entier | Mode de gestion de la double authentification : <br><br>- 0 : Double authentification inactive. La double authentification n'est pas activée pour l'utilisateur et elle n'est pas obligatoire. <br><br>- 1 : Double authentification active. La double authentification est activée pour l'utilisateur. <br><br>- 2 : Double authentification à activer. La double authentification est obligatoire mais n'a pas été activée. <br><br><br><br><br>Remarque : Cette propriété est prise en compte uniquement pour le groupware utilisateur en mode WDBaaS et le Store Privé. |
| DroitAccès | Constante de type Entier | Droits d'accès de l'utilisateur : <br><br>- *gpwDroitAjoutApplication* : Droit d'ajouter une application dans le Store Privé. <br><br>- *gpwDroitAucun* : Aucun droit d'ajout. <br><br><br><br><br> |
| Email | Chaîne de caractères | Adresse email de l'utilisateur. |
| IDExterne | Chaîne de caractères | Identifiant de l'utilisateur dans une base externe au groupware. Cette propriété permet par exemple de lier l'utilisateur du groupware utilisateur au fichier Utilisateur de l'application utilisant le groupware en référençant son identifiant. <br><br> |
| ListeDeConfiance | Buffer | Si la double authentification est activée, l'utilisateur a la possibilité d'indiquer que la machine en cours est une machine de confiance. Cette propriété liste tous les appareils, ordinateurs et navigateurs de confiance définis par l'utilisateur. <br><br>Remarque : Dans l'interface du profil utilisateur proposé par le groupware utilisateur, il est possible de supprimer des machines de confiance. |
| Login | Chaîne de caractères | Login de l'utilisateur. |
| MotDePasse | Chaîne de caractères | Mot de passe de l'utilisateur.<br><br>Ce mot de passe n'est pas disponible lors de la récupération des informations concernant l'utilisateur courant avec la fonction [gpwRecupInfoUtilisateur](../WDLang6/3041004.md). |
| MotDePasseASaisir | Booléen | <br><br>- <u><u><u><u>Vrai</u></u></u></u> si l'utilisateur doit indiquer son mot de passe lors de la première connexion, <br><br>- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.<br><br><br> |
| Nom | Chaîne de caractères | Nom de l'utilisateur. |
| Prénom | Chaîne de caractères | Prénom de l'utilisateur. |
| Superviseur | Booléen | <br><br>- <u><u><u><u>Vrai</u></u></u></u> si l'utilisateur est superviseur, <br><br>- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.<br><br><br> |
| Téléphone | Chaîne de caractères | Numéro de téléphone de l'utilisateur. |

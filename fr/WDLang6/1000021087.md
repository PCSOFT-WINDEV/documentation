
## gpwRechercheUtilisateur (Fonction)

***En anglais : gpwFindUser***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Récupère les caractéristiques d'un utilisateur du groupware utilisateur.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Recherche de l'utilisateur
clUtilisateur est un gpwUtilisateur = gpwRechercheUtilisateur(gsEmail)
// Utilisateur non trouvé
SI ErreurDétectée ALORS
	RETOUR
FIN

// Modification du mot de passe
clUtilisateur.MotDePasse = SAI_MotPasseNouveau
SI gpwModifieUtilisateur(clUtilisateur) = Faux ALORS
	Info("Impossible de modifier l'utilisateur.")
	RETOUR
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = gpwRechercheUtilisateur(<Identifiant>)`
---

**`<Résultat> : (Variable de type gpwUtilisateur)`**

Nom de la variable de type [gpwUtilisateur](../WDLang6/1000021088.md) contenant les caractéristiques de l'utilisateur recherché. 

**`<Identifiant> : (Chaîne de caractères)`**

Login ou email de l'utilisateur. Si une adresse email est spécifiée, la recherche sera effectuée dans la propriété **Email** de la variable [gpwUtilisateur](../WDLang6/1000021088.md), dans le cas contraire, la recherche sera effectuée dans la propriété **Login** de la variable [gpwUtilisateur](../WDLang6/1000021088.md).




## &lt;Variable Scanneur&gt;.Capacité (Fonction)

***En anglais : &lt;Scanner variable&gt;.Capability***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de savoir si un scanneur possède une capacité donnée.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonScanneur est un Scanneur

// Indique si le scanneur supporte le recto-verso
RectoVerso est un booléen = MonScanneur.Capacité(scanCapacitéRectoVerso)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Scanneur>.Capacité(<Capacité>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le scanneur a la capacité &lt;Capacité&gt;, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. En cas d'erreur, la variable [ErreurDétectée](../WDLang1/3087001.md) vaut <u><u><u><u>Vrai</u></u></u></u> et le message d'erreur correspondant peut être connu avec la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Scanneur> : (Variable de type Scanneur)`**

Nom de la variable de type [Scanneur](../WDLang6/1410087808.md) initialisée avec les informations du scanneur dont on veut connaître la capacité.

**`<Capacité> : (Constante de type Entier)`**

Information recherchée concernant la capacité du scanneur :  



|   |   |
| --- | --- |
| *scanCapacitéChargeur* | Permet de savoir si le scanneur a un chargeur. |
| *scanCapacitéPlateau* | Permet de savoir si le scanneur a un plateau. |
| *scanCapacitéRectoVerso* | Permet de savoir si le scanneur supporte le recto-verso. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Il est également possible de connaître les capacités d'un scanneur avec la fonction [&lt;Variable Scanneur&gt;.Propriété](../WDLang6/1410087863.md). Cette fonction renvoie une combinaison des constantes correspondant aux capacités supportées par le scanneur. La fonction **&lt;Variable Scanneur&gt;.Capacité** simplifie le code permettant de savoir si le scanneur a une capacité donnée.

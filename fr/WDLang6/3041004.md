


## gpwRecupInfoUtilisateur (Fonction)

***En anglais : gpwGetUserInfo***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Récupère des informations sur l'utilisateur en cours (application utilisant le groupware utilisateur).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Récupération du prénom de l'utilisateur
Prénom est une chaîne 
Prénom = gpwRecupInfoUtilisateur(gpwInfoPrénom)

// Récupération de la liste des groupes de l'utilisateur
sListeGroupe est une chaîne = gpwRecupInfoUtilisateur(gpwInfoGroupe)

POUR TOUTE CHAÎNE sGroupe DE sListeGroupe SEPAREE PAR TAB
        Trace(sGroupe)
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = gpwRecupInfoUtilisateur(<Information>)`
---

**`<Résultat> : (Chaîne de caractères, Booléen ou variable de type gpwUtilisateur)`**



- Information demandée.

- Variable de type [gpwUtilisateur](../WDLang6/1000021088.md), si la constante *gpwComplet* a été utilisée.




**`<Information> : (Constante)`**

Type d'information recherchée :


|   |   |
| --- | --- |
| *gpwComplet* | Permet de récupérer une variable de type [gpwUtilisateur](../WDLang6/1000021088.md) contenant toutes les informations sur l'utilisateur courant.<br>Remarque : la propriété **MotDePasse** de la variable n'est pas renseignée. |
| *gpwInfoDoubleAuthentification* | Permet de savoir si la double authentification est activée (&lt;Résultat&gt; sera à <u><u><u><u>Vrai</u></u></u></u>) ou non (&lt;Résultat&gt; sera à <u><u><u><u>Faux</u></u></u></u>) pour l'utilisateur connecté (cette information est donnée par application). |
| *gpwInfoGroupe* | Nom du groupe de l'utilisateur, ou liste des groupes séparés par une tabulation. |
| *gpwInfoLogin* | Login de l'utilisateur ou constante correspondant au type de l'utilisateur : <br><br>- *gpwSuperviseur* : Superviseur de l'application.<br><br>- *gpwVisiteur* : Visiteur.<br><br><br> |
| *gpwInfoNom* | Nom de l'utilisateur |
| *gpwInfoPrénom* | Prénom de l'utilisateur. |
| *gpwInfoSuperviseur* | Permet de savoir si l'utilisateur connecté est un administrateur (&lt;Résultat&gt; sera à <u><u><u><u>Vrai</u></u></u></u>) ou non (&lt;Résultat&gt; sera à <u><u><u><u>Faux</u></u></u></u>). |

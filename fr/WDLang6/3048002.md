


## Fonctions Java
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Ces fonctions permettent d'exécuter simplement du code Java : 



|   |   |
| --- | --- |
| [JavaCharge](../WDLang6/3048001.md) | Indique l'emplacement des classes de l'utilisateur nécessaires à l'exécution de l'application ou de l'applet Java. |
| [JavaExécute](../WDLang6/3048003.md) | Lance une application ou une applet Java en appelant la méthode "Main" d'une classe Java donnée. |
| [JavaExécuteFonction](../WDLang6/3048004.md) | Lance une fonction statique spécifique d'une classe Java donnée. |

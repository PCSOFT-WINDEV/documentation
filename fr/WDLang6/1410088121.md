


## PDFRécupèreDateSignature (Fonction)

***En anglais : PDFGetSignatureDate***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Récupère la date de l'une des signatures d'un document PDF.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonPDF est un pdfDocument = PDFOuvre("test.pdf")

// Nombre de signatures dans le fichier PDF
nNbSignature est un entier = PDFNbSignature(MonPDF)

// Récupération des informations sur les signatures
POUR i = 1 _A_ nNbSignature
	// Récupération de la signature
	bufSignature est un Buffer = PDFRécupèreSignature(MonPDF, i)
	
	// Récupération de la date de la signature
	dhDateSignature est un DateHeure = PDFRécupèreDateSignature(MonPDF, i)

	// Récupération du certificat
	Certif est un Certificat = CertificatExtrait(bufSignature)
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = PDFRécupèreDateSignature(<Document PDF> , <Indice>)`
---

**`<Résultat> : (DateHeure)`**

Date de la signature examinée. 

**`<Document PDF> : (Nom de champ ou variable de type pdfDocument)`**

Document PDF à analyser. Ce document peut correspondre : 

- à une variable de type [pdfDocument](../WDLang6/1000024909.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) à un champ Lecteur PDF.




**`<Indice> : (Entier)`**

Indice de la signature dans le document (le nombre de signatures est connu par la fonction [PDFNbSignature](../WDLang6/1410088119.md)).

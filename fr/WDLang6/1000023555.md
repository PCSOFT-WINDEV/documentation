
## gpwSuperviseur (Fonction)

***En anglais : gpwSupervisor***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le login du compte "Superviseur" selon le framework utilisé. Ce compte est créé automatiquement à la mise en place du groupware. 

Remarque : Le groupware et les fichiers du groupware étant multilingues, il est ainsi possible de récupérer la valeur de la constante représentant le nom du compte "Superviseur".
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI sCompteEncours = gpwSuperviseur() ALORS
	// Programmer ci-dessous le code à exécuter 
	// lorsque le compte en cours est le superviseur
	...
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = gpwSuperviseur()`
---

**`<Résultat> : (Chaîne de caractères)`**

Login du compte superviseur selon le framework utilisé. Correspond à : 

- "Superviseur" lorsque le framework est en français, 

- "Supervisor" lorsque le framework est en anglais.

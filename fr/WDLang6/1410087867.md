
## Fonctions de gestion des scanneurs (syntaxe préfixée)

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions de gestion des scanneurs (syntaxe préfixée) sont les suivantes :



|   |   |
| --- | --- |
| [&lt;Variable Scanneur&gt;.AvecInterfaceUtilisateur](../WDLang6/1410087860.md) | Enregistre un document provenant d'un scanneur dans un ou plusieurs fichiers, en utilisant l'interface utilisateur du scanneur. |
| [&lt;Variable Scanneur&gt;.Capacité](../WDLang6/1410088721.md) | Permet de savoir si un scanneur possède une capacité donnée. |
| [&lt;Variable Scanneur&gt;.Propriété](../WDLang6/1410087863.md) | Permet de connaître ou de modifier les caractéristiques d'un scanneur. |
| [&lt;Variable Scanneur&gt;.VersImage](../WDLang6/1410087864.md) | Récupère un document provenant d'un scanneur dans une variable de type [Image](../WDLang1/1000019650.md). |
| [&lt;Variable Scanneur&gt;.VersPDF](../WDLang6/1410087866.md) | Enregistre un document provenant d'un scanneur dans un fichier PDF. |




## TwainSelectSource (Fonction)

***En anglais : TwainSelectSource***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Affiche une boîte de dialogue standard listant les différents périphériques Twain disponibles sur le poste en cours. L'utilisateur pourra choisir le périphérique Twain à utiliser par défaut. Ce périphérique sera pris en compte pour toutes les fonctions Twain suivantes.<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=TwainSelectSource.gif)
**Remarques** : 

- La fonction [TwainSourceEnCours](../WDLang6/3078015.md) permet de connaître le périphérique Twain utilisé par défaut par les fonctions Twain.

- En 64 bits, seuls les périphériques ayant un driver 64 bits sont listés. 



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affiche la boîte de dialogue standard listant les différents
// périphériques Twain disponibles sur le poste en cours
SI TwainSelectSource() = Faux ALORS
	Erreur("L'utilisateur n'a pas choisi de périphérique Twain")
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = TwainSelectSource()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si un périphérique Twain a été sélectionné par l'utilisateur, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Le message d'erreur correspondant peut être connu avec la fonction [ErreurInfo](../WDLang1/3013008.md).  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Changer le périphérique Twain par défaut :
<a name="changer_peripherique_twain_par_defaut_ELTPARAGRAPHE000048"></a>Pour changer le périphérique Twain utilisé par défaut, il est possible :

- soit d'utiliser la fonction [TwainSourceEnCours](../WDLang6/3078015.md).

- soit d'afficher la liste des périphériques Twain installés grâce à la fonction **TwainSelectSource**. L'utilisateur pourra alors choisir le périphérique Twain à utiliser par défaut.





### Fonctionnement dans des applications 64 bits
<a name="fonctionnement_dans_des_applications_64_bits_ELTPARAGRAPHE000183"></a>Dans une application 64 bits, cette fonction utilisera automatiquement le driver Twain 64 bits. Si ce driver n'est pas installé, le périphérique TWAIN ne pourra pas être manipulé.




## TwainSourceEnCours (Fonction)

***En anglais : TwainCurrentSource***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de connaître et de changer le périphérique Twain utilisé par défaut. Ce périphérique sera utilisé par défaut pour toutes les fonctions Twain suivantes.

**Remarque** : Pour connaître l'ensemble des périphériques Twain installés sur le poste en cours, utilisez la fonction [TwainListeSource](../WDLang6/3078005.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Périphérique Twain utilisé par défaut ?
Trace("Le périphérique utilisé par défaut est : " + TwainSourceEnCours())
```
<a name="Exemple2"></a>

```wl
// Changer le périphérique Twain utilisé par défaut
TwainSourceEnCours("Logitech Camera")
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Changer le périphérique Twain utilisé par défaut

`<Résultat> = TwainSourceEnCours(<Nom du driver>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le périphérique spécifié a été pris en compte, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Le message d'erreur correspondant peut être connu avec la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Nom du driver> : (Chaîne de caractères)`**

Nom du driver du périphérique Twain à utiliser par défaut. Ce nom doit correspondre à un des noms renvoyé par la fonction [TwainListeSource](../WDLang6/3078005.md).  


<a name="SYNTAXE2"></a>

### Connaître le périphérique Twain utilisé par défaut

`<Résultat> = TwainSourceEnCours()`
---

**`<Résultat> : (Chaîne de caractères)`**

Nom du driver du périphérique Twain utilisé par défaut.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Changer le périphérique Twain par défaut
<a name="changer_peripherique_twain_par_defaut_ELTPARAGRAPHE000058"></a>Pour changer le périphérique Twain utilisé par défaut, il est possible :

- soit d'utiliser la fonction **TwainSourceEnCours**.

- soit d'afficher la liste des périphériques Twain installés grâce à la fonction [TwainSelectSource](../WDLang6/3078013.md). L'utilisateur pourra alors choisir le périphérique Twain à utiliser par défaut.





### Fonctionnement dans des applications 64 bits
<a name="fonctionnement_dans_des_applications_64_bits_ELTPARAGRAPHE000199"></a>Dans une application 64 bits, cette fonction utilisera automatiquement le driver Twain 64 bits. Si ce driver n'est pas installé, le périphérique TWAIN ne pourra pas être manipulé.

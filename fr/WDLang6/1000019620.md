
## paDésactiveAction (Fonction)

***En anglais : apDisableAction***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Désactive l'exécution d'une action présente dans un plan d'action. 

Cette fonction peut être utilisée uniquement dans les traitements d'un plan d'action. Il est par exemple possible de désactiver certaines actions dans le code d'initialisation du plan d'action en fonction de paramètres. 

**Remarque** : Cette fonction est une fonction permettant de manipuler l'éditeur de WINDEV, WEBDEV ou WINDEV Mobile. Elle est donc disponible dans les trois produits. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Désactivation de l'action "AuditStatique"
SI paramAudit = Faux ALORS
	paDésactiveAction(AuditStatique)
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`paDésactiveAction(<Nom de l'action>)`
---

**`<Nom de l'action> : (Chaîne de caractères)`**

Nom de l'action à désactiver. L'action désactivée ne sera pas exécutée et donc ne provoquera pas d'erreur.

**Rappel** : Le nom de l'action est défini dans la fenêtre de description de l'action.


## VerrouAppliDébloque (Fonction)

***En anglais : LockAppOff***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Déverrouille une application bloquée par le verrouillage automatique ou la fonction [VerrouAppliBloque](../WDLang6/1000017011.md).

**Remarque** : Cette fonction est notamment utilisée dans la fenêtre de déverrouillage de l'application. Elle peut également servir à déverrouiller une application par programmation (par exemple utilisation d'un raccourci clavier spécifique).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
VerrouAppliDébloque()
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`VerrouAppliDébloque()`

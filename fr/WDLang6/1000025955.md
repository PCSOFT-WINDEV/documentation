
## Constantes OCR
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les constantes spécifiques à l'OCR sont les suivantes :

| Constante | Détail | Fonction / Type |
| --- | --- | --- |
| **ocrPrécis** | Mode de précision : La précision de la lecture est privilégiée. | Type [ocrOption](../WDLang6/1000025768.md) |
| **ocrRapide** | Mode de précision : La rapidité de la lecture est privilégiée | Type [ocrOption](../WDLang6/1000025768.md) |


## ConsoleEcrit (Fonction)

***En anglais : ConsoleWrite***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ecrit une chaîne de caractères dans le flux de sortie standard "stdout" (également appelé "console").
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
ConsoleEcrit("Opération réussie")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`ConsoleEcrit(<Message>)`
---

**`<Message> : (Chaîne de caractères)`**

Chaîne à écrire sur la sortie standard. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Attention : sous Windows, l'exécutable généré doit être en mode console pour pouvoir voir la chaîne écrite.

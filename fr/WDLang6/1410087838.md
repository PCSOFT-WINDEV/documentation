


## ScanVersImage (Fonction)

***En anglais : ScanToImage***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Récupère un document provenant d'un scanneur dans une variable de type [Image](../WDLang1/1000019650.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonScanneur est un Scanneur 
MonScanneur = ScanSélecteur()

// Affiche le document présent dans le scanneur dans un champ Image
IMG_MonImage = ScanVersImage(MonScanneur)
SI ErreurDétectée ALORS
	Erreur("Le document n'a pas été scanné", ErreurInfo())
FIN

// Scan de toutes les pages du chargeur
ScanVersImage(MonScanneur, TraitePageScan)
// Traitement d'une page scannée
PROCEDURE INTERNE TraitePageScan(nNumPage est un entier, ImageScan est une Image)
	dSauveImageJPEG(ImageScan, "page" + nNumPage + ".jpg")
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Scanner une seule page

`<Résultat> = ScanVersImage(<Scanneur>)`
---

**`<Résultat> : (Variable de type Image)`**

Variable de type [Image](../WDLang1/1000019650.md) récupérée du scanneur. En cas d'erreur, la variable [ErreurDétectée](../WDLang1/3087001.md) vaut <u><u><u><u>Vrai</u></u></u></u> et le message d'erreur correspondant peut être connu avec la fonction [ErreurInfo](../WDLang1/3013008.md).

**`<Scanneur> : (Variable de type Scanneur)`**

Nom de la variable de type [Scanneur](../WDLang6/1410087808.md) initialisée avec les informations du scanneur à utiliser.


<a name="SYNTAXE2"></a>

### Scanner plusieurs pages (avec chargeur)

`<Résultat> = ScanVersImage(<Scanneur> [, <Nombre pages>] , <Fin de scan d'une page>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le scan a été effectué sans erreur,

- <u><u><u><u>Faux</u></u></u></u> en cas d'erreur durant le scan. Le message d'erreur correspondant peut être connu avec la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Scanneur> : (Variable de type Scanneur)`**

Nom de la variable de type [scanneur](../WDLang6/1410087808.md) initialisée avec les informations du scanneur à utiliser.

**`<Nombre pages> : (Entier optionnel)`**



- Nombre de pages à scanner, 

- Constante *scanNombrePageToutes* pour scanner toutes les pages du chargeur (valeur par défaut).




**`<Fin de scan d'une page> : (Nom de procédure)`**

Nom de la procédure WLangage (également nommée "callback") appelée lors de la fin du scan d'une page. Cette procédure est de la forme : 

```txt
<Nom de la procédure>(<Numéro de page>, <Image>)
```
où : 

- &lt;Numéro de page&gt; : Numéro de la page scannée. 

- &lt;Image&gt; : Variable de type [Image](../WDLang1/1000019650.md) contenant la page scannée. Cette variable est valable uniquement lors de l'appel à la procédure : elle doit donc être copiée ou sauvée dans un fichier (il ne faut pas prendre de référence sur cette variable).


Si la procédure renvoie <u><u><u><u>Faux</u></u></u></u>, le scan est interrompu.

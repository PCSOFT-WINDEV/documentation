
## pdfSignature (Type de variable)

***En anglais : pdfSignature***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **pdfSignature** permet de définir toutes les caractéristiques avancées d'une signature à appliquer à un fichier PDF : 

- lors de la signature d'un PDF existant (via la fonction [PDFSigne](../WDLang6/1000022241.md)). 

- lors d'une impression (via la fonction [iImprimeSignature](../WDLang5/1000021690.md). 


Les caractéristiques de cette signature peuvent être définies et modifiées à l'aide de différentes propriétés WLangage. 

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MaSignature est un pdfSignature
MaSignature.Certificat = CertificatCharge("test.pfx", "mdp")[1]
MaSignature.Libellé = "Signé par test"
MaSignature.X = 10
MaSignature.Y = 10
MaSignature.Largeur = 150
MaSignature.Hauteur = 50
PDFSigne("MonFichierPDF.pdf", MaSignature)
```

## Remarques
<a name="NOTE0_1"></a>


### Propriétés spécifiques à la description des variables de type pdfSignature
<a name="proprietes_specifiques_description_des_variables_type_pdfsignature_ELTPARAGRAPHE000040"></a>Les propriétés suivantes peuvent être utilisées pour manipuler une signature de PDF :

| Nom de la propriété | Type manipulé | Effet |
| --- | --- | --- |
| BrosseFond | Constante de type Entier | Motif de remplissage du cadre de la signature : <br><br>- *iCroisée* : Brosse croisée.<br><br>- *iCroiséeDiagonale* : Brosse croisée en diagonale.<br><br>- *iDiagonaleDescendante* : Brosse diagonale descendante.<br><br>- *iDiagonaleMontante* : Brosse diagonale ascendante.<br><br>- *iHorizontale* : Brosse horizontale.<br><br>- *iPleine* (valeur par défaut) : Brosse pleine.<br><br>- *iVerticale* : Brosse verticale.<br><br><br> |
| CadrageHorizontal | Constante de type Entier | Cadrage horizontal du texte de la signature : <br><br>- *iCentreH* : Centrage horizontal du texte.<br><br>- *iGauche* (valeur par défaut) : Alignement du texte à gauche.<br><br>- *iDroite* : Alignement du texte à droite.<br><br><br> |
| CadrageVertical | Constante de type Entier | Cadrage vertical du texte de la signature : <br><br>- *iBas* : Alignement du texte en bas.<br><br>- *iCentreV* : Centrage vertical du texte.<br><br>- *iHaut* (valeur par défaut) : Alignement du texte en haut.<br><br><br> |
| Cadre | Variable de type [Cadre](../Motscles/15140100.md) | Description du cadre de la signature. |
| Certificat | Variable de type [Certificat](../WDLang1/1000019287.md) | Certificat à utiliser pour signer le PDF. |
| CouleurFond | Constante de type Entier | Couleur de fond utilisée pour la zone de la signature. Ce paramètre peut correspondre à :<br><br>- une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)),<br><br>- une couleur TSL (obtenue avec la fonction [TSL](../WDLang1/3029057.md)),<br><br>- [une couleur prédéfinie du WLangage](../WDLang5/3010002.md).<br><br><br><br><br> |
| Hauteur | Réel | Hauteur du rectangle de la signature dans la page (en millimètres). Cette hauteur doit être supérieure à 0. |
| Image | Variable de type [Image](../WDLang1/1000019650.md) | Image de la signature. <br><br>Pour une signature invisible, cette propriété et la propriété **Libellé** ne doivent pas être spécifiées. |
| Largeur | Réel | Largeur du rectangle de la signature dans la page (en millimètres). Cette largeur doit être supérieure à 0. |
| Libellé | Chaîne de caractères | Texte de la signature. <br><br>Pour une signature invisible, cette propriété et la propriété **Image** ne doivent pas être spécifiées. Cette propriété **Libellé** peut également correspondre à une chaîne vide.  |
| Opacité | Entier | Opacité de la couleur de fond de la zone de signature. Valeur comprise entre 0 et 100 : <br><br>- 0 : Fond transparent, <br><br>- 100 : Fond opaque. <br><br><br> |
| Page | Entier | Numéro de la page où la signature est affichée. Par défaut, cette propriété vaut 0 (c'est-à-dire affichage de la signature en fin de document). |
| X | Réel | Abscisse de la position de la zone de signature dans la page (en millimètres). |
| Y | Réel | Ordonnée de la position de la zone de signature dans la page (en millimètres). |




<a name="NOTE0_2"></a>


### Signature visible ou invisible
<a name="signature_visible_invisible_ELTPARAGRAPHE000184"></a>Si les propriétés **Libellé** et **Image** ne sont pas précisées, la signature sera invisible : le document sera signé et Adobe Reader fera apparaître un bandeau bleu mais la signature ne sera pas visible dans le document. 

Dans le cas contraire, la signature apparaîtra dans le document avec les paramètres spécifiés par les différentes propriétés du type **pdfSignature**. 
<a name="NOTE0_3"></a>

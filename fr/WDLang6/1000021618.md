
## gpwVérifieMotDePasse (Fonction)

***En anglais : gpwCheckPassword***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Vérifie un mot de passe (crypté ou non crypté).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI gpwVérifieMotDePasse(sLogin, sMotDePasse) ALORS
	RENVOYER gpwOk
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = gpwVérifieMotDePasse(<Login> , <Mot de passe>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le mot de passe correspond à l'utilisateur, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 




**`<Login> : (Chaîne de caractères)`**

Login de l'utilisateur. 

**`<Mot de passe> : (Chaîne de caractères)`**

Mot de passe associé au login de l'utilisateur.

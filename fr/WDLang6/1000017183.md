


## Fonctions SAP
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Ces fonctions permettent d'utiliser les fonctions / BAPI d'un serveur SAP :  



|   |   |
| --- | --- |
| [SAPConnecte](../WDLang6/1000017120.md) | Connecte un système SAP R/3 à partir des informations données dans la [structure SAP](../WDLang6/1000017184.md). |
| [SAPDéconnecte](../WDLang6/1000017121.md) | Déconnecte un système SAP R/3. |
| [SAPExécute](../WDLang6/1000017122.md) | Appelle une fonction ou BAPI synchrone sur un système SAP. |






- ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=LesfonctionsSAP.gif) ***Exemples unitaires (WINDEV)*** : **Les fonctions SAP** <br>Utilisation des fonctions WLangage SAPXxx.<br>Ces fonctions permettent d'utiliser les BAPI SAP nativement dans WINDEV et dans WEBDEV.<br>Cela permet donc d'accéder à vos données SAP directement depuis vos applications WINDEV.

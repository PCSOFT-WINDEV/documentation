## Syntaxe
<a name="SYNTAXE1"></a>

### Connecter un utilisateur au groupware utilisateur

`<Résultat> = gpwConnecteUtilisateur(<Login> , <Mot de passe>)`
---

**`<Résultat> : (Constante de type Entier)`**

Résultat de la connexion : 


|   |   |
| --- | --- |
| *gpwErreur* | Une erreur est survenue. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md). |
| *gpwMotPasseInvalide* | Le mot de passe de l'utilisateur est inconnu. |
| *gpwOK* | La connexion de l'utilisateur a été réalisée. |
| *gpwUtilisateurInconnu* | L'utilisateur est inconnu : il n'est pas présent dans la base de données du groupware utilisateur. |



**`<Login> : (Chaîne de caractères)`**

Login de l'utilisateur. 

**`<Mot de passe> : (Chaîne de caractères)`**

Mot de passe associé au login de l'utilisateur. 


<a name="SYNTAXE2"></a>

### Connecter un utilisateur identifié par son adresse email au groupware utilisateur

`<Résultat> = gpwConnecteUtilisateur(<Email de l'utilisateur>)`
---

**`<Résultat> : (Constante de type Entier)`**

Résultat de la connexion : 


|   |   |
| --- | --- |
| *gpwErreur* | Une erreur est survenue. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md). |
| *gpwMotPasseInvalide* | Le mot de passe de l'utilisateur est inconnu. |
| *gpwOK* | La connexion de l'utilisateur a été réalisée. |
| *gpwUtilisateurInconnu* | L'utilisateur est inconnu : il n'est pas présent dans la base de données du groupware utilisateur. |



**`<Email de l'utilisateur> : (Chaîne de caractères)`**

Email de l'utilisateur récupéré lors d'une authentification via OpenID (propriété **Email** de la variable [OpenIDIdentité](../WDLang3/1000023567.md)).


## Graver un CD/DVD
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000135"></a>
Plusieurs fonctions WLangage sont à votre disposition pour graver des CD ou des DVD. Il est possible de créer : 

- des CD/DVD de type données.

- des CD de type audio.




**Remarque** : Il est possible uniquement de graver des fichiers sur un CD ou un DVD. Il n'est pas possible de copier directement un CD ou un DVD.


### Configuration nécessaire
<a name="configuration_necessaire_ELTPARAGRAPHE000019"></a>![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) La gravure de CD est disponible uniquement sous Windows XP et supérieur. La gravure de DVD est disponible sous Windows Vista, et sous Windows XP en utilisant la mise à jour KB932716 pour Windows XP (cette mise à jour doit être téléchargée depuis le site de Microsoft et installée car elle n'est pas prise en compte par le mécanisme de Live Update).

## Comment graver un CD/DVD?
<a name="comment_graver_cddvd_ELTTEXTE000165"></a>
Pour graver un CD/DVD :

1. Déterminez si nécessaire le graveur à utiliser : 
	


|   |   |
| --- | --- |
| [GraveurChemin](../WDLang6/3061004.md) | Renvoie le chemin du graveur en cours. |
| [GraveurEtat](../WDLang6/3061007.md) | Renvoie l'état actuel du graveur en cours. |
| [GraveurListe](../WDLang6/3061009.md) | Liste les graveurs installés sur le poste en cours. |
| [GraveurPropriété](../WDLang6/3061011.md) | Permet de connaître ou de modifier les propriétés du graveur en cours. |
| [GraveurSélectionne](../WDLang6/3061012.md) | Permet de sélectionner le graveur utilisé par défaut. |

2. Ouvrez si nécessaire le tiroir du graveur en cours (fonction [GraveurEjecte](../WDLang6/3061014.md)).
	**Remarque** : Il est possible de savoir si le graveur contient un support grâce à la fonction [GraveurMédiaPrésent](../WDLang6/1000019910.md). 

3. Spécifiez si nécessaire le type du CD/DVD à créer :
	


|   |   |
| --- | --- |
| [GraveurInfoMédia](../WDLang6/3061008.md) | Récupère les caractéristiques du CD/DVD présent dans le graveur en cours. |
| [GraveurTypeMédia](../WDLang6/3061013.md) | Permet de connaître ou de modifier le format du CD ou DVD à graver. |

4. Effacez si nécessaire les fichiers déjà présents sur le CD/DVD à l'aide de la fonction [GraveurEfface](../WDLang6/3061005.md). Lors de l'effacement, une procédure WLangage ("callback") est appelée régulièrement. Cette procédure permet de gérer les différents événements rencontrés.

5. Indiquez si nécessaire le nom à donner au CD/DVD à graver en utilisant la fonction [GraveurNomVolume](../WDLang6/3061019.md).

6. Sélectionnez les fichiers à graver sur le CD/DVD :
	


|   |   |
| --- | --- |
| [GraveurAjouteFichier](../WDLang6/3061001.md) | Ajoute un fichier à la compilation. |
| [GraveurAjouteRépertoire](../WDLang6/3061002.md) | Ajoute la totalité des fichiers d'un répertoire à la compilation. |

7. Récupérez si nécessaire les caractéristiques de la compilation (fonction [GraveurInfoCompilation](../WDLang6/3061018.md)).

8. Gravez les fichiers sélectionnés sur le CD/DVD avec la fonction [GraveurEnregistre](../WDLang6/3061006.md). Lors de la gravure, une procédure WLangage ("callback") est appelée régulièrement. Cette procédure permet de gérer les différents événements rencontrés.

9. Annulez si nécessaire la gravure en cours (fonction [GraveurAnnuleGravure](../WDLang6/3061003.md)).



<a name="NOTE2_2"></a>


### Remarques
<a name="remarques_ELTPARAGRAPHE000080"></a>

- Les fonctions de gravure permettent de graver des CD multi-sessions.

- Il n'est pas possible de graver des DVD multi-sessions. Il est cependant possible de copier le DVD sur le disque dur puis de le re-graver avec le nouveau contenu.

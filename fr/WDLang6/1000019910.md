
## GraveurMédiaPrésent (Fonction)

***En anglais : BurnerMediaFound***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de savoir si le graveur est plein ou vide. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MédiaPrésent est un booléen
MediaPrésent = GraveurMédiaPrésent()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = GraveurMédiaPrésent()`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si un média est présent dans le graveur. 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.

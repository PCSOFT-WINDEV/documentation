
## pdfPage (Type de variable)

***En anglais : pdfPage***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **pdfPage** permet de définir toutes les caractéristiques avancées d'une page d'un document PDF chargé en mémoire. Les caractéristiques de cette page peuvent être définies et modifiées à l'aide de différentes propriétés WLangage. 

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonPDF est un pdfDocument = "test.pdf"

POUR TOUTE PageDocument de MonPDF.Page
	Trace(PageDocument.Hauteur)
FIN

PDFSauve(MonPDF, "testmdp.pdf")

// Il est également possible d'utiliser la syntaxe préfixée
// MonPDF.Sauve("testmdp.pdf")
```



```wl
MonDocPDF est un pdfDocument = PDFOuvre("C:\temp\Monguide.pdf")
Trace(MonDocPDF.VersionPDF)
Trace(MonDocPDF.Auteur)
Trace(MonDocPDF.Page.Occurrence)
Trace(MonDocPDF.Page[1].Largeur)
Trace(MonDocPDF.Page[36].Hauteur)

POUR TOUT UnElémentTexte DE MonDocPDF.Page[2].ElémentTexte
	Trace(UnElémentTexte.Texte + " X = " + 
	UnElémentTexte.X + " Y = " + UnElémentTexte.Y + " L = " + 
	UnElémentTexte.Largeur + " H = " + UnElémentTexte.Hauteur)
FIN

MonDocPDF2 est un pdfDocument = "C:\temp\plan.pdf"
// Ajout de pages
Ajoute(MonDocPDF2.Page, MonDocPDF.Page[5])
// Insertion de pages
Insère(MonDocPDF2.Page, 1, MonDocPDF.Page[32])
// Enregistrement du PDF modifié
PDFSauve(MonDocPDF2, "MonDocPDF2.pdf")
LanceAppliAssociée("MonDocPDF2.pdf")
```

## Remarques
<a name="NOTE0_1"></a>


### Propriétés spécifiques à la description des variables de type pdfPage
<a name="proprietes_specifiques_description_des_variables_type_pdfpage_ELTPARAGRAPHE000055"></a>Les propriétés suivantes peuvent être utilisées pour manipuler une page d'un document PDF :

| Nom de la propriété | Type manipulé | Effet |
| --- | --- | --- |
| ElémentTexte | Tableau de [pdfElémentTexte](../WDLang6/1000024556.md) | Informations de texte contenu dans la page. |
| Hauteur | Réel | Hauteur de la page en millimètres. <br><br>Si cette hauteur est modifiée, la page est automatiquement redimensionnée et les éléments contenus sont recadrés. <br><br> |
| Largeur | Réel | Largeur de la page en millimètres. <br><br>Si cette largeur est modifiée, la page est automatiquement redimensionnée et les éléments contenus sont recadrés. <br><br> |
| Orientation | Constante de type Chaîne | Permet de connaître et de modifier l'orientation de la page : <br><br>- *Orient0* : Orientation par défaut.<br><br>- *Orient90* : Orientation de la page à 90°.<br><br>- *Orient180* : Orientation de la page à 180°.<br><br>- *Orient270* : Orientation de la page à 270°.<br><br><br> |

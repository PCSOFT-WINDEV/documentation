
## gpwMode (Fonction)

***En anglais : gpwMode***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le mode d'exécution du groupware utilisateur.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Si l'authentification est entièrement faite par le groupware utilisateur
SI gpwMode() = gpwModeWindows ALORS
	// Vérification du login en allant lire dans le fichier
	VerifLogin()
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = gpwMode()`
---

**`<Résultat> : (Constante de type Entier)`**

Mode d'exécution du groupware utilisateur : 


|   |   |
| --- | --- |
| *gpwModeGroupware* | L'authentification est entièrement réalisée par le groupware utilisateur. |
| *gpwModeLDAP* | L'authentification est réalisée par un annuaire LDAP. |
| *gpwModeWindows* | L'authentification est réalisée par Windows. |

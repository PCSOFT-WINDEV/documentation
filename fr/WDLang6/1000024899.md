
## pdfPièceJointe (Type de variable)

***En anglais : pdfAttachment***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **pdfPièceJointe** permet de définir toutes les caractéristiques avancées d'une pièce jointe dans un document PDF. Les caractéristiques de cette pièce jointe peuvent être définies et modifiées à l'aide de différentes propriétés WLangage. 

Il est ainsi possible d'extraire le contenu d'une pièce jointe, d'ajouter ou de supprimer une pièce jointe à un document PDF.

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonPDF est un pdfDocument = "test.pdf"
POUR TOUT UnePièceJointe DE MonPDF.PièceJointe
	fSauveBuffer(UnePièceJointe.Nom, UnePièceJointe.Contenu)
FIN
```





<a name="NOTE0"></a>

## Remarques
<a name="NOTE0_1"></a>


### Propriétés spécifiques à la description des variables de type pdfPièceJointe
<a name="proprietes_specifiques_description_des_variables_type_pdfpiecejointe_ELTPARAGRAPHE000055"></a>Les propriétés suivantes peuvent être utilisées pour manipuler une pièce jointe d'un document PDF :

| Nom de la propriété | Type manipulé | Effet |
| --- | --- | --- |
| Contenu | Buffer | Contenu de la pièce jointe. |
| DateCréation | DateHeure | Date de création de la pièce jointe.<br><br> |
| DateModification | DateHeure | Date de la dernière modification de la pièce jointe.<br><br> |
| Nom | Chaîne de caractères | Nom de la pièce jointe. |

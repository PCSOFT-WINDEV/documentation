
## Constantes utilisées par les fonctions Fabrique
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les constantes utilisées par les fonctions de gestion de la fabrique logicielle sont les suivantes : 



| Constante | Description | Utilisé par les fonctions |
| --- | --- | --- |
| **paEtatDéconnecté** | L'automate d'exécution du plan d'action ne répond plus. | [PlanActionExécute](../WDLang6/1000021332.md) |
| **paEtatEnAttente** | Le plan d'action est en attente d'exécution. | [PlanActionExécute](../WDLang6/1000021332.md) |
| **paEtatEnCours** | Le plan d'action en cours d'exécution. | [PlanActionExécute](../WDLang6/1000021332.md) |
| **paEtatLancement** | Le coordinateur va lancer le plan d'action. | [PlanActionExécute](../WDLang6/1000021332.md) |
| **paEtatTerminéEchec** | L'exécution du plan s'est terminé avec erreur. | [PlanActionExécute](../WDLang6/1000021332.md) |
| **paEtatTerminéOK** | L'exécution du plan s'est terminé sans erreur. | [PlanActionExécute](../WDLang6/1000021332.md) |
| **paInfoAnnulationEnCours** | Information voulue sur le plan d'action : Annulation en cours. | [PlanActionInfo](../WDLang6/1000021829.md) |
| **paInfoAvancement** | Information voulue sur le plan d'action : Pourcentage d'avancement. | [PlanActionInfo](../WDLang6/1000021829.md) |
| **paInfoDébut** | Information voulue sur le plan d'action : Date et heure du début du plan d'action. | [PlanActionInfo](../WDLang6/1000021829.md) |
| **paInfoEtat** | Information voulue sur le plan d'action : Etat du plan d'action. | [PlanActionInfo](../WDLang6/1000021829.md) |
| **paInfoFin** | Information voulue sur le plan d'action : Date et heure de fin du plan d'action. | [PlanActionInfo](../WDLang6/1000021829.md) |
| **paInfoJournal** | Information voulue sur le plan d'action : Contenu du journal d'exécution du plan d'action. | [PlanActionInfo](../WDLang6/1000021829.md) |
| **paInfoMachine** | Information voulue sur le plan d'action : Nom de la machine ayant exécuté le plan d'action. | [PlanActionInfo](../WDLang6/1000021829.md) |
| **paInfoUtilisateur** | Information voulue sur le plan d'action : Nom de l'utilisateur ayant lancé le plan d'action. | [PlanActionInfo](../WDLang6/1000021829.md) |

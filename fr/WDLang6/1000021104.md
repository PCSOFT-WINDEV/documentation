
## gpwVérifieUtilisateur (Fonction)

***En anglais : gpwCheckUser***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Vérifie l'existence et la validité du mot de passe d'un utilisateur du groupware utilisateur.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
nRes est un entier = gpwVérifieUtilisateur(Login, MotPasse)
// Si le login a échoué
SI nRes <> gpwOk ALORS
        SELON nRes
                CAS gpwErreur : Erreur("Erreur à l'initialisation du groupware.")
                CAS gpwUtilisateurInconnu : Erreur("Utilisateur inconnu.")
                CAS gpwMotPasseInvalide : Erreur("Mot de passe invalide.")
        FIN
        FinProgramme()
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = gpwVérifieUtilisateur(<Login> , <Mot de passe>)`
---

**`<Résultat> : (Entier)`**

Résultat de la vérification : 


|   |   |
| --- | --- |
| *gpwErreur* | Une erreur est survenue. Pour plus de détails sur l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md). |
| *gpwMotPasseInvalide* | Le mot de passe de l'utilisateur n'est pas valide. Il ne correspond pas au mot de passe de l'utilisateur décrit dans la base de données du groupware utilisateur.  |
| *gpwOK* | L'utilisateur est connecté. |
| *gpwUtilisateurInconnu* | L'utilisateur est inconnu : il n'est pas présent dans la base de données du groupware utilisateur. |



**`<Login> : (Chaîne de caractères)`**

Login de l'utilisateur. 

**`<Mot de passe> : (Chaîne de caractères)`**

Mot de passe associé au login de l'utilisateur.

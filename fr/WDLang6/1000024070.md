


## Fonctions de gestion des Exécutables en mode Console
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions permettant de manipuler les différents flux des exécutables en mode "console" sont les suivantes : 



|   |   |
| --- | --- |
| [ConsoleEcrit](../WDLang6/1000024022.md) | Ecrit une chaîne de caractères dans le flux de sortie standard "stdout" (également appelé "console"). |
| [ConsoleErreur](../WDLang6/1000024023.md) | Ecrit une chaîne de caractères dans le flux de sortie d'erreurs "stderr". |
| [ConsoleLit](../WDLang6/1000024024.md) | Lit une chaîne sur le flux d'entrée standard "stdin". |

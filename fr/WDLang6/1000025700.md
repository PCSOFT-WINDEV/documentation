
## OCRRépertoireModèle (Fonction)

***En anglais : OCRModelPathDirectory***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de définir le répertoire contenant les modèles de langue pour l'OCR natif.

Attention : Tous les modèles de langue doivent être dans le même répertoire.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
OCRRépertoireModèle("C:\OCR\CheminModèles")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = OCRRépertoireModèle([<Répertoire>])`
---

**`<Résultat> : (Chaîne de caractères)`**

Chemin du répertoire des modèles de langue. 

Si le paramètre &lt;Répertoire&gt; n'est pas précisé, &lt;Résultat&gt; correspond au répertoire courant. 

**`<Répertoire> : (Chaîne de caractères optionnelle)`**

Nouveau chemin du répertoire des modèles de langue. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Trois langues sont proposées dans le produit : Français, Anglais, Espagnol. En cas d'utilisation d'une de ces langues, le chemin du répertoire des modèles de langue ne doit pas être spécifié. En effet, les modèles de langue correspondants sont automatiquement placés à côté de l'exécutable. En cas de combinaison avec d'autres modèles personnels, tous les modèles doivent être placés dans le même répertoire.

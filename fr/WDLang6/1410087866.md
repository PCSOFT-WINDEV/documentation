


## &lt;Variable Scanneur&gt;.VersPDF (Fonction)

***En anglais : &lt;Scanner variable&gt;.ToPDF***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Enregistre un document provenant d'un scanneur dans un fichier PDF.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonScanneur est un Scanneur

// Scanne toutes les pages du chargeur dans un fichier PDF
MonScanneur.VersPDF("Fichier.PDF")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Scanneur>.VersPDF([<Nombre pages>] [, <Paramètres PDF>] , <Fichier PDF>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le document a été enregistré,

- <u><u><u><u>Faux</u></u></u></u> si le document n'a pas été enregistré. Le message d'erreur correspondant peut être connu avec la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Scanneur> : (Variable de type Scanneur)`**

Nom de la variable de type [Scanneur](../WDLang6/1410087808.md) correspondant au scanneur à utiliser.

**`<Nombre pages> : (Entier optionnel)`**



- Nombre de pages à scanner, 

- Constante *scanNombrePageToutes* pour scanner toutes les pages du chargeur (valeur par défaut).




**`<Paramètres PDF> : (Variable de type pdfParamètre)`**

Nom de la variable de type [pdfParamètre](../WDLang6/1410087857.md) décrivant les caractéristiques du fichier PDF à créer.

**`<Fichier PDF> : (Chaîne de caractères)`**

Nom et chemin complet (ou relatif) du fichier (260 caractères maximum). L'extension du fichier n'est pas obligatoire. Un chemin de type UNC peut être utilisé.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Sans chargeur, pour scanner plusieurs pages dans un unique fichier PDF, il faut scanner chaque page dans un fichier PDF puis utiliser la fonction [PDFFusionne](../WDLang6/1000019712.md).

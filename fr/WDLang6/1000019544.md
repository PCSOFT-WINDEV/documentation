


## Fonctions de gestion des plans d'action (Fabrique logicielle)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions de gestion des plans d'action de la fabrique logicielle sont les suivantes :



|   |   |
| --- | --- |
| [paDésactiveAction](../WDLang6/1000019620.md) | Désactive l'exécution d'une action présente dans un plan d'action. |
| [paExécute](../WDLang6/1000019457.md) | Exécute un plan d'action. |
| [paLog](../WDLang6/1000019632.md) | Enregistre une information dans un fichier "log" pendant l'exécution d'un plan d'action. |
| [paNomCompletProjet](../WDLang6/1000019640.md) | Récupère le nom physique complet d'un projet ouvert pendant l'exécution d'un plan d'action. |
| [PlanActionExécute](../WDLang6/1000021332.md) | Exécute une instance de plan d'action dans la fabrique logicielle. |
| [PlanActionInfo](../WDLang6/1000021829.md) | Récupère une information concernant l'exécution d'une instance d'un plan d'action. |
| [PlanActionParamètre](../WDLang6/1000021598.md) | Paramètre un plan d'action dans la fabrique logicielle. |

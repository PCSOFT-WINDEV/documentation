
## PDFEstProtégé (Fonction)

***En anglais : PDFIsProtected***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Vérifie si le fichier PDF nécessite un mot de passe pour être lu.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Teste si le fichier PDF est protégé
SI PDFEstProtégé("C:\Temp\MonDocument.pdf") = Vrai ALORS
	Info("Fichier PDF avec mot de passe")
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = PDFEstProtégé(<Fichier PDF>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si un mot de passe est nécessaire pour ouvrir le fichier PDF, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Fichier PDF> : (Chaîne de caractères)`**

Nom et chemin du fichier PDF à analyser.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Seul le mot de passe nécessaire à l'ouverture du fichier est pris en compte. Si le fichier PDF nécessite uniquement un mot de passe pour modifier les mesures de protection, ce fichier PDF n'est pas considéré comme protégé (la fonction **PDFEstProtégé** renverra <u><u><u><u>Faux</u></u></u></u>).




## TwainListeSource (Fonction)

***En anglais : TwainListSource***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Liste les drivers des périphériques Twain connectés au poste en cours.

**Remarque** : La fonction [TwainSourceEnCours](../WDLang6/3078015.md) permet de connaître le périphérique Twain qui sera utilisé par défaut par les fonctions Twain.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affiche la liste des drivers des périphériques Twain 
// du poste en cours dans le champ LISTE_DriverPériphérique
ListeAjoute(LISTE_DriverPériphérique, TwainListeSource())
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = TwainListeSource()`
---

**`<Résultat> : (Chaîne de caractères)`**

Liste des drivers des périphériques Twain installés sur le poste en cours, sous la forme :

```txt
<Nom du driver 1> + RC + <Nom du driver N>
```




<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Affichage du résultat dans un champ Liste ou un champ Combo
<a name="affichage_resultat_dans_champ_liste_champ_combo_ELTPARAGRAPHE000037"></a>&lt;Résultat&gt; peut être directement affecté à un champ Liste ou à un champ Combo grâce à la fonction [ListeAjoute](../WDLang1/3049004.md). Chaque ligne contiendra le nom du driver correspondant à chaque périphérique Twain installé sur le poste en cours.
<a name="NOTE0_2"></a>


### Pré-requis sous Windows 2008 serveur 64 bits
<a name="prerequis_sous_windows_2008_serveur_64_bits_ELTPARAGRAPHE000047"></a>Dans le cas d'une application 32 bits devant effectuer une numérisation sous Windows serveur 2008 64 bits, la fonctionnalité "Expérience utilisateur" de Windows est requise. Pour ajouter cette fonctionnalité :

1. Sélectionnez les fonctionnalités du gestionnaire de serveur. 

2. Cliquez sur "Ajouter une fonctionnalité". 

3. Cochez la case "Expérience utilisateur" ("Desktop experience" pour un Windows anglais). 

4. Le serveur est redémarré en fin d'installation.





### Fonctionnement dans des applications 64 bits
<a name="fonctionnement_dans_des_applications_64_bits_ELTPARAGRAPHE000184"></a>Dans une application 64 bits, cette fonction utilisera automatiquement le driver Twain 64 bits. Si ce driver n'est pas installé, le périphérique TWAIN ne pourra pas être manipulé.


## iaParamètreDétectionContour (Type de variable)

***En anglais : aiConfigureContourDetection***
			
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **iaParamètreDétectionContour** permet de définir les paramètres de la fonction de détection de contours. Les caractéristiques de ces paramètres peuvent être définies et modifiées à l'aide de différentes propriétés WLangage.

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
monContour est un iaParamètreDétectionContour
monContour.AlgorithmeDétection = iaDétectContourCanny
monContour.Dilatation = Vrai
monContour.TailleNoyauFiltreGaussien = 3
monContour.ValeurSeuil = 150
monContour.ValeurSeuilMax = 255

monPolygone est un Polygone
monPolygone = IADétecteContour("Image_test_contour_3.png", monContour)
```





<a name="PROPS"></a>

## Propriétés


### Propriétés spécifiques à la description des variables de type iaParamètreDétectionContour
<a name="proprietes_specifiques_description_des_variables_type_iaparametredetectioncontour_ELTPARAGRAPHE000028"></a>Les propriétés suivantes peuvent être utilisées pour manipuler une variable de type **iaParamètreDétectionContour** : 

| Nom de la propriété | Type manipulé | Effet |
| --- | --- | --- |
| AlgorithmeDétection | Constante de type Entier | Algorithme utilisé pour la détection des contours : <br><br>- *iaDétectContourCanny* (Valeur par défaut) : algorithme de J.F.Canny (edge detector) de 1986.<br><br>- *iaDétectContourBinaryThresholding* : algorithme de Binary Thresholding.<br><br><br> |
| Dilatation | Booléen | Permet d'ajouter de la dilatation à l'image, ce qui, dans certains cas peut permettre une meilleure détection : <br><br>- <u><u><u>Vrai</u></u></u> pour ajouter de la dilatation à l'image, <br><br>- <u><u><u>Faux</u></u></u> (valeur par défaut) dans le cas contraire. <br><br><br> |
| TailleNoyauFiltreGaussien | Entier | Réduction du bruit floutant l’image à l’aide d’un flou Gaussien. <br><br>Par défaut, cette propriété correspond à la valeur 3. <br><br>Remarque : Cette valeur doit être impaire. |
| ValeurSeuil | Réel | Cette propriété est utilisée lors de l’application de l'algorithme Thresholding. <br><br>Tout pixel ayant une valeur supérieure à **ValeurSeuil** sera défini sur la valeur de **ValeurSeuilMax**.<br><br>Par défaut, cette propriété correspond à la valeur 150,0. |
| ValeurSeuilMax | Réel | Cette propriété est utilisée lors de l’application de l'algorithme Thresholding.<br><br>Tout pixel ayant une valeur supérieure à **ValeurSeuil** sera défini sur la valeur de **ValeurSeuilMax**.<br><br>Par défaut, cette propriété correspond à la valeur 255,0. |

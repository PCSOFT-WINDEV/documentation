
## SAPConnecte (Fonction)

***En anglais : SAPConnect***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Connecte un système SAP R/3 à partir des informations données dans la [structure SAP](../WDLang6/1000017184.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SAP.Adresse = "MINISAP.test.com"
SAP.NuméroSystème = "0"
SAP.NuméroClient = "000"
SAP.Utilisateur = "BCUSER"
SAP.MotDePasse = "MINISAP"
SAP.Langue = ""
IDSAP est un entier = SAPConnecte()
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = SAPConnecte()`
---

**`<Résultat> : (Entier)`**



- Identifiant de la connexion, 

- -1 en cas d'erreur. La fonction [ErreurInfo](../WDLang1/3013008.md) permet d'obtenir plus de renseignements sur le problème rencontré.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
La fonction **SAPConnecte** renvoie -1 uniquement si le serveur n'est pas joignable. Dans le cas d'une erreur de login, de mot de passe ou de compte client qui n'existerait pas, SAP accepte la connexion et renverra une erreur lors d'un appel de procédure.

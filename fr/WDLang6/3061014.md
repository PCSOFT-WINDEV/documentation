
## GraveurEjecte (Fonction)

***En anglais : BurnerEject***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ouvre ou ferme le tiroir du graveur en cours. Ce graveur peut être sélectionné grâce à la fonction [GraveurSélectionne](../WDLang6/3061012.md).

**Remarque** : Pour connaître l'ensemble des graveurs disponibles à partir du poste en cours, utilisez la fonction [GraveurListe](../WDLang6/3061009.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ouvre le tiroir du graveur
ResOuverture est un booléen 
ResOuverture = GraveurEjecte() 
// Erreur rencontrée ? 
SI ResOuverture = Faux ALORS 
	// Affichage de l'erreur 
	Erreur(ErreurInfo(errMessage)) 
SINON 
	Info("Veuillez insérer le CD ou le DVD dans le graveur") 
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = GraveurEjecte([<Ouverture>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'opération demandée a été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour connaître le libellé de l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Ouverture> : (Booléen optionnel)`**



- <u><u><u><u>Vrai</u></u></u></u> (par défaut) si le tiroir du graveur doit s'ouvrir, 

- <u><u><u><u>Faux</u></u></u></u> si le tiroir du graveur doit se fermer.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Ouverture et fermeture du tiroir du graveur
<a name="ouverture_fermeture_tiroir_graveur_ELTPARAGRAPHE000060"></a>Si le tiroir du graveur est déjà ouvert, la ligne de code suivante sera sans effet : 

```wl
GraveurEjecte(Vrai)
```


Inversement, si le tiroir du graveur est fermé, la ligne de code suivante sera également sans effet : 

```wl
GraveurEjecte(Faux)
```


Certains graveurs ignorent la demande de fermeture du tiroir. Dans ce cas, la fonction **GraveurEjecte** renvoie <u><u><u><u>Vrai</u></u></u></u> bien que le tiroir du graveur reste ouvert.


<a name="NOTE0_2"></a>




### Configuration nécessaire 
<a name="configuration_necessaire_ELTPARAGRAPHE000082"></a>

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) La gravure de CD est disponible uniquement sous Windows XP et supérieur. La gravure de DVD est disponible sous Windows Vista, et sous Windows XP en utilisant la mise à jour KB932716 pour Windows XP (cette mise à jour doit être téléchargée depuis le site de Microsoft et installée car elle n'est pas prise en compte par le mécanisme de Live Update).

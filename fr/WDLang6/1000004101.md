
## CDEjecte (Fonction)

***En anglais : CDEject***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Ouvre ou ferme le tiroir du lecteur de CD/DVD sélectionné sur le poste en cours.

**Remarque** : Pour connaître l'ensemble des lecteurs CD/DVD disponibles à partir du poste en cours, utilisez la fonction [CDListe](../WDLang6/1000004102.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ouvre le tiroir du lecteur E:
ResOuverture est un booléen 
ResOuverture = CDEjecte("E:")
// Erreur rencontrée ?
SI ResOuverture = Faux ALORS
	// Affichage de l'erreur
	Erreur(ErreurInfo(errMessage))
SINON
	Info("Veuillez insérer le CD dans le lecteur")
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = CDEjecte(<Lecteur> [, <Ouverture>])`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'opération demandée a été effectuée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour connaître le libellé de l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Lecteur> : (Chaîne de caractères)`**

Lecteur pour lequel le tiroir doit s'ouvrir ou se fermer.

**`<Ouverture> : (Booléen optionnel)`**



- <u><u><u><u>Vrai</u></u></u></u> (par défaut) si le tiroir du lecteur doit s'ouvrir, 

- <u><u><u><u>Faux</u></u></u></u> si le tiroir du lecteur doit se fermer.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Ouverture et fermeture du tiroir du lecteur
<a name="ouverture_fermeture_tiroir_lecteur_ELTPARAGRAPHE000059"></a>

- Si le tiroir du lecteur est déjà ouvert, **CDEjecte**(&lt;Lecteur&gt;,Vrai) sera sans effet. Inversement, si le tiroir du lecteur est fermé, **CDEjecte**(&lt;Lecteur&gt;, Faux) sera également sans effet.

- Certains lecteurs ignorent la demande de fermeture du tiroir. Dans ce cas, la fonction **CDEjecte** renvoie <u><u><u><u>Vrai</u></u></u></u> bien que le tiroir du lecteur reste ouvert.

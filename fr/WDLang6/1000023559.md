
## gpwUtilisateursEnregistrés (Fonction)

***En anglais : gpwRegisteredUsers***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie la valeur correspondant au mot "Utilisateurs enregistrés" selon le framework utilisé. Le mot "Utilisateurs enregistrés" est utilisé lorsqu'un utilisateur est enregistré dans le groupware.

Remarque : Le groupware et les fichiers du groupware étant multilingues, il est ainsi possible de récupérer la valeur de la constante représentant le mot "Utilisateurs enregistrés".


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
sCstgpwUtilisateursEnregistrés est une chaîne
sCstgpwUtilisateursEnregistrés = gpwUtilisateursEnregistrés()
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = gpwUtilisateursEnregistrés()`
---

**`<Résultat> : (Chaîne de caractères)`**

Valeur du mot "Utilisateurs enregistrés" selon le framework utilisé : 

- "Utilisateurs enregistrés" lorsque le framework est en français,

- "Registered users" lorsque le framework est en anglais.

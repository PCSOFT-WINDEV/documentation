
## TuileSupprimeTout (Fonction)

***En anglais : TileDeleteAll***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Supprime toutes les modifications faites à la tuile de l'application. La tuile de l'application sera la tuile définie en édition. 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
TuileSupprimeTout()
```

## Syntaxe

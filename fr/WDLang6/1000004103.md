


## Fonctions CD/DVD
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les fonctions de gestion des lecteurs CD/DVD sont les suivantes :



|   |   |
| --- | --- |
| [CDEjecte](../WDLang6/1000004101.md) | Ouvre ou ferme le tiroir du lecteur de CD/DVD sélectionné sur le poste en cours. |
| [CDListe](../WDLang6/1000004102.md) | Liste les lecteurs de CD/DVD installés sur le poste en cours. |

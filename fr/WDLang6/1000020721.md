
## TuileAfficheBadge (Fonction)

***En anglais : TileDisplayBadge***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Affiche un badge (notification numérique ou image) sur la tuile d'une application Universal Windows 10 App. 

Par exemple, la tuile suivante propose une notification numérique affichant le nombre 12 et indiquant que 12 messages sont à lire. 
<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=BadgeAffiche.gif)



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
TuileAfficheBadge(nNbEmailsNonLu)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`TuileAfficheBadge(<Valeur>)`
---

**`<Valeur> : (Entier ou constante de type entier)`**



- Entier de 0 à 99 : la valeur choisie sera affichée dans la tuile.  La valeur 0 permet de supprimer le badge. 

- Constante permettant d'afficher une image dans la tuile. Cette constante peut être choisie parmi les valeurs suivantes : 
	


|   |   |
| --- | --- |
| *badgeAlerte* | <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=badge_Alerte.gif)<br> |
| *badgeAttention* | <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Badge_Attention.gif)<br> |
| *badgeAucun* | Le badge est supprimé. |
| *badgeDisponible* | <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=badge_disponible.gif)<br> |

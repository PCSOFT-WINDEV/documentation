
## Variable SAP
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000128"></a>
La variable SAP permet de configurer les paramètres de la connexion au serveur SAP. Ces paramètres seront utilisés par la fonction [SAPConnecte](../WDLang6/1000017120.md).



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Variable SAP
<a name="variable_sap_ELTTEXTE000152"></a>
La variable SAP est composée des éléments suivants :


|   |   |
| --- | --- |
| SAP.Adresse | Chaîne de caractères<br><br>Adresse du système SAP. |
| SAP.Connexion | Chaîne de caractères<br><br>Description de la connexion SAP. Pour plus de détails sur le contenu de cette description, consultez la documentation spécifique SAP.<br>Exemple :<br><br>SAP.Connexion = "TYPE=3 ASHOST=MINISAP.test.com SYSNR=00 <br>CLIENT=000 USER=BCUSER PASSWD=minisap"<br><br><br><br>Remarque : si ce paramètre est précisé, les autres variables SAP sont ignorées. |
| SAP.Langue | Chaîne de caractères<br><br>Définit la langue utilisée pour la connexion (par exemple "en" pour l'anglais) |
| SAP.MotDePasse | Mot de passe de l'utilisateur qui se connecte. |
| SAP.NuméroClient | Chaîne de caractères<br><br>Numéro du client qui se connecte. |
| SAP.NuméroSystème | Entier<br><br>Numéro du système SAP. |
| SAP.Utilisateur | Chaîne de caractères<br><br>Nom de l'utilisateur qui se connecte. |







<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Exemple
<a name="exemple_ELTTEXTE000176"></a>


### Connexion à un serveur SAP
<a name="connexion_serveur_sap_ELTPARAGRAPHE000078"></a>


```wl
SAP.Adresse = "MINISAP.test.com"
SAP.NuméroSystème = "0"
SAP.NuméroClient = "000"
SAP.Utilisateur = "BCUSER"
SAP.MotDePasse = "MINISAP"
SAP.Langue = ""
IDSAP est un entier = SAPConnecte()
```

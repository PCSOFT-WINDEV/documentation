
## GraveurPropriété (Fonction)

***En anglais : BurnerProperty***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Permet de connaître ou de modifier les propriétés du graveur en cours. Ce graveur peut être sélectionné grâce à la fonction [GraveurSélectionne](../WDLang6/3061012.md).

**Remarque** : Pour connaître l'ensemble des graveurs disponibles à partir du poste en cours, utilisez la fonction [GraveurListe](../WDLang6/3061009.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Vitesse du graveur
ResVitesse est un entier 
ResVitesse = GraveurPropriété(graveurVitesse)

Info("La vitesse de gravure est : " + ResVitesse + "x")
```


<a name="Exemple2"></a>



```wl
// Gestion du "SmartBurn"
GraveurPropriété(graveurSmartBurn, 1)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître les propriétés du graveur

`<Résultat> = GraveurPropriété(<Propriété du graveur>)`
---

**`<Résultat> : (Entier)`**

Valeur de la propriété demandée.

**`<Propriété du graveur> : (Constante de type Entier)`**

Propriété à demander :


|   |   |
| --- | --- |
| *graveurSecondesEntrePistes* | Nombre de secondes insérées entre chaque piste. |
| *graveurSmartBurn* | Gestion du SmartBurn :<br><br>- 0 : le SmartBurn n'est pas géré.<br><br>- 1 : le SmartBurn est géré.<br><br><br> |
| *graveurVitesse* | Vitesse actuelle de gravure |
| *graveurVitesseMax* | Vitesse maximale de gravure |




<a name="SYNTAXE2"></a>

### Modifier les propriétés du graveur

`<Résultat> = GraveurPropriété(<Propriété du graveur> , <Nouvelle valeur>)`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la propriété a été modifiée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Pour connaître le libellé de l'erreur, utilisez la fonction [ErreurInfo](../WDLang1/3013008.md).




**`<Propriété du graveur> : (Constante de type Entier)`**

Propriété à modifier :


|   |   |
| --- | --- |
| *graveurSecondesEntrePistes* | Nombre de secondes insérées entre chaque piste. |
| *graveurSmartBurn* | Gestion du SmartBurn :<br><br>- 0 : le SmartBurn n'est pas géré.<br><br>- 1 : le SmartBurn est géré.<br><br><br> |
| *graveurVitesse* | Vitesse de gravure. |



**`<Nouvelle valeur> : (Entier)`**

Nouvelle valeur de la propriété demandée.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### SmartBurn
<a name="smartburn_ELTPARAGRAPHE000102"></a>La technologie Smart Burn (Smart Monitoring And Adapting Recording Technology for Burning) optimise la vitesse de gravure en fonction de la qualité du CD utilisé et permet d'éviter les erreurs du type "buffer underruns".

Cette technologie est disponible uniquement sur certains types de graveur.


<a name="NOTE0_2"></a>


### Configuration nécessaire 
<a name="configuration_necessaire_ELTPARAGRAPHE000112"></a>![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) La gravure de CD est disponible uniquement sous Windows XP et supérieur. La gravure de DVD est disponible sous Windows Vista, et sous Windows XP en utilisant la mise à jour KB932716 pour Windows XP (cette mise à jour doit être téléchargée depuis le site de Microsoft et installée car elle n'est pas prise en compte par le mécanisme de Live Update).

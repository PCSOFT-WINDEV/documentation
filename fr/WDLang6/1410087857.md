
## pdfParamètre (Type de variable)

***En anglais : pdfParameter***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **pdfParamètre** permet de définir toutes les caractéristiques d'un fichier PDF à générer. Les caractéristiques de ce fichier PDF peuvent être définies et modifiées à l'aide de différentes propriétés WLangage. 

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affichage de deux pages entières avec défilement et page de couverture
Params est un pdfParamètre
Params.Qualité = iQualitéElevée
Params.Zoom = zoomPleineLargeur
Params.AffichagePage = affDeuxPages + affAvecDéfilement + affAvecCouverture
iParamètrePDF(Params)
iDestination(iPDFGénérique)
iImprimeEtat(MonEtat)

LanceAppliAssociée(iDernierFichier())
```



```wl
// Affichage d'une page avec défilement et avec zoom au contenu
param est un pdfParamètre
param.Zoom = zoomContenu
param.AffichagePage = affUnePage + affAvecDéfilement
iParamètrePDF(param)
iDestination(iPDFGénérique, gsNomPDF)
iImprimeEtat(ETAT_Base)

LanceAppliAssociée(gsNomPDF)
```



```wl
// Affichage d'une page en 100% avec défilement
param est un pdfParamètre
param.Zoom = 100
param.AffichagePage = affAvecDéfilement
iParamètrePDF(param)
iDestination(iPDFGénérique, gsNomPDF)
iImprimeEtat(ETAT_Base)

LanceAppliAssociée(gsNomPDF)
```

## Remarques
<a name="NOTE0_1"></a>


### Propriétés spécifiques à la description des variables de type pdfParamètre
<a name="proprietes_specifiques_description_des_variables_type_docparampagetitleshort_ELTPARAGRAPHE000036"></a>Les propriétés suivantes peuvent être utilisées pour manipuler les paramètres d'un fichier PDF :

| Nom de la propriété | Type manipulé | Effet |
| --- | --- | --- |
| AffichagePage | Constante de type Entier | Mode d'affichage de la page dans le lecteur PDF utilisé. Correspond à une combinaison des constantes suivantes : <br><br>- *affAvecCouverture* : Considère la première page du fichier PDF comme étant une couverture. La première page est affichée à droite lors d'un affichage en mode 2 pages. <br><br>- *affAvecDéfilement* : Affichage avec défilement : Affiche les pages selon une colonne verticale ininterrompue occupant la largeur d'une page. Le défilement peut être activé pour un affichage 1 page ou 2 pages. <br><br>- *affDéfaut* (Valeur par défaut) : Affichage par défaut. <br><br>- *affDeuxPages* : Affichage de deux pages à la fois, côte à côte du ficher PDF dans le lecteur PDF. <br><br>- *affUnePage* : Affichage d'une seule page à la fois du fichier PDF dans le lecteur PDF. Les parties adjacentes ne sont pas affichées. <br><br><br> |
| FormatCouleur | Constante de type Entier | Gestion de la couleur dans le fichier PDF : <br><br>- *pdfCouleur* : Création d'un fichier PDF en couleurs. <br><br>- *pdfNiveauGris* : Création d'un fichier PDF en noir et blanc. <br><br><br> |
| MotDePasseModification | Chaîne de caractères | Mot de passe utilisé pour permettre la modification des propriétés du PDF. |
| MotDePasseOuverture | Chaîne de caractères | Mot de passe à demander lors de l'ouverture du PDF. |
| Protection | Constantes de type Entier | Options de protection du fichier PDF généré. Correspond à la combinaison des constantes suivantes : <br><br>- Options de format : <br><br>	- *iPDFA* ou *iPDFA1b* : PDF au format PDF/A-1b. Tous les éléments nécessaires à l'affichage ou l'impression du fichier PDF sont intégrés au fichier généré. La taille du fichier généré est donc plus importante. <br>			Remarque : Ce format implique certaines limitations : <br><br>		- Transparence des images non gérée. <br><br>		- Polices de type OpenType non gérées. <br><br>		- Utilisation de polices incorporables.<br><br><br><br><br>- *iPDFA3b* : PDF au format PDF/A-3b. Tous les éléments nécessaires à l'affichage ou l'impression du fichier PDF sont intégrés au fichier généré. La taille du fichier généré est donc plus importante. Il est possible d'ajouter au fichier PDF, des fichiers au format XML, CSV, traitement de texte, tableur, etc. <br>			Remarque : Ce format implique certaines limitations : <br><br>	- Transparence des images non gérée. <br><br>	- Polices de type OpenType non gérées. <br><br>	- Utilisation de polices incorporables.<br><br>- Options de protection : <br><br>	- *iProtectionAucune* : Aucune protection du fichier PDF.<br><br>	- *iProtectionCommentaire* : Protection contre l'ajout ou la modification de commentaires.<br><br>	- *iProtectionImpression* : Le fichier PDF ne pourra pas être imprimé.<br><br>	- *iProtectionModification* : Le contenu du fichier PDF ne pourra pas être modifié.<br><br>	- *iProtectionSélection* : Le contenu du fichier PDF ne pourra pas être copié ou extrait du fichier.<br><br><br><br><br><br> |
| Qualité | Constante de type Entier | Qualité de compression des couleurs et des niveaux de gris : <br><br>- *iQualitéAutomatique* : Qualité calculée automatiquement. <br><br>- *iQualitéElevée* : Qualité de compression élevée.<br><br>- *iQualitéFaible* : Qualité de compression faible.<br><br>- *iQualitéMaximale* : Qualité de compression maximale.<br><br>- *iQualitéMinimale* : Qualité de compression minimale.<br><br>- *iQualitéMoyenne* : Qualité de compression moyenne.<br><br><br> |
| Signet | Booléen | <br><br>- <u><u><u><u>Vrai</u></u></u></u> si les signets doivent être pris en compte dans le PDF. Un signet est automatiquement défini pour chaque page du PDF. Chaque signet a pour nom "Page &lt;numéro de la page&gt;" (par exemple "Page 1").<br><br>- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.<br><br><br> |
| Unicode | Booléen | <br><br>- <u><u><u><u>Vrai</u></u></u></u> si le PDF est au format Unicode, <br><br>- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.<br><br><br><br><br> |
| Version | Constante de type Entier | Version du PDF généré : <br><br>- *iPDF14* : PDF généré au format PDF 1.4 (valeur par défaut). <br><br>- *iPDF15* : PDF généré au format PDF 1.5<br><br>- *iPDF16* : PDF généré au format PDF 1.6<br><br>- *iPDF17* : PDF généré au format PDF 1.7<br><br><br> |
| Zoom | Entier ou constante de type Entier | Zoom appliqué par défaut au fichier PDF à son ouverture : <br><br>- Valeur de zoom comprise en 25% et 6400%, <br><br>- *zoomContenu* : Zoom permettant d'afficher la page de sorte que le texte et les images occupent toute la largeur du visualisateur utilisé. <br><br>- *zoomDéfaut* : Zoom par défaut. <br><br>- *zoomPageEntière* : Zoom permettant d'afficher une page entière. <br><br>- *zoomPleineHauteur* : Zoom permettant d'afficher la page entière dans sa hauteur. <br><br>- *zoomPleineLargeur* : Zoom permettant d'afficher la page entière dans sa largeur. <br><br>- *zoomTailleRéelle* : Zoom à 100%. <br><br><br> |

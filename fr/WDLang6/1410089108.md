
## IADétecteDessin (Fonction)

***En anglais : AIDetectDrawing***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Classifie les dessins à main levée parmi cercle, carré et triangle.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
tabIARésultatImage est un tableau de iaRésultatImage

QUAND EXCEPTION DANS 
	tabIARésultatImage = IADétecteDessin("rond.png")
FAIRE 
	Trace(ExceptionInfo(errComplet))
FIN

iElement est un iaRésultatImage
POUR TOUT ÉLÉMENT iElement DE tabIARésultatImage
	Trace(iElement.iadTypeElément, " ", iElement.IndiceConfiance)
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = IADétecteDessin(<Image>)`
---

**`<Résultat> : (Tableau)`**

Tableau de 3 variables de type [iaRésultatImage](../WDLang6/1410087965.md), trié par ordre décroissant suivant l'indice de confiance. 

**`<Image> : (Nom de champ, variable de type Image, chaîne de caractères)`**

Image à analyser. Cette image peut correspondre à : 

- un champ Image,

- une variable de type [Image](../WDLang1/1000019650.md), 

- un mémo Image, 

- le chemin d'un fichier image ou d'un fichier PDF. 


Cette image doit correspondre à une image sur fond blanc avec un trait noir. L'épaisseur du trait doit être relativement épaisse afin d'améliorer la détection et la classification. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
La fonction **IADétecteDessin** renvoie un tableau contenant 3 variables de type [iaRésultatImage](../WDLang6/1410087965.md). Chacune correspond à une des valeurs possibles (cercle, carré ou triangle), classés selon la probabilité.

Pour chaque valeur de classification possible, les informations suivantes sont données : 

- le type (*iadCercle*, *iadCarré* et *iadTriangle*), 

- la probabilité (en pourcentage). 


Le polygone n'est pas renseigné.

Attention, il est important de noter qu'il s'agit d'un classifieur. Si le dessin ne ressemble ni à un cercle, ni à un carré, ni à un triangle, le classifieur va quand même donner des probabilités pour le cercle, le carré et le triangle qui peuvent ne pas être 0.

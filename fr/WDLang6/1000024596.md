
## gpwVérificationBiométrique (Fonction)

***En anglais : gpwBiometricAuthentication***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Active la vérification biométrique pour l'utilisateur connecté ou récupère l'état actuel pour l'utilisateur connecté.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
bEtat est un booléen = gpwVérificationBiométrique()
gpwVérificationBiométrique(PAS bEtat)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Activer la vérification biométrique

`gpwVérificationBiométrique([<Active>])`
---

**`<Active> : (Booléen optionnel)`**



- <u><u><u><u>Vrai</u></u></u></u> pour activer la vérification biométrique, 

- <u><u><u><u>Faux</u></u></u></u> pour désactiver la vérification biométrique. 





<a name="SYNTAXE2"></a>

### Récupérer l'état de la vérification biométrique pour l'utilisateur actuel

`<Actif> = gpwVérificationBiométrique()`
---

**`<Actif> : (Booléen)`**

Etat courant de la vérification biométrique pour l'utilisateur connecté : 

- <u><u><u><u>Vrai</u></u></u></u> si la vérification biométrique est activée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 








<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Si le groupware utilisateur est paramétré par défaut avec la vérification biométrique, cette fonction est sans effet.

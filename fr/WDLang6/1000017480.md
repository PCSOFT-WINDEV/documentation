
## PDFNombreDePages (Fonction)

***En anglais : PDFNumberOfPages***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le nombre total de pages d'un fichier PDF.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
NbPages est un entier
NbPages = PDFNombreDePages("C:\temp\MonDocument.pdf")
Info("Nombre de pages du PDF : " + NbPages)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = PDFNombreDePages(<Fichier PDF> [, <Mot de passe>])`
---

**`<Résultat> : (Entier)`**

Nombre de pages total du fichier PDF.

**`<Fichier PDF> : (Chaîne de caractères)`**

Nom et chemin du fichier PDF à analyser.

**`<Mot de passe> : (Chaîne de caractères optionnelle)`**

Si le fichier PDF est protégé en ouverture, mot de passe nécessaire pour ouvrir le fichier.


## ocrOption (Type de variable)

***En anglais : ocrOption***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **ocrOption** permet de définir toutes les options générales de l'OCR (Optical Character Recognition) natif. Les caractéristiques de ces options peuvent être définies et modifiées à l'aide de différentes propriétés WLangage. 

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Création des options
MesOCROptions est une ocrOption

// Ne souhaite que les nombres lors de la lecture d'une image
MesOCROptions.NombreUniquement = Vrai

OCRParamètre(MesOCROptions)
```





<a name="NOTE0"></a>

## Remarques
<a name="NOTE0_1"></a>


### Propriétés spécifiques à la description des variables de type ocrOption
<a name="proprietes_specifiques_description_des_variables_type_ocroption_ELTPARAGRAPHE000029"></a>Les propriétés suivantes peuvent être utilisées pour manipuler les options avancées de l'OCR natif :

| Nom de la propriété | Type manipulé | Effet |
| --- | --- | --- |
| AvecCorrecteurLinguistique | Booléen | <br><br>- <u><u><u><u>Vrai</u></u></u></u> (valeur par défaut) pour activer le correcteur linguistique, <br><br>- <u><u><u><u>Faux</u></u></u></u> pour le désactiver. <br><br><br> |
| ListeMot | Tableau de chaînes |   |
| NombreUniquement | Booléen | <br><br>- <u><u><u><u>Vrai</u></u></u></u> si l'OCR doit renvoyer uniquement les chiffres arabes, <br><br>- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. <br><br><br><br><br> |
| Précision | Constante de type Entier |   |

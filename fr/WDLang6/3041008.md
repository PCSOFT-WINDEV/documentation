
## gpwRecupInfoFichier (Fonction)

***En anglais : gpwGetFileInfo***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Récupère le répertoire des fichiers de données du groupware utilisateur.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Récupération du répertoire du fichier des utilisateurs
RepFicUtilisateur est une chaîne = gpwRecupInfoFichier(gpwInfoRepUtilisateur)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = gpwRecupInfoFichier(<Information>)`
---

**`<Résultat> : (Chaîne de caractères)`**

Information demandée.

**`<Information> : (Constante)`**

Type d'information recherchée :


|   |   |
| --- | --- |
| *gpwInfoRepDroit* | Répertoire du fichier de données des droits. |
| *gpwInfoRepUtilisateur* | Répertoire du fichier de données des utilisateurs. |

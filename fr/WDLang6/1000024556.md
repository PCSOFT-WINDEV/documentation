
## pdfElémentTexte (Type de variable)

***En anglais : pdfTextElement***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **pdfElémentTexte** permet de connaître toutes les caractéristiques d'un élément textuel dans une page PDF. Les caractéristiques de cet élément peuvent être connues à l'aide de différentes propriétés WLangage. Il est par exemple possible de connaître la position du texte et son contenu, pour faire des recherches.

Attention : Toutes les propriétés de la variable **pdfElémentTexte** sont en lecture seulement car le texte d'un fichier PDF n'est pas modifiable. 

**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonPDF est un pdfDocument = "test.pdf" 

POUR TOUTE PageDocument de MonPDF.Page
	POUR TOUT ElémentPage de PageDocument.ElémentTexte
		Trace(ElémentPage.Texte)
	FIN
FIN
```





<a name="NOTE0"></a>

## Remarques
<a name="NOTE0_1"></a>


### Propriétés spécifiques à la description des variables de type pdfElémentTexte
<a name="proprietes_specifiques_description_des_variables_type_pdfelementtexte_ELTPARAGRAPHE000058"></a>Les propriétés suivantes peuvent être utilisées pour manipuler un élément texte du fichier PDF :

| Nom de la propriété | Type manipulé | Effet |
| --- | --- | --- |
| Hauteur | Réel | Hauteur du rectangle indiquant la position et la taille de l'élément dans la page (en millimètres). <br><br>**Cette propriété est disponible en lecture seulement**. |
| Largeur | Réel | Largeur du rectangle indiquant la position et la taille de l'élément dans la page (en millimètres). <br><br>**Cette propriété est disponible en lecture seulement**. |
| PoliceNom | Chaîne de caractères | Nom de la police utilisée pour dessiner l'élément textuel.  <br><br>**Cette propriété est disponible en lecture seulement**.<br><br> |
| PoliceTaille | Réel | Taille de la police utilisée pour dessiner l'élément textuel (en point typographique, 72PPP).  <br><br>**Cette propriété est disponible en lecture seulement**.<br><br> |
| Texte | Chaîne de caractères | Texte de l'élément.<br><br>**Cette propriété est disponible en lecture seulement**. |
| X | Réel | Position X du rectangle indiquant la position et la taille de l'élément dans la page (en millimètres). <br><br>**Cette propriété est disponible en lecture seulement**. |
| Y | Réel | Position Y du rectangle indiquant la position et la taille de l'élément dans la page (en millimètres).  <br><br>**Cette propriété est disponible en lecture seulement**. |

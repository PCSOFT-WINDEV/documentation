## Comment créer une archive ?
<a name="comment_creer_une_archive_ELTTEXTE000255"></a>
Pour créer une archive :

1. Lancez WDZIP : sous le volet "Outils", dans le groupe "Utilitaires", cliquez sur "WDZip". La fenêtre de WDZip s'affiche :<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=WDZip%20-%20HC%20N%B0001.gif&type=thumb)


2. Cliquez sur le bouton "Nouveau" ou sélectionnez l'option "Fichier .. Nouvelle archive...".  Un sélecteur de fichiers s'affiche.

3. Saisissez le nom de l'archive à créer puis cliquez sur "Enregistrer". Par défaut, l'extension de l'archive créée sera ".WDZ". Pour créer une archive au format ZIP, CAB ou 7z, précisez l'extension dans le nom du fichier.

## Comment ajouter des fichiers dans une archive ?
<a name="comment_ajouter_des_fichiers_dans_une_archive_ELTTEXTE000279"></a>
Pour ajouter un ou plusieurs fichiers dans une archive : 

1. Ouvrez si nécessaire l'archive dans laquelle des fichiers doivent être ajoutés (cliquez sur le bouton "Ouvrir" ou sélectionnez l'option "Fichier .. Ouvrir une archive").

2. Cliquez sur le bouton "Ajouter" ou sélectionnez l'option "Action .. Ajouter un fichier".

3. Sélectionnez le ou les fichiers à ajouter. 
	Rappel : Pour sélectionner plusieurs fichiers, maintenez la touche Ctrl enfoncée lors de la sélection des fichiers.

4. Choisissez le mode de sauvegarde de l'arborescence du ou des fichiers à archiver. Les modes de sauvegarde de l'arborescence sont les suivants :

	- Conserver l'arborescence : le ou les fichiers seront archivés avec leur arborescence relative (*&lt;Répertoires...&gt; + &lt;Fichier&gt;*).

	- Inclure les fichiers système et cachés : permet d'archiver les fichiers systèmes et les fichiers cachés.

	- Stocker le chemin absolu : le ou les fichiers seront archivés avec leur arborescence complète (*&lt;Disque&gt; + &lt;Répertoires...&gt; + &lt;Fichier&gt;*)

	- Protéger par mot de passe : le ou les fichiers nécessiteront le mot de passe indiqué pour toute manipulation réalisée (remplacement, extraction, ...)




5. Indiquez si nécessaire le niveau de compression voulu (0 aucune compression, 9 compression maximale). 

6. Validez.




Remarques :

- Pour ajouter des fichiers dans une archive, il est également possible d'effectuer un "Drag and Drop" de l'explorateur vers **WDZip**. Dans ce cas, les fichiers ajoutés sont automatiquement archivés avec leur arborescence.

- Le pourcentage de compression des fichiers (taille des fichiers compressés par rapport à leur taille originale) est indiqué à côté du nom des fichiers.

- L'ajout de fichiers dans une archive peut être réalisé uniquement si l'archive est mono-partie. Pour effectuer cette opération sur une archive multi-parties, il suffit de regrouper les sous-archives pour obtenir une archive mono-partie.

## Comment ajouter des répertoires dans une archive ?
<a name="comment_ajouter_des_repertoires_dans_une_archive_ELTTEXTE000303"></a>
Pour ajouter un répertoire dans une archive :

1. Ouvrez si nécessaire l'archive dans laquelle des fichiers doivent être ajoutés (cliquez sur le bouton "Ouvrir" ou sélectionnez l'option "Fichier .. Ouvrir une archive").

2. Cliquez sur le bouton "Ajouter" ou sélectionnez l'option "Action .. Ajouter un fichier".

3. Sélectionnez le répertoire à ajouter puis cliquez sur le bouton "Ouvrir". 

4. Choisissez le mode de sauvegarde de l'arborescence du ou des fichiers présents dans le répertoire à archiver. Les modes de sauvegarde de l'arborescence sont les suivants :

	- Conserver l'arborescence : le ou les fichiers présents dans le répertoire seront archivés avec leur arborescence relative (*&lt;Répertoires...&gt; + &lt;Fichier&gt;*).

	- Inclure les fichiers système et cachés : permet d'archiver les fichiers systèmes et les fichiers cachés présents dans le répertoire.

	- Stocker le chemin absolu : le ou les fichiers présents dans le répertoire seront archivés avec leur arborescence complète (*&lt;Disque&gt; + &lt;Répertoires...&gt; + &lt;Fichier&gt;*)

	- Protéger par mot de passe : le ou les fichiers présents dans le répertoire nécessiteront le mot de passe indiqué pour toute manipulation réalisée (remplacement, extraction, ...)




5. Sélectionnez le ou les répertoires à ajouter. 
	**Rappel** : Pour sélectionner plusieurs répertoires, maintenez la touche Ctrl enfoncée lors de la sélection des répertoires.

6. Validez.




Remarques :

- Pour ajouter des répertoires dans une archive, il est également possible d'effectuer un "Drag and Drop" de l'explorateur vers **WDZip**. Dans ce cas, le ou les répertoires et leurs sous-dossiers sont sauvegardés avec leur arborescence relative.

- L'ajout de répertoires dans une archive peut être réalisé uniquement si l'archive est mono-partie. Pour effectuer cette opération sur une archive multi-parties, il suffit de regrouper les sous-archives pour obtenir une archive mono-partie.

## Découper et regrouper une archive
<a name="decouper_regrouper_une_archive_ELTTEXTE000327"></a>
Une archive multi-parties se présente sous la forme de plusieurs fichiers (ou sous-archives) de taille identique. La taille de chaque sous-archive est fixée lors de la création de l'archive multi-parties. Pour plus de détails, consultez [Archive mono-partie / multi-partie](../WDLang3/3082015.md).

A partir d'une archive multi-parties, il est uniquement possible de :

- Regrouper les différentes parties,

- Extraire les fichiers.




**Découper une archive en plusieurs sous-archives** : 

1. Ouvrez si nécessaire l'archive à découper (cliquez sur le bouton "Ouvrir" ou sélectionnez l'option "Fichier .. Ouvrir une archive").

2. Cliquez sur le bouton "Découper" ou sélectionnez l'option "Action .. Découper l'archive".

3. La fenêtre de paramétrage du découpage de l'archive s'affiche.

4. Choisissez la taille des sous-archives (la taille proposée va de 360 ko à 64 Mo). 
	Remarque : Si vous sélectionnez "Autre", saisissez la taille de l'archive en kilo-octets ou méga-octets selon l'option choisie. 
	Attention : La taille des sous archives ne peut pas être inférieure à 360Ko.

5. Si le fichier "WDZ" ou "ZIP" original doit être supprimé après le découpage de l'archive, cochez la case "Supprimer l'archive originale après le découpage".

6. Validez.




Pour les archives au format WDZ : La première sous-archive est un fichier de type ".WDZ". Les sous-archives suivantes sont des fichiers numérotés de 1 à N grâce à leur extension (".001", ".002", ..., ".NNN").

Pour les archives au format ZIP : La première sous-archive est un fichier de type ".ZIP". Les sous-archives suivantes sont des fichiers numérotés de 1 à N grâce à leur extension (".Z01", ".Z02", ..., ".ZNN").

**Remarque** : Certaines fonctions d'archivage sont utilisables uniquement sur des archives mono-partie (ajout de fichiers, suppression de fichiers, création d'un exécutable auto-extractible, ...). Pour pouvoir utiliser ces fonctionnalités, vous devez [regrouper l'archive en une archive mono-partie](../WDZip/3527003.md).
<a name="NOTE5_2"></a>
Une archive mono-partie se présente sous la forme d'un seul fichier de type ".WDZ" : un fichier archive regroupe tous les fichiers compressés.

L'archive occupe la place nécessaire pour contenir tous les fichiers compressés.

**Pour regrouper une archive multi-partie en une archive mono-partie** : 

1. Ouvrez si nécessaire le premier fichier de l'archive multi-partie (bouton "Ouvrir" ou option "Fichier .. Ouvrir une archive"). Cette archive porte l'extension ".WDZ" ou ".ZIP".

2. Sélectionnez l'option "Action .. Regrouper les archives".

3. L'archive regroupée est maintenant mono-partie.

## Archive auto-extractible
<a name="archive_autoextractible_ELTTEXTE000357"></a>


### Créer une archive auto-extractible
<a name="creer_une_archive_autoextractible_ELTPARAGRAPHE000168"></a>Une archive auto-extractible est une archive sous forme d'un exécutable (fichier ".EXE"). Lors du lancement de cet exécutable, les fichiers de l'archive sont automatiquement décompressés sur le poste en cours.

La création d'une archive auto-extractible ne peut être réalisée que si :

- L'archive existe.

- L'archive est accessible en lecture et en écriture.

- L'archive est mono-partie.




**Pour créer une archive auto-extractible** :

1. Ouvrez si nécessaire l'archive à convertir en archive auto-extractible (clic sur le bouton "Ouvrir" ou sélection de l'option "Fichier .. Ouvrir une archive").

2. Cliquez sur le bouton "Exécutable" ou sélectionnez l'option "Action .. Créer archive auto-extractible".

3. Spécifiez dans quel répertoire les fichiers doivent être décompressés lors du lancement de l'exécutable.

4. Indiquez si une boîte de dialogue doit être affichée lors de l'extraction. Cette boîte de dialogue permet par exemple à l'utilisateur de choisir le chemin d'extraction.

5. Validez. L'archive auto-extractible est créée.

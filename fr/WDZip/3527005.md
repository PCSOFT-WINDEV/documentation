## Présentation
<a name="presentation_ELTTEXTE000126"></a>
WDZip permet de compresser et de regrouper vos fichiers sous forme d'archives.

Une archive est un fichier physique de type ".WDZ", ".ZIP, ".RAR", ".CAB" ou ".7z". Ce fichier regroupe plusieurs fichiers de tous types (".pdf", ".fic", ".mmo", ...) automatiquement compressés.

WDZip gère les archives de type ".7z".Grâce à une compression optimisée, vos fichiers occupent moins d'espace disque et sont facilement transmissibles sur différents supports (disquettes, CD, Internet, ...).
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ARCHIVE.gif)
WDZip vous permet par exemple de :

- sauvegarder régulièrement vos fichiers tout en réduisant la taille de stockage,

- faciliter la transmission de vos fichiers sur Internet / Intranet,

- créer des archives auto-extractibles,

- etc.




**Rappel** : Tous les formats d'archive sont également manipulables grâce aux [fonctions ZIP](../WDLang3/3082006.md) du WLangage.

WDZip peut être lancé :

- directement à partir d'un fichier archive (double-clic sur un fichier ".WDZ" par exemple)

- depuis WINDEV, WEBDEV ou WINDEV Mobile : sous le volet "Outils", dans le groupe "Utilitaires", cliquez sur "WDZip".

- en lançant directement le programme "WDZip.exe" présent dans le répertoire de WINDEV, WEBDEV ou WINDEV Mobile. 




WDZip est un outil non redistribuable. WDZip ne peut pas être installé avec les applications développées avec WINDEV, WINDEV Mobile ou WEBDEV.

Pour utiliser WDZip, WINDEV, WINDEV Mobile ou WEBDEV doit obligatoirement être installé sur le poste en cours.

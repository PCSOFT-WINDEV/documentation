
## WDZip : Manipuler les fichiers d'une archive
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000137"></a>
Les fichiers compressés et rassemblés dans une archive peuvent être manipulés avec **WDZip**.

Les fichiers contenus dans l'archive peuvent être :

- [Extraits (décompressés) vers un répertoire de destination](#NOTE2_1).

- [Supprimés de l'archive](#NOTE3_1).

- [Recherchés, si l'archive comporte beaucoup de fichiers par exemple](#NOTE4_1).






<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Extraction de fichiers d'une archive
<a name="extraction_fichiers_une_archive_ELTTEXTE000161"></a>
Extraire un fichier d'une archive consiste à décompresser le fichier contenu dans l'archive vers un répertoire de destination. Le fichier n'est pas supprimé de l'archive.

**Pour extraire un ou plusieurs fichiers contenus dans une archive** :

1. Ouvrez si nécessaire l'archive ".wdz", ".zip", ".cab" ou ".7z" contenant les fichiers à extraire (option "Fichier .. Ouvrir une archive" de WDZIP).

2. Cliquez sur le bouton "Extraire" (ou "Extraire tout") ou sélectionnez l'option de menu "Action .. Extraire le fichier sélectionné ..." (ou "Action .. Extraire tous les fichiers ..."). La fenêtre de paramétrage de l'extraction s'affiche.

3. Indiquez le répertoire dans lequel le ou les fichiers seront décompressés.

4. Indiquez si les fichiers doivent être extraits ***avec leur arborescence*** : Les fichiers seront décompressés dans le répertoire spécifié auquel s'ajoute l'arborescence mémorisée du fichier (le disque n'est pas conservé). Si les répertoires n'existent plus, ils sont automatiquement recréés.
	Si cette option n'est pas cochée, les fichiers seront tous décompressés dans le répertoire spécifié.
	**Remarque** : Si la sauvegarde des chemins des fichiers n'a pas été précisée lors de la création de l'archive, les fichiers seront automatiquement extraits sans leur arborescence.

5. Validez.




**Attention** : Pour pouvoir extraire un fichier dans une archive, il est nécessaire que l'archive soit [mono-partie](../WDLang3/3082015.md) et accessible en lecture/écriture.

## Suppression de fichiers d'une archive
<a name="suppression_fichiers_une_archive_ELTTEXTE000185"></a>
Supprimer un fichier d'une archive consiste à l'effacer de la liste des fichiers contenus dans l'archive. Ce fichier n'est pas décompressé.

**Pour supprimer un ou plusieurs fichiers contenus dans une archive** :

1. Ouvrez si nécessaire l'archive ".wdz", ".zip", ".cab" ou ".7z" contenant les fichiers à supprimer. Sélectionnez l'option "Fichier .. Ouvrir une archive" de WDZIP. 

2. Sélectionnez le ou les fichiers à supprimer.
	**Rappel** : Pour sélectionner plusieurs fichiers, maintenez la touche Ctrl enfoncée lors de la sélection des fichiers.

3. Cliquez sur le bouton "Supprimer" ou sélectionnez l'option "Action .. Supprimer un fichier".
	**Remarque** : Si aucun fichier n'est sélectionné, WDZIP vous proposera de supprimer automatiquement tous les fichiers de l'archive.

4. Confirmez la suppression du ou des fichiers sélectionnés.




**Attention** : Pour pouvoir supprimer un fichier dans une archive, il est nécessaire que l'archive soit [mono-partie](../WDLang3/3082015.md) et accessible en écriture.

## Recherche de fichiers dans une archive
<a name="recherche_fichiers_dans_une_archive_ELTTEXTE000209"></a>
Pour rechercher un fichier dans une archive : 

1. Ouvrez si nécessaire l'archive ".wdz", ".zip", ".cab" ou ".7z" contenant les fichiers à rechercher (sélectionnez l'option "Fichier .. Ouvrir une archive" de WDZip).

2. Cliquez sur le bouton "Rechercher" ou sélectionnez l'option "Action .. Rechercher un fichier". Une fenêtre de recherche s'affiche.

3. Dans cette fenêtre de recherche, indiquez le nom ou le chemin complet du fichier à rechercher et validez. Le fichier recherché est automatiquement sélectionné.


**Attention** : Pour pouvoir rechercher un fichier dans une archive, il est nécessaire que l'archive soit [mono-partie](../WDLang3/3082015.md) et accessible en lecture/écriture.

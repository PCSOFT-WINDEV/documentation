## Présentation
<a name="presentation_ELTTEXTE000133"></a>
Votre application est terminée. Nous avons déjà vu comment s'assurer du bon fonctionnement de l'application grâce aux fonctionnalités suivantes :  

- le débogueur, pour comprendre les erreurs de fonctionnement ([Leçon 2.7. Le débogueur](../TutoWD/1410087580.md)), 

- les tests automatiques, pour éviter les régressions ([Leçon 4.12. Surveillez l'évolution de vos applications](../TutoWD/1410087527.md)). 




Vous voulez déployer cette application chez vos clients.

Avez-vous pensé à utiliser les outils de WINDEV pour optimiser votre application ? WINDEV propose une série d'outils et de fonctionnalités permettant d'optimiser rapidement votre application et d'éviter simplement des "bugs" grossiers. 

Cette partie présente en détail ces outils et leur utilisation.

Pour mieux manipuler ces fonctionnalités, nous avons préparé un projet à optimiser !
<a name="NOTE2_2"></a>


### Ouverture du projet
<a name="ouverture_projet_ELTPARAGRAPHE000045"></a>

- Lancez WINDEV 28 (si ce n'est déjà fait). 




- Si nécessaire affichez la page d'accueil de WINDEV (Ctrl + &lt;). 




- Ouvrez le projet "WD Optimisation".
	Pour cela, dans la page d'accueil, cliquez sur "Tutoriel" puis dans la zone "Partie 6 - Optimiser un projet", double-cliquez sur "Optimiser un projet".

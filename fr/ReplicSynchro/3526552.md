## ReplicSynchro et ReplicAdmin
<a name="replicsynchro_replicadmin_ELTTEXTE000106"></a>
ReplicSynchro et ReplicAdmin sont deux outils permettant de réaliser la synchronisation de données d'applications :

- **ReplicSynchro** permet uniquement de réaliser la synchronisation. C'est l'outil à utiliser pour effectuer une simple synchronisation des données.

- **ReplicAdmin** permet de configurer les synchronisations et d'effectuer la synchronisation. C'est l'administrateur de réplications.

## ReplicAdmin
<a name="replicadmin_ELTTEXTE000130"></a>


### Lancement de ReplicAdmin
<a name="lancement_replicadmin_ELTPARAGRAPHE000022"></a>ReplicAdmin peut être exécuté directement ou depuis l'outil ReplicSynchro.


<a name="NOTE2_2"></a>


### Utilisation de ReplicAdmin
<a name="utilisation_replicadmin_ELTPARAGRAPHE000030"></a>ReplicAdmin liste les différentes synchronisations disponibles sur le poste en cours, avec la date de dernière synchronisation.

L'icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Replication.gif) permet de lancer la réplication sélectionnée.

L'icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Connexion_directe.gif) /  ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Connexion_directe1.gif) permet de passer du mode connecté au mode déconnecté.

L'icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ICO_Cr%E9er_cpt.gif) permet d'ajouter une nouvelle réplication.

L'icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Supprimer.gif) permet de supprimer la réplication sélectionnée dans la liste.


<a name="NOTE2_3"></a>


### Condition d'utilisation
<a name="condition_utilisation_ELTPARAGRAPHE000056"></a>ReplicAdmin est un outil redistribuable.

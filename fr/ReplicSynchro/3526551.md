## ReplicSynchro et ReplicAdmin
<a name="replicsynchro_replicadmin_ELTTEXTE000158"></a>
ReplicSynchro et ReplicAdmin sont deux outils permettant de réaliser la synchronisation de données d'applications :

- **ReplicSynchro** permet uniquement de réaliser la synchronisation. C'est l'outil à utiliser pour effectuer une simple synchronisation des données.

- **ReplicAdmin** permet de configurer les synchronisations et d'effectuer la synchronisation. C'est l'administrateur de réplications.

## ReplicSynchro
<a name="replicsynchro_ELTTEXTE000182"></a>


### Lancement de ReplicSynchro
<a name="lancement_replicsynchro_ELTPARAGRAPHE000021"></a>ReplicSynchro peut être exécuté par le raccourci "Démarrer .. Outils PC SOFT .. ReplicSynchro".
<a name="NOTE2_2"></a>


### Utilisation de ReplicSynchro
<a name="utilisation_replicsynchro_ELTPARAGRAPHE000028"></a>ReplicSynchro liste les différentes réplications disponibles sur le poste en cours, avec la date de la dernière synchronisation.

L'icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Replication.gif)
 permet de lancer la réplication sélectionnée.

L'icône ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Admin.gif)
 permet de lancer l'administrateur de réplication.
<a name="NOTE2_3"></a>

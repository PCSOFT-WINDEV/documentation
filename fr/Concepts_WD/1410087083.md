<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Protocoles POP3 ou IMAP et SMTP
<a name="protocoles_pop3_imap_smtp_ELTTEXTE000134"></a>
Ces protocoles sont les standards Internet pour la gestion des emails. Ils sont normalisés par les documents RFC 1939, 3501 et 5321. Ces protocoles vous permettent de dialoguer directement avec le serveur, disponible chez votre fournisseur d'accès. 

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=P6-Gestion%20des%20emails%20Protocoles%20POP3%20ou%20IMAP%20et%20SMTP.gif)


- POP3 (Post Office Protocol) est un protocole de lecture d'emails.

- IMAP (Internet Message Access Protocol) est un protocole de lecture d'emails à distance.

- SMTP (Simple Mail Transfer Protocol) est un protocole d'envoi d'emails.




<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## API "Simple Mail API (appelé aussi SMAPI ou Simple MAPI)"
<a name="api_simple_mail_api_appele_aussi_smapi_simple_mapi_ELTTEXTE000158"></a>

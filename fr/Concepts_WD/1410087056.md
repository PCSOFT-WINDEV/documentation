L'éditeur d'analyses permet de décrire la structure des fichiers de données.

Selon les informations saisies sous l'éditeur d'analyses, différents fichiers sont créés physiquement.

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=creation_fichier_HFSQL.svg)
**Remarque** : Ce schéma présente uniquement les principaux fichiers créés. D'autres fichiers spécifiques peuvent être créés si le fichier de données utilise la journalisation, les transactions ou encore la réplication.


## 6. Fenêtre interne
			

| [Page précédente](../Concepts_WD/1410086971.md) | [Sommaire](../Concepts_WD/1410087098.md) | [Page suivante](../Concepts_WD/1410086973.md) |

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Le champ Fenêtre interne permet d'inclure une fenêtre (et son code) dans une autre fenêtre. A l'exécution, la fenêtre interne sera dynamiquement fusionnée à la fenêtre hôte.


|   |   |
| --- | --- |
| **1. Création d'une fenêtre interne**.<br><br>Pour créer une fenêtre interne :<br><br>- Cliquez sur ![Créer un élément](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ICO_Nouveau.gif)<br> parmi les boutons d'accès rapide. <br><br>- La fenêtre de création d'un nouvel élément s'affiche : cliquez sur "Fenêtre" puis sur "Fenêtre interne".<br><br><br> |
| Une fenêtre interne est une fenêtre spécifique qui ne comporte ni barre de titre, ni menu.<br>Dans cette fenêtre, tous les types de champs peuvent être utilisés.<br><br>Dans l'exemple, une fenêtre interne de sélection de fichiers est définie. | <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=P1_Fen%EAtre%20interne%20-%20HC%20N%B0002.gif)<br> |




|   |   |
| --- | --- |
| **2. Utilisation d'une fenêtre interne**. |
| Pour utiliser une fenêtre interne :<br><br>- Créez un champ de type fenêtre interne.<br><br>- Dans la fenêtre de description du champ, sélectionnez la fenêtre interne à utiliser.<br><br>- Validez la fenêtre de description du champ.<br><br><br> | <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=P1_Fen%EAtre%20interne%20-%20HC%20N%B0001.gif&type=thumb)<br> |
| **Remarques** :<br><br>- Il est possible de modifier par programmation la fenêtre interne utilisée dans le champ "Fenêtre interne".<br><br>- Limitations : La zone d'accueil est rectangulaire et aucune surcharge n'est possible. Pour réaliser des surcharges, il est conseillé d'utiliser des modèles de champs.<br><br><br> |

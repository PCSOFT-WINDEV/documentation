<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
HFSQL Client/Serveur est un puissant système de gestion de bases de données relationnelles (SGBD/R) en mode Client/Serveur.

Les caractéristiques de HFSQL Client/Serveur sont les suivantes :

- Une application HFSQL Client/Serveur peut être exécutée sur différents postes utilisateur (appelés machines clientes) qui communiquent à travers le réseau avec le serveur de base de données.

- Les fichiers de données sont présents sur un poste serveur. Seul le serveur accède physiquement aux fichiers de données.

- L'ensemble des traitements (requête, lecture/ajout dans un fichier de données, ...) est réalisé sur le serveur.

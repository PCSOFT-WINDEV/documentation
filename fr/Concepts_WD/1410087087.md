
## 5. Les Webservices

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
WINDEV vous permet d'utiliser très simplement des Webservices (ou services Web XML). Un Webservice est une application disponible sur le Web. WINDEV permet d'accéder à des Webservices SOAP (Simple Object Access Protocol) comme à des Webservices REST (Representational State Transfer).

Toute application WINDEV peut utiliser un Webservice existant. 

Pour utiliser un Webservice SOAP, il suffit de connaître l'adresse du Webservice et d'importer sa description dans l'application. 

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=P6-Les%20Webservices.gif)


Pour utiliser un Webservice REST, il suffit de connaître les caractéristiques du Webservice et d'utiliser la fonction [RESTEnvoie](../WDLang3/1000021476.md). 

**Remarque** : il est également possible d'exposer une ou plusieurs collections de procédures d'une application WINDEV sous la forme d'un Webservice SOAP ou REST. Il faut pour cela : 

- créer une configuration de projet dédiée à cet usage, 

- dans le cas d'un Webservice REST, créer une description du Webservice REST,

- générer le Webservice, 

- le déployer sur un Serveur d'Application WEBDEV.

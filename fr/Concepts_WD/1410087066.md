
## 4. Groupware utilisateur

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Le Groupware utilisateur permet d'inclure simplement une gestion des utilisateurs et de leurs droits dans une application WINDEV.

Deux types d'utilisateurs sont pris en compte :

- utilisateur simple, utilisant directement l'application.

- superviseur, pouvant configurer les utilisateurs et leurs droits.

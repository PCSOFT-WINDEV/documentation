


## Centre de Suivi de Projets : Statistiques
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000108"></a>
Lors de la gestion d'un projet, il est nécessaire de suivre certains indicateurs pour comprendre l'évolution du projet, ses dépassements, ...

Le Centre de Suivi de Projets propose une fenêtre de statistiques complète, permettant d'afficher :

- le temps passé par projet pour un ou plusieurs intervenants,

- le temps passé par intervenant sur un projet,

- le temps passé par tâche pour un projet et un intervenant,

- le temps passé par tâche d'interruption, 

- une courbe d'avancement (ou "Burn down"). 




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Afficher les statistiques
<a name="afficher_les_statistiques_ELTTEXTE000132"></a>
Pour afficher les statistiques du Centre de Suivi de Projets : 

1. Affichez l'onglet "Statistiques" (ou créez un nouvel onglet de type "Statistiques"). 

2. Dans le ruban, sélectionnez :

	- le type de statistiques,

	- le projet et les intervenants concernés,

	- la liste d'exigences (pour la courbe d'avancement),

	- le type de tâches (pour la courbe d'avancement),

	- la période,

	- les options d'affichage (l'échelle utilisée, l'affichage de la légende, ...).




3. Affichez le graphe (cliquez sur l'image affichée par défaut ou sur le bouton "Actualiser").




**Remarque** : Pour afficher une courbe d'avancement, il est nécessaire d'activer la collecte quotidienne des données sur le serveur (option proposée par le Centre de Suivi de Projets lors de la sélection de ce type de statistiques). La courbe d'avancement sera alors réalisée à partir de la date d'activation de cette collecte.


### Courbe d'avancement (Burn down)
<a name="courbe_avancement_burn_down_ELTPARAGRAPHE000052"></a>La courbe d'avancement est constituée d'une courbe et de deux droites de tendance.

- La courbe représente la charge de travail planifiée. Le nombre de jours planifiés évolue au fur et à mesure du projet. La vitesse réelle de développement est impactée par la "réalité" (par exemple: réunions, séminaires, formation, débogage d'anciennes applications, tâches oubliées, absences, ...)

- La première droite de tendance affichée représente la tendance de la courbe. Cette tendance indique donc la date théorique de fin du projet (ou de l'itération).

- La deuxième droite de tendance représente la tendance en incluant l'expérience des projets précédents. Dans le groupe "Affichage" de l'onglet "Statistiques", il est possible de choisir : 

	- l'exigence ou la liste d'exigences pour l'itération précédente. 

	- la vitesse prise en compte pour l'itération précédente.


## Centre de Suivi de Projets : Gestion des Tags
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000180"></a>
Les tags permettent d'associer des mots-clé aux incidents présents dans le Centre de Suivi de Projets. Grâce aux tags, il est ensuite possible de filtrer, classer ou encore réaliser des statistiques sur les différents incidents associés à un projet, une exigence, ... 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000204"></a>


### Gérer les tags
<a name="gerer_les_tags_ELTPARAGRAPHE000016"></a>La gestion des différents tags est réalisée depuis le volet "Incidents" du Centre de Suivi de Projets. 

Pour afficher la fenêtre de gestion des tags depuis le Centre de Suivi de projets :

1. Affichez si nécessaire un onglet de type "Incidents". 

2. Dans le ruban, dans le groupe "Gestion", cliquez sur "Tags". <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CC_Suivi_Tag%20-%20HC%20N%B0001%201.gif&type=thumb)


3. La fenêtre de gestion des tags s'affiche. <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CC_Suivi_Tag%20-%20HC%20N%B0003.gif&type=thumb)
Cette fenêtre permet de : 

	- définir les tags : création, modification, ...

	- voir la répartition des incidents par tag. 








### Créer, modifier, supprimer un tag
<a name="creer_modifier_supprimer_tag_ELTPARAGRAPHE000033"></a>**Pour créer un tag** : 

1. Affichez si nécessaire un onglet de type "Incidents". 

2. Dans le ruban, dans le groupe "Gestion", cliquez sur "Tags". 

3. Dans l'onglet "Liste des tags" de la fenêtre qui s'affiche, cliquez sur "+". La fenêtre de création d'un tag s'affiche : <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CC_Suivi_Tag%20-%20HC%20N%B0002.gif)
Dans cette fenêtre, vous devez indiquer : 

	- Le nom du tag (obligatoire). Ce nom sera utilisé pour identifier le tag. Ce nom ne doit pas contenir d'espace. 

	- La description du tag (optionnel) : Permet de décrire les conditions d'utilisation du tag. 

	- La visibilité du tag : 

		- Si le tag est public, il sera visible par tous les intervenants. 

		- Si le tag est privé, il sera visible uniquement par son propriétaire. 




- La couleur du tag : Permet d'identifier rapidement les éléments associés à un tag. Ce paramètre est obligatoire.  

4. Validez. Le nouveau tag apparaît immédiatement dans la liste des tags. 




**Pour modifier un tag** : 

1. Affichez si nécessaire un onglet de type "Incidents". 

2. Dans le ruban, dans le groupe "Gestion", cliquez sur "Tags". 

3. Dans l'onglet "Liste des tags" de la fenêtre qui s'affiche, sélectionnez un tag et cliquez sur le bouton de modification ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CC_Suivi_Tag%20-%20HC%20N%B0003%201.gif). La fenêtre de description d'un tag s'affiche. 

4. Dans cette fenêtre, il est possible de modifier  : 

	- Pour un tag privé, les différentes caractéristiques du tag. Le tag peut être transformé en tag public. Il est également possible de changer le propriétaire d'un tag privé (attention : dans ce cas, le tag n'apparaîtra plus dans la liste des tags). 

	- Pour un tag public, les différentes caractéristiques du tag. Cependant, un tag public ne peut pas être modifié en tag privé. 




5. Validez. La modification est automatiquement prise en compte.  




**Pour supprimer un tag** : 

1. Affichez si nécessaire un onglet de type "Incidents". 

2. Dans le ruban, dans le groupe "Gestion", cliquez sur "Tags". 

3. Dans l'onglet "Liste des tags" de la fenêtre qui s'affiche, sélectionnez un tag et cliquez sur "-". 

4. Une confirmation de la suppression est demandée puis le tag est supprimé. Si le tag est associé à des incidents, cette association sera également supprimée. 







### Associer un tag à un incident
<a name="associer_tag_incident_ELTPARAGRAPHE000074"></a>Il existe plusieurs façons d'associer un tag à un incident : 

- depuis la fiche de l'incident. <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CC_Suivi_Tag%20-%20HC%20N%B0004.gif&type=thumb)


- depuis la liste des incidents. <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CC_Suivi_Tag%20-%20HC%20N%B0001.gif&type=thumb)


- depuis la fenêtre de création d'un incident via la messagerie WDBal. 
	Remarque : Selon le texte du message correspondant à l'incident, des tags sont automatiquement proposés.




Remarque : Tous les ajouts ou suppression d'un tag public à un incident sont notés dans l'historique de l'incident. 




### Filtrer les incidents en utilisant les tags
<a name="filtrer_les_incidents_utilisant_les_tags_ELTPARAGRAPHE000092"></a>Pour filtrer les incidents correspondant à un tag : 

1. Affichez si nécessaire un onglet de type "Incidents". 

2. Dans la zone "Filtrer" affichée au dessus de la liste des incidents : <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CC_Suivi_Tag%20-%20HC%20N%B0006.gif)


	- Indiquez le ou les tag(s) à prendre en compte. 

- Indiquez le mode de filtre à utiliser en cliquant sur le lien.

3. Le résultat est immédiatement affiché. 







### Voir la répartition des incidents par tag
<a name="voir_repartition_des_incidents_par_tag_ELTPARAGRAPHE000106"></a>Pour voir la répartition des incidents par tag : 

1. Affichez si nécessaire un onglet de type "Incidents". 

2. Dans le ruban, dans le groupe "Gestion", cliquez sur "Tags". 

3. Dans la fenêtre qui s'affiche, affichez l'onglet "Répartition des incidents par tag". <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CC_Suivi_Tag%20-%20HC%20N%B0005.gif&type=thumb)
Cette fenêtre permet de consulter la répartition des incidents par tag, selon les critères de tri définis. Pour chaque tag affiché, un menu contextuel permet d'effectuer les opérations suivantes : 

	- Masquer le tag. 

	- Modifier le tag. 

	- Afficher la liste des incidents. Dans ce cas, un nouvel onglet "Incidents" est ouvert dans le Centre de Suivi de projets, contenant uniquement les incidents correspondant au tag sélectionné. 





Cette fenêtre permet de réaliser des filtres sur les tags affichés : 

- Affectation des tags, 

- Auteur

- Lot d'incidents, 

- Type d'incident, 

- Etat, 

- Tags privés ou publics, 

- Incidents sans tag.

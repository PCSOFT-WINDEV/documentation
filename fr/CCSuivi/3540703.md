


## Centre de Suivi de projets : Gestion des intervenants
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000131"></a>
Les intervenants correspondent aux personnes qui peuvent participer à un ou plusieurs projets.

La gestion des intervenants permet de :

- créer un intervenant.

- modifier un intervenant (par exemple en lui donnant les droits d'administrateur)

- supprimer un intervenant.




Ces opérations peuvent être réalisées uniquement par un administrateur (par défaut, le login et le mot de passe correspond à "ADMIN").

La manipulation des intervenants est réalisée à partir de la liste des intervenants, affichée grâce à l'option "Gestion .. Gestion des intervenants".
Pour manipuler les intervenants, déroulez ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CC_Suivi_Configuration%20-%20HC%20N%B0001.gif) puis sélectionnez "Intervenants".

## Gestion des intervenants
<a name="gestion_des_intervenants_ELTTEXTE000155"></a>


### Ajouter un intervenant
<a name="ajouter_intervenant_ELTPARAGRAPHE000032"></a>Pour ajouter un intervenant :

1. Déroulez ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CC_Suivi_Configuration%20-%20HC%20N%B0001.gif) puis sélectionnez "Intervenants" pour afficher la liste des intervenants.

2. Cliquez sur le bouton "Nouvel intervenant".

3. Remplissez les différentes informations de la fenêtre de création de l'intervenant : <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Ajout_Intervenant.gif&type=thumb)
Les informations importantes sont les suivantes :

	- Login et mot de passe : permet à l'intervenant de s'identifier lors du lancement du Centre de Contrôle.

	- Cet intervenant dispose des droits "Administrateur" : permet de donner les droits d'administrateur à un intervenant. Seul un administrateur a accès à cette option. Pour les autres utilisateurs, cette option est grisée.

	- le type d'intervenant : fixe, en déplacement ou provenant d'une société extérieure. 




4. Validez.




**Remarque** : Lors de l'identification d'un nouvel utilisateur dans la fenêtre de login de WINDEV, WEBDEV ou WINDEV Mobile, il est possible d'ajouter automatiquement ce nouvel utilisateur.
<a name="NOTE2_2"></a>


### Modifier un intervenant
<a name="modifier_intervenant_ELTPARAGRAPHE000057"></a>Pour modifier un intervenant :

1. Déroulez ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CC_Suivi_Configuration%20-%20HC%20N%B0001.gif) puis sélectionnez "Intervenants" pour afficher la liste des intervenants.

2. Sélectionnez l'intervenant à modifier.

3. Cliquez sur le bouton "Afficher l'intervenant".

4. Modifiez l'information voulue.

5. Validez.




**Remarque** : Seul un administrateur ou l'intervenant lui-même peut modifier sa fiche.
<a name="NOTE2_3"></a>


### Supprimer un intervenant
<a name="supprimer_intervenant_ELTPARAGRAPHE000074"></a>Pour supprimer un intervenant :

1. Déroulez ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CC_Suivi_Configuration%20-%20HC%20N%B0001.gif) puis sélectionnez "Intervenants" pour afficher la liste des intervenants.

2. Sélectionnez l'intervenant à supprimer.

3. Cliquez sur le bouton "Supprimer l'intervenant".




**Remarque** : **Seul un administrateur peut supprimer un intervenant**.

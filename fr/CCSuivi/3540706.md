


## Centre de Suivi de Projets : Configuration
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000194"></a>
Pour que le planning et le suivi de projets soient conformes à la réalité, il est nécessaire que chaque intervenant du projet saisisse la configuration de son temps de travail.

Cette configuration prend en compte :

- les jours ouvrés et les heures ouvrées.

- les congés.

- les périodes de télétravail.




Pour configurer ces paramètres dans le Centre de Suivi de Projets, déroulez ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CC_Suivi_Configuration%20-%20HC%20N%B0001.gif)
 et sélectionnez "Horaires et congés".

**Remarque** : Si l'aide à la gestion du temps est activée, la configuration des jours ouvrés et des heures ouvrées n'est pas prise en compte lors du calcul du temps passé sur une tâche. Ces informations sont prises en compte uniquement lors de l'établissement du planning prévisionnel, pour le calcul des délais. Pour plus de détails, consultez [Utiliser le planning et la gestion des tâches](../CCSuivi/3540713.md).

## Configuration des jours et heures ouvrés
<a name="configuration_des_jours_heures_ouvres_ELTTEXTE000218"></a>


### Configuration des jours ouvrés
<a name="configuration_des_jours_ouvres_ELTPARAGRAPHE000035"></a>
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CC_Suivi_Configuration%20-%20HC%20N%B0002.gif&type=thumb)
La configuration des jours ouvrés consiste à définir les jours de travail (au niveau des demi-journées).

Pour chaque jour de la semaine, il est possible de définir si le matin et / ou l'après-midi est travaillé.
<a name="NOTE2_2"></a>


### Configuration des heures ouvrées
<a name="configuration_des_heures_ouvrees_ELTPARAGRAPHE000045"></a>La configuration des heures ouvrées permet de définir pour chaque demi-journée de travail, quelles sont les heures travaillées.

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CC_Suivi_Configuration%20-%20HC%20N%B0002_1.gif)
**Remarques** :

- Lors du repas de midi, aucune action spécifique ne doit être réalisée : la tâche ne doit pas être arrêtée

- Lorsque la tâche sera arrêtée (bloquée ou validée), les configurations données pour les jours ouvrés et les heures ouvrées seront automatiquement prises en compte.

## Configuration des congés et des jours fériés
<a name="configuration_des_conges_des_jours_feries_ELTTEXTE000248"></a>
La configuration des congés permet de définir les périodes non travaillées. 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CC_Suivi_Configuration%20-%20HC%20N%B0003.gif&type=thumb)
Le planning prendra en compte ces données.

Deux périodes de congés peuvent être visualisées :

- **Congés personnels** : ce sont les congés saisis par l'intervenant.

- **Congés suggérés** : ce sont les congés proposés par l'administrateur (cette option est réservée aux administrateurs). En effet, pour assurer le bon déroulement d'un projet, il est souvent nécessaire que les différents participants partent en congés en même temps (ou dans des périodes proches). Le Centre de Suivi de Projets permet donc à l'administrateur de suggérer des périodes de congés pour tous les intervenants.




**Pour ajouter une période de congés personnels** : 

1. Cliquez si nécessaire sur le bouton "Congés personnels".

2. Saisissez les dates et horaires du premier jour et du dernier jour de la période. Vous pouvez directement cliquer sur les différentes dates du calendrier pour saisir votre période de congés. Le nombre de jours ouvrés correspondant est automatiquement calculé.

3. Saisissez les raisons de vos congés.

4. Cliquez sur le bouton "Ajouter le congé personnel".




**Pour supprimer une période de congés personnels**, il suffit de sélectionner la période et de cliquer sur le bouton "Supprimer le congé personnel".

**Pour ajouter une période de congés suggérés** : 

1. Cliquez si nécessaire sur le bouton "Congés suggérés".

2. Saisissez les dates et horaires du premier jour et du dernier jour de la période. Vous pouvez directement cliquer sur les différentes dates du calendrier pour saisir la période de congés. Le nombre de jours ouvrés correspondant est automatiquement calculé.

3. Saisissez les raisons des congés suggérés.

4. Cliquez sur le bouton "Ajouter le congé suggéré".




**Pour supprimer une période de congés suggérés**, il suffit de sélectionner la période et de cliquer sur le bouton "Supprimer le congé suggéré".

**Remarques** : 

- Un intervenant peut uniquement saisir ses congés personnels. Un administrateur dispose d'un sélecteur lui permettant de sélectionner dans quel type de congés sa saisie va être effectuée.

- Les périodes de congés sont prises en compte par l'application de messagerie lors de l'écriture d'un message : lors de la saisie du nom d'un ou de plusieurs intervenants, la liste des destinataires en congés sera affichée. 

- L'administrateur a la possibilité de voir et de modifier les périodes de congés d'un intervenant. Il suffit de cliquer sur le lien correspondant au nom de l'intervenant en haut à droite de la fenêtre et de choisir l'intervenant voulu. Les informations concernant les congés de l'intervenant sont affichées. 



<a name="NOTE4_2"></a>


### Jours fériés
<a name="jours_feries_ELTPARAGRAPHE000117"></a>L'option "Traiter automatiquement les jours fériés (si cette option n'est pas cochée, vous devrez saisir les jours fériés manuellement comme des congés)" permet de prendre en compte automatiquement les jours fériés dans le planning. Les jours fériés pris en compte sont ceux du calendrier français.

Si cette option n'est pas cochée, il est nécessaire de saisir manuellement les jours fériés (comme des jours de congés).

## Configuration des périodes de télétravail
<a name="configuration_des_periodes_teletravail_ELTTEXTE000278"></a>
Le Centre de suivi de projets permet de définir les périodes en télétravail. 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CC_Suivi_Configuration%20-%20HC%20N%B0004.gif&type=thumb)


**Pour ajouter une période de télétravail** : 

1. Saisissez les dates et horaires du premier jour et du dernier jour de la période. Vous pouvez directement cliquer sur les différentes dates du calendrier pour saisir votre période de télétravail. 

2. Saisissez les raisons de la mise en place du télétravail.

3. Cliquez sur le bouton "Ajouter".




**Pour supprimer une période de télétravail**, il suffit de sélectionner la période et de cliquer sur le bouton "Supprimer la période".

**Remarques** : 

- Un intervenant peut uniquement saisir ses propres périodes de télétravail. 

- Les périodes de télétravail sont prises en compte par l'application de messagerie lors de l'écriture d'un message : lors de la saisie du nom d'un ou de plusieurs intervenants, la liste des destinataires en télétravail sera affichée.

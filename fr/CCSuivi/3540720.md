## Comment créer la base de données des Centres de Contrôle ?
<a name="comment_creer_base_donnees_des_centres_controle_ELTTEXTE000220"></a>


### Installation
<a name="installation_ELTPARAGRAPHE000022"></a>Lors de l'installation de WINDEV, WEBDEV ou WINDEV Mobile, le programme d'installation propose :

- soit de créer la base des Centres de Contrôle. Par défaut, cette base sera créée  dans le répertoire spécifié (dans le sous-répertoire "\\Personnel\\Centres de Contrôle") au premier lancement de WINDEV, WEBDEV ou WINDEV Mobile. Cette base de données sera par défaut au format HFSQL Classic.

- soit de partager une base de données des Centres de Contrôle existante.




Si vous avez choisi les paramètres par défaut, au premier lancement de WINDEV, WEBDEV ou WINDEV Mobile, la base des Centres de Contrôle est créée en local.
<a name="NOTE2_2"></a>


### Travail seul ou en équipe
<a name="travail_seul_equipe_ELTPARAGRAPHE000034"></a>**Si vous travaillez seul**, vous pouvez conserver la base de données des Centres de Contrôle en local. Cependant, si vous utilisez plusieurs produits, il est conseillé de partager cette base locale entre les différents produits.

**Si vous travaillez en équipe**, il est recommandé de partager la base de données des Centres de Contrôle. Il est possible :

- soit d'utiliser une base HFSQL Classic partagée.

- soit d'utiliser une base HFSQL Client/Serveur.




La configuration du type de base de données est réalisée dans les options de WINDEV, WEBDEV ou WINDEV Mobile : 

- sous le volet "Accueil", dans le groupe "Environnement", déroulez "Options" et sélectionnez "Options générales de WINDEV/WEBDEV/WINDEV Mobile". 

- dans l'onglet "Utilisateur", cliquez sur "Paramètres des Centres de Contrôle".



<a name="NOTE2_3"></a>


### Base de données HFSQL Classic partagée
<a name="base_donnees_hfsql_classic_partagee_ELTPARAGRAPHE000067"></a>**Pour utiliser une base de données HFSQL Classic partagée,** il suffit de :

- Créer un répertoire partagé accessible par tous les développeurs. Ce répertoire doit être en Lecture et Ecriture pour tous les intervenants.

- Indiquer le chemin de ce répertoire dans les options de configuration des Centres de Contrôle. Tous les développeurs de l'équipe doivent avoir la même configuration.




**Rappel** : La configuration du type de base de données est réalisée dans les options de WINDEV, WEBDEV ou WINDEV Mobile : 

- sous le volet "Accueil", dans le groupe "Environnement", déroulez "Options" et sélectionnez "Options générales de WINDEV/WEBDEV/WINDEV Mobile". 

- dans l'onglet "Utilisateur", cliquez sur "Paramètres des Centres de Contrôle".




La base de données sera automatiquement créée à la première utilisation. Les différents intervenants devront si nécessaire ressaisir leur login et mot de passe.

Remarque : Les informations saisies par les différents développeurs dans les Centres de Contrôle utilisés en local ne pourront être récupérées.
<a name="NOTE2_4"></a>


### Base de données HFSQL Client/Serveur
<a name="base_donnees_hfsql_clientserveur_ELTPARAGRAPHE000102"></a>**Pour utiliser une base de données HFSQL Client/Serveur**, il suffit de :

- Installer un serveur HFSQL Client/Serveur.
	Pour installer un serveur HFSQL Client/Serveur, lancez le package d'installation de votre produit et cliquez sur l'option "Installation du serveur HFSQL Client/Serveur".
	**Remarque** : En WINDEV, le programme d'installation d'un serveur HFSQL Client/Serveur est également livré dans le sous-répertoire "\\Install\\Serveur HFSQL" du répertoire d'installation de votre produit.

- Indiquer les caractéristiques de la base dans les options de configuration des Centres de Contrôle :

	- le nom ou l'adresse IP du serveur.

	- le nom de la base HFSQL Client/Serveur à utiliser. Cette base doit être vide (sans fichiers) ou inexistante (dans ce cas, elle sera automatiquement créée).

	- le port à utiliser pour la connexion.







Tous les développeurs de l'équipe doivent avoir la même configuration.

**Rappel** : La configuration du type de base de données est réalisée dans les options de WINDEV, WEBDEV ou WINDEV Mobile : 

- sous le volet "Accueil", dans le groupe "Environnement", déroulez "Options" et sélectionnez "Options générales de WINDEV/WEBDEV/WINDEV Mobile". 

- dans l'onglet "Utilisateur", cliquez sur "Paramètres des Centres de Contrôle".




**Remarque** : Si vous utilisez une base HFSQL Classic partagée, il est possible de migrer cette base en mode HFSQL Client/Serveur. Pour plus de détails, consultez [Utiliser la base des Centres de Contrôle en mode Client/Serveur](../CCSuivi/3540712.md).

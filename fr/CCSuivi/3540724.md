
## Centre de Suivi de projets : Diagramme de Gantt
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000106"></a>
Le Centre de Suivi de Projets permet de suivre les tâches d'un ou de plusieurs intervenants grâce : 

- à la liste des tâches (onglet de type "Tâches"), 

- au planning (onglet de type "Planning"),

- au diagramme de Gantt (onglet de type "Diagramme de Gantt"). 




Grâce au diagramme de Gantt, il devient facile d'organiser les tâches des différents intervenants dans les projets. 


![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=cc_Suivi_gantt%20-%20HC%20N%B0001.gif&type=thumb)


Le diagramme de Gantt affiche pour chaque intervenant sélectionné l'enchainement des différentes tâches. 
Pour chaque tâche, la durée restante est indiquée. Cette information est conservée lors de l'impression.

<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000130"></a>


### Afficher le diagramme de Gantt
<a name="afficher_diagramme_gantt_ELTPARAGRAPHE000030"></a>Le diagramme de Gantt des tâches peut être affiché : 

- Via un onglet de type "Tâches" : dans le ruban, dans le groupe "Gestion", cliquez sur "Diagramme de Gantt". 

- Via un nouvel onglet de type "Diagramme de Gantt".  




Dans le diagramme de Gantt, il est possible de choisir : 

- la période affichée, 

- le ou les intervenants concernés, 

- le mode de gestion des congés (réels ou suggérés), 

- la liste d'exigences concernée, 

- la granularité de l'affichage. 





### Manipuler un diagramme de Gantt
<a name="manipuler_diagramme_gantt_ELTPARAGRAPHE000046"></a>Dans un onglet "Diagramme de Gantt", il est possible de : 

- Modifier une tâche en double-cliquant sur la tâche. 

- Augmenter ou diminuer la durée d'une tâche en utilisant la souris pour manipuler les extrémités de la zone représentant la tâche. 

- Modifier la position des tâches et leur ordre d'exécution. 

- Lier une tâche à une tâche préalable (option "Lier une tâche préalable" du menu contextuel puis clic sur la tâche préalable). 

- Supprimer les tâches préalables (option "Supprimer tous les préalables" du menu contextuel).





### Imprimer un diagramme de Gantt
<a name="imprimer_diagramme_gantt_ELTPARAGRAPHE000057"></a>Dans un onglet "Diagramme de Gantt", il est possible d'imprimer le diagramme de Gantt (option "Imprimer" dans le ruban de l'onglet "Diagramme de Gantt".




## Maintenance des fichiers des Centres de Contrôle
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000110"></a>
Lors d'une utilisation normale d'une base de données, il peut être nécessaire de faire des opérations de maintenance sur les fichiers de données. Ces opérations permettent par exemple d'optimiser les temps d'accès aux données ou encore de libérer des transactions bloquées.

**Quand utiliser ces options ?**
Lors d'un problème technique (coupure de courant, ...), au lancement des Centres de Contrôle, un message peut demander une annulation des transactions bloquées ou une réindexation.
Remarque : Si la base de données des Centres de Contrôle est importante, une ré-indexation régulière est conseillée.

**Attention : Ces opérations sont bloquantes : les données des Centres de Contrôle ne sont plus accessibles pendant l'opération**.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000134"></a>
Pour lancer la maintenance des fichiers des Centres de Contrôle : 

1. Sélectionnez au choix :

	- dans le Centre de Suivi de projets :  déroulez ![Menu Centre de suivi de Projets](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CC_Suivi_Configuration%20-%20HC%20N%B0001.gif)
 puis sélectionnez "Maintenance".

	- dans WINDEV, WEBDEV ou WINDEV Mobile : sous le volet "Outils", dans le groupe "Centres de Contrôle", cliquez sur "Maintenance de la base".




2. Dans la fenêtre qui s'affiche, indiquez :

	- la base concernée.

	- l'opération à effectuer :

		- Réindexer la base : option conseillée si la base de données est importante

		- Annuler les transactions bloquées : option conseillée si un message indique que des transactions sont bloquées au lancement d'un Centre de Contrôle.

		- Migrer la base dans le format le plus récent : option conseillée pour profiter de toutes les nouveautés de la version en cours.

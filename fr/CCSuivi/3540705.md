


## Centre de Suivi de Projets : Gestion du planning
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000161"></a>
Les intervenants d'un projet peuvent gérer leurs tâches directement depuis le planning ou la liste des tâches.

Le planning présente sous forme graphique les différentes tâches d'un ou de plusieurs intervenants à réaliser, tous projets confondus. La présentation peut être faite avec un planning à l'heure, à la journée ou à la semaine.

Le planning est affiché par le Centre de Suivi de Projets dans un onglet de type "Planning". Ce planning prend en compte les tâches standard affichées dans les onglets de type "Tâches". 

L'onglet de type "Tâches" donne une vision générale des tâches d'un intervenant à effectuer sur un projet.

L'onglet de type "Planning" donne une vision générale des tâches d'un intervenant et des tâches simultanées de plusieurs intervenants.

## Manipuler le planning
<a name="manipuler_planning_ELTTEXTE000185"></a>


### Affichage et impression du planning
<a name="affichage_impression_planning_ELTPARAGRAPHE000030"></a>Le planning peut être affiché via un onglet de type "Planning". Par défaut, le planning de l'intervenant en cours est affiché. 

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Gestion_Planning%20-%20HC%20N%B0001.gif&type=thumb)
Le planning de chaque intervenant peut être affiché à l'heure, la journée ou à la semaine, via le menu contextuel du planning. 

Pour visualiser le planning de plusieurs intervenants, il suffit : 

- soit de sélectionner les différents intervenants à l'aide du bouton "..." situé à droite du champ "Intervenants".

- soit d'écrire le nom des intervenants voulus dans le champ "Intervenants". Ces noms doivent être séparés par un point virgule. 




Le planning peut être imprimé : 

- via l'option "Imprimer" du menu contextuel. 

- via le bouton "Imprimer" présent dans le ruban de l'onglet de type "Planning". 




Pour voir le détail d'une tâche affichée dans le planning, il suffit de double-cliquer sur cette tâche. La tâche est alors en édition et peut être modifiée.

<a name="NOTE2_2"></a>


### Durées prises en compte pour le planning
<a name="durees_prises_compte_pour_planning_ELTPARAGRAPHE000051"></a>Le planning prévisionnel (pour les jours à venir) prend en compte les horaires et les jours de travail saisis par chaque intervenant.

Le planning réalisé (des jours précédents) prend en compte :

- Si les intervenants ont activé la gestion du temps, la durée réelle passée sur chaque tâche (y compris les heures "supplémentaires").

- Si les intervenants n'ont pas activé la gestion du temps, les horaires et les jours de travail configurés dans le Centre de Suivi de Projets.

## Manipuler la liste des tâches
<a name="manipuler_liste_des_taches_ELTTEXTE000215"></a>


### Présentation
<a name="presentation_ELTPARAGRAPHE000065"></a>L'onglet de type "Tâches" permet à chaque intervenant de voir les tâches qui lui sont affectées. Dans cet onglet, l'intervenant peut les ré-organiser en utilisant les flèches à droite de la liste des tâches. Le planning (affiché dans les onglets de type "Planning") prend automatiquement en compte cette ré-organisation.

<a name="NOTE3_2"></a>


### Gestion des tâches en cours
<a name="gestion_des_taches_cours_ELTPARAGRAPHE000072"></a>L'intervenant peut facilement indiquer sa tâche en cours : dans un onglet de type "Tâches", il suffit de sélectionner la tâche et de sélectionner l'option "Démarrer cette tâche". Le nom de la tâche en cours apparaît alors :

- dans une bulle d'aide dans la barre des tâches (au-dessus de l'icône du Centre de Suivi de Projets).

- dans la fenêtre de gestion des tâches si [l'aide à la gestion du temps](../CCSuivi/3540711.md) est activée.




Le Centre de Suivi de Projets peut être réduit (bouton "Réduire" ou touche Echap du clavier).

**Pour indiquer que la tâche est terminée**, il suffit : 

- d'afficher le Centre de Suivi de Projets, 

- d'afficher un onglet de type "Tâches". 

- de sélectionner l'option "Réaliser la tâche" dans le menu contextuel de la tâche. Le temps passé sur la tâche est alors mis à jour.




**Si l'aide à la gestion du temps est activée**, le calcul du temps passé sur une tâche prend en compte :

- les informations indiquées dans la roue des tâches et dans la fenêtre de gestion de la tâche en cours. Les horaires et les jours de travail indiqués dans les options du Centre de Suivi de Projets ne sont pas pris en compte.

- le blocage automatique de la tâche en cours lors de la fermeture de la session Windows.

- le blocage automatique de la tâche en cours lors de la fermeture du Centre de Suivi de Projets de WINDEV, WEBDEV et WINDEV Mobile.




**Si l'aide à la gestion du temps n'est pas activée**, le calcul du temps passé sur une tâche prend en compte uniquement les horaires et les jours travaillés spécifiés dans la configuration des horaires. Les heures "supplémentaires" ne sont pas prises en compte.

## Dépassements de planning
<a name="depassements_planning_ELTTEXTE000245"></a>
Le Centre de Suivi de Projets permet de visualiser rapidement les dépassements de planning dans les onglets de type "Tâches" et dans les onglets de type "Planning". Les tâches concernées sont mises en évidence grâce à une couleur spécifique ou des hachures particulières.

**Pour visualiser les dépassements de planning** :

1. Affichez le volet de type "Tâches" ou de type "Planning". 

2. Dans le ruban du volet, cliquez sur "Dépassements de planning".

3. Dans la fenêtre qui s'affiche, indiquez la couleur à utiliser pour : 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Gestion_Planning%20-%20HC%20N%B0002.gif)


	- Visualiser les tâches dont le temps passé est proche de la durée estimée. 

- Visualiser les tâches dont la date d'échéance approche ou est dépassée. 

4. Validez.

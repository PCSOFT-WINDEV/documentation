
## Centre de Suivi de Projets : Tendance de création/correction des incidents
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000078"></a>
Pour améliorer le suivi d'un projet, le Centre de Suivi de Projets propose de consulter l'évolution des incidents créés et corrigés sur une période donnée, par un ou plusieurs intervenants.  

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Afficher la tendance
<a name="afficher_tendance_ELTTEXTE000102"></a>
Pour afficher la tendance des incidents créés et corrigés depuis le Centre de Suivi de projets :

1. Affichez si nécessaire un onglet de type "Incidents". 

2. Dans le ruban, dans le groupe "Gestion", cliquez sur "Tendance". 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=CC_Suivi_Tag%20-%20HC%20N%B0001%202.gif&type=thumb)


3. La fenêtre de statistiques s'affiche. 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ccsuivi-tendance.gif&type=thumb)
Cette fenêtre permet de voir la tendance des créations et corrections selon plusieurs critères : 

	- le mode de vision : 

		- "Développement" : Incidents créés ou corrigés sur la période et affectés aux intervenants sélectionnés. 

		- "Test" : Incidents créés ou corrigés sur la période dont les intervenants sont les auteurs. 




- le ou les intervenants pris en compte dans la tendance. Si aucun intervenant n'est sélectionné, tous les intervenants seront pris en compte. 

- le lot d'incidents si nécessaire. Si aucun lot n'est sélectionné, tous les incidents seront pris en compte.  

- la période prise en compte.


## Instruction POUR TOUT / POUR TOUS (parcours de combinaisons)

***En anglais : FOR EACH / FOR ALL***
				

<a name="description"></a>
L'instruction **POUR TOUT** permet de parcourir les options disponibles dans les combinaisons.  

**Remarque** : Les instructions **POUR TOUT, POUR TOUS, POUR CHAQUE** sont acceptées. Dans cette documentation, seule l'instruction **POUR TOUT** est utilisée et peut être remplacée par **POUR TOUS** ou **POUR CHAQUE**.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Code de déclaration
// Déclarer une combinaison
TypeCombinaison est une Combinaison
	CréationFic
	LectureFic
	EcritureFic
	SuppressionFic
FIN
```

```wl
// Syntaxe 1 : Parcours des options de la combinaison
POUR TOUT eOption DE TypeCombinaison
	Trace(eOption..Nom)
FIN
```

```wl
// Affectation de la variable
ActionFichier = LectureFic + EcritureFic

// Syntaxe 2 : Parcours des options actives
POUR TOUT eOption DE ActionFichier
	Trace(eOption..Nom)
FIN
```





<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Syntaxe simplifiée

`POUR TOUT <MonOption> DE <Combinaison>
    	 ...
FIN
`
---

**`<POUR TOUT> : ()`**

Signale le début du bloc d'instructions.

**`<MonOption> : ()`**

Variable de type [Description d'Option](../WDLang1/1000023072.md) correspondant à l'option de la combinaison. Il n'est pas nécessaire de déclarer cette variable. 

**`<Combinaison> : ()`**



- [Combinaison](../Motscles/1514074.md) contenant les valeurs à parcourir. 

- Variable de type [Combinaison](../Motscles/1514074.md) contenant les options à parcourir. Toutes les options (actives ou inactives) sont parcourues. 





<a name="SYNTAXE2"></a>

### Parcours des options actives d'une variable de type Combinaison

`POUR TOUT <MonOption> DE <Combinaison>
    	 ...
FIN
`
---

**`<POUR TOUT> : ()`**

Signale le début du bloc d'instructions.

**`<MonOption> : ()`**

Variable de type [Description d'Option](../WDLang1/1000023072.md) correspondant à l'option de la combinaison. Il n'est pas nécessaire de déclarer cette variable. 

**`<Combinaison> : ()`**

Variable de type [Combinaison](../Motscles/1514074.md) contenant les options à parcourir. Seules les options actives sont parcourues.

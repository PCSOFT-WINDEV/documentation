
## UTILISER DANS (Chronomètre)

***En anglais : USING IN***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
L'instruction **UTILISER ... DANS** permet d'effectuer automatiquement les opérations de déclenchement et d'arrêt d'un chrono. 

Cette instruction permet principalement de simplifier le cumul de chronomètres sur plusieurs itérations. 

Si le chronomètre est arrêté au début du bloc, il est automatiquement déclenché en sortie du bloc. 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
c est un Chrono

UTILISER c DANS
	ThreadPause(2s)
	SI Vrai ALORS
		SORTIR
		// Le calcul du chronomètre s'arrête
	FIN
FIN

ThreadPause(2s) // Ce code n'est pas pris en compte dans le calcul du chronomètre

UTILISER c DANS
	ThreadPause(2s)
FIN

Info(c)
```





<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`UTILISER <Chronomètre> DANS 

     ...
FIN
`
---

**`<UTILISER DANS> : ()`**

Signale le début du bloc d'instructions.

**`<Chronomètre> : (Variable de type Chrono)`**

Nom de la variable de type [Chrono](../WDLang1/1000023560.md) correspondant au chronométrage à réaliser.

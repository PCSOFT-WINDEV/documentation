
## MaFenêtreCoulissanteGauche (Mot réservé)

***En anglais : MyLeftSlidingWindow***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le mot-clé **MaFenêtreCoulissanteGauche** permet de manipuler la fenêtre coulissante gauche associée à la fenêtre courante. Lors de l'exécution de la fenêtre, le mot-clé **MaFenêtreCoulissanteGauche** est remplacé par la fenêtre et non par le nom de la fenêtre. Le mot-clé **MaFenêtreCoulissanteGauche** s'utilise alors comme une fenêtre.

**Intérêt** : Le mot-clé **MaFenêtreCoulissanteGauche** permet de manipuler depuis la fenêtre en cours les champs présents dans la fenêtre coulissante gauche associée à la fenêtre en cours.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MaFenêtreCoulissanteGauche.ConnecteAprès(Faux)
FenCoulissanteVisible(fcGauche, Vrai)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`MaFenêtreCoulissanteGauche`
---



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Manipulation des champs de la fenêtre coulissante depuis la fenêtre en cours
<a name="manipulation_des_champs_fenetre_coulissante_depuis_fenetre_cours_ELTPARAGRAPHE000033"></a>Le mot-clé **MaFenêtreCoulissanteGauche** est toujours remplacé par la fenêtre coulissante gauche définie pour la fenêtre en cours. Par exemple, utilisation de la propriété [Couleur](../Proprietes/2510071.md) sur un champ Libellé :


```wl
// Change la couleur d'un libellé dans la fenêtre coulissante à gauche
MaFenêtreCoulissanteGauche.LIB_Libellé.Couleur = RougeClair
```


## MonParent  (Mot réservé)

***En anglais : MyParent***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le mot-clé **MonParent** permet de manipuler :

- Le superchamp courant à partir d'un élément du superchamp. 
	Lors de l'exécution de la fenêtre, le mot-clé **MonParent** est remplacé par le superchamp et non par le nom du superchamp. Le mot-clé **MonParent** s'utilise alors comme un superchamp.
	Intérêt : Le mot-clé **MonParent** permet de rendre le code des champs du superchamp (champ, bouton, ...) ou des fonctions associées au superchamp indépendant du nom du superchamp.

- L'état courant à partir d'un des champs de l'état.

- La fenêtre courante à partir d'un des champs de la fenêtre.





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Code du bouton de sélection
NomFichier = fSélecteur(fExtraitChemin(NomFichier, fRépertoire), ...
	fExtraitChemin(NomFichier, fFichier + fExtension), ...
	"Sélectionner un fichier", "Tous les fichiers (*.*)" + TAB + "*.*", "")

// La valeur du superchamp correspond au fichier sélectionné
MonParent..Valeur = NomFichier
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`MonParent`

<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Manipulation du superchamp en cours
<a name="manipulation_superchamp_cours_ELTPARAGRAPHE000074"></a>

- Le mot-clé **MonParent** est toujours remplacé par le superchamp en cours. Par exemple, passage en paramètre :
	
	```wl
	AppelProcedure(MonParent)
	```


- Le mot-clé **MonParent** peut être utilisé uniquement dans les traitements manipulant le superchamp en cours (par exemple, dans les traitements associés à un champ du superchamp, dans les procédures associées au superchamp, ...).





<a name="NOTE0_2"></a>




### Utilisation de MonParent dans une procédure
<a name="utilisation_monparent_dans_une_procedure_ELTPARAGRAPHE000086"></a>Le mot-clé **MonParent** peut être utilisé dans une procédure associée au superchamp en cours. Le mot-clé **MonParent** fait alors référence au superchamp en cours.




## Coloration de procédures
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000172"></a>
Il est possible d'attribuer une couleur (de texte) aux procédures afin de les classer et de les retrouver facilement.

Cette fonctionnalité est très pratique dans les gros projets.
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Coloration_proc%E9dure%20-%20HC%20N%B0003.GIF)
Il est également possible de mettre en gras le nom de certaines procédures, par exemple pour identifier une procédure principale.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000196"></a>


### Colorer une ou plusieurs procédures
<a name="colorer_une_plusieurs_procedures_ELTPARAGRAPHE000022"></a>Pour colorer une ou plusieurs procédures :

1. Affichez le volet "Code" si nécessaire : sous le volet "Accueil", dans le groupe "Environnement", déroulez "Volets" et sélectionnez "Volets" puis "Code".

2. Dans le volet "Code", sélectionnez l'option "Procédures globales" ou "Procédures locales".

3. Sélectionnez la ou les procédures à colorer.
	**Rappel** : Pour sélectionner simultanément plusieurs procédures, conservez la touche Ctrl enfoncée lors de la sélection.

4. Affichez le menu contextuel de la sélection (clic droit). 

5. Sélectionnez l'option "Classer" puis "Classer dans le groupe xxx" ou xxx correspond à la couleur voulue. ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Coloration_proc%E9dure%20-%20HC%20N%B0001.GIF)




<a name="NOTE2_2"></a>


### Décolorer une ou plusieurs procédures
<a name="decolorer_une_plusieurs_procedures_ELTPARAGRAPHE000050"></a>Pour décolorer une ou plusieurs procédures :

1. Affichez le volet "Code" si nécessaire : sous le volet "Accueil", dans le groupe "Environnement", déroulez "Volets" et sélectionnez "Volets" puis "Code".

2. Dans le volet "Code", sélectionnez l'option "Procédures globales" ou "Procédures locales".

3. Sélectionnez la ou les procédures à décolorer.
	**Rappel** : Pour sélectionner simultanément plusieurs procédures, conservez la touche Ctrl enfoncée lors de la sélection.

4. Affichez le menu contextuel de la sélection (clic droit). 

5. Sélectionnez l'option "Classer" puis "Classer dans aucun groupe". ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Coloration_proc%E9dure%20-%20HC%20N%B0001%201.GIF)




<a name="NOTE2_3"></a>


### Visualiser les procédures appartenant à un groupe de couleurs
<a name="visualiser_les_procedures_appartenant_groupe_couleurs_ELTPARAGRAPHE000078"></a>Pour visualiser les procédures appartenant à un groupe de couleurs :

1. Affichez le volet "Code" si nécessaire : sous le volet "Accueil", dans le groupe "Environnement", déroulez "Volets" et sélectionnez "Volets" puis "Code".

2. Dans le volet "Code", sélectionnez l'option "Procédures globales" ou "Procédures locales".

3. Cliquez sur le bouton "Groupes" et sélectionnez l'icône de couleur correspondant au groupe que vous voulez afficher. ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Coloration_proc%E9dure%20-%20HC%20N%B0002.GIF)




<a name="NOTE2_4"></a>


### Mettre en gras le nom d'une ou de plusieurs procédures
<a name="mettre_gras_nom_une_plusieurs_procedures_ELTPARAGRAPHE000102"></a>Pour mettre en gras le nom d'une ou de plusieurs procédures :

1. Affichez le volet "Code" si nécessaire : sous le volet "Accueil", dans le groupe "Environnement", déroulez "Volets" et sélectionnez "Volets" puis "Code".

2. Dans le volet "Code", sélectionnez l'option "Procédures globales" ou "Procédures locales".

3. Sélectionnez la ou les procédures voulues.
	**Rappel** : Pour sélectionner simultanément plusieurs procédures, conservez la touche Ctrl enfoncée lors de la sélection.

4. Affichez le menu contextuel de la sélection (clic droit). 

5. Sélectionnez l'option "Classer" puis "Favori". ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Coloration_proc%E9dure%20-%20HC%20N%B0001%202.GIF)

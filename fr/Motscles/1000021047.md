
## MembreVariant (Type de variable)

***En anglais : VariantMember***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **MembreVariant** est un type avancé, correspondant à un tableau de membres, chaque membre étant une variable de type [variant](../Motscles/1514019.md).

Ce type de variable ne doit être utilisé que dans des cas très spécifiques, par exemple lors du traitement du résultat de la fonction [JSONVersVariant](../WDLang1/1000021052.md).

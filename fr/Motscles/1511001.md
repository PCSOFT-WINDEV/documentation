
## MaSource (Mot réservé)

***En anglais : MySource***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le mot-clé **MaSource** permet de manipuler la source de données (fichier, vue ou requête) courante associée à un état ou une table fichier. Lors de l'exécution du projet, ce mot-clé est automatiquement remplacé par la source de données en cours.

**Intérêt** : Le mot-clé **MaSource** permet de :

- Rendre le code indépendant de la source de données en cours.

- Accéder à la valeur d'une rubrique de l'enregistrement en cours dans la source de données en cours.

- Récupérer le nom de la source de données pour effectuer un traitement HFSQL.

- Indiquer la fenêtre contenant une table à imprimer dans un [état sur table](../WDChamp/1011012.md).





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Récupération du nom de la source de données en cours dans l'état
NomSourceDonnée = MaSource..Nom

// Récupération du nom du client en cours d'impression
SI MaSource.NomCli = "Martin" ALORS
	CALC_TotalHT.CouleurFond = RougeClair
FIN
```


<a name="Exemple2"></a>



```wl
// Modifier la source de données d'un état sur table dans son code d'ouverture
PROCEDURE ETAT_Table_Client(sNomTableSource)
MonEtat.MaSource = sNomTableSource
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`MaSource`
---



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Le mot-clé **MaSource** est utilisable uniquement dans les événements et/ou traitements associés à :

	- un état,

	- un champ Table fichier.




- Le mot-clé **MaSource** n'est pas utilisable dans les champs dont la source de données est une variable.

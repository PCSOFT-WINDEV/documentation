
## Cadre (Type de variable)

***En anglais : Border***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **Cadre** permet de manipuler un cadre par programmation. Les caractéristiques de ce cadre peuvent être modifiées à l'aide de [différentes propriétés du WLangage](#NOTE0_2).

Un type **Cadre** peut être utilisé : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) dans les champs d'une fenêtre (propriété [Cadre](../Proprietes/1000018157.md) des champs d'une fenêtre).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) dans les cellules d'un champ Table d'une fenêtre (propriété [Cadre](../Proprietes/1000018157.md) des cellules des champs Table).

- dans les champs d'un état (propriété [Cadre](../Proprietes/1000018157.md) des champs d'un état).
	

- dans les fonctions de dessin (fonction [dCadre](../WDLang1/1000021894.md)).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) dans les cellules Excel (propriété [Cadre](../WDLang5/1000017464.md) des cellules d'un document Excel).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) dans les paragraphes d'un document de type Traitement de texte (propriété **Cadre** sur les options de mise en page des variables de type [docParagraphe](../WDLang1/1000022483.md)). 




**Remarque** : Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).

<a name="sample_code"></a>

## Exemple


```wl
// Définition des caractéristiques du cadre pour les champs de l'état
MonCadre est un Cadre
MonCadre.Couleur = RougeClair
MonCadre.Epaisseur = 5

// Utilisation du cadre pour le champ NumNbPage de l'état
NumNbPage.Cadre = MonCadre
// Modification d'une caractéristique du cadre du champ NumNbPage
NumNbPage.Cadre.Epaisseur = 1

// Récupération des caractéristiques du cadre du champ NumNbPage
MonCadre2 est un Cadre = NumNbPage.Cadre
```


<a name="Exemple2"></a>

```wl
// Définition des caractéristiques du cadre
MonCadre est un Cadre
MonCadre.Couleur = RougeClair
MonCadre.Epaisseur = 5

// Utilisation du cadre pour le champ NumNbPage de l'état
NumNbPage.Cadre = MonCadre
// Modification d'une caractéristique du cadre du champ NumNbPage
NumNbPage.Cadre.CoinHautGauche.Largeur = 2
NumNbPage.Cadre.CoinHautGauche.Hauteur = 3
```
<a name="Exemple3"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 
```wl
// Paramètre le cadre
MonCadre est un Cadre
MonCadre.Couleur = RougeClair
MonCadre.Epaisseur = 5

// Spécificité du trait bas
MonCadre.TraitBas.Type = TraitTiret

// Applique le cadre à la cellule 2, 2 (2ème ligne, 2ème colonne)
TABLE_MaTable[2,2].Cadre = MonCadre
// ou 
TABLE_MaTable.COL_SansNom2[2].Cadre = MonCadre
```

## Remarques




### Caractéristiques par défaut d'un cadre
<a name="caracteristiques_par_defaut_cadre_ELTPARAGRAPHE000110"></a>**Par défaut, dans un état**, un cadre a les caractéristiques suivantes :

- épaisseur 0. Le cadre est visible si l'épaisseur est supérieure à 0. Le cadre est invisible si l'épaisseur vaut 0.

- couleur noir.

- trait continu.

- coins non arrondis.

- tous les bords du cadre sont affichés.




![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) **Par défaut, dans une feuille XLS**, le cadre d'une cellule a les mêmes caractéristiques. Cependant, dans le cas d'une cellule XLS, l'épaisseur ainsi que l'arrondi des coins ne sont pas gérés.

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) **Par défaut, dans une cellule d'un champ Table**, un cadre a les caractéristiques suivantes :

- épaisseur 0. Le cadre est visible si l'épaisseur est supérieure à 0. Le cadre est invisible si l'épaisseur vaut 0.

- couleur transparente.

- trait continu.

- coins non arrondis.

- aucun bord du cadre n'est affiché.




Pour définir les caractéristiques d'un cadre par programmation, utilisez les propriétés de gestion des cadres.


<a name="NOTE0_2"></a>




### Propriétés WLangage utilisables avec le type Cadre
<a name="proprietes_wlangage_utilisables_avec_type_cadre_ELTPARAGRAPHE000143"></a>Les propriétés suivantes peuvent être utilisées pour manipuler une variable de type **Cadre**. 

| Nom de la propriété | Type manipulé | Effet |
| --- | --- | --- |
| Arrondi | Réel | Valeur de l'arrondi des coins du cadre. Correspond au rayon de l'arrondi.  <br><br><br><br>- En modification, modifie l'arrondi de tous les coins. <br><br>- En lecture, renvoie : <br><br>	- la largeur et la hauteur des coins si ces paramètres sont identiques pour tous les coins. <br><br>	- la constante *ArrondiIndéfini* si tous les coins n'ont pas la même largeur et hauteur.<br><br><br><br><br><br><br><br>Non géré : <br><br>- dans les cellules XLS.<br><br>- dans les paragraphes des document de type Traitement de texte.<br><br><br> |
| Bords | Constante de type entier | Renvoie et modifie la présence des bords du cadre. Cette propriété peut correspondre à une combinaison des options suivantes :<br><br>- *BordAucun* : le cadre n'a aucun bord.<br><br>- *BordBas* : le cadre a un bord bas.<br><br>- *BordDroite* : le cadre a un bord droit.<br><br>- *BordGauche* : le cadre a un bord gauche.<br><br>- *BordHaut* : le cadre a un bord haut.<br><br>- *BordTous* : tous les bords du cadre sont affichés.<br><br><br> |
| CoinBasDroite | <br><br> | Caractéristiques du coin bas droit du cadre : <br><br>- Propriété **Largeur** : largeur du coin du cadre (réel). <br><br>- Propriété **Hauteur** : hauteur du coin du cadre (réel). <br><br><br><br><br>Non géré dans les cellules XLS. |
| CoinBasGauche | <br><br> | Caractéristiques du coin bas gauche du cadre : <br><br>- Propriété **Largeur** : largeur du coin du cadre (réel). <br><br>- Propriété **Hauteur** : hauteur du coin du cadre (réel). <br><br><br><br><br>Non géré dans les cellules XLS. |
| CoinHautDroite | <br><br> | Caractéristiques du coin haut droit du cadre : <br><br>- Propriété **Largeur** : largeur du coin du cadre (réel). <br><br>- Propriété **Hauteur** : hauteur du coin du cadre (réel). <br><br><br><br><br>Non géré dans les cellules XLS. |
| CoinHautGauche | <br><br> | Caractéristiques du coin haut gauche du cadre : <br><br>- Propriété **Largeur** : largeur du coin du cadre (réel). <br><br>- Propriété **Hauteur** : hauteur du coin du cadre (réel). <br><br><br><br><br>Non géré dans les cellules XLS. |
| Couleur | Entier | Couleur du cadre. Cette couleur peut correspondre à :<br><br>- une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)),<br><br>- [une couleur prédéfinie](../WDLang5/3010002.md).<br><br><br><br><br>Cette propriété peut être utilisée : <br><br>- En modification : la couleur de tous les bords est modifiée. <br><br>- En lecture, cette propriété renvoie : <br><br>	- la couleur des bords si cette couleur est identique pour tous les bords. <br><br>	- la constante *CouleurIndéfinie* si tous les bords n'ont pas la même couleur.<br><br><br><br><br><br><br><br> |
| Epaisseur | Réel | Renvoie et modifie l'épaisseur du cadre (en millimètres dans les impressions et en pixels dans les fonctions graphiques). <br><br><br><br>- En modification, modifie l'épaisseur de tous les bords. <br><br>- En lecture, renvoie : <br><br>	- l'épaisseur des bords si ce paramètre est identique pour tous les bords. <br><br>	- la constante *EpaisseurIndéfinie* si tous les bords n'ont pas la même épaisseur.<br><br><br><br><br><br><br><br>Non géré dans les cellules XLS. |
| Trait | Constante de type Entier | Renvoie et modifie le type du trait du cadre (voir paragraphe suivant).<br><br><br><br>- En modification, modifie le type de trait de tous les bords. <br><br>- En lecture, renvoie : <br><br>	- le type de trait des bords si ce paramètre est identique pour tous les bords. <br><br>	- la constante *TypeIndéfini* si tous les bords n'ont pas le même trait.<br><br><br><br><br><br><br><br> |
| TraitBas | <br><br> | Caractéristiques du trait bas du cadre : <br><br>- Propriété **Couleur** : Couleur du trait : Cette couleur peut correspondre soit à une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)), soit à [une couleur prédéfinie](../WDLang5/3010002.md).<br><br>- Propriété **Epaisseur** (Réel) : Epaisseur du trait (en millimètres dans les impressions et en pixels dans les fonctions graphiques). <br><br>- Propriété **Type** (constante de type Entier) : Type du trait (voir paragraphe suivant).<br>	<br><br><br><br><br>Non géré dans les cellules XLS. |
| TraitDroite | <br><br> | Caractéristiques du trait droit du cadre : <br><br>- Propriété **Couleur** : Couleur du trait : Cette couleur peut correspondre soit à une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)), soit à [une couleur prédéfinie](../WDLang5/3010002.md).<br><br>- Propriété **Epaisseur** (Réel) : Epaisseur du trait (en millimètres dans les impressions et en pixels dans les fonctions graphiques). <br><br>- Propriété **Type** (constante de type Entier) : Type du trait (voir paragraphe suivant).<br>	<br><br><br><br><br>Non géré dans les cellules XLS. |
| TraitGauche | <br><br> | Caractéristiques du trait gauche du cadre : <br><br>- Propriété **Couleur** : Couleur du trait : Cette couleur peut correspondre soit à une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)), soit à [une couleur prédéfinie](../WDLang5/3010002.md).<br><br>- Propriété **Epaisseur** (Réel) : Epaisseur du trait (en millimètres dans les impressions et en pixels dans les fonctions graphiques). <br><br>- Propriété **Type** (constante de type Entier) : Type du trait (voir paragraphe suivant).<br>	<br><br><br><br><br>Non géré dans les cellules XLS. |
| TraitHaut | <br><br> | Caractéristiques du trait haut du cadre : <br><br>- Propriété **Couleur** : Couleur du trait : Cette couleur peut correspondre soit à une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)), soit à [une couleur prédéfinie](../WDLang5/3010002.md).<br><br>- Propriété **Epaisseur** (Réel) : Epaisseur du trait (en millimètres dans les impressions et en pixels dans les fonctions graphiques). <br><br>- Propriété **Type** (constante de type Entier) : Type du trait (voir paragraphe suivant).<br>	<br><br><br><br><br>Non géré dans les cellules XLS. |


Ces propriétés peuvent être utilisées :

- soit directement sur la variable **Cadre**. Exemple :
	
	```wl
	MonCadre est un Cadre
	MonCadre.Couleur = RougeClair
	MonCadre.Epaisseur = 5
	```


- soit sur la propriété **Cadre** de l'élément manipulé. 
	
	```wl
	// Modification d'une caractéristique du cadre de l'élément NumNbPage
	NumNbPage.Cadre.Epaisseur = 1
	```





**Remarque** : Il est également possible d'affecter directement les caractéristiques d'un cadre défini par programmation à un élément :

```wl
// Définition des caractéristiques du cadre
MonCadre est un Cadre
MonCadre.Couleur = RougeClair
MonCadre.Epaisseur = 5

// Utilisation du cadre pour le champ NumNbPage de l'état
NumNbPage.Cadre = MonCadre
```

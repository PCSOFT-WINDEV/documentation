
## Police (Type de variable)

***En anglais : Font ***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Une variable de type **Police** permet de créer une police par programmation. Les caractéristiques de cette police peuvent être modifiées à l'aide de différentes propriétés WLangage.

Une variable de type **Police** peut être utilisée : 

- dans les dessins,

- dans les graphes,

- pour les impressions,

- dans les champs d'une fenêtre (propriété [Police](../Motscles/1514045.md) des champs d'une fenêtre),

- dans les champs d'une page (propriété [Police](../Motscles/1514045.md) des champs d'une page),

- dans les champs d'un état (propriété [Police](../Motscles/1514045.md) des champs d'un état).




**Remarques** : 

- Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Pour modifier les caractéristiques d'une cellule d'un champ Tableur, utilisez la fonction [TableurStyleSélection](../WDLang1/1000021338.md).









<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Déclaration de variables de type Police
MaPolice est une Police
MaPolice1, MaPolice2 sont des Polices
```


<a name="Exemple2"></a>



```wl
// Variable initialisée avec la police créée avec la fonction PoliceCrée
MaPolice = PoliceCrée("Arial", 12, iGras)
```


<a name="Exemple3"></a>



```wl
// Modification des caractéristiques de la police
MaPolice.Nom = "Arial"
MaPolice.Taille = 12
MaPolice.Gras = Vrai
```


<a name="Exemple4"></a>





```wl
// Utilisation d'une police sur un champ d'une fenêtre
SAI_Nom.Police = MaPolice

// Récupération de la police d'un champ
MaPolice1 = SAI_Prénom.Police
```

## Remarques




### Caractéristiques d'une police
<a name="caracteristiques_une_police_ELTPARAGRAPHE000110"></a>Par défaut, une variable Police a les caractéristiques suivantes :

- "Arial", 

- taille 12, 

- couleur noir,

- sans attribut.




Pour définir les caractéristiques d'une police par programmation, utilisez :

- Soit la fonction [PoliceCrée](../WDLang1/3060002.md).

- Soit les propriétés de gestion des polices (voir ci-dessous)

- Soit la fonction [PoliceSelect](../WDLang1/3060004.md) qui ouvre la fenêtre standard Windows de sélection des polices.





<a name="NOTE0_2"></a>




### Propriétés de gestion des polices
<a name="proprietes_gestion_des_polices_ELTPARAGRAPHE000136"></a>Les propriétés suivantes peuvent être utilisées pour manipuler une variable de type **Police** :

| Nom de la propriété | Effet |
| --- | --- |
| [Alphabet](../Proprietes/2512008.md) | Renvoie et modifie l'alphabet utilisé par une variable de type Police.<br><br> |
| [Angle](../Proprietes/1410086632.md) | Renvoie et modifie l'angle (en degrés) d'une variable de type Police.<br><br> |
| [Barré](../Proprietes/2513005.md) | Vérifie si une variable de type Police est barrée et barre (ou non) une variable de type Police. |
| [Couleur](../Proprietes/2510071.md) | Vérifie et modifie la couleur d'une variable de type Police. |
| [Gras](../Proprietes/2513001.md) | Teste si une variable de type Police est en gras et modifie l'attribut gras d'une variable de type Police. |
| [Interlettre](../Proprietes/1410088367.md) | Renvoie et modifie l'espacement ajouté entre les caractères d'une variable de type Police. |
| [Interligne](../Proprietes/1410088366.md) | Renvoie et modifie le coefficient d'espacement entre les lignes pour une variable de type Police.  |
| [Italique](../Proprietes/2513007.md) | Vérifie si une variable de type Police est en italique et modifie l'attribut italique d'une variable de type Police. |
| [Nom](../Proprietes/2510082.md) | Renvoie et modifie le nom d'une variable de type Police. |
| [Souligné](../Proprietes/2513013.md) | Vérifie si une variable de type Police est soulignée et souligne (ou non) une variable de type Police. |
| [Taille](../Proprietes/2510125.md) | Renvoie et modifie la taille (en points pica) d'une variable de type Police. |
| [UnitéTaille](../Proprietes/1410088365.md) | Renvoie et modifie l'unité utilisée pour manipuler la taille d'une variable de type Police. |





Ces propriétés peuvent être utilisées directement sur la variable de type **Police**. Exemple :


```wl
MaPolice est une Police
MaPolice.Gras = Vrai
MaPolice.Couleur = RougeClair
```


![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Seules les propriétés suivantes sont utilisables avec le type **Police** : [Angle](../Proprietes/1410086632.md), [Barré](../Proprietes/2513005.md), [Couleur](../Proprietes/2510071.md), [Gras](../Proprietes/2513001.md), [Italique](../Proprietes/2513007.md), [Nom](../Proprietes/2510082.md), [Souligné](../Proprietes/2513013.md), [Taille](../Proprietes/2510125.md).

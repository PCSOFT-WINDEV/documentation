
## Instruction POUR TOUT / POUR TOUS (parcours d'énumérations)

***En anglais : FOR EACH / FOR ALL***
				

<a name="description"></a>
L'instruction **POUR TOUT** permet de parcourir les valeurs disponibles dans les énumérations. 

**Remarque** : Les instructions **POUR TOUT, POUR TOUS, POUR CHAQUE** sont acceptées. Dans cette documentation, seule l'instruction **POUR TOUT** est utilisée et peut être remplacée par **POUR TOUS** ou **POUR CHAQUE**.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Parcours des options d'une énumération
// ---------------------------------------
// Code de déclaration
// Déclarer une énumération
TypeAutomate est une Enumération
	Marche
	Arrêté
	EnMaintenance
	EnPanne
FIN
```

```wl
// Parcours des valeurs
POUR TOUT Option DE TypeAutomate
	Trace(Option..Nom)	
FIN
```





<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`POUR TOUT <MaValeur> DE <Enumération>
    	 ...
FIN
`
---

**`<POUR TOUT> : ()`**

Signale le début du bloc d'instructions.

**`<MaValeur> : ()`**

Variable de type [Description de Valeur](../WDLang1/1000023061.md) correspondant à la valeur de l'énumération. Il n'est pas nécessaire de déclarer cette variable. 

**`<Enumération> : ()`**

[Enumération](../Motscles/1514067.md) contenant les valeurs à parcourir.


## Instruction BOUCLE

***En anglais : LOOP***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le bloc d'instructions est répété à l'infini. Le nombre d'itérations du bloc d'instructions n'est pas contrôlé, il n'y a pas d'expression à évaluer.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Sortie selon une condition SI
BOUCLE
	// Lecture d'une ligne du fichier texte
	UneLigne = fLitLigne(NumFic)
	 SI UneLigne = EOT ALORS SORTIR
	TraiteLigne(UneLigne)
FIN
```


<a name="Exemple2"></a>



```wl
// Sortie selon une condition TANTQUE
BOUCLE
	// Lecture d'une ligne du fichier texte
	UneLigne = fLitLigne(NumFic)
	TraiteLigne(UneLigne)
A FAIRE TANTQUE UneLigne <> EOT
```




<a name="Exemple3"></a>





```wl
// Sortie selon une itération
BOUCLE (10)
	// Lecture d'une ligne du fichier texte
	UneLigne = fLitLigne(NumFic)
	TraiteLigne(UneLigne)
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Boucle avec sortie selon une condition "SI"

`BOUCLE
    ...
    SI <Condition> ALORS SORTIR
    ...
FIN
`
---

**`<BOUCLE> : ()`**

Signale le début du bloc d'instructions.

**`<SORTIR> : ()`**

Permet de quitter le bloc d'instructions.

**`<FIN> : ()`**

Signale la fin du bloc d'instructions.  


<a name="SYNTAXE2"></a>

### Boucle avec sortie selon une condition "TANTQUE"

`BOUCLE
...
A FAIRE TANTQUE <Condition>
`
---

**`<BOUCLE> : ()`**

Signale le début du bloc d'instructions.

**`<A FAIRE TANTQUE> : ()`**

Signale la fin du bloc d'instructions. Permet de quitter le bloc d'instructions. Les lignes de la boucle situées avant cette instruction sont exécutées.  


<a name="SYNTAXE3"></a>

### Boucle avec sortie selon le nombre d'itérations 

`BOUCLE (<Nombre d'itérations>)
...
FIN
`
---

**`<BOUCLE> : ()`**

Signale le début du bloc d'instructions.

**`<Nombre d'itérations> : ()`**

Nombre d'itérations à exécuter. La sortie de la boucle se fera lorsque le nombre d'itérations sera atteint.

**`<FIN> : ()`**

Signale la fin du bloc d'instructions.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Code à exécuter
<a name="code_executer_ELTPARAGRAPHE000100"></a>Le code à exécuter est placé entre les instructions **BOUCLE** et **FIN**.


<a name="NOTE0_2"></a>




### Sortir d'une boucle
<a name="sortir_une_boucle_ELTPARAGRAPHE000109"></a>Plusieurs instructions sont possibles :

- [RETOUR](../Motscles/1510009.md) : Sortie de la boucle et sortie du traitement (ou de la procédure) en cours.

- [RENVOYER](../Motscles/1510007.md) : Renvoi d'un compte-rendu au traitement appelant. Sortie de la boucle et sortie du traitement (ou de la procédure) en cours.

- [SORTIR](../Motscles/1510012.md) : Sortie de la boucle et exécution de la suite du traitement en cours.




La fonction [Ferme](../WDLang1/3038018.md) permet de sortir de la boucle et de fermer la fenêtre en cours.

**Attention** : Il n'est pas possible d'utiliser les instructions [RETOUR](../Motscles/1510009.md) et [RENVOYER](../Motscles/1510007.md) **dans un même traitement**.


<a name="NOTE0_3"></a>




### Exécution de l'itération suivante
<a name="execution_iteration_suivante_ELTPARAGRAPHE000141"></a>Pour exécuter directement l'itération suivante **sans terminer le code de l'itération en cours**, utilisez l'instruction [CONTINUER](../Motscles/1510003.md) :


```txt
BOUCLE
	...
	SI <Condition> ALORS CONTINUER		// Retour au mot-clé BOUCLE
	...
FIN
```



<a name="NOTE0_4"></a>




### Boucle sans fin
<a name="boucle_sans_fin_ELTPARAGRAPHE000155"></a>Lors de la compilation du projet, une boucle sans fin évidente (absence des instructions [SORTIR](../Motscles/1510012.md), [RETOUR](../Motscles/1510009.md) ou [RENVOYER](../Motscles/1510007.md)) est signalée par un warning.


<a name="NOTE0_5"></a>




### Variable externe dans une boucle
<a name="variable_externe_dans_une_boucle_ELTPARAGRAPHE000173"></a>Une variable déclarée avec le mot-clé [EXTERNE](../Motscles/1511013.md) ne peut pas être utilisée dans une boucle.

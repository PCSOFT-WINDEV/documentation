
## Opérations disponibles sur les dates, heures, durées, ...
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000309"></a>
Les opérations suivantes sont disponibles sur les variables de type : [Date](../Motscles/1514066.md), [Heure](../Motscles/1514068.md), [DateHeure](../Motscles/1514070.md) et [Durée](../Motscles/1514069.md) :

- addition

- soustraction

- comparaison






**Remarque** : Le WLangage dispose aussi de fonctions permettant de manipuler des dates et les heures (voir fonctions de gestion des dates et des heures).



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Addition
<a name="addition_ELTTEXTE000333"></a>
Les opérations d'addition entre les différents types de variables Date, Heure, DateHeure et Durée sont présentées dans le tableau ci-dessous. Ce tableau présente pour chaque combinaison, le type du résultat :

| Type additionné<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=fleche2.gif)<br> | Date | Heure | DateHeure | Durée |
| --- | --- | --- | --- | --- |
| **Date** | Concaténation des éléments dans une chaîne | Concaténation des éléments dans une chaîne. Cette chaîne peut être utilisée par la fonction [DateHeureDifférence](../WDLang1/3027020.md) | Concaténation des éléments dans une chaîne | Variable de type Date (partie Heure ignorée) |
| **Heure** | Concaténation des éléments dans une chaîne | Concaténation des éléments dans une chaîne | Concaténation des éléments dans une chaîne | Variable de type Heure (partie date ignorée) |
| **DateHeure** | Concaténation des éléments dans une chaîne | Concaténation des éléments dans une chaîne | Concaténation des éléments dans une chaîne | Variable de type DateHeure |
| **Durée** | Variable de type Date | Variable de type Heure | Variable de type DateHeure | Variable de type Durée |






```wl
// Addition d'une variable de type DateHeure et d'une variable de type Heure
DateHeureDébut est une DateHeure = "200112051530"
DateHeureFin est une DateHeure = "200112051730"
DuréeSéminaire est une Durée = DateHeureFin - DateHeureDébut
 
HeureEnPlus est une Heure = "0330"
DateHeureFinal est une DateHeure
DateHeureFinal = DuréeSéminaire + HeureEnPlus
```



```wl
// Addition d'une variable de type DateHeure et d'une variable de type Durée
dhMaDate est une DateHeure = DateSys() + HeureSys()
dhMaDateRésultat est une DateHeure

Trace("DateHeure de début : " + dhMaDate)

HeureDébut est une Heure = "0117"
HeureFin est une Heure = "2353"  
duMaDurée est une Durée = HeureFin - HeureDébut
Trace("Durée en heures : " + NumériqueVersChaîne(duMaDurée.EnHeures) + "h")

dhMaDateRésultat = dhMaDate + duMaDurée
Trace("DateHeure additionnée à une durée : " + dhMaDateRésultat)
```

## Soustraction
<a name="soustraction_ELTTEXTE000357"></a>
Les opérations de soustraction entre les différents types de variables Date, Heure, DateHeure et durée sont présentées dans le tableau ci-dessous. Ce tableau présente pour chaque combinaison, le type du résultat :

| Type soustrait<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=fleche2.gif)<br> | Date | Heure | DateHeure | Durée |
| --- | --- | --- | --- | --- |
| **Date** | Variable de type Durée | Erreur | Erreur | Variable de type Date |
| **Heure** | Erreur | Variable de type Durée | Erreur | Variable de type Heure |
| **DateHeure** | Erreur | Erreur | Variable de type Durée | Variable de type DateHeure |
| **Durée** | Erreur | Erreur | Erreur | Variable de type Durée |






```wl
DateHeureDébut est une DateHeure = "200112051530"
DateHeureFin est une DateHeure = "200112071730"
DuréeSéminaire est une Durée = DateHeureFin - DateHeureDébut
```

## Comparaison
<a name="comparaison_ELTTEXTE000381"></a>
Les opérations de comparaison (=, &lt;&gt;, &lt;, &gt;, &lt;=, &gt;=) entre les différents types de variables Date, Heure, DateHeure et Durée sont présentées dans le tableau ci-dessous. Ce tableau présente pour chaque combinaison, si la comparaison est possible.

| Type comparé<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=fleche2.gif)<br> | Date | Heure | DateHeure | Durée |
| --- | --- | --- | --- | --- |
| **Date** | Possible | Erreur | Possible (partie heure ignorée) | Erreur |
| **Heure** | Erreur | Possible | Possible (partie Date ignorée) | Erreur |
| **DateHeure** | Possible (partie heure ignorée) | Possible (partie Date ignorée) | Possible | Erreur |
| **Durée** | Erreur | Erreur | Erreur | Possible |






```wl
DateHeureDébut est une DateHeure = "200112051530"
DateHeureFin est une DateHeure = "200115051730"
SI DateHeureFin > DateHeureDébut ALORS
	Info("Les dates sont valides")
FIN
```

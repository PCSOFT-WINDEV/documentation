
## BLOC

***En anglais : BLOCK***
				

<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le mot-clé **&lt;BLOC&gt;** est utilisé dans l'éditeur de code pour déclarer des blocs de code personnalisés. Ces blocs de code permettent de mettre en évidence les différentes phases d'un algorithme, de regrouper des déclarations de variables ou de membres. 

Le bloc de code défini pourra être déplié ou replié. 

Il est possible d'associer un commentaire au bloc. Ce commentaire sera toujours affiché en mode replié. 

Ce mot-clé n'a aucune influence en exécution. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
<BLOC Structures de sérialisation d'un champ>

ObjetChamp est une Structure
	Nom est une chaîne
	Type est un entier
	Propriétés est un tableau associatif de Variant
FIN

MémoireObjetChamp est une Structure
	NombreOccurrence est un entier
	ListeObjets est un tableau de ObjetChamp
FIN

<FIN>
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Déclaration d'un bloc de code

`<BLOC>
   <Code à exécuter>
<FIN>
`
---

**`<Code à exécuter> : ()`**

Code WLangage à exécuter associé au bloc.  


<a name="SYNTAXE2"></a>

### Déclaration d'un bloc de code avec affichage d'un commentaire en mode replié

`<BLOC Commentaire toujours affiché>
   <Code à exécuter>
<FIN>
`
---

**`<Code à exécuter> : ()`**

Code WLangage à exécuter associé au bloc.


## DateHeure  (Type de variable)

***En anglais : DateTime***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **DateHeure** permet de manipuler simplement une date et une heure. Le type DateHeure peut être utilisé par exemple pour réaliser des calculs (différence, addition, ...) sur les dates, les heures et les durées. Pour plus de détails, consultez [Opérations disponibles sur le type DateHeure](../Motscles/1514040.md).

**Remarques** :

- Plusieurs propriétés peuvent être utilisées avec le type **DateHeure**, pour récupérer par exemple uniquement la date ou l'heure, une partie de la date, ou une partie de l'heure.

- Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
DateDébut est une DateHeure = "200112311524"     // 31/12/2001 à 15h24 minutes
```

## Remarques




### Déclaration et initialisation
<a name="declaration_initialisation_ELTPARAGRAPHE000073"></a>

- Une variable de type **DateHeure** peut être initialisée avec une valeur de type :

	- **Date** : les heures et les minutes seront initialisées à 0.

	- **DateHeure**

	- **Chaîne de caractères**, au format

		- AAAAMMJJ

		- AAAAMMJJHH

		- AAAAMMJJHHmm

		- AAAAMMJJHHmmSS

		- AAAAMMJJHHmmSSCC

		- AAAAMMJJHHmmSSCCC
						(AAAA : année, MM : mois, JJ : Jour, HH : heure, mm : minutes, SS : secondes, CC : centièmes de seconde, CCC : Millièmes de seconde)




- Par défaut, une variable de type **DateHeure** est initialisée avec la date du jour (date système) et l'heure système. Pour régler la valeur par défaut affectée aux variables de type [Date](../Motscles/1514066.md), [Heure](../Motscles/1514068.md) et **DateHeure**, utilisez la fonction [DateHeureParDéfaut](../WDLang1/3027015.md).




Remarque : Il est également possible d'initialiser une variable de type **DateHeure** avec la date et l'heure système en utilisant la fonction [Maintenant](../WDLang1/3027024.md). 
<a name="NOTE0_2"></a>




### Les propriétés utilisables sur les variables de type DateHeure 
<a name="les_proprietes_utilisables_sur_les_variables_type_dateheure_ELTPARAGRAPHE000111"></a>


|   |   |
| --- | --- |
| [Année](../Proprietes/2512095.md) | Renvoie ou modifie l'année de la date spécifiée (sous forme d'entier). |
| [Heure](../Proprietes/2512038.md) | Renvoie ou modifie l'heure (sous forme d'entier). |
| [Jour](../Proprietes/2512043.md) | Renvoie ou modifie le jour de la date spécifiée (sous forme d'entier). |
| [Milliseconde](../Proprietes/2512062.md) | Renvoie ou modifie le nombre de millisecondes (sous forme d'entier). |
| [Minute](../Proprietes/2512050.md) | Renvoie ou modifie le nombre de minutes (sous forme d'entier). |
| [Mois](../Proprietes/2512052.md) | Renvoie ou modifie le mois de la date spécifiée (sous forme d'entier). |
| [PartieDate](../Proprietes/2512065.md) | Renvoie ou modifie la date de la variable DateHeure (sous forme de date). |
| [PartieHeure](../Proprietes/2512069.md) | Renvoie ou modifie l'Heure de la variable DateHeure (sous forme d'heure). |
| [Seconde](../Proprietes/2512081.md) | Renvoie ou modifie le nombre de secondes (sous forme d'entier). |


```wl
DateDébut est une DateHeure = "200112011530"
// Ajoute 5 ans à la date
DateDébut.Année += 5
// Modifie le mois
DateDébut.Mois = 5
// Affiche la date
Info("Date " + DateDébut.PartieDate)
// Affiche l'heure
Info("Heure " + DateDébut.PartieHeure)
```

### Intervalle de données
<a name="intervalle_donnees_ELTPARAGRAPHE000166"></a>Le type **DateHeure** permet de gérer des données comprises entre le 01/01/0001 à 00h00 et le 31/12/9999 à 23h59.


## Liste des mots réservés
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les mots réservés du WLangage sont les suivants :


|   |   |
| --- | --- |
| [Allouer](../Motscles/1511011.md) | Permet de réserver de la place en mémoire pour contenir un tableau dynamique, un objet de classe dynamique, une structure dynamique. |
| [BLOC](../Motscles/1511026.md) | Permet de définir un bloc de code sous l'éditeur de code. |
| [EXTERNE](../Motscles/1511013.md) | Permet soit d'intégrer un fichier texte contenant du WLangage dans une application, soit de déclarer un objet externe |
| [Libérer](../Motscles/1511012.md) | Permet de libérer explicitement la mémoire occupée par un tableau dynamique, un objet de classe dynamique, une structure dynamique. |
| [MaFenêtre](../Motscles/1511004.md) | Permet de manipuler la fenêtre courante. |
| [MaFenêtreCoulissanteDroite](../Motscles/1511023.md) | Permet de manipuler la fenêtre coulissante droite associée à la fenêtre courante. |
| [MaFenêtreCoulissanteGauche](../Motscles/1511022.md) | Permet de manipuler la fenêtre coulissante gauche associée à la fenêtre courante. |
| [MaFenêtreFille](../Motscles/1511021.md) | Permet de manipuler la fenêtre fille de la fenêtre courante. |
| [MaFenêtreInterne](../Motscles/1511024.md) | Permet de manipuler la fenêtre interne courante. |
| [MaPage](../Motscles/1511017.md) | Permet de manipuler la page courante. |
| [MaSource](../Motscles/1511001.md) | Permet de manipuler la source de données (fichier, vue ou requête) courante. |
| [MesParamètres](../Motscles/1511018.md) | Permet de manipuler les paramètres d'une procédure WLangage (procédure classique ou procédure à nombre de paramètres variable) |
| [Modulo](../Motscles/1511002.md) | Retourne le reste d'une division entière. |
| [MoiMême](../Motscles/1511007.md) | Permet de manipuler le champ en cours. |
| [MonEtat](../Proprietes/2510086.md) | Permet de manipuler l'état courant |
| [MonParent](../Motscles/1511016.md) | Permet de manipuler le superchamp en cours à partir d'un de ses champs |
| [Null](../Proprietes/2512064.md) | Permet de :<br><br>- indiquer qu'un paramètre ne doit pas être pris en compte dans une requête : Null et les requêtes<br><br>- spécifier qu'une variable de type variant ne contient pas de valeur : Null et les variants<br><br>- comparer une valeur à 0 : Null et les numériques<br><br>- dans certaines fonctions du WLangage, indiquer que le paramètre ne doit pas être pris en compte : Null et les fonctions du WLangage<br><br><br> |
| [STOP](../Motscles/1511014.md) | Permet d'appeler le débogueur du WLangage lors d'un test sous l'éditeur. Après exécution de ce mot-clé, le test en cours s'effectuera dans le débogueur. |


A cette liste doivent être ajoutés : 

- les différents types de variables disponibles,

- les différentes fonctions WLangage,

- les différentes propriétés WLangage.

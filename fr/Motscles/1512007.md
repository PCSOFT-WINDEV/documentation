
## Opérateur d'échange (<=>)

***En anglais : Exchange operator (<=>)***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
L'opérateur **&lt;=&gt;** permet d'échanger les contenus de deux éléments. Si nécessaire, les conversions de type sont effectuées. Retourne l'adresse d'une variable sous la forme d'un entier.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
I est un entier = 5 
J est un entier = 6 
I <=> J
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Élément 1> <=> <Élément 2>
`
---

**`<Élément 1> : (Type simple)`**

Nom de la variable, du champ, de la rubrique HFSQL dont le contenu doit être échangé avec le contenu de &lt;Élément 2&gt;.

**`<Élément 2> : (Type simple)`**

Nom de la variable, du champ, de la rubrique HFSQL dont le contenu doit être échangé avec le contenu de &lt;Élément 1&gt;.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
L'opérateur d'échange ne fonctionne qu'avec des types simples.


## Désignation de l'élément en cours (Mot réservé)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les mots-clés permettant de désigner l'élément en cours sont :


|   |   |
| --- | --- |
| [MaFenêtre](../Motscles/1511004.md) | Permet de manipuler la fenêtre courante. |
| [MaFenêtreCoulissanteDroite](../Motscles/1511023.md) | Permet de manipuler la fenêtre coulissante droite associée à la fenêtre courante. |
| [MaFenêtreCoulissanteGauche](../Motscles/1511022.md) | Permet de manipuler la fenêtre coulissante gauche associée à la fenêtre courante. |
| [MaFenêtreInterne](../Motscles/1511024.md) | Permet de manipuler la fenêtre interne courante. |
| [MaPage](../Motscles/1511017.md) | Permet de manipuler la page courante |
| [MaSource](../Motscles/1511001.md) | Permet de manipuler la source de données courante |
| [MoiMême](../Motscles/1511007.md) | Permet de manipuler le champ courant |
| [MonChampPopup](../Motscles/1511005.md) | Permet de manipuler le champ qui a ouvert une fenêtre de type popup |
| [MonEtat](../Motscles/1511010.md) | Permet de manipuler l'état courant |
| [MonFichier](../Motscles/1511019.md) | Permet de manipuler le fichier qui a déclenché le trigger à partir du code du trigger. |
| [MonParent](../Motscles/1511016.md) | Permet de manipuler le superchamp en cours à partir d'un de ses champs |
| [Objet](../POO/6010002.md) / [This](../POO/6010002.md) | Permet de manipuler la classe en cours. |
| MonThread | Permet de manipuler l'objet Thread qui exécute le code courant. |

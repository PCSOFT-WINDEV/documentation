
## MaFenêtreCoulissanteDroite (Mot réservé)

***En anglais : MyRightSlidingWindow***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le mot-clé **MaFenêtreCoulissanteDroite** permet de manipuler la fenêtre coulissante droite associée à la fenêtre courante. Lors de l'exécution de la fenêtre, le mot-clé **MaFenêtreCoulissanteDroite** est remplacé par la fenêtre et non par le nom de la fenêtre. Le mot-clé **MaFenêtreCoulissanteDroite** s'utilise alors comme une fenêtre.

**Intérêt** : Le mot-clé **MaFenêtreCoulissanteDroite** permet de manipuler depuis la fenêtre en cours les champs présents dans la fenêtre coulissante droite associée à la fenêtre en cours.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MaFenêtreCoulissanteDroite.ConnecteAprès(Faux)
FenCoulissanteVisible(fcDroite, Vrai)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`MaFenêtreCoulissanteDroite`
---



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Manipulation des champs de la fenêtre coulissante depuis la fenêtre en cours
<a name="manipulation_des_champs_fenetre_coulissante_depuis_fenetre_cours_ELTPARAGRAPHE000034"></a>Le mot-clé **MaFenêtreCoulissanteDroite** est toujours remplacé par la fenêtre coulissante droite définie pour la fenêtre en cours. Par exemple, utilisation de la propriété [Couleur](../Proprietes/2510071.md) sur un champ Libellé :


```wl
// Change la couleur d'un libellé dans la fenêtre coulissante à droite
MaFenêtreCoulissanteDroite.LIB_Libellé.Couleur = RougeClair
```

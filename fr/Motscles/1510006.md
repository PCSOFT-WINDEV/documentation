## Instructions conditionnelles
<a name="instructions_conditionnelles_ELTTEXTE000135"></a>
Les instructions conditionnelles du WLangage sont les suivantes : 



|   |   |
| --- | --- |
| [SELON](../Motscles/1510008.md) | Exécution d'une action ou d'une autre en fonction d'une expression |
| [SI](../Motscles/1510011.md) | Exécution d'une action ou d'une autre en fonction d'une condition |

## Instructions répétitives
<a name="instructions_repetitives_ELTTEXTE000159"></a>
Les instructions répétitives du WLangage sont les suivantes : 



|   |   |
| --- | --- |
| [BOUCLE](../Motscles/1510004.md) | Bloc d'instructions répété à l'infini |
| [POUR](../Motscles/1510001.md) | Bloc d'instructions répété jusqu'à une valeur limite |
| [POUR TOUT](../Motscles/1510016.md) | Parcours de tableaux |
| [POUR TOUT](../Motscles/1510018.md) | Parcours de chaînes |
| [POUR TOUT](../Motscles/1510017.md) | Parcours de champs |
| [POUR TOUT](../Motscles/1510013.md) | Parcours HyperFileSQL (parcours complet, parcours avec recherche, parcours avec filtre) |
| [TANTQUE](../Motscles/1510010.md) | Bloc d'instructions répété en fonction d'une condition |


L'instruction POUR TOUT peut également être utilisée pour parcourir les objets DotNet qui implémentent l'interface IEnumerable.


## Utilisation des types avancés
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000163"></a>
Les types avancés (variables de type XLS, ggl, ...) permettent de manipuler des données à l'aide de propriétés.

Pour manipuler ces types, il faut déclarer une variable correspondant au type à manipuler et instancier cette variable.

Cette page d'aide présente :

- comment manipuler les propriétés d'un type avancé.

- l'affectation des types avancés.






<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Manipuler les propriétés d'un type avancé
<a name="manipuler_les_proprietes_type_avance_ELTTEXTE000187"></a>


### Présentation
<a name="presentation_ELTPARAGRAPHE000026"></a>Une propriété d'un type avancé est une donnée associée à la variable.


<a name="NOTE3_2"></a>


### Utiliser une propriété d'une variable d'un type avancé
<a name="utiliser_une_propriete_une_variable_type_avance_ELTPARAGRAPHE000034"></a>Pour appeler une propriété d'une variable, la syntaxe à utiliser est :


```txt
<Nom de la variable> . <Nom de la propriété>
```


ou 


```txt
<Nom de la variable> .. <Nom de la propriété>
```



<a name="NOTE3_3"></a>


### Remarque
<a name="remarque_ELTPARAGRAPHE000048"></a>Il est possible d'enchaîner les accès aux propriétés. Par exemple :


```wl
// Enchaînements de propriétés
MaCampagne..GéoCiblage..PaysCible..Pays = "FR" + RC + "DE"
```

## Affectation des variables des types avancés
<a name="affectation_des_variables_des_types_avances_ELTTEXTE000223"></a>


### Règles d'affectation
<a name="regles_affectation_ELTPARAGRAPHE000060"></a>Les règles d'affectation entre des variables de type avancé et entre des variables dynamiques de type avancé sont les suivantes :


|   |   |   |   |   |   |   |
| --- | --- | --- | --- | --- | --- | --- |
|   | **Opérateur =** | **Opérateur &lt;=** | **Opérateur &lt;-** |
| Variable de type avancé | Variable dynamique de type avancé | Variable de type avancé | Variable dynamique de type avancé | Variable de type avancé | Variable dynamique de type avancé |
| Variable de type avancé | Copie | Copie | Copie | Copie | Prise de référence | Prise de référence |
| Variable dynamique de type avancé | Prise de référence | Prise de référence | Copie | Copie | Prise de référence | Prise de référence |

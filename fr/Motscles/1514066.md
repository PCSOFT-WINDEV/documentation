
## Date (Type de variable)

***En anglais : Date***
				



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le type **Date** permet de manipuler simplement des dates. Ce type permet de gérer de manière quasi automatique les conversions (fonctions [ChaîneVersDate](../WDLang1/3027013.md), [DateVersChaîne](../WDLang1/3027025.md), [EntierVersDate](../WDLang1/3027009.md), [DateVersEntier](../WDLang1/3027014.md), ...).

Ce type peut être utilisé aussi bien :

- pour récupérer puis manipuler des rubriques HFSQL au format Date.

- pour récupérer puis manipuler les champs de saisie de type Date.

- dans les fonctions du WLangage permettant de gérer les dates.

- pour réaliser des calculs sur les dates (différence, addition, ...). Pour plus de détails, consultez [Opérations disponibles sur le type Date](../Motscles/1514040.md).




**Remarques** :

- Plusieurs propriétés peuvent être utilisées avec le type **Date**, pour récupérer par exemple qu'une partie de la date.

- Pour plus de détails sur la déclaration de ce type de variable et l'utilisation des propriétés WLangage, consultez [Déclaration d'une variable](../Motscles/1514032.md).







<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
DateDébut est une Date = 1205
DateFin est une Date = "20011231"     // 31/12/2001
DateJour est une Date = DateSys()
```

## Remarques




### Déclaration et initialisation
<a name="declaration_initialisation_ELTPARAGRAPHE000100"></a>Une variable de type **Date** peut être initialisée avec une valeur de type :

- **Date**

- **DateHeure** : seule la date sera prise en compte.

- **Chaîne de caractères**, au format AAAAMMJJ (AAAA : année, MM : Mois, JJ : Jour)

- **Entier** : la date correspondra alors au nombre de jours écoulés entre le 1er janvier 1800 et la date spécifiée.


Par défaut, une variable de type **Date** est initialisée avec la date du jour (date système). Pour régler la valeur par défaut affectée aux variables de type **Date**, [Heure](../Motscles/1514068.md) et [DateHeure](../Motscles/1514070.md), utilisez la fonction [DateHeureParDéfaut](../WDLang1/3027015.md).

Remarque : Il est également possible d'initialiser une variable de type **Date** avec la date système en utilisant la fonction [Maintenant](../WDLang1/3027024.md). 


<a name="NOTE0_2"></a>




### Propriétés utilisables sur les dates 
<a name="proprietes_utilisables_sur_les_dates_ELTPARAGRAPHE000132"></a>Les propriétés WLangage utilisables sur les dates sont les suivantes : 


|   |   |
| --- | --- |
| [Année](../Proprietes/2512095.md) | Renvoie ou modifie l'année de la date spécifiée (sous forme d'entier). |
| [Jour](../Proprietes/2512043.md) | Renvoie ou modifie le jour de la date spécifiée (sous forme d'entier). |
| [Mois](../Proprietes/2512052.md) | Renvoie ou modifie le mois de la date spécifiée (sous forme d'entier). |





Pour plus de détails sur les opérations possibles sur les dates (addition, soustraction, comparaison), consultez [Opérations disponibles sur le type Date](../Motscles/1514040.md).



```wl
DateDébut est une Date = "20011201"
// Ajoute 5 ans à la date
DateDébut.Année += 5
// Modifie le mois
DateDébut.Mois = 5
// Calcule la fin (30 jours après)
soit DateFin = DateDébut
DateFin.Mois++
DateFin.Jour--
```



<a name="NOTE0_3"></a>




### Plage de dates
<a name="plage_dates_ELTPARAGRAPHE000165"></a>Le type **Date** permet de gérer des dates comprises entre le 01/01/0001 et le 31/12/9999.

**Remarque** : Les calculs sur les dates réalisés avec les [fonctions WLangage](../WDLang1/3027029.md) ou les [propriétés WLangage](../Proprietes/2514010.md) sont fiables à partir du 1er Janvier 1583.

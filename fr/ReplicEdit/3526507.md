
## Description de la réplication (ReplicEdit)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000124"></a>
ReplicEdit permet de créer une réplication à l'aide d'un assistant mais des paramètres supplémentaires peuvent être connus ou configurés via la fenêtre de description de la réplication. 

**Pour éditer la description d'une réplication**, sélectionnez l'option "Réplication .. Description de la réplication". 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Les différentes options
<a name="les_differentes_options_ELTTEXTE000148"></a>


### Onglet "Général"
<a name="onglet_general_ELTPARAGRAPHE000018"></a>L'onglet "Général" présente les informations suivantes : 

- Nom de la réplication, 

- Description de la réplication

- Sens de réplication, 

- Gestion des conflits. 




Ces informations ont été saisies lors de la création de la réplication et peuvent être modifiées à tout moment. 


### Onglet "Paramètres"
<a name="onglet_parametres_ELTPARAGRAPHE000030"></a>L'onglet "Paramètres" indique les différents paramètres attendus par la réplication. 

Ces paramètres correspondent aux filtres posés sur les fichiers à répliquer. 

Il est ainsi possible d'obtenir l'ordre des paramètres utilisés par la fonction [RéplicInitialise](../WDLang4/1000017195.md). 



### Onglet "Fichiers maîtres"
<a name="onglet_fichiers_maitres_ELTPARAGRAPHE000042"></a>Cet onglet permet d'indiquer : 

- lors de la réplication d'une base de données HFSQL Classic, le répertoire de base des fichiers maîtres HFSQL Classic. 

- le mode d'accès et la localisation des fichiers maîtres de la réplication. 





### Onglet "Avancé"
<a name="onglet_avance_ELTPARAGRAPHE000050"></a>Cet onglet permet d'indiquer les options avancées de la réplication : 

- **** : Cette option permet de déterminer automatiquement l'ordre des fichiers répliqués (selon les contraintes d'intégrité). Si cette option n'est pas sélectionné, l'ordre est déterminé par l'ordre défini dans l'onglet "Paramètres". 

- **** : Cette option permet de prendre en compte la modification automatique des données. Dans ce cas : 

	- Une modification de la structure de la base de données maître sera reportée sur la base de données abonnée.

	- Les nouvelles rubriques seront prises en compte par la réplication. 




- <u><u><u><u>Faux</u></u></u></u> (valeur par défaut). La modification automatique des données effectuée sur la base de données maître n'est pas reportée sur la base de données abonnée.  Attention : ce paramètre est disponible uniquement pour une réplication entre des bases de données HFSQL (Classic ou Client/Serveur).

## Présentation
<a name="presentation_ELTTEXTE000172"></a>


### L'utilitaire ReplicEdit
<a name="utilitaire_replicedit_ELTPARAGRAPHE000011"></a>ReplicEdit est un outil permettant de définir les caractéristiques d'une réplication. Cette réplication sera ensuite utilisée par la réplication universelle assistée soit en mode automatique (avec les outils ReplicSynchro et ReplicAdmin), soit en mode programmé. 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Lancement de ReplicEdit
<a name="lancement_replicedit_ELTTEXTE000196"></a>


### Lancement de ReplicEdit
<a name="lancement_replicedit_ELTPARAGRAPHE000020"></a>Pour lancer ReplicEdit :

- sous le volet "Outils", dans le groupe "Base de données", déroulez "Réplication" et sélectionnez "ReplicEdit - Editeur de réplication".

- lancez directement le programme "ReplicEdit.EXE".



<a name="NOTE2_2"></a>


### Condition d'utilisation
<a name="condition_utilisation_ELTPARAGRAPHE000042"></a>ReplicEdit est un outil redistribuable.
<a name="NOTE2_3"></a>

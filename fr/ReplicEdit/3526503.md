
## ReplicEdit : Prise en compte des modifications de l'analyse
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000081"></a>
Lors de la modification de l'analyse d'une application répliquée (ajout d'une rubrique, d'un fichier, ...), il est nécessaire de prendre en compte cette modification dans la description de la réplication.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comment le faire ?
<a name="comment_faire_ELTTEXTE000105"></a>
Pour prendre en compte les modifications effectuées dans l'analyse dans le fichier .WER de la réplication :

1. Ouvrez le fichier .WER voulu sous ReplicEdit.

2. Sélectionnez l'option "Réplication .. Se mettre à jour de la description de l'analyse".





<a name="NOTE2_2"></a>


### Attention
<a name="attention_ELTPARAGRAPHE000028"></a>Si les connexions définies dans l'analyse nécessitent des mots de passe, il est nécessaire de ressaisir ces mots de passe dans ReplicEdit (pour plus de détails, consultez [Options de réplication](../ReplicEdit/3526502.md)).

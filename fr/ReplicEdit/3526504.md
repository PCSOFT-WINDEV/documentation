
## Ajouter des fichiers dans une réplication (ReplicEdit)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000109"></a>
Lorsque la réplication est décrite, il est nécessaire d'ajouter les fichiers de données pris en compte dans la réplication. Cette opération est réalisée à l'aide d'un assistant.

## Ajouter des fichiers de données dans une réplication
<a name="ajouter_des_fichiers_donnees_dans_une_replication_ELTTEXTE000133"></a>


### Ajout du fichier de base
<a name="ajout_fichier_base_ELTPARAGRAPHE000021"></a>

1. Sélectionnez l'option "Réplication .. Ajouter un fichier de données". L'assistant d'ajout de fichier se lance.

2. La liste des fichiers de données décrits dans l'analyse en cours ou présents sur le serveur est affichée. 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ReplicEdit_FichierDeBase.gif&type=thumb)
Vous pouvez sélectionner un ou plusieurs fichiers de données à synchroniser. Par défaut, tous les enregistrements sont répliqués. 

3. Pour les fichiers de données sélectionnés, vous pouvez définir : 

	- le sens de réplication des fichiers de données : 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ReplicEdit_FichierSens.gif&type=thumb)


	- la gestion des conflits de modification. 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ReplicEdit_FichierConflit.gif&type=thumb)

			Il est possible de choisir : 

	- le mode par défaut défini précédemment (conseillé). 

	- un autre mode. 

4. L'assistant est terminé. La réplication est décrite. L'éditeur de réplication liste les fichiers de données pris en compte dans la réplication. 




Les fichiers affichés en gris sont des fichiers de données liés : un fichier de données lié est un fichier de données qui est lié à un fichier de base (appartenant déjà à la réplication). Selon la réplication effectuée, les données du fichier lié peuvent être ajoutées ou non. 

Vous pouvez enregistrer la réplication (option "Fichier .. Enregistrer"). La réplication est enregistrée sous forme d'un fichier de type ".Wer". 

Ce fichier contient la description de la réplication. Il devra être présent à côté de l'exécutable sur les postes abonnés.


### Ajout d'un fichier lié
<a name="ajout_fichier_lie_ELTPARAGRAPHE000047"></a>Pour ajouter un fichier lié : 

1. Double-cliquez sur le fichier de données lié (affiché en gris) sous ReplicEdit. L'assistant d'ajout d'un fichier de données lié se lance.

2. Par défaut, les données du fichier lié sont celles correspondantes au fichier de base. Ainsi, si le fichier de base est filtré, les enregistrements du fichier lié utilisent le même filtre. 
	Remarque : Pour mettre un filtre sur le fichier de base, affichez la [description du fichier](../ReplicEdit/3526506.md). 

3. Indiquez si un filtrage supplémentaire doit être effectué. Si vous ajouter un filtrage, donnez les caractéristiques nécessaires. 

4. Dans la suite de l'assistant, indiquez le sens de réplication du fichier lié ainsi que le mode de gestion des conflits.

5. Validez l'assistant. Le fichier lié apparaît désormais en fichier à répliquer.

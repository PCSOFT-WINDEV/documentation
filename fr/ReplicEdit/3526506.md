
## Description des fichiers répliqués (ReplicEdit)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000119"></a>
Une réplication est composée de un ou plusieurs fichiers de données. RepliEdit permet à tout moment d'éditer la description d'un fichier de données pris en compte par la réplication. 

Pour éditer la description d'un fichier de données : 

1. Sélectionnez le fichier de données dans la liste affichée sous l'éditeur de réplication. 

2. Double-cliquez sur son nom ou sélectionnez l'option "Propriétés" du menu contextuel.

## Les différentes options
<a name="les_differentes_options_ELTTEXTE000143"></a>


### Onglet "Général"
<a name="onglet_general_ELTPARAGRAPHE000023"></a>L'onglet "Général" présente les informations suivantes : 

- Nom du fichier, 

- Mode d'accès au fichier maître, 

- Sens de réplication, 

- Gestion des conflits. 




Ces informations ont été saisies lors de l'ajout du fichier dans la réplication et peuvent être modifiées à tout moment. 


### Onglet "Filtre"
<a name="onglet_filtre_ELTPARAGRAPHE000035"></a>L'onglet "Filtre" permet de filtrer les données à répliquer. 

Prenons un exemple simple : Un commercial veut récupérer localement la base des clients. Au lieu de mettre à jour toute la base, il ne prend que les clients habitant en France. 

Pour créer un filtre sur un fichier de données : 

1. Dans l'onglet "Filtre", sélectionnez l'option "Sélection". 

2. La liste des rubriques présentes dans le fichier de données apparaît. 

3. Sélectionnez la ligne correspondant à la rubrique à filtrer. 

4. Dans la colonne "Condition", sélectionnez la condition à appliquer à la rubrique. 

5. Dans la colonne "Valeur ou paramètre", indiquez : 

	- Soit la valeur voulue (par exemple "France").

	- Soit un paramètre (par exemple "Ville"). Ce paramètre sera initialisé avec la valeur voulue au moment de la synchronisation 







**Remarque** : Il est également possible de saisir directement un filtre manuel. 

**Conseils** :

- Filtrez les données répliquées. Ainsi, la réplication sera plus rapide, le nombre d'enregistrements répliqué étant limité.

- Donnez un nom explicite aux paramètres.

	- Ces paramètres pourront être utilisés dans la fonction [ReplicInitialise](../WDLang4/1000017195.md).  

	- Ces paramètres seront visualisés par les utilisateurs finaux qui lanceront la réplication à l'aide de ReplicSynchro.










### Onglet "Rubriques"
<a name="onglet_rubriques_ELTPARAGRAPHE000067"></a>L'onglet "Rubriques" permet de configurer les rubriques à répliquer. Par défaut, toutes les rubriques sont répliquées. 

Cet onglet permet également d'indiquer la rubrique de datation utilisée pour la réplication.

## Présentation
<a name="presentation_ELTTEXTE000245"></a>


### L'utilitaire WDOptimiseur
<a name="utilitaire_wdoptimiseur_ELTPARAGRAPHE000011"></a>Les différentes opérations pouvant être réalisées avec WDOptimiseur sont les suivantes :

- **Optimisation des index**
	Cette option permet d'effectuer plusieurs types d'opérations :

	1. **Vérifier les index** : Vérifie la cohérence entre l'index et les fichiers de données.

	2. **Optimiser la vitesse des index (Re-calcul des statistiques)** : Optimise les index en calculant les statistiques sur les index. Ces statistiques permettent d'optimiser les filtres, les requêtes et les vues HFSQL. 
			Remarque : Plus le fichier de données est modifié et moins les statistiques reflètent le contenu du fichier de données HFSQL. Plus le fichier de données contient d'enregistrements et moins la modification d'un enregistrement a d'impact sur les statistiques.

	3. **Reconstruire les index** : Optimise l'accès aux enregistrements du fichier de données et re-calcule les statistiques. Tous les enregistrements rayés sont automatiquement supprimés. 

	4. **Reconstruire les index et les mémos** : Optimise la totalité du fichier de données, de son index et ses mémos.

	5. ** Réviser et compresser les index et les mémos** : Optimise la totalité du fichier de données, de son index et ses mémos. Les mémos seront compressés. Des options avancées sont disponibles ("Options de compression des mémos").




- **Visualisation des propriétés d'un fichier**
	Affiche les caractéristiques d'un fichier de données (nom, extension, format, cryptage, ...).

- **Édition et modification d'un fichier ".REP"**
	Optimise l'accès aux fichiers de données.

- **Annulation d'une transaction ou libération d'enregistrements en transaction**
	Permet d'annuler une transaction sans interrompre l'exécution du programme.

## Utilisation
<a name="utilisation_ELTTEXTE000275"></a>


### Lancement de WDOptimiseur
<a name="lancement_wdoptimiseur_ELTPARAGRAPHE000043"></a>WDOptimiseur peut être lancé en mode interactif grâce à une des méthodes suivantes :

- Depuis WINDEV, WEBDEV ou WINDEV Mobile, sous le volet "Outils", dans le groupe "Base de données", cliquez sur "WDOptimiseur".

- Lancez directement le programme "WDOptimiseur.EXE".




Les différentes opérations pouvant être réalisées avec WDOptimiseur sont les suivantes :

1. Optimiser les index. Pour plus de détails, consultez [Optimisation des index](../WDOptimiseur/3519004.md).

2. Visualiser les propriétés d'un fichier. Ce fichier de données peut être sélectionné dans la liste affichée à l'écran, ou sélectionné grâce au sélecteur de fichiers.

3. Éditer un fichier .REP : Il suffit de sélectionner le fichier .REP voulu. Pour plus de détails, consultez [Édition et modification d'un fichier ".REP"](../WDOptimiseur/3519001.md)

4. Manipuler les transactions. Pour plus de détails, consultez [Annulation d'une transaction ou libération d'enregistrements en transaction](../WDOptimiseur/3519006.md)




**Remarque** : Sur le poste de développement, WDOptimiseur peut aussi être lancé directement depuis WDOutil.

Il est également possible de lancer WDOptimiseur en ligne de commande. Pour plus de détails, consultez [Lancer WDOptimiseur en ligne de commande](../WDOptimiseur/3519005.md).
<a name="NOTE2_2"></a>


### Conditions d'utilisation
<a name="conditions_utilisation_ELTPARAGRAPHE000086"></a>WDOptimiseur est un outil redistribuable. WDOptimiseur peut être installé avec les applications développées avec WINDEV ou WEBDEV. La licence d'utilisation de WINDEV ou WEBDEV s'applique intégralement.

Lors de la création de la procédure d'installation, WDOptimiseur est automatiquement livré avec vos applications.

**Rappel** : La procédure d'installation peut être créée : 

- soit à partir de l'assistant : sous le volet "Projet", dans le groupe "Génération", cliquez sur "Procédure d'installation". 

- soit à partir de l'éditeur d'installation : sous le volet "Outils", dans le groupe "Utilitaires", cliquez sur "WDInst".

## Rappel : Index d'un fichier
<a name="rappel_index_fichier_ELTTEXTE000311"></a>
L'index d'un fichier de données est un fichier spécifique (.NDX). Ce fichier est créé en fonction des clés de parcours définies sur le fichier de données. Il permet de retrouver rapidement des données dans un fichier de données.

Pour un meilleur fonctionnement des recherches dans vos fichiers de données, il est conseillé d'optimiser l'index régulièrement.

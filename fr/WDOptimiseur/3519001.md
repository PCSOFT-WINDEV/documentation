
## WDOptimiseur : Édition et modification d'un fichier ".REP"
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000105"></a>
Le fichier &lt;MonProjet&gt;.REP est un fichier contenant la liste des fichiers manipulés par l'application (identifiant, nom logique et chemin complet du fichier physique).

Le GUID de l'analyse est l'identifiant unique de l'analyse liée au projet, contenant la description des fichiers. Cet identifiant peut être connu sous l'éditeur d'analyses, dans la description du de l'analyse (onglet "Options").

Le GUID du fichier correspond à l'identifiant du fichier logique. Cet identifiant peut être connu sous l'éditeur d'analyses, dans la description du fichier (onglet "Notes").

Ce fichier est automatiquement créé dans le répertoire de l'application et renseigné par le moteur HFSQL.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Le fichier .REP
<a name="fichier_rep_ELTTEXTE000129"></a>


### A quoi sert le fichier .REP ?
<a name="quoi_sert_fichier_rep_ELTPARAGRAPHE000025"></a>Le fichier .REP permet de localiser facilement les fichiers de données qui ont été utilisés par l'application WINDEV ou le site WEBDEV.

L'application WINDEV ou le site WEBDEV met à jour automatiquement le fichier .REP mais ne se sert pas (ou rarement) du fichier .REP.

Ce fichier est utilisé par tous les outils devant manipuler les fichiers de l'application, et principalement par la mise à jour automatique des fichiers, etc.


<a name="NOTE2_2"></a>


### Exemple : Mise à jour d'une application avec modification de l'analyse
<a name="exemple_mise_jour_une_application_avec_modification_analyse_ELTPARAGRAPHE000037"></a>Lors de la mise à jour d'une application WINDEV, la modification automatique des fichiers de données est automatiquement lancée en cas de modifications sur la structure de la base de données.

Cette procédure utilise le fichier .REP pour retrouver les fichiers physiques utilisés par l'application afin de les modifier.

## Éditer et modifier un fichier .REP
<a name="editer_modifier_fichier_rep_ELTTEXTE000159"></a>


### Editer et modifier un fichier .REP
<a name="editer_modifier_fichier_rep_ELTPARAGRAPHE000049"></a>Pour éditer et modifier un fichier .REP :

1. Lancez WDOptimiseur.

2. Sélectionnez l'option "Éditer et modifier un fichier .REP".

3. Sélectionnez le fichier .REP. Les fichiers contenus dans le fichier .REP sélectionné sont listés.

4. Pour supprimer l'ensemble des fichiers qui n'existent pas, cliquez sur le bouton "Nettoyer".

5. Pour supprimer le fichier sélectionné, cliquez sur le bouton "Supprimer".


## Erreur 1013 : L'accès au membre de la classe est restreint par un héritage protégé
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000070"></a>
Vous tentez d'accéder à un membre d'une classe précédemment dérivée par un héritage protégé. Il n'est pas possible d'accéder à ce membre depuis un code extérieur à la classe dérivée (ou extérieur à ses propres classes dérivées).



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correction
<a name="correction_ELTTEXTE000094"></a>
Deux possibilités : 

- Accédez à ce membre par une méthode de la classe dérivée ou d'une de ses propres classes dérivées. 

- Transformez l'héritage protégé en héritage public : le membre sera alors accessible aussi bien depuis le code des classes dérivées que depuis un code extérieur à la classe.

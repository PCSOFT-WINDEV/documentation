
## Erreur 34 : 'Sortir' est interdit en dehors des boucles
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000088"></a>
Vous avez utilisé le mot-clé SORTIR pour interrompre une procédure.

**Rappel** : Le mot-clé SORTIR doit être utilisé uniquement pour interrompre une boucle.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correction
<a name="correction_ELTTEXTE000112"></a>
Pour sortir d'une procédure, utilisez les mots-clés RETOUR ou RENVOYER (si une valeur doit être renvoyée au traitement appelant).



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Exemple
<a name="exemple_ELTTEXTE000136"></a>


### Utilisation du mot-clé SORTIR pour interrompre une procédure
<a name="utilisation_motcle_sortir_pour_interrompre_une_procedure_ELTPARAGRAPHE000029"></a>

**Code provoquant l'erreur** 


```wl
PROCEDURE Compare(Chiffre1, Chiffre2)
SI Chiffre1>Chiffre2 ALORS 
	Info(Chiffre1 + "est supérieur à" + Chiffre2)
	SORTIR
FIN
```




**Correction proposée**

Le mot-clé SORTIR doit être remplacé par le mot-clé RETOUR.



```wl
PROCEDURE Compare(Chiffre1, Chiffre2)
SI Chiffre1>Chiffre2 ALORS 
	Info(Chiffre1 + "est supérieur à" + Chiffre2)
	RETOUR
FIN
```


## Erreur 30 : Il est interdit de déclarer un traitement d'exception à l'intérieur d'un  traitement ...
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000085"></a>
Le traitement en cours utilise un traitement d'exception. Dans ce traitement d'exception, une nouvelle gestion des exception est mise en place. Il est impossible de gérer les exceptions dans un traitement d'exception.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Conseil
<a name="conseil_ELTTEXTE000109"></a>
Le code de traitement d'une exception doit être vérifié avec soin. En effet, si ce code contient une erreur, cette erreur ne pourra pas être interceptée par un traitement d'exception. 



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Exemples
<a name="exemples_ELTTEXTE000133"></a>


### Un traitement d'exception permet de gérer une erreur intervenue dans le premier traitement d'exception
<a name="traitement_exception_permet_gerer_une_erreur_intervenue_dans_premier_traitement_exception_ELTPARAGRAPHE000027"></a>

**Code provoquant l'erreur**


```wl
SI EnModeTest() = Faux ALORS
	QUAND EXCEPTION
	Erreur("Erreur:" + ExceptionInfo(errMessage) + " Code " + ExceptionInfo(errCODE) + ...
			 " à la ligne " + ExceptionInfo(errLIGNE) + " de "  + ...
			  ExceptionInfo(errTRAITEMENT) , "EXC")
		QUAND EXCEPTION
			// Si on arrive pas a écrire dans le .ini, on quitte le programme
			FinProgramme() 
		FIN
		// Procédure d'écriture dans un fichier INI
		bEcritDansIni("")
	FIN
FIN
```


**Correction proposée**

Supprimez le traitement d'exception inutile et vérifiez le traitement d'exception.



```wl
SI EnModeTest() = Faux ALORS
	QUAND EXCEPTION
	Erreur("Erreur:" + ExceptionInfo(errMessage) + " Code " + ExceptionInfo(errCODE) + ...
			 " à la ligne " + ExceptionInfo(errLIGNE) + " de "  + ...
			  ExceptionInfo(errTRAITEMENT) , "EXC")
		// Procédure d'écriture dans un fichier INI
		bEcritDansIni("")
	FIN
FIN
```

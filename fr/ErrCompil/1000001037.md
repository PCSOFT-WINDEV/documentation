
## Erreur 1038 : Ce membre existe déjà dans la classe
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000085"></a>
Vous déclarez une classe utilisant deux membres de même nom. Cette erreur peut apparaître suite à une faute de frappe par exemple dans le nom d'un des membres de la classe.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correction
<a name="correction_ELTTEXTE000109"></a>
Renommez ou supprimez le membre en trop.

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Exemple
<a name="exemple_ELTTEXTE000133"></a>


### Déclaration d'une classe avec deux membres de même nom
<a name="declaration_une_classe_avec_deux_membres_meme_nom_ELTPARAGRAPHE000025"></a>**Code provoquant l'erreur** 


```wl
Coord est une classe
	PosX est un entier
	PosX est un entier
FIN
```


 
**Correction proposée**
Renommer un des membres.



```wl
Coord est une classe
	PosX est un entier
	PosY est un entier
FIN
```

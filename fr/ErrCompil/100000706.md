
## Erreur 706 : La propriété n'existe pas sur ce type de champ
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000086"></a>
Vous tentez d'exécuter une propriété. Cette propriété ne peut pas être utilisée sur le type de champ spécifié.

**Remarque** : La complétion automatique propose automatiquement les propriétés correspondant au type de champ manipulé. Si la propriété n'est pas présente dans cette liste, elle ne pourra pas être utilisée sur ce type de champ.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correction
<a name="correction_ELTTEXTE000110"></a>
Consultez l'aide en ligne pour rechercher la propriété voulue et le type de champ sur lequel cette propriété peut être appliquée. 



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Exemple
<a name="exemple_ELTTEXTE000134"></a>


### Utilisation de la propriété Abréviation sur un champ
<a name="utilisation_propriete_abreviation_sur_champ_ELTPARAGRAPHE000029"></a>

**Code provoquant l'erreur** 


```wl
Trace(BTN_Bouton1..RubriqueAffichée)
```




**Correction proposée**

Utiliser une autre propriété.

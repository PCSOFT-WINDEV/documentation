
## Erreur 2 : Ligne incomplète
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000125"></a>
La ligne de code en cours est incomplète : l'instruction du WLangage en cours n'est pas terminée. Les cas les plus courants sont :

- L'instruction utilisée n'est pas terminée : il manque une parenthèse par exemple.

- La ligne de code se prolonge sur la ligne suivante.




**Rappel** : Dans l'éditeur de code, le passage à la ligne suivante (utilisation de la touche ENTREE) est considérée comme un séparateur d'instructions.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correction
<a name="correction_ELTTEXTE000149"></a>
Vérifiez la syntaxe de la fonction et complétez la ligne.



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Exemples
<a name="exemples_ELTTEXTE000173"></a>


### La syntaxe de la fonction est incomplète : il manque la parenthèse fermante
<a name="syntaxe_fonction_est_incomplete_manque_parenthese_fermante_ELTPARAGRAPHE000032"></a>

**Code provoquant l'erreur** 


```wl
HLitPremier(Client, Nom
```




**Correction proposée**

Ajoutez la parenthèse fermante en fin de ligne.



```wl
HLitPremier(Client, Nom)
```



<a name="NOTE3_2"></a>


### Expression de calcul incorrecte
<a name="expression_calcul_incorrecte_ELTPARAGRAPHE000049"></a>

**Code provoquant l'erreur** 


```wl
MaVariable = 3 +
```




**Correction proposée**

Complétez si nécessaire la ligne de code.



```wl
MaVariable = 3 + 3
```



<a name="NOTE3_3"></a>


### La ligne de code continue sur la ligne suivante
<a name="ligne_code_continue_sur_ligne_suivante_ELTPARAGRAPHE000066"></a>

**Code provoquant l'erreur** 


```wl
// Ajout dans le champ Table par programmation
TableAjoute(TABLE_TABLE1, CLIENT.NOMCONTACT + TAB + CLIENT.NOMSOCIETE +
	TAB + CLIENT.VILLECLIENT
```




**Correction proposée**
Utilisez la ponctuation "..." en fin de ligne pour indiquer que la ligne de code en cours continue sur la ligne suivante.



```wl
// Ajout dans le champ Table par programmation
TableAjoute(TABLE_TABLE1, CLIENT.NOMCONTACT + TAB + CLIENT.NOMSOCIETE + ...
	TAB + CLIENT.VILLECLIENT)
```

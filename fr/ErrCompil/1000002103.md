
## Information 2103 : 'Double' n'existe plus : les réels sont sur 8 octets
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000068"></a>
Vous tentez d'utiliser un réel double : les réels doubles n'existent plus en WINDEV, tous les réels sont sur 8 octets.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correction
<a name="correction_ELTTEXTE000092"></a>
Utilisez un réel.

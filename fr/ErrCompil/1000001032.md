
## Erreur 1032 : L'héritage est interdit dans une structure
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000086"></a>
Vous tentez de définir un héritage dans la définition d'une structure. L'héritage est interdit dans une structure.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correction
<a name="correction_ELTTEXTE000110"></a>
Supprimez la syntaxe d'héritage. 



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Exemple
<a name="exemple_ELTTEXTE000134"></a>


### Tentative d'héritage dans une structure
<a name="tentative_heritage_dans_une_structure_ELTPARAGRAPHE000027"></a>

**Code provoquant l'erreur** 


```wl
Coord est une structure 
	PosX est un entier
	PosY est un entier
FIN
 
TailleObjet est une structure
   un objet coord
	Hauteur est un entier
	Largeur est un entier
FIN
```




**Correction proposée**

Supprimer la syntaxe d'héritage. 



```wl
Coord est une structure 
	PosX est un entier
	PosY est un entier
FIN
 
TailleObjet est une structure
	Hauteur est un entier
	Largeur est un entier
FIN
```


## Erreur 809 : Le caractère * ne peut pas être utilisé comme paramètre obligatoire d'une fonction
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000086"></a>
Vous tentez d'utiliser le caractère \* pour remplacer un paramètre d'une fonction du WLangage. Ce paramètre est obligatoire et ne peut être remplacé par le caractère \*.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correction
<a name="correction_ELTTEXTE000110"></a>
Remplacez le caractère \* par la valeur du paramètre.



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Exemple
<a name="exemple_ELTTEXTE000134"></a>


### Utilisation du caractère \* dans la fonction ListeAjoute
<a name="utilisation_caractere_star_dans_fonction_listeajoute_ELTPARAGRAPHE000027"></a>

**Code provoquant l'erreur** 


```wl
ListeAjoute(*, "Marc")
```




**Correction proposée**

Utiliser le nom de la liste.



```wl
ListeAjoute(LISTE_Liste1, "Marc")
```

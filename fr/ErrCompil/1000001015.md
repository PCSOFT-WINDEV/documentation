
## Erreur 1015 : La méthode de la classe est protégée
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000072"></a>
Vous tentez d'accéder à une méthode protégée de la classe à partir d'un code différent d'une méthode de cette classe ou d'une classe dérivée.

**Rappel** : Une méthode protégée peut être manipulée uniquement à partir d'une méthode de la classe ou d'une classe dérivée de cette classe.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correction
<a name="correction_ELTTEXTE000096"></a>
Deux possibilités :

- Remplacer la méthode protégée par une méthode **publique** : l'accès à la méthode ne sera pas restreint.

- Ne pas manipuler cette méthode protégée depuis un des codes du projet.

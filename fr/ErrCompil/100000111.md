
## Erreur 111 : La taille de la chaîne pascal %1 est trop grande
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000089"></a>
Vous utilisez une chaîne Pascal dont la taille dépasse la taille maximale autorisée.


**Rappel** : La taille maximale d'une chaîne Pascal est 255 caractères.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correction
<a name="correction_ELTTEXTE000113"></a>
Modifiez la taille de votre chaîne Pascal en tenant compte de la valeur maximale autorisée.



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Exemple
<a name="exemple_ELTTEXTE000137"></a>


### La taille de la chaîne fixe est trop grande
<a name="taille_chaine_fixe_est_trop_grande_ELTPARAGRAPHE000030"></a>

**Code provoquant l'erreur** 


```wl
B est une chaîne Pascal de 700
```




**Correction proposée**

Diminuez la taille pour respecter la valeur maximale autorisée.



```wl
B est une chaîne Pascal de 200
```

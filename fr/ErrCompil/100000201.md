
## Erreur 201 : La méthode a déjà un attribut virtuelle
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000087"></a>
Vous déclarez une méthode virtuelle : vous avez utilisé deux fois le mot-clé "VIRTUELLE" lors de la déclaration de la méthode virtuelle.

**Rappel** : Par défaut, lors de sa création, une méthode n'est pas virtuelle.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correction
<a name="correction_ELTTEXTE000111"></a>
Supprimez le mot-clé "VIRTUELLE" en trop.



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Exemple
<a name="exemple_ELTTEXTE000135"></a>


### Déclaration d'une méthode virtuelle
<a name="declaration_une_methode_virtuelle_ELTPARAGRAPHE000029"></a>

**Code provoquant l'erreur** 


```wl
PROCEDURE VIRTUELLE PRIVEE VIRTUELLE Calcule_Bourse()
```


**Correction proposée**

Supprimez le mot-clé VIRTUELLE en trop.


```wl
PROCEDURE PRIVEE VIRTUELLE Calcule_Bourse()
```

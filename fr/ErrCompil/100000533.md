
## Erreur 533 : L'expression d'un { } doit être une chaîne
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000103"></a>
Vous utilisez un opérateur d'indirection sur une valeur ou une variable qui n'est pas une chaîne.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correction
<a name="correction_ELTTEXTE000127"></a>
Vérifiez et modifiez le type de la variable sur laquelle l'opérateur d'indirection est utilisé. 



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Exemples
<a name="exemples_ELTTEXTE000151"></a>


### Indirection sur un numérique
<a name="indirection_sur_numerique_ELTPARAGRAPHE000027"></a>

**Code provoquant l'erreur** 


```wl
{5}..Note = "Note"
```




**Correction proposée**

Remplacer la valeur numérique par une chaîne de caractères. 



```wl
{"MonChamp"}..Note = "Note"
```



<a name="NOTE3_2"></a>


### Indirection sur une variable de type numérique
<a name="indirection_sur_une_variable_type_numerique_ELTPARAGRAPHE000044"></a>

**Code provoquant l'erreur** 


```wl
MonChamp est un réel = 5
{MonChamp}..Note = "Note"
```




**Correction proposée**

Remplacer la valeur numérique par une chaîne de caractères. 



```wl
MonChamp est une chaîne = "Bouton1"
{MonChamp}..Note = "Note"
```

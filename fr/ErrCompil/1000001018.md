
## Erreur 1018 : La méthode est globale : utilisez l'opérateur :: pour y accéder
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000096"></a>
Vous tentez d'accéder à une méthode globale de la classe en utilisant la syntaxe permettant d'accéder à une méthode non globale.

Vous utilisez la syntaxe suivante :


```txt
NomClasse:NomMethode
```


Cette syntaxe ne permet pas d'accéder à une méthode globale.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correction
<a name="correction_ELTTEXTE000120"></a>
Utilisez la syntaxe permettant d'accéder à une méthode globale de la classe :


```txt
NomClasse::NomMéthode
```




<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Exemple
<a name="exemple_ELTTEXTE000144"></a>


### Utilisation de la méthode globale Détruire de la classe Dessin
<a name="utilisation_methode_globale_detruire_classe_dessin_ELTPARAGRAPHE000037"></a>

**Code provoquant l'erreur** 


```wl
Dessin:Détruire()
```




**Correction proposée**

Corriger la syntaxe utilisée : 



```wl
Dessin::Détruire()
```

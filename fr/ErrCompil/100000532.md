
## Erreur 532 : La taille de la chaîne à extraire doit être strictement positive
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000095"></a>
Vous utilisez les opérateurs d'extraction de chaîne ([[ ou ]]). La taille de la chaîne à extraire doit être positive.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correction
<a name="correction_ELTTEXTE000119"></a>
Modifiez la valeur de la taille de la chaîne à extraire.



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Exemple
<a name="exemple_ELTTEXTE000143"></a>


### Extraction d'une partie de chaîne de caractères 
<a name="extraction_une_partie_chaine_caracteres_ELTPARAGRAPHE000037"></a>

**Code provoquant l'erreur**


```wl
sChaîne est une chaîne ="WEBDEV est formidable"
Trace(sChaîne[[1 sur 0]])
```


**Correction proposée**

Modifier la valeur extraite afin que la taille de la chaîne extraite soit positive.



```wl
sChaîne est une chaîne ="WEBDEV est formidable"
Trace(sChaîne[[1 sur 5]])
```

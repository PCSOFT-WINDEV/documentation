
## Erreur 1005 : Pour accéder explicitement à un élément global de la classe '%1', utilisez la syntaxe 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000092"></a>
Vous tentez d'utiliser un élément global (membre, méthode ou constante) de la classe de base depuis la classe dérivée. 

La syntaxe utilisée n'est pas correcte. Vous utilisez la syntaxe suivante :

```txt
:NomClasseDeBase:NomElément
```




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correction
<a name="correction_ELTTEXTE000116"></a>
Pour utiliser un élément global (membre, méthode ou constante) de la classe de base depuis une classe dérivée, il est nécessaire de préfixer l'élément par le nom de la classe de base.

```txt
NomClasseDeBase:NomElément
```




<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Exemple
<a name="exemple_ELTTEXTE000140"></a>


### Utilisation du membre global pDernier défini dans la classe dessin à partir d'une méthode de la classe dérivée ROND
<a name="utilisation_membre_global_pdernier_defini_dans_classe_dessin_partir_une_methode_classe_derivee_rond_ELTPARAGRAPHE000033"></a>

**Code provoquant l'erreur** 


```wl
:Dessin::pdernier
```




**Correction proposée**

Supprimer les deux points (caractère :) devant le nom de la classe de base.



```wl
Dessin::pdernier
```

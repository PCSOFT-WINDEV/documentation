
## Erreur 110 : La taille de la chaîne ASCIIZ est trop grande
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000088"></a>
Vous utilisez une chaîne ASCIIZ dont la taille dépasse la taille maximale autorisée.

**Rappel** : La taille maximale d'une chaîne ASCIIZ est 64 Ko (65535 octets).



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correction
<a name="correction_ELTTEXTE000112"></a>
Modifiez la taille de votre chaîne ASCIIZ en tenant compte de la valeur maximale autorisée.



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Example
<a name="example_ELTTEXTE000136"></a>


### La taille de la chaîne fixe est trop grande
<a name="taille_chaine_fixe_est_trop_grande_ELTPARAGRAPHE000029"></a>

**Code provoquant l'erreur** 


```wl
B est une chaîne ASCIIZ de 70000
```




**Correction proposée**

Diminuez la taille pour respecter la valeur maximale autorisée.



```wl
B est une chaîne ASCIIZ de 60000
```

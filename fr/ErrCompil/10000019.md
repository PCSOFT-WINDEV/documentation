
## Erreur 19 : Le pas de cette boucle POUR doit être constant
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000088"></a>
Cette ligne de code utilise une instruction répétitive de type POUR. Le PAS de cette boucle répétitive est susceptible de varier à chaque répétition.

**Remarque** : Seules les boucles sur une variable locale de type entier ou réel acceptent un pas variable.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correction
<a name="correction_ELTTEXTE000112"></a>
Modifiez le PAS de votre instruction répétitive.



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Exemple
<a name="exemple_ELTTEXTE000136"></a>


### Utilisation d'une instruction répétitive avec un pas variable pour initialiser un tableau
<a name="utilisation_une_instruction_repetitive_avec_pas_variable_pour_initialiser_tableau_ELTPARAGRAPHE000029"></a>

**Code provoquant l'erreur** 


```wl
J est un entier = 0
I est un entier = 0
Montableau est un tableau de 5 entiers
POUR I = 1 A 5 PAS J+1
	Montableau[I] = I
FIN
```




**Correction proposée**

Utiliser un pas non variable.



```wl
J est un entier = 0
I est un entier = 0
Montableau est un tableau de 5 entiers
POUR I = 1 A 5 PAS J
	Montableau[I] = I
FIN
```

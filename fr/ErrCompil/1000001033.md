
## Erreur 1033 : Les directives d'accès sont interdites dans une structure
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000086"></a>
Vous essayez d'utiliser des directives d'accès (mots-clés : PUBLIC, PRIVÉ, PROTÉGÉ) lors de la définition d'une structure. Il n'est pas possible de restreindre l'accès à un membre d'une structure : tous les membres d'une structure sont publics.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correction
<a name="correction_ELTTEXTE000110"></a>
Supprimez le mot-clé correspondant à une directive d'accès : PUBLIC, PRIVÉ, PROTÉGÉ.



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Exemple
<a name="exemple_ELTTEXTE000134"></a>


### Définition d'une structure
<a name="definition_une_structure_ELTPARAGRAPHE000027"></a>

**Code provoquant l'erreur** 


```wl
Coord est une structure 
	PRIVÉ
	 PosX est un entier
	 PosY est un entier
FIN
```




**Correction proposée**

Supprimer le mot PRIVÉ.



```wl
Coord est une structure 
	PosX est un entier
	PosY est un entier
FIN
```


## Erreur 403: La rubrique n'appartient pas au fichier
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000090"></a>
Vous manipulez une rubrique d'un fichier de données en utilisant une syntaxe de type : 

&lt;NomDuFichier&gt;.&lt;NomDeLaRubrique&gt;.

La rubrique ainsi désignée n'existe pas dans le fichier de données spécifié.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correction
<a name="correction_ELTTEXTE000114"></a>
Vérifiez l'orthographe de la rubrique.

Vérifiez si la rubrique existe. Créez la rubrique si nécessaire.



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Exemple
<a name="exemple_ELTTEXTE000138"></a>


### La rubrique PrixTTC n'existe pas dans le fichier Commande
<a name="rubrique_prixttc_nexiste_pas_dans_fichier_commande_ELTPARAGRAPHE000033"></a>

**Code provoquant l'erreur**


```wl
Commande.PrixTTC = PrixHT*TVA
```




**Correction proposée**

Créez la rubrique dans le fichier de données ou utilisez une autre rubrique du fichier de données.

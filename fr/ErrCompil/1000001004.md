
## Erreur 1004 : Pour accéder explicitement à un élément de la classe de base '%1', utilisez la syntaxe
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000091"></a>
Vous tentez d'utiliser un élément (membre, méthode ou constante) de la classe de base depuis la classe dérivée.

La syntaxe utilisée n'est pas correcte. Vous utilisez la syntaxe suivante :

```txt
:NomClasseDeBase:NomElément
```




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correction
<a name="correction_ELTTEXTE000115"></a>
Pour utiliser un élément (membre, méthode ou constante) de la classe de base depuis une classe dérivée, il est nécessaire de préfixer l'élément par le nom de la classe de base.

```txt
NomClasseDeBase:NomElément
```




<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Exemple
<a name="exemple_ELTTEXTE000139"></a>


### Utilisation du membre CouleurBase de la classe Dessin dans la classe dérivée Rond
<a name="utilisation_membre_couleurbase_classe_dessin_dans_classe_derivee_rond_ELTPARAGRAPHE000033"></a>

**Code provoquant l'erreur** 


```wl
:Dessin:CouleuBase = RougeClair
```


**Correction proposée**

Enlever les deux points (caractère :) devant le nom de la classe de base.



```wl
Dessin:CouleurBase = RougeClair
```

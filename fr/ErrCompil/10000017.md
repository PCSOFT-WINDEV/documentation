
## Erreur 17 : AUTRES CAS doit être en dernier dans le SELON
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000087"></a>
Vous utilisez une instruction conditionnelle de type SELON ... CAS. L'instruction "AUTRES CAS" doit être le dernier CAS utilisé.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correction
<a name="correction_ELTTEXTE000111"></a>
Déplacez le bloc d'instructions commençant par "AUTRES CAS" à la fin de l'instruction conditionnelle SELON.



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Exemple
<a name="exemple_ELTTEXTE000135"></a>


### L'instruction AUTRES CAS n'est pas la dernière instruction
<a name="instruction_autres_cas_nest_pas_derniere_instruction_ELTPARAGRAPHE000027"></a>**Code provoquant l'erreur** 


```wl
SELON TypeProduit
	CAS "1"
	    Message("Produit alimentaire")
	AUTRES CAS
	    Message("Promotion exceptionnelle")
	CAS "2"
	    Message("Produit de nettoyage")
FIN
```


 
**Correction proposée**
Déplacez le bloc d'instructions commençant par "AUTRES CAS" à la fin de l'instruction conditionnelle SELON.



```wl
SELON TypeProduit
	CAS "1"
	    Message("Produit alimentaire")
	CAS "2"
	    Message("Produit de nettoyage")
	AUTRES CAS
	    Message("Promotion exceptionnelle")
FIN
```

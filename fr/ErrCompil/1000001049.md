
## Erreur 1049 : Les directives d'accès sont interdites dans une variable composée
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000088"></a>
Vous essayez d'utiliser des directives d'accès (mots-clés : PUBLIC, PRIVÉ, PROTÉGÉ) lors de la définition d'une variable composée. 

Il n'est pas possible de restreindre l'accès à un membre d'une variable composée : tous les membres d'une variable composée sont publics.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correction
<a name="correction_ELTTEXTE000112"></a>
Supprimez le mot-clé correspondant à une directive d'accès : PUBLIC, PRIVÉ, PROTÉGÉ.



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Exemple
<a name="exemple_ELTTEXTE000136"></a>


### Définition d'une structure
<a name="definition_une_structure_ELTPARAGRAPHE000029"></a>

**Code provoquant l'erreur** 


```wl
Coord est composée de
	PRIVÉ
	 PosX est un entier
	 PosY est un entier
FIN
```




**Correction proposée**

Supprimer le mot PRIVÉ.



```wl
Coord est composée de
	PosX est un entier
	PosY est un entier
FIN
```


## Erreur 1045 : Une classe ne peut pas être déclarée en dehors d'un code de déclaration de classe
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000086"></a>
Le code en cours correspond à une déclaration de classe. Or, ce code est présent dans un événement différent de l'événement "Déclaration de la classe".



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correction
<a name="correction_ELTTEXTE000110"></a>
Vérifiez que la classe est créée dans votre projet et copiez votre code de déclaration de classe dans l'événement "Déclaration de la classe".



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Exemples
<a name="exemples_ELTTEXTE000134"></a>


### Code de déclaration de classe utilisé dans le code d'un bouton
<a name="code_declaration_classe_utilise_dans_code_bouton_ELTPARAGRAPHE000027"></a>

**Code provoquant l'erreur** 


```wl
// -- Clic sur BTN_Bouton1
UnFichier est une classe 
	Nom est une chaîne
	Extension est une chaîne
	Répertoire est une chaîne
FIN
```




**Correction proposée**

Création de la classe UnFichier et copie du code de déclaration de la classe UnFichier dans l'événement "Déclaration de la classe".



```wl
// -- Déclaration de la classe UnFichier
UnFichier est une classe 
	Nom est une chaîne 
	Extension est une chaîne
	Répertoire est une chaîne
FIN
```

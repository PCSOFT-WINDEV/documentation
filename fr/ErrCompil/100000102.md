
## Erreur 102 : Les constantes peuvent être définies dans le code d'initialisation du projet, ...
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000095"></a>
Vous tentez de déclarer une constante dans un traitement où ce type de déclaration est interdit.

**Rappel** : Les constantes doivent être déclarées dans :

- Le code d'initialisation du projet pour être utilisable dans tout les traitements du projet (codes du projet, codes des fenêtres et codes des champs).

- Le code d'ouverture d'une fenêtre ou d'une page pour être utilisable dans tous les traitements de la fenêtre ou de la page (codes de la fenêtre/page et codes des champs présents dans la fenêtre/page).






<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correction
<a name="correction_ELTTEXTE000119"></a>
Déclarez votre constante dans le code approprié.



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Exemple
<a name="exemple_ELTTEXTE000143"></a>


### Une constante est déclarée dans le code de clic d'un bouton de la fenêtre "FEN_Menu"
<a name="une_constante_est_declaree_dans_code_clic_bouton_fenetre_fenmenu_ELTPARAGRAPHE000032"></a>

**Code provoquant l'erreur** 


```wl
// -- Code de clic du bouton BTN_AFFICHE
Constante 
	TauxTVA = 22.6
FIN
```


**Corrections proposées**
Déclarez la constante dans le code d'ouverture de la fenêtre.


```wl
// -- Code d'ouverture de la fenêtre FEN_Menu
Constante
	TauxTVA = 22.6
FIN
```


Déclarez la constante dans le code de déclaration des globales de la page.



```wl
// -- Code de déclaration des globales de la page PAGE_Menu
Constante 
	TauxTVA = 22.6
FIN
```


## Erreur 25 : Il manque le ALORS associé à ce SI
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000101"></a>
Vous avez utilisé une instruction conditionnelle de type SI. Cette instruction doit comporter le mot-clé ALORS.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correction
<a name="correction_ELTTEXTE000125"></a>
Vérifiez le code de votre instruction conditionnelle et ajoutez si nécessaire le mot-clé ALORS manquant.



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Exemples
<a name="exemples_ELTTEXTE000149"></a>


### Utilisation d'une instruction SI sur une ligne
<a name="utilisation_une_instruction_sur_une_ligne_ELTPARAGRAPHE000027"></a>**Code provoquant l'erreur**


```wl
SI TABLE_Cours..Vide 
	IMG_Dessiner..Etat = Grisé
```


**Correction proposée**
Ajoutez le mot-clé ALORS entre la condition et l'action à effectuer si la condition est remplie.


```wl
SI TABLE_Cours..Vide ALORS IMG_Dessiner..Etat = Grisé
```



<a name="NOTE3_2"></a>


### Utilisation d'une instruction SI sur plusieurs lignes
<a name="utilisation_une_instruction_sur_plusieurs_lignes_ELTPARAGRAPHE000043"></a>

**Code provoquant l'erreur**


```wl
SI TABLE_Cours..Vide 
	IMG_Dessiner..Etat = Grisé
FIN
```


**Correction proposée**
Ajoutez le mot-clé ALORS entre la condition et l'action à effectuer si la condition est remplie.


```wl
SI TABLE_Cours..Vide ALORS
	IMG_Dessiner..Etat = Grisé
FIN
```

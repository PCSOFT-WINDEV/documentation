
## Erreur 705 : Propriété inexistante sur ce type
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000086"></a>
Vous tentez d'exécuter une propriété. Cette propriété ne peut pas être utilisée sur le type spécifié.

**Remarque** : La complétion automatique propose automatiquement les propriétés correspondant au type d'objet manipulé. Si la propriété n'est pas présente dans cette liste, elle ne pourra pas être utilisée sur ce type d'objet.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correction
<a name="correction_ELTTEXTE000110"></a>
Consultez l'aide en ligne pour rechercher la propriété voulue et le type sur lequel cette propriété peut être appliquée.



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Exemple
<a name="exemple_ELTTEXTE000134"></a>


### Utilisation de la propriété Nom sur une variable de type Booléen
<a name="utilisation_propriete_nom_sur_une_variable_type_booleen_ELTPARAGRAPHE000029"></a>

**Code provoquant l'erreur** 


```wl
b est un booléen
b..Nom = "C"
```




**Correction proposée**

Utiliser une autre propriété, une autre fonction, etc.

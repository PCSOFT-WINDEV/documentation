
## Erreur 1046 : Mauvais paramètre pour le constructeur de la classe
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000087"></a>
Vous tentez de définir un objet d'une classe. Le nombre de paramètres spécifiés ne correspond pas au nombre de paramètres attendus par la classe.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correction
<a name="correction_ELTTEXTE000111"></a>
Vérifiez les paramètres attendus par le constructeur de la classe et modifiez votre code.



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Exemple
<a name="exemple_ELTTEXTE000135"></a>


### Passage de paramètres à un constructeur
<a name="passage_parametres_constructeur_ELTPARAGRAPHE000027"></a>

**Code provoquant l'erreur** 


```wl
// -- Code de clic sur le bouton BTN_CréerRond
pDessin est un Rond dynamique = allouer un Rond(gpDessin, x, y, larg)
```

```wl
// -- Constructeur de la classe "Rond"
PROCEDURE Rond::Constructeur(pt, aa, bb, cc, dd)
```




**Correction proposée**

Passer le nombre de paramètres attendu.



```wl
// -- Code de clic sur le bouton BTN_CréerRond
pDessin est un Rond dynamique = allouer un Rond(gpDessin, x, y, larg, haut)
 
// -- Constructeur de la classe "Rond"
PROCEDURE Rond::Constructeur(pt, aa, bb, cc, dd)
```

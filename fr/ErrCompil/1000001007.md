
## Erreur 1007 : Le membre est global : utilisez l'opérateur :: pour y accéder
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000094"></a>
Vous tentez d'accéder à un membre global de la classe en utilisant la syntaxe permettant d'accéder à un membre non global.

Vous utilisez la syntaxe suivante :

```txt
NomClasse:NomMembre
```


Cette syntaxe ne permet pas d'accéder à un membre global.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correction
<a name="correction_ELTTEXTE000118"></a>
Utilisez la syntaxe permettant d'accéder à un membre global de la classe : 

```txt
NomClasse::NomMembre
```




<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Exemple
<a name="exemple_ELTTEXTE000142"></a>


### Utilisation du membre global pDernier de la classe Dessin
<a name="utilisation_membre_global_pdernier_classe_dessin_ELTPARAGRAPHE000035"></a>

**Code provoquant l'erreur** 


```wl
Dessin:pdernier
```




**Correction proposée**

Corriger la syntaxe utilisée.


```wl
Dessin::pdernier
```


## Erreur 305 : L'élément n'existe pas dans la fenêtre
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000088"></a>
Vous manipulez un élément de votre fenêtre en utilisant une syntaxe du type :

&lt;NomDeLaFenêtre&gt;.&lt;NomElément&gt;

L'élément ainsi désigné n'existe pas dans la fenêtre spécifiée : ce n'est ni un champ, ni une variable globale, ni une procédure définie dans la fenêtre.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correction
<a name="correction_ELTTEXTE000112"></a>
Vérifiez l'orthographe de l'élément utilisé. Créez si nécessaire le champ, la variable ou la procédure.



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Exemple
<a name="exemple_ELTTEXTE000136"></a>


### Le champ SAI_SaisiePrix n'existe pas dans la fenêtre "FEN_Commande"
<a name="champ_saisaisieprix_nexiste_pas_dans_fenetre_fencommande_ELTPARAGRAPHE000032"></a>

**Code provoquant l'erreur** 


```wl
FEN_Commande.SAI_SaisiePrix = SAI_PrixHT*SAI_TVA
```


**Correction proposée**

Vérifiez l'orthographe du champ ou créez le champ dans la fenêtre.

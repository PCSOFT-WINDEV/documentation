
## Erreur 530 : Opérateur -- interdit
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000085"></a>
Vous tentez d'utiliser l'opérateur arithmétique -- sur une variable ne permettant pas l'utilisation de cet opérateur.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correction
<a name="correction_ELTTEXTE000109"></a>
Modifiez votre code et utilisez l'opérateur ou la fonction appropriée.



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Exemple
<a name="exemple_ELTTEXTE000133"></a>


### Utilisation de l'opérateur arithmétique -- sur une chaîne de caractères
<a name="utilisation_operateur_arithmetique_sur_une_chaine_caracteres_ELTPARAGRAPHE000027"></a>

**Code provoquant l'erreur**


```wl
MaChaîne est une chaîne = "ABCD"
MaChaîne--
```


**Correction proposée**

Utiliser la fonction Gauche pour retrancher la lettre D à la variable MaChaîne.



```wl
MaChaîne est une chaîne = "ABCD"
MaChaîne = Gauche(MaChaîne, 3)
```

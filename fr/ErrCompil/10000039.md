
## Erreur 39 : Il manque le FIN qui marque la fin de déclaration de la variable composée
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000086"></a>
Vous déclarez une variable composée. Le mot FIN signalant la fin de la déclaration de la variable composée n'est pas présent. 



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correction
<a name="correction_ELTTEXTE000110"></a>
Précisez la fin de la déclaration de la variable composée grâce au mot-clé FIN. 



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Exemple
<a name="exemple_ELTTEXTE000134"></a>


### Déclaration incomplète d'une variable composée
<a name="declaration_incomplete_une_variable_composee_ELTPARAGRAPHE000027"></a>

**Code provoquant l'erreur** 


```wl
RefProduit est composé de
	CodeFamille est une chaîne fixe sur 8
	CodeProduit est une chaîne fixe sur 10
```




**Correction proposée**

Ajouter le mot-clé FIN pour terminer la déclaration de la variable composée. 



```wl
RefProduit est composé de
	CodeFamille est une chaîne fixe sur 8
	CodeProduit est une chaîne fixe sur 10
FIN
```

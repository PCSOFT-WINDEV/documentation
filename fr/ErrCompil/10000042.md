
## Erreur 42 : Label hors de portée du GOTO
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000087"></a>
Une étiquette de GOTO est utilisée dans une boucle de type POUR. Cependant, le GOTO correspondant ne se trouve pas dans l'instruction POUR.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correction
<a name="correction_ELTTEXTE000111"></a>


- Intégrer l'instruction GOTO et l'étiquette correspondante dans l'instruction POUR. 

- Supprimer l'étiquette de l'instruction GOTO de la boucle POUR.






<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Exemple
<a name="exemple_ELTTEXTE000135"></a>


### Accès à une étiquette déclarée dans un code d'instruction répétitive 
<a name="acces_une_etiquette_declaree_dans_code_instruction_repetitive_ELTPARAGRAPHE000028"></a>

**Code provoquant l'erreur** 


```wl
I est un entier
MonTableau est un tableau de 10 entiers
...
GOTO ECRIT2
 
POUR I = 1 A 10
	MonTableau[I] = 0
 
	ECRIT2 : 
	MonTableau[I-1]=I
 
FIN
```




**Correction proposée**

Supprimer l'étiquette.



```wl
I est un entier
MonTableau est un tableau de 10 entiers
...
 
POUR I = 1 A 10
	MonTableau[I] = 0
 
	MonTableau[I-1]=I
 
FIN
```

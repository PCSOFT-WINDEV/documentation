
## Erreur 802 : Aucune syntaxe de cette fonction ne correspond
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000087"></a>
Vous tentez d'utiliser une fonction du WLangage en utilisant une syntaxe incorrecte.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correction
<a name="correction_ELTTEXTE000111"></a>
Vérifiez les syntaxes possibles pour cette fonction dans l'aide en ligne et corrigez votre ligne de code.



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Exemple
<a name="exemple_ELTTEXTE000135"></a>


### Trop de paramètres sont passés à la fonction Hasard
<a name="trop_parametres_sont_passes_fonction_hasard_ELTPARAGRAPHE000027"></a>**Code provoquant l'erreur** 


```wl
Hasard(5, 7, 8)
```




 
**Correction proposée**
Utilisation de 2 paramètres après vérification de la syntaxe dans l'aide en ligne.



```wl
Hasard(5, 7)
```

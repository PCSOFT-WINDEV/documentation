
## Warning 127 : Vous utilisez une source de données locale au traitement
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000086"></a>
Cette ligne de code déclare une variable locale de type "source de données". Cette variable sera détruite à la fin du traitement en cours.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Conseil
<a name="conseil_ELTTEXTE000110"></a>
Pour utiliser une variable de type "Source de données" dans tous les traitements de votre application, définissez cette variable en tant que variable globale.



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Exemples
<a name="exemples_ELTTEXTE000134"></a>


### Utilisation d'une source de données locale à la fenêtre "FEN_Gestion"
<a name="utilisation_une_source_donnees_locale_fenetre_fengestion_ELTPARAGRAPHE000027"></a>

**Code provoquant le warning**



```wl
MonFichierClient est une Source de Données
```



**Correction proposée**
Définition de la source de données globale à la fenêtre.



```wl
// -- Déclarations globales de la fenêtre FEN_Gestion
GLOBALE
	MonFichierClient est une Source de Données
```

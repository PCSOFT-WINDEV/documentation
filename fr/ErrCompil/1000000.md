
## Erreur 1037 : Ce membre existe déjà dans la structure 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000087"></a>
Vous déclarez une structure utilisant deux membres de même nom. Cette erreur peut apparaître suite à une faute de frappe par exemple dans le nom d'un des membres de la structure.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correction
<a name="correction_ELTTEXTE000111"></a>
Renommez ou supprimez le membre en trop.



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Exemple
<a name="exemple_ELTTEXTE000135"></a>


### Déclaration d'une structure avec deux membres de même nom
<a name="declaration_une_structure_avec_deux_membres_meme_nom_ELTPARAGRAPHE000027"></a>

**Code provoquant l'erreur** 


```wl
Coord est une Structure 
	PosX est un entier
	PosX est un entier
FIN
```


 
**Correction proposée**
Renommer un des membres.



```wl
Coord est une Structure 
	PosX est un entier
	PosY est un entier
FIN
```


## Erreur 116 : Une dimension de tableau doit être une valeur entière
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000107"></a>
Vous utilisez un tableau : les dimensions du tableau sont incorrectes. En effet, pour définir la dimension, vous devez utilisez une valeur entière.

**Rappel** : Lors de la déclaration d'un tableau, il est nécessaire d'utiliser des valeurs de type ENTIER pour définir les dimensions du tableau.



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Correction
<a name="correction_ELTTEXTE000131"></a>
Modifiez le type utilisé pour définir les dimensions du tableau. Utilisez un entier.



<a name="NOTE4"></a>
<a name="NOTE4_1"></a>


## Exemples
<a name="exemples_ELTTEXTE000155"></a>


### Utilisation d'une valeur numérique incorrecte : cette valeur ne correspond pas au type attendu
<a name="utilisation_une_valeur_numerique_incorrecte_cette_valeur_correspond_pas_type_attendu_ELTPARAGRAPHE000029"></a>

**Code provoquant l'erreur** 


```wl
MonTableau est un tableau de 5.5 entiers
```


**Corrections proposées**

Pour utiliser un tableau à deux dimensions, séparez les dimensions du tableau par une virgule : remplacez 5.5 par **5,5**.


```wl
MonTableau est un tableau de 5,5 entiers
```


Pour utiliser un tableau à une seule dimension, modifiez la valeur numérique : remplacez 5.5 par **5**.



```wl
MonTableau est un tableau de 5 entiers
```



<a name="NOTE4_2"></a>


### Utilisation d'une chaîne de caractères pour déclarer la dimension d'un tableau
<a name="utilisation_une_chaine_caracteres_pour_declarer_dimension_tableau_ELTPARAGRAPHE000049"></a>

**Code provoquant l'erreur** 


```wl
NbLignes est une chaîne
MonTableau est un tableau de NbLignes entiers
```


**Correction proposée**

Modifiez le type de la variable NbLignes pour utiliser un entier.


```wl
NbLignes est un entier
MonTableau est un tableau de NbLignes entiers
```


## Erreur 104 : 'Dynamique' ne peut être utilisé que pour les classes et les objets Automation
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000092"></a>
Dans cette ligne de code, vous utilisez le mot-clé **Dynamique** pour définir un objet qui ne peut pas être dynamique.

**Rappel** : Le mot-clé **Dynamique** peut être utilisé uniquement avec les types suivants :

- classe

- tableau

- structure

- objet automation






<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correction
<a name="correction_ELTTEXTE000116"></a>
Supprimez le mot-clé **dynamique**.



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Exemples
<a name="exemples_ELTTEXTE000140"></a>


### Un entier est déclaré en entier dynamique
<a name="entier_est_declare_entier_dynamique_ELTPARAGRAPHE000034"></a>

**Code provoquant l'erreur** 


```wl
NbCommande est un entier dynamique
```


**Correction proposée**

Supprimez le mot **Dynamique**.



```wl
NbCommande est un entier
```

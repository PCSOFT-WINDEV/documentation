
## Erreur 1016 : La méthode de la classe est privée
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000077"></a>
Vous tentez d'accéder à une méthode privée de la classe à partir d'un code différent d'une méthode de cette classe.

**Rappel** : Une méthode privée peut être manipulée uniquement à partir d'une méthode de la classe ou d'une classe dérivée de cette classe.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correction
<a name="correction_ELTTEXTE000101"></a>
Deux possibilités :

- Remplacer la méthode privée par :

	- Une méthode **publique** : l'accès à la méthode ne sera pas restreint.

	- Une méthode **protégée** : l'accès à cette méthode pourra être effectué depuis une classe dérivée.




- Ne pas manipuler cette méthode privée depuis un des codes du projet.


## Erreur 404 : Le fichier du parcours HFSQL ne peut pas être une expression de ce type
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000086"></a>
Ce code utilise l'instruction POUR TOUT. Le nom du fichier de données attendu dans cette instruction ne correspond pas à un nom de fichier de données valide.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correction
<a name="correction_ELTTEXTE000110"></a>
Vérifiez le code de l'instruction POUR TOUT et utilisez un nom de fichier de données HyperFileSQL.



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Exemple
<a name="exemple_ELTTEXTE000134"></a>


### Utilisation d'un entier dans une boucle POUR TOUT
<a name="utilisation_entier_dans_une_boucle_pour_tout_ELTPARAGRAPHE000027"></a>

**Code provoquant l'erreur** 


```wl
B est un entier
POUR TOUT B
	ListeAjoute(LISTE_Liste1, NomClient)
FIN
```




**Correction proposée**

Utiliser un nom de fichier de données et non une variable de type scalaire.



```wl
POUR TOUT CLIENT
	ListeAjoute(LISTE_Liste1, NomClient)
FIN
```

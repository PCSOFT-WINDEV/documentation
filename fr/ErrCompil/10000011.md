
## Erreur 11 : Le label est inconnu
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000087"></a>
Cette ligne de code utilise l'instruction GOTO. L'étiquette utilisée dans l'instruction GOTO n'a pas été définie dans le code en cours.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correction
<a name="correction_ELTTEXTE000111"></a>
Vérifiez que l'étiquette existe et utilisez son nom dans l'instruction GOTO (la coloration syntaxique permet de vérifier facilement si le nom de l'étiquette existe).



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Exemple
<a name="exemple_ELTTEXTE000135"></a>


### Une faute de frappe a été faite dans le nom de l'étiquette
<a name="une_faute_frappe_ete_faite_dans_nom_etiquette_ELTPARAGRAPHE000027"></a>

**Code provoquant l'erreur**


```wl
Res = fOuvre(NomFichier, foEcriture)
SI Res = -1 ALORS GOTO ERROUVRE
Res = fEcrit(Res,"Traitement OK")
SI Res = -1 ALORS GOTO ERRCRIT
...
RETOUR
 
ERROUVRE : 
Info("Fichier " + NomFichier + " ne peut être ouvert. Vérifiez son existence.")
 
ERRECRIT : 
Info("Il est impossible d'écrire dans le fichier " + NomFichier)
```


**Correction proposée**

Vérifiez le nom des étiquettes et corrigez la faute de frappe. Dans cet exemple, ERRECIT est remplacé par ERRECRIT.


```wl
Res = fOuvre(NomFichier, foEcriture)
SI Res = -1 ALORS GOTO ERROUVRE
Res = fEcrit(Res,"Traitement OK")
SI Res = -1 ALORS GOTO ERRECRIT
...
RETOUR
 
ERROUVRE : 
Info("Fichier " + NomFichier + " ne peut être ouvert. Vérifiez son existence.")
 
ERRECRIT : 
Info("Il est impossible d'écrire dans le fichier " + NomFichier)
```

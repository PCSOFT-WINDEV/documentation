
## Erreur 105 : Redéfinition de la variable locale 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000103"></a>
Vous tentez de déclarer une variable locale qui a déjà été déclarée dans le traitement en cours.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correction
<a name="correction_ELTTEXTE000127"></a>
Renommez une des variables ou supprimez la déclaration inutile.



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Exemples
<a name="exemples_ELTTEXTE000151"></a>


### Deux variables locales de même nom mais de type différent sont utilisées
<a name="deux_variables_locales_meme_nom_mais_type_different_sont_utilisees_ELTPARAGRAPHE000027"></a>

**Code provoquant l'erreur** 


```wl
b est un entier
 
b est une chaîne
```




**Correction proposée**

Renommez la seconde variable.



```wl
b est un entier
 
c est une chaîne
```



<a name="NOTE3_2"></a>


### Deux variables locales ont le même nom et le même type
<a name="deux_variables_locales_ont_meme_nom_meme_type_ELTPARAGRAPHE000044"></a>

**Code provoquant l'erreur** 


```wl
I est un entier
POUR I = 1 A 10
...
FIN
...
 
I est un entier
POUR I = 1 A 10
...
FIN
```




**Correction proposée**

Supprimez la seconde déclaration (déclaration inutile).



```wl
I est un entier
POUR I = 1 A 10
...
FIN
...
 
POUR I = 1 A 10
...
FIN
```

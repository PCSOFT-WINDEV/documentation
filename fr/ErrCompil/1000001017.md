
## Erreur 1017 : La méthode n'est pas globale : utilisez l'opérateur : pour y accéder
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000094"></a>
Vous tentez d'accéder à une méthode de la classe en utilisant la syntaxe permettant d'accéder à une méthode globale.

Vous utilisez la syntaxe suivante :

```txt
NomClasse::NomMéthode
```


Cette syntaxe ne permet pas d'accéder à une méthode non globale.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correction
<a name="correction_ELTTEXTE000118"></a>
Utilisez la syntaxe permettant d'accéder à une méthode (non globale) de la classe : 

```txt
NomClasse:NomMéthode
```




<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Exemple
<a name="exemple_ELTTEXTE000142"></a>


### Utilisation de la méthode non globale Afficher de la classe Dessin
<a name="utilisation_methode_non_globale_afficher_classe_dessin_ELTPARAGRAPHE000035"></a>

**Code provoquant l'erreur** 


```wl
Dessin::Afficher()
```




**Correction proposée**

Utiliser la syntaxe correcte.



```wl
Dessin:Afficher()
```

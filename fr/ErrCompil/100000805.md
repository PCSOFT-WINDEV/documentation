
## Erreur 805 : Pas de syntaxe sans paramètres
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000088"></a>
Vous utilisez une fonction du WLangage (ou une fonction ou une procédure) sans indiquer les parenthèses. Or, cette fonction n'accepte pas de syntaxe sans aucun paramètre et donc sans parenthèses.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correction
<a name="correction_ELTTEXTE000112"></a>
Complétez l'appel de la fonction ou de la procédure en précisant les paramètres obligatoires.

**Remarque** : Pour des fonctions du WLangage, consultez l'aide en ligne pour obtenir la syntaxe exacte de la fonction.



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Exemple
<a name="exemple_ELTTEXTE000136"></a>


### Utilisation de la fonction Ln
<a name="utilisation_fonction_ELTPARAGRAPHE000029"></a>

**Code provoquant l'erreur** 


```wl
Res = Ln
```




**Correction proposée**

Compléter la syntaxe.



```wl
Res = Ln(5)
```

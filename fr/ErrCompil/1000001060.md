
## Erreur 1060 : Appel de méthode sans paramètre indisponible : rajoutez ()
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000087"></a>
Vous tentez d'utiliser une méthode sans indiquer les parenthèses.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correction
<a name="correction_ELTTEXTE000111"></a>
Ajoutez les parenthèses et si nécessaire les paramètres attendus par la méthode.



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Exemple
<a name="exemple_ELTTEXTE000135"></a>


### Utilisation de la méthode Détruire de la classe Dessin
<a name="utilisation_methode_detruire_classe_dessin_ELTPARAGRAPHE000027"></a>**Code provoquant l'erreur** 


```wl
Dessin::Detruire
```


 
**Correction proposée**


Ajouter les parenthèses.


```wl
Dessin::Detruire()
```

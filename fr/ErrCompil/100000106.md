
## Erreur 106 : Redéfinition de la variable globale 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000086"></a>
Vous tentez de déclarer une variable globale. Le nom de cette variable a déjà été utilisé pour une autre variable globale du projet ou de la fenêtre en cours.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correction
<a name="correction_ELTTEXTE000110"></a>
Renommez une des variables ou supprimez la déclaration inutile.



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Exemple
<a name="exemple_ELTTEXTE000134"></a>


### Deux variables globales de même nom mais de type différent sont utilisées
<a name="deux_variables_globales_meme_nom_mais_type_different_sont_utilisees_ELTPARAGRAPHE000027"></a>

**Code provoquant l'erreur** 


```wl
// -- Code d'initialisation du projet
Globale
	b est un entier
 
	b est une chaîne
```




**Correction proposée**

Renommez la seconde variable.



```wl
// -- Code d'initialisation du projet
Globale
	b est un entier
 
	c est une chaîne
```

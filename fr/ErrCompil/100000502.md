
## Erreur 502 : Type Incompatible
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000084"></a>
Vous tentez de réaliser une opération arithmétique ou logique entre deux expressions de type incompatible.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correction
<a name="correction_ELTTEXTE000108"></a>
Ce type d'opération est impossible. Recherchez les fonctions du WLangage vous permettant de réaliser cette opérations.



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Exemples
<a name="exemples_ELTTEXTE000132"></a>


### Le code suivant tente de faire une soustraction entre deux chaînes de caractères
<a name="code_suivant_tente_faire_une_soustraction_entre_deux_chaines_caracteres_ELTPARAGRAPHE000027"></a>

**Code provoquant l'erreur** 


```wl
chaîne1 est une chaîne
chaîne2 est une chaîne
Trace(chaîne1-chaîne2)
```




**Correction proposée**

La soustraction de chaînes de caractères est impossible en WLangage.


## Erreur 701 : La propriété est en lecture seule
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000084"></a>
Cette ligne de code tente d'affecter une valeur à une propriété qui est utilisable uniquement en lecture.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correction
<a name="correction_ELTTEXTE000108"></a>
Modifiez votre code pour utiliser cette propriété en lecture et non en affectation.



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Exemple
<a name="exemple_ELTTEXTE000132"></a>


### Utilisation de la propriété Abréviation 
<a name="utilisation_propriete_abreviation_ELTPARAGRAPHE000027"></a>

**Code provoquant l'erreur** 


```wl
CLIENT..Abréviation = "AB"
```




**Correction proposée**

Supprimez la ligne de code.


## Erreur 3 : La chaîne n'a pas de " final
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000120"></a>
Cette ligne de code contient des guillemets pour délimiter une chaîne de caractères. Ces guillemets ont été "ouverts" et n'ont pas été refermés avant la fin de la ligne de code.

**Rappel** : Dans l'éditeur de code, le passage à la ligne suivante (utilisation de la touche ENTREE) est considérée comme un séparateur d'instructions.

Pour indiquer que la ligne de code en cours continue sur la ligne suivante, il est nécessaire d'utiliser la ponctuation "..." en fin de ligne.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correction
<a name="correction_ELTTEXTE000144"></a>
Vérifiez votre ligne de code et terminez la chaîne de caractères par des guillemets.

**Remarque** : La coloration syntaxique permet de visualiser la fin de la chaîne de caractères.



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Exemples
<a name="exemples_ELTTEXTE000168"></a>


### Affichage d'un message
<a name="affichage_message_ELTPARAGRAPHE000033"></a>Code provoquant l'erreur :

```txt
Info("Vous ne pouvez pas réaliser cette opération)
```


**Correction proposée**

Ajoutez le guillemet final à la fin de la chaîne de caractères.


```wl
Info("Vous ne pouvez pas réaliser cette opération")
```

<a name="NOTE3_2"></a>


### Affichage d'un message contenant une information stockée dans une variable
<a name="affichage_message_contenant_une_information_stockee_dans_une_variable_ELTPARAGRAPHE000047"></a>Code provoquant l'erreur :

```txt
Info("Vous ne pouvez pas enregistrer le fichier + NomFichier)
```


**Correction proposée**

Ajoutez un guillemet pour séparer la fin de la chaîne de caractères et la valeur de la variable.


```wl
Info("Vous ne pouvez pas enregistrer le fichier" + NomFichier)
```



<a name="NOTE3_3"></a>


### Affichage d'un message écrit sur plusieurs lignes de code
<a name="affichage_message_ecrit_sur_plusieurs_lignes_code_ELTPARAGRAPHE000062"></a>

Code provoquant l'erreur :


```txt
Info("Vous ne pouvez pas réaliser cette opération
	vous n'avez pas les droits nécessaires")
```


**Correction proposée**

Ajoutez le guillemet final en fin de ligne ainsi que les caractères "+ ..." pour indiquer que la chaîne de caractères se poursuit sur la ligne suivante.


```wl
Info("Vous ne pouvez pas réaliser cette opération"+...
	"vous n'avez pas les droits nécessaires")
```

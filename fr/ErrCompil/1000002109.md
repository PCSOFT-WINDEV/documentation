
## Erreur 2109 : Vous utilisez la fonction InfoObjet
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000164"></a>
Vous tentez d'utiliser la fonction ***InfoObjet***. Cette fonction n'existe plus dans WINDEV. Utilisez une des fonctions de remplacement.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correction
<a name="correction_ELTTEXTE000188"></a>
Remplacez la fonction ***InfoObjet*** par une des nouvelles fonctions du WLangage :


| <br><br> | Description | Fonction WLangage correspondante |
| --- | --- | --- |
| **InfoObjet(xx, 0)** | Liste des éléments d'une fenêtre, d'un groupe de champs, d'un onglet, etc. | [EnumèreChamp](../WDLang1/3025005.md) |
| **InfoObjet(xx,1)** |
| **InfoObjet(2)** | Nom de l'élément contenant le champ (nom de la fenêtre, nom du champ Table, nom du superchamp, etc.) | [ParentObjet](../WDLang1/3025031.md) |
| **InfoObjet(4)** | Nom de l'onglet contenant le champ | [OngletDuChamp](../WDLang1/3025028.md) |
| **InfoObjet(11)** | Nom du champ survolé par la souris | [ChampSurvol](../WDLang1/3025021.md) |
| **InfoObjet(15,4)** | Handle des éléments d'un champ Combo (zone de saisie du champ Combo ou liste du champ Combo) | [Handle](../WDLang1/3015005.md) |
| **InfoObjet(22)** | Paramètre le délai d'ouverture d'une fenêtre système (ouverte par les fonctions Erreur, Info, etc.) | [DélaiAvantFermeture](../WDLang1/3038040.md) |
| **InfoObjet(26)** | Nom d'une colonne à partir de son indice | [TableEnumèreColonne](../WDLang1/3074012.md) |
| **InfoObjet(34)** | Modification des couleurs dans une bulle d'aide | [BulleCouleur](../WDLang1/3025015.md) |
| **InfoObjet(35)** | Fixe une colonne d'un champ Table | [TableFixeColonne](../WDLang1/3074014.md) |
| **InfoObjet(36)** | Déplace une colonne dans un champ Table | [TableDéplaceColonne](../WDLang1/3074011.md) |
| **InfoObjet(39)** | Liste des colonnes triées | [TableColonnesTriées](../WDLang1/3074010.md) |
| **InfoObjet(43)** | Combo auto-alimentée : autorise ou non la récupération de la valeur définie par la fonction gLien dans un champ Liste ou dans un champ Combo | [gLienActive](../WDLang1/3029036.md), [gValeurMémoriséeActive](../WDLang1/1410088364.md) |

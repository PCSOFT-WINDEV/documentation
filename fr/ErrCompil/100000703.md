
## Erreur 703 : Propriété Inexistante
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000086"></a>
La propriété utilisée n'existe pas dans le WLangage. Cette erreur est sans doute due à une faute de frappe.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correction
<a name="correction_ELTTEXTE000110"></a>
Vérifiez le nom de la propriété dans l'aide en ligne et modifiez votre code en conséquence.



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Exemple
<a name="exemple_ELTTEXTE000134"></a>


### Utilisation de la propriété VALEUR
<a name="utilisation_propriete_valeur_ELTPARAGRAPHE000027"></a>

**Code provoquant l'erreur** 


```wl
BTN_Bouton1..Valeu
```




**Correction proposée**

Le nom de la propriété est Valeur.



```wl
BTN_Bouton1..Valeur
```

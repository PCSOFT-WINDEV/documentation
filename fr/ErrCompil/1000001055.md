
## Erreur 1055 : 'Ancêtre' ne peut être utilisé que dans le code d'une classe dérivée d'une  classe
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000072"></a>
Vous tentez d'utiliser le mot-clé 'Ancêtre' dans le code d'une classe non dérivée.


 
**Rappel** : Le mot-clé 'Ancêtre' permet de spécifier, dans le code d'une classe dérivée, la classe ancêtre associée à la méthode ou au membre à manipuler.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correction
<a name="correction_ELTTEXTE000096"></a>
N'utilisez pas le mot-clé 'Ancêtre'.

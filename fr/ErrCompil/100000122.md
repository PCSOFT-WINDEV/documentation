
## Erreur 122 : Le nom d'un objet Automation doit être une chaîne
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000087"></a>
Vous déclarez un objet automation sans utiliser une chaîne (ou une variable de type chaîne) pour définir le nom de l'objet automation.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correction
<a name="correction_ELTTEXTE000111"></a>
Utilisez une chaîne de caractères (ou une variable de type chaîne de caractères) pour définir le nom de l'objet automation.



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Exemple
<a name="exemple_ELTTEXTE000135"></a>


### Déclaration d'un objet OLE Automation
<a name="declaration_objet_ole_automation_ELTPARAGRAPHE000027"></a>**Code provoquant l'erreur** 


```wl
MonDocument est un objet automation 5
```


 
**Correction proposée**
Donnez un nom à l'objet Automation. 



```wl
MonDocument est un objet automation "Word.Application"
```


## Erreur 38 : Il manque le FIN qui marque la fin de déclaration de la structure
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000086"></a>
Vous déclarez une structure. Le mot FIN signalant la fin de la déclaration de la structure n'est pas présent. 



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correction
<a name="correction_ELTTEXTE000110"></a>
Précisez la fin de la déclaration de la structure grâce au mot-clé FIN.



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Exemple
<a name="exemple_ELTTEXTE000134"></a>


### Déclaration d'une structure incomplète
<a name="declaration_une_structure_incomplete_ELTPARAGRAPHE000027"></a>

**Code provoquant l'erreur** 


```wl
MaClasse est une structure
	Élève est une chaîne
	Niveau est un entier
	Année est une Date
```




**Correction proposée**

Précisez la fin de la structure à l'aide du mot-clé FIN.



```wl
MaClasse est une structure
	Élève est une chaîne
	Niveau est un entier
	Année est une Date
FIN
```


## Erreur 16 : CAS non compatible
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000059"></a>
Vous utilisez une instruction conditionnelle de type SELON ... CAS. Les éléments de comparaison indiqués dans le "CAS" sont incompatibles avec l'expression utilisée dans le "SELON".

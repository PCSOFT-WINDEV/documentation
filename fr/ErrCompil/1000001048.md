
## Erreur 1048 : L'héritage est interdit dans une variable composée
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000086"></a>
Vous tentez de définir un héritage dans la définition d'une variable composée. L'héritage est interdit dans une variable composée.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correction
<a name="correction_ELTTEXTE000110"></a>
Supprimez la déclaration de l'héritage. 



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Exemple
<a name="exemple_ELTTEXTE000134"></a>


### Héritage dans une variable composée
<a name="heritage_dans_une_variable_composee_ELTPARAGRAPHE000027"></a>

**Code provoquant l'erreur** 


```wl
Coordonnée est composé de
	x est un réel
	y est un réel
FIN
Coordonnée1 est composé de
	un objet Coordonnée
	x1 est un réel
	y1 est un réel
FIN
```




**Correction proposée**

Supprimer l'héritage. 



```wl
Coordonnée est composé de
	x est un réel
	y est un réel
FIN
Coordonnée1 est composé de
	x1 est un réel
	y1 est un réel
FIN
```

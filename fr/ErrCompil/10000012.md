
## Warning 12 : Le label est inutilisé
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000088"></a>
Cette ligne de code utilise l'instruction GOTO avec des étiquettes de branchement. Dans le traitement en cours, cette étiquette est définie mais elle n'est utilisée par aucune instruction GOTO.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Conseil
<a name="conseil_ELTTEXTE000112"></a>
Vérifiez que cette étiquette est vraiment inutile :

- Si l'étiquette est inutile, supprimez-la.

- Si l'étiquette est utile, corrigez l'instruction GOTO afin d'utiliser la bonne étiquette.






<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Exemple
<a name="exemple_ELTTEXTE000136"></a>


### Suite à une erreur de frappe, une étiquette n'est utilisée par aucun GOTO
<a name="suite_une_erreur_frappe_une_etiquette_nest_utilisee_par_aucun_goto_ELTPARAGRAPHE000030"></a>

**Code provoquant l'erreur**


```wl
Res = fOuvre(NomFichier, foEcriture)
SI Res = -1 ALORS GOTO ERROUVRE
Res = fEcrit(Res, "Traitement OK")
SI Res = -1 ALORS GOTO ERROUVRE
...
RETOUR
 
ERROUVRE : 
Info("Fichier " + NomFichier + " ne peut être ouvert. Vérifiez son existence.")
 
ERRECRIT : 
Info("Il est impossible d'écrire dans le fichier " + NomFichier)
```


**Corrections proposées**

Modifiez le code de l'instruction GOTO et remplacez l'étiquette en double par l'étiquette correspondante.
 

```wl
Res = fOuvre(NomFichier, foEcriture)
SI Res = -1 ALORS GOTO ERROUVRE
Res = fEcrit(Res,"Traitement OK")
SI Res = -1 ALORS GOTO ERRECRIT
...
RETOUR
 
ERROUVRE : 
Info("Fichier " + NomFichier + " ne peut être ouvert. Vérifiez son existence.")
 
ERRECRIT : 
Info("Il est impossible d'écrire dans le fichier " + NomFichier)
```

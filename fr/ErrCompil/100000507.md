
## Erreur 507 : Opérateur '++' interdit
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000086"></a>
Vous tentez d'utiliser l'opérateur arithmétique ++ sur une variable ne permettant pas l'utilisation de cet opérateur.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correction
<a name="correction_ELTTEXTE000110"></a>
Modifiez votre code et utilisez l'opérateur ou la fonction appropriée.



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Exemple
<a name="exemple_ELTTEXTE000134"></a>


### Utilisation de l'opérateur arithmétique ++ sur une chaîne de caractères
<a name="utilisation_operateur_arithmetique_sur_une_chaine_caracteres_ELTPARAGRAPHE000027"></a>

**Code provoquant l'erreur** 


```wl
MaChaîne est une chaîne = "A"
MaChaîne++
```




**Correction proposée**

Utiliser l'opérateur += pour ajouter une valeur à la variable MaChaîne.



```wl
MaChaîne est une chaîne = "A"
MaChaîne+=MaChaîne
```

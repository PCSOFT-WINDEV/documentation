
## Erreur 1054 : 'Ancêtre' ne peut pas être utilisé en dehors du code d'une classe
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000086"></a>
Vous tentez d'accéder à un membre ou une méthode de la classe de base d'une classe dérivée en utilisant le mot-clé 'Ancêtre'. Ce mot-clé ne peut être utilisé que dans un code d'une classe dérivée.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correction
<a name="correction_ELTTEXTE000110"></a>
Pour utiliser explicitement une méthode ou un membre de la classe de base à partir d'un code extérieur à la classe, spécifiez le nom de la classe ancêtre à utiliser.



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Exemple
<a name="exemple_ELTTEXTE000134"></a>


### Code provoquant l'erreur
<a name="code_provoquant_erreur_ELTPARAGRAPHE000027"></a>


```wl
// -- Code de clic du bouton 
Ancetre::pDernier
```




 
**Correction proposée**
Préciser le nom de la classe. 


```wl
// -- Code de clic du bouton 
Dessin::pDernier
```

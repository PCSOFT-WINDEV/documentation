
## Erreur 1022 : L'accès à la méthode de la classe est restreint par un héritage protégé
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000070"></a>
Vous tentez d'accéder à une méthode d'une classe elle-même dérivée par un héritage protégé. Il n'est pas possible d'accéder à cette méthode depuis un code extérieur à cette classe.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correction
<a name="correction_ELTTEXTE000094"></a>
Deux possibilités : 

- Accédez à cette méthode par une méthode de la classe dérivée ou d'une de ses propres classes dérivées. 

- Transformez l'héritage protégé en héritage public : la méthode sera alors accessible aussi bien depuis le code des classes dérivées que depuis un code extérieur à la classe.

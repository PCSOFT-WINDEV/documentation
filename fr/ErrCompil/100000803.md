
## Erreur 803 : Utilisation des mots clés Renvoyer et Retour dans le même traitement
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000090"></a>
Le traitement en cours utilise les mots-clés RENVOYER et RETOUR. Ces deux mots-clés ne peuvent pas être utilisés dans le même traitement.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correction
<a name="correction_ELTTEXTE000114"></a>
Harmonisez les traitements effectués dans votre boucle ou dans votre procédure.

- Soit la procédure / la boucle renvoie un résultat, et utilise toujours le mot-clé RENVOYER.

- Soit la procédure / la boucle ne renvoie pas de résultat et utilise toujours le mot-clé RETOUR.






<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Exemple
<a name="exemple_ELTTEXTE000138"></a>


### Procédure utilisant les mots-clés RETOUR et RENVOYER
<a name="procedure_utilisant_les_motscles_retour_renvoyer_ELTPARAGRAPHE000030"></a>

**Code provoquant l'erreur** 


```wl
FONCTION Trouve(Fichier, Cle, Valeur)
HLitRecherche(Fichier, Cle, Valeur)
SI HTrouve()=Vrai ALORS
	RENVOYER Vrai
SINON
	RENVOYER Faux
FIN
RETOUR
```



<a name="NOTE3_2"></a>


### Correction proposée
<a name="correction_proposee_ELTPARAGRAPHE000040"></a>Dans cet exemple, le mot-clé RETOUR est inutile et doit être supprimé.


## Erreur 109 : La taille de la chaîne fixe est trop grande
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000088"></a>
Vous utilisez une chaîne fixe dont la taille dépasse la taille maximale autorisée.

Rappel : La taille maximale d'une chaîne fixe est 32 Ko (32767 octets).



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correction
<a name="correction_ELTTEXTE000112"></a>
Modifiez la taille de votre chaîne fixe en tenant compte de la valeur maximale autorisée.



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Exemple
<a name="exemple_ELTTEXTE000136"></a>


### La taille de la chaîne fixe est trop grande
<a name="taille_chaine_fixe_est_trop_grande_ELTPARAGRAPHE000029"></a>

**Code provoquant l'erreur** 


```wl
B est une chaîne fixe de 40000
```




**Correction proposée**

Diminuez la taille pour respecter la valeur maximale autorisée.



```wl
B est une chaîne fixe de 30000
```


## Erreur 1044 : La classe dérive déjà de cette classe
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000086"></a>
Vous tentez de rédéfinir un héritage. Le même héritage est défini deux fois dans le même code.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correction
<a name="correction_ELTTEXTE000110"></a>
Supprimez la syntaxe supplémentaire. 



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Exemple
<a name="exemple_ELTTEXTE000134"></a>


### Déclaration d'une classe dérivée
<a name="declaration_une_classe_derivee_ELTPARAGRAPHE000027"></a>

**Code provoquant l'erreur** 


```wl
Rond est une classe
	un objet Dessin
	un objet Dessin
FIN
```




**Correction proposée**

Supprimer la déclaration supplémentaire. 



```wl
Rond est une classe
	un objet Dessin
FIN
```

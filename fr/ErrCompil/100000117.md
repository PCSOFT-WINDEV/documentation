
## Erreur 117 : Le tableau possède X dimensions et vous y accédez avec Y dimensions
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000092"></a>
Vous manipulez un tableau en prenant en compte plus de dimensions qu'il n'en a été défini.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correction
<a name="correction_ELTTEXTE000116"></a>
Modifiez la déclaration de votre tableau ou la syntaxe permettant de manipuler un élément du tableau.



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Exemple
<a name="exemple_ELTTEXTE000140"></a>


### Manipulation d'un tableau déclaré avec une seule dimension
<a name="manipulation_tableau_declare_avec_une_seule_dimension_ELTPARAGRAPHE000027"></a>

**Code provoquant l'erreur** 


```wl
// Déclaration du tableau
VenteMin est un tableau de 12 entiers
i est un entier = 1
// Affectation d'un élément de VenteMin
VenteMin[15,2]=15
```




**Correction proposée**

Modification de la déclaration du tableau.



```wl
// Déclaration du tableau
VenteMin est un tableau de 12 par 15 entiers
i est un entier = 1
// Affectation d'un élément de VenteMin
VenteMin[15,2]=15
```


**Correction proposée**

Modification de l'affectation.



```wl
// Déclaration du tableau
VenteMin est un tableau de 12 entiers
i est un entier = 1
// Affectation d'un élément de VenteMin
VenteMin[2]=15
```

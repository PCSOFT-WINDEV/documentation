
## Erreur 40 : Il manque le FIN qui marque la fin de déclaration de la classe
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000086"></a>
Vous déclarez une classe dans le code de déclaration de votre classe. La déclaration de la classe ne se termine pas par le mot-clé FIN. 



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correction
<a name="correction_ELTTEXTE000110"></a>
Utilisez le mot-clé FIN pour terminer la déclaration de votre classe. 



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Exemple
<a name="exemple_ELTTEXTE000134"></a>


### Déclaration d'une classe sans le mot-clé FIN
<a name="declaration_une_classe_sans_motcle_fin_ELTPARAGRAPHE000027"></a>

**Code provoquant l'erreur** 


```wl
// -- Déclaration de la classe "Fichier"
Fichier est une classe 
	Nom est une chaîne 
	Extension est une chaîne
	Répertoire est une chaîne
```




**Correction proposée**

Utiliser le mot-clé FIN.



```wl
// -- Déclaration de la classe "Fichier"
Fichier est une classe 
	Nom est une chaîne 
	Extension est une chaîne
	Répertoire est une chaîne
FIN
```

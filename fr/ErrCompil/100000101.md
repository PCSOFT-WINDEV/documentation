
## Erreur 101 : Déclaration de variable globale interdite
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000093"></a>
Vous tentez de déclarer une variable globale dans un traitement où ce type de déclaration est interdit.


**Rappel** : Les variables globales doivent être déclarées dans :

- Le code d'initialisation du projet pour être utilisable dans tout les traitements du projet (codes du projet, codes des fenêtres et codes des champs).

- Le code d'ouverture d'une fenêtre pour être utilisable dans tous les traitements de la fenêtre (codes de la fenêtre et codes des champs présents dans la fenêtre).

- Le code de chargement de la page pour être utilisable dans tous les traitements navigateur de la page (codes de la page et codes des champs présents dans la page).






<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correction
<a name="correction_ELTTEXTE000117"></a>
Déclarez votre variable globale dans le code approprié.



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Exemple
<a name="exemple_ELTTEXTE000141"></a>


### Une variable globale est déclarée dans le code de clic d'un bouton de la fenêtre "FEN_Menu"
<a name="une_variable_globale_est_declaree_dans_code_clic_bouton_fenetre_fenmenu_ELTPARAGRAPHE000034"></a>

**Code provoquant l'erreur** 


```wl
// -- Code de clic du bouton BTN_Affiche
Global
	Resultat_Lu est un booléen
```




**Correction proposée**

Déclarez la variable globale dans le code "Déclarations globales" de la fenêtre.



```wl
// -- Déclarations globales de la fenêtre FEN_MENU
Global
	Resultat_Lu est un booléen
```

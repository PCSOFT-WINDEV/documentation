
## Information 2102 : 'Long' n'existe plus : les entiers sont sur 4 octets
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000067"></a>
Vous tentez d'utiliser un entier long : les entiers longs n'existent plus en WINDEV. Tous les entiers sont maintenant par défaut sur 4 octets.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correction
<a name="correction_ELTTEXTE000091"></a>
Utilisez un entier.

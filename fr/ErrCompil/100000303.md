
## Erreur 303 : ambiguïté sur le nom du champ
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000087"></a>
Vous tentez de manipuler une colonne d'un champ Table (Table fichier ou Table par programmation). Dans la fenêtre en cours, une colonne de même nom existe dans plusieurs champs Table.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correction
<a name="correction_ELTTEXTE000111"></a>
Précisez le nom complet de la colonne en préfixant le nom de la colonne par le nom du champ Table.



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Exemple
<a name="exemple_ELTTEXTE000135"></a>
La colonne "COL_PrixTTC" existe dans le champ "TABLE_Commande" et dans le champ Table "TABLE_LigneCde".

**Code provoquant l'erreur** 


```wl
// -- Code de clic du bouton "BTN_Calcule"
COL_PrixTTC = COL_PrixHT*COL_TVA
```


 
**Correction proposée**

Préfixez le nom de la colonne avec le nom du champ Table.



```wl
// -- Code de clic du bouton "BTN_Calcule"
TABLE_LigneCde.COL_PrixTTC = COL_PrixHT*COL_TVA
```

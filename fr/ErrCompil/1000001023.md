
## Erreur 1014 : L'accès au membre de la classe est restreint par un héritage privé
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000073"></a>
Vous tentez d'accéder à un membre d'une classe précédemment dérivée par un héritage privé. Il n'est pas possible d'accéder à ce membre depuis :

- Un code extérieur à la classe dérivée (ou extérieur à ses propres classes dérivées).

- Un code des classes dérivées de la première classe dérivée par un héritage privé.






<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correction
<a name="correction_ELTTEXTE000097"></a>
Deux possibilités : 

- Accédez à ce membre par une méthode de la classe dérivée. 

- Transformez l'héritage privé en héritage public : le membre sera alors accessible aussi bien depuis le code des classes dérivées que depuis un code extérieur à la classe.


## Erreur 14 : Instruction SINON sans SI
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000085"></a>
Vous avez utilisé le mot-clé SINON sans utiliser l'instruction conditionnelle SI correspondante.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correction
<a name="correction_ELTTEXTE000109"></a>
Vérifiez le code de votre instruction conditionnelle.



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Exemple
<a name="exemple_ELTTEXTE000133"></a>


### Le début de l'instruction conditionnelle n'a pas été saisie
<a name="debut_instruction_conditionnelle_pas_ete_saisie_ELTPARAGRAPHE000027"></a>

**Code provoquant l'erreur**


```wl
SAI_Cours = "Français"
SINON
	SAI_Cours = "Anglais"
FIN
```


**Corrections proposées**
Complétez l'instruction conditionnelle.



```wl
SI TableSelect(TABLE_Cours) = 1 ALORS
	SAI_Cours = "Français"
SINON
	SAI_Cours = "Anglais"
FIN
```

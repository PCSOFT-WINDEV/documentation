
## Erreur 531 : Les indices négatifs ou nuls sont interdits
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000095"></a>
Vous utilisez les opérateurs d'extraction de chaîne ([[ ou ]]). L'indice d'extraction d'une chaîne en WLangage commence toujours à 1 : le premier caractère a pour indice 1.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correction
<a name="correction_ELTTEXTE000119"></a>
Modifiez la valeur spécifiée pour l'indice d'extraction de chaîne.



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Exemple
<a name="exemple_ELTTEXTE000143"></a>


### Extraction du premier caractère de la chaîne 
<a name="extraction_premier_caractere_chaine_ELTPARAGRAPHE000037"></a>

**Code provoquant l'erreur**


```wl
sChaîne est une chaîne = "WEBDEV est formidable"
Trace(sChaîne[[0]])
```


**Correction proposée**

Remplacer le caractère 0 par le caractère 1 (premier caractère de la chaîne).



```wl
sChaîne est une chaîne = "WEBDEV est formidable"
Trace(sChaîne[[1]])
```

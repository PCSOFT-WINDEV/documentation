
## Erreur 4 : Il faut un espace après le caractère ':'
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000089"></a>
Cette ligne de code permet par exemple d'affecter ou d'accéder à un membre d'une classe dans une boucle de type SELON. Le caractère ':' est utilisé successivement pour :

- désigner la condition (CAS : ),

- désigner le membre de la classe.






<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correction
<a name="correction_ELTTEXTE000113"></a>
Insérez le caractère espace entre les deux "deux points". 



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Exemple
<a name="exemple_ELTTEXTE000137"></a>


### Affectation d'un membre de la classe courante
<a name="affectation_membre_classe_courante_ELTPARAGRAPHE000030"></a>

**Code provoquant l'erreur** 


```wl
SELON NbRep
	CAS 1 : i=1
	CAS 2 : i=2
	CAS 5 ::b=4
FIN
```




**Correction proposée**

Insérer un espace entre les caractères **: :**. 



```wl
SELON NbRep
	CAS 1 : i=1
	CAS 2 : i=2
	CAS 5 : : b=4
FIN
```

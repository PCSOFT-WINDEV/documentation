


## Erreur 301 : Identificateur inconnu
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000106"></a>
Vous utilisez un nom de champ, de constante ou de variable qui n'existe pas ou qui n'a pas été déclaré auparavant.

**Attention** : Cette erreur peut apparaître en tant qu'erreur ou seulement comme warning en fonction du mode de gestion des identificateurs inconnus choisi dans la description du projet : 

1. Sous le volet "Projet", dans le groupe "Projet", cliquez sur "Description".

2. Affichez l'onglet "Compilation".

3. Le mode de compilation des identificateurs inconnus peut être : 

	- Mode rigoureux : provoque une erreur de compilation. 

	- Mode permissif : effectue une déclaration implicite et provoque un warning.







<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correction
<a name="correction_ELTTEXTE000130"></a>
Vérifiez l'orthographe de l'objet utilisé et modifiez son nom si nécessaire.

**Rappel** : La coloration syntaxique présente dans l'éditeur de code permet de savoir rapidement si vous utilisez un identificateur connu ou non : si l'identificateur est inconnu, son nom apparaît en rouge.

Si l'orthographe est correcte, vérifiez l'existence du champ, de la variable ou de la procédure, et créez si nécessaire ce champ, cette variable ou cette procédure.

**Astuce** : Pour créer un procédure directement à partir de son nom, positionnez le curseur sur le nom et utilisez la touche F4.

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Exemple
<a name="exemple_ELTTEXTE000154"></a>


### Erreur dans l'orthographe d'une variable
<a name="erreur_dans_orthographe_une_variable_ELTPARAGRAPHE000052"></a>**Code provoquant l'erreur**

```wl
MonProjet est une chaîne
...
MoProjet = fRepEnCours() + NomProjet
```
**Correction proposée**
Corrigez l'orthographe de la variable.

```wl
MonProjet est une chaîne
...
MonProjet = fRepEnCours() + NomProjet
```

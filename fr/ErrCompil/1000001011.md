
## Erreur 1011 : Pour accéder au membre global de la classe, utilisez la syntaxe NOMClasse::NomMembre
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000088"></a>
Vous tentez d'accéder à un membre global en utilisant la syntaxe permettant d'accéder à un membre de la classe.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correction
<a name="correction_ELTTEXTE000112"></a>
Utilisez la syntaxe permettant d'accéder à un membre global de la classe dans le code de la classe :

```txt
NomClasse::NomMembreGlobal
```




<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Exemple
<a name="exemple_ELTTEXTE000136"></a>


### Initialisation d'un membre global dans le code de déclaration de la classe
<a name="initialisation_membre_global_dans_code_declaration_classe_ELTPARAGRAPHE000029"></a>

**Code provoquant l'erreur** 


```wl
// Initialisation d'un membre global
Dessin:pDernier = NULL
```




**Correction proposée**

Utilisez la syntaxe appropriée.



```wl
// Initialisation d'un membre global
Dessin::pDernier = NULL
```

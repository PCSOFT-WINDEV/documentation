
## Erreur 24 : Il manque le FIN associé à ce BOUCLE
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000106"></a>
Vous avez utilisé une instruction répétitive de type BOUCLE. Cette instruction doit se terminer par le mot-clé FIN.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correction
<a name="correction_ELTTEXTE000130"></a>
Vérifiez le code de votre instruction répétitive et ajoutez si nécessaire le mot-clé FIN manquant.



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Exemples
<a name="exemples_ELTTEXTE000154"></a>


### Utilisation d'un BOUCLE sans FIN
<a name="utilisation_boucle_sans_fin_ELTPARAGRAPHE000027"></a>**Code provoquant l'erreur** 


```wl
BOUCLE
	// Lecture d'une ligne du fichier texte
	UneLigne = fLitLigne(NumFic)
	SI UneLigne = EOT ALORS SORTIR
	TraiteLigne(UneLigne)
```


 
 
**Correction proposée**


Ajoutez le mot-clé FIN à la fin de la répétition.



```wl
BOUCLE
	// Lecture d'une ligne du fichier texte
	UneLigne = fLitLigne(NumFic)
	SI UneLigne = EOT ALORS SORTIR
	TraiteLigne(UneLigne)
FIN
```



<a name="NOTE3_2"></a>


### Instructions imbriquées : une instruction BOUCLE et une instruction SI sont imbriquées
<a name="instructions_imbriquees_une_instruction_boucle_une_instruction_sont_imbriquees_ELTPARAGRAPHE000047"></a>**Code provoquant l'erreur** 


```wl
BOUCLE
	// Lecture d'une ligne du fichier texte
	UneLigne = fLitLigne(NumFic)
	SI UneLigne = EOT ALORS 
		SORTIR
	TraiteLigne(UneLigne)
FIN
```
 


**Correction proposée**
Vérifiez que toutes les instructions imbriquées se terminent correctement (par l'instruction FIN par exemple). Dans notre exemple, l'instruction SI n'a pas de FIN associée. Ajoutez une instruction FIN.



```wl
BOUCLE
	// Lecture d'une ligne du fichier texte
	UneLigne = fLitLigne(NumFic)
	SI UneLigne = EOT ALORS 
		SORTIR
	FIN
	TraiteLigne(UneLigne)
FIN
```

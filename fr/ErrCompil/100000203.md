
## Erreur 203 : La procédure a déjà un attribut de portée (locale, globale)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000090"></a>
Vous déclarez une procédure en essayant de lui donner deux attributs de portée.

**Rappel** : Une procédure ne peut avoir qu'un seul attribut de portée :

- Globale : la procédure pourra être appelée dans tous les traitements des fenêtres, champs, états du projet.

- Locale (valeur par défaut) : la procédure pourra être appelée uniquement dans les traitements de la fenêtre (et des champs de la fenêtre) où cette procédure a été créée.






<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correction
<a name="correction_ELTTEXTE000114"></a>
Utilisez un seul attribut de portée lors de la déclaration de la procédure.



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Exemple
<a name="exemple_ELTTEXTE000138"></a>


### Déclaration d'une procédure
<a name="declaration_une_procedure_ELTPARAGRAPHE000032"></a>

**Code provoquant l'erreur**



```wl
PROCEDURE GLOBALE LOCALE Bourse()
```


**Correction proposée**

Spécifiez uniquement un seul attribut de portée pour cette procédure.


```wl
PROCEDURE GLOBALE Bourse()
```

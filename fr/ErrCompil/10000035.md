
## Erreur 35 : 'Continuer' n'est disponible que dans les boucles
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000089"></a>
Vous avez utilisé le mot-clé CONTINUER dans un traitement différent d'une boucle.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correction
<a name="correction_ELTTEXTE000113"></a>
Intégrez le mot-clé CONTINUER dans une instruction répétitive ou supprimez ce mot-clé.

**Remarque** : Le mot-clé CONTINUER utilisé dans une instruction répétitive permet de terminer l'itération courante immédiatement. Le code présent à la suite du mot-clé CONTINUER n'est pas exécuté. L'itération suivante est automatiquement exécutée.



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Exemple
<a name="exemple_ELTTEXTE000137"></a>


### Utilisation du mot-clé CONTINUER dans une procédure
<a name="utilisation_motcle_continuer_dans_une_procedure_ELTPARAGRAPHE000029"></a>

**Code provoquant l'erreur** 


```wl
PROCEDURE Compare(Chiffre1, Chiffre2)
SI Chiffre1>Chiffre2 ALORS 
	RETOUR
SINON
	CONTINUER
FIN
```




**Correction proposée**

Suppression du mot-clé CONTINUER.


```wl
PROCEDURE Compare(Chiffre1, Chiffre2)
SI Chiffre1>Chiffre2 ALORS 
	RETOUR
SINON
	Info(Chiffre2 + "est supérieur à " + Chiffre1)
FIN
```

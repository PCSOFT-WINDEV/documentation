
## Erreur 202 : La méthode a déjà un attribut d'accès (publique, protégée, privée)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000091"></a>
Vous utilisez une méthode de classe. Vous avez attribué plusieurs niveaux d'accès à cette méthode. Or une méthode ne peut avoir qu'un seul niveau d'accès.

**Rappel** : une méthode peut avoir l'un des 3 niveaux d'accès suivants : 

- Privé : accès autorisé seulement à l'intérieur du code de la classe.

- Protégé : accès autorisé à l'intérieur du code de la classe et des classes dérivées.

- Public : accès non restreint (mode d'accès par défaut).






<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correction
<a name="correction_ELTTEXTE000115"></a>
Modifiez votre code afin de conserver un seul niveau d'accès.



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Exemple
<a name="exemple_ELTTEXTE000139"></a>


### Déclaration d'une méthode de classe
<a name="declaration_une_methode_classe_ELTPARAGRAPHE000033"></a>

**Code provoquant l'erreur** 


```txt
METHODE PUBLIC PROTEGE CalculeBourse(Action)
```


**Correction proposée**

Spécifiez uniquement un seul niveau d'accès pour cette méthode.


```txt
METHODE PUBLIC CalculeBourse(Action)
```

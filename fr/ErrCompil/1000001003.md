
## Erreur 1003 : L'élément n'existe pas dans la classe
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000089"></a>
Vous tentez d'accéder à un membre, une méthode ou une constante qui n'a pas été définie dans la classe.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correction
<a name="correction_ELTTEXTE000113"></a>
Deux solutions : 

- Vérifier l'orthographe de l'objet à manipuler.

- Créer l'objet dans la classe.






<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Exemple
<a name="exemple_ELTTEXTE000137"></a>


### Utilisation d'une méthode mal orthographiée
<a name="utilisation_une_methode_mal_orthographiee_ELTPARAGRAPHE000030"></a>

**Code provoquant l'erreur** 


```wl
pDessin est un Dessin dynamique = allouer un Dessin(gpDessin,x,y,larg,haut)
pDessin:Affiche()
```




**Correction proposée**

La méthode Affiche n'existe pas : le nom de la méthode est Afficher. 



```wl
pDessin est un Dessin dynamique = allouer un Dessin(gpDessin,x,y,larg,haut)
pDessin:Afficher()
```

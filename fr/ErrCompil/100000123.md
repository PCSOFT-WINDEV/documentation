
## Erreur 123 : La constante existe déjà
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000103"></a>
Vous tentez de déclarer une constante qui a déjà été déclarée dans le projet ou la fenêtre en cours.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correction
<a name="correction_ELTTEXTE000127"></a>
Renommez une des constantes ou supprimez la déclaration inutile.



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Exemples
<a name="exemples_ELTTEXTE000151"></a>


### Deux constantes sont utilisées dans le même traitement avec la même valeur
<a name="deux_constantes_sont_utilisees_dans_meme_traitement_avec_meme_valeur_ELTPARAGRAPHE000027"></a>

**Code provoquant l'erreur** 


```wl
// -- Code d'initialisation du projet
constante B=5
 
Constante
	B = 5
FIN
```




**Correction proposée**

Une des déclarations est inutile. Supprimez la constante en trop.



```wl
// -- Code d'initialisation du projet
constante B=5
```



<a name="NOTE3_2"></a>


### Une constante est définie dans le code du projet puis dans le code d'initialisation d'une fenêtre
<a name="une_constante_est_definie_dans_code_projet_puis_dans_code_initialisation_une_fenetre_ELTPARAGRAPHE000044"></a>

**Code provoquant l'erreur** 


```wl
// -- Code d'initialisation du projet
constante B = 5
 
// -- Déclarations globales de la fenêtre
Constante
	B = 7
FIN
```




**Correction proposée**

Renommez la seconde déclaration.



```wl
// -- Code d'initialisation du projet
constante B = 5
 
// -- Déclarations globales de la fenêtre
Constante
	D = 7
FIN
```


## Erreur 1029 : L'accès à la classe de base est restreint par un héritage protégé
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000070"></a>
Vous tentez d'accéder à une classe de base d'un héritage. Cette classe a été dérivée par un héritage protégé : il n'est pas possible d'accéder à cette classe depuis un code extérieur à la classe ou à une de ses classes dérivées.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correction
<a name="correction_ELTTEXTE000094"></a>
Deux possibilités : 

- Accédez à la classe de base par une méthode de la classe dérivée.

- Utilisez un héritage public : la classe sera accessible aussi bien par un code extérieur à la classe que par un code de ses classes dérivées.

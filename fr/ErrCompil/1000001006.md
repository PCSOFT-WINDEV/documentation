
## Erreur 1006 : Le membre n'est pas global : utilisez l'opérateur : pour y accéder
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000098"></a>
Vous tentez d'accéder à un membre de la classe en utilisant la syntaxe permettant d'accéder à un membre global.


Vous utilisez la syntaxe suivante :



```wl
NomClasse::NomMembre
```


Cette syntaxe ne permet pas d'accéder à un membre non global.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correction
<a name="correction_ELTTEXTE000122"></a>
Utilisez la syntaxe permettant d'accéder à un membre (non global) de la classe :


```wl
NomClasse:NomMembre
```


<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Exemples
<a name="exemples_ELTTEXTE000146"></a>


### Utilisation du membre CouleurBase de la classe Dessin
<a name="utilisation_membre_couleurbase_classe_dessin_ELTPARAGRAPHE000037"></a>

**Code provoquant l'erreur**



```wl
Dessin::CouleurBase
```



**Correction proposée**

Utiliser la syntaxe correcte.



```wl
Dessin:CouleurBase
```

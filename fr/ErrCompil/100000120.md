
## Erreur 120 : Redéfinition de la structure
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000103"></a>
Vous tentez de déclarer deux structures du même nom dans le même traitement.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correction
<a name="correction_ELTTEXTE000127"></a>
Renommez une des structures ou supprimez la déclaration inutile.



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Exemples
<a name="exemples_ELTTEXTE000151"></a>


### Deux structures du même nom sont déclarées dans le même traitement
<a name="deux_structures_meme_nom_sont_declarees_dans_meme_traitement_ELTPARAGRAPHE000027"></a>**Code provoquant l'erreur**


```wl
Coord_Point est une structure
	x,y sont des entiers
FIN
 
...
 
Coord_Point est une structure
	x,y sont des entiers
FIN
```




**Correction proposée**

Supprimez la seconde structure (déclaration inutile).



```wl
Coord_Point est une structure
	x,y sont des entiers
FIN
```



<a name="NOTE3_2"></a>


### Deux structures de même nom mais utilisant des membres différents sont présentes dans le même traitement
<a name="deux_structures_meme_nom_mais_utilisant_des_membres_differents_sont_presentes_dans_meme_traitement_ELTPARAGRAPHE000044"></a>**Code provoquant l'erreur**


```wl
Coord_Point est une structure
	x,y sont des entiers
FIN
 
...
 
Coord_Point est une structure
	Coord_x,Coord_y sont des réels
FIN
```




**Correction proposée**

Supprimez la seconde déclaration (déclaration inutile).



```wl
Coord_Point est une structure
	x,y sont des entiers
FIN
 
...
 
Coord_Point_Image est une structure
	Coord_x,Coord_y sont des réels
FIN
```

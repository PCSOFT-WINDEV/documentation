
## Information 2104 : Cette fonction est conservée dans cette version pour compatibilité
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000062"></a>
Vous tentez d'utiliser une fonction du WLangage qui n'est conservée dans cette version que pour compatibilité avec les versions précédentes du produit.

Cette fonction a été renommée, modifiée, optimisée et disparaîtra dans les prochaines versions. Il est donc conseillé d'utiliser la nouvelle fonction.

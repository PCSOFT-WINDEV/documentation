
## Warning 124 : Masquage de la variable globale
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Vous avez déclaré une variable : une variable globale du même nom existe déjà dans la fenêtre en cours ou dans le projet.

Si vous conservez ce nom, la variable "globale" ne sera plus visible dans le traitement en cours : c'est la variable locale qui sera utilisée.

Attention : Si vous redéfinissez une variable globale au projet dans le code d'initialisation de la fenêtre, la nouvelle variable sera utilisée dans tous les traitements de la fenêtre et de ses champs.

**Rappel** : Portée des variables

- Les variables globales déclarées dans le code d'initialisation du projet sont visibles dans tout le projet. 

- Les variables globales déclarées dans le code d'ouverture des fenêtres sont visibles dans tous les traitements de la fenêtres et de ses champs. 






<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Conseil
<a name="conseil_ELTTEXTE000099"></a>
Évitez d'utiliser les mêmes identifiants pour des variables différentes.

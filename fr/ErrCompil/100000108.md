
## Erreur 108 : La taille de la variable est interdite
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Cause
<a name="cause_ELTTEXTE000115"></a>
Vous utilisez une variable de type entier ou réel en précisant une taille spécifique. Cette taille n'est pas disponible pour ce type de variable.

Les tailles disponibles pour le type entier sont :

- 1 octet

- 2 octets

- 4 octets

- 8 octets




Les tailles disponibles pour le type réel sont :

- réel sur 4 octets

- réel sur 8 octets






<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Correction
<a name="correction_ELTTEXTE000139"></a>
Utilisez une taille disponible pour les variables de type entier ou réel.



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Exemples
<a name="exemples_ELTTEXTE000163"></a>


### Une variable de type entier utilise une taille non reconnue
<a name="une_variable_type_entier_utilise_une_taille_non_reconnue_ELTPARAGRAPHE000039"></a>

**Code provoquant l'erreur** 


```wl
MonEntier est un entier sur 3 octets
```




**Correction proposée**

Utilisez une taille reconnue par le WLangage.



```wl
MonEntier est un entier sur 2 octets
```



<a name="NOTE3_2"></a>


### Une variable de type réel utilise une taille non reconnue
<a name="une_variable_type_reel_utilise_une_taille_non_reconnue_ELTPARAGRAPHE000056"></a>

**Code provoquant l'erreur** 


```wl
Prix est un réel sur 2 octets
```




**Correction proposée**

Utilisez une taille reconnue par le WLangage.



```wl
Prix est un réel sur 4 octets
```

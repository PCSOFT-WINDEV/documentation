


## Cadre (Propriété)

***En anglais : Border***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Cadre** permet de connaître et de modifier les caractéristiques du cadre : 

- d'un champ d'un état ou d'un bloc d'un état.
	

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) d'un champ d'une fenêtre.
	Remarque : Cette propriété n'est pas disponible sur les champs Bouton. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) d'une cellule d'un champ Table.









<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Définition des caractéristiques du cadre pour les champs de l'état
MonCadre est un Cadre
MonCadre.Couleur = RougeClair
MonCadre.Epaisseur = 5

// Utilisation du cadre pour le champ NumNbPage de l'état
NumNbPage.Cadre = MonCadre
// Modification d'une caractéristique du cadre du champ NumNbPage
NumNbPage.Cadre.Epaisseur = 1

// Récupération des caractéristiques du cadre du champ NumNbPage
MonCadre2 est un Cadre = NumNbPage.Cadre
```
<a name="Exemple2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 
```wl
// Paramètre le cadre
MonCadre est un Cadre
MonCadre.Couleur = RougeClair
MonCadre.Epaisseur = 5

// Spécificité du trait bas
MonCadre.TraitBas.Type = TraitTiret
MonCadre.TraitBas.Couleur = RougeFoncé

// Applique le cadre à la cellule 2, 2 (2ème ligne, 2ème colonne)
TABLE_MaTable[2,2].Cadre = MonCadre
// ou 
TABLE_MaTable.COL_SansNom2[2].Cadre = MonCadre
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Récupérer les caractéristiques d'un cadre d'un élément

`<Caractéristiques> = <Elément manipulé>.Cadre`
---

**`<Caractéristiques> : (Variable de type Cadre)`**

Variable de type [Cadre](../Motscles/15140100.md) contenant les caractéristiques du cadre pour l'élément spécifié.

**`<Elément manipulé> : (Nom d champ)`**



- Nom du champ ou du bloc manipulé. Cet élément est présent dans l'état en cours. 
	

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Nom du champ de la fenêtre manipulé.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Cellule d'un champ Table en utilisant une des syntaxes suivantes :

	- 
		```txt
		<Champ Table>[NumLigne,NumColonne]
		```


	- 
		```txt
		<Champ Table>.<Nom de la colonne>[NumLigne]
		```









<a name="SYNTAXE2"></a>

### Définir un cadre par programmation

`<Elément manipulé>.Cadre = <Caractéristiques du cadre>`
---

**`<Elément manipulé> : (Nom d champ)`**



- Nom du champ ou du bloc manipulé. Cet élément est présent dans l'état en cours.
	

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Nom du champ de la fenêtre manipulé.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Cellule d'un champ Table en utilisant une des syntaxes suivantes :

	- 
		```txt
		<Champ Table>[NumLigne,NumColonne]
		```


	- 
		```txt
		<Champ Table>.<Nom de la colonne>[NumLigne]
		```








**`<Caractéristiques du cadre> : (Variable de type Cadre)`**

Variable de type [Cadre](../Motscles/15140100.md) contenant les caractéristiques du cadre à appliquer à l'élément spécifié.

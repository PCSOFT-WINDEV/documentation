


## LargeurImageDossier (Propriété)

***En anglais : FolderImageWidth***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **LargeurImageDossier** permet de connaître et de modifier la largeur de l'image correspondant aux dossiers dans les champs Arbre et Table hiérarchique.

Cette propriété correspond à l'option : 

- "Largeur de l'image des dossiers" présente dans l'onglet "Détail" de la fenêtre de description des champs Arbre. 

- "Largeur" de la zone "Image des dossiers" dans l'onglet "Détail" de la fenêtre de description des champs Table hiérarchique. 





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Supprimer l'affichage des dossiers
ARBRE_MonArbre.LargeurImageDossier = 0

// Restaurer l'affichage des dossiers
ARBRE_MonArbre.LargeurImageDossier = -1 // Valeur par défaut
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître la largeur de l'image correspondant au dossier

`<Largeur actuelle> = <Champ manipulé>.LargeurImageDossier`
---

**`<Largeur actuelle> : (Entier)`**



- Largeur actuelle de l'image affichée (exprimée en pixels), 

- 0 si l'image n'est pas affichée,

- -1 si l'image est affichée avec sa largeur d'origine. 




**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler :

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Champ Arbre, 

- Champ Table hiérarchique. 





<a name="SYNTAXE2"></a>

### Modifier la largeur de l'image correspondant au dossier

`<Champ manipulé>.LargeurImageDossier = <Nouvelle largeur>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler :

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Champ Arbre,

- Champ Table hiérarchique. 




**`<Nouvelle largeur> : (Entier)`**

Nouvelle largeur de l'image affichée (exprimée en pixels).

Si ce paramètre vaut : 

- 0 si l'image n'est pas affichée,

- -1 si l'image est affichée avec sa largeur d'origine.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Il est possible de modifier la largeur de l'image affichée pour une ligne spécifique d'un champ Table hiérarchique ou d'un champ Arbre en utilisant la syntaxe suivante :

```txt
<Nom_champ>[Numéro_Ligne].LargeurImageDossier = Valeur
```

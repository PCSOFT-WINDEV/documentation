
## NomFeuille (Propriété)

***En anglais : WorksheetName***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **NomFeuille** permet de connaître ou modifier le nom de la feuille de calcul en cours dans un champ Tableur. 

**Rappel** : Si le champ Tableur gère plusieurs feuilles de calcul, le nom de la feuille apparaît dans l'onglet permettant de sélectionner la feuille (en bas du champ Tableur).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Recherche si une feuille de calcul a pour nom "FR_Stat"
POUR I = 1 A TBLR_MonTableur.NbFeuille
	TBLR_MonTableur.FeuilleEnCours = i
	SI TBLR_MonTableur.NomFeuille = "FR_Stat" ALORS
		RENVOYER I // Feuille de calcul trouvée
	FIN
FIN
// Feuille de calcul pas trouvée
RENVOYER 0
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître le nom de la feuille de calcul en cours

`<Résultat> = <Champ Tableur>.NomFeuille`
---

**`<Résultat> : (Chaîne de caractères)`**

Nom de la feuille de calcul en cours dans le champ Tableur. 

**`<Champ Tableur> : (Nom de champ)`**

Nom du champ Tableur à manipuler. 


<a name="SYNTAXE2"></a>

### Modifier le nom de la feuille de calcul en cours

`<Champ Tableur>.NomFeuille = <Nouveau nom>`
---

**`<Champ Tableur> : (Nom de champ)`**

Nom du champ Tableur à manipuler. 

**`<Nouveau nom> : (Chaîne de caractères)`**

Nouveau nom de la feuille de calcul. La taille maximale de ce nom est 32 caractères.

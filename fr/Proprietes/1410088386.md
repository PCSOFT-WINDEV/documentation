
## mdpsDérivation (Propriété)

***En anglais : spDerivation***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **mdpsDérivation** permet de :

- Connaître l'algorithme de dérivation de clé utilisé pour le hachage d'une rubrique de type "Mot de passe". Cette propriété peut être utilisée pour une rubrique de type "Mot de passe" définie sous l'éditeur d'analyses ou définie par programmation. 

- Définir l'algorithme de dérivation de clé utilisé pour le hachage (lors de la description par programmation de la rubrique "Mot de passe" uniquement). 



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Description du fichier UnFichier
...

// Description de la rubrique "Mot_Passe"
MaRubrique est une Description de Rubrique
MaRubrique.Nom = "Mot_Passe"
MaRubrique.Type = hRubMotDePasseSécurisé
MaRubrique.mdpsDérivation = MDPS_PBKDF2
MaRubrique.mdpsHash = MDPS_SHA2_256
MaRubrique.mdpsNombreItération = 20000
MaRubrique.mdpsTaille = 256

// Validation de la description de la rubrique "Mot_Passe"
HDécritRubrique(UnFichier, MaRubrique)

// Validation de la description du fichier UnFichier
...
```





<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître l'algorithme de dérivation de clé utilisé (rubrique "Mot de passe")

`<Algorithme de dérivation> = <Fichier de données>.<Rubrique>.mdpsDérivation`
---

**`<Algorithme de dérivation> : (Constante de type Entier)`**

Algorithme de dérivation de clé utilisé pour le hachage d'une rubrique de type "Mot de passe" : 



|   |   |
| --- | --- |
| *MDPS_PBKDF2* | L'algorithme PBKDF2 est utilisé. |



**`<Fichier de données> : (Chaîne de caractères)`**

Nom du fichier manipulé. Ce nom a été défini sous l'éditeur d'analyses ou grâce au type [Description de Fichier](../WDLang4/1514065.md).

**`<Rubrique> : (Chaîne de caractères)`**

Nom de la rubrique manipulée. Ce nom a été défini sous l'éditeur d'analyses ou grâce au type [Description de Rubrique](../WDLang4/1514071.md).  


<a name="SYNTAXE2"></a>

### Définir l'algorithme de dérivation de clé utilisé (rubrique "Mot de passe" décrite par programmation)

`<Rubrique>.mdpsDérivation = <Algorithme de dérivation>`
---

**`<Rubrique> : (Chaîne de caractères)`**

Nom de la rubrique manipulée. Ce nom vient d'être défini grâce au type [Description de Rubrique](../WDLang4/1514071.md).

**`<Algorithme de dérivation> : (Constante de type Entier)`**

Algorithme de dérivation de clé utilisé pour le hachage d'une rubrique de type "Mot de passe" : 



|   |   |
| --- | --- |
| *MDPS_PBKDF2* | L'algorithme PBKDF2 est utilisé. |

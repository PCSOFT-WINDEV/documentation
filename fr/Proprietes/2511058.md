


## NbEnrImprimé (Propriété)

***En anglais : NbPrintedRec***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **NbEnrImprimé** permet de connaître le nombre d'enregistrements déjà imprimés.

**Remarque** : La propriété **NbEnrImprimé** ne tient pas compte des blocs répétés jusqu'en bas de page.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Si plus de 5 enregistrements ont été imprimés
SI ETAT_EtatClient.NbEnrImprimé > 5 ALORS
	// L'impression de l'état "ETAT_EtatClient" est arrêtée
	iFinEtat()
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Etat manipulé>.NbEnrImprimé`
---

**`<Résultat> : (Entier)`**

Nombre d'enregistrements déjà imprimés.

**`<Etat manipulé> : (Nom d'état)`**

Nom de l'état à manipuler.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Nombre d'enregistrements
<a name="nombre_enregistrements_ELTPARAGRAPHE000050"></a>

- La propriété [NbEnrLu](../Proprietes/2511041.md) renvoie le nombre d'enregistrements sélectionnés par la source de l'état (requête, vue, zone mémoire ou champ Table).

- La propriété **NbEnrImprimé** renvoie le nombre d'enregistrements sélectionnés à la fois par la source de l'état ET par le code de filtre (s'il existe).
	Dans le cas d'un état sur fichier, les propriétés [NbEnrLu](../Proprietes/2511041.md) et **NbEnrImprimé** retournent la même valeur : le nombre d'enregistrements du fichier ET sélectionnés par le code de filtre (s'il existe).

- La propriété [NbEnrTotal](../Proprietes/2511047.md) permet de connaître et de modifier le nombre d'enregistrements à imprimer. Le nombre d'enregistrements à imprimer sera compris entre 1 et **NbEnrImprimé**.



<a name="NOTE0_2"></a>


### Limite
<a name="limite_ELTPARAGRAPHE000079"></a>La propriété **NbEnrImprimé** ne peut pas être utilisée sur un état sans source de données.

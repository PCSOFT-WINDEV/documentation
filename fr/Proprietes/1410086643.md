
## VersionHTTP (Propriété)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## XXX
<a name="xxx_ELTTEXTE000076"></a>
La propriété **VersionHTTP** permet de gérer la version du protocole HTTP utilisée par le serveur. Cette propriété est utilisable sur :

- une variable de type [httpRequête](../WDLang3/1000021158.md). 

- une variable générée automatiquement pour représenter [un Webservice importé](../WDLang3/3086001.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable générée automatiquement pour représenter [un Webservice REST importé](../WDLang3/1410087809.md).

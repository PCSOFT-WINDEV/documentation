


## ValeurRecherche (Propriété)

***En anglais : SearchValue***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **ValeurRecherche** permet de connaître et modifier la valeur du champ de recherche dans une Action Bar. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Evénement "A chaque modification de la recherche" d'un champ Action Bar
n = TableCherche(TABLE_Client.COL_NOM, ACTB_ACTIONBAR.ValeurRecherche)
SI n != -1 ALORS
	TABLE_Client = n
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Récupérer la valeur du champ de recherche

`<Résultat> = <Champ Action Bar>.ValeurRecherche`
---

**`<Résultat> : (Chaîne de caractères)`**

Valeur du champ de recherche de l'Action Bar. 

**`<Champ Action Bar> : (Nom de champ)`**

Nom du champ Action Bar à manipuler. 


<a name="SYNTAXE2"></a>

### Modifier la valeur du champ de recherche

`<Champ Action Bar>.ValeurRecherche = <Valeur>`
---

**`<Champ Action Bar> : (Nom de champ)`**

Nom du champ Action Bar à manipuler.

**`<Valeur> : (Chaîne de caractères)`**

Nouvelle valeur du champ de recherche.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Le champ de recherche de l'Action Bar peut être affiché avec la fonction [ActionBarRechercheVisible](../WDLang1/1000021599.md).

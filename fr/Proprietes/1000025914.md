
## Body (Propriété)

***En anglais : Body***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Body** permet de connaître et de modifier le corps HTML d'un champ d'affichage HTML ou d'un champ Editeur HTML.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Modifie la description de l'article dans le fichier de données Article
Article.Description = HTM_Navigateur.Body
HModifie(Article)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître le corps HTML d'un champ d'affichage HTML ou d'un champ Editeur HTML

`<Résultat> = <Champ manipulé>.Body`
---

**`<Résultat> : (Chaîne de caractères)`**

Code HTML du body. 

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler : 

- Champ d'affichage HTML.

- Champ Editeur HTML.  





<a name="SYNTAXE2"></a>

### Modifier le corps HTML d'un champ d'affichage HTML ou d'un champ Editeur HTML

`<Champ manipulé>.Body = <Nouvelle valeur>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler : 

- Champ d'affichage HTML.

- Champ Editeur HTML. 




**`<Nouvelle valeur> : (Chaîne de caractères)`**

Nouveau code HTML du body. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- La propriété **Body** permet de modifier uniquement le corps du code HTML présent dans le champ sans modifier son entête. Il est ainsi possible de conserver les feuilles CSS chargées par exemple.

- Si le champ est lié à une rubrique d'un fichier de données, la propriété **Body** sera utilisée par les fonctions [FichierVersEcran](../WDLang4/3044210.md) et [EcranVersFichier](../WDLang4/3044146.md).

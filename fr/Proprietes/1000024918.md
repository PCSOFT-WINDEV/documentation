
## PoliceTailleMin (Propriété)

***En anglais : FontMinSize***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **PoliceTailleMin** permet de connaître et de modifier la taille minimale de la police utilisée pour un champ Libellé avec l'option "Réduire la police" (onglet "Détail" de la fenêtre de description du champ Libellé). 

Rappel : Cette option permet de réduire la police pour permettre à un texte long d'être affiché en entier dans le champ Libellé. La police minimale peut également être spécifiée dans l'onglet "Détail" de la fenêtre de description du champ Libellé. 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Fixe la taille minimale de la police
LIB_MonTitre.PoliceTailleMin = 8
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître la taille minimale de la police utilisée par un champ Libellé

`<Résultat> = <Champ Libellé>.PoliceTailleMin`
---

**`<Résultat> : (Entier)`**



- 0 si la fonctionnalité n'est pas active (valeur par défaut). Dans ce cas, la police n'est pas réduite. 

- Taille minimale de la police. Cette taille est exprimée dans l'unité spécifiée pour l'élément (connue via la propriété [UnitéTaille](../Proprietes/1410088365.md)). 




**`<Champ Libellé> : (Nom de champ)`**

Nom du champ Libellé à manipuler.


<a name="SYNTAXE2"></a>

### Modifier la taille minimale de la police utilisée par un champ Libellé

`<Champ Libellé>.PoliceTailleMin = <Taille de la police>`
---

**`<Champ Libellé> : (Nom de champ)`**

Nom du champ Libellé à manipuler.

**`<Taille de la police> : (Entier)`**



- 0 pour désactiver la fonctionnalité. Dans ce cas, la police n'est pas réduite. 

- Taille de la police minimale. Cette taille est exprimée dans l'unité spécifiée pour l'élément (connue via la propriété [UnitéTaille](../Proprietes/1410088365.md)). 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Cette propriété est sans effet sur les champs Libellé RTF.

- Si le texte est trop long, une ellipse est affichée. 

- La taille maximale de la police est donnée par la propriété [PoliceTaille](../Proprietes/2510072.md). 

- La taille est exprimée dans l'unité spécifiée pour l'élément (connue via la propriété [UnitéTaille](../Proprietes/1410088365.md)).

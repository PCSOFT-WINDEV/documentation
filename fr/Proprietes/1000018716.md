
## Confidentialité (Propriété)

***En anglais : Sensitivity***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Confidentialité** permet de connaître et de modifier le niveau de confidentialité d'un email. 

Cette propriété est utilisable sur :

- la structure [Email](../WDLang3/3032029.md).

- une variable de type [Email](../WDLang3/1000018713.md).

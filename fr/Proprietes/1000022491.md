
## PositionDébut (Propriété)

***En anglais : StartPosition***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **PositionDébut** permet de connaître la position de début d'un élément dans un document Traitement de texte. Cette propriété est utilisable sur :

- les variables de type [docElément](../WDLang1/1000022484.md). 

- les variables de type [docFragment](../WDLang1/1000022482.md)

- les variables de type [docParagraphe](../WDLang1/1000022483.md).


## Habillage (Propriété)

***En anglais : Wrapping***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Habillage** permet de connaître et de modifier les caractéristiques d'habillage d'un élément. Cette propriété est utilisable sur :

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [docElémentImage](../WDLang1/1000022696.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [docElémentZoneDeTexte](../WDLang1/1000023420.md).

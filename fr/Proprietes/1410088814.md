
## AttributSérialise (Propriété)

***En anglais : AttributeSerialize***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **AttributSérialise** permet de connaître le nom de sérialisation du type en fonction de son nom et de l'attribut d'extension &lt;Sérialise&gt;. Cette propriété est disponible sur : 

- les variables de type [Définition](../WDLang1/1000019540.md). 

- les variables de type [Description de variable](../WDLang1/1000019560.md).

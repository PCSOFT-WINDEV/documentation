


## Réplication (Propriété)

***En anglais : Replication***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Réplication** permet de :

- connaître le mode de réplication utilisé pour un fichier de données (fichier défini sous l'éditeur d'analyses ou défini par programmation).

- spécifier le mode de réplication utilisé pour un fichier de données (lors de la description par programmation du fichier uniquement).




**Cette propriété est utilisable uniquement dans le cas d'une réplication journalée.**
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Teste si le fichier de données est marqué comme pouvant être répliqué (réplication journalée)
SI Client.Réplication = hNonRépliqué ALORS Info("Réplication impossible")
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le mode de réplication

`<Mode actuel> = <Fichier de données>.Réplication`
---

**`<Mode actuel> : (Constante de type entier)`**

Mode de réplication sont :


|   |   |
| --- | --- |
| *hNonRépliqué* | Le fichier de données n'est pas prévu pour effectuer une réplication journalée. Pour mettre en place la réplication journalée, cochez l'option "Réplication" dans les caractéristiques du fichier sous l'éditeur d'analyses. |
| *hRéplicaAbonné* | Le fichier de données est un réplica abonné (la fonction [HCréeRéplicaAbonné](../WDLang4/3044253.md) a été utilisée pour créer le réplica abonné) |
| *hRéplicaMaître* | Le fichier de données est un réplica maître (la fonction [HCréeRéplicaMaître](../WDLang4/3044175.md) a été utilisée pour créer le réplica maître) |
| *hRépliqué* | Le fichier de données est configuré pour être répliqué. |



**`<Fichier de données> : (Chaîne de caractères)`**

Nom du fichier manipulé. Ce nom a été défini sous l'éditeur d'analyses ou grâce au type [Description de Fichier](../WDLang4/1514065.md).  


<a name="SYNTAXE2"></a>

### Définir le mode de réplication

`<Fichier de données>.Réplication = <Mode de réplication>`
---

**`<Fichier de données> : (Chaîne de caractères)`**

Nom du fichier manipulé. Ce nom vient d'être défini grâce au type [Description de Fichier](../WDLang4/1514065.md).

**`<Mode de réplication> : (Constante de type entier)`**

Mode de réplication sont :


|   |   |
| --- | --- |
| *hNonRépliqué* | Le fichier de données n'est pas prévu pour effectuer une réplication journalée. |
| *hRépliqué* | Le fichier de données est configuré pour être répliqué. |

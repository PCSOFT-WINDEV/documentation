
## TexteSaisieInvalide (Propriété)

***En anglais : InvalidInputMessage***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **TexteSaisieInvalide** permet de connaître et de définir le texte qui sera affiché si le champ est invalide (masque ou borne). Si la propriété est vide, une chaine par défaut sera utilisé.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SAI_RefProduit.TexteSaisieInvalide = "Veuillez respecter le format des références de produits"
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître le texte affiché en cas de saisie invalide

`<Résultat> = <Champ manipulé>.TexteSaisieInvalide`
---

**`<Résultat> : (Chaîne de caractères)`**



- Message d'erreur utilisé en cas de saisie invalide, 

- Chaîne vide si le message d'erreur par défaut est utilisé ("Saisie invalide"). 




**`<Champ manipulé> : (Nom de champ)`**

Nom du champ manipulé : 

- Champ de saisie. 

- Champ Combo. 





<a name="SYNTAXE2"></a>

### Modifier le texte affiché en cas de saisie invalide

`<Champ manipulé>.TexteSaisieInvalide = <Message>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ manipulé : 

- Champ de saisie. 

- Champ Combo. 




**`<Message> : (Chaîne de caractères)`**



- Message d'erreur utilisé en cas de saisie invalide, 

- Chaîne vide si le message d'erreur par défaut est utilisé ("Saisie invalide").

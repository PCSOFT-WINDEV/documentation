
## ValeurAssociée (Propriété)

***En anglais : AssociatedValue***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **ValeurAssociée** permet de connaître la valeur associée. Cette propriété est utilisable sur :

- une variable de type [Description d'Option](../WDLang1/1000023072.md). 

- une variable de type [Description de Valeur](../WDLang1/1000023061.md).

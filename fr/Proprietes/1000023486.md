
## Parent (Propriété)

***En anglais : Parent***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Parent** permet de connaître l'élément parent d'un élément. Cette propriété est utilisable sur :

- une variable de type [docElément](../WDLang1/1000022484.md).

- une variable de type [docParagraphe](../WDLang1/1000022483.md).




## HauteurJour (Propriété)

***En anglais : DayHeight***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **HauteurJour** permet de connaître et de modifier la hauteur des jours dans un champ Planning avec les jours en ligne et les ressources en colonne. 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI INT_Zoom = Vrai ALORS
	PLN_MonPlanning.HauteurJour = 200
SINON
	PLN_MonPlanning.HauteurJour = 100
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître la hauteur des jours

`<Résultat> = <Champ Planning>.HauteurJour`
---

**`<Résultat> : (Entier)`**



- Hauteur des jours en pixels si la hauteur des jours a été précédemment modifiée par programmation (avec la propriété **HauteurJour**). 

- -1 dans les cas suivants : 

	- si la hauteur des jours est automatiquement calculée (grâce aux informations données dans la fenêtre de description du champ). 

	- si les ressources sont en ligne. Dans ce cas, il faut utiliser les propriétés [LargeurJour](../Proprietes/1000020108.md) et [HauteurRessource](../Proprietes/1000020109.md). 







**`<Champ Planning> : (Nom de champ)`**

Nom du champ Planning manipulé.  


<a name="SYNTAXE2"></a>

### Modifier la hauteur des jours

`<Champ Planning>.HauteurJour = <Nouvelle hauteur>`
---

**`<Champ Planning> : (Nom de champ)`**

Nom du champ Planning manipulé.

**`<Nouvelle hauteur> : (Entier)`**



- Nouvelle hauteur des jours exprimée en pixels. 

- -1 pour calculer automatiquement la hauteur des jours à partir des informations données dans la fenêtre de description du champ. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Si la hauteur des jours est fixée avec la propriété **HauteurJour**, plus aucun "ancrage" de la taille des jours en fonction de la taille disponible n'est effectuée. 

- Seule la valeur -1 peut être utilisée. Les autres valeurs négatives sont ignorées.

- Pour modifier la largeur des ressources, utilisez la propriété [LargeurRessource](../Proprietes/1000020110.md). 

- Si des ruptures sont présentes sur les jours, leur hauteur est connue grâce à la propriété [HauteurRuptureJour](../Proprietes/1000020105.md).




## CxInitialCatalog (Propriété)

***En anglais : CtInitialCatalog***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **CxInitialCatalog** permet de connaître la base de données par défaut définie pour l'accès. Il est par exemple possible de connaître la base de données par défaut définie pour la connexion.

Ces informations ont été spécifiées :

- soit lors de la création de la connexion sous l'éditeur d'analyses,

- soit lors de la définition de la connexion dans les fonctions [HConnecte](../WDLang4/3044263.md), [HDécritConnexion](../WDLang4/3044205.md) ou [HOuvreConnexion](../WDLang4/3044107.md).




Cette propriété peut être utilisée pour un fichier de données ou une connexion défini sous l'éditeur d'analyses ou défini par programmation.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connaître la base de données à utiliser sur la source de données
// de la connexion "MaConnexion"
Info(MaConnexion.CxInitialCatalog)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Initial Catalog> = <Nom du fichier/connexion>.CxInitialCatalog`
---

**`<Initial Catalog> : (Chaîne de caractères)`**



- Base de données à utiliser sur la source de données de la connexion.

- Chaîne vide ("") si aucune base de données spécifique n'est nécessaire pour la connexion.




**`<Nom du fichier/connexion> : (Chaîne de caractères)`**



- Nom du fichier de données manipulé. Ce nom a été défini sous l'éditeur d'analyses ou grâce au type [Description de Fichier](../WDLang4/1514065.md).

- Nom de la connexion manipulée. Ce nom a été défini sous l'éditeur d'analyses ou avec les fonctions [HDécritConnexion](../WDLang4/3044205.md) ou [HOuvreConnexion](../WDLang4/3044107.md).  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
La propriété **CxInitialCatalog** permet de connaître la base de données par défaut d'une connexion. Cette propriété est conservée par compatibilité. Il est conseillé d'utiliser la propriété [BaseDeDonnées](../Proprietes/2512101.md) qui permet de lire et de modifier la base de données associée à une connexion.


## MasquageAuScroll (Propriété)

***En anglais : HideWhileScrolling***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **MasquageAuScroll** permet de :

- Savoir si le champ épinglé est rendu invisible pendant le scroll, 

- Rendre ou non le champ épinglé invisible pendant le scroll.  




Remarques : 

- Lorsque le conteneur du champ épinglé est scrollé, le champ épinglé peut être rendu invisible. Dans ce cas, le champ épinglé réapparait à la fin du scroll.

- Cette propriété correspond à l'option  "Masquer le champ lors du scrolling" disponible dans l'onglet "UI" de la fenêtre de description du champ (bouton de gestion du parallaxe et des champs épinglés).



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
BTN_NouveauMessage.Epinglé = Vrai
BTN_NouveauMessage.MasquageAuScroll = Vrai
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Savoir si un champ est masqué lors du scroll de son conteneur

`<Résultat> = <Champ manipulé>.MasquageAuScroll`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le champ est masqué lors du scroll de son conteneur, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 




**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler.


<a name="SYNTAXE2"></a>

### Masquer un champ lors du scroll de son conteneur

`<Champ manipulé>.MasquageAuScroll = <Epinglé ?>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler.

**`<Epinglé ?> : ()`**



- <u><u><u><u>Vrai</u></u></u></u> si le champ doit être masqué lors du scroll de son conteneur,

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 










<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
La propriété **MasquageAuScroll** a un effet uniquement si le champ manipulé est épinglé ou possède un taux de parallaxe.




## CompacteBarreOutils (Propriété)

***En anglais : CompactToolbar***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **CompacteBarreOutils** permet de :

- savoir si les barres d'outils de la fenêtre sont compactées.

- compacter ou non les barres d'outils.




La compression des barres d'outils consiste à enlever les espaces lorsque :

- une option de la barre d'outils devient visible ou invisible. Si une option devient invisible, la barre d'outils est automatiquement redimensionnée.

- une barre d'outils devient visible ou invisible. Si une barre d'outils devient invisible, les autres barres d'outils sont automatiquement repositionnées pour ne pas laisser d'espace libre.

- une barre d'outils est dockée ou dédockée.




Les barres d'outils ne sont pas compactées si l'utilisateur déplace les barres en les laissant dockées.

**Remarque** : Cette propriété correspond à l'option "Compacter les barres d'outils" disponible dans l'onglet "Détail" de la description des fenêtres.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
FEN_FenPrincipale.CompacteBarreOutils = Vrai
BAR_Barre1.Visible = Vrai
FEN_FenPrincipale.CompacteBarreOutils = Faux
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Savoir si les barres d'outils sont compactées

`<Résultat> = <Fenêtre manipulée>.CompacteBarreOutils`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si les barres d'outils de la fenêtre sont compactées, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Fenêtre manipulée> : (Nom de fenêtre)`**

Nom de la fenêtre manipulée.  


<a name="SYNTAXE2"></a>

### Définir la compression des barres d'outils

`<Fenêtre manipulée>.CompacteBarreOutils = <Compacter>`
---

**`<Fenêtre manipulée> : (Nom de fenêtre)`**

Nom de la fenêtre manipulée.

**`<Compacter> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si les barres d'outils de la fenêtre doivent être compactées, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.

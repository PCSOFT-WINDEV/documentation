
## Utilisateur (Propriété)

***En anglais : User***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Utilisateur** permet de connaître et de modifier l'utilisateur d'une fonctionnalité. Cette propriété est utilisable sur :

- une variable de type [Connexion](../WDLang4/1514073.md). Pour plus de détails, consultez [Utilisateur](../Proprietes/2512107.md).

- une variable de type [hDescriptionSauvegarde](../WDLang4/1000017455.md). 

- une variable de type [hOptimisationPlanifiée](../WDLang4/1000018854.md).

- une variable générée automatiquement pour représenter [un Webservice importé](../WDLang3/3086001.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable générée automatiquement pour représenter [un Webservice REST importé](../WDLang3/1410087809.md).

- une variable de type [httpRequête](../WDLang3/1000021158.md).

- une variable de type [restRequête](../WDLang3/1000021481.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png) une variable de type [sshSession](../WDLang3/1000021157.md).

- une variable de type [hTâchePlanifiée](../WDLang4/1000017448.md).

- une variable de type [hPlanificationVueMatérialisée](../WDLang4/1000021444.md).

- une variable de type [hdfsConnexion](../WDLang4/1000021650.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [mqttSession](../WDLang3/1000023023.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [memcachedConnexion](../WDLang4/1000023339.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [saasVisite](../WDLang3/1000024560.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [adminWBAuth](../WDLang2/1410089328.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [adminWBCompte](../WDLang2/1410089382.md).

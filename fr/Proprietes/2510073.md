


## Nouveau (Propriété)

***En anglais : NewLine***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Nouveau** permet de savoir :

- dans un champ Table, si la ligne en cours vient d'être créée grâce aux fonctions [TableAjoute](../WDLang1/3074017.md), [TableAjouteLigne](../WDLang1/3074006.md), [TableInsère](../WDLang1/3074025.md) ou [TableInsèreLigne](../WDLang1/3074026.md).

- dans un champ Zone répétée, si la répétition en cours vient d'être créée grâce aux fonctions [ZoneRépétéeAjoute](../WDLang2/3083006.md), [ZoneRépétéeAjouteLigne](../WDLang2/3083004.md), [ZoneRépétéeInsère](../WDLang2/3083003.md) ou [ZoneRépétéeInsèreLigne](../WDLang2/3083002.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) dans un champ Agenda, si un rendez-vous vient d'être créé par le menu contextuel. Cette propriété peut être utilisée dans les événements "Entrée en saisie d'un rendez-vous" et "Sortie de saisie d'un rendez-vous".

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) dans un champ Planning, si un rendez-vous vient d'être créé par le menu contextuel. Cette propriété peut être utilisée dans les événements "Entrée en saisie d'un rendez-vous" et "Sortie de saisie d'un rendez-vous".




Remarque : La propriété **Nouveau** peut être utilisée indifféremment sur un champ lié à un fichier de données et sur un champ manipulé par programmation.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Vérifie si la ligne en cours dans le champ Table "TABLE_TableClient" vient d'être créée
ResNouveau = TABLE_TableClient.Nouveau
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Champ manipulé>.Nouveau`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si :

	- dans le champ Table, la ligne en cours vient d'être créée grâce aux fonctions [TableAjoute](../WDLang1/3074017.md), [TableAjouteLigne](../WDLang1/3074006.md), [TableInsère](../WDLang1/3074025.md) ou [TableInsèreLigne](../WDLang1/3074026.md).

	- dans le champ Zone répétée, la répétition en cours vient d'être créée grâce aux fonctions [ZoneRépétéeAjoute](../WDLang2/3083006.md), [ZoneRépétéeAjouteLigne](../WDLang2/3083004.md), [ZoneRépétéeInsère](../WDLang2/3083003.md) ou [ZoneRépétéeInsèreLigne](../WDLang2/3083002.md).

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) dans un champ Agenda, un rendez-vous vient d'être créé par le menu contextuel. 

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) dans un champ Planning, un rendez-vous vient d'être créé par le menu contextuel. 




- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire (par exemple si l'élément en cours dans le champ provient du fichier lié).




**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler : 

- champ Table, 

- champ Zone répétée,

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) champ Agenda.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) champ Planning. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Initialisation de la propriété **Nouveau**
<a name="initialisation_propriete_docparampagetitleshort_ELTPARAGRAPHE000143"></a>La propriété **Nouveau** est initialisée à <u><u><u><u>Vrai</u></u></u></u> par :

- les fonctions [TableAjoute](../WDLang1/3074017.md), [TableAjouteLigne](../WDLang1/3074006.md), [TableInsère](../WDLang1/3074025.md) et [TableInsèreLigne](../WDLang1/3074026.md)

- les fonctions [ZoneRépétéeAjoute](../WDLang2/3083006.md), [ZoneRépétéeAjouteLigne](../WDLang2/3083004.md), [ZoneRépétéeInsère](../WDLang2/3083003.md) et [ZoneRépétéeInsèreLigne](../WDLang2/3083002.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) l'ajout d'un rendez-vous par le menu contextuel automatique du champ Agenda.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) l'ajout d'un rendez-vous par le menu contextuel automatique du champ Planning.




Cette propriété est initialisée à <u><u><u><u>Faux</u></u></u></u> après l'exécution de l'événement "Sortie d'une ligne" sauf si ce traitement contient  :

- les fonctions [TableAjoute](../WDLang1/3074017.md), [TableAjouteLigne](../WDLang1/3074006.md), [TableInsère](../WDLang1/3074025.md) ou [TableInsèreLigne](../WDLang1/3074026.md)

- les fonctions [ZoneRépétéeAjoute](../WDLang2/3083006.md), [ZoneRépétéeAjouteLigne](../WDLang2/3083004.md), [ZoneRépétéeInsère](../WDLang2/3083003.md) ou [ZoneRépétéeInsèreLigne](../WDLang2/3083002.md).

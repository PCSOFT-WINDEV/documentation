


## TitreNote (Propriété)

***En anglais : NoteTitle***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **TitreNote** permet de :

- Connaître le titre des notes utilisé pour la documentation technique d'un champ, d'une fenêtre, d'une page ou d'un état. 

- Modifier le titre des notes utilisé pour la documentation technique d'un champ, d'une fenêtre, d'une page ou d'un état.




Bien souvent, cette propriété permet au développeur de stocker une information sans avoir d'influence sur l'exécution de l'application.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Initialiser le champ "SAI_Prospect" avec le titre des notes
SAI_Prospect = SAI_Prospect.TitreNote
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le titre des notes

`<Résultat> = <Elément manipulé>.TitreNote`
---

**`<Résultat> : (Chaîne de caractères)`**

Titre des notes utilisé pour la documentation technique de l'élément spécifié.

**`<Elément manipulé> : (nom de champ, de fenêtre, de page ou d'état)`**

Nom de l'élément (champ, fenêtre, page ou état) à manipuler.  


<a name="SYNTAXE2"></a>

### Modifier le titre des notes

`<Elément manipulé>.TitreNote = <Nouveau titre>`
---

**`<Elément manipulé> : (nom de champ, de fenêtre, de page ou d'état)`**

Nom de l'élément (champ, fenêtre, page ou état) à manipuler. 

**`<Nouveau titre> : (Chaîne de caractères)`**

Nouveau titre des notes utilisé pour la documentation technique de l'élément spécifié.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Divers
<a name="divers_ELTPARAGRAPHE000090"></a>

- Le titre des notes utilisé pour la documentation technique d'un champ, d'une fenêtre ou d'une page est défini dans l'onglet "Note" de la description du champ, de la fenêtre, de la page ou de l'état. Ce champ permet de saisir une description rapide du fonctionnement de l'élément en cours.

- Il est également possible d'associer une note au champ. Cette note sera également utilisée dans la documentation technique. La propriété permettant de modifier les notes est [Note](../Proprietes/2510085.md).





### Limitations
<a name="limitations_ELTPARAGRAPHE000099"></a>Dans un état, la propriété **TitreNote** s'applique à tous les champs, aux blocs et aux états.

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Dans une fenêtre, la propriété **TitreNote** ne peut pas être utilisée sur :

- un champ Forme,

- une option d'un champ Interrupteur ou Sélecteur,

- une ligne d'un champ Table ou une ligne d'un champ Liste ou Combo,

- un menu,

- un volet d'onglet,

- un groupe de champs,

- un élément d'un champ Arbre.

- un élément d'un champ Liste image.


![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) La propriété **TitreNote** s'applique uniquement aux champs Combo dans cette version.

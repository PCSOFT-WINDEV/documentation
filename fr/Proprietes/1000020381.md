


## RégletteModifiable (Propriété)

***En anglais : RulerModifiable***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **RégletteModifiable** permet de : 

- Savoir si la réglette d'un champ TimeLine peut être déplacée par l'utilisateur. 

- Rendre déplaçable ou non la réglette d'un champ TimeLine par l'utilisateur. 




**Rappel** : La réglette d'un champ TimeLine correspond à un trait vertical "gras" permettant de visualiser un instant donné dans le champ. Le style de la réglette est défini dans l'onglet "Style" du champ TimeLine (élément "Temps sélectionné").
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Clic sur le bouton BTN_Modifier
TL_TimeLine.RégletteModifiable = Vrai

// A chaque modification du champ TimeLine
// (Code exécuté à chaque modification de la réglette par l'utilisateur)
// Affiche la position sélectionnée dans le champ Libellé
LIB_PositionRéglette = TL_TimeLine.RégletteValeur
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Savoir si la réglette est modifiable par l'utilisateur

`<Résultat> = <Champ TimeLine>.RégletteModifiable`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la réglette est déplaçable par l'utilisateur, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Champ TimeLine> : (Nom du champ)`**

Nom du champ TimeLine manipulé. 


<a name="SYNTAXE2"></a>

### Rendre la réglette modifiable ou non par l'utilisateur

`<Champ TimeLine>.RégletteModifiable = <Modifiable ou non>`
---

**`<Champ TimeLine> : (Nom du champ)`**

Nom du champ TimeLine manipulé. 

**`<Modifiable ou non> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> pour rendre la réglette déplaçable par l'utilisateur, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- La réglette peut être modifiée par l'utilisateur : 

	- si l'option "Réglette modifiable" est cochée dans la fenêtre de description du champ TimeLine. 

	- si la propriété **RégletteModifiable** est à <u><u><u><u>Vrai</u></u></u></u>. 




- La position par défaut de la réglette correspond à la valeur de la propriété [DébutEtendueTotale](../Proprietes/1000020392.md). 

- L'utilisateur final pourra modifier la valeur de la réglette en faisant glisser la ligne à la souris. Cette action déclenche l'événement "A chaque modification" du champ TimeLine. 

- La nouvelle position de la réglette peut être connue grâce à la propriété [RégletteValeur](../Proprietes/1000020380.md).

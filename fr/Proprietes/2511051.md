


## MargeBasse (Propriété)

***En anglais : BottomMargin***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **MargeBasse** permet de :

- Connaître la marge basse d'un état.

- Modifier la marge basse d'un état.




**Attention** : Il est possible de modifier la marge basse d'un état uniquement dans l'événement "Ouverture de l'état".

**Remarque** : Les différentes marges d'un état sont définies dans le format de la page (option "Mise en page" du menu contextuel de l'état).<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=MargeBasse%20-%20HC%20N%B0001.GIF)



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Modifier la marge basse de l'état "ETAT_Client"
ETAT_Client.MargeBasse = 25
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître la marge basse d'un état

`<Résultat> = <Etat manipulé>.MargeBasse`
---

**`<Résultat> : (Réel)`**

Marge basse (en millimètres) de l'état spécifié.

**`<Etat manipulé> : (Nom d'état)`**

Nom de l'état à manipuler.  


<a name="SYNTAXE2"></a>

### Modifier la marge basse d'un état

`<Etat manipulé>.MargeBasse = <Nouvelle marge basse>`
---

**`<Etat manipulé> : (Nom d'état)`**

Nom de l'état à manipuler.

**`<Nouvelle marge basse> : (Réel)`**

Nouvelle marge basse (en millimètres) de l'état spécifié.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Marge physique / Marge logique
<a name="marge_physique_marge_logique_ELTPARAGRAPHE000063"></a>Chaque imprimante gère une marge physique dans laquelle aucune impression n'est possible. Les propriétés **MargeBasse**, [MargeDroite](../Proprietes/2511053.md), [MargeGauche](../Proprietes/2511048.md) et [MargeHaute](../Proprietes/2511043.md) permettent de définir des marges "logiques" d'impression.

- **Si la marge physique est supérieure à la marge logique** : La marge logique n'est pas prise en compte. L'état sera compressé lors de l'impression pour pouvoir tenir sur une seule page. Pour interdire la compression de la taille de l'état, utilisez l'option "Imprimer l'état tel quel" présente dans la fenêtre de description de l'état (onglet "Options").

- **Si la marge physique est inférieure à la marge logique** : La marge logique est prise en compte. La marge physique est ignorée.




La fonction [iFinImprime](../WDLang5/3046053.md) annule les marges.
<a name="NOTE0_2"></a>


### Limite
<a name="limite_ELTPARAGRAPHE000094"></a>La propriété **MargeBasse** s'applique uniquement aux états.

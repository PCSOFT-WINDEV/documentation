


## RéorganisationAutomatique (Propriété)

***En anglais : AutomaticReorganization***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **RéorganisationAutomatique** permet de connaître et de modifier le mode de réorganisation automatique des tâches dans une colonne de type "Diagramme de Gantt". 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI INT_DéplacementLibre ALORS
	COL_Gantt.RéorganisationAutomatique = Faux 
SINON
	COL_Gantt.RéorganisationAutomatique = Vrai
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Savoir si la réorganisation automatique des tâches est activée

`<Résultat> = <Colonne Gantt>.RéorganisationAutomatique`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la réorganisation automatique des tâches est activée. 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 




**`<Colonne Gantt> : (Nom de colonne)`**

Nom de la colonne (d'un champ Table ou Table hiérarchique) de type Diagramme de Gantt à manipuler.


<a name="SYNTAXE2"></a>

### Modifier la réorganisation automatique des tâches

`<Colonne Gantt>.RéorganisationAutomatique = <Réorganisation automatique>`
---

**`<Colonne Gantt> : (Nom de colonne)`**

Nom de la colonne (d'un champ Table ou Table hiérarchique) de type Diagramme de Gantt à manipuler.

**`<Réorganisation automatique> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> pour activer la réorganisation automatique des tâches. 

- <u><u><u><u>Faux</u></u></u></u> pour désactiver la réorganisation automatique des tâches. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Si la réorganisation automatique des tâches est activée, les tâches sont automatiquement déplacées pour qu'elles vérifient leurs prérequis.

Cette réorganisation est effectuée dans les cas suivants : 

- si l'utilisateur déplace une tâche,

- si l'utilisateur redimensionne une tâche, 

- si l'utilisateur ajoute une tâche en prérequis à une autre tâche. 




Si la réorganisation automatique n'est pas activée, aucun déplacement n'est effectué : certaines tâches peuvent donc être situées avant leur prérequis. Il est possible d'appeler la fonction [GanttOrganiseTâche](../WDLang1/1000020890.md) pour lancer cette réorganisation par programmation.

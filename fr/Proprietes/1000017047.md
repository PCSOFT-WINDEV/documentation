


## NbReçus (Propriété)

***En anglais : NbReceived***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Renvoie le nombre de paramètres réellement reçus par une procédure.  

**Remarque** : La propriété [Occurrence](../Proprietes/2510097.md) permet de connaître le nombre de paramètres utilisables dans la procédure.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Appel de la procédure
MaProc("A")
```

```wl
// Déclaration de la procédure
PROCEDURE MaProc(p1, po2 = "Y", po3 = "Z")
Info(MesParamètres.NbReçus) // Renvoie 1
Info(MesParamètres.Occurrence) // Renvoie 3
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <MesParamètres>.NbReçus`
---

**`<Résultat> : (Entier)`**

Nombre de paramètres réellement reçus par la procédure. Ce nombre ne prend pas en compte les paramètres optionnels pour lesquels aucune valeur n'a été spécifiée lors de l'appel.

**`<MesParamètres> : (Mot-clé)`**

Mot-clé du WLangage permettant de désigner l'ensemble des paramètres d'une procédure.

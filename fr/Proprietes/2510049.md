


## Groupe (Propriété)

***En anglais : Group***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Groupe** permet de : 

- savoir si le champ appartient ou non à un groupe de champs. Si le champ appartient à un ou plusieurs groupes, la propriété **Groupe** permet d'identifier ces groupes de champs.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) associer un champ à un groupe de champs. 




**Rappel** : L'association d'un champ à un groupe de champs est réalisée sous l'éditeur lors de la description du champ (onglet "UI", bouton "Groupe").


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connaître le nom du groupe de champs du champ MonChamp
NomGroupeChamps = MonChamp.Groupe
```
<a name="Exemple2"></a>

```wl
SI Position(BTN_Fermer.Groupe, "Btn_Défaut")>0 ALORS
	BTN_Fermer.Etat = Actif
FIN
// OptionFermer est le nom d'une option de menu
Trace(OptionFermer.Groupe)
```
<a name="Exemple3"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) 
```wl
// Le groupe "GR_Options" a été créé sous l'éditeur
SAI_Options.Groupe = GR_Options
GR_Options.Visible = Faux
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le nom du groupe associé à un champ

`<Nom du groupe> = <Champ manipulé>.Groupe`
---

**`<Nom du groupe> : (Chaîne de caractères)`**



- Nom du ou des groupes auxquels appartient le champ. Si le champ appartient à plusieurs groupes, les noms des différents groupes sont séparés par une virgule (par exemple "GR_GROUPE1,GR_GROUPE2").

- Chaîne vide ("") si le champ n'appartient à aucun groupe.




**`<Champ manipulé> : (Nom de champ)`**

Nom du champ pour lequel on veut connaître le ou les groupes auxquels il appartient.  


<a name="SYNTAXE2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) 
### Associer un champ à un groupe de champs

`<Champ manipulé>.Groupe = <Nom du groupe>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler.

**`<Nom du groupe> : (Chaîne de caractères)`**



- Nom du ou des groupes auxquels le champ doit être associé. Pour désigner plusieurs groupes, les noms des différents groupes doivent être séparés par une virgule (par exemple "GR_Groupe1,GR_Groupe2"). 

- Chaîne vide ("") si le champ n'appartient à aucun groupe.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
La propriété **Groupe** s'applique uniquement :

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) aux champs d'un état.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) aux champs d'une fenêtre (y compris les colonnes de table) et aux options de menus.
	Cette propriété ne peut pas être utilisée avec des groupes de champs, des lignes de table, des fenêtres, ...




![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) La propriété **Groupe** s'applique uniquement aux champs suivants :

- Bouton.

- Libellé.

- Champ de saisie.

- Image.

- Interrupteur.

- Sélecteur.

- Liste.

- Combo.

- Arbre.

- Onglet.

- Table.

- Colonne de table.

- Superchamps.

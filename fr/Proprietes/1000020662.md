


## AxeZMin (Propriété)

***En anglais : ZAxisMin***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **AxeZMin** permet de connaître la borne minimale des altitudes (axe Z) d'un champ Graphe (graphe Surface).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affiche la borne minimale des altitudes
Info("La borne minimale de l'axe des altitudes (axe Z) est " + GRF_MonGraphe.AxeZMin)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Champ Graphe>.AxeZMin`
---

**`<Résultat> : (Réel)`**

Borne minimale des altitudes.

**`<Champ Graphe> : (Nom de champ)`**

Nom du champ Graphe (de type Surface) à manipuler.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
La propriété **AxeZMin** fonctionne uniquement sur les champs Graphe de type Surface. Cette propriété n'est pas utilisable sur les graphes affichés dans une fenêtre ou dans un champ Image, ou sur les autres types de champs Graphe.

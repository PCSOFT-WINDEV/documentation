


## AjoutTerminé (Propriété)

***En anglais : AdditionCompleted***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **AjoutTerminé** permet de connaître ou de modifier l'état de l'ajout d'un élément dans une file, une liste ou une pile. Cet état (ajout en cours ou ajout terminé) est utilisé lors de la gestion du multithread.

En effet, cette propriété permet par exemple au thread qui ajoute dans la file d'indiquer au thread qui traite les éléments de la file que l'ajout d'éléments est terminé.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple
<a class="notetitle" target="_blank" href="$DOC$=2000020824&name=ajouttermine_propriete&product=WD">Voir des exemples supplémentaires</a>

```wl
// Voir l'exemple complet de la propriété pour plus de détails
gFilesnames.AjoutTerminé = Vrai
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître l'état de l'ajout d'un élément dans une file, une liste ou une pile

`<Mode d'ajout> = <Nom de l'élément>.AjoutTerminé`
---

**`<Mode d'ajout> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'ajout de l'élément dans la file, la liste ou la pile est terminé.

- <u><u><u><u>Faux</u></u></u></u> si l'ajout de l'élément dans la file, la liste ou la pile n'est pas terminé.




**`<Nom de l'élément> : (File, Liste ou Pile)`**

Nom de la variable File, Liste ou Pile manipulée.  


<a name="SYNTAXE2"></a>

### Modifier l'état de l'ajout d'un élément dans une file, une liste ou une pile

`<Nom de l'élément>.AjoutTerminé = <Nouveau mode>`
---

**`<Nom de l'élément> : (File, Liste ou Pile)`**

Nom de la variable File, Liste ou Pile manipulée.

**`<Nouveau mode> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'ajout de l'élément dans la file, la liste ou la pile est terminé.

- <u><u><u><u>Faux</u></u></u></u> si l'ajout de l'élément dans la file, la liste ou la pile n'est pas terminé.

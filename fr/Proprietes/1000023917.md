
## Destination (Propriété)

***En anglais : Destination***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Destination** permet de définir : 

- la destination d'une sauvegarde. Cette propriété est utilisable sur  une variable de type [hDescriptionSauvegarde](../WDLang4/1000017455.md). 

- le fichier résultat d'une requête. Cette propriété est utilisable sur :

	- une variable de type [httpRequête](../WDLang3/1000021158.md). 

	- une variable de type [restRequête](../WDLang3/1000021481.md). 




- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [iInfoSortie](../WDLang5/1410088954.md).




## Vignette (Propriété)

***En anglais : Thumbnail***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Vignette** permet de : 

- connaître la valeur d'une vignette d'une rubrique. Cette rubrique a été définie sous l'éditeur d'analyses ou définie par programmation.





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Valeur de la première vignette de la rubrique Client.Photo
ValeurVignette est une chaîne
ValeurVignette = Client.Photo.Vignette[1]
// équivalent à ValeurVignette = Client.Photo.Vignette

// Valeur nulle ?
SI ValeurVignette <> "" ALORS
	Info("La valeur de cette vignette est : " + ValeurVignette)
SINON
	ChaîneErreur = Client.Photo.InfoGénérationVignette[1]
	Erreur("Erreur de génération", ChaîneErreur)
FIN
```

## Syntaxe

### Connaître la valeur de la vignette d'une rubrique

`<Valeur> = <Fichier de données>.<Rubrique>.Vignette`
---

**`<Valeur> : (Chaîne de caractères)`**

Image de la vignette spécifiée sous forme de mémo binaire.

**`<Fichier de données> : (Chaîne de caractères)`**

Nom du fichier de données manipulé. Ce nom a été défini sous l'éditeur d'analyses ou grâce au type [Description de Fichier](../WDLang4/1514065.md).

**`<Rubrique> : (Chaîne de caractères)`**

Nom de la rubrique de type mémo binaire manipulée. Ce nom a été défini sous l'éditeur d'analyses ou grâce au type [Description de Rubrique](../WDLang4/1514071.md).

## Remarques


### Vignette d'un rubrique
<a name="vignette_rubrique_ELTPARAGRAPHE000106"></a>La propriété **Vignette** est disponible :

- en HFSQL Classic et en HFSQL Mobile : sur les rubriques d'un fichier de données, d'une vue ou d'une requête.

- en HFSQL Client/Serveur : sur les rubriques d'un fichier de données uniquement.




![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) La propriété **Vignette** est disponible uniquement en HFSQL Client/Serveur sur les rubriques d'un fichier de données.




## Utilisateur (Propriété)

***En anglais : User***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Utilisateur** permet de gérer l'utilisateur d'une connexion (HFSQL Client/Serveur, OLE DB, ...). Il est possible de :

- Connaître l'utilisateur d'une connexion (HFSQL Client/Serveur, OLE DB, ...).

- Définir l'utilisateur d'une connexion (HFSQL Client/Serveur, OLE DB, ...). Dans ce cas, si la connexion est actuellement ouverte, elle sera fermée, puis ré-ouverte avec les nouveaux paramètres lors d'un premier accès à un fichier utilisant cette connexion.




**Remarque** : La connexion peut correspondre :

- soit à une connexion décrite sous l'éditeur d'analyses,

- soit à une connexion décrite avec les fonctions [HConnecte](../WDLang4/3044263.md), [HDécritConnexion](../WDLang4/3044205.md) ou [HOuvreConnexion](../WDLang4/3044107.md).

- soit à une connexion décrite avec une variable de type [Connexion](../WDLang4/1514073.md).









<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MaConnexion est une Connexion
// Description de la connexion
MaConnexion.Utilisateur = "FP"
MaConnexion.MotDePasse = "Test"
MaConnexion.Serveur = "MONSERVEUR" 
MaConnexion.BaseDeDonnées = "Base de données"
MaConnexion.Provider = hAccèsHFClientServeur
MaConnexion.Accès = hOLectureEcriture
MaConnexion.InfosEtendues = "Infos étendues"
MaConnexion.OptionsCurseur = hCurseurClient

HOuvreConnexion(MaConnexion)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître l'utilisateur d'une connexion

`<Nom de l'utilisateur> = <Connexion manipulée>.Utilisateur`
---

**`<Nom de l'utilisateur> : (Chaîne de caractères)`**

Nom de l'utilisateur de la connexion.

**`<Connexion manipulée> : (Nom de la connexion)`**

Nom de la connexion manipulée. Ce nom a été défini :

- soit sous l'éditeur d'analyses,

- soit avec les fonctions [HDécritConnexion](../WDLang4/3044205.md) ou [HOuvreConnexion](../WDLang4/3044107.md),

- soit avec une variable de type [Connexion](../WDLang4/1514073.md).  





<a name="SYNTAXE2"></a>

### Définir l'utilisateur d'une connexion

`<Connexion manipulée>.Utilisateur = <Nouveau nom de l'utilisateur>`
---

**`<Connexion manipulée> : (Nom de la connexion)`**

Nom de la connexion manipulée. Ce nom a été défini :

- soit sous l'éditeur d'analyses,

- soit avec les fonctions [HDécritConnexion](../WDLang4/3044205.md) ou [HOuvreConnexion](../WDLang4/3044107.md),

- soit avec une variable de type [Connexion](../WDLang4/1514073.md).




**`<Nouveau nom de l'utilisateur> : (Chaîne de caractères)`**

Nom de l'utilisateur de la connexion.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
La propriété [CxUtilisateur](../Proprietes/2512026.md) permet de définir l'utilisateur d'une connexion. Cette propriété est conservée par compatibilité. Il est conseillé d'utiliser la propriété **Utilisateur**.

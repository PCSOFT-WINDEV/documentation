
## SaisieInvalideBloquante (Propriété)

***En anglais : InvalidInputPreventExit***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **SaisieInvalideBloquante** permet de savoir s'il est possible ou non de sortir du champ en cas de saisie invalide. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI SAI_MotDePasse.SaisieInvalideBloquante = Vrai ALORS
	SaisieInvalideAfficheBulle("Saisie non valide.")
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître le mode de gestion de la saisie invalide

`<Résultat> = <Champ manipulé>.SaisieInvalideBloquante`
---

**`<Résultat> : ()`**



- <u><u><u><u>Vrai</u></u></u></u> : il n'est pas possible de sortir du champ si la saisie est invalide et/ou si le champ est vide. 

- <u><u><u><u>Faux</u></u></u></u> : il est possible de sortir du champ dans tous les cas. 




**`<Champ manipulé> : (Nom de champ)`**

Nom du champ manipulé. Ce champ peut être : 

- un champ de saisie, 

- un champ Combo avec saisie.

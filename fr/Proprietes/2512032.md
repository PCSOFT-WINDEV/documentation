


## FichierJournal (Propriété)

***En anglais : LogFile***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **FichierJournal** permet de savoir si un fichier de données est un fichier journal ou non.

**Rappel** : Un fichier journal permet de stocker les différentes opérations réalisées sur un fichier de données journalé. Un fichier journal est créé si des options de journalisation ont été définies pour un ou plusieurs fichiers de données de l'analyse.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
EXTERNE FournisseurJNL
HDéclareExterne("C:\WinDev\Exemples\GestCom\FournisseurJNL.fic", "FournisseurJNL")
SI FounisseurJNL.FichierJournal = Vrai ALORS
	Info("Vous manipulez un fichier journal")
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Fichier de données>.FichierJournal`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le fichier est un fichier journal, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 


Remarque : Les fichiers de données décrits dans l'analyse ne sont pas des fichiers "journal".

**`<Fichier de données> : (Chaîne de caractères)`**

Nom logique du fichier de données manipulé. Peut correspondre à un nom d'alias d'un fichier déclaré par la fonction [HDéclareExterne](../WDLang4/3044204.md).

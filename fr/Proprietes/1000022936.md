
## SélectionEnAffichage (Propriété)

***En anglais : SelectionIfReadOnly***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **SélectionEnAffichage** permet de savoir et d'indiquer si le contenu d'un champ de saisie peut être sélectionné lorsqu'il est en lecture seule.

**Remarque** : Cette propriété correspond à l'option "Sélection en affichage" présente dans l'onglet "Détail" de la fenêtre de description du champ de saisie. 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SAI_Date.SélectionEnAffichage = Faux
SAI_Date.Etat = Inactif
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Savoir si le contenu d'un champ de saisie peut être sélectionné lorsqu'il est en lecture seule

`<Résultat> = <Champ de saisie>.SélectionEnAffichage`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le contenu d'un champ de saisie peut être sélectionné lorsqu'il est en lecture seule, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 




**`<Champ de saisie> : (Nom de champ)`**

Nom du champ de saisie à manipuler. 


<a name="SYNTAXE2"></a>

### Indiquer si le contenu d'un champ de saisie peut être sélectionné lorsqu'il est en lecture seule

`<Champ de saisie>.SélectionEnAffichage = <Nouvelle valeur>`
---

**`<Champ de saisie> : (Nom de champ)`**

Nom du champ de saisie à manipuler.

**`<Nouvelle valeur> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le contenu d'un champ de saisie peut être sélectionné lorsqu'il est en lecture seule, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Détails
<a name="details_ELTPARAGRAPHE000058"></a>

- Par défaut, la propriété **SélectionEnAffichage** vaut <u><u><u><u>Vrai</u></u></u></u>. 

- Si la propriété **SélectionEnAffichage** vaut <u><u><u><u>Faux</u></u></u></u> et si le champ de saisie est en lecture seule, le clic sur le champ est sans effet.

- Si le champ est grisé, la propriété **SélectionEnAffichage** est ignorée. 

- Si le champ est actif, cette propriété **SélectionEnAffichage** est ignorée.




## Vignette (Propriété)

***En anglais : Thumbnail***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Vignette** permet de connaître et de modifier l'image affichée en vignette dans un champ Vignette.

En Ajax, cette propriété est disponible en lecture seulement.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MaVignette.Image = "Image_Origin.gif"
MaVignette.Vignette = "Image_Vignette.gif"
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître l'image utilisée pour la vignette

`<Image> = <Champ Vignette>.Vignette`
---

**`<Image> : (Chaîne de caractères)`**



- Chemin du fichier image utilisé pour l'image de la vignette, 

- Chaîne vide ("") si l'image de la vignette est générée automatiquement.




**`<Champ Vignette> : (Nom de champ)`**

Nom du champ Vignette manipulé.

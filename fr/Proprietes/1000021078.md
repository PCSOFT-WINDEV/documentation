


## Membre (Propriété)

***En anglais : Member***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Membre** sur les variables de type Variant permet d'obtenir le tableau des éléments nommés. 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
v est un variant = JSONVersVariant(JSON)
POUR TOUT x DE v..Membre
	Trace("Le membre " + x.Nom + " a la valeur " + x.Valeur)
FIN
```
<a name="Exemple2"></a>

```wl
jedi est un JSON = [
	{
	"id": 5,
	"name": "Luke Skywalker",
	"pilotingScore": 98,
	"shootingScore": 56,
	"isForceUser": true
	}
]

v est un Variant = JSONVersVariant(jedi)
Trace("Dans le JSON Jedi :")
POUR TOUT x DE v..Membre
	Trace("Le membre " + x.Nom + " a la valeur " + x.Valeur)
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Obtenir le tableau des éléments nommés d'une variable de type Variant

`<Résultat> = <Variant>.Membre`
---

**`<Résultat> : (Tableau)`**

Tableau des éléments nommés. Il est possible de parcourir ce tableau via la syntaxe POUR TOUT. 

**`<Variant> : (Chaîne de caractères)`**

Nom de la variable de type Variant à manipuler.

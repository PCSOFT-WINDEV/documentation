
## CompressionMmo (Propriété)

***En anglais : MmoPackMethod***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **CompressionMmo** permet de paramétrer le mode de compression du fichier MMO (contenant les mémo binaires et texte) associé au fichier de données. Il est ainsi possible de :

- connaître le mode de compression d'un fichier mémo (fichier de données défini sous l'éditeur d'analyses ou défini par programmation).

- modifier le mode de compression des mémos (fichier de données défini par programmation uniquement).







<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Description par programmation d'un fichier de données
UnFichier est une Description de Fichier
// Modification du mode de compression de ce fichier de données
UnFichier.CompressionMmo = hCompressionZLW
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le mode de compression des mémos

`<Compression actuelle> = <Fichier de données>.CompressionMmo`
---

**`<Compression actuelle> : (Constante de type entier)`**

Les deux modes de compression sont :


|   |   |
| --- | --- |
| *hCompressionNon* | Aucune compression du fichier mémo n'est effectuée. |
| *hCompressionZLW* | Le fichier mémo est compressé selon l'algorithme de Ziw-Lempel, implémentation de Welch. |



**`<Fichier de données> : (Chaîne de caractères)`**

Nom du fichier de données manipulé. Ce nom a été défini sous l'éditeur d'analyses ou grâce au type [Description de Fichier](../WDLang4/1514065.md).  


<a name="SYNTAXE2"></a>

### Définir le mode de compression des mémos

`<Fichier de données>.CompressionMmo = <Nouveau mode de compression>`
---

**`<Fichier de données> : (Chaîne de caractères)`**

Nom du fichier de données manipulé. Ce nom vient d'être défini grâce au type [Description de Fichier](../WDLang4/1514065.md).

**`<Nouveau mode de compression> : (Constante de type entier)`**

Les deux modes de compression sont :


|   |   |
| --- | --- |
| *hCompressionNon* | Aucune compression du fichier mémo n'est effectuée. |
| *hCompressionZLW* | Le fichier mémo est compressé selon l'algorithme de Ziw-Lempel, implémentation de Welch. |

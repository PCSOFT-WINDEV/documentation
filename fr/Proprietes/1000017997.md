


## DateFin (Propriété)

***En anglais : EndDate***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **DateFin** permet de connaître et de modifier la date de fin d'une période. Cette propriété est utilisable sur :

- les champs Calendrier, Agenda et Planning. Pour plus de détails, consultez [Propriété DateFin](../Proprietes/1000017427.md).

- une variable de type [saasAbonnement](../WDLang3/1000019088.md).

- une variable de type [gglEvénement](../WDLang5/1000017440.md). 

- une variable de type [RendezVous](../WDLang1/1000019244.md).

- une variable de type [Répétition](../WDLang1/1000019262.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [TâcheGantt](../WDLang1/1000020881.md).

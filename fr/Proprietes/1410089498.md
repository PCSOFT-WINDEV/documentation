
## TypeAutorisation (Propriété)

***En anglais : GrantType***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **TypeAutorisation** permet de connaître le type d'autorisation disponible. Cette propriété est utilisée par :

- une variable de type [OAuth2Paramètres](../WDLang3/1000022212.md). 

- une variable de type [OpenIDParamètres](../WDLang3/1000023561.md).

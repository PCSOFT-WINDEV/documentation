
## Extra (Propriété)

***En anglais : Extra***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Extra** permet de :

- Connaître les informations supplémentaires associées à un champ, à une fenêtre, ou à une page.

- Modifier les informations supplémentaires associées à un champ, à une fenêtre ou à une page.






Les informations supplémentaires sont destinées aux développeurs et sont disponibles et manipulables uniquement par programmation. 

Cette propriété permet au développeur de stocker une information avancée sans avoir d'influence sur l'exécution de l'application. Par exemple, il  est ainsi possible de stocker une valeur de n'importe quel type, un tableau, etc. Il est également possible d'ajouter des membres à la propriété **Extra**. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SAI_Lib.Extra.ValeurPrécédente = SAI_Ville
SAI_Lib.Extra.DateModif = DateSys()
```

```wl
MonChamp.Extra = Valeur
MonChamp.Extra.Membre = Valeur
MonChamp.Extra["Membre"] = Valeur
```





<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître les informations supplémentaires associées à un élément (champ, fenêtre, page, etc.)

`<Informations> = <Elément manipulé>.Extra`
---

**`<Informations> : (Types des informations)`**

Informations associées à l'élément spécifié.

**`<Elément manipulé> : (Nom de champ, nom de fenêtre ou nom de page)`**

Nom de l'élément (champ, fenêtre ou page) à manipuler.   


<a name="SYNTAXE2"></a>

### Modifier les informations supplémentaires associées à un élément (champ, fenêtre, page, etc.)

`<Elément manipulé>.Extra = <Nouvelles informations>`
---

**`<Elément manipulé> : (Nom de champ, nom de fenêtre ou nom de page)`**

Nom de l'élément (champ, fenêtre ou page) à manipuler.  

**`<Nouvelles informations> : (Types des informations)`**

Informations associées à l'élément spécifié.

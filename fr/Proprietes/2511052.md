


## LargeurPixel (Propriété)

***En anglais : WidthInPixel***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **LargeurPixel** permet de connaître la largeur d'un champ d'un état en pixels.

**Remarque** : Le cadre du champ est compris dans sa largeur. La propriété [Largeur](../Proprietes/2510059.md) permet de connaître et de modifier la largeur d'un champ en millimètres.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connaître la largeur du champ "RUB_NomClient"
ResLargeur = RUB_NomClient.LargeurPixel
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Champ manipulé>.LargeurPixel`
---

**`<Résultat> : (Réel)`**

Largeur (en pixels) du champ spécifié.

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ (d'un état) à manipuler.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
La propriété **LargeurPixel** ne peut pas être utilisée sur :

- un état dans son ensemble,

- un bloc de l'état,

- un groupe de champs,

- un champ Interrupteur, 

- un champ de saisie, 

- un champ Agenda,

- un champ Planning, 

- un champ Diagramme de Gantt, 

- un champ Signature.

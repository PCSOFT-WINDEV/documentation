
## DescriptionCodeEtat (Propriété)

***En anglais : DescriptionStatusCode***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **DescriptionCodeEtat** permet de connaître la description correspondant à l'état de la réponse à la requête. Cette propriété est utilisable sur :

- une variable de type [httpRéponse](../WDLang3/1000021165.md). 

- une variable de type [restRéponse](../WDLang3/1000021477.md).

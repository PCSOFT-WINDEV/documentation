


## MargeGauche (Propriété)

***En anglais : LeftMargin***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **MargeGauche** permet de :

- Connaître la marge de gauche d'un état.

- Modifier la marge de gauche d'un état.




**Attention** : Il est possible de modifier la marge de gauche d'un état uniquement dans l'événement "Ouverture de l'état".

**Remarque** : Les différentes marges d'un état sont définies dans le format de la page (option "Mise en page" dans le menu contextuel de l'état).
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=MargeBasse%20-%20HC%20N%B0001.GIF)



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Modifier la marge de gauche de l'état "ETAT_Client"
ETAT_Client.MargeGauche = 25
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître la marge de gauche d'un état

`<Résultat> = <Etat manipulé>.MargeGauche`
---

**`<Résultat> : (Réel)`**

Marge de gauche (en millimètres) de l'état spécifié.

**`<Etat manipulé> : (Nom d'état)`**

Nom de l'état à manipuler.  


<a name="SYNTAXE2"></a>

### Modifier la marge de gauche d'un état

`<Etat manipulé>.MargeGauche = <Nouvelle marge de gauche>`
---

**`<Etat manipulé> : (Nom d'état)`**

Nom de l'état à manipuler.

**`<Nouvelle marge de gauche> : (Réel)`**

Nouvelle marge de gauche (en millimètres) de l'état spécifié.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Marge physique / Marge logique
<a name="marge_physique_marge_logique_ELTPARAGRAPHE000064"></a>Chaque imprimante gère une marge physique dans laquelle aucune impression n'est possible. Les propriétés [MargeBasse](../Proprietes/2511051.md), [MargeDroite](../Proprietes/2511053.md), **MargeGauche** et [MargeHaute](../Proprietes/2511043.md) permettent de définir des marges "logiques" d'impression.

- **Si la marge physique est supérieure à la marge logique** : La marge logique n'est pas prise en compte. L'état sera compressé lors de l'impression pour pouvoir tenir sur une seule page. Pour interdire la compression de la taille de l'état, utilisez l'option "Imprimer l'état tel quel" présente dans la fenêtre de description de l'état (onglet "Options").

- **Si la marge physique est inférieure à la marge logique** : La marge logique est prise en compte. La marge physique est ignorée.




La fonction [iFinImprime](../WDLang5/3046053.md) annule les marges.
<a name="NOTE0_2"></a>


### Limite
<a name="limite_ELTPARAGRAPHE000095"></a>La propriété **MargeGauche** s'applique uniquement aux états.


## Serveur (Propriété)

***En anglais : Server***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Serveur** permet de gérer les serveurs. Cette propriété est utilisable sur :

- une connexion HFSQL. Pour plus de détails, consultez [Serveur](../Proprietes/2512105.md).

- une variable de type [hRSConfig](../WDLang4/1000020484.md).

- une variable de type [hbConnexion](../WDLang4/1000021685.md).

- une variable de type [memcachedConnexion](../WDLang4/1000023339.md).

- une variable de type [redisConnexion](../WDLang4/1000023527.md).

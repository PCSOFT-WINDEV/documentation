


## CadrageHorizontal (Propriété)

***En anglais : HorizontalAlignment***
	



<a name="XPresentation"></a>
<a name="Presentation"></a>
Le cadrage horizontal d'un champ consiste à définir la position du texte dans le champ par rapport aux bords du champ. Ainsi, il est possible de cadrer le contenu d'un champ Libellé par rapport aux bords de la zone du libellé.

Plusieurs cadrages sont à votre disposition :

- cadrage à gauche ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Cadrage_gauche.gif)


- cadrage à droite ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Cadrage_Droite.gif)


- centré ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Cadrage_centre.gif)


- justifié



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **CadrageHorizontal** permet de connaître et de modifier le cadrage horizontal d'un champ.

Le cadrage initial du champ est défini sous l'éditeur lors de la description du champ.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Si le champ est cadré à gauche, on le cadre à droite
SI LIB_libellé1.CadrageHorizontal = chGauche ALORS
	LIB_libellé1.CadrageHorizontal = chDroite
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le cadrage horizontal d'un champ

`<Cadrage actuel>.CadrageHorizontal = <Champ manipulé>`
---

**`<Cadrage actuel> : (Constante de type entier)`**

Cadrage horizontal actuel du champ. Les valeurs possibles sont les suivantes : 



|   |   |
| --- | --- |
| *chCentre* | Le texte est centré. |
| *chDroite* | Le texte est aligné à droite. |
| *chGauche* | Le texte est aligné à gauche. |
| *chJustifié* | Le texte est justifié. |
| *chNonDéfini* | Cadrage défini par l'héritage du CSS.<br>Cette constante est disponible uniquement en WEBDEV. |



**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler.  


<a name="SYNTAXE2"></a>

### Modifier le cadrage horizontal d'un champ

`<Champ manipulé>.CadrageHorizontal = <Nouveau cadrage>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler. 

**`<Nouveau cadrage> : (Constante de type entier)`**

Nouveau cadrage horizontal du champ. Les valeurs possibles sont les suivantes : 



|   |   |
| --- | --- |
| *chCentre* | Le texte est centré. |
| *chDroite* | Le texte est aligné à droite. |
| *chGauche* | Le texte est aligné à gauche. |
| *chJustifié* | Le texte est justifié. |
| *chNonDéfini* | Cadrage défini par l'héritage du CSS.<br>Cette constante est disponible uniquement en WEBDEV. |

### Cadrage horizontal et texte de taille supérieure au champ
<a name="cadrage_horizontal_texte_taille_superieure_champ_ELTPARAGRAPHE000154"></a>Si le texte ne rentre pas complètement dans le champ, les informations suivantes sont affichées :

- Sous l'éditeur d'états, la partie gauche du texte est affichée.

- Sous l'éditeur de fenêtres :

	- dans un libellé, la partie droite du texte est affichée.

	- dans un champ de saisie, la partie gauche du texte est affichée.

### Limitation
<a name="limitation_ELTPARAGRAPHE000177"></a>![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Dans une fenêtre, la propriété **CadrageHorizontal** permet de connaître et de modifier le cadrage horizontal des champs de type :

- Libellé

- Champ de saisie

- Combo (avec ou sans saisie)

- Interrupteur / Sélecteur

- Liste

- Liste image

- Image

- Colonne de table et colonne de table hiérarchique

- Case de barre de message

- Code-barres

- Graphe

- Jauge. La propriété permet de connaître et modifier le cadrage horizontal du pourcentage de progression affiché dans la jauge.

- Champ Notation

- Organigramme


 
 
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Dans un état, la propriété **CadrageHorizontal** permet de connaître et de modifier le cadrage horizontal des champs de type :

- Champ Libellé.

- Champ Calculé.

- Champ Prédéfini.

- Champ Graphe.

- Groupes de champs. Cette propriété s'appliquera uniquement aux champs du groupe gérant le cadrage horizontal.


![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) La propriété **CadrageHorizontal** s'applique uniquement aux champs suivants :

- Libellé.

- Champ de saisie.

- Image.

- Interrupteur.

- Sélecteur.

- Liste.

- Combo.

- Colonne de table (de type texte, combo ou interrupteur).

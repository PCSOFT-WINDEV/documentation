
## NuméroPage (Propriété)

***En anglais : PageNumber***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **NuméroPage** permet de connaître et de modifier le numéro d'une page. Cette propriété est utilisable sur :

- un champ Image ou un état. Pour plus de détails, consultez [ NuméroPage](../Proprietes/2510106.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [ pdfSignet](../WDLang6/1410089195.md).

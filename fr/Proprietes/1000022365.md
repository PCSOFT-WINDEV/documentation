


## CodeCSS (Propriété)

***En anglais : CodeCSS***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **CodeCSS** permet de lire et modifier le code CSS associé à un champ Composant Web.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
Trace(CMP_MonComposantWeb.CodeCSS)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître le code CSS associé au champ Composant Web

`<Résultat> = <Champ Composant Web>.CodeCSS`
---

**`<Résultat> : (Chaîne de caractères)`**

Code CSS associé au champ Composant Web. Ce code CSS sera écrit dans la page. 

**`<Champ Composant Web> : (Nom de champ)`**

Nom du champ Composant Web à manipuler. 


<a name="SYNTAXE2"></a>

### Modifier le code CSS associé au champ Composant Web

`<Champ Composant Web>.CodeCSS = <Code CSS>`
---

**`<Champ Composant Web> : (Nom de champ)`**

Nom du champ Composant Web à manipuler. 

**`<Code CSS> : (Chaîne de caractères)`**

Nouveau code CSS associé au champ.

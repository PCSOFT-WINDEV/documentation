***En anglais : ZOrder***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Altitude** permet de connaître et de modifier l'altitude d'un élément. Cette propriété est utilisable sur :

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) les champs d'une fenêtre ou d'un état. Pour plus de détails, consultez l'aide de la propriété [Altitude](../Proprietes/2510002.md).
	

- une variable de type [géoPosition](../WDLang3/1000019191.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [docElémentImage](../WDLang1/1000022696.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [docElémentZoneDeTexte](../WDLang1/1000023420.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [Marqueur](../WDLang3/1000019940.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [CarteCercle](../WDLang3/1000025504.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [CarteImage](../WDLang3/1000025523.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [diagActionAutorisée](../WDLang1/1410088840.md).

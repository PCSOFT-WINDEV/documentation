
## Modification (Propriété)

***En anglais : Modification***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Modification** permet de spécifier les modifications à effectuer. Cette propriété est disponible sur : 

- une variable de type [mongoOpérationModificationUn](../WDLang4/1410089164.md).

- une variable de type [mongoOpérationModificationTout](../WDLang4/1410089165.md).


## CelluleDéfaut

***En anglais : DefaultCell***
	

<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **CelluleDéfaut** permet de gérer le style des cellules dans les documents XLS. 

Cette propriété est utilisable sur :

- une variable de type [xlsColonne](../WDLang5/1000017471.md).

- une variable de type [xlsLigne](../WDLang5/1000017325.md).

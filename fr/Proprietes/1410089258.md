
## Document (Propriété)

***En anglais : Document***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le mot-clé **Document** peut correspondre : 

- à la variable de type [Document](../WDLang1/1000022461.md), utilisée pour manipuler des documents de type Traitement de texte. 

- à la propriété **Document**. Cette propriété est utilisable sur :

	- une variable de type [mongoOpérationAjout](../WDLang4/1410089163.md).

	- une variable de type [mongoOpérationRemplacementUn](../WDLang4/1410089166.md).

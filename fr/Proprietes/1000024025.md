


## JetonSupprimable (Propriété)

***En anglais : TokenDeletable***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **JetonSupprimable** permet de : 

- Savoir si la suppression des jetons par l'utilisateur final est autorisée ou non dans un champ de saisie de type "Jetons de texte". 

- Autoriser ou non la suppression des jetons par l'utilisateur final dans un champ de saisie de type "Jetons de texte".



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI INT_AutoriseSuppression ALORS
	SAI_Email.JetonSupprimable = Vrai 
SINON
	SAI_Email.JetonSupprimable = Faux
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Savoir si la suppression des jetons par l'utilisateur final est autorisée 

`<Résultat> = <Champ de saisie>.JetonSupprimable`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la suppression des jetons par l'utilisateur final est autorisée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 




**`<Champ de saisie> : (Nom de champ)`**

Nom du champ de saisie de type "Jetons de texte" manipulé. 


<a name="SYNTAXE2"></a>

### Autoriser ou non la suppression des jetons par l'utilisateur final 

`<Champ de saisie>.JetonSupprimable = <Autorisation>`
---

**`<Champ de saisie> : (Nom de champ)`**

Nom du champ de saisie de type "Jetons de texte" manipulé. 

**`<Autorisation> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la suppression des jetons est autorisée,

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Une erreur est générée : 

	- si le champ de saisie n'est pas de type "Jetons de texte". 

	- Si la propriété [JetonActif](../Proprietes/1000023782.md) correspond à <u><u><u><u>Faux</u></u></u></u>.




- Si la propriété **JetonSupprimable** correspond à <u><u><u><u>Vrai</u></u></u></u> (valeur par défaut), l'utilisateur a le droit de supprimer les jetons avec la croix de suppression ou la touche Suppr. 

- Si la propriété **JetonSupprimable** correspond à <u><u><u><u>Faux</u></u></u></u>, la croix de suppression n'est pas affichée et la touche Suppr est sans effet.

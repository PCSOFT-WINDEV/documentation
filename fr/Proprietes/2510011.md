
## AvanceRapide (Propriété)

***En anglais : ScrollValue***
	



<a name="XPresentation"></a>
<a name="Presentation"></a>
L'avance rapide d'un champ Ascenseur correspond au déplacement généré par un clic dans la barre située entre la cage de l'ascenseur et l'une des flèches de défilement. 

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=avancerapide.gif)


L'avance rapide d'un champ Spin correspond au pas d'incrémentation du spin. 
<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **AvanceRapide** permet de gérer la valeur de l'avance rapide. Il est possible de :

- Connaître la valeur de l'avance rapide d'un champ Ascenseur, Spin ou Range Slider.

- Modifier la valeur de l'avance rapide d'un champ Ascenseur, Spin ou Range Slider.





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Définir l'avance rapide
ASC_Ascenseur1.AvanceRapide = 10
// Simuler l'avance rapide d'une page
ASC_Ascenseur1 += ASC_Ascenseur1.AvanceRapide
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître l'avance rapide

`<Avance rapide actuelle> = <Champ manipulé>.AvanceRapide`
---

**`<Avance rapide actuelle> : (Entier)`**

Déplacement généré par un clic sur la barre du champ Ascenseur (ou du champ Range Slider) ou sur le bouton du champ Spin.

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ manipulé : 

- Ascenseur,

- Spin,

- Range Slider.





<a name="SYNTAXE2"></a>

### Modifier l'avance rapide

`<Champ manipulé>.AvanceRapide = <Nouvelle avance rapide>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ manipulé : 

- Ascenseur,

- Spin,

- Range Slider.




**`<Nouvelle avance rapide> : (Entier)`**

Nouvelle valeur du déplacement généré par un clic sur la barre du champ Ascenseur (ou du champ Range Slider) ou sur le bouton du champ Spin.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Avance rapide et bornes
<a name="avance_rapide_bornes_ELTPARAGRAPHE000076"></a>Si les propriétés [BorneMin](../Proprietes/2510008.md) et [BorneMax](../Proprietes/2510009.md) sont inférieures à **AvanceRapide**, la propriété **AvanceRapide** prend automatiquement la valeur suivante : (BorneMax - BorneMin).
<a name="NOTE0_2"></a>




### Limites
<a name="limites_ELTPARAGRAPHE000096"></a>La propriété **AvanceRapide** s'applique uniquement aux champs d'une fenêtre : 

- Ascenseur. 

- Spin.

- Range Slider.




![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) En Linux, cette propriété est disponible uniquement pour les champs Ascenseur.

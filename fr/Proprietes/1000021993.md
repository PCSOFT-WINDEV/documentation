


## ScreenCastEnCours (Propriété)

***En anglais : ScreenCastInProgress***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **ScreenCastEnCours** permet de savoir si la vidéo jouée dans un champ Multimédia est actuellement jouée sur un périphérique externe (compatible Air Play). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI MM_Multi.ScreenCastEnCours = Vrai ALORS
	// Code si la vidéo est actuellement jouée sur un périphérique externe (compatible Air Play)
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Champ Multimédia>.ScreenCastEnCours`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la vidéo est actuellement jouée sur un périphérique externe, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Champ Multimédia> : (Nom de champ)`**

Nom du champ Multimédia à manipuler.




## DateDébutPériodeAffichée (Propriété)

***En anglais : StartDatePeriodDisplayed***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **DateDébutPériodeAffichée** permet de connaître la date de début de la période affichée dans un champ Planning ou Agenda d'un état.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Traitement avant impression du bloc d'entête
PLN_MonPlanning.LibellePériodeEntête = ...
	" Semaine du " + DateVersChaîne(PLN_MonPlanning.DateDébutPériodeAffichée) + ...
	" au " + DateVersChaîne(PLN_MonPlanning.DateFinPériodeAffichée)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître la date de début de la période affichée dans l'état

`<Date de début> = <Champ manipulé>.DateDébutPériodeAffichée`
---

**`<Date de début> : (Chaîne de caractères ou variable de type Date)`**

Date du début de la période affichée dans l'état.  

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ Planning ou Agenda présent dans l'état.


## RGPD (Propriété)

***En anglais : GDPR***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **RGPD** permet de : 

- savoir si une rubrique d'un fichier de données est concernée par la RGPD. 

- définir si une rubrique doit être concernée par la RGPD (lors de la description de la rubrique par programmation uniquement).




Rappel : La définition des rubriques concernées par la RGPD est réalisée sous l'éditeur d'analyses.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Description du fichier UnFichier
...

// Description de la rubrique "NOM"
MaRubrique est une Description de Rubrique
MaRubrique.Nom = "NOM"
MaRubrique.Type = hRubTexte
MaRubrique.Taille = 40
MaRubrique.TypeClé = hCléUnique
MaRubrique.RGPD = Vrai

// Validation de la description de la rubrique "NOM"
HDécritRubrique(UnFichier, MaRubrique)

// Validation de la description du fichier UnFichier
...
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Savoir si la rubrique est concernée par la RGDP

`<Résultat> = <Fichier de données>.<Rubrique>.RGPD`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la rubrique associée à l'enregistrement est concernée par la RGPD,

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.  


Nom logique du fichier de données manipulé. Ce nom a été défini sous l'éditeur d'analyses ou grâce au type [Description de Fichier](../WDLang4/1514065.md).

**`<Rubrique> : (Chaîne de caractères)`**

Nom de la rubrique manipulée. Ce nom a été défini sous l'éditeur d'analyses ou grâce au type [Description de Rubrique](../WDLang4/1514071.md).


<a name="SYNTAXE2"></a>

### Définir si une rubrique est concernée par la RGPD

`<Rubrique>.RGPD = <Mode de gestion>`
---

**`<Rubrique> : (Chaîne de caractères)`**

Nom de la rubrique manipulée. Ce nom vient d'être défini grâce au type [Description de Rubrique](../WDLang4/1514071.md).

**`<Mode de gestion> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la rubrique est concernée par la RGPD, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.


## CorrectionAutomatique (Propriété)

***En anglais : AutoCorrect***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **CorrectionAutomatique** permet de définir un dictionnaire des corrections automatiques à appliquer dans un champ Traitement de texte pendant la saisie.  
<a name="Exemple1"></a>
<a name="1410088985_sample_code"></a>

## Exemple


```wl
c est un docRemplacementAutomatique
c.NouvelleChaîne = "10 fois plus vite"
TT_MondocCommercial.CorrectionAutomatique.Remplacement["10f"] = c

// Autre syntaxe acceptée (sans passer par la variable docRemplacementAutomatique)
TT_MondocCommercial.CorrectionAutomatique.Remplacement["20f"] = "20 fois plus vite"
```
<a name="1410088985_Exemple2"></a>

```wl
d est un Document
DocInsère(d, 1, "Bonjour")

TT_MondocCommercial.CorrectionAutomatique.Remplacement["bb"] = d

d2 est un Document 
d2 <- TT_MondocCommercial..CorrectionAutomatique.Remplacement["bb"]..NouvelleChaîneFormatée
```



<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître la valeur d'un remplacement automatique

`<Résultat> = <Champ Traitement de texte>.CorrectionAutomatique.Remplacement[<Chaîne originale>]`
---

**`<Résultat> : (Variable de type docRemplacementAutomatique)`**

Variable de type [docRemplacementAutomatique](../WDLang1/1410088985.md) correspondant au remplacement effectué pour la chaîne originale. 

**`<Champ Traitement de texte> : (Nom de champ)`**

Nom du champ Traitement de texte manipulé. 

**`<Chaîne originale> : (Chaîne de caractères)`**

Chaîne de caractères présente dans le champ Traitement de texte à remplacer lors de la saisie. 


<a name="SYNTAXE2"></a>

### Définir la valeur d'un remplacement automatique

`<Champ Traitement de texte>.CorrectionAutomatique.Remplacement[<Chaîne originale>] = <Nouvelle valeur>`
---

**`<Champ Traitement de texte> : (Nom de champ)`**

Nom du champ Traitement de texte manipulé. 

**`<Chaîne originale> : (Chaîne de caractères)`**

Chaîne de caractères présente dans le champ Traitement de texte à remplacer lors de la saisie. 

**`<Nouvelle valeur> : (Chaîne de caractères ou variable de type docRemplacementAutomatique)`**



- Chaîne à utiliser pour effectuer le remplacement. 

- Nom de la variable de type [docRemplacementAutomatique](../WDLang1/1410088985.md) à utiliser pour effectuer le remplacement.

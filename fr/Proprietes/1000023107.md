
## Major (Propriété)

***En anglais : Major***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Major** permet de connaître et de modifier le numéro Major d'une balise Beacon. Cette propriété est utilisable sur : 

- une variable de type [beaconInfoDétection](../WDLang3/1000023109.md). 

- une variable de type [beaconGroupe](../WDLang3/1000023105.md).

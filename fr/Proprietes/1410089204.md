
## Carte (Propriété)

***En anglais : Card***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Carte** permet de manipuler les différentes cartes associées à une liste de Kanban. Il est ainsi possible par exemple de parcourir, ajouter, modifier ou supprimer les cartes d'une liste de Kanban. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Modifie toutes les cartes de la liste "KANBAN_Liste_TODO"
// Remplace la date par la date du jour
POUR i = 1 _A_ KANBAN_Liste_TODO.Carte.Ocurrence()
	KANBAN_Liste_TODO.Carte[i].Date = DateSys
FIN

// Ajoute une carte dans liste "KANBAN_Liste_TODO"
T1 est une kbCarte
T1.Titre = "Documenter"
T1.Texte = "Ecrire la documentation"
KANBAN_Liste_TODO.Carte.Ajoute(T1)
```

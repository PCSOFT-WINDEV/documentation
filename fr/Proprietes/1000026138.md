
## Fragment (Propriété)

***En anglais : Fragment***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Fragment** permet de manipuler un fragment. Cette propriété est utilisable sur :

- une variable de type [URI](../WDLang3/1000024027.md). 

- une variable de type [edtHTMLSélection](../WDLang1/1000026093.md).




## QuadrillageVisible (Propriété)

***En anglais : GridlinesVisible***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **QuadrillageVisible** permet de : 

- Savoir si le quadrillage est visible ou non dans un champ. 

- Modifier la visibilité du quadrillage dans un champ. 




Cette propriété est disponible : 

- pour les champs Tableur présents :

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) dans une fenêtre.

	- dans un état.




- pour les champs Editeur de diagrammes.



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Bouton [Quadrillage masqué]
TBLR_ChampTableur.QuadrillageVisible = Faux
```

```wl
// Le champ Interrupteur INT_Options propose les options suivantes : 
// Bibliothèque
// Styles
// Barre d'outils
// Grille
EDIAG_Diagramme.PanneauBibliothèqueVisible = INT_Options[1]
EDIAG_Diagramme.PanneauModifieurVisible = INT_Options[2]
EDIAG_Diagramme.BarreOutilsVisible = INT_Options[3]
EDIAG_Diagramme.QuadrillageVisible = INT_Options[4]
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Savoir si le quadrillage d'un champ Tableur est visible 

`<Résultat> = <Champ manipulé>.QuadrillageVisible`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le quadrillage du champ est visible, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 




**`<Champ manipulé> : (Nom de champ)`**

Nom du champ manipulé. Ce champ peut être : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) un champ Tableur d'une fenêtre, 

- un champ Tableur d'un état.  

- un champ Editeur de diagrammes d'une page ou d'une fenêtre.





<a name="SYNTAXE2"></a>

### Modifier la visibilité du quadrillage dans un champ Tableur

`<Champ manipulé>.QuadrillageVisible = <Visibilité>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ manipulé. Ce champ peut être : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) un champ Tableur d'une fenêtre, 

- un champ Tableur d'un état.  

- un champ Editeur de diagrammes d'une page ou d'une fenêtre.




**`<Visibilité> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le quadrillage du champ Tableur doit être visible, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.

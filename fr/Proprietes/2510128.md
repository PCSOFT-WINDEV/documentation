


## ValeurInitiale (Propriété)

***En anglais : InitialValue***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **ValeurInitiale** permet de connaître la valeur initiale d'un champ d'une fenêtre ou d'une page. La valeur initiale correspond à la valeur initiale définie sous l'éditeur (onglet "Contenu" de la fenêtre de description du champ).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Initialiser la valeur du champ "SAI_Client" avec sa valeur d'origine
SAI_Client = SAI_Client.ValeurInitiale
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Valeur initiale> = <Champ manipulé>.ValeurInitiale`
---

**`<Valeur initiale> : (Tout type)`**

Valeur initiale du champ spécifié.

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 

### Champ ActiveX
<a name="champ_activex_ELTPARAGRAPHE000077"></a>Pour un champ ActiveX, la valeur initiale correspond à l'identifiant COM (ProgId) de l'ActiveX sélectionné lors de la création de la fenêtre.
<a name="NOTE0_2"></a>
<a name="NOTE0_3"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 

### Champ Graphe
<a name="champ_graphe_ELTPARAGRAPHE000101"></a>Pour un champ Graphe, la valeur initiale correspond à l'image affichée dans le fond du champ, sélectionnée lors de la création de la fenêtre / page.
<a name="NOTE0_4"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) 

### Champ Tiroir
<a name="champ_tiroir_ELTPARAGRAPHE000110"></a>Pour un champ Tiroir, la propriété **ValeurInitiale** correspond à l'état du champ (déroulé (1) ou enroulé (0)) sélectionné lors de la création du champ.
<a name="NOTE0_5"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) 

### Champ Combo
<a name="champ_combo_ELTPARAGRAPHE000122"></a>Pour un champ Combo, la propriété **ValeurInitiale** permet de relire la chaîne initiale affecté à l'élément.

Par exemple, si le code suivant est utilisé pour ajouter un élément : 

```wl
ListeAjoute(COMBO_Combo1, gImage(nNumImage,i) + "Client.NomClient" + i + ...
	gValeurMémorisée("Client.NomClient" + i))
```
le code suivant permet de relire la chaîne d'origine : 

```wl
COMBO_Combo1[1].ValeurInitiale
```


Remarque : Le résultat de la propriété [ValeurAffichée](../Proprietes/2510129.md) ne permet pas de récupérer les informations sur le contenu ajouté avec la fonction [gValeurMémorisée](../WDLang1/1410088363.md). 
<a name="NOTE0_6"></a>


### Limites
<a name="limites_ELTPARAGRAPHE000147"></a>![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Dans une fenêtre, la propriété **ValeurInitiale** ne peut pas être utilisée sur :

- une fenêtre,

- un champ Libellé,

- un champ Bouton,

- un champ Forme,

- une option de sélecteur ou d'interrupteur,

- une ligne de liste (y compris Liste image) ou de table,

- une cellule ou une colonne de table,

- un menu,

- un volet d'onglet,

- un groupe de champs,

- un élément d'un champ Arbre.


![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) La propriété **ValeurInitiale** s'applique uniquement aux éléments suivants :

- Champ de saisie.

- Champ Image.

- Champ Interrupteur.

- Champ Sélecteur.

- Champ Liste.

- Champ Combo.

- Champ Table.

- Superchamp.

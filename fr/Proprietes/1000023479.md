
## NuméroPageAffichée (Propriété)

***En anglais : NumberDisplayedPage***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **NuméroPageAffichée** permet de connaître et de modifier le numéro de la page actuellement affichée dans le champ Lecteur PDF ou dans le champ Traitement de texte. Cette page est affichée à plus de 50% dans le champ. 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affiche le numéro de la page affichée dans un champ
LIB_pages = "Page " + PDF_LECTEUR_MonPDF.NuméroPageAffichée
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître le numéro de la page affichée

`<Résultat> = <Champ manipulé>.NuméroPageAffichée`
---

**`<Résultat> : (Entier)`**

Numéro de la page actuellement affichée dans le champ spécifié. Cette page est affichée à plus de 50%.  

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler : 

- Champ Lecteur PDF, 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Champ Traitement de texte. 





<a name="SYNTAXE2"></a>

### Modifier le numéro de la page affichée

`<Champ manipulé>.NuméroPageAffichée = <Page à afficher>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler : 

- Champ Lecteur PDF, 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Champ Traitement de texte. 




**`<Page à afficher> : (Entier)`**

Numéro de la page à afficher dans le champ spécifié. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 

### Spécificités pour le champ Traitement de texte
<a name="specificites_pour_champ_traitement_texte_ELTPARAGRAPHE000064"></a>

- Dans un champ Traitement de texte, le numéro de la page affichée peut être différent du numéro de la page en saisie (par exemple, saisie dans la page N+1 alors que la page N est affichée à 70% dans le champ). 

- Lors de la modification de la page affichée avec la propriété **NuméroPageAffichée**, la position du curseur de saisie n'est pas modifiée dans le champ.

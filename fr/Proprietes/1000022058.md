
## UUID (Propriété)

***En anglais : UUID***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **UUID** permet de connaître l'identifiant de l'élément. Cette propriété est utilisable sur :

- une variable de type [btleCaractéristique](../WDLang3/1000022060.md).

- une variable de type [btleDescripteur](../WDLang3/1000022064.md). 

- une variable de type [btleService](../WDLang3/1000022057.md).

- une variable de type [beaconGroupe](../WDLang3/1000023105.md).

- une variable de type [beaconInfoDétection](../WDLang3/1000023109.md).

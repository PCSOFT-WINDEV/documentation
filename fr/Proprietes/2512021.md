


## CxLibelléDécrit (Propriété)

***En anglais : CtDescribedCaption***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **CxLibelléDécrit** permet de gérer le libellé de la connexion à la source de données OLE DB. Il est possible de :

- Connaître le libellé de la connexion à la source de données OLE DB. Cette source de données a été décrite sous l'éditeur d'analyses ou lors de la description par programmation du fichier de données.

- Définir le libellé de la connexion OLE DB (lors de la description par programmation du fichier de données uniquement).



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connaître le libellé de la connexion à la source de données OLE DB
ResLibelléSourceOLEDB = Client.CxLibelléDécrit
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le libellé de la connexion

`<Libellé> = <Fichier de données>.CxLibelléDécrit`
---

**`<Libellé> : (Chaîne de caractères)`**

Libellé de la connexion.

**`<Fichier de données> : (Chaîne de caractères)`**

Nom du fichier de données manipulé. Ce nom a été défini sous l'éditeur d'analyses ou grâce au type [Description de Fichier](../WDLang4/1514065.md).  


<a name="SYNTAXE2"></a>

### Définir le libellé de la connexion

`<Fichier de données>.CxLibelléDécrit = <Libellé>`
---

**`<Fichier de données> : (Chaîne de caractères)`**

Nom du fichier de données manipulé. Ce nom vient d'être défini grâce au type [Description de Fichier](../WDLang4/1514065.md).

**`<Libellé> : (Chaîne de caractères)`**

Libellé de la connexion.




## PageSource (Propriété)

***En anglais : SourcePage***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **PageSource** permet de connaître la source d'une page interne (c'est-à-dire la page interne affichée par le champ de type Page Interne).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
Trace(PI_ChampPageInterne1.PageSource) // Affiche PI_PageInterne2
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Page interne utilisée> = <Champ Page Interne>.PageSource`
---

**`<Page interne utilisée> : (Chaîne de caractères)`**

Nom de la page interne utilisée par le champ de type "Page interne".

**`<Champ Page Interne> : (Nom de champ)`**

Nom du champ Page interne.




## PointDAttacheArrivée (Propriété)

***En anglais : AttachPointEnd***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **PointDAttacheArrivée** permet de connaître et de modifier la position du point d'attache d'arrivée d'un champ Fil.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
IdPointAttacheDépart est un entier
IdPointAttacheArrivée est un entier

FIL_Fil_de_liaison est un Champ

FIL_Fil_de_liaison <- ChampCrée("Liaison", typFil)


FIL_Fil_de_liaison.ChampDépart = IMG_Depart
FIL_Fil_de_liaison.ChampArrivée = IMG_Arrivee

IdPointAttacheDépart = PointDAttacheAjoute(IMG_Depart, 100, 25)
IdPointAttacheArrivée = PointDAttacheAjoute(IMG_Arrivee, 0, 75)

FIL_Fil_de_liaison.PointDAttacheDépart = IdPointAttacheDépart
FIL_Fil_de_liaison.PointDAttacheArrivée = IdPointAttacheArrivée
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître la position du point d'attache d'arrivée d'un champ Fil

`<Résultat> = <Champ Fil>.PointDAttacheArrivée`
---

**`<Résultat> : (Entier ou constante de type Entier)`**



- Identifiant du point d'attache si le point d'attache a été personnalisé. Cet identifiant correspond à la valeur renvoyée par la fonction [PointDAttacheAjoute](../WDLang1/1000021902.md). 

- Constante correspondant à la position du point d'attache (si cette position n'a pas été personnalisée). 
	


|   |   |
| --- | --- |
| *ptAutomatique*<br>(Valeur par défaut) | WINDEV choisit automatiquement le meilleur point d'attache. |
| *ptBasCentre* | Le point d'attache est situé au centre du bord bas du champ. |
| *ptHautCentre* | Le point d'attache est situé au centre du bord haut du champ. |
| *ptMilieuDroite* | Le point d'attache est situé au milieu du bord droit du champ. |
| *ptMilieuGauche* | Le point d'attache est situé au milieu du bord gauche du champ. |






**`<Champ Fil> : (Nom de champ)`**

Nom du champ Fil à manipuler. 


<a name="SYNTAXE2"></a>

### Modifier la position du point d'attache d'arrivée d'un champ Fil

`<Champ Fil>.PointDAttacheArrivée = <Nouvelle valeur>`
---

**`<Champ Fil> : (Nom de champ)`**

Nom du champ Fil à manipuler.

**`<Nouvelle valeur> : (Entier ou constante de type Entier)`**

Cette valeur peut correspondre :

- à l'identifiant du point d'attache renvoyé par la fonction [PointDAttacheAjoute](../WDLang1/1000021902.md).

- à une des constantes suivantes : 
	


|   |   |
| --- | --- |
| *ptAutomatique*<br>(Valeur par défaut) | WINDEV choisit automatiquement le meilleur point d'attache. |
| *ptBasCentre* | Le point d'attache est situé au centre du bord bas du champ. |
| *ptHautCentre* | Le point d'attache est situé au centre du bord haut du champ. |
| *ptMilieuDroite* | Le point d'attache est situé au milieu du bord droit du champ. |
| *ptMilieuGauche* | Le point d'attache est situé au milieu du bord gauche du champ. |








<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Le point d'attache de départ peut être personnalisé par la propriété [PointDAttacheDépart](../Proprietes/1000021907.md). 

- Si les points d'attache par défaut ont été supprimés avec la fonction [PointDAttacheSupprimeTout](../WDLang1/1000021903.md), les constantes de position ne peuvent plus être utilisées.




## CodeJS (Propriété)

***En anglais : CodeJS***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **CodeJS** permet de lire et modifier le code Javascript associé à un champ Composant Web.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
Trace(CMP_MonComposantWeb.CodeJS)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître le code Javascript associé au champ Composant Web

`<Résultat> = <Champ Composant Web>.CodeJS`
---

**`<Résultat> : (Chaîne de caractères)`**

Code Javascript associé au champ Composant Web. Ce code Javascript sera écrit dans la page. 

**`<Champ Composant Web> : (Nom de champ)`**

Nom du champ Composant Web à manipuler. 


<a name="SYNTAXE2"></a>

### Modifier le code Javascript associé au champ Composant Web

`<Champ Composant Web>.CodeJS = <Code Javascript>`
---

**`<Champ Composant Web> : (Nom de champ)`**

Nom du champ Composant Web à manipuler. 

**`<Code Javascript> : (Chaîne de caractères)`**

Nouveau code Javascript associé au champ.


## Précision (Propriété)

***En anglais : Accuracy***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Précision** permet de manipuler la précision d'un élément. Cette propriété est utilisable sur : 

- une variable de type [géoPosition](../WDLang3/1000019191.md). 

- une variable de type [ocrOption](../WDLang6/1000025768.md).

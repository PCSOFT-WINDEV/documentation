
## Défaut (Propriété)

***En anglais : Default***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Défaut** permet de gérer les valeurs par défaut. Cette propriété est utilisable sur :

- une procédure. Pour plus de détails, consultez [Défaut](../Proprietes/1000017046.md).

- une variable de type [gglNotification](../WDLang5/1000026200.md).


## Attache (Propriété)

***En anglais : Attach***
	

<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Attache** permet de connaître et de modifier les pièces jointes d'un email. 

Cette propriété est utilisable sur :

- la structure [Email](../WDLang3/3032029.md)

- une variable de type [Email](../WDLang3/1000018713.md).

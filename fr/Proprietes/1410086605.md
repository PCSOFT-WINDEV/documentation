
## Niveau

***En anglais : Level***
	

<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Niveau** permet de connaître et de modifier le niveau d'une numérotation. Cette propriété est utilisable sur :

- une variable de type [docParagraphe](../WDLang1/1000022483.md), 

- une variable de type [docNumérotation](../WDLang1/1000022794.md).




## Jeton (Propriété)

***En anglais : Token***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Jeton** permet de manipuler les différents jetons associés à un champ de saisie de type "Texte avec jetons".


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
J est un Jeton
J.Valeur = "Mon jeton"
SaisieAjouteJeton(SAI_Email, J)

// équivalences :  
// SAI_Email.AjouteJeton(J)
// Ajoute(SAI_Email.Jeton[J])

Info(SAI_Email.Jeton[1].Valeur)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître la valeur d'un jeton

`<Résultat> = <Champ de saisie>.Jeton[<Indice du jeton>]`
---

**`<Résultat> : (Chaîne de caractères)`**

Valeur du jeton. 

**`<Champ de saisie> : (Nom de champ)`**

Nom du champ de saisie de type "Texte avec jetons". 

**`<Indice du jeton> : (Entier)`**

Indice du jeton à manipuler. Les indices correspondent à l'ordre d'affichage actuel des jetons (et non l'indice initial d'ajout ou d'insertion du jeton).

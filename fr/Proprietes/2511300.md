
## Source (Propriété)

***En anglais : Source***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Source** permet de connaître et de modifier la source d'un élément. Cette propriété est utilisable sur :

- un champ Graphe ou une connexion. Pour plus de détails, consultez [Propriété Source sur un graphe ou une connexion](../Proprietes/2510147.md).

- une variable de type [hDescriptionSauvegarde](../WDLang4/1000017455.md). 

- une variable de type [rssEntrée](../WDLang5/1000017792.md). 

- la structure [Email](../WDLang3/3032029.md)

- une variable de type [Email](../WDLang3/1000018713.md). 

- une variable de type [OpenIDIdentité](../WDLang3/1000023567.md).

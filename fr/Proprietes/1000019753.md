
## DroiteAGauche (Propriété)

***En anglais : RightToLeft***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **DroiteAGauche** permet de connaître et de modifier le sens d'écriture d'un élément. Cette propriété est utilisable sur :

- un champ, une fenêtre ou un état. Pour plus de détails, consultez l'aide de la propriété [DroiteAGauche](../Proprietes/1000017228.md). 

- une variable de type [xlsDocument](../WDLang5/1000017464.md).

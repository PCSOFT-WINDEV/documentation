


## Cochée (Propriété)

***En anglais : Checked***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Cochée** permet de connaître et modifier l'état de la coche d'un élément. 

Cette propriété peut être utilisée : 

- sur les options de menus principaux et sur les options de menus contextuels. Dans ce cas, il est possible de savoir si la coche est affichée ou non devant l'option. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) sur les lignes des champs Arbre. Dans ce cas, il est possible de savoir si la coche est affichée, non affichée ou dans un état indéterminé. 



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Clic sur l'option "Afficher les Totaux"
SI OPT_OptionAfficher.Cochée ALORS
	OPT_OptionAfficher.Cochée = Faux
	LIB_TOTAL.Visible = Faux
SINON
	OPT_OptionAfficher.Cochée = Vrai
	LIB_TOTAL.Visible = Vrai
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître l'état de la coche d'un élément

`<Résultat> = <Elément à manipuler>.Cochée`
---

**`<Résultat> : (Booléen ou entier)`**



- Pour une option de menu : 

	- <u><u><u><u>Vrai</u></u></u></u> si l'option de menu possède une marque, 

	- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Pour une ligne d'un champ Arbre : 

	- 1 si l'option est cochée

	- 0 si l'option est décochée

	- -1 si l'option est dans un état indéterminé.







**`<Elément à manipuler> : (Chaîne de caractères)`**

Nom de l'élément à manipuler : 

- Nom de l'option de menu. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Ligne du champ Arbre spécifiée avec la notation suivante : &lt;Champ Arbre&gt;[Indice de la ligne]. 





<a name="SYNTAXE2"></a>

### Modifier l'état de la coche d'un élément

`<Elément à manipuler>.Cochée = <Affichée>`
---

**`<Elément à manipuler> : (Chaîne de caractères)`**

Nom de l'élément à manipuler : 

- Nom de l'option de menu. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Ligne du champ Arbre spécifiée avec la notation suivante : &lt;Champ Arbre&gt;[Indice de la ligne]. 




**`<Affichée> : (Booléen ou entier)`**



- Pour une option de menu : 

	- <u><u><u><u>Vrai</u></u></u></u> si l'option de menu possède une marque, 

	- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Pour une ligne d'un champ Arbre : 

	- 1 si l'option est cochée

	- 0 si l'option est décochée

	- -1 si l'option est dans un état indéterminé.

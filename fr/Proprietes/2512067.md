


## Numérique (Propriété)

***En anglais : Numerical***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Numérique** permet de savoir si une rubrique est de type numérique. **Cette propriété est disponible en lecture seulement**.

**Remarque** : Les rubriques considérées comme des rubriques de type numérique sont les suivantes :

- entier,

- réel,

- monétaire, 

- numérique,

- booléen.





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// La rubrique Client.Téléphone est-elle de type numérique ?
ResNum = Client.Téléphone.Numérique
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Fichier de données>.<Rubrique>.Numérique`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la rubrique est numérique, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Fichier de données> : (Chaîne de caractères)`**

Nom logique du fichier de données manipulé.

**`<Rubrique> : (Chaîne de caractères)`**

Nom de la rubrique manipulée.

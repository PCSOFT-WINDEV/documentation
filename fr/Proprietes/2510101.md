


## PoliceLarge (Propriété)

***En anglais : FontLarge***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **PoliceLarge** permet de gérer l'espacement des caractères de la police. Il est possible de :

- Savoir si les caractères du texte affiché sont élargis ou non.

- Élargir ou non les caractères du texte affiché.




Les différents types d'espacement entre les caractères gérés en WLangage sont :

- Normal : l'espacement entre les caractères est normal.

- Condensé : l'espacement entre les caractères est diminué (propriété [PoliceCondensée](../Proprietes/2510096.md)).

- Étendu : l'espacement entre les caractères est augmenté (propriété [PoliceEtendue](../Proprietes/2510104.md)).

- Large : l'espacement entre les caractères est augmenté au maximum (propriété **PoliceLarge**).





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Élargir les caractères du texte affiché dans le champ "LIB_ChampLibellé"
LIB_ChampLibellé.PoliceLarge = Vrai
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Savoir si les caractères du texte affiché dans un champ sont élargis

`<Résultat> = <Champ manipulé>.PoliceLarge`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si les caractères du texte affiché sont élargis, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Champ manipulé> : (Nom de champ)`**

Nom du champ (champ Libellé ou du champ Calculé) à manipuler.


<a name="SYNTAXE2"></a>

### Élargir les caractères du texte affiché dans un champ 

`<Champ manipulé>.PoliceLarge = <Larges/Non larges>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ (champ Libellé ou du champ Calculé) à manipuler.

**`<Larges/Non larges> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si les caractères du texte affiché doivent être élargis, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Variable de type Police
<a name="variable_type_police_ELTPARAGRAPHE000081"></a>Si des polices identiques (même taille, même attribut, ...) doivent être appliquées sur plusieurs champs, il est conseillé d'utiliser une variable de type Police.

Une police peut être créée à partir d'une variable de type [Police](../Motscles/1514045.md).

Pour définir les caractéristiques de la police, utilisez :

- soit la fonction [PoliceCrée](../WDLang1/3060002.md),

- soit les propriétés [Angle](../Proprietes/1410086632.md), [Barré](../Proprietes/2513005.md), [Couleur](../Proprietes/2510071.md), [Etendu](../Proprietes/2513002.md), [Gras](../Proprietes/2513001.md), [Italique](../Proprietes/2513007.md), [Large](../Proprietes/2513012.md), [Nom](../Proprietes/2510082.md), [Souligné](../Proprietes/2513013.md) et [Taille](../Proprietes/2510125.md).




Cette police pourra être utilisée :

- Pour écrire du texte dans des dessins (fonction [dPolice](../WDLang1/3029020.md)).

- Pour écrire dans les graphes (fonctions [grPoliceEtiquette](../WDLang3/3042012.md), [grPoliceLégende](../WDLang3/3042037.md), [grPoliceTitre](../WDLang3/3042048.md) et [grPoliceTitreAxe](../WDLang3/3042059.md)).

- Pour imprimer (fonction [iPolice](../WDLang5/3046014.md)).

- Dans les champs d'une fenêtre (propriété [Police](../Proprietes/2510095.md) des champs d'une fenêtre).

- Dans les champs d'un état (propriété [Police](../Proprietes/2510095.md) des champs d'un état).



<a name="NOTE0_2"></a>


### Police utilisée par les fonctions d'impression
<a name="police_utilisee_par_les_fonctions_impression_ELTPARAGRAPHE000163"></a>Pour créer une police utilisable par les [fonctions d'impression](../WDLang5/3046065.md) ([iImprimeBloc](../WDLang5/3046030.md), [iImprimeMot](../WDLang5/3046034.md), ...), utilisez :

- soit la fonction [iCréePolice](../WDLang5/3046045.md).

- soit une variable de type [Police](../Motscles/1514045.md).



<a name="NOTE0_3"></a>


### Limites
<a name="limites_ELTPARAGRAPHE000188"></a>Dans un état, la propriété **PoliceLarge** peut être utilisée uniquement sur :

- un champ Libellé.

- un champ Lien URL.

- un champ Calculé.

- un champ prédéfini.




Dans une fenêtre, la propriété **PoliceLarge** ne peut pas être utilisée sur :

- un champ Spin,

- un champ Web Camera,

- un champ Conférence.


Remarque : Cette propriété s'applique sur les champs "Liste image" en mode d'affichage "Liste classique" uniquement.

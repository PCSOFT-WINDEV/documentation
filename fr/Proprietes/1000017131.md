


## EffacementAutomatique (Propriété)

***En anglais : AutomaticErase***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **EffacementAutomatique** permet de :

- savoir si un champ de saisie est en mode "Effacement automatique".

- modifier la caractéristique "Effacement automatique" d'un champ de saisie.




**Remarque** : Si un champ de saisie est en mode "Effacement automatique", la valeur présente dans le champ est automatiquement sélectionnée lors de l'entrée dans le champ. Cette caractéristique est paramétrable dans l'onglet "Détail" de la fenêtre de description du champ.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Désactive temporairement l'effacement automatique 
// pour ne pas tout sélectionner lors de l'appel à la fonction DonneFocus

// Mémorise l'effacement automatique actuel
EffacementAuto est un booléen 
EffacementAuto = SAI_Saisie1.EffacementAutomatique
SAI_Saisie1.EffacementAutomatique = Faux

// Redonne le focus au champ de saisie
DonneFocus(SAI_Saisie1) 

// Restaure l'effacement automatique
SAI_Saisie1.EffacementAutomatique = Vrai
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le mode "Effacement automatique"

`<Mode actuel> = <Champ manipulé>.EffacementAutomatique`
---

**`<Mode actuel> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le mode "Effacement automatique" est activé, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Champ manipulé> : (Nom de champ)`**

Nom de l'élément manipulé : 

- Champ de saisie, 

- Colonne de table de type Texte.  





<a name="SYNTAXE2"></a>

### Modifier le mode "Effacement automatique"

`<Champ à manipuler>.EffacementAutomatique = <Nouveau mode>`
---

**`<Champ à manipuler> : (Nom de champ)`**

Nom de l'élément manipulé : 

- Champ de saisie, 

- Colonne de table de type Texte.




**`<Nouveau mode> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> pour activer le mode "Effacement automatique", 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.

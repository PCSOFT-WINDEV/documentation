
## MotDePasseCléPrivée (Propriété)

***En anglais : PrivateKeyPassword***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **MotDePasseCléPrivée** permet de gérer le mot de passe du fichier de clés (dans le cas d'une connexion par clé privée). Cette propriété est utilisée : 

- par les variables de type [sshSession](../WDLang3/1000021157.md). 

- par les variables de type [mqttSession](../WDLang3/1000023023.md).

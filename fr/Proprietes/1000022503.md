
## PoliceTaille (Propriété)

***En anglais : FontSize***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **PoliceTaille** permet de gérer la taille de la police utilisée pour un élément. Cette propriété est utilisable sur : 

- les champs. Pour plus de détails, consultez [PoliceTaille](../Proprietes/2510072.md). 

- une variable de type [docMiseEnForme](../WDLang1/1000022817.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [pdfElémentTexte](../WDLang6/1000024556.md).

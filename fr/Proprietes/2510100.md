


## PoliceNom (Propriété)

***En anglais : FontName***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **PoliceNom** permet de :

- Connaître la police utilisée dans un champ.

- Modifier la police utilisée dans un champ.





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Modifier la police utilisée dans le champ "SAI_NomClient"
SAI_NomClient.PoliceNom = "Arial"
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître la police utilisée dans un champ

`<Résultat> = <Champ manipulé>.PoliceNom`
---

**`<Résultat> : (Chaîne de caractères)`**

Nom de la police utilisée dans le champ spécifié.

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler.  


<a name="SYNTAXE2"></a>

### Modifier la police utilisée dans un champ

`<Champ manipulé>.PoliceNom = <Nouvelle police>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler.  

**`<Nouvelle police> : (Chaîne de caractères)`**

Nouvelle police utilisée dans le champ spécifié.

## Remarques


### Limites
<a name="limites_ELTPARAGRAPHE000084"></a>![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Dans un état, la propriété **PoliceNom** peut être utilisée uniquement sur :

- un champ Lien URL.

- un champ Libellé.

- un champ Calculé.

- un champ Prédéfini.


![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Dans une fenêtre, la propriété **PoliceNom** ne peut pas être utilisée sur :

- un champ Spin,

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) un champ Web Camera,

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) un champ Conférence.


Remarque : Cette propriété s'applique sur les champs "Liste image" en mode d'affichage "Liste classique" uniquement.![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) La propriété **PoliceNom** s'applique uniquement aux éléments suivants :

- Champ Bouton.

- Champ Libellé.

- Champ Combo.

- Champ Arbre.

- Colonne d'un champ Table.

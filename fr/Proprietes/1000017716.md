


## LongueurMinimale (Propriété)

***En anglais : MinimumLength***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **LongueurMinimale** permet de paramétrer la longueur minimale des mots à indexer (cas d'un index full-text). Il est ainsi possible de :

- Connaître la longueur minimale des mots à indexer pour un index full-text (défini sous l'éditeur d'analyses ou défini par programmation).

- Définir la longueur minimale des mots à indexer pour un index full-text (lors de la description de l'index full-text par programmation).





<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître la longueur minimale des mots à indexer

`<Longueur> = <Fichier de données>.<Index full-text>.LongueurMinimale`
---

**`<Longueur> : (Entier)`**



- Longueur minimale des mots à indexer, 

- 0 pour indexer tous les mots sans taille minimale.




**`<Fichier de données> : (Nom de champ)`**

Nom du fichier de données manipulé. Ce nom a été défini sous l'éditeur d'analyses ou grâce au type [Description de Fichier](../WDLang4/1514065.md).

**`<Index full-text> : (Chaîne de caractères)`**

Nom de l'index full-text manipulé. Ce nom a été défini sous l'éditeur d'analyses ou grâce au type [Description d'IndexFullText](../WDLang4/1000017461.md).  


<a name="SYNTAXE2"></a>

### Définir la longueur minimale des mots à indexer

`<Index full-text>.LongueurMinimale = <Nouvelle longueur>`
---

**`<Index full-text> : (Chaîne de caractères)`**

Nom de l'index full-text manipulé. Ce nom vient d'être défini grâce au type [Description d'IndexFullText](../WDLang4/1000017461.md).

**`<Nouvelle longueur> : (Entier)`**



- Longueur minimale des mots à indexer, 

- 0 si tous les mots doivent être indexés.

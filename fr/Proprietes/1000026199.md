
## PièceJointe (Propriété)

***En anglais : Attachment***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **PièceJointe** permet de gérer les pièces jointes. Cette propriété est utilisable sur : 

- une variable de type [pdfDocument](../WDLang6/1000024909.md). 

- une variable de type [gglEvénement](../WDLang5/1000017440.md).

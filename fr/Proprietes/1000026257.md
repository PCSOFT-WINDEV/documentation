
## PageInternePopup (Propriété)

***En anglais : InternalPagePopup***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **PageInternePopup** permet de connaître et de modifier le nom de la page interne utilisée pour personnaliser le contenu de la popup affichée pour les marqueurs du champ Carte. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
CARTE_Carte.PageInternePopup = "PI_Popup"
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître la page interne utilisée pour personnaliser la popup affichée pour les marqueurs du champ Carte

`<Résultat> = <Champ Carte>.PageInternePopup`
---

**`<Résultat> : (Chaîne de caractères)`**



- Nom de la page interne utilisée pour la popup,

- Chaîne vide ("") si la popup utilise un affichage par défaut.




**`<Champ Carte> : (Nom de champ)`**

Nom du champ Carte à manipuler.   


<a name="SYNTAXE2"></a>

### Modifier la page interne utilisée pour personnaliser la popup affichée pour les marqueurs du champ Carte

`<Champ Carte>.PageInternePopup = <Nouvelle page interne>`
---

**`<Champ Carte> : (Nom de champ)`**

Nom du champ Carte à manipuler.

**`<Nouvelle page interne> : (Nom de page)`**



- Nom de la page interne utilisée pour la popup,

- Chaîne vide ("") si la popup utilise un affichage par défaut.










<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Si un nom de page interne a été renseigné pour la propriété **PageInternePopup**, la page interne sera chargée à chaque affichage d'une popup dans le champ Carte. La variable [Marqueur](../WDLang3/1000019940.md) correspondant à la popup affichée est passée à l'événement "Déclarations globales" de la page interne à ouvrir. Ce paramètre est passé par référence et il est considéré comme une variable globale à la page.

- La page interne spécifiée est ouverte en mode "affichage uniquement" : aucune interaction avec l'utilisateur n'est possible dans les champs de la page interne.

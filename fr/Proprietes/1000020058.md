
## MémoriseFiltreFAA (Propriété)

***En anglais : StoreFilterAAF***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **MémoriseFiltreFAA** permet de savoir et d'indiquer si les filtres posés par l'utilisateur sur un champ Table ou Table hiérarchique sont conservés entre 2 lancements de l'application.

Ce filtre a été mis en place grâce aux boutons de filtre présents dans les colonnes de la table (clic droit sur la loupe ou l'entonnoir). 

Si le filtre est conservé, à la prochaine ouverture de la fenêtre contenant la table, un message furtif sera affiché indiquant à l'utilisateur le filtre actuellement mis en place, puis le filtre sera automatiquement réappliqué. 
<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=MemoriseFiltreFAA.gif)


**Remarque** : Cette propriété est identique à l'option "Mémoriser les filtres utilisateur" dans l'onglet "Détail" de la fenêtre de description du champ Table. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Modification de INT_MémoriseFiltre
SI INT_MémoriseFiltre = Vrai ALORS
	TABLE_PRIX.MémoriseFiltreFAA = Vrai
SINON
	TABLE_PRIX.MémoriseFiltreFAA = Faux
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Savoir si les options de filtre sont conservées

`<Résultat> = <Champ Table>.MémoriseFiltreFAA`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si les options de filtre définis par l'utilisateur sur le champ Table sont conservées. 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire (valeur par défaut). 




**`<Champ Table> : (Nom de champ)`**

Nom du champ Table ou Table hiérarchique à manipuler. 


<a name="SYNTAXE2"></a>

### Modifier la gestion des options de filtre

`<Champ Table>.MémoriseFiltreFAA = <Mémorisation>`
---

**`<Champ Table> : (Nom de champ)`**

Nom du champ Table ou Table hiérarchique à manipuler. 

**`<Mémorisation> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> pour conserver les options de filtre définis par l'utilisateur sur le champ Table entre 2 lancements de l'application. 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.

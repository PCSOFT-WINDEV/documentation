
## Visible (Propriété)

***En anglais : Visible***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Visible** permet de gérer la visibilité d'un élément. Cette propriété est utilisable sur : 

- les champs d'une fenêtre, d'une page ou d'un état. Pour plus de détails, consultez [Visible](../Proprietes/2510138.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [picCalque](../WDLang1/1000024604.md).

- une variable de type [kbCarte](../WDLang1/1410089167.md).




## PoliceContenuRendezVous (Propriété)

***En anglais : FontAppointmentContent***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **PoliceContenuRendezVous** permet de :

- Connaître les caractéristiques de la police utilisée pour le contenu des rendez-vous dans les **états** de type Planning et Agenda. 

- Modifier les caractéristiques de la police utilisée pour le contenu des rendez-vous dans les **états** de type Planning et Agenda.





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Caractéristiques de la police utilisée pour le champ PLN_Planning dans l'état ETAT_Planning
MaPolice est une Police

MaPolice = PLN_PLANNING.PoliceContenuRendezVous
Trace(MaPolice.Nom)
Trace(MaPolice.Gras)
Trace(MaPolice.Souligné)
Trace(MaPolice.Taille)
Trace(MaPolice.Couleur)
```
<a name="Exemple2"></a>

```wl
// Définition de polices 
MaPolice est une Police
MaPolice.Nom = "Comic Sans MS"
MaPolice.Taille = 18
MaPolice.Couleur = BleuPastel

MaPolice2 est une Police
MaPolice2.Nom = "Courier new"
MaPolice2.Taille = 14
MaPolice2.Couleur = VertPastel

// Modification des polices utilisées dans l'état
PLN_PLANNING.PoliceTitreRendezVous = MaPolice
PLN_PLANNING.PoliceContenuRendezVous = MaPolice2
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître la police utilisée pour le contenu des rendez-vous (Etat Planning ou Agenda)

`<Résultat> = <Champ manipulé>.PoliceContenuRendezVous`
---

**`<Résultat> : (Police)`**

Variable de type [Police](../Motscles/1514045.md) contenant les caractéristiques de la police utilisée pour le contenu des rendez-vous. 

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ Planning ou Agenda présent dans un état Planning ou Agenda.  


<a name="SYNTAXE2"></a>

### Modifier la police utilisée pour le contenu des rendez-vous (Etat Planning ou Agenda)

`<Champ manipulé>.PoliceContenuRendezVous = <Nouvelle police>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ Planning ou Agenda présent dans un état Planning ou Agenda.

**`<Nouvelle police> : (Police)`**

Variable de type [Police](../Motscles/1514045.md) contenant les caractéristiques de la police à utiliser pour le contenu des rendez-vous.   



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
La propriété **PoliceContenuRendezVous** s'applique uniquement aux champs Agenda et Planning présents dans des états.

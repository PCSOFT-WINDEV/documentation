


## EffetVisuel (Propriété)

***En anglais : VisualEffect***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **EffetVisuel** permet de gérer un effet visuel sur un champ ou un groupe de champs d'une fenêtre. Il est possible de : 

- Connaître l'effet visuel utilisé pour un champ ou un groupe de champs. 

- Modifier l'effet visuel utilisé pour un champ ou un groupe de champs. 



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Modification de l'effet visuel utilisé
GR_MonGroupe.EffetVisuel = evVoileBleu
JAUGE_Silver.EffetVisuel = evAucun
BTN_BoutonOK.EffetVisuel = evGfi
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître l'effet visuel utilisé

`<Résultat> = <Champ manipulé>.EffetVisuel`
---

**`<Résultat> : (Constante de type Entier)`**

Type d'effet visuel actuellement utilisé par le champ : 


|   |   |
| --- | --- |
| *evAucun* | Aucun effet visuel spécifique n'est appliqué au champ ou au groupe de champs. L'affichage correspond à un affichage standard. |
| *evGfi* | Le champ ou le groupe de champs est affiché en grisé : l'effet visuel est identique à celui utilisé par le GFI (Grisage des Fenêtres Inaccessibles). |
| *evVoileBleu* | Un voile bleuté est affiché sur le champ ou le groupe de champs. |



**`<Champ manipulé> : (Nom de champ)`**

Nom du champ ou du groupe de champs à manipuler.


<a name="SYNTAXE2"></a>

### Modifier l'effet visuel utilisé

`<Champ manipulé>.EffetVisuel = <Nouvel effet visuel>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ ou du groupe de champs à manipuler. 

**`<Nouvel effet visuel> : (Constante de type Entier)`**

Nouvel effet visuel à appliquer au champ : 


|   |   |
| --- | --- |
| *evAucun* | Aucun effet visuel spécifique n'est appliqué au champ ou au groupe de champs. L'affichage correspond à un affichage standard. |
| *evGfi* | Le champ ou le groupe de champs est affiché en grisé : l'effet visuel est identique à celui utilisé par le GFI (Grisage des Fenêtres Inaccessibles). |
| *evVoileBleu* | Un voile bleuté est affiché sur le champ ou le groupe de champs. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Limites
<a name="limites_ELTPARAGRAPHE000069"></a>Cette propriété n'est pas disponible sur les types de champs suivants : 

- Champ OLE,

- Champ ActiveX, 

- Champ HTML, 

- Champ WebCaméra, 

- Champ Flux,

- Champ Séparateur, 

- Champ Graphe, 

- Champ Xaml, 

- Champ Carrousel, 

- Champ Cube,

- Champ Vidéo.

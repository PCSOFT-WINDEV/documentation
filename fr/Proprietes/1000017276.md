


## Compression (Propriété)

***En anglais : Compression***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Compression** permet de gérer la compression des données lors d'une connexion (HFSQL Client/Serveur, ...). Il est possible de :

- Savoir si la connexion transmet des données compressées.

- Définir si les données transmises entre le serveur et le client doivent être compressées. Dans ce cas, si la connexion est actuellement ouverte, elle sera fermée, puis ré-ouverte avec les nouveaux paramètres lors d'un premier accès à un fichier utilisant cette connexion.




**Rappel** : La compression des données permet de réduire le volume de données transitant par le média de transmission (ce qui peut être un avantage pour des transmissions facturées aux données transmises). La compression des données est gérée à partir de la version 14 du serveur HFSQL. Pour plus de détails, consultez [Compression de trames](../WDLang4/1000017306.md).

**Remarque** : La connexion peut correspondre :

- soit à une connexion décrite sous l'éditeur d'analyses,

- soit à une connexion décrite avec les fonctions [HConnecte](../WDLang4/3044263.md), [HDécritConnexion](../WDLang4/3044205.md) ou [HOuvreConnexion](../WDLang4/3044107.md).

- soit à une connexion décrite avec une variable de type [Connexion](../WDLang4/1514073.md).









<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MaConnexion est une Connexion
// Description de la connexion
MaConnexion.Utilisateur = "USER"
MaConnexion.MotDePasse = "PASSWORD"
MaConnexion.Serveur = "MONSERVEUR" 
MaConnexion.BaseDeDonnées = "Test"
MaConnexion.Provider = hAccèsHFClientServeur
MaConnexion.Accès = hOLectureEcriture
MaConnexion.InfosEtendues = "Infos étendues"
MaConnexion.OptionsCurseur = hCurseurClient
MaConnexion.Compression = Vrai

HOuvreConnexion(MaConnexion)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Savoir si la connexion transmet des données compressées

`<Résultat> = <Connexion>.Compression`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si les données transmises sont compressées, 

- <u><u><u><u>Faux</u></u></u></u> si les données transmises ne sont pas compressées.




**Remarque** : La gestion de la compression est disponible à partir de la version 14 du serveur HFSQL.

**`<Connexion> : (Chaîne de caractères ou variable de type Connexion)`**

Nom de la connexion manipulée. Ce nom a été défini :

- soit sous l'éditeur d'analyses,

- soit avec les fonctions [HDécritConnexion](../WDLang4/3044205.md) ou [HOuvreConnexion](../WDLang4/3044107.md),

- soit avec le type [Connexion](../WDLang4/1514073.md).  





<a name="SYNTAXE2"></a>

### Définir la compression des données d'une connexion

`<Connexion>.Compression = <Compression des données>`
---

**`<Connexion> : (Chaîne de caractères ou variable de type Connexion)`**

Nom de la connexion manipulée. Ce nom a été défini :

- soit sous l'éditeur d'analyses,

- soit avec les fonctions [HDécritConnexion](../WDLang4/3044205.md) ou [HOuvreConnexion](../WDLang4/3044107.md),

- soit avec le type [Connexion](../WDLang4/1514073.md).




**`<Compression des données> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> pour compresser les données, 

- <u><u><u><u>Faux</u></u></u></u> pour ne pas compresser les données.




**Remarque** : La gestion de la compression est disponible à partir de la version 14 du serveur HFSQL.

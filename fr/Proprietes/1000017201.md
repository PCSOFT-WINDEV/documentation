


## Zoom (Propriété)

***En anglais : Zoom***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Zoom** permet de connaître ou de modifier la valeur du zoom utilisée : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) dans un champ Image. Ce champ Image doit avoir la gestion des ascenseurs automatiques et du zoom activée (option "Ascenseurs et zoom automatiques" dans l'onglet "Détail" de la fenêtre de description du champ).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) dans un champ Carte. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) dans un champ Organigramme.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) dans un champ Table, Table hiérarchique, Tableau croisé dynamique, Tableur ou Liste. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) dans une fenêtre.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) dans un champ Traitement de texte.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) dans un champ Lecteur PDF.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) dans un champ Editeur de diagrammes. 





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 
```wl
// Effectue un zoom à 40% du fichier PDF affiché dans le champ Image
IMG_Image_PDF.Zoom = 40
```

## Syntaxe

### Connaître le zoom effectué

`<Résultat> = <Elément manipulé>.Zoom`
---

**`<Résultat> : (Entier)`**

Valeur du zoom actuellement réalisé dans l'élément. Cette valeur correspond : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Dans un champ Image : au pourcentage de zoom.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Dans un champ Organigramme : au pourcentage de zoom.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Dans un champ Table, Table hiérarchique, Tableau croisé dynamique ou Tableur : au pourcentage de zoom.  

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Dans un champ Carte : à une valeur comprise entre 1 (affichage de la terre entière) et 21. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Dans une fenêtre : au pourcentage de zoom.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Dans un champ Traitement de texte : au pourcentage de zoom.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Dans un champ Lecteur PDF : au pourcentage de zoom.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Dans un champ Editeur de diagrammes : au pourcentage de zoom.




**`<Elément manipulé> : (Nom de champ ou nom de fenêtre)`**

Nom de l'élément à manipuler :  

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Nom du champ Image.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Nom du champ Organigramme. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Nom du champ Carte. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Nom du champ Table, Table hiérarchique, Tableau croisé dynamique, Tableur ou Liste. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Nom de la fenêtre.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Nom du champ Traitement de texte.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Nom du champ Lecteur PDF.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Nom du champ Editeur de diagrammes.

### Modifier le zoom effectué

`<Elément manipulé>.Zoom = <Nouveau zoom>`
---

**`<Elément manipulé> : (Nom de champ ou nom de fenêtre)`**

Nom de l'élément à manipuler :  

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Nom du champ Image.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Nom du champ Organigramme. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Nom du champ Carte. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Nom du champ Table, Table hiérarchique, Tableau croisé dynamique, Tableur ou Liste. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Nom de la fenêtre.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Nom du champ Traitement de texte.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Nom du champ Lecteur PDF.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Nom du champ Editeur de diagrammes.




**`<Nouveau zoom> : (Entier ou constante de type Entier)`**

Nouvelle valeur du zoom à appliquer à l'élément. 

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) **Pour un champ Lecteur PDF**, cette valeur peut correspondre :

- à un pourcentage.

- à une des constantes suivantes :
	


|   |   |
| --- | --- |
| *zoomAdapteHauteur* | Affiche le fichier PDF en pleine hauteur dans le champ Lecteur PDF. |
| *zoomAdapteLargeur* | Affiche le fichier PDF en pleine largeur dans le champ Lecteur PDF. |
| *zoomAdapteTaille* | Affiche le fichier PDF en entier dans le champ Lecteur PDF. |




**Remarque** : quand un nouveau fichier PDF est chargé dans le champ, la valeur du zoom n'est pas conservée. 

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) **Pour un champ Image**, cette valeur peut correspondre :

- à un pourcentage.

- à une des constantes suivantes :
	


|   |   |
| --- | --- |
| *zoomAdapteHauteur* | Affiche l'image en pleine hauteur dans le champ Image. |
| *zoomAdapteLargeur* | Affiche l'image en pleine largeur dans le champ Image. |
| *zoomAdapteTaille* | Affiche l'image en entier dans le champ Image. |




**Remarque** : quand une nouvelle image est chargée dans le champ, la valeur du zoom n'est pas conservée. 

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) **Pour un champ Carte** : 

- cette valeur est comprise entre 1 (affichage de la terre entière) et 21. 

- cette valeur peut correspondre à la constante suivante : 
	


|   |   |
| --- | --- |
| *zoomAdapteTaille* | Adapte le zoom du champ Carte pour que tous les marqueurs soient visibles en même temps. |




![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) **Pour un champ Organigramme**, cette valeur correspond à un pourcentage.  
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) **Pour un champ Table, Table hiérarchique, Tableau croisé dynamique, Tableur ou Liste** : cette valeur correspond à un pourcentage.  ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) **Pour une fenêtre**, cette valeur correspond à un pourcentage.![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) **Pour un champ Traitement de texte**, cette valeur correspond à un pourcentage.![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) **Pour un champ Editeur de diagrammes**, cette valeur correspond au pourcentage de zoom.

### Fonctions de dessin, zoom et ascenseurs automatiques
<a name="fonctions_dessin_zoom_ascenseurs_automatiques_ELTPARAGRAPHE000348"></a>L'option "Ascenseurs et zoom automatiques" disponible sur un champ Image est incompatible avec la fonction [dDébutDessin](../WDLang1/3029009.md). Si cette option est utilisée sur le champ Image, la fonction [dDébutDessin](../WDLang1/3029009.md) entraîne un affichage figé dans le champ. Le zoom est également figé. Le comportement redevient normal lors de l'appel de la fonction [dFinDessin](../WDLang1/3029019.md).

Le même effet peut être constaté avec les fonctions [dPixelCouleur](../WDLang1/3029014.md) et [dRotation](../WDLang1/3029024.md).

<a name="NOTE0_3"></a>




## Multisélection (Propriété)

***En anglais : Multiselection***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Multisélection** permet de connaître et de modifier le mode de sélection :

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) d'un champ Table.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) d'un champ Table hiérarchique.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) d'un champ Tableau croisé dynamique.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) d'un champ Liste.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) d'un champ Liste image.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) d'un champ Zone répétée.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) d'un champ Calendrier.





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Modifie le mode de sélection selon la valeur du champ Interrupteur
SI INT_Multisel = Vrai ALORS
	TABLE_Table1.Multisélection = Vrai
SINON
	TABLE_Table1.Multisélection = Faux
FIN
```

<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le mode de sélection

`<Mode de sélection> = <Champ manipulé>.Multisélection`
---

**`<Mode de sélection> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le champ est multisélections, 

- <u><u><u><u>Faux</u></u></u></u> si le champ est monosélection.




![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png) Cas des champs Table et Table hiérarchique : Seul le mode de sélection des lignes est pris en compte. Le mode de sélection des colonnes et des cellules n'est pas pris en compte.

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Champ Table.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Champ Table hiérarchique.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Champ Tableau croisé dynamique.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Champ Liste.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Champ Liste image.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Champ Zone répétée.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Champ Calendrier.





<a name="SYNTAXE2"></a>

### Modifier le mode de sélection

`<Champ manipulé>.Multisélection = <Nouveau mode de sélection>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Champ Table.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Champ Table hiérarchique.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Champ Tableau croisé dynamique.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Champ Liste.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Champ Liste image.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Champ Zone répétée.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Champ Calendrier.




**`<Nouveau mode de sélection> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le champ doit être multisélections, 

- <u><u><u><u>Faux</u></u></u></u> si le champ doit être monosélection. Si le champ passe en mode monosélection alors que plusieurs éléments sont sélectionnés, seul le premier élément sélectionné est conservé.  




![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png) Cas des champs Table et Table hiérarchique : Seul le mode de sélection des lignes est modifié. Le mode de sélection des colonnes et des cellules n'est pas modifié.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) 

### Limites pour le champ Calendrier
<a name="limites_pour_champ_calendrier_ELTPARAGRAPHE000187"></a>Pour un champ Calendrier, la propriété **Multisélection** permet uniquement de passer du mode monosélection au mode multisélection. Si le mode "Sélection de période" est sélectionné sous l'éditeur (onglet "Détail" de la fenêtre de description du champ), la propriété **Multisélection** n'a aucun effet.
<a name="NOTE0_2"></a>
<a name="NOTE0_3"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 

### Fonctionnement du champ Tableau Croisé Dynamique
<a name="fonctionnement_champ_tableau_croise_dynamique_ELTPARAGRAPHE000216"></a>Pour un champ Tableau Croisé Dynamique, la propriété **Multisélection** permet uniquement de passer du mode monosélection au mode multisélection. L'utilisateur pourra alors sélectionner plusieurs cellules juxtaposées. 

Cette propriété est équivalente à l'option "Autoriser la sélection multiple des lignes, colonnes et cellules" présente dans l'onglet "UI" de la fenêtre de description du champ. 
<a name="NOTE0_4"></a>


### Limites
<a name="limites_ELTPARAGRAPHE000232"></a>![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) La propriété **Multisélection** est sans effet :

- sur les champs Table en saisie sans sélection : elle renvoie <u><u><u><u>Faux</u></u></u></u>.

- sur les champs Table en affichage sans sélection : elle renvoie <u><u><u><u>Faux</u></u></u></u>.

- sur les champs Table "Compatibles 5.5".


La propriété **Multisélection** fonctionne pour les champs suivants :

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Champ Table.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Champ Table hiérarchique.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Champ Tableau croisé dynamique.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Champ Liste.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Champ Liste image.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Champ Zone répétée.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Champ Calendrier.

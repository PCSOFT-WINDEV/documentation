
## Ressource

***En anglais : Resource***
	

<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Ressource** permet de manipuler la ressource associée à un rendez-vous. 

Cette propriété est utilisable sur :

- un champ Planning. Pour plus de détails, consultez [Ressource](../Proprietes/1000019308.md). 

- une variable de type [RendezVous](../WDLang1/1000019244.md).


## ModeBacASable (Propriété)

***En anglais : SandboxMode***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **ModeBacASable** permet de savoir si le mode bac à sable est utilisé. Cette propriété est utilisable sur :

- une variable de type [sfInfoSession](../WDLang5/1000018416.md). 

- une variable de type [sfConnexion](../WDLang5/1000017579.md).

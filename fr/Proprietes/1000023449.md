
## ModeManipulation (Propriété)

***En anglais : UseMode***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **ModeManipulation** permet de connaître ou de modifier le mode de manipulation du champ Lecteur PDF via la souris. La souris peut permettre de sélectionner du texte dans le document ou bien de se déplacer dans les pages du document.

Par défaut, le champ Lecteur PDF est en mode sélection. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Passe le champ en mode déplacement
PDF_MonPDF.ModeManipulation = modeManipulationDéplacement
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître le mode de manipulation du champ Lecteur PDF via la souris

`<Résultat> = <Champ Lecteur PDF>.ModeManipulation`
---

**`<Résultat> : (Constante de type Entier)`**

Mode de manipulation actuel du document PDF : 


|   |   |
| --- | --- |
| *modeManipulationDéplacement* | Le champ est en mode déplacement : <br><br>- le bouton gauche de la souris permet de se déplacer dans le document. <br><br>- le curseur de la souris se change en main. <br><br>- si le ruban est affiché, l'option "Déplacement" est sélectionnée (volet "Affichage", groupe "Mode"). <br><br><br><br><br> |
| *modeManipulationOCR* | Le champ est en mode sélection : <br><br>- le bouton gauche de la souris permet de sélectionner les éléments du document. <br><br>- le curseur de la souris se change en lasso de sélection. <br><br>- si le ruban est affiché, l'option "OCR" est sélectionnée (volet "Affichage", groupe "Mode"). <br><br><br> |
| *modeManipulationSélection* | Le champ est en mode sélection : <br><br>- le bouton gauche de la souris permet de sélectionner les éléments du document. <br><br>- le curseur de la souris se change en curseur de saisie. <br><br>- si le ruban est affiché, l'option "Sélection" est sélectionnée (volet "Affichage", groupe "Mode"). <br><br><br><br><br> |



**`<Champ Lecteur PDF> : (Nom de champ)`**

Nom du champ Lecteur PDF à manipuler. 


<a name="SYNTAXE2"></a>

### Modifier le mode de manipulation du champ Lecteur PDF via la souris

`<Champ Lecteur PDF>.ModeManipulation = <Nouveau mode>`
---

**`<Champ Lecteur PDF> : (Nom de champ)`**

Nom du champ Lecteur PDF à manipuler. 

**`<Nouveau mode> : (Constante de type Entier)`**

Nouveau mode de manipulation du document PDF :

| *modeManipulationOCR* | Le champ est en mode sélection : <br><br>- le bouton gauche de la souris permet de sélectionner les éléments du document. <br><br>- le curseur de la souris se change en lasso de sélection. <br><br>- si le ruban est affiché, l'option "OCR" est sélectionnée (volet "Affichage", groupe "Mode"). <br><br><br> |
| *modeManipulationSélection* | Le champ est en mode sélection : <br><br>- le bouton gauche de la souris permet de sélectionner les éléments du document. <br><br>- le curseur de la souris se change en curseur de saisie. <br><br>- si le ruban est affiché, l'option "Sélection" est sélectionnée (volet "Affichage", groupe "Mode"). <br><br><br><br><br> |

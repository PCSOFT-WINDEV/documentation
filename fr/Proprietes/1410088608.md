
## LargeurImageInitiale (Propriété)

***En anglais : ImageInitialWidth***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **LargeurImageInitiale** permet de connaître ou de modifier la largeur par défaut de l'image créée dans le champ Editeur d'images.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
EI_MonEditeurImage.LargeurImageInitiale = 800
EI_MonEditeurImage.HauteurImageInitiale = 600
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître la largeur par défaut de l'image créée dans le champ Editeur d'images

`<Résultat> = <Champ Editeur d'images>.LargeurImageInitiale`
---

**`<Résultat> : (Entier)`**

Largeur par défaut (en pixels) d'une image créée dans le champ Editeur d'images. 

**`<Champ Editeur d'images> : (Nom de champ)`**

Nom du champ Editeur d'images manipulé.


<a name="SYNTAXE2"></a>

### Modifier la largeur par défaut de l'image créée dans le champ Editeur d'images

`<Champ Editeur d'images>.LargeurImageInitiale = <Nouvelle largeur>`
---

**`<Champ Editeur d'images> : (Nom de champ)`**

Nom du champ Editeur d'images manipulé.

**`<Nouvelle largeur> : (Entier)`**

Nouvelle largeur par défaut (en pixels) d'une image créée dans le champ Editeur d'images. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Pour modifier les caractéristiques par défaut du champ Editeur d'images, il est également possible d'utiliser : 

- la propriété [HauteurImageInitiale](../Proprietes/1410088607.md) pour définir la hauteur par défaut de l'image créée dans le champ Editeur d'images.

- la propriété [OutilInitial](../Proprietes/1410088609.md) pour définir l'outil utilisé par défaut dans le champ Editeur d'images.

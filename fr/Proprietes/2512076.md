


## RépertoireJournalOpération (Propriété)

***En anglais : OperationLogDirectory***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **RépertoireJournalOpération** permet de gérer le répertoire du fichier des opérations du journal associé à un fichier de données journalé. Il est ainsi possible de :

- Connaître le répertoire du journal des opérations lié à un fichier de données journalé défini sous l'éditeur d'analyses ou défini par programmation.

- Définir le répertoire du journal des opérations pour un fichier de données (lors de la description du fichier par programmation).




**Rappel** : Le fichier des opérations du journal (JournalOpération.fic) et le fichier d'identification des fichiers journalés (journalIdentification.fic) ne sont pas créés par défaut lors de la journalisation d'un fichier de données. Ces fichiers sont automatiquement créés lors de la journalisation en lecture d'un fichier de données ou lors de la mise en réplication d'un fichier de données.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Définit le répertoire du journal des opérations
// pour le fichier de données Commande (décrit par programmation)
Commande.RépertoireJournalOpération = "C:\MesFichiers\Opérations"
...
// Valide la description du fichier de données
HDécritFichier(Commande)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le répertoire du fichier des opérations d'un fichier de données journalé

`<Répertoire actuel> = <Fichier de données>.RépertoireJournalOpération`
---

**`<Répertoire actuel> : (Chaîne de caractères)`**



- Nom du répertoire du fichier "JournalOpération.fic" associé au fichier de données journalé, 

- "." si le répertoire correspond au répertoire en cours.




**`<Fichier de données> : (Chaîne de caractères)`**

Nom (logique) du fichier de données manipulé. Ce nom a été défini sous l'éditeur d'analyses ou grâce au type [Description de Fichier](../WDLang4/1514065.md).  


<a name="SYNTAXE2"></a>

### Définir le répertoire du fichier des opérations pour un fichier de données journalé décrit par programmation

`<Fichier de données>.RépertoireJournalOpération = <Nouveau répertoire>`
---

**`<Fichier de données> : (Chaîne de caractères)`**

Nom (logique) du fichier de données manipulé. Ce nom vient d'être défini grâce au type [Description de Fichier](../WDLang4/1514065.md).

**`<Nouveau répertoire> : (Chaîne de caractères)`**

Nom du répertoire du fichier "JournalOpération.fic" associé au fichier de données journalé.




## NomComplet (Propriété)

***En anglais : FullName***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **NomComplet** permet de connaître le nom complet :

- d'un champ.

- d'une fenêtre.

- d'une option de menu.

- d'une colonne de table.





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) 
```wl
Info("Le nom complet du champ est " + SAI_Saisie1.NomComplet)
// Affiche "Le nom complet du champ est FEN_Fenêtre1.SAI_Saisie1"
```
<a name="Exemple2"></a>
<a name="Exemple3"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) 
```wl
Info("Le nom complet du champ est " + COL_Colonne1.NomComplet)
// Affiche "Le nom complet du champ est FEN_Fenêtre1.TABLE_Table1.COL_Colonne1"
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le nom complet de l'élément spécifié

`<Résultat> = <Elément manipulé>.NomComplet`
---

**`<Résultat> : (Chaîne de caractères)`**



- Nom complet de l'élément spécifié.

- Chaîne vide ("") pour : 

	- les lignes des champs Table, Liste, Liste image ou Combo,

	- les cellules des champs Table.







**`<Elément manipulé> : (Chaîne de caractères)`**

Nom de l'élément à manipuler.


## AuthToken (Propriété)

***En anglais : AuthToken***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le mot-clé **AuthToken** peut correspondre : 

- une variable de type [AuthToken](../WDLang3/1000022220.md). 

- une propriété d'une variable de type [httpRequête](../WDLang3/1000021158.md). 

- une propriété d'une variable de type [emailSessionIMAP](../WDLang3/1000018957.md). 

- une propriété d'une variable de type [emailSessionSMTP](../WDLang3/1000018765.md).

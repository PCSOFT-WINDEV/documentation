
## FiltreFichier (Propriété)

***En anglais : FileFilter***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **FiltreFichier** permet de connaître et de modifier le filtre utilisé pour l'affichage des fichiers : 

- lors de la saisie assistée dans un champ de saisie ou une colonne de table avec l'option "Assister la saisie de chemin de fichier" activée (dans l'onglet "Général" de sa description), 

- dans le sélecteur de fichier affiché quand on sélectionne l'option "Parcourir..." du menu contextuel d'un champ de saisie, d'une colonne de table ou d'un champ Combo.



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Filtre sur les fichiers de log
SAI_SaisieFichierLog.FiltreFichier = "Fichiers de log (*.log)" + TAB + "*.log"
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître le filtre utilisé pour l'affichage des fichiers

`<Valeur du filtre> = <Champ manipulé>.FiltreFichier`
---

**`<Valeur du filtre> : (Chaîne de caractères)`**

Valeur actuelle du filtre sur les fichiers affichés : 

- lors de la saisie assistée.

- dans le sélecteur de fichier affiché par l'option "Parcourir..." du menu contextuel d'un champ. 




Cette valeur peut correspondre : 

- à une chaîne de caractères de la forme : 
	
	```txt
	<Type de fichier> + TAB + <Liste des extensions> [ + RC + ...
	<Type de fichier 2> + TAB + <Liste des extensions 2>]
	```

	Où :

	- &lt;Type de fichier&gt; : chaîne de caractères décrivant le type de fichier ("BMP (\*.BMP;\*.RLE)" par exemple).

	- &lt;Liste des extensions&gt; : extensions correspondant au type décrit ("\*.BMP;\*.RLE" par exemple).


 Par exemple :
	"BMP (\*.BMP;\*.RLE)"+TAB+"\*.BMP;\*.RLE"+RC+"Icône (\*.ICO)"+TAB+"\*.ICO"+RC+"Autres"+TAB+"\*.GIF;\*.TIF;\*.JPG"

- à une chaîne vide ("") si aucun filtre n'est utilisé. 




**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler : 

- Champ de saisie.

- Colonne d'un champ Table. 

- Champ Combo. 





<a name="SYNTAXE2"></a>

### Modifier le filtre utilisé pour l'affichage des fichiers

`<Champ manipulé>.FiltreFichier = <Nouveau filtre>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler : 

- Champ de saisie.

- Colonne d'un champ Table. 

- Champ Combo. 




**`<Nouveau filtre> : (Chaîne de caractères)`**

Valeur actuelle du filtre sur les fichiers affichés : 

- lors de la saisie assistée.

- dans le sélecteur de fichier affiché par l'option "Parcourir..." du menu contextuel d'un champ.


Cette valeur peut correspondre :  

- à une chaîne de caractères de la forme : 
	
	```txt
	<Type de fichier> + TAB + <Liste des extensions> [ + RC + ...
	<Type de fichier 2> + TAB + <Liste des extensions 2>]
	```

	Où :

	- &lt;Type de fichier&gt; : chaîne de caractères décrivant le type de fichier ("BMP (\*.BMP;\*.RLE)" par exemple).

	- &lt;Liste des extensions&gt; : extensions correspondant au type décrit ("\*.BMP;\*.RLE" par exemple).


 Par exemple :
	"BMP (\*.BMP;\*.RLE)"+TAB+"\*.BMP;\*.RLE"+RC+"Icône (\*.ICO)"+TAB+"\*.ICO"+RC+"Autres"+TAB+"\*.GIF;\*.TIF;\*.JPG"

- au résultat des fonctions [fFiltreImage](../WDLang1/3036009.md) ou [zipFiltre](../WDLang3/1000026293.md).

- à une chaîne vide ("") pour annuler le filtre en cours et n'utiliser aucun filtre.

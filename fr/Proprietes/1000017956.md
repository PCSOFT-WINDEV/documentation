
## Type (Propriété)

***En anglais : Type***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Type** permet de connaître et de modifier le type d'un élément. Cette propriété est utilisable sur :

- les champs d'une fenêtre ou d'une page. Pour plus de détails, consultez [Propriété Type](../Proprietes/2510131.md).

- une variable de type [gglAdressePostale](../WDLang5/1000017575.md).

- une variable de type [gglContact](../WDLang5/1000017434.md). 

- une variable de type [gglEmail](../WDLang5/1000017571.md).

- une variable de type [gglMessagerieInstantanée](../WDLang5/1000017569.md).

- une variable de type [gglSociété](../WDLang5/1000017756.md).

- une variable de type [gglTéléphone](../WDLang5/1000017573.md).

- une variable de type [rssDocumentJoint](../WDLang5/1000017789.md).

- une variable de type [xlsCellule](../WDLang5/1000017472.md).

- une variable de type [xmlNoeud](../WDLang5/1000018786.md).




- une variable d'un type généré automatiquement lors de l'[import d'un Webservice](../WDLang3/3086001.md).




- une variable de type [Définition](../WDLang1/1000019540.md). 

- une variable de type [Répétition](../WDLang1/1000019262.md).

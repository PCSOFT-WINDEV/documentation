


## Valeur (Propriété)

***En anglais : Value***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Valeur** permet de :

- Connaître et modifier la valeur d'un champ d'un état.

- Connaître et modifier la valeur d'un champ ou le titre d'une fenêtre.

- Connaître l'identifiant d'un flux. 

- Connaître et modifier la valeur d'un champ ou le titre d'une page.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Centrer la carte présente dans un champ Carte sur une position spécifique.

- Connaître la valeur associée à une valeur d'une variable de type [Enumération](../Motscles/1514067.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Connaître l'identifiant ou le handle associé à un champ Conteneur natif.




**Remarque** : La propriété **Valeur** est rarement utilisée et peut être remplacée par une syntaxe équivalente (voir Remarques).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Modifier le fichier image associé au champ Image "IMG_Image"
IMG_Image.Valeur = "C:\MesImages\Image.BMP"
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître la valeur d'un élément

`<Résultat> = <Elément manipulé>.Valeur`
---

**`<Résultat> : (Tout type)`**

Ce résultat dépend du type de l'élément.

**`<Elément manipulé> : (Type de l'élément)`**

Nom de l'élément à manipuler.  


<a name="SYNTAXE2"></a>

### Modifier la valeur d'un élément

`<Elément manipulé>.Valeur = <Nouvelle valeur>`
---

**`<Elément manipulé> : (Type de l'élément)`**

Nom de l'élément à manipuler.

**`<Nouvelle valeur> : (Tout type)`**

Nouvelle valeur de l'élément spécifié. Ce paramètre dépend du type de l'élément spécifié.

### Valeur d'un champ de saisie
<a name="valeur_champ_saisie_ELTPARAGRAPHE000260"></a>La propriété **Valeur** permet de récupérer ou de modifier la valeur saisie dans un champ de saisie.

&lt;Résultat&gt; est de type :

- chaîne de caractères si le sous-type du champ de saisie correspond à "Texte", "Date" ou "Heure",

- entier ou réel si le sous-type du champ de saisie correspond à "Numérique",

- numérique si le sous-type du champ de saisie correspond à "Monétaire" ou "Monétaire + Euro".




Pour connaître la valeur saisie dans un champ de saisie, les syntaxes suivantes sont équivalentes :

```txt
ResValeur = <Champ de saisie>.Valeur

ResValeur = <Champ de saisie>
```

### Valeur d'un champ Interrupteur ou d'une option d'un champ Interrupteur
<a name="valeur_champ_interrupteur_une_option_champ_interrupteur_ELTPARAGRAPHE000315"></a>La propriété **Valeur** permet de :

- Cocher ou décocher une option d'un champ Interrupteur.

- Savoir si une option d'un champ Interrupteur est cochée ou décochée




&lt;Résultat&gt; est un booléen : 

- <u><u><u><u>Vrai</u></u></u></u> si l'option est cochée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




Pour savoir si une option d'un champ Interrupteur est cochée ou non, les syntaxes suivantes sont équivalentes :

```txt
ResCoché = <Champ Interrupteur>[<Numéro de l'option>].Valeur

ResCoché = <Champ Interrupteur>[<Numéro de l'option>]

ResCoché = <Nom de l'option>.Valeur

ResCoché = <Nom de l'option>
```
Remarque : Si le paramètre &lt;Numéro de l'option&gt; n'est pas spécifié, la première option est sélectionnée par défaut.

### Valeur d'un champ Sélecteur
<a name="valeur_champ_selecteur_ELTPARAGRAPHE000345"></a>La propriété **Valeur** permet de :

- Connaître l'option active d'un champ Sélecteur.

- Sélectionner une option.




&lt;Résultat&gt; est un entier.

Pour connaître l'option active d'un champ Sélecteur, les syntaxes suivantes sont équivalentes :

Remarque : Par défaut, un champ Sélecteur est toujours positionné sur une position (1 par défaut). Pour afficher un champ Sélecteur sans option par défaut, il suffit d'affecter la valeur du champ à -1 lors de son initialisation :

```wl

```

### Valeur d'un champ Liste, Liste image ou Table
<a name="valeur_champ_liste_liste_image_table_ELTPARAGRAPHE000367"></a>Deux cas distincts se présentent.

- Le champ Liste (champ Liste image ou champ Table) est un champ par programmation OU la propriété [RubriqueMémorisée](../Proprietes/2510110.md) correspond à une chaîne vide ("").
	La propriété **Valeur** permet de :

	- Connaître la ligne sélectionnée. Si aucune ligne n'est sélectionnée, la propriété **Valeur** retourne -1. Si le champ est multisélection, le numéro de la dernière ligne sélectionnée est renvoyé.
			Pour connaître la ligne sélectionnée, les syntaxes suivantes sont équivalentes :
			
		```txt
		ResLigne = <Nom du champ>.Valeur
		
		ResLigne = <Nom du champ>
		```


	- Déplacer la sélection en indiquant le numéro de la nouvelle ligne à sélectionner. La ou les lignes précédemment sélectionnées sont dé-sélectionnées.   




- Le champ Liste (champ Liste image ou champ Table) est un champ fichier ET la propriété [RubriqueMémorisée](../Proprietes/2510110.md) n'est pas vide.
	La propriété **Valeur** permet de connaître la rubrique mémorisée pour l'enregistrement associé à la ligne sélectionnée. Si aucune ligne n'est sélectionnée, la propriété **Valeur** retourne une chaîne vide ("").
	Pour connaître la rubrique mémorisée associée à la ligne sélectionnée, les syntaxes suivantes sont équivalentes :
	
	```txt
	ResRubMem = <Nom du champ>.Valeur
	
	ResRubMem = <Nom du champ>.RubriqueMémorisée
	```

### Valeur d'une cellule d'un champ Table
<a name="valeur_une_cellule_champ_table_ELTPARAGRAPHE000419"></a>La propriété **Valeur** permet de connaître ou de modifier la valeur d'une cellule d'un champ Table.

&lt;Résultat&gt; est une chaîne de caractères.

Pour connaître la valeur d'une cellule, les syntaxes suivantes sont équivalentes :

```txt
ResCellule = <Champ Table>[<Numéro de ligne>][<Numéro de colonne>].Valeur

ResCellule = <Champ Table>[<Numéro de ligne>][<Numéro de colonne>]

ResCellule = <Colonne>[<Numéro de ligne>].Valeur

ResCellule = <Colonne>[<Numéro de ligne>]
```


**Cas particulier : Valeur d'une cellule de la ligne en cours**
Si seul le nom de la colonne est spécifié, la propriété **Valeur** permet de :

- Connaître la valeur de la cellule identifiée par la colonne spécifiée et par la ligne en cours dans le champ Table. Si aucune ligne n'est en cours, une chaîne vide ("") est renvoyée.

- Modifier la valeur de la cellule identifiée par la colonne spécifiée et par la ligne en cours dans le champ Table.




&lt;Résultat&gt; est une chaîne de caractères.

Pour connaître la valeur de la cellule identifiée par la colonne spécifiée et par la ligne en cours dans le champ Table, les syntaxes suivantes sont équivalentes :

```txt
ResColonne = <Nom de la colonne>.Valeur

ResColonne = <Nom de la colonne>
```

### Valeur d'un champ Combo

### Valeur d'un champ HTML
<a name="valeur_champ_html_ELTPARAGRAPHE000686"></a>La propriété **Valeur** permet de connaître l'adresse associée à la page HTML affichée dans le champ. Si la page :

- est une page construite directement en HTML, la valeur correspond à "about:blank".

- est une page d'un site, la valeur correspond à l'adresse de la page (par exemple www.pcsoft.fr).

- est un fichier HTML, la valeur correspond à l'adresse du fichier.


![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) La propriété **Valeur** renvoie l'URL du site mais ne peut pas renvoyer le code HTML du champ. Cette propriété ne peut pas être utilisée pour associer un fichier htm local au champ HTML.

### Valeur d'un champ Carte
<a name="valeur_champ_carte_ELTPARAGRAPHE000711"></a>La propriété **Valeur** permet de centrer la carte (présente dans un champ Carte) sur la position correspondant à une variable de type [géoPosition](../WDLang3/1000019191.md). 

Exemple : 

```wl
// Centre la carte affichée par le champ "CARTE_Position" sur une position géographique
MaPosition est une geoPosition

CARTE_Position.Valeur = MaPosition
```
**Remarques** : 

- L'utilisation de la propriété **Valeur** est équivalente à la fonction [CarteAffichePosition](../WDLang3/1000019787.md). Seule différence : lors de l'utilisation de la fonction [CarteAffichePosition](../WDLang3/1000019787.md), le positionnement sur la position demandée est réalisé à l'aide d'une animation.

### Limites
<a name="limites_ELTPARAGRAPHE000881"></a>![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Pour les états, la propriété **Valeur** ne peut pas être utilisée sur :

- un état.

- un bloc d'état.

- un champ Interrupteur.

- un champ Forme.

- un champ Agenda.

- un champ Planning. 

- un champ Diagramme de Gantt. 

- un champ Signature.


![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Dans une fenêtre, la propriété **Valeur** ne peut pas être utilisée sur :

- une option de sélecteur.

- un volet d'onglet.

- un groupe de champs.


 
 
![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) La propriété **Valeur** s'applique uniquement aux éléments suivants :

- Champ Bouton.

- Champ Libellé.

- Champ de saisie.

- Champ Image.

- Champ Interrupteur.

- Champ Sélecteur.

- Champ Liste.

- Champ Combo.

- Champ Onglet.

- Fenêtre.

- Champ Table.

- Colonne de table.

- Superchamp.

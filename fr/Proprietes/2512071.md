


## ProtégéParMotDePasse (Propriété)

***En anglais : PasswordProtected***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **ProtégéParMotDePasse** permet de savoir si un fichier de données est protégé par mot de passe.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Savoir si le fichier de données Client est protégé par un mot de passe 
ResProtection = Client.ProtégéParMotDePasse
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Protégé> = <Fichier de données>.ProtégéParMotDePasse`
---

**`<Protégé> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le fichier de données est protégé par mot de passe, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Fichier de données> : (Chaîne de caractères)`**

Nom du fichier de données manipulé. Ce fichier de données a été décrit sous l'éditeur d'analyses.

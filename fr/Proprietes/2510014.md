


## DndSource (Propriété)

***En anglais : DndSource***
	



<a name="XPresentation"></a>
<a name="Presentation"></a>
Pour certains types de champs, WINDEV et WEBDEV proposent une [gestion du "Drag and Drop" automatique](../WDLang1/3030012.md) : il suffit de cocher l'option correspondante dans l'onglet "Détail" de la description du champ. Un champ peut alors être défini comme source ou cible du "Drag and Drop".

Le Drag and Drop automatique est disponible : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) pour les champs de saisie, les listes, les listes image, les arbres.


Lors d'un [Drag and Drop programmé](../WDLang1/3030013.md), les propriétés **DndSource** et [DndCible](../Proprietes/2510016.md) permettent de définir les champs source et cible du Drag and Drop.
<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **DndSource** permet de connaître et de modifier le comportement d'un champ (champ de saisie, Liste, Liste image, Zone répétée, Arbre, ...) vis-à-vis du "Drag and Drop". Ce champ sera source du "Drag and Drop", c'est-à-dire que le "Drag and Drop" débutera depuis ce champ.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// -- Initialisation du champ LISTE_Source
// LISTE_Source n'est pas cible
LISTE_Source.DndSource = dndInactif

// -- Initialisation du champ LISTE_Cible
LISTE_Cible.DndCible = dndProgrammé
DndEvénement("Lâcher", "LISTE_Cible", dndLâcher)
DndEvénement("Survol", "LISTE_Cible", dndSurvol)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le comportement d'un champ source du "Drag and Drop"

`<Gestion du Drag and Drop> = <Champ manipulé>.DndSource`
---

**`<Gestion du Drag and Drop> : (Constante)`**

Identifie le comportement du champ source du "Drag and Drop" :


|   |   |
| --- | --- |
| *dndAuto* | Gestion automatique du "Drag and Drop" |
| *dndInactif* | Le "Drag and Drop" à partir de ce champ est inactif |
| *dndProgrammé* | La gestion du "Drag and Drop" à partir de ce champ est à la charge du développeur |



**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler.  


<a name="SYNTAXE2"></a>

### Modifier le comportement d'un champ source du "Drag and Drop"

`<Champ manipulé>.DndSource = <Gestion du Drag and Drop>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler.

**`<Gestion du Drag and Drop> : (Constante)`**

Identifie le comportement du champ source du "Drag and Drop".


|   |   |
| --- | --- |
| *dndAuto* | Gestion automatique du "Drag and Drop" |
| *dndInactif* | Le "Drag and Drop" à partir de ce champ est inactif |
| *dndProgrammé* | La gestion du "Drag and Drop" à partir de ce champ est à la charge du développeur |

## Remarques


### "Drag and Drop" programmé ou automatique
<a name="drag_and_drop_programme_automatique_ELTPARAGRAPHE000105"></a>![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Un "Drag and Drop" automatique est disponible uniquement pour les champs de saisie, les listes, les listes image, les tables et les arbres.

Un "Drag and Drop" programmé est disponible pour les champs suivants :

- Champ de saisie

- Champ Liste et champ Liste image

- Champ Arbre

- Champ Table

- Champ Image

- Champ Code-Barres

- Champ Bouton

- Champ Zone répétée

- Champ Graphe

- Champ Table hiérarchique

- Champ Calendrier

- Champ Agenda

- Champ Planning




![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Seul le "Drag and Drop" automatique est disponible.


<a name="NOTE0_2"></a>


### Limite
<a name="limite_ELTPARAGRAPHE000152"></a>![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) La propriété **DndSource** peut être utilisée uniquement avec les champs suivants :

- Champ de saisie,

- Champ Liste,

- Champ Liste image,

- Champ Arbre,

- Champ Zone répétée,

- Champ Image,

- Champ Graphe, 

- Champ Table, 

- Champ Table hiérarchique,

- Champ Calendrier, 

- Champ Agenda,

- Champ Planning,

- Champ Bouton,

- Champ Notation.




![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) La propriété **DndSource** s'applique uniquement aux champs de saisie, listes et arbres.

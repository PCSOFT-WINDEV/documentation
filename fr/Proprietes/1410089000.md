
## Markdown (Propriété)

***En anglais : Markdown***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Markdown** permet de : 

- savoir si un champ Libellé d'un état est de type Markdown ou non. 

- modifier le type d'un champ Libellé présent dans un état pour qu'il devienne un libellé de type Markdown. 




Remarque : Le type Markdown est une option des champs Libellé. Cette option est située dans l'onglet "Général" de la description du champ.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Si le champ Libellé "LIB_Nom" est de type Markdown, il devient visible
SI LIB_Nom.Markdown = Vrai ALORS
	LIB_Nom.Visible = Vrai
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Savoir si le champ est de type Markdown

`<Résultat> = <Champ Libellé>.Markdown`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'élément spécifié est de type Markdown, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Champ Libellé> : (Nom de champ)`**

Champ Libellé d'un état. 


<a name="SYNTAXE2"></a>

### Modifier le type Markdown d'un champ

`<Champ Libellé>.Markdown = <Type>`
---

**`<Champ Libellé> : (Nom de champ)`**

Champ Libellé d'un état. 

**`<Type> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'élément spécifié doit être de type Markdown, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.

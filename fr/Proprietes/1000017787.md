


## Chemin (Propriété)

***En anglais : Path***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Chemin** permet de connaître et de modifier un chemin. Cette propriété est utilisable sur :

- une variable de type [gglParamètreCarte](../WDLang5/1000017505.md).

- une variable de type [rssFlux](../WDLang5/1000017795.md).

- une variable de type [rssCloud](../WDLang5/1000017785.md).

- une variable de type [Cookie](../WDLang3/1000019222.md).

- une variable de type [hdfsInformation](../WDLang4/1000021973.md).


## Y (Propriété)

***En anglais : Y***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Y** permet de connaître et de modifier l'ordonnée d'un élément. Cette propriété est utilisable sur :

- les champs, les fenêtres, les pages et les états. Pour plus de détails, consultez [Y](../Proprietes/2510137.md).

- une variable de type [Point](../Motscles/1000018908.md).

- une variable de type [Rectangle](../Motscles/1514079.md).

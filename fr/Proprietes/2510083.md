


## NombreColonne (Propriété)

***En anglais : NumberColumn***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **NombreColonne** permet de :

- Connaître le nombre de colonnes présentes dans :

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) un champ Liste multicolonne.

	- un champ Table ou Table hiérarchique. 

	- un champ Zone répétée.

	- un champ Sélecteur.

	- un champ Interrupteur. 

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) un champ Tableau croisé dynamique.

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) un champ Combo Table.

	- un champ Tableau de Bord.

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) un champ Tableur. 




- Modifier le nombre de colonnes présentes : 

	- dans un champ Liste multicolonne. Le nombre de colonnes d'un champ Liste multicolonne ne peut pas être inférieur à 2.

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) dans un champ Zone répétée.







![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) **Rappel** : Pour rendre un champ Liste multicolonne (champ Liste ou champ Liste image affiché en mode classique) :

1. Affichez la fenêtre de description du champ (option "Description" du menu contextuel du champ).

2. Dans l'onglet "Détail", saisissez le nombre de colonnes voulu dans l'option "Nombre de colonnes".





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// --Traitement de modification de taille de la fenêtre
// Si la largeur de la fenêtre est modifiée :
// - la largeur du champ Liste est modifiée
LISTE_MaListe.Largeur = MaFenêtre.Largeur - 30
// - le nombre de colonnes présentes dans le champ Liste est modifié
// selon la nouvelle largeur de la fenêtre
LISTE_MaListe.NombreColonne = LISTE_MaListe.Largeur/50
```

## Syntaxe

### Connaître le nombre de colonnes d'un champ

`<Nombre de colonnes> = <Champ manipulé>.NombreColonne`
---

**`<Nombre de colonnes> : (Entier)`**



- Nombre de colonnes présentes dans le champ Liste multicolonne spécifié.

- Nombre de colonnes présentes dans le champ Table spécifié.

- Nombre de colonnes présentes dans le champ Interrupteur ou Sélecteur spécifié.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Nombre de colonnes présentes dans le champ Tableau croisé dynamique spécifié.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Nombre de colonnes présentes dans le champ Combo Table spécifié.

- Nombre de cellules en largeur présentes dans le champ Tableau de bord spécifié.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Nombre maximum de colonnes ayant des données (valeur ou formule) dans un champ Tableur. 

- Nombre de colonnes présentes dans le champ Zone répétée spécifié. 

- 1 si l'élément spécifié ne correspond pas à un champ pouvant être traité.




**`<Champ manipulé> : (Nom de champ)`**

Nom du champ dont on veut connaître le nombre de colonnes.  


<a name="SYNTAXE2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 
### Modifier le nombre de colonnes présentes dans un champ

`<Champ manipulé>.NombreColonne = <Nombre de colonnes>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler : 

- Champ Liste multicolonne

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Champ Zone répétée.




**`<Nombre de colonnes> : (Entier)`**

Nouveau nombre de colonnes. 

Pour les listes, ce nombre doit être supérieur à 1 (il n'est pas possible de transformer un champ Liste multicolonne en liste standard ou inversement).

<a name="NOTE0_1"></a>

## Remarques


### Limites
<a name="limites_ELTPARAGRAPHE000150"></a>La propriété **NombreColonne** s'applique uniquement aux :

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Champs Liste multicolonne (champs Liste image affichés en mode classique).

- Champs Table ou Table hiérarchique. 

- Champs Sélecteur.

- Champs Interrupteur.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Champs Tableau croisé dynamique.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Champs Combo Table.

- Champs Tableau de bord.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Champs Tableur.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Champs Zone répétée.




![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) La propriété **NombreColonne** s'applique uniquement aux éléments suivants :

- Champs Interrupteur.

- Champs Sélecteur.

- Champs Table.

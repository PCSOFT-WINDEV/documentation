
## Associé (Propriété)

***En anglais : Associated***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Associé** permet de savoir si un élément appartient à la classe de base ou à une classe associée. Cette propriété est utilisée par :

- une variable de type [Description de procédure](../WDLang1/1000019317.md).

- une variable de type [Description de Propriété](../WDLang1/1000019324.md).

- une variable de type [Description de Variable](../WDLang1/1000019560.md).

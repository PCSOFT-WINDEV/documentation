
## Etiquette (Propriété)

***En anglais : Label***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Etiquette** permet de connaître et de définir l'étiquette d'un élément. Cette propriété est utilisable sur :

- une variable de type [AdresseEmail](../WDLang3/1000020244.md).

- une variable de type [AdresseSIP](../WDLang3/1000020245.md).

- une variable de type [AdresseWeb](../WDLang3/1000020246.md).

- une variable de type [Evénement](../WDLang3/1000020248.md). 

- une variable de type [MessagerieInstantanée](../WDLang3/1000020247.md). 

- une variable de type [Téléphone](../WDLang3/1000020241.md).

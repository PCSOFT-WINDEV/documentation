


## OngletDynamique (Propriété)

***En anglais : DynamicTab***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **OngletDynamique** permet de : 

- Connaître le type d'un champ Onglet (onglet statique ou onglet dynamique). 

- Modifier le type d'un champ Onglet (onglet statique ou onglet dynamique). 




**Rappel** : Un champ Onglet dynamique est un champ Onglet qui supporte l'ajout, la suppression et le déplacement des volets.

Remarque : Cette propriété remplace la propriété [OngletMDI](../Proprietes/1000021471.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Vérifie si l'onglet est un onglet dynamique
SI ONG_Onglet1.OngletDynamique = Vrai ALORS
	// Fermeture de tous les volets
	OngletFermeTout(ONG_Onglet1)
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Savoir si un champ Onglet est un onglet dynamique

`<Résultat> = <Champ Onglet>.OngletDynamique`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le champ Onglet est un onglet dynamique, 

- <u><u><u><u>Faux</u></u></u></u> si le champ Onglet est un onglet statique. 




**`<Champ Onglet> : (Nom de champ)`**

Nom du champ Onglet à manipuler. 


<a name="SYNTAXE2"></a>

### Modifier le type d'un champ Onglet

`<Champ Onglet>.OngletDynamique = <Type du champ Onglet>`
---

**`<Champ Onglet> : (Nom de champ)`**

Nom du champ Onglet à manipuler. 

**`<Type du champ Onglet> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> pour que le champ Onglet soit un champ Onglet dynamique, 

- <u><u><u><u>Faux</u></u></u></u> pour que le champ Onglet soit un champ Onglet statique. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Champ Onglet dynamique
<a name="champ_onglet_dynamique_ELTPARAGRAPHE000065"></a>Si la propriété **OngletDynamique** est à <u><u><u><u>Vrai</u></u></u></u> : 

- les fonctions [OngletXXX](../WDLang1/1000021391.md) peuvent être appelées sur le champ.

- l'opérateur [] sur l'onglet doit être appelé avec l'alias du volet. 

- l'utilisateur final peut : 

	- supprimer des volets si la propriété [AvecBoutonFermeture](../Proprietes/1000021374.md) est à <u><u><u><u>Vrai</u></u></u></u>. 

	- ajouter des volets si la propriété [AvecBoutonNouveau](../Proprietes/1000021377.md) est à <u><u><u><u>Vrai</u></u></u></u>.

	- modifier l'ordre des volets par Drag and Drop. 

	- détacher les volets de la fenêtre si la propriété [VoletDétachable](../Proprietes/1000021382.md) est à <u><u><u><u>Vrai</u></u></u></u>.

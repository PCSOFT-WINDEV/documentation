


## MotsClés (Propriété)

***En anglais : Keywords***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **MotsClés** permet de connaître et modifier les mots-clés associés à une page. Les mots-clés peuvent être également associés à une page depuis l'éditeur de pages.

La propriété **MotsClés** permet dans le cas de pages AWP de choisir les mots-clés associés à la page dynamiquement (par exemple en fonction du contenu affiché dans la page).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Modifie les mots-clés de la page en fonction de son contenu
// Ajoute des nouveaux mots-clés en plus de ceux existant
MaPage.MotsClés = MaPage.MotsClés + ", Bateau, Voile, Moteur"
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître les mots-clés associés à une page

`<Liste des mots-clés> = <Page manipulée>.MotsClés`
---

**`<Liste des mots-clés> : (Chaîne de caractères)`**

Mots-clés associés à la page. Les différents mots-clés associés à la page sont séparés par des virgules (",").

**`<Page manipulée> : (Nom de page)`**

Nom de la page à analyser.  


<a name="SYNTAXE2"></a>

### Modifier les mots-clés associés à une page 

`<Page manipulée>.MotsClés = <Liste des mots-clés>`
---

**`<Page manipulée> : (Nom de page)`**

Nom de la page pour laquelle des mots-clés doivent être définis.

**`<Liste des mots-clés> : (Chaîne de caractères)`**

Mots-clés à associer à la page. Les différents mots-clés associés sont séparés par des virgules (","). Si la page a déjà des mots-clés, ces mots-clés sont remplacés. Cette chaîne doit respecter les règles suivantes :

- Ne pas contenir de Retour Chariot (RC)

- Ne pas contenir d'espaces, guillemets.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Mots-clés associés à une page
<a name="motscles_associes_une_page_ELTPARAGRAPHE000045"></a>Les mots-clés associés à une page sont écrits dans l'entête HTML de la page (balise &lt;META NAME="Keywords" CONTENT="..."&gt;). Ces mots-clés sont souvent utilisés par les moteurs de recherche pour référencer les pages.

Les mots-clés doivent être définis dans le code d'initialisation de la page. De plus, pour être efficaces, les mots-clés doivent être définis sur des pages accessibles par une URL de la forme "http://".

Quelques conseils : 

- Les 3 premiers mots-clés de la liste sont très importants.

- Ne pas saisir plus de 20 mots-clés par page.

- Ne pas répéter le même mot-clé.

- Penser aux variantes (singulier / pluriel, masculin / féminin, nom / verbe).

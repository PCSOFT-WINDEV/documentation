


## CxInfosEtendues (Propriété)

***En anglais : CtExtendedInfo***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **CxInfosEtendues** permet de connaître les informations supplémentaires insérées dans la chaîne de connexion à la base de données. Il est possible par exemple de connaître les informations supplémentaires spécifiées pour la connexion à une table par OLE DB.

Ces informations ont été spécifiées :

- soit lors de la création de la connexion sous l'éditeur d'analyses.

- soit lors de la définition de la connexion dans les fonctions [HConnecte](../WDLang4/3044263.md), [HDécritConnexion](../WDLang4/3044205.md) ou [HOuvreConnexion](../WDLang4/3044107.md).




Cette propriété peut être utilisée pour un fichier de données ou une connexion défini sous l'éditeur d'analyses ou défini par programmation.

**Rappel** : Informations étendues : Information insérée dans la chaîne de connexion à la base de données.

Pour plus de détails sur ces informations étendues, consultez l'aide du fournisseur correspondant à l'accès utilisé.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connaître les informations étendues de la connexion utilisée
// par le fichier de données Client
Info(Client.CxInfosEtendues)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Informations étendues> = <Fichier de données/connexion>.CxInfosEtendues`
---

**`<Informations étendues> : (Chaîne de caractères)`**

Information insérée dans la chaîne de connexion à la base de données. 

**`<Fichier de données/connexion> : (Chaîne de caractères)`**



- Nom du fichier de données manipulé. Ce nom a été défini sous l'éditeur d'analyses ou grâce au type [Description de Fichier](../WDLang4/1514065.md).

- Nom de la connexion manipulée. Ce nom a été défini sous l'éditeur d'analyses ou avec les fonctions [HDécritConnexion](../WDLang4/3044205.md) ou [HOuvreConnexion](../WDLang4/3044107.md).

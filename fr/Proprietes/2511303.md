


## Description (Propriété)

***En anglais : Description***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Description** permet de connaître et de modifier la description d'un élément. Cette propriété est utilisable sur :

- une variable de type [gglAgenda](../WDLang5/1000017418.md).

- une variable de type [hDescriptionSauvegarde](../WDLang4/1000017455.md). 

- une variable de type [hOptimisationPlanifiée](../WDLang4/1000018854.md).

- une variable de type [hTâchePlanifiée](../WDLang4/1000017448.md).

- une variable de type [Marqueur](../WDLang3/1000019940.md).

- une variable de type [rssCanal](../WDLang5/1000017769.md).

- une variable de type [rssFlux](../WDLang5/1000017795.md).

- une variable de type [rssImage](../WDLang5/1000017799.md). 

- une variable de type [rssEntrée](../WDLang5/1000017792.md).

- une variable de type [rssSaisieTexte](../WDLang5/1000017802.md)

- une variable de type [hRSConfig](../WDLang4/1000020484.md).

- une variable de type [inAppProduit](../WDLang3/1000020868.md).

- une variable de type [hPlanificationVueMatérialisée](../WDLang4/1000021444.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [notificationCatégorie](../WDLang3/1000023981.md).

- une variable de type [CarteCercle](../WDLang3/1000025504.md).

- une variable de type [CarteImage](../WDLang3/1000025523.md).

- une variable de type [CartePolygone](../WDLang3/1000025476.md).

- une variable de type [CartePolyligne](../WDLang3/1000025489.md).

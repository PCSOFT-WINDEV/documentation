


## Message (Propriété)

***En anglais : Message***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Message** permet de :

- Connaître le message d'aide associé à un champ ou une option de menu d'une fenêtre.

- Modifier le message d'aide associé à un champ ou une option de menu d'une fenêtre.




**Remarques** :

- Le message d'aide s'affiche dans la barre de message de la fenêtre.

- Le message d'aide est défini dans l'onglet "Aide" de la description du champ (option "Texte de la barre de message"), par la propriété **Message** ou par la fonction [Message](../WDLang1/3021006.md).



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Modifier le message d'aide du champ de saisie "SAI_FichierImage"
SAI_FichierImage.Message = "Le fichier image doit être une image en noir && blanc"
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le message d'aide

`<Résultat> = <Elément manipulé>.Message`
---

**`<Résultat> : (Chaîne de caractères)`**



- Message d'aide de l'objet spécifié, 

- Chaîne vide ("") si aucun message d'aide n'est associé à l'objet.




**`<Elément manipulé> : (Chaîne de caractères)`**

Nom de l'objet (champ ou option de menu) dont on veut connaître le message d'aide.  


<a name="SYNTAXE2"></a>

### Modifier le message d'aide

`<Elément manipulé>.Message = <Nouveau message d'aide>`
---

**`<Elément manipulé> : (Chaîne de caractères)`**

Nom de l'objet (champ ou option de menu) dont on veut modifier le message d'aide.

**`<Nouveau message d'aide> : (Chaîne de caractères)`**

Nouveau message d'aide de l'objet spécifié.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Affichage du message d'aide
<a name="affichage_message_aide_ELTPARAGRAPHE000052"></a>

- **Pour un champ** (sauf les champs Bouton) : le message d'aide apparaît dans la barre de message lorsque le champ prend le focus.

- **Pour un champ Bouton** : le message d'aide apparaît dans la barre de message dès que le bouton gauche de la souris est enfoncé.

- **Pour une option de menu** : le message d'aide apparaît dans la barre de message lorsque l'option est survolée.



<a name="NOTE0_2"></a>


### Caractère &
<a name="caractere_ELTPARAGRAPHE000061"></a>Pour afficher le caractère "&" dans le message d'aide, doublez ce caractère. Sinon, le caractère suivant le caractère "&" sera souligné et le caractère "&" ne sera pas affiché. Par exemple :

```wl
SAI_FichierImage.Message = "Le fichier image doit être une image en noir && blanc"
```

<a name="NOTE0_3"></a>


### Limites
<a name="limites_ELTPARAGRAPHE000069"></a>La propriété **Message** ne peut pas être utilisée sur :

- un champ Image,

- un champ Forme,

- une option d'un champ Interrupteur,

- une option d'un champ Sélecteur,

- une ligne d'un champ Liste, d'un champ Combo ou d'un champ Table,

- une cellule de table,

- un onglet,

- un groupe de champs,

- un élément d'un champ Arbre.

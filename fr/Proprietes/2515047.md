


## Propriétés HFSQL des requêtes et des vues
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Ces propriétés peuvent être utilisées sur des requêtes ou des vues HFSQL.




|   |   |
| --- | --- |
| [BorneMax](../Proprietes/2510009.md) | La propriété **BorneMax** permet de :<br><br>- connaître et de modifier la borne maximale d'un état basé sur un fichier de données. <br><br>- connaître la borne maximale du filtre en cours.<br><br>- connaître la valeur maximale d'une série d'un graphe.<br><br>- connaître et modifier la valeur maximale de certains types de champs.<br><br><br> |
| [BorneMin](../Proprietes/2510008.md) | La propriété **BorneMin** permet de :<br><br>- connaître et de modifier la borne minimale d'un état basé sur un fichier de données. <br><br>- connaître la borne minimale du filtre en cours.<br><br>- connaître la valeur minimale d'une série d'un graphe.<br><br>- connaître et modifier la valeur minimale de certains types de champs. <br><br><br> |
| [CodeSQL](../Proprietes/1000017045.md) | La propriété **CodeSQL** permet de connaître le code SQL d'une requête créée avec l'éditeur de requêtes (fichier d'extension ".WDR"). |
| [ConditionFiltre](../Proprietes/2512096.md) | La propriété **ConditionFiltre** permet de connaître la condition de sélection mise en place par la fonction [HFiltre](../WDLang4/3044100.md) sur un fichier de données, une vue HFSQL ou une requête. |
| [ExécutionTerminée](../Proprietes/2512117.md) | La propriété **ExécutionTerminée** permet de savoir si une requête ou une vue HFSQL Classic ou Client/Serveur a fini de s'exécuter. |
| [FiltreAvecBornes](../Proprietes/2512097.md) | La propriété **FiltreAvecBornes** permet de savoir si des bornes ont été précisées sur le filtre mis en place par la fonction [HFiltre](../WDLang4/3044100.md) sur un fichier de données, une vue HFSQL ou une requête. |
| [IDAutoMax](../Proprietes/2512023.md) | La propriété **IDAutoMax** permet de connaître l'identifiant automatique maximum sur un fichier de données répliqué. |
| [IDAutoMin](../Proprietes/2512039.md) | La propriété **IDAutoMin** permet de connaître l'identifiant automatique minimum sur un fichier de données répliqué. |
| [NbRubrique](../Proprietes/2512055.md) | La propriété **NbRubrique** permet d'obtenir le nombre total de rubriques (y compris les clés composées) présentes dans un enregistrement de :<br><br>- un fichier décrit sous l'éditeur d'analyses,<br><br>- un fichier décrit par programmation,<br><br>- une vue HFSQL,<br><br>- une requête.<br><br><br> |
| [NbRubriqueClé](../Proprietes/2512056.md) | La propriété **NbRubriqueClé** permet d'obtenir le nombre total de rubriques clé (y compris les clés composées) présentes dans un enregistrement de :<br><br>- un fichier décrit sous l'éditeur d'analyses,<br><br>- un fichier décrit par programmation,<br><br>- une vue HFSQL,<br><br>- une requête.<br><br><br> |
| [NbRubriqueMémo](../Proprietes/2512057.md) | La propriété **NbRubriqueMémo** permet d'obtenir le nombre total de rubriques mémo présentes dans un enregistrement de :<br><br>- un fichier décrit sous l'éditeur d'analyses,<br><br>- un fichier décrit par programmation,<br><br>- une vue HFSQL,<br><br>- une requête.<br><br><br> |
| [Nom](../Proprietes/2510082.md) | La propriété **Nom** permet de connaître :<br><br>- le nom de la police associée à une variable de type Police.<br><br>- le nom d'un champ, d'un état ou d'un bloc d'un état.<br><br>- le nom d'un champ, d'un groupe de champs ou d'une fenêtre.<br><br>- le nom d'un champ, d'un groupe de champs ou d'une page.<br><br><br> |
| [NomDécrit](../Proprietes/2512066.md) | La propriété **NomDécrit** permet de gérer le nom logique des fichiers de données HFSQL (sans tenir compte des éventuelles modifications effectuées par la fonction [HAlias](../WDLang4/3044176.md)). |
| [RubriqueFiltrée](../Proprietes/2512086.md) | La propriété **RubriqueFiltrée** permet de connaître la rubrique sur laquelle un filtre a été mis en place par la fonction [HFiltre](../WDLang4/3044100.md). |
| [TailleEnregistrement](../Proprietes/2512090.md) | La propriété **TailleEnregistrement** permet d'obtenir la taille (en octets) d'un enregistrement de :<br><br>- un fichier décrit sous l'éditeur d'analyses,<br><br>- un fichier décrit par programmation,<br><br>- une vue HFSQL,<br><br>- une requête.<br><br><br> |
| [Type](../Proprietes/2510131.md) | La propriété **Type** permet de connaître :<br><br>- le type d'un élément décrit par programmation (description de rubrique ou de fichier) ou non (fichier ou rubrique défini sous l'éditeur d'analyses, vue ou requête).<br><br>- le type d'un objet.<br><br>- le type de la valeur stockée dans un variant.<br><br><br> |

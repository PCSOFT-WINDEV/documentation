


## AncrageAuContenu (Propriété)

***En anglais : AnchorToContent***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **AncrageAuContenu** permet de connaître et de modifier le mode d'adaptation du champ par rapport à son contenu. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Agrandir le libellé selon son contenu
LIB_MonTexte.AncrageAuContenu = aacAdapteEnHauteur
LIB_MonTexte = Produit.Description
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître le mode d'adaptation du champ par rapport à son contenu

`<Résultat> = <Champ manipulé>.AncrageAuContenu`
---

**`<Résultat> : (Constante de type Entier)`**

Mode d'adaptation actuellement utilisé pour le champ : 



|   |   |
| --- | --- |
| *aacAdapteEnHauteur* | Lorsque le contenu du champ change, sa hauteur change et s'adapte au nouveau contenu.  |
| *aacAdapteEnLargeur* | Lorsque le contenu du champ change, sa largeur change et s'adapte au nouveau contenu. |
| *aacAucun* | Lorsque le contenu du champ change, le champ ne change pas de taille. |



**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler : Champ de saisie, Libellé, Champ d'affichage HTML, Interrupteur, Sélecteur, modèle de champs. 


<a name="SYNTAXE2"></a>

### Modifier le mode d'adaptation du champ par rapport à son contenu

`<Champ manipulé>.AncrageAuContenu = <Nouveau mode d'adaptation>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler : Champ de saisie, Libellé, Champ d'affichage HTML, Interrupteur, Sélecteur, modèle de champs. 

**`<Nouveau mode d'adaptation> : (Constante de type Entier)`**

Nouveau mode d'adaptation utilisé pour le champ : 



|   |   |
| --- | --- |
| *aacAdapteEnHauteur* | Lorsque le contenu du champ change, sa hauteur change et s'adapte au nouveau contenu.  |
| *aacAdapteEnLargeur* | Lorsque le contenu du champ change, sa largeur change et s'adapte au nouveau contenu. |
| *aacAucun* | Lorsque le contenu du champ change, le champ ne change pas de taille.  |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Si le champ est ancré et présent dans un champ Disposition, la cellule est agrandie pour que la taille du champ soit adaptée selon son ancrage.

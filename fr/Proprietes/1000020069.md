


## MémoriseTriFAA (Propriété)

***En anglais : StoreSortAAF***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **MémoriseTriFAA** permet de savoir et d'indiquer si le tri effectué par l'utilisateur sur un champ Table ou Table hiérarchique est conservé entre 2 lancements de l'application. 

Ce tri a été réalisé grâce aux boutons de tri présents dans les colonnes de la table.

Si le tri est conservé, à la prochaine ouverture de la fenêtre contenant la table, le tri sera automatiquement réappliqué. 

**Remarque** : Cette propriété est identique à l'option "Mémoriser les tris utilisateur" dans l'onglet "Détail" de la fenêtre de description du champ Table. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Modification de INT_MémoriserTri
SI INT_MémoriserTri = Vrai ALORS
	TABLE_PRIX.MémoriseTriFAA = Vrai
SINON
	TABLE_PRIX.MémoriseTriFAA = Faux
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Savoir si les options de tri sont conservées

`<Résultat> = <Table manipulée>.MémoriseTriFAA`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si les options de tri définies par l'utilisateur sur le champ Table sont conservées. 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire (valeur par défaut). 




**`<Table manipulée> : (Nom de champ)`**

Nom du champ Table ou Table hiérarchique à manipuler. 


<a name="SYNTAXE2"></a>

### Modifier la gestion des options de tri

`<Table manipulée>.MémoriseTriFAA = <Mémorisation>`
---

**`<Table manipulée> : (Nom de champ)`**

Nom du champ Table ou Table hiérarchique à manipuler. 

**`<Mémorisation> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> pour conserver les options de tri définies par l'utilisateur sur le champ Table entre 2 lancements de l'application. 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Les informations de tri sont sauvées avec les autres informations persistantes (pour plus de détails consultez l'aide de la fonction [SauveParamètre](../WDLang1/3025046.md)).




## VisibleEnExportEtImpression (Propriété)

***En anglais : VisibleInExportAndPrint***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **VisibleEnExportEtImpression** permet de : 

- Savoir si une colonne d'un champ Table ou Table hiérarchique est visible pour l'export et l'impression.

- Modifier la visibilité d'une colonne d'un champ Table ou Table hiérarchique pour l'export et l'impression.




Remarque : Cette propriété correspond à l'option "Export et impression" disponible dans l'onglet "Général" de la fenêtre de description d'une colonne d'un champ Table ou Table hiérarchique. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Bouton [Imprimer sans détail]
TABLE_CLIENT.COL_Détail.VisibleEnExportEtImpression = Faux
iImprimeEtat(ETAT_EtatSurTableClient)
TABLE_CLIENT.COL_Détail.VisibleEnExportEtImpression = Défaut
```
<a name="Exemple2"></a>

```wl
// Export vers Excel :
// 1. La colonne COL_Détail est exportée
TABLE_MaTable.COL_Détail.VisibleEnExportEtImpression = Faux
TableVersExcel(TABLE_MaTable, sFic, taSansTitre + taAvecColonnesInvisibles)
// 2. La colonne COL_Détail n'est pas exportée
TABLE_MaTable.COL_Détail.VisibleEnExportEtImpression = Faux
TableVersExcel(TABLE_MaTable, sFic, taSansTitre)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Savoir si une colonne d'un champ Table ou Table hiérarchique est visible pour l'export et l'impression

`<Résultat> = <Colonne>.VisibleEnExportEtImpression`
---

**`<Résultat> : (Constante de type Entier)`**

Mode de gestion de la colonne : 


|   |   |
| --- | --- |
| *Défaut* | La colonne est exportée ou imprimée uniquement si la colonne est visible à l'affichage. |
| *Faux* | La colonne ne peut jamais être exportée ou imprimée. <br>**Attention** : Lors d'un export réalisé avec une des fonctions d'export ([TableVersExcel](../WDLang1/3074033.md), [TableVersTexte](../WDLang1/1000017061.md), [TableVersWord](../WDLang1/3074036.md), [TableVersXML](../WDLang1/3074037.md), [TableVersPressePapier](../WDLang1/1000017062.md)), si la constante *taAvecColonnesInvisibles* est utilisée, la colonne sera tout de même exportée. |
| *Vrai* | La colonne peut être exportée ou imprimée (même si la colonne est invisible à l'affichage). |



**`<Colonne> : (Nom de champ)`**

Nom de la colonne à manipuler. Cette colonne peut être une colonne d'un champ Table ou Table hiérarchique. 


<a name="SYNTAXE2"></a>

### Modifier la visibilité d'une colonne d'un champ Table ou Table hiérarchique pour l'export et l'impression

`<Colonne>.VisibleEnExportEtImpression = <Visibilité>`
---

**`<Colonne> : (Nom de champ)`**

Nom de la colonne à manipuler. Cette colonne peut être une colonne d'un champ Table ou Table hiérarchique. 

**`<Visibilité> : (Constante de type Entier)`**

Mode de gestion de la colonne : 


|   |   |
| --- | --- |
| *Défaut* | La colonne est exportée ou imprimée uniquement si la colonne est visible à l'affichage. |
| *Faux* | La colonne ne peut jamais être exportée ou imprimée.<br>**Attention** : Lors d'un export réalisé avec une des fonctions d'export ([TableVersExcel](../WDLang1/3074033.md), [TableVersTexte](../WDLang1/1000017061.md), [TableVersWord](../WDLang1/3074036.md), [TableVersXML](../WDLang1/3074037.md), [TableVersPressePapier](../WDLang1/1000017062.md)), si la constante *taAvecColonnesInvisibles* est utilisée, la colonne sera tout de même exportée. |
| *Vrai* | La colonne peut être exportée ou imprimée (même si la colonne est invisible à l'affichage). |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
La propriété **VisibleEnExportEtImpression** est prise en compte dans les cas suivants :  

- impression réalisée via la FAA d'impression (Fonctionnalité Automatique de vos Applications) : impression par exemple par l'option "Imprimer" du menu contextuel du champ Table.  

- impression réalisée via les FAA d'export disponibles dans le menu contextuel du champ Table,

- export via les fonctions d'export ([TableVersExcel](../WDLang1/3074033.md), [TableVersTexte](../WDLang1/1000017061.md), [TableVersWord](../WDLang1/3074036.md), [TableVersXML](../WDLang1/3074037.md), [TableVersPressePapier](../WDLang1/1000017062.md)).




## RichEdit (Propriété)

***En anglais : RichEdit***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **RichEdit** permet de gérer le type RTF (Rich Text Format) : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) d'un champ d'une fenêtre.
	**Rappel** : Le type RTF est une option des champs de saisie et des champs Libellé. Cette option est située dans l'onglet "Détail" de la description du champ.

- d'un champ Libellé d'un état.
	**Rappel** : Le type RTF est une option des champs Libellé. Cette option est située dans l'onglet "Général" de la description du champ.




Il est possible de : 

- savoir si le champ est de type RTF. 

- modifier le type RTF d'un champ.







<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Si le champ Libellé "LIB_Nom" est de type RTF, il devient visible
SI LIB_Nom.RichEdit = Vrai ALORS
	LIB_Nom.Visible = Vrai
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Savoir si le champ est de type RTF

`<Résultat> = <Champ manipulé>.RichEdit`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'élément spécifié est de type RTF, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler : 

- Champ de saisie ou champ Libellé dans une fenêtre. 

- Champ Libellé dans un état.





<a name="SYNTAXE2"></a>

### Modifier le type RTF d'un champ

`<Champ manipulé>.RichEdit = <Type>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler : 

- Champ de saisie ou champ Libellé dans une fenêtre. 

- Champ Libellé dans un état.




**`<Type> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'élément spécifié doit être de type RTF, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png) 

### Manipuler la valeur d'un champ RTF
<a name="manipuler_valeur_champ_rtf_ELTPARAGRAPHE000102"></a>

- Si l'affectation (ou la lecture) est directe ou réalisée avec la propriété [Valeur](../Proprietes/2510130.md) : la valeur du champ de type RTF est au format RTF (cette valeur contient des balises RTF).

- Si l'affectation (ou la lecture) est réalisée avec la fonction [RTFVersTexte](../WDLang1/3025032.md) : la valeur du champ de type RTF n'est pas au format RTF (cette valeur ne contient pas de balises RTF).



<a name="NOTE0_2"></a>


### Limites
<a name="limites_ELTPARAGRAPHE000116"></a>Dans une fenêtre, la propriété **RichEdit** s'applique uniquement aux champs suivants :

- champs de saisie,

- champs Libellé. 

- colonne de table de type texte. 


Dans un état, la propriété **RichEdit** s'applique uniquement aux champs Libellé.

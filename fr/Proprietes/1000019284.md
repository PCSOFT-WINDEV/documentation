


## ScrollAuDoigt (Propriété)

***En anglais : ScrollWithFinger***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **ScrollAuDoigt** permet de : 

- Savoir si le contenu d'un champ peut être déplacé avec le doigt ("scroll au doigt").

- Autoriser ou non un champ à être manipulé directement avec le doigt. 




![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Cette propriété correspond à l'option "Déplacement au doigt" disponible dans l'onglet "Détail" des champs Liste, Table et Zone répétée.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI INT_InterfaceTactile = Vrai ALORS
	TABLE_TableClient.ScrollAuDoigt = Vrai
SINON
	TABLE_TableClient.ScrollAuDoigt = Faux
FIN
```
<a name="Exemple2"></a>

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Savoir si un champ permet le déplacement avec le doigt

`<Résultat> = <Champ manipulé>.ScrollAuDoigt`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le champ permet le déplacement avec le doigt, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Champ manipulé> : (Nom de champ)`**

Nom du champ manipulé. Ce champ peut être : 

- un champ Liste.

- un champ Table.

- un champ Zone répétée. 





<a name="SYNTAXE2"></a>

### Permettre le déplacement avec le doigt ou non

`<Champ manipulé>.ScrollAuDoigt = <Défilement au doigt>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ manipulé. Ce champ peut être : 

- un champ Liste.

- un champ Table.

- un champ Zone répétée. 




**`<Défilement au doigt> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si champ autorise le défilement au doigt, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Limite
<a name="limite_ELTPARAGRAPHE000109"></a>![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) La propriété **ScrollAuDoigt** peut uniquement être utilisée sur les champs Liste, Table et Zone répétée. 
<a name="NOTE0_2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 

### Cas particuliers
<a name="cas_particuliers_ELTPARAGRAPHE000135"></a>Sur certains systèmes, un défilement au doigt automatique (fourni par le système) peut être actif même si le défilement au doigt du champ est désactivé.

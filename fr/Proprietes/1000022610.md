


## TypeInterligne (Propriété)

***En anglais : InterlineType***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **TypeInterligne** permet de connaître et de modifier le type d'interligne utilisé. Cette propriété est utilisable sur :

- les variables de type [docParagraphe](../WDLang1/1000022483.md).

- les variables de type [docNiveauNumérotation](../WDLang1/1000022799.md).

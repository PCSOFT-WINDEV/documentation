
## Alias (Propriété)

***En anglais : Alias***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Alias** permet de connaître l'alias d'un élément. Cette propriété est utilisable sur : 

- un élément WINDEV/WEBDEV/WINDEV Mobile : 

	- pour connaître ou de modifier l'alias d'une fenêtre (WINDEV). 

	- pour connaître ou de modifier l'alias d'un volet d'onglet dynamique (WINDEV)

	- pour connaître le nom interne HTML / Javascript d'un champ (WEBDEV).


 Pour plus de détails, consultez l'aide de la propriété [Alias](../Proprietes/2510004.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [mongoGridFSInfoFichier](../WDLang4/1000022417.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [docStyle](../WDLang1/1000022486.md). 

- une variable de type [Description d'Option](../WDLang1/1000023072.md). 

- une variable de type [Description de valeur](../WDLang1/1000023061.md). 

- une variable de type [YAML](../Motscles/1000024438.md).

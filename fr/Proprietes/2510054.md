


## CadrageVertical (Propriété)

***En anglais : VerticalAlignment***
	



<a name="XPresentation"></a>
<a name="Presentation"></a>
Le cadrage vertical d'un champ consiste à définir la position du texte dans le champ par rapport aux bords du champ. Ainsi, il est possible de cadrer le contenu d'un champ de saisie par rapport aux bords de la zone de saisie. 

Trois cadrages différents sont à votre disposition : 

- Cadrage en haut<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Cardage_Haut.gif)


- Cadrage au milieu<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Cadrage_Milieu.gif)


- Cadrage en bas<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Cadrage_Bas.gif)




<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **CadrageVertical** permet de connaître et de modifier le cadrage vertical d'un champ.

Le cadrage initial du champ est défini lors de la description du champ.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Si le champ est cadré en haut, on le cadre en bas
SI LIB_Libellé.CadrageVertical = cvHaut ALORS
	LIB_Libellé.CadrageVertical = cvBas
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le cadrage vertical d'un champ

`<Cadrage actuel> = <Champ manipulé>.CadrageVertical`
---

**`<Cadrage actuel> : (Constante de type entier)`**

Cadrage vertical actuel du champ. Les valeurs possibles sont les suivantes :



|   |   |
| --- | --- |
| *cvBas* | le texte est placé en bas du champ |
| *cvHaut* | le texte est placé en haut du champ |
| *cvMilieu* | le texte est placé au milieu du champ |




**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler.  


<a name="SYNTAXE2"></a>

### Modifier le cadrage vertical d'un champ

`<Champ manipulé>.CadrageVertical = <Nouveau cadrage>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler.

**`<Nouveau cadrage> : (Constante de type entier)`**

Nouveau cadrage du champ. Les valeurs possibles sont les suivantes : 



|   |   |
| --- | --- |
| *cvBas* | le texte est placé en bas du champ |
| *cvHaut* | le texte est placé en haut du champ |
| *cvMilieu* | le texte est placé au milieu du champ |

## Remarques


### Cadrage horizontal
<a name="cadrage_horizontal_ELTPARAGRAPHE000191"></a>Pour définir le cadrage horizontal du champ, utilisez la propriété [CadrageHorizontal](../Proprietes/2510023.md).
<a name="NOTE0_2"></a>


### Limitations
<a name="limitations_ELTPARAGRAPHE000201"></a>![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Dans une fenêtre, la propriété **CadrageVertical** permet de connaître et de modifier le cadrage vertical des champs de type :

- Libellé

- Interrupteur ou sélecteur

- Image

- Colonne de table ou de table hiérarchique

- Liste

- Combo

- Code-barres

- Graphe

- Champ Notation

- Organigramme


![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Dans un état, la propriété **CadrageVertical** permet de connaître et de modifier le cadrage vertical des champs de type :

- Champ Libellé.

- Champ Calculé.

- Champ prédéfini.

- Champ Graphe.

- Groupe de champs : Cette propriété s'appliquera uniquement aux champs du groupe gérant le cadrage vertical.


![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) La propriété **CadrageVertical** s'applique uniquement aux champs suivants :

- Libellé.

- Image.

- Interrupteur.

- Sélecteur.

- Combo.


## Police

***En anglais : ***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le mot-clé **Police** peut correspondre : 

- à la variable de type [Police](../Motscles/1514045.md) qui permet de définir les caractéristiques d'une police. 

- à la propriété **Police** qui permet de connaître et de modifier la police utilisée dans les champs d'une page, d'une fenêtre ou d'un état. Pour plus de détails, consultez [Police sur les champs](../Proprietes/2510095.md).

- à la propriété **Police** disponible sur : 

	- une variable de type [xlsCellule](../WDLang5/1000017472.md). 

	- une variable de type [Filigrane](../WDLang5/1000019922.md).

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [picTexte](../WDLang1/1000025178.md).

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [diagTexte](../WDLang1/1410088393.md).

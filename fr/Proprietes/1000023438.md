
## ActiveDirectory (Propriété)

***En anglais : ActiveDirectory***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **ActiveDirectory** permet de gérer une connexion avec authentification à un serveur HFSQL par Active Directory. Il est possible de :

- Savoir si la connexion utilise l'authentification par Active Directory.

- Définir si la connexion utilise l'authentification par Active Directory.




**Remarque** : La connexion peut correspondre :

- soit à une connexion décrite sous l'éditeur d'analyses,

- soit à une connexion décrite avec les fonctions [HConnecte](../WDLang4/3044263.md), [HDécritConnexion](../WDLang4/3044205.md) ou [HOuvreConnexion](../WDLang4/3044107.md).

- soit à une connexion décrite avec une variable de type [Connexion](../WDLang4/1514073.md).





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
cnx_sso est une Connexion
cnx_sso.Provider = hAccèsHFClientServeur
cnx_sso.Serveur = "srv1.mondomaine.com"
cnx_sso.ActiveDirectory = Vrai
HOuvreConnexion(cnx_sso)
```



```wl
// Forcer l'utilisation d'un autre compte de domaine
cnx est une Connexion
cnx.Provider = hAccèsHFClientServeur
cnx.Serveur = "srv1.mondomaine.com"
cnx.ActiveDirectory = Vrai
cnx.Utilisateur = "Nom_compte@mondomaine.com"
cnx.MotDePasse = "xxxxxx"
HOuvreConnexion(cnx)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Savoir si la connexion utilise l'authentification par Active Directory

`<Résultat> = <Connexion>.ActiveDirectory`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la connexion utilise l'authentification par Active Directory, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 




**`<Connexion> : (Chaîne de caractères ou variable de type Connexion)`**

Nom de la connexion manipulée. Ce nom a été défini :

- soit sous l'éditeur d'analyses,

- soit avec les fonctions [HDécritConnexion](../WDLang4/3044205.md) ou [HOuvreConnexion](../WDLang4/3044107.md),

- soit avec le type [Connexion](../WDLang4/1514073.md).  





<a name="SYNTAXE2"></a>

### Définir si la connexion utilise l'authentification par Active Directory

`<Connexion>.ActiveDirectory = <Gestion Active Directory>`
---

**`<Connexion> : (Chaîne de caractères ou variable de type Connexion)`**

Nom de la connexion manipulée. Ce nom a été défini :

- soit sous l'éditeur d'analyses,

- soit avec les fonctions [HDécritConnexion](../WDLang4/3044205.md) ou [HOuvreConnexion](../WDLang4/3044107.md),

- soit avec le type [Connexion](../WDLang4/1514073.md).




**`<Gestion Active Directory> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la connexion utilise l'authentification par Active Directory, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.

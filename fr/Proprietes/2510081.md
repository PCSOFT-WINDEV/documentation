


## NbLignesParPage (Propriété)

***En anglais : NbLinesPerPage***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **NbLignesParPage** permet de :

- Connaître et modifier le nombre maximum de lignes par page d'un champ Table.

- Connaître et modifier le nombre maximum de répétitions par page d'un champ Zone répétée. 
	Attention : si le champ Zone répétée est multicolonne, la propriété **NbLignesParPage** correspond au nombre total maximum d'occurrences affichées dans le champ Zone répétée (nombre de colonnes multiplié par le nombre de lignes).




**Rappel** : Le nombre de lignes par page correspond au nombre maximum de lignes d'un champ Table ou Zone répétée pouvant être affichées sur une page. Cette option permet d'optimiser la taille des pages. Ce nombre est défini dans la description du champ :

- Pour les champs Zone répétée : onglet "Général" de la description du champ.

- Pour les champs Table : onglet "Détail" de la description du champ.





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affiche 20 lignes du champ Zone répétée sur une page
ZR_ZoneRépétée1.NbLignesParPage = 20
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le nombre maximum de lignes (ou de répétitions) par page

`<Nombre de lignes> = <Champ manipulé>.NbLignesParPage`
---

**`<Nombre de lignes> : (Entier)`**

Nombre maximum de lignes actuellement défini.

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ manipulé : champ Table ou champ Zone répétée.  


<a name="SYNTAXE2"></a>

### Modifier le nombre maximum de lignes (ou de répétitions) par page

`<Champ manipulé>.NbLignesParPage = <Nouveau nombre de lignes>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ manipulé : champ Table ou champ Zone répétée. 

**`<Nouveau nombre de lignes> : (Entier)`**

Nombre maximum de lignes par page. Ce nombre ne doit pas être égal à 0.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Il n'est pas nécessaire d'appeler la fonction [TableAffiche](../WDLang1/3074003.md) ou [ZoneRépétéeAffiche](../WDLang2/3083001.md) pour prendre en compte la modification du nombre maximal de lignes par page.

- Cette propriété ne fonctionne pas si le champ Zone répétée et/ou le champ Table est en mode Ajax.

- Cette propriété n'est pas disponible sur les champs Table hiérarchique.

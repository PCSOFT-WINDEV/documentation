


## VérifieOrthographe (Propriété)

***En anglais : CheckSpell***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **VérifieOrthographe** permet d'activer ou non la vérification automatique des fautes d'orthographe. 

Les mots n'existant pas dans le dictionnaire sont automatiquement soulignés d'un trait rouge : 

- dans les champs de saisie. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) dans les colonnes des champs Table.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) dans les champs Traitement de texte.




**Remarque** : Cette propriété est identique à l'option "Vérifier l'orthographe" présente : 

- dans l'onglet "Détail" de la fenêtre de description des champs de saisie, 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) dans l'onglet "Détail" de la fenêtre de description des colonnes des champs Table.





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Active la vérification orthographique dans un champ de saisie
SAI_Texte.VérifieOrthographe = Vrai
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Savoir si la correction orthographique est activée

`<Résultat> = <Champ manipulé>.VérifieOrthographe`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la correction orthographique est activée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler : 

- champ de saisie, 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) colonne d'un champ Table, 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) champ Traitement de texte.





<a name="SYNTAXE2"></a>

### Activer ou non la correction orthographique

`<Champ manipulé>.VérifieOrthographe = <Activer / Désactiver>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler : 

- champ de saisie, 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) colonne d'un champ Table, 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) champ Traitement de texte.




**`<Activer / Désactiver> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la correction orthographique doit être activée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.

## Remarques
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) 

### Correcteur orthographique utilisé dans les champs de saisie et les colonnes d'un champ Table
<a name="correcteur_orthographique_utilise_dans_les_champs_saisie_les_colonnes_champ_table_ELTPARAGRAPHE000114"></a>Dans les champs de saisie et les colonnes d'un champ Table d'une application WINDEV, la propriété **VérifieOrthographe** utilise : 

- soit le dictionnaire de OpenOffice 2.0. 
	Pour plus de détails sur l'installation de ce logiciel et de ses dictionnaires, consultez [Vérification d'orthographe avec OpenOffice](../Editeurs/1013146.md).

- soit le correcteur intégré.
	Dans ce cas : 

	- les dictionnaires français et anglais sont automatiquement copiés (avec leur licence respectives) dans le répertoire de l'exécutable (option paramétrable dans la [fenêtre de description du projet, onglet "Options"](../Editeurs/9500227.md)). 

	- les dictionnaires français et anglais sont chargés par défaut. 

	- pour utiliser d'autres dictionnaires, il est nécessaire de les associer à une langue et de les charger en mémoire avec la fonction [DictionnaireCharge](../WDLang1/1000022958.md). 




- Pour plus de détails, consultez [Vérification d'orthographe basée sur les dictionnaires Hunspell](../WDChamp/1410087030.md).




La fonction [ParamètreOrthographe](../WDLang1/3025052.md) permet de : 

- choisir le correcteur orthographique utilisé. 

- activer le correcteur orthographique pour tous les champs de saisie de l'application. 

- paramétrer la langue utilisée (dans le cas d'une correction orthographique via OpenOffice). 




Sur les mots non reconnus, le menu contextuel par défaut de WINDEV propose les options suivantes :

- Remplacement du mot par un mot valide (les 8 premières propositions de OpenOffice ou du correcteur intégré sont proposées).

- Ajout du mot dans le dictionnaire.

### Correcteur orthographique utilisé dans les champs Traitement de texte
<a name="correcteur_orthographique_utilise_dans_les_champs_traitement_texte_ELTPARAGRAPHE000169"></a>Dans les champs Traitement de texte, la propriété **VérifieOrthographe** utilise une vérification orthographique basée sur les dictionnaires au format Hunspell.

Par défaut, si votre projet contient un champ Traitement de texte : 

- les dictionnaires français et anglais sont automatiquement copiés (avec leur licence respectives) dans le répertoire de l'exécutable (option paramétrable dans la [fenêtre de description du projet, onglet "Options"](../Editeurs/9500227.md)). 

- les dictionnaires français et anglais sont chargés par défaut. 




Pour utiliser d'autres dictionnaires, il est nécessaire de les associer à une langue et de les charger en mémoire avec la fonction [DictionnaireCharge](../WDLang1/1000022958.md). 

Pour plus de détails, consultez [Vérification d'orthographe basée sur les dictionnaires Hunspell](../WDChamp/1410087030.md).

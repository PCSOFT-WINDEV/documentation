
## Zoom (Propriété)

***En anglais : Zoom***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Zoom** permet de connaître et de modifier le zoom appliquer à un élément. Cette propriété est utilisable sur : 

- un champ ou une fenêtre. Pour plus de détails, consultez [Zoom](../Proprietes/1000017201.md). 

- une variable de type [pdfParamètre](../WDLang6/1410087857.md).

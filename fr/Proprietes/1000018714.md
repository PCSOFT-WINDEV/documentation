
## AccuséRéception (Propriété)

***En anglais : ReturnReceipt***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **AccuséRéception** permet de connaître et de modifier la demande d'accusé de réception d'un email. 

Cette propriété est utilisable avec :

- la structure [Email](../WDLang3/3032029.md).

- une variable de type [Email](../WDLang3/1000018713.md).

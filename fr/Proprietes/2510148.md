


## SourceDescription (Propriété)

***En anglais : SourceDescription***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **SourceDescription** permet de connaître le nom de l'élément utilisé comme source de données d'une série.

**Remarques** :

- La fonction [grSourceSérie](../WDLang3/3042067.md) permet de définir la source de données d'une série.

- La propriété [Source](../Proprietes/2510147.md) permet de connaître le type de source utilisé (colonne de table, liste, ...).











<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI GRF_MonGraphe[3].Source = grColonneTable ET ...
	GRF_MonGraphe[3].SourceDescription = "MaColonne" ALORS
	// Change la source de données
	grSourceSérie(GRF_MonGraphe, 3, grListe, "MaListe")
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le nom de l'élément source de données d'une série

`<Résultat> = <Série>.SourceDescription`
---

**`<Résultat> : (Chaîne de caractères)`**

Nom de l'élément source de données. Ce nom peut correspondre :

- à un nom de colonne de table, si la source de données est de type "Colonne de table".

- à un nom de procédure WLangage, si la source de données est une procédure WLangage.

- à nom de champ Liste si la source de données est une liste.

- à une liste de valeurs, si la source de données est définie par programmation.

- au nom d'une rubrique (sous la forme &lt;Nom fichier&gt;.&lt;Nom Rubrique&gt;), si la source de données est la rubrique d'un fichier de données.

- au nom d'un tableau WLangage, si la source de données est un tableau WLangage.




Le type de la source de données peut être connu par la propriété [Source](../Proprietes/2510147.md).

**`<Série> : (Chaîne de caractères)`**

Série à manipuler. Utilisez la notation : *&lt;Nom du graphe&gt;[&lt;Numéro de la série&gt;]*.

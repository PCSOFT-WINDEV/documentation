
## TexteSaisieObligatoire (Propriété)

***En anglais : MandatoryInputMessage***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **TexteSaisieObligatoire** permet de connaître et de définir le texte qui sera affiché si le champ est vide (avec la propriété [SaisieObligatoire](../Proprietes/1000026174.md) activée).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SAI_DateNaissance.TexteSaisieObligatoire = ...
	"Donner votre date de naissance, on vous souhaitera votre anniversaire"
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître le texte affiché en cas de saisie obligatoire

`<Résultat> = <Champ manipulé>.TexteSaisieObligatoire`
---

**`<Résultat> : (Chaîne de caractères)`**



- Message d'erreur utilisé en cas de saisie obligatoire (propriété [SaisieObligatoire](../Proprietes/1000026174.md) à <u><u><u><u>Vrai</u></u></u></u>), 

- Chaîne vide si le message d'erreur par défaut est utilisé ("Saisie obligatoire"). 




**`<Champ manipulé> : (Nom de champ)`**

Nom du champ manipulé. Ce champ peut être : 

- un champ de saisie, 

- un champ Interrupteur, 

- un champ Sélecteur, 

- un champ Combo, 

- un champ Notation.





<a name="SYNTAXE2"></a>

### Modifier le texte affiché en cas de saisie obligatoire

`<Champ manipulé>.TexteSaisieObligatoire = <Message>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ manipulé. Ce champ peut être : 

- un champ de saisie, 

- un champ Interrupteur, 

- un champ Sélecteur, 

- un champ Combo, 

- un champ Notation.




**`<Message> : (Chaîne de caractères)`**



- Message d'erreur utilisé en cas de saisie obligatoire (propriété [SaisieObligatoire](../Proprietes/1000026174.md) à <u><u><u><u>Vrai</u></u></u></u>), 

- Chaîne vide si le message d'erreur par défaut est utilisé ("Saisie obligatoire").

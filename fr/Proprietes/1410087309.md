
## TailleFichier (Propriété)

***En anglais : FileSize***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **TailleFichier** permet de connaître la taille des fichiers. Cette propriété est disponible sur : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [mongoBaseInfo](../WDLang4/1000022412.md).




## SensSélection (Propriété)

***En anglais : SelectionDirection***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **SensSélection** permet de connaître et modifier le sens de la sélection (gauche à droite ou droite à gauche) dans un champ Traitement de Texte. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
T est un entier = 30
POUR i = TT_MonDoc.Curseur _A_ TT_MonDoc.Curseur + TT_MonDoc.LongueurSélection
	f est docFragment = TT_MonDoc[i SUR 1]
	f.Style.PoliceTaille = T
	SI TT_MonDoc.SensSélection = 1 ALORS
		T += 5
	SINON
		T -= 5
	FIN
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître le sens de la sélection dans un champ Traitement de texte

`<Résultat> = <Champ Traitement de texte>.SensSélection`
---

**`<Résultat> : (Entier)`**

Sens de la sélection :

- 1 : La sélection a été réalisée dans le sens de l'écriture (de gauche à droite). 

- 2 : La sélection a été réalisée dans le sens inverse de l'écriture (de droite à gauche). 




**`<Champ Traitement de texte> : (Nom de champ)`**

Nom du champ Traitement de texte à manipuler.


<a name="SYNTAXE2"></a>

### Modifier le sens de la sélection dans un champ Traitement de texte

`<Champ Traitement de texte>.SensSélection = <Nouvelle valeur>`
---

**`<Champ Traitement de texte> : (Nom de champ)`**

Nom du champ Traitement de texte à manipuler.

**`<Nouvelle valeur> : (Entier)`**

Nouveau sens de la sélection :

- 1 : La sélection a été réalisée dans le sens de l'écriture (de gauche à droite). 

- 2 : La sélection a été réalisée dans le sens inverse de l'écriture (de droite à gauche).

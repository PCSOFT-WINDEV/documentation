
## DateModification (Propriété)

***En anglais : DateModified***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **DateModification** permet de connaître la date de modification d'un élément. Cette propriété est utilisable sur :

- une variable de type [gglContact](../WDLang5/1000017434.md). 

- une variable de type [zipFichierArchivé](../WDLang3/1000019044.md).

- une variable de type [hdfsInformation](../WDLang4/1000021973.md).

- une variable de type [pdfDocument](../WDLang6/1000024909.md).

- une variable de type [pdfPièceJointe](../WDLang6/1000024899.md).

- une variable de type [FichierDisque](../WDLang1/1410088883.md).




## Adresse (Propriété)

***En anglais : Address***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Adresse** permet de :

- Connecter le champ Conférence (et le flux associé) avec un correspondant. Cette propriété est équivalente à la fonction [FluxConnecte](../WDLang3/3039008.md).

- Connaître l'adresse correspondant à la dernière demande de connexion sortante.




A partir de la version 20, les champs Conférence sont supportés sous Windows Vista et supérieur. Si vous souhaitez utiliser le champ Conférence avec une version antérieure de Windows : 

1. Affichez la fenêtre de description du champ. 

2. Dans l'onglet "UI", cochez l'option "Utiliser les API TAPI 3.x / Windows XP
(pour compatibilité)".

3. Validez. 



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connecter le flux à un correspondant
CONF_Conférence1.Adresse = "123.101.7.56"
SI CONF_Conférence1 <> 0 ALORS Info("La communication est établie")
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître l'adresse correspondante à la dernière connexion

`<Résultat> = <Champ Conférence>.Adresse`
---

**`<Résultat> : (Chaîne de caractères)`**

Adresse de la dernière demande de connexion sortante. Cette adresse peut être :

- Une adresse IP au format XXX.XXX.XXX.XXX (par exemple, 125.5.110.100).

- Une adresse IP contenant le nom du serveur (par exemple, pop3.freesbee.fr).




**`<Champ Conférence> : (Nom de champ)`**

Nom du champ Conférence associé au flux.  


<a name="SYNTAXE2"></a>

### Connecter le flux à un correspondant

`<Champ Conférence>.Adresse = <Adresse du correspondant>`
---

**`<Champ Conférence> : (Nom de champ)`**

Nom du champ Conférence associé au flux.

**`<Adresse du correspondant> : (Chaîne de caractères)`**

Nouvelle adresse du correspondant. Cette adresse peut être :

- Une adresse IP au format XXX.XXX.XXX.XXX (par exemple, 125.5.110.100).

- Une adresse IP contenant le nom du serveur (par exemple, pop3.freesbee.fr). Cette syntaxe est conseillée.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Pour savoir si la connexion a réussi, vous pouvez :

- soit vérifier la valeur de la variable [ErreurDétectée](../WDLang1/3087001.md).

- soit vérifier la valeur du champ Conférence (propriété [Valeur](../Proprietes/2510130.md)). Si &lt;Champ Conférence&gt;.Valeur = 0, alors la connexion a échoué.

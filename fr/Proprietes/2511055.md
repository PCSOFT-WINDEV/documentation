


## LargeurEtiquette (Propriété)

***En anglais : LabelWidth***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **LargeurEtiquette** permet de :

- Connaître la largeur des étiquettes d'un état de type "Étiquette".

- Modifier la largeur des étiquettes d'un état de type "Étiquette".




**Remarque** : La largeur des étiquettes a été définie lors de la création de l'état au format étiquette. Cette option peut être connue et/ou modifiée sous l'éditeur d'états dans l'option "Format des étiquettes" dans le menu contextuel de l'état.<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Largeuretiquette.gif)



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Augmente la largeur des étiquettes de l'état "ETAT_EtiqClient"
ETAT_EtiqClient.LargeurEtiquette = ETAT_EtiqClient.LargeurEtiquette + 35
// Est équivalent à : ETAT_EtiqClient.LargeurEtiquette += 35
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître la largeur des étiquettes

`<Résultat> = <Etat manipulé>.LargeurEtiquette`
---

**`<Résultat> : (Réel)`**

Largeur (en millimètres) des étiquettes de l'état spécifié.

**`<Etat manipulé> : (Nom d'état)`**

Nom de l'état de type "Étiquette" à manipuler.  


<a name="SYNTAXE2"></a>

### Modifier la largeur des étiquettes

`<Etat manipulé>.LargeurEtiquette = <Nouvelle largeur>`
---

**`<Etat manipulé> : (Nom d'état)`**

Nom de l'état de type "Étiquette" à manipuler.

**`<Nouvelle largeur> : (Réel)`**

Nouvelle largeur (en millimètres) des étiquettes de l'état spécifié.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Etat de type "Étiquette"
<a name="etat_type_etiquette_ELTPARAGRAPHE000061"></a>

- Pour connaître ou modifier la hauteur des étiquettes d'un état de type "Étiquette", utilisez la propriété [HauteurEtiquette](../Proprietes/2511069.md).

- Pour connaître ou modifier l'espacement horizontal entre les étiquettes d'un état de type "Étiquette", utilisez la propriété [EspaceH](../Proprietes/2511067.md).

- Pour connaître ou modifier l'espacement vertical entre les étiquettes d'un état de type "Étiquette", utilisez la propriété [EspaceV](../Proprietes/2511065.md).

- Pour connaître ou modifier le nombre d'étiquettes imprimées sur la largeur d'une page, utilisez la propriété [NbEtiquette](../Proprietes/2511050.md).



<a name="NOTE0_2"></a>


### Limite
<a name="limite_ELTPARAGRAPHE000083"></a>La propriété **LargeurEtiquette** s'applique uniquement aux états de type "Étiquette".

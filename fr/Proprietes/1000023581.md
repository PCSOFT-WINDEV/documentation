
## Mode (Propriété)

***En anglais : Mode***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Mode** permet de gérer : 

- le nombre de noeuds minimum devant avoir acquitté le traitement pour considérer l'écriture effectuée (variable de type [mongoOptionEcriture](../WDLang4/1000022597.md)). 

- le mode de gestion des sections multicolonnes (variable de type [docFormatMulticolonne](../WDLang1/1000023580.md)).

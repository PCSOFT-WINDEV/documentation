
## ImageFond (Propriété)

***En anglais : BackgroundImage***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **ImageFond** permet de connaître et de modifier l'image de fond d'un élément. Cette propriété est utilisable sur :

- une fenêtre ou un champ (bouton, jauge, potentiomètre, champ de saisie HTML). Pour plus de détails, consultez [ImageFond](../Proprietes/2510056.md).

- une variable de type [OrgaElement](../WDLang1/1000019713.md).




## Texte (Propriété)

***En anglais : Text***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Texte** permet de savoir si une rubrique est une rubrique de type texte. **Cette propriété est disponible en lecture seulement**.

**Remarque** : Les rubriques considérées comme des rubriques de type texte sont les suivantes :

- chaîne de caractères,

- mémo texte,

- caractère.







<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// La rubrique Client.Téléphone est-elle de type texte ?
ResTexte = Client.Téléphone.Texte
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Fichier de données>.<Rubrique>.Texte`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la rubrique est de type texte, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Fichier de données> : (Chaîne de caractères)`**

Nom logique du fichier de données manipulé.

**`<Rubrique> : (Chaîne de caractères)`**

Nom de la rubrique manipulée.

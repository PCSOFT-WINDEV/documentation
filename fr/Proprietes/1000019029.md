
## NombreExécution (Propriété)

***En anglais : NumberExecution***
	

<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **NombreExécution** permet de connaître et de modifier le nombre d'exécutions d'une tâche. Cette propriété est utilisable sur : 

- une variable de type [hOptimisationPlanifiée](../WDLang4/1000018854.md). 

- une variable de type [hTâchePlanifiée](../WDLang4/1000017448.md).

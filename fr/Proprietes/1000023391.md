


## Disposition (Propriété)

***En anglais : Layout***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Disposition** permet de connaître la disposition des cellules dans un champ Disposition. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Elargir le libellé si le champ Disposition affiche une cellule par colonne
SI DISP_MaDisposition.Disposition = dispEmpilementVertical ALORS
	DISP_MaDisposition.LIB_Nom.Largeur = 320
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître la disposition des cellules

`<Résultat> = <Champ Disposition>.Disposition`
---

**`<Résultat> : (Constante de type Entier)`**

Disposition des cellules : 



|   |   |
| --- | --- |
| *dispEmpilementHorizontal* | Le champ Disposition affiche une cellule par ligne, les unes sous les autres. |
| *dispEmpilementVertical* | Le champ Disposition affiche une cellule par colonne, les unes à côté des autres. |





**`<Champ Disposition> : (Nom de champ)`**

Nom du champ Disposition manipulé.

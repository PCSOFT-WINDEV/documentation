
## CxAccès (Propriété)

***En anglais : CtAccess***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **CxAccès** permet de gérer le type d'accès au fichier lors de la connexion OLE DB à une table spécifique. Il est possible de :

- Connaître le type d'accès au fichier de données utilisé pour la connexion à une table par OLE DB.
	Le type d'accès peut correspondre :

	- soit au type d'accès défini sous l'éditeur d'analyses,

	- soit au type d'accès spécifié dans les fonctions [HConnecte](../WDLang4/3044263.md), [HDécritConnexion](../WDLang4/3044205.md) ou [HOuvreConnexion](../WDLang4/3044107.md).
			Cette propriété peut être utilisée pour un fichier de données ou une connexion défini sous l'éditeur d'analyses ou défini par programmation.




- Définir le type d'accès au fichier de données utilisé pour la connexion à une table par OLE DB (lors de la description par programmation du fichier de données uniquement).
	Dans ce cas, cette propriété est équivalente à la propriété [CxAccèsDécrit](../Proprietes/2512048.md). 




**Rappel** : La fonction [HConnecte](../WDLang4/3044263.md) permet de redéfinir un ou plusieurs paramètres de la connexion OLE DB sur une table. Par défaut, les caractéristiques de la connexion OLE DB sont définies :

- soit sous l'éditeur d'analyses lors de la description du fichier de données,

- soit en programmation lors de la description du fichier de données.



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connaître le type d'accès du fichier de données Client
ResTypeAccès = Client.CxAccès
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le type d'accès au fichier de données par une connexion

`<Type d'accès> = <Fichier de données ou Connexion>.CxAccès`
---

**`<Type d'accès> : (Constante)`**

Renvoie le type d'accès au fichier de données.


|   |   |
| --- | --- |
| *hOLecture* | Le fichier de données est accessible uniquement en lecture. Seule la lecture des enregistrements est possible. Il n'est pas possible d'en ajouter ou d'en supprimer. |
| *hOLectureEcriture* | Le fichier de données est accessible en lecture et en écriture. Il est possible aussi bien de lire des enregistrement que d'écrire de nouveaux enregistrements. |



**`<Fichier de données ou Connexion> : (Chaîne de caractères)`**



- Nom du fichier de données manipulé. Ce nom a été défini sous l'éditeur d'analyses ou grâce au type [Description de Fichier](../WDLang4/1514065.md).

- Nom de la connexion manipulée. Ce nom a été défini sous l'éditeur d'analyses ou avec les fonctions [HDécritConnexion](../WDLang4/3044205.md) ou [HOuvreConnexion](../WDLang4/3044107.md).  





<a name="SYNTAXE2"></a>

### Définir le type d'accès au fichier par une connexion

`<Fichier de données ou Connexion>.CxAccès = <Type d'accès>`
---

**`<Fichier de données ou Connexion> : (Chaîne de caractères)`**



- Nom du fichier de données manipulé. Ce nom a été défini sous l'éditeur d'analyses ou grâce au type [Description de Fichier](../WDLang4/1514065.md).

- Nom de la connexion manipulée. Ce nom a été défini sous l'éditeur d'analyses ou avec les fonctions [HDécritConnexion](../WDLang4/3044205.md) ou [HOuvreConnexion](../WDLang4/3044107.md).




**`<Type d'accès> : (Constante)`**

Définit le type d'accès au fichier de données.


|   |   |
| --- | --- |
| *hOLecture* | Le fichier de données est accessible uniquement en lecture. Seule la lecture des enregistrements est possible. Il n'est pas possible d'en ajouter ou d'en supprimer. |
| *hOLectureEcriture* | Le fichier de données est accessible en lecture et en écriture. Il est possible aussi bien de lire des enregistrement que d'écrire de nouveaux enregistrements. |







<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Divers
<a name="divers_ELTPARAGRAPHE000112"></a>La propriété [CxAccèsDécrit](../Proprietes/2512048.md) permet de connaître et de modifier le type d'accès spécifié pour la connexion OLE DB lors de la description du fichier de données.
<a name="NOTE0_2"></a>


### Equivalence
<a name="equivalence_ELTPARAGRAPHE000122"></a>La propriété **CxAccès** permet de définir le type d'accès sur un fichier de données manipulé par une connexion. Cette propriété est conservée par compatibilité. Il est conseillé d'utiliser la propriété [Accès](../Proprietes/2512100.md). La propriété [Accès](../Proprietes/2512100.md) permet de définir le type d'accès sur les fichiers de données manipulés par une connexion.

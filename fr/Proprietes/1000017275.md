


## BoutonCalendrier (Propriété)

***En anglais : CalendarButton***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **BoutonCalendrier** permet de savoir et d'indiquer si un champ de saisie de type Date propose un bouton pour afficher un calendrier en popup.  <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=BoutonCalendrier.gif)



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affiche un bouton Calendrier si nécessaire
SI SAI_Date.BoutonCalendrier = Faux ALORS 
	SAI_Date.BoutonCalendrier = Vrai
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Savoir si un bouton Calendrier est affiché

`<Résultat> = <Champ de saisie>.BoutonCalendrier`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si un bouton Calendrier est affiché, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Champ de saisie> : (Nom de champ)`**

Nom du champ de saisie de type Date à manipuler.  


<a name="SYNTAXE2"></a>

### Afficher ou non un bouton Calendrier

`<Champ de saisie>.BoutonCalendrier = <Afficher>`
---

**`<Champ de saisie> : (Nom de champ)`**

Nom du champ de saisie de type Date à manipuler.

**`<Afficher> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si un bouton Calendrier doit être affiché, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.

## Remarques
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) 

### Fonctionnement du champ Calendrier
<a name="fonctionnement_champ_calendrier_ELTPARAGRAPHE000075"></a>Par défaut, en exécution, les champs de saisie de type Date proposent l'option "Calendrier". Cette option permet d'afficher un calendrier pour sélectionner une date.

Si la propriété **BoutonCalendrier** est utilisée pour afficher un bouton Calendrier dans le champ de saisie de type Date, l'option "Calendrier" n'est alors plus disponible dans le menu contextuel du champ Date.

**Remarque** : Il est possible de désactiver pour un projet ou un champ l'affichage du calendrier (via le bouton ou le menu contextuel) grâce à la fonction [FAADésactive](../WDLang1/1000022018.md).

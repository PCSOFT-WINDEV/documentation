
## Entête

***En anglais : Header***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Entête** permet de connaître et de modifier un entête. Cette propriété est utilisable sur :

- une variable de type [Email](../WDLang3/1000018713.md). 

- une variable de type [httpRequête](../WDLang3/1000021158.md).

- une variable de type [httpRéponse](../WDLang3/1000021165.md).

- une variable de type [restRequête](../WDLang3/1000021481.md).

- une variable de type [restRéponse](../WDLang3/1000021477.md).

- une variable de type [apiRequête](../WDLang3/1410088935.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [tblrSélection](../WDLang1/1000023498.md).

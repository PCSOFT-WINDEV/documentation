


## Connexion (Propriété)

***En anglais : Connection***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Connexion** permet de connaître et de modifier la connexion actuellement associée à un fichier de données.

Cette connexion a été définie :

- soit depuis l'éditeur d'analyses.

- soit grâce aux fonctions [HDécritConnexion](../WDLang4/3044205.md), [HOuvreConnexion](../WDLang4/3044107.md) ou [HChangeConnexion](../WDLang4/3044150.md).




Cette propriété peut être utilisée pour un fichier de données défini sous l'éditeur d'analyses ou défini par programmation.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connaître la connexion actuellement associée au fichier de données Client
Info(Client.Connexion)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître la connexion actuellement associée à un fichier de données

`<Connexion courante> = <Fichier de données>.Connexion`
---

**`<Connexion courante> : (Chaîne de caractères)`**



- Nom de la connexion actuellement associée au fichier de données spécifié. 

- Chaîne vide ("") si aucune connexion n'est actuellement associée au fichier de données spécifié.




**`<Fichier de données> : (Chaîne de caractères)`**

Nom du fichier de données manipulé. Ce nom a été défini sous l'éditeur d'analyses ou grâce au type [Description de Fichier](../WDLang4/1514065.md).  


<a name="SYNTAXE2"></a>

### Modifier la connexion actuellement associée à un fichier de données

`<Fichier de données>.Connexion = <Nouvelle connexion>`
---

**`<Fichier de données> : (Chaîne de caractères)`**

Nom du fichier de données manipulé. Ce nom a été défini sous l'éditeur d'analyses ou grâce au type [Description de Fichier](../WDLang4/1514065.md).

**`<Nouvelle connexion> : (Chaîne de caractères ou variable de type Connexion)`**



- Nom de la connexion à associer au fichier de données spécifié. 

- Chaîne vide ("") si aucune connexion n'est actuellement associée au fichier de données spécifié.

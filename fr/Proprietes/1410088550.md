
## PointDePassage (Propriété)

***En anglais : WayPoint***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **PointDePassage** permet de gérer les points de passage d'un lien. Cette propriété est utilisable sur : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) un champ Fil. Pour plus de détails, consultez l'aide de la propriété [PointDePassage](../Proprietes/1000021957.md). 

- une variable de type [diagLien](../WDLang1/1410088454.md).

- une variable de type [diagActionAutorisée](../WDLang1/1410088840.md).

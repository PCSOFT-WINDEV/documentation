
## Paramètre (Propriété)

***En anglais : Parameter***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Paramètre** permet de connaître et de modifier certains paramètres. Cette propriété est utilisable sur :

- une variable de type [hTâchePlanifiée](../WDLang4/1000017448.md).

- une variable de type [Description d'API](../WDLang1/1000019149.md).

- une variable de type [Description de TâcheParallèle](../WDLang1/1000021586.md).

- une variable de type [Description de Syntaxe](../WDLang1/1410089154.md).

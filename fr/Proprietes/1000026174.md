
## SaisieObligatoire (Propriété)

***En anglais : MandatoryInput***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **SaisieObligatoire** permet de connaître et modifier la caractéristique de saisie obligatoire d'un champ. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI m_bCompteCritique ALORS
	SAI_MotDePasse.SaisieObligatoire = Vrai
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Savoir si la saisie obligatoire est activée ou non sur un champ

`<Résultat> = <Champ manipulé>.SaisieObligatoire`
---

**`<Résultat> : ()`**



- <u><u><u><u>Vrai</u></u></u></u> si la saisie obligatoire est activée. Dans ce cas, le champ est considéré en erreur dans les cas suivants : 

	- Champ de saisie : Champ en erreur si le contenu est vide. 

	- Champ Combo en saisie : Champ en erreur si le contenu est vide. 

	- Champ Combo sans saisie : Champ en erreur si aucune ligne n'est sélectionnée. 

	- Champ Interrupteur : Champ en erreur si aucune option n'est cochée. 

	- Champ Sélecteur : Champ en erreur si aucune option n'est cochée. 

	- Champ Notation : Champ en erreur si aucune étoile n'est affichée. 




- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 




**`<Champ manipulé> : (Nom de champ)`**

Nom du champ manipulé. Ce champ peut être : 

- un champ de saisie, 

- un champ Interrupteur, 

- un champ Sélecteur, 

- un champ Combo, 

- un champ Notation. 





<a name="SYNTAXE2"></a>

### Activer ou non la saisie obligatoire sur un champ

`<Champ manipulé>.SaisieObligatoire = <Saisie obligatoire>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ manipulé. Ce champ peut être : 

- un champ de saisie, 

- un champ Interrupteur, 

- un champ Sélecteur, 

- un champ Combo, 

- un champ Notation. 




**`<Saisie obligatoire> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la saisie obligatoire est activée. Dans ce cas, le champ est considéré en erreur dans les cas suivants : 

	- Champ de saisie : Champ en erreur si le contenu est vide. 

	- Champ Combo en saisie : Champ en erreur si le contenu est vide. 

	- Champ Combo sans saisie : Champ en erreur si aucune ligne n'est sélectionnée. 

	- Champ Interrupteur : Champ en erreur si aucune option n'est cochée. 

	- Champ Sélecteur : Champ en erreur si aucune option n'est cochée. 

	- Champ Notation : Champ en erreur si aucune étoile n'est affichée. 




- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.


## DateDernierAccès (Propriété)

***En anglais : DateLastAccessed***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **DateDernierAccès** permet de connaître la date du dernier accès au fichier. Cette propriété est utilisable sur : 

- une variable de type [FichierDisque](../WDLang1/1410088883.md). 

- une variable de type [hdfsInformation](../WDLang4/1000021973.md). 

- une variable de type [zipFichierArchivé](../WDLang3/1000019044.md).


## FormatAffichage (Propriété)

***En anglais : DisplayFormat***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **FormatAffichage** permet de connaître et de modifier le format attendu pour l'affichage d'un libellé : texte simple, texte au format RTF ou Markdown.

Ce libellé peut correspondre : 

- au libellé d'un champ Libellé. 

- au libellé d'un champ de saisie.

- au libellé d'une colonne d'un champ Table.  

- au libellé d'une option de menu.  



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
LIB_Info.FormatAffichage = faMarkdown
LIB_Info = "Texte **gras** et *italique*"
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître le format attendu pour l'affichage dans le libellé d'un champ

`<Résultat> = <Champ manipulé>.FormatAffichage`
---

**`<Résultat> : (Constante de type Entier)`**

Format attendu par le champ Libellé : 



|   |   |
| --- | --- |
| *faMarkdown* | Texte au format Markdown. |
| *faRTF* | Texte au format RTF. |
| *faTexteSimple* | Texte sans format spécifique. |





**`<Champ manipulé> : (Nom de champ)`**

Nom du champ manipulé. Ce champ peut correspondre : 

- au nom d'un champ Libellé, 

- au nom d'un champ de saisie, 

- au nom d'une colonne d'un champ Table,

- au nom d'une option de menu.  





<a name="SYNTAXE2"></a>

### Modifier le format attendu pour l'affichage dans le libellé d'un champ

`<Champ manipulé>.FormatAffichage = <Nouveau format>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ manipulé. Ce champ peut correspondre : 

- au nom d'un champ Libellé, 

- au nom d'un champ de saisie, 

- au nom d'une colonne d'un champ Table,

- au nom d'une option de menu.     




**`<Nouveau format> : (Constante de type Entier)`**

Nouveau format attendu par le champ Libellé : 



|   |   |
| --- | --- |
| *faMarkdown* | Texte au format Markdown. |
| *faRTF* | Texte au format RTF. |
| *faTexteSimple* | Texte sans format spécifique. |

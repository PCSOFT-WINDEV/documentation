
## Propriété (Propriété)

***En anglais : Property***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Propriété** permet de connaître et de modifier les propriétés d'un élément. Cette propriété est utilisable sur :

- une variable de type [Définition](../WDLang1/1000019540.md).

- une variable de type [btleCaractéristique](../WDLang3/1000022060.md).

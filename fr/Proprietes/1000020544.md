
## Compression (Propriété)

***En anglais : Compression***
	

<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Compression** permet de connaître et de modifier le mode de compression des données. 

Cette propriété est utilisable sur :

- une connexion (HFSQL Client/Serveur, ...). Pour plus de détails, consultez [Compression](../Proprietes/1000017276.md).

- une variable de type [hDescriptionSauvegarde](../WDLang4/1000017455.md).

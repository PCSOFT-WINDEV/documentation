
## InfoTrafic (Propriété)

***En anglais : TrafficInfo***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **InfoTrafic** permet de : 

- savoir si les informations sur le trafic routier sont affichées dans le champ Carte. 

- afficher ou non les informations sur le trafic routier dans le champ Carte.



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affiche les informations sur le trafic
CARTE_Carte.InfoTrafic = Vrai
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Savoir si les informations sur le trafic sont affichées dans un champ Carte

`<Résultat> = <Champ Carte>.InfoTrafic`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si les informations concernant le trafic doivent être affichées, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 




**`<Champ Carte> : (Nom de champ)`**

Nom du champ Carte à manipuler.   


<a name="SYNTAXE2"></a>

### Afficher ou non les informations sur le trafic dans un champ Carte

`<Champ Carte>.InfoTrafic = <Info trafic>`
---

**`<Champ Carte> : (Nom de champ)`**

Nom du champ Carte à manipuler.

**`<Info trafic> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si les informations concernant le trafic doivent être affichées, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.

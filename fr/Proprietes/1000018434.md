
## NomComplet (Propriété)

***En anglais : FullName***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **NomComplet** permet de connaître le nom complet d'un élément. Cette propriété est utilisable sur :

- un champ, une fenêtre ou une option de menu. Pour plus de détails, consultez [NomComplet](../Proprietes/2514005.md).

- une variable de type [sfInfoUtilisateur](../WDLang5/1000018423.md).

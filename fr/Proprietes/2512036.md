
## FichierTransaction (Propriété)

***En anglais : TransactionFile***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **FichierTransaction** permet de savoir si un fichier de données est un fichier de transaction ou non.

**Rappel** : Un fichier de transaction permet de stocker les différentes opérations effectuées lors d'une transaction. Ces opérations sont ensuite soit validées, soit annulées. Le fichier de transaction est créé lors au début de la transaction par la fonction [HTransactionDébut](../WDLang4/3044002.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
EXTERNE GESCOM_TRS
HDéclareExterne("C:\WinDev\Exemples\GestCom\GESCOM_TRS.fic", "GESCOM_TRS")
SI GESCOM_TRS.FichierTransaction = Vrai ALORS
	Info("Vous manipulez un fichier de transaction.")
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Fichier de données>.FichierTransaction`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le fichier est un fichier de transaction, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 
	Remarque : Les fichiers décrits dans l'analyse ne sont pas des fichiers de transaction.




**`<Fichier de données> : (Chaîne de caractères)`**

Nom logique du fichier manipulé. Peut correspondre à un nom d'alias d'un fichier déclaré par la fonction [HDéclareExterne](../WDLang4/3044204.md).




## AxeSecondaire (Propriété)

***En anglais : SecondaryAxis***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **AxeSecondaire** permet de : 

- savoir si une série d'un graphe est placée sur l'axe secondaire. 

- placer une série d'un graphe sur l'axe secondaire.




**Remarque** : Cette propriété est équivalente à la fonction [grSérieAxeSecondaire](../WDLang3/1000019188.md).






<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Colore la série 2 du graphe GRF_MonGraphe si elle est placée sur l'axe secondaire
SI GRF_MonGraphe[2].AxeSecondaire = Vrai  ALORS
	GRF_MonGraphe[2].CouleurSérie = RougeClair
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Savoir si une série est placée sur l'axe secondaire

`<Résultat> = <Série manipulée>.AxeSecondaire`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la série est sur l'axe secondaire, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Série manipulée> : (Nom de champ)`**

Série à manipuler. Utilisez la notation : 

```txt
<Nom du graphe>[<Numéro de la série>]
```



<a name="SYNTAXE2"></a>

### Modifier la position d'une série

`<Série manipulée>.AxeSecondaire = <Nouvelle position>`
---

**`<Série manipulée> : (Nom de champ)`**

Série dont la position doit être modifiée. Utilisez la notation : 

```txt
<Nom du graphe>[<Numéro de la série>]
```


**`<Nouvelle position> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> pour placer la série sur l'axe secondaire, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Cette propriété est disponible uniquement avec les graphes possédant des axes. Elle ne fonctionne pas avec des graphes de type secteur/radar, ...

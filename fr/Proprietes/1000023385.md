
## CheminFichier (Propriété)

***En anglais : FilePath***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **CheminFichier** permet de connaître le chemin du fichier associé à un champ. Cette propriété est utilisable sur :

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) un champ bureautique présent dans une fenêtre (champ Tableur, champ Traitement de texte, champ Editeur d'images, champ Lecteur PDF). Pour plus de détails, consultez [CheminFichier](../Proprietes/1000023104.md).

- une variable de type [Document](../WDLang1/1000022461.md).

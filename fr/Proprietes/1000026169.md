
## RépertoireDeTravail (Propriété)

***En anglais : WorkingDirectory***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **RépertoireDeTravail** permet de : 

- connaître ou modifier le répertoire utilisé par le champ Editeur HTML pour stocker les images et feuilles CSS d'une page HTML.

- connaître le répertoire utilisé par le champ d'affichage HTML pour stocker les images et feuilles CSS d'une page HTML.






**Remarques** : 

- Champ Editeur HTML : lors de l'utilisation du champ, un répertoire de travail est automatiquement créé. Ce répertoire est supprimé à la fermeture de la fenêtre ou de l'application.

- Champ d'affichage HTML : lors de l'utilisation de la fonction [HTMLDepuisEmail](../WDLang1/1410087263.md), un répertoire de travail est automatiquement créé pour enregistrer les images associées à l'email. Ce répertoire est supprimé à la fermeture de la fenêtre ou de l'application.



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Copie une image dans le répertoire et insère l'image dans le document
fCopieFichier("logo.png", ComplèteRep(EDHTML_SansNom1.RépertoireDeTravail) + "logo.png")
EDHTML_SansNom1.Insère("<img src='logo.png'>")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître le répertoire utilisé par le champ pour stocker les fichiers 

`<Résultat> = <Champ manipulé>.RépertoireDeTravail`
---

**`<Résultat> : (Chaîne de caractères)`**



- Champ Editeur HTML : Répertoire utilisé par le champ Editeur HTML pour stocker les fichiers (images, feuilles CSS, ...). 

- Champ d'affichage HTML : Répertoire utilisé par le champ d'affichage HTML. Si l'URL associée au champ est un fichier, la propriété **RépertoireDeTravail** renvoie le répertoire correspondant, sinon elle renvoie chaine vide (""). 




**`<Champ manipulé> : (Nom de champ)`**

Nom du champ manipulé : 

- Champ Editeur HTML, 

- Champ d'affichage HTML. 





<a name="SYNTAXE2"></a>

### Modifier le répertoire utilisé par le champ Editeur HTML pour stocker les fichiers (images, feuilles CSS, ...)

`<Champ Editeur HTML>.RépertoireDeTravail = <Nouveau répertoire>`
---

**`<Champ Editeur HTML> : (Nom de champ)`**



- ![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png) Nom du champ Editeur HTML manipulé. 




**`<Nouveau répertoire> : (Chaîne de caractères)`**

Répertoire utilisé pour stocker : 

- ![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png) les fichiers du champ Editeur HTML (images, feuilles CSS, ...).






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Les images ou feuilles de styles CSS avec une URL de fichier (c'est-à-dire commençant par "file://") ne peuvent pas être affichées par défaut durant l'édition dans le champ Editeur HTML. 

- Pour afficher les images ou feuilles de styles CSS avec une URL de fichier (c'est-à-dire commençant par "file://"), ces fichiers doivent : 

	- être présents dans le répertoire spécifié par la propriété **RépertoireDeTravail**.

	- être relatives. Par exemple : &lt;img src='monLogo.png' ...




- Ce répertoire est initialisé avec un répertoire temporaire. Ce répertoire est supprimé à la fermeture de la fenêtre ou de l'application. Il est possible de le modifier pour les champs Editeur HTML avec la propriété **RépertoireDeTravail**. Dans ce cas, le répertoire spécifié n'est pas supprimé en fermeture.

- L'option du ruban du champ Editeur HTML qui permet d'insérer des images dans la page réalise automatiquement la copie des images dans ce répertoire.

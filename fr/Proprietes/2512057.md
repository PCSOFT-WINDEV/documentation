


## NbRubriqueMémo (Propriété)

***En anglais : NbMemoItem***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **NbRubriqueMémo** permet d'obtenir le nombre total de rubriques mémo présentes dans un enregistrement de :

- un fichier de données décrit sous l'éditeur d'analyses,

- un fichier de données décrit par programmation,

- une vue HFSQL,

- une requête.





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connaître le nombre de rubriques mémo présentes dans un enregistrement du fichier de données Client
ResNbRubriqueMémo = Client.NbRubriqueMémo
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Nombre de rubriques> = <Elément manipulé>.NbRubriqueMémo`
---

**`<Nombre de rubriques> : (Entier)`**

Nombre de rubriques mémo d'un enregistrement de l'objet spécifié.

**`<Elément manipulé> : (Chaîne de caractères)`**

Nom de l'élément dont on veut connaître le nombre de rubriques mémo pour un enregistrement :

- nom d'un fichier de données décrit sous l'éditeur d'analyses ou par programmation,

- nom d'une vue HFSQL,

- nom d'une requête.

<a name="NOTE0_1"></a>

## Remarques


### Statistiques sur un enregistrement 
<a name="statistiques_sur_enregistrement_ELTPARAGRAPHE000056"></a>

- La propriété [NbRubrique](../Proprietes/2512055.md) permet de connaître le nombre total de rubriques d'un enregistrement.

- La propriété [NbRubriqueClé](../Proprietes/2512056.md) permet de connaître le nombre total de rubriques clé d'un enregistrement.

- La propriété **NbRubriqueMémo** permet de connaître le nombre total de rubriques mémo d'un enregistrement.

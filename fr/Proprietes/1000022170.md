


## RTFAvecImagesEtTableaux (Propriété)

***En anglais : RTFWithImagesAndTables***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **RTFAvecImagesEtTableaux** permet de gérer la prise en charge des images et tableaux dans un champ Libellé de type RTF présent dans un état. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Activer la prise en charge des images et tableaux 
// dans le RTF du champ LIB_Descriptif
LIB_Descriptif.RTFAvecImagesEtTableaux = Vrai
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître le mode de gestion des images et tableaux en RTF

`<Résultat> = <Champ Libellé>.RTFAvecImagesEtTableaux`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si les images et tableaux sont gérées dans le champ Libellé de type RTF, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 




**`<Champ Libellé> : (Nom de champ)`**

Nom du champ Libellé à manipuler. Ce champ est présent dans un état. 


<a name="SYNTAXE2"></a>

### Modifier le mode de gestion des images et tableaux en RTF

`<Champ Libellé>.RTFAvecImagesEtTableaux = <Nouveau mode>`
---

**`<Champ Libellé> : (Nom de champ)`**

Nom du champ Libellé à manipuler. Ce champ est présent dans un état.

**`<Nouveau mode> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> pour gérer les images et tableaux dans le champ Libellé de type RTF, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Cette propriété permet de changer le mode de gestion des images et des tableaux pour un champ. Il est également possible de modifier globalement ce mode de gestion (c'est-à-dire pour tous les champs) en utilisant la fonction [iParamètre](../WDLang5/3046008.md) avec la constante *iRTFAvecImagesEtTableaux*.

- La fonction [iParamètre](../WDLang5/3046008.md) utilisée avec la constante *iRTFAvecImagesEtTableaux* est prioritaire sur la propriété **RTFAvecImagesEtTableaux**. La propriété **RTFAvecImagesEtTableaux** est donc ignorée si la fonction [iParamètre](../WDLang5/3046008.md) est appelée avec la constante *iRTFAvecImagesEtTableaux*.

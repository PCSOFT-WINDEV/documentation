
## ZoomMin (Propriété)

***En anglais : MinZoom***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **ZoomMin** permet de connaître la valeur minimale du zoom utilisable pour le champ Caméra (pour la caméra en cours d'utilisation).

Remarque : Cette propriété est utilisable uniquement pour les champs Caméra qui ne sont pas en "Mode compatible 26". 

Remarque : 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Donne les valeurs maxi et mini pour un champ Potentiomètre
// permettant de régler le zoom du champ Caméra 
POT_Zoom.BorneMin = CAM_MaCaméra.ZoomMin
POT_Zoom.BorneMax = CAM_MaCaméra.ZoomMax
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Champ Caméra>.ZoomMin`
---

**`<Résultat> : (Réel)`**

Zoom minimal du champ Caméra. 

**`<Champ Caméra> : (Nom de champ)`**

Nom du champ Caméra manipulé.

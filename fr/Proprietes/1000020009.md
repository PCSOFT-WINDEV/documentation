


## BoutonCalculatrice (Propriété)

***En anglais : CalculatorButton***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **BoutonCalculatrice** permet de savoir et d'indiquer si un champ de saisie numérique ou monétaire propose un bouton pour afficher une calculatrice en popup. 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=BoutonCalculatrice%20-%20HC%20N%B0001.gif)

<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Clic sur BTN_ACTIVER_CALCULATRICE
SAI_Tarif.BoutonCalculatrice = Vrai
SAI_Remise.BoutonCalculatrice = Vrai
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Savoir si un bouton Calculatrice est affiché

`<Résultat> = <Champ de saisie>.BoutonCalculatrice`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le bouton Calculatrice est affiché. 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 




**`<Champ de saisie> : (Nom de champ)`**

Nom du champ de saisie de type numérique ou monétaire à manipuler. 


<a name="SYNTAXE2"></a>

### Afficher un bouton Calculatrice

`<Champ de saisie>.BoutonCalculatrice = <Afficher>`
---

**`<Champ de saisie> : (Nom de champ)`**

Nom du champ de saisie de type numérique ou monétaire à manipuler. 

**`<Afficher> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le bouton Calculatrice doit être affiché. 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Fonctionnement de la calculatrice
<a name="fonctionnement_calculatrice_ELTPARAGRAPHE000056"></a>Par défaut, en exécution, les champs de saisie numériques et monétaires proposent l'option "Calculatrice" dans leur menu contextuel. Cette option permet d'afficher une calculatrice pour réaliser un calcul. 

Si la propriété **BoutonCalculatrice** est utilisée pour afficher un bouton Calculatrice  dans le champ de saisie numérique ou monétaire, l'option "Calculatrice" n'est alors plus disponible dans le menu contextuel du champ. 

**Remarque** : Il est possible de désactiver pour un projet ou un champ l'affichage de la calculatrice (via le bouton ou le menu contextuel) grâce à la fonction [FAADésactive](../WDLang1/1000022018.md).

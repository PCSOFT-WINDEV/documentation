


## GrosFichier (Propriété)

***En anglais : HugeFile***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **GrosFichier** permet de paramétrer la taille maximale d'un fichier de données. Il est ainsi possible de :

- Savoir si un fichier de données est configuré pour contenir plus de 2 Go de données (fichier défini sous l'éditeur d'analyses ou défini par programmation).

- Définir les caractéristiques de taille d'un fichier de données (lors de la description du fichier par programmation uniquement).





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Définir la taille maximale du fichier de données Client
Client.GrosFichier = Vrai
...
// Valide la description du fichier de données
HDécritFichier(Client)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître la taille maximale du fichier de données

`<Résultat> = <Fichier de données>.GrosFichier`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le fichier de données est configuré pour contenir plus de 2 Go de données, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Fichier de données> : (Chaîne de caractères)`**

Nom du fichier de données manipulé. Ce nom a été défini sous l'éditeur d'analyses ou grâce au type [Description de Fichier](../WDLang4/1514065.md).


<a name="SYNTAXE2"></a>

### Définir la taille maximale du fichier de données (fichier défini par programmation)

`<Fichier de données>.GrosFichier = <Taille 2Go>`
---

**`<Fichier de données> : (Chaîne de caractères)`**

Nom du fichier de données manipulé. Ce nom vient d'être défini grâce au type [Description de Fichier](../WDLang4/1514065.md).

**`<Taille 2Go> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le fichier de données doit être configuré pour contenir plus de 2 Go de données, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Fichiers de données de plus de 2 Go
<a name="fichiers_donnees_plus_2_ELTPARAGRAPHE000085"></a>Les fichiers de données de plus de 2 Go ne peuvent être créés que sur des disques de type NTFS. De plus, le système d'exploitation doit lui aussi gérer et supporter les fichiers de plus de 2 Go.




## TypeMIME (Propriété)

***En anglais : MIMEType***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **TypeMIME** permet de connaître et de modifier le type MIME d'un élément. Cette propriété est utilisable sur :

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [upnpImage](../WDLang3/1000020905.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [gglDocument](../WDLang5/1000018001.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [gglPièceJointe](../WDLang5/1000026182.md).


## DernièreDateCouverte (Propriété)

***En anglais : LatestDateCovered***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **DernièreDateCouverte** permet de connaître la date du dernier enregistrement sélectionné. Cette propriété est utilisable sur :

- une variable de type [sfRésultatRechercheModifié](../WDLang5/1000018561.md).

- une variable de type [sfRésultatRechercheSupprimé](../WDLang5/1000018562.md).

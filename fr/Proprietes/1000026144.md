
## CheminFichierParDéfaut (Propriété)

***En anglais : DefaultFilePath***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **CheminFichierParDéfaut** permet de connaître et de définir le nom complet du fichier (chemin et nom du fichier) proposé par défaut dans le cas d'un opération de type "Enregistrer sous" dans un champ Traitement de texte ou Editeur HTML.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
TT_ChampTT.CheminFichierParDéfaut = "C:\mesdoc\doc.docx"
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître le nom complet du fichier proposé par défaut

`<Résultat> = <Champ manipulé>.CheminFichierParDefaut`
---

**`<Résultat> : (Chaîne de caractères)`**

Nom complet du fichier (chemin et nom du fichier) proposé par défaut lors d'une opération de type "Enregistrer sous". 

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ manipulé : 

- Champ Traitement de texte. 

- Champ Editeur HTML. 





<a name="SYNTAXE2"></a>

### Modifier le nom complet du fichier proposé par défaut

`<Champ manipulé>.CheminFichierParDefaut = <Chemin par défaut>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ manipulé : 

- Champ Traitement de texte. 

- Champ Editeur HTML. 




**`<Chemin par défaut> : (Chaîne de caractères)`**

Nom complet du fichier (chemin et nom du fichier) proposé par défaut lors d'une opération de type "Enregistrer sous".

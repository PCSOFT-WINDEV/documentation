
## Action (Propriété)

***En anglais : Action***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Action** permet de spécifier l'action à réaliser. Cette propriété est utilisable sur :

- une variable de type [sfRequêteTraitementTâche](../WDLang5/1000018520.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [sysNotification](../WDLang1/1410087893.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [sysNotificationBouton](../WDLang1/1410087947.md).




## EtatInitial (Propriété)

***En anglais : InitialState***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **EtatInitial** permet de connaître l'état initial d'un champ. L'état initial correspond à l'état défini lors de la description du champ (onglet "UI").


|   |   |
| --- | --- |
| <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=EtatInitial_Champ.gif)<br>Etat initial d'un champ de saisie (onglet "UI") | <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Etat_init_champ_btn.gif)<br>Etat initial d'un champ (bouton par exemple) |
| <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Table_initial.gif)<br>Etat initial d'un champ Table (onglet "UI") | <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Etat_init_col.gif)<br>Etat initial d'une colonne de table (onglet "UI") |

Pour savoir si un champ est visible ou non, utilisez la propriété [Visible](../Proprietes/2510138.md).






<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI BTN_Fermer.EtatInitial = Grisé ALORS
	BTN_Fermer.Etat = Actif
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Etat initial> = <Champ>.EtatInitial`
---

**`<Etat initial> : (Constante de type entier)`**

Etat initial du champ, défini sous l'éditeur.


|   |   |
| --- | --- |
| *Actif* | Champ actif, visible et en saisie |
| *AffichageSansSélection* | Pour les champs Table uniquement : Champ affiché, sans aucune saisie possible.<br>Dans ce cas, l'utilisateur ne pourra pas sélectionner une ligne de la table à l'aide de la souris ou des flèches. Une ligne pourra être sélectionnée uniquement en programmation, grâce à la fonction [TableSelectPlus](../WDLang1/3074031.md). Cette ligne sera sélectionnée grâce au bandeau de sélection.<br><br>![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Cette constante n'est pas disponible. |
| *AffichageSeulement ou Inactif* | Champ affiché, sans aucune saisie ou action possible<br>Pour les champs Table, l'utilisateur pourra sélectionner une ligne de la table. Il sera également possible de la sélectionner par programmation grâce à la fonction [TableSelectPlus](../WDLang1/3074031.md). |
| *Grisé* | Champ grisé |



**`<Champ> : (Nom de champ ou de groupe de champs)`**

Nom du champ ou du groupe de champs.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Etat d'un groupe de champs
<a name="etat_groupe_champs_ELTPARAGRAPHE000136"></a>Si la propriété **EtatInitial** est utilisée sur un groupe de champs, l'état renvoyé correspond à l'état initial du premier champ du groupe.
<a name="NOTE0_2"></a>


### Limites
<a name="limites_ELTPARAGRAPHE000146"></a>

- La propriété **EtatInitial** ne peut pas être utilisée sur un menu ou une option de menu.

- ![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) La propriété **EtatInitial** est sans effet sur les champs HTML.

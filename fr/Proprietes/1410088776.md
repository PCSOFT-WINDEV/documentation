
## Déplacer

***En anglais : Move***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Déplacer** permet d'autoriser ou non le déplacement d'un élément. Cette propriété est disponible sur : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [diagActionAutorisée](../WDLang1/1410088840.md). 

- une variable de type [kbActionAutorisée](../WDLang1/1410089541.md).

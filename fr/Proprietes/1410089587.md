
## VisualisationMarkdown (Propriété)

***En anglais : DisplayAsMarkdown***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **VisualisationMarkdown** permet d'indiquer ou non si un champ de saisie affiche son contenu au format Markdown. 

Remarque : Cette propriété permet de changer le type du champ de saisie en le rendant de type "Visualisation Markdown". 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Clic sur [Prévisu en Markdown]
SAI_Promo.VisualisationMarkdown = Vrai

// Clic sur [Modification]
SAI_Promo.VisualisationMarkdown = faux
DonneFocus(SAI_Promo)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Savoir si le champ de saisie affiche son contenu au format Markdown

`<Résultat> = <Champ de saisie>.VisualisationMarkdown`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le champ de saisie est en lecture seule avec le contenu affiché au format Markdown. Le défilement et la sélection sont possibles (sauf si la propriété [Etat](../Proprietes/2510039.md) vaut *Grisé*). La propriété [Etat](../Proprietes/2510039.md) du champ est inchangée. Le champ de saisie est de type "Visualisation Markdown". 

- <u><u><u><u>Faux</u></u></u></u> si le champ de saisie est en saisie.  




**`<Champ de saisie> : (Nom de champ)`**

Nom du champ de saisie manipulé. 


<a name="SYNTAXE2"></a>

### Indiquer si un champ de saisie affiche ou non son contenu au format Markdown

`<Champ de saisie>.VisualisationMarkdown = <Mode de gestion>`
---

**`<Champ de saisie> : (Nom de champ)`**

Nom du champ de saisie manipulé.    

**`<Mode de gestion> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> : le champ de saisie passe en lecture seule avec le contenu affiché au format Markdown. Le défilement et la sélection sont possibles (sauf si la propriété [Etat](../Proprietes/2510039.md) vaut *Grisé*). La propriété [Etat](../Proprietes/2510039.md) du champ est inchangée.  Le champ de saisie passe en champ de saisie de type "Visualisation Markdown". 

- <u><u><u><u>Faux</u></u></u></u> : le champ de saisie repasse en saisie. Il est également nécessaire d'utiliser la propriété [Etat](../Proprietes/2510039.md) pour le rendre actif. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- L'option "Support du Markdown" présente dans l'onglet "UI" de la fenêtre de description des champs de saisie s'applique uniquement au libellé, à la bulle et aux jetons associés au champ de saisie. Cette propriété ne s'applique pas au contenu du champ de saisie. 

- Pour afficher des données au format Markdown, le champ de saisie doit : 

	- soit être de type "Visualisation Markdown". 

	- soit avoir la propriété **VisualisationMarkdown** à <u><u><u><u>Vrai</u></u></u></u>.

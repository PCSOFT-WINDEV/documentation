


## NbElémentTableau (Propriété)

***En anglais : NbArrayElement***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **NbElémentTableau** permet de paramétrer le nombre d'éléments d'une rubrique tableau. Il est ainsi possible de :

- Connaître le nombre d'éléments d'une rubrique tableau (définie sous l'éditeur d'analyses ou définie par programmation).

- Définir le nombre d'éléments d'une rubrique tableau (lors de la description de la rubrique par programmation uniquement).





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Définir le nombre d'éléments de la 
// rubrique tableau Civilité du fichier de données Client
Civilité.NbElémentTableau = 3
...
// Valide la description du fichier de données
HDécritFichier(Client)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le nombre d'éléments d'une rubrique tableau

`<Nombre d'éléments> = <Fichier de données>.<Rubrique>.NbElémentTableau`
---

**`<Nombre d'éléments> : (Entier)`**



- Nombre d'éléments de la rubrique tableau spécifiée,

- 1 si la rubrique spécifiée est une rubrique non tableau.




**`<Fichier de données> : (Chaîne de caractères)`**

Nom du fichier de données manipulé. Ce nom a été défini sous l'éditeur d'analyses ou grâce au type [Description de Fichier](../WDLang4/1514065.md).

**`<Rubrique> : (Chaîne de caractères)`**

Nom de la rubrique tableau manipulée. Ce nom a été défini sous l'éditeur d'analyses ou grâce au type [Description de Rubrique](../WDLang4/1514071.md).  


<a name="SYNTAXE2"></a>

### Définir le nombre d'éléments d'une rubrique tableau

`<Rubrique>.NbElémentTableau = <Nouveau nombre d'éléments>`
---

**`<Rubrique> : (Chaîne de caractères)`**

Nom de la rubrique tableau manipulée. Ce nom vient d'être défini grâce au type [Description de Rubrique](../WDLang4/1514071.md).

**`<Nouveau nombre d'éléments> : (Entier)`**



- Nombre d'éléments de la rubrique tableau spécifiée, 

- 1 si la rubrique spécifiée est une rubrique non tableau.

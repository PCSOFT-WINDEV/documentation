
## FormatCouleur (Propriété)

***En anglais : ColorFormat***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **FormatCouleur** permet de gérer le format de la couleur (couleur ou niveau de gris). Cette propriété est disponible sur : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [Scanneur](../WDLang6/1410087808.md), 

- une variable de type [pdfParamètre](../WDLang6/1410087857.md).




## ImpressionAChaquePage (Propriété)

***En anglais : PageByPagePrint***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **ImpressionAChaquePage** permet de :

- Demander l'impression d'un état page par page. Chaque page de l'état sera envoyée séparément au spooler de l'imprimante. Cette option permet par exemple de minimiser la taille du spooler d'impression.

- Connaître le mode d'impression de l'état (page par page ou en totalité).




**Remarque** : Pour demander l'impression page par page d'un état, il est également possible de cocher l'option "Envoyer chaque page séparément à l'imprimante" dans l'onglet "Options" de la description de l'état.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Lancer l'impression à chaque page
ETAT_NomEtat.ImpressionAChaquePage = Vrai
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le mode d'impression de l'état

`<Résultat> = <Etat manipulé>.ImpressionAChaquePage`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'impression de l'état est effectuée pour chaque page, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Etat manipulé> : (Nom d'état)`**

Nom de l'état à manipuler.  


<a name="SYNTAXE2"></a>

### Demander l'impression d'un état à chaque page

`<Etat manipulé>.ImpressionAChaquePage = <Mode d'impression>`
---

**`<Etat manipulé> : (Nom d'état)`**

Nom de l'état à manipuler.

**`<Mode d'impression> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'impression de l'état est effectuée pour chaque page, 

- <u><u><u><u>Faux</u></u></u></u> si l'état est imprimé en totalité.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Dans quel événement WLangage est-il possible utiliser la propriété ImpressionAChaquePage ? 
<a name="dans_quel_evenement_wlangage_estil_possible_utiliser_propriete_docparampagetitleshort_ELTPARAGRAPHE000078"></a>Il est conseillé d'utiliser la propriété **ImpressionAChaquePage** dans l'événement "Ouverture" de l'état. Dans les autres événements associés à l'état, cette propriété sera ignorée.
<a name="NOTE0_2"></a>


### Limite : Champ calculé utilisant le nombre total de pages de l'état
<a name="limite_champ_calcule_utilisant_nombre_total_pages_etat_ELTPARAGRAPHE000091"></a>Si l'état à imprimer contient un champ calculé de type "Nombre de pages", il est déconseillé de demander l'impression de cet état page à page. En effet, pour calculer le nombre de pages, l'état doit être complètement exécuté. Si l'impression page à page est tout de même demandée, le nombre de pages sera remplacé par "...".

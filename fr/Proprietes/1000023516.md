
## ImageType (Propriété)

***En anglais : ImageType***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **ImageType** permet de connaître le type de l'image en cours d'édition dans un champ Editeur d'images (icône ou image).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
Trace(EI_Editeur.ImageType)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Champ Editeur d'images>.ImageType`
---

**`<Résultat> : (Constante de type Entier)`**

Type de l'image manipulée : 


|   |   |
| --- | --- |
| *picIcône* | Image de type icône. |
| *picImage* | Image de type image. |



**`<Champ Editeur d'images> : (Nom de champ)`**

Nom du champ Editeur d'images manipulé.


## Expéditeur (Propriété)

***En anglais : Sender***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Expéditeur** permet de connaître et de modifier l'adresse d'expédition d'un email. 

Cette propriété est utilisable sur :

- la structure [Email](../WDLang3/3032029.md).

- une variable de type [Email](../WDLang3/1000018713.md).

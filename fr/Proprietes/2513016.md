
## ImageEnroulée (Propriété)

***En anglais : CollapsedImage***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **ImageEnroulée** permet de : 

- connaître et/ou de modifier l'image par défaut d'une ligne enroulée dans un champ Table hiérarchique.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) connaître et/ou de modifier l'image par défaut d'une ligne enroulée dans un champ Arbre.

- connaître et/ou de modifier l'image d'une ligne spécifique enroulée dans un champ Table hiérarchique.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) connaître et/ou modifier l'image d'une ligne enroulée dans une rubrique hiérarchique d'un état de type "Tableau hiérarchique". 




Cette propriété permet par exemple de définir : 

- l'image qui sera utilisée par défaut lors des prochains ajouts dans le champ Table hiérarchique. Cette image sera prise en compte par les fonctions [TableAjoute](../WDLang1/3074017.md), [TableAjouteLigne](../WDLang1/3074006.md), [TableAjouteFils](../WDLang1/1000010004.md), [TableInsère](../WDLang1/3074025.md), [TableInsèreLigne](../WDLang1/3074026.md), [TableInsèreFils](../WDLang1/1000010005.md), ...

- l'image enroulée d'une des lignes du champ Table hiérarchique.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) l'image qui sera utilisée par défaut lors des prochains ajouts dans le champ Arbre. Cette image sera prise en compte par les fonctions [ArbreAjoute](../WDLang1/3018012.md) et [ArbreInsère](../WDLang1/3018023.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) l'image enroulée des lignes d'un état de type "Tableau hiérarchique".









<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Définit l'image utilisée pour une ligne enroulée
TABLEH_Recette.ImageEnroulée = "C:\MesImages\Image.BMP"
```


<a name="Exemple2"></a>



```wl
// Change l'image enroulée de la ligne 4
TABLEH_Recette[4].ImageEnroulée = "C:\MesImages\Image.BMP"
```


<a name="Exemple3"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) 
```wl
// Change l'image enroulée dans l'état
// -- Avant impression de RUB_Hiérarchie
SI MoiMême.ProfondeurHiérarchique = 1 ALORS MoiMême.ImageEnroulée = "C:\Images\outil.png"
```







<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître l'image par défaut d'une ligne enroulée ou l'image associée à une ligne spécifique

`<Résultat> = <Champ manipulé>.ImageEnroulée`
---

**`<Résultat> : (Chaîne de caractères)`**



- Nom et chemin complet de l'image associée par défaut aux lignes enroulées du champ. 

- Chaîne vide ("") si l'image utilisée par défaut n'a pas été modifiée.




**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler : 

- Champ Table hiérarchique. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Champ Arbre.




Pour connaître l'image enroulée associée à une ligne spécifique d'un champ Table hiérarchique, il suffit d'utiliser la notation suivante : 

```txt
<Table hiérarchique>[<Numéro de la ligne>]
```



<a name="SYNTAXE2"></a>

### Modifier l'image par défaut d'une ligne enroulée ou l'image associée à une ligne spécifique

`<Champ manipulé>.ImageEnroulée = <Nouvelle image>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler : 

- Champ Table hiérarchique. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Champ Arbre.




Pour modifier l'image enroulée associée à une ligne spécifique d'un champ Table hiérarchique, il suffit d'utiliser la notation suivante : 

```txt
<Table hiérarchique>[<Numéro de la ligne>]
```


**`<Nouvelle image> : (Chaîne de caractères)`**

Nom et chemin complet de la nouvelle image associée par défaut à une ligne enroulée.  




<a name="SYNTAXE3"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) 
### Connaître l'image d'une rubrique hiérarchique enroulée dans un état "Tableau hiérarchique"

`<Résultat> = <Rubrique hiérarchique>.ImageDéroulée`
---

**`<Résultat> : (Chaîne de caractères)`**



- Nom et chemin complet de l'image associée à une ligne enroulée de la rubrique hiérarchique. 

- Chaîne vide ("") si aucune image n'est utilisée.




**`<Rubrique hiérarchique> : (Chaîne de caractères)`**

Nom de la rubrique hiérarchique à manipuler.


<a name="SYNTAXE4"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) 
### Modifier l'image d'une rubrique hiérarchique enroulée dans un état "Tableau hiérarchique"

`<Rubrique hiérarchique>.ImageDéroulée = <Nouvelle image>`
---

**`<Rubrique hiérarchique> : (Chaîne de caractères)`**

Nom de la rubrique hiérarchique à manipuler.  

**`<Nouvelle image> : (Chaîne de caractères)`**

Nom et chemin complet de la nouvelle image associée à une ligne enroulée de la rubrique hiérarchique.

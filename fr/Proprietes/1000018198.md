
## Lien (Propriété)

***En anglais : Link***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Lien** permet de connaître et de modifier le lien d'un élément. Cette propriété est utilisable sur :

- une variable de type [rssCanal](../WDLang5/1000017769.md).

- une variable de type [rssEntrée](../WDLang5/1000017792.md).

- une variable de type [rssImage](../WDLang5/1000017799.md).

- une variable de type [rssSaisieTexte](../WDLang5/1000017802.md).  

- une variable de type [hdfsInformation](../WDLang4/1000021973.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [docElément](../WDLang1/1000022484.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [docElémentLien](../WDLang1/1000022970.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [diagPointDePassage](../WDLang1/1410088548.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [diagSélection](../WDLang1/1410088172.md).

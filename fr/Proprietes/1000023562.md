
## URLRedirection (Propriété)

***En anglais : RedirectionURL***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **URLRedirection** permet de connaître l'URL utilisée pour la redirection du résultat. Cette propriété est utilisée par :

- une variable de type [OAuth2Paramètres](../WDLang3/1000022212.md). 

- une variable de type [OpenIDParamètres](../WDLang3/1000023561.md).

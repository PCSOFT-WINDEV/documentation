
## NuméroSérie (Propriété)

***En anglais : SerialNumber***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **NuméroSérie** permet de gérer les numéros de série. Cette propriété est utilisable sur : 

- une variable de type [Certificat](../WDLang1/1000019287.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [upnpPériphérique](../WDLang3/1000020903.md).

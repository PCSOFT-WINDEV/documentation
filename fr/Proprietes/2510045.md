


## Miniature (Propriété)

***En anglais : ThumbnailImage***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Miniature** permet de :

- Connaître l'image associée à un élément d'un champ Liste image.

- Modifier l'image (la miniature) associée à un élément d'un champ Liste image.



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png) 
```wl
// Associer une miniature au premier élément du champ Liste image
LSI_ListeImage1[1].Miniature = "C:\Temp\Image1.Bmp"
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le chemin de l'image associée à un élément d'un champ Liste image

`<Résultat> = <Champ Liste image>[<Numéro de l'élément>].Miniature`
---

**`<Résultat> : (Chaîne de caractères)`**

Chemin complet de l'image utilisée pour l'élément du champ Liste image.

**`<Champ Liste image> : (Nom de champ)`**

Nom du champ Liste image à manipuler.

**`<Numéro de l'élément> : (Entier ou chaîne de caractères)`**

Indice de l'élément à manipuler.  


<a name="SYNTAXE2"></a>

### Modifier l'image associée à un élément d'un champ Liste image

`<Champ Liste image>[<Numéro de l'élément>].Miniature = <Nouvelle image>`
---

**`<Champ Liste image> : (Nom de champ)`**

Nom du champ Liste image à manipuler.

**`<Numéro de l'élément> : (Entier ou chaîne de caractères)`**

Indice de l'élément à modifier.

**`<Nouvelle image> : (Chaîne de caractères)`**

Chemin de l'image associée à l'élément du champ Liste image. Ce paramètre peut correspondre au nom d'un champ Image contenant l'image à associer à l'élément du champ Liste image.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
La propriété **Miniature** peut uniquement être utilisée sur les champs de type Liste image.

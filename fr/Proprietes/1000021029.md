


## MargeHauteur (Propriété)

***En anglais : MarginHeight***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **MargeHauteur** permet de : 

- Connaître la marge verticale entre les Widgets présents dans un champ Tableau de bord. 

- Modifier la marge verticale entre les Widgets présents dans un champ Tableau de bord.




Remarque : La marge verticale entre les Widgets d'un champ Tableau de bord est définie dans l'onglet "Détail" de la fenêtre de description du champ Tableau de bord. 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Modification de INT_EspacementWidget
SI INT_EspacementWidget ALORS
	TDB_Principal.MargeLargeur = 20
	TDB_Principal.MargeHauteur = 20
SINON
	TDB_Principal.MargeLargeur = 0
	TDB_Principal.MargeHauteur = 0
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître la marge verticale entre les Widgets

`<Résultat> = <Champ Tableau de bord>.MargeHauteur`
---

**`<Résultat> : (Entier)`**

Marge verticale (en pixels) entre les Widgets présents dans le champ Tableau de bord spécifié. 

**`<Champ Tableau de bord> : (Nom de champ)`**

Nom du champ Tableau de bord à manipuler. 


<a name="SYNTAXE2"></a>

### Modifier la marge verticale entre les Widgets

`<Champ Tableau de bord>.MargeHauteur = <Nouvelle marge>`
---

**`<Champ Tableau de bord> : (Nom de champ)`**

Nom du champ Tableau de bord à manipuler. 

**`<Nouvelle marge> : (Entier)`**

Nouvelle marge verticale (en pixels) entre les Widgets du champ Tableau de bord spécifié. Cette valeur doit être positive ou nulle. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Cette marge ne prend pas en compte le cadre défini dans le style.

- Pour modifier la marge horizontale, utilisez la propriété [MargeLargeur](../Proprietes/1000021028.md).

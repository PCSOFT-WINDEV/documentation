


## NbColonne (Propriété)

***En anglais : NbColumn***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **NbColonne** permet de connaître le nombre de colonnes d'un état. Cette propriété permet notamment de savoir si un état est un état multicolonne. 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Changement de la couleur de fond d'un bloc selon le numéro de colonne 
// (cas d'un état à 2 colonnes) 
// Dans l'événement "Avant impression du bloc"
SI MonEtat.NbColonne = 2 ALORS
	UneColonneCourante est un entier = iNumColonne()
	SELON UneColonneCourante
		CAS 1:
		MoiMême.CouleurFond = RougeClair
		CAS 2:
		MoiMême.CouleurFond = VertClair
	FIN
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Etat manipulé>.NbColonne`
---

**`<Résultat> : (Entier)`**

Nombre de colonnes présentes dans l'état. 

**`<Etat manipulé> : (Nom d'état)`**

Nom de l'état à manipuler.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- La propriété **NbColonne** s'applique uniquement aux états (et non aux champs ou blocs d'un état).

- La propriété **NbColonne** doit être utilisée dans le code des événements associés à l'état.

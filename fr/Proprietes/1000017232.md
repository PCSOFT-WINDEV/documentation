


## SousLibellé (Propriété)

***En anglais : SubCaption***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **SousLibellé** permet de connaître et de modifier : 

- le libellé secondaire d'une option d'un champ Sélecteur ou Interrupteur.
	**Remarque** : Le champ Sélecteur ou Interrupteur doit gérer les sous-libellés (champ de type "Standard + sous-libellés" ou "Vista + sous-libellés").

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) le libellé secondaire d'un élément d'un champ TreeMap. 





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Modification du libellé secondaire d'un champ Sélecteur
SEL_Choix[2].SousLibellé = "Pour activer le mode avancé"
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le sous-libellé d'un élément

`<Texte du sous-libellé> = <Champ manipulé>[<Numéro de l'élément>].SousLibellé`
---

**`<Texte du sous-libellé> : (Chaîne de caractères)`**

Correspond : 

- au sous-libellé de l'option pour un champ Sélecteur ou Interrupteur. 

- au libellé secondaire de l'élément pour un champ TreeMap. 




**`<Champ manipulé> : (Nom de champ)`**

Nom du champ manipulé. Ce champ peut correspondre à un champ Sélecteur, un champ Interrupteur ou un champ TreeMap. 

**`<Numéro de l'élément> : (Entier)`**

Correspond : 

- au numéro de l'option manipulée pour un champ Sélecteur ou Interrupteur. 

- à l'indice de l'élément manipulé pour un champ TreeMap. 





<a name="SYNTAXE2"></a>

### Modifier le sous-libellé d'un élément

`<Champ manipulé>[<Numéro de l'élément>].SousLibellé = <Nouveau sous-libellé>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ manipulé. Ce champ peut correspondre à un champ Sélecteur, un champ Interrupteur ou un champ TreeMap. 

**`<Numéro de l'élément> : (Entier)`**

Correspond : 

- au numéro de l'option manipulée pour un champ Sélecteur ou Interrupteur. 

- à l'indice de l'élément manipulé pour un champ TreeMap. 




**`<Nouveau sous-libellé> : (Chaîne de caractères)`**

Texte du sous-libellé à afficher pour l'élément spécifié.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Champ Interrupteur et Sélecteur
<a name="champ_interrupteur_selecteur_ELTPARAGRAPHE000077"></a>La hauteur du sous-libellé est automatiquement adaptée à la taille du texte affiché. Il est possible de modifier cette hauteur grâce à la propriété [Hauteur](../Proprietes/2510050.md).
<a name="NOTE0_2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 

### Champ TreeMap
<a name="champ_treemap_ELTPARAGRAPHE000089"></a>Le libellé secondaire d'un élément sera affiché uniquement si la place nécessaire est disponible.

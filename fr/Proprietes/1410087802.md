
## Cliquable (Propriété)

***En anglais : Clickable***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Cliquable** permet de connaître et modifier le comportement d'un champ Image par rapport aux clics souris. 

**Remarque** : Cette option correspond à l'option "Cette image est une zone de clic" de l'onglet "Détail" de la fenêtre de description du champ Image. 


	<B><B><B>Avertissement</B></B></B>

	 Dans les versions précédentes, cette propriété avait pour nom **Clicable**. 

<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Rendre un champ Image inactif aux clics de souris
IMG_MonImage.Cliquable = Faux
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître le comportement d'un champ Image par rapport aux clics souris

`<Champ Image> = <Comportement>.Cliquable`
---

**`<Champ Image> : (Nom de champ)`**

Nom du champ Image manipulé. 

**`<Comportement> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le champ Image est une zone de cliquage. Dans ce cas, tous les évènements souris (enfoncé, relâché, survol, double-clic, clic droit, etc.) sont capturés par le champ Image.

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Dans ce cas, le champ n'intercepte aucun évènement souris. Les événements souris sont capturés par les champs en dessous ou par la fenêtre. 





<a name="SYNTAXE2"></a>

### Modifier le comportement d'un champ Image par rapport aux clics souris

`<Champ Image>.Cliquable = <Nouveau comportement>`
---

**`<Champ Image> : (Nom de champ)`**

Nom du champ Image manipulé.

**`<Nouveau comportement> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le champ Image est une zone de cliquage. Dans ce cas, tous les évènements souris (enfoncé, relâché, survol, double-clic, clic droit, etc.) sont capturés par le champ Image.

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Dans ce cas, le champ n'intercepte aucun évènement souris. Les événements souris sont capturés par les champs en dessous ou par la fenêtre.

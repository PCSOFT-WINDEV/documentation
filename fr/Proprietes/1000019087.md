
## Client (Propriété)

***En anglais : Client***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Client** permet de connaître le compte client SaaS associé à :

- une variable de type [saasAbonnement](../WDLang3/1000019088.md),

- une variable de type [saasUtilisateur](../WDLang3/1000019083.md).


## PoliceItalique (Propriété)

***En anglais : FontItalic***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **PoliceItalique** permet de gérer la police italique dans un texte. Cette propriété est utilisable sur : 

- les champs. Pour plus de détails, consultez [PoliceItalique](../Proprietes/2510102.md). 

- une variable de type [docMiseEnForme](../WDLang1/1000022817.md).

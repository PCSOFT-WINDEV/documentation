
## JetonSaisieEnCours (Propriété)

***En anglais : TokenCurrentInput***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **JetonSaisieEnCours** permet de lire ou modifier la valeur en cours de saisie dans un champ de saisie de type "Jetons de texte".
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affiche un libellé d'avertissement si la saisie n'est pas valide
LIB_SaisieErronée.Visible = PAS bSaisieValide(SAI_Saisie.JetonSaisieEnCours)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Lire la valeur en cours de saisie dans un champ de saisie de type "Jetons de texte"

`<Résultat> = <Champ de saisie>.JetonSaisieEnCours`
---

**`<Résultat> : (Chaîne de caractères)`**



- Valeur en cours de saisie dans le champ de saisie de type "Jetons de texte".

- Chaîne vide ("") si aucune valeur n'est en cours de saisie. 




**`<Champ de saisie> : (Nom de champ)`**

Nom du champ de saisie de type "Jetons de texte" manipulé. 


<a name="SYNTAXE2"></a>

### Modifier la valeur en cours de saisie dans un champ de saisie de type "Jetons de texte"

`<Champ de saisie>.JetonSaisieEnCours = <Nouvelle valeur>`
---

**`<Champ de saisie> : (Nom de champ)`**

Nom du champ de saisie de type "Jetons de texte" manipulé. 

**`<Nouvelle valeur> : (Chaîne de caractères)`**

Nouvelle valeur du champ de saisie de type "Jetons de texte".

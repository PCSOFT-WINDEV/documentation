
## Existe (Propriété)

***En anglais : Exist***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Existe** permet de savoir si un élément existe. 

Cette propriété est utilisable sur :

- les tableaux associatifs ou leurs éléments. Pour plus de détails, consultez [Existe](../Proprietes/1000022167.md).

- une variable de type [Variant](../Motscles/1514019.md). 

- une variable de type [XMLNoeud](../WDLang5/1000018786.md). 

- une variable de type [JSON](../Motscles/1000023611.md). 

- une variable de type [YAML](../Motscles/1000024438.md). 

- une variable de type [htmlAttribut](../WDLang5/1000025912.md).

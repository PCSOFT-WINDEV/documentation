
## Orientation (Propriété)

***En anglais : Orientation***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Orientation** permet de connaître et de modifier l'orientation d'un élément. Cette propriété est utilisable sur :

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [docSection](../WDLang1/1000022685.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [pdfPage](../WDLang6/1000024904.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [iInfoSortie](../WDLang5/1410088954.md).

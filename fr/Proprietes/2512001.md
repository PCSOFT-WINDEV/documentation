
## Binaire (Propriété)

***En anglais : Binary***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Binaire** permet de savoir si une rubrique est de type binaire ou non. **Cette propriété est disponible en lecture seulement**.

**Remarque** : Les rubriques considérées comme des rubriques de type binaire sont les suivantes :

- chaîne binaire,

- mémo binaire,

- clé composée.







<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// La rubrique Client.Photo est-elle mémo texte ou mémo binaire ?
ResBinaire = Client.Photo.Binaire
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Fichier de données>.<Rubrique>.Binaire`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la rubrique est binaire, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Fichier de données> : (Chaîne de caractères)`**

Nom logique du fichier de données manipulé.

**`<Rubrique> : (Chaîne de caractères)`**

Nom de la rubrique manipulée.

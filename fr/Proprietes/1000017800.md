
## Largeur (Propriété)

***En anglais : Width***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Largeur** permet de connaître et de modifier la largeur d'un élément. Cette propriété est utilisable sur :

- un élément d'une fenêtre, d'une page ou d'un état. Pour plus de détails, consultez [Largeur](../Proprietes/2510059.md).

- une variable de type [rssFlux](../WDLang5/1000017795.md).

- une variable de type [rssImage](../WDLang5/1000017799.md).

- une variable de type [xlsDocument](../WDLang5/1000017464.md).

- une variable de type [xlsColonne](../WDLang5/1000017471.md).

- une variable de type [Image](../WDLang1/1000019650.md).

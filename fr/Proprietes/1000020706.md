


## Jauge (Propriété)

***En anglais : ProgressBar***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Jauge** permet de connaître et de modifier le champ Jauge utilisé durant le calcul d'un champ Tableau croisé dynamique (fonctions [TCDCalculeTout](../WDLang1/1000020539.md) et [TCDCalculeMiseAJour](../WDLang1/1000020534.md)). 
<a name="sample_code"></a>

## Exemple


```wl
TCD_Ventes.Jauge = JAUGE_TCDVentes.Nom
TCDCalculeTout(TCD_Ventes)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître la jauge utilisée pour le calcul d'un champ Tableau croisé dynamique

`<Résultat> = <Champ Tableau croisé dynamique>.Jauge`
---

**`<Résultat> : (Chaîne de caractères)`**



- Nom du champ Jauge utilisé. 

- Chaîne vide ("") si la jauge automatique du champ Tableau croisé dynamique est utilisée. 




**`<Champ Tableau croisé dynamique> : (Nom de champ)`**

Nom du champ Tableau croisé dynamique. 


<a name="SYNTAXE2"></a>

### Modifier la jauge utilisée pour le calcul d'un champ Tableau croisé dynamique

`<Champ Tableau croisé dynamique>.Jauge = <Nouvelle jauge>`
---

**`<Champ Tableau croisé dynamique> : (Nom de champ)`**

Nom du champ Tableau croisé dynamique à manipuler.  

**`<Nouvelle jauge> : (Chaîne de caractères)`**

Nouveau nom de la jauge à utiliser. Ce nom peut correspondre : 

- au nom d'un champ Jauge (propriété [Nom](../Proprietes/2510082.md) utilisée sur le champ Jauge). 

- à une chaîne vide ("") si la jauge automatique du champ Tableau croisé dynamique doit être utilisée. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Il n'est pas possible d'utiliser une case de barre de message de type Jauge. 

- Si le champ Jauge est invisible, il est automatiquement rendu visible le temps du calcul, puis rendu à nouveau invisible à la fin du calcul.

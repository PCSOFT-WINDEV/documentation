
## Unicode (Propriété)

***En anglais : Unicode***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Unicode** permet de connaître et de modifier le format Unicode d'un élément. Cette propriété est utilisable sur :

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) un champ. Pour plus de détails, consultez [Unicode](../Proprietes/1000017001.md).

- une variable de type [pdfParamètre](../WDLang6/1410087857.md).

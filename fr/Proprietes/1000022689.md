
## MargeGauche (Propriété)

***En anglais : LeftMargin***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **MargeGauche** permet de définir la marge gauche d'un élément. Cette propriété est utilisable : 

- sur les états. Pour plus de détails, consultez [MargeGauche](../Proprietes/2511048.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) sur les variables de type [docSection](../WDLang1/1000022685.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) sur les variables de type [xlsMiseEnPage](../WDLang5/1000023139.md).

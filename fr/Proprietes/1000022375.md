


## RetraitDétailLigne (Propriété)

***En anglais : IndentLineDetails***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **RetraitDétailLigne** permet de connaître ou de modifier la valeur du retrait à gauche de la fenêtre interne utilisée pour afficher les détails d'une ligne d'un champ Table.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Clic sur [Avec Détails]
TABLE_Commande.FenêtreDétailLigne =  "FI_DetailCommande" 
TABLE_Commande.RetraitDétailLigne =  16  // 16 pixels de décalage
TableDéroule(TABLE_Commande, 1)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître le retrait à gauche de la fenêtre interne de détail d'une ligne

`<Résultat> = <Champ Table>.RetraitDétailLigne`
---

**`<Résultat> : (Entier)`**

Valeur du retrait à gauche (en pixels) de la fenêtre interne utilisée pour afficher le détail de la ligne.

**`<Champ Table> : (Nom de champ)`**

Nom du champ Table à manipuler.


<a name="SYNTAXE2"></a>

### Modifier le retrait à gauche de la fenêtre interne de détail d'une ligne

`<Champ Table>.RetraitDétailLigne = <Nouveau retrait>`
---

**`<Champ Table> : (Nom de champ)`**

Nom du champ Table à manipuler.

**`<Nouveau retrait> : (Entier)`**

Nouvelle valeur du retrait à gauche (en pixels) de la fenêtre interne utilisée pour afficher le détail de la ligne.

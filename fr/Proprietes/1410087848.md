
## Torche (Propriété)

***En anglais : Torch***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Torche** permet de connaître et de modifier le mode d'utilisation de la torche par un champ Caméra. 

Remarques : 

- Cette propriété est utilisable uniquement pour les champs Caméra qui ne sont pas en "Mode compatible 26". 



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
CAM_MaCaméra.Torche = camTorcheOn
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître le mode d'utilisation de la torche par champ Caméra

`<Résultat> = <Champ Caméra>.Torche`
---

**`<Résultat> : (Constante de type Entier)`**

Mode d'utilisation de la torche de la caméra : 


|   |   |
| --- | --- |
| *camTorcheAuto* | Torche déclenchée automatiquement selon la luminosité.  |
| *camTorcheOff* | La torche est toujours désactivée. |
| *camTorcheOn* | La torche est toujours activée. |


 Par défaut, l'usage de la torche est désactivé. 

**`<Champ Caméra> : (Nom de champ)`**

Nom du champ Caméra à manipuler.


<a name="SYNTAXE2"></a>

### Modifier le mode d'utilisation de la torche par champ Caméra

`<Champ Caméra>.Torche = <Nouveau mode>`
---

**`<Champ Caméra> : (Nom de champ)`**

Nom du champ Caméra à manipuler.

**`<Nouveau mode> : (Constante de type Entier)`**

Mode d'utilisation de la torche de la caméra : 


|   |   |
| --- | --- |
| *camTorcheAuto* | Torche déclenchée automatiquement selon la luminosité.  |
| *camTorcheOff* | La torche est toujours désactivée. |
| *camTorcheOn* | La torche est toujours activée. |


 Par défaut, l'usage de la torche est désactivé.

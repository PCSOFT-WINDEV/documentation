


## NbEtapeAnimation (Propriété)

***En anglais : NbAnimationSteps***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **NbEtapeAnimation** permet de connaître et de spécifier le nombre de dessins présent dans l'image définissant l'animation du champ Jauge infinie personnalisé. Cette image a été spécifiée sous l'éditeur ou par la propriété [Image](../Proprietes/2510034.md). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
JAUGE_Infinie1.Image = "MonAnim.png"
JAUGE_Infinie1.NbEtapeAnimation = 10 // 10 images dans dans "MonAnim.png"
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître le nombre de dessins constituant l'animation

`<Résultat> = <Champ Jauge infinie>.NbEtapeAnimation`
---

**`<Résultat> : (Entier)`**

Nombre d'étapes défini pour l'animation du champ Jauge infinie. 

**`<Champ Jauge infinie> : (Nom de champ)`**

Nom du champ Jauge infinie à manipuler. 


<a name="SYNTAXE2"></a>

### Modifier le nombre de dessins constituant l'animation

`<Champ Jauge infinie>.NbEtapeAnimation = <Nouvelle valeur>`
---

**`<Champ Jauge infinie> : (Nom de champ)`**

Nom du champ Jauge infinie à manipuler. 

**`<Nouvelle valeur> : (Entier)`**

Nombre d'étapes à faire pour l'animation du champ Jauge infinie. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Cette propriété est sans effet si le champ est natif. Le champ doit être un champ Jauge infinie personnalisé. 

- L'image est découpée verticalement en "bandes" pour chaque étape d'animation.

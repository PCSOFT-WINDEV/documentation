


## BandeauVisible (Propriété)

***En anglais : BarVisible***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **BandeauVisible** permet de gérer le bandeau de sélection sur une colonne d'un champ Table (ou Table hiérarchique). Il est possible de :

- Savoir si le bandeau de sélection est visible ou non dans la colonne.

- Rendre le bandeau de sélection visible ou invisible dans la colonne.





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// N'affiche pas le bandeau de sélection sur les colonnes 1 et 5
COL_Colonne1.BandeauVisible = Faux
COL_Colonne5.BandeauVisible = Faux
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Savoir si le bandeau de sélection est visible sur une colonne

`<Visible/Invisible> = <Colonne manipulée>.BandeauVisible`
---

**`<Visible/Invisible> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le bandeau de sélection est visible, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Colonne manipulée> : (Nom de champ)`**

Nom de la colonne (champ Table ou Table hiérarchique) à manipuler.  


<a name="SYNTAXE2"></a>

### Rendre le bandeau de sélection visible ou invisible sur une colonne

`<Colonne manipulée>.BandeauVisible = <Visible/Invisible>`
---

**`<Colonne manipulée> : (Nom de champ)`**

Nom de la colonne (champ Table ou Table hiérarchique) à manipuler.

**`<Visible/Invisible> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le bandeau de sélection doit être visible sur la colonne, 

- <u><u><u><u>Faux</u></u></u></u> pour rendre le bandeau de sélection invisible sur la colonne.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- La propriété **BandeauVisible** peut être utilisée sur les colonnes des champs Table et Table hiérarchique.

- La propriété **BandeauVisible** peut être utilisée sur les champs Table multisélections : le bandeau de sélection est visible/invisible dans la colonne indiquée pour toutes les lignes sélectionnées.

- La propriété **BandeauVisible** est disponible sur les bandeaux de sélection associés à une couleur ou à une image.

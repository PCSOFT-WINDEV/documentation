
## DateCréation (Propriété)

***En anglais : DateCreated***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **DateCréation** permet de connaître la date de création d'un élément. Cette propriété est utilisable sur : 

- une variable de type [zipFichierArchivé](../WDLang3/1000019044.md). 

- une variable de type [docPropriété](../WDLang1/1000022945.md). 

- une variable de type [pdfDocument](../WDLang6/1000024909.md).

- une variable de type [pdfPièceJointe](../WDLang6/1000024899.md).

- une variable de type [FichierDisque](../WDLang1/1410088883.md).

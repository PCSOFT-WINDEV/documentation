
## Journalisation (Propriété)

***En anglais : LogMethod***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Journalisation** permet de :

- connaître le mode de journalisation utilisé pour un fichier de données (fichier défini sous l'éditeur d'analyses ou défini par programmation).

- définir le mode de journalisation utilisé pour un fichier de données (lors de la description du fichier par programmation uniquement).



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI Client.Journalisation = hJournalisationNon ALORS Info("Réplication impossible")
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le mode de journalisation 

`<Mode actuel> = <Fichier de données>.Journalisation`
---

**`<Mode actuel> : (Constante de type entier)`**

Les formats de journalisation sont : 



|   |   |
| --- | --- |
| *hJournalisationEcriture* | Journalisation des opérations d'écriture réalisées dans le fichier de données. |
| *hJournalisationLectureEcriture* | Journalisation des opérations de lecture et d'écriture réalisées dans le fichier de données. |
| *hJournalisationNon* | Pas de journalisation. |



**`<Fichier de données> : (Chaîne de caractères)`**

Nom du fichier de données manipulé. Ce nom a été défini sous l'éditeur d'analyses ou grâce au type [Description de Fichier](../WDLang4/1514065.md).  


<a name="SYNTAXE2"></a>

### Définir le mode de journalisation 

`<Fichier de données>.Journalisation = <Nouveau mode>`
---

**`<Fichier de données> : (Chaîne de caractères)`**

Nom du fichier de données manipulé. Ce nom vient d'être défini grâce au type [Description de Fichier](../WDLang4/1514065.md).

**`<Nouveau mode> : (Constante de type entier)`**

Les formats de journalisation sont :



|   |   |
| --- | --- |
| *hJournalisationEcriture* | Journalisation des opérations d'écriture réalisées dans le fichier de données. |
| *hJournalisationLectureEcriture* | Journalisation des opérations de lecture et d'écriture réalisées dans le fichier de données. |
| *hJournalisationNon* | Pas de journalisation. |


## ProcédureTrace (Propriété)

***En anglais : ProcedureTrace***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **ProcédureTrace** permet de spécifier le nom de la procédure WLangage utilisée pour connaître les entêtes et les données envoyées. Cette propriété est utilisable sur :

- une variable de type [httpRequête](../WDLang3/1000021158.md). 

- une variable de type [restRequête](../WDLang3/1000021481.md).




## Port (Propriété)

***En anglais : Port***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Port** permet de connaître et de modifier le port de connexion à un serveur. Cette propriété est utilisable sur :

- une variable de type [emailSessionIMAP](../WDLang3/1000018957.md).

- une variable de type [emailSessionPOP3](../WDLang3/1000018759.md).

- une variable de type [emailSessionSMTP](../WDLang3/1000018765.md).

- une variable de type [rssFlux](../WDLang5/1000017795.md). 

- une variable de type [rssCloud](../WDLang5/1000017785.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png) une variable de type [sshSession](../WDLang3/1000021157.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png) une variable de type [mqttSession](../WDLang3/1000023023.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [memcachedConnexion](../WDLang4/1000023339.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [adminWBAuth](../WDLang2/1410089328.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable générée automatiquement pour représenter [un Webservice REST importé](../WDLang3/1410087809.md).

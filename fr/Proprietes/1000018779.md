


## NomFichierNavigateur (Propriété)

***En anglais : NameBrowserFile***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **NomFichierNavigateur** permet de connaître le nom original du fichier à uploader présent sur le poste de l'internaute. Cette propriété est disponible uniquement pour les champs Upload multi-fichiers.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Les fichiers uploadés sont copiés dans un répertoire sur le serveur avec leur nom d'origine
POUR i = 1 _A_ UPL_Upload.Occurrence
	UploadCopieFichier(UPL_Upload, gsCheminFichiersUploadés, "", i)
	Trace(UPL_Upload[i].NomFichierNavigateur, UPL_Upload[i].NomFichierServeur)
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le nom original du fichier à uploader présent sur le poste de l'internaute

`<Résultat> = <Champ Upload>[<Identifiant du fichier>].NomFichierNavigateur`
---

**`<Résultat> : (Chaîne de caractères)`**

Nom original du fichier uploadé (c'est-à-dire nom du fichier présent sur le poste de l'internaute).

**`<Champ Upload> : (Nom de champ)`**

Nom du champ Upload à manipuler.

**`<N> : (Entier)`**

Numéro du fichier considéré présent dans le champ Upload.

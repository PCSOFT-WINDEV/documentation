


## NumTAB (Propriété)

***En anglais : TABOrder***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **NumTAB** permet de connaître et de modifier la position d'un champ dans [l'ordre de navigation](../Editeurs/2026006.md) d'une fenêtre.

**Remarques** :

- L'ordre de navigation des champs d'une fenêtre décrit l'ordre de prise de focus des champs lorsque l'utilisateur utilise la touche Tabulation.

- Pour connaître l'ordre de navigation de l'ensemble des champs présents dans une fenêtre, sous le volet "Fenêtre", dans le groupe "Ordre", déroulez "Navigation" et sélectionnez "Éditer".

- Pour modifier l'ordre de navigation, sous le volet "Fenêtre", dans le groupe "Ordre", déroulez "Navigation" et sélectionnez "Éditer". Utilisez les flèches situées sur la droite.





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ordre de navigation du champ "SAI_NomClient"
ResNavigation = SAI_NomClient.NumTAB
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître la position d'un champ dans l'ordre de navigation

`<Résultat> = <Champ à manipuler>.NumTAB`
---

**`<Résultat> : (Entier)`**



- Position du champ spécifié dans l'ordre de navigation, 

- -1 si le champ spécifié ne peut pas être accessible par la touche [Tabulation].




**`<Champ à manipuler> : (Nom de champ)`**

Nom du champ à manipuler.  


<a name="SYNTAXE2"></a>

### Modifier la position d'un champ dans l'ordre de navigation

`<Champ à manipuler>.NumTAB = <Nouvelle position>`
---

**`<Champ à manipuler> : (Nom de champ)`**

Nom du champ à manipuler.

**`<Nouvelle position> : (Entier)`**

Position du champ spécifié dans l'ordre de navigation. Cette position doit être comprise entre 1 (premier champ) et le nombre de champs de la fenêtre. Si la nouvelle position correspond à une position déjà attribuée, toutes les positions des champs suivants seront décalées.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Champ accessible par la touche Tabulation
<a name="champ_accessible_par_touche_tabulation_ELTPARAGRAPHE000088"></a>Pour qu'un champ soit accessible par la touche Tabulation, cochez l'option "Accessible par TAB" dans l'onglet "UI" de la description du champ.

Par défaut, l'option "Accessible par TAB" est cochée lors de la création d'un champ.

Si le champ est accessible par la touche Tabulation, ce champ est automatiquement intégré dans l'ordre de navigation de la fenêtre. Par défaut, l'ordre de navigation correspond à l'ordre de création des champs dans la fenêtre.
<a name="NOTE0_2"></a>


### Limites
<a name="limites_ELTPARAGRAPHE000107"></a>La propriété **NumTAB** ne peut pas être utilisée sur :

- une fenêtre,

- un champ Libellé, Forme ou Image,

- une option d'un champ Sélecteur ou d'un champ Interrupteur,

- une ligne d'un champ Liste (y compris Liste image), Combo ou Table,

- une cellule d'un champ Table,

- une colonne d'un champ Table,

- un menu,

- un volet d'onglet,

- un groupe de champs,

- un élément d'un champ Arbre.




![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) La propriété **NumTAB** s'applique uniquement aux éléments suivants :

- Champ Bouton.

- Champ de saisie.

- Champ Interrupteur.

- Champ Sélecteur.

- Champ Liste.

- Champ Combo.

- Champ Arbre.

- Champ Onglet.

- Champ Table.

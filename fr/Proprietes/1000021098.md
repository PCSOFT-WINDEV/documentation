


## Volet (Propriété)

***En anglais : Pane***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Volet** permet de : 

- Connaître les volets d'onglet associés à un champ. 

- Définir les volets d'onglet associés à un champ. 




**Remarque** : Cette propriété fonctionne uniquement sur les volets d'onglet classiques. Cette propriété ne fonctionne pas sur les volets d'onglet dynamiques. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Associer le bouton Valider aux volets 1, 2 et 4
BTN_Valider.Volet = "1,2,4"

// Connaître les volets associés à un bouton
SI BTN_Valider.Volet [=] "2" ALORS Info("Le champ est associé au volet 2")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître les volets d'onglets associés à un champ

`<Résultat> = <Champ manipulé>.Volet`
---

**`<Résultat> : (Entier ou chaîne de caractères)`**



- Numéro du volet d'onglet associé au champ. Le premier volet a pour indice 1. 

- Chaîne de caractères contenant la liste des volets d'onglets associés au champ sous la forme : 
	
	```txt
	"<Numéro volet 1>,<Numéro volet 2>,...,<Numéro volet N>"
	```


- 0 si le champ n'appartient à aucun volet. 




**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler.


<a name="SYNTAXE2"></a>

### Définir les volets d'onglet associés à un champ

`<Champ manipulé>.Volet = <Nouvelle valeur>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler.

**`<Nouvelle valeur> : (Entier ou chaîne de caractères)`**



- Numéro du volet d'onglet associé au champ. Le premier volet a pour indice 1. 

- Chaîne de caractères contenant la liste des volets d'onglets associés au champ sous la forme : 
	
	```txt
	"<Numéro volet 1>,<Numéro volet 2>,...,<Numéro volet N>"
	```







<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Si le champ n'appartient à aucun onglet, il n'est pas possible de l'associer à un ou plusieurs volets d'onglet.

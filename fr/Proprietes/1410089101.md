
## IgnoreErreur (Propriété)

***En anglais : IgnoreError***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **IgnoreErreur** permet de connaître et de modifier les différentes erreurs de certificats ignorées par le champ d'affichage HTML et par le champ Page WEBDEV.

Cette propriété permet de faire fonctionner des sites avec des certificats non conformes.
**Attention** : L'utilisation de cette propriété peut présenter des risques de sécurité et doit être utilisée uniquement si le site à afficher dans le champ est légitime.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
HTM_PageWeb.IgnoreErreur = httpIgnoreCertificatExpiré + httpIgnoreCertificatInvalide
HTM_PageWeb = "https://www.monsiteavecunmauvaiscertificat.com/login"
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître les différentes erreurs ignorées par le champ d'affichage HTML et par le champ Page WEBDEV

`<Résultat> = <Champ manipulé>.IgnoreErreur`
---

**`<Résultat> : (Constante ou combinaison de constantes de type Entier)`**

Erreurs de certificat ignorées : 


|   |   |
| --- | --- |
| *httpIgnoreCertificatExpiré* | La date du certificat est ignorée |
| *httpIgnoreCertificatInvalide* | Le certificat est ignoré. |
| *httpIgnoreNomCertificatInvalide* | Le nom du site figurant dans le certificat est ignoré. |
| *httpIgnoreRenvoiHTTP* | Le renvoi vers un serveur non sécurisé est autorisé. |





**`<Champ manipulé> : (Nom de champ)`**

Nom du champ d'affichage HTML ou du champ Page WEBDEV à manipuler. 


<a name="SYNTAXE2"></a>

### Modifier les différentes erreurs ignorées par le champ d'affichage HTML et par le champ Page WEBDEV

`<Champ manipulé>.IgnoreErreur = <Erreur à ignorer>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ d'affichage HTML ou du champ Page WEBDEV à manipuler. 

**`<Erreur à ignorer> : (Constante ou combinaison de constantes de type Entier)`**

Erreurs de certificat ignorées : 



|   |   |
| --- | --- |
| *httpIgnoreCertificatExpiré* | La date du certificat est ignorée |
| *httpIgnoreCertificatInvalide* | Le certificat est ignoré. |
| *httpIgnoreNomCertificatInvalide* | Le nom du site figurant dans le certificat est ignoré. |
| *httpIgnoreRenvoiHTTP* | Le renvoi vers un serveur non sécurisé est autorisé. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Cette propriété est disponible pour les champs basés sur Chromium CEF (champ d'affichage HTML et champ Page WEBDEV).

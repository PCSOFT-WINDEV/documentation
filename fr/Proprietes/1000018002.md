
## Opacité (Propriété)

***En anglais : Opacity***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Opacité** permet de connaître et de modifier l'opacité d'un élément. Cette propriété est utilisable sur :

- les champs d'une fenêtre ou d'une page. Pour plus de détails, consultez [Propriété Opacité sur les champs](../Proprietes/2514006.md).

- une variable de type [gglChemin](../WDLang5/1000017433.md). 

- une variable de type [Couleur](../WDLang1/1000019661.md).

- une variable de type [Filigrane](../WDLang5/1000019922.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [pdfSignature](../WDLang6/1000022255.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [Toast](../WDLang1/1000024638.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [picTexte](../WDLang1/1000025178.md).

- une variable de type [CarteImage](../WDLang3/1000025523.md).

- une variable de type [Marqueur](../WDLang3/1000019940.md).

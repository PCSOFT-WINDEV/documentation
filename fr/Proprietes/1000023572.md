
## Valide (Propriété)

***En anglais : Valid***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Valide** permet de savoir si un élément est valide. Cette propriété est utilisable sur :

- une variable de type [Image](../WDLang1/1000019650.md).

- une variable de type [AuthToken](../WDLang3/1000022220.md). 

- une variable de type [OpenIDIdentité](../WDLang3/1000023567.md).

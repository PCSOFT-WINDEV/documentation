
## CadrageHorizontal (Propriété)

***En anglais : HorizontalAlignment***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **CadrageHorizontal** permet de connaître et de modifier le cadrage horizontal d'un champ ou d'un texte. Cette propriété est utilisable sur : 

- les champs d'une fenêtre ou d'une page. Pour plus de détails, consultez [CadrageHorizontal](../Proprietes/2510023.md). 

- une variable de type [pdfSignature](../WDLang6/1000022255.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [Toast](../WDLang1/1000024638.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [picTexte](../WDLang1/1000025178.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [diagTexte](../WDLang1/1410088393.md).

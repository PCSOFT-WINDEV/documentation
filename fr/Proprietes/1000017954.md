
## Couleur (Propriété)

***En anglais : Color***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Couleur** permet de connaître et de modifier la couleur d'un élément. Cette propriété est utilisable sur :

- les champs d'une fenêtre ou d'une page. Pour plus de détails, consultez [Couleur sur les champs d'une fenêtre ou d'une page](../Proprietes/2510071.md).

- une variable de type [gglChemin](../WDLang5/1000017433.md).

- une variable de type [gglMarqueur](../WDLang5/1000017442.md).

- une variable de type [gglParamètreCarte](../WDLang5/1000017505.md). 

- une variable de type [Cadre](../Motscles/15140100.md).

- une variable de type [Couleur](../WDLang1/1000019661.md).

- une variable de type [Fond](../WDLang1/1000022038.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [Jeton](../WDLang1/1000023942.md).

- une variable de type [MarqueurImage](../WDLang3/1000025796.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [gglEvénement](../WDLang5/1000017440.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [diagTexte](../WDLang1/1410088393.md).

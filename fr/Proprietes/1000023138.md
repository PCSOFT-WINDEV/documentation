
## MiseEnPage (Propriété)

***En anglais : PageLayout***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **MiseEnPage** permet de connaître et de modifier la mise en page : 

- du diagramme en cours dans le champ Editeur de diagrammes. La mise en page est décrite dans une variable de type [diagMiseEnPage](../WDLang1/1410088610.md).

- de la feuille en cours d'un champ Tableur. La mise en page est décrite dans une variable de type [xlsMiseEnPage](../WDLang5/1000023139.md).



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Feuille en cours [SansMarges]
xMiseEnPage est un xlsMiseEnPage  = TBLR_ChampTableur.MiseEnPage
xMiseEnPage.MargeGauche = 0
xMiseEnPage.MargeDroite = 0
TBLR_ChampTableur.MiseEnPage = xMiseEnpage
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître la mise en page d'un diagramme

`<Résultat> = <Champ Editeur de diagrammes>.MiseEnPage`
---

**`<Résultat> : (Variable de type diagMiseEnPage)`**

Variable de type [diagMiseEnPage](../WDLang1/1410088610.md) correspondant à la mise en page actuelle du diagramme.

**`<Champ Editeur de diagrammes> : (Nom de champ)`**

Nom du champ Editeur de diagrammes manipulé.


<a name="SYNTAXE2"></a>

### Modifier la mise en page d'un diagramme

`<Champ Editeur de diagrammes>.MiseEnPage = <Nouvelle mise en page>`
---

**`<Champ Editeur de diagrammes> : (Nom de champ)`**

Nom du champ Editeur de diagrammes manipulé.

**`<Nouvelle mise en page> : (Variable de type diagMiseEnPage)`**

Nom de la variable de type [diagMiseEnPage](../WDLang1/1410088610.md) correspondant à la nouvelle mise en page du diagramme.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
<a name="NOTE0_2"></a>
Cette mise en page est utilisée lors de l'impression du diagramme via le champ Editeur de diagrammes ou via la fonction [iImprimeDiagramme](../WDLang5/1410088189.md).

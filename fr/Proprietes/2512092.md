


## ValeurParDéfaut (Propriété)

***En anglais : DefaultValue***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **ValeurParDéfaut** permet de :

- Connaître la valeur par défaut d'une rubrique. Cette rubrique a été définie sous l'éditeur d'analyses ou définie par programmation.

- Définir la valeur par défaut d'une rubrique (lors de la description de la rubrique par programmation uniquement).





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Définir la valeur par défaut de la rubrique Commande.Civilité
Civilité.ValeurParDéfaut = 1
...
// Valide la description du fichier de données
HDécritFichier(Commande)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître la valeur par défaut d'une rubrique

`<Résultat> = <Fichier de données>.<Rubrique>.ValeurParDéfaut`
---

**`<Résultat> : (Type de la rubrique)`**

Valeur par défaut de la rubrique.

**`<Fichier de données> : (Chaîne de caractères)`**

Nom du fichier de données manipulé.

**`<Rubrique> : (Chaîne de caractères)`**

Nom de la rubrique manipulée.

### Définir la valeur par défaut d'une rubrique décrite par programmation

`<Rubrique>.ValeurParDéfaut = <Valeur>`
---

**`<Rubrique> : (Chaîne de caractères)`**

Nom de la rubrique manipulée. Ce nom vient d'être défini grâce au type [Description de Rubrique](../WDLang4/1514071.md).

**`<Valeur> : (Type de la rubrique)`**

Valeur par défaut de la rubrique.

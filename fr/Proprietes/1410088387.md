
## mdpsTaille (Propriété)

***En anglais : spLength***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **mdpsTaille** permet de :

- Connaître la taille de la clé après hachage. Cette propriété peut être utilisée pour une rubrique de type "Mot de passe" définie sous l'éditeur d'analyses ou définie par programmation. 

- Définir la taille de la clé après hachage (lors de la description par programmation de la rubrique "Mot de passe" uniquement). 



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Description du fichier UnFichier
...

// Description de la rubrique "Mot_Passe"
MaRubrique est une Description de Rubrique
MaRubrique.Nom = "Mot_Passe"
MaRubrique.Type = hRubMotDePasseSécurisé
MaRubrique.mdpsDérivation = MDPS_PBKDF2
MaRubrique.mdpsHash = MDPS_SHA2_256
MaRubrique.mdpsNombreItération = 20000
MaRubrique.mdpsTaille = 256

// Validation de la description de la rubrique "Mot_Passe"
HDécritRubrique(UnFichier, MaRubrique)

// Validation de la description du fichier UnFichier
...
```





<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître la taille de la clé après hachage (rubrique "Mot de passe")

`<Taille> = <Fichier de données>.<Rubrique>.mdpsTaille`
---

**`<Taille> : (Entier)`**

Taille de la clé après hachage (exprimée en bits). Cette clé doit être un multiple de 8, supérieur à 160 bits et inférieur à 2048 bits. Il est conseillé d'utiliser une clé de 256 ou 512 bits. 

**`<Fichier de données> : (Chaîne de caractères)`**

Nom du fichier manipulé. Ce nom a été défini sous l'éditeur d'analyses ou grâce au type [Description de Fichier](../WDLang4/1514065.md).

**`<Rubrique> : (Chaîne de caractères)`**

Nom de la rubrique manipulée. Ce nom a été défini sous l'éditeur d'analyses ou grâce au type [Description de Rubrique](../WDLang4/1514071.md).  


<a name="SYNTAXE2"></a>

### Définir la taille de la clé après hachage (rubrique "Mot de passe" décrite par programmation)

`<Rubrique>.mdpsTaille = <Taille>`
---

**`<Rubrique> : (Chaîne de caractères)`**

Nom de la rubrique manipulée. Ce nom vient d'être défini grâce au type [Description de Rubrique](../WDLang4/1514071.md).

**`<Taille> : (Entier)`**

Taille de la clé après hachage (exprimée en bits). Cette clé doit être un multiple de 8, supérieur à 160 bits et inférieur à 2048 bits. Il est conseillé d'utiliser une clé de 256 ou 512 bits.

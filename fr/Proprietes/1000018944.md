
## EnDehors (Propriété)

***En anglais : Out***
	

<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **EnDehors** permet de savoir si un email a été lu correctement sur le serveur. 

Cette propriété est utilisable sur :

- la structure [Email](../WDLang3/3032029.md).

- une variable de type [Email](../WDLang3/1000018713.md). 

- une variable de type [tcdPosition](../WDLang1/1000020726.md).

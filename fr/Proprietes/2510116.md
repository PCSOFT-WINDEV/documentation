


## Triable (Propriété)

***En anglais : Sortable***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Triable** permet de :

- Savoir si une colonne d'un champ Table est triable.

- Rendre une colonne d'un champ Table triable ou non.




Une colonne "triable" est une colonne que l'utilisateur pourra utiliser pour trier les informations affichées dans le champ Table.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Permettre la sélection par l'utilisateur de la colonne "COL_NomClient"
// comme critère de tri de la table
COL_NomClient.Triable = Vrai
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Savoir si une colonne est triable

`<Résultat> = <Colonne manipulée>.Triable`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la colonne spécifiée est triable, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Colonne manipulée> : (Nom de champ)`**

Nom de la colonne à manipuler.  


<a name="SYNTAXE2"></a>

### Rendre une colonne triable ou non

`<Colonne manipulée>.Triable = <Triable ?>`
---

**`<Colonne manipulée> : (Nom de champ)`**

Nom de la colonne à manipuler.

**`<Triable ?> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la colonne spécifiée doit être triable, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Colonne "triable"
<a name="colonne_triable_ELTPARAGRAPHE000077"></a>

- Par défaut, lors de sa création, une colonne est triable. Il est possible de modifier cette option sous l'éditeur de fenêtres dans l'onglet "Général" de la description de colonne.

- Lorsque l'option "Triable" est cochée, l'utilisateur pourra trier le champ Table en cliquant sur l'entête de la colonne.

- Si une colonne est triable, une icône représentant une flèche double apparaît dans l'entête de la colonne.



<a name="NOTE0_2"></a>


### Tri dans un champ Table fichier
<a name="tri_dans_champ_table_fichier_ELTPARAGRAPHE000086"></a>Une colonne peut être triée par l'utilisateur uniquement si :

- La rubrique associée à la colonne est une clé.

- Aucun filtre n'existe sur ce fichier de données.


La propriété **Triable** peut tout de même retourner et/ou être initialisée à <u><u><u><u>Vrai</u></u></u></u>. Cependant, aucune icône n'apparaît dans l'entête de la colonne et l'utilisateur ne peut pas trier cette colonne.

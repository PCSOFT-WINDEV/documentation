


## URL (Propriété)

***En anglais : URL***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **URL** permet de :

- Connaître l'adresse associée à un champ.

- Modifier l'adresse associée à un champ.




Ce champ peut correspondre :

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png) sous l'éditeur de fenêtres à un Champ d'affichage HTML.

- sous l'éditeur de pages, à un lien, un bouton, une image cliquable, un champ iFrame, un champ Page cornée, un champ Code-barres ou une option de menu (définie sous l'éditeur ou par programmation).

- sous l'éditeur d'états, à un lien.





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
LIEN_LIEN1.URL = "http://www.pcsoft.fr"
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître l'adresse associée à un champ

`<Résultat> = <Champ manipulé>.URL`
---

**`<Résultat> : (Chaîne de caractères)`**

Adresse associée au champ.

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler :

- Champ d'une fenêtre : champ d'affichage HTML. 

- Champ d'une page : Lien, Bouton, Image, iFrame, Page cornée, Code-barres ou option de menu.

- Champ d'un état : Lien.  





<a name="SYNTAXE2"></a>

### Modifier l'adresse associée à un lien

`<Champ manipulé>.URL = <Nouvelle adresse>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler :

- Champ d'une fenêtre : champ d'affichage HTML. 

- Champ d'une page : Lien, Bouton, Image, iFrame, Page cornée, Code-barres ou option de menu.

- Champ d'un état : Lien.




**`<Nouvelle adresse> : (Tout type)`**

Nouvelle adresse associée au lien. Cette adresse doit contenir "http://".

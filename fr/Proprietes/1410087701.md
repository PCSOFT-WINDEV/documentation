
## AutoFocusAuClic (Propriété)

***En anglais : TapToFocus***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **AutoFocusAuClic** permet de : 

- Savoir si le focus au clic est activé dans un champ Caméra. 

- Activer ou non le focus au clic dans un champ Caméra. 


Si le focus au clic est activé, le focus de la caméra s'effectuera à l'endroit cliqué par l'utilisateur. 

Remarques : 

- Cette propriété est utilisable uniquement pour les champs Caméra qui ne sont pas en "Mode compatible 26". 

- Cette propriété correspond à l'option "Autoriser l'autofocus au clic" disponible dans l'onglet "Général" de la fenêtre de description du champ Caméra. 



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI CAM_MaCaméra.AutoFocusAuClic = Vrai ALORS
	ToastAffiche("Cliquez pour effectuer le focus.")
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Savoir si un champ Caméra autorise le focus au clic

`<Résultat> = <Champ Caméra>.AutoFocusAuClic`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le focus au clic est actif, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 




**`<Champ Caméra> : (Nom de champ)`**

Nom du champ Caméra à manipuler. 


<a name="SYNTAXE2"></a>

### Autoriser ou non le focus au clic dans un champ Caméra

`<Champ Caméra>.AutoFocusAuClic = <Autorisation>`
---

**`<Champ Caméra> : (Nom de champ)`**

Nom du champ Caméra à manipuler. 

**`<Autorisation> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> pour activer le focus au clic, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.

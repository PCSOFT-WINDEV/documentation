
## Virtuel (Propriété)

***En anglais : Virtual***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Virtuel** permet de savoir si un élément est virtuel ou non. Cette propriété est utilisée par :

- une variable de type [Description de procédure](../WDLang1/1000019317.md).

- une variable de type [Description de Propriété](../WDLang1/1000019324.md).

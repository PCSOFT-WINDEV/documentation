
## Latitude (Propriété)

***En anglais : Latitude***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Latitude** permet de connaître et de modifier la latitude d'un élément. Cette propriété est utilisable sur :

- une variable de type [gglCoordonnée](../WDLang5/1000017489.md).

- une variable de type [gglMarqueur](../WDLang5/1000017442.md).

- une variable de type [géoPosition](../WDLang3/1000019191.md).


## AxeYMax (Propriété)

***En anglais : YAxisMax***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **AxeYMax** permet de connaître la borne maximale des ordonnées (axe Y) d'un champ de type Graphe. Cette propriété peut être utilisée par exemple dans l'événement "Zoom" d'un champ Graphe pour identifier la partie affichée.
<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Graphe_axes2%201.gif)



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affiche la borne maximale des ordonnées
Info("La borne maximale de l'axe des Y est " + GRF_MonGraphe.AxeYMax)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Champ Graphe>.AxeYMax`
---

**`<Résultat> : (Réel)`**

Borne maximale des ordonnées.

**`<Champ Graphe> : (Nom de champ)`**

Nom du champ Graphe à manipuler.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- La propriété **AxeYMax** fonctionne uniquement sur les champs de type Graphe. Cette propriété n'est pas utilisable sur les graphes affichés dans une fenêtre ou dans un champ Image.

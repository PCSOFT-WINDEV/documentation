
## Evénement

***En anglais : Event***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le mot-clé **Evénement** peut correspondre :  

- à la fonction **[Evénement](../WDLang1/3015011.md)**, qui permet d'intercepter un événement Windows. 

- à la propriété **Evénement** d'une variable de type [gglAgenda](../WDLang5/1000017418.md).




## EnSaisie (Propriété)

***En anglais : InputEnabled***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **EnSaisie** permet de : 

- Savoir si un champ ou un groupe de champs est en saisie ou en lecture seule.

- Modifier les options de saisie d'un champ ou d'un groupe de champs. 









<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI SAI_Prix < 100 ALORS
	SAI_FraisDeLivraison.EnSaisie = Vrai
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Savoir si un champ ou un groupe de champs est en saisie

`<Résultat> = <Elément manipulé>.EnSaisie`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le champ ou le groupe de champs est en saisie, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 




**`<Elément manipulé> : (Nom de champ)`**

Nom du champ ou du groupe de champs à manipuler. 


<a name="SYNTAXE2"></a>

### Modifier les options de saisie d'un champ ou d'un groupe de champs

`<Elément manipulé>.EnSaisie = <Nouvelle valeur>`
---

**`<Elément manipulé> : (Nom de champ)`**

Nom du champ ou du groupe de champs à manipuler.

**`<Nouvelle valeur> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le champ ou le groupe de champs doit être en saisie, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Limites
<a name="limites_ELTPARAGRAPHE000084"></a>La propriété **EnSaisie** s'applique uniquement : 

- aux champs de saisie,

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) aux combos avec saisie,

- aux champs Table,

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) aux champs Table hiérarchique, 

- aux champs Zone répétée, 

- aux groupes de champs.







<a name="NOTE0_2"></a>


### Cas particuliers
<a name="cas_particuliers_ELTPARAGRAPHE000129"></a>

- Equivalence : La propriété **EnSaisie** à <u><u><u><u>Vrai</u></u></u></u> est équivalente à la propriété [Etat](../Proprietes/2510039.md) égale à la constante *Actif* sauf pour les champs grisés qui conservent l'état grisé.

- Pour les champs Table et Zone répétée, la propriété **EnSaisie** permet de basculer du mode "avec saisie" au mode "sans saisie".




## Mémo (Propriété)

***En anglais : Memo***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Mémo** permet de savoir si une rubrique est de type mémo ou non.

**Remarque** : La rubrique peut avoir été créée sous l'éditeur d'analyses ou par programmation.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// La rubrique Client.Photo est-elle mémo ?
ResMémo = Client.Photo.Mémo
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Fichier de données>.<Rubrique>.Mémo`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la rubrique est de type mémo, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Fichier de données> : (Chaîne de caractères)`**

Nom logique du fichier de données manipulé.

**`<Rubrique> : (Chaîne de caractères)`**

Nom de la rubrique manipulée.


## CouleurTitre (Propriété)

***En anglais : TitleColor***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **CouleurTitre** permet de : 

- connaître la couleur du texte du titre d'une colonne d'un champ Table (ou Table hiérarchique).

- modifier la couleur du texte du titre d'une colonne ou de toutes les colonnes d'un champ Table (ou Table hiérarchique).



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI bCLientImportant ALORS
	COL_Remarque.CouleurTitre = RougeClair
SINON
	COL_Remarque.CouleurTitre = CouleurDefaut
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître la couleur du texte du titre d'une colonne d'un champ Table ou Table hiérarchique

`<Résultat> = <Elément manipulé>.CouleurTitre`
---

**`<Résultat> : (Constante de type Entier)`**

Couleur utilisée actuellement pour le texte du titre de la colonne spécifiée. Pour plus de détails, consultez [couleur prédéfinie du WLangage](../WDLang5/3010002.md). 

**`<Elément manipulé> : (Nom de champ)`**



- Nom de la colonne à manipuler. 

- Nom du champ Table. Dans ce cas, seule la première colonne du champ Table est considérée. 

- Nom du champ Table hiérarchique. Dans ce cas, seule la première colonne du champ Table hiérarchique est considérée.





<a name="SYNTAXE2"></a>

### Modifier la couleur du texte du titre d'une ou de toutes les colonnes d'un champ Table ou Table hiérarchique

`<Elément manipulé>.CouleurTitre = <Nouvelle couleur>`
---

**`<Elément manipulé> : (Nom de champ)`**



- Nom de la colonne à manipuler.

- Nom du champ Table (pour modifier la couleur du titre de toutes les colonnes du champ Table). 

- Nom du champ Table hiérarchique (pour modifier la couleur du titre de toutes les colonnes du champ Table hiérarchique). 




**`<Nouvelle couleur> : (Entier ou constante de type Entier)`**

Nouvelle couleur du texte du titre. Ce paramètre peut correspondre à l'une des options suivantes : 

- une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)),

- une couleur TSL (obtenue avec la fonction [TSL](../WDLang1/3029057.md)),

- [une couleur prédéfinie du WLangage](../WDLang5/3010002.md).

- la constante *CouleurDéfaut*. Dans ce cas, le titre de la colonne utilisera la couleur commune à toutes les colonnes (c'est-à-dire la couleur définie dans le style du champ Table).

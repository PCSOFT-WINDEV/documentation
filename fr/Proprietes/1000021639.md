


## MultiFeuille (Propriété)

***En anglais : MultiWorksheet***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **MultiFeuille** permet de : 

- Savoir si un champ Tableur gère plusieurs feuilles de calcul. 

- Modifier le mode de gestion des feuilles de calcul d'un champ Tableur. 



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Passe le champ Tableur en mode multi feuille pour charger un fichier xlsx
TBLR_MonTableur.MultiFeuille = Vrai
TableurCharge(TBLR_MonTableur, "DocComplet.xlsx")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître le mode de gestion des feuilles de calcul

`<Résultat> = <Champ Tableur>.MultiFeuille`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le champ Tableur gère plusieurs feuilles de calcul, 

- <u><u><u><u>Faux</u></u></u></u> si le champ Tableur gère une seule feuille de calcul.  




**`<Champ Tableur> : (Nom de champ)`**

Nom du champ Tableur à manipuler.


<a name="SYNTAXE2"></a>

### Modifier le mode de gestion des feuilles de calcul

`<Champ Tableur>.MultiFeuille = <Nouveau mode de gestion>`
---

**`<Champ Tableur> : (Nom de champ)`**

Nom du champ Tableur à manipuler. 

**`<Nouveau mode de gestion> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le champ Tableur doit gérer plusieurs feuilles de calcul, 

- <u><u><u><u>Faux</u></u></u></u> si le champ Tableur doit gérer une seule feuille de calcul.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Si le mode de gestion multifeuille est activé : 

- Des boutons de gestion des feuilles de calcul sont affichés dans le champ. 

- Les documents XLSX chargés et sauvés gèrent toutes les feuilles (fonctions [TableurCharge](../WDLang1/1000021340.md) et [TableurSauve](../WDLang1/1000021341.md)).


Si le mode de gestion multifeuille n'est pas activé : 

- La fonction [TableurCharge](../WDLang1/1000021340.md) ne charge qu'une seule feuille de calcul. 

- Aucun bouton de gestion des feuilles de calcul n'est proposé.

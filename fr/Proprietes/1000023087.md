
## Société (Propriété)

***En anglais : Company***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Société** permet de connaître et de modifier la société associée à un élément. Cette propriété est disponible sur : 

- les variables de type [gglContact](../WDLang5/1000017434.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) les variables de type [docPropriété](../WDLang1/1000022945.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) les variables de type [adminWBCompte](../WDLang2/1410089382.md).

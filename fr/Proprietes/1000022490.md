
## Elément (Propriété)

***En anglais : Element***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Elément** permet de manipuler les éléments d'un document Traitement de texte. Cette propriété est utilisable sur :

- les variables de type [docParagraphe](../WDLang1/1000022483.md).

- les variables de type [docFragment](../WDLang1/1000022482.md).

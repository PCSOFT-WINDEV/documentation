
## TrameFond (Propriété)

***En anglais : BackgroundPattern***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **TrameFond** permet de connaître et de modifier la couleur de la trame de fond d'un élément. Cette propriété est utilisable sur :

- les variables de type [docCellule](../WDLang1/1000022927.md).

- les variables de type [docFragment](../WDLang1/1000022482.md).

- les variables de type [docParagraphe](../WDLang1/1000022483.md).

- les variables de type [docNiveauNumérotation](../WDLang1/1000022799.md).

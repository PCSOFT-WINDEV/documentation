
## CléReliée (Propriété)

***En anglais : LinkedKey***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **CléReliée** permet de paramétrer les caractéristiques d'une liaison. Il est ainsi possible de :

- Connaître la rubrique du fichier de données relié utilisée dans la liaison.

- Modifier la rubrique du fichier de données relié utilisée dans la liaison.



<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=cleReliee.gif)





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Modifier la clé reliée
LiaisonClubPersonne.CléReliée = Personne.IDClub
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître la clé reliée de la liaison

`<Nom de la clé> = <Nom de la liaison>.CléReliée`
---

**`<Nom de la clé> : (Chaîne de caractères)`**

Nom de la clé utilisée par la liaison dans le fichier de données relié.

**`<Nom de la liaison> : (Chaîne de caractères)`**

Nom de la liaison définie sous l'éditeur d'analyses ou définie grâce au type [Description de Liaison](../WDLang4/1514039.md).  


<a name="SYNTAXE2"></a>

### Modifier la clé reliée de la liaison

`<Nom de la liaison>.CléReliée = <Nouvelle clé>`
---

**`<Nom de la liaison> : (Chaîne de caractères)`**

Nom de la liaison définie sous l'éditeur d'analyses ou définie grâce au type [Description de Liaison](../WDLang4/1514039.md).

**`<Nouvelle clé> : (Chaîne de caractères)`**

Nom de la nouvelle clé utilisée par la liaison dans le fichier de données relié. Cette clé a été définie sous l'éditeur d'analyses ou grâce au type [Description de Rubrique](../WDLang4/1514071.md).

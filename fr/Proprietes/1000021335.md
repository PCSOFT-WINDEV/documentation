
## EntêteVisible (Propriété)

***En anglais : HeaderVisible***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **EntêteVisible** permet de : 

- Savoir si les entêtes de lignes et colonnes sont affichés dans un champ Tableur. 

- Afficher ou non les entêtes de lignes et colonnes dans un champ Tableur.




Rappel : 

- Dans un champ Tableur, les entêtes permettent d'identifier rapidement les cellules.

- Cette propriété correspond à l'option "Afficher les entêtes de lignes et de colonnes" présente dans l'onglet "Détail" de la fenêtre de description du champ Tableur.



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI INT_Entête ALORS
	TBLR_Tableur.EntêteVisible = Vrai
SINON
	TBLR_Tableur.EntêteVisible = Faux
FIN
```

<a name="SYNTAXE1"></a>

### Savoir si les entêtes de ligne ou de colonne sont visibles

`<Résultat> = <Champ Tableur>.EntêteVisible`
---

**`<Résultat> : (Booléen)`**



- <u><u><u>Vrai</u></u></u> si les entêtes sont visibles, 

- <u><u><u>Faux</u></u></u> dans le cas contraire.




**`<Champ Tableur> : (Nom de champ)`**

Nom du champ Tableur à manipuler.


<a name="SYNTAXE2"></a>

### Afficher ou non les entêtes de lignes et colonnes 

`<Champ Tableur>.EntêteVisible = <Etat de l'entête>`
---

**`<Champ Tableur> : (Nom de champ)`**

Nom du champ Tableur à manipuler. 

**`<Etat de l'entête> : (Booléen)`**



- <u><u><u>Vrai</u></u></u> si l'entête est visible, 

- <u><u><u>Faux</u></u></u> dans le cas contraire.

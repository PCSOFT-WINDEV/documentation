


## FormeExtrémitéArrivée (Propriété)

***En anglais : ShapeExtremityEnd***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **FormeExtrémitéArrivée** permet de connaître et de modifier le type de forme utilisé pour l'extrémité d'arrivée d'un champ Fil.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Création et initialisation d'un champ Fil 
// Ce fil relie le champ BTN_ChampA et le champ BTN_ChampB
NomFil est un Champ

NomFil <- ChampCrée("", typFil)
NomFil.ChampDépart = BTN_ChampA
NomFil.ChampArrivée = BTN_ChampB
NomFil.FormeExtrémitéDépart = formeLosange
NomFil.FormeExtrémitéArrivée = formeRond
NomFil.FormeExtrémitéTaille = 8
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître le type de forme utilisé pour l'extrémité d'arrivée d'un champ Fil

`<Résultat> = <Champ Fil>.FormeExtrémitéArrivée`
---

**`<Résultat> : (Constante de type Entier)`**

Type de forme utilisé : 


|   |   |
| --- | --- |
| *formeAucune* | Aucune forme. |
| *formeCarré* | Forme de l'extrémité d'arrivée : Carré. |
| *formeFlèche* | Forme de l'extrémité d'arrivée : Flèche. |
| *formeLosange* | Forme de l'extrémité d'arrivée : Losange. |
| *formeRond* | Forme de l'extrémité d'arrivée : Rond. |
| *formeTriangle* | Forme de l'extrémité d'arrivée : Triangle. |



**`<Champ Fil> : (Nom de champ)`**

Nom du champ Fil à manipuler. 


<a name="SYNTAXE2"></a>

### Modifier le type de forme utilisé pour l'extrémité d'arrivée d'un champ Fil

`<Champ Fil>.FormeExtrémitéArrivée = <Nouvelle forme>`
---

**`<Champ Fil> : (Nom de champ)`**

Nom du champ Fil à manipuler. 

**`<Nouvelle forme> : (Constante de type Entier)`**

Nouvelle forme à utiliser à l'arrivée du champ Fil : 


|   |   |
| --- | --- |
| *formeAucune* | Aucune forme. |
| *formeCarré* | Forme de l'extrémité d'arrivée : Carré. |
| *formeFlèche* | Forme de l'extrémité d'arrivée : Flèche. |
| *formeLosange* | Forme de l'extrémité d'arrivée : Losange. |
| *formeRond* | Forme de l'extrémité d'arrivée : Rond. |
| *formeTriangle* | Forme de l'extrémité d'arrivée : Triangle. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- La couleur de la forme au départ et à l'arrivée du champ Fil est fixée par la propriété [Couleur](../Proprietes/2510071.md).

- La taille de la forme au départ et à l'arrivée du champ Fil est fixée par la propriété [FormeExtrémitéTaille](../Proprietes/1000021960.md).

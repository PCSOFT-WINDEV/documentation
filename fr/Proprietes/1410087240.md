
## LargeurAvecAnimation (Propriété)

***En anglais : WidthWithAnimation***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **LargeurAvecAnimation** permet de modifier la largeur d'une cellule d'un champ Disposition en jouant l'animation "Changement de taille d'une cellule" définie : 

- soit dans l'onglet "Avancé" de la fenêtre de description du projet (bouton "Animation des champs"). 

- soit dans l'onglet "Style" du champ ou du projet. 



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Augmenter de 100 px la largeur de la 3ème cellule avec animation
DISP_Données[3].LargeurAvecAnimation = DISP_Données[3].Largeur + 100
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE2"></a>

### Modifier la largeur d'une cellule d'un champ Disposition en jouant une animation

`<Cellule>.LargeurAvecAnimation = <Nouvelle largeur>`
---

**`<Cellule> : (Nom de champ)`**

Cellule d'un champ Disposition. La notation utilisée pour désigner la cellule du champ Disposition est la suivante : 

```txt
<Champ Disposition>[<Numéro cellule>]
```


**`<Nouvelle largeur> : (Entier)`**

Nouvelle largeur du champ. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Les propriétés de l'animation (durée par exemple) peuvent être décrites au niveau du projet et surchargées dans le style du champ. 

- Cette propriété permet de redimensionner le champ Disposition avec une animation.

- Cette propriété fonctionne uniquement sur les champs Dispositions horizontaux (avec les cellules placées les unes à coté des autres).

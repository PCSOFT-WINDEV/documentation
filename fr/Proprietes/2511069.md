
## HauteurEtiquette (Propriété)

***En anglais : LabelHeight***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **HauteurEtiquette** permet de :

- Connaître la hauteur des étiquettes d'un état de type Étiquette.

- Modifier la hauteur des étiquettes d'un état de type Étiquette.




**Remarque** : La hauteur des étiquettes a été définie lors de la création de l'état de type "Étiquette". Cette option peut être connue et/ou modifiée dans le format de la page (option "Format des étiquettes" dans le menu contextuel de l'état).

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Hauteuretiquette.gif)





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Augmente la hauteur des étiquettes de l'état "ETAT_EtiqClient"
ETAT_EtiqClient.HauteurEtiquette = ETAT_EtiqClient.HauteurEtiquette + 35
// Est équivalent à : ETAT_EtiqClient.HauteurEtiquette += 35
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître la hauteur des étiquettes d'un état

`<Résultat> = <Etat manipulé>.HauteurEtiquette`
---

**`<Résultat> : (Réel)`**

Hauteur (en millimètres) des étiquettes de l'état spécifié.

**`<Etat manipulé> : (Nom d'état)`**

Nom de l'état à manipuler.  


<a name="SYNTAXE2"></a>

### Modifier la hauteur des étiquettes d'un état

`<Etat manipulé>.HauteurEtiquette = <Nouvelle hauteur>`
---

**`<Etat manipulé> : (Nom d'état)`**

Nom de l'état à manipuler.

**`<Nouvelle hauteur> : (Réel)`**

Nouvelle hauteur (en millimètres) des étiquettes de l'état spécifié.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
La propriété **HauteurEtiquette** s'applique uniquement aux états de type "Étiquette".

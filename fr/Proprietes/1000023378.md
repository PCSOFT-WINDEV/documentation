


## ModeAffichage (Propriété)

***En anglais : DisplayMode***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **ModeAffichage** permet de connaître et de modifier : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) le mode d'affichage d'un champ Traitement de texte, 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) le mode d'affichage d'un champ Lecteur PDF,

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) le mode d'affichage d'un champ Editeur HTML, 

- le mode d'affichage d'un champ Kanban. 





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Bouton [Mode Web]
TT_ChampDoc.ModeAffichage = modeAffichageWeb
```



```wl
SI TypeEcran() = teTablette ALORS
	KANBAN_TACHES.ModeAffichage = modeAffichageKanbanMultiListe
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître le mode d'affichage d'un champ Bureautique

`<Résultat> = <Champ manipulé>.ModeAffichage`
---

**`<Résultat> : (Constante de type Entier)`**

Mode d'affichage actuel du champ : 



|   |   |
| --- | --- |
| *modeAffichageCodeSourceHtml* | Le code source HTML est affiché dans le champ Editeur HTML.   <br><br>Remarque : Cette constante est disponible uniquement pour le champ Editeur HTML. |
| *modeAffichageContinu* | Les pages du document sont affichées en continu à l'écran. <br><br>Remarque : Cette constante est disponible uniquement pour le champ Editeur HTML.  |
| *modeAffichageDeuxPages* | Les pages du document sont affichées deux par deux à l'écran.  <br><br>Remarque : Cette constante n'est pas disponible pour le champ Editeur HTML. |
| *modeAffichageDeuxPagesAvecCouverture* | Les pages du document sont affichées deux par deux à l'écran. La première page est considérée comme une couverture. La première page est affichée à droite. <br><br>Remarque : Cette constante n'est pas disponible pour le champ Traitement de texte ou le champ Editeur HTML. |
| *modeAffichageKanbanMonoListe* | Une seule liste sera affichée dans le champ Kanban. Les autres listes seront accessibles via l'ascenseur horizontal. Cette constante correspond à l'option "Liste centrale avec défilement" dans la fenêtre de description du champ Kanban (onglet "Détails").   <br><br>Remarque : Cette constante est disponible uniquement pour le champ Kanban. |
| *modeAffichageKanbanMultiListe* | Plusieurs listes pourront être affichées simultanément dans le champ Kanban. Cette constante correspond à l'option "Plusieurs listes" dans la fenêtre de description du champ Kanban (onglet "Détails").       <br><br>Remarque : Cette constante est disponible uniquement pour le champ Kanban. |
| *modeAffichageUnePage*<br>(Valeur par défaut) | Le document est affiché tel qu'il sera imprimé. <br><br>Remarque : Cette constante n'est pas disponible pour le champ Editeur HTML. |
| *modeAffichageWeb* | Le document est affiché sous la forme d'une page Web. <br><br>Remarque : Cette constante n'est pas disponible pour le champ Lecteur PDF. |



**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler : 

- Champ Traitement de texte.

- Champ Lecteur PDF à manipuler. 

- Champ Editeur HTML.  

- Champ Kanban.





<a name="SYNTAXE2"></a>

### Modifier le mode d'affichage d'un champ Bureautique

`<Champ manipulé>.ModeAffichage = <Nouveau mode>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler : 

- Champ Traitement de texte.

- Champ Lecteur PDF à manipuler.

- Champ Editeur HTML.

- Champ Kanban.




**`<Nouveau mode> : (Constante de type Entier)`**

Mode d'affichage à appliquer au champ : 



|   |   |
| --- | --- |
| *modeAffichageCodeSourceHtml* | Le code source HTML est affiché dans le champ Editeur HTML.   <br><br>Remarque : Cette constante est disponible uniquement pour le champ Editeur HTML. |
| *modeAffichageContinu* | Les pages du document sont affichées en continu à l'écran. <br><br>Remarque : Cette constante est disponible uniquement pour le champ Editeur HTML.  |
| *modeAffichageDeuxPages* | Les pages du document sont affichées deux par deux à l'écran.  <br><br>Remarque : Cette constante n'est pas disponible pour le champ Editeur HTML. |
| *modeAffichageDeuxPagesAvecCouverture* | Les pages du document sont affichées deux par deux à l'écran. La première page est considérée comme une couverture et est affichée à droite. <br><br>Remarques : <br><br>- L'utilisation de cette constante change le mode d'affichage mais ne change pas le zoom du document dans le champ Lecteur PDF. Pour modifier le zoom, utilisez la propriété [Zoom](../Proprietes/1000017201.md). <br><br>- Cette constante n'est pas disponible pour le champ Traitement de texte ou le champ Editeur HTML. <br><br><br> |
| *modeAffichageKanbanMonoListe* | Une seule liste sera affichée dans le champ Kanban. Les autres listes seront accessibles via l'ascenseur horizontal. Cette constante correspond à l'option "Liste centrale avec défilement" dans la fenêtre de description du champ Kanban (onglet "Détails").   <br><br>Remarque : Cette constante est disponible uniquement pour le champ Kanban. |
| *modeAffichageKanbanMultiListe* | Plusieurs listes pourront être affichées simultanément dans le champ Kanban. Cette constante correspond à l'option "Plusieurs listes" dans la fenêtre de description du champ Kanban (onglet "Détails").       <br><br>Remarque : Cette constante est disponible uniquement pour le champ Kanban. |
| *modeAffichageUnePage*<br>(Valeur par défaut) | Le document est affiché tel qu'il sera imprimé. <br><br>Remarque : Cette constante n'est pas disponible pour le champ Editeur HTML. |
| *modeAffichageWeb* | Le document est affiché sous la forme d'une page Web. <br><br>Remarque : Cette constante n'est pas disponible pour le champ Lecteur PDF. |

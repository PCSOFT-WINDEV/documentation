


## RépertoireRéplication (Propriété)

***En anglais : ReplicationDirectory***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **RépertoireRéplication** permet de gérer le répertoire du réplica (fichier ".RPL" ou ".RPM"). Il est ainsi possible de :

- Connaître le répertoire du réplica pour un fichier de données défini sous l'éditeur d'analyses ou défini par programmation.

- Définir le répertoire du réplica lors de la description par programmation d'un fichier de données.



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Répertoire du réplica du fichier de données Client
Info(Client.RépertoireRéplication)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le répertoire du réplica d'un fichier de données

`<Répertoire actuel> = <Fichier de données>.RépertoireRéplication`
---

**`<Répertoire actuel> : (Chaîne de caractères)`**



- Nom du répertoire du réplica du fichier de données spécifié, 

- "." si le répertoire correspond au répertoire en cours.




**`<Fichier de données> : (Chaîne de caractères)`**

Nom logique du fichier de données manipulé. Ce nom a été défini sous l'éditeur d'analyses ou grâce au type [Description de Fichier](../WDLang4/1514065.md).  


<a name="SYNTAXE2"></a>

### Définir le répertoire du réplica lors de la description par programmation d'un fichier de données

`<Fichier de données>.RépertoireRéplication = <Nouveau répertoire>`
---

**`<Fichier de données> : (Chaîne de caractères)`**

Nom logique du fichier de données défini grâce au type [Description de Fichier](../WDLang4/1514065.md).

**`<Nouveau répertoire> : (Chaîne de caractères)`**

Nom du répertoire du réplica.




## PoliceSoulignée (Propriété)

***En anglais : FontUnderlined***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **PoliceSoulignée** permet de :

- Savoir si le texte affiché est souligné ou non.

- Souligner ou non le texte.







<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Souligner le texte affiché dans le champ "LIB_ChampLibellé"
LIB_ChampLibellé.PoliceSoulignée = Vrai
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Savoir si le texte affiché dans un champ est souligné

`<Résultat> = <Champ manipulé>.PoliceSoulignée`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le texte est souligné, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler.  


<a name="SYNTAXE2"></a>

### Modifier la mise en forme souligné du texte affiché dans un champ

`<Champ manipulé>.PoliceSoulignée = <Souligné/Non souligné>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler.

**`<Souligné/Non souligné> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le texte affiché doit être souligné, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Limites
<a name="limites_ELTPARAGRAPHE000121"></a>![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Dans un état, la propriété **PoliceSoulignée** peut être utilisée uniquement sur :

- un champ Libellé.

- un champ Calculé.

- un champ Lien URL.

- un champ Prédéfini.


![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Dans une fenêtre, la propriété **PoliceSoulignée** ne peut pas être utilisée sur :

- un champ Spin,

- un champ Web Camera,

- un champ Conférence.


Remarque : Cette propriété s'applique sur les champs "Liste image" en mode d'affichage "Liste classique" uniquement.

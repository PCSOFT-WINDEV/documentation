
## Epaisseur (Propriété)

***En anglais : Thickness***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Epaisseur** permet de connaître et de modifier l'épaisseur d'un élément. Cette propriété est utilisable sur :

- les champs d'une fenêtre : 

	- les graphes de type courbe. 

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) les champs Fil. 


 Pour plus de détails, consultez [Epaisseur](../Proprietes/2510144.md).

- une variable de type [gglParamètreCarte](../WDLang5/1000017505.md). 

- une variable de type [gglChemin](../WDLang5/1000017433.md).

- une variable de type [Cadre](../Motscles/15140100.md).

- une variable de type [CarteCercle](../WDLang3/1000025504.md).

- une variable de type [CartePolygone](../WDLang3/1000025476.md).

- une variable de type [CartePolyligne](../WDLang3/1000025489.md).


## SectionCritique

***En anglais : CriticalSection***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le mot-clé **SectionCritique** peut correspondre :

- à la fonction [SectionCritique](../WDLang1/1000024373.md) qui permet d'activer une section critique nommée ou sur une variable dans une instruction de type [UTILISER ... DANS](../Motscles/1510025.md).

- au type de variable [SectionCritique](../WDLang1/1000023972.md) qui permet de définir des sections critiques afin de limiter l'exécution simultanée d'un code (procédure, ligne de code, ...) **à un thread** à un instant donné dans une seule application.




## TexteSélectionnéAvecBalise (Propriété)

***En anglais : SelectedTextWithTag***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **TexteSélectionnéAvecBalise** permet de récupérer ou de modifier le texte sélectionné dans un champ de saisie riche (au format RTF ou HTML) avec les balises de formatage HTML ou RTF.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonTexte est une chaîne 
MonTexte = SAI_HTML.TexteSélectionnéAvecBalise
SI Position(MonTexte, "<A ") > 0 ALORS
	Message("En sélection d'un lien")
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Récupérer le texte sélectionné avec les balises

`<Texte> = <Champ de saisie>.TexteSélectionnéAvecBalise`
---

**`<Texte> : (Chaîne de caractères)`**

Texte contenu dans le champ de saisie avec les balises de formatage.

**Attention** : Si le champ est un champ Unicode, la chaîne de caractères renvoyée est une chaîne de caractères Unicode.

**`<Champ de saisie> : (Nom de champ)`**

Nom du champ de saisie riche à manipuler.


<a name="SYNTAXE2"></a>

### Modifier le texte sélectionné avec les balises

`<Champ de saisie>.TexteSélectionnéAvecBalise = <Nouveau texte>`
---

**`<Champ de saisie> : (Nom de champ)`**

Nom du champ de saisie riche à manipuler.

**`<Nouveau texte> : (Chaîne de caractères)`**

Nouveau texte à afficher dans le champ de saisie avec les balises de formatage.

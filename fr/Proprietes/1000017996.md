


## DateDébut (Propriété)

***En anglais : StartDate***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **DateDébut** permet de connaître et de modifier la date de début d'une période. Cette propriété est utilisable sur :

- les champs Calendrier, Agenda et Planning. Pour plus de détails, consultez [Propriété DateDébut](../Proprietes/1000017426.md).

- une variable de type [saasAbonnement](../WDLang3/1000019088.md).

- une variable de type [gglEvénement](../WDLang5/1000017440.md). 

- une variable de type [RendezVous](../WDLang1/1000019244.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [TâcheGantt](../WDLang1/1000020881.md).

- une variable de type [hOpérationServeur](../WDLang4/1000022471.md).

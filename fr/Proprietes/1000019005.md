


## NbJourAffiché (Propriété)

***En anglais : NbDayDisplayed***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **NbJourAffiché** permet de : 

- connaître et de modifier le nombre de jours affichés dans un champ Agenda ou dans un champ Planning. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) connaître le nombre de jours affichés dans un champ Diagramme de Gantt présent dans un état. 




**Remarque** : Ce nombre de jours est utilisé uniquement pour l'affichage en mode "semaine".


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affiche 2 jours dans le champ Planning
PLN_Planning1.NbJourAffiché = 2
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le nombre de jours affichés

`<Résultat> = <Champ manipulé>.NbJourAffiché`
---

**`<Résultat> : (Entier)`**

Nombre de jours affichés dans le champ. 

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ manipulé : 

- Champ Agenda.

- Champ Planning. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Champ Diagramme de Gantt d'un état.





<a name="SYNTAXE2"></a>

### Modifier le nombre de jours affichés

`<Champ manipulé>.NbJourAffiché = <Nouveau nombre de jours>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ manipulé : 

- Champ Agenda.

- Champ Planning. 




**`<Nouveau nombre de jours> : (Entier)`**

Nombre de jours à afficher dans le champ. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- **Utilisation de la fonction AgendaChangeMode** : 
	La fonction [AgendaChangeMode](../WDLang1/1000019497.md) permet de modifier la plage temporelle affichée par un champ Agenda (Jour/Semaine/Mois). Si le nombre de jours affichés a été paramétré avec la propriété **NbJourAffiché**, la fonction [AgendaChangeMode](../WDLang1/1000019497.md) écrase cette valeur avec la valeur spécifiée lors de l'appel à la fonction. 

- **Utilisation de la fonction PlanningChangeMode** : 
	La fonction [PlanningChangeMode](../WDLang1/1000019440.md) permet de modifier la plage temporelle affichée par un champ Planning (Jour/Semaine/Mois). Si le nombre de jours affichés a été paramétré avec la propriété **NbJourAffiché**, la fonction [PlanningChangeMode](../WDLang1/1000019440.md) écrase cette valeur avec la valeur spécifiée lors de l'appel à la fonction.




## NuméroPage (Propriété)

***En anglais : PageNumber***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **NuméroPage** permet de : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) connaître et de modifier le numéro de la page affichée dans un champ Image (fichiers image de type "multi-pages").

- connaître et de modifier le numéro de la page d'un fichier PDF affiché dans un champ Image d'une fenêtre. 

- connaître et de modifier le numéro de la page d'un fichier PDF affiché en fond d'un état. 




**Rappel** : Certains formats d'images (le format TIFF par exemple) peuvent contenir plusieurs images dans le même fichier. Chaque image correspond à une page du fichier. Ces formats sont appelés format "multi-pages".

Remarque : Pour connaître et modifier le numéro de la page affichée dans un champ Traitement de texte ou dans un champ Lecteur PDF, utilisez la propriété [NuméroPageAffichée](../Proprietes/1000023479.md). 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png) 
```wl
// Variables
i est un entier
// Modification du fichier image associé au champ "IMG_MonImage" dans une fenêtre
IMG_MonImage = "FichierMultipage.TIF"
// Affichage successif des différentes pages présentes dans le fichier image
POUR i = 1 A IMG_MonImage.NombrePage
	IMG_MonImage.NuméroPage = i
	// Temporisation du changement des images
	Multitâche(-100)
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le numéro de la page affichée

`<Numéro de la page> = <Elément manipulé>.NuméroPage`
---

**`<Numéro de la page> : (Entier)`**



- Numéro de la page affichée actuellement dans le champ Image spécifié ou en fond de l'état spécifié.

- 1 si le format du fichier image n'est pas "multi-pages" ou si la première page du fichier "multi-pages" est affichée.

- 0 si le champ ou l'état spécifié n'est pas associé à un format d'image supporté.




**`<Elément manipulé> : (Nom de champ ou nom d'état)`**

Nom du champ Image ou de l'état à manipuler.  


<a name="SYNTAXE2"></a>

### Modifier le numéro de la page affichée

`<Elément manipulé>.NuméroPage = <Numéro de la page>`
---

**`<Elément manipulé> : (Nom de champ ou nom d'état)`**

Nom du champ Image ou de l'état à manipuler.

**`<Numéro de la page> : (Entier)`**

Numéro de la page à afficher dans le champ Image ou en fond de l'état spécifié.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Modifier la page affichée dans un champ Image
<a name="modifier_page_affichee_dans_champ_image_ELTPARAGRAPHE000072"></a>Lorsqu'un fichier image ou PDF est associé à un champ (directement depuis l'éditeur de fenêtres, grâce à la propriété [Image](../Proprietes/2510034.md), ...), l'image affichée par défaut correspond à la première page du fichier "multi-pages".

Pour modifier la page affichée dans le champ Image, il suffit d'utiliser la propriété **NuméroPage**.

**Attention** : La propriété **NuméroPage** s'applique uniquement sur le fichier image actuellement associé au champ Image.
<a name="NOTE0_2"></a>


### Modifier la page du PDF affichée en fond d'un état
<a name="modifier_page_pdf_affichee_fond_etat_ELTPARAGRAPHE000092"></a>Si un fichier PDF est utilisé comme formulaire d'un état, il est possible de choisir dans la description de l'état, la page du fichier PDF à afficher (onglet "Fond", option "N° de page PDF"). Pour modifier par programmation la page affichée en fond de l'état, il suffit d'utiliser la propriété **NuméroPage**.

**Attention** : La propriété **NuméroPage** s'applique uniquement sur le fichier image actuellement utilisé comme formulaire de l'état.
<a name="NOTE0_3"></a>


### Nombre de pages présentes dans un fichier "multi-pages"
<a name="nombre_pages_presentes_dans_fichier_multipages_ELTPARAGRAPHE000107"></a>Pour connaître le nombre de pages présentes dans un fichier image de type "multi-pages", utilisez la propriété [NombrePage](../Proprietes/2510084.md).
<a name="NOTE0_4"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) 

### Dessin dans un champ Image associé à un image "multi-pages"
<a name="dessin_dans_champ_image_associe_image_multipages_ELTPARAGRAPHE000119"></a>Si un dessin (réalisé avec les [fonctions de dessin](../WDLang1/3029033.md)) est effectué dans le champ Image, ce dessin sera automatiquement effacé lors de la modification de la page affichée avec la propriété **NuméroPage**.

Il est nécessaire de re-dessiner ce dessin chaque fois que la page affichée sera modifiée. Pour re-dessiner ce dessin, il est conseillé d'utiliser une procédure.
<a name="NOTE0_5"></a>


### Limites
<a name="limites_ELTPARAGRAPHE000134"></a>La propriété **NuméroPage** s'applique uniquement :

- aux champs de type image présents dans les fenêtres.

- aux états.

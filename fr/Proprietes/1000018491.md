
## IDUtilisateur (Propriété)

***En anglais : UserID***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **IDUtilisateur** permet de connaître l'identifiant de l'utilisateur. Cette propriété est utilisable sur :

- une variable de type [sfInfoSession](../WDLang5/1000018416.md).

- une variable de type [sfInfoUtilisateur](../WDLang5/1000018423.md).

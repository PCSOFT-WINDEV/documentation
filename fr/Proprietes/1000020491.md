
## Planification (Propriété)

***En anglais : Scheduling***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Planification** permet de connaître et de modifier les options de planification. Cette propriété est utilisable sur :

- une variable de type [hOptimisationPlanifiée](../WDLang4/1000018854.md). 

- une variable de type [hRSConfig](../WDLang4/1000020484.md). 

- une variable de type [hPlanificationVueMatérialisée](../WDLang4/1000021444.md).


## PiedDePage (Propriété)

***En anglais : PageFooter***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **PiedDePage** permet de connaître et de modifier la hauteur d'un pied de page. Cette propriété est utilisable sur :

- une variable de type [docSection](../WDLang1/1000022685.md).

- une variable de type [xlsMiseEnPage](../WDLang5/1000023139.md).

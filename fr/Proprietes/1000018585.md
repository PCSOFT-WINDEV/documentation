
## IDApprobateurSuivant (Propriété)

***En anglais : NextApproverID***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **IDApprobateurSuivant** permet de gérer les utilisateurs réalisant la prochaine action. Cette propriété est utilisable sur :

- une variable de type [sfRequêteTraitementSoumission](../WDLang5/1000018516.md).

- une variable de type [sfRequêteTraitementTâche](../WDLang5/1000018520.md).

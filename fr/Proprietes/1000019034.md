


## TypeFichiers (Propriété)

***En anglais : TypeFiles***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **TypeFichiers** permet de connaître et de modifier le filtre sur les fichiers proposé dans le sélecteur de fichiers du champ Upload.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
UPL_Upload1.TypeFichiers = "Fichier Texte (*.txt)" + TAB + "*.txt" + RC + "Tous" + TAB + "*.*"
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître le filtre des fichiers proposé par le champ Upload

`<Résultat> = <Champ Upload>.TypeFichiers`
---

**`<Résultat> : (Chaîne de caractères)`**

Description des types de fichiers pouvant être sélectionnés. Cette description est de la forme :

```txt
<Types Fichiers>+TAB+<Liste Extensions>[+RC+ ...
	<Types Fichiers 2>+TAB+<Liste Extensions 2>]
```
Où :
&lt;Types Fichiers&gt; : chaîne de caractères décrivant le type de fichier ("BMP (\*.BMP;\*.RLE)" par exemple).
&lt;Liste Extensions&gt; : extensions correspondant au type décrit ("\*.BMP;\*.RLE" par exemple).

Par exemple :
"BMP (\*.BMP;\*.RLE)" + TAB + "\*.BMP;\*.RLE" + RC + "Icône (\*.ICO)" + TAB + "\*.ICO" + RC + "Autres" + TAB + "\*.GIF;\*.TIF;\*.JPG"

**Remarque** : Si ce paramètre correspond à une chaîne vide (""), tous les types de fichiers seront proposés. 

**`<Champ Upload> : (Nom de champ)`**

Nom du champ Upload manipulé. 


<a name="SYNTAXE2"></a>

### Modifier le filtre des fichiers proposé par le champ Upload

`<Champ Upload>.TypeFichiers = <Nouveau filtre>`
---

**`<Champ Upload> : (Nom de champ)`**

Nom du champ Upload manipulé

**`<Nouveau filtre> : (Chaîne de caractères)`**

Description des types de fichiers pouvant être sélectionnés. Cette description est de la forme :

```txt
<Types Fichiers>+TAB+<Liste Extensions>[+RC+ ...
	<Types Fichiers 2>+TAB+<Liste Extensions 2>]
```
Où :
&lt;Types Fichiers&gt; : chaîne de caractères décrivant le type de fichier ("BMP (\*.BMP;\*.RLE)" par exemple).
&lt;Liste Extensions&gt; : extensions correspondant au type décrit ("\*.BMP;\*.RLE" par exemple).

Par exemple :
"BMP (\*.BMP;\*.RLE)" + TAB + "\*.BMP;\*.RLE" + RC + "Icône (\*.ICO)" + TAB + "\*.ICO" + RC + "Autres" + TAB + "\*.GIF;\*.TIF;\*.JPG"

**Remarque** : Si ce paramètre correspond à une chaîne vide (""), tous les types de fichiers seront proposés.

<a name="NOTE0_1"></a>

## Remarques


### Support de .TypeFichiers selon le navigateur
<a name="support_typefichiers_selon_navigateur_ELTPARAGRAPHE000069"></a>Certains navigateur conservent leur filtre par défaut. La liste des extensions définie avec la propriété **TypeFichiers** vient s'ajouter à la liste des extensions définie par le navigateur. Ce fonctionnement est notamment observé avec le navigateur Edge.

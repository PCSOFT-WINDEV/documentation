
## RechercheFAA (Propriété)

***En anglais : SearchAAF***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **RechercheFAA** permet de : 

- savoir si la recherche via la FAA est active sur un champ Combo sans saisie, 

- activer ou non la recherche via la FAA sur un champ Combo sans saisie. 




Remarque : Cette propriété est équivalente à l'option "Avec recherche" disponible dans l'onglet "Détail" de la fenêtre de description du champ Combo. 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=RechercheFAA%20-%20HC%20N%B0001.gif)
Un champ de recherche est alors affiché lorsque la combo est déroulée. ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=RechercheFAA%20-%20HC%20N%B0002.gif)

<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Clic sur le bouton [Activer la recherche]
COMBO_Client.RechercheFAA = Vrai
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Savoir si la recherche est active sur un champ Combo sans saisie

`<Résultat> = <Champ manipulé>.RechercheFAA`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la recherche est active, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 




**`<Champ manipulé> : (Nom de champ)`**

Nom du champ Combo sans saisie à manipuler.


<a name="SYNTAXE2"></a>

### Activer ou non la recherche sur un champ Combo sans saisie

`<Champ manipulé>.RechercheFAA = <Activation/Désactivation>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ Combo sans saisie à manipuler.

**`<Activation/Désactivation> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> pour activer la recherche, 

- <u><u><u><u>Faux</u></u></u></u> pour désactiver la recherche.




## SaisieDirecteRDV (Propriété)

***En anglais : DirectInputAPT***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **SaisieDirecteRDV** permet de savoir et d'indiquer si l'utilisateur peut modifier directement le titre d'un rendez-vous : 

- dans un champ Planning, 

- dans un champ Agenda.




Cette propriété correspond à l'option "Autoriser la saisie directe d'un rendez-vous" présente dans l'onglet "Détail" de la fenêtre de description des champs Planning et Agenda. 




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Gestion des options d'un champ Planning
SI gbModeModif = Vrai ALORS
	PLN_MonPlanning.DéplacementRDV = Vrai
	PLN_MonPlanning.SaisieDirecteRDV = Vrai
	PLN_MonPlanning.SélectionPériode = Vrai
SINON
	PLN_MonPlanning.DéplacementRDV = Faux
	PLN_MonPlanning.SaisieDirecteRDV = Faux
	PLN_MonPlanning.SélectionPériode = Faux
FIN
```


<a name="1000019993_Exemple2"></a>



```wl
// Gestion des options d'un champ Agenda
SI gbModeModif = Vrai ALORS
	AGD_MonAgenda.DéplacementRDV = Vrai
	AGD_MonAgenda.SélectionPériode = Vrai
	AGD_MonAgenda.SaisieDirecteRDV = Vrai
	SI PAS EnModeAndroid() _ET_ PAS EnModeIOS() ALORS
		AGD_MonAgenda.ModificationDuréeRDV = Vrai
	FIN
SINON
	AGD_MonAgenda.DéplacementRDV = Faux
	AGD_MonAgenda.SélectionPériode = Faux
	AGD_MonAgenda.SaisieDirecteRDV = Faux
	SI PAS EnModeAndroid() _ET_ PAS EnModeIOS() ALORS
		AGD_MonAgenda.ModificationDuréeRDV = Faux
	FIN
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Savoir si l'utilisateur peut saisir directement un rendez-vous

`<Résultat> = <Champ manipulé>.SaisieDirecteRDV`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'utilisateur peut saisir directement un rendez-vous, 

- <u><u><u><u>Faux</u></u></u></u> si l'utilisateur ne peut pas saisir directement un rendez-vous.




**`<Champ manipulé> : (Nom de champ)`**

Nom du champ manipulé. Ce champ peut correspondre à : 

- un champ Agenda.

- un champ Planning.  





<a name="SYNTAXE2"></a>

### Rendre ou non les rendez-vous directement saisissables par l'utilisateur

`<Champ manipulé>.SaisieDirecteRDV = <Déplaçable>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ manipulé. Ce champ peut correspondre à : 

- un champ Agenda.

- un champ Planning.




**`<Déplaçable> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> pour que l'utilisateur puisse directement saisir les rendez-vous, 

- <u><u><u><u>Faux</u></u></u></u> pour l'utilisateur ne puisse pas saisir directement les rendez-vous.

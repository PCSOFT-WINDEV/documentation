


## HauteurImage (Propriété)

***En anglais : ImageHeight***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **HauteurImage** permet de :

- Connaître la hauteur de la portion de l'image originale qui est affichée dans un champ Image ou dans le fond d'un champ Graphe.

- Modifier la hauteur de la portion de l'image originale qui est affichée dans un champ Image ou dans le fond d'un champ Graphe.




**Remarque** : Le mode d'affichage de l'image sélectionné dans la description du champ Image sera appliqué à la portion d'image.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
IMG_ChampImage.XImage = 20
IMG_ChampImage.YImage = 20
IMG_ChampImage.HauteurImage = 40
IMG_ChampImage.LargeurImage = 40
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître la hauteur de la portion affichée

`<Hauteur portion> = <Champ manipulé>.HauteurImage`
---

**`<Hauteur portion> : (Entier)`**



- Hauteur de la portion d'image à afficher dans le champ Image ou dans le fond du champ Graphe (exprimée en pixels). 

- -1 si toute l'image est affichée dans le champ Image ou dans le fond du champ Graphe.




**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler : 

- Champ Image, 

- Champ Graphe. 





<a name="SYNTAXE2"></a>

### Modifier la hauteur de la portion affichée

`<Champ manipulé>.HauteurImage = <Nouvelle hauteur>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler : 

- Champ Image, 

- Champ Graphe.




**`<Nouvelle hauteur> : (Entier)`**



- Hauteur de la portion d'image à afficher dans le champ Image ou dans le fond du champ Graphe (exprimée en pixels). 

- -1 pour afficher toute l'image dans le champ Image ou dans le fond du champ Graphe.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Définition de la portion d'image
<a name="definition_portion_image_ELTPARAGRAPHE000069"></a>Un champ Image permet d'afficher une portion de l'image contenue dans le fichier image associé. Pour définir la portion d'image affichée dans le champ Image, utilisez les propriétés suivantes :

- [XImage](../Proprietes/2510139.md) : permet de définir l'origine de la portion d'image.

- [YImage](../Proprietes/2510046.md) : permet de définir l'origine de la portion d'image.

- **HauteurImage** : permet de définir la hauteur de la portion d'image.

- [LargeurImage](../Proprietes/2510066.md) : permet de définir la largeur de la portion d'image.

### Limites
<a name="limites_ELTPARAGRAPHE000097"></a>Cette propriété s'applique uniquement aux champs Image et aux champs Graphe présents dans les fenêtres.
![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) La propriété **HauteurImage** s'applique uniquement aux champs Image.

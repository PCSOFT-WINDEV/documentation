


## JourDuMois (Propriété)

***En anglais : DayOfMonth***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **JourDuMois** permet de connaître et de modifier le numéro du jour du mois pour lequel la tâche planifiée doit être exécutée. 

Cette propriété est utilisable sur :

- une variable de type [hTâchePlanifiée](../WDLang4/1000017448.md).

- une variable de type [hPlanification](../WDLang4/1000018853.md).

- une variable de type [Répétition](../WDLang1/1000019262.md).

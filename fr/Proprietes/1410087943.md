
## Epinglé (Propriété)

***En anglais : Pinned***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Epinglé** permet de : 

- Epingler ou non un champ : le champ épinglé sera positionné par rapport à la zone visible de son conteneur au lieu de la zone du contenu.
	Dans une fenêtre mobile, cette propriété permet d'obtenir un champ fixe par rapport à l'écran même lorsque la fenêtre scrolle.

- Savoir si un champ est épinglé. 




Remarque : Cette propriété correspond à l'option "Champ épinglé" disponible dans l'onglet "UI" de la fenêtre de description du champ (bouton de gestion du parallaxe et des champs épinglés). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
BTN_NouveauMessage.Epinglé = Vrai 
BTN_NouveauMessage.MasquageAuScroll = Vrai
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Savoir si un champ est épinglé

`<Résultat> = <Champ manipulé>.Epinglé`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le champ est épinglé (le champ est fixe par rapport à son conteneur), 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 




**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler.


<a name="SYNTAXE2"></a>

### Epingler un champ

`<Champ manipulé>.Epinglé = <Epinglé ?>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler.

**`<Epinglé ?> : ()`**



- <u><u><u><u>Vrai</u></u></u></u> si le champ doit être épinglé (le champ sera fixe par rapport à son conteneur), 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




## Etendu (Propriété)

***En anglais : Extended***
	

<DIV class="specObsolete">
	<IMG style="float:left;margin-right:10px;" src="https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=warning_big.png"><B>Avertissement</B><BR>
	A partir de la version <B>27</B>,  cette propriété est conservée par compatibilité. Il est conseillé d'utiliser la nouvelle propriété [Interlettre](../Proprietes/1410088367.md) permettant de gérer l'interlettrage. 
</DIV><a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Etendu** permet de :

- Savoir si une variable de type Police est étendue.

- Étendre ou non une variable de type Police.




**Remarque** : L'étirement des caractères d'une police peut être :

- Normal : la largeur des caractères est normale.

- Condensé : la largeur des caractères est diminuée (propriété [Condensé](../Proprietes/2513006.md)).

- Étendu : la largeur des caractères est augmentée.

- Large : la largeur des caractères est augmentée au maximum (propriété [Large](../Proprietes/2513012.md)).





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Définir les caractéristiques d'une variable de type police
Police1 est une Police
Police1.Nom = "Arial"
Police1.Taille = 14
Police1.Etendu = Vrai
// Appliquer la police au champ "MonChamp"
MonChamp.Police = Police1
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Savoir si une variable de type Police est étendue

`<Résultat> = <Police manipulée>.Etendu`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la police spécifiée est étendue, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Police manipulée> : (Variable de type Police)`**

Nom de la variable de type [Police](../Motscles/1514045.md) à manipuler.  


<a name="SYNTAXE2"></a>

### Étendre ou non une variable de type Police

`<Police manipulée>.Etendu = <Étendu/Non étendu>`
---

**`<Police manipulée> : (Variable de type Police)`**

Nom de la variable de type [Police](../Motscles/1514045.md) à manipuler.

**`<Étendu/Non étendu> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la police spécifiée doit être étendue, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Variable de type Police
<a name="variable_type_police_ELTPARAGRAPHE000090"></a>Il est possible de créer une police à partir d'une variable de type [Police](../Motscles/1514045.md).

Pour définir les caractéristiques de la police, utilisez :

- soit la fonction [PoliceCrée](../WDLang1/3060002.md),

- soit les propriétés [Alphabet](../Proprietes/2512008.md), [Angle](../Proprietes/1410086632.md), [Barré](../Proprietes/2513005.md), [Condensé](../Proprietes/2513006.md), [Couleur](../Proprietes/2510071.md), [Gras](../Proprietes/2513001.md), [Italique](../Proprietes/2513007.md), [Large](../Proprietes/2513012.md), [Nom](../Proprietes/2510082.md), [Souligné](../Proprietes/2513013.md) et [Taille](../Proprietes/2510125.md).




Cette police pourra être utilisée :

- Pour écrire du texte dans des dessins (fonction [dPolice](../WDLang1/3029020.md)).

- Pour écrire dans les graphes (fonctions [grPoliceEtiquette](../WDLang3/3042012.md), [grPoliceLégende](../WDLang3/3042037.md), [grPoliceTitre](../WDLang3/3042048.md) et [grPoliceTitreAxe](../WDLang3/3042059.md)).

- Pour imprimer (fonction [iPolice](../WDLang5/3046014.md)).

- Dans les champs d'une fenêtre (propriété [Police](../Motscles/1514045.md) des champs d'une fenêtre).

- Dans les champs d'un état (propriété [Police](../Motscles/1514045.md) des champs d'un état).



<a name="NOTE0_2"></a>


### Limites
<a name="limites_ELTPARAGRAPHE000173"></a>Les propriétés permettant de définir l'étirement des caractères (propriétés [Condensé](../Proprietes/2513006.md), **Etendu** et [Large](../Proprietes/2513012.md)) sont gérées uniquement si la variable de type [Police](../Motscles/1514045.md) spécifiée est utilisée pour l'impression (police d'impression ou police des champs d'un état).

Dans les autres cas (police utilisée pour les dessins, les graphes ou les champs d'une fenêtre), ces caractéristiques ne sont pas prises en compte.

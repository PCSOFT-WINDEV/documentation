
## NoeudRacine (Propriété)

***En anglais : RootNode***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **NoeudRacine** permet de connaître les noeuds racine d'un document HTML ou XML. Cette propriété est utilisable sur :

- une variable de type [xmlDocument](../WDLang5/1000018782.md). 

- une variable de type [htmlDocument](../WDLang5/1000025910.md).

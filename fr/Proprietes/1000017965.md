
## Identifiant (Propriété)

***En anglais : Identifier***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Identifiant** permet de connaître et de modifier l'identifiant d'un élément. Cette propriété est utilisable sur :

- les flux. Pour plus de détails, consultez [Propriété Identifiant sur les flux](../Proprietes/2510036.md).

- une variable de type [gglAgenda](../WDLang5/1000017418.md).

- une variable de type [gglContact](../WDLang5/1000017434.md).

- une variable de type [gglDocument](../WDLang5/1000018001.md).

- une variable de type [gglEvénement](../WDLang5/1000017440.md).

- une variable de type [gglGroupeContact](../WDLang5/1000017576.md).

- une variable de type [xlsDocument](../WDLang5/1000017464.md).

- une variable de type [emailAttache](../WDLang3/1000018752.md).

- une variable de type [hDescriptionSauvegarde](../WDLang4/1000017455.md). 

- une variable de type [hOptimisationPlanifiée](../WDLang4/1000018854.md). 

- une variable de type [hRSConfig](../WDLang4/1000020484.md).

- une variable de type [rssEntrée](../WDLang5/1000017792.md).

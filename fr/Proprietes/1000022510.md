
## CouleurSouligné (Propriété)

***En anglais : UnderlineColor***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **CouleurSouligné** permet de définir la couleur utilisée pour souligner un élément. Cette propriété est utilisable sur : 

- une variable de type [docMiseEnForme](../WDLang1/1000022817.md). 

- une variable de type [docElémentTexte](../WDLang1/1000022493.md) (caractéristiques de mise en forme). 

- une variable de type [docFragment](../WDLang1/1000022482.md) (caractéristiques de mise en forme).

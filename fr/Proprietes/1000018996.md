


## BulleAutomatique (Propriété)

***En anglais : AutomaticTooltip***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **BulleAutomatique** permet d'autoriser ou non l'affichage des bulles automatiques sur : 

- les champs Liste. 

- les champs Table.

- les champs Table hiérarchique. 

- les champs Arbre.


**Rappel** : Les bulles automatiques sont des bulles d'aide apparaissant automatiquement dès que le contenu d'une cellule ou d'une ligne est tronqué. L'utilisateur peut ainsi voir le contenu complet de l'élément sans effectuer de manipulations spécifiques.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Désactive la bulle automatique
TABLE_MaTable.BulleAutomatique = Faux
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Savoir si les bulles automatiques sont activées ou non

`<Résultat> = <Champ manipulé>.BulleAutomatique`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si les bulles automatiques sont activées, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 


Remarque : Par défaut, les bulles automatiques sont activées.

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ manipulé : 

- Champ Table,

- Champ Table hiérarchique,

- Champ Liste,

- Champ Arbre.





<a name="SYNTAXE2"></a>

### Activer ou désactiver les bulles automatiques

`<Champ manipulé>.BulleAutomatique = <Nouvelle valeur>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ manipulé : 

- Champ Table,

- Champ Table hiérarchique,

- Champ Liste,

- Champ Arbre.




**`<Nouvelle valeur> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> pour activer les bulles automatiques, 

- <u><u><u><u>Faux</u></u></u></u> pour les désactiver.










<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Si les bulles d'aide ont été personnalisées pour le projet, le texte des bulles automatiques sera affiché dans les bulles d'aide personnalisées. Pour plus de détails, consultez [Bulles d'aide personnalisées](../Editeurs/9500233.md).

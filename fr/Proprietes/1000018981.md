


## JaugeSystème (Propriété)

***En anglais : ProgressBarSystem***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **JaugeSystème** permet de connaître et de modifier la jauge utilisée comme jauge système (dans la barre des tâches) avec Windows 7 (et supérieur). 

**Rappel** : A partir de Windows 7, il est possible d'afficher une jauge dans la barre des tâches pour une fenêtre donnée. Par défaut, cette jauge correspond :

- soit à la jauge de la barre d'état si cette jauge est présente. 

- soit la première jauge visible de la fenêtre si la fenêtre possède une jauge.


Ce comportement automatique peut être modifié avec la propriété **JaugeSystème**. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Champ jauge lié à la jauge 7
MaFenêtre.JaugeSystème = JAUGE_Jauge1..Nom
JAUGE_Jauge1.BorneMin = 0
JAUGE_Jauge1.BorneMax = 100
POUR i = 1 A 100
	JAUGE_Jauge1.Valeur = i
	Multitâche(10)
FIN
// Jauge de la barre des tâches liée à la jauge 7
MaFenêtre.JaugeSystème = JaugeBarre
POUR i = 1 A 100
	Jauge(i, 100)
	Multitâche(10)
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître la jauge utilisée comme jauge système par Windows 7

`<Résultat> = <Fenêtre manipulée>.JaugeSystème`
---

**`<Résultat> : (Chaîne de caractères)`**

Nom de la jauge utilisée dans la barre des tâches de la fenêtre en Windows 7. Ce nom peut correspondre : 

- au nom d'un champ Jauge. 

- à une des constantes suivantes : 
	


|   |   |
| --- | --- |
| *JaugeAucune* | L'affichage de la jauge est désactivé. |
| *JaugeBarre* | La jauge de la barre des tâches est utilisée dans tous les cas. |
| *JaugeDéfaut* | Restaure le comportement par défaut. |






**`<Fenêtre manipulée> : (Nom de fenêtre)`**

Nom de la fenêtre sur laquelle la jauge système sera rendue visible. 


<a name="SYNTAXE2"></a>

### Modifier la jauge utilisée comme jauge système par Windows 7

`<Fenêtre manipulée>.JaugeSystème = <Nouvelle valeur>`
---

**`<Fenêtre manipulée> : (Nom de fenêtre)`**

Nom de la fenêtre sur laquelle la jauge système doit être gérée. 

**`<Nouvelle valeur> : (Chaîne de caractères)`**

Nouveau nom de la jauge utilisée dans la barre des tâches de la fenêtre en Windows 7. Ce nom peut correspondre : 

- au nom d'un champ Jauge. 

- à une des constantes suivantes : 
	


|   |   |
| --- | --- |
| *JaugeAucune* | L'affichage de la jauge est désactivé. |
| *JaugeBarre* | La jauge de la barre des tâches est utilisée dans tous les cas. |
| *JaugeDéfaut* | Restaure le comportement par défaut. |








<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- La modification de la propriété **JaugeSystème** réinitialise la jauge de la barre des tâches. Toute valeur de la jauge avant l'appel à la propriété **JaugeSystème** sera perdue.

- Cette propriété est disponible uniquement sous Windows 7 ou supérieur.

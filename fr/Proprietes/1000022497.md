


## ZoomAuDoigt (Propriété)

***En anglais : ZoomWithFinger***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **ZoomAuDoigt** permet de : 

- savoir si un champ Image ou Caméra autorise le zoom au doigt. 

- autoriser ou non le zoom au doigt dans un champ Image ou Caméra. 





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// En mode dessin, pas de zoom au doigt
SI bDessin = Vrai ALORS
	IMG_Image1.ZoomAuDoigt = Faux
SINON
	IMG_Image1.ZoomAuDoigt = Vrai
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Savoir si un champ Image ou le champ Caméra autorise le zoom au doigt

`<Résultat> = <Champ manipulé>.ZoomAuDoigt`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le zoom au doigt est actif, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 




**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler : 

- Champ Image, 





<a name="SYNTAXE2"></a>

### Autoriser ou non le zoom au doigt dans un champ Image ou le champ Caméra

`<Champ manipulé>.ZoomAuDoigt = <Autorisation>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler : 

- Champ Image, 




**`<Autorisation> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> pour activer le zoom au doigt, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Utilisation dans un champ Image
<a name="utilisation_dans_champ_image_ELTPARAGRAPHE000090"></a>

- Cette propriété permet notamment d'interrompre temporairement le zoom au doigt pour pouvoir dessiner dans un champ Image. De la même façon, il est possible d'interrompre le scroll au doigt avec la propriété [ScrollAuDoigt](../Proprietes/1000019284.md). 

- L'appel à la fonction [dDébutDessin](../WDLang1/3029009.md) désactive automatiquement les fonctionnalités de zoom et scroll au doigt.

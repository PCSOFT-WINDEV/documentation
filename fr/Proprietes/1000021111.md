
## BitParPixel (Propriété)

***En anglais : BitPerPixel***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **BitParPixel** permet de connaître et de modifier le nombre de bits par pixels d'une image. Cette propriété est utilisable sur :

- une variable de type [Image](../WDLang1/1000019650.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [upnpImage](../WDLang3/1000020905.md).

- ![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png) une variable de type [Icône](../WDLang1/1000020811.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [Scanneur](../WDLang6/1410087808.md).


## Réplication (Propriété)

***En anglais : Replication***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Réplication** permet de connaître et de modifier le mode de réplication. Cette propriété est utilisable sur :

- un fichier de données. Pour plus de détails, consultez [Réplication](../Proprietes/2512073.md).

- une variable de type [hOpérationServeur](../WDLang4/1000022471.md).

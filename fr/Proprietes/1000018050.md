
## Multiligne (Propriété)

***En anglais : Multiline***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Multiligne** permet de savoir si un élément est multiligne. Cette propriété est utilisable sur :

- les champs d'une fenêtre, d'une page ou d'un état. Pour plus de détails, consultez [Multiligne sur les champs](../Proprietes/2510105.md).

- une variable de type [xlsCellule](../WDLang5/1000017472.md).

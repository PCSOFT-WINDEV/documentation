
## Point

***En anglais : Point***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le mot-clé **Point** permet de manipuler un point. Ce mot-clé peut correspondre à : 

- une propriété de la variable [gglChemin](../WDLang5/1000017433.md). 

- une variable de type [Point](../Motscles/1000018908.md). 

- une propriété d'une variable de type [Polygone](../WDLang1/1000025624.md).

- une propriété d'une variable de type [CartePolygone](../WDLang3/1000025476.md).

- une propriété d'une variable de type [CartePolyligne](../WDLang3/1000025489.md).

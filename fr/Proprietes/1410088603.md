
## XPourcentage (Propriété)

***En anglais : XPercent***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **XPourcentage** permet de connaître et de modifier l'abscisse d'un point exprimée en pourcentage. Cette propriété est utilisable sur : 

- une variable de type [diagPoint](../WDLang1/1410088140.md). 

- une variable de type [diagPointDAttache](../WDLang1/1410088457.md).




## CxBaseDeDonnéesDécrite (Propriété)

***En anglais : CtDescribedDatabase***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **CxBaseDeDonnéesDécrite** permet de gérer la source de données OLE DB lors de la connexion OLE DB à une table spécifique. Il est possible de :

- Connaître la source de données OLE DB utilisée pour la connexion à une table par OLE DB. Cette source de données a été décrite sous l'éditeur d'analyses ou lors de la description par programmation du fichier de données.
	Attention : la propriété **CxBaseDeDonnéesDécrite** ne tient pas compte des éventuelles modifications réalisées avec la fonction [HConnecte](../WDLang4/3044263.md).
	Cette propriété peut être utilisée pour un fichier de données défini sous l'éditeur d'analyses ou défini par programmation.

- Définir la source de données OLE DB utilisée pour la connexion à une table par OLE DB (lors de la description par programmation du fichier de données uniquement). Cette propriété est équivalente à la propriété [CxBaseDeDonnées](../Proprietes/2512047.md). 




**Rappel** : La fonction [HConnecte](../WDLang4/3044263.md) permet de redéfinir un ou plusieurs paramètres de la connexion OLE DB sur une table. Par défaut, les caractéristiques de la connexion OLE DB sont définies :

- soit sous l'éditeur d'analyses lors de la description du fichier de données,

- soit lors de la description par programmation du fichier de données.



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connaître la source de données OLE DB
ResBaseOLEDB = Client.CxBaseDeDonnéesDécrite
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître la source de données OLE DB

`<Source de données OLE DB> = <Fichier de données>.CxBaseDeDonnéesDécrite`
---

**`<Source de données OLE DB> : (Chaîne de caractères)`**

Chemin complet de la source de données OLE DB, précisée lors de la description du fichier.

**`<Fichier de données> : (Chaîne de caractères)`**

Nom du fichier de données manipulé. Ce nom a été défini sous l'éditeur d'analyses ou grâce au type [Description de Fichier](../WDLang4/1514065.md).  


<a name="SYNTAXE2"></a>

### Définir la source de données OLE DB

`<Fichier de données>.CxBaseDeDonnéesDécrite = <Source de données OLE DB>`
---

**`<Fichier de données> : (Chaîne de caractères)`**

Nom du fichier de données manipulé. Ce nom vient d'être défini grâce au type [Description de Fichier](../WDLang4/1514065.md).

**`<Source de données OLE DB> : (Chaîne de caractères)`**

Chemin complet de la source de données OLE DB.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
La propriété [CxBaseDeDonnées](../Proprietes/2512047.md) permet de connaître et de modifier la source de données spécifiée pour la connexion OLE DB.


## AttributNote (Propriété)

***En anglais : AttributeNote***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **AttributNote** permet de connaître la valeur de l'attribut d'extension &lt;Note&gt;. Cette propriété est disponible sur : 

- les variables de type [Définition](../WDLang1/1000019540.md). 

- les variables de type [Description de variable](../WDLang1/1000019560.md).

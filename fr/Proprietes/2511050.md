


## NbEtiquette (Propriété)

***En anglais : NbLabel***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **NbEtiquette** permet de :

- Connaître le nombre d'étiquettes imprimées sur la largeur d'une page.

- Modifier le nombre d'étiquettes imprimées sur la largeur d'une page.




**Remarque** : Le nombre d'étiquettes imprimées sur la largeur d'une page peut être connu et/ou modifié sous l'éditeur d'états : sous le volet "Modification", dans le groupe "Etat", sélectionnez "Format et étiquettes".
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=nbetiquette.gif)



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Augmente le nombre d'étiquettes à imprimer dans la largeur de la page
ETAT_EtiqClient.NbEtiquette = ETAT_EtiqClient.NbEtiquette + 2
// Est équivalent à : ETAT_EtiqClient.NbEtiquette += 2
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le nombre d'étiquettes imprimées sur la largeur d'une page

`<Résultat> = <Etat manipulé>.NbEtiquette`
---

**`<Résultat> : (Entier)`**

Nombre d'étiquettes à imprimer sur la largeur de la page.

**`<Etat manipulé> : (Nom d'état)`**

Nom de l'état de type "Étiquette" à manipuler.  


<a name="SYNTAXE2"></a>

### Modifier le nombre d'étiquettes imprimées sur la largeur d'une page

`<Etat manipulé>.NbEtiquette = <Nouveau nombre>`
---

**`<Etat manipulé> : (Nom d'état)`**

Nom de l'état de type "Étiquette" à manipuler.

**`<Nouveau nombre> : (Entier)`**

Nouveau nombre d'étiquettes à imprimer sur la largeur de la page.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Modification du nombre d'étiquettes
<a name="modification_nombre_etiquettes_ELTPARAGRAPHE000073"></a>Lors de la modification du nombre d'étiquettes par programmation, les étiquettes ne sont pas automatiquement re-dimensionnées. Il est donc conseillé de vérifier la taille des étiquettes.
<a name="NOTE0_2"></a>


### Modification des dimensions des étiquettes
<a name="modification_des_dimensions_des_etiquettes_ELTPARAGRAPHE000080"></a>

- Pour connaître ou modifier la hauteur des étiquettes d'un état de type "Étiquette", utilisez la propriété [HauteurEtiquette](../Proprietes/2511069.md).

- Pour connaître ou modifier la largeur des étiquettes d'un état de type "Étiquette", utilisez la propriété [LargeurEtiquette](../Proprietes/2511055.md).

- Pour connaître ou modifier l'espacement horizontal entre les étiquettes d'un état de type "Étiquette", utilisez la propriété [EspaceH](../Proprietes/2511067.md).

- Pour connaître ou modifier l'espacement vertical entre les étiquettes d'un état de type "Étiquette", utilisez la propriété [EspaceV](../Proprietes/2511065.md).



<a name="NOTE0_3"></a>


### Limite
<a name="limite_ELTPARAGRAPHE000102"></a>La propriété **NbEtiquette** s'applique uniquement aux états de type "Étiquette".

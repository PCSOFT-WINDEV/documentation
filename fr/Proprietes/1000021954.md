


## TimestampType (Propriété)

***En anglais : TimestampType***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **TimestampType** permet de :

- Connaître le type d'horodatage d'une rubrique. Cette rubrique a été définie sous l'éditeur d'analyses ou définie par programmation.

- Définir le type d'horodatage d'une rubrique de type Date, Heure ou DateHeure (lors de la description de la rubrique par programmation uniquement).




Rappel : Une rubrique de type "Horodatage" ne peut pas être affectée par programmation. Elle est automatiquement affectée avec la date de création ou de dernière modification de l'enregistrement.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
UnFichier est une Description de Fichier
UneRubrique est une Description de Rubrique
UneRubrique.Nom = "DateCmd"
UneRubrique.TimestampType = htsCréation
UneRubrique.TimestampUTC = Faux
HDécritRubrique(UnFichier, UneRubrique)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître le type d'horodatage d'une rubrique

`<Résultat> = <Fichier de données>.<Rubrique>.TimestampType`
---

**`<Résultat> : (Constante de type Entier)`**

Type d'horodatage de la rubrique :


|   |   |
| --- | --- |
| *htsAucun* | Aucun horodatage. |
| *htsCréation* | Horodatage uniquement lors de la création de l'enregistrement. |
| *htsDernièreModification* | Horodatage à chaque modification de l'enregistrement. |



**`<Fichier de données> : (Chaîne de caractères)`**

Nom du fichier de données manipulé.

**`<Rubrique> : (Chaîne de caractères)`**

Nom de la rubrique manipulée.

### Définir le type d'horodatage d'une rubrique de type Date, Heure ou DateHeure

`<Rubrique>.TimestampType = <Type d'horodatage>`
---

**`<Rubrique> : (Chaîne de caractères)`**

Nom de la rubrique manipulée. Ce nom vient d'être défini grâce au type [Description de Rubrique](../WDLang4/1514071.md).

**`<Type d'horodatage> : (Constante de type Entier)`**

Type d'horodatage de la rubrique :


|   |   |
| --- | --- |
| *htsAucun* | La rubrique n'est pas une rubrique d'horodatage. |
| *htsCréation* | La rubrique est une rubrique d'horodatage renseignée uniquement lors de la création de l'enregistrement. |
| *htsDernièreModification* | La rubrique est une rubrique d'horodatage renseignée à chaque modification de l'enregistrement. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Sous l'éditeur d'analyses, il est possible de définir une rubrique de type Horodatage. 

- Pour définir une rubrique Horodatage par programmation : 

	- la rubrique doit être de type DateHeure, Date ou Heure, 

	- le type d'horodatage est spécifié grâce à la propriété **TimestampType**.

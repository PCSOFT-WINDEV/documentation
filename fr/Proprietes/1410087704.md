
## Caméra (Propriété)

***En anglais : Caméra***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Caméra** permet de connaître et de modifier la caméra utilisée (caméra dorsale ou frontale). 

Remarques : 

- Si une enregistrement vidéo est en cours, il n'est pas possible de modifier la caméra. 

- Cette propriété est utilisable uniquement pour les champs Caméra qui ne sont pas en "Mode compatible 26". 

- Cette propriété correspond à l'option "Caméra" de l'onglet "Général" de la fenêtre de description du champ Caméra. 





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI CAM_MaCaméra.Caméra = camCaméraDorsale ALORS
	CAM_MaCaméra.Caméra = camCaméraFrontale
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître la caméra utilisée par un champ Caméra

`<Résultat> = <Champ Caméra>.Caméra`
---

**`<Résultat> : (Constante de type Entier)`**

Caméra utilisée : 


|   |   |
| --- | --- |
| *camCaméraDorsale* | Caméra dorsale du périphérique.   |
| *camCaméraFrontale* | Caméra frontale du périphérique. |


 Par défaut, la caméra dorsale est utilisée. 

**`<Champ Caméra> : (Nom de champ)`**

Nom du champ Caméra à manipuler.


<a name="SYNTAXE2"></a>

### Modifier la caméra utilisée par un champ Caméra

`<Champ Caméra>.Flash = <Nouvelle caméra>`
---

**`<Champ Caméra> : (Nom de champ)`**

Nom du champ Caméra à manipuler.

**`<Nouvelle caméra> : (Constante de type Entier)`**

Caméra utilisée : 


|   |   |
| --- | --- |
| *camCaméraDorsale* | Caméra dorsale du périphérique.   |
| *camCaméraFrontale* | Caméra frontale du périphérique. |


 Par défaut, la caméra dorsale est utilisée.

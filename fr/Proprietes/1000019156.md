
## DéchargeDLL

***En anglais : FreeDLL***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le mot-clé **DéchargeDLL** peut correspondre : 

- à la fonction **DéchargeDLL** qui permet de décharger une librairie 32 bits (DLL) qui a été précédemment chargée en mémoire. Pour plus de détails, consultez [DéchargeDLL](../WDLang1/3014009.md). 

- à la propriété **DéchargeDLL** disponible sur une variable de type [Description d'API](../WDLang1/1000019149.md).

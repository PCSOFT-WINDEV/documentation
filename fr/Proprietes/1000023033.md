
## Sélection (Propriété)

***En anglais : Selection***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Sélection** permet d'obtenir les caractéristiques de la sélection (ou du curseur) : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) dans un champ Traitement de texte.
	Remarque : Cette sélection est située dans la partie en cours d'édition dans le champ (corps, entête ou pied de page). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) dans un champ Tableur.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) dans un champ Editeur HTML.

- dans un champ Editeur de diagrammes.





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 
```wl
// Récupération du fragment sélectionné par l'utilisateur
MaSélection est un docFragment = TT_Doc.Sélection.Fragment
// Mise en gras de la sélection de l'utilisateur
MaSélection.MiseEnForme.PoliceGras = Vrai
```

```wl
// Récupération de la première section du texte sélectionné par l'utilisateur
MaSection1 est un docSection = TT_Doc.Sélection.Section[1]
```



```wl
// Obtenir la sélection dans un champ Editeur de diagrammes
MaSélection est un diagSélection <- EDIAG_MonDiagramme.Sélection
SI MaSélection.Forme.Occurrence > 0 ALORS
	// Obtenir les formes
	POUR TOUT stForme DE MaSélection.Forme	
		LIB_INFO_SELECTION = "Sélection de " + stForme.Nom + 
		" [ " + stForme.X + ", " + stForme.Y + " - " + stForme.Largeur + 
		"x" + stForme.Hauteur + " ]"	
	FIN
SINON
	LIB_INFO_SELECTION = "Cliquez sur une forme dans le diagramme pour connaître la sélection"
FIN
```

## Syntaxe
<a name="SYNTAXE1"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 
### Récupérer les propriétés de la sélection dans un champ Traitement de texte

`<Résultat> = <Champ Traitement de texte>.Sélection.<Propriété>`
---

**`<Résultat> : (Type correspondant à la propriété utilisée)`**

Résultat attendu (voir "Remarques").  

**`<Champ Traitement de texte> : (Nom de champ)`**

Nom du champ Traitement de texte.

**`<Propriété> : ()`**

Propriété à utiliser sur la sélection (voir Remarques). 




<a name="SYNTAXE2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 
### Récupérer les propriétés de la sélection dans un champ Tableur

`<Résultat> = <Champ Tableur>.Sélection`
---

**`<Résultat> : (Variable de type tblrSélection)`**

Variable de type [tblrSélection](../WDLang1/1000023498.md) décrivant la sélection.   

**`<Champ Tableur> : (Nom de champ)`**

Nom du champ Tableur. 




<a name="SYNTAXE3"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 
### Récupérer les propriétés de la sélection dans un champ Editeur HTML

`<Résultat> = <Champ Editeur HTML>.Sélection`
---

**`<Résultat> : (Variable de type edtHTMLSélection)`**

Variable de type [edtHTMLSélection](../WDLang1/1000026093.md) décrivant la sélection.   

**`<Champ Editeur HTML> : (Nom de champ)`**

Nom du champ Editeur HTML.




<a name="SYNTAXE4"></a>

### Récupérer les propriétés de la sélection dans un champ Editeur de diagrammes

`<Résultat> = <Champ Editeur de diagrammes>.Sélection`
---

**`<Résultat> : (Variable de type diagSélection)`**

Variable de type [diagSélection](../WDLang1/1410088172.md) décrivant la sélection.   

**`<Champ Editeur de diagrammes> : (Nom de champ)`**

Nom du champ Editeur de diagrammes.

## Remarques
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 

### Sélection dans un champ Traitement de texte
<a name="selection_dans_champ_traitement_texte_ELTPARAGRAPHE000107"></a>

| Propriété | Explications |
| --- | --- |
| Fragment | &lt;Résultat&gt; correspond à une variable de type [docFragment](../WDLang1/1000022482.md) contenant les caractéristiques du fragment sélectionné. <br><br>Exemple : &lt;Résultat&gt; = &lt;Champ Traitement de texte&gt;.Sélection.Fragment |
| Section[X] | Permet de récupérer la section associée à la sélection. <br><br>- X correspond au numéro de la section à récupérer.<br><br>- &lt;Résultat&gt; correspond à une variable de type [docSection](../WDLang1/1000022685.md) contenant les caractéristiques de la section associée à la sélection. <br><br><br><br><br>Exemple : &lt;Résultat&gt; = &lt;Champ Traitement de texte&gt;.Sélection.Section[1] |

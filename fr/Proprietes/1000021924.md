


## BoutonSuppression (Propriété)

***En anglais : DeleteButton***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **BoutonSuppression** permet de connaître et modifier la présence et la position d'un bouton de suppression automatique dans un champ de saisie de type Texte. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Active le bouton de suppression sur SAI_Saisie1
// Le bouton est toujours visible
SAI_Saisie1.BoutonSuppression = bsToujoursVisible
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître la position du bouton de suppression du champ de saisie

`<Résultat> = <Champ de saisie>.BoutonSuppression`
---

**`<Résultat> : (Constante de type Entier)`**

Position du bouton de suppression : 



|   |   |
| --- | --- |
| *bsAucun* | Le bouton de suppression n'est pas affiché. |
| *bsEnSaisie* | Le bouton de suppression est affiché uniquement lorsque le champ de saisie est en mode édition. |
| *bsHorsSaisie* | Le bouton de suppression est affiché uniquement lorsque le champ de saisie n'est pas en mode édition. |
| *bsToujoursVisible* | Le bouton de suppression est toujours affiché. |



**`<Champ de saisie> : (Nom de champ)`**

Champ de saisie de type Texte à manipuler. 


<a name="SYNTAXE2"></a>

### Modifier la position du bouton de suppression du champ de saisie

`<Champ de saisie>.BoutonSuppression = <Bouton de suppression>`
---

**`<Champ de saisie> : (Nom de champ)`**

Champ de saisie de type Texte à manipuler. 

**`<Bouton de suppression> : (Constante de type Entier)`**

Position du bouton de suppression : 



|   |   |
| --- | --- |
| *bsAucun* | Le bouton de suppression n'est pas affiché. |
| *bsEnSaisie* | Le bouton de suppression est affiché uniquement lorsque le champ de saisie est en mode édition. |
| *bsHorsSaisie* | Le bouton de suppression est affiché uniquement lorsque le champ de saisie n'est pas en mode édition. |
| *bsToujoursVisible* | Le bouton de suppression est toujours affiché. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Le bouton de suppression est disponible uniquement sur les champs de saisie de type texte. Il n'est pas disponible pour les champs de saisie multilignes, numérique, date, heure, ...

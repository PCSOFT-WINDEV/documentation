
## PartieDécimale (Propriété)

***En anglais : DecimalPart***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **PartieDécimale** permet de paramétrer les rubriques de type Numérique. Il est possible de :

- Connaître le nombre de chiffres de la partie décimale d'une rubrique de type Numérique.

- Définir le nombre de chiffres de la partie décimale d'une rubrique de type Numérique.




**Remarque** : La propriété [PartieEntière](../Proprietes/2512118.md) permet de définir et de connaître le nombre de chiffres de la partie entière d'une rubrique de type Numérique.






<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Description du fichier de données UnFichier
...

// Description de la rubrique "Montant"
UneRubrique.Nom = "MONTANT"
UneRubrique.Type = hRubNumérique
UneRubrique.PartieEntière = 15
UneRubrique.PartieDécimale = 7

// Validation de la description de la rubrique "Montant"
HDécritRubrique(UnFichier, UneRubrique)

// Validation de la description du fichier de données UnFichier
...
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le nombre de chiffres de la partie décimale d'une rubrique de type Numérique

`<Résultat> = <Fichier de données>.<Rubrique>.PartieDécimale`
---

**`<Résultat> : (Entier)`**

Nombre de chiffres pris en compte par la partie décimale de la rubrique.

**`<Fichier de données> : (Chaîne de caractères)`**

Nom du fichier de données manipulé. Ce nom a été défini sous l'éditeur d'analyses ou grâce au type [Description de Fichier](../WDLang4/1514065.md).

**`<Rubrique> : (Chaîne de caractères)`**

Nom de la rubrique manipulée. Ce nom a été défini sous l'éditeur d'analyses ou grâce au type [Description de Rubrique](../WDLang4/1514071.md).  


<a name="SYNTAXE2"></a>

### Définir le nombre de chiffres de la partie décimale d'une rubrique de type Numérique

`<Rubrique>.PartieDécimale = <Nombre de chiffres>`
---

**`<Rubrique> : (Chaîne de caractères)`**

Nom de la rubrique manipulée. Ce nom vient d'être défini grâce au type [Description de Rubrique](../WDLang4/1514071.md).

**`<Nombre de chiffres> : (Entier)`**

Nombre de chiffres à prendre en compte pour la partie décimale de la rubrique de type Numérique.

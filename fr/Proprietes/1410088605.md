
## StyleCoche (Propriété)

***En anglais : CheckMarkStyle***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **StyleCoche** permet de connaître et de modifier le style de la coche d'une option de menu (menu principal ou menu contextuel). Cette coche apparaît uniquement si l'option de menu est cochée. 

Cette propriété permet d'obtenir des groupes d'options qui simulent un sélecteur dans un menu, comme sous Windows 11. Cette propriété est utilisable que le menu soit au look Windows 11 ou non. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
OPTION_Haut.StyleCoche = scRond
OPTION_Milieu.StyleCoche = scRond 
OPTION_Bas.StyleCoche = scRond 
OPTION_Milieu.Cochée = Vrai
```

<a name="XSYNTAXE"></a>

## Syntaxe

### Connaître le style de la coche d'une option de menu

`<Résultat> = <Elément à manipuler>.StyleCoche`
---

**`<Résultat> : (Constante de type Entier)`**

Style de la coche : 



|   |   |
| --- | --- |
| *scDéfaut* | Coche par défaut. |
| *scRond* | Coche ronde. |



**`<Elément à manipuler> : (Nom de champ)`**

Nom de l'option de menu.


<a name="SYNTAXE2"></a>

### Modifier le style de la coche d'une option de menu

`<Elément à manipuler>.StyleCoche = <nouvelle valeur>`
---

**`<Elément à manipuler> : (Nom de champ)`**

Nom de l'option de menu.

**`<nouvelle valeur> : ()`**

Style à appliquer à la coche : 


|   |   |
| --- | --- |
| *scDéfaut* | Coche par défaut. |
| *scRond* | Coche ronde. |

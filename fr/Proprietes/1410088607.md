
## HauteurImageInitiale (Propriété)

***En anglais : ImageInitialHeight***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **HauteurImageInitiale** permet de connaître ou de modifier la hauteur par défaut de l'image créée dans le champ Editeur d'images.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
EI_MonEditeurImage.LargeurImageInitiale = 800
EI_MonEditeurImage.HauteurImageInitiale = 600
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître la hauteur par défaut de l'image créée dans le champ Editeur d'images

`<Résultat> = <Champ Editeur d'images>.HauteurImageInitiale`
---

**`<Résultat> : (Entier)`**

Hauteur par défaut (en pixels) d'une image créée dans le champ Editeur d'images. 

**`<Champ Editeur d'images> : (Nom de champ)`**

Nom du champ Editeur d'images manipulé.


<a name="SYNTAXE2"></a>

### Modifier la hauteur par défaut de l'image créée dans le champ Editeur d'images

`<Champ Editeur d'images>.HauteurImageInitiale = <Nouvelle hauteur>`
---

**`<Champ Editeur d'images> : (Nom de champ)`**

Nom du champ Editeur d'images manipulé.

**`<Nouvelle hauteur> : (Entier)`**

Nouvelle hauteur par défaut (en pixels) d'une image créée dans le champ Editeur d'images.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Pour modifier les caractéristiques par défaut du champ Editeur d'images, il est également possible d'utiliser : 

- la propriété [LargeurImageInitiale](../Proprietes/1410088608.md) pour définir la largeur par défaut de l'image créée dans le champ Editeur d'images.

- la propriété [OutilInitial](../Proprietes/1410088609.md) pour définir l'outil utilisé par défaut dans le champ Editeur d'images.

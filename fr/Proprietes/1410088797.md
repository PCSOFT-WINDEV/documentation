
## Supprimer

***En anglais : Remove***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Supprimer** permet d'autoriser ou non la suppression d'un élément. Cette propriété est disponible sur : 

- une variable de type diagActionAutorisée. 

- une variable de type kbActionAutorisée.

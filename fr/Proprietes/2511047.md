


## NbEnrTotal (Propriété)

***En anglais : TotalNbRec***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **NbEnrTotal** permet de :

- Connaître le nombre d'enregistrements à imprimer (cas d'un état sur un fichier, sur une zone mémoire ou sur une requête).

- Modifier le nombre d'enregistrements à imprimer (c'est-à-dire fixer une limite, par exemple imprimer uniquement les 50 premiers enregistrements).









<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Modifier le nombre maximum d'enregistrements à imprimer
ETAT_Client.NbEnrTotal = 25
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le nombre maximum d'enregistrements à imprimer

`<Résultat> = <Etat manipulé>.NbEnrTotal`
---

**`<Résultat> : (Entier)`**



- Nombre d'enregistrements à imprimer,

- 0 si tous les enregistrements doivent être imprimés.




**`<Etat manipulé> : (Nom d'état)`**

Nom de l'état à manipuler.  


<a name="SYNTAXE2"></a>

### Modifier le nombre maximum d'enregistrements à imprimer

`<Etat manipulé>.NbEnrTotal = <Nouveau nombre maximum>`
---

**`<Etat manipulé> : (Nom d'état)`**

Nom de l'état à manipuler.

**`<Nouveau nombre maximum> : (Entier)`**



- Nouveau nombre d'enregistrements à imprimer,

- 0 si tous les enregistrements doivent être imprimés.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Nombre d'enregistrements
<a name="nombre_enregistrements_ELTPARAGRAPHE000063"></a>

- La propriété [NbEnrLu](../Proprietes/2511041.md) renvoie le nombre d'enregistrements sélectionnés par la source de l'état (requête, vue, zone mémoire ou champ Table).

- La propriété [NbEnrImprimé](../Proprietes/2511058.md) renvoie le nombre d'enregistrements sélectionnés à la fois par la source de l'état ET par le code de filtre (s'il existe).

- Dans le cas d'un état sur fichier, les propriétés [NbEnrLu](../Proprietes/2511041.md) et [NbEnrImprimé](../Proprietes/2511058.md) retournent la même valeur : le nombre d'enregistrements du fichier ET sélectionnés par le code de filtre (s'il existe).

- La propriété **NbEnrTotal** permet de connaître et de modifier le nombre d'enregistrements à imprimer. Le nombre d'enregistrements à imprimer sera compris entre 1 et la valeur de la propriété [NbEnrImprimé](../Proprietes/2511058.md).



<a name="NOTE0_2"></a>


### Type d'état
<a name="type_etat_ELTPARAGRAPHE000091"></a>Si l'état est lié à :

- ***un fichier de données HFSQL***, les enregistrements à imprimer sont sélectionnés dans le début du fichier de données.

- ** *une requête***, le nombre maximum d'enregistrements à imprimer peut être connu et/ou modifié dans la description des options avancées de la requête. Pour afficher les options avancées de la requête liée à l'état : 

	- Sélectionnez l'option "Editer la requête" du menu contextuel de l'état. L'éditeur de requêtes s'affiche. 

	- Sélectionnez l'option "Description de la requête" du menu contextuel de la requête. 

	- Dans la fenêtre qui s'affiche, cliquez sur le bouton "Avancé".






<a name="NOTE0_3"></a>


### Limite
<a name="limite_ELTPARAGRAPHE000105"></a>La propriété **NbEnrTotal** ne peut pas être utilisée sur un état sans source de données.

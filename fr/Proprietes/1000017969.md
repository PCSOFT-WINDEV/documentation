
## Etat (Propriété)

***En anglais : State***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Etat** permet de connaître et de modifier l'état d'un élément. Cette propriété est utilisable sur :

- **les champs, les fenêtres, les pages, les états**. Pour plus de détails, consultez [Propriété Etat sur les champs](../Proprietes/2510039.md).

- une variable de type [TâcheParallèle](../WDLang1/1000021220.md).

- une variable de type [Thread](../WDLang1/1000024879.md).

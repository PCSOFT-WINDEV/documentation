


## PersistanceRubriqueCalculée (Propriété)

***En anglais : PersistedCalculatedItem***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **PersistanceRubriqueCalculée** permet de :

- Savoir si la valeur d'une rubrique calculée est mémorisée dans le fichier de données. Cette rubrique a été définie sous l'éditeur d'analyses ou définie par programmation.

- Définir si la valeur d'une rubrique calculée est mémorisée dans le fichier de données (lors de la description de la rubrique par programmation uniquement).




Remarque : Les rubriques calculées correspondent aux rubriques "Formule" sous l'éditeur d'analyses. 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
UnFichier est une Description de Fichier
UneRubrique est une Description de Rubrique
UneRubrique.Nom = "DateCmd"
UneRubrique.FormuleRubriqueCalculée = "'Ecrit par ' + CURRENT_USER + ' le ' + SYSDATE"
UneRubrique.PersistanceRubriqueCalculée = Vrai
HDécritRubrique(UnFichier, UneRubrique)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Savoir si la valeur d'une rubrique calculée est mémorisée dans le fichier de données

`<Résultat> = <Fichier de données>.<Rubrique>.PersistanceRubriqueCalculée`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la valeur de la rubrique calculée est mémorisée dans le fichier de données.

- <u><u><u><u>Faux</u></u></u></u> si la valeur de la rubrique est recalculée à chaque lecture.




**`<Fichier de données> : (Nom de fichier)`**

Nom du fichier de données manipulé.

**`<Rubrique> : (Nom de rubrique)`**

Nom de la rubrique manipulée.

### Indiquer si la valeur d'une rubrique calculée doit être mémorisée dans le fichier de données

`<Rubrique>.PersistanceRubriqueCalculée = <Valeur>`
---

**`<Rubrique> : (Nom de rubrique)`**

Nom de la rubrique manipulée. Ce nom vient d'être défini grâce au type [Description de Rubrique](../WDLang4/1514071.md).

**`<Valeur> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la valeur de la rubrique calculée doit être mémorisée dans le fichier de données.

- <u><u><u><u>Faux</u></u></u></u> si la valeur de la rubrique doit être recalculée à chaque lecture.

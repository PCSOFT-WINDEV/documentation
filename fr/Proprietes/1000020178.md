


## HTMLEntête (Propriété)

***En anglais : HTMLHeader***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **HTMLEntête** permet de connaître et de modifier le code HTML ajouté dans l'entête de la page.  

Remarque : Il est également possible de spécifier le code HTML ajouté à l'entête de la page dans l'onglet "Avancé" de la fenêtre de description de la page (onglet "Html"). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ajout de meta tag facebook
sCodeFacebook est une chaîne = [
<a name="fb_share"></a> 
<script src="http://static.ak.fbcdn.net/connect.php/js/FB.Share" 
        type="text/javascript">
</script>
]

MaPage.HTMLEntête = sCodeFacebook
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le contenu HTML inséré dans l'entête de la page

`<Contenu HTML> = <Page manipulée>.HTMLEntête`
---

**`<Contenu HTML> : (Chaîne de caractères)`**

Contenu HTML inséré dans l'entête de la page.

**`<Page manipulée> : (Nom de page)`**

Nom de la page à manipuler.  


<a name="SYNTAXE2"></a>

### Modifier le contenu HTML inséré dans l'entête de la page

`<Page manipulée>.HTMLEntête = <Contenu HTML>`
---

**`<Page manipulée> : (Nom de page)`**

Nom de la page à manipuler.

**`<Contenu HTML> : (Chaîne de caractères)`**

Contenu HTML à insérer dans l'entête de la page.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Le code HTML est inséré au milieu de l'entête de la page (après la déclaration des .js).

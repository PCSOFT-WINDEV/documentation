


## Importance (Propriété)

***En anglais : Importance***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Importance** permet connaître et de modifier le niveau d'importance d'un rendez-vous. Cette propriété est utilisable sur :

- une variable de type [EvénementTimeline](../WDLang1/1000020612.md).

- une variable de type [RendezVous](../WDLang1/1000019244.md).


## BaseDeDonnées (Propriété)

***En anglais : Database***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **BaseDeDonnées** permet de connaître et de modifier la base de données associée à une connexion. Cette propriété est utilisable sur :

- une connexion. Pour plus de détails, consultez l'aide de la propriété [BaseDeDonnées](../Proprietes/2512101.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [mongoConnexion](../WDLang4/1000022409.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [mongoGridFS](../WDLang4/1000022416.md).

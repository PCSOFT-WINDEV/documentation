


## MasqueTitreHeure (Propriété)

***En anglais : MaskTitleTime***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **MasqueTitreHeure** permet de connaître ou de modifier le masque utilisé pour le titre des colonnes des heures : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) dans un champ Agenda.

- dans un champ Planning. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) dans un champ TimeLine. 





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
PLN_Planning.MasqueTitreHeure = "HH:MM:SS" // Affiche "01:00:00"
```
<a name="Exemple2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 
```wl
TL_TimeLine.MasqueTitreHeure = "SS'CC" // Affiche "01'23"
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître le masque des titres de colonnes des heures

`<Résultat> = <Champ manipulé>.MasqueTitreHeure`
---

**`<Résultat> : (Chaîne de caractères)`**

Masque d'affichage actuel des titres de colonnes des heures. Ce masque correspond à : 

- " " (chaîne vide) si le masque utilisé est le masque par défaut. 

- le masque d'affichage utilisé (voir Remarques). 




**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) champ Agenda.

- champ Planning.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) champ TimeLine. 





<a name="SYNTAXE2"></a>

### Modifier le masque des titres de colonnes des heures

`<Champ manipulé>.MasqueTitreHeure = <Nouveau masque>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) champ Agenda.

- champ Planning.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) champ TimeLine.




**`<Nouveau masque> : (Chaîne de caractères)`**

Nouveau masque d'affichage des titres de colonnes des heures. Ce masque peut correspondre : 

- à un masque prédéfini ou personnalisé (voir Remarques).

- à "" (chaîne vide) pour utiliser le masque d'affichage par défaut. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Formats d'affichage disponibles
<a name="formats_affichage_disponibles_ELTPARAGRAPHE000086"></a>Le format d'affichage peut correspondre à : 

- un masque prédéfini : 

	- 'HH:MM'

	- 'HH:MM:SS'

	- 'HH:MM:SS:CC'

	- 'HH:MM:SS:CCC'

	- 'HH:MM AM'




- un masque personnalisé. Dans ce cas, il est possible d'utiliser les éléments suivants : 

	- 'HH' (heure)

	- 'MM' (minutes)

	- 'SS' (secondes)

	- 'CC' ou 'CCC' (centièmes de seconde sur 2 ou 3 caractères)

	- 'AM' (pour gérer l'heure au format AM/PM).

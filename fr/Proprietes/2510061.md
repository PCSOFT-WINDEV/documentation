


## Driver (Propriété)

***En anglais : Driver***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Driver** permet de connaître le driver de capture Vidéo auquel est associé le champ Caméra.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
Info("Le champ Caméra de cette fenêtre est associé au driver " + CAM_Caméra1.Driver)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Champ Camera>.Driver`
---

**`<Résultat> : (Chaîne de caractères)`**



- Chaîne vide ("") si le champ caméra n'est associé à aucun driver de capture Vidéo. Aucune fonction Vidéo ne pourra être utilisée.

- Driver de capture vidéo associé au champ Caméra spécifié. Ce résultat est sous la forme : 
	
	```txt
	<Nom du driver> + TAB + <Version du driver>
	```

	Où : 

	- &lt;Nom du driver &gt; correspond au nom du driver vidéo,

	- &lt;Version du driver&gt; correspond à la version du driver.







**`<Champ Camera> : (Nom de champ)`**

Nom du champ Caméra à manipuler

<a name="NOTE0_1"></a>

## Remarques

<a name="gestion_des_drivers_capture_video_ELTPARAGRAPHE000040"></a>La fonction [VidéoChangeDriver](../WDLang1/3079010.md) permet de modifier le driver de capture Vidéo associé au champ Caméra et la fonction [VidéoListeDriver](../WDLang1/3079008.md) permet d'obtenir la liste des drivers de capture Vidéo installés sur le poste.

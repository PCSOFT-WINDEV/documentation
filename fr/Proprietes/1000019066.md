


## EtendueTotale (Propriété)

***En anglais : TotalRange***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **EtendueTotale** permet de connaître et de modifier le nombre total de données représentées par le champ Range Slider. Par exemple, si l'étendue totale est de 100, le champ Range Slider représente des données allant de 1 à 100. 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=RangeSlider_Etendue.gif)
L'étendue totale peut également être paramétrée dans la fenêtre de description du champ Range Slider (onglet "Général"). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affiche les propriétés du champ Range Slider
SAI_Valeur = MoiMême..Valeur
SAI_BORNE_MIN = MoiMême.BorneMin
SAI_BORNE_MAX = MoiMême.BorneMax
SAI_ETENDUE_TOTALE = MoiMême.EtendueTotale
SAI_ETENDUE_VISIBLE = MoiMême.EtendueVisible
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître l'étendue totale du champ Range Slider

`<Résultat> = <Champ Range Slider>.EtendueTotale`
---

**`<Résultat> : (Entier)`**

Etendue totale du Range Slider. 

**`<Champ Range Slider> : (Nom de champ)`**

Nom du champ Range Slider manipulé. 


<a name="SYNTAXE2"></a>

### Modifier l'étendue totale du champ Range Slider

`<Champ Range Slider>.EtendueTotale = <Nouvelle étendue totale>`
---

**`<Champ Range Slider> : (Nom de champ)`**

Nom du champ Range Slider manipulé.

**`<Nouvelle étendue totale> : (Entier)`**

Nouvelle étendue totale du Range Slider.




## ModeAjustement (Propriété)

***En anglais : AdjustmentMode***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **ModeAjustement** permet de connaître et de modifier le mode d'ajustement d'un champ Tableur dans un état.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Traitement Initialisation du champ TBLR_Satistiques
TBLR_Satistiques = "DefileMode.xlsx"
TBLR_Satistiques.ModeAjustement = modeAjustementLignesEtColonnes
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître le mode d'ajustement d'un champ Tableur dans un état

`<Résultat> = <Champ Tableur>.ModeAjustement`
---

**`<Résultat> : (Constante de type Entier)`**

Mode d'ajustement du champ Tableur : 


|   |   |
| --- | --- |
| *modeAjustementAucun* | Aucune mise à l'échelle n'est effectuée. |
| *modeAjustementColonnes* | Toutes les colonnes sont ajustées à la largeur du champ. |
| *modeAjustementLignes* | Toutes les lignes sont ajustées à la hauteur du champ. |
| *modeAjustementLignesEtColonnes* | La feuille de calcul est ajustée aux dimensions du champ Tableur. |



**`<Champ Tableur> : (Nom de champ)`**

Nom du champ Tableur à manipuler dans l'état. 


<a name="SYNTAXE2"></a>

### Modifier le mode d'ajustement d'un champ Tableur dans un état

`<Champ Tableur>.ModeAjustement = <Nouvel ajustement>`
---

**`<Champ Tableur> : (Nom de champ)`**

Nom du champ Tableur à manipuler dans l'état. 

**`<Nouvel ajustement> : (Constante de type Entier)`**

Nouveau mode d'ajustement du champ Tableur : 


|   |   |
| --- | --- |
| *modeAjustementAucun* | Aucune mise à l'échelle n'est effectuée. |
| *modeAjustementColonnes* | Toutes les colonnes sont ajustées à la largeur du champ. |
| *modeAjustementLignes* | Toutes les lignes sont ajustées à la hauteur du champ. |
| *modeAjustementLignesEtColonnes* | La feuille de calcul est ajustée aux dimensions du champ Tableur. |

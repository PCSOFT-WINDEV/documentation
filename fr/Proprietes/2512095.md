
## Année (Propriété)

***En anglais : Year***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Année** permet de :

- Récupérer l'année d'une variable ou d'une rubrique de type [Date](../Motscles/1514066.md) ou [DateHeure](../Motscles/1514070.md).

- Modifier l'année d'une variable ou d'une rubrique de type [Date](../Motscles/1514066.md) ou [DateHeure](../Motscles/1514070.md).




Remarque : La propriété **Année** permet de simplifier les opérations de décalage sur les dates : ajouter une année, etc.

**Rappel** : Les rubriques et variables de type Date permettent de gérer :

- les dates simples "Année - Mois - Jour" (format AAAAMMJJ).

- les dates et les heures "Année - Mois - Jour - Heures - Minutes - Secondes - Millièmes de seconde" (format AAAAMMJJHHmmSSCCC).

- les durées "Nombre de jours - Nombre d'heures - Nombre de minutes - Nombre de secondes - Nombre de millièmes de seconde" (format +JHHMMSSCCC).





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
DateDébut est une Date = DateSys()
// Ajouter 1 an à la date
DateDébut.Année++
// Modifier les années
DateDébut.Année = 2003
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître l'année d'une variable ou d'une rubrique de type Date ou DateHeure

`<Résultat> = <Date>.Année`
---

**`<Résultat> : (Entier)`**

Année sur 4 chiffres.

**`<Date> : (Date ou DateHeure)`**

Nom de la variable ou de la rubrique de type [Date](../Motscles/1514066.md) ou [DateHeure](../Motscles/1514070.md) à manipuler. 




<a name="SYNTAXE2"></a>

### Modifier l'année d'une variable ou d'une rubrique de type Date ou DateHeure

`<Date>.Année = <Nouvelle année>`
---

**`<Date> : (Date ou DateHeure)`**

Nom de la variable ou de la rubrique de type [Date](../Motscles/1514066.md) ou [DateHeure](../Motscles/1514070.md) à manipuler.



**`<Nouvelle année> : (Entier ou chaîne de caractères)`**

Nouvelle année en chiffres (compris entre 1 et 9999). Remplace l'année de la date manipulée.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Opérateurs disponibles sur les années
<a name="operateurs_disponibles_sur_les_annees_ELTPARAGRAPHE000151"></a>Les opérateurs arithmétiques suivants peuvent être utilisés avec la propriété **Année** :

- ++ et --

- += et -=





```wl
DateDébut est une Date = "20011212"
DateDébut.Année++  // Ajouter 1 an à la date
DateDébut.Année+=5 // Ajouter 5 ans à la date
DateDébut.Année-=5 // Retrancher 5 ans à la date
```



<a name="NOTE0_2"></a>


### Années bissextiles
<a name="annees_bissextiles_ELTPARAGRAPHE000167"></a>Si la date manipulée correspond au 29 février d'une année bissextile, l'augmentation ou la diminution d'une année de la date entraîne également le changement de jour. Par exemple :


```wl
DateDébut est une Date = "19960229"
// Ajouter 1 an à la date
DateDébut.Année++ 
// DateDébut vaut "19970228"
```



<a name="NOTE0_3"></a>


### Calculs sur les dates
<a name="calculs_sur_les_dates_ELTPARAGRAPHE000177"></a>Le format de stockage des dates permet de stocker des dates comprises entre le 01/01/0001 et le 31/12/9999.

Les calculs sur les dates réalisés avec les [fonctions WLangage](../WDLang1/3027029.md) ou les [propriétés WLangage](../Proprietes/2514010.md) sont fiables à partir du 1er Janvier 1583.

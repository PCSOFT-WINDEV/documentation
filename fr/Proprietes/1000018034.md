
## Modifié (Propriété)

***En anglais : Modified***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Modifié** permet de savoir si un élément a été modifié. Cette propriété est utilisable sur :

- les champs d'une fenêtre. Pour plus de détails, consultez [Modifié sur les champs d'une fenêtre](../Proprietes/2510080.md).

- une variable de type [xlsDocument](../WDLang5/1000017464.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [Document](../WDLang1/1000022461.md).




## ModeLoupe (Propriété)

***En anglais : MagnifierMode***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **ModeLoupe** permet connaître et de modifier le mode de gestion de la loupe dans le titre des colonnes d'un champ Table (ou Table hiérarchique).

**Remarque** : Le mode de gestion de la loupe indique ce que doit faire le champ Table lorsque l'utilisateur clique sur la loupe présente dans le titre des colonnes. Ce mode de gestion est également utilisé par la fonction [TableSaisieLoupe](../WDLang1/3074024.md).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Définition du mode d'utilisation de la loupe
TABLE_Table1.ModeLoupe = filtreContient
// Utilisation de la loupe
TableSaisieLoupe(TABLE_Table1.COL_Colonne1, "")
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le mode de gestion de la loupe dans un champ Table

`<Résultat> = <Champ manipulé>.ModeLoupe`
---

**`<Résultat> : (Constante de type Entier)`**

Mode de gestion actuel de la loupe dans le champ Table ou la colonne spécifiée. Ce paramètre peut correspondre à une des constantes suivantes :


|   |   |
| --- | --- |
| *filtreCommencePar* | Un clic sur la loupe lancera un filtre de type "Commence par" : seules seront affichées les lignes dont la valeur de la colonne "Commence par" les caractères indiqués dans la zone de saisie de la colonne. |
| *filtreContient* | Un clic sur la loupe lancera un filtre de type "Contient" : seules seront affichées les lignes dont la valeur de la colonne "Contient" les caractères indiqués dans la zone de saisie de la colonne. |
| *filtreDifférent* | Un clic sur la loupe lancera un filtre de type "Différent" : seules seront affichées les lignes dont la valeur de la colonne "est différente" des caractères indiqués dans la zone de saisie de la colonne. |
| *filtreEgal* | Un clic sur la loupe lancera un filtre de type "Est égal" : seules seront affichées les lignes dont la valeur de la colonne "est égale" aux caractères indiqués dans la zone de saisie de la colonne. |
| *filtreInférieur* | Un clic sur la loupe lancera un filtre de type "est inférieur" : seules seront affichées les lignes dont la valeur de la colonne "est inférieure" aux caractères indiqués dans la zone de saisie de la colonne. |
| *filtreInférieurOuEgal* | Un clic sur la loupe lancera un filtre de type "est inférieure ou égale" : seules seront affichées les lignes dont la valeur de la colonne "est inférieure ou égale" aux caractères indiqués dans la zone de saisie de la colonne. |
| *filtreNeCommencePasPar* | Un clic sur la loupe lancera un filtre de type "Ne commence pas par" : seules seront affichées les lignes dont la valeur de la colonne "ne commence pas par" les caractères indiqués dans la zone de saisie de la colonne. |
| *filtreNeContientPas* | Un clic sur la loupe lancera un filtre de type "ne contient pas" : seules seront affichées les lignes dont la valeur de la colonne "ne contient pas" les caractères indiqués dans la zone de saisie de la colonne. |
| *filtreNeTerminePasPar* | Un clic sur la loupe lancera un filtre de type "Ne termine pas par" : seules seront affichées les lignes dont la valeur de la colonne "ne se termine pas par" les caractères indiqués dans la zone de saisie de la colonne. |
| *filtreSupérieur* | Un clic sur la loupe lancera un filtre de type "Est supérieur" : seules seront affichées les lignes dont la valeur de la colonne "est supérieure" aux caractères indiqués dans la zone de saisie de la colonne. |
| *filtreSupérieurOuEgal* | Un clic sur la loupe lancera un filtre de type "Est supérieur ou égal" : seules seront affichées les lignes dont la valeur de la colonne "est supérieure ou égale" aux caractères indiqués dans la zone de saisie de la colonne. |
| *filtreTerminePar* | Un clic sur la loupe lancera un filtre de type "Termine par" : seules seront affichées les lignes dont la valeur de la colonne "se termine par" les caractères indiqués dans la zone de saisie de la colonne. |
| *RechercheCommencePar* | Un clic sur la loupe triera la table selon la colonne spécifiée, passera cette colonne en mode recherche : seules seront affichées les lignes dont la valeur de la colonne "commence par" les caractères indiqués dans la zone de saisie de la colonne. |
| *RechercheDéfaut* | Indique que la colonne utilisera le mode de gestion de la loupe défini pour la table associée. Cette constante est disponible uniquement pour les colonnes de table. |



**`<Champ manipulé> : (Nom de champ)`**

Nom de la colonne de table, du champ Table ou Table hiérarchique à manipuler.  


<a name="SYNTAXE2"></a>

### Modifier le mode de gestion de la loupe dans un champ Table

`<Champ manipulé>.ModeLoupe = <Nouveau mode de gestion>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom de la colonne de table, du champ Table ou Table hiérarchique à manipuler.

Si la modification est effectuée sur le champ Table, cette modification affecte toutes les colonnes avec la constante *RechercheDéfaut* (si c'est possible).

Si la modification est effectuée sur une colonne de table, l'effet est limité à cette colonne.

<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
La propriété **ModeLoupe** est utilisable uniquement sur les champs Table, les colonnes de champ Table/Table hiérarchique et les champs Table hiérarchique.

La propriété **ModeLoupe** est sans effet :

- sur les champs Table fichier de type "Accès direct".

- sur les champs Table fichier avec un ascenseur à rebond.

- sur les colonnes qui n'affichent pas de loupe et qui ne sont pas triables.

- sur les colonnes ne supportant pas le type de filtre demandé (par exemple, l'utilisation d'un filtre de type *filtreContient* sur une colonne numérique).




## CouleurFond (Propriété)

***En anglais : BackgroundColor***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **CouleurFond** permet de connaître ou de modifier la couleur de fond de : 

- un champ.

- une fenêtre. 

- une page.

- un bloc d'un état.

- un libellé d'une option de menu.




**Remarque** : Selon le type de champ, les effets de la propriété **CouleurFond** peuvent varier (consultez le paragraphe "Remarques" pour plus de détails).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Selon la valeur affichée, la couleur de fond est modifiée
SI SAI_Total > 15000 ALORS
	SAI_Total.CouleurFond = RougeClair
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître la couleur de fond

`<Couleur de fond> = <Elément manipulé>.CouleurFond`
---

**`<Couleur de fond> : (Entier ou constante)`**

Couleur de l'élément spécifié. Correspond à l'une des options suivantes :

- une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)),

- [une couleur prédéfinie](../WDLang5/3010002.md).




**`<Elément manipulé> : (Type de l'élément)`**

Nom de l'élément à manipuler : champ, fenêtre, page ou bloc de l'état.  


<a name="SYNTAXE2"></a>

### Modifier la couleur de fond

`<Elément manipulé>.CouleurFond = <Nouvelle couleur>`
---

**`<Elément manipulé> : (Type de l'élément)`**

Nom de l'élément à manipuler : champ, fenêtre, page ou bloc de l'état.

**`<Nouvelle couleur> : (Entier ou constante)`**

Nouvelle couleur de l'élément spécifié. Correspond à l'une des options suivantes :

- une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)),

- une couleur TSL (obtenue avec la fonction [TSL](../WDLang1/3029057.md)),

- [une couleur prédéfinie](../WDLang5/3010002.md).

### Couleur de fond d'une cellule, d'une colonne, d'une ligne d'un champ Table et d'un champ Table
<a name="couleur_fond_une_cellule_une_colonne_une_ligne_champ_table_champ_table_ELTPARAGRAPHE000267"></a>Utilisée sur un champ Table dans son ensemble, la propriété **CouleurFond** permet de :

- Supprimer les couleurs particulières aux cellules et colonnes grâce à la syntaxe suivante :
	
	```txt
	<Champ Table>.CouleurFond = CouleurDéfaut
	```


- Modifier la couleur de fond de toutes les colonnes du champ Table (colonnes et cellules non colorées spécialement). Par exemple le code suivant colore toutes les colonnes en bleu :
	
	```txt
	<Champ Table>.CouleurFond = BleuFoncé
	```





Cette propriété permet également de connaître ou de modifier la couleur de fond d'une cellule du champ Table. Il est nécessaire d'utiliser la syntaxe suivante pour désigner la cellule : 

```txt
<Champ Table>[NumLigne][NumColonne]
```


Exemple : Le code suivant permet de modifier la couleur de fond de la cellule de la première ligne, deuxième colonne.

```wl
TABLE_Table1[1][2].CouleurFond = BleuFoncé
```
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=table.gif)
**Attention** : Pour conserver la couleur de fond lors de la sélection d'une ligne dans un champ Table fichier, il est nécessaire d'utiliser la propriété **CouleurFond** dans le traitement intitulé "Sélection d'une ligne de table".
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) **Remarque** : Il est possible de modifier le style de la couleur de fond d'une cellule, d'une colonne ou d'une ligne d'un champ Table ou Table hiérarchique grâce à la propriété [StyleFond](../Proprietes/2513019.md).

### Couleur de fond d'un champ Onglet
<a name="couleur_fond_champ_onglet_ELTPARAGRAPHE000330"></a>Pour les champs Onglet :

- Pour le volet d'onglet : 

	- La propriété [Couleur](../Proprietes/2510071.md) permet de connaître et de modifier la couleur du texte des volets d'onglet.

- Pour le champ Onglet :  La propriété **CouleurFond** permet de connaître et de modifier la couleur de fond du champ Onglet (couleur de fond des onglets, des volets d'onglet et couleur apparaissant derrière les onglets).

### Limites
<a name="limites_ELTPARAGRAPHE000379"></a>![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) **Limites sur les champs d'un état**
La propriété **CouleurFond** n'a aucun effet sur :

- les états (dans leur totalité).

- les champs Etat Interne.

- les champs Signature.


![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) **Limites sur les champs des fenêtres WINDEV**
La propriété **CouleurFond** n'a aucun effet sur les champs suivants :

- les champs HTML,

- les champs Web caméra,

- les champs Potentiomètre. 


 
 
![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) La propriété **CouleurFond** s'applique uniquement aux éléments suivants :

- Champ Bouton.

- Champ Libellé.

- Champ de saisie.

- Champ Image.

- Champ Interrupteur.

- Champ Sélecteur.

- Champ Liste.

- Champ Combo.

- Champ Arbre.

- Champ Onglet.

- Fenêtre.

- Champ Table.

- Colonne de champ Table.




## Mois (Propriété)

***En anglais : Month***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Mois** permet de connaître et de modifier le mois d'un élément. Cette propriété est utilisable sur :

- une variable ou une rubrique de type Date, Heure et DateHeure. Pour plus de détails, consulter [Propriété Mois sur les types Date, Heure et DateHeure](../Proprietes/2512052.md).  

- une variable de type [hTâchePlanifiée](../WDLang4/1000017448.md). 

- une variable de type [hPlanification](../WDLang4/1000018853.md).

- une variable de type [Répétition](../WDLang1/1000019262.md).

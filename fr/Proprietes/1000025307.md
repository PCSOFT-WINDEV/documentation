
## PortionAffichée (Propriété)

***En anglais : DisplayedSection***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **PortionAffichée** permet de connaître et de modifier la portion d'image affichée dans un champ Image.

Cette propriété permet notamment de zoomer une image en affichant uniquement une portion de l'image. Cette propriété doit être utilisée dans un champ Image autorisant le zoom. 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Sauve la portion affichée
rectSauve est un Rectangle
rectSauve = IMG_Photo.PortionAffichée
// Restaure la portion affichée
IMG_Photo.PortionAffichée = rectSauve

// Zoom X2 en haut à droite
rectX2 est un Rectangle
rectX2.X = 1000
rectX2.Y = 0
rectX2.Largeur = 500
rectX2.Hauteur = 500
IMG_Photo.PortionAffichée = rectX2

rectX2.PortionAffichée = O
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître la portion d'image affichée dans un champ Image

`<Résultat> = <Champ Image>.PortionAffichée`
---

**`<Résultat> : (Variable de type Rectangle)`**

Variable de type [Rectangle](../Motscles/1514079.md) correspondant à la portion de l'image affichée.

Si le champ Image ne contient pas d'image (ou contient une image générée (fonctions de dessin)), le rectangle renvoyé correspond au champ.  

**`<Champ Image> : (Nom de champ)`**

Nom du champ Image à manipuler.


<a name="SYNTAXE2"></a>

### Modifier la portion d'image affichée dans un champ Image

`<Champ Image>.PortionAffichée = <Nouvelle portion affichée>`
---

**`<Champ Image> : (Nom de champ)`**

Nom du champ Image à manipuler.

**`<Nouvelle portion affichée> : (Variable de type Rectangle)`**



- Nom de la variable de type [Rectangle](../Motscles/1514079.md) correspondant à la portion de l'image à afficher. 

- 0 pour afficher toute l'image dans le champ Image. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Le rectangle est en pixels de l'image. 

- Si l'image est multi-DPI, le rectangle correspond au grossissement 1x. 

- Lors de la modification de la portion d'image affichée (syntaxe 2) : 

	- Le zoom est calculé en fonction de la taille du rectangle fournie. 

	- Si le rectangle n'a pas le même ratio x/y que le champ Image, le plus grand ratio est utilisé. 

	- Si le rectangle est vide, l'affectation est sans effet.

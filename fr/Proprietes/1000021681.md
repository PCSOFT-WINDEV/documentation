


## ModeTransparence (Propriété)

***En anglais : TransparencyMode***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **ModeTransparence** permet de connaître et de modifier le mode de transparence d'une image.

Remarque : Le mode de transparence est défini sous l'éditeur grâce à l'option "Transparence" présente dans la fenêtre de description des champs. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Modification du mode de transparence de l'image du champ "IMG_ImageVoiture"
IMG_ImageVoiture.ModeTransparence = trsImage
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître le mode de transparence actuel

`<Transparence actuelle> = <Champ manipulé>.ModeTransparence`
---

**`<Transparence actuelle> : (Constante de type Entier)`**

Mode de transparence actuel de l'image :


|   |   |
| --- | --- |
| *trsCoinHautGauche* | Les points de la même couleur que le point haut gauche de l'image seront transparents. |
| *trsImage*  | La transparence de l'image est conservée. |
| *trsMagenta* | Le magenta présent dans l'image sera transparent à l'affichage. |



**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler : 

- Champ Image.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Colonne de table de type Image.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Fond d'un champ Graphe.





<a name="SYNTAXE2"></a>

### Modifier le mode de transparence

`<Champ manipulé>.ModeTransparence = <Nouveau mode>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler : 

- Champ Image.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Colonne de table de type Image.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Fond d'un champ Graphe.




**`<Nouveau mode> : (Constante de type Entier)`**

Nouveau mode de transparence de l'image :


|   |   |
| --- | --- |
| *trsCoinHautGauche* | Les points de la même couleur que celle du point haut gauche de l'image seront transparents. |
| *trsImage*  | La transparence de l'image est conservée. |
| *trsMagenta* | Le magenta présent dans l'image sera transparent à l'affichage. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Image de fond d'un champ Graphe
<a name="image_fond_champ_graphe_ELTPARAGRAPHE000097"></a>Lors de la modification du mode de transparence de l'image de fond d'un champ Graphe, il est nécessaire de rafraîchir l'affichage du graphe (par exemple avec la fonction [grDessine](../WDLang3/3042026.md)) pour faire apparaître le changement de mode de transparence.
<a name="NOTE0_2"></a>


### Limites
<a name="limites_ELTPARAGRAPHE000107"></a>La propriété **ModeTransparence** s'applique uniquement aux champs suivants :

- Champ Image.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Colonne de table de type Image.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Fond d'un champ Graphe.

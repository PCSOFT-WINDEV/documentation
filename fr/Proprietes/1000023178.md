
## ModeAjustement (Propriété)

***En anglais : AdjustmentMode***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **ModeAjustement** permet de connaître et de modifier le mode d'ajustement d'un champ Tableur pour l'impression. Cette propriété est utilisable sur :

- un champ Tableur dans un état. Pour plus de détails, consultez [ModeAjustement](../Proprietes/1000021790.md).

- une variable de type [xlsMiseEnPage](../WDLang5/1000023139.md).

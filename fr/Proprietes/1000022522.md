


## LongueurSélection (Propriété)

***En anglais : SelectionLength***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **LongueurSélection** permet de connaître et de modifier la longueur de la sélection effectuée dans un champ Traitement de texte.

**Astuce** :  Cette propriété permet d'afficher une sélection dans le champ Traitement de texte. Le point de départ de la sélection correspond à la position courante du curseur (manipulable avec la propriété [Curseur](../Proprietes/2510025.md)).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Passe en gras la sélection courante
SI TT_MonDoc.LongueurSélection <> 0 ALORS
	f est docFragment(TT_MonDoc, TT_MonDoc.Curseur, TT_MonDoc.LongueurSélection)
	f.Style.PoliceGras = Vrai
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître la longueur de la sélection

`<Résultat> = <Champ Traitement de texte>.LongueurSélection`
---

**`<Résultat> : (Entier)`**

Nombre de caractères constituant la sélection. 

**`<Champ Traitement de texte> : (Nom de champ)`**

Nom du champ Traitement de texte à manipuler. 


<a name="SYNTAXE2"></a>

### Modifier la longueur de la sélection

`<Champ Traitement de texte>.LongueurSélection = <Nouvelle longueur>`
---

**`<Champ Traitement de texte> : (Nom de champ)`**

Nom du champ Traitement de texte à manipuler. 

**`<Nouvelle longueur> : (Entier)`**

Nombre de caractères à sélectionner. Si la longueur de sélection est négative ou nulle, seul le curseur simple est affiché dans le champ.

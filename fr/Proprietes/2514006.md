


## Opacité (Propriété)

***En anglais : Opacity***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Opacité** permet de connaître et définir le pourcentage d'opacité d'une fenêtre ou des champs présents dans une fenêtre ou une page. Il est ainsi possible de rendre transparente une fenêtre (ou un champ présent dans une fenêtre ou une page).

**Remarques** : 

- Dans le cas d'une variable de type [Couleur](../WDLang1/1000019661.md), la valeur de l'opacité n'est pas comprise entre 0 et 100 mais entre 0 et 255. 





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 
```wl
// Rendre la fenêtre invisible petit à petit (fade out)
i est un entier
POUR i = 1 A 100
	FEN_FenêtreSaisie.Opacité = 100 - i
FIN
Ferme()
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le pourcentage d'opacité d'une fenêtre ou d'un champ (d'une fenêtre ou d'une page)

`<Pourcentage d'opacité> = <Elément manipulé>.Opacité`
---

**`<Pourcentage d'opacité> : (Entier)`**

Pourcentage d'opacité de l'élément spécifié (fenêtre ou champ). Ce pourcentage est compris entre 0 (fenêtre complètement invisible) et 100 (fenêtre complètement opaque).

**`<Elément manipulé> : (Nom de fenêtre, nom de champ ou nom de groupe de champs)`**

Nom de l'élément à manipuler : 

- Nom de la fenêtre,
	

- Nom du champ, 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Nom du groupe de champs.





<a name="SYNTAXE2"></a>

### Modifier le pourcentage d'opacité d'une fenêtre ou d'un champ (d'une fenêtre ou d'une page)

`<Elément manipulé>.Opacité = <Pourcentage d'opacité>`
---

**`<Elément manipulé> : (Nom de fenêtre, nom de champ ou nom de groupe de champs)`**

Nom de l'élément à manipuler : 

- Nom de la fenêtre, 
	

- Nom du champ, 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Nom du groupe de champs.




**`<Pourcentage d'opacité> : (Entier)`**

Nouveau pourcentage d'opacité à appliquer à l'élément spécifié (fenêtre ou champ). Ce pourcentage est compris entre 0 (fenêtre complètement invisible) et 100 (fenêtre complètement opaque). 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 

### Limite : système d'exploitation et carte graphique
<a name="limite_systeme_exploitation_carte_graphique_ELTPARAGRAPHE000095"></a>La propriété **Opacité** peut être utilisée uniquement sur les systèmes supérieurs à Windows 7.

Si la carte graphique ne gère pas l'effet de transparence, la propriété **Opacité** n'aura aucun effet.
<a name="NOTE0_2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 

### Limite : types de fenêtres
<a name="limite_types_fenetres_ELTPARAGRAPHE000112"></a>Le système Windows ne permet pas de gérer l'opacité des fenêtres de type Fille MDI.
<a name="NOTE0_3"></a>
<a name="NOTE0_4"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)


## Prénom (Propriété)

***En anglais : FirstName***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Prénom** peut être utilisée sur :

- une variable de type [SaasUtilisateur](../WDLang3/1000019083.md). 

- une variable de type [gpwUtilisateur](../WDLang6/1000021088.md).

- une variable de type [OpenIDIdentité](../WDLang3/1000023567.md).

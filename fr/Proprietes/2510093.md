


## ParcoursAuto (Propriété)

***En anglais : AutoBrowse***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **ParcoursAuto** permet de savoir si le parcours d'un champ Liste, Zone répétée, Table ou Combo fichier est automatique ou programmé.






<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Le parcours de la table "TABLE_Client" est-il automatique ou programmé ?
ResParcours = TABLE_Client.ParcoursAuto
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Champ manipulé>.ParcoursAuto`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le parcours de l'élément spécifié est automatique.

- <u><u><u><u>Faux</u></u></u></u> si le parcours de l'élément spécifié est programmé.




**`<Champ manipulé> : (Nom de champ)`**

Nom du champ manipulé : champ Liste, Table, Zone répétée ou Combo.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Type de parcours
<a name="type_parcours_ELTPARAGRAPHE000070"></a>

- ***Parcours automatique*** : le parcours du fichier de données et l'affichage des données dans l'élément sont automatiques. Le développeur n'a aucun code spécifique à saisir.
	Le tri est effectué sur la rubrique de parcours définie dans l'onglet "Contenu" de la description de l'élément. Si un filtre est spécifié (option "Filtre sur la rubrique de parcours" dans l'onglet "Contenu" de la description de l'élément), ce filtre est utilisé. Aucun traitement de parcours de l'élément spécifié n'est exécuté.

- ***Parcours programmé*** : Le développeur doit programmer le parcours du fichier et l'affichage des données dans l'élément.
	Les traitements "Lecture du premier enregistrement", "Lecture de l'enregistrement précédent", "Lecture de l'enregistrement suivant" et "Lecture du dernier enregistrement" sont exécutés. Un parcours programmé permet un contrôle total sur l'ordre et le filtrage des enregistrements. Cependant, un parcours programmé est plus complexe et plus lent qu'un parcours automatique.



<a name="NOTE0_2"></a>


### Limites
<a name="limites_ELTPARAGRAPHE000082"></a>La propriété **ParcoursAuto** s'applique uniquement aux :

- champs Liste fichier,

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) champs Table fichier,

- champs Combo fichier,

- champs Zone répétée fichier,

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) champs Liste image fichier.




![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) La propriété **ParcoursAuto** s'applique uniquement aux champs Table.

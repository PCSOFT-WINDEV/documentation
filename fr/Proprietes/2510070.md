


## VisibleInitial (Propriété)

***En anglais : InitialVisible***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **VisibleInitial** permet de savoir si :

- un champ était visible lors du premier affichage de la fenêtre ou la page associée.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une fenêtre était visible lors de son premier affichage.




**Remarques** :

- La visibilité initiale d'un champ ou d'une fenêtre correspond à celle fixée sous l'éditeur (onglet "Détail" de la description de l'objet). Cette option sera utilisée lors de l'initialisation du champ ou de la fenêtre.

- La visibilité d'un champ ou d'une fenêtre peut être modifiée grâce aux propriétés [Visible](../Proprietes/2510138.md) et [Etat](../Proprietes/2510039.md).









<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Remettre le champ "SAI_Société" dans son état de visibilité initiale
SAI_Société.Visible = SAI_Société.VisibleInitial
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Elément manipulé>.VisibleInitial`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'objet spécifié était visible lors de son premier affichage, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Elément manipulé> : (Nom de champ ou nom de fenêtre)`**

Nom de l'objet (champ ou fenêtre) à manipuler. 

Pour un champ associé à un onglet, utilisez la notation suivante : 

```txt
<Nom de l'onglet>.<Nom du champ>
```




<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Attribut d'un champ
<a name="attribut_champ_ELTPARAGRAPHE000106"></a>La propriété **VisibleInitial** ne modifie pas les autres attributs d'état du champ (sélection sans saisie, actif ou grisé).


### Limites
<a name="limites_ELTPARAGRAPHE000114"></a>La propriété **VisibleInitial** ne peut pas être utilisée sur :

- une option d'interrupteur ou de sélecteur,

- une ligne de table, de liste ou de combo,

- une cellule de table,

- un menu de fenêtre,

- un groupe de champs,

- un élément d'un champ Arbre.




![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) La propriété **VisibleInitial** s'applique uniquement aux éléments suivants :

- Champ Bouton.

- Champ Libellé.

- Champ de saisie.

- Champ Image.

- Champ Interrupteur.

- Champ Sélecteur.

- Champ Liste.

- Champ Combo.

- Champ Arbre.

- Champ Onglet.

- Fenêtre.

- Champ Table.

- Colonne de table.

- Superchamp.

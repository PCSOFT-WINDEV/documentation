
## DateHeureDébut (Propriété)

***En anglais : StartTimestamp***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **DateHeureDébut** permet de gérer la date et l'heure de début d'une action. Elle est disponible sur les variables de type :

- [hbLecture](../WDLang4/1000021701.md),

- [hbParamètreScan](../WDLang4/1000021715.md).

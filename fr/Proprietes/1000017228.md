


## DroiteAGauche (Propriété)

***En anglais : RightToLeft***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **DroiteAGauche** permet de connaître le sens d'écriture réel, effectif sur un champ, une colonne, une fenêtre ou un état. 

Le sens d'écriture peut être modifié avec la propriété [SensEcriture](../Proprietes/2513020.md). Si le sens d'écriture utilisé dans le champ est le sens d'écriture défini dans les options du projet, la propriété [SensEcriture](../Proprietes/2513020.md) renvoie la constante *seDéfaut*. Dans ce cas, la propriété **DroiteAGauche** permet de connaître le sens d'écriture réellement utilisé.

Cette propriété s'applique à :

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Une fenêtre ou les champs d'une fenêtre (sauf : Champ indicé, Champ Web caméra, Splitter, Calendrier et Agenda).

- Un état ou les champs suivants d'un état : 

	- Champ Libellé, 

	- Champ Calculé, 

	- Champ texte RTF, 

	- Champ Prédéfini, 

	- Lien et Champ de saisie, 

	- Champ Graphe.







![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Attention : Cette propriété est disponible uniquement dans les états.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 
```wl
// Le sens d'écriture de la colonne est le sens par défaut
TABLE_Table1.COL_Colonne2.SensEcriture = seDéfaut
// Recherche du sens d'écriture par défaut
SI TABLE_Table1.COL_Colonne2.DroiteAGauche = Vrai ALORS
	SAI_Saisie1.SensEcriture = seDroiteAGauche
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le sens d'écriture effectif utilisé

`<Résultat> = <Nom de l'élément>.DroiteAGauche`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le sens d'écriture utilisé est de droite à gauche, 

- <u><u><u><u>Faux</u></u></u></u> si le sens d'écriture utilisé est de gauche à droite.




**`<Nom de l'élément> : (Nom de fenêtre, nom d'état ou nom de champ)`**

Nom de l'élément pour lequel le sens d'écriture effectif est recherché : 

- Nom de la fenêtre, 

- Nom des champs de la fenêtre, 

- Nom de l'état, 

- Nom des champs de l'état. 




![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Ce paramètre correspond uniquement au nom de l'état ou au nom d'un champ de l'état.

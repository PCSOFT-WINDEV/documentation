
## SourceXML (Propriété)

***En anglais : XMLSource***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **SourceXML** permet de connaître le code source XML. Cette propriété est utilisable avec : 

- une variable de type [wsRéponse](../WDLang3/1000020568.md). 

- une variable de type [xmlNoeud](../WDLang5/1000018786.md).


## TypeSaisie (Propriété)

***En anglais : InputType***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **TypeSaisie** permet de connaître ou modifier le type de saisie permis par un champ : Durée, Date, Numérique, ...

Cette propriété s'applique aux champs suivants :

- Champ de saisie.

- Colonne d'un champ Table.

- Champ Libellé.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Colonne de table affichée par un champ Combo.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Champ Combo avec saisie.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Champ Tableau croisé dynamique.




**Remarque** : Le type d'un champ est défini dans l'onglet "Général" de la description du champ.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connaître le type du champ de saisie "SAI_Date"
SI SAI_Date.TypeSaisie = typSaisieDate ALORS
	Message("Saisissez une date")
FIN
```



```wl
// Connaître le type du champ de saisie "SAI_Date"
SI SAI_Date.TypeSaisie = typSaisieDate ALORS
	Message("Saisissez une date")
FIN
```



```wl
SAI_dynamique.TypeSaisie = SAI_DateHeure.TypeSaisie
SAI_dynamique.FormatMémorisé = "AAAAMMJJHHmmSSCCC"
SAI_dynamique.MasqueSaisie = SAI_DateHeure.MasqueSaisie 
SAI_dynamique = SAI_DateHeure
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le type d'un champ

`<Résultat> = <Champ manipulé>.TypeSaisie`
---

**`<Résultat> : (Constante)`**

Type du champ. Les valeurs possibles sont :


|   |   |
| --- | --- |
| *typSaisieDate* | Type date. |
| *typSaisieDurée* | Type durée. |
| *typSaisieHeure* | Type heure. |
| *typSaisieMonétaire* | Type monétaire. |
| *typSaisieNum* | Type numérique (entier, réel, ...). |
| *typSaisieNumériquePrécis* | Type numérique (uniquement pour les champs de saisie associés au type Numérique). |
| *typSaisieTexte* | Type texte. |



**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler.  


<a name="SYNTAXE2"></a>

### Modifier le type d'un champ

`<Champ manipulé>.TypeSaisie = <Nouveau type>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler.

**`<Nouveau type> : (Constante)`**

Nouveau type, les valeurs possibles sont :


|   |   |
| --- | --- |
| *typSaisieDate* | Type date. |
| *typSaisieDurée* | Type durée. |
| *typSaisieHeure* | Type heure. |
| *typSaisieMonétaire* | Type monétaire. |
| *typSaisieNum* | Type numérique (entier, réel, ...). |
| *typSaisieNumériquePrécis* | Type numérique (uniquement pour les champs de saisie associés au type Numérique). |
| *typSaisieTexte* | Type texte. |

## Remarques

### Champ Tableau croisé dynamique
<a name="champ_tableau_croise_dynamique_ELTPARAGRAPHE000166"></a>Pour connaître et de modifier le type affiché pour une valeur d'un champ Tableau croisé dynamique, il est possible d'utiliser les syntaxes suivantes : 


```txt
{{"Nom_Valeur"}}.TypeSaisie = typSaisieDurée
```


ou bien : 


```txt
Nom_Valeur.TypeSaisie = typSaisieDurée
```


## Connexion

***En anglais : Connection***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le mot-clé **Connexion** peut correspondre :

- au type de variable [Connexion](../WDLang4/1514073.md).

- à la propriété **Connexion** qui permet de connaître et de modifier la connexion actuellement associée à un fichier de données. 
	Pour plus de détails, consultez [Connexion](../Proprietes/2512019.md).

- à la propriété **Connexion** d'une variable de type [hbParamètreScan](../WDLang4/1000021715.md).

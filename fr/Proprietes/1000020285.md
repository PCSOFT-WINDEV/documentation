
## AvecSaisie (Propriété)

***En anglais : WithInput***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **AvecSaisie** permet de savoir si un champ Combo est avec ou sans saisie et permet de modifier cet état. 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI bModeExpert = Vrai ALORS
	COMBO_AdresseInternet.AvecSaisie = Vrai
FIN

SI COMBO_AdresseInternet.AvecSaisie = Vrai ALORS
	sAdresseNet = COMBO_AdresseInternet
SINON
	sAdresseNet = COMBO_AdresseInternet[COMBO_AdresseInternet]
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Savoir si un champ Combo est avec saisie ou non

`<Résultat> = <Champ Combo>.AvecSaisie`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le champ Combo est un champ Combo avec saisie, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Champ Combo> : (Nom de champ)`**

Nom du champ Combo à manipuler.  


<a name="SYNTAXE2"></a>

### Permettre ou non de saisir dans un champ Combo

`<Champ Combo>.AvecSaisie = <Type de saisie>`
---

**`<Champ Combo> : (Nom de champ)`**

Nom du champ Combo à manipuler. 

**`<Type de saisie> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le champ Combo doit être avec saisie, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.  










<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- La valeur renvoyée par le champ Combo est différente si le champ est en saisie (propriété **AvecSaisie** égale à <u><u><u><u>Vrai</u></u></u></u>) ou sans saisie (propriété **AvecSaisie** égale à <u><u><u><u>Faux</u></u></u></u>). 

- Dans un champ Combo en saisie (propriété **AvecSaisie** égale à <u><u><u><u>Vrai</u></u></u></u>), l'utilisateur peut saisir une valeur dans le champ au clavier. Dans le cas contraire, l'utilisateur peut uniquement choisir une valeur dans la liste/table/popup déroulée par la combo.

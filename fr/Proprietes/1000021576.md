
## AvancementEnvoi (Propriété)

***En anglais : UploadProgress***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **AvancementEnvoi** permet d'obtenir la notification de l'état d'avancement de l'envoi des données POST. Cette propriété est utilisable sur :

- une variable de type [httpRequête](../WDLang3/1000021158.md). 

- une variable de type [restRequête](../WDLang3/1000021481.md).

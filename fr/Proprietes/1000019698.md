
## Domaine

***En anglais : Domain***
	

<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Domaine** permet de connaître et de modifier les domaines gérés. 

Cette propriété est utilisable sur :

- une variable de type [Cookie](../WDLang3/1000019222.md).

- une variable de type [rssCatégorie](../WDLang5/1000017782.md). 

- une variable de type [rssCloud](../WDLang5/1000017785.md).

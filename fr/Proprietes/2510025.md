


## Curseur (Propriété)

***En anglais : Cursor***
	



<a name="XPresentation"></a>
<a name="Presentation"></a>


### Définition
<a name="definition_ELTPARAGRAPHE000010"></a>Le curseur de souris permet de visualiser facilement l'endroit où l'action va se dérouler. Dans le cas d'un champ en saisie, le curseur se transforme en trait vertical clignotant et permet d'identifier rapidement la position du prochain caractère saisi. 

Si une sélection est effectuée dans le champ, les caractères sélectionnés sont mis en surbrillance. La position du curseur correspond au début de la zone affichée en surbrillance. 

Quelques exemples :

| --- | --- |
| ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Position_fin.gif)<br> | Le curseur de saisie est placé à la fin de la zone de saisie. Sa position est 7 dans cet exemple. |
| ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=position_sel.gif)<br> | Le curseur de saisie est placé au début de la zone en en surbrillance (position 3 dans cet exemple). <a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Curseur** permet de connaître et de modifier la position du curseur de la souris dans un des champs suivants :

- Champ de saisie.

- Champ Combo avec saisie.

- Colonne d'un champ Table de type "Champ de saisie" et colonne d'un champ Table de type "Combo avec saisie".

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Champ Traitement de texte.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Champ Editeur HTML.




**Remarque** : Pour connaître le texte sélectionné dans un champ de saisie, un champ Combo avec saisie ou une colonne d'un champ Table, il est possible d'utiliser la propriété [TexteSélectionné](../Proprietes/1000017193.md).



**Astuce** : Pour sélectionner une partie d'un texte d'un champ de saisie à partir d'un champ Bouton :

1. Le champ de saisie ne doit pas avoir l'option "Effacement automatique" cochée (sauf si tout le contenu du champ doit être supprimé).

2. Le bouton doit être de type "Interruption" et contenir par exemple le code suivant :
	
	```wl
	// -- Clic du bouton BTN_Sélection (bouton de type Interruption)
	// Sélectionner tout le texte
	SAI_Saisie1.Curseur = 1
	SAI_Saisie1.FinCurseur = Taille(SAI_Saisie1) + 1
	EcranPremier(SAI_Saisie1)
	```

## Syntaxe

### Connaître la position du curseur de saisie

`<Valeur actuelle> = <Champ manipulé>.Curseur`
---

**`<Valeur actuelle> : (Entier)`**

Position actuelle du curseur de saisie dans le champ. Cette position peut varier entre :

- 0 (le champ n'est pas en saisie et il n'y a pas de curseur de saisie).

- Le nombre de caractères affichés dans le champ + 1 (le curseur de saisie se trouve à la fin de la zone de saisie).




**`<Champ manipulé> : (Nom de champ)`**

Nom du champ dans lequel le curseur de saisie est affiché : 

- Champ de saisie.

- Champ Combo avec saisie.

- Colonne d'un champ Table de type "Champ de saisie" et colonne d'un champ Table de type "Combo avec saisie".

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Champ Traitement de texte.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Champ Editeur HTML.





<a name="SYNTAXE2"></a>

### Modifier la position du curseur de saisie

`<Champ manipulé>.Curseur = <Nouvelle valeur>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ dans lequel la position du curseur de saisie doit être modifiée : 

- Champ de saisie.

- Champ Combo avec saisie.

- Colonne d'un champ Table de type "Champ de saisie" et colonne d'un champ Table de type "Combo avec saisie".

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Champ Traitement de texte.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Champ Editeur HTML.




**`<Nouvelle valeur> : (Entier)`**

Nouvelle position du curseur de saisie, comprise entre 1 et le nombre de caractères actuellement affichés dans le champ + 1.

## Remarques

### Cas particulier : Champ RTF
<a name="cas_particulier_champ_rtf_ELTPARAGRAPHE000172"></a>Le nombre de caractères d'un champ RTF peut être différent si ce champ contient des Retours Chariots (RC).

En effet, dans un champ non RTF, le caractère RC correspond à 2 caractères, et dans un champ RTF, le caractère RC correspond à un caractère.
<a name="NOTE0_2"></a>


### Limite : Modification de la position du curseur de saisie
<a name="limite_modification_position_curseur_saisie_ELTPARAGRAPHE000181"></a>La modification de la position du curseur de saisie n'est possible que si le champ est en cours de saisie. Si le champ n'est pas en cours de saisie, la propriété **Curseur** renvoie 0.

**Rappel** : Il est possible de forcer la saisie dans un champ en utilisant les fonctions [EcranPremier](../WDLang1/3025039.md) ou [DonneFocusEtRetourUtilisateur](../WDLang1/1410088107.md).


## CodeEtat

***En anglais : StatusCode***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **CodeEtat** permet de connaître l'état de la réponse à la requête. Cette propriété est utilisable sur :

- une variable de type [httpRéponse](../WDLang3/1000021165.md). 

- une variable de type [restRéponse](../WDLang3/1000021477.md).

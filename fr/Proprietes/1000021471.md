***En anglais : TabMDI***
	

<DIV class="specObsolete">
	<IMG style="float:left;margin-right:10px;" src="https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=warning_big.png"><B>Avertissement</B><BR>
	A partir de la version <B>20 (57)</B>,  il est conseillé d'utiliser la propriété [OngletDynamique](../Proprietes/1000021626.md). 
</DIV><a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **OngletMDI** permet de : 

- Connaître le type d'un champ Onglet (onglet simple ou onglet dynamique - MDI). 

- Modifier le type d'un champ Onglet (onglet simple ou onglet dynamique - MDI). 




**Rappel** : Un champ Onglet dynamique est un champ Onglet qui supporte l'ajout, la suppression et le déplacement des volets.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Vérifie si l'onglet est un onglet MDI
SI ONG_Onglet1.OngletMDI = Vrai ALORS
	// Fermeture de tous les volets
	OngletFermeTout(ONG_Onglet1)
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Savoir si un champ Onglet est un onglet MDI (dynamique)

`<Résultat> = <Champ Onglet>.OngletMDI`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le champ Onglet est un onglet MDI (dynamique), 

- <u><u><u><u>Faux</u></u></u></u> si le champ Onglet est un onglet simple. 




**`<Champ Onglet> : (Nom de champ)`**

Nom du champ Onglet à manipuler. 


<a name="SYNTAXE2"></a>

### Modifier le type d'un champ Onglet

`<Champ Onglet>.OngletMDI = <Type du champ Onglet>`
---

**`<Champ Onglet> : (Nom de champ)`**

Nom du champ Onglet à manipuler. 

**`<Type du champ Onglet> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> pour que le champ Onglet soit un champ Onglet MDI (dynamique), 

- <u><u><u><u>Faux</u></u></u></u> pour que le champ Onglet soit un champ Onglet simple.

## Remarques


### Champ Onglet MDI (dynamique)
<a name="champ_onglet_mdi_dynamique_ELTPARAGRAPHE000066"></a>Si la propriété **OngletMDI** est à <u><u><u><u>Vrai</u></u></u></u> : 

- les fonctions [OngletXXX](../WDLang1/1000021391.md) peuvent être appelées sur le champ.

- l'opérateur [] sur l'onglet doit être appelé avec l'alias du volet. 

- l'utilisateur final peut : 

	- supprimer des volets si la propriété [AvecBoutonFermeture](../Proprietes/1000021374.md) est à <u><u><u><u>Vrai</u></u></u></u>. 

	- ajouter des volets si la propriété [AvecBoutonNouveau](../Proprietes/1000021377.md) est à <u><u><u><u>Vrai</u></u></u></u>.

	- modifier l'ordre des volets par Drag and Drop. 

	- détacher les volets de la fenêtre si la propriété [VoletDétachable](../Proprietes/1000021382.md) est à <u><u><u><u>Vrai</u></u></u></u>.

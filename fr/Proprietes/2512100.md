


## Accès (Propriété)

***En anglais : Access***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Accès** permet de gérer le type d'accès d'une connexion (HFSQL Client/Serveur, OLE DB, ...). Il est possible de :

- Connaître le type d'accès utilisé pour une connexion (HFSQL Client/Serveur, OLE DB, ...).

- Définir le type d'accès utilisé pour la connexion  (HFSQL Client/Serveur, OLE DB, ...). Dans ce cas, si la connexion est actuellement ouverte, elle sera fermée, puis ré-ouverte avec les nouveaux paramètres lors d'un premier accès à un fichier de données utilisant cette connexion.




**Remarque** : La connexion peut correspondre :

- soit à une connexion décrite sous l'éditeur d'analyses,

- soit à une connexion décrite avec les fonctions [HConnecte](../WDLang4/3044263.md), [HDécritConnexion](../WDLang4/3044205.md) ou [HOuvreConnexion](../WDLang4/3044107.md).

- soit à une connexion décrite avec une variable de type [Connexion](../WDLang4/1514073.md).







<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MaConnexion est une Connexion
// Description de la connexion
MaConnexion.Utilisateur = "USER"
MaConnexion.MotDePasse = "PASSWORD"
MaConnexion.Serveur = "MONSERVEUR" 
MaConnexion.BaseDeDonnées = "Base de données"
MaConnexion.Provider = hAccèsHFClientServeur
MaConnexion.Accès = hOLectureEcriture
MaConnexion.InfosEtendues = "Infos étendues"
MaConnexion.OptionsCurseur = hCurseurClient

HOuvreConnexion(MaConnexion)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le type d'accès d'une connexion

`<Type d'accès> = <Connexion>.Accès`
---

**`<Type d'accès> : (Constante de type Entier)`**

Renvoie le type d'accès de la connexion : 


|   |   |
| --- | --- |
| *hOLecture* | Les fichiers de données manipulés par la connexion sont accessibles uniquement en lecture. Seule la lecture des enregistrements est possible. Il n'est pas possible d'en ajouter ou d'en supprimer.<br>![OLE DB](https://doc.pcsoft.fr/ext/images/fr/OLEDB.png) Cette constante est disponible uniquement lors d'une connexion OLE DB. |
| *hOLectureEcriture* | Les fichiers de données manipulés par la connexion sont accessibles en lecture et en écriture. Il est possible aussi bien de lire des enregistrements que d'écrire de nouveaux enregistrements. |



**`<Connexion> : (Chaîne de caractères ou variable de type Connexion)`**

Nom de la connexion manipulée. Ce nom a été défini :

- soit sous l'éditeur d'analyses,

- soit avec les fonctions [HDécritConnexion](../WDLang4/3044205.md) ou [HOuvreConnexion](../WDLang4/3044107.md),

- soit avec le type [Connexion](../WDLang4/1514073.md).





<a name="SYNTAXE2"></a>

### Définir le type d'accès d'une connexion

`<Connexion>.Accès = <Type d'accès>`
---

**`<Connexion> : (Chaîne de caractères ou variable de type Connexion)`**

Nom de la connexion manipulée. Ce nom a été défini :

- soit sous l'éditeur d'analyses,

- soit avec les fonctions [HDécritConnexion](../WDLang4/3044205.md) ou [HOuvreConnexion](../WDLang4/3044107.md),

- soit avec le type [Connexion](../WDLang4/1514073.md).




**`<Type d'accès> : (Constante de type Entier)`**

Définit le type d'accès de la connexion : 


|   |   |
| --- | --- |
| *hOLecture* | Les fichiers de données manipulés par la connexion sont accessibles uniquement en lecture. Seule la lecture des enregistrements est possible. Il n'est pas possible d'en ajouter ou d'en supprimer.<br>![OLE DB](https://doc.pcsoft.fr/ext/images/fr/OLEDB.png) Cette constante est disponible uniquement lors d'une connexion OLE DB. Elle est ignorée dans les autres cas. |
| *hOLectureEcriture* | Les fichiers de données manipulés par la connexion sont accessibles en lecture et en écriture. Il est possible aussi bien de lire des enregistrements que d'écrire de nouveaux enregistrements. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
La propriété [CxAccès](../Proprietes/2512015.md) permet de définir le type d'accès sur un fichier de données manipulé par une connexion. Cette propriété est conservée par compatibilité. Il est conseillé d'utiliser la propriété **Accès**.

Attention : La propriété **Accès** s'applique uniquement sur un type "Connexion". La propriété [CxAccès](../Proprietes/2512015.md) s'applique sur un fichier de données.

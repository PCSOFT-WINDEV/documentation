
## MotClé (Propriété)

***En anglais : Keyword***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **MotClé** permet de connaître et de modifier un mot-clé. Cette propriété est utilisable sur :

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [docPropriété](../WDLang1/1000022945.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [pdfDocument](../WDLang6/1000024909.md).

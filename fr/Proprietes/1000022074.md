


## CouleurTexteAutomatique (Propriété)

***En anglais : AutomaticTextColor***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **CouleurTexteAutomatique** permet de connaître et de modifier la couleur à utiliser pour tous les libellés des éléments utilisant la couleur "automatique". 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Passer en inverse vidéo 
FEN_Présentation.CouleurFond = Noir
FEN_Présentation.CouleurTexteAutomatique = Blanc
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître la couleur associée à la couleur automatique

`<Résultat> = <Elément manipulé>.CouleurTexteAutomatique`
---

**`<Résultat> : (Entier ou constante de type Entier)`**

Couleur associée à la couleur automatique. Cette couleur peut correspondre à :

- une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)),

- [une couleur prédéfinie](../WDLang5/3010002.md).




**`<Elément manipulé> : (Nom de fenêtre ou nom de champ)`**

Nom de l'élément à manipuler : fenêtre, fenêtre interne, superchamp, champ Onglet, champ Zone multiligne, champ Ruban. 


<a name="SYNTAXE2"></a>

### Modifier la couleur associée à la couleur automatique

`<Elément manipulé>.CouleurTexteAutomatique = <Nouvelle couleur>`
---

**`<Elément manipulé> : (Nom de fenêtre ou nom de champ)`**

Nom de l'élément à manipuler : fenêtre, fenêtre interne, superchamp, champ Onglet, champ Zone multiligne, champ Ruban. 

**`<Nouvelle couleur> : (Entier ou constante de type Entier)`**

Couleur associée à la couleur automatique. Cette couleur peut correspondre à :

- une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)),

- [une couleur prédéfinie](../WDLang5/3010002.md).






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Lors de la modification de la couleur automatique, tous les éléments utilisant la couleur "automatique" dans leur style sont modifiés. Les éléments n'utilisant pas cette couleur ne sont pas modifiés.

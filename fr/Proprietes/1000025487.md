
## Géodésique (Propriété)

***En anglais : Geodesic***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Géodésique** permet d'indiquer si le dessin doit suivre la courbure de la Terre. Cette propriété est utilisable sur : 

- une variable de type [CartePolygone](../WDLang3/1000025476.md).

- une variable de type [CartePolyligne](../WDLang3/1000025489.md).

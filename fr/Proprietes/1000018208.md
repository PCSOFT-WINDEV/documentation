
## Principal (Propriété)

***En anglais : Primary***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Principal** peut être utilisée :

- sur le type [gglAdressePostale](../WDLang5/1000017575.md).

- sur le type [gglEmail](../WDLang5/1000017571.md).

- sur le type [gglMessagerieInstantanée](../WDLang5/1000017569.md).

- sur le type [gglSociété](../WDLang5/1000017756.md).

- sur le type [gglTéléphone](../WDLang5/1000017573.md).


## LectureSeule (Propriété)

***En anglais : ReadOnly***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **LectureSeule** permet de savoir si un élément est en lecture seulement. Cette propriété est utilisable sur :

- une variable de type [NFCTag](../WDLang3/1000020411.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [Document](../WDLang1/1000022461.md).

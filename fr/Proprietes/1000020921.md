
## DéplacementTâche (Propriété)

***En anglais : MovementTask***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **DéplacementTâche** permet de : 

- savoir si le déplacement des tâches par l'utilisateur est autorisé dans une colonne de type Diagramme de Gantt. 

- autoriser ou non le déplacement des tâches par l'utilisateur dans une colonne de type Diagramme de Gantt.



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI INT_Modif ALORS
	COL_Gantt.DéplacementTâche = Vrai
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Savoir si le déplacement des tâches dans un Diagramme de Gantt est autorisé ou non 

`<Résultat> = <Colonne Gantt>.DéplacementTâche`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le déplacement des tâches est autorisé. 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.  




**`<Colonne Gantt> : (Nom de champ)`**

Nom de la colonne (d'un champ Table ou Table hiérarchique) de type Diagramme de Gantt à manipuler.


<a name="SYNTAXE2"></a>

### Autoriser ou interdire le déplacement des tâches dans un Diagramme de Gantt

`<Colonne Gantt>.DéplacementTâche = <Nouveau mode>`
---

**`<Colonne Gantt> : (Nom de champ)`**

Nom de la colonne (d'un champ Table ou Table hiérarchique) de type Diagramme de Gantt à manipuler.

**`<Nouveau mode> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> pour autoriser le déplacement des tâches par l'utilisateur, 

- <u><u><u><u>Faux</u></u></u></u> pour ne pas autoriser le déplacement des tâches par l'utilisateur.

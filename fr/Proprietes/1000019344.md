


## Icone (Propriété)

***En anglais : Icon***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Icone** permet de :

- Connaître l'icône affichée dans la barre de titre d'une fenêtre ou d'un champ Panneau dockable. 

- Modifier l'icône de la barre de titre d'une fenêtre ou d'un champ Panneau dockable.



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connaître l'icône affichée dans la fenêtre "FEN_SaisieClient"
ResIcône = FEN_SaisieClient.Icone
```



```wl
// Modifier l'icône affichée dans la fenêtre "FEN_SaisieClient"

// En utilisant un fichier ".ico" : 
FEN_SaisieClient.Icone = "MonIcone.ico"

// En utilisant un champ Image :
FEN_SaisieClient.Icone = dSauveImagePNG(IMG_Image1, enMémoire)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Récupérer l'icône d'une fenêtre

`<Résultat> = <Elément manipulé>.Icone`
---

**`<Résultat> : (Chaîne de caractères)`**

Icône de l'élément spécifié. Le nom de l'icône est retourné sans son chemin.

**`<Elément manipulé> : (Nom de champ ou nom de fenêtre)`**



- Nom de la fenêtre à manipuler.

- Nom du champ Panneau dockable à manipuler. 





<a name="SYNTAXE2"></a>

### Modifier l'icône d'une fenêtre

`<Elément manipulé>.Icone = <Nouvelle valeur>`
---

**`<Elément manipulé> : (Nom de champ ou nom de fenêtre)`**



- Nom de la fenêtre à manipuler.

- Nom du champ Panneau dockable à manipuler. 




**`<Nouvelle valeur> : (Chaîne de caractères ou variable de type Buffer)`**

Correspond : 

- Nom et chemin (complet, relatif ou UNC) de la nouvelle icône. Cette icône doit être au format ".ico".
	Remarques : 

	- Si le chemin n'est pas précisé, le fichier icône sera recherché dans le répertoire de l'exécutable.

	- Si le fichier icône n'existe pas, l'icône actuelle sera conservée.




- Chaîne d'octets renvoyée par l'une des fonctions suivantes :

	- [dSauveImageBMP](../WDLang1/3029026.md).

	- [dSauveImageGIF](../WDLang1/3029040.md).

	- [dSauveImagePNG](../WDLang1/3029027.md).

	- [dSauveImageJPEG](../WDLang1/3029038.md).









<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- L'icône de la première fenêtre du projet se retrouve :

	- Dans la barre des tâches de Windows.

	- Dans la fenêtre de navigation lors de l'utilisation de la combinaison de touches Alt + Tab.




- La modification d'une icône peut être effectuée uniquement sur les fenêtres ouvertes. Si la fenêtre est fermée, une erreur WLangage sera générée.




## NuméroAide (Propriété)

***En anglais : HelpNumber***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **NuméroAide** permet de connaître et de modifier le numéro d'aide associé au champ. Ce numéro correspond au champ "Numéro d'aide" présent dans l'onglet "Aide" de la fenêtre de description du champ.

Ce numéro d'aide correspond au numéro de la page d'aide à afficher lorsque l'utilisateur demande l'aide contextuelle du champ (soit en appuyant sur la touche F1, soit en cliquant sur le "?" dans la barre de menu puis sur le champ).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI Mode = "Ajouter" ALORS
	// Afficher l'aide sur "Ajouter"
	BTN_Bouton1.NuméroAide = 12
SINON
	// Afficher l'aide sur "Modifier"
	BTN_Bouton1.NuméroAide = 48
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le numéro d'aide d'un champ

`<Numéro d'aide> = <Champ manipulé>.NuméroAide`
---

**`<Numéro d'aide> : (Entier)`**

Numéro d'aide associé au champ.

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ manipulé.  


<a name="SYNTAXE2"></a>

### Modifier le numéro d'aide d'un champ

`<Champ manipulé>.NuméroAide = <Nouveau numéro d'aide>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ manipulé.

**`<Nouveau numéro d'aide> : (Entier)`**

Nouveau numéro d'aide associé au champ.

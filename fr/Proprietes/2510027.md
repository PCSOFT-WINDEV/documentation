


## DoubleClic (Propriété)

***En anglais : DoubleClick***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **DoubleClic** permet de connaître et de modifier le nom du champ Bouton qui sera exécuté lors d'un double-clic sur un champ de type Liste, Liste image, Table, Zone répétée, Interrupteur, Sélecteur ou Arbre. 

Par défaut, il est possible de définir le nom du champ Bouton exécuté lors d'un double-clic sous l'éditeur de fenêtres, lors de la description du champ (onglet "Détail").


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Changer le champ Bouton associé au double-clic sur le champ Liste "LISTE_Carte"
LISTE_Carte.DoubleClic = "BTN_Assistant"
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le champ Bouton associé au double-clic

`<Champ Bouton> = <Champ manipulé>.DoubleClic`
---

**`<Champ Bouton> : (Nom de champ)`**



- Nom du champ Bouton dont le code sera exécuté lors d'un double-clic sur le champ spécifié. 

- Chaîne vide ("") si aucun champ Bouton n'est associé au double-clic.




**`<Champ manipulé> : (Nom de champ)`**

Nom du champ (Liste, Liste image, Arbre, Zone répétée, Interrupteur, Sélecteur ou Table) pour lequel on veut connaître le nom du bouton associé au double-clic.  


<a name="SYNTAXE2"></a>

### Modifier le champ Bouton associé au double-clic

`<Champ manipulé>.DoubleClic = <Nouveau champ Bouton>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ (Liste, Liste image, Arbre, Zone répétée, Interrupteur, Sélecteur ou Table) pour lequel on veut modifier le nom du bouton associé au double-clic.

**`<Nouveau champ Bouton> : (Nom de champ)`**



- Nom du champ Bouton dont le code sera exécuté lors d'un double-clic sur le champ spécifié, 

- Chaîne vide ("") si aucun champ Bouton n'est associé au double-clic.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
La propriété **DoubleClic** s'applique uniquement aux champs suivants :

- champs Arbre,

- champs Liste et Liste image,

- champs Zone répétée,

- champs Table et Table hiérarchique,

- champs Interrupteur, 

- champs Sélecteur,

- champs Calendrier, 

- champs TreeMap,

- champs Agenda,

- champs Planning.




![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) La propriété **DoubleClic** s'applique uniquement aux champs Liste et aux champs Arbre.

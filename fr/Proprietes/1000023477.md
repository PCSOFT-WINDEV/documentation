
## NombrePageAccessible (Propriété)

***En anglais : NumberAccessiblePages***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **NombrePageAccessible** permet de connaître le nombre de pages actuellement chargées dans le champ Lecteur PDF ou dans le champ Traitement de texte.

Lors de l'ouverture d'un fichier PDF avec un nombre de pages important, le champ Lecteur PDF charge chaque page successivement : l'ensemble des pages n'est pas immédiatement accessible. Cette propriété permet de connaître le nombre de pages du PDF actuellement chargées sans attendre la fin du chargement.

Le comportement est identique pour un fichier DOCX ouvert par un champ Traitement de texte. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Récupère le nombre de pages actuellement chargées dans le champ Lecteur PDF
Trace(PDF_Lecteur.NombrePageAccessible)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Champ manipulé>.NombrePageAccessible`
---

**`<Résultat> : (Entier)`**

Nombre de pages actuellement chargées dans le champ Lecteur PDF ou dans le champ Traitement de texte. 

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler : 

- Champ Lecteur PDF, 

- Champ Traitement de texte. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
La propriété [NombrePage](../Proprietes/2510084.md) attend la fin du chargement du fichier PDF ou DOCX par le champ et renvoie le nombre de pages totales du fichier. Cette propriété est bloquante.

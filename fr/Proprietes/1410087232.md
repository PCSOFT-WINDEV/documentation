


## VisibleAvecAnimation (Propriété)

***En anglais : VisibleWithAnimation***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **VisibleAvecAnimation** permet de rendre un champ visible ou invisible en jouant l'animation "Apparition/Disparition". Cette animation est définie : 

- soit dans l'onglet "Avancé" de la fenêtre de description du projet (bouton "Animation des champs"). 

- soit dans l'onglet "Style" du champ ou du projet. 



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Rend le champ Bouton APPLIQUER visible
BTN_Appliquer.VisibleAvecAnimation = Vrai
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE2"></a>

`<Champ manipulé>.VisibleAvecAnimation = <Visible>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ, du groupe de champs ou de la cellule d'un champ Disposition à manipuler. 

**`<Visible> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> pour rendre le champ visible avec l'animation "Apparition/Disparition", 

- <u><u><u><u>Faux</u></u></u></u> pour rendre le champ invisible avec l'animation "Apparition/Disparition". 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Les propriétés de l'animation (durée par exemple) peuvent être décrites au niveau du projet et surchargées dans le style du champ. 

- Cette propriété est disponible uniquement pour les champs, les groupes de champs et les cellules du champ Disposition.

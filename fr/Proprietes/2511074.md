


## BasPageCollé (Propriété)

***En anglais : PageEndStick***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **BasPageCollé** permet de gérer la position du bloc de bas de page. Le bloc de bas de page peut être :

- collé au bloc précédent, le bas de page n'étant pas obligatoirement situé au bas de la page imprimée. Il est ainsi possible d'utiliser le bas de page pour réaliser des calculs et les afficher au bas du bloc affichant le tableau.

- non collé au bloc précédent, le bloc de bas de page sera imprimé à sa place habituelle (en bas de page). C'est la gestion par défaut du bas de page.




La propriété **BasPageCollé** permet de :

- Savoir si le bas de page doit être collé au dernier bloc avant le bas de page.

- Modifier la gestion du bloc de pas de page.







<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connaître le mode de gestion du bas de page
SI MonEtat.BasPageCollé = Vrai ALORS
	Trace("Bas de page collé au bloc précédent")
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le mode de gestion du bas de page

`<Gestion actuelle> = <Etat manipulé>.BasPageCollé`
---

**`<Gestion actuelle> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le bas de page est collé au bloc précédent, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Etat manipulé> : (Nom d'état)`**

Nom de l'état manipulé.  


<a name="SYNTAXE2"></a>

### Modifier le mode de gestion du bas de page

`<Etat manipulé>.BasPageCollé = <Nouvelle gestion>`
---

**`<Etat manipulé> : (Nom d'état)`**

Nom de l'état manipulé.

**`<Nouvelle gestion> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le bas de page doit être collé au bloc précédent, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
La propriété **BasPageCollé** peut être utilisée dans n'importe quel code de l'état, en lecture comme en écriture.


## PanneauModifieurVisible (Propriété)

***En anglais : ModifierPanelVisible***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **PanneauModifieurVisible** permet de : 

- savoir si le panneau "Modifieur" est affichée dans un champ Editeur de diagrammes.  

- afficher ou non le panneau "Modifieur" dans un champ Editeur de diagrammes.




Rappel : Le panneau "Modifieur" d'un champ Editeur de diagrammes correspond au panneau permettant de modifier l'élément sélectionné (couleur, épaisseur du trait, etc.). Ce panneau est affiché à droite du champ Editeur de diagrammes.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Le champ Interrupteur INT_Options propose les options suivantes : 
// Bibliothèque
// Styles
// Barre d'outils
// Grille
EDIAG_Diagramme.PanneauBibliothèqueVisible = INT_Options[1]
EDIAG_Diagramme.PanneauModifieurVisible = INT_Options[2]
EDIAG_Diagramme.BarreOutilsVisible = INT_Options[3]
EDIAG_Diagramme.QuadrillageVisible = INT_Options[4]
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Savoir si le panneau "Modifieur" est affiché

`<Résultat> = <Champ Editeur de diagrammes>.PanneauModifieurVisible`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le panneau "Modifieur" est visible,

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 




**`<Champ Editeur de diagrammes> : (Nom de champ)`**

Nom du champ Editeur de diagrammes manipulé. 


<a name="SYNTAXE2"></a>

### Afficher ou non le panneau "Modifieur"

`<Champ Editeur de diagrammes>.PanneauModifieurVisible = <Affichage>`
---

**`<Champ Editeur de diagrammes> : (Nom de champ)`**

Nom du champ Editeur de diagrammes manipulé. 

**`<Affichage> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> pour rendre le panneau "Modifieur" visible,

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.


## AffichageSignets (Propriété)

***En anglais : DisplayBookmarks***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **AffichageSignets** permet : 

- de connaître le mode d'affichage des signets dans un champ Traitement de texte. 

- d'activer ou de désactiver l'affichage des signets dans un champ Traitement de texte.



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Active l'affichage des signets
TT_MonDocument.AffichageSignets = Vrai
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître le mode d'affichage des signets

`<Résultat> = <Champ Traitement de texte>.AffichageSignets`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le champ affiche les signets, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire




**`<Champ Traitement de texte> : (Nom de champ)`**

Nom du champ Traitement de texte à manipuler. 


<a name="SYNTAXE2"></a>

### Activer ou de désactiver l'affichage des signets

`<Champ Traitement de texte>.AffichageSignets = <Mode d'affichage>`
---

**`<Champ Traitement de texte> : (Nom de champ)`**

Nom du champ Traitement de texte à manipuler.

**`<Mode d'affichage> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> pour afficher les signets dans le champ, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.

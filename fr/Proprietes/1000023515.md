
## VoletVisible (Propriété)

***En anglais : PaneVisible***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **VoletVisible** permet de : 

- savoir si le volet du champ Editeur d'images est visible ou non, 

- rendre visible ou invisible le volet du champ Editeur d'images.



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Ne pas afficher le volet dans l'éditeur
EI_Editeur.VoletVisible = Faux
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Savoir si le volet du champ Editeur d'images est visible

`<Résultat> = <Champ Editeur d'images>.VoletVisible`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le volet du champ est visible, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 




**`<Champ Editeur d'images> : (Nom de champ)`**

Nom du champ Editeur d'images à manipuler.  


<a name="SYNTAXE2"></a>

### Rendre le volet du champ Editeur d'images visible ou invisible

`<Champ Editeur d'images>.VoletVisible = <Visibilité>`
---

**`<Champ Editeur d'images> : (Nom de champ)`**

Nom du champ Editeur d'images à manipuler.

**`<Visibilité> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le volet du champ doit être visible, 

- <u><u><u><u>Faux</u></u></u></u> si le volet du champ doit être invisible.

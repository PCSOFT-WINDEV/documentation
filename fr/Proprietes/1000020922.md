


## LibelléHTML (Propriété)

***En anglais : CaptionHTML***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **LibelléHTML** permet de manipuler le libellé d'un champ sous forme de code HTML.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
LIEN_CGV.LibelléHTML = "Afficher les <strong>conditions générales de vente</strong>"
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Récupérer la valeur du libellé du champ

`<Résultat> = <Champ à manipuler>.LibelléHTML`
---

**`<Résultat> : (Chaîne de caractères)`**

Libellé du champ au format HTML.

**`<Champ à manipuler> : (Nom de champ)`**

Nom du champ à manipuler : Zone de Texte Riche, Bouton, ...


<a name="SYNTAXE2"></a>

### Modifier la valeur du libellé du champ

`<Champ à manipuler>.LibelléHTML = <Nouvelle valeur>`
---

**`<Champ à manipuler> : (Nom de champ)`**

Nom du champ à manipuler : Zone de Texte Riche, Bouton, ...

**`<Nouvelle valeur> : (Chaîne de caractères)`**

Nouveau libellé du champ. Cette valeur doit correspondre à un code HTML valide.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Libellé et libellé HTML
<a name="libelle_libelle_html_ELTPARAGRAPHE000049"></a>Un champ possède un libellé ou un libellé HTML : il ne peut pas posséder les deux à la fois. L'affectation de la propriété **LibelléHTML** supprime la valeur du libellé affectée avec la propriété [Libellé](../Proprietes/2510053.md) et réciproquement.

Un accès en lecture à la propriété [Libellé](../Proprietes/2510053.md) alors que le libellé du champ est un libellé HTML déclenche une erreur. En revanche, la lecture de la propriété **LibelléHTML** alors que le libellé du champ est affectée avec la propriété [Libellé](../Proprietes/2510053.md) retourne la valeur du libellé encodée en HTML. 

Tableau récapitulatif : 


|   |   |   |
| --- | --- | --- |
|   | Propriété affectée dans le champ : [Libellé](../Proprietes/2510053.md) | Propriété affectée dans le champ : **LibelléHTML** |
| Propriété lue : [Libellé](../Proprietes/2510053.md) | Retourne la valeur. | Erreur. |
| Propriété lue : **LibelléHTML** | Retourne la valeur encodée en HTML. | Retourne la valeur. |




### Validité du code HTML
<a name="validite_code_html_ELTPARAGRAPHE000100"></a>Aucune vérification n'est effectuée sur la validité du HTML. Si une valeur HTML invalide est affectée dans le champ, la page sera probablement mal affichée par le navigateur. Il faut faire en particulier attention :

- Aux balises (et attributs) mal fermées ou invalides pour la norme HTML de la page (HTML 4, HTML 5, ...). 

- A l'encodage des caractères spéciaux. Il est possible d'utiliser la fonction [TexteVersHTML](../WDLang1/3025050.md).





### Limitation
<a name="limitation_ELTPARAGRAPHE000111"></a>Cette propriété est disponible sur les champs suivants : 

- Champ Bouton, 

- Champ Lien,

- Champ Zone de Texte Riche,

- Champ de saisie,

- Champ Interrupteur,

- Champ Sélecteur,

- Champ Liste,

- Champ d'affichage formaté,

- Champ Combo,

- Option d'un champ Interrupteur,

- Option d'un champ Sélecteur, 

- Option de menu.

- Colonne de table.

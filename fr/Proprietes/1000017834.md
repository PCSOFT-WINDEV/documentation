


## Région (Propriété)

***En anglais : Region***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Région** permet de connaître et de modifier une région utilisée par le service publicitaire Google AdWords. Cette propriété est utilisable sur :

- une variable de type [sfInfoUtilisateur](../WDLang5/1000018423.md).

- une variable de type [Adresse](../WDLang3/1000019944.md).

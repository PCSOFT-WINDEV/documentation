


## TauxAncrageHauteur (Propriété)

***En anglais : AnchorRateHeight***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **TauxAncrageHauteur** permet de :

- Connaître le taux d'ancrage en hauteur d'un champ

- Modifier le taux d'ancrage en hauteur d'un champ.




Le taux d'ancrage permet de définir la manière dont le champ se déplace ou change de taille en fonction de l'évolution de la taille de la fenêtre (ou de l'état) à laquelle il est ancré. Le taux d'ancrage correspond au pourcentage de déplacement du champ par rapport à la fenêtre (ou à l'état). Ainsi, si le taux d'ancrage est de 0, le champ ne sera pas déplacé, si le taux d'ancrage est de 1000, l'ancrage du champ sera normal.

Le taux d'ancrage en hauteur (propriété **TauxAncrageHauteur**) est appliqué si le champ est ancré vers le haut (constante *ancreHauteur* utilisée pour la propriété [Ancrage](../Proprietes/2510001.md)).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Le champ Liste se déplacera de 50% de l'agrandissement de la fenêtre.
LISTE_ListePersonne.TauxAncrageHauteur = 500
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le taux d'ancrage d'un champ ancré en hauteur

`<Résultat> = <Champ manipulé>.TauxAncrageHauteur`
---

**`<Résultat> : (Entier)`**

Taux d'ancrage compris entre 0 (champ non étiré) et 1000 (ancrage normal)

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler. Ce champ peut être un champ : 

- d'une fenêtre.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) d'un état.





<a name="SYNTAXE2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) 
### Modifier le taux d'ancrage d'un champ ancré en hauteur

`<Champ manipulé>.TauxAncrageHauteur = <Taux d'ancrage>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler.

**`<Taux d'ancrage> : (Entier)`**

Nouveau taux d'ancrage compris entre 0 (champ non étiré) et 1000 (ancrage normal)

## Remarques


### Ancrage d'un champ
<a name="ancrage_champ_ELTPARAGRAPHE000104"></a>

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Dans une fenêtre, la propriété [Ancrage](../Proprietes/2510001.md) permet de définir le comportement du champ (dimensions et déplacement) lors du changement de taille de la fenêtre. Si le champ est associé à un champ Onglet, l'ancrage est défini par rapport à l'onglet. 
	Pour plus de détails, consultez [Ancrage des champs dans une fenêtre](../WDChamp/2026002.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Dans un état, la propriété [Ancrage](../Proprietes/2510001.md) permet de connaître le comportement du champ (dimensions et déplacement) lors du changement de taille de l'état. Ce comportement a été défini sous l'éditeur d'états. 
	Pour plus de détails, consultez [Ancrage des champs dans un état](../WDChamp/1013298.md). 



<a name="NOTE0_2"></a>


### Limites
<a name="limites_ELTPARAGRAPHE000134"></a>![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) La propriété **TauxAncrageHauteur** s'applique à tous les champs d'une fenêtre.

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) La propriété **TauxAncrageHauteur** s'applique à tous les champs d'un état. Dans ce cas, il est uniquement possible de connaître le taux d'ancrage utilisé.

![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) La propriété **TauxAncrageHauteur** s'applique uniquement aux éléments suivants :

- Bouton.

- Libellé.

- Champ de saisie.

- Image.

- Interrupteur.

- Sélecteur.

- Liste.

- Combo.

- Arbre.

- Onglet.

- Table.

- Superchamp.

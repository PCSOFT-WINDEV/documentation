


## Bulle (Propriété)

***En anglais : ToolTip***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Bulle** permet de connaître et de modifier le texte d'une bulle d'aide. Cette propriété est utilisable sur :

- les champs d'une fenêtre ou d'une page. Pour plus de détails, consultez [Bulle](../Proprietes/2510005.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [EvénementTimeline](../WDLang1/1000020612.md).

- une variable de type [RendezVous](../WDLang1/1000019244.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [OrgaElément](../WDLang1/1000019713.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [TâcheGantt](../WDLang1/1000020881.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [docElémentLien](../WDLang1/1000022970.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [Jeton](../WDLang1/1000023942.md).

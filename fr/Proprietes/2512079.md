


## RubriqueOrigine (Propriété)

***En anglais : InitialItem***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **RubriqueOrigine** permet de connaître le nom original d'une rubrique présente dans une requête. Ce nom est connu à partir de l'alias de cette rubrique.

**Remarque** : La propriété [FichierOrigine](../Proprietes/2512029.md) permet de connaître le nom original du fichier de données associé à une rubrique présente dans une requête. Ce nom est connu à partir de l'alias de cette rubrique.






<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
HExécuteRequêteSQL("RequêteExemple", "SELECT Client.NomCli AS MonSuperClient FROM Client")
Trace(RequêteExemple.MonSuperClient.FichierOrigine) // Client
Trace(RequêteExemple.MonSuperClient.RubriqueOrigine) // NomCli
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Requête>.<Alias de la rubrique>.RubriqueOrigine`
---

**`<Résultat> : (Chaîne de caractères)`**



- Nom original de la rubrique associé à l'alias de la rubrique, 

- Chaîne vide ("") si l'alias de la rubrique est un agrégat ou correspond à un compteur.




**`<Requête> : (Chaîne de caractères)`**

Nom de la requête contenant la rubrique. Ce paramètre peut également correspondre à un nom de fichier de données HFSQL, un nom de vue HFSQL, ...

**`<Alias de la rubrique> : (Chaîne de caractères)`**

Nom de la rubrique dans le résultat de la requête (ou dans le fichier de données, la vue, ...).

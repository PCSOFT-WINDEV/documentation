
## Numérotation

***En anglais : Numbering***
	

<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Numérotation** permet de décrire la numérotation d'un élément. Cette propriété est utilisable : 

- sur une variable de type [docParagraphe](../WDLang1/1000022483.md).

- sur une variable de type [Document](../WDLang1/1000022461.md).

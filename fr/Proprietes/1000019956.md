


## ModeCarte (Propriété)

***En anglais : MapMode***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **ModeCarte** permet de connaître et de modifier le mode d'affichage de la carte dans le champ Carte. Cette carte peut être affichée en mode plan ou en mode satellite. 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=ModeCarte.gif)



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Modifier le mode d'affichage du champ Carte
CARTE_Position.ModeCarte = carteModeSatellite
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le mode d'affichage d'un champ Carte

`<Résultat> = <Champ Carte>.ModeCarte`
---

**`<Résultat> : (Constante de type Entier)`**

Mode d'affichage actuel de la carte dans le champ Carte. Ce paramètre peut correspondre à une des constantes suivantes :


|   |   |
| --- | --- |
| *carteModePlan* | La carte affichée dans le champ Carte est en mode plan. |
| *carteModeSatellite* | La carte affichée dans le champ Carte est en mode satellite. |



**`<Champ Carte> : (Nom de champ)`**

Nom du champ Carte à manipuler.  


<a name="SYNTAXE2"></a>

### Modifier le mode d'affichage d'un champ Carte

`<Champ Carte>.ModeCarte = <Nouveau mode d'affichage>`
---

**`<Champ Carte> : (Nom de champ)`**

Nom du champ Carte à manipuler.

**`<Nouveau mode d'affichage> : (Constante de type Entier)`**

Nouveau mode d'affichage de la carte dans le champ Carte. Ce paramètre peut correspondre à une des constantes suivantes :


|   |   |
| --- | --- |
| *carteModePlan* | La carte affichée dans le champ Carte est en mode plan. |
| *carteModeSatellite* | La carte affichée dans le champ Carte est en mode satellite. |


## Interligne (Propriété)

***En anglais : Interline***
	



<a name="description"></a>
La propriété **Interligne** permet de gérer l'interligne d'un élément. Cette propriété est utilisable sur :

- une variable de type [docParagraphe](../WDLang1/1000022483.md) (caractéristiques de mise en page).

- une variable de type [docNiveauNumérotation](../WDLang1/1000022799.md).




## FormeExtrémitéDépart (Propriété)

***En anglais : ShapeExtremityStart***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **FormeExtrémitéDépart** permet de connaître et de modifier le type de forme utilisé pour l'extrémité de départ d'un champ Fil.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Création et initialisation d'un champ Fil 
// Ce fil relie le champ BTN_ChampA et le champ BTN_ChampB
NomFil est un Champ

NomFil <- ChampCrée("", typFil)
NomFil.ChampDépart = BTN_ChampA
NomFil.ChampArrivée = BTN_ChampB
NomFil.FormeExtrémitéDépart = formeLosange
NomFil.FormeExtrémitéArrivée = formeRond
NomFil.FormeExtrémitéTaille = 8
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître le type de forme utilisé pour l'extrémité de départ d'un champ Fil

`<Résultat> = <Champ Fil>.FormeExtrémitéDépart`
---

**`<Résultat> : (Constante de type Entier)`**

Type de forme utilisé : 


|   |   |
| --- | --- |
| *formeAucune* | Aucune forme. |
| *formeCarré* | Forme de l'extrémité de départ : Carré. |
| *formeFlèche* | Forme de l'extrémité de départ : Flèche. |
| *formeLosange* | Forme de l'extrémité de départ : Losange. |
| *formeRond* | Forme de l'extrémité de départ : Rond. |
| *formeTriangle* | Forme de l'extrémité de départ : Triangle. |



**`<Champ Fil> : (Nom de champ)`**

Nom du champ Fil à manipuler. 


<a name="SYNTAXE2"></a>

### Modifier le type de forme utilisé pour l'extrémité de départ d'un champ Fil

`<Champ Fil>.FormeExtrémitéDépart = <Nouvelle forme>`
---

**`<Champ Fil> : (Nom de champ)`**

Nom du champ Fil à manipuler. 

**`<Nouvelle forme> : (Constante de type Entier)`**

Nouvelle forme à utiliser au départ du champ Fil : 


|   |   |
| --- | --- |
| *formeAucune* | Aucune forme. |
| *formeCarré* | Forme de l'extrémité de départ : Carré. |
| *formeFlèche* | Forme de l'extrémité de départ : Flèche. |
| *formeLosange* | Forme de l'extrémité de départ : Losange. |
| *formeRond* | Forme de l'extrémité de départ : Rond. |
| *formeTriangle* | Forme de l'extrémité de départ : Triangle. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- La couleur de la forme au départ et à l'arrivée du champ Fil est fixée par la propriété [Couleur](../Proprietes/2510071.md).

- La taille de la forme au départ et à l'arrivée du champ Fil est fixée par la propriété [FormeExtrémitéTaille](../Proprietes/1000021960.md).

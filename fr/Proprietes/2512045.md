


## Format (Propriété)

***En anglais : Format***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Format** permet de paramétrer le format des données du fichier de données. Il est ainsi possible de :

- Connaître le format des données utilisé pour un fichier de données (fichier défini sous l'éditeur d'analyses ou défini par programmation).

- Définir le format des données d'un fichier de données (lors de la description du fichier par programmation uniquement).







<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Définir le format des données
Client.Format = hFormatUnicode
...
// Valide la description du fichier de données
HDécritFichier(Client)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le format des données 

`<Format actuel> = <Fichier de données>.Format`
---

**`<Format actuel> : (Constante de type entier)`**

Un des formats disponibles : 


|   |   |
| --- | --- |
| *hFormatAnsi* | Format Ansi (utilisé par Windows). |
| *hFormatUnicode* | Format Unicode. |



**`<Fichier de données> : (Chaîne de caractères)`**

Nom du fichier de données manipulé. Ce nom a été défini sous l'éditeur d'analyses ou grâce au type [Description de Fichier](../WDLang4/1514065.md).  


<a name="SYNTAXE2"></a>

### Définir le format des données

`<Fichier de données>.Format = <Nouveau format>`
---

**`<Fichier de données> : (Chaîne de caractères)`**

Nom du fichier de données manipulé. Ce nom vient d'être défini grâce au type [Description de Fichier](../WDLang4/1514065.md).

**`<Nouveau format> : (Constante de type entier)`**

Un des format disponibles : 


|   |   |
| --- | --- |
| *hFormatAnsi* | Format Ansi (utilisé par Windows). |
| *hFormatUnicode* | Format Unicode. |

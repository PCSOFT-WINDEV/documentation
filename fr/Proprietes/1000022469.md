


## RendezVousContinu (Propriété)

***En anglais : AppointmentContinuous***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **RendezVousContinu** permet de connaître ou de modifier la méthode d'affichage des rendez-vous sur plusieurs jours dans un champ Planning.  

Rappel : Un rendez-vous étalé sur plusieurs jours dans un champ Planning peut être affiché : 

- soit comme un seul rendez-vous continu. 

- soit comme un rendez-vous par jour. 




Remarque : Cette propriété correspond à l'option "Affichage continu si plusieurs jours" présente dans la fenêtre de description des champs Planning. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
PLN_MonPlanning.RendezVousContinu = Vrai
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître le mode d'affichage des rendez-vous sur plusieurs jours

`<Résultat> = <Champ Planning>.RendezVousContinu`
---

**`<Résultat> : (Constante de type Entier)`**

Mode d'affichage : 


|   |   |
| --- | --- |
| *Défaut* | L'affichage des rendez-vous dépend des paramètres d'affichage du champ Planning. |
| *Faux* | Les rendez-vous sont affichés sur chaque jour. |
| *Vrai* | Les rendez-vous sont affichés en continu. |



**`<Champ Planning> : (Nom de champ)`**

Nom du champ Planning à manipuler.


<a name="SYNTAXE2"></a>

### Modifier le mode d'affichage des rendez-vous sur plusieurs jours

`<Champ Planning>.RendezVousContinu = <Nouveau mode d'affichage>`
---

**`<Champ Planning> : (Nom de champ)`**

Nom du champ Planning à manipuler.

**`<Nouveau mode d'affichage> : (Constante de type Entier)`**

Mode d'affichage à utiliser : 


|   |   |
| --- | --- |
| *Défaut* | L'affichage des rendez-vous dépend des paramètres d'affichage du champ Planning. |
| *Faux* | Les rendez-vous sont affichés sur chaque jour. |
| *Vrai* | Les rendez-vous sont affichés en continu. |

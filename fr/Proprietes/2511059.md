


## NbExemplaireEtiquette (Propriété)

***En anglais : NbLabelCopy***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **NbExemplaireEtiquette** permet de :

- Connaître le nombre d'exemplaires d'étiquettes identiques imprimés.

- Modifier le nombre d'exemplaires d'étiquettes identiques à imprimer.





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Imprimer 5 fois chaque étiquette présente dans l'état "ETAT_Etat1"
ETAT_Etat1.NbExemplaireEtiquette = 5
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le nombre d'exemplaires d'étiquettes identiques

`<Résultat> = <Etat manipulé>.NbExemplaireEtiquette`
---

**`<Résultat> : (Entier)`**

Nombre d'exemplaires d'étiquettes identiques à imprimer.

**`<Etat manipulé> : (Nom de l'état)`**

Nom de l'état "Étiquette" à manipuler.  


<a name="SYNTAXE2"></a>

### Modifier le nombre d'exemplaires d'étiquettes identiques

`<Etat manipulé>.NbExemplaireEtiquette = <Nombre d'exemplaires>`
---

**`<Etat manipulé> : (Nom de l'état)`**

Nom de l'état de type "Étiquette" à manipuler.

**`<Nombre d'exemplaires> : (Entier)`**

Nombre d'exemplaires d'étiquettes identiques à imprimer.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### File d'attente de l'imprimante
<a name="file_attente_imprimante_ELTPARAGRAPHE000058"></a>L'utilisation de la propriété **NbExemplaireEtiquette** augmente la file d'attente (appelée également spool) de l'imprimante utilisée.

Par exemple, si les 5 étiquettes présentes dans un état sont imprimées 10 fois chacune, la file d'attente de l'imprimante contiendra un état de 50 étiquettes.

Pour éviter de saturer la file d'attente de l'imprimante utilisée, il est préférable d'imprimer plusieurs fois le même état (propriété [NbCopie](../Proprietes/2511042.md)).

Ainsi, dans notre exemple, l'état sera imprimé 10 fois. La file d'attente de l'imprimante contiendra donc 10 états de 5 étiquettes.
<a name="NOTE0_2"></a>


### Limite
<a name="limite_ELTPARAGRAPHE000077"></a>La propriété **NbExemplaireEtiquette** s'applique uniquement aux états de type "Étiquette".

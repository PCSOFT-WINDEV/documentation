


## NbThreadAttente (Propriété)

***En anglais : NbPendingThread***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **NbThreadAttente** permet de connaître le nombre de threads en attente d'une file ou d'une pile. Cette propriété est utilisée lors d'une gestion multithreads.




<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Nombre de threads> = <Elément manipulé>.NbThreadAttente`
---

**`<Nombre de threads> : (Entier)`**

Nombre de threads en attente.

**`<Elément manipulé> : (File ou pile)`**

Nom de la variable de type [File](../WDLang1/1514021.md) ou de type [Pile](../WDLang1/1514022.md) manipulée.

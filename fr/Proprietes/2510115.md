
## Triée (Propriété)

***En anglais : Sorted***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Triée** permet de :

- Savoir si un champ est trié. 

- Trier ou non un champ. 




![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Cette propriété est disponible sur les champs Liste, Liste image et Combo par programmation.



**Remarques** : 

- L'option **Triée** permet de trier les éléments du champ Liste ou du champ Combo lors de son affichage. Si le champ est trié, les ajouts ([ListeAjoute](../WDLang1/3049004.md)) seront réalisés en tenant compte du tri (il s'agira en fait d'insertion d'éléments).

- Dans le cas des champs Liste image, le tri est réalisé en fonction des libellés des éléments.

- Cette option peut être choisie sous l'éditeur de fenêtres ou de pages lors de la description du champ (onglet "Détail", option "Liste triée").







<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
ListeAjoute(LISTE_Liste1, "CC")
ListeAjoute(LISTE_Liste1, "DD")
ListeAjoute(LISTE_Liste1, "BB")
LISTE_Liste1.Triée = Vrai
// L'ordre dans le champ Liste devient BB, CC, DD
ListeAjoute(LISTE_Liste1, "AA") // Sera inséré en début de liste
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Savoir si un champ est trié

`<Résultat> = <Champ manipulé>.Triée`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le champ spécifié est trié, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler : 

- Nom du champ Liste (y compris champ Liste image).

- Nom du champ Combo.





<a name="SYNTAXE2"></a>

### Trier ou non un champ

`<Champ manipulé>.Triée = <A trier>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à trier : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Nom du champ Liste (y compris un champ Liste image).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Nom du champ Combo.




**`<A trier> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le champ spécifié doit être trié, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Limites
<a name="limites_ELTPARAGRAPHE000127"></a>![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) La propriété **Triée** s'applique uniquement aux champs Liste par programmation (y compris champ Liste image) et aux champs Combo par programmation. La propriété **Triée** ne s'applique pas aux champs Liste de type Table ou aux champs Combo de type Table.

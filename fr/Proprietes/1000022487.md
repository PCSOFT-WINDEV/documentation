
## Style (Propriété)

***En anglais : Style***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Style** permet de : 

- récupérer ou d'affecter le style complet d'un champ (y compris les surcharges). Pour plus de détails, consultez l'aide de la propriété [Style](../Proprietes/1000021472.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) connaître les styles associés à un document de type Traitement de texte. Cette propriété est utilisable sur les variables de type [Document](../WDLang1/1000022461.md).

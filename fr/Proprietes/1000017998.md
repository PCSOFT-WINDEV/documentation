
## Contenu (Propriété)

***En anglais : Content***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Contenu** permet de connaître et de modifier le contenu d'un élément. Cette propriété est utilisable sur :

- les champs de type liste ou combo. Pour plus de détails, consultez [Propriété Contenu sur les champs](../Proprietes/1000017242.md).

- une variable de type [gglEvénement](../WDLang5/1000017440.md). 

- une variable de type [emailAttache](../WDLang3/1000018752.md).

- une variable de type [EvénementTimeline](../WDLang1/1000020612.md).

- une variable de type [zipFichierArchivé](../WDLang3/1000019044.md).

- une variable de type [RendezVous](../WDLang1/1000019244.md).

- une variable de type [CodeBarres](../WDLang5/1000019451.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [OrgaElément](../WDLang1/1000019713.md).

- une variable de type [Notification](../WDLang3/1000019441.md). 

- une variable de type [Enregistrement](../WDLang4/1000020962.md). 

- une variable de type [httpRequête](../WDLang3/1000021158.md).

- une variable de type [httpRéponse](../WDLang3/1000021165.md).

- une variable de type [restRequête](../WDLang3/1000021481.md).

- une variable de type [restRéponse](../WDLang3/1000021477.md).

- une variable de type [notificationFormat](../WDLang3/1000021301.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [docCellule](../WDLang1/1000022927.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [docElémentLien](../WDLang1/1000022970.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [docElémentZoneDeTexte](../WDLang1/1000023420.md).

- une variable de type [pdfPièceJointe](../WDLang6/1000024899.md).

- une variable de type [MotDePasseSécurisé](../Motscles/1000021064.md).

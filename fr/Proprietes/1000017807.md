


## Téléphone (Propriété)

***En anglais : PhoneNumber***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Téléphone** permet de connaître et de modifier le numéro de téléphone. Cette propriété est utilisable sur :

- une variable de type [gglContact](../WDLang5/1000017434.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [gpwUtilisateur](../WDLang6/1000021088.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [adminWBCompte](../WDLang2/1410089382.md).

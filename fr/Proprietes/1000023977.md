
## Jeton

***En anglais : Token***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le mot-clé **Jeton** peut correspondre :

- à la propriété [Jeton](../Proprietes/1000023976.md) qui permet de manipuler les différents jetons associés à un champ de saisie de type "Texte avec jetons".

- à une variable de type [Jeton](../WDLang1/1000023942.md) qui permet de manipuler un jeton associé à un champ de saisie de type "Texte avec jetons".

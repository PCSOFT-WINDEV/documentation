
## Affichage (Propriété)

***En anglais : Display***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Affichage** permet de connaître et de modifier la vidéo actuellement affichée par le champ Conférence. Le champ Conférence peut utiliser la vidéo entrante, la vidéo émise ou aucune vidéo.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Le champ affiche la vidéo entrante
CONF_Conférence1.Affichage = FluxAffichageRéception
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître la vidéo actuellement affichée

`<Résultat> = <Champ Conférence>.Affichage`
---

**`<Résultat> : (Constante de type entier)`**

Vidéo actuellement utilisée :


|   |   |
| --- | --- |
| *FluxAffichageAucun* | Aucune vidéo n'est affichée |
| *FluxAffichageEmission* | La vidéo affichée correspond à la vidéo émise par le poste en cours. |
| *FluxAffichageRéception* | La vidéo affichée correspond à la vidéo reçue sur le poste en cours. |



**`<Champ Conférence> : (Nom de champ)`**

Nom du champ Conférence associé au flux.  


<a name="SYNTAXE2"></a>

### Modifier la vidéo actuellement affichée

`<Champ Conférence>.Affichage = <Vidéo affichée>`
---

**`<Champ Conférence> : (Nom de champ)`**

Nom du champ Conférence associé au flux.

**`<Vidéo affichée> : (Constante de type Entier)`**

Nouvelle vidéo affichée :


|   |   |
| --- | --- |
| *FluxAffichageAucun* | Aucune vidéo n'est affichée |
| *FluxAffichageEmission* | La vidéo affichée correspond à la vidéo émise par le poste en cours. |
| *FluxAffichageRéception* | La vidéo affichée correspond à la vidéo reçue sur le poste en cours. |







<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- La propriété **Affichage** modifie ce qui est affiché à l'écran, mais elle ne modifie pas le contenu de la transmission.

- **Attention** : La modification de l'affichage n'est pas effective pour la connexion en cours. Elle est effective uniquement pour la prochaine connexion.


## CardinalitéSourceMax (Propriété)

***En anglais : MaxSourceCardinality***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **CardinalitéSourceMax** permet de paramétrer les caractéristiques d'une liaison. Il est possible de :

- Connaître la cardinalité maximale de la rubrique source dans la liaison.

- Modifier la cardinalité maximale de la rubrique source dans la liaison.



<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=cardinaliteSource.gif)





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Modifier la cardinalité maximale
LiaisonClubPersonne.CardinalitéSourceMax = hCardinalitéInfinie
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître la cardinalité maximale de la rubrique source

`<Cardinalité> = <Liaison>.CardinalitéSourceMax`
---

**`<Cardinalité> : (Entier)`**

Cardinalité maximale de la rubrique source de la liaison. La constante *hCardinalitéInfinie* permet d'identifier la cardinalité "n".

**`<Liaison> : (Chaîne de caractères)`**

Nom de la liaison définie sous l'éditeur d'analyses ou définie grâce au type [Description de Liaison](../WDLang4/1514039.md).  


<a name="SYNTAXE2"></a>

### Modifier la cardinalité maximale de la rubrique source

`<Liaison>.CardinalitéSourceMax = <Nouvelle cardinalité>`
---

**`<Liaison> : (Chaîne de caractères)`**

Nom de la liaison définie sous l'éditeur d'analyses ou définie grâce au type [Description de Liaison](../WDLang4/1514039.md).

**`<Nouvelle cardinalité> : (Entier)`**

Nouvelle cardinalité maximale de la rubrique source de la liaison. La constante *hCardinalitéInfinie* permet de définir la cardinalité "n". Par défaut, lors de la description d'une liaison par programmation, la cardinalité maximale de la rubrique source est égale à 1.


## URI

***En anglais : URI***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le mot-clé URI peur correspondre à : 

- une variable de type [URI](../WDLang3/1000024027.md). 

- la propriété URI permettant gérer un emplacement dans un fichier XML. Cette propriété est utilisable sur :

	- une variable de type [xmlNamespace](../WDLang5/1000018803.md). 

	- une variable de type [xmlSignatureRéférence](../WDLang3/1000020746.md).




## NbFeuille (Propriété)

***En anglais : NbWorksheet***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **NbFeuille** permet de connaître le nombre de feuilles de calcul dans un champ Tableur. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Parcours des feuilles du champ TABLR_MonTableur
POUR I = 1 A TBLR_MonTableur.NbFeuille
	TBLR_MonTableur.FeuilleEnCours = I
	TBLR_MonTableur["A1"] = "Sur la feuille " + I
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Champ Tableur>.NbFeuille`
---

**`<Résultat> : (Entier)`**

Nombre de feuilles de calcul présentes dans le champ Tableur. 

**`<Champ Tableur> : (Nom de champ)`**

Nom du champ Tableur à manipuler.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Pour gérer plus d'une feuille de calcul, le champ Tableur doit autoriser la gestion de plusieurs feuilles de calcul : 

	- Sous l'éditeur : Dans l'onglet "Détail" de la fenêtre de description du champ, cochez l'option "Autoriser la gestion de plusieurs feuilles de calcul". 

	- Par programmation, utilisez la propriété [MultiFeuille](../Proprietes/1000021639.md).




- Les feuilles peuvent être créées : 

	- par l'utilisateur final, 

	- par la fonction [TableurAjouteFeuille](../WDLang1/1000021642.md) ou la fonction [TableurInsèreFeuille](../WDLang1/1000021648.md),  

	- en chargeant un fichier xlsx contenant plusieurs feuilles (fonction [TableurCharge](../WDLang1/1000021340.md)).

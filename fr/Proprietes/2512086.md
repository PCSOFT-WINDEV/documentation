


## RubriqueFiltrée (Propriété)

***En anglais : FilteredItem***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **RubriqueFiltrée** permet de connaître la rubrique sur laquelle un filtre a été mis en place par la fonction [HFiltre](../WDLang4/3044100.md). Cette propriété est utilisable sur un fichier de données, une vue HFSQL ou une requête.






<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Active le filtre
HFiltre(Client, NomClient, "D" + hValMin, "F" + hValMax, "Ville='Paris'")
Trace(Client.RubriqueFiltrée)  // Indique "NomClient"

// Désactive le filtre
HDésactiveFiltre(Client)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Rubrique> = <Elément manipulé>.RubriqueFiltrée`
---

**`<Rubrique> : (Chaîne de caractères)`**



- Rubrique sur laquelle s'applique le filtre défini avec la fonction [HFiltre](../WDLang4/3044100.md), 

- Chaîne vide ("") si aucun filtre n'est actif.




**`<Elément manipulé> : (Chaîne de caractères)`**

Nom du fichier de données, de la requête ou de la vue HFSQL sur laquelle le filtre a été défini.

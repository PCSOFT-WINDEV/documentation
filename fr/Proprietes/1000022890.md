


## RetraitHaut (Propriété)

***En anglais : TopIndent***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **RetraitHaut** permet de connaître et de modifier l'espace laissé en haut du texte (marge haute) dans un champ.

Cette propriété est disponible sur : 

- Les champs de saisie.

- Les champs Libellé.




**Remarque** : Cette propriété est équivalente à la définition de la marge haute : 

- Pour le champ Libellé : la marge haute est définie dans l'onglet "Style" de la fenêtre de description du champ (élément "Cadre extérieur", bouton "Marges", option "Marge haut").

- Pour le champ de saisie, la marge haute est définie dans l'onglet "Style" de la fenêtre de description du champ (élément "Zone de saisie", bouton "Marges", option "Marge haut").



<a name="Exemple1"></a>
<a name="1000022891_sample_code"></a>

## Exemple


```wl
// Réserver 4 pixels de marge en haut et en bas du champ Libellé
LIB_Date.RetraitBas = 4
LIB_Date.RetraitHaut = 4
```
<a name="1000022891_Exemple2"></a>

```wl
// Réserver 5 pixels de marge en haut et en bas de la zone de saisie
SAI_NomVoiture.RetraitBas = 5
SAI_NomVoiture.RetraitHaut = 5
```



<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître la valeur de la marge en haut du texte dans un champ

`<Résultat> = <Champ manipulé>.RetraitHaut`
---

**`<Résultat> : (Entier)`**

Nombre de pixels utilisé pour la marge en haut du texte. Par défaut, ce retrait est de 0 pixel (pas de retrait).  

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ manipulé : 

- Champ Libellé, 

- Champ de saisie.





<a name="SYNTAXE2"></a>

### Modifier la valeur de la marge en haut du texte dans un champ

`<Champ manipulé>.RetraitHaut = <Nouvelle valeur>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ manipulé : 

- Champ Libellé, 

- Champ de saisie.




**`<Nouvelle valeur> : (Entier)`**

Nouvelle valeur de la marge en haut du texte (en pixels). Cette valeur doit être positive.

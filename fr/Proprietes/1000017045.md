


## CodeSQL (Propriété)

***En anglais : SQLCode***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **CodeSQL** permet de connaître le code SQL d'une requête créée avec l'éditeur de requêtes (fichier d'extension ".WDR"). Les commentaires saisis sous l'éditeur de requêtes sont visibles dans le code SQL obtenu.






<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affecte le code SQL de la requête à une chaîne
MonCodeSQL est une chaîne
MonCodeSQL = REQ_RequêteCalcul.CodeSQL
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Obtenir le code SQL d'une requête

`<Code SQL> = <Requête manipulée>.CodeSQL`
---

**`<Code SQL> : (Chaîne de caractères)`**

Code SQL de la requête indiquée. Si le fichier ".WDR" contient des commentaires, ces commentaires seront présents dans le code SQL de la requête (les commentaires commencent par les caractères "--").

**`<Requête manipulée> : (Nom de requête)`**

Nom logique de la requête manipulée. Ce nom a été défini sous l'éditeur de requêtes.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Mise en forme du résultat
<a name="mise_forme_resultat_ELTPARAGRAPHE000050"></a>Le code SQL de la requête est identique à celui affiché sous l'éditeur de requêtes par l'option "Code SQL" du menu contextuel de la requête : les espaces, tabulations et retours à la ligne sont conservés.
<a name="NOTE0_2"></a>


### Utilisation dans les procédures stockées HFSQL
<a name="utilisation_dans_les_procedures_stockees_hfsql_ELTPARAGRAPHE000057"></a>La propriété **CodeSQL** peut être utilisée dans le code d'une procédure stockée pour connaître le code d'une requête stockée présente sur le serveur.

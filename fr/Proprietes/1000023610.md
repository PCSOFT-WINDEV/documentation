
## Séparateur (Propriété)

***En anglais : Separator***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Séparateur** permet de gérer les séparateurs. Cette propriété est utilisable sur : 

- les variables de type [docNiveauNumérotation](../WDLang1/1000022799.md). 

- les variables de type [docFormatMulticolonne](../WDLang1/1000023580.md).

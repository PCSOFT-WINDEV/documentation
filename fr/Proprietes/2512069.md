
## PartieHeure (Propriété)

***En anglais : Time***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **PartieHeure** permet de :

- Récupérer uniquement l'heure d'une variable de type [DateHeure](../Motscles/1514070.md).

- Modifier uniquement l'heure d'une variable de type [DateHeure](../Motscles/1514070.md).

- Récupérer l'heure d'une rubrique de type Date (au format "Date et Heure").
	

- Modifier l'heure d'une rubrique de type Date (au format "Date et Heure").
	




**Rappel** : Les rubriques de type Date permet de gérer :

- les dates simples "Année - Mois - Jour" (format AAAAMMJJ)

- les dates et les heures "Année - Mois - Jour - Heures - Minutes - Secondes - Millièmes de seconde" (format AAAAMMJJHHmmSSCCC)

- les durées "Nombre de jours - Nombre d'heures - Nombre de minutes - Nombre de secondes - Nombre de millièmes de seconde" (format +JHHMMSSCCC)







<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Manipulations d'une variable de type DateHeure
DateDébut est une DateHeure = "200112121515"
HeureFin est une Heure = "1530"
// Changer l'heure
DateDébut.PartieHeure = "153050"// Modifier la partie Heure
DateDébut.PartieHeure = HeureFin
```


<a name="Exemple2"></a>





```wl
// Manipulations d'une rubrique de type DateHeure
Travail.DateDébut = "200112121515"
Travail.HeureFin = "1530"
// Changer l'heure
Travail.DateDébut.PartieHeure = "153050"// Modifier la partie Heure
Travail.DateDébut.PartieHeure = Travail.HeureFin
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître l'heure d'une variable de type DateHeure

`<Résultat> = <Variable DateHeure>.PartieHeure`
---

**`<Résultat> : (Chaîne de caractères ou heure)`**

Heure extraite de la variable de type [DateHeure](../Motscles/1514070.md).

**`<Variable DateHeure> : (DateHeure)`**

Nom de la variable de type [DateHeure](../Motscles/1514070.md) à manipuler.


<a name="SYNTAXE2"></a>

### Modifier l'heure d'une variable de type DateHeure

`<Variable DateHeure>.PartieHeure = <Nouvelle heure>`
---

**`<Variable DateHeure> : (DateHeure)`**

Nom de la variable de type [DateHeure](../Motscles/1514070.md) à manipuler.

**`<Nouvelle heure> : (Chaîne de caractères ou heure ou entier)`**

Nouvelle heure qui remplace l'heure de la variable manipulée.

Cette heure peut être au format :

- Entier : nombre de centièmes de seconde écoulés depuis minuit "00:00:00:00"

- HHMM

- HHMMSS

- HHMMSSCC

- HHMMSSCCC




(HH : heures, MM : minutes, SS : secondes, CC : centièmes de seconde, CCC : millièmes de seconde).


<a name="SYNTAXE3"></a>

### Connaître l'heure d'une rubrique de type Date au format "Date et Heure"

`<Résultat> = <Fichier de données>.<Rubrique>.PartieHeure`
---

**`<Résultat> : (Chaîne de caractères ou heure)`**

Heure extraite de la variable de type [DateHeure](../Motscles/1514070.md).

**`<Fichier de données> : (Chaîne de caractères)`**

Nom du fichier de données manipulé. Ce nom a été défini sous l'éditeur d'analyses ou grâce au type [Description de Fichier](../WDLang4/1514065.md).

**`<Rubrique> : (Chaîne de caractères)`**

Nom de la rubrique manipulée. Ce nom a été défini sous l'éditeur d'analyses ou grâce au type [Description de Rubrique](../WDLang4/1514071.md).  


<a name="SYNTAXE4"></a>

### Modifier l'heure d'une rubrique de type Date au format "Date et Heure"

`<Fichier de données>.<Rubrique>.PartieHeure = <Nouvelle heure>`
---

**`<Fichier de données> : (Chaîne de caractères)`**

Nom du fichier de données manipulé. Ce nom a été défini sous l'éditeur d'analyses ou grâce au type [Description de Fichier](../WDLang4/1514065.md).

**`<Rubrique> : (Chaîne de caractères)`**

Nom de la rubrique manipulée. Ce nom a été défini sous l'éditeur d'analyses ou grâce au type [Description de Rubrique](../WDLang4/1514071.md).

**`<Nouvelle heure> : (Chaîne de caractères ou heure ou entier)`**

Nouvelle heure qui remplace l'heure de la rubrique manipulée. Cette heure peut être au format :

- Entier : nombre de centièmes de seconde écoulés depuis minuit "00:00:00:00"

- HHMM

- HHMMSS

- HHMMSSCC

- HHMMSSCCC




(HH : heures, MM : minutes, SS : secondes, CC : centièmes de seconde, CCC : millièmes de seconde).

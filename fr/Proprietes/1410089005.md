
## ModeEdition (Propriété)

***En anglais : EditMode***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **ModeEdition** permet de connaître et de modifier le mode d'édition du champ Editeur de diagrammes : sélection ou dessin à main levée. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Active l'édition à main levée
EDIAG_MonDiagramme.ModeEdition = diagModeDessinMainLevée
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître le mode d'édition en cours dans le champ Editeur de diagrammes

`<Résultat> = <Champ Editeur de diagrammes>.ModeEdition`
---

**`<Résultat> : (Constante de type Entier)`**

Mode d'édition en cours dans le champ : 



|   |   |
| --- | --- |
| *diagModeDessinMainLevée* | Le mode d'édition en cours dans le champ est le dessin à main levée. |
| *diagModeSélection* | Le mode d'édition en cours dans le champ est la sélection. |





**`<Champ Editeur de diagrammes> : (Nom de champ)`**

Nom du champ Editeur de diagrammes manipulé. 


<a name="SYNTAXE2"></a>

### Modifier le mode d'édition en cours dans le champ Editeur de diagrammes

`<Champ Editeur de diagrammes>.ModeEdition = <Nouveau mode d'édition>`
---

**`<Champ Editeur de diagrammes> : (Nom de champ)`**

Nom du champ Editeur de diagrammes manipulé. 

**`<Nouveau mode d'édition> : (Constante de type Entier)`**

Nouveau mode d'édition en cours dans le champ : 



|   |   |
| --- | --- |
| *diagModeDessinMainLevée* | Le mode d'édition en cours dans le champ est le dessin à main levée. |
| *diagModeSélection* | Le mode d'édition en cours dans le champ est la sélection. |




## Taille (Propriété)

***En anglais : Size***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Taille** permet de connaître et modifier la taille d'un élément. 

Cette propriété peut être utilisée pour connaître et modifier : 

- la taille (en picas) d'une variable de type [Police](../Motscles/1514045.md). 
	**Remarque** : le nom d'une police doit correspondre à une des polices installées sur le poste en cours. Par défaut, la taille de la police est 12.

- le nombre maximum de caractères contenus dans :

	- un champ d'une fenêtre (champ de saisie, colonne de table ou combo en saisie)

	- un champ d'une page (champ de saisie)

	- un champ Libellé d'un état.
			**Remarque** : Le nombre maximum de caractères contenus dans un champ est défini dans l'onglet "Général" de la description du champ.




- la taille maximale d'une rubrique de type texte (chaîne de caractères, mémo texte, caractère) définie sous l'éditeur d'analyses ou définie par programmation.

- la taille maximale d'une rubrique de type Heure définie sous l'éditeur d'analyses ou définie par programmation.




Cette propriété permet également de connaître : 

- le nombre de colonnes d'un champ Table.







<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple
<a class="notetitle" target="_blank" href="$DOC$=1000002510125&name=taille_propriete&product=WD">Voir des exemples supplémentaires</a>

```wl
// Définir le nombre maximum de caractères contenus dans la rubrique "AdresseLiv"
AdresseLiv.Taille = 50
...
// Définir le nombre maximum de caractères contenus dans la rubrique "HeureLiv"
HeureLiv.Taille = 6 // Format HHMMSS
...
// Valide la description du fichier de données
HDécritFichier(Client)
```
<a name="Exemple2"></a>

```wl
// Modifier le nombre maximum de caractères contenus dans le champ "SAI_Nom"
SAI_Nom.Taille = 25
```

## Syntaxe

### Connaître la taille d'une variable de type Police, le nombre maximum de caractères contenus dans un champ

`<Résultat> = <Elément à manipuler>.Taille`
---

**`<Résultat> : (Entier)`**



- Taille (en pica) de la police spécifiée.

- Nombre maximum de caractères contenus dans le champ spécifié ou 0 si le champ spécifié n'a aucune limite de saisie.

- Nombre de colonnes d'une table si le champ spécifié est une table.




**`<Elément à manipuler> : (Variable de type Police ou nom de champ)`**



- Nom de la variable de type [Police](../Motscles/1514045.md) à manipuler.

- Nom du champ à manipuler  





<a name="SYNTAXE2"></a>

### Modifier la taille d'une variable de type Police, le nombre maximum de caractères contenus dans un champ

`<Elément à manipuler>.Taille = <Nouvelle taille>`
---

**`<Elément à manipuler> : (Variable de type Police ou nom de champ)`**



- Nom de la variable de type [Police](../Motscles/1514045.md) à manipuler.

- Nom du champ à manipuler.




**`<Nouvelle taille> : (Entier)`**



- Nouvelle taille (en pica) de la police spécifiée.

- Nombre maximum de caractères contenus dans le champ spécifié ou 0 si le champ spécifié n'a aucune limite de saisie.




Remarque : Il n'est pas possible de modifier le nombre de colonnes présentes dans un champ Table.  


<a name="SYNTAXE3"></a>

### Connaître la taille d'une rubrique

`<Résultat> = <Fichier de données>.<Rubrique>.Taille`
---

**`<Résultat> : (Entier)`**



- Nombre maximum de caractères gérés par la rubrique spécifiée, 

- 0 pour les mémos.




**`<Fichier de données> : (Chaîne de caractères)`**

Nom du fichier de données manipulé. Ce nom a été défini sous l'éditeur d'analyses ou grâce au type [Description de Fichier](../WDLang4/1514065.md).

**`<Rubrique> : (Chaîne de caractères)`**

Nom de la rubrique manipulée. Ce nom a été défini sous l'éditeur d'analyses ou grâce au type [Description de Rubrique](../WDLang4/1514071.md).

***Si cette rubrique est de type texte***, nombre de caractères de la rubrique.

***Si cette rubrique est de type heure***, le nombre de caractères permet d'identifier le format de la rubrique :

| Nombre de caractères<br>de la rubrique Heure | Format de la rubrique Heure |
| --- | --- |
| 2 | HH |
| 4 | HHMM |
| 6 | HHMMSS |
| 8 | HHMMSSCC |
| 9 | HHMMSSCCC |

### Définir la taille (nombre de caractères) d'une rubrique décrite par programmation

`<Rubrique>.Taille = <Nombre de caractères>`
---

**`<Rubrique> : (Chaîne de caractères)`**

Nom de la rubrique manipulée. Ce nom vient d'être défini grâce au type [Description de Rubrique](../WDLang4/1514071.md).

***Si cette rubrique est de type texte***, nombre de caractères de la rubrique.

***Si cette rubrique est de type heure***, le nombre de caractères permet d'identifier le format de la rubrique :

| Nombre de caractères de la rubrique Heure | Format de la rubrique Heure |
| --- | --- |
| 2 | HH |
| 4 | HHMM |
| 6 | HHMMSS |
| 8 | HHMMSSCC |
| 9 | HHMMSSCCC |



**`<Nombre de caractères> : (Entier)`**

Nouveau nombre maximum de caractères gérés par la rubrique spécifiée.

## Remarques
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) 

### Variable de type Police
<a name="variable_type_police_ELTPARAGRAPHE000237"></a>Une police peut être créée à partir d'une variable de type [Police](../Motscles/1514045.md).

Pour définir les caractéristiques de la police, utilisez :

- soit la fonction [PoliceCrée](../WDLang1/3060002.md).
	

- soit les propriétés [Alphabet](../Proprietes/2512008.md), [Angle](../Proprietes/1410086632.md), [Barré](../Proprietes/2513005.md), [Couleur](../Proprietes/2510071.md), [Gras](../Proprietes/2513001.md), [Italique](../Proprietes/2513007.md), [Interlettre](../Proprietes/1410088367.md), [Interligne](../Proprietes/1410088366.md), [Nom](../Proprietes/2510082.md), [Souligné](../Proprietes/2513013.md), [Taille](../Proprietes/2510125.md) et [UnitéTaille](../Proprietes/1410088365.md).
	![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Seules les propriétés suivantes sont utilisables avec le type **Police** : [Angle](../Proprietes/1410086632.md), [Barré](../Proprietes/2513005.md), [Couleur](../Proprietes/2510071.md), [Gras](../Proprietes/2513001.md), [Italique](../Proprietes/2513007.md), [Nom](../Proprietes/2510082.md), [Souligné](../Proprietes/2513013.md), [Taille](../Proprietes/2510125.md).
	




Cette police pourra être utilisée :

- Pour écrire du texte dans des dessins (fonction [dPolice](../WDLang1/3029020.md)).

- Pour écrire dans les graphes (fonctions [grPoliceEtiquette](../WDLang3/3042012.md), [grPoliceLégende](../WDLang3/3042037.md), [grPoliceTitre](../WDLang3/3042048.md) et [grPoliceTitreAxe](../WDLang3/3042059.md)).

- Pour imprimer (fonction [iPolice](../WDLang5/3046014.md)).

- Dans les champs d'une fenêtre (propriété [Police](../Motscles/1514045.md) des champs d'une fenêtre).

- Dans les champs d'un état (propriété [Police](../Motscles/1514045.md) des champs d'un état).





<a name="NOTE0_2"></a>


### Nombre de caractères
<a name="nombre_caracteres_ELTPARAGRAPHE000244"></a>Si le nombre maximum de caractères spécifié est plus petit que le nombre de caractères actuellement présents dans le champ :

- Le contenu du champ n'est pas modifié à l'affichage.

- La valeur du champ comprend l'ensemble des caractères affichés dans le champ.



<a name="NOTE0_3"></a>


### Champ de saisie multiligne
<a name="champ_saisie_multiligne_ELTPARAGRAPHE000254"></a>Si le nombre maximum de caractères dans un champ de saisie multiligne est modifié avec la propriété **Taille**, il sera possible d'affecter par programmation un texte utilisant un nombre de caractères supérieur à celui défini. Par exemple :

```wl
// "SAI_Adresse" est un champ de saisie multiligne
SAI_Adresse.Taille = 5
SAI_Adresse = "ABCDEFGHI" 
// "SAI_Adresse" contient "ABCDEFGHI"
```

<a name="NOTE0_4"></a>


### Champ avec masque de saisie
<a name="champ_avec_masque_saisie_ELTPARAGRAPHE000265"></a>La propriété **Taille** ne doit pas être utilisée pour modifier le nombre maximum de caractères contenus dans les champs comportant un masque de saisie. En effet, le nombre maximum de caractères d'un champ est alors fonction du masque de saisie défini pour ce champ.
<a name="NOTE0_5"></a>


### Limites
<a name="limites_ELTPARAGRAPHE000275"></a>**Dans un fichier de données HFSQL**, la propriété **Taille** s'applique uniquement aux rubriques de type texte et heure.

**Dans un état**, la propriété **Taille** peut être utilisée uniquement sur un champ Libellé.

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) **Dans une fenêtre**, la propriété **Taille** s'applique uniquement aux :

- champs de saisie,

- tables (uniquement en lecture),

- colonnes de table,

- combos en saisie.






![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) La propriété **Taille** s'applique uniquement aux éléments suivants :

- Champ de saisie.

- Combo.

- Table.

- Colonne de table.

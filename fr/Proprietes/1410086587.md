
## Colonne

***En anglais : Column***
	

<a name="description"></a>
La propriété **Colonne** permet de gérer une ou plusieurs colonnes. Cette propriété est disponible pour les variables de type :

- [hbCellule](../WDLang4/1000021707.md).

- [hbDescriptionTable](../WDLang4/1000021688.md).

- [hbLecture](../WDLang4/1000021701.md).

- [hbParamètreScan](../WDLang4/1000021715.md).

- [hbRésultatScan](../WDLang4/1000021724.md). 

- [xlsDocument](../WDLang5/1000017464.md).

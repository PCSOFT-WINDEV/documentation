
## Ligne (Propriété)

<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Ligne** permet de gérer une ou plusieurs lignes. Elle est disponible sur les variables de type :

- [hbEcriture](../WDLang4/1000021710.md).

- [hbLecture](../WDLang4/1000021701.md).

- [hbRésultatScan](../WDLang4/1000021724.md).

- [TâcheGantt](../WDLang1/1000020881.md).

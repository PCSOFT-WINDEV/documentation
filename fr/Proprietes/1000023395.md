
## NamespaceDéclaré (Propriété)

***En anglais : NamespaceDeclared***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **NamespaceDéclaré** permet de gérer les espaces de nommage définis. Cette propriété est utilisable sur :

- une variable de type [xmlNoeud](../WDLang5/1000018786.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [xmlLecteur](../WDLang5/1000023392.md).

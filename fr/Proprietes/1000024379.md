


## ParRéférence (Propriété)

***En anglais : ByReference***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **ParRéférence** permet de savoir si un paramètre d'une procédure a été passé par référence ou non.

Si le paramètre a été passé par référence, sa valeur peut être modifiée dans le code de la procédure.

Remarques : 

- Cette propriété est identique à la propriété [ParAdresse](../Proprietes/1000017048.md).

- Les termes "Passage de paramètre par référence" et "Passage de paramètre par adresse" sont identiques. 





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
Indice est un entier = 1
Indice2 est un entier = 3
Indice3 est un entier = 4
// Avant l'appel de la procédure, les différentes variables ont leurs valeurs par défaut
AjouteUn(Indice, Indice2, Indice3)
// Après l'appel de la procédure, les variables ont toutes changé de valeur
```

```wl
// -- Déclaration de la procédure
PROCEDURE AjouteUn(*)
POUR I = 1 _A_ MesParamètres..Occurrence
	SI MesParamètres[I]..ParRéférence = Vrai ALORS
		MesParamètres[I] += 1
	FIN
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Paramètre manipulé>.ParRéférence`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le paramètre a été passé par référence, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Paramètre manipulé> : (Chaîne de caractères)`**

Nom du paramètre manipulé. Ce paramètre est désigné grâce au mot-clé **MesParamètres**, en spécifiant l'indice du paramètre manipulé. Par exemple : MesParamètres[1] pour manipuler le premier paramètre passé à la procédure.

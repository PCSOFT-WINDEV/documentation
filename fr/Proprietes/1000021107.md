


## NullSiVide (Propriété)

***En anglais : NullIfEmpty***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **NullSiVide** permet de : 

- Savoir si la valeur renvoyée par le champ est NULL si son contenu est vide. 

- Paramétrer la valeur renvoyée par le champ si son contenu est vide.




Remarque : Cette propriété est équivalente à l'option "Retourner NULL si vide" de la fenêtre de description du champ (onglet "Détail").
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SAI_PrixMin.NullSiVide = Vrai
SAI_PrixMax.NullSiVide = Vrai
REQ_Rech.ParamPrixMin = SAI_PrixMin
REQ_Rech.ParamPrixMax = SAI_PrixMax
HExécuteRequête(REQ_Rech)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Savoir si la valeur renvoyée par le champ est NULL si son contenu est vide

`<Résultat> = <Champ manipulé>.NullSiVide`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la valeur renvoyée par le champ est NULL si son contenu est vide, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 




**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler. Ce champ peut correspondre à un champ de saisie ou un champ Combo en saisie. 


<a name="SYNTAXE2"></a>

### Paramétrer la valeur renvoyée par le champ si son contenu est vide.

`<Champ manipulé>.NullSiVide = <Nouvelle gestion>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler. Ce champ peut correspondre à un champ de saisie ou un champ Combo en saisie. 

**`<Nouvelle gestion> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la valeur renvoyée par le champ est NULL si son contenu est vide, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Cette propriété permet de réaliser facilement des recherches multicritères via une requête paramétrée dans laquelle les valeurs des paramètres correspondent à des champs de saisie. En effet, les paramètres d'une requête ayant pour valeur NULL sont ignorés. Pour plus de détails, consultez [Utilisation d'une requête paramétrée](../Editeurs/2032032.md).  

- Cette propriété n'a pas d'effet lors de l'affectation du champ avec une valeur.

- Cette propriété est disponible uniquement pour les champs de saisie et les champs Combo en saisie.

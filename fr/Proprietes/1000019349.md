


## VisibleHorsFenêtre (Propriété)

***En anglais : VisibleOutsideWindow***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **VisibleHorsFenêtre** permet de : 

- savoir si un champ Bouton est visible même si sa position (X,Y) le met en dehors du cadre de la fenêtre (entièrement ou partiellement). 

- modifier la visibilité d'un champ Bouton "hors fenêtre".




**Attention** : Pour utiliser la propriété **VisibleHorsFenêtre**, la fenêtre en cours doit posséder un cadre détouré. Dans le cas contraire, la propriété est sans effet.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Met le champ Bouton OK sous la fenêtre
BTN_OK.Y = MaFenêtre.Hauteur + 10
// Rend le champ Bouton visible 
BTN_OK.VisibleHorsFenêtre = Vrai
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Savoir si un champ Bouton est visible "Hors fenêtre"

`<Résultat> = <Champ Bouton>.VisibleHorsFenêtre`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le champ Bouton est visible hors fenêtre, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 




**`<Champ Bouton> : (Nom de champ)`**

Nom du champ Bouton à manipuler.


<a name="SYNTAXE2"></a>

### Modifier la visibilité d'un champ Bouton hors fenêtre

`<Champ Bouton>.VisibleHorsFenêtre = <Visibilité>`
---

**`<Champ Bouton> : (Nom de champ)`**

Nom du champ Bouton à manipuler

**`<Visibilité> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le champ Bouton doit être visible hors fenêtre, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Si le champ Bouton est au dessus ou à gauche de la fenêtre (propriété Y&lt;0 ou propriété X&lt;0), le champ Bouton ne sera pas visible. Pour être rendu visible ou invisible, le champ Bouton doit être placé à droite ou en dessous de la fenêtre par exemple. 

- Si la propriété [Visible](../Proprietes/2510138.md) du champ Bouton est à <u><u><u><u>Faux</u></u></u></u>, le champ Bouton ne sera pas visible.

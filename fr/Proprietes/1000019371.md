
## AngleTitre (Propriété)

***En anglais : OrientationTitle***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **AngleTitre** permet de : 

- Savoir si les titres des colonnes d'un champ Table sont inclinés. 

- Incliner ou non les titres des colonnes d'un champ Table (par exemple à 45 degrés).




**Remarque** : Si le titre est incliné, la hauteur du titre est adaptée en conséquence pour laisser voir les titres des colonnes. 

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=AngleTitre.gif)

<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
TABLE_TABLE1.COL_Colonne1.Titre = "Titre vraiment long"
TABLE_TABLE1.AngleTitre = 50
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Savoir si le titre d'une colonne est incliné

`<Inclinaison> = <Champ manipulé>.AngleTitre`
---

**`<Inclinaison> : (Entier)`**

Degré d'inclinaison du titre de la colonne. Ce degré d'inclinaison peut être compris entre 0 (titre horizontal) et 90 (titre vertical). Par défaut, l'inclinaison est de 0 (titre horizontal)

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ manipulé : champ Table ou champ Table hiérarchique. 


<a name="SYNTAXE2"></a>

### Incliner ou non les titres des colonnes

`<Champ manipulé>.AngleTitre = <Inclinaison>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ manipulé : champ Table ou champ Table hiérarchique. 

**`<Inclinaison> : (Entier)`**

Degré d'inclinaison du titre de la colonne. Ce degré d'inclinaison peut être compris entre 0 (titre horizontal) et 90 (titre vertical).







<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Cas particuliers
<a name="cas_particuliers_ELTPARAGRAPHE000049"></a>

- Si un angle est précisé, la position des différentes icônes des colonnes (colonne triée, colonne triable, recherche et filtre) devient fixe.

- Les angles inférieurs à 10 degrés sont représentés par des angles de 10 degrés au moins.

- La propriété est sans effet sur les champs Table orientés à 90 degrés (c'est-à-dire avec les titres situés à gauche et non en haut). 

- Si les fonctions de gestion des graphiques ([gImage](../WDLang1/3029042.md), [gPoliceGras](../WDLang1/3029061.md), ...) sont utilisées dans les titres des colonnes, lorsque le titre est incliné avec la propriété **AngleTitre**, ces fonctions n'ont plus d'effet.  



<a name="NOTE0_2"></a>


### Limite
<a name="limite_ELTPARAGRAPHE000068"></a>La propriété **AngleTitre** s'applique uniquement aux champs Table et Table hiérarchique d'une fenêtre.

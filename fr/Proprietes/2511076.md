


## ToutModifiableAvecEtatsEtRequêtes (Propriété)

***En anglais : AllModifiableWithReportsAndQueries***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **ToutModifiableAvecEtatsEtRequêtes** permet de savoir si un état est modifiable sous le logiciel Etats & Requêtes. Lorsqu'un état est modifiable sous le logiciel Etats & Requêtes, il est possible de :

- modifier son interface.

- modifier son code.

- supprimer un ou plusieurs de ses champs.




**Remarques** :

- Cette fonctionnalité est définie sous l'éditeur lors de la description de l'état (sous l'onglet "Options" de la fenêtre de description, option "État modifiable sous États et Requêtes", sous-option "Tout modifiable (interface et code)").

- La propriété [InterfaceModifiableAvecEtatsEtRequêtes](../Proprietes/2511075.md) permet uniquement de savoir si l' **interface** d'un état est modifiable sous le logiciel Etats & Requêtes.

- Pour plus de détails sur le logiciel Etats & Requêtes, consultez [Logiciel Etats & Requêtes](../Presentation/3088003.md).





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Etat modifiable sous Etats & Requêtes ?
SI ETAT_MonEtat.ToutModifiableAvecEtatsEtRequêtes = Vrai ALORS
	// Mode WINDEV ?
	SI EnModeWindows() = Vrai ALORS
		// Ouverture de l'état sous Etats & Requêtes
		LanceEtatsEtRequêtes(ETAT_MonEtat)
	SINON
		Info("Cet état est modifiable sous Etats & Requêtes")
	FIN
SINON
	Info("Cet état n'est pas modifiable sous Etats & Requêtes")
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Etat manipulé>.ToutModifiableAvecEtatsEtRequêtes`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'état spécifié est modifiable sous le logiciel Etats & Requêtes, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Etat manipulé> : (Nom d'état)`**

Nom de l'état à manipuler.

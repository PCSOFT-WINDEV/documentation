
## RetraitPremièreLigne (Propriété)

***En anglais : IndentFirstLine***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **RetraitPremièreLigne** permet de connaître et de modifier le retrait de première ligne d'un élément. Cette propriété est utilisable sur :

- les variables de type [docParagraphe](../WDLang1/1000022483.md).

- les variables de type [docNiveauNumérotation](../WDLang1/1000022799.md).

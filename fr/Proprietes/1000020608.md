


## CompacteOption (Propriété)

***En anglais : CompactOption***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **CompacteOption** permet de :

- savoir si les options d'un champ (Sélecteur, Interrupteur, Tableau de bord) sont en mode compacté.

- passer les options d'un champ (Sélecteur, Interrupteur, Tableau de bord) en mode compacté.




En mode compacté, les options d'un champ se resserrent : 

- lorsqu'une option est rendue invisible. 

- lorsqu'une option est supprimée.




Si les options ne sont pas compactées, elles conservent leur position d'origine. 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 
```wl
INT_Options.CompacteOption = Vrai
// Supprime l'option numéro 3 d'un champ Interrupteur
InterrupteurSupprime(INT_Options, 3)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Savoir si les options d'un champ sont compactées

`<Résultat> = <Champ manipulé>.CompacteOption`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si les options sont compactées, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Champ manipulé> : (Nom de champ)`**

Nom du champ manipulé. Ce champ peut être : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) un champ Interrupteur, 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) un champ Sélecteur, 

- un champ Tableau de bord.





<a name="SYNTAXE2"></a>

### Définir le compactage des options d'un champ

`<Champ manipulé>.CompacteOption = <Compacter>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ manipulé. Ce champ peut être : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) un champ Interrupteur, 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) un champ Sélecteur, 

- un champ Tableau de bord.




**`<Compacter> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si les options du champ doivent être compactées, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Cette propriété est disponible pour : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) les champs Interrupteur. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) les champs Sélecteur. 

- les champs Tableau de bord.




**Remarques** : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Champs Interrupteur et Sélecteur : Cette propriété correspond à l'option "Compacter les options invisibles" disponible dans l'onglet "Détail" de la description du champ.

- Champ Tableau de bord : Cette propriété correspond à l'option "Compacter les Widgets" disponible dans l'onglet "Détail" de la description du champ.

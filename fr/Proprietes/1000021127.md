


## Synonyme (Propriété)

***En anglais : Synonym***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Synonyme** permet de définir les synonymes dans un index full-text défini par programmation.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonIndexFulText est une Description d'IndexFullText
...
MonIndexFulText.Synonyme = "Voiture Auto Automobile" + RC + "Maison Villa Chalet"
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Nom de l'index full-text>.Synonyme = <Synonymes>`
---

**`<Nom de l'index full-text> : (Chaîne de caractères)`**

Nom de l'index full-text manipulé. Ce nom vient d'être défini grâce au type [Description d'IndexFullText](../WDLang4/1000017461.md).

**`<Synonymes> : (Chaîne de caractères)`**

Liste des mots devant être considérés comme synonymes. Les différents mots sont séparés par un espace, et les différents synonymes sont séparés par un Retour Chariot (RC). 

```txt
"MotA SynonymeA1 SynonymeA2" + RC + "MotN SynonymeN1 SynonymeN2"
```

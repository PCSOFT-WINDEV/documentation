


## WDD55 (Propriété)

***En anglais : WDD55***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **WDD55** permet de :

- Connaître le nom et le chemin complet du fichier WDD au format Hyper File 5.5. Ce fichier est utilisé pour accéder aux fichiers au format Hyper File 5.5 présents dans une analyse au format HFSQL Classic.

- Définir le nom et le chemin complet du fichier WDD au format Hyper File 5.5. Ce fichier est utilisé pour accéder aux fichiers au format Hyper File 5.5 présents dans une analyse au format HFSQL Classic. Cette propriété est utile lors de la description par programmation d'un fichier de données au format 5.5 dans une analyse au format HFSQL Classic.




**Attention** : Cette propriété est utilisable uniquement pour des fichiers de données Hyper File 5.5 non migrés dans une analyse au format HFSQL Classic.

Le nom du fichier WDD au format 5.5 peut être retrouvé dans la description du fichier de données sous l'analyse :
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=WDD55%20-%20HC%20N%B0001.gif)



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connaître le chemin du fichier de données Commande
ResChemin = Commande.WDD55
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le chemin du fichier WDD au format Hyper File 5.5

`<Résultat> = <Fichier de données>.WDD55`
---

**`<Résultat> : (Chaîne de caractères)`**

Chemin du WDD correspondant à l'analyse initiale au format 5.5. Si le répertoire de l'analyse en exécution correspond au répertoire en cours, ce chemin est relatif au répertoire en cours.

**`<Fichier de données> : (Chaîne de caractères)`**

Nom du fichier de données manipulé au format Hyper File 5.5.  


<a name="SYNTAXE2"></a>

### Définir le chemin du fichier WDD au format Hyper File 5.5

`<Fichier de données>.WDD55 = <Chemin>`
---

**`<Fichier de données> : (Chaîne de caractères)`**

Nom du fichier de données manipulé. Ce nom a été défini grâce au type [Description de Fichier](../WDLang4/1514065.md).

**`<Chemin> : (Chaîne de caractères)`**

Chemin du WDD correspondant à l'analyse initiale au format 5.5.

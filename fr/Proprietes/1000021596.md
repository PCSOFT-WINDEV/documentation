


## DescriptionType (Propriété)

***En anglais : TypeDescription***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **DescriptionType** permet de : 

- Connaître le type utilisé par une rubrique de type Enumération ou Combinaison. Cette rubrique a été définie sous l'éditeur d'analyses ou définie par programmation.

- Définir le type utilisé par une rubrique de type Enumération ou Combinaison (lors de la description de la rubrique par programmation uniquement).





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Code du projet
EtatChien est une Enumération
	Assis = 1
	Couché = 2
	Debout = 3
	PrêtAJouer = 4
FIN

EtatVacances est une Enumération
	Passé = 1
	Encours = 2
	A_Venir = 3
	Annulé = 4
FIN
```



```wl
// Code d'un bouton "Création fichier"
UnFichier est une Description de Fichier
UneRubrique est une Description de Rubrique
CLIENT est une Source de Données

// Description du fichier de données "CLIENT"
UnFichier.Nom = "CLIENT"
UnFichier.Type = hFichierNormal
UnFichier.CryptageFic = hCryptageStandard

// Description de la rubrique "EtatChien"
UneRubrique.Nom = "Rub_EtatChien"
UneRubrique.Type = hRubEnumération
UneRubrique.DescriptionType = EtatChien
HDécritRubrique(UnFichier, UneRubrique)

// Description de la rubrique "EtatVacances"
UneRubrique.Nom = "Rub_EtatVacances"
UneRubrique.Type = hRubEnumération
UneRubrique.DescriptionType = EtatVacances
HDécritRubrique(UnFichier, UneRubrique)

// Valide la description du fichier de données 
HDécritFichier(UnFichier)

// Création du fichier de données Client
HCréation(CLIENT)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le type d'une rubrique Combinaison ou Enumération

`<Résultat> = <Fichier de données>.<Rubrique>.DescriptionType`
---

**`<Résultat> : (Chaîne de caractères)`**

Nom du type.

**`<Fichier de données manipulé> : (Nom de fichier)`**

Nom du fichier de données manipulé. Ce nom a été défini sous l'éditeur d'analyses ou grâce au type [Description de Fichier](../WDLang4/1514065.md).

**`<Rubrique manipulée> : (Nom de rubrique)`**

Nom de la rubrique clé manipulée. Ce nom a été défini sous l'éditeur d'analyses ou grâce au type [Description de Rubrique](../WDLang4/1514071.md).


<a name="SYNTAXE3"></a>

### Définir le type d'une rubrique Combinaison ou Enumération définie par programmation

`<Rubrique manipulée>.DescriptionType = <Nouveau type>`
---

**`<Rubrique manipulée> : (Nom de rubrique)`**

Nom de la rubrique de type Combinaison ou Enumération manipulée. Ce nom vient d'être défini grâce au type [Description de Rubrique](../WDLang4/1514071.md).

**`<Nouveau type> : (Chaîne de caractères)`**

Type de la rubrique Enumération ou Combinaison. Ce type doit avoir été défini par programmation.


## Trait (Propriété)

***En anglais : Line***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Trait** permet de définir le style de trait utilisé. Cette propriété est utilisable sur :

- une variable de type [Cadre](../Motscles/15140100.md). 

- une variable de type [docElémentZoneDeTexte](../WDLang1/1000023420.md).

- une variable de type [CarteCercle](../WDLang3/1000025504.md).

- une variable de type [CarteImage](../WDLang3/1000025523.md).

- une variable de type [CartePolygone](../WDLang3/1000025476.md).

- une variable de type [CartePolyligne](../WDLang3/1000025489.md).

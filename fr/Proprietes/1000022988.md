


## MotDePasseUtilisateur (Propriété)

***En anglais : UserPassword***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **MotDePasseUtilisateur** permet de gérer le mot de passe de l'utilisateur utilisé dans le cadre d'une authentification par mot de passe. Cette propriété est utilisée : 

- par les variables de type [sshSession](../WDLang3/1000021157.md). 

- par les variables de type [mqttSession](../WDLang3/1000023023.md).

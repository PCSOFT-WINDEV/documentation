


## TexteSélectionné (Propriété)

***En anglais : SelectedText***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **TexteSélectionné** permet de connaître et de modifier le texte sélectionné dans un champ de saisie, un champ Combo en saisie ou une colonne d'un champ Table en saisie.

Remarque : Cette propriété permet d'éviter l'utilisation des propriétés [Curseur](../Proprietes/2510025.md) et [FinCurseur](../Proprietes/2510047.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI SAI_Saisie1.TexteSélectionné = "<E>" ALORS
	SAI_Saisie1.TexteSélectionné = "<Entrer>"
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le texte sélectionné

`<Texte> = <Champ manipulé>.TexteSélectionné`
---

**`<Texte> : (Chaîne de caractères)`**

Texte sélectionné par l'utilisateur dans le champ de saisie.

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler. Ce champ peut correspondre à : 

- un champ de saisie, 

- un champ Combo en saisie, 

- une colonne d'un champ Table en saisie.  





<a name="SYNTAXE2"></a>

### Modifier le texte sélectionné

`<Champ manipulé>.TexteSélectionné = <Texte à sélectionner>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler. Ce champ peut correspondre à : 

- un champ de saisie, 

- un champ Combo en saisie, 

- une colonne d'un champ Table en saisie.




**`<Texte à sélectionner> : (Chaîne de caractères)`**

Texte devant être mis en surbrillance dans le champ. Le texte remplace le texte actuellement sélectionné.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Cas particuliers
<a name="cas_particuliers_ELTPARAGRAPHE000056"></a>

- Si le champ est au format RTF, le texte sélectionné est un texte sans formatage.

- Si le champ est un champ Unicode, la chaîne renvoyée est une chaîne Unicode.



<a name="NOTE0_2"></a>


### Limites
<a name="limites_ELTPARAGRAPHE000064"></a>La propriété **TexteSélectionné** est disponible pour les champs suivants :

- champ de saisie,

- combo avec saisie,

- colonne d'un champ Table de type texte.

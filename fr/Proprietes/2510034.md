


## Image (Propriété)

***En anglais : Image***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Image** permet de connaître et de modifier l'image de fond associée à un élément.
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Dans un état, la propriété **Image** permet de connaître et de modifier l'image de fond d'un champ ou d'un bloc.![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Dans une fenêtre, la propriété **Image** permet de connaître et de modifier l'image :

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) affichée dans la barre d'icônes d'une fenêtre de type mère MDI (uniquement en WINDEV),

- associée au libellé d'un bouton,

- associée à un volet d'onglet ou à un onglet,

- associée aux volets d'une boîte à outils ou à un bouton d'un volet de la boîte à outils,

- associée au libellé d'une option de menu,

- utilisée comme curseur dans une jauge ou dans un potentiomètre non rotatif,

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) utilisée pour la coche des interrupteurs ou des sélecteurs (uniquement en WINDEV),

- utilisée dans les options des interrupteurs et sélecteurs au look Vista,

- utilisée en image de fond d'un champ Calendrier.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) utilisée dans un champ Notation.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) utilisée pour les marqueurs d'un champ Carte.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) utilisée pour un regroupement lorsque le regroupement d'un ruban est fermé.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) utilisée pour représenter une jauge infinie.


**Remarque** : Ces images sont définies sous l'éditeur dans la description du champ, de la fenêtre ou de la page (onglet "Général" ou "Détail").

## Remarques
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) 

### Image associée à un champ ou à un bloc d'un état
<a name="image_associee_champ_bloc_etat_ELTPARAGRAPHE000275"></a>

- L'image d'un champ Image peut être connue et/ou modifiée sous l'éditeur d'états dans l'onglet "Général" de la description du champ.

- L'image de fond d'un champ ou d'un bloc peut être connue et/ou modifiée sous l'éditeur d'états, dans l'onglet "Style" de la description du champ ou du bloc. Il suffit de sélectionner l'élément "Image de fond".



<a name="NOTE0_2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) 

### Image scannée dans un état
<a name="image_scannee_dans_etat_ELTPARAGRAPHE000285"></a>La propriété **Image** ne permet pas de modifier l'image scannée d'un état. En effet, l'image scannée d'un état permet de :

- faciliter le positionnement des différents champs dans l'état,

- visualiser le résultat lors de l'aperçu avant impression.



<a name="NOTE0_3"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) 

### Image dans une option de menu
<a name="image_dans_une_option_menu_ELTPARAGRAPHE000300"></a>Si une image est associée à une option de menu marquée, l'image remplace la marque.
<a name="NOTE0_4"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) 

### Image d'un onglet ou d'un volet d'onglet
<a name="image_onglet_volet_onglet_ELTPARAGRAPHE000309"></a>Pour modifier l'image d'un onglet (c'est-à-dire l'image des boutons de l'onglet), il suffit d'utiliser le code suivant :

```wl
ONG_Onglet.Image = "Exemple.jpg"
```
Pour modifier l'image affichée dans le bouton de l'onglet (à côté du libellé), il suffit de préciser l'indice de l'onglet à modifier :

```wl
ONG_Onglet[1].Image = "Exemple.jpg"
```

<a name="NOTE0_5"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) 

### Image des coches d'interrupteur et de sélecteur
<a name="image_des_coches_interrupteur_selecteur_ELTPARAGRAPHE000321"></a>La propriété **Image** permet de changer la coche des interrupteurs et des sélecteurs. Cette propriété correspond à l'option "Image des coches" dans l'onglet "Détail" de la description du champ.

Si l'image de la coche ne doit pas être animée : l'image utilisée doit être constituée des deux ou trois états de la coche : Coché, Décoché et Indéfini si nécessaire.

Si l'image de la coche est animée, l'image utilisée doit être une image GIF, contenant pour chaque état principal (Coché, Décoché et Indéfini), les images suivantes :

- Coche active (pas d'animation)

- Coche enfoncée (animation)

- Coche grisée (pas d'animation)

- Coche survolée (animation)




Exemple d'image utilisée pour une coche animée :<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Image_Interrup.gif)
1. Images pour l'état coché.

2. Images pour l'état décoché.

3. Images pour l'état indéfini.

### Limites
<a name="limites_ELTPARAGRAPHE000370"></a>![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Dans un état, la propriété **Image** ne peut pas être utilisée sur :

- un état.

- un code-barres.

- un champ Interrupteur.


Formats d'images autorisés : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Les formats d'images pouvant être affichés sont les suivants : tous les formats décrits dans la fonction [fFiltreImage](../WDLang1/3036009.md).


![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) La propriété **Image** peut être utilisée uniquement sur :

- une fenêtre de type MDI,

- un bouton,

- un volet d'onglet ou un onglet,

- un volet d'une barre d'outils ou un bouton d'un volet,

- une option de menu,

- une jauge,

- un potentiomètre non rotatif,

- un interrupteur,

- un sélecteur,

- un champ Calendrier,

- un champ Notation.

- un marqueur d'un champ Carte.

- un champ Jauge infinie personnalisé.




![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) La propriété **Image** s'applique uniquement aux éléments suivants :

- Bouton.

- Fenêtre.




## BarreDeMessage (Propriété)

***En anglais : StatusBar***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **BarreDeMessage** permet de :

- savoir si la barre de message est affichée ou non,

- afficher ou non la barre de message d'une fenêtre.




**Remarque** : Cette propriété correspond à l'option "Barre de message" de l'onglet "Style" de la description de la fenêtre.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affiche la barre de message
MaFenêtre.BarreDeMessage = Vrai
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Savoir si la barre de message est affichée ou non

`<Barre affichée> = <Fenêtre>.BarreDeMessage`
---

**`<Barre affichée> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la barre de message est affichée, 

- <u><u><u><u>Faux</u></u></u></u> si la barre de message est invisible.




**`<Fenêtre> : (Nom de fenêtre)`**

Nom de la fenêtre manipulée.  


<a name="SYNTAXE2"></a>

### Afficher ou non la barre de message

`<Fenêtre>.BarreDeMessage = <Barre de message>`
---

**`<Fenêtre> : (Nom de fenêtre)`**

Nom de la fenêtre manipulée.  

**`<Barre de message> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> pour afficher la barre de message, 

- <u><u><u><u>Faux</u></u></u></u> pour supprimer la barre de message.

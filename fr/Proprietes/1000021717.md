
## Filtre

***En anglais : Filter***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le mot-clé **Filtre** peut correspondre à : 

- la fonction [Filtre](../WDLang1/1000024845.md) qui renvoie un tableau contenant les éléments d'un tableau source pour lesquels une procédure renvoie <u><u><u>Vrai</u></u></u>.

- la propriété **Filtre** permet de connaître et de modifier les caractéristiques d'un filtre. 
	Cette propriété est utilisable sur :

	- les champs Liste, Combo, Table fichier. Pour plus de détails, consultez l'aide de la propriété [Filtre](../Proprietes/2510042.md).

	- une variable de type [hbParamètreScan](../WDLang4/1000021715.md).

	- une variable de type [mongoOpérationModificationUn](../WDLang4/1410089164.md).

	- une variable de type [mongoOpérationModificationTout](../WDLang4/1410089165.md).

	- une variable de type [mongoOpérationRemplacementUn](../WDLang4/1410089166.md).

	- une variable de type [mongoOpérationSuppressionTout](../WDLang4/1410089162.md).

	- une variable de type [mongoOpérationSuppressionUn](../WDLang4/1410089161.md).

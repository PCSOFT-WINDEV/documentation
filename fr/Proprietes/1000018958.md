


## HeureOuvrableDébut (Propriété)

***En anglais : WorkingHourStart***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **HeureOuvrableDébut** permet de connaître et de modifier l'heure de début des heures ouvrables utilisées : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) par un champ Agenda.

- par un champ Planning.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) par une colonne de type Diagramme de Gantt (dans un champ Table ou Table hiérarchique).




Remarques :

- La plage horaire des heures ouvrables peut être modifiée également dans la fenêtre de description du champ (onglet "Détail"). 

- Les rendez-vous ou les tâches peuvent être déplacés en dehors des heures ouvrables. 





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
AGD_Agenda_1.HeureOuvrableDébut = "0830"
AGD_Agenda_1.HeureOuvrableFin = "1800"
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître l'heure de début des heures ouvrables

`<Résultat> = <Champ manipulé>.HeureOuvrableDébut`
---

**`<Résultat> : (Heure)`**

Heure de début des heures ouvrables (au format HHMM). 

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ manipulé : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) champ Agenda.

- champ Planning.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) colonne de type Diagramme de Gantt (dans un champ Table ou Table hiérarchique).





<a name="SYNTAXE2"></a>

### Modifier l'heure de début des heures ouvrables

`<Champ manipulé>.HeureOuvrableDébut = <Nouvelle heure>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ manipulé :  

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) champ Agenda.

- champ Planning.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) colonne de type Diagramme de Gantt (dans un champ Table ou Table hiérarchique).




**`<Nouvelle heure> : (Chaîne de caractères ou variable de type Heure)`**

Nouvelle heure de début des heures ouvrables du champ. Cette heure est au format "HHMM" ou "HH".



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Heure de fin des heures ouvrables 
<a name="heure_fin_des_heures_ouvrables_ELTPARAGRAPHE000114"></a>L'heure de fin des heures ouvrables peut être connue et modifiée grâce à la propriété [HeureOuvrableFin](../Proprietes/1000019056.md). 
<a name="NOTE0_2"></a>


### Limitation
<a name="limitation_ELTPARAGRAPHE000124"></a>Cette propriété n'est pas disponible pour les champs Agenda et pour les champs Planning présents dans les états.

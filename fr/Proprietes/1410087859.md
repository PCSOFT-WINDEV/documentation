
## Signet (Propriété)

***En anglais : Bookmark***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Signet** permet de gérer les signet d'un document. Cette propriété est utilisable sur : 

- une variable de type [Document](../WDLang1/1000022461.md). 

- une variable de type [pdfParamètre](../WDLang6/1410087857.md). 

- une variable de type [pdfDocument](../WDLang6/1000024909.md).

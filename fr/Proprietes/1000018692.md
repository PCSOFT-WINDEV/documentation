
## Fichier (Propriété)

***En anglais : File***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Fichier** permet de gérer les fichiers pris en compte par une fonctionnalité. Cette propriété est utilisable sur : 

- une variable de type [xlsDocument](../WDLang5/1000017464.md).

- une variable de type [hOptimisationPlanifiée](../WDLang4/1000018854.md). 

- une variable de type [zipArchive](../WDLang3/1000018679.md).

- une variable de type [hRSConfig](../WDLang4/1000020484.md).




Remarque : Le mot-clé **Fichier** peut aussi correspondre à une [Description de Fichier](../WDLang4/1514065.md).

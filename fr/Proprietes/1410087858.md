
## Qualité (Propriété)

***En anglais : Quality***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Qualité** permet de connaître et de définir le niveau de qualité d'un élément. Cette propriété est utilisable sur : 

- ![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) une variable de type [VoixDeSynthèse](../WDLang3/1000025986.md). 

- une variable de type [pdfParamètre](../WDLang6/1410087857.md).


## MotVide (Propriété)

***En anglais : StopWord***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **MotVide** permet de définir les mots creux dans un index full-text défini par programmation.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonIndexFulText est une Description d'IndexFullText
...
MonIndexFulText.MotVide = "le la à a un une"
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Index full-text>.MotVide = <Mots creux>`
---

**`<Index full-text> : (Chaîne de caractères)`**

Nom de l'index full-text manipulé. Ce nom vient d'être défini grâce au type [Description d'IndexFullText](../WDLang4/1000017461.md).

**`<Mots creux> : (Chaîne de caractères)`**

Liste des mots devant être considérés comme des mots creux. Les différents mots peuvent être séparés par : 

- un espace,

- une virgule

- un point virgule,

- un point,

- deux points,

- un retour chariot (RC),

- une tabulation (TAB).

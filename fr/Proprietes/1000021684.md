


## Certificat (Propriété)

***En anglais : Certificate***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Certificat** permet de connaître et de modifier le certificat utilisé par un champ Signature dans un état.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
UnCertificat est un Certificat
tabCertificat est tableau de Certificats = CertificatListe()
// Choix d'un certificat
UnCertificat = tabCertificat[1]
// Association du certificat au champ Signature
SIG_MaSignature.Certificat = UnCertificat
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître le certificat utilisé

`<Résultat> = <Champ Signature>.Certificat`
---

**`<Résultat> : (Variable de type Certificat)`**

Variable de type [Certificat](../WDLang1/1000019287.md) contenant la description du certificat utilisé. 

**`<Champ Signature> : (Nom de champ)`**

Champ Signature à manipuler.


<a name="SYNTAXE2"></a>

### Modifier le certificat utilisé

`<Champ Signature>.Certificat = <Nouveau certificat>`
---

**`<Champ Signature> : (Nom de champ)`**

Champ Signature à manipuler.

**`<Nouveau certificat> : (Variable de type Certificat)`**

Variable de type [Certificat](../WDLang1/1000019287.md) contenant la description du certificat à utiliser.

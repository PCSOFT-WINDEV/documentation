


## PartieDate (Propriété)

***En anglais : Date***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **PartieDate** permet de :

- Récupérer uniquement la date d'une variable de type [DateHeure](../Motscles/1514070.md).

- Modifier uniquement la date d'une variable de type [DateHeure](../Motscles/1514070.md).

- Récupérer la date d'une rubrique de type Date (au format "Date et Heure").
	

- Modifier la date d'une rubrique de type Date (au format "Date et Heure").
	




**Rappel** : Les rubriques de type Date permettent de gérer :

- les dates simples "Année - Mois - Jour" (format AAAAMMJJ)

- les dates et les heures "Année - Mois - Jour - Heures - Minutes - Secondes - Millièmes de seconde" (format AAAAMMJJHHmmSSCCC)

- les durées "Nombre de jours - Nombre d'heures - Nombre de minutes - Nombre de secondes - Nombre de millièmes de seconde" (format +JHHMMSSCCC)





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Manipulations d'une variable de type DateHeure
DateDébut est une DateHeure = "200112121515"
DateFin est une Date = "20011209"
// Changer la date
DateDébut.PartieDate = "20011225"
// Modifier la partie Date
DateDébut.PartieDate = DateFin
```


<a name="Exemple2"></a>

```wl
// Manipulations d'une rubrique de type DateHeure
MonTravail.DateDébut = "200112121515"
MonTravail.DateFin = "20011209"
// Changer la date
MonTravail.DateDébut.PartieDate = "20011225"
// Modifier la partie Date
MonTravail.DateDébut.PartieDate = MonTravail.DateFin
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître la date d'une variable de type DateHeure

`<Résultat> = <DateHeure>.PartieDate`
---

**`<Résultat> : (Date)`**

Date extraite de la variable de type [DateHeure](../Motscles/1514070.md).

**`<DateHeure> : (DateHeure)`**

Nom de la variable de type [DateHeure](../Motscles/1514070.md) à manipuler.  


<a name="SYNTAXE2"></a>

### Modifier la date d'une variable de type DateHeure

`<DateHeure>.PartieDate = <Nouvelle date>`
---

**`<DateHeure> : (Date ou DateHeure)`**

Nom de la variable de type [DateHeure](../Motscles/1514070.md) à manipuler.

**`<Nouvelle date> : (Chaîne de caractères ou date ou entier)`**

Nouvelle date au format :

- AAAAMMJJ

- Entier : nombre de jours écoulés entre le 1er janvier 1800 et la date spécifiée.




Remplace la date de la variable manipulée.  


<a name="SYNTAXE3"></a>

### Connaître la date d'une rubrique de type Date (format Date et Heure)

`<Résultat> = <Fichier de données>.<Rubrique>.PartieDate`
---

**`<Résultat> : (Date)`**

Date extraite de la rubrique.

**`<Fichier de données> : (Chaîne de caractères)`**

Nom du fichier de données manipulé. Ce nom a été défini sous l'éditeur d'analyses ou grâce au type [Description de Fichier](../WDLang4/1514065.md).

**`<Rubrique> : (Chaîne de caractères)`**

Nom de la rubrique manipulée. Ce nom a été défini sous l'éditeur d'analyses ou grâce au type [Description de Rubrique](../WDLang4/1514071.md).  


<a name="SYNTAXE4"></a>

### Modifier la date d'une variable de type Date (format Date et Heure)

`<Fichier de données>.<Rubrique>.PartieDate = <Nouvelle date>`
---

**`<Fichier de données> : (Chaîne de caractères)`**

Nom du fichier de données manipulé. Ce nom a été défini sous l'éditeur d'analyses ou grâce au type [Description de Fichier](../WDLang4/1514065.md).

**`<Rubrique> : (Chaîne de caractères)`**

Nom de la rubrique manipulée. Ce nom a été défini sous l'éditeur d'analyses ou grâce au type [Description de Rubrique](../WDLang4/1514071.md).

**`<Nouvelle date> : (Chaîne de caractères ou date ou entier)`**

Nouvelle date au format :

- AAAAMMJJ

- Entier : nombre de jours écoulés entre le 1er janvier 1800 et la date spécifiée.




Remplace la date de la date manipulée.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Les types [Date](../Motscles/1514066.md) et [DateHeure](../Motscles/1514070.md) permettent de gérer des dates du 01/01/0001 au 31/12/9999.

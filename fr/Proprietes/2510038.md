
## Enfoncé (Propriété)

***En anglais : Pushed***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Enfoncé** permet de connaître et modifier l'état d'un bouton poussoir (bouton enfoncé ou non).






<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Enfonce le champ Bouton BTN_Bouton1
BTN_Bouton1.Enfoncé = Vrai

// Modification de la taille de la fenêtre
SI BTN_Détail.Enfoncé = Vrai ALORS
	FEN_Fenêtre1.Hauteur = 400
SINON
	FEN_Fenêtre1.Largeur = 200
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître l'état d'un champ Bouton poussoir (on/off)

`<Résultat> = <Champ Bouton>.Enfoncé`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le champ Bouton est enfoncé, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Champ Bouton> : (Nom de champ)`**

Nom du champ Bouton poussoir (option "Bouton On/Off" cochée dans l'onglet "UI" de la description du champ).  


<a name="SYNTAXE2"></a>

### Modifier l'état d'un champ Bouton poussoir (on/off)

`<Champ Bouton>.Enfoncé = <Nouvel état>`
---

**`<Champ Bouton> : (Nom de champ)`**

Nom du champ Bouton poussoir (option "Bouton On/Off" cochée dans l'onglet "UI" de la description du champ).

**`<Nouvel état> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le champ Bouton doit être enfoncé, 

- <u><u><u><u>Faux</u></u></u></u> si le champ Bouton doit être dans son état normal.

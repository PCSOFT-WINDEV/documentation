
## AttributMapping (Propriété)

***En anglais : AttributeMapping***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **AttributMapping** permet de connaître la valeur de l'attribut d'extension &lt;Mapping&gt;. Cette propriété est disponible sur : 

- les variables de type [Définition](../WDLang1/1000019540.md). 

- les variables de type [Description de variable](../WDLang1/1000019560.md).




## Image (Mot-clé)

***En anglais : Image***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le mot-clé**Image** peut correspondre : 

- à une variable de type [Image](../WDLang1/1000019650.md). 

- à la propriété **Image** qui permet de connaître et de modifier l'image associée à un élément. Cette propriété est utilisable sur :

	- un champ. Pour plus de détails, consultez [Propriété Image sur un champ](../Proprietes/2510034.md).

	- une variable de type [EvénementTimeline](../WDLang1/1000020612.md).

	- une variable de type [Marqueur](../WDLang3/1000019940.md).

	- une variable de type [RendezVous](../WDLang1/1000019244.md).

	- une variable de type [rssCanal](../WDLang5/1000017769.md).

	- une variable de type [rssFlux](../WDLang5/1000017795.md).

	- une variable de type [OrgaElément](../WDLang1/1000019713.md).

	- une variable de type [Icône](../WDLang1/1000020811.md).

	- une variable de type [Tiff](../WDLang1/1000021025.md).

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [TâcheGantt](../WDLang1/1000020881.md).

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [upnpImage](../WDLang3/1000020905.md).

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [pdfSignature](../WDLang6/1000022255.md).

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [docElément](../WDLang1/1000022484.md).

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [WDPic](../WDLang1/1000023503.md).

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [CodeBarres](../WDLang5/1000019451.md).

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [picCalque](../WDLang1/1000024604.md).

	- une variable de type [CarteImage](../WDLang3/1000025523.md).

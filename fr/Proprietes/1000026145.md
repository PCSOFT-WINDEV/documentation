
## Document (Propriété)

***En anglais : Document***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Document** permet de connaître ou de modifier le document HTML associé à un champ d'affichage HTML ou à un champ Editeur HTML.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Récupère le code HTML du Body
sCodeHTML est chaîne = HTM_MonDocument.Document.Body.OuterHTML
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître le document HTML

`<Résultat> = <Champ manipulé>.Document`
---

**`<Résultat> : (Variable de type htmlDocument)`**

Variable de type [htmlDocument](../WDLang5/1000025910.md) contenant le document HTML associé au champ. 

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ manipulé : 

- Champ d'affichage HTML.

- Champ Editeur HTML.  





<a name="SYNTAXE2"></a>

### Modifier le document HTML associé à un champ d'affichage HTML

`<Champ manipulé>.Document = <Document HTML>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ manipulé : 

- Champ d'affichage HTML.

- Champ Editeur HTML. 




**`<Document HTML> : (Variable de type htmlDocument)`**

Nom de la variable de type [htmlDocument](../WDLang5/1000025910.md) contenant le document HTML associé au champ. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Cette propriété renvoie une copie du document.

- Les modifications effectuées dans cette copie ne sont donc pas répercutées sur le champ.

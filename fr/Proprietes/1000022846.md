
## StyleID (Propriété)

***En anglais : StyleID***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **StyleID** permet de connaître et de modifier l'identifiant du style associé à un élément. 

Cette propriété est utilisée par les variables de type : 

- [docFragment](../WDLang1/1000022482.md).

- [docParagraphe](../WDLang1/1000022483.md). 

- [docStyle](../WDLang1/1000022486.md).

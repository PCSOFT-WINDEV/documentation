
## LargeurMaxBouton (Propriété)

***En anglais : MaxButtonWidth***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **LargeurMaxBouton** permet de connaître et de définir la largeur maximale d'un bouton d'un volet d'onglet. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
ONG_MonOngletDyn.LargeurMaxBouton = 200
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître la largeur maximale d'un bouton d'un volet d'onglet.

`<Résultat> = <Champ Onglet>.LargeurMaxBouton`
---

**`<Résultat> : (Entier)`**



- 0, aucune limite n'est définie pour la largeur maximale d'un bouton d'un volet d'onglet. 

- Largeur maximale (en pixels) d'un bouton d'un volet d'onglet. 




**`<Champ Onglet> : (Nom de champ)`**

Nom du champ Onglet à manipuler. 


<a name="SYNTAXE2"></a>

### Modifier la largeur maximale d'un bouton d'un volet d'onglet.

`<Champ Onglet>.LargeurMaxBouton = <Largeur>`
---

**`<Champ Onglet> : (Nom de champ)`**

Nom du champ Onglet à manipuler.

**`<Largeur> : (Entier)`**



- Largeur maximale (en pixels) d'un bouton d'un volet d'onglet. 

- 0 pour ne donner aucune limite. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- La propriété **LargeurMaxBouton** permet de limiter la taille des boutons lorsque le le champ Onglet utilise l'option "Volets de taille : Proportionnelle au libellé" (onglet "Détails" de la fenêtre de description du champ). 

- Cas des onglets dynamiques : Le libellé de l'onglet est calculé automatiquement. La propriété **LargeurMaxBouton** permet d'éviter des libellés trop longs et donc des boutons trop larges. 

- Si le libellé des boutons est plus large que la taille spécifiée, le libellé est affiché avec une ellipse en fin ("...").

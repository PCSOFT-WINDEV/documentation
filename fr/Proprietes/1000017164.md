


## Description (Propriété)

***En anglais : Description***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Description** permet de connaître et modifier la description associée à la page. La description apparaît (souvent tronquée) dans l'affichage du résultat de la recherche dans les moteurs de recherche. La description peut être également saisie depuis l'éditeur de pages.

La propriété **Description** permet dans le cas des pages AWP de saisir dynamiquement la description associée à la page (par exemple en fonction du contenu de la page).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Modifie la description de la page en fonction de son contenu
MaPage.Description = "Vente de bateaux en ligne"
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître la description associée à une page

`<Description de la page> = <Page manipulée>.Description`
---

**`<Description de la page> : (Chaîne de caractères)`**

Phrase utilisée pour la description de la page.

**`<Page manipulée> : (Nom de page)`**

Nom de la page à analyser.  


<a name="SYNTAXE2"></a>

### Modifier la description associée à une page 

`<Page manipulée>.Description = <Description de la page>`
---

**`<Page manipulée> : (Nom de page)`**

Nom de la page pour laquelle la description doit être définie.

**`<Description de la page> : (Chaîne de caractères)`**

Description à associer à la page. Cette phrase doit inciter l'internaute à venir sur la page. Cette chaîne doit respecter les règles suivantes :

- Ne pas contenir de Retour Chariot (RC)

- Être courte (200 caractères maximum)  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Description associée à une page
<a name="description_associee_une_page_ELTPARAGRAPHE000045"></a>La description associée à une page est écrite dans l'entête HTML de la page (balise &lt;META NAME="Description" CONTENT="..."&gt;). Cette description est utilisée par les moteurs de recherche : cette valeur est affichée comme description de la page dans la liste des résultats de la recherche.

La description doit être définie dans le code d'initialisation de la page. De plus, pour être efficace, la description doit être définie sur des pages accessibles par une URL de la forme "http://".


## NbMaxSegment (Propriété)

***En anglais : MaxNbSegment***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **NbMaxSegment** permet de connaître et de modifier le nombre maximum de segments de droite pour dessiner un champ Fil.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// On ajoute un point de passage au fil pour qu'il prenne le chemin que l'on souhaite
PointPassage1, PointPassage2 sont des Points 

PointPassage1.X = 280
PointPassage1.Y = 54

PointPassage2.X = 320
PointPassage2.Y = 54
FIL_MonFil.NbMaxSegment = 20

FIL_MonFil.PointDePassage = [PointPassage1, PointPassage2]
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître le nombre maximum de segments de droite d'un champ Fil

`<Résultat> = <Champ Fil>.NbMaxSegment`
---

**`<Résultat> : (Entier)`**



- Nombre maximum de segments de droite constituant le champ Fil, 

- 1 correspond au dessin d'une seul droite entre deux champs. 




**`<Champ Fil> : (Nom de champ)`**

Nom du champ Fil à manipuler.


<a name="SYNTAXE2"></a>

### Modifier le nombre maximum de segments de droite d'un champ Fil

`<Champ Fil>.NbMaxSegment = <Nouveau nombre maximum>`
---

**`<Champ Fil> : (Nom de champ)`**

Nom du champ Fil à manipuler.

**`<Nouveau nombre maximum> : (Entier)`**



- Nombre maximum de segments de droite constituant le champ Fil, 

- 1 correspond au dessin d'une seul droite entre deux champs. 




Remarque : Si il est possible de dessiner le champ Fil avec moins de segments, le tracé est automatiquement simplifié.




## Style (Propriété)

***En anglais : Style***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Style** permet de récupérer ou d'affecter le style complet d'un champ (y compris les surcharges). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Copie de styles entre 2 champs de saisie
SAI_SaisiePrix.Style = SAI_Exemple.Style
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Récupérer le style complet d'un champ

`<Résultat> = <Champ manipulé>.Style`
---

**`<Résultat> : (Buffer)`**

Buffer contenant le style complet du champ. Ce buffer peut par exemple : 

- être utilisé pour copier le style d'un champ dans un autre. 

- être sauvé dans un fichier ou une rubrique mémo afin de restaurer le style plus tard. 




**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler. 


<a name="SYNTAXE2"></a>

### Modifier le style d'un champ

`<Champ manipulé>.Style = <Nouveau style>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler.

**`<Nouveau style> : (Buffer)`**

Buffer contenant le nouveau style complet du champ. 

**Attention** : 

- Le style utilisé doit correspondre au style d'un champ de même type. Dans le cas contraire, une erreur WLangage est affichée. 

- Si le style utilise des images, ces images doivent être présentes dans la WDL ou accessibles sur le disque.




## CodeHTML (Propriété)

***En anglais : CodeHTML***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **CodeHTML** permet de lire et modifier le code HTML associé à un champ Composant Web.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
Trace(CMP_MonComposantWeb.CodeHTML)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître le code HTML associé au champ Composant Web

`<Résultat> = <Champ Composant Web>.CodeHTML`
---

**`<Résultat> : (Chaîne de caractères)`**

Code HTML associé au champ Composant Web. Ce code HTML sera écrit dans la page. 

**`<Champ Composant Web> : (Nom de champ)`**

Nom du champ Composant Web à manipuler. 


<a name="SYNTAXE2"></a>

### Modifier le code HTML associé au champ Composant Web

`<Champ Composant Web>.CodeHTML = <Code HTML>`
---

**`<Champ Composant Web> : (Nom de champ)`**

Nom du champ Composant Web à manipuler. 

**`<Code HTML> : (Chaîne de caractères)`**

Nouveau code HTML associé au champ.


## Sujet (Propriété)

***En anglais : Subject***
	

<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Sujet** permet de connaître et de modifier le sujet d'un élément. 

Cette propriété est utilisable sur :

- la structure [Email](../WDLang3/3032029.md).

- une variable de type [Email](../WDLang3/1000018713.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [docPropriété](../WDLang1/1000022945.md).

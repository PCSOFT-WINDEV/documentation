
## PoliceSoulignée (Propriété)

***En anglais : FontUnderlined***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **PoliceSoulignée** permet de gérer le soulignement d'un texte. Cette propriété est utilisable sur : 

- les champs. Pour plus de détails, consultez [PoliceSoulignée](../Proprietes/2510099.md). 

- une variable de type [docMiseEnForme](../WDLang1/1000022817.md).

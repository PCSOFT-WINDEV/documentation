
## MiseEnForme (Propriété)

***En anglais : Formatting***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **MiseEnforme** permet de décrire la mise en forme d'un élément. Cette propriété est utilisable : 

- sur une variable de type [docElémentFormule](../WDLang1/1000022937.md). 

- sur une variable de type [docElémentTexte](../WDLang1/1000022493.md). 

- sur une variable de type [docFragment](../WDLang1/1000022482.md).

- sur une variable de type [docNiveauNumérotation](../WDLang1/1000022799.md).

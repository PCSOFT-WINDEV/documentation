


## DeviseAffichage (Propriété)

***En anglais : DisplayCurrency***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **DeviseAffichage** permet de connaître et modifier la devise utilisée lors de l'affichage et/ou de la saisie dans un champ de type "Monétaire + Euro".

Remarques : 

- La devise affichée peut être différente de la devise retournée au programme ou enregistrée dans le fichier de données HFSQL correspondant. La devise utilisée lors de l'enregistrement est définie par la propriété [DeviseMémoire](../Proprietes/2510013.md).

- Si la devise affichée et la devise d'enregistrement sont différentes, la conversion est automatiquement réalisée lors de l'enregistrement. 





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple

![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) 
```wl
// Modification de la devise d'affichage dans le champ de saisie SAI_Saisie1
SAI_Saisie1.DeviseAffichage = eITL	// Lire italienne
```

## Remarques


### Devise supplémentaire
<a name="devise_supplementaire_ELTPARAGRAPHE000353"></a>Il est possible de définir des devises utilisateurs : elles permettent de choisir une devise supplémentaire, créée par programmation avec la fonction [eInitDevise](../WDLang1/3033003.md). Pour choisir une de ses devises, utilisez les valeurs 16 à 19 :

- 16 : Devise personnalisée numéro 1

- 17 : Devise personnalisée numéro 2

- 18 : Devise personnalisée numéro 3

- 19 : Devise personnalisée numéro 4



<a name="NOTE0_2"></a>


### Divers
<a name="divers_ELTPARAGRAPHE000368"></a>Lors de la modification de la devise d'affichage d'un champ, la valeur contenue dans le champ est automatiquement convertie.
<a name="NOTE0_3"></a>


### Limites
<a name="limites_ELTPARAGRAPHE000375"></a>La propriété **DeviseAffichage** s'applique :

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) lors de l'affichage d'une valeur dans un champ calculé de type "Monétaire + Euro" (dans un état).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) lors de l'affichage et la saisie d'une valeur dans un champ de saisie de type "Monétaire + Euro" (dans une fenêtre).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) lors de l'affichage et la saisie d'une valeur dans une colonne d'un champ Table de type "Monétaire + Euro" (dans une fenêtre).

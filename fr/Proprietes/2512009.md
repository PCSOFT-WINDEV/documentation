


## Abréviation (Propriété)

***En anglais : Abbreviation***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Abréviation** permet d'obtenir l'abréviation du fichier de données. Cette abréviation a été définie sous l'éditeur d'analyses lors de la description du fichier de données. Cette propriété est utilisable en lecture uniquement.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Abréviation du fichier de données Commande
ResAbréviation = Commande.Abréviation
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Abréviation du fichier de données> = <Fichier de données>.Abréviation`
---

**`<Abréviation du fichier de données> : (Chaîne de caractères)`**

Deux lettres permettant d'identifier l'abréviation du fichier de données.

**`<Fichier de données> : (Chaîne de caractères)`**

Nom du fichier de données à manipuler.

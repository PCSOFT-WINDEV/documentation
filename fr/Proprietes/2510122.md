


## TotauxActif (Propriété)

***En anglais : TotalsEnabled***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **TotauxActif** permet de :

- Connaître le mode de fonctionnement des calculs de colonnes (automatiques ou personnalisés) pour un champ Table ou Table hiérarchique,

- D'interdire temporairement ou de forcer les calculs de colonnes (automatiques ou personnalisés) dans un champ Table ou Table hiérarchique.




Remarques : 

- Pour plus de détails sur la mise en place et la programmation des calculs dans les champs Table, consultez : 

	- [Effectuer des calculs automatiques dans les colonnes des champs Table](../WDChamp/1013201.md).

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) [Effectuer des calculs personnalisés dans les colonnes des champs Table](../WDChamp/1013331.md).




- Si la propriété [AffichageActif](../Proprietes/2510015.md) est utilisée sur le champ Table et correspond à <u><u><u>Faux</u></u></u>, les calculs ne sont pas mis à jour. 



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
TABLE_TABLE1.TotauxActif = Faux // plus de totaux calculés
POUR I = 1 A 50
	TableAjoute(TABLE_TABLE1, MonCalculContenuLigne(I))
	Multitache(-1) // la table est réaffichée sans les totaux
FIN
TABLE_TABLE1.TotauxActif = Vrai
Trace(" Total = " + COL_COLTOTAL[ligneTotal])
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le mode de fonctionnement des calculs de colonnes (automatiques ou personnalisés) 

`<Résultat> = <Champ manipulé>.TotauxActif`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si les calculs de colonnes (automatiques ou personnalisés) sont effectués immédiatement, 

- <u><u><u><u>Faux</u></u></u></u> si les calculs de colonnes (automatiques ou personnalisés) ne sont pas effectués.




**`<Champ manipulé> : (Nom de champ)`**

Nom du champ Table ou Table hiérarchique contenant des calculs de colonnes (automatiques ou personnalisés).  


<a name="SYNTAXE2"></a>

### Modifier le mode de fonctionnement des calculs de colonnes (automatiques ou personnalisés) 

`<Champ manipulé>.TotauxActif = <Nouveau mode de fonctionnement>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ Table ou Table hiérarchique contenant des calculs de colonnes (automatiques ou personnalisés).

**`<Nouveau mode de fonctionnement> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> pour forcer l'exécution des calculs de colonnes (automatiques ou personnalisés) dans le champ, 

- <u><u><u><u>Faux</u></u></u></u> pour ne pas effectuer les calculs de colonnes (automatiques ou personnalisés).

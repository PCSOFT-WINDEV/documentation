
## AffichageLED (Propriété)

***En anglais : DisplayLED***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **AffichageLED** permet de gérer l'affichage de la LED lors de l'affichage des notifications. Cette propriété est utilisable sur :

- une variable de type [Notification](../WDLang3/1000019441.md).

- une variable de type [notificationCatégorie](../WDLang3/1000023981.md).


## Indice (Propriété)

***En anglais : Index***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Indice** permet de connaître l'indice d'un élément. Cette propriété est utilisable sur :

- une rubrique d'un enregistrement. Pour plus de détails, consultez l'aide de la propriété [Indice](../Proprietes/2512040.md).

- une variable de type [htmlNoeud](../WDLang5/1000026081.md).

- une variable de type [kbCarte](../WDLang1/1410089167.md).

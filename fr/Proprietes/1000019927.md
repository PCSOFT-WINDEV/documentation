


## Position

***En anglais : Position***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le mot-clé **Position** peut correspondre :

- à la fonction **[Position](../WDLang1/3024027.md)** qui permet de rechercher la position d'une chaîne de caractères dans une autre chaîne de caractères.

- à la propriété **Position** disponible sur :

	- une variable de type [Adresse](../WDLang3/1000019944.md).

	- une variable de type [Marqueur](../WDLang3/1000019940.md).

	- une variable de type [Filigrane](../WDLang5/1000019922.md).

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [docTabulation](../WDLang1/1000022573.md). 

	- une variable de type [CarteImage](../WDLang3/1000025523.md).

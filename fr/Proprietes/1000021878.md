


## ChampArrivée (Propriété)

***En anglais : EndControl***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **ChampArrivée** permet de connaître ou de modifier le champ d'arrivée d'un champ Fil.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Création et initialisation d'un champ Fil 
// Ce fil relie le champ BTN_ChampA et le champ BTN_ChampB
NomFil est un Champ
NomFil <- ChampCrée("", typFil)
NomFil.ChampDépart = BTN_ChampA
NomFil.ChampArrivée = BTN_ChampB
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître le champ d'arrivée d'un champ Fil

`<Résultat> = <Champ manipulé>.ChampArrivée`
---

**`<Résultat> : (Chaîne de caractères)`**

Nom du champ d'arrivée. 

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ Fil. 


<a name="SYNTAXE2"></a>

### Modifier le champ d'arrivée d'un champ Fil

`<Champ manipulé>.ChampArrivée = <Nouveau champ d'arrivée>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ Fil. 

**`<Nouveau champ d'arrivée> : (Chaîne de caractères)`**

Nom du nouveau du champ d'arrivée. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Le fil est affiché lorsque les propriétés [ChampDépart](../Proprietes/1000021877.md) et **ChampArrivée** sont renseignées et visibles. 

- Les points d'attache utilisés peuvent être modifiés à l'aide des propriétés [PointDAttacheDépart](../Proprietes/1000021907.md) et [PointDAttacheArrivée](../Proprietes/1000021908.md).

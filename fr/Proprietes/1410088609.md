
## OutilInitial (Propriété)

***En anglais : InitialTool***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **OutilInitial** permet de connaître ou de modifier l'outil de dessin initial sélectionné dans le champ Editeur d'images.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
EI_EditeurImage.OutilInitial = outilFlèche
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître l'outil de dessin initial du champ Editeur d'images

`<Résultat> = <Champ Editeur d'images>.OutilInitial`
---

**`<Résultat> : (Constante de type Entier)`**

Outil sélectionné par défaut dans le champ Editeur d'images : 



|   |   |
| --- | --- |
| *outilCrayon* | Par défaut, l'outil sélectionné dans le champ Editeur d'images est le crayon. |
| *outilEllipse* | Par défaut, l'outil sélectionné dans le champ Editeur d'images est l'ellipse. |
| *outilFiligrane* | Par défaut, l'outil sélectionné dans le champ Editeur d'images est le filigrane. |
| *outilFlèche* | Par défaut, l'outil sélectionné dans le champ Editeur d'images est la flèche. |
| *outilFlou* | Par défaut, l'outil sélectionné dans le champ Editeur d'images est le floutage. |
| *outilGomme* | Par défaut, l'outil sélectionné dans le champ Editeur d'images est la gomme. |
| *outilOCR* | Par défaut, l'outil sélectionné dans le champ Editeur d'images est l'OCR. |
| *outilRecadrage* | Par défaut, l'outil sélectionné dans le champ Editeur d'images est l'outil de recadrage. |
| *outilRectangle* | Par défaut, l'outil sélectionné dans le champ Editeur d'images est le rectangle. |
| *outilRemplissage* | Par défaut, l'outil sélectionné dans le champ Editeur d'images est le remplissage. |
| *outilTexte* | Par défaut, l'outil sélectionné dans le champ Editeur d'images est le texte. |
| *outilTrait* | Par défaut, l'outil sélectionné dans le champ Editeur d'images est le trait. |



**`<Champ Editeur d'images> : (Nom de champ)`**

Nom du champ Editeur d'images manipulé.


<a name="SYNTAXE2"></a>

### Modifier l'outil de dessin initial du champ Editeur d'images

`<Champ Editeur d'images>.OutilInitial = <Outil par défaut>`
---

**`<Champ Editeur d'images> : (Nom de champ)`**

Nom du champ Editeur d'images manipulé.

**`<Outil par défaut> : (Constante de type Entier)`**

Outil à sélectionner par défaut dans le champ Editeur d'images : 



|   |   |
| --- | --- |
| *outilCrayon* | Par défaut, l'outil sélectionné dans le champ Editeur d'images est le crayon. |
| *outilEllipse* | Par défaut, l'outil sélectionné dans le champ Editeur d'images est l'ellipse. |
| *outilFiligrane* | Par défaut, l'outil sélectionné dans le champ Editeur d'images est le filigrane. |
| *outilFlèche* | Par défaut, l'outil sélectionné dans le champ Editeur d'images est la flèche. |
| *outilFlou* | Par défaut, l'outil sélectionné dans le champ Editeur d'images est le floutage. |
| *outilGomme* | Par défaut, l'outil sélectionné dans le champ Editeur d'images est la gomme. |
| *outilOCR* | Par défaut, l'outil sélectionné dans le champ Editeur d'images est l'OCR. |
| *outilRecadrage* | Par défaut, l'outil sélectionné dans le champ Editeur d'images est l'outil de recadrage. |
| *outilRectangle* | Par défaut, l'outil sélectionné dans le champ Editeur d'images est le rectangle. |
| *outilRemplissage* | Par défaut, l'outil sélectionné dans le champ Editeur d'images est le remplissage. |
| *outilTexte* | Par défaut, l'outil sélectionné dans le champ Editeur d'images est le texte. |
| *outilTrait* | Par défaut, l'outil sélectionné dans le champ Editeur d'images est le trait. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Pour modifier les caractéristiques par défaut du champ Editeur d'images, il est également possible d'utiliser les propriétés [LargeurInitiale](../Proprietes/2510067.md) et [HauteurInitiale](../Proprietes/2510043.md) pour définir la taille de l'image créée par défaut dans le champ Editeur d'images.

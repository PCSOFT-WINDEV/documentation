


## NomFichierServeur (Propriété)

***En anglais : NameServerFile***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **NomFichierServeur** permet de connaître le nom du fichier uploadé présent sur le serveur. Cette propriété est disponible uniquement pour les champs Upload multi-fichiers.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Les fichiers uploadés sont copiés dans un répertoire sur le serveur avec leur nom d'origine
POUR i = 1 _A_ UPL_Upload.Occurrence
	UploadCopieFichier(UPL_Upload, gsCheminFichiersUploadés, "", i)
	Trace(UPL_Upload[i].NomFichierNavigateur, UPL_Upload[i].NomFichierServeur)
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le nom du fichier uploadé présent sur le serveur

`<Résultat> = <Champ Upload>[<N>].NomFichierServeur`
---

**`<Résultat> : (Chaîne de caractères)`**

Nom du fichier uploadé (c'est-à-dire nom du fichier présent sur le serveur).

**`<Champ Upload> : (Nom de champ)`**

Nom du champ Upload à manipuler.

**`<N> : (Entier)`**

Numéro du fichier considéré présent dans le champ Upload.

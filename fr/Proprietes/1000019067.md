


## EtendueVisible (Propriété)

***En anglais : VisibleRange***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **EtendueVisible** permet de connaître et de modifier la largeur de la cage d'ascenseur du champ Range Slider. <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=RangeSlider_Etendue.gif)
L'étendue visible peut également être paramétrée : 

- dans la fenêtre de description du champ Range Slider (onglet "Général"). 

- par l'utilisateur lorsqu'il modifie la place des extrémités de la cage d'ascenseur. Dans ce cas, l'événement "Modification de l'étendue" du champ Range Slider est automatiquement exécuté. 



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affiche les propriétés du champ Range Slider
SAI_Valeur = MoiMême.Valeur
SAI_BORNE_MIN = MoiMême.BorneMin
SAI_BORNE_MAX = MoiMême.BorneMax
SAI_ETENDUE_TOTALE = MoiMême.EtendueTotale
SAI_ETENDUE_VISIBLE = MoiMême.EtendueVisible
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître l'étendue visible du champ Range Slider

`<Résultat> = <Champ Range Slider>.EtendueVisible`
---

**`<Résultat> : (Entier)`**

Etendue visible du Range Slider.

**`<Champ Range Slider> : (Nom de champ)`**

Nom du champ Range Slider manipulé.


<a name="SYNTAXE2"></a>

### Modifier l'étendue visible du champ Range Slider

`<Champ Range Slider>.EtendueVisible = <Nouvelle étendue visible>`
---

**`<Champ Range Slider> : (Nom de champ)`**

Nom du champ Range Slider manipulé.

**`<Nouvelle étendue visible> : (Entier)`**

Nouvelle étendue visible du Range Slider. 
**Remarque** : Pour un Range Slider horizontal, seule l'extrémité droite est modifiée. Pour un range Slider vertical, seule l'extrémité basse est modifiée. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Lors de la modification de l'étendue visible du Range Slider par la propriété **EtendueVisible**, l'événement "Modification de l'étendue" du champ Range Slider n'est pas exécuté.

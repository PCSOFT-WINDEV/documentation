
## TimeoutConnexion (Propriété)

***En anglais : ConnectionTimeout***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **TimeoutConnexion** permet de gérer le temps d'attente de connexion. Cette propriété est disponible sur :

- une variable de type [sshSession](../WDLang3/1000021157.md).

- une variable de type [httpRequête](../WDLang3/1000021158.md).

- une variable de type [restRequête](../WDLang3/1000021481.md).

- une variable de type [mqttSession](../WDLang3/1000023023.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [apiRequête](../WDLang3/1410088935.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [memcachedConnexion](../WDLang4/1000023339.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [redisConnexion](../WDLang4/1000023527.md).

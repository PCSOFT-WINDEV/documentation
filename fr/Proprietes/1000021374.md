


## AvecBoutonFermeture (Propriété)

***En anglais : WithClosingButton***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **AvecBoutonFermeture** permet de : 

- Savoir si tous les volets d'un onglet dynamique possèdent une croix de fermeture. 

- Configurer tous les volets d'un onglet dynamique pour afficher une croix de fermeture. 




Rappel : La croix de fermeture permet à l'utilisateur final de fermer un volet d'onglet. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
ONG_OngletDynamique.AvecBoutonFermeture = Vrai
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Savoir si tous les volets d'un onglet dynamique possèdent une croix de fermeture

`<Résultat> = <Champ Onglet>.AvecBoutonFermeture`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si tous les volets d'un champ Onglet dynamique possèdent une croix de fermeture,

- <u><u><u><u>Faux</u></u></u></u> si les volets d'un champ Onglet dynamique ne possèdent pas une croix de fermeture. 




**`<Champ Onglet> : (Nom de champ)`**

Nom du champ Onglet à manipuler. 


<a name="SYNTAXE2"></a>

### Configurer tous les volets d'un onglet dynamique pour afficher une croix de fermeture

`<Champ Onglet>.AvecBoutonFermeture = <Croix de fermeture>`
---

**`<Champ Onglet> : (Nom de champ)`**

Nom du champ Onglet à manipuler. 

**`<Croix de fermeture> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si les volets d'un champ Onglet dynamique doivent afficher une croix de fermeture,

- <u><u><u><u>Faux</u></u></u></u> si les volets d'un champ Onglet dynamique ne doivent pas afficher une croix de fermeture. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Si les volets possèdent une croix de fermeture, lorsque l'utilisateur clique sur cette croix l'événement "Fermeture d'un volet" est exécuté. Si cet événement n'empêche pas la fermeture du volet (instruction "RENVOYER Faux" ou fonction [DonneFocusEtRetourUtilisateur](../WDLang1/1410088107.md)), ce dernier est fermé.

- Si les volets ne possèdent pas de croix de fermeture, les Fonctionnalités Automatiques de vos Applications (FAA) suivantes présentes dans le menu contextuel du volet sont désactivées :

	- Fermer,

	- Fermer tous les autres,

	- Fermer tous à droite,

	- Fermer tous à gauche.




- Il n'est pas possible de supprimer ou d'afficher la croix de fermeture sur un seul volet d'onglet : soit tous les volets d'onglet affichent la croix de fermeture, soit aucun.

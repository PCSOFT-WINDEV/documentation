
## ModeAffichageAutoAuChargement (Propriété)

***En anglais : DisplayModeAutoWhenLoading***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **ModeAffichageAutoAuChargement** permet de connaître et de modifier le comportement du mode d'affichage lors du chargement d'un document PDF. Le mode d'affichage (1 page, 2 pages, 2 pages avec couverture) utilisé peut correspondre : 

- soit au mode d'affichage spécifié dans le champ. 

- soit au mode d'affichage préféré spécifié dans le fichier PDF.




Par défaut, le champ Lecteur PDF respecte les préférences d'affichage du fichier PDF. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Force l'affichage du document en mode 1 page
PDF_MonLecteurPDF.ModeAffichageAutoAuChargement = Faux
PDF_MonLecteurPDF.ModeAffichage = modeAffichageUnePage
LecteurPDFOuvre(PDF_MonLecteurPDF, "MonDocument.pdf")
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître le comportement du mode d'affichage par rapport aux préférences du document PDF

`<Résultat> = <Champ Lecteur PDF>.ModeAffichageAutoAuChargement`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le mode d'affichage utilisé lors du chargement d'un document PDF dans le champ est celui spécifié dans le document PDF. Le mode d'affichage utilisé sera automatiquement modifié pour correspondre aux préférences précisées dans le fichier PDF.  

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Le mode d'affichage utilisé ne sera pas changé lors de l'ouverture d'un fichier PDF. 




**`<Champ Lecteur PDF> : (Nom de champ)`**

Nom du champ Lecteur PDF à manipuler.


<a name="SYNTAXE2"></a>

### Modifier le comportement du mode d'affichage par rapport aux préférences du document PDF

`<Champ Lecteur PDF>.ModeAffichageAutoAuChargement = <Nouveau mode>`
---

**`<Champ Lecteur PDF> : (Nom de champ)`**

Nom du champ Lecteur PDF à manipuler.

**`<Nouveau mode> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le mode d'affichage utilisé lors du chargement d'un document PDF dans le champ est celui spécifié dans le document PDF. Le mode d'affichage utilisé sera automatiquement modifié pour correspondre aux préférences précisées dans le fichier PDF.  

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Le mode d'affichage utilisé ne sera pas changé lors de l'ouverture d'un fichier PDF.

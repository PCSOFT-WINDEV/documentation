


## HauteurPixel (Propriété)

***En anglais : HeightInPixel***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **HauteurPixel** permet de connaître la hauteur d'un champ d'un état en pixels.

**Remarque** : Le cadre du champ est compris dans sa hauteur. La propriété [Hauteur](../Proprietes/2510050.md) permet de connaître et de modifier la hauteur d'un champ en millimètres.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connaître la hauteur du champ "RUB_NomClient"
ResHauteur = RUB_NomClient.HauteurPixel
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Champ manipulé>.HauteurPixel`
---

**`<Résultat> : (Réel)`**

Hauteur (en pixels) du champ spécifié.

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ de l'état à manipuler.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Hauteur
<a name="hauteur_ELTPARAGRAPHE000055"></a>Un champ doit être entièrement contenu dans le bloc auquel il appartient.
<a name="NOTE0_2"></a>


### Limites
<a name="limites_ELTPARAGRAPHE000062"></a>La propriété **HauteurPixel** ne peut pas être utilisée sur :

- un état dans son ensemble,

- un bloc de l'état,

- un groupe de champs,

- un champ Interrupteur, 

- un champ de saisie, 

- un champ Agenda,

- un champ Planning, 

- un champ Diagramme de Gantt, 

- un champ Signature.

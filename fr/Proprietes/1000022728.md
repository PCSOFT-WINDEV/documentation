
## MargeHaute (Propriété)

***En anglais : TopMargin***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **MargeHaute** permet de définir la marge haute d'un élément. Cette propriété est utilisable : 

- sur les états. Pour plus de détails, consultez [MargeHaute](../Proprietes/2511043.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) sur les variables de type [docSection](../WDLang1/1000022685.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) sur les variables de type [xlsMiseEnPage](../WDLang5/1000023139.md).

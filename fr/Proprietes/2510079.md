


## ModeListeImage (Propriété)

***En anglais : ListViewMode***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **ModeListeImage** permet de :

- Connaître le mode d'affichage d'un champ Liste image. 

- Modifier le mode d'affichage d'un champ Liste image.




**Remarque** : Le mode d'affichage d'un champ Liste image peut également être modifié sous l'éditeur, dans la fenêtre de description du champ (onglet "Détail").

**Rappel** : Un champ Liste image peut être affiché : 

- en mode classique : le champ est affiché sans les images. 

- en mode liste image : le champ est affiché avec les images. 

- en mode panorama : le champ est affiché avec les images comme si elles étaient présentes sur un mur incurvé. Dans ce mode, seules 20 images maximum peuvent être affichées.  



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Passage en mode classique
LSI_ListeImage1.ModeListeImage = mliListeSimple
```

<a name="SYNTAXE"></a>

### Connaître le mode d'affichage d'un champ Liste image

`<Résultat> = <Champ Liste image>.ModeListeImage`
---

**`<Résultat> : (Constante de type Entier)`**

Mode d'affichage du champ Liste image : 


|   |   |
| --- | --- |
| *mliListeImage* | Champ Liste image affiché en mode liste image. |
| *mliListeSimple* | Champ Liste image affiché en mode liste classique. |
| *mliPanorama* | Champ Liste image affiché en mode panorama. |





**`<Champ Liste image> : (Nom de champ)`**

Nom du champ Liste image à manipuler.  


<a name="SYNTAXE4"></a>

### Modifier le mode d'affichage d'un champ Liste image

`<Champ Liste image>.ModeListeImage = <Nouveau mode d'affichage>`
---

**`<Champ Liste image> : (Nom de champ)`**

Nom du champ Liste image à manipuler.

**`<Nouveau mode d'affichage> : (Constante de type Entier)`**

Mode d'affichage du champ Liste image : 


|   |   |
| --- | --- |
| *mliListeImage* | Champ Liste image affiché en mode liste image. |
| *mliListeSimple* | Champ Liste image affiché en mode liste classique. |
| *mliPanorama* | Champ Liste image affiché en mode panorama. |




<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
La propriété **ModeListeImage** peut uniquement être utilisée sur les champs de type Liste image.

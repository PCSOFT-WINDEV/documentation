
## Animation (Propriété)

***En anglais : Animation***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Animation** permet de gérer l'animation d'une image. 

Il est possible de :

- Connaître l'état actuel de l'image animée ou du texte animé.

- Lancer ou arrêter l'animation d'une image ou d'un texte (par exemple, l'animation d'une image de type GIF animée).




Cette propriété s'applique aux champs suivants :

- ![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png) Champs Image,

- ![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png) Champs Bouton,

- ![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Champs Libellé,

- ![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png) Champs Carrousel,

- ![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png) Champs Jauge.




![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) La propriété **Animation** permet également l'animation d'un texte dans un champ Libellé.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple

![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png) 
```wl
// Initialisation du champ Image avec Flèche.GIF (GIF animé)
IMG_Image1 = "C:\Temp\Flèche.gif"
// Lancement de l'animation
IMG_Image1.Animation = Vrai
```


<a name="Exemple2"></a>


![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) 
```wl
// Initialisation du champ Libellé avec le texte "WINDEV est formidable"
LIB_Libellé1 = "WINDEV est formidable"
// Lancement de l'animation
LIB_Libellé1.Animation = Vrai
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître l'état actuel de l'image animée ou du texte animé

`<Etat> = <Champ manipulé>.Animation`
---

**`<Etat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'image ou le texte actuellement présent dans le champ est animé, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Champ manipulé> : (Nom de champ)`**

Nom du champ manipulé.  


<a name="SYNTAXE2"></a>

### Lancer ou arrêter l'animation

`<Champ manipulé>.Animation = <Mode d'animation>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ manipulé.

**`<Mode d'animation> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> pour lancer l'animation, 

- <u><u><u><u>Faux</u></u></u></u> pour arrêter l'animation.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png) 

### Animation d'une image
<a name="animation_une_image_ELTPARAGRAPHE000105"></a>Pour animer l'image affichée dans le **champ Image**, cette image peut être :

- soit une image de type animé (par exemple, Gif animé).

- soit une image animée par WINDEV. Dans ce cas, le type de l'animation peut être choisi soit dans la fenêtre de description de l'image (onglet "Détail"), soit par programmation avec la fonction [AnimationChangeType](../WDLang1/3017001.md).




Dans un champ Bouton, seule une image de type animée peut être associée au champ (par exemple Gif Animé).
<a name="NOTE0_2"></a>
![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) 

### Animation d'un texte
<a name="animation_texte_ELTPARAGRAPHE000122"></a>Pour animer un texte dans un champ Libellé, il est nécessaire de choisir l'animation effectuée :

- soit dans la fenêtre de description du libellé (onglet "Détail"),

- soit par programmation avec la fonction [AnimationChangeType](../WDLang1/3017001.md).



<a name="NOTE0_3"></a>
![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png) 

### Animation d'une jauge
<a name="animation_une_jauge_ELTPARAGRAPHE000137"></a>Dans une jauge, l'animation se fait lors d'une variation importante de la valeur de la jauge : au lieu de se positionner immédiatement sur la nouvelle valeur, la jauge est incrémentée (ou décrémentée) progressivement pour obtenir un effet d'animation.

Remarque : cet effet est particulièrement adapté pour les jauges circulaires.


<a name="NOTE0_4"></a>


### Divers
<a name="divers_ELTPARAGRAPHE000147"></a>Par défaut, l'animation est à <u><u><u><u>Vrai</u></u></u></u>.

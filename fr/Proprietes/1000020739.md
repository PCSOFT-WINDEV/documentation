
## Clé (Propriété)

***En anglais : Key***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Clé** permet de gérer les clés :

- dans une variable de type [xmlSignature](../WDLang3/1000020736.md). 

- dans une variable de type [memcachedRésultat](../WDLang4/1000023359.md).

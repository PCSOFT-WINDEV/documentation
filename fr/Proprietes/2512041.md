


## InfosEtendues (Propriété)

***En anglais : ExtendedInfo***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **InfosEtendues** permet de connaître les informations supplémentaires :

- d'un fichier de données accédé par un Connecteur Natif ou HFSQL Client/Serveur.

- d'une rubrique accédé par un Connecteur Natif ou HFSQL Client/Serveur.

- d'une connexion. Dans ce cas, cette propriété est équivalente à la propriété [CxInfosEtendues](../Proprietes/2512017.md).




Ces informations étendues peuvent être modifiées pour un fichier, une connexion ou une rubrique définis par programmation.

Pour plus de détails sur ces informations étendues, consultez l'aide du fournisseur de la base de données utilisée.


**Important** : Actuellement, cette propriété est utilisable uniquement avec des fichiers de données (et des rubriques) AS/400 et des connexions HFSQL Client/Serveur. Sur les connexions HFSQL Client/Serveur, seul le cryptage peut être géré par la propriété **InfosEtendues**. 


Ces informations sont affichées sous l'éditeur d'analyses :

- dans la fenêtre de description d'un fichier de données.

- dans la fenêtre de description des rubriques d'un fichier de données.

- dans la fenêtre de description d'une connexion : sous le volet "Analyse", dans le groupe "Connexion", cliquez sur "Connexions".




Cette propriété peut être utilisée :

- en lecture pour un fichier ou une rubrique défini sous l'éditeur d'analyses ou défini par programmation.

- en écriture, uniquement pour un fichier de données défini par programmation ou une connexion.



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connaître les informations étendues du fichier de données Client
Info(Client.InfosEtendues)
// Connaître les informations étendues de la rubrique NomClient du fichier de données Client
Info(Client.NomClient.InfosEtendues)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître les informations étendues

`<Informations étendues> = <Fichier de données/Rubrique/Connexion>.InfosEtendues`
---

**`<Informations étendues> : (Chaîne de caractères)`**

Informations étendues de l'élément spécifié. Les différentes informations sont séparées par des points-virgule (';').

**`<Fichier de données/Rubrique/Connexion> : (Chaîne de caractères)`**



- Nom du fichier de données manipulé. Ce nom a été défini sous l'éditeur d'analyses ou grâce au type [Description de Fichier](../WDLang4/1514065.md).

- Nom de la rubrique manipulée (sous la forme &lt;Nom du fichier de données&gt;.&lt;Nom de la rubrique&gt;). Ce nom a été défini sous l'éditeur d'analyses ou grâce au type [Description de Rubrique](../WDLang4/1514071.md).

- Nom de la connexion manipulée. Cette connexion peut correspondre :

	- soit à une connexion décrite sous l'éditeur d'analyses,

	- soit à une connexion décrite avec les fonctions [HConnecte](../WDLang4/3044263.md), [HDécritConnexion](../WDLang4/3044205.md) ou [HOuvreConnexion](../WDLang4/3044107.md).

	- soit à une connexion décrite avec une variable de type [Connexion](../WDLang4/1514073.md).  








<a name="SYNTAXE2"></a>

### Définir les informations étendues

`<Fichier de données/Rubrique/Connexion>.InfosEtendues = <Informations étendues>`
---

**`<Fichier de données/Rubrique/Connexion> : (Chaîne de caractères)`**



- Nom du fichier de données manipulé. Ce nom a été défini grâce au type [Description de Fichier](../WDLang4/1514065.md).

- Nom de la rubrique manipulée (sous la forme &lt;Nom du fichier de données&gt;.&lt;Nom de la rubrique&gt;). Ce nom a été défini grâce au type [Description de Rubrique](../WDLang4/1514071.md).

- Nom de la connexion manipulée. Cette connexion peut correspondre :

	- soit à une connexion décrite sous l'éditeur d'analyses,

	- soit à une connexion décrite avec les fonctions [HConnecte](../WDLang4/3044263.md), [HDécritConnexion](../WDLang4/3044205.md) ou [HOuvreConnexion](../WDLang4/3044107.md).

	- soit à une connexion décrite avec une variable de type [Connexion](../WDLang4/1514073.md).







**`<Informations étendues> : (Chaîne de caractères)`**

Informations étendues de l'élément spécifié. Les différentes informations doivent être séparées par des points-virgule (';'). 

Remarque : La liste des informations étendues reconnues par OLE DB et les Connecteurs Natifs est présentée dans la page [Informations optionnelles de connexion](../WDLang4/9000124.md).

### Propriété **InfosEtendues** et variable de type Connexion
<a name="propriete_docparampagetitleshort_variable_type_connexion_ELTPARAGRAPHE000138"></a>La modification des informations concernant le cryptage d'une connexion HFSQL Client/Serveur grâce à la propriété **InfosEtendues** met automatiquement à jour les informations renvoyées par la propriété [Cryptage](../Proprietes/2512102.md) (et inversement).

Ainsi la ligne de code :

```wl
MaConnexion.Cryptage = hCryptageRC5_16
```
modifiera l'option CRYPTAGE dans la chaîne d'informations étendues.

De même la ligne de code :

```wl
MaConnexion.InfosEtendues = "CRYPTAGE=RC5_16"
```
modifiera la propriété [Cryptage](../Proprietes/2512102.md).

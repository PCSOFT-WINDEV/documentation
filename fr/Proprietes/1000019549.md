


## CocheVisible (Propriété)

***En anglais : CheckmarkVisible***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **CocheVisible** permet de : 

- Savoir si la coche est visible ou non devant une ligne d'un champ Arbre. 

- Modifier la visibilité de la coche devant une ligne d'un champ Arbre. 



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
ARBRE_Arbre1["Ligne1"].CocheVisible = Vrai
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Savoir si la coche est visible ou non

`<Résultat> = <Ligne du champ Arbre>.CocheVisible`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la coche est visible, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 




**`<Ligne du champ Arbre> : (Chaîne de caractères)`**

Ligne du champ Arbre identifiée grâce à une des notations suivantes : 

- &lt;Champ Arbre&gt; [ &lt;Indice de l'élément&gt;]

- &lt;Champ Arbre&gt; [ &lt;Chemin de élément&gt;]





<a name="SYNTAXE2"></a>

### Rendre la coche visible ou non

`<Ligne du champ Arbre>.CocheVisible = <Visible>`
---

**`<Ligne du champ Arbre> : (Chaîne de caractères)`**

Ligne du champ Arbre identifiée grâce à une des notations suivantes : 

- &lt;Champ Arbre&gt; [ &lt;Indice de l'élément&gt;]

- &lt;Champ Arbre&gt; [ &lt;Chemin de élément&gt;]




**`<Visible> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la coche doit être visible, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Le champ Arbre doit être un champ Arbre sans saisie avec gestion des coches. Dans le cas contraire, il ne sera pas possible de rendre la coche visible. 

- Si le champ Arbre est un champ Arbre sans saisie et sans gestion des coches, la propriété **CocheVisible** renvoie toujours <u><u><u><u>Faux</u></u></u></u>.

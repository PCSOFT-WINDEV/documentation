
## Certificat

***En anglais : Certificate***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le mot-clé **Certificat** peut correspondre :

- au type de variable [Certificat](../WDLang1/1000019287.md). 

- à la propriété **Certificat** d'une variable de type [Email](../WDLang3/1000018713.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) à la propriété [Certificat](../Proprietes/1000021684.md) utilisée sur les champs Signature dans les états.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [pdfSignature](../WDLang6/1000022255.md).

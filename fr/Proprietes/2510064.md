


## HauteurLigneMax (Propriété)

***En anglais : MaxLineHeight***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **HauteurLigneMax** permet de connaître et de modifier la hauteur maximale : 

- des lignes d'un champ Table (cas des lignes multilignes à redimensionnement automatique).

- des lignes d'un champ Table hiérarchique. 

- des répétitions d'un champ Zone répétée. 




**Remarque** : La hauteur maximale des lignes d'un champ Table est définie sous l'éditeur de fenêtres, dans l'onglet "Détail" de la description du champ.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Indique que la hauteur maximale d'une ligne 
// permet d'afficher 2 lignes de texte.
TABLE_Table1.HauteurLigneMax = 2
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître la hauteur maximale des lignes ou des répétitions

`<Hauteur maximale actuelle> = <Champ manipulé>.HauteurLigneMax`
---

**`<Hauteur maximale actuelle> : (Entier)`**

Hauteur maximale (en caractères) des lignes ou des répétitions pour le champ spécifié.

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler : champ Table, Table hiérarchique ou champ Zone répétée. 


<a name="SYNTAXE2"></a>

### Modifier la hauteur des lignes ou des répétitions

`<Champ manipulé>.HauteurLigneMax = <Nouvelle hauteur maximale>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler : champ Table, Table hiérarchique ou champ Zone répétée.

**`<Nouvelle hauteur maximale> : (Entier)`**

Nouvelle hauteur maximale (en caractères) : 

- des lignes du champ Table ou Table hiérarchique spécifié. 

- des répétitions du champ Zone répétée spécifié. 


Seules les nouvelles lignes/répétitions affichées seront modifiées. Les lignes/répétitions déjà affichées ne seront pas modifiées.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Hauteur maximale par défaut 
<a name="hauteur_maximale_par_defaut_ELTPARAGRAPHE000073"></a>Par défaut, la hauteur maximale d'une ligne/répétition vaut 0. Dans ce cas, la hauteur maximale d'une ligne correspond à la hauteur par défaut de la ligne.
<a name="NOTE0_2"></a>


### Limites
<a name="limites_ELTPARAGRAPHE000080"></a>

- Les valeurs possibles vont de 0 à +2147483647 (0x7FFFFFFF).

- La propriété **HauteurLigneMax** peut être utilisée uniquement sur les champs de type :

	- Table.

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Table hiérarchique. 

	- Zone répétée. 




- La hauteur d'une ligne/répétition modifiée en programmation par la propriété [Hauteur](../Proprietes/2510050.md) n'est pas affectée par ce maximum.




## Source (Propriété)

***En anglais : Source***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Source** peut être utilisée sur :

- **un champ Graphe** : dans ce cas, la propriété **Source** permet de connaître la source de la série : Colonne de table, liste de valeurs, ...
	Remarques :

	- La fonction [grSourceSérie](../WDLang3/3042067.md) permet de définir la source de données d'une série.

	- La propriété [SourceDescription](../Proprietes/2510148.md) permet de connaître le nom et/ou le contenu de la source de données.




- **une connexion** : dans ce cas, la propriété **Source** permet de :

	- Connaître la source d'une connexion (HFSQL Client/Serveur, OLE DB, ...).

	- Définir la source d'une connexion (HFSQL Client/Serveur, OLE DB, ...). Dans ce cas, si la connexion est actuellement ouverte, elle sera fermée, puis ré-ouverte avec les nouveaux paramètres lors d'un premier accès à un fichier utilisant cette connexion.

## Syntaxe
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) 
### Connaître la source de données d'une série

`<Résultat> = <Série>.Source`
---

**`<Résultat> : (Constante de type entier)`**

Type de source de données de la série :  


|   |   |
| --- | --- |
| *grColonneTable* | La source de données est une colonne de table. |
| *grFonction* | La source de données est une fonction WLangage. |
| *grListe* | La source de données est un champ Liste. |
| *grProgrammation* | La source de données est une liste de valeurs. |
| *grRubrique* | La source de données est une rubrique d'un fichier. |
| *grTableau* | La source de données est un tableau. |



**`<Série> : (Chaîne de caractères)`**

Série à manipuler. Utilisez la notation : *&lt;Nom du graphe&gt;[&lt;Numéro de la série&gt;]*.

### Connaître la source d'une connexion

`<Source de données> = <Connexion manipulée>.Source`
---

**`<Source de données> : (Chaîne de caractères)`**



- Nom du serveur (dans le cas d'une connexion HFSQL Client/Serveur, SQL Server, Oracle, MySQL, ...)

- Nom du fichier source de données (dans la cas d'une connexion Access, XML, xBase, ...)




**`<Connexion manipulée> : (Chaîne de caractères ou variable de type Connexion)`**

Nom de la connexion manipulée. Ce nom a été défini :

- soit sous l'éditeur d'analyses,

- soit avec les fonctions [HDécritConnexion](../WDLang4/3044205.md) ou [HOuvreConnexion](../WDLang4/3044107.md),

- soit avec le type [Connexion](../WDLang4/1514073.md).

### Définir la source de données de la connexion

`<Connexion manipulée>.Source = <Source de données>`
---

**`<Connexion manipulée> : (Chaîne de caractères ou variable de type Connexion)`**

Nom de la connexion manipulée. Ce nom a été défini :

- soit sous l'éditeur d'analyses,

- soit avec les fonctions [HDécritConnexion](../WDLang4/3044205.md) ou [HOuvreConnexion](../WDLang4/3044107.md),

- soit avec le type [Connexion](../WDLang4/1514073.md).




**`<Source de données> : (Chaîne de caractères)`**



- Nom du serveur ou adresse IP du serveur (dans le cas d'une connexion HFSQL Client/Serveur, SQL Server, Oracle, MySQL, ...).

- Nom du fichier source de données (dans la cas d'une connexion Access, XML, xBase, ...).

### Equivalence
<a name="equivalence_ELTPARAGRAPHE000163"></a>La propriété [Serveur](../Proprietes/2512105.md) est équivalente à la propriété **Source**.

Ces deux propriétés sont conservées pour améliorer la lisibilité du code. En effet, lors de la connexion (HFSQL Client/Serveur, SQL Server, Oracle, MySQL, ...), la propriété [Serveur](../Proprietes/2512105.md) a plus de sens car il est nécessaire d'indiquer un nom de serveur.

Par contre, lors de la connexion Access, XML, xBase, la propriété **Source** a plus de sens car il est nécessaire d'indiquer un fichier.

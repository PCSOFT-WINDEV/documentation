***En anglais : Value***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Valeur** permet de connaître et de modifier la valeur d'un élément. Cette propriété est utilisable sur :

- un champ d'une fenêtre, d'un état ou d'une page. Pour plus de détails, consulter [Propriété Valeur sur les champs](../Proprietes/2510130.md).  

- une variable de type [gglLieu](../WDLang5/1000017441.md).

- une variable de type [rssFlux](../WDLang5/1000017795.md).

- une variable de type [rssSource](../WDLang5/1000017803.md).

- une variable de type [rssIdentifiant](../WDLang5/1000017797.md).

- une variable de type [rssCatégorie](../WDLang5/1000017782.md).

- une variable de type [xlsCellule](../WDLang5/1000017472.md).

- une variable de type [xlsDocument](../WDLang5/1000017464.md).




## HauteurRuptureJour (Propriété)

***En anglais : DayBreakHeight***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **HauteurRuptureJour** permet de connaître et de modifier la hauteur des ruptures entre les jours dans un champ Planning avec les jours en ligne et les ressources en colonne. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI INT_AvecRuptures = Vrai ALORS
	PLN_MonPlanning.HauteurRuptureJour = 25
SINON
	PLN_MonPlanning.HauteurRuptureJour = 0
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître la hauteur des ruptures entre les jours

`<Résultat> = <Champ Planning>.HauteurRuptureJour`
---

**`<Résultat> : (Entier)`**



- Hauteur des ruptures en pixels. 

- 0 si il n'y a aucune rupture. 




**`<Champ Planning> : (Nom de champ)`**

Nom du champ Planning manipulé.  


<a name="SYNTAXE2"></a>

### Modifier la hauteur des ruptures entre les jours

`<Champ Planning>.HauteurRuptureJour = <Nouvelle hauteur>`
---

**`<Champ Planning> : (Nom de champ)`**

Nom du champ Planning manipulé.

**`<Nouvelle hauteur> : (Entier)`**



- Nouvelle hauteur des ruptures entre les jours exprimée en pixels. 

- 0 pour désactiver les ruptures entre les jours.


## ActionClic (Propriété)

***En anglais : ActionClick***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **ActionClic** permet de définir la procédure exécutée lors du clic sur un élément. Cette propriété est utilisable sur :

- une variable de type [Marqueur](../WDLang3/1000019940.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [Toast](../WDLang1/1000024638.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [CarteCercle](../WDLang3/1000025504.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [CarteImage](../WDLang3/1000025523.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [CartePolygone](../WDLang3/1000025476.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [CartePolyligne](../WDLang3/1000025489.md).


## RéduitAutomatique (Propriété)

***En anglais : AutomaticallyMinimized***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **RéduitAutomatique** permet de : 

- Connaître le délai avant la réduction automatique du champ Note repositionnable.

- Définir le délai avant la réduction automatique du champ Note repositionnable. 




Remarque : Cette propriété est identique à l'option "Réduire automatiquement la note après" dans l'onglet "UI" de la fenêtre de description du champ. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Déplier un champ Note repositionnable et laisser le champ déplié
NOTE_MaNote.Réduit = Faux
NOTE_MaNote.RéduitAutomatique = 0
```



```wl
// Replier le champ Note repositionnable au bout d'une minute
NOTE_MaNote.Réduit = Vrai
NOTE_MaNote.RéduitAutomatique = 60
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître la durée avant la réduction automatique du champ Note repositionnable

`<Résultat> = <Champ Note repositionnable>.RéduitAutomatique`
---

**`<Résultat> : (Entier)`**

Durée (en secondes) avant la réduction automatique du champ Note repositionnable. 

**`<Champ Note repositionnable> : (Nom de champ)`**

Nom du champ Note repositionnable à manipuler. 


<a name="SYNTAXE2"></a>

### Modifier la durée avant la réduction automatique du champ Note repositionnable

`<Champ Note repositionnable>.RéduitAutomatique = <Enroulement automatique>`
---

**`<Champ Note repositionnable> : (Nom de champ)`**

Nom du champ Note repositionnable à manipuler. 

**`<Enroulement automatique> : (Entier ou Durée)`**

Durée (en secondes) avant la réduction automatique du champ Note repositionnable. Ce paramètre peut correspondre à : 

- un entier correspondant au nombre de secondes, 

- une variable de type Durée, 

- la durée en clair (par exemple '1s' ou '10 s'). 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Fonctionnement
<a name="fonctionnement_ELTPARAGRAPHE000062"></a>

- Par défaut, le champ Note repositionnable n'est jamais réduit (option "Réduire automatiquement la note après" dans l'onglet "UI" positionnée à "Jamais").

- Si l'utilisateur déplie le champ Note repositionnable, le champ reste déplié. 

- Si la propriété **RéduitAutomatique** est affectée avec une valeur XX, au bout de XX secondes, le champ Note repositionnable sera automatiquement réduit. 





### Note
<a name="note_ELTPARAGRAPHE000076"></a>

- La propriété [Réduit](../Proprietes/1000022005.md) permet de savoir si un champ Note repositionnable est réduit.

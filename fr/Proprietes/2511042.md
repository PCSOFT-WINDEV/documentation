


## NbCopie (Propriété)

***En anglais : NbCopy***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **NbCopie** permet de connaître et de modifier le nombre d'exemplaires à imprimer d'un état.

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) **Remarques** :

- Le visualisateur de rapports permet aussi à l'utilisateur final de paramétrer directement le nombre de copies à imprimer.

- Cette propriété fonctionne uniquement lors d'une impression sur une imprimante.

- Si un nombre de copies est spécifié par l'utilisateur (fonction [iParamètre](../WDLang5/3046008.md) ou fonction [iParamètreEtat](../WDLang5/3046077.md)) avant de lancer l'impression de l'état, ce nombre de copies sera prioritaire sur celui spécifié avec la propriété **NbCopie**.





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Imprimer l'état "ETAT_Client" 10 fois
ETAT_Client.NbCopie = 10
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le nombre d'exemplaires d'un état

`<Résultat> = <Etat manipulé>.NbCopie`
---

**`<Résultat> : (Entier)`**

Nombre d'exemplaires à imprimer.

**`<Etat manipulé> : (Nom d'état)`**

Nom de l'état à manipuler.  


<a name="SYNTAXE2"></a>

### Modifier le nombre d'exemplaires d'un état

`<Etat manipulé>.NbCopie = <Nombre de copies>`
---

**`<Etat manipulé> : (Nom d'état)`**

Nom de l'état à manipuler.

**`<Nombre de copies> : (Entier)`**

Nombre de copies à imprimer.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Donner la possibilité à l'utilisateur de déterminer le nombre de copies d'un état
<a name="donner_possibilite_utilisateur_determiner_nombre_copies_etat_ELTPARAGRAPHE000069"></a>La propriété **NbCopie** ne pouvant être utilisée que dans le code de l'état, il est nécessaire de passer le nombre de copies en paramètre à l'état lors de l'impression.

Par exemple :

1. Code permettant d'imprimer l'état.
	
	```wl
	iImprimeEtat(ETAT_MonEtat, 3)
	```


2. Code d'ouverture de l'état.
	
	```wl
	PROCEDURE ETAT_MonEtat(nNbCopie)
	ETAT_MonEtat.NbCopie = nNbCopie
	```




<a name="NOTE0_2"></a>


### Limites
<a name="limites_ELTPARAGRAPHE000088"></a>

- La propriété **NbCopie** s'applique uniquement aux états (et non aux champs ou blocs d'un état).

- Cette propriété est prise en compte uniquement lors d'une impression réalisée sur une imprimante.

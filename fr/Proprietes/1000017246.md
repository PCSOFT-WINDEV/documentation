


## TexteJauge (Propriété)

***En anglais : TextProgressBar***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **TexteJauge** permet de connaître et de modifier le texte affiché dans une jauge. Ce texte est affiché à la place du pourcentage. 
**Rappel** : Par défaut, le texte affiché dans une jauge correspond au pourcentage d'avancement de la jauge.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
COL_Jauge[2].TexteJauge = "12 Go / 17 Go"
COL_Jauge.TexteJauge = "Email 3/67 (%1)"
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le texte affiché dans une jauge

`<Texte> = <Champ manipulé>.TexteJauge`
---

**`<Texte> : (Chaîne de caractères)`**

Texte affiché dans la jauge.

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler. Ce champ peut correspondre à : 

- un champ Jauge, 

- un champ de saisie, 

- un champ Combo en saisie, 

- une colonne d'un champ Table en saisie.





<a name="SYNTAXE2"></a>

### Modifier le texte affiché dans une jauge

`<Champ manipulé>.TexteJauge = <Nouveau texte>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler. Ce champ peut correspondre à : 

- un champ Jauge, 

- un champ de saisie, 

- un champ Combo en saisie, 

- une colonne d'un champ Table en saisie.




**`<Nouveau texte> : (Chaîne de caractères)`**

Nouveau texte à afficher dans la jauge. 

- Si &lt;Nouveau texte&gt; contient le chaîne "%1", cette chaîne est remplacée par le pourcentage en cours. 

- Si &lt;Nouveau texte&gt; correspond à chaîne vide (""), la jauge n'affiche plus de pourcentage.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Cas particulier
<a name="cas_particulier_ELTPARAGRAPHE000063"></a>Si la propriété **TexteJauge** est utilisée pour modifier le texte affiché sur la jauge, l'option "Afficher le pourcentage de progression" est automatiquement cochée dans l'onglet "Détail" de la description du champ Jauge.
<a name="NOTE0_2"></a>


### Limitations
<a name="limitations_ELTPARAGRAPHE000077"></a>La propriété **TexteJauge** est disponible :

- sur les champs Jauge,

- sur les cellules de colonne de champ Table de type Jauge.




Cette propriété n'est pas disponible sur les cellules de colonne de table hiérarchique de type Jauge.


## AdresseServeur (Propriété)

***En anglais : ServerAddress***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **AdresseServeur** permet de connaître et de modifier l'adresse d'un serveur. Cette propriété est utilisable sur :

- une variable de type [emailSessionIMAP](../WDLang3/1000018957.md).

- une variable de type [emailSessionPOP3](../WDLang3/1000018759.md). 

- une variable de type [emailSessionSMTP](../WDLang3/1000018765.md). 

- une variable de type [adminWBAuth](../WDLang2/1410089328.md).




## CxUtilisateurDécrit (Propriété)

***En anglais : CtDescribedUserName***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **CxUtilisateurDécrit** permet de paramétrer le nom de l'utilisateur utilisé lors de la connexion OLE DB à une table spécifique. Il est possible de :

- Connaître le nom de l'utilisateur utilisé pour la connexion à une table par OLE DB d'un fichier de données (fichier de données défini sous l'éditeur d'analyses ou défini par programmation). Ce nom d'utilisateur a été spécifié lors de la description du fichier de données.

- Définir le nom de l'utilisateur utilisé pour la connexion à une table par OLE DB d'un fichier de données (lors de la description par programmation du fichier de données uniquement). 




**Rappel** : La fonction [HConnecte](../WDLang4/3044263.md) permet de redéfinir un ou plusieurs paramètre de la connexion OLE DB sur une table. Par défaut, les caractéristiques de la connexion OLE DB sont définies sous l'éditeur d'analyses lors de la description du fichier de données.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Définir le nom de l'utilisateur
Client.CxUtilisateurDécrit = "Julie"
...
// Valide la description du fichier de données
HDécritFichier(Client)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le nom de l'utilisateur

`<Utilisateur> = <Fichier de données>.CxUtilisateurDécrit`
---

**`<Utilisateur> : (Chaîne de caractères)`**

Nom de l'utilisateur de la source OLE DB, défini lors de la description du fichier sous l'éditeur d'analyses.

**`<Fichier de données> : (Chaîne de caractères)`**

Nom du fichier de données manipulé. Ce nom a été défini sous l'éditeur d'analyses ou grâce au type [Description de Fichier](../WDLang4/1514065.md).  


<a name="SYNTAXE2"></a>

### Définir le nom de l'utilisateur

`<Fichier de données>.CxUtilisateurDécrit = <Nouveau nom de l'utilisateur>`
---

**`<Fichier de données> : (Chaîne de caractères)`**

Nom du fichier de données manipulé. Ce nom vient d'être défini grâce au type [Description de Fichier](../WDLang4/1514065.md).

**`<Nouveau nom de l'utilisateur> : (Chaîne de caractères)`**

Nom de l'utilisateur de la source OLE DB.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
La propriété [CxUtilisateur](../Proprietes/2512026.md) permet de connaître et de modifier le nom de l'utilisateur spécifié pour la connexion OLE DB lors de l'utilisation de la fonction [HConnecte](../WDLang4/3044263.md).

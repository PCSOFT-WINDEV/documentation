


## Langue (Propriété)

***En anglais : Language***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Langue** permet de connaître et de modifier la langue utilisée. Cette propriété est utilisable sur :

- les rubriques des fichiers de données pour gérer la langue utilisée pour les comparaisons et le tri. Pour plus de détails, consultez [Propriété Langue sur les rubriques](../Proprietes/1000017053.md).

- une variable de type [gglParamètreCarte](../WDLang5/1000017505.md).

- une variable de type [rssCanal](../WDLang5/1000017769.md).

- une variable de type [sfInfoUtilisateur](../WDLang5/1000018423.md).

- ![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) une variable de type [VoixDeSynthèse](../WDLang3/1000025986.md).




## Grisé (Propriété)

***En anglais : Grayed***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Grisé** permet de : 

- Savoir si un champ, un groupe de champs ou une fenêtre est grisé ou non. 

- Griser ou dégriser un champ, un groupe de champs ou une fenêtre. 





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Grise le groupe de champs GR_FraisDeLivraison en fonction de la valeur du champ SAI_Prix
SI SAI_Prix > 100 ALORS
	GR_FraisDeLivraison.Grisé = Vrai
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Savoir si un élément est grisé

`<Résultat> = <Elément manipulé>.Grisé`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'élément est grisé, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Elément manipulé> : (Nom de champ ou nom de fenêtre)`**

Nom de l'élément à manipuler : 

- Nom du champ, 

- Nom du groupe de champs,

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Nom de la fenêtre. 





<a name="SYNTAXE2"></a>

### Modifier l'état grisé pour un élément

`<Elément manipulé>.Grisé = <Nouvelle valeur>`
---

**`<Elément manipulé> : (Nom de champ ou nom de fenêtre)`**

Nom de l'élément à manipuler : 

- Nom du champ, 

- Nom du groupe de champs,

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Nom de la fenêtre. 




**`<Nouvelle valeur> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'élément doit être grisé, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Equivalence
<a name="equivalence_ELTPARAGRAPHE000114"></a>La propriété **Grisé** à <u><u><u><u>Vrai</u></u></u></u> est équivalente à la propriété [Etat](../Proprietes/2510039.md) égale à la constante *Grisé* sauf pour les champs en lecture seule qui conservent l'état "en lecture seule". 


### Limites
<a name="limites_ELTPARAGRAPHE000129"></a>![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) La propriété **Grisé** s'applique à tous les types de champs (sauf pour les cases des barres de message).  

![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) La propriété **Grisé** est sans effet sur les champs HTML.

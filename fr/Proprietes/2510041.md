


## HauteurLigne (Propriété)

***En anglais : LineHeight***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **HauteurLigne** permet de connaître et de modifier la hauteur des lignes :

- d'un champ Liste,

- d'un champ Liste image (mode liste classique uniquement),

- d'un champ Table,

- d'un champ Combo,

- d'un champ Zone répétée (hauteur de la répétition).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) d'un champ Agenda.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) d'un champ Tableur.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) d'un champ Arbre.




**Remarques** : 

- Champ Liste ou Combo : La hauteur des lignes est définie sous l'éditeur de fenêtres, dans l'onglet "Style" de la description du champ.

- Champ Table, Arbre, Zone répétée, Tableur : La hauteur des lignes est définie sous l'éditeur de fenêtres, dans l'onglet "Détail" de la description du champ.







<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Augmentation de la hauteur des lignes du champ "TABLE_TableClient"
TABLE_TableClient.HauteurLigne = TABLE_TableClient.HauteurLigne + 100
// Équivalent à : TABLE_TableClient.HauteurLigne +=100
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître la hauteur des lignes d'un champ

`<Hauteur actuelle> = <Champ manipulé>.HauteurLigne`
---

**`<Hauteur actuelle> : (Entier)`**

Hauteur (en pixels) des lignes du champ spécifié.

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler.  


<a name="SYNTAXE2"></a>

### Modifier la hauteur des lignes d'un champ

`<Champ manipulé>.HauteurLigne = <Nouvelle hauteur>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler.

**`<Nouvelle hauteur> : (Entier)`**

Nouvelle hauteur (en pixels) des lignes du champ spécifié.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Champ Liste, champ Table et champ Combo de type fichier
<a name="champ_liste_champ_table_champ_combo_type_fichier_ELTPARAGRAPHE000103"></a>Lors de la modification de la hauteur des lignes d'un champ Table, champ Liste ou champ Combo de type fichier, seules les lignes visibles à l'écran conservent la nouvelle hauteur de lignes.
<a name="NOTE0_2"></a>


### Champ Liste
<a name="champ_liste_ELTPARAGRAPHE000110"></a>Si nécessaire, un ascenseur vertical peut être affiché dans un champ Liste après modification de la taille des lignes.
<a name="NOTE0_3"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 

### Champ Liste image
<a name="champ_liste_image_ELTPARAGRAPHE000119"></a>La propriété **HauteurLigne** est prise en compte uniquement si l'affichage du champ Liste image est effectué en mode classique.
<a name="NOTE0_4"></a>


### Limites
<a name="limites_ELTPARAGRAPHE000129"></a>![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) La propriété **HauteurLigne** s'applique uniquement aux champs Zone répétée.![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) La propriété **HauteurLigne** s'applique uniquement aux : 

- Champs Liste, 

- Champ Combo, 

- Champs Table.

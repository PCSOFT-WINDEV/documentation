


## BalayageActif (Propriété)

***En anglais : SwipeEnabled***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **BalayageActif** active ou désactive la gestion du mouvement de balayage par l'utilisateur : 

- sur un champ Fenêtre interne. 

- sur une ligne d'un champ Zone répétée.



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
CFI_FenInterne.BalayageActif = Faux
```


<a name="Exemple2"></a>

```wl
ZR_Produit[2].BalayageActif = Faux
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître le mode de gestion du balayage

`<Résultat> = <Champ manipulé>.BalayageActif`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le balayage est autorisé, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 




**`<Champ manipulé> : (Nom de champ)`**



- Nom du champ Fenêtre interne manipulé. 

- Ligne du champ Zone répétée manipulé. Dans ce cas, la notation utilisée est : 
	
	```txt
	Champ_Zone_Repétée[Numéro_Ligne]
	```






<a name="SYNTAXE2"></a>

### Modifier le mode de gestion du balayage

`<Champ manipulé>.BalayageActif = <Gestion du balayage>`
---

**`<Champ manipulé> : (Nom de champ)`**



- Nom du champ Fenêtre interne manipulé. 

- Ligne du champ Zone répétée manipulé. Dans ce cas, la notation utilisée est : 
	
	```txt
	Champ_Zone_Repétée[Numéro_Ligne]
	```





**`<Gestion du balayage> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le balayage est autorisé, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Balayage dans une fenêtre interne
<a name="balayage_dans_une_fenetre_interne_ELTPARAGRAPHE000067"></a>Par défaut, le balayage est activé : 

- lorsque sur un champ Fenêtre interne l'option "Autoriser le changement de contenu par balayage" est activée (onglet "UI" de la fenêtre de description du champ), 

- lorsque plus d'une fenêtre interne a été ajoutée par la fonction [FIListeAjoute](../WDLang1/1000020558.md). 


Dans ce cas, l'utilisateur peut modifier la fenêtre interne affichée par un simple mouvement de doigt. 

La propriété **BalayageActif** permet de désactiver puis de réactiver le balayage par l'utilisateur. Il est ainsi possible de désactiver le balayage momentanément pour effectuer une action spécifique. 


### Balayage dans un champ Zone répétée
<a name="balayage_dans_champ_zone_repetee_ELTPARAGRAPHE000089"></a>Par défaut, le balayage sur un champ Zone répétée est activé lorsque : 

- l'option "Balayage d'une ligne : Balayage personnalisable" est activée (onglet "Détail" de la fenêtre de description du champ), 

- une fenêtre interne a été définie pour le balayage personnalisé.  


Dans ce cas, l'utilisateur peut afficher la fenêtre interne par un simple mouvement de doigt sur la ligne du champ Zone répétée. 

La propriété **BalayageActif** permet de désactiver puis de réactiver le balayage par l'utilisateur. Il est ainsi possible de désactiver le balayage momentanément pour effectuer une action spécifique.

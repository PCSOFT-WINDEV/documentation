
## DuréeNonRéponse (Propriété)

***En anglais : TimeOut***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **DuréeNonRéponse** permet de connaître et de modifier le délai d'attente maximum. Cette propriété est utilisable sur :

- le flux lors de l'utilisation d'un champ VisioConférence. Pour plus de détails, consultez [DuréeNonRéponse](../Proprietes/2510052.md).

- une variable de type [httpRequête](../WDLang3/1000021158.md).

- une variable de type [restRequête](../WDLang3/1000021481.md).

- une variable de type [wsRequête](../WDLang3/1000020577.md).

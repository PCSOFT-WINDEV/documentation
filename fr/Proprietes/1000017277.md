


## TailleClé (Propriété)

***En anglais : KeySize***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **TailleClé** permet de paramétrer les caractéristiques d'une rubrique clé de type mémo texte. Il est possible de :

- Connaître la taille de la partie indexée du mémo pour une rubrique définie sous l'éditeur d'analyses ou définie par programmation.

- Définir la taille de la partie indexée du mémo (lors de la description de la rubrique par programmation uniquement).   





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonIndexMémo est une Description de Rubrique
MonIndexMémo.Nom = "Commentaires"
MonIndexMémo.Type = hRubMémoTexte
MonIndexMémo.TypeClé = hCléUnique
MonIndexMémo.TypeTri = hCroissant
MonIndexMémo.SensibleAccentuation = Faux
MonIndexMémo.SensibleCasse = Faux
MonIndexMémo.SensiblePonctuation = Faux
MonIndexMémo.TailleClé = 50
HDécritRubrique(Livre, MonIndexMémo)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître la taille de la partie indexée du mémo

`<Taille> = <Fichier de données>.<Rubrique>.TailleClé`
---

**`<Taille> : (Entier)`**

Nombre de caractères du mémo texte qui seront indexés.

**`<Fichier de données> : (Chaîne de caractères)`**

Nom du fichier de données manipulé. Ce nom a été défini sous l'éditeur d'analyses ou grâce au type [Description de Fichier](../WDLang4/1514065.md).

**`<Rubrique> : (Chaîne de caractères)`**

Nom de la rubrique manipulée. Ce nom a été défini sous l'éditeur d'analyses ou grâce au type [Description de Rubrique](../WDLang4/1514071.md).  


<a name="SYNTAXE2"></a>

### Définir la taille de la partie indexée du mémo

`<Rubrique>.TailleClé = <Taille indexée>`
---

**`<Rubrique> : (Variable de type Description de rubrique)`**

Nom de la rubrique manipulée. Ce nom vient d'être défini grâce au type [Description de Rubrique](../WDLang4/1514071.md).

**`<Taille indexée> : (Entier)`**

Taille de la partie indexée du mémo, exprimée en nombre de caractères.

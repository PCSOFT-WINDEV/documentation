
## Heure (Mot-clé)

***En anglais : Hour/Time***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le mot-clé **Heure** peut correspondre : 

- à une variable de type [Heure](../Motscles/1514068.md). 

- à la propriété **Heure**. Cette propriété permet de connaître et de modifier l'heure d'un élément. Cette propriété est utilisable sur :

	- une variable ou une rubrique de type Date, Heure et DateHeure. Pour plus de détails, consultez [Propriété Heure sur les rubriques et les types Date, Heure et DateHeure](../Proprietes/2512038.md). 

	- une variable de type [hTâchePlanifiée](../WDLang4/1000017448.md).

	- une variable de type [hPlanification](../WDLang4/1000018853.md).

	- une variable de type [Répétition](../WDLang1/1000019262.md).




## NombreLigne (Propriété)

***En anglais : NumberRow***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **NombreLigne** renvoie le nombre maximum de lignes avec des données dans un champ Tableur. Ces lignes contiennent des données, une formule, un formatage ou une hauteur non standard. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
POUR I  = 1 _A_ TBLR_MonTableur.NombreLigne
	POUR J = 1 _A_ TBLR_MonTableur.NombreColonne
			Trace(TBLR_MonTableur[I, J])
		FIN
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître le nombre de lignes d'un champ Tableur

`<Nombre de lignes> = <Champ Tableur>.NombreLigne`
---

**`<Nombre de lignes> : (Entier)`**

Nombre maximum de lignes ayant des données (valeur ou formule) dans un champ Tableur. 

**`<Champ Tableur> : (Nom de champ)`**

Nom du champ Tableur dont on veut connaître le nombre de colonnes.

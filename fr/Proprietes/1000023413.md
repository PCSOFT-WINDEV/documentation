


## MargeInterCellules (Propriété)

***En anglais : MarginInterCell***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **MargeInterCellules** permet de : 

- connaître la marge entres les lignes ou les colonnes d'un champ Disposition. 

- modifier la marge entres les lignes ou les colonnes d'un champ Disposition. 
	



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// 16 pixels entre chaque ligne de DISP_Enligne
DISP_EnLigne.MargeInterCellules = 16
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître la marge entres les lignes ou les colonnes d'un champ Disposition

`<Résultat> = <Champ Disposition>.MargeInterCellules`
---

**`<Résultat> : (Entier)`**

Marge (en pixels) entres les lignes ou les colonnes du champ Disposition spécifié. 

**`<Champ Disposition> : (Nom de champ)`**

Nom du champ Disposition à manipuler. 


<a name="SYNTAXE2"></a>

### Modifier la marge entres les lignes ou les colonnes d'un champ Disposition

`<Champ Disposition>.MargeInterCellules = <Nouvelle marge>`
---

**`<Champ Disposition> : (Nom de champ)`**

Nom du champ Disposition à manipuler. 

**`<Nouvelle marge> : (Entier)`**

Nouvelle marge (en pixels) entres les lignes ou les colonnes du champ Disposition spécifié. Cette valeur doit être positive ou nulle.

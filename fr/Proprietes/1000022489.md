
## Paragraphe (Propriété)

***En anglais : Paragraph***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Paragraphe** permet de manipuler les paragraphes d'un document Traitement de texte. Cette propriété est utilisable sur :

- les variables de type [docElément](../WDLang1/1000022484.md). 

- les variables de type [Document](../WDLang1/1000022461.md). 

- les variables de type [docFragment](../WDLang1/1000022482.md).

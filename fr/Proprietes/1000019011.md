


## Invité (Propriété)

***En anglais : Guest***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Invité** permet de connaître et de modifier une liste d'invités. Cette propriété est utilisable sur :

- une variable de type [gglEvénement](../WDLang5/1000017440.md)

- une variable de type [RendezVous](../WDLang1/1000019244.md).

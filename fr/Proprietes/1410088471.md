
## PointDAttache (Propriété)

***En anglais : ConnectionPoint***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **PointDAttache** permet de gérer les points d'attache d'une forme présente dans un diagramme affiché dans un champ Editeur de diagrammes. Cette propriété est utilisable sur : 

- une variable de type [diagForme](../WDLang1/1410088082.md).

- une variable de type [diagChemin](../WDLang1/1410088384.md),

- une variable de type [diagGroupe](../WDLang1/1410088127.md),

- une variable de type [diagImage](../WDLang1/1410088414.md),

- une variable de type [diagLien](../WDLang1/1410088454.md),

- une variable de type [diagOvale](../WDLang1/1410088135.md),

- une variable de type [diagPolygone](../WDLang1/1410088137.md),

- une variable de type [diagRectangle](../WDLang1/1410088083.md), 

- une variable de type [diagExtrémité](../WDLang1/1410088470.md), 

- une variable de type [diagActionAutorisée](../WDLang1/1410088840.md).

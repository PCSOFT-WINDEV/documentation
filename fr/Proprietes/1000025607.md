
## Boussole (Propriété)

***En anglais : Compass***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Boussole** permet : 

- d'afficher ou non la boussole sur un champ Carte, 

- de savoir si la boussole est affichée sur un champ Carte. 




Attention : La boussole est affichée uniquement en cas de rotation de la carte. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affiche la boussole
CARTE_MaCarte.Boussole = Vrai
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Savoir si la boussole est affichée sur un champ Carte

`<Résultat> = <Champ Carte>.Boussole`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la boussole est affichée sur le champ Carte, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 




**`<Champ Carte> : (Nom de champ)`**

Nom du champ Carte manipulé. 


<a name="SYNTAXE2"></a>

### Afficher ou non la boussole sur un champ Carte

`<Champ Carte>.Boussole = <Autorisation>`
---

**`<Champ Carte> : (Nom de champ)`**

Nom du champ Carte manipulé. 

**`<Autorisation> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la boussole doit être affichée sur le champ Carte, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




## LargeurJour (Propriété)

***En anglais : DayWidth***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **LargeurJour** permet de connaître et de modifier la largeur des jours : 

- dans un champ Planning avec les jours en colonne et les ressources en ligne.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) dans une colonne de type Diagramme de Gantt.





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI INT_Zoom = Vrai ALORS
	PLN_MonPlanning.LargeurJour = 200
SINON
	PLN_MonPlanning.LargeurJour = 100
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître la largeur des jours

`<Résultat> = <Champ manipulé>.LargeurJour`
---

**`<Résultat> : (Entier)`**



- Largeur des jours en pixels si la largeur des jours a été précédemment modifiée par programmation (avec la propriété **LargeurJour**).

- -1 dans les cas suivants : 

	- si la largeur des jours est automatiquement calculée (grâce aux informations données dans la fenêtre de description du champ). 

	- si les ressources sont en colonne. 







**`<Champ manipulé> : (Nom de champ)`**

Nom du champ manipulé : 

- Champ Planning, 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Colonne de type Diagramme de Gantt. 





<a name="SYNTAXE2"></a>

### Modifier la largeur des jours

`<Champ manipulé>.LargeurJour = <Nouvelle largeur>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ manipulé : 

- Champ Planning, 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Colonne de type diagramme de Gantt. 




**`<Nouvelle largeur> : (Entier)`**



- Nouvelle largeur des jours exprimée en pixels. 

- -1 pour calculer automatiquement la largeur des jours à partir des informations données dans la fenêtre de description du champ.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Si la largeur des jours est fixée avec la propriété **LargeurJour**, plus aucun "ancrage" de la taille des jours en fonction de la taille disponible n'est effectué. 

- Seule la valeur -1 peut être utilisée. Les autres valeurs négatives sont ignorées.

- Si le champ Planning est configuré avec l'option "Afficher les heures", la taille donnée sera ajustée par tranche de 24h.

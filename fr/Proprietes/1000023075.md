
## CouleurFondTitre (Propriété)

***En anglais : TitleBackgroundColor***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **CouleurFondTitre** permet de connaître et de modifier la couleur de fond d'un titre de colonne d'un champ Table ou Table hiérarchique. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI bCLientImportant ALORS
	COL_Remarque.CouleurFondTitre = RougeClair
SINON
	COL_Remarque.CouleurFondTitre = CouleurDefaut
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître la couleur de fond du titre d'une colonne

`<Couleur de fond> = <Colonne manipulée>.CouleurFondTitre`
---

**`<Couleur de fond> : (Entier ou constante)`**

Couleur de fond du titre de la colonne. Cette couleur peut correspondre à :

- une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)),

- [une couleur prédéfinie](../WDLang5/3010002.md).




**`<Colonne manipulée> : (Nom de champ)`**

Nom d'une colonne de champ Table ou Table hiérarchique. 


<a name="SYNTAXE2"></a>

### Modifier la couleur de fond du titre d'une colonne

`<Colonne manipulée>.CouleurFondTitre = <Nouvelle couleur de fond>`
---

**`<Colonne manipulée> : (Nom de champ)`**

Nom d'une colonne de champ Table ou Table hiérarchique. 

**`<Nouvelle couleur de fond> : (Entier ou constante)`**

Nouvelle couleur de fond du titre de la colonne. Cette couleur peut correspondre à :

- une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)),

- [une couleur prédéfinie](../WDLang5/3010002.md), 

- la constante *CouleurDéfaut*. Dans ce cas, la couleur de fond correspond à la couleur de fond commune à tous les titres de colonne (définie dans le style du champ). 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Si le fond du titre de la colonne est une image, le fond du titre est surchargé avec cette couleur.

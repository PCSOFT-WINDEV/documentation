


## HTMLFinPage (Propriété)

***En anglais : HTMLEndPage***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **HTMLFinPage** permet de connaître et de modifier le code HTML ajouté à la fin de la page.

Remarque : Il est également possible de spécifier le code HTML ajouté à la fin de la page dans l'onglet "Avancé" de la fenêtre de description de la page (onglet "Html").

<a name="sample_code"></a>

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le contenu HTML inséré à la fin de la page

`<Contenu HTML> = <Page manipulée>.HTMLFinPage`
---

**`<Contenu HTML> : (Chaîne de caractères)`**

Contenu HTML inséré à la fin de la page.

**`<Page manipulée> : (Nom de page)`**

Nom de la page à manipuler.  


<a name="SYNTAXE2"></a>

### Modifier le contenu HTML inséré à la fin de la page

`<Page manipulée>.HTMLFinPage = <Contenu HTML>`
---

**`<Page manipulée> : (Nom de page)`**

Nom de la page à manipuler.

**`<Contenu HTML> : (Chaîne de caractères)`**

Contenu HTML à insérer à la fin de la page.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Le code HTML est inséré à la fin de la page, juste avant la fermeture de la balise "BODY".

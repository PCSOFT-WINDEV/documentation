
## Page (Propriété)

***En anglais : Page***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Page** permet de manipuler une page. Cette propriété est utilisable sur :

- une variable de type [pdfSignature](../WDLang6/1000022255.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [pdfDocument](../WDLang6/1000024909.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) le champ Page WEBDEV. Cette propriété permet de manipuler depuis la fenêtre, la page chargée dans le champ Page WEBDEV. Pour plus de détails, consultez [Page](../Proprietes/1410089371.md).

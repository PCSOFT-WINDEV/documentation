


## Ancrage (Propriété)

***En anglais : Anchor***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Ancrage** permet de gérer l'ancrage d'un champ. Il est possible de :

- Connaître l'ancrage actuel d'un champ dans une fenêtre ou un état.

- Modifier les caractéristiques d'ancrage d'un champ dans une fenêtre.




**Rappel** : L'ancrage d'un champ permet de définir le comportement du champ (dimensions et déplacement) lors du changement de taille de la fenêtre (ou de l'état). 

Plusieurs propriétés permettent de définir ou de connaître le taux d'ancrage d'un champ. Ces propriétés sont les suivantes :

- [TauxAncrageBas](../Proprietes/2510113.md).

- [TauxAncrageDroite](../Proprietes/2510114.md).

- [TauxAncrageHauteur](../Proprietes/2510119.md).

- [TauxAncrageLargeur](../Proprietes/2510120.md).





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Définition des caractéristiques d'ancrage
// Le champ sera agrandi en hauteur et largeur
LISTE_Liste1.Ancrage = ancreLargeur + ancreHauteur
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître l'ancrage d'un champ

`<Ancrage actuel> = <Champ manipulé>.Ancrage`
---

**`<Ancrage actuel> : (Constante de type entier)`**

Caractéristique d'ancrage actuellement définie pour le champ : 



|   |   |
| --- | --- |
| *ancreBas* | Le champ se déplace vers le bas. |
| *ancreDroite* | Le champ se déplace vers la droite. |
| *ancreHauteur* | Le champ s'étire vers le bas. |
| *ancreHautGauche* | Aucune action. |
| *ancreLargeur* | Le champ s'étire vers la droite. |


Les constantes *ancreDroite* et *ancreLargeur* peuvent être combinées avec les constantes *ancreBas* et *ancreHauteur*.

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ manipulé (champ d'une fenêtre ou d'un état).

### Modifier l'ancrage d'un champ

`<Champ manipulé>.Ancrage = <Nouvel ancrage>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ manipulé (champ d'une fenêtre ou d'un état).

**`<Nouvel ancrage> : (Constante de type entier)`**

Nouvelle caractéristique d'ancrage définie pour le champ : 



|   |   |
| --- | --- |
| *ancreBas* | Le champ se déplace vers le bas. |
| *ancreDroite* | Le champ se déplace vers la droite. |
| *ancreHauteur* | Le champ s'étire vers le bas. |
| *ancreHautGauche* | Aucune action. |
| *ancreLargeur* | Le champ s'étire vers la droite. |


Les constantes *ancreDroite* et *ancreLargeur* peuvent être combinées avec les constantes *ancreBas* et *ancreHauteur*.

## Remarques


### Ancrage
<a name="ancrage_ELTPARAGRAPHE000165"></a>Rappel : l'ancrage d'un champ permet de définir le comportement du champ (dimensions et déplacement) lors du changement de taille de la fenêtre ou de l'état. 

Les positions et largeurs utilisées pour calculer les nouvelles tailles et positions sont celles définies sous l'éditeur. Elles peuvent être connues grâce aux propriétés [XInitial](../Proprietes/2510140.md), [YInitial](../Proprietes/2510087.md), [LargeurInitiale](../Proprietes/2510067.md) et [HauteurInitiale](../Proprietes/2510043.md).

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Remarques : 

- Dans le cas d'une colonne d'un champ Table ou Table hiérarchique, la propriété **Ancrage** n'est pas disponible. L'ancrage d'une colonne se spécifie directement avec la propriété [TauxAncrageLargeur](../Proprietes/2510120.md).

- Si le champ est associé à un volet d'onglet, l'ancrage est défini par rapport au champ Onglet.



<a name="NOTE0_2"></a>


### Limites
<a name="limites_ELTPARAGRAPHE000199"></a>![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) La propriété **Ancrage** s'applique à tous les champs d'une fenêtre.

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) La propriété **Ancrage** s'applique à tous les champs d'un état. Dans ce cas, il est uniquement possible de connaître le type d'ancrage utilisé. 

![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) La propriété **Ancrage** s'applique uniquement aux champs suivants :

- Champ Bouton.

- Champ Libellé.

- Champ de saisie.

- Champ Image.

- Champ Interrupteur.

- Champ Sélecteur.

- Champ Liste.

- Champ Combo.

- Champ Arbre.

- Champ Onglet.

- Champ Table.

- Superchamp.

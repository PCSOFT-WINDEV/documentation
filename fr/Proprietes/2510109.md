


## RubriqueAffichéeImage (Propriété)

***En anglais : DisplayedItemImage***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **RubriqueAffichéeImage** permet de :

- Connaître la rubrique correspondant à l'image affichée dans un champ Liste image.

- Modifier la rubrique correspondant à l'image affichée dans un champ Liste image.


Remarque : Cette rubrique peut provenir d'un fichier de données, d'une vue ou d'une requête.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Modifier la rubrique affichée dans le champ "LSI_ListeClient"
LSI_ListeClient.RubriqueAffichéeImage = Client.Photo
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître la rubrique correspondant à l'image affichée

`<Rubrique affichée> = <Champ Liste image>.RubriqueAffichéeImage`
---

**`<Rubrique affichée> : (Chaîne de caractères)`**

Nom de la rubrique correspondant à l'image affichée dans le champ Liste image.

**`<Champ Liste image> : (Nom de champ)`**

Nom du champ Liste image à manipuler.  


<a name="SYNTAXE2"></a>

### Modifier la rubrique correspondant à l'image affichée 

`<Champ Liste image>.RubriqueAffichéeImage = <Nouvelle rubrique>`
---

**`<Champ Liste image> : (Nom de champ)`**

Nom du champ Liste image à manipuler.

**`<Nouvelle rubrique> : (Chaîne de caractères)`**

Nom de la nouvelle rubrique à utiliser pour afficher l'image du champ Liste image. Cette rubrique doit appartenir au fichier de données parcouru par le champ (connu ou modifié grâce à la propriété [FichierParcouru](../Proprietes/2510040.md)).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Image et libellé
<a name="image_libelle_ELTPARAGRAPHE000048"></a>La propriété **RubriqueAffichéeImage** permet de connaître et de modifier uniquement la rubrique correspondant aux images affichées dans le champ.

Pour connaître et modifier la rubrique correspondant aux libellés des éléments affichés dans le champ, utilisez la propriété [RubriqueAffichée](../Proprietes/2510123.md).
<a name="NOTE0_2"></a>


### Manipuler des champs Liste image grâce aux propriétés 
<a name="manipuler_des_champs_liste_image_grace_aux_proprietes_ELTPARAGRAPHE000063"></a>

- Les enregistrements affichés dans un champ Liste image fichier proviennent du fichier de données ou de la requête spécifiée avec la propriété [FichierParcouru](../Proprietes/2510040.md).

- Le sens de tri est fixé par la propriété [RubriqueParcourue](../Proprietes/2510111.md).

- Un filtrage des enregistrements peut être réalisé avec la propriété [Filtre](../Proprietes/2510042.md) ou la fonction [HFiltre](../WDLang4/3044100.md).



<a name="NOTE0_3"></a>


### Limite
<a name="limite_ELTPARAGRAPHE000084"></a>La propriété **RubriqueAffichéeImage** s'applique uniquement aux champs Liste image.

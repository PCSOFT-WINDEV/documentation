


## Identifiant (Propriété)

***En anglais : Identifier***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Identifiant** permet de connaître et modifier le nom sous lequel le poste en cours apparaît pour les correspondants du flux audio ou vidéo. Ce nom correspond par exemple à la présentation du numéro.

Par défaut, l'identifiant du flux est une chaîne vide.

Remarque : En réception, ce nom sera connu avec la fonction [FluxInfo](../WDLang3/3039009.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Modification du nom de l'identifiant
MonFlux.Identifiant = "PosteJulie"
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître l'identifiant du flux

`<Résultat> = <Nom du flux>.Identifiant`
---

**`<Résultat> : (Chaîne de caractères)`**

Nom sous lequel le poste en cours apparaît à ses correspondants.

**`<Nom du flux> : (Chaîne de caractères)`**

Nom du Flux ou nom du champ Conférence associé au flux.  


<a name="SYNTAXE2"></a>

### Modifier l'identifiant du flux

`<Nom du flux>.Identifiant = <Nouveau Nom>`
---

**`<Nom du flux> : (Chaîne de caractères)`**

Nom du Flux ou nom du champ Conférence associé au flux.

**`<Nouveau Nom> : (Chaîne de caractères)`**

Nom sous lequel le poste en cours apparaît à ses correspondants.


## Angle (Propriété)

***En anglais : Orientation***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Angle** permet de connaître et de modifier l'angle d'un élément. Cette propriété est utilisable sur :

- une variable de type [Police](../Motscles/1514045.md). Pour plus de détails, consultez l'aide de la propriété [Angle](../Proprietes/1410086632.md). 

- une variable de type [Filigrane](../WDLang5/1000019922.md).

- une variable de type [Fond](../WDLang1/1000022038.md).

- une variable de type [CarteImage](../WDLang3/1000025523.md).

- une variable de type [diagForme](../WDLang1/1410088082.md).

- une variable de type [diagSélection](../WDLang1/1410088172.md).


## Forme (Propriété)

***En anglais : Shape***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Forme** permet de gérer les différentes formes d'un élément. Cette propriété est disponible sur : 

- une variable de type [MarqueurImage](../WDLang3/1000025796.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [Diagramme](../WDLang1/1410088055.md).

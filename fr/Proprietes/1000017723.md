


## TypeRemplissage (Propriété)

***En anglais : FillType***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **TypeRemplissage** permet de connaître le type de source utilisé pour remplir les champs de type Table, Liste, Liste image, Combo et Zone répétée. Cette propriété est utilisable en lecture uniquement. 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Type de remplissage d'un champ Table
ResRemplissage = TABLE_Commande.TypeRemplissage
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Type de remplissage> = <Champ manipulé>.TypeRemplissage`
---

**`<Type de remplissage> : (Constante de type Entier)`**

Type de remplissage du champ :


|   |   |
| --- | --- |
| *treFichierAccèsDirect* | Champ fichier en mode accès direct. |
| *treFichierChargéEnMémoire* | Champ fichier en mode chargé en mémoire. |
| *treMémoire* | Champ par programmation (fonctions [TableAjoute](../WDLang1/3074017.md) ou [TableAjouteLigne](../WDLang1/3074006.md) pour un champ Table). |
| *treVariable* | Champ sur variable lié à une variable WLangage. |



**`<Champ manipulé> : (Nom de champ)`**

Nom du champ manipulé : champ Table, Liste, Liste image, Combo ou Zone répétée.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Champs Liste, Liste image, Combo et Table fichier
<a name="champs_liste_liste_image_combo_table_fichier_ELTPARAGRAPHE000066"></a>Les propriétés suivantes définissent les enregistrements qui sont affichés.

- [FichierParcouru](../Proprietes/2510040.md) : Nom du fichier de base.

- [RubriqueParcourue](../Proprietes/2510111.md) : Nom de la rubrique utilisée pour le parcours (le tri est effectué sur cette rubrique).

- [Filtre](../Proprietes/2510042.md) : Si ce filtre n'est pas vide, seuls les enregistrements correspondant au filtre sont affichés.

- [ParcoursAuto](../Proprietes/2510093.md) : Indique si l'on utilise les traitements de parcours associés à l'objet.



<a name="NOTE0_2"></a>


### Equivalence
<a name="equivalence_ELTPARAGRAPHE000086"></a>Cette propriété remplace la propriété [Mémoire](../Proprietes/2510076.md) et permet d'obtenir des renseignements plus détaillés.

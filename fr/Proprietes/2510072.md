


## PoliceTaille (Propriété)

***En anglais : FontSize***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **PoliceTaille** permet de :

- Connaître la taille de la police utilisée dans un champ.

- Modifier la taille de la police utilisée dans un champ.







<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Modifier la taille de la police utilisée dans le champ "SAI_NomClient"
SAI_NomClient.PoliceTaille = 16
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître la taille de la police utilisée dans un champ

`<Résultat> = <Champ manipulé>.PoliceTaille`
---

**`<Résultat> : (Entier)`**

Taille de la police utilisée dans l'objet spécifié.


**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler.  


<a name="SYNTAXE2"></a>

### Modifier la taille de la police utilisée dans un champ

`<Champ manipulé>.PoliceTaille = <Taille de la police>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler.

**`<Taille de la police> : (Entier)`**

Nouvelle taille de la police utilisée dans le champ spécifié.

## Remarques


### Limites
<a name="limites_ELTPARAGRAPHE000091"></a>![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Dans un état, la propriété **PoliceTaille** peut être utilisée uniquement sur :

- un champ Libellé.

- un champ Calculé.

- un champ Prédéfini.

- un champ Lien.


![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Dans une fenêtre, la propriété **PoliceTaille** ne peut pas être utilisée sur :

- un champ Spin,

- un champ Web Camera,

- un champ Conférence.


Remarque : Cette propriété s'applique sur les champs "Liste image" en mode d'affichage "Liste classique" uniquement.
![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) La propriété **PoliceTaille** s'applique uniquement aux éléments suivants :

- Bouton,

- Libellé,

- Combo,

- Arbre,

- Colonne de table.

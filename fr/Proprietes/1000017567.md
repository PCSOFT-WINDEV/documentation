


## ImageFondLigne (Propriété)

***En anglais : BackgroundImageLine***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **ImageFondLigne** permet de connaître et de modifier l'image de fond utilisée pour chaque ligne : 

- d'un champ Table, 

- d'un champ Table hiérarchique, 

- d'un champ Zone répétée.




Cette image n'est pas utilisée pour les lignes sélectionnées ou survolées.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Changement de l'image utilisée pour les lignes du champ Table
TABLE_Mouvements.ImageFondLigne = "Dollars.gif"
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître l'image de fond des lignes d'un champ

`<Résultat> = <Champ manipulé>.ImageFondLigne`
---

**`<Résultat> : (Chaîne de caractères)`**

Nom et chemin complet de l'image de fond associée aux lignes du champ.

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler : 

- Champ Table, 

- Champ Table hiérarchique, 

- Champ Zone répétée.





<a name="SYNTAXE2"></a>

### Modifier l'image de fond des lignes d'un champ

`<Champ manipulé>.ImageFondLigne = <Nouvelle image>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler : 

- Champ Table, 

- Champ Table hiérarchique, 

- Champ Zone répétée.




**`<Nouvelle image> : (Chaîne de caractères)`**

Nom et chemin complet de la nouvelle image de fond associée aux lignes du champ. Un chemin de type UNC peut être utilisé. Si l'image est présente dans la bibliothèque, le chemin complet n'est pas nécessaire.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Caractéristiques de l'image
<a name="caracteristiques_image_ELTPARAGRAPHE000060"></a>Le mode 9 images qui s'applique à l'image est celui défini dans les options de style concernant le bandeau (onglet "Style" de la description du champ, élément "Ligne sélectionnée").

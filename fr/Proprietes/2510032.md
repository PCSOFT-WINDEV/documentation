
## Déplaçable (Propriété)

***En anglais : Movable***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Déplaçable** permet de :

- Savoir si une colonne d'un champ Table peut être déplacée ou non par l'utilisateur,

- Autoriser et/ou empêcher le déplacement d'une colonne d'un champ Table par l'utilisateur.





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Permettre le déplacement de la colonne "Nom" par l'utilisateur
COL_Nom.Déplaçable = Vrai
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Savoir si une colonne d'un champ Table peut être déplacée ou non

`<Résultat> = <Colonne>.Déplaçable`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la colonne spécifiée peut être déplacée par l'utilisateur, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Colonne> : (Nom de champ)`**

Nom de la colonne de champ Table à manipuler.  


<a name="SYNTAXE2"></a>

### Autoriser et/ou empêcher le déplacement d'une colonne d'un champ Table

`<Colonne>.Déplaçable = <Déplacement autorisé>`
---

**`<Colonne> : (Nom de champ)`**

Nom de la colonne du champ Table à manipuler.

**`<Déplacement autorisé> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la colonne spécifiée peut être déplacée par l'utilisateur, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Par défaut, une colonne d'un champ Table peut être déplacée par l'utilisateur. Il est possible de modifier cette option sous l'éditeur de fenêtres dans l'onglet "Général" de la description de la colonne.

- Lorsque l'option "Déplaçable" d'une colonne est cochée, l'utilisateur peut modifier la position de cette colonne (vers la droite ou vers la gauche). Pour réaliser cette action, l'utilisateur doit :

	- cliquer sur l'entête de cette colonne et maintenir le bouton gauche de la souris enfoncé,

	- déplacer la colonne jusqu'à sa nouvelle position puis relâcher le bouton gauche de la souris.

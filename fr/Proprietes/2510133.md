


## Vertical (Propriété)

***En anglais : Vertical***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Vertical** permet de :

- Savoir si un champ est horizontal ou vertical.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Modifier le sens d'affichage des champs Table. Un champ Table peut être affiché verticalement (mode classique) ou horizontalement. Dans ce cas, les colonnes du mode classique deviennent les lignes du champ Table.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Modifier le sens d'affichage des champs Zone répétée. Un champ Zone répétée peut être affiché verticalement (mode classique) ou horizontalement. 




**Remarques** : 

- Le sens d'orientation d'un champ est défini dans l'onglet "Général" de la description du champ.

- ![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) La propriété **Vertical** est utilisable uniquement sur les champs de type Jauge.





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 
```wl
// Connaître le sens d'orientation du champ "ASC_Ascenseur"
ResSens = ASC_Ascenseur.Vertical
```


<a name="Exemple2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 
```wl
// Passer le champ Table TABLE_Client en affichage horizontal
TABLE_Client.Vertical = Faux
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le sens d'orientation des champs

`<Résultat> = <Champ manipulé>.Vertical`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le champ spécifié est vertical, 

- <u><u><u><u>Faux</u></u></u></u> si le champ spécifié est horizontal.




**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler.  


<a name="SYNTAXE2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 
### Modifier le sens d'orientation des champs Table

`<Champ Table>.Vertical = <Nouveau sens>`
---

**`<Champ Table> : (Nom de champ)`**

Nom du champ Table à manipuler.

**`<Nouveau sens> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le champ Table doit être affichée verticalement (mode d'affichage classique des champs Table), 

- <u><u><u><u>Faux</u></u></u></u> si le champ Table doit être affichée horizontalement.  





<a name="SYNTAXE3"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 
### Modifier le sens d'orientation des champs Zone répétée

`<Champ Zone répétée>.Vertical = <Nouveau sens>`
---

**`<Champ Zone répétée> : (Nom de champ)`**

Nom du champ Zone répétée à manipuler.

**`<Nouveau sens> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le champ Zone répétée doit être affiché verticalement (mode d'affichage classique des champs Zone répétée), 

- <u><u><u><u>Faux</u></u></u></u> si le champ Zone répétée doit être affichée horizontalement.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Limites
<a name="limites_ELTPARAGRAPHE000125"></a>![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) La propriété **Vertical** s'applique uniquement aux :

- champs Jauge.

- champs Spin.

- champs Potentiomètre.

- champs Ascenseur.

- champs Séparateur.

- champs Carrousel.

- champs Table.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) champs Range slider.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) champs Planning.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) champs Zone répétée.


![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) La propriété **Vertical** s'applique uniquement aux champs Jauge.

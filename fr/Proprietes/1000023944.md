
## LibelléHTML (Propriété)

***En anglais : CaptionHTML***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **LibelléHTML** permet de manipuler un libellé sous forme de code HTML. Cette propriété est utilisable sur :

- les champs d'une page. Pour plus de détails, consultez l'aide de la propriété [LibelléHTML](../Proprietes/1000020922.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [diagTexte](../WDLang1/1410088393.md).


## Adresse

***En anglais : Address***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le mot-clé **Adresse** peut correspondre : 

- à la propriété **Adresse** utilisable sur un champ Conférence. Pour plus de détails, consultez [Adresse](../Proprietes/2510017.md).

- à la propriété **Adresse** utilisable sur : 

	- une variable de type [gglAdressePostale](../WDLang5/1000017575.md).

	- une variable de type [gglContact](../WDLang5/1000017434.md).

	- une variable de type [gglEmail](../WDLang5/1000017571.md).

	- une variable de type [gglMessagerieInstantanée](../WDLang5/1000017569.md).

	- une variable de type [COMObjet](../WDLang1/1000018983.md).

	- une variable générée automatiquement pour représenter [un Webservice importé](../WDLang3/3086001.md).

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable générée automatiquement pour représenter [un Webservice REST importé](../WDLang3/1410087809.md).

	- une variable de type [Description d'API](../WDLang1/1000019149.md).

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png) une variable de type [sshSession](../WDLang3/1000021157.md).

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png) une variable de type [mqttSession](../WDLang3/1000023023.md).

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [adminWBCompte](../WDLang2/1410089382.md).

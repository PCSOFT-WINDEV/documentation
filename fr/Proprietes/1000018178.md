
## Cadre (Propriété)

***En anglais : Border***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Cadre** permet de connaître et de modifier le cadre d'un élément. Cette propriété est utilisable sur :

- le cadre du champ d'un état. Pour plus de détails, consultez [Cadre](../Proprietes/1000018157.md).

- une variable de type [xlsCellule](../WDLang5/1000017472.md). 

- une variable de type [gglParamètreCarte](../WDLang5/1000017505.md).

- une variable de type [OrgaElément](../WDLang1/1000019713.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [pdfSignature](../WDLang6/1000022255.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [docParagraphe](../WDLang1/1000022483.md) (caractéristiques de mise en page).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [docNiveauNumérotation](../WDLang1/1000022799.md).

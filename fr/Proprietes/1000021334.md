
## BarreDeFormuleVisible (Propriété)

***En anglais : FormulaBarVisible***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **BarreDeFormuleVisible** permet de : 

- Savoir si une barre de formule est affichée dans un champ Tableur. 

- Afficher ou non une barre de formule dans un champ Tableur. 





![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Tableur_Barre_Formule.gif)


Rappel : 

- Dans un champ Tableur, la barre de formule permet la saisie de formules et de noms.

- Cette propriété correspond à l'option "Avec barre de saisie de formule" présente dans l'onglet "Détail" de la fenêtre de description du champ Tableur.



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI INT_BarreFormule ALORS
	TBLR_Tableur.BarreDeFormuleVisible = Vrai
SINON
	TBLR_Tableur.BarreDeFormuleVisible = Faux
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Savoir si une barre de formule est affichée dans un champ Tableur

`<Résultat> = <Champ Tableur>.BarreDeFormuleVisible`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la barre de formule est visible,

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 




**`<Champ Tableur> : (Nom de champ)`**

Nom du champ Tableur à manipuler. 


<a name="SYNTAXE2"></a>

### Afficher ou non une barre de formule dans un champ Tableur

`<Champ Tableur>.BarreDeFormuleVisible = <Affichage>`
---

**`<Champ Tableur> : (Nom de champ)`**

Nom du champ Tableur à manipuler. 

**`<Affichage> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> pour rendre la barre de formule visible,

- <u><u><u><u>Faux</u></u></u></u> pour rendre la barre de formule invisible. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

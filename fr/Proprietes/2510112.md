


## Sélectionnée (Propriété)

***En anglais : Selected***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Sélectionnée** permet de :

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Savoir si une ligne est sélectionnée (dans un champ Liste, Liste image ou Table) et sélectionner une ligne (dans un champ Liste, Liste image ou Table). 
	**Remarque** : A la différence de la fonction [ListeSelect](../WDLang1/3049011.md), cette propriété permet de savoir directement si la ligne à l'indice spécifié est sélectionnée ou non.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Savoir si une colonne ou une cellule d'un champ Table est sélectionnée et sélectionner une colonne ou une cellule d'un champ Table (cas des champs Table avec sélection de la ligne ou de la colonne).   





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) 
```wl
// Sélectionne la ligne 5 du champ Liste VILLE
LISTE_Ville[5].Sélectionnée = Vrai
```

## Syntaxe
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) 
### Savoir si un élément est sélectionné

`<Résultat> = <Champ manipulé>[<Indice de la ligne>].Sélectionnée`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'élément est sélectionné, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler : 

- Nom du champ Liste, Liste image ou Table : permet de savoir si la ligne est sélectionnée.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Nom de la colonne : permet de savoir si la cellule est sélectionnée. Si l'indice de la ligne n'est pas spécifié, permet de savoir si la colonne est sélectionnée. 




**`<Indice de la ligne> : (Entier)`**

Indice de la ligne à manipuler.  


<a name="SYNTAXE2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) 
### Sélectionner un élément

`<Champ manipulé>[<Indice de la ligne>].Sélectionnée = <Sélection>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler : 

- Nom du champ Liste, Liste image ou Table : permet de sélectionner ou non une ligne.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Nom de la colonne : permet de sélectionner ou non une cellule. Si L'indice de la ligne n'est pas spécifié, permet de sélectionner ou non la colonne de la table. 




**`<Indice de la ligne> : (Entier)`**

Indice de la ligne à manipuler.

**`<Sélection> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'élément doit être sélectionné, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.

## Remarques
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) 

### Champ Liste monosélection et multisélection
<a name="champ_liste_monoselection_multiselection_ELTPARAGRAPHE000150"></a>Dans le cas d'un champ Liste multisélection, la propriété **Sélectionnée** permet de :

- connaître les lignes sélectionnées.

- sélectionner plusieurs lignes.




Dans le cas d'un champ Liste monosélection, lorsque une ligne est sélectionnée avec la propriété **Sélectionnée**, la ligne précédemment sélectionnée est automatiquement dé-sélectionnée.


<a name="NOTE0_2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) 

### Limites
<a name="limites_ELTPARAGRAPHE000174"></a>La propriété **Sélectionnée** s'applique uniquement aux :

- Champs Table.

- Champs Liste.

- Champs Liste image.

- Colonnes de table.

- Cellules de table.


![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) La propriété **Sélectionnée** s'applique uniquement aux champs Liste.




## BalayageVertical (Propriété)

***En anglais : VerticalSwipe***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **BalayageVertical** permet de connaître et de modifier le sens du balayage dans une fenêtre interne avec balayage.

Remarque : Cette propriété correspond à l'option "Orientation du balayage" présente dans l'onglet "UI" de la description du champ Fenêtre Interne. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Evénement : Changement d'orientation de la fenêtre
// Le code suivant permet de toujours balayer sur le même axe
FI_MaFenInterne1.BalayageVertical = PAS FI_MaFenInterne1.BalayageVertical
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître le sens du balayage

`<Résultat> = <Champ Fenêtre interne>.BalayageVertical`
---

**`<Résultat> : (booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le balayage s'effectue verticalement, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire : le balayage s'effectue horizontalement.  




**`<Champ Fenêtre interne> : (Nom de champ)`**

Nom du champ Fenêtre interne manipulé. 


<a name="SYNTAXE2"></a>

<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Cette propriété est sans effet si la fonctionnalité de balayage n'est pas activée sur le champ Fenêtre interne. Pour plus de détails, consultez [Changement du contenu d'une fenêtre par balayage](../WDChamp/1013277.md).

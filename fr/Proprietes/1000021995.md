


## TypeSignature (Propriété)

***En anglais : SignatureType***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **TypeSignature** permet de définir et de connaître le type de signature à utiliser dans un champ Signature.

Remarque : Cette propriété correspond à l'option "Type de signature" présente dans l'onglet "Général" de la fenêtre de description du champ Signature. Ce champ est disponible uniquement dans un état. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Le champ Signature utilise une signature de certification
SIG_MaSignature.TypeSignature = typeSignatureCertification
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître le type de signature utilisé par le champ Signature

`<Résultat> = <Champ Signature>.TypeSignature`
---

**`<Résultat> : (Constante de type Entier)`**

Type de signature : 


|   |   |
| --- | --- |
| *typeSignatureApprobation* | Signature d'approbation.<br>Une signature d'approbation correspond à une marque appliquée par une tierce personne pour indiquer qu'elle approuve le contenu du document. |
| *typeSignatureCertification* | Signature de certification.<br>Une signature de certification correspond à un sceau appliqué sur le document pour garantir son intégrité. |



**`<Champ Signature> : (Nom de champ)`**

Champ Signature à manipuler. 


<a name="SYNTAXE2"></a>

### Modifier le type de signature utilisé par le champ Signature

`<Champ Signature>.TypeSignature = <Nouveau type>`
---

**`<Champ Signature> : (Nom de champ)`**

Champ Signature à manipuler.

**`<Nouveau type> : (Constante de type Entier)`**

Type de signature à utiliser : 


|   |   |
| --- | --- |
| *typeSignatureApprobation* | Signature d'approbation.<br>Une signature d'approbation est comme une marque appliquée par une tierce personne pour indiquer qu'elle approuve le contenu du document. |
| *typeSignatureCertification* | Signature de certification. <br>Une signature de certification est comme un sceau appliqué sur le document pour garantir son intégrité. |

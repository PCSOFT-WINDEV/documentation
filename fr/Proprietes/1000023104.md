
## CheminFichier (Propriété)

***En anglais : FilePath***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **CheminFichier** permet de connaître : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) le nom du fichier xlsx associé à un champ Tableur. Ce nom est initialisé par les fonctions [TableurCharge](../WDLang1/1000021340.md) et [TableurSauve](../WDLang1/1000021341.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) le nom du fichier associé à un champ Editeur d'images. Ce nom est initialisé par les fonctions [PicOuvre](../WDLang1/1000023379.md) et [PicSauve](../WDLang1/1000023381.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) le nom du fichier PDF associée à un champ Lecteur PDF. Ce nom est initialisé par la fonction [LecteurPDFOuvre](../WDLang1/1000023452.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) le nom du fichier DOCX associé à un champ Traitement de texte. Ce nom est initialisé par les fonctions [DocOuvre](../WDLang1/1000022622.md) et [DocSauve](../WDLang1/1000022387.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) le nom du fichier wddiag associé à un champ Editeur de diagrammes. Ce nom est initialisé par les fonctions [DiagrammeCharge](../WDLang1/1410088123.md) et [DiagrammeSauve](../WDLang1/1410088122.md). 





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 
```wl
SI TBLR_MonTableur.CheminFichier <> "" ALORS
	LIB_TitreTableur = TBLR_MonTableur.CheminFichier
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître le nom du fichier associé au champ 

`<Résultat> = <Champ à manipuler>.CheminFichier`
---

**`<Résultat> : (Chaîne de caractères)`**

Nom du fichier associé au champ. 

**`<Champ à manipuler> : (Nom du champ)`**

Nom du champ à manipuler : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Champ Tableur, 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Champ Editeur d'images, 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Champ Traitement de texte, 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Champ Lecteur PDF. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Champ Editeur de diagrammes.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Par défaut, la propriété **CheminFichier** correspond à une chaîne vide.

- Cette propriété est disponible uniquement sur : 

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) les champs Tableur présents dans une fenêtre. 

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) les champs Editeur d'images.

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) les champs Traitement de texte.

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) les champs Lecteur PDF. 

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) les champs Editeur de diagrammes.




## PoliceGras (Propriété)

***En anglais : FontBold***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **PoliceGras** permet de :

- Savoir si le texte affiché est en gras ou non.

- Modifier la graisse du texte.





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Mettre le texte affiché dans le champ "LIB_ChampLibellé" en gras
LIB_ChampLibellé.PoliceGras = Vrai
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Savoir si le texte affiché dans un champ est en gras

`<Résultat> = <Champ manipulé>.PoliceGras`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le texte affiché est en gras, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler.  


<a name="SYNTAXE2"></a>

### Modifier la mise en forme gras du texte affiché dans un champ

`<Champ manipulé>.PoliceGras = <Gras/Non gras>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler.

**`<Gras/Non gras> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le texte affiché doit être en gras, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Limites
<a name="limites_ELTPARAGRAPHE000098"></a>Dans un état, la propriété **PoliceGras** peut être utilisée uniquement sur :

- un champ Libellé.

- un champ Lien URL.

- un champ Calculé.

- un champ Prédéfini.




![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Dans une fenêtre, la propriété **PoliceGras** ne peut pas être utilisée sur :

- un champ Spin,

- un champ Web Camera,

- un champ Conférence.


Remarque : Cette propriété s'applique sur les champs "Liste image" en mode d'affichage "Liste classique" uniquement.

![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) En Java, la propriété **PoliceGras** s'applique uniquement aux éléments suivants :

- Champ Bouton.

- Champ Libellé.

- Champ Combo.

- Champ Arbre.

- Colonne de table.

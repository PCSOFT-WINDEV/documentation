
## Avancement (Propriété)

***En anglais : Progress***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Avancement** permet de connaître l'état d'avancement d'une tâche. Cette propriété est utilisable sur :

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [TâcheGantt](../WDLang1/1000020881.md).

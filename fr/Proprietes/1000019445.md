


## RessourceSélectionnée (Propriété)

***En anglais : SelectedResource***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **RessourceSélectionnée** renvoie le nom de la ressource correspondant à la sélection de l'utilisateur dans un champ Planning. 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Code de clic sur le bouton BTN_NOUVEAU_RDV
MonRDV est un RendezVous
SI PLN_Planning1.RessourceSélectionnée = "" ALORS
	RETOUR
FIN

MonRDV.DateDébut = PLN_Planning1.DateDébut
MonRDV.DateFin = PLN_Planning1.DateFin
MonRDV.Ressource = PLN_Planning1.RessourceSélectionnée
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Champ Planning>.RessourceSélectionnée`
---

**`<Résultat> : (Chaîne de caractères)`**



- Nom de la ressource sélectionnée, 

- Chaîne vide ("") si aucune ressource n'est sélectionnée. 


Si les ressources du planning ont été créées avec la fonction [PlanningAjouteRessource](../WDLang1/1000019279.md) en utilisant la fonction [gValeurMémorisée](../WDLang1/1410088363.md), le nom de la ressource correspondra à la valeur précisée dans la fonction [gValeurMémorisée](../WDLang1/1410088363.md) pour cette ressource.  

**`<Champ Planning> : (Nom de champ)`**

Nom du champ Planning manipulé.

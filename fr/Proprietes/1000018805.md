
## Namespace (Propriété)

***En anglais : Namespace***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Namespace** permet de gérer les espaces de nommage. Cette propriété est utilisable sur :

- une variable de type [xmlAttribut](../WDLang5/1000018799.md).

- une variable de type [xmlNoeud](../WDLang5/1000018786.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [xmlLecteur](../WDLang5/1000023392.md).

- une variable de type [memcachedConnexion](../WDLang4/1000023339.md).


## Alignement

***En anglais : Alignment***
	

<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Alignement** permet de gérer l'alignement d'un élément. Cette propriété est utilisable sur :

- une variable de type [Marqueur](../WDLang3/1000019940.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [docParagraphe](../WDLang1/1000022483.md) (caractéristiques de mise en page).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [docNiveauNumérotation](../WDLang1/1000022799.md).

- une variable de type [CarteImage](../WDLang3/1000025523.md).

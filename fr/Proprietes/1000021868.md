


## RetourALaLigneAuto (Propriété)

***En anglais : AutoLineWrap***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **RetourALaLigneAuto** permet de : 

- savoir si le retour à la ligne est automatiquement géré dans un champ de saisie multiligne, dans une colonne d'un champ Table (ou Table hiérarchique) ou dans un entête de ligne d'un champ Tableau croisé dynamique. 

- indiquer le mode de gestion du retour à la ligne dans un champ de saisie multiligne, dans une colonne d'un champ Table (ou Table hiérarchique) ou dans un entête de ligne d'un champ Tableau croisé dynamique.





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Modification de l'interrupteur "Retour à la ligne automatique"
SAI_TexteMultiligne.RetourALaLigneAuto = INT_RetourAuto
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Savoir si le retour à la ligne est automatiquement géré

`<Résultat> = <Champ manipulé>.RetourALaLigneAuto`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le retour à la ligne automatique est actif, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler : 

- Champ de saisie multiligne, 

- Colonne d'un champ Table ou Table hiérarchique.

- Champ Tableau croisé dynamique. 





<a name="SYNTAXE2"></a>

### Indiquer le mode de gestion du retour à la ligne

`<Champ manipulé>.RetourALaLigneAuto = <Gestion du retour à la ligne>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler : 

- Champ de saisie multiligne, 

- Colonne d'un champ Table ou Table hiérarchique.

- Champ Tableau croisé dynamique. 




**`<Gestion du retour à la ligne> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> pour activer le retour à la ligne automatique sur le champ,   

- <u><u><u><u>Faux</u></u></u></u> pour désactiver le retour à la ligne automatique sur le champ.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Si la propriété **RetourALaLigneAuto** vaut <u><u><u><u>Faux</u></u></u></u>, un scrolling horizontal est possible si les lignes sont trop longues (avec un ascenseur visible ou non selon les paramètres du champ de saisie). 

- Si la propriété **RetourALaLigneAuto** vaut <u><u><u><u>Vrai</u></u></u></u>, les lignes trop longues sont découpées sur plusieurs lignes. 

- Cette propriété est sans effet sur un champ RTF, HTML ou un champ de saisie non multiligne.


## NombreColonne (Propriété)

***En anglais : NumberColumn***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **NombreColonne** permet de connaître et de modifier le nombre de colonnes d'un élément. Cette propriété est utilisable sur :

- les champs d'une fenêtre. Pour plus de détails, consultez [NombreColonne sur les champs d'une fenêtre](../Proprietes/2510083.md).

- une variable de type [xlsDocument](../WDLang5/1000017464.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [docFormatMulticolonne](../WDLang1/1000023580.md).

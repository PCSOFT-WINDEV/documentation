
## Bordure (Propriété)

***En anglais : Border***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Bordure** permet de définir les caractéristiques de la bordure d'un document. Cette propriété est utilisable sur : 

- une variable de type [diagForme](../WDLang1/1410088082.md). 

- une variable de type [diagSélection](../WDLang1/1410088172.md).

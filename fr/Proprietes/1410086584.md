
## Compile

***En anglais : Compile***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le mot-clé **Compile** peut correspondre :

- à la fonction **[Compile](../WDLang1/3013015.md)** qui permet de compiler dynamiquement une procédure dont le code source est fourni.. 

- à l'instruction de compilation **&lt;COMPILE SI&gt;** qui permet de compiler ou non un code WLangage selon une condition (notamment utilisé pour compiler un code selon une plateforme ou un type de configuration).

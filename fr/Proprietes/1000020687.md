
## Calendrier

***En anglais : Calendar***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le mot-clé **Calendrier** peut correspondre : 

- à une variable de type [Calendrier](../WDLang3/1000020663.md). 

- à la propriété **Calendrier** d'une variable de type [RendezVous](../WDLang1/1000019244.md).

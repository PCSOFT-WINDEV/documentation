
## ModeSignature (Propriété)

***En anglais : SignatureMode***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **ModeSignature** permet de connaître et de modifier le mode de signature d'un champ Signature présent dans un état. 

Remarque : Cette propriété correspond à l'option "Mode de signature" présente dans l'onglet "Général" de la fenêtre de description du champ Signature. Ce champ est disponible uniquement dans un état. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI SIG_MaSignature.ModeSignature = iSignature_PKCS7 ALORS
	bModeChorus = Faux
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le mode de signature d'un champ Signature

`<Résultat> = <Champ Signature>.ModeSignature`
---

**`<Résultat> : (Constante de type entier)`**

Mode de signature du champ spécifié. Les valeurs possibles sont les suivantes : 



|   |   |
| --- | --- |
| *iSignature_CAdES_B* | Format de signature électronique CAdES Basic. |
| *iSignature_CAdES_C* | Format de signature électronique CAdES C. |
| *iSignature_CAdES_T* | Format de signature électronique CAdES Trust token. |
| *iSignature_CAdES_X* | Format de signature électronique CAdES X Type 2. |
| *iSignature_CAdES_XL* | Format de signature électronique CAdES XL Type 2. |
| *iSignature_PKCS7* | Format de signature PKCS7. Ce format est un format standard de Adobe non approuvé par la norme ETSI et Chorus. |


Rappel : CAdES (CMS Advanced Electronic Signature) est une norme qui permet la signature "enveloppée" ou "détachée".

**`<Champ Signature> : (Nom de champ)`**

Nom du champ Signature à manipuler.  


<a name="SYNTAXE2"></a>

### Modifier le mode de signature d'un champ Signature

`<Champ Signature>.ModeSignature = <Nouveau mode de signature>`
---

**`<Champ Signature> : (Nom de champ)`**

Nom du champ Signature à manipuler.  

**`<Nouveau mode de signature> : (Constante de type entier)`**

Nouveau mode de signature du champ spécifié. Les valeurs possibles sont les suivantes : 



|   |   |
| --- | --- |
| *iSignature_CAdES_B* | Format de signature électronique CAdES Basic. |
| *iSignature_CAdES_C* | Format de signature électronique CAdES C. |
| *iSignature_CAdES_T* | Format de signature électronique CAdES Trust token. |
| *iSignature_CAdES_X* | Format de signature électronique CAdES X Type 2. |
| *iSignature_CAdES_XL* | Format de signature électronique CAdES XL Type 2. |
| *iSignature_PKCS7* | Format de signature PKCS7. Ce format est un format standard de Adobe non approuvé par la norme ETSI et Chorus. |


Rappel : CAdES (CMS Advanced Electronic Signature) est une norme qui permet la signature "enveloppée" ou "détachée".

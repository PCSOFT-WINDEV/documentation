


## ImageFondEtat (Propriété)

***En anglais : BackgroundImageState***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **ImageFondEtat** permet de connaître et de modifier le nombre de dessins présents dans l'image associée au cadre d'un champ Bouton.

**Remarques** :

- Chaque dessin de l'image correspond à un état d'un bouton. Les dessins doivent suivre l'ordre suivant : Normal, Enfoncé, Grisé, Focus et Survol.

- Le nombre de dessins présents dans une image est défini sous l'éditeur de fenêtres dans l'onglet "Général" de la description du champ.







<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Modification de l'image associée au cadre du bouton "BTN_Aide"
BTN_Aide.ImageFond ="C:\MesImages\Cadre.bmp"
// Initialisation du nombre de dessins présents dans l'image
// associée au cadre du bouton "BTN_Aide"
BTN_Aide.ImageFondEtat = 3
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le nombre de dessins présents dans l'image associée au cadre d'un champ Bouton

`<Nombre de dessins actuel> = <Champ Bouton>.ImageFondEtat`
---

**`<Nombre de dessins actuel> : (Entier)`**

Nombre de dessins présents dans l'image associée au cadre du bouton spécifié. Ce nombre est compris entre 1 et 5.

**`<Champ Bouton> : (Nom de champ)`**

Nom du champ Bouton à manipuler.  


<a name="SYNTAXE2"></a>

### Modifier le nombre de dessins présents dans l'image associée au cadre d'un champ Bouton

`<Champ Bouton>.ImageFondEtat = <Nouveau nombre de dessins>`
---

**`<Champ Bouton> : (Nom de champ)`**

Nom du champ Bouton à manipuler.

**`<Nouveau nombre de dessins> : (Entier)`**

Nouveau nombre de dessins présents dans l'image associée au cadre du bouton spécifié. Ce nombre doit être compris entre 1 et 5.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
La propriété **ImageFondEtat** s'applique uniquement aux champs Bouton.

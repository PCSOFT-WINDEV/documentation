
## ValeurRenvoyée (Propriété)

***En anglais : ReturnedValue***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **ValeurRenvoyée** permet de gérer la valeur renvoyée. Cette propriété est utilisable sur :

- les fenêtres, les pages et les champs. Pour plus de détails, consultez [ValeurRenvoyée](../Proprietes/2510134.md).

- une variable de type [TâcheParallèle](../WDLang1/1000021220.md).

- une variable de type [Thread](../WDLang1/1000024879.md).

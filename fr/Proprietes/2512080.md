


## RubriqueTexteComplétée (Propriété)

***En anglais : TextItemCompleted***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **RubriqueTexteComplétée** permet de paramétrer la gestion des rubriques texte dans un fichier de données HFSQL. Les rubriques texte peuvent :

- être complétées automatiquement par des espaces (fonctionnement identique à celui de WINDEV 5.5),

- ne pas être complétées par des espaces.




Il est ainsi possible de :

- Savoir si un fichier de données est configuré pour compléter ou non les rubriques texte (fichier de données défini sous l'éditeur d'analyses ou défini par programmation).

- Définir les caractéristiques des rubriques texte (lors de la description du fichier de données par programmation uniquement).









<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Caractéristiques des rubriques texte du fichier de données Commande
Commande.RubriqueTexteComplétée = Vrai
...
// Valide la description du fichier de données
HDécritFichier(Commande)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Savoir si les rubriques texte seront complétées par des espaces

`<Rubriques complétées> = <Fichier de données>.RubriqueTexteComplétée`
---

**`<Rubriques complétées> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le fichier de données est configuré pour que les rubriques texte soient complétées par des espaces, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Fichier de données> : (Chaîne de caractères)`**

Nom du fichier de données manipulé. Ce nom a été défini sous l'éditeur d'analyses ou grâce au type [Description de Fichier](../WDLang4/1514065.md).  


<a name="SYNTAXE2"></a>

### Définir la gestion des rubriques texte (fichier créé par programmation)

`<Fichier de données>.RubriqueTexteComplétée = <Rubriques complétées>`
---

**`<Fichier de données> : (Chaîne de caractères)`**

Nom du fichier de données manipulé. Ce nom vient d'être défini grâce au type [Description de Fichier](../WDLang4/1514065.md).

**`<Rubriques complétées> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le fichier de données doit être configuré pour que les rubriques texte soient complétées par des espaces, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.

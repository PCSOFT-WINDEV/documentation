
## Début (Propriété)

***En anglais : Start***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Début** permet de connaître le début d'un élément ou événement. Cette propriété est utilisable sur :

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [EvénementTimeline](../WDLang1/1000020612.md). 

- une variable de type [edtHTMLSélection](../WDLang1/1000026093.md).

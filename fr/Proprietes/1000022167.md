


## Existe (Propriété)

***En anglais : Exist***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Existe** permet de savoir si un élément d'un tableau associatif existe ou non. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Tableau associatif d'entiers 
// indexé sur des chaînes et avec doublons
taIDParClient est un tableau associatif d'entiers
// Ajout de l'identifiant du client "A"
taIDParClient["A"] = 55
// Ajout de l'identifiant du client "B" 
taIDParClient["B"] = 321

SI taIDParClient["A"]..Existe = Vrai ALORS 
	Info("Il existe au moins un élément A")
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Elément manipulé>.Existe`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'élément spécifié existe (n'est pas vide), 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Elément manipulé> : (Elément d'un tableau associatif)`**

Nom de l'élément à manipuler (élément d'un tableau associatif).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
La propriété **Existe** permet de savoir si un élément d'un tableau associatif existe ou non. 

- Dans le cas d'un tableau associatif sans doublons, la propriété **Existe** est à <u><u><u><u>Vrai</u></u></u></u> si l'élément existe. 

- Dans le cas d'un tableau associatif avec doublons, la propriété **Existe** est à <u><u><u><u>Vrai</u></u></u></u> si au moins un élément existe.


## GUIDFichier (Propriété)

***En anglais : GUIDFile***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **GUIDFichier** renvoie le GUID du fichier de données défini dans l'analyse qui a permis de créer le fichier de données. Cet identifiant est utilisé dans le fichier .REP (pour plus de détails, consultez [Gestion du fichier .REP](../WDLang4/3044186.md)).

**Remarque** : Le GUID du fichier peut également être connu dans l'éditeur d'analyses (onglet "Notes" de la description du fichier de données).




<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=GUIDAnalyse%20-%20HC%20N%B0001.gif)



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// GUID du fichier de données "Client"
Trace(Client.GUIDFichier)
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<GUID Fichier> = <Fichier de données>.GUIDFichier`
---

**`<GUID Fichier> : (Chaîne de caractères)`**



- GUID du fichier défini dans l'analyse qui a permis de créer le fichier de données. 

- Chaîne vide ("") si le fichier de données a été créé par programmation ou si le fichier de données utilisé n'est pas un fichier au format HFSQL Classic ou Client/Serveur.




**`<Fichier de données> : (Chaîne de caractères)`**

Nom du fichier de données pour lequel le GUID est recherché.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
La propriété [GUIDAnalyse](../Proprietes/2512098.md) permet de connaître le GUID de l'analyse qui a permis de créer le fichier de données.

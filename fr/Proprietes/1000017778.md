


## Catégorie (Propriété)

***En anglais : Category***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Catégorie** permet de connaître et de modifier la catégorie d'un élément. Cette propriété est utilisable sur :

- une variable de type [Email](../WDLang3/1000018713.md). 

- une variable de type [EvénementTimeline](../WDLang1/1000020612.md).

- une variable de type [RendezVous](../WDLang1/1000019244.md).

- une variable de type [rssCanal](../WDLang5/1000017769.md).

- une variable de type [rssEntrée](../WDLang5/1000017792.md).

- la structure [Email](../WDLang3/3032029.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [docPropriété](../WDLang1/1000022945.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [Notification](../WDLang3/1000019441.md).




## IDAutoMax (Propriété)

***En anglais : MaxAutoID***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **IDAutoMax** permet de connaître l'identifiant automatique maximum sur un fichier de données répliqué. **Cette propriété est disponible en lecture seulement**.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connaître l'identifiant automatiquement maximum du fichier de données Client 
ResIDAutoMax = Client.IDAutoMax
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Fichier de données>.IDAutoMax`
---

**`<Résultat> : (Entier)`**



- Valeur maximale de l'identifiant automatique défini pour le fichier de données, 

- 0 si les bornes n'ont pas été définies ou si le fichier de données n'est pas répliqué.




**`<Fichier de données> : (Chaîne de caractères)`**

Nom logique du fichier de données manipulé.

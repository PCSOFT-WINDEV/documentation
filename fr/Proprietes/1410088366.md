
## Interligne (Propriété)

***En anglais : LineSpacing***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Interligne** permet de connaître et de modifier l'interligne défini pour une variable de type [Police](../Motscles/1514045.md). La valeur de l'interligne correspond au facteur multiplicatif de l'interligne simple.

Rappel : L'interligne correspond à l'espace entre les lignes d'un texte. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MaPolice est une Police
MaPolice.Nom = "arial"
MaPolice.UnitéTaille = unitéPoint
MaPolice.Taille = 12
MaPolice.Interligne = 1.5
MaPolice.Interlettre = 5
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître l'interligne défini pour une variable de type Police

`<Résultat> = <Police>.Interligne`
---

**`<Résultat> : (Réel)`**

Facteur multiplicatif de l'interligne simple. 

**`<Police> : (Variable de type Police)`**

Nom de la variable de type [Police](../Motscles/1514045.md) à manipuler. 


<a name="SYNTAXE2"></a>

### Modifier l'interligne défini pour une variable de type Police

`<Police>.Interligne = <Interligne>`
---

**`<Police> : (Variable de type Police)`**

Nom de la variable de type [Police](../Motscles/1514045.md) à manipuler.

**`<Interligne> : (Réel)`**

Facteur multiplicatif de l'interligne simple.

## Remarques


### Style d'un champ : Sélection de l'interligne sous l'éditeur
<a name="style_champ_selection_interligne_sous_editeur_ELTPARAGRAPHE000053"></a>Le style d'un champ permet de définir la police utilisée pour les différents éléments du champ (Onglet "Style" de la fenêtre de description du champ). Lors de la saisie des informations de taille de la police sous l'éditeur, il est possible de choisir l'interligne et l'interlettrage pour la police en cours. <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Style_Unit%E9_interligne%20-%20HC%20N%B0005.gif)





### Variable de type Police
<a name="variable_type_police_ELTPARAGRAPHE000060"></a>Une police peut être créée à partir d'une variable de type [Police](../Motscles/1514045.md).

Pour définir les caractéristiques de la police, utilisez :

- soit la fonction [PoliceCrée](../WDLang1/3060002.md).
	

- soit les propriétés [Alphabet](../Proprietes/2512008.md), [Angle](../Proprietes/1410086632.md), [Barré](../Proprietes/2513005.md), [Couleur](../Proprietes/2510071.md), [Gras](../Proprietes/2513001.md), [Italique](../Proprietes/2513007.md), [Interlettre](../Proprietes/1410088367.md), [Interligne](../Proprietes/1410088366.md), [Nom](../Proprietes/2510082.md), [Souligné](../Proprietes/2513013.md), [Taille](../Proprietes/2510125.md) et [UnitéTaille](../Proprietes/1410088365.md).
	![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Seules les propriétés suivantes sont utilisables avec le type **Police** : [Angle](../Proprietes/1410086632.md), [Barré](../Proprietes/2513005.md), [Couleur](../Proprietes/2510071.md), [Gras](../Proprietes/2513001.md), [Italique](../Proprietes/2513007.md), [Nom](../Proprietes/2510082.md), [Souligné](../Proprietes/2513013.md), [Taille](../Proprietes/2510125.md).
	




Cette police pourra être utilisée :

- Pour écrire du texte dans des dessins (fonction [dPolice](../WDLang1/3029020.md)).

- Pour écrire dans les graphes (fonctions [grPoliceEtiquette](../WDLang3/3042012.md), [grPoliceLégende](../WDLang3/3042037.md), [grPoliceTitre](../WDLang3/3042048.md) et [grPoliceTitreAxe](../WDLang3/3042059.md)).

- Pour imprimer (fonction [iPolice](../WDLang5/3046014.md)).

- Dans les champs d'une fenêtre (propriété [Police](../Motscles/1514045.md) des champs d'une fenêtre).

- Dans les champs d'un état (propriété [Police](../Motscles/1514045.md) des champs d'un état).

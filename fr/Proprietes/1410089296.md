
## Liste (Propriété)

***En anglais : List***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Liste** permet de manipuler les différentes listes d'un champ Kanban. Il est ainsi possible par exemple de parcourir, ajouter, modifier ou supprimer les listes d'un champ Kanban.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
POUR TOUT uneListe DE KANBAN_Planning.Liste
	Trace(uneListe.Titre)
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Champ Kanban>.Liste`
---

**`<Résultat> : (Tableau)`**

Tableau de listes de Kanban. 

**`<Champ Kanban> : (Nom de champ)`**

Nom de champ Kanban.

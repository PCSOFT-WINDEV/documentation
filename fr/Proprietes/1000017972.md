


## URL (Propriété)

***En anglais : URL***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **URL** permet de connaître et de modifier une URL. Cette propriété est utilisable sur :

- un lien, un bouton, une image cliquable ou un champ iFrame. Pour plus de détails, consultez [URL](../Proprietes/2510132.md).

- une variable de type [rssDocumentJoint](../WDLang5/1000017789.md).

- une variable de type [rssImage](../WDLang5/1000017799.md).

- une variable de type [rssSource](../WDLang5/1000017803.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [upnpPériphérique](../WDLang3/1000020903.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [upnpImage](../WDLang3/1000020905.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [upnpService](../WDLang3/1000020908.md).

- une variable de type [httpRequête](../WDLang3/1000021158.md).

- une variable de type [restRequête](../WDLang3/1000021481.md).

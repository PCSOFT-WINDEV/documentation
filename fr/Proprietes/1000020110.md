


## LargeurRessource (Propriété)

***En anglais : ResourceWidth***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **LargeurRessource** permet de connaître et de modifier la largeur des ressources dans un champ Planning avec les ressources en colonne. 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI INT_Zoom = Vrai ALORS
	PLN_MonPlanning.LargeurRessource = 200
SINON
	PLN_MonPlanning.LargeurRessource = 100
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître la largeur des ressources

`<Résultat> = <Champ manipulé>.LargeurRessource`
---

**`<Résultat> : (Entier)`**



- Largeur des ressources en pixels si la largeur des ressources a été précédemment modifiée par programmation (avec la propriété **LargeurRessource**).

- -1 dans les cas suivants : 

	- si la largeur des ressources est automatiquement calculée (grâce aux informations données dans la fenêtre de description du champ).

	- si les ressources sont en ligne. 







**`<Champ manipulé> : (Nom de champ)`**

Nom du champ Planning manipulé.  


<a name="SYNTAXE2"></a>

### Modifier la largeur des ressources

`<Champ manipulé>.LargeurRessource = <Nouvelle largeur>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ Planning manipulé.

**`<Nouvelle largeur> : (Entier)`**



- Nouvelle largeur des ressources exprimée en pixels. 

- -1 pour calculer automatiquement la largeur des ressources à partir des informations données dans la fenêtre de description du champ.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Si la largeur des ressources est fixée avec la propriété **LargeurRessource**, plus aucun "ancrage" de la taille des ressources en fonction de la taille disponible n'est effectuée. 

- Seule la valeur -1 peut être utilisée. Les autres valeurs négatives sont ignorées.

- La largeur d'une ressource particulière peut être modifiée grâce à la propriété [Largeur](../Proprietes/2510059.md) en utilisant la syntaxe suivante :
	
	```txt
	Nom_Planning.Ressource["MaRessource"].Largeur = <Nouvelle valeur>
	```

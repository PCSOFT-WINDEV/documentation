


## RetraitDroit (Propriété)

***En anglais : RightIndent***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **RetraitDroit** permet de connaître et modifier l'espace laissé à droite du texte dans un champ. 

Cette propriété est disponible sur : 

- Les champs de saisie. 

- Les champs Libellé.

- Les colonnes de champ Table et de champ Table hiérarchique (entre la fin de la colonne et la fin du texte).





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Réserver 4 pixels de marge à droite
SAI_Date.RetraitDroit = 4
```
<a name="Exemple2"></a>

```wl
// Réserver 4 pixels de marge à droite
LIB_Date.RetraitDroit = 4
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître la valeur de la marge à droite du texte dans un champ

`<Résultat> = <Champ manipulé>.RetraitDroit`
---

**`<Résultat> : (Entier)`**

Nombre de pixels utilisé pour le retrait à droite du texte. Par défaut, le retrait utilisé est de 0 pixels (pas de retrait).  

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler : 

- Champ de saisie. 

- Champ Libellé.

- Colonnes de champ Table et de champ Table hiérarchique.





<a name="SYNTAXE2"></a>

### Modifier la valeur de la marge à droite du texte dans un champ

`<Champ manipulé>.RetraitDroit = <Nouveau retrait>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler : 

- Champ de saisie. 

- Champ Libellé.

- Colonnes de champ Table et de champ Table hiérarchique.




**`<Nouveau retrait> : (Entier)`**

Nombre de pixels à utiliser pour le retrait à droite du texte sur le champ spécifié.

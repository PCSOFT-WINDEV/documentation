


## BarreOutilsVisible (Propriété)

***En anglais : ToolbarVisible***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **BarreOutilsVisible** permet : 

- de savoir si la barre d'outils ou le ruban est affichée dans un champ.  

- d'afficher ou non la barre d'outils ou le ruban dans un champ. 




Dans un champ Tableur, la propriété **BarreOutilsVisible** permet de : 

- savoir si la barre d'outils ou le ruban est affichée. 

- afficher ou non la barre d'outils ou le ruban.


Dans un champ Traitement de texte, Editeur d'images, Lecteur PDF, Editeur HTML ou Editeur de diagrammes, la propriété **BarreOutilsVisible** permet de  :  

- savoir si le ruban est affiché. 

- afficher ou non un ruban.




Rappel :  Cette propriété correspond à l'option : 

- "Barre d'outils" présente dans l'onglet "Détail" de la fenêtre de description du champ Tableur. Il est possible d'afficher une barre d'outils ou un ruban. Cette propriété permet d'afficher ou de masquer l'élément choisi dans la fenêtre de description. Si l'option "Aucune barre" est choisie, la propriété **BarreOutilsVisible** permettra d'afficher ou de rendre invisible la barre d'outils. 

- "Avec ruban" présente dans l'onglet "Général" de la fenêtre de description des champs Traitement de texte, Editeur d'images, Editeur HTML, Editeur de diagrammes ou Lecteur PDF.





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI INT_BarreOutils ALORS
	TBLR_Tableur.BarreOutilsVisible = Vrai
SINON
	TBLR_Tableur.BarreOutilsVisible = Faux
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Savoir si une barre d'outils (ou le ruban) est affichée dans le champ

`<Résultat> = <Champ manipulé>.BarreOutilsVisible`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la barre d'outils (ou le ruban) est visible,

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 




**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler : 

- Champ Tableur. 

- Champ Traitement de texte.

- Champ Editeur d'images.

- Champ Lecteur PDF.

- Champ Editeur HTML.

- Champ Editeur de diagrammes.





<a name="SYNTAXE2"></a>

### Afficher ou non une barre d'outils (ou un ruban) dans un champ

`<Champ manipulé>.BarreOutilsVisible = <Affichage>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler : 

- Champ Tableur. 

- Champ Traitement de texte.

- Champ Editeur d'images.

- Champ Lecteur PDF.

- Champ Editeur HTML.

- Champ Editeur de diagrammes.




**`<Affichage> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> pour rendre la barre d'outils (ou le ruban) visible,

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




## FormuleRubriqueCalculée (Propriété)

***En anglais : CalculatedItemExpression***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **FormuleRubriqueCalculée** permet de :

- Connaître la formule SQL définissant la valeur d'une rubrique de type formule (ou rubrique calculée). Cette rubrique a été définie sous l'éditeur d'analyses ou définie par programmation.

- Définir la formule SQL donnant la valeur d'une rubrique de type formule (lors de la description de la rubrique par programmation uniquement).




![Connecteurs Natifs (Accès Natifs)](https://doc.pcsoft.fr/ext/images/fr/AN.png) Cette propriété est disponible pour le Connecteur Natif MySQL. 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
UnFichier est une Description de Fichier
UneRubrique est une Description de Rubrique
UneRubrique.Nom = "DateCmd"
UneRubrique.FormuleRubriqueCalculée = "'Ecrit par ' + CURRENT_USER + ' le ' + SYSDATE"
UneRubrique.PersistanceRubriqueCalculée = Vrai
HDécritRubrique(UnFichier, UneRubrique)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître la formule SQL définissant la valeur d'une rubrique calculée

`<Résultat> = <Fichier de données>.<Rubrique>.FormuleRubriqueCalculée`
---

**`<Résultat> : (Chaîne de caractères)`**

Formule SQL définissant la valeur de la rubrique.

**`<Fichier de données manipulé> : (Nom de fichier)`**

Nom du fichier de données manipulé.

**`<Rubrique manipulée> : (Nom de rubrique)`**

Nom de la rubrique manipulée.


<a name="SYNTAXE2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) 
### Définir la formule SQL donnant la valeur d'une rubrique calculée

`<Rubrique manipulé>.FormuleRubriqueCalculée = <Formule SQL>`
---

**`<Rubrique manipulé> : (Nom de rubrique)`**

Nom de la rubrique manipulée. Ce nom vient d'être défini grâce au type [Description de Rubrique](../WDLang4/1514071.md).

**`<Formule SQL> : (Chaîne de caractères)`**

Formule SQL définissant la valeur de la rubrique.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Une rubrique de type formule (ou rubrique calculée) est une rubrique dont le contenu est calculé en fonction de la formule SQL associée et des valeurs de rubriques dans l'enregistrement courant. Pour plus de détails, consultez [Editeur d'analyses : les différents types de rubriques](../Editeurs/2011026.md).

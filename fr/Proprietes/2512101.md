


## BaseDeDonnées (Propriété)

***En anglais : Database***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **BaseDeDonnées** permet de gérer la base de données associée à une connexion (HFSQL Client/Serveur, OLE DB, ...). Il est possible de :

- Connaître la base de données associée à une connexion (HFSQL Client/Serveur, OLE DB, ...).

- Définir la base de données associée à une connexion  (HFSQL Client/Serveur, OLE DB, ...). Dans ce cas, si la connexion est actuellement ouverte, elle sera fermée, puis ré-ouverte avec les nouveaux paramètres lors d'un premier accès à un fichier utilisant cette connexion.




**Remarque** : La connexion peut correspondre :

- soit à une connexion décrite sous l'éditeur d'analyses,

- soit à une connexion décrite avec les fonctions [HConnecte](../WDLang4/3044263.md), [HDécritConnexion](../WDLang4/3044205.md) ou [HOuvreConnexion](../WDLang4/3044107.md).

- soit à une connexion décrite avec une variable de type [Connexion](../WDLang4/1514073.md).











<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MaConnexion est une Connexion
// Description de la connexion
MaConnexion.Utilisateur = "USER"
MaConnexion.MotDePasse = "PASSWORD"
MaConnexion.Serveur = "MONSERVEUR" 
MaConnexion.BaseDeDonnées = "Test"
MaConnexion.Provider = hAccèsHFClientServeur
MaConnexion.Accès = hOLectureEcriture
MaConnexion.InfosEtendues = "Infos étendues"
MaConnexion.OptionsCurseur = hCurseurClient

HOuvreConnexion(MaConnexion)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître la base de données associée à une connexion

`<Base de données associée> = <Connexion>.BaseDeDonnées`
---

**`<Base de données associée> : (Chaîne de caractères)`**



- Base de données utilisée comme source de données de la connexion.

- Chaîne vide ("") si aucune base de données spécifique n'est associée à la connexion.




**`<Connexion> : (Nom ou variable de type Connexion)`**

Nom de la connexion manipulée. Ce nom a été défini :

- soit sous l'éditeur d'analyses,

- soit avec les fonctions [HDécritConnexion](../WDLang4/3044205.md) ou [HOuvreConnexion](../WDLang4/3044107.md),

- soit avec le type [Connexion](../WDLang4/1514073.md).

### Définir la base de données associée à une connexion

`<Connexion>.BaseDeDonnées = <Nouvelle base de données associée>`
---

**`<Connexion> : (Nom ou variable de type Connexion)`**

Nom de la connexion manipulée. Ce nom a été défini :

- soit sous l'éditeur d'analyses,

- soit avec les fonctions [HDécritConnexion](../WDLang4/3044205.md) ou [HOuvreConnexion](../WDLang4/3044107.md),

- soit avec le type [Connexion](../WDLang4/1514073.md).




**`<Nouvelle base de données associée> : (Chaîne de caractères)`**



- Base de données utilisée comme source de données de la connexion.

- Chaîne vide ("") si aucune base de données spécifique n'est associée à la connexion.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
La propriété [CxInitialCatalog](../Proprietes/2512018.md) permet de connaître la base de données par défaut d'une connexion. Cette propriété est conservée par compatibilité. Il est conseillé d'utiliser la propriété **BaseDeDonnées**.

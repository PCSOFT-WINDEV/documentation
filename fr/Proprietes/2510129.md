
## ValeurAffichée (Propriété)

***En anglais : DisplayedValue***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **ValeurAffichée** permet de connaître la valeur actuellement affichée à l'écran dans un champ d'une fenêtre ou d'une page.






<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connaître la valeur affichée à l'écran dans le champ "Date"
SAI_Date = 18031972
Trace(SAI_Date.ValeurAffichée) // Affiche "18/03/1972"
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Valeur affichée> = <Champ manipulé>.ValeurAffichée`
---

**`<Valeur affichée> : (Chaîne de caractères)`**

Valeur actuellement affichée à l'écran dans le champ spécifié.

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Valeur affichée dans un champ de saisie
<a name="valeur_affichee_dans_champ_saisie_ELTPARAGRAPHE000086"></a>La propriété **ValeurAffichée** retourne la chaîne formatée avec son masque de saisie.


<a name="NOTE0_2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) 

### Valeur affichée dans un champ d'affichage HTML
<a name="valeur_affichee_dans_champ_affichage_html_ELTPARAGRAPHE000099"></a>La propriété **ValeurAffichée** retourne le code HTML contenu entre les balises &lt;BODY&gt; et &lt;/BODY&gt;.

Remarques : 

- ![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) La propriété **ValeurAffichée** est sans effet sur les champs d'affichage HTML : cette propriété ne renvoie rien. 



<a name="NOTE0_3"></a>




### Valeur affichée dans un champ Combo
<a name="valeur_affichee_dans_champ_combo_ELTPARAGRAPHE000122"></a>La propriété **ValeurAffichée** retourne la chaîne formatée affichée :

- dans le champ de saisie si le champ Combo est en saisie,

- dans la ligne en cours si le champ Combo n'est pas en saisie.




**Remarque : utilisation des fonctions gImage et gValeurMémorisée**
Pour récupérer le contenu ajouté avec les fonctions [gImage](../WDLang1/3029042.md) et [gValeurMémorisée](../WDLang1/1410088363.md), utilisez la propriété [ValeurInitiale](../Proprietes/2510128.md). 
Par exemple, si le code suivant est utilisé pour ajouter un élément : 

```wl
ListeAjoute(COMBO_Combo1, gImage(nNumImage,i) + "Client.NomClient" + i + ...
	 ValeurMémorisée("Client.NomClient" + i))
```
le code suivant permet de relire la chaîne d'origine : 

```wl
COMBO_Combo1[1].ValeurInitiale
```

<a name="NOTE0_4"></a>




### Valeur affichée dans un champ Liste
<a name="valeur_affichee_dans_champ_liste_ELTPARAGRAPHE000158"></a>La propriété **ValeurAffichée** retourne :

- la chaîne formatée affichée dans la ligne spécifiée si un numéro de ligne est indiqué,

- la chaîne formatée affichée dans la ligne en cours si aucun numéro de ligne n'est spécifié.





<a name="NOTE0_5"></a>




### Valeur affichée dans une colonne d'un champ Table
<a name="valeur_affichee_dans_une_colonne_champ_table_ELTPARAGRAPHE000173"></a>La propriété **ValeurAffichée** retourne la chaîne formatée affichée dans la cellule correspondant à la colonne spécifiée et à la ligne en cours.


<a name="NOTE0_6"></a>




### Valeur affichée dans une cellule d'un champ Table
<a name="valeur_affichee_dans_une_cellule_champ_table_ELTPARAGRAPHE000185"></a>La propriété **ValeurAffichée** retourne la chaîne formatée affichée dans la cellule correspondant à la colonne spécifiée et à la ligne spécifiée.


<a name="NOTE0_7"></a>




### Valeur affichée dans une ligne d'un champ Table
<a name="valeur_affichee_dans_une_ligne_champ_table_ELTPARAGRAPHE000197"></a>La propriété **ValeurAffichée** retourne les valeurs de chaque colonne de la ligne spécifiée séparées par des *TAB*.

Remarque : La propriété **ValeurAffichée** ne retourne pas la valeur des colonnes invisibles.


<a name="NOTE0_8"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 

### Valeur affichée dans un champ Liste image
<a name="valeur_affichee_dans_champ_liste_image_ELTPARAGRAPHE000215"></a>La propriété **ValeurAffichée** retourne :

- le libellé de l'élément affiché dans la ligne spécifiée si un numéro de ligne est indiqué,

- le libellé de l'élément affiché dans la ligne en cours si aucun numéro de ligne n'est spécifié.



<a name="NOTE0_9"></a>




### Limite
<a name="limite_ELTPARAGRAPHE000229"></a>La propriété **ValeurAffichée** s'applique uniquement aux :

- Champs de saisie,

- Champs Combo,

- Champs Liste et lignes de champ Liste,

- Colonnes et lignes de champ Table,

- Cellules de champ Table,

- Champs Libellés,

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Champs HTML.




![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) En Java, la propriété **ValeurAffichée** s'applique uniquement aux éléments suivants :

- Champ de saisie.

- Champ Liste.

- Champ Combo.

- Colonne de champ Table.


## Login (Propriété)

***En anglais : Login***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Login** permet de connaître et de modifier le login. Cette propriété est utilisable sur :

- une variable de type [sfConnexion](../WDLang5/1000017579.md). 

- une variable de type [sfInfoUtilisateur](../WDLang5/1000018423.md).

- une variable de type [saasUtilisateur](../WDLang3/1000019083.md).

- une variable de type [gpwUtilisateur](../WDLang6/1000021088.md).

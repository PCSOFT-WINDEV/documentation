
## Format (Propriété)

***En anglais : Format***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Format** permet de connaître et de modifier le format d'un élément. Cette propriété est utilisable sur :

- les fichiers de données. Pour plus de détails, consultez [Propriété Format sur les fichiers de données](../Proprietes/2512045.md).

- une variable de type [gglParamètreCarte](../WDLang5/1000017505.md). 

- une variable de type [Icône](../WDLang1/1000020811.md).

- une variable de type [Notification](../WDLang3/1000019441.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [docNiveauNumérotation](../WDLang1/1000022799.md).

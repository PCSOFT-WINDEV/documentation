


## RepeatableReadSupporté (Propriété)

***En anglais : RepeatableReadSupported***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **RepeatableReadSupporté** permet de : 

- Savoir si le fichier de données (fichier défini sous l'éditeur d'analyses ou défini par programmation) peut être utilisé en HFSQL Client/Serveur dans une transaction ayant un mode d'isolation "Repeatable Read".

- Autoriser l'utilisation d'une transaction ayant un mode d'isolation "Repeatable Read" pour un fichier de données (lors de la description du fichier par programmation uniquement).




Attention : si cette option est activée, le fichier est incompatible avec les versions 18 et antérieures.

Cette propriété correspond à l'option "Autoriser les transactions REPEATABLE READ (nécessite HFSQL version 19 ou supérieure)", disponible dans l'onglet "Détail" de la fenêtre de description du fichier dans l'analyse. 

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=RepeatableReadSupport%E9%20-%20HC%20N%B0001.gif)

<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
Si Client.RepeatableReadSupporté = Faux ALORS Info("Utilisation du mode Repeatable Read impossible")
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Savoir si le fichier est utilisable en mode "REPEATABLE READ" 

`<Résultat> = <Fichier de données>.RepeatableReadSupporté`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le fichier peut être utilisé en mode "REPEATABLE READ", 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.  




**`<Fichier de données> : (Nom de fichier)`**

Nom du fichier de données manipulé. Ce nom a été défini sous l'éditeur d'analyses ou grâce au type [Description de Fichier](../WDLang4/1514065.md).  


<a name="SYNTAXE2"></a>

### Définir si le fichier est utilisable en mode "REPEATABLE READ" 

`<Fichier de données>.RepeatableReadSupporté = <Mode "REPEATABLE READ">`
---

**`<Fichier de données> : (Nom de fichier)`**

Nom du fichier de données manipulé. Ce nom vient d'être défini grâce au type [Description de Fichier](../WDLang4/1514065.md).

**`<Mode "REPEATABLE READ"> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le fichier peut être utilisé en mode "REPEATABLE READ", 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.

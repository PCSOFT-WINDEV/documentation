
## MotDePasseCertificatClient (Propriété)

***En anglais : ClientCertificatePassword***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **MotDePasseCertificatClient** permet de gérer le mot de passe associé à un certificat client. Cette propriété est utilisable sur :

- une variable de type [httpRequête](../WDLang3/1000021158.md). 

- une variable de type [restRequête](../WDLang3/1000021481.md). 

- une variable de type [wsRequête](../WDLang3/1000020577.md). 

- une variable de type [apiRequête](../WDLang3/1410088935.md).


## Générateur (Propriété)

***En anglais : Generator***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Générateur** permet de connaître le logiciel générateur d'un élément. Cette propriété est utilisable sur : 

- une variable de type [rssCanal](../WDLang5/1000017769.md).  

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [pdfDocument](../WDLang6/1000024909.md).

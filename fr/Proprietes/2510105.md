


## Multiligne (Propriété)

***En anglais : Multiline***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Dans une fenêtre ou une page, la propriété **Multiligne** permet :

- de savoir si un champ de saisie est multiligne ou de rendre un champ de saisie multiligne.

- de savoir si une colonne de table est multiligne ou de rendre ou non une colonne de table multiligne. Pour que le changement soit pris en compte, il est nécessaire de ré-afficher le champ Table (fonction [TableAffiche](../WDLang1/3074003.md) par exemple) ou de remplir à nouveau le champ Table.




Dans un état, la propriété **Multiligne** permet de :

- savoir si un champ est multiligne

- modifier l'option "Multi-ligne" d'un champ.




**Remarques** : 

- L'option "Multi-ligne" permet d'afficher le contenu d'un champ sur plusieurs lignes et de ne pas tronquer le contenu d'un champ.

- Dans les états, le multiligne est une option des champs Libellé, Rubrique, Calculé et Lien. Cette option est située dans l'onglet "Détail" de la description du champ.





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Le champ "SAI_Adresse" est-il multiligne ?
ResMultiligne = SAI_Adresse.Multiligne
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Savoir si un champ est multiligne

`<Résultat> = <Champ manipulé>.Multiligne`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le champ spécifié est multiligne, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler.  


<a name="SYNTAXE2"></a>

### Modifier l'option "Multiligne" d'un champ

`<Champ manipulé>.Multiligne = <Est multiligne ?>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler.

**`<Est multiligne ?> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le champ spécifié doit être multiligne, 

- <u><u><u><u>Faux</u></u></u></u> si le champ spécifié ne doit pas être multiligne.

## Remarques

### Champs multilignes dans un état
<a name="champs_multilignes_dans_etat_ELTPARAGRAPHE000108"></a>Si un angle d'orientation est appliqué à un champ multiligne dans un état, les retours chariot automatiques ne sont pas pris en compte. Seuls les retours manuels à la ligne  sont pris en compte.
<a name="NOTE0_2"></a>


### Limites
<a name="limites_ELTPARAGRAPHE000115"></a>![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Dans un état, la propriété **Multiligne** s'applique uniquement aux :

- champs Libellé,

- champs Rubrique,

- champs Calculé,

- champs Lien.




![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Dans une fenêtre, la propriété **Multiligne** s'applique uniquement aux :

- champs de saisie.

- colonnes dans les champs Table.




![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) La propriété **Multiligne** s'applique uniquement aux champs de saisie.

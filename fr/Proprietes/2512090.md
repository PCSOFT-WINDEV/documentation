


## TailleEnregistrement (Propriété)

***En anglais : RecordSize***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **TailleEnregistrement** permet d'obtenir la taille (en octets) d'un enregistrement de :

- un fichier de données décrit sous l'éditeur d'analyses,

- un fichier de données décrit par programmation,

- une vue HFSQL,

- une requête.




Cette taille est calculée sans tenir compte des clés composées.






<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connaître la taille d'un enregistrement du fichier de données Commande
ResTaille = Commande.TailleEnregistrement
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Taille> = <Elément manipulé>.TailleEnregistrement`
---

**`<Taille> : (Entier)`**

Taille de l'enregistrement (en octets) sans tenir compte des clés composées.

**`<Elément manipulé> : (Chaîne de caractères)`**

Nom de l'élément dont on veut connaître la taille d'un enregistrement :

- nom d'un fichier de données décrit sous l'éditeur d'analyses ou par programmation,

- nom d'une vue HFSQL,

- nom d'une requête.

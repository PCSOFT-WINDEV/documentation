
## TailleMax (Propriété)

***En anglais : MaxSize***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **TailleMax** permet de connaître et de définir une taille maximale. Cette propriété est disponible sur : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [mongoCollectionOption](../WDLang4/1000022413.md).

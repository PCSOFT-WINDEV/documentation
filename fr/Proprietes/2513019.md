
## StyleFond (Propriété)

***En anglais : BackgroundStyle***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **StyleFond** permet de connaître et de modifier le style d'affichage du fond des cellules de tables et / ou de tables hiérarchiques. L'effet de cette propriété est visible uniquement si la ligne, colonne ou cellule utilise une couleur de fond spécifiée avec la propriété [CouleurFond](../Proprietes/2510022.md). Grâce à cette propriété, il est possible de mettre en relief une cellule spécifique de la table.

**Exemple** : La cellule "Lundi" est colorée en vert clair avec le style "Tube".

![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=StyleFond.gif)

<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Coloration de la cellule JOUR de la ligne en cours
COL_Jour[1].CouleurFond = VertClair
COL_Jour[1].StyleFond = styleTube
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le style de la couleur de fond

`<Style de fond> = <Elément à manipuler>.StyleFond`
---

**`<Style de fond> : (Constante de type Entier)`**

Style de fond actuellement utilisé :


|   |   |
| --- | --- |
| *styleDéfaut* | Valeur par défaut. Aplat simple de couleurs. |
| *styleDégradé* | Dégradé simple (vers le bas). |
| *styleHachureCroixDiagonale* | Hachures du type xxxxx |
| *styleHachureCroixDroite* | Hachures du type +++++ |
| *styleHachureDiagonaleDroite* | Hachures du type \\\\\\\\\\\\ |
| *styleHachureDiagonaleGauche* | Hachures du type ////// |
| *styleHachureHorizontal* | Hachures du type ------ |
| *styleHachureVertical* | Hachures du type \\\\\ |
| *styleTube* | 2 dégradés de la couleur de base, permettant d'obtenir un effet tube. |



**`<Elément à manipuler> : (Chaîne de caractères)`**

Identifie la cellule, la ligne ou la colonne d'un champ Table ou d'un champ Table hiérarchique à manipuler.  


<a name="SYNTAXE2"></a>

### Modifier le style de la couleur de fond

`<Elément à manipuler>.StyleFond = <Nouveau style>`
---

**`<Elément à manipuler> : (Chaîne de caractères)`**

Identifie la cellule, la ligne ou la colonne d'un champ Table ou d'un champ Table hiérarchique à manipuler.  

**`<Nouveau style> : (Constante de type Entier)`**

Style de fond à utiliser :


|   |   |
| --- | --- |
| *styleDéfaut* | Aplat simple de couleurs. Cette option est utilisée par défaut lors de la création d'une table. |
| *styleDégradé* | Dégradé simple (vers le bas). |
| *styleHachureCroixDiagonale* | Hachures du type xxxxx |
| *styleHachureCroixDroite* | Hachures du type +++++ |
| *styleHachureDiagonaleDroite* | Hachures du type \\\\\\\\\\\\ |
| *styleHachureDiagonaleGauche* | Hachures du type ////// |
| *styleHachureHorizontal* | Hachures du type ------ |
| *styleHachureVertical* | Hachures du type \\\\\ |
| *styleTube* | 2 dégradés de la couleur de base, permettant d'obtenir un effet tube. |







<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques




### Fonctionnement de la propriété **StyleFond**
<a name="fonctionnement_propriete_docparampagetitleshort_ELTPARAGRAPHE000145"></a>L'effet de la modification de la propriété **StyleFond** est visible sur une cellule, ligne ou colonne uniquement si la couleur de cette cellule, ligne ou colonne a été définie avec la propriété [CouleurFond](../Proprietes/2510022.md).

**Utilisation des hachures** : La couleur des hachures est fixée par la propriété [Couleur](../Proprietes/2510071.md). Par défaut, la couleur des hachures correspond à la couleur du texte. Pour utiliser une couleur différente pour le texte, utilisez la fonction [gStylo](../WDLang1/3029039.md).

Exemple : 


```wl
TABLE_SansNom1.Couleur = Noir
TABLE_SansNom1.CouleurFond = JauneClair
POUR i = 1 A 10
	TableAjouteLigne(MoiMême)
	TABLE_SansNom1[i].StyleFond = styleHachureDiagonaleGauche
	TABLE_SansNom1.COL_SansNom1[i] = gStylo(RougeClair) + "Ligne " + i
FIN
```



<a name="NOTE0_2"></a>




### Limite
<a name="limite_ELTPARAGRAPHE000175"></a>La propriété **StyleFond** est disponible uniquement sur les cellules, lignes et colonnes des champs Table et Table hiérarchique.


## Flash (Propriété)

***En anglais : Flash***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Flash** permet de connaître et de modifier le mode d'utilisation du flash dans un champ Caméra. 

Remarques : 

- Cette propriété est utilisable uniquement pour les champs Caméra qui ne sont pas en "Mode compatible 26". 

- Cette propriété correspond à l'option "Flash" de l'onglet "Général" de la fenêtre de description du champ Caméra. 





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI CAM_MaCaméra.Flash = camFlashOff ALORS
	CAM_MaCaméra.Flash = camFlashAuto
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître le mode d'utilisation du flash dans un champ Caméra

`<Résultat> = <Champ Caméra>.Flash`
---

**`<Résultat> : (Constante de type Entier)`**

Mode de gestion du flash de la caméra : 


|   |   |
| --- | --- |
| *camFlashAuto* | Flash déclenché automatiquement selon la luminosité.  |
| *camFlashOff* | Le flash est toujours désactivé. |
| *camFlashOn* | Le flash est toujours activé. |


 Par défaut, l'usage du flash est désactivé. 

**`<Champ Caméra> : (Nom de champ)`**

Nom du champ Caméra à manipuler.


<a name="SYNTAXE2"></a>

### Modifier le mode d'utilisation du flash dans un champ Caméra

`<Champ Caméra>.Flash = <Nouveau mode>`
---

**`<Champ Caméra> : (Nom de champ)`**

Nom du champ Caméra à manipuler.

**`<Nouveau mode> : (Constante de type Entier)`**

Mode de gestion du flash de la caméra : 


|   |   |
| --- | --- |
| *camFlashAuto* | Flash déclenché automatiquement selon la luminosité.  |
| *camFlashOff* | Le flash est toujours désactivé. |
| *camFlashOn* | Le flash est toujours activé. |


 Par défaut, l'usage du flash est désactivé.

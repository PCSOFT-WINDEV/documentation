
## URLAuth (Propriété)

***En anglais : AuthURL***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **URLAuth** permet de connaître et de définir l'URL d'autorisation à utiliser (première URL du standard OAuth 2.0). Cette propriété est utilisable sur : 

- une variable de type [OAuth2Paramètres](../WDLang3/1000022212.md). 

- une variable de type [gpwOAuthParamètres](../WDLang6/1000024751.md).




## Défaut (Propriété)

***En anglais : Default***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Défaut** permet de savoir si le paramètre passé à une procédure a été renseigné ou a été affecté avec sa valeur par défaut.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Appel de la procédure
MaProc("A")
```

```wl
// Déclaration de la procédure
PROCEDURE MaProc(p1, po2 = "Y", po3 = "Z")
```

```wl
// Dans cet exemple : 
MesParamètres[1].Défaut // vaut Faux
p1.Défaut               // vaut Faux
MesParamètres[2].Défaut // vaut Vrai
po2.Défaut              // vaut Vrai
MesParamètres[3].Défaut // vaut Vrai
po3.Défaut              // vaut Vrai
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Paramètre manipulé>.Défaut`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le paramètre n'a pas été renseigné lors de l'appel de la procédure (la valeur par défaut a donc été utilisée), 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Paramètre manipulé> : (Nom du paramètre)`**

Nom du paramètre manipulé. Ce paramètre est désigné :

- soit en utilisant le mot-clé **MesParamètres**, en spécifiant l'indice du paramètre manipulé. Par exemple : MesParamètres[1] pour manipuler le premier paramètre passé à la procédure. 

- soit en utilisant directement le nom du paramètre spécifié dans la déclaration de la procédure.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- La propriété **Défaut** prend en compte le passage de paramètres uniquement pour la procédure en cours. Si plusieurs procédures sont imbriquées et transmettent le même paramètre en utilisant ou non la valeur par défaut, la propriété **Défaut** prend en compte uniquement la valeur du paramètre dans la procédure où cette propriété est utilisée.

- La propriété **Défaut** est utilisable uniquement dans le traitement qui déclare l'élément.

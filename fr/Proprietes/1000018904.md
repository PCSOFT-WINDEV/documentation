


## Poids (Propriété)

***En anglais : Weight***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Poids** permet de gérer le poids d'un élément d'un champ TreeMap. Il est possible de : 

- Connaître le poids d'un élément d'un champ TreeMap. 

- Modifier le poids d'un élément d'un champ TreeMap.




**Remarque** : La modification du poids de l'élément entraîne la modification de la taille de cet élément dans le champ TreeMap. Le poids a été spécifié lors de l'ajout de l'élément dans le TreeMap grâce à la fonction [TreeMapAjoute](../WDLang1/1000018814.md). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Récupère l'indice de l'élément sélectionné dans le champ TreeMap
IndiceSélection est un entier = TMAP_TreeMap

// Le libellé affiche le poids de l'élément sélectionné
LIB_TailleFichierNoZippe = TMAP_TreeMap[IndiceSélection].Poids

// Si l'option "Fichier zippé" est cochée, 
// le poids de l'élément du TreeMap correspond à la taille zippée de l'élément 
SI INT_ZIP = Vrai ALORS
	TMAP_TreeMap[IndiceSélection].Poids = PoidsZippe(TMAP_TreeMap[IndiceSélection]) 
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le poids d'un élément d'un champ TreeMap

`<Résultat> = <Elément du champ TreeMap>.Poids`
---

**`<Résultat> : (Réel ou Entier)`**

Poids actuel de l'élément du champ TreeMap spécifié. 

**`<Elément du champ TreeMap> : (Nom de champ)`**

Elément du champ TreeMap dont le poids doit être connu. Ce paramètre est de la forme : 

```txt
<Champ TreeMap>[Indice de l'élément]
```



<a name="SYNTAXE2"></a>

### Modifier le poids d'un élément d'un champ TreeMap

`<Elément du champ TreeMap>.Poids = <Nouveau poids>`
---

**`<Elément du champ TreeMap> : (Nom de champ)`**

Elément du champ TreeMap dont le poids doit être modifié. Ce paramètre est de la forme : 

```txt
<Champ TreeMap>[Indice de l'élément]
```


**`<Nouveau poids> : (Réel ou entier)`**

Nouveau poids de l'élément spécifié.

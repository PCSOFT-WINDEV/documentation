


## Réception (Propriété)

***En anglais : IncomingData***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Réception** permet de connaître et modifier le type de données reçues par le flux.

Par défaut, un flux reçoit du son et de la vidéo.

**Attention** : La transmission en cours n'est pas modifiée. Les changements effectués par cette propriété seront pris en compte uniquement lors de la prochaine connexion.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Modifie le type de données reçues
MonFlux.Reception = FluxSon
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le type de données reçues par le flux

`<Résultat> = <Nom du flux>.Réception`
---

**`<Résultat> : (Constante ou combinaison de constantes de type entier)`**

Type de données actuellement reçues :


|   |   |
| --- | --- |
| *FluxSon* | Les données de type son sont reçues. |
| *FluxVidéo* | Les données de type Vidéo sont reçues. |



**`<Nom du flux> : (Chaîne de caractères)`**

Nom du Flux ou nom du champ Conférence associé au flux.  


<a name="SYNTAXE2"></a>

### Modifier le type de données reçues par le flux

`<Nom du flux>.Réception = <Nouveau type de données>`
---

**`<Nom du flux> : (Chaîne de caractères)`**

Nom du Flux ou nom du champ Conférence associé au flux.

**`<Nouveau type de données> : (Constante ou combinaison de constantes de type Entier)`**

Nouveau type de données à recevoir :


|   |   |
| --- | --- |
| *FluxSon* | Les données de type son sont reçues. |
| *FluxVidéo* | Les données de type Vidéo sont reçues. |

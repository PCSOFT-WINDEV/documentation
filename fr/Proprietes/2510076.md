


## Mémoire (Propriété)

***En anglais : Memory***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Mémoire** permet de savoir si le champ spécifié est un champ "par programmation" (également appelé "mémoire") ou un champ fichier.

Les champs sur lesquels s'applique la propriété **Mémoire** sont les champs Liste, Liste image, Combo, Table, Table hiérarchique ou Zone répétée (suivant le produit utilisé : WINDEV, WINDEV Mobile ou WEBDEV).

**Rappel** : 

- Les champs Liste, Liste image, Combo, Table, Table hiérarchique et Zone répétée **par programmation** manipulent des données stockées en mémoire (définies par programmation). 

- Les champs Liste, Liste image, Combo, Table, Table hiérarchique et Zone répétée **fichier** manipulent des données provenant de fichiers de données ou de requêtes.




**Important : Cette propriété est conservée par compatibilité**. Il est conseillé d'utiliser la propriété [TypeRemplissage](../Proprietes/1000017723.md). 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI LISTE_Liste1.Mémoire = Vrai ALORS
	ListeAjoute(LISTE_Liste1, "Paris")
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Champ manipulé>.Mémoire`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le contenu du champ provient d'ajouts effectués par programmation.

- <u><u><u><u>Faux</u></u></u></u> si le contenu du champ provient d'un fichier de données.




**`<Champ manipulé> : (Nom de champ)`**

Nom du champ (Liste, Liste image, Combo, Zone répétée, Table ou Table hiérarchique) à manipuler.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Champs Liste, Liste image, Combo et Table fichier
<a name="champs_liste_liste_image_combo_table_fichier_ELTPARAGRAPHE000083"></a>Les propriétés suivantes définissent les enregistrements qui sont affichés.

- [FichierParcouru](../Proprietes/2510040.md) : Nom du fichier de données de base.

- [RubriqueParcourue](../Proprietes/2510111.md) : Nom de la rubrique utilisée pour le parcours (le tri est effectué sur cette rubrique).

- [Filtre](../Proprietes/2510042.md) : Si ce filtre n'est pas vide, seuls les enregistrements correspondant au filtre sont affichés.

- [ParcoursAuto](../Proprietes/2510093.md) : Indique si les événements de parcours associés à l'élément sont utilisés.



<a name="NOTE0_2"></a>


### Limite
<a name="limite_ELTPARAGRAPHE000103"></a>![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) En Java, la propriété **Mémoire** s'applique uniquement sur les champs Liste.




## DéplacementRDV (Propriété)

***En anglais : MovementAPT***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **DéplacementRDV** permet de savoir et d'indiquer si l'utilisateur peut déplacer un rendez-vous : 

- dans un champ Planning, 

- dans un champ Agenda.




Cette propriété correspond à l'option "Déplacement" présente dans l'onglet "Détail" de la fenêtre de description des champs Planning et Agenda. 




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Gestion des options d'un champ Planning
SI gbModeModif = Vrai ALORS
	PLN_MonPlanning.DéplacementRDV = Vrai
	PLN_MonPlanning.SaisieDirecteRDV = Vrai
	PLN_MonPlanning.SélectionPériode = Vrai
SINON
	PLN_MonPlanning.DéplacementRDV = Faux
	PLN_MonPlanning.SaisieDirecteRDV = Faux
	PLN_MonPlanning.SélectionPériode = Faux
FIN
```


<a name="Exemple2"></a>



```wl
// Gestion des options d'un champ Agenda
SI gbModeModif = Vrai ALORS
	AGD_MonAgenda.DéplacementRDV = Vrai
	AGD_MonAgenda.SélectionPériode = Vrai
	AGD_MonAgenda.SaisieDirecteRDV = Vrai
	SI PAS EnModeAndroid() _ET_ PAS EnModeIOS() ALORS
		AGD_MonAgenda.ModificationDuréeRDV = Vrai
	FIN
SINON
	AGD_MonAgenda.DéplacementRDV = Faux
	AGD_MonAgenda.SélectionPériode = Faux
	AGD_MonAgenda.SaisieDirecteRDV = Faux
	SI PAS EnModeAndroid() _ET_ PAS EnModeIOS() ALORS
		AGD_MonAgenda.ModificationDuréeRDV = Faux
	FIN
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Savoir si l'utilisateur peut déplacer un rendez-vous

`<Résultat> = <Champ manipulé>.DéplacementRDV`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si les rendez-vous sont déplaçables par l'utilisateur, 

- <u><u><u><u>Faux</u></u></u></u> si les rendez-vous ne sont pas déplaçables par l'utilisateur.




**`<Champ manipulé> : (Nom de champ)`**

Nom du champ manipulé. Ce champ peut correspondre à : 

- un champ Agenda.

- un champ Planning.  





<a name="SYNTAXE2"></a>

### Rendre ou non les rendez-vous déplaçables par l'utilisateur

`<Champ manipulé>.DéplacementRDV = <Déplaçable>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ manipulé. Ce champ peut correspondre à : 

- un champ Agenda.

- un champ Planning.




**`<Déplaçable> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> pour rendre les rendez-vous déplaçables par l'utilisateur, 

- <u><u><u><u>Faux</u></u></u></u> pour ne pas rendre les rendez-vous déplaçables par l'utilisateur.

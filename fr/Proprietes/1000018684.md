
## TailleCompressée (Propriété)

***En anglais : CompressedSize***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **TailleCompressée** permet de connaître la taille compressée d'un élément. Cette propriété est utilisable sur :

- une variable de type [zipArchive](../WDLang3/1000018679.md).

- une variable de type [zipFichierArchivé](../WDLang3/1000019044.md).

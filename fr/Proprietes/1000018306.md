


## ImageTitre (Propriété)

***En anglais : TitleImage***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **ImageTitre** permet de connaître et/ou de modifier l'image affichée à côté du titre d'une colonne d'un champ Table. L'image est positionnée à gauche du titre de la colonne (ou bien à droite si le sens de l'écriture est inversé). 

Cette propriété est également disponible pour les champs Table hiérarchique.

**Remarque** : Cette propriété permet d'inclure simplement une image dans un titre de colonne. La fonction [gImage](../WDLang1/3029042.md) n'est plus nécessaire.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Met une image dans le titre de la colonne
COL_Colonne1.ImageTitre = "smallblue.gif"
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître l'image associée au titre d'une colonne

`<Nom de l'image> = <Colonne manipulée>.ImageTitre`
---

**`<Nom de l'image> : (Chaîne de caractères)`**



- Nom de l'image utilisée dans le titre de la colonne, 

- Chaîne vide ("") si aucune image n'est associée à la colonne.




**`<Colonne manipulée> : (Nom de champ)`**

Nom de la colonne du champ Table ou de Table hiérarchique à manipuler.  


<a name="SYNTAXE2"></a>

### Modifier l'image associée au titre d'une colonne

`<Colonne manipulée>.ImageTitre = <Nouvelle image>`
---

**`<Colonne manipulée> : (Nom de champ)`**

Nom de la colonne du champ Table ou de Table hiérarchique à manipuler.

**`<Nouvelle image> : (Chaîne de caractères)`**



- Nom de l'image (avec son chemin si nécessaire) utilisée dans le titre de la colonne, 

- Chaîne vide ("") si aucune image ne doit être associée à la colonne. 


Si l'image indiquée n'existe pas, aucune image ne sera affichée.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Position de l'image
<a name="position_image_ELTPARAGRAPHE000051"></a>La position de l'image dans le titre de la colonne varie en fonction du sens d'écriture choisi pour l'application :

- Si le sens d'écriture est de droite à gauche, l'image sera placée à droite du titre de la colonne.

- Si le sens d'écriture est de gauche à droite, l'image sera placée à gauche du titre de la colonne.




Rappel : Le sens d'écriture d'une application peut être défini par :

- les options linguistiques du projet : sous le volet "Projet", dans le groupe "Projet", cliquez sur "Description". Dans l'onglet "Langues", choisissez les options linguistiques de type "Divers" et paramétrez l'option "Sens d'écriture".

- la propriété [SensEcriture](../Proprietes/2513020.md).



<a name="NOTE0_2"></a>


### Limite
<a name="limite_ELTPARAGRAPHE000081"></a>La propriété **ImageTitre** s'applique uniquement aux champs de type Table et Table hiérarchique.

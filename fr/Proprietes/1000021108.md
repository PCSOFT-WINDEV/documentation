
## TypeBouton (Propriété)

***En anglais : ButtonType***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **TypeBouton** permet de connaître et de modifier le type d'un champ Bouton : Annulation, Interruption, Validation, ...

Remarque : Cette propriété correspond à l'option "Type" disponible dans l'onglet "UI" de la fenêtre de description d'un  champ Bouton. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
B1 est un Champ
B1 <- ChampCrée("B1", typBouton, 100, 50, 32, 32)
B1.TypeBouton = btnInterruption
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître le type d'un bouton

`<Résultat> = <Champ Bouton>.TypeBouton`
---

**`<Résultat> : (Constante de type Entier)`**

Type du bouton : 


|   |   |
| --- | --- |
| *btnAbandon* | Bouton de type Abandon. |
| *btnAide* | Bouton de type Aide. |
| *btnInterruption* | Bouton de type Interruption. |
| *btnNormal* | Bouton de type Normal. |
| *btnValidation* | Bouton de type Validation. |



**`<Champ Bouton> : (Nom de champ)`**

Nom du champ Bouton à manipuler. 


<a name="SYNTAXE2"></a>

### Modifier le type d'un bouton

`<Champ Bouton>.TypeBouton = <Nouveau type>`
---

**`<Champ Bouton> : (Nom de champ)`**

Nom du champ Bouton à manipuler. 

**`<Nouveau type> : (Constante de type Entier)`**

Type du bouton : 


|   |   |
| --- | --- |
| *btnAbandon* | Bouton de type Abandon. |
| *btnAide* | Bouton de type Aide. |
| *btnInterruption* | Bouton de type Interruption. |
| *btnNormal* | Bouton de type Normal. |
| *btnValidation* | Bouton de type Validation. |

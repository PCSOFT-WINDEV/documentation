
## CadrePageVisible (Propriété)

***En anglais : PageBorderVisible***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **CadrePageVisible** permet de : 

- savoir si le cadre des pages est affiché dans un champ Editeur de diagrammes.  

- afficher ou non le cadre des pages dans un champ Editeur de diagrammes.




Remarque : Lors de l'impression du diagramme, le cadre permet de visualiser les différentes pages nécessaires à l'impression. La taille des pages peut être définie avec :  

- la propriété "MiseEnPage" utilisée sur le champ Editeur de diagrammes, 

- la propriété "MiseEnPage" de la variable de type [Diagramme](../WDLang1/1410088055.md).  



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affiche le cadre des pages et le quadrillage
EDIAG_Diagramme.CadrePageVisible = Vrai
EDIAG_Diagramme.QuadrillageVisible = Vrai
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Savoir si le cadre des pages est affiché

`<Résultat> = <Champ Editeur de diagrammes>.CadrePageVisible`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le cadre des pages est visible,

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 




**`<Champ Editeur de diagrammes> : (Nom de champ)`**

Nom du champ Editeur de diagrammes manipulé. 


<a name="SYNTAXE2"></a>

### Afficher ou non le cadre des pages

`<Champ Editeur de diagrammes>.CadrePageVisible = <Affichage>`
---

**`<Champ Editeur de diagrammes> : (Nom de champ)`**

Nom du champ Editeur de diagrammes manipulé. 

**`<Affichage> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> pour rendre le cadre des pages visible,

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.


## DateHeureFin (Propriété)

***En anglais : EndTimestamp***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **DateHeureFin** permet de gérer la date et l'heure de fin d'une action. Elle est disponible sur les variables de type :

- [hbLecture](../WDLang4/1000021701.md)

- [hbParamètreScan](../WDLang4/1000021715.md)


## Permission (Propriété)

***En anglais : Permission***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le mot-clé **Permission** permet de connaître et de modifier les permissions. 

Ce mot-clé peut correspondre : 

- à la propriété **Permission** utilisable sur :

	- une variable de type [btleCaractéristique](../WDLang3/1000022060.md).

	- une variable de type [btleDescripteur](../WDLang3/1000022064.md). 

	- une variable de type [fbSession](../WDLang5/1000021851.md).

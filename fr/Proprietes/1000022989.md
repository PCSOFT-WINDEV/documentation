
## CléPrivée (Propriété)

***En anglais : PrivateKey***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **CléPrivée** permet de gérer les connexions par clés privées (connexion SSH ou MQTT). Cette propriété est utilisée : 

- par les variables de type [sshSession](../WDLang3/1000021157.md). 

- par les variables de type [mqttSession](../WDLang3/1000023023.md).

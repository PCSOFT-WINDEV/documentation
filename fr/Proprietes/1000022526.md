
## PoliceBarrée (Propriété)

***En anglais : FontStrikeOut***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **PoliceBarrée** permet de gérer si un texte est barré ou non. Cette propriété est utilisable sur : 

- les champs. Pour plus de détails, consultez [PoliceBarrée](../Proprietes/2510091.md). 

- une variable de type [docMiseEnForme](../WDLang1/1000022817.md).




## Propriétés de gestion des polices
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>




Ces propriétés peuvent être utilisées sur une variable de type [Police](../Motscles/1514045.md). 



|   |   |
| --- | --- |
| [Alphabet](../Proprietes/2512008.md) | La propriété **Alphabet** permet de : <br><br>- Connaître l'alphabet actuellement utilisé par une variable de type Police.<br><br>- Modifier l'alphabet utilisé par une variable de type Police.<br><br>- Connaître l'alphabet utilisé par un fichier de données HFSQL. <br><br><br> |
| [Angle](../Proprietes/1410086632.md) | La propriété **Angle** permet de :<br><br>- Connaître l'angle (en degrés) d'une variable de type Police.<br><br>- Modifier l'angle (en degrés) d'une variable de type Police.<br><br><br> |
| [Barré](../Proprietes/2513005.md) | La propriété **Barré** permet de :<br><br>- Savoir si une variable de type Police est barrée.<br><br>- Barrer (ou non) une variable de type Police.<br><br><br> |
| [Condensé](../Proprietes/2513006.md) | La propriété **Condensé** permet de :<br><br>- Savoir si une variable de type Police est condensée.<br><br>- Condenser (ou non) une variable de type Police.<br><br><br>**Cette propriété est conservée par compatibilité.** |
| [Couleur](../Proprietes/2510071.md) | La propriété **Couleur** permet de connaître ou modifier la couleur de la police de :<br><br>- une variable de type Police.<br><br>- un texte d'un champ Libellé ou calculé d'un état.<br><br>- un texte d'un champ d'une fenêtre.<br><br>- un texte d'un champ d'une page.<br><br><br> |
| [Etendu](../Proprietes/2513002.md) | La propriété **Etendu** permet de :<br><br>- Savoir si une variable de type Police est étendue.<br><br>- Étendre ou non une variable de type Police.<br><br><br>**Cette propriété est conservée par compatibilité.** |
| [Gras](../Proprietes/2513001.md) | La propriété **Gras** permet de :<br><br>- Savoir si une variable de type Police est en gras.<br><br>- Modifier l'attribut gras d'une variable de type Police.<br><br><br> |
| [Interlettre](../Proprietes/1410088367.md) | La propriété **Interlettre** permet de connaître et de modifier l'espacement ajouté entre les caractères. |
| [Interligne](../Proprietes/1410088366.md) | La propriété **Interligne** permet de connaître et de modifier l'interligne défini pour une variable de type [Police](../Motscles/1514045.md). |
| [Italique](../Proprietes/2513007.md) | La propriété **Italique** permet de :<br><br>- Savoir si une variable de type Police est en italique.<br><br>- Modifier l'attribut italique d'une variable de type Police.<br><br><br> |
| [Large](../Proprietes/2513012.md) | La propriété **Large** permet de :<br><br>- Savoir si une variable de type Police est élargie.<br><br>- Élargir ou non une variable de type Police.<br><br><br>**Cette propriété est conservée par compatibilité.** |
| [Nom](../Proprietes/2510082.md) | La propriété **Nom** permet de connaître :<br><br>- le nom de la police associée à une variable de type Police.<br><br>- le nom d'un champ, d'un état ou d'un bloc d'un état.<br><br>- le nom d'un champ, d'un groupe de champs ou d'une fenêtre.<br><br>- le nom d'un champ, d'un groupe de champs ou d'une page.<br><br><br> |
| [Souligné](../Proprietes/2513013.md) | La propriété **Souligné** permet de :<br><br>- Savoir si une variable de type Police est soulignée.<br><br>- Souligner (ou non) une variable de type Police.<br><br><br> |
| [Taille](../Proprietes/2510125.md) | La propriété **Taille** permet de connaître et modifier la taille d'un élément. |
| [UnitéTaille](../Proprietes/1410088365.md) | La propriété **UnitéTaille** permet de connaître et de modifier l'unité utilisée pour manipuler la taille d'une police. |

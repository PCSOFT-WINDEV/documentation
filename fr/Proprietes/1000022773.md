
## MiseEnPage (Propriété)

***En anglais : PageLayout***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **MiseEnPage** permet de définir les caractéristiques de mise en page d'un document. Cette propriété est utilisable sur : 

- le champ Editeur de diagrammes. Pour plus de détails, consultez [MiseEnPage](../Proprietes/1000023138.md). 

- une variable de type [docFragment](../WDLang1/1000022482.md). 

- une variable de type [docParagraphe](../WDLang1/1000022483.md). 

- une variable de type [docNiveauNumérotation](../WDLang1/1000022799.md). 

- une variable de type [diagMiseEnPage](../WDLang1/1410088610.md).




## Groupe (Propriété)

***En anglais : Group***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Groupe** permet de connaître et de modifier les groupes d'éléments. Cette propriété est utilisable sur :

- les groupes de champs. Pour plus de détails, consultez [Groupe sur les groupes de champs](../Proprietes/2510049.md).

- une variable de type [gglContact](../WDLang5/1000017434.md). 

- une variable de type [Notification](../WDLang3/1000019441.md).

- une variable de type [hdfsInformation](../WDLang4/1000021973.md).

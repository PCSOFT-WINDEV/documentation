


## CouleurTexteIndication (Propriété)

***En anglais : HintTextColor***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **CouleurTexteIndication** permet de connaître et de modifier la couleur du texte d'indication d'un champ de saisie. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SAI_Nom.Indication = "Saisir un nom valide"
SAI_Nom.CouleurTexteIndication = RougePastel
DonneFocusEtRetourUtilisateur(SAI_Nom)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître la couleur du texte d'indication

`<Résultat> = <Champ de saisie>.CouleurTexteIndication`
---

**`<Résultat> : (Entier ou constante de type Entier)`**

Couleur associée au texte d'indication. Cette couleur peut correspondre à :

- une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)),

- [une couleur prédéfinie](../WDLang5/3010002.md).




**`<Champ de saisie> : (Nom de champ)`**

Nom du champ de saisie à manipuler. 


<a name="SYNTAXE2"></a>

### Modifier la couleur du texte d'indication

`<Champ de saisie>.CouleurTexteIndication = <Nouvelle couleur>`
---

**`<Champ de saisie> : (Nom de champ)`**

Nom du champ de saisie à manipuler. 

**`<Nouvelle couleur> : (Entier ou constante de type Entier)`**

Couleur associée au texte d'indication. Cette couleur peut correspondre à :

- une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)),

- [une couleur prédéfinie](../WDLang5/3010002.md).

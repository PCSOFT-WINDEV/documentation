


## HTMLAvant (Propriété)

***En anglais : HTMLBefore***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **HTMLAvant** permet de connaître et de modifier le code HTML ajouté avant le champ.

**Remarque** : Le code HTML généré avant le champ peut être saisi dans l'onglet "Avancé" de la fenêtre de description du champ.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Modifier dynamiquement le style d'un bouton
BTN_BoutonDyn.HTMLAvant = "<DIV style=""..."">"
BTN_BoutonDyn.HTMLAprès = "</DIV>"
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le contenu HTML inséré avant le champ

`<Contenu HTML> = <Champ manipulé>.HTMLAvant`
---

**`<Contenu HTML> : (Chaîne de caractères)`**

Contenu HTML inséré avant le champ.

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler.  


<a name="SYNTAXE2"></a>

### Modifier le contenu HTML inséré avant le champ

`<Champ manipulé>.HTMLAvant = <Contenu HTML>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler.

**`<Contenu HTML> : (Chaîne de caractères)`**

Contenu HTML inséré avant le champ.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Cette option est disponible sur tous les champs possédant un code HTML pouvant être inséré avant. Cette propriété n'est pas disponible sur les champs suivants :

- Champ HTML.

- Champ Table.

- Champ Zone répétée.

- Champ Table hiérarchique.




## NomZoneMem (Propriété)

***En anglais : MemZoneName***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **NomZoneMem** permet de :

- connaître le nom de la zone mémoire utilisée comme source de données d'un état.

- modifier le nom de la zone mémoire utilisée comme source de données d'un état.





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connaître le nom de la source de données de l'état "ETAT_EtatClient"
ResNomVue = ETAT_EtatClient.NomZoneMem
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le nom de la zone mémoire associée à un état

`<Résultat> = <Etat manipulé>.NomZoneMem`
---

**`<Résultat> : (Chaîne de caractères)`**

Nom de la zone mémoire utilisée comme source de données de l'état spécifié.

**`<Etat manipulé> : (Nom d'état)`**

Nom de l'état associé à une zone mémoire à manipuler. Si ce paramètre ne correspond pas à un état associé à une zone mémoire, une erreur du WLangage est générée.  


<a name="SYNTAXE2"></a>

### Modifier le nom de la zone mémoire associée à un état

`<Etat manipulé>.NomZoneMem = <Nom de la zone mémoire>`
---

**`<Etat manipulé> : (Nom d'état)`**

Nom de l'état à manipuler. Si ce paramètre ne correspond pas à un état associé à une zone mémoire, une erreur du WLangage est générée.

**`<Nom de la zone mémoire> : (Chaîne de caractères)`**

Nom de la zone mémoire à utiliser comme source de données de l'état spécifié.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
La propriété **NomZoneMem** peut être utilisée uniquement sur un état associé à une zone mémoire.

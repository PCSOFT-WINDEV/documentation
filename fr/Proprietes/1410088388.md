
## mdpsNombreItération (Propriété)

***En anglais : spIterationCount***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **mdpsNombreItération** permet de :

- Connaître le nombre d'itérations de l'algorithme de hachage utilisé pour une rubrique de type "Mot de passe". Cette propriété peut être utilisée pour une rubrique de type "Mot de passe" définie sous l'éditeur d'analyses ou définie par programmation. 

- Définir le nombre d'itérations de l'algorithme de hachage utilisé pour une rubrique de type "Mot de passe" (lors de la description par programmation de la rubrique "Mot de passe" uniquement). 



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Description du fichier UnFichier
...

// Description de la rubrique "Mot_Passe"
MaRubrique est une Description de Rubrique
MaRubrique.Nom = "Mot_Passe"
MaRubrique.Type = hRubMotDePasseSécurisé
MaRubrique.mdpsDérivation = MDPS_PBKDF2
MaRubrique.mdpsHash = MDPS_SHA2_256
MaRubrique.mdpsNombreItération = 20000
MaRubrique.mdpsTaille = 256

// Validation de la description de la rubrique "Mot_Passe"
HDécritRubrique(UnFichier, MaRubrique)

// Validation de la description du fichier UnFichier
...
```





<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le nombre d'itérations de l'algorithme de hachage d'une rubrique de type "Mot de passe"

`<Itérations> = <Fichier de données>.<Rubrique>.mdpsNombreItération`
---

**`<Itérations> : (Entier)`**

Nombre d'itérations de l'algorithme de hachage utilisé.

**`<Fichier de données> : (Chaîne de caractères)`**

Nom du fichier manipulé. Ce nom a été défini sous l'éditeur d'analyses ou grâce au type [Description de Fichier](../WDLang4/1514065.md).

**`<Rubrique> : (Chaîne de caractères)`**

Nom de la rubrique manipulée. Ce nom a été défini sous l'éditeur d'analyses ou grâce au type [Description de Rubrique](../WDLang4/1514071.md).  


<a name="SYNTAXE2"></a>

### Définir le nombre d'itérations de l'algorithme de hachage d'une rubrique de type "Mot de passe" (rubrique décrite par programmation)

`<Rubrique>.mdpsNombreItération = <Itérations>`
---

**`<Rubrique> : (Chaîne de caractères)`**

Nom de la rubrique manipulée. Ce nom vient d'être défini grâce au type [Description de Rubrique](../WDLang4/1514071.md).

**`<Itérations> : (Entier)`**

Nombre d'itérations de l'algorithme de hachage utilisé. Il est conseillé d’utiliser une valeur comprise entre 10000 et 30000.

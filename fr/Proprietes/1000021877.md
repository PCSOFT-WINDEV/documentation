


## ChampDépart (Propriété)

***En anglais : StartControl***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **ChampDépart** permet de connaître ou de modifier le champ de départ d'un champ Fil.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Création et initialisation d'un champ Fil 
// Ce fil relie le champ BTN_ChampA et le champ BTN_ChampB
NomFil est un Champ
NomFil <- ChampCrée("", typFil)
NomFil.ChampDépart = BTN_ChampA
NomFil.ChampArrivée = BTN_ChampB
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître le champ de départ d'un champ Fil

`<Résultat> = <Champ manipulé>.ChampDépart`
---

**`<Résultat> : (Chaîne de caractères)`**

Nom du champ de départ. 

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ Fil. 


<a name="SYNTAXE2"></a>

### Modifier le champ départ d'un champ Fil

`<Champ manipulé>.ChampDépart = <Nouveau champ de départ>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ Fil. 

**`<Nouveau champ de départ> : (Chaîne de caractères)`**

Nom du nouveau champ de départ. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Le fil est affiché lorsque les propriétés **ChampDépart** et [ChampArrivée](../Proprietes/1000021878.md) sont renseignées et visibles.

- Les points d'attache utilisés peuvent être modifiés à l'aide des propriétés [PointDAttacheDépart](../Proprietes/1000021907.md) et [PointDAttacheArrivée](../Proprietes/1000021908.md).

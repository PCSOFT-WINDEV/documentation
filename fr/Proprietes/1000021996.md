


## ValeurParEtoile (Propriété)

***En anglais : ValuePerStar***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **ValeurParEtoile** permet de connaître et de modifier la valeur d'une étoile dans un champ Notation. 

Remarque : Cette propriété correspond à l'option "Valeur d'une 'étoile'" disponible dans l'onglet "Général" de la fenêtre de description du champ Notation. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Pour utiliser des valeurs de 0 à 100 dans NOTE_Notation1

// Le champ affiche de 0 à 5 étoiles
NOTE_Notation1.BorneMax = 5 
// Chaque étoile a pour valeur 20
NOTE_Notation1.ValeurParEtoile = 20 
// Les valeurs affichées dans le champ vont de 0 à 100 (5 * 20)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître la valeur d'une étoile dans un champ Notation

`<Résultat> = <Champ Notation>.ValeurParEtoile`
---

**`<Résultat> : (Réel)`**

Valeur associée à une étoile. 

**`<Champ Notation> : (Nom de champ)`**

Nom du champ Notation à manipuler. 


<a name="SYNTAXE2"></a>

### Modifier la valeur d'une étoile dans un champ Notation

`<Champ Notation>.ValeurParEtoile = <Nouvelle valeur>`
---

**`<Champ Notation> : (Nom de champ)`**

Nom du champ Notation à manipuler.

**`<Nouvelle valeur> : (Réel)`**

Nouvelle valeur associée à une étoile (ou autre pictogramme graphique utilisé dans le champ Notation). Cette valeur doit être supérieure à 0. Dans le cas contraire, cette valeur est ignorée. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Par défaut, la valeur d'une étoile (ou autre pictogramme) du champ Notation est 1. 

- La propriété [Valeur](../Proprietes/2510130.md) du champ Notation correspond au nombre d'étoiles multiplié par la valeur de la propriété **ValeurParEtoile**. 

- Le nombre maximum d'étoiles d'un champ Notation est donné par la propriété [BorneMax](../Proprietes/2510009.md).

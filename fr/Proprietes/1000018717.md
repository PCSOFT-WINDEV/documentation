
## ConfirmationLecture (Propriété)

***En anglais : DispositionNotification***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **ConfirmationLecture** permet de connaître et de modifier la demande de confirmation de lecture d'un email. 

Cette propriété est utilisable sur :

- la structure [Email](../WDLang3/3032029.md).

- une variable de type [Email](../WDLang3/1000018713.md).

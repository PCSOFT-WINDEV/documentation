


## TauxAncrageLargeur (Propriété)

***En anglais : AnchorRateWidth***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **TauxAncrageLargeur** permet de :

- Connaître le taux d'ancrage en agrandissement d'un champ (ou d'une colonne de table) sur l'axe horizontal.

- Modifier le taux d'ancrage en agrandissement d'un champ (ou d'une colonne de table) sur l'axe horizontal.




Le taux d'ancrage permet de définir la manière dont le champ se déplace ou change de taille en fonction de l'évolution de la taille de la fenêtre (ou de l'état) à laquelle il est ancré. Le taux d'ancrage correspond au pourcentage de déplacement du champ par rapport à la fenêtre (ou à l'état). Ainsi, si le taux d'ancrage est de 0, le champ ne sera pas déplacé, si le taux d'ancrage est de 1000, l'ancrage du champ sera normal.

Le taux d'ancrage en largeur (propriété **TauxAncrageLargeur**) est appliqué si le champ est ancré en largeur (constante *ancreLargeur* utilisée pour la propriété [Ancrage](../Proprietes/2510001.md)).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Le champ Liste s'élargit de 50% de l'agrandissement de la fenêtre (500 pour 1000).
LISTE_ListePersonne.TauxAncrageLargeur = 500
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le taux d'ancrage en agrandissement d'un champ ancré en largeur

`<Résultat> = <Champ manipulé>.TauxAncrageLargeur`
---

**`<Résultat> : (Entier)`**

Taux d'ancrage en agrandissement compris entre 0 (champ non étiré) et 1000 (ancrage normal)

**`<Champ manipulé> : (Nom du champ)`**

Nom du champ (ou de la colonne) à manipuler. Ce champ peut être un champ : 

- d'une fenêtre.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) d'un état.





<a name="SYNTAXE2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) 
### Modifier le taux d'ancrage en agrandissement d'un champ ancré en largeur

`<Champ manipulé>.TauxAncrageLargeur = <Taux d'ancrage>`
---

**`<Champ manipulé> : (Nom du champ)`**

Nom du champ (ou de la colonne) à manipuler.

**`<Taux d'ancrage> : (Entier)`**

Nouveau taux d'ancrage en agrandissement compris entre 0 (champ non étiré) et 1000 (ancrage normal)

## Remarques


### Ancrage d'un champ
<a name="ancrage_champ_ELTPARAGRAPHE000110"></a>

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Dans une fenêtre, la propriété [Ancrage](../Proprietes/2510001.md) permet de définir le comportement du champ (dimensions et déplacement) lors du changement de taille de la fenêtre. Si le champ est associé à un champ Onglet, l'ancrage est défini par rapport à l'onglet.
	Pour plus de détails, consultez [Ancrage des champs dans une fenêtre](../WDChamp/2026002.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Dans le cas d'une colonne d'un champ Table ou Table hiérarchique, la propriété [Ancrage](../Proprietes/2510001.md) n'est pas disponible. L'ancrage d'une colonne se spécifie directement avec la propriété **TauxAncrageLargeur**.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Dans un état, la propriété [Ancrage](../Proprietes/2510001.md) permet de connaître le comportement du champ (dimensions et déplacement) lors du changement de taille de l'état. Ce comportement a été défini sous l'éditeur d'états. 
	Pour plus de détails, consultez [Ancrage des champs dans un état](../WDChamp/1013298.md). 



<a name="NOTE0_2"></a>


### Limites
<a name="limites_ELTPARAGRAPHE000149"></a>![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) La propriété **TauxAncrageLargeur** s'applique à tous les champs d'une fenêtre.

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) La propriété **TauxAncrageLargeur** s'applique à tous les champs d'un état. Dans ce cas, il est uniquement possible de connaître le taux d'ancrage utilisé.

![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) La propriété **TauxAncrageLargeur** s'applique uniquement aux éléments suivants :

- Bouton.

- Libellé.

- Champ de saisie.

- Image.

- Interrupteur.

- Sélecteur.

- Liste.

- Combo.

- Arbre.

- Onglet.

- Table.

- Superchamp.

- Regroupement de ruban.

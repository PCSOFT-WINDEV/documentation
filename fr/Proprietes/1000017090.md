


## NouvelEnregistrement (Propriété)

***En anglais : NewRecord***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **NouvelEnregistrement** permet de savoir si l'enregistrement est un nouvel enregistrement ou provient du fichier de données.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Savoir si l'enregistrement vient d'être saisi
SI Client.NouvelEnregistrement = Vrai ALORS
	Info("Cet enregistrement vient d'être saisi")
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Savoir si l'enregistrement est nouveau

`<Résultat> = <Fichier de données>.NouvelEnregistrement`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'enregistrement est nouveau (la fonction [HRAZ](../WDLang4/3044118.md) a été utilisée), 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Fichier de données> : (Chaîne de caractères)`**

Nom du fichier de données manipulé.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Par défaut, la propriété **NouvelEnregistrement** est à <u><u><u><u>Vrai</u></u></u></u>. Cette propriété est également à <u><u><u><u>Vrai</u></u></u></u> si la fonction [HRAZ](../WDLang4/3044118.md) a été utilisée.

Elle est remise à <u><u><u><u>Faux</u></u></u></u> lors de l'appel de l'une des fonctions suivantes :

- [HAjoute](../WDLang4/3044147.md).

- [HEcrit](../WDLang4/3044092.md).

- [HLit](../WDLang4/3044047.md).

- [HLitDernier](../WDLang4/3044140.md).

- [HLitPrécédent](../WDLang4/3044048.md).

- [HLitPremier](../WDLang4/3044051.md).

- [HLitRecherche](../WDLang4/3044050.md).

- [HLitRecherchePremier](../WDLang4/3044036.md).

- [HLitSuivant](../WDLang4/3044037.md).

- [HModifie](../WDLang4/3044042.md).

- [HRaye](../WDLang4/3044121.md).

- [HRetourPosition](../WDLang4/3044137.md).

- [HSupprime](../WDLang4/3044018.md).

- [POUR TOUT Fichier](../Motscles/1510013.md).




Remarque : L'affectation d'une variable de type [Enregistrement](../WDLang4/1000020962.md) à un fichier modifie également la valeur de la propriété **NouvelEnregistrement**.

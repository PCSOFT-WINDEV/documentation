
## Message

***En anglais : Message***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le mot-clé **Message** peut correspondre :

- à la fonction **[Message](../WDLang1/3021006.md)** qui permet d'afficher un message dans la barre de message. 

- à la propriété **[Message](../Proprietes/2510078.md)** qui permet de connaître et modifier le message d'aide associé à un champ ou une option de menu d'une fenêtre. 

- à la propriété **Message** disponible sur :

	- une variable de type [SFErreur](../WDLang5/1000018455.md). 

	- la structure [Email](../WDLang3/3032029.md)

	- une variable de type [Email](../WDLang3/1000018713.md).

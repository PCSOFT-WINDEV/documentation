
## Surentête (Propriété)

***En anglais : TitleHeader***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Surentête** permet de : 

- Connaître le titre et l'aspect d'un sur-entête de colonne de table (champ Table ou Table hiérarchique). 

- Connaître le nombre de sur-entêtes d'une colonne de table (champ Table ou Table hiérarchique).

- Modifier le titre et l'aspect d'un sur-entête de colonne de table (champ Table ou Table hiérarchique).




Attention : Cette propriété ne peut pas être affectée ou copiée directement. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Modifie le titre d'un sur-entête sur la ligne 1
COL_DATE.Surentête[2].Libellé = "Nouveau titre"

// Modifie le titre d'un sur-entête sur la ligne 2
COL_DATE.Surentête[1].Libellé = "sous-titre"
// Modifie la police du titre d'un sur entête sur la ligne 2
COL_DATE.Surentête[1].Police.Gras = Vrai

// Modifie la couleur du sous-titre sur la ligne 1
COL_DATE.Surentête[1].Couleur = RougeClair
COL_DATE.Surentête[1].PoliceGras = Vrai
POUR I = 1 A COL_DATE.Surentête.Occurrence 
	COL_DATE.Surentête[I].Couleur = RougeClair
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Récupérer la valeur d'une propriété d'un sur-tête de colonne

`<Résultat> = <Colonne>.Surentête[<Numéro de ligne>].<Propriété>`
---

**`<Résultat> : (Type attendu)`**

Valeur de la propriété.

**`<Colonne> : (Nom de champ)`**

Nom de la colonne du champ Table manipulée. 

**`<Numéro de ligne> : (Entier)`**

Numéro de la ligne du sur-entête à manipuler. La ligne 1 correspond à la première ligne de titre de la colonne en partant du bas. ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Surentete_ligne.gif)


**`<Propriété> : (Propriété WLangage)`**

Propriété WLangage à utiliser pour manipuler le sur-entête. Les propriétés disponibles sont : 


|   |   |
| --- | --- |
| [Couleur](../Proprietes/2510071.md) | Permet de connaître la couleur du texte du sur-entête. |
| [CouleurFond](../Proprietes/2510022.md) | Permet de connaître la couleur de fond du sur-entête. |
| [Libellé](../Proprietes/2510053.md) | Permet de connaître le texte du sur-entête. |
| [Police](../Proprietes/2510095.md) | Permet de connaître la police du sur-entête. <br><br>Remarque : Il est possible d'utiliser directement les propriétés du type Police. |
| [PoliceBarrée](../Proprietes/2510091.md) | Permet de savoir si le texte du sur-entête est barré. |
| [PoliceCondensée](../Proprietes/2510096.md) | Permet de savoir si le texte du sur-entête est condensé. |
| [PoliceEtendue](../Proprietes/2510104.md) | Permet de savoir si le texte du sur-entête est étendu.  |
| [PoliceGras](../Proprietes/2510103.md) | Permet de savoir si le texte du sur-entête est en gras.  |
| [PoliceItalique](../Proprietes/2510102.md) | Permet de savoir si le texte du sur-entête est en italique.  |
| [PoliceLarge](../Proprietes/2510101.md) | Permet de savoir si le texte du sur-entête est large. |
| [PoliceNom](../Proprietes/2510100.md) | Permet de connaître le nom de la police utilisée pour le texte du sur-entête. |
| [PoliceSoulignée](../Proprietes/2510099.md) | Permet de savoir si le texte du sur-entête est souligné. |
| [PoliceTaille](../Proprietes/2510072.md) | Permet de connaître la taille du texte du sur-entête. |




<a name="SYNTAXE2"></a>

### Connaître le nombre de sur-entêtes

`<Résultat> = <Colonne>.Surentête.Occurrence`
---

**`<Résultat> : (Entier)`**

Nombre de sur-entêtes. 

**`<Colonne> : (Nom de champ)`**

Nom de la colonne du champ Table manipulée. 


<a name="SYNTAXE3"></a>

### Modifier une propriété d'un sur-tête de colonne

`<Colonne>.Surentête[<Numéro de ligne>].<Propriété> = <Valeur>`
---

**`<Colonne> : (Nom de champ)`**

Nom de la colonne du champ Table manipulée. 

**`<Numéro de ligne> : (Entier)`**

Numéro de la ligne du sur-entête à manipuler. La ligne 1 correspond à la première ligne de titre de la colonne en partant du bas. ![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Surentete_ligne.gif)


**`<Propriété> : (Propriété WLangage)`**

Propriété WLangage à utiliser pour manipuler le surentête. Les propriétés disponibles sont : 


|   |   |
| --- | --- |
| [Couleur](../Proprietes/2510071.md) | Permet de modifier la couleur du texte du sur-entête. |
| [CouleurFond](../Proprietes/2510022.md) | Permet de modifier la couleur de fond du sur-entête. |
| [Libellé](../Proprietes/2510053.md) | Permet de modifier le texte du sur-entête. <br><br>Remarque : La modification du libellé de la ligne 1 correspond à l'utilisation de la propriété [Titre](../Proprietes/2510090.md) sur la colonne. |
| [Police](../Proprietes/2510095.md) | Permet de modifier la police du sur-entête. <br><br>Remarque : Il est possible d'utiliser directement les propriétés du type [Police](../Motscles/1514045.md). |
| [PoliceBarrée](../Proprietes/2510091.md) | Permet de barrer le texte du sur-entête. |
| [PoliceCondensée](../Proprietes/2510096.md) | Permet de condenser le texte du sur-entête. |
| [PoliceEtendue](../Proprietes/2510104.md) | Permet d'étendre le texte du sur-entête. |
| [PoliceGras](../Proprietes/2510103.md) | Permet de mettre en gras le texte du sur-entête. |
| [PoliceItalique](../Proprietes/2510102.md) | Permet de mettre en italique le texte du sur-entête. |

| [PoliceNom](../Proprietes/2510100.md) | Permet de modifier le nom de la police utilisée pour le texte du sur-entête. |
| [PoliceSoulignée](../Proprietes/2510099.md) | Permet de souligner le texte du sur-entête. |
| [PoliceTaille](../Proprietes/2510072.md) | Permet de modifier la taille du texte du sur-entête. |



**`<Valeur> : (Type attendu)`**

Valeur de la propriété.

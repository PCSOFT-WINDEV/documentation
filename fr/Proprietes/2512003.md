
## CléSource (Propriété)

***En anglais : SourceKey***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **CléSource** permet de paramétrer les caractéristiques d'une liaison. Il est ainsi possible de :

- Connaître la rubrique du fichier de données source utilisée dans la liaison.

- Modifier la rubrique du fichier de données source utilisée dans la liaison.



<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=cleSource.gif)





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Modifier la clé source
LiaisonClubPersonne.CléSource = Club.IDClub
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître la clé source de la liaison

`<Clé> = <Liaison>.CléSource`
---

**`<Clé> : (Chaîne de caractères)`**

Nom de la clé utilisée par la liaison dans le fichier de données source.

**`<Liaison> : (Chaîne de caractères)`**

Nom de la liaison définie sous l'éditeur d'analyses ou définie grâce au type [Description de Liaison](../WDLang4/1514039.md).  


<a name="SYNTAXE2"></a>

### Modifier la clé source de la liaison

`<Liaison>.CléSource = <Nouvelle clé>`
---

**`<Liaison> : (Chaîne de caractères)`**

Nom de la liaison définie sous l'éditeur d'analyses ou définie grâce au type [Description de Liaison](../WDLang4/1514039.md).

**`<Nouvelle clé> : (Chaîne de caractères)`**

Nom de la nouvelle clé utilisée par la liaison dans le fichier de données source. Cette clé a été définie sous l'éditeur d'analyses ou grâce au type [Description de Rubrique](../WDLang4/1514071.md).




## RetraitGauche (Propriété)

***En anglais : LeftIndent***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **RetraitGauche** permet de connaître et de modifier l'espace laissé à gauche du texte dans un champ. 

Cette propriété s'applique sur les champs suivants : 

- les colonnes des champs Table et Table hiérarchique (entre le début de la colonne et le début du texte).

- les champs de saisie.

- les champs Liste et Combo.

- les champs Bouton.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) les champs Ruban.

- les champs Onglet.

- les champs Libellé.




Cet espace est exprimé en pixels.

![Champ Table sans retrait à gauche](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=RetraitGauche.gif)


![Champ Table avec un retrait à gauche de 20 pixels](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=RetraitGauche2.gif)



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Applique un retrait de 20 pixels pour toutes les colonnes du champ Table
TABLE_Table1.RetraitGauche = 20
// Applique un retrait supplémentaire à la colonne Jour de la ligne en cours
COL_Jour.RetraitGauche = 50
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le retrait gauche d'un champ

`<Résultat> = <Champ manipulé>.RetraitGauche`
---

**`<Résultat> : (Entier)`**

Nombre de pixels utilisé pour le retrait du texte. Par défaut, le retrait utilisé est de 0 pixels (pas de retrait). 

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler : 

- colonne d'un champ Table.

- colonne d'un champ Table hiérarchique. 

- champ de saisie.

- champ Liste ou Combo.

- champ Bouton.

- champ Ruban.

- champ Onglet.

- champ Libellé.





<a name="SYNTAXE2"></a>

### Modifier le retrait gauche d'un champ

`<Champ manipulé>.RetraitGauche = <Nouveau retrait>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler : 

- colonne d'un champ Table.

- colonne d'un champ Table hiérarchique. 

- champ de saisie.

- champ Liste ou Combo.

- champ Bouton.

- champ Ruban.

- champ Onglet.

- champ Libellé.




**`<Nouveau retrait> : (Entier)`**

Nombre de pixels à utiliser pour le retrait du texte sur le champ spécifié. 
Remarque : Retrait sur tout le champ Table : Si une des colonnes possède un retrait spécifique, le nouveau retrait ne sera pas appliqué à cette colonne.

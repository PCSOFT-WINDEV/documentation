


## Extension (Propriété)

***En anglais : Extension***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Extension** permet de :

- connaître l'extension d'un fichier de données défini sous l'éditeur d'analyses ou défini par programmation.

- définir l'extension d'un fichier de données (lors de la description du fichier par programmation uniquement).







<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connaître le nom et le chemin complet du fichier de données Client
NomCheminCompletFichier est une chaîne
NomCheminCompletFichier = Client.Répertoire + "\" + ...
	Client.NomPhysique + Client.Extension
// NomCheminCompletFichier correspond à "C:\MesFichiers\Client.Fic"
```
<a name="Exemple2"></a>

```wl
// Définir l'extension du fichier UnFichier
UnFichier est une Description de Fichier
UnFichier.Nom = "CLIENT"
UnFichier.Extension = ".AAA"
// Le fichier créé sera "CLIENT.AAA"
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître l'extension d'un fichier de données

`<Extension du fichier> = <Fichier de données>.Extension`
---

**`<Extension du fichier> : (Chaîne de caractères)`**



- Extension du fichier de données manipulé (précédée d'un point). 

- Chaîne vide ("") si le fichier de données manipulé n'a pas d'extension.




**`<Fichier de données> : (Chaîne de caractères)`**

Nom du fichier de données manipulé. Ce nom a été défini sous l'éditeur d'analyses ou grâce au type [Description de Fichier](../WDLang4/1514065.md).  


<a name="SYNTAXE2"></a>

### Définir l'extension d'un fichier de données

`<Fichier de données>.Extension = <Nouvelle extension>`
---

**`<Fichier de données> : (Chaîne de caractères)`**

Nom du fichier de données manipulé. Ce nom vient d'être défini grâce au type [Description de Fichier](../WDLang4/1514065.md).

**`<Nouvelle extension> : (Chaîne de caractères)`**

Extension du fichier de données manipulé (précédée d'un point).

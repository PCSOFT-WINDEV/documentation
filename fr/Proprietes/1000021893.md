
## AutoriseMasquageActionBar (Propriété)

***En anglais : AllowActionBarHiding***
	



<a name="XPresentation"></a>
<a name="Presentation"></a>
**Fonctionnement du masquage automatique de l'Action Bar** : 

- Par défaut, l'Action Bar est toujours visible. 

- L'option "Masquer automatiquement l'Action Bar lors du Scroll" (onglet "Détail" de la fenêtre de description de l'Action Bar) permet de masquer l'Action Bar lors du scroll (dans une fenêtre, fenêtre interne, superchamp, table ou zone répétée). 
	Dans certains cas, lorsque cette option est active, il peut être nécessaire de ne pas masquer l'Action Bar. Ce paramétrage peut être réalisé avec la propriété **AutoriseMasquageActionBar**. 



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **AutoriseMasquageActionBar** permet de gérer le masquage automatique de l'Action Bar. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Forcer l'Action Bar à rester visible
ACTB_MonActionBar.Visible = Vrai
FI_Client.AutoriseMasquageActionBar = Faux
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Savoir si le scroll au doigt masque automatiquement l'Action Bar

`<Résultat> = <Elément à manipuler>.AutoriseMasquageActionBar`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'Action Bar est automatiquement masquée lors du scroll dans l'élément, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 




**`<Elément à manipuler> : (Nom de champ ou nom de fenêtre)`**

Nom de l'élément à "scroller" : fenêtre, fenêtre interne, superchamp, table, zone répétée. 


<a name="SYNTAXE2"></a>

### Modifier le fonctionnement du masquage automatique de l'Action Bar

`<Elément à manipuler>.AutoriseMasquageActionBar = <Masquage automatique>`
---

**`<Elément à manipuler> : (Nom de champ ou nom de fenêtre)`**

Nom de l'élément à "scroller" : fenêtre, fenêtre interne, superchamp, table, zone répétée. 

**`<Masquage automatique> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'Action Bar est automatiquement masquée lors du scroll dans l'élément, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Si l'option "Masquer automatiquement l'Action Bar lors du Scroll" (onglet "Détail" de la fenêtre de description du champ Action Bar) n'est pas activée, cette propriété n'a aucun effet. 

- Le champ Action Bar peut être rendu visible ou non via la propriété [Visible](../Proprietes/2510138.md). 

- Si le champ ou la fenêtre ne supporte pas de scroll vers le haut ou le bas, la propriété **AutoriseMasquageActionBar** est sans effet.

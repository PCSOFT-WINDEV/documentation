


## Emission (Propriété)

***En anglais : OutgoingData***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Emission** permet de connaître et modifier le type des données émises par le flux.

Par défaut, un flux émet du son et de la vidéo.

**Attention** : La transmission en cours n'est pas modifiée. Les changements effectués par cette propriété seront pris en compte uniquement lors de la prochaine connexion.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Gère uniquement le son
MonFlux.Emission = FluxSon
MonFlux.Adresse = "123.123.123"
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le type de données émises par le flux

`<Résultat> = <Nom du flux>.Emission`
---

**`<Résultat> : (Constante ou combinaison de constantes de type entier)`**

Type de données actuellement émises :


|   |   |
| --- | --- |
| *FluxSon* | Les données de type son sont émises. |
| *FluxVidéo* | Les données de type Vidéo sont émises. |



**`<Nom du flux> : (Chaîne de caractères)`**

Nom du Flux ou nom du champ Conférence associé au flux.  


<a name="SYNTAXE2"></a>

### Modifier le type de données émises par le flux

`<Nom du flux>.Emission = <Nouveau type de données>`
---

**`<Nom du flux> : (Chaîne de caractères)`**

Nom du Flux ou nom du champ Conférence associé au flux.

**`<Nouveau type de données> : (Constante ou combinaison de constantes de type Entier)`**

Nouveau type de données à émettre. Cette modification sera prise en compte uniquement lors de la prochaine connexion.


|   |   |
| --- | --- |
| *FluxSon* | Les données de type Son sont émises. |
| *FluxVidéo* | Les données de type Vidéo sont émises. |

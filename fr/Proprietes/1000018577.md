
## Erreur

***En anglais : Error***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le mot-clé **Erreur** permet de gérer les erreurs. Ce mot-clé peut correspondre :

- au mot-clé "CAS ERREUR" utilisé pour la [gestion automatique des erreurs](../WDLang1/3034010.md). 

- à la fonction **[Erreur](../WDLang1/3021013.md)**.

- à la propriété **Erreur** utilisée sur une variable de type [sfRésultatConvertitPiste](../WDLang5/1000018739.md). 

- à la propriété **Erreur** utilisée sur une variable de type [sfRésultatDéconnecte](../WDLang5/1000018560.md). 

- à la propriété **Erreur** utilisée sur une variable de type [sfRésultatFusionne](../WDLang5/1000018499.md).

- à la propriété **Erreur** utilisée sur une variable de type [sfRésultatRestaure](../WDLang5/1000018526.md).

- à la propriété **Erreur** utilisée sur une variable de type [sfRésultatSauvegarde](../WDLang5/1000018460.md).

- à la propriété **Erreur** utilisée sur une variable de type [sfRésultatSupprime](../WDLang5/1000018546.md).

- à la propriété **Erreur** utilisée sur une variable de type [sfRésultatVideCorbeille](../WDLang5/1000018467.md).

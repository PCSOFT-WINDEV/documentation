***En anglais : Count***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le mot-clé **Occurrence** peut correspondre à : 

- la fonction **[Occurrence](../WDLang1/3025009.md)**. Cette fonction renvoie le nombre d'occurrences d'un champ dans une fenêtre, une page ou un groupe de champs.

- la propriété **Occurrence** qui permet de connaître un nombre d'éléments. Cette propriété est utilisable sur :

	- les champs d'une fenêtre ou d'une page. Pour plus de détails, consultez [Occurrence](../Proprietes/2510097.md).

	- une variable de type [xmlNoeud](../WDLang5/1000018786.md).

	- une variable de type [JSON](../Motscles/1000023611.md). 

	- une variable de type [YAML](../Motscles/1000024438.md).




## Note (Propriété)

***En anglais : Note***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Note** permet de :

- Connaître les notes associées à un champ, à une fenêtre, à une page ou un état.

- Modifier les notes associées à un champ, à une fenêtre, à une page ou un état.




Les notes sont destinées aux développeurs. Elles apparaissent lors de l'impression de la documentation technique. Il est également possible d'indiquer un titre pour les notes avec la propriété [TitreNote](../Proprietes/2510121.md).

Bien souvent, cette propriété permet au développeur de stocker une information sans avoir d'influence sur l'exécution de l'application.

**Remarque** : Les notes d'un champ, d'une fenêtre, d'une page, d'un état ou d'un bloc sont définies dans l'onglet "Note" de la description de l'élément.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Modifier les notes associées au champ "SAI_Contact"
SAI_Contact.Note = "Désigne la personne à contacter dans la société"
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître les notes associées à un élément (champ, fenêtre, page, état, bloc, ...)

`<Notes> = <Elément manipulé>.Note`
---

**`<Notes> : (Chaîne de caractères)`**

Notes associées à l'élément spécifié.

**`<Elément manipulé> : (Nom de champ, nom de fenêtre, nom de page ou nom d'état)`**

Nom de l'élément (champ, fenêtre, page ou état) à manipuler.   


<a name="SYNTAXE2"></a>

### Modifier les notes associées à un élément (champ, fenêtre, page, état, bloc, ...)

`<Elément manipulé>.Note = <Nouvelles notes>`
---

**`<Elément manipulé> : (Nom de champ, nom de fenêtre, nom de page ou nom d'état)`**

Nom de l'élément (champ, fenêtre, page ou état) à manipuler.  

**`<Nouvelles notes> : (Chaîne de caractères)`**

Nouvelles notes associées à l'élément spécifié.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Limite
<a name="limite_ELTPARAGRAPHE000109"></a>Dans un  état, la propriété **Note** s'applique à tous les champs, aux blocs et aux états.

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Dans une fenêtre, la propriété **Note** ne peut pas être utilisée sur :

- une option d'un champ Interrupteur ou Sélecteur,

- une ligne d'un champ Table ou une ligne d'un champ Liste ou Combo,

- un menu,

- un volet d'onglet,

- un groupe de champs,

- un élément d'un champ Arbre.


![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) La propriété **Note** s'applique uniquement aux éléments suivants :

- Bouton.

- Libellé.

- Champ de saisie.

- Image.

- Interrupteur.

- Sélecteur.

- Liste.

- Combo.

- Arbre.

- Onglet.

- Fenêtre.

- Table.

- Colonne de table.

- Superchamp.

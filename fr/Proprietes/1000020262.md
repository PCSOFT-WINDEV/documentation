


## MessagerieInstantanée (Propriété)

***En anglais : InstantMessaging***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le mot-clé **MessagerieInstantanée** peut correspondre :  

- à la propriété **MessagerieInstantanée** d'une variable de type [Contact](../WDLang3/1000020254.md). 

- à une variable de type [MessagerieInstantanée](../WDLang3/1000020247.md).

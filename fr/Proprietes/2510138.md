


## Visible (Propriété)

***En anglais : Visible***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Visible** permet de :

- Savoir si un élément est visible.

- Rendre un élément visible ou invisible.




Cette propriété s'applique aux :

- champs et groupes de champs d'une fenêtre, d'une page ou d'un état.

- blocs d'un état,

- options de menu.




**Remarques** :

- Sous l'éditeur de fenêtres et de pages, la visibilité d'un champ est définie dans l'onglet "UI" de la description du champ.

- Sous l'éditeur d'états, la visibilité d'un champ est définie dans l'onglet "Détail" de la description du champ.





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Rendre visible le champ "SAI_Société" s'il est invisible
SI SAI_Société.Visible = Faux ALORS
	SAI_Société.Visible = Vrai
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Savoir si un élément est visible

`<Résultat> = <Elément manipulé>.Visible`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'élément spécifié est visible, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Elément manipulé> : (Type de l'élément)`**

Nom de l'élément à manipuler.  


<a name="SYNTAXE2"></a>

### Rendre visible/invisible un champ ou un bloc d'un état

`<Elément manipulé>.Visible = <Visible/Invisible>`
---

**`<Elément manipulé> : (Type de l'élément)`**

Nom de l'élément à manipuler.

**`<Visible/Invisible> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'élément spécifié doit être visible, 

- <u><u><u><u>Faux</u></u></u></u> si l'élément spécifié doit être invisible.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Attribut d'un champ
<a name="attribut_champ_ELTPARAGRAPHE000114"></a>La propriété **Visible** ne modifie pas les autres attributs d'état du champ (sélection sans saisie, actif ou grisé).
<a name="NOTE0_2"></a>


### Groupe de champs
<a name="groupe_champs_ELTPARAGRAPHE000124"></a>Si l'objet spécifié est un groupe de champs, l'état de visibilité de l'ensemble des champs appartenant au groupe est modifié.
<a name="NOTE0_3"></a>

### Fonctionnement des lettres d'appel en fonction de l'état d'un champ
<a name="fonctionnement_des_lettres_appel_fonction_etat_champ_ELTPARAGRAPHE000133"></a>Il est impossible d'utiliser les lettres d'appel d'un champ invisible. Par exemple, un bouton invisible ne fonctionne pas via l'utilisation des lettres d'appel.
<a name="NOTE0_4"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) 

### Champs sur onglets
<a name="champs_sur_onglets_ELTPARAGRAPHE000142"></a>Si un champ présent sur un volet d'onglet est visible, mais si ce volet n'est pas affiché, la propriété **Visible** renverra <u><u><u><u>Faux</u></u></u></u>.
<a name="NOTE0_5"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) 

### Lignes d'un champ Table
<a name="lignes_champ_table_ELTPARAGRAPHE000158"></a>Il est possible rendre visible ou invisible une ligne d'un champ Table. Par exemple :

```wl
// Rend la seconde ligne du champ Table invisible
TABLE_Table[2].Visible = Faux
```
Si le champ Table utilise des bandeaux de couleurs alternées, l'alternance de couleurs sera respectée.

Remarque : Si la propriété **Visible** est utilisée sur une cellule de table, toutes les cellules de la colonne correspondante seront rendues invisibles.

### Limites
<a name="limites_ELTPARAGRAPHE000205"></a>![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) La propriété **Visible** ne s'applique pas aux états.![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) La propriété **Visible** ne peut pas être utilisée sur :

- une ligne d'un champ Liste ou Combo,

- un menu de fenêtre

- un élément d'un champ Arbre.




Si la propriété **Visible** est utilisée sur une cellule de table, toutes les cellules de la colonne correspondante seront rendues invisibles. ![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) La propriété **Visible** s'applique uniquement aux éléments suivants :

- Champ Bouton.

- Champ Libellé.

- Champ de saisie.

- Champ Image.

- Champ Interrupteur.

- Champ Sélecteur.

- Champ Liste.

- Champ Combo.

- Champ Arbre.

- Champ Onglet.

- Fenêtre.

- Champ Table.

- Colonne d'un champ Table.

- Superchamp.


 
 
![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) La propriété **Visible** ne s'applique pas aux fenêtres.

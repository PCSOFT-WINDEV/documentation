


## GUIDAnalyse (Propriété)

***En anglais : GUIDAnalysis***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **GUIDAnalyse** renvoie le GUID de l'analyse qui a permis de créer le fichier de données. Cet identifiant est utilisé dans le fichier .REP (pour plus de détails, consultez [Gestion du fichier .REP](../WDLang4/3044186.md)). 

**Remarque** : Le GUID de l'analyse peut également être connu dans l'éditeur d'analyses (onglet "Notes" de la description du fichier).
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=GUIDAnalyse%20-%20HC%20N%B0001.gif)







<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
Trace(Client.GUIDAnalyse)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<GUID Analyse> = <Fichier de données>.GUIDAnalyse`
---

**`<GUID Analyse> : (Chaîne de caractères)`**



- GUID de l'analyse qui a permis de créer le fichier de données. 

- Chaîne vide ("") si le fichier de données a été créé par programmation ou si le fichier utilisé n'est pas un fichier de données au format HFSQL Classic ou Client/Serveur.




**`<Fichier de données> : (Chaîne de caractères)`**

Nom du fichier de données pour lequel le GUID de l'analyse est recherché.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
La propriété [GUIDFichier](../Proprietes/2512099.md) permet de connaître le GUID du fichier. Ce GUID a été défini dans l'analyse qui a permis de créer le fichier de données.

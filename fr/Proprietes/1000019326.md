
## Hérité (Propriété)

***En anglais : Inherited***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Hérité** permet de savoir si un élément est hérité ou non. Cette propriété est utilisée par :

- une variable de type [Description de procédure](../WDLang1/1000019317.md).

- une variable de type [Description de Propriété](../WDLang1/1000019324.md).

- une variable de type [Description de Variable](../WDLang1/1000019560.md).

- une variable de type [Description d'Option](../WDLang1/1000023072.md). 

- une variable de type [Description de Valeur](../WDLang1/1000023061.md).

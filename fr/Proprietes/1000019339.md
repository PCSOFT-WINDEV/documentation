
## Définition (Propriété)

***En anglais : Definition***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Définition** permet de gérer la définition d'un type. Cette propriété est utilisée par :

- une variable de type [Description de Propriété](../WDLang1/1000019324.md).

- une variable de type [Description de Variable](../WDLang1/1000019560.md).

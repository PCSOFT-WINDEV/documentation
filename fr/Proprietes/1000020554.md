
## AvecZoom (Propriété)

***En anglais : WithZoom***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **AvecZoom** permet d'indiquer et de savoir si l'utilisateur peut zoomer : 

- dans un champ d'affichage HTML présent dans une application Android, iPhone ou iPad. 

- dans une carte affichée dans un champ Carte présent dans une application Android, iPhone ou iPad.



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Désactiver le zoom sur le champ HTM_NavigationPCS
HTM_NavigationPCS.AvecZoom = Faux
```


<a name="Exemple2"></a>

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Savoir si un champ HTML ou Carte permet le zoom ou non

`<Résultat> = <Champ manipulé>.AvecZoom`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le champ permet le zoom, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler : 

- Champ d'affichage HTML.

- Champ Carte.





<a name="SYNTAXE2"></a>

### Permettre ou non le zoom dans un champ HTML ou dans un champ Carte

`<Champ manipulé>.AvecZoom = <Zoom>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler : 

- Champ d'affichage HTML.

- Champ Carte.




**`<Zoom> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le champ doit permettre le zoom, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Si le zoom est autorisé, l'utilisateur pourra modifier le zoom en écartant/resserrant ses doigts sur l'écran.

- La propriété n'a aucun effet en mode simulateur.

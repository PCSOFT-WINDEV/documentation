
## Activé (Propriété)

***En anglais : Enabled***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Activé** permet d'activer ou non la gestion d'un élément. Cette propriété est utilisable sur :

- une variable de type [hDescriptionSauvegarde](../WDLang4/1000017455.md).

- une variable de type [hOptimisationPlanifiée](../WDLang4/1000018854.md). 

- une variable de type [InfoVersion](../WDLang1/1000018905.md).

- une variable de type [gpwUtilisateur](../WDLang6/1000021088.md).

- une variable de type [hTâchePlanifiée](../WDLang4/1000017448.md).

- une variable de type [hPlanificationVueMatérialisée](../WDLang4/1000021444.md).

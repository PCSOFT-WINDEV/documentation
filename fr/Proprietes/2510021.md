


## ContenuInitial (Propriété)

***En anglais : InitialContent***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **ContenuInitial** permet de connaître le contenu initial :

- d'un champ Liste par programmation.

- d'un champ Combo par programmation.

- d'une colonne d'un champ Table de type "Liste de valeurs".

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) d'un champ Tableur.




Ce contenu initial a été défini :

- lors de la description du champ (onglet "Général" du champ Liste, Combo ou de la colonne de type "Liste de valeurs").

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) lors de l'initialisation du champ Tableur sous l'éditeur de fenêtres.




Remarque : Il est également possible de modifier le contenu initial des champs Liste ou Combo par programmation s'ils sont présents dans un champ Zone répétée. 

**Important** : Un champ Zone répétée permet de créer un attribut 'Contenu initial' sur un champ de type Combo ou Liste. Il est donc possible d'avoir pour chaque occurrence de cette zone répétée un contenu différent pour les champs Liste ou Combo.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Vide la liste et remet son contenu initial
ListeSupprimeTout(LISTE_Liste1)
ListeAjoute(LISTE_Liste1, LISTE_Liste1.ContenuInitial)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le contenu initial d'un champ

`<Contenu initial> = <Champ manipulé>.ContenuInitial`
---

**`<Contenu initial> : (Chaîne de caractères)`**

Liste des éléments saisis sous l'éditeur de fenêtres ou de pages pour le contenu initial de l'objet. Chaque élément est séparé par le caractère RC.

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ dont on veut connaître le contenu initial. Ce champ doit être un champ de type : 

- Liste par programmation, 

- Combo par programmation, 

- Colonne de type "Liste de valeurs", 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Tableur initialisé directement sous l'éditeur de fenêtres.





<a name="SYNTAXE2"></a>

### Modifier le contenu initial d'un champ Liste, Combo ou colonne présent dans un champ Zone répétée

`<Nom de l'attribut>.ContenuInitial = <Contenu initial>`
---

**`<Nom de l'attribut> : (Chaîne de caractères)`**

Nom de l'attribut lié au champ dont le contenu initial doit être modifié. Le champ lié à l'attribut doit être un champ de type Liste par programmation, Combo par programmation ou colonne de type "Liste de valeurs" présent dans un champ Zone répétée. Ce paramètre est de la forme : 

```txt
<Nom de l'attribut>[Indice de la répétition]
```


**`<Contenu initial> : (Chaîne de caractères)`**

Liste des éléments définissant le contenu initial de l'objet. Chaque élément est séparé par le caractère RC.

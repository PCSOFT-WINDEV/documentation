


## MsgTriSel (Propriété)

***En anglais : MsgSortSel***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **MsgTriSel** permet de :

- Connaître le message d'attente affiché lors du tri et de la sélection des enregistrements de l'état.

- Modifier le message d'attente affiché lors du tri et de la sélection des enregistrements de l'état.




**Remarque** : Le message d'attente est défini dans les caractéristiques de l'état (onglet "UI" de la description de l'état).




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
ETAT_MonEtat.MsgTriSel = "Veuillez patienter, sélection des factures à imprimer"
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le message d'attente

`<Résultat> = <Etat manipulé>.MsgTriSel`
---

**`<Résultat> : (Chaîne de caractères)`**

Message d'attente de l'état spécifié.

**`<Etat manipulé> : (Nom d'état)`**

Nom de l'état à manipuler.  


<a name="SYNTAXE2"></a>

### Modifier le message d'attente

`<Etat manipulé>.MsgTriSel = <Nouveau message>`
---

**`<Etat manipulé> : (Nom d'état)`**

Nom de l'état à manipuler.

**`<Nouveau message> : (Chaîne de caractères)`**

Nouveau message d'attente pour l'état spécifié.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
La propriété **MsgTriSel** s'applique uniquement aux états.

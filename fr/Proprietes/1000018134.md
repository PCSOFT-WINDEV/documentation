


## JourDeLaSemaine (Propriété)

***En anglais : DayOfWeek***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **JourDeLaSemaine** permet de connaître et de modifier le jour de la semaine concerné par une planification. Cette propriété est utilisable sur :

- une variable de type [hTâchePlanifiée](../WDLang4/1000017448.md).

- une variable de type [hPlanification](../WDLang4/1000018853.md).

- une variable de type [Répétition](../WDLang1/1000019262.md).

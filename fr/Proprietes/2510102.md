


## PoliceItalique (Propriété)

***En anglais : FontItalic***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **PoliceItalique** permet de :

- Savoir si le texte affiché est en italique ou non.

- Modifier la mise en forme italique du texte.







<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Mettre le texte affiché dans le champ "LIB_ChampLibellé" en italique
LIB_ChampLibellé.PoliceItalique = Vrai
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Savoir si le texte affiché dans un champ est en italique

`<Résultat> = <Champ manipulé>.PoliceItalique`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le texte affiché est en italique, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler.  


<a name="SYNTAXE2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) 
### Modifier la mise en forme italique du texte affiché dans un champ

`<Champ manipulé>.PoliceItalique = <Italique/Non italique>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler.

**`<Italique/Non italique> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le texte affiché doit être en italique, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Limites
<a name="limites_ELTPARAGRAPHE000097"></a>Dans un état, la propriété **PoliceItalique** peut être utilisée uniquement sur :

- un champ Libellé.

- un champ Lien URL.

- un champ Calculé.

- un champ Prédéfini.




![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Dans une fenêtre, la propriété **PoliceItalique** ne peut pas être utilisée sur :

- un champ Spin,

- un champ Web Camera,

- un champ Conférence.


Remarque : Cette propriété s'applique sur les champs "Liste image" en mode d'affichage "Liste classique" uniquement.![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) La propriété **PoliceItalique** s'applique uniquement aux éléments suivants :

- Champ Bouton.

- Champ Libellé.

- Champ Combo.

- Champ Arbre.

- Colonne d'un champ Table.




## ConditionFiltre (Propriété)

***En anglais : FilterCondition***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **ConditionFiltre** permet de connaître la condition de sélection mise en place par la fonction [HFiltre](../WDLang4/3044100.md) sur un fichier de données, une vue HFSQL ou une requête.






<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// On active le filtre
HFiltre(Client, NomClient, "D" + hValMin, "F" + hValMax, "Ville=Paris")
// Insérez ici votre traitement
Trace(Client.BorneMax)
Trace(Client.BorneMin)
Trace(Client.ConditionFiltre)  // Indique "Ville='Paris'"

// On désactive le filtre
HDésactiveFiltre(Client)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Condition de sélection> = <Elément manipulé>.ConditionFiltre`
---

**`<Condition de sélection> : (Chaîne de caractères)`**



- Condition spécifiée avec la fonction [HFiltre](../WDLang4/3044100.md), 

- Chaîne vide ("") si aucun filtre n'est actif ou si aucune condition de sélection n'a été indiquée. 


Cette condition de sélection peut être une chaîne de caractères au format Ansi ou Unicode.

**`<Elément manipulé> : (Chaîne de caractères)`**

Nom du fichier de données, de la requête ou de la vue HFSQL sur laquelle le filtre a été défini.

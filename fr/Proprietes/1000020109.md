


## HauteurRessource (Propriété)

***En anglais : ResourceHeight***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **HauteurRessource** permet de connaître et de modifier la hauteur des ressources dans un champ Planning avec les ressources en ligne. 




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI INT_Zoom = Vrai ALORS
	PLN_MonPlanning.HauteurRessource = 200
SINON
	PLN_MonPlanning.HauteurRessource = 100
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître la hauteur des ressources

`<Résultat> = <Champ Planning>.HauteurRessource`
---

**`<Résultat> : (Entier)`**



- Hauteur des ressources en pixels si la hauteur des ressources a été précédemment modifiée par programmation (avec la propriété **HauteurRessource**).

- -1  dans les cas suivants : 

	- si la hauteur des ressources est automatiquement calculée (grâce aux informations données dans la fenêtre de description du champ).

	- si les ressources sont en colonne. 







**`<Champ Planning> : (Nom de champ)`**

Nom du champ Planning manipulé.  


<a name="SYNTAXE2"></a>

### Modifier la hauteur des ressources

`<Champ Planning>.HauteurRessource = <Nouvelle hauteur>`
---

**`<Champ Planning> : (Nom de champ)`**

Nom du champ Planning manipulé.

**`<Nouvelle hauteur> : (Entier)`**



- Nouvelle hauteur des ressources exprimée en pixels. 

- -1 pour calculer automatiquement la hauteur des ressources à partir des informations données dans la fenêtre de description du champ.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Si la hauteur des ressources est fixée avec la propriété **HauteurRessource**, plus aucun "ancrage" de la taille des ressources en fonction de la taille disponible n'est effectuée. 

- Seule la valeur -1 peut être utilisée. Les autres valeurs négatives sont ignorées.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) La hauteur d'une ressource spécifique peut être modifiée grâce à la propriété [Hauteur](../Proprietes/2510050.md) en utilisant la syntaxe suivante :
	
	```txt
	Nom_Planning.Ressource["MaRessource"].Hauteur = <Nouvelle valeur>
	```


## FichierSource (Propriété)

***En anglais : SourceFile***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **FichierSource** permet de paramétrer les caractéristiques d'une liaison. Il est ainsi possible de :

- Connaître le nom du fichier source utilisé dans la liaison.

- Changer le fichier source utilisé dans la liaison.




![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=FichierSource.gif)





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Modifier le fichier source
GestionProjet.FichierSource = GRPPROJ
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le fichier source de la liaison

`<Fichier de données> = <Liaison>.FichierSource`
---

**`<Fichier de données> : (Chaîne de caractères)`**

Nom du fichier source de la liaison.

**`<Liaison> : (Chaîne de caractères)`**

Nom de la liaison définie sous l'éditeur d'analyses ou définie grâce au type [Description de Liaison](../WDLang4/1514039.md).  


<a name="SYNTAXE2"></a>

### Changer le fichier source de la liaison

`<Liaison>.FichierSource = <Nouveau fichier source>`
---

**`<Liaison> : (Chaîne de caractères)`**

Nom de la liaison définie sous l'éditeur d'analyses ou définie grâce au type [Description de Liaison](../WDLang4/1514039.md).

**`<Nouveau fichier source> : (Chaîne de caractères)`**

Nom du fichier source utilisé par la liaison. Ce fichier a été défini sous l'éditeur d'analyses ou par programmation grâce au type [Description de Fichier](../WDLang4/1514065.md).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
La propriété [FichierRelié](../Proprietes/2512031.md) permet de connaître et de modifier le fichier relié de la liaison.

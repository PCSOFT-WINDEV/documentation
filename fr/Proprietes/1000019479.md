
## Icone

***En anglais : Icon***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le mot-clé **Icône** peut correspondre : 

- à la propriété **Icone**. Cette propriété permet de connaître et de modifier l'icône d'un élément. Cette propriété est utilisable sur :

	- une fenêtre. Pour plus de détails, consultez [Propriété Icone](../Proprietes/1000019344.md).

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [upnpPériphérique](../WDLang3/1000020903.md).




- à une variable de type [Icône](../WDLang1/1000020811.md).


## GrandeIcone (Propriété)

***En anglais : LargeIcon***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **GrandeIcone** permet de définir la grande icône utilisée dans les notifications. Cette icône sera affichée à gauche du titre et du message de la notification.

Cette propriété est utilisée : 

- par une variable de type [Notification](../WDLang3/1000019441.md). 

- par une variable de type [notificationFormat](../WDLang3/1000021301.md).


## IDContact (Propriété)

***En anglais : ContactID***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **IDContact** permet de gérer les identifiants de contacts. Cette propriété est utilisable sur : 

- une variable de type [sfRésultatConvertitPiste](../WDLang5/1000018739.md). 

- une variable de type [sfConversionPiste](../WDLang5/1000018742.md).

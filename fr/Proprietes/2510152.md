
## PériodeAnimation (Propriété)

***En anglais : AnimationPeriod***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **PériodeAnimation** permet de connaître ou modifier la vitesse de rotation de l'animation des champs Cube ou Carrousel. Plus exactement, cette propriété permet de régler la durée d'une rotation effectuée par le champ.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
CUBE_CUBE1.PériodeAnimation = 2000 // 20 secondes pour faire un tour
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître la période de l'animation des champs

`<Résultat> = <Champ manipulé>.PériodeAnimation`
---

**`<Résultat> : (Entier)`**

Durée d'une rotation complète de l'image autour de l'axe en centièmes de seconde.

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ Cube ou du champ Carrousel à manipuler.  


<a name="SYNTAXE2"></a>

### Modifier la période de l'animation des champs

`<Champ manipulé>.PériodeAnimation = <Nouvelle durée>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ Cube ou du champ Carrousel à manipuler.

**`<Nouvelle durée> : (Entier ou Durée)`**

Nouvelle durée d'une rotation complète de l'image autour de l'axe de rotation en centièmes de seconde. Ce paramètre peut correspondre à : 

- un entier correspondant au nombre de centièmes de seconde, 

- une variable de type Durée, 

- la durée en clair (par exemple '1cs' ou '10 ms'). 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Par défaut, le champ Cube met 350 centièmes de seconde pour effectuer une rotation complète (soit 3.5 secondes). 

Par défaut, le champ Carrousel met 1200 centièmes de seconde pour effectuer une rotation complète (soit 12 secondes).

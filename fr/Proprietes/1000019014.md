


## Note (Propriété)

***En anglais : Note***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Note** permet de connaître et de modifier un texte libre associé à un élément. Cette propriété est utilisable sur :

- Les fenêtres, les pages ou les champs. Pour plus de détails, consultez [Note](../Proprietes/2510085.md).

- une variable de type [EvénementTimeline](../WDLang1/1000020612.md).

- une variable de type [RendezVous](../WDLang1/1000019244.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [OrgaElément](../WDLang1/1000019713.md).

- une variable de type [Marqueur](../WDLang3/1000019940.md).

- une variable de type [websocketClient](../WDLang3/1000025952.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [diagForme](../WDLang1/1410088082.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [sysNotification](../WDLang1/1410087893.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [sysNotificationBouton](../WDLang1/1410087947.md).

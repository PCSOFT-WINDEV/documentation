


## VisibleUtilisateurFinal (Propriété)

***En anglais : VisibleEndUser***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **VisibleUtilisateurFinal** permet de :

- Savoir si une rubrique est visible ou non par les utilisateurs finaux. Cette rubrique a été définie sous l'éditeur d'analyses ou définie par programmation.

- Définir si une rubrique doit être visible ou non par les utilisateurs finaux (lors de la description de la rubrique par programmation uniquement).




Lorsqu'une rubrique est visible par les utilisateurs finaux :

- cette rubrique est manipulable sous le [logiciel Etats & Requêtes](../Presentation/3088003.md). Les utilisateurs finaux pourront utiliser cette rubrique dans leurs états et leurs requêtes personnalisés.

- les valeurs de cette rubrique seront modifiables dans les différentes [fenêtres d'assistance automatique HFSQL](../WDLang4/3044188.md) (en cas d'erreur de doublons par exemple). 




**Remarque** : La visibilité d'une rubrique est définie sous l'éditeur lors de la description des rubriques d'un fichier (option "Visible par l'utilisateur final").






<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Définir la visibilité de la rubrique Commande.DateCde
DateCde.VisibleUtilisateurFinal = Vrai
...
// Valide la description du fichier de données
HDécritFichier(Commande)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Savoir si la rubrique est visible par l'utilisateur final

`<Résultat> = <Fichier de données>.<Rubrique>.VisibleUtilisateurFinal`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la rubrique est visible par les utilisateurs finaux, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Fichier de données> : (Chaîne de caractères)`**

Nom du fichier de données manipulé.

**`<Rubrique> : (Chaîne de caractères)`**

Nom de la rubrique manipulée.  


<a name="SYNTAXE2"></a>

### Définir la visibilité par l'utilisateur final d'une rubrique décrite par programmation

`<Rubrique>.VisibleUtilisateurFinal = <Visibilité>`
---

**`<Rubrique> : (Chaîne de caractères)`**

Nom de la rubrique manipulée. Ce nom vient d'être défini grâce au type [Description de Rubrique](../WDLang4/1514071.md).

**`<Visibilité> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la rubrique doit être visible par les utilisateurs finaux, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.

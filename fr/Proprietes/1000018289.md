
## Jauge / Fonction Jauge (Propriété)

***En anglais : ProgressBar***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le WLangage propose :

- La fonction [Jauge](../WDLang1/3021008.md) qui permet d'afficher une jauge de progression.

- La propriété **Jauge**, utilisable sur : 

	- une variable de type [hDescriptionSauvegarde](../WDLang4/1000017455.md). 

	- une variable de type [httpAvancement](../WDLang3/1000021222.md).

	- un champ Tableau croisé dynamique. Pour plus de détails, consultez [Jauge](../Proprietes/1000020706.md).

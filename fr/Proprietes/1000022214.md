
## ClientSecret (Propriété)

***En anglais : ClientSecret***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **ClientSecret** permet de connaître le code secret du client défini lors de la déclaration (ou de l'enregistrement) de l'application. Cette propriété est utilisée par :

- une variable de type [gglConnexion](../WDLang5/1000018009.md). 

- une variable de type [OAuth2Paramètres](../WDLang3/1000022212.md). 

- une variable de type [OpenIDParamètres](../WDLang3/1000023561.md).

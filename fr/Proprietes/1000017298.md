
## EnSecondes (Propriété)

***En anglais : InSeconds***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **EnSecondes** permet de convertir une variable de type Durée en secondes. Il est ainsi possible de :

- Connaître le nombre de secondes correspondant à une durée.

- Modifier le nombre de secondes correspondant à une durée.







<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MaDurée est une Durée
MaDurée.EnHeures = 210
Trace(MaDurée.EnSecondes)  // 756000
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le nombre de secondes correspondant à une variable de type Durée

`<Résultat> = <Durée manipulée>.EnSecondes`
---

**`<Résultat> : (Réel)`**

Nombre de secondes correspondant à la durée spécifiée.

**`<Durée manipulée> : (Durée)`**

Nom de la variable de type [Durée](../Motscles/1514069.md) à manipuler.  


<a name="SYNTAXE2"></a>

### Modifier le nombre de secondes correspondant à une variable de type Durée

`<Durée manipulée>.EnSecondes = <Nouvelle durée>`
---

**`<Durée manipulée> : (Durée)`**

Nom de la variable de type [Durée](../Motscles/1514069.md) à manipuler.

**`<Nouvelle durée> : (Réel)`**

Nombre de secondes correspondant à la durée spécifiée.




## SélectionSeule (Propriété)

***En anglais : SelectionOnly***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **SélectionSeule** permet de :

- Savoir quels enregistrements du champ Table ou Table hiérarchique sont imprimés : impression de tous les enregistrements ou uniquement des enregistrements sélectionnés.

- Modifier le mode d'impression des lignes du champ Table ou Table hiérarchique : impression de tous les enregistrements ou uniquement des enregistrements sélectionnés.




Cette propriété peut être utilisée uniquement dans le code d'ouverture d'un état "Etat sur table" ou "Etat sur table hiérarchique".

![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Cette propriété est disponible par compatibilité. Les états sur table ne sont pas disponibles.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// -- Code d'ouverture de l'état
// Imprime uniquement les lignes sélectionnées dans le champ Table
ETAT_Etat1.SélectionSeule = Vrai
```
<a name="Exemple2"></a>

```wl
// -- Code d'ouverture de l'état
// Imprime toutes les lignes du champ Table
ETAT_Etat1.SélectionSeule = Faux
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le mode d'impression des lignes du champ Table ou Table hiérarchique

`<Résultat> = <Etat manipulé>.SélectionSeule`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'état va imprimer uniquement les lignes actuellement sélectionnées dans le champ Table ou Table hiérarchique,

- <u><u><u><u>Faux</u></u></u></u> si l'état va imprimer toutes les lignes du champ.




**`<Etat manipulé> : (Nom d'état)`**

Nom de l'état de type "Etat sur table" ou "Etat sur table hiérarchique" à manipuler.


<a name="SYNTAXE2"></a>

### Modifier le mode d'impression des lignes du champ Table ou Table hiérarchique

`<Etat manipulé>.SélectionSeule = <Type d'impression>`
---

**`<Etat manipulé> : (Nom d'état)`**

Nom de l'état de type "Etat sur table" ou "Etat sur table hiérarchique" à manipuler.

**`<Type d'impression> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> pour imprimer uniquement les lignes sélectionnées dans le champ Table ou Table hiérarchique,

- <u><u><u><u>Faux</u></u></u></u> pour imprimer toutes les lignes du champ.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
La propriété **SélectionSeule** s'applique uniquement aux états de type "Etat sur table" et "Etat sur table hiérarchique" et ne peut être utilisée que dans le code d'ouverture de l'état.

***En anglais : ID***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **ID** permet de connaître et de modifier un identifiant. Cette propriété est utilisable sur :

- une variable de type [Email](../WDLang3/1000018713.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [EvénementTimeline](../WDLang1/1000020612.md).

- une variable de type [RendezVous](../WDLang1/1000019244.md).

- une variable de type [sfEnregistrementSupprimé](../WDLang5/1000018751.md).

- une variable de type [sfObjet](../WDLang5/1000018439.md). 

- une variable de type [sfRésultatFusionne](../WDLang5/1000018499.md).

- une variable de type [sfRésultatRechercheModifié](../WDLang5/1000018561.md).

- une variable de type [sfRésultatRestaure](../WDLang5/1000018526.md).

- une variable de type [sfRésultatSauvegarde](../WDLang5/1000018460.md).

- une variable de type [sfRésultatSupprime](../WDLang5/1000018546.md).

- une variable de type [sfRésultatVideCorbeille](../WDLang5/1000018467.md).

- une [structure Email](../WDLang3/3032029.md).

- une variable de type [OrgaElément](../WDLang1/1000019713.md).

- une variable de type [TâcheGantt](../WDLang1/1000020881.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [mongoGridFSInfoFichier](../WDLang4/1000022417.md).

- ![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) une variable de type [Contact](../WDLang3/1000020254.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [sysNotification](../WDLang1/1410087893.md).

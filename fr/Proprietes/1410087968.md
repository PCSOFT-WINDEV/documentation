
## IndiceConfiance (Propriété)

***En anglais : ConfidenceLevel***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **IndiceConfiance** permet de connaître l'indice de confiance d'une analyse de données. Cette propriété est utilisable sur :

- une variable de type [ocrBlocTexte](../WDLang6/1000025639.md).

- une variable de type [iaRésultatImage](../WDLang6/1410087965.md).

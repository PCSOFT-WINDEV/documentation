


## Police (Propriété)

***En anglais : Font***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Police** permet de :

- Connaître l'identifiant de la police utilisée dans un champ d'un état, d'une page ou d'une fenêtre.

- Modifier la police utilisée dans un champ d'un état, d'une page ou d'une fenêtre.





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 
```wl
MaPolice est une police
MaPolice = PoliceCrée("Arial", 12, iGras)
SAI_Adresse.Police = MaPolice
```
<a name="Exemple2"></a>

```wl
// Définir les caractéristiques d'une variable de type Police
Police1 est une Police
Police1.Nom = "Arial"
Police1.Couleur = JauneClair
Police1.Taille = 14
Police1.UnitéTaille = unitéPoint
Police1.Angle = 45
// Appliquer la police au champ "MonChamp"
MonChamp.Police = Police1
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître la police utilisée dans un champ

`<Résultat> = <Champ manipulé>.Police`
---

**`<Résultat> : (Police)`**

Variable de type [Police](../Motscles/1514045.md) contenant les caractéristiques de la police utilisée dans le champ spécifié.

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler.  


<a name="SYNTAXE2"></a>

### Modifier la police utilisée dans un champ

`<Champ manipulé>.Police = <Nouvelle police>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler.

**`<Nouvelle police> : (Police)`**

Variable de type [Police](../Motscles/1514045.md) contenant les caractéristiques de la nouvelle police à utiliser dans le champ spécifié. Les caractéristiques de la police ont été définies grâce à la fonction [PoliceCrée](../WDLang1/3060002.md) ou grâce aux [propriétés de gestion des polices](../Proprietes/2515045.md).

### Limites
<a name="limites_ELTPARAGRAPHE000155"></a>![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Pour un état, la propriété **Police** peut être utilisée sur :

- un champ Libellé.

- un champ Calculé.

- un champ Lien.

- un champ Prédéfini.

- un champ de saisie.


Pour une fenêtre et une page, la propriété **Police** peut être utilisée sur :

- un champ Libellé,

- un champ de saisie,

- un champ Bouton,

- un champ Interrupteur,

- un champ Sélecteur,

- un champ Liste,

- une ligne de liste (y compris liste image) ou de combo,

- un champ Table,

- un champ Table hiérarchique,

- une cellule de table,
	

- une colonne de table,

- un champ d'affichage formaté,

- un champ HTML,

- un champ Combo,

- un champ Lien,

- un champ Libellé HTML,

- un champ Réglette,

- un champ Chemin de navigation,

- un champ Arbre,

- un Champ Menu,

- une case de barre de message.

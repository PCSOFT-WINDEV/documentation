


## MaxFeuilleParLigne (Propriété)

***En anglais : MaxLeafPerRow***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **MaxFeuilleParLigne** permet de connaître et de modifier le nombre maximum d'éléments en ligne dans un champ Organigramme. Lorsque ce nombre maximum est atteint pour le dernier niveau de la hiérarchie, les éléments sont présentés sur plusieurs lignes (organigramme vertical) ou sur plusieurs colonnes (organigramme horizontal).  

**Remarque** : Le nombre maximum de feuilles par ligne du champ Organigramme est définie dans l'onglet "Général" de la fenêtre de description du champ (option "Passer sur plusieurs lignes à partir de"). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI INT_EnColonne = Vrai ALORS
	ORGA_Principal.MaxFeuilleParLigne = 1
SINON
	ORGA_Principal.MaxFeuilleParLigne = 0 // sans limite
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître le nombre maximal de feuilles

`<Résultat> = <Champ Organigramme>.MaxFeuilleParLigne`
---

**`<Résultat> : (Entier)`**

Nombre maximal de feuilles par ligne. Par défaut, cette valeur vaut 3. 

Si cette valeur correspond à 0, il n'y a pas de limite. 

Si cette valeur correspond à 1, toutes les feuilles sont les unes sous les autres au lieu d'être cote à cote (cas d'un organigramme vertical). 

**`<Champ Organigramme> : (Nom de champ)`**

Nom du champ Organigramme à manipuler. 


<a name="SYNTAXE2"></a>

### Modifier le nombre maximal de feuilles

`<Champ Organigramme>.MaxFeuilleParLigne = <Nouvelle valeur>`
---

**`<Champ Organigramme> : (Nom de champ)`**

Nom du champ Organigramme à manipuler. 

**`<Nouvelle valeur> : (Entier)`**

Nombre maximal de feuilles par ligne. Par défaut, cette valeur vaut 3. 

Si cette valeur correspond à 0, il n'y a pas de limite. 

Si cette valeur correspond à 1, toutes les feuilles sont les unes sous les autres au lieu d'être côte à côte (cas d'un organigramme vertical).

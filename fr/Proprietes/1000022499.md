
## PoliceGras (Propriété)

***En anglais : FontBold***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **PoliceGras** permet de gérer le gras dans un texte. Cette propriété est utilisable sur : 

- les champs. Pour plus de détails, consultez [PoliceGras](../Proprietes/2510103.md). 

- une variable de type [docMiseEnForme](../WDLang1/1000022817.md).

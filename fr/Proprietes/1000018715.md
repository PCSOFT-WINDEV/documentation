
## AdresseExpéditeur (Propriété)

***En anglais : SenderAddress***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **AdresseExpéditeur** permet de connaître et de modifier l'adresse à laquelle la réponse au message doit être envoyée (Reply-To).

Cette propriété est utilisable avec :

- la structure [Email](../WDLang3/3032029.md).

- une variable de type [Email](../WDLang3/1000018713.md).

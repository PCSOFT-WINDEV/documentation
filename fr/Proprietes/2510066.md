


## LargeurImage (Propriété)

***En anglais : ImageWidth***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **LargeurImage** permet de connaître et de modifier la largeur de la portion de l'image originale qui est affichée dans un champ Image ou dans le fond d'un champ Graphe.

**Remarque** : Le mode d'affichage de l'image sélectionné dans la description du champ Image sera appliqué à la portion d'image.




<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
IMG_ChampImage.XImage = 20
IMG_ChampImage.YImage = 20
IMG_ChampImage.HauteurImage = 40
IMG_ChampImage.LargeurImage = 40
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître la largeur de la portion affichée

`<Largeur portion> = <Champ manipulé>.LargeurImage`
---

**`<Largeur portion> : (Entier)`**



- Largeur (exprimée en pixels) : 

	- de la portion d'image à afficher dans le champ Image. 

	- de la portion d'image à afficher dans le fond du champ Graphe. 




- -1 si toute l'image est affichée dans le champ Image ou dans le fond du champ Graphe.




**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler.  


<a name="SYNTAXE2"></a>

### Modifier la largeur de la portion affichée

`<Champ manipulé>.LargeurImage = <Nouvelle largeur>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler.

**`<Nouvelle largeur> : (Entier)`**



- Largeur (exprimée en pixels) : 

	- de la portion d'image à afficher dans le champ Image. 

	- de la portion d'image à afficher dans le fond du champ Graphe. 




- -1 pour afficher toute l'image dans le champ Image ou dans le fond du champ Graphe.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Définition de la portion d'image
<a name="definition_portion_image_ELTPARAGRAPHE000068"></a>Un champ Image permet d'afficher une portion de l'image contenue dans le fichier image associé. Pour définir la portion d'image affichée dans le champ Image, utilisez les propriétés suivantes :

- [XImage](../Proprietes/2510139.md) : permet de définir l'origine de la portion d'image.

- [YImage](../Proprietes/2510046.md) : permet de définir l'origine de la portion d'image.

- [HauteurImage](../Proprietes/2510044.md) : permet de définir la hauteur de la portion d'image.

- **LargeurImage** : permet de définir la largeur de la portion d'image.

<a name="NOTE0_2"></a>


### Limites
<a name="limites_ELTPARAGRAPHE000096"></a>Dans les fenêtres, cette propriété s'applique uniquement aux champs suivants : 

- Champ Image. 

- Champ Graphe. 




![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) La propriété **LargeurImage** s'applique uniquement aux champs Image.

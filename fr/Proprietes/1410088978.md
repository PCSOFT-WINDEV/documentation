
## ValeurMémorisée (Propriété)

***En anglais : StoredValue***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **ValeurMémorisée** permet de connaître et de modifier la valeur mémorisée d'un élément. Cette propriété est utilisable sur :

- les lignes des champs Liste et des champs Table, ainsi que les champs de saisie avec jetons. Pour plus de détails, consultez [ValeurMémorisée](../Proprietes/2510127.md).

- une variable de type [Jeton](../WDLang1/1000023942.md). 

- une variable de type [SaisieAssistée](../WDLang1/1410088983.md).

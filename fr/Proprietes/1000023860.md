
## NomFeuille (Propriété)

***En anglais : WorksheetName***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **NomFeuille** permet de connaître et de modifier le nom d'une feuille dans un document XLS. Cette propriété est utilisable sur :

- Un champ Tableur. Pour plus de détails, consultez l'aide de la propriété [NomFeuille](../Proprietes/1000021646.md).

- Une variable de type [ xlsDocument](../WDLang5/1000017464.md).


## Fin

***En anglais : End***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le mot-clé "FIN" peut correspondre : 

- à l'étiquette "FIN :" qui permet de définir un code qui sera exécuté systématiquement à la fin de l'exécution d'un traitement. Pour plus de détails, consultez [Etiquette "FIN :"](../Motscles/1510014.md). 

- au mot-clé signalant la fin d'une instruction composée. Pour plus de détails, consultez [Instructions composées](../Motscles/1510006.md). 

- au mot-clé signalant la fin d'une procédure interne. Pour plus de détails, consultez [Procédure interne](../Motscles/1514075.md). 

- à la propriété **Fin** qui permet de connaître la fin d'un élément. Cette propriété est utilisable sur :

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [EvénementTimeline](../WDLang1/1000020612.md). 

	- une variable de type [edtHTMLSélection](../WDLang1/1000026093.md).

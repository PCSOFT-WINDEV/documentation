
## CouleurLED (Propriété)

***En anglais : ColorLED***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **CouleurLED** permet de gérer la couleur de la LED lors de l'affichage des notifications. Cette propriété est utilisable sur :

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [Notification](../WDLang3/1000019441.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [notificationCatégorie](../WDLang3/1000023981.md).

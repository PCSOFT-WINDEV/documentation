
## ValeurInitiale (Propriété)

***En anglais : InitialValue***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **ValeurInitiale** permet de connaître et/ou de modifier une valeur initiale. Cette propriété est utilisable sur :

- les champs d'une fenêtre ou d'une page. Pour plus de détails, consultez [ ValeurInitiale](../Proprietes/2510128.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [docNiveauNumérotation](../WDLang1/1000022799.md).


## Occulté (Propriété)

***En anglais : Hidden***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Occulté** permet de : 

- Savoir si un champ d'un état est occulté. 

- Occulter un champ d'un état en le floutant ou en le biffant.




Remarques : 

- Le flou et la biffure d'un champ sont pris en compte quelque soit le mode d'impression sélectionné (PDF, Visualisateur de rapports, ...). 

- Par défaut, le niveau de flou utilisé est 100%. Cette valeur est paramétrable avec la propriété [NiveauFlou](../Proprietes/1410088348.md). 



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
LIB_LibelléTitreEtat.NiveauFlou = 50
LIB_LibelléTitreEtat.Occulté = Flou
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Savoir si un champ d'un état est occulté. 

`<Résultat> = <Champ à manipuler>.NumTAB`
---

**`<Résultat> : (Constante de type Entier)`**

Mode d'occultation du champ : 


|   |   |
| --- | --- |
| *Biffure* | Le champ est remplacé par un rectangle noir. |
| *Faux* | Le champ n'est pas occulté. |
| *Flou* | Le champ est flouté. <br><br>![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) Non disponible. |



**`<Champ à manipuler> : (Nom de champ)`**

Nom du champ présent dans un état à manipuler.  


<a name="SYNTAXE2"></a>

### Occulter un champ d'un état en le floutant ou en le biffant

`<Champ à manipuler>.NumTAB = <Mode d'occultation>`
---

**`<Champ à manipuler> : (Nom de champ)`**

Nom du champ présent dans un état à manipuler. 

**`<Mode d'occultation> : (Constante de type Entier)`**

Mode d'occultation du champ : 


|   |   |
| --- | --- |
| *Biffure* | Le champ est remplacé par un rectangle noir. |
| *Faux* | Le champ n'est pas occulté. |
| *Flou* | Le champ est flouté. <br><br>![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) Non disponible. |

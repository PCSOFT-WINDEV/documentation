


## MotDePasseWDD55 (Propriété)

***En anglais : WDD55Password***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **MotDePasseWDD55** permet de spécifier le mot de passe de l'analyse au format 5.5 lors de la description par programmation d'un fichier de données au format Hyper File 5.5 dans une analyse au format HFSQL Classic.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Définit le mot de passe associé à l'analyse 
// contenant le fichier de données Client
Client.MotDePasseWDD55 = "MotDePasse"
...
// Valide la description du fichier
HDécritFichier(Client)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Fichier de données>.MotDePasseWDD55 = <Mot de passe>`
---

**`<Fichier de données> : (Chaîne de caractères)`**

Nom du fichier de données défini grâce au type [Description de Fichier](../WDLang4/1514065.md).

**`<Mot de passe> : (Chaîne de caractères)`**

Mot de passe du WDD au format 5.5.

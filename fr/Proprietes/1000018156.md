


## CouleurJauge (Propriété)

***En anglais : ProgressBarColor***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **CouleurJauge** permet de connaître et de modifier la couleur de progression (c'est-à-dire la couleur de la zone qui avance) dans un des champs suivants : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) champ Jauge, 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) colonne de type Jauge, 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) cellule de type Jauge, 




![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Cette propriété fonctionne uniquement sur les champs Jauge.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) 
```wl
// Modification de la couleur de la jauge pour les cellules d'un champ Table
COL_Jauge[2].CouleurJauge = VertFoncé
COL_Jauge[3].CouleurJauge = RougeClair
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître la couleur de progression d'une jauge

`<Couleur de progression> = <Champ manipulé>.CouleurJauge`
---

**`<Couleur de progression> : (Entier ou constante)`**

Couleur de progression de la jauge affichée dans l'élément spécifié. Cette couleur peut correspondre à :

- une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)),

- [une couleur prédéfinie du WLangage](../WDLang5/3010002.md).




**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler. Ce champ peut correspondre à : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) un champ Jauge,

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une colonne de type Jauge,

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une cellule de type Jauge, 





<a name="SYNTAXE2"></a>

### Modifier la couleur de progression d'une jauge

`<Champ manipulé>.CouleurJauge = <Couleur de progression>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler. Ce champ peut correspondre à : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) un champ Jauge,

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une colonne de type Jauge,

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une cellule de type Jauge, 




**`<Couleur de progression> : (Entier ou constante)`**

Couleur de progression de la jauge affichée dans l'élément spécifié. Cette couleur peut correspondre à :

- une couleur RVB (obtenue avec la fonction [RVB](../WDLang1/3029012.md)),

- [une couleur prédéfinie du WLangage](../WDLang5/3010002.md).






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) 

### Définir la couleur de progression de la jauge sous l'éditeur de fenêtres
<a name="definir_couleur_progression_jauge_sous_editeur_fenetres_ELTPARAGRAPHE000130"></a>La couleur de progression d'une jauge est définie dans l'onglet "Style" de la description du champ (option "Couleur" de l'élément "Jauge").

Si la couleur de progression est une couleur dégradée, toutes les couleurs seront modifiées en conservant la même luminosité.
<a name="NOTE0_2"></a>


### Limites
<a name="limites_ELTPARAGRAPHE000139"></a>

- La propriété **CouleurJauge** est utilisable :

	- sur les champs Jauge,

	- sur les colonnes de type Jauge (pour les champs Table et Table hiérarchique),

	- sur les cellules de type Jauge (pour les champs Table et Table hiérarchique).




- Si la jauge utilise une image pour représenter la progression, la propriété **CouleurJauge** est sans effet.




![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Cette propriété fonctionne uniquement sur les champs Jauge.

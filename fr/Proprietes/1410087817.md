
## Zone (Propriété)

***En anglais : Area***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Zone** permet de définir une zone. Cette propriété est utilisable sur : 

- une variable de type [gglParamètreCarte](../WDLang5/1000017505.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [Scanneur](../WDLang6/1410087808.md).

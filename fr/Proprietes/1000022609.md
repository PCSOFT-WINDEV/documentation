
## TypeContenu (Propriété)

***En anglais : ContentType***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **TypeContenu** permet de connaître et définir le contenu d'un élément. Cette propriété peut être utilisée : 

- sur une variable de type [CodeBarres](../WDLang5/1000019451.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) sur une variable de type [mongoGridFSInfoFichier](../WDLang4/1000022417.md).




## XImage (Propriété)

***En anglais : XImage***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **XImage** permet de connaître et de modifier l'origine (abscisse) de la portion de l'image qui sera affichée dans un champ Image ou dans le fond d'un champ Graphe.

**Rappel** : L'origine de l'image correspond au point de l'image qui va être affiché dans le coin haut gauche du champ. Par défaut, **XImage** = 0.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connaître l'origine de l'image affichée dans le champ "IMG_Car"
AbscisseSource = IMG_Car.XImage
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître l'abscisse de la portion d'image affichée

`<Résultat> = <Champ manipulé>.XImage`
---

**`<Résultat> : (Entier)`**

Abscisse de la portion d'image qui sera affichée dans le champ (en pixels).

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler.  


<a name="SYNTAXE2"></a>

### Modifier l'abscisse de la portion d'image affichée

`<Champ manipulé>.XImage = <Nouvelle abscisse>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler.

**`<Nouvelle abscisse> : (Entier)`**

Nouvelle abscisse de la portion d'image qui sera affichée dans le champ (en pixels). Cette valeur doit être positive.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Définition de la portion d'image
<a name="definition_portion_image_ELTPARAGRAPHE000050"></a>Un champ Image permet d'afficher une portion de l'image contenue dans le fichier image associé. Pour définir la portion d'image affichée dans le champ Image, utilisez les propriétés suivantes :

- **XImage** : permet de définir l'origine X de la portion d'image.

- [YImage](../Proprietes/2510046.md) : permet de définir l'origine Y de la portion d'image.

- [HauteurImage](../Proprietes/2510044.md) : permet de définir la hauteur de la portion d'image.

- [LargeurImage](../Proprietes/2510066.md) : permet de définir la largeur de la portion d'image.

<a name="NOTE0_2"></a>


### Limites
<a name="limites_ELTPARAGRAPHE000078"></a>La propriété **XImage** s'applique uniquement aux champs Image et aux champs Graphe d'une fenêtre. Cette propriété ne s'applique pas aux champs Image associés à une image animée.

![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) La propriété **XImage** s'applique uniquement aux champs Image.

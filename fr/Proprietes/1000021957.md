


## PointDePassage (Propriété)

***En anglais : Waypoint***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **PointDePassage** permet de connaître et de modifier les points de passage obligatoires pour un champ Fil.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
x est un Champ
x <- ChampCrée("", typFil)
x.ChampDépart = BTN_SansNom1
x.ChampArrivée = BTN_SansNom2
x.PointDePassage = "10,50,10,120"
```



```wl
// On ajoute un point de passage au fil pour qu'il prenne le chemin que l'on souhaite
PointPassage1, PointPassage2 sont des Points 

PointPassage1.X = 280
PointPassage1.Y = 54

PointPassage2.X = 320
PointPassage2.Y = 54
FIL_MonFil.NbMaxSegment = 20

FIL_MonFil.PointDePassage = [PointPassage1, PointPassage2]
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître les points de passage obligatoires pour un champ Fil

`<Résultat> = <Champ Fil>.PointDePassage`
---

**`<Résultat> : (Tableau de points)`**

Tableau de variables de type [Point](../Motscles/1000018908.md) contenant les coordonnées des points de passage. 

Attention : Par défaut, les coordonnées des points de passage sont les coordonnées relatives à la position de départ du fil. Pour modifier ce comportement, utilisez la propriété [OriginePointDePassage](../Proprietes/1000022161.md).

**`<Champ Fil> : (Nom de champ)`**

Nom du champ Fil à manipuler.


<a name="SYNTAXE2"></a>

### Modifier les points de passage obligatoires pour un champ Fil

`<Champ Fil>.PointDePassage = <Nouveaux points>`
---

**`<Champ Fil> : (Nom de champ)`**

Nom du champ Fil à manipuler

**`<Nouveaux points> : (Tableau de points)`**



- Tableau de variables de type [Point](../Motscles/1000018908.md) contenant les coordonnées des points de passage.

- Chaîne de caractères contenant les différentes coordonnées séparées par des virgules. 

- Chaîne vide ("") pour supprimer tous les points de passage. 




Attention : Par défaut, les coordonnées des points de passage sont les coordonnées relatives à la position de départ du fil. Pour modifier ce comportement, utilisez la propriété [OriginePointDePassage](../Proprietes/1000022161.md).




## JetonActif (Propriété)

***En anglais : TokenEnabled***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **JetonActif** permet de savoir si un champ de saisie est un champ de saisie de type "Jetons de texte". 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI SAI_Email.JetonActif = Vrai ALORS 
	SAI_Email.JetonListeSéparateur = ";.:" + TAB
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Savoir si le champ de saisie est de type "Jetons de texte"

`<Résultat> = <Champ de saisie>.JetonActif`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le champ de saisie est de type "Jetons de texte", 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 




**`<Champ de saisie> : (Nom de champ)`**

Nom du champ de saisie manipulé. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Si la propriété **JetonActif** est à <u><u><u><u>Vrai</u></u></u></u>, lorsque l'utilisateur frappe un caractère séparateur ou un retour chariot (RC), un jeton est automatiquement ajouté et la saisie continue dans le champ.

Un jeton est une zone de texte encadrée qui permet à l'utilisateur d'identifier les différents éléments d'un saisie multiple. Par exemple, les jetons permettent d'identifier facilement les différents destinataires d'un email.

La valeur renvoyée ou mémorisée par un champ de saisie de type "Jetons de texte" est une chaîne séparée par RC.
Par exemple : paul@mail1.com&lt;RC&gt;mar@mail2.fr

Les séparateurs qui déclenchent la création de nouveaux jetons durant la saisie sont définis par la propriété [JetonListeSéparateur](../Proprietes/1000023783.md). 

La gestion des doublons est paramétrée par la propriété [JetonAutoriseDoublon](../Proprietes/1000023784.md). 

Il est possible d'obtenir une saisie assistée sur les jetons valides avec la fonction [SaisieAssistéeAjoute](../WDLang1/1000020043.md).

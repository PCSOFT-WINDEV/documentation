


## JourDuMoisOuDeLaSemaine (Propriété)

***En anglais : DayOfMonthOrDayOfWeek***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **JourDuMoisOuDeLaSemaine** permet de gérer l'exécution d'une tâche planifiée en précisant le numéro de jour de la semaine ou le numéro de jour du mois. Exemple : exécution tous les lundis 1er jour du mois.

Cette propriété est utilisable sur :

- une variable de type [hTâchePlanifiée](../WDLang4/1000017448.md).

- une variable de type [hPlanification](../WDLang4/1000018853.md).

- une variable de type [Répétition](../WDLang1/1000019262.md).

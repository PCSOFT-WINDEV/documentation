


## Nom55 (Propriété)

***En anglais : Name55***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Nom55** permet de gérer le nom logique d'un fichier de données au format Hyper File 5.5 non migré et présent dans une analyse au format WINDEV 28. Il est ainsi possible de :

- Connaître le nom logique d'un fichier de données.

- Définir le nom d'un fichier de données au format 5.5 créé par programmation dans une analyse au format WINDEV 28.



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connaître le nom logique du fichier de données Client
ResNom55 = Client.Nom55
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le nom logique d'un fichier de données au format 5.5 

`<Nom logique> = <Fichier de données>.Nom55`
---

**`<Nom logique> : (Chaîne de caractères)`**

Nom logique du fichier de données au format 5.5.

**`<Fichier de données> : (Chaîne de caractères)`**

Nom logique du fichier de données dans l'analyse au format WINDEV 28.  


<a name="SYNTAXE2"></a>

### Définir le nom logique 5.5 d'un fichier de données créé par programmation

`<Fichier de données>.Nom55 = <Nouveau nom logique>`
---

**`<Fichier de données> : (Chaîne de caractères)`**

Nom du fichier de données défini par programmation grâce au type [Description de Fichier](../WDLang4/1514065.md).

**`<Nouveau nom logique> : (Chaîne de caractères)`**

Nom logique du fichier de données au format 5.5.


## Taille (Propriété)

***En anglais : Size***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le mot-clé **Taille** peut correspondre à : 

- La fonction [Taille](../WDLang1/3024031.md) permet de connaître la taille d'une chaîne de caractères ou d'un buffer.

- La propriété **Taille** permet de connaître et de modifier la taille d'un élément. Cette propriété est utilisable sur :

	- les champs d'une fenêtre ou d'un état. Pour plus de détails, consultez l'aide de la propriété [Taille](../Proprietes/2510125.md).

	- les rubriques de type Texte et Heure d'un fichier de données. Pour plus de détails, consultez l'aide de la propriété [Taille](../WDLang1/3024031.md).

	- une variable de type [Police](../Motscles/1514045.md). Pour plus de détails, consultez l'aide de la propriété [Taille](../Proprietes/2510125.md).

	- une variable de type [gglMarqueur](../WDLang5/1000017442.md). 

	- une variable de type [rssDocumentJoint](../WDLang5/1000017789.md).

	- une variable de type [sfRésultatExécuteRequête](../WDLang5/1000018558.md).

	- une variable de type [InfoVersion](../WDLang1/1000018905.md).

	- une variable de type [zipArchive](../WDLang3/1000018679.md).

	- une variable de type [zipFichierArchivé](../WDLang3/1000019044.md).

	- une variable de type [Filigrane](../WDLang5/1000019922.md).

	- une variable de type [hdfsInformation](../WDLang4/1000021973.md).

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [mongoGridFSInfofichier](../WDLang4/1000022417.md).

	- une variable de type [MarqueurImage](../WDLang3/1000025796.md).




- une variable de type [MotDePasseSécurisé](../Motscles/1000021064.md).

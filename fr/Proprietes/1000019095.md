
## Site (Propriété)

***En anglais : Site***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Site** permet de connaître le site SaaS associé à :

- une variable de type [saasService](../WDLang3/1000019096.md),

- une variable de type [saasTarification](../WDLang3/1000019091.md).

- une variable de type [saasVisite](../WDLang3/1000024560.md).

- une variable de type [adminWBConnexion](../WDLang2/1410089344.md).

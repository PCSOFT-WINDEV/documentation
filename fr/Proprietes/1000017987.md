
## DateMiseAJour (Propriété)

***En anglais : UpdateDate***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **DateMiseAJour** permet de connaître la date de mise à jour d'un élément. Cette propriété est utilisable sur :

- une variable de type [gglGroupeContact](../WDLang5/1000017576.md).

- une variable de type [gglDocument](../WDLang5/1000018001.md).

- une variable de type [gglEvénement](../WDLang5/1000017440.md).

- une variable de type [rssCanal](../WDLang5/1000017769.md).

- une variable de type [rssFlux](../WDLang5/1000017795.md).

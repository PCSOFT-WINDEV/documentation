


## RègleVisible (Propriété)

***En anglais : RuleVisible***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **RègleVisible** permet de : 

- Savoir si les règles sont visibles ou invisibles dans un champ Traitement de texte. 

- Rendre les règles visibles ou invisibles dans un champ Traitement de texte. 




**Rappel** : La règle d'un champ Traitement de texte permet de visualiser et de modifier les tabulations et les marges du document.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affichage des règles
TT_MonDoc.RègleVisible = Vrai
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Savoir si les règles sont visibles ou invisibles

`<Résultat> = <Champ Traitement de texte>.RègleVisible`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si les règles sont visibles dans le champ, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Champ Traitement de texte> : (Nom de champ)`**

Nom du champ Traitement de texte manipulé. 


<a name="SYNTAXE2"></a>

### Rendre visible/invisible les règles

`<Champ Traitement de texte>.RègleVisible = <Visible/Invisible>`
---

**`<Champ Traitement de texte> : (Nom de champ)`**

Nom du champ Traitement de texte manipulé.

**`<Visible/Invisible> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si les règles doivent être visibles, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.

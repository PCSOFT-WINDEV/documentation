


## Avance (Propriété)

***En anglais : Scroll***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Avance** permet de connaître et de modifier le pas d'avancement d'un champ Ascenseur ou Range slider. Ce pas d'avancement est utilisé si l'utilisateur clique sur le bouton suivant ou le bouton précédent du champ. 

**Remarque** : Cette propriété est utile par exemple si l'ascenseur possède une étendu importante. Elle permet d'avancer rapidement d'un élément à l'autre.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Définition de la borne maximale
ASC_MonAscenseur.BorneMax = 10000
// Définition de l'avance rapide 
// (clic entre la cage de l'ascenseur et les boutons de déplacement)
ASC_MonAscenseur.AvanceRapide = 1000
// Définition du pas d'avancement (clic sur les boutons de déplacement)
ASC_MonAscenseur.Avance = 100
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître le pas d'avancement d'un champ Ascenseur ou Range slider

`<Résultat> = <Champ manipulé>.Avance`
---

**`<Résultat> : (Entier)`**

Pas d'avancement du champ. 

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ Ascenseur ou Range slider à manipuler. 


<a name="SYNTAXE2"></a>

### Modifier le pas d'avancement d'un champ Ascenseur ou Range slider

`<Champ manipulé>.Avance = <Nouveau pas d'avancement>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ Ascenseur ou Range slider à manipuler. 

**`<Nouveau pas d'avancement> : (Entier)`**

Nouveau pas d'avancement du champ. Cette valeur doit être supérieure ou égale à 1.




## RégletteVisible (Propriété)

***En anglais : RulerVisible***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **RégletteVisible** permet de : 

- Savoir une réglette est visible dans un champ TimeLine. 

- Rendre une réglette visible ou invisible dans un champ TimeLine. 




**Rappel** : La réglette d'un champ TimeLine correspond à un trait vertical "gras" permettant de visualiser un instant donné dans le champ. Le style de la réglette est défini dans l'onglet "Style" du champ TimeLine (élément "Temps sélectionné").
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Positionne la réglette à 5 secondes
TL_TimeLine.RégletteVisible = Vrai
TL_TimeLine.RégletteValeur = 5
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Savoir si une réglette est visible

`<Résultat> = <Champ manipulé>.RégletteVisible`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la réglette est visible dans le champ, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Champ manipulé> : (Nom de champ)`**

Nom du champ TimeLine manipulé. 


<a name="SYNTAXE2"></a>

### Rendre visible/invisible une réglette

`<Champ manipulé>.RégletteVisible = <Visible/Invisible>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ TimeLine manipulé.

**`<Visible/Invisible> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la réglette doit être visible, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- La réglette peut être modifiée par l'utilisateur : 

	- si l'option "Réglette modifiable" est cochée dans la fenêtre de description du champ TimeLine. 

	- si la propriété [RégletteModifiable](../Proprietes/1000020381.md) est à <u><u><u><u>Vrai</u></u></u></u>.




- La réglette est visible par l'utilisateur : 

	- si l'option "Réglette visible" est cochée dans la fenêtre de description du champ TimeLine. 

	- si la propriété **RégletteVisible** est à <u><u><u><u>Vrai</u></u></u></u>.




- La position par défaut de la réglette correspond à la valeur de la propriété [DébutEtendueTotale](../Proprietes/1000020392.md). 

- L'utilisateur final pourra modifier la valeur de la réglette en faisant glisser la ligne à la souris. Cette action déclenche l'événement "Modification" du champ TimeLine. 

- La nouvelle position de la réglette peut être connue grâce à la propriété [RégletteValeur](../Proprietes/1000020380.md).




## NuméroGénération (Propriété)

***En anglais : GenerationNumber***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **NuméroGénération** permet de connaître le numéro de génération du fichier de données physique associé au fichier de données logique spécifié.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connaître le numéro de génération du fichier de données Client
ResNumGénération = Client.NuméroGénération
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Fichier de données>.NuméroGénération`
---

**`<Résultat> : (Entier)`**

Numéro de génération du fichier de données physique actuellement associé au fichier de données logique spécifié.

**`<Fichier de données> : (Chaîne de caractères)`**

Nom logique du fichier de données manipulé.

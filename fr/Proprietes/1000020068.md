


## HauteurElément (Propriété)

***En anglais : ElementHeight***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **HauteurElément** permet de :

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Connaître ou modifier la hauteur des éléments d'un champ Organigramme.

- Connaître ou modifier la hauteur des cellules d'un champ Tableau de bord.




Remarques : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) La hauteur minimale des éléments d'un champ Organigramme est définie dans l'onglet "Général" de la fenêtre de description du champ Organigramme. 

- La hauteur d'une cellule d'un champ Tableau de bord est définie dans l'onglet "Détail" de la description d'un champ Tableau de bord. 





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 
```wl
ORGA_Principal.LargeurElément = 200
ORGA_Principal.HauteurElément = 100

SI INT_Zoom = Vrai ALORS
	ORGA_Principal.LargeurElément = 400
	ORGA_Principal.HauteurElément = 200
FIN
```
<a name="Exemple2"></a>

```wl
// Réduit la hauteur de toutes les cellules du champ Tableau de bord
SI TDB_TableauDeBord.HauteurElément > 50 ALORS 
	TDB_TableauDeBord.HauteurElément = 50
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître la hauteur des éléments ou des cellules

`<Résultat> = <Champ manipulé>.HauteurElément`
---

**`<Résultat> : (Entier)`**



- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Hauteur (en pixels) des éléments du champ Organigramme spécifié.

- Hauteur (en pixels) des cellules du champ Tableau de bord spécifié.




**`<Champ manipulé> : (Nom de champ)`**



- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Nom du champ Organigramme à manipuler. 

- Nom du champ Tableau de bord à manipuler.





<a name="SYNTAXE2"></a>

### Modifier la hauteur des éléments ou des cellules

`<Champ manipulé>.HauteurElément = <Nouvelle hauteur>`
---

**`<Champ manipulé> : (Nom de champ)`**



- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Nom du champ Organigramme à manipuler.

- Nom du champ Tableau de bord à manipuler.




**`<Nouvelle hauteur> : (Entier)`**



- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Nouvelle hauteur (en pixels) des éléments du champ Organigramme spécifié.

- Nouvelle hauteur (en pixels) des cellules du champ Tableau de bord spécifié.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
A la hauteur des éléments s'ajoute la marge entre les éléments.

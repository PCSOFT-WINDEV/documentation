


## NomVue (Propriété)

***En anglais : ViewName***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **NomVue** permet de :

- connaître le nom de la vue HFSQL, source de données d'un état

- modifier le nom de la vue HFSQL, source de données d'un état.









<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connaître le nom de la source de données de l'état "ETAT_EtatClient"
ResNomVue = ETAT_EtatClient.NomVue
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le nom de la vue HFSQL associée à un état

`<Résultat> = <Etat manipulé>.NomVue`
---

**`<Résultat> : (Chaîne de caractères)`**

Nom de la vue HFSQL, source de données de l'état spécifié.

**`<Etat manipulé> : (Nom d'état)`**

Nom de l'état à manipuler. Si ce paramètre ne correspond pas à un état associé à une vue HFSQL, une erreur du WLangage est générée.  


<a name="SYNTAXE2"></a>

### Modifier le nom de la vue HFSQL associée à un état

`<Etat manipulé>.NomVue = <Vue HFSQL>`
---

**`<Etat manipulé> : (Nom d'état)`**

Nom de l'état à manipuler. Si ce paramètre ne correspond pas à un état associé à une vue HFSQL, une erreur du WLangage est générée.

**`<Vue HFSQL> : (Chaîne de caractères)`**

Nom de la vue HFSQL à utiliser comme source de données de l'état spécifié.

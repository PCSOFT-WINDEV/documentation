


## NbVignetteDécrit (Propriété)

***En anglais : NbDescribedThumbnail***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **NbVignetteDécrit** permet de connaître le nombre de vignettes attendues pour une rubrique. Cette rubrique a été définie sous l'éditeur d'analyses ou par programmation.

Remarque : Le nombre de vignettes attendues peut être différent du nombre de vignettes réellement générées (connu avec la propriété [NbVignette](../Proprietes/2512113.md)).






<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connaître le nombre de vignettes générées 
// et le nombre de vignettes attendues
// pour la rubrique Client.Photo
NombreVignette est un entier
NombreVignetteAttendu est un entier
NombreVignette = Client.Photo.NbVignette
NombreVignetteAttendu = Client.Photo.NbVignetteDécrit
// Nombre égal ?
SI NombreVignette = NombreVignetteAttendu ALORS
	Info("Toutes les vignettes ont été générées")
SINON
	POUR Nombre = 1 A NombreVignetteAttendu
		ChaîneErreur += "Vignette " + Nombre + Client.Photo.InfoGénérationVignette[n] + RC
	FIN
	Erreur("Erreur de génération", ChaîneErreur)
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Nombre de vignettes attendu> = <Fichier de données>.<Rubrique>.NbVignetteDécrit`
---

**`<Nombre de vignettes attendu> : (Entier)`**

Nombre de vignettes demandé pour la rubrique.

**`<Fichier de données> : (Chaîne de caractères)`**

Nom du fichier de données manipulé. Ce nom a été défini sous l'éditeur d'analyses ou grâce au type [Description de Fichier](../WDLang4/1514065.md).

**`<Rubrique> : (Chaîne de caractères)`**

Nom de la rubrique de type mémo binaire manipulée. Ce nom a été défini sous l'éditeur d'analyses ou grâce au type [Description de Rubrique](../WDLang4/1514071.md).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
La propriété **NbVignetteDécrit** est disponible :

- en HFSQL Classic : sur les rubriques d'un fichier de données, d'une vue ou d'une requête.

- en HFSQL Client/Serveur : sur les rubriques d'un fichier de données uniquement.


![Linux](https://doc.pcsoft.fr/ext/images/fr/LX.png) La propriété **NbVignetteDécrit** est disponible uniquement en HFSQL Client/Serveur sur les rubriques d'un fichier de données.

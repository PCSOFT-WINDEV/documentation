
## LibelléAction (Propriété)

***En anglais : ActionLabel***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **LibelléAction** permet de connaître et de modifier le libellé du bouton d'action de la notification. Cette propriété est utilisable sur :

- une variable de type [Notification](../WDLang3/1000019441.md). 

- une variable de type [notificationAction](../WDLang3/1000021304.md).


## Timestamp (Propriété)

***En anglais : Timestamp***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Timestamp** permet de gérer l'horodatage des valeurs. Cette propriété est disponible sur :

- une variable de type [hbRésultatScan](../WDLang4/1000021724.md).

- une variable de type [hbCellule](../WDLang4/1000021707.md).

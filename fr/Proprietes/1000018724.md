
## HTML (Propriété)

***En anglais : HTML***
	

<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **HTML** permet de paramétrer l'utilisation du HTML. Cette propriété est utilisable sur : 

- un état. Pour plus de détails, consultez la propriété [HTML](../Proprietes/2511040.md).

- la structure [Email](../WDLang3/3032029.md).

- une variable de type [Email](../WDLang3/1000018713.md). 

- ![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png) une variable de type [edtHTMLSélection](../WDLang1/1000026093.md).

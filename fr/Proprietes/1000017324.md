
## Arrondi (mot-clé)

***En anglais : Round***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le mot-clé **Arrondi** peut correspondre :

- à la fonction WLangage [Arrondi](../WDLang1/3050063.md).

- à la propriété **Arrondi** utilisable sur: 

	- une variable de type [Cadre](../Motscles/15140100.md).

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [diagSélection](../WDLang1/1410088172.md).

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [diagRectangle](../WDLang1/1410088083.md).


## AnimationInitiale (Propriété)

***En anglais : InitialAnimation***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **AnimationInitiale** permet de connaître l'état initial de l'animation dans un champ Image, un champ Bouton, un champ Libellé ou un champ Carrousel. L'état initial correspond à l'état de l'animation défini lors de la description du champ (onglet "Détail").


![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=AnimationInitiale%20-%20HC%20N%B0001.gif)

<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple

![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png) 
```wl
// Ré-initialiser l'image à l'état initial
IMG_Image1.Animation = IMG_Image1.AnimationInitiale
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Etat> = <Champ manipulé>.AnimationInitiale`
---

**`<Etat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'image présente dans le champ Image, le champ Bouton ou le champ Carrousel (ou le texte présent dans le champ Libellé) est animée à la création, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 


Par défaut, l'animation est à <u><u><u><u>Vrai</u></u></u></u>.

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ manipulé. Ce champ peut correspondre à : 

- ![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png) un champ Image, 

- ![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png) un champ Bouton, 

- un champ Libellé, 

- ![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png) un champ Carrousel. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png) 

### Animation d'une image
<a name="animation_une_image_ELTPARAGRAPHE000062"></a>Pour animer l'image affichée dans le champ Image, cette image peut être :

- soit une image de type animé (par exemple, Gif animé).

- soit une image animée par WINDEV. Dans ce cas, le type de l'animation peut être choisi soit dans la fenêtre de description du champ Image (onglet "Détail"), soit par programmation avec la fonction [AnimationChangeType](../WDLang1/3017001.md).




Animation d'une image dans un champ Bouton : Seule une image de type animée peut être associée au champ (par exemple Gif Animé).
<a name="NOTE0_2"></a>
![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) 

### Animation d'un texte dans un champ Libellé
<a name="animation_texte_dans_champ_libelle_ELTPARAGRAPHE000079"></a>Pour animer un texte dans un champ Libellé, il est nécessaire de choisir l'animation effectuée :

- soit dans la fenêtre de description du champ Libellé (onglet "Détail"),

- soit par programmation avec la fonction [AnimationChangeType](../WDLang1/3017001.md).



<a name="NOTE0_3"></a>


### Limites
<a name="limites_ELTPARAGRAPHE000092"></a>La propriété **AnimationInitiale** s'applique uniquement : 

- ![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png) aux champs Image, Bouton, Libellé, Jauge et Carrousel d'une fenêtre.

- ![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) aux champs Libellé d'une fenêtre.

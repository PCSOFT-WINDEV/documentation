


## FichierRelié (Propriété)

***En anglais : LinkedFile***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **FichierRelié** permet de paramétrer les caractéristiques d'une liaison. Il est ainsi possible de :

- Connaître le nom du fichier de données relié utilisé dans la liaison.

- Changer le fichier de données relié utilisé dans la liaison.


![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=FichierRelie.gif)



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Modifier le fichier de données relié
GestionProjet.FichierRelié = Projet
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le fichier de données relié de la liaison

`<Fichier de données> = <Liaison>.FichierRelié`
---

**`<Fichier de données> : (Chaîne de caractères)`**

Nom du fichier de données relié de la liaison.

**`<Liaison> : (Chaîne de caractères)`**

Nom de la liaison définie sous l'éditeur d'analyses ou définie grâce au type [Description de Liaison](../WDLang4/1514039.md).  


<a name="SYNTAXE2"></a>

### Changer le fichier de données relié de la liaison

`<Liaison>.FichierRelié = <Fichier de données>`
---

**`<Liaison> : (Chaîne de caractères)`**

Nom de la liaison définie sous l'éditeur d'analyses ou définie grâce au type [Description de Liaison](../WDLang4/1514039.md).

**`<Fichier de données> : (Chaîne de caractères)`**

Nom du fichier de données relié utilisé par la liaison. Ce fichier de données a été défini sous l'éditeur d'analyses ou par programmation grâce au type [Description de Fichier](../WDLang4/1514065.md).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
La propriété [FichierSource](../Proprietes/2512034.md) permet de connaître et de modifier le fichier de données source de la liaison.

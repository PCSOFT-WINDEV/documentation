
## Priorité (Propriété)

***En anglais : Priority***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Priorité** permet de connaître et de modifier le niveau de priorité d'un élément. 

Cette propriété est utilisable sur :

- la structure [Email](../WDLang3/3032029.md).

- une variable de type [Email](../WDLang3/1000018713.md). 

- une variable de type [Notification](../WDLang3/1000019441.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [notificationCatégorie](../WDLang3/1000023981.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [Document](../WDLang1/1000022461.md).

- une variable de type [Thread](../WDLang1/1000024879.md).




## SaisieFormule (Propriété)

***En anglais : InputFormula***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **SaisieFormule** permet de : 

- savoir si l'utilisateur final peut saisir ou modifier des formules dans un champ Tableur. 

- autoriser ou non l'utilisateur final à saisir ou modifier des formules dans un champ Tableur.




Cette propriété permet d'éviter que l'utilisateur final puisse lancer l'exécution de code.

Remarque : Cette propriété correspond à l'option "Autoriser la saisie de formules" présente dans l'onglet "Détail" de la fenêtre de description du champ Tableur.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI INT_LectureSeule ALORS
	TBLR_Tableur.SaisieFormule = Vrai
SINON
	TBLR_Tableur.SaisieFormule = Faux
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Savoir si l'utilisateur peut modifier ou ajouter des formules

`<Résultat> = <Champ Tableur>.SaisieFormule`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'utilisateur peut modifier, ajouter des formules ou charger un fichier XLSX, 

- <u><u><u><u>Faux</u></u></u></u> si l'utilisateur ne peut pas modifier, ajouter des formules ou charger un fichier XLXS.




**`<Champ Tableur> : (Nom de champ)`**

Nom du champ Tableur à manipuler.


<a name="SYNTAXE2"></a>

### Autoriser l'utilisateur à modifier ou ajouter des formules

`<Champ Tableur>.SaisieFormule = <Autorisation>`
---

**`<Champ Tableur> : (Nom de champ)`**

Nom du champ Tableur à manipuler.

**`<Autorisation> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> pour que l'utilisateur puisse modifier, ajouter des formules ou charger un fichier XLSX. 

- <u><u><u><u>Faux</u></u></u></u> pour que l'utilisateur ne puisse pas modifier, ajouter des formules ou charger un fichier XLSX.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Si la propriété **SaisieFormule** vaut <u><u><u><u>Faux</u></u></u></u> et si la propriété [SaisieValeur](../Proprietes/1000021336.md) vaut <u><u><u><u>Vrai</u></u></u></u>, l'utilisateur final pourra saisir et modifier les valeurs dans le champ Tableur.


## Section (Propriété)

***En anglais : Section***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Section** permet de gérer les sections dans les documents de type Traitement de texte. Cette propriété est disponible : 

- sur les variables de type [docParagraphe](../WDLang1/1000022483.md). 

- sur les variables de type [Document](../WDLang1/1000022461.md). 

- avec la propriété [Sélection](../Proprietes/1000023033.md).

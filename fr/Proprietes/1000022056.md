
## Service (Propriété)

***En anglais : Service***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Service** permet de connaître et de modifier les permissions. Cette propriété est utilisable sur :

- une variable de type [btleService](../WDLang3/1000022057.md).

- une variable de type [upnpService](../WDLang3/1000020908.md).


## Attribut (Propriété)

***En anglais : Attribute***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Attribut** permet de gérer des attributs. Cette propriété est disponible sur :

- une variable de type [xmlNoeud](../WDLang5/1000018786.md).

- une variable de type [xmlLecteur](../WDLang5/1000023392.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [hbDescriptionTable](../WDLang4/1000021688.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [hbDescriptionColonne](../WDLang4/1000021694.md).

- une variable de type [xpathRésultat](../WDLang5/1000021967.md).

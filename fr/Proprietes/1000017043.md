


## Indication (Propriété)

***En anglais : Hint***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Indication** permet de connaître et de modifier le texte d'indication affiché lorsque le champ de saisie est vide.
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Indication.gif)
Ce texte d'indication est affiché directement dans le champ et il disparaît dès la saisie d'un caractère dans le champ. Ce texte d'indication peut être saisi :

- en WINDEV et WINDEV Mobile, dans l'onglet "Aide" de la fenêtre de description du champ de saisie,

- en WEBDEV, dans l'onglet "Contenu" de la fenêtre de description du champ.





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SAI_Saisie1.Indication = "Entrez le nom du client ici"
SAI_Password.Indication = "Mot de passe"
SAI_Date.Indication = "Date de sortie, ex 12/06/2007"
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le texte d'indication

`<Texte d'indication> = <Champ manipulé>.Indication`
---

**`<Texte d'indication> : (Chaîne de caractères)`**

Texte qui sera affiché dans le champ spécifié lorsque les conditions suivantes seront remplies :

- Champ de saisie (ou champ d'une combo en saisie) vide.

- L'utilisateur n'est pas en cours de saisie dans le champ.




**`<Champ manipulé> : (Nom de champ)`**

Nom du champ manipulé : champ de saisie ou combo en saisie.  


<a name="SYNTAXE2"></a>

### Modifier le texte d'indication

`<Champ manipulé>.Indication = <Texte d'indication>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ manipulé : champ de saisie ou combo en saisie.

**`<Texte d'indication> : (Chaîne de caractères)`**

Texte qui sera affiché dans le champ spécifié lorsque les conditions suivantes seront remplies :

- Champ de saisie (ou champ d'une combo en saisie) vide.

- L'utilisateur n'est pas en cours de saisie dans le champ.


Si ce paramètre correspond à chaîne vide (""), aucune indication ne sera affichée.

Ce texte peut correspondre à une chaîne graphique (utilisant par exemple les fonctions [gFond](../WDLang1/3029037.md), [gImage](../WDLang1/3029042.md), ...).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Pour que le texte d'indication apparaisse dans les champs de saisie de type Numérique, Monétaire, Monétaire + Euro, il faut cocher l'option "Mise à blanc si 0".

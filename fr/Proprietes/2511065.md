


## EspaceV (Propriété)

***En anglais : VSpace***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **EspaceV** permet de connaître et de modifier l'espacement vertical entre les étiquettes d'un état. Cet espace a été défini lors de la création de l'état au format étiquette.<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=EspaceV.gif)



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Modification de l'espace vertical entre les étiquettes
ETAT_EtatEtiquette.EspaceV += 10   // Ajoute 10 mm
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître l'espace vertical entre les étiquettes

`<Espace actuel> = <Etat Etiquette>.EspaceV`
---

**`<Espace actuel> : (Réel)`**

Espace vertical actuel entre les étiquettes. Cet espace est exprimé en millimètres.

**`<Etat Etiquette> : (Nom d'état)`**

Nom de l'état de type Étiquette.  


<a name="SYNTAXE2"></a>

### Modifier l'espace vertical entre les étiquettes

`<Etat Etiquette>.EspaceV = <Nouvel espace>`
---

**`<Etat Etiquette> : (Nom d'état)`**

Nom de l'état de type Étiquette.

**`<Nouvel espace> : (Réel)`**

Espace vertical à définir entre les étiquettes. Cet espace est exprimé en millimètres.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Hauteur et largeur des étiquettes
<a name="hauteur_largeur_des_etiquettes_ELTPARAGRAPHE000056"></a>Les propriétés [LargeurEtiquette](../Proprietes/2511055.md) et [HauteurEtiquette](../Proprietes/2511069.md) permettent de définir la taille des étiquettes par programmation.

La propriété [EspaceH](../Proprietes/2511065.md) permet de définir l'espace horizontal entre les étiquettes d'un état.
<a name="NOTE0_2"></a>


### Divers
<a name="divers_ELTPARAGRAPHE000074"></a>Cette propriété peut être utilisée dans n'importe quel événement de l'état.

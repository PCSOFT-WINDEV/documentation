
## Date

***En anglais : Date***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le mot-clé **Date** peut correspondre à : 

- une variable de type [Date](../Motscles/1514066.md). 

- la propriété **Date** d'une variable de type [InfoVersion](../WDLang1/1000018905.md).

- la propriété **Date** d'une variable de type [mongoGridFSInfoFichier](../WDLang4/1000022417.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) la propriété **Date** d'une variable de type [saasVisite](../WDLang3/1000024560.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) la propriété **Date** d'une variable de type [sysNotification](../WDLang1/1410087893.md).

- la propriété **Date** d'une variable de type [kbCarte](../WDLang1/1410089167.md).

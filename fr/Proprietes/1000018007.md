
## MotDePasse (Propriété)

***En anglais : Password***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **MotDePasse** permet de connaître et de modifier le mot de passe. Cette propriété est utilisable :

- pour savoir si un champ de saisie est de type "Mot de passe". Pour plus de détails, consultez [MotDePasse](../Proprietes/2510086.md).

- pour définir le mot de passe d'une connexion. Pour plus de détails, consultez [MotDePasse](../Proprietes/2510086.md).

- sur une variable de type [sfConnexion](../WDLang5/1000017579.md).

- sur une variable de type [emailSessionIMAP](../WDLang3/1000018957.md).

- sur une variable de type [emailSessionNotes](../WDLang3/1000018768.md).

- sur une variable de type [emailSessionPOP3](../WDLang3/1000018759.md).

- sur une variable de type [emailSessionSMTP](../WDLang3/1000018765.md).

- sur une variable de type [hParamètreFichier](../WDLang4/1000018900.md).

- sur une variable de type [saasUtilisateur](../WDLang3/1000019083.md).

- sur une variable de type [zipArchive](../WDLang3/1000018679.md).

- sur une variable générée automatiquement pour représenter [un Webservice importé](../WDLang3/3086001.md).

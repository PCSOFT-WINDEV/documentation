


## ImageFondImprimée (Propriété)

***En anglais : BackgroundImagePrinted***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **ImageFondImprimée** permet de : 

- Savoir si l'image de fond d'un état (formulaire ou non) doit être imprimée.

- Imprimer ou non l'image de fond d'un état. 


**Remarque** : Cette propriété est sauvegardée par page : il est ainsi possible de renseigner un paramétrage différent pour chaque page imprimée. 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MonEtat.ImageFond = PDFExtraitPage("fiche.pdf", 2)
MonEtat.ImageFondImprimée = Vrai
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Savoir si l'image de fond d'un état (formulaire ou non) doit être imprimée

`<Résultat> = <Etat manipulé>.ImageFondImprimée`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'image de fond doit être imprimée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 




**`<Etat manipulé> : (Nom d'état)`**

Nom de l'état manipulé.  


<a name="SYNTAXE2"></a>

### Imprimer ou non l'image de fond d'un état

`<Etat manipulé>.ImageFondImprimée = <Impression de l'image>`
---

**`<Etat manipulé> : (Nom d'état)`**

Nom de l'état manipulé.

**`<Impression de l'image> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'image de fond doit être imprimée, 

- <u><u><u><u>Faux</u></u></u></u> si l'image de fond ne doit pas être imprimée. 


Le nouveau paramétrage sera pris en compte à partir du prochain saut de page. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- La propriété **ImageFondImprimée** est sauvegardée par page : il est ainsi possible de renseigner un paramétrage différent pour chaque page imprimée. 

- Si le paramétrage est modifié avec la propriété **ImageFondImprimée**, le nouveau paramétrage sera pris en compte à partir du prochain saut de page.

- Cette propriété est disponible dans tous les types d'états. 

- Cette propriété agit sur l'image de fond des états formulaire ou celle renseignée via la propriété [ImageFond](../Proprietes/2510056.md).

- La gestion des marges spécifiée dans l'onglet "Fond" de la description de l'état (option "Ignorer les marges physiques de l'imprimante") est prise en compte.


## UnitéTaille (Propriété)

***En anglais : SizeUnit***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **UnitéTaille** permet de connaître et de modifier l'unité utilisée pour manipuler la taille d'une police définie avec une variable de type [Police](../Motscles/1514045.md).  
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
p est une Police
p.Nom = "arial"
p.UnitéTaille = unitéPoint
p.Taille = 12
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître l'unité utilisée pour définir la taille d'une police

`<Résultat> = <Police>.UnitéTaille`
---

**`<Résultat> : (Constante de type Entier)`**

Unité utilisée : 



|   |   |
| --- | --- |
| *unitéPixel* | L'unité utilisée est le pixel CSS. |
| *unitéPoint* | L'unité utilisée est le point. |
| *unitéPointW* | L'unité utilisée est le point WINDEV (unité permettant de retrouver la taille utilisée dans les versions précédentes). |



**`<Police> : (Variable de type Police)`**

Nom de la variable de type [Police](../Motscles/1514045.md) à manipuler. 


<a name="SYNTAXE2"></a>

### Modifier l'unité utilisée pour définir la taille d'une police

`<Police>.UnitéTaille = <Nouvelle unité>`
---

**`<Police> : (Variable de type Police)`**

Nom de la variable de type [Police](../Motscles/1514045.md) à manipuler. 

**`<Nouvelle unité> : (Constante de type Entier)`**

Unité à utiliser : 



|   |   |
| --- | --- |
| *unitéPixel* | L'unité à utiliser est le pixel CSS. |
| *unitéPoint* | L'unité à utiliser est le point. |
| *unitéPointW* | L'unité à utiliser est le point WINDEV (unité permettant de retrouver la taille utilisée dans les versions précédentes). |

## Remarques


### Style d'un champ : Sélection de l'unité de la police sous l'éditeur
<a name="style_champ_selection_unite_police_sous_editeur_ELTPARAGRAPHE000078"></a>Le style d'un champ permet de définir la police utilisée pour les différents éléments du champ (Onglet "Style" de la fenêtre de description du champ). Lors de la saisie des informations de taille de la police sous l'éditeur, il est possible de choisir l'unité à utiliser. 
![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Style_Unit%E9_interligne%20-%20HC%20N%B0002.gif&type=thumb)





### Variable de type Police
<a name="variable_type_police_ELTPARAGRAPHE000085"></a>Une police peut être créée à partir d'une variable de type [Police](../Motscles/1514045.md).

Pour définir les caractéristiques de la police, utilisez :

- soit la fonction [PoliceCrée](../WDLang1/3060002.md).
	

- soit les propriétés [Alphabet](../Proprietes/2512008.md), [Angle](../Proprietes/1410086632.md), [Barré](../Proprietes/2513005.md), [Couleur](../Proprietes/2510071.md), [Gras](../Proprietes/2513001.md), [Italique](../Proprietes/2513007.md), [Interlettre](../Proprietes/1410088367.md), [Interligne](../Proprietes/1410088366.md), [Nom](../Proprietes/2510082.md), [Souligné](../Proprietes/2513013.md), [Taille](../Proprietes/2510125.md) et [UnitéTaille](../Proprietes/1410088365.md).
	![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Seules les propriétés suivantes sont utilisables avec le type **Police** : [Angle](../Proprietes/1410086632.md), [Barré](../Proprietes/2513005.md), [Couleur](../Proprietes/2510071.md), [Gras](../Proprietes/2513001.md), [Italique](../Proprietes/2513007.md), [Nom](../Proprietes/2510082.md), [Souligné](../Proprietes/2513013.md), [Taille](../Proprietes/2510125.md).
	




Cette police pourra être utilisée :

- Pour écrire du texte dans des dessins (fonction [dPolice](../WDLang1/3029020.md)).

- Pour écrire dans les graphes (fonctions [grPoliceEtiquette](../WDLang3/3042012.md), [grPoliceLégende](../WDLang3/3042037.md), [grPoliceTitre](../WDLang3/3042048.md) et [grPoliceTitreAxe](../WDLang3/3042059.md)).

- Pour imprimer (fonction [iPolice](../WDLang5/3046014.md)).

- Dans les champs d'une fenêtre (propriété [Police](../Motscles/1514045.md) des champs d'une fenêtre).

- Dans les champs d'un état (propriété [Police](../Motscles/1514045.md) des champs d'un état).

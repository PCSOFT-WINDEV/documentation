


## FinCurseur (Propriété)

***En anglais : CursorEnd***
	



<a name="XPresentation"></a>
<a name="Presentation"></a>


### Définition
<a name="definition_ELTPARAGRAPHE000010"></a>Le curseur de souris permet de visualiser facilement l'endroit où l'action va se dérouler. Dans le cas d'un champ en saisie, le curseur se transforme en trait vertical clignotant et permet d'identifier rapidement la position du prochain caractère saisi. 
Si une sélection est effectuée dans le champ, les caractères sélectionnés sont affichés en "inverse vidéo". La position du curseur correspond au début de la zone affichée en "inverse vidéo". 

Quelques exemples : 


| <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=position1.gif)<br> | Le curseur de saisie est placé au début de la zone de saisie. Sa position est 1. |
| --- | --- |
| <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=Position_fin.gif)<br> | Le curseur de saisie est placé à la fin de la zone de saisie. Sa position est 7 dans cet exemple. |
| <br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=position_sel.gif)<br> | Le curseur de saisie est placé au début de la zone en "inverse vidéo" (position 3 dans cet exemple). La propriété **FinCurseur** permet de connaître et de modifier la fin de la zone sélectionnée. |
|   |   |


<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **FinCurseur** permet de connaître et de modifier la position de fin de sélection du curseur de la souris dans l'un des champs suivants :

- Champ de saisie.

- Combo avec saisie.

- Colonne d'un champ Table de type "Champ de saisie" et colonne d'un champ Table de type "Combo avec saisie".

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Champ Traitement de texte. 




Remarque : Pour connaître le texte sélectionné dans un champ de saisie, un champ Combo avec saisie ou une colonne d'un champ Table, il est possible d'utiliser la propriété [TexteSélectionné](../Proprietes/1000017193.md).



**Astuce** : Pour sélectionner une partie d'un texte d'un champ de saisie à partir d'un bouton :

1. Le champ de saisie ne doit pas avoir l'option "Effacement automatique" cochée (sauf si tout le contenu du champ doit être supprimé).

2. Le bouton doit être de type "Interruption" et contenir par exemple le code suivant (voir exemple).





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// -- Code de clic du bouton BTN_Sélection (bouton de type Interruption)
// Sélectionner tout le texte
SAI_Saisie1.Curseur = 1
SAI_Saisie1.FinCurseur = Taille(SAI_Saisie1) + 1
```
<a name="Exemple2"></a>

```wl
// Dé-sélectionner le texte
SAI_Saisie1.FinCurseur = SAI_Saisie1.Curseur
```
<a name="Exemple3"></a>

```wl
// Placer le curseur à la fin du texte
SAI_Saisie1.FinCurseur = Taille(SAI_Saisie1) + 1
SAI_Saisie1.Curseur = SAI_Saisie1.FinCurseur
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître la position du dernier caractère sélectionné

`<Position> = <Champ manipulé>.FinCurseur`
---

**`<Position> : (Entier)`**

Position du dernier caractère sélectionné + 1. Si aucun caractère n'est sélectionné, les propriétés [Curseur](../Proprietes/2510025.md) et **FinCurseur** sont équivalentes. Si le champ n'est pas en saisie, la position est 0.
<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=image17.gif)


**`<Champ manipulé> : (Nom de champ)`**

Nom du champ dans lequel une sélection a été effectuée.  


<a name="SYNTAXE2"></a>

### Sélectionner un ensemble de caractères

`<Champ manipulé>.FinCurseur = <Nombre de caractères sélectionnés>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ dans lequel une sélection doit être effectuée.

**`<Nombre de caractères sélectionnés> : (Entier)`**

La chaîne sélectionnée est comprise entre la position du curseur et la valeur transmise à la propriété **FinCurseur**. 
La plus petite valeur possible est 1 (curseur à gauche). La plus grande valeur correspond au nombre de caractères dans le champ + 1. 
**Attention** : Si la propriété **FinCurseur** est affectée avec son propre contenu, le texte est dé-sélectionné : il n'y a plus de caractères en inverse vidéo.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Limite : Sélection en inverse vidéo
<a name="limite_selection_inverse_video_ELTPARAGRAPHE000140"></a>La sélection en inverse vidéo n'est possible que si le champ est en cours de saisie. Si le champ n'est pas en cours de saisie, la propriété **FinCurseur** renvoie 0.

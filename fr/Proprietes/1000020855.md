
## HeureUTC (Propriété)

***En anglais : TimeUTC***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **HeureUTC** permet de savoir ou de définir si une heure est une heure UTC (temps universel) ou non. 

Cette propriété est utilisable sur : 

- une variable de type [hTâchePlanifiée](../WDLang4/1000017448.md).

- une variable de type [hPlanification](../WDLang4/1000018853.md).

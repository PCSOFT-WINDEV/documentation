


## YImage (Propriété)

***En anglais : YImage***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **YImage** permet de connaître et de modifier l'origine (ordonnée) de la portion de l'image qui sera affichée dans un champ Image ou dans le fond d'un champ Graphe.

**Rappel** : L'origine de l'image correspond au point de l'image qui va être affiché dans le coin haut gauche du champ. Par défaut, la propriété **YImage** est égale à 0. 


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connaître l'origine de l'image affichée dans le champ "IMG_Image_Car"
OrdonnéeSource = IMG_Image_Car.YImage
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître l'ordonnée de la portion d'image affichée

`<Résultat> = <Champ manipulé>.YImage`
---

**`<Résultat> : (Entier)`**

Ordonnée de la portion d'image qui sera affichée dans le champ (en pixels).

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler (champ Image ou champ Graphe).  


<a name="SYNTAXE2"></a>

### Modifier l'ordonnée de la portion d'image affichée

`<Champ manipulé>.YImage = <Nouvelle ordonnée>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler (champ Image ou champ Graphe).

**`<Nouvelle ordonnée> : (Entier)`**

Nouvelle ordonnée de la portion d'image qui sera affichée dans le champ (en pixels). Cette valeur doit être positive.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Définition de la portion d'image
<a name="definition_portion_image_ELTPARAGRAPHE000050"></a>Un champ Image permet d'afficher une portion de l'image contenue dans le fichier image associé. Pour définir la portion d'image affichée dans le champ Image, utilisez les propriétés suivantes :

- [XImage](../Proprietes/2510139.md) : permet de définir l'origine de la portion d'image.

- **YImage** : permet de définir l'origine de la portion d'image.

- [HauteurImage](../Proprietes/2510044.md) : permet de définir la hauteur de la portion d'image.

- [LargeurImage](../Proprietes/2510066.md) : permet de définir la largeur de la portion d'image.

<a name="NOTE0_2"></a>


### Limites
<a name="limites_ELTPARAGRAPHE000079"></a>La propriété **YImage** s'applique uniquement aux champs Image et aux champs Graphe d'une fenêtre. Cette propriété ne s'applique pas aux champs Image associés à une image animée.

![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) La propriété **YImage** s'applique uniquement aux champs Image.

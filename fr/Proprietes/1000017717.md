


## GèreHTML (Propriété)

***En anglais : ManageHTML***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **GèreHTML** permet de paramétrer le mode de gestion du HTML dans un index full-text. Il est ainsi possible de :

- Savoir si le texte des rubriques d'un index full-text peut contenir ou non du HTML. L'index full-text a été défini sous l'éditeur d'analyses par programmation.

- Définir si le texte des rubriques d'un index full-text peut contenir ou non du HTML. L'index full-text a été défini par programmation.





<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le mode de gestion du HTML

`<Mode de gestion> = <Fichier de données>.<Index full-text>.GèreHTML`
---

**`<Mode de gestion> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le texte peut contenir du HTML, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Si le texte peut contenir du HTML, les balises HTML ne seront pas indexées.




**`<Fichier de données> : (Chaîne de caractères)`**

Nom du fichier de données manipulé. Ce nom a été défini sous l'éditeur d'analyses ou grâce au type [Description de Fichier](../WDLang4/1514065.md).

**`<Index full-text> : (Chaîne de caractères)`**

Nom de l'index full-text manipulé. Ce nom a été défini sous l'éditeur d'analyses ou grâce au type [Description d'IndexFullText](../WDLang4/1000017461.md).  


<a name="SYNTAXE2"></a>

### Définir le mode de gestion du HTML

`<Index full-text>.GèreHTML = <Nouveau mode de gestion>`
---

**`<Index full-text> : (Chaîne de caractères)`**

Nom de l'index full-text manipulé. Ce nom vient d'être défini grâce au type [Description d'IndexFullText](../WDLang4/1000017461.md).

**`<Nouveau mode de gestion> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le texte peut contenir du HTML, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. Si le texte peut contenir du HTML, les balises HTML ne seront pas indexées.

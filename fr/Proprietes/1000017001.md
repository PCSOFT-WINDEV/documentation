


## Unicode (Propriété)

***En anglais : Unicode***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Unicode** permet de :

- Savoir si la valeur d'un champ est une chaîne Unicode.

- Indiquer si la valeur d'un champ est une chaîne Unicode.




**Remarque** : Cette propriété est particulièrement utile pour gérer les champs reliés à une rubrique HFSQL en Unicode.

<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Teste si la valeur du champ SAI_MessageEmail est en Unicode
SI SAI_MessageEmail.Unicode = Faux ALORS
	SAI_MessageEmail.Unicode = Vrai
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Savoir si la valeur du champ est en Unicode

`<Résultat> = <Champ manipulé>.Unicode`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la valeur du champ est en Unicode, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler.  


<a name="SYNTAXE2"></a>

### Indiquer si la valeur d'un champ est en Unicode

`<Champ manipulé>.Unicode = <Unicode/Ansi>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler.

**`<Unicode/Ansi> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la valeur du champ doit être en Unicode, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Gestion des alphabets non latins
<a name="gestion_des_alphabets_non_latins_ELTPARAGRAPHE000060"></a>Pour gérer les langues utilisant un alphabet non latin, il n'est pas nécessaire d'utiliser des champs Unicode. Il suffit d'utiliser la fonction [ChangeAlphabet](../WDLang1/3054001.md).
<a name="NOTE0_2"></a>


### Limites
<a name="limites_ELTPARAGRAPHE000070"></a>La propriété **Unicode** est disponible pour les champs suivants :

- Champ de saisie,

- Champ Combo,

- Champ Libellé,

- Champ Liste,

- Colonne de champ Table,

- Champ Liste image.

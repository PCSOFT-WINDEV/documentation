
## Poids (Propriété)

***En anglais : Weight***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Poids** permet de connaître et de modifier le poids d'un élément. Cette propriété est utilisable sur :

- un champ TreeMap. Pour plus de détails, consultez [Poids](../Proprietes/1000018904.md).

- une variable de type [memcachedConnexion](../WDLang4/1000023339.md).




## EditionHTML (Propriété)

***En anglais : HTMLEdit***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **EditionHTML** permet de : 

- savoir si un champ Traitement de texte est affiché en mode optimisé pour l'édition HTML, 

- modifier un champ Traitement de texte pour l'afficher ou non en mode optimisé pour l'édition HTML. 




Remarque : Si la propriété **EditionHTML** est activée, la propriété [ModeAffichage](../Proprietes/1000023378.md) ne peut pas être modifiée et renvoie la constante *modeAffichageWeb*. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Bouton [Mode Web]
TT_MonDocument.EditionHTML = Vrai
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Savoir si un champ Traitement de texte est affiché en mode optimisé pour l'édition HTML

`<Résultat> = <Champ Traitement de texte>.EditionHTML`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le champ Traitement de texte est affiché en mode optimisé pour l'édition HTML, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 




**`<Champ Traitement de texte> : (Nom de champ)`**

Nom du champ Traitement de texte à manipuler.


<a name="SYNTAXE2"></a>

### Modifier un champ Traitement de texte pour l'afficher ou non en mode optimisé pour l'édition HTML

`<Champ Traitement de texte>.EditionHTML = <Nouvelle valeur>`
---

**`<Champ Traitement de texte> : (Nom de champ)`**

Nom du champ Traitement de texte à manipuler.

**`<Nouvelle valeur> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le champ Traitement de texte est affiché en mode optimisé pour l'édition HTML, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.


## FuseauHoraire (Propriété)

***En anglais : TimeZone***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **FuseauHoraire** permet de paramétrer le fuseau horaire utilisé. Cette propriété est utilisable sur :

- une variable de type [gglAgenda](../WDLang5/1000017418.md). 

- une variable de type [sfInfoUtilisateur](../WDLang5/1000018423.md). 

- une variable de type [RendezVous](../WDLang1/1000019244.md).

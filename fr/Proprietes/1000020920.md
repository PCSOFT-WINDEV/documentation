


## AfficheLien (Propriété)

***En anglais : DisplayLink***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **AfficheLien** permet de connaître ou de modifier le mode d'affichage des liens dans une colonne de type Diagramme de Gantt. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SELON SEL_SansNom1 
	CAS 1 : COL_Gantt.AfficheLien = alLienTous 
	CAS 2 : COL_Gantt.AfficheLien = alLienSélection
	CAS 3 : COL_Gantt.AfficheLien = alLienAucun
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître le mode d'affichage actuel des liens dans un Diagramme de Gantt

`<Résultat> = <Colonne Gantt>.AfficheLien`
---

**`<Résultat> : (Constante de type Entier)`**

Mode d'affichage des liens dans la colonne spécifiée : 


|   |   |
| --- | --- |
| *alLienAucun* | Aucun lien n'est affiché. |
| *alLienSélection* | Seuls les liens de la tâche sélectionnée sont affichés. |
| *alLienTous* | Tous les liens sont affichés. |



**`<Colonne Gantt> : (Nom de champ)`**

Nom de la colonne (d'un champ Table ou Table hiérarchique) de type Diagramme de Gantt à manipuler.


<a name="SYNTAXE2"></a>

### Modifier le mode d'affichage actuel des liens dans un Diagramme de Gantt

`<Colonne Gantt>.AfficheLien = <Nouveau mode>`
---

**`<Colonne Gantt> : (Nom de champ)`**

Nom de la colonne (d'un champ Table ou Table hiérarchique) de type Diagramme de Gantt à manipuler. 

**`<Nouveau mode> : (Constante de type Entier)`**

Nouveau mode d'affichage des liens dans la colonne spécifiée :


|   |   |
| --- | --- |
| *alLienAucun* | Aucun lien n'est affiché. |
| *alLienSélection* | Seuls les liens de la tâche sélectionnée sont affichés. |
| *alLienTous* | Tous les liens sont affichés. |

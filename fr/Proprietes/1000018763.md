
## Option (Propriété)

***En anglais : Option***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Option** permet de connaître et de modifier certaines options. Cette propriété est utilisable sur :

- un champ Code-barres. Pour plus de détails, consultez [Option](../Proprietes/2510154.md).

- une variable de type [emailSessionIMAP](../WDLang3/1000018957.md).

- une variable de type [emailSessionPOP3](../WDLang3/1000018759.md).

- une variable de type [emailSessionSMTP](../WDLang3/1000018765.md).

- une variable de type [hParamètreFichier](../WDLang4/1000018900.md). 

- une variable de type [Description d'API](../WDLang1/1000019149.md).

- une variable de type [Description de TâcheParallèle](../WDLang1/1000021586.md).

- une variable de type [Description d'IndexFullText](../WDLang4/1000017461.md).

- une variable de type [mqttSession](../WDLang3/1000023023.md). 

- une variable de type [mongoOpérationAjout](../WDLang4/1410089163.md).

- une variable de type [mongoOpérationModificationUn](../WDLang4/1410089164.md).

- une variable de type [mongoOpérationModificationTout](../WDLang4/1410089165.md).

- une variable de type [mongoOpérationRemplacementUn](../WDLang4/1410089166.md).

- une variable de type [mongoOpérationSuppressionTout](../WDLang4/1410089162.md).

- une variable de type [mongoOpérationSuppressionUn](../WDLang4/1410089161.md).


## Ombre (Propriété)

***En anglais : Shadow***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Ombre** permet de gérer l'ombre d'une forme présente dans un diagramme. Cette propriété est utilisable sur : 

- une variable de type [diagForme](../WDLang1/1410088082.md).

- une variable de type [diagSélection](../WDLang1/1410088172.md).

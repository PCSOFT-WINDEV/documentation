
## Age

***En anglais : Age***
	

<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le mot-clé **Age** peut correspondre :

- à la fonction **[Age](../WDLang1/3027034.md)** qui permet de calculer l'âge à partir de la date de naissance. 

- à la propriété **Age** disponible sur une variable de type [MotDePasseSécurisé](../Motscles/1000021064.md).




## CouleurFond (Propriété)

***En anglais : BackgroundColor***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **CouleurFond** permet de connaître et de modifier la couleur de fond d'un élément. Cette propriété est utilisable sur :

- les champs d'une fenêtre ou d'une page. Pour plus de détails, consultez l'aide de la propriété [CouleurFond](../Proprietes/2510022.md).

- une variable de type [EvénementTimeline](../WDLang1/1000020612.md).

- une variable de type [RendezVous](../WDLang1/1000019244.md).

- une variable de type [xlsCellule](../WDLang5/1000017472.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [OrgaElément](../WDLang1/1000019713.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [pdfSignature](../WDLang6/1000022255.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [docCellule](../WDLang1/1000022927.md).




## JetonListeSéparateur (Propriété)

***En anglais : TokenListSeparator***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **JetonListeSéparateur** récupère ou modifie les séparateurs qui déclenchent la création de nouveaux jetons durant la saisie dans un champ de saisie de type  "Jetons de texte". 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI SAI_Email.JetonActif = Vrai ALORS
	SAI_Email.JetonListeSéparateur = ";.:"
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître les séparateurs déclenchant la création des jetons

`<Résultat> = <Champ de saisie>.JetonListeSéparateur`
---

**`<Résultat> : (Chaîne de caractères)`**

Liste des séparateurs autorisés pour la création de jetons.
Remarque : La touche Entrée correspond toujours à un séparateur de jetons. 

**`<Champ de saisie> : (Nom de champ)`**

Nom du champ de saisie manipulé. 


<a name="SYNTAXE2"></a>

### Modifier les séparateurs déclenchant la création des jetons

`<Champ de saisie>.JetonListeSéparateur = <Nouveaux séparateurs>`
---

**`<Champ de saisie> : (Nom de champ)`**

Nom du champ de saisie manipulé. 

**`<Nouveaux séparateurs> : (Chaîne de caractères)`**

Liste des séparateurs autorisés pour la création de jetons.
Remarque : La touche Entrée correspond toujours à un séparateur de jetons.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Une erreur est générée : 

	- si le champ de saisie n'est pas de type "Jetons de texte". 

	- si la propriété [JetonActif](../Proprietes/1000023782.md) correspond à <u><u><u><u>Faux</u></u></u></u>.




- Chaque caractère spécifié avec la propriété **JetonListeSéparateur** est traité séparément. Par exemple, si la propriété **JetonListeSéparateur** correspond à " ;:", la saisie de ':'  suffit à déclencher la création d'un jeton.

- L'appui sur la touche Entrée déclenche toujours la création d'un jeton, il n'est donc pas nécessaire de l'ajouter la liste des séparateurs de la propriété **JetonListeSéparateur**.

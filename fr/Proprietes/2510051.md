


## Ellipse (Propriété)

***En anglais : Ellipsis***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Ellipse** permet de connaître et modifier le mode de gestion de l'ellipse dans : 

- un champ Libellé, 

- un champ de saisie, 

- un champ Liste, 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png) un champ Combo,

- un bouton, 

- une colonne de table de type Libellé, 

- un champ Libellé d'un état.




**Rappel** : L'ellipse consiste à indiquer qu'un libellé est trop long pour être affiché en entier. Les caractères suivants sont alors ajoutés au libellé : "...". La propriété **Ellipse** permet d'ajouter ces caractères.

Par exemple :<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=EllipseFin.gif)<br>![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=EllipseChemin.gif)


![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) La propriété **Ellipse** est disponible uniquement sur les champs Libellé et les champs de saisie.![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Cette propriété est disponible uniquement dans les états. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Changement du mode de gestion de l'ellipse dans le champ SAI_MonChamp
SI SAI_MonChamp.Ellipse = ellipseFin ALORS SAI_MonChamp.Ellipse = ellipseChemin
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le mode de gestion de l'ellipse

`<Résultat> = <Champ manipulé>.Ellipse`
---

**`<Résultat> : (Constante de type entier)`**

Mode de gestion actuel de l'ellipse pour le champ spécifié : 


|   |   |
| --- | --- |
| *ellipseChemin* | Les caractères "..." remplacent les caractères au milieu du texte. Cette constante est conseillée pour afficher un chemin vers un fichier. |
| *ellipseFin* | Les caractères "..." remplacent les caractères qui ne peuvent pas être affichés à la fin du champ. |
| *ellipseInactive* | Les caractères "..." ne sont pas utilisés. L'ellipse n'est pas effectuée. |



**`<Champ manipulé> : (Nom de champ)`**



- Nom du champ de la fenêtre qui doit gérer l'ellipse : 

	- Libellé, 

	- Champ de saisie, 

	- Bouton, 

	- Champ Liste, 

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png) Champ Combo,

	- Colonne de type Libellé




- Nom du champ Libellé présent dans l'état qui doit gérer l'ellipse.


![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Cette propriété est disponible uniquement sur les champs Libellé et les champs de saisie.


<a name="SYNTAXE2"></a>

### Modifier le mode de gestion de l'ellipse

`<Champ manipulé>.Ellipse = <Nouveau mode de gestion>`
---

**`<Champ manipulé> : (Nom de champ)`**



- Nom du champ qui doit gérer l'ellipse : 

	- Libellé, 

	- Champ de saisie, 

	- Bouton, 

	- Champ Liste, 

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/fr/WINDOWS.png) Champ Combo,

	- Colonne de type Libellé




- Nom du champ Libellé présent dans l'état qui doit gérer l'ellipse.


![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Cette propriété est disponible uniquement sur les champs Libellé et les champs de saisie.

**`<Nouveau mode de gestion> : (Constante ou combinaison de constantes de type Entier)`**

Nouveau mode de gestion de l'ellipse pour le champ spécifié.


|   |   |
| --- | --- |
| *ellipseChemin* | Les caractères "..." remplacent les caractères au milieu du texte. Cette constante est conseillée pour afficher un chemin vers un fichier. |
| *ellipseFin* | Les caractères "..." remplacent les caractères qui ne peuvent pas être affichés à la fin du champ. |
| *ellipseInactive* | Les caractères "..." ne sont pas utilisés. L'ellipse n'est pas effectuée. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Champ de type Liste
<a name="champ_type_liste_ELTPARAGRAPHE000190"></a>Il est déconseillé d'utiliser la propriété **Ellipse** dans un champ Liste avec ascenseur horizontal. Deux cas peuvent se présenter :  

- L'ascenseur horizontal est utilisé pour visualiser la totalité de la ligne.

- La propriété **Ellipse** est utilisée. La ligne est tronquée, mais l'ascenseur horizontal devient inutile.



<a name="NOTE0_2"></a>


### Limites
<a name="limites_ELTPARAGRAPHE000206"></a>

- La propriété **Ellipse** ne peut pas être utilisée sur un champ de saisie RTF ou un libellé RTF.

- Si la propriété **Ellipse** est utilisée sur une colonne de table multiligne, le mode multiligne sera désactivé.

- La propriété **Ellipse** n'est pas prise en compte lors de l'utilisation des [fonctions de dessin dans les listes et combos](../WDLang1/3029028.md).

- ![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) Cette propriété est disponible uniquement sur les champs Libellé et les champs de saisie.

- ![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Cette propriété est disponible uniquement dans les états.


## Email (Propriété)

***En anglais : Email***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Email** permet de connaître et de modifier une adresse email. Cette propriété est utilisable sur :

- une variable de type [gglAuteur](../WDLang5/1000017746.md).

- une variable de type [gglConnexion](../WDLang5/1000018009.md).

- une variable de type [gglPersonne](../WDLang5/1000017446.md).

- une variable de type [sfInfoUtilisateur](../WDLang5/1000018423.md).

- une variable de type [gpwUtilisateur](../WDLang6/1000021088.md).

- une variable de type [saasUtilisateur](../WDLang3/1000019083.md). 

- une variable de type [OpenIDIdentité](../WDLang3/1000023567.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [adminWBCompte](../WDLang2/1410089382.md).

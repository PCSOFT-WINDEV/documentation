
## Titre (Propriété)

***En anglais : Title***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Titre** permet de :

- Connaître le titre d'une colonne d'un champ Table, d'un champ Graphe, d'une fenêtre ou d'une page. 

- Modifier le titre d'une colonne d'un champ Table, d'un champ Graphe, d'une fenêtre ou d'une page.




Dans les états, cette propriété permet de connaître et modifier le titre d'un champ Graphe. 










<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 
```wl
// Connaître le titre de la fenêtre "FEN_FormulaireSaisie"
ResTitre = FEN_FormulaireSaisie.Titre
```


<a name="Exemple2"></a>


![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) 
```wl
// Ajout d'une image dans le titre d'une colonne d'un champ Table
TABLE_MaTable.COL_MaColonne.Titre = gImage("MonImage.BMP") + "Mon titre"
```


<a name="Exemple3"></a>


![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 
```wl
// Placer la date à droite du titre de la fenêtre
MaFenêtre.Titre = EOT + gCadrage("D", DateVersChaîne(DateDuJour(), ...
		"JJJJ JJ MMMM AAAA"))
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le titre d'un élément

`<Résultat> = <Nom de l'élément>.Titre`
---

**`<Résultat> : (Chaîne de caractères)`**



- Titre de la fenêtre ou de la page spécifiée.

- Titre de la colonne spécifiée.

- Titre de la colonne à laquelle appartient la cellule de table spécifiée.

- Titre du champ Graphe spécifié.




**`<Nom de l'élément> : (Chaîne de caractères)`**

Nom de l'élément (colonne, graphe, cellule de table, fenêtre ou page) à manipuler. 
Une cellule de table n'a pas de nom. Pour récupérer le titre de la colonne à laquelle appartient une cellule de table, utilisez la syntaxe : 

```txt
<Nom de la table>[Numéro de ligne][Numéro de colonne]
```

Par exemple : 

```txt
TABLE_Table1[1][2]
```



<a name="SYNTAXE2"></a>

### Modifier le titre d'un élément

`<Nom de l'élément>.Titre = <Nouveau titre>`
---

**`<Nom de l'élément> : (Chaîne de caractères)`**

Nom de l'élément (colonne, graphe, cellule de table, fenêtre ou page) à manipuler. 
Une cellule de table n'a pas de nom. Pour récupérer le titre de la colonne à laquelle appartient une cellule de table, utilisez la syntaxe : 

```txt
<Nom de la table>[Numéro de ligne][Numéro de colonne]
```

Par exemple : 

```txt
TABLE_Table1[1][2]
```


**`<Nouveau titre> : (Tout type)`**



- Nouveau titre de la fenêtre ou de la page spécifiée. Les fonctions graphiques peuvent être utilisées dans ce paramètre. 
	**Cas particulier sous Windows Vista (et supérieur)** : Les fenêtres utilisant le look "Aero system" ne peuvent pas avoir de titre graphique : la partie graphique est ignorée. Si "Aero" n'est pas actif, le titre graphique sera affiché normalement.

- Nouveau titre de la colonne.

- Nouveau titre du champ Graphe.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 

### Cas des fenêtres détourées
<a name="cas_des_fenetres_detourees_ELTPARAGRAPHE000147"></a>Sur les fenêtres détourées, la propriété **Titre** permet également de connaître et de modifier le titre de la fenêtre. Cependant, ce titre n'est pas affiché dans la fenêtre.  

Si vous avez utilisé un gabarit détouré, le titre de la fenêtre est affiché dans un champ de type Libellé. Le contenu de ce champ n'est pas pris en compte par la propriété **Titre**.
<a name="NOTE0_2"></a>


### Équivalence
<a name="equivalence_ELTPARAGRAPHE000162"></a>La propriété **Titre** est équivalente aux propriétés [Valeur](../Proprietes/2510130.md) et [Libellé](../Proprietes/2510053.md).

Pour connaître le titre d'une fenêtre, les syntaxes suivantes sont équivalentes :


```txt
ResTitre = <Nom de la fenêtre>.Titre

ResTitre = <Nom de la fenêtre>.Valeur

ResTitre = <Nom de la fenêtre>.Libellé

ResTitre = <Nom de la fenêtre>
```


Pour connaître le titre d'une colonne, les syntaxes suivantes sont équivalentes :

```txt
ResTitre = <Nom de la colonne>.Titre

ResTitre = <Nom de la colonne>.Libellé
```


![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Pour connaître le titre de la colonne à laquelle appartient une cellule d'un champ Table, les deux syntaxes suivantes sont équivalentes :


```txt
ResTitre = <Nom du champ Table>[<Numéro de ligne>][<Numéro de colonne>].Titre

ResTitre = <Nom du champ Table>[<Numéro de ligne>][<Numéro de colonne>].Libellé
```



<a name="NOTE0_3"></a>




### Limites
<a name="limites_ELTPARAGRAPHE000193"></a>La propriété **Titre** s'applique uniquement aux :

- fenêtres et pages,

- colonnes d'un champ Table (et d'un champ Table hiérarchique),

- cellules d'un champ Table (et d'un champ Table hiérarchique),

- champs Graphe, 




![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Dans un état, la propriété **Titre** s'applique uniquement sur les champs Graphe. 

![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) En Java, la propriété **Titre** s'applique uniquement aux éléments suivants :

- Fenêtre.

- Colonne d'un champ Table.

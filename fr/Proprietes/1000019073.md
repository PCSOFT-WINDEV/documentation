
## Numéro (Propriété)

***En anglais : Number***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Numéro** permet de connaître le numéro d'un élément. Cette propriété est utilisable sur :

- une variable de type [Description d'API](../WDLang1/1000019149.md).

- une variable de type [gglTéléphone](../WDLang5/1000017573.md).  

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [docSection](../WDLang1/1000022685.md).




## ValeurParDéfautCalculée (Propriété)

***En anglais : CalculatedDefaultValue***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **ValeurParDéfautCalculée** permet de :

- Connaître la formule SQL définissant la valeur par défaut d'une rubrique. Cette rubrique a été définie sous l'éditeur d'analyses ou définie par programmation.

- Définir la formule SQL donnant la valeur par défaut d'une rubrique (lors de la description de la rubrique par programmation uniquement).





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
UnFichier est une Description de Fichier
UneRubrique est une Description de Rubrique
UneRubrique.Nom = "DateCmd"
UneRubrique.Type = hRubDateHeure
UneRubrique.ValeurParDéfautCalculée = "SYSDATE"
HDécritRubrique(UnFichier, UneRubrique)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître la formule SQL définissant la valeur par défaut d'une rubrique

`<Résultat> = <Fichier de données>.<Rubrique>.ValeurParDéfautCalculée`
---

**`<Résultat> : (Chaîne de caractères)`**

Formule SQL définissant la valeur par défaut de la rubrique.

**`<Fichier de données> : (Nom de fichier)`**

Nom du fichier de données manipulé.

**`<Rubrique> : (Nom de rubrique)`**

Nom de la rubrique manipulée.

### Définir la formule SQL donnant la valeur par défaut d'une rubrique décrite par programmation

`<Rubrique>.ValeurParDéfautCalculée = <Formule SQL>`
---

**`<Rubrique> : (Nom de rubrique)`**

Nom de la rubrique manipulée. Ce nom vient d'être défini grâce au type [Description de Rubrique](../WDLang4/1514071.md).

**`<Formule SQL> : (Chaîne de caractères)`**

Formule SQL définissant la valeur par défaut de la rubrique. Cette formule ne doit pas utiliser d'autres rubriques.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Pour définir une valeur par défaut constante, utilisez la propriété [ValeurParDéfaut](../Proprietes/2512092.md).

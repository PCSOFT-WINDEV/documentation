
## CléComposée (Propriété)

***En anglais : CompositeKey***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **CléComposée** permet de savoir si une rubrique est une clé composée. **Cette propriété est disponible en lecture seulement**.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// La rubrique Client.NomPrénom est-elle une clé composée ?
ResCléComp = Client.NomPrénom.CléComposée
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Fichier de données>.<Rubrique>.CléComposée`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la rubrique est une clé composée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Fichier de données> : (Chaîne de caractères)`**

Nom logique du fichier de données manipulé.

**`<Rubrique> : (Chaîne de caractères)`**

Nom de la rubrique manipulée.


## Colonnes (Propriété)

***En anglais : Columns***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Colonnes** permet d'accéder : 

- à la configuration d'une colonne d'un tableau dans un document au format Traitement de texte (variable [docTableau](../WDLang1/1000022924.md)).

- à la configuration d'une colonne de texte dans un document multicolonne (variable [docFormatMulticolonne](../WDLang1/1000023580.md)).

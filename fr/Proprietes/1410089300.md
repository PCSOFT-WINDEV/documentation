
## CodeSQLExécuté (Propriété)

***En anglais : SQLCodeExecuted***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **CodeSQLExécuté** permet de récupérer le code SQL d'une requête après remplacement des paramètres par leur valeur. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
REQ_CLIENT.pCodePostal = "34080"
HExécuteRequête(REQ_CLIENT)
Trace(REQ_CLIENT.CodeSQLExécuté)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Requête>.CodeSQLExécuté`
---

**`<Résultat> : (Chaîne de caractères)`**

Code SQL de la requête. 

**`<Requête> : (Nom de requête)`**

Requête manipulée. Cette requête peut correspondre à : 

- une requête WDR précédemment exécutée, 

- une source de données d'une requête, 

- une variable de type [Requête SQL](../Motscles/1514082.md) exécutée.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
La requête doit avoir été exécutée. Elle ne doit pas avoir été libérée.




## Indice (Propriété)

***En anglais : Index***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Indice** permet de connaître l'indice physique (l'ordre physique) de la rubrique dans l'enregistrement.

Remarque : La rubrique peut avoir été créée sous l'éditeur d'analyses ou par programmation.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
Info("La rubrique Nom est la " + Client.Nom.Indice + ...
		"ème rubrique physique du fichier CLIENT")
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Fichier de données>.<Rubrique>.Indice`
---

**`<Résultat> : (Entier)`**



- Indice de la rubrique, 

- 0 pour une rubrique clé composée.




**`<Fichier de données> : (Chaîne de caractères)`**

Nom logique du fichier de données manipulé.

**`<Rubrique> : (Chaîne de caractères)`**

Nom de la rubrique manipulée.




## Rubriques (Propriété)

***En anglais : Items***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Rubriques** permet de paramétrer les différentes rubriques composant un index full-text. Il est ainsi possible de :

- Connaître les différentes rubriques d'un index full-text (défini sous l'éditeur d'analyses ou défini par programmation).

- Définir les différentes rubriques d'un index full-text (lors de la description de l'index full-text par programmation).





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Description de l'index full-text
MonIndexFullText.LongueurMinimale = 4
MonIndexFullText.Nom = "IndexFullText1"
MonIndexFullText.Rubriques = "NOM+PRENOM"
MonIndexFullText.SensibleAccentuation = Vrai
MonIndexFullText.SensibleCasse = Faux
HDécritIndexFullText(UnFichier, MonIndexFullText)
...
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître la formule d'un index full-text

`<Formule de l'index> = <Fichier de données>.<Index full-text>.Rubriques`
---

**`<Formule de l'index> : (Chaîne de caractères)`**

Noms des différentes rubriques composant l'index full-text. Ces noms de rubriques sont séparés par le caractère "+".

**`<Fichier de données> : (Nom de champ)`**

Nom du fichier de données manipulé. Ce nom a été défini sous l'éditeur d'analyses ou grâce au type [Description de Fichier](../WDLang4/1514065.md).

**`<Index full-text> : (Chaîne de caractères)`**

Nom de l'index full-text manipulé. Ce nom a été défini sous l'éditeur d'analyses ou grâce au type [Description d'IndexFullText](../WDLang4/1000017461.md).  


<a name="SYNTAXE2"></a>

### Définir la formule d'un index full-text

`<Index full-text>.Rubriques = <Nouvelle formule>`
---

**`<Index full-text> : (Chaîne de caractères)`**

Nom de l'index full-text manipulé. Ce nom vient d'être défini grâce au type [Description d'IndexFullText](../WDLang4/1000017461.md).

**`<Nouvelle formule> : (Chaîne de caractères)`**

Nom des différentes rubriques composant l'index full-text. Ces noms de rubriques sont séparés par le caractère "+".

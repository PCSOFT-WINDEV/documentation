
## Null (mot-clé)

***En anglais : Null***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le mot-clé "Null" peut correspondre : 

- au mot-clé Null. Pour plus de détails, consultez [Null (Mot réservé)](../Motscles/1511015.md). 

- à la propriété [Null](../Proprietes/2512064.md). Cette propriété permet de gérer la valeur NULL dans les rubriques des fichiers de données HFSQL ou dans les rubriques des fichiers de données accédés par les Connecteurs Natifs.

- à la constante Null. Cette constante correspond à un entier initialisé à 0. Pour plus de détails, consultez [Constantes diverses](../WDLang1/3010001.md).

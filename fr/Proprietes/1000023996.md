
## Ruban (Propriété)

***En anglais : Ribbon***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Ruban** permet d'accéder au champ Ruban du champ associé (champ Traitement de texte, Lecteur PDF, Tableur, Editeur d'images, Editeur HTML ou Editeur de diagrammes). Il est ainsi possible par exemple de rendre visible ou invisible un champ du ruban. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Cache le bouton "Imprimer" du ruban
MonRuban est un Champ <- TT_Document.Ruban
{MonRuban.NomComplet + ".BTN_WDFAA_Imprimer", indChamp}..Visible = Faux
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Champ manipulé>.Ruban`
---

**`<Résultat> : (Champ)`**

Champ Ruban associé au champ spécifié. 

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ manipulé possédant un ruban : 

- Traitement de texte, 

- Lecteur PDF, 

- Tableur, 

- Editeur d'images, 

- Editeur HTML.

- Editeur de diagrammes.

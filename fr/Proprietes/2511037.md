


## NomEtatInterne (Propriété)

***En anglais : InternalReportName***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **NomEtatInterne** permet de :

- Connaître le nom de l'état imprimé dans un champ de type "[Etat Interne](../WDChamp/1013044.md)".

- Changer l'état imprimé dans un champ de type "[Etat Interne](../WDChamp/1013044.md)".





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Changer l'état imprimé dans le champ "ETATINT_EtatInterne2"
ETATINT_EtatInterne2.NomEtatInterne = ETAT_TableauProduit
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le nom de l'état imprimé dans le champ "Etat Interne"

`<Résultat> = <Champ Etat interne>.NomEtatInterne`
---

**`<Résultat> : (Chaîne de caractères)`**

Nom de l'état actuellement associé au champ de type "Etat interne" spécifié.

**`<Champ Etat interne> : (Nom de champ)`**

Nom du champ de type "Etat interne" à manipuler.  


<a name="SYNTAXE2"></a>

### Changer l'état imprimé dans le champ "Etat Interne"

`<Champ Etat interne>.NomEtatInterne = <Etat à imprimer>`
---

**`<Champ Etat interne> : (Nom de champ)`**

Nom du champ de type Etat Interne à manipuler.

**`<Etat à imprimer> : (Chaîne de caractères)`**

Nom de l'état à imprimer dans le champ "Etat interne" spécifié.

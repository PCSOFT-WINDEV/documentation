
## Déplaçable (Propriété)

***En anglais : Movable***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Déplaçable** permet de gérer la possibilité de déplacer un élément. Cette propriété est utilisable sur :

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une colonne d'un champ Table. Pour plus de détails, consultez l'aide de la propriété [Déplaçable](../Proprietes/2510032.md).

- une variable de type [Marqueur](../WDLang3/1000019940.md).




## ExécutionTerminée (Propriété)

***En anglais : ExecutionCompleted***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **ExécutionTerminée** permet de savoir si une requête ou une vue HFSQL Classic ou Client/Serveur a fini de s'exécuter.






<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI MaRequête.ExécutionTerminée = Vrai ALORS
	Info("Le nombre d'enregistrements de la vue est : " + HNbEnr(hEtatActif))
SINON
	Info("Le nombre d'enregistrements déjà récupérés est : " + HNbEnr(hNonBloquant))
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Elément manipulé>.ExécutionTerminée`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'exécution de la vue ou de la requête est terminée, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Elément manipulé> : (Chaîne de caractères)`**

Nom de la requête ou de la vue à manipuler.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- La propriété **ExécutionTerminée** peut être utilisée uniquement sur des vues et des requêtes HFSQL Classic et Client/Serveur. 

- Pour tous les autres éléments (fichiers de données, requêtes utilisant les Connecteurs Natifs, ...), la propriété **ExécutionTerminée** renvoie toujours <u><u><u><u>Vrai</u></u></u></u>.

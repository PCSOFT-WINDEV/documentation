
## Replié (Propriété)

***En anglais : Reduced***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Replié** permet de connaître ou modifier l'état replié/déplié d'un champ Panneau dockable.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// -- Clic d'un champ Bouton 
// Ce code permet de replier ou de déplier le champ Panneau dockable 
PADO_MonPanneauDockable.Replié = PAS PADO_MonPanneauDockable.Replié
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Savoir si un champ Panneau dockable est replié ou non

`<Résultat> = <Champ Panneau dockable>.Replié`
---

**`<Résultat> : ()`**



- <u><u><u><u>Vrai</u></u></u></u> si le champ Panneau dockable est replié, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Champ Panneau dockable> : (Nom de champ)`**

Nom du champ Panneau dockable à manipuler.


<a name="SYNTAXE2"></a>

### Replier ou non un champ Panneau dockable

`<Champ Panneau dockable>.Replié = <Replié/déplié>`
---

**`<Champ Panneau dockable> : (Nom de champ)`**

Nom du champ Panneau dockable à manipuler.

**`<Replié/déplié> : ()`**



- <u><u><u><u>Vrai</u></u></u></u> si le champ Panneau dockable doit être replié, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
Si le champ Panneau dockable est dédocké : 

- la propriété **Replié** renvoie toujours <u><u><u><u>Faux</u></u></u></u>,

- la modification de la propriété **Replié** est sans effet (le champ Panneau dockable reste déplié et la valeur de la propriété **Replié** reste à <u><u><u><u>Faux</u></u></u></u>).

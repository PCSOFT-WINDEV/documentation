


## Propriétés de gestion des dates et des heures
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Les différentes propriétés de gestion des dates et des heures sont les suivantes :



|   |   |
| --- | --- |
| [Année](../Proprietes/2512095.md) | La propriété **Année** permet de :<br><br>- Récupérer l'année d'une variable ou d'une rubrique de type [Date](../Motscles/1514066.md) ou [DateHeure](../Motscles/1514070.md).<br><br>- Modifier l'année d'une variable ou d'une rubrique de type [Date](../Motscles/1514066.md) ou [DateHeure](../Motscles/1514070.md).<br><br><br> |
| [EnHeures](../Proprietes/1000017296.md) | La propriété **EnHeures** permet de convertir une variable de type Durée en heures. |
| [EnJours](../Proprietes/1000017295.md) | La propriété **EnJours** permet de convertir une variable de type Durée en jours. |
| [EnMillisecondes](../Proprietes/1000017299.md) | La propriété **EnMillisecondes** permet de convertir une variable de type Durée en millisecondes. |
| [EnMinutes](../Proprietes/1000017297.md) | La propriété **EnMinutes** permet de convertir une variable de type Durée en minutes. |
| [EnSecondes](../Proprietes/1000017298.md) | La propriété **EnSecondes** permet de convertir une variable de type Durée en secondes. |
| [Heure](../Proprietes/2512038.md) | La propriété **Heure** permet de :<br><br>- Récupérer l'heure d'une variable de type [Heure](../Motscles/1514068.md), [DateHeure](../Motscles/1514070.md) ou récupérer le nombre d'heures d'une variable de type [Durée](../Motscles/1514069.md).<br><br>- Modifier l'heure d'une variable de type [Heure](../Motscles/1514068.md), [DateHeure](../Motscles/1514070.md) ou modifier le nombre d'heures d'une variable de type [Durée](../Motscles/1514069.md).<br><br>- Récupérer l'heure ou le nombre d'heures d'une rubrique de type [Date](../Motscles/1514066.md) (au format "Date et heure" ou "Durée") ou d'une rubrique de type [Heure](../Motscles/1514068.md).<br><br>- Modifier l'heure ou le nombre d'heures d'une rubrique de type [Date](../Motscles/1514066.md) (au format "Date et heure" ou "Durée") ou d'une rubrique de type [Heure](../Motscles/1514068.md).<br><br><br> |
| [Jour](../Proprietes/2512043.md) | La propriété **Jour** permet de :<br><br>- Récupérer le jour d'une variable de type [Date](../Motscles/1514066.md), [DateHeure](../Motscles/1514070.md) ou récupérer le nombre de jours d'une variable de type [Durée](../Motscles/1514069.md).<br><br>- Modifier le jour d'une variable de type [Date](../Motscles/1514066.md), [DateHeure](../Motscles/1514070.md) ou modifier le nombre de jours d'une variable de type [Durée](../Motscles/1514069.md).<br><br>- Récupérer le jour ou le nombre de jours d'une rubrique de type Date (au format "Date", "Date et heure" ou "durée")<br><br>- Modifier le jour ou le nombre de jours d'une rubrique de type Date (au format "Date", "Date et heure" ou "durée")<br><br><br> |
| [Milliseconde](../Proprietes/2512062.md) | La propriété **Milliseconde** permet de :<br><br>- Récupérer le nombre de millisecondes d'une variable de type [Heure](../Motscles/1514068.md), [DateHeure](../Motscles/1514070.md) ou [Durée](../Motscles/1514069.md).<br><br>- Modifier le nombre de millisecondes d'une variable de type [Heure](../Motscles/1514068.md), [DateHeure](../Motscles/1514070.md) ou [Durée](../Motscles/1514069.md).<br><br>- Récupérer les millisecondes ou le nombre de millisecondes d'une rubrique de type Date (au format "Date et heure" ou "durée")<br><br>- Modifier les millisecondes ou le nombre de millisecondes d'une rubrique de type Date (au format "Date et heure" ou "durée")<br><br><br> |
| [Minute](../Proprietes/2512050.md) | La propriété **Minute** permet de :<br><br>- Récupérer les minutes d'une variable de type [Heure](../Motscles/1514068.md), [DateHeure](../Motscles/1514070.md) ou le nombre de minutes d'une [Durée](../Motscles/1514069.md).<br><br>- Modifier les minutes d'une variable de type [Heure](../Motscles/1514068.md), [DateHeure](../Motscles/1514070.md) ou le nombre de minutes d'une [Durée](../Motscles/1514069.md).<br><br>- Récupérer les minutes ou le nombre de minutes d'une rubrique de type Date (au format "Date et heure" ou "durée") ou d'une rubrique de type Heure.<br><br>- Modifier les minutes ou le nombre de minutes d'une rubrique de type Date (au format "Date et heure" ou "durée") ou d'une rubrique de type Heure.<br><br><br> |
| [Mois](../Proprietes/2512052.md) | La propriété **Mois** permet de :<br><br>- Récupérer le mois d'une variable de type [Date](../Motscles/1514066.md) ou [DateHeure](../Motscles/1514070.md).<br><br>- Modifier le mois d'une variable de type [Date](../Motscles/1514066.md) ou [DateHeure](../Motscles/1514070.md).<br><br>- Récupérer le mois d'une rubrique de type Date (au format "Date simple" ou "Date et Heure")<br><br>- Modifier le mois d'une rubrique de type Date (au format "Date simple" ou "Date et Heure")<br><br><br> |
| [PartieDate](../Proprietes/2512065.md) | La propriété **PartieDate** permet de :<br><br>- Récupérer uniquement la date d'une variable de type [DateHeure](../Motscles/1514070.md).<br><br>- Modifier uniquement la date d'une variable de type [DateHeure](../Motscles/1514070.md).<br><br>- Récupérer la date d'une rubrique de type Date (au format "Date et Heure") .<br><br>- Modifier la date d'une rubrique de type Date (au format "Date et Heure")<br><br><br> |
| [PartieHeure](../Proprietes/2512069.md) | La propriété **PartieHeure** permet de :<br><br>- Récupérer uniquement l'heure d'une variable de type [DateHeure](../Motscles/1514070.md).<br><br>- Modifier uniquement l'heure d'une variable de type [DateHeure](../Motscles/1514070.md).<br><br>- Récupérer l'heure d'une rubrique de type Date (au format "Date et Heure")<br><br>- Modifier l'heure d'une rubrique de type Date (au format "Date et Heure").<br><br><br> |
| [Seconde](../Proprietes/2512081.md) | La propriété **Seconde** permet de :<br><br>- Récupérer le nombre de secondes d'une variable de type [Heure](../Motscles/1514068.md), [DateHeure](../Motscles/1514070.md) ou [Durée](../Motscles/1514069.md).<br><br>- Modifier le nombre de secondes d'une variable de type [Heure](../Motscles/1514068.md), [DateHeure](../Motscles/1514070.md) ou [Durée](../Motscles/1514069.md).<br><br>- Récupérer les secondes ou le nombre de secondes d'une rubrique de type Date (au format "Date et heure" ou "Durée")<br><br>- Modifier les secondes ou le nombre de secondes d'une rubrique de type Date (au format "Date et heure" ou "Durée")<br><br><br> |

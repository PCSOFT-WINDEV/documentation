


## TypeTri (Propriété)

***En anglais : SortType***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **TypeTri** permet de :

- Connaître le sens de parcours défini pour une rubrique ou pour une composante d'une clé composée. Cette rubrique a été définie sous l'éditeur d'analyses ou par programmation.

- Définir le sens de parcours d'une rubrique ou d'une composante de clé composée (lors de la description de la rubrique par programmation uniquement).





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connaître le sens de tri de la composante Nom de la clé composée NomPrénom
Res est un entier
Res = Client.NomPrénom[1].TypeTri
SELON Res
	CAS hCroissant : Info("Tri croissant sur Nom")
	CAS hDécroissant : Info("Tri décroissant sur Nom")
	CAS hNonTrié : Info("Pas de tri sur Nom")
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le sens de parcours d'une rubrique

`<Résultat> = <Fichier de données>.<Rubrique>.TypeTri`
---

**`<Résultat> : (Constante de type entier)`**

Tupe de tri : 



|   |   |
| --- | --- |
| *hCroissant* | Ordre croissant (ascendant) |
| *hDécroissant* | Ordre décroissant (descendant) |
| *hNonTrié* | Rubrique non triée |



**`<Fichier de données> : (Chaîne de caractères)`**

Nom du fichier de données manipulé.

**`<Rubrique> : (Chaîne de caractères)`**

Nom de la rubrique manipulée. 
Si la rubrique correspond à une composante de clé composée, le paramètre &lt;Nom de la rubrique&gt; correspond à :** &lt;Nom de la clé composée&gt;[&lt;Indice de la composante&gt;]**.
Exemple : NomPrénom[1].

### Définir le sens de parcours d'une rubrique décrite par programmation

`<Rubrique>.TypeTri = <Sens du tri>`
---

**`<Rubrique> : (Chaîne de caractères)`**

Nom de la rubrique manipulée. Ce nom vient d'être défini grâce au type [Description de Rubrique](../WDLang4/1514071.md). 
Si la rubrique correspond à une composante de clé composée, le paramètre &lt;Nom de la rubrique&gt; correspond à :** &lt;Nom de la clé composée&gt;[&lt;Indice de la composante&gt;]**.
Exemple : NomPrénom[1].

**`<Sens du tri> : (Constante de type Entier)`**

Type de tri : 



|   |   |
| --- | --- |
| *hCroissant* | Ordre croissant (ascendant) |
| *hDécroissant* | Ordre décroissant (descendant) |
| *hNonTrié* | Rubrique non triée |

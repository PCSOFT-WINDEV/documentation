


## Serveur (Propriété)

***En anglais : Server***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Serveur** permet de gérer la source de données d'une connexion (HFSQL Client/Serveur, OLE DB, ...). Il est possible de :

- Connaître la source de données utilisée pour une connexion (HFSQL Client/Serveur, OLE DB, ...).

- Définir la source de données utilisée pour la connexion  (HFSQL Client/Serveur, OLE DB, ...). Dans ce cas, si la connexion est actuellement ouverte, elle sera fermée, puis ré-ouverte avec les nouveaux paramètres lors d'un premier accès à un fichier utilisant cette connexion.




Attention : si votre base de données est distante et si vous utilisez une connexion Internet pour y accéder, il est nécessaire de rediriger le port d'entrée du routeur sur le serveur HFSQL. Pour plus de détails, consultez [Gestion des ports](../WDLang4/3044334.md).

**Remarque** : La connexion peut correspondre :

- soit à une connexion décrite sous l'éditeur d'analyses,

- soit à une connexion décrite avec les fonctions [HConnecte](../WDLang4/3044263.md), [HDécritConnexion](../WDLang4/3044205.md) ou [HOuvreConnexion](../WDLang4/3044107.md).

- soit à une connexion décrite avec une variable de type [Connexion](../WDLang4/1514073.md).









<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
MaConnexion est une Connexion
// Description de la connexion
MaConnexion.Utilisateur = "USER"
MaConnexion.MotDePasse = "PASSWORD"
MaConnexion.Serveur = "MONSERVEUR" 
MaConnexion.BaseDeDonnées = "Base de données"
MaConnexion.Provider = hAccèsHFClientServeur
MaConnexion.Accès = hOLectureEcriture
MaConnexion.InfosEtendues = "Infos étendues"
MaConnexion.OptionsCurseur = hCurseurClient

HOuvreConnexion(MaConnexion)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître la source de données de la connexion

`<Source de données> = <Connexion manipulée>.Serveur`
---

**`<Source de données> : (Chaîne de caractères)`**



- Nom du serveur (dans le cas d'une connexion HFSQL Client/Serveur, SQL Server, Oracle, MySQL, ...)

- Nom du fichier source de données (dans la cas d'une connexion Access, XML, xBase, ...)




**`<Connexion manipulée> : (Chaîne de caractères ou variable de type Connexion)`**

Nom de la connexion manipulée. Ce nom a été défini :

- soit sous l'éditeur d'analyses,

- soit avec les fonctions [HDécritConnexion](../WDLang4/3044205.md) ou [HOuvreConnexion](../WDLang4/3044107.md),

- soit avec le type [Connexion](../WDLang4/1514073.md).  





<a name="SYNTAXE2"></a>

### Définir la source de données de la connexion

`<Connexion manipulée>.Serveur = <Source de données>`
---

**`<Connexion manipulée> : (Chaîne de caractères ou variable de type Connexion)`**

Nom de la connexion manipulée. Ce nom a été défini :

- soit sous l'éditeur d'analyses,

- soit avec les fonctions [HDécritConnexion](../WDLang4/3044205.md) ou [HOuvreConnexion](../WDLang4/3044107.md),

- soit avec le type [Connexion](../WDLang4/1514073.md).




**`<Source de données> : (Chaîne de caractères)`**



- Nom du serveur ou Adresse IP du serveur (dans le cas d'une connexion HFSQL Client/Serveur, SQL Server, Oracle, MySQL, ...)

- Nom du fichier source de données (dans la cas d'une connexion Access, XML, xBase, ...)  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
La propriété **Serveur** est équivalente à la propriété [Source](../Proprietes/2510147.md).

Ces deux propriétés sont conservées pour améliorer la lisibilité du code. En effet, lors de la connexion HFSQL Client/Serveur, SQL Server, Oracle, MySQL, la propriété **Serveur** est conseillée car il est nécessaire d'indiquer un nom de serveur.

Par contre, lors de la connexion Access, XML, xBase, la propriété [Source](../Proprietes/2510147.md) est conseillée car il est nécessaire d'indiquer un fichier de données.

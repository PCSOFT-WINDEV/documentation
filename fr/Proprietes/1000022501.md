
## PoliceNom (Propriété)

***En anglais : FontName***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **PoliceNom** permet de gérer le nom de la police utilisée. Cette propriété est utilisable sur : 

- les champs. Pour plus de détails, consultez [PoliceNom](../Proprietes/2510100.md). 

- une variable de type [docMiseEnForme](../WDLang1/1000022817.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [pdfElémentTexte](../WDLang6/1000024556.md).

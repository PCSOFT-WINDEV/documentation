


## TexteAlternatif (Propriété)

***En anglais : AlternativeText***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **TexteAlternatif** permet de connaître et de modifier le texte alternatif des champs de la page.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affiche l'image du produit à partir du fichier de données Produit
IMG_Produit = Produit.Image
IMG_Produit.TexteAlternatif = Produit.Nom
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître le texte alternatif associé au champ

`<Résultat> = <Champ manipulé>.TexteAlternatif`
---

**`<Résultat> : (Chaîne de caractères)`**

Texte alternatif du champ.

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ dont le texte alternatif est recherché. Ce champ peut être : 

- un champ Image ou Image cliquable, 

- un champ Captcha, 

- un champ Vignette, 

- un champ Code-barres,

- un champ Graphe. 





<a name="SYNTAXE2"></a>

### Modifier le texte alternatif associé au champ

`<Champ manipulé>.TexteAlternatif = <Texte alternatif>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler. Ce champ peut être : 

- un champ Image ou Image cliquable, 

- un champ Captcha, 

- un champ Vignette, 

- un champ Code-barres,

- un champ Graphe. 




**`<Texte alternatif> : (Chaîne de caractères)`**

Nouveau texte alternatif du champ. Ce texte est un texte simple, sans balises HTML. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Cette propriété est équivalente à la propriété [Libellé](../Proprietes/2510053.md) sur les champs Image, Image cliquable, Captcha, Vignette, Code-barres et Graphe.

- Le texte alternatif d'une image (attribut "alt" en HTML) indique le texte affiché par le navigateur quand l'image n'est pas disponible. Ce texte est aussi utilisé à la place de l'image pour l'accessibilité des pages.

- Le texte alternatif peut également être défini sous l'éditeur dans la fenêtre de description du champ. Par exemple : 

	- dans l'onglet "Aide" pour les champs Image, Code-barres, Vignette, 

	- dans l'onglet "Détail" pour les champs Captcha, 

	- dans l'onglet "UI" pour les champs Graphe.

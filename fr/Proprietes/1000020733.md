
## Méthode (Propriété)

***En anglais : Method***
	

<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Méthode** permet de préciser la méthode utilisée. Cette propriété est utilisable sur :

- une variable de type [httpRequête](../WDLang3/1000021158.md). 

- une variable de type [restRequête](../WDLang3/1000021481.md).

- une variable de type [gglSurcharge](../WDLang5/1000026204.md).

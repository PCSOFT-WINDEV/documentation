
## HTML (Propriété)

***En anglais : HTML***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **HTML** permet de : 

- savoir si l'état en cours d'impression s'imprime en mode HTML. Il est ainsi possible par exemple de rendre invisibles certains champs inutiles lors d'une impression en mode HTML (numéro de page par exemple).

- gérer le type HTML d'un champ Libellé présent dans un état.




**Remarques** :

- L'impression en mode HTML est réalisée grâce à la fonction [iDestination](../WDLang5/3046074.md).

- La propriété **HTML** ne peut pas être utilisée dans le code d'ouverture de l'état. Cette propriété peut être utilisée dans tous les autres codes de l'état et de ses champs ou blocs (code d'initialisation d'un bloc par exemple).







<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// -- Code d'initialisation de fin de document
// Si l'impression de l'état en cours est en HTML
// Le numéro de page n'est pas imprimé. 
// Le numéro de page est associé au groupe de champs GR_GROUPE1
SI ETAT_Etat2.HTML = Vrai ALORS GR_GROUPE1.Visible = Faux
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Savoir si l'état en cours d'impression s'imprime en mode HTML

`<Résultat> = <Nom de l'état>.HTML`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'impression est effectuée en mode HTML, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Nom de l'état> : (Chaîne de caractères)`**

Nom de l'état à manipuler.  


<a name="SYNTAXE2"></a>

### Savoir si le champ Libellé est de type HTML

`<Résultat> = <Nom du champ>.HTML`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le champ Libellé est de type HTML, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Nom du champ> : (Chaîne de caractères)`**

Nom du champ Libellé à manipuler.  




<a name="SYNTAXE3"></a>

### Modifier le type HTML d'un champ Libellé

`<Nom du champ>.HTML = <Nouveau type>`
---

**`<Nom du champ> : (Chaîne de caractères)`**

Nom du champ Libellé à manipuler. 

**`<Nouveau type> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le champ Libellé doit être de type HTML, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.








<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- La propriété **HTML** ne peut pas être utilisée dans le code d'ouverture de l'état.

- La propriété **HTML** s'applique uniquement : 

	- aux états. 

	- aux champs Libellé des états.

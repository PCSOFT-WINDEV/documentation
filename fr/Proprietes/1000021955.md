


## TimestampUTC (Propriété)

***En anglais : TimestampUTC***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **TimestampUTC** permet de :

- Savoir si une rubrique de type horodatage utilisera une heure UTC ou locale. Cette rubrique a été définie sous l'éditeur d'analyses ou définie par programmation.

- Définir si une rubrique de type horodatage utilisera une heure UTC ou locale (lors de la description de la rubrique par programmation uniquement).





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
UnFichier est une Description de Fichier
UneRubrique est une Description de Rubrique
UneRubrique.Nom = "DateCmd"
UneRubrique.TimestampType = htsCréation
UneRubrique.TimestampUTC = Faux
HDécritRubrique(UnFichier, UneRubrique)
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Savoir si une rubrique d'horodatage utilisera une heure UTC

`<Résultat> = <Fichier de données>.<Rubrique>.TimestampUTC`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la rubrique de type horodatage utilise une heure UTC.

- <u><u><u><u>Faux</u></u></u></u> si la rubrique de type horodatage utilise une heure locale.




**`<Fichier de données> : (Chaîne de caractères)`**

Nom du fichier de données manipulé.

**`<Rubrique> : (Chaîne de caractères)`**

Nom de la rubrique manipulée.

### Indiquer si une rubrique d'horodatage utilisera une heure UTC

`<Rubrique>.TimestampUTC = <Valeur>`
---

**`<Rubrique> : (Chaîne de caractères)`**

Nom de la rubrique manipulée. Ce nom vient d'être défini grâce au type [Description de Rubrique](../WDLang4/1514071.md).

**`<Valeur> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la rubrique de type horodatage doit utiliser une heure UTC.

- <u><u><u><u>Faux</u></u></u></u> si la rubrique de type horodatage doit utiliser une heure locale.

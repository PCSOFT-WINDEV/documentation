


## FenêtreSource (Propriété)

***En anglais : SourceWindow***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **FenêtreSource** permet de connaître la source d'une fenêtre interne (c'est-à-dire la fenêtre interne affichée par le champ de type Fenêtre Interne).

**Remarque** : Pour modifier la fenêtre interne associée à un champ de type "Fenêtre interne" utilisez la fonction [ChangeFenêtreSource](../WDLang1/3038053.md).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
Trace(FI_ChampFenInterne1.FenêtreSource) // Affiche FI_FenInterne2
```
<a name="Exemple2"></a>

```wl
// Changement de la source de la fenêtre interne
SI FI_ChampFen1.FenêtreSource = "FI_SaisieParam" ALORS
	ChangeFenêtreSource(FI_ChampFen1, FI_SaisieParam2)
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Fenêtre interne utilisée> = <Champ Fenêtre Interne>.FenêtreSource`
---

**`<Fenêtre interne utilisée> : (Nom de fenêtre)`**

Nom de la fenêtre interne utilisée par le champ de type "Fenêtre interne".

**`<Champ Fenêtre Interne> : (Nom de champ)`**

Nom du champ Fenêtre interne.

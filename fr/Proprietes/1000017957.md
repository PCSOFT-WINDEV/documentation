
## Libellé (Propriété)

***En anglais : Caption***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Libellé** permet de connaître et de modifier le libellé d'un élément. Cette propriété est utilisable sur :

- les champs, les fenêtres, les pages et les états. Pour plus de détails, consultez [Libellé](../Proprietes/2510053.md).

- une variable de type [gglAdressePostale](../WDLang5/1000017575.md).

- une variable de type [gglContact](../WDLang5/1000017434.md).

- une variable de type [gglEmail](../WDLang5/1000017571.md).

- une variable de type [gglEvénement](../WDLang5/1000017440.md).

- une variable de type [gglLieu](../WDLang5/1000017441.md).

- une variable de type [gglMessagerieInstantanée](../WDLang5/1000017569.md).

- une variable de type [gglPersonne](../WDLang5/1000017446.md).

- une variable de type [gglSociété](../WDLang5/1000017756.md).

- une variable de type [gglTéléphone](../WDLang5/1000017573.md).

- une variable de type [inAppProduit](../WDLang3/1000020868.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [pdfSignature](../WDLang6/1000022255.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [Jeton](../WDLang1/1000023942.md).

- une variable de type [MarqueurImage](../WDLang3/1000025796.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [diagTexte](../WDLang1/1410088393.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [sysNotificationBouton](../WDLang1/1410087947.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [sysNotificationEntrée](../WDLang1/1410088024.md).

- une variable de type [SaisieAssistée](../WDLang1/1410088983.md).

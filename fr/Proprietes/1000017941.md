
## Version (Propriété)

***En anglais : Version***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Version** permet de connaître et de modifier la version d'un élément. Cette propriété est utilisable sur :

- une variable de type [rssFlux](../WDLang5/1000017795.md).

- une variable de type [xlsDocument](../WDLang5/1000017464.md).

- une variable de type [xmlDocument](../WDLang5/1000018782.md).

- une variable de type [hbLecture](../WDLang4/1000021701.md).

- une variable de type [pdfParamètre](../WDLang6/1410087857.md).




## NbEnrLu (Propriété)

***En anglais : NbRecRead***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **NbEnrLu** permet de connaître le nombre d'enregistrements lus : sélectionnés à la fois par la source de l'état ET par le code de filtre (s'il existe).


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Si plus de 100 enregistrements ont été lus
SI ETAT_EtatClient.NbEnrLu > 100 ALORS
	// L'impression de l'état "ETAT_EtatClient" est arrêtée
	iFinEtat()
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Etat manipulé>.NbEnrLu`
---

**`<Résultat> : (Entier)`**

Nombre d'enregistrements lus.

**`<Etat manipulé> : (Nom d'état)`**

Nom de l'état à manipuler.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Nombre d'enregistrements
<a name="nombre_enregistrements_ELTPARAGRAPHE000047"></a>

- La propriété **NbEnrLu** renvoie le nombre d'enregistrements sélectionnés par la source de l'état (requête, vue, zone mémoire ou champ Table).

- La propriété [NbEnrImprimé](../Proprietes/2511058.md) renvoie le nombre d'enregistrements sélectionnés à la fois par la source de l'état ET par le code de filtre (s'il existe).

- Dans le cas d'un état sur fichier, les propriétés **NbEnrLu** et [NbEnrImprimé](../Proprietes/2511058.md) retournent la même valeur : le nombre d'enregistrements du fichier ET sélectionnés par le code de filtre (s'il existe).

- La propriété [NbEnrTotal](../Proprietes/2511047.md) permet de connaître et de modifier le nombre d'enregistrements à imprimer. Le nombre d'enregistrements à imprimer sera compris entre 1 et [NbEnrImprimé](../Proprietes/2511058.md).



<a name="NOTE0_2"></a>


### Limite
<a name="limite_ELTPARAGRAPHE000075"></a>La propriété **NbEnrLu** ne peut pas être utilisée sur un état sans source de données.

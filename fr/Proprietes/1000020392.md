


## DébutEtendueTotale (Propriété)

***En anglais : StartTotalRange***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **DébutEtendueTotale** permet de : 

- connaître la première date ou heure affichable : 

	- dans un champ Planning. 

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) dans un champ TimeLine.




- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) modifier la première heure affichable dans un champ TimeLine.





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 
```wl
// Champ TimeLine sur 2 minutes au total
TimelineChangeMode(TL_TimeLine, plnModeSeconde, 120)

// équivalent à :
TL_TimeLine.DébutEtendueTotale = 0  // commence à 0 sec
TL_TimeLine.FinEtendueTotale = 120 // fin à 120 sec 

// Rendre visible les 30 premières secondes 
TL_TimeLine.DébutEtendueVisible = 0 // commence à 0 sec
TL_TimeLine.FinEtendueVisible = 30 // taille à 30 sec
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître la première date ou heure affichable

`<Résultat> = <Champ manipulé>.DébutEtendueTotale`
---

**`<Résultat> : (Entier ou chaîne de caractères au format Date/Heure)`**

Première valeur affichée dans le champ. Cette valeur correspond : 

- à un entier pour : 

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) un champ TimeLine, 

	- un champ Planning en mode seconde, milliseconde ou microseconde. 




- à une date au format AAAAMMJJHHSSCC pour un champ Planning en mode jour, semaine, mois.




**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler. Ce champ peut correspondre à : 

- un champ Planning. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) un champ TimeLine. 





<a name="SYNTAXE2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) 
### Modifier la valeur de la première heure affichable

`<Champ TimeLine>.DébutEtendueTotale = <Nouvelle valeur>`
---

**`<Champ TimeLine> : (Nom de champ)`**

Nom du champ TimeLine à manipuler. 

**`<Nouvelle valeur> : (Entier)`**

Première valeur à afficher dans le champ. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- L'ensemble des événements représentés par le champ est compris entre **DébutEtendueTotale** et [FinEtendueTotale](../Proprietes/1000020393.md). Cela correspond à l'étendue totale de l'ascenseur ou du range slider associé au champ. 

- L'ensemble des événements visibles dans le champ est compris entre [DébutEtendueVisible](../Proprietes/1000020394.md) et [FinEtendueVisible](../Proprietes/1000020395.md). Cela correspond à la taille de la cage de l'ascenseur ou du range slider associé au champ.

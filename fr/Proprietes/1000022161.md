


## OriginePointDePassage (Propriété)

***En anglais : OriginWaypoint***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **OriginePointDePassage** permet de connaître et de définir l'origine des points dont les coordonnées sont spécifiées dans la propriété [PointDePassage](../Proprietes/1000021957.md). 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Le champ Fil passe par le champ LIB_Centre
FIL_MonFil.OriginePointDePassage = filOrigineFenêtre  
pt1 est un Point = [LIB_Centre.X, LIB_Centre.Y]
FIL_MonFil.PointDePassage = [pt1]
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Connaître l'origine des points spécifiés dans la propriété PointDePassage

`<Résultat> = <Champ Fil>.OriginePointDePassage`
---

**`<Résultat> : (Constante de type Entier)`**

Origine actuelle prise en compte pour les coordonnées des points : 



|   |   |
| --- | --- |
| *filOrigineChampDeDépart* | Les coordonnées sont relatives au point haut gauche du champ de départ du champ Fil. |
| *filOrigineFenêtre* | Les coordonnées sont relatives à l'origine de la fenêtre. <br><br>Remarque : Cette constante permet d'obliger le champ Fil à passer par une position fixe même si le champ est déplacé. |
| *filOriginePointDeDépart* | Les coordonnées sont relatives au point de départ du champ Fil. |



**`<Champ Fil> : (Nom de champ)`**

Nom du champ Fil à manipuler.


<a name="SYNTAXE2"></a>

### Modifier les points de passage obligatoires pour un champ Fil

`<Champ Fil>.PointDePassage = <Nouvelle origine>`
---

**`<Champ Fil> : (Nom de champ)`**

Nom du champ Fil à manipuler

**`<Nouvelle origine> : (Constante de type Entier)`**

Nouvelle origine prise en compte pour les coordonnées des points : 



|   |   |
| --- | --- |
| *filOrigineChampDeDépart* | Les coordonnées sont relatives au point haut gauche du champ de départ du champ Fil. |
| *filOrigineFenêtre* | Les coordonnées sont relatives à l'origine de la fenêtre. <br><br>Remarque : Cette constante permet d'obliger le champ Fil à passer par une position fixe même si le champ est déplacé. |
| *filOriginePointDeDépart* | Les coordonnées sont relatives au point de départ du champ Fil. |




## NbFilsTotal (Propriété)

***En anglais : TotalNbChildren***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **NbFilsTotal** renvoie le nombre total de fils d'une branche : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) d'un champ Arbre,

- d'un champ Table hiérarchique.



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Liste les fils de l'élément sélectionné dans le champ Table hiérarchique
NbFils est entier = TABLEH_MaTable[TABLEH_MaTable].NbFilsTotal
POUR i = 1 à NbFils
	Trace(TABLEH_MaTable[n + i])
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Elément>.NbFilsTotal`
---

**`<Résultat> : (Entier)`**

Nombre total de fils. 

**`<Elément> : (Chaîne de caractères)`**

Elément à manipuler. Cet élément peut être : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une branche d'un champ Arbre.

- Une branche d'un champ Table hiérarchique.

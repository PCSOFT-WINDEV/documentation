


## EnMinutes (Propriété)

***En anglais : InMinutes***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **EnMinutes** permet de convertir une variable de type Durée en minutes. Il est ainsi possible de :

- Connaître le nombre de minutes correspondant à une durée.

- Modifier le nombre de minutes correspondant à une durée.





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
DateDébut est une DateHeure
DateFin est une DateHeure
MaDurée est une Durée

DateDébut = "200812251535"
DateFin = "200812271812"
MaDurée = DateFin - DateDébut
Trace(MaDurée.EnJours)   // 2.109027777778
Trace(MaDurée.EnHeures)  // 50.61666666667
Trace(MaDurée.EnMinutes) // 3037
```


<a name="Exemple2"></a>

```wl
MaDurée est une Durée
MaDurée.EnJours = 1.5
Trace(MaDurée.EnHeures)  // 36
Trace(MaDurée.EnMinutes) // 2160
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le nombre de minutes correspondant à une variable de type Durée

`<Résultat> = <Durée manipulée>.EnMinutes`
---

**`<Résultat> : (Réel)`**

Nombre de minutes correspondant à la durée spécifiée.

**`<Durée manipulée> : (Durée)`**

Nom de la variable de type [Durée](../Motscles/1514069.md) à manipuler.  


<a name="SYNTAXE2"></a>

### Modifier le nombre de minutes correspondant à une variable de type Durée

`<Durée manipulée>.EnMinutes = <Nouvelle durée>`
---

**`<Durée manipulée> : (Durée)`**

Nom de la variable de type [Durée](../Motscles/1514069.md) à manipuler.

**`<Nouvelle durée> : (Réel)`**

Nombre de minutes correspondant à la durée spécifiée.

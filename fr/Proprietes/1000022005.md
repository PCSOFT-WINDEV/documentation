
## Réduit (Propriété)

***En anglais : Minimized***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Réduit** permet de : 

- savoir si une note repositionnable est réduite, 

- réduire ou restaurer une note repositionnable. 



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Déplier un champ Note repositionnable et laisser le champ déplié
NOTE_MaNote.Réduit = Faux
NOTE_MaNote.RéduitAutomatique = 0
```



```wl
// Replier le champ Note repositionnable au bout d'une minute
NOTE_MaNote.Réduit = Vrai
NOTE_MaNote.RéduitAutomatique = 60
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Savoir si une note repositionnable est réduite

`<Résultat> = <Champ Note>.Réduit`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le champ Note repositionnable est réduit. La note est affichée en haut à droite de la fenêtre, 

- <u><u><u><u>Faux</u></u></u></u> si le champ Note repositionnable est affiché entièrement. 




**`<Champ Note> : (Nom de champ)`**

Nom du champ Note à manipuler. 


<a name="SYNTAXE2"></a>

### Réduire ou restaurer une note repositionnable

`<Champ Note>.Réduit = <Réduction>`
---

**`<Champ Note> : (Nom de champ)`**

Nom du champ Note à manipuler. 

**`<Réduction> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le champ Note repositionnable doit être réduit, 

- <u><u><u><u>Faux</u></u></u></u> si le champ Note repositionnable doit être affiché entièrement. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Lorsque la note repositionnable est réduite (propriété **Réduit** à <u><u><u><u>Vrai</u></u></u></u>), la note réduite est positionnée en haut à droite de la fenêtre. 

- La propriété [RéduitAutomatique](../Proprietes/1000021963.md) permet de connaître et définir le délai avant la réduction automatique de la note repositionnable.


## Bulle (Propriété)

***En anglais : ToolTip***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Bulle** permet de connaître et de modifier le texte affiché dans la bulle d'aide associée à un champ d'une fenêtre ou d'une page.

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) **Remarques** :

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) La propriété **Bulle** permet d'insérer des images et du texte coloré dans la bulle grâce aux [fonctions gXXX](../WDLang1/3029028.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) La propriété **Bulle** permet d'insérer du texte au format RTF dans les bulles d'aide.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) L'utilisateur final a la possibilité de modifier les bulles d'aide de son application. Pour plus de détails, consultez [Bulle d'aide des champs (FAA)](../Editeurs/1000021821.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Le texte de la bulle d'aide peut être affiché dans une bulle d'aide standard ou personnalisée. Pour plus de détails, consultez [Bulles d'aide personnalisées](../Editeurs/9500233.md).




**Rappel** : Pour saisir le message affiché dans la bulle d'aide d'un champ sous l'éditeur : 

1. Affichez la fenêtre de description du champ (option "Description" du menu contextuel). 

2. Dans l'onglet "Aide", dans la zone "Bulle d'aide", saisissez le texte de la bulle d'aide. 
	Remarques : 

	- Dans une application multilingue, le texte de la bulle d'aide peut être saisi dans les différentes langues. 

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Si l'option "RTF" est cochée, le texte peut être mis en forme grâce aux options de la barre d'outils RTF affichée. En exécution, la mise en forme sera respectée.

## Exemple

## Syntaxe

### Connaître le texte de la bulle d'aide

`<Texte de la bulle> = <Champ manipulé>.Bulle`
---

**`<Texte de la bulle> : (Chaîne de caractères)`**

Texte actuel de la bulle d'aide associée au champ.

**Remarque** : Si le texte de la bulle est en RTF, le texte renvoyé contiendra les balises RTF. 

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ dont on veut connaître le texte de la bulle d'aide.  


<a name="SYNTAXE2"></a>

### Modifier le texte de la bulle d'aide

`<Champ manipulé>.Bulle = <Nouveau texte de la bulle>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ dont on veut modifier le texte de la bulle d'aide.

**`<Nouveau texte de la bulle> : (Chaîne de caractères)`**

Nouveau texte de la bulle d'aide associée au champ. Si ce texte correspond à une chaîne vide (""), aucune bulle d'aide ne sera affichée pour ce champ.

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Ce texte peut être mis en forme et contenir des images en utilisant les [fonctions gXXX du WLangage](../WDLang1/3029028.md).

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) Ce texte peut correspondre à un texte RTF (fichier RTF ou texte avec les balises RTF).

## Remarques
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) 

### Bulle d'un champ Onglet
<a name="bulle_champ_onglet_ELTPARAGRAPHE000163"></a>Pour les champs Onglet, il est possible de définir une bulle d'aide pour chacun des volets d'onglet.
<a name="NOTE0_2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) 

### Bulle d'un champ Sélecteur et/ou d'un champ Interrupteur
<a name="bulle_champ_selecteur_etou_champ_interrupteur_ELTPARAGRAPHE000172"></a>Il est possible de définir une bulle d'aide pour le champ et pour chaque option. Par exemple :


```wl
SEL_MonSelecteur[1].Bulle = "Option Numéro 1"
```

<a name="NOTE0_3"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) 

### Bulle d'une ligne d'un champ Arbre
<a name="bulle_une_ligne_champ_arbre_ELTPARAGRAPHE000183"></a>Pour les champs Arbre, il est possible de définir une bulle d'aide pour chacun des éléments de l'arbre (fonctions [ArbreAjoute](../WDLang1/3018012.md), [ArbreInsère](../WDLang1/3018023.md) et [ArbreModifie](../WDLang1/3018013.md)).
<a name="NOTE0_4"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) 

### Bulle d'un champ Table et d'une colonne d'un champ Table
<a name="bulle_champ_table_une_colonne_champ_table_ELTPARAGRAPHE000201"></a>L'éditeur de fenêtres permet de définir une bulle d'aide pour un champ Table et pour chacune de ses colonnes. La bulle d'aide concernant le champ Table est affichée uniquement si aucune bulle d'aide n'a été définie pour la colonne survolée.
<a name="NOTE0_5"></a>
<a name="NOTE0_6"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png) 

### Limites
<a name="limites_ELTPARAGRAPHE000238"></a>

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) La propriété **Bulle** s'applique uniquement aux fenêtres et aux champs d'une fenêtre (y compris aux colonnes et cellules d'une table). Cette propriété ne peut pas être utilisée avec des menus. 



![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) La propriété **Bulle** s'applique uniquement aux champs suivants :

- Champ Bouton.

- Champ Libellé.

- Champ de saisie.

- Champ Image.

- Champ Interrupteur.

- Champ Sélecteur.

- Champ Liste.

- Champ Combo.

- Champ Arbre.

- Champ Onglet.

- Champ Table.

- Champ Colonne de table (de type texte, combo ou interrupteur).

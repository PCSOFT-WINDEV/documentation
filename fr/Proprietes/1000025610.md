
## AvecScroll (Propriété)

***En anglais : WithScroll***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **AvecScroll** permet : 

- d'autoriser ou non la gesture de scroll sur un champ Carte, 

- de savoir si la gesture de scroll est autorisée sur un champ Carte. 



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Autorise la gesture de scroll
CARTE_MaCarte.AvecScroll = Vrai
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Savoir si la gesture de scroll est autorisée sur un champ Carte

`<Résultat> = <Champ Carte>.AvecScroll`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la gesture de scroll est autorisée sur le champ Carte, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 




**`<Champ Carte> : (Nom de champ)`**

Nom du champ Carte manipulé. 


<a name="SYNTAXE2"></a>

### Autoriser ou non la gesture de scroll sur un champ Carte

`<Champ Carte>.AvecScroll = <Autorisation>`
---

**`<Champ Carte> : (Nom de champ)`**

Nom du champ Carte manipulé. 

**`<Autorisation> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la gesture de scroll est autorisée sur le champ Carte, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Gesture de scroll
<a name="gesture_scroll_ELTPARAGRAPHE000058"></a>La gesture de scroll consiste à faire glisser un doigt sur la carte vers le haut, vers le bas, vers la gauche ou vers la droite.




## CxMotDePasseDécrit (Propriété)

***En anglais : CtDescribedPassword***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **CxMotDePasseDécrit** permet de définir le mot de passe utilisé pour ouvrir le fichier de données lors d'une connexion OLE DB (fichier défini par programmation uniquement). 


**Rappel** : La fonction [HConnecte](../WDLang4/3044263.md) permet de redéfinir un ou plusieurs paramètres de la connexion OLE DB sur une table. Par défaut, les caractéristiques de la connexion OLE DB sont définies :

- soit sous l'éditeur d'analyses lors de la description du fichier de données,

- soit lors de la description par programmation du fichier de données.



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Définir le mot de passe utilisé pour ouvrir le fichier de données Client
Client.CxMotDePasseDécrit = "MotDePasse"
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Fichier de données>.CxMotDePasseDécrit = <Mot de passe>`
---

**`<Fichier de données> : (Chaîne de caractères)`**

Nom du fichier de données manipulé. Ce nom a été défini grâce au type [Description de Fichier](../WDLang4/1514065.md).

**`<Mot de passe> : (Chaîne de caractères)`**

Mot de passe du fichier à ouvrir par une connexion OLE DB.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
La propriété [MotDePasse](../Proprietes/2510086.md) permet de définir le mot de passe associé à un utilisateur et une connexion.

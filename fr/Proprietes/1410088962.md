
## NomFichier (Propriété)

***En anglais : FileName***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Nomfichier** permet de connaître le nom d'un fichier (fichier WDPic ou fichier généré lors de l'impression). Cette propriété est utilisable sur :

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [iInfoSortie](../WDLang5/1410088954.md).

- une variable de type [WDPic](../WDLang1/1000023503.md).

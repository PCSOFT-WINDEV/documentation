


## ModeTest (Propriété)

***En anglais : TestMode***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **ModeTest** permet de savoir si la publicité affichée dans le champ Publicité est une publicité de test. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI PUB_MaPub.ModeTest = Faux _ET_ PUB_MaPub.PublicitéChargée = Vrai ALORS
	NbPubVue = NbPubVue + 1
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

`<Résultat> = <Champ Publicité>.ModeTest`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la publicité affichée est une publicité de test. 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.  




**`<Champ Publicité> : (Nom de champ)`**

Nom du champ Publicité à manipuler. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Fonctionnement du mode test
<a name="fonctionnement_mode_test_ELTPARAGRAPHE000040"></a>AdMob n'autorise pas l'éditeur de l'application à cliquer sur ses propres annonces ou à augmenter artificiellement le nombre d'affichage de publicités. 

Le mode test du champ Publicité permet d'afficher de "fausses publicités" sur lesquelles le clic n'engendre pas de revenu.

Le mode test est automatiquement : 

- activé lorsque l'application est lancée sur l'émulateur. 

- désactivé en exécution sur le téléphone ou la tablette.


## Minute (Propriété)

***En anglais : Minute***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Minute** permet de connaître et de modifier les minutes d'un élément. Cette propriété est utilisable sur :

- une variable ou une rubrique de type Date, Heure et DateHeure. Pour plus de détails, consulter [Propriété Minute sur les types Date, Heure et DateHeure](../Proprietes/2512050.md).  

- une variable de type [hTâchePlanifiée](../WDLang4/1000017448.md). 

- une variable de type [hPlanification](../WDLang4/1000018853.md).

- une variable de type [Répétition](../WDLang1/1000019262.md).

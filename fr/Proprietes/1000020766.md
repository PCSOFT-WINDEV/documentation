


## VideSiZéro (Propriété)

***En anglais : EmptyIfZero***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **VideSiZéro** permet de : 

- Connaître le mode de gestion de la valeur zéro dans un champ.  

- Modifier le mode de gestion de la valeur zéro dans un champ. 





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SAI_Saisie1 = 0
SI INT_TOUS = Vrai ALORS
	SAI_Saisie1.VideSiZéro = Faux // Affiche "0,00"
SINON
	SAI_Saisie1.VideSiZéro = Vrai // Affiche ""
FIN
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le mode de gestion de la valeur zéro

`<Résultat> = <Champ manipulé>.VideSiZéro`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le champ est vide lorsque sa valeur correspond à zéro, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler : 

- Champ de saisie numérique, 

- Colonne de table ou de table hiérarchique de type Interrupteur.

- Combo ou combo affichant une table. 

- Champ Libellé.





<a name="SYNTAXE2"></a>

### Modifier le mode de gestion de la valeur zéro

`<Champ manipulé>.VideSiZéro = <Mode de gestion>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ à manipuler : 

- Champ de saisie numérique, 

- Colonne de table ou de table hiérarchique de type Interrupteur.

- Combo ou combo affichant une table. 

- Champ Libellé.




**`<Mode de gestion> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le champ est vide lorsque sa valeur correspond à zéro, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
La propriété **VideSiZéro** est utilisable sur : 

- les champs de saisie numérique. 
	Lorsqu'un champ de saisie numérique affiche la valeur 0, il est possible soit d'afficher la valeur, soit de laisser le champ vide. 
	La propriété **VideSiZéro** correspond à l'option "Mise à blanc si zéro" disponible dans l'onglet "Détail" de la fenêtre de description d'un champ de saisie numérique.

- les colonnes de table et table hiérarchique de type Interrupteur en affichage. 
	La case de l'interrupteur peut être affichée ou non si la case n'est pas cochée. 
	La propriété **VideSiZéro** correspond à l'option "Masquer la coche si décoché (en affichage)" disponible dans l'onglet "Détail" de la fenêtre de description d'une colonne de type Interrupteur.

- les champs Combo affichant une liste ou les champs Combo affichant une table. 

- les champs Libellé.

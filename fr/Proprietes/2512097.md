


## FiltreAvecBornes (Propriété)

***En anglais : FilterWithBounds***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **FiltreAvecBornes** permet de savoir si des bornes ont été précisées sur le filtre mis en place par la fonction [HFiltre](../WDLang4/3044100.md) sur un fichier de données, une vue HFSQL ou une requête.






<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Active le filtre
HFiltre(Client, NomClient, "D" + hValMin, "F" + hValMax, "Ville='Paris'")
SI Client.FiltreAvecBornes = Vrai ALORS Trace("Des bornes ont été précisées pour le filtre")
// Désactive le filtre
HDésactiveFiltre(Client)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Elément manipulé>.FiltreAvecBornes`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si des bornes ont été spécifiées pour le filtre avec la fonction [HFiltre](../WDLang4/3044100.md), 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Elément manipulé> : (Chaîne de caractères)`**

Nom du fichier de données, de la requête ou de la vue HFSQL sur laquelle le filtre a été défini.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Valeur des bornes
<a name="valeur_des_bornes_ELTPARAGRAPHE000070"></a>Si des bornes ont été précisées, leurs valeurs peuvent être connues grâce aux propriétés [BorneMin](../Proprietes/2510008.md) et [BorneMax](../Proprietes/2510009.md).
<a name="NOTE0_2"></a>


### Rappel : hValMin et hValMax utilisées dans un filtre
<a name="rappel_hvalmin_hvalmax_utilisees_dans_filtre_ELTPARAGRAPHE000083"></a>Pour réaliser **un filtre générique sur une valeur**, il suffit de :

- compléter la borne minimale avec la constante ***hValMin*** pour lui donner la valeur minimale.

- compléter la borne maximale avec la constante ***hValMax*** pour lui donner la valeur maximale.




Par exemple, pour sélectionner les clients dont le nom commence par "Dupon" :

```wl
HFiltre(Client, Nom, "Dupon" + hValMin, "Dupon" + hValMax)
```
Les clients "Dupon" et "Dupontel" seront sélectionnés.

Pour plus de détails, consultez l'aide de la fonction [HFiltre](../WDLang4/3044100.md).

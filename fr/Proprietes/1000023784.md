


## JetonAutoriseDoublon (Propriété)

***En anglais : TokenAllowDuplicate***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **JetonAutoriseDoublon** permet de : 

- Autoriser ou non la saisie de doublons dans un champ de saisie de type "Jetons de texte". 

- Savoir si les doublons sont autorisés dans un champ de saisie de type "Jetons de texte".



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
SI INT_Doublons ALORS
	SAI_Email.JetonAutoriseDoublon = Vrai
SINON
	SAI_Email.JetonAutoriseDoublon = Faux
FIN
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Savoir si les doublons sont autorisés dans un champ de saisie de type "Jetons de texte"

`<Résultat> = <Champ de saisie>.JetonAutoriseDoublon`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si les doublons sont autorisés,

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<Champ de saisie> : (Nom de champ)`**

Nom du champ de saisie manipulé. 


<a name="SYNTAXE2"></a>

### Autoriser ou non les doublons dans un champ de saisie de type "Jetons de texte"

`<Champ de saisie>.JetonAutoriseDoublon = <Autorisation>`
---

**`<Champ de saisie> : (Nom de champ)`**

Nom du champ de saisie manipulé. 

**`<Autorisation> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si les doublons sont autorisés,

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


- Une erreur est générée : 

	- si le champ de saisie n'est pas de type "Jetons de texte". 

	- Si la propriété [JetonActif](../Proprietes/1000023782.md) correspond à <u><u><u><u>Faux</u></u></u></u>.




- Si la propriété **JetonAutoriseDoublon** correspond à <u><u><u><u>Faux</u></u></u></u> (valeur par défaut) : 

	- l'ajout d'un jeton avec la même valeur mémorisée déclenche une erreur non fatale dans les fonctions [SaisieAjouteJeton](../WDLang1/1000023953.md) et [SaisieInsèreJeton](../WDLang1/1000023954.md).

	- lors d'une saisie directe dans le champ, l'ajout d'un jeton avec la même valeur mémorisée n'est pas prise en compte : la saisie ne se transforme pas en jeton. 




- Si la propriété **JetonAutoriseDoublon** correspond à <u><u><u><u>Vrai</u></u></u></u>, le même jeton peut être ajouté plusieurs fois.

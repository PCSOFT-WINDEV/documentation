
## Formule (Propriété)

***En anglais : Formula***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Formule** permet de décrire une formule dans un document ou une feuille de données. Cette propriété est utilisable sur : 

- une variable de type [xlsCellule](../WDLang5/1000017472.md). 

- une variable de type [docElément](../WDLang1/1000022484.md).

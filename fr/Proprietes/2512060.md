


## TypeClé (Propriété)

***En anglais : KeyType***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **TypeClé** permet de paramétrer les caractéristiques d'une rubrique clé. Il est ainsi possible de :

- Savoir si une rubrique est clé pour une rubrique définie sous l'éditeur d'analyses ou définie par programmation.

- Définir une rubrique clé (lors de la description de la rubrique par programmation uniquement).





<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Définir les caractéristiques la rubrique clé Commande.IdCde
Commande.IDCde.TypeClé = hCléUnique
...
// Valide la description du fichier de données
HDécritFichier(Commande)
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Savoir si une rubrique est clé

`<Caractéristique> = <Fichier de données>.<Rubrique>.TypeClé`
---

**`<Caractéristique> : (Constante de type entier)`**

Les caractéristiques d'une rubrique sont :


|   |   |
| --- | --- |
| *hCléDoublon* | Clé avec doublon. Cette rubrique pourra avoir la même valeur pour plusieurs enregistrements du fichier de données. |
| *hCléPrimaire* | Clé primaire. La valeur affectée à cette rubrique sera uniquement dans le fichier de données et non nulle. |
| *hCléUnique* | Clé unique. La valeur affectée pour cette rubrique sera unique dans le fichier de données. Aucun enregistrement du fichier n'aura la même valeur pour cette rubrique. |
| *hNonClé* | La rubrique n'est pas une clé. |



**`<Fichier de données> : (Chaîne de caractères)`**

Nom du fichier de données manipulé. Ce nom a été défini sous l'éditeur d'analyses ou grâce au type [Description de Fichier](../WDLang4/1514065.md).

**`<Rubrique> : (Chaîne de caractères)`**

Nom de la rubrique manipulée. Ce nom a été défini sous l'éditeur d'analyses ou grâce au type [Description de Rubrique](../WDLang4/1514071.md).

### Définir une clé par programmation

`<Rubrique>.TypeClé = <Type de la clé>`
---

**`<Rubrique> : (Chaîne de caractères)`**

Nom de la rubrique manipulée. Ce nom vient d'être défini grâce au type [Description de Rubrique](../WDLang4/1514071.md).

**`<Type de la clé> : (Constante de type entier)`**

Les caractéristiques d'une clé peuvent être :


|   |   |
| --- | --- |
| *hCléDoublon* | Clé avec doublon. Cette rubrique pourra avoir la même valeur pour plusieurs enregistrements du fichier de données. |
| *hCléPrimaire* | Clé primaire. La valeur affectée à cette rubrique sera uniquement dans le fichier de données et non nulle. |
| *hCléUnique* | Clé unique. La valeur affectée pour cette rubrique sera unique dans le fichier de données. Aucun enregistrement du fichier n'aura la même valeur pour cette rubrique. |
| *hNonClé* | La rubrique n'est pas une clé. |

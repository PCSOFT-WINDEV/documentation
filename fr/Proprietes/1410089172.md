
## Extra (Propriété)

***En anglais : Extra***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Extra** permet de gérer des information supplémentaires. Cette propriété est utilisable sur : 

- les champs, les fenêtres et les pages. Pour plus de détails, consultez l'aide de la propriété [Extra](../Proprietes/1410089327.md).

- les variables de type [kbCarte](../WDLang1/1410089167.md).


## ContentType (Propriété)

***En anglais : ContentType***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **ContentType** permet de connaître et de modifier le type d'un message. Cette propriété est utilisable sur :

- une variable de type [emailAttache](../WDLang3/1000018752.md).

- une variable de type [httpRequête](../WDLang3/1000021158.md).

- une variable de type [httpRéponse](../WDLang3/1000021165.md).

- une variable de type [restRequête](../WDLang3/1000021481.md).

- une variable de type [restRéponse](../WDLang3/1000021477.md).


## EntêteBrut (Propriété)

***En anglais : RoughHeader***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **EntêteBrut** permet de connaître l'entête qui a été retourné avec la réponse. Cette propriété est utilisable sur :

- une variable de type [httpRéponse](../WDLang3/1000021165.md). 

- une variable de type [restRéponse](../WDLang3/1000021477.md).

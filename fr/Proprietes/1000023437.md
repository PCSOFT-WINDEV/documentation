
## VoletNavigationVisible (Propriété)

***En anglais : NavigationPaneVisible***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **VoletNavigationVisible** permet de gérer l'affichage du volet des vignettes dans le champ Lecteur PDF. 
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Affiche le volet de navigation
PDF_MonPDF.VoletNavigationVisible = Vrai
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Savoir si le volet des vignettes est affiché dans le champ Lecteur PDF

`<Résultat> = <Champ Lecteur PDF>.VoletNavigationVisible`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le volet des vignettes est visible, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 




**`<Champ Lecteur PDF> : (Nom de champ)`**

Nom du champ Lecteur PDF à manipuler.


<a name="SYNTAXE2"></a>

### Modifier l'affichage du volet des vignettes dans le champ Lecteur PDF

`<Champ Lecteur PDF>.VoletNavigationVisible = <Nouvelle valeur>`
---

**`<Champ Lecteur PDF> : (Nom de champ)`**

Nom du champ Lecteur PDF à manipuler.

**`<Nouvelle valeur> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si le volet des vignettes est visible, 

- <u><u><u><u>Faux</u></u></u></u> si le volet des vignettes est caché.

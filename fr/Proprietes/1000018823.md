
## Vide (Propriété)

***En anglais : Empty***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Vide** permet de savoir si un élément est vide. Cette propriété est utilisable sur :

- les éléments suivants : 

	- les champs Zone répétée, Graphe, ...

	- les tableaux, les files, les piles. 


 Pour plus de détails, consultez [Vide](../Proprietes/2510126.md).

- une variable de type [xmlAttribut](../WDLang5/1000018799.md).

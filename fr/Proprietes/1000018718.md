
## DateRéception (Propriété)

***En anglais : ReceiveDate***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **DateRéception** permet de connaître la date de réception d'un email. 

Cette propriété est utilisable sur :

- la structure [Email](../WDLang3/3032029.md).

- une variable de type [Email](../WDLang3/1000018713.md).


## MessageSecondaire (Propriété)

***En anglais : SecondaryMessage***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **MessageSecondaire** permet de connaître et de modifier le message secondaire affiché en dessous du message principal de la notification.

Cette propriété est disponible sur : 

- les variables de type [Notification](../WDLang3/1000019441.md). 

- les variables de type [notificationFormat](../WDLang3/1000021301.md).

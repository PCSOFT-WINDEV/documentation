
## Commentaire (Propriété)

***En anglais : Comment***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Commentaire** permet de gérer les commentaires. Cette propriété est utilisable sur :

- une variable de type [sfRequêteTraitementSoumission](../WDLang5/1000018516.md).

- une variable de type [sfRequêteTraitementTâche](../WDLang5/1000018520.md).

- une variable de type [Cookie](../WDLang3/1000019222.md).

- une variable de type [InfoVersion](../WDLang1/1000018905.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [docPropriété](../WDLang1/1000022945.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [diagForme](../WDLang1/1410088082.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [diagSélection](../WDLang1/1410088172.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [mongoLotOpérationsOption](../WDLang4/1410089271.md).


## AvecInclinaison (Propriété)

***En anglais : WithTilt***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **AvecInclinaison** permet : 

- d'autoriser ou non la gesture d'inclinaison sur un champ Carte, 

- de savoir si la gesture d'inclinaison est autorisée sur un champ Carte. 



<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Autorise la gesture d'inclinaison
CARTE_MaCarte.AvecInclinaison = Vrai
```

<a name="XSYNTAXE"></a>

## Syntaxe
<a name="SYNTAXE1"></a>

### Savoir si la gesture d'inclinaison est autorisée sur un champ Carte

`<Résultat> = <Champ Carte>.AvecInclinaison`
---

**`<Résultat> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la gesture d'inclinaison est autorisée sur le champ Carte, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 




**`<Champ Carte> : (Nom de champ)`**

Nom du champ Carte manipulé. 


<a name="SYNTAXE2"></a>

### Autoriser ou non la gesture d'inclinaison sur un champ Carte

`<Champ Carte>.AvecInclinaison = <Autorisation>`
---

**`<Champ Carte> : (Nom de champ)`**

Nom du champ Carte manipulé. 

**`<Autorisation> : (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si la gesture d'inclinaison est autorisée sur le champ Carte, 

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Gesture d'inclinaison
<a name="gesture_inclinaison_ELTPARAGRAPHE000058"></a>La gesture d'inclinaison consiste à faire glisser la carte vers le haut (ou vers le bas) par un mouvement de doigts.

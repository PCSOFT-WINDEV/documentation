


## NumEtiquetteDeDépart (Propriété)

***En anglais : LabelNumberToStart***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **NumEtiquetteDeDépart** permet de :

- Connaître le numéro de l'étiquette sur la planche d'étiquettes à partir de laquelle l'impression est effectuée.

- Modifier le numéro de l'étiquette à partir de laquelle l'impression est effectuée sur la planche d'étiquettes.




Grâce à cette propriété, il est possible d'imprimer sur une planche d'étiquettes où une précédente impression a été réalisée. L'impression ne commencera qu'à partir du numéro d'étiquette spécifiée.

**Remarque** : Cette propriété doit être utilisée dans le code d'ouverture de l'état.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// -- Code d'ouverture de l'état
// Imprime à partir de l'étiquette n°6 de la page
ETAT_EtiqClient.NumEtiquetteDeDépart = 6
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le numéro de l'étiquette où l'impression doit commencer

`<Résultat> = <Etat manipulé>.NumEtiquetteDeDépart`
---

**`<Résultat> : (Entier)`**

Numéro de l'étiquette à partir de laquelle l'impression va commencer. Ce numéro d'étiquette est donné en fonction de la planche d'étiquettes utilisée (les étiquettes étant numérotées de gauche à droite et de haut en bas). 
Par défaut, le numéro de l'étiquette de départ de l'impression est 1.

**`<Etat manipulé> : (Nom d'état)`**

Nom de l'état de type "Étiquette" à manipuler.  


<a name="SYNTAXE2"></a>

### Modifier le numéro de l'étiquette où l'impression doit commencer

`<Etat manipulé>.NumEtiquetteDeDépart = <Nouveau numéro>`
---

**`<Etat manipulé> : (Nom d'état)`**

Nom de l'état de type "Étiquette" à manipuler.

**`<Nouveau numéro> : (Entier)`**

Numéro de l'étiquette à partir de laquelle l'impression va commencer. Ce numéro d'étiquette doit être donné en fonction de la planche d'étiquettes utilisée (les étiquettes étant numérotées de gauche à droite et de haut en bas).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques


### Conditions d'utilisation
<a name="conditions_utilisation_ELTPARAGRAPHE000064"></a>La propriété **NumEtiquetteDeDépart** doit être utilisée dans le traitement d'ouverture de l'état. Il est par exemple possible de passer le numéro de l'étiquette de départ en paramètre à l'état lors du lancement de l'impression.

Cette propriété ne fonctionne que pour les impressions réalisées sur une planche d'étiquettes.

Cette propriété ne peut pas être utilisée si l'état possède un haut et un bas de page.
<a name="NOTE0_2"></a>


### Limite
<a name="limite_ELTPARAGRAPHE000078"></a>La propriété **NumEtiquetteDeDépart** s'applique uniquement aux états de type "Étiquette".

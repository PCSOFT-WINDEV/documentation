
## Handle_R (Propriété)

***En anglais : Handle_R***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le mot-clé **Handle** permet de connaître et de modifier le handle d'un élément. Ce mot-clé peut correspondre : 

- à la fonction [Handle](../WDLang1/3015005.md). Pour plus de détails, consultez [Handle](../WDLang1/3015005.md).

- à la propriété **Handle** d'une variable de type [ FichierDisque](../WDLang1/1410088883.md).

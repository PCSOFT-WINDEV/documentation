


## AngleObservateur (Propriété)

***En anglais : ObserverOrientation***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **AngleObservateur** permet de connaître ou modifier l'angle secondaire d'un champ Carrousel. Cet angle secondaire fait "pencher" le carrousel :

- vers l'avant ou vers l'arrière pour un champ horizontal

- vers la gauche/droite pour un champ vertical.




**Remarque** : L'angle est remis à sa valeur par défaut si la propriété [Vertical](../Proprietes/2510133.md) est modifiée sur le champ.
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
CAR_Carrousel1.AngleObservateur += 15 // Le carrousel penche de 15°
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître l'angle secondaire d'un champ Carrousel

`<Résultat> = <Champ Carrousel>.AngleObservateur`
---

**`<Résultat> : (Entier)`**

Valeur de l'angle secondaire (en degrés, de 0 à 360). 

**`<Champ Carrousel> : (Nom de champ)`**

Nom du champ Carrousel dont on veut connaître l'angle secondaire.  


<a name="SYNTAXE2"></a>

### Modifier l'angle secondaire d'un champ Carrousel

`<Champ Carrousel>.AngleObservateur = <Nouvel angle>`
---

**`<Champ Carrousel> : (Nom de champ)`**

Nom du champ Carrousel dont on veut modifier l'angle secondaire.

**`<Nouvel angle> : (Entier)`**

Valeur de l'angle secondaire (en degrés, de 0 à 360).

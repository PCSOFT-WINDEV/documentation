


## NbComposante (Propriété)

***En anglais : NbComponent***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **NbComposante** permet de connaître le nombre d'éléments d'une clé composée.

**Remarque** : La rubrique peut avoir été créée sous l'éditeur d'analyses ou par programmation.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Connaître le nombre de composante de la clé composée Client.NomPrénom
ResNbComp = Client.NomPrénom.NbComposante
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

`<Résultat> = <Fichier de données>.<Rubrique>.NbComposante`
---

**`<Résultat> : (Entier)`**



- Nombre d'éléments de la clé composée, 

- 1 si la clé n'est pas une clé composée.




**`<Fichier de données> : (Chaîne de caractères)`**

Nom logique du fichier de données manipulé.

**`<Rubrique> : (Chaîne de caractères)`**

Nom de la rubrique clé manipulée.

<a name="NOTE0_1"></a>

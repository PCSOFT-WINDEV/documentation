
## Prix (Propriété)

***En anglais : Price***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Prix** permet de gérer le prix d'un élément. 

Cette propriété est utilisable sur :

- une variable de type [saasTarification](../WDLang3/1000019091.md).

- une variable de type [inAppProduit](../WDLang3/1000020868.md).




## ClicDroit (Propriété)

***En anglais : RightClick***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **ClicDroit** permet de connaître et de modifier le nom du champ Bouton qui sera exécuté lors d'un clic droit sur un objet.

Par défaut, il est possible de définir le nom du champ Bouton exécuté lors d'un clic droit sous l'éditeur de fenêtres, lors de la description du champ (onglet "Détail").

**Remarque** : Si les options "Clic Droit" et "Menu Contextuel" sont renseignées pour le même champ, seule l'action définie pour le clic droit sera exécutée.


<a name="Exemple1"></a>
<a name="sample_code"></a>

## Exemple


```wl
// Changer le bouton associé au clic droit du champ Bouton BTN_Nouveau
BTN_Nouveau.ClicDroit = "Btn_Assistant"
```

<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntaxe

### Connaître le champ Bouton associé au clic droit

`<Champ Bouton> = <Champ manipulé>.ClicDroit`
---

**`<Champ Bouton> : (Chaîne de caractères)`**



- Nom du champ Bouton dont le code sera exécuté lors d'un clic droit sur le champ spécifié, 

- Chaîne vide ("") si aucun champ Bouton n'est associé au clic droit.




**`<Champ manipulé> : (Nom de champ)`**

Nom du champ pour lequel on veut connaître le nom du champ Bouton associé au clic droit.  


<a name="SYNTAXE2"></a>

### Modifier le champ Bouton associé au clic droit

`<Champ manipulé>.ClicDroit = <Nouveau champ Bouton>`
---

**`<Champ manipulé> : (Nom de champ)`**

Nom du champ pour lequel on veut modifier le nom du bouton associé au clic droit.

**`<Nouveau champ Bouton> : (Nom de champ)`**



- Nom du champ Bouton dont le code sera exécuté lors d'un clic droit sur le champ spécifié, 

- Chaîne vide ("") si aucun bouton n'est associé au clic droit.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarques
La propriété **ClicDroit** s'applique uniquement aux fenêtres et aux champs d'une fenêtre (y compris aux colonnes de table). Cette propriété ne peut pas être utilisée avec des menus, etc.

Cette propriété n'est pas disponible pour les champs suivants :

- Champ ActiveX,

- Champ Forme,

- Champ Image non cliquable,

- Champ Jauge,

- Champ Libellé.

<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Le mot-clé **Notification** peut correspondre : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) à une variable de type [Notification](../WDLang3/1000019441.md), qui permet de gérer les notifications iOS et Android.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) à la propriété **Notification** d'une variable de type [gglEvénement](../WDLang5/1000017440.md).


## RetraitGauche (Propriété)

***En anglais : LeftIndent***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **RetraitGauche** permet de connaître et de modifier le retrait à gauche d'un élément. Cette propriété est utilisable sur :

- les champs de saisie et les champs Libellé. Pour plus de détails, consultez [RetraitGauche](../Proprietes/2513018.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) les variables de type [docParagraphe](../WDLang1/1000022483.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) les variables de type [docNiveauNumérotation](../WDLang1/1000022799.md).

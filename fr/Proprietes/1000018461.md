
## Succès (Propriété)

***En anglais : Success***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Succès** permet de savoir si l'opération a été effectuée avec succès. Cette propriété est utilisable sur :

- une variable de type [sfRésultatConvertitPiste](../WDLang5/1000018739.md)

- une variable de type [sfRésultatDéconnecte](../WDLang5/1000018560.md).

- une variable de type [sfRésultatExécuteTraitement](../WDLang5/1000018559.md).

- une variable de type [sfRésultatFusionne](../WDLang5/1000018499.md).

- une variable de type [sfRésultatRestaure](../WDLang5/1000018526.md).

- une variable de type [sfRésultatSauvegarde](../WDLang5/1000018460.md).

- une variable de type [sfRésultatSupprime](../WDLang5/1000018546.md).

- une variable de type [sfRésultatVideCorbeille](../WDLang5/1000018467.md).


## Texte (Propriété)

***En anglais : Text***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Texte** permet de connaître et de modifier un texte. Cette propriété est utilisable sur :

- les rubriques d'un fichier de données : elle permet de savoir si une rubrique est une rubrique de type texte. Pour plus de détails, consultez [Propriété Texte](../Proprietes/2512089.md).

- une variable de type [xmlNoeud](../WDLang5/1000018786.md).

- une variable de type [Filigrane](../WDLang5/1000019922.md).

- ![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) une variable de type [Tuile](../WDLang6/1000020713.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [docElément](../WDLang1/1000022484.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [docFragment](../WDLang1/1000022482.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [docNiveauNumérotation](../WDLang1/1000022799.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [Toast](../WDLang1/1000024638.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [pdfElémentTexte](../WDLang6/1000024556.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [picCalque](../WDLang1/1000024604.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [picTexte](../WDLang1/1000025178.md).

- une variable de type [ocrBlocTexte](../WDLang6/1000025639.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [diagForme](../WDLang1/1410088082.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [diagSélection](../WDLang1/1410088172.md).

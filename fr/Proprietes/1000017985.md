
## Titre (Propriété)

***En anglais : Title***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Titre** permet de connaître et de modifier le titre d'un élément. Cette propriété est utilisable sur :

- les fenêtres, les pages ou les champs Graphe. Pour plus de détails, consultez [Propriété Titre](../Proprietes/2510090.md).

- une variable de type [EvénementTimeline](../WDLang1/1000020612.md).

- une variable de type [gglAgenda](../WDLang5/1000017418.md). 

- une variable de type [gglDocument](../WDLang5/1000018001.md).

- une variable de type [gglEvénement](../WDLang5/1000017440.md).

- une variable de type [gglSociété](../WDLang5/1000017756.md).

- une variable de type [Notification](../WDLang3/1000019441.md).

- une variable de type [notificationFormat](../WDLang3/1000021301.md).

- une variable de type [RendezVous](../WDLang1/1000019244.md).

- une variable de type [rssCanal](../WDLang5/1000017769.md)

- une variable de type [rssEntrée](../WDLang5/1000017792.md)

- une variable de type [rssImage](../WDLang5/1000017799.md)

- une variable de type [rssSaisieTexte](../WDLang5/1000017802.md)


## JournéeEntière

***En anglais : WholeDay***
	

<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **JournéeEntière** permet de configurer un événement pour une durée d'une journée entière. Cette propriété est utilisable sur :

- une variable de type [gglEvénement](../WDLang5/1000017440.md).

- une variable de type [RendezVous](../WDLang1/1000019244.md).


## RetraitDroit (Propriété)

***En anglais : RightIndent***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **RetraitDroit** permet de connaître et de modifier le retrait à droite d'un élément. Cette propriété est utilisable sur :

- les champs de saisie et les champs Libellé. Pour plus de détails, consultez [RetraitDroit](../Proprietes/1000021110.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) les variables de type [docParagraphe](../WDLang1/1000022483.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) les variables de type [docNiveauNumérotation](../WDLang1/1000022799.md).

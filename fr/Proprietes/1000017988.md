
## Auteur (Propriété)

***En anglais : Author***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Auteur** permet de connaître et de modifier l'auteur d'un élément. Cette propriété est utilisable sur :

- une variable de type [EvénementTimeline](../WDLang1/1000020612.md).

- une variable de type [gglEvénement](../WDLang5/1000017440.md).

- une variable de type [RendezVous](../WDLang1/1000019244.md).

- une variable de type [rssEntrée](../WDLang5/1000017792.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [docPropriété](../WDLang1/1000022945.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) une variable de type [pdfDocument](../WDLang6/1000024909.md).

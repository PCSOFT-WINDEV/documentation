


## Nom (Propriété)

***En anglais : Name***
	



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
La propriété **Nom** permet de connaître :

- le nom de la police associée à une variable de type [Police](../Motscles/1514045.md).
	**Remarque** : Le nom d'une police doit correspondre à une des polices installées sur le poste en cours.

- le nom d'un champ, d'un état ou d'un bloc d'un état.

- le nom d'un champ, d'un groupe de champs ou d'une fenêtre.

- le nom d'un champ, d'un groupe de champs ou d'une page.
	




La propriété **Nom** permet de modifier le nom de la police associée à une variable de type Police.

La propriété **Nom** permet également de gérer le nom des différents objets HFSQL. Il est ainsi possible de :

- Connaître le nom d'une connexion.

- Connaître le nom d'un objet manipulé par indirection :

	- fichier de données défini sous l'éditeur d'analyses ou défini par programmation,

	- vue,

	- requête préalablement exécutée avec la fonction [HExécuteRequête](../WDLang4/3044080.md),

	- rubrique définie sous l'éditeur d'analyses ou définie par programmation,

	- liaison définie sous l'éditeur d'analyses ou définie par programmation.




- Définir le nom de l'élément décrit :

	- lors de la description d'un fichier par programmation,

	- lors de la description d'une rubrique par programmation,

	- lors de la description d'une liaison par programmation,

	- lors de la description d'un index full-text par programmation.







![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) Utilisée sur le mot-clé [MaFenêtreFille](../Motscles/1511021.md), la propriété **Nom** permet de récupérer le nom de la dernière fenêtre fille fermée dans une application mobile.

## Exemple

![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png)

```wl
Police1.Taille = 14

// Appliquer la police au champ "MonChamp"
MonChamp.Police = Police1
```

## Syntaxe

### Connaître le nom de l'élément spécifié

`<Résultat> = <Elément manipulé>.Nom`
---

**`<Résultat> : (Chaîne de caractères)`**

Nom de l'élément spécifié.

**`<Elément manipulé> : (Chaîne de caractères)`**

Nom de l'élément à manipuler.  


<a name="SYNTAXE2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png)![Code Utilisateur (MCU)](https://doc.pcsoft.fr/ext/images/fr/MCU.png)![HFSQL Classic](https://doc.pcsoft.fr/ext/images/fr/HF.png)![HFSQL Client/Serveur](https://doc.pcsoft.fr/ext/images/fr/HFCS.png) 
### Modifier le nom de l'élément spécifié

`<Elément manipulé>.Nom = <Nouveau nom>`
---

**`<Elément manipulé> : (Chaîne de caractères)`**

Nom de l'objet dynamique manipulé :

- fichier défini par programmation grâce au type [Description de Fichier](../WDLang4/1514065.md),

- rubrique définie par programmation grâce au type [Description de Rubrique](../WDLang4/1514071.md),

- liaison définie par programmation grâce au type [Description de Liaison](../WDLang4/1514039.md).

- index full-text défini par programmation grâce au type [Description d'IndexFullText](../WDLang4/1000017461.md).




***ou***

Nom de la variable de type [Police](../Motscles/1514045.md) à manipuler.

**`<Nouveau nom> : (Chaîne de caractères)`**

Nouveau nom de l'élément spécifié

Cette police pourra être utilisée :

- Pour écrire du texte dans des dessins (fonction [dPolice](../WDLang1/3029020.md)).

- Pour écrire dans les graphes (fonctions [grPoliceEtiquette](../WDLang3/3042012.md), [grPoliceLégende](../WDLang3/3042037.md), [grPoliceTitre](../WDLang3/3042048.md) et [grPoliceTitreAxe](../WDLang3/3042059.md)).

- Dans les champs d'une fenêtre (propriété [Police](../Motscles/1514045.md) des champs d'une fenêtre).

- Dans les champs d'un état (propriété [Police](../Motscles/1514045.md) des champs d'un état).

### Alias d'une fenêtre
<a name="alias_une_fenetre_ELTPARAGRAPHE000196"></a>Une même fenêtre (fille ou soeur) peut être ouverte plusieurs fois en parallèle. Pour différencier et identifier les fenêtres ouvertes, un alias est attribué à chaque fenêtre.

Pour définir un alias, utilisez les fonctions [OuvreSoeur](../WDLang1/3038045.md) ou [OuvreFille](../WDLang1/3038021.md). Cet alias est utilisé pour identifier les fenêtres avec les fonctions acceptant un nom de fenêtre en paramètre ([TitreEnCours](../WDLang1/3038023.md), [TitreSuivant](../WDLang1/3038043.md), [DonneFocusEtRetourUtilisateur](../WDLang1/1410088107.md), etc.) ou pour manipuler un champ d'une fenêtre donnée.

Pour connaître l'alias d'une fenêtre, utilisez les fonctions [FenEnCours](../WDLang1/3038033.md), [FenPrécédente](../WDLang1/3038014.md) ou la propriété [Alias](../Proprietes/2510004.md).

Si aucun alias n'est précisé lors de l'ouverture d'une fenêtre, WINDEV attribue automatiquement un alias à cette fenêtre.

### Type d'objet
<a name="type_objet_ELTPARAGRAPHE000253"></a>La propriété **Nom** renvoie une chaîne vide ("") pour :			

- les lignes des champs Table, Liste, Liste image ou Combo,

### Nom d'une connexion
<a name="nom_une_connexion_ELTPARAGRAPHE000268"></a>Une connexion peut être créée grâce à une variable de type [Connexion](../WDLang4/1514073.md). Le nom de la connexion correspond alors au nom de la variable. 

Si une connexion de même nom existe déjà (dans l'analyse ou décrite précédemment), le nom de la connexion est défini par le moteur HFSQL qui suffixe le nom de la variable par un numéro.

Dans ce cas, la propriété **Nom** permet de connaître le nom de la connexion définit par le moteur HFSQL.
<a name="NOTE0_6"></a>
![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/fr/UNIVERSALAPP.png) **Connaître le nom de la dernière fenêtre fille fermée**
Dans les applications mobiles, l'ouverture des fenêtres est effectuée par la fonction [OuvreFenêtreMobile](../WDLang1/1000021018.md) (ou la fonction [OuvreFille](../WDLang1/3038021.md)) et non la fonction [Ouvre](../WDLang1/3038035.md). 

Il est possible de connaître le nom de la dernière fenêtre fille fermée dans l'événement "Fermeture d'une fenêtre fille" de la fenêtre ouvrant la fenêtre fille (celle qui a utilisé la fonction [OuvreFenêtreMobile](../WDLang1/1000021018.md) ou la fonction [OuvreFille](../WDLang1/3038021.md)).

Il suffit d'utiliser la propriété **Nom** sur le mot-clé [MafenêtreFille](../Motscles/1511021.md) : 

```wl
// Fermeture d'une fenêtre fille
NomFenêtreFille = MaFenêtreFille.Nom
```
**Remarque** : La propriété [ValeurRenvoyée](../Proprietes/2510134.md) permet de connaître la valeur renvoyée par la fenêtre fille lors de sa fermeture.

### Limites
<a name="limites_ELTPARAGRAPHE000326"></a>![WINDEV](https://doc.pcsoft.fr/ext/images/fr/WD.png) La propriété **Nom** ne peut pas être utilisée sur :

- une option d'un champ Sélecteur ou Interrupteur,

- un élément d'un champ Arbre.


![Java](https://doc.pcsoft.fr/ext/images/fr/JAVA.png) La propriété **Nom** s'applique uniquement aux éléments suivants :

- Champ Bouton.

- Champ Libellé.

- Champ de saisie.

- Champ Image.

- Champ Interrupteur.

- Champ Sélecteur.

- Champ Liste.

- Champ Combo.

- Champ Arbre.

- Champ Onglet.

- Fenêtre.

- Champ Table.

- Colonne de table.

- Superchamp.


## WDXView : Utilisation
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000137"></a>
Cette page d'aide vous présente les différentes fonctionnalités de WDXView :

- Connaître les propriétés, les méthodes, les événements et les constantes d'un ActiveX.

- Tester un ActiveX et ses fonctionnalités.






<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Connaître les propriétés, les méthodes, les événements et les constantes d'un ActiveX
<a name="connaitre_les_proprietes_les_methodes_les_evenements_les_constantes_activex_ELTTEXTE000161"></a>
Pour afficher les informations relatives à un ActiveX :

1. Lancez WDXView

2. Sélectionnez un ActiveX dans la combo "ActiveX". Cette combo liste l'ensemble des ActiveX présents sur le poste. 
	**Remarque** : L'icône de recherche (![](https://doc.pcsoft.fr/fr-FR/images/image.awp?langid=5&name=recherche.jpg)) permet d'effectuer une sélection de l'ActiveX par mots-clés.
	Les différentes informations relatives à l'ActiveX s'affichent.

3. Choisissez l'élément dont vous voulez connaître la description. Cet élément peut correspondre à :

	- une méthode,

	- un événement,

	- une propriété,

	- une constante.







**Remarques** :

- La description des caractéristiques d'un ActiveX correspond aux informations données par le constructeur. Selon les ActiveX, ces informations peuvent être plus ou moins détaillées.   

- Un ActiveX ne possède pas obligatoirement des événements, des méthodes, des propriétés ou des constantes.

## Tester un ActiveX et ses fonctionnalités
<a name="tester_activex_ses_fonctionnalites_ELTTEXTE000185"></a>




### Tester un ActiveX et ses fonctionnalités
<a name="tester_activex_ses_fonctionnalites_ELTPARAGRAPHE000052"></a>Pour tester un ActiveX :

1. Sous WDXView, sélectionnez l'ActiveX voulu.

2. Cliquez sur le bouton "Aperçu". L'ActiveX sélectionné est automatiquement exécuté. 




Pour tester une méthode ou un événement de l'ActiveX sélectionné :

- soit double-cliquez sur le nom de la méthode ou de l'événement à tester,

- soit sélectionnez l'option "Exécuter" du menu contextuel de la méthode ou de l'événement.




Si la méthode attend des paramètres, une boîte de dialogue s'affiche pour saisir ces paramètres. Si plusieurs paramètres doivent être fournis, ces paramètres doivent être séparés par un point-virgule.


<a name="NOTE3_2"></a>




### Modifier une propriété d'un ActiveX
<a name="modifier_une_propriete_activex_ELTPARAGRAPHE000071"></a>Pour modifier une propriété d'un ActiveX :

- soit double-cliquez sur le nom de la propriété,

- soit sélectionnez l'option "Modifier" du menu contextuel de la propriété.




Donnez la nouvelle valeur de la propriété puis validez.


<a name="NOTE3_3"></a>




### Notes
<a name="notes_ELTPARAGRAPHE000085"></a>Seules les propriétés de type **Int**, **Long**, **Short**, **Real**, **Double**, **Date**, **Boolean** et **String** peuvent être modifiées.

Seules les fonctions avec des paramètres et renvoyant un résultat de type **Int**, **Long**, **Short**, **Real**, **Double**, **Date**, **Boolean** et **String** peuvent être exécutées.

## Présentation
<a name="presentation_ELTTEXTE000096"></a>


### L'utilitaire WDXView
<a name="utilitaire_wdxview_ELTPARAGRAPHE000011"></a>WDXView est un visualiseur d'ActiveX. Cet outil permet de connaître, de visualiser et de tester les différents ActiveX installés sur votre poste.

WDXView permet de :

- Connaître les propriétés, les méthodes, les événements et les constantes associés à un ActiveX. Les informations obtenues peuvent ensuite être utilisées pour piloter entièrement un ActiveX par programmation.

- Tester un ActiveX et ses fonctionnalités.




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Lancement de WDXView
<a name="lancement_wdxview_ELTTEXTE000120"></a>
WDXView peut être lancé :

- depuis l'éditeur : sous le volet "Outils", dans le groupe "Utilitaires", cliquez sur "WDXView",

- en exécutant le fichier "WDXVIEW.EXE" présent dans le sous-répertoire "Programmes" du répertoire d'installation de WINDEV ou WINDEV Mobile.



<a name="NOTE2_2"></a>

<a name="conditions_utilisation_ELTPARAGRAPHE000045"></a>WDXView n'est pas redistribuable avec vos applications. WDXView ne peut pas être installé avec les applications développées avec WINDEV et/ou WINDEV Mobile.
